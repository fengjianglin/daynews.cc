大馬司機遇堵車給交警發鈔票，交警立馬給“老闆”開路鞠躬送行

天天要聞
7 天 ago

都說錢不是萬能的，但是沒有錢是萬萬不能的。用錢買「特權」並不是什麼光彩的事情，但是社會上仍有很多人選擇這種歪門邪道，花錢給自己帶來各種便利。

據馬來西亞新聞網站12月11日報道，馬來西亞一名男子為了快速通過擁堵路段，竟賄賂了一群交警。令人驚奇的是，這些交警居然當眾受賄，之後給該司機「開綠燈」，讓他駛離了擁堵路段。

一位名叫Gary Lim的網友在Facebook上發布了一段視頻，並配上了文字:「當你足夠富有時，你可以用錢來避開交通擁堵。」

一段視頻顯示，一名男子為了快速通過一個十字路口，當眾賄賂了兩名交警。隨後，這兩名交警開始叫停其他車輛，以便讓該男子通過馬路。

後來，一群交警聽聞有「老闆」發錢，一個接一個地走到該男子的車邊，總共大約有十人左右。他們都收到了那名司機的現金賄賂，並最終讓這名男子順利地通過了路口，還有一名警察鞠躬為他送行。

目前，Gary Lim的帖子已經被分享了2萬多次。大量網友對該司機的賄賂行為進行了譴責和批評，但也有一些不一樣的聲音，認為該司機「有錢任性」、「交警不受賄，司機就不會行賄。」

類似於此的賄賂事件其實時有發生。1月3日23點47分，南京交警五大隊執勤五中隊在江東北路上，發現一輛白色寶馬轎車行駛異常，於是立刻上前讓司機熄火下車接受檢查。

經初步檢測，該男子存在酒駕嫌疑。在這時，他突然靠近交警說：「我喝了酒，你看，2萬塊打點費夠不夠？」然而，交警們不為所動，以身作則，堅決按規矩辦事。經檢測，該男子涉嫌醉酒駕駛。最終，他因涉嫌危險駕駛罪被捕。

雖然有些時候錢能買到便利，但是我們不能因此去做違法犯罪的事情，像賄賂執法人員，妄圖用財消災的情況，應當徹底杜絕才是。
