美以衛星發現秘密隧道，以軍懷疑伊朗武器中轉通道，準備襲擊摧毀

2019-12-11 1,486條評論 2,423次閱讀 1179人點贊

伊朗再次陷入危機，以色列是最為活躍的。以色列一直都把伊朗當作最大的敵人，在打擊伊朗的問題上也最為積極。以色列不僅僅鼓動美國在打壓伊朗問題上祭出更多的手段，而且還直接針對伊朗的海外基地進行打擊。當然，伊朗與以色列之前的情報滲透與破壞也是一直在進行當中。也就是說伊朗和以色列之間的明爭暗鬥此起彼伏，始終都沒有間斷。伊朗和以色列之間還經常互相放狠話，威脅攻打對方首都等等。針對伊朗的各種軍事行動，以色列也是充分利用衛星圖片等手段進行偵察，並且實施針對性打擊。

以色列對外表示，衛星圖片顯示伊邊境大型隧道。《以色列時報》12月10日報道稱，以色列軍方對外表示，在獲取的最新衛星圖片中，出現了一個超級打隧道。以色列軍方指出，這個大隧道位於伊拉克邊境地區，明顯是一種軍事設施。以色列軍方認為，這條超級隧道起始點就是伊朗伊瑪目阿里軍事基地，應該是供伊朗儲存導彈、炸藥、車輛等軍事武器彈藥的。以色列軍方指出，這條隧道就是伊朗海外軍事基地武器彈藥周轉的證據，也是伊朗在海外實施軍事行動的重要物資供應地。

美國網站披露更多隧道細節，指出該隧道經營已久。美國「福克斯新聞」網披露，該隧道開挖已久，應該是2018年11月份就開始挖掘。，美媒指出目前該隧道已經到了最後階段，距離最終的投入使用不遠了，具體尺寸大概為長約121米，寬至4.5米，深約3.9米，能容納數千名士兵以及大量導彈。美媒指出，有證據現實伊朗可能已經秘密的向伊拉克運送了一批導彈等武器。美媒指出，伊朗支持伊拉克境內的武裝力量，主要是用來對付當地的美國駐軍，以及對以色列實施襲擊行動。

以色列誓言給予摧毀，將不惜採取一切必要手段。以色列時報12月10日報道，以色列軍方指出，伊朗挖掘的這個大隧道，將會成為武器彈藥運輸的重要中轉地，必然將給以色列帶來致命威脅。以色列軍方指出，以軍方將採取一切手段將這個超大隧道給予摧毀。以色列向來善於秘密破壞，並且以色列針對伊朗海外軍事基地的轟炸也頗具成效，所以說以色列真的有能力將這個隧道破壞。以色列的特工世界有名，破壞手段也是不能小視，所以說以色列警告給予破壞，絕對不僅僅是一種威懾，很可能很快就採取行動。

針對可能的襲擊，伊朗必須採取應對措施。伊朗在以色列的襲擊中確實遭遇了較大損失，伊朗多個海外基地曾遭遇以色列空襲。所以這次如果超級隧道真的是伊朗建造，伊朗真的需要做好針對性防範。當然，伊朗如果不能採取有效措施保護這條隧道，那麼只能將其放棄，不然一旦遭遇襲擊，後果將會很嚴重。美以衛星發現秘密隧道，以軍懷疑伊朗武器中轉通道，準備襲擊摧毀，未來的局勢如何走，值得持續關注。
