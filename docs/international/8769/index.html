國際銳評：美國政客對中國治疆“睜眼瞎”簡直是滑天下之大稽

2019-12-09 3,509條評論 4,515次閱讀 3570人點贊

來源:央視網

近日，中央廣播電視總台旗下中國環球電視網(CGTN)陸續推出的英文紀錄片《中國新疆，反恐前沿》和《幕後黑手——「東伊運」與新疆暴恐》，以嚴酷的事實展示了暴力恐怖主義和宗教極端主義在新疆製造的一系列暴行，系統闡述了中國為解決問題所做出的巨大努力。並不令人意外的是，一向在反恐上持雙重標準的某些西方媒體和政客對這兩部紀錄片「選擇性失明」、集體失聲。

人們記憶猶新，當「9•11」恐怖襲擊發生後，全球譴責恐怖暴行，同情美國人民。包括中國在內的世界許多國家主動與美國攜手，積極開展反恐合作。然而，一些美國政客似乎好了傷疤忘了疼，居然以人權為幌子替發生在中國新疆的暴力恐怖主義和宗教極端主義鳴冤叫屈，甚至充當暴恐分子「保護傘」，這簡直是滑天下之大稽！「睜眼瞎」的美國政客這種荒唐之舉，是對國際反恐合作的恣意破壞，也給國際恐怖勢力發出極其危險的信號。

恐怖主義和宗教極端勢力是全人類的敵人，是國際社會共同面臨的安全挑戰，任何一個國家都不能獨善其身。近年來，國際恐怖勢力組織形態由集團化、規模化向多中心、擴散化轉變。報告顯示，2018年上半年，全球42個國家共發生639起恐怖襲擊。

但是，針對恐怖主義和極端勢力如何開展全球治理，新疆依法治理的真實情況如何，這些都不是西方政客真正關心的。他們玩弄「人權」「民主」把戲，對中國治疆政策極盡抹黑污衊之能事，目的就是為了分裂和搞亂中國。

謊言終歸是謊言。CGTN播出的紀錄片以詳盡事實展示了新疆巨大反恐成就以及前所未有的安定繁榮局面，這是任何造謠污衊都無法抹煞的。

由於受到宗教極端思想的侵蝕，新疆曾遭受嚴重衝擊。據不完全統計，自1990年至2016年底，新疆地區發生數千起暴力恐怖案(事)件，造成大量無辜群眾被害，經濟社會發展嚴重受阻。為應對「三股勢力」威脅，保障新疆民眾的基本人權，中國政府從法律、經濟、國際合作等方面入手，在新疆採取了一系列預防性反恐舉措，並取得切實成效。紀錄片以寫實鏡頭，對新疆設立職業技能教育培訓中心等系統性反恐舉措進行了真實記錄。教培工作開展以來，新疆連續3年未發生暴力恐怖案（事）件，人們「快樂的時候跳舞，高興的時候唱歌」。

《國際銳評》評論員今年7月曾參訪過新疆教培中心，所見所聞與紀錄片完全一致。這裡根本不是什麼西方某些媒體和政客聲稱的「集中營」，而是寄宿制學校。學員們對這裡的生活和學習表示很滿意。這裡也沒有什麼鐵絲網，學員的人身自由並未受到任何限制。從這裡畢業的絕大多數學員找到了工作，重新融入社會。去年底以來，70餘批國家和地區官員、國際組織、新聞媒體、宗教團體、專家學者等共一千餘人蔘訪新疆。大家在實地考察之後，充分肯定新疆反恐和去極端化工作的成績做法。

不少外國網友在看過CGTN播出的英文紀錄片後也紛紛感慨：「真相就在那裡」，「是時候去點明那些西方國家的偽善和雙標了」……新疆的和平穩定來之不易，這是國際反恐事業的重要組成部分。對於眼明心亮、真正關心新疆社會發展和當地少數民族人權事業的人來說，紀錄片為他們提供了新疆反恐和去極端化的真實情況。

中國新疆，已經為國際反恐合作做出了重要貢獻，這是任何勢力抹煞不了的。希望「睜眼瞎」的美西方政客們懸崖勒馬、儘快回頭。如果不擇手段到為恐怖主義和極端勢力撐腰打氣的地步，那就失去了人類的道德和理智底線。這種不計後果的瘋狂之舉，遲早是要付出慘重代價的！（國際銳評評論員）
