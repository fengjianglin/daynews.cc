前首相約翰·梅傑當初是如何從威廉和哈里口袋裡拿走40多萬英鎊的

2019-12-07 854條評論 1,808次閱讀 468人點贊

有報道稱，約翰·梅傑爵士被指控在擔任威廉王子和哈里王子的法定監護人時犯了一個巨大的「錯誤」，他「實際上從倆孩子的口袋裡拿走了」40多萬英鎊。

約翰·梅傑「從威廉和哈里的口袋裡拿走了40萬英鎊」

根據《每日鏡報》1998年的報道，這位前領導人也遇到了一些與皇室有關的麻煩。

1997年黛安娜王妃去世後，梅傑被任命為哈里王子和威廉王子的特別監護人，負責法律和行政事務。

據報道，梅傑的任命是在查爾斯王子的建議下做出的，他被要求在關於母親遺囑的談判中保護哈里和威廉的利益。

前首相約翰·梅傑

據《鏡報》報道，約翰爵士被指控在擔任他們的「合法監護人」時犯了巨大的「錯誤」。

據報道，他當時僱傭了一組律師為他提供建議，但在短短6個月內，這組律師們就花掉了40多萬英鎊的經費。

儘管梅傑本人並沒有從中受益，但據說該法案的規模之大令查爾斯王子感到震驚。

報告中寫道:「直到梅傑先生向查爾斯提交了最終的發票供他批準時，查爾斯才知道這些費用被越來越多地花銷掉了。

「這張賬單是5月份提交給查爾斯王子的，但據信梅傑在把賬單交給查爾斯之前已經保管了3個月。」

梅傑先生在黛安娜死後被任命為哈里王子和威廉王子的特別監護人

此外，威斯敏斯特的一位消息人士告訴《鏡報》:「這件事會引起一場可怕的爭吵。

「約翰·梅傑完全把事情搞砸了。

「他似乎讓這項法案失去了控制，即使他密切關注過它，但他卻忽視了讓人們了解情況，實施監督的過程。

他說:「公眾輿論顯然會非常敵對。

「人們想知道什麼樣的建議才有可能價值近50萬英鎊。

威廉王子和哈里王子與約翰·梅傑爵士

約翰•梅傑爵士

約翰爵士是哈里王子和梅根·馬克爾婚禮上唯一的英國政治家

我無法想像查爾斯王子怎麼會同意支付這麼多的錢，這些錢實際上都是從威廉和哈里的口袋裡掏出來的。」

這位消息人士補充道:「一群律師在一位失敗的首相的指導下，將從黛安娜的遺產中攫取大量現金，而這筆錢本應歸這些男孩所有，這種想法讓我們極為反感。」

儘管該法案和當時招致的批評，威廉和哈里似乎沒有表現出任何怨恨。

約翰爵士是2018年哈里王子和梅根·馬克爾婚禮上唯一的英國政治家。
