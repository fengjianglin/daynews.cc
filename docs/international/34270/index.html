媽媽好幫手！英國夜間護士服務火了：整晚照顧嬰兒，一晚1730元

2019-12-24 632條評論 2,423次閱讀 935人點贊

（央視財經《天下財經》）最近，產科夜間護士服務深受英國倫敦媽媽們的喜愛，儘管收費不便宜，但使用了這項服務的媽媽們紛紛表示，這是幫助她們更快恢復正常工作的最佳投資。

這名叫艾娃的嬰兒雖然還只有四周大，但從她小生命誕生的第一周起，她就開始接受「特別」的訓練。而這樣的訓練則是來自艾娃父母選擇的倫敦私人產科護士阿勒曼德。阿勒曼德曾是衛生服務機構受訓的兒科護士，現在的她正在為倫敦有需求的客戶提供一對一的嬰兒相關諮詢和護理服務。在護理服務中最受歡迎的要當屬夜間護理服務了。

根據孩子的需求，夜間護士通常會在晚上8-9點左右到達住家，整晚留下來照顧孩子，轉天早上6點左右再離開。據了解，一個合格的專業夜間護理人員提供一次標準夜間服務的費用大約為190英鎊，約合人民幣1730元。

新生兒母親蘭姆：我認為一個夜間保姆絕對是必要的，這能讓你感到精神煥發，而且因為你也得到了休息，白天就能更好地照顧你的孩子。

經過兩周的夜間護理，艾娃的睡眠習慣得到了很大程度的改善。小寶寶也學會了信任阿勒曼德，聽話地讓她喂自己吃飯和抱起抱下。到了晚上，燈一滅，艾娃看起來睡得很香。

近幾年來，英國倫敦當地越來越流行這樣的產科夜間護士服務。英國當地一家最大的育兒網站編輯蘇西·布恩表示，儘管目前夜間護士服務價格較高，但隨著需求的增加，未來價格將出現下降，能否更好滿足職場媽媽們的需求。

英國某育兒網站編輯 蘇西·布恩：如果你能負擔得起的話，產科夜間護士是一個了不起的解決方案。隨著需求的增加，供應也在增加，我們可以看到價格在下降，這是非常好的，雖然現在還只是一部分人可以負擔得起這樣的服務。

供稿編輯：周陽洋（實習）
