「調研」科迪華洞察：氣候變化威脅加劇，全球食品安全水平下降

2019-12-17 373條評論 4,788次閱讀 1070人點贊

今日，科迪華農業科技®攜手經濟學人智庫發布《2019年全球食品安全指數》報告。

報告顯示，氣候變化對全球食品安全威脅日漸加劇，而創新為降低相關風險提供了可行路徑。在該報告的綜合榜單當中，新加坡以87.4的得分連續第二年位居榜首，中國綜合得分為71，較去年大幅提升11位，來到第35位。

「全球食品安全指數」（Global Food Security Index）是科迪華與經濟學人智庫共同發起的全球食品安全評估體系。該報告2018年10月首度推出，是世界上首個綜合了系統承載力、可用性和產品質量三大維度來觀察食品安全問題的調研。相關研究覆蓋全球113個國家與地區，旨在審視食品體系的活力以及全球環境變化對其造成的影響。

（圖片來源於網路）

科迪華農業科技®對外事務與可持續發展高級副總裁Dana Bolden表示：「通過發布全球食品安全指數，我們希望可以幫助食品生態系統中的所有利益攸關方更清楚地認識到我們在捍衛食品安全上所面臨的機遇與挑戰。從農戶到行業組織，從政府到企業再到消費者，食品價值鏈中的所有環節都有相應的角色需要扮演。科迪華相信，我們有責任以知促行，打造更可持續的生產體系，提高食品的營養以及可獲得性，最終進一步鞏固全球食品安全。」

本期指數新增了農業基礎設施、食品成本和營養標準等幾大關鍵內容，新加坡、愛爾蘭和美國綜合得分前三，與去年相同。而中國則以71的綜合分數居於第35名，在亞洲國家中與新加坡、卡達、日本、阿聯酋、科威特、馬來西亞及韓國皆排名靠前。

新增指標：自然稟賦及修復能力

食品安全與自然環境常被視為相互對立的兩個方面，農業生產被認為會增加碳排放，加劇土地壓力和加速資源枯竭，由此引發的環境危機對農業和食品安全有極大負面影響。2006至2016年十年間在53個發展中國家進行的74項災後需求評估研究顯示，以乾旱、暴雨和洪水為主要形式的自然災害造成了23％的農業損失。

鑒於此，本期指數報告新加入了一項評估指標：自然稟賦及修復能力，以衡量各國對氣候變化風險的承受力。在添加了這一指標後，大部分國家的食品安全表現在新框架之下均有下降，亞洲國家更為突出。食品進口依存度較高的新加坡、阿聯酋和菲律賓的排名分別大幅下降了11位、9位和8位，這凸顯了全球食品體系在面對乾旱、洪水和海平面上升等自然環境威脅時的脆弱。

在「自然稟賦及修復能力」指標之下考察，中國排位第69，僅處於所有國家平均偏下水準，這顯示出中國農業和食品生產所面臨的巨大挑戰：不僅要著眼每年的穩產豐產，更需要確立可持續發展模式，提升糧食生產體系的穩定性和抗風險能力。報告同時指出，數字技術，人工智慧等系列創新有望為中國在應對這一挑戰上帶來突破。中國正積極探索基於機器學習的氣候預測，這或將增強其對氣候危機的抵禦能力，進而整體改善食品安全。此外報告同時指出，減少有害的種植和生產方式，調整農業實踐以適應日趨惡化的氣候並強化食品價值鏈中各環節的創新，也是有效應對環境負面影響，支持農業獲得彈性發展的通用路徑。

基礎設施與營養水平有待強化

報告指出，農業基礎設施建設是推進全球食品安全的關鍵因素之一。雖然目前多數國家的農業基礎設施已得到顯著改善，但可用於灌溉的耕地比例仍無法滿足全球需求，聯合國糧農組織數據顯示，近70％的國家其灌溉設施完備耕地面積不足10%。此外在全球範圍內，農業方面的公共支出也依然不理想。聯合國數據顯示，自21世紀以來，全球在農業領域的公共支出水平遠低於該領域對GDP的貢獻，在東亞和東南亞地區這一情況尤為明顯。

過去15年間，全球農業生產波動趨於穩定，而穩定的農業生產可以幫助各國更好地預測其食品供應能力。本期報告顯示，大部分國家（88%）的人口都得到了充足的食品供應。然而根據聯合國糧農組織關於世界食品安全和營養狀況的報告，全世界仍然有超過8. 2億人正在挨餓。在指數覆蓋的國家中，更有超過三分之一的國家其10%以上的人口營養不良，產業各方在確保「量」的同時也尤待在「質」的提升上取得突破。

此外報告還發現，超過30%的國家缺乏維持正常視力、健康免疫系統和器官功能所需要的維生素A。約四分之一的國家缺乏維持健康免疫系統和新陳代謝功能所必要的營養素鋅。根據預測，到2050年，氣候變化可能會進一步降低農作物中蛋白質、鋅和鐵的含量，導致額外1.75億人口缺鋅，1.22億人口蛋白質攝入不足。
