同樣遭彈劾，特朗普比柯林頓支持率低得多，為何卻更遊刃有餘？

天天要聞
2019-12-17

（1998年，柯林頓即將在白宮發表道歉聲明，同一天眾議院司法委員會就其彈劾條款表決投票）

即便眾議員司法委員會已經開始準備就彈劾條款投票，林賽·格雷厄姆依然試圖在最後一刻前達成一項協議。如果總統承認他所做的一切，他就有可能擺脫那些重罪和輕罪指控。格雷厄姆甚至為總統草擬好了一份聲明稿。

隨著總統本人緩步走向攝像機，司法委員會的成員也放下手頭的會議來到電視機前。在之後發表的聲明中，他承認了自己做下的錯事，但這並沒有達到令格雷厄姆滿意的程度。在他結束演講的九分鐘後，委員會成員按照黨派劃分投票彈劾柯林頓總統。

21年過去了，上周，眾議員司法委員會成員針對另一位總統的彈劾條款展開投票。對於那些目睹了上次總統彈劾事件始末的人來說，眼下發生的一切頗有番似曾相識之感，這其中包括幾乎一模一樣的辯論說辭以及政治人物按黨派站隊等等。

儘管兩件彈劾案十分相似，但它們也存在著眾多不同的地方。

柯林頓彈劾案就好比是電視劇大結局，其情節之緊張讓全世界矚目，整個過程跌宕起伏，讓人焦急不安，在最後一刻到來前，沒人有把握整個事件會如何落幕。

而特朗普案則更像是一場已經持續了三年的全面戰爭中的一次重大戰役，其中所發生的一切都與此前的事存在千絲萬縷的聯繫，它的結局相較之下也明顯少了幾分懸念。

當時林賽·格雷厄姆作為一個共和黨人認為自己能夠在最後時刻同民主黨達成協議避免彈劾的想法 在今天看來近乎是異想天開。這位當年可以協調兩黨達成協議的年輕眾議員現在已經成了一位資深參議員以及特朗普總統的忠誠衛士。柯林頓彈劾案也許是整整一代政治人物所經歷過的分歧最大的時刻。但如今看來，它顯然無法同美國政治今天的分裂局面相提並論。

（1998年12月，時任眾議員林賽·格雷厄姆在彈劾聽證會上）

（如今已是資深共和党參議員的林賽·格雷厄姆在特朗普的彈劾聽證會上）

「當年也存在著許多分歧，不過一個大的區別是現在兩黨缺乏對事實的認同，」加州民主黨眾議員佐伊·洛夫格倫總結道，當年審理過柯林頓案的司法委員會成員中有五位至今還留在委員會中，洛夫格倫是其中之一。

針對柯林頓的彈劾調查正好趕上了福克斯新聞和德拉吉報道等媒體的強勢崛起，當時，整個事件看上去就像是一場充斥著醜陋場面的大型政治真人秀。色情之王拉里·弗林懸賞100萬美元尋找議員們的私生活黑料並成功扳倒了包括候任眾議員議長在內的多位共和黨人。而柯林頓的政敵則繞開主流媒體，選擇在互聯網上散播有關這位總統的八卦流言和陰謀論。

回過頭來看，當時的社會對立和現實撕裂尚處在初期階段。而現在當特朗普彈劾案發生時，在新聞和社交媒體環境中吃香的是那些音量最大，最憤怒的聲音，這種趨勢讓越來越多的美國人陷入到自己的信息孤島上。

陰謀論橫行而散播它們的陰謀論者就在白宮和國會中。柯林頓雖然能在攝像機前向全國民眾講話，但當時還沒有推特平台供他每天發布123條短消息，也沒有福克斯新聞台日日夜夜幫他宣傳自己的政治觀點。

在過去的21年里，政黨內部變得更加同質化，而政治分歧也日趨尖銳。特朗普是現代歷史上最兩極分化的總統，他善於操縱種族，意識形態和經濟裂痕而不是設法彌補它們。儘管柯林頓否認自己違法，但他至少承認了自己犯下的錯誤。相比之下，特朗普不但不認錯，反而還要求政治盟友無條件為自己辯護。

特朗普的批評者早已被其言行激怒，以至於在他上台前就已經開始討論彈劾他。在特朗普涉嫌要求外國政府協助自己打擊國內政敵的事情曝光後，自由派向眾議員議長佩洛西和其他民主黨高層施壓要求他們全力支持對該總統的彈劾。

一些人擔憂彈劾現在正變成一項政治武器。「如果某個政黨掌控白宮而另一個政黨在眾議院占多數的情況在日後繼續出現的話，我們將會看到更多類似的彈劾調查，」作為五位經歷過柯林頓彈劾調查的現任司法委員會成員之一，俄亥俄州共和黨眾議員史蒂夫·夏波擔憂地說。「這在造成分裂的同時讓我們無法專註於其它事務。」

（1999年，時任民主党參議員克里斯托弗·杜德表達對共和黨傳喚柯林頓本人作為證人的不滿）

（2019年，共和黨眾議員路易·戈莫特表達自己的沮喪之情）

（1998年，時任司法委員會主席亨利·海德特同民主黨眾議員小康耶斯在彈劾聽證會結束時交談）

（現任司法委員會主席傑利·納德勒突然推遲針對特朗普的彈劾條款投票，讓共和黨議員道格·柯林斯十分不滿）

民主黨也曾有過這樣的想法。當時，佩洛西和其他眾議員成員寫信給時任共和党參議院領袖，抨擊柯林頓彈劾案中出現「極端黨同伐異」的現象，並聲稱將柯林頓趕出白宮「將違背廣大美國民眾的意願」。

針對柯林頓的控告簡單卻又敏感——在宣誓中作假並阻撓一項性騷擾的司法調查——這些都是明明白白的犯罪行為。不過值得一提的是，這些罪行都是與人而非體制相關。一位總統宣誓維護法律但他沒有做到，但這並未影響到合眾國的利益。有關柯林頓的爭論並非圍繞他做了什麼，而是該對他施以什麼樣的懲罰。

雖然特朗普面臨的指控同樣簡單明了，但沒有柯林頓那麼狗血，對於一些美國人來說這些指控還略微難懂——向外國政府施壓以獲取國內政敵黑料的同時打壓本國國家安全部門。針對特朗普的彈劾條款與違法犯罪無關，但涉及到總統濫用人民賦予他的權力。而且更不要忘記此案又是在特朗普多次打破前任總統普遍遵守的常規的背景之下發生的。

兩位總統都欣慰地看到強盛的經濟表現在彈劾調查期間一定程度上挽救了他們的民調，但在此期間，柯林頓的受歡迎度明顯要更高，其支持率基本維持在60%以上，在被眾議院彈劾後的第二天甚至達到了73%，而特朗普的平均支持率則始終在44%上下。

然而，特朗普對本方政黨的掌控顯然要高於柯林頓。儘管一些共和黨議員在私下裡對他們的總統十分不屑，但在公開場合他們卻團結地表達了對他的支持。沒有一位共和黨眾議員投票支持彈劾調查，可以預見的是也不會有一位為彈劾條款投贊成票。

對比之下，當初有31位民主黨眾議員支持柯林頓的彈劾調查。儘管一些民主黨人並不希望本黨總統被彈劾，但他們在譴責他的不當行為時毫不留情。由於擔心來自民主黨內的壓力會爆發出來並最終迫使自己辭職，柯林頓政府在彈劾調查期間始終處在焦慮的狀態中。

（1998年，反對柯林頓的示威者在等待他的到來）

（上個月，示威者在紐約表達對特朗普的不滿）

眾議院民主黨領袖曾一度判斷將有100位本黨議員投票支持彈劾柯林頓。還有一次，白宮列出了十餘位在他們看來即將「拋棄」柯林頓總統的民主党參議員。柯林頓本人的白宮辦公廳副主任曾和自由派利益集團秘密接觸，討論迫使總統下台的可能。

所有的這一切促使柯林頓團隊選擇了一種能夠拯救他的戰略——儘可能地讓整個彈劾過程帶上黨派鬥爭的色彩。只要維持住敵我分明的態勢，整個民主黨都會聚集在總統周圍，這樣一來即使總統被共和黨占多數的眾議院彈劾，參議院也湊不齊三分之二的贊成票數給他定罪。

這也是為什麼當柯林頓的白宮辦公廳主任在聽說參議院兩黨領袖決定摒棄黨派立隔閡讓議員們自由投票時徹底慌了神。共和民主兩黨達成一致對柯林頓來說可以是致命的。

特朗普則無需擔心這一點。參議院多數黨領袖，共和黨人麥康奈爾目前正積極同白宮內的盟友，而非少數黨領袖舒默商討參議院審判時的規則。

並不是說民主党參議院黨團從未和柯林頓團隊溝通過。在審判期間，參議員可以通過向首席大法官侖奎斯特遞卡片的方式提出有關指控和辯護的問題，負責主持審判的大法官會大聲將卡片上的問題念出。

柯林頓的辯護律師查爾斯·魯夫曾在私底下和時任參議院民主黨領袖達施樂的職員商定好暗號：如果他想獲得反駁眾議院共和黨控方的機會，他就將自己的筆平放在面前的桌上。民主黨一方的助手隨即會以民主党參議員的名義提交問題，詢問魯夫是否要對控方的主張發表看法。

這兩次彈劾調查的相似之處也十分引人矚目。在上周眾議院司法委員會的會議上，有幾個熟悉的場景讓人彷彿看到了一部原版演員參演的經典老劇翻拍：多數黨大談法制而總統所在的政黨則抱怨黨同伐異；一方面白宮的盟友提出反對意見稱沒有「事實證人」，另一方面司法委員會領袖則針鋒相對地堅稱相關證據確鑿無誤。

正如當初民主黨人詆毀負責調查柯林頓的獨立律師肯尼斯·史塔那樣，現在的共和黨人也在攻擊領導調查特朗普的民主黨眾議員亞當·希夫。隨後的結果也完全相同，議員們按照自己的黨派身份針對彈劾條款投票。

（上周四，特朗普在白宮出席活動，與此同時，眾議院司法委員會正在討論針對他的彈劾條款）

這一刻對雙方來說都不是那麼的光彩。「這讓人感到膽寒，」民主黨眾議院希拉·傑克遜·李說，這是她第二次在司法委員會就總統彈劾條款投票。在完成投票離開的路上，她駐足對自己的共和黨同僚，另一位經歷過柯林頓彈劾且至今仍留在司法委員會的眾議員吉姆·森森布倫納說：「我們又走到了這一步，我們一定能熬過這一切。」
