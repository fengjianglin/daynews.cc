特朗普宏大的美墨邊境工程目標，竟然被小小蝴蝶逼停

2019-12-05 1,364條評論 2,854次閱讀 500人點贊

美國總統特朗普之所以能夠上台，與他充滿煽動性的政見迎合了美國人的種種訴求不無關係。

而在他的施政措施之中，一項標誌性的措施就是在得克薩斯州修一堵廣闊的邊境牆，將來自墨西哥的非法移民全都擋在牆外。

在特朗普當選後，作為一國元首的他不僅動用國家機器修牆，還號召支持他的團體建牆。「我們修牆（We Build the Wall）」組織就是一個支持特朗普建牆的大型「粉絲團」，該組織由美國空軍退伍軍人創建。

上個月，「我們修牆」 組織在得克薩斯州邊境開啟了建牆工程。在不過在近日，這場施工被德州的一位法官臨時叫停了，因為他們的行為被動物保護組織抗議，動保組織稱這堵牆影響到野生蝴蝶的棲息。

佔地100公頃的美國國家蝴蝶研究中心坐落於建牆地的附近，該研究中心也是一個大型野生蝴蝶保護區。

顯而易見，該蝴蝶保護區為蝴蝶們提供了廣闊的棲息場所。在他們看來，這座旨在將墨西哥移民擋在美國門外的邊境牆是「非法」的，對該地區脆弱的生物多樣性產生了「威脅」。因此，保護區代表要求法院干涉築牆組織的施工過程。

本周二，德州伊達爾戈郡地方法院法官基諾·瓦奎茲發出臨時禁制令，稱建牆工程有可能對蝴蝶保護區造成「不可逆」 的損害，並叫停了該工程。

蝴蝶研究中心的律師哈維·皮納聲稱，修牆組織的不當言辭及該工程的破壞性，迫使他們採取法律行動。

自從築牆組織開始施工以來，該組織和蝴蝶保護區方就一直有齟齬。

修牆組織創始人布萊恩·柯法奇在推特上曾經貼出視頻，要求該蝴蝶研究中心搬走。

其實，一直在關注美墨邊境動態的國際邊境與水資源委員會，對該組織的修牆工程表示過擔憂，他們要求該組織提供關於建造過程的更多細節，同時阻止其在堤壩上使用重型工程車輛。

這已經不是該組織第一次在美墨邊境因大興土木引發騷亂了。今年5月，該組織在新墨西哥州的施工中就破壞了邊境居民的私有財產。
