未來版哈雷，來自印尼的NEO BOARD TRACKER

天天要聞
2019-12-20

來自印尼的改裝廠牌Smoked Garage此前打造的Royal Enfield讓人印象深刻，也讓他們在東南亞改裝領域有了一定的影響力，這家改裝廠牌在巴厘島以及澳大利亞布里斯班都有工作室，所以這家的改裝作品也有一定的中西結合，既有東方的靈動，也有西方的氣派，這一點從他們之前的作品就能體現出來。

這一次為了參加Suryanation Motorland活動，Smoked Garage打造了一台獨一無二的標誌性作品，歷時三個月，完成了這輛名為「NEO BOARD TRACKER」的作品。與其說是改裝，不如說設計師重塑了一台車，除了保留了哈雷Softail的1340mL發動機以外，這款車的其它部分全部重新打造，整體的車架由鋼管搭建，從搭建的雛形就能看出這款車有多未來了。

除了創新的車架，兩個輪轂也採用了空心造型，不過設計師保留了東南亞的細輪風格，採用了20寸輪轂配Shinko 777輪胎，可以看到這款車並沒有為了突顯氣質用很寬的輪胎，這也是東南亞改裝風格的一大體現。不過在側面造型上依舊能夠感受到很強的未來感和霸氣，比例也把控得很好。




而在細節方面，前減震採用的是FOX單筒減震器，被隱藏在了車殼裡面，由於是硬尾設計，所以設計師在這款車的車座上下文章，設計了一套聯動系統，和整車的風格完美融合。制動系統也是定製的，Brembo卡鉗被巧妙地隱藏了起來，以保證整體的協調性，一同被隱藏的還有排氣系統，整套排氣系統的存在感極低，幾乎很難發現它們的存在。


車把以及腳踏都是為這款車量身打造了，而整車的點睛之筆正是LED燈的設計，輪轂中間的LED燈經過特殊編程，可以隨著速度的提升而改變亮度，搭配全車的暗黑配色，讓人不禁想起了《創世紀》中的那輛經典車型。不過工頭看到這款車的騎姿還是覺得這款車只是靜靜擺在那裡好了。
