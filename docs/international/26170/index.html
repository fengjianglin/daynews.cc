到2035年俄羅斯石油產量可能會上升至6億噸/年以上，約合1200萬桶/日

2019-12-21 1,193條評論 3,989次閱讀 1153人點贊

【能源人都在看，點擊右上角加’關注’】

俄羅斯石油生產商盧克石油公司(Lukoil)預測，如果全球能源供應大幅增加，到2035年，俄羅斯原油和凝析油產量可能會上升至6億噸/年以上，約合1200萬桶/日。

該公司在周二發布的一份關於2035年全球液態碳氫化合物市場主要趨勢的報告中表示，俄羅斯可能主要通過提高石油採收率和開發緻密油儲量來提高產量。。

該公司為該行業到2035年的發展建立了三個場景的模型。該公司估計，如果能源轉型取得進展，原油和凝析油產量可能會降至5億噸/年以上。與此同時，該公司表示，採用激進的氣候政策，將使產量降至4億噸/年以下。

2019年俄羅斯石油產量預計在5.6億噸左右，相當於1125萬桶/日。

主要增長驅動因素

盧克石油預計，難以開採的石油儲量將在俄羅斯產量中發揮越來越大的作用，特別是在2025年後的巴扎諾夫地層。據估計，在保守的開發方案下，巴扎諾夫油田的產量將在2035年達到3500萬噸/年。如果需求增長明顯加快，這一數字可能高達7700萬噸/年。

盧克石油還指出，俄羅斯的低採收率是一個潛在的開發領域，尤其是如果政府引入稅收激勵來提高採收率的話。盧克石油估計，俄羅斯的石油採收率應該是38%，但要低得多，只有22%。

「如果政府提供額外的稅收優惠，石油採收率可能會提高到45%，與美國和挪威相當，」盧克石油說。

該公司認為，北極近海開發可能會因為缺乏有關儲量的信息、高成本以及西方在此期間對俄羅斯的制裁而受到阻礙。

該預測稱，「考慮到俄羅斯企業目前擁有更具吸引力的投資機會，我們估計2030年前應該不會有新的北極生產項目啟動的預期.」

西方的制裁迫使俄羅斯石油生產商重新評估其深水和難以開採的儲量，以及北極近海的儲量。不過，俄羅斯已經在進口替代計划上取得了一些成功。

盧克石油稱，「我們的預估是，最早在2030年，俄羅斯將能夠在對石油工業高效發展至關重要的所有領域大幅降低對進口的依賴.」

俄羅斯公司優先發展國內技術，並加強與亞洲合作夥伴的合作。

稅收，特別是試點的超額利潤稅，也可能對未來十年的生產前景產生潛在的重大影響。

Lukoil表示:「與現有的稅收制度不同，財務結果是要納稅的，而不是每生產一噸石油就要納稅。」「將主要的稅收負擔轉移到油田開發的後期階段，可以通過開發以前不盈利的地區來顯著提高產量，從而增加公共收入。」

俄羅斯將於2020年5月對超額利潤稅試點結果進行審查，試點範圍可能擴大到整個西西伯利亞地區。俄羅斯將在2022年決定是否將這一稅收體系應用於整個行業。

OPEC +合作

盧克石油還預計，根據OPEC+協議，俄羅斯與其他主要產油國的合作將是長期的。

儘管一些俄羅斯生產商抱怨稱，這些限制使長期規劃變得困難，並限制了他們爭奪市場份額的能力，但盧克石油預計不會對長期機會產生負面影響。

該公司表示：「這種生產管理方法不會對儲層質量產生重大影響，因此，幾乎不會對該國的生產機會產生影響。」

盧克石油預計，這一安排將繼續支持市場穩定，並降低價格波動的幅度。

盧克石油公司在2025年至2035年期間對三種不同的油價情景進行了建模。如果全球能源供應大幅增加，油價最高可達90美元/桶。如果能源轉型顯著，油價最高可達70美元/桶;如果採取最積極的氣候政策，油價最高可達50美元/桶。

該公司預計美元將繼續主導石油交易，儘管包括俄羅斯在內的一些國家最近採取行動，在石油貿易中增加使用替代貨幣。

盧克石油表示：「採用替代石油支付方式的一個里程碑式事件是2018年在上海推出了人民幣計價的石油期貨合約。」「儘管如此，在我們看來，在2035年之前的時間範圍內，美元將保持其在國際結算中的主導地位，並繼續影響石油定價。
