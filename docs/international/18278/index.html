英國29歲退伍士兵退伍後因無處可去而自殺，父親說未得到軍隊補助

2019-12-17 1,535條評論 2,869次閱讀 1099人點贊

軍人是鐵骨錚錚的硬漢形象代表，但堅硬外表下的他們也是普通人。據《太陽報》12月16日消息，英國29歲的退伍軍人韋恩·格林因為精神問題在離開軍隊19個月後，於今年11月17日被發現死在朋友家中。

2017年，韋恩辭職入伍，實現了他的軍人夢。退役之前他在蘭開斯特公爵軍團，軍團總部在英國北約克郡的卡特里克。據了解，韋恩退役之後沒有得到軍隊的任何補助，韋恩的父親在接受《星期日人民報》採訪時說到：「他只是想要擁有一個他自己的小地方，但是他離開軍隊後沒有得到過軍隊的幫扶。」

對軍隊執著的韋恩退伍後無法接受退役現實，他整日窩在朋友的沙發上或者是其他的一些旅店中。作為退役軍人，他認為沒有工作的自己是累贅，是負擔。他無所事事，只能抱著手機像個「廢人」，韋恩有著自己的驕傲，但沒有年輕人的朝氣和拼搏向上的勇氣，所以，他最後只能被迫，又主動的選擇死亡。

雖然博爾頓武裝部隊退伍軍人中心在韋恩無家可歸之時給予了他幫助，但這微薄的不解決根本問題的幫助，並沒有辦法讓迷茫的韋恩走出迷茫。據悉，在英國，像韋恩這樣的退伍軍人並不少見，2018年，47歲的達倫·格林菲爾德12月在愛丁堡街頭乞討，凍死在街頭，2015年，一名82歲的前武裝部成員也被發現凍死在睡袋中。僅在今年聖誕，英國就有多達35000名退伍軍人無家可歸。

軍人，是保家衛國的第一人，然而，退役後的他們卻流落街頭，無家可歸，屍骨寒涼，讓人心寒。英國博爾頓武裝部隊退伍軍人中心的負責人表示：「政府和其他當局應該做的多一點，應該為這些無家可歸的人們提供幫助。」
