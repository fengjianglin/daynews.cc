戰鬥在索馬利亞的中械部隊！看看烏干達維和部隊都用啥？全是中國造

天天要聞
7 天 ago

隨著我國武器外貿市場的逐漸擴大，在廣大亞非拉地區，一支支全部裝備「中國製造」的中械部隊也如雨後春筍般冒了出來。在這些部隊中，除了常見的56式槍族仍然經常出現外，還有一大批這幾年我國主推的新裝備也開始在他們當中普及了。下面我們就以在索馬利亞執行維和任務的烏干達軍隊為例，來看看他們這些年都進口了哪些我國外貿的輕重武器。首先就是下圖裡的56式自動步槍，其三棱軍刺太容易識別了，而它也是我國外貿最多的槍械。

與清一色的56式步槍相比，烏干達軍隊在機槍方面就比較豐富了，他們裝備了3款從我國進口的機槍。第一款就是上圖裡的85式12.7mm高機，該槍結構非常簡單，其槍身看起來就像是一根水管子，而且重量只有18公斤，比較容易攜帶和搬運。它所配的三腳架可以將高度調的很低，在降低火線的同時，也增強了槍手射擊時的穩定性。85式高機除了能打54式高機彈藥外，還能打84式12.7mm脫殼穿甲彈，因此有一定的反裝甲能力。

W85式高機

除了85式高機之外，烏干達陸軍還裝備了另一款W85式高機。上面介紹的85式高機主要是以裝備我軍為目的研製，配備給步兵營做支援火器，後來才對外出口，而這個W85式高機則就是以外貿為主要目的，所以它的出口數量要遠大於85式，基本上目前全世界各熱點戰場上都能看到它，風頭完全蓋過了美製M2高機和前蘇聯德什卡高機的風頭。不過它雖然以外貿為目的，但是廠家以它為基礎改進的88式車載高機卻大量裝備我軍各裝甲車輛。

80式通用機槍

除了在高射機槍上選擇我國產品外，烏干達陸軍在通用機槍上也選擇了我國外貿的80式通用機槍，該槍為我國廠家山寨的前蘇聯PKM通用機槍，其性能與PKM基本一致，但是在生產工藝上有一些小差別。注意圖中照片上機匣蓋上的4顆鉚釘，這就是80式與PKM最明顯的不同，PKM的機匣蓋用的一體衝壓生產，所以在這個位置沒有鉚接的地方。80式通機在我軍只是小批量裝備，外貿露面也不是很多，除了烏干達外，柬埔寨軍隊也有裝備。

87式自動榴彈發射器

看完了機槍，再來看榴彈發射器。烏干達軍隊裝備了兩款從我國進口的榴彈發射器，一款是87式自動榴彈發射器，該發射器採用彈鼓供彈，有兩腳架和三腳架兩種狀態。與常見的美製MK19和蘇制AGS-17相比，87式的重量要輕太多了，其兩腳架狀態下才12公斤左右，加上3腳架也才20公斤，所以步兵可以直接手持射擊。不過和西方普遍使用的MGL轉輪榴彈槍相比，它就重了一些。MGL全重只有5.3公斤，不到87式的一半。不過87式雖然比MGL重，但是其彈藥射程要比MGL遠很多，最遠射程達到了1700米，當然這種情況下得使用三腳架打才能保證準頭，如果是手持射擊的話，有效射程其實和MGL的400米也差不了太多。在烏干達軍隊中，87式基本是以兩腳架的輕便射擊狀態為主。

LG2槍掛40mm榴彈發射器

除了自動榴彈發射器外，烏干達陸軍還裝備有LG2槍掛40mm榴彈發射器，其外形和使用方式都比較類似美製的M203榴彈發射器，也是後膛裝填，只是它採用的是側開方式裝填，並不是M203的前推方式。在彈藥方面，除了國產的40mm榴彈外，LG2還能直接使用北約的M433和M406系列榴彈。在使用國產的破甲殺傷彈時，由於其破甲威力達到了40mm級別，所以該榴彈發射器能夠對一些輕裝甲車輛產生相當大的威脅。

另外，在連用迫擊炮方面，烏干達軍隊也選擇了我國產品，而且買的還是近幾年我軍大量裝備使用的PP89式60mm迫擊炮的外貿版：WA121型60迫。該炮最早是在兩山輪戰時期研製的，主要是為替換63式60mm迫擊炮，但是它大規模進入部隊的時間卻並不長，近幾年露面的機會比較多。在我軍中，該炮主要是裝備在山地和高原地區作戰的輕型合成部隊，與89式重機槍一樣屬於連屬支援火器。
