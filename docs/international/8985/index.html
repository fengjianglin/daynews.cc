日媒：日本人經濟泡沫時期囤鑽石 社會轉型後大量賣給中國等新興國家買主

2019-12-07 776條評論 2,092次閱讀 213人點贊

中國小康網12月6日訊 老馬 沉睡在日本家庭里的鑽石吸引了海外的關注。在日本國內最大的珠寶批發商業區，外國買家瞄準了日本經濟泡沫時期日本人從全球買來的鑽石。由於鑽石主人的高齡化，以及家人整理遺物等原因，這些「泡沫時期的遺產」開始出現在二手市場，並不斷流入經濟持續增長的中國和印度等新興市場國。

二手鑽戒

日本經濟新聞報道，會場上立即傳出了「70萬日元！」、「75萬日元！」的聲音。每隔幾秒鐘，價格就會抬高。當外國買家喊出了「86萬日元！」後，會場內恢復了平靜，拍賣錘一錘定音。沒能成功拍下的買家發出了嘆息聲。

在近千家珠寶商聚集的東京御徒町的綜合大樓內，APRE公司和DAIYA Corporation公司10月聯合舉辦了拍賣會，競爭十分激烈。

日本的二手鑽石市場吸引了很多外國珠寶商。APRE表示，在經常參加拍賣會的約80家珠寶商中，外籍競拍者佔到了約40%。這一比例在1年時間內幾乎翻了一番。

尤其是印度和中國的買家。在印度人經營的珠寶批發公司負責競拍的一位男性工作人員表示，「我們將拍到的二手鑽石帶回國進行研磨再加工，價格會漲很多」。

這些漂洋過海來到日本的二手鑽石又繼續流向全世界。除中國和中東的迪拜外，鑽石貿易公司的相關人士表示，「隨著經濟增長，富裕階層不斷增加的東南亞和以色列等地的需求也在高漲」。

日本的二手鑽石為何會備受青睞呢？APRE的社長菊地溫以認為，「因為泡沫時期流入日本的很多高級珠寶在日本家庭處於休眠狀態」。一位中國買家也肯定地表示，日本經濟泡沫時期的二手鑽石比任何國家的品質都好，而且便宜。

日本人的結婚戒指通常會選擇鑽戒。日本在1980年代迎來泡沫經濟時期後，鑽石的需求進一步高漲。據業內相關人士介紹，泡沫時期日本國內賣出了相當於數十萬億日元的珠寶，而鑽石占其中的大部分。

泡沫破滅後日本經濟下滑，鑽石交易量減少。很多鑽石在家庭成為「壓箱底之物」，但最近越來越多的鑽石迴流到了二手市場。因為泡沫時期的鑽石主人已經步入老齡化，或者已經去世。

從事珠寶整理和處置諮詢的Jewelry Asset Managers公司董事上之原匡表示，「很多人是繼承了父母遺產或被老人贈送，過來出售鑽石的人在這4、5年間增加了很多」。

二手珠寶交易店Gold Plaza的鑒定師川崎拓表示，「以年輕人為中心，越來越多的人抱著與其擁有高級珠寶，不如賣了拿錢旅遊的想法。這一觀念的變化也是二手鑽石重現市場的背景之一」。

與拉動全球經濟的國家更迭一樣，鑽石也開始從日本向新興市場國轉移。來自日本的鑽石在異國他鄉重放光彩可能性會越來越多。
