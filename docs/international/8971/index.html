美國又傳來噩耗，十幾輛警車公路上狂奔，激烈槍戰爆發，多人死亡

2019-12-07 905條評論 2,622次閱讀 57人點贊

眾所周知，美國因為不禁槍，所以美國的槍擊案發生率排世界第一，據美國統計數據顯示，每100個美國公民就擁有88把槍，而美國每年就會有1萬人因槍擊案死亡，美國的犯罪率和死亡率一直居高不下，因此美國也一度被稱為「最不安全的國家之一」。近日美國本土又傳來噩耗，據海外網12月5日報道，當地時間5日，美國佛羅里達州上演了一場警匪大戰，十幾輛警車在公路上狂奔，並且還爆發了激烈的槍戰，追擊現場猶如美國大片異常激烈。

據了解，當地時間下午，兩名劫匪持槍搶劫了一家珠寶店，並且與店主發生了激烈的槍戰，現場的一名婦女當場中槍身亡，劫匪在搶劫完後，劫持了一輛卡車進行了逃亡，當地警察接到報警後，立即出動了十多輛警車進行追捕，在追捕期間許多車輛受到驚嚇，紛紛讓道，導致交通一度停滯，不過所幸在警方的努力下，最終劫匪在75號州際公路上被圍住，在一場激烈的槍戰後，兩名劫匪被擊斃，卡車上的人質被救下。事後警方也發布了消息稱，搶劫案導致多人死亡，對此深感歉意，警察機構將會對此事繼續調查。

美國的槍擊案屢禁不止，其根本原因還是因為美國對於槍支的管束不到位，這次槍擊案已經是美國本月發生的第二起案件了，在12月1日上個周日，在美國新奧爾良法國區就發生了一起槍擊案，造成至少11人受傷，其中2人受傷嚴重已經被送往附近的醫院就診，目前這一案件還在調查中，尚未有人被逮捕。

像這類的情況時有發生，有不少網友調侃到：「美國的警察每天都在調查槍擊案件，剛過去的都還沒有調查清楚，新的案件又來了，美國警察真夠忙的。」雖然這只是網友的一句玩笑話，但是不否認美國槍擊案多的事實，美國如果繼續對槍支不加以管束，那麼此後還會有更多此類事件發生。

對於美國的槍擊案，有關人士分析過原因，除了美國不禁槍這個直接原因外，還有三個間接原因：其一就是非法移民的影響，使社會不穩定加劇，犯罪率上升，非法移民過多也容易造成很多矛盾；其二就是美國經濟貧富差距問題，即使再發達的美國貧富差距仍然存在，有些人為了金錢的誘惑就不可避免的走上犯罪的道路；其三美國對未成年人的法治教育問題，美國每年都會關押大量的未成年人，但是卻得不到好的法律教育，因此被釋放之後依然會從事犯罪活動。諸多原因加在一起因此才造成了美國槍擊案高發的原因，如果美國政府不能認識到這些問題並採取必要的措施，那麼美國犯罪率可能還會繼續上升。
