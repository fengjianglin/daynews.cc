備受寵愛的萊昂諾爾公主會成為未來的西班牙女王嗎？

2019-12-21 1,272條評論 4,461次閱讀 2556人點贊

現如今雖然說已進入21世紀，但是在君主立憲制的國家中，王室高顏值的公主依舊是人們矚目的焦點。而作為童話故事裡的寵兒，而這些公主幾乎就是美貌、善良、高貴的代名詞。但是，今天要講的這位西班牙公主，不僅擁有了以上所有美好的辭彙，她還會是將來的西班牙女王，但是想要成為女王，還必須具備以下條件。那麼，這些條件又是為什麼呢？

西班牙為什麼不修改法律？

作為為數不多的保留君主制的國家，西班牙的法律也是沿襲了以往的傳統，而這就是令人頭等的男性繼承王位的順序要優於女性，時至21世紀，西班牙還沒有將這條法律改善的原因卻是十分複雜的。

而這項法律還可以追溯到1978年，正是通過參考當時的《王位繼承法》費利佩王子才成為了西班牙王位第一順位繼承人，其實在費利佩王子的前面還有著兩個姐姐。假若說是按照出生的順序來繼承王位的話，那麼壓根就輪不到費利佩。

其實西班牙王室經歷了動蕩不安的時期，而西班牙擺脫獨裁統治也只不過四十幾年，因此，很多議員認為現在的《王位繼承法》會給予西班牙政治提供一定的穩定性。

因此，假若想要將這部法律修改的話，目前為止似乎是比較困難的一件事。因為此項法律還涉及了與王室有關的事務，包括繼承規則。

更改法律需要具備哪些條件？

更改此項法律中的內容都需要以下條件：1.批准議會對此項法律進行修改，但是需要舉行多票制，也就是需要超過三分之二的人同意才能進行下一步。

2.廢除舊《王位繼承法》，並立即修改新的《王位繼承法》。3.新的《王位繼承法》出台後，通過西班牙議會進行投票，並再次以多數票選通過，也就是超過三分之二的人通過新的《王位繼承法》。

4.最後還要進行全民公投，以民意決定新的《王位繼承法》是否可以通過。

西班牙曾有過修改法律的意向

在2005年，費利佩有了第一個孩子，但是這個孩子卻是一個女孩，也就是萊昂諾爾公主。

公主的降生，讓人們再次審視了《王位繼承法》是否過時了，當時的主要政治力量以及西班牙民眾都認為以男性優先的《王位繼承法》已經不合時宜。

萊昂諾爾公主應該成為最優先繼承王位的人，而不是以男女之分來選擇。而西班牙政府也聽到了民眾的呼聲，於是在原有《王位繼承法》的基礎上提出了以下協議：1.新的《王位繼承法》依舊適用於費利佩王子，因為當時的他是王位第一順序繼承人，而他的兩個姐姐因歷史原因被排除在外。2.費利佩的第一個孩子無論男女，都會成為王位第一順位繼承人。

2008年，西班牙議會對此項協議進行了討論，但是這件事情便開始變得複雜，因為部分政黨想要趁機增加其他改革項目，於是協議裡面又增加了其他的內容，在錯綜複雜的情況之下，這項協議並沒有通過。

此外，當時的萊蒂齊亞王妃又生下了第二個女兒，並且他們似乎也沒有再想要孩子的打算。因此，《王位繼承法》的修訂成為了迫在眉睫的事情。

萊昂諾爾公主會成為西班牙女王嗎？

無巧不成書，2008年西班牙發生了醜聞事件，在王室涉案的成員中，只有費利佩沒有參與其中，這也讓費利佩贏得了眾人的好感。

但是，《王位繼承法》依舊是擺在萊昂諾爾公主面前的絆腳石，萊昂諾爾公主也只是西班牙暫時的王位第一順位繼承人而已，因為除了妹妹索菲亞之外，她再也沒有其他兄弟姐妹了。

而萊蒂齊亞王后今年也已經47歲，雖然年齡並不是很大，但是想要再生育孩子似乎也不太可能。從另一層面上來說，即便是無法更改《王位繼承法》，她還是有可能成為西班牙未來的女王。

雖然西班牙走向共和制的可能性也不是沒有，不過隨著新老國王的交接成功，費利佩低調而積極形象也贏得了民意。

而費利佩一直以來也以謹慎、低調、謙和的形象示人。根據調查，西班牙大部分人也非常支持君主立憲制，這表明費利佩有著較高的支持率，因此，西班牙在短暫時間內是不可能改變體制，而萊昂諾爾公主也很極有可能繼位，並成為西班牙女王。
