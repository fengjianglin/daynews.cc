美國素食女網紅自曝嘗試肉食後相信吃葷比吃素好 粉絲稱被她騙了

2019-12-07 237條評論 1,401次閱讀 399人點贊

據英國《每日郵報》12月6日報道，國外視頻網站上一名推行素食主義的女網紅近日遭到眾多粉絲謾罵，稱被她騙了，原因是她爆料自己在嘗試了30天的純肉飲食後覺得吃葷比吃素好。

這位名叫愛麗絲·帕克（Alyse Parker）的女網紅在YouTube上因長期分享吃素和健身小妙招吸引了70多萬名粉絲關注。然而近日她更新的一則視頻和相關照片給自己惹來了不小的事端，引發粉絲們集體聲討和攻擊。

在視頻中，這名家住美國康涅狄格州的女子稱，她過去30天嘗試了一項挑戰，即為了更好地理解非素食主義者而天天只吃肉食。她不但發布了相關視頻，還分享了幾張照片。讓她沒有想到的是，自己的這項大膽的挑戰卻招來粉絲們極大的憤怒。

愛麗絲在視頻里表示，她挑戰吃肉食是為了體驗這種不同的飲食方式，讓自己的思想對其他飲食方式秉持開放態度。這次挑戰教會了她「要理解他人，而不是評判他們」。她還說挑戰後她發現純肉食實際上促進了自己的消化系統，甚至還讓她的體重減少了兩斤。這比素食給她帶來的效果要好。最後，她還說，以後她雖然不會百分百吃肉食，但是她看到了這種不一樣的飲食對身體的好處。

粉絲們表示他們對愛麗絲放棄持續了近5年的素食主義的決定感到失望透頂，因為他們受到她的啟發和鼓勵，已經改為吃素食了。其中一名粉絲憤怒地寫道：「作為一名素食主義者，你就以為你明白吃肉食對身體的影響了嗎？你就以為你明白這樣對環境更重要了嗎？你失去了我們所有對你的敬意！」

另一名粉絲留言道：「你起先還是啟發我的導師，而現在你卻支持完全相反的飲食法。你的改變讓我感到很遺憾。」還有人懇求她停止向粉絲們宣揚「極端飲食法」，稱「我們能不能停止極端飲食法，停止將其掩飾為健康和改造自我的做法？這太毒害人了！」
