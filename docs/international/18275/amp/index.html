美國大妞雲南走村入寨研究少數民族方言 范秀琳獲評2019年雲南省外國專家“彩雲獎”

天天要聞
2019-12-17

范秀琳走訪彝族村寨。 受訪者供圖

昆明信息港訊（昆明日報 記者張怡）彝語方言錯綜複雜,每個方言區內部都存在語言差異。美國學者范秀琳用13年時間,在彝語西部、中部和南部方言區做了大量田野調查和研究,發現並典藏了部分面臨消失的彝語方言,促進了國內外學界對彝語方言分類的了解。其在瀕危語言記錄和典藏、社會語言學語音變異和研究等方面貢獻突出,於今年獲雲南省外國專家「彩雲獎」。

與雲南結下情緣

范秀琳出生在美國印第安納州,從小就是個熱愛自由的姑娘,為了更好地闖世界她學習了多國的語言。1996年,范秀琳作為交換生來到了杭州,初次來到中國,除了被風景所吸引,漢語與英語的諸多不同也勾起了她的興趣。1997年、1999年、2003年,范秀琳又被學校分別派到北京體育大學、雲南民族大學進一步學習,也是在這幾年范秀琳的漢語水平突飛猛進。

2004年在雲南民族大學學習期間,范秀琳與同在該校教書的一位雲南籍老師相遇相戀了,並於2006年結婚。婚後的范秀琳隨丈夫留在了昆明,並開始謀劃自己未來的工作方向。「我發現雲南的山歌語調優美,歌詞富有詩意,少數民族文化有值得探索的部分,這讓我萌發了研究少數民族語言的想法。」范秀琳說,丈夫很巧就是迪慶州的少數民族,於是她開始接觸起了傈僳語和彝語。

走村入寨收集資料

范秀琳喜歡少數民族語言更多因為語言結構和發音。「藏緬語系習慣把動詞放在賓語後,如『我吃飯』會表達為『我飯吃』,與英語和漢語有很大不同;彝語母音分為常態發音和緊喉兩種,可區分不同意思。我覺得語言是了解一個民族文化和歷史的窗口,能推斷民族文化演變的過程。」隨著學習的深入,彝族方言錯綜複雜的特點讓范秀琳愈加感興趣。

由於地區和語言內部差異大,現有研究大多圍繞六大方言區開展,地方話和土語則資料較少,面臨衰退,很是瀕危。2006年,范秀琳將彝語確定為主要研究方向後,她走進彝族村寨,一次進村就會待一周甚至更久。每次她都會從當地人的發音中尋找研究線索,她準備了很多生活工具、稱謂等日常用語,讓當地人根據詞語發音,再一一記錄。村裡來了外國學者,村民們都很新奇,熱情地配合范秀琳工作。

但是風俗習慣的不同也讓范秀琳吃了不少苦頭。「村民會招待我吃蜂蛹,天哪,我沒有辦法吃那個。」范秀琳笑著告訴記者,正是因為這些小插曲,也讓她更深入地了解民族文化,為掌握語言演變過程奠定了基礎。

願投入更多個13年

從表面上看,范秀琳的研究工作太過理論化,但她認為,這些研究對認識少數民族歷史文化至關重要。如生活在臨滄的彝族認為自己的祖先來自大理巍山,卻鮮有典籍記錄過。一次偶然走訪,范秀琳發現巍山的彝族來到臨滄後,很多語言發音及習慣與當地彝族人很類似。她就此展開研究,為兩地彝族來自同一祖先的民間傳說提供了歷史語言學佐證。

「研究服務雲南民族語言發展,使其得以保留和傳承是一件有意義的工作。」范秀琳說,在研究中,她發現會說新平縣拉祜語拉烏話的人數量已不多,且年齡大多超過70歲,語言面臨消亡;永德縣彝語俐米話則保留了部分古彝語的語音特點,在語言學上有較高研究價值。基於此,她發表了相關語音系統和語言歷史文章,讓國內外學者更好地了解這兩種民族語言。

13年來,除了在彝語方言學研究方面作出了很多研究,范秀琳還發表了楊柳鄉臘羅話的聲調變異、俐侎話聲調的形成等文章,幫助人們了解聲調演化過程,也幫助其他學者了解東亞、東南亞複雜聲調系統的變化趨勢。其研究成果發表的論文和專著被引用70多次,受到了國際語言學界的關注。被雲南師範大學漢藏語研究院聘為碩士研究生導師後,她常帶領學生深入普洱、臨滄、文山等地,了解當地土語,往往去10多個地方,僅能完成一種方言的某個方向研究。

值得一提的是,國外語言學研究理論、方法和成果多以英文發表,對國內英語水平不夠好的少數民族學生而言,理解和學習起來較為困難。范秀琳熟悉國外語言學研究範式,能熟練地使用中英文進行教學和學術交流,為少數民族學生了解國外最新研究理論和方法搭建了橋樑。

她表示,雲南少數民族語言和文化資源豐富而寶貴,其保護和傳承需要更多民族母語人才及高層次民族母語語言學者。她願意繼續投入更多個13年研究雲南少數民族語言,培養雲南學子,為雲南民族文化傳承不斷檔貢獻自己的力量。
