非洲兄弟的五大本土汽車，造型還挺好看， 就是車標設計有點奇怪

提起汽車的話，非洲這個廣袤的大地似乎一直都是消費者的身份，即便是作為生產者，好像也是一些歐洲品牌的製造工廠。不過變革不管在什麼樣的地方都是存在的，一些非洲企業家為了不再被排除在汽車行業之外，現如今非洲本土品牌汽車也開始在當地的汽車市場上有所斬獲，下面這些就是非洲人為非洲生產的幾款車型。

Kantanka

現如今有些非洲人已經擁有了這款讓人非常自豪的迦納汽車。他們的顏色以及型號都非常的充足，車型也比較的廣泛，有SUV車型，也有皮卡車型。這個汽車品牌的製造廠在迦納的中部，據悉這款車是適配迦納當地道路條件的一款本土汽車。並且這個品牌的領導也承認，他們正在和一個中國合作夥伴進行秘密的合作。

Innoson

Innocent Ifediaso Chukwuma是奈及利亞Innoson汽車公司的創始人與開發者。這個公司是奈及利亞本土第1家專門打造汽車的公司，他們生產的車型有小型汽車以及卡車等等。現如今這款車型已經在上市，並且投入使用當中，同時這款車絕對不會限量生產，他們有不少於500台汽車在上市，銷售價格方面在150萬到350萬奈拉之間。

除此之外這款車型還提供引擎和車主保修在內的一系列售後服務，整個的售後表現是非常有誠意的。

Kiira電動車

第3個我們要說的就是Kiira電動車，這是烏干達製造的混合動力的電動汽車，這個車型能夠用電也可以使用柴油作為動力源泉。據悉這款車是一個學生髮起的發明，現如今他們也得到了烏干達政府的支持，在未來或許會在當地有著不錯的影響力。

Mobius

Mobius Motors如今已經發布了他們第1款低成本的汽車。這款車型的價格大概在1萬美元，說實話確實並不算昂貴，並且它是一台多功能汽車，專門為非洲的市場進行打造的。這款汽車，主要就是圍繞著大陸地形以及交通用途進行設計的一款性能汽車，他的空間非常的大，屬於8座車，在載貨空間方面值得肯定。

Turtle

Turtle依然是來自迦納製造的一款車，同時也是為了滿足當地的需求設計的。這款車型結合吉普車以及卡車的外觀，整體的感覺是非常不錯的。

上面這幾款車型都是來自非洲本土，車型表現意外的讓人覺得還不錯。對此大家有什麼想說的嗎？
