“基建狂魔”又一力作，肯亞總統拉中國代表手，豎起大拇指點贊

2019-12-15 2,786條評論 4,945次閱讀 2049人點贊

中國作為「基建狂魔」早已在世界打響了自己的名號，尤其是近年來中國在海外大範圍投資基建項目。最近中國在非洲的又一個工程項目投入運營，中國基建再度成為焦點，與此同時日本在海外的基建卻屢遭碰壁，為何中日海外基建會走向不同的方向？

據媒體報道，12月13日，中企承建的肯亞加里薩50兆瓦光伏發電站正式投入運營，這座東非最大的光伏發電項目將讓肯亞乃至整個非洲都獲益匪淺，可以說這是「基建狂魔」的又一力作，而肯亞人也對中國給予了很高的評價。

在投運啟動儀式現場，肯亞總統親自出席，他還拉住中國代表的手，豎起大拇指為中國企業點贊，並給予了該項目充分的讚揚。近年來，中國在肯亞投資了鐵路、能源等諸多項目，雙方合作廣泛，可以說這也成為「中國基建」的一個很好的宣傳平台。然而這一幕卻讓不少日本人感到觸景生情，因為就在同一個月，日本在海外投資的發電項目黃了。

肯亞加里薩50兆瓦光伏發電站

12月初，越南宣布已經取消了國內的兩個核電站項目，其中一個項目是日本投資的，而另一個是俄羅斯投資的，日本在越南寧順2核電項目也曾投入了很多，包括提供設計方案、建造項目設施以及為越南學員提供培訓。

根據分析，之所以項目會被取消，主要就是越南沒錢了。同時日本在越南的高鐵項目同樣因為資金問題而被無限期擱置，而日本在泰國的高鐵項目也是因為資金沒有談攏導致難以進行下去，實際上這也反映出了中日同樣在海外基建投資，但結果往往不同的重要原因。

核電站項目

對於經濟條件更加不好的非洲地區，中國都能順利建造鐵路、水電站等設施，一方面中國有價格優勢，另外中國在對海外投資的時候，往往會提供優惠的貸款政策，能吸引更多的發展中國家願意接受中國的投資。但日本在海外投資的項目常常缺乏長遠的打算，往往更看中眼前利益。

實際上日本也不是沒有學習過中國的做法，他們為拿下印度高鐵的競標，承諾為印度高鐵提供81%的貸款，而高鐵總費用預計高達11000億盧比，相當於1100億人民幣，日本可以說拿出了很大的誠意。然而上個月，印度官員卻宣布要重新審視高項目，因為印度人要對該高鐵的開發工作、成本估算、障礙和最後期限進行評估，也就是說日本在印度的高鐵項目有被叫停的風險，明明高鐵項目已經開始4年了，卻被印度人來了這麼一出，而這也反映出中日海外基建的另一個不同。

印度高鐵設計

中國在非洲等地的基建項目往往強調中方做好指導工作，並事先對親自對該國的環境進行實地考察排除絕大部分不利因素，而在經過充分的調查之後便於當地進行協商，給出讓雙方都滿意的的方案後就會動工，因此在施工期間也很少出岔子。

相反日本在海外的基建，在很多時候都沒有考慮當地的情況，反而更多依託當地的部門自己開解決問題，這就導致項目容易被各種因素所左右。就像這次其他國家缺錢就把項目停了，而日方也沒有應對措施。此外在印度高鐵上，高鐵項目修了一年就突然遭到沿線居民的抵制，因為他們並不想因修高鐵而搬家，而印度當地官員做了2年的調查都沒意識到這個問題，可見日本投資項目的漏洞有多大。

因此在海外的基建投資上，日本比起「基建狂魔」的差距，不僅僅是在成本和技術上，在管理經驗上也差了不少火候。（子文）
