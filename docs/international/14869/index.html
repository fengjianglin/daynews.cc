砸下141億巨資，拒絕美俄技術的支持，獨立打造屬於自已的太空站

2019-12-16 574條評論 2,900次閱讀 397人點贊

印度的存在，一直成為了全球笑料的「大餅」，但是印度這個擁有13億人口的大國，一直擁有著大國雄心夢。而且向來是把鄰國當成了自己追趕的對象，不光是鄰國有什麼大舉動，印度肯定跟著也一起弄。比如我們自己造了國產航母，印度也要弄。我們進行太空探月計劃，印度也來個「月船2號」進行登陸，但是沒有想到的是，在關鍵時刻，印度慘遭失敗的最終結果。而最近隨著我們國際空間的建設完成，並即將進入太空進行試驗。

印度得知這個消息後，也是不甘寂寞。於是根據印度航天的計劃，2022年將會實現載人航天飛行，同時也將會在5-7年之前建設自己的空間戰，按照印度人的計劃，這個空間站將會容納三名宇航員，同時印度航天部門也表示，不會加入國際空間站，而在未來會獨立運行這座空間站，雖然印度並沒有透露這座空間站會耗資多少錢，但是據英國政府表示，英國政府已經為印度的載人航天登月投入了至少15億美元。

雖然目前印度的空間站，仍然停留在紙面上，但是印度人也表示，自己在登月和火星探索項目上積累了豐富的經驗，而未來中國也會成為俄美之後的第三個航天大國。對於此前印度登月上的失敗教訓，但是印度人的解釋稱，此次探月在最後2.1公里之後突然失聯，但是這也並不意味著登月失敗了，而且印度已經把探月器送到了月球上面，就憑這一點就有很多國家都做不到的。

自從印度公布這個計劃之後，美俄都向印度發出橄欖枝，並表示會提供技術支持。但是印度此次卻表現的非常強硬，表示要憑自己的實力進行研製，不會加入美俄等國的國際空間站。印度此前並沒有類似的經驗，於是就把俄羅斯的空間站圖紙為模板進行改進，同時印度的國際空間站距離地表的距離140公里的距離，而這件20噸重的空間站，但是無率是技術還是規模上都是鄰國的「天宮二號」相媲美。印度計劃在2020年到2021年分別進行兩次無人飛試費，而且花費7年時間去建自己的空間站，同時進行植物培育、重力實驗室進行研究。據印度航天知情人士透露，此次航天計劃，至少要花費141億美元，看來印度這回也是下了血本了。
