齊奧塞斯庫因何下台？表面是一個匈牙利牧師，是其昏招所致

2019-12-24 1,239條評論 4,075次閱讀 1086人點贊

東歐劇變後，唯一個被自己國民處決的元首就是羅馬尼亞前總統齊奧塞斯庫。當時齊奧塞斯庫在羅馬尼亞的國內統治，已經引起民眾不滿，這是羅馬尼亞人反感齊奧塞斯庫的一個重要原因。但真正導致齊奧塞斯庫下台，並非是本國民眾，而是一個外國人。

事件起因

羅馬尼亞的蒂米什瓦拉市，有世代居住在這裡的匈牙利少數民族，1989年10月，該族有一個撒路·透克斯的牧師批評了羅馬尼亞針對少數民族的不當言論。撒路·透克斯是匈牙利人。此人具體是否受西方指派？並無具體記載。但撒路·透克斯批評羅馬尼亞對少數民族人權政策後，羅馬尼亞政府並沒有在意，但羅馬尼亞的主教卻做了一件事，引起民族不滿。

該主席把撒路·透克斯調往羅馬尼亞偏僻的農村去傳教，此時就有信教的人不滿了，並大肆造謠：「此事震撼全國！」雖然言過其實，但在當時網路不發達的情況下，此謠言一出，一起並不是事的宗教言論，引發更多民眾的不滿。

事件發酵

此後針對這起事件的謠言不斷，羅馬尼亞政府開始出面了，認為這是「煽動民族仇恨」。的確這就是煽動民族仇恨，可以低調冷處理此事，此時撒路·透克斯夫婦是自由的。不過羅馬尼亞用了一個很低級手法，本來這個宗教事件快傳聞快要消退了，但羅巴尼亞政府的官方媒體卻開足馬力，大肆批評這起事件。

羅馬尼亞民眾並不知道這起宗教事件，而且也不關心這樣的事。但官方媒體大肆放大此事，並上升為敵我矛盾，老百姓的興趣就來了。一些人開始反對羅政府這樣做。此時蒂米什瓦拉市長也做一件錯事，他一看民眾對政府不滿，於是就公開表示，要對撒路·透克斯進行處分。這位市長要以此轉移民眾視線，但他此後又做了一件蠢事。

這位市長此後不久去首都開會，回來後又宣布支持羅政府驅逐撒路·透克斯的事。將一個宗教事件上綱上線的弄成了敵我矛盾，利用官方媒體大肆抨擊，本來不關心這件事的老百姓也開始感興趣起來，蒂米什瓦拉市長發現勢頭不妙，立刻表示將重新考慮透克斯的處分問題，隨後可能和上級領導開了個會，回來又說他支持驅逐的決定。

老百姓此時徹底不滿了，他們對這位市長出爾反爾的決定更加不滿，於是走上街頭進行示威。12月16日，第一次示威遊行時，只有幾百人，但情緒不滿，出現了激烈的對立場面，而羅馬尼亞的警察和民兵處置不當，也引發民眾不滿。

此時那位「蹩腳」的市長再次出現，他讓示威的民眾回家，保證今天不驅透克斯牧師。民眾被這位市長的話更加不滿，民眾示威的人更加多，而且大部分民眾是羅馬尼亞人。

遊行示威的消息傳開，又有一些人開始在教堂附近聚集。警察卻向示威發射了催淚斯，此時遊行隊伍中發出了「齊奧塞斯庫下台」示威口號。

遊行示威從聲援透克斯牧師的匈牙利少數人，到更多的羅馬尼亞人加入，而且反對聲音也變調，開始反對齊奧塞斯庫執政了。到了12月16日晚上，示威遊行突然演變成騷亂，蒂米什瓦拉市發生了打砸搶事件，有人還燒了汽車，砸了商店。

到了午夜時分，羅馬尼亞的軍隊的坦克和裝甲車進入蒂米什瓦拉市平亂，使騷亂進一步升級。在這平息騷亂中，羅馬尼亞政府又做了一個低級事，居然在示威人群抓住了透克斯和其妻子，並送入監獄。不過軍警在平亂過程中並沒有開槍。

對於那些騷亂分子，這是早有預謀的。而抓捕透克斯完全可以偷偷進行，沒必要當著民眾面抓捕，而且騷亂升級後，已經不是一起宗教事件了。而是一起反政府的暴亂，抓捕一個牧師，只會使事件進一步升級。

而派軍隊也很低級，更讓一些造謠者找到了口實。

毫不在意

到了12月17日早晨，蒂米什瓦拉市區平靜下來，此時清潔工也開始清掃垃圾。蒂米什瓦拉市長也長舒一口氣，他以為騷亂已經平息，但不久，他得知郊外工廠有罷工的消息，而且他還得到上級開槍的命令。

因為齊奧塞斯庫對這起騷亂事件也進行了應對，他召開會議，研究了此事，認為此事目標是他，其中也有西方間諜搗亂，下令抓捕而已。在齊奧塞斯庫眼裡，這樣的事不叫事，他也沒放在心上，隨後就飛往伊朗進行訪問了。

但12月17日中午時，羅馬尼亞的民眾在政府前的現勝利廣場再次聚集，警察進行驅逐，但沒有成功。到了下午，有人衝進政府大樓，他們打碎辦公大樓的玻璃，燒了有關文件，還砸了齊奧塞斯庫的雕像。

到晚上6點左右，謠言開始出，說是軍隊開始開槍。但近年來，從當時參加示威的人回憶，當時示威的人手裡也有槍，而軍方平亂軍人，並沒有向示威人群開火，只是警告性射擊。

但到了18日時，有30個羅馬尼亞人爬到教堂頂樓揮動旗幟，他們被警察射擊。警察認為不驅散他們，騷亂就會再次升級。

羅馬尼亞政府面對國內亂局，暫不對外國人進行簽證。但此時更大謠言出現，有人說這起騷亂死亡6萬人。而後來有報道，平民死了97人，傷了210人，就這一數字，近幾年也沒人提了。
