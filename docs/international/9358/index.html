為什麼日本女性不坐月子，產後恢復也很快？這3點中國女性做不到

2019-12-04 2,063條評論 2,981次閱讀 2052人點贊

為什麼日本女人不坐月子，產後恢復也很快？這3點中國女性做不到

生孩子是個很辛苦的過程，中國媽媽在生完孩子後都會做月子，身體才會逐漸恢復，而且在這個期間，花費的金額也相當高，有些有錢人還會把產婦送去月子中心。

相比之下，國外有些國家都不需要坐月子的，但是因為種族不同、習慣也並不一樣，很多人也並不意外。但是同為亞洲黃種人的日本女性，在生活習慣和體質上都和中國女性十分相似，那為什麼她們在生完孩子之後也不需要坐月子，而且產後恢復也很快呢？這3點原因中國女性可能都做不到。

日本女性對身體的要求很嚴格

在日本，准媽媽懷孕後都會定時去做產檢，在醫院醫生都會通知孕婦要適當的保養，最重要的就是控制體重，不能過度肥胖。當然國內醫院的醫生也這樣說過，可是因為有些老一輩的人，希望能夠給孕婦補充更多的營養，希望寶貝可以健康，所以說在懷孕期間就吃的特別胖，根據各國的調查，現如今日本在控制體重方面，做的其實非常好。

日本孕婦就算是懷孕，體態也不會特別臃腫，雖然肚子很大，但從背後看的話，並沒有什麼太大的變化。更何況日本女性對自己的身材要求也相當高，在成年人當中肥胖率只佔到整體人數的3.7%。

食材里有非常多的營養

日本人在吃的方面也很重視，在1985年的時候，他們就制定了比較健康的飲食指南，倡導百姓一天當中吃30多種食材，這樣就可以有效的吸收更多的營養了。更何況日本原本就是一個島國，有非常多的海產品，平時吃的東西多多少少都會有一些魚類。

經過專業的研究人員表明，長時間吃深海魚，可以有效的降低人身體當中的有害物質，尤其是膽固醇。還可以減少心血管病發病的概率，再加上他們平時吃鹽的量相對較少，身體素質會更好一些，補充營養的同時，就可以更好的保持身材。

日本人非常喜歡走路

日本人的環保意識很強，在路上也很少看到私家車，平時就算是家裡有車，可能也喜歡乘坐公共交通或者是騎自行車上班。一般狀態下運動很長一段時間，一個人只要每天保持運動30分鐘以上，那麼對心臟就會有其他的幫助，可以有效的促進血液的流動，但是相比之下，國人的經濟壓力生活壓力都非常大，平時根本就沒有那麼多的時間去運動，如果走路的話根本就來不及上班，所以沒有辦法，必須要打車或者是坐公交車。

沒有了鍛煉的機會，身體素質自然就會慢慢變差，日本的女性之所以不用坐月子，就是因為她們提前已經做好了準備，如果孕媽希望在日後恢復的速度能夠快一點，那在平時也要多多健身，這樣就不會非常辛苦了。
