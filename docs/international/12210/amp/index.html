芬蘭女總理當選背後的女性福利：小孩出生後，母親可帶薪休假三年

天天要聞
2019-12-10

當地時間12月8日，芬蘭交通部長桑娜·馬林擊敗黨內競爭對手，成為芬蘭執政聯盟中最大黨社會民主黨的黨魁，成為芬蘭的新一任總理。

34歲的馬林現在是全世界最年輕的在任總理，同時也是芬蘭第三位女性領導人。

成長於一個由兩個媽媽組成的「彩虹家庭」中，馬林的童年並不一帆風順，但是兩位母親給了她絕對的信任和支持。

27歲時，她就憑藉著自己的努力成為了芬蘭第三大城市坦佩雷市議會主席；2015年進入國會，成為社民黨副主席；今年6月接棒交通和通信部部長，並在前總理林內生病期間代理黨主席職務。

雖然這位準總理在受訪時直言「從來沒有想過自己的性別或年齡問題」，但不可否認的是女性力量在芬蘭政壇日益崛起。

女性力量引領芬蘭政壇

馬林將要領導的聯合政府由五個黨派組成，所有黨魁均是女性，其中有四位年齡不到35歲。連前總理，保守黨成員亞歷山大·斯圖布都不禁發推讚賞芬蘭的現代和進步。

分析人士認為本屆政府的全女子軍團可能是個巧合，但女性角色在芬蘭政壇越來越耀眼確實是肉眼可見的事實，女性對於政治權利的爭取在芬蘭也是一個歷史悠久的課題。

芬蘭是歐洲第一、全球第三個女性獲得投票權的國家，也是世界上第一個女性擁有被選舉權的國家。

1907年，芬蘭女性獲得被選舉權的次年，就有19位女性當選議會委員，占議會席位近十分之一。

但爭取到政治權利後芬蘭女性，面對的並非坦途。1917年，芬蘭從沙皇俄國獨立，女性政治代表的人數卻有所下降，在兩次世界大戰期間，女性議員的人數只有一次達到了10%，並在1930年的選舉中進入低谷，只有11為女性議員（5.5%）當選。

這大概是因為戰時的芬蘭人並不需要女性在議會中出現，他們認為孕產和家庭才是女性最重要的工作和工作場所。

二次大戰後，芬蘭開始和平發展，議會中對女性人數也開始穩步上升。到80年代，女性國會議員對比例已經超過了30%。90年代初期，女性開始打破政治領域的天花板，深入到各個政治領域和所有等級的政治職位中。

上升到高位的芬蘭女性領導人往往都有一些共性。她們的支持者不僅包括自己的黨內的女性組織，通常還有許多女性利益集團。除此之外，她們在男性群體中也有良好的聲譽，非常願意並能夠和黨內黨外的男性同僚合作。

2000年，芬蘭選出了第一位女性總統塔里婭·哈洛寧。根據芬蘭平等法案的規定，國家管理委員會、諮詢委員會委員會以及市政委員會等所有相似機構都必須遵守男性和女性僱員都不得少於40%的配額要求。現在，芬蘭議會的女性議員佔比47%。

芬蘭議會中女性議員人數的變化 / 網路

不僅女性和政治

在芬蘭，不僅僅是女性的政治訴求能夠得到充分表達，對於平權的追求早已延伸到了所有性別的各個社會領域中。性別平等一直都是芬蘭的核心價值觀。

20世紀60年代，性別活動家們成立了男女平權聯合會「第九組」，男女平權問題在芬蘭社會得到廣泛討論，他們尤其關注男性和女性在育兒問題上面的平等問題。

此後十年，芬蘭一直跟進這一理念，並建立了第一個育兒假制度，法律也首次通過了幼兒日托服務的相關規定。現在在芬蘭，新生兒出生後，母親將獲得四個月的帶薪休假，父親也會獲得超過兩個月的帶薪休假。

即便是在育兒假結束後，父母有權留在家中帶孩子，每月領取450歐元，直到孩子三歲，再回到原來的工作崗位繼續工作。

性教育是性別關係中另一個重要的問題。從20世紀70年代年代起，芬蘭開始允許墮胎，並在小學課程中納入了關係和性教育課程，並通過立法保證了節育服務。

芬蘭還建立了LGBT權利保護機制，不再將同性戀行為定位犯罪，為受到暴力的女性建立了第一個庇護所。

在僱傭領域，芬蘭在1972年成立了兩性平等為理事會，一年前通過僱傭合同法案禁止了基於性別的就業歧視。2000年，禁止所有範圍的歧視已被明確列在芬蘭憲法中，1987年開始實行的男女平權法案在2005年得到更新。

除了這些個別的法案，芬蘭也設置了相關機構推動並監管平權問題。性別平等問題由芬蘭社會事務和衛生部負責，接受國務院的協助。其下設立了許多部門在不同領域努力，包括性別平等委員會，平等問題監察部門等。

此外，幾乎所有政府部位都設立了性別平等工作組，許多公共機構都有性別平等專家和性別平等聯絡點網路。各大城市和勞動市場也有自己的性別平等機構。

在這些努力下，芬蘭成為了全球兩性差別最小，性別平等指數最高的國家之一。

爭取性別平等，步履永遠不停

雖然芬蘭在性別平權方面已經遙遙領先，但芬蘭爭取性別平等的鬥爭還遠遠沒有結束。據統計，芬蘭仍然存在非常大比率的基於性別的暴力。有47%的芬蘭女性都曾經受過身體或性暴力，在歐盟國家中位列第二。

同時，對芬蘭的移民和薩米原住民女性都存在著歧視現象。在勞動市場上，移民女性受雇的可能性小於在芬蘭出生的女性。而現實是芬蘭的移民數量正在逐年增加，在2015年，移民約佔芬蘭人口的6%。

不管怎樣，芬蘭性別領域取得的成績和付出的努力都值得其他國家借鑒。
