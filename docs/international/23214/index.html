日本公司的"大失敗獎"火了！員工虧了5000萬，不僅沒被開除，還能得獎！就連老闆都拿過這個獎

轉載：日本設計小站（ID：japandesign）

自古做的好的有獎勵

做的差的遭懲罰

但你有聽說過

「大失敗獎」嗎？

就是失敗越大、給公司造成的損失越多

越能得獎的那種

日站君發誓

絕對沒有騙人

這事就發生在日本大阪的一家公司

20年前

這家做汽車配件的公司

都是把活兒外包出去的

為了激發自家員工的創意

有一天老闆決定

像那些車載香氛、方向盤之類的小配件

自己來開發試試唄


然而，想法很美好

可現實hin殘酷

員工做出來的那些

完全賣不出去…

一下子就讓公司虧了5000萬日元！

（相當於300萬人民幣）

天了嚕，5000萬誒

要是擱在我們這兒

絕對就是

挨罵！罰錢！開除！

虧了公司5000萬的人

不開留著過年嗎？？

但這家神奇的公司偏偏反其道而行之

不僅沒開除

還給他頒了一個「大失敗獎」！

而且還煞有介事的弄了個

公開的頒獎典禮

就是這頒獎的場面么

略微有些尷尬哈哈哈哈哈哈哈

領獎的人不好意思地

摸摸後腦勺

下面的同事

向你露出善意？的微笑

喏，獎狀長這樣

社長說，頒這個獎

就是為了告訴大家

即使失敗也沒關係！

最重要的是要勇於挑戰！

畢竟在摸索過程中難免會失敗

但那也能為之後留下寶貴的經驗

所以就算失敗了也要表揚

要給予大家繼續挑戰的動力

社長人也太好了吧！！

我要是那名員工

聽到那席話

可能會當場感動到哭出來

在這樣的公司里

我甘願再工作100年！

而如此好的社長本人

當然也拿過這個獎哈哈哈哈哈哈哈


本來因為5000萬日元的損失

所有員工的獎金都被削減了

整個公司都籠罩在一種低迷、壓抑的氛圍中

大家在開發新產品時更加束手束腳了

生怕一個不小心

就虧損了下一個5000萬…

但頒了這個獎之後

員工們又開始積極起來了

怕啥？

就算虧錢了還能被表揚呢

而那位得獎的員工

自此之後發奮努力

最後成功在上海開了工廠

把公司營業額提高了5倍！

現在已經晉陞成公司的董事啦

這是什麼神反轉劇情

簡直比電視劇還勵志

事情傳到網上

中國網友看了紛紛表示




喂…「大失敗獎」不是為了鼓勵失敗

而是鼓勵創新、鼓勵挑戰啊

下面這位網友的評論

也讓日站君深有感觸

對失敗的寬容更能讓人心生感激

而這種情感的影響

絕對會比憤恨更正向

但目前，這家公司還只是「別人家的公司」

眾所周知，日本是出了名的社畜集中營

多的是各種慘無人道的

奴、隸、制、度

比如你敢相信

他們為了防止員工打瞌睡

開發了一個「眼皮監測系統」

當檢測到有人昏昏欲睡時

就立馬調低空調溫度

把員工活生生凍醒…

再比如一些公司

在員工出去跑業務時

強制他們使用一款APP

專門記錄你走了多少步

在全組人排名多少

所有這一切上司都實時監控

就問你還敢不敢偷懶！


但最苦的莫過於新幹線的員工

幾乎是用繩命在工作！

因為2015年曾有一次車廂零件脫落

導致一名女乘客受傷

從此以後，一項可怕的「體感研修」就成了

西日本安全管理工作人員的必修課

受訓員工必須戴著安全帽和防風鏡

蹲在軌道旁一個深約1米的坑裡

當列車經過受訓點時

會故意把速度提到最高！

可憐的員工得近距離地

感受時速300公里的列車

從身邊呼嘯而過

其心情估計跟「從鬼門關前走了一遭」沒兩樣

鐵路公司是想藉此驚醒員工

一點失誤都不能有

否則自己也會遭殃！

但員工知道零件脫落的危險性

不就行了嗎

這種「玩命訓練」

簡直不把員工當人看

都算不上是訓練、而是欺凌！

猶記得前段時間

萬達女高管因為業績不達標

被施壓要撤了她的總經理職務

結果不堪重負，選擇跳樓

看的人心惶惶

項目都不敢做失敗了

否則要麼滾、要麼死

真心希望中國的企業

能多一些人情味兒

我們不鼓勵失敗

但可以對失敗多一些寬容

畢竟上班族們

都真的真的真的…
