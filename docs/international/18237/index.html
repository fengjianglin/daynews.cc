王室評論員：查爾斯從性格孤僻變得愛開懷大笑，是卡米拉改變了他

2019-12-17 597條評論 1,900次閱讀 510人點贊

英國康沃爾公爵夫人卡米拉，因為和查爾斯王儲的感情故事，多年來一直飽受爭議。但這些一直沒有影響到他倆之間的感情，倆人一直情比金堅，而且查爾斯似乎還因為卡米拉改變了不少。

據《每日郵報》12月16日報道，一位王室評論員告訴英國媒體，卡米拉把查爾斯王子，從一個「不愛笑的憂鬱角色，徹底轉變成了一個快樂自信的未來國王。」

查爾斯王子的童年並不快樂，在他的成長期，他的母親剛成為英國女王，忙於工作對他疏忽照顧。在菲利普親王的堅持下，查爾斯被送到軍事化管理學校上學，這對一向懦弱的查爾斯來說，留下了很深的童年陰影。在查爾斯成年後，也不止一次的埋怨過英女王對她不好，他一直是個悶悶不樂的王儲。他一直認為，相比起他的弟弟妹妹，他得到的母愛並不多。

英國皇家傳記作家彭妮朱諾向媒體透露，在過去的幾年裡，她發現查爾斯發生了巨大的變化，你在屏幕上經常看到他開懷大笑的樣子，而這些在以前是很少看到的。以前的查爾斯孤僻，嚴肅，笑得不多，是卡米拉改變徹底改變了他。彭妮說，現在的查爾斯王子終於是一個快樂的人。

今年72歲的卡米拉比查爾斯大一歲，她現在為王室承擔很多公職，在不少活動中都可以看到卡米拉有著非常幽默的一面。本月初卡米拉出席電影首映禮的時候，還將演員逗得哈哈大樂。王室評論員認為，卡米拉的幽默性格是讓查爾斯有所改變的關鍵，卡米拉把查爾斯在憂鬱狀態中拉了出來，她徹底改變了他。

儘管卡米拉和查爾斯的感情一直被負面報道了20多年，但兩人似乎還依然愛得熱烈，經常能在公開活動中看到他們深情對望。卡米拉近年來擔任的公職任務也越來越多，這也代表著英女王和王室對她的認可度越來越高。
