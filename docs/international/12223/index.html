強大的俄羅斯鐵甲洪流，或捲土重來，新型裝甲車和步戰車將要量產

2019-12-10 1,942條評論 2,904次閱讀 92人點贊

近日，俄羅斯媒體報道，”迴旋鏢”裝甲車的裝甲運兵車版本K-16和步兵戰車版本K-17的量產將於2022年開始。

2015年紅場閱兵，俄羅斯展示了以T-14″阿瑪塔”坦克，T-15步兵戰車，”庫爾幹人-25″和”迴旋鏢”輪式裝甲車為代表的新一代坦克裝甲車輛。著實讓世界驚艷了一把。

那個時候俄羅斯可謂豪情壯志，當時的俄羅斯陸軍總司令奧列格·薩留科夫向官方媒體表示，到2020年，俄羅斯將擁有2300輛”阿瑪塔”坦克，屆時將佔到俄羅斯坦克庫存的70%。

結果烏克蘭事件爆發，俄羅斯遭到制裁+能源價格大幅下跌，俄羅斯經濟狀況陷入困難，新武器的研發和生產也受到了影響。

整個2019年，T-14大概只有12輛坦克和和4輛裝甲回收車輛改型在俄羅斯陸軍第一近衛坦克軍測試，由於經濟困難，T-14的量產化工作一推再推。最開始說2016，後來推到了2018，再後來又推到2019，現在才說”將要量產”，這都已經2019年12月了，還沒見到量產型的T-14坦克下線。

T-14主戰坦克在西部軍區第1近衛坦克集團軍測試 ​​​​

其實要說T-14起碼一直還有點消息，2015年和T-14同台亮相的”庫爾幹人-25″，T-15步戰車和”迴旋鏢”裝甲車連報道都不多，也就是參加參加各種武器展會才能見一面。比如俄羅斯每年一度的”軍隊”系列國際軍事技術論壇。

在”軍隊-2018″上展的安裝57mm炮通用無人炮塔的T-14步兵戰車。

“迴旋鏢”在2017年還被砍了一波配置，在”軍隊-2017″論壇上展出的”迴旋鏢”裝甲車把炮塔換成了一個升級版的BMP-2″別列若克”炮塔。這個炮塔倒不是差，事實上配置挺不錯的，但是比起原本的Epoch遙控無人炮塔還是差了一個檔次。

BMP-2炮塔版本

Epoch遙控無人炮塔版本

不過”迴旋鏢”後續發展進度挺快，俄羅斯衛星通訊社2019年4月26日報道，俄羅斯軍事工業公司、最新一代裝甲運兵車”迴旋鏢”研製企業總經理亞歷山大·克拉索維茨基表示，”迴旋鏢”裝甲車將按計劃在今年啟動國家試驗。

而後據俄羅斯衛星網2019年11月25日報道，俄羅斯陸軍總司令奧列格·薩柳科夫在接受俄軍機關報《紅星報》採訪時表示，已簽署向俄軍供應試用批次步戰車”庫爾幹人-25″和”迴旋鏢”的國家合同。但是沒說明具體數量，而且僅僅是試用批次，還不到量產階段。而且也不知道量產版本的”迴旋鏢”用的是哪一款炮塔。

“迴旋鏢”是俄羅斯計劃中的下一代通用輪式底盤，定位類似解放軍的ZBL-08系列，有多種衍生型號，結果到現在一個實車都沒見到，就有一個搭載125mm炮的輪式坦克計劃和一個防禦增強版的計劃。防護增強版的戰鬥全重將從現有的24噸增加到34噸，能夠全向防禦”布雷德利”的25毫米機關炮。

125mm輪式坦克版”迴旋鏢”

但是無論如何，2019年我們最多能看到”迴旋鏢”進入俄軍中進行測試，量產至少要等到2022。相比起T-14，輪式裝甲車的製造要簡單一些，筆者認為2022年”迴旋鏢”量產是能做到的，但是很可能僅限於步戰車版本和裝甲運兵車版本（運兵車版本就是12.7mm機槍）。

面對新武器難產，而北約步步緊逼的現狀，俄羅斯陸軍迫切需要加強軍備，不得已走起了改進手頭現有武器的路子，最典型的就是俄羅斯主戰坦克的”現代化三兄弟”：T-72B3M，T-90M和T-80BVM。尤其是T-72B3M頗受重視，以前風光一時的T-90A反而受到冷落。目前俄羅斯陸軍大概有1850輛T-72B3M，是俄羅斯坦克部隊的絕對主力。另有500多輛T-90和T-80系列正在進行升級。升級後的坦克在面對北約坦克的時候至少不會落於下風。

2019年5月9日紅場閱兵T-72B3M上了T-90A沒上就很能說明問題了，連BMPT都上了。

俄羅斯陸軍總司令奧列格·薩柳科夫2019年6月2日在接受《紅星報》採訪時稱，俄陸軍2019年將接收2500多件新式型號的武器、軍事和特種技術裝備。但是這些新裝備大多數還是老裝備的改進型，比如S-300V4、”山毛櫸-M3″、”道爾-M2″、”柳樹”等防空導彈系統，地面武器就是T-72B3M，T-90M，T-80BVM，BMP-2M和BMP-1AM。

BMP-1AM就是BMP-1換了BTR-82A炮塔

俄羅斯主管國防工業的副總理鮑里索夫也更傾向於用採購新型主戰坦克的經費來升級俄軍現役的T-72、T-80和T-90主戰坦克。

不過這回俄羅斯方面是真不打算拖下去了，包括T-14和蘇-57在內的新武器，都咬死了要在2019年投入量產，哪怕是一架或一輛也好，也要進入量產階段，俄羅斯實在是太需要新武器了。

蘇-57的量產型一號機，圖中這是2019年11月初的狀態，如無意外，2019年應該能交付。

俄羅斯這個舉動其實是可以理解的，新武器在試驗台上永遠無法形成戰鬥力，也永遠無法形成威懾。退一步來說，對於這些武器的潛在買家來講，一直到現在俄羅斯自己都沒有裝備量產型，那麼我為什麼要購買呢？俄羅斯可是已經吧T-14和蘇-57都開始向外銷售了。

俄羅斯這境況還真是不容易，他不是沒技術，也不是沒想法，更不是沒有生產能力，就是沒有錢來擴大生產和採購，實屬窩心。

這個表情包真的很能體現俄羅斯現在的心情……
