暖心！普京單膝跪地，與殘疾人運動員自拍合影

2019-12-04 1,678條評論 3,272次閱讀 1602人點贊

​12月3日是國際殘疾人日。當天，正在索契視察的俄總統普京專程前往俄羅斯國際奧林匹克大學，看望在那裡學習的殘疾人運動員。

普京觀看了兩個展覽，並與殘疾人運動員們親切交流。

「我們在倫敦比賽的時候，當時是莫斯科時間凌晨1點，但總統依然關注著我們的比賽。」一位倫敦奧運會銀牌得主握著普京的手，感謝總統一直以來對他們的關注和鼓勵。

「我看了比賽！很為您感到緊張揪心！」普京說。

​普京指出，國際奧林匹克大學創辦已經有十年了，吸引來自全國各個地區的運動員代表前來學習深造。目前，學校獨獨缺少來自楚科奇的學員，但相信很快會覆蓋到這一地區。

普京強調說，所有人都應該從事體育運動，而殘疾人運動員身殘志堅，為大家樹立了很好的榜樣。

與坐在輪椅上的殘疾人運動員合影時，普京很自然地蹲下身來，單膝跪坐地板，將笑容定格在手機屏幕上。

​交流期間，足球奧運冠軍伊萬-波捷辛向普京鄭重提出請求，希望能夠降低國際奧林匹克大學的入學門檻，讓更多有意者前來深造，重啟自己的職業生涯。

「說實話，這個我都沒有考慮到，」普京很誠懇地感謝波捷辛的提醒，並承諾一定照辦。
