5人死刑！沙特記者碎屍案疑掩蓋真相，卡舒吉之子卻說裁判“公正”

2019-12-24 969條評論 2,998次閱讀 1388人點贊

12月23日，沙特公布了一年多前震驚全世界的「卡舒吉案」的判決結果：有五名被告因參與謀殺被判死刑，另三名被告則因隱瞞犯罪，被判處總計24年刑期。

以上就是關於這次判決，外界所知的全部。

從這起案件的調查階段開始，沙特方面從未公開過案子的調查名單，這種保密措施也維持到了判決宣布：五名被判死刑的被告的身份迄今依然無人知曉，庭審在高度保密的情況下進行，聯合國調查員要求旁聽的申請遭沙特方面反覆駁回，少數出席了庭審的外交官則被要求發誓保密。

雖然不知道被控有罪的是什麼人，但卻有至少三個此前因涉入謀殺被解職的高官已經無罪釋放：沙特王儲穆罕默德·本·薩勒曼的高級顧問沙特·卡塔尼（Saud al-Qahtani），時任沙特駐伊斯坦布爾領事館總領事穆罕默德·奧塔比Mohammed al-Otaibi，和情報局副局長艾哈邁德·阿西里（Ahmed al-Asiri）。

「卡塔尼出席了庭審，但沒有任何證據對他不利。「負責此案的檢察官說道。阿西里和卡塔尼和王儲關係密切，通常被認為是王儲「內部圈子」的人。而卡塔尼和奧塔比早已在一年前就被美國財政部列為了制裁對象，理由是被指參與謀殺。

卡舒吉的長子，負責協調卡舒吉一家和沙特政府關係的薩拉赫也表示：「我們對沙特的司法系統抱有信心，因為他們的判決結果帶來了正義。」

今年四月，有消息指卡舒吉一家也已經和沙特政府達成協議，獲賠價值上千萬美元的現金和資產。外界質疑這是沙特政府給卡舒吉一家的「封口費」，但薩拉赫在推特上否認了這一消息。

卡舒吉的未婚妻哈蒂絲在推特上則表示，無法接受沙特政府宣布的結果，並說「永遠不會原諒那些參與謀殺的人」，以及「試圖掩蓋真相的人」。

卡舒吉和未婚妻哈蒂絲的合照 / 網路

主動調查這一轟動全球的大案，沙特的目的原本是儘可能挽回一邊倒的國際輿論，但從昨天有關案件判決的公告來看，沙特距離實現這個目標，只怕還很遠。

匿名死刑名單

「卡舒吉案」起於2018年10月2日，當天流亡海外的沙特著名記者及異見人士傑邁勒·卡舒吉為辦理相關證明進入沙特駐土耳其伊斯坦布爾的領事館，但卻在領事館裡遭到殺身之禍：等著他的是來自沙特情報部門的特工人員，他們對他進行了酷刑折磨，最後將其殺害並肢解了屍體。隨後的毀屍滅跡行動靜悄悄地發生，沙特領事館極力否認遮掩，導致十三天後他的死訊才由土耳其政府確認。直到今天，卡舒吉的遺體仍未找到——有媒體報道相信，遺體已經被沙特特工用強酸徹底腐蝕溶化。

這起案件的殘忍程度震驚了全世界，更為引人注目的是它絕非一場普通謀殺——殺人的是特工，藏屍的是領事館，這已足夠證明它是一場極高級別的政治暗殺。

外界的懷疑對象從一開始就指向了沙特目前的實質當權者，卡舒吉此前幾年的言論已經數次觸怒王儲，而自2015年開始掌權以來，這位沙特王儲在消滅反對聲音方面的記錄一直不少。

沙特王儲穆罕默德·本·薩勒曼 / 網路

案發四十天後，美國CIA得出了「謀殺命令出自王儲本人」的結論，美國國會隨即要求沙特王儲為此事承擔責任。2019年6月，聯合國發布了有關該案的調查報告，正式將沙特王儲穆罕默德·本·薩勒曼列為相關「應受到調查和制裁」的人員。

但在沙特，政府不僅否認王儲在謀殺中發揮過任何作用，而且對此事進一步貫徹了掩飾、迴避、否認的處理原則，口徑多次變化。儘管在承認卡舒吉的確死於沙特領事館以後，據稱沙特逮捕了18名嫌疑人（後增加到21個），但直至目前，這份逮捕名單從未公布過。

除了駭人聽聞的殘忍情節，捲入這起案件當中的還有錯綜複雜的國際政治：卡舒吉在逃離沙特後已定居美國，又是在土耳其境內被殺，三者間隨時可能改變的微妙關係，給這起案件的追查造成了多重變數：美國總統特朗普打算維護沙特盟友的意圖明顯，美方曾承諾公布的證據細節最終不了了之，而土耳其自案發起一直將此案當作一個國際談判籌碼，在當前土耳其出兵敘利亞、與美國及中東關係都極為敏感的當口上，準備在何種程度上進行干預仍是未知。

多重因素疊加的結果，就是當前沙特這份不知道具體有誰，也不知道他們如何參與了案件的死刑判決名單——據沙特方面披露的有限消息，五名被告均已上訴，考慮到沙特迄今仍採取公開方式執行死刑，這些神秘的被告人究竟是誰，或許需要到最後一刻才能知道。

或許會適得其反

但在這份判決公布之際，沙特距離自己試圖「洗白」的目標，似乎是越來越遠了。

由於秘密庭審和秘密調查，外界難以確認的不但有案件的被告名單，也包括立案的相關證據——這讓判決看起來不像是伸張正義，倒更像是殺人滅口。

在本次判決中被確定無罪開釋的卡塔尼時任王儲的心腹顧問，此前已經被各方共同指為謀殺的直接策劃者，並因此被列入美國制裁名單。然而在沒有提供任何解釋的情況下，沙特法院宣布已經釋放卡塔尼，他甚至沒有被法庭起訴。

卡塔尼和王儲 / 網路

情況類似的還有時任情報局副局長阿西里，他們都因涉入謀殺而被解職，但最終結論都是無罪釋放。

聯合國特別調查員卡拉瑪德此前提出，這起案件並不適合由沙特司法系統進行調查，因其「難以抵禦政治干預」。

判決公布後，卡拉瑪德也在推特上寫道：「殺手被判罪名成立並處以死刑，而其主謀不僅自由來去，甚至很少被調查和審判觸及。」

聯合國一位專家稱，這場判決「完全是正義的反面」，土耳其外交部則回應，判決「達不到土耳其和國際社會對這起謀殺案的所有方面進行澄清和伸張正義的期望」。

自當權以來，沙特王儲穆罕默德·本·薩勒曼一直是個爭議人物。在國內他大力推行自由派改革，似乎想要用現代經濟與意識形態改造沙特；但與此同時，這位王儲也在國內外針對不同意見發動了規模空前的清洗運動，這摧毀了他的國際聲譽，卡舒吉案正是其中最受矚目的一樁。

有關判決的聲明公布以後，沙特媒體迅速響應，異口同聲地稱在這份判決當中「正義已經得到伸張」。

但在國際輿論看來，這只是沙特內部密不透風的政治高壓氣氛的又一次直接證明。

「卡舒吉案的判決剛剛公布，」CNN寫道，「但它所解答的問題遠不及它帶來的那麼多。」
