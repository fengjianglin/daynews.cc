醫院護士誤操作呼吸機致英國著名銀行家死亡，家屬獲賠1940萬

2019-12-17 1,878條評論 3,964次閱讀 2397人點贊

圖：恩滕曼一家

據英國《每日郵報》12月16日報道，英國一位非常著名的銀行家在一家私人醫院因護士的失誤死亡，他的遺孀和孩子因此獲得了210萬英鎊（約合人民幣1940萬元）的賠償金，他的兩個孩子亞歷山大和克洛伊現年分別為13歲和11歲。

銀行家羅伯特·恩滕曼去世時年僅57歲，他是統一信貸銀行電子商務全球主管，深受家人和倫敦金融城同事的喜愛。

13日，本案的聽證會在英國倫敦高等法院舉行，聽證會記錄了他本應在2015年5月在倫敦橋醫院進行相當簡單的心臟手術。生活在奇斯爾赫斯特的 恩滕曼生前患有二尖瓣疾病、2型糖尿病以及關節炎。

然而，事實證明他的身體情況比預期的要複雜，在心臟直視手術後，他住進了醫院的重症監護室，並用上了呼吸機。法官馬丁·斯賓塞說：「手術大約一周後，一名護士犯了一個『悲劇性的錯誤』，導致這位深受愛戴的銀行家死亡。在她關閉呼吸機中的加濕器後，他的呼吸道中積聚了分泌物，導致呼吸和心跳停止。」

2015年5月28日，因他無法從不可逆的腦損傷中恢復，他的生命維持機器被關閉。王室法律顧問亨利·威特科姆說：「恩滕曼是一個非常受人喜愛的人，他總是抽時間陪伴每個人，尤其是他的孩子。」

法官表示，對恩滕曼死亡原因的調查發現，他在醫院接受手術後的護理存在嚴重缺陷。恩滕曼家的律師控告醫院的運營商HCA國際有限公司失職，法官表示，值得稱道的是，該公司在早期就承擔了這部分責任。威特科姆在法庭上表示，經過協商，當事公司同意賠償給恩滕曼的家人210萬英鎊。

當斯賓塞法官向恩滕曼表示敬意時，他的妻子同意了和解然後哭了。法官說：「也許我對恩滕曼先生最大的恭維就是他是一個我非常想見到的人。他聽起來是一個令人欽佩的，有著文藝復興時期氣質的人，對周圍的世界有著百科全書般的了解。他對他的工作，特別是對他的家庭盡心儘力。」

「生於美國的恩滕曼在英國生活了30多年，但他在可愛的孩子們還很小的時候就去世了，孩子們肯定特別容易受到年幼喪父的影響。我真誠地希望這次和解能夠幫助這個家庭，儘可能地把這件事拋在腦後。」

在和解之後，恩滕曼的妻子表示，悲劇唯一可能產生的積極影響就是醫院從這次錯誤中吸取教訓，確保類似的事情再也不會發生。保羅·麥克尼爾是恩滕曼家族的律師代表，他表示羅伯特·恩滕曼的妻子和兩個孩子在他去世後不得不堅強的活下去，承受著原本不該承受的痛苦，他希望他們一切都好。
