新公民法案攪動下的印度：有人燒火車抗議，也有人切蛋糕慶祝

天天要聞
2019-12-17

為反對《公民身份法修正案》（以下簡稱修正案），印度阿薩姆邦連日發生大規模暴力示威遊行。據《衛報》報道，遊行已造成6人死亡。儘管如此，示威者表示仍將繼續遊行。

《公民身份法修正案》於上周四在印度議會通過後，由印度總統拉姆·納特·科溫德批准成為一項正式法律。該法律將給予因受「迫害」逃離巴基斯坦、孟加拉國、阿富汗來到印度的「非穆斯林」宗教少數群體印度公民身份。該修正案除了引發穆斯林群體「被歧視」的擔憂外，印度東北地區民眾擔心法律的實行會造成大批孟加拉國印度教徒湧入，與當地民眾爭搶資源，危及他們的語言、文化傳統，並使當地居民在政治、經濟、文化上成為少數。

「阿薩姆萬歲」

在此次遊行示威的「震中」——阿薩姆邦的最大城市古瓦哈提，局勢仍然非常緊張。據《衛報》報道，周日（15日），五千多人在上百警員的注視下，開始了新一輪的示威。他們舉著標牌、橫幅，高呼「阿薩姆萬歲」。

抗議者卡蘭·米利的同事在示威中遇難， 「阿薩姆一定會繼續抗議下去的。印度是一個民主國家，政府必須聽取我們的意見。」 卡蘭·米利說。

另一名示威者普拉蒂瑪·夏爾馬說，「我們不希望暴力發生，但抗議活動將繼續……阿薩姆人不會停止，直到政府撤銷法律。」

據《今日印度》報道，居住在英國倫敦的阿薩姆人於周六來到倫敦的印度高級專員公署外，要求印度政府撤銷《公民身份法修正案》。

卡盧納·薩加爾·達斯，一位較有聲望的阿薩姆僑民對印度報業托拉斯說，「我們希望在全球範圍內尋求聲援，並對家中的親朋好友表示支持。」

倫敦政治經濟學院的一名來自阿薩姆邦的研究生說：「所有阿薩姆人團結起來反對修正法……我們不接受威脅到我們語言和文化的法律。」

暴力示威遊行現場

印度多地發生暴力示威

在印度西孟加拉邦，抗議示威遊行已持續四天，抗議者高呼反對執政黨印人黨政府的口號，燒毀總理納倫德拉·莫迪和聯邦內政部長阿米特·沙阿的肖像，並在高速公路和鐵軌上靜坐，燒毀火車和公共汽車，導致某些地區的火車停運。

另據印度News Nation電視台16日報道，西孟加拉邦首席部長、草根國大黨主席馬瑪塔·班納吉在呼籲和平示威的同時，公開反對中央政府推行《公民身份法修正案》，稱其為「對人性的折磨」，並於周一（16日）走上西孟加拉邦首府加爾各答街頭，領導反對修正案的和平抗議遊行活動。草根國大黨領導人、西孟加拉邦教育部長帕莎·查特吉說：「我們希望立即廢除這項帶有分裂國家性質的公民法。我們的邦政府已經表示將不會在西孟加拉邦實施該修正案。我們呼籲人民以和平方式進行抗議。」

周日，在印度首都新德里國立伊斯蘭大學附近爆發了大規模暴力示威遊行，多輛巴士被焚燒，警方向示威者發射催淚瓦斯。據印度報業托拉斯報道，約35人在示威中受傷送醫，地方政府官員稱周一將關閉示威地區學校。另據《衛報》報道，警方還進入了國立伊斯蘭大學中拘捕了100多名學生。

周日的騷亂導致部分地區交通癱瘓，地鐵停運。一位市民告訴印度亞洲通訊社，市民經過國立伊斯蘭大學時非常恐慌，很多人甚至「丟下車逃命去了。」另有市民表示，很多當地人已經屯好糧食以備不時之需。

印人黨與反對派的論戰

總理莫迪將動亂歸咎於國大黨等反對派。 據印度《論壇報》15日報道，莫迪在賈爾坎德邦東部舉行的一個集會上說：「為了給予那些逃往印度並被迫以難民身份生活的人以尊嚴，兩院均通過了《公民身份法修正案》。國大黨及其盟友卻以《公民身份法修正案》為由到處煽風點火引發暴力事件。東北各邦人民不希望訴諸暴力……他們（國大黨支持者）到處縱火就是因為沒達到他們的目的……他們的行為正說明通過修正案的決定是百分之一千正確的。」

對於伊斯蘭團體、反對派、維權人士而言，修正案被視為莫迪印度民族主義議程的一部分，目的是將印度2億穆斯林邊緣化。莫迪否認了這一指控。人權組織和穆斯林政黨正向最高法院對這項法律提出質疑，認為其違反了憲法和印度的世俗化原則。

在印巴基斯坦裔印度教徒燃起希望

在由《公民身份法修正案》引起的暴力衝突和激烈論戰背景下，600多名住在印度旁遮普邦賈朗達爾、來自巴基斯坦的印度教徒燃起了早日獲得印度公民身份的希望。

據印度《論壇報》報道，這些來自巴基斯坦的印度教徒移民聲稱，他們是在20世紀90年代用訪客簽證進入印度的，並嘗試多次延長簽證以求留在印度。雖然身份仍為巴基斯坦籍，拿的是巴基斯坦護照，但他們現在與巴基斯坦幾乎沒有聯繫，並認為自己已經是印度人。最近，這些巴基斯坦裔印度教徒通過聚在一起切蛋糕的方式來慶祝《公民身份法修正案》的通過。

法拉克·拉格·巴加特如今44歲，在他22歲的時候，與其他50名親友一起離開巴基斯坦進入印度。在做了幾年零工後，他終於覺得自己定居在這裡了。如今，他製造手術設備並出售給當地醫院。

法拉克說，「印巴分治的時候，我們社區的一些成員成功逃出巴基斯坦並進入印度，但更多人因為某些原因不得不留在巴基斯坦。其他宗教群體要求我們改宗，我們一直活在壓力和恐懼之下。直到1998年，我們終於得以登上開往印度的火車並在印度的土地上安家。」他補充道，「但是在印度的生活也非常不容易，因為我們拿的是訪客簽證。甚至我們結婚生子了，也需要每年去更新簽證。每次要花個2000-3000盧比（約197-296元人民幣）」。
