真·冰火兩重天！南北半球兩城市溫差超90℃

天天要聞
2019-12-22

來源:央視網

當地時間19日，俄羅斯遠東薩哈雅庫特共和國首府雅庫茨克遭遇嚴寒天氣襲擊，市內最低溫度達到零下46攝氏度。

而在澳大利亞，這一周被熱浪席捲，多地山火肆虐。澳大利亞更是連續兩天創下高溫紀錄。

俄羅斯雅庫茨克市

遭遇-46℃嚴寒天氣

當天，俄羅斯雅庫茨克市政府宣布小學一至五年級停課，這也是該地本年度第一次因嚴寒天氣停課。根據當地法律規定，最低溫度達到零下45攝氏度以下時小學停課，達到零下50攝氏度時全部學校將停課。

由於惡劣天氣，雅庫茨克機場多個進出港航班也受到影響被迫取消。

俄羅斯薩哈雅庫特共和國有40%的領土位於北極圈內，是北半球冬季最長和溫度最低的地區，最低溫度可達零下70℃。

澳大利亞氣溫高達45.3℃

高溫助推山火肆虐

在南半球的澳大利亞，高溫天氣仍在持續。受高溫和野火的雙重「烘烤」，考拉們忍不住乾渴，大著膽子向路人討水喝，有的甚至「組團」上門，令人心疼。

對此，澳大利亞一家動物保護機構指出，極端高溫可能對野生動物造成「毀滅性後果」，他們還建議人們在陰涼的地方把新鮮、乾淨的水放在淺碗里。

在南澳大利亞州首府阿德萊德市，19日的氣溫達到了45.3攝氏度。當地一名女子拍下了自己的丈夫用水為一隻貓頭鷹降溫的畫面，這名女子說，這隻貓頭鷹似乎是從樹上掉下來的，當時正大口喘著氣，看到周圍的水迅速喝了起來。

與此同時，阿德萊德市北部的兩處林火則在高溫的助推下繼續肆虐，依然沒有得到控制，給附近住宅造成威脅。

當地消防部門出動了36輛消防車，以及8架飛機參與滅火。在新南威爾士州，目前仍有100多處林火正在燃燒。19日，兩名志願者在該州首府悉尼附近參與滅火時不幸遇難。

面對國內不斷高漲的輿論壓力，澳大利亞總理莫里森不得不宣布縮短自己在夏威夷的假期，莫里森在聲明中稱，對自己此時出國度假的行為表示後悔，並表示將儘快趕回澳大利亞。
