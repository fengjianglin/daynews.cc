印度暴徒放火焚燒警車，警察開槍打死4人，一夜逮捕數十名大學生

2019-12-19 239條評論 4,728次閱讀 2034人點贊

9日，印度下議院以311票贊成、80票反對通過了《公民身份法修正案》（以下簡稱修正案）；15日，莫迪正式宣布修正案通過。其規定，在2014年以前，來自巴基斯坦、孟加拉國、阿富汗的民眾將獲得申請成為印度公民的資格。

但是，讓印度民眾難以接受的是，這條法案規定，信奉印度教、佛教、拜火教、錫克教，基督教的人都可以獲得合法身份，但是信奉穆斯林的教徒卻不能享受此法案。這樣的宗教歧視，使得印度民間非常不滿，很快數十萬人上街進行遊行，抗議活動席捲了多個城市。

在首都新德里，抗議者放火焚燒了四輛公交車和兩輛警車，甚至還包括火車，並堵塞道路阻礙交通。眼看著示威活動範圍越來越大，莫迪果斷採取了暴力鎮壓的手段，據悉印度的警察已經全部調動，並且為了增加鎮壓的效率，印度警察都配備實彈。

而且，讓外界更加嘩然的是，莫迪為了對付遊行民眾，居然調集了軍隊，關閉了幾個重大暴亂城市的無線網路。對此，白宮強烈批評莫迪的這一行為不符合「人權」規定，並警告稱將制裁引入此項修正案的印內政部長沙米特·沙阿，及部分其他印度官員。

直至今天，印度國內的暴亂依舊沒有平息，甚至因為政府的強硬手段，暴亂者變得更加瘋狂，到目前為止，已經有6人死亡，其中有4人是被警察開槍擊斃。隨著人員的傷亡情況不斷擴大，牽扯進來的暴亂者也越來越多，據悉印度國內的大學也出現了不少暴亂者。

伊斯蘭教大學幾乎所有的學生都參與到了抗議活動當中，在學校內部的組織下，這些學生用石頭毀壞民房和公物，還與警察發生衝突，造成60多人受傷，一夜之間，警察逮捕了數十名大學生，不過為了避免事情鬧大，第二天凌晨這些學生都被釋放。

據國外媒體報道，英美等西方國家已經向民眾發出了旅行警告，禁止民眾前往印度。由此來看，印度此次暴亂，對於國內的經濟將會是一場巨大的災難。據悉，印度在2019年第三季度的增長率只有4.5%，已經創下了15年新低。

如果印度不能迅速平定暴亂，那麼接下來的經濟恐怕還會繼續突破新低。可笑的是，印度此前還放言要在5年內成長為世界第三大經濟體，結果經濟沒提上去，反而深陷暴亂當中。

面對國內的亂局，印度政府本應該採用協商等和平手段，來穩定民眾情緒。但是，莫迪卻一味地依靠暴力鎮壓，意圖迅速平定亂局，民眾自然反抗更加強烈，可以預見，印度的暴力衝突，還會持續很長一段時間。
