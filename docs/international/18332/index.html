緬甸億萬老夫妻變賣所有財產獲近1億，現在住爛窩棚！發生什麼了

2019-12-16 1,144條評論 4,573次閱讀 1783人點贊

【緬甸中文網訊】緬甸是一個佛教國家，有85%以上的民眾信仰佛教。而緬甸國內更是佛塔林立，可以說佛塔是緬甸最多的宗教建築，特別是緬甸歷史文化之城——蒲甘。蒲甘是緬甸佛塔最多的城市，現存佛塔有幾千座，被譽為「佛塔之都」，也正因歷史悠久，佛塔林立，蒲甘古城被列為世界遺產。

為什麼蒲甘乃至緬甸會有那麼多佛塔呢？可以說，民眾有很大的功勞。很多佛塔的建立都離不開民眾的參與，他們會捐錢修建佛塔、佛寺、講經堂等，還會捐贈佛塔寶傘、捐贈供奉寶物，捐贈佛像……

緬甸民眾對佛教到底有多虔誠？可以這樣說，有可能你在大街上看到一位民眾非常普通，衣著看上去十分寒酸，住所也破爛不堪。但是，他可能已經捐了好幾座佛像或佛塔了！這在緬甸，的確是可能發生的事，小編了解到的一對老夫妻，就做了類似的事情。

這對老夫妻為曼德勒省敏建鎮人士，老爺爺名為吳耶，老奶奶名為杜丁褪。這對老夫妻已經年過花甲，正是享受天倫之樂的時候，做出令人意想不到的事情，他們竟然將辛苦一生「創造」的財富變賣。

這對老夫妻變賣財產的目的是什麼呢？目的只有一個——捐錢修建講經堂。這對老夫妻變賣了他們的房產和地產，獲得了9500多萬緬幣，然後將這筆錢全部捐出，只為修建一座講經堂。

你可能有疑問：變賣了所有房產，老爺爺老奶奶以後住哪裡呢？原來，變賣了所有房產和地產後，他們使用棕櫚的樹枝搭建窩棚居住，雖然日子很清貧，但兩人卻過得十分幸福。

對於老爺爺老奶奶的做法，有人為他們的行為點贊，覺得他們十分虔誠。但也有民眾稱老爺爺老奶奶的做法雖然令人佩服，但是他們應該捐錢蓋學校或者修路，修建講經堂不太值得。您怎麼看？
