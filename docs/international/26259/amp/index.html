韓國將在2750年從地球上消失？可能等不到那時了

天天要聞
2019-12-20

「如果出生率沒有變化，韓國將成為2750年，地球上第一個消失的國家。」

這是2014年，韓國出生率1.25，位居OECD國家倒數第一時，韓國保健福祉委員會得出的研究結果。

而人們恐怕不會想到，與此後的5年相比，2014年的出生率已是巔峰。

根據韓國統計廳的數據，2018年韓國出生率跌破1。即平均1名女性，一生生育不到1名子女。2019年第三季度，這一數值更是跌至0.88，再創新低。

一路惡化的生育率和老齡化問題，使得這個「亡國滅種」的期限，可能提前。

出生率逐年降低

生育率降低的原因

大多數情況下，韓國把不結婚而生孩子的女性稱為「未婚媽媽」，給予不友善的目光。韓國的婚外生子率低於2%，在27個OECD成員國中最低。

也就是說，「結了婚才能生孩子」一直是韓國社會的共識。因此，結婚率和生育率息息相關。

1980年韓國結婚件數約為40萬，此後逐年遞減。而到了2018年，結婚人口急劇減少，結婚件數只有25萬左右。2019年的數據雖然尚未統計，但是情況依然不容樂觀。

韓國新婚夫妻

韓國年輕人越來越恐懼結婚生子，原因有以下幾點：

① 自我中心的YOLO文化

YOLO是「You Only Live Once」，顧名思義就是「人生只有一次，要為自己享受生活」。

31歲的李賢載，大學畢業後從家鄉大田來到首爾工作和生活。他每天最開心的時刻，就是下了班和女朋友相約看場電影，或者去大學附近的酒吧街喝上一杯。「有了孩子，就要與這種開心的生活告別，難以想像」。

人們將更多的注意力投入到工作、學習、和豐富的業餘生活中，不願為了結婚生子而犧牲自己的人生。因此，韓國「一人家庭」（即獨自居住的「家庭」構成）數量也在逐年增長。

崇尚YOLO的年輕人

② 女性的困境

與YOLO族不同的是，一部分韓國女性希望結婚生子，但擺在她們面前的現實卻是殘酷的。

《82年生金智英》在韓國大火，甚至被一些中學列入必讀書目。電影充分反映了韓國女性，在結婚生子後所面臨的窘境。女性必須放棄自己的工作和生活，才能迎接一個新生命的到來。而這一切的結果，只是變成「媽蟲」（韓國對「花老公錢」的年輕媽媽的歧視詞）而已。這樣的生活，很難和「幸福」聯繫起來。

「職場經歷中斷女性」是近年來流行的詞語，指因育兒而被迫中斷職場生涯的女性。

與中國略有不同的是，韓國隔輩帶娃現象並不常見，很多時候，仍然需要媽媽辭職照顧寶寶。職場與家庭不可兼得的窘境，成了韓國女性不願結婚生子的原因之一。

《82年生金智英》

③ 高昂的結婚費用

高額的結婚費用也成了年輕人走向婚姻的一座大山。

✔ 平均結婚費用：

以最近2年內結婚的1000名新婚夫婦為對象進行的調查結果顯示，結婚平均費用為2.31億韓元，約合138萬人民幣。其中住宅資金高達1.68億韓元，約合100萬人民幣。其他支出還包括禮金、婚禮和蜜月等等。

而這僅僅是全國的平均費用，在大城市和首都圈的結婚花費更高。

高昂的結婚費用

✔ 首爾人平均工資：

根據新韓銀行《2019普通人金融生活報告書》，首爾人均工資為358萬韓元，約合人民幣2.1萬元。也就是說，兩個首爾年輕人，需要不吃不喝工作2.74年，才能湊夠全國平均的結婚費用。

而結了婚就是苦難的終結？不，那只是焦慮的開始！

《首爾市職場人金融地圖》

✔ 高房價：

與生育率相反，首爾近兩年房價連續暴漲。2019年首爾新房均價為韓幣778萬/平方米，約合人民幣4.7萬/平方米。結婚即意味著漫長的貸款還款期的開始。

首爾房子

韓國的生育鼓勵政策

2005年，韓國政府驚覺出生率過低。於是在截至2019年的14年間，出台了各項生育鼓勵政策，投入了上百萬億韓元的預算。

鼓勵生育的標語

這一系列生育鼓勵政策包括：

  • 產檢補貼：每位懷孕女性單胎60萬韓幣（約3600人民幣），雙胎100萬韓幣（約6000人民幣）
  • 營養補充劑：為每位懷孕女性無償提供葉酸和鐵劑
  • 月子補貼：產婦生產後，按新生兒數計算，每人至少50萬韓元（約3000人民幣）
  • 一次性生產獎勵：根據居住地不同，每個新生兒10萬至500萬韓幣（約600至3萬人民幣）不等，子女數越多獎勵越高
  • 每月育兒工資：為選擇不上幼兒園而自己撫養孩子的父母提供育兒工資，根據子女年齡，每人每月10萬至20萬韓幣（約600至1200人民幣）不等
  • 每月寶寶工資：為7歲以下寶寶每人每月發放工資10萬韓幣（約600人民幣）
  • 尿不濕和奶粉費用：為特殊情況家庭（國際結婚、單親、殘疾、疾病、低收入等等）免費提供2年
  • 學費全免：公立幼兒園至高中學費全免，甚至提供免費午餐
  • 保姆服務：孩子12歲之前，根據家庭收入提供補貼，政府最多可承擔85%的保姆費用
  • 育兒休職：為8歲或小學2年級以下子女的父母提供，為期一年，工資按照休職前50%發放，最低限額70萬韓幣（約4200人民幣），最高限額150萬韓幣（約9000人民幣）
  • 其他各種生育宣傳和鼓勵政策

生育鼓勵政策手冊

然而效果卻並不明顯，新生兒出生率仍在持續下降。

金美妍因為首爾的高房價，被迫搬到首爾近郊，每天坐1個半小時上下班。「朋友中沒有因為生孩子而提高生活質量的」，「生育鼓勵政策只是杯水車薪，孩子的一生會面臨更多壓力，我都結不起婚了，卻要擔心如何負擔孩子的結婚費用？」

韓國社會的高度競爭，促使課外補習盛行，每月花費在孩子補習班上的費用高達幾千甚至上萬人民幣。與孩子幼年的一點點補助相比，高昂的補習費用，以及今後留學、結婚的花費都讓人難以承受。

韓國有名的補習聖地「大峙洞」燈火通明

2017年9月，IMF總裁克里斯蒂娜拉加德訪問韓國，他警告說：「嚴重的低出生率問題，將導致韓國成為『集體自殺社會』。」

韓國的低生育率問題，在其他世界發達城市均有出現。如果不能正視人口負增長和老齡化問題，地球人口都有可能面臨「消失」的危險。

也許，解決「性別不平等」問題，同時讓年輕人感到「結婚生子也能過得幸福」的社會，才是解決低生育率問題的鑰匙。
