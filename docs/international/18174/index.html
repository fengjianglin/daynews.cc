印度公民法案導致多地騷亂，女學生為保護同學，手無寸鐵抵抗警察

2019-12-17 184條評論 1,368次閱讀 209人點贊

據英國BBC新聞12月17日報道，《公民身份修正案》在印度多地引發的示威活動仍在繼續。印度一些女大學生為保護朋友，手無寸鐵抵抗警察。

這場騷亂的起因是印度議會此前通過的《公民身份修正案》。該修正案提議印度政府授予2014年12月31日前因宗教原因而來到印度的巴基斯坦、孟加拉國和阿富汗三國非法移民印度國民身份。

這其中包括印度教、錫克教、佛教、耆那教、拜火教和天主教徒，但唯獨把穆斯林排除在外。

在有爭議的《公民身份修正案》通過後，隨即在印度國內外引起了強烈的不滿和抗議。

其中印度東北地區特別是阿薩姆邦甚至發生了大規模騷亂，且出現了人員傷亡。截止16日凌晨，已造成6人死亡，至少50人受傷。

拉迪達和她的朋友們在12月15日德里的賈米亞·米利亞大學參加了抗議活動。

12月16日，她的臉被割傷後，她仍然繼續參與抗議示威活動。她告訴記者：「新出台的法案，不會僅僅發生在我和我身邊的朋友身上，它會發生在這個國家每一個穆斯林身上。」

「請每個人意識到這一點，站出來吧，團結起來，與政府抗爭，為了自己，也為了這個國家。」

她最後說：「如果我們每一個人都選擇沉默，那麼誰會站出來替我們發聲呢？」

據了解，在法案通過後，聯合國人權辦公室就表達了對該法案的擔憂，稱該法案具有歧視性。

但是印度教民族主義的印度人民黨政府說，沒有宗教偏見，此舉是為了照顧那些因宗教問題逃離的人。

它說，穆斯林不受新法律的保護，因為他們在巴基斯坦、孟加拉國和阿富汗不是宗教少數派，因此不需要印度的保護。但是對於拉迪達和她的朋友們來說，該法案的最終目的似乎顯而易見:「每個人都知道該法案是針對穆斯林的。」這就是她們決定加入抗議示威活動的原因。

但是隨著示威活動的發展，矛盾不斷升級，警方開始了鎮壓行動，突襲校園，在大學圖書館內發射催淚瓦斯。

最後，當警察毆打她朋友的時候，拉迪達站了出來，用雙手推開警察，手無寸鐵的她靠著勇敢，抵抗著武裝齊全的警察。但是，12月16日警方發言否認使用了過度的武力。德里警方發言人蘭哈瓦告訴記者，警方已「採取了最大限度的剋制」。

他的言論並沒有平息全國學生的憤怒，這些學生團結在賈米亞·米利亞大學的學生周圍。

「我們很高興看到這種事情在印度各地發生。」拉迪達告訴記者。

「他們在抵製法案，他們在大聲訴求。」

但這並不是拉迪達的朋友阿耶莎·雷納希望人們從這次事件中得到的唯一東西，她希望其他女性可以加入她們的抗議活動。

「走出去，讓政府聽到女性的聲音。男人們總是讓你坐下，不要出聲。這不是我們想要的結果，我們希望女性都可以站起來，大聲的表達觀點，因為這是我們的權利。」
