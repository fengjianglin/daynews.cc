俄網紅橘貓的貓生巔峰：將和普京共同見證克里米亞大橋鐵路通車

2019-12-24 717條評論 2,454次閱讀 422人點贊

據俄羅斯新聞通訊社12月23日報道，克里米亞大橋新聞中心發布消息，普京將於23日參加克里米亞鐵路運輸通車儀式，與他一起見證開通儀式的還有一位特殊的「監工」：一隻橘貓，不僅如此，它還將在開通儀式後的一段時間內繼續監督該大橋列車運行情況。

克里米亞大橋

據了解這隻橘貓自克里米亞大橋建造之日起就一直在建築現場。當建築工人第一次到達刻赤海峽岸邊時，見到了這隻「浪跡天涯」的小貓，於是工人們就把它放在自己的房間里養著了。

網紅橘貓

如今大橋建成了，這隻貓也長大了，成為了建築工地的「檢查員」。它和工人們在工地一起生活了4年，一同見證了大橋打樁、建立支撐點以及主體結構的施工整個過程。如今它不僅開通了自己的社交賬號，還成了一名貓界「網紅」，每天發文數篇，吸引了來自於世界上50多個國家的粉絲。

橘貓的社交賬號

在克里米亞大橋的施工現場，這隻走上巔峰的橘貓還「創建」了自己的「監察部門」：一支由海鷗、海豚、蜥蜴、吉普賽犬和狐狸組成的「施工監督隊」。

日前克里米亞新聞中心項目負責人凱拉塔稱，橋樑的建設已完成，目前已經收到了關於該建設項目與技術法規和設計文件相符的文件，已經取得調試許可，大橋鐵路部分已準備好接收首列火車。

凱拉塔

凱拉塔說：「正如我們所承諾的那樣，我們將在預定的時間2019年12月底開通鐵路運輸。我們相信，這是對克里米亞半島和俄羅斯人民特殊的新年禮物。」

克里米亞大橋全長19公里，從克拉斯諾達爾地區通往克里米亞，是俄羅斯最長的公路鐵路兩用大橋。公路運輸於去年5月16日開始通車，俄羅斯總統普京出席了開通儀式。
