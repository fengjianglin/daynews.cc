北美第三大華人街區，但卻並不繁華，而且有些冷清的“中國城”

2019-12-11 1,789條評論 3,115次閱讀 2156人點贊

溫哥華市中心其實平淡如斯，街道沒有北京的寬，人流要比上海少，建築甚至沒有中國的二線城市新，對於一個城市，當古典和繁華都占不上的時候，它基本上是就城市群里的絲。絲的狀態有一種別樣的真實，在這個喧囂的年代裡，它不張揚不虛偽不做作，平常紮實地存在。

溫哥華城市裡的建築首先表現出來這姿態，除了主商業區那些商業用房，它們多數注重結構和外形的美感設計，習慣於在建築材料表面塗上各色的塗料來做整體處理，而不過多使用精細和奢侈的裝飾。所以，那些建築多是一幅穿休閑衣匡威鞋的氣質。域外的所有中國城都會讓中國人在意。

溫哥華的中國城是北美除舊金山唐人街外最大的華人街市。這裡是中國紅的主色調，紅色的牌樓、紅色的建築外牆、紅色的商鋪標牌以至紅色的路燈。街道並沒有想像中的繁華，而且幾乎是有一絲冷清的，這與近年來旁邊有一條毒品街有關係吧。

在溫哥華的當地華人說，在中國城可以買到屬於中國的一切。像是國內街邊攤的小雜貨店裡，「王致和」與「老乾媽」、花椒和桂皮、陳醋牽手醬油親切現身，讓被番茄醬和乳酪澆灌了數日味蕾接近崩潰的你，幾乎淚流滿面。

在溫哥華，文藝男女們極易愛上蓋士鎮(Gastown)。這裡是溫哥華最古老的街區，1971年時即成為文化保留區，古老建築物因而得到很好的保存。鵝卵石鋪就的街道，街兩邊維多利亞式的建築，咖啡館、古董店、精品店和餐廳都開在古建築里，很像國內的仿古一條街，只是身在其中，發現這裡一邊是更為古樸，一邊是更為時尚。著名的蒸汽鍾（Steam Clock）是蓋世鎮的地標，它也是全世界第一座利用地下蒸汽帶動時鐘發條旋轉的鐘。每15分鐘一次，蒸汽鍾噴出蒸汽，發出呼呼的汽笛聲，儘管時間不準，人們還是歡呼著，露出笑臉和老蒸汽鍾合影。

蓋士鎮的好時光是黃昏和傍晚，逛街加閑坐，用腳底感覺那些街道圓石的歷史，買一些可有可無的紀念品，在街邊喝一杯價格不低味道一般的咖啡，即使這樣依然會感覺這一刻不錯。而最美的時刻，是開了Google Maps在蓋士鎮周邊沒有目的只保留方向感地慢走。主街後面的街道安靜下來，人很少，那些沒有經過整修的老建築，粗糙、破舊卻更有過往的自然。許多植物在許多地方儘可能地出現，窗檯、門口，從花盆到修剪，看得到一致的用心。

不管多遠都是生活，生活的美好其實總是隱藏在細節中的小物品之上。在一個帶院子的大門口，看到不時有人進出，好奇地觀察。一位正出門的女士微笑推開大門，告訴我們這裡是教堂，可以進去參觀。這是沒有彩繪玻璃、沒有穹頂繪畫的教堂，它的椅子甚至有摺疊的，可是一旦進入，那宗教的氣息即時讓你安靜而舉止輕柔。

最後走累了，在開在半地下的小超市裡買了一瓶不能準確翻譯的飲料，趴在街邊的扶欄上一口一口地喝著，抬頭碰到街對面窗口裡婦人的目光，她淺淺地微笑，然後離開。世界如果是幸福並且大同著，也就是那一刻的樣子。
