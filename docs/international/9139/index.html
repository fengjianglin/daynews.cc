美國新澤西華人2019參選總結研討會：團結是關鍵

2019-12-06 624條評論 1,714次閱讀 329人點贊

東布朗士維克學委洪利武。(美國《世界日報》記者 謝哲澍╱攝)

中國僑網12月6日電 據美國《世界日報》報道，美國新澤西華人聯合總會3日舉行華人選舉總結研討會，參與今年選舉的華人及團隊、小區成員出席，探討和分享此次選舉的經驗。總會主席林潔輝指出，選舉有成敗，但參與、討論政治總是華人小區最寶貴的經歷。

以最高票連任安歌塢崖(Englewood Cliffs)市議員的胡世遠(Bill Woo)分享其參選策略，指出注重解決居民的現實問題是此次選舉大勝的原因之一，「安歌塢崖面臨開發商的大舉進入，我們從一開始就明確反對，但參選對手卻遲遲不敢表態，導致最終輸掉民心。」

胡世遠還分享團隊競選策略，包括注重意見領袖、團結志願者的力量，以及在各族裔、不同團體都有負責人，每周報告不同小區的意見和建議，「我在勝選後致電同為華裔的對手，稱讚他的努力。我希望華人能夠不分黨派，支持願意出來服務的華人。」

彭曉寒也以最高票連任愛迪生學區委員，「每次在遇到困難的時候，總會和主席都會第一時間伸出援手，這是後盾的力量，也是我這次再次參選的動力。」她同時強調使用主流社交網站、加強網路宣傳的重要性，並呼籲華人小區應該統一發聲，杜絕分裂現象。

洪利武同樣以最高票連任東布朗士維克學區委員，他感謝志願者以及福建同鄉會的支持，並分享自己拜票時的小故事：「我帶著兒子插競選牌子的時候，他突然說，爸爸我長大了要選美國總統。我說好啊，到時候爸爸陪你去插好多好多的競選牌。這段對話也讓我明白，孩子是在看著我們的一舉一動，他們從小就會把參政的種子埋進心底。」

劉鄭瓊英以三票的細微差距，領跑含德市(Holmdel)委員的選戰，但對手已提出重新計票，並將在5日進行。前密爾本(Millburn)學區委員會副主席劉建翔、華夏總校公關部長葉晶分享含德市選情，並呼籲華人團結一致支持劉鄭瓊英。

西溫莎─平原學區(WW-P)委員仲宇(Yu Taylor Zhong)則在今年的選戰中遺憾落敗，他坦言今年自己未做動員工作，「學委在很多重大投票時，往往會遭遇民意分歧，這樣的情況讓我覺得十分棘手。考慮到華人小區也有不同意見，為了避免爭吵，我今年未發動志願者拜票。感謝總會的研討會以及大家的分享，讓我受益良多。」

愛迪生學委會主席施景渭表示，華人參選需要團結其他族裔，因為光靠華人的力量不夠，「我們兩次選舉勝利都是與印度裔合作，少數族裔聯合起來才能獲得成功。」(記者 謝哲澍)
