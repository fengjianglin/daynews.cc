日本家長不接送孩子真的僅僅是治安好嗎？是另有原因

2019-12-07 763條評論 2,070次閱讀 688人點贊

國內的小學校門口一到上學放學的時間就熱鬧嘈雜，擠滿了接送孩子的人和車。

可是日本的小學校則看不到這種現象，為什麼日本人不接送孩子呢？日本的小學生怎麼上下學呢？

除去日本治安良好，人和車各行其道等大環境之外，還有一個重要原因，那就是在日本養育孩子是社會性的，而不是單純的家庭養育型。

日本的公立中小學校和國內一樣，是按學區劃分的，沒有特殊情況不允許跨學區就讀，當然也有私立小學可以擇校就讀，但絕大多數的日本人還是按部就班讓孩子在所屬學區內，讀小學和初中，高中則完全是按照實力去考的。

公立市立的小學初中和高中的教師，不會一直固定在一個學校教學，這也是為了公平起見，大約每4年教師就會人事變動（異動），所以不會存在師資力量偏倚的現象。

各個公立市立的中小學校都和所在地域緊密聯絡。中學生可以自己上下學，但是小學生必須加入小區的兒童會，集體上下學。

兒童會由兒童家長負責，一名會長，一名副會長，一名書記，所有參加兒童會的家長，在每個新學年開始前要聚在一起開個碰頭會，定下來一年的活動安排，比如說迎新會（目的是讓新入學的孩子和家長認識老會員），夏季活動（有一起看電影也有野遊等），目的是增強團結性。然後制定早上的「當番」—就是送孩子上學的值日表。每天早上由家長輪流去送孩子，幫助他們過馬路，將他們送到小學校門口。

學校里也有一名老師對應負責1~2個兒童會，在放學時將他們送至安全的小區附近。孩子們在排隊上學時，是六年級的孩子牽著一年級孩子的手，六年級男孩子領頭，六年級女孩壓後，中間是三四年級。新學年開始的第一個月，每天早上由六年級孩子到一年級孩子家門口接上一起去集合地點排隊。

兒童會內部如果發生糾紛，比如說孩子打架，以大欺小等，則由兒童會擔當老師，所屬班級的班主任，兒童會會長，家長四方協商解決，解決不了的則由校長。市教育委員會和警察介入，當然，一般的小打小鬧很快就會平息。

小學校放學時，各個班的孩子會迅速跑到操場上（雨天在體育館），找到自己所屬兒童會的位置，兒童會的擔當老師在所定位置等著，排好隊做放學集訓，主要是說一下當天學校里有無大事發生，地區內有無危險分子出現（針對兒童的犯罪行為等，警察會迅速將這些情況反饋給各學校）。

然後路遠的兒童會，在擔當老師率領下，最先出校門，別的兒童會陸續回家，各個學校，每年都會募集義工，守護孩子上下學，有退休的老人，有主婦，一到上學放學的時間，義工們穿著鮮艷的黃色橙色或綠色的衣服，帶著同色帽子護送孩子。在危險的交差點，市教育委員會還會專門派一個人守在路口，指揮孩子過馬路。

所以這種地域緊密性地養育孩子，護送孩子的方式讓日本人不用每個家庭每天接送孩子上學放學。
