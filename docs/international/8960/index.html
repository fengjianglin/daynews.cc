女王遲遲不願意給查爾斯王子讓位，到底她在糾結什麼？

2019-12-07 191條評論 1,516次閱讀 575人點贊

英國女王伊麗莎白二世最近頻繁地得到她兒子查爾斯王子的幫助，這位威爾士王子在安德魯王子接受BBC晚間新聞採訪的餘波中挺身而出，維護了皇室成員尊嚴。

然而，有消息透露，女王目前並沒有打算讓她兒子有擔當攝政王的跡象。她到底在糾結什麼？

據稱，在今年圍繞安德魯王子的爭議之後，查爾斯王子成為了「影子國王」。據稱，查爾斯王子在對弟弟放棄王室職責中起到了關鍵作用，他還在10月份的議會開幕儀式上代替父親露面。

最近有越來越多的猜測，宣稱女王計劃利用她95歲的生日永久退居幕後，讓查爾斯王子成為攝政王。

一旦作為攝政王，查爾斯就將擁有他母親的權力，只是不會擁有國王的頭銜，也即充當女王代理人的角色。

2017年在接受《每日郵報》採訪時，王室評論員羅伯特·約伯森表示，女王伊麗莎白二世正在考慮讓王位通過「無縫過渡對接」的方式得以完成。

他說:「據我所知，女王對這件事已經考慮了很久了，如果她95歲還活著，她會認真考慮把王位傳給查爾斯的。」

女王的糾結: 為什麼女王不願為查爾斯王子退位？

查爾斯最近在皇室工作中承擔了更多的職責

女王陛下的確考慮過她的年齡，希望確保當時機成熟時，王冠的交接就應該是無縫的。但現實的情況是，女王並沒有宣布讓位，說明在她內心深處，她還在糾結，如何才能做到皇權的無縫對接，顯然她認為現在時機還不成熟。

菲利普親王96歲退休了，人們希望君主也能如此，但這也只是人們的一種企盼罷了。

女王的糾結在於她的年齡，但更糾結的是，她還在擔心查爾斯的處事能力，儘管在安德魯王子事件中表現出來堅定的做派，但這還只是頭一次。

只是，最近據來自與王室關係密切人士的消息來源傳言表明，女王不太在近期有可能退位。

一位消息人士告訴《名利場》:「到目前為止，從女王的嘴裡尚未透露出半點有任何移交權力或退休的跡象。」

然而，女王並沒有答應給查爾斯一個攝政王的頭銜

威爾士親王的一位發言人補充說:「目前並沒有女王在95歲或其他年齡需要交接職責的計劃。」

在這一消息傳出之際，查爾斯王子因對安德魯王子捲入傑弗里•愛潑斯坦(Jeffrey Epstein)醜聞的處理事件中而被譽為了「影子國王」。

正是有了上述場景的發生，這才引發了人們的猜測，是否會有女王讓位之說。

在安德魯的「車禍」採訪後，查爾斯幫助指導了王室職責工作

在接受《太陽報》採訪時，有消息稱，在安德魯王子接受《新聞之夜》採訪後，查爾斯王子採取了「果斷」的行動。

據說，安德魯王子經與公司高級成員「進行過討論」，以及查爾斯本人的一句話之後，已經從他的皇室工作崗位上退了下來。

據報道，他告訴女王:「安德魯必須離開……這是唯一的辦法。」

據稱，去年11月查爾斯王子結束皇室之旅回國後，就直接與父親菲利普親王討論了王室如何應對這些指控。

雖然對安德魯的指控被證明對皇室有潛在的危害，但據另一消息來源透露，其實皇室的結構並沒有要即將發生變化的跡象或可能，這是女王洞察過形勢後的自我判斷。

他們在接受《每日郵報》採訪時透露，女王仍然在「堅定地驅使著」這個家庭在前行。

他們說:「似乎有一種推斷，因受到安德魯王子事件的影響，好像皇室要有巨大情況發生似的，但它並沒有。

「王室是由女王主導的，恐怕以後依然還會是這樣。」
