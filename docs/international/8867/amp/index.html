迪拜又來放大招，竟要建漂浮大樓

天天要聞
2019-12-08

說起迪拜只能用土豪二字來形容，有錢就是任性，蓋樓都要不走尋常路。

他們有世界上最高的摩天大樓哈利法塔。

去年剛剛完工的大相框叫做迪拜框架（Dubai Frame）。

還有一直還沒有動工的旋轉摩天大樓「達.芬奇塔」。

以及即將要建的水上城市Floating Venice，完全仿照義大利的著名城市威尼斯建造，甚至連聖馬可廣場都有。

總之，都是些奇葩的建築。

如今，地上和水裡都容不下啦，他們竟然要讓大樓漂浮在空中。

它叫做Rising Oases，翻譯過來的大概意思就是「升起的綠洲」。

這是由迪拜美國大學研究創新設計中心的Georges Kachaamy提出的概念，最初他還做了個原型，但不過很小，只有大約10厘米。

而在上個月的迪拜設計周上，他又展示了最新的更大模型，大約有兩米。

採用類似磁懸浮列車的技術，使用3D列印的超輕材料製造而成。

那這樣的大樓有什麼好處呢？

因為浮動建築不佔陸地，所以下面就可以隨便種植樹木做綠化，從而改善環境。此外，它也能讓人們免受地震，颶風和洪水等自然災害的侵擾。

大樓不受地域的限制，在沙漠上也能建造。

當然，在城市裡更沒有問題，還能解決城市交通堵塞的問題。

雖然看起來好像有些荒唐，可是磁懸浮技術現在已經並不是什麼紙上談兵，除了磁懸浮列車之外，磁懸浮電梯也在研發中了。

只不過專家表示，目前這個漂浮建築原型，還不能懸停在空中太高，要突破技術和材料的瓶頸，才能建造全尺寸的漂浮大樓。

迪拜又來放大招，竟要建漂浮大樓！也許未來我們真的會住在空中。
