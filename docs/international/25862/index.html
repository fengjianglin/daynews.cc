罵王子懟女王、100歲還在飆車，不愧王室最倔老頭…

2019-12-22 158條評論 3,341次閱讀 2212人點贊

再有四天，就是西方最重要的節日——聖誕節了。

這個日子是家人團聚的時刻，就跟我們過春節一樣。不過對於英國女王來說，今年的聖誕節，一家人沒一個讓她省心的……

先是孫子孫媳婦表示要去美國過節，現在又出了新狀況：她老公菲利普親王住進了醫院。

今天，女王一個人坐上了前往英國諾福克郡的桑德林漢姆莊園的列車。

菲利普親王於2017年退休後，大多數時間都待在這個莊園休養。

這兩天因為身體抱恙，他從桑德林漢姆府回到倫敦、住進了愛德華七世國王醫院，兩人就這樣華麗麗地錯過了。

不過也不用太擔心，這次送醫並不是什麼緊急情況，而是預防措施，因為他之前就有一些身體問題。

人家都不是被救護車送到醫院，而是自己走進的醫院大門。

說起來，菲利普親王明年就要99歲了，可還是像年輕人一樣愛折騰，不少事故都是自己「作」出來的。

但最神奇的是他倒是沒啥大事；外加他超級剛的性格，「鐵公爵」（iron duke）的外號真不是白叫的……

菲利普親王有多堅強呢？

自從成為「90後」，他和很多耄耋老人一樣成了醫院常客。

菲利普親王近年來得過膀胱炎，11年聖誕節時因為胸痛住院、接受「預防性檢查」，還做了冠脈心臟支架。

17年身體出現「緊急情況」住院，同年5月份親王就宣布卸任公職了。親王：你們玩兒吧，拜拜咯！

（圖源：NBC news）

18年聖誕節，菲利普親王患上了重感冒，沒能和王室家族一起去教堂。

今年可好，英媒接到消息說，「公爵狠狠地摔了一跤，在床上躺了好幾天。他沒摔壞哪兒，但給大家都嚇得夠嗆。」

大概……聖誕節是菲利普親王的劫。

不過他作為一位近百歲的男士，真的是老頭中的戰鬥機了。

很多人年紀輕輕就聽力退化，而直到14年親王才使用助聽器，那時候他已經93歲了。

18年3月底，菲利普親王髖部不適，4月，他做了個髖關節置換手術。手術大動筋骨，對任何人來說都是很傷元氣的，更別說98歲的老人。

然鵝這位公爵老哥是真硬核，恢復不到一個月，5月份參加孫子孫媳婦——哈里和梅根的婚禮時，扶都不用扶，簡直震驚醫護人員全家。

今年8月他在巴爾莫勒爾莊園度假，遠遠看過去，哪裡像快100歲的人呢？

而且誰也想不到，今年1月，他還在桑德林漢姆莊園不小心出了場車禍。

今事故看起來非常可怕：菲利普親王的路虎側翻在路邊，對面司機都骨折了。

而菲利普親王只有一點皮肉傷……

目擊者表示，人們把親王從路虎弄出來時，他意識清醒；但是受到了驚嚇，抖個不停，也把大家嚇出一身冷汗。

後來他寫信給對方道了歉，這事兒才算完。

你以為出車禍他就歇了？錯！

僅僅兩天不到，英國媒體就發現愛丁堡公爵開著一輛嶄新的路虎，繼續高齡上路，而且沒系安全帶。

（圖源：daily mirror）

公爵這種行為被大家罵了一通：拜託上路注意安全好嗎，您老爺爺不怕撞，我們路人怕啊！

這位鋼鐵老爺子除了身體夠硬，性格也特別剛。其實那次車禍之前，就有人提醒過他一把年紀就省省吧，他非不聽……

菲利普親王對自己的「蜜汁自信」和的倔脾氣，早就不是啥秘密了。

Netflix的劇集《王冠》里是這麼演的：女王即位後，他倆之間的矛盾越來越尖銳。菲利普性格執拗，但奈何老婆是一國之君，很多事兒由不得他做決定。

按照英國憲法，不管你是女王親媽還是親夫，都要遵從王權。對於菲利普來說，莉莉貝特（伊麗莎白二世的昵稱）先是女王，後是妻子。

然而年輕氣盛的菲利普面對各種規定太憋屈了，啥都不讓做，還不如去死一死……

其中他最不爽的就是名字的問題。

根據1952年女王通過的一項法案，英國王室後人只能姓溫莎。所以當女王即位，孩子們要隨女王的姓氏「溫莎「，而菲利普姓「蒙巴頓」。

在劇中，他非常生氣地說「你奪走了我的事業，我的家……」

他從小流亡海外，在多個國家之間輾轉，菲利普親王對自己的身份認知一直十分迷茫。

和伊麗莎白二世結婚，意味著很多事情都沒有自主權，這種感覺可能和倒插門女婿差不多。只不過他要服從的不是女方家庭，是整個國家。

可能戲劇有所誇張，但公爵老哥的剛，很真實。據說，菲利普親王曾經自嘲：「在英國，只有我的兒子不能跟父親的姓。”

經過長期抗爭，女王終於宣布，自己的後代可以選擇姓蒙巴頓-溫莎。菲利普的堅持贏下一分。

為了緩解壓抑的心情，他很會找樂子，比如學飛行。

飛行危險係數很高，即使他自己的姐姐就死於空難，也不能阻止他的愛好；親王非要搞這麼一出，十頭牛都拉不住。就算首相內閣都指責他，他也不管。

在治理家庭方面，菲利普親王也是出了名的鐵手腕，特別是對大兒子查爾斯王子。

菲利普年輕時曾經親眼看著父親花天酒地最後悲慘死去、母親精神失常被關進醫院，所以他下定決心，自己的孩子絕對不能搞成這個鬼樣子。老婆忙著治理國家，他就開始執掌家庭。

然而老爸管事，對查爾斯來說就是噩夢的開始……

親王對孩子那叫一個嚴苛，導致小時候的查爾斯非常怕他，不敢和他說話，每次見到父親菲利普都是一付冷漠的面孔，查爾斯甚至懷疑自己是不是菲利普親生的？

等長大點，他就被老爸送到了位於遙遠的蘇格蘭的寄宿學校Gordonstoun，就算和奶奶撒嬌求放過也沒用。

查爾斯和父親性格太不一樣了。親王被人稱為天生leader，而小王子卻害羞內向。結果他一到那邊就開始被同學欺負、孤立。求查爾斯的心理陰影面積。

等查爾斯的兩個弟弟安德魯和愛德華長大了，也被送到那裡去上學，而他們過得也並不像爸爸想像中那麼好。

But，親王一直覺得自己的決定完全沒問題。

或許是童年受到老爸管控太嚴格，現在查爾斯對兒子們都比較寬容仁慈。

老父親和長子的矛盾沒有隨著他長大成人而結束。

查爾斯和戴妃出現婚姻問題時，菲利普對這個傻兒子感到相當失望。在其他人都沒咋吱聲的時候，他給兒媳婦寫過信：「我還是那句話。如果有需要，我一定盡己所能去幫助你和查爾斯，但是現在必須承認，我在婚姻諮詢上沒什麼天賦。

「我不支持你們中任何一個有外遇。查爾斯很蠢，身處如此位置還為卡米拉冒險。」

由於菲利普親王在成長過程中沒有一個正面的父親形象，他對自己的孩子能做到這樣很不容易了；只是對於查爾斯來說，他仍然不是一個好父親。所以這對父子到現在都關係很僵……

唯一讓老親王感到心都化了的，就是和孫子曾孫們在一起的時候，每次和威廉哈里同框，他都是笑意盈盈啊。

這位硬核親王從年輕倔到老，別人勸他什麼，他完全聽不進去。王室專家都說，他「倔得像頭牛」（tough as an ox）。

助理也吐槽：親王特別犟，想要勸他？哈哈，那是不可能的。畢竟人家是全世界待機時間最長的君主的配偶、英國最資深皇夫。

逢此佳節，親王住院，女王奶奶卻連看都沒去看老公，看來不是什麼大事兒。這個超倔老頭大概還能陪伴愛妻很多年吧……
