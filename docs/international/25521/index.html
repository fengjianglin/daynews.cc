“與中國的相遇讓我不斷成長，更有自信”——羅馬高中生鄭彥柯的2019中國故事

2019-12-23 1,119條評論 2,304次閱讀 956人點贊

羅馬國立住讀學校高中生鄭彥柯

國際在線報道（記者 宋承傑）：在義大利，有一個名叫鄭彥柯的高中生，他在鄭州舉辦的「漢語橋」世界中學生中文比賽中一路過關斬將，獲得全球總冠軍，著實火了一把。鄭彥柯說，2019年與中國的相遇讓自己不斷成長、更有自信，同時他也切身感受到了中國的開放與包容。

鄭彥柯今年17歲，是羅馬國立住讀學校高中五年級學生。2019年對於鄭彥柯而言是難忘的一年，這一年他代表義大利前往中國參加第十二屆「漢語橋」世界中學生中文比賽，經過長達兩周的多輪比拼，最終在來自105個國家的124名參賽選手中脫穎而出，獲得全球總冠軍。實現了從學習漢語以來就一直懷著的「中國夢」，「因為我相信天道酬勤，如果你付出努力就會得到想要的結果。當初覺得世界總冠軍很遠，參加比賽的很多人至少有一個比我厲害的，其實也是這樣。但是，到了自己當上歐洲冠軍之後，剩下五個人，其中的一個是我。所以一切都是有可能的。還有另外一件事情就是，我接觸到了來自全世界的年輕人，還有這些中國的志願者和學生，我還有他們的微信，都聊一樣的話題，最大的感受就是年輕人之間沒有什麼不同。」

鄭彥柯操著一口流利的中文，講起書法和太極來頭頭是道。他與中國和漢語的結緣開始於五年前，鄭彥柯就讀的羅馬國立住讀學校在義大利久負盛名，學校從2009年起開設了五年制中文國際理科高中，學生學習漢語和中國文化蔚然成風。2019年3月，在中國國家主席習近平對義大利進行國事訪問之際，學校師生給習主席寫了一封信，並收到親切回信。作為親歷者之一，鄭彥柯表示寫信的目的是感謝習近平主席給了他們那麼多機會去學習中文，了解中國，至今他仍記得回信中的內容，「我們本來沒有想到他真的會回信， 我們給他寫的那封信是感謝他給了我們那麼多機會去學習中文，去了解中國。收到回信之後，第一反應是很驚訝，大家都在談論這件事情，我到現在還記得最後兩句話：『青春正好，不負韶華』，他希望我們年輕人能夠一起付出努力，讓世界變得更好，幫助中國和義大利變得更發達。」

2019年鄭彥柯通過了漢語水平最高等級的六級考試，同時也即將面臨高中畢業。對於未來的選擇，他非常淡定從容，心裡已經有了主意。鄭彥柯表示，2019年是新中國成立70周年，2020年是義大利與中國建交50周年，雖然他只見證了最後的五年，但希望能在未來的歲月里親身參與兩國關係的發展，「我想去中國，繼續學習中文，繼續了解中國。中國大學有播音主持專業，這是義大利沒有的。還有我想學習國際關係，希望能夠幫助兩個國家之間建立更好的關係。我原來都不知道 義大利和中國有這麼長久的外交關係，像我一樣年輕人大部分都不知道，所以需要讓義大利年輕人更好地了解中國。」

五年前接觸漢語開始，鄭彥柯每年都會去中國，從參加夏令營到成為交換生，對中國的了解早已不局限於長城和熊貓，他會騎著小黃車去上書法興趣班，或是打完太極後手機下單一杯外賣奶茶。「中國總是讓人看不夠」，對於近幾年來的變化，尤其是巨大的開放和發展成就，鄭彥柯也頗有感受，「就像看一部好的電影，第一次會知道故事，大概是什麼樣子；第二次會發現更多的細節；第三次就會關注拍電影時的技巧。我覺得去中國也是一樣，每次都會發現不同的東西。我媽媽在上海工作，如果有人問她有什麼變化， 她就會說，我走在路的時候，經常會發現今天突然出現了另外一座樓，她是建築師可能比較注意這些。」

如今已小有名氣的 鄭彥柯過著忙碌而充實的生活，平日里學中文的師弟師妹總會來「取經」，年底他還受邀去上海曾經學習過的母校客串一把主持人。總結2019年，鄭彥柯用了「成長」、「自知」和「自信」三個詞來概括，他表示，所有這些收穫都是和中國密不可分的，「剛開始學中文的時候，大家都會問我為什麼學中文，好像這是一個很勇敢的選擇。現在中國越來越受到關注，也讓我更有自信繼續學習中文，現在大家問的更多是中國是什麼樣的？你打算去那裡做什麼？中國有了越來越重要的地位和影響力，我現在也像他一樣更有自信，和同齡人以及世界上其他地方年輕人比起來更知道自己要做什麼，我的未來有很多選擇，有很多的機會。」

2020年，羅馬高中生鄭彥柯或許會成為中國某地的一名大學生，也可能在世界上其他地方學習生活，但是中文已經融入了他的生活，中國文化已經成為他思想的重要組成。鄭彥柯喜歡用「女排精神」勉勵自己努力拚搏，不斷進取。問他偶像是誰？他說：「女排總教練郎平！因為我很喜歡打排球，對全世界排球運動的情況很了解。郎平也來過義大利摩德納，並且帶領摩德納隊贏得義大利冠軍和歐洲冠軍。她身上具有堅韌不拔，拼搏到底的品質，她對我的影響已經超過了體育本身。」
