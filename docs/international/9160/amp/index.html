中國學生成績問鼎全球，德國人表示：不服

天天要聞
2 週 ago

本周二，國際經濟合作與發展組織（OECD）宣布了2018年最新PISA成績。

PISA即「國際學生評估項目」，是針對15歲學生閱讀、數學、科學四項能力評價研究項目。從2000年開始，PISA每三年進行一次測評，被譽為「教育界的世界盃」競賽。

本次測試共有79個國家（地區）參加。在最新公布的結果中，我國四省市（北京、上海、江蘇、浙江）作為一個整體取得全部3項科目第一的好成績！

而一向以發達的教育體系著稱的德國，測試成績卻差強人意，甚至比往年有了明顯倒退：

上圖是按閱讀成績排序的前30名。可以看到，德國名列20位，三項成績都僅僅略高於平均值。

這一結果令德國人大跌眼鏡，媒體一片哀嚎。

《法蘭克福彙報》報道稱：「最新Pisa結果顯示，德國15歲的學生中有五分之一只能讀懂簡單的句子。」報道稱這些不愛讀書的人（Lesemuffel）水平相當於文盲（Analphabet）。

在閱讀方面，20%的15歲學生甚至沒能達到小學畢業標準；在數學方面，2018年的PISA成績要明顯低於2012年；在自然科學領域，2018年的平均成績低於2006年！

《世界報》也表示「德國學生Pisa成績太差引發擔憂」

令人意外的是，在一向以愛讀書著稱的德國，學生對閱讀的興趣也明顯下降。只有四分之一表示自己的最大愛好是讀書，二分之一稱自己只有在「必須這樣做以獲得所需信息」時才看看文字，而居然有34%的學生直言閱讀是「浪費時間」！

圖片來源：tagesschau.de

總之，這次的測評結果令德國人感到扎心，也引起了各界對於德國教育的大討論。

很多教育專家都在反省，有人認為是原因是年輕人看手機太多，閱讀太少；也有人歸咎於學生動不動就罷課；還有人認為政府只看重高等教育而忽視了基礎教育，導致中小學師資力量不足……

而也有一些德國媒體表示：我不服！

《圖片報》在昨天的報道中，對這一測評結果提出了質疑：「我們成績這麼差，是不是因為別的國家作弊了？」並且還在開頭附了一張中國學生的照片，圖注寫著「中國的Pisa成績特別好，是不是因為只有最優秀的學生參加測試了？」


報道引用杜塞爾多夫大學教授Heiner Barz的觀點說，Pisa測試結果有很大的解讀空間，因為德國嚴格遵守了測試規則，而其他國家很可能並沒有。

「也就是說，其他國家有可能在Pisa測試這一天，建議那些較差的學生請病假。」

該教授還表示，中國只有教育水平最好的省市參加了測評，因此測試結果是不公平的。

而另一位教育專家則認為這個測試意義不大：「分數上的細微差距會導致名次上下差距巨大。但是在日常生活中其實無關緊要。就像100米跑步，用時9.8秒還是9.9秒會決定獎牌歸屬，但是我能不能趕上公交車，其實也不在於這0.1秒」

對於德媒的這個態度，小編想說：你們開心就好~

但是，「讓差學生請假」這個猜測有什麼依據呢？這腦洞是不是有點不負責任了……

其實，德國在Pisa測試中慘敗也不是第一次了。2000年進行的第一次Pisa測試中，德國在32個參與國家（地區）中名列22位，當時這一事件震驚了德國，被稱為「Pisa-Schock」。此後德國提高了對基礎教育的重視，排名有所上升，但從上次測試（2015年）開始，又開始連續下滑。

德媒把這個鍋甩給了難民。

《圖片報》文章說：「為什麼其他國家比我們好那麼多？德國在79個參賽國家中下滑至第20位，比波蘭、中國、瑞典都要差。其中一個原因是：德國的非母語者比例在提升。從2015年至今，有20萬名難民在德國上學。」

中國學生的PISA成績在2009年、2012年連續兩年排名第一，2015年退步到第十，如今再次回歸第一。這份報告對中國教育界人士來說，可謂是一個鼓舞人心的報告。

當然，中國人也確實不應盲目樂觀。京滬江浙四地的確是中國教育水平的高地，這是不爭的事實。經合組織秘書長古里亞（Angel Gurria）也強調，中國東部的這四個省市”遠不能代表中國的整體情況”。

但上海市教育科學研究院PISA研究中心專家王湖濱表示，根據測試數據，即使是京滬江浙四地中最弱勢的學生，其閱讀能力都超過了測試國家/地區的平均水平。

而一向奉行快樂教育的德國，PISA 成績卻一路下滑，這的確值得思考。

在青少年教育方面，中德教育方式究竟有什麼差異，孰優孰劣，對此你有什麼看法？
