加拿大航空從最好變最爛！美聯航為倒數第一

2019-12-07 257條評論 1,181次閱讀 241人點贊

聖誕季和中國春節季馬上到來了，各位華人們如果準備定加航出國玩，一定要先看看這篇文章！

一項排名顯示：加拿大航空成為全世界最差的國際航空公司之一！

12月4日，美國信息研究公司JD Power公布兩項群眾對國際航空公司的滿意度的排名，綜合了九個不同的因素：機艙服務、成本和費用、機組人員、登機手續辦理、登機、移民、行李和預訂，加拿大航空已被列入最差的國際航空公司之一！

在2019年9月~10月期間，JD Power收集了來自全球各地的6,287名乘客的反饋進行分析，列出了兩項排名：

在飛往亞洲目的地的排行中，加拿大航空排在倒數第三，令人詫異的是…中國東方航空公司竟比加航還差，排在了倒數第二，而美聯航為倒數第一！

在飛往歐洲目的地的排行中，加拿大航空排在倒數第二，僅在挪威穿梭航空之前。

儘管飛往亞洲和歐洲的旅客對加拿大航空如此不滿意，但加拿大航空公司在今年8月，獲得了北美最佳航空公司的獎項。

當時，加航官方把這個消息發到了推特上，結果卻遭網友們瘋狂吐槽…

“是因為其它航空公司真的超級差勁，才能讓加航榮獲最佳嗎”

“假的！這怎麼可能？”

“絕對是用不正當手段操控的！加航消費者們的體驗都很差！”

近兩年，加航的負面新聞越來越多…

飛機延誤，種族歧視，安全隱患，乘客被鎖飛機上，服務態度極差…

光是今年下半年，隨便一搜，加航的負面新聞就一大片！↓
