大跌眼鏡！敘利亞高級軍官出現重大傷亡，俄制坦克也遭到“重挫”

2019-12-10 86條評論 1,420次閱讀 103人點贊

自從去年美國總統特朗普宣布要全面撤出在敘利亞的美軍之後，雖然到現在已經快一年了，在敘美軍仍然沒有要走的跡象。但這一年，美軍對敘利亞確實沒有什麼動作，而是放任俄羅斯和敘利亞聯軍猛攻敘利亞叛軍。而當前叛軍主要盤踞在伊德利卜地區。這就令人奇怪了，當世第二軍事強國俄羅斯聯合地頭蛇敘利亞政府軍竟然撼動不了小小的叛軍。

而且從前方傳出來的消息來看，敘利亞軍隊損失還真不小。僅今年4月份以來，敘利亞軍隊在前線就有1名准將，4名上校，2名少校等7名高級軍官陣亡，士兵的傷亡可能更大。除了人員之外，敘利亞重型裝備損失也比較慘重，這主要體現在俄羅斯援助敘利亞的T-62M坦克上，已經有數輛該型坦克被敘利亞叛軍擊毀。這種戰況實在令人大跌眼鏡，難道叛軍真的非常彪悍嗎？其實形成這種局面的原因是多方面的。

敘利亞叛軍之所以能夠長期盤踞在伊德利卜地區，主要有以下幾種原因：一是該地區幾乎是叛軍的最後陣地，可以說是生死存亡的地方，叛軍處於逆境之中，爆發出了強烈的抵抗意志。第二方面叛軍大量收縮在伊德利卜地區，使得該地區的抵抗力量也大大增強了。第三方面是該地的地形比較複雜，有丘陵、有山地、還有果園等樹林阻隔，使得俄敘聯軍重型裝備難以施展。第四方面是由於該地區背靠土耳其，叛軍可以得到以輕型反坦克裝備為主的外部支援，這對俄敘聯軍的重型裝備尤其是坦克形成了嚴重的威脅。

據統計，敘利亞軍隊在過去的7年內戰中，遭到叛軍毀傷的各型車輛，就已經超過了5000輛，這其中坦克就損失了2000餘輛，幾乎趕上了一場大型的戰役損失數量。當然這些坦克基本上都是俄制的坦克。在敘利亞戰場上上場的俄制坦克既有T-90，也有T-72，還有從比較老舊的T-62改進而來的T-62m坦克，而敘利亞叛軍一方則得到了陶式反坦克導彈，甚至是俄制的短號-E反坦克導彈(很可能來自於土耳其)的支援，這裡幾乎已經成為了戰爭實驗室。

那麼這些坦克在戰場上的表現到底如何呢，以T-90坦克來說，該坦克可以說什麼是比較先進的坦克，但在戰場上損失也比較嚴重。據統計，自2015年以來，俄羅斯援助敘利亞的60輛該型坦克，現在已經損毀了二十多輛。T-72坦克其實更為慘重。

為了彌補這種損失，俄羅斯將由T-62坦克改進的T-62M也大量援助敘利亞。為了發揮這款老式坦克的性能，在改進型中，俄羅斯也花費了不少力氣，進行了現代化改進，雖然未更換主炮，但是為該坦克配備了9k118激光制導炮射導彈，使得其射程達到了4000米，裝備KTD-2激光測距儀和BV-62彈道計算機，以及微光瞄準鏡和火炮穩定系統，都使得該坦克的火控系統反應更為靈敏。

為了增加防護力，坦克的車體上還加裝了BBD被動裝甲，該裝甲由鋼板和泡沫阻燃塑料組成，極大的加持了坦克的防護能力。在場上，該坦克的表現甚至要優於T-72坦克。不過，還是抵不住叛軍優良的反坦克武器和地形對裝甲部隊性能發揮的影響，前方仍然不斷傳來該坦克被摧毀的消息。看來想要消滅敘利亞叛軍，還真沒有那麼簡單。
