印度人民抗議浪潮升級，印度政府讓通訊公司停掉人們的手機

2019-12-20 170條評論 1,099次閱讀 82人點贊

印度政府發布了全國性的集會禁令

據英國《每日郵報》12月19日報道，近日，印度人民反抗《公民身份法》的熱潮不斷升溫，這部法案提議印度政府授予2014年12月31日前因「宗教迫害」來到印度的巴基斯坦、孟加拉國和阿富汗三國非法移民印度國民身份，包括印度教、錫克教、佛教、耆那教、拜火教和天主教徒都有資格擺脫非法移民身份。但其最大爭議點在於，穆斯林被排除在外。

12月19日，印度人民再次無視全國性的集會禁令，對這部反穆斯林的公民法發動集體抗議。在此前的幾天抗議活動中，已有4人被槍殺，數十人受傷，數百人被捕，許多車輛被焚毀。

一名女士在今日的抗議中對警察怒吼

印度德里警方告訴當地記者，由於在首都發生了多起暴力抗議，目前德里已有14個地鐵站被關閉，其中包括本周早些時候被抗議者包圍的警察總部附近的一個地鐵站。這造成了嚴重的交通堵塞。

為了防止首都德里的抗議再度升級，電信巨頭沃達豐（Vodafone）和巴帝電信（Airtel）宣布，它們已切斷德里市部分地區的手機通訊以及互聯網服務，據悉，移動通訊公司Jio也將在不久之後切斷部分地區的服務。

警察在維持秩序

對此，活動人士憤怒地表示，在切斷互聯網與通訊服務這一方面，印度可以說是走在「世界前列」。目前，印度東北部部分地區和北方邦的互聯網服務已經受到限制，而北方邦正好是一個龐大的穆斯林少數民族聚居地。

一名男子因在抗議中喊口號而被警察抓捕

但是，切斷人民的通訊並不能停下抗議的浪潮，12月18日，印度知名演員兼導演法爾漢·阿赫塔爾在推特上表示：「僅在社交媒體上表達抗議的時代已經結束。」人們紛紛走上大街，公然表示對政府的憤怒。當日，在穆斯林人口密集的新德里，示威活動演變成了暴力事件，人們點燃了車輛，抗議者還向警察投擲石塊。
