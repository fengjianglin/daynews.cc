防止員工上廁所偷懶，英國發明傾斜馬桶，最長只能坐15分鐘

2019-12-18 2,144條評論 3,609次閱讀 2214人點贊

據英國媒體12月18日報道，英國廁所協會(British Toilet Association)正在推廣一種新型的馬桶，這種馬桶可以縮短公司員工上廁所打電話、玩手機的時間，有效防止員工借口上廁所而偷懶。根據統計，一般人在工作時上廁所花費的時間比在家裡多25%。

這種馬桶的主要特點是向下傾斜，目前英國廁所協會推薦的是向下傾斜13度的馬桶產品。馬桶製造商表示，因為馬桶呈現向下傾斜的狀態，所以長時間坐在馬桶上，會導致腿部肌肉酸痛，一般情況下使用者坐5分鐘就會想站起來，普通人最長也只能在這樣的馬桶上堅持15分鐘左右。

根據獨立調查機構在英國國內八個城市的統計，一般員工上班的時候平均每天在廁所里待著的時間是28分鐘35秒。折算下來，公司在每個員工身上每周損失的工作時間超過了兩個小時。因此，馬桶製造商表示，根據他們的測算，這項改進，可以為英國的企業每年節省160億英鎊的商業成本。

英國廁所協會的工作人員介紹稱，這種馬桶除了有利於縮短馬桶使用時間，防止員工偷懶之外，更重要的是它其實比普通的馬桶更有利於健康。使用普通馬桶容易導致痔瘡和盆骨肌肉萎縮，但使用這種傾斜馬桶，因為腿部一直在用力，所以可以促進人體下半身血液循環，減少這些疾病的產生。

目前，除了上述傾斜13度的馬桶之外，馬桶製造商還設計了傾斜35度的馬桶，據他們測算，向下傾斜35度的馬桶，可以將員工上廁所的時間輕鬆減少30％。

但是這種致力於減少員工上廁所時間的設計發明也招致了很多批評的聲音。英國有相關專家批評稱，這種設計完全就是上員工上廁所和正常工作對立起來看待，但實際上”工作期間員工想躲在廁所，只能說明公司組織管理不善，員工壓力過大”。
