塔利班炸死一美國軍人，還曝光身份證！現場血跡斑斑，人都碎了

2019-12-24 1,258條評論 3,864次閱讀 2890人點贊

12月23日，塔利班稱他們的武裝分子在當天炸死了一名美國軍人，並發布了一個血跡斑斑的背包和一名美國士兵的身份證的照片來證明這一點。美國軍方在一份聲明中說，一名美國軍人「在行動中陣亡」，但沒有透露細節，並在通知其近親之前向媒體隱瞞了這名軍人的姓名。

但塔利班發言人扎比胡拉·穆賈希德說，他們在北部昆都士省的一次爆炸中殺死了這名美軍成員。穆賈希德稱除了炸死1名美國軍人之外，還打傷了一名阿富汗突擊隊隊員。從阿富汗方面公布的照片來看，塔利班方面的信息基本屬實。除了一個血跡斑斑的背包之外，現場出現了大量血跡，還有幾乎被撕碎的軍服。可見這名美軍士兵不但凶多吉少，而且基本上連全屍都沒有。

值得注意的是，這次塔利班公布了美國士兵的身份證信息，從一個側面可以看出美軍官兵的個人信息的數字化管理手段。這實際是一種被稱為通用訪問卡 (Common Access Card)的智能卡。其性能外觀跟我們使用的最新的借記卡或者信用卡沒有區別，即前面有IC晶元，而背面有磁條，用於對建築物和受控空間進行物理訪問的主要卡，並提供對國防計算機網路和系統的訪問。

卡片的正面包含有軍人的姓名、軍種（包含陸海空、陸戰隊、文職、承包商等各類）、卡片到期時間、薪酬等級、職銜等。從塔利班公布的照片可以得知，這名被炸死的美軍士兵叫科貝爾·米切爾·詹姆斯，薪酬等級為E7，職銜是SFC，即Sergeant First Class，相當於我軍的上士，一般擔任排士官。此人屬於美軍部隊基層部隊士官的骨幹力量，連排長都要靠這些人指揮管理部隊。

E-7的美軍人員，基本月薪應該在3900美元左右（人民幣2.7萬），基本上在部隊可能已經幹了10年左右。除了基本月薪之外，米切爾·詹姆斯還應該有每年2萬美元的房屋津貼，另外還有每年4千美元的基本生活津貼，總共算起來1年大約能有7萬美元的收入。當然這裡面基本工資是要扣稅的，大約在6.5萬美元左右，在加上戰區的一些津貼，大約還能增加1千美元的收入，因此總收入預估在6.6萬美元左右（人民幣46萬）。而卡片的背面則包含軍人的血型、福利保障號碼、出生年月以及此人在日內瓦公約中享受待遇的種類。

據報道，大約2萬名外國士兵在阿富汗，其中大部分是美國人，這是美國領導的北約任務的一部分，目的是訓練、協助阿富汗部隊並為其提供建議。一些美軍對伊斯蘭激進組織進行反恐行動。2019年，已有十幾名外國部隊成員在阿富汗被打死。過去一年，美國一直在與塔利班武裝分子就一項協議進行談判，根據該協議，美國可以從其最長的戰爭中撤軍，以換取塔利班的安全保證。

12月，美國談判代表表示，在塔利班武裝分子對一個大型美軍基地發動自殺式襲擊後，他們將暫停談判。阿富汗政府軍在美軍的支持下，最近加強了在昆都士打擊塔利班的行動。但是美國承認，塔利班控制的領土比他們2001年被趕下台以來的任何時候都要大。
