果然是用完就扔！美國終於選擇退縮 中止對烏克蘭的巨額援助

天天要聞
2019-12-22

利刃/縱橫論

美國作為世界老大，在全球有著許多的盟友，而美國的態度也一向鮮明，那就是只要聽從美國安排，世界老大就會給足夠的援助和保護。然而，美國的保護承諾已被多次證明並不像他們自己保證的那樣牢固，就在剛剛傳來了一個令烏克蘭人心寒的消息。

據俄羅斯衛星網12月21日援引美國華盛頓郵報消息報道，白宮方面正打算放棄向烏克蘭劃撥軍事援助的條款。據悉，在20日通過的7380億美元的2020年國防預算中，內部包含對烏克蘭的軍事援助資金，理論上在接下來的45天內，華盛頓就需要向烏克蘭方面提供約2.5億美元的資金。但是現在，由於白宮的那位生意人強烈反對繼續向烏克蘭提供軍援，烏克蘭能否拿到這筆價值不菲的援助很成問題。

值得注意的是，美國常年的軍事預算中，通常都包含一些對世界上重點地區的對外軍事撥款，而用於俄羅斯的資金，以及資助反俄盟友的資金佔了很大一部分比例。目前，美國對外軍事援助分量最大的一筆資金，就是用於援助烏克蘭的撥款。在此前，美國已經向烏克蘭方面提供了大量步兵輕裝備，包括醫療、通訊、偵查和多種非致命性裝備，此外，例如標槍多用途反坦克導彈也被美國交付烏克蘭。在大筆的美國軍援支持下，烏克蘭人似乎還有繼續為美國充當馬前卒，牢牢拖住俄羅斯的鬥志。

然而，美國對烏克蘭的軍援其實是有著很大風險的，如果軍援不足，那麼不能讓烏克蘭有足夠的底氣去挑釁俄羅斯。這方面非常容易理解，畢竟俄烏的實力不是一個層次，唯有西方大力支持，烏克蘭才能發揮自己牽制俄羅斯的角色。但是，西方的軍援必須小心翼翼地進行，一旦軍援過火激怒俄羅斯，在俄軍直接動手的情況下，軍援越多，烏克蘭會敗得越慘。即使是最強硬的美國軍人也必須承認，烏克蘭並不是一個好的和俄羅斯對抗的戰場，在這裡爆發局部的大規模劇烈衝突，美軍也不能保證取勝。甚至可以說，一旦將俄羅斯徹底惹怒，俄軍將不惜一切代價求得勝利，而美國是否做好了為烏克蘭，和俄羅斯陷入一場決死的大戰的準備呢？答案似乎是沒有。

正因如此，這幾年來，美國和西方對烏克蘭的軍援是小心翼翼的，大部分是非致命性武器。即使是致命性裝備，仍然以步兵輕武器為主，美國根本不打算向烏克蘭提供先進的重型武器。而在經過4年的援助之後，烏克蘭地區的局勢卻日趨穩定，西方難以讓烏克蘭掀起更大的風浪。此時的烏克蘭如同一顆棄子，暫時不能為美國發揮太大的作用，而美國又沒有決心冒著激怒俄羅斯的風險大規模軍援。因此，暫時放棄烏克蘭成了美國的選擇。
