據透露，碧翠絲公主選定了自己的婚禮地點，此一選擇有些令人傷感

2019-12-14 2,747條評論 4,812次閱讀 2261人點贊

碧翠絲公主的婚禮最近因圍繞她父親安德魯王子的爭議和對他的指控而被蒙上了陰影。

皇室專家安吉拉·莫拉德宣布，她相信公主的婚禮會在她未婚夫家人居住地義大利舉行。這位皇室專家補充說，選擇這個目的地可能是為了逃避她父親在英國收到指控的影響。

安吉拉·莫拉德稱碧翠絲公主的婚禮將在義大利舉行

莫拉德女士說:「我認為，全面了解碧翠絲公主婚禮前後的情況將會非常有趣。這是因為父親的事件讓這場婚禮蒙上了陰影。

另外，碧翠絲公主與她的妹妹歐也妮公主的性格也完全不同，歐也妮是安德魯王子的第二個孩子，其性格更外向一點，但碧翠絲卻是一個非常注重隱私和安靜的人。

碧翠絲公主的婚禮地點已經公布

「我認為他們很有可能會在她未婚夫的家鄉義大利舉行婚禮，儘管傳統上的皇室新娘婚禮都是在英國舉行的。」

「但在這種情況下，去義大利舉行婚禮會更好。這位皇室專家指出，「如果是我，我也會這麼選擇的。

圍繞安德魯王子的爭議給他女兒的婚禮蒙上了陰影

或者她可以走聖喬治教堂的路線，這也是公眾非常關注的。

「我想看看是否會在電視上播出也會很有趣，因為我們都知道安德魯和菲姬曾很努力地想讓尤金妮的婚禮在電視上播出。

「但英國廣播公司拒絕了它，因為他們認為它的收視率不夠高，英國獨立電視台接受了它，然後英國廣播公司只播放了片段。

「但我不認為碧翠絲公主想要一個電視直播的婚禮。」

安吉拉·莫拉德還質疑婚禮是否會在電視上播出

當然，過去幾個月圍繞著她父親的情況表明，她會變得更加安靜。

「她甚至還不能確定婚禮舉行的確切日期。」
