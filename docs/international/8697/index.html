真事！東京——舊金山只需5.5小時 3年後啟用

2019-12-09 2,081條評論 3,097次閱讀 386人點贊

你還記得它嗎？超音速客機協和號（Concorde）速度快的不凡印象一直讓世人津津樂道。雖然已經成為歷史，但如今超音速飛機製造商Boom Supersonic研發Overture噴氣式飛機正試圖重塑超音速客機的形象。

日本航空（JAL）與維珍航空（Virgin）這兩家航空公司已經一共預訂了30架飛機。未來從舊金山到東京之間的飛行時間只需5.5小時，將使全球旅行具有更大的靈活性，客機將於2023年開始進行商業運營。

2014年成立的Boom Supersonic，推出超音速飛機Overture 圖片來自網路

什麼是協和飛機

協和飛機（法語、英語：Concorde）是一款由法國宇航和英國飛機公司聯合研製的中程超音速客機，它和蘇聯圖波列夫設計局的圖-144同為世界上少數曾投入商業使用的超音速客機。協和飛機在1969年首飛、1976年投入服務，主要用於執行從倫敦希思羅機場和巴黎戴高樂國際機場往返於紐約肯尼迪國際機場的跨大西洋定期航線。飛機能夠在18000米的高空以2.02倍音速巡航，從巴黎飛到紐約只需約3小時20分鐘。1996年2月7日，協和飛機從倫敦飛抵紐約僅耗時2小時52分鐘59秒，創下了航班飛行的最快紀錄。但是由於票價昂貴和飛機雜訊污染等問題，全球所有的協和客機已經在2003年全部退役。

協和飛機共生產了20架，其中僅有16架投入營運，協和飛機代表著航空技術史上的技術進步，因此即使退役後，協和飛機仍然是航空歷史上的重要象徵，而英法聯合研製的經驗更為後來歐洲多國共同組建空客公司鋪路。

英國航空「協和號」 圖片來自網路

負擔得起的出行新方式

美國三家公司：Boom Supersonic、Aerion Supersonic 和 Spike Aerospac 競相成為第一個將超音速客機推向市場的公司。而2014年成立的Boom Supersonic， 目標是讓所有人都能買得起超音速客機的機票，因此推出了新型飛機Overture，飛行時速為2.2 馬赫、約 2,333 公里，飛行速度是一般民航機的兩倍以上，噪音則比協和號大幅減低，但是票價只是目前航空票價的2-3倍，成為大多數乘客可以負擔得起的出行方式。

日本航空（JAL）預購多達20架Boom Supersonic的飛機 圖片來自網路

飛越太平洋不到6小時

日本航空（JAL）客機 圖片來自網路

Overture飛機內部使用全商務艙設計，機艙內可提供55個座位。日本航空（JAL）預定購買多達20架Boom Supersonic的飛機。未來從舊金山到東京之間的飛行時間，可望縮短到只有5.5小時，將使全球旅行具有更大的靈活性。
