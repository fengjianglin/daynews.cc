非洲強國熱衷採購中國造，軍艦導彈無人機都是中國貨，連衛星也是

2019-12-15 41條評論 3,743次閱讀 2718人點贊

作者：戰忽局編外

海軍，中國戰艦的忠實擁躉

阿爾及利亞海軍中大到導彈護衛艦，小到導彈艇，都是中國製造，堪稱中國戰艦的忠實擁躉。中國不但向其出口成品軍艦，還按照阿爾及利亞海軍的需求提供艦炮、防空導彈、反艦導彈等，並協助其安裝在阿爾及利亞自己建造的軍艦上，支持其戰艦國產化。更加難能可貴的是，中國還為阿爾及利亞援建了海軍船廠，以提升其國防工業水平。

C28A型出口型導彈護衛艦

目前，阿爾及利亞裝備有3艘中國製造的C28A型出口型導彈護衛艦。該型護衛艦滿載排水量為2800噸，於2013年開始建造，外觀和我國自用的053H3型導彈護衛艦類似，也融入了某些054型護衛艦的特點，具有一定隱身性能。

C28A擁有1門76毫米主艦炮、1座8聯裝紅旗-7近程防空導彈、8枚C-803反艦導彈、2座730近防炮系統及輕型魚雷發射裝置，還能夠攜帶一架中型艦載直升機。可以說是防空反潛反艦能力應有盡有，性能均衡，是一款作戰能力非常強的新型高端軍艦。除了阿爾及利亞方面要求加裝的少量歐洲電子設備之外，其他全部為中國製造。

T50A綜合訓練艦「高峰」號

阿爾及利亞海軍「高峰」號訓練艦，雙37炮充滿中國風格。

阿爾及利亞海軍還有一艘大型綜合訓練艦——「高峰」號也是中國建造的。「高峰」號型號為T50A綜合訓練艦，排水量為5500噸，建造時的舷號為「中阿937」號，該艦一度是阿爾及利亞海軍中排水量最大的軍艦，直到近年才被一艘排水量為9000噸的兩棲攻擊艦超越。「高峰」號從艦體到武器裝備都呈現出濃郁的中國特色。

吉貝欽奴阿級導彈艇

阿爾及利亞生產的「哈桑巴巴爾」號導彈艇裝備全套中國火炮導彈武器。

除了整艦出口外，中國還為阿爾及利亞自己建造的「吉貝欽奴阿」級導彈艇提供武器系統。「吉貝欽奴阿」級導彈艇，阿方稱之為輕型護衛艦，是阿爾及利亞自主建造的第一款軍艦，除了艦殼外，該級艦全套武器裝備都為中方提供，如：C-802反艦導彈和H/PJ-26型76毫米艦炮。

除此之外，阿爾及利亞還計劃引進1500噸級的海警巡邏艦艇，目前，這艘艦艇正在中國造船廠建造中。

空軍，察打一體無人機唱主角

阿爾及利亞空軍武器裝備方面採購的中式武器主要是大量彩虹3和彩虹4型察打一體無人機。未來，不能排除其購買中國的戰鬥機等更加先進航空裝備的可能。

阿爾及利亞彩虹-4無人機等武器裝備。

彩虹-3無人機

在阿爾及利亞空軍基地進行彙報飛行的彩虹3（圖源：社交媒體）

彩虹-3基本型於2005年開始研製，2007年首飛成功。該型無人機採用鴨式布局和翼身融合後掠翼設計布局，主翼翼尖各有一個較大的翼稍小翼，提高了隱身能力，使得低空/超低空機動性能更好，且升力也更大。

該型無人機通過掛載光電載荷、合成孔徑雷達、電子戰等設備來實現不同類型的戰術任務。是國內首款察打一體無人機，可攜帶光電偵察載荷和2枚AR-1型小型空面導彈，也可掛裝50千克級別的FT系列精確制導炸彈。攻擊精度小於1.5米，適用於偵察和對地面固定和移動目標進行精確打擊。

「彩虹3」先後出口到巴基斯坦、奈及利亞、土庫曼、緬甸、尚比亞、阿爾及利亞等多個國家，是中國首款實現批量出口且出口數量最大的攻擊性無人機。

阿爾及利亞電視台曝光的CH-4，注意其背部的天線。（圖源：社交媒體）

彩虹-4整體性能指標優於美國「捕食者A」無人機系統，被稱為「中國捕食者」。 彩虹-4無人機是目前國內同級別無人機系統中掛載能力最強、飛行性能最優的無人機系統之一，由於機體相對較大，設備較完善，包括照相、攝像等裝置以及光電偵察載荷、SAR雷達、衛星通信天線等，機上搭載的電子設備比同類型的美國「捕食者」無人機更加高端，可針對不同天氣採取不同制導形式。

翼下設有4個掛點，通常可掛4枚導彈或炸彈。在使用最大載荷時，內側的兩個掛點使用複合掛架可分別掛載兩枚導彈，意味著一次出動可對6個目標實施精確打擊。阿爾及利亞購買的彩虹-4是比較高端的版本，裝備有衛星通信天線，可以在千里之外對該機進行控制，從而極大地提升了該機的作戰效能，減少了地面控制站被攻擊的可能。

除此之外，阿爾及利亞還從中國購買了本國第一顆通信衛星——阿爾及利亞一號通信衛星，並由中國的火箭發射入軌，提供了從衛星研製、發射、軌道控制在內的一條龍服務。該星於2017年12月11日成功發射並順利入軌。
