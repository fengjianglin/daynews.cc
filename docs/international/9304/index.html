這些韓國高考生居然提前查到了分數

2019-12-04 1,154條評論 2,261次閱讀 353人點贊

今日話題

12月4日是韓國公布高考成績的日子，然而提前3天就有三百多名考生提前查詢到了成績單。韓媒稱，這是自1994年韓國實行高考制度以來首次高考成績泄露事件。由於險些釀成大禍，招致韓國社會的廣泛批評。

據韓國KBS電視台報道，2020學年大學修學能力考試（相當於「高考」）成績，將於12月4日公布。然而，12月1日晚上，竟有考生提前在一個網路論壇「曬」出了成績單，並配文稱：「憑這個成績，我能逃出現在就讀的大學嗎？」經查證，發帖人是一名已讀大學後又復讀重新參加高考的學生。他自稱找到了系統漏洞，所以提前獲知了成績。

那麼，成績是如何提前泄露的呢？它源於負責公布成績的「韓國教育課程評價院」網站。根據該名考生在論壇上的「解密」，在12月4日之前，該網站原本只能查詢到1995-2019年的高考成績。但他發現，使用一款「瀏覽器開發者工具」的軟體，便可以將「查詢學年」一欄中的年份改為2020年，最終他成功查詢到自己的成績。這條「解密」帖一時間吸引了大量考生的關注——僅3個半小時內，就有312名考生利用這一系統漏洞，提前查詢了自己的成績。

韓媒稱，這是韓國自1994年實行高考制度以來，首次發生成績提前泄露事件。韓國教育課程評價院對此解釋稱，當時他們正進行發布成績前的測試，然而在測試中系統出現了漏洞。

韓國教育界一致表示，高考成績提前泄露可能導致嚴重後果，政府機關網路系統亟需嚴查。值得慶幸的是，考生選擇錄取考察方式早在系統出現漏洞前已結束。儘管如此，成績泄露事件招致廣泛批評。韓國輿論認為，作為負責國內最大規模考試的公共機關，安全系統竟如此不堪一擊，令人難以接受。
