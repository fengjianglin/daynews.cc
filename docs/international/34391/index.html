日本人的瞎帳事：為籌備奧運會搬遷世界最大海鮮市場，如今有點黃

2019-12-24 1,921條評論 3,583次閱讀 1347人點贊

今天，我們來了解的是一個曾經的輝煌—-世界最大海鮮市場，日本築地市場。它佔地23萬平方米，地處鬧市區，這裡經營空間比較局促，周圍居民也被海鮮味道滿滿包圍。

但是，這並不妨礙它的火。平均每天有近5萬人進出，賣出各類海鮮商品2000噸，最高的時候更是達到3200噸。

海鮮市場經營來自日本海和非洲、美洲的大約四五百種海鮮，其中金槍魚最為有名。很多美食專家認為，這裡的金槍魚是世界上味道最足的。

每天凌晨四點左右，新上岸的金槍魚就開始擺上市場了。它們首先被放在地板上，一條條編號，凌晨5點左右的時候，金槍魚的交易正式開始，這個過程大約持續90分鐘左右的時間，期間，其他海鮮產品的交易同時進行。

雖然這裡最著名的是金槍魚交易，但是很少有遊客觀看到整個交易的過程。因為為了不干擾交易，清晨的金槍魚拍賣現場不能自由參觀。

遊客要在七點以後，到築地市場的觀光管理部門登記排號，然後組織統一參觀，每日名額最多限150人。由於名額實在有限，很多遊客就早早起來，裝扮成前來購物的商販，得以混進市場。

這裡的一天開始的非常早，每天早上三點左右就活躍起來。一艘艘滿載蝦類、貝類、魚類等海鮮的大船來到這裡，熟門熟路找到自己對應的停靠碼頭，然後卸貨，第一時間擺上櫃檯。

據了解，這裡涵蓋了幾乎所有的海鮮，大約有近500種。

來這裡購買海鮮產品你可以放心，因為這裡商品全部明碼標價，它號稱是「日本人的廚房」，但是，來這裡採購的，卻以男人們為主，經營者也是男人的天下。

築地市場誕生於1935年，近百年的經營，在最高峰期間，一天內有3200噸水產品成交。這裡的海鮮，支撐了日本的大半壁江山。

但是，真正讓築地成為旅遊熱地的，是它獨特的市場拍賣。台下的競拍者一邊打手勢一邊抬高自己的價碼，數秒之內便能完成競拍。現場有種驚心動魄的感覺，但這種感覺滿足了所有人的感官享受。

2018年10月6日的清晨5點，築地市場關門。這是為了給2020 年舉辦的東京奧運會騰地方，這一天，來自青森縣的金槍魚以每斤約1.35萬日元的價格，競拍出了當天全場的最高價。

但是，很多日本人對於關閉這座擁有近百年歷史的市場感到不理解，認為奧運會不值當的這樣下大力氣。更有當地居民去相關部門上訪，要求恢復市場。

新的市場搬遷到了日本豐洲，更大更氣派也更現代化，光面積便是原來的1.7倍，本以為會續寫輝煌，但是，新市場目前並沒有出現讓人滿意的購銷兩旺，特別是遊客幾乎消失殆盡，讓人感覺有點黃了的味道。

特別是新市場開張第一天，就出現了道路擁堵、卡車著火等情況，甚至還有現場示威的反對者。這種尷尬局面其實也在情理之中。

對於很多日本當地人和外國遊客來說，築地市場有近百年歷史，不僅僅是一個市場，更是一個時代的符號，來這裡不僅僅為了買魚，更為體驗傳承的味道，或者說是一種鄉愁。而當承載鄉愁的載體消失了，繁華落幕也是自然。
