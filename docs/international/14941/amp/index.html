為什麼說美國是全世界最浪費的國家？看看他們的超市就知道了

天天要聞
2019-12-15

說到美國，最先想到的就是這是一個發達且經濟實力非常強的國家。美國是一個超級大國，這事毋庸置疑的一個事實。就是因為這個國家的強悍，所以也讓很多朋友對美國人有一個認知：美國人很有錢。確實美國很有錢，但也很浪費。在浪費這一點上，美國甚至可以說是全世界最浪費的國家，沒有之一！看看他們的超市就知道了。

在美國有很多像沃爾瑪這樣的倉儲式超市，這樣的超市裡面的商品種類非常多，據統計沃爾瑪的商品的種類達到29萬種，是一個相當龐大的數字了。不僅如此，沃爾瑪超市裡面有些東西的保質期只有三天，由於商品的保質期比較短，商品的價格也是非常便宜的，超市在這些東西上面根本拿不到利潤。如果在這幾天之內，這些商品沒有賣完，就要全部扔掉，不會再出現在櫃檯上。

還有一點就是有些加工好的食物，如果溫度降到了一定的程度，也是要被清理掉的，美國每天光超市裡扔掉的食物就有很多了。而且這些超市還時不時的搞搞促銷，使買到的商品就跟不要錢似的，造成美國普通百姓家裡也大量堆積著商品。這些商品的定價也是美國人浪費的原因之一吧。

另一個原因大概就是和他們的居住環境有關了，眾所周知美國是一個地廣人稀的國家，美國家家戶戶基本上都有車。在這樣人煙稀少的地方，沒有車就相當於殘疾人沒有拐杖一樣，去那都很費勁。所以每家每戶都有至少2台車，而且他們的汽油和車輛的價格都是非常便宜的，開車出行也是他們的標配了。這也就導致了個人的能源消耗也是非常高的，最終也造成了一種能源浪費。

這也是為什麼說美國是一個浪費的國家，除了它以外，估計也沒有國家可以這樣浪費了吧！
