女子上演“竊聽風雲”，借女兒玩偶暗中監聽前夫

2019-12-22 2,175條評論 4,556次閱讀 3153人點贊

《紐約郵報》資料圖

海外網12月21日電 美國一名女子為了監控其前夫，在送給女兒的米妮老鼠玩具中秘密安裝了竊聽器。目前，該女子因竊聽等罪名被捕入獄。

綜合美國《紐約郵報》等媒體報道，美國新澤西州警方近日逮捕了一名涉嫌竊聽罪和騷擾罪的33歲女子希瑟·班福德(Heather Bamford)。這名女子給女兒的米妮玩偶中安裝了錄音裝置，用以監聽其前夫。

警方稱，該女子的前夫多米尼克·班福德(Dominic Bamfor)在上周日(12月15日)無意間按下玩具的肚子，本來是希望聽到玩偶內置的說話聲，結果卻聽到了女兒和自己的聲音。這一發現讓他不寒而慄，隨即報了警。

警察調查後，以竊聽罪和騷擾罪將希瑟·班福德抓捕。目前尚不清楚該女子的竊聽行為持續了多久。該案件正等待審理。

據悉，2009年美國也發生過一起類似的案件。內布拉斯加州一名女子為了爭奪孩子的監護權，在女兒的泰迪熊里安裝了竊聽器，準備利用錄音帶爭取法官支持，不過最終被判證據無效，且該女子和她的父親還被判要向其前夫賠償12萬美元(約合人民幣84萬元)。
