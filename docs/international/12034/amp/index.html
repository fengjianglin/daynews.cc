優雅了90年的英國女王伊麗莎白二世，比小說更傳奇的一生

天天要聞
7 天 ago

近日，據英媒體報道：超長待機的英國女王伊麗莎白二世或許會在18個月後——即2021年滿95歲時退休。

這已經是她「職業生涯」中經歷的第14位首相——「鐵打的女王，流水的首相。」

這個英國政壇上面的傳奇女人，一生都可以用華麗來說。

作為這個世界上最著名的皇室女性之一，女王伊麗莎白二世年輕時並不像現在這麼素氣。她享受過無數的奢侈服裝以及名貴珠寶。

20世紀50年代由諾曼·哈特內爾為伊麗莎白二世設計的晚禮服。刺繡工藝唯有文藝復興時的禮服可以相比。

1957年4月8日至11日，伊麗莎白二世對巴黎進行國事訪問。這件禮服上裝飾有法國圖案，包括法國花，大金蜂（拿破崙的象徵）。這張照片是第一個晚上(4月8日)，在愛麗舍宮(Elysee Palace)的總統Rene Coty主持的國宴上，女王穿著這件禮服觀看了歌劇表演。
