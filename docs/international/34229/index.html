日本明仁上皇迎86歲，家族成員齊現身，愛子悠仁放學後赴慶生宴

2019-12-25 84條評論 1,063次閱讀 98人點贊

當地時間23日，日本的上皇明仁，迎來了他86歲的生日，日本皇室的工作人員發布了一組照片來為他慶祝。明仁上皇和相濡以沫60年的妻子美智子一同站在東京皇居的花園中，從兩位老人的服裝看，照片應該拍攝有些日子了。

84歲的美智子皇太后，自從淡出人們視線以來身體狀況一直堪憂，4月是被爆出體重持續下降，8月她接受了乳腺癌手術，據悉手術效果不錯，癌細胞也沒有擴散。而據《女性自身》雜誌報道稱，自9月以來美智子已經有數次嘔血了。不過具體原因皇室方面沒有進一步說明，可能跟服用的藥物有關，不過也跟精神壓力大不無關係。

曾有消息稱，美智子的精神壓力來自大孫女真子公主的婚事，和小室圭的若有似無的婚約，不僅讓真子本人苦惱，也讓她的祖母為之擔憂。不過最新消息稱，皇太后的壓力更多的來自她欺負兒媳婦的流言，儘管多年前就有「雅子因為生育問題而受到來自婆婆的打壓」這樣的說法，但隨著雅子成為皇后，類似於這樣的消息在日本國內網路上就愈演愈烈，而且還有很多聲稱是宮內人士發布的。而美智子早在多年前就習慣在網路上看新聞了，她對這樣的東西不會視而不見的，尤其是對於她那樣一位心思細膩的女性來說，看到之後所承受的壓力會更加巨大。

為了慶祝上皇的86歲生日，德仁天皇和雅子皇后當天午後到父母所住的「吹上仙洞御所」去拜壽。穿著水藍色套裝的雅子笑得很是燦爛，還不停地向圍觀群眾揮手致意。成為皇后之後的雅子，多少有點兒揚眉吐氣的味道，笑容也比以前更多更真實了。

相比較起來，紀子妃似乎就沒有嫂子那般的好心情了，臉上掛著的是習慣性的「禮貌笑」。

待嫁閨中的真子公主和佳子公主也一起乘車前往拜壽，姐妹二人的劉海很是有趣，分別撇向了不同方向。據《女性周刊》報道稱，真子公主的未婚夫小室圭，目前正在美國留學，為了考取紐約州的國際律師資格證而努力中。但那個資格證只能在當地才有用，如此說來他和真子的婚約是否已經「名存實亡」了呢？

當天晚些時候，德仁天皇夫婦再次乘車到「吹上仙洞御所」，準備參加慶生晚宴，這一次剛剛過完18歲生日的愛子公主也加入了，她正在準備考大學。

在文仁王儲的車中，悠仁王子也出現了，這位初中一年級的男孩穿著傳統的日式禮服，顯得很是乖巧。愛子公主和悠仁王子都沒有在白天出現，應該是跟上學有關，畢竟他們還都是在校生。
