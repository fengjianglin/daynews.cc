店鋪關門！世界“購物天堂”空置率上升5倍，著名的“第五大道”怎麼了？

2019-12-18 1,666條評論 2,998次閱讀 959人點贊

美國紐約曼哈頓的第五大道以高端設計和平價品牌而聞名，是世界上著名的購物目的地之一。然而隨著近年來網上購物的火爆，傳統商鋪銷售額受到打壓，第五大道的許多商店選擇關閉店鋪。

為了讓第五大道恢復生機，當地管理部門希望通過改造這一區域，重新吸引遊客。

與法國巴黎的香榭麗舍大街和日本東京的銀座一樣，紐約曼哈頓第五大道是各大品牌旗艦店的聚集地。除了各類日用品商店外，還有許多由世界頂級設計師設計的高檔櫥窗和展品。由於品牌齊全，品類豐富，也成為遊客們不想錯過的旅遊目的地之一。


美國亞利桑那州遊客 梅麗莎⋅馬丁內斯：我們只來過紐約一次，這是我們的第一次，第五大道是我們一直想要來的地方。

美國新澤西州遊客 雷琳達⋅波頓：每次我們都會來這裡，對我們來說已經成為了一種習慣。

如今由於世界各地的購物者越來越多的選擇在網上購物，紐約第五大道零售商的生意變得越來越難做。僅在今年，奢侈品珠寶商亨利⋅本德爾、羅德與泰勒百貨商店、休閑時尚品牌蓋普就關閉了在第五大道的店鋪。

美國房地產諮詢公司高緯物業的數據顯示，在過去10年間，第五大道的租金逐年下降，但店鋪的空置率卻上升了五倍多。

美國高緯物業執行董事 史蒂文⋅戴克：從歷史上看，第五大道一直是紐約租金最高的地方。我認為，在過去的兩三年里，第五大道的租金大幅下降，房東們正在盡其所能把店鋪租出去，零售商也會用更少的空間做更多的事情。

為了讓第五大道恢復往日的活力，這條街的管理者第五大道協會也在積極尋找改善遊客體驗的方法。比如引入更多的快閃店，美國第三方支付服務商貝寶近期在第五大道開設了一家假日店，專為小企業提供支付處理器。

而傳統的零售商則將引入更多新元素，改善消費者的購物體驗。

美國某廣告設計公司行政副總裁 泰勒⋅赫爾姆斯：我們注意到第五大道的很多商店都關門了，但其它商店卻在改變經營方式。商店再也無法只通過提供產品來實現銷售，必須提供更多的東西，讓消費者走出家門，進店消費，而不是在網上購物。

美國高緯物業執行董事 史蒂文⋅戴克：未來幾年，我們希望第五大道能夠做出一些改變，讓第五大道成為一個永不打烊、充滿活力的地方，在這片區域增加更多的美食和娛樂項目。
