馬哈蒂爾說美國制裁伊朗違反國際法，是耿直老將沒錯了

天天要聞
2019-12-16

亞太日報綜合 杜瑋

據雅虎新聞報道，馬來西亞總理馬哈蒂爾·穆罕默德（Mahathir Mohamad）周六在卡達首都多哈的一次會議上明確表示，美國對伊朗的經濟制裁是非法的，該制裁違反了《聯合國憲章》和國際法。馬來西亞不支持美國這一舉措，因為這些限制措施限制了馬來西亞進入這一重要市場和貿易夥伴的渠道。

馬哈蒂爾在多哈論壇上直言：「馬來西亞不支持美國重新對伊朗實施單方面制裁。」他認為，美國對伊朗的制裁非常橫行霸道，越俎代庖。因為這麼做不僅令馬來西亞和其他國家失去了一個「大市場」，更是直接干預到了聯合國的職權範圍。他還表示：「美國的這種制裁顯然違反了《聯合國憲章》和國際法，而眾所周知，這類制裁只能由聯合國根據《憲章》實施。」

事實上，這已經不是馬哈蒂爾第一次在公開場所表明自己反對美國的態度了。在今年的第74屆聯合國大會上，各國代表紛紛登台發聲，其中馬哈蒂爾的發言，引發了在場各國共鳴，使得美國瞬間就成為了過街老鼠，被多個國家代表在眾目睽睽之下譴責。

馬哈蒂爾當時在聯合國大會的發言很有感染力，同時也很非常巧妙，他並沒有指名道姓的說美國，而是抱怨了伊朗問題，他表示當對一個國家實施制裁時，其他國家也會受到制裁，比如說伊朗，伊朗受到了制裁，馬來西亞和其他國家就失去了伊朗的市場，同時也失去了伊朗的石油。他還暗示，隨意制裁他國已經成為某些富國強國的特權，這令人悲哀。

馬哈蒂爾發聲之後，立刻就引發了多國的共鳴，譬如，頻頻遭受美國簽證限制的俄羅斯和伊朗當時也不約而同地提出同一個問題：既然美國如此刁難「來客」，是不是應該把聯合國總部搬出美國？一些人甚至開始討論搬到哪兒更好。在分析人士看來，雖然這一建議還不成熟，甚至有點兒「烏托邦」，但這也足以反映這些國家對美國的怒火已經不是一朝一夕的事兒了。

對此，美聯社總結稱，馬哈蒂爾表達的意思很簡單，美國可以為所欲為，弱勢方只能苦苦掙扎。馬哈蒂爾敢頻頻在國際會議上批評美國的行為，也令外界驚嘆，因為他已經94歲高齡，還如此耿直率真，敢於直言，實在令人佩服。雖說美國目前尚未對此事做出回應，但小編還是想勸一句特朗普，善良點吧！
