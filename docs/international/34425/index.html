印度學中巴合作模式，密謀和美國搞防務合作，卻先被宰150億美元

2019-12-23 347條評論 3,645次閱讀 2113人點贊

《印度教徒報》報道，在12月18日華盛頓舉行「2+2」對話會上，印度和美國達成了一系列的共識，並簽署了重要的《工業安全附件》協議，據悉，這個協議將大大加強印度的國防工業實力。

本次「2+2」對話是繼去年9月在新德里舉辦的美印「2+2」對話後，雙方第二次舉行該對話，從結果上看，本次對話對於美印雙方都十分重要，在會後舉辦的記者會上，美國國防部長埃斯伯表示此次會談簽署了多個重要的涉及到各種高科技技術轉讓和高科技技術的共同研發選項，具有重要的意義。美國國務卿蓬佩奧也表示，2019年的這個對話讓美印關係提高到了一個新的高度，雙方的戰略夥伴關係也在持續增長。印度方面對此次會談顯然也十分滿意，印度國防部長辛格表示，他和美國國防部長埃斯伯進行了極為友好的會面，而且還對美印雙方防務合作的過往進行了回顧，這些令他非常高興。印度外交部長蘇傑生則在記者會上興高采烈的表示，期待與美國方面繼續加強諸如防務領域的其他合作、兩國進行的貿易往來及其他人文交流。

對於此次對話雙方取得的關鍵成果，《工業安全附件》協議則是美印雙方之間的重頭戲，有了這個協議，美印之間可以分享大量的敏感技術，這意味著美印雙方在軍事領域的合作取得了重大進展，美國國務院負責南亞和中亞地區的代理助理國務卿愛麗絲韋爾斯則表示，這個協議將為美國和印度在有關軍事和其他方面的合作帶來便利，對加深美印之間的關係大有好處。

不過，美國人的東西從來不是那麼好拿的，根據報道，印度和美國敲定這個協議後，首先印度要拿出150多億美元去購買美國110架快過時的F-16戰鬥機，由於在今年和巴基斯坦的空戰中印度一口咬定他們的米格-21戰鬥機是被巴基斯坦F-16擊落，這讓印度對這種戰鬥機產生了濃厚的興趣，美國方面由於波音公司最近深陷各種麻煩，財務不斷惡化，也樂於積極推銷自己的產品，況且F-16在美國也屬於即將淘汰的戰鬥機，就算給了印度，美國也不吃虧，還能發揮餘熱繼續賺一筆。

▲巴基斯坦的F-16戰鬥機

而對印度來說，其落後的航空工業根本無法為其提供先進的戰鬥機，本國研究了三十年還沒弄明白的LCA光輝戰鬥機早就成了笑話，至今不堪一戰，面對美國轉讓軍事技術的機會，印度方面自然想起了中巴成功合作的梟龍戰鬥機，於是立刻像抓救命稻草一樣撲上去死死抓住，企圖獲得先進技術武裝自己，雖然印度得不到美國最先進的F-35戰鬥機，得到F-16研究研究也不錯，而且對面巴基斯坦用的也是這個型號，就當知己知彼了，不過美國絕對不會誠心相待，印度在後續中繼續被宰那也是可以預見的。
