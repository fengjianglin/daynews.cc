飼養條件差7隻已慘死，美國停止向中東派遣炸彈嗅探犬

2019-12-25 742條評論 2,164次閱讀 844人點贊

據路透社24日報道，美國國務院周一表示，已停止向約旦和埃及運送炸彈嗅探犬，原因是擔心被部署的犬只因中暑和中毒等原因死亡。

今年9月，美國國務院監察長辦公室確認被送往約旦的兩隻嗅探犬死亡。約旦是接收這些嗅探犬最多的國家。上周五發布的第二份報告稱，死亡的嗅探犬總數為7隻。

大約135隻嗅探犬參加了反恐援助項目，該項目幫助8個國家加強邊境和航空安全。美國國務院一名官員對記者說，已經在約旦和埃及工作的嗅探犬將繼續留在那裡，與此同時，美國當局要求採取措施改善這些動物的生活條件和飼養方式。

9月的報告發布後，在有人投訴後，監察長發現，又有兩條被送往約旦的嗅探犬死亡，一條因中暑死亡，另一條因狗舍內或附近噴洒的殺蟲劑而中毒死亡。

報告發現，被送往埃及的10隻嗅探犬中，有3隻也死亡了。一隻死於肺癌，一隻死於膽囊破裂，另一隻死於中暑。

根據9月份的報告，一名獸醫告訴監察長辦公室，「中暑是由於疏忽和護理不當造成的，不是意外事故」，死於中暑的狗「死得很慘」。

美國官員發現，被送到外國合作夥伴那裡的嗅探犬沒有簽署概述護理標準的書面協議，而且沒有對嗅探犬的健康狀況進行充分的後續檢查。

另一位美國國務院官員說：「改善嗅探犬健康和飼養是一件持續的事情。不會一夜之間完成，我們正在積極努力，改善養犬條件，改善檢查嗅探犬的次數。」
