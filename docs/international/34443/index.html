四個英國小哥看鄰居家的狗太可愛就寫了封擼狗申請書，好笑又感人

2019-12-23 95條評論 1,010次閱讀 98人點贊

平胸而論，對於喜歡狗的人來說，因為一些客觀原因而不能養狗的話，那真是一件很煩惱的事情。

來自英國的Jack小哥和他的三個室友就面對著這樣的艱辛：他們四個人都是從小跟小動物一起長大的：有的養貓有的養狗有的養魚有的養荷蘭豬。但是由於他們現在租的這處房子的房東不允許在屋子裡養寵物，他們自然也就沒什麼機會接觸到小動物了，這可把他們真的憋壞了，直到有一天……

有一天他們看到隔壁人家家裡的狗子出來遛彎，這四個大老爺們兒當時就fong了！狗啊！好可愛的狗子啊！看呀看呀在遛彎呢！好想rua一把啊！

可這麼不打招呼上去就擼人家狗也確實不合適，於是他們就很正式地給隔壁鄰居寫了一封「擼狗申請書」：

大概意思就是簡單介紹了一下自己，然後說明了房東不讓養動物但他們都很喜歡小動物，偶然間下班回來看到了鄰居的狗真是太可愛了，想問問有時間有機會的話能不能讓狗狗陪他們玩玩兒，或者是幫忙遛狗也行啊！這沒有狗的日子可怎麼過啊，拜託讓我們rua一把狗狗吧！

這封感人至深的陳情書送達之後沒多久，Jack他們就收到了一封回信，信封上有一個大大的爪印，收件人寫著：小哥哥們收。

當時他們就開心壞了：

這事兒成了！

原來這隻狗子叫作Stevie Ticks，是主人Sarah從救助中心領養回家的汪，現在已經兩歲多啦。它的性格超棒，一點都不怕生跟誰都親，見到陌生人也能很快跟人家打成一片。

主人在回信里以Stevie的口吻介紹了自己，表示欣然接受rua狗的請求並把自己的聯繫方式給了小哥們，rua狗的那一天指日可待！

Jack小哥一開始就把這事兒po到了twitter上，當然也沒忘了把這封來自Stevie的來信一起更新上去。而最近的幾張照片就是他們四個人一起和狗子開心玩耍的畫面啦！

小哥說，他們的公寓不能養狗，而平時他們也有空餘的時間幫Stevie的主人遛它，也算是幫人家分擔了一點點的工作，這樣也能說雙贏：小哥們能擼狗，主人也有了點點空餘時間，大家都開心啊！

這四個小哥說起來都是工程師，平時的工作可能不免枯燥，現在下班回家有機會出門走走還有狗可以擼。真是好日子啊啊啊！

說起來這條twitter現在已經收穫了將近50000的贊和10000多的轉發，而狗子Stevie的ins賬號也有了4000多的粉絲。真是，有狗真好啊！
