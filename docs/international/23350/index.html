在中國，漢堡王為什麼干不過麥當勞？

2019-12-19 186條評論 2,172次閱讀 651人點贊

第一知道漢堡王，是因為一位美國「錦鯉」。

在美國，有這麼一個小夥子，在facebook上，每天堅持給漢堡王留言，這一堅持就堅持了兩年多的時間，寫下了637條評論。

在這位小伙的念念不忘之下，漢堡王終於有了「迴響」；於是，在那一年的聖誕節，這位小伙收到了漢堡王的聖誕禮物——終生會員。哪，就是下面這位小伙了。

有了這個終生會員卡，小伙在這家漢堡王的店裡，不僅停車場有自己的專屬車位；

餐桌上有自己的專屬餐盤；


店裡有自己的專屬餐桌；

甚至連店名都以他的名字命名；

最重要的是，漢堡王還給了他一張金卡，不僅是終身免費，連同行的人也免費。


這待遇，分分鐘就是漢堡界錦鯉的配置。

漢堡王這一波實力寵粉，經過互聯網的傳播，迅速收割了路人的好感和一批新粉，品牌知名度也由此打開。

然後，一直堅持火烤的烹飪方式，在一眾油炸快餐視頻中，也有了一席之地

但是，在進入中國之後，一切好似變了模樣。

先是有漢堡王前工作人員表示，漢堡王的便宜漢堡，除了豬肉都是油炸的，所有的小吃也是油炸的。

▲截圖自知乎

爆料的真假未知，只是在知乎上，一大波我用戶在吐槽「漢堡王」，諸如工作人員工作技能不到家、出錯貨拒不認錯、服務員不禮貌等等等等，不一而足。若不是這些用戶說出具體的地址和經歷，你會以為自己看到漢堡王大型黑粉現場。

▲截圖自知乎

在眾多吐槽當中，「為什麼漢堡王在中國市場火不起來」這一話題還等上了知乎熱搜，瀏覽量達3500萬。

話題下面，大部分是對漢堡王服務不到位的吐槽。

事實上，鮮姐覺得，漢堡王之所以火不起來，服務只是一部分，畢竟只要實力夠硬，火起來還是早晚的事情；之所以不火，鮮姐覺得，一方面是漢堡王沒能明白自己的定位，另一方面是對市場的錯估。

犄角旮旯里的店面

漢堡最大的特色是什麼？是便捷。

而需要便捷的場景往往是這樣的；

上班路上趕時間，到店裡買個漢堡，邊趕路邊吃；飢腸轆轆的趕到車站機場，距離發車還有一點時間，但又沒有足夠的時間吃正餐，這時候買個漢堡帶杯飲料走起；逛街走累了，找個地方休息一下。

這時候，快餐店的地理位置優勢就顯現出來了，麥當勞肯德基的店面選址基本在地鐵口、商場口、電梯口這些人流量集中的地方；而反觀漢堡王的選址，有時開著導航都不一定能找到。

就地理位置這一點上，漢堡王還被麥當勞當面吐槽：2016年，在法國一個僅有6700人的小鎮Brioude的公路旁邊豎起了一個戶外廣告牌。較常規廣告牌不同的是，這個廣告牌有兩塊，一塊短的和一塊長的。

短的上面文字描述是「前面5公里處有麥當勞出沒」，而另一塊，是留給漢堡王的「在開258公里就能吃到漢堡王了」。

對的，就是欺負漢堡王在人流聚集地沒有那麼多的門店。

事實上，漢堡王和麥當勞在店面的數量上確實有很大差距；麥當勞進入中國後，1990年到2008年之間，每年以53家門店的速度擴張；2008年之後，每年以345家門店的速度擴張，到了近兩年，每年新開420家。

再看漢堡王，2005年進入中國市場，在之後的7年時間裡，僅在國內部分城市開出了52家門店。到了2018年，漢堡王在上海開出了第1000家門店，即便是野蠻擴張時期，漢堡王立下flag也是三年內開1000家門店。

店面選址不好，數量還跟不上。再加上進入中國市場晚了十幾年，主流商圈的商鋪早就被搶佔一空，就算漢堡王想瘋狂擴張，也很難在主流商圈找到合適的店面。想要彎道超車，只能著眼於新商圈的開發，而新商圈周邊的配套設施通常又不是很完善，很難快速圍攏人流，這也給我們造成一個錯覺，漢堡王總是開在不知名的地方。

再加上，漢堡王自己實力作死。

文化歧視，實力作死

如果說，上述的市場定位不明確還只能算戰略失誤，那接下來漢堡王這波操作，真的是實力作死。

2019年4月，漢堡王在Instagram上分享一個視頻，視頻中「顧客」手拿尺寸誇張筷子，分外吃力的啃著漢堡，更有意思的，筷子並不是單手操作使用，而是一手一隻，動作笨拙吃相難看：

講真，在杜嘉班納出現筷子風波之後，漢堡王還能出這樣一個視頻，雖非故意，但可以看出漢堡王作為一個跨國企業，對自己即將要打開的亞洲市場知之甚少。

雖然，漢堡王隨後將爭議廣告移除並進行道歉。「我們對這則冒犯到（亞裔）社區的廣告表示誠摯歉意。它並不能體現我們追求多元化以及包容性的品牌價值觀。」但從側面反映出，漢堡王對自己要進入的亞洲市場不夠了解，至少是對亞洲文化的不夠了解。這種不了解，延伸到產品上，就非常致命。

漢堡王推出了的一些產品，並不符合普通國人的口味。同樣的產品，麥當勞和肯德基在進入中國市場之後，不停的在調整優化自己的產品，更加的本土化，到了漢堡王畫風就變了。別說根據市場調整產品線，甚至出現「反其道而行之」的行為。

以至於成了漢堡界的段子還不自知——「聽說大部分中國人不喜歡吃洋蔥，也不喜歡生吃東西，所以我們在我們的所有牛肉漢堡尤其是招牌皇堡裡面保留了原汁原味的大量生洋蔥」

▲截圖自知乎

除了口味調整，在日常的服務處理上，漢堡王並不如麥當勞這些老大哥這些做的走心。拿兒童餐來說，麥當勞會經常有些當下流行的元素、玩具、主題去包裝兒童餐，基本上搞定了熊孩子，一餐飯吃的就不會鬧心；而漢堡王，儘管後來也加入了兒童套餐，但服務真的是「一言難盡」

▲截圖自知乎

店少還欺客，產品不本土化服務還差，再加上價格些微「貴」那麼一丟丟，能不能火起來，自己心裡還沒點X數嗎？！
