儘管領先優勢縮小 約翰遜得到1.99億美元信任票

2019-12-12 401條評論 1,642次閱讀 277人點贊

金融界美股訊 在英國首相鮑里斯·約翰遜有望贏得本周大選之際，規模最大的英國小盤股交易所交易基金（ETF）的投資者發出了看好該國經濟前景的信號。

貝萊德iShares安碩MSCI英國小盤UCITS ETF在過去三周累計流入資金逾1.51億英鎊（1.99億美元），整個12月有望創出最高紀錄。這支規模3.43億英鎊的產品本月以來資產增長了50%以上。

大量資金投入這支ETF，被視作對保守黨政府會通過財政刺激和脫歐使企業受益的信任票，即便是一項關鍵民調顯示保守黨領先優勢縮小。

資金流入「與押注保守派獲多數席位、英國脫歐前景相對明朗、經濟增長前景向好有關，」Raymond James的歐洲策略師Chris Bailey表示。「我感覺一些投資者一直在平掉低配倉位，而要影響小盤股並不需要太多流動性。」

在全球風險偏好上升以及對該國不會硬脫歐的樂觀情緒推動下，英鎊和英國股市都有所上漲。約翰遜承諾在1月31日前脫歐，2020年底前與歐盟達成貿易協定。他的工黨對手傑里米·科爾賓主張包括提高公司稅和國有化在內的左翼政策。
