日本最“貴”的古寺，參拜費高達三千日元，進去還要提前預約

2019-12-09 750條評論 2,112次閱讀 795人點贊

現在人們的生活水平逐漸提高，閑暇也就有了更多時間出門旅遊，大部分有條件的會選擇出國，而我們國內的遊客一般去的都是周邊的國家，像是東南亞地區的泰國，印度尼西亞，還有韓國日本等等，首先說到日本我們都知道，因為地理位置很特殊，所以這裡受到海洋的影響很大，內部的自然景觀非常漂亮。

但其實這座島國除了遍地都是浪漫的櫻花，還有很多古色古香的寺廟，其中有些是模仿了唐朝時期的建築風格，給人一種非常熟悉的美感。而今天要說的這個就不一樣了，在日本一眾寺廟裡應該算是最貴且講究的一個，你能猜出是哪裡嗎？它就是位於京都市西京區的西芳寺，又被人們稱作苔寺。

其實如果有了解過它背後的歷史就會知道，這個寺廟原來是飛鳥時代聖德太子的別莊，到了奈良時代一位僧人就在這裡建造了一座寺廟，從此以後它就一直以這樣的狀態存在，可是歷經百年之後，西芳寺也沒有逃過戰火的侵襲，一次次的被摧毀，又一次次的重建，之後又遭遇了不同程度的自然災害。

因為洪水不斷的侵入，這個地方在恢復原樣之後就留下了厚厚的苔蘚，放眼望去全都是綠色，加上周圍環境也非常的適宜，所以就給它們提供了非常適宜的生產環境，而重建之後，很多藝術家也對這裡做出了相應的改造，現在的西芳寺處處都透露著一股天然的枯山水氣息，對於僧人冥想有著很好的作用。

而正是憑藉著這一點，西芳寺名氣大增，開始吸引了越來越多的遊客，可是想要進入到寺廟內部參觀，也不是什麼容易的事情，需要提前半個月或者一個月預約，還要以手寫信的方式投到寺內，至於會不會被接受，那就要看運氣了，而且成功之後還要掏3000日元的參拜費，不得不說真的太貴了！

不過能夠見識到廟內的景觀，即使「肉疼」也會覺得非常值得，大家看到這裡是不是也很感興趣呢？之後有機會可以親自去體驗一下，但是能不能進去也要看個人運氣哦！
