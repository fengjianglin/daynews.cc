美國希望成為全球人工智慧領域的領頭羊，而且是在軍事領域

2019-12-06 348條評論 1,676次閱讀 290人點贊

導讀

  1. AI發展於民用，國家在軍事應用中的推進慢，但美國政府希望成為「領頭羊」；
  2. AI在軍事中的應用主要有無人機群、偵察等領域。

AI領先世界面臨困局

美國把人工智慧(AI)納入了第三次抵消策略，第三次抵消策略是2014年時任國防部長Chuck Hagel以重啟美國軍事技術優勢而啟動的，基於這個概念，第三次抵消策略主要集中在機器人和自動化方面，人工智慧是其中的關鍵，奧巴馬和特朗普政府都意於將美國打造為人工智慧領域的全球領先者。

2018年5月，特朗普為了此目標宣稱：「在美國，我們就是要最大可能的允許科學家和技術人員自由的發展他們下一代偉大的發明」。然而，儘管這樣，美國在人工智慧領域領先世界的路上仍然面臨國內外形勢的困局。

困局何在？

首先，和核武不一樣，人工智慧發展於非政府機構，比如谷歌和微軟公司，不是政府主導的。這些機構不是總願意與政府進行合作，這將阻礙人工智慧在軍事領域的應用；

其次，政府自身的國家人工智慧戰略推進緩慢，這裡需要強調的一點是，美國是最後一個發布正式的人工智慧國家戰略的國家，而加拿大是第一個(2017年正式發布)；

外部因素主要來自於其它國家對美國AI領先地位的挑戰，如中國和俄羅斯通過AI技術提升軍事武器系統的性能，以挑戰美國的軍事優勢。

因此，下面簡要分析奧巴馬和特朗普政府的主要國家政府舉措，負責人工智慧的研究和轉化的機構，以及目前人工智慧在軍事上的主要應用。

美國軍事AI的發展

美國第一個有關AI的聲明產生於2016年，當時奧巴馬政府發布了一份確認美國要在人工智慧領域佔據領導地位的重要性的路線圖。

該路線圖主要側重於監管政策問題，同年，白宮科技政策辦公室組織了多個專題研討會，並成立了機器學習和人工智慧小組委員會(MLAI)。主要用於協調國家AI計劃，並在奧巴馬主政時期發布了三項主要報告：

1. 為AI未來做準備

2. 國家AI研究與開發戰略計劃

3. 人工智慧、自動化與經濟

第一個報告建議的是與AI管治相關的策略、公共研發(R＆D)、道德和安全；國家AI研究與發展戰略計劃主要側重於研發策略；而最後一份報告則在考慮必要的政策以增強AI的優勢並降低其成本。

另一個重要節點是國土安全部在2017年發布了一份關於AI的分析報告，重點是如何更好地理解人工智慧帶來的利益和威脅。

同年，AI在「國家安全戰略」(2017年)中首次被提到，並聚焦於人工智慧在信息管理、武器化和監視方面的作用。

2018年的《國防戰略》指出，人工智慧能「確保我們(美國)能夠發起並贏得未來的戰爭，新的商業技術將改變社會，並最終改變戰爭的性質」，美國國防部在其AI戰略報告中呼籲立即啟動國家AI戰略。

因為：「 其它國家，尤其是中國和俄羅斯，正在為軍事目的在AI上進行大量投資，並將其應用於國際規範和人權問題等方面。這有可能侵害我們的技術和運營優勢，並破壞自由開放的國際秩序」 。

2018年5月，負責技術政策的總統副助理邁克爾·奎蒂奧斯(Michael Kratsios)在2018年白宮美國工業人工智慧峰會上重提特朗普的人工智慧目標：

  1. 優先為人工智慧研發提供資金；
  2. 消除創新障礙；
  3. 培訓未來的美國勞動力；
  4. 獲得戰略軍事優勢，引領人工智慧；
  5. 發揮人工智慧在政府服務方面的應用；
  6. 領導國際人工智慧磋商。

為了實現所有這些目標，奎蒂奧斯在峰會期間宣布成立一個新的人工智慧特設委員會，以「改善協調與人工智慧相關的聯邦計劃，並確保美國持續在人工智慧領域發揮領導作用」。

2018年7月，總統辦公廳發布了一份關於2020財政年度行政研究和發展預算優先事項的備忘錄。人工智慧是其中的最高研發優先事項之一，並強調了引領人工智慧的重要性(與量子信息科學和戰略計算一道)。

在2019年2月簽署的行政命令也指出：「美國是人工智慧研究中的世界領導者，維護了這個領導地位對美國來說是至關重要的」，最後，有必要強調AI第一次被視為政府優先事項是在特朗普政府的FY2019預算中，其中「預算的重點領域包括人工智慧、自主系統和高超音速研究。」

此外，2018年，DARPA宣布對新的和現有的人工智慧項目進行超過20億美元的長期投資，稱為「下一代人工智慧」計劃。

特朗普政府於2018年5月10日成立了國家科學技術委員會下屬的人工智慧特設委員會，該委員會是白宮在人工智慧研發優先事項方面的主要顧問，同時負責政府與工業界和學術界在人工智慧研究和開發方面的合作。

人工智慧在軍事中的應用

人工智慧這一技術必然對國家安全產生影響，據此美國和其他國家正在開發AI的軍事應用，並在這一領域進行了許多不同的實驗，可以說，到目前為止，包括中國和俄羅斯在內的其他國家都還沒有達到美國目前的水平，為了證明這一點，本小節將詳細介紹美國國防部範圍內進行的一些主要測試/實驗。

無人機和無人機群

2013年，美國海軍的X-47B原型無人機實現自主著陸，2015年該型無人機進行了自主空中加油，在這兩個試驗中，操作員只是在飛機飛行過程中下達了降落或加油的命令，具體操作是由軟體完成的。

軍事行動的組織將隨著AI應用于軍事和新的戰術構想而改變。無人機群就是這樣的一個例子，2016年，美國展示了103架無人機一起自主飛行，五角大樓把它描述為「一個集體的有機體，共享一個分散式的決策大腦。能像自然界的蜂群一樣互相配合」；

美國海軍在2016年11月進行了類似的實驗，由五艘無人駕駛的船組成的船群在切薩皮克灣的一個特定區域實現自主巡邏，並攔截了一艘「入侵者」船隻；其他關於水下無人駕駛艦艇的實驗預計將在不久的將來進行。

人工智慧用於情報、監視和偵察

美國參與的最著名和討論最多的案例是Maven項目，該項目將人工智慧應用於情報、監視和偵察行動，到目前為止，Maven項目一直被用於支持在伊拉克和敘利亞打擊ISIS的戰爭。

此外，美國軍事部門的重點是將人工智慧應用於半自主和自主的車輛，包括戰鬥機、無人機、地面車輛和海軍船。「忠誠僚機計劃」就是這樣一個例子。

該計劃使用上一代戰鬥機(F-16或B-1，經過無人化改造)或低成本無人機(XQ-58，AValkyRIE)搭配有人戰鬥機(F-35或F-22)，無人戰鬥機通過自主行動(不需要預先規劃編程)就能保護有人飛機；

陸軍和海軍也進行了類似試驗，原型車在戰場上跟蹤和保護士兵或車輛；「海上狩獵者」(Sea Hunter)是連續跟蹤反潛戰無人船原型，由DARPA首先開發，隨後將其轉移到海軍研究辦公室。「海上狩獵者」是第一個能自主地從加利福尼亞到夏威夷然後又回到加利福尼亞的無人船。

最後，人工智慧相關的法規是非常重要的，然而，美國尚未在這方面宣布任何測試或試驗。
