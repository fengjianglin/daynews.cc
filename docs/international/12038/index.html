加拿大多倫多舉辦學者論壇，紀念12.13南京大屠殺遇難同胞

2019-12-11 238條評論 1,269次閱讀 123人點贊

來源：交匯點新聞客戶端

交匯點訊 多倫多當地時間12月8日下午，多倫多北約克市政廳會議室，由加拿大和楓會主辦的「南京大屠殺遇難者 12.13 紀念日論壇」在此間舉行，五位在歷史、社會、文化、戰爭史等領域有研究的學者做了專題報告。多位專家學者教授以及港澳台同胞和社會各界人士、為79號議案和66號動議的長期推動者在自由討論中踴躍發言，共同表達了銘記歷史、珍愛和平的心愿。

出席論壇的有中國駐多倫多總領事館領事楊葆華，副領事曹培英，安大略省南京大屠殺紀念日動議（79號議案/66號動議）的提出者、安大略省前省議員黃素梅，多倫多市議員詹嘉里（Jim Karygiannis），萬錦市議員楊綺清等。

論壇開始後，全體與會者起立為南京大屠殺遇難者祭奠默哀一分鐘，表達哀思與銘記。接著黃素梅發表了講話。她首先感謝組織這次紀念活動的主辦方，並感謝詹嘉里議員2016年12月在多倫多市議會提出確認12月13日南京大屠殺事件的動議，與黃素梅在省議會提出的79號議案同步呼應。 黃素梅說，設立南京大屠殺紀念日與我們的下一代息息相關。許多大學教授加入今天的論壇，表示要努力讓全世界都知道這段歷史。我們正在籌建亞太和平博物館，取得的成績令人欣慰。安大略省作為一個擁有眾多亞裔學生的大省，應該讓每一個區的教育委員會都知道南京大屠殺，如同全加拿大人都知道的猶太人大屠殺、亞美尼亞大屠殺、對加拿大原住民的大屠殺一樣。一個對南京大屠殺遇難者的遭遇視而不見的社會不可能走向進步。

主持人王體博士向與會者介紹：「2012年，多倫多福特市長宣布12月13日為南京大屠殺紀念日，之後，市議員黃惠文（Kristyn Wong-Tam）和詹嘉里（Jim Karygiannis）於2016年推動多倫多市政府再次確認了這個紀念日。」 詹嘉里在發言中指出，他的祖父母家族中就有被送到希臘後，被德國納粹屠殺的受害者。再過兩天的12月10日是1948年的聯合國《世界人權宣言》發表紀念日，該宣言旨在防止歷史重演大屠殺悲劇。詹嘉里曾到過南京大屠殺遇難同胞紀念館，當他看到「災難之牆」的「遇難者 300000」有希臘文等文字鐫刻時，他的眼淚奪眶而出。他說，使用這些文字的國家和中國一樣，都曾有過被大屠殺的慘烈歷史。我們必須知道我們的歷史，這些大屠殺不僅僅是某個族裔或國家的歷史，而是全人類共同面臨的問題。所以，無論你身處何地，膚色種族為何，都要一起加入到這個推動人類和平的事業當中。 楊葆華感謝黃素梅與詹嘉里議員到場支持南京大屠殺紀念日論壇活動。他說：「我是帶著感動和感謝來參加此次論壇。在西方社會，對亞洲二戰歷史講得不多。華人社區珍視與祖籍國的聯繫，願意為宣傳南京大屠殺的真相做一些工作，特別是長期做南京大屠殺研究的人員和社區熱心人士，如果能一年年把工作堅持下去，會起到應有的效果，這不光是為我們華人社區做的事情，也是為中華民族和世界和平作出的貢獻。」

楊綺清所在的馬克海姆市，在2017年就在議會通過了12.13南京大屠殺遇難者紀念日。她說：「南京大屠殺事件，是一個警鐘，不能再讓悲劇發生，大家要珍惜和平。」 本次論壇組織策劃者、加拿大和楓會會長余承璋作了《不忘歷史 珍惜和平——感念在二戰日本侵華及南京大屠殺期間聲張正義的國際人道人士》。2017年9月4日聯合國向全世界發出公告，南京成為全球第169個和平城市。成為和平城市，是因為這座城市承載沉重的過去和市民們嚮往永遠的和平。當年那些國際友人以無私無畏的行動保護了幾十萬中國百姓的生命，為了民族的尊嚴，為了子孫後代，今天我們應該盡我們最大的努力為維護世界和平作出貢獻。 麥吉爾大學物理學博士郭曉明說：「自從2017年安大略省議會通過66號動議後，我們就應該年年祭奠南京大屠殺死難者。南京大屠殺是東京審判有結論的歷史事件，希望各社區各團體在每年入冬時節能都舉辦公祭南京大屠殺的活動，正視歷史，純潔良心，張揚加拿大人崇尚和平與人權的價值觀。 多倫多大學社會學系教授張衛國表示：「我們應該把南京大屠殺紀念活動制度化，理直氣壯地在全社會紀念南京大屠殺遇難者。我們推動設立紀念日是為了和平而非仇恨。我們應該以加強合作、研究、交流和傳承來推動在華人社區、加拿大社會和國際社會的教育工作。」 和楓會秘書長管建宏介紹說，加拿大有三個國家法定紀念日紀念猶太人大屠殺，在四個國家級紀念館都重點展示了猶太人大屠殺的內容。南京大屠殺在中國之外幾乎被遺忘。生活在加拿大的中國人有義務向加拿大主流社會展示這個人類歷史上罕見的冷血屠殺、戰爭暴行。加拿大的猶太人社區工作相當出色，是我們華人社區學習的榜樣。 今年10月參加聯邦國會議員選舉、獲得選區第二名的張川先生專程從滑鐵盧趕到多倫多，他說：「華人要融入主流社會，必須為族裔發聲。我們有義務把這件曾發生在人類歷史上最黑暗時期的種族屠殺事件，分析徹底，然後告訴年輕一代人，吸取教訓，永遠捍衛世界和平。 華文教育工作者張瑪琍發言指出，加拿大的華人後代對南京大屠殺非常不了解，作為教育工作者，我們應該鼓勵華人家庭對子女講解，讓他們了解祖輩的歷史，對下一代的教育是我們應該擔當的責任。

交匯點記者 吳昌紅 龔淑賢（實習生） 通訊員 張波
