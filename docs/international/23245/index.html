汲取邁阿密槍戰教訓“FBI”配發黃色字樣外套及防彈背心

2019-12-20 1,822條評論 3,396次閱讀 2267人點贊

由於SWAT人數有限，面對窮凶極惡的持槍匪徒，美國執法機構仍時常感到有心無力。1986年4月11日，美國聯邦調查局（FBI）邁阿密分局探員在高速公路上發現銀行搶劫案嫌疑人的車輛行蹤。該局迅速出動8名探員、5輛警車前往追蹤並伺機抓捕。在追蹤過程中，FBI探員被嫌犯發現後，決定改為強攻，使用車輛逼停對方車輛。

FBI負責人馬可尼爾認為己方人多勢眾，嫌犯只能乖乖投降。但他低估了對方的準備——嫌犯邁克爾·普拉特曾服役於美國陸軍「遊騎兵」團，嫌犯馬迪克斯是海軍陸戰隊隊員，兩人都經過良好的軍事訓練且裝備精良。在車輛被逼停後，坐在副駕駛位置上的普拉特迅速反應過來，取出一把Mini-14半自動步槍向警方車輛猛射，駕駛座上的馬迪克斯也拿起霰彈槍激烈抵抗。相比之下，FBI探員人數雖多，但僅裝備有幾支左輪手槍和一支霰彈槍，火力上明顯處於下風。在持續5分鐘的激烈槍戰中，雙方共發射了145發子彈，高速公路變成腥風血雨的戰場——雖然兩名嫌犯被擊斃，但參戰的8名FBI探員中有2人殉職、3人重傷、2人輕傷。

總結這次邁阿密槍戰，FBI和美國各執法機構深感手中火力不足，尤其是此前FBI標配的9毫米子彈威力不足，普拉特被這種子彈擊中5次後仍殺死了兩名探員。FBI後來認為，需要裝備大彈容量的自動手槍代替火力持續性弱的左輪手槍，並要使用更大威力的彈藥。

FBI在邁阿密槍戰中還汲取了一個教訓：槍戰爆發時FBI探員都穿著便衣，以至於目擊者並不知道這是抓捕行動，還以為是黑幫之間的火併。所以他們沒有立即報警，而是等到槍戰結束後才報警。為避免類似的誤會，此後FBI給每名便衣探員發了一件標誌性的裝備——背後印有醒目黃色「FBI」字樣的外套和防彈背心。
