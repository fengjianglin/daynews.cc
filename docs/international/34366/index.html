繼日本後，韓國出口列車也曝出缺陷！曾邀請中國共研高鐵標準

2019-12-24 1,882條評論 2,962次閱讀 656人點贊

本文為「金十數據」原創文章，未經許可，禁止轉載，違者必究。

中國、日本與韓國都是當今世界的高鐵強國之一，不過近年來日本與韓國的出口列車卻頻出事故。在日本2017年首次成功將高鐵出口英國後，首批交付列車第一次投入運營卻出現尷尬一幕，不僅晚點，途中也出現空調大面積漏水事故，這也打消了英國原本打算將主要高鐵訂單都給日本的想法，轉而將部分訂單投向中國。那麼，韓國高鐵技術又如何呢？

繼日本後，韓國出口澳洲列車發生重大”設計缺陷”

其實韓國的高鐵技術也可以稱得上是很不錯了，尤其是最近出口澳大利亞的一批列車，經一項審查發現，安全系統的先進性可達到世界前列，這也是吸引澳大利亞購買的重要因素。澳大利亞新洲當地更是已經拍板購買554節韓國現代集團在韓國製造的車廂，預計明年投入使用。

然而據澳洲新聞集團援引鐵路、電車和公交聯盟（RTBU）的看法，這批耗資23億澳元（約合96.87億人民幣）來自韓國現代集團製造的列車有設計上的安全風險，澳大利亞新洲當地並沒有好好管理交通項目。也就是說明年韓國列車投入運營的概率大大降低。

既然韓國的高鐵技術不錯，為什麼還會出現如此缺陷呢？其實就是列車沒有因地制宜，與當地基礎設施以及習慣並不配套。其一，在韓國新列車系統，列車警衛的工作方式將會改變，只能在車廂內為旅客提供旅行信息以及其他幫助。原本如果乘客在車外呼救時，警衛是可以聽到提供幫助的，然而新列車啟動後，警衛只能被關在車內看閉路監控。

另一方面，也是最主要的一個設計缺陷是，韓國的新列車沒有根據當地的隧道情況進行製造，尺寸過於寬了。韓國新款列車比澳洲目前的列車2.9米的寬還要寬20厘米，這小小的差異也意味著新列車可能會與隧道相撞。

澳大利亞新州的主要運輸和公路機構TfNSW在一份環境因素評估報告中說，如果要列車安全投入運營，必須在將在某些地方挖出一大塊，以拓寬所有隧道，甚至是重新調整軌道，重整軌道成本十分高昂，但是澳大利亞已經斥巨資買了韓國新列車，也只能如此了。

中國高鐵技術位居世界前列，韓國曾邀請中國一起研究

中國高鐵起步較晚，卻後來居上，據統計，截至2018年，中國高鐵營業總里程已經高達2.9萬公里,佔世界高鐵總里程的60%以上。在國際上，中國也是屢獲大單，不說為印尼修建成其國內的首條高鐵雅萬高鐵，東南亞中其他的寮國、緬甸、泰國、柬埔寨等國都有中國高鐵的足跡，甚至今年有望進軍歐洲發達國家，英國的5000億大單項目組正派人來中國洽談。

中國強勁的高鐵實力也引起了同為高鐵強國韓國的注意。據韓國媒體11月11日報道，當日，中國國家鐵路集團有限公司與韓國鐵道公社（KORAIL）在首爾簽署了鐵路合作備忘錄。接著，12月3日，韓國更是提出要和中國就”東亞國家間交通連接標準”展開合作研究，為中韓之外的第三方市場的基建合作提供支持。

對此，中國也做出了積極的回應，據媒體12月4日的援引中國代表人士的消息，未來中國將積極與韓國探討在海外基建合作領域開展第三方合作的可能性。
