印度家庭聚會，光用蔬菜300多斤，看到做好的美味，網友：很厲害

2019-12-18 2,375條評論 3,780次閱讀 1388人點贊

經常生活在農村的朋友們咱們也都是知道的，平時總會遇到一些紅白喜事，但是如果一旦遇到這些事情的話，肯定都是會請來親朋好友在一起聚餐的，而且準備的東西都是非常的豐富的，不管是什麼雞鴨魚肉，基本上都是有的或者是燉雞，燉鴨，燉魚，紅燒魚，燉羊肉等等也算得上是非常熱鬧的，但是在印度的農村同時也是有著像類似這樣的聚餐，今天小編就來帶你們看一看印度人在聚會的時候都會吃一些什麼吧。

​

其實他們準備了非常多的新鮮的蔬菜，而且這麼多的量，看上去差不多也有300多斤，這些蔬菜也都是非常豐富的，想必大家也都是知道現在有很多的蔬菜是要比肉的價格還要貴的，接下來我們就來看看他們都準備的都是一些什麼樣的蔬菜吧，都有西紅柿，胡蘿蔔，茄子，大頭菜，菜花，青椒等等，還有就是各種各樣的調料。

​

印度人先把這麼多的蔬菜全部都給切碎之後，再把全部切好，之後的蔬菜全部都放在一個大盆子裡面。等到把菜切好之後，又會再拿來一大口鍋，然後在鍋裡面放入一些油，把切好的辣椒洋蔥全部都放在鍋裡面翻炒，然後再把之前所有的蔬菜也全部都放中鍋中去炒，最後再加上非常多的咖喱粉，接著又往鍋中加入了非常多的水，把這些全部都放在一起燉煮，差不多經過一個小時燉煮之後，這道美食就已經做好了。

​

做好之後完全就像是一頓亂燉呀，但是看到印度人吃的還是非常的開心的，而且他們也都是非常喜愛這種美食的，看上去他們吃的也都是非常美味的，不過卻有很多的網友，看到那麼多的蔬菜被做成了這麼一大鍋的黃色的東西，都會覺得特別的無語，而且甚至還有很多的網友都說，這真的是糟蹋了那麼多的蔬菜呀，甚至還有很多的網友都說這真的能吃嗎？那印度人也真的是很厲害了，難道這也叫做美食嗎？不知道你們看到這些美食是什麼樣的感受呢？
