英國大選重磅：保守黨將大獲全勝，英鎊暴漲至9.49

2019-12-13 225條評論 1,931次閱讀 88人點贊

原創： UKPIWeekly 英國房產周刊 2019-12-13

英國當地時間12日22:00，大選投票結束。

根據英國BBC、ITV和SkyTV三大電視台聯合委託的投票站出口民調顯示，保守黨將在英國下議院獲得368席（比目前增長50個席位，比所有反對黨席位多出86席的絕對優勢)，工黨可能獲得191席（丟掉71個席位）。

由於以往的出口民調結果的準確度非常接近最終的投票結果，而備受關注。具體的真實投票結果，將陸續統計公布，要到13號周五中午全部完成。

出口民調結果一出，英鎊大漲。英國兌美元匯率立刻飆至1.34美元。兌歐元也漲至1.20歐元。英鎊對人民幣漲至9.49元。

英媒如何反應？

在出口民調結果公布後，12月13日各大英媒的封面都報道了保守黨以多數優勢贏得大選的民調結果。

13日關鍵時刻看點

儘管英國全部650選區的計票結果要到13日中午才能最終公布，但對於鮑里斯領導的保守黨而言，12日的夜晚並不那麼漫長。在投票正式結束後，隨著10個支持工黨的關鍵選區結果的公布，英國大選的結果即將正式揭曉。

Cumbria的Workington預計是最早公布結果的關鍵選區，大約在13日凌晨1點。

另外，其他9個決定大選結果的關鍵選區預計於凌晨3點公布結果。這些選區的共同點都是：2016年脫歐公投中選擇脫歐，但目前掌握在工黨手上。

1 : 30am：Darlington、Wrexham、Wrexham（3選區支持脫歐的比例為58%）

2 : 00am：Burnley(66%脫歐)

3 : 00am：Great Grimsby(71%脫歐)、Peterborough(63%脫歐)、BlackpoolSouth(68%脫歐)、Newcastle-under-Lyme(62%脫歐)、BishopAukland(60%脫歐)

在大選結果正式公布後，如果與出口民調的預測一致，保守黨以多數優勢當選，鮑里斯·約翰遜將於13日下午到白金漢宮覲見女王。女王將任命鮑里斯·約翰遜為英國首相，並要求他組建政府。按照慣例，鮑里斯將在覲見女王后回到唐寧街10號首相府門前發表就職演說。

獲得了議會的多數優勢後，鮑里斯將在議會通過此前與歐盟達成的脫歐協議，啟動《脫歐法案》。

按照鮑里斯·約翰遜的承諾，他將在1月31日脫歐最後期限前脫歐。然而這只是脫歐的第一階段，接下來英國還要與歐盟之間進行貿易談判。

按照與歐盟之間達成的協議，英國脫歐的過渡期將持續到2020年12月31日。過渡期期間，所有規則保持不變，以便歐盟與英國就未來關係展開談判。過渡期可延長，但期限只有1-2年，且獲得雙方同意。

鮑里斯首相在歐盟問題之外，還要處理蘇格蘭不斷要求二次獨立公投的棘手問題。

大選結果對英國意味著什麼？

如果與出口民調預測一致，工黨再一次輸掉大選，191個席位更是工黨自上世紀30年代以來在議會席位最少的一次，無疑是嚴重的歷史性失敗。

保守黨在此次大選中贏得的多數席位將結束英國議會無作為的局面。對鮑里斯·約翰遜而言，86席位的優勢幾乎賦予了他重新書寫英國政局的權利。然而，除了要完成英國脫歐，保守黨還為英國未來5年承諾了什麼？

稅收

保守黨承諾將國民保險的起征點從現在的£8,632提高至明年的£9,500。對於年收入2.5萬英鎊的人來說，可大約節約104鎊。

一旦當選執政，將不會提高個人所得稅和增值稅的稅率。

房產

根據非營利組織的預測，如果保守黨大獲全勝，全英房產均價到2030年時將上漲26%；如果工黨執政，將上漲19%。

承諾引入新的長期固定利率抵押貸款，只需5％首付；同時推出了「首套房」計劃，允許首次購房者在特定區域購買新房時享受30%優惠；計劃在未來5年建設100萬套新建住宅。

此外，保守黨計劃對非居民的海外買家征額外3%印花稅。

遺產稅

根據現行規定，明年起家庭的遺產稅免稅額將增加至100萬英鎊。目前，價值超過32.5萬鎊的房產徵收40%遺產稅，但夫妻二人的免稅額累加可至65萬鎊。

此外，非主要住所遺產稅的免稅額可以為每人增加額外15萬鎊。目前，這一政策讓擁有房產並有直接繼承人的業主享受一定的免稅額，個人15萬鎊，夫婦30萬鎊。2020年4月起，這一免稅額將分別提升至17.5萬鎊和35萬鎊。

家庭財務

計劃將目前30小時的免費托兒服務增加至每年38小時(3歲孩子)和48小時（4歲孩子）。這將為全職家長每年在每個孩子身上節約1410鎊。

養老金

保守黨表示將保留老年人冬季取暖費、老年公交卡和其他退休人員福利，以確保國家養老金至少增長2.5%。本周一，鮑里斯表示如果執政，將取消154.5英鎊的BBC執照費，並表示應為75歲以上老人免去這一費用。

保守黨還提出針對所謂的「Taper problem」進行解決，該問題導致醫生們因為擔心過高的養老金稅單而拒絕輪班，並承諾與醫療屆進行對話以解決這一問題。

承諾將全面評估如何對收入在1萬-1.25萬鎊的僱員，進行養老金自動補助的可能。

儲蓄

目前英國央行基準利率為0.75%。基準利率的調整將影響到儲戶的存款收益。

研究機構Capital Economics預測，如果保守黨獲勝，基準利率在2020年內將維持0.75%，並在2021年結束時升值1%。對2萬英鎊的存款而言，可多獲得50鎊利息。

投資

Interactive Investor的最新調查顯示，接近三分之四的投資者表示，保守黨獲勝對他們的資金最為有利。

Interactive Investor的股權戰略負責人Lee Wild表示，保守黨多數當選是金融城分析師唯一想看到的結果，他們認為這將為脫歐帶來確定性。

政策上的確定性會吸引海外投資，市場對選舉結果的任何積極反應都會說服國內投資者終於是時候行動了。

只要能夠避免無協議脫歐，股票和英鎊都會表現良好。

家庭賬單

如果鮑里斯連任，將撥款63億英鎊用來專門用於住房改善，包括改善低收入家庭的保溫隔熱效果或提高鍋爐效率。

保守黨表示他們將維持目前能源賬單的上限。這阻止了能源供應商向採用標準費率的住家收費超過每年£1,179。

上限每6個月進行一次審查，預計2020年中不會出現大幅調整。

交通

保障退休人員享受免費公交卡。目前倫敦和威爾士地區60歲以上的人可以享受該福利。

但保守黨未對火車票價做出任何承諾，表示計劃向北部和中部地區的鐵路網投入5億英鎊，以扭轉上世紀60年代削減計劃帶來的影響。

聖誕季大選

2019年的聖誕月大選成為1923年以來首次在12月舉行的全民大選。由於臨近聖誕節，從競選到投票的全過程都充滿了聖誕氛圍。

12日早晨起，就有選民通過社交網路曬出投票站門口排起的長龍。網友Matt發布推文稱在Battersea住了20年，從沒為了投票而排隊。

圖說：

在英國的大學，學生的投票率也是非常的高，隊伍甚至排到教學樓之外。

社交網路上還掀起了一股狗狗在投票站的接龍。
