老齡悲劇！日本88歲老婦殺死70歲女兒，留下遺書後自殺

2019-12-18 646條評論 4,868次閱讀 3767人點贊

據今日日本網站17日報道，周一早上，福岡市一名88歲的老婦坐在公園長椅上用刀自殺身亡，她70歲的女兒也被警方發現死在附近的一家養老院里，此前她曾長期卧床不起。

據富士電視台報道，警方認為這名88歲的老人先是殺死了自己的女兒，然後自殺。公園長椅上留有她的遺書和一把刀。

據警方稱，一名路人在早上7點半左右發現這名女性癱倒在西北區一個公園的長椅上，隨後立刻報了警。該女子因刀傷大量出血，被送到醫院後不久後去世。

警方前往該老人和她的女兒居住的養老院，發現她的女兒躺在地上，已經死於刺傷。她70歲的女兒在過去幾年一直卧床不起。

今年早些時候，日本也發生過謀殺後自殺的案件。據日本放送協會5月份報道，一名50多歲男性用菜刀連續刺傷了多名正在等待校車的小學生和大人。據日本警方透露，受害的學生當中已有一名女孩死亡，另有一名30多歲的男性也被確認死亡。行兇男子雖然在附近被確認了身份，但由於他自己用刀捅傷了脖子，也在送往醫院後死亡。
