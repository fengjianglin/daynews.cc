361公里的黑海海峽為何完全由土耳其控制，而不是被希臘控制？

2019-12-22 2,722條評論 4,750次閱讀 549人點贊

黑海作為一個深入歐亞大陸內部的陸間海，上萬年來，在人類的歷史中都是爭奪激烈的區域。因為黑海和地中海，地處歐亞非三大陸的交通樞紐的位置，無論是原始社會的人類遷徙，還是進入階級社會以後幾千年來的東西方的衝突，黑海和地中海都是一個重要的戰場區域和十字路口。而黑海只通過一條狹長的水道和地中海聯繫。這條水道雖然寬度變化很大，卻可以全程通行5萬噸以上的巨輪。因此也是黑海周邊的國家，走向外海遠洋的唯一通道。而黑海海峽本身，也是歐亞之間的天然地理分割。黑海海峽在誰家的手裡，其實對全球的地緣政治都有重大的影響。早在2500年前希臘和波斯爭霸的時代，歐亞之間誰強誰弱？首先就要看誰能控制黑海海峽。比如一開始波斯方面比較強大，控制了黑海海峽之後，就可以對整個希臘大舉入侵。開啟了亞洲和歐洲文明武力碰撞的先河。

而波斯人屢次入侵希臘沒有得手後，反倒被希臘方面崛起的馬其頓人反攻，一直打到了今天的南亞半島。可見從一開始，黑海海峽的控制權在誰手裡，就已經顯示出極強的重要性。而到了13世紀之後，一直到15世紀中期，歐洲人不但丟了黑海海峽的控制權，甚至連整個黑海周邊和巴爾幹的控制權基本全丟了。這就是奧斯曼土耳其的興起。這個自稱來自中亞的小族群，居然打遍中東和非洲無敵手。囊括了今天三大洲幾十個國家。而歐洲人在歷史上，從來沒有被長期入侵的如此之深。整個歐洲東南部，包括今天的匈牙利、烏克蘭、前南斯拉夫地區的大部分都成了奧斯曼的行省。作為歐洲文明發源地的希臘就更不用說了。奧斯曼鼎盛時期，全國共分為31個省，250個縣，總面積高達800萬平方公里以上。凡是挨著他的國家幾乎全部被征服。就算擴張成性的沙俄，經過12次戰爭，在奧斯曼人身上也沒找到太大的便宜。

黑海海峽控制權的徹底丟失，瀚海狼山認為對整個歐洲的影響是非常巨大的。直接導致了以義大利城市為核心的，環地中海古代商貿中心的衰落。因為傳統的東西方貿易通道完全被奧斯曼人控制，商旅來往被收重稅。而歐洲的貿易和繁榮中心被迫向西移動到了大西洋沿岸和英倫三島。大航海的原本目的，就是要尋找到亞洲的新航路，結果卻有一系列意外的地理大發現，也促進了現代工業文明的興起。而機器生產的出現和現代槍炮技術的發明，讓過去打群架很少輸的奧斯曼土耳其人，也迎來他的全面衰敗期。勢力範圍不斷縮小的同時，只能在西歐工業和軍事大國中被迫站隊。最早奧斯曼和英法合作，打贏了克里木戰爭，讓沙俄向南發展的企圖受挫。一戰以前，土耳其人本來想同英國結盟。但是一戰的突然爆發，英國人扣留了土耳其人訂購的2艘戰列艦緊急自用，既不立即還錢也不還艦，此事讓土耳其人大怒。結果德國人趁虛而入。土耳其加入協約國參加了一戰。而英法也很重視。在黑海海峽爆發了前所未有的海戰。

不過這次土耳其人站錯了隊。土耳其不但在1戰中陣亡了多達170萬人，而且戰後連小亞細亞的本土都快保不住了。英法和義大利，都想徹底把非JD教和非歐洲的土耳其人徹底的趕出歐洲。因此不但分別佔領了黑海海峽和靠近愛琴海的小亞細亞的大片西側土地。還支持已經復國的希臘人大舉向東行動。眼看整個黑海海峽，都要徹底被歐洲人收入囊中。

但是誰能想到此時土耳其冒出了一個凱末爾將軍，在新興蘇聯的支持下大舉反攻。不但把佔據小亞細亞的英法義大利的歐洲軍隊都趕走了。還收復了黑海海峽和歐洲東南部一角的3萬多平方公里的土地。這樣黑海海峽重新被土耳其人完全控制。1923年簽訂了洛桑條約。承認了此處的領土現狀一直到今天。不過蘇聯在1922年全力幫助土耳其人反攻，一定程度上，屬於為未來自找麻煩。因為沒想到土耳其二戰後又加入了北約。黑海海峽成了圍堵蘇聯海軍出入深海的重要關卡。如果當年不幫土耳其人，黑海海峽最後是同為DZ教的希臘人完全控制的話，前蘇聯和俄羅斯黑海艦隊面臨的戰略態勢明顯要更好一些。
