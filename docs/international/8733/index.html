40.6億盧比！新加坡勝科工業收購印度子公司剩餘5.95%股份

2019-12-09 1,372條評論 2,889次閱讀 721人點贊

國際太陽能光伏網訊：近日，新加坡綜合能源公司勝科工業集團(Sembcorp)簽署了一項協議，收購其印度子公司勝科能源印度有限公司(SEIL)當地合作夥伴Gayatri能源公司(Gayatri energy)剩餘5.95%的股份，收購價格為40.6億盧比，將通過內部資金和借款的組合來融資。

該公司聲明稱，在擬議收購完成後，勝科將成為SElL的唯一所有者。擬議中的收購將使勝科作為獨資擁有者，以便能靈活地評估可再生能源領域的市場機會，尋求合適的股權窗口來上市，或選擇其他資本回收。

據悉，此次收購預計將在今年年底前完成，前提是滿足某些先決條件，包括股東對GPL的批准。

此前，勝科於7月份宣布向SEIL注入約51.69億盧比的資金，用於發行2.75億股新股。

此次股權注入旨在支持印度可再生能源業務的增長，使得這家新加坡公司在SEIL的總持股比例從93.73%增至94.05%。

據關注，SEIL前身是印度熱力動力技術公司，領先的獨立電力生產商，專註於發展清潔能源組合。目前，該公司擁有超過4300MW多元化熱能和可再生能源資產組合，其太陽能資產位於拉賈斯坦邦和古吉拉特邦。

印度市場是勝科的第二大收入來源。2018年，該公司總收入為116.34億新元(5870億盧比)，其中1.685億新元(850億盧比)來自SEIL的電力業務。
