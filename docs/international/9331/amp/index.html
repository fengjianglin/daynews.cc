俄羅斯旅遊巧遇兩對新人，婚禮後的去處令人深思，我們又該怎麼辦

天天要聞
2019-12-04

小編前往俄羅斯遠東第三大城市布拉戈維申斯克市旅遊時，在大街上偶遇了兩對新婚新人，因不懂俄語的緣故，只能靠手語和面部表情同對方交流，雖略有尷尬，但他們的態度還是蠻熱情的，尤其是漂亮的俄羅斯新娘和美女們，更是非常熱情。圖為面對鏡頭，俄羅斯新娘開心的主動揮手示好。（圖\文 北疆記錄者）

據悉，俄羅斯《聯邦家庭法》修正案規定：在符合聯邦主體最高執行機關規定，得到父母、養父母或監護人的同意，存在特殊情況（已懷孕或已生子）的條件下，14至16周歲的公民可以結婚。而俄羅斯現行法律規定，公民滿18周歲才可以結婚。

圖為一對新人拍攝婚紗照時，旁邊的親朋好友們還端著杯子，因為不懂俄語的緣故，小編也沒弄懂他們這究竟是拍攝婚紗照，還是結婚呢。

在俄羅斯國內，很多女孩一旦過了25歲還沒有結婚，都被稱之為「老姑娘」了，父母也開始催婚了。圖為面對中國朋友的熱情，這對新人和他們的朋友們也都很高興。說實話，布拉戈維申斯克市絕大部分俄羅斯人對中國人的態度，還是非常友好又熱情的。

因為俄羅斯女多男少的緣故，男性在擇偶上有很多的優勢，很多男性也比較花心，畢竟他們可選擇的餘地太大了，分手後不愁找不到更漂亮的美女，反而是很多漂亮的俄羅斯女孩因為種種原因而找不到心儀的另一半。圖為帥氣的俄羅斯新郎正在親吻自己的新娘子。

「結婚要趁早」這句話在俄羅斯絕對好使。對女孩而言，結婚早的好處就是可以在自己最美的年齡找到滿意的另一半，不好的地方就是一旦看走眼，可能就會遭遇中年危機。其實，對於美女如雲的俄羅斯而言，很多女性到不了中年就遭遇婚姻危機了。也因此，一些性格不夠強勢或情商很高的女性，會有點「慣著」自己的另一半，多少有點委曲求全的意思，不過遇到渣的了，終究還是很難挽留住。圖為一對新人和親朋好友們開心合影。

在俄羅斯是沒有結婚要彩禮一說的。對此，有中國遊客調侃道：「先不說俄羅斯的風俗，就單憑男少女多，結婚新娘子要彩禮這一做法在俄羅斯就沒有市場。」圖為同行的葉老師和一對新人合影。

在俄羅斯，對於子女的婚姻大事，當父母的不是什麼都管，也不是撒手不管，他們的態度是開放而積極的，他們會給子女一些好的建議，但又尊重子女的選擇。不過，也有很多父母對子女的管教很嚴格，生怕子女選錯人婚姻不幸福。

圖為行走到布拉戈維申斯克市江邊不遠處的廣場，我們又巧遇了一對已舉辦完婚禮的俄羅斯新人。

值得一提的是，俄式婚禮是家庭儀式中重要的一種， 婚禮是由很多步驟組成，不同的地方區別非常大。不過，近年來越來越多的年輕人選擇更加簡單的結婚儀式。有的女孩沒有經過父母同意，回到家中拿到戶口本就和新郎領取結婚證了，然後兩家人邀請親朋好友舉辦一場簡單的宴會就算結婚了。甚至還有的新人邀請一些要好的親朋好友，在大街上喝點酒拍點照舉辦個簡單的儀式，也算結婚了。

圖為面對鏡頭，這對新人及親人都顯得很開心。

在俄羅斯遠東第三大城市布拉戈維申斯克市，有著一個不成文的風俗，那就是很多新人新婚結束後會前往江邊的「凱旋門」感恩。圖為這對新人在親朋好友的陪伴下，準備前往不遠處的「凱旋門」感恩。

據悉，布拉戈維申斯克市江邊的「凱旋門」是為紀念末代沙皇尼古拉二世1891年來到布市而修建的，因此在大門上方標有羅曼諾夫王朝的皇家徽記和1891字樣。布拉戈維申斯克原名叫「海蘭泡」，原屬中國，1858年《中俄璦琿條約》簽訂後被沙俄割占並改今名，在俄語里是「喜訊」的意思。「凱旋門」也是他們為當年取得勝利而建的標誌性建築，有著更多的歷史意義和精神力量。

如今，這座「凱旋門」不但成為布拉戈維申斯克市民眾開展愛國主義教育的地方，也成為很多新人婚宴結束後感恩的場所。尤其難得的是，俄羅斯人的很多愛國行為多是自發的，這就是最大的區別，也是最大的力量所在。

編後語：面對俄羅斯民眾這種從每個個體身上散發出來的愛國主義精神，我們又該怎麼辦？真的應該三思。其實，我們身邊的烈士墓和紀念塔也不少，但我們又有多少人把愛國主義精神融入了自己的生活和生命中呢？一個民族要想真正強大，必須得有強大的精神，也得足夠霸氣，哪怕有點匪氣！
