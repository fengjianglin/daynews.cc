金星榮獲世界經濟論壇水晶獎，感謝這份特別的禮物

2019-12-18 1,382條評論 3,699次閱讀 1517人點贊

第26屆「水晶獎」公布中國舞蹈家金星獲獎，表彰她在塑造具有包容性的文化規範方面起到的領導作用。

世界經濟論壇水晶獎專門表彰那些致力於改善世界狀況的傑出文化領袖。這些文化領袖為推動包容、可持續的變革作出了重大貢獻。與金星一同獲獎的有美國藝術家Theaster Gates，印度女演員Deepika Padukone和澳大利亞藝術家Lynette Wallworth。

第50屆世界經濟論壇年會將於下月21至24日在瑞士達沃斯舉行，聚焦主題「凝聚全球力量，實現可持續發展」 。水晶獎頒獎典禮將於20日舉行。

金星：「」本周末慶祝上海金星舞蹈團建團20周年紀念演出前，收到一份特別的禮物！」

這應該是金星第三次參加世界經濟論壇，其實早在2017年1月，金星就已經應邀參加在瑞士舉行的冬季達沃斯論壇。

2018年1月，金星第二次出現在冬季達沃斯的舞台上。她說，自己馬上就50歲了，但是仍然對世界充滿好奇，而且有勇氣應對挑戰。

金姐的選擇相信很多人都了解，這裡小編就不在贅述了，對於她的選擇，金星從未後悔，她說：「你可以不認可我的生活方式，存在方式，那是你的問題，不是我的問題。堅持自己的事情就可以了，時間會告訴你正確與否！」

今年是上海金星舞蹈團成立二十周年，期待精彩的演出！
