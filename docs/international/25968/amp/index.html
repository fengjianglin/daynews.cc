2019-12-21

石頭在我們的生活中

是最為常見的

平時總能看到幼兒園布置作業

會將石頭繪製成各色生動有趣的圖案

最近米醋發現一位

來自日本的寶藏石頭彩繪藝術家

Nakata Akie

她繪製的石頭作品看一眼便被震驚

更有人稱她為「石頭藝術家」

其實她本質的職業

是日本銀座靖山畫廊的畫師

更因其繪製的寫實石頭畫作名聲鵲起

作品一放到社交網上

便吸引了好幾十萬網友的關注

更有網友說道：

石頭居然被變成了小可愛！

簡直心都融化了~

Nakata Akie平時喜歡利用

從河岸邊撿的石頭

根據石頭特有的形態來描繪動物形象

經過她細膩富有靈動的繪畫技巧

每塊石頭都好似被施了魔法

變得生動形象栩栩如生

Akie Nakata 曾說道：
「當我把石頭放在掌心，

我會想到這顆石頭從山上一塊巨石

變成現在的大小和形狀所花費的時間，

我感覺到每一顆石頭都有生命的呼吸。」

Akie Nakata就好似

能感受到石頭的靈魂一般

無論石頭有縫隙還是完好無損

她都能將石頭繪製成屬於他們

特有氣質的動物

一個個活靈活現的小眼神里

都透露出Akie Nakata精湛的繪畫技藝

毛絨絨的倉鼠、皮糙肉厚的河馬、

滑溜質感的青蛙

彷彿石頭披上了動物的外衣一般

看了這麼多成品在來看看

前後作品草圖到成品的對比照

而最近聽說 Nakata Akie

還為這些石頭動物開辦了展覽

其實除了這些動物系列

她還繪製過其他創意石頭畫作一樣驚艷
