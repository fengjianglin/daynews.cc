開個鎖要價600歐，德國警方終於出手了

2019-12-25 1,936條評論 4,784次閱讀 1422人點贊

還記得之前我們說過的奧地利小夥子請人開鎖被刷走2800歐的事情嗎？鎖匠在騙到錢後人間蒸發，小夥子只能自認倒霉。不過在德國，警方就抓獲了一名無良鎖匠，前幾天法庭進行了宣判，不僅嚴罰1000歐，還被判入獄10個月。

圖說：無良鎖匠Daniel K.。圖片來源：歐洲時報德國版

23歲的Daniel K.近日被奧格斯堡地方法院判處10個月監禁，緩期執行，並處罰金1000歐元。據 歐洲時報德國版報道，在德國，開鎖糾紛時有發生，僅去年就有1000多起。這些開鎖糾紛事件中，金額最高的一張賬單竟高達3167歐。為了遏制此類現象，警方於去年開展執法行動，揪出了Daniel K.這名無良鎖匠。

事情追溯到去年7月，下決心整治亂象的奧格斯堡警方設置了一處「陷阱」，把一個房間直接鎖上，隨後撥打開鎖熱線。 一個小時後，Daniel K.出現了。在折騰了近2個小時，他才將門鎖打開，隨後向警方遞去一張高達634.86歐的賬單。其中列著： 189歐元「特殊案例出工費」、40歐元車費、94.5歐元夜間加班費以及每一刻鐘45歐元的工費。

換做其他熟練鎖匠，一般1分鐘就能搞定。這裡也不得不佩服下警方，硬是等了2個小時才收網下班。面對這個「挫貨」，警方說，專業開鎖的收費最多也就109歐，， Daniel K.是水平不行，要價還高 。

在法庭上，Daniel K.承認自己並沒有接受過專業的開鎖培訓，只是一位朋友曾經為他演示過「如何用一張卡片劃開房門」。至於他收走的超額費用，他辯解稱自己只能拿到三分之一，其餘的都要上交給熱線電話，以保證自己以後還能通過熱線接到活。

說到熱線電話，德國很多城市的居民都看到過印在小廣告上的「德國開鎖服務中心」服務電話。當你按照電話撥過去後，「德國開鎖服務中心」會熱情服務，然後聯繫合作鎖匠。這些鎖匠們往往技術不嫻熟，還會搞些小動作，比如把鎖徹底破壞，就能開出更高的賬單。最終，鎖匠會將收費金額的一大半返還「德國開鎖服務中心」。多年來，該中心通過這一手段賺取了上千萬歐元。後來，德國檢方以欺詐、逃稅等罪名起訴「德國開鎖服務中心」的兩名老闆。最終，58歲的Karl S.被判入獄6年半，39歲的Christian S.被判3年9個月。

新民晚報記者 梅瓔迪
