這族韓國人自稱中國後裔，拿出族譜證明，還到河南為祖立碑

2019-12-12 1,473條評論 2,439次閱讀 235人點贊

中華文明上下五千年，世界深受中國文化的影響。加上中國人口眾多，炎黃子孫也遍布全世界。然而，確有這樣一族韓國人，在尋根問祖時發現自己竟是明朝後裔。經他們多番考究，最終在河南的一個地方為其祖先立下了紀念碑。

　　

　　這支韓國人，就是韓國千姓族人。要說千姓，在國內也是比較罕見的。但在朝鮮半島，那可是名門望族。你要說趙錢孫李這樣的大姓，子孫眾多遍及國外也有可能。這麼小眾的姓，跑來認親會不會有假？

　　對此，韓國千姓宗親千龍鳳會長拿出了保存的《韓國千姓大同譜傳》，為大家揭開了秘密。

　　據記載，朝鮮半島史上就沒有「千」這個姓。唯一有記載的，就是河南登封潁陽的明朝將軍千萬里，他是落籍朝鮮半島的千姓第一人。

　　

　　那麼，千萬里將軍為什麼就落籍朝鮮半島了呢？這還有從一次朝鮮外族入侵戰爭說起。1592年，日本入侵朝鮮半島。當時朝鮮半島屬於中國的藩屬，朝鮮國王眼看打不過，趕緊派人來向萬曆皇帝求助。於是，就有了千萬里將軍出征朝鮮的事迹。

　　明朝大軍天兵神將，一到朝鮮就把日本倭寇打得倉皇而逃。戰爭結束後，千萬里將軍被封為了一等功臣，然後就定居在了朝鮮花山。歷經數百年，千氏子孫繁衍日盛。但他們一直以中國潁陽為本貫，奉千萬里的先祖千岩為始祖、世代發揚中原文化，也算是人才輩出。

　　

　　炎黃子孫都喜歡尋根問祖，韓國千氏一族也不例外。據說，自1987年起，韓國的千氏後裔都會跑到河南登封潁陽鎮尋根問祖。但是苦於地方志及實物欠缺，他們也一直沒有找到實際的先祖痕迹。

　　不過，天無絕人之路。一位叫宋植芝的老人，幫助了他們。宋植芝是登封潁陽鎮本地人，他了解到韓國千姓後裔急於尋根的事情後，主動把這活兒給攬下下來。雖然他已經81歲高齡，但他依然每天騎著車子在潁陽各個村子轉悠，終於有所發現。

　　

　　有一天，他在一名姓張的村民院子里發現了一塊碑刻，這塊碑上赫然寫著多個千氏的名字。他與這位張姓村民聊起這塊碑的來歷，才得知這裡曾經居住過姓千的人。千姓的祖先蹤跡，算是初步找到了。

　　隨後，經過潁陽鎮政府與韓國有關千氏後裔考證，最終確定了這個村子為韓國千姓祖居地。

　　

　　在得知尋找到自己的「根」後，韓國千氏後裔歡呼雀躍。當年，韓國千氏宗親研究會的會長千本立等一行13人來到了河南登封市潁陽鎮安寨村，築起了「根在潁陽」四字的千氏始祖紀念碑。

　　自此，這支流落韓國的明朝後裔，算是有「根」可尋，有祖可拜了。
