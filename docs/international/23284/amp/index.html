2019年最大的變化就是成了日本羅大力財團獎學生

天天要聞
2019-12-20

2019年四月起，我拿到了日本民間最大財團的獎學金。ロータリー米山奨學金，這個獎學金除了每個月給我14萬日元的經濟支持以外。還有很多參加活動的機會，讓我見識到了不一樣的人和不一樣的世界。接下來就讓我為大家介紹一下吧。

下圖是2019年兵庫地區的獎學生合照。學生們來自中國，韓國，越南，紐西蘭，蒙古，突尼西亞，印度，印度尼西亞等國。

在一次介紹會議上，我知道了以前拿過獎學金的一個留學生。在日本留學之後回到中國，回到了自己出生的地方，甘肅省的一個小地方從政。全身心投入到解決當地貧困事業的問題中，在45歲的時候，不幸去世，中央台還為他做了一個專題報道。原來在羅大力獎學生裡面還有這麼優秀的中國人。

我認識了財團的牙醫，山田醫生。他在上大學的時候，父親因為車禍意外去世，他得到社會的幫助才可以完成自己的學業。現在已經有自己的私人牙科醫院，有二十多個員工。為了回報社會，他一直不斷參加很多貢獻社會的活動。

下面這張圖是醫生和他們醫院的員工們。

這是牙科醫院的外景。我在這裡免費接受牙科的治療，他們採用的設備都是最先進的，服務非常友好，耐心。位於寶冢市，一個小地區就有這麼好的醫院，我很讚歎日本醫療水準之高，服務之好。

山田醫生還請我和我對象一起去京都的川床料理體驗了最正宗的日本料理。開車來回四個小時帶我們去了這個地方，從環境到美食都是無可挑剔的。非常感謝醫生。

這是川床料理的外部環境。

我們還體驗了和食的製作，整個過程非常有趣。這是在京都完成的。

我去了醫院兩次，醫生不收我錢，我覺得很不好意思，給醫生們做了餃子帶去，他們還特意拿過來讓我拍照留戀。

在參加羅大力財團活動的時候，我遇到了很多會長社長的夫人。她們平易近人給了我一次特別奇妙的體驗。

在活動中我還體驗了日式的太鼓。

我們還看到了蒙古留學生跳了正宗的蒙古舞蹈。

我和財團一起去了伊豆半島。住進了日式的房間，這裡拉開窗帘，窗外就是美景。但是價格真的很貴，是非常奇妙的體驗。到了日本一定要體驗一下日式的溫泉和日本料理哦。

每天都吃非常精緻的日本料理。

那天我生日，山田醫生又請我們到了高級的餐廳吃飯。這個餐廳位於大阪市中心，20層樓的位置。這裡可以看到大阪的夜景。看完了最日式的風景又看到了璀璨輝煌的夜景，有時候都不知道，到底那個才是真的日本。是傳統還是超現代？

那天財團的成員們為我一起過了生日。我自己都很少給自己過生日的。這裡面的人有內科醫院院長，婦產科醫院院長，建築師，會計師，稅理士，土地測量師。圖中和我一起拿著蛋糕的是大平 友規子，她是一個非常優秀的人。比我大十歲左右，二十多歲的時候母親發現有癌症，她照顧了十幾年自己的母親，也耽誤自己找對象。柔弱的外表之下，有一顆非常堅強，非常照顧別人的心。我很欣賞她。在羅大力財團里，這一年我體會了很多，我遇到很多人，我吃了很多好吃的。這是一種很難以用語言形容的感覺，能遇到你們真的很幸運。

希望大家，在異國他鄉漂泊努力的時候，不要放棄。

因為一定有人和你一樣遇到困難，但是他們選擇了堅持和克服。

有可能，就會有一片新的天地，一片更美好的天地在等著你。

這就是我的2019

和羅大力財團在一起的故事。

謝謝大家。
