墨西哥大毒梟的兒子被引渡到美國，兩個兄弟已被美國判刑

2019-12-22 627條評論 2,239次閱讀 342人點贊

墨西哥的毒梟因為向美國境內輸送毒品，因此被捕後一般都會被美國政府引渡回國受審。綜合英國路透社和《每日郵報》等媒體12月21日報道，墨西哥臭名昭著的販毒組織「錫那羅亞集團」創始人之一，大毒梟古斯曼的老搭檔伊斯梅爾·贊巴達·加西亞的第三個兒子日前被引渡到美國，準備在美國受審。

伊斯梅爾·贊巴達·因佩里亞爾

​​據媒體報道，當地時間12月19日晚間，大毒梟伊斯梅爾·贊巴達·加西亞的第三個兒子，35歲的伊斯梅爾·贊巴達·因佩里亞爾被引渡到美國加利福尼亞州的聖迭戈。伊斯梅爾·贊巴達·因佩里亞爾此次被引渡到美國，將面臨多起牽涉其父親的毒品走私案的審判。而目前伊斯梅爾·贊巴達·因佩里亞爾的父親，大毒梟伊斯梅爾·贊巴達·加西亞還在逃。

伊斯梅爾·贊巴達·因佩里亞爾和父親（左）

據媒體報道，因佩里亞爾簡直就是其父親的翻版，目前因佩里亞爾被指控向美國走私了大量的冰毒、可卡因、海洛因以及大麻等毒品。因佩里亞爾是伊斯梅爾·贊巴達·加西亞的第三個兒子，他是2014年被墨西哥警方逮捕，此後受到了美國加利福尼亞州當局的通緝。

而在因佩里亞爾之前，他的兩個兄弟早已經被引渡到美國並受到審判。因佩里亞爾的長兄文森特·贊巴達·涅布拉今年五月已經被美國法院判處入獄15年，目前正在一所美國聯邦監獄中服刑，因為早在2010年就被引渡，他預計將於2022年刑滿釋放。而·因佩里亞爾29歲的弟弟瑟拉芬·贊巴達·奧爾蒂斯也在去年被美國一家法院判處五年刑期，當時瑟拉芬·贊巴達·奧爾蒂斯還在法庭上辯解稱自己從小成長在父親所創立的販毒集團的陰影下，因此請求法院輕判，目前他已經提前出獄。

據媒體報道，伊斯梅爾·贊巴達·加西亞是墨西哥販毒組織「錫那羅亞集團」的創始人之一，自從前任古斯曼鋃鐺入獄後外界盛傳伊斯梅爾·贊巴達·加西亞已經接手了該集團。
