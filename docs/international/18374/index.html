民調：82.7%的民眾認為聲援塔納通將使泰國經濟發展難上加難

2019-12-16 345條評論 1,319次閱讀 367人點贊

泰國新泰日報社訊 泰媒報道，12月14日，泰國SUPER POLL研究中心負責人納帕敦公布了一組民眾對於「擔心政權鬥爭影響」看法的相關調查數據，該項調查主要以定量調研和定性調研法為主，通過Social Media Voice以及Traditional Voice網路平台分別對11703名以及1195名群眾進行了民意調查。

數據顯示，有85.9%的人認為政權鬥爭從未帶來良好的影響；有14.1%的人認為政權鬥爭為人民帶來了好處；有70.5%的人認為趕巴育下台義跑活動、集會示威等活動都只是為了政黨自身的利益，有29.5%的人認為這些活動是出於國家和人民的利益；有82.7%的人擔心這些政治活動將導致社會動亂，國家的經濟發展將因此再度下滑；有17.3%的人並不擔心這些政治動亂。

納帕敦教授的助理表示，Social Media Voice有800多萬網民關注，其中有10多萬民眾關注政治事件，而其中有53.8%的人表示不喜歡政斗，有46.2%的人表示能接受政斗。而數據調查顯示出，實際上泰國大部分人都在擔心政治鬥爭帶來的不良影響，而那些不擔心動亂帶來影響的人多是因為動亂並不會對他們造成損失，這部分人在動亂期間依然能夠過著舒適的生活，但那些窮苦人士則會非常擔心他們未來的生存狀況。那些網路調查之外的民眾也多是希望有一個和諧的社會環境，希望國民能夠團結起來。
