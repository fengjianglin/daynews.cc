英國男子網上買個聖誕老人回來發現有二層樓高，6個人才能拽住它

2019-12-20 627條評論 4,887次閱讀 1402人點贊

據《每日郵報》12月19日報道，英國默西塞德郡的32歲男子馬蒂·詹姆斯從網上買了一個充氣聖誕老人，他本來以為這個聖誕老人最多只有真人那麼高，沒想到寄來之後才發現，這個聖誕老人充好氣後竟然和他家房子一樣高，這樣一個龐然大物放在門外樂壞了詹姆斯的鄰居們。

快聖誕節了，詹姆斯決定給家裡增加點兒聖誕氣氛，於是上網花100英鎊買了個聖誕老人，他本來以為這個聖誕老人應該和真人差不多高。等到收到貨後，詹姆斯開始給聖誕老人充氣，不過他越充越感覺不對勁兒，等到充好氣詹姆斯才發現，這個聖誕老人已經和自家的房子一樣高了，竟然高達25英尺（約合7.6米）。

詹姆斯說雖然自己在小區里素來以常常做出古怪的事而出名，但是這次這個烏龍實在太好笑了，當鄰居們看到他家門口放著這麼大一個聖誕老人時，鄰居們都笑瘋了。這樣一個聖誕老人放在門外，確實是夠拉風的，但是當刮大風的時候就悲劇了。前幾天當地是大風天氣，一米八多、一身肌肉的詹姆斯算得上是身強力壯，但是颳風時他一個人根本拉不住這個聖誕老人，詹姆斯說：「我眼看著風越來越大，我不得不叫上5個鄰居，我們6個人一起用繩子才能綁住它，不然它早被風刮跑了。」

詹姆斯和鄰居們把聖誕老人用繩子綁在房子上，但是詹姆斯開始擔心如果風再刮大一點兒，說不定自家的房子可能都會被這個聖誕老人帶著一起颳走，詹姆斯說：「我們小區離海岸很近，前幾天我聽說有暴風雨，我怕海風太大，這個聖誕老人會帶著我家房子飛走，我就把它放了氣收回來了。不過我希望天氣轉好，這樣我才能夠儘快把它恢復原狀。」

詹姆斯說當時買這個聖誕老人的初衷是受到了萬聖節的啟示：「萬聖節的時候我買了一些充氣殭屍，覺得挺好玩兒，然後聖誕節我也想買個充氣聖誕老人，但沒想到這個這麼大，它站在我家房子門口，當我早上醒來往外看時，聖誕老人的頭會擋住我的窗戶，當我拉開窗帘打開窗戶時，一個大大的聖誕老人就會撲到臉上，我現在已經習慣了每天看到聖誕老人的頭和屁股在我窗外若隱若現。」
