美國同盟是一種主僕關係；所以中俄不會結盟，但關係比盟友更鐵

2019-12-21 158條評論 3,716次閱讀 2169人點贊

作者：刺客

普京在莫斯科召開2019年度大型記者招待會期間，稱「俄羅斯不打算與中國建立軍事同盟」。據《環球時報》最新報道，在談到中俄關係時，普京表示稱「至於聯盟，我們目前與中國沒有建立軍事聯盟，我們也不打算建立」。此外，他還表示，俄羅斯幫助中國建立導彈預警系統，也只是防禦性的。

普京之前在「瓦爾代」國際辯論俱樂部會議上曾聲稱「俄羅斯正在幫助中國建立導彈預警系統」。他指出，這將從根本上提升中國的國防能力，因為目前只有美國和俄羅斯擁有這種系統。

眾所周知，導彈預警系統是一種由太空偵察、預警衛星、地面和海基大型雷達、系統組成的反導預警網路，堪稱是一個國家導彈防禦體系的眼睛和大腦。沒有這種能力，任憑你的導彈再厲害，打出去也會因為無法捕捉到目標而抓瞎。

按照美國的標準，如果沒有到達結盟地步的國家，是不會將如此敏感的軍事情報進行分享的。值得一提的是，美國與盟友之間所謂的反導合作，實際上就是通過在盟友的境內部署、或者迫使盟友掏錢購買雷達、反導彈藥等軍事設備、為美國的國家和戰區導彈防禦系統服務。

由於導彈預警系統核心技術掌握在美國人手裡，各地區的盟友所部署或者購買的美國反導系統，設么時候發射、往什麼方向打，都得由美國人說了算。就連向日本這樣砸巨資採購美國反導系統的國家，除了有權利按下攔截彈發射按鈕外，其他的所有程序都必須由美軍來「代勞」。

換句話說，美國與盟友間的「反導合作」，實際上就是在確保美國絕對安全的前提下，在「方便」 的時候才為盟友提供保護。這一點與美國在盟友境內駐軍一樣，首要任務就是維持其全球戰略，「保護」盟友僅僅只是副業。

綜上所述，美國與盟友間的關係，其實就是一種極不平等的主僕關係，就是強者對弱者的欺凌，同時也是弱者對強者的屈服。這樣的同盟關係，根本就不適合用於中俄之間的合作關係。

反觀中俄在反導領域上的合作，俄羅斯幫助中國建立導彈預警系統，目的並未像美國對待盟友一樣，要通過利用或者控制中國的軍事資源，在反導領域上為俄羅斯服務，而是在中俄有著各自反導體系的前提下，利用俄羅斯在導彈預警領域上的技術優勢，幫助中國提高自身反導能力。

所以說，如果以美國的標準來衡量，中俄兩國在反導領域上如此深度的合作，實際上已經遠遠超越美式同盟關係了。至少，中俄在包括反導在內的所有領域上的合作，本身就是一種平等互利的合作關係，這種關係要遠比美國與盟友之間主僕關係更加靠譜。

實際上，俄羅斯幫助中國建立導彈預警系統，僅僅只是中俄之間密切軍事合作的一個縮影。近年來，中俄兩國在面對共同對手的背景下，通過定期舉行聯合軍演、中方軍隊參加俄羅斯高度機密戰略演習等舉措，本身就是在向外界展示中俄之間遠超結盟的合作關係。

更重要的是，在面對共同威脅的背景下，中俄兩國背靠背互相取暖，通過優勢互補的方式增強各自應對美國單邊主義的威脅，目前中國已經成為俄羅是第一大貿易夥伴國，今年兩國間貿易額有望達到1100億美元，更進一步顯示中俄之間牢不可破的全天候合作夥伴關係。

由此可見，普京做出「不打算與中國結盟」的表態，並不意味著中俄關係就達不到美國與盟友間關係的層次。相反，單從俄羅斯無條件幫助中國建立導彈預警系統的舉措來看，中俄兩國之間的合作關係，實際上早已經遠遠超越美國與盟友所建立在利益基礎上的合作關係。
