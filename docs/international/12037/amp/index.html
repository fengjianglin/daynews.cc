英國國家醫療服務體系受質疑 醫院超負荷運作 新生兒甚至睡地板

天天要聞
7 天 ago

英國《鏡報》12月10日報導：一個生病的嬰兒，要排隊11個小時才能看病。更有甚者，有些孩子沒有病床，睡在地板上。英國的國家醫療服務體系，到底怎麼了？

​

圖源：Mirror

科爾圖.貝碧科是2個孩子的媽媽。談到小女兒看病的經歷，她禁不住淚流滿面。

​

她的女兒布蘭卡，才3周大，被診斷出支氣管炎，整整花了一周時間才痊癒。

家裡決定要把這個就診的經歷曝光，因為事實背後隱藏著巨大的隱患。

​

科爾圖今年30歲，她丈夫本傑明36歲。

本傑明覺得醫生和護士表現無可挑剔，問題出在醫療服務體系。

「就像是一艘即將沉沒的大船，而船上的樂隊仍然繼續演奏。」

他們是11月22日下午5點45份帶著布蘭卡去醫院的，因為她一直咳嗽。

等待治療的11個小時，讓人精疲力竭。

​

晚上8點半的時候，他們被分派到所謂的「兒童部」，實際上就是又黑又亂的走廊。

等到布蘭卡接受治療的時候，已經是凌晨5點了。可氣的是，醫院辯稱是2點鐘。

​

科爾圖說「寶寶一直在咳嗽，我們要時不時的把她豎起來拍拍。」

「醫院的環境非常惡劣，空氣污濁，熱的要命。」

「護士一直說快了，但是我們一直等到半夜。後來想想都後怕，情況真的是很嚴重，她還那麼小。」

​

本傑明是一個軟體開發工程師，他說：「有孩子就睡在沙發上，凳子上。」

「有些是臨時用椅子拼湊起來的床，而父母就睡在旁邊的地板上，看著讓人心酸。」

「我們把事情說出來，是希望大家都知道我們到底經歷了什麼，情況真的不容樂觀。」

無獨有偶，9個月大的莉莉，在柴郡埃爾斯米爾港的切斯特伯爵夫人醫院整整等了6個小時。

​

本說：「過去的10年，我們的醫療服務體系碰到嚴重的挑戰。」

「我媽媽以前是個護士，我的兩個阿姨一直做護士到現在。」

「我的祖父祖母是從波蘭和奧地利逃難過來的。我祖父曾經是波蘭空軍，後來加入英國皇家空軍，參加了諾曼底戰役。」

「如果他們看到現在的醫療狀況，怕是難以瞑目。」

​

麗莎.格蘭特，是利茲教學醫院的護士長。

她說：「我們衷心向布蘭卡和她家人道歉。」

「以前兒童部最忙的時候每天有50名患者。而現在，每天病人超過了90位。醫院太忙了，我們的醫護人員還是盡心儘力。我很高興布蘭卡家庭有肯定我們的付出。」

「我們明年會擴建兒童部，增加10個病床和相應的空間。2025年計劃建成一座全新的兒童醫院。」

​

希望計劃能夠儘快落實，也給孩子們一個乾淨、舒適的就診環境。
