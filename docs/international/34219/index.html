世界美術中的奇葩——波斯細密畫

2019-12-25 2,035條評論 4,298次閱讀 330人點贊

作者：M·辰

#伊朗篇-23#

親愛的小夥伴們，感謝您一路支持、跟隨「走遍世界博物館」從文明古國系列的埃及、印度、希臘、墨西哥以及歐洲系列的俄羅斯一直走到亞洲系列的新加坡、印度尼西亞、土耳其等。現在，2019年重頭戲——伊朗篇正在進行中……速度圍觀啦！

（波斯細密畫，15世紀；伊朗國家博物館藏）

「奇葩」一詞近年來多有嘲諷之意，實際上其原本之意乃是讚美之詞：指奇特而美麗的花朵。用以比喻非同尋常的人或物，也常指優異、突出的文藝作品。今天咱們要說的波斯細密畫，就是世界美術史當中的一朵奇特又美麗的花朵，它是世界美術中的一個重要畫種，也是世界美術的重要組成部分。

（本文圖片由作者拍攝，版權所有）

（波斯手抄本中的細密畫，伊朗國家博物館藏）

什麼是細密畫？

簡單粗暴的解釋：畫畫所用的筆特別細，用這種筆畫出來的畫就叫細密畫（miniature）。

世界上最早的細密畫被發現於公元前10世紀埃及法老的陪葬物中。而波斯細密畫的起源大約是在公元3世紀中葉，它以插畫的形式出現在經典的抄本中。這些抄本或文稿大多是宗教類的內容，由於當時人們的文化水平不高，因此傳道者便以繪畫的形式來圖解其中的內容，久而久之，這種插畫的形式被傳承下來，成為了一種獨特的繪畫門類。之後，細密畫也常被用做書籍的封面以及盒子、鏡框等裝飾畫。

（波斯細密畫，16世紀；美國大都會博物館藏）

細密畫的特點

1.尺幅小。細密畫作為文稿、抄本中的插圖屬於小型繪畫作品，它的尺幅最大不超過書頁的大小。以古代書籍來說，細密畫的大小最大也就相當於現在的32開本，而實際上很多作品都非常小。

2.題材特色。細密畫的題材主要是以神話故事、民間傳說以及宮廷生活為主要內容。

3.色澤明麗。細密畫的最大特點是顏色亮麗鮮艷，這可能是因為其顏料取自於天然礦物成分，因而能夠保持色彩鮮艷經久彌新。

（前圖局部）

4. 畫風明快。細密畫的畫風通常都非常生動明快，它們或表現日常生活場景，或展現美麗田園風情，再或是描繪了奢華富麗的皇宮庭院等，其畫面生動細膩，情趣盎然。

5.藝術特色。細密畫儘管尺幅不大，但作品卻是短小精湛。其作為插畫的特點，奠定了它在整體風格上追求裝飾性和平面性的效果。

總體來說，細密畫更像是一道溫潤淳美的下午茶，雖不夠濃烈但卻蘊含著精緻與細膩。

波斯細密畫

波斯細密畫雖然起源於公元3世紀，但真正發展起來並成為一種畫派則是公元15-16世紀的事了。

公元1501年，薩法維王朝的締造者伊斯邁爾建都大不里士。這位慣於徵戰的帝王不僅驍勇善戰，卻也是一位藝術愛好者與贊助者。定都大不里士後，伊斯邁爾廣羅藝術人才並聘請他們作為宮廷畫師留在宮中作畫。在帝王的熱心贊助與扶持下，波斯細密畫在這一時期得到了蓬勃發展，湧現出眾多繪畫大家。

繼伊斯邁爾之後登上王位的沙·塔赫馬斯普也是一位藝術骨灰粉，他對文化藝術的熱愛比其前任有過之而無不及。沙·塔赫馬斯普在位52年，造就了波斯細密畫半個世紀的黃金時代，這一時期波斯細密畫成為波斯文化圈的流行畫種。

（印度細密畫，16世紀；印度國家博物館藏）

在去伊朗旅行之前，辰館曾於八年前到訪過印度國家博物館，在那裡辰館看到了眾多的印度細密畫。據當時所查資料顯示，印度細密畫源於波斯細密畫的傳入……此番伊朗之行，終於圓滿了卻了該問題的疑惑。

當年，1544年印度莫卧兒王朝的皇帝胡馬雍流亡到波斯國（伊朗舊稱），住在薩法維王朝的皇宮裡，並因此結識了宮廷畫家賽義德·阿里。胡馬雍非常欣賞阿里的繪畫才能，他把阿里召至身邊，隨侍左右。1555年，胡馬雍恢復莫卧兒帝國，重返德里，他邀請阿里前往德里並任命其為宮廷圖書館館長，負責編繪《埃米爾·哈姆扎傳統故事集》中的插畫。阿里與幾十名印度畫家一起工作了數年，繪製了2400多幅插圖……

正因如此，波斯細密畫得以在印度傳播開來，從而推動了印度細密畫的繁榮。

（波斯細密畫，15世紀；伊朗國家博物館藏）

波斯細密畫，作為世界美術史上的一朵奇葩，至今保持著旺盛的藝術生命力。
