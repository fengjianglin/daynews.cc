新加坡：領銜生態建築，精美絕倫

天天要聞
2019-12-14

隨著人們的環保意識越來越強，建築物也開始注重環保方面的事情，一些新的詞語，比如「綠色建築、生態建築」慢慢出現在了人們面前。除了採用更多的高科技之外，人們始終沒有忘記最原始的美好，大自然給我們帶來的感官享受是其他東西無法比擬的，建築與自然的結合，將是接下來的發展趨向。

新加坡一直號稱花園國家、花園城市，它地處熱帶地區，氣候濕潤，國土面積也非常的小，把整個國家打造成花園似乎也並不難，秉承著建設花園城市的理念，新加坡打造了諸多綠色生態建築。小編為大家分享點評幾個較為經典的，不得不說，它們真的很美。

皮克林賓樂雅酒店

從它不凡的外形就可以看出，這是一座高端星級酒店。建築沒有追求高度，外觀較為複雜，整體來看用了一種視覺懸空的設計，剛柔並濟、時尚現代。

Erwin Soo

建築物基本上分為了三個部分，上部分的主體為玻璃幕牆方盒子，被大量的圓柱支撐了起來，下部分圍繞柱子，採用了柔美的曲線造型，中間部分幾乎是掏空的。樓頂庭院現在已經很常見了，但是建築物中部掏空的庭院，還是挺特別的。

作為一座綠色生態建築，跟綠植的結合是它的招牌，可以看得出來，經過了精心的布局安排，設計的非常巧妙，跟建築物每一部分都融合得很好，充滿細節且不突兀，不過，這應該會招來不少的昆蟲類的生物吧，不知道是一個優點還是缺點呢？

Oasia酒店

這座建築採用了難度非常大的紅綠配色，要知道，這種搭配稍有不慎可能就會很鄉土，不過還好，憑藉著前衛的造型，整體還挺好看的，好像叢林小屋的放大版。可以看得出來，建築物的中間部分也進行了掏空，形成了一個獨特的空間，仔細觀察你還會發現，它外立面覆蓋的紅色材料是有縫隙的，這樣可以讓藤類植物更容易地攀爬生長。

Simon_sees

跟上一座相比，它採用的是一種比較放任綠植生長的方式，顯得更為接近自然，只是小編在想，這種方式萬一控制不住，會不會產生植物把整座建築完全包起來的情況，採光便會成為大問題，所以應該還是會有人定期打理。跟綠植這麼一結合，感覺整座大樓都在生長，充滿了生機與活力。

濱海灣花園

室內景觀

濱海灣花園的名氣就非常大了，相信很多朋友都見過，而且，只要到新加坡旅遊的人，基本都會來這裡觀光。

這個地方可以說展現了新加坡花園城市理念的精髓，最顯眼的當然是那幾棵碩大的天空樹，整體形狀模仿了大樹，但看得出來，加入了很多主觀想法，底部爬滿了各種植物，冠頂為枝杈造型，很有設計感，夜晚搭配上絢麗的燈光，美輪美奐。

遠遠看去，好似科幻大片里的某些場景，可謂近觀精美，遠觀大氣磅礴，當然，這些大樹只是觀光設施，並不是真正住人的樓房。

SOTA新加坡藝術學院

超前的設計和想法出現在藝術學院里，其實也挺正常的，不瞞大家說，小編也來自於藝術院校，感覺搞藝術的，做出什麼來都不誇張。

這座大樓的綠植也是根據樓體形狀，經過了特意設計，使它們長在了指定的位置，起到了很強的裝飾作用，在這樣一座大樓里上課，應該會激發出很多的靈感吧。

南洋理工大學

Gramicidin

不知道為什麼，一看到這個名字，總感覺是中國南方的某所大學，當然，人家南洋理工可是一所世界一流大學，在這裡上學的，都是棟樑之才。

這所大學裡有多座建築都很漂亮，例如那個屋頂長滿草的，還有這個一節一節的，極具張力，留給了人們很多想像的空間，跟綠植結合是一方面，其實建築物本身就非常棒，讓人忍不住駐足觀賞。感覺大學這種地方，就應該前衛一些才對。

樟宜機場

機場可以說是一個城市的門面，很多人對某座城市的第一印象就是機場，樟宜機場就是非常注重和討好遊客的那種，極為重視景觀的營造，給人一種溫馨自然之感。

著名的星耀樟宜景觀蜚聲世界，新加坡不愧是旅遊城市、名不虛傳，一下飛機就先給你來個景點。星耀樟宜確實值得一看，巨大的圓形玻璃穹頂，非常具有想像力的室內瀑布，精心栽種的植物森林，絕對可以給你帶來獨特的體驗，它會讓你懂得，生活沒必要一直匆匆忙忙，適時地也可以慢下腳步。當然，您如果真的是去機場趕飛機，那可千萬不要被它迷住，一不小心忘了登機可就悲劇了。

結語：新加坡倡導綠色生態建築跟它的城市定位有關，其實也得益於當地的氣候，說實在的，熱帶多雨的地方，隨便弄弄那植物都得瘋長，這要是在北方，還真是有點難度。好了，看了這麼多，你的內心是否有種去看看衝動呢？
