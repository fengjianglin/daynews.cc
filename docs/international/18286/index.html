中國在海外最大電站正式投產，印網友：來幫我們修基建吧

2019-12-17 672條評論 2,216次閱讀 307人點贊

國華印尼爪哇電廠

說起基礎建設，就不得不提到被稱為「基建狂魔」的中國，隨著科技的發展，我國在基礎建設這一領域的技術越來越高，逐漸走向世界前列，漸漸的我國基建開始走出國門，開始幫助各國修建基礎設施。

據媒體報道，中國企業在海外建設的國華印尼爪哇的7號項目1號機組正式投產，這是我國企業在海外建設的單機容量最大、擁有自主知識產權的火電機組，同時也是印度尼西亞店裡建設史上裝機容量最大、技術最先進的環保型電站。

印尼爪哇電廠1號機組正式投產

據了解，爪哇7號項目應用的海水淡化技術是我國國華電力的自主知識產權，這項技術從開始的基礎原理研究做起，一直到最後的工程應用完全是我國自主研發出來的，這個由我國建設的爪哇7號發電站創下了很多世界之最，比如：擁有目前世界體積最大的褐煤鍋爐、最大容量的發動機等，然而這已經不是我國第一次幫助其他國家建設發電站了。

眾所周知，伊拉克是中東地區一個石油產量豐富的國家，正是由於這一點，導致伊拉克飽受戰爭之苦，由於經過了戰爭的洗禮，當地的基礎設施重建變得異常艱難，其中，用電荒就是他們面臨的一項重要難題，伊拉克之前每天斷電時間高達16個小時，就連他們的首都巴格達曾經也有半個夏季都處於停電狀態。為此，伊拉克方面選擇求助作為「基建狂魔」的我國，我國出於大國風範，自然而然地接受了他們的請求。

​我國幫助伊拉克修建的發電廠

隨後，我國在當地的底格里斯河畔建起一座發電廠，經過中國企業的幫助後，伊拉克很大程度的緩解了他們用電荒的問題，並且這次幫助伊拉克修建的發電站的項目建設中，全部採用了中國設備和中國標準。但是有人會感到疑問，我國為什麼要大力幫助這些小的國家建立火電站這一類的基礎設施呢？

為此有專家分析說：我國幫助別國首先是出於友好互助的目的，還有一點就是，我國幫助別的國家建設基礎設施，是一種自信的表現，因為首先我們相信自己的技術，才會去選擇幫助別的國家修建基礎建設，這樣也是為了能夠吸引更多的國家與我們合作，讓雙方達到一種互惠共贏的結果，同時也不用擔心說，這些小國萬一沒有資金償還怎麼辦，就拿伊拉克來說，他們經過長年的戰爭，肯定是沒有多少資金用來修建基礎設施，但是他們的石油豐富，石油也可以作為一種支付手段，所以說這種問題是不用擔心的。

總的來說，我國「基建狂魔」的名片已經發往全球，這個稱號也實至名歸，同時我們也希望有越來越多的國家與我們合作，這次在印度尼西亞建造的發電站不得不說也是一項偉大的壯舉，這次又創輝煌，也贏得了國外網友的一致好評，為此有印度網友說到：來幫我們修基建吧。（曲奇）
