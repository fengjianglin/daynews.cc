利比亞國民軍宣布挫敗民族團結政府的進攻計劃

2019-12-22 1,365條評論 2,275次閱讀 1195人點贊

本欄目為中東研究通訊《周觀中東》系列欄目之《北非地區一周觀察》。

埃及

據阿拉比亞電視台12月22日報道，埃及議會將批准部門改組。據埃及消息人士表示，「議會將批准大約10個部門的變動，其中最主要的是社會團結部、農業部、工業部、司法部、旅遊部和議會事務部」。並有消息表示，旅遊部和文物部將合併，並成立新聞部。

埃及先前的內閣，圖源：阿拉比亞電視台

消息人士表示，埃及計劃、監督和行政改革部部長哈勒·賽義德將擔任經濟事務副總理一職，此外還將設立安全事務副總理和服務事務總理的職位。

消息人士證實，國防、外交和內政部長將繼續存在，以及水資源和灌溉部，教育部和移民部部長也將繼續行使其職能。

蘇丹

據《阿拉伯耶路撒冷報》12月22日報道，從本月19日開始，蘇丹人舉行了推翻奧馬爾·巴希爾政權的慶祝活動，該前總統曾統治蘇丹長達三十年。

慶祝活動，圖源：阿拉伯耶路撒冷報

在這些慶祝活動之前，在蘇丹的許多地區已發生象徵性的遊行，如森納爾省的Myerno地區，去年12月6日革命時的第一次示威活動發生於此。

示威活動於12月9日在多地舉行，該日期是蘇丹1955年在議會內部宣布獨立的日子，由此為革命增添了一分民族色彩。這些示威者的規模數以萬計，他們高呼口號，要求對革命中死去的示威者報仇，並重新呼喊起那句最著名的「自由，和平與正義」。

社交媒體上也能看見許多慶祝行動，其中有按從去年一直到今年年初順序發布的視頻和照片，記錄了民眾運動的不同階段。街道上掛滿了國旗，人們悼念犧牲者，官方媒體也通過播放特別節目的形式表示慶祝，國家電視台則放映了有關青年在革命中發揮作用的影片。

利比亞

據《艾恩新聞》（Al-ain）12月21日報道，利比亞國民軍周六宣布，其武裝部隊挫敗了民族團結政府手下的武裝民兵的進攻計劃，該計劃企圖襲擊塔魯納市以東的達沃恩地區。

利比亞國民軍，圖源：《艾恩新聞》

利比亞武裝部隊總司令的一份聲明說：「民族團結政府最近在米蘇拉塔的軍事基地的襲擊事件中受挫，並造成重大損失，故指示民兵襲擊塔魯納市以東的達沃恩地區，而僅靠該城市的一些力量就把它打敗。這次失敗的襲擊表明，民族團結政府未受原則或國家利益約束，並有奪取年輕人的生命的想法。」利比亞的塔魯納地區在的黎波里戰役中具有戰略性的地理位置和決定性的作用。

另有來自哈里發·哈夫塔爾武裝部隊的軍事消息表明，該部隊宣布在一連串的空襲事件後將向利比亞首都以東的馬斯拉塔邁進，該舉動旨在追擊從塔魯納和馬斯拉塔撤離的民族團結政府武裝。

摩洛哥

據《摩洛哥國門》12月19日報道，第14屆法國摩洛哥高級會議於周四在巴黎舉行，該會議由摩洛哥首相薩阿德丁·奧斯曼尼與法國總理愛德華·菲利普共同主持。

法摩高級會議，圖源：《摩洛哥國門》

高級會議舉行前，摩洛哥首相與法國總理進行了雙邊會談，旨在增進兩國之間的多方位特殊的戰略夥伴關係，並加強雙邊政治對話。該會議在法摩友好的框架內舉行，評估了兩國在政治、經濟、文化、安全等各領域的合作，並將簽署涵蓋各個合作領域的一系列雙邊協定。

在該會議結束時，兩國領導人發表聯合聲明，重申兩國政府將努力鼓勵並加強經濟交流，重申「建立框架」的必要性，並表示今年是企業之間促進和恢復交流與投資的有利年份。」

兩國還稱讚了法國開發署的工作。自2017年以來，法國開發署承諾每年提供約為4億歐元投入發展，僅在2019年簽署的協議框架內就承諾提供5.9億歐元。這表明雙方渴望與開發署加強在貸款方面的合作關係，這些貸款將被用於支持教育領域、職業培訓、衛生領域的改革，社會保障，以及振興農村地區的經濟。
