為什麼要在東京過聖誕節？因為是日劇啊

天天要聞
2019-12-23

在東京，12月的大事件，就是聖誕燈會了。每年的冬季檔的日劇里，都能看見它們的身影。

よみうりランド

關東地區第一燈會地標，橫跨東京都稻城市和神奈川縣川崎市兩大區域。遊樂場一般的氛圍，令它成為日劇日影的最愛，多年前木村拓哉和香里奈主演的富士月九《Priceless》，最終話就是在這裡取景的。

東京迪斯尼海洋

東京迪斯尼的聖誕燈會從11月8號就開始預熱了，持續到12月25日。在日本年輕人之中，海洋世界的聖誕之夜要比迪斯尼世界要更浪漫一點點。

橫濱紅磚倉庫

西洋風情的紅磚倉庫，聖誕之間的一個月里，夜晚裝點了燈飾。同時還會舉辦德國傳統的聖誕市集，除了巨大的聖誕樹帶來的浪漫的氛圍之外，還能吃到正宗的德國啤酒和香腸，度過美食之夜。

東京中城

亮燈時間從11月14日持續到12月25日，東京市中心的人氣約會之地，充滿了夢幻的童話風情。和其他大型燈會相比，它規模較小，但擁有自己獨創的燈光技術，也令人驚艷。這裡也設置了可以合照的聖誕樹。

惠比壽花園

雖然燈光秀從11月2日延續到來年的2月24日，整個冬天都看得到，但是巨大的聖誕樹過了聖誕節就會被撤走，還是要趁早看。日劇里的有錢人都是在這裡約會的，不妨來感受一下它金光閃閃的氣息。

Caretta汐留

每年的燈會都有主題，今年靈感來自迪斯尼的最新電影《阿拉丁》，現場搭配以原聲音樂，每隔15分鐘會有一場特別演出，一定能將你帶回童年的夢境中。

東京德國村

關東三大燈光秀之一，使用的燈泡數量高達300萬個。每年主題各有不同，今年已經延續到第14屆了，和往年不同的是，第一次出現了中國「自貢燈彩」的主題區域，能看到很多熟悉的中國工藝元素。

江之島

江之島燈塔，年輕的戀人們必去的約會之地，拍過數不計數的戀愛日劇。亮起燈來的江之島，可是被稱為「湘南的寶石」的。

大理石村Lock Heart城

在群馬縣重現的歐洲中世紀古城，能夠在日本遇到一個純正的英國風情的小城，也是蠻有趣的。這裡面還有一個「世界聖誕老人博物館」，形態長相各異的聖誕老公公，聚集了超過1100個。

足利花卉公園

以紫藤花而聞名的公園，其實擁有各種各樣的珍奇花卉，這裡舉辦的燈光秀「光之花的庭園」，就以花為主題來設計，已經被認定為日本的夜景遺產，也是日本公認的三大燈光秀之一。

相模湖度假村 Pleasure Forest

大約600萬個燈泡組成的大型燈光秀，規模盛大而壯觀，搭乘期間限定的熱氣球升上天空，珍貴的俯瞰景象定是獨一無二的體驗。

東京王子公園鐵塔酒店

沒人能將東京塔私有，但是在這裡卻可以。不僅鐵塔近在遲尺，酒店裡還推出了各種聖誕節限定套餐，看著這一天的東京塔亮燈熄燈，日劇里最浪漫也不過如此了。
