中國國家知識產權局與歐洲專利局加強PCT框架內合作

2019-12-04 90條評論 1,875次閱讀 725人點贊

中國國家知識產權局和歐洲專利局在近日舉辦的中歐第十三次局長會議上，就指定歐洲專利局為向中國國家知識產權局提出的PCT國際申請的國際檢索單位達成重要共識，藉此加強兩局在《專利合作條約》（PCT）框架內的雙邊合作。該合作有望於2020年實施，並針對以英文提交的國際申請進行兩年的試點。

據了解，中歐兩局互為全面戰略夥伴，30多年來，雙方保持著長期緊密的合作關係，所取得的豐碩成果為中歐兩地的用戶和公眾帶來了福祉。此次，雙方就向中國國家知識產權局提出的PCT國際申請可指定歐洲專利局作為國際檢索單位達成了共識，將為中國乃至全球的PCT申請人提供更多便利，也是中國在更大力度加強知識產權保護國際合作方面邁出的堅實一步。

此次合作將為中國PCT申請人優化其國際專利戰略提供更多選擇。選擇歐洲專利局作為國際檢索單位，可為有意願加快申請速度的中國PCT申請人節省近一年的時間，其PCT國際申請可提早進入歐洲階段，無需歐洲補充檢索即可直接進行案卷審查。

此外，在近日於江蘇南京舉辦的第四屆紫金知識產權國際峰會上，中國國家知識產權局和歐洲專利局聯合發布了中歐計算機實施發明專利審查研究報告，主要涉及相關領域專利審查實踐的異同等內容。近年來，計算機實施發明（CII）領域的技術發展迅猛，CII相關專利申請顯著增加。隨著CII技術越來越多地滲透到其他創新領域，各國知識產權機構及申請人都將面臨新的挑戰。為應對這些挑戰，中國國家知識產權局與歐洲專利局著手開展了CII領域的聯合研究工作。報告的研究成果將為申請人在中國和歐洲提交CII領域專利申請時提供實踐指導。（知識產權報　董妍）
