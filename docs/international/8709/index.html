這才是哈利波特鐵杆粉！印度軍警竹竿當駿馬，這兩國卻遠比它搞笑

2019-12-09 1,026條評論 2,918次閱讀 1375人點贊

《哈利波特》系列電影中，騎著魔法掃帚在天上飛來飛去的一幕也曾吸引了不少兒童競相模仿，而把一根竹竿當成戰馬，一邊跑一邊喊著駕駕，也是不少人的童年回憶。然而誰也不會想到，這種常人看來只有幾歲小孩才會做的兒戲，居然被印度軍警直接照搬在訓練任務中！

印度軍警把竹竿當馬騎，一邊高喊一面沖向假想敵

據印度NDTV電台11月13日報道，印度北方的菲羅扎巴德邦當時舉行了一次特殊的治安演習，一群軍警得到通知，他們要騎馬衝散試圖引發混亂的人群。然而可笑的是，這些軍警居然連一匹可供騎乘的馬都沒有，每個人都只能手握一根長竹竿，將其想像成駿馬，然後一邊高喊著一邊快速沖向扮演暴亂者的人群。印度軍警表示，這次演習雖然有其規範和標準，但由於物質條件的限制，所以參演軍警只能用這樣讓人看起來匪夷所思的方式完成訓練。這段十幾秒鐘的視頻頓時帶來了幾十萬的觀看量，不難想像這種和小孩過家家一樣的軍警訓練會讓多少人捧腹大笑。

一百多年過去了，印度讓義和團現象變相復活

不少觀看者戲稱，這些印度軍警可能都是哈利波特的鐵杆粉絲，他們自以為自己擁有了超自然魔法。某種程度這也是清末義和團的重現，義和團大師兄們施法時往往先把黃紙燒成灰，將其撒入所謂神水中，喝掉所謂神水後大吼刀槍不入，其師弟們此時還用未開刃的大刀長矛配合其所謂神靈上身。此外，請神上身的義和團師兄們還會往地上撒黃豆，並坐在板凳上將板凳視為戰馬。一百多年過去了，號稱自身軍事力量即將趕超中國的印度不但連訓練用戰馬都湊不出來，甚至還表現出義和團式的荒唐，結合其日常的吹牛充大，只能為世人增添笑柄。然而，有兩個國家在荒唐方面比印度卻有過之無不及。

劫持事件中一些菲律賓警察用手比作持槍的動作參與戰鬥

2010年8月23日，菲律賓首都馬尼拉發生襲擊者劫持人質事件。巡警出身的門多薩單槍匹馬持槍劫持一輛搭載21名乘客的大巴車。按說這種孤狼式襲擊發生在一國首都，必然會被該國軍警和特戰隊迅速彈壓解決。然而菲律賓特警官兵們的表現卻讓人大跌眼鏡，門多薩在此期間多次出現在車門處，本就犯了大忌，因為這種毫無遮擋和防禦動作的行為幾乎就是給對方狙擊手創造良好靶標。可就是這種比蠢較量中，菲律賓特警最終技高一籌，他們在現場連基本的狙擊意識都沒有，更無專業狙擊槍械，白白浪費了戰機。這還不算，據媒體報道一些參與圍堵任務的菲律賓軍警居然沒有手槍，只能用手做出持槍的樣子，然後嘴裡發出類似於開槍的聲音！如果說印度警察把竹竿當馬騎還可以歸結為日常訓練圖省事，但菲律賓軍警在實戰中卻依舊以過家家的態度應對，絕對暴露該國武裝力量嚴重缺乏基本武備和訓練的問題。

德軍用掃帚冒充機槍，搞笑程度不亞於印度和菲律賓

可能會有人認為，菲律賓國小兵弱，印度則一貫以不靠譜的戰備著稱，但一個素以軍事技術先進且歷史上戰功赫赫的國家卻在荒唐程度上不逞多讓。這個國家就是德國，在2014年9月的北約軍事演習中，德軍士兵將一支掃帚塗成黑色，並將其安放在一台拳擊手裝甲車上。德國國防部卻表示他們無法解釋此事，而這一情況也是由某士兵告知督查人員的。這台部署掃帚機槍的裝甲車是一輛機動指揮車，無需深入一線戰鬥，所以它的車載武器如此糊弄貌似也說得過去。然而，深入調查卻發現德軍可悲的一幕，涉事部隊371裝甲擲彈兵營是北約快速反應部隊之一，按照規定如果烏克蘭戰爭需要介入，那麼該部就很可能和俄軍攤牌。但這支特戰任務的部隊不但缺乏通用機槍和夜視儀等武器和設備，甚至41%的士兵沒有手槍！事後有人評價，讓武器都配不全的部隊去面對俄軍，就和讓人空手和獅子搏鬥一樣荒唐。而近年來，德軍的戰機、坦克乃至戰艦和潛艇的出勤率都已降至新低，問題嚴重時甚至海軍全部潛艇同時趴窩，所以掃帚當機槍這種荒唐之事足以讓德軍在搞笑方面躋身印度、菲律賓之列。
