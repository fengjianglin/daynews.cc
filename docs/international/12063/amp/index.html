阿拉伯半島早期墓葬，出土埃及“神眼”，考學學家：信仰複雜

天天要聞
2019-12-11

阿曼是個位於阿拉伯半島東南部的國家，2012年，一支義大利考古隊在此意外發現了鐵器時代墓葬，隨後，他們與阿曼當地的考古學家一同對墓葬進行了發掘，而這一挖就是7年，今年又有了新發現，考古學家們在一個大型墓葬中挖出了12個人和1隻「眼睛」。

墓葬位於阿曼最北部穆桑代姆省的迪巴，此前經過多年發掘的一個大型墓葬出土了豐富的文物，除了金飾、貝殼飾品、青銅器、寶石、陶器等生活用品，還有不少箭頭、匕首這樣的武器。

這個墓葬的使用年代，從公元前1600年一直延續到了公元300年，屬於集體埋葬，一個墓葬坑中有大量人骨遺骸，隨葬品也遍布其中。截止目前，這個墓葬中已經發現了至少188個人的遺骸，超過3500件文物。

此次找到12個人和1隻「眼睛」的是另一個墓葬，雖然埋葬的人數上不及之前的墓，但是從已清理出的3000多件隨葬品的規格看，這次發現的墓葬等級明顯更高。

墓葬呈現橢圓形，隨葬品中的陶器大部分都上有釉，青銅器有精美的動物裝飾，金銀飾品也表現出外來的風格，顯然是從鄰近的其他文明中進口而來，其中最引人矚目的當屬一個「荷魯斯之眼」。

荷魯斯之眼其實是埃及法老的守護神鷹頭神的眼睛，在古埃及有著特殊的含義，既代表法老神聖的權力，也蘊含著復活和重生的神奇力量。在古埃及葬禮中，常常會用荷魯斯之眼隨葬，它通常會被製作成手鐲、項鏈佩戴，當作死者的護身符。

這是阿曼發現的第一個荷魯斯之眼，也是當地已知的第二種隨葬用的護身符。此前在迪巴遺址，考古人員曾經找到另一種護身符，其上刻有「Gula」的字樣，據說「Gula」是美索不達米亞文明中掌管生死的神靈，有著治癒生者、保護死者的力量。

同一個遺址，同樣的集體墓葬，有著不同來源的隨葬品和兩種來自不同文明的護身符，這也表明阿曼所在的阿拉伯半島東南部，曾經在人群交流和貿易往來中有過獨特的作用。當然，除了隨葬品，考古學家對一個墓中同時出現的12個人很感興趣，不知道當初集體葬禮意味著什麼。
