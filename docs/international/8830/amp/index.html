日本交通事故頻發，原因中國需警惕

天天要聞
1 週 ago

日本以擁有世界上最高效，最全面的公共交通系統之一而聞名，但它也是一個駕駛員和愛車人員最多的國家，路上行駛著近8000萬輛汽車。現在，作為地球上最老的國家之一，有五分之一的公民年齡在70歲或以上，它面臨著一個敏感的問題：隨著人們的年齡增長，如何減少交通事故的發生？

這是一個重要的問題：去年在日本，由75歲或75歲以上駕駛員引起的致命交通事故比例從2008年的8.7％上升到14.8％。儘管去年日本的整體交通死亡人數是1948年以來最低的，但65歲的老人佔總死亡人數的56％，創歷史新高。

根據日本政府在6月份的一份報告，與75歲及以上的駕駛員相比，2018年造成的致命事故數量是年輕駕駛員的兩倍多。更具體地說，超過75歲的人在道路上每10萬人中有8.2人死亡，這是「 74歲以下年輕人造成的死亡人數的2.4倍 」。

涉及年紀較大的駕駛員的致命事故繼續成為全國新聞。目前在日本，年齡超過75歲的老人必須每三年進行一次認知測試，才能成功更新其駕照，並且在夏季提出的提案旨在讓長者只能駕駛帶有先進自動剎車系統的汽車。

儘管採取了單方面的安全措施，但最大的問題仍然是，沒有任何一種萬能的解決方案可以確保老年人安全駕駛。那是因為並非每個年齡或人生階段的人都會一樣。「您不能說在某個人的年齡X點，他們可能會經歷特定的下降，」世界衛生組織殘疾與康復協調員Alana Officer說。

另外，一些研究表明，最年輕的駕駛員比最年長的駕駛員更危險；日本政府在6月發布的同一份報告中發現，年齡在16至19歲之間的駕駛員是最危險的人群，每100,000名有執照的駕駛員造成11.1致命事故。更重要的是，老年人死於車禍的風險更高，也可能是由於與年齡相關的對醫療併發症的敏感性增加。

軍官說，如果您制定的政策嚴格限制了年齡-也許以公共安全的名義為所有年齡以上的人撤銷了執照-您將面臨歧視性歧視年齡的風險。

那麼可以做什麼，日本在做什麼呢？答案還不是很明確，但是謹慎的政策和新技術的結合可能為未來指明了道路。

一名年長的司機撞上古迹後在神戶造成多人受傷。在日本，老年司機引起的類似事故也很普遍（來源：Alamy）

需要獨立和尊重

在任何國家，與年長的家庭成員就放棄駕駛進行討論可能都是困難的。「重要的是，採取步驟支持公共安全，同時有禮貌地對待年長的駕駛員，」華盛頓特區非營利組織人口參考局的高級研究人員Toshiko Kaneda說，該組織分析了人口趨勢和統計數據。

在日本中南部靜岡縣清水市，豐田汽車質量顧問兼汽車銷售員牧野友美（Tomomi Makino）親眼目睹了生活方式的改變對年長駕駛員的直接傷害。她在博客中介紹了她與年長客戶之間的經驗，並說更多的人選擇選擇放棄他們的許可證。發生這種情況時，汽車經銷商會來到他們家，將汽車開回經銷商處再賣掉。

老年人可以從政府那裡得到一些好處，例如打車和乘車折扣，但是告別駕駛可能會讓人情緒激動。牧野回憶起一位顧客的電話，該顧客之所以打電話是因為他放棄了駕照，需要她把車拿走。他在電話中對她說：「在傷害別人之前，我應該停下來。」 當牧野露面出現時，她說那人哭了。

她說：「許多人輕易地討論了老年駕駛員應歸還駕駛執照，但我們不應忘記這些人的感受。」 他們的汽車和駕駛「已成為他們生活的重要組成部分」。

根據國家警察局的統計分析， 2017年，日本有超過40萬名老年人放棄了執照，這是自1998年實施該計劃以來的最高數字。但是，日本國家老年醫學與老年醫學中心主席荒井英典（Hidenori Arai）認為，放棄執照發放的趨勢並不是「好趨勢」。在這個估計患有痴呆症的人數為500萬人的國家，他贊成定期對年長的駕駛員進行認知測試，並贊成對駕駛技能進行再培訓以「延長駕駛時間」。

失去執照可能會打擊農村地區的老年人，農村地區的老年人口最大，公共交通受到限制，尤其是困難重重。「沒有汽車，他們就無法生存，」 Arai說。「他們不能去購物或見朋友。為了享受生活，汽車是必要的。即使[雖然]一些老年人意識到他們的駕駛技能受損，但他們仍需要繼續駕駛汽車以維持他們的日常生活。」

隨著年長的駕駛員在日本道路上造成越來越多的事故，老年人駕駛學校如雨後春筍般湧現，以對駕駛員進行再培訓

通過創新妥協

那麼，為減少事故數量又使老年人保持活躍而計劃的計劃是什麼？可以自己開車的計程車。

近年來，日本汽車和科技公司加大了在這一領域的研究力度，特別側重於幫助老年人。（他們還希望這些機器人計程車中的一些能夠在2020年夏季奧運會之前及時在東京運動場館之間穿梭於外國遊客。）自動駕駛汽車的道路測試始於2016年，當時無人駕駛的普銳斯在道路上纏繞在海邊小鎮和其他農村地區。

長期以來，長者一直是無人駕駛汽車的主要受眾之一。在世界其他地方，一家名為Voyage的初創公司籌集了數千萬美元，用於測試退休社區的艦隊，例如佛羅里達州一個40平方英里，12.5萬居民的社區。

但是，我們距離自動駕駛汽車成為現實且隨時可用的解決方案還差幾年。在短期內，除了那些定期的認知測驗外，警察部門還試圖為那些確實認知功能或駕駛技能受損的人推出「有限駕駛執照」 。它們可以駕駛，但只能駕駛帶有特殊內置安全支持的某些類型的汽車：例如，自動制動系統。（在年長的駕駛員中，致命事故的常見原因是制動踏板和油門混合在一起。）

在短期內，公司正在努力定製新車以滿足老年人的需求。本月，豐田汽車推出了一款微型的兩人座電動汽車，專為短途駕駛而設計，最高時速為60公里/小時，專門針對希望保持活躍狀態​​的老年駕車者。

日本已將加速自動駕駛汽車送入道路列為優先事項，這可以為該國越來越多的老年駕駛員提供幫助（來源：Getty

所有人的解決方案

多年來，日本一直要求在初學者和老年駕駛員的汽車上都貼上特殊的保險杠標籤，以便與其他駕駛者識別。這些標籤實質上是對路上其他人的提醒。

儘管在某些情況下這些信號可能是有用的信號，但世衛組織Alana官員表示，政府必須謹慎標記年齡段的任何一方，以應對交通禍害，而無需提供任何解決方案。她說，否則，這是歧視性法律的一個滑坡。

為了實現有效的變革，幹事呼應了Arai提出的一項更全面的計劃，該計劃使年長的駕駛員能夠適應新的生活階段並保持更長的駕駛時間。她指出，應繼續進行駕駛教育，並提供職業治療方法，以幫助因骨關節炎等疾病導致頭部動作受限的駕駛員更好地左右檢查，以及輔助技術或汽車改裝。

她說：「如果建立人們的認知能力，就很有可能擴展到安全駕駛，」她說，就像維護人的多任務能力一樣，這是駕駛中使用的關鍵技能。「我認為我們要做的很多事情不是將老年人歸入這些類別，而是，而是要看待衰老過程，這需要特定的政策以使人們繼續做出選擇來安全駕駛嗎？ 」

無論您身在世界何處，對可能結束自己的駕駛職業生涯達成妥協都可能會難以接受。這是很難進行的調整，尤其是如果沒有足夠的政策來幫助您進行調整時。然而，在日本，越來越多的人口將面對這一問題。

對於某些人來說，可能是懷舊之情。如今的老年人見證了汽車工業的興起，並且可能已成為1960年代和70年代人們獲得駕駛執照的早期浪潮的一部分。」人口參考局的金田說。「從某些方面來說，這是時代的終結。」
