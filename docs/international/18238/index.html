中國已繪製一個宏偉藍圖，巴西人：是時候向中國學習，去實現夢想

2019-12-17 1,171條評論 2,187次閱讀 345人點贊

巴西海軍航母

巴西的海軍正在遭到嚴峻的問題，巴西海軍陸戰學校教授卡布拉爾非常直接的指出，是時候向中國學習了，中國人已經繪製並實施了一個宏偉藍圖，巴西海軍應該遵循這樣的藍圖去實現自己的夢想。巴西海軍正在同德國組成一個合作關係，但是遭到了質疑。巴西海軍將要簽署的4艘軍艦的合同，因為招標不規範的問題受到檢視。

巴西海軍需要什麼樣的軍艦，巴西海軍工業又該如何發展，成為關注的焦點。卡布拉爾教授指出，新的4艘巴西海軍護衛艦將在伊塔亞伊船廠建造，第一艘國產化率為31.6%，其他三艘的國產化率為41%。對方稱將轉讓技術，但是這個比例顯然不夠。

巴西船廠

卡布拉爾教授稱，上世紀70到80年代，巴西海軍工業可以為自己建造船隻，但是這些技能在上世紀90年代開始就逐漸喪失了。這期間有很多種原因，包括採購了其他國家退役的航母，巴西海軍實際上都沒有怎麼使用過，就宣布再次退役。實際上，巴西海軍及海軍工業在這些採購活動中什麼都沒有獲得。

現在巴西應該要重新審視自己的海軍工業，當前4艘護衛艦的建造是一個機會。海軍工業未來的軍艦、潛艇軍應該國產化。巴西應該重新獲得這些國防技術，建造更大、更複雜的軍艦。巴西是全球第八大經濟體，海軍需要巡邏至少450萬平方公裏海洋，這些都促使巴西需要發展自己的海軍工業。

中國海軍

是時候向中國學習了，中國海軍及海軍工業的已經繪製並實施了一個宏偉的藍圖，從輕型護衛艦開始3000噸的軍艦到8萬噸的航母，中國海軍工業從建造簡單的軍艦中脫穎而出，已經完善了整個海軍工業體系，巴西海軍應該遵循我們的夥伴中國這樣的藍圖去實現自己的海軍夢想。

巴西似乎更加具有優勢，德國人在同巴西海軍軍艦建造的協議中，承諾了轉讓技術，巴西擁有這樣的機會，完全可以完成自己的海軍工業重建。巴西的不少軍事分析人員也支持這樣的觀點，並且認為，巴西重建海軍工業比中國更有優勢，那就是可以從全球先進的海軍國家引進更多的技術。事實上也是如此，就如當前，德國承諾在聯合建造巴西四艘護衛艦時提供技術轉讓，這在中國海軍工業中不可能出現這樣的現象。

巴西海軍巴納伊巴號

但是分析指出，這是巴西的優勢，但是可能恰恰是最致命的劣勢。因為本國軍工行業發展的初期，建造出的軍艦未必有直接採購自發達國家的軍艦先進，海軍能否忍受並包容這種現象？

在印度，已經進行的十多年的武器裝備國產化的進程中，就遇到這樣的現象。印度的海軍和空軍總是在抱怨他們的國產化裝備，以至於建造出來了遲遲無法入役使用，最終還是從其他國家採購武器裝備。印度的LCA光輝戰機就是鮮明的例子，在印度空軍中，這款戰機一直沒有地位，在印度國防部的強迫下，空軍雖然採購了數十架，但是也沒有任何積極性。這對印度航空工業幫助微乎其微。

印度航空工業工廠

另外，在武器裝備國產化的進程中，會觸及很多國際軍火商的利益，他們會不遺餘力地摧毀該國的本土軍工業。還是以印度為例，在上世紀本來還算不錯的印度航空工業，幾乎沒有任何發展，技術還穩定地保持在上世紀的水平。雖然俄羅斯和美國的軍火商都承諾並實際轉讓了不少技術給他們。

在海軍工業上，同樣美俄法德都承諾給印度海軍工業技術轉讓，但是實際並不理想，印度船廠看起來能夠建造小型軍艦也能建造航母，甚至可以建造核潛艇，但是所有人都明白，這些技術依舊停留在上世紀末的水平。因此，雖然中國繪製並實施了一個偉大的藍圖，但是並不是誰都可以學習的，這一點真的很難。（再言）
