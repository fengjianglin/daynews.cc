美國海軍隊戰勝陸軍隊，贏得第120屆“美國比賽”

2019-12-17 447條評論 1,462次閱讀 56人點贊

美國第120屆「陸軍-海軍」橄欖球比賽已於上周六在費城結束，海軍學院的「海軍中尉」隊以31:7戰勝西點軍校「黑騎士」隊，結束了最近三年的三連敗。

這項比賽起源於1890年，是美國大學橄欖球中最傳統、最悠久的賽事。也體現了美國武裝部隊在軍種之間競爭的精神，被稱為「美國比賽」。

「海軍-空軍（空軍學院獵鷹隊）」比賽在10月的第一個周六舉行；「陸軍-空軍」比賽在11月的第一個周六舉行；「陸軍-海軍」比賽在12月的第二個周六舉行。三場比賽的積分最高者獲得「總司令杯」，由美國總統在白宮頒發。不過，最具影響力的還是這場「海陸大戰」。
