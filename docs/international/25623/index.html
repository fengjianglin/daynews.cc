宏都拉斯監獄爆發黑幫衝突　18死16傷

2019-12-22 1,493條評論 4,082次閱讀 2067人點贊

為遏制監獄血腥暴力，中美洲國家宏都拉斯17日宣布由安全部隊接管監獄。但據知情人士披露，截至20日安全部隊仍未能接管全部監獄，北部地區一所監獄當天爆發幫派衝突，造成至少18人死亡、16人受傷。

再現監獄暴力

宏都拉斯北部港口城市特拉一所監獄20日爆發血腥幫派衝突，至少18名在押人員死亡，16人受傷。政府部門隨後說，安全部隊已進入監獄恢復秩序。

這是宏都拉斯近期發生的最嚴重監獄暴力事件之一。

由於近期頻繁發生監獄謀殺和暴力事件，宏都拉斯總統胡安·奧蘭多·埃爾南德斯本月17日宣布全國監獄進入緊急狀態，由安全部隊接管監獄，為期6個月。

一名不願意在媒體報道中公開姓名的高級軍官20日告訴路透社，安全部隊還沒來得及正式接管位於特拉的這所監獄，獄中便爆發黑幫衝突。

拉美地區經常發生監獄暴力以及越獄事件。巴拿馬首都附近拉荷伊塔監獄17日突發槍戰，至少12人死亡、13人受傷，警方繳獲至少5把手槍和3支長管槍械，尚未查明這些槍械如何流入監獄。

墨西哥中部莫雷洛斯州一所監獄今年10月30日發生幫派衝突，致6名囚犯死亡、2名囚犯受傷，墨西哥當局出動大約350名警察和士兵平息衝突。

美國難逃干係?

宏都拉斯監獄早就人滿為患，街頭幫派競相爭奪監獄內「控制權」，導致獄中暴力鬥毆乃至槍戰屢見不鮮。

在各大幫派中，「野蠻薩爾瓦多人」臭名昭著。這一跨國黑幫起源於美國加利福尼亞州洛杉磯地區，上世紀90年代遭打壓後，眾多成員被驅逐到宏都拉斯、薩爾瓦多、瓜地馬拉三個中美洲國家，並在那裡重新壯大，使得中美洲北部成為全球最暴力危險的地區之一。

美國政府部門20日宣布，在美國紐約、馬里蘭州巴爾的摩等地和薩爾瓦多抓獲超過230名疑似「野蠻薩爾瓦多人」幫派分子。其中，多達96人逮捕自美國紐約長島，這一地區被視為該黑幫的一個大本營。

美國總統唐納德·特朗普20日晚些時候經由社交媒體「推特」發帖，慶祝這一打擊成果。

墨西哥同樣飽受監獄暴力和黑幫衝突等問題困擾。墨西哥方面估計，犯罪團伙所持武器超過八成由美國流入。墨西哥總統安德烈斯·曼努埃爾·洛佩斯·奧夫拉多爾今年10月與特朗普通電話時提及此事，呼籲美國「斷供」。
