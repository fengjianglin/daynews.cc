潘基文：聯合國實現全球發展願景離不開中國的全力支持

2019-12-23 115條評論 1,064次閱讀 75人點贊

中新社北京12月23日電 題：潘基文：聯合國實現全球發展願景離不開中國的全力支持

中新社記者 肖欣 范思憶

聯合國前秘書長潘基文近日在北京接受中新社記者專訪指出，中國的發展繁榮、積極承擔國際責任參與全球治理，以及對多邊主義的堅定支持，在不同的歷史時期都極大地支持和推動了聯合國的全球發展願景的實現。

資料圖：聯合國前秘書長潘基文。中新社記者 曾靜寧 攝

過去數十年中，潘基文多次到訪中國，見證了中國的發展變化。在擔任聯合國秘書長的十年間，與中國政府和中方同事們愉快的合作經歷令他記憶猶新。他告訴記者，「我深知中國為全球可持續發展所做的重要貢獻。」他甚至認為，「如果沒有中國的全力支持，我作為聯合國秘書長可能很難完成預期的全球發展願景」。

潘基文指出，中國自加入聯合國以來一直扮演著重要角色。作為聯合國安理會常任理事國，中國承擔了維護國際和平與安全的重要責任；在維護世界多邊主義的過程中，中國積极參与並支持全球事務；作為迅速增長的經濟體，中國與其他國家保持著密切的合作，表明了共同發展繁榮的意願。

「消除貧困和飢餓」是聯合國2030可持續發展目標中的重要一項，潘基文表示，中國的貢獻首先來自於其自身實現了脫貧致富。自新中國成立以來的70年間，中國通過非凡的努力和繁榮成就贏得了全世界的高度認可和尊重，特別是自改革開放以來，僅用短短40年，從一個貧窮國家一躍成為世界第二大經濟體。早在2010年，中國便已實現聯合國千年發展目標中的減貧等多項目標，比千年發展目標計劃的2015年提前了5年。現在，中國又提出在2020年全面建成小康社會，這比聯合國2030可持續發展目標設定的2030年又提前了10年。他相信，中國將如期實現這一充滿雄心的目標，屆時將成為全球範圍內最先達成聯合國可持續發展目標的國家。

基於全球可持續發展的視角，潘基文更為看重的是，中國為其他國家提供了「可借鑒的發展樣本」。他回憶說，「我曾會晤許多非洲國家的領導人，他們均高度讚賞中國一直以來關注非洲的發展，提供了持續不斷的支持與合作，特別是近年來通過『一帶一路』倡議的合作。例如在『一帶一路』倡議框架下成立亞洲基礎設施投資銀行，支持了很多發展中國家的基礎設施建設。」

潘基文尤為讚賞的是，富裕起來的中國在參與國際治理中所提出的方案，始終強調維護多邊合作、實現共同發展。他指出，過去百年間，在全世界人民的共同努力下，全球科技領域取得革命性的創新，很多國家在此期間贏得獨立，人民生活處境得到極大改善。但當今世界仍有很多人尚未實現溫飽、基本權利仍未得到尊重和保障，因此，聯合國自誕生伊始明確的宗旨和原則並未過時，多邊主義仍富有指導意義。此時維護並尊重全球規則、維護多邊主義、倡導多邊合作顯得更為重要。

他對於近期單邊主義、民粹主義抬頭的浪潮回應稱，「基於多邊主義的框架，世界上大大小小的國家都生活在和平之中，創造了很多繁榮，我們都為之受益。在維護多邊主義的過程中，沒有哪個國家能夠單獨去做這件事」。

潘基文認為，近年來中國領導人提出的「一帶一路」倡議、「人類命運共同體」等發展理念，極大地支持了聯合國為實現全球共同繁榮、和諧、可持續發展所制定的目標。他對中國國家主席習近平2015年9月在聯合國總部出席第七十屆聯合國大會一般性辯論時發表的講話印象深刻，為講話中提出的「構建以合作共贏為核心的新型國際關係，打造人類命運共同體」的願景感到振奮，也聽到來自各界的讚譽認同之聲。

展望未來，潘基文稱，「衷心希望在未來的很長一段時間裡，中國能在政治、經濟、科技、文化等各領域成為全球治理中一支重要的領導力量」，期待「我們將見到更多的發展、更多的革新、更多的合作」。(完)
