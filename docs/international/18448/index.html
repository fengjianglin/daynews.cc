伊朗控制現金流：規定個人24小時現金交易上限

2019-12-16 1,888條評論 3,487次閱讀 1021人點贊

每天關注壹行伊朗 了解中東實事與動態

伊朗《德黑蘭時報》12月11日報道編譯。伊央行行長赫馬提在央行定期會議上與各銀行總經理舉行會議。會議探討了管理現金流、加快完成為所有銀行賬戶實施一次性密碼(OTP)計劃及高效信用證操作計劃(GAM)等問題。赫強調了央行控制和管理現金流的堅定決心。並稱，從周三開始，伊朗個人24小時現金交易上限為10億里亞爾(約合2.38萬美元)。

伊前8個月貿易逆差為13億美元

伊朗《梅赫爾通訊社》12月11日報道。伊朗海關總署負責人邁赫迪（Mehdi Mir-Ashrafi）表示，伊歷今年前8個月(3月21日至11月22日)，伊朗的出口總值為270億美元，進口總值為283億美元（約有2210萬噸產品進口到伊朗），貿易差額為負13億美元。

伊中小企業本年已出口15億美元商品

伊朗《德黑蘭時報》12月10日報道，伊朗小企業和產業園區組織（ISIPO）官員賈萬瑪爾德對媒體表示，本伊歷年前8個月（3月21日至11月23日）伊朗中小企業已出口15億美元商品，這些商品主要是由產業園區內的590家中小企業完成的。賈強調，一旦中小企業獲得足夠的支持，他們的出口能力至少還可以翻兩番。為了達到該目標，其組織採取了多項措施，包括認證專業的貿易代理、組建出口企業聯盟等方式，以支持中小企業。據ISIPO介紹，目前伊朗共有中小企業43650家，分布於全國800多個產業園區，其中33800個企業經營活動較為活躍。

伊能源部長稱未來三個月內將啟動9個電力項目

伊朗《德黑蘭時報》12月10日報道，伊朗能源部長阿爾達卡尼向媒體透露，未來三個月內（至本伊歷年底）將有9個主要電力項目在不同省份啟動。這些項目均屬於「水電伊朗」計劃的一部分，該計劃旨在向每個省平均投資10萬億里亞爾（約2.38億美元）用於電力項目建設。阿還表示，在下財年預算中能源部將獲得90萬億里亞爾（約21.4億美元）的資金安排，較本年有所增加。此外，從俄羅斯貸款的50億美元中也已有22億美元用於水電項目建設，剩餘28億美元則將用於製造2000節地鐵車廂、建設錫斯坦-俾路支斯坦至南呼羅珊省的鐵路、升級阿瓦士電站以及在伊朗西部建設一座水電站。
