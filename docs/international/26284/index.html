日本天價草莓1顆900元 什麼草莓這麼貴（圖）中國公司一次買下108顆 稱“買回去觀賞”

2019-12-20 1,701條評論 3,509次閱讀 2494人點贊

中國小康網 草莓我們都吃過，但是最貴的草莓能賣到多貴呢？日媒16日報道，奈良縣產的一盒3種共30顆草莓的高級禮盒拍出150萬日元(約合9萬7千人民幣)的高價，一顆草莓值約900人民幣。

工作人員展示其中一種草莓

據日本媒體12月17日報道，這家奈良縣的農場以培養優質草莓著稱，這次在大阪推出的草莓禮盒裡裝有他們最新培育出的草莓品種。

禮盒中有三種新品種的草莓，名字也是取得十分高雅：分別是「古都華」、「淡雪」、「白珍珠」。每個品種36顆，禮盒一共裝有108顆草莓。

草莓一共108顆一盒

在當天的市場拍賣中，一家中國香港公司以150萬日元的高價將這盒草莓拍下，並立刻空運送往香港銷售。該農場負責人表示這批草莓是優中選優，從200公斤草莓中才選出這樣5公斤的優質草莓銷售。

不過這家農場負責人也表示，這些草莓的銷售對象也都是海外富人和精英階層，所以這個售價也不算離譜。

買家表示這草莓就值這個價，要拿回去觀賞。
