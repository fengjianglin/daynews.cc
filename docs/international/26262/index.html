商務部：中歐已交換關於投資市場准入的清單改進出價

2019-12-20 901條評論 2,885次閱讀 619人點贊

　　中新經緯客戶端12月19日電 19日，商務部新聞發言人高峰表示，第25輪中歐投資協定談判正在舉行，雙方交換了關於投資市場准入的清單改進出價，談判取得積極進展。

　　商務部新聞發言人高峰。來源：商務部官網

　　在19日的商務部例行新聞發布會上，有記者提問，注意到目前中國和歐盟正在就投資協定展開新一輪談判。請問本輪談判涉及哪些重要議題？雙方是否在一些議題上已經取得了一致？

　　高峰表示，12月16日到19日，第25輪中歐投資協定談判正在布魯塞爾舉行。雙方繼續圍繞協定文本展開談判，並交換了關於投資市場准入的清單改進出價，談判取得積極進展。

　　高峰提到，為落實中歐雙方領導人關於加快推進談判的指示，雙方經貿團隊今年共舉行了6輪正式談判，三次會間會，文本和清單談判均取得了積極進展。

　　高峰表示，中方願與歐方共同努力，按照雙方領導人達成的重要共識，進一步加大談判投入，保持談判的良好勢頭，相向而行，爭取早日達成一致，使協定早日惠及雙方的企業和人民。(中新經緯APP)
