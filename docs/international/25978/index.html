華為，韓國在華為構建自主供應鏈幫助很大，今年採購額13兆韓元

2019-12-21 2,460條評論 3,973次閱讀 3025人點贊

-華為在韓記者座談會
-預計明年上半年在韓設立R&D中心，內部檢討中

據消息，華為今年在韓國採購額達到13兆韓元。華為5G移動通信有關業務正在正式化，明年也將加大在韓國方面的採購和投資。

華為公司全球媒體傳播副總裁Karl Song於20日在韓國首爾舉辦的記者座談會中稱「因美問題，華為正經歷著前所未有的試煉，在這過程中，華為建立了自主的供應鏈，其中韓國起到了很重要的作用」。

Karl Song強調「華為這段時間在全球簽訂了60多個5G商用化合同，供應基站40萬台」，「在韓國為LG U+供應了1萬8000台基站，幫助了LG U+在5G領域領先」。

預計繼續推進明年上半年在韓國構建R&D中心。

華為韓國CEO孟少雲提到「在韓國設立R&D中心的意志很強」，「目前內部檢討中，之後如果有確切消息將進行公開」。

華為將在5G 28GHz 帶寬、 SA設備上進行努力，供LG U+等韓國運營商選擇。

孟少雲強調「韓國客戶要求的技術水平很高，會供應高品質的產品來獲得客戶信任」，「28GHz帶寬設備·SA設備，也會根據客戶要求，隨時供應」。

只是，華為在韓國的採購額去年是106美元，約12兆韓元，與今年相比並沒有大幅度的增加。主要是與三星電子和SK海力士的交易。
