俄稱大量武器被仿製，卻照樣大肆出售，越南人：俄製造商很奇怪

天天要聞
3 天 ago

蘇-35戰機

近幾年來，俄羅斯武器裝備製造逐漸復甦，軍火交易所佔的市場份額有所增長，最典型的就是S-400防空導彈系統，以其優越的性能，打破美國的封鎖，續中國之後，土耳其、印度紛紛達成購買協議，乃至石油大國沙特也垂誕不已。蘇-35戰機也頗有斬獲。

眾所周知，美國是世界第一大軍火商，但美國對武器裝備銷售附加的限制條件很苛刻。日前，美國負責軍備控制和國際安全事務的副國務卿安德里亞·湯普森曾致信巴基斯坦空軍指揮官穆賈希德·安瓦爾·汗，譴責巴基斯坦「轉移，維持和運營」戰鬥機。

「 AIM-120C-5」字樣的導彈碎片

美國出售F-16給巴基斯坦時，附帶條件規定該飛機僅用於「加強反恐行動的能力」的目的。不能用於對付印度，甚至部署在那些基地都有要求。但在3月份的時候印度展示了擊落米格-21戰機帶有「 AIM-120C-5」字樣的導彈碎片，這是F-16戰機裝備的美製中程地對空導彈殘片。被認為巴基斯坦用F-16戰機擊落了印度米格-21。湯普森稱這種舉動可能會將美國的敏感技術移交給第三方。

巴基斯坦被稱為「巴鐵」，與中國的關係毋庸置疑，美國嚴防死守的目的不言而喻。F-16戰機並不是最先進的武器尚且如此，也可以看出美國出售武器的附加條件有多麼苛刻，有網友戲言：「連司機是男是女都有要求」。

F-16

相比美國，俄羅斯的附加條件則寬鬆的多，而且對先進的武器裝備的出售也沒有那麼多顧慮，S-400導彈、蘇-57戰機等，可以說是俄羅斯最先進的武器，都可以賣。這也使得俄羅斯軍火交易中更具競爭力，很多國家轉向購買俄制裝備，但也導致一些武器製造技術被購買國掌握。但這些年有大量的俄羅斯武器被仿製。

俄羅斯媒體12月13日引述俄羅斯Rostech國防公司知識產權部門負責人葉夫根尼·利瓦德尼的話說，在最近的17年中發現有超過500宗未經許可複製俄羅斯軍事硬體的案件。表示在國外工作的專家經常發現非法複製案件。實際上是否如此呢？

越南媒體

越南媒體對此消息進行了報道並指出，美國雷神公司和英國BAE系統公司等這些主要的武器製造商為保護其製造技術會在國外進行專利註冊，據悉擁有5000多項專利。而俄羅斯製造商很奇怪，很少在國外註冊專利。也就是說，使用這些技術並不違反法律。俄制武器裝備之所以能突破美國的封鎖，與這一點有很大關係。

報道還評論稱，最重點的是俄羅斯武器裝備的核心技術根本無法被仿製，一般都有自保機制。如果買一個設備過來就可以將整個技術學到手，那麼印度、土耳其又為何要花巨額資金與俄羅斯聯合研發武器裝備，自己拆了研製就可以了。

雖然利瓦德尼提及技術被複制，但俄羅斯並沒有提高武器裝備出售門檻，照樣大肆出售先進武器，其實俄羅斯在沒有過多附加條件的情況下，已經是默許了部分技術將被購買國掌握，只有如此才能與美製武器競爭。（風調雨順）
