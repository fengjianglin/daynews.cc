印度公然投美，俄羅斯立即反應，普京政府向巴鐵發強烈信號

天天要聞
2019-12-24

第一軍情作者：君劍

過去幾十年來，俄羅斯和印度一直保持著良好的防務合作關係，雙方合作可以說是順風順水，印度現有的軍備體系，也和俄羅斯的背後支持有密切關係。9月4日普京和印度總理莫迪會談時，還強調印度是俄羅斯的關鍵夥伴之一。之後，俄羅斯與印度協商2030年前軍事技術合作計劃，印度國防部長拉傑納特·辛格聲稱儘管美國施壓，但新德里仍將和莫斯科保持防務合作關係，預計印方將在至少5年內繼續採購俄制軍品零部件。最近，印度還和俄羅斯聯合開啟「因陀羅-2019」演習，對外展現了雙方軍隊緊密的合作關係。

儘管已經拍胸脯做保證，但是在美國的拉攏下，印度其實早就已經動搖了。現在的新德里政府，就是吃著碗里瞧著鍋里。特別是在美國提出所謂的「印太戰略」，將印度捧得很高之後，新德里更是和美國不斷眉來眼去。12月18日美印兩國防長和外長「2+2」對話會在華盛頓舉行了，這一次會議上，印度和美國達成一系列協議並重申加強戰略夥伴關係。而最重要的看點，就是雙方簽署了《工業安全附件》（The Industrial Security Annex，ISA），該文件不僅將允許美國公司與印度國有企業合作，而且允許它們與印度私營公司分享敏感的專有防務技術，這標誌著兩國在深化軍事合作上又迎來了一個重大進展。

在過去，美國對於賣武器送技術是沒什麼興趣的，但是現在為了把印度拉上賊船，美國也算是下血本了，《工業安全附件》的簽署，實際上就是為印度生產美國武器掃清了道路，如果美國向印度提供技術，就意味著是要和俄羅斯展開「血拚」了。因為印度買俄羅斯武器的一個重要原因，就是俄羅斯承諾提供技術。現在美國也這麼干，就是硬要把俄羅斯給擠出去。印度感覺左右逢源，自然是來者不拒。

然而，俄羅斯顯然不允許印度這樣四處張望。就在美印兩國防長和外長「2+2」對話會舉行之際，應巴基斯坦方面邀請，俄羅斯海軍總司令尼古拉·葉夫緬諾夫上將於12月18日至21日對巴基斯坦進行訪問。俄媒報道稱，訪問期間，葉夫緬諾夫與巴基斯坦國防部長舉行了會談。雙方討論了俄巴海軍合作問題，包括互派代表團，開展兩國海軍教育和文化領域的合作。並且，雙方還討論了將在2020年落實的活動。

巴基斯坦是印度的死對頭，雙方最近因為克什米爾問題而劍拔弩張，巴方甚至警告稱克什米爾問題可能引發熱核戰爭。而印度《公民身份法》修正案的出台，又使得印巴關係更加緊張。在這樣的情況下，俄羅斯突然和巴基斯坦走近，這顯然不是「巧合」。事實上，今年6月份的時候，巴基斯坦總理伊姆蘭·汗就確認巴方計劃從俄羅斯購買武器，並且正就此與俄羅斯開展接觸。俄羅斯戰略與技術分析中心副主任馬基延科曾聲稱，巴基斯坦對俄羅斯戰機、防空導彈系統、坦克和戰艦都感興趣，巴方可能需要價值80至90億美元的俄羅斯武器。

顯而易見，巴基斯坦和俄羅斯走近的跡象越來越明顯，而這正是普京政府向印度發出的一個警告性信號。俄羅斯無法阻止印度和美國的合作，但同樣地，印度也不可能阻止俄羅斯和巴基斯坦的合作。如果印度真的上了美國的賊船，那麼俄羅斯必然會和巴基斯坦合作，並通過支持巴基斯坦，達到牽制印度的目的。現在擺在印度面前的，是非常嚴肅的事實。印度想要繼續和俄羅斯合作，就不能一味和美國走近。否則，俄羅斯肯定不會給好臉色看。
