特朗普有份“暗深勢力”黑名單？班農：別鬧，陰謀論只適用於瘋子

2019-12-19 712條評論 3,528次閱讀 334人點贊

當地時間今天，美國國會眾議院就要對彈劾總統特朗普的條款進行全體表決了。

究竟特朗普是不是真的「變節」了，究竟是不是需要給他定罪、彈劾他？這個問題讓美國人很糾結。

不過，還沒被定罪的特朗普已經暗暗地在心中給一大群人定了罪，因為他們是想要搞垮政府的「暗深勢力」成員。

他們無處不在？

「所以，特朗普是不是認為他的聯邦調查局局長現在也成了『暗深勢力』成員？」

在司法部監察長10日發布了一份關於「通俄門」調查起源的新報告後，美國聯邦調查局局長克里斯·雷表示支持這份否認了試圖通過陰謀對總統特朗普發動「政變」的「暗深勢力」存在的報告。立即，雷就遭到了特朗普斥責。而美國有線電視新聞網也很快打出了這樣一則標題，嘲諷特朗普的「暗深勢力」成員名單又添「新人」。

CNN嘲諷特朗普「暗深勢力」黑名單又添「新人」。

究竟什麼是「暗深勢力」？

「暗深勢力」的英文說法是deep state，也常常有人將其譯作「國中之國」「深層政府」。這個來源於土耳其語的單詞原本特指奧斯曼帝國垮台前就存在的秘密政治社團，後來延伸用來指那些並非經過選舉產生的，由政治官僚、公務員、軍事工業複合體、金融界、財團、情報機構組成的，為保護其既得利益，幕後真正並實際控制國家的集團。

2014年，美國前共和黨議員助理麥克·洛夫格倫出版了一本題為《暗深勢力：憲法的隕落及影子政府的崛起》，並在書中明確了「暗深勢力」的定義。雖然這本書得到了一些好評，但並沒有引起太多的關注。而講述國家安全委員會歷史的《白宮勇士》一書的作者約翰·甘斯也寫道：「華盛頓一直沒有把『暗深勢力』的說法當回事，甚至是把它當做一種榮譽或者內部笑話來對待。」

直到特朗普出任美國總統。

作為一個來自共和黨、被指「通俄」、還是個政治素人的新總統，特朗普對「不忠的人」格外敏感，尤其是那些曾經為奧巴馬政府工作過、後來被派到白宮的職業官僚。

2017年6月，特朗普轉發了福克斯新聞的肖恩·漢尼替的一條推文，算是第一回「認可」了「暗深勢力」這個詞。同年11月30日，特朗普發出了自己的第一條提及「暗深勢力」的推文，稱公眾在希拉里電子郵件門的問題上被誤導了，「電子郵件危害了國家安全，為什麼我們的暗深勢力當局沒有給予關注？是被操縱了，還是受賄了？」這一年，「暗深勢力」這個單詞在特朗普的推文里就出現了兩次。

但到了2018年「暗深勢力」出現5次，並且從當年8月開始，特朗普在演講和媒體採訪中也越來越多使用這個詞，甚至還在10月的一次共和黨晚宴上提到了「暗深勢力」。而2019年1月至今「暗深勢力」的推文有10條，另外還在各種不同場合提到「暗深勢力」。10月的一場公開演說中，特朗普痛斥民主黨人和他們在「暗深政府」里的「死黨」，稱他們正在「串通一氣試圖搞垮大選」。而在路易斯安那州的一次集會上，他說民主黨、媒體和「暗深勢力」的官僚之間存在著「邪惡聯盟」。

寧可錯殺一千？

究竟哪些人會上特朗普的這份「暗深勢力」成員「黑名單」？

在美國國務院工作了43年後，安妮·帕特森獲得了一個去五角大樓工作的提名。在帕特森看來，要想榮耀地結束自己漫長的職業生涯，在退休前到這個事實上掌握著美國外交政策的地方任職，是一個不錯的選擇。

安妮·帕特森

於是，帕特森懷著滿心的期待等待著調令，但卻在右翼媒體上看到了對她提名的質疑。有質疑很正常，但問題是批評者將帕特森描畫成了一個潛伏在暗處、肆機破壞特朗普政府的「暗深勢力」成員、民主黨的「秘密游擊隊員」。當類似的表述在互聯網上鋪天蓋地地傳播，時任國防部長的馬蒂斯撤回了對帕特森的提名。光榮退休，帕特森是別想了，只能草草結束了自己職業官僚生涯。

在聯邦政府中幹了15年外事工作的薩哈爾·諾羅盧扎德也被右翼媒體描述成「暗深勢力」成員、對特朗普不忠的人。雖然向自己的上司抱怨過，但似乎在諾羅盧扎德看來，聯邦政府似乎並不想糾正這些錯誤的報道。相反，她被調離了原來的崗位。而對於諾羅盧扎德的遭遇，美國國務院的一份報告只是說是她的政治觀點、與前政府的聯繫和原始國籍導致了諾羅盧扎德被降級。

說我的政治觀點沒和政府保持一致，我承認；但說我屬於想要顛覆特朗普政府的「暗深勢力」，開什麼玩笑？隨著彈劾總統特朗普的進程深入，像帕特森這樣被「卷」入「暗深勢力」中、被踢出圈子的美國職業官僚越來越多。

一方面，被懷疑為「暗深勢力」成員而被踢出的聯邦政府行政機構中的職業官僚越來越多，另一方面，因為所謂「忠誠」而受特朗普青睞並獲得提名的人也越來越多。

《紐約時報》援引美國外交服務協會的數據稱，在特朗普手下任職的166名大使中，45%是根據忠誠度和競選獻金挑選的政治任命官員，這是史上最高比例。

但別以為因為「忠誠」而被選中就撇清了與「暗深勢力」的關係。「你看看絕對敗類的聯邦調查局高層都幹了什麼。你看看司法部都發生了什麼。」在10月的一次訪談中，特朗普說道，「（前司法部長）傑夫·塞申斯是個災難，徹底的災難……我把他放在（司法部長的）那個位子上是因為他支持我，並且我迫切需要他的支持。」

還有特朗普親自提拔的前國家安全事務助理赫伯特·雷蒙德·麥克馬斯特，就因為禁止手下人稱職業官僚為「奧巴馬的留任者」，並且提拔了其中一些人，也被當時特朗普的「軍師」、鼓吹所謂「暗深勢力」陰謀「政變」的班農利用自家的新聞網站布賴特巴特（Breitbart）批成了「暗深勢力」成員而備受攻擊。

無論當初多麼地你好我好、互相欣賞，只要一朝惹得特朗普不高興，就會身後挨刀，成了「暗深勢力」的一份子。

所以，簡單地說，奧巴馬任期的官僚、甚至是整個聯邦政府的行政體系和所有那些對特朗普不忠的人，在特朗普看來都屬於「暗深勢力」。而特朗普對付他們的方法，便是寧可錯殺一千，絕不放過一個。

「最偉大的成就」？

「我認為打敗『暗深勢力』是我做過的最偉大的事情之一。」特朗普毫不諱言自己有多看重打擊「暗深勢力」，稱與「暗深勢力」的鬥爭對他的總統任期至關重要，「我已經對它造成了絕對巨大的破壞，這會是我最偉大的成就之一。」

不過有趣的是，正是這個當初鼓吹「暗深勢力」陰謀論、幫著特朗普到處「捉拿」不忠者的班農，最近卻親口否認了「暗深勢力」的存在。

班農說，美國有一個強大的政府官僚機構，但並沒有什麼藏在陰暗深處的「暗深勢力」，一切觸手可及。一本新書援引了班農的話稱，「『暗深勢力』的陰謀論只適用於瘋子」。他甚至還表示，以「特別邪惡的『暗深勢力』成員」的說法炒了詹姆斯·科米的聯邦調查局局長一職是美國「現代政治史上最大的錯誤」。

班農。

如果就像班農所說的那樣，「暗深勢力」並不存在，那麼「請考慮這樣一個問題：如果在過去很長時間裡傑出地工作的那些人，只是因為陳述已知的事實而被解僱，那麼我們確實處在一個非常非常暗深的地方。」美國有線電視新聞網意有所指地評論道。

撰稿 深海三文魚
