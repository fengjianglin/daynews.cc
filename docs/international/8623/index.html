亞歷克斯·霍諾德：無保護攀登酋長岩

2019-12-10 372條評論 3,046次閱讀 740人點贊

酋長岩，位於美國加州，是高914米的全球最大的花崗岩巨型獨石，酋長岩因其陡峭和艱險著稱，被譽為全世界最難攀爬的岩石，但就是這一塊艱險的岩石，卻成為幾代人的夢想，1958年，沃倫·哈定第一次攀登酋長岩，歷時16個月，摯登過程46天，因為實際攀爬的難度遠比預計的大，體力的迅速流失，讓他們無法連續攀爬，從那時開始，酋長岩就成了全世界攀岩愛好者的聖地。攀岩是世界上最危險的運動之一，就像參加奧運會，你必須拿到冠軍，否則您就會摔下懸崖粉身碎骨。而無保護攀岩則更讓人感到「死亡近在咫尺」，失誤所要付出的巨大代價讓只有不到1%的攀岩者願意嘗試這種近乎瘋狂的運動形式。但在2017 年 6 月 3 日，美國攀岩大師亞歷克斯· 霍諾德 ( Alex Honnold )在無輔助情況下，無保護徒手攻克美國約塞米蒂國家公園 3000 英尺 ( 約 914 米 ) 高的酋長石 ( El Capitan ) ，用時 3 小時 56 分鐘。成為全球首位徒手快速登頂酋長岩的人。

亞歷克斯.霍諾德

那些把「無保護攀岩運動」當成生活重心的人全都死了，約韓·巴卡爾是無保護攀岩的先驅人物，但他在2009年不幸摔落身亡。肖恩·利里，在2014年在猶他州攀岩失誤導致死亡。德里克·赫西，在攀登約塞米蒂岩時死亡…….。亞歷克斯的母親狄德莉·沃羅尼克在接受採訪時說：「我不希望他去無保護攀岩，我很感謝他每次去無保護攀岩時都不告訴我，否則我肯定會說：「你不能去」，這樣就會和他疏遠，因為攀岩就是他的生命，他是無保護攀岩時是最有生命力的，最能體會一切…….」。

「我無法理解這種行為，只要一次小小的失誤，只要一次沒抓穩，就會從高達幾百米的岩壁上掉下去摔死，為什麼還要去從事這樣一種工作？」美國一訪問節目主持人這樣問，「可能是因為我太熟悉攀岩，反而感覺不到危險的臨近」。亞歷克斯如是說。

酋長岩

亞歷克斯是個素食主義者，大約從20多歲開始，為了保護環境，他決定不再吃肉開始適應各種蔬菜，亞歷克斯是一個有點古怪的人，在外人看來有點自閉，小時候他便開始攀岩，他經常獨自去無保護攀岩，不想認識人，也不想和別人說話，夢想的種子在自己的小天地里生根發芽，亞歷克斯性格的養成和他的父親可能有很大關係，亞歷克斯的父親查理是一個少言寡語的人，和誰都不親近，總是很陰鬱，有點像今天的「阿斯伯格綜合症患者」查里唯一喜歡做的就是旅行，旅行對於查里就像攀登對於亞歷克斯，後來亞歷克斯的父母離婚了，第二年亞歷克斯的父親便去世了。互歷克斯從不向別人談論自己的攀岩計劃，在他看來，無保護攀岩是個十分古怪的個人行為。在大多數人眼裡，我想是的，當別人談及這種瘋狂的舉動，大多會說，他可能心理有問題，亞歷克斯曾去做過一次腦部核磁共震，在檢查過程中醫生給亞歷克斯看了大量的血肉模糊的照片或者糞便堵住馬桶的照片，但結果發現，亞歷克斯的杏仁體（大腦恐懼情緒中樞）沒有任何活動，醫生得到結論，亞歷克斯的杏仁體需要很高水平的刺激才會活躍，這也可以解釋亞歷克斯是如何克服無保護攀岩的恐懼的。

在亞歷克斯成功攀登酋長岩之前，他己經是美國著名的專業攀登者，亞歷克斯已經做出了成績，way? 為什麼要去無保護攀登?他這樣回答： 「我不想墜落身亡，可是成動挑戰自己能帶來滿足，面對死亡的時候，滿足感更強烈，因為絕不能犯一個錯誤。如果你想追求完美，無保護攀岩是最好的方式」。

「我攀登過很多次酋長岩，但是沒有進行過無保護攀登，沒有人做過，稍微想想就知道，很嚇人的」亞歷克斯·霍諾德，從2009年開始，亞歷克斯每年都會考慮無保護攀登酋長岩，但都放棄了，但亞歷克斯從未停止為攀登酋長岩做準備，他需要熟悉攀登路線上的每一個繩段、每一塊石板、每一個踩點，他需要一張詳細的路線圖。2012年，亞歷克斯和湯米·考德威爾來到摩洛哥攀岩，為攀登酋長岩做訓練。後來湯米接受採訪時表示：「和亞歷克斯在山間白由的攀爬今人著迷，他面對風險的態度，會讓人感到自己非常強大，這種自信，很吸引人。」

亞歷克斯正在攀登酋長岩

無保護攀登酋長岩，亞歷克斯從未停止努力過，他一邊又一邊試攀，「通過訓練，讓恐懼不再影響我，逐漸擴大自己的舒適區。」最讓亞歷克斯恐懼的是「極限平板」段，能落腳的地方很窄，只有岩壁上微小的突起，一旦腳下打滑，手也無法抓穩，只有兩個很小的接觸點防止你摔下山崖。亞歷克斯從2009年一直到2018年攀登過成百上千次酋長岩，第一段，從左側一路向上，雙腿分開，很安全。第二段，相信右腳，推重心，右手出到最後一個反提。第八段，很容易，快速通過。第九段，倒攀時保持在外側，小心落石。第二十六段，沿內角對抗動作向上，左手關鍵捏點，右腳在較低腳點上折膝，左腳抵住岩壁站起來，左手抓住大的耳形把手點，換腳並手抓住把手點，左腳卡入岩縫，右手從把手點頂端向下拉……右手手指插入小的指洞，左手抓住一個反提……。每一個動作都融進了他的血液。

2017年四月三十日，烏里·斯特克在珠峰附近喪生。

2017年6月3日，亞歷克斯·霍諾德成為無保護登頂酋長岩的第一人。
