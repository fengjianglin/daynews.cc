特朗普欲在明年修訂美國現有汽車排放法案

2019-12-10 811條評論 3,253次閱讀 187人點贊

文：懂車帝原創 李德喆

[懂車帝原創 行業]美國總統特朗普日前表示，他將在明年完成對奧巴馬時代汽車尾氣排放標準的修訂。業內人士預計，這將引發包括加利福尼亞州在內的新的法律挑戰。

最近幾個月來，特朗普政府一直在致力於推動修改燃油排放標準的提案。特朗普方面辯稱，從經濟和安全形度考慮，重新調整是非常必要的。然而，來自加州的環保人士卻一直對修訂標準持反對意見，他們認為，這將導致消費者在燃油成本上多支出數千億美元的開支。

去年8月，特朗普總統提議，從2020年開始，要將美國的汽車排放標準要求推遲6年，即在2026年時，車輛的平均燃油效率達到37英里/加侖（約為百公里7.6升），而根據現行2012年通過的規定，到2026年，車輛的平均油耗需達到46.7英里/加侖（約為百公里6.0升）。

此外，特朗普總統還希望，到本世紀30年代，將美國石油的日消耗量提升50萬桶左右，並將美國本土的汽車製造商監管成本減少逾3000億美元。

特朗普在上周五說：「我們有足夠的燃料，新的規定將為消費者帶來更安全、更實惠的汽車產品。」

然而，想要推動修訂案的落實沒那麼簡單，現在的狀況是，美國的州政府正在與特朗普總統「唱反調」。就在上個月，加州和美國其他22個州提起訴訟，對美國政府在今年9月作出的，撤銷加州制定嚴格的車輛尾氣排放規定以及增加零排放車輛數量的決定提出質疑。

但是這一次，幾大汽車廠商卻與特朗普站在了一邊。包括通用、豐田以及菲亞特·克萊斯勒在內的多家汽車製造商支持特朗普禁止加州設定汽車尾氣排放標準的努力。

值得一提的是，美國司法部在今年8月對四家與加州達成協議的汽車製造商展開了反壟斷調查，並同時向福特、本田、寶馬和大眾汽車發出民事傳票。

總的來說，特朗普政府的最終要求是適度提升車輛的燃油效率，而不是「一刀切」。部分汽車製造商預計，產品燃油效率提升將保持在每年1.5%左右，而這遠沒有奧巴馬時期規定的嚴格。
