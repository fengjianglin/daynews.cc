"北溪2號"項目遭美製裁 俄方強硬回應：將對等報復

2019-12-24 209條評論 1,771次閱讀 502人點贊

來源:海外網

圖源：俄羅斯衛星通訊社

海外網12月23日電美國方面宣布將對俄羅斯與歐洲天然氣管道項目「北溪2號」進行制裁後，該項目於周六（21日）停工。俄羅斯方面回應稱將對此實施對等報復。

據美聯社23日消息，克里姆林宮發言人佩斯科夫當日表示，俄羅斯認為制裁不可接受，俄方將對制裁採取「對等」（reciprocity）措施。此外，佩斯科夫還表示，克里姆林宮仍希望完成「北溪2號」，「如何以及何時完成這項工作仍然是俄羅斯國家利益的問題」。

俄外長拉夫羅夫（Sergey Lavrov）亦重申了這一觀點。拉夫羅夫說，在經過對局勢進行「冷靜的分析」之後，俄羅斯將對「這些絕對令人無法接受的，且赤裸裸地違反國際法所有準則以及外交和人權尊嚴」的行為做出回應。

當日晚些時候，俄政府總理梅德韋傑夫（Dmitry Medvedev）也表示，他已責成內閣負責調查可能對美國施加的「限制」，以報復美國方面施加的制裁。

據悉，「北溪2號」項目旨在鋪設一條由俄羅斯經波羅的海海底到德國的天然氣管道，可繞過烏克蘭把俄羅斯天然氣輸送到德國，再通過德國幹線管道輸送到其他歐洲國家。長期以來，美國一直是該管道的直言不諱的反對者，認為此舉將增加歐洲對俄羅斯能源的依賴。美國政府於上周簽署一項制裁「北溪2號」相關企業和個人的法案，此舉促使瑞士的一家企業暫停了其參與的「北溪2號」項目的所有工作。

德國方面抨擊白宮的「歐洲能源多樣化」訴求是「干涉德國內政」。德國總理默克爾誓言要與美國討論此事。（編譯/海外網 侯興川）
