末代漢諾威公主，拒絕當維多利亞女王的兒媳，寧願終身不嫁

天天要聞
2019-12-09

1866年，普魯士王國吞併漢諾威王國之後，維多利亞女王的堂弟，漢諾威國王喬治五世一下子成為了亡國之君，儘管他從來沒有承認過亡國。他的兩個女兒弗雷德里卡和瑪麗，面對突如其來的變故，因心氣不同，做出了迥異的人生選擇。

弗雷德里卡是姐姐，1848年出生，比妹妹瑪麗大一歲。雖然是亡國公主，但是頭銜還在，而且作為英王喬治三世的曾孫女，兩姐妹在英國也是公主，所以在婚戀市場上，還是很有競爭力的。各路王室親戚們，也希望弗雷德里卡和瑪麗，能夠嫁進在位王室，重新建立地位。

（弗雷德里卡）

不過弗雷德里卡不這麼認為，反而覺得爸爸丟掉王位，對她的婚姻來說，是一件好事情，因為她的婚姻可以自己做主了，再也不需要充當聯姻公主的角色了。於是，弗雷德里卡遵從自己的內心，嫁給了她爸爸的隨從，定居英國，仰賴英王室，投身於慈善事業，舉案齊眉。

而瑪麗呢，一直沒放下亡國之恨，在維多利亞女王非常想要她做兒媳的情況下，依然選擇拒絕。

（瑪麗）

1875年初，維多利亞女王的三兒子亞瑟，正在尋找新娘，他拜訪瑪麗的爸爸時，看上了瑪麗。維多利亞女王挺高興的，覺得瑪麗會是她心目中的好兒媳，很期待兒子能帶來好消息。然而瑪麗面對亞瑟的熱情，很冷漠。維多利亞女王很震驚，不敢相信瑪麗竟然不想嫁給她的寶貝兒子！

不死心的亞瑟繼續追求瑪麗，他覺得如果瑪麗一家來英國度假，會是求婚的好時機，但是1876年5月，瑪麗一家真的來英國小住時，婚事依然沒有轉機。當維多利亞女王向瑪麗的媽媽探口風時，瑪麗媽媽說一切以女兒的意願為主。瑪麗的哥哥更是找託辭，說瑪麗曾有一段深刻的、不匹配的戀情，所以無法接受亞瑟的心意。

（維多利亞女王和兒子亞瑟）

1877年初，亞瑟再次拜訪了瑪麗的爸爸，希望這段婚事能成，然而瑪麗讓她爸爸開出了條件——如果她嫁給亞瑟，必須保證她將永遠不會和亞瑟的大伯薩克森·科堡和哥達公爵、或者亞瑟的姐夫普魯士王儲碰面。瑪麗恨這兩人，因為他們在吞併漢諾威王國上出了一份力。

亞瑟為了娶到瑪麗，同意了這個條件。維多利亞女王為了自己看中的兒媳人選，也作出了讓步，承諾儘可能地滿足。但是令人意外的是，瑪麗似乎鐵了心，並不滿意這個讓步，她最後一次拒絕了亞瑟。

（瑪麗和她媽媽）

維多利亞女王很氣憤，覺得漢諾威非常愚蠢，其餘的王室親戚更是認為瑪麗是一個自私的白痴。最後亞瑟娶了普魯士的公主，而瑪麗終身未婚，與她媽媽相伴到老。
