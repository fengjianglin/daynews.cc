美國一直以來都以打壓中國為目標，不論是軍事、政治、還是科技等等各方面，無不充斥著美國人的身影。

隨著中國的實力越來越強盛，越來越得到世界各國的友好示意，美國再次坐不住了！

畢竟中國的強盛會影響到美國的國際強國地位，這是美方所不容允許的，以前的美國把蘇聯作為打壓目標，而現在儼然變成了中國。

美方不服中華文化？”鬥茶”引200萬美國人觀看，美國承認中國地位

近段時間發生了一件事，引發各界人們的關注！

在紐約舉行了一場「鬥茶」活動，我們都知道鬥茶是我國古時候的文人娛樂活動，為了促使弘揚我國的中華文化，特意在美國舉行了一場鬥茶活動。

此次活動舉辦的十分成功，現場引200萬美國人觀看，近萬美國人參與品茶活動，現場很多美國人都表示：「我愛中國，我愛茶文化」。

我們聽到他們大家讚賞我國的茶文化，是非常欣慰的。

說到這，可能很多人不知道什麼是”鬥茶”？

「鬥茶」其實是我國古時候品茶的一種方式。

在宋朝時期的茶文化，發展到盛況空前的地步，幾乎全國上下每個人家裡都有茶杯和茶葉，飲茶已經成了那時代不可或缺的東西。

在宋朝時期，鬥茶活動十分受歡迎，不論是達官貴人亦或者是普通的老百姓，都非常喜歡參與這項活動。

人們通過品茶來鑒別茶的好壞，比較茶的優和劣，具有強烈的賽事色彩。

這種茶文化一直流傳至今，在現代很多地方都會舉辦民間的「鬥茶」活動。

鬥茶一般會選在清明節期間舉行，因為此時的新茶都慢慢露出頭，最適合參斗。

鬥茶的比賽一般是小組間的競賽，現場分為兩組甚至更多組，每組5-8個人為宜，每個人都拿出自己珍藏的茶葉，泡完之後讓現場的民眾飲用，讓大家為這些茶葉打分。

現場實行積分制，得到積分最多的那一組即可獲得勝利。

比賽內容包括茶葉的色相與芳香度、茶湯香醇度，茶具的優劣、煮水火候的緩急等等。

這一文化不僅民間很受歡迎，連皇家都非常流行這種「鬥茶」的活動。

當然，隨著現在社會的發展，很多地方已經丟失了這樣的傳統，國家也在大力倡導中華文化，不能讓老祖宗的智慧結晶斷層，這需要大家的共同努力。
