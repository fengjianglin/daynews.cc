屠新泉：WTO遭美重傷，但不會死

天天要聞
2019-12-07

專家簡介

屠新泉，全球化智庫（CCG）特邀高級研究員，對外經濟貿易大學中國WTO研究院院長。

12月10日，兩位WTO上訴機構成員將退休，而原本7人的WTO「最高法院」將只剩下1人，從而喪失其審理案件的能力和權力。上訴機構是WTO爭端解決機制中的重要一環，其停擺也將對WTO造成嚴重傷害。

之所以造成這一局面，主要是因為美國特朗普政府上台以來，不顧其他WTO成員的呼籲，頑固拒絕延長現有法官任期或者任命新的法官。美國自稱這麼做的理由是因為上訴機構經常超越其授權，濫用其權力解釋WTO規則，從而使成員(當然主要是指美國)承擔超出其實際承諾的義務。上訴機構的使命就是確保爭端解決機制的獨立性和權威性，美國對上訴機構的破壞極大損害了WTO審議和解決成員間爭端的能力。

面對美國的惡意破壞，其他163個WTO成員正在積極做出努力，維護以規則為基礎的多邊貿易體制。中國和歐盟已經聯合提出維護和改革WTO爭端解決機制的建議，其他多個成員也在以建設性的姿態提出相關提案，力爭維持爭端解決機制的運轉，也曾力圖勸說美國改弦更張。但從當前的形勢來看，美國已經鐵了心要癱瘓上訴機構。在這種情況下，一些成員提出一些臨時性的解決方案，比如自願接受專家組裁決為最終裁決，自願在相關成員間建立仲裁機制以替代上訴機制等。儘管各方在具體方案上還沒有形成一致，但只要大家共同維護爭端解決機制的權威性，即便美國不再參與，WTO仍將維繫其爭端解決功能。

需要指出的是，上訴機構的癱瘓並不意味著整個WTO爭端解決機制的終結。從程序上看，爭端解決機制的前兩個階段分別是磋商和專家組。從1995-2018年間，總共有573個案例被提起，其中40%的案例都是通過磋商得到解決的，而真正產生專家組報告的只有249個，而其中1/3的案例並沒有被提起上訴。也就是說，真正進入上訴階段的案例只有166個(佔573個案例的不到30%)。因此，只要相關成員之間有足夠的意願解決彼此間的爭端，即便沒有上訴機制，大部分的爭端仍然可以得到解決。

從WTO的功能來看，爭端解決也絕非唯一或者最重要的部分。事實上，WTO的最重要功能是將這164個成員的貿易政策置於同一框架下，通過最惠國待遇原則實現彼此間的、相互交叉的利益交換和捆綁，形成一個貿易共同體。

美國雖然破壞了WTO爭端解決機制中的上訴機制部分，但美國並未真的退出WTO，美國仍然在總體上維持其對其他WTO成員的承諾，同時其他成員之間以及其他成員對美國也仍然維持原有的承諾，這些承諾才是保障世界貿易順利運轉的關鍵。爭端解決機製為維護這些承諾的可信度提供了重要保障，但更重要的是各成員維護和推動貿易自由化的決心和意願。實際上WTO仍然是唯一能夠提供這種多邊承諾的場所，而且仍然在為制定新的承諾做出努力。目前有70多個成員參與的電子商務談判正在WTO進行，其目的是為了建立更加自由的跨境電子商務和數字貿易政策環境。

經濟全球化是歷史潮流，雖然WTO當前面臨嚴峻挑戰，但各國之間開展緊密經貿合作的需求沒有變也不會變，即便是美國也不可能真的孤立於世界。我們應當相信，經過國際社會和WTO成員的共同努力，以規則為基礎的多邊貿易體制必將繼續展現其生命力，為經濟全球化的向前發展提供有力基礎。也希望美國政府能夠早日放棄錯誤做法，與國際社會一道維護多邊主義的國際體系。

文章選自《環球時報》，2019年12月6日
