日本一家公園掛了一張牌，寫明針對越南人

2019-12-18 1,509條評論 3,789次閱讀 2738人點贊

據越南快訊網報道，因為越南人丟垃圾引起周圍人的抱怨，日本Iwase公園限制越南人進出該公園。

越南媒體報道截圖 12月15日，日本富山行政辦公室告知，從8月開始，大約300名越南人在富山市Iwase體育公園組織足球比賽。比賽結束後他們在角落留下了一些吃剩下的食物和一些塑料空瓶而沒有收拾乾淨。

Iwase體育公園本身沒有垃圾桶，要求人們要自覺帶走垃圾。在住在周邊的居民抱怨說有臭味之後，公園已經安排了工作人員收拾了這些垃圾，用兩輛貨車拉走垃圾。公園管理處表示，從去年的夏天開始已經掛牌就丟垃圾的事情進行了警告，但是情況沒有得到改善。因此，在今年的事件發生之後，他們只好掛牌限制越南人，並宣布禁止越南人在三個月內使用該公園，時間從8月至11月。

8月越南人組織足球賽的球場 一名富山縣的官員說：「這是其他人可以較為舒適的使用該公園的一個合理辦法」。 一名富山縣的律師承認，越南人在文化、習俗、語言方面有些差異性的地方。雖然如此，如果你沒有處理好垃圾，你就不能使用公園。 公園也在討論向想要使用公園的越南球隊增收垃圾處理費用的相關事宜。 日本Weibun報道說，Iwase公園辦公室的人員說，這個規定也許很多人會反對，認為有歧視，但是他們也只能這樣做才能改善有人在公園亂丟垃圾的狀況。一個生活在附近的80歲的老先生對一些人在公園丟垃圾會造成不好的先例表示擔憂，擔心其他人也效仿這種行為。

據中越加報道，去年，一名32歲的越南實習生因為在東京一家公園打死兩隻野生鴨子被抓捕。這個實習生想把這兩隻鴨帶回去煮粥，因為不適應當地食物的口味，但是卻不知道在公園抓鴨子違反了日本的野生動物保護法。
