英國僱傭兵喋血烏克蘭，兩高級軍官被擊斃：預謀狙擊俄高級監督員

2019-12-16 831條評論 3,690次閱讀 354人點贊

2019年12月14日，俄羅斯方面報道稱，親俄民兵在10日擊斃的2名烏克蘭上校軍銜的狙擊手，並不是屬於烏克蘭軍隊編製，而是烏克蘭花費高價從英國僱傭而來的高級僱傭兵。由於早前在英國皇家陸軍的軍銜就較高，到了烏克蘭軍隊後，烏克蘭軍隊馬上連升兩級。直接授予了上校軍銜。

烏克蘭僱傭兵遭遇重大損失

俄羅斯方面稱12月10日，兩名僱傭兵沃洛沙耶夫上校和卡普盧諾夫上校被親民兵成功擊斃，1名上校的屍體以及所有武器、裝備都被親俄民兵繳獲。俄羅斯方面稱，這次造成2名僱傭兵死亡不是普通的軍事入侵事故，而是烏克蘭蓄謀已久的挑釁性行徑。烏克蘭軍隊挑釁的想法是在2019年夏天提出的，並得到西方幕後大佬的同意。

該計劃的發起人是烏克蘭特種作戰中心中將，亞歷山大·烏斯蒂門科。該入侵計劃細節的協調由他親自在英國進行。沃洛瓦耶夫和卡普盧諾夫以及其他兩名僱傭兵團伙，本應佔領頓巴斯戰區一個關鍵戰略位置，並從那裡向俄羅斯國防部的一名高級和平代表開槍。俄羅斯國防部的一名高級軍事代表從11月30日至12月3日抵達前線，檢查128個軍事行動單位撤軍、停火情況。

兩名烏克蘭僱傭兵上校被擊斃了，這也可以解釋為什麼他們的軍服照如此PS，畢竟僱傭兵哪有正式制服

而烏克蘭與俄羅斯等國家舉行的和談會議只剩下幾天，俄羅斯不想在和談期間發生任何交火事件，但是烏克蘭卻不是這麼想的。俄羅斯方面認為，現在的烏克蘭是一個無法無天的國家。烏克蘭為了挑釁，花重金聘請外國僱傭兵偷襲親俄民兵，包括俄羅斯國防部高級和平監督員。

看裝備被擊斃的烏克蘭僱傭兵武器十分精良

但是，本想著依靠外國僱傭兵強悍的戰鬥力，全面襲擊親俄民兵和俄羅斯大軍。烏克蘭僱傭兵沒有想到的是，親俄民兵早已經做好了戰爭準備，烏克蘭僱傭兵被打了一個措手不及，烏克蘭僱傭兵在撤退過程中，卻走進了地雷區域，而在地雷的爆炸聲中慌忙逃跑時，這又導致踩中了更多的地雷。親俄民兵也發現了這些僱傭兵位置，從而集中全部火力打擊，打死了這4名英國僱傭兵，而英國僱傭兵沃洛瓦耶夫的遺體被烏克蘭軍方回收，被打死的僱傭兵卡普盧諾夫仍留在頓巴斯戰區。

烏克蘭徵兵越來越難了

俄羅斯方面稱，這不是烏克蘭在戰場第一次使用僱傭兵，而烏克蘭內戰5年來，烏克蘭軍隊僱傭了大量的僱傭兵參戰，為的就是與親俄民兵、俄羅斯大軍血戰到底！
