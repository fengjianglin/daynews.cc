混血女孩東南亞運動會上獲跆拳道銀牌，網友卻讓她去參加選美

2019-12-10 2,200條評論 3,999次閱讀 2850人點贊

東南亞運動會每兩年舉辦一次，是東南亞地區最受歡迎的體育盛事之一。2019年的東南亞運動會在菲律賓舉行，經過一番激烈的角逐，於12月10日落下了帷幕。各個項目的優秀運動員們，也都拿到了屬於自己的榮譽。其中一名跆拳道運動員，卻因超高顏值引發關注。

據外媒12月10日報道，這名運動員是現年21歲的Casandre Tubbs，她在女子62公斤級的跆拳道比賽中拿到了銀牌，實力非常出眾。有趣的是，比賽結束後，網友們不僅被Casandre Tubbs的實力征服，還被她柬埔寨和美國混血的高顏值吸引，說她去參加選美比賽也能拿獎。

這位人見人愛的美女運動員，究竟有著什麼樣的經歷呢？原來，Casandre Tubbs的母親是柬埔寨人，父親是美國人。她在美國的德克薩斯州出生和長大，完美繼承了父親和母親身上的種種優點，既有著精緻的面孔，又有著高挑的身材和良好的運動天賦。

Casandre Tubbs從小身體健康，非常熱愛運動，這點父親功不可沒。因為父親是一名跆拳道教練，他有意識地培養女兒的運動能力，堅信良好的身體狀態有助於更好的生活。在父親的培養下，Casandre Tubbs小小年紀就參加了很多大大小小的跆拳道比賽。

幾年前，Casandre Tubbs對母親生長的地方很有興趣，於是搬到柬埔寨生活，還成功進入了柬埔寨的跆拳道國家隊。今年冬天，Casandre Tubbs代表柬埔寨參加東南亞運動會，憑藉出眾的表現，拿到了女子62公斤級的跆拳道比賽中拿到了銀牌。

有趣的是，東南亞的網友們在觀看比賽過後，很多人都被Casandre Tubbs的顏值圈粉。現年21歲的她有著180厘米的身高，腿長腰細、身材勻稱，是大家眼中的女神外形。比賽結束後，網友們紛紛翻找Casandre Tubbs的社交賬戶，發現她在日常生活中看起來也很漂亮。

雖然比賽已經結束，大家對Casandre Tubbs的討論依舊熱烈。其中很多人都誇讚她在跆拳道比賽中的實力，也有人完全沉迷於她的美麗。

有網友說，Casandre Tubbs不需要過多打扮，看起來也優雅美麗，完全可以去參加選美。Casandre Tubbs確實是名副其實的美女運動員，即使不去參加選美，也能以健康有活力的形象獲得大家的喜歡。
