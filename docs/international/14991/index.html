12個到了泰國的中國人加上2000泰銖，今時今日竟成了幾十億資產

2019-12-15 342條評論 2,337次閱讀 53人點贊

你可能聽說過一句話，在泰國，宗教第一，皇室第二，華人第三。這句話其實沒有誇大，在泰國這個國度，華人的影響力及社會地位確實無與倫比。除開經濟和政治不說，其實還有一個原因。可能很多來過泰國有看到他們的身影，但卻並不清楚他們到底是誰，今天夜貓就來跟大家簡單的說一下。

在泰國，有一個民間組織，叫「報德堂」，這個「報德堂」，在泰國可謂是無人不知，它是於1910年由泰國華僑領袖鄭智勇等12人共同發起創立，當時主要所做的善事是收斂無名屍。因為早年來泰國的華人生活貧困，非常多人從事重勞力工作又或者是危險係數高的工作，而且當其時孤身前來泰國的人不在少數，這就導致了經常出現，有華人發生意外身故，屍首卻無人認領。由於習俗不同，加上無人認領，也就談不上說辦理後事，「報德堂」成立後，就挺身而出。當年的報德堂，靠著成立初期僅有的2000泰銖，專門為這些命喪他鄉的無名屍舉辦喪禮，讓其入土為安。當時在泰國，只要遇到身上穿著中式衣服的無名屍，便會通知報德堂。

後來的報德堂影響力逐漸誇大，在收屍的基礎上，還在泰國做了大量的施醫義葬、扶貧濟幼等慈善工作。信仰佛教的泰國人，也開始加入到報德堂，一起為有需要的人做出自己的貢獻。

社會大眾受到了報德堂的幫助，反過來報恩，以自己的錢財或人力支持慈善事業。報德堂在得到支持後又做更多的善事，受益者又感恩之後報恩，如此循環不已，慈善事業不斷擴大發展。

100年間，風雨無阻的幫助他人，也讓報德堂的到無數被援助者的幫助。從成立註冊資金只有2000泰銖到後來，成為一個擁有幾十億泰銖資產的「慈善巨頭」，報德堂，也從未停下它為人服務的腳步。

1937年，報德堂創建華僑助產院，1939年，擴大為華僑醫院，1979年再擴建為22層樓大廈。

1994年，由報德堂發動，得到泰華社會的支持，報德堂成立了華僑崇聖大學，並在該校開辦中文系，教授中文。

1997年，始建新義山莊和萬人墓。

2000年，華僑中醫院七層樓大廈奠基。

中間經過幾代人堅持，一點點把慈善事業擴展開來，帶來了，除了慈善公益，還有泰國人民對我們中國華人接納。經過長期的不斷善舉，保德堂得到了泰國社會的認可，為泰國各族人民讚賞，也得到泰國王室和官府支持。

現在的泰國，只要有困難的地方，就必定會有像「報德堂」這樣的善堂機構的志願者的身影，並且在所有的案發或事故現場，他們也永遠比警察先到達和解決現場的問題。

這些志願者們是來自各行各業，他們通常利用工作之餘的時間來免費當志願者。

他們也抱著像老一輩的泰國華人一樣的看法，認為今天救助了別人，說不準哪天自己需要別人救助的時候，就會有人伸出援手。

直至今時今日，報德堂除了是慈善的代表，也同時讓中國的文化，在泰國流傳著。
