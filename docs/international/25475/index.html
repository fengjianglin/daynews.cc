我以為伊朗沒什麼好買的，但是我錯了，大、錯、特、錯

2019-12-23 263條評論 4,181次閱讀 223人點贊

每天關注壹行伊朗 了解中東實事與動態

出發前一直在網路搜尋伊朗到底有什麼好買，但是資料不多。

大多數都是說地毯，波斯地毯自古聞名，現在又分為手工編織和機器編織，來到伊朗當然要買手工編織地毯。但也都提到手工地毯的價格昂貴，畢竟手工地毯編織曠日耗時，加上打結數量多寡、材質、用色多寡都會影響價錢。

其他東西，大概就是蕃紅花（也稱為藏紅花），伊朗的蕃紅花產量佔了全球90％以上，在這裡可以買到品質精純的蕃紅花。點擊文底「閱讀原文」，購買伊朗最好的藏紅花！

魚子醬，但我此行只有在機場免稅店才看到魚子醬，也許是我走的都是中部大城市，地處內陸，而魚子醬以裏海為生產地有關。

另外還有提到駱駝骨製品(這我沒興趣……)，其他資訊其實真的不多，只片段看到有人提到民族風首飾之類很好買，但我又對這些首飾不感興趣。

我以為伊朗沒什麼好買，但是我錯了，大、錯、特、錯！

後來根本是後悔因為怕旅費不夠不敢放手大買，結果還剩錢回家，然後懊悔買不夠

之所以會有這個遺憾其實也跟行程受影響有關係。原本安排在Esfahan(伊斯法罕)住上四天三夜，就是為了好好逛Bazaar(巴扎)。

「伊斯法罕半個天下」，搜尋伊朗資料時必定會看到這句話，指的便是當年伊斯法罕的巴扎是世界貿易中心，只要在巴扎逛上一圈幾乎就像是走了半個世界一樣，可以買到各地物產。

伊斯法罕是我倒數第二個城市，本就打算在這裡好好逛逛，這樣就算買了東西也不用帶太久。但因為大雪耽誤一天行程，我無奈地臨時砍掉一天Eafahan行程，連帶逛巴扎的時間大受影響，以至於我市區有些景點沒走，巴扎也逛得不夠過癮，真是遺憾。

[ 圖：伊斯法罕，伊瑪目廣場旁紀念品藝廊 ]

先說說我這次在伊斯法罕買的東西。時間不夠，買的東西真的很少，連我家那不愛買東西的另一半都說：好好買，可惜時間不夠。

買不起絲質手工地毯，我最後還是買了一張羊毛手工地毯。依材質分，地毯可略分為絲質、羊毛，絲質當然比羊毛貴上許多。

打結數越多、用色越多的代表手工越細，所需耗費的時間越多，當然也就越貴。

幾乎每個城市都有地毯工廠，但Kerman是最大的生產地，品質也最好，Kahsan則是第二大生產城市（以上資料為計程車司機告知）

[ 圖：伊斯法罕地毯店 ]

[ 圖：亞茲德地毯店 ]

當我們來到Kashan時，的確經常看到許多車子載著地毯前往其他城市，路邊的地毯店也多餘其他城市。

舊巴扎里竟然還可見到已經有4、50年的地毯，雖然因為使用許久且放在一旁而堆滿灰塵，但色澤依然亮麗，見識手工地毯的耐用。

而我也在這裡看到此行看過的地毯中覺得最美的。那不像我們在其他觀光區進去的地毯店一般，一直不斷地展示地毯，且價錢驚人。

Kashan舊巴扎里的那間店，東西其實不多，但老闆從身後柜子里取出的那張地毯，一打開，我跟另一半眼睛完全一亮：美！美極了！

無論是用色、圖案和細密度，都遠在我們之前所看的地毯之上！而且價錢真的比其他地方來得實惠。

就像旅行社老闆跟我說的：Kashan本身就是個地毯城市，許多大城市的地毯其實都來自Kashan，尤其Eafahan。

所以那些城市的店家會把售價提高然後給你折扣，其實他還是貴。但是Kashan不一樣，老闆開給你的價錢都是最實惠的，所以你拿到的折扣不會多，但還是比你在大都市買便宜。

我心裡也知道似乎是這麼一回事，但這張開價二千美金的地毯還是超出我的預算與負荷。或許哪天我存夠錢了，有機會再帶她回家吧！

[ 圖：Kashan地毯店，全絲質，無論用色、手工、圖案皆屬上乘的地毯 ]

[ 圖：Kahsan地毯店，這是一張羊毛混絲質的大型地毯，開價三千美金 ]

[ 圖：Kashan地毯店，看似圖畫，其實都是地毯編織而成，手工之細膩令人讚歎 ]


來到伊朗，預算夠的話真的可以帶條地毯，這可是伊朗傳承至少2500年以上歷史的波斯地毯哪！

黃銅/紅銅工藝品。

伊朗許多城市都可看到銅製器具，小從銅杯大至銅鍋，各種尺寸、款式，而且價格比起日本或法國製造便宜非常多。但是礙於攜帶的方便性，最後只帶了小銅鍋、銅杯和銅盤。

本來想帶小茶壺，但有人一直阻止我…

手工打上的圖案，這一個盤子不用9美金啊！

伊朗的手工藝品非常有名，許多店裡都可看到工匠們直接在現場繪製圖案或手工打造。

我一路上一直看到搪瓷彩繪，本來只覺得漂亮，沒很動心要買。

但是走著走著，竟然走到了巴扎里的手工藝區（伊朗的巴扎或店鋪很有趣，相同類別的幾乎都會在同一條路上），我才發現這些全都是手工一個一個繪製的！

我原本以為只有這個工坊是這樣，後來才發現是所有的店裡賣的都是手工一筆一筆畫出的，仔細看便會發現這些東西圖案沒一個是重複的！真的是純手工繪製啊！




一樣因為攜帶問題，最後我只能買個盤子回家，這一個大約台幣500。一想到這個手工，根本捨不得殺價啊！而且老闆直接拿鑰匙在上面刮，完全沒掉色也沒刮傷，非常堅固耐用。只可惜，後來想多買，但離開伊斯法罕後就再也沒見到了。

除此以外，我在伊斯法罕還買了個小磁磚，準備當成隔熱墊用，還買了個時鐘。伊朗的時鐘都好漂亮，但只有這個造型最簡單的比較好帶，只好帶這個回家。

民族風在伊朗無所不見，我還買了個側背包和化妝包，價錢都十分便宜。

在伊朗餐廳吃飯，他們都是坐在床上吃飯，床上會有靠枕、地毯，上菜時會鋪一條桌巾防臟，所以我也買了一條桌巾

不實用的裝飾品只買了這些，真的好恨時間不夠啊啊啊啊～～～～～

吃的東西，不能錯過的就是蕃紅花。（蕃紅花有分等級，關於蕃紅花的知識請可以點擊右側文章鏈接，了解更多番紅花知識伊朗藏紅花比黃金更金貴）

市場里、雜貨店裡其實都買得到蕃紅花，但是品質良莠不一，尤其是雜貨店裡的更是不能期待品質。

我的蕃紅花是下大雪被迫退回亞茲德那天，同是天涯淪落人的香港人推薦我一個司機，他說司機帶他去買到品質很好的蕃紅花，而且價錢很便宜。

這段故事很長，總之，我很幸運地竟然碰到同一位司機，我也同樣要求司機帶我去買蕃紅花。

賣我蕃紅花的是司機的朋友，本身就有種植蕃紅花，所以才能以便宜的價格直接販售給我們，而且是A級Sargol不帶黃色的蕃紅花。

打開便聞到那奇特的香味，拿出一根放在舌尖，香氣直撲鼻腔，嘴裡那股香味即便過了十分鐘也不散，而且舌尖呈現蕃紅花特殊的橘紅色。

至於價錢，我只能說，是台灣購物網站的1/10而已…

除了蕃紅花，司機在車上還請我們吃玫瑰糖。Kashan是玫瑰城市，城市裡到處都是玫瑰蒸餾水工廠，伊朗人又特愛吃甜食，而這玫瑰糖便是Kashan的特產，我在其他城市並無見到。

玫瑰糖一吃下去，滿嘴都是玫瑰芳香，還能吃到乾燥玫瑰花瓣，真是一種充滿少女心的糖果啊！

伊朗盛產各式堅果，以堅果做成糖果也是當然要帶回的紀念品，我帶了兩種。至於圖片下方的黃色糖棒，是在參加tour時司機示範，我們才知道原來伊朗人是這樣喝紅茶。

平常他們會去店裡買糖塊，喝茶的時候攪拌；但是外出時，他們會買這種像棒棒糖的，放杯子里慢慢攪拌。

伊朗人非常愛喝茶，到店裡便會端出一杯紅茶。甚至某天早上，我家先生外出拍照，還被拉到旅館裡喝一杯茶（是真的喝一杯紅茶XD），只可惜這東西帶的過程中難免會壓碎……

每次出國我對這種方便的義大利面都很感興趣，但因為對伊朗的錢很搞不清楚，倉促間只各買一包，回旅館後發現很便宜，想多買卻再也沒時間去。

回家後發現這個是伊朗製造的就更後悔沒多買個幾包…都怪某人，我要買的時候一直皺眉頭，說沒吃過不要亂買>」<

在Kashan這個玫瑰城市，除了玫瑰水外，玫瑰相關產品也不少。有乾燥玫瑰可泡茶，當然也有玫瑰香水、玫瑰油、玫瑰乳霜等等，味道都十分天然舒服，而且價錢十分便宜。



除玫瑰外，還有許多其他香料，如茉莉、薄荷等等，是愛好香氣、花草茶者的天堂。

另外還有街上到處可見的堅果行，我們買一大包開心果、瓜子，路上慢慢嗑著吃，價錢依然比台灣便宜許多。核桃也是，帶殼的、去殼的，應有盡有。

沒了，我就買這樣而已了…

真的買不夠啊啊啊啊啊～～～～～～～～～～～～～～

後來有剩錢，但已經沒時間再逛，連我家那不愛逛街的另一半都說好可惜，伊朗的東西真的很有特色又很好買。

以下都是隨拍，有興趣的人以後可別忘了多留點時間逛街啊！
