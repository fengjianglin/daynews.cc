埃及男子在家裡挖出法老雕像，考古部門到場後，通知警察逮捕了他

天天要聞
2019-12-16

埃及一男子最近遇見了糟心事兒，他在自家後院挖出法老雕像，原本是個意外地驚喜，但沒想到卻給他惹來一場「大災禍」。

這位又幸運又倒霉的男子今年62歲，家住在埃及吉薩市的米特·拉西納村（Mit Rahina），這個村莊不一般，離它不遠處就是古埃及孟菲斯神廟的遺址。上個月，老頭家後院積水，挖渠排水時，一個巨大的石頭出現了。

老頭原本以為能將石頭鏟掉，沒想到它實在是太大了，堪稱巨石，而且挖著挖著石頭上竟然有一張臉，接著往下挖，老頭才意識到這是塊巨大的法老雕像。挖出法老雕像的消息，在村子裡不脛而走，也傳到了當地文物部門的耳朵里.

考古隊立刻前往了老頭家，對現場進行了全面發掘，隨後又有19塊石塊被挖了出來，這些石塊上刻著各式各樣的儀式場面，研究人員推測，雕像所刻畫的是埃及法老拉美西斯二世（公元前1303年至公元前1213年）。

雕像由紅色花崗岩雕刻而成，雖是個半身像，但是其高度足足有105厘米，寬55厘米，厚45厘米，重要的是其頭上戴著「卡（ka）」。「卡」在古埃及信仰中具有神奇的力量，人從生到死都有「卡」的陪伴。

「卡」即一個人的生命和靈魂，象徵著力量、生命力和精神，古埃及人相信，人在去世之後，會以「卡」的形式繼續存活。在雕塑中，「卡」則是一種戴在頭上的飾品，呈U字型，就像是高高舉起的雙臂。

此前埃及只發現過一個戴有「卡」的雕塑，那個雕塑為木雕全身像，1894年出土於一座墓葬中，所雕刻的形象為埃及十三王朝法老賀爾（約公元前1750年），木製雕塑現在保存於埃及開羅博物館，被視作鎮館之寶。

其餘石塊上雕刻的儀式場面據說是一種名為「赫卜賽德（Heb Sed）」的節日慶典，埃及法老在位滿30年就可以開始舉辦，每3年一次，拉美西斯二世公元前1279年即位，公元前1213年去世，在位共66年，一共舉辦了14次赫卜賽德，它們全都被記錄在了石刻之上。

此次發現拉美西斯二世雕塑的老頭，可以說是「國寶」的發現者，但是在埃及，未經授權的挖掘是非法的，相關規定表明，任何人私自將文物挖出，都會使文物面臨被破壞和買賣的危險，老頭也因此被捕。

對埃及人來說，發現文物後，不能自己動手挖，只有第一時間將發現文物的消息通知給相關部門，才能免於牢獄之災。
