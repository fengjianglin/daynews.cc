印度新政受到反對，莫迪面臨嚴峻考驗，已有6人死亡

天天要聞
2019-12-18

眾所周知，印度是世界上人口第二多的國家，而這些人口中不僅包含合法公民，也包括非法移民。針對這一情況，印度政府近期頒布了一項「公民身份法案」，該法案規定2014年12月31日之前，來自孟加拉、阿富汗、巴基斯坦三國的非法移民可以獲得身份。對此，莫迪表示支持，並稱這展現了印度的包容性，是具有標誌性的一天。但實際情況並未與莫迪設想一樣，北部城市發生了大規模的抗議，並且影響範圍在逐漸擴大。

據環球網報道，截至16日凌晨已經有6人因這場暴亂死亡，其中4人被警察開槍打死。而暴亂最嚴重的時候有數千人在首都抗議示威，示威者打砸搶燒製造暴力事件已經讓不少城市陷入混亂。為了控制事件政府已經派遣士兵進駐，而外圍更是有數千名士兵整裝待發。除此之外，示威者甚至將抗議的橫幅拉到了英國倫敦印度官員辦公樓前，另外日本首相安倍晉三和孟加拉外長也因印度局勢延期訪問，此事的影響已經越來越嚴重。

之所以會有如此多的人反對，是因為這個法案存在一定的局限性。據了解，這份法案雖然讓許多非法移民獲得了身份，但並不是所有的移民，所以剩下的一些移民表示不滿。對此，他們認為這個法案不公平、具有歧視，他們強烈要求莫迪做出決定，取消法案或者讓法案變得更公平。另外就是反對派影響，由於政治上的對立，反對派勢必會幹涉莫迪的政策，加上如此多民眾不滿，印度已然陷入了危機。

據悉，面對不斷失控的局勢，印度內政部在暗示可能會對法案進行修改。這可以看作是取得了階段性的勝利，這種情況實屬罕見，要知道莫迪可是除了名的鐵腕，今年8月發生的事件就是最好的例子。當時政府宣布廢除憲法第370條，結束了印控克什米爾長達多年的特殊地位。此政策一經頒布，立即引起了很多人士不滿，為了控制局勢，莫迪立即派遣大量軍隊進入印控克什米爾，當地斷電、斷網持續了很久。

而要論政策的大膽，莫迪曾經還實行過廢除紙幣的政策。因為印度的貪腐嚴重，而大額紙幣更是為貪腐提供了便利，所以為了遏制貪腐就廢除了大額紙幣。現在印度的新政又受到了反對，並且聲勢浩大，莫迪面臨了一個非常嚴峻的考驗。
