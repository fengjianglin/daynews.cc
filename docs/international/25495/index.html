美前外交官：制裁“北溪2號”最後努力，是華盛頓的“絕望表現”

2019-12-23 115條評論 1,290次閱讀 249人點贊

作者：王德華

前美國外交官吉姆•賈特拉斯告訴《今日俄羅斯》，儘管美國實施了最新的制裁，但北溪2號管道仍將完工。試圖阻止其發完工的最後努力是華盛頓的絕望姿態。

美國總統唐納德•特朗普周五簽署的《2020年國防授權法案》，將對參與北溪2號管道建設的外國承包商實施嚴厲制裁。

該法案要求這些公司在被列入黑名單、簽證被吊銷、資產被凍結之前，有30天的時間來逐步關閉業務。Allseas是一家瑞士-荷蘭公司，一直在為丹麥海岸外的「前溪2號」鋪設深海管道。由於面臨制裁威脅，Allseas已經宣布暫停作業，並將逐步停止運營。

然而， 賈特拉斯認為，如果有什麼區別的話，這只不過是一個小挫折，不會對項目造成大範圍的影響。

他說，「北溪二號將會完工。我認為最新一輪的制裁和工人的罷工是一個小問題。它將完工並投入使用。」

這名外交官辯稱，美國試圖阻撓的最後努力意味著，華盛頓建制派「不能承認他們的政策失敗了」。他說，這種行為類似於一個懷恨在心的人，如果事情不順他的意，他會試圖破壞周圍所有人的情緒。

我們只有一個現實，那就是俄羅斯是壞的，這個把俄羅斯和歐洲連接起來的東西也是壞的。我們必須盡我們所能來阻止它，如果我們不能阻止它，那就讓每個人的生活變得困難。

杰特拉斯辯稱，制裁很少會產生非常有效的效果。他表示，他希望歐洲國家能勇敢地面對美國，因為美國扮演著全球警察的角色，而對其盟友的利益視而不見。

按他的說法，美國人曾說，我們想與歐洲建立夥伴關係，我們想與歐洲建立最好的關係。這是一個朋友的聲音嗎? 美國的制裁，是發出一個統治者的聲音，他對其他國家發號施令，就好像它們是附庸國或附庸國一樣。

杰特拉斯指出，美國習慣於讓盟友為所欲為，但每一次都有越來越多的歐洲人(尤其是商界人士)表示，他們厭倦了被這樣對待。他總結道:「我希望他們醒來，聞到咖啡的味道，看到自己的興趣所在。」

這條管道將在距美國數千英里外建造，五角大樓卻稱這是對國防的威脅。可能非洲的獅子也是對美國國家利益的威脅。里根總統曾說，像格瑞那達這樣的小島對美國是一種威脅，如果這個小島是一種威脅，如果一個天然氣管道也是安全威脅，那就表明美國沒有他們說的那麼強大。

有網民稱，這是對《2020年國防法案》的嚴重濫用。無論如何， 「北溪2號「都不會構成任何國防威脅，而美國正試圖讓歐盟購買美國的天然氣，從而證明了這一點。這是美國對歐盟的公開戰爭行為。如果歐盟不迅速覺醒，他們將被沼澤淹沒，他們的主權將不復存在。

美國獲得權力不僅是因為其經濟實力和由此產生的軍事實力，更主要的原因是其令人印象深刻的「軟實力」(文化欺騙)，而這種實力不僅在減少。本質上，美國是一個殘忍的自戀者，希望主宰所有人，並因此而受到崇拜。如果人們不再羨慕，不再卑躬屈膝，美國人的心態就會陷入危機，因為內心的空虛會顯露出來。自戀者除了拚命報復外，沒有其他手段可以利用。
