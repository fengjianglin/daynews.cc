英國年輕男子將頭等艙座位，給88歲退休護士，空姐被暖心之舉感動

天天要聞
2019-12-20

空姐們除了擁有較好的容貌和不俗的氣質外，她們的「見多識廣」也是很多行業很難比擬的，畢竟她們會接觸到各種各樣的乘客，和性格各異的人打交道。按理說空姐們不應該有什麼最喜歡的乘客，或是最討厭的乘客，但維珍航空的一位叫做莉亞·艾米（Leah Amy）的空姐，卻直言不諱地在個人社交媒體說出了她最喜歡的兩位乘客。

據《每日郵報》倫敦時間18日晚的消息稱，莉亞分享了一個暖心的故事，講述了一位年輕男乘客如何將自己的頭等艙座位（upper class cabin，維珍的豪華商務艙），讓給一位88歲的老人的事情。據悉飛機是從紐約飛往倫敦的，一位叫做傑克的男子和他的家人都購買了頭等艙的座位，但當傑克登機後，他直接去了經濟艙，讓88歲的瓦奧萊特 (Violet)去坐原本屬於他的位置。

空姐莉亞稱，傑克坐在了緊挨經濟艙廁所的那排座位上，而在飛行期間，小夥子一直都保持十分自然的狀態，就好像他原本就應該坐在那裡似的，而事實上根本沒有人要求他那麼做，完全是出自本心。莉亞發文稱，在自己飛過的數百班航班中，服務過足球運動員、超模，甚至還有好萊塢明星，但傑克和瓦奧萊特是她最喜歡的兩位乘客。

據莉亞介紹稱，瓦奧萊特住在倫敦，她結束了到紐約探望女兒的旅行，乘坐了這次航班回家。因為剛剛接受了膝關節置換手術，她的行動不是非常方便。瓦奧萊特非常感激傑克的善心，並且稱能坐在頭等艙是她一直以來的夢想。空姐莉亞表示，當瓦奧萊特說話時，臉上充滿了發自內心的感動。

莉亞稱在社交媒體分享傑克和瓦奧萊特的照片時，已經得到了他們的同意。很多網友為這個小故事所感動，稱傑克是真正的紳士，而瓦奧萊特能大方接受對方的善意，並且發自內心的感激，同意也值得稱讚。
