塔利班汽車炸彈襲擊對美軍為首的北約軍隊造成巨大的心理陰影

2019-12-14 124條評論 1,450次閱讀 388人點贊

12月11日，一輛裝有大量爆炸物的汽車，沖向阿富汗北約巴格拉姆空軍基地附近的在建設施並引發巨大爆炸；本起恐怖襲擊事件，已經造成了73人受傷和2人死亡，其中包括5名喬治亞軍人。塔利班已經對外宣布，對這起恐怖襲擊事件負責。

對駐紮在阿富汗的北約軍隊發動汽車炸彈襲擊，對於抵抗組織塔利班來說，似乎已經是輕車熟路的套路，一般採取汽車炸彈爆炸衝擊波以後，隨即其他人員跟上猛烈槍擊的戰術，已經被在敘利亞等地的反對派武裝和伊斯蘭國恐怖分子大量借鑒，且取得了不少的戰果。

目前，阿富汗境內發生的汽車爆炸襲擊，其實，已經難以統計了，反正汽車炸彈襲擊，造成的傷亡人數真的不在少數，即便路邊炸彈，都對美軍為首的北約軍隊造成了重大人員傷亡。

實話實說，這種自殺性的汽車炸彈爆炸，真的是慘無人道，駕駛車輛者也即襲擊者，都是抱著必死的信仰，與北約軍隊或阿富汗軍警同歸於盡。

對於包括美軍在內的北約軍隊和阿富汗軍警來說，使用汽車炸彈等爆炸物，進行各種各樣的零星襲擊戰術，雖然沒有什麼多大的技術含量，但卻是始終無法破解的戰術，揮散不去的陰影，造成了他們巨大的心理障礙。

阿富汗，真的是個多災多難的國家，不斷發生的汽車自殺性爆炸、路邊炸彈、冷槍和暗殺等行動，使得美軍和北約的控制區，其安全都根本得不到有效保障，北約的軍事事基地，塔利班照樣策劃並實施了汽車炸彈+人員槍擊的戰術，讓北約軍隊疲於應付，關鍵是精神高度緊張，不知道什麼時候汽車炸彈襲擊又來了。

美軍為首的北約軍隊雖然強大，但是在和塔利班的持久衝突中，居然占不到什麼便宜，強大的軍事進攻將塔利班政權推翻易如反掌，但低烈度的治安戰，塔利班零敲牛皮糖的戰術，打冷槍、暗殺、路邊炸彈、汽車自殺性炸彈等，長年累月下來，竟然也令美軍為首的北約軍隊也損失慘重。

本次塔利班對北約軍事基地實施汽車炸彈+槍擊組合的襲擊戰術，是在美軍與塔利班武裝的談判以失敗告終的背景下發生的，肯定是有預謀的行動，目的就是警告美軍一而再再而三的不守信用，對美軍的態度可謂十分不滿意，包括拖延撤軍時間。
