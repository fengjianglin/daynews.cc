新聞“全歐了”：1杯咖啡=3聽可樂，節日限定卻成甜蜜的負擔

天天要聞
2019-12-03

【歐洲時報網】近日，星巴克在中國市場發布了2019年的聖誕紙杯。今年的聖誕紙杯共有四款，除了人們習慣的聖誕紅杯，還有墨綠和白色設計。同時，星巴克還推出了太妃榛果拿鐵、聖誕薑餅風味拿鐵、瑞雪黑森林摩卡等多款季節限定款飲品，充滿了濃濃的聖誕氣氛。

圖為星巴克在中國上市的聖誕限定飲品。（圖片來源：星巴克中國官網）

上海界面新聞報道，限定的聖誕飲品已經成為當下人們消費儀式感的重要見證，麥當勞的McCafe、Costa等品牌都會在聖誕季相應推出限定款飲品。

這類節日限定款每年都會受到消費者的歡迎，其背後的儀式感和參與度是重要原因，尤其是即將到來的聖誕節。

聖誕節是西方文化中人們每年最期待的節日，如今在其他文化中也扮演著重要角色——它意味著家人團聚、年底的放鬆假期、交換禮物等各種美好事物，也是幸福感的來源。

當消費者手持一杯帶有特殊裝飾的節日紙杯和限定款的甜蜜飲品時，立刻就有了節日氛圍，幸福感油然而生。這幸福感源自一杯街邊便能買到的咖啡，又如每年英國百貨John Lewis的聖誕廣告片或是美國梅西百貨的聖誕大遊行一樣，對大多數人來說，都是觸手可及的能產生節日幸福感的方式。

圖為位於英國倫敦海德公園的2019「冬日仙境」聖誕市場。（圖片來源：新華社）

但是，當人們享受節日帶來的幸福時，是否會意識到，這類飲品的含糖量已經「超標」了呢？

BBC報道，英國慈善組織Action on Sugar對超過200款飲品進行分析，發現其中多數產品的含糖量超乎想像。

其中，使用燕麥奶製作的帶奶油的星巴克大杯裝（venti）經典焦糖熱巧克力的含糖量最高，包括93.7克的糖和758卡路里，相當於3聽可樂的含糖量。聖誕限定款薑餅拿鐵的含糖量排在第三位——含56.6克的糖和523卡路里。

Action on Sugar表示，因為缺少標識，消費者大多是在不知不覺中選擇含糖量高的飲品；而且很多消費者認為選擇純素的配料更健康，但事實上，有些商店使用的是預先加過糖的牛奶替代品。

季節限定咖啡飲品含糖量排行榜上包括了星巴克、Costa、Pret、麥當勞等商家。（圖片來源：BBC網站截圖）

其實，改變一些配料便會有很大的不同。例如，將星巴克薑餅拿鐵中的燕麥奶替換成杏仁奶，這杯飲品的含糖量將減少18克。

營養學家Holly Gabriel表示，這一調查結果令人震驚。咖啡店必須進一步減少飲品中的含糖量，增加低糖產品的使用，不要放任消費者「盡情放縱」。

星巴克的新聞發言人則表示，顧客可以自行定製飲品的規格，包括是否使用低脂牛奶、少加或不加奶油等。同時，星巴克自2015年就一直致力於在所有產品中減少含糖量，尤其是含糖量較高的香草、焦糖和榛子產品。

近年來，減少糖的攝入已經成為追求健康飲食的共識，有些國家甚至為了國民健康加征糖稅。聖誕季即將到來，聖誕限定飲品是否成為了你的「甜蜜負擔」？
