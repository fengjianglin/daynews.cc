世界上用太陽來燒烤“第一人”，一本萬利賺大錢，每天只賣6小時

天天要聞
2019-12-14

嘴大吃天下，燒烤，想必是夜宵的必備選擇。木炭則是燒烤必不可少的用料，當然還有現在比較流行的電烤，那你有沒有想過還可以用別的能源代替呢？

泰國的一對老夫妻想到了，而且還是無污染，一本萬利的辦法，就是利用太陽能源。

這位開著燒烤店的老人，一直以來都用著傳統的木炭來製作燒烤，可是後來由於木炭價格上漲，導致生意難以維持，經過一番思索後，老人突發奇想，想到了利用太陽反射熱量的這種既環保又無需成本的辦法，因此還獲得了碧武里皇家師範大學的科學榮譽學位，成為世界上用太陽來烤雞的「第一人」

這款太陽能燒烤裝置，是他自己設計的，使用了一千多塊鏡子來進行安裝，也可以看出是多麼龐大的一個「反光鏡」。

燒烤過程中，這些鏡子利用陽光反射的原理，大概能夠收集到與烤箱同等的熱量，12分鐘就可以烤熟一隻雞，並且更加的健康。

烤雞過程中，需要根據太陽的位置變化，不斷調整反射鏡和烤雞的角度，以此保證它們能均勻的接受陽光的照射。 當然，這就是我覺得唯一不太方便的地方。不過，細想下來，炭烤與電烤也是需要這樣的操作。

這對夫妻的燒烤攤每天能夠烤40隻整雞和一些牛排，生意非常火爆，每天只開工6個小時，想要品嘗得要先預定才能吃到這有限的「供應」。

當然，這個對於太陽的收集也是有一定的危險度的，會給我們脆弱的眼睛造成傷害。圖中就是老人用手保護著面部，不去看反射板。也有網友表示：這麼做很危險，接近烤架必須有安全措施，否則容易燙傷，甚至把眼鏡烤瞎。然後就是效率不高，主要是因為操作複雜，溫度又不能準確調控。唯一的好處只是只要噱頭。個人認為不如用，太陽能板轉化成電能，再用電能燒烤。不會有光污染，安全，操作方便，溫度可調。

而前來品嘗太陽能烤雞的食客吃後表示：烤雞香嫩的表皮把可口的肉汁都鎖在裡面，口感更加嫩滑，有別於傳統的炭烤與電烤。你們怎麼看這種燒烤的方式呢？
