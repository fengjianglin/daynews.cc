男子在婚禮前4天出門送請柬，不幸遇車禍離世準新娘只等來葬禮

2019-12-07 176條評論 1,248次閱讀 165人點贊

結婚對很多人來說都是一件值得高興的事情，兩個家庭一起商討細節，認真準備所有需要的物品，然後在大喜的日子裡與親戚朋友熱鬧慶祝，共同創造美好的回憶。可是在越南義安省，一名準新郎在送請柬的路上遭遇車禍，不幸離開了人世。

據越南媒體12月7日報道，Dau Xuan Th今年27歲，來自越南義安省，與未婚妻Ngoc計劃在12月8日舉行婚禮。然而，Dau Xuan Th在婚禮前4天約朋友見面並準備去送請柬，沒想到在路上遇到車禍，因傷勢過重離開了人世，準新娘只等來葬禮。

事情發生在義安省南壇縣Nam Linh公社。男子Dau Xuan Th與未婚妻Ngoc戀愛多年，感情甜蜜。在雙方父母的支持下，兩個年輕人將婚禮定在了今年的12月8日。他們原本就決定在今年冬天舉行婚禮，也早早開始準備，思考婚禮的細節流程，購買需要的物品。

只是之前沒有定下準確的日子，所以在準備的過程中沒有通知親戚朋友。好不容易把婚禮日期定下來，眼看就沒有多少時間了，Dau Xuan Th只能通過見面送請柬或打電話的方式挨個通知親戚，表明自己的歉意，並且希望大家都能到現場一起慶祝。

12月4日，離婚禮只有4天了，Dau Xuan Th想到自己雖然打電話通知了幾個關係很好的朋友，但還是應該在婚禮前見他們一面，給他們送去正式的結婚請柬。那天晚上，Dau Xuan Th與朋友們約好見面的地點，然後把請柬揣在兜里，騎著摩托車開心赴約。

不幸的是，Dau Xuan Th騎車經過Nam Linh公社9號村的路邊時發生車禍，他受到很重的傷，雖然被送到醫院搶救，但還是離開了人世。當時準新娘Ngoc通過電話得知Dau Xuan Th出車禍的消息，連忙讓親人騎車送她去醫院。

因為兩個人都比較著急，Ngoc乘坐的摩托車在去醫院的路上也遇到了車禍，其中Ngoc只是輕微擦傷，載她的親人則是腿部骨折，也需要住院治療。目前兩個家庭都非常悲傷，他們打電話取消了定好的婚禮，還向親友們傳達了Dau Xuan Th離世的噩耗。

每個聽到消息的人都非常難過，幾乎沒有人遇到過類似的情況，明明離結婚只剩4天了，準新郎和準新娘卻接連遇到2場車禍。現在Dau Xuan Th的葬禮正在進行，他的父母和Ngoc都悲痛欲絕，始終無法接受現實。

消息傳出後，有網友說，Dau Xuan Th和Ngoc太不幸了，眼看著就可以舉辦婚禮，開始全新的婚姻生活，這下只剩下無盡的悲傷了。還有網友說，越著急越容易出錯，尤其是行車途中，很容易因為大意出問題。我們在同情Dau Xuan Th一家的同時，只能更加註意，保護好自己和家人的安全。
