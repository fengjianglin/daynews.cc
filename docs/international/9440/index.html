在印度尼西亞，為了可觀的利益他們將槍口對準了大猩猩

2019-12-03 715條評論 2,723次閱讀 820人點贊

在印度尼西亞蘇門答臘島和婆羅洲島上，雨林密布，大猩猩們在此生活，仍然生活在野外的有約6600隻猩猩。

照片里這隻14歲的雌性大猩猩，正在猩猩保護計劃中心等待體檢，他被發現氣槍金屬顆粒嵌入他體內的棕櫚油植物中。

在過去十年中，這裡的野生猩猩種群減少了約50％，人類行為給大猩猩們帶來了生存危機。

特別是焚燒森林的行為，讓大猩猩們賴以生存的環境被破壞，無家可歸。

因為印度尼西亞是全球棕櫚油生產大國，為了給棕櫚樹更大的種植土地，他們選擇焚燒雨林。

這是森林滅火隊的隊員，他正在撲滅這些人為的森林大火。在印度尼西亞，五分之四的熱帶雨林正在消失，並被棕櫚油種植園取代。

森林毀滅，大猩猩們流離失所，所以常常被偷獵者抓住，被送到市場上出售。

這隻小猩猩是蘇門答臘猩猩保護組織在交易市場上發現的，目前，該組織已經幫助280多隻猩猩從棕櫚油種植園，偷獵者和寵物主人手中獲救。

這組作品來自德國女攝影師桑德拉（Sandra Hoyn），她專註於社會，環境和人權問題，感謝她帶來這些真實的照片。
