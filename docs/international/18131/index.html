馬來西亞前總理保鏢：總理曾下令殺害蒙古女模特

2019-12-18 738條評論 2,115次閱讀 186人點贊

封面新聞記者 湯晨 編譯

據路透社報道，2015年，一名叫阿齊拉的馬來西亞警官因殺害一名蒙古女模特沙裡布而被判處死刑。近日，阿齊拉指控前總理納吉布·拉扎克下令讓他殺死這名女子。

2015年，時任總理保鏢的阿齊拉和另一名警察希魯·烏瑪因殺害28歲的蒙古模特沙裡布被判處死刑。

2015年阿齊拉與希魯被判死刑

納吉布在去年的馬來西亞的大選中落敗，如今他面臨大規模的腐敗指控。納吉布否認認識這名女子，但究竟是誰下令殺害這名女子的問題，一直沒有得到官方回答。

馬來西亞新聞門戶網站Malaysiakini周一報道稱，阿齊拉在一份法庭文件中表示，納吉布命令他「逮捕並消滅」沙裡布。

被殺害的蒙古女模特沙裡布

「我問他(納吉布)說的『逮捕並消滅』外國間諜是什麼意思，他回答說:『開槍殺死』，並以『割喉』的手勢示意。」阿齊拉在庭上說。

阿齊拉的律師庫爾迪普庫馬爾周二證實了這一報道，但拒絕進一步置評。馬來西亞聯邦法院將於明年4月20日審理阿齊拉的指控。

前總理納吉布否認了這些指控，並指責馬來西亞政府利用阿齊拉策劃針對他的政治攻擊。

納吉布在他的臉書主頁上說:「有人告訴我，這個新故事將為政府提供一種不保釋就逮捕和監禁我的方式，因為謀殺案件不允許保釋。」

納吉布·拉扎克

2006年，蒙古女模特沙裡布在吉隆坡附近的森林中被擊斃，她的屍體被軍用級炸藥炸毀。

馬來西亞民間社會團體稱，沙裡布的被殺與她在2002年馬來西亞從法國造船巨頭DCNS購買兩艘潛艇時，為納吉布的前副手阿卜杜勒•拉扎克擔任翻譯有關。2008年，阿卜杜勒·拉扎克被判無罪，他曾說自己與沙裡布有染。

時任總理副手阿卜杜勒•拉扎克

沙裡布家人的律師呼籲針對阿齊拉最新的指控，政府應對的沙裡布死亡進行新的調查。

本案除阿齊拉外的另一名被告希魯·烏瑪在判決下達前不久逃到了澳大利亞。希魯去年表示，如果他能得到完全的赦免，他將會配合任何新的謀殺案調查。
