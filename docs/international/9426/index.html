巴勒斯坦如何看待內塔尼亞胡或將下台，稱以色列誰上台都一樣

2019-12-03 399條評論 1,621次閱讀 546人點贊

一位巴勒斯坦專家認為，即使本傑明·內塔尼亞胡離開以色列的政治舞台，和平仍遙遙無期。不僅是因為以色列沒有產生遠見，而且還因為巴勒斯坦人陷入了自己的政治混亂，以致他們無法集中精力解決以巴衝突。

以色列總理本傑明·內塔尼亞胡有30天的時間要求以色列議會的豁免權，這將保護他免受審判。

內塔尼亞胡被指控接受一個富有捐贈者的非法禮物，並為自己購買了正面報導以換取逃稅，內塔尼亞胡否認這些指控。

別無選擇

但總部設在拉馬拉的巴勒斯坦政策研究和戰略研究中心總幹事哈尼•馬斯里表示，內塔尼亞胡可能受到的審判並不是值得慶祝的理由。

他說：「內塔尼亞胡是一個危險和極端的人，但他不會沒有戰鬥就離開。他在利庫德集團很有實力，甚至計劃參加下一輪大選。」他指的是最早可能在2020年2月舉行的選舉。

然而，多年來一直關注以色列政治的馬西里認為，新一輪選舉不會導致陷入僵局的巴以和平進程取得進展，僅僅是因為以色列目前的政治版圖沒有一個尋求和平、希望有所作為的領導人。

他說：「甘茨可能比內塔尼亞胡強，但他沒有別的選擇。事實上，兩者觀點相同，政策相同。兩人都計劃吞併巴勒斯坦領土。」他指的是內塔尼亞胡在選舉前承諾吞併約旦河谷，甘茨支持這一舉動。

這位權威人士認為，其他政客，包括那些在以色列政壇被視為左翼政黨的梅雷茨的政客，也不能成為和平的夥伴。

馬西里說：「梅雷茨已經和埃胡德·巴拉克合併，他並不完全以和平努力聞名。」

巴拉克是以色列最得意洋洋的指揮官之一，他在2008年以色列「鑄鉛行動」中扮演的角色最為人們所銘記，當時他作為國防部長，在加沙殺害了1000多名巴勒斯坦人，其中許多人是平民。

2010年，他被指控對以色列襲擊一批前往加沙的國際人權活動人士負有責任，這批人乘坐的船隊旨在打破以色列對哈馬斯控制的飛地的封鎖。

呼籲團結

因此，馬西里認為，巴勒斯坦人除了自己之外，沒有其他人可以依靠。

他說：「我們的軍事抵抗未能實現建立巴勒斯坦國的目標。權力的平衡不在巴勒斯坦一方，因此在軍事上對抗以色列是沒有意義的，外交努力也是徒勞的。這就是為什麼我們需要重新考慮我們的戰略，走一條不同的道路。」

馬西里認為，這條新道路將包括抵制以色列、國際壓力和各種宣傳工具，有助於向國際聽眾解釋巴勒斯坦的立場。

但是，在巴勒斯坦人開始採取這些步驟之前，馬斯里認為他們應該首先克服自己的分裂。

他強調：「首先，法塔赫和哈馬斯需要認識到，他們沒有多數人可以單獨執政，因此他們需要彼此維持下去。其次，吞併約旦河谷不僅是生活在西岸的巴勒斯坦人的問題，也是一個民族悲劇。以色列對加沙的圍困不僅是加沙人的問題，也是所有巴勒斯坦人的一大難題。這就是為什麼我們需要這種團結，向以色列和國際社會施加壓力，以承認我們的事業。」

儘管困難重重，馬西里仍然樂觀。他說：「團結不是一種選擇，而是一種必要。至於以色列人，他們也需要改變，我相信他們最終會選出一位領導人，希望與巴勒斯坦人實現和平。但這不會在一夜之間發生。實現這一目標需要數年時間。」
