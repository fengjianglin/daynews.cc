遊行示威浪潮讓瑞士大城市商家不堪重負

2019-12-17 1,084條評論 4,204次閱讀 2111人點贊

瑞士主流媒體「瑞士資訊」日前發表題為《民主的不滿？示威浪潮讓瑞士不堪重負》一文指出，瑞士遊行示威持續增多，這不僅讓瑞士居民、上下班通勤者以及外來遊客深受其擾，還間接威脅到各大城市無數商家的興衰存亡。

該報道說，瑞士居民比以往任何時候都更樂意走上街頭遊行抗議，預計首都伯爾尼將迎來前所未有的一年300場街頭集會示威記錄；當地居民和通勤族對此頗為不滿和反感，商家更是憂心如焚。比如，讓無數人期盼的「黑色星期五」本是一年中最佳商機之一，消費者想「血拚」一把，商家想賺得盆滿缽溢，沒想到被遊行示威攪黃了。今年「黑色星期五」當天，瑞士第一大城市蘇黎世警方警告市民，從下午4點開始即交通高峰時段，因舉行氣候變化示威遊行，警方將在老城區以及蘇黎世湖畔周邊區域實施交通管制。警方還「溫馨提示」，因有軌電車屆時將繞行，忙著去購物的民眾應該改道步行。這場抗議活動剛結束沒多久，新的一場未遵循固定路線前行的自行車遊行又開始了，直接導致交通堵塞。次日，蘇黎世又爆發一場涉及他國政治事務的抗議遊行。在那個「黑色星期五」周末，整個蘇黎世市區交通被不少於兩場抗議示威活動攪亂。


圖片選自「瑞士資訊」網站

另據瑞士《周日報》日前發表的題為「瑞士城市在示威浪潮中呻吟」報道所述，瑞士首都伯爾尼受遊行示威影響最為嚴重；在伯爾尼狹窄街巷上，平均每個周末會發生6場抗議示威活動，有些周末甚至會同時發生4場遊行示威。該報道指出，瑞士遊行示威持續增多，2018年首都伯爾尼全年抗議示威次數達299次，今年截至10月底已累計達280次。報道強調，頻繁的遊行示威不僅讓瑞士居民、上下班通勤者以及外來遊客深受其擾，還間接威脅到各大城市無數商家的興衰存亡。蘇黎世一家時尚女裝品牌店員工在接受媒體採訪時表示，該店經歷了30年以來首次周六閉門歇業；周六是一周中服裝店客流量最高、生意最繁忙的日子，如今顧客們因為周六街頭遊行示威頻現而不願上街購物。蘇黎世老城商業協會會長、瑞士一家百年巧克力品牌旗艦店老闆米蘭告訴媒體記者，大量遊行示威給鬧市區旺鋪造成數十萬瑞郎的高額損失；經常性的道路封鎖、電車改道勢必讓很多人覺得周末來城裡購物是件麻煩事，人們有時甚至擔心街頭示威可能會演變成暴力事件。

「瑞士資訊」報道文章指出，並非所有遊行示威組織者都會依照程序向官方申請遊行示威許可，也並非所有遊行示威者都會循規蹈矩，有些遊行示威組織者就期待出現諸如交通堵塞之類的城市混亂效果。對此，「瑞士資訊」引述《周日報》一篇社論稱，（瑞士）絕大多數抗議示威活動沒有任何效果，因為遊行者所表達的大部分訴求已是眾所周知議題，或者實際上完全能夠藉助於其他方式得到妥善解決。社論強調，「現在已經到了瑞士政府部門收緊發放示威許可，嚴令禁止城市中心區域遊行示威，並對未經授權的遊行抗議活動施以罰款的關鍵時刻。」

據了解，瑞士遊行示威的主題五花八門，不僅突顯瑞士國內政治經濟社會各種問題，還緊跟國際形勢及熱點，呈現訴求趨於雜多散亂、頻率居高不下和擾民現象日益嚴重的趨勢。近年來，氣候變化、人權問題、難民問題、男女平等、同性戀權利和動物權利等，成為遊行示威訴求熱門主題，一些小眾議題也開始登台亮相。很多遊行示威陷入擾民擾商討嫌、當地民眾不知曉和不具實際意義的尷尬境地。筆者今年10月初赴蘇黎世採訪時，在當日交通高峰時段，突遇一場遊行示威。當時，筆者乘坐的有軌電車途中突然停駛，司機在廣播里說，由於前方有遊行示威，電車不走了，所有乘客須下車自行解決交通問題，而且不知道何時能恢復通車。筆者問周圍乘客，什麼人為什麼遊行示威？大家都聳肩搖頭說不知道。當筆者無奈改為乘坐計程車繞行時，筆者向計程車司機提出同樣問題，司機也說不知道。試想，一場造成市區交通大堵塞的遊行示威，連消息靈通的計程車司機都不知情，可見這座城市遊行示威次數多到何等程度！
