歷時七年修復，美國空軍受損F-22終於重新首飛

2019-12-20 1,972條評論 2,952次閱讀 1366人點贊

作為美軍現役最先進的戰鬥機，F-22的數量相較於他的性能來說相當可憐。美國空軍原計劃採購750架這種先進戰機，但由於蘇聯解體以及成本考慮，最終F-22的生產數量僅有187架，是美國空軍現役主力戰鬥機中數量最少的一型。

由於生產線在2012年就已經關閉，並且重開生產線的費用高達數百億美元。對於美國空軍來說F-22顯得極為珍貴，可謂是摔一架少一架。從2005年服役至今，已經發生了3起F-22墜毀事故，另外還有多起人為操作原因或自然災害導致F-22受損無法使用的事故。

2012年5月31日，一架編號02-4037 / TY的F-22戰機在延德爾空軍基地進行觸地復飛時，由於飛行員操作失誤，過早的將起落架收回。導致這架F-22以機腹在跑道上滑行，一直滑行了853米才停下來。而這名飛行員此前只有2次駕駛F-22的經驗，這是他第3次駕駛F-22升空，他本人也在事故中受到輕傷。

2012年發生事故的F-22

事故發生後美國空軍立即對這架受損的F-22進行評估，評估結果顯示這架飛機的機體結構嚴重受損，不僅機腹蒙皮和結構受損嚴重，同時還要更換機身中部、尾部以及部分機翼的結構件。美國空軍內部對此次事故的定性是A級事故，也就是造成100萬美元以上的損失或是造成重大人員傷亡的事故。

通常情況下這種事故發生後需要將飛機送回原廠進行大修，然而此時F-22的生產線已經關閉，相關的供應商已經不再提供結構部件。

另一方面美國空軍也不甘心將這架F-22進行退役處理，經過慎重評估美國空軍決定在自己的維修站進行修復工作。隨後這架受損的F-22被運到猶他州的希爾空軍基地進行維修，並與洛克希德馬丁公司、波音公司成立專門的維修專家小組進行維修，當時預估的維修費用為3500萬美元，相比於該機1.42億美元的造價也算是值得的。

不過這一維修過程是極為漫長的，修復專家小組先是對受損的F-22進行結構性修復，更換了內部兩個大型組件，包括機身艙壁和機翼的一部分，這些部件需要大量特製複合材料和碳纖維補丁。此後還要對戰機的功能性進行評估測試，同時還要將受損戰機升級到現役F-22同樣的版本，這些都需要大量的時間。

就在上周經過7年漫長的修復以後，這架受損的F-22由美國空軍第514飛行測試中隊的菲利普.強生少校重新駕駛升空。不過目前還沒有證據顯示這架飛機已經重新加入美國空軍服役，未來可能還將進行大量試飛工作對飛機的可靠性進行嚴重。

不過對於美國空軍來說，對於F-22的修復還沒有結束。2018年4月15日，美國空軍一架F-22在內達華州舉行的飛行表演中又發生了同樣的事故，並且這起事故是明顯的人為因素造成的。

2018年發生事故的F-22

這架F-22原本是參加美國海軍戰鬥機武器學校（也就是電影《壯志凌雲》中的Top Gun）的畢業典禮，而駕駛這架F-22的空軍飛行員想在海軍飛行員面前秀一把超機動，按當天的氣象條件F-22需要在速度達到143節時，也就是265公里每小時的情況下拉起。

但當時F-22的飛行員拉起時速度僅為136節，也就是251公里每小時，並過早的將起落架收起。由於升力不夠這架F-22重重的摔在了跑道上，滑行了6541英尺（約2公里）後才停止。

而這架F-22同樣機腹受損嚴重，無法繼續執行作戰任務，只能和2012年受損的F-22一樣進行修復。事後美國空軍對相關責任人員進行了處罰，並要求F-22飛行員今後禁止在起降過程中做這一危險動作。
