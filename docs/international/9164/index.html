倫敦國家畫廊展示出的傑出戰艦及海景繪畫作品，都是英雄的象徵

2019-12-06 2,602條評論 4,686次閱讀 2224人點贊

倫敦國家畫廊，是榮耀都市倫敦中的一處散發著濃鬱氣息的典雅的美美術殿堂。堪稱彙集了文藝復興的經典名作的藝術寶庫，其座落於美麗、宏偉的特拉加廣場北側，創設於1824年，是歐洲藝術名作彙集的寶庫。國家畫廊收藏了許多西歐著名的繪畫，包括了波提切利、達芬奇、米開朗琪羅、塞尚等人的作品。國家畫廊的收藏品按照年代分別在不同展覽室展出。所以，能在這裡展出的繪畫基本上都是大師幾級別的。

今天我們看到這些傑出的繪畫作品，都是倫敦國家美術館展示出的戰艦及海景繪畫作品，它們都是英雄的象徵。每一幅畫都是一段故事。像這樣大量的海景和戰艦繪畫作品實屬罕見。也許是為了銘記歷史吧。不過這樣的作品對於繪畫愛好者，到是很好的學習臨摹作品，希望大家好好珍惜！

國家肖像畫廊專門收藏人物的肖像畫，從15世紀到現代的都有，當然也少不了英國女王，還有黛安娜王妃，除了肖像畫之外，也不定時的展出人物攝影作品。
