1米88黑人4百年前登陸日本，引發舉國轟動！黑武士成大軍閥寵臣

2019-12-18 1,298條評論 2,799次閱讀 901人點贊

悲慘而殘酷的奴隸貿易是非洲歷史上眾所周知的一個方面。直到今天，黑人仍然成為美國一個重要而敏感的種族問題。不過很多人想像不到的是，在遙遠的16世紀，奴隸貿易竟然給日本帶來了一位非洲「黑武士」。

這位非洲「黑武士」名叫彌助，據考證很可能是當時非洲東北部丁卡族人的一員，也就是現在的南蘇丹。彌助在孩提時代就被販賣到了印度。在那裡，彌助成為亞洲首席耶穌會士亞歷山德羅·瓦利尼亞諾的奴僕，瓦利尼亞諾當時正在印度考察，然後最終目的地是日本，1579年，瓦利尼亞諾和他的保鏢兼貼身侍從彌助抵達日本。

瓦利尼亞諾在1581年對當時的日本首都京都的訪問，成為彌助在日本的第一份官方記錄。甚至彌助帶來的震動還要超過瓦利尼亞諾，因為日本人從來沒有見過渾身黑色皮膚的人。在日本在途中，耶穌會的隊伍經過港口城市酒井，大量的人群高聲嚷著要一睹非洲人的風采，導致了騷亂。建築物在觀眾的重壓下倒塌，耶穌會的遊行隊伍被嚴重擾亂。為了逃脫，彌助不得不騎馬擺脫一群著迷的當地人。

他不僅是他們看到的第一個非洲人，而且按照日本人的標準，彌助是一個「巨人」——身高188厘米，看上去英俊、聰明、有趣，而且力大無窮。而且，因為在日本的佛經中佛陀經常被描繪成黑皮膚，因此許多日本人認為他是一個神聖的訪客。這種喧鬧的接待在京都重演，數千人到場見證彌助的到來。彌助在耶穌會教堂躲著，但愛看熱鬧的日本人破門而入，一睹奇特的非洲人風采，一些觀眾甚至被壓死。

當時，日本最有權勢的軍閥織田信長正在附近。聽到外面的混亂，他下令調查到底是誰擾亂了他的平靜，當他聽說彌助的事時，命令立即接見他。那時彌助已經學會了相當多的日語，所以能夠交談了。這位著迷的軍閥甚至拿布來擦拭彌助的皮膚，看看皮膚的顏色是真的還是染上去的。發現彌助的皮膚完全是自然的黑色，高興的織田信長為這位令人震驚的客人舉辦了一個聚會，並贈送了一大筆現金給彌助，使他立即發了財。

幾天後，織田信長問瓦利尼亞諾是否可以讓彌助為他服務。最後織田信長成了彌助的新主人，後者專門攜帶織田信長的武器，這是一個非常光榮的職位，另外彌助還成為一些海外事務的顧問。幾個月後，彌助收到了一份津貼，在京都東北部的安土城有了一套房子，還配了僕人和一把武士刀。武士刀是武士的象徵，所以日本傳統上認為彌助是第一個有記載的非日本出生的武士。

在接下來的一年裡，彌助至少陪同信長參加了一次戰鬥。他們正在征服富士山以北的領土，武田家族長期控制著這片領土，武田家族是織田信長的死敵。彌助在作戰返回安土城的旅途中，在富士山以南曾被一名日本日記作家觀察到。

1582年6月，織田信長發起了一場針對他西邊的宿敵毛利氏的重大戰役。一支龐大的軍隊開往戰場，織田信長帶著大約30個人出發前往前線，其中包括彌助。在京都，他們睡在本能寺，這是彌助15個月前會見織田信長的地方。在6月21日黎明前的一個小時內，他們遭到了明智光秀13000人的的襲擊，他曾是信長最信任的將軍之一，伏兵襲擊了院落，大多數守衛者被殺死。

在戰鬥中，寺廟著火了。當火焰席捲寺廟時，織田信長切腹自殺。根據傳說，他的最後一個命令是下達給彌助，告訴這位非洲武士不要讓他的頭落入敵人手中。最後彌助逃到了織田信長的兒子和繼承人織田信忠那裡；然而織田信忠只有大約200人的援兵，叛軍很快就把他們包圍了，織田信忠也被迫結束了自己的生命。彌助被帶到明智光秀面前，後者命令他回到耶穌會士那裡。

這是日本歷史上最後一次提到彌助。在弗洛伊斯的《日本史》中記載，在沖田畷之戰中，有馬晴信的家臣中有一位善使鐵炮的黑人參戰。電視競猜節目《世界不思議發現》曾經給出說法，稱此黑人是彌助，他與有馬家的人相識，在本能寺之變後可能投奔有馬家。今天，非洲武士已經重生，成為電腦遊戲、動漫、漫畫、電影、書籍和戲劇中的一個角色。
