9名中國船員非洲入獄背後：國際紅木走私猖獗，"真兇"不斷變換主體，根本查不出；無辜船員慘被拋棄

2019-12-18 2,742條評論 4,571次閱讀 1823人點贊

《飛來橫禍：非洲監獄裡的中國船員系列報道之二》

作者|陳龍 編輯|覃旭

給獄警小費後，15名FLYING號貨輪的船員在監獄用手機拍了張合照——9名中國人、2名緬甸人、4名孟加拉人。那情景，很像100多年前美洲的華人勞工。

2018年12月18日凌晨的印度洋海面上，中國貨輪FLYING號遭到一艘小艇追擊，隨後遭到猛烈的槍炮攻擊。被俘後，貨輪上的17名船員被押往馬達加斯加圖阿馬西納港。其間，兩名關鍵船員趁治療槍傷之機偷渡回國，激怒了馬國警方，剩下15名船員被投入監獄。

被馬國政府俘獲並沒收後，停泊在Toamasina港的FLYING貨輪。在當地比較顯眼。

原來，FLYING號貨輪在2015年至2016年多次前往馬國走私珍稀木材紅木，其後更改船名，於2018年10月企圖再次前往馬國。10月至12月，他們在印度洋上謹慎窺探，逡巡不前。殊不知，馬達加斯加軍方已經布好了一張逮捕大網。

2019年3月27日，15名船員以”非法入境罪”被當地法庭判監禁5年。他們申冤無門，原本保證全力營救的國內船東，後來金蟬脫殼，外交和老家的外事、海事部門也無能為力。2019年10月13日，二審維持原判。

紅木走私是一項國際性的暴利經濟犯罪。FLYING號的福建船東商人只是國際非法走私紅木貿易網中的一個代理人，其背後，有三家背景頗深的香港、福州公司，他們狡兔三窟，藏身嚴密。船員家屬發現，在先前幾個月的”緩兵”時間裡，疑似走私集團內部密集變動資料，企圖從麻煩里隱身。

走私紅木國際犯罪

2018年12月20日早上，FLYING剛靠泊馬達加斯加Toamasina港，世界公共船舶數據網站FleetMon就發出了一則報道，稱18日在馬國東北部安東吉爾灣，軍方經過2小時追捕，攔截一艘”載有中國和迦納船員”（實際為9名中國、2名緬甸和4名孟加拉國船員，編者注）的普通貨船，並稱”該船涉嫌長期走私珍稀紅木，第一次被發現走私是在2015年1月，當時船舶是另外一個名字。”

第二天，印度海事局在Facebook轉發了該報道，同時披露此船建造於1997年，屬於香港海凌船務有限公司。當月，物流巴巴網翻譯轉載了FleetMon和印度海事局的報道，標題為《中國貨船因走私稀有紅木木材遭扣押》。

中國駐馬達加斯加大使館自始至終關注著此事。FLYING號在馬國靠港後不久，大使館就派了一名領事專門處理此事。

使館人員告訴他們：早在被抓前幾天，馬國的電視台就開始連續報道FLYING號貨輪。新聞中說，這條船2015年和2016年來馬國走私過紅木，現在他們又來了，警方接到情報，”要把他們抓住”……想起審訊時警察說的”你們這條船早就被盯上了”，FLYING號貨輪大副申文波不寒而慄。

什麼是馬達加斯加紅木？申文波和船員們都不知道，更沒有見過。入獄前後，他們詢問警察，對方都回答：這種紅木全世界只有馬國有，非常珍貴。

2009年馬達加斯加軍事政變，政局動蕩，造成紅木、烏木採伐觸目驚心，走私泛濫，原始森林面積銳減。

公開資料顯示，馬達加斯加紅木，是一種黃檀科紫檀，生長緩慢，因芳香、顏色火紅而成為製作傢具的上等材料。在中國，其價格可達每噸50萬至100萬元人民幣。

2009年，馬達加斯加發生軍事政變，之後引起國際上的經濟制裁，該國陷入政局動蕩和持續貧困。過去多年，馬國東北部的原始森林裡，數萬立方米的紅木和烏木消失。而過去的50年里，馬達加斯加的森林覆蓋率下降40%。

紅木走私，有一張龐大的全球生意網。本地伐木工、木匠、採伐商、運輸商，甚至包括國際網路黑手黨和一些馬國政府官員，無不牽涉其中。2009年，政變上台的過渡政府批准了紅木和烏木出口的決議，允許東北部採伐出口，導致砍伐與走私更加泛濫。此舉遭到國際社會廣泛譴責。迫於壓力，2010年，馬國政府頒布法令，禁止紅木、烏木出口。

2013年，新修訂的《瀕危野生動植物國際貿易公約》（CITES）將7種紅木樹種列入禁止非法開採和貿易附錄，其中就包含馬達加斯加紅木、烏木及巴黎桑。

2015年12月，馬國國民議會通過新的打擊紅木和烏木走私法，還專門成立了打擊走私的特別法庭。

但走私屢禁不絕。據估計，僅2013年，從馬國非法出口的木材價值就在2.5億至3億美元之間，大部分是紫檀木，即紅木。這些木材先被運到中間國家，如法屬留尼汪島、模里西斯等，在那裡偽造文件，使貨物合法化，然後送入中國等有龐大胃口的市場。禁令、出口量也常常攪動著市場行情，中國國內紅木傢具價格在某一年內暴漲60%。

2013年年底，福州海關截獲4起走私紅木案，涉及紅木400多噸，價值人民幣1億元以上。2014年3月，新加坡截獲近3萬根馬達加斯加紅木，約3235噸，價值5000多萬美元，創下該類案件紀錄。

2019年，僅3月的專項行動中，中國的上海和江蘇張家港海關分別查扣438噸、330方走私紅木；廣州黃埔海關更立案16宗，抓獲23人，截獲走私紅木總案值5.39億元。

但是，馬國法庭對FLYING號船員的審判中，卻隻字未提”紅木走私”，因為真正的走私犯並未被抓到，被抓的17名船員中，除船長和船東代表可能牽涉此前的走私外，其他船員都是在2017年之後才上船，而且對此行的航運目的事先毫不知情。

018年8月，申文波上船後簽訂的協議書。結尾蓋章為香港蓮華國際貿易有限公司。

實際上的走私紅木罪行指控，到了法庭變成了”非法入境”。申文波說，”把我們從公海上抓來的，卻說我們沒合法手續入了境。”

FLYING號貨船幕後疑雲

FLYING號涉嫌走私背後的承運商楊建豐（福州民豐船務）、派遣方（大連華商勞務）與香港的海凌船務公司、蓮華國際貿易公司之間，是一條怎樣的走私貿易鏈條，讓船員們捉摸不透。

“那麼大的一個案子，上線下線這麼多環節，哪個環節跟船員有關係？船東在中國，他們沒能力抓，那馬國本國提供貨物的人呢？船東在法律上，只是個承運人，砍伐人、運輸人、發貨人、收貨人，等等，你去查啊。”申文波說，”問題是，它不查了，不管了，就抓住船員。”

入獄後不久，15名船員開始自救。他們從當地一個華人餐館老闆那裡買到一部二手小米手機，聯繫上家人和船東楊建豐、巫秀青夫婦。從2019年3月至7月，家屬們到處奔走，三次前往福州尋找楊建豐夫婦。

船員大多來自山東、東北、江蘇的農村，家屬們沒多少交涉能力。前兩次去時，楊建豐夫婦熱情招待。2019年4月，巫秀青在上海一家餐廳面見了船員孟范義的兒子孟朝生、符偉剛的弟弟符瑞峰，二人作為代表前往馬國港口城市圖阿馬西納探監。

另一方面，涉嫌犯罪的每家公司都在極力”隱身”。除了兩三家香港公司身影模糊，此次FLYING號出事後，大連華商勞務公司在2019年5月緊急變更了工商數據，將註冊資金從原先的200萬變為5萬，直減97.5%，幾乎成為空殼公司。

目前的工商信息中可以查到，楊建豐夫婦曾持有一家已被吊銷的”福州通四海水產有限公司”。但歷史記錄顯示，他曾投資過”福建閩豐輪船有限公司”，該公司注資達5000萬。其妻子巫秀青則持有一家廣告公司和工藝品店。

但2019年7月，楊建豐因借款合同糾紛等被凍結了其在一家公司的股權，被限制高消費；7月22日又因債務糾紛被福建龍岩法院列為失信被執行人；10月9日他又被上海法院列為被執行人……

在馬國的監獄裡，唯一可能參與過2015年走私的，就是船長於天財，他和船東姐夫在這條船上工作了4年。出於自保，他不肯說當時的情況。一年過去了，在大家的催問下，他透露了一些信息：楊建豐在2014年買下了這條船，當時船名是MIN FENG，從2014年到2016年，他們來馬達加斯加走私過三次。第三次為2016年，紅木在香港卸貨時被香港海關查獲。”楊建豐通過關係把這事擺平了。之後這條船就開始跑小東南亞航線。”2017年6月8日，這條船在福州新遠船廠重新噴漆改造，更改為現在的名字——FLYING號。

在馬達加斯加探監的半個月，孟朝生、符瑞峰看到停泊在港口的FLYING號。”那艘船在國內只算是個小破船，在圖阿馬西納港卻很顯眼。”符瑞峰說，很多當地民眾都知道此船走私的事，議論紛紛。有一天他們碰到一位當地華人，對方拿出手機，展示了2015年某一天的照片，當時這艘還叫MINFENG的船舶正從海里吊紅木。

原來多次表示要全力救船員的楊建豐夫婦，到2019年7月後，態度突然發生180度轉變。楊建豐夫婦拉黑了大部分船員家屬，也很少再接船員們的電話。

直到這時，孟朝生才反應過來，船東之前的善意態度都是在爭取時間。出事後的幾個月，所有涉事公司都開始消除可能涉及犯罪的各種痕迹，包括可能在做財產安置和轉移。

除了中介派遣公司注資縮水97.5%，楊建豐、巫秀青夫婦位於福州市倉山區金山水岸路1號的福州民豐船務有限公司的工商信息也消失了。僅有的”民豐船務有限公司”，是一家在香港註冊的公司。

事實上，這條紅木國際走私鏈條上，的確有著福建、香港的多重外殼。民豐船務有限公司最早成立於1992年，後來註銷，2013年重新註冊。楊建豐夫婦，可能是其在大陸的代理人。孟朝生猜測，走私紅木的實際操盤者，可能是協議上蓋章的香港蓮華國際貿易有限公司。

而FLYING號真正的主人，是香港海凌船務有限公司，公司編號0982297，董事為陳彬。該公司在香港的辦公室為香港北角渣華道18號嘉匯商業大廈，實際上只是個馬甲。工商資料顯示，陳彬在大陸的福建、上海共擁有6家公司，其中5家持股比例都在95%以上。這些公司多為船務公司，且註冊資本許多都是500萬，其中兩家的注資高達1000萬和5000萬。

FLYING貨輪近景，約攝於2016年。圖片來自公共船舶數據網船訊網。

一個偶然的機會，孟朝生髮現，2015年1月，另一艘中國貨輪閩泰冷號（MIN TAI LENG）在馬國東北部地區涉嫌走私紅木也被馬海軍扣押。當時，中國駐馬大使館曾通報這一消息，並稱閩泰冷號（MIN TAI LENG）屬海凌（香港）船務有限公司——與FLYING號同屬一家。可是如今再查，這艘IMO號8627309、掛蒙古國國旗的MIN TAI LENG，不知何時已改名為SHUI JING LING。

“這些紅木走私進來，渠道、銷路，肯定有一連串的關係，但是沒有人去查。”孟朝生感到遺憾——在龐大的走私系統中，船員是最表面的一層，他們大部分只是第一次登上FLYING號，與該船的前科毫無關係，大部分人也不知道此行的真實目的，在航運過程中還一度質疑和抵制來自船東的不合理指令。

但如今，”真兇”隱身潛伏，作為替罪羊的船員則在異國監獄裡煎熬，公正和自由遙遙無期。

（文中孟朝生、符瑞峰為化名。）
