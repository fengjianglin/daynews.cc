各國招待“貴賓”的美食，中國和迪拜最大氣，印度的最“奇葩”

2019-12-23 106條評論 1,900次閱讀 986人點贊

每個國家的生活環境不同，喜歡吃的東西也都是不一樣的，我們中國的美食種類比較多，但是平時用來招待貴賓的話，一般都會挑選自己認為比較好的東西，才能夠體現出誠意來，每個國家來招待貴賓的美食都是不一樣的，這幾個國家，這最具有代表性的，其中中國和迪拜的是最大氣的，印度的可以說是最奇葩的，還有一個美國的，雖然看上去好，但是吃起來卻不怎麼樣，不過這種美食在美國的地位就很高。

首先看一下迪拜用來招待貴賓的位置，我們都知道迪拜是一個沙漠國家，他們那裡有很多的駱駝，一般人招待貴賓的時候，一般都是用烤駱駝招待的，而且還是烤一整隻駱駝，在駱駝的體內，會放一些雞鴨魚肉這些東西，如果在迪拜別人用這個招待你的話，那你想用的就是最高的待遇了，實在是太大氣了，想想都要流口水。

還有就是中國用來招待貴賓的美食，尤其是夏天的時候，吃燒烤的比較多，如果別人用烤全羊來招待你，說明你在他的心目中地位是很高的，一整隻羊放到火上，烤過之後，外焦里嫩，吃起來非常好吃，雖然沒有迪拜的烤全駱駝霸氣，但是看上去也是很大氣的，所以夏天烤全羊是最受歡迎的。

接下來就是美國用來招待貴賓的美食了，在美國人的心目中，火雞的地位是很高的，在重大節日的時候，美國人一般都會製作一隻烤雞，如果他們用這個來招待你的話，說明是把你當成貴賓來看待的，只不過美國的烤火雞，看起來色香味俱全，但是吃起來，裡面的肉一點味道都沒有，是非常難吃的，不要是因為他的個頭太大，每個人還堅持一整隻烤。

最後就是印度用來招待貴賓的美食了，在印度鴕鳥是一種稀有動物，印度人把鴕鳥蛋當成了一種珍品，不過印度人卻不擅長烹飪，他們都是直接把鴕鳥蛋放到鍋里煮，煮過之後，直接就可以吃，很多外地人都不知道怎麼下口，這種招待貴賓的美食，實在是太奇葩，你們覺得呢？
