中俄貿易額破千億後，又出現小插曲：對俄魚類出口檢出質量問題？

天天要聞
2019-12-20

目前，中國穩居俄羅斯第一大貿易夥伴的地位，並且在今年前11個月內，雙方的進出口總額再次突破1000億美元，預計今年全年總額可達1100億美元，再創新高。中俄的經濟合作無疑正在朝著不斷擴大深化的方向前進。不過近日，卻發生了一段「小插曲」。

據俄羅斯媒體20日消息，俄羅斯聯邦動植物衛生監督局發布消息稱，在從中國出口至俄羅斯的魚類和海鮮中檢測出質量不達標的問題，如果該趨勢持續，俄羅斯將考慮對上述產品的進口實行臨時限制。今年7月，俄羅斯也曾暫停對中國梨果類和核果類水果的進口，理由同樣是質量未達標。

自數年前俄羅斯暫停從歐盟、加拿大、挪威等國家進口食品以來，俄羅斯一直致力於提高本國的農產品產量，以降低對進口食品的依賴，而中國則在此期間填補了俄羅斯市場的部分空缺，水產品也不例外。僅在今年內，俄羅斯就先後在2月、9月共計批准了16家中國水產品企業的對俄出口資格。不過與此同時，俄羅斯對水產品的進口標準也相當嚴苛。

據了解，按照俄羅斯的規定，如果出口企業的產品第一次被檢出存在質量問題，將限制其對俄出口，第二次被檢出時，則直接暫停對該企業產品的進口。再加上俄羅斯的檢測流程及標準並未完全與國際標準接軌，因此對於水產品出口企業而言，俄羅斯是一個不確定風險較高的出口市場。

但是從另一方面而言，只要產品質量過硬，出口商們依然有進軍俄羅斯市場的可能性。更何況，在今年6月，俄羅斯企業和中國簽署了價值1320萬美元（約9250萬元人民幣）的食品合作協議，意味著雙方在食品進出口方面還將迎來更多機遇。事實上，除了俄羅斯設立嚴格的進口標準以外，我國也在不斷提高進口標準。

比如自今年以來，我國要求水果供應國不但要提供產品衛生證明，還要提供產品的來源地信息，並且對產品的包裝和運輸標準也進行了相應的提高。在今年前7個月內，越南水果對華出口量銳減68%，主要原因正是未能符合我國的進口標準。
