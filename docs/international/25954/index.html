「關注」大規模示威1月9日將再次襲法！巴黎交通停擺無限期

2019-12-21 396條評論 1,342次閱讀 143人點贊

巴黎12月20日電（魯佳）法國政府與工會之間針對養老金改革的分歧難以彌合，工會發出號召，將於2020年1月9日舉行第四輪「全國總動員」。

據《巴黎人報》報道，法國總理愛德華·菲利普18日在官邸馬提尼翁宮與多家主要工會負責人及僱主代表會面，試圖找到解決危機的辦法，但未能達成一致，尤其關於64歲這一退休關鍵年齡的衝突無法調和。法國總工會（CGT）秘書長菲利普·馬丁內茲（Philippe Martinez）隨後宣布，將於2020年1月9日舉行新一輪動員，這將是自12月5日以來的第四次大規模示威活動。法國工人民主聯盟（CFDT）秘書長洛朗·貝爾熱（Laurent Berger）也加入其中。但法國自治工會聯盟（Unsa）主席洛朗·埃斯庫爾（Laurent Escure）態度緩和，呼籲法國國家鐵路公司（SNCF）罷工人員在聖誕節期間暫時休戰。

工會方面抱怨稱，此次談判中，政府的態度「毫無新意」，並沒有聽到抗議者的呼聲。但總理菲利普認為，談判有所進展，他對此持樂觀態度，同時堅持64歲的「均衡年齡」方案。菲利普表示，人們可以選擇到62歲退休，也可以工作到64歲，該方案為每個人提供了個性化選擇。

據知情人士稱，談判尚有迴旋餘地，雖然餘地不大，政府可能在「均衡年齡」這一關鍵點上就不同行業進行調整。

聖誕節將至，車次和出行問題困擾乘客。根據已知的消息，19日至22日的列車預計50%可正常運行，23日和24日的列車則只有41%可正常運行。

罷工對巴黎大區交通造成嚴重且持續的影響。巴黎大區議會主席、大區流動交通署主席瓦萊麗·佩克雷斯（Valérie Pécresse）表示，法國國家鐵路公司和巴黎大眾運輸公司（RATP）沒有遵守最低限度運行計劃，鑒於與巴黎大區流動交通署的協議，兩家公司將在罷工結束後對遭受損失的乘客作出賠償，比如返還交通卡費用。巴黎大眾運輸公司管理層19日也表示，意識到了社會運動對用戶的影響，並準備在有條件的情況下予以償還。據悉，賠償計劃有可能在2020年1月末啟動。瓦萊麗·佩克雷斯建議，對所有罷工日子裡所有線路乘客進行全額賠償，她估計，截至目前的兩周停擺已造成約1億歐元損失。

巴黎市政府也於19日宣布，將對罷工中遭遇損失的商家提供250萬歐元的援助，但不會採取直接補貼方式，而是通過免除部分稅費的形式進行。
