美國試圖給菲律賓挖坑，但老杜不傻，馬上就拒絕了

2019-12-22 879條評論 2,181次閱讀 584人點贊

第一衝鋒號作者：刺客

自從老杜帶領菲律賓逐步擺脫對美國以來之後，美國可謂是心急如焚，幾乎使盡一切手段，試圖將老杜所領導的菲律賓拉回到自己的懷抱中。然而，無論美國是如何的威脅利誘，老杜就是軟硬不吃，仍然我行我素，讓菲律賓與美國的關係漸行漸遠。

尤其是在老杜之前訪問俄羅斯、並向俄羅斯購買大批直升機之後，更進一步疏遠菲律賓與美國之間的關係。然而，美國似乎還是不死心，繼續對老杜使陰招。據《菲律賓世界日報》最新報道稱，美國波音公司向菲律賓政府推銷其最先進的「阿帕奇」攻擊直升機、和「支奴干」運輸直升機，並聲稱「能夠幫助菲律賓應對地區爭端」。

波音公司印度洋-直升機市場銷售主任布羅斯特羅姆表示，他建議菲律賓軍方購買該公司最先進的兩款直升機，因為菲律賓「是時候應該擁有這些武器了，特別是在有南部和國內威脅、以及來自外部威脅的情況下」。

美國的意圖非常明顯，那就是通過向菲律賓提供先進攻擊性武器等手段，在獲得經濟利益的同時，也慫恿老杜和前阿基諾政府一樣，追隨美國製造地區緊張局勢，繼續充當其印太戰略的炮灰。毫無疑問，美國此舉不但是要菲律賓的錢，而且還想要把菲律賓拉下水。

此外，由於之前美國的盟友加拿大因為菲律賓國內禁毒問題，單方面撕毀了向該國提供直升機的合同，從而迫使老杜政府轉而向俄羅斯采供直升機，以滿足國內軍方的要求。與此同時，日前老杜政府還宣布向印度採購具有俄羅斯血統的「布拉莫斯」超音速導彈，逐步將菲律賓軍隊從美式體系轉向俄制體系。

在這樣的背景下，美國如果繼續麻木不仁，沒有積極尋求向菲律賓提供武器，那麼，用不了多久，將會有更多的俄羅斯武器進入菲律賓，從而讓美國軍事裝備永遠與這個東南亞國家絕緣。

所以，美國這時候急於向菲律賓提供直升機，本身就有一箭雙鵰的戰略考量，既能夠利用菲律賓軍方繼續使用美式武器體系，方便對老杜政府的軍事和外交政策實施控制，同時又能從俄羅斯手裡將菲律賓這個軍火市場搶回來。賺到了錢又獲得戰略利益，何樂而不為呢？

可惜美國太過高估自身對菲律賓的影響力了，儘管菲律賓目前還有70%以上的民意對美國有好感，但可惜老杜對美國的好感已經蕩然無存了。據《菲律賓世界日報》最新報道稱，菲律賓總統杜特爾特日前公開宣布，稱「菲律賓政府將不再向美國購買武器，也決不允許美國的計劃在菲律賓實現，我們不需要AH-64E，因為菲律賓沒有與別國開戰的考慮，所以不需要這種直升機」。

實際上，早在之前老杜訪俄並宣布將採購俄羅斯的直升機、軍艦、導彈甚至潛水艇的計劃之後，美國隨後就迫不及待的跳出來，一面向該國推銷二手直升機，另一面警告稱「菲律賓政府此舉會破壞美菲同盟關係」，試圖迫使老杜政府放棄對俄羅斯採購武器的計劃。

然而，美國的威脅利用手段在老杜的面前根本就起不到什麼作用，購買俄羅斯直升機的合同簽了，購買具有俄羅斯血統的印度導彈合同也簽署了。美國這時候才想起要向菲律賓軍方提供先進直升機，這不是在自取其辱嗎？

顯而易見，老杜經過與美國多年的打交道，早就摸清華盛頓那幫政客對菲律賓打的是什麼算盤了。他們這時候向菲律賓軍方推銷攻擊直升機，實際上就是在給老杜政府挖坑，因為老杜一旦接受美國的「施捨」，後果必然會毀掉自己好不容易才爭取來的周邊環境的安全局面，同時也讓菲律賓與俄羅斯剛剛建立起來的軍事合作關係、受到致命性的打擊。

所以，老杜根本就沒有美國所想像中的那麼傻，為了獲得所謂的「擁有最先進技術的殺人直升機」，最終讓自己跳入美國所挖的坑裡。在這樣的背景下，拒絕購買美國直升機，不再給華盛頓留下一點面子，自然就成為老杜的必然選擇了。
