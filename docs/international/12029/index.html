真是雪上加霜！俄“庫茲涅佐夫”號航母維修再遇新麻煩

2019-12-11 660條評論 2,673次閱讀 689人點贊

前段時間，俄羅斯海軍「庫茲涅佐夫」號航母大修工作迎來利好消息，社交媒體上的照片顯示第35海軍修船廠的干船塢合併工程進展順利，這意味著「庫茲涅佐夫」號進塢維修的場所有望在2020年內完成施工，該艦可以按照計划進塢並在2021年完成預定大修工作。然而好消息沒幾天，「庫茲涅佐夫」號航母的新問題就來了。

◎停泊在摩爾曼斯克州第35修船廠碼頭的「庫茲涅佐夫」號航母

據俄羅斯國際文傳電訊社消息，俄羅斯聯合造船廠集團總裁阿列克謝·拉赫曼諾夫透露，由於「庫茲涅佐夫」號航母在檢修過程中出現了新問題，該艦需要接受額外的維修工序，不過拉赫曼諾夫否認額外的維修工序將延長「庫茲涅佐夫」號航母的維修工期，該艦將「按照預定計劃於2022年夏天完成大修交付海軍」。

阿列克謝·拉赫曼諾夫表示「庫茲涅佐夫」號航母的新問題出現在動力系統，具體部位是蒸汽輪機主渦輪機組。「庫茲涅佐夫」號航母是常規動力航母，動力系統由8台鍋爐和4台蒸汽輪機組成。

◎製造過程中的蒸汽輪機渦輪機組

這裡簡單說一下，不管是核動力航母還是常規動力航母，把熱能轉換成機械能傳遞到推進螺旋槳的過程中，都需要用到核心部件——蒸汽輪機。通俗點兒說，蒸汽輪機的作用是將核反應堆或者增壓鍋爐燒出來的水蒸氣，通過渦輪葉片轉換成動力，然後經過減速齒輪機組的逐級減速來驅動螺旋槳轉動，這是目前大多數艦艇的驅動方式（燃氣輪機和全電推進除外）。沒有蒸汽輪機，不管原動力是強大的核反應堆還是重油鍋爐，能量都無法轉化為驅動螺旋槳的機械能，航母也就寸步難行了。

◎2019年11月，第35海軍修船廠干船塢合併工程最新進度

由此可見，蒸汽輪機在「庫茲涅佐夫」號航母動力系統中所起到的作用是非常關鍵的，蒸汽輪機出問題，「庫茲涅佐夫」號就動不了。此次出現問題的是蒸汽輪機的主渦輪機組，主渦輪組一旦發生故障，鍋爐燒出來的水蒸氣就無法驅動渦輪葉片轉動，自然也就不能帶動減速齒輪機組工作了。

對於「庫茲涅佐夫」號而言，剛換好鍋爐，蒸汽輪機又出現問題，這可太頭疼了，要修好蒸汽輪機主渦輪機組，「庫茲涅佐夫」號需要再次「開膛破肚」，問題嚴重的話甚至可能需要更換整套蒸汽輪機主渦輪機組，這可不是小打小鬧，不影響維修工期是不可能的。但作為聯合造船廠集團總裁，阿列克謝·拉赫曼諾夫也只能對外說這問題不會影響工期了，不然沒法安撫俄羅斯國防部和俄羅斯海軍代表。
