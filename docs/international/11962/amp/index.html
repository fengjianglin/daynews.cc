從智利軍機空難看極地飛行之難：超低溫能把燃油都給凍住

天天要聞
2019-12-12

【環球時報報道 駐巴西特派記者 李曉驍 特約記者 章 節 記者 劉揚】近日，一架智利空軍的C-130運輸機在向南極基地運輸人員和物資過程中失聯，據信已經墜毀。目前還沒有證據表明此次事故與極地飛行的特殊性有關，但在極地飛行確實要比常規航線更為複雜。如果在南極洲內運營，就會更麻煩些，通常需要配備專門改裝的飛機。

智利空軍C-130南極失聯

據防務世界網站10日報道，智利空軍的1架C-130H「大力神」運輸機，在飛往南極基地的途中無線電失去聯絡。該機從蓬塔阿雷納斯起飛，原本是飛往南極洲喬治國王島上的蒙塔爾瓦總統基地（智利開設的南極科學考察站）。

報道稱，這架飛機上載有17名機組人員和21名乘客，其中包括3名平民。機組人員的工作是前往智利南極基地，運送物資，維修設施設備，另外還有科學家以及學術研究人員。智利空軍表示，飛機並未啟動任何求救信號，突然消失了，可能因為迷航，燃料耗盡而迫降，或墜落。報道稱，智利是最接近南極的國家，因此也建立了最多的南極基地，總共有9座基地，分布在120萬平方公里的土地上。

智利空軍的C-130運輸機

智利電視台「T13」稱，智利軍方表示，飛機起飛後，在距南極目的地500公里左右的德雷克海峽失去聯繫。智利國防部長埃斯皮納稱，找到倖存者的機會比較渺茫，但仍會全力以赴。事故原因不排除任何可能性。

阿根廷新聞網站Infobae稱，此次失聯飛機於1978年製造，在美國服役至2008年。2012年智利花費700萬美元購入，2015年進入智利空軍服役。Infobae稱，德雷克海峽是智利本土通往南極基地最短航程的必經之路，這裡是太平洋和大西洋水流的匯合處，沒有任何陸地阻擋，該海域一直以惡劣天氣著稱，氣溫極低且常有嚴重暴風雨。據不完全統計，目前已有800艘船隻沉入德雷克海峽，造成兩萬人死亡。智利廣播電台Biobio稱，智利軍方表示，飛機起飛時，飛機狀況和天氣狀況均良好。搜尋行動將持續至少6天，並可延長4天。

極地飛行難在哪

失聯事件發生後，很多分析人士將它與南極惡劣氣象條件聯繫起來。很多航空公司都開設有極地航線，特別是通過北極航線可大幅縮短歐洲和亞洲或北美洲和亞洲之間洲際航行的路程，降低運行成本。而南極航線主要用於支援科考活動或旅遊。總體來說，極地上空飛行，要比普通地區飛行操作更複雜。包括低溫條件下的燃油管理、更複雜的導航、通信等。

專家表示，極地航路運行的一個重要特點是，低溫條件使燃油溫度接近最低使用溫度，可能威脅到正常飛行與安全。因此，必須有一套切實可行的管理監控措施，來保證飛機在極地航路安全運行。通常要使用低溫燃油，並在飛行中密切監控燃油溫度，防止燃油因低溫結冰。當油箱燃油溫度接近最低使用溫度時，還要採取改變飛行高度、增大馬赫數、改航脫離低溫區等措施提高燃油溫度，以保證飛行安全。此外，低溫還會給氣壓式高度錶帶來誤差。

在南極冰面降落的飛機

兩極地區的地球磁場也異於低緯度地區。地球磁場收斂於地磁南北極，這使得兩極地區的磁傾角很大，磁場水平分量很小，使得磁羅盤很難準確測量磁航向，而且由於磁經線迅速收縮，磁差變化比較大。加之地磁風暴、宇宙射線等影響，使得磁極經常出現漂移。這就造成了羅盤定向誤差，甚至無法正確定向。而GPS等衛星導航系統由於星座分布原因，在兩極也會導致精度降低，在極地上空飛行要有完善的備份導航手段。

宇宙射線以及太陽的周期性活動對電磁波的干擾在兩極也比較明顯，可能引發無線電通信不穩定甚至中斷。而衛星通信同樣不可靠，由於絕大多數通信衛星位於赤道上空的地球靜止軌道，無法完全覆蓋兩極地區。此外，極地飛行備降機場少、地面導航設施少，都為飛行帶來不便。

極地內航線多用特種飛機

僅就飛越極地而言，飛機無須特殊改裝。很多客機也可以降落到能夠起降大型固定翼飛機的南極機場。目前在南極大陸能夠起降大型飛機的機場大約8個，其中包括3個陸基岩石跑道機場和5個藍冰或者冰雪壓實的跑道。3個陸基岩石跑道在南極半島方向，也就是從智利和阿根廷方向進出南極的航班使用。

但在南極洲內航線飛行，由於多為簡易機場，就需要經過改裝的中小型飛機，例如BT-67、雙水獺等，當然還需要經過特殊改裝。美國巴斯勒（Basler）公司的BT-67（中國首架極地固定翼飛機「雪鷹601」號就屬於此型號），就是在二戰期間著名的DC-3飛機（軍用型為C-46運輸機）的基礎上，根據極地特點改裝而來的。巴斯勒公司將該機的活塞式發動機改裝為加拿大普惠公司PT6A-67R渦輪螺槳發動機，並對機體結構進行了微小改動，採用全新的燃油、液壓、電氣和航空電子系統，增設適合南極低溫條件的加溫、除冰裝置，於1990年2月獲得美國聯邦航空局（FAA）頒發的「補充型號合格證」（STC）。巴斯勒公司已經改裝了超過60架BT-67。最大起飛重量約13噸，最大有效載荷（含油料）5.9噸，巡航速度388公里/小時。

此外，一些軍用飛機由於具有良好的野外起降能力，對氣溫適應能力也較強，通常可以適應正負50攝氏度的溫度條件，所以也可以在這些極地機場起降。
