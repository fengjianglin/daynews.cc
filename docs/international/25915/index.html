澳洲前總理一線救火，現任總理到海外度假！7.7萬人要求總理下台

2019-12-21 264條評論 2,204次閱讀 302人點贊

澳洲前總理艾伯特一線救火，現任總理莫里森卻悠閑地在海外度假，兩人舉動形成強烈對比！目前，已有7.7萬名澳人在網上請願，要求莫里森下台。

《每日郵報》報道稱，本周五，前總理艾伯特再次「披掛上陣」，作為一名志願消防員前往悉尼西南部的Green Wattle Creek與山火搏鬥，而他身穿制服與民眾合影的照片也迅速在網上傳開。

艾伯特身穿志願消防員制服與民眾合影（圖片來源：《每日郵報》）

艾伯特身穿志願消防員制服與民眾合影（圖片來源：《每日郵報》）

與此同時，也有網友曬出了現任總理莫里森在夏威夷陪妻子度假的照片，兩任總理在山火危機期間的舉動形成了強烈對比。

莫里森陪伴妻女在夏威夷度假（圖片來源：《每日郵報》）

莫里森陪伴妻女在夏威夷度假（圖片來源：《每日郵報》）

據悉，艾伯特在過去的十幾年間一直是志願消防員。不少人甚至呼籲，62歲的艾伯特應該重新成為澳洲的領導人。

有網友表示：「我從來就不是艾伯特的支持者，但他在山火危機時前往一線救火。而現任總理莫里森卻在夏威夷度假，說些什麼『寄託哀思和為他們禱告』之類的屁話！」

另外一名網友也表示：「我不是艾伯特的粉絲，但這件事讓我開始反思。有多少澳洲政客真的在想辦法撲滅大火？」

莫里森攜妻女前往海外度假一事被曝光後，引發了澳洲民眾的強烈不滿。而他本人也出面道歉，並辯解稱「我又不是消防員，也不是指揮，相關負責人正在做著他們的工作」。

莫里森如此回應也徹底激怒了澳洲民眾，有人在網上發起請願，要求莫里森下台。請願發出18小時內，已有超過7.7萬人簽名。

兩名志願消防員殉職（圖片來源：澳洲九號台新聞）

本周，兩名新州志願消防員殉職，而莫里森也宣布提前結束假期，於今日返回悉尼。莫里森表示：「考慮到最近發生的這些悲劇，我決定儘早返回悉尼。」

澳洲九號台新聞報道稱，莫里森計劃於本周末前往新州鄉村消防局總部，而他的妻子和女兒會繼續留在夏威夷。
