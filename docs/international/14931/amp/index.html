身陷彈劾指控 特朗普看陸海軍球賽 受全場熱烈歡迎

天天要聞
3 天 ago

當地時間14日，深陷彈劾案的美國總統特朗普抽空來到費城老鷹隊的主場林肯金融體育場。為在此舉行的年度陸海軍橄欖球對抗賽拋幣開球，他受到現場7萬名觀眾的熱烈歡迎。

當天，特朗普戴著寫有”讓美國繼續偉大”字樣的紅色運動帽，來到了球員更衣室。有目擊者稱，他與那些球員一同合影，並拍了拍他們的背，向他們表示問候。隨後他來到球場，按傳統習俗拋硬幣，為比賽揭幕。看台上約7萬名觀眾歡呼雀躍，向他致意。

開賽前，陸軍隊的中後衛科勒·克里斯蒂安森還向他贈送了印有45號的橄欖球衣。

據悉，美國陸海軍大學橄欖球比賽起始於19世紀90年代，已有約120年的歷史，是陸海軍軍事院校大學生之間最具傳統的一項體育比賽。在此之前，一共有10位美國總統出席過這項年度賽事，其中就包括奧巴馬、小布希和肯尼迪。特朗普一直自稱是美國軍隊的偉大捍衛者，近年來，他參加了兩場陸軍海軍學院的足球賽。

這與之前他在華盛頓特區觀看的美國職業棒球比賽世界大賽的待遇形成了鮮明的對比。今年10月27日，特朗普在通過大屏幕向在場觀眾致意時，在場球迷爆發出了噓聲，甚至還有人喊道”把他抓起來！”（編輯：徐思韻）
