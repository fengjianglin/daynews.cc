傑克羅素梗失蹤三天，大鬍子鏟屎官竟然挖地三尺將它找回…

2019-12-03 1,835條評論 4,457次閱讀 3382人點贊

「掘地三尺」多用於形容尋找某物的範圍甚廣，尋找得十分細緻。而最近一位英國的鏟屎官為了找回自己走失的狗狗，可真的是掘地三尺……

11月20日，鏟屎官艾倫·惠頓(Alan Whitton)帶著自己的狗狗傑克羅素梗米茨（Mitzi）去奈特頓森林（KnightonWood）周圍散步。

米茨遇到了一隻巴吉度犬，兩隻小狗狗一見如故，於是艾倫就解開了牽引繩。讓它們在森林裡開心的追逐玩耍。

可是，過了一會巴吉度犬自己回來了，而米茨卻不見了蹤影。艾倫焦急的四處尋找米茨，妻子和兩個孩子知道了米茨不見了，也趕忙一起到森林尋找它。

這一家四口從一直從下午找到天黑，也沒見到米茨的蹤影。他們只好回家第二天繼續尋找。

第二天，艾倫還製作了大量的尋狗啟示，分發給住附近的居民後。繼續在茂密的森林裡的尋找。這次米茨失蹤，是第一次離開艾倫，艾倫內心也是無比的焦急，擔心它在森林裡遭遇不幸……

米茨是艾倫一年前領養的狗狗。那時艾倫的第一隻狗狗Tinker剛剛因病去世，艾倫心裡無比的難過，直到在收容所遇到可愛的米茨……

見到它的第一眼，艾倫就被這個機靈的小傢伙打動了，馬上就決定收養它，給它一個家。

可惜，第二天的找尋還是一無所獲……

第三天，艾倫不放棄，再次又回到森林尋找米茨。

在上午11點左右，他接到了一個狗友的電話：他說他家就在森林附近，他發現的他的狗狗一直在一個狐狸洞周圍嗅來嗅去，猜測米茨可能是掉進狐狸洞里了。

艾倫馬上趕到那個狐狸洞附近，他帶來米茨的玩具，在狐狸洞口處發出吱吱聲，聽到一陣微弱的嗚咽聲作為回應。米茨就在裡面！

很多得知米茨被困狐狸洞狗友，也馬上趕來幫忙，他們帶來了鏟子，來幫艾倫一起挖開狐狸洞，救出米茨。

他們足足挖了2個多小時，才挖到米茨所在的位置。艾倫把手伸到新挖出的洞口，米茨馬上從洞里鑽了出來。

這個年近50歲的外形硬漢的大鬍子鏟屎官艾倫，再也忍不住了，抱著失而不得的米茨失聲痛哭。來幫忙的朋友也開始為他們的團聚而歡呼聲……

米茨後來經過檢查，還好它沒有受傷，只是身上有一些跳蚤。艾倫現在計劃為米茨買一個GPS項圈，在牽好牽引繩，以避免將來再發生類似的事情。

他知道再也不能失去米茨了……
