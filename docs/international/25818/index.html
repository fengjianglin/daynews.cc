“天價軍費”對美軍不是一個好預兆？或成壓死駱駝最後一根稻草

2019-12-22 770條評論 2,006次閱讀 160人點贊

美軍

又到財政年底，美國國會通過了新法案，高昂的軍費竟佔世界總和三分之一！美國人沾沾自喜，頗感榮耀。然而，這份美軍的「聖誕禮物」竟是個潘多拉魔盒？

據報道，對於新財年國防授權法案獲得國會通過，美國一些媒體和分析人士難掩興奮之情，認為這是軍隊和軍工利益集團提前收穫的「聖誕禮物」，標誌著特朗普2017年提出的「重建美軍」計劃「向前邁進一大步」。

但客觀來看，「天價軍費」對美軍來說卻不是一個好的預兆。

據新華社近日報道，該法案授權美國2020財年軍費支出7380億美元，這一數字佔據了世界軍費總和的三分之一，相當於15個國家總和，更是俄羅斯軍費的十倍。國內專家表示，美國軍費連年上漲，全是民眾買單，對國民經濟負擔太重，或成壓死駱駝最後一根稻草。

美軍

過去美國依靠金融壟斷、美元壟斷、科技壟斷、軍事威脅來收割其他國家的羊毛，但這條路未來只會越來越差。俄羅斯受打壓後把美國國債拋得差不多了，人民幣國際化受響應的國家越來越多，中國加大了晶元等科技領域的長期投資，航母打擊群軍事受到新型導彈，無人機，高超音速的威脅，並不能造成壓倒性勝利。

現在新軍事裝備研發，美國也遇到了天花板。隨著中國的追趕步伐，比如隱形戰機，隱形轟炸機，航母，大驅等裝備的研發生產，美國軍備對中國的代差優勢正在逐漸喪失。

復旦大學教授沈丁立

復旦大學教授沈丁立認為，美國2.8%國防預算增長幅度也只是「充充門面」而已。山姆大叔背後的心理在於，不能適應中國等國的快速崛起，以及由此帶來的力量對比變化，不甘心過去對他國的「碾壓優勢」和「絕對領導地位」被打破。在沈丁立看來，如今的美國對其他大國越來越難構成威脅。

另一方面，值得注意的是，美國更想著眼於未來科技。這次通過的法案除了支持新型武器技術的研發投入，例如無人機和高超音速裝備，還批准美成立太空部隊。

就此分析人士稱，美國現在大量投錢發展太空建設，即不和平也很不值，因為將會增加軍備競賽的風險，將全球向戰爭推進。過去的戰爭可以掠奪資源，是有利可圖的，而現代戰爭由於利益交織只有合作才能共贏，所以越來越多的錢投軍事，最終就沒錢發展經濟。而這正是美國現在最應看重的部分。

​示意圖

而且，美國想搶先進行太空部署進行壟斷，是不切實際的。網友也笑稱，接下來讓美國自己看著辦吧，中俄兩國想跟進美國的太空戰力不是不可能。

想想蘇聯那麼強大為什麼解體？投資主要集中在長期經濟效益差的軍事產業上。反觀美國經濟增長速度低緩，還撥天價軍費，難保「絕對領導地位」不說，全由民眾買單。如此亂花錢充門面，難道美國不是在步蘇聯後塵嗎？（育宏）
