全球幾十個國家落入中等收入陷阱，李稻葵：2035中國晉陞發達國家

天天要聞
2019-12-12

如何從中等偏上收入國家跨入高收入（發達國家）國家，近一個世紀以來，一直是一個世界難題。

按照世界銀行公布的數據，目前世界上的高收入國家（發達國家）有80個左右。其中有20個左右是人口超過一千萬的人口大國。

這些國家從中上等收入國家邁入高收入的發達國家。平均用了12年又4個月時間。而那20個人口超過1千萬的國家，從中上等收入邁入高收入國家的時間大多較短，不到12年時間，平均只用了11年又9個月。

但是這些人口大國的經濟，在進入中上等收入國家前的這段起飛前的時間比較長，也就是起飛前的準備時間比較長，克服困難所花的時間更久。而當他們一旦邁入中上等收入階段後，這20個人口大國的收入提升速度就很快，而且是越到後期速度越快。

目前的問題是，從上世紀70年代以來的半個世紀里，全世界100多個發展中國家中，只有13個成功邁入高收入國家。更多的要麼仍然停留在貧窮落後階段，要麼卡在中等收入階段，幾十年時間裡都邁不過去。人們把這種現象叫做中等收入陷阱。

比如上世紀70年代拉丁美洲的十幾個國家，都已經進入中等收入階段。但現在50多年過去了，仍然沒有取得任何進步。俗稱拉美漩渦。

上世紀80年代，東南亞的馬來西亞，印度尼西亞，泰國，菲律賓等國，也已經達到中上等收入，現在40多年過去了，經濟發展一直停滯不前，俗稱東亞泡沫。

上世紀90年代，北非國家，比如埃及，敘利亞，利比亞，突尼西亞，葉門等這些國家，也達到中上等收入標準，卻到現在也沒有邁過去。很多國家反而陷入了政治危機，社會動蕩的泥潭裡。這些國家的現象被稱為西亞北非危機。

整個上世紀70,80,90年代，橫跨亞非拉廣大地區的這麼多國家，都先後陷入中等收入陷阱。

外界總結的原因就是：條件變了，發展方式沒有變，缺乏可持續發展能力。

按世界銀行公布的數據，2018年的最新收入分組標準為：人均國民總收入低於995美元為低收入國家，在996至3895美元之間為中等偏下收入國家，在3896至12055元之間為中等偏上收入國家，高於12055美元為高收入國家。

在中國，換算成人民幣就是家庭人均年收入2.7萬-8.5萬之間是中等偏上收入，家庭人均年收入高於8.5萬，就算高收入。根據國家統計局2018年發布的《新中國成立70周年經濟社會發展成就系列報告之一》。報告中顯示2018年我國人均國民總收入達到9732美元（約67,150人民幣），高於中等收入國家平均水平，但並未邁入高收入國家標準線。

也即是說，中國已經正式步入中等偏上收入國家的隊列。

具體情況是，我國目前有超過4億人達到中等偏上收入標準。其中關於高收入人群的統計數據，以最近的2017年為例，統計中國高收入人群數量達到7200萬左右，人均年收入16.6萬。中國的人群收入比例由低到高大致應該是10 : 3.2 : 0.8。

按照既往的國際經驗，中國邁入中等偏上收入階段後，經濟發展的環境和條件都將隨之改變，這時就需要逐步從舊有的發展軌道上尋求轉型，尋找新的經濟增長動能，這樣才能順應變化，成功邁過中等收入陷阱，進入發達國家的行列。

但是作為14億人口的大國，人口分布和結構複雜，人群收入比例在低中高三個階段里所佔絕對數量都不小，這種情況在全世界歷史上都是沒有出現過的。在經濟轉型過程中，如何兼顧科技研發為主的高端實業競爭力，和可以促進就業的傳統經濟之間的比例平衡關係，就成為了不得不考慮的重中之重。

如果不升級產業，比拼科技實力，就難以晉陞到產業鏈前端，獲得產品的高附加值利潤。但是高科技產業能容納的就業崗位有限，所以中國14億人口的就業市場，又離不開傳統製造業創造的崗位。

國家前幾年就已經針對中國現階段的發展，制定了中國製造2025規劃和供給側改革等方案，目的就是在經濟發展新階段，切換新舊產能，順利邁過中等收入陷阱。在2019年經濟壓力持續加大的情況下，我們也許可以從以下幾個角度思考：

1. 金融業在全國工商業所創造的利潤里所佔比例是否偏高？會不會擠壓了第二產業獲利空間，進而影響到產業升級？

2. 傳統製造業在稅負，高廠房租金，人力成本時高漲的現實困境下，從哪些方面可以緩解其壓力，幫助他們拓展生存空間，進而消化社會就業。

3. 電商和實體商業之間的再平衡有沒有必要？

4. 高房價後緊接著的房租上漲，是否會進一步抬高人力成本，和整個社會的生產生活成本。

5. 如何實現從產品生產大國向裝備製造業大國邁進？

6. 面對核心科技被「卡脖子」的窘境，政府和企業的研發資金投嚮應該怎麼安排更合理？

7. 內需市場遲遲難以啟動的核心問題在哪裡？

宏觀經濟是一盤大棋，裡面的每一步也都牽扯著一個小問題的解決。我們將以怎樣的姿態邁過中等收入陷阱的答案，可能就隱藏在這其中。

不過雖然面對困難，清華大學中國與世界經濟研究中心主任李稻葵卻認為，因為中國的產業鏈很全，國內市場大，所以經濟韌性很高。中國從現在開始，頭十一年如果保持6%的增速，緊接著的十年保持4%的增速，到2035年，我們將穩穩地進入世界發達國家的行列，在老百姓生活水平、人均發展水平、富裕程度、百姓期望能過上的美好生活的程度上，在人口超過500萬的中大型國家裡面，我們將進入30強。」

不知道條友們現在對自己身邊所處的經濟環境感受是不是如本文中所述，或者有什麼不同呢？經濟冷暖，歡迎在評論區討論。
