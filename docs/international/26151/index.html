澳洲街頭的公共設施究竟有多人性？每一個細節都令人感動

2019-12-21 1,954條評論 4,727次閱讀 2061人點贊

公共設施的完善程度體現著一座城市的發展水平，也反映出當地居民的生活狀態。智能化、人性化的公共設施會讓居民獲得更多的幸福感。很多人都知道，在這方面國外很多地方都比我們要先進，這是事實。前不久我到澳洲旅行，在悉尼街頭就拍下了一些他們的公共設施，還是挺有感觸的。

公園，是城市的標配，也是城市居民周末休閑的好去處。澳洲的街心公園綠草如茵，與我們國家所不同的是，這裡的草坪上沒有「禁止踩踏」的警示牌，相反，這裡的草坪就是供人們使用的。所以你會看到這裡的人們可以隨意在草坪上踩踏、奔跑乃至睡覺。

澳洲公園裡不僅有休閑座椅，還有這些設施讓人暖心。首先是公共餐桌。一家人出來到公園裡玩耍，要做好像郊遊的準備，到「野餐」時間了，就在這寬敞的大桌子上吃頓美味。

我注意到，這些公共餐桌上都安裝有一個鐵質的方形設備，暫且叫它「鐵環」吧。剛開始我也搞不懂這個鐵環是幹什麼用的，一問才知道，這是個拴狗環。原來，這是公園為帶寵物出遊的家庭準備的，當大家在用餐的時候，小動物可能會亂跑，那麼就拴到這個鐵環上吧，保證安全，真是人性化。

除了公共餐桌，澳洲的公園裡還有免費的燒烤箱，也就是說，你預備好食材，就可以在公園裡燒烤了，免費使用哦。看到這些，我忽然想，如果我們的公園裡也有免費的燒烤爐可以用，那我周末都不想出遠門，就在家門口擼串就好了啊。

吃飯前吃飯後我們都要洗洗手，這就是澳洲的戶外公共水池了，這樣的水池很多，用起來也非常的方便。最重要的是，在這裡不僅可以洗手，口渴了還能直接喝水。

注意到洗手池上方這個小小的像噴嘴一樣的小孔了嗎？這就是出水口，澳洲所有的公共洗手池都是直飲水，包括廁所。所以在澳洲的廁所，你會經常看到有人洗了手，再夠上去對著洗手池的水喝幾口，這種情況在我們看來不可思議，對於澳洲人來說再平常不過了。

澳洲的公園裡還有一個裝置特別醒目，可以說是隨處可見。乍一看有點像咪表，一般人也不會注意，但實際上，這個裝置用處大了。可以說，這才是澳洲公共設施人性化的最大體現。

這是一個應急呼叫設施，是給在遊玩過程中突然發生疾病或者羽然遇到困難的人用來呼救的應急裝置。最主要的是，人家還有中文接聽。看到這些，作為一個中國遊客，我心裡反正是暖暖的。

澳洲也是一個禁煙的社會，只可以在指定的區域抽煙，他們會設置一些扔煙蒂的設施，你看這樣的像我們的蜂窩煤，扔進去不會掉出來。

在澳洲的公園裡，所有的護欄都不是90度稜角的，而是像圖中這樣做成了圓滑的邊角，就是為了防止小孩和特殊人群磕碰，細節處處顯示了人文關懷。

在公園的的垃圾桶里，我還看到有的放置著一些綠色的塑料袋，垃圾桶上也會有狗狗的標誌。原來這是為了那些牽狗遛彎的人準備的，有些人出門會忘記帶撿屎袋，於是就在垃圾桶上為他們免費準備，當然如果你需要用來裝其他隨手扔的垃圾，也可以抽取。但是不可以貪便宜裝回家哦，因為這是公共資源。

最後我們再來看看澳洲公路上的一些細節處理。公園的馬路牙子上，安裝了很多柳丁和鐵角，經過詢問我了解到，在澳洲玩滑板的年輕人很多，他們尤其喜歡在馬路牙子上玩特技，所以為了避免馬路牙子受到損害，也為了年輕人的安全考慮就安裝了這些柳丁和鐵角。

怎麼樣？看著澳洲公園裡這些人性化的設置，你有什麼感觸呢？在這裡我們不評判誰的更先進，而是希望能借鑒別人的長處，來改進我們的短板，希望我們的公共設施也更加的人性化。我是路燈攝影，每天為你分享旅途故事。
