俄羅斯遭4年禁賽背後：大規模系統服用禁藥，涉事人員離奇死亡

2019-12-10 287條評論 1,622次閱讀 693人點贊

「俄羅斯的審判日：我們的體育被毀滅了！」一家俄羅斯體育媒體使用了這樣的標題。

「體育史上最嚴重的處罰。」在地球的另一頭，西班牙第一大報《世界報》寫道。

北京時間12月9日下午六點，世界反興奮劑組織（WADA）執行委員會一致同意了對俄羅斯執行四年禁賽令的決定，其中包括未來四年內舉辦的奧運會和各種項目的世界盃。這意味著：

俄羅斯將無法參加2020年的東京奧運會和2022年的北京冬奧會，同時還將錯過2022年的卡達足球世界盃和整整四屆冰球世界盃。

2019年12月9日，瑞士洛桑，世界反興奮劑機構(WADA)主席克雷格·里迪在關於俄羅斯興奮劑數據操縱的會議後的新聞發布會上發言 / IC Photo

這是四年內俄羅斯第三次遭遇國際禁賽令。

2016年里約奧運會，俄羅斯被禁止參加田徑項目；2018年平昌冬奧會，俄羅斯被剝奪了以國家名義參與的權利；而包括本次在內，所有制裁都來自同一個理由：禁藥嫌疑。

或許唯一的好消息是「全面禁賽」的傳聞並未成真：與2018年平昌冬奧會一樣，俄羅斯運動員仍可以個人身份參加體育賽事，前提是需要證明自己的清白。另外，俄羅斯仍有上訴權——按照國際體育規則，俄羅斯可以選擇在未來21天期限內向國際體育仲裁法庭（CAS）提出上訴。

「我們一定會上訴的。」前述俄羅斯體育媒體《世界盃》寫道。

禁藥疑雲

籠罩俄羅斯的興奮劑醜聞陰雲起自2014年底。

這一年12月，德國一家電視台公布了一部紀錄片，稱超過99%的俄羅斯運動員在使用禁藥。這部片子沒有引起太多公眾關注，但卻引起了世界反興奮劑組織的注意。

2015年11月9日，世界反興奮劑組織召開新聞發布會，宣布撤銷俄羅斯反興奮劑機構（RUSADA）的認證授權，因為他們已經掌握了大量俄羅斯運動員系統性服用禁藥的證據。世界反興奮劑組織同時表示，這樣的制裁措施僅僅是第一步。

一個公眾極為陌生的名字在接下來的一地雞毛中逐漸浮出水面，此人名叫羅德岑科夫，是當時莫斯科反興奮劑實驗室的主任。在世界反興奮劑組織最初的新聞發布會中，羅德岑科夫被指是俄羅斯興奮劑黑幕的最大嫌疑人，2016年1月，他選擇了逃往美國。

羅德岑科夫 / 網路

事情的發展似乎證明他的決定很有道理。

2016年2月，羅德岑科夫在俄羅斯反興奮劑機構（RUSADA）的兩位前同事相繼不明不白地突然辭世，間隔不到兩個星期，而死因至今眾說紛紜。

2016年5月，羅德岑科夫與另一位揭發者、俄羅斯前奧運冠軍斯捷潘諾娃共同站了出來，以俄羅斯政府為主角的「大規模、系統性服用禁藥」醜聞就此揭開。

2016年7月，俄羅斯田徑運動員因這起禁藥醜聞而被宣告無緣里約奧運，但對於此後三年的風暴，這次禁令不過是個序曲。

隨著世界反興奮劑組織的調查結果一步步公開，曾被俄羅斯人視為國家驕傲的2014索契冬奧會很快成了史上最大一次國家主導的禁藥行動的背景板：俄羅斯在這次家門口舉辦的冬奧會上力壓全部對手，壟斷了獎牌榜榜首的位置。

而在這背後，卻是由克里姆林宮下令、俄羅斯特工執行、各級反興奮劑工作人員配合其中的一場規模宏大的密謀。

儘管在事發當時，俄羅斯不乏將此事指為「西方操縱」、「政治動機」的聲音，甚至有主流媒體將之與納粹德國對蘇聯的侵略行動相提並論，但體育界的反應已經說明了很多問題：兩屆撐桿跳奧運冠軍伊辛巴耶娃與俄羅斯體育部長穆特科都宣稱：「禁藥人人在吃，其他國家運動員一樣不幹凈」。

2015年11月俄羅斯因涉嫌「系統性」使用興奮劑遭國際田聯全面禁賽，撐桿跳女皇伊辛巴耶娃出席發布會答記者問 / IC Photo

而克里姆林宮顯然更想在這個問題上息事寧人，2016年，在奧林匹克委員會裁定「乾淨的」運動員可以以個人身份參加奧運會的時候，克里姆林宮立刻出面稱讚了這一決定。

里約奧運會前，普京親自出面為這些「個人身份運動員」送行，在講話中也再次含糊承認，俄羅斯體育界在此前的賽事中確有不當之處。

2016里約奧運會前，俄羅斯全隊舉行歡送會，普京出席發表講話 / IC Photo

風波再起

2017年，世界反興奮劑組織的最終判決下達，俄羅斯被禁止以國家名義參加在韓國舉辦的平昌冬奧會，處罰力度之重為體育史上所未有。

但對於大多數人來說，這起禁藥風波也已經隨之告一段落： 在2018年6月舉辦的俄羅斯世界盃並未受到禁藥風波的影響，2018年9月，世界反興奮劑組織正式取消了針對俄羅斯反興奮劑機構的處罰（停止授權），唯一的條件是要求俄羅斯反興奮劑機構將2012至2015年期間的葯檢樣品及結果移交世界反興奮劑組織。

但大家沒有想到的是，事情並沒有完。

五個月後的2019年1月，世界反興奮劑組織工作人員在莫斯科反興奮劑實驗室中收集了一批過去的檢測結果；2019年9月，情況急轉直下：世界反興奮劑組織認為，這批提交上來的數據中的許多已經遭到篡改，或者乾脆就是刪除。

2014年，索契冬奧會的一個興奮劑檢測點 / IC Photo

由於此事與一年前取消處罰時的條款直接相悖，11月底，世界反興奮劑組織提出，建議對俄羅斯執行四年禁賽。

審判日就在今天，對於這樣的結果，許多人並不感到意外：畢竟，媒體報道稱世界反興奮劑組織的這次表決結果是「一致通過」。

「我們會適應」

在如今的俄羅斯，禁藥問題似乎已經難以激起太多情緒：12月8日，與克里姆林宮關係匪淺的寡頭烏斯馬諾夫發表了一封寫給世界反興奮劑組織的公開信，呼籲世界反興奮劑組織要「保護那些乾淨的運動員」。

「只要決定是公正的，我完全支持對任何違反興奮劑規則的人採取最嚴厲的制裁，」烏斯馬諾夫寫道。他同時也是國際擊劍聯合會的俄方主席，還曾是阿森納足球俱樂部的大股東。

俄羅斯冰壺聯合會主席斯維什切夫看上去甚至更加淡定：「這樣的打擊並不可怕……我們將找到機會，無論何種制裁……我們會適應。」他在世界反興奮劑組織表決前幾小時說。

俄羅斯莫斯科，一名男子走過俄羅斯奧委會莫斯科總部 / IC Photo

世界反興奮劑組織表決結果公布後的幾小時，俄羅斯運動員們給出的第一波反應也同樣平靜得驚人。三次跳高項目世界冠軍拉西采科內說，「完全不意外這樣的結果」，四屆奧運冬季兩項冠軍吉洪諾夫則更為直接：「這是我們應得的。」

當然，也仍有關於「政治動機」的反擊。俄羅斯夏季奧運會項目聯合會主席季托夫對媒體稱，「這不關於體育。這完全是政治。」

未來將會如何？俄羅斯反興奮劑機構將在12月19日前做出是否上訴的決定，而假設中的上訴結果仍難於預測，但此時此刻，一些此前的例子或許有些參考價值：

2017年底，在平昌冬奧會禁賽令下達當時，俄羅斯外交部發言人扎哈羅娃曾在指責「政治動機」之餘回應稱俄羅斯並不會因此垮掉，」世界大戰沒能做到，蘇聯解體也沒能做到。「

而兩個月後的平昌冬奧會期間，俄羅斯國內則在「抵制奧運」的呼聲中維持了驚人的平靜——在當時靜悄悄的莫斯科街頭，與對此事隻字不提的俄羅斯媒體上，你甚至意識不到幾千公里外正在舉行一場冬季奧運會。（文/路塵 責編/朱凱）
