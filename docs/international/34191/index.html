有惡意軟體以支持瑞典環保少女Greta Thunberg的名義進行傳播

2019-12-25 1,730條評論 3,960次閱讀 1695人點贊

Proofpoint 網路安全研究人員剛剛發現，有一款名叫 Emotet 的惡意軟體，正在以「支持 Greta Thunberg.doc」的名義進行傳播。儘管看似一封電子郵件，但其本質上是一款竊取 Windows PC 上金融信息的網銀木馬。攻擊者試圖以這位瑞典環保少女的名義，邀請受害者參加聖誕前夕的氣候變化抗議活動。

早些時候，Greta Thunberg 因一系列旨在提升全球環保意識的活動，被《時代》雜誌評為了 2019 年度人物。

對於那些毫無戒心的網友們來說，需要對這類基於電子郵件的惡意軟體傳播活動提高警惕。

Proofpoint 安全研究人員指出，這輪攻擊主要針對美國、歐洲和亞洲地區擁有 .edu 電子郵件地址的學生群體，甚至出現了使用多種語言的版本。

（圖自：Proofpoint，via TechSpot）

Proofpoint 補充道：「我們發現受攻擊的 .edu 域名數量超過與任何特定國家 / 地區相關的域名。鑒於 Thunberg 得到了學生和年輕人群體的有力支持，黑客的套路也是有些講究的」。

遺憾的是，轉發此電子郵件，並不會對緩解氣候危機有任何幫助，反而只會讓更多計算機不小心感染上 Emotet 惡意軟體。

這款網銀木馬偽裝成了微軟Word 文檔附件，並且使用了與郵件主題相同的「支持 Greta Thunberg.doc」文件名。

Proofpoint 提醒道：「假日期間，攻擊者會毫不猶豫地瞄準和利用人們的善意，也算是一種另類的公益意識晴雨表了」。
