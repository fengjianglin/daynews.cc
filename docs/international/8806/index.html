錯綜複雜的日本交通網，常見的交通方式，你都知道幾個？

2019-12-08 403條評論 4,185次閱讀 424人點贊

文/星醬

「只要搞定日本交通，就能輕鬆搞定日本自由行。」這句話，對於即將去日本旅遊的小夥伴來說，絕對是真理！

日本的交通非常發達，出行便利，但是交織如網的城市鐵路和地鐵，聽著也是讓人挺頭大的。

今天，星醬就為大家，粗略整理了一下日本的出行方式，讓你的旅途不再緊張~

飛機

目前日本國內主要有兩家航空公司，分別是日本航空（JAL）和全日空（ANA）。JAL和ANA的服務較好，因此價格稍貴。

另外，還有日本天馬航空，日本天籟亞洲航空，樂桃航空等航空公司，價格更便宜些。

全國共有東京、大阪等數十個航空港。我們可根據不同航空公司的優惠折扣，選擇合適的航空公司與航班。

而且日本人非常注重服務，航空服務也是非常人性化的。

新幹線

新幹線（Shinkansen）是連接日本全國的高速鐵路系統。

1964年10月1日，日本第一條東海道新幹線開通，連接東京、名古屋和大阪，連通日本三大都市圈。

該線路也是全世界第一條投入商業營運的高速鐵路系統。至今，新幹線仍是世界上先進的高速鐵路系統之一。

從大阪到東京大概兩個半小時，速度還是很快的。

由於新幹線價格也不便宜，所以如果不是跨地區活動的話，就不太需要乘坐新幹線。

JR鐵路

JR列車跟國內的火車比較相似。

JR列車有3種速度，特急車速度最快，停靠的站最少。

特急指的是在既有線（普鐵）上跑的中長途直達列車，通常使用特製的列車。有更加舒適的車廂和乘坐體驗，相比起來，也更貴一些。

快速、普通每站都停，速度很慢。大家出行的時候，可以根據停站多少，發車間隔，盡量選擇速度快的列車。

JR列車通常分為兩種坐席。自由席是可以隨便坐的，但是沒有座位就只能站著，指定席是對號入座的，但是要另外給錢。

地下鐵

地下鐵是擔任大城市市內交通且因為用地緊張，而不得不基本修在地下的軌道交通。

在日本像中國那樣很長一段，都在地下的「地下鐵」倒是很少。

除了東京、大阪和名古屋三個城市的地下鐵系統規模較大之外，其他幾個有地鐵的城市，都只有寥寥幾條線路。

日本的地鐵票有3種類型：一次性票，日票和月票。

一般情況下，一次性票或日票比較適合短時間旅行的遊客。日票的話，可乘坐東京Metro地鐵全線。

觀光巴士

觀光巴士是一種非常經濟實惠的交通出行方式，也是了解當地社會文化的重要渠道。

行程較近的地方，可以乘坐觀光巴士，觀光沿線地區的自然景點，領略日本美麗的景色。

而且語言服務是吸引外國遊客的重要因素，不失為出行的一個良好選擇。

輪渡

眾所周知，日本是一個島國，擁有狹長的海岸線，日本的海上交通網相當完善。

無數大小船隻成為溝通全國各個島嶼的橋樑，安全又方便。

在許多提供遊玩服務的島嶼，遊客還可以乘坐觀光遊覽船或者水上快艇。

而且日本的海鮮非常便宜，乘坐輪渡可以在觀賞日本海岸線景色，體驗日本海洋風情的同時，買到實惠便宜的新鮮產品。

計程車

由於維護成本高、人工費貴、國家政策調控等原因，日本計程車相當昂貴。

日本全國的計程車費大致相同，起步價為660日元（約合40元人民幣），超過兩公里後就按每274米增加80日元計費。

和上海採用分時計價一樣，在日本過了晚上11點，所有的計程車費均上浮30％。從東京新宿車站打車到成田機場，單程就要2萬6千多日元，約合1595元人民幣，比東京到上海的單程全價機票還要貴。

在日本，計程車是一種不太推薦的交通出行方式，日本政府倡導綠色出行。

由於出行成本相當昂貴，所以日本街頭雖不缺少計程車的影子，但不是大家經常選擇的出行方式。

JR Pass

為了促進旅遊業，日本交通方面出了很多，只有遊客才可以使用的交通優惠券。包括JR Pass，關西廣域券，京都大阪周遊券，富士山箱根周遊券等等。

日本鐵路周遊券（JR PASS）是由JR集團的6家公司（JR北海道、JR東日本、JR西日本、JR東海、JR九州、JR四國）共同提供的，專為外國人遊客準備的乘車優惠通票。

是一種日本全境都可以使用的通票，範圍廣，實用性價比很高。

JR Pass只能在日本以外國家的指定旅行代理店出售，在日本國內不可購買。所以大家不要想著抵達日本後再購買。

到達日本後，拿著訂單到JR旅行服務中心（JR Travel Service Centre，大部分主要JR車站內的綠色窗口處以及成田和關西國際機場都有），出示護照以激活通票。

激活後走人工通道，就可以無限次乘坐JR相關的交通工具啦。
