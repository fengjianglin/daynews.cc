戰爭之下誰能倖免？兩名俄軍、一名美軍死於突然襲擊

2019-12-25 259條評論 1,276次閱讀 150人點贊

兵凶戰危，無論是強悍的美軍還是不要命的俄軍，只要上了戰場隨時都有危險，更何況他們還要跟神出鬼沒的”游擊隊”作戰，死傷是在所難免的。

根據敘利亞ZAMANAl Wasl網站消息稱，當地時間22日晚上，敘利亞南部德拉省一個俄軍檢查站遭到一夥武裝分子的襲擊，由於事發突然，俄軍被打了個措手不及，導致2名俄軍死亡，多人受傷。雖然俄羅斯跟當地武裝達成了和解協議，但仍然遭到不明襲擊。

要知道，德拉省此前就是敘利亞叛軍集中地，在敘利亞局勢還沒有明朗之前，他們早就滲透到了該省的各個角落，打算利用堅固的堡壘跟俄敘聯軍死磕到底。2019年俄敘聯軍一舉攻佔東古塔地區後，該地的叛軍變得士氣低落了，在對方的猛烈攻擊下不得不選擇了繳械投降。

誰能想到，這批人並不是真心歸附，時刻都想著捲土重來。本月以來，俄軍遭遇的襲擊也不少。12月2日，一輛俄軍裝甲車在阿勒頗巡邏時，遭遇路邊炸彈襲擊，造成3人受傷。12月23日，兩架無人機試圖攻擊俄駐敘郝梅爾空軍基地，結果被俄軍提前發現併當場擊落。

頻繁的攻擊表明，敘利亞叛軍有了死灰復燃的跡象，這背後如果沒有勢力的支持，似乎說不過去。目前敘利亞叛軍被死死壓制在了伊德利卜省，眼看就要滅亡了，結果由於土耳其和西方勢力的強勢干預，導致叛軍仍然活著，只要叛軍一天不消滅，那麼俄軍就得繼續”繼續站崗放哨”。

如今俄軍檢查站被襲擊表明，武裝分子並不甘心失敗，他們要通過頻繁的攻擊逼迫俄軍撤離。但從俄敘聯軍掌握主動權的形勢看，這種以卵擊石的方法並不能為叛軍帶來什麼好處，反而會惹來俄軍的強勢報復，相信只要有合適的時機，俄軍消滅這些頑固分子是遲早的事情。

值得注意的是，在2名俄軍遇襲身亡的第二天，又一名美軍犧牲了。根據阿富汗黎明新聞電視台報道稱，美軍證實，一名叫邁克爾•詹姆斯•高布爾的美國軍人，在阿富汗北部昆都士省巡邏時被塔利班的自製炸彈當場炸死。這給原本緊張的美塔和談又蒙上了一層陰影，接下來雙方又會圍繞撤兵問題而激烈爭吵，不過主動權似乎回到了塔利班手裡了。

坦白說，如今美軍深陷阿富汗戰場無法自拔，18年來死了數千人花費了2萬億美元，不僅沒有扶持起來一個強大的阿富汗親美政權，反而逼得很多阿富汗人站到了塔利班的隊伍里。長年戰爭，已經導致美軍厭戰，國內反對，急得白宮不知道如何是好。

特別是在美債危機爆發，製造業空心化，全球霸權動搖的情況下，對於塔利班的頻繁攻擊，美軍是有心無力，他們只能通過操控輿論，對外傳遞美軍還能控制局面的假象，但誰都明白，美軍早就不得人心了，最終還是要撤兵。

之所以還不走，就是因為白宮要給這場戰爭畫個句號，要讓世界覺得美軍打贏了仗，是自己主動撤離的，而不是被塔利班趕走的，只有這樣，才能保持美軍繼續強大的形象。

但塔利班偏偏不給美軍這種機會，他們抱著，要走就趁早，不走就用炸彈招呼你的想法給美軍不斷製造麻煩，他們相信，接二連三的美軍死亡一定會給白宮帶來巨大壓力，如果特朗普還不撤軍，那麼他的票倉就會出問題了，到時候恐怕連民主黨都饒不了他。
