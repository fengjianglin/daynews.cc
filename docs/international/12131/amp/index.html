價值200萬的箱子失而復得，美籍華人張先生感慨：也就在中國能這麼快找回

天天要聞
2019-12-11

錢江晚報·小時新聞記者 朱麗珍 圖片由張先生提供

張先生夫婦是美籍華人，在洛杉磯開了一家拍賣行，也是珠寶收藏家。12月5日，他們準備回洛杉磯，可乘坐網約專車到達義烏機場時，發現最重要的一隻行李箱不見了。

裡面不僅有準備拿來拍賣的大量珠寶，還有美國護照、美國駕照、10張信用卡、小朋友的美國出生證、辦理簽證的資料、筆記本電腦、2個IPAD以及部分現金。

箱子里，光珠寶的價值就超過200萬元，更別提那些補辦起來很麻煩的證件了。

8點15分，發現箱子不見了；10點，飛機就要起飛。

怎麼辦？張先生一個大男人，差點急哭。

遺失的箱子

7:50——搭乘專車到機場趕飛機

張先生一家四口，這次到義烏是來辦事的，住了一個月。這段時間，就租住在義烏市區楊一村望湖公館。

住的時間比較長，臨走前，一家人整理了三個航空箱和三個小箱子，一共6隻箱子。

12月4日，張先生預約了第二天早上7點50分的滴滴禮橙專車。他們要搭乘早上10點的飛機飛北京，之後轉下午3點的飛機回洛杉磯。

12月5日上午，專車司機傅天德按照預約時間來接張先生一家。

當天天氣很冷，傅師傅幫忙裝箱子，上車前還特地看了看周圍，確定沒有遺落物件才開車出發。

8:15——最重要的一隻行李箱不見了

約25分鐘後，張先生一行抵達機場，清點行李時，張先生慌了：一隻銀色的18寸密碼箱不見了！

張先生一下子懵了。那隻箱子里，不僅裝著很多即將要進行拍賣的名貴珠寶，還有美國護照、美國駕照、10張信用卡、小朋友的美國出生證、辦理簽證的資料、筆記本電腦、2個IPAD以及現金，價值起碼超過200萬元。

箱子去哪裡了？誰都不知道。相比張先生，張太太還算冷靜，她和兩個孩子在機場等著，讓張先生馬上回住處去找找看。

傅師傅二話不說，就開車子帶著張先生往回趕。

箱子里很多證件

8:30——一路通電話，希望出現了

為了幫助張先生尋找行李箱，傅師傅取消了下一個訂單。在車裡，氣氛異常緊張。張先生都有些語無倫次，急得聲音發抖。

傅師傅一邊開車一邊安撫，又讓張先生不管怎麼樣先報警。哪怕東西丟了，到處都是監控，肯定能找回來。

張先生報警後，又給房東和打掃衛生的阿姨打電話，讓他們在原地幫忙尋找。聽說這事，房東也很著急，從電梯口開始，一路找到了大門口，可壓根就沒有箱子的蹤影。

「你要麼在門口左右兩邊都找找看？」張先生幾乎要絕望了，抱著最後一點希望，請求房東。

房東看了大門右邊，說沒有。電話這頭，張先生心都涼了：「哪怕真在路邊，上班高峰期人來人往，這隻箱子十有八九也被人撿走了。」

沒想到，房東又看了一眼左邊，立馬音量提高了：「你那個箱子是不是白色的呀？」

雖然張先生的箱子是銀色的，但他覺得，房東說的可能就是他的箱子。

8:40——箱子找到，東西一樣不少

「真的不知道怎麼形容當時的心情，司機師傅還在一路狂奔往回開，我一把抓住他的胳膊，靠在他的肩膀上一直說『找到了，找到了』。」張先生說。傅師傅安慰了他一路，一直到公館大門口，看到房東拎著箱子站在那裡焦急地張望，他懸著的心才總算放下來。

這隻箱子，不知道怎麼回事，居然跑到了門口兩隻垃圾桶的中間。因為視線遮擋，裝行李的時候大家都沒看見。

「估計當時行李太多，那裡又有點下坡，箱子滑到旁邊去了。」傅師傅猜測。

箱子沒有上鎖，在路邊待了40多分鐘，張先生下車後趕緊打開檢查，發現東西一樣不少。

「心徹底放下來了，那個心理過程我真的沒法用語言來形容，大悲大喜，失而復得。」張先生說。

9:00——抵達機場，順利趕上飛機

來不及喘氣，傅師傅帶著張先生又立馬返回機場，原本想讓張夫人和孩子們先登機，哪知道當時太著急，登機證件還在張先生隨身的包里。

傅師傅盡量用最快的速度，趕到了機場。最終，張先生一家順利登機。

分別前，張先生特地加了傅師傅的微信，給他發了一個紅包。

「這是我應該做的啊，還發來紅包，我挺不好意思的。」傅師傅今年37歲，7月份剛認證做了專車司機。收到紅包後，他還特地趕到公司諮詢，「我問公司這種紅包能不能收，他們說能收的，我才放心收下。」

禮橙專車司機傅天德

最新進展：以後來義烏，一定要約師傅吃個飯

經過這段像過山車一般的插曲，張先生一家四口順利抵達洛杉磯的家。

「當時忙著趕飛機，也來不及好好謝謝傅師傅，還有好心的房東。」張先生說，自己經常世界各地飛，受時差困擾，迷迷糊糊的，丟東西也是常事，「以前在墨西哥也丟過東西，損失不小，最後沒找回來。」

這次箱子里的珠寶，有些是馬上要拍賣的，有些是古代玉器，價值無法估計，起碼200萬元以上。失而復得，讓他很感動。「等再回義烏，我會約師傅吃飯，當面再好好感謝。」

而另一邊，為了表揚傅師傅助人為樂的行為，滴滴出行授予他「滴滴公益榜樣」稱號，「滴滴正能量在路上專項公益計劃」獎勵傅師傅2000元現金。
