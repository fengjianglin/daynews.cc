韓國接收首架全球鷹，一舉進入戰略偵察時代，未服役就亮出了肌肉

2019-12-25 1,156條評論 4,608次閱讀 3321人點贊

RQ-4全球鷹無人機，是世界上最著名的軍用無人機之一，因其優異的高空偵察能力以及較遠的航程而備受各國的青睞，除了美國以外，其他國家也一直想擁有這種先進的無人偵察機。就在近日，1架RQ-4B全球鷹高空無人偵察機便從遙遠的美國飛到了韓國，隨後這架無人偵察機抵達了韓國慶尚南道泗川空軍基地，並在這裡等待韓國軍方的接收。據了解，早在2011年的時候，韓國軍方便以對外軍事銷售向美國採購了全球鷹無人偵察機，這架全球鷹無人偵察機便是成品。

也是在時隔8年以後首次交付韓國，其餘的3架會在明年上半年交付，由韓國軍方檢驗以後加入部隊服役。在擁有了全球鷹無人機以後，韓國軍隊的監視範圍將擴散至周邊，通信監聽能力也能覆蓋白頭山附近地區，這對於提升韓國軍隊的情報收集能力有著極大的幫助，此外該無人機的交付也將讓韓國空軍直接進入了戰略偵察的時代。

RQ-4系列全球鷹高空無人偵察機的定位有點類似於當初叱吒天下的U-2偵察機，作為一款裝備有高解析度合成孔徑雷達並且能夠提供長程長時間全區域動態監視的無人機，對軍隊來說有著十分重要的價值，尤其是在綜觀戰場或是細部目標監視方面全球鷹可以把重要的信息傳輸到指揮部，甚至它還有潛在能力可以進行波譜分析的諜報工作，能夠提前發現危機和衝突，這如果在戰爭時候就能夠起到很大的作用。

韓國的RQ-4B其實是RQ-4系列全球鷹無人機的衍生版本，還有包括RQ-4A和RQ-4C，在航程方面，全球鷹絲毫不輸一些大型有人偵察機，它的最高速度可以達到每小時650千米，航程則為2.5萬千米，並且能夠在2萬米的高空對目標進行監視，這種高度一般的防空導彈根本就是束手無策。不過由於RQ-4系列球鷹無人機的單價過於昂貴，即便是美國軍隊也只能小批量的採購。

據了解一架RQ-4全球鷹包含地面站全系統的價格在7370萬美元，而美國海軍所使用的改進版RQ-4C則高達1.2億美元，與一架F-35隱身戰鬥機的價格無異了。但貴也有貴的好處，全球鷹無人偵察機的性能出眾，幾乎是世界上最先進的無人機，這將給韓國軍隊帶來質的飛躍，值得警惕的是，韓國空軍此次裝備全球鷹之後，其情報體系將會和美國更加緊密，更何況這次RQ-4B是越洋飛行而來，未服役就亮出了肌肉，只怕是想給對手一個下馬威吧。
