澳大利亞新州人口最新預測：2036年將超過1000萬

天天要聞
3 天 ago

準備好迎接人口大爆炸了嗎？新州政府今天發布的人口預測報告顯示，到2036年，新州居民人數將超過1000萬，相比2008年預測多了100萬。

據《每日電訊報》報道，10年前，州政府預計到2036年，新州人口為910萬，但現在這一數字已修訂為1010萬。此外，到2041年，預計新州人口增加至1060萬。規劃和公共空間部長Rob Stokes表示，新州政府的基礎設施計劃能夠容納不斷增長的人口，因此不必為此感到恐慌。

規劃和公共空間部長Rob Stokes

他指出，該預測報告也表明，新州各個區仍是澳人最理想的居住地。Stokes說道：「我們擁有完善的基礎設施計劃，以確保我們的子孫後代所需的房屋、醫院、學校、道路和其他基礎設施和服務，這些預測只是數據，而不是命運，但會有助於我們在一些決策上作出決定。」

該報告預計新州女性人口預期壽命將從2016年85.8歲提高到2041年的88.9歲，男性則從同期的81.8歲增加到86歲。

此外，在未來的22年，65歲及65歲以上人口預計將增長100萬以上，達到220萬以上。悉尼將不會受到人口繁榮的全部衝擊，因為每一人移居悉尼，就有兩個人搬離。而離開城市的多為年輕家庭和退休人士。
