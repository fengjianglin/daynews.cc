這種魚體型巨大，卻因太笨差點被巴西人吃滅絕，在中國卻不一樣

2019-12-08 186條評論 3,088次閱讀 1428人點贊

隨著經濟水平的不斷提升和發展，人們有閑情的時候就會選擇去世界各地旅行，南美洲的巴西是一個很不錯的國家，不少的中國人都喜歡去當地旅行，這裡有熱情的人們、足球健將等，但是巴西存在一個世界最大的熱帶雨林，中國地區有著豐富且多樣化的物種，生態環境是出奇的好，在雨林深處有一種雨類在這裡混的十分差勁，甚至被吃到絕種。

它的名字叫做巨骨舌魚，這可是個大傢伙，並且還是世界上最大的淡水魚之一，它的性格非常的怪異，沒事在岸邊聽到了一些動靜就會主動的前去查看情況，這時候的人們看見了當然沒有放過這個大傢伙，它身上的肉都可以供幾個人飽餐一頓了，甚至有時候只需要一根木頭就可以將其捕獲。

這種魚類成熟後的身體最長可以達到6米，體重也是有著相當驚人的三百斤，這是經過人們在亞馬孫流域的河流里捕獲的數據，並且這些魚類的價格也不是很便宜，一條小小的10公分的巨骨舌魚幼魚就可以賣到三百塊錢上下不等的價格，當然成年的這些魚，假如有2米的話，那就可以在市上賣到九千左右的金額，但是必須是原生態捕獲的。

由於這些魚類的肉質肥美且營養成分十分的高，許多的人就為了謀求利益，大量的捕捉這些巨骨舌魚，因此到後來這些魚類就持續的減少，為了保護起這一淡水中存在最大的魚類，政府嚴令禁止捕捉這些魚類。為此，你是怎麼看的呢？
