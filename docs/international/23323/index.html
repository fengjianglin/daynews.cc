排第一的2019年加拿大年度新聞讓人語塞

2019-12-19 23條評論 1,507次閱讀 419人點贊

這是我們居住的地球，如果人類消失了，地球會變成什麼樣子？

彷彿覺得人類從來都不屬於地球！

相信不少人都聽說過，2012年世界末日的傳說，七年過去了，地球依然健康地存在著。

但是，今天一條登上Global News的新聞，讓情報員心裡不是滋味。

我們以為依舊健康的地球，其實正飽受摧殘，而地球也正用它自己的方式，報復著地球上的物種…

登上2019年加拿大年度新聞

9月下旬，成千上萬的加拿大人走遍全國各地，要求政府應對氣候變化，而這也成為了加拿大歷史上規模最大的抗議活動之一，這正與瑞典少女Greta Thunberg的國際氣候罷工運動遙相呼應。

許多環境保護認識就認為，這意味著加拿大氣候變化時代的到來。加拿大氣候行動網路負責人Catherine Abreu表示：「2019年就像加拿大的氣候喚醒之年。」而氣候變化也是加拿大全國各地的記者和編輯集中報道的話題，被選為2019年加拿大年度新聞第一名。

看似距離我們很遠的「氣候問題」，其實正悄悄地影響著我們的日常生活，像溫水煮青蛙似的，在不知不覺中已經造成了無法挽回的創傷。

十年前，氣候變化比學術問題更具現實意義，但近年來，幾乎沒有加拿大人直接受到因氣候變化所引起的極端天氣的影響，如：洪水，山火，大風暴，寒潮，熱浪，冬季更長，夏季更短。

6月，當議會投票宣布我們正面臨氣候危機時，安大略省東部，魁北克省和新不倫瑞克省的部分地區剛剛擺脫三年來第二次百年一遇的洪水。

氣候變化細思極恐

接下來這組數據，可以說是細思極恐……

加拿大環境部的氣象學家David Phillips表示，在過去的135年，全球氣溫升高了一度。但是，在加拿大部分地區，過去70年，氣溫上升了4至4.5度。也就是說，加拿大隻用了一半的時間，氣溫上升了全球平均值的兩倍。

加拿大環境部繪製的從1948年到2012年，全加拿大境內的溫度增長圖

氣候變化行動網組織的負責人Catherine Abreu表示，海岸附近的居民正在經歷海平面上升，侵蝕，和水災。尤其是原住民部落，他們因天氣變化失掉了原有的生活方式。而在過去的10到15年，加拿大東部受氣候變化的影響也逐漸顯現。

約克大學假設在人類不做出任何改變的情況下，製作出了氣候變化模型。結果顯示，2050年，多倫多將會有51天氣溫高於30度。而在2070年，這個數字會達到77天。目前，這個數字是16天。

2013年，多倫多地區的冰風暴，政府事後花費約為1億零6百萬加元。

而到了冬天，因為氣溫變化，冰雨天氣將會增加50%到60%。在草原省份，農作物生長周期會隨天氣而延長，與此同時，森林山火和霧霾天氣也隨之增加。

報告顯示，從1983年到2008年，災害保險每年支付的數字是2.5億加幣，但是，從2009年開始，巨大災害保險大幅上升，現在每年的平均支出是18億加元。

BC 省森林大火造成的霧霾

加拿大西北地區因紐維克（Inuvik ），越來越多的灰熊出現在北極波弗特海域附近的陸地和島嶼上，專家認為這是氣候變暖所致。加拿大西北地區的因紐特人散居在約100,000平方公里的北極苔原區域，這裡一向有灰熊出沒，但現在出現在那裡的灰熊數量大增。

因紐維克地區的生態系統在過去30年中變化巨大，通常每年8月開始冰凍季節，但這幾年冰凍季節推遲到9月底甚至10月初。融雪和生長季節大大延長，改變了灰熊的生存條件。

焦慮與改變並行

與氣候變化隨之而來的是空氣污染，淡水資源缺乏，人類面對更多的不確定性。

心理學家發現，極端天氣也會對人的心理造成影響，被稱為「生態焦慮症」。意即面對自己無法掌控的氣候變化，尤其是經歷過水災森林火災，瞬間失去家園的人，其心理傷害遠遠比之前想像的嚴重。

據Global News的報道，許多編輯之所以將氣候變化這一話題排在第一位，是因為這一嚴峻的問題切切實實地對我們生活造成了影響。卡爾加里先驅報和卡爾加里太陽報的高級編輯保羅·哈維說：「沒有比人類對我們自己的星球進行人工改造更大的故事了，這是明顯存在的事實。」

此前加拿大新任環境部長Jonathan Wilkinson競選總統，主要是因為他想採取行動解決氣候變化這一問題。

而加拿大政府正面臨著一個難題：如何在不以石油，天然氣，製造和運輸為關鍵的經濟困境的情況下大幅減少排放。與歐洲的一些小國不同，加拿大人居住分散，冬天的供熱和電力需求很高，很難在對氣候採取積極行動的同時，又兼顧經濟繁榮。所以，一切針對氣候的國家層面的改變，全指望這一屆政府能夠拿出些切實有效的方案來。

而在我們的日常生活中，低碳、環保也必將時刻圍繞在我們身邊。如果不做出改變，那在百年甚至千年以後，人類也只能在無孽可做之時，將千瘡百孔的生存家園一點點還給地球！
