當今的烏克蘭人最喜歡美元，但是最歡迎的卻不是美國人

2019-12-21 3,595條評論 4,829次閱讀 3146人點贊

冬季的烏克蘭首都基輔的街頭，其實是相對非常寒冷的。基輔這個城市雖然在整個歐洲算是比較靠南邊的首都，比倫敦、柏林、華沙、莫斯科的緯度都要更靠南，但是這裡的冬季氣溫，並不比以上4個更靠北的日常氣溫高。因為雖然名義上比較靠南，卻和東亞的大城市哈爾濱已經在一條大體的緯度線上。而其他4個更靠北的首都，瀚海狼山認為或多或少，都會受到北大西洋暖流這個歐洲天然超級暖氣升溫增濕的關照。而周邊都是陸地的基輔就很難享受這種大洋暖流的待遇了。基輔再往南幾百公里是封閉的黑海，這裡冬季沒有暖流反倒有惡劣的風暴，因此是指望不上的。基輔街頭的寒意，不但來自成堆的積雪，更來自越來越不景氣的經濟。當然現在的基輔街頭，也會偶爾出現價值百萬的豪車，但是那是極少數富豪的座駕。大多數普通的車輛，

和歐洲西邊的國家對比，明顯不在一個檔次；市內的車輛和行人也不多，市面的蕭條可以感受的出來。不過基輔街頭，比街上行人還多的，是大大小小的外幣兌換店鋪，幾乎到了幾十步就有一家的程度。當今的基輔根本算不上一個歐洲著名的旅遊城市，可以見到的外國遊客並不多，而冬季則更少了。那麼為何有如此多的外幣兌換店呢？很簡單，這些店鋪大多時候服務的對象，並不是天然持有外幣需要兌換的外國人，而是更多的服務本國的國民換匯。獨立以來，由於烏克蘭的法定貨幣格里夫納，對全球主要貨幣的幣值越來越低，最近幾年更是貶值起來，烏克蘭政F部門都不知道會貶值到哪裡，屬於貶到哪裡算哪裡。因此普通烏克蘭人，只要一拿到當月的工資或者退休金，就趕快到貨幣兌換店兌換成美元。認為這樣起碼能部分保值。

不至於一個月的退休金到了最後，連幾個麵包都買不起。當然手裡本就不多的工資和退休金都換成了美元，平時買菜，交水電費還需要支付本國貨幣，就只能把美元再部分的換回來。這樣本國人來來回回的高頻兌換，自然讓一些頭腦靈活的人看到了「商機」，因此只要稍微有一點資本和背景的基輔人，就會在街頭開一個外幣兌換點，這樣自然就會有利潤送上門。因此不但街頭固定的外幣兌換店比便利店還多，便利店往往也兼營外幣兌換。還有大量的、各種街頭流動兌換的外幣販子，他們給的匯率會比有固定店鋪的兌換點更「優惠些」，但是一旦收到假幣或者被J C當非法兌換抓了還沒收資金，就只能自認倒霉。

烏克蘭是歐洲國家，也一直想加入歐盟「享福」，但是就是不過關。本來以為烏克蘭人更喜歡歐元，誰知道在烏克蘭最被接受的是美元。感覺好像每個烏克蘭人都在積攢美元，準備最後移民美國一樣。其實99%的烏克蘭人一輩子都實現不了這樣的夢想。自己手裡的一點工資和退休金雖然可以任意兌換成美元，但是還不夠買張去西歐的火車票的，更不用說到美國的簽證費和機票錢了。

很少有烏克蘭人，有能力拿出上千的美元去兌換。如果有人突然拿出數千甚至上萬美元現鈔在街頭兌換，往往會被認為是騙子。去年就發生過AN225大飛機80歲高齡的設計師，在基輔街頭先被流動換外幣的混混們襲擊，而後又被隨後趕來的當地JC再次打了一頓的事件。很大一個原因，很可能就在於這位上了年紀的設計師，有特殊外匯來源，而又想多換點本幣改善生活，結果去找流動攤販兌換而被當成了假幣騙子暴打。

至於當今烏克蘭的飛機廠和坦克工廠的大多數普通職工，連被外幣販子打一頓的機會也沒有。因為他們甭說美元外快，就是本幣的工資都36個月到48個月沒發放了。都在盼救世主一樣的盼望公司和工廠被好心人收購。才能補發拖欠多年的工資和取暖費。但是有些已經談好的收購，超級大國居然橫插一杠，無理阻攔；但是自己又不想出哪怕一美分來接盤。因此現在從烏克蘭普通工人到烏克蘭總統，都對超級大國的這套把戲厭倦了。當今的烏克蘭人，雖然還最喜歡美元，但是卻對發行美元的國家有了新想法。對誰誰家到烏克蘭的商務客人和旅遊者都非常的熱情。因為誰能真正幫助烏克蘭，現在是人就能看得清楚。
