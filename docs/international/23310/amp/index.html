墨西哥黑幫多可怕？與政府激戰十年死傷十萬，壟斷世界毒品市場

天天要聞
2019-12-19

提及毒品，我們總是會情不自禁想到一個著名的販毒基地——墨西哥。

幾十年前，墨西哥便已經淪為了犯罪者的天堂。俗話說得好，”離天堂太遠，離美國太近”，美國廣闊的毒品消費市場是墨西哥販毒集團猖獗的根本原因。

墨西哥政府時常需要看毒販臉色度日，縱然這些匪徒行兇作惡，時常讓人橫屍街頭，可墨西哥政府也拿他們沒有絲毫辦法。可是萬物都有發展過程，墨西哥毒梟也並非從一開始就如此強大，那這群猛獸是如何一步步騎到政府頭上的呢？他們又能猖獗到何種地步呢？這還需要回溯近百年來的墨西哥歷史。

19世紀二三十年代的墨西哥並未同今天這般，毒品泛濫成災，那時候”革命制度黨”一直把持著墨西哥社會生活的方方面面，包括毒品。在70年代以前，墨西哥政府在這個黨派的帶領下，為國內創造了相對穩定的環境。雖然當時在墨西哥政府默許下，墨西哥人喜歡販賣大麻到美國，但都是小打小鬧。

然而，時間進入70年代以後，一切都開始變了。當時拉丁美洲局勢動蕩，經濟不景氣，可各方勢力都需要錢來打仗，於是他們將注意力放到了古柯葉這玩意兒身上，因為這種葉子能提煉出新型毒品可卡因，其毒性比原先的鴉片更能讓人感到興奮。於是拉丁美洲的人就大肆製作可卡因，隨後將其售往美國。

80年代，哥倫比亞毒梟們便佔據了美國市場。這批毒梟將毒品運往美國的渠道有兩條，一條是通過加勒比海海運，另一條是從墨西哥陸上輸送。由於當時這批毒梟將佛羅里達州第一大城市邁阿密變成兇殺之都，美國里根總統便開始打擊他們的海上渠道，於是他們就只能借道墨西哥。

於是乎，墨西哥的黑幫們就開始當起了哥倫比亞到美國販毒交易的中間商。但是這個中間商的差價沒有這麼好賺，因為他們有極高的運送風險和成本。如果一旦他們中途將貨物弄丟，那必須按照市場價格賠償給哥倫比亞毒梟。此時的墨西哥黑幫組織處於弱勢地位，又想分一杯羹，自然爽口答應了。

漸漸地，墨西哥”中間商”的規模越來越大，越多黑幫的加入使得墨西哥政府開始插手毒品貿易。然而，在墨西哥官員的絕對強權下，這批毒販不過是待宰羔羊。所以這些黑幫便成了政府圈養的肥羊，他們做生意的不少好處都需要上交政府，賄賂官員，美其名曰納稅。所以墨西哥政府為了明面上過得去，特意將各個大黑幫組織成”卡特爾”。

卡特爾是什麼呢？術語解釋為”由一系列生產類似產品的獨立企業所構成的組織，集體行動的生產者，目的是提高該類產品價格和控制其產量。”也就是說，當時的墨西哥黑幫成為了社會上合法的生產組織。毫不誇張地說，當時的”卡特爾”就是政府的黨屬社會事業單位。

卡特爾模型

本來，按照這樣的走勢下去，當時的革命制度黨能夠通過控制這些毒販的頭目，逼迫卡特爾永遠其效命。然而，一場意外卻打亂了這一切。

1984年，卡特爾的0.7噸可卡因被美國警察沒收，他們不得不賠償近億美元給哥倫比亞毒梟。怒火中燒的卡達大佬們竟然幹掉了美國緝毒局特工，並讓其橫屍街頭。這一下美國火了，當即逼迫墨西哥執政黨政府交人。

其實早先美國政府就曾向其他政府要求叫出毒販，但當地政府不以為然，結果美國特種兵直接進入當地抓走了該毒梟，拘捕回國審判。所以不敢得罪美國的革命制度黨便將當時的頭目們交了出去。

掀起炫富狂潮的墨西哥毒梟

就在這件事發生後不久，冷戰結束，墨西哥的特務機構也紛紛解散，一系列變化下來，政府也缺乏了對中層卡特爾毒販的控制。與此同時，哥倫比亞的”老大哥們”也被美國關押或者刺殺，墨西哥中層卡特爾們迎來了前所未有的發展契機。

整個90年代，是卡特爾內部之間的血腥爭奪與資源整合的時代。此時湧現出來的新卡特爾大佬們不再同第一代大佬一樣服從政府的管理，他們的爭奪已經開始干涉到了國家的穩定，橫屍街頭和肢解屍體扔到市區等街景已經讓人們習以為常。此時的革命制度黨卻因為有其他黨派的出現，無暇顧及這群魔鬼。90年代後期的墨西哥毒梟已經有了當年哥倫比亞毒梟的猖狂和風采。

墨西哥毒梟的瘋狂報復

但是同樣的，他們的囂張跋扈也讓他們遭遇了和哥倫比亞毒梟一樣的下場，美國政府再度插手干預墨西哥。但是舊的魔鬼倒下，總有新的魔鬼來取代他的位置。如果說80年代的毒梟是腐敗政府的走狗，90年代的毒梟是叱吒風雲的大亨，那麼新世紀的毒梟則是手握重兵的軍閥。而代號”Z字頭”的卡特爾便是這一切的催化劑。

本來Z字頭組織只是墨西哥一個卡特爾大佬的衛隊，當時這個卡特爾大佬在墨西哥政府忙於黨爭的時候，向墨西哥特種單位挖人來保護自己。當時的墨西哥特種兵也看見革命制度黨搖搖欲墜，沒有前途，便欣然接受了豐厚條件，為毒梟效命。由於這支部隊之前軍方稱號為”Z”，所以人稱他們為Z字頭。

原本Z字頭只是保護卡特爾頭目的存在，但隨著時局動蕩，這個組織也開始被用來在卡特爾之間搶奪地盤。由於他們原本就作戰能力極強，”戰功累累”，於是乎，當初聘用他們的卡特爾大佬就獎勵了他們屬於自己的地盤。

隨著時間的流逝，這個組織實力越發壯大，可在卡特爾組織內部卻沒有相對等的權利，於是他們便和老東家火併。就在此時，其他卡特爾組織也想從二者火併中撈點好處，結果逼的這個具有軍事化傳統的Z字頭卡特爾走向極端，以軍事手段來血腥鎮壓。

由於毒販之間得火併儼然向專門的軍事化發展，Z字頭也開始大量招募軍中士兵。後來，不管新成員是否有過訓練，都要接受Z字頭卡特爾的再次軍事培訓，以實現專業的指揮作戰。現今的Z字頭顯示是一個成熟的軍事組織。在Z字頭的逼迫下，其他的組織也變得更加恐怖，現在墨西哥就宛若軍閥割據的狀態。

2008年，當新的黨派成功逼迫革命制度黨下台，可是當該黨執政以後卻發現，革命制度黨還是和卡特爾狼狽為奸，所以便決定消滅他們。可是這個新的民主黨派不會料到他們遇到的是多麼極端和變態的對手，以致於在近十年的時間中，墨西哥因毒販和政府軍火併而喪失了10萬6千人性命。

時至今日，墨西哥的卡特爾們不再是當初的毒販，而是許多的恐怖組織，美國的市場也不再是他們販毒的唯一理由，因為他們已經將生意做到全世界，甚至壟斷了英國和東歐的毒品市場。

除此之外，他們干涉別國秩序，只要對方利益足夠打動他們，他們為派人為顧客刺殺敵國的政要。而且他們也同世界其他地區諸多黑幫和恐怖組織合作。現今的墨西哥，可謂已經成為卡特爾們了犯罪的天堂。
