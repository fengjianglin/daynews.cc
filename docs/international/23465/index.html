美國7個月大的嬰兒贏得拍賣，成為美國最年輕的市長

2019-12-19 317條評論 1,829次閱讀 510人點贊

據《新聞周刊》12月16日報道，12月15日下午，7個月大的嬰兒——威廉·查爾斯·麥克米利安（William Charles McMillian）成為了美國最年輕的市長，他被任命為美國德克薩斯州白廳（Whitehall, Texas）的名譽市長。

當日下午4點30分左右，查爾斯身著燕尾服出席了市長任命儀式，其養父母查德（Chad）和南希·麥克米利安（Nancy McMillian）在一側陪伴著他，約150名當地社區的成員參與了慶祝儀式，見證了這位年齡最小的市長的就職典禮。

參加典禮的41歲的喬西·富爾茨（Josh Fultz）是德克薩斯州納瓦索塔鎮（Navasota）的一名議員。據富爾茨稱，在任職典禮的過程中，查爾斯只是坐在那裡，看上去只是一個快樂的小孩子。

富爾茨表示，整個典禮都是由查爾斯的父母精心策劃的，像以往的就職典禮一樣，只是規模要小上一些。由於還是嬰兒的「小市長」看上去十分可愛，幾乎在場的人都想抱著他，和他一起散步，大家都很愛他。

查爾斯甚至有自己量身定製的就職宣誓：「我，威廉·查爾斯·麥克米利安，鄭重承諾，我將忠實履行白廳市長的職責，並將盡我所能，善待每一個人，提升生活水平、推動領養和良好、乾淨的鄉村生活，鋪設所有的碎石路，給志願消防隊帶餅乾，捕捉最大的鯰魚，保護白廳社區，所以請幫助我，媽媽和爸爸！」

據報道，每一年，市長的職位都會在白廳志願者消防局舉辦的燒烤募捐會上進行拍賣，出價最高的人將贏得這一榮譽。10月20日，查爾斯成為了今年的獲勝者，美國也就此誕生了史上年齡最小的市長。

下個月，麥克米利安一家將前往華盛頓特區會見德克薩斯州參議員特德·克魯茲（Ted Cruz）。
