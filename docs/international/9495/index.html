開放共贏 外資持續看好中國市場

2019-12-03 1,253條評論 5,015次閱讀 3606人點贊

央視網消息：在全球經濟依然面臨較大下行壓力的背景下，中國進一步擴大開放，為世界經濟注入新活力，同時不斷優化營商環境，吸引著越來越多的外資通過與中國合作來實現共贏。

前幾天，在武漢舉行的2019全球採購大會吸引了來自俄羅斯、塔吉克、新加坡等國的採購商。來自俄羅斯的魏歌德說得一口流利的漢語。他告訴記者，今年8月剛剛在中國註冊了一家採購平台公司，不到兩個多月，已經有2000多家中國企業成為平台會員。

俄羅斯國家電子採購專業委員會副會長 魏歌德：中國企業的需求很大，所以我們的市場規模變得更大了，供應商也很有興趣跟中國的企業合作，這給我們帶來了一個非常好的機遇。

不只是俄羅斯商人看準了中國市場，美國供應管理協會與中國物流與採購聯合會近期簽署了戰略合作協議，要在中國合作培養供應鏈管理人才。作為一家採購經理指數的統計機構，美國供應管理協會一直關注著中國經濟。

美國供應管理協會總裁 托馬斯：我不認為中國會出現經濟衰退，中國經濟的發展是強勁的，「一帶一路」將為中國帶來新的市場。我相信中國經濟能夠很好地應對短期挑戰，並繼續保持增長活力。

在全球貿易活動低迷的背景下，加強同中國的合作成了很多外資的共識。全球最大的集裝箱航運集團之一的馬士基，不久前在寧波投資建設國際物流中心，在建的梅山一期倉庫將是馬士基全球第一個全自動化倉庫。

馬士基亞太區供應鏈產品總監 安東尼·蘭德爾：對馬士基來說，中國市場非常重要、也是我們最大的一個市場。這裡有充滿活力的經濟和快速增加的中等收入群體。

外資紛紛進入中國，這與中國營商環境的持續改善是分不開的。上周末在財政部和世界銀行主辦的優化營商環境研討會上，外國駐華商會代表們紛紛給中國營商環境點贊。

中國歐盟商會副主席 彥辭：我們不少會員企業都認為，中國現在的研發創新環境要優於全球平均水平。

中國美國商會會長 畢艾倫：中國在持續營造透明、公開、開放的市場，公共採購領域也放開了。

中國日本商會會長 小野元生：《外商投資法》與《優化營商環境條例》明年起就要實施了，我們抱有很大期待。

商務部最新數據顯示，1-10月全國新設外商投資企業33407家，實際使用外資7524.1億元人民幣，同比增長6.6%。中國持續擴大開放，進一步優化營商環境，與世界共發展，贏得了外商、外資的尊重與認可。
