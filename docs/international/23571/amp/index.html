現實版變形記，九百萬鎊小女孩去貧窮人家，大呼看不懂

天天要聞
2019-12-18

作者：黑桃K

現在的人們最喜聞樂見的，就是通過電視看看別人的生活。前有美國真人秀《與卡戴珊同行》記錄卡戴珊一家「壕無人性」的日常生活，後有中國《變形記》描寫城市裡的孩子去到偏遠山村如何生存。

《變形記》里有的山村實在是離我們的生活太遠了，可能是為了形成強烈對比吧，就算是讓菌菌生活下來可能也並不容易。

而菌菌今天要講的現實版變形記，這是英國Channel 5的節目富家孩子《 Skint Christmas》，但是要比《變形記》看起來真實的多啦！

就好像是把真正富家的孩子放到你我身邊的故事了。我們先一起來看看這個身價900萬英鎊的18歲小女孩阿普爾的日常生活是什麼樣子的吧～

阿普爾來自白金漢郡，她和父母住在一棟價值900萬英鎊，有7個卧室和10個浴室的豪宅里。別墅裡面有三棵聖誕樹，電影院，健身房，桑拿房，酒吧，私人電梯和一個保齡球館。

阿普爾的父親是個事業有成的商人，他在IT領域做的生意做的非常成功。阿普爾也在父親的幫助下創辦了自己的服裝公司。

在他們家中，一共有五個裝滿名牌服飾的衣帽間，其中包括400英鎊的運動鞋和500英鎊的高跟鞋，以及各種各樣的奢侈品手錶，比如Cartier等等。

阿普爾這別墅可能比貝克漢姆一家豪宅還要壕！可是說是吃喝玩樂應有盡有了，菌菌要是住在這裡，怕是一個月都不用出門，光在別墅里隨便玩玩一個月就過去了吧～

阿普爾一家人的聖誕節其實沒有固定的預算，他們一般花費5,000到10,000英鎊，他們認為聖誕節每年只有一次，所以一定要好好過呀！

下面再來說說那個普通人家，伯克一家由母親琳賽、父親艾倫、16歲的克萊頓、11歲的懷亞特和4歲的崔妮蒂三個孩子組成。

他們每周的生活費只有160英鎊，所以對於阿普爾一家最重視的聖誕節，他們只能拿出15英鎊作為聖誕晚餐的經費。

艾倫曾經是一位成功的兒童藝人，周薪約800英鎊，但是之前經歷了幾次中風和喉嚨不適，沒辦法的他只好辭職。母親琳賽也不得不辭掉她的餐飲業工作，照顧她的丈夫和他們的三個孩子，成為了家庭主婦。

就這樣，生長在富豪家庭的阿普爾和英國的一個普通大家庭相遇了，他們一起提前享用了聖誕晚餐。

阿普爾和父親艾倫一起準備晚餐，這應該是這位小公主人生第一次給土豆削皮吧哈哈～

阿普爾震驚地發現，他們的聖誕晚餐裡面沒有火雞，而是用普通的雞代替了。用雞來替代火雞是他們家省錢的方法之一，晚餐中的蔬菜，也是從超市買來的冷凍蔬菜。

這個家庭也為了節省電費晚餐桌上所有食物幾乎都是油炸的，因為如果使用大烤箱的話會很浪費電費。

阿普爾說：「雖然薯條很棒，但是我不太喜歡油炸食品，而且我也不認為聖誕晚餐里會有油炸食品…」但是無論怎樣，她還是開開心心的吃完了這頓飯。

晚餐後，這個家庭的媽媽和阿普爾聊天時說到：「當你想給孩子們想要的東西，卻由於預算原因你不能給孩子時，這真的很讓我難過。」

當阿普爾意識到她腳下穿的這雙400英鎊的運動鞋可以支付孩子們所有的聖誕節禮物時，她哭了起來。是被觸動了吧，她從來沒想過在英國還有人過著這樣的生活，她應該不知道，這樣的日子才是大部分人現在正在過的，她是個幸運的人。

等她回到自己家中，阿普爾說：「我會很感激有這樣的一個機會，因為我一直在想每個人都會有一頓豐盛的聖誕晚餐，你可以省點錢，但我想這是不可能節省的，我第一次意識到有人會為了省錢而全吃油炸食品。」

與《變形計》不同的是，富家的小公主謙遜有禮貌，普通人家的孩子也沒有覺得自己的生活有多麼不堪，整個節目沒有刻意的製造衝突和對比。

這雖然是一個英國的階層故事，但並不殘酷不是嗎？
