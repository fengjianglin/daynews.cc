英國飛行員長年虐待未婚妻，導致她精神失常殺死兒子後自殺

2019-12-22 1,928條評論 4,158次閱讀 1510人點贊

據英國《每日星報》2019年12月20日報道，本周英國曼徹斯特刑事法庭宣判了一樁令人唏噓的案件，33歲的媽媽萊西卡因殺死自己4歲的兒子被判入獄15年，她的飛行員未婚夫彼得·奇爾弗斯因常年虐待她被判18個月監禁。在過去8年的生活里，萊西卡承受了太多來自未婚夫精神和身體的虐待，導致她精神失常企圖自殺，而在自殺之前，她親手殺死了她們的孩子，雖然她自己最終被搶救了過來，但這對她來說也不知道是好事還是壞事。

圖：2011年奇爾弗斯和萊西卡第一次見面

奇爾弗斯是瑞安航空的一名飛行員，他和萊西卡第一次見面是在2011年，當時他們一見鍾情迅速墜入愛河。後來他們同居住在利物浦的一間小別墅里。但從那時開始，奇爾弗斯的行為變得越來越糟糕，他開始用各種方法虐待萊西卡，而且越來越嚴重。

據悉，奇爾弗斯的轉變，源於他的嫉妒心理，他始終覺得萊西卡和其他異性關係走得太近。於是，從一開始只是在親吻時開玩笑似的捏住萊西卡的鼻子使她窒息，到後來逐漸發展成直接對萊西卡拳腳相向。

圖：2012年奇爾弗斯和萊西卡訂婚

圖：奇爾弗斯

在2012年他們還是訂婚了，好在訂婚之後萊西卡就出差到迪拜工作6個月，短暫的分離對他們的關係來說或許是件好事。但是，當萊西卡回到英國後，她和奇爾弗斯搬到了特倫特河畔斯托克，從那開始萊西卡噩夢般的生活又開始了。後來，萊西卡發現奇爾弗斯竟然和同事發生了外遇，她想分手，卻發現懷了他的孩子。奇爾弗斯對她懷孕的消息非常憤怒，他要求她墮胎，理由是孩子會毀掉他的生活和事業。他還指責萊西卡想要孩子是自私且混蛋的行為。

圖：萊西卡和兒子詹姆斯

2015年9月，他們的兒子詹姆斯出生了，當萊西卡帶著兒子去投奔奇爾弗斯時，又一輪虐待開始了。奇爾弗斯認為，兒子奪走了他的自由，覺得萊西卡很噁心。於是他變本加厲地虐待萊西卡，比如掐她的脖子，強迫她吃掉自己的頭髮等。奇爾弗斯還威脅她：「如果你試圖逃跑和躲藏，我會找到你並殺了你。」

2016年，他們搬到了曼徹斯特郊外一個名為Wythenshawe的新住宅。直到2017年8月，萊西卡發現奇爾弗斯和他的情婦還繼續來往，並且他的情婦還和他在同一個地方工作，這成了壓垮萊西卡的最後一根稻草。

萊西卡精神崩潰決定自殺，但是在她自殺之前，她先上樓將兒子詹姆斯殺死。當天，奇爾弗斯晚上11點55分回家後發現萊西卡受了重傷，於是把她送進了醫院。凌晨12點05分，警方趕到現場，發現孩子已經死亡。

後來，萊西卡被拘留，她在牢房裡發表了一份聲明，稱自己感到「醜陋、毫無價值、無助、羞愧和內疚」。聲明中寫道：「讓公眾了解彼得·奇爾弗斯對我的虐待是很重要的，他毀了我的一生。」萊西卡承認了過失殺人罪，她被判處15年監禁。而彼得·奇爾弗斯只被判入獄18個月

在本周的量刑聽證會上，法官帕特里克·菲爾德說：「奇爾弗斯對待萊西卡就像對待自己的財產一樣。他並沒有對自己的行為有悔意，不僅如此還試圖把錯誤和責任轉移到他人身上。」
