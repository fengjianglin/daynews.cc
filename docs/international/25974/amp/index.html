英國單親媽媽從不採取措施，九個娃之後又產子，全職養娃自得其樂

天天要聞
2019-12-21

圖源：babycouture.in

照顧孩子是一件很辛苦的事，尤其是照顧小的孩子，吃喝拉撒睡都需要面面俱到。一般一個家庭有兩個孩子就夠父母忙的了，但是英國有位女子，雖為單親媽媽，在交友時卻從不採取防護措施，連生九個孩子之後，不顧醫生的勸阻，堅持生下了第十個孩子。

英國《鏡報》12月18日報道，英國女子索尼婭·羅克林住在英國南部港口城市南安普敦，她的大家庭由她和十個孩子組成：尼克拉，19歲；拉亞，17歲；香農，16歲；科恩，14歲；葉林，13歲；凱，10歲；察娜，7歲；路易斯，5歲和萊克西，1歲，還有一名新生兒丹尼爾。

因為要一個人照顧這麼多孩子，索尼婭根本沒有辦法工作，帶著孩子們住在政府的救濟公寓里，靠救助金生活。每個月的錢除了交房租，其他的都花在孩子身上了，其中最大的開銷就是孩子們的吃。

索尼婭每天早晨六點半起床，儘管孩子們吵鬧讓她忙得不可開交，她還是堅持親手做飯給他們吃，然後收拾準備讓孩子們都按時到校不遲到，是一項很艱巨的任務。

把孩子們送到學校，索尼婭就去超市購買一家人的東西，儘管每個月的生活費都很緊巴，索尼婭還是有辦法精打細算把開銷控制在預算之內，買東西都是打包，這樣平均單價會便宜一些。每個月的錢大部分都花在了孩子們的吃上，不管怎樣餵飽孩子們的肚子都是她的首選項。

圖源：Parents

有人說索尼婭不工作，就可以帶著孩子們開開心心地生活，還有錢花，但是索尼婭覺得說這話的人不了解她的辛苦，讓他們在自己的位置上生活，他們可能連一天都堅持不到。為了讓孩子們可以花更少的錢吃得更好，索尼婭每天就堅持自己做飯給孩子們吃，有時候她都覺得自己像是開飯館的。

儘管如此，索尼婭還要生第十個孩子，但是醫生並不建議她繼續生，在生了這麼多孩子之後，索尼婭並不擔心，生下了她的第十個寶寶。生這個孩子產程持續了十三個小時，但是索尼婭回到家以後就開始洗衣做飯帶孩子，並沒有受太大的影響。

很多人擔心單親家庭孩子的心理問題，但這在索尼婭家裡似乎不是個問題，孩子們都很體諒母親的辛苦，也都很習慣並歡迎新的家庭成員的到來，兄弟姐妹互相玩鬧幫助，也是很熱鬧的一家人。
