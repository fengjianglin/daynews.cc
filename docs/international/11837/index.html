英國女王正在經歷著一個艱難的時刻，她正努力撰寫聖誕演講

2019-12-13 934條評論 3,458次閱讀 566人點贊

12月11日英國媒體《每日星報》報道了英國女王和聖誕節的事情。英國女王現在正在努力的撰寫自己的聖誕節演講稿。如今已經93歲高齡的英國女王如今有些心力交瘁。對於她來說，在經歷了動-盪的一年後，她覺得自己的精力有些不夠用了。

​

這一年可以說英國皇室問題不斷。最近安德魯王子的事情就搞得大家都心情不好。他和愛潑斯坦之間的友誼已經引起了很大的爭議。

​

最近愛潑斯坦的醜聞牽扯到了不少人。而安德魯王子自己也因為最近爆出的醜聞也讓皇室感到蒙羞。他本人也不得不退出了所有的皇室職務。而安德魯王子的女兒最近的日子也不是很好過。

​

而在今年早些時候，讓大家炒作的一個話題就是女王的兩個孫子之前的事情了。今年威廉和哈里的之間的矛盾也被大家熱議。

​

甚至哈里還公開的說了那樣的話，對於整個皇室來說，這都是令人蒙羞的。不少皇室成員都無法接受哈里和梅根在採訪時說的那些不利於團結的話。

​

以往每年的女王聖誕節演講，她都會提到今年一年的皇室亮點，但是對於今年，真的有點太難為女王了。一位皇室專家表示，今年的聖誕節演講可能正處於起草階段，但是對於女王來說，可沒那麼簡單。據悉，這是1992年黛安娜和查爾斯的鬧劇之後，女王最艱難的一年。
