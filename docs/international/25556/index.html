為拯救一個大陸，3人冒死靠近核輻射源頭，最終竟沒被感染

2019-12-23 1,073條評論 3,853次閱讀 1061人點贊

　　1986年4月26日凌晨1時23分，烏克蘭邊境城鎮普里皮亞季，小城居民早已進入夢鄉。突然，一聲巨響，驚醒了整座城鎮，也驚動了全世界。普里皮亞季鄰近的切爾諾貝利核電廠的第四號反應堆發生了爆炸。連續的爆炸引發了大火併散發出大量高能輻射物質到大氣層中，這些輻射塵涵蓋了大面積區域。這次災難所釋放出的輻射線劑量是二戰時期爆炸於廣島的原子彈的400倍以上。該事故被認為是歷史上最嚴重的核電事故。

　　

　　而在這起事故發生之前，普里皮亞季人口達5萬，每年還在不斷增加。然而，僅僅過了一個晚上的時間，普里皮亞季就有203人被送往了醫院，其中31人因過量輻射，甚至都來不及搶救就離開了這個世界，普里皮亞季因此被廢棄。不過，更為可怕的是，當時的死亡根本不算什麼。據統計，在切爾諾貝利核事故發生後20年來，陸續有9萬多人因輻射死亡，數十萬人患癌，大多數輻射致死者都來自普里皮亞季和清理廢墟人員。這些人的處境比當場死亡更可怕，他們身體內的細胞是一點點死亡的，承受著常人難以想像的痛苦，很多人因精神受到極大摧殘而自殺。

　　

　　80年代的人們，對核輻射還沒有太多的概念，所以一開始都以為，那只是一場普通的爆炸，把火撲滅了，人們仍舊能繼續生活下去。但是核輻射的影響，竟然比人們想像中要大的多，甚至大到令人不可思議的地步。

　　

　　核輻射究竟有多可怕呢?不僅僅是毀滅了普里皮亞季這個小城市，它還影響到了烏克蘭全國，甚至蔓延至周圍的白俄羅斯、俄羅斯等國。後來蘇聯公開了核電站事件的真相，事故發生一周之後，歐洲各部分甚至日本都檢測出了輻射。很明顯，整個歐洲大陸都受到了此次核輻射的影響，超過5億的人籠罩在核輻射的陰霾之下，人們人心惶惶，不敢想像自己會因此遭受多麼壞的結果。不過，由於大部分西歐國家離切爾諾貝利核電站數千公里，因此雖然檢測到的核輻射超標，但並未造成大傷亡。

　　

　　而歐洲並未受到較大影響的原因，除了因為距離遠以外，還有另一個重要的原因。就是有三位英雄的冒死相救，若沒有他們三位，切爾諾貝利核電站將會持續釋放放射性物質，不但歐洲大陸核輻射越來越高，連歐洲水源都將被污染，那造成的後果將不可想像。

　　

　　當時，4號反應堆爆炸後，數萬噸冷卻水漏到反應堆底部。而堆芯由於高溫正在慢慢的融化地板，用不了多久便會掉到底部，把這些含有高輻射的冷卻水，加熱到蒸汽狀態。當蒸汽壓力達到一定的極限後，便會把整個反應堆外殼炸開，核材料將飄到整個歐洲。那時候的切爾諾貝利就是一個即將爆炸的超級高壓鍋，一旦爆炸，會產生更加嚴重的後果。為防止二次爆炸，必須得有人將地下室水抽出來。

　　

　　這時，這三位蘇聯英雄站了出來，他們分別是鮑里斯·巴拉諾夫、瓦列里·別斯帕羅夫、阿列克謝·阿納年科。這三個人完成這次任務有得天獨厚的條件，因為他們是核電站的工程師，非常熟悉核電站的情況，由他們潛入水中把冷卻水放掉，一定比其他人完成難度小很多。不過，雖然他們穿著防護服，但地下室靠近輻射源頭，可見其輻射會有多強烈。可以說，這是一次有來無回的任務。

　　

　　但他們三個依然沒有猶豫，為了國家，為了整個地球，他們必須得這樣做。儘管後果可能已經預料，但他們依舊選擇自願參加這次行動。幸運的是，憑著對反應堆的熟悉，三名工程師及時找到並打開了閥門。水位開始下降，地下室的水被順利排空，他們及時的阻止了更為嚴重的爆炸。更令人驚喜的是，他們竟然平安的出來了，並且核輻射並未影響到他們。

　　

　　之後，三人身體一直都很健康，除了巴拉諾夫2005年因心臟病去世外，別斯帕羅夫和阿納年科二人至今仍然健康活著，身體沒什麼明顯變化，這實在是令人無比驚喜的結果。不過，為什麼離核輻射很遠的人，都可能死亡。而這三個人離的那麼近，反而一點事都沒有呢?這令人匪夷所思的事情，科學家們至今也沒有給出合理的解釋。不過，這是否意味著核輻射或許並沒有我們想像中的那麼恐怖?
