印尼女學霸56歲才出嫁：男方小學畢業，成為當地佳話

2019-12-14 458條評論 1,509次閱讀 167人點贊

來源：海外網

圖源：印尼當地媒體「Tribunnews」網站。

海外網12月13日電在當今社會，不少事業成功的男女由於種種原因都選擇了晚婚。在印度尼西亞也有這樣一位優秀的女士，直到56歲才找到屬於自己的「真命天子」，嫁給了意中人。

綜合印尼當地媒體「Tribunnews」網站等媒體13日報道，這名女子名叫哈旺，來自印尼蘇拉威西省省會肯達里。在上世紀80年代，哈旺在完成中學學業後，還考上了師範類學校，是那個年代少有的女性「學霸」。

圖源：印尼當地媒體。

當時，不少男士都曾追求過哈旺，但她一直沒有找到滿意的人選。而在印尼，隨著經濟和社會的不斷發展，現在也有更多女性獲得了大學乃至更高的學歷。

一轉眼，哈旺也到了50多歲的年紀。直到去年，她終於遇到了一位名叫謝祖丁的男子。他雖然是小學畢業，但也為人老實。

於是，兩人在近日舉辦了隆重的婚禮，一時間還成為了城中佳話。不少網友也紛紛表示祝福，有網友說：「如果你一直沒找到真愛，別擔心，可能只是時間的問題。」（海外網 吳倩）
