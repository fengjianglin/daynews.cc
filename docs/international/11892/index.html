布拉德·皮特今年聖誕好孤單：六個孩子來三個，比去年又少一個

2019-12-12 458條評論 3,039次閱讀 1501人點贊

據英國媒體12月11日報道，布拉德·皮特將和他的孩子中的三個——13歲的夏伊洛、11歲的雙胞胎諾克斯和薇薇安一起度過平安夜。

據報道，這將是他自2016年與安吉麗娜·朱莉分手後第一個家裡沒有監視器的聖誕節。

不過據知情人士透露，14歲的女兒扎哈拉不會參加。這位消息人士說，皮特「今年不指望她能來」。

▲安吉麗娜·朱莉和女兒扎哈拉、兒子馬多克斯

布拉德和夏伊洛、諾克斯、薇薇安和扎哈拉一起度過了去年的聖誕節。

據悉，18歲的馬多克斯和16歲的帕克斯預計也不會參加。

2016年，馬多克斯和父親在飛往洛杉磯的私人飛機上發生爭執，之後兩人關係疏遠。儘管布拉德沒有做錯任何事，但這起事件導致了兩人關係趨向緊張。據報道，這也是朱莉在與皮特結婚兩年、共同生活了11年後，提出離婚的原因。

馬多克斯目前在韓國上大學。 據稱，儘管朱莉一直在努力修補兩人的關係，試圖彌合兩人之間的裂痕，但這位少年卻始終不願接受。

▲2014年皮特一家的照片

布拉德認為這次裂痕是一個「巨大的損失」。他與16歲的另一個兒子帕克斯的關係也很緊張。一位消息人士在10月份透露，皮特希望帕克斯有興趣與他和解，因為他的哥哥馬多克斯在國外上學。

皮特和朱莉在2004年拍攝《史密斯夫婦》時相識，共有6個孩子。

他們在一起10年後於2014年8月結婚，但安吉麗娜於2016年9月提出離婚。

▲布拉德·皮特

法官在他們最終離婚時，給予他們合法的單身身份。這對曾經的情侶已經就孩子的監護權達成了協議，目前正在整理財產和財務，特別是希望就誰將擁有他們曾經的共同財產——法國釀酒廠Mirava達成一致——那裡也是他們的婚禮舉辦地。

（陳賽）
