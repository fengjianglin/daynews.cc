航母即將服役，就是買不到艦載機，土耳其的大國航母夢遙遙無期

天天要聞
3 天 ago

在埃爾多安的帶領下，土耳其也在做一個大國夢，作為這個大國夢的重要一部分，土耳其至少要能控制地中海的相當一部分，這就需要有一支強大的海上力量，而在地中海，土耳其海軍的實力明顯不夠，作為擴充土耳其海軍的重要部分，在西班牙方面的支持下，土耳其海軍開始建造「安納托利亞」號兩棲攻擊艦，在設計上大幅借鑒了西班牙海軍的「胡安·卡洛斯一世」號戰略投送艦。

在建造中，土耳其海軍對其進行了修改，包括滑躍甲板和耐燒蝕甲板等，按照土耳其方面的預想，該艦在2020年服役後，將裝備美製F35B垂直短距起降戰鬥機，讓土耳其海軍擁有一艘戰力可觀的輕型航母，而不僅僅是一艘裝備直升機的兩棲攻擊艦，屆時，土耳其海軍將大幅強化其在地中海的影響力，老對頭希臘海軍自不必說，甚至可以憑藉F35B的優勢，媲美法國海軍和義大利海軍，甚至是俄羅斯海軍。

不過，隨著引進俄制S400防空系統導致美國停止向土耳其出售F35戰鬥機，即將於2020年服役的「安納托里斯」號，將被迫繼續扮演兩棲攻擊艦的角色，12月13日，美國媒體報道了這艘即將服役的土耳其海軍最大軍艦，「安納托里斯」號最大排水量達28000噸，已經超過了一戰期間的「嚴厲」號戰列巡洋艦（原德國海軍「戈本」號）。美國媒體稱，「安納托里斯」號將會是一艘沒有固定翼戰機的航母，只能搭載直升機作戰。

按照最初的設計，「安納托里斯」號上有八個直升機起降點，其中四個能夠操作CH47重型直升機，一個能夠操作V22「魚鷹」傾轉旋翼機，預計土耳其製造的T129武裝直升機也將部署到「安納托里斯」號上，不過，鑒於目前土耳其與美國和其他北約盟友之間的關係，恐怕土耳其海軍很難為「安納托里斯」號配齊足夠的新型直升機，土耳其製造的首架T70通用直升機剛首飛（美製S70的仿製版本），但距離量產還有很長一段時間。與美國關係疏遠，嚴重影響了土耳其軍隊的整體戰鬥力，雖然近期土耳其方面由於經濟等緣故，不斷向美國示好，希望修補美土關係，但目前還沒有看到美國方面的積極反應，土耳其海軍現在頂多只能算擁有一艘直升機航母，距離真正的航母還差的遠。
