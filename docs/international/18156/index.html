俄官員：中俄首座跨境鐵路橋明年完工，將促進兩國貿易

2019-12-17 189條評論 1,294次閱讀 195人點贊

據今日俄羅斯（RT）17日報道，俄羅斯總統遠東聯邦地區特使尤里·特魯特涅夫表示，期待已久的中俄鐵路橋將於明年年底完工。

這一橋樑最初計劃在2020年上半年完工，但由於遭遇困難，不得不重新進行設計。

「我們將不得不開始一個新的工程項目，因為我們（俄羅斯）一側涉及的河流，以及已經安裝的橋樑柱子，比設計的要更深、更高。客觀地說，這個項目需要變更。」

他補充說，計劃在不吸引額外聯邦預算資金的情況下，對該項目進行修改。

這座連接俄羅斯遠東地區和中國黑龍江省的橋樑，長達2209米，預計將把雙邊貿易總額提升到新高度。中鐵大橋局集團總工程師李華超（音譯）表示，該項目旨在開發一條連接中國東北鐵路網和俄羅斯西伯利亞鐵路網的國際走廊。

阿穆爾河大橋的高速公路部分已於上月竣工。當這條路線開始運營時，貨運量預計將增長8倍，達到400萬噸。

俄羅斯阿穆爾州州長瓦西里·奧爾洛夫表示：「這座橋對我們具有重要的戰略意義。給我們地區來帶來希望。」

中俄兩國經過28年的談判，於2013年就這個雄心勃勃的3億美元建設項目達成協議。建築工程於2016年正式開工。俄羅斯計劃通過這一通道向中國出口鐵礦石、煤炭、礦物肥料、木材和其他商品。
