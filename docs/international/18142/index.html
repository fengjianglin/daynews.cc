「國防工業」蘭德公司分析俄羅斯國防經費及國防工業

2019-12-17 1,059條評論 3,564次閱讀 1312人點贊

———- 獲取更多信息，請關注我們———-

近期蘭德公司發布了《俄羅斯武裝力量趨勢——預算及能力概況》報告，其中對俄羅斯過去二十年的國防經費預算情況及未來趨勢、國防工業現狀及不足等進行了分析。

圖1 報告封面

一、俄羅斯的國防經費預算情況及趨勢

蘭德公司通過俄羅斯官方提交給聯合國的軍事支出數據及俄羅斯財政部公布的預算數據等開源渠道，對俄羅斯過去二十年國防經費情況進行了分析，並根據這一分析預測俄羅斯未來幾年可能的軍事花銷情況。

下圖為俄羅斯1996-2016年間的國防經費情況。在1998年的經濟危機後，俄羅斯國防預算開始實際增長（以扣除通貨膨脹後的盧布計價），儘管存在一些起伏；2005~2009年及2011~2015年間預算增長尤為顯著。其中，2011年到2015年，俄羅斯國防經費預算每年平均增長10.1%，但2016年受國際油價下降及西方制裁等影響，俄羅斯國防預算開始下降，2016年下降6.3%，2017年更是下降了14.8%。

圖2 俄羅斯1996-2016年間國防經費情況

俄羅斯可能在未來幾年削減國防支出，增大民生項目投入。2017年，俄羅斯經濟總量僅為歐盟的11.6%，美國的10.4%，歐盟和美國總量之和的5.5%。從經濟上來看，俄羅斯的影響力仍然僅限於歐洲，還不具備全球影響力。俄羅斯必須找到促進經濟的方法以防止與歐盟的差距進一步擴大。此外，國防預算下降意味著俄羅斯軍隊將面臨困難，不得不在重點資源投入領域作出艱難選擇。蘭德公司推測，俄羅斯國防預算將在2020年前持續下降，預計2020年預算將低於過去10年任何一年的水平，此後到2025將小幅回升。

圖3 俄羅斯到2025年的國防經費推測

下圖為俄羅斯2000~2016年國防經費（以2014年不變價格盧布計價）構成情況，包括人員、運行及維護、採購、研發。其中，人員費用在這17年間佔比最大，平均達46%；近四分之一的費用花在運行及維護上，另外還有四分之一用於採購；軍事研發費用平均佔比5%。

圖4 俄羅斯2000-2016年間國防經費構成情況

下圖為俄羅斯2001~2016年間國防經費軍種分配情況，包括陸軍、海軍、空軍、其它作戰部隊、准軍事部隊及管理費用。不同年份的數據變化較大，但總的來看，陸軍獲得的經費佔比最大，平均為23%;其它作戰部隊平均獲得12.8%的經費，海軍平均為13.5%，空軍為10.9%；管理費用佔比11%，准軍事部隊費用佔比6.9%。

圖5 俄羅斯2001-2016年間國防經費軍種分配情況

二、俄羅斯的國防工業現狀及不足

俄羅斯國防工業是世界上規模最大的國防工業之一。蘭德公司估計俄羅斯整個國防工業銷售額在2013年達到了252億美元的規模。與大多數國家類似，俄羅斯國防工業產值在其整個工業體系中的佔比不是很大，2013年為3%，2009到2013年間為2.8%。過去數年裡俄羅斯本國採購的武器增長迅速，而武器出口也在國防工業中持續扮演主要角色。1998~2013年，俄羅斯武器出口額均超過了本國採購額，僅有2014年本國採購超過了武器出口。部分年份武器出口額甚至超過本國採購額的兩倍。俄羅斯連續多年位列世界第二大武器出口國，僅次於美國。印度和中國是其主要出口對象，加起來佔比超過50%。出口武器類型以飛機為主，佔比近50%，其次是導彈、艦船及裝甲車輛。

圖6 俄羅斯2000-2014年間國防工業銷售額

工業結構方面，俄羅斯國防工業由近1500家機構組成，包括從前蘇聯繼承下來的研究所、設計局、製造廠。儘管一些公司在近些年投資建設了新的組裝線，但這些工業設施大體上還是基於蘇聯時期的場址和建築。過去十年，俄羅斯政府致力於建設大型控股公司以鞏固國防工業。幾乎所有的飛機工業都整合到了聯合飛機公司（UAC）；大部分造船企業都整合進了聯合造船公司，而在此之前，近五分之二的企業為私有。俄羅斯的裝甲車輛工業可能是全世界最大的，擁有四家主要企業，2016年武器銷售額達16.8億美元；造船工業牢固且技術先進，可製造小型和大型水面艦以及潛艇，主要企業是聯合造船公司，2016年銷售額達40.3億美元；軍用航空工業穩固，是俄羅斯技術最為先進的工業之一，主要企業有聯合飛機公司、聯合發動機公司、俄羅斯直升機公司等6家，2016年總銷售額達136.6億美元；此外，太空防禦和導彈工業及核武器工業也十分強大。

儘管俄羅斯擁有僅次於美國的強大國防工業體系，但其弱點也很突出。蘇聯解體後由於缺乏投資，很多設備老舊，導致質量問題頻發；受困於產能過剩，難以支持新的研發投入；部分關鍵部件依賴進口，尤其是電子和光學器件等。

（藍海星）
