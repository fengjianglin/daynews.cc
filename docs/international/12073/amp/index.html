“歐洲最美公主”出席諾貝爾獎晚宴，高貴典雅如女神

天天要聞
2019-12-11

瑞典國王的小女兒瑪德琳公主是公認的「歐洲最美公主」，她在諾貝爾獎晚宴上留下那張戴著王冠的照片幾乎驚艷了全世界。

今年諾貝爾獎的頒獎晚宴，瑞典媒體很早就得了消息，瑪德琳公主將出席今年的頒獎晚會，這給諾貝爾獎的頒獎儀式增添了更多色彩。

瑪德琳公主在晚宴上

據《每日郵報》報道，37歲的瑪德琳公主穿著一件粉紅色的無肩帶晚禮服長裙，頭戴海藍寶石王冠，挽著2019年諾貝爾生理學或醫學獎得主威廉·凱林（William G. Kaelin）款款走下斯德哥爾摩音樂廳的台階，顯得高貴典雅又儀態萬方。

瑪德琳公主出席2019諾貝爾獎頒獎晚宴

每一年的諾貝爾獎頒獎晚會都是瑞典王室的年度大事，幾乎全員出席。瑪德琳公主歷年來在諾獎晚宴上的著裝都讓人驚艷。

2005年，瑪德琳公主穿著一襲緞面蕾絲緊身禮服。

2012年，瑪德琳公主連續缺席兩年後重返諾貝爾獎。她穿著淺色薄紗無肩帶禮服，戴了瑞典王室的紫水晶頭飾。

2012

2013年瑪德琳公主又缺席了，因為她嫁給了美國銀行家克里斯托弗·奧尼爾，搬到了美國定居。

但是隨後的2014年，瑪德琳公主參加了頒獎晚宴。她穿著一件用絲綢、歐根紗和義大利薄紗製成的Fadi El Khoury禮服，長裙上有手工刺繡，同時鑲嵌了染成灰褐色和酒紅色的玻璃珠。

2014

2015年的頒獎典禮和宴會上，瑪德琳身穿令人驚艷的灰色Fadi El Khoury長袖晚禮服，頭戴海藍寶石頭冠，用鑽石花卉耳環搭配。

2015

2016年瑪德琳公主的衣服過於繁複厚重，未免美中不足。

2016

公主自從嫁給美國富商之後就定居美國，在短短的五年時間裡連生了三個孩子，生活的重心以撫育這三個孩子為主。並且為了讓孩子能過上正常的生活，瑪德琳公主和她的哥哥都放棄了孩子的王室尊號和職責，未來也不會領取王室津貼。

2017年，瑪德琳公主正懷著第三個孩子，所以她是挺著大肚子、帶著老公奧尼爾一起出席的諾獎晚宴。

2017

2018年瑪德琳公主生孩子了，諾獎晚宴自然也沒有出席。所以今年傳出她要出席晚宴的消息，令瑞典媒體振奮了好一陣子。

瑪德琳公主動圖

12月10日是瑞典科學家諾貝爾的逝世紀念日。自1901年起，除因戰亂中斷外，每年的這一天，諾貝爾獎組委會都會在瑞典首都斯德哥爾摩舉行隆重的頒獎儀式，為當年諾貝爾獎得主們頒獎。

諾獎得主們會在音樂廳接受由瑞典國王親自頒發的證書、獎章和獎金支票，然後再到市政廳參加晚宴。

2019諾貝爾獎晚宴

瑞典王室會盛裝出席晚宴。所以，除了備受矚目的諾獎得主，瑞典王室的珠寶首飾、衣著造型等時尚也是報紙媒體津津樂道的話題。
