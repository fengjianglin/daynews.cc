美國警告：俄羅斯管線危害歐洲安全，莫斯科的新朋友將令人失望

2019-12-18 1,290條評論 4,450次閱讀 552人點贊

據歐洲媒體12月11日報道，在歐洲一場新冷戰已宣告爆發，全球大部分大國都捲入其中，然而其所用的武器卻看不見摸不著，原來都是天然氣。北流天然氣管道二線，簡稱”北流二線”，是這場冷戰的核心。這是一條通過波羅的海直接連接俄羅斯和德國的天然氣管道，全長1230公里，與已在去年開始全面運營的北流一線平行鋪設，估計二線修建工程將耗資約95億歐元。輿論認為這項工程一旦完工，將令美國在新冷戰中節節敗退。

過去那場美蘇之間持續半個世紀之久的冷戰，源自1947年前英國首相丘吉爾發表的”鐵幕”演說，即一道”鐵幕從波羅的海到黑海徐徐降下”。如今，這場新冷戰同樣出現了一道”鐵幕”，即東歐諸多國家如波羅的海三國、波蘭和烏克蘭，在冷戰時代要麼是蘇聯的加盟共和國，要麼是華約成員國，如今卻成為了西方國家的”死忠粉”，遵從美國的意志執行堅決的反俄政策。這些國家同樣構成了一道從波羅的海到黑海的漫長”鐵幕”，只不過和冷戰時期的”鐵幕”是關閉通向西方的大門相反，如今這道”鐵幕”是為了關閉通向俄羅斯的大門，儘管俄羅斯仍保持著門戶開放政策。

對於俄羅斯來說，雖然通向亞太地區的道路越來越通暢，但歐洲特別是西歐國家，仍然是其經貿合作的最大夥伴，是俄羅斯出售各種資源、獲得各種高級工業製品和消費品的必然合作對象。因此，既然東歐這些國家不給與合作，那麼繞開它們就行了。”北流二線”宏大工程由俄羅斯天然氣工業股份公司領導進行，德國、奧地利、法國的能源集團以及英荷殼牌石油集團作為投資方也參與其中。倘若沒有政治干預，北流二線工程有望在明年初完工。

絲毫不出人意料，波蘭是最明確反對該項目的國家，完全依靠俄羅斯天然氣的三個波羅的海國家也表示不願意。而冷戰時代的對抗最前沿德國，如今又成為爭議的中心，其立場既微妙又矛盾。默克爾政府強調，北流二線是一個”純經濟”項目，可以確保德國的天然氣供應，也有助於減少碳排放，這個管道不是專為德國利益而建，而是為了整個歐洲考慮。

但是，美國政府已經譴責該管道將德國變成俄羅斯的”人質”。美國國務卿蓬佩奧表示，北流二線對歐洲能源安全和穩定構成”威脅”，歐洲可能淪為莫斯科的”受害者”。近期，德國已經承受不住美國的壓力，大幅度增加了貢獻給北約的防務預算。因此，莫斯科如果在這場新冷戰中指望柏林成為其堅定的新朋友，恐怕最後會完全失望的，儘管德國的經濟實力是歐洲第一，比俄羅斯的經濟規模要大好幾倍，卻很難保持對美獨立的立場。
