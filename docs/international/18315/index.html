語言、認同、經濟：西班牙為何留不住加泰羅尼亞的“心”？

2019-12-17 3,505條評論 4,757次閱讀 1820人點贊

西班牙地形圖

「你願意付出多少代價，為了追求獨立？」

一個位在地中海沿岸的自治區，舉辦一場獨立公投，中央政府不但不承認，還派出鎮暴警察，佔領投票所、沒收選票。投票當天下著雨，仍澆不熄民眾的投票意願，但手無寸鐵的民眾卻遭鎮暴警察粗暴對待，被打得頭破血流。

自加泰2017年獨立運動以來，獨立呼聲未曾中斷。

即便如此，這些人依然勇敢站出來，徒手對抗鎮暴警察，捍衛獨立公投。這不是電影情節，而是在2017年10月1日，西班牙自治區加泰羅尼亞真實上演的場景。

時間來到2019年10月14日，9名支持加泰羅尼亞獨立的領袖在不準保釋下囚禁近兩年後，遭重判入獄9至13年，另有3人罪成但只被判罰款，不用服刑。巴塞羅那隨即發聲明，表明「牢獄不是解決辦法」，呼籲政治領袖通過對話解決紛爭。隨後，引發區內大規模示威。

示威

事件起因就是2017年加泰羅尼亞一批政治及民權領袖在當地發起獨立公投引發，但被西班牙政府視為非法活動而大力打壓。

西班牙地形圖

壹

由宿敵巴塞皇馬：看懂加泰羅尼亞與西班牙恩怨情仇

從上帝的視角看巴塞羅那

加泰自治區是西班牙國內最富有兼最具生產力的地區，人口約750萬，佔全國總人口的16%，有自己的歷史、語言、旗幟、甚至警察部隊與政制，與西班牙截然不同。加泰歷史可追溯至1000年前，加泰自亞拉岡王國時期仍擁有相當的自治地位。直至1714年加泰在西班牙王位繼承戰爭中戰敗，巴塞隆那被攻陷，加泰從此成為西班牙的一部分，持續至今。加泰的獨立聲浪不斷高漲，特別是2008年金融海嘯後，更多加泰人想與西班牙分家。

加泰羅尼亞位置圖

加泰羅尼亞民眾不滿西班牙中央集權的概念，中央政府禁絕本土文化，「西班牙法律限制了加泰羅尼亞語在日常生活中的使用，毫不尊重加泰特有文化。」

巴塞羅那

除了文化和語言的差異，經濟亦是原因之一，據統計，加泰羅尼亞是全國經濟最繁榮的地區，貢獻全國近兩成的稅收，以及三成的生產總值，是西班牙收入的重大來源。加泰羅尼亞在2016年的統計中貢獻西班牙高達20.1％的GDP，高居西班牙各行政區之冠。繳交稅款與獲得的財政資助比例不成正比，加泰羅尼亞地區的基建投資、服務和福利方面發展不足，所交稅款反而補貼了馬德里及巴利阿里群島等地區。

這樣的經濟負擔，在歐債危機爆發後更讓加泰羅尼亞人感到沉重，成為追求獨立的一個重要因素。經濟基礎影響政治，政治從而影響文化，甚至體育。

巴塞

「如果加泰羅尼亞有一天獨立了，那麼巴塞必然在其中發揮了作用。」

《金融時報》一篇文章分析說。這句話也許說中了許多加泰人的心聲。

巴塞羅那足球俱樂部

巴塞球會成立於1899年，仔細看球會的會徽展示出傳統紅黃間的加泰旗幟，會歌也以加泰羅尼亞語填詞，管理層大多由擁有加泰羅尼亞血統的人士出任，不會像其他西甲球會般，允許外國注資，正如巴塞的前主席所言：「巴塞永遠都不會賣出去的。」球場內外，都充加泰地區都充斥著強烈的身份認同感，也是這個原因在西班牙獨裁者佛朗哥掌政時期，巴塞球會處處受到打壓。

部分加泰人基於政治、文化及經濟原因，支持脫離西班牙獨立建國。

公投前，在加泰羅尼亞自治區首府巴塞羅那，享譽國際的足球隊「巴塞羅那足球俱樂部」（簡稱巴塞）早於2017年5月發表聲明，公開支持加泰羅尼亞發起獨立公投，為「加泰是否脫離西班牙統治」這一議題進行自決。對加泰人來說，巴塞不單是一支足球隊，更是一個民族身份的象徵。

巴塞球迷在魯營球場揮動加泰旗幟，高喊政治口號。

公投當天，為免政治帶進球場釀成騷亂，巴塞在主場迎戰拉斯彭馬斯的西甲賽事最終改為閉門作賽，不開放給球迷觀戰。

公投當天，巴塞在主場迎戰拉斯彭馬斯的賽事最終改為閉門作賽，昔日座無虛席的魯營，竟然會呈現出這片光境。

加泰羅尼亞的球迷每年看西甲聯賽，進場時手中緊握象徵獨立的加泰旗幟，在比賽開始後，默默等待17分14秒這一刻的來臨。

每次巴塞對皇馬，球員都踢得份外落力，充滿火藥味。圖為巴塞後衛碧基（左）與皇馬隊長沙治奧拉莫斯（右）。

球賽來到17分14秒，紅黃相間的藍底白星旗幟高高飄揚，傳來一陣響亮的樂聲，在場人士高喊獨立口號，這裡不是公眾廣場，而是西甲巴塞羅那主場。選在這個時刻，是因為巴塞羅那城在1714年陷落於西班牙軍隊之手。頃刻之間，球場就變成一片示威地。

球迷經常在魯營球場內高舉示威標語，包括表達民主訴求。

加泰羅尼亞想獨立，除了經濟因素，還有歷史上兩段婚姻的造化弄人。加泰羅尼亞變成西班牙國土，其實是場美麗的錯誤。

貳

一場相差23歲的婚姻，加泰羅尼亞成為亞拉岡統治者

紅圈標註即為羅馬帝國統治下的加泰羅尼亞

公元711年，信仰伊斯蘭教的摩爾人，從北非跨越直布羅陀海峽進攻今日西班牙所在的伊比利半島。位在半島東北部的加泰羅尼亞居民，聯合北方法蘭克人抵擋摩爾人入侵，此後法國給予當地相當程度的自主權，這便是加泰羅尼亞作為獨立政治實體的濫觴。

法國國王在加泰羅尼亞分封許多伯爵，其中以靠海的巴塞羅那伯爵勢力最大，成為加泰羅尼亞實質上的領袖。

11-15世紀的西班牙

直到1137年，一場世紀婚禮為加泰羅尼亞帶來改變。當時的巴塞羅那伯爵拉蒙‧貝倫格爾四世在政治考量下，以24歲之齡，和鄰近的亞拉岡王國年僅1歲的女王佩特羅妮拉訂婚。巴塞羅那伯爵成為亞拉岡女婿，加泰羅尼亞也隨之成為亞拉岡的「親王國」。

12-13世紀西班牙和葡萄牙

佩特羅妮拉女王在1164年退位，由她和拉蒙‧貝倫格爾四世的兒子阿方索二世繼承王位，此後亞拉岡國王改由巴塞羅那伯爵一脈繼承。

十四世紀的伊比利半島，黃色部分即為卡斯提爾王國，右半部則為亞拉岡

叄

亞拉岡的第二段婚姻，加泰羅尼亞終成西班牙國土

亞拉岡與卡斯提爾

時間快轉到15世紀，伊比利半島上有兩個主要王國，分別是亞拉岡與卡斯提爾。1469年，亞拉岡王子費爾南多二世與卡斯提爾公主伊莎貝拉一世達成政治聯姻，雙方仍保有「祖國」的王位繼承權。

關係圖

這段婚姻為兩件事情奠定基礎：西班牙王國統一、加泰羅尼亞成為西班牙國土。

短短10年內，他們兩人分別當上「祖國」國王，等於一對夫婦、一個王朝同時治理兩國，因此費爾南多二世繼承亞拉岡王位的1479年，被多數歷史學者認定為「西班牙王國」的濫觴。

1476年伊比利亞半島

西班牙王國建立後，包含加泰羅尼亞在內的地區，雖仍有相當自治權，但在文化上卻是以相對強勢的卡斯提爾為中心。

伊莎貝拉一世的女兒胡安娜，在其死後繼位為卡斯提爾女王，且1516年費爾南多二世過世後，再兼領亞拉岡女王，自此兩個國家的王座同屬一人，未合併卻已達成名義上的統一。加泰羅尼亞的巴塞羅那伯爵血脈，從亞拉岡變成整個西班牙的國王。

西班牙國徽包含了多個組成西班牙的國家地區

由於胡安娜嫁給神聖羅馬帝國哈布斯堡王室的「美男子」菲利浦，他們的兒子查理五世，同時繼承神聖羅馬帝國和西班牙王位，西班牙轉而被哈布斯堡王室統治。

加泰羅尼亞曾短暫併入過阿拉貢王國

肆

西班牙王位繼承戰，種下加泰羅尼亞心結

哈布斯堡王室統治西班牙將近200年，1700年逝世的國王卡洛斯二世並未留下子嗣，在遺詔安排他姊姊和法王路易十四的孫子腓力五世繼位，但同屬哈布斯堡王室的奧地利認為，不能把王位傳給法國的波旁王室，旋即引髮長達14年的「西班牙王位繼承戰」。

加泰羅尼亞曾獲得法國政府幫助

加泰羅尼亞在戰爭初期支持法國波旁王室，後來奧地利哈布斯堡王室開出許多優渥條件，轉而對抗法國。

戰爭後期，列強紛紛簽訂合約停戰，同意腓力五世接掌西班牙，但他必須放棄法國王位繼承權。

加泰羅尼亞位置圖

這個看似圓滿的結局，加泰羅尼亞竟然被蒙在鼓裡，孤軍對抗法國。首府巴塞羅那在1714年9月11日於西班牙王位繼承戰爭中被法軍攻陷，自此併入西班牙領土，被西班牙菲利浦五世剝奪所有財政、行政、經濟自治權。加泰羅尼亞視之為恥辱，成為今天加泰羅尼亞國慶「淪陷日」的由來。

因戰事後期的頑強抵抗，腓力五世拿下加泰羅尼亞後便予以打壓，廢除上百年的自治特權，並禁止當地說加泰羅尼亞語，加泰羅尼亞對西班牙的心結由此誕生。

關係圖

伍

從佛朗哥獨裁到民主化，加泰羅尼亞的恩怨情仇

19世紀末，加泰羅尼亞成為了全國工業中心，至20世紀初加泰取得了不同程度的自治權力，並受到注重地方語言和民俗的「浪漫主義」影響，助長獨立聲浪。加泰羅尼亞憑著經濟實力逐步爭取到自治權，1933年重新成立自治政府。

加泰羅尼亞與西班牙比較

三年後，西班牙政府與保皇派的佛朗哥將軍爆發一場死傷慘重的內戰，當時西班牙人口約2300萬，內戰造成50萬人死亡。最終，佛朗哥在納粹德國的協助下贏得勝利，但沒有恢復王室，反而是自己擔任「攝政王」，西班牙進入長達37年的軍事獨裁。

佛朗哥

佛朗哥軍政府反對區域民族意識，加泰羅尼亞的語言、文字被禁用，自治權也被剝奪，在佛朗哥執政期間再次遭遇沉重打擊。

1975年佛朗哥去世後七天，胡安卡洛斯一世登上王位，西班牙波旁王室失勢多年後重回政治舞台。胡安卡洛斯一世大力推動西班牙民主化，包含加泰羅尼亞在內的許多地區恢復自治、重新開放「母語」。

加泰羅尼亞

陸

歐債之後：加泰羅尼亞再度踏上獨立之路

2008年金融海嘯席捲全球，西班牙政府要求加泰羅尼亞承擔更多經濟壓力，補助款從16％砍到9.5％，還要倒貼財政困難的中央政府140億歐元。西班牙經濟的「扯後腿」，再度點燃加泰羅尼亞的獨立之火。

集會

加泰羅尼亞議會是當地的權力核心，只要能在135席中取得過半，就能擔任自治區主席。西班牙民主化以來，歷屆加泰羅尼亞議會選舉的最大黨，都是主張獨立的加泰羅尼亞民主匯合黨。

該黨領導人、時任自治區主席的馬斯，在2014年不顧西班牙政府和憲法法院的反對，仗著在議會多數，舉辦沒有法律效力的獨立公投。之所以沒有法律效力，是因《西班牙憲法》不允許任何地區透過公投獨立。

2014年11月9日，加泰羅尼亞舉辦史上首次獨立公投，選民穿著代表加泰羅尼亞旗幟的服裝，投下神聖的一票。

這場公投有兩道題目，分別是：

1.你是否希望加泰羅尼亞成為一個國家？

2.如果是的話，你是否希望這個國家獨立？

兩道題目都同意的比例佔80.8％，都反對的為4.5％。但更值得關注的是投票率僅有41.6％，原因是反對黨杯葛，加上公投沒有合法性，無法做出實際改變，選民顯得意興闌珊，公投象徵意義大於實質意義。

馬斯在2015年的議會改選，和其他獨立派政黨再度取得議會過半。這次更進一步在議會通過一份為期18個月的獨立方針，在2017年10月1日舉辦第二次獨立公投，只要通過就會在48小時內宣布加泰羅尼亞獨立；若遭否決就解散議會進行大選。

10月14日最高法院宣判結果後，加泰爆發大規模示威，與防暴警察衝突。

2017年的公投題目非常簡明扼要：

你希望加泰羅尼亞以共和國的形式成為獨立國家嗎？

這次公投並未設立投票率門檻，只要開票結果是多數人贊成獨立，加泰羅尼亞自治政府就可以啟動獨立程序。不過，西班牙總理拉荷義與憲法法院都不願承認公投的合法性。

大批民眾走上巴塞羅那的公路

公投前夕，西班牙政府派出鎮暴警察進駐加泰羅尼亞，還宣稱「不會有人走進投票所」。加泰羅尼亞選民則是直接睡在投票所，為的是確保不會被警察佔據而無法投票。

2017年10月1日公投當日，加泰羅尼亞傳出許多衝突事件，鎮暴警察推擠、摔倒民眾，造成超過800人受傷，阻止民眾參與公投。由於鎮暴警察的阻礙，2300多個投票所約有1300個被封閉，不少選民因此無法投票。最終開票結果，贊成獨立的選民高達92％，投票率則為42.3％。

象徵加泰羅尼亞共和國的星條旗 (Estelada)，由當地獨派廣泛使用。 圖為獨派居民將其懸掛在自家露台。

不過，西班牙也有「殺手鐧」，西班牙威脅啟動《西班牙憲法》第155條接管加泰隆尼亞。《西班牙憲法》第155條規定中央政府如何暫停全國17個自治區的自治權。條文載明：「若自治區未能盡到《憲法》或其他法律要求的義務，或行事方式嚴重損害西班牙整體利益，政府可能採取必要措施，強迫（自治區）必須盡到應盡義務，否則將會維護上述整體利益。」

但是條文還規定，政府須先通知自治區政府主席將暫停自治權，若未接獲任何回復，還須獲得西班牙參議院絕對多數同意才能採取上述措施。

加泰羅尼亞

柒

語言、認同、經濟：西班牙為何留不住加泰羅尼亞的「心」？

待在西班牙數百年之久的加泰羅尼亞，為何近來一直想鬧獨立？除了歷史恩怨、浪漫主義催生民族意識、歐債風暴的影響之外，還與當地獨特的語言有關。

風景秀麗的加泰羅尼亞

經歷佛朗哥時期的語言破壞，加泰羅尼亞自治政府在西班牙民主化後，立法保障加泰羅尼亞語地位，不僅公務員需要經加語、西語的雙語認證，還成立加語媒體，學校除西班牙語課程，也一律使用加語。

西班牙的加語的人口集中在加泰羅尼亞，約86％的居民聽得懂加語、亦有76％會說。保障「母語」之舉，重新塑造加泰羅尼亞居民的自我認同，因為「我講的話跟西班牙人不一樣。」

這項轉變不僅限以加語為「母語」的人，只要你住在加泰羅尼亞、會說加語，透過語言對當地文化產生認同，不必有血緣認證也能成為加泰羅尼亞人。這樣的民族意識與種族無關，而是讓社會產生全民意志的「公民民族主義」。

加泰羅尼亞徽章

這點也是加泰羅尼亞與其他獨立運動最大的差異之處，因為有屬於當地、全球獨一無二的語言，就會產生很強的歸屬感與認同感。例如英國的蘇格蘭多數人還是說英語，加拿大的魁北克雖然是用法文，但世界上講法語的國家並不少。

今日西班牙地圖，淡黃色部分為西班牙國土，其中紅色部分即為加泰羅尼亞。

這樣的區域認同，在金融海嘯時被激發出來。即便歐盟等國際組織以「不合乎西班牙憲法」為由反對獨立，但自衝突發生以來，西班牙政府只想著鎮壓、制止，沒有正視與解決加泰羅尼亞的問題，總理拉荷義甚至在公投結束後說「什麼事都沒發生。」

加泰羅尼亞受不了西班牙的索求無度，以自身經濟實力做後盾，選擇走上這條艱辛、不被祝福的獨立之路。
