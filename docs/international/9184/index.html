加拿大車輛盜竊猖獗這些車最易被偷！手把手教你防盜賊

2019-12-06 1,055條評論 2,560次閱讀 57人點贊

近年，多倫多地區車輛盜竊頻發

尤其是今年

盜竊現象尤其嚴重

盜賊手段不同花樣百出

時不時就能看到車輛被盜的新聞

更誇張的是

安省Petawawa的地區

自今年4月以來

共有約100多輛車被盜

9月份更有一夥盜賊從該地區的

豐田車行一次性偷走了7輛車

整個過程只花了60秒

連車鑰匙都不用!

多倫多更是爆出過年僅19歲小毛孩

以嫻熟的盜車技術

在一個月內偷走36輛車的新聞

被捕後多倫多警方對其

做出了高達71項的指控

然而該青年只是盜車集團的一份子而已

背後的集團仍然活躍

讓人細思極恐

。。。

到底是特定車輛容易被偷

還是竊賊無所顧忌有啥偷啥?

加拿大保險局(IBC)

表示，汽車盜竊給加拿大人帶來的

損失近10億加幣

隨著汽車技術的不斷發展

盜賊的偷竊技巧也在與時俱進

現在「老練」的盜賊擅長

利用高科技產品

跳過汽車安全系統

能輕易將車開走還不被發現

IBC的調查服務總監Bryan Gast表示

越來越多汽車採用

「無鑰匙進入系統」

而盜賊正是利用某些使用這種系統

但卻沒有點火防盜裝置車輛的漏洞

有目標性的發起「攻擊」

(IBC)近日公布了2019年

加拿大最常被盜車輛年度清單

如果擁有以下車輛的車主可得小心了!

福特的350SD和250SD

凌志的RX350 / RX350L / RX450h /

RX450hL 4DR AWD 2018

本田思域Si 2DR Coupe 1998

而盜竊團伙一般在偷車後

會迅速將這些被盜車輛出手，拆賣

或者轉售給非洲或者中東國家

之前就有華人爆料自己車被偷

警方在魁北克邊境追蹤到失竊車輛

差一點車就要漂洋過海去非洲了

那麼該如何防範車被偷呢?：

1.切勿在無人看管的情況下啟動車輛

2.停車時保證車輛門窗關閉並鎖車

3.將車輛停放在光線充足

且安全的車庫或者區域中

4.安裝行車記錄儀

5.安裝方向盤鎖或者制動踏板鎖

6.切勿將值錢的物品留在車內

希望警方能加大對盜車團伙的打擊力度

不然在這麼寒冷的冬天

少了車可怎麼出門哪?
