中國大熊貓遭外國人瘋狂獵殺，美國總統羅斯福的兒子是獵殺第一人

2019-12-11 147條評論 1,606次閱讀 324人點贊

1869年，法國某傳教士首次向世界介紹了大熊貓，在世界上掀起了一股「熊貓熱」。此後，外國探險家、動物學家等紛紛來到中國，試圖捕捉大熊貓。其中包括美國前總統西奧多·羅斯福的兩個兒子。

​1926年，羅斯福兄弟來到中國四川寶興縣，以打獵為名，暗中進行大熊貓盜獵活動。兄弟二人很快就成功捕獲了一隻大熊貓，並偷偷運往美國。不幸的是，這隻大熊貓在漫長的運輸途中染病死亡，羅斯福兄弟只好將它做成了標本。

​1928年，兄弟倆再次來到寶興縣。為了掩人耳目，他們宣稱是在捕獵獨角獸，還找來了當地老百姓做嚮導。但當地百姓發現了他們在捕獵大熊貓後，立即將兩人趕走。他倆又輾轉來到了涼山越西縣，終於發現了一隻成年雌性大熊貓。因這隻大熊貓的體形龐大、兇猛敏捷，羅斯福兄弟不得不放棄了活體捕捉的計劃，選擇射殺大熊貓：他們同時對那隻漸行漸遠的熊貓背影開了槍，兩槍都準確命中。熊貓應聲倒地，但很快又爬起來，跑進了茂密的叢林中……不得不說，是羅斯福兄弟首開了西方人獵殺中國大熊貓的先例。

​此後，西方探險家開始了對中國大熊貓的瘋狂獵殺。從1936年至1938年間，從中國運出的活體大熊貓共計12隻，流往西方博物館的大熊貓標本更是多達70餘具。
