日本貧民窟vs富人區？目擊最直觀的貧富差距

2019-12-11 894條評論 1,850次閱讀 448人點贊

貧富差距這個問題，不管是大城市還是小鄉村，發達國家還是發展中國家都客觀存在的問題。

日本曾被大家稱為亞洲貧富差距最小的國家，也被無數人嚮往，還號稱一億三千萬人口，有一億人是中產階級，儘管如此，仍然有著相對懸殊的差距，畢竟有人的地方就有階級，這是無法完全消除的。

一位up主就用自己的鏡頭，以自己的視角，向我們把大部分日本人眼中的富人區麻布十番，以及貧民窟壽町進行記錄。

首先來到富人區麻布十番。

街上沒有遮天蔽日的大樓，通透敞亮的感覺撲面而來，顯然街上歐美面孔的人們更多些，街上更是一片祥和安靜的氛圍。

駛過的國際學校的校車

發現一家很走心地過萬聖節的一家人。

街上歐美面孔的孩子，和朋友歡樂的玩耍。

大人們則是帶著孩子一起在足球場、網球場做運動，享受歡樂時光。

聚集在這裡的機構也不一般，各國大使館以及日本最大的圖書館——東京都立圖書館就在這裡。

離開這個明亮的地方，轉戰下一個日本橫濱的壽町，被稱為日本最大的貧民窟之一。出站後，可以看到這裡的建築都很高，遮天蔽日，陽光都很難照射進來。

環顧四周，發現年輕人很少，大多都是老年人，街上的垃圾多是啤酒瓶之類的。

這裡的孩子們都是自己騎著自行車，身邊並沒有家人的陪伴，甚至有些孩子還會對陌生人出口成臟。

環顧這裡的公寓就是政府提供的廉租房，但是統統給裝上了空調，因為房間沒有浴室，還非常貼心地設置了投幣式洗浴房，一起的還有投幣式洗衣機。

邊走邊看，發現街上有類似社區服務中心的地方，也是這裡的圖書館，每個月還可以免費看電影，附有診療室、浴場。

設立了供應貧窮的人免費吃飯的地方，一日三餐都免費，只不過每個人使用不能超過3個月，就得自己找工作養活自己，對於困境中的人們來說，這個地方可以說很貼心了。

*圖源來自Linlin的奇妙冒險

看完貧富兩街區的對比，一邊感嘆兩邊生活環境的差距，一邊為日本健全的福利制度點贊，可以真正的給需要幫助的人，切實地給到細節上的幫助。同時不忘給這裡的人生活的希望。

而在差距明顯的環境中成長的孩子，我也相信終有一天，她們能觸碰光明，迎來生命的綻放！
