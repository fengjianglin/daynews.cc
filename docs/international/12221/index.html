世界上竟然有這麼多人在學中文！網紅"孔子"到底有多火？

2019-12-10 407條評論 1,344次閱讀 332人點贊

自2004年6月15日，全球首家孔子學院在烏茲別克塔什干正式設立，15年來，孔子學院和孔子課堂正在世界各國遍地開花。

義大利羅馬國立住讀學校，是當地一所名校。2009年，該校率先在高中開設漢語必修課，成為義大利第一所將漢語作為第一外語納入必修課體系的國立理科高中。

2010年，羅馬國立住讀學校孔子課堂掛牌。

鄭彥柯

四年前，因為被學長一口流利的中文吸引住，該校學生鄭彥柯迷上了中文。

如今，這個17歲的義大利男孩不僅能說一口流利的中文，還獲得了2019年”漢語橋”世界中學生中文比賽冠軍。在接受記者採訪時，鄭彥柯時不時就能出口成章、引經據典。

“我一直很喜歡學習中文，享受學習的過程，就像孔子說的那句話”知之者不如好之者，好之者不如樂之者”，興趣是最好的老師。”

從2009年首次開設漢語必修課，到目前，義大利已開設了12所孔子學院和41個孔子課堂。

十多年來，累計學員近23萬人，文化活動參與人數逾133萬人次。

羅馬國立住讀學校孔子課堂負責人陳晨：

“義大利的漢語教育以前更多的是集中在大學、高等教育階段，在這些年，開始有了低齡化的發展，特別是在基礎教育階段，有越來越多的中學把漢語作為必修課，而且也將漢語作為高中畢業考試的項目，而且現在越來越低齡化，甚至低齡到可能3歲、4歲、5歲就想要開始學習漢語。”

在15年的發展歷程中，孔子學院已經成為世界認識中國的一個重要平台。

來自馬達加斯加塔那那利佛大學孔子學院的本土教師吳妮，曾在孔子學院中文班學習，後來來中國留學兩年，畢業回國後，她選擇重返孔子學院成為一名本土老師。吳妮說，”學好漢語有出路”，已成為馬達加斯加和眾多非洲國家年輕人的共識。

“學好漢語，掌握專業技能，將會給非洲年輕人更廣闊的發展空間，為他們的夢想插上騰飛的翅膀。非洲與中國雖遠隔萬里，但’一帶一路’倡議讓我們不再遙遠，漢語讓我們心靈相通。”

隨著中國日益增強的國力和日漸擴大的影響力，學習漢語逐漸成為一種國際流行趨勢。

目前，全世界已有3萬多所中小學開設了中文課程，4000多所大學設立了中文院系或課程，還有4.5萬所華文學校和培訓機構開展中文教育，全球學習中文的人數超過2500萬人。
