世界環球小姐出爐 是畫風跑偏還是我不會欣賞美？

天天要聞
2019-12-10

吳哥時報訊 今天世界環球小姐選舉結果出爐，南非佳麗奪得2019環球小姐。雖然這是個選美比賽，但在民族服裝比賽單元畫風似乎有些跑偏，融入民族特色後的服裝就變得有些清奇。

南非佳麗佐茲比尼·通齊成為2019環球小姐

小編覺得，這是最考驗環球小姐的個人實力的環節，畢竟一般人實在難以駕馭這些「奇裝異服」。

南非：國旗配色

首先我們來看看本次的冠軍，來自南非的佳麗，腳踩恨天高，裝飾用色鮮艷，十分具有南非蓬勃之氣。

柬埔寨：apsara

柬埔寨佳麗的服裝每個細節都源自高棉傳統文化藝術，意在希望柬埔寨婦女能走在前列，讓世界更加熟悉柬埔寨及其文化。雖然靈感來源是高棉傳統文化，但是真的沒有借鑒一點點孔雀開屏嗎？

大片大片的彩色的馬賽克靈感來自哥斯大黎加的彩繪，它是哥斯大黎加的國家象徵。

馬來西亞：馬來西亞食品

馬來西亞佳麗的民族服裝是整個比賽中最複雜的服裝之一。它重超過60磅，並配有一桌馬來西亞傳統食品。

薩爾瓦多：農作物

這「座「」色彩鮮艷的服裝結合了薩爾瓦多的民族農業，主要農作物包括咖啡、糖和玉米。為啥大家都喜歡在衣服上放點吃的？

孟加拉國：黃包車

孟加拉國佳麗穿著黃包車風格的服裝，配有標誌性的孟加拉國人力車的樣板兜帽。實用性第一名，又可以遮陽又可以擋雨。

烏克蘭：綠色

烏克蘭的一身綠的十分搶眼，意在保護環境，守護大地的綠色。只是這個綠色的帽子吧……

中國：舞蹈

這一局，小編覺得中國輸了，因為我們不夠奇特，只是美了。
