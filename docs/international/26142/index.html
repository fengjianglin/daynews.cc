鮑里斯在會議上無聊玩手機被點名批評回懟：請講點趣事！全場鬨笑

2019-12-21 431條評論 2,551次閱讀 178人點贊

近日，英國最重要的事情當然是大選，最終看起來不靠譜的鮑里斯率領保守黨大獲全勝。不過，他的日子也不會太好過，畢竟英國目前看起來也是危機四伏，除了要繼續解決脫歐問題，英國的「人民內部矛盾」也一直是個老大難問題。這不，據12月20日的《每日快報》報道，近日，鮑里斯在解決人民內部矛盾時慘遭蘇格蘭民族黨議員怒懟。

要說事件的起因，還得從大選說起。就在鮑里斯獲得大選勝利的同時，在蘇格蘭選區，蘇格蘭民族黨在議席中的席位也大幅增加，成為蘇格蘭選區的最大贏家。

結果，蘇格蘭政府首席大臣、蘇格蘭民族黨黨魁妮可拉·斯特金立即表示，要在聖誕節前向鮑里斯遞交信函，要求舉行第二次蘇格蘭獨立公投。

而鮑里斯對此已經明確表態，他認為2014年的蘇格蘭獨立公投是「一代人僅此一次」的機會，他拒絕考慮就這一問題再度進行公投。

而在近日英國議會下議院的一次會議中，不可避免地又討論到蘇格蘭公投問題。或許是早已對此問題感到厭煩，鮑里斯竟然自顧自地在台下玩起了手機。

當時，蘇格蘭民族黨下議院領袖伊安·布萊克福德正在台上就蘇格蘭公投慷慨陳詞，但他同時注意到鮑里斯的小動作，於是直接懟道：「我能對首相說，看著他在那裡玩手機而不是傾聽蘇格蘭民族黨的合理要求，實在不是什麼光彩的事嗎？」

早已憋了一肚子火的鮑里斯也不是軟柿子，直接反擊道：「那就請說點更有趣的事情吧。」

兩人的這一番唇槍舌戰引得下議院爆發出一陣鬨笑。

雖然只是下議院會議中的一個小插曲，但可以想見，作為英國首相，鮑里斯·約翰遜未來的路並不好走。不知道未來他是否能一邊玩著手機，一邊就把脫歐問題妥善解決掉。
