伊朗外長喊話歐洲國家：行使主權吧，不要屈服於美國欺凌

2019-12-06 494條評論 4,786次閱讀 2387人點贊

伊朗外長扎里夫（圖源：美聯社）

海外網12月5日電在伊核問題上，伊朗曾多次公開指責歐洲各國不履行《伊核協議》，並表示仍在等待其採取切實可行的措施落實協議內容。就在北約峰會剛結束一天後，伊朗外長扎里夫再次向歐洲各國喊話，讓其行使主權，不要再屈服於美國的欺凌。

據俄羅斯衛星通訊社報道，5日，扎里夫在社交平台提到了一封歐洲3國近期交給聯合國秘書長的信，扎里夫稱這封有關導彈研發的信就是一個「徹底的謊言」，它體現了歐洲國家在履行《伊核協議》上的無能。

扎里夫表示：「如果歐洲3國還想獲得一點全球可信度的話，就應該從行使主權開始，而不是屈服於美國的欺凌。」扎里夫說這番話之前，3日，伊朗副外長阿巴斯在同日本外相茂木敏會面時重申，伊朗並未尋求放棄《伊核協議》，但前提是要取消對伊朗的制裁。

美國政府去年5月退出伊核協議，恢復並追加對伊制裁，試圖以極限施壓方式迫使伊方重新談判。美方退出伊核協議的同時向歐洲盟友施壓，要求重新啟動對伊制裁。歐洲方面雖然表示繼續支持協議，但同時警告稱，他們的支持取決於伊朗對協議的全面承諾。

自今年5月以來，伊朗一直在減少履行其對於協議的承諾以應對美國制裁，同時警告歐洲締約方，除非採取更多措施，保護伊朗經濟免受美國懲罰，否則將繼續取消對本國核計劃的限制。但歐洲則是強烈要求伊朗配合國際原子能組織(IAEA)的監察活動。（海外網 魏雪巍）
