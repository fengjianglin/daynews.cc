7年2起惡性案件，印度150萬人掀起抗議：強姦案越來越多

2019-12-14 83條評論 1,028次閱讀 90人點贊

據媒體12月13日報道，12月12日，印度全國各地婦女發動大規模抗議活動，她們對11月一名女獸醫在印度海德拉巴被輪姦殺害並焚屍趕到極度恐慌，其中許多人對自己的安全表示擔憂。

在被安全問題困擾幾天後，一位年輕的24歲女性在網上請願，她是該網站上500個請願書其中之一。在過去的兩周中，有數百萬人在網站留言，他們要求政府培養正義敏銳的警察並開設求助熱線，以保證婦女的人身安全。

印度民眾抗議遊行

2012年，印度新德里一名23歲女學生，在公共汽車上遭到6名男子輪姦並被殘忍殺害。當時很多印度人走上街頭髮起抗議遊行，他們呼籲政府嚴懲犯罪分子。但是近年來，印度的強姦案發生率越來越高，請願者坎德拉昆塔告訴記者，強姦案件的頻繁發生警示我們必須做出改變。

她說：”每當媒體報道此類罪行時，人們就會呼籲對犯罪者處以死刑，但是到現在都沒有人解決婦女在街道上的安全問題”，她請求政府安裝更多路燈點亮海德拉巴，並確保行人的安全。

印度女性

2019年11月，印度一名女獸醫被幾名男子輪姦並焚屍，事件報道後，印度很多地方爆發了大規模抗議活動。有媒體報道稱印度是女性生活最糟糕的地方之一，2018年，印度更是在關於女性生活最危險的國家列表中名列前茅。

根據最新的政府數據顯示，印度2017年記錄了超過32500起強姦案，平均每天90起。但法院裁決的案件約有18300起，截止2017年底，有超過12萬起案件目前還懸而未決。

印度請願網站負責人哈桑說：”只需要看看網站主頁上的請願書標籤，就可以知道國家對此類案件的處理非常遲緩。在海德拉巴獸醫被強姦謀殺後，網站上新增了500份請願書，前五份請願書下面有超過150萬個留言”。

印度女性

新德里公交車輪姦案受害者的母親德維在網站上發起了一項請願書，她希望針對婦女的強姦案能夠迅速得到審判，她寫道：”我知道所有這些受害者的家人都經歷了什麼，我希望政府能夠對這些案件儘快處理，還受害者及其家庭一個公道”。請願書發表後短短几個小時，就獲得了8萬多個留言。

她還在請願書中說：”如果我們想擺脫婦女生活”不安全國家”的標籤，那麼我們必須站出來伸張正義”。喬普拉在她的請願書中寫道：”我們不應該對這些事情置之不理，我擔心下一個會是我！”。

印度一位作家表示，關於改善海德拉巴街道照明的請願書是一個很有效的解決方案。
