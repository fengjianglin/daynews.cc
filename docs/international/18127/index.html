西行漫記4：印度本地供應商考察，簡直就是一段魔鬼般的旅程

2019-12-18 1,296條評論 4,123次閱讀 2138人點贊

2010年10月4日，在國內還在舉國歡慶過大節的時候，我們取經四人組已經開始了新的一周工作。

到印度的第三周，異國的第一次供應商考察終於開始了。早上6點半就得起床，三哥的地頭，路況差，得考慮早點車子少一些出行順暢些。

7點45分到達GMTCI後，又等了一會，才等來了最後一位姍姍來遲的印度工程師，8點10分左右打車出發了。顛簸了一個多小時，9點10分我們來到了BOSCH印度的技術中心。

同樣的差不多要接受機場安檢的待遇，花了20多分鐘才進到了BOSCH的會議室，在0樓進入他們的辦公區的時候，也需要刷卡才能經過國內只有進地鐵站台才有的那種門，真是有五步一崗，十步一哨的感覺啊。現在進我們公司也要這樣操作了，信息保密方面印度或許比我們領先了很多。

在BOSCH技術中心會議室，聽完了他們對公司和產品做的介紹以後，就到11點了。首先在會議的過程中有茶水和小甜點隨意取用，這倒是在國內少有的。

因為BOSCH的研發和生產是分開的，到生產場地去做生產場地評審還得花2個小時的車程，路程只有區區的72公里，在國內一個小時肯定能到了。車子出城後，發現和國內一樣也是有收費站的，但收費多少沒看得懂，收費站上面的語言都是用的印地語。過了收費站，車子開上了疑似高速公路上。經過兩個多小時的顛簸，來到了BOSCH和曼胡墨爾合資的一個生產工廠。

到這裡是當地時間下午1點鐘了，進到會議室一陣寒暄後，就看見有人送了一堆食物進來，難道要在會議室吃午餐。果真如此在印度經歷了一次會議室午餐。這樣的做法在國內肯定會被認為對客人不尊重，但在印度我倒認為這是提高工作效率的一種表現。

大家花了半個小時左右就吃完了。收拾乾淨後，BOSCH馬上就開始了工廠介紹和產品展示。最後是工廠參觀和生產過程介紹。雖然濾清器不是我們管的零件，但通過這樣的參觀後我們也對該產品的生產工藝和製造過程有了相當的了解。參觀過程大概花了1個半小時左右。

因為還有下一站是考察實驗設備，而且試驗室又是在另外一個地方，好在是在回程的路上，但也需要坐1個小時的車才能抵達。因此相當於我們從下午4點開始踏上了歸程。一個小時以後來到了曼胡莫爾在班加羅爾市內的試驗中心。在這裡的參觀考察又花了一個小時，6點了，到了印度人的下班的時間，我們魔鬼般的歸程也開始了。

由於通用在印度虧損9年，每年賣的車不超過10萬輛，沒有一款車型的銷量是超過1萬的，供應商肯定對這樣的主機廠不重視，因此用車送你，門都沒有，只好自己叫計程車。

或許是班加羅爾計程車數量不夠，也可能是正趕上下班時間堵車吧，計程車遲遲不到，BOSCH的人只好讓我們上車，一起開到外面路邊去等出租。

在路邊整整等了一個小時，才等來了預訂的計程車。從BOSCH的試驗中心到我們住的賓館，預計要開兩個小時，現在的印度時間以及過7點了。如果準時到達吃完晚餐，國內肯定過了11點，看來今天又無法與老婆孩子聊Q了。

如果不經歷這天晚上的魔鬼歸程，不會對印度的基礎設施的落後有這麼深刻的感受，車子在路上感覺簡直是在爬。

班加羅爾是印度第五大城市，但市內的道路比柳州拉堡都不如，路窄車多，摩托車，突突、三輪車，公交車、計程車和私家車全部都擠在一條路上，而且路中間還沒有明顯的分界線。下班時間車子又多，行人過馬路也很隨機，所以前面一個小時基本沒走多少路。還好後面快了，道路也寬了，中間有隔離帶了。熬到9點10分左右，總算回到賓館了。吃晚飯到房間，果然快10點了。老婆孩子都睡了無法聊天了。

這還是在班加羅爾市內進行的，後續的德里、普內等其他城市的供應商考察，困難可想而知。

說了這麼多印度的差的方面，好的方面也是要說一下的，那就是環境，在國內很少能看見天空那麼純凈那麼藍。沒有得到充分發展就意味著環境沒有受到破壞，班加羅爾到處是大樹，綠地很多，空氣很好，沒有過多的雜訊和污染，這些倒是國內稍顯不如的。
