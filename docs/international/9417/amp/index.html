又一土豪超豪華車庫曝光，王思聰不及他一半，最貴一輛超四千萬

天天要聞
2019-12-04

此前，車行哥跟大家說過一個全球豪車最多的人汶萊蘇丹，他是汶萊的國家元首，其在購買豪車上的花費超過了一百多億元，可以說他的車庫就是一個豪車的展覽會。不過，像是這樣的私人豪華車庫的人，全球並不少，在我國就有好幾位，比如說王思聰，京城四少等等。但是大家都知道，這些人都是富二代，一出生就註定了可以擁有很多豪車。而接下來要說到的這位是完全靠著自己的努力的，讓自己的車庫變成了了一個超級豪華的車庫，他就是美國伊朗籍的億萬美元富翁曼尼霍斯賓。

曼尼霍斯賓在購買豪車花費了超過了4億多元，而他目前的財產超過了70億。他的車庫中有著不少限量版的頂級豪車以及頂級超跑。但是他並不是一位富二代，他出生的家庭非常的貧窮，甚至有時候吃不飽飯。而出來工作之後，雖然沒有穩定的工作，但是經常做零工賺錢，而後靠著一筆小積蓄，自己賣起了水果，再賺到錢之後又再去投資，慢慢的積累財富，從賣水果到加油站再到房地產，目前曼尼霍斯賓才剛40出頭，便已經超過了70億的財產。

因此，在成功之後的他，也可以有資本去實現小時候的夢想，買豪車開超跑。在曼尼霍斯賓的車庫收藏了十幾輛豪車，其中最便宜的便的車是一台全球限量50台的邁凱倫650S CAN-AM特別版，超過了33萬美元。而最豪華的車分別是的全球只限量三台的布加迪傳說系列的倫勃朗布加迪以及私人定製的帕加尼Huayra愛馬仕版，前面一款車超過了2000萬人民幣，而私人定製加尼Huayra愛馬仕版雖然不知道價格，但是估計也不會低於4000萬。

單單就這兩款車就已經超過了6000多萬的價格，在加上其他十幾輛豪車，包括了勞斯萊斯轎跑以及一些限量的頂級跑車，其數量雖然不是非常的多，但是其整個車庫的價格也超過了4億多人民幣。可以說，曼尼霍斯賓是一個從貧窮小子到成功人士的典範。因此，千萬被覺得自己現在剛畢業出來什麼都沒有，覺得出生不公平，很多時候，靠著自己的努力也能過上自己所夢想的生活。
