非洲有一個以廢墟命名的國家，遺址還挖出中國瓷器

2019-12-21 536條評論 1,851次閱讀 730人點贊

辛巴威最令人嘆為觀止的歷史古迹無疑就是大辛巴威城遺址了。”辛巴威”這個名字是一個英語化的非洲單詞，意思是”石屋”「石頭城」，它橫跨辛巴威東南部地區，面積近1800英畝。雖然這處遺址聽起來是以國家的名字命名的，但實際上卻恰恰相反。

據估計，大辛巴威城的建設持續跨越了300多年，但是，距今為止誰也不知道大辛巴威到底是由誰建造的。雖然歷經滄桑最終被遺棄，然而，許多建築至今仍屹立不倒。

大辛巴威城遺址是鐵器時代辛巴威王國的首都，它坐落在該國東南部的山丘上，被認為是撒哈拉以南非洲最重要和規模最大的石頭城遺迹。1986年聯合國教科文組織將此列為世界遺產，大辛巴威城遺留的塔樓和圍牆是由數以萬計的石頭建造而成，彼此完美地壘砌在一起。

考古發現，辛巴威城是在11世紀由肖納人建立的，肖納人是班圖人的一支。1000多年前，居住在辛巴威城的肖納人用黃金和象牙同來自印度洋海岸的阿拉伯和印度的客商換取瓷器、布和玻璃珠子。由於他們深諳商貿之道，於是變得越來越富有，大辛巴威城成為當時非洲大陸最繁華的首都，並且在14世紀達到頂峰。在這個時期，有超過15000人居住在這些令人印象深刻的石頭建築內。

然而，在15世紀的曙光來臨之時，這座繁華的都市由於太過繁華命運開始改變。由於人口的暴增，城市周圍樹木被砍伐殆盡，動物逃離，生存環境變得惡劣。到1450年時，大辛巴威被另一座城市卡米替代。1505年，當葡萄牙殖民者來到這一地區尋找傳說中的黃金城時，大辛巴威已經淪落為廢墟。

大辛巴威城遺址發現的皂石鳥雕

對於大辛巴威的出身，長期以來充滿了爭議。早期的歐洲探險家在偶然發現了這片廢墟後，許多關於它起源的不同理論就被不斷提及，也被不斷質疑。德國地理學家卡爾·莫赫曾假設，遺址與所羅門王和示巴女王有關聯。而英國考古學家西奧多·本特則聲稱發現了腓尼基人或阿拉伯商人建造的證據。帝國主義者鑽石大王塞西爾·羅德斯贊助了考古和挖掘工作，由於受到殖民主義信仰的影響，他斷言非洲土著人缺少文明，無法建造這樣一座建築先進的城市。

1905年考古和人類學家對遺址進行的第一次科學發掘，出土的班圖族文物，無可爭議地駁斥了這些種族主義信仰者的結論。英國考古學家格特魯德·卡頓·湯普森後來的研究證實了該遺址百分之百由非洲人建造，許多考古學證據和人類學知識使越來越多的科學家支持這一理論，即該遺址是由非洲土著肖納人的祖先建造的事實，此結論自20世紀50年代以來再也沒有受到爭議。

在大辛巴威城遺址發現的皂石鳥雕成為國家象徵，至今仍描繪在辛巴威國旗上。

有人認為辛巴威城是以現在的辛巴威國家的名字命名的，實際上事實卻是完全相反的。在辛巴威城遺址被發現後，西方殖民者試圖否認大辛巴威城的非洲血統，但黑人民族主義團體把大辛巴威城作為非洲的成就和抵抗的象徵。1980年，當羅得西亞重生獲得獨立時，穆加貝領導的非洲獨立聯盟受到了這個鐵器時代就已經非常輝煌的古代非洲王國首都的啟發，把辛巴威作為獨立後的共和國名稱，於是就有了現在的辛巴威。

大辛巴威城城牆遺址俯視圖

今天，大辛巴威城的廢墟是該國最吸引人的景點之一。它們被分為三個不同的區域，既山丘遺迹、大圍牆和山谷遺迹。山丘廢墟在一座山上，形成了一座衛城，大多數考古學家認為這裡曾是皇宮，是國王居住的地方。大圍牆包括了幾個社區，由一系列14世紀建造的石牆隔開。最後一塊山谷廢墟建於19世紀，建成年代相對較晚。

假如有一天去辛巴威旅遊，在探索這三個獨特的景點之前，一定要先參觀辛巴威博物館。它展示了考古學家發現的許多文物，包括阿拉伯硬幣和中國瓷器，它們證明了這個定居點的貿易歷史。

去辛巴威城並不費事，距大辛巴威最近的城市是馬辛戈，車程只有25分鐘。這裡有幾家接待旅遊者的旅館，它是探索廢墟的最方便的基地。附近的大辛巴威酒店有38間乾淨舒適的客房，此外還有游泳池、排球場、餐廳和停車場。如果你想在日出和日落時觀看廢墟，可以考慮住得更近一些，或者就在廢墟附近露營。
