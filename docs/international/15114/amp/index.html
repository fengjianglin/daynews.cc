特朗普外孫女1歲半學中文，媽媽被誇有遠見，如今她讓女兒學編程

天天要聞
2019-12-14

一直以為中國父母很雞血，對孩子的教育很瘋狂，但沒想到美國也是一樣的雞血和瘋狂！

來，看下總統家的親外孫女~

特朗普一家恐怕是美國歷史上最具「網紅」基因的總統家族了。

繼特朗普本人、特朗普女兒伊萬卡刷屏網路後，他家的寶貝外孫女阿拉貝拉，憑藉著一口流利的中文，再次引起了全世界的關注。

《憫農》《三字經》《茉莉花》……這些中國小孩才熟知的經典，沒想到這個「美國第一外孫女」張口就來。

據說，阿拉貝拉才1歲半的時候，就被送到紐約上東區的語言學校去學中文了，不僅如此，伊萬卡還要求保姆在家必須用中文跟孩子交流。

不得不說，作為媽媽的伊萬卡在教育上的確很有遠見。特朗普當選總統後，阿拉貝拉的中文表演在中國人民心中拉了一大波好感，小小年紀就擔起了中美友誼小使者的重任。

前不久，伊萬卡又在網上曬出了阿拉貝拉學習的新技能——她竟然帶著5歲大的女兒學編程了！並且配文表示「編程是一種未來的語言」。

「今晚和阿拉貝拉一起學編程」

本以為大家會驚呼，總統家的孩子都開始學編程了，萬萬沒想到，美國網友的反應竟然是：

「編程不是未來的語言，而是現在的語言，伊萬卡你落後了」

甚至還有網友評論說：「學習編程是美國的未來！」

孩子學編程在美國已經這麼普遍了嗎？

原來現在整個美國都在大力推廣編程教育，15年政府投入40億美元，16年強制要求高中必修計算機編程學分。據統計，全美有超過1500萬孩子正在學習編程。

這種如火如荼的局面，讓人不禁想問：

美國人是瘋了嗎？

編程難道不是計算機專業的課程嗎？

我的孩子又不當程序員有必要學嗎？

2

當然有必要，因為時代變化了。

不知道你有沒有發現，雙十一的物流速度快了很多，以前要等上半個月的快遞，現在基本上兩三天就到了，有些訂單甚至第二天就到貨了。

原來天貓、京東、蘇寧等各大電商平台都早已建立了自己的「無人倉」。

機器人24小時不眠不休工作，一旦顧客下單，就會立馬根據演算法和商品銷量的歷史數據進行處理。

我們的生活的確是便捷了，但還是讓人忍不住倒吸一口涼氣：

人工智慧真的開始跟人類搶飯碗了！

除了這種體力勞動的行業，曾經我們認為不可能被取代的創意行業也漸漸被佔領。

在藝術領域，人工智慧通過大量學習、模仿人類風格做出的畫作在巴黎拍賣出了1萬歐元（約合人民幣8萬元）的高價；

你能猜出哪張是AI畫的嗎？

前段時間，新華社「AI合成主播」正式上線，人工智慧不但可以模仿主持人的面部表情和聲音，還能精準播報不出差錯。

其實，全世界最頂級的管理諮詢公司麥肯錫早就預測過——

到2030年，也就是12年後，全球將有8億人因為人工智慧而失業。

在未來，不管你的孩子從事哪個行業，但凡想要有所成就，都需要懂編程。

3

既然編程這麼重要，一定很難吧？恐怕只是少數人的特權。

恰恰相反！

任何一個關注教育的父母大概都發現了，「編程學習從娃娃抓起」 已經是當下政策的大趨勢：

早在2017年，中國國務院發布《新一代人工智慧發展規劃》，人工智慧上升為國家發展戰略。

規劃明確提出：「在中小學階段設置人工智慧相關課程，逐步推廣編程教育。」此次規划出台，將會加快編程及相關人工智慧課程進入中小學課堂。

浙江新高考改革將信息技術納入2017年高考選考科目。

與此同時，浙江省公布的選考科目範圍中，有48所「211」高校的338個專業，指定信息技術為選考科目。

在高校公布的選考科目範圍中，信息學的上榜率排名第三，僅次於物理和化學。

在今年全國政府工作報告四次提及「智能」，並特別指出要加強新一代人工智慧研發應用，在醫療、養老、教育、文化、體育等多領域推進「互聯網+」。

可以看出，我國政府高度重視人工智慧的發展以及人才的培養，著力於普及和推廣編程教育，以便讓更多的孩子學會編程，擁有改變未來的可能！
