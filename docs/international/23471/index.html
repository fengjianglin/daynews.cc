英國女王和凱特王妃常戴王冠出席重要活動，為什麼梅根沒有呢？

2019-12-19 1,129條評論 3,561次閱讀 1011人點贊

當我們想起王室成員時，腦海里經常會想像他們，就像童話故事裡的王子公主一樣，無論在什麼場合都會一直戴著王冠。

梅根 · 馬克爾

梅根只在結婚那天戴過王冠

但是，在現實生活中，王室成員們並非每天都會戴著王冠。姑且不論王冠上一般飾有貴重的寶石或者珍珠，如果在日常生活中都戴著如此閃耀奢華的王冠，未免顯得誇張。

有些王冠還很沉重，穿戴起來原本就不方便，如果要戴著王冠行走，那就是難上加難了。

據悉，泰國國王瑪哈·哇集拉隆功，在今年5月國王加冕儀式上，佩戴的王冠就重達7公斤，比英國女王伊麗莎白二世的「帝國王冠」還要重7倍。

看來，欲戴王冠,必承其重，這句話真是一點不假啊。

雖然一般在隆重的場合，王室成員們才會戴上珍貴的王冠，穿著華貴的衣服閃亮登場。而我們也經常看到英女王伊麗莎白二世，以及威廉王子的妻子凱特王妃戴著王冠參加國家活動，或者接待其他國家的重要領導人。

但是，與她們相較，英國哈里王子的夫人梅根卻似乎顯得比較寒磣。

自從與哈里王子結婚那天，頂著由英女伊麗莎白二世王借給她的，原屬於瑪麗皇后的鑽石王冠（The Filigree Tiara），度過了人生最重要的日子後，梅根就再也沒有被看到戴過王冠了。

而這究竟又是什麼原因呢？

梅根沒有機會戴王冠

英國王室禮儀學校主管、前英國王室管家格蘭特·哈羅德在接受採訪時透露，「傳統上，王冠是婚姻的一個標誌。因此，通常新娘會在結婚當天佩戴王冠。結婚後，每當要參加高級別的正式晚宴時，成為王室成員的新娘也可以佩戴王冠。而單身的女士則一般都不會帶王冠，除非他們出生於王室，是公主。」

但是，既然梅根已經嫁給了哈里王子，按理說，已經滿足了格蘭特所說的條件，為什麼至今在出席重要的國家活動時，梅根還只是佩戴著簡單的飾品，而沒有戴上王冠呢？

哈里王子妻子梅根（資料圖）

熟悉英國王室的評論員理查德·菲茨威廉姆斯給了我們一個解釋，「實際上，答案非常簡單，因為她根本沒有合適的機會。」

截至目前，梅根所參加的活動中，還沒有哪個活動正式到需要佩戴王冠。而王室成員一般不興戴王冠參加休閑活動，因為這樣看起來太誇張了。

所以儘管公眾已經有很長一段時間沒有看到梅根戴王冠了，但這只是時間問題。

理查德也提到，「毫無疑問，未來梅根將會在適當的時候，在適當的場合戴上王冠，但是這顯然取決於她和哈里王子是否想要履行高級王室成員的職責。」

【「繽FUN世界」點評】

自認識哈里王子並嫁入英國王室以來，頂著非裔、有過婚史、大齡等「黑歷史」的梅根，便成為英國媒體和民眾關注的焦點。而梅根的大多數行為，都遭到英國媒體和民眾無情地指責。舉止不得體、有損英國王室形象、奢侈浪費納稅人的錢成為梅根身上的標籤。梅根還甚至被傳與王室成員不和，不受英女王伊麗莎白二世待見。所以，梅根一直沒戴上王冠的原因也飽受猜疑。

也許如王室評論員理查德所說，梅根不是不能戴王冠，只是時候未到。但是，也很難說他的解釋就一定正確。畢竟梅根自與哈里王子結婚以來，已經有一年半了，才只戴過一次王冠。這個時間未免也太長了。

梅根究竟能不能再次戴上王冠，確實難以預料。但是，我們唯一能知道的是，當她再次戴上王冠時，肯定會成為英國一大轟動的消息。

（花花）
