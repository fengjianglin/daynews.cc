世界上最難偽造的八種紙幣，我國有兩種上榜

2019-12-05 375條評論 3,793次閱讀 2821人點贊

隨著高科技防偽技術逐步得到改善，世界鈔票的防偽水平也大大提高。2009年，國際貨幣事務聯合會評選出世界上最難偽造的八張鈔票……看看你手裡有幾種吧。

第一種：面額，萬元堅戈

國家：哈薩克

一萬元堅戈是第一種使用安全絲(security/thread)的鈔票，安全絲在鈔票兩面均清晰可見。安全絲具有獨特的標記，而且還是透明的，下面多出的安全特性即看得清清楚楚。

第二種：面額，50比索

國家：墨西哥

製造這種鈔票的材料是多層塑料聚合物，在將鈔票傾斜時，塑料聚合物就會改變顏色，使用自製設備仿造這種鈔票幾乎不可能。而透明視窗又加了一道安全保險，令其更難偽造。

第三種：面額，千元克朗

國家：瑞典

千元面值的瑞典克朗融入了獨特的水印，是第一種使用顯微條紋(microlens/strip)的鈔票，同即將應用於新版百元面值的美元鈔票上的技術類似。

第四種：面額，10港元

地區：中國香港

凸起的油墨，比如，在這張照片中看到的書寫字體，使得這種鈔票具有獨特的質地。另外，印在鈔票兩面的一匹駿馬圖像完美地排列著。只用粗糙的自製設備，相信很難偽造出具有上述兩種防偽技術的鈔票。

第五種：面額，10元盧比

國家：尼泊爾

尼泊爾的10元盧比鈔票有一個清晰的聚合物視窗——一般的紙張很難或不可能複製，細細的金屬絲還融入到鈔票之中。

第六種：面額，千元第納爾

國家：伊拉克

鈔票上有凸起的字母符號，令其擁有獨特的質地，上面的油墨只能在紫外光線下才能看得見。

第七種：面額，20英鎊

國家：英國

新版20英鎊面值的鈔票有一個顯著的全息條。一旦將鈔票傾斜，全息條上的圖像能在英鎊標誌和數字「20」之間來迴轉換。

第八種：面額，100人民幣

國家：中國

2015年版100元紙幣增加了防偽性能較高的光彩光變數字、光變鏤空開窗安全線、磁性全埋安全線等防偽特徵，提升了人像水印等防偽性能。

世界上最難偽造的八種紙幣，單我國就有兩種上榜，可見我國造幣水平世界領先，收藏價值不言而喻？
