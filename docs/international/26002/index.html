1.22萬億、1.21萬億！前三季度，韓國GDP險勝俄羅斯，那全年呢？

2019-12-21 1,205條評論 2,965次閱讀 1839人點贊

2018年，俄羅斯完成的名義GDP約為1.648萬億美元，在全球排在第11名。之後是韓國，完成的名義GDP約為1.62萬億美元，全球排第12名。兩國經濟總量差距很小，且韓國經濟增速要比俄羅斯更高。

因此，網友們普遍認為在2019年韓國的GDP很可能實現對俄羅斯的趕超，從而使得排名上升到全球第11名。從今年上半年的情況來看，的確如此——即韓國的GDP已經超過了俄羅斯。

但在第三季度，俄羅斯經濟實際增速又開始恢復了，由第一季度的0.5、二季度的0.9%，上升為1.7%，從而使得前三季度俄羅斯的GDP總量達到了1.21萬億美元（實際增長1.1%）。與此同時，前三季度韓國GDP約為1.22萬億美元（實際增長1.9%）。

這就是說，前三季度韓國GDP與俄羅斯的差距又縮小了，僅約100億美元左右。韓國雖然仍險勝俄羅斯，但俄羅斯經濟增速在回升。那問題來了，2019年全年，韓國GDP最終能否實現對俄羅斯的趕超呢？

2019年全年，韓國與俄羅斯的GDP預測

我們先看韓國。今年第一季度，韓國GDP同比實際增長了1.7%，第二和第三季度同比實際增速都是2%。其中，第三季度韓國的GDP平減指數下降1.6%（-1.6%），為1999年第二季度（-2.7%）以來的最大降幅。

所謂GDP平減指數，指的是：沒有剔除物價變動前的GDP增長與剔除了物價變動後的GDP增長之商，是綜合反映貨物和服務價格變化的宏觀指標。簡單的說，GDP平減指數的降低會導致韓國經濟的名義增速在下降。

俄羅斯在今年第三季度的經濟實際增速恢復到1.7%，GDP平減指數同比增長1.4%，雖然也不是很高的數值，但要比韓國好一些。此外，三季度俄羅斯經濟多項數據在上升，俄羅斯央行預測在今年第四季度，其經濟增速有望再上漲到1.9%，全年增速有望由前三季度的1.1%，提升到1.3%。

總之，就像第三季度那樣，雖然GDP實際增速韓國更高（2%對比1.7%），但GDP平減指數俄羅斯更優，導致兩國經濟的名義增速偏向俄羅斯。加上韓元貶值幅度超過俄羅斯盧布，因此數據再度偏向俄羅斯了。

按照IMF的預測：2019年全年，韓國經濟實際增速預計為2%，要比俄羅斯更高。但GDP轉換成美元後，韓國約為1.63萬億美元，俄羅斯為1.64萬億美元——仍小幅領先韓國。

韓國人均GDP依然超過俄羅斯

按照IMF的預測，很可能在2019年俄羅斯的GDP總量仍在韓國之上，但俄羅斯的人均GDP不到1.2萬美元，而韓國的人均GDP卻超過了3萬美元（和西班牙、義大利的人均相當），從人均角度來看韓國依然佔優。

當然了，這是從匯率角度來對比的！如果採用世界銀行的購買力平價GDP，那俄羅斯的經濟總量超過了英國、法國和德國接近。與韓國相比，俄羅斯的領先優勢就更大了。此外，俄羅斯在政治、軍事、外交、地緣影響力、能源等多個方面也領先韓國，綜合國力仍不是韓國可以相提並論的。本文由【南生】整理並撰寫，無授權請勿轉載、抄襲！
