拉脫維亞網友上傳“將俄羅斯國旗插狗屎”照片，俄方封禁圖片網站

天天要聞
2019-12-04

據日本Gigazine網站12月4日報道，美國知名圖片網站Shutterstock因收存了多張俄羅斯國旗插在糞便上的圖片，被俄羅斯官方列入黑名單。

據報道，俄羅斯通信、信息技術和媒體監督局(Roskomnadzor)已在11月13日將該圖片網站拉黑，目前，Shutterstock網站已經在俄羅斯境內被正式屏蔽。

此前，一位名叫aijaphoto的拉脫維亞網友在Shutterstock網站上發布了數張俄羅斯國旗被插在糞便上的照片。aijaphoto對照片的描述是「俄羅斯國旗插在狗屎上」，似乎是在表達對俄羅斯的抗議。

目前，這些圖片已經被刪除，但是在網上還是可以找到略縮圖。其中不僅有俄羅斯國旗插在狗屎上的照片，還有俄羅斯國旗插在各種糞便上的照片。除此之外，拍攝者還從各個角度進行了拍攝。

俄羅斯聯邦議院(參議院)於2019年3月通過了一項法律，對那些傳播虛假信息，或在網路上傳播 「侮辱國家」信息的個人施以嚴厲懲罰。普京總統簽署了該法律。

這項法律規定，有傳播行為者，最高可處以罰款10萬盧布(約1.1萬人民幣)。而如果有人在網上對國家、當局、公民、俄羅斯國旗和憲法進行了「明顯的侮辱」，將會被處以兩倍的罰款或最多15天的監禁。

隨著俄羅斯政府加強網路監管， 俄羅斯通信、信息技術和媒體監督局認定這張有爭議的圖片是在「侮辱俄羅斯」，並將Shutterstock列入黑名單，以阻止俄羅斯國內的網路訪問。自2019年11月28日起，該網站域名在俄羅斯國內無法訪問。

Shutterstock也意識到了來自俄羅斯的封鎖，正在處理這個問題，以便恢復訪問。Shutterstock評論道:「有問題的內容已經被刪除，Shutterstock將繼續刪除圖片縮略圖。」

11月下旬，一個在德國註冊的用戶Yulia Raketic在在線社會公益請願平台（Change.org）上，發起了一份請願書，呼籲Shutterstock將 aijaphoto拉黑。Raketic說，該帳戶應被禁止，因為他不尊重俄羅斯並侮辱了俄羅斯的國旗，還導致Shutterstock網站被封鎖。」目前，請願書已有200多個簽名。

今年7月，俄羅斯通信、信息技術和媒體監督局還設法讓Facebook刪除了一張照片，照片上的俄羅斯徽章上印有納粹標誌，此外該機構還讓Twitter刪除了一條推文，稱推文內容對俄羅斯人民來說是一種侮辱和褻瀆。
