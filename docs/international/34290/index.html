英媒：爆發豬瘟疫情的背景下 泰國雞肉被中國市場“拯救”

2019-12-24 1,254條評論 4,318次閱讀 1274人點贊

根據報道，英國路透社12月19日發表文章，稱在爆發豬瘟疫情的背景下，泰國對中國雞肉出口註定會增加。泰國一個行業協會在周四稱，預計泰國明年對中國雞肉出口將增加60%，因中國在爆發豬瘟疫情後尋求替代肉類供應。

（圖源：泰國頭條新聞）

該文稱中國當局表示，中國的生豬數量比一年前減少了大約40%。泰國雞肉加工出口商協會經理庫克里特·阿萊帕格恩說，中國是世界上最大的豬肉消費國，預計中國對肉類替代品需求的增加將把明年泰國對華雞肉出口量提升至8萬噸，這將比今年的5萬噸出口量增加60%，而今年的對中國出口量已經是2018年對華出口1.8萬噸的近3倍。

「隨著獲准對中國出口的加工廠越來越多，對中國出口勢必會增加。」庫克里特說。2018年3月，泰國有7個雞肉加工廠獲准向中國出口。上個月，又有8個加工廠獲准進入中國市場，樂觀估計，這些加工廠將於2020年開始向中國出口雞肉。儘管對中國出口的預測很樂觀，但是根據該協會提供的數據，泰國明年的雞肉出口總量預計僅增加4.26%，達到98萬噸。

日本和歐盟是泰國雞肉的前兩大買主，但是2019年泰國對這兩個地區的雞肉出口量卻下降了，原因是泰銖走強，導致出口（雞肉）價格與主要競爭對手巴西相比失去了競爭力。

但是對中國出口而言，庫克里特說，泰國會從中國和東盟簽署的自由貿易協定上受益，再加上泰中兩國又是鄰國，交貨期只有7天。
