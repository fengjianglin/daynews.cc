富士山不是日本的，日本每年需交天價租金，隨時可能血本無歸

天天要聞
2019-12-03

一說起日本，很多人第一時間想到便是櫻花和富士山，富士山被日本人民譽為”聖岳”，是日本民族的象徵。而它作為日本的國家象徵之一，在全球享有盛譽，每年去富士山旅遊的中國遊客也非常多。

日本的櫻花和富士山

象徵著日本自然、歷史、現代的三大景點之一的富士山，是位於日本靜岡縣的一座活火山，它的海拔高度為3776米，是日本最高的山峰，還是世界上最大的活火山之一，它在日本的地位不亞於中國的泰山。當然了你可能不知道，富士山並不是日本在政府的財產，而是屬於私人所有!富士山不是日本的，每年日本需要交天價租金，還隨時可能血本無歸。

夜幕下靜謐的富士山

這座山的歸屬權一直是屬於淺間神社，屬於私人所有。這座山原本是屬於德川家康家族的,後來德川家康把這座山送給了淺間神社，而對於日本來說他們是非常尊重個人的利益的，因此這座山實際上是屬於淺間神社這個社團的。雖然在這後來經歷過很多次的歸屬交換，但是在現在的時候富士山3000米以上的位置還是不屬於日本政府，是屬於別的社團或者個人的，所以每年日本政府都需要支付天價的租金，以供富士山的旅遊開發。

淺間神社

但是日本政府並不虧，因為富士山是國山，不愁沒遊客。每年慕名前往富士山的旅客絡繹不絕，上富士山雖然不收門票錢,但修建了豪華的旅館和配套設施,而遊客上下山，往往在山上停留兩到三天，這兩三天時間就足夠賺回本錢了。但日本政府投資富士山,還有一種致命危險，還可能導致隨時血本無歸，那就是難以預測的自然災害所帶來的危險。眾所周知，富士山是一所活火山，別看表面冰雪一層，似乎永遠沉睡的樣子，其實核心熔岩涌動，隨時準備噴發。而且日本多發地震，火山地下的岩漿和熱水容易受到破壞，更易引起群發性地震。

地震致日本列島部分沉沒，面積約相當於大半個東京

最可怕的是，科學家推算，由於富士山巨大，如果發生7級噴發，日本列島將不在適合人類居住，日本這個國家很有可能就不存在了。雖然距離上次噴發是在1707年，但近幾年的日本經常遭受地震山洪等自然災害侵害，火山提前爆發也是極有可能，所以日本政府目前也是一直做火山爆發的準備。畢竟半個日本都可能被吞併，整個國家都可能會血本無歸。
