印政府再出手：首都部分地區被斷網

2019-12-20 562條評論 3,067次閱讀 802人點贊

【文/觀察者網 王慧】

局勢緊張就斷網，這已成為印度政府的慣常操作，一度創下了5年斷網357次的超高記錄。自上周印度國會上議院通過一項涉嫌排斥穆斯林的「公民身份修正法案」（CAA）後，引發該國連日示威和騷亂，新德里和孟買成了「重災區」。

綜合《經濟時報》、《印度斯坦時報》等印媒12月19日報道，為「應對」抗議，印度政府又使出了斷網大招：印度三大電信巨頭Bharti Airtel，Vodafone Idea和Reliance Jio均收到了政府指示，19日在新德里許多地區暫停語音、簡訊和互聯網服務。

電信運營商Airtel於19日上午10點鐘起開始暫停上述服務，該公司代表稱，他們也不能確定服務暫定的具體期限。

「就像我們所說的，我們按照政府機構指示，目前在你所在位置暫停語音、網路和簡訊服務。一旦暫停指令取消，我們的服務將會完全恢復正常。」 Airtel公司的一位客服主管說。

除Airtel外，在新德里暫停服務的運營商還有Vodafone Idea和Reliance Jio。

消息人士稱，Vodafone從早上9點到下午1點在新德里的一些地區暫停了語音、數據和簡訊服務。該公司在其官方推特上向用戶發出通知：「根據政府指示，我們有計劃地暫停服務。一旦收到當局的下一個指令，我們會立刻做出調整。感謝你的耐心和合作。」

印度政府本次斷網和該國國會上議院日前通過的一項新的修正案——「公民身份修正法案」有關。該修正案將給予印度三個鄰國的非法移民公民身份，但前提是他們不是穆斯林。

然而，上述修正案很快引起了印度國內民眾的強烈抗議與反對。穆斯林聚集的阿薩姆邦、西孟加拉邦成為近期騷亂的重災區，自上周起相繼被政府強行斷網。

12月19日新德里示威活動中與警察對峙的印度婦女 圖源：法新社

諷刺的是，為了澆滅示威者的怒火，印度總理莫迪16日曾連發5條推特呼籲各方冷靜，而英國廣播公司（BBC）就此評論道，考慮到有的地方已經斷網，尚不清楚有多少人看到了莫迪的推文。

印度有著超高頻率的斷網操作。據《今日印度》12月18日公布的統計報告，顯示2014年以來印度政府關閉網路已達357次：2014年6次；2015年上升到14次；2016年達到31次；2017年達到79次；2018年達到峰值134次；2019年截至12月15日， 互聯網服務關閉共計93次。
