加拿大亞裔母親申請難民造假 結果兩個孩子要被遞解出境

天天要聞
2019-12-10

一名來自南亞國家孟加拉的女子於6年前入境加拿大的時候，使用假名和遞交假文件申請難民，結果移民難民局認定她的證詞「模糊又有誤導性」，屬於故意欺騙，拒絕了她的難民申請。

但是在聯邦法院近日作出裁決，不是這名女子被驅逐出境，而是她的兩個孩子。

圖片來源：Aaron Lynett/National Post

女子名叫Sabina Shahrin Nipa，她的兩個孩子Redwan和Shuruvi Mozumder分別19歲和18歲。兩個孩子從十多歲起就住在多倫多，已經是典型的多倫多孩子。小女兒Shuruvi剛被多倫多大學錄取。Redwan是市政府的員工，負責校後班照看兒童。但是，如果他們的律師不能說服法官停止遞解程序，很快兩人就將被押送到邊境，在邊境官的護送下進入紐約州水牛城，被迫在一個他們沒有任何家人和聯繫的國家重新開始生活。

律師Richard Wazana說，兩個孩子完全是無辜的。

難民申請被拒的案例在加拿大很常見，每天都有人被遞解。但是這個案例又非常特殊，做錯事的人是母親Nipa，但是她還不會被遞解，一旦被遞解，可能是回到孟加拉，而不是美國。

從孟加拉到美國再到加拿大

律師Wazana說，如果這位母親的移民經驗多一點點誠實，加拿大邊境局也不會這麼急得遞解她的兩個孩子。她的小女兒剛滿18歲。

Nipa的申請個案相當混亂繁雜，充滿謊言，並拖延了多年。她稱第一次以人道主義和同情理由提交難民申請並使用了假名，是因為聽了一位移民顧問的建議，而她以為那人是個律師。

Nipa在申請中稱，她早年在孟加拉與一名男子結婚，因為遭到家人的反對沒有登記註冊，在生下兒子後就遭到家人和族群的指責。於是，為了逃離，她同意與一名在美國的男子結婚獲取綠卡。當美國丈夫發現她已婚並懷有別人的孩子後，就開始「虐待和恐嚇」她。Nipa逃回到孟加拉，又於2001年回到美國生下女兒Shuruvi，之後再回到孟加拉。

直到2013年，Nipa帶著兒子Redwan和女兒Shuruvi到加拿大申請難民。申請和上訴都被拒，拒絕的理由是認為這家人可能會去美國，因為Nipa有美國綠卡，Shuruvi和Redwan都是美國公民。

但是這個裁決之後，當局沒有任何行動。孩子們很快適應了這裡的生活。Redwan說，加拿大開始變成他們的家，如果有人問我你是哪裡人，我會說「Canadian」。

今年春季，Shuruvi從東約克高中畢業，被多大生命科學系錄取，她打算明年秋季入學。但是11月1日，也就是在她18歲生日的兩周之後，邊境局（CBSA）下令於12月9日遞解兩個孩子。現在Shuruvi不知道自己要怎麼辦。

國會議員求情

Beaches-East York選區的自由黨國會議員Nathaniel Erskine-Smith向法庭致信，稱將兩個孩子遞解到美國的決定「不公正」。

信中寫道：「他們雖然在美國出生，但是從來沒有在那裡生活，與那裡沒有任何聯繫。其次，遞解孩子也將拆散這個家庭。孩子們並沒有錯。」

Erskine-Smith希望遞解令暫停，至少等到Nipa的裁決結果出來，三人一同留下，或者被送往同一個國家。

Redwan和Shuruvi表示，如果自己真的被驅逐到美國，會先找到一個落腳和生活的地方，然後找份工作，等到安定下來，會想辦法重回學校。

律師Wazana認為，Nipa未必會被遣返回孟加拉。如果Redwan和Shuruvi離開加拿大，他們將沒有機會再回來。他要求在評估人道主義和同情申請之後再執行遞解令。

「邊境局的決定應該引起公眾的不安。這樣拆散家族應該讓所有人感到不安。」

邊境局發言人回應說，基於隱私法，不能對這個案子發表評論。「但是我們可以說的是，將某個人驅逐出加拿大，不是輕輕鬆鬆的決定。」

邊境局媒體關係主任Ashley Lemire說：「移民和難民保護法規定，驅逐令必須儘快執行。加拿大邊境服務局會堅定地執行。
