沒想到美國，竟然也會去進口25年前的"洋垃圾"，而且是高價

2019-12-16 1,741條評論 2,785次閱讀 1600人點贊

去過香港、澳洲的朋友，心裡會很羨慕當地人，因為他們可以開著一些經典情懷老車馳騁在馬路上，尤其是當漫哥看到《頭文字D》動漫的那些車，我真的是挪不開離開的步伐滴。但羨慕香港、澳門可不止是我們國內車友，就連老美人民也表示羨慕。

為啥呢？1988年，時任美國總統的羅納德·里根簽署了《進口汽車安全合規法案》，該法案規定所有進口整車必須符合美國的汽車安全法規，因此所有非官方總代理渠道進口到美國車輛都被拒之門外了。

而到了1998年，苦逼的美國車友卻迎來了一個喜大普奔的好消息：NHTSA美國高速公路安全管理局出台新政，允許以個人形式進口下線滿25年以上的車型。所以，廣大美國車友都期待解鎖自己心儀車型的那一天，也就是可以正式引入這些”洋垃圾”了。

舉例說明下吧，比如如果你想在2020年1月進口一輛95年款的Skyline GT-R（R33），這輛車的生產時間必須是1995年1月才行。看到這裡，漫哥我化身成為一顆檸檬精，賊酸的那種，我心裡也有一個”戰神夢”，即便是開一台”口碑不佳”的R33，我也開心得像是一個200斤的胖子呀！

馬上就2020年了，已經有”好心”海外網站已經在盤點部分2020年可以合法進口的經典車了。首先，人氣最高要屬本田Integra Type-R（DC2），這款車最初於1995年9月發售，美國車迷要等到2020年9月後才能合法進口這款經典性能車了。

天天有人說咱們國內消費者人傻錢多，實際上老美腦熱的人也大有人在，前不久，一台不太純粹的謳歌Integra Type R就以驚人的8.2萬美元（約合58.4萬元）被人帶回了家，只因為想側面品一品DC2的魅力，因為這台上世紀90年代的老車身體躺著本田萬轉紅頭引擎——B18C，可以帶你體驗最純粹的”爆TEC”那種不可描述的感受。

不只是國內的”破三菱”EVO才是理財產品，而在老美同樣是。到了2020年，美國就可以合法進口1995年2月推出的Lancer GSR Evolution III和Lancer RS Evolution III了。而熟悉《頭文字D》的朋友看到這個車型，就知道是《頭文字D》里須藤京一的戰車了。

另外，呼聲很高的還有保時捷”風冷絕唱”，代號993的末代風冷911，當年不能進口到美國的車型，如今老美人民也能翻993的牌子了。

看到這裡，漫哥心裡真是羨慕嫉妒恨，我心中也有海外經典車型的情結。所以想問問大傢伙，如果可以進口經典車型，你最想擁有哪台情懷座駕呢？
