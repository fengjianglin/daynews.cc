全球首批1500V 200kW級以上光伏項目在印度浦那併網

天天要聞
2019-12-13

【能源人都在看，點擊右上角加’關注’】

近日，隨著印度馬哈拉施特拉邦浦那BOSCH光伏電站項目成功併網，讓印度成為全球首批1500V 200kW級以上光伏項目併網的國家之一，是名副其實的印度「領跑者」。項目全部採用特變電工最先進的行業最高容配比1500V組串光伏解決方案。

特變電工1500V組串光伏解決方案圍繞核心產品TS208KTL-HV 型逆變器從「1500V系統+大方陣+高容配比+智能運維」四大特點實現度電成本最優，光伏電站綜合收益提升7%以上。

◆1500V+大方陣+高容配，最優LCOE

就當前而言，1500V逆變器在海外已然成為了主流趨勢，通過高超配的設計實現LCOE已成為實現電站收益最大的有效途徑。特變電工1500V TS208KTL-HV組串式逆變器在印度市場中採用「Y+1」接線方式的容配比可提升至1.7倍以上，可支持3.75MW方陣以上單元設計，是目前全球最高容配比大功率組串逆變器。

在印度BOSCH項目採用8.7MW的大方陣設計，每個方陣配置32台TS208KTL-HV組串式逆變器，每台逆變器接入了30串組件，每串由30塊345W組件組成，逆變器直流側輸入310.5kW，容配比高達1.49，在長達25年的電站生命周期內，可以降低LCOE7%以上。

◆動態過載能力高，完美匹配雙面組件

TS208KTL-HV組串式逆變器具備高動態過載能力，可以根據環境溫度自動調整過載能力，50℃支持滿載長期運行能力，40℃支持1.1倍長期過載能力，25℃支持1.2倍過載能力，每路MPPT的最大電流26A，完美匹配雙面組件。

印度BOSCH光伏項目位於印度西部城市浦那，年最高氣溫高達50℃，TS208KTL-HV組串式逆變器可以穩定長期滿載運行於該工況，為客戶帶來更高發電量和收益。

◆IP66防護等級，C5防腐等級‍

TS208KTL-HV組串式逆變器外部部件防護等級IP66，在器件選擇和結構設計上保證了整機的防護性能。逆變器的外置風扇防護等級IP68，滿足複雜環境下的可靠運行。同時為了適應高鹽霧、高濕度環境，該逆變器產品在櫃體表面使用新一代靜電粉末噴塗處理技術，防腐性能強，25年不脫落、不鏽蝕。此設計可有效應對惡劣自然環境對特變電工逆變器的損壞，減少故障，延長使用壽命。

◆寬頻PLC通訊 降低投資成本

◆寬頻PLCC通訊，降低投資成本

寬頻PLCC通訊，降低投資成本‍

TS208KTL-HV組串式逆變器，具有PLC通訊功能，完美替代傳統RS485通訊方式，節省了專門鋪設通訊線的物料和建設成本。特變電工的PLC通訊採用寬頻載波技術，載波頻率2MHz~12MHz自適應，比傳統的窄帶載波技術更穩定，通訊速率更高。

印度BOSCH光伏項目採用了PLC通訊方式後，全站調度時間比傳統RS485方式降低了30%以上，達到3S以內。

特變電工新能源全資公司—特變電工西安電氣科技有限公司作為國內最早從事逆變器企業之一。憑藉19年逆變器研發製造經驗，始終以技術創新來降低了逆變器成本，產品具有高容配比、高效率、高可靠性的特點，應用於大型地面、水面、複雜山地、屋頂等各種環境，服務於集中式、分散式等各類光伏電站1000多座，業務遍及全球4大洲20多個國家，為全球光伏發電提供最優LCOE解決方案。
