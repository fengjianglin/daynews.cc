美國小偷偷走龍蝦卡車，公司派另一輛車去追，網友：這很波士頓

2019-12-20 1,185條評論 2,511次閱讀 1266人點贊

據美國媒體12月19日報道，美國波士頓警方接到報案，稱兩輛裝滿龍蝦的貨車相撞，需要他們到現場處理。

▲工作人員手握龍蝦

如果你以為這只是簡單的交通事故就錯了。據了解，相撞的兩車中的一輛是從附近的海鮮批發公司偷來的——事發時海鮮批發公司的卡車停在碼頭上裝龍蝦，一名29歲男子趁工作人員不注意，偷偷溜進駕駛室企圖駕車潛逃。

想到被偷的車裡裝著價值10000美元（約70000元人民幣）的龍蝦，海鮮批發公司立馬派出另一輛裝著龍蝦的貨車去追。雖然海鮮批發公司的員工很快就追上了小偷，但是他們未能說服小偷把車停下來，小偷反而開車撞上了員工的車。

兩車相撞後，員工順勢「抓住了嫌疑人，等候警察趕到現場」。目前，警方已經逮捕了偷車賊。波士頓警方稱，該盜賊將會面臨偷車，吊銷駕照後駕車、使用危險武器襲擊、毆打他人，辱罵、恐嚇他人等指控。

▲波士頓警方公告

事件報道後，周圍居民和網友們紛紛表示，「還有比這更『波士頓』的行為嗎？」

海鮮批發公司銷售市場部經理彼得·拉戈里奧（Peter Lagorio）在接受採訪時說：「這可能是有史以來最『波士頓』的事，這對每個參與者來說都是一次非常『波士頓』的經歷。」

（賈琦）
