美駐德大使稱制裁俄羅斯“非常親歐”，俄媒嗆：哦，真的嗎？

2019-12-23 2,940條評論 3,923次閱讀 1900人點贊

溫家越

【環球網報道 記者 溫家越】美國總統特朗普日前批准對俄羅斯向德國輸氣的「北溪-2」項目實施制裁，引德國政府不滿。多家俄媒22日稱，美國駐德國大使理查德•格雷內爾表示，美國對該項目制裁是「一個非常親歐的決定」，對歐洲有利。對此，「今日俄羅斯」(RT)22日直接在標題「回嗆」道：哦，真的嗎？(Oh, really？)

RT報道截圖

RT稱，格雷內爾在接受德國《圖片報》和美聯社採訪時表示，特朗普簽署國防授權法案，設想對參與「北溪-2」天然氣管道項目的企業實施嚴厲制裁後，他收到歐洲外交官表達感謝的消息。俄羅斯衛星通訊社稱，格雷內爾特別提到，「15個歐洲國家、歐盟委員會和歐洲議會」都表達了對「北溪-2」項目的擔憂。

俄羅斯衛星通訊社報道截圖

俄衛星社援引格雷內爾的話說，對「北溪- 2」項目實施制裁是「一個非常親歐的決定」，因為華盛頓一直「從我們的歐洲夥伴那裡聽到，美國應該支持他們(作出)的努力」。格雷內爾表示：「目前在德國，有不少關於更多支持歐洲的言論，我們相信，在『北溪-2』項目上，我們採取了非常親歐的立場。我一整天都聽到歐洲外交官感謝我採取了這一措施。」不過據RT描述，格雷內爾沒有詳細說明，哪些國家的官員「排著隊」向他表示感謝。

RT提到，長期以來，德國一直強烈反對外界干預其與俄羅斯的合作項目，一些人甚至呼籲「以牙還牙」，比如禁止德國從美國進口液化天然氣。而對於格雷內爾稱制裁是「非常親歐的決定」的說法，RT在22日的報道標題中反問道：「哦，真的嗎？」

「北溪-2」項目旨在鋪設一條由俄羅斯經波羅的海海底到德國的天然氣管道，可繞過烏克蘭把俄天然氣輸送至德國，並通過德國幹線管道輸送到其他歐洲國家。美國一直阻撓這一項目，認為「北溪-2」項目將使歐洲各國加深對俄羅斯能源的依賴，同時會損害烏克蘭的經濟和戰略安全。

當地時間20日，特朗普簽署了2020財年國防授權法案，內容包括對參與「北溪-2」天然氣管道項目的企業實施制裁，措施包括禁止相關企業人員去美國旅行，以及凍結他們在美國的財產等。德國聯邦政府發言人21日回應稱，德國政府拒絕接受此類「域外製裁」，此制裁影響到德國和歐盟企業，構成了對德方和歐方內政的干涉。當地時間22日，俄羅斯外交部發言人扎哈洛娃也對制裁進行猛烈抨擊，稱背負巨額國債的美國正試圖阻止其它國家的經濟增長。
