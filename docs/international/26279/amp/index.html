/ 01 /

印度：有毒霧霾持續籠罩 首都空氣污染達嚴重等級

2019年12月13日，印度首都新德里(New Delhi)連續第2天籠罩在有毒霧霾中，由於氣溫及風速降低，致命污染物滯留在空氣中，導致能見度降低。空污質量達到「嚴重」(severe)等級。

/ 02 /

日本：阿蘇山火山噴出煙塵 氣象廳發布預警

2019年12月13日，日本熊本縣的阿蘇火山噴出煙塵，日本氣象廳目前已發出警報，阿蘇火山或將於近日爆發。

/ 03 /

俄羅斯：執法人員在莫斯科抓獲5名恐怖分子

2019年12月13日，俄羅斯聯邦安全局表示，該部門與俄內務部在莫斯科抓獲5名「伊斯蘭國」成員。

/ 04 /

尼日：西部一軍營遇襲至少71人死亡

2019年12月10日晚，尼日安全部門消息稱尼西部一軍營遭不明身份武裝分子襲擊，造成至少71名士兵死亡，10多人失蹤。

/ 05 /

澳大利亞：提出漲薪要求遭拒絕 維州警方舉行罷工

2019年12月12日，澳大利亞維多利亞州警察工會提出漲薪4%的要求被拒絕，警方開始舉行罷工。這是警方自2011年以來的第一次罷工。

/ 06 /

巴基斯坦：200名律師襲擊醫院火燒警車 3名病人失救死亡

2019年12月11日，巴基斯坦200多名律師前往拉合爾一處醫院進行暴力抗議，他們不僅損壞了醫院的內部設備、砸碎了門窗，還放火點燃了醫院外的警車。這起混亂也導致3名患者因為未能得到醫生的及時治療而死亡。

/ 07 /

丹麥：警方在反恐行動中拘留約20人

2019年12月11日，丹麥警方開展了大型反恐行動，並逮捕了約20名嫌犯。

/ 08 /

烏克蘭：議會決定將「特殊地位法」延長一年

2019年12月12日，烏克蘭最高議會通過表決，將烏克蘭頓涅茨克州和盧甘斯克州部分地區地方自治特殊制度法（簡稱「特殊地位法」）的期限延長一年，至2020年12月31日。
