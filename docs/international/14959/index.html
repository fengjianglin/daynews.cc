小國越南 為何分出60多個省？可不是只為增加當官數量

2019-12-15 362條評論 2,730次閱讀 1542人點贊

木叔這篇國際評論談談鄰居越南。

一個有趣的話題是——越南面積大約32萬平方公里，比鄰居雲南省的38萬平方公里還要小。但是卻分了59個省，4個直轄市。

所以越南人自己所謂的省，其實就相當於我們的縣那麼大。

比如雲南有16個市州，130個左右的縣區。越南設立了60多個省，顯然平均下來看的話，我們雲南的大約兩個縣區，就相當於他們一個省那麼大。

所以，不要在乎他們把自己的地方行政單位叫什麼，無論是叫省，還是叫縣，或者叫州，地方大小是不變的，就是那麼大點地方。因此名稱對越南地方實際管理而言，沒啥太大的意義。

不過不要嘲笑越南，說人家地方小，還弄出一個一個小的省和地方組織機構，有啥意義？

這可不是只為了增加官員數量，當然也不是說會造成分裂。

其實對於任何國家，無論國土大小，都不可能從中央直接管理到鄉村一級。

一定要在中間設立多層行政管理機構，可能是一層，也可能是2-3層。

這是一個全球治理的問題，也是一國行政區劃和行政管理的客觀現實。

比如以色列——

只有大約2個天津那麼大，才2萬多平方公里，比越南還要小16倍，但是以色列卻設立了幾十個城市。

有些地區在我們看來就是一個小村子，幾千人，但他們也把這裡叫做市。

所以本質上，這種設置的地方行政區劃和越南的很小範圍的省，其實是一個道理。都是要在中央以下有一個中間層級的行政區劃來管理更基層的事務。

基本上從經濟管理和發展的角度，也比較得心應手。

比如越南的這種制度，會讓該國的地方經濟持續釋放活力。

再舉個面積大的國家——俄羅斯。

該國有1700萬平方公里的土地，80多個省一級行政區劃，有的區劃比國際上大多數國家的國土面積都要大。

這就讓俄羅斯聯邦政府管理起來不太方便，因為地方太大，人太少，可能照顧不到就會出問題。

於是普京在2001年想出了一個辦法。在80多個地方行政區劃以上，設立了一個大區制。

也就是把全國80多個地區分成8個區域。每個區域他派遣一名總統特別助理來負責監督本區的情況，直接向他彙報。

這就加強了對地方當局的掌控力度。

任何國家都會設立符合他們國家現實和歷史的行政管理制度，無論國家大小，都如此。

越南的60多個小省是符合越南發展情況的一種手段，俄羅斯弄出8個聯邦大區，也是防止該國地方出現問題的有效方法，各有各的特色和道理。
