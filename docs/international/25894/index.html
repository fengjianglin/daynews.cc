普京向美喊話：不願軍備競賽，他還這樣描述特朗普彈劾案

2019-12-21 1,604條評論 4,988次閱讀 180人點贊

俄羅斯總統弗拉基米爾·普京19日舉行年度記者會，就俄美關係以及與烏克蘭糾紛等話題，一一回應媒體記者提問。

普京 新華社記者 白雪騏 攝

俄美關係 延長軍控條約

談及俄美關係，普京表示不希望兩國展開軍備競賽。

隨著《中導條約》失效，《新削減戰略武器條約》成為俄美之間最後一份主要軍控條約。《新削減戰略武器條約》2010年簽署，2021年2月到期。

俄方先前向美方提議，把《新削減戰略武器條約》有效期延長5年。普京在年度記者會上說，美方迄今沒有回復俄方關於延長條約的提議，「如果沒有《新削減戰略武器條約》，那就完全沒有什麼能遏制全球軍備競賽了。在我看來，那就壞事了」。

按照普京說法，美國制裁措施會影響俄美關係，「我們仍努力工作。如果有必要，我們會作出對等回應」。

美彈劾案 兩黨政治角力

俄方同樣關注美國總統唐納德·特朗普彈劾案。普京19日把這樁彈劾案描述為美國兩黨之間的「政治角力」，預測特朗普不太可能遭彈劾下台。

普京說，特朗普彈劾案將交由參議院審理，「據我所知，共和黨人佔據參議院多數席位……這不過就是美國國內政治角力的延續」。美國民主黨「曾經指責特朗普串通俄羅斯，結果沒查出什麼陰謀，現在又把矛頭指向烏克蘭」。

不過，普京19日表示樂意維護俄美關係，「這與誰入主白宮沒關聯，與哪個政黨掌控美國國會沒關聯」。

俄烏糾紛 堅持明斯克協議

就俄烏糾紛，普京說：「除了明斯克協議，我們沒有其他協議了。澤連斯基有關可能修改明斯克協議的言論……當然引起我的關切。如果重新審議這份協議，局勢可能徹底走入一條死胡同。」

烏克蘭東部頓巴斯地區2014年爆發政府軍和民間武裝間衝突，時任法國總統弗朗索瓦·奧朗德同年創建「諾曼底模式」。2015年2月明斯克峰會上，各方簽署明斯克協議，就停火和烏東地區政治安排等達成一致。不過，各方分歧仍然較大，停火也沒有得到嚴格執行。

本月9日，普京與澤連斯基在「諾曼底模式」機制下首次會晤，俄烏雙方同意年底前互換在押人員，但沒有就烏東部頓巴斯地區狀態等核心議題達成共識。
