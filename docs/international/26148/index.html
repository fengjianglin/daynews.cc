為什麼英國女王在國會開幕式上沒有戴冠冕

2019-12-21 1,415條評論 4,552次閱讀 2539人點贊

英國女王伊麗莎白二世和威爾士親王坐在英國議會開幕前的會議廳里

今天，女王的演講不那麼隆重了，女王坐上了一輛汽車而不是馬車，戴著帽子代替了皇冠。

正如唐寧街所描述的那樣，國家議會的開幕式有意地「減少的儀式元素」。

據議會網站信息，根據10月女王參加議會的討論，由於當前選情的特殊情況，所以議會的開幕式將遠離盛典模式。

除了使用賓利汽車，女王穿了一天的普通套裝和帽子，而不是禮服和冠冕。

查爾斯王子則穿著晨衣套裝參加。菲利普親王已經推掉了大部分的公共事務。

儘管女王穿著日常的衣服而不是正式的盛典隆裝，但這次活動仍保留著許多著名的儀式。

英國女王伊麗莎白二世在倫敦威斯敏斯特宮的上議院開幕前，士兵們遊行通過君主的入口處。

騎兵衛隊穿著生機勃勃的紅色制服，搜查議會的地下室，這是一項儀式任務，可追溯到1605年蓋伊·福克斯試圖摧毀威斯敏斯特宮時。

議會的國家開幕式是新的議會年的開始，總是從女王的演講開始，從中提出了她的政治立法方案。

她在上議院宣讀法案，下議院的成員在下面傾聽。然後，這些政治家們在女王離開後對立案進行辯論。

這個儀式可以追溯到1852年，當時的英國下議院在火災後開始啟動。

此前，女王進行過一次更低調的開幕式，那就是倒霉的特蕾莎·梅在臨時選舉之後，於2017年舉行的國會開放儀式。
