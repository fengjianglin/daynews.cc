俄羅斯四代機首墜，距離中俄邊境僅三百千米

2019-12-24 1,520條評論 2,869次閱讀 1021人點贊

據俄官方報道，一架尚未交付俄羅斯空軍的SU-57戰鬥機在阿穆爾河畔共青城加加林飛機製造廠進行試驗時發生事故。飛機在距離製造廠所屬的jomig機場大約110千米處發生事故，飛機墜機，而飛行員成功逃生，據稱已經被軍方安排的米-8搜救直升機接走。

俄媒報道

目前尚無圖片流出，也不清楚飛機墜毀在機場的什麼位置上，只能以大約110千米作圓猜測飛機的墜毀地。唯一可以確認的是，澤姆吉空軍基地距離中俄邊境也就是300多千米，飛機墜毀位置可能會在距離中俄邊境200-400千米的位置上。這一地區在大興安嶺以北，氣溫比我國東北溫度更低，這個季節氣溫也就是在零下二三十度左右。鑒於飛行員很快被救援直升機接走，說明可能飛機並非突發故障，飛行員還有機會和地面取得聯繫。但在更多消息披露之前，還有一種可能就是測試機被密切關注，一旦發生事故，救援機就已經出發接應。雖然算不得多麼光彩的事情，但是在航空事故處理的問題上，俄羅斯確實是經驗豐富。

基地與製造商工廠

澤姆吉空軍基地是共青城飛機製造廠的測試機場，也是俄羅斯在遠東地區的重要機場。從衛星圖上可以看到多款俄羅斯主力戰機、預警機、運輸機的身影。也是俄羅斯測試蘇-57的重要基地。阿穆爾河畔共青城加加林飛機製造廠更是為國人所熟知。中國空軍第一批重型三代戰機就是這家工廠交付的。儘管在今天說起蘇-27SK，軍迷們往往嘴一撇眼一斜「這什麼破玩意」，但是在上世紀九十年代，乃至二十一世紀最初的5-10年，這些戰機可是絕對的主力。其實就在兩三年前，解放軍空軍引進的蘇-35戰機也是由這家老牌航空製造廠生產的。

阿穆爾廠交付解放軍的蘇-27SK

澤姆吉基地沒有IATA代碼，ICAO代碼為UHKD，當地海拔高度27米，擁有一條跑道，方向角01/19，場2480米。在機場跑道盡頭和中部擁有大量停機位，直接與共青城飛機製造廠相連，方便剛剛生產出的飛機進行測試。

外媒報道

有一些外媒的報道比較奇怪，並沒有稱之為蘇-57，而是提到T-50這個原始代號。目前可以確認蘇-57的各類原型機13架，其中0號機為地面測試機，2號機，也就是KNS機為靜力測試機，T-50-1到50-5是第一批原型機，6-11號機為第二批原型機，其中7號機用於第二批的地面測試。如果媒體不是報道出現了偏差而確實是如此的話，很可能墜機的是T-50的6號機或者8-11號機。

2011年發動機起火的T-50

俄羅斯今年年底可謂流年不利，直升機墜機，航母起火，船塢翻沉還帶走了一艘老潛艇。如果說以上都還是老舊裝備的故障，那麼一架蘇-57墜毀，對於俄羅斯而言，可能還需要全面審視飛機設計、製造中的各類隱患、缺陷。那麼這款剛剛投入批量生產的（2019年7月29日蘇霍伊宣傳冊中提到的信息）被無數人詬病為「偽四代機」的俄羅斯空軍下一代戰機的希望，是否會受到此次墜機影響。這次墜機到底是偶發事件還是設計缺陷，還需要等待進一步調查。但是對俄羅斯而言，這終歸不是個好消息。
