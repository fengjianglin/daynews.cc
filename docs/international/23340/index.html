美國首位月子中心業者被判刑 當天釋放立刻遣返

2019-12-19 3,490條評論 4,761次閱讀 3619人點贊

在橙郡爾灣經營月子中心的中國公民李冬媛（Dongyuan Li）於本月16日正式在加州中區聯邦法院接受判刑。作為這起美國史上最大的月子中心案中首名認罪並被判刑的被告，李冬媛被判入獄10個月，沒收總額價值100萬美元的房產、存款和汽車。

由於李冬媛自今年1月案發被捕後一直在押，她將於出庭當天便被釋放並立刻被遣返回中國。本案檢察官Charles Pell在接受記者採訪時表示，對此判刑「非常失望」，但是尊重法官的判斷。

這起月子中心案件，歷時5年卧底調查取證並審判，引起主流媒體關注，《美國新聞速遞》的記者作為當天到場的唯一華人媒體全程出席了宣判聽證會。此案檢察官當天不斷建議法官考慮嚴判李冬媛，但是法官最終判處她入獄10個月。

李冬媛的家屬離開法庭。龐可陽攝

李冬媛是首位認罪並被判刑的月子中心業者，她於今年9月正式就」串謀移民欺詐和簽證欺詐「的罪名認罪。她在認罪協議中承認，在2013年到2015年3月間，她在爾灣經營一家名為「優孕」（You Win USA Vacation Services）的月子中心，協助懷孕的外國人，通常來自中國，在美國生孩子，幫助他們的孩子獲得美國公民身份。

該公司宣傳網站上稱， 已為超過500名中國客戶提供服務。起訴書中指出，李冬媛和丈夫閆強及商業夥伴陳超在爾灣租下20套公寓。他們向客戶收取4萬美金到8萬美金不等的費用，兩年多里，李冬媛收到來自中國的300萬美元的電匯。

加州中區聯邦法院。龐可陽攝

李冬媛的公司會教導一些客戶，在簽證申請和在美國移民官員面前做虛假陳述。包括錯誤地表達他們將在美國停留僅兩周時間，而實際上，他們計劃留下在美國最多三個月的時間生育。

這起案件中還有快樂寶寶月子中心（Happy Baby Inc）的業者-53歲華人劉維岳（Michael Wei Yueh Liu）與42歲的董晶（Jing Dong）被捕。李冬媛是他們中的首位認罪並被判刑的被告。另外還有16名涉案者被通緝，其中包括4名月子中心業者和12名赴美生子的孕婦及其家屬。他們面臨的指控包括移民欺詐、洗錢和身份盜竊等多項刑事罪名。
