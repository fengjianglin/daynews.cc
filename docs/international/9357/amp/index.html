員工必讀 | 12月5號罷工日～不上班攻略拿走不謝

天天要聞
2019-12-04

巴黎12月2日電（魯佳）法國現在眾所周知，5號是一個很難的日子。不管是僱主還是僱員，面對交通系統全面大罷工也只能適應，畢竟法律上沒有靈丹妙藥。

有時，不上班是必須的，現在的問題是勞資雙方如何就解決方案達成一致。可能不少員工對此還比較困惑，而《巴黎人報》就相關法律條文進行了解讀。

首先，在可以遠程工作的行業和崗位，有集體協議的情況下，遭遇交通困難等情況可以選擇在家工作，無需修改勞動合同；若沒有集體協議，可以在罷工前起草一份勞動合同補充協議，或者直接取得僱主同意。

勞動法律師弗朗索瓦·瓦卡羅（François Vaccaro）解釋說，遠程辦公需要履行一定的程序，比如僱員與僱主之間通過電子郵件或書面交流，員工可發送郵件向僱主申請，若達成一致，僱主應予郵件回復，表明接受僱員在罷工當天遠程工作。若無法達成一致，在擁有員工代表的公司，建議員工先向代表徵求意見。法律上，僱主須保障僱員權利，包括工作時間、離線權及必要情況下遠程辦公權，為僱員的健康和安全提供基本保障。

阿斯塔律師事務所（Astaé）創辦人菲利普·拉維西（Philippe Ravisy）強調，僱員遠程辦公最好先取得公司同意，否則，在未達成一致的情況下，一旦在家發生工作事故，有可能無法獲得安全保險方面的認可。弗朗索瓦·瓦卡羅表示，雖然無法強加於人，但呼籲資方採取務實態度，「我們只能建議僱主，在罷工期間儘可能准許員工採取遠程工作模式。」

當然，並非所有職業和崗位都有遠程工作機會。法國2017年的一項調查顯示，只有16%到20%的員工可遠程辦公。如果電腦是主要工作工具，這比較容易實現，但對於工人、醫護人員、保安人員、實體商業從業者等很多職業來說，就不太現實。這些不能遠程工作的員工，可以申請減薪或休假，僱主可接受也可不接受，但另一方面，僱主也不能強迫員工休假。律師埃里克·羅切布拉夫（Eric Rocheblave）談到，在法律上，員工有義務安排自己到崗上班事宜。

罷工當天，若被困途中導致上班遲到或缺勤，由於不可抗力減少工作時間是可能的，但要視乎公司規定，僱主也可選擇扣發工資。律師建議，萬一上班途中遇到交通阻滯，應盡量搜集證明材料，比如拍下車次取消或延遲的通知。須注意，若員工自行選擇打車上班，僱主沒有義務報銷車資等額外費用。
