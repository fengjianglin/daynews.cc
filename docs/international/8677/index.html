俄羅斯遭禁賽，日本獲利！奧運獎牌榜前3更穩了，迎創歷史良機

2019-12-09 2,219條評論 4,560次閱讀 3039人點贊

北京時間12月9日，根據塔斯社、BBC等多家權威外媒報道，世界反興奮劑組織，將禁止俄羅斯參加未來4年的國際大賽。這個重磅消息，波及的體育領域不計其數，對於無數的運動員都造成了不可估量的影響。作為體育強國，俄羅斯遭到禁賽，同樣會影響到多個項目的格局，近在眼前的2020年東京奧運會的整體形勢，將會發生不小的變化。

根據此前格雷斯諾特體育數據公司，7月公布的2020年東京奧運會的獎牌榜預測，日本被看好最終能夠排在第三，他們領先俄羅斯的優勢微乎其微，如今俄羅斯若是缺席奧運會，日本獎牌榜前三的席位無疑更加穩固，這個此前獎牌數最多排在第五的國家，也有望在本土創造歷史。

根據此前的預測，美國將以51金以及126塊獎牌，排名獎牌榜第一，中國代表團將以38金以及81塊獎牌排在第二，東道主日本則是排在第三，總計67塊獎牌，包括了29枚金牌。

而排在第四的正是俄羅斯，根據這份權威預測，俄羅斯的獎牌數將達到65塊，將是東道主日本的最大威脅。如今俄羅斯遭到禁賽，日本最終的獎牌數極有可能穩定在前三。

縱觀日本參加奧運會的歷史，他們的最佳戰績出現在1964年，彼時他們以16金5銀8銅，排在獎牌榜第三位。雅典奧運會上，日本拿到16金9銅12銀，排在獎牌榜第五位，是總獎牌數最為突出的一屆。

對於日本代表團來說，即將到來的東京奧運會，挑戰美國和中國代表團的可能性不大，但在俄羅斯遭到禁賽之後，他們排名第三也更穩了。而在金牌和獎牌數量上，超越此前的1964年東京奧運會，幾乎是成為定局，他們迎來了在本土創造歷史的良機。
