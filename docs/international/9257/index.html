日本NHK贊：中國姑娘：你們彎下了腰，挺起的卻是中國脊樑

2019-12-05 426條評論 4,454次閱讀 1531人點贊

異國他鄉，即使一個人，也要挺起中國的脊樑。

「我在日本撿垃圾」

去年這個時候，當在日本開麻辣燙小店的中國姑娘aya被日本知名電視台舉著鏡頭追問：如何看待西川口因為中國人的進入，變得髒亂差時，她一時陷入語塞與恐慌。

日本的西川口，本是一個熱鬧非凡的地方，近年來以「小中華街」美譽著稱。

隨著越來越多的中國人到來，更是繁華了當地的街區：中國美食、網吧、撞球、餐飲娛樂一應俱全。

然而繁華背後，卻是越來越嚴重的環境問題。

日本媒體多次探訪，看到的都是垃圾滿地，雜亂堆放的糟心場景。

甚至有日本人用蹩腳的中文貼出告示：這裡不是中國！

如果任由此類報道鋪天蓋地，會影響多少中國人在日本民眾心中的形象？況且間或被報道的：中國姑娘欠租逃跑，房間令人作嘔。個別人沒素質的行為，實力坑害了全體中國同胞。

比如現在中國留學生在日租房，一是特別難，二是需學校擔保，而其他國家的學子根本不需要。

aya自己也切身感受到因為日本人對中國人印象差，而造成的種種不便與煩惱

當初aya租賃店鋪時，僅僅因為她是中國人，很多房東斷然拒絕出租，一點商量的餘地都沒有。更可怕的是，這種不好的印象直接影響了下一代。很多中國孩子在日本的學校慢慢的不喜歡，也不敢說中文，有人譴責孩子忘本，但歸根結底，大人有沒有給孩子在學校說中文的驕傲？

為了改觀國人形象，

aya走上街頭，

做了一件讓所有人大跌眼鏡的事：

撿垃圾。

最初只有aya一個人，但慢慢地，身邊的國人被aya的行為打動，主動參與進來。

他們中有店鋪老闆、留學生、公司職員、媒體人···每個人都抱著相同的信念：提升國人在日本的形象。每月第一個周六，一群戴熊貓口罩的中國人，就會自發準時出現在街頭。

起初一些日本人對走上街頭撿垃圾的中國人持懷疑態度，甚至有人認為：這不過是偶爾的作秀。

但一年多時間堅持下來，連日本人也被我們國人感染，帶著小孩子自發參與到街頭撿垃圾的活動。

雖然語言不通，但一個大拇指已能代表所有。中國人走上街頭，自發撿垃圾的行為，被雅虎、NHK等知名媒體報道後，日本網友更是好評不斷：

「我認為這是非常棒的活動」

還有日本市民特意寄來手寫信和明信片表示感謝。

如今越來越多的中國人，帶著孩子參與到這場撿垃圾的行動中來。「我們雖不能改變世界，卻也讓這個世界因為我們有了一點改變。」
