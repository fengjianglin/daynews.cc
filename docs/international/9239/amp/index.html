11000億項目面臨停擺，日本製造在印度翻車？或只有中國製造能救

天天要聞
2019-12-05

印度火車

過去，以新幹線為代表的日本製造，一度被封奉為神壇，然而近年來「神話」色彩卻褪色，為此日本也想扳回一城，而他們選擇在印度重新崛起，但以「奇葩」著稱的印度人會給日本製造機會么？

據媒體報道，近日，印度馬哈拉施特拉邦首席部長宣布將重新審核孟買-艾哈邁達巴德高鐵項目，而不久前，他還叫停了當地地鐵3號線的車棚項目，很多人擔心印度的這個高鐵項目也可能面臨停擺。印度官員給出的解釋是，他們要對正在進行的開發工作、成本估算、障礙和最後期限進行更新，然後再來決定做什麼。然而對於這個開工兩年多的地鐵項目而言，突然中斷並不是什麼好消息，尤其是對於承包商日本企業而言更不是什麼好事。

日本列車

2015年的時候，日本企業拿下來印度高鐵的修建工程，為此日本也算是付出了血本。該高鐵項目預計耗資11000億盧比（相當於1100億人民幣），而本人承諾81%的資金將由日本方面提供貸款，貸款期限為50年，並且由印度的國營企業來執行該項目。之所以日本願意給印度提供這麼多便利來不遠萬里綁印度修高鐵，因為這在日本人看來，就是一場維護「日本製造」名聲的關鍵之戰。

提起日本的高鐵，當年的新幹線也是風靡一時，成為了「日本製造神話」的代表。然而隨著近年來中國高鐵的崛起，日本的高鐵面臨著前所未有的挑戰。中日高鐵首先在東南亞進行了角逐，包括泰國、印度尼西亞、馬來西亞與新加坡都出現了中國高鐵的競標，而目前中國高鐵拿到三個項目，日本只拿下一個。另外日本在泰國的高鐵項目因為資金沒談攏而擱置，在越南的項目也因為建設周期太長而面臨叫停的風險。

日本飛機

日本高鐵的衰落，就像是日本製造神話也本身的破滅的一個縮影，除了高鐵上目，上個月日本三菱的支線飛機也慘遭國外用戶退貨，日本製造真的陷入了一種困境，而拿下印度的高鐵項目本來是日本製造的一個翻身的機會。如今提起日本在海外的大型項目，也就是南亞大國印度的高鐵項目能提一下，然而如今面臨這樣的情況，莫非「日本製造神話」來到神奇的印度後，最終還是要翻車？歸其原因，實際上還是日本人太相信印度人了。

日本為印度高鐵不僅墊了很多錢，還把施工權都交給印度，而在考察2年後，印度高鐵項目總算動工，而然開工不到一年高鐵項目立馬遇到困難，原因是沿線居民反對建造高鐵，他們並不想因為高鐵而搬家。也就是說這兩年來，印方連居民思想工作都沒做好。而建設2年來，也是各種原因建建停停，到目前已經花費了300多億盧比，原本高鐵計劃2023年通車，如今看來都是個「無底洞」項目。沒想到的是，印度人還首先發難，這讓日本人慾哭無淚。

​中國高鐵

我們也知道印度近年來一直想改善鐵路交通狀況，因此也願意投入精力造高鐵，但從目前情況來看，或許只有中國製造能拯救印度高鐵了。中國不僅高鐵造得好，在海外的鐵路建設也打響了自己的名聲。畢竟我們在教育普及程度更低的非洲都能充分調動當地的積極性把鐵路建好，印度要選中國來幫忙建高鐵哪會是今天這樣的局面。（子文）
