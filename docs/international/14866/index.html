美國女孩在廁所遭男子騷擾，父親一怒之下將對方打死，被判8年

2019-12-16 1,696條評論 3,226次閱讀 2279人點贊

據《每日郵報》2月14日報道，近日，美國亞利桑那州一名父親因將一名圖謀不軌的男子毆打致死而被正式宣判8年監禁，涉事男子試圖闖入這名父親16歲女兒的衛生間，父親迫於無奈才將其打死。

事情發生在2018年8月，40歲的梅爾文·哈里斯(Melvin Harris III) 去接女兒和她的朋友下班，路過亞利桑那州菲尼克斯市一家便利店，朋友們提議停車去買幾瓶水，女生順便去上個廁所。

當他們在商店裡的時候，27歲的流浪漢利昂·列文·阿姆斯特朗(Leon Leevon Armstrong)過來向哈里斯要錢。哈里斯給了他一些零錢，阿姆斯特朗轉身走進衛生間。

哈里斯的女兒說，阿姆斯特「咔嗒」一聲關上了她所在的衛生間隔間的門，並大力搖晃。她先報了警，然後回到父親的車裡，告訴父親發生了什麼事。當他們準備離開的時候，阿姆斯特朗從商店裡出來，女兒告訴父親就是這名男子剛才騷擾了他，於是哈里斯下了車找這名男子理論。

目擊者說，他們看到兩人發生了爭執，哈里斯不斷地對阿姆斯特朗拳打腳踢。哈里斯告訴警方，其實是阿姆斯特朗先動手的。據悉，阿姆斯特朗有著長期犯罪史。

保安人員接到報警後通知了警方，但在警方趕到之前哈里斯已經在停車場失手把阿姆斯特朗打死了。該男子最終死於腦腫脹、鼻骨折和缺氧。

哈里斯最初被控二級謀殺，但他後來接受了認罪協議，指控被減為過失殺人罪，但根據協議，他將服刑8年。

據《刑法》規定，犯過失殺人罪，過失致人死亡的，處三年以上七年以下有期徒刑;情節較輕的，處三年以下有期徒刑。然而哈里斯最初被控的二級謀殺，據美國《刑法》規定，應處20年以上有期徒刑或無期徒刑。

哈里斯被捕後，他的伴侶為他辯護。她說哈里斯其實並沒有做錯什麼，他只是在保護他們的女兒。

伴侶表示：「我並不生他的氣，我不覺得他做錯了什麼，他只是為我們的孩子做了他應該做的事情，你不能說保護自己的孩子是錯誤的。而且換作是我，我也會這麼做的，我一點也不為他的行為感到內疚，我只是覺得那個男的死得太輕鬆了。」
