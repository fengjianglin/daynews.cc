“永生建築”即將面世？荷蘭發明能“自動修復”的建築混凝土

2019-12-19 2,466條評論 4,302次閱讀 2579人點贊

荷情，荷理，荷你一起發現創新

現代建築多以鋼筋混凝土為材料，這些建築的壽命一般僅為50年。即使再堅固的混凝土，在風吹日晒下遲早也會裂開。

而每年光是修復這些裂縫所需要的費用，更是一筆龐大的支出。據不完全統計，僅在歐盟橋樑和隧道的維護費用，每年就高達60億歐元（約463億人民幣）。

而這一切即將改變！荷蘭的微生物學家、發明家亨德里克·瓊克斯（Hendrik Jonkers）發明了一種混凝土。這種生物混凝土最神奇的特點就是：能進行自我修復！

這是怎麼做到的呢？

亨德里克·瓊克斯的腦洞大開，是受到了人體能通過鈣化形成骨細胞的啟發。

他試圖尋找一種可以生活在鹼性環境里，並且還能夠自己製造石灰石的細菌，經過三年多的專研他終於發現了桿菌細菌！桿菌細菌可以在石灰石里生存，並能產生孢子。

「自行修復」是如何進行的呢？

桿菌細菌被塗抹進待修復的裂縫裡，當有水滲透進去時，細菌就會生產出石灰石，因此裂縫將被自動修復好。

使用這種方法修復裂縫，一般3周之內就能完成。

那這種細菌能存活多久呢？

這種細菌可以在混凝土中存活200年，這就意味著，使用了它的建築物經過兩個世紀都不會有裂紋。目前這項技術已經應用在荷蘭的一個老建築上。

這種可以「自行療傷」的混凝土不僅延長了建築物的使用壽命，也解決了歷史建築修復的難題，因此這項發明也獲得了歐洲的創新大獎！

有了這種材料，建築彷彿有了生命。像人體細胞一樣自行進行更新修復。最最重要的是，如果用它來蓋房子，可能200年內都會完好無損，一輩子都不用修。這可為大家省了一大筆錢呀！
