下圖中的這對夫婦你們認識嗎？

圖中男子是全球最年輕的自行創業億萬富豪，著名的社交網路Facebook的創始人——馬克·扎克伯格。右邊的人是他的亞裔妻子，普莉希拉·陳。

過去曾有人質疑：「她嫁給了這樣的老公就是運氣好啊」「難以理解洋人的審美」「怎麼一個大富豪會娶這麼普通的女子」……

很多人都想不明白，地球上最有錢的80後，怎麼就看上了一個皮膚黝黑長相普通的姑娘呢？

而事實卻是，扎克伯格說：「如果說高攀，那隻能是我高攀她。」

01

扎克伯格的寵妻之路

近日，扎克伯格和妻子普莉希拉·陳於家中接受了CBS採訪，這也是公開了他們在加州帕洛阿爾托的家。

自從兩人結婚以來，他們的婚後生活便一直是媒體的焦點。然而，對於身家過千億的夫婦來說，他們的婚後生活並沒有想像中的高調張揚，而是十分溫馨平淡。

今年3月份，在北卡羅萊納農工州立大學的一次演講中，扎克伯格表示：「相比於Facebook，我生活的樂趣和幸福感來源於我的家庭！」

扎克伯格有多寵妻呢？

作為一個丈夫，扎克伯格看妻子的眼神，始終透露出藏不住的愛意，從相識到結婚，一直都是滿眼星星地看著對方。

扎克伯克曾在ins上展示了他的一項「寵妻發明」，他覺得做媽媽很辛苦，這個「睡眠盒子」可以幫助妻子在晚上安然入睡。

扎克伯格每周的工作時間大約是50-60小時，儘管工作再忙，他每年都會在12月份抽出兩周時間和妻子一起去旅行。

兩人還有一個不成文的規定：每周一次的Date Night，這是維繫夫妻感情的小秘密，在約會之夜那晚，不談工作，只享受美好的家庭時光。

有很多人都覺得這是王子愛上了灰姑娘，質疑普莉希拉·陳：她的運氣是不是太好了？

在2012年，Facebook上市的第二天，扎克伯格就宣布要與女友舉辦婚禮，人們都相當好奇，糾結是誰能配得上年輕帥氣多金的扎克伯格呢？

看到普莉希拉·陳後，有不少人都大跌眼鏡。

其實，他們不是什麼灰姑娘遇見王子的故事，而是一個毛頭小子和亞裔學霸互相欣賞的劇情。

02

『她嫁給我，是我賺了』

扎克伯克曾說：「陳嫁給我，是我賺了。她從來沒有從我這裡得到什麼，反而是我一直在受她的影響改變。」

陳出生在80年代的華裔難民家庭，父母做著底層的工作，每天需要工作十幾個小時，在陳還不滿3歲時，就開始幫忙做家務了，比如踩著板凳在水池前洗碗。

雖然出身平凡，但陳始終抱有積極向上、樂觀堅強的心態。在她13歲時，就諮詢老師：「如何才能考上哈佛？」 之後她果然做到了，她持全額獎學金進入哈佛大學生物學系。

在哈佛的那段時間很不容易，陳覺得：「自己不屬於這裡，聰明是我唯一的優勢，但在哈佛，比我聰明優秀的人太多了。」 她最終沒有放棄，做義工時遇到的貧困家庭兒童成為了她的動力：「我必須留下來，留在哈佛，有所作為。這樣我才能幫助更多人打開新世界的大門。」

再後來的故事，就和扎克伯格有關了，學校的派對上，她遇上了瀕臨退學的「學渣」扎克伯格。9年後，志同道合的兩人舉行了婚禮。

他們不僅僅是夫妻，還是最佳搭檔。在扎克伯格創業初期，陳在背後全力支持，離開了自小生活的東岸，跟隨扎克伯格到美國西岸的加州打拚。

陳也有自己的事業，作為一名兒科醫生和陳-扎克伯格基金會的負責人，她常常忙得不可開交。曾經因為在工作上投入過多，超過身體負荷，導致了3次流產。

陳的生活很簡樸，並且不遺餘力地在做慈善事業：「把錢用來做慈善，錢才有了真的價值和意義。」夫妻兩人做的慈善項目，涵蓋了醫療、教育、科學等多重領域。

早在2013年，陳就登上了《慈善紀事報》，成了那年美國最大的慈善家；在2016年又被《Time》選為全球最具影響力100人。

她不僅僅是扎克伯格的太太，更是一個獨立優秀的個體。

普莉希拉·陳給我們的印象是：哈佛學霸、兒科醫生、慈善家……或許在她諸多身份中，最不起眼的那個，才是富豪之妻。

她是一個靠著自己力量實現夢想的難民家庭女孩。看了她的事迹後，都會被她的人格魅力給打動。

沒有良好的原生家庭、相貌平凡、身材普通，但她的美，我們始終能看得見。就算不嫁給扎克伯格，她也光芒萬丈。

03

背後的女人都是相似的

在一段婚姻里，對男人來說，最大的底牌是妻子。

當我們想這些女子到底有什麼魔力，憑啥可以嫁給xxx時，我們可以發現，她們自己本身就是很強大的。

就像股神巴菲特曾說過：「我這一生最重要的投資，不是購買了哪支股票，而是選擇了誰成為我的伴侶。」

百度CEO李彥宏的妻子馬東敏，也是一位強大的「背後的女人。」

馬東敏是天才少女，畢業於中國科技大學少年班，後到國外修得生物學博士。

最初在李彥宏回國創業時，馬東敏選擇留在美國發展，01年百度遇到了創業以來最大的困難。李彥宏給妻子打電話問了一個問題：「如果沒有車和房，你還能接受嗎？」

李彥宏得到的回應是被掛電話，那時心都涼了。沒想到在第二天，就看到連夜趕回來，從天而降的妻子，在寒冬的李彥宏那時感到溫暖無比。

李彥宏說：「在百度創業歷程中，每一步都是她推著我走的。」

還有一位值得一提的是，新東方教育集團創始人俞敏洪的妻子楊桂青。

在新東方成立之初，市場上培訓班已經遍地開花，作為行業新秀的新東方無人問津。俞敏洪開始沒日沒夜地在大街小巷上貼廣告，妻子提出和丈夫一起去貼。

當時是寒冷的冬天，戶外零下十幾度，俞敏洪本想拒絕，但是妻子說：「等貼到一天有一百人報名了，我就回前台。」

愛情的魔力或許在這一刻就起效了，新東方漸漸有了生源。

在放下xxx的妻子的身份後，她們都是一個個優秀獨立的人，真正強大的女性，都有著無與倫比的風采。

與其去糾結富豪們的妻子是不是膚白貌美氣質佳，不如去看到在她們身上獨立豐富的精神世界，以及由內而外的人格魅力，這些才是更重要的。

普莉希拉·陳、馬東敏、楊桂青……這些女人，值得我們尊敬，不是因為她們的老公，而是因為她們本人。
