外媒：杜伊斯堡建巨型集裝箱碼頭 開往中國貨運班列將增至100輛

2019-12-13 3,126條評論 4,036次閱讀 2668人點贊

2018年4月16日在德國西部城市杜伊斯堡DIT貨運場站拍攝的中歐班列集裝箱。（新華社記者 羅歡歡 攝）

國際在線專稿：據法國《歐洲時報》12月11日援引德國西德意志廣播電台報道，德國杜伊斯堡港股份有限公司（Duisburger Hafen AG）近日表示，明年將在杜伊斯堡港新建一座巨型集裝箱碼頭，隨著新碼頭的建成，每周開往中國的貨運列車數量將增至100輛。

杜伊斯堡港股份有限公司稱，這座巨型集裝箱碼頭被看作是「杜伊斯堡門戶碼頭」（Duisburg Gateway Terminal）。該公司表示，同中國加強鐵路運輸合作可以帶來諸多利好，一方面可以節省航空運輸的開支；另一方面，「杜伊斯堡門戶碼頭」投入使用後，預計每年可減少約6000萬噸的二氧化碳排放，這能大大改善德國北威州的二氧化碳排放狀況。

報道指出，新建的巨型集裝箱碼頭佔地面積約為35個足球場大小，耗資1億歐元，將於2022年投入使用。它將在杜伊斯堡港煤炭島的基礎上建成。煤炭島佔地24萬平方米，其中22萬平方米將用於建造碼頭，2萬平方米用於建造儲物倉。整個工程分兩期進行，期間將配備6架龍門起重機、5處裝貨場、3個內航停船泊位以及一個6萬平方米的集裝箱存放區。此外還將鋪設12條用於停放整輛貨運列車的鐵軌，每條鐵軌長730米。

杜伊斯堡港股份有限公司表示，做出新建碼頭這一決定考慮了兩方面因素，一方面是在能源轉型的背景下，港口作為煤炭中轉地的區位功能在急劇下降；另一方面是和中國的貿易往來態勢良好。

杜伊斯堡港在歐洲和中國的貿易往來中具有著重要的區位優勢。目前，平均每周有40輛貨運列車從杜伊斯堡港出發，開往中國。

《歐洲時報》援引德國萊茵郵報網報道指出，杜伊斯堡港越來越重視同中國的商業往來，合作領域涉及貨物運輸、國際運輸服務以及商品交易。兩國重要企業間的戰略合作能夠助力杜伊斯堡成為「歐洲運輸中心」。
