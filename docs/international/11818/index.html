中國人在國外做臘肉被舉報稱其為恐怖片現場網友讓他嘗點就解決了

2019-12-13 692條評論 3,854次閱讀 1157人點贊

中國人吃臘肉已經成為了一種習俗，以前是為了能夠把肉保存的更久，而發明的做法，現在則是成為了很多人心中的情懷。

尤其是在過年的時候，臘肉和香腸是在日常生活中家家戶戶的餐桌上非常常見的一種食物。不僅僅是因為其美味，更重要的是準備臘肉和香腸的時候，才真正的有種要迎接新年的氛圍。

臘肉的味道十分獨特，而且十分百搭，不管是炒菜燜飯，都十分的美味。就算只是在腌制好後，上鍋蒸熟後，直接拌飯吃，味道也是十分美味，肥而不膩，香氣撲鼻。

就算是身在異鄉的遊子也無法割捨，但是對於這種習俗，外國人是不太理解。據報道，曾經有加拿大華人在家製作臘肉和香腸，卻遭鄰居舉報，原因竟然是因為，一些沒見過曬臘肉的小孩子被嚇哭了，稱其為「恐怖片現場」，對此中國人也表示十分哭笑不得了。

當然這樣的做法也讓很多外國人覺得，吃的是不新鮮的肉會得病，因此有很多外國友人會好心的，前來勸解的鄰居，對此中國網友調侃：只要讓他們嘗一點就解決了所有問題。
