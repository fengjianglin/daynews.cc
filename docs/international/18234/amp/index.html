盤點那些說俄語的國家！俄語如今的國際影響力怎麼樣？

天天要聞
1 天 ago

最近，俄羅斯可以說是在媒體和網路上頻頻「露臉」，狂刷了一波存在感……

雖然這些與之相關的消息有好有壞，但總還是站在了輿論的風口浪尖，掀起一陣波瀾。

上周（12月9日），世界反興奮劑機構（World Anti-Doping Agency，簡稱「WADA」）決定禁止俄羅斯在未來四年內參加奧運會和世錦賽等國際重大賽事。

這可謂是體育史上的最重的一次懲罰了，此消息一出，人們大跌眼鏡，紛紛圍觀。

12月12日，因為德國政府將兩名俄外交官驅逐出境，俄羅斯採取報復性措施，驅逐了兩名德國外交官，這一事件讓兩國關係降至冰點……

而前天（12月15日），俄羅斯《觀點報》又報導出「烏克蘭記者要求讓說俄語的孩子『消失』」的文章。

烏克蘭的一個知名電視節目主持人在社交網路上聲稱：「整個基礎教育系統應該旨在確保在烏克蘭沒有一個孩子（沒有一個孩子！）不懂烏克蘭語或者不會運用它……講俄語的孩子應該從我們的國家消失。……只說俄語的人應該作為一個人種消失。」

該主持人的這番言論，明顯是針對普京此前在四國峰會後的發言——」希望烏克蘭全境的俄語居民享有平等的民主權利」。

其實，烏克蘭有38%的公民認為自己是講俄語的。但是，從明年開始，烏克蘭所有的俄語學校都將轉向烏克蘭語教學……

所以，俄語在烏克蘭的前途可以說並不太光明。

哪些國家是說俄語的呢？俄語的國際影響力怎麼樣？

俄語是俄羅斯的官方語言，還是白俄羅斯、哈薩克、吉爾吉斯斯坦，以及聯合國的官方語言之一。

在世界上約有13個說俄語的國家，這些國家都是前蘇聯的國家和現在的獨聯體國家。

俄語在這些國家被作為官方語言或第二語言，不可置疑的是俄語在這些國家都能通用。

在這些國家電視和廣播有俄語節目，學校有俄語教學，年輕人用的是俄羅斯社交網站，用的是俄羅斯的「百度」——YANDEX，俄語在這些國家和地區影響深遠。

根據俄羅斯官方統計，講俄語的人數位列全球第四。

說俄語的國家

  • 俄羅斯
  • 白俄羅斯

俄語是其官方語言

  • 烏克蘭

烏克蘭語和俄語有很多相似處，在之前俄語可通用，但是正如前面所說的，近年來烏克蘭正在大力發展本國語言。

  • 摩爾多瓦

俄語在該國通用，學校有教授俄語，本國語言和羅馬尼亞語一樣，是俄羅斯的勢力範圍。

  • 波羅的海三國：愛沙尼亞、拉脫維亞、立陶宛

居民不說俄語，年輕人大多不懂俄語，少數學校有俄語教學，很多俄羅斯人生活在這三國。

  • 喬治亞

俄語通用，有本國語言，俄語地位有所削弱。

  • 亞美尼亞

俄語通用，有本國語言，和俄羅斯關係較好。

  • 亞塞拜然

居民能通曉俄語，本國語言亞塞拜然語和土耳其語很接近。

  • 哈薩克、吉爾吉斯斯坦

俄語是其官方語言。

  • 烏茲別克、塔吉克、土庫曼

俄語通用。
