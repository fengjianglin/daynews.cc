世界軍力最強的美國最不安全？對外讓馬仔當盾，對內恐嚇國民撈錢

2019-12-25 2,013條評論 4,984次閱讀 2276人點贊

原創不易，請隨手關注！

作者：毅品文團隊大水牛，無授權禁轉！

美國是一個奇怪的國家，軍費最多卻總覺得不夠花，實力越強越覺得不安全。2019年度，美國發布了《世界彈道導彈與巡航導彈威脅評估報告》中，再次對俄羅斯、伊朗等國家的導彈發展品頭論足一番。無論美國如何誇大其詞，刻意渲染，但在我們眼中，這種做法僅僅是一種病態的行為，真正的意圖有2點。

為推進導彈防禦項目開路

（俄羅斯彈道導彈）

（伊朗彈道導彈）

反導系統是美國，蘇聯軍事對抗的結果，是削弱對方進攻能力，保證己方戰略優勢的重要手段。冷戰結束後，美國以受到導彈威脅為理由，並沒有停止建設該系統的步伐。在新的「三位一體」核力量結構中，美國明確提出導彈防禦系統為其重要組成部分。在美國看來導彈防禦是實現眾多軍事戰略目標的支柱之一。其目的在於威懾和安全保證。說白了，就是現在那幾個核俱樂部國家湊在一個小黑屋裡，誰也不敢動，因為大家都是赤身裸體。動一動，就避免不了受傷，會發生世界大戰。現在美國乾脆拋開限制，研製出導彈防禦系統，自己先穿上一身盔甲，那屋子裡其他漢子是不是急了？

這裡還有其他原因，導彈防禦有助於美國對眾多馬仔小弟的安全承諾；同時，這個世界警察到處在地球上殺人放火，他部署了導彈防禦系統就可以讓其他國家難以阻止或者破壞美國的土匪一般的軍事行動。

（薩德反導系統）

（陸基宙斯盾系統）

現在美國到處部署末端高空區域防禦系統和宙斯盾反導系統。比如韓國的薩德系統和日本的陸基宙斯盾系統。而地基中段攔截系統小幅度增長，比如以色列地區部署了一些。從這些國家和地區，你就會發現，都是一些新聞熱點的關注焦點，時時都有點衝突發生。

未來10年內，美國的導彈防禦系統技術會更加成熟。按照其計劃，在亞歐大陸的東西兩側的反導系統全部部署就位，就可以構建一張天網，在這裡他可以罩得住一切，實現對世界能與其抗衡的兩個核大國戰略合圍。目的就是使其他國家的核武器失效，讓美國擁有絕對戰略優勢。流氓打架，除了拿刀砍人，還會拿著一張盾牌，這個盾牌很大，多數時候還是讓那些馬仔小弟去扛，替他擋子彈。但這個其實作用不大，因為反導系統只會激起新一輪的軍備競賽，很快就有新的突防性能的彈道導彈問世，倒霉的只是這些馬仔國家。

為應對軍費削減計劃造勢

錢是一個大問題，沒有錢的日子誰都不好過。尤其是大手大腳花錢過日子的美國兵。蘇聯倒了以後，美軍在世界上找不到勢均力敵的對手，但是軍費開支仍是接近世界軍費總額的50%，超過世界軍費前10位的其他9個國家的總和。即使是曾經很牛的英國人，也只敢在海軍費用上下點賭注，就是保持噸位在兩個海軍強國的總和以上。現在美國人經濟又不是很好的情況下，又碰上貿易戰，國會那些老爺們會要求美軍5年內削減費用2000億左右美金。

但實際上，美國人通過輿論造勢，誇大他國的導彈威脅論。就是恐嚇那些國會裡的白髮蒼蒼的老頭老太，如果你不給我錢，我沒工資發給那些軍人，到時候你家的小孫子和孫女沒人保護，一發俄羅斯的核彈直接讓你家的小孩在後花園玩耍的時候直接化為水蒸氣。就是一張流氓赤裸裸的要保護費的臉，當然這些話還是讓美國防長埃斯珀去委婉的對國會去講。

（國防部長埃斯珀，他負責要保護費）

（到了國會，就要用三寸不爛之舌向老頭老太恐嚇要保護費）

即使要到了錢以後，這些款多數沒有花在防禦性項目上。更多時候，美國在一些優先項目上花錢，比如導彈方面，有超音速隱身巡航導彈，和最近試射的中程導彈。他們在經費節省的情況下，要重點發展尖端科技，確保軍事技術領域的絕對領先。

（從國會要了錢，多數都是拿來造這種進攻性的F35這種玩意，什麼防禦？美軍是進攻型的）

作為世界頭號軍事強國的美國，炒作他國威脅論已經不是什麼新鮮事了。特別是近年來，美國的國際地位和影響力有所下降，其他大國的地位相對上升，美國對俄羅斯和東亞地區等戒備心理開始增強。美國防長在國會聽證會報告上直接聲稱俄羅斯擁有100枚以上能直接攻擊美國本土的導彈，美軍有責任也必須做好準備，裝備有防禦系統應對這些潛在敵人。

但實際上，美軍擁有世界上最大的核武庫，技術和性能上領先全球。他的核彈能打到亞洲地區就有超過1000枚。一邊在對國會和民眾恐嚇狼來了，一邊在研製自己的劃時代軍事裝備。從F35到全球即時打擊系統，美軍要的不是一兩件應對威脅的裝備，而是未來威脅別人的優勢地位。參考資料：《美國熱炒導彈威脅論》有什麼意見，歡迎在下方留言討論！（請支持毅品文團隊的各種原創文章及實體書，獨立專業有種有料）
