英國新生兒兩次被醫生診斷為普通感冒，一周後卻突然死亡

2019-12-23 73條評論 1,322次閱讀 166人點贊

新生兒通常是很脆弱的，有時候一點小小的疾病就會對他們產生很大的影響，如果不能及時對症下藥，後果可能是無法想像的。近日，在英國，就有一位五周大的男孩出現了呼吸困難等癥狀，兩次去到醫院後醫生都說只是簡單的感冒，但男孩一周後卻最終死於膿毒症。

Luchii Gavilescu

據《每日星報》12月21日報道，Luchii Gavilescu出生於10月26日，他的父親Vlado Gavrilescu和母親Laura Cooke對於他的到來都非常高興。但Luchii當5周大時，Laura注意到他出現了呼吸困難的情況，而且身上起了很多疹子，但因當地的醫生無法立刻預約，所以Laura於11月29日帶他去了伊麗莎白女王醫院（QEQM），醫生診斷Luchii只是患有普通的支氣管炎，而且告訴了Laura沒有什麼好擔心的，也沒有必要驗血。

Luchii Gavilescu和父母

但隨後Luchii的病情越來越嚴重，Laura再次帶他去了急症，醫生卻再一次告訴她，她的孩子只是感冒了，她只能開了點葯帶著Luchii回家。然而，Luchii在接下來的幾天里時而清醒，時而昏迷，還伴有尿色發黑等癥狀。

12月6日，Laura帶Luchii去探望她的母親，她的母親也認為孩子呼吸很不正常，於是他們立刻叫了一輛救護車，Luchii於中午就被送到了伊麗莎白女王醫院，但是直到晚上6點兒童病房裡都沒有空位給Luchii。在等待的過程中，醫護人員給Luchii驗了血，但結果讓醫護人員變得越來越恐慌，隨後一位兒科醫生告訴Laura，她的兒子需要緊急治療膿毒症。

Laura在接受《每日郵報》採訪時表示：「我當時僵住了，眼淚立刻就流了下來而且很慌張，不知道說什麼也不知道怎麼做。我們在病房時，一位會診醫生看了一眼Luchii，就知道他病得很重，這位醫生還對另一個兒科醫生說『為什麼這個孩子沒有早一點來接受治療？』。」

當Luchii的其他家人趕到醫院時，他已經被緊急送往手術室接受治療，在那裡他暫時停止了呼吸，醫生們不得不對他進行心肺復甦。之後有一段時間，Luchii似乎穩定下來了，醫生還告訴Laura，她的兒子會康復的。

但不久之後，另一位醫生來告訴她，Luchii的心臟已經停止跳動，醫護人員正試圖挽救他。

45分鐘後，這家人收到了最壞的消息——Luchii已經死亡。

Luchii Gavilescu的家人

Laura說:「當時一個醫生走進來，他說的第一句話就是『對不起』，然後我癱倒在了地板上大哭，我們都不敢相信結果會是這樣。」

Laura把兒子的死歸咎於醫院的無能，她表示：「這個醫院對待Luchii的方式是令人震驚的，我們每個人都希望能把孩子的健康託付給負責任和有能力的醫生，這一切本來都是可以避免的，我們會堅持到最後，為Luchii討回公道，我們也不希望這種悲劇再發生在其他家庭。」

目前，相關組織已經開始對Luchii的死亡進行調查。

《每日星報》已經聯繫了這家醫院和Laura，並請他們對此事發表評論。
