洛杉磯丨住山頂房車民宿，與好萊塢巨星為鄰

2019-12-08 76條評論 2,657次閱讀 747人點贊

洛杉磯一個佔地16萬㎡的山頭上，有一家從復古房車改來的民宿，客人可以在山頂露營、賞日落、看星空，還能俯瞰尼古拉斯·凱奇的家！其擁有者是一對好萊塢名流夫婦，5年來，他們接待了7000多個客人，還有很多明星前來。

這家房車民宿，鄰居都是大明星

這間民宿（AIRSTREAM VINTAGE TRAILER ADVENTURE）的主人是一對神奇的夫婦。
在美國洛杉磯附近的一座山上，
大明星尼古拉斯·凱奇的私宅附近，
住著一對神奇的夫婦。
太太是好萊塢獨立製片人，
她出品的真人秀《狗語者》
8年出了9季，一共在200個國家播出，
光是在美國，每周就有1000萬觀眾等著追劇，
還獲得過「艾美獎」提名。
丈夫做過機修工、舞台設計等8種工作，
蓋房子、改房車、養馬、養蜜蜂……
2012年，夫妻倆為了慶祝第27個結婚紀念日，
把復古房車搬到了自家山頂，
喝紅酒、吃大餐、看日出……
比年輕人還會玩浪漫！
之後還不過癮，兩人索性把房車留在山頂，
變成了房車民宿，沒想到竟然異常火爆，
因為這裡充滿著愛的回憶，
很多男客人，都選擇在這裡求婚，成功率超高！

他們把其中的2輛拿出來，作為民宿，
只有2間房，最多可以住4個人，
不僅要自己做飯，水電還得省著用，
但5年卻住了7000人，幾乎每天都滿房。
想要訂到房間，你至少需要提前2個月！
《洛杉磯時報》的編輯悄悄來住過之後，
寫了一篇報道，把Murray的房車民宿，
列為洛杉磯最值得探訪的7個地方之一。
《辛普森一家》的主要製作人Ricky來住過之後，
在民宿的留言本上寫了感謝信……

Murray和Kay在結婚前，
做了10年的好朋友，
直到女方離婚，
他們的友誼才變成了愛情。
1994年，兩人買下了一片16萬平方米的山頭，
這裡距離洛杉磯45分鐘車程。
Murray不僅自己動手蓋房子，
還把自己喜歡的復古房車搬了過來。

5年前的10月，是兩人第27個結婚紀念日，
Murray想給太太一個驚喜，
結果在網上找來找去，
都沒找到合適的地方，
Murray突然想到，
其實自家山頭就有一片好地方。
Murray家有一個山頂平台，
可以360度看到漂亮的聖莫尼卡山脈，
以及遠處的太平洋，
還能隱約看到好萊塢大明星尼古拉斯·凱奇的私宅。

於是，他瞞著太太，
悄悄把房車搬到了山頂，
還在房車前撐了把遮陽傘，
擺了躺椅和紅酒，
第27個結婚紀念日，他們就在這裡度過。

洛杉磯最值得探訪的7個地方之一

浪漫的結婚紀念日過後，
Murray並沒有急著把房車從山頂上搬下來，
他想，這麼酷的1957年款復古房車，
再加上全世界獨一無二的風景，
幹嘛不把房車放到Airbnb愛彼迎上面去，
讓更多人能來體驗呢？

那個時候正是美國加州寒冷的冬天，
Murray原本以為，
大家不會對戶外的民宿有興趣，
結果僅僅過了幾天，
竟然就有客人下單了……

房車裡設備齊全，
客人可以在裡面自己做飯，
細心的Murray還為小朋友準備了桌游，
一家三口來這裡住，完全不會覺得無聊。

不過，房車維護起來很辛苦，
Murray和Kay沒有僱傭員工，
所有的工作都親力親為，
他們親自從山下搬水上來，
客人走後還要清理洗手間，
房車靠太陽能發電，
來住的客人一般都很自覺，
節約用水、用電。

5年來，Murray夫婦一共接待了7000位客人，
房車幾乎每天都是滿房，
夫妻倆簡直忙到跳腳！
但他們不但沒有抱怨，
還很開心能認識從世界各地來的客人。

《洛杉磯時報》的編輯悄悄來住過之後，
寫了一篇報道，把Murray的房車民宿，
列為洛杉磯最值得探訪的7個地方之一。

《辛普森一家》的主要製作人Ricky來住過之後，
在Murray的留言本上寫了感謝信，
順便還畫了一幅辛普森的小像，
Murray非常喜歡，沒事就拿出來看看。

還有無數男客人，
想在Murray的房車前向女友求婚。
有一次，一對情侶請Murray給他們在房車前拍照留念，
Murray拍到一半，照片里的人突然「沒了」，
原來這位男士居然單膝下跪，開始求婚了！

有一回，Kay還碰到了一位粉絲：
有一個女孩子獨自來房車民宿住，
她看起來心情不好，Kay就陪她聊天，
才知道她的媽媽剛剛過世。
她說，以前一直和媽媽坐在沙發上看《狗語者》，
Kay說，我就是這個片子的製片人！
女孩子既開心又感動，
覺得肯定是媽媽冥冥之中的安排！

打理房車民宿之餘，
Murray還要照顧十幾隻小動物，
他家有2隻狗、2隻貓、1隻鸚鵡、2匹馬，
還有若干只小雞，
這些都是Murray救助來的動物。

偶爾，Murray還會去照顧蜂箱，
這裡的氣候能生產出世界上最好的蜂蜜之一，
Murray閑來無事，竟然自學養蜂術，
還小有成就，
去年冬天，就收穫了56公斤的蜂蜜，
作為聖誕禮物送給了親朋好友，大受歡迎！

我們問Murray，要開民宿、要喂馬、還要養蜂，
日子過得累不累？有沒有想過退休？
他說，完全不累，更沒想過退休。
美國人一般65歲就退休了，
而Murray今年已經71歲了，Kay比丈夫還大幾歲。

Murray和Kay

只要身體允許，Murray想把房車民宿一直開下去，
讓越來越多喜歡房車的人，有機會住進來。
但說起因為忙碌，不能常常陪妻子出門旅遊的事，
Murray還是有點內疚。
不過最近，他在忙著修理自己的房車，
他計劃著，今年聖誕節的時候，
可以帶著Kay一起去墨西哥玩，彌補遺憾！
