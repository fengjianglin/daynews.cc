初代樂高分揀機：小哥的靈感來源

Daniel在視頻中提到，2011年一位來自日本的樂高愛好者Akiyuki，就曾發明了一款樂高分揀機，那是他最初的靈感來源。當時的分揀機雖然沒有那麼華麗，但也算很厲害了。

‍

Akiyuki當時的資料庫圖像較少，而且識別和分揀的速度也比較慢。

原視頻：

如今，Daniel的這台機器由10000多塊樂高零件組成，帶有6個樂高電機和9個伺服電機，為傳送帶和攪拌器提供動力。

能夠將近3000種樂高零件分揀到18個不同的盒子，每2秒就可以分揀一個零件，這效率可以的！

怪不得Daniel稱他的這款分揀機為世界上「第一台樂高通用分揀機」，是因為它使用了最先進的人工智慧技術，通過卷積神經網路能夠識別和分類任何已經生產出的樂高部件，甚至是機器從未見過的零件。

Daniel也計劃在將來公布代碼。

How it works？

在過去的兩年中，Daniel一直在設計和製造可以識別並分類樂高零件的機器，用了超過10000塊樂高零件，終於大功告成！藉助計算機視覺演算法，可以識別分揀出任何樂高零件。

小哥很有心的把分揀機的實現過程製作成了視頻，快來和文摘菌一起看看吧~

原視頻：

分揀機的核心部分是「Capture Unit」，一個擁有傳送帶、照相機和照明燈的小空間。看起來好像沒什麼大不了，實際上要想讓它順利完成工作，有很多需要注意的事情。

相機會拍攝沿著傳送帶輸送的樂高零件，然後將照片上傳到運行著AI演算法的伺服器，從數千個可能的樂高元素中識別零件。

核心問題是需要將傳送帶的實時視頻流，轉換成神經網路可以識別的各個部分的獨立圖像。

最終目標：從原始視頻（左）到大小均勻的圖像（右），然後發送到神經網路。（動圖的速度比實時視頻慢了50%左右）

這裡用到了目標檢測，即檢測目標的存在、位置和大小，以便零件在每一幀都能生成邊界框。表面上看起來很簡單，實際上卻困難重重。

要想順利實現零件的識別與分揀，還有許多需要注意的地方，比如相機的位置和角度；光源要保證充足；另外零件還不能與傳送帶的顏色一樣，否則無法扣除背景，小哥選擇的傳送帶顏色是淡粉色。

在訓練神經網路上也需要耗費大量時間和精力，什麼是神經網路呢？可以把它看作一個虛擬的大腦，它可以通過接受輸入並轉換成相應的輸出來完成特定任務。

一般來說，輸入給神經網路的數據越多，它完成任務的能力就越強。樂高分揀機的神經網路是通過輸入樂高零件的圖像，輸出相應零件的編號。

由於樂高的零部件有成千上百種類型，顏色多樣，並且從不同角度看形狀也不一樣。因此，收集正確的訓練數據集是整個工作中最難的一部分。小哥在分揀機運行幾天後捕獲了30萬張圖像，這是其中的一部分。

所有樂高愛好者都頭疼的一件事

沒錯，就是分類整理！

Daniel有了這個歸納整理小能手，再也不用對著那一桶桶零件發愁了！不過咱也只有眼饞的份啊，想著那些年被堆到桶里的樂高，還是只得長嘆一句，唉~

論樂高到目前為止究竟有多少種零件，還真的找不到確切的數字，因為分類方式不一樣，統計出來的種類自然也就不一樣，而且似乎每天都有新的樂高零件誕生…

樂高愛好者們只得互相交流，探討整理樂高零件的心得體會，文摘菌看了一下豆瓣和知乎上廣大網友的熱心分享，大致的分類方式有按大小分、按顏色分、按系列分，甚至還有按照零件上的顆粒數量分。裝零件的器皿多是一些收納盒，整理箱。
