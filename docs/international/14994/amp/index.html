日本巨頭將投資12億在越南建廠，原崑山、重慶和東莞工廠或受影響

天天要聞
3 天 ago

據國際商業諮詢機構AODOK的消息，日本電產（Nidec尼得科）旗下的超眾科技計劃於2020年開始，向越南投資1.72億美元（約合人民幣12億）在首都河內市建造新工廠。

超眾科技於1973年在中國台灣成立，是全球領先的電子產品熱管理解決方案提供商，其產品散熱模組廣泛應用於智能手機、電腦、汽車電子，5G基站，自動化設備和人工智慧設備等。

2018年，日本電產（Nidec尼得科）集團斥資約45億新台幣，收購了超眾科技48%的股份，成為第一大股東和實際控制人。從股權結構來說，超眾科技屬於日本電產（Nidec尼得科）的旗下企業。

日本電產（Nidec尼得科）成立於1973年，是全球首屈一指的精密馬達製造商，目前全球擁有10萬名員工，在2018財年，公司的營業額高達1.475萬億日元（約合946億人民幣）。

1989年，日本電產就進駐了中國市場，目前在中國有近百家工廠和銷售公司，還有多個通訊、產業自動化、物流和5G技術的研發中心，擁有員工約4萬名。日本電產徹底貫徹中國速度和中國價格策略，爭取要在2020年的中國市場實現500億人民幣的銷售目標。

日本電產（Nidec尼得科），這家日企在中國普通消費者心中沒有絲毫的存在感。不過它的馬達產品廣泛應用於手機、電腦、家電、汽車、IoT和機器人等。

在蘋果、三星、華為、松下、夏普、大金、東芝、日立、三菱、LG、伊萊克斯、海爾、海信、美的、格力和長虹等這些大家熟悉的品牌中，基本都使用了日本電產的馬達零部件，另外中國有超過70%的汽車天窗也使用了日本電產的馬達。

日本電產（Nidec尼得科）是日本製造業工匠精神的完美體現。

日本電產（Nidec尼得科）的創始人永守重信大學讀的是電機工程專業，畢業後曾在兩家日本知名企業擔任過工程師，1973年7月23日，年僅28歲永守重信帶領三位朋友，在京都市的一間陋室創辦了日本電產株式會社，當時主要生產精密小型交流電AC馬達產品。

創業之初困難重重，公司也幾度走到了破產的邊緣，最慘的時候永守重信抵押了自己的全部身家（甚至包括人壽保單），但「要成為世界第一」的雄心壯志激勵著永守重信咬牙堅持著。

日本電產（Nidec尼得科）創業時的陋室！

經過近十年的堅守後，上個世紀80年代美國PC市場逐漸起步，日本電產開始專註於研發生產電腦硬碟驅動器馬達，憑藉過硬的產品質量和具有競爭力的價格，日本電產迅速打開了市場局面。

1984年，日本電產收購美國托林公司的軸流風扇部門，這是日本電產的首次併購，通過這次併購完成了美國的市場布局後，此後幾年間又分別在日本和全球各地設立了新工廠，讓日本電產一躍成為全球的知名企業。

如今的日本電產（Nidec尼得科）已從單一的馬達生產商，轉變為面向「轉動體與移動體」的綜合機電服務商，並成為了這一細分領域當之無愧的世界第一。

在《福布斯2019年日本富豪排行榜》上，日本電產（Nidec尼得科）創始人永守重信以45億美元的財富凈值名列第9位。

現在全球都在爭奪5G的制高點，日本電產（Nidec尼得科）旗下的超眾科技大手筆投資越南，除了是產能的轉移外，還涉及到新技術的應用！

在5G技術加速普及的過程中，從基站系統、通信網路設備到智能手機等終端，都對散熱有了更高的要求。5G高速率、大帶寬和低延時的特徵決定了5G設備里的核心部件需要更大的運算量，同時也會產生更多的熱量，如果沒有合適的外部手段輔助散熱，就會影響到運轉速度甚至死機。

也就是說5G產品的散熱如何，將直接決定產品的性能和檔次。

單從手機來說，現在智能手機用到散熱片的比重只有5%，進入5G之後，3年後這個比重可能會提高到50%， 現在蘋果、華為、小米、Oppo、Vivo、三星和LG等都在積極的尋找新的手機散熱方案。

超眾科技目前的生產基地除了有中國台灣的總部外，其它的三個分布在中國大陸的江蘇崑山、重慶市和廣東東莞。

這次超眾科技重點投資布局越南，除了中國大陸的產能面臨轉移外，還可能會把全球領先的散熱新技術研發和新產品製造放在越南的基地。
