紐西蘭火山噴發已致18死，搜救仍在繼續

2019-12-15 1,753條評論 3,290次閱讀 1603人點贊

崔舒飛

搜救隊潛水搜尋未果(圖片來源：法新社)

【環球網報道 記者 崔舒飛】據法新社15日最新消息，紐西蘭懷特島火山噴發已造成18人死亡，包括兩名遺體未能回收的遇難者。

法新社報道截圖

報道稱，15日搜救隊分別於上午和下午進行了一次陸地搜救和一次海中搜救，但均未能找到失蹤的二人。警方稱，事發當天目擊者看到遇難者遺體的水域，水流是一路流向大海的。兩名失蹤者的遺體都「極有可能」還在水裡，且搜救隊已經確認堤岸附近水域沒有兩人的遺體。警方表示，儘管搜救隊「感到沮喪」，但他們仍未放棄希望。

報道稱，9日火山噴發時，島上共有47人。隨著一名澳大利亞傷者在醫院不治身亡，遇難者人數已上升至18人。14日，紐西蘭警方開始正式公布遇難者的姓名和國籍。在事故中受傷的兩名中國公民都已恢復意識，情況正在好轉。紐西蘭總理阿德恩14日在一份聲明中宣布，紐西蘭全國將於當地時間16日下午2時為遇難者默哀一分鐘。目前，在26名倖存者中，有20人處於「情況危急」的狀態，正在紐西蘭和澳大利亞的醫院接受治療。

懷特島位於紐西蘭北島豐盛灣以北約50公里，是一座活火山島。毛利人稱這座島為「神奇的火山」。該島火山長期活躍，經常噴發濃煙和氣體，地表溫度高，形成了奇特景觀，是紐西蘭一處著名旅遊景點，人們可以乘船或坐直升機遊覽該島。在安全條件許可的情況下，遊客甚至可以進入主火山口。
