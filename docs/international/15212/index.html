吹響勝利號角！民眾出城夾道歡迎俄軍，曾經邪惡國度首都被解放

2019-12-13 173條評論 1,162次閱讀 189人點贊

據俄羅斯媒體12月9日報道，俄羅斯軍隊已經進入了敘利亞城市拉卡，並進行人道主義救援。這是迄今為止最明顯的例子，證明俄羅斯吹響了在敘利亞的勝利號角，填補了美國總統決定從敘利亞撤軍而造成的真空。

拉卡是敘利亞的戰略重鎮，這座城市曾有多年時間作為邪惡的”IS國”的首都。極端主義組織”IS”趁著敘利亞內戰的混亂局勢，於2014年初宣布所謂”建國”，將拉卡設為其首都，其”國家元首”就是不久前被美軍特種部隊擊斃的巴格達迪。這個邪惡的國家由於搶佔了敘利亞的絕大部分油田，一時之間財大氣粗，竟像模像樣的開始訓練”行政官員骨幹”討論如何組織教育、天然資源、工業、外交關係、公關等領域的政府部門。

但是這個所謂的”國家”完全沒有脫離極端主義本色。在其掌控的地區，”IS”禁止平民吸煙，懲罰不戴面紗的婦女，而且還實施最嚴厲的懲罰行為，比如砍頭等。該組織還在代爾祖爾草率地處死了三名阿拉維派人士，掌管敘利亞政府的阿薩德家族就屬於這個派系。此外，”IS”甚至還開槍射殺那些抗議該組織殘暴行為的敘利亞平民。”IS”還不斷利用現代社交手段擴大其全球影響力。

如此邪惡的”國家”，國際社會理應一致團結起來將其消滅。2014年9月，美國就組建了包括英國、法國等54個國家和北約以及阿盟等在內的國際聯盟以打擊”IS”。然而，打擊大本營位於敘利亞境內的”IS”同時，美國以及西方國家竟然絲毫不考慮與敘利亞政府軍及其盟友俄羅斯軍隊聯手。更有甚者，英國情報機構所培養的”白頭盔”還不時製造”化武疑雲”，讓美國和西方國家有理由在反對派形勢不利的時候空襲敘利亞政府軍。

西方這種雙重標準、三心二意的態度，使得對抗”IS”的戰事曠日持久，將敘利亞大片地區化為了廢墟，人民遭受無盡痛苦。終於，去年年末敘利亞政府宣布在該國南方的所有”IS”佔領區域已經被徹底解放。至於拉卡，很快也被美軍及其庫爾德武裝盟友攻克，宣告這個邪惡組織至少在表面上走向覆滅。但如今，隨著美軍的撤退，俄軍給所有這些戰亂之地帶來了真正的解放。俄軍進入拉卡地區時受到了民眾出城夾道歡迎，士兵們與兒童握手，並向居民分發了2000袋食品，俄羅斯軍醫也準備向當地居民提供高水平的醫療服務。
