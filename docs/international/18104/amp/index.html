因礦難造成41人死亡，美國五巨頭遭起訴！中國無人礦車已投入商用

天天要聞
2019-12-18

科學技術的進步，也伴隨著各種礦產資源的開掘，但隨之帶來一個沉重的話題——礦難。儘管採礦公司會做好安全措施，挖礦工人們在礦產資源的挖掘過程中還是可能會伴隨著不可預知的危險，近期向來慣於盤剝童工的剛果（金）鈷礦就出現塌方，造成41人死亡。

因鈷礦難美國五巨頭遭國際人權組織起訴

對此消息，國際人權組織代表14個剛果（金）家庭在美國正式提起訴訟，訴訟名單中美國五巨頭蘋果、谷歌、微軟、特斯拉、戴爾赫然在列。理由為，這些公司明知道剛果（金）的鈷礦公司長期盤剝童工，並未監管好供應鏈，還助長了鈷的供應鏈，從剝削中獲利，促使剛果（金）的兒童在危險的鈷礦下工作，甚至因此死亡。

據悉，鈷作為一種金屬材料，主要用於鋰電池的生產，應用於新能源汽車、智能手機、筆記本電腦等高科技產品。而剛果（金）鈷礦資源十分豐富，供應量可達全球60%的需求。然而該國的鈷礦開採通常有非法採礦、盤剝童工等問題。

事實上，早在2017年美國媒體就曝光剛果（金）童工採礦的惡劣環境，蘋果回應稱暫時停止採購剛果（金）鈷礦，要求剛果（金）鈷礦公司整頓採礦環境，並宣稱將與剛果（金）當地機構合作，改善工人工作環境，禁止僱傭童工。

此外，蘋果、谷歌母公司Alphabet、戴爾、微軟和特斯拉都曾公開要求在其供應鏈中禁止使用童工，甚至有公司政策頒布，然而，如今看來，為了持續獲得廉價鈷礦，他們似乎選擇了不徹底實施該政策。對此，美國五巨頭暫無回應。

中國無人礦車橫空出世，市場高達6000億

而中國煤炭資源十分豐富，採礦過程中也曾發生事故。為進一步改善煤礦工人的工作環境，提高煤礦採礦現場的安全指數，中國企業也開始搶灘無人駕駛，國家政策也在鼓勵礦山向無人化、信息化、數字化方向發展。

2019年1月，中國自研的無人駕駛礦車正式橫空出世，開啟中國的無人礦車商用元年。在內蒙古的寶利煤炭礦區，9月份，就有三輛無人駕駛翻斗車在煤礦現場試行作業，在背後操縱這三台翻斗車的，是來自慧拓智能機器有限公司的「愚公」智慧礦山無人化運輸系統，具體包括礦車自動駕駛系統、機群調度系統、遠程管控系統等。

據悉，礦山的無人駕駛市場，主要分為礦區運輸和礦用車的無人化改裝。實施無人駕駛，通過系統操控後，不僅大大提高了礦區作業效率，還大大降低了用人成本，同時還緩解了礦區司機招工難的問題，更大大降低了礦區的安全隱患。

據慧拓智能CEO介紹，目前中國礦區運輸類無人駕駛、礦車無人化的前後端改裝各有3000多億元的市場規模，預計未來市場還會有爆髮式增長。而中國中車在這方面涉入較早，早在2017年5月就生產出240噸級別礦用自卸車，受到南美市場歡迎，近日更是獲得澳大利亞必和必拓智利分公司2019-2025年採購90台該礦車的訂單。
