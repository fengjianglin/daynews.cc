印尼發布新規定：外國互聯網公司需要納稅

2019-12-09 769條評論 4,372次閱讀 268人點贊

根據周3公布的一項新規定，印度尼西亞政府已表示，在印度尼西亞蓬勃發展的互聯網經濟中佔有重要地位的海外公司，必須在本國任命一名代表，並繳納所有適用的稅款。

根據谷歌、淡馬錫和貝恩公司10月份的報告，印度尼西亞的互聯網經濟是東南亞地區最大、增長最快的互聯網經濟，有望在2025年突破1300億美元大關。

這項新規定於11月25日通過後立即生效，該規定要求在印度尼西亞積極進行電子商品或服務貿易的外國公司被視為等同於在印度尼西亞實際存在，因此必須遵守所有新規定。

新規定適用於所有符合特定標準的公司，包括從印度尼西亞產生大量流量或達到一定數量的交易額或交易量的公司。這些公司將被要求在印度尼西亞管轄範圍內任命一名代表，代表其行事。

印度尼西亞經濟部部長Susiwijono Mugiharso表示：「這樣做是為了創造一個公平的競爭環境。一直以來，我們只是一個市場，但是現在，他們正在積極利用我們的市場，那麼他們也必須在這裡出現。」

印度尼西亞財政部長Sri Mulyani Indrawati於11月22日表示，她將在本月向國會提交一項法案，該法案將允許當局將具有重要經濟地位的任何大型互聯網公司視為印度尼西亞納稅居民，必須繳納增值稅和公司稅。

印度尼西亞電子商務協會主席Ignatius Untung表示，對該措施表示歡迎，稱此舉對當地企業公平，舊的稅收規定在當前條件下已不再適用。

新法規還涵蓋了針對本地在線企業的更廣泛規則，例如要求他們保護客戶數據，提供客戶服務並出於統計目的與有關部門共享數據。

由於出口收入下降，印度尼西亞政府可能難以實現今年的稅收目標。
