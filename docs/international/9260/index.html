漲知識！12月外貿月曆，全球買家節日一覽

2019-12-05 790條評論 2,114次閱讀 532人點贊

12月1日中非國慶日

中非國慶日，中非政府於1958年規定，每年的12月1日為中非國慶日。

12月2日阿聯酋國慶日

1971年3月1日，英國宣布同波斯灣各酋長國簽訂的條約於年底終止，同年12月2日，阿拉伯聯合大公國宣告成立，由阿布扎比、迪拜、沙迦、富查伊拉、烏姆蓋萬和阿治曼6個酋長國組成聯邦國家。1972年2月10日，哈伊馬角加入聯邦。

12月5日泰國國王日

國王在泰國享有至高無上的地位，所以，泰國的國慶日也定在國王普密蓬·阿杜德的生日12月5日，同時也是泰國的父親節。每當國王的生日來臨，曼谷的大街小巷都懸掛國王普密蓬·阿杜德和王后詩麗吉的肖像。同時，身著禮服的泰國士兵將在曼谷市銅馬廣場參加盛大的閱兵式。

12月6日芬蘭獨立日

1917年12月6日芬蘭宣告獨立， 成為主權國家。每年芬蘭獨立紀念日最大的活動焦點就是芬蘭總統府舉行的正式慶祝宴會。據說總統會邀請這一年來對芬蘭社會有突出貢獻的人士出席宴會，在電視上可以看到來賓排隊入場一一與總統夫婦握手。

12月12日肯亞獨立日

1890年，英、德瓜分東非，肯亞被劃歸英國，英政府於1895年宣布肯為其「東非保護地」，1920年改為其殖民地 。1960年3月，肯亞非洲民族聯盟（簡稱「肯盟」）和肯亞非洲民主聯盟成立。1962年2月倫敦制憲會議決定由上述兩黨組成聯合政府。1963年5月肯舉行大選，肯盟獲勝。同年6月1日成立自治政府，12月12日宣告獨立。

12月18日卡達國慶日

在每年的12月18日，卡達都會舉行大型活動以慶祝國慶日，紀念1878年12月18日，賈西姆•本•穆罕默德•阿勒薩尼從他父親穆罕默德•本•塔尼那裡繼承了卡達半島的統治權。在他繼承王位之前，卡達的民眾普遍認為他通過抗擊外來勢力，為半島的所有部落贏得了相當的自主權。

12月23日以色列光明節

光明節是為紀念古代以色列人反抗希臘侵略者並取得勝利而設立的，其後演變成猶太人的感恩節，與美國的感恩節異曲同工。每年光明節的主要慶祝活動是點燃九燭台，節日為期八天，但是上班族不休假。

12月24日平安夜

平安夜是指12月24日是聖誕前夕，稱為聖誕夜，也稱為平安夜。屆時，千千萬萬的歐美人風塵僕僕地趕回家中團聚。聖誕之夜必不可少的節目是Party或聚會。聖誕之夜，父母們會悄悄地給孩子們準備禮物放在長筒襪里。

12月25日聖誕節

聖誕節是基督教世界最大的宗教節日，傳說是耶穌降生的誕辰。它是一個宗教節日，全球有許多深受基督教、天主教影響的國家，有著過聖誕節的傳統，聖誕會有假期，人們也會有一些對應的宗教慶典活動。

值得一提的是，聖誕節不再僅僅是宗教的節日。很多國家地區包括中國都會趁著聖誕節應應景，或者放假。聖誕前夕可以和客戶確認他們具體的放假時間，在節後才能相應跟進上。

伊朗里亞爾兌外幣匯率呈走低趨勢

伊朗《經濟論壇報》11月26日報道，伊朗里亞爾兌美元、歐元等外幣匯率在過去數周內一直保持相對穩定，但在政府實施汽油配額和漲價政策後，呈現快速走低趨勢。截至26日，自由外匯市場上，1美元兌12.348萬里亞爾，較前一天上漲2950里亞爾，漲幅為2.45%；1歐元兌13.544萬里亞爾，較前一天上漲2440里亞爾，漲幅為1.79%。二級外匯市場（NIMA）匯率則為1美元兌10.88萬里亞爾，1歐元兌12.029萬里亞爾。

對於外匯市場的波動，伊央行將責任指向了非油出口商，稱「出口商們試圖利用最近汽油漲價形成的氛圍，以較高的價格出售他們的外匯收入」。央行行長赫馬提表示，近期外匯市場價格是不真實的，因為沒有那麼多需求。但是市場分析者認為，接近年底時間外匯價格上漲是正常的，因為大部分海外合作夥伴都會在年底前封賬並要求伊朗客戶儘快承付賬款，因此這一段時間外匯需求會上升。

伊媒稱關閉互聯網帶來巨大損失

伊朗《SetarehSobh》報11月24日報道，稱關閉互聯網給伊朗帶來巨大損失。現編譯如下：

伊朗商業、金融領域經濟人士指出「斷絕」互聯網帶來巨額損失。

前伊朗商會主席賈拉波在媒體採訪時提到了（切斷）互聯網對進出口貿易領域的影響，他表示，在沒有互聯網的一周時間裡，幾乎無法與國外客戶進行業務往來，包括運輸、銷售和商務通信。互聯網是市場不可或缺的組成部分，沒有網路就意味著沒有市場和客戶，這方面的損失「我們承擔不起」。據伊朗官方統計，該國經濟部門在過去一周遭受了15億美元的損失。

德黑蘭商會商務委員會負責人瑪內什表示，商務領域所有的往來都需要通過國際互聯網，互聯網被切斷，商務往來也中斷了。儘管安全問題很重要，但至少要保證生產者和貿易商連接到互聯網，這樣他們就不會遭受損失。

伊朗-卡達商會主席瑪薩波表示，互聯網對出口具有決定性影響。「出口即意味著與世界交流，而最重要的交流方式是互聯網」，切斷互聯網雖不至於使得出口降低到零，但已經給出口帶來困難，使貿易商在財務和信貸方面遭受損害。

厄瓜多75%進口玩具來自中國

厄瓜多《宇宙報》11月29日報道，2019年1-9月，厄玩具進口額達1.07億美元，中國是第一大進口來源國，厄75%的進口玩具來自中國。

土耳其與卡達簽署提高貨幣互換至50億美元

據土耳其《晨報》11月29日報道，土耳其央行發布公告稱，埃爾多安總統訪問卡達期間，該行與卡達央行簽署一項貨幣互換協議，將2018年兩國貨幣互換30億美元額度提高至50億美元，以支持雙邊貿易本幣支付及為金融穩定提供流動性支持。雙方還簽署了旨在加強投資、貿易和技術等一系列合作協議。卡達為土耳其重要投資夥伴，雙邊貿易2018年為13億美元，10年來增長84%。

1-11月份越南吸收外資318億美元

越南計劃投資部外資局網站在線11月28日報道，今年1-11月份越南共吸收外商投資協議額近318億美元，同比增加3.1%；實際到位金額176.9億美元，同比增長7.2%。其中，新批項目3478個，協議額146.88億美元，同比下降7%；批准1256個項目增資申請，協議額為58.7億美元，同比下降10.7%；外商進行股權併購8561宗，涉及金額112.4億美元，同比增長47.1%，占外資總額35.4%。

今年吸收外資最多的仍是加工製造業，協議額215.6億美元，佔67.8%；其次是房地產經營業，協議額33.1億美元，佔10.4%。中國香港特區位居117個外資來源地首位，協議額66.9億美元；韓國57.3億美元位居第二；新加坡44.7億美元居第三；中國35.7億美元居第四。由於中美貿易戰的原因，中國內地和中國香港對越投資額分別是去年同期的2倍和3.9倍。
