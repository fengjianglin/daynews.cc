印度決定用美國的AH-64E，“阿帕奇”，取代俄羅斯直升機

2019-12-13 995條評論 1,930次閱讀 895人點贊

印度軍隊在歷史傳統上，大都使用俄羅斯的直升機。但是，最近它們的這個趨勢有了新的變化。有媒體報道，在2015年底，印度與美國簽署了20億美元的合同，用來採購美國的軍事裝備。這件事情的表象是，印度軍方採購的俄羅斯Mi-25和Mi-35直升機現在有些落後，無法滿足印度現在的需求。另一方面，AH-64「阿帕奇」能夠執行多種任務，攜帶更多的武器，並且與俄羅斯Mi系列的直升機相比技術更加先進。

這種邏輯似乎很適合印度武裝部隊的採購計劃。實際上，AH-64「阿帕奇」儘管是上世紀70年代設計和研發的，但AH-64仍採用了複合材料，搭載有開放式體系結構的新型電子設備以及其他現代技術。但是這並不能說明就「阿帕奇」一定是印度的最好選擇。

曾經有專家說，如果印度軍方真正想要尋找一架擁有現代化技術的武裝直升機，那麼在目前俄羅斯的武裝直升機中會有多重選擇。但是，印度沒有這樣做。據稱，印度空軍的理由是需要廉價，高效的武裝直升機，並且要在最短的時間內完成換裝。說俄羅斯的新裝備太昂貴了，這種價格的說法顯然不成立，俄羅斯的價格與美國比相當有競爭力。在這次軍事裝備的換裝中，印度選擇了「阿帕奇」，是否有其它方面的考慮我們不得而知。

近年來，印度一直在改變它的策略。先是決定購買法國戰鬥機，現在又是武裝直升機，同時總是給出牽強附會的解釋。大家認為印度這麼做的原因是什麼，歡迎小夥伴們在評論區發言討論。
