重磅！吉利德/百時美的CAR-T專利之爭審判落錘丨醫麥猛爆料

2019-12-16 122條評論 1,061次閱讀 77人點贊

近兩年，CAR-T領域的兩大參與者一直為其專利權而爭奪不休。近日，兩大公司在加利福尼亞對簿公堂，進行了為期兩周的審判，隨後陪審團宣布了審判結果：百時美（BMS）勝訴，吉利德敗訴，吉利德旗下的Kite公司因此需要向BMS旗下的Juno公司及合作夥伴支付7.52億美元。

BMS發言人表示對此次判決結果滿意，吉利德則堅持認為BMS持有的此項專利無效以及其並沒有侵犯該專利。另外，吉利德表示將通過審判後動議和潛在上訴來解決其擔憂。

CAR-T專利之爭的由來

眾所周知，CAR-T療法的出世引起了醫藥界的廣泛關注，作為一次性治癒的新型療法，其治療的標價一直居高不下，這也引起了國內外不少企業的相繼競爭，其專利權更是他們關注的重點。

此次兩家公司的專利爭奪主要是由「190專利」授予許可引起。以下便是該起事件的發展歷程：

  • 早在2007年，Kite與一位名為「A」的科學合作者向「190專利發明人之一」的”B”博士請教了一些問題。訴訟稱，基於A作為美國國立癌症研究所（NCI）研究員的地位，B向A介紹了2002年一篇關於該專利的期刊文章，因此Juno認為Kite間接性的獲取了190專利，並用於實踐。
  • 2013年，Juno從斯隆·凱特林和紀念斯隆凱特林癌症中心（MSKCC）獲得「190專利」的獨家授權許可。藉此，Juno認為Kite侵犯了其190專利的專屬權。
  • 隨後2017年，吉利德斥資120億美元收購Kite公司以獲得其CAR-T療法Yescarta，同年，該重磅療法獲得FDA批准上市。
  • 故此，2017年，Juno公司起訴吉利德的Kite公司，稱其通過抄襲斯隆－凱特琳的研究成果（即190專利權），進而推動其在CAR-T療法方面的進展，最終使得Yescarta獲得FDA批准上市。
  • 在起訴的過程中，2018年，新基以90億美元收購了Juno公司，全面布局CAR-T療法。
  • 2019年，BMS豪擲740億美元收購了新基公司。於是，最初Juno對Kite的起訴轉為了BMS和吉利德的對簿公堂。

一波剛平一波又起的CAR-T專利之爭

作為CAR-T——190專利的獲得者，Juno利用其專利權先後起訴了全球僅兩款上市的藥物開發企業：諾華和吉利德。

在CAR-T領域，除了吉利德上市的Yescarta之外，諾華的首款CAR-T療法Kymriah早在2017年8月就獲得了FDA的批准上市。在其上市之前，諾華與Juno就發起了專利糾紛，此後諾華交付了1225萬美元，外加未來的里程碑付款和CAR-T療法的特許使用費，才使Juno同意放棄了訴訟。就此，平息了關於CAR-T專利權的第一次風波。

隨後，在一系列的交易收購下（新基收購Juno;BMS收購新基），Juno隸屬於BMS，有著強大的動力支撐，不久就發起了對吉利德的專利起訴，近日獲得勝訴，目前該公司利用其專利，更是擁有多項I期和II期階段的CAR-T療法候選藥物。

結語

此次審判結果對吉利德來說，可謂是一大打擊。前不久的ASH會議上，其子公司Kite才公布了其第二款CAR-T細胞療法KTE-X19的積極臨床數據，並隨後向FDA遞交了其生物製品許可(BLA)申請，不知此次結果是否會影響該公司對CAR-T藥物的研發進展以及上市進度。
