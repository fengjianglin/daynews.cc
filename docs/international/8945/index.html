章瑩穎案兇手服刑監獄及作息曝光！每月小賣部消費上限320美元

2019-12-07 1,750條評論 3,217次閱讀 1272人點贊

當地時間12月6日，有美媒報道稱，殺害中國訪美學者章瑩穎的兇手克里斯滕森可能會在肯塔基州東南部的麥克雷里（McCreary）監獄度過餘生。今年7月，克里斯滕森因犯有綁架致死罪等三項罪名被判處終身監禁。

2017年9月至2019年5月，克里斯滕森一直都被關在利文斯頓縣監獄。直到今年10月中旬，他才被轉移到俄克拉荷馬城聯邦轉移中心。目前，克里斯滕森已經被列入麥克雷里監獄的囚犯名單。

今年6月，克里斯滕森被裁定犯有綁架致死罪等三項罪名，7月18日，他被判處終身監禁。據南都此前報道，判決結果出來後，聯邦監獄管理局將決定把克里斯滕森轉移到哪間監獄，這個過程可能需要一個月。在決定過程中，聯邦監獄管理局需要考慮監獄的安全等級、管理等因素，以及犯人可能需要的心理健康治療、醫療等方面的特殊需要。史蒂夫貝克特表示，在此前等待審判時，克里斯滕森在監獄中還擁有一定的自由甚至特殊待遇，包括獨立浴室、電視、平板電腦等，但接下來要他要被送往的監獄並不會向此前的利文斯頓縣監獄一樣，擁有好的住宿條件。

據悉，麥克雷監獄開放於2004年，囚犯的年齡從20歲到86歲不等，囚犯的平均逗留時間近16年。當前，這間安全係數高的聯邦監獄裡共有1335位男性囚犯和378名員工。

據美媒報道披露，麥克雷里監獄的囚犯必須在早上6點醒來，並在早上7時45分開始工作，工作任務包括食品服務、設備維護、看門、文書工作等。此外，囚犯不得在每日22時之後離開牢房。囚犯的個人物品僅限於放置在儲物櫃或床底下，每個月在監獄小賣部的消費上限為320美元。此外，監獄內提供了較大的休閑區域，可供團隊或個人進行體育活動、做手工或看電視等。

在章瑩穎案的審理過程中，陪審團曾在2個小時內就作出決定，克里斯滕森犯有綁架致死罪等三項罪名。今年7月18日，章瑩穎案被告克里斯滕森被判處終身監禁。判決結果出來後，章瑩穎案主審法官沙迪曾表示，他希望會在監獄裡「孤獨終老」的克里斯滕森能在死前向章瑩穎的家人表達歉意。

采寫：南都記者 秦楚喬
