生薑和這種食材搭配做成的暖胃茶，成本只要8毛錢，我每天一杯

2019-11-10 22條評論 1,064次閱讀 91人點贊

導語：大家好，我是梓棋，每天堅持自己下廚，用最簡單的食材，做最可口的菜肴。

現在很多人因為工作或生活的原因，經常熬夜，飲食不規律，不愛運動，導致自己脾胃就越來越差。很多年輕人仗著身體年輕，不重視，脾胃就非常容易受到影響，而脾胃虛寒的人，如果也不加以重視，不僅會導致肥胖的產生，身體健康的問題也會越來越差。

日常生活中我們調理脾胃的方法也很多，首先要養成良好的生活習慣，盡量做到早睡早起，不要熬夜，在日常飲食中，多加入驅寒功效的食材，比如生薑，胡椒等，這些食材可以多吃一些。有助於祛濕的食物，比如赤小豆，茯苓等，這些對於脾胃的調理也有一定的作用。今天分享的這款能調理脾胃的茶所用的食材，也許很多人日常中都丟棄的，那就是生薑烏賊骨茶，《中國藥典》里也有記載，生薑性辛，具有解表散寒，溫中止嘔的功效，有助於溫暖脾胃，但生薑屬於昇陽的食物，所以盡量選擇上午食用，而烏賊骨又叫海螵蛸，能中胃酸，將兩者搭配在一起，對脾胃的調理效果會更佳。

【生薑烏賊骨茶】

準備的兩至三人份食材：生薑15g，烏賊骨15g，黃冰糖適量按個人口味添加。

做法：1.生薑選取老薑的根部，取15g，洗凈切片或者切絲都可以。老人常說：「早上三片姜，勝過喝參湯」，所以常食生薑益處多多。

2.把洗凈的薑片和烏賊骨一同放入養生壺中，加入1000ml左右的清水，用水煮15分鐘即可盛出飲用。生活中人們常將烏賊骨丟棄，想用的時候卻不知道去哪裡買，最好去藥店購買。今天我買了15g只需要8毛錢，烏賊骨一定要用夠量才有效果。

這樣一杯暖胃祛濕的生薑烏賊骨茶就完成了，是不是很簡單很實惠。這款茶入口不會很辛辣，我是喝得慣的，如果一點也受不了生薑的辛辣的小夥伴，可以加點黃冰糖調味，早上一家人來上一杯，胃暖暖的，上班也精神。其實調理身體沒那麼難，關注我每日分享一款健康食譜，總有一款適合你的。

各位小夥伴，關於生薑你還有哪些新的做法嗎？歡迎留言分享。
