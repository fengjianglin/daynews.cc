關於茶葉的一些誤解，你知道多少？

2019-12-03 657條評論 2,245次閱讀 72人點贊

喝茶已經是生活里的一種習慣，是健康養生的一種方式。但是茶葉就只是葉子，而且品種多樣，種類繁多。導致很多人只知道喝。卻不了解它。也就導致對茶葉不一樣的解讀，或者錯誤的誤導。

那麼對茶葉知識的誤會有哪些呢？

一、茶到底能不能解酒？

以為能解酒的是因為茶能提神醒腦，而酒會讓人不清醒。對於這個問題我們還是要相信李時珍。他說：「酒後飯茶傷腎」等癥狀。傷腎你還要喝嗎？特別是濃茶，還會增加心血管負擔。所以酒後喝茶，應該適量，不喝濃茶。喝完酒，回家睡覺最好。你說呢？

二、隔夜茶能不能喝？

這個其實有很好理解，隔夜的東西很多你也不吃，後者放冰箱。但是有時候你也吃隔夜的東西。其實它是沒有壞的，就是怕有細菌。很多人說會致癌，是因為亞硝酸鹽跟胃酸會產生亞硝胺，而這是致癌的一種物質。但是其實在隔夜茶里所含有的量，幾乎可以忽略。所以你如果喝了隔夜茶，不要害怕。其實沒什麼事。

三、沖茶時，產生的泡沫，是農殘嗎？

不是。農殘，是茶葉裡面有種物質叫茶皂素引起的。這個物質其實是有益的。但是在自己泡茶時都會弄掉。沖茶的時候不要高沖，產生的泡沫就比較少。

四、喝茶會不會上火？

對於這個問題提供個人意見：上火屬於中醫範疇，深奧得很。其中有一個就是體質。個人體質決定你需要注意什麼。如果你喝某一款茶種，感覺上火了，那你就少喝。茶葉品種很多，總有一款適合你的。所以喝茶上不上火看個人，看茶的品種。

五、晚上喝茶睡不著？

茶葉裡面含有咖啡因，兒茶素。這些物質都是提高精神。經常喝茶的人，習慣了，就不會怎麼影響。有些不怎麼喝茶的，影響就比較大，特別是濃茶。其實這個跟喝咖啡差不多 有人加班，就需要喝咖啡提神，後者喝濃茶。知道自己喝茶睡不著，晚上睡覺前三個小時不喝茶，後者吃點甜點。喝淡茶等。

六、對於茶葉一些知識誤解

安吉白茶是屬於綠茶，福鼎白茶才是真的白茶。大紅袍不是紅茶而是烏龍茶，鐵觀音不是綠茶而是烏龍茶。普洱茶屬於黑茶。鳳凰單叢而不是鳳凰單樅。茉莉花綠茶不是加香精茶，而是加了茉莉花製作而成等。

七、茶葉是不是越陳越好？

不是。還是要看茶葉品種，普洱茶，湖南黑茶，六堡茶，藏茶等可以存放。綠茶，黃茶，鐵觀音，紅茶等及時喝，放太久不好喝。沒有季節的鮮味了。

喝茶是生活的一种放松，喝茶是生活的一種調味劑。不要因為一些誤解，讓你不敢喝茶。朋友們你們知道哪些誤解呢？歡迎評論，討論。#頭條公開課繁星計劃#
