寒冬來臨，謹防心腦血管病，高危人群日常注意事項

2019-11-21 2,817條評論 3,820次閱讀 1567人點贊

冬季是心腦血管病的高發期，而一旦腦血管病發作，致死率和致殘率都很高。寒冷的冬天已來臨，對心腦血管病高危人群來說是一個充滿挑戰的季節。人們需做好相應措施預防急性心腦血管病的發病。下面介紹一下日常注意事項。

(1)飲食方面

①限制鈉鹽的攝入。飲食應以清淡為宜，少吃鹹食。吃鹽過多，可使血管硬化和血壓升高。每天吃鹽應在5克以下為宜。

②少吃動物脂肪。動物脂肪膽固醇含量高，可加速動脈硬化，如動物肝、心等應少吃。

③少吃甜食。甜食含糖量高，可在體內轉化成脂肪，容易促進肥胖和動脈硬化。

④戒煙忌酒。有煙酒嗜好的患者，可因煙酒過多引起心肌梗死、腦血管病等。

⑤宜多食含鉀食物。鉀在體內能緩解鈉的有害作用，促進鈉的排出，可以降壓。含鉀的食物有：土豆、茄子、葛筍、豆類、番茄、乳製品、海帶、蘑菇及各種綠葉蔬菜；含鉀的水果有：橘子、蘋果、香蕉、梨、菠蘿、獼猴桃、核桃、山楂、西瓜等。

⑥宜多食含蛋白和維生素的食物。如魚、牛奶、瘦肉、豆製品等。

⑦多食含鈣食物。美國醫學專家認為，高血壓患者每天堅持食人高鈣食物，能有明顯的降壓效果。含鈣的食物有：乳製品、豆製品、花生、大棗、海帶、蝦皮等。

（2）起居方面

運動對機體的脂質代謝具有積極的影響，能加速脂質的運轉、分解和排泄。很多醫學專家都認為，加強運動鍛煉是高脂血症積極的防治措施。

一般來說，患有高脂血症而無其他併發症者應保持中等強度運動量，即每天達到慢跑3000～5000米的運動量。對有輕度高血壓、肥胖等並發疾病的患者應自行掌握，以鍛煉時不發生明顯的身體不適為原則。

運動方式可根據自己的情況及環境而定，可進行適量的有氧運動：慢跑、體操、太極拳、運動操、養生功、游泳、爬山、騎自行車及健身器鍛煉等活動，每次不少於30分鐘，每周不少於3次。只要持之以恆，保持一定強度的運動量，就能達到預防和治療高脂血症、降低冠心病等心腦血管疾病的患病率。

（3）良好的心態：俗話說：「笑一笑，十年少。」保持良好的心理狀態，穩定情緒，培養自己的各種愛好。可以通過欣賞音樂，學習繪畫和書法等來陶冶情操；種花草、養寵物也可以使人獲得良好的心態，緩解緊張的情緒。

（4）遵守醫囑，堅持服藥治療：繼續服用血小板解聚葯、降壓藥、降血脂葯，以及補腦健腦藥物等。

(5)其他 定期到醫院複查。

一旦發現有心腦血管徵兆，一定及時就診，做到早發現，早診治。
