五臟六腑都有“火”及時降火才能不傷身

2019-12-19 1,518條評論 4,161次閱讀 572人點贊

冬季因為天乾物燥，因此是很容易上火的。那麼，五臟六腑有「火」，怎麼才能降火呢？

　　冬季是很容易上火，因為天乾物燥，再加上喜歡吃溫補類的食物。請記住，五臟六腑都有「火」，如果你不及時降火的話，身體會垮掉的。那麼五臟六腑有「火」怎麼降火呢?這個需要好好了解一下，如此才能更有效率的降火。

　　五臟六腑有「火」怎麼降火

　　1、心火

　　心火重人，舌頭就會發紅或者說長泡，而且口苦，容易有脾胃疾病。另外，排出的小便是赤黃的。

　　去火方法：利用蓮子心煮粥食用，或者說用蓮子心、車前草、梔子花之類加水煎煮成茶湯飲用，當然味道比較難聞，若是喝不下的話，可以加些甘草一起煮，有助改善口味。

　　2、肝火

　　當人體肝火旺盛的時候，常見的癥狀有脾氣大、嘴裡有苦味，眼屎多，兩肋脹痛、性子急等。這是因為陽氣比較亢奮所致。

　　去火方法：如果你肝火旺盛的話，想要清瀉肝火的話，建議吃些苦瓜、茼蒿、蘆薈等，這些都是有助清肝火的。另外，可以準備一些菊花和枸杞，將它們放入杯中，加入熱水沖泡飲用。

　　3、脾火

　　當人脾臟有「火」時，往往會覺得嘴甜而粘，伸出舌頭能夠看到很厚的舌苔，而且中間是黃苔。還可能出現肚子脹痛、口乾、便秘等等問題，每次排便都是先干後軟，整個人很是乏力。

　　去火方法：建議買點山藥，刮皮後洗凈切成小段與大米(或者說小米)熬成粥食用，有助降脾火，而且效果挺不錯的。

　　4、肺火

　　肺臟有火的時候，容易出現口乾舌燥、乾咳少痰、氣喘咳嗽，咳出的痰略帶黃色甚至黃綠色。另外，臉上容易長癤子，而且口腔容易有辣味。

　　去火方法：建議買些沙參和大棗，清洗乾淨後一起煮水飲用。或者說選擇荸薺、雪梨、甘蔗這些水果榨汁飲用，都是能夠清理肺火的。

　　5、腎火

　　當腎可是先天之本，祖國醫學認為，腎臟是相火寄存的臟腑，當腎有「火」的時候，往往出現小便赤黃或者說澀痛，甚至還會有浮腫。此外，還可能出現口乾、大便干、盜汗、耳鳴等等問題。

　　去火方法：建議次用滋陰降火方面下手，比如說用金銀花、蒲公英、車前草等煮水當茶飲用。或者說煮一些具有清熱利尿作用的紅小豆當做粥食用，或者而準備些枸杞泡水喝。

　　冬季如何預防上火

　　1、室內溫度不要太高

　　冬季天冷很多人待在室內的時候，總是習慣將空調或者取暖器調到很高的溫度。長時間待在這樣的環境里，而且冬季天氣乾燥，很容易導致人體上火的。所以說，要控制室內的溫度。

　　2、規律作息

　　保持充足的睡眠，會給人體帶來活力，如果長期熬夜，導致睡眠不足的話，人體的免疫力會下降，而且還會出現其他大大小小的問題，甚至會加速人體衰老。如此一來，上火這個問題也是在所難免的。所以說，請大家戒掉熬夜習慣，堅持早睡的習慣，讓作息保持規律。

　　3、清淡飲食

　　冬季天冷很多人喜歡吃溫補類的食物，幫助身體禦寒，這是可以的，但是記住進食量要有所控制。尤其是一些熱性的食物，吃多了身體會積熱而上火。所以說，辣椒、胡椒、生薑等要少吃一些。
