手腳冰冷怎麼也“捂不熱”？4付熱湯改善手腳冰涼 | 吃出健康

2019-12-24 1,098條評論 3,593次閱讀 362人點贊

康復小編：每年只要一進入冬天，很多人都會特別怕冷，手腳永遠都是冰涼的。這樣不僅會讓自己身體感覺特別難受，而且還很容易會引發一些其他的疾病出現。

手冷得還以為自己是Elsa

冬天血液循環的速度就會相對減慢，因為氣溫比較低，所以血管收縮迴流的能力也是比較弱的，手腳等一些部位血液循環無法達到神經末梢。

寒冷的冬天對於怕冷的人來說是非常難熬的，女人的手腳一般情況下比男人要冷些。其實，如果可以好好調理身體，也是可以暖暖過冬的。

你為什麼會怕冷？

平時害怕寒冷和自身的體質是有一定關係的。寒性體質、腎陽虛的人群通常都比較害怕寒冷，常見於女性。但除了這個原因，手腳冰涼也可能是由於疾病所導致的。

凍 jio jio

1.甲狀腺分泌不足

甲狀腺激素可以讓機體物質加速代謝，而且還具有使糖分、蛋白質、脂肪等快速燃燒能量的作用，同時還會使心跳速度加快，加速血液循環的速度，增加身體熱量。所以如果甲狀腺激素分泌不足的話，熱量就會減少，導致身體怕冷。

2.鐵元素缺乏

身體中的鐵元素可以促進鐵紅素的合成，這樣就能夠提高血液供氧的能量，使身體產生熱量。一旦出現缺鐵的情況，外圍組織供氧量就會相對的減少，能量代謝會產生一定的障礙。所以在這種情況下也會導致身體熱量不足，而感到寒冷。

3.低血壓

在平日生活中血壓低的人群也是很容易會出現末梢血液循環不良的現象，尤其是手和腳等部位出現供血不足。組織在能量代謝的時候，沒有得到所需要的氧氣和能源，就會讓人產生一種害怕寒冷的感覺，且在這種情況下很容易發生凍瘡。

手腳冰涼應該怎麼辦？

1.忌食生冷，食補暖身

手腳冰涼的患者在平時到時候可以多吃溫熱的食物，通過食補的方法，就能夠讓身體這些癥狀得到很好的緩解。（下文有食譜~）

對於女性來說，平時多吃堅果類的食物，比如說松子芝麻等都是有很多好處的，且無論在什麼時候都不要去吃生冷的食物。

2.多做運動，促進循環

進入冬天之後，溫度比較低，血液循環的速度也會相對減慢。對於手腳冰涼的患者來說，應該在早上起床以後適量的做一些運動，比如慢跑、甩手、健走、爬樓梯、瑜伽、健身操等。

在運動的過程當中，氣血的運行會順暢一些。在早上的時候做運動會使血液循環的速度加快，從而緩解手腳冰涼的癥狀。

4付湯，食療改善手腳冰涼

平時一定要做到飲食均衡，保證每天攝入充足的蛋白質等營養物質，平時少吃生冷食物，比如冰淇淋等涼飲。今天就為大家推薦幾款益氣補血的靚湯，過個暖暖的秋冬～

1.補血雞肉湯

功效：適合身體虛弱、氣血不足、營養不良的人食用

食材：雞肉600克、蓮子50克、山藥50克、枸杞50克、薑片10克、冷開水1800毫升、料酒1湯匙、鹽1茶匙

做法：

1. 雞肉洗乾淨後切成塊，放入沸水中焯燙一下撈起備用。

2. 山藥去皮，洗凈後切成片。

3. 將雞肉和山藥、蓮子、枸杞一起放入鍋中，加入清水，用大火煮開。

4. 加入薑片、料酒之後，換成小火燉煮1個小時，直到雞肉軟爛。

2.茶樹菇燉排骨湯

功效：排骨被中醫認為是能夠補中益氣、調理氣血的好食材

食材：排骨300克、茶樹菇30克、紅棗10顆、姜2片、鹽1茶匙

做法：

1. 茶樹菇去掉根部，然後用清水浸泡約1個小時，切段。

2. 紅棗洗乾淨，將兩頭剪開口，瀝干水分。

3. 排骨洗乾淨，切成大塊，將排骨放入沸水中焯燙去掉血水，然後撈起，沖洗掉浮沫。

4. 將焯燙好的排骨放入湯鍋中，加入茶樹菇、紅棗和薑片，倒入淹沒食材的清水。

5. 用大火燒開，然後換成小火熬煮2個小時。

3.田七烏雞湯

功效：具有補血調經、除煩退熱之功，對於氣血不足、肝腎陰虛而見潮熱盜汗者療效更顯著，但不適宜孕期婦女飲用

食材：烏雞1隻(約1200克)、田七15克、紅棗8顆、陳皮2塊、姜2片、鹽1/2茶匙、清水2500毫升

做法：

1. 田七洗乾淨後壓成小碎塊。

2. 烏雞洗乾淨，斬成大塊。

3. 把烏雞放入鍋中，倒入清水，大火煮沸後繼續煮2分鐘，撇清浮沫後撈出，並用清水洗乾淨雞肉上的浮沫。

4. 鍋中倒入清水，放入烏雞、陳皮、紅棗、薑片和田七碎塊。

5. 大火煮沸後，換成小火燉煮至少3個鐘頭。

4.金針豬肝湯

功效：豬肝能明目、補血益氣

食材：干金針30克、豬肝片150克、姜3片、高湯400毫升、鹽1/4茶匙

做法：

1. 將食材洗乾淨後，干金針放入清水中浸泡15分鐘，撈起後，再用清水洗乾淨。

2. 豬肝洗乾淨，放入清水中浸泡半個鐘頭，期間換2-3次水，然後切成片。

3. 豬肝片用沸水焯燙後撈起備用。

4. 湯倒入鍋中，加入薑絲和金針煮沸，換成小火煮2分鐘。

5. 最後放入豬肝片煮滾。
