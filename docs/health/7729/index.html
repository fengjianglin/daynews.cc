“江南人蔘”——荸薺，秋天吃最適合

2019-10-22 915條評論 4,831次閱讀 600人點贊

荸薺，又稱馬蹄，皮色紫黑，肉質潔白，味甜多汁，清脆可口，自古有「地下雪梨」之美譽，北方人視之為「江南人蔘」。

荸薺口感甜脆，營養豐富，含有蛋白質、脂肪、粗纖維、胡蘿蔔素、維生素B、維生素C、鐵、鈣、磷和碳水化合物，可以用來烹調，可制澱粉。而且經常被用來做飲料，和雪梨、蜂蜜搭配便是一道潤燥清肺的好飲品。

藥用功效

荸薺性寒味甘，能清熱生津，涼血解毒，利濕化痰，解酒，降血壓。其所含荸薺素對金黃色葡萄球菌、大腸桿菌及綠膿桿菌等有抑制作用，但荸薺素不耐高溫，以生食為宜。

荸薺可以治療熱病傷津煩渴、咽喉腫痛、口腔炎、濕熱黃疸、肺熱咳嗽、痔瘡出血、痢疾下血、婦女崩漏、醉酒乾渴、大小便不利等症。

葯膳方

1.荸薺蜇皮湯

材料：荸薺250克，海蜇皮100克，精鹽適量。

做法：

  • 先將荸薺洗凈，去皮切成厚片，再將海蜇皮洗凈，切成粗絲；
  • 然後將荸薺、海蜇皮同放砂鍋內，用小火燉至酥爛，加精鹽適量即可食用。

功效：此湯有滋陰降壓的作用，是陰虛陽亢型高血壓病人的理想食療佳品。

2.荸薺蛋苗餚

材料：荸薺若干，雞蛋2～3個，熟火腿1～2個，新鮮豌豆苗若干，精鹽、味精、料酒、澱粉、雞湯等。

做法：

  • 豌豆苗擇洗乾淨，用開水稍燙後，放涼待用；
  • 熟火腿切末，荸薺去皮切成碎丁；
  • 雞蛋入碗攪拌均勻，加入荸薺丁、精鹽、料酒、澱粉、雞湯，再打均勻；
  • 炒鍋入油，燒至八成熟，將打好的雞蛋入鍋翻炒，炒成糊狀時加味精入盤，再撒上火腿末、豌豆苗即可食用。

功效：滋陰潤燥，清熱生津，和中下氣，止渴止瀉。

3.荸薺三金粥

材料：荸薺150 克， 雞內金20克，金錢草30克，海金沙15克，粳米100克。

做法：

  • 先用水煎金錢草和海金沙，過濾取汁備用；
  • 將荸薺搗爛擠汁，將雞內金烘乾研細；
  • 再將此荸薺汁、雞內金粉和粳米加適量水煮粥，待粥半熟時加入葯汁，煮至米爛粥稠後代早餐食用。

功效：可清熱消積，利尿排石，通淋止痛。常服此粥對於尿路結石導致的尿色黃赤混濁、小便艱澀灼痛以及時而尿流中斷、尿道刺痛等，效果較理想。

4.紅蘿蔔馬蹄雪梨湯

材料：紅蘿蔔200克、馬蹄200克、200~300克胡雪梨1~2個（2~3人量）。

做法：

  • 各物分別洗凈，去皮，雪梨去芯，切塊；
  • 先把紅蘿蔔、馬蹄放進瓦煲內，加入清水1500毫升，武火滾沸後，改文火滾約30分鐘，下雪梨再滾沸約15~20分鐘便可，可1日分上下午進飲。

功效：開胃消食，清熱生津。 特別適合胃口較差、口乾心煩、消化不良的兒童。

飲食宜忌

脾胃虛寒泄瀉、女子月經來潮期間，均忌食生冷荸薺；因含糖分較多，糖尿病患者亦應少食。

來源：《家庭醫藥增刊·葯膳大全》，本文有刪改。
