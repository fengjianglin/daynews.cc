冬天，這6款養生粥，我是一定要喝的

2019-11-25 906條評論 2,417次閱讀 1464人點贊

冬季養生粥

撰文　王建明

編輯　保健君

「小雪」節氣已過，天氣越來越冷了，冬季多喝粥有助於增加人體熱量，增加營養，還可以補充水分避免血液黏稠，幫助消化，提高機體的抗病能力。現介紹六款適合冬季食用的養生粥，供大家參考選用。

核桃羊肉粥

核桃仁10克，羊肉100克，羊腎1對，大米100克，蔥、姜、鹽等調味品適量。先將羊肉洗凈，切碎；羊腎剖開，去筋膜，切碎。將大米放入水中煮沸，放入羊肉、羊腎，煮至粥熟後，加入蔥、姜、鹽等調味品後即可食用。

適用人群：此粥具有溫補腎陽的作用，適合陽虛怕冷者食用。

肉桂山楂粥

肉桂5克，山楂30克，糯米100克，紅糖適量。先將肉桂水煎20分鐘取汁100毫升備用；另用砂鍋放入水，將山楂、糯米一同浸泡15分鐘後先用武火煮沸，再用文火慢熬，待粥煮至濃稠時放入肉桂汁和適量紅糖再稍煮片刻，即可食用。

適用人群：肉桂溫中散寒，能擴張血管，使血液循環旺盛，疏通血脈；山楂活血化瘀、健脾養胃，可促進氣血流通。兩者配伍相得益彰，適用於脾腎陽虛引起的手腳冰涼、脘腹冷痛、食欲不振等症者食用。

山藥桂圓粥

山藥100克，粳米50克，桂圓15克，荔枝8克，五味子3克，白砂糖30克。將山藥去皮切成薄片，粳米淘洗乾淨。山藥片與桂圓、荔枝、五味子、粳米同置鍋內，加水適量煮粥，粥熟後加白糖調味即成。

適用人群：此粥具有補益心腎、安神益智的功效，適宜心悸失眠、眩暈健忘、神疲乏力者食用，尤其適宜中老年腦力勞動者食用。

首烏芹菜粥

制何首烏20克，芹菜20克，豬瘦肉餡50克，粳米100克，鹽、味精適量。先將制何首烏入砂鍋加水煎煮30分鐘，取藥液濃汁。另將粳米洗凈，加適量的水和首烏藥液汁，煮至粥將熟時入精瘦肉餡和芹菜末，再煮至米爛肉熟，加入鹽、味精等調料，早晚服食。

適用人群：此粥有補益氣血、降脂降壓的作用，對防治心血管系統疾病和延緩動脈粥樣硬化均有一定效果。

杞精山藥粥

枸杞子15克，黃精20克，山藥30克，粳米100克，白糖適量。將黃精、山藥切片備用。加水先煮粳米至半熟，再下枸杞子、黃精、山藥，一同煮至米熟粥稠，白糖調味，單獨食用或佐餐。

適用人群：此粥有補肝腎、益精血的作用，適用於肝腎精血不足、脾氣衰弱，症見頭昏耳鳴、健忘、消瘦少食等。

茯苓粳米粥

茯苓30克，粳米100克，紅糖適量。先將茯苓研末備用。粳米淘洗乾淨放鍋內加適量清水煮粥，待粥煮至濃稠時，再放入茯苓粉拌均勻稍煮一會兒，以紅糖調味即可。溫熱空腹食用，每日早、晚各服1次。

適用人群：此粥有健脾益胃、利水滲濕、益智寧神的作用，適合有食欲不振、腹脹便溏、小便不利、四肢無力、痰多水腫、體虛浮腫等症者服食。中老年人常服有強身健體、抗衰防老、延年益壽之功效。
