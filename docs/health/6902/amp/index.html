人蔘

性味歸經：

一年四季，養在冬。春天、夏天、秋天，都是人體消耗的季節。只有冬天，人體進入收斂狀態，身體亟需營養，以幫助受損細胞和器官進行修復工作。走進中藥老字號店，各種滋補品琳琅滿目，佔據C位的當屬人蔘。但是「參」這一家子成員眾多，個個都兵強馬壯，功效各異。

下面就讓我來帶大家了解一下吧。溫，味甘、微苦。歸脾、肺、心、腎經。

功效：

大補元氣，補脾益肺，生津止渴，安神益智。適用於大病、久病體虛患者。

分類：

  1. 野山參：生長於深山野嶺，時間長達數十年乃至上百年，極其難得，市面上鮮有。

  2. 園參：人工栽培於園地，故稱園參，市面上多為此種。由於加工方法不同又分生曬參、白參和紅參。

  3. 生曬參：性平和，補氣為主，尤適用於氣陰不足者。

  4. 白參：與生曬參功效相似。

  5. 紅參：性溫味甘，有溫燥之性，長於振奮陽氣，尤適用於冬天怕冷，陽虛體弱者。

鑒別：

  1. 看根莖。稱為蘆頭，圓柱形完全的短根莖。

  2. 看鬚根。長條須，老而韌，清疏而長，其上綴有小米粒狀的小疙瘩，名為「珍珠點」，色白而嫩脆者為次品。

  3. 看紋理。毛根上端肩膀頭處有細密而深的螺絲狀橫紋，橫紋粗糙、浮淺而不連貫者為次品。

  4. 看主根。根順理且直，單腿或雙腿併攏者為次品。

  5. 看錶皮。老皮，黃褐色，質地緊密有光澤。皮嫩而白者為次品。

服用方法:

  1. 含化法。2-3片人蔘片含於口中細嚼。

  2. 沖泡法

  3. 燉服法。人蔘烏雞湯等各種養生湯。

  4. 磨粉法。磨成細粉，每日服用。

  5. 泡酒法。切成薄片裝入瓶內用白酒浸泡，每日酌情服用。

注意事項：

體弱滋補者、動脈硬化、高血液黏度、失眠、胃病、膽囊炎和膽結石者，以及兒童、孕婦和哺乳期婦女，不宜過多服用人蔘。

忌與蘿蔔同服。

高麗參

紅參的一種，專指朝鮮半島出產的人蔘（選取六年以上的），又稱別直參。

性味歸經：

性溫、味甘，歸心、脾、肺經。

功效：

大補元氣，益血，養心安神，提高免疫力。

鑒別：

  1. 看蘆頭。最好是雙蘆頭或馬鞍蘆，蘆碗6個以上為佳。

  2. 看色澤。顏色醬紅，表皮堅實，少有褶皺，強光的透視下完全半透明。

  3. 脖子短而粗，扁而寬。

  4. 比例適中，與腿的長度比例為6:4至7:3為佳。

（文/同濟大學附屬東方醫院 陸麗君）
