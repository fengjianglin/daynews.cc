早起不要著急吃飯，空腹5粒花生米

天天要聞
2019-12-22

花生在我們生活當中可以算得上是最常見不過的了，每到逢年過節招待親朋好友的時候通常花生豆是少不了的堅果食材。

花生一般分為兩種，一種是紅皮花生，一種是白皮花生，紅皮花生外表就是紅顏色的皮對血小板過少的人具有很好的恢復作用，而且花生的吃法也非常的簡單，有的人喜歡熬粥，有的人喜歡生吃，有的人喜歡煲湯等等。



有專家說，早上空腹吃花生對身體的作用是發揮到最大的表現，今天小編就來談一談：早起不要著急吃飯，空腹「5粒花生米」，不出兩周，3大好處找你！

1、能夠健脾養胃、補氣血

花生中含有的營養物質，能夠起到一定的養胃作用，能夠保護胃黏膜，預防胃病，對於腸胃不太好和經常出現胃痛腹脹的人來說，早晨起來空腹吃幾粒花生，對我們的腸胃健康有很好的幫助。而且花生中含有的鐵元素，具有益氣補血之功效，常吃能夠有效預防貧血癥狀，很多女性朋友容易氣血不足，體質虛寒，可以吃點花生，有一定的緩解作用。

2、增強記憶力，防治心腦血管疾病

花生中含有豐富的維生素E和鋅，是大腦發育不可缺少的營養成分，能夠有效延緩大腦功能衰退，增強記憶力，花生中含有的生物活性還可抑制腫瘤疾病發生，防止動脈硬化，預防心腦血管病，降低高血壓和心臟病的幾率。

3、降低膽固醇

不少有心血管疾病或者是膽固醇較高的人，都喜歡隨手吃花生，因為花生中有一種油酸的成分，它不僅對人體心血管有益，而且還可以降低人體中的有害膽固醇。所以也可以說它是一種天然的滋補佳品，食療功效很好，膽固醇高的人，可以每天吃一小把，體質好的人也可以常吃，能預防不少慢性疾病呢。


煮熟或生吃都行，不建議炒食！
