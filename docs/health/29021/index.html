長壽蛋白SIRT6還可以預防脂肪肝和脂肪肝疾病

2019-12-22 60條評論 1,176次閱讀 92人點贊

脂肪肝或肝脂肪變性會在人體產生過多脂肪或無法充分有效地代謝脂肪時發生，影響全球約25％的人口。多餘的脂肪儲存在肝細胞中，在肝細胞中積累並會引起脂肪肝和其他疾病。

在剛剛發表在《細胞報告》雜誌上的一項研究中，研究人員首次發現SIRT6是一種參與調節許多生物過程（例如衰老，肥胖，胰島素抵抗，炎症和新陳代謝）的蛋白質，在燃燒和調節中也起著至關重要的作用。肝脂肪代謝。

SIRT6通過激活另一種稱為過氧化物酶體增殖物激活的受體α（PPAR-alpha）的蛋白質來調節脂肪代謝。這種蛋白質促進肝臟中脂肪的燃燒。這項研究的主要作者，巴伊蘭大學米娜分校和埃弗拉德·古德曼生命科學學院的首席作者海姆·科恩教授說：「 SIRT6就像是一個雜耍者，在體內的代謝過程之間保持平衡和協調。」 「通過與PPAR-alpha一起工作，SIRT6實際上可以向身體發出燃燒更多脂肪的信息。這種合作是SIRT6 預防脂肪肝和脂肪肝疾病以及與肥胖相關的損害的一種方式。」

Cohen及其同事先前的研究表明，增加SIRT6可以延長使用壽命。為了測試這種蛋白質還如何延長健康壽命，本研究的研究人員增加了常規SIRT6的水平，以抵消高脂飲食和脂肪肝疾病中SIRT6的減少。他們使用由卡內基梅隆大學（Carnegie Mellon University）的Ziv Bar Joseph教授開發的計算工具，研究了不同的代謝狀態，例如禁食和定期飲食，發現SIRT6的更高表達導致更多脂肪的燃燒，特別是在肝臟中。

Shoshana說：「 SIRT6不僅可以與PPAR-α結合使用，從而使身體燃燒更多的脂肪並協調肝臟中的脂肪代謝，而且還可以調節與肝臟中脂肪相關的其他代謝途徑，例如膽固醇和甘油三酸酯代謝。」奈曼是巴伊蘭大學Mina博士和Everard Goodman生命科學學院的博士生，他是該研究的共同作者。
