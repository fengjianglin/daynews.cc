雞蛋中膽固醇不低，如果血糖高吃雞蛋好嗎？

天天要聞
2019-12-17

血糖高的朋友可以正常吃雞蛋，雞蛋中的碳水化合物含量並不高，對血糖的影響其實不大，雞蛋還能為我們提供豐富的營養，特別是優質蛋白，雞蛋是唯一一個「全蛋白」食物，也就是說氨基酸的種類、佔比和人體極其相似，人體的吸收率極高，可高達97%以上，所謂「全蛋白」，就是可以作為其他含蛋白質食物的楷模來配比其中的蛋白質！雞蛋還含有豐富的礦物質、脂溶性蛋白質，對於糖友來說也是營養攝入的保證之一。

不過，雞蛋中可能會有一些讓高血糖朋友擔憂的問題，比如雞蛋的熱量、脂肪，雞蛋的膽固醇，畢竟如果高油高脂食物攝入過多，會增加胰島素抵抗，同樣對血糖不利，而膽固醇的過多攝入可能是對血脂的不利，對於糖友來說可能更容易誘發高血脂，從而誘發多種併發症的發作。100g雞蛋提供的能量是144大卡，脂肪含量8.8g，一個雞蛋大約70g左右，而每人每天推薦攝入的雞蛋就是1枚，所以熱量越100g左右，脂肪不超過6g，這點熱量和脂肪咱們一個上午的時間就基本消耗殆盡了，並不能算作是高脂高熱。不過雞蛋黃中的膽固醇不能忽略了吧？畢竟100g雞蛋含有膽固醇超過280mg，已經列入高膽固醇食物的行列了。

部分嚴重高血脂、高血壓患者或者糖尿病合併較嚴重的高血脂患者，最好是在醫生的指導下判斷自己是否要吃雞蛋黃，不過一般的高血糖或糖尿病患者其實並不用過於擔心雞蛋中膽固醇的攝入。高膽固醇血症其實並不是因為外源性膽固醇攝入過多造成的，而是和遺傳因素、長期高油高脂食物攝入緩慢引起，而高油高脂食物中的膽固醇含量普遍較高，從而更容易引起高血脂症。外源性膽固醇對總膽固醇的影響很小，它的佔比只有20%，隨著攝入量的升高吸收量還會不斷減少，影響更大的是內源性膽固醇，例如低密度脂蛋白偏高，或者高密度脂蛋白偏低，都可能會造成高膽固醇血症。

雞蛋的攝入實際上有助提高「好膽固醇」（高密度脂蛋白）的含量，它能夠幫助膽固醇更快被運輸到肝臟代謝，平穩血清膽固醇，反倒有助改善高血脂。雞蛋中還含有磷脂成分，卵磷脂能夠幫助我們抑制脂蛋白在血管壁的堆積，有助預防動脈粥樣硬化，保護血管，總體來說，每日合適地吃雞蛋反倒是有助三高患者的選擇。
