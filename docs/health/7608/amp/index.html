多數人做冰糖雪梨止咳無效果，原來是方法不對，正確的步驟在這裡

天天要聞
1 個月 ago

導語：秋季是梨子多產的季節，這個時候隨時適合頓一碗冰糖雪梨來潤一潤嗓子，尤其是家中有孩子和老人的，自身抵抗力就低，夏天的炎熱會在體內堆積了不少火氣。

這時候吃梨止咳最好了。我家寶寶夏天就喜歡吃冷飲，每年到了秋季都會咳嗽，我都會給她燉煮幾個冰糖雪梨來止咳，連吃3天就能見好。

但是有很多人會問，為什麼我的做冰糖雪梨吃完之後就沒有止咳效果，這是為什麼呢？原來是你的方法不對，方法不到位，吃再多冰糖雪梨也沒有效果的。

下面就看看我是怎麼做的吧，保准你吃了之後清肺又止咳！

冰糖紅棗雪梨盅

By 飄零星

配料：

雪梨 4個、紅棗 10餘顆、冰糖 10餘粒、小蘇打粉 10克、精鹽 1小勺

步驟1.準備好所有的食材。

步驟2.將雪梨放入淡鹽水中浸泡十來分鐘，然後用小蘇打粉搓洗表皮，再用清水沖洗乾淨。雪梨不需要去皮，燉好之後是連著皮一起吃的，所以一定要將表皮清洗乾淨。

步驟3.沿著雪梨頂端三分之一的位置切開，藉助小刀和勺子將梨核和部分梨肉挖出來。

步驟4.紅棗用清水浸泡十來分鐘，清洗乾淨褶皺裡面的灰塵，然後去掉紅棗核，剪成紅棗圈。如果嫌麻煩的話，可以直接剪成片狀。

步驟5.將挖出來的梨肉重新填回挖空的梨窩裡，然後放入紅棗圈和冰糖。

步驟6.按照上面的步驟，將四個雪梨全部都做好，然後放入深碗或者深盤裡面，將切下來的梨蓋重新蓋回去，可以藉助牙籤固定住。雪梨燉好之後會流出很多湯汁，所以要選用深一點的容器。

步驟7.將雪梨放入蒸箱裡面，設置溫度100度，時間90分鐘。

步驟8.燉至雪梨表皮呈現深黃色、梨肉呈現透明狀的時候，就表示燉好啦。取出來，享用吧。雪梨和湯汁一起吃掉，潤肺止咳效果很好喲。

烹飪小貼士：

  1. 如果沒有蒸箱的話，可以用普通的蒸鍋，中大火，隔水蒸1個小時至1個半小時，時間根據你家的火力，只要梨肉完全軟化就可以了。
  2. 如果覺得挖空梨核比較麻煩的話，可以直接將雪梨去皮去核，切成小塊，然後放入冰糖和紅棗，再入蒸鍋蒸軟。
  3. 梨窩裡面還可以放入枸杞、百合、銀耳等食材，豐富口感和營養。
  4. 風熱咳嗽的話，可以加入川貝母一起蒸，風寒咳嗽的話，則可以將川貝母換成麻黃。
  5. 除了隔水蒸，還可以直接煮，只是煮的雪梨效果沒有蒸的好。
