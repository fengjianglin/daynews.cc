哪些食物會加劇腹瀉？這2點是常見誤區

2019-12-18 784條評論 2,887次閱讀 60人點贊

腹瀉其實是一個非常常見的問題，很多種原因都可能導致腹瀉，比如疾病、飲食或者藥物，甚至精神壓力什麼的都會有影響，在腹瀉的情況下飲食上應當怎樣注意就成了一件很重要的事情。

常見誤區

首先膳食纖維是可以吃的，比如說像煮熟的胡蘿蔔、香蕉什麼的，對於你腹瀉時候的腸道是有幫助的，特別是香蕉其實並不是像所謂有些人想像是滑腸的，其實香蕉很適合腹瀉患者，還可以補充糖 電解質 可溶性膳食纖維，少量單寧甚至對於大便形狀的改變也有影響。

有一些常見的錯誤是覺得一定要禁食，其實這是沒有必要的。多數情況下的腹瀉，在補充水分的同時還是可以吃一些容易消化的食物，但有一些食物可能值得注意。

  • 牛奶

很多人其實存在乳糖不耐受的問題，當你攝入非常多乳糖的情況下，你的小腸無法快速的分解，有可能會在大腸內被細菌發酵，使得你的病情加劇，出現乳糖不耐受的癥狀，比如腹瀉、腹痛等問題。

常見的除了純牛奶以外還有乳酪、冰激凌也有可能會導致類似的癥狀。

  • 酒精

其實平時我們都不建議以任何健康為目的去飲酒。

酒精一方面會對你的胃黏膜產生刺激，另外也會影響你肝臟的代謝，甚至有可能會在引用比較多的情況下加劇你消化道的癥狀，腹瀉時酒精也會加劇脫水，酒精還有可能影響某些藥物的代謝。

  • 咖啡

第三種就是咖啡了，其實正常情況下有些人可能能夠耐受咖啡因，咖啡對你的腸道蠕動沒有特彆強的刺激作用。但是咖啡因可能會加劇你的脫水癥狀，因此在腹瀉的時候我們不建議喝咖啡。

  • 辣椒

辣椒很顯然會加速對腸道的蠕動，有可能加劇你的腹瀉，同時也容易引起一些腹脹以及燒灼的感覺。

再加上腹瀉本來就會使得你比較衰弱，非常強的刺激也有可能引起其他不適。

  • 部分豆製品

再有是一些沒有經過完全熟透和加工的豆製品，其實豆製品中往往會含有一些容易引發脹氣的成分，儘管多數人是可以耐受的，但最好也不要大量食用。

  • 糖醇類

懷腹瀉期間還要注意糖醇類甜味劑成分，比如說山梨糖醇、木糖醇這些，它們都是有可能加劇腹瀉的。

所以如果你在吃口香糖、一些甜的糕點的時候，最好注意一下它的配料表，看是否用了這些成分。

  • 堅果

類似的還有堅果，首先，如果你本身對堅果不過敏的話，吃堅果是沒有問題的，並不是絕對不能吃。

但堅果的問題在於多數人其實不是完全能夠把堅果消化吸收，這時候對於一個腹瀉患者來說就沒有什麼必要了。

  • 果汁和蜂蜜

果汁的滲透壓過高，一般不建議榨汁之後大量飲用，少量飲用還是可以的，還有像蜂蜜，它的果糖含量也很高，有些人可能會果糖不耐受，反而加劇腹瀉。

該怎麼吃？

其實在腹瀉期間，正常的白米飯、土豆、燕麥粥、雞蛋、香蕉、木瓜、雞肉這些都是不錯的。

另外注意少食多餐，可以每3到4個小時吃一些，注意大量飲水以及補充電解質。
