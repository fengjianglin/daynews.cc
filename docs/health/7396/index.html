術後補蛋白質？你out了，小分子肽才是首選

2019-11-14 313條評論 2,180次閱讀 525人點贊

世界衛生組織指出：健康長壽的影響指數中，遺傳佔15%、社會佔10%、醫療佔8%、氣候佔7%、自我保健佔60%。營養改善也被作為重要的策略列入聯合國可持續發展目標及許多國家的健康促進規劃中。新型營養素肽對術後病人及其他體弱者便有著營養支持的幫助。

外科病人的臨床治療過程中，常會有一些疾病妨礙食物的攝入，或引起食慾減退、代謝紊亂、消化功能下降，影響食物在胃腸道正常消化吸收，造成患者營養不良和免疫功能低下。

此外，手術創傷及併發症或伴發的感染等使機體處於高代謝狀態。蛋白質是更新和修補創傷組織的原料。如果這時不注意蛋白質的攝取，就會引起血容量減少，血漿蛋白降低，傷口癒合能力減弱，免疫功能下降的現象。

臨床實驗表明，對外科疾病患者進行合理的營養支持可以改善患者的營養狀況，促進蛋白質合成和組織修復，改善腸粘膜的功能，調節患者免疫功能，最終對患者疾病的治療效果和康復產生積極的作用。補充小分子肽營養補充劑能明顯改善危重病人的營養狀況，提高患者的總蛋白、白蛋白、前白蛋白水平，對改善免疫力、防止腸道菌群移位具有積極作用。

老人或小孩中體弱多病人群，通常因為腸胃消化功能紊亂，造成營養物質無法正常吸收代謝，也同樣存在營養不良的情況。處於社會中流砥柱的中青年人亞健康狀況也比較嚴重，有很多人一直處於慢性疲勞的狀態。

小分子肽無需消化分解即可直接被腸上皮細胞吸收利用，適用於胃腸功能不全或胰液分泌不足的病人；另外有較低的滲透壓，能減少胃腸道副反應，耐受性更強。危重病人的應激係數高，營養及能量消耗主要用於維持重要內臟器官的功能代謝，消化道的消化吸收功能順應性降低，此期給予短肽型腸內營養可以減輕消化道吸收負擔，為機體提供能量及蛋白。
