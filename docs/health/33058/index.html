冬至飲食有講究，孩子吃飯注意這4點，健康過冬

2019-12-24 1,040條評論 3,107次閱讀 756人點贊

冬至是時年八節之一，古時民間有在「八節」拜神祭祖的習俗。冬至被視為冬季的大節日，在民間有「冬至大如年」的講法，所以古人稱冬至為「亞歲」或「小年」。那冬至節氣小朋友飲食方面應該注意哪些呢？

1、食宜多樣

身體的滋養需要多種營養，冬至節氣一定要注意飲食的均衡性，保證飲食營養的多樣化。 最好是做到谷、果、肉、蔬合理搭配，並適當選擇一些高鈣食品。

2、食宜清淡

冬至節氣最好不要吃太多油膩的食物，尤其小朋友消化能力比較弱，建議保持飲食的清淡。 如果吃太多油膩的食物，易造成消化不良，出現健康狀況。

現代營養學的金標準是小朋友飲食要做到「三多三少」：攝入蛋白質、維生素、膳食纖維多多，攝入糖類、脂肪、鹽保證少量。

3、食宜溫熱熟軟

冬天的時候小朋友脾胃比較虛弱，而脾喜溫熱、厭寒冷，所以吃東西的時候一定要保持溫熱，這樣可以保護脾胃。而且小朋友牙口沒有發育完全，建議遵循健康飲食原則，保證溫熱熟軟的飲食攝入，放棄冷硬的食物。

4、食宜少緩

冬至節氣一定要注意保持飲食的適度攝入，做到飲食八分飽，吃太飽會給身體的消化系統帶來不利影響。吃東西時要細嚼慢咽，讓食物充分地被咀嚼，這樣不僅對脾胃比較好，而且對小朋友身體健康非常有利。

冬至建議食譜：胡蘿蔔鮮蝦豬肉水餃

食材用料：

麵粉500g，豬肉200g，鮮蝦仁200g，胡蘿蔔2根，水260ml，食用油2勺，生抽1勺，老抽半勺，蚝油半勺，香油1勺，鹽適量，木耳適量。

製作步驟：

  1. 準備好所有食材，蝦仁剁碎，豬肉剁碎，胡蘿蔔剁碎擠出汁，木耳提前泡發剁碎。
  2. 加入所有調料拌勻備用。
  3. 麵粉加水和成麵糰醒十分鐘，麵糰分成小劑子按扁，擀成餃子皮。
  4. 全部包好，水開下入餃子煮八分鐘出鍋。

胡蘿蔔裡面含有豐富的維生素以及胡蘿蔔素，這些營養元素對於寶寶來說都是非常重要的，不僅能夠促進寶寶的生長發育，還能夠幫助保護孩子的視力，是非常好的一個選擇；且鮮蝦裡面含有豐富的鈣元素，能夠幫助孩子補鈣，兩者搭配營養又美味。
