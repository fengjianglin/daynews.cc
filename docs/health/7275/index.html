能改善關節炎的食物有哪些？推薦這5種

2019-11-21 402條評論 1,349次閱讀 355人點贊

有關節炎的患者，都知道這是一種非常痛苦的疾病，會影響到日常活動。我們經常會聽到「食療」這個詞，其實就是通過食物來改善身體健康狀況。對於關節炎患者來說，日常生活中，可以多吃下面幾種有名的關節炎「剋星」食物，以幫助緩解關節炎的癥狀。

1. 堅果

日常生活中常常見到的核桃、開心果、杏仁等堅果中，富含抗炎單不飽和脂肪、蛋白質以及纖維，這些營養物質可以幫助緩解關節炎的癥狀。因此，關節炎患者應該每天堅持吃一點堅果，用以緩解關節疼痛的癥狀。

2. 漿果

我們都知道漿果類水果中有豐富的抗氧化劑，可以清除體內的自由基，提高免疫系統的預防能力，還可以起到抗炎的作用。因此，平時多吃一些藍莓、草莓、樹莓或者櫻桃等漿果類食物，可以減輕關節炎的癥狀。

3. 洋蔥

有一些人不愛吃洋蔥，因為洋蔥有強烈的辛辣味。但你可能並不知道，洋蔥中的抗氧化劑，可以幫助減輕關節炎患者體內的炎症，減輕其疼痛的問題。

4. 橄欖油

橄欖油中的脂肪是對人體有益的單不飽和脂肪，這種脂肪可以幫助對抗炎症，對於關節炎患者十分有利。不過需要注意的是，在選用橄欖油的時候，一定要選擇特級初榨的橄欖油。

5. 豆類食物

豆類食物既便宜，營養價值又高，非常適合關節炎患者食用。不管是黑豆、紅豆還是菜豆，這些食物中都含有大量的抗氧化劑、抗炎化合物、蛋白質、纖維、鎂、鐵、鋅等，是關節炎患者理想的食物之一。

最後要提醒的是，以上5類食物中，豆類食物是不適合痛風患者食用的，因為痛風和普通的關節炎不同，而豆類食物含有較多的嘌呤，會加重痛風的發作。
