科學家有望揭開長壽的秘密：過量自噬蛋白p62可延長線蟲壽命一周

2019-12-17 404條評論 4,646次閱讀 1994人點贊

文 / 蔣力 編 / 袁月

【搜狐健康】長壽的秘密是什麼？近日發表在《自然》雜誌上的一項研究發現，過量的自噬蛋白P62可以使線蟲的壽命延長20%-30%。該研究可幫助發現與衰老有關疾病的治療方法，例如阿爾茨海默氏病。

「包括我們在內的研究表明，通過增強自噬，即一種用來降解和回收衰老、破碎以及受損的細胞成分的過程，可以延長壽命。」該研究的作者Malene Hansen博士說。「在這項工作開展前，我們知道自噬是一個與衰老有關的過程，但是作為一種選擇性自噬蛋白，p62蛋白對壽命的影響尚不清楚。」

科學家曾認為，對於所有細胞垃圾，細胞回收功能都是以相同的方式起作用。近年來，研究人員發現，自噬可以具有高度選擇性。這意味著該細胞可以使用獨特的「回收卡車」（例如p62蛋白）將不同類型的垃圾輸送到細胞的「回收中心」。例如，目前可以確定的是，p62蛋白可將聚集的蛋白質和破舊的線粒體（細胞的動力裝置）選擇性地輸送到回收中心。

為了更好地了解p62蛋白在細胞再生和長壽中的作用，科學家使用了短壽命，透明的秀麗隱桿線蟲進行研究。先前，Hansen的團隊發現對線蟲進行短暫的熱休克後，p62蛋白的水平會升高。事實證明，輕度熱休克對線蟲是有益的，這可以使他們更長壽。

在這一前提下，科學家對秀麗隱桿線蟲進行了基因工程改造，以生產出過量水平的p62蛋白。結果發現，這些線蟲超過了通常只有三周的壽命，生活了一個月，這相當於延長了20％到30％的壽命。研究人員甚至欣喜地發現，通過增加p62蛋白的水平，「回收中心」變得更加富餘，能夠回收更多的「垃圾」，這表明p62蛋白推動了整個回收過程。

自噬受體p62 / SQST-1 通過誘導自噬促進線蟲的蛋白穩定和長壽 圖片源自《Nature》

「現在我們已經確認選擇性自噬對於長壽很重要，我們將繼續進行下一步，即確定它能去除哪些有害的細胞『垃圾』。基於此，我們希望找到影響長壽的危險因素的特定細胞成分。」Hansen實驗室的研究助理教授，該研究的主要作者Caroline Kumsta博士說。

許多與年齡有關的疾病都是由有毒、錯誤摺疊的蛋白質引起的，包括阿爾茨海默氏病和亨廷頓氏病。Hansen和Kumsta先前的研究表明，在患有亨廷頓氏疾病的秀麗隱桿線蟲的模型中，p62蛋白水平的提升能夠延長壽命。科學家們希望通過研究選擇性自噬，可以找到清除那些有害於健康長壽的蛋白質的療法。

據數據統計，十年後大約20％的美國人（約7100萬人）將65歲以上，罹患慢性病的風險更高。隨著美國人口的老齡化，尋找可能的與衰老有關的疾病的治療途徑變得越來越重要。

p62蛋白的水平增加對延長壽命具有很大的潛力，這鼓勵科學家們進行下一步的研究。然而，最近對果蠅的研究發現，p62蛋白似乎在進化上是很保守的，這警示他們不能將這種方法直接用於人類。因為已有研究證明，高水平的p62與人類癌症有關。

「鑒於p62和癌症之間的已知聯繫，從頭到尾了解選擇性自噬過程更為重要。有了這些信息，我們也許能夠增強p62在選擇性自噬中的有益功能，並找到促進健康、治療衰老的療法。」Hansen博士說。#健康科普大賽# #2019算數健康大會# #我要上頭條#
