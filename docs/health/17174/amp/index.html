如何控制高血壓遠離腦出血？來看看醫生提出的4個注意事項

天天要聞
2019-12-14

高血壓如果長期不控制，會讓心腦血管長期承受過高的壓力，加速心腦血管的硬化問題，有些長期高血壓的朋友，血壓以及其他相關基礎疾病的沒有做好控制，腦動脈硬化較為嚴重，如果在某些因素影響下，血壓出現驟然升高的情況，就會出現腦出血的風險，腦出血一旦發生，死亡率高，預後也往往較差，因此，做好血壓的控制，做好腦出血的預防，確實是非常重要的事情。

在談注意事項之前，還是要說一下防大於控的問題，如果能夠在出現高血壓，高血脂等問題的初期，就積極的進行干預控制，需要用藥的情況下合理用藥，做好血壓、血脂、血糖的控制，同時生活上注意合理膳食，控制體重，戒煙限酒，加強運動，通過生活方式干預+合理用藥的方式，把這些相關慢病以及不良生活習慣對心腦血管的影響降到最小，減緩腦動脈硬化速度，控制血壓平穩，才是真正減少腦出血風險的最佳方法。如果等到慢病不控制多年後，已經明確的有了腦血管硬化的問題，血壓也更加的難以控制的時候，再去控制血壓平穩，減少腦出血風險，雖然通過合理的用藥和生活干預，也能夠有降低腦出血風險的效果，但其風險性比起早預防早干預的朋友來說，還是要大的很多了！

對於高血壓患者來說，控制血壓主要是考慮2個方面，一是達標，二是平穩。達標我們說了很多了，基礎目標是140/90以下，身體情況較好的中老年朋友，可以控制到130/80以下則更佳，而在控制血壓達標的同時，能夠平穩的把高血壓控制住，保持血壓不產生劇烈的波動，也是減少心腦血管疾病風險的重要方面。

想要控制血壓平穩，避免血壓劇烈波動引起腦出血風險，在生活上，高血壓患者要避免以下這些事項——

1. 平和心態，避免情緒劇烈波動

很多人會忽略情緒對於血壓的影響，但實際上，這是控制血壓平穩的最重要方面之一。生氣、激動、緊張或其他的一些激烈情緒影響，會刺激身體的神經中樞，導致血管緊張，血壓驟升，引起硬化的腦血管的出血風險。因此，不管是在輔助控制血壓，還是預防高危患者腦出血風險的情況，不管是飯前還是飯後，高血壓患者都要控制好自己的情況下，保持平和樂觀的心態，不以物喜不以己悲，良好的心態，是血壓保持平穩的前提。

2. 避免用力排便

很多朋友受到便秘問題的困擾，但如果有便秘問題，一定要積極的進行調理。如果本身就有高血壓的問題，在便秘用力排便時，會使血壓有一個驟然升高的過程，在晨峰高血壓期間，因過度用力排便血壓進一步升高，而導致腦出血的病例也並不少見，因此，高血壓患者想要保持血壓平穩，減少心腦血管疾病風險，也應該注意避免過度用力排便。

3. 避免劇烈運動

如果是飯後，那就更要避免了，飯後如果接著進行劇烈運動，不但不利於心血管健康，也不利於消化系統的健康。有高血壓問題的朋友，我們提倡加強運動鍛煉來加強血壓的控制調理，但一定要注意在控制好血壓的前提下，如果不控制血壓，而運動強度又過大，在劇烈運動的過程中，心跳加速，血壓進一步飆升，腦出血風險就會進一步加大，得不償失。因此，高血壓患者想要減少腦出血風險，一定要注意在運動時控制好血壓，也要控制好運動強度，結合自身情況，以中等強度的有氧運動為主即可。

4. 避免飯後立即洗澡

飯後身體為了加強食物的消化吸收，會有更多的血液供應到胃腸道的消化系統，如果此時洗澡，通常身體會反射性的收縮血管以保持各個臟器的血壓和血供，而此時如果去洗澡，洗頭時熱水會導致頭部血管的血液循環加速，腦血流速度增加，加大腦出血風險。因此，高血壓患者要避免飯後立即洗澡，如果需要洗澡，可以在餐後1.5~2小時後進行，如果有較高腦出血風險的朋友，也要注意調節好水溫，不要過熱，先洗身體再洗頭，這樣能夠更好的減少腦出血風險。#清風計劃# #心臟英雄# #健康真社探#
