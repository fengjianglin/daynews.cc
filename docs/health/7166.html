冬季飲食有講究，這8道月子餐特別適合冬季吃，趕緊收藏

2019-11-26 510條評論 2,365次閱讀 803人點贊

春生、夏長、秋收、冬藏，冬季是一個主「收藏」的季節，冬季的飲食大多以滋陰潛陽、增加熱量為主，那麼冬天的月子餐有什麼講究呢？吃什麼更好？今天十三姨就好好跟大家分享一下。

冬季月子餐要點

1、補氣補血為主

冬天坐月子飲食應以補氣補血為主。這樣可以保暖身體，有助於子宮收縮。

可多食用溫補的食物，因為食用溫補的食物可促進血液循環，達到氣血雙補的目的，而且筋骨較不易扭傷，腰背也較不會酸痛。

2、多補水、重滋養

冬季氣溫低，再加上新媽媽的活動量較小，因此排汗量也減少。但是大腦和身體各器官都需要補充大量水分，以保證正常的母乳及新陳代謝，所以需要多補水，

此外，由於氣候乾燥，還要多吃潤肺滋陰的食物。

3、多補鈣和維生素

冬季坐月子，還要記住一點就是要補充維生素D，以促進對鈣的吸收。如果媽媽缺鈣嚴重，容易導致骨密度降低，從而出現骨質疏鬆、腰酸背痛、牙齒鬆動等癥狀。

推薦食譜

下面推薦的這8道月子餐，非常適合冬季吃

山楂白蘿蔔煲

食材：

羊肉500克，白蘿蔔300克，山楂20克，姜、鹽、香菜葉各適量。

做法：

  1. 羊肉洗凈、切塊，放入沸水中汆燙一下、撈出、洗凈血水。
  2. 白蘿蔔去皮、洗凈、切塊備用。
  3. 姜洗凈、切片備用，山楂洗凈。
  4. 在沙鍋里放入羊肉、白蘿蔔、山楂、薑片，加入清水(水量以淹沒材料為宜)，用大火煮沸後，改小火煮約1小時。
  5. 加入鹽、香菜葉調味即可。

蓮子薏米煲鴨湯

食材：

鴨子1隻、蓮子、百合、薏米、蔥姜、鹽、雞精、料酒、白糖。

做法：

  1. 鴨子切塊，放開水中焯一下，撈出後放入煲中。
  2. 鍋中依次放入蔥姜、蓮子、百合、薏米，加入少許料酒，白糖，雞精，倒入適量開水。
  3. 待湯煲好後出鍋時加少量鹽即可食用。

黃花豬心湯

食材：

黃花菜20克，小油菜50克，豬心半個

做法：

  1. 豬心洗凈，放入開水蒸燙，然後撈起放入涼水中用手擠壓去血水，反覆換水直到去凈血水。
  2. 去凈血水的豬心加3碗水，大火燒開後轉小火煮約15分鐘，然後取出切薄片。
  3. 黃花菜去蒂、泡水洗凈，小油菜洗凈備用。
  4. 用兩碗水，加入黃花菜煮，水燒開後將小油菜、豬心片放入略煮，加鹽調味即可食用。

草菇炒山藥

食材：

草菇300g，山藥300g，黑木耳100g，食鹽、蔥、植物油、生粉適量。

做法：

  1. 草菇清洗乾淨後對半切。
  2. 黑木耳提前浸泡好；山藥去皮洗凈切片備用。
  3. 鍋中加入適量的食用油，再加入上面所有的食材翻炒一下。
  4. 用生粉勾薄芡，撒上蔥花再淋上香油即可出鍋。

蘿蔔燉牛肉

食材：

牛腱子肉、蘿蔔、大蔥。

做法：

  1. 牛腱子肉洗凈切成小塊，蘿蔔洗凈去皮切成小塊，大蔥洗凈切成小段備用。
  2. 鍋內加入少量的水煮開放入牛腱子肉塊焯水
  3. 高壓鍋內放入適量的熱水，依次放入焯水後的牛腱子肉、蘿蔔、蔥段
  4. 最後加入料酒、老抽、蚝油煮30分鐘即可，出鍋前加入適量的鹽

大棗豬腳花生湯

食材：

豬腳1隻，花生250克，黃芪100克，米酒2瓶，紅棗150克，當歸20克。

做法：

  1. 將豬腳切成塊狀，先以滾水汆燙，撈出後再浸泡於冷水中。
  2. 花生浸泡於冷水中4小時，再入鍋煮至略軟。
  3. 將豬腳、黃芪、紅棗及花生同時放入燉鍋中，並且加入米酒及水，蓋過所有材料。
  4. 燉至花生及豬腳熟爛，再加入當歸燜煮五分鐘即可。

榛蘑燉土雞

食材

土雞1隻，榛蘑100克。

做法：

  1. 將土雞洗凈切塊，榛蘑用溫水泡發洗凈。
  2. 鍋內加水，放入土雞塊，大火燒開，撇去血沫，加入薑片小火燉煮40分鐘。
  3. 加入榛蘑、枸杞，再燉20分鐘，調味即可。

乳鴿枸杞雲吞

食材：

乳鴿1隻，雲吞10個，枸杞5克，油菜適量。

做法：

  1. 將乳鴿洗凈、切塊，放入沸水中汆燙一下、撈出洗凈。
  2. 將處理好的乳鴿與適量蔥姜一起放入高壓鍋里煮熟。
  3. 從鍋里取出乳鴿湯肉，再加入適量清水煮開，之後下入雲吞、油菜、枸杞，大約煮3分鐘左右，調味即可。
