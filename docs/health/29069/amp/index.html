流感高發身體要扶正氣，中醫師教您做防感茶、防感粥

天天要聞
2019-12-22

時下流感高發，據國家衛計委發布的消息看兒科是重災區，說明兒童免疫力低下易於「中招」，另外，體弱的老年人群、原有多種慢性疾病者，也是易感人群，即使是青壯年，如果過度勞累免疫力下降也會有發病機會。因此，要想預防不病，平安度過這波流感危機，扶助正氣，增進免疫就顯得尤為重要！《黃帝內經》早有「正氣存內，邪不可干」之說，為此，江蘇省中醫院呼吸科史鎖芳主任醫師給各位讀者介紹些扶正防感舉措，以供大家預防流感！

黃芪、党參泡茶，補足氣、不流感

黃芪是重要的補氣葯，全身之氣皆能補益，尤能補益肺氣！它和党參均屬補氣良藥，党參（生曬參）偏重於大補元氣、回陽救逆，常用於虛脫、休克等急症；而黃芪則以補虛為主，常用於體衰日久、言語低弱、脈細無力的人。經常疲勞的流感患者，不妨喝這款參芪茶。

參芪茶

取生薑5片，蔥白5條，党參15克（或生曬參6克），黃芪30克。加水煎煮30分鐘即可。

該方具有補益肺脾、發汗解表的功效，用於體虛流感風寒證，有惡寒、頭痛、胃痛欲吐、四肢酸痛、鼻塞、體倦乏力等癥狀。早晨服用， 可連服7天。

家備生薑，吃出溫暖來

民間有「朝含三片姜，不用開藥方」,「冬有生薑，不怕風霜」,「冬吃蘿蔔夏吃薑，不勞醫生開藥方」,「家備小姜，小病不慌」等說法。而且在流感發燒時，很多人都會用生薑煮湯給患者喝，可以發汗，把寒氣逼出體外。

生薑辛溫，吃過後，人會有身體發熱的感覺，這是因為它能使血管擴張，血液循環加快，促使身上的毛孔張開，這樣不但能把多餘的熱帶走，同時還能把體內的寒氣一同帶出。所以，受寒流感後，生薑是首選，尤其適宜於當下寒濕型流感。

生薑泡茶

取生薑5克，紅糖20克，將生薑洗凈，打爛，放入鍋中，加水兩碗，大火燒開，加入紅糖，調勻即可。

具有辛溫解表的功效，可用於流感風寒證，症見惡寒發熱、鼻塞、流涕、打噴嚏等。早晚各1次，可連服用3天。

生薑煮粥

取生薑5片，蔥白5條，糯米50克。將糯米煮至將熟，加入蔥、生薑，再煮數分鐘即可。

具有發汗解表的功效，用於流感風寒證，症見惡寒、頭痛、胃痛欲吐、四肢酸痛、鼻塞等。早晨服用， 可連服7天。

姜蘇茶

取生薑、蘇葉10克，紅糖15克。先將生薑洗凈搗爛，蘇葉洗凈，共放鍋中，加水適量，小火煎開，加入紅糖調勻即可。本方具有祛風解表、溫散和胃的功效，可用於風寒流感，症見惡寒、頭痛、四肢酸痛、鼻塞、流清涕、噁心、胃部不適等。每日2次，早晚溫服。半月為1個療程。
