冬，一個喝“熟普”的季節（這些人必看）

2019-11-11 1,375條評論 3,725次閱讀 2640人點贊

春生，夏長，秋收，冬藏。轉眼氣溫驟降，你的茶杯續滿健康了嗎？冬天，不僅是進補的季節，人體也易出現各種虛弱、乾燥的癥狀。清代宮廷就有冬喝普洱夏喝龍井的傳統，龍園君覺得這個時期，就應該順時節、養生進補，飯後來上一杯熟普洱解油膩，怡養身心，潤肺解燥。

冬季喝熟普有哪些好處？

1、消食、解膩

發酵過程形成多種有益菌群，菌群作用可以減少小腸對甘油三脂和糖的吸收、提高酵素分解腰腹部脂肪，同時還能改善腸道微生物環境，具有順腸胃的功能。

2、暖胃、養氣

含有豐富有益菌群的發酵熟普洱茶,在進入人體後不會對胃產生刺激作為，而且能夠在胃的表層形成附著膜，對胃不好的人有保護作用。熟茶性溫味甘，能夠聚集體內陽氣，使身體發熱起到暖胃作用，還能讓人體增強對寒冷氣候適應能力。

3、提高免疫力

熟普洱茶有茶紅素、茶黃素、茶褐素（TB）、沒食子酸和維生素C等，這些物質對提高人體免疫系統功能發揮重要的作用，更可以有效預防感冒。

4、補充微量元素

根據云南省權威機構發布的報告顯示，每500毫升普洱茶中，所含人體所需的礦物、微量元素達三十種之多。多酚類物質、蛋白質、維生素、微量元素鐵、鉀、磷、鈣、鎂、錳、等。這些元素對人體作用極大，所以在秋季飲用普洱茶可以補充人體元素從而達到養生保健功效。

哪些人更適合喝熟普？

1、體寒之人：平時怕冷，四肢不溫，喜熱飲，體溫常偏低，冬最宜溫熱性茶品。

2、腸胃問題之人：現代都市人，由於煙酒、熬夜、進食不規律等多種原因，很多腸胃不好。飲用溫順平和的熟普，能起到護胃、養胃的作用。

3、老年體弱之人：現代普洱茶現代工藝研究，證實熟茶有血管舒張、血壓暫時下降和腦部血流量減少等功效，發酵加工過程中微生物作用產生多種物質，對提高人體免疫系統發揮了重要作用，故老年人體弱者很適宜飲熟茶。

4、減肥愛美之人：普洱茶臨床試驗證明其對減少類脂化合物，膽固醇含量有良好效果。昆明醫學院也對普洱茶治療高脂血病的療效作了55例臨床試驗，並與降脂較好的藥物安妥明治療的31例作對比，普洱茶的療效高於安妥明。長期飲用熟茶有治療肥胖症及美容的功用。

5、辦公室之人：辦公室的白領用腦用眼較多，泡茶講究便利性。熟茶，甜滑苦澀味低，取適量，無論是玻璃杯，飄逸杯，甚至是保溫杯（要去蓋，少燜），都可輕鬆搞定一杯舒心好茶。
