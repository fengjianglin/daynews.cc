秋天身體最愛4種“白色食物”，照著吃清肺、潤燥、補脾、健胃

2019-10-19 251條評論 1,187次閱讀 143人點贊

隨著天氣漸涼，人們的食慾也逐漸提高。秋季陽氣日減，陰寒日生，且氣候乾燥，易傷肺陰，胃腸道抵抗力較差，不宜多吃生涼的瓜果，過食辛辣食物會加重秋燥，引起乾咳等疾病，因而應多選些中性食物，並循序漸進地選用一些食物進行調節，以平衡人體營養。

所謂中性食物是按食物的屬性來說的，即指性質較為平和的食物（不寒不熱的食物）。

中醫認為，食物的顏色與五臟有關，青入肝、赤入心、黃入脾、白入肺、黑入腎，正是因為白色食物具有補肺的作用，秋天適當食用白色食物對健康是有益的。

傳統中醫認為，常吃「白色食物」有清肺潤肺的功效。白色食物指的是像雪梨、銀耳、百合、金針菇、蘿蔔等表皮呈白色的蔬菜、水果和菌菇類食物。當然從現代營養學來說，或許觀點不同，但幾種應季的白色食物確實對身體有諸多好處。

白蘿蔔

白蘿蔔含水量高，幾乎可以達到90%以上，有生津止渴、清熱排毒等多種功效。現代醫學研究發現，蘿蔔之中含有芥子油以及澱粉酶和粗纖維，所以食用之後能夠很好地促進消化，並且對於止咳化痰也具有很好的作用。

中醫方面也認為，白蘿蔔性涼，在秋冬乾燥的季節，吃一些蘿蔔，確實可以達到補水、潤肺的效果。

白蘿蔔無論是直接食用或者是煮熟之後食用都是可以的。

  • 推薦美食：蘿蔔羊肉湯

取新鮮白蘿蔔一個，羊肉適量，將準備好的白蘿蔔洗凈切塊。加入羊肉燉熟趁熱食用即可。

羊肉大溫大補，可驅散寒冷、滋養肝臟，但羊肉吃多了容易上火，蘿蔔清火通氣，羊肉湯中加入蘿蔔，既減輕羊肉的熱性，有清涼作用，又可去掉羊膻味，而且還能解油膩，化痰瀉火，實在是絕配！

蓮藕

秋天，天氣乾燥，吃些藕，不僅可以清心安神，還能潤燥止渴。藕裡面含有大量的碳水化合物，蛋白質，身體所需的各種維生素，以及膳食纖維。膳食纖維能促進腸胃蠕動，對改善便秘癥狀來說非常不錯，它能有效促使有害物質排出，對身體健康來說非常有好處。

不管是生吃還是清炒，都是很好的做法。

  • 推薦美食：蓮藕燉排骨

百合

新鮮百合里含有鈣磷鎂等礦物質，和多種維生素、充足的水分，女性朋友常吃的話，能起到一定的美容抗衰老的作用。

中醫常用百合來清肺潤燥、滋陰清熱、理脾健胃。多食百合還可以寧心安神、清心除煩，對於調節身體節奏，保持良好心態也頗有裨益。

  • 推薦美食：百合蓮子粥

但吃百合時要注意，生百合里含有一種行名為秋水仙鹼的物質，具有很強的毒性。如果不將它除掉，很容易引起噁心嘔吐的癥狀，嚴重的甚至還會損傷肝腎。把百合烹飪熟後，才可以徹底消除這種有毒物質。

山藥

山藥有「食物葯」的功效，可以滋腎益精、健脾益胃、益肺止咳。補而不滯、不熱不燥，一般人群都適用。

山藥既可以作為蔬菜，又可以作為主食，是秋季首選的理想保健食物，其含豐富的澱粉酶、多酚氧化酶，對我們脾胃的消化吸收有很好的功效。

再就是，裡面含粘液蛋白，也就是山藥為什麼黏糊糊的，而這種粘液蛋白具有降低血糖的作用。因為山藥的營養豐富，自古以來就被視為物美價廉的補虛佳品。

山藥吃法多種多樣，作為秋補以煮粥為佳。

  • 推薦美食：山藥粥

山藥有養胃的作用，患腹瀉等腸胃疾病的人不妨多吃。但山藥有收澀作用，大便燥結者不宜食用。

白色食物里，還有一些適合減肥期間吃的，比如豆腐、茭白等。它們營養豐富、熱量低，富含纖維素。

這種食物纖維，作為人體的清潔工，幫助腸道排泄廢物，還為胃腸的蠕動助一臂之力，最顯著的效果就是如果你長期堅持，就再也不會為便秘而苦惱。

最後提醒大家：

有三種白色食物要少吃，那就是白酒、肥肉和食鹽，現在我國高血壓、高血脂等慢性病頻繁。跟飲食中高油脂、高糖高鹽、酒精攝入量偏高有關，大家要盡量避免。
