天天要聞
1 個月 ago

深秋陽氣逐漸內斂，寒氣上升，

一旦受寒氣入侵，容易傷及五臟。

中醫道：

秋天一碗湯，不用醫生幫！

在秋天經常喝湯，你會受益無窮！

秋天喝湯三大補

01 補肺

秋天80%的病是由呼吸道引起的，比如哮喘或者支氣管炎，秋天常喝湯能補肺潤氣，防止呼吸疾病，還能提高免疫力！

02 補腎

俗話說「一秋腎三虛」，一年當中，秋天最容易出現腎虛的季節，尤其對比較瘦弱的人，通過喝湯，可以滋陰補腎，減少疾病。

03 補脾

秋天易內燥，引發口乾眼乾，皮膚騷癢，這是脾虛的表現，脾能產生津液，潤澤全身，脾虛可以常喝湯來補充。

秋天一碗湯 不用醫生幫


蒸冬瓜配菌菇湯

-材 料-

冬瓜、乾花菇、海帶

黃花菜、干百合、鹽、生抽

-做 法-

1. 乾花菇、干黃花菜、海帶、百合泡發。

2. 海帶泡發後洗乾淨，切成粗細長短差不多的細絲。

3. 香菇百合清洗乾淨，直接放入燉鍋中。

4. 放入料理好的黃花菜、海帶絲。

5. 加適量水大火燒開後轉小火熬煮3小時左右。

6. 煨湯最後幾分鐘，放鹽和生抽稍煮片刻關火備用。

7. 冬瓜切開剝去籽，切大塊，蒸出來吃著很過癮，煮好的香菇涼置後切小塊。

8. 所有食材擺盤，加少許牛肝菌油。

9. 放入蒸鍋中，繼續蒸30分鐘左右，冬瓜熟透即可。


蓮藕板栗湯

-材 料-

蓮藕、板栗、核桃仁

桂圓、百合、生薑

-做 法-

1. 板栗用刀去殼，去殼的板栗連皮倒入冷水中煮沸。

2. 煮開後放入提前準備的涼水中，經過熱脹冷縮皮很容易剝下來。

3. 準備一碗熱水，將核桃仁放入碗里浸泡10分鐘左右，核桃皮就很好去除。

4. 干桂圓兩手用力捏碎，去除外皮。

5. 蓮藕洗凈去皮，斜著切小塊。

6. 生薑切成薄片，準備好所有材料。

7. 準備一個鍋倒入適量水，下入蓮藕、板栗、生薑，大火煮沸，加一勺花生醬。8. 煮開後放入干桂圓和百合。

9. 中火煮30分鐘左右。

10. 最後放入核桃仁大約煮2-3分鐘，起鍋前放入枸杞。


素煲湯

-材 料-

紅棗3顆、蟲草花 一小把

眉豆、胡蘿蔔、花生適量

鹽、鮮蔬粉適量、姜1片

-做 法-

1. 食材洗凈，生薑切片，胡蘿蔔切塊

2. 以上材料加入燉盅，隔水燉1.5小時

3. 出鍋前加鹽和鮮蔬粉調味即可


素羅宋湯

-材 料-

西紅柿 2個、土豆 1個

橄欖油 10g、椰菜 150g

鹽 3g、鮮蔬粉 1茶匙

番茄醬 2湯勺

-做 法-

1. 鍋里加入橄欖油，再加入番茄，煮至水份蒸發

2. 出番茄皮，加入番茄醬翻炒

3. 加入切好的椰菜拌勻

4. 加入1升水；加入切丁的土豆，蓋蓋中火煲15分鐘

5. 加入鹽、鮮蔬粉，試味調整酸度即可熄火

6. 吃的時候可以灑上黑胡椒碎或滴些辣椒仔

蘿蔔蓮藕素湯

-材 料-

西紅柿 2個、山藥 半根

土豆 1個、香菇5朵

胡蘿蔔 1根、蓮藕 一段

玉米 半根、花生米 10粒

豌豆 適量、生薑5片

海鹽、黑胡椒粉 適量

亞麻籽油/食用油 少許

-做 法-

1. 將所有主料切成小塊備用。

2. 鍋中倒入少許亞麻籽油（可以用食用油代替），將切好的西紅柿倒入，小火熬成西紅柿醬。

3. 將熬好的西紅柿醬倒入砂鍋中（沒有砂鍋用普通能熬湯的容器也可以，需要有蓋子），加入生薑和大量的水。

4. 將準備好的所有食材全部倒入砂鍋中，大火燒開後，調至中小火，蓋上鍋蓋煲1小時。

5. 高湯新鮮出爐，加入海鹽即可，也可以根據自己口味加入些黑胡椒，原汁原味就很鮮美了噢


菌菇湯

-材 料-

白蟹味菇70克、灰蟹味菇70克

干杏孢菇10克、姜3克

香菜、鹽適量、白鬍椒少許

-做 法-

1. 干杏鮑菇洗凈用清水泡發2小時以上，改刀切成條狀。

2. 白蟹味菇和灰蟹味菇洗凈，用淡鹽水浸泡10分鐘後瀝水備用。

3. 鍋里放油爆香薑末，輕炒杏鮑菇，加入雙色蟹味菇。

4. 加入適量清水大火煮開，轉小火燉30分鐘。

5. 加入鹽、白鬍椒調味，撒香菜出鍋。

海筍玉米蘑菇湯

-材 料-

玉米1根，海筍10克

姜2片，胡椒粉1小勺

海鮮菇，白玉菇

口蘑，鹽適量

-做 法-

1. 玉米切段，口蘑切成薄片備用。

2. 將各種菌菇放入沸水中略焯後撈出備用。

3. 將切段的玉米放在砂鍋中，大火煮開。

4. 加入海筍和薑片，再次煮開。

5. 轉小火，加入焯好的菌菇。

6. 燉約二十分鐘左右，加入鹽調味。

7. 再略燉一會，加入少許胡椒粉即可食用。

養身四物湯

-材 料-

紅棗100g，干桂圓120g

枸杞15g，干蓮子50g

-做 法-

1. 把紅棗和蓮子先放入砂鍋燉煮，水開後，小火

2. 20分鐘後，加入干桂圓，燉煮30分鐘左右。試一下是否已經熟透。

3. 加入枸杞，燉5分鐘，關火即可。

4. 清熱安神，補氣益血，孕婦禁

蓮藕花生湯

-材 料-

蓮藕，綠豆，花生

冬菇，姜，當歸兩片

鹽，蔬果調味粉

-做 法-

1. 綠豆洗凈泡上四個小時左右，蓮藕洗凈去皮

2. 將蓮藕的頭切去備用，蓮藕內孔沖洗乾淨

3. 將泡好的綠豆填入蓮藕中，一定要壓實哦

4. 蓋上剛切下的蓮藕頭，用牙籤固定好

5. 冬菇泡發，花生洗凈泡上一個小時左右，姜切片；

6. 鍋內加入足夠的水，將釀好的蓮藕、冬菇、花生、姜、當歸一起加入

7. 大火煲開後轉小火煲一個小時，撈出蓮藕，切成約1cm厚的片片

8. 再加入湯中，這時同時加入鹽、蔬果粉，再煲二十分鐘左右，大功告成！

陸豐五果湯

-材 料-

紅豆，芡實，蓮子

薏米，桂圓，冰糖

-做 法-

1. 把準備好的紅豆、薏米、蓮子、芡實洗凈，接著再浸泡3小時

2. 3小時後便可把紅豆、薏米、蓮子、芡實的水分瀝干，再撈起放至砂鍋中，倒入清水煮半小時

3. 加入冰糖，桂圓，攪拌均勻，再煮十分鐘即可


番茄南瓜湯

-材 料-

南瓜300克，番茄2個

黃瓜1根，青豆100克

胡蘿蔔1根，芹菜100克

姜適量，鹽少許

-做 法-

1. 番茄切小塊，南瓜切小塊，芹菜切段，可以保留一點芹菜葉子，黃瓜切小塊，胡蘿蔔切丁，豌豆焯水，姜切片

2. 鍋底放油，油熱炒香姜

3. 加入胡蘿蔔丁，翻炒一分鐘

4. 加入番茄，翻炒一下

5. 加入芹菜，翻炒一下，放水沒過所有的菜，加鹽

6. 水開加青豆，南瓜和黃瓜，繼續煮半個小時就可以了

蘋果銀耳羹

-材 料-

銀耳半朵，大蘋果1個

紅棗10克，枸杞5克，冰糖適量

-做 法-
1. 銀耳用適量的水泡發，泡發好後的銀耳去除蒂頭，清洗乾淨後 撕成小塊

2. 紅棗、枸杞洗凈

3. 把銀耳放入鍋中，加入適量的水，中火煮40分鐘

4. 銀耳塊煮好的時候，醬蘋果去皮、去核，切成小塊

5. 加入蘋果塊、紅棗，繼續煲10分鐘左右

6. 10分鐘後，加入枸杞、冰糖，攪拌均勻，蓋蓋子繼續煲2分鐘即可關火

紅豆蓮藕湯

-材 料-

紅小豆100克

蓮藕300克、冰糖適量

-做 法-

1. 紅小豆淘洗乾淨放入電飯煲，加入足量的水

2. 蓮藕去皮切丁，沖洗乾淨

3. 切好的藕丁和冰糖一起放入電飯煲中

4. 啟動湯粥程序

5. 程序結束，一份簡單潤燥的蓮藕紅豆湯就完成了

芡實薏米銀耳羹

-材 料-

銀耳半朵，薏米30克

芡實25克，枸杞、冰糖適量

-做 法-

1. 銀耳事先用冷水泡開，去掉黃蒂，撕成碎朵

2. 芡實、薏仁泡水1小時以上洗乾淨。

3. 洗乾淨後，加上銀耳，入鍋。大火煮開後，轉小火煮1.5到兩小時。

4. 關火前10分鐘加入適量冰糖。

5. 枸杞入鍋關火燜一下即可。

冬瓜素丸湯

-材 料-

素丸、姜、枸杞

冬瓜、胡椒粉、鹽、香菜

-做 法-

1. 準備好素丸，香菜切末

2. 冬瓜切小片，將鍋燒熱，倒入薑片、枸杞和切好的冬瓜片，加水燒沸。

3. 燒沸後，倒入素丸，蓋上鍋蓋再煮10分鐘左右。

4. 煮好後調入鹽、胡椒粉、香菜末攪勻即可。

絲瓜豆腐湯

-材 料-

南豆腐200克，絲瓜1根

姜5克，鹽適量

-做 法-

1. 絲瓜去皮洗凈切成滾刀塊；南豆腐切成小方塊；姜去皮切絲

2. 熱鍋冷油，加入薑絲炒出香味

3. 加入絲瓜、少許清水，煮開後轉小火將絲瓜煮至7成軟

4. 煮好加入豆腐，繼續將食材煮熟

5. 最後嘗下鹹淡加鹽調味拌勻即可

銀耳雪梨湯

-材 料-

雪梨、銀耳、冰糖、枸杞

-做 法-

1. 將銀耳泡在水裡，泡發後，去掉根部，並清洗。

2. 將銀耳倒入鍋中，加上清水和冰糖煮沸。煮沸後轉小火。

3. 15分鐘後，將雪梨切塊，倒入鍋中，並加入枸杞，繼續小火煮10分鐘左右。

4. 一碗銀耳雪梨湯煮好啦， 除了冰糖雪梨化痰止咳，我們又get了一種銀耳雪梨湯，同樣有這種功效。


百合綠豆湯

-材 料-

綠豆150g，百合1個

冰糖60g，枸杞少許

-做 法-

1. 將綠豆洗凈，瀝水後放入高壓鍋，加入適量開水壓一刻鐘

2. 把百合洗凈剝片，入水浸泡半小時

3. 取一小鍋，倒入綠豆湯和百合，再倒入適量開水，加入冰糖燒開後轉小火半小時就可以啦

4. 乘熱把枸杞放入就可以品嘗啦，一次不建議食用太多，因為性寒


紅糖生薑番薯糖水

-材 料-

番薯400g，紅棗8粒

紅糖適量，生薑4片

-做 法-

1. 紅薯去皮切塊，將清水沖洗幾次，姜去皮切片。

2. 紅棗洗凈後去核一切為二。

3. 鍋中加足量水，放入薑片、紅棗、紅薯塊，大火煮開後轉中小火煮30分鐘

4. 30分鐘以後，加入紅糖，再煮5分鐘左右，糖融化就可以了

紅棗南瓜百合甜湯
-材 料-

紅棗15克，南瓜1塊

鮮百合4個，干桂花適量

-做 法-

1. 紅棗用工具去核洗凈備用

2. 南瓜去皮切塊備用

3. 鮮百合去根去黑邊，洗凈泥土後瀝干備用

4. 將所有材料放入壓力鍋內膽中，加足水

5. 啟動電壓力鍋，以雜糧粥模式，壓力將至最低，燉煮30分鐘即可。
