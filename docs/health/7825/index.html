皮膚最愛的維生素C，還有這麼多好處，你知道嗎？

2019-10-18 717條評論 2,338次閱讀 649人點贊

維生素C，人體不能自行製造，所以只能從食物和營養補品中獲取。今天就來聊一聊為什麼要補充維生素C？維生素C對人體有哪兒些好處呢？

維生素C，又叫抗壞血酸，是一種水溶性維生素。維生素C在我們人體中起著重要的作用。

1. 是組織生長及修補、腎上腺功能、健康牙齦所必需的抗氧化劑。

2. 預防有害的感染及癌症。

3. 增強免疫力。

4. 可以降低膽固醇及高血壓，預防動脈硬化。

5. 是膠原質形成的必需物質。

6. 是強抗氧化劑，它可以使致癌物失去作用，避免細胞產生癌變。

7. 維生素C和維生素E相互作用，在一起發揮的作用很大。維生素E負責消除細胞膜上的危險自由基，維生素C則負責打斷體液內的自由基鏈。

8. 是新細胞及組織製造生長必不可少的物質，還能預防濾過性病毒滲透到細胞膜。

缺乏維生素C時。

1. 身上的傷口不易癒合，血管變得脆弱，牙齦容易出血，極易導致壞血病。

2. 牙齒鬆軟、脫落，琺琅質容易被侵蝕而形成蛀牙。

3. 骨骼基本結構被破壞，礦物質流失。

4. 維生素C與骨折的癒合有密切的關係。缺乏維生素C，無法構成骨骼的膠原蛋白，打斷的骨骼便無法結合。

維生素C的功效我們也要了解一下。

1. 防治壞血症。

2. 可以美容養顏，避免皮膚在日晒後產生黑色素的沉澱，使肌膚美白。

3. 有效預防感冒、流行性病毒感冒、病毒性肝炎、癌症、自體免疫疾病及其他的病毒性疾病。

4. 促進干擾素的產生，干擾素可以強化細胞對病毒的抵抗力。

5. 有抗癌作用，有助於防止亞硝胺（致癌物質）的形成。

6. 治療牙齦出血。

7. 預防濾過性病毒和細菌的感染，增強免疫系統功能。

8. 增加對無機鐵的吸收。

吃什麼食物可以補充維生素C呢？

1. 針葉櫻桃。

2. 酸棗。

3. 鮮棗。

4. 野莧菜。

5. 辣椒。

6. 甜椒。

7. 番石榴。

8. 獼猴桃。

9. 苦瓜。

10. 西藍花。

現在市面上常見的有很多維生素C的營養補充劑也是我們可以考慮的。維生素C可以長期吃，但是要堅持，也可以隨時想起來隨時吃。

一些日常烹調的方法也會幫助我們保留蔬菜中的維生素C。

1. 長時間的燉煮會破壞蔬菜中維生素C，加熱對其破壞很嚴重。

2. 先洗後切。

3. 微波加熱可以更好的保留維生素C。

4. 水果的攪拌、榨汁、腌漬都會讓維生素C損失。最好生吃。

5. 選擇大火快炒或者油炸。
