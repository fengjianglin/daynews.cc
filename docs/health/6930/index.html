雞蛋海鮮放心吃，飲食均衡不怕膽固醇

2019-12-08 80條評論 4,065次閱讀 1444人點贊

說到膽固醇，您想到什麼？是雞蛋、內臟還是海鮮？不管腦海浮現的是哪種食物，我們總有一種不能吃進太多膽固醇的觀念，不過，近來越來越多的研究指出，其實飲食中的膽固醇，不只沒有想像中的可怕，還對人體相當重要；而且，許多高膽固醇的食物，像是雞蛋、全脂乳製品或魚油等，反而都富含許多營養，今天我們就來談談膽固醇。

膽固醇和心臟病的關係

過去認為攝取膽固醇是造成心臟病的主因之一，但是其實導致心臟病的因素還有很多種，包括身體發炎、壓力、高血壓或吸煙等。而且，雖然心臟病和膽固醇脫不了關係，但真正的兇手是運送膽固醇的脂蛋白，飲食中的膽固醇反而影響不大。有學者認為，氧化膽固醇在血液中的濃度，可能才是導致心臟病的主因之一。

飲食中膽固醇≠血液膽固醇

雖然吃什麼補什麼看似很合理，但事實是，飲食中攝取的膽固醇，跟血液中的膽固醇其實是兩回事。因為身體製造近8成的膽固醇，只有不到2成是由飲食中攝取，也因此，飲食中的膽固醇攝取量，對血液膽固醇濃度影響很小。

另外，約有40%較敏感的人，容易受到食物影響導致膽固醇濃度變化，一般認為這是因為某些基因所導致，但是，這群人雖然可能因攝取高膽固醇食物，而有較高的低密度脂蛋白(又稱壞膽固醇)，但心臟病的風險卻沒有因此增加，這是因為他們所增加的是大顆粒的低密度脂蛋白，並非小顆粒的低密度脂蛋白，而當大顆粒的低密度脂蛋白較多時，患心臟病的風險反而較低。

此外，人們在攝取高膽固醇食物時，高密度脂蛋白(又稱好膽固醇)的濃度也會上升，所以整體而言，不論是一般人或較敏感族群，飲食中的膽固醇對心臟病的患風險，影響並不大。

美國新飲食指南解除膽固醇上限

值得注意的是，美國舊版的飲食指南建議，每人每天的膽固醇攝取量應低於300毫克，但根據新公布的飲食指南，竟取消了每日膽固醇的攝取上限，也推翻了過去應控制雞蛋和海鮮攝取量的觀念。不過，營養吸收畢竟還是會有個體差異，因此人們如果在攝取高膽固醇食物後，發現有負面影響，還是應限制其攝取量。

蛋黃解禁糖尿病患應少吃

至於過去認為蛋黃的膽固醇較高，應避免食用過多的說法，但今年許多研究都指出，攝取蛋黃並不會增加心臟病的患病風險，甚至還能有助於改善好壞脂蛋白的比例，降低患病風險。

更有學者比較吃蛋白和吃全蛋，兩者對於膽固醇的影響，研究結果顯示，每天吃3顆全蛋的人，高密度脂蛋白的數量上升較多，且壞膽固醇的濃度也下降較多。不過，如果飲食較偏西式的糖尿病患者，應該特別小心雞蛋的攝取，因為有研究指出，糖尿病患如果吃雞蛋，可能會增加患心臟病的風險。

飲食均衡很重要

血液中膽固醇過高是心臟病的主因之一，但對大部分的人來說，飲食中的膽固醇其實和血中膽固醇濃度關係不大，而且，再加上目前的研究並沒有明確的證據證明，攝取膽固醇會增加心臟病風險；因此，人們不必再小心翼翼地吃各式美食，但仍建議人們，任何食物以適量、均衡攝取最好。
