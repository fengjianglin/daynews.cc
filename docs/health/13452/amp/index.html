別人都在補，就你沒食慾？食欲不振四大類型，應對方法各不同

天天要聞
7 天 ago

冬季到來，雖然不少因為暑熱而導致的身體問題逐漸偃旗息鼓，但不少新問題也開始出現，首當其衝的就是因為寒邪而導致的食欲不振。

冬季的食欲不振大致有四種類型：感受寒邪型、濕熱上火型、飲食所傷型、脾胃虛弱型。那麼，不同類型的食欲不振，在飲食方面都有哪些需要注意的呢？

一、感受寒邪型

胃是最怕冷的，冬季氣溫低，寒邪極易入侵，出現胃部的疾病。感受寒邪型的表現多見腹部不適，腹痛或腹脹，腹部怕冷，食欲不振，口淡，腹瀉或大便爛，得溫則緩解。

推薦：可以用生薑粥、紫蘇粥進行調理。

生薑粥

材料：生薑25克，秈米100克，飴糖150克。

做法：將生薑切末煮汁，入米煮粥，待粥熟時，加飴糖攪勻。煮沸即成。 空腹服食，取微汗。

作用： 消痰止咳，散寒止吐。適用於傷風感冒、鼻塞咳嗽、胃寒腹痛等。

二、濕熱上火型

外面天氣越來越冷，屋裡的溫度卻越來越高。到了冬天，不少人為了取暖而將暖氣、空調溫度開得很高，門窗緊閉，再加之飲食滋補肥膩，很容易上火，濕、熱一同侵犯人體。表現為頭暈頭痛，疲倦乏力，噁心嘔吐，無食慾或胃口差，煩熱，腹脹腹痛，小便黃，腹瀉或大便爛，舌紅苔黃膩。

推薦：癥狀較輕的可以用荷葉粥調理。

荷葉粥

材料：大米100克，荷葉30克，冰糖30克。

做法：取大米煮粥，待粥熟後加適量冰糖攪勻，趁熱將荷葉撕碎覆蓋粥面上，待粥呈淡綠色取出荷葉即可食用。

作用：清熱利濕，降血壓、降血脂。適用於濕熱而致的頭昏腦脹、胸悶煩渴、小便短赤等。

三、飲食所傷型

到了冬天，很多人喜歡「冬補」。大魚大肉，而且飲食過於辛辣、肥膩，或者暴飲暴食、飲食不定時，都會導致脾胃功能受損，出現食欲不振，噁心欲嘔等表現。

若是由於暴飲暴食，飲食過於肥膩引起的胃腸積滯，則出現食欲不振、噁心欲嘔，還會伴有大便不通或大便臭，口氣重等等癥狀。

推薦：輕者可食用山楂或者山楂糖水調理，癥狀重可以服用保和丸進行治療。

桂花山楂糖水

材料：山楂200克，冰糖80克。

做法：山楂洗凈、去核，鍋中放入山楂、冰糖，加水大火煮開，放入一勺糖桂花，小火熬煮10分鐘即可。

作用：消食健胃，行氣散瘀，化濁降脂。

四、脾胃虛弱型

有部分人素體脾胃功能較差，在冬季寒冷時，更是加重脾胃的負擔，出現脾胃虛弱型厭食症。它的表現是食欲不振，不知飢餓，疲倦乏力，面色萎黃，大便爛，容易出現消化不良。

推薦：用党參、五指毛桃、薏苡仁、淮山等等煲湯或者做菜，健脾益氣，如五指毛桃豬骨湯。

五指毛桃豬骨湯

材料：排骨500克，五指毛桃50克，土茯苓20克。鹽適量。

做法：五指毛桃、土茯苓洗凈，浸泡十五分鐘；豬骨斬塊，洗凈，氽水撈起；煮沸清水，放入所有材料，武火煮20分鐘，轉文火煲一個半小時，下鹽調味即可飲用。

作用：健脾益氣，清肝潤肺。

冬季對付厭食，我們還可以適當吃一點水果，蘋果是開胃水果之王，是最正氣的水果。或者喝一些檸檬茶，這樣也會使胃口好點。
