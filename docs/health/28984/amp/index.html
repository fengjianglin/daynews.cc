風天氣燥，夫妻不和都容易「上火」。「上火」為民間俗語，又稱「熱氣」，可以從中醫理論解釋，屬於中醫熱證範疇。

冬天易上火

一般認為「火」可以分為「實火」和「虛火」兩大類，

實火臨床表現：面紅目赤、口唇乾裂、口苦燥渴、口舌糜爛、咽喉腫痛、牙齦出血、鼻衄出血、耳鳴耳聾、癤瘡乍起、身熱煩躁、尿少便秘、尿血便血、舌紅苔黃、脈數。

虛火臨床表現：多因內傷勞損所致，可進一步分為陰虛火旺和氣虛火旺（氣虛內熱）兩型。陰虛火旺多表現為潮熱盜汗、形體消瘦、口燥咽干、五心煩熱、躁動不安、舌紅無苔、脈細數。氣虛火旺常見癥狀有全身低熱、午前為甚、畏寒怕風、喜熱怕冷、身倦無力、氣短懶言、自汗不已、尿清便溏、脈大無力、舌淡苔薄。以五臟六腑為綱進行劃分，上火主要有肝火、心火、胃火、肺火、腎火。

1、經常熬夜引發第一把火：肝火

愛發脾氣

典型癥狀：

頭痛頭暈、睡眠不穩、愛發脾氣、甚至突發耳聾、小便少且黃、大便干、舌紅苔黃。

追劇熬夜、工作熬夜、看球賽熬夜、玩手機熬夜等，這些現象都會打亂人體正常的作息時間，加重我們肝臟本身的負擔，肝火也就隨之「點燃」。

因為肝臟不能按時休息和排毒，於是只好加班加點將體內積存的雜質和毒素清除。

【防火措施】

防肝火，睡眠是很重要的一點，每天晚上12點前盡量入睡，如果做不到不熬夜，後面一定要補足睡眠。

日常可以選擇菊花、決明子泡茶喝，蔬菜里的苦瓜、馬齒莧等都是清肝火的，可以常吃。

太沖穴

防肝火主要穴位：期門 、太沖 。

2、思慮過度引發第二把火：心火

口腔潰瘍

典型癥狀：

口腔潰瘍、咽痛咽干、失眠（中醫說「勞心思慮，過耗其真」，大致意思是太勞心了，就容易傷神。）、胸中感覺熱、口渴、臉色發紅、口舌生瘡、小便黃、舌尖發紅等。

心火有虛實之分。虛火表現為反覆口腔潰瘍、低熱、盜汗、心悸、心煩、失眠、健忘等，有心血管病的老年人常有此類癥狀。實火則表現為小便短赤、心煩易怒等，身體健壯的中青年人容易出現。

【防火措施】

控制情緒，減少緊張，少生心事煩事。

可以選用蓮子心、金銀花、菊花、淡竹葉等清心火中藥，單選一種來泡水，代茶飲用。平日也可以多吃點綠豆，有去心火的作用。

神門穴

防心火主要穴位：心俞、神門 。

3、飲食失調引發第三把火：胃火

長痘痘

典型癥狀：

長痘痘、牙齦腫、口角爛、輕微咳嗽、胃口不好、便秘、舌紅、少苔。

【防火措施】

既然火氣是吃出來的，那麼對付也要從吃這個源頭上抓起。多吃粗糧可以預防胃火。貪吃辛辣、燒烤等刺激性食物，會使脾胃受損，內蘊化熱，嚴重者口角會出現糜爛。

大家熟悉的金銀花，可以清脾胃之火，選其來泡水，代茶飲用即可。另外，也可以選用蒲公英和綠豆煮湯喝。

足三里

防胃火主要穴位：中脘、足三里 。

4、受涼或抽煙引發第四把火：肺火

咽喉干疼

典型癥狀：

咽喉干疼、口鼻乾燥、鼻炎反覆發作、咳痰時一般是黃痰且黏稠、舌紅苔黃等。

一些小孩睡覺時蓋不住被子，喜歡俯卧，喜吃冷飲，口唇發紅，也是肺火旺的表現。

【防火措施】

除肺火可用呼吸咳嗽洗肺法，通過深呼吸和主動咳嗽，幫助呼吸道排出分泌物，增強免疫力。

可選擇空氣新鮮的地方，反覆進行吸氣呼氣，盡量排出肺內氣體，另外每天主動咳嗽幾下，也是積極的保護性反射動作，能有效清潔肺部。

常用的清肺熱食材有橄欖、羅漢果、胖大海，可以單選一種來泡水，代茶飲用。平時可多吃點牛蒡、梨、百合、薄荷等食物，有清肺熱的作用。

少商穴

防肺火主要穴位：少商 。

5、陰虛火旺引發地第五把火：腎陰虛

腰膝酸軟

典型癥狀：

手心、足心、心口熱，部分人只是下午或晚上有點低熱，晚上醒後出汗（盜汗）。同時，還伴有咽干口燥、飲水多、大便乾結、尿黃量少、面部顴紅、煩躁不安等表現。

中醫診病的過程中，腎陰虛多為陰虛火旺的虛證，五心煩熱、潮熱盜汗、頭暈目眩、耳鳴、耳聾、牙疼尤其夜間疼得厲害但牙齦不腫，或有反覆發作的口腔潰瘍，還有尿路感染，腰膝酸軟等癥狀都是腎陰虛的表現。

【防火措施】

養生原則是補陰清熱，滋養肝腎。可選用滋陰生津中藥如女貞子、山茱萸、枸杞子、旱蓮子、知母等，選擇一種泡水喝，代茶飲用。

湧泉穴

防肺腎陰虛主要穴位：湧泉。

生活中要注意勞逸結合，飲食上要注意多吃富含維生素的蔬菜水果，多喝水，少吃辛辣煎炸食品，少抽煙喝酒。上火和心理狀態也有密不可分的關係，保持樂觀積極的生活態度其實是最好的滅火劑。
