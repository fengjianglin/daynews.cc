1碗熱粥，3個動作，從冬天開始補養腎氣

2019-12-18 207條評論 1,872次閱讀 321人點贊

寒來暑往，秋收冬藏，立冬之後，陽氣潛藏，陰氣盛極，寒邪當令，萬物閉藏。這時候，只有順應冬氣，將陽氣好好養藏起來，才能為來年春天的生機勃發打好基礎。

對於女性而言，冬天「藏」得好，來年更能氣血足，美麗又健康。從立冬到立春，養藏的關鍵在於：護陽、斂陽，就是要保護陽氣，不讓陽氣被過分消耗，並斂藏陽氣。

護陽的關鍵：不受寒

冬季，寒氣當令，避免受寒是養護陽氣的重要一步。立冬後，北方地區越來越冷，南方地區寒意正悄然而至。

寒邪傷人，最傷陽氣，如果沒有及時保暖，身體就會調動陽氣來抵禦寒邪的侵襲，消耗陽氣。

如果一直不重視避寒保暖，對陽氣是極大的損耗，一到秋冬，就會比其他人更容易手腳冰冷、反覆感冒，抵抗力比較差。

立冬後，女性尤其要重視避寒就溫，不要受寒。

✔ 及時添衣

立冬後氣候變化比較大，穿衣要薄厚適度，天冷及時添衣。穿得太少、太薄，容易受寒，損耗陽氣；穿得太多、太厚，會使皮膚腠理開泄，既不利於陽氣潛藏，又容易讓寒邪入侵。

✔ 養護背部

保持背部溫暖，是一個既簡單又重要的保護陽氣的方法。

背部的督脈是人體陽經的總督，是陽氣的「發電廠」；兩邊的足太陽膀胱經是抵禦寒邪和保護陽氣的「第一道防線」。

1、重視背部保暖，天冷後，可以穿件背心，保護背部不受寒；

2、洗澡的時候，用熱水沖一下背，幫助陽氣升發；

3、出太陽的時候，出去走走，晒晒背，既能溫陽，又能讓整個人暖和起來，非常舒服。

斂陽的關鍵：補養腎

腎主藏，腎氣充足，才能「養精蓄銳」，更好地斂藏。《素問·上古天真論》中記載：「腎者主水，受五臟六腑之精而藏之。」冬主腎，陽氣斂藏在腎水之中，養腎就是在幫助陽氣斂藏。

腎中精氣化生的陽氣，是最強大的「元氣」。腎是氣血的「大本營」。腎中陰陽被稱為「元陰」和「元陽」，是人體陽氣和陰血津液的根本。

立冬後，滋養腎精，最能幫助來年的陽氣生髮；反之，「冬不藏精，春必病溫」，封藏不足，來年氣血更不足，更容易生病。

輕鬆養腎的方法

1、摩擦腰部。「腰者，腎之府」，摩擦腰部至微微發熱，對養腎有一定作用；

2、揉耳朵。「腎氣通於耳」，揉捏耳廓和耳垂對補腎氣有一定作用；

3、叩齒。「腎者，主身之骨髓及齒」，上下牙相叩，有益腎的功效。

補腎益腎的食物

✔ 芝麻：補肝腎、潤五臟，《本草經疏》稱其：「氣味和平，不寒不熱，補肝腎之佳谷也。」

✔ 栗子：補中益氣，濃腸胃，補腎氣。

✔ 豇豆：補腎健脾，《本草綱目》稱其：「理中益氣，補腎健胃，生精髓。」

✔ 淡菜：補肝腎，益精血，《隨息居飲食譜》稱其：「補腎，益血，填精。」

✔ 瑤柱：《隨息居飲食譜》稱其：「甘溫，補腎，鮮脆勝之，為海味冠。」

✔ 海參：益精血，補腎氣，潤腸燥，《隨息居飲食譜》中記載：「海參，咸溫，滋腎，補血，健陽，潤燥，調經，養胎，利產。」

立冬養腎食療方：養元參米粥

很多地方都有立冬補冬的習俗，天冷的時候，適當進補能很好地抵禦寒冷。

但是，進補前要先引補，讓脾胃有一個適應過程，這時候，一碗溫熱的粥，就非常熨帖。

冬天是非常適合煲粥燉湯的。一碗熱粥，一碗冒著熱氣的湯，不僅能讓身體暖和起來，還能滋脾養腎，讓食材的功效發揮到極致。立冬後，給大家推薦一道海參小米粥。

海參，味甘、咸，性溫，具有補腎益精、養血潤燥的功效，既能補腎陽，又能滋腎陰。

《本草綱目拾遺》中記載：「海參，味甘咸，補腎，益精髓，攝小便，其性溫補，足敵人蔘，故名海參。」

比較奇妙的是，海參的再生能力非常強，活著的海參被切斷後，能重新長完整。

小米，味甘、咸，專入腎，兼入脾、胃，養腎氣，健脾胃，清熱解渴。《本草綱目》記載其：「治反胃熱痢，煮粥食，益丹田，補虛損，開腸胃。」

尤其適合內熱及脾胃虛弱的人，當胃口欠佳時，食用小米既開胃又養胃。

養元參米粥

功效：健脾養胃，補腎益精。

做法：海參1個，小米100克。海參以40℃的溫水泡軟後，剪開參體，除去內臟，洗凈，再用開水煮30分鐘，後用涼水浸泡至軟，宜反覆沖洗並更換涼水。

海參處理好後，切成小塊，加入小米、薑絲，用小火燉煮至粥水黏稠。

用法：

一般女性日常保健一周1次。脾腎兩虛的女性更適用，如經常感到食欲不振、神疲睏倦、手腳冰冷、腰膝酸軟無力、夜尿頻繁、舌淡者一周可用2-3次。

女性提醒：

✔ 孕期可用；

✔ 經期可用；

✔ 肌瘤、增生女性可用。

閆主任點評：

1、小米養胃，其中，最養胃的是小米的米油，就是浮在粥面的那層薄薄的皮和一層很黏稠的液體，對於健脾養胃非常好；

2、《隨息居飲食譜》里記載：「脾弱不運，痰多，便滑，客邪未凈者，均不可食。」因此，外邪未清除的時候，不吃海參；脾胃長期虛弱，運化不暢，痰多便滑的人，要少吃。

同時，海參最好不要與醋、酸味水果等一同進食。

3、現代研究認為，海參含豐富的蛋白質、鈣、鐵、碘、磷等，有些品種的海參還含有硫酸軟骨素、粘多糖等成分，具有增強體質、延緩衰老、調整血糖血脂、抗腫瘤等作用。

4、海參以參體黑褐色、鮮亮、半透明狀、內外膨脹均勻呈圓形狀、內部無硬心的為佳，特別以體形大、肉質厚、體內無砂者為上品；

劣質的海參參體發紅，體軟且發黏，參體枯瘦、肉薄、坑陷大，肉刺倒伏，尖而不直。另外，海參可大致分為光參、刺參兩種，其中，肉刺完整的刺參質量優於光參。
