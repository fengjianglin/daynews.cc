終於找到心臟支架長壽的秘笈！血管通暢，還要做好4件事

2019-12-13 818條評論 1,905次閱讀 479人點贊

冠心病、心臟支架、心梗….這些辭彙在生活中越來越常見，幾乎每個人身邊都能找出患有心血管疾病的親戚朋友。據統計，我國僅2018年就做了915256例心臟介入手術，而且這一數字還在呈現逐年增長的趨勢。心臟支架後如何重新回歸生活？能活多久？如何保持支架的通暢？成為了壓在這群患者心中的3座大山，那麼心臟支架要注意哪些呢？如何才能讓支架一直暢通，不再受到心血管健康的威脅呢？我們一起來了解一下。

▲數據來自於地方醫院1788家及軍隊醫院[1]

心臟支架後能活多久？

心臟支架後能活多久，10年？15年？這是患者們最為關注的問題。事實上，心臟放支架，是臨床中幫助患者解決心肌缺血，恢復冠狀動脈血運的一種有效治療手段，支架本身對於患者的壽命並不會造成不良影響，反而是幫助患者撐起了堵塞的血管，保住了性命，希望大家不要對心臟放支架產生誤解。

而心臟支架後能活多久，完全取決於患者自身對於冠狀動脈粥樣硬化這一疾病本質的控制情況。確保不發生新的血管堵塞、延緩甚至逆轉冠狀動脈粥樣硬化，要從藥物治療與非藥物治療兩方面入手，雙管齊下。按醫囑用藥，調整好自己的生活方式，定期複查，將血糖、血壓、血脂、體重控制在良好狀態，即：

空腹血糖低於6，餐後2小時在7左右或以下；

血壓控制在80/130；

體重指數（即體重/身高的平方）控制在22-24左右；

低密度脂蛋白膽固醇（LDL-C）控制在＜1.8mmol/L，不低於1.0mmol/L

控制了這些讓心臟血管變硬、堵塞、老化的高危因素，在這場心臟血管保衛戰中，你就基本取得了勝利，手術才不會白做。

進口支架、國產支架，哪個更好？

現在臨床中用的最多的支架就是藥物洗脫支架，國產支架價格在8000-12000元；進口支架價格在18000-20000元。目前國產支架的生產工藝完全可以比得上進口支架，因此絕大部分醫生都會建議患者使用國產，除非有患者需要的支架型號特殊或自己要求用進口支架醫生才會使用。

但是，支架的國產與進口，在對於康復的影響上沒有差異、治療效果也不受影響，心臟放入支架後的管理才是關鍵。因此，患者在選擇的時候不必在這方面糾結，根據自己的經濟情況決定就可以了。

心臟支架後注意這些，血管才不會再堵！

  • 嚴格遵從醫囑用藥

這樣既能促進已經做過支架的血管恢復，又能幫助尚未堵塞的血管繼續保持暢通，同時還能減緩已經出現狹窄和堵塞的血管的進一步惡化。常用的藥物治療方法，依照目的主要分為兩大類：

1. 預防心梗，改善預後。常用阿司匹林（防止血栓形成）、氯吡格雷（用於替代阿司匹林）、他汀類藥物（改善血脂水平，減緩血栓的形成和發展）、ACEI或ARB（降低心梗風險，包括卡托普利、雷米普利等）。

2. 改善心肌缺血情況，減輕心絞痛發作。常用β受體拮抗劑（如比索洛爾、美托洛爾）、硝酸酯類（如硝酸甘油、單硝酸異山梨酯）、鈣通道阻滯劑（如維拉帕米、硝苯地平）。

  • 積極調整生活方式

飲食上盡量多蔬果少油膩，戒煙限酒，並保持適度的體力活動，增強心肺功能。

  • 定期複查

術後患者要進行定期複查，通常剛出院的第1 個月複查第1次，如果化驗結果較好、病情穩定，會逐步延長複查間隔時間，之後會間隔3個月、6個月、1年，之後每年複查一次即可，特殊情況例外。

  • 自我心理調適

受限於患者對於冠狀脈支架手術的相關知識了解不夠，往往會使病人出現一系列負性情緒如自我負擔、抑鬱、恐怖等，嚴重影響了患者預後效果和術後生活。[2]

往往在這個時候，支架術後患者們應該快速調整自己的角色位置，並正確認識到：「患上心臟病治療是一個長期過程」、「支架手術並不是治癒心臟病，還需要日常用藥和養護來保持血管通暢」 ，積極面對，輕鬆、冷靜接受自己患者的事實。[3]

平時保證充足的睡眠，積极參加社區心腦血管健康知識講座活動，樹立積極健康的的人生觀，始終相信只要利用科學方法進行術後養護。
