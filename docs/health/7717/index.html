β-胡蘿蔔素好處多，這3種蔬果中的含量比胡蘿蔔多

2019-10-22 62條評論 1,642次閱讀 520人點贊

簡單來說：β-胡蘿蔔素很重要，但平時多吃顏色豐富的蔬菜水果（不一定要是胡蘿蔔），均衡飲食就不用吃相關補充劑來補充維生素A。

目錄：

1. 什麼是β-胡蘿蔔素？

2. 維生素A有什麼用？

3. 如何補充維生素A？

4. 如何計算?

5. 吃什麼補充劑？

以下信息僅為科普，不應視為診療建議，不能取代醫生對特定患者的個體化判斷。

前天我的文章《我奶奶又又又亂花錢買保健品了 | 葉黃素有用嗎？》發完後，有一條很有意思的留言，有人問胡蘿蔔吃得少，要吃β-胡蘿蔔素嗎？

看來，β-胡蘿蔔素在人們心中有著無法忽視的功效。那它主要有什麼功效，又該如何攝取β-胡蘿蔔素呢？

1. 什麼是β-胡蘿蔔素？

β-胡蘿蔔素是一種食物中常見的「維生素A原」。

簡單來說，胡蘿蔔素是一類黃色或者紅色的色素，在很多深綠色、黃色的蔬菜、水果中含量都很豐富，此外包括牛奶、蛋黃以及人體的脂肪中也都會有。

胡蘿蔔素常見的有四種形式，分別是α-、β-、γ-和δ-。（後兩個讀作「伽馬」和”德爾塔” ，就是導致綠巨人變異的那個伽馬射線的伽馬），它們都可以轉化成維生素A，也因此被視為維生素A原（能夠在人體內轉成化為維生素A的成分的統稱）。

4種胡蘿蔔素當中，β-胡蘿蔔素是食物中最為常見的，也是維生素A原的典型代表。一般哺乳動物所獲得的維生素A中，有2/3是來自β-胡蘿蔔素，像是綠葉蔬菜、橙色蔬菜（胡蘿蔔、紅薯、南瓜等）還有雞蛋都有β-胡蘿蔔素。

我們之所以認為β-胡蘿蔔素非常重要，也正是因為它維生素A原的屬性。

2. 維生素A有什麼用？

維生素A是人體所必需的一種維生素，首先它在保障視力方面非常重要，眼睛需要產生某些色素來使得視網膜正常工作，而如果缺乏維生素A，這種色素就無法產生，會導致夜盲症。

同時眼睛還需要維生素A來滋養角膜。如果缺乏維生素A，眼睛會無法產生足夠的水分來保持濕潤。

因此，嚴重的維生素A缺乏症會導致兒童失明、孕婦的夜盲症。

此外維生素A也和免疫功能、呼吸系統、生長發育等等密不可分。曾經有一項隨機對照試驗對新生兒進行維生素A的補充，將有營養不良風險的尼泊爾兒童乾眼症患病率下降了63%。

3. 如何補充維生素A？

對於維生素A缺乏，常見的解決方案包括服用維生素A製劑、吃富含維生素A的強化食品、天然食物等。多攝入富含β-胡蘿蔔素（維生素A原）的水果、蔬菜，使膳食多樣化，也是有幫助的。

除了胡蘿蔔之外，菠菜、羽衣甘藍，還有西藍花的β-胡蘿蔔素含量算是比較高的了。半杯做熟了的西藍花大約就可以滿足一天1/4的維生素A的需要。

但需要注意的是，過去網上流傳的版本以及市面上大多數軟體中，西藍花的β-胡蘿蔔素的含量極高（7210微克/100克），然而按照最新版的《中國食物成分表》，每100克的西藍花β-胡蘿蔔素只有151微克，視黃醇活性當量是13微克，比常規蔬菜是高了一些，但也絕對沒有到「超級食物」那麼誇張。

所以如果需要精確計算，最好注意一下。一般人也不用特別的注意。

4. 如何計算?

需要注意的是，在評估食物的營養價值時，我們都是把各種維生素A原摺合成「視黃醇」來評價它的營養價值。

以中國營養學會膳食營養素參考攝入量為例，維生素A的部分都會特別註明，單位是「視黃醇活性當量（RAE）」，不同來源換算的比例是不一樣的。

之所以要用「視黃醇」當量來評估，主要是因為在人體血液中的維生素A原一般來說都是以「血清視黃醇」的形式存在。

當然這個方法也有過於簡化的問題，但對於預防維生素A缺乏非常有意義的。

從上圖紅框里的這個公式你會發現，如果是β-胡蘿蔔素需要除以12，其他膳食維生素A類胡蘿蔔素需要除以24。

再有一個單位是「國際單位」（IU），1IU相當於0.3微克，不過這種單位已經在逐漸淘汰中。

5. 吃什麼補充劑？

有的人可能說，既然食物這麼難算，我直接吃補充劑難道不好嗎？

如果是真的維生素A缺乏高發的人群（一般來說發展中國家的嬰幼兒、孕婦、哺乳期女性算是），還是有必要吃維生素A補充劑或者吃維生素A原補充劑（比如β-胡蘿蔔素補充劑）的。

但還得警惕維生素A過量的風險，維生素A中毒會導致噁心、食慾下降、嘔吐、視力模糊、脫髮、肌肉和胃部疼痛、嗜睡等等癥狀，長期較高的劑量也可能導致骨折。

哪怕是β-胡蘿蔔素，由於需要轉化，當人體內維生素A已經較多時，轉化率會下降，因此相比維生素A安全，頂多是吃得粘膜、皮膚髮黃。但也有研究顯示吸煙人群補充β-胡蘿蔔素之後（劑量大約是20-30毫克），會增加肺癌的風險。

所以能改善飲食就不要亂補，補充也要控制劑量在安全範圍（一般世面上流行的成年人用複合型補充劑、嬰幼兒吃的維生素AD滴劑按照標準劑量單獨吃都沒問題），更好的選擇還是吃天然的食物，多吃各種顏色多樣的蔬菜、水果，同時保證動物性食物的維生素A。
