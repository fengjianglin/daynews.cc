冬至飲食養生，推薦4種食材1款食療，大家可以試試

2019-12-18 501條評論 2,155次閱讀 709人點贊

文/癌上健康，本文原創歡迎個人轉發分享！

大雪節氣就快過完了，很多的小夥伴呢沒有趕上大雪節氣的養生，隨著冬至這個節氣就來了，冬至就是一年中最冷的開始，我們南北方的習俗都不一樣，但是養生的方法都是一樣的。

今天就給大家科普一下，冬至節氣我們怎麼養生？有什麼好的食材養生？

冬至養生食材：

1. 紅棗

紅棗是我們最常見的養生食材，紅棗有補血益氣、健脾胃的效果，我們在喝水的時候就可以泡上幾個，還可以和枸杞、桂圓等食材一起泡水喝，中醫養生常說「一日三顆棗，永遠不顯老」這就說出了紅棗的好處。

2. 花生

花生也是我們養生和飲食常見，花生的紅衣也有補血益氣的效果，我們經常用花生豬腳一起燉著吃，有下乳的效果，如果出現胃酸、胃燒心，也可以吃點花生可以緩解，所以花生也可以緩解胃病的效果。

3. 糯米

糯米在南方吃的比較多，一般吃湯圓、粽子、米酒等，並且糯米還有健脾開胃、緩解氣虛、滋補禦寒等效果。

4. 黃豆

黃豆也是我們經常見的食材，我們飯桌上出現的黃豆芽等，並且黃豆還有提高免疫力、健腦等效果，黃豆、花生、豬腳一起燉，也是有下乳的效果。

給大家推薦一款糯米食材的食療：

烏雞糯米粥

需要食材：糯米、烏雞腿、蔥、調味料等食材；

先把：烏雞腿清洗乾淨，切成塊，糯米淘洗乾淨，蔥切成絲，開火放鍋，添加適當水，把烏雞腿塊放進去煮，先大火煮開，轉小火煮，這個時候把糯米放進去，繼續大火煮，煮沸以後轉小火，等到煮好以後，把蔥絲和調味料放進去就可以食用了。

功效：滋陰補腎的效果。

以上的食材都是我們生活中的食材，其實養生就是我們的飲食合理化，食材之間的搭配要準確，這樣才可以起到滋補的作用。
