腸胃病：長年喝中藥為什麼不見效？

2019-11-28 1,966條評論 4,377次閱讀 3392人點贊

現今，生活習慣的改變、巨大的精神壓力、工作壓力造成了慢性腸胃病患者越來越多，且越來越年輕化，淺表性胃炎，萎縮性胃炎，胃潰瘍，結腸炎，腸胃功能紊亂，胃酸，胃脹，腹瀉，便秘，消化不良……，多達三十餘種的腸胃問題，已悄然成為危害我國國民健康的頭號疾病。

據某資料顯示，我國腸胃病患病率高達25%，患者人數如此之多的腸胃病，西醫和中醫是否有好的辦法呢？

西醫在治療慢性腸胃病上，療法很少，療效不理想，副作用大，往往一時控制癥狀，過後更加重病情，根本無法治本。但對於一些重大腸胃疾病，以及急性腸胃問題，通過西醫的一些特殊療法，可以得到有效的控制與緩解。

腸胃問題可以說是中醫的強項

現今中醫治療腸胃病，基本是以內服中藥為主，但有相當比例的腸胃病患者，長年服用中藥方劑，換了一個又一個中醫大夫，卻始終不見好轉，或者一時見效，過後又複發如初，這是為什麼呢？

張建銀老師（蓮花醫學創始人、扶陽派鄭盧醫學傳人）認為：

1. 現今中藥材質量的下降是我們無法迴避的問題，而現代中藥代煎機或中藥顆粒製劑則損失藥性，這是當代中醫面臨的嚴峻問題；

2. 中藥方劑對人體起效，首先要通過腸胃的消化吸收，而腸胃病患者自身的消化機能很差，平時消化一般食物尚難保證，更別說對中藥材的吸收。對於腸胃病患者，喝下去的湯藥，真正能夠被吸收用來療愈病灶的成分，往往是大打折扣的。這是非常重要的原因；

3. 很多腸胃病是由植物神經紊亂誘發，而植物神經紊亂與個人的精神思想、生活狀態有很大關係，而其他因素導致的腸胃病發展到一定程度，也會反過來引起植物神經紊亂。與植物神經紊亂相關的腸胃病，單靠中藥內服是無法康復的；

4. 一些腸胃病是由腹部組織創傷誘發，如闌尾炎手術，女性剖腹產等，這些創傷會破壞與腸胃相關的經脈運作與氣血循環，而這種結構性損傷，是單純中藥內服無力解決的。

5. 許多腸胃病是由長期外受寒涼，或者保持不良姿態習慣所致，這些都會導致與腸胃功能相關的肌群和筋膜組織的變異，最終造成對腸胃器官的壓迫或者牽拉，使得消化機能無法正常運作，而形成腸胃病。而腸胃病自身發展到一定程度，也會反過來導致外部肌群和筋膜組織的病變，導致惡性循環的出現。這種與肌群筋膜病變相關的腸胃病，也是單純中藥內服無法解決的。

我們已經了解了腸胃問題久治不愈的根源所在，該怎麼辦呢？

在張建銀老師看來，腸胃病的治療，應當堅持內外調理相結合的思路，也就是中藥內服與手法、針灸、理療等外治法的相互配合，內病外治的運用，是許多頑固慢性腸胃病獲得康復的關鍵所在。

1. 通過理療和手法外調，包括對經穴、筋膜的調理，率先打通與腸胃相關的「交通線」，消除腹部痞塊，排出一部分的寒濕瘀毒，為後續治療打下基礎；

2. 通過一些特殊的外治法，如整脊法和張建銀老師的「臟腑別通法」，恢復植物神經的運作平衡，在中醫上，實際上也就是對任督二脈以及其他相關重要經脈的調理，由此為腸胃疾病的康復，奠定良好的內環境；

3. 通過筋膜松解法、理筋法等，恢復與腸胃相關的肌群和筋膜的健康狀態，最大限度地松解創傷疤痕和其他增生硬化組織，解除它們對臟腑組織的牽壓限制，為腸胃疾病的康復，奠定良好的外環境；

4. 在上述基礎上，辨證施以中藥方劑，修復損傷的腸胃組織。

事實上，由於人體自身有著強大的自愈力，當經脈、筋膜等能夠對腸胃機能造成限制的客觀因素解除後，很多腸胃病患者可以不藥而癒。

附兩則典型醫案：

一位鄉村木工，長年腸胃不適，非常消瘦，多年間四處求醫問葯，內服湯藥不計其數，但並沒有顯著效果，去年求診於張建銀老師，張老師經檢查後發現，其腳部的脾胃反射區有非常明顯的按痛感，腳背處有許多筋結，問詢得知，該木工在年輕時不慎被機器壓到腳背處，自此腳部受創後，行走多有不便，隨後便出現了長達三十餘年的腸胃不適，隨後，張老師施以手法為其松解了腳部筋膜，疏通經絡，一星期後複診時，該患者自述已然多日腸胃未曾難受不適，飲食、排便等都已正常，至今，反饋良好。

一位中年男性從江蘇匆匆趕到山西太原，求診於張建銀老師，自言腹瀉、脹氣等腸胃問題已有三四年之久，喝了許多湯藥都收效甚微，但實在不堪忍受腸胃病之苦，朋友介紹後連夜趕到。張老師腹診後發現其腹部充滿緊硬痞塊，辨證其為腹部長期受寒所致，後以手法外調及一些理療手段，為其消除了痞塊，排出了一部分寒濕瘀毒，留診三天內，該男性再無出現腹瀉、脹氣等問題，腸胃問題已然消除，張老師叮囑其切忌飲食生冷，切忌腹部受寒涼，至今，反饋良好。

由於現代中醫醫院的模式，和大眾對中醫的既有印象，腸胃病這類內科慢性病，常常都被認為只能靠中藥去調理，但事實上，中醫內病外治法對於腸胃病，可能有著更神奇的功效！
