到了中年就“發福”，真的是躲不過去的“土肥圓”?

2019-12-18 1,829條評論 3,317次閱讀 242人點贊

戳右邊關注我們，每天都有最靠譜、實用的健康知識為你呈現呦，→_→

人們常說：「三十而立！」過去大家都認為，一個年過三十的人，就應該在事業上小有成就了；與此同時，家庭生活也應該步入成熟和穩定的階段。大多數人年過三十以後，還有一個比較顯著的變化，那就是開始「發福了」。很多朋友抱怨說，我平時非常注意養生，吃得比二十多歲的時候少了，還經常參加體育鍛煉，可身體還是不停使喚地一天天胖了起來，這是為什麼呢？

這種情況的發生，並不是個例，無論男女幾乎每個人都會「中招」。原因在於，我們的身體已經結束了生長發育的階段，已開始步入成熟和穩定的階段了。這個階段的一個顯著特點，就是身體對高熱量食物的需求，呈現出逐年下降的趨勢。用一個專業的名詞來解釋，就是我們在30歲以後，身體總的新陳代謝率每年都會降低千分之一點五左右。總新陳代謝率的降低，是導致中年人身體肥胖的重要原因之一，也是罹患心血管疾病和內分泌疾病的基礎原因。

看到這裡，可能有些朋友就會說，如果能夠想辦法提高身體的總新陳代謝率，是不是就能夠保持優美的身姿了呢？理論上是可以這麼理解的。但是，一個人的新陳代謝率也不能過高，這種情況也是一種病態。比如，甲亢患者的新陳代謝率就會病態地升高。對於大多數年過三十的朋友來說，保持身體現有的新陳代謝率，才是比較健康的選擇。

多年的實踐表明，保持一個健康成人新陳代謝率的穩定，做好以下幾件事情是非常重要的：

1、堅持參加體育鍛煉

年齡在30-65歲的中青年人群，每周都應該進行150分鐘的有氧運動，如快步走、慢跑、游泳或者舞蹈等；最好還能夠安排兩次力量訓練，如引體向上、仰卧起坐、深蹲等，每次力量訓練不少於10分鐘；

2、選擇優質蛋白類食物

相比脂肪組織來說，肌肉組織消耗熱量的效率更高一些。除了增加力量訓練和有氧運動之外，優質蛋白飲食與高糖、高脂肪和高膽固醇飲食相比，更有利於增加體內的肌肉總量。多吃富含維生素和礦物質且熱量較低的蔬菜和水果，也有利於人體吸收和利用蛋白質類食物。

3、補充足夠的水分

水是維持身體新陳代謝不可或缺的物質。除了從食物中獲取之外，成年人每天應該喝1000毫升以上的水。大家喝水，最好能夠選擇白開水，也可以適當喝一些淡茶水或者咖啡。由於含有較高的糖分和熱量，盡量不要選擇含糖飲料和碳水飲料來補水。

我們身體總的新陳代謝率，受到了年齡、性別、體重、呼吸、心率和基礎體溫的影響，也與我們運動強度和飲食結構有關，兩者之間也是相互影響的。為了保持健康的體魄和適宜的體重，改變我們的運動和飲食習慣是必然的選擇。
