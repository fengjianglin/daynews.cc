健康中國 溫補輕養新時代——阿膠肽＆蟲草素科技創新發展論壇

2019-12-16 93條評論 1,215次閱讀 184人點贊

健康中國 溫補輕養新時代

——阿膠肽＆蟲草素科技創新發展論壇在江蘇舉辦

保健品的安全健康問題是當今社會生活中人們較為關注的熱點課題。保健品對人體是否有益，如何正確合理地食用保健、養生食品……這些都是消費者比較關心的問題。但是，由於缺乏科學、正確的宣傳與指導，人們對保健品產生了越來越多的疑問，老百姓在身體需要補充營養的時候只能望而卻步。

中醫科學院研究員、央視健康之路欄目醫學顧問張立教授對這個問題，有著自己的見解，她認為在「溫補輕養」的新時代，現代人追求的健康養生應當是根據東方人的體質，根據身體的不同狀態，進行階段性、長期的營養補給。「補要有道理，養要有方法」「溫補輕養」的理念將會被越來越多的老百姓所接受。

中醫科學院研究員、央視健康之路欄目醫學顧問張立教授

聚焦健康產業，共享綠色生活。12月14日—15日，由東阿阿膠股份有限公司及江蘇康能生物工程股份有限公司共同主辦的「阿膠肽＆蟲草素 科技融合 創新發展論壇——溫補輕養新時代」大會在江蘇召開。會上，攜載8項國內外專利的新產品——魅卡蛹蟲草阿膠低聚肽飲（複合果汁飲料）閃亮登場，新產品所代表的溫補輕養的健康生活方式，受到了業內人士的廣泛關注。

倡導傳統食養新理念，開啟溫補輕養新時代。本次論壇積極響應「健康中國2030」規劃綱要、「國民營養計劃（2017-2030）」，堅持以人民為中心的發展思想，牢固樹立和貫徹落實創新、協調、綠色、開放、共享的發展理念，共同探討傳統食養產品的研發及產業升級換代，大力宣傳科學合理的溫補輕養生活方式，為健康中國助力創新。

會上，世界健康產業大會執行主席黃明達主席以「科技融合成就中國大健康產業未來」為主題進行演講。他表示隨著社會經濟的快速發展，注重健康已經成為當今人們的共識，健康即是財富，擁有健康也就是擁有了未來。大健康已被納入國家戰略，國家在「十三五」規劃中特彆強調健康美麗新中國。同時，他強調要發揮滋補食品物質本身原有的價值，利用現代科技賦能，為人類健康產業做出積極貢獻，是每個健康企業應該擔負起的社會責任。

世界健康產業大會執行主席 黃明達

東阿阿膠健康管理連鎖有限公司總經理田向欣在會議上談到，阿膠有近三千年的歷史，如何再傳三千年，需要創新，也需要現代的技術。阿膠肽和蟲草素的科技融合，不僅是一種傳承，更是科技的創新。東阿阿膠始終奉行「壽人濟世」的使命，堅守「厚道、地道、傳承、創新」的價值觀，光大中華古老瑰寶，讓傳承千年的東阿阿膠走進千家萬戶，守護中國人民和世界人民的健康。

東阿阿膠健康管理連鎖有限公司 田向欣

為了更好地推動「阿膠肽＆蟲草素的創新發展，本次論壇上，東阿阿膠與康能生物確立戰略合作。在數十家媒體的見證下，來自全國各地一百家新零售平台代表參與「溫補輕養」百名聯合發起人活動，共同推進新品發布和「溫補輕養」的生活理念。
