什麼時候喝水、喝多少水有利於控制血糖？我的經驗是這樣的

天天要聞
2019-11-10

一天，有一糖友問我：「你怎麼每天喝那麼多的水？都說糖尿病是吃得多、喝得多、尿得多、體重減少，喝水多了對糖尿病不利啊！

有這種觀點的人還真不少，認為患糖尿病後應該控制喝水，這是大錯特錯的，「三多一少」的早期表現，現在已經不是絕對的了，而且有的糖友並沒有癥狀。

其實，糖尿病人每天喝水很重要。那麼，糖尿病人應該怎樣喝水呢？以下是我的經驗：

每天早晨起床必須喝一杯溫開水

我每天早晨起床必須喝一杯溫白開水。因為，糖尿病人是很需要水的，這是毋庸置疑的。

從中醫理論上認為，糖尿病是陰精虧損、肌體津虧燥熱的是一種消渴疾病。所以，水在調整人體各種營養元素代謝紊亂中起到重要的作用。相反，一旦糖尿病人缺水嚴重，就會導致急性併發症，甚至會威脅生命。為了預防這種嚴重併發症，最好方法是保持身體水分，在缺水時及時補水。

我之所以常年堅持每天早晨起來要喝一杯200ml左右的溫開水，是發現這杯水確實很有好處。

一是溫水容易被腸道吸收，不會馬上排出去。

二是早晨往往肌體缺水，這個時候喝可以恰到好處的補充。

三是早上器官是剛剛開始運行時間，這個時候補水會潤滑腸道，促進腸道蠕動加快排便，清理毒素。

四是補充肝臟的水分有利於提高清理體內毒素的功效。

五是及時稀釋血液,凈化血液中的垃圾,預防心腦血管疾病的發生。

藥王孫思邈在《千金方》中就提出了「初起首服水法」的養生方法，認識到了早晨起床後喝一杯水有利於健康的科學道理。我也聽到過這樣的觀點，說早上不感覺口渴，肯定是不缺水。其實，早上在缺水的時候，糖尿病人由於神經系統失調或病變，是感覺不出來的，這個時候如果不及時補水，就會導致血稠、運行慢，帶來的後果就是血壓、血糖都會波動，對糖尿病人危害極大。

所以，我認為，不渴不喝水是個錯誤的習慣，糖尿病人需要及時補水。

每天補水量必須達到2500毫升

我每天都堅持喝水，從飲食到喝水，總量都能達到2000ml以上，不是暴飲，而是多次少飲，讓水分在機體內慢慢吸收，為我所用。

每天補水量是有科學研究的，據美國洛杉磯國際醫藥研究所的研究，成年人每天飲水量的標準是每公斤體重每天應該補充40ml水。也就是說，成年人每天肌體中最低也要保證補充2000ml左右的水。

有人認為，現在的自來水有污染，不如喝純凈水和蒸餾水比較安全，那麼，到底喝什麼水好呢？

世界衛生組織提出的健康飲用水必須符合六個條件，即：不含有害物質、含有適量的礦物質、硬度適中、含氧豐富、水分子團小、pH值為弱鹼性。

純凈水及蒸餾水基本上不含礦物質，硬度接近於0，含氧量極少，水分子團大。顯然純凈水和蒸餾水不符合健康水的條件，營養及安全性並不及礦泉水或白開水。所以還是補充白開水比較好。

但每天喝水不能當任務去被動的完成，更不能為了喝夠水量一大缸子水一口氣喝下去，這都是不對的補水方法，正確的喝法是勤喝少喝，一天的量按時間均勻持續的喝。《中國糖尿病膳食指南（2017）》中提出白開水是糖友最好的飲料。

平常最好少量多次喝水，每次1杯（200mL），不能等口渴時才喝，夏天出汗多時更要主動多喝。一般情況下，糖尿病人每天飲食中的水分加上平時的補水總量以2000～2500毫升為宜。

如果患糖尿病後控制喝水不但不能治療糖尿病，反而使糖尿病更加嚴重，可引起酮症酸中毒或高滲性昏迷，是非常危險的。

當然，也不是人人都要多飲水的，那些有腎功能障礙的患者，不能一味多喝水，而應該根據他的尿量多少決定飲水量。如果有腎功能障礙，但尿量很少，這個時候，建議還是合理的補充一些水分為好。

每天睡覺前必須喝一杯白開水

晚上睡覺之前半小時，為了身體一夜新陳代謝作水分儲備，我會再喝一杯200ml左右的溫開水。因為整個一個晚上，我們一般很少主動補充水分，所以晚上的水分流失很大，人體在睡眠的時候會自然發汗，在不知不覺中流失了水分及鹽分，而睡眠的八小時內，身體都無法補充水分，這就是為什麼早晨起床會覺得口乾舌燥的原因了。

因此建議在睡前半小時要預先補充水分，讓身體在睡眠中仍能維持平衡的狀態。

美國埃默里大學醫學院臨床醫學教授詹姆斯·加文博士表示，飲水不足對2型糖尿病的影響與過量攝入膽固醇十分相似。法國有研究小組為了探討「喝水量」與「糖尿病」二者之間的關係，對三千多名志願者進行了一項研究調查，研究者每隔三年會對這些人員進行一次調查，分析他們日常生活中的飲水狀況，9年之後，再為這些志願者做一次血糖測試。

調查結果顯示，那些每天喝水量較多的人，患高血糖症（前期高血糖）的風險比飲水量少的人低21%。研究者還總結稱，每天喝4杯（227g/杯）水有助於預防高血糖症，有助於避免得糖尿病的風險。

那麼，為什麼喝水太少容易血糖升高？

研究表明，這主要是因為糖尿病與調節人體水分的相關激素（後葉加壓素）存在著某種聯繫，當飲水量不足時，後葉加壓素會提高，進而導致血糖提高，久而久之，就會增加患糖尿病的風險。所以，糖尿病人除了每天補充水分外，更要補好睡前這杯水。

總之，適量多飲水，有利肌體代謝，降低血漿滲透壓，改善血循環和微循環系統機能，減少酮症酸中毒、微血管等併發症的發生。建議糖尿病人補充白開水為宜，也可以喝些淡茶水、牛奶、豆漿。

最後需要強調的是，如果腎功能下降，患有糖尿病腎病，並出現水腫等癥狀的糖尿病人，是不能隨意多喝水的，一定要及時就診、科學限制。
