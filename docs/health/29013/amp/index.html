推拿降壓到底是否有效？淺談中醫降壓的一種治療手段

天天要聞
2019-12-22

眾所周知，高血壓是臨床常見的疾病，可將其分為繼發性高血壓和原發性高血壓，其中原發性高血壓患者，其體循環動脈壓升高，可引發腎、 腦、心等障礙，若患者的病情較為嚴重，則可導致其死亡。在我國高血壓患者中，中老年人群占絕大多數，加強原發性高血壓患者的有效治療干預是必要的。

臨床上口服降壓藥是對原發性高血壓進行治療的主要方法，其主要的藥物為鈣通道阻滯劑、 β 受體阻滯劑、利尿劑等，雖然可獲得一定的效果， 但是若患者長時間接受降壓藥物治療，則會出現血鉀降低、心率增快等不良反應的情況。隨著研究的深入，發現中醫在原發性高血壓患者治療中應用，可獲得較好的效果。

推拿降壓，穴位刺激是否能夠有效降壓？

中醫認為，患者精神長時間處於高度緊張的狀態，會使得其肝氣鬱結、肝火過旺，從而引發血壓水升高。

研究顯示，為患者實施中醫推拿的降壓效果如下：

1、能夠通過足底反射區、人體穴位等，來對大腦皮層相關反射功能觸發，獲得中樞神經調控的效果。為原發性高血壓患者實施推橋弓干預， 可促進血壓水平的降低， 獲得較好的治療效果，主要是其可對頸動脈竇按摩， 獲得較好的壓力感受器效果，從而達到血管擴張、心率減緩的作用。

2、能夠促進血管壁切應力的提高，改善平滑肌細胞以及血管內皮細胞，從而改善生理層面，對高血壓患者的相關生理反應調節進行調整，獲得活血化瘀的效果，對血壓水平控制。

中醫推拿的實際推廣意義何在？

中醫推拿的應用，能夠獲得較高的價值，可更好控制患者的血壓水平，具有可反覆操作、 無創等特點，治療效果好，併發症發生率低，對改善原發性高血壓患者的預後， 有著積極的意義，值得推廣應用。
