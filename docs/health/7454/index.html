90後颳起一陣養生熱！這些網傳的神級養生產品紅黑榜了解一下

2019-11-12 88條評論 4,833次閱讀 376人點贊

不知不覺，90後也開始踏入了養生的行列，

比如小編一個11月份還在光腿的作死女孩兒，

現在已經開始每天泡jio了……

90後們察覺到自己開始脫髮，膝蓋開始疼了之後，

發展了一堆養生表情包，畢竟亞健康也意味著不健康！

是啊，表情包是做出來了，你們真的做到了嗎？

其實90後的養生也不算真正意義上的養生，

啤酒+枸杞，暴飲暴食完吃健胃消食片，

一邊火鍋蛋糕冰淇淋一邊嚷嚷要去健身房……

所以「養生朋克」這個詞就出來了！

為了能夠更方便又洋氣的養生，90後開始尋找養生產品，

商家們也抓緊時機推出各種「黑科技」，

就這樣，我們身邊突然掀起了一股「養生風」，

然而有些大熱的產品是真的雞肋！騙你都不要錢的！

【黑榜】

1.肩頸按摩儀

這種比較老式的肩頸按摩儀，長輩家一般都有，

但其實它真的挺雞肋的，因為實在是太硌了！！！

哪有這樣的按摩儀啊一點兒都不舒服好嘛！

同樣的，現在還有這種挎在脖子上像個小耳機一樣的肩頸按摩儀，

（這裡不指定牌子，圖是我隨便找的，沒有黑誰的意思，求生欲很強！）

這種按摩儀在按摩頸部的時候確實還是挺舒服的，

但是按摩的面積太小了，有的連斜方肌都按不到。

2.自發熱護膝

為什麼我特意找了這樣一張圖呢？

因為去年冬天我給我媽買了一套，還是日本的，

也不知道圖啥，護膝不論對兒賣論支賣，一次得買兩盒，而且還不便宜！

結果我媽戴了兩天問我，咋沒發熱呢？而且還起球兒！

3.養生鍋

這個東西我是萬萬不可能買的，我的砂鍋用的也蠻好：）

但是同事小姐姐就買了一個，結果最後也閑置了，

主要是這種養生鍋一燉就是三四個小時，實在是太慢了，

你們想喝啥湯，砂鍋它不香嗎？何必呢……

小編還有個朋友家的長輩，買了個類似於這個玩意兒的「黑科技」，

（只是類似昂！不是同一款！沒有黑誰的意思，求生欲很強！）

說是什麼納米科技、電磁脈衝啥啥的說的天花亂墜！

搞得朋友家的長輩花了幾萬塊錢，結果屁用沒有！

真的，說的越神的東西越別信！別叫智商稅了求你們了昂~

【紅榜】

1.肩頸按摩儀

沒錯，又是肩頸按摩儀！只不過這次的有點大隻，

這種像披肩一樣的肩頸按摩儀的按摩區域比較大，

有些還可以按摩到背部，雖然笨重了點兒，但是效果不錯。

2.養生壺

其實小編想了很久到底是放紅榜還是放黑榜，

這東西本身沒什麼雞肋的地方，顏值也不錯，

但問題就是很多人買回來用幾次就閑置了，

根本原因還是太懶了，養生壺本身是沒錯的！

3.艾灸儀/貼

艾灸儀這種東西一般都是醫院啊美容院會有，

覺得哪裡不舒服了就去用艾灸儀照一照，

但有人說艾灸是玄學，可能確實沒有宣傳的那麼神奇，

但是照一照熱乎乎的還是很舒服的，也有一定作用。

還有這種艾灸貼，稍微有點燙燙的，用起來也是超級舒服，

聽一個朋友說她每天貼，結果一年下來都沒有感冒過！

4.足貼

足貼也是我很糾結的一個，不知道紅榜還是黑榜，

現在足貼的牌子太多了，難免有攪屎棍在裡面，

有些足貼是真的好用，而有些足貼就是逗你玩兒呢！

還有一點就是足貼很容易弄髒床單襪子什麼的這點很頭疼~

最後就是如果你買到了好的足貼，它不只是貼在腳下，

查一查每個穴位有什麼作用，可以對症下藥！

感覺我們現在跟風買養生用品，就像老年人看電視購物一樣，

現在我終於理解為什麼那麼多老人都容易被騙了，

莫名其妙就被種草然後跟風買爆款的我們，不是一樣的道理嗎？

還有什麼雞肋的或者特別好用的養生產品，來一起種草拔草！
