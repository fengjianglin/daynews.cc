從便便看健康，醫生教你如何科學的觀察便便

2019-11-14 1,435條評論 2,995次閱讀 165人點贊

中醫診斷疾病以望、聞、問、切為主，其中「望」為首，舌苔、面色、指甲等等，都是「望」的內容。自古以來中醫就很重視觀察便便以幫助診斷疾病，大便的次數、性狀、顏色、氣味等，可以說明很多問題。醫生在查問病情時，一般也都會問一下患者的便便情況，以便更好地進行辯證，以儘早採取有效的對症治療。因此，我們平時最好也不要嫌臟畏臭，要時常注意觀察便便，留心有什麼樣的改變，這樣我們就能簡單的通過觀察來判斷身體的近況。

那麼我們應該從哪幾個方面觀察便便呢？

次數

正常：正常情況大便次數一般為1~2天一次。

異常：大便次數突然減少並且大便乾燥，稱為便秘；大便次數突然增多或突然變稀，同時內容物有變化，則稱為腹瀉。

性狀

正常：正常情況大便呈軟條狀或糊狀。

異常：

1、細菌性痢疾：大便除變稀外，若出現較多黏液或混有血液大便，大多是細菌性痢疾；

2、霍亂病：大便如果是淘米水一樣的情況，排便的時候沒有腹痛，但出現脫水、抽搐、休克，則可能是患了霍亂病；

3、感染性腹瀉：如果大便呈湯樣，可能是細菌或病毒引起的感染性腹瀉；

4、金葡萄腸炎：大便如果是海水樣大便，則可能是金葡萄腸炎；

5、黴菌性腸炎：如果大便形似豆腐渣樣，則黴菌性腸炎的可能性比較大。

結論：一般臨床上初步診斷會認為大便清稀，大便有不消化食物，或如鴨溏樣便，屬寒濕困脾或脾胃氣虛，大腸功能紊亂；便糞如水，或兼惡風、發熱，是風瀉；大便乾結，面紅，身熱，舌紅少津，多屬實熱傷津液或陰虛內熱；大便乾結如羊屎，排出困難，為腸道津液缺乏。

顏色

正常：正常情況大便為黃色或棕色。

異常：

1、膽道梗阻：大便如果呈白色，並伴有全身黃染、騷癢是膽道梗阻的表現；

2、出血性壞死性腸炎：大便像赤豆湯樣暗紅色伴惡臭，可能是出血性壞死性腸炎；

3、腸套疊或阿米巴痢疾：果醬色大便有可能是腸套迭或阿米巴痢疾所致；

4、上消化道出血：如果出現柏油樣黑便，在排除服鐵劑和大量食用動物肝血之後，可診斷為上消化道出血；

5、直腸或肛門出血性疾病：若孩子的大便呈鮮紅色血液狀，直腸或肛門出血性疾病的可能性比較大。

結論：一般臨床上初步診斷會認為大便紅如桃醬，排便不暢，舌苔黃膩是胃腸蘊熱，迫血下行；大便色黑如柏油樣，兼面色不華，或脘腹隱痛為胃絡出血；大便如膿涕，色白或紅，兼見腹痛肛灼，里急後重者為濕熱痢疾；大便白多赤少，偏濕，病在氣分；血多色赤者，偏熱，病在血分；若紅白夾雜，是氣血兩傷。

氣味

正常：正常情況下腸道細菌分解食物殘渣都會產生吲哚、氨等物質，帶來一定臭味，但不太重。

異常：

1、大便酸臭加重並帶有泡沫，日常飲食中含糖或澱粉類過多，導致腸胃不能完全消化吸收而過度發酵引起的；

2、大便突然變惡臭，發出臭雞蛋一樣的氣味，可能是日常飲食中蛋白質比例過多引起的；

3、大便腥臭伴油膩感，為脂肪類飲食過多；

4、大便惡臭且量又多，但同時身體很消瘦，是腸吸收不良綜合症的表現。

結論：一般臨床上初步診斷會認為大便色深，有惡臭屬濕熱泄瀉；大便酸臭味重屬消化不良，脾胃虛弱。
