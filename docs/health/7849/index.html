「來自北方的家」北方的美食往往都是簡單而又溫暖

2019-10-17 1,846條評論 2,839次閱讀 1691人點贊

白蘿蔔營養豐富，民間有「冬吃蘿蔔夏吃薑，一年四季保安康」的說法。胡蘿蔔含有大量β胡蘿蔔素，可起到保護視力的作用。蘿蔔的營養價值自古以來就被廣泛肯定，所含的多種營養成分能增強人體的免疫力。蘿蔔含有能誘導人體自身產生干擾素的多種微量元素，對防癌，抗癌有重要意義。

羊肉性溫熱、補氣滋陽、暖中補氣、開胃健脾，還可以去濕氣避寒冷，暖胃寒。羊肉對一般風寒咳嗽、慢性氣管炎、虛寒哮喘、腎虧陽痿、腹部冷痛、體虛怕冷、腰膝酸軟、面黃肌瘦、氣血兩虧、病後或產後身體虛虧等一切虛狀均有治療和補益效果。胡蘿蔔羊肉是傳統搭配，羊肉胡蘿蔔一起燉能提高營養價值。

白蘿蔔燉羊肉

準備食材：白蘿蔔、新鮮羊肉、花生油、鹽、大蔥、姜、蔥、花椒、雞精、枸杞、香油、香菜、胡椒粉、十三香、紅棗

做法：

  1. 將蘿蔔切塊，蔥切段，姜切片，香菜切末，羊肉切塊；
  2. 鍋中倒水加熱，放入羊肉焯水撈出肉沫，關火；
  3. 將肉撈出備用，鍋中下油煮熟，放入蔥姜花椒，將羊肉倒入，倒入香油，撒入十三香；
  4. 加水，煮至開鍋，加入白蘿蔔，加鹽，撒入胡椒粉，加入雞精，攪拌均勻；
  5. 開鍋後加入紅棗和枸杞，關火裝入碗中，撒香菜即可。

胡蘿蔔燒羊肉

準備食材：羊肉、胡蘿蔔、油、蔥姜蒜、五香粉、鹽、醬油、郫縣豆瓣醬、料酒、雞精

做法：

  1. 羊肉剁小塊焯水瀝干，胡蘿蔔洗凈切滾刀塊，蔥姜蒜切末；
  2. 起鍋熱油爆香蔥姜蒜、郫縣豆瓣醬、五香粉，煸炒羊肉片刻；
  3. 放料酒和1碗水小火燜煮1小時（用壓力鍋15分鐘）；
  4. 放醬油和胡蘿蔔燜煮5分鐘；
  5. 羊肉爛時收湯，放鹽和雞精調味即可。
