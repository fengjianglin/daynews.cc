脾虛是不是西醫的胃病？80後老中醫教您觀舌象調脾胃學養生

2019-10-18 1,282條評論 4,361次閱讀 2974人點贊

中醫藏象學說認為：脾胃五行屬土，處於中焦，共同承擔著化生氣血的重任。脾胃同為「氣血生化之源」，中醫認為人體的氣血由脾胃將食物轉化而來的。

又說，脾胃是「後天之本」，是人生存的根本。人從出生之後成長，長大以後學習、工作、娛樂等都需要大量的能量，而這些能量都是要通過飲食而來，但是飲食必須要由脾胃共同工作才能正常的轉化為氣血能量。

所以說，一個人健康與否跟脾胃的關係非常密切。

舌象這樣的人脾胃不好

中醫理論認為，舌象是身體、健康的一面鏡子。通過望舌這種簡易無創的途徑，可提供一個了解自身健康狀況的方法。

據李醫生的臨床觀察，如果舌象長這樣，通常脾胃都不算好。

舌質淡白，苔薄白或少或剝，中有裂紋，邊有齒痕。

這是很典型的脾虛舌象，提示脾臟運化功能失常，其主要表現為食欲不振，腹部脹滿，大便溏薄。如何再結合癥狀可以提前預判疾病的發生。

易患哪些胃病？

①胃炎：如果在進食後出現胃痛，或是食後飽脹、噯氣，那可能是胃炎的徵兆。

②胃潰瘍：一旦進餐後胃部疼痛發作或加重則要考慮是否得了胃潰瘍，這類病人在進食鹼性食物，如饅頭、鹼水面後， 疼痛可減輕。

③食道反流：如果餐後出現泛酸、燒心、暖氣、胸骨後痛(平卧或身體前屈或腹壓增加時更明顯)，要考慮胃食管反流病。這類人平時應避免吃得太飽，要多吃流食。

④胃下垂：進餐後出現胃部不適或腹脹噯氣加重(飯後活動或站立者尤為明顯)，平卧後能減輕的則要考慮胃下垂，尤其是體態偏瘦的人，更是如此。這些人不適宜「飯後百步走」，應少食多餐，餐後平卧休息30-60 分鐘，且不適宜做劇烈運動。

⑤胃癌：少部分人過去曾有過胃病史，但近來癥狀不具有原來的規律和特點，或出現易飢、易飽，且臉色、體力越來越差，則更應提高警惕，很可能是胃癌的信號。

如何調理?

具有上述特徵的人群該如何調理？李醫生認為還是要從飲食習慣入手：避免進食生冷、辛辣、油膩及刺激性食物，少吃甜食及酸性食物，不要吸煙飲酒，更不能暴飲暴食，飲食要營養均衡，潔凈衛生，定時定量，少食多餐，細嚼慢咽等等。

如果已經出現比較嚴重的癥狀，一定要到醫院查明病因，進行針對性治療。
