吃什麼補什麼！豬肚養胃真的如此嗎？聽醫生怎麼說

2019-10-18 2,102條評論 4,680次閱讀 749人點贊

豬肚是在我們平常生活中最常見的一種食材，它吃起來味道非常的獨特，並且有很多種烹調方法，可以用來煮湯，或者爆炒等，它是日常生活中我們常吃的一種食物，但是最近有一位就聽家裡的老人說豬肚還有養胃的功效，來問筱青這是真的嗎？吃豬肚可以養胃嗎？

其實關於養胃這個問題，十個人里八個都有過養胃的經歷，比如今天吃了很多不健康的東西，胃就會不舒服，那就開始養胃吧。很多人都用一些軟爛的食物進行養胃，那麼豬肚養胃的說法是從何而來呢？

大家認為它養胃，就是源於中國人以形補形吃什麼補什麼的說法，在民間還有一道關於豬肚非常有名的菜，豬肚雞，是豬肚和雞肉一起熬制而成的，看起來就非常清淡，會給人一種非常養胃的感覺，並且喝起來口感也很不錯，所以受到很多人的喜歡，但是吃什麼補什麼，以形補形這個說法是沒有任何科學依據的。

多吃豬肚不僅不能養胃，並且豬肚脂肪含量還比較高，從數據來看，每100克的豬肚含有含5.1克脂肪、15.2克蛋白質、165毫克膽固醇，食用要適量。嘌呤含量高，高尿酸和痛風患者應完全避免。除此之外，豬肚裡有一定量的的鉀、磷、鐵等礦物質。營養含量和正常的豬肉營養是差不多的，所以說豬肚養胃的說法是不靠譜的。

另外由於豬肚的脂肪和膽固醇含量比較高，如果經常吃，還採用了一些高油高鹽的烹飪方法反而會更加容易增加腸胃的負擔，不僅不養胃，還會傷胃。吃豬肚適量即可，按照新膳食指南，建議大家每天的畜禽肉適宜量為40～75克。

現在很多人的胃都不太好，身邊很多人都會時不時胃疼。養胃也成了一個大問題，經常也能收到很多朋友問筱青，到底怎麼樣才能養胃？身體髮膚受之父母，中醫上講，父母給我們的身體是「腎有餘而脾不足」的。如果後天沒有注意保養，脾胃不和就是難免的了。

在生活中大家說的最多的就是喝粥養胃了，因為粥很容易消化吸收，喝粥不給胃增加負擔。如果是急性的胃腸炎，胃潰瘍的患者，或者胃手術後的一段時間裡面，因為胃的功能不好，食物的消化吸收受到了影響，這時候確實應該以流食或者半流質飲食為主，但是，如果長期喝粥，那麼胃的功能就會慢慢變弱，對胃的健康反而會不利。而且，粥的營養非常單一，長期喝粥也會出現營養障礙。

胃不好的人，首先要找儘可能的找到病因，明確診斷。引起慢性胃病，常見的病因有不健康的生活飲食習慣，某些藥物，膽汁反流，不良的情緒等等。找到病因之後才能接受規範正規的治療。三餐保持規律，不要因為工作忙或者其他原因硬扛著，非要等到餓的胃疼，才想起來去吃飯。吃飯細嚼慢咽，給胃部減少負擔。切記不要狼吞虎咽。吃飯到七分飽就好，太飽容易加重胃的負擔，胃內壓力升高，造成食物反流

對於有胃動力不足情況的朋友，筱青建議沒有堅持一個小時的有氧運動。堅持一段時間，你自己就能看到改變，並且心肺功能和體重也會增強不少。

胃不好的朋友要避免高鹽飲食，高鹽使胃粘膜受到損害，在我們國家由於長時間重口味的飲食，導致胃癌發病率居高不下。大家在做菜的時候一定要注意少放鹽，少放醬，每天攝入的鹽分不可大於6克。

同時還要做到攝入營養的均衡，不挑食不偏食，保證碳水化合物，脂肪，蛋白質，等物質的攝入量。

不戒掉一些不良的飲食習慣就算再好的胃也承受不住。並且胃是有一定的自我修復能力的，只要你在日常生活中正常健康的飲食，保持好的作息，堅持幾年，胃自然會好很多。然而，有的人胃不好還非常的作，不吃早餐是常事，暴飲暴食，飢一頓飽一頓，飲食重口味，油膩，經常大量的飲酒，喝濃茶，吃各種刺激性的食物以及飲料，喜歡吃鹹的，各種腌菜，泡菜隨便吃。這樣的生活習慣，你的胃還能好的了嗎？養胃也需要一個過程，只要不做傷胃的事情，就會慢慢變好的，更多的健康養生知識，歡迎關注筱青。
