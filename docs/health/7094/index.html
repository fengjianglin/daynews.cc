關於心臟病的飲食誤區，還有很多人在深信不疑

2019-11-29 930條評論 2,068次閱讀 630人點贊

關於心臟病，很多人更加關心的是預防心臟病吃什麼好？人到中年後就要特別注重自身各器官的健康狀況，很多疾病都是在中年以後開始接踵而至的，其中心臟病就是最常見的疾病之一。想要預防心臟病可以多注重日常的飲食，而很多關於心臟病的飲食存在著誤區。以下這幾個飲食誤區，就是很多人在深信不疑的。

1. 吃西紅柿和葡萄能預防心臟病？

以形補形的觀念很深刻的影響著大多數人的飲食。人們認為吃核桃能補腦、吃牡蠣能壯陽、吃牛鞭能壯陽、吃西紅柿和葡萄能強健心臟，因為西紅柿切開以後的樣子和人的心臟結構很像。以形補形並不是完全有道理的，吃西紅柿和葡萄並不能有效的預防心臟病。

2. 經常喝紅酒對心血管健康有利

有的人認為你經常喝點紅酒是有利於心臟的健康的。紅酒少喝是可以促進血液循環的，但是並沒有數據表明喝紅酒可以起到預防心臟病的作用。而且大多數人在喝紅酒的時候都是掌握不好飲酒量的，往往喝的不是一星半點。所以喝紅酒和喝其他的酒一樣，都是會增加心臟病的發病風險的。並不提倡大家用少量喝紅酒的方式來預防心血管疾病，效果往往適得其反！

3. 吃魚肝油能預防心臟病

人們認為魚肝油的主要成分是不飽和脂肪酸，適當食用是可以減低甘油三酯的。有利於減肥，同時也對心臟的健康有益。其實，適量的食用魚肝油對心臟有些好處，但是一旦食用過量，會導致出現胃腸不適的癥狀。其實，建議大家可以在生活中多吃一些深海魚類來增加不飽和脂肪酸的攝入，預防心血管疾病以及降低甘油三酯，少吃各種膳食補充劑。

4. 患有心臟病只能吃素？

有的人在患有心臟病以後為了不增加心臟的負擔，避免心臟病病情加重。將自己的飲食習慣改成了純素食主義者。其實完全沒有這個必要的！純素食如果自己搭配不好飲食很容易造成營養成分的缺乏，而且肉類中還含有一些不飽和脂肪酸也是對心臟有益的。沒有必要完全素食，可以適當增加一些魚蝦食物的攝入。

除了以上這幾個關於心臟病的飲食誤區以外，還建議大家每天適當食用堅果食物，對保護心血管也是有好處的，不要吃太多，帶殼的堅果每天一把就可以了。
