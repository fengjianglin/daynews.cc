養腎就是養命，正確做法你get了嗎？

2019-12-16 1,013條評論 2,228次閱讀 775人點贊

腎，也就是大家所說的「腰子」，它的主要功能，是將血液中的廢棄物與不需要的物質收集起來，在排除的同時，將必要的成分再吸收，可以說是人體任勞任怨的「垃圾回收處理站」。

但，它也有罷工偷懶的時候，這個時候人體的垃圾就不能實現良性的運轉了。在中國，慢性腎臟病的發病率為11%左右，我國人口眾多，慢性腎臟病患者的總數龐大。

一旦腎功能惡化就很難恢復，腎衰竭晚期的患者進入尿毒症階段之後將可能會出現短期無尿的情況，思考能力降低、頭疼、失眠等。

並且由於生命無法維持，所以必須進行血液透析或腎臟移植手術，這時候所需要的不僅僅是一個好醫生，還有普通家庭難以支付的巨額醫藥費。

難道我們對於腎臟問題真的就束手無策了嗎？

不是的，腎臟出問題之後，我們的身體會給我們一些信號，能夠讓我們提早預防。

身體出現這些信號，要警惕腎臟問題

圖片來源：pixabay

尿液泡沫：正常人的泡沫尿，很快就會消失，但是腎病患者的尿中有蛋白質，所以泡沫細小且多，很難消失。

高血壓：沒有高血壓家族史的年輕患者，若血壓持續升高，應高度懷疑是否患有腎病。

水腫：常出現於眼瞼、足踝及背臀部。要注意的是腎病水腫清晨比傍晚嚴重。

起夜增多：慢性腎臟病有一個特殊的表現就是：夜間的尿比白天的多，所以晚上起夜增多。

當然了，除此之外還有乏力、腰疼、血尿等癥狀，這些癥狀比較複雜，導致的原因也很多，不便於自查，所以風哥就點到即止。如果出現上述幾種癥狀，那麼一定要趕緊去醫院檢查一下自己的腎臟。

養腎，其實很簡單

當然，在腎臟出問題之前，我們要想養好腎，不用枸杞不用生蚝不用各種鞭，只需注意以下：

圖片來源：pixabay

• 每天早晨起來喝一杯溫水，及時給腎臟補充水分；

• 千萬千萬別憋尿，憋久了，腎可是會罷工的；

• 炒菜時少放鹽，醬油、雞精、味精這些也含鹽，放這些調料時，鹽少放一點，高鹽會傷腎；

• 少吃煎炸食品、少喝碳酸飲料，多吃新鮮水果，膳食纖維對腎好；

• 少吃火腿腸、培根、臘肉、熏肉、午餐肉、肉罐頭等，腎臟不喜歡它們；

• 沒事少吃中藥，腎作為代謝藥物的主要器官，老讓它加班它也不樂意；

• 尿酸一定好控制好，痛風反覆發作會殃及腎；

• 止痛藥雖然能止痛，但是對腎不好，能不吃就別吃；

圖片來源：pixabay

• 腎不好，也能有性生活，腎不好和腎虛是兩碼事。

腎臟患病率非常之高，腎臟疾病因其低知曉率、高發病率、高死亡率被稱為「沉默的殺手」，很多患者本身患了腎病，可是卻又難以發現，早發現早護理，有時候能避免很多問題。
