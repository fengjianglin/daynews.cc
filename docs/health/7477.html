氣溫下降手腳冰涼？兩道食譜幫你緩解癥狀，讓你做個秋冬暖寶寶

2019-11-11 52條評論 2,148次閱讀 438人點贊

天氣逐漸變冷，慢慢開始有很多人出現手腳冰凍的現象，我們也能經常聽到女性朋友說一到冬天就開始手腳冰冷，晚上睡覺也暖不熱，同時還能見到很多女性在冬天的時候常常會抱著暖手寶取暖。

當然導致這些癥狀的出現主要還是有這幾點原因：

1. 自我保護

當人體處於寒冷的環境下，為了保證體內的器官的正常運轉，機體會自動開啟保護功能，皮膚會關閉身體上的毛細血管，使其身體能夠保持一定的溫度，這樣也使得更多血流能夠流向心臟等重要器官，如此一來，四肢的血流量就會有所減少，所以很容易變得手腳冰冷。

2. 氣血偏虛

對於體質偏虛的人群，往往很容易引起四肢冰冷的癥狀，體內新陳代謝速度變慢，細胞活躍度偏低，因此除了會出現手腳冰冷的癥狀，同時還可能伴隨著食慾不佳、精神不振等癥狀出現，以及情緒不穩定都會導致手腳冰冷的現象。

3. 激素分泌紊亂

女性出現手腳冰冷大多跟激素分泌有關，比如經期不調或者孕期的時候體內激素會有所變化，激素分泌過少從而導致神經系統細胞的分泌減少，體內一些器官沒有這些細胞的傳遞信號減少易導致血流量的減少以及皮膚血管的收縮，所以手腳冰涼的最明顯的表現。

4. 營養不足

比如過度減肥或者是飢餓過頭導致體內血糖濃度的降低，以及體內一些營養無法及時跟上，血壓也會有所降低，血液不循環容易導致手腳冰冷的情況。

與此同時，有些不良的生活習慣也會導致手腳冰冷以及畏寒的情況：

1. 久坐不動

導致血液流通速度變慢，同時產生的熱量也有所降低，因此人在寒冷的環境下常會出現發抖的情況，這也是體內自動產熱的一種表現。

2. 穿緊身衣

使得身體某些部位血液不流通常常受到擠壓，血液循環不流暢極易導致手腳冰冷的癥狀。

3. 挑食

有些人不愛吃主食導致人體內熱量不足，因此變得更加不耐寒。

4. 吸煙喝酒

嚴重影響血液循環，以及血管是收縮跟堵塞，血流明顯下降，容易出現手腳冰冷的情況。

在這康康給你推薦兩款冬季食譜，能夠有效的緩解手腳冰冷的癥狀：

1. 紅棗枸杞羊肉湯

材料：紅棗、枸杞、生薑、大蔥、食鹽；

做法：羊肉切塊用開水煮出血水後撈出再放入清水中加入紅棗、枸杞、生薑、大蔥煮40分鐘左右，再出鍋前加入少許食鹽即可。

2. 紅棗小米粥

材料：小米、紅棗、紅糖；

做法：紅棗去核後跟小米一起放入鍋中加入清水煮沸轉文火熬20分鐘，加入適量的紅糖即可。

除此之外，在平時生活還可以做到以下幾個點，同樣也能緩解手腳冰冷的癥狀：

1. 多運動：改善血液循環，增加身體熱量，同時也能促進新陳代謝；

2. 泡腳：同樣也能改善血液循環，不過建議浸泡時間不宜過長同時還要注意水溫；

3. 多曬太陽：提高身體免疫力，同時還能給身體帶來熱量

4. 注意保暖：做好保暖工作，避免血液循環受阻。

好啦！以上都是關於冬季如何保暖的知識點內容，這些你都情況了嗎？
