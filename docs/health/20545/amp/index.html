想要沒有蛀牙？改變你的飲食習慣

天天要聞
2019-12-17

人們通常所說的「蛀牙」，即齲病，是人類的常見病、多發病之一。齲病是由於口腔內可引起齲病發生的細菌附著在牙面上，通過分解牙面上的食物殘渣等物質產生酸，牙齒逐漸被酸蝕分解而形成的。細菌產酸需要一定的食物，而這些食物正是我們自己吃進嘴裡的。因此，我們可以通過改變自己的飲食習慣，少吃對致齲細菌有利的食物，多吃對致齲細菌有害的食物，來降低自己患齲病的風險。

糖類物質是致齲細菌最喜歡的食物，也是我們攝入最多的食物。大米、麵條、餅乾、飲料、糖果等，其中都含有大量的、不同種類的糖。糖類為細菌提供營養，細菌消化糖類後產酸，對牙齒造成較大的破壞。為了降低患蛀牙的風險，飲料、糖果、糕點等，應盡量少吃。糖類從被細菌吸收到產生酸，需要一定的時間，因此，吃完含糖量高且容易滯留在牙齒上的食物後，對牙齒進行清潔，如漱口或刷牙，可有效去除牙面上的食物殘渣，避免其被細菌利用。糖類物質也並非都是「壞人」，如糖醇類物質，特別是木糖醇，不僅致齲能力比較弱，還能抑制致齲細菌的生長。所以在吃了其他容易致齲的食物後，咀嚼木糖醇口香糖，不僅能清潔牙齒上的食物殘渣，還能促進唾液的分泌，使口腔內的酸性減弱，大大減少患齲的機會。

酸含量高的物質，如碳酸飲料、檸檬水、酸奶等，不僅可以營造一個適合耐酸致齲細菌生活的酸性環境，而且本身即可以使牙齒酸蝕脫礦，形成牙體缺損。

因此，若想降低齲病的患病風險，應減少食用酸性飲料、酸性物質。但如果實在想喝怎麼辦？可以通過使用吸管，減少飲料與牙齒的接觸。喝完後及時喝水、漱口、刷牙或咀嚼無糖口香糖，也是很好的挽救措施。

多喝水不僅可以對牙齒表面進行清潔，沖刷掉一些食物殘渣和尚未牢固附著的細菌，而且可以稀釋牙面上已產生的酸，升高PH，減少其對牙齒的酸蝕。一些礦物質含量高的水，包括牛奶，其中的無機鹽類也可以促進已經被酸蝕礦化牙齒的再礦化，減緩齲病的發生。

膳食纖維主要來自於植物，我們日常飲食中的瓜果蔬菜中含有較高的膳食纖維。膳食纖維為一種常見的多糖，其不能被人體消化吸收，也不易被細菌利用。一方面，膳食纖維含量高的食物在咀嚼的過程中，可以像牙刷一樣清潔牙面，清除附著在牙齒上和牙齒間的食物殘渣，。另一方面，其在咀嚼的過程中，可以刺激唾液的分泌，達到減少患齲的目的。如果在飲食中多用「粗糧」代替「精糧」，也能降低齲病的患病率。

除此之外，懷孕的媽媽和牙齒萌出之前的寶寶，如果蛋白質、維生素和各種礦物成分攝入不足，也會影響牙齒的正常發育，從而導致寶寶的牙齒結構發育不良，患齲齒的風險也大大增加。孕媽和寶寶應注意攝入足夠的動物蛋白，富含A、C、D族維生素的食物，如瓜果蔬菜、乳製品等，還有適量的鈣、磷等礦物質，為寶寶的生長發育提供足夠的營養，同時也能降低寶寶牙齒的患齲風險。
