沙療可調養腰背疼痛，腰間盤突出

2019-12-13 945條評論 2,475次閱讀 322人點贊

腰間盤突出是很多人的痛，沙療能夠減輕這種痛苦，除了腰間盤，風濕等也可以通過沙療來輔助。

很多人年紀輕輕，腰椎不適怎麼造成的呢,我們要知道人的脊柱是由24脊椎串聯成的，連接相鄰脊椎椎體的彈性墊就叫椎間盤，連接腰椎的就叫腰椎間盤。正常的腰椎間盤是待在腰椎骨頭和骨頭之間的，起一個緩衝的作用，承受著壓力，但是隨著年齡的增長，腰椎間盤各部分有不同程度的退行性改變後，再在外力因素的作用下，就會造成腰椎不適

腰椎不適

腰背疼痛是腰椎不適常見的癥狀，過度勞累受風著涼就容易誘發。

下肢放射痛

腰椎負重時從臀部、大腿後外側、小腿外側直到足跟部或足背的放射痛，主要表現為一側坐骨神經疼痛，咳嗽、噴嚏、用力排便可加重。

馬尾神經綜合症

重度腰椎突出嚴重會出現大小便失去控制甚至下肢癱瘓，這就是臨床上說的馬尾神經綜合症。

秋冬時節正是腰頸椎有問題多發的時候，這個季節氣溫偏寒，運動量減少，脊椎、肩部抵抗力下降，就會出現脊椎的勞損。人的脊椎負責支撐人體的所有重量，坐、卧、跑、跳，各種姿勢、活動都得依靠脊椎的支撐。脊椎的重要性不言而喻了。

沙療調養腰椎問題，沙療中所用的礦物沙經過加熱，會讓你的身體感到溫熱，熱流直達身體各個經絡，沙療有溫經散寒、調和氣血、補益下焦、清熱利濕的功效，可調養腰痛。沙療對不僅這種辦公室職業病，腰間盤突出、類風濕性關節炎等，都很好。
