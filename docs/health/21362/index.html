小分子肽治療銀屑病有奇效？保健品終歸只是輔助

2019-12-18 963條評論 3,735次閱讀 138人點贊

　　現在人們對於健康、養生是越來越關注了。前幾天有患者諮詢，說他了解到一個高科技產品——小分子肽，對治療銀屑病有奇效!所以想去嘗試一下能不能用小分子肽治好銀屑病。在展開對這個問題的論述之前，我先強調一下：對於銀屑病的治療，保健品終歸只是輔助!

　　一、市面上盛行的小分子肽究竟是什麼?

　　由2至15個氨基酸組成的就叫做小分子肽，而人體是最大的化學工場，蛋白質進入我們人體，是會被分解為肽類以及氨基酸供給身體需要的。市面上的小分子肽只是作為一種額外的補充。

　　有一點可能很多患者都不知道：醫院裡很多昂貴的藥物都是肽類的!而醫用肽效果比傳統藥物好但價格極其昂貴，是按毫克來計算的，不在醫保報銷範圍之內。像羊胎素、幹細胞，胰島素、催產素這些名字不含有肽的其實也是肽!

　　從以上我們不難看出，肽在醫療方面確實功能強大。但市面上盛行的小分子肽可就不見得了。隨著肽逐漸走進人們的視野，各種牌子的小分子肽也是層出不窮，但歸根結底，都是「食用級別」的肽。

　　二、網上宣傳的小分子肽對銀屑病有何「奇效」?

　　①加強營養：銀屑病的病因錯綜複雜，而有些患者就是因為消化系統不好導致缺乏某些微量元素，進而導致銀屑病。而且銀屑病會讓皮損處鱗屑脫落，流失大量的營養成分。小分子肽百分百吸收，銀屑病患者服用後可以快速的補充營養，靠細胞自己去修復疾病。

　　②提高免疫力：免疫異常是銀屑病重要的發病機制之一，而小分子肽中的一些氨基酸對合成人體免疫蛋白是有著非常重要的作用，能調節銀屑病患者的免疫功能。

　　③打通微循環障礙：微循環障礙是很多久治不愈的銀屑病患者都有的問題，細胞中的「垃圾」不能及時運出，「營養」不能及時補給，就會發生廣泛的微血管形態異常，也就是銀屑病的皮損。而小分子肽內服外敷，因為能透過皮膚被直接吸收，所以對銀屑病患者的微循環障礙有著很好的治療效果。

　　三、客觀論述小分子肽對銀屑病的作用

　　①能加強銀屑病患者的營養?我們人體是有一個自己修復的機制的，當它修復不了的時候就會生病，這個時候就需要做針對性的治療。而小分子肽即使外敷於銀屑病的患處效果也是一樣的：為細胞提供營養。「靠細胞自己去修復疾病」聽起來是很健康，但它真的能戰勝頑固的銀屑病嗎?當銀屑病已經在身體里「紮根」，就得祛除「病根」才能得到良好的治療效果。

　　②能提高銀屑病患者的免疫力?不僅是小分子肽，好像所有的保健品，都說有一個提高免疫力的作用?這時候肯定會有人反駁：小分子肽不是保健品，這是種食品，是細胞的食物!那這就是屬於食療，食療起效更慢，我們都知道黑芝麻能烏髮，但那是以年為計算單位的。

　　③能打通銀屑病患者的微循環障礙?「小分子肽能被直接吸收進細胞，為細胞提供能量，加快銀屑病患處的修復。」假設這是真的，也處在理想的情況下。問：我們人體有多少細胞需要補充肽?而小分子肽進入人體後會有多少肽會對銀屑病的患處起到作用?它只是食療級的，能讓人精力充沛，對身體健康有一定的作用，但沒法靶向銀屑病，對銀屑病的治療只能起到一個輔助作用。

　　④我們應該也聽說過「xx保健品吃出人命」的新聞，那種情況很少見但確實存在，補錯了地方反而會加重病情。而小分子肽屬於食補，相對來說很安全了，但也不可能達到「包治百病」作用。那些保健品是需要長期服用的，而且價格也比較貴，有經濟條件的患者可以酌情購買。

　　總而言之，不管是小分子肽還是其他的一些保健品，我們不能否認它們的作用，但那些是屬於「預防」而不是「治療」，而對於銀屑病的治療，保健品終歸只是輔助。
