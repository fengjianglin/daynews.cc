100克營養成分的加餐食物是多少

2019-12-12 107條評論 1,483次閱讀 504人點贊

尿病患者因為要控制飲食，有時候會容易肚子餓，不過適當加餐對身體是沒有壞處的，加餐得當還會有利於治療。很多糖友會選擇堅果、水果、蔬菜、餅乾、薯類或者小零食來加餐，這都是可取的，不過糖友可要算好總能量。

100克營養成分的加餐食物是多少呢

　　這些加餐食物的能量如何估算呢，下面就為您列舉常見的15種加餐食物，每100克含有多少能量、蛋白質、脂肪和碳水化合物。這樣，加餐時對食物的量和能量都能做到心中有數。

　　堅果組

　　南瓜子(炒)

　　南瓜子的可食部分為68%，圖為100克帶殼南瓜子的量。該示例份南瓜子含能量390千卡，蛋白質24.5克，脂肪31.3克，碳水化合物5.4克，膳食纖維2.8克。

　　葵花籽

　　炒制的葵花籽可食部分為56%，圖為100克帶殼葵花籽的量，約3小把。該示例份葵花籽含能量321千卡，蛋白質11.8克，脂肪27.5克，碳水化合物9.0克，膳食纖維2.5克。

　　扁桃仁

　　扁桃仁可食部為100%，圖為100克某品牌扁桃仁的量。據該扁桃仁的營養成分表，該示例份扁桃仁含能量538千卡，蛋白質23.5克，脂肪37.8克，碳水化合物25.9克，鈉81毫克。

　　堅果類零食還有很多，上述三種僅是舉例，希望「糖友」對100克堅果的量有一個直觀概念。在超市購買堅果類食物時，注意閱讀食品標籤，並有意識地控制食用數量。

　　餅乾組

　　蘇打餅乾

　　圖為100克某品牌蘇打餅乾的量，16片。據該品牌蘇打餅乾的營養成分表，該示例份蘇打餅乾含能量571千卡，蛋白質8.5克，脂肪36.1克，碳水化合物53克，鈉480毫克。

　

　 五穀雜糧餅乾

　　圖為100克某品牌五穀雜糧餅乾的量，8片。據該品牌五穀雜糧餅乾的營養成分表，該示例份蘇打餅乾含能量468千卡，蛋白質7.6克，脂肪21.9克，碳水化合物60克。平均1片能量50千卡以上。

　　鮮奶薄餅

　　圖為100克某品牌鮮奶薄餅的量，9包。據該品牌鮮奶薄餅的營養成分表，該示例份鮮奶薄餅含能量439千卡，蛋白質15克，脂肪11克，碳水化合物70克。1包所含能量近50千卡。

　　餅乾是糖尿病患者的常見加餐零食，每種餅乾的能量均不一樣。糖尿病患者最好能夠明確每一片餅乾的能量，這對糖尿病患者控制總能量攝入大有好處。通過以上數據可以看出，同等重量下，蘇打餅乾的能量並不低。

　　果蔬組

　　哈密瓜

　　圖為100克哈密瓜的量。該示例份哈密瓜含能量34千卡，蛋白質0.5克，脂肪0.1克，碳水化合物7.9克，膳食纖維0.2克。

　　葡萄

　　葡萄的可食部86%，圖為100克葡萄的量，約7粒。該示例份葡萄含能量37.4千卡，蛋白質0.4克，脂肪0.2克，碳水化合物8.9克，膳食纖維0.3克。

　　聖女果

　　圖為100克聖女果的量，10粒。該示例份聖女果含能量22千卡，蛋白質1克，脂肪0.2克，碳水化合物5.8克，膳食纖維1.8克。

　　迷你黃瓜

　　圖為100克迷你黃瓜的量。該示例份迷你黃瓜含能量12千卡，蛋白質1克，脂肪0.2克，碳水化合物2.5克，膳食纖維0.9克。

　　糖尿病患者不必和水果絕緣，只要血糖控制穩定，只要控制好總能量攝入，什麼水果都可以吃，但不能多吃，不必計較水果的甜度。聖女果和迷你黃瓜是是適合血糖控制不佳的糖尿病患者食用的蔬菜。

　　小零食組

　　山楂餅

　　圖為100克某品牌山楂餅的量，約8卷。該示例份含能量356千卡，蛋白質0克，脂肪4克，碳水化合物80克。平均每卷含能量45千卡。

　　紅薯

　　紅薯可食部為90%，圖為100克紅薯的量。該示例份紅薯含能量88千卡，蛋白質1克，脂肪0.2克，碳水化合物22克，膳食纖維1.4克。

　　玉米

　　鮮玉米的可食部為46%，圖為100克鮮玉米的量。該示例份玉米含能量45千卡，蛋白質1.7克，脂肪0.5克，碳水化合物9.6克，膳食纖維1.2克。

　　巧克力

　　圖為100克巧克力的量，1袋德芙純黑巧克力的重量為80克。據該巧克力的食物成分表，該示例份巧克力能量618千卡，蛋白質6.3克，脂肪44.9克，碳水化合物47.1克。平均1格巧克力能量約為60千卡。

　　乳酪

　　圖為100克乳酪的量，約6片。據該乳酪的營養標籤，該示例份乳酪含能量239千卡，蛋白質17.5克，脂肪12.5克，碳水化合物14.2克。平均1片約含40千卡能量。

　　小零食種類繁多，牛奶、酸奶、豆漿、豆腐乾等都是適合糖尿病患者的零食。建議糖尿病患者盡量選擇能量低的品種，最好自己製作，這樣更容易控制食物的能量。

　　本文列舉的食物僅是日常加餐食物的一部分，旨在培養糖尿病患者養成估算食物重量和能量的意識，然後通過稱量和查閱工具書確認，日積月累，對這些數值的估算將會越來越精確。如果糖尿病患者對吃的每一口食物的能量心中有個估算值，那控制總能量攝入又有何難!
