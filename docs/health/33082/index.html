預防眼乾眼澀妙招，護眼秘笈學起來

2019-12-23 3,967條評論 4,972次閱讀 3143人點贊

乾眼症在醫學上的定義是：「任何原因引起的淚液質和量異常，並伴有眼部不適，導致眼表組織病變」。乾眼症會引起眼睛乾澀，灼痛，眼屎較多，眼酸，眼癢，怕光和視力減退等癥狀，甚至會誘髮結膜炎、眼部感染，嚴重時可造成失明。所以乾眼症的預防和治療是不容忽視的。

研究表明，以下這些人群較容易患上乾眼症：

藥物使用者：某些藥物可使淚液分泌減少，導致乾眼癥狀，如抗抑鬱葯、抗組胺葯、利尿劑、降血壓葯、麻醉劑、抗潰瘍葯等。

長期使用某種眼藥水者：如長期使用血管收縮性眼藥水，很容易導致乾眼症。

受環境因素影響者：煙霧、紫外線、空氣污染、高溫、空調和氣候乾燥，都可加速淚液的蒸發，導致乾眼症。

某些疾病患者：患有類風濕性關節炎、糖尿病、甲狀腺異常、哮喘、白內障、青光眼及紅斑狼瘡等症的患者，較易患有乾眼症。

瞼板腺功能障礙者：患有瞼板腺功能障礙的患者，因淚液分泌和排出減少，容易導致乾眼症。

年齡超過65歲的人群：年齡超過65歲的人群中有75%患有乾眼症，因為老年人淚液的產生隨年齡的增加而減少。淚液分泌減少使眼睛易受刺激，有時會引起劇烈的流淚反應，即溢淚。

如何預防乾眼症？推薦幾種方法！

方法一：緩解視疲勞

每天努力工作的同時也應適當地放鬆一下眼睛，但並不需要總是眯著眼睛看身邊的所有人。例如，在打電話或考慮計劃時，可以閉上眼睛幾秒鐘。
如果每天在電腦前工作6到8個小時，建議每2至3小時休息一下，可以做眼保健操等眼球運動。

教你簡易3招動眼方式，以減輕雙眼疲勞：

1、調節觀遠看近

(1)選一個距離較近的景物，先觀看約5秒，然後再閉眼約5秒。

(2)轉換視線調到距離較遠的景物，再觀看約5秒鐘，然後閉眼約5秒，反覆幾次即可。

2、左、右交替觀看

(1)先雙眼向右看，保持約5秒，然後再閉眼約5秒。

(2)接著轉換雙眼向左看約5秒，再閉眼約5秒，重複數次後即可。

3、上、下交替觀看

(1)先雙眼向上看，保持約5秒，然後再閉眼約5秒。

(2)接著轉換雙眼向下看約5秒，再閉眼約5秒，重複數次後即可。

方法二：多眨眼，避免用眼過度

一般人每分鐘眨眼少於5次會使眼睛乾燥。一個人在電腦工作時眨眼次數只及平時的三分之一，因而減少了眼內潤滑劑和酶的分泌。應該多眨眼，每隔一小時至少讓眼睛休息一次。

方法三：手蒙住眼做深呼吸

搓手直到手掌發燙，然後用手掌遮住眼睛，深吸一口氣，想像黑暗中的寧靜。每天這樣做20分鐘可以減輕眼睛疲勞。

方法四：多喝綠茶防輻射又護眼

綠茶中的茶多酚，具有很強的抗氧化活性，可幫助清除人體內過量的自由基，從而起到抗輻射、增強機體免疫力的作用。綠茶有很好的防輻射、護眼效果，非常適合在電腦前工作的人。

方法五：多吃蔬菜給眼睛補充營養

薺菜中含有豐富的胡蘿蔔素，多吃薺菜能預防乾眼症，也可以減輕眼睛乾澀不適的癥狀。人眼的成像部位是視網膜，而視網膜有一個「黃斑」，裡面的黃色營養來源，就是葉黃素和玉米黃素。這兩類營養物質能減少紫外線對眼睛的傷害，延緩眼睛的老化，預防視網膜黃斑變性和白內障等眼疾。蛋黃中顏色為深黃色的，正是葉黃素和玉米黃素的來源。

菠菜深綠色葉菜中也含有大量葉黃素，根據研究，綠葉蔬菜還是維生素B2和β-胡蘿蔔素的好來源。維生素B2充足的時候，眼睛便不容易布滿血絲;而β-胡蘿蔔素，可以在體內轉變成維生素A，預防「乾眼病」等症。

乾眼症是一種慢性病，如果不給予足夠的重視會嚴重影響我們的日常生活。反過來許多不良的生活習慣也會加重乾眼症，久而久之形成惡性循環。除了養成良好的生活習慣外，外部的輔助治療也是很有必要的。
