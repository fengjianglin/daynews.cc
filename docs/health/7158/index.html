每個家庭都需要自己的“家庭營養師”

2019-11-26 1,282條評論 2,582次閱讀 284人點贊

//

民以食為天

//

民以食為天，我們每天都要依靠飲食生存，幾乎所有的養生專家都在講食物是最好的醫藥，然而大多數人並不知道如何利用飲食的營養保證身體健康。

在歐美、日本等發達國家，營養師與律師、醫師並稱為三師，營養師的社會地位是很高的。

在西醫診療體系中，營養也是非常重要的環節，然而，西醫引進到中國後，營養治療和干預的功能弱化，很多大中型醫院的營養科基本上淪為雞肋，地位很低。

在當前西醫過度醫療激化醫患矛盾的情形下，營養被人為弱化，是不是中國人對於西醫有「取其糟粕，去其精華」之悲哀呢！

//

我們什麼時候需要營養師？

//

現在的孕婦，從備孕、孕中到產後都特別注重營養，合理膳食、控制合理體重，即保證胎兒的健康發育，又減少孕期糖尿病、高血壓等威脅到孕婦健康疾病的發生。

到了孩子出生，新生兒的餵養又是一個大問題，如何優化母親乳汁、如何增加輔食、選擇什麼樣的輔食、如何製作輔食等、如何幫助新生兒提高免疫力等等。

再長大一點到學齡時期，如何讓孩子長的高一點都非常注重，生怕自己的孩子輸在了起跑線上……

然而，人到中年，人生中壓力最大的時期，也是人體健康的分水嶺，事業家庭兩頭都要照顧，卻忽略了自己的健康，總是以壓力大、應酬多、加班多、休息少作為借口，而疾病卻沒有任何借口，已在悄悄的積累了。

營養說難也難，說簡單也簡單……隨時隨地只要知道一些簡單的原理，就可以預防大部分疾病的積累。

//

為什麼要學營養師？

//

餐飲業的發展，促進了我國經濟的發展，但是，在社會公眾吃出的慢性病的發病率也著實令人吃驚。

餐飲業銷售額排名第一、第二位的火鍋和洋快餐。

據調查統計們吃火鍋時由於量或方法不合理，出現寄生蟲病、兒童哮喘病、痛風、膽囊炎、胰腺炎的發病率提高，甚至高出常人的幾倍。兒童肥胖與大量攝入洋快餐有直接關係等等。

我國14億人口，營養師不足40萬人。

並且，一部分已經大齡，退休休息，一部分在醫院做臨床營養師，從事疾病的治療。

而學校、企事業單位、健身中心、幼兒園、社區等需要普及營養知識的地方普遍缺相關的營養師。

僅解決我國居民營養師需求。大約需要400萬營養師。

一部分具有中西醫營養知識和技能的營養師，備受國際營養師市場的青睞。

//

營養師與醫藥的區別？

//

營養師的武器是均衡的飲食，特殊功能的食品，醫生的武器是藥劑和手術刀，醫藥只能賑災。

真正治好疾病而讓人保持的健康的只有全面的營養，因此，預防疾病，延緩疾病的發生，成為家庭人員的營養師是保持家庭成員健康的不二法門。

//

營養師與廚師的區別？

//

食物是最好的醫藥，一個合格的營養師應該從廚房開始。

營養師是透過食物或者特殊膳食與個體疾病、體質等進行合理搭配，來提供全面的營養。廚師只是將有營養的食物做的更美味，製作時，並不考慮是否符合享受美味的人的身體狀況。

早在兩千多年前的《黃帝內經》中就提出「五穀為養，五果為助，五畜為益，五菜為充」這一觀點，為我們提供了符合現代營養學觀點的平衡膳食原則。從大自然賜予我們的食物中汲取營養是最科學、最經濟、最安全的途徑。

現在，很多人越來越注重健康，在家給家人做各種養生鍋、滋補湯。但殊不知，食物是有屬性的，人的體質也是有差別的，有的人適合吃，有的人則不適合吃。

只有選對了食物，汲取的才是於身體有好處的營養素，否則就是有害物質，現在發病率較高的「富貴病」——高血壓、高血脂、高血糖、糖尿病、痛風……等，都和飲食有著密不可分的關係。

今天，移動互聯網的發展，給我們成為一個營養師提供了便捷的支持，使用工具，了解事物的屬性，了解人體的特徵，合理搭配，營養師的生涯就此開始……

一人懂營養，全家更健康，關愛家人，請從一日三餐、全面營養開始。
