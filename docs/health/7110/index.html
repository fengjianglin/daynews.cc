從中醫養生理論看頭采小青柑的清、潤、和、滋、養

2019-11-28 346條評論 1,475次閱讀 360人點贊

在中醫的理論上，講究五行相生相剋，以五髒的心、脾、肺、腎、肝對應五行的火、土、金、水、木。以五行相生關係形成中醫獨特的預防優於治理；而運用五行相剋規律，又形成中醫有別於西醫「頭痛醫頭，腳痛醫腳」的醫療方法。

茶以養生保健功效成為國飲，不是拿著元素表參照就可以曆數千年不衰的，靠的是先人對百草藥性的嘗試、運用、用中醫五行理論進行論證，最後作出總結。

寶福林是率先用五行論證頭采小青柑的保健作用的，總結出頭采小青柑「清、潤、和、滋、養」的五行相生特點。

/ NO.01 /

頭采小青柑·清心火

2019 寶福林頭采小青柑 花果香系列

寶福林頭采小青柑具有清心火功效。心火旺盛者有心煩、失眠、便秘、尿黃、口舌生瘡等癥狀，常見於過食辛辣刺激、溫補之品，或暑熱所致，而頭采小青柑有很好的緩解作用。

/ NO.02 /

頭采小青柑·潤肺燥

花果香·頭采小青柑

寶福林頭采小青柑具有潤肺燥功效。肺燥者有乾咳無痰、咽喉癢、聲音嘶啞等癥狀，常見於乾燥季節，又可由心火盛而轉化成。寶福林頭采小青柑取自新會7月頭采果，營養豐富，具有平喘理氣、清潤肺燥的天然功效，七月頭採的小青柑表面易起白霜，功效更為顯著。

/ NO.03 /

頭采小青柑·和脾胃

冰糖甜·頭采小青柑

寶福林頭采小青柑具有健脾和胃功效。現代人飲食常無節制，經常冷熱交錯，容易造成氣虛乏力、胃脹胃痛、拉肚子、眼睛浮腫，這是脾胃虛弱的癥狀。頭采小青柑對於調理脾胃的功效是很理想的，經長期存放的效果更是顯著。

/ NO.04 /

頭采小青柑·滋腎水

蜜韻·生普小青柑

寶福林頭采小青柑具有滋腎水的功效。長期的飲食不當、喜喝酸性飲料或喜食太咸、飲水少都會導致加重腎臟的負擔，伴有精神難以集中、尿頻、肥胖等。頭采小青柑裝填的是優質雲南陳年熟普/生普，具有提神、利尿、降脂的效力，頭采柑皮具有顯著的理氣功效，使肺氣能清肅下降，兩者結合對減輕腎臟壓力有一定的幫助。

/ NO.05 /

頭采小青柑·養肝血

寶福林頭采小青柑具有養肝血的功效。肝司職全身氣機的暢通，調節人體精、氣、神、血、水的正常運轉。若肝之疏泄失職，就會出現氣機不調，血行不暢，進而肝火上亢、肝氣鬱結，使體內毒素無法分解和排除。頭采小青柑采自新會柑核心產區天馬村的老樹柑果園，樹齡大都有7、8年之上，所產柑皮厚實油潤，具有很好的疏肝破氣功效，藥效更勝一籌。

有些消費者得知青柑皮具有「疏肝破氣」功效，就認為是會損元氣，這是完全錯誤的。悶氣積聚在胸中，現代稱憂鬱症，破氣指的是散鬱結，這與傷元氣是不同的。

結語：以上就是關於「頭采小青柑清、潤、和、滋、養」的養生價值。值得一提的是，頭采小青柑在味道上更勝一籌，入口極致鮮爽，回味香甜，在柑普茶中是難得一品之極韻，至今也是眾多老茶客喝柑普茶的佳選。
