心血管健康飲食原則？如何避免意外發生？如何運動保護血管？| 健康問答 086 期

2019-10-22 470條評論 2,642次閱讀 1319人點贊

秋日天氣忽冷忽熱，早晚溫差顯著加大，室內外溫差也非常明顯，此時更是心血管疾病的多發季節。

心血管

Q1：心血管患者在飲食上要注意什麼？哪些良好習慣助於調理心血管疾病？

一般來說，心血管患者應清淡飲食，少吃辛辣刺激油膩食物，忌煙酒。

1、少食多餐，減少高熱量食物的攝入。

2、低鈉、低脂、低膽固醇飲食，少食動物內臟或動物脂肪；每日膽固醇量控制在300毫克以下或200毫克以下。

3、多食豆類及其替代品，以獲取優質的植物蛋白。

4、增加膳食纖維的攝入量，多選清蒸、燉、煮、燴等烹飪方式，避免油炸、油煎等烹飪方式。

5、多關注血壓、血脂和血糖的情況，以及遵醫囑按時服藥。

Q2：秋冬季節如何避免發生心血管意外？

秋冬季節因多變的氣溫及寒冷的刺激，易誘發心血管疾病的意外發生。

首先要防寒保暖，根據氣候改變適時添衣，避免著涼。

其次，在暖氣供應的室內及時補充水分，避免體液流失過多導致血液黏度過高，引發心肌梗死的意外。

再次要適量運動，天氣晴朗溫度適宜時，可多增加室外活動或室外逗留時間，以提高機體的禦寒能力。

但老年心血管患者不建議外出，尤其是晨練運動。

若出現各種嚴重的胸悶、胸痛癥狀，要立即就醫或是撥打120。

Q3：心血管患者該如何健康運動？

適當的運動能明顯改善心血管功能，改善情緒和提高睡眠質量。

合理的運動應包括熱身、訓練和放鬆的環節：10分鐘左右熱身、20-30分鐘訓練、10分鐘左右放鬆。

運動期間忌憋氣，進行體力範圍內的有氧運動，如散步、慢走、快走、爬樓梯或緩坡。

下午4—6點運動為宜，並延長熱身時間，運動期間出現不適，逐漸放慢動作，待癥狀緩解後再休息。

預防心血管，從家庭養護著手

心血管疾病是人類健康的頭號殺手，每年死於心血管疾病的人數高於其它任何病因的人數。

高血脂、高血壓、高血糖及煙酒是心血管的四大高危因素，除此之外過度疲勞、劇烈運動、寒冷、情緒激動都會影響心血管健康。

因此不妨空閑時，煲一份秋日安神養心的護心湯——丹參田七雞湯，溫補又養神。

取10g丹參，3g田七和一份雞肉，由於湯中加入丹參和田七，小朋友和經期女性不可服用哦~

用清水浸泡雞肉一小時，隨後取出鋪在燉盅底部；洗凈丹參田七放入燉盅，倒1000毫升清水（4人份，可斟酌增加或減少），燉煮三小時即可飲用。
