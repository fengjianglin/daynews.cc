這種黑不溜秋的蒜，和我們日常吃的蒜有什麼區別嗎？

天天要聞
2019-12-15

在烹飪或腌制美食時加入蒜是一種家常便飯的做法。但是，近年有一種黑蒜「異軍突起」，憑藉極高的營養價值極高，而且味道酸酸甜甜，紅遍了大江南北，正逐漸被人們認識和認可，正逐步走向百姓生活。

食用黑蒜的歷史可以追溯到古代埃及時代，大蒜即因是滋補強身和解除疲勞的特效食品而被推崇，不僅口感「甜、軟、糯」而且具有具有抗氧化、抗酸化的功效，對糖尿病、高血壓、高血脂、癌等疾病亦有療效。

黑蒜還有最好的抗癌食品的口碑。在1990年美國國立癌症研究所發表的《Designer Foods Program》中大蒜因預防癌症最有效而被列在48種常見的抗癌食品之首。同時，它可降低血液中的多餘膽固醇及糖份，也可調整胃腸的蠕動。

黑蒜是如何製作成的？

黑蒜是用新鮮的生蒜，帶皮放在高溫高濕的發酵箱里發酵60~90 天，讓其自然發酵製成的食品。

簡單的步驟是選材——浸泡清洗——甩干晾曬——分級挑選——發酵成品

  • 酶促發酵：需要10天，是黑蒜的酶素激活期。
  • 中溫發酵：需要30天，分解出多種氨基酸。
  • 高溫熟成：需要50天，微量元素含量顯著增加。
  • 常溫後熟：需要30天，讓發酵更充分。

怎麼做更好吃？

黑蒜的做法可以豐富多彩的，這裡推薦幾個經典的菜品。

  • 黑蒜燉乳鴿

選料：凈土雞、赤肉、乳鴿、姜蔥、黑蒜、鹽

做法：

1、把凈土雞1隻和赤肉1500克放入鍋里，加姜蔥並摻清水，用小火煨9小時左右，然後除去湯汁里的油脂，即得湯底。

2、把乳鴿塊和黑蒜分別放入多個湯盅內，調入少許鹽，摻入湯底，然後送入蒸箱里蒸4小時，即成。

  • 黑蒜叉燒

選料：五花腩1斤2兩，黑蒜50g、姜、蔥、叉燒醬各適量。

做法

1.五花腩用黑蒜、姜、蔥、叉燒醬先腌制大概2個鐘頭，以讓花肉入口松化。

2.掛爐燒，中火燒45—50分鐘，然後再掛蜜汁，再放入燒，反覆2次。

3.出餐時鋪上黑蒜即可。

  • 黑蒜橡樹果火腿

選料： 西班牙橡樹果火腿100克、黑蒜50克、番茄風味粉30克、香甜沙拉醬30克、淡奶油15克、檸檬汁3克

做法：

1.沙拉醬、風味汁粉、淡奶油、檸檬汁調勻，放入黑蒜拌勻，再裹上風味汁粉。

2. 火腿切成超薄的片狀，捲起裝盤，配上醬汁和黑蒜即可。

黑蒜保存方法

黑蒜質感比較軟，不宜抽真空包裹太緊，否則會擠壓壞黑蒜，黑蒜拆封后，吃完封嚴袋口，例如比較好的黑蒜，黑蒜一般可以保存3個月，不拆封可以保存時間較長，但必須放在陰涼乾燥處保存。
