中醫望耳診病，哪些變化才能說明臟腑出了問題？

2019-10-23 287條評論 1,430次閱讀 399人點贊

耳郭是人體相對獨立的外在器官，雖然僅占人體總面積的1%左右，但卻是最能敏感反映人體臟腑生理、病理信息的窗口。《靈樞·師傳》就有「視耳好惡，以知其性」的記載。因此，通過觀察耳郭色澤、形態、皮疹、脫屑等變化，分析、辨別臟腑功能狀態和病理改變，以輔助診斷疾病，這樣的「望耳診病」方法為歷代醫家所重視和應用。基於「有諸內者，必形諸外」和「司外揣內」等中醫學傳統理論，通過觀察大量病例的耳診種種表現，歸納、整理望耳診病的原理及其在臨床實踐中的觀察內容及注意事項。

望耳診病原理

耳與經絡的關係

中醫經絡理論認為，人體經絡系統內屬臟腑、外絡肢節，是人體氣血運行的通路，五臟六腑通過經絡與機體組織器官、四肢百骸相互關聯。耳郭之所以能夠反映人體臟腑組織器官的生理病理變化，也是通過遍布周身的經絡系統的聯絡功能得以實現的。《靈樞·邪氣臟腑病形》首先提出：「十二經脈，三百六十五絡，其血氣皆上於面而走空竅……其彆氣走於耳而為聽。」元代《衛生寶鑒》也指出：「五臟六腑，十二經絡有絡於耳者。」說明耳與全身是一個統一的有機整體。

耳與臟腑的關係

中醫藏象學說認為，經絡的聯絡功能使人體五臟六腑與五官七竅密切聯繫，而耳與五臟功能的聯繫尤為重要。如《靈樞·脈度》指出「腎氣通於耳，腎和則耳能聞五音矣」，《素問·金匱真言論》曰：「南方赤色，入通於心，開竅於耳，藏精於心」，《證治準繩》曰：「腎為耳竅之主，心為耳竅之客」，《素問·玉機真臟論》認為「脾為孤臟……不及則令人九竅不通」，《素問·臟氣法時論》提出「肝病者……虛則……耳無所聞……氣逆則頭痛，耳聾不聰」，《難經·四十難》則記載了「肺主聲，令耳聞聲」，以及清代《雜病源流犀燭》提出「肺主氣，一身之氣貫於耳」的見解，都說明五臟與耳的關係密不可分。

常用耳穴分布規律

耳穴是臟腑經絡之氣輸注於耳郭皮膚表面的部位，也是人體臟腑經絡、組織器官與耳郭皮膚表面的相應之處。因此，耳穴不但可以反映臟腑、經絡的生理功能和病理變化，還可用來診斷疾病。

人體在耳郭上的投影位置猶如一個倒置的胎兒，即頭面在下，位於耳垂；內臟居中，主要位於耳甲腔、耳甲艇；腳在上，位於對耳輪上、下腳。因此，耳穴的分布也具有相應的規律，如與頭面部相應的穴位在耳垂；與上肢相應的穴位在耳舟；與軀幹相應的穴位在對耳輪體部；與內臟相應的穴位在耳甲腔和耳甲艇；與下肢相應的穴位在對耳輪上、下腳；與盆腔內部分生殖器官相應的穴位在三角窩。通過常用耳穴分布規律分析，可明確疾病病變的臟腑部位。

常見耳郭病理反應

根據學者的研究和我們在臨床實踐中的長期觀察，常見耳郭病理反應的主要表現可以概括為耳郭局部顏色改變和形態改變兩種類型。

顏色改變

常見耳郭皮膚的顏色改變主要有白色、紅色、青紫色、褐色、灰色和黑色幾種類型。

白色：耳郭皮膚的白色改變多見於慢性疾病，如慢性炎症、缺血性疾病等氣血不足的病理改變，屬中醫的虛證、寒證範疇。

紅色：耳郭皮膚的紅色改變又可分為鮮紅色、暗紅色、中間白色周圍紅色3種類型。

（1）鮮紅色：耳郭皮膚的鮮紅色改變常見於新病、急性病，如急性痛症、急性炎症、出血性疾病等，屬中醫實證、熱證範疇。也見於某些正常生理狀態，如月經期、剛進食不久、耳郭受到揉捏等刺激時。

（2）暗紅色：耳郭皮膚的暗紅色改變多見於痛證、病程較長者，如慢性或陳舊性損傷、久病、痛證等氣血瘀阻、運行不暢的病證，屬中醫瘀證、實證範疇。

（3）中間白色、周圍紅色：耳郭皮膚出現中間白色、周圍紅暈包繞的改變多見於慢性疾病急性發作期，如慢性病、慢性炎症急性發作，多屬中醫虛實夾雜的證型。

青紫色：耳郭皮膚出現青紫色改變多見於疼痛性疾病，《靈樞·論疾診尺》指出「耳間青脈起者掣痛」，故可見於外傷疼痛、痛經等病證，屬中醫血瘀證、寒證、實證範疇。

褐色：耳郭皮膚的褐色改變可見於陳舊性疾病、勞損、疾病癒後，如陳舊性外傷、肌肉勞損、頭痛、痔瘡、中風後遺症等，多屬中醫氣滯血瘀、痰濕阻滯之證。

灰色：耳郭局部皮膚的灰色改變多見於慢性疾病、鈣化病灶、惡性腫瘤，屬於中醫邪毒內阻、氣血經絡失暢等證。

黑色：耳郭皮膚的黑色改變雖可見於痣點，但有時也可提示該部位存在病症，屬於中醫瘀證、痹證的範疇，多由氣血失和、脈絡瘀阻而成。有些則可能與家族病史有關。

形態改變

耳郭常見的形態病理改變主要表現為凸起、凹陷、脫屑、丘疹、皺褶、囊皰等幾種反應類型。

凸起

（1）點狀凸起：圓點狀、高於耳郭皮膚的凸起，多見於臟腑組織器官的急性炎症、慢性炎症、增生、結石、息肉、肌瘤等。

（2）絲、線狀凸起：略高於耳郭皮膚的細絲狀或細線狀血管充盈，可見於外傷後、痛症、神經衰弱、失眠等病症。

（3）條嵴狀凸起：明顯高於耳郭皮膚的條狀血管充盈、軟骨增生所致的凸起，可見於較明顯或較劇烈的腰背、關節痛症，或外傷所致的筋骨損傷等病症。

（4）片狀凸起：略高於耳郭皮膚的薄片狀凸起，可見於陳舊性肌肉勞損、炎性疾病恢復期、瘢痕遺留等病症。

（5）塊狀凸起：界限清晰、質地較柔軟、明顯高於耳郭皮膚的各種凸起，可見於腫瘤（多為良性）、腦血管硬化、中風後遺症等病症。

（6）結節狀凸起：表面不光滑、形態不規則、質地較硬的塊狀凸起，可見於痛風結石、骨質增生、腫瘤等病症。

（7）串珠樣凸起：連續2～3個或多個形狀相近、順序排列的軟骨凸起，主要見於脊椎退行性增生。

隆起：與凸起相比較，隆起局部的耳郭皮膚呈光滑圓潤膨出，質地更柔軟，多見於脂肪肝、肥厚性胃炎、腫脹（水腫、血腫、氣腫）等病症。

凹陷

（1）點狀凹陷：低於耳郭皮膚表面、形狀較小的凹窪，可見於組織器官的缺損性疾病，如胃潰瘍、十二指腸潰瘍、膽囊摘除術後、肺結核鈣化點等。也見於女性排卵第1、2天。

（2）條溝樣凹陷：低於耳郭皮膚表面的線狀、條狀凹槽，常見於術後瘢痕印記、骨折等。有時與皺褶相伴出現，如冠狀動脈粥樣硬化時出現的耳折征。

（3）片狀凹陷：耳郭皮膚表面不規則、面積稍大的淺凹，可見於胃出血、胃潰瘍、潰瘍性結腸炎、心肌供血不足、臟器摘除術後等。

脫屑：耳郭皮膚上出現的白色糠皮樣或鱗片狀、擦去復生的皮屑，常依據皮屑出現部位及臨床癥狀不同而診斷各異，多見於各類皮膚病、消化不良、咽炎、生殖系統疾病、老年性皮膚瘙癢症等病症。

丘疹：高於耳郭皮膚、頂部較尖的細小單個或多個針尖樣凸起（類似雞皮疙瘩或痱子），呈紅色或白色，多見於炎症，尤以黏膜炎症為常見，如婦科疾病、腸道疾病、腎炎、心肌炎、房室傳導阻滯、慢性氣管炎等。

皺褶：耳輪及耳垂部位的表面皮膚擠壓、摺疊所出現的垂直或斜行條溝狀彎曲、下陷變形。臨床上多見於心臟血管動脈硬化、糖尿病等病症。

囊皰：高於耳郭皮膚表面的水皰狀凸起，水皰內可有液態內容物，常見於炎症、積液、息肉、囊腫等病症。

望耳診病注意事項

望耳時直接觀察耳部特定部位，即觀察耳穴皮膚上出現的變色、變形、脫屑、丘疹以及血管變化等觀察時注意以下方面：

環境：採光充足，室溫適宜，環境安靜，受試者情緒穩定。

光線：視診時最好選擇自然光線，必要時可藉助燈光，以便觀察耳郭的細微變化，同時應避免周圍有色物體的反光影響。

操作方法：受試者可選用坐位或卧位，盡量使受視者耳郭位置與檢視者兩眼平行；去除耳郭飾物，充分暴露檢查部位。不易觀察的部位，如耳甲艇、三角窩等處，可用手指在耳背輕輕頂起，或用探棒撥開影響視線的部位進行檢視，必要時用放大鏡進行觀察；視診前不要擦洗、消毒、提拉耳郭，以免引起血管擴張、充血而致皮膚顏色改變，也可避免造成陽性物消失。耳郭不潔時，可用棉球輕輕擦凈，休息片刻再進行視診；視者可以一手拇、食指輕輕牽拉受視者耳郭，對著光線，自上而下，由外而內地按耳郭表面解剖部位順序仔細觀察；發現陽性反應物時，應仔細辨別其位置、大小、色澤、形態等變化，必要時可用手指或探棒探查反應物的範圍、硬度、移動性及有無壓痛等；若反應物不易辨別時，可將繃緊的皮膚慢慢放鬆，再慢慢繃緊，進行反覆觀察；必須將兩側耳郭進行對照觀察。
