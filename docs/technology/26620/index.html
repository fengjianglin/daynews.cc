新車智能情報站12月03期丨新勢力來襲，合資不甘示弱，自主持續發力

2019-12-22 465條評論 1,658次閱讀 741人點贊

在過去的七天，有多款新車上市，包括新邁騰、長安福特銳際、愛馳U5、寶沃BX3、銷量王新款吉利遠景等，這些車型都搭載了最新的車載互聯繫統。其中，造車新勢力愛馳U5終於上市，既然是新勢力造車企業，車機智能化絕對是很重要的一環，相比之下，合資品牌也不甘示弱，大眾最新的「眾行家」智能網聯繫統也終於在邁騰上上車，而自主品牌在智能化方面更是領先一小步。接下來，我們就一起盤點下本周上市的重點新車，看看都有哪些智能化亮點。

雪佛蘭創界雪乎藍限量版

上市日期：12月16日

指導售價：16.59萬元

搭載配置：MyLink智能車載互聯繫統（支持OTA）

技術亮點：雪佛蘭目前「最強」的車載系統

技術點評：

這套系統涵蓋了智能導航、娛樂音樂、資訊資訊、實時天氣、交通生活等實用功能，只需用手機號在車機上創建個人賬戶並登錄，就能實現手機與車載共有app的一鍵貫通，可支持CarPlay、CarLife連接。另外，該車搭載的安吉星車聯繫統，可實現碰撞自助求助、緊急救援協助、道路導航、遠程操控免提電話、車況檢測等服務。

新款邁騰/邁騰GTE

上市日期：12月18日

售價區間：25.39-26.89萬元

搭載配置：「眾行家」智能網聯繫統（支持OTA）、IQ.DRIVE智能駕駛技術（L2級）

技術亮點：充分展現大眾汽車對於未來出行的前沿技術和理念

技術點評：

1、「眾行家」智能網聯繫統是隨新邁騰首次面世，該系統具備語音識別、手機遠程控制、藍牙鑰匙、組隊出行、遠程控制智能家居、在線支付、車載WIFI等功能，支Carplay/Carlife/MirrorLink等主流智能互聯，已經達到業內的主流水準。在未來，充電樁的密度和位置、自動泊車、甚至ETC支付等都會被整合進「眾行家」內，為消費者出行道路上遇到的問題提供便利。

2、IQ.DRIVE智能駕駛技術包括複雜環境下的輕易泊車、Aera View 360°鳥瞰式全景可視泊車系統、一鍵travel assist、0-160km/h全速域橫縱控制、ACC3.0自適應巡航系統、24小時在線的全旅程自動剎車、前後預碰撞保護系統、HOD方向盤離手監測等等。

長安福特銳際

上市日期：12月19日

售價區間：18.98-21.88萬元

搭載配置：SYNC+車機系統（支持OTA）、Co-Pilot360智行駕駛輔助系統（L2級）

技術亮點：語音智能水平顯著升級

技術點評：

1、SYNC+車載娛樂系統、福特以前自研的SYNC系統在語音交互、網聯在線和娛樂出行服務等功能上都是比較欠缺的，所以福特也是與百度深度合作，帶來了整合百度人工智慧的車載OS SYNC+，利用百度的人工智慧、語音語義識別技術以及百度豐富互聯網生態，並結合福特整車以及雲端建立平台。在日常行車中，還可以直接完成百度智能導航、個性化語音播報這些基本功能。除此之外，還結合了QQ音樂和愛奇藝等娛樂平台。

此外，福特派Ford Pass遠程互聯繫統，可實現遠程解鎖、啟動車輛以及開關空調等功能。

2、Co-Pilot360智行駕駛輔助系統的強大在於，系統以車身的攝像頭和雷達為基礎，快速的進行演算法分析，在堵車的時候，ACC全速自適應巡航+智能感應制動可以解放你的雙手雙腳；在路況複雜的時候，盲區監測+車道保持輔助可以更加精準的進行提醒；在狹窄的停車場中，360度全景影像+主動泊車輔助又可以幫你避免剮蹭，可以說非常實用。

愛馳U5

上市日期：12月19日

補貼後售價區間：19.79-29.21萬元

NEDC工況續航：503km

搭載配置：四維智聯（支持OTA）、智能自動駕駛輔助系統（L2級）

技術亮點：智能交互融入各個環節

技術點評：

1、車機系統方面，愛馳汽車攜手四維智聯打造了智能互聯和智能語音交互系統，將愛馳 ID貫穿於購車、駕駛、智能應用交互、甚至用車生活的各個環節，實現了對每個用戶進行專屬服務。包括：遠程服務系統AI-Key、多端互聯繫統AI-Life、出行服務系統AI-Go、車輛管家系統AI-Keeper、語音交互系統AI-Talk、個性定義系統AI-Me、社交娛樂系統AI-Joy、創新服務系統AI-X等功能。通過手機遠程控制APP，用戶就可通過AI-ID體系的雲同步，實現和愛馳U5智能座艙的多端互聯，體驗AI技術帶來的便利。

2、愛馳U5 的自動駕駛水平為L2級，根據愛馳汽車的說法，其從感測器、控制器、執行機構的選取，到整車的系統集成和測試，均達到了行業最新水平。具體來看，愛馳U5的智能自動駕駛輔助系統，可在全速段（低速城市擁堵、高速公路）本車道內（含彎道）保持跟車或自動巡航，可實現高階AEB（自動緊急制動系統）功能、開門預警、後側方預警、變道輔助、全場景自動泊車等功能。

寶沃BX3

上市日期：12月20日

售價區間：9.68-13.08萬元

搭載配置：B-Link智能互聯繫統（不支持OTA）

技術亮點：交互方式多樣

技術點評：

該系統支持蘋果和安卓系統，能夠涵蓋信息資訊、休閑娛樂、安全防護、交通駕駛、診斷保養和便捷服務6大方面49項功能，以綜合精準的操作和貼心的內容、高速互聯的車聯網雲服務，全面覆蓋使用者出行生活的每一個角落，全面滿足年輕消費者對更智慧、更個性、更酷炫的消費追求。


新車還支持通過手機上的寶沃汽車APP進行遠程控制車輛，比如在夏天和冬天，能夠提前通過手機打開車內的空調，還是非常實用的，發動機開關、車身門鎖、車窗升降、座椅加熱等都可以遠程控制。寶沃汽車APP還可以查看車輛情況，比如行駛里程、電瓶情況、油量情況都可以通過手機查看，同時還支持一鍵呼叫、一鍵尋車、導航倒車等功能。

奔騰X40EV 460

上市日期：12月20日

指導售價：18.78-19.38萬元（尚不清楚享受多少國家補貼）

NEDC工況續航：352km

搭載配置：D-LIFE智能車載互聯繫統（支持OTA）

技術亮點：自帶社交屬性

技術點評：

D-LIFE作為一款帶著社交屬性的車聯網產品，它的用戶活躍度達到了78.5%，用戶平均使用時長為2h/天。我們再看一看當下主流的微信和QQ用戶活躍數據，微信用戶活躍度為93%，QQ的用戶活躍度為71%。從數據的角度來看，D-LIFE是一款成功的車聯網產品。2.0版本又增加了語音控制和遠程控制兩大功能，實現了整車手機全面操控。

新款吉利遠景

上市日期：12月20日

指導售價：5.39-7.39萬元

搭載配置：G-LINK手機交互系統（支持OTA）

技術亮點：經濟實用的人車互聯

技術點評：

全新遠景提供便捷的智能互聯體驗，8英寸大屏顯示清晰，觸摸按鍵簡潔、響應靈敏，集成倒車影像、GPS導航生活資訊等功能，並搭載智能語音控制系統與G-LINK手機交互系統，幫助用戶通過口令輕鬆實現播放音樂、收聽廣播、開關空調、地圖導航等車內功能，簡化駕乘操作，充分感受智能駕駛的豐富樂趣。
