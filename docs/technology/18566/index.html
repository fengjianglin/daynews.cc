盤點2019年最聰明的五台車，最貴的是特斯拉，最便宜的僅不足10萬

2019-12-18 483條評論 3,203次閱讀 1533人點贊

國內汽車市場風起雲湧，各大車企紛紛大力布局，在10多年的飛速發展中，中國汽車消費市場已屹立於世界之巔。但物極必反，多年的飛速增長一直迎來了2018年的市場寒冬，並且還延續至2019年，這也加速了國內汽車行業的洗牌。只有真正把控消費者需求的優質產品，才能佔據自己的一席之地，並且在智能科技融入汽車行業的今天，一台車的智能程度尤為重要，我們今天就來盤點一下2019年最聰明且最值得關注的五台車，其中包括自主品牌與合資品牌，最貴的產品是特斯拉，最便宜的還不足10萬。

最值得關注且最聰明的車第五名：特斯拉中國 Model3，售價35.58萬

隨著特斯拉超級工廠的建成，特斯拉國產化的腳步也在跟進，不僅將Model3的售價拉低，同時組裝工藝也所有提升。Model3外觀沿用家族式設計，內飾功能控制也以一塊15英寸大屏為主，整合了車聯網系統與OTA在線升級，也支持語音控制，這是聰明的一部分。

聰明還體現於特斯拉的輔助駕駛系統，這也是它引以為傲的技術，全速自適應巡航為標配，還可以選裝完全自動駕駛，其召喚功能甚至可以在停車場讓車輛尋找用戶，聰明程度可見一斑，基本可以解放雙手雙腳。但由於純電動的能源類型、高昂的售價以及選裝包的5.6萬價格，它只能屈居第五名。

第四名：理想ONE，售價32.8萬

作為國內造車新勢力的代表車企，理想ONE的到來，為消費者提供新的選擇，增程式的能源類型解決了里程焦慮。理想ONE定位中大型SUV，提供六座及七座兩款車型，但售價相同。內部的四塊大屏是理想ONE的亮點，它也擁有車聯網與OTA在線升級功能，語音識別支持控制導航、天窗、空調等功能。當然，它的聰明同樣也體現於輔助駕駛系統。

理想ONE也擁有全速自適應巡航，在主動剎車與車道保持輔助等一系列輔助系統的加持下，可以實現L2級別的智能駕駛輔助，這也是它聰明的最重要部分。但是它與Model3的問題同樣在與高昂的售價，另外，部分消費者可能也不會接受增程式的能源類型，所以它排名第四。

第三名：2020款沃爾沃S60，售價區間28.69萬-37.99萬

上一代的S60L已銷售多年，所以它的換代勢在必行，2020款的S60採用最新的家族設計語言，將整車的氣質提升不少，更符合現代的審美。作為一台換代車型，沃爾沃S60標配了車聯網系統，支持CarPlay，同時也擁有語音識別功能，能控制空調、導航等功能，這是它聰明的基礎。

一向以安全著名的沃爾沃，為這台S60標配了主動剎車系統，包括車道保持輔助也是標配，並由此整合了全速自適應巡航，高配車型還擁有道路指示牌識別功能，達到L2級別的駕駛輔助系統，為消費者提升很大的便捷性。不過由於剛上市，沃爾沃S60售價較為堅挺，並且全系車型均沒有OTA在線升級功能，所以它排名第三。

第二名：2020款哈弗H9，售價區間20.98萬-27.28萬

作為國內自主品牌SUV的頂尖選手，2020款哈弗H9不僅通過”三把鎖”將自身硬派越野車的身份發揮到極致，在智能化程度方面也與時俱進。9英寸中控屏也整合了車聯網、OTA在線升級以及語音控制功能，多媒體系統的聰明程度緊跟時代。

聰明也體現於哈弗H9的輔助駕駛系統，車道保持輔助、主動剎車等各項配置的加入，衍生出ACC自適應巡航，在高速路段，基本也能解放駕駛者的雙手雙腳。不過，輔助駕駛並不是全速自適應巡航，但它出色的越野性能也非常值得關注，所以它排名第二。

第一名：新寶駿RS-5，售價區間9.78萬-13.58萬

作為誕生於新時代的車企，新寶駿所有的產品都帶有互聯網特色，這台新寶駿RS-5也不例外。10.25英寸中控屏整合了車聯網系統、OTA在線升級、語音識別功能，並且還是全系標配。通過與手機連接，它幾乎可以實現手機支持的所有功能，甚至還能用語音回復微信。

當然，它的輔助駕駛也不遜色。新寶駿RS-5的全速自適應巡航能達到L2級別的駕駛輔助，車道保持、道路交通指示牌識別等功能一應俱全，城市道路與高速路段都能為駕駛者提供便捷性。多項行業首創功能的加持、合理的售價，讓新寶駿RS-5的第一名當之無愧。

總結：通過對這五颱風格迥異，定位與售價完全不同的產品分析之後，不難發現，我們的自主品牌如今並不遜色於高端的合資品牌，甚至是豪華品牌。尤其是新寶駿RS-5，更是在某些部分做到超越合資品牌，它的第一也理所應當，它也堪稱10萬級別家用車的最佳選，期待它能有更好的市場表現。
