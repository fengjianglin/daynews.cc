最新：埃及多家銀行開通人民幣業務；美國將對92%日本商品零收費

2019-12-04 2,417條評論 3,364次閱讀 833人點贊

美國谷歌面臨重大人事調整，兩位創始人宣布辭職

據媒體12月4日報道稱，美國科技巨頭谷歌兩位聯合創始人拉里·佩奇和謝爾蓋·布林宣布，將控制權轉交給谷歌現任首席執行高管。與此同時，這兩位創始人也卸任在谷歌職位，但會繼續作為谷歌母公司Alphabet的董事會成員。此消息一出，Alphabet股價當日上漲近0.5個百分點。據了解，自今年以來谷歌的財報一直不盡人意，其運營的核心數字廣告業務明顯出現放緩跡象。Alphabet打算加大對其他業務的投入，其中包括Waymo和Verily等公司。

蘋果公司股價下跌3%，創數月來最大單日跌幅

據媒體12月4日報道稱，本周二美國蘋果公司股價意外下跌多達3%，創下自8月23日以來最大的單日跌幅。據悉，蘋果股價自6月創下最低點以來已經上漲了近50%。但在iPhone11系列產品的熱度逐漸褪去之後，蘋果股價已經連續第三日下跌。值得注意的是，本周一摩根大通還預測稱蘋果或推出重大戰略：將蘋果手機的發行時間改為每年兩次。此外，該機構還表示蘋果將會在明年推出4款5G手機。可以看出，為了逆轉頹勢，蘋果正在積極做出行動和改變。

埃及多家銀行開通人民幣業務 人民幣成全球第六大支付貨幣

據媒體消息，在當地有關監管部門的批准下，近日埃及銀行、埃及商業國際銀行開通了人民幣業務，企業可辦理匯兌、結算、出具信用證等業務。據悉，早在2016年，為了便利雙邊貿易和投資，埃及央行就與中國人民銀行簽署了總額180億元人民幣的貨幣互換協議。隨著人民幣國際化進程加快，根據環球同業銀行金融電訊協會（SWIFT）的數據，截至2019年10月，人民幣在國際支付中的市場佔有率為1.65%，為全球第六大支付貨幣。

日本通過美日貿易協定 美國將對92%日本商品零收費

據媒體最新消息，今日（12月4日），日本有關部門通過了與美國的貿易協議，將於明年1月1日起生效。根據該協定，對美國牛肉的”額外費用”將從目前的38.5%逐步降至2033年的9%。一位日本官員透露，根據美日達成的貿易協定，若按價值來計算，美國將豁免約92％的日本商品的費用，而日本將對84％的美國商品零收費。不過，在日本最在意的汽車收費問題上，目前美國依舊沒有給出明確的說法。

2701.1億元蟬聯中國首富，馬雲再登《福布斯》雜誌封面

近日，馬雲再次登上了美國知名雜誌《福布斯》的封面，此前馬雲在2000年阿里巴巴剛剛成立時也曾登上雜誌封面。另外在福布斯12月3日發布的”2019中國富豪榜”上，馬雲也以2701.1億元的財富蟬聯中國首富。儘管擁有巨大的財富，但馬雲這次被福布斯關注主要是其公益基金會等新事業，而馬雲在接受採訪時也表示，雖然已經退休，但是他不會離開這個舞台。

美國每年55萬人無家可歸，紐約因”趕人”被告上法庭

儘管美國一直聲稱該國的經濟和股市表現不錯，但另一個數據或許能夠揭露美國經濟社會的”真面目”——無家可歸人數。數據顯示，2018年全美國有55萬人無家可歸，其中僅紐約市就有78676人，佔比高達14%。近日，附近的紐瓦克市市長將這個美國最大的城市告上法庭，指責紐約市將這些無家可歸者驅趕至自己的城市。生活在底層的人們因為無家可歸，被這些城市”趕來趕去”，很難想像這是發生在美國這個世界第一大經濟體的事情。
