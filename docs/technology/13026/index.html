海南新能源汽車售後服務規範徵求意見公布

2019-12-11 578條評論 2,389次閱讀 934人點贊

海南省商務廳12月5日發布《海南省新能源汽車售後服務規範（徵求意見稿）》（以下簡稱《規範》），向社會公開徵求意見。】《規範》涉及術語和定義、經營者義務、三包責任及爭議處理、廢舊動力電池組回收要求、安全風險防控要求等方面內容。新能源汽車指小型、微型純電驅動載客汽車以及商用汽車，包括純電動汽車、插電式混合動力汽車及燃料電池汽車。

一、在經營者義務方面，《規範》提出，售後服務承諾至少包括產品質量保證承諾、續駛里程承諾、售後服務網路建設、動力電池回收、安全風險防控、索賠處理等內容。續駛里程承諾包括正常續駛里程、高低溫使用環境下的續駛里程衰減程度、關鍵零部件包修期內續駛里程衰減程度承諾等。

二、　在三包責任及爭議處理方面，《規範》提出，

1、新能源汽車包修期不低於3年或12萬公里，以先到者為準。

2、新能源汽車動力電池及管理系統、驅動電機及其控制器、整車控制器等關鍵零部件包修期不低於8年或12萬公里，以先到者為準。

3、新能源汽車三包有效期不低於2年或5萬公里，以先到者為準。

4、新能源汽車自銷售者開具購車發票之日起60日內或者行駛里程3000公里之內，動力電池、驅動電機的主要零件出現產品質量問題的，消費者可以選擇免費更換動力電池、驅動電機。

5、新能源汽車自銷售者開具購車發票之日起60日內或者行駛里程3000公里之內，新能源汽車出現強電漏電等產品質量問題，消費者選擇換車或退車的，銷售者應當負責免費換車或退車。

6、在三包有效期內，動力電池、驅動電機累計更換2次後，或者動力電池、驅動電機的同一主要零件因其質量問題，累計更換2次後，仍不能正常使用的，動力電池、驅動電機與其主要零件更換次數不重複計算，消費者選擇換車或退車的，銷售者應當負責換車或退車。海南省新能源汽車推廣應用中，保留了天然氣汽車。

意見反饋截止日期為12月27日。聯繫人：陳大富，聯繫電話：65351653，郵箱：441241827@qq.com.
