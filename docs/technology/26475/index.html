美國第3大晶元企業賣資產，正是華為急需的，可惜買不到

2019-12-22 1,827條評論 3,313次閱讀 250人點贊

眾所周知，這幾年雖然中國的半導體產業蓬勃發展，但由於起步慢，再受到一些技術的封鎖，而半導體產業周期長，投資大，門檻高，所以總體不是太如意的。

所以這些年以來，很多的中國企業都想通過收購一些國外先進的半導體企業，來完善自己的產業結構，進而提高自己。

所以每當國外一些半導體企業想要變賣資產時，國內的企業都是蠢蠢欲動，甚至拋出橄欖枝，可惜很多時候並不能成功，因為不一定能夠通過國外審核機構的審核，畢竟某些國家不願意中國在半導體領域變得強大起來。

近日，美國第三大晶元企業也要變賣自己的資產了，而這部分資產是華為急需的，甚至可以說是目前中國半導體界急需的，可惜的是，雖然情況是這樣，但基本是買不到的，美國不可能通過審核，所以大家也只能看看就好。

這家美國第三大晶元公司就是博通，目前在全球半導領域可以排第五，而在美國可以排在第三，僅次於英特爾、美光之後，博通想要變賣的是射頻（RF）晶元產業。

為什麼說射頻晶元是華為，甚至是整個中國急需要的？在2017年時國產射頻晶元市佔率僅有2%。到2018年時，提升了一些，但份額還不到5%，可見還是差得很遠的。

而射頻晶元是移動通信系統的核心組件，尤其當5G到來時，射頻晶元更顯得重要了，而博通是這一領域的前3強之一，尤其它生產的基於智能手機的濾波器，以使信號更清晰，其技術是該行業的「市場領導者」，蘋果都是其客戶。

當然，正如前面所言，國內廠商是想買也買不到，實在有點可惜了，要是能夠買下來，一定能夠改變國內射頻晶元的格局，尤其如果華為能夠買下來，就真的太好了，可惜啊。

目前網上的傳聞是蘋果或100億美元接盤，畢竟蘋果也急需射頻晶元，當然目前的結果如何還不知道，不過國內企業是真的看著眼紅，卻又吃不到。
