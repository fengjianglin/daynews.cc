絕不向霸權主義屈服！法國再次把矛頭對準美，向谷歌開出11億罰單

2019-12-22 352條評論 1,840次閱讀 54人點贊

據央視新聞客戶端本月21日報道，法國競爭事務監察總署20日發布公告稱，因美國谷歌公司「濫用在互聯網廣告市場的主導地位」，該機構決定對谷歌罰款1.5億歐元（約合11億元人民幣），並要求谷歌進行整改。

該監察總署認為，谷歌對平台用戶施加的運營規則和方法不透明，內容晦澀難懂，存在誤導客戶的嫌疑，而修改規則的權力一直是谷歌自己的，這種行為不平等且非常武斷，最終導致廣告商和用戶的權利雙雙受損，要求谷歌闡明廣告平台規則。

僅從法方給出的這個理由來看，谷歌並沒有犯什麼大錯，廣告費用是谷歌收入的一個大頭，規則本就非常多，由於是面向全球廣告商的，要做到面面俱到，冗雜自然是無法避免的。也就是說，法國開出這張巨額罰單的理由非常牽強。

法國為什麼要把矛頭指向谷歌呢？從眼下的利益來講，法國對谷歌開刀，就是對美國的反擊，法國在今年下半年的時候，通過了數字費用的提案，將對互聯網企業收取額外費用，而歐洲的物聯網企業基本上全都來自美國，因此法國這個法案被特朗普認為是在針對美國。

美國很快就還以顏色，要對法國的葡萄酒等核心產品多收費，預計在明年1月份開始實施。不過法國並沒有被嚇到，而是選擇跟美國死磕到底，再次對美國的物聯網巨頭下手，用實際行動告訴美國，絕對不會屈服於霸權主義。

從長遠的利益來看，歐洲要想擺脫對美國互聯網企業的依賴，必須要發展自己的互聯網行業，而這些美國企業已經成為了攔路虎，因此歐盟一直都對這些美國企業非常苛刻，正所謂欲加之罪何患無辭，谷歌這種企業只要挑毛病，絕對能挑出來。

當前歐美的關係越來越緊張，在英國脫歐的大背景下，法國作為歐盟當中僅剩的五常國家，當仁不讓地成為了歐盟的領頭羊，常言道能力越大責任越大，因此法國必須要出這個頭，跟美國死磕到底。

美國在跟各方都達成了協議之後，矛頭已經再次對準歐盟，對於美國來說，歐盟剛剛換了新領導，必然要趁其立足未穩之時發起攻勢，這樣才能把歐盟徹底安排明白，確保今後歐盟能夠「聽話」，然而目前來看，美國想達到這個目的並不簡單，歐盟新掌門人也不是什麼善茬，不過不管誰笑到了最後，西方世界的分裂已經無法避免。
