2020年初，國內或將出現巨大財富機會，馬雲、劉強東已搶先布局？

天天要聞
2019-12-18

縱觀世界各大佬，無論是世界首富美國亞馬遜的老闆貝索斯還是中國首富阿里巴巴的馬雲，都是做電商平台起家的。不難看出，前十位的富豪基本都是做互聯網或者IT行業的。但是，還有一個行業非常造富，那就是零售行業，多年蟬聯世界500強榜首的沃爾瑪，就是全球最大的的零售企業，要不是沃爾頓將財產分給了幾位子女，那他應該就能當上世界首富。包括剛剛進入中國的costco超市，在上海開了第一家門店，開業三小時被迫關門，原因是商品直接被搶空，門都被擠爆了。

說到零售業，那我們不得不說說我們國家的零售市場，據統計，2018我國零售總額超38.1萬億，已經連續五年持續穩定增長，目前穩居世界第二位，零售和餐飲總額與美國的差距越來越小。2019年的社零總額可能直逼世界第一的位置。

2004、2005、2008年的中國首富國美電器的黃光裕是第一個零售業的首富，但是由於後來的網上零售的平台產生，線下的實體店受到巨大衝擊，生意一落千丈。接下來就是淘寶，京東的崛起，也就是馬雲，劉強東的故事，馬雲也是一躍成為中國首富。那時的馬雲曾說過：實體店將會被電商全面取締。馬雲是一位非常具有前瞻眼光的企業家，當時的實體店確實舉步維艱，門可羅雀。就連大潤發這樣的零售巨頭都無奈含淚退場……但是2016的雲棲大會上面馬雲提出「新零售」，並講到：純電商的時代已經過去了，未來只有線上線下相結合，才能誕生真正的新零售。」第一次零售時代被馬雲說准了，這次電商行業估計將沒落，迎來的將是互聯網+實體零售。

2年前，萬達掌門人王健林開玩笑地說，近年來中國零售業許多新名詞，比如張近東提出「智慧零售」，馬雲提出「新零售」，劉強東提出「無界零售」，「現在就等馬化騰再提個什麼」。玩笑歸玩笑，但這句大實話沒毛病，直接反映出阿里、京東、蘇寧在爭奪新一輪新零售變革話語權時的膠著狀態。可以看出，在新零售這個領域，各界大佬也是紛紛布局，爭取這第一杯羹，就連做遊戲的馬化騰也要參與其中了。

過去的十年，抓住房地產和金融二者之一，基本上不會過得太差。如果說上一輪資產飆升，是「膽大的」淘汰「膽小的」，那麼下一輪經濟轉型，就是「有文化的」淘汰「沒有文化的」。因為在5G、人工智慧、互聯網的催促之下，未來沒有眼光和智慧的人，將徹底邊緣化，完全沒有存在感。

現階段，好的財富風口在哪裡？

隨著拼多多一躍成為全球第二大電商平台，大家瞬間不敢小看三四線下沉市場的購買力了。在線上拼團玩得波詭雲譎，線下也有一股力量在不安躁動。那就是社區電商的雛形，社區拼團。

有人說，社區電商是線下版「拼多多」。的確在2018年總共有十幾家社區電商獲得了融資，合計超過45億元，眾多資本紛紛看好這片萬億級藍海。

可以看出，當今，所有做得好的生意，都有社群的元素在裡面！無社群，不營銷！
