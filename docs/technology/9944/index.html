淘集集併購重組失敗 啟動破產清算程序

2019-12-08 1,310條評論 4,766次閱讀 2560人點贊

科技訊 12月8日消息，淘集集正式走上破產道路。淘集集CEO張正平稱，公司併購重組的確失敗了，投資人已經簽了投資協議，但是在最後打款階段出現多次拖延；公司各賬戶被訴前保全，司法凍結，特別是支付寶賬戶被凍結，使得公司無運營資金；公司按照破產重整、破產清算兩條路來走。

今年10月15日，淘集集CEO張正平在官方微博發布了道歉信，承認資金鏈斷裂。張正平表示，目前公司負債主要分為兩部分，供應商的貨款和廣告代理投放的欠款，負債總額為16億元左右。淘集集於2018年8月上線，僅僅幾個月的時間就獲得了1.3億註冊用戶。

在道歉信中，張正平稱，「進入（今年）7月，由於內外部一些因素，業績增長受到了極大的影響，銷售額出現停滯，這裡我犯了一個極大的錯誤，過多的時間花在了融資身上，想通過融資款來解決當前增長的問題，延誤了最黃金的自救期，策略上邊選擇了繼續虧損獲取用戶。進入9月，由於融資遲遲得不到確認，現金流開始下降，危機已經來臨。」

10月16日淘集集宣布，主要經營模式由商家入駐模式，調整為合伙人自營模式。淘集集稱，現有主要供應商轉為淘集集股東合伙人，與淘集集同榮辱共進退。

10月31日，淘集集官方發布重組併購進程通報內容是：「10月23號，供應商債權人完成51%債務重組協議簽定;10月28號，收到資方書面TS，簽定投資意向書。當前淘集集併購重組進展順利;公司運營穩定;平台運營穩定。

在11月19日，淘集集官方微博還稱，「融資重組已進入收尾階段，請各位夥伴耐心等待官方的發文，再次感謝各位夥伴與淘集集一起同舟共濟，共度難關。」（一橙）
