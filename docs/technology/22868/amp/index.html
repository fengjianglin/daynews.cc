中國汽車健康指數結果發布，10款車型5個五星，長安CS75PLUS賺了

天天要聞
2019-12-20

2019年12月19日，中國汽車健康指數管理中心在中國汽車健康指數官網、官微發布了中國汽車健康指數2019年第二批車型測評結果暨推薦車型。

​

本次線上發布2019年第二批共計10款車型VOC/VOI及EMR測評結果，發布車型涵蓋自主品牌（4款）、合資品牌（5款）及進口車(1款)，囊括燃油車（8款）和純電動車（2款）。

​

​

同時發布了中國汽車健康指數2019年第二批推薦車型和分板塊推薦車型，獲得推薦車型的分別是：

科魯澤2019款320T手動欣快版國VI（VOC&VOI板塊得分94.5，評級為5星；EMR板塊得分94.75，評級為5星）

​

特斯拉 Model 3 2019款長續航全輪驅動版（VOC&VOI板塊得分92.5，評級為5星；EMR板塊得分99.19，評級為5星）

​

長安CS75 plus2020款自動豪華型（VOC&VOI板塊得分96，評級為5星；EMR板塊得分89.49，評級為5星）

​

VOC/VOI板塊推薦車型：長安CS75 plus2020款自動豪華型

​

EMR板塊推薦車型：特斯拉 Model 3 2019款長續航全輪驅動版

​

推薦車型根據《中國汽車健康指數「推薦車型」評選辦法（2019）》執行。

從中國汽車健康指數VOC/VOI板塊的測評結果來看：本次測評的十款車型中，有五款車型達到五星標準，它們分別是重慶長安汽車股份有限公司的CS75plus、上汽通用汽車有限公司的科魯澤、特斯拉汽車（北京）有限公司的Model 3、北京賓士汽車有限公司的GLC260、長安福特汽車有限公司的福克斯。

其中重慶長安汽車股份有限公司的CS75plus以96分奪魁。

​

從中國汽車健康指數EMR板塊的測評結果來看：本次測評的十款車型中，有七款車型達到五星標準，它們分別是重慶長安汽車股份有限公司的CS75plus、上汽通用汽車有限公司的科魯澤、特斯拉汽車（北京）有限公司的Model3、北京賓士汽車有限公司的GLC260、長安福特汽車有限公司的福克斯、中國第一汽車集團有限公司的T77、廣州小鵬汽車科技有限公司的G3。

其中特斯拉汽車（北京）有限公司的Model 3以99.19分奪魁。

​

12月20日，中國汽車健康指數將正式發布年度推薦車型，車評酷也將在後期針對2019年第二批車型測評結果進行解讀，請持續關注哦~
