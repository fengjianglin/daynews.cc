亞馬遜起訴：特朗普有偏見，讓我們丟百億美元合同

2019-12-10 278條評論 2,912次閱讀 1675人點贊

科技訊12月10日消息，據外媒報道，美國時間周一，亞馬遜旗下亞馬遜網路服務公司(Amazon Web Services Inc.)在公布的一份訴訟書中稱，該公司參與了美國國防部價值100億美元的雲服務合同的競標，但美國國防部在招標過程中未能做出公正地評判，導致其競標失敗，背後的原因是美國總統唐納德·特朗普將該公司創始人傑弗里·貝索斯（Jeffrey Bezos ）視為自己的「政敵」。


亞馬遜表示，儘管微軟未能滿足所謂的「聯合企業防禦基礎設施(Joint Enterprise Defense Infrastructure，簡稱JEDI)」合同的要求，但美國國防部無視亞馬遜的先進技術，將合同授予了微軟。

AWS指控稱，美國國防部犯那些錯誤，是由於特朗普的不當干涉。AWS在起訴書中表示，特朗普「多次公開或幕後攻擊，引導JEDI合同遠離AWS，以傷害他認為的‘政治敵人’貝索斯。」根據這份起訴書，特朗普長期以來一直批評貝索斯擁有《華盛頓郵報》的所有權。

上月，亞馬遜向法院起訴美國國防部，正式抗議其失去美國國防部的雲服務合同。美國國防部的JEDI項目旨在整合其雲計算基礎設施並使其技術系統現代化。該合同價值高達100億美元，為期10年。更重要的是，贏家可能在蓬勃發展的雲市場中佔有更大優勢。（劉春）
