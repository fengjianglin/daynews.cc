神經網路入門的第一本書，你應該選擇它

天天要聞
2019-12-15

如果只讀一本神經網路入門圖書，請選擇這本《Python神經網路編程》[英] 塔里克·拉希德（Tariq Rashid） 著。基於Python 3.5，對神經網路精彩解讀。

本書所探討的就是神經網路，讓你了解神經網路如何工作，幫你製作出自己的神經網路，訓練神經網路來識別人類的手寫字元。如果使用傳統的方法來執行這個任務，那麼將是非常困難的。

讀這本書需要數學知識嗎？

本書將帶領您進行一場妙趣橫生卻又有條不紊的旅行——從一個非常簡單的想法開始，逐步理解神經網路的工作機制。您無需任何超出中學範圍的數學知識，並且本書還給出易於理解的微積分簡介。本書的目標是讓儘可能多的普通讀者理解神經網路。讀者將學習使用Python開發自己的神經網路，訓練它識別手寫數字，甚至可以與專業的神經網路相媲美。

讀者群：

本書適合想要了解深度學習、人工智慧和神經網路的讀者閱讀，尤其適合想要通過Python編程進行神經網路開發的讀者參考。

《Python神經網路編程》共分３章：

  • 在第1章中，我們將如清風拂面般，一覽在簡單的神經網路中所用的數學思想。我們有意不介紹任何計算機編程知識，以避免喧賓奪主地干擾了本書的核心思想。
  • 在第2章中，我們將學習足以實現自己的神經網路的Python知識。我們將訓練神經網路，識別手寫數字，並且會測試神經網路的性能。
  • 在第3章中，我們將進一步了解簡單的神經網路，這超出了了解基本神經網路知識的範疇，但是我們這樣做只是為了獲得一些樂趣。我們將嘗試一些想法，進一步改善神經網路的性能，我們將觀察已受訓練的神經網路內部，看看我們是否理解神經網路所學習到的知識，是否理解神經網路是如何做出決定進行回答的。

我們使用的軟體工具都是免費開源的，你無須支付任何費用。你也不需要一台昂貴的計算機製作自己的神經網路。本書中的所有代碼都已經經過了測試，可以在價廉物美的樹莓派Zero上運行。在本書的末尾，附錄B介紹了如何讓你的樹莓派準備就緒。

豆瓣評分9.2
