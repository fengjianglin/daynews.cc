買下3萬畝荒地，如今價值等於雷軍整個小米，堪稱最大的隱形富豪

天天要聞
2019-12-21

最近晨興創投再次拋售小米的股份，套現了20億。晨興創投已經減持了小米大約10%的股份，套現了估計在200億以上。另外晨興創投的還擁有小米大約10%的股份，目前價值也在200億左右。

說起晨興創投就不得不提起香港的陳啟宗家族。晨興創投就是陳啟宗家族創辦的創投公司，最初的資金大部分都是來自於陳家的家族資金。

其實在香港還有一個隱形富豪，單單是他們現在家族擁有的地皮，就足以等於整個小米的價值。目前小米的市值是2千億左右，而這個家族在多年之前，在廣州買下大量的荒地，如今這些荒地價值高達2千億。這個豪門就是霍英東家族。

1978年，霍英東來到廣州南沙考察。霍英東的眼光確實非常驚人。當時從香港來南沙，需要坐船再換車再換船，前後需要3天的時間，不過霍英東卻認為南沙的發展潛力驚人。

當時霍英東在地圖上用尺子一量，他發現南沙從海上去香港才90公里。同時南沙去澳門也非常近。霍英東量了一下，原來南沙是整個珠三角地區的幾何中心。南沙的北邊是廣州主城，東邊是深圳、東莞，西邊是珠海，南邊出海是香港。

當時的南沙幾乎是一篇荒地，只有一些人在挖礦，連一條像樣的路都沒有。不過基於這種考慮，霍英東馬上決定開發南沙。其實能夠有眼光看中南沙的人，肯定不止霍英東一個人。為何卻只有霍英東一個人去南沙呢？

因為在當時來說，開發南沙的成本太高了，從投資回報的角度來說，回報率太低了，甚至隨時會虧本。霍英東願意投資南沙，是因為他不是從投資的角度來看，他是希望真正幫國家把南沙發展起來。把南沙發展成為第二個項目，這也是霍英東提出來的。

霍英東初到南沙，就買下了21平方公里的地皮，總計3萬畝地皮。這些地皮當時都是荒地，而且是一片灘涂。霍英東當時認為這3萬畝地皮是南沙的精華，擁有南沙的7公里的黃金海岸線。

拿下這地皮之後，霍英東就開始自己掏錢在南沙修建公路，如今南沙的南沙大道跟進港大道等路都是霍英東出資修建的。這些路都是20多年之前就修建的，但是如今看來依然一點都不落後。另外霍英東還修建了南沙大橋等等。

當時南沙許多地都是坑坑窪窪，霍英東自己投入非常多資金才把這些地給填平了，另外南沙許多基建設施也是霍英東出資建設的，包括了南沙碼頭。

霍英東前前後後在南沙投入了60億的資金。把基建打好了，一個區域還得有人，才可以發展起來，特別是人才。為了吸引人才到南沙發展，霍英東想了很多辦法，包括在南沙興建學校跟酒店等等。不過這些投入大部分都是入不敷出。霍英東在南沙囤積的地皮，直到他去世的時候，都沒有賣掉一平方米，也沒有開發一平方米的房地產。

霍英東是真正從長遠角度來發展這片地方的。如今南沙已經逐漸發展起來，並且有了許多政策的扶持。王健林跟許家印也先後在這裡砸下重金。王健林開發了南沙的萬達廣場，許家印則把新能源汽車的產業中心放在了裡面。

南沙的地價也是飛漲。霍英東去世13年之後，霍家才在南沙推出了第一個房地產項目。如今霍家還手握3萬畝地方，如果以一畝地700萬計算，那麼價格就高達2千億，剛好可以把雷軍的小米都買下。

買下3萬畝荒地，如今價值等於雷軍整個小米，堪稱最大的隱形富豪，最近幾年已經去財富榜上看不到霍家的身影了，因為他的許多財富都隱形起來了，可以說現在霍英東的子孫依然是香港最大的隱形富豪。
