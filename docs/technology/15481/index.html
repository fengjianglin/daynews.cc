templatetags里有多少個模塊沒有限制，因為｛% load %｝會載入給定模塊名的標籤過濾器，而不是應用的名稱

為了成為一個可用的標籤庫，必須註冊，放在模塊頂部

from django import template
register=template.Library()

自定義標籤過濾器

自定義過濾器就是一個帶有一個或兩個參數的Python 函數：

  • （輸入的）變數的值 —— 不一定是字元串形式。
  • 參數的值 —— 可以有一個初始值，或者完全不要這個參數。

例如，在{{ var|foo:”bar” }}中，foo過濾器應當傳入變數var和參數 “bar”。

下面是這個過濾器應該如何使用：

{{ somevariable|cut:"0" }}

大多數過濾器沒有參數。在這種情況下，你的函數不帶這個參數即可。示例︰
