科技類股票繼續飛，存儲器借力，40億投入國產之光等待崛起

2019-12-18 964條評論 3,383次閱讀 878人點贊

電子產業的基礎，半導體行業具高投資價值：作為技術密集型產業，半導體行業具有較高技術壁壘，隨著今年以來行業景氣周期見底，半導體行業需求逐步回升，疊加5G、物聯網帶來的增量機會，多重政策催化，國內半導體行業廠商將顯著受益，行業具有較高投資價值。

而存儲器又是半導體銷售市場上佔比最高的，生產企業主要集中在在美國、韓國、中國大陸以及中國台灣地區，主要產品類別分為DRAM (內存條顆粒)、NAND Flash (硬碟存儲器)，其中DRAM市場份額為59%佔比最高，其次是NAND Flash佔比40%。

DRAM領域主要生產廠商有三星(份額第一)、 SK海力士(份額第一)以及美光(份額第一)。NAND Flash市場，三星、東芝、西數、美光、SK海力士五家廠商市場佔有率合計超過90%，市場壟斷程度較高。

雖然在2019年上半年，前有MY摩擦，後又日韓對原材料禁運，半導體市場受多面夾擊，加上很多廠商去庫存還未結束，存儲器的價格也處於下跌通道，導致整個半導體產業都是在連續下滑。但是要知道2018年存儲器在大幅降價的情況下，全球銷售額仍能保持1579億美元，增速仍有27.4%。

現在5G、物聯網（IOT）、AI、汽車電子、區塊鏈、新能源六大需求開始崛起，硬體使用量和設備終端數量將顯著增長，到2020年左右或集中爆發，形成景氣周期共振，帶動半導體增量機會，存儲器自然也是一樣迎來紅利，在2016年之前，存儲器銷售額佔到整體半導體市場銷售規模20%左右，但是在2016年，由於存儲器價格在2017年及2018年出現大幅上漲，銷售額佔比一度高於30%，預計在2019年佔比仍將保持在30%左右。

我國又是全球最大的存儲器銷售市場，據統計201 8年我國存儲器市場銷售額就高達5575億元,相比2017年的4303億元，銷售額增長了34.2%，佔全球市場份額的約55%，份額未來還將持續提升。

某公司是一家晶元設計企業。公司核心人員組成，每位都曾在全球著名IC企業工作多年。

公司立足已有Flash和MCU 業務，主要產品分為快閃記憶體晶元產品、微控制器產品以及感測器晶元產品，其中快閃記憶體晶元產品主要為NOR Flash和NAND Flash兩類。整合新加入的基於觸控和指紋識別的感測器業務，包括指紋識別晶元，包括電容式、光學式以及超聲波式指紋識別晶元，微控制器覆蓋率穩居市場前列。推進產業整合，拓展戰略布局，打造存儲、控制、感測、互聯、以及邊緣計算的一體化解決方案，未來有望大量應用於物聯網、汽車、工業控制等領域。

公司研發投入增速一直保持高於營收增速，2019Q3研發投入為2.43億元， 同比增長46%， 佔到營業總收入比例的11%，創歷史新高。

公司在10月擬募資近40億元投入到DRAM晶元研發及產業化項目，該項目主要是研發1Xnm級(19nm、17nm)工藝製程下的DRAM技術，設計和開發DDR3、LPDDR3、DDR4、LPDDR4系列DRAM晶元，該項目的實施有助於豐富自身產品線，完善公司在存儲領域的布局，成為國內高端通用存儲器領域領軍企業。

公司立足存儲領域，布局感測和控制產業鏈，是國內IC 設計企業中在物聯網時代最為受益的標的，所處賽道市場空間大，國產替代需求強烈，未來業績成長具備大幅想像空間。
