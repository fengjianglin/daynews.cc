熱點新聞

  1. 紐約數字投資集團（NYDIG）已獲得美國證券交易委員會（SEC）的批准，向機構投資者提供專註於比特幣期貨的新型基金。
  2. 以太坊2.0零階段正在進入測試最後階段，有望於2020年初啟動零階段主網。
  3. 北京地方金融監督管理局黨組書記、局長霍學文表示，未來北京將重點在兩個區域開展監管沙箱探索，一是西城與海淀相鄰地區的金融科技與專業服務創新示範區，二是城市副中心。
  4. 安全公司PeckShield表示，PlusToken跑路資金EOS部分正通過SWFT平台混淆轉移。

長沙多家養老機構爆雷：單個公司涉案金額達十餘億元

近日，長沙多起養老機構集體「爆雷」，上萬老人中招，涉案金額超10億。湖南警方在通告中指出，2018年湖南已偵破此類案件45起，涉案金額高達35億。

幾萬名老人含辛茹苦幾十年攢下的養老錢，這其中不光有「缺乏判斷力」的老人，還有央企職員、幹部家屬、大學教授。

總有一個騙局適合你，這樣一派美好的景象幾乎在中國每一個城市每個月都在發生，結局大都是「終於還清了房貸，卻住進了養老院，最後還被養老院掃地出門」的慘劇。

有人說，在中國，兩種錢最好賺，一種是孩子的錢，一種是老人的錢。為了孩子的教育，老人的健康，人們很容易落入圈套。

雪崩發生時，沒有一片雪花是無辜的。

都說我國的養老市場是一個千億級的藍海，但這麼多年過去，養老行業卻陷入了一個怪圈：

養老越來越難，60歲以上老人高達2.5億，養老院床位提前一年預約，都不一定能排的上；官方支持的正規養老院盈利卻是遙遙無期，開一家黃一家，難以為繼；騙子養老機構賺的盆滿缽滿，無數人養老錢被搜刮一空；一切又回到原點，老人不光沒地養老，連養老錢都沒了。最終騙子也逃不過法網，但沒有一個人真正受益。
