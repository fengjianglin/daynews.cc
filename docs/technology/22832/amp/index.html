聚划算賣豬肉

19塊9一斤！全網最低價！

16日上午，阿里旗下團購網站聚划算助力安徽大別山土豬出道，項目上線後，當天豬肉總售量高達2噸。

其中，經補貼後19塊9一斤的近千份特價豬肉一推出即被消費者「秒無」，搶購熱潮異常火爆。

要知道，在現在的肉價市場，豬肉動輒即三四十塊錢一斤。

這19塊9一斤的土豬肉，直接就是低於線下超市價格近半的全網最低價豬肉了。

在當下的肉價市場，打出如此重磅的折扣價？逆天了吧。

聚划算這麼自信，主要有以下幾方面的原因。

第一，大別山土豬肉供應採用的是基地規模化供應方式，這很大程度上降低了豬肉的養殖成本。即在源頭上下功夫把控好成本。

第二，原產地直供、冷鏈物流直送到家的模式，直接打掉層層代理商，避免中間商賺差價。

第三，實時進行售量監控，精準洞察消費者的需求，確保按需屠宰，避免庫存壓力。

第四，進行簡單粗暴的「百億補貼」，進一步把價格壓至全網最低。

經過一系列組合式的有效操作後，19塊9一斤的平價大別山土豬肉新鮮出爐。

可能有人會說，這小量的特價土豬肉，在中國偌大的消費市場，簡直微不足道。而且，許多消費者還不一定就能搶得到呢。

是的，聚划算這一舉動，要想解決中國幾億人餐桌上的豬肉問題，的確是異想天開。

但這看似不起眼的一步，實際上卻釋放出一個重磅信號：豬肉價格將會進一步下降，市場態勢將會朝良好方向持續發展。

為什麼這樣說呢？

這得從豬肉價格狂漲的原因說起。

電商助農，進一步刺激豬農養豬積極性

今年下半年以來，豬肉價格節節攀升，用「瘋狂」來形容也一點不為過。

做個對比：去年8月20日生豬的價格是13.87元/公斤，一年後的同段時間，價格漲到了24.34元/公斤，相比去年足足上漲了75%。

而到了菜市場，零售價幾乎翻了倍，普通豬肉已經在30-40元/斤區間。若是大型超市的品牌豬肉，其價格更是漲至近50元/斤起。

管你是壹號土豬，還是東升牌土豬、安康走地豬，通通都站在了風口之上，讓老百姓望而卻步。

那為什麼會出現這種情況呢？

簡單來說，就是市場需求大、供應少了，尤其是後者。

受今年非洲豬瘟影響，在半年間，中國生豬被撲殺數飆升至100萬，致使各地豬肉供應大幅度減少。

除此以外，受農村人居環境整治、畜禽養殖環境污染整治等多重因素影響，國內生豬的產能亦急劇下降，存欄量在年中跌至低點。

故此，在消費需求龐大，而供應量嚴重不足的情況下，國內豬肉價格一路狂飆，創下歷史新高。

所以說，要想徹底解決高價豬肉的問題，提高市場供應量，實現市場供需平衡是關鍵所在。

我們現在回過頭來看，那聚划算的「助農賣空豬肉」到底意義何在呢？

一方面，它是一次扶貧活動，助力農村把農產品賣出去，幫助其脫貧致富。

但另一方面，它也在進一步帶動農村地區人們自主創業的激情，刺激他們積極生產。

在今年，聚划算先後賣空雲南千畝玫瑰花田、四千畝特優級沃柑林、四百萬枚綿陽雞蛋、三百噸海口火山荔枝、四百噸棗陽皇桃……頻繁創下中國電商農產品上行紀錄。

這似乎在打包票說：你使勁生產，儘管生產，不管有多少，我們都可以幫你賣空掉。

毫無疑問，這大大地刺激了農村地區農產品的生產力度。

回到豬肉上來說，也是同樣的道理。在電商的促動下，養殖戶的積極性得到大大提高，從而進一步擴大其生豬養殖量。

待養殖量一旦跟上來，市場的豬肉供應量必定大增，那豬肉價格恢復正常將指日可待。

今年春節能吃上便宜豬肉嗎？

眾所周知，保障豬肉供應，不僅影響著百姓餐桌上的選擇，還事關物價的穩定和「三農」的健康發展：

所以，在意識到事態嚴重性後，國家早就重拳出擊應對了。

比如，中央儲備凍豬肉和牛羊肉，適時投放增加市場供應。

比如，加強非洲豬瘟疫情防控，儘快恢復生豬產能。

再比如，嚴厲打擊囤積居奇、串通漲價行為！

還有，出台補貼政策，鼓勵豬農擴大生豬養殖規模。

因為舉措既切中問題痛點，又順應市場規律，所以很快就收到了立竿見影的效果：進入11月份以來，持續瘋漲的豬肉價格開始降價，其出廠價已經連續三周環比下跌，而且跌幅逐漸擴大。

但養豬業是一個生產周期頗長的行業，養殖戶從補欄到增加豬肉供應，亦需耗費一年半載的時間。

也就是說，生豬的供應恢復可能還需要一段時間，價格的高位運行也可能會持續一段時間。

豬肉價格雖然在近期會出現一定的回落，但真正的拐點還遠遠沒有到來。

這也就意味著，今年春節吃上便宜豬肉還只能是一個奢望了。
