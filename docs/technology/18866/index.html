美團打車助力計程車行業升級，提供流量、技術、運營三大支撐

2019-12-16 230條評論 1,755次閱讀 805人點贊

網約車內參訊（文/彭小偉）12月13日，由中國交通報社主辦，美團打車承辦的出租汽車行業數字化升級專題研討會在北京舉行，來自政府部門、研究機構和企業的代表，分享了對出租汽車行業藉助數字化推動行業高質量發展的思路與建議。

行業迎來數字化升級窗口

當天，來自交通運輸部運輸服務司、國家發展改革委綜合運輸研究所、廣州市交通運輸局、瀋陽市交通運輸局、湖北省交通運輸廳道路運輸管理局、交通運輸部管理幹部學院、東南大學、美團打車等政府部門、研究機構和企業代表紛紛就出租汽車行業藉助數字化發展建言獻策。

交通運輸部運輸服務司城市交通管理處副調研員席錦池表示，交通運輸部運輸服務司根據黨中央、國務院決策部署和部黨組工作安排，高度重視並指導各地交通運輸部門持續推進深化出租汽車行業改革工作，包括：推動出租汽車運價改革，加快健全運價機制形成，建立運價動態調整機制；推廣各地好經驗好做法，提升行業監管能力和服務水平；關愛改善出租汽車司機生存環境，提升出租汽車行業形象和服務水平；促進新老業態融合發展，推進巡遊車「+互聯網」服務，促進運營效率和服務水平提升。

美團巡遊車業務負責人林小翀認為，巡遊計程車行業的網約化，主要是解決乘客、司機、車輛、企業和監管部門的線上化連接，為價格動態調整創造了前提，而這正是巡遊計程車市場化的關鍵。巡遊計程車的市場化，可以通過完善價格結構，提高巡遊計程車的運力效率，激發市場活力。此外，巡遊計程車的網約化和市場化也需要運營能力保障。

三大抓手驅動行業發展

會上，林小翀系統介紹了美團打車在巡遊計程車數字化升級中的實踐。據他介紹，美團打車以流量能力、技術能力、運營能力為三大抓手，與傳統企業深度合作做巡遊計程車數字化升級的「驅動器」，推動以數字化為基礎的網約化、市場化和監督管理信息化，助力巡遊計程車行業數字化升級，實現多方共贏。

自2017年2月美團自營網約車在南京上線以來，美團打車的網約車產品已經過了兩年的迭代，在技術及運營能力上有了深厚的積累。美團打車大型專業研發團隊，能夠提供完善的系統能力支持，同時美團年度交易用戶總數達4.4億，可作為城市巡遊計程車服務平台的啟動流量，並且可以第一時間接入平台，解決自建平台缺少流量的問題。

此外，林小翀還介紹了美團打車的三大行業解決方案，包括「網約SaaS方案」、「服務線上化」以及「信用體系建設」。其中，「網約SaaS方案」打通手機與4G智能車機，促進巡遊計程車的巡網融合。部分城市的實踐結果表明，車機潛力的發揮有助於提升巡遊車網約化水平。「服務線上化」 指的是通過掃碼實現線下訂單信息&服務線上化，其關鍵價值在於增強監管和管理的信息化，提升乘客體驗。「信用體系建設」則可以實現精準管控，約束失信司乘儘快修複信用，有效提升行業服務水平。

美團模式構建行業新生態

這次研討會上，美團交通政策研究院院長田翔表示，在助力巡遊計程車數字化升級方面，美團具備獨特的生態優勢。

田翔表示，美團「Food+Platform」 戰略，以吃為核心，構建從需求側到供給側的多層次科技服務平台。通過科技創新觸達用戶、服務用戶，為用戶打造到店、到家、旅行、出行全場景的消費服務體驗，用戶可以在平台上完成從信息獲取、服務預訂、交易支付、評價及分享的全過程。

在美團的「出行+消費」生態下，將實現多場景串聯，豐富城市居民的生活體驗；可以利用大數據分析等科技手段，提升司乘雙邊效率和服務安全；同時結合美團的能力優勢和合作夥伴深厚的行業積累，共同推動行業和諧發展。

網約車內參研究員盧布表示，網約車行業的監管只會越來越嚴格，越來越規範，已經告別了野蠻生長的時代。網約車平台未來的發展方向只有兩個，一是加大平台的合規化，司機有網約車駕駛員證，車輛有網約車運輸證，平台在當地有網約車經營許可證。二是積極融入到各城市計程車的轉型升級中，幫助當地計程車行業數字化升級。很顯然，美團打車已經開始布局，並且取得了不錯的成績。
