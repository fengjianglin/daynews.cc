機構報告：寵物、美妝消費品增速超過50%，四五線城市消費總額增速領跑

2019-12-19 1,317條評論 2,925次閱讀 320人點贊

每經記者：張鍾尹 每經編輯：陳旭

12月17日，京東數字科技集團在京發布《基於京東大數據的中國消費市場研究報告》（以下簡稱報告）。

報告將京東平台的商品分為必需消費品（家用電器、食品飲料、母嬰等7類）和可選消費品（手機通訊、旅遊出行等13類）兩大類。數據顯示，當前可選消費品增速和結構均弱於必需消費品，這表明經濟下行壓力依然較大。

根據報告的分析研究，2019前10月，大部分消費品類的增速均快於2018年，而食品飲料、本地生活/出行旅行、文娛消費品類增速慢於2018年。

具體來看，2019年，寵物生活、美妝護膚類消費品的增速均超過50%，家居日用、圖書、母嬰、醫藥保健、服飾內衣等品類的增速在20%~40%區間，增速均快於2018年。這些消費品類可作為未來一段時間促進消費的發力點。

數據來源：國家統計局

京東數字科技研究院院長、首席經濟學家沈建光向記者表示，提振消費應持續推進居民收入提升計劃和中等收入群體壯大計劃。多渠道增加居民財產性收入以改善初次分配，在再分配環節建立靈活多樣的稅收調節機制。

他同時表示，中國居民儲蓄率居世界前列，消費率相對不足，應通過完善社會保障、醫療等措施提升居民的邊際消費傾向。

此外，報告顯示，低線級城市的消費總額增速快於高線級城市，四五線城市的消費總額增速領跑其他線級城市。這也表明四五線城市這一下沉消費市場具有強大的消費潛力。

從整體消費走勢來看，京東平台中縣域消費總額一直以高於平台整體的速度增長，且今年以來兩者差距逐步擴大。不過與非縣域城市相比，縣域地區的消費結構仍有待改善，未來存在較大的消費升級潛力。
