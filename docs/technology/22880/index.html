遊戲行業2019年收入超過2300億元 1468款遊戲拿到版號

2019-12-20 225條評論 2,556次閱讀 638人點贊

每經記者：許戀戀 每經編輯：杜毅

曾經，版號凍結困擾著遊戲行業，隨著版號放開，遊戲公司業績復甦明顯。這一跡象，在年底的行業大會上得到了體現。

12月17日，在2019年度中國遊戲產業年會上，中宣部出版局副局長馮士新在致辭中透露，今年以來，國家新聞出版署已批准出版運營遊戲1468款，基本滿足了企業正常發展需求。

根據會上發布的2019年度中國遊戲產業報告，中國遊戲行業實際銷售收入達2308億元，同比增長7.7%。7.7%的增速相比去年的5.3%有所提升，在總量調控等多重政策影響下，中國遊戲產業正在經歷一個大浪淘沙的階段。

圖片來源：主辦方提供

僅1468款新游過關 但收入未縮水

根據每經記者一年來和遊戲行業人士的交流，今年遊戲版號下發數量減少是不爭的事實。在這場遊戲行業的年會上，官方公布的具體數據確認了這一點，1468款遊戲和往年相比，減少了好幾倍。

但這未必是一件壞事。從總量上看，雖然不到1500款，但是遊戲行業的收入並未縮水。增幅也比去年有所提升，對於經歷多年發展已經高度成熟的遊戲行業來說，7.7%的增速或許不夠振奮，但是依然給了行業以信心。

「一方面，很多老遊戲依然在持續貢獻營收，另一方面，政策管控讓市場上的換皮遊戲少了很多，能夠拿到版號上線的，幾乎可以說都是品質較好的遊戲。」一位現場的行業人士向每經記者表示。

這印證了此前遊戲從業者對精品時代的判斷，在這股風潮下，大一點的遊戲廠商優勢將會更大，遊戲產業或將進入一個「強者恆強」的時代，中小廠商除非拿出能夠留住用戶的創新，否則被淘汰已經是板上釘釘。

對於版號，馮士新建議，希望廣大遊戲企業精確定位方向、精心打磨產品、力戒浮躁風氣，用好每一個版號資源，做一款精一款。

從細分市場觀察，移動遊戲佔整體營銷收入近七成，處於主導地位。客戶端與網頁遊戲佔比分別降至26.6%和4.3%。2019年網頁遊戲營銷收入98.7億元，同比下降22.0%。與之相應，2019年用戶規模下降至1.9億，同比下降15.2%，下滑趨勢極為明顯。移動遊戲依然保持較好的增長速度，但是端游和頁游發展疲軟較為明顯。

值得注意的是，報告中提到，2019年中國自主研發遊戲海外市場營銷收入115.9億美元，折算人民幣約為825億元，同比增長21%。相當數量不同類型自主研發遊戲在全球上百個不同地區的下載榜和暢銷榜進入頭部。此外，女性遊戲、二次元遊戲也被提及，2019年女性用戶規模為3億，佔國內遊戲用戶總規模的46.2%，營銷收入526.8億，佔總收入的22.8%，基本保持穩定。2019年二次元遊戲營銷收入達215.6億元，同比增長12.9%。與之相應，用戶規模1.16億，同比增長10.7%，呈穩步上升趨勢。

遊戲行業發展依然有短板

2019年是中宣部承擔遊戲行業管理職能的第一個整年。審批遊戲、印發防止未成年人沉迷網路遊戲的通知、起草制訂新的網路遊戲管理辦法、開展道德評議等等措施，對行業影響巨大。

不過馮士新表示，在肯定成績的同時也要看到，我國遊戲行業的發展還存在不少問題和薄弱環節。比如，未成年人沉迷網路遊戲現象依然突出，社會各界對此仍十分關切；遊戲產品整體結構有待優化、內容質量有待提高，精品供給仍顯不足；一些企業片面追求經濟效益、忽視社會效益，商業味濃、文化味淡，甚至作品價值導向出現偏差；部分企業經營出現較大困難，心態浮躁、急功近利的現象比較突出等等。

馮士新提出了幾點建議，首先是著力保護未成年人身心健康。今年10月底國家新聞出版署印發了《關於防止未成年人沉迷網路遊戲的通知》，對未成年人使用網路遊戲的時段時長、消費金額等，提出了具體標準和明確要求。

他也透露，國家新聞出版署正在抓緊建設統一的實名驗證平台，為遊戲企業做好防沉迷工作提供必要條件。我們將指導各地出版行業主管部門，對防沉迷要求落實情況開展專項檢查。我們還將加快推動遊戲適齡提示工作，為不同年齡段玩家提供明確指引。

每經記者注意到，在這次大會上，行業老大騰訊表示，2019年，由於技術原因無法接入健康系統的32款遊戲已經強制退市或在退市流程中。騰訊公司高級副總裁馬曉軼在今年8月時表示，為了確保所有未成年用戶都可以受到保護，如果有遊戲因為技術等硬性原因無法接入健康系統的——即便依然可以貢獻營收——騰訊都會毫不猶豫地把它們停運和下架。

上述遊戲從業者則向每經記者分析，遊戲大廠去年至今在未成年人保護上確實進展較大，未來遊戲廠商在遊戲的文化價值上將會著力深挖，5G、人工智慧、雲計算、大數據、區塊鏈等新技術新應用和遊戲行業的結合也會更深，在政策導向已經很明確的當下，就看哪家廠商能夠提前佔據下一個風口。
