​軟體+車聯網龍頭標的（300496）：機構上調目標價！要引爆行情

2019-12-16 1,217條評論 2,572次閱讀 1025人點贊

中科創達(300496)

公司簡介

中科創達(300496)軟體股份有限公司主營業務為移動智能終端操作系統產品的研發、銷售及提供相關技術服務。公司提供的主要產品和服務主要包括軟體開發服務、聯合實驗室、軟體產品及軟硬體一體化解決方案。 　　

目前，公司擁有超過1000項自主研發的技術專利及軟體著作權，在全球範圍內擁有約3800名員工，其中研發人員佔比接近90%。

重磅事件

中科創達近日再度被券商關注，有分析師認為，中科創達以底層軟體作為核心競爭力，目前三大業務板塊發展邏輯清晰，長期成長性優異，結合公司歷史估值水平，當前估值仍有提升空間。

①車載業務：汽車行業智能化、聯網化、電動化變革重塑產業鏈，軟體價值得到較大增加，軟體廠商在車載產業鏈地位隨之提升。中科創達已經基於智OS+RighwareKanzi3D+智能視覺三大技術優勢構建完整智能駕駛艙生態平台，多年底層技術積澱以及市場先發優勢助力公司成為行業領跑者；

②IoT業務：5G商用驅動物聯網等諸多下游應用發展，諸多應用場景有望進入爆發期。中科創達2016年推出「核心板+操作系統+核心演算法」一體化的SoM(SystemonModule)產品，近年來開始向TurboX平台發展，從底層重塑行業生態；

③ADAS業務：車載視覺演算法在汽車ADAS產業鏈中地位重要性提升，中科創達2017年12月收購保加利亞視覺技術提供商MMSolutions切入ADAS領域，MMS在圖像處理演算法開發、優化和集成領域全球領先。

中科創達兩大業務處於高速發展階段，智能汽車滲透率持續提升，車載業務將保持高速增長態勢；5G商用化催化物聯網行業，IOT業務未來三年將較快增長。

券商分析師預計中科創達2019-2021年歸母公司凈利潤增速分別為43%、40%、45%，2019年EPS為0.58元/股，將公司目標價上調至56.7元/股，維持「買入」評級。
