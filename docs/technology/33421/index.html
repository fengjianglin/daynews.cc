數據中心行業為何再次渴望採用伺服器

2019-12-24 637條評論 1,982次閱讀 321人點贊

伺服器銷售量是一個很好的指標，可以反映不同規模和行業的企業對其特定業務的看法。從今年第三季度的伺服器銷售數量和市場總收入來看，儘管存在諸多不確定性，但伺服器廠商的感覺應該不錯。

通常情況下，越來越多伺服器會進入市場，並且會按照現行準則按五年時間表進行折舊。但對於使用壽命很長，並且工作負荷相對穩定的大型NUMA伺服器，甚至在超過其折舊日期的情況下仍在使用這些伺服器。大型伺服器的早期情況並非如此，當時即使性能提高並隨摩爾定律的速度更新換代，但計算能力的增長絕對值要小得多。在20年前，操作系統升級更為頻繁，可能是三年升級一次。而現在卻很少有人升級系統，通常是淘汰或更換，因為還有其他更緊迫的問題要處理，例如如何支持10億用戶。對於超大規模生產商和雲計算製造商而言，毫不奇怪的是，他們每年更換大約三分之一的伺服器產品，並增加新容量以支持不斷增長的工作負載。

有趣的是：根據市場研究機構IDC公司的調查，儘管伺服器市場收入在第三季度出現下滑，但銷售量和出貨量都是連續上升的，已經成為歷史上第二好的第三季度，2019年的銷售額為219.9億美元，機器出貨量為307萬台。2019年伺服器市場收入下降6.7%，而出貨量下降3%。市場收入下降的一半以上可能歸因於AMD公司在x86伺服器市場上與英特爾公司的競爭日益激烈，與此同時，存儲器和快閃記憶體的價格從過去兩年的高位向下回落。隨著AMD公司與Nvidia公司的競爭越來越激烈，GPU的價格也在下降，這也對高端伺服器市場產生了影響，並緩解了企業對於更多核心處理器的需求。在這裡要說的是，將2019年第三季度與2018年第三季度進行比較是一個非常艱難的比較，預計同樣的事情將在今年第四季度再次發生。當然，這取決於美國的谷歌、微軟、亞馬遜和Facebook和中國的阿里巴巴、百度、騰訊、京東這8家超級巨頭以及十幾家大型服務提供商的需求，這些服務提供商的運營規模很大，但並不是超大規模廠商。

重要的是要記住，世界上大量的大型企業和中型企業仍然是每個季度的伺服器出貨量的主要來源，毫無疑問，伺服器製造商從與這些企業的交易中獲得的利潤都在彌補將伺服器出售給超大規模生產商和雲計算供應商的虧損。

這是一項艱巨的任務，正如調研機構之前所說，希望那些生產伺服器的企業能有更多的利潤。但是，大部分利潤仍將流向英特爾公司，AMD公司和Arm公司在這方面要經過多年的努力才能獲得成功，這並不意味著利潤會從英特爾公司轉移到伺服器廠商。市場競爭很可能會減少很多利潤，而當這一切都結束時，很少有人從中受益。這正是小型計算機在20世紀80年代進入數據中心，並對大型機產生重大影響時所發生的事情;而在20世紀90年代，Unix伺服器與專有伺服器和大型機開展有效的競爭;2000年左右， Linux伺服器和x86伺服器團隊更多地進入數據中心。IDC公司可能為此追蹤這些年的伺服器利潤，但是結果可能會讓每個人都感到意外。

好消息是，伺服器製造商似乎喜歡為獲得更多的銷售機會而生產伺服器，這是有利可圖的。調研公司表示，對於那些利潤微薄的公司來說這是很好的一件事，然而，絕大多數利潤都流向一兩家供應商這說明市場發展並不健康。

市場收入和出貨量是很好的統計數據，但是需要更好地衡量消耗多少計算容量以及價格/性能計費。由於x86伺服器現在幾乎主導數據中心的應用，因此通過x86伺服器的收入和出貨量計算出平均售價。為了說明過去10年，伺服器市場發生了多大變化，查看下圖可以了解一下經濟大蕭條以來x86伺服器的收入與非x86伺服器的市場收入之間的差距。

在2009年第一季度，x86伺服器的全球市場收入為50億美元，而非x86伺服器的全球市場收入也是50億美元。如今，全球伺服器市場的收入和出貨量增長了2.2倍，但非x86伺服器市場收入卻在今年在第三季度降至14億美元(下降了70%)，與去年同期相比減少了13.1%。與2018年第三季度相比，IBM公司的所有RISC/Unix供應商業績表現良好。

通過了解x86伺服器的發貨量和x86伺服器的總收入，因此可以計算出平均售價(見下圖)：

可以看出，儘管沒有達到用戶的預期，但伺服器的平均價格確實已經下降。在今年第一季度，伺服器達到7,236美元的峰值價格，這是自RISC/Unix機器在上世紀90年代後期佔據主導地位以來，從未達到這麼高的價格。直到2017年初，各種伺服器的平均銷售價格都在5,000美元至6,000美元之間，當時RISC/Unix和大型機商店大多擁有性能相當強大的伺服器，並且隨著越來越多的中小型企業開始採用雲計算基礎設施，其價格也開始上漲。在許多情況下，對於只有很小一部分容量的軟體服務開始採用雲計算基礎設施。有趣的是，隨著時間的推移，已安裝的伺服器的總利用率發生了顯著變化，平均利用率可能翻了一番或增長三倍，達到CPU容量的25%到30%，這是因為用戶通過虛擬化、容器和其他技術，在驅動核心處理器方面做得更好。伺服器收入不高並且供應商利潤不高的原因之一是，很多用戶在使用和購買伺服器方面有著豐富的經驗，充分挖掘伺服器的潛力。

在任何情況下，全球企業的IT預算都不可能在一年內購買1800億美元的伺服器。用戶會發現無需大量計算就能獲得成功的方法。儘管展望摩爾定律的未來發展，但CPU最終將變得越來越昂貴，而不是變得越來越便宜，可以看到伺服器市場收入隨著核心處理器成本的上升而增加。這種事情已經發生在內存和快閃記憶體產品上，那麼為什麼不能發生在CPU?

這是一個真正有趣的趨勢：每個季度銷售的伺服器的總計算能力以及該容量的相對成本(見下圖)：

這種情況反映了過去十年來主導x86伺服器市場的現實。這個圖表列出了已售出的伺服器數量，每台伺服器的核心處理器平均數量，以及由於每一代伺服器的架構更改而導致的性能改進，並得出了計算容量相對性能指標的總和。在這個圖表中，在過去十年中，也就是從2009年第一季度到2019年第三季度，計算總量增長了28倍，但是單位CPU的平均成本減少了12倍。這是因為半導體技術取得了驚人的進步，在這種情況下，主要原因是摩爾定律和CPU架構的改進。

在這個圖表中有幾件事值得注意。首先，就研究機構的估計而言，這是已交付的總計算容量創紀錄的季度，這沒有反映在IDC公司公布的伺服器市場收入或發貨數量中。實際上，由於時鐘指令的適度增加，以及每台伺服器的核心處理器數量大幅增加，因此已交付的總計算能力提高了24%。其次，伺服器單位計算的平均成本同比下降了約25%，2019年第三季度與2018年第三季度持平，這是一個巨大的進步。每台伺服器的成本並不能說明這一點。這些只是估算值，希望IDC公司能夠提供更準確的數字，並指明市場發展方向。
