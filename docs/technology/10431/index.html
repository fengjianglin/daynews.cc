上市首日破發！這家河南企業成科創板首例，專家稱註冊制味道越來越濃

2019-12-05 197條評論 1,183次閱讀 186人點贊

華夏時報（chinatimes.net.cn）記者賈謹嫣 陳鋒 北京報道

12月4日，建龍微納（688357.SH）開盤不久後即破發，成為首例上市當天破發的科創板企業。

建龍微納發行價格為43.28元/股，今日開盤價報44.00元/股，開盤不久股價即下跌，9點43分左右報43.26元/股，跌破發行價，此後股價一直保持在發行價下方，上市首日股價收於42.35元/股，下跌2.15%。

《華夏時報》記者致電建龍微納董秘辦，工作人員在聽到記者詢問破發一事後笑出聲，工作人員稱，公司將持續做好主營業務，發行價是否定高一事不便回應，公司認為股價破發一事和市場整體行情相關。

放牛塘聯合創始人、資深投行人士周興用在接受《華夏時報》記者採訪時表示，上市首日破發，在註冊製成熟的證券市場是司空見慣的事情，甚至很多時候首日破發的占多數。企業和承銷商根據投資者的認購情況，確定對自己有利的發行價，這並沒有錯誤，只能說明投資者自己不理智，盲目申購。企業的職責在於合法披露信息和充分提示風險，以讓投資者做出合理判斷。上市首日破發的出現，讓科創板的註冊制味道越來越濃。

作為河南省首家完成科創板上市輔導的企業，建龍微納在IPO初期就受到市場諸多關注。值得注意的是，就在2018年11月1日，科創板正式宣布設立的前4天，全國股轉公司發布了建龍微納在全國中小企業股份轉讓系統終止掛牌的公告。

公開資料顯示，建龍微納主要從事無機非金屬多孔晶體材料分子篩吸附劑相關產品研發、生產、 銷售及技術服務的業務，主要產品為分子篩原粉、分子篩活化粉和成型分子篩三大類。

事實上，建龍微納屬於無機化工領域，目前整個化工行業動態市盈率不高。私募人士杜坤維在接受《華夏時報》記者採訪時表示，化工行業動態市盈率約為22倍，建龍微納的市盈率是27倍，估值適度偏高，僅僅按照估值水平，新股有一定溢價，不應破發，關鍵在於整個科創板估值水平下移，很多個股跌幅巨大，市場已經沒有多少炒作新股的勇氣。

杜坤維認為，科創板實行投資者適格性制度，科創板相對於主板應該有一個抑價而不是溢價，可是從IPO價格來看，偏偏出現了明顯的溢價，這給炒新帶來潛在的機會。除此之外，建龍微納發行絕對價格偏高，也不適合炒作，國內投資者還是偏好中低價股，這也是今天盤中破發的一個因素，從行業來看，建龍微納雖然業績增長較好，但缺少高科技想像空間。

上市之際，建龍微納的財務數據曾被市場關注。最為矚目的是，2013年-2017年公司凈利潤從未過千萬，2018年「飛躍」至4774萬元，同比增長423%，而同期收入增幅只有55%。

財務數據顯示，2013年-2016年，建龍微納營業收入分別為1.02億元、1.26億元、1.32億元和1.30億元，扣非凈利潤分別為-436萬元、753萬元、636萬元和669萬元。

2017年、2018年，建龍微納營業收入2.44億元、3.78億元，同比大幅增長88%和55%；歸屬於母公司所有者凈利潤分別為-1018.82萬元、4707.24萬元。

針對2017財年公司歸屬母公司股東的凈利潤為-1018.82萬元，建龍微納表示，主要是因為主合同債務人海龍精鑄逾期未償還銀行貸款，公司作為保證人代主合同海龍精鑄向債權人累計支付了債務3262萬元。

天眼查數據顯示，建龍微納共有12條動產抵押信息，涉及金額2.9億元。《華夏時報》記者梳理後發現，自2015年5月開始，建龍微納共將數千台機器設備進行抵押借款。

值得注意的是，記者在梳理中發現，在12條動產抵押信息中，有3條均涉及1172台機器設備的抵押信息，抵押權人為洛陽銀行股份有限公司偃師支行，抵押時間上具有連續性，第一次抵押時間為2015年9月至2016年9月，第二次抵押時間為2016年9月至2017年6月，第三次抵押時間為2017年6月至2019年7月。

這或許意味著，建龍微納或因資金周轉問題出現連續循環借貸，不得已將機器設備常年抵押籌資。

除此之外，記者查詢裁判文書網後發現，涉及建龍微納的裁判文書共有25條，多為買賣合同糾紛。遞交科創板IPO申請後，新增6條裁判文書。
