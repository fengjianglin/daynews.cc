阿里最新架構調整：張建鋒卸任CTO 蔣凡兼管阿里媽媽

天天要聞
2019-12-19

12月19日，阿里巴巴宣布新一輪面向未來的升級，集中發力推進全球化、內需、大數據和雲計算三大戰略。

當天上午，阿里巴巴集團董事局主席兼CEO張勇、螞蟻金服集團董事長井賢棟分別發出全員信。張勇在信中表示，阿里巴巴歷來的習慣就是在最好的時刻為未來變陣。


他說，全球化、內需、大數據和雲計算是阿里數字經濟體未來三大戰略；在實施戰略、兌現使命的過程中，業務戰略、文化戰略和組織戰略三位一體，缺一不可。「這對我們提出的基本要求就是，在我們的數字經濟體里，通業務、通文化、通技術、通人才、通組織保障。」

在此背景下，阿里系一系列組織和人事的調整也隨之展開。

為把握新消費時代的變革，淘寶、天貓將和阿里媽媽合力推進用戶產品和商業產品統一策略下的創新；同時，阿里媽媽的廣告業務中台將持續優化升級，服務於阿里巴巴集團其他業務場景。蔣凡在淘寶天貓總裁的職責基礎上，將代表集團分管阿里媽媽事業群。

阿里巴巴集團B2B事業群總裁戴珊在負責原業務的基礎上，將代表集團分管盒馬事業群，全面負責打通盒馬、村淘、智慧農業等業務。

據介紹，目前，阿里已實現核心系統100%上雲，同時在人工智慧、大數據、區塊鏈等技術上取得突破性進展。張建鋒將繼續擔任阿里巴巴技術委員會主席、達摩院院長、阿里雲智能事業群總裁，領導阿里巴巴未來的技術總戰略、達摩院的建設，並致力於阿里雲智能業務的進一步突破。螞蟻金服原CTO程立將擔任阿里巴巴集團CTO，兼任阿里巴巴技術委員會副主席，負責阿里數字經濟體內各業務的全面技術打通。螞蟻金服原副CTO胡喜出任螞蟻金服CTO。

螞蟻金服集團董事長井賢棟在全員信中宣布，全球化市場的未來超乎想像，他需要在全球化中投入更多精力。井賢棟將繼續管理螞蟻國際事業群、智能科技事業群、HR、財務和戰略投資板塊，不再擔任螞蟻金服CEO，螞蟻金服總裁胡曉明將出任螞蟻金服CEO。趙穎將擔任螞蟻金服國際事業群總裁，全面協助井賢棟做好全球化，同時繼續兼任飛豬總裁。

「在這一張關乎阿里未來二十年基本走向的戰略布局中，我們必須體現阿里人的管理創新和擔當。一個籬笆三個樁，一個好漢三個幫。現在，我們的每個戰略板塊，可以清晰地看到多個業務事業群攜手作戰的場景。我們不是一個人在戰鬥，我們是一群人為同一個目標在戰鬥。十指相扣，指指連心！」張勇在全員信中寫到。

各位阿里人：

最近集團喜事不斷，讓人振奮。雙11再創紀錄，阿里巴巴香港成功上市，王堅博士成為我們國家第一位來自民營企業的院士，雙12又極大地激發了小商家的活力。

我相信很多同事都在期待集團組織架構的調整。因為阿里巴巴歷來的習慣就是在最好的時刻，為未來變陣。是的，我們一直在為此精心規劃，有些調整已經在進行並取得意料之中的亮點，但這遠遠不夠。

我們在阿里成立20周年之際，重申了阿里數字經濟體未來三大戰略：全球化、內需、大數據和雲計算。業務戰略、文化戰略和組織戰略三位一體，缺一不可。這對我們提出的基本要求就是，在我們數字經濟體里，通業務、通文化、通技術、通人才、通組織保障。確保全阿里數字經濟體能「一張圖、一顆心、一場仗」，實現更加完美的戰略一體化。

基於此，集團決定：

1. 張建鋒（行癲）將卸任阿里集團CTO工作，繼續擔任阿里巴巴技術委員會主席、達摩院院長、阿里雲智能事業群總裁，領導阿里巴巴未來的技術總戰略，達摩院的建設，以及致力於阿里雲智能業務的進一步突破。程立（魯肅）將擔任阿里集團CTO，向我彙報。阿里集團新零售技術事業群總裁吳澤明（范禹）向程立彙報，阿里搜索及廣告技術事業部負責人周靖人（靖人）向程立和淘寶天貓總裁蔣凡雙實線彙報。程立並將兼任阿里巴巴技術委員會副主席，負責阿里數字經濟體內各業務的全面技術打通。

2. 蔣凡在現有淘寶天貓總裁的職責基礎上，將代表集團分管阿里媽媽事業群，總裁張憶芬（趙敏）向蔣凡彙報。阿里媽媽和淘寶、天貓將合力推進用戶產品和商業產品統一策略下的創新，並且更堅定地建設好阿里媽媽廣告業務中台，為集團其他業務場景服務。

3. 集團B2B事業群總裁戴珊（蘇荃）在負責ICBU、1688、村淘、零售通、速賣通業務基礎上，將代表集團分管盒馬事業群，全面負責打通盒馬、村淘、智慧農業等業務。盒馬總裁侯毅（老菜）向戴珊彙報。

4. 趙穎（芷雪）擔任螞蟻金服國際事業群總裁的同時，繼續兼任飛豬總裁。

以上決定即日起生效。

在這一張關乎阿里未來二十年基本走向的戰略布局中，我們必須體現阿里人的管理創新和擔當。一個籬笆三個樁，一個好漢三個幫。現在，我們的每個戰略板塊，可以清晰地看到多個業務事業群攜手作戰的場景。我們不是一個人在戰鬥，我們是一群人為同一個目標在戰鬥。十指相扣，指指連心！

因為信任，所以簡單；因為相信，所以看見！阿里人，讓我們一起加油！

張勇（逍遙子）

阿里巴巴集團董事局主席兼首席執行官
