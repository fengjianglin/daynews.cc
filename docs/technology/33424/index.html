晶元之後，中國又一領域實現重大突破！國內價格不到進口的1/3？

2019-12-24 455條評論 2,290次閱讀 1024人點贊

本文為「金十數據」原創文章，未經許可，禁止轉載，違者必究。

近代以來，創新和科技水平對於促進各國經濟發展的作用越來越強。而在過去很長一段時間裡，各個領域的頂尖科技幾乎都掌握在主要發達國家手中。不過，近些年來，隨著中國製造業進入轉型階段，中國在越來越多領域開始出現突破。在近期中國晶元出現重大突破之後，最新報道稱，中國又一領域出現了零突破，並且還創造了國際標準。

中國晶元取得重大突破，有望打破美日韓壟斷

2018年初，中國開始意識到自身在晶元製造領域的空白，而這種空白對於很多高科技產品的競爭來說都是不太樂觀的現象，自此中國開始加大投資，推動國產晶元的研發和生產。而在經過一年多的努力之後，今年9月份中企終於取得重大突破。

據外媒此前報道，中國半導體企業紫光集團旗下的長江存儲已經開始對自主研發的Xtacking架構的64層三維快閃記憶體（3D NAND）進行量產。分析認為，此舉有望令中國的存儲晶元的自給率從8%提升至40%，這對降低晶元對外依存度有著重要的意義。

要知道，在中國加入晶元製造的競爭之前，全球晶元製造長期以來都被美日韓三國的企業壟斷。而在中國首款64層3D NAND實現量產之後，有望令中國在該領域與全球頂尖企業的技術差距縮小1到2年，是打破美日韓壟斷的關鍵。

中國又一領域出現零突破，還創造了國際標準

值得一提的是，在晶元領域取得重大突破之後，上周五（12月20日）中國又一重要領域迎來歷史性時刻，不只出現零突破，而且還創造了國際標準。

據報道，12月20日中國石葯集團高血壓專利葯玄寧已經獲得美國食品樣品監督管理局審評通過，這成為中國本土企業第一個獲得美國完全批准的創新葯。而由於玄寧是美國食品藥品監督管理局批准的首個左旋氨氯地平，中國研發的降血壓新葯還成為了國際標準。

這無疑是中國製藥出海的一個里程碑，代表了中國製藥的新高度。事實上，中國製藥和其他眾多製造業領域一樣，為了實現突破，在過去幾十年的研發過程中都歷經艱辛。然而，對於中國葯企來說，在美國申請新葯上市的非常少，而能在美國批准上市，就意味著該藥品拿到了在全球上市銷售的許可證，因此這具有非常重要的意義。

另外，藥品作為一種特殊的商品，此次成為國際標準不只意味著”中國創新”又攻克一個難關，而且對於中國減少抗高血壓藥物的對外依賴性，從而保障國民健康來說也有著重要意義。

有統計數據顯示，目前全球約有10億成年人為高血壓患者，而中國2018年的研究報告顯示，我國有高達2.45億人患有高血壓，幾乎佔全球高血壓患者總數的四分之一。然而，目前國際抗高血壓藥物的製造仍被外企壟斷，外國產品長期佔領中國最大的市場，高額的葯價也成為患者的另一大負擔。

而玄寧此次打破外國產品的壟斷之後，在國內的單片價格還不到進口產品的三分之一，這不僅可以降低國內患者用藥的經濟負擔，而且還可能從外企手中搶回中國市場，加大國內市場的競爭，從而有望拉低高血壓藥品的整體價格。
