最全閱讀類App測評報告，移動閱讀市場細分化趨勢明顯

2019-12-14 1,835條評論 3,881次閱讀 279人點贊

文/李富陽

本文超多乾貨，預計4分鐘閱讀完畢

商務君按：現在市面上各類閱讀App讓人眼花繚亂，幫助大家避免選擇恐懼症，商務君盤點了比較主流的23款App，分析了一下他們各自的特點，找到合適的閱讀App。如果以下沒有你中意的那款，或者盤點的App中有些寶藏功能沒被商務君發現，歡迎你告訴商務君喲~


曾經，人們一度以為電子書會取代紙質書的閱讀，但近幾年，隨著電子書市場發展的逐漸穩定，越來越多的出版界人士認為電子書與紙質書不是替代關係，在內容為王的前提下，二者未來將相互靠攏融合，並各自深耕細分市場。

目前，電子書市場經歷過早期的激烈競爭，如今的發展趨於平穩，各種類型的閱讀APP發展得都較為成熟，各種類型的閱讀APP都有自身的垂直受眾，競爭的範圍也變的更為細分。

以下商務君盤點了目前市面上的23款閱讀類APP，並對他們進行了分類，從幾個維度對它們進行了評價，不知道你更會中意哪款？

綜合類閱讀APP：藏書豐富、各有側重點

所謂的綜合類閱讀APP指的是既包括傳統已經出版的作品，也包括網路文學作品的閱讀器。目前市面上比較知名的綜合類閱讀app主要有掌閱、QQ閱讀、多看閱讀、愛奇藝閱讀、咪咕閱讀、天翼閱讀等。就藏書量來說，這些綜合性閱讀app的圖書儲備量都不小，但是因為各自依託的平台不同，所以側重點也有所不一。掌閱起步較早，因此無論是網路文學還是已出版的作品發展得比較均衡，QQ閱讀因為背靠閱文集團，所以在網路文學儲備這塊有豐富的資源。愛奇藝比較側重網路文學，目前正在大力發展自己平台的作者，簽約了一批網路文學知名作家，比如唐家三少等，主要致力於生產自己的IP,雖然愛奇藝並沒有QQ閱讀背後閱文集團豐富的資源，不過從發力IP運營的角度來說，兩者有一定的相似之處。咪咕閱讀因為背靠中國移動，目前推出了5G的聽書體驗，不僅可以提供普通話版本，也有四川話、廣東話之類的方言版本，比較適合想要嘗試新技術體驗的讀者。

就使用體驗而言，這幾款綜合性閱讀APP大多都比較流暢，但有些比如天翼閱讀，在搜索圖書時不會特別流暢。不過相對來說，天翼閱讀的界面設計也比較簡單直觀，其他的大部分綜合性閱讀App在界面設計上都有些複雜。

在優惠力度方面，各家都有自己的優惠政策，比較相似，例如都會推出一些小活動比如積分打卡的小活動，在月卡、年卡之類上面都會有一些優惠，但具體來說並沒有特別大的優惠政策。大抵是因為，在經過多年耕耘之後，這些綜合型閱讀平台的用戶群體都已經比較飽和，不需要再靠大幅度的優惠引流。

目前各大綜合型閱讀器已經轉戰精品、新品競爭路線，能否給用戶帶來真正切入核心需求的服務，是各大綜合類閱讀器目前競爭的主要方向。目前各大綜合類閱讀APP也在不斷拓展新的業務板塊，如漫畫、同人等，並且鼓勵讀者進行創作。此外，掌閱、Q閱讀、以及愛奇藝文學等都設置了書圈的功能，社交化已經成為綜合類閱讀app的一大趨勢。

出版類閱讀APP :新入局者優惠多

所謂的出版類閱讀APP，就是專註已經出版作品的閱讀類APP。大眾比較熟悉的圖書三大電商噹噹、京東亞馬遜都有各自的讀書類APP。在這些app上，讀者可以看到目前市面上絕大多數已經出版作品的電子版，資源非常豐富，但大多都要付費購買。就這三款傳統電商開發的閱讀類APP而言，kindle的使用體驗比較卡頓，界面的感覺也比較死板，用戶體驗有待提升。京東讀書和噹噹雲閱讀的使用體驗相對流暢一些，不過就界面設計而言，京東的界面設計相對噹噹更加清晰明了一些，使用起來也相對流暢。因為背後是電商平台，所以讀者們在這幾個平台上閱讀電子書時，若是想要購買紙質版本，除了亞馬遜（紙質書業務已經退出國內），在京東讀書和噹噹雲閱讀就可以直接購買。

除了傳統的電商平台，近幾年，網易蝸牛讀書和微信讀書的加入，給出版類閱讀APP的市場帶來一些新的氣息。在傳統的電商平台閱讀app發展趨於平穩時，這兩款產品的入局，便用大量的優惠政策吸引了眾多讀者的眼光。微信讀書是經常推出30天讀優惠政策，分享卡券免費讀書等優惠，網易蝸牛則是推出了每天免費閱讀1小時政策，讀者按照閱讀時長付費，而不是電子書本身的定價。這些大力的優惠舉措對於許多讀者而言都有著不小的吸引力。

由於背靠大型互聯網集團，網易蝸牛讀書和微信讀書的使用體驗都相對比較流暢。此外，由於有著微信這一強大的社交背景，因此，微信讀書相對網易蝸牛讀書而言具備了強大的社交特色：讀者既可以以書見人：用戶可以通過好友書架內容間接了解好友閱讀偏好及行為喜好；同時，亦可以通過排名點贊、召回讀書、買一贈一、贈一得一等方式加強與友人的社會聯繫；還可以以書會友：用戶可以通過書摘分享、想法交流等方式與好友探討讀書心得；同時，不定期的話題討論也有助於形成書友社群。由於微信還有微信讀書的小程序，因此，對於想要使用微信讀書又不想下載APP的讀者而言，通過小程序入口進入微信讀書也是一個非常方便的選擇。

網易蝸牛讀書的監製是韓寒，整體風格比較文藝，就圖書的分類來說，相對微信讀書比較成體系一些。網易蝸牛讀書的分類比較細緻，為讀者提供了各大讀書榜單，分出了新書榜，在讀榜，豆瓣8分以上榜單，冷門榜單，以及按照字數的分類專門划出了小說、名著、青春等區域。總體上來說，為讀者找書提供了較為細緻的嚮導。而微信讀書的分類則相對來說較為簡單，就書城來說，只分為小說、聽書、漫畫、公眾號四類，比較簡單。總體上來說，相較微信讀書，網易蝸牛讀書更加文藝范一些。

知識服務類產品：得到領跑，深受自媒體用戶群特性影響

隨著近幾年知識服務的興起，興起了一批解構知識，主打聽書課程的APP，如得到、新世相讀書會、十點讀書、有書等。這些APP中，如得到、新世相讀書會和十點讀書都有比較強大的自媒體背景，許多用戶都是從從微信公眾號轉化而來。其中，得到是這些知識服務類產品中領頭羊般的存在，無論是知識服務課程的類型、講師的豐富程度，都是最豐富的。除了各類知識服務聽書課程，得到今年還推出了電子書服務，使得用戶在聽完解讀之後，可以在平台上立即讀到相應的書籍。

就圖書種類來說，職場、心理以及商業是目前此類知識服務類產品中比較熱門的種類。除此之外，因為新世相讀書會和十點讀書的受眾比較偏向文藝向，除了熱門的幾種非虛構圖書，文藝類圖書的解讀在兩者的平台上也比較豐富。就受眾群體來說，新世相和十點讀書比較相似，以女性居多，不過相較新世相，十點讀書的女性定位相對明顯一些，十點讀書有專門的變美變瘦、情感家庭等女性向氣質明顯的板塊，而新世相讀書會整體的板塊呈現都相對比較中性。

相較以上三者，有書主打共讀模式，在幫助有強烈閱讀願望但又不知道讀什麼書、不能堅持閱讀、閱讀效率低的書友開始高效閱讀。如果說，前面幾種知識服務產品比較適合自主學習能力比較強的讀者，有書共讀則是通過書圈的形式，讓書友之間彼此激勵、彼此監督，加強讀書這一行為的社交屬性。有書的平台上既有知識服務項目，也有電子書，比較偏向於非虛構。分類方面，有書的分類比較清晰明了，直接根據圖書的種類比如職場、心理、財經、文學等進行分類，讀者可以根據自己感興趣的種類進行選擇。

創作型平台：優質作者成關鍵，平台調性不一

對於創作型的閱讀平台而言，優質作者的數量很大程度上決定了平台用戶額粘性。對於豆瓣閱讀、簡書、ONE一個這樣的平台而言，平台上的內容質量都是依靠現有作者創作而成。因此，平台對於優質內容和作者的扶持非常重要。

豆瓣閱讀因為背後有著豆瓣這樣一個文藝社交平台，因此，吸引大量作者聚集。近幾年，從豆瓣閱讀上走出了出版界許多的優質作者，比如王占黑、遠子等。豆瓣閱讀較多長篇作品，很多都為寫作試驗，頗具創新性，比較偏向嚴肅文學路線。適合偏愛小眾文學，以及適用於挖掘文學新人。對於有志於創作的年輕讀者也比較適合，因為在豆瓣閱讀上經常會舉行徵文比賽，而且豆瓣讀書也是各家出版機構關注的平台，許多優秀的新人作品都有機會通過該平台出版。

相較豆瓣閱讀，簡書整體作品的篇幅較為短小，而且就風格而言，情感類作品作品佔比較大。簡書上也有進行長篇連載的作家，不過比較偏向網路文學路線，有比較多傳統網路文學類型的作品比如言情、玄幻等，不過偶爾也有一些比較嚴肅風格的作品，比如歷史類小說。簡書上曾經培育了許多目前微信公眾號的情感類KOL，此外也有一些作者被出版社挖掘，出版了作品。目前簡書也在進行自身的出版計劃。總體上而言，簡書上的作品比較適合輕閱讀。想要試水文學創作的新人也可以嘗試一下簡書，因為簡書上經常會推出一些寫作教程，比較適合寫作入門者。

One一個是韓寒創辦的APP,聚合高質量的文章、圖片、音樂、影視等內容，推送給用戶,主打文藝、精緻的風格,APP每天推送給用戶首頁的內容不多，主題大部分是生活、文學相關的類型。內容篇幅不大，適合當今時代快速節奏下人們碎片化閱讀的習慣。「ONE·一個」團隊會對短篇文章進行整理，並以文集的形式進行出版。截至目前，已經出版了較多圖書，如《很高興見到你》、《去你家好嗎》、《不散的宴席》等作品。讀者在上面既可以創作，也可以對喜歡的作品進行打賞，還可以記錄自己的生活。比較感性，熱愛生活和文藝，文化水平較高的年輕人。

網路文學平台：適合各自垂直用戶群體，app使用體驗差距較大

目前市面上比較知名的網路文學平台如晉江、紅袖添香、起點、塔讀、書旗小說等都有各自獨立的app。這些app依靠平台優秀作者培養起來的粘性讀者，各自擁有自身的垂直用戶。晉江、紅袖添香主要針對女性讀者，起點比較受男性讀者的歡迎，塔讀則比較偏向年輕Z世代（95後）群體，設置了一系列同人、幻想等受年輕人歡迎的欄目。而書旗小說則將小鎮青年作為主要用戶群體，以年輕化、大眾化為主要標籤。

但這些app的使用體驗相差較大。就流暢性和界面設計而言，晉江小說app相對其它幾款較為弱勢，使用時偶爾會出現卡頓。不過憑藉多年優質作者資源的積累，晉江小說閱讀app的用戶粘性在這幾款app中應當是領先的。由於晉江小說並沒有被其他大型網路文學平台收購，在許多優質作品上，具有獨家優勢。

起點讀書和紅袖讀書雖然在作者資源和使用體驗上都比較佔據優勢，不過由於這兩款都是閱文集團旗下產品，在QQ閱讀上也可以讀到這兩個平台的優質作品。從一定程度上來說，起點和紅袖讀書並不具備獨家優勢。

相較以上幾款產品，塔讀和書旗小說在優質作者資源上缺乏一定優勢，因此二者另闢蹊徑，培養自身具有針對性的讀者群。從去年開始，塔讀文學針對年輕人受眾這以群體特點，通過與李沁、王祖藍、黃子韜、蔡依林等明星藝人進行合作，結合節日熱點從而為平台進行宣傳傳播，與用戶形成共識；並在客戶端內發起明星話題討論，增強平台社區屬性。而書旗小說基於阿里大數據，在個性化推薦上進行了更深層的嘗試。提升作品分發效率，讓書和人之間形成匹配。此外，作為阿里文學旗下的主要閱讀APP，書旗小說的另外一個優勢則是來自阿里大文娛生態所帶來的內容紅利。已經在優酷熱播的《大軍師司馬懿》《白夜追兇》《烈火如歌》《北京女子圖鑑》《上海女子圖鑑》等劇集改編而成的小說也在書旗小說上有獨家呈現。
