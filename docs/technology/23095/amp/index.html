你的文娛生活已被統治，佔據音樂版權80%

天天要聞
2019-12-18

騰訊音樂娛樂集團（以下簡稱「騰訊音樂」、「TME」）發布了2019年第二季度及截至2019年6月30日的六個月的未經審計財務報告。2019年第二季度的總營業收入同比增長31.0%，至人民幣59億元（8.59億美元）。其中，在線音樂服務收入由去年同期的人民幣13億元增長20.2%，至人民幣15.6億元（2.28億美元）；社交娛樂服務收入由去年同期的人民幣32億元增長35.3%，至人民幣43.4億元（6.32億美元）。

在2017年1月，QQ音樂和中國音樂集團整合成立了新音樂集團，並更名為騰訊音樂娛樂集團，同時將QQ音樂、酷狗音樂和酷我音樂進行整合，合併後的騰訊音樂集團擁有環球、華納、索尼、華誼兄弟、韓國YG娛樂等在中國的獨家版權和數字分銷權。國內音樂市場幾乎全部熱門歌曲的獨家音樂版權被其包攬，成為在線音樂平台中實力最強的一方。在QQ音樂和其他的多個移動應用中，共有7億月活躍用戶。根據2017年中國數字音樂平台排行榜的榜單顯示，排在TOP5的分別是，QQ音樂、酷狗音樂、酷我音樂、網易雲音樂和蝦米音樂。據此測算，擁有QQ音樂、酷狗音樂和酷我音樂的騰訊音樂已經吞掉了中國70%的數字音樂市場，在版權方面也更是處於統治地位。

據了解，騰訊音樂的業務分六大板塊，其中之一是版權管理部，和QQ音樂、酷狗音樂和酷我音樂平齊。而根據中商產業研究院整理的數據，騰訊音樂集團手中握有的版權包括：環球、華納、索尼、英皇、YG、CJ、華誼、福茂、傑威爾

據了解，和網易雲、蝦米音樂靠版權轉授的運營方式不同，騰訊音樂在對待版權合作方面的態度更加簡單粗暴，就是買買買。這也是讓騰訊在把版權緊握在手之後，在中國數字音樂市場上能夠傲視群雄的最大原因。

其實在網易雲發展初期，騰訊曾積極尋求合作，對抗當時的競爭對手蝦米音樂。而隨著網易雲音樂的壯大，騰訊遂而停止和網易合作，轉而向阿里音樂完成版權轉授合作，這種扶一個打壓另一個的套路，騰訊可以說是玩的信手拈來。到了今年年初，騰訊在國家版權局的推動下，又分別於網易和阿里簽訂了版權轉授協議，至此可以發現，中國的數字音樂市場都在騰訊的一手操控之下

隨著互聯網的普及，音樂作品的消費渠道越來越集中於數字音樂平台，根據IFPI國際唱片業協會發布《全球音樂報告》，中國音樂市場中數字音樂佔比高達96%，如果音樂版權被一家壟斷，用戶聽音樂將別無選擇。缺乏競爭的市場，用戶體驗也無從保障。單一的渠道也有可能導致盜版現象抬頭，音樂市場再次回歸無秩序的狀態。

再者，音樂有被二次開發的價值，通過翻唱，原本不為人所知的歌曲也有可能廣為流傳。而獨家版許可權制了音樂愛好者對音樂的二次創作，同樣不利於音樂市場繁榮。儘管騰訊音樂曲庫在2000萬首，百度音樂也宣稱有1800萬首，網易雲音樂的曲庫在1000萬首，但是播放量高的作品只有四五萬首，而這些歌曲大多來自環球、華納、索尼全球三大唱片公司，它們的版權都已被騰訊音樂包攬。

《中華人民共和國反壟斷法》第三條規定，經營者濫用市場支配地位的行為屬於壟斷行為。

《反壟斷法》還認為有下列情形之一的，可以推定經營者具有市場支配地位:

一個經營者在相關市場的市場份額達到二分之一的;

(二)兩個經營者在相關市場的市場份額合計達到三分之二的;

(三)三個經營者在相關市場的市場份額合計達到四分之三的。

騰訊音樂旗下有三大音樂平台，它們在今年上半年的月活總量在4億左右，而在線音樂平台的月活總量為5億上下，騰訊音樂佔據了80%的市場份額。符合上述情形的第一條和第三條，因而在音樂市場佔據支配地位。在網易雲音樂發展正盛，對其有所威脅之時，騰訊音樂與網易雲音樂發生版權糾紛，也有濫用市場支配地位的嫌疑。由此可見，騰訊音樂對版權的壟斷行為並未空穴來風。

然而尷尬的是，我國《反壟斷法》的適用範圍並不包括知識產權，因而沒有法律或政策能明確斷定騰訊音樂的行為是否屬於壟斷，儘管該行為已經基本符合壟斷的條件。
