現在還堅持用三星手機的，一般都是什麼人？

2019-12-23 2,223條評論 4,390次閱讀 797人點贊

在幾年前，一提到三星手機機友們肯定都會豎起大拇指，全球第一大手機廠商可不是白叫的。但現在，一說到三星，機友們只剩下一聲嘆息了，甚至有些機友都快忘記這家手機巨頭了。

目前三星依舊是全球第一大手機廠商，但在國內市場三星早已經不成氣候了，銷量份額甚至不到1%。雖然三星在國內幾乎銷聲匿跡，可依舊有一些人在用三星手機，好物君分析了一波，現在還在堅持用三星手機的，主要分為以下三類群體。

第一類：三星死忠粉

首先要說的自然是三星的死忠粉了。

別看三星現在在國內賣得不怎樣，但痴迷三星手機的機友大有人在，因為三星本身就是一家實力非常雄厚的手機廠商，這一點從它的全球銷量就能看得出來。

三星手機之所以能在全球各地都很暢銷，與它手機的高水準是分不開的，沒有人會為不好的手機買單，三星能連續多年佔據全球銷量排行榜榜首的位置，那說明它旗下的手機確實讓人很滿意，既然如此，機友們會成為三星的死忠粉也就不足為奇了。

第二類：科技愛好者

對手機參數和配置了如指掌的科技愛好者們也很喜歡三星手機，因為三星手機往往就代表著目前手機的最高水準。

這句話並不是誇大，三星手機的水準有些時候要比蘋果和華為更高，就比如之前蘋果發布了第一款劉海屏手機iPhone X，在手機圈掀起一股劉海屏風潮，機友們並不喜歡這種外觀，但幾乎各家廠商都開始效仿蘋果，唯一一家不使用劉海屏手機的廠商就是三星。

三星不僅不使用劉海屏，還推出了一種全新的打孔屏外觀，比劉海屏更好看，這正是明年旗艦手機的主流外觀，三星再次成了領跑者。不光在外觀方面有自己獨特的設計理念，三星在屏幕、處理器、內存、相機甚至是電池等眾多手機配件方面都廣有涉獵，其他友商很多時候都得依靠三星。

第三類：手機發燒友

第三類使用三星手機的人就是對手機要求很高的手機發燒友。現在的手機五花八門，數量很多，但真正品質過硬的機型並不多，甚至就連蘋果手機也有不少讓人吐槽的缺點。但三星手機卻幾乎能做到零槽點。

三星的旗艦機向來都被認為是手機圈最頂級的機型之一，從顏值到配置，從拍照到續航，甚至就連屏幕和音效都能做到無可挑剔，這絕對不是一件容易的事，可三星總是能滿足我們對於手機的全部幻想，打造真正完美的手機。

三星手機雖然在國內銷量不佳，但內行人都知道，三星旗艦是手機圈裡出了名的高品質手機，要是預算充足的話，很多機友甚至會捨棄蘋果和華為，轉而入手一款最新的三星手機。

當然，機友們應該也看到了上面好物君說了一個前提條件，預算得足夠，三星手機哪裡都好，唯獨價格讓人猶豫，這也是它在國內火不起來的一個重要原因，機友們身邊有沒有用三星手機的人呢？
