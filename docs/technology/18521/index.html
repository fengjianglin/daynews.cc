實力對比

性 能 對 比

》》》》續駛能力：Model 3續駛里程EPA標準為496公里，超過任何一款未來5年內上市的車型；

》》》》加速性能：Model 3百公里加速相比較其他兩輛新能源車還是具有極大優勢的，0-100km/h加速僅4.7秒；

》》》》充電功率：快充充電功率Model 3 120kw（可通過軟體升級進行優化），I-PACE 100kw，MARVEL X 50kw；快充從20%至80%都只需40分鐘；

系 統 對 比

》》》》懸架形式：MARVEL X 麥弗遜式懸架；I-PACE同樣是雙叉臂結構形式，下擺臂由兩個單臂組成A型叉臂；Model 3是雙叉臂結構形式，下擺臂採用雙球節的形式；

》》》》車身：MARVEL X採用純鋼車身；I-PACE採用全鋁車身；Model 3 車身由於採用大量的超高強鋼、使整車性能大幅度提升，從而也更好的保護了電池包，所以Model 3 的電池包可以做的更輕只有469.5公斤，這是創新的設計理念。

》》》》熱管理系統：MARVEL X空調採用熱棒系統，效率高、能耗較低。電池包採用的是自然風冷方式，模組與托盤底板之間通過散熱膠傳遞熱量，殼體通過自然風冷進行散熱，成本相對比較低，由於電池包儲能性能受溫度影響很大，因此電池性能大大降低，冬季在中國寒冷的北方普及難度較大；

I-PACE 電機電池空調製冷和取暖四套獨立循環系統，通過控制閥體關閉流經系統冷卻交換器的流量來控制溫度；

Model 3根據不同的工況形成主要的四種循環散熱迴路，確保電池工作在最優化、最一致的溫度區間，使電芯的環境溫度均衡一致，延長電池循環壽命；

》》》》電機：MARVEL X三電機（前置1個永磁同步+後置2個永磁同步）雖然有三個電機，但單電機最大功率僅85kW；I-PACE雙電機（前置永磁同步+後置永磁同步）單電機最大功率147kW；Model 3雙電機（前置交流感應電機+後置交流永磁同步電機），單電機最大功率180kW；

》》》》動力電池：MARVEL X電池可用容量52.5kwh，採用的是寧德時代方塊電芯；I-PACE電池可用容量76.2kwh，採用的是LG軟包電芯；Model 3電池可用容量75kwh，採用的是松下2170圓柱電芯；在容量上可見I-PACE優於Model 3 和MARVEL X；

》》》》集成度：Model 3 採用的是電機、減速箱、電控三合一集成，I-PACE 和MARVEL X 採用的是電機、減速箱二合一集成；

MARVEL X採用了三電機四驅（二合一）設計，為了滿足大功率高動力性，採用了後置雙電機，相對較龐大，各電機控制器就近布置，故後部空間無剩餘，車載充電機，DCDC模塊均布置在前艙，相同動力需求下，動力技術上的差距，造成動力系統過於龐大，空間利用率低，前艙無法布置儲物空間;

I-PACE前後電驅（二合一）採用的同心傳動，傳動軸穿過電機中心，極大的節省了空間，為前電機控制器，DCDC模塊，車載充電機，不僅以類似於「三明治」疊裝在前電機上部提供了足夠的空間，而且提供了一個前艙儲物空間；

Model 3為了節省空間電機控制器集成到了電驅中（三合一），DCDC模塊與車載充電機集成到了一起，整體採用了扁平化設計，安裝到了二排座椅下部，電池包上方，部件的高集程度，輕鬆的在前艙提供了一個儲物空間;

結合國內外專家對新能源汽車領域的深入研究，我們認為Model 3 整車集成度更高、智能化程度更先進、零件數量更少、成本更低、技術較領先。
