央視評視頻網站會員再加點播費：別過早薅禿一隻羊

2019-12-15 527條評論 2,159次閱讀 879人點贊

文/李曉亮

最近視頻網站薅羊毛有點兒狠：「《慶余年》50元超前點播惹爭議」「《慶余年》50元得罪全網，畸形收費從何而來？」有此疑問的不止一兩個人。先要說一下，會員收費和超前點播差別。當然，要說明網友怨氣，還得再往回倒，先說會員套路。

「坑太深！視頻網站會員還有專屬廣告，看好片要另出錢」。最初，視頻前廣告秒數，也就以十位數計，後來幾乎均在三位數往上。看幾分鐘短片，恨不能給你插兩三分鐘廣告。再者，熱播大片，甚至全線產品，都優先甚至只限會員，不少最終結局也設限，變相逼你充值。忍無可忍充了會員，才發現還細分黃金、鑽石、鉑金等，層層加碼，誘導消費。更過分的，一般會員許可權外，還有單片單獨付費。

會員只免前端廣告，熱播大劇基本都有中插的硬廣軟廣，有些甚至不能跳過。躲過開頭，躲不過中間。最狠的還是，套餐期滿，多默認勾選自動續費。防不勝防，坑你沒商量，即便申請取消，也是麻煩重重，苦不堪言。

有此種種不堪「前科」，還不識相地鼓搗出個50塊錢「超前點播」，不犯眾怒才怪。這是遊戲里常見套路，充值無底洞，培育「人民幣玩家」。但真正優質平台服務，不該如此貪得無厭、吃相難看。

如此心急或因回本壓力，是為之前圈內各種天價亂象買單。但薅羊毛也得耐心等待，且不能只擱一頭羊上重複薅。內容付費，才剛剛起步。大環境還未成熟，習慣還未完全養成之前，就竭澤而漁、殺雞取卵，是飲鴆止渴，會遭反噬。長期持續供給優質內容和服務，才能反哺孕育會員體系健康良性成長，該是常識。
