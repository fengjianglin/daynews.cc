Discuz X3.4免插件採用定時任務自動生成sitemap.xml

2019-12-07 1,708條評論 4,212次閱讀 2848人點贊

為了更好的利於SEO，加快百度、360、谷歌、bing、搜狗等收錄，我們需要網站生成sitemal.xml，有些搜索引擎只支持.xml,不支持.txt,所以我們必須在511遇見論壇下自動生成sitemap.xml，由於我們採用了偽靜態，所以生成的鏈接必須和偽靜態規則一致，在discuz後台插件搜索了幾個，姑且不談收費免費，發現很不穩定，且靜態規則無法兼容，這裡我們採用discuz的定時任務，自動生成itemap.xml

手動建立一個cron_sitemap.php文件複製以下內容

注意自己discuz的編碼，我這裡utf-8,注意手動修改。

<?php
/*
 * $ cron_sitemap.php 2019-11-23 網站地圖 計劃任務版 bbs.511yj.com
 */
 
if(!defined('IN_DISCUZ')) {
	exit('Access Denied');
}
 
$filename='sitemap.xml';
//以下五項根據具體情況修改即可
$cfg_updateperi='60';//協議文件更新周期的上限，單位為分鐘
$web_root=$_G['siteurl'];//根網址
$CHARSET='utf-8';// or gbk //選擇編碼方式
/***********************************************************************************************/
//網站地圖sitemap.xml
$sitemap="<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n";
$sitemap.="<urlset\n";
$sitemap.="xmlns=\"http://www.sitemaps.org/schemas/sitemap/0.9\"\n";
$sitemap.="xmlns:xsi=\"http://www.w3.org/2001/XMLSchema-instance\"\n";
$sitemap.="xsi:schemaLocation=\"http://www.sitemaps.org/schemas/sitemap/0.9\n";
$sitemap.="http://www.sitemaps.org/schemas/sitemap/0.9/sitemap.xsd\">\n";
 $querys = DB::query("SELECT a.tid FROM ".DB::table('forum_thread')." a inner join ".DB::table('forum_forum')." b on a.fid=b.fid ORDER BY a.tid DESC LIMIT 0,10000");
 while($threadfid = DB::fetch($querys))
 {
	 $turl=$web_root.'thread-'.$threadfid['tid'].'-1-1.html';//注意靜態規則
 $link = $turl;
		$t=time();
		$riqi=date("Y-m-d",$t);
		$priority=rand(1,10)/10;
//date("D F d Y",$t);
$sitemap.="<url>\n";
$sitemap.="<loc>$link</loc>\n";
$sitemap.="<priority>$priority</priority>\n";
$sitemap.="<lastmod>$riqi</lastmod>\n";
$sitemap.="<changefreq>weekly</changefreq>\n";
$sitemap.="</url>\n";
 }
 $sitemap .= "</urlset>\n";
 $fp = fopen(DISCUZ_ROOT.'/'.$filename,'w');
 fwrite($fp,$sitemap);
 fclose($fp);
?>

把cron_sitemap.php文件上傳到論壇目錄

把cron_sitemap.php上傳至source\include\cron目錄

discuz後台添加定時任務

後台–工具–計劃任務–新增，名字隨便，提交
然後編輯，任務腳本：cron_sitemap.php
