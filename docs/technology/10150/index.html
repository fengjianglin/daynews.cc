Mybatis-Plus 3.3.0 發布，讓你專註業務快速敏捷開發

2019-12-07 305條評論 1,375次閱讀 217人點贊

Mybatis-Plus 3.3.0 發布了，Mybatis-Plus是一款MyBatis的增強工具包，簡化 CRUD 操作。啟動載入 XML 配置時注入單表 SQL 操作 ，為簡化開發工作、提高生產率而生。Mybatis-Plus 啟動注入非攔截實現、性能更優，讓你專註業務快速敏捷開發

更新日誌：

  • BaseMapper 介面兩個 page 方法優化
  • IService 以及 ServiceImpl 對應 page 方法優化,個別返回 collection 的方法修改為返回 list
  • 邏輯刪除欄位的兩個表示已刪除和未刪除的定義支持字元串 “null”
  • 修復批量操作未清空緩存
  • 批量操作異常轉換為DataAccessException
  • mybatis up 3.5.3, mybatis-spring up 2.0.3, jsqlparser up 3.1
  • mapper 選裝件包調整, chainWrapper 包調整
  • 新增 ChainWrappers 工具類
  • 新增 IdentifierGenerator 介面,支持自定義Id生成
  • 代碼生成工具廢棄正則表名匹配,新增likeTable與notLikeTable
  • 分頁插件支持自定義處理頁數限制與溢出總頁數處理
  • 修復SqlExplainInterceptor導致的Oracle序列自增兩次
  • 分頁二級緩存支持
  • 擴展p6spy日誌列印
  • DbConfig加入新屬性propertyFormat,TableFieldInfo移除屬性related
  • 優化序列生成器,過時KeySequence的clazz屬性
  • 修復Ognl表達式關鍵字導致的null值判斷失效
  • 修復更新填充開關失效
  • 優化填充邏輯
  • ISqlRunner支持selectPage
  • 支持全局邏輯刪除欄位
  • BaseMapper的方法可自定義
  • 添加【虛谷】【Oracle12c】【Kingbase】資料庫支持
  • 解決資料庫欄位與實體欄位名稱不同時出現null as xxx的情況
  • 過時ID_WORKER_STR,自動識別主鍵類型
  • 配置開啟註解，TableName也強制生成
