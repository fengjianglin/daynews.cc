廣東刷新全球最大氫燃料客車訂單，2支氫能源個股迎來重大利好

2019-12-08 1,263條評論 3,110次閱讀 1863人點贊

日前廣東佛山市南海區2019年燃料電池公交車項目招標結果公示，雲南五龍汽車成功中標，由此獲得了200輛氫燃料汽車和47輛純電動汽車、總銷售額為4.26億元的訂單。據了解，這是全球氫燃料客車一次中標數量最多的訂單。

業內表示，氫能汽車技術突飛猛進，不久前本田發布的第二代clarityfcv車型氫能汽車，加氫只需3分鐘，最高續航750公里。我國《新能源汽車產業發展規劃（2021-2035年）》（徵求意見稿），明確提出要大力發展燃料電池汽車，要有序推進氫燃料供給體系建設。這無疑對氫能源汽車發展帶來了利好消息。

點評：目前，國內主流車企都在加深對氫燃料電池汽車領域的布局，據不完全統計，中國已有超40家車企開始研發燃料電池汽車。機構分析稱，燃料電池行業是所有能源形式裡面唯一可以橫跨發電、儲能、交通三大領域的能源，未來市場達萬億元。

雄韜股份（002733）首條全自動氫燃料電池電堆產線日前正式投產。

嘉化能源（600273）擁有1萬噸低成本副產氫資源，強強聯手部署產氫、儲氫、加氫等產業鏈重要環節。
