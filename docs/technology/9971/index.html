Jean Tirole：區塊鏈技術不可能取代銀行地位

2019-12-08 238條評論 2,076次閱讀 653人點贊

摘要： 2014年諾貝爾經濟學獎獲得者Jean Tirole發表主旨演講表示，區塊鏈技術不可能取代銀行的地位，比如銀行承擔著存款儲蓄的功能，這是需要滿足長期、安全的要求的。

鏈得得（ChainDD）12月8日現場報道，在由鈦媒體主辦、鏈得得協辦的2019 T-EDGE 新金融峰會暨 CHAINSIGHTS 金融科技與區塊鏈中國峰會上，2014年諾貝爾經濟學獎獲得者Jean Tirole發表主旨演講表示，區塊鏈會帶來更好的收益，更低的成本，更好的服務，獲得更多的融資渠道。但對於某些國家和某些業務而言，它帶來的是破壞效應和負面效果。很多人覺得新興技術會是決定性的作用，從某些角度來說確實是如此，但是這樣想是錯誤的。

Jean Tirole認為，區塊鏈技術不可能取代銀行的地位，比如銀行承擔著存款儲蓄的功能，而這是需要滿足長期、安全的要求，而區塊鏈是沒有辦法替代這些長期的銀行服務的。

2019 CHAINSIGHTS金融科技與區塊鏈中國峰會於北京·大興 星光視界中心正式舉行。鏈得得、ChainDD攜手鈦媒體盡攬全球金融界領袖以及區塊鏈尖端人物，薈聚北京，集中探討區塊鏈和金融科技將引領這個分野與共識並重的時代走向何方。
