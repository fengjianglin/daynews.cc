P的病歷能發起10萬愛心捐款？水滴籌：嚴厲譴責

2019-12-10 2,492條評論 4,015次閱讀 2528人點贊

澎湃新聞記者 王去愚

連續道歉的水滴籌，又被曝出P的病歷也能發起愛心捐款。

12月9日晚間，有微信公眾號發文稱，水滴籌道歉6天後，「我P的病歷還能發起10萬愛心捐款」。該文詳述了自己如何測試用虛假病歷在水滴籌募捐的過程。


12月10日上午，北京水滴互聯科技有限公司（簡稱水滴公司）給澎湃新聞（www.thepaper.cn）發來回應稱，平台近期發現多起虛假籌款項目，在調查取證過程中發現部分項目存在「釣魚」情形。同時，「我們也注意到某公眾號PS虛假病歷發起水滴籌籌款申請的文章，引發了公眾對水滴籌平台的誤解。」

前述微信公號文章稱，測試者以「丁某某」患有精神分裂症為由，在水滴籌發起10萬元愛心籌款。在收到1元錢籌款後，水滴籌系統以簡訊通知當事人，「您的籌款效果不佳」，可以通過多種方式轉發募捐信息。與此同時，籌款人收到水滴保險商城的「600萬醫療保障」險的投保鏈接。

該文認為，水滴籌前腳剛紅著眼圈給癌症病人捐完款，後腳就提醒你購買保險避免這樣慘。「堪稱販賣焦慮的終極版本」。

對此，水滴籌方面回應稱，作為 「救急難」的個人社交網路大病求助平台，水滴籌會在籌款發起、傳播、提現等整個過程中，通過材料審核、微信等熟人社交網路傳播驗證、第三方驗證等多種手段對籌款進行全流程動態審核。

上述回應稱，求助人在發起時提交身份證明材料、病情證明材料等相關材料，通過平台初步審核之後，還需要根據實時治療進展提供醫療花費票據、檢查報告、出入院證明等醫療證明材料，並經過社交網路的監督驗證、提現公示驗證等環節，才能最終完成提現。同時，水滴籌除了證明材料審核外，還有醫院實地探訪、大數據等核實方式保證籌款方式準確。

「對於僅僅提交了基礎信息而沒有通過後續層層審核的籌款，既無法籌集到所需資金，也無法最終完成提現，籌款也會原路退還給贈與人。」水滴籌方面稱。

前述微信公號文章也提及，其這次提交的籌款申請，尚未完全獲得水滴籌的完全審核通過，但可先行籌款，系統自動把金額調成了10萬。「而在審核未全過的情況下，不影響轉發朋友圈眾籌。」

對此，水滴籌方面稱，該文作者僅僅是發起了求助，看似可以籌款，但因其並未通過社交網路傳播驗證、大數據驗證、第三方驗證等後續環節，在水滴籌的全流程動態監控過程中，隨時會被定位為虛假籌款行為，凍結籌款、終止項目、退還款項甚至追究法律責任。

關於在發起申請環節只需提交基礎醫療證明、身份證明等材料，更多證明材料需後續補充，水滴籌方面重申，大病救助本身是一個與病魔鬥爭和時間賽跑的過程，在這一階段患者準備齊全所有相關證明有很大難度且需要較長時間，而另一方面常常是患者等錢做手術治療，我們在積極尋求此間的平衡。但如上所述，從平台全流程的審核機制來看，如果一個項目不符合籌款條件，縱使申請發起了，最終也很難通過平台審核提現。

水滴籌還「嚴厲譴責」通過製作、購買虛假材料騙取網路同情、消耗社會資源和愛心的不當行為，「我們亦會採取包括法律手段在內的多種手段以捍衛贈與人的善心。今年以來水滴籌就聯合多地公安嚴懲了採取類似方式涉嫌詐騙的5名籌款人。」
