廈門副市長孟芊：發展信息產業、建設智慧城市，最缺的還是人才

2019-12-07 2,049條評論 4,095次閱讀 477人點贊

為全面落實新時代全國高校本科教育工作會議精神，推進新時代計算機專業建設和人才培養，提高高校服務經濟社會發展能力，推動計算機教育和產業的深度融合，攜手構建計算機教育新生態，12月7日，主題為「新時代、新計算、新教育」的2019中國計算機教育大會在廈門隆重舉行，就中國產業發展及產教融合等一系列問題展開深入交流，並聚焦信息技術發展前沿問題，深入探討信息技術，特別是人工智慧快速發展下，計算機教育發展創新的思路和舉措。

　　本次大會由教育部高等學校四大計算機類教學指導委員會首次聯合舉辦，是新世紀以來規格最高、規模最大的面向國內高校和相關產業界的計算機教育盛會。會議由中國工程院院士，教育部高等學校計算機類專業教學指導委員會主任委員，中國計算機教育聯合會主任委員，清華大學計算機科學與技術系主任、教授吳建平主持。

廈門市副市長 孟芊

　　開幕式上，廈門市副市長孟芊代表廈門市委政府發表精彩致辭。他表示，本次大會對新時代計算機學科建設、人才培養以及產學研合作具有非常重要的意義，對於廈門來說，也是難得的發展學科、推動產業、引進人才的機會。

　　當今，計算機學科滲透到人類生活各個領域，成為深刻影響國家發展、科技進步以及新一輪科技創新、驅動產學研發展的重要學科。我們處在一個飛速發展的人工智慧時代，計算機學科教育跨界融合發展、科教協同、產學研協作等領域面臨許多需要大家共同研究探討的問題。 「新時代、新計算、新教育」成為教育界和產業界高度關注的重要課題。孟芊表示，在此背景下，本次大會具有十分重要的意義，尤其是許多專家學者帶來的權威報告和前沿趨勢熱點問題的深度交流和研討，必將為我國計算機學科的發展以及計算機教育和相關產業發展深度融合提供更多的解決方案。

　　孟芊強調，廈門市是隨改革開放而興盛起來的一座新興之城，「兩高兩化」是廈門市的城市定位和發展目標。具體在產業和教育上，IT產業是廈門市發展的重要產業，是廈門目前最大的產業集群。無論是計算機製造還是軟體信息，都具有相當的規模和水平。在人才培養方面，廈門市高等重視高等院校計算機學科建設和人才培養工作。廈門大學、華僑大學、集美大學和廈門理工學院等十多所本地大中專院校均已開設計算機專業，全市計算機專業在校生約2.5萬人，每年畢業生約6000人。

　　同時，廈門市政府還積極引導各高等院校和企業開展共建特色專業、共建實習基地等形式的人才培養合作，著力培育應用型計算機和軟體信息人才。他強調，正是由於這些年對相關學科的重視，才有力推動了產業迅速發展。2018年全市計算機與通信設備產業、軟體信息服務業均已突破千億，數字經濟規模達到6000億左右。

　　當前，廈門市全面加快信息化基礎設施建設，大力推動互聯網和實體經濟融合發展，以企業上雲推動傳統企業轉型升級，實現新舊動能轉換。推動雲計算、大數據等現代信息技術在智慧城市中的示範應用和有力融合，爭取實現政務信息互聯互通和無障礙信息共享。

　　在發展信息產業，在建設智慧城市上，最缺乏的還是人才。孟芊表示，僅僅就軟體信息這一個領域來說，廈門市每年需要的人才規模就在13000人以上，而廈門市本地高校的培養能力還不能完全滿足企業的需要。他表示，廈門熱切地歡迎全國計算機學科的領導、專家把學生送到廈門來實習、就業，更歡迎各位專家學者帶著技術成果過來，廈門在全國有著最好的法治環境、最好的營商環境和最好的人才發展環境和最好的居住環境，真誠期盼各位專家學者為廈門的建設發展建言獻策，提供寶貴的意見建議。

　　據悉，除主論壇外，大會期間還開設了一流本科專業建設、大數據教育、產學合作協同育人、網路安全人才培養、一流課程建設、人工智慧教育等15個專題分論壇，以及4個教育部教學指導委員會年會和數十個創新應用展，以前沿創新技術多維度賦能教育領域，打造計算機教育業界盛宴。
