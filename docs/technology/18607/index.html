宣布完成收購德國ARADEX 濰柴動力在新能源領域要飛天

2019-12-17 2,067條評論 3,729次閱讀 2786人點贊

2019年12月16日，濰柴動力股份有限公司（香港聯交所股份代碼：2338；深圳證券交易所股份代碼：000338）宣布已完成對德國ARADEX股份有限公司（以下簡稱「德國ARADEX」）的戰略收購。本次收購是繼濰柴動力與英國錫里斯動力攜手及戰略投資加拿大巴拉德後，在新能源動力系統領域上的又一次國際化戰略布局。至此，濰柴動力掌握了重型商用車傳統動力總成領域的關鍵核心技術，亦同時成功構架起「電池+電機+電控」為一體的新能源動力系統集成優勢，以德國ARADEX新科技賦能濰柴新能源推動全產業鏈快速發展，為實現「2030年新能源業務要引領全球行業發展」戰略目標奠定先進的產品技術基礎。

德國ARADEX公司自1989年成立以來，致力於工業及交通運輸行業用電機、電控和電源的研究開發工作，是大功率變頻和電源設備的技術領先者，擁有多種數控機床用伺服電機和驅動器、新能源商用車用電機控制器、電機、燃料電池DC/DC變換器等產品的設計開發和系統集成能力，產品設計壽命達3萬小時以上，性能優異，並廣受讚譽，已有數百種成功應用案例。

未來，雙方將充分發揮協同優勢，共同致力於新能源商用車動力系統的開發及集成工作。濰柴豐富的全球整車整機、研發創新中心及動力系統核心部件等優勢資源將助力德國ARADEX繼續發展壯大；德國ARADEX領先的新科技優勢將助推濰柴動力進一步加快產業升級和結構調整。雙方深度融合將促進節能減排，為打贏「藍天保衛戰」和新能源產業發展提供可持續的動力保證。
