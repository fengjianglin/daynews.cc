被大家低估的中興：晶元、操作系統、5G都表現突出

2019-12-21 1,758條評論 3,563次閱讀 2202人點贊

說起國內的晶元、操作系統、5G我想大家都會提華為，因為目前華為5G是全球第一的、而晶元是中國第一，操作系統方面，鴻蒙的影響也是非常大。

但其實我想大家可能低估了中興，這家老牌企業，其實在晶元、操作系統、5G上面都表現突出，只是因為華為太耀眼了，遮住了大家的眼光。

先說說中興的晶元，中興旗下有一家晶元公司叫做中興微電子，2017年在IC設計領域排名國內第3，2018年排名第4，也算是國內非常牛的了。

但它不出名，是因為它不生產手機晶元，生產的是基帶晶元，機頂盒晶元，路由器，交換機晶元，以及基站晶元等，再加上營銷水平不行，所以不被大家熟悉。

之前中興表示目前中興是國內除華為外唯一擁有7nm晶元的廠商，也是華為外唯一研究5nm晶元的廠商，其實說的就是微電子研究的5nm的基帶晶元。

再說說中興的操作系統，中興有一款系統叫做中興新支點，目前出貨量已經超過2億套。它之所以不出名，也一方面是因為不會營銷和吹牛，另外一個原因是因為它沒有面向個人用戶，主要是面向行業用戶，比如教育、政府、金融等領域。

這款系統是基於linux修改的，支持ARM、PowerPC、MIPS、X86等多種架構。而除了這款系統之外，前段時間中興和深度、CEC、誠邁科技四家單位聯合研發操作系統UOS，目前也有了測試版發布了。

再說說中興的5G，這從兩個方面來說明，一個方面是5G專利，最新的數據不可考，但按照IPlytics發布的5G專利報告顯示，截止至今年4月份，中興的專利數量排名全球第三，僅次於華為、諾基亞。

而近日IHS Markit統計了全球5G市場的發貨量情況，在2019年第三季度，華為、三星、中興排名前三，發貨量佔比分別為33%、21%、18%；今年的第一、第二和第三季度，前三甲都是華為、三星、中興。

可見中興一直就沒有掉隊，在5G時代，甚至成為了前三，超過了諾基亞、愛立信了。

這麼一分析之後，大家是不是覺得低估了中興了？以為去年這樣被一擊便倒，所以中興沒實力？那你是想錯了，中興正好不研發手機晶元、也不研發手機系統，所以才這以容易被擊倒的，但在它自己的專業領域，可是秀厲害的哦。
