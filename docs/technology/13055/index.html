全球72家供應商推出183款5G設備，下一個超級終端的機會在哪？

2019-12-11 101條評論 4,692次閱讀 3418人點贊

國際知名市場分析機構IHS Markit10日在深圳發布《智能互融：藉助5G、人工智慧和雲技術，釋放機遇》白皮書稱，2019年，全球見證了第一波5G商業發布。截至2019年10月，已有50家運營商在27個國家和地區推出了符合3GPP(第三代合作夥伴計劃)要求的5G商業服務。

今年年底，5G將在全國50個城市普及

白皮書也分析了目前世界各國5G部署的現狀。其中，中國和韓國已經大規模部署了超過10000個5G NR/gNB（基站）。在澳大利亞、英國、沙烏地阿拉伯、瑞士、阿聯酋和美國，已經有數百個5G NR/GNB的小規模戰術部署。

而在中國，5G及其應用即將迎來大規模爆發。白皮書認為，近年來，中國一直在為5G商用化做準備。2019年11月初，中國三大移動運營商——中國移動、中國電信和中國聯通推出了商用5G服務，比計劃提前了幾個月。預計到今年年底，這三家公司將部署約10萬個5G基站，2020年將增至100萬個。到今年年底，5G將在全國50個城市普及。

全球72家供應商宣布推出183款5G設備

事實上，5G設備生態系統的發展已經成為了各領域發力的主要方向。

報告介紹，截至2019年11月中旬，全球共有72家供應商宣布推出183款5G設備，包括智能手機、室內和室外CPE、機器人、無人機等。其中超過40款已投入商用。

白皮書認為，5G時代將開啟以個人體驗為中心，融合技術、設備與服務的萬物互融時代。5G、AI與雲技術作為這一互融時代的三大支撐，將不僅能實現設備之間的互聯互通，以及跨地域、跨場景數據的無縫智能溝通，還將與個人用戶緊密聯繫，與用戶生活中的方方面面相融合。

在遊戲層面，雲遊戲和VR/AR也將迎來嶄新機遇。雲遊戲可大幅降低消費者的硬體門檻，在更小的設備上隨時隨地享受高體驗遊戲，並減少延遲，進一步催生VR/AR遊戲的全面發展。白皮書表示，早在2018年，這一市場的全球價值便超過了1000億美元。

此外。視頻監控、自動駕駛、工業自動化都蘊含了大量的萬物互融場景與機會。

到2023年，全球4.25億部手機將集成5G

根據白皮書預測，到2023年，全球17億部手機中約四分之一將集成5G。智能手機將越來越具備人工智慧能力：IHS Markit預測，到2025年，三分之二的智能手機將預裝人工智慧硬體和功能。

在萬物互聯的時代中，智能手機作為個人進入數字世界入口的角色不僅沒有削弱，反而更加重要，可以說是下一個超級終端。作為數字世界中交互最易、連接最廣、算力最強的超級終端，手機的功能與形態將在滿足外觀極簡與便攜中不斷發展。通過5G管道，AI與雲技術將成為手機應用的後台能力，手機的算力並不會因為雲或AI的到來而削弱，而將不斷提升。
