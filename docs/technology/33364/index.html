電動車創企Rivian融資 13億美元 亞馬遜等參與

2019-12-24 1,807條評論 2,906次閱讀 957人點贊

科技訊 12月24日消息，據國外媒體報道，電動皮卡製造商Rivian從以投資公司普信（T. Rowe Price）為首的投資者那裡籌集到13億美元的新資金。


圖示：R1T插電式皮卡

據悉，Rivian此前已經得到亞馬遜和福特汽車的支持，將於明年推出首款汽車。以普信牽頭的融資是Rivian今年所獲得得第四筆主要投資，這也使Rivian獲得的總融資額達到了28.5億美元。這家總部位於密歇根州普利茅斯的製造商表示，亞馬遜、福特和貝萊德管理的基金也參與了本輪融資。

Rivian吸引了大量想要轉向電動汽車的企業。過去七年中，公司首席執行官R.J.斯卡林格(R.J. Scaringe)一直致力於開發皮卡和SUV。這款車將以通用架構為基礎打造，將來也可以用於製造其他車型。斯卡林格將這個平台稱之為「滑板」，也是他願景的重要組成部分。


Rivian的一些投資者也是其未來客戶。亞馬遜已向該公司訂購了10萬輛電動送貨車，用於「最後一英里」商品交付業務。而福特計劃將Rivian開發的汽車平台用於一款新型電動汽車。Rivian計劃在明年晚些時候推出R1T插電式皮卡和R1S SUV。

普信對Rivian的投資是其今年對未來汽車行業未來進行一系列投資的最新舉措。今年5月，普信曾加入本田汽車和軟銀願景基金的行列，為自動駕駛初創企業Cruise LLC共同提供11.5億美元的融資。目前通用汽車持有Cruise LLC的多數股權。

普信發言人表示，這筆對Rivian的投資資金來自普信提供諮詢服務的某些共同基金和其他賬戶。（辰辰）
