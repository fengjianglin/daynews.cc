小米正式進入日本：首款手機是小米CC9 Pro國際版

2019-12-09 275條評論 1,241次閱讀 130人點贊

IT之家12月9日消息 小米手機今日正式進入日本市場，同時首款進入日本市場的手機便是小米CC9 Pro的國際版本Mi Note 10。


對此小米日本官方也在第一時間在推特上公布此事，據介紹，本次進入日本市場的Mi Note 10將擁有極光綠、冰川白、午夜黑三種配色，採用全曲面機身，搭載6.47英寸雙曲面屏，搭載驍龍730G處理器，內置5260mAh條形快充電池，標配30W閃充。

同時小米日本官推還詳細介紹了Mi Note 10所採用的1億像素CMOS：


相機方面，Mi Note 10採用了和小米CC9 Pro相同的五攝四閃方案，後置五攝為超長焦攝像頭（支持10倍混合變焦和50倍數字變焦）+1200萬像素50mm經典人像鏡頭+一億像素主攝像頭+2000萬的廣角攝像頭（117°大視角）+超近微距攝像頭。
