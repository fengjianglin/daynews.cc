比特幣熊市影響凸顯 研究表明入行區塊鏈最好時機已經到來

2019-12-19 982條評論 3,016次閱讀 1002人點贊

政策法規

歐洲央行：正開發保護用戶隱私的央行數字貨幣付款系統

據Cointelegraph消息，歐洲中央銀行（ECB）最近的研究聲稱，正在開發保護用戶隱私的中央銀行數字貨幣（CBDC）付款系統。根據這份名為《探索央行數字貨幣的匿名性》的報告，歐洲央行系統建立了一個CBDC中匿名的概念驗證(PoC)，這是其正在進行的CBDC研究及其對公眾潛在利益的一部分。

德國聯邦金融監管局評估「市場參與者風險」以嚴格監管加密資產

據Finextra消息，德國聯邦金融監管局（BaFin）正在對由加密貨幣支持的衍生產品進行市場調查。該調查將重點關注比特幣和以太坊等加密貨幣，並徵詢了投資者，消費者保護協會和其他利益集團的意見。Bafin主要評估「市場參與者的風險」，理由是投資者近年來對風險較高的加密貨幣支持衍生產品（例如差價合約）的興趣日益濃厚，監管機構應採取措施對加密資產進行更嚴格審查。

印度所得稅部門正在培訓官員調查加密貨幣

據CoinTelegraph今日報道，印度所得稅部門正在秘密訓練官員調查加密貨幣，該部門發行的一本內部指南解釋了加密貨幣、它們的特點、「比特幣的陰暗面」以及稅務官員的最佳調查做法。據悉，印度政府不認為加密貨幣是法定貨幣，但也沒有宣布它們完全非法，通過加密貨幣賺錢是印度加密社區的灰色地帶。

數字金融

數據顯示：市值前十加密資產EOS鏈和幣安鏈完整節點同步難度係數最高

12月17日，區塊鏈研究平台Coin Metrics發布最新文章，其對加密市值排名前十的加密項目節點操作排名顯示，市值排名前十加密項目中，BTC、BCH、LTC和BSV完整節點同步、更新和維護的難易程度較為簡單。ETH和USDT（Omn節點）完整節點同步、更新和維護的難易程度適中。

EOS和幣安鏈節點同步更新和維護的難易程度等級最高。Coin Metrics解釋稱，EOS擁有負責模型，提取必要數據以審核過於複雜，同時，同步節點還需要運行帶有額外插件的存檔節點，此外EOS借貸方數據量過大，也增加了同步難度。Coin Metrics稱，同步一個完整的歸檔EOS節點花費了一個多月時間，且需要一台裝有TB級NVMe SSD計算機（已擁有很快存儲）；幣安鏈DEX收取一定費用，使得獲取數據比較困難，同時其也是唯一未公布源代碼的鏈，其僅將簽名的二進位文件提供給可能的節點維護者。

最後，Ripple和Stellar節點未能進行同步。Coin Metrics稱，少數資產（特別是Ripple）需要特別大的存儲量（數十TB），以至於無法運行和同步節點。

報告顯示：目前是在區塊鏈領域工作的最好時機

據Decrypt消息，近期兩項研究報告顯示，區塊鏈行業工作現在是求職者的機會。在求職人數減少的同時，可供選擇的區塊鏈工作數量正在增加，這給求職者提供了機遇。12月17日，LinkedIn發布新興工作報告稱，在未來一年的趨勢中，比特幣熊市的直接後果是，沒有開發人員接手區塊鏈的工作。同時，求職公司Indeed最近一項研究顯示，自去年以來，區塊鏈部門職位空缺實際上增加了26%。這項研究也證實了LinkedIn報告的發現，人們對區塊鏈行業的興趣正在下降。但此外，根據LinkedIn的說法，印度的情況恰恰相反。LinkedIn上周發布的印度就業報告顯示，到2020年，區塊鏈行業工作是印度發展最快的職業。

彭博社：BTC若跌破下一支撐線6500美元 則可能急劇下跌至4000美元

據Dailyhodl消息，據彭博社發表分析文章稱，新的技術分析表明BTC的下一個支撐線是6,500美元，BTC可能會在該水平附近找到買家。但是如果跌破該線，比特幣就有可能急劇下跌至4,000美元。彭博社的分析主要基於比特幣一直持續走低的事實，並表示BTC可能尚未觸底。金融市場分析師Mati Greenspan也表示，比特幣跌破7,000美元並不是什麼新鮮事，BTC在不改變其長期看漲軌跡的情況下，可能跌至約5,800美元。

北京鏈安：部分Upbit被盜ETH正通過去中心化渠道轉為穩定幣

北京鏈安Chainsmap監測系統發現，Upbit交易所被盜ETH近期正頻繁的通過去中心化交易渠道將ETH轉為穩定幣，進行後續的洗錢行為，目前，這些被盜ETH主要被交易轉化為USDT，以及部分TUSD、PAX、USDC。交易後，這些穩定並會回到涉黑地址，並進一步聚合轉移，我們曾經發現一個地址在六小時內即聚合轉移了價值14.78萬美元的穩定幣。

有數據分析師分析，通過ETH向穩定幣，而且是多種穩定幣的轉化，進而再轉移，將進一步混淆整個洗錢過程，而從目前來看，被盜ETH被洗出的部分依然只是總量的少數。

資金動向&企業合作

ErisX已啟動其加密貨幣期貨市場 目前僅支持比特幣期貨合約

據The Block 12月17日消息，加密貨幣交易所運營商ErisX已啟動其加密貨幣期貨市場。該公司最初計劃推出綁定多種代幣的期貨產品，但目前僅支持比特幣期貨合約。根據新聞稿， 儘管期貨市場今天開放 ，但客戶要等到明年年初才能參與。根據產品頁面，ErisX將以月度和季度合同開始。此前消息，加密貨幣交易所運營商ErisX將於12月17日啟動加密貨幣期貨市場。

美國晶元巨頭AMD宣布加入區塊鏈遊戲聯盟

據PCMSG消息，美國晶元巨頭AMD宣布加入區塊鏈遊戲聯盟（Blockchain Game Alliance/BGA），並為該聯盟提供所需的CPU和GPU晶元，以幫助遊戲開發商在其平台上有效運行基於區塊鏈的相關功能。AMD的區塊鏈負責人Joerg Roskowetz表示，「下一代區塊鏈遊戲平台將使遊戲玩家能夠訪問專有的在線內容，並真正擁有遊戲。此外，它還將為遊戲發行商提供發行數字遊戲內容的新渠道。」

Circle已將其場外交易平台出售給加密交易所Kraken

據The Block 12月17日消息，加密交易所Kraken購買了Circle的場外交易平台服務。消息人士稱，自今年夏天以來，Circle一直在為其OTC櫃檯尋找買家。Circle的外交易平台在2018年的名義交易量為240億美元。

應用動態

上海法院應用區塊鏈技術便捷財產保全

據新華社18日消息，上海市第二中級人民法院17日開通「智慧保全服務平台」。該平台將區塊鏈技術應用於法院財產保全業務，有利於杜絕虛假保函、惡意篡改保函等現象。據悉，「智慧保全服務平台」採用區塊鏈技術，將各家保險公司的保函數據「上鏈」。承辦法官通過系統輸入保函信息即可調取相關數據，並關聯至案件電子卷宗，同時將保函的受理信息即時反饋至保險公司，實現分散式存儲，防止保函信息被篡改。

科羅拉多州高等教育部計劃利用區塊鏈技術將教育系統去中心化

據Decrypt 12月17日消息，科羅拉多州高等教育部公布一項計劃稱，將推出一個類似於「生物實驗室」的專門設施「C-Lab」，利用區塊鏈技術將該州的高等教育系統去中心化。據悉，該實驗室旨在為該州提供一個統一但去中心化的生態系統，用於創新、試點、研究和協作，為教育工作服務。

工行利用區塊鏈技術打造的數字信用憑證，將企業信用延伸到「長尾客戶」

據人民網消息，在昨日舉辦的首屆「工行小微客戶節」上，工商銀行行長谷澍發布了「萬家小微成長計劃」。據了解，2018年工商銀行推出普惠金融服務1.0，發布了「一平台、三產品」，一年多來已取得顯著成效。其中，信用類的「經營快貸」已為近百萬客戶授信超5000億元；抵質押類的「e抵快貸」餘額突破1200億元；交易類的數字供應鏈方面，已與3000多家龍頭企業建立合作，並利用區塊鏈技術打造「可拆分、可流轉、防篡改」的數字信用憑證，將核心企業信用延伸到產業鏈末端的「長尾客戶」，以金融力量賦能全產業鏈發展。
