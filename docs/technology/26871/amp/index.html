Model X對比唐EV 價格差一倍多《新能源高考》只差1分？

天天要聞
2019-12-20

任何以排名形式呈現的榜單，榜首之爭都是最激烈、最受人關注的。上周我們發布了《新能源高考》評測體系2019年度榜單，價格差距相當大的特斯拉Model X和比亞迪唐EV分別位居狀元和榜眼之位，且在總分750分維度中僅相差1分，引起了不小的爭議，憑什麼、為什麼等疑問接踵而來，本文咱們就進入「講卷」時間……

Ps：搜狐汽車·E電園新能源車評測體系在2019年高考之際迎來了全新升級，新的評測體系將以高考為切入點，分為6個維度，分值權重與高考一致，即3項150分+3項100分的科目。

[·總分上的1分之差 差在了哪些「單科」·]

《新能源高考》評測體系共有6大維度：充電續航（動力核心）、動態表現、空間、製造水平及配置、車機系統以及價格質保。其中5大項目直指產品本身，Model X和唐EV這兩款車型有著大個頭兒、高性能、長續航的相同屬性，產品方面的不「偏科」，正是二者獲得高分數的根源。而究竟差在了哪些「單科」，咱們先來看看總分，以下逐一為您詳解。

[·價格這個性價比的「分母」是二者相提並論的源頭·]

參加「考試」的Model X 長續航版（100D）標準配置、不算任何選裝的價格為80.99萬元，唐EV600頂配四驅智聯創世版的售價為35.99萬元，將兩款價格相差一倍多的車型放在一起對比，源於《新能源高考》評測體系對性價比的考量。細說性價比這個關鍵詞，乃是「分子」的性能與「分母」的價格，花一半錢能買到80%水平的產品，必然是要在分數中所有體現的。

[·續航Model X佔優 充電還真不一定·]

Model X（100D）的100kWh電池容量直追大巴等商用車，本身的硬指標就要優於唐EV600的82.8kWh。同時，特斯拉作為一家有著很強科技屬性的汽車企業，在電控系統等軟體方面一直處於行業領先，體現在行駛電耗以及續航方面，Model X確實要略勝一籌。

而換到電動車的另一大核心充電效率方面，可就不一定了。特斯拉的超充自然是具備優勢，Model X相比Model 3來說的老架構，也還沒到落伍的地步。但現階段隨著車輛增加，超充站排隊的情況很常見，即便不排隊也要與其它車分享充電功率，很難「馬力全開」。

2018年初特斯拉改成國標充電介面，這種「吃著碗里」超充、還要「看著鍋里」國標的做法，並沒有達到最初的預期。特斯拉品牌自成一派的通信協議，與國標充電樁的兼容性存在問題，即便正常連接因平台電壓不高、全靠電流，也難有高效表現。

今年年初我們測試之時，Model X在國家電網充電站換到第6根電樁才充上電，多次充電中充電功率「最佳狀態」為46kW，面對比亞迪唐EV也是完敗。而同一時間段內測試的唐EV，在60kW國標快充電樁上，一直能頂著電樁上限充電，接收端車輛經過損失之後，充電功率穩穩的在56kW以上。

[·加速唐EV四驅版勝出 不帶「P」特斯拉沒那麼「暴躁」·]

特斯拉可以說是掀起電動車性能的始作俑者，此前展示的全新一代Roadster加速0-100km/h在2秒以內完成，已經不帶任何燃油版超跑玩了。而特斯拉非Performance版本的車型，其實沒有想像中那麼「暴躁」，Model X 100D的0-100km/h加速實測為5.6秒，而四驅雙電機版本的唐EV則是4.5秒。

[·空氣懸掛是電動車動態表現的最佳拍檔·]

相對於評判性能本身上限的加速測試，日常行駛表現更為重要，電動車動力電池帶來了更大自重，讓底盤等關於動態表現部件要比燃油車的要求還高不少，因為懸掛系統面對更大自重帶來的負荷，操控性與舒適性都會有一定的犧牲。從我們的測試來看，能夠更好應對不同行駛狀態的空氣懸掛，乃是電動車的絕配。

裝備空氣懸掛的特斯拉（不含Model 3）乃至蔚來出色的行駛質感，都在《新能源高考》評測體系中獲得更多的分數。比亞迪全新一代王朝系列平台，相比此前確實有著非常明顯的提升，動態表現也對得起其30萬元+的售價，只是定位與成本的先天因素，限制了它的上限。

[·飛翼門是Model X的加分項 唐EV成敗皆7座·]

關於空間，唐EV絕對當得起大空間SUV的名頭，並且與燃油版一樣提供7座版本。但「成也7座、敗也7座」，三種動力版本的唐的第三排都只是「有」的程度，而且為了給第三排座椅「擠」出位置，我們測試的5座版本還犧牲了一定的第二排空間。

反觀空間倒也沒有太大優勢的Model X，後排向上開啟的飛翼式車門不光吸引眼球，狹窄區域好開啟、方便上下車等實用、好用的優勢，讓此處亮點成為了決定分數的一大關鍵點。

[·製造工藝非特斯拉長項 配置只輸在了駕駛輔助·]

要說製造工藝，特斯拉車主在以往舉出了很多的「反例」，而唐EV乃至唐這個車系作為比亞迪的SUV旗艦，在工藝、用料方面確實下了不少功夫。一個與其價格並不符、另一個是品牌門面，讓這個與成本直接掛鉤的單項，唐EV只落後了區區2分。

配置方面，特斯拉品牌的科技屬性在Model X等產品中有著直觀的體現。而配置一直是自主品牌的強項，且我們測試的唐EV還是全系頂配，你能想到的主流功能並不遜色於 Model X，只在特斯拉行業領先的駕駛輔助系統上處於劣勢，倒也在情理之中。

[·特斯拉多媒體自成體系 比亞迪定製安卓的DiLink真好用·]

智能手機的普及，讓車載多媒體系統這個汽車智能化的實際載體，成為了不少消費者選車時非常看重的產品點，為此我們專門拿出了100分的權重。這方面，特斯拉很有遠見的電子架構，讓整車所有電子系統都能像手機一樣持續在線升級，確實完勝比亞迪後期要通過OBD「刷電腦」的方式。

而升級更多的還是看未來，與大家現在的實際用車關係沒那麼大，所以多媒體系統這個「單科」還是唐EV的一次完勝。比亞迪深度定製的安卓系統，讓 DiLink具備了真正意義上的汽車「網聯」，兼容300多萬款手機APP或是在線功能。在功能豐富程度上，這套系統的對手已經不是汽車品牌了，而是IT品牌。

[·全文總結·]正如文中的性價比「分子」和「分母」之說，比亞迪唐EV以不到一半的價格，產品水平達到了特斯拉Model X的70%以上，那麼它就值得我們給出高分。而二者只相差1分，那麼它們是競爭對手嗎？答案當然是否定的。我們做的是將產品表現以分數的形式直觀展現，而真正選車中還有很多如品牌價值、因人而異的外觀好不好看等無法量化的因素。《新能源高考》評測體系只是一個考量車型的工具，一把衡量所有新能源車的「尺子」，供大家作為選車時的參考，您要覺得有幫助，敬請持續關注搜狐汽車·E電園後續的更多測試內容。
