提拉過後的Model 3？7座的Model Y究竟有無優勢？

2019-12-04 1,779條評論 3,513次閱讀 707人點贊

對於特斯拉，要說它現在是全球最出名的汽車品牌也不太過分。這個新能源汽車的獨角獸到目前，前前後後已經推出過不少車型了，涉及轎車、轎跑、SUV、甚至還有皮卡等，並且每一輛都稱得上是熱度爆表。所以，特斯拉也開始布局更多的產品層面，這就有了接下來要說到的7座中型SUV Model Y。

Model Y是特斯拉發布的第五款車型，將於明年第一季度正式開始交付，比預期要提前半年，在這點上還是很佩服特斯拉的效率。而在臨近交付之前，國外有人拍到了一張該車內的座椅布局照片，重點當然是在第三排座椅。

三排坐人必須犧牲二排空間

雖然在產品發布會上就有內飾的官圖放出，並且大家對Model Y的第三排空間也沒有太大的期望，畢竟車身比Model 3大不了多少。不過當實車照片曝光的時候還是讓圈哥驚訝了一番。也許是由於採用了隱藏式的座椅頭枕（在背椅後上方，需提拉抽出），導致看上去完全不像是座椅的大小。但即使是這樣，Model Y的第三排座椅空間似乎也太小了，如果保持第二排正常坐姿，那麼第三排前後的距離肯定是容不下一個正常成年人的。既然坐不了成年人，那小孩也不可能單獨坐在第三排，除非讓第二排座椅相當靠前，不過這樣的話對於整車乘坐者可是一個相當壓抑的過程。

溜背還要第三排？頭枕竟需隱藏？

大家都知道，特斯拉的所有車型對於溜背造型格外偏愛，每一款車都少不了大溜背。Model Y的整體造型其實就像是把Model 3向上提了一下，但是卻沒有顯得很高，反而是看著更短了些。溜背造型繼續延續，不過畢竟向上提拉過，第二排頭部空間還是不用擔心的，但是第三排就難了。

不得不說，馬斯克真的是相當聰明，在發布Model Y的車內圖片時，故意放了張削頂的照片，顯得頭部空間還很充裕。現在看到實車照片，《車壹圈》更加確定那是有意為之了。

隱藏式的頭枕的確可以在座椅放倒時形成一個全平的地台，對儲物方面有很好的幫助。不過頭枕直接露出來估計會頂到車頂或許也是一個重要原因。《車壹圈》認為這是綜合考慮過後為了節約空間的設計，並不是為了增加科技前衛感的小心思。

《車壹圈》觀點

根據特斯拉中國官網公布，Model Y在中國的預估售價為44.4-53.5萬元，還有更入門的標準續航版暫未公布預售價。就價格來看，該車比Model 3可是要貴了不少，而除了車身更高，它最大的優勢就是多了第三排兩個座椅，而這個座椅的選裝費用是2.78萬元。全國排名前十的《車壹圈》認為，對於喜歡SUV車型的朋友，加裝第三排座椅其實沒有必要，多一個大後備箱可能更實用；而對於真正有7座需求的朋友們，Model Y可能缺少了實用性。所以圈哥對這輛車的產品力並不看好，不過特斯拉的車型是逃不過「真香」定律的，即使不看好，但圈哥還是相信它的銷量會不錯。
