王思聰被取消限制令，有博主發問“所以，老婆們你們還在嗎？”

2019-12-25 367條評論 1,710次閱讀 672人點贊

12月24日，有媒體曝出，王思聰的三條限制令已經撤銷，之前王思聰被凍結財產等消息至此也煙消雲散，成為了過去。

為何之前還鬧得沸沸揚揚的王思聰限制令會撤得如此之快，按照王思聰的家庭條件來說，不足以出現這樣的情況才對，但王思聰能夠迅速解決問題，不禁讓人想到了王思聰的父親王健林曾經說過的一句話，從裡面我們或許能夠找到一些答案。

之前在參加節目採訪的時候，王健林曾訂過一個小目標，「比如說先掙它一個億」，在別人看來是誇下海口，王健林卻很快就完成了目標，後來還超出了許多，掙了好幾個億。

不得不承認王健林的這番話很霸氣，和王思聰的「壕」比起來，高級了不止一個檔次，但也正是王健林在說出小目標時所散發出的那種自信，讓王思聰在出現限制令消息的時候，震驚了一片網友，有這樣的老爸，王思聰還不是分分鐘就能夠解決問題，何至於拖了這麼長的時間。

王健林的話透露著王思聰有能夠解決問題的底氣，無論自己是否能有力支付欠款，王健林始終都會是王思聰背後的大樹，關鍵時候都能夠幫一把，但直到王思聰被撤銷三條限制令後，並沒有出現王健林幫忙的聲音出現，這也就解釋了王思聰為何會拖了這麼久，才將事情完全解決掉。

王思聰雖然這些年在事業上取得了一些成績，但和父親王健林相比，還是有著非常大的差距，在解決問題的時候，往往也就會需要花費更多的時間，而每個人處理問題的方法也不盡相同，在李嘉誠的兒子李澤楷出現資金問題的時候，李嘉誠立馬出手幫兒子度過了危機，而顯然王健林就沒有那麼做，更多的是要看王思聰個人的能力，如果無路可走，相信王健林到那時才會出手幫忙。

很難說李嘉誠和王健林誰對誰錯，但在最終的結果上，都是將問題解決掉了，只是兩個人看待問題的出發點不同，才會有了不一樣的手段出現。

此次王思聰把欠款支付後，不少網友紛紛表示，王思聰終於可以安穩的過年了，不過就在此前，王思聰還被拍到和好友一起現身日本，不僅吃了拉麵，還一起滑雪，玩得不亦樂乎，從來身邊不缺乏女伴的王思聰，身邊依舊有美女同行，對於消費限制令，王思聰似乎也從來都沒有擔心過，這也能側面看出，王思聰有解決問題的能力，這才是讓自己玩得開心的關鍵所在。

而在王思聰的限制令出現時，網上就有了許多調侃的話語出現，多半都和王思聰身邊應接不暇的美女有關，此次限制消費令撤銷後，就有博主發問「所以，老婆們你們還在嗎？」有網友也玩笑口吻的回應「從未離開」，還放了一張狗頭的表情包，可以說很有趣味了。

從限制令出現，到支付欠款取消限制令，不管中途有著怎樣曲折的過程，好在問題都已經解決，也證明了王思聰是娛樂圈頂流的地位從未改變過啊。
