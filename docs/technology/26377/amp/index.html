5G、區塊鏈、數字經濟，專家眼中新技術如何成為經濟增長點？

天天要聞
2019-12-23

12月21日，由亞太經濟創新研究院、AEIC學術交流中心聯合主辦的「數字經濟：粵港澳大灣區高質量發展新引擎」2019國際科技創新學術交流大會暨廣東亞太創新經濟研究院第二屆粵港澳大灣區科技創新論壇在廣州舉行。

會議現場，同濟大學軟體學院院長、國際歐亞科學院院士趙生捷，中國信息通信研究院廣州分院副院長王洪嶺，廣州無線電集團黨委副書記、總經理、副董事長黃躍珍，國家信息與標準化委員會委員、創鏈數據董事長許洪波等專家學者，圍繞人工智慧、5G、大數據、區塊鏈等領域暢談，共同解讀數字經濟的現狀和探討發展趨勢。

「數字網路生態」背後凝聚的是人才

根據《中國數字經濟發展與就業白皮書（2019年）》顯示，我國數字經濟持續快速發展，2018年數字經濟規模達到31.3萬億元，增長20.9%，佔GDP比重為34.8%，尤其是粵港澳大灣區數字經濟發展居全國前列。

在會上，亞太創新經濟研究院理事長李志堅首先發言。他認為，發展數字經濟，不僅要關注基礎設施「硬聯通」，還應關注人文生態「軟聯通」。

「發展數字經濟，勢必依託網路生態的構建。所謂網路生態，即連點成線、連線成面，在更大範圍內促進溝通、交流與共享。」在李志堅看來，城市構成城市群、城市群構成大灣區，進而輻射全國、聯通世界，是網路生態的構建，個人組成團體，又通過平台與其他團體的人聚合、碰撞、交流，進而讓知識和智慧傳播、影響，在更廣範圍產生更大能量。

而打造「數字網路生態」背後凝聚的是人才。同濟大學軟體學院院長、國際歐亞科學院院士趙生捷以「數字經濟產業與創新創業人才培養與實踐」為題發表主旨演講。他以同濟大學為例，介紹了該校在軟體工程設計方面的人才培養模式。

「我們目前的團隊有軟體工程理論與技術、軟體人工智慧、大數據系統與軟體、軟體與網路通信、軟體與數字媒體技術，這是我們規劃的2017年到2021年的團隊，這也是根據我們自身的特點進行規劃的。」趙生捷表示。

5G能有效激發產業創造力

目前，在數字經濟領域，5G已經進入應用落地階段，即將迎來爆髮式增長。「在未來五年中，通過5G與相關產業的結合，預計將帶動15萬億的增長。」王洪嶺說。

據測算，去年我國的數字經濟規模是35萬億元。「如果5G能夠對數字經濟產生15萬億元的增長，也就相當於我們現有數字經濟總量40%的水平了。」王洪嶺表示，5G對數字經濟的帶動作用非常強，關鍵在於其能有效激發產業創造力。

5G有哪些應用場景？王洪嶺歸結為三大類：「一是增強移動寬頻，這是比較成熟的；二是高可靠低時延，正在逐步成熟；三是海量信息通信，這也是在逐步成熟的。其他信息技術如邊緣計算、人工智慧，與5G組合在一起，才能夠實現5G向工業領域的拓展。」

「廣州擁有實力雄厚的製造業基礎，最近剛剛和深圳、汕頭一道成為全省首批三個5G產業園。在工業製造、城市管理以及民生領域，都將成為5G在廣州的主要應用場景。」王洪嶺特別指出，廣州將城市路面上的智慧燈桿作為5G基站新載體，不僅能有效鋪開5G信號覆蓋面，還能將5G技術與智慧城市建設相結合，創造了許多全新的應用場景。

「可以預見的是，2020年廣州5G無論是在上中下游產業，還是應用創新領域，都將迎來許多突破，非常值得期待。」王洪嶺說。

區塊鏈將刺激零售業新活力

在數字時代，區塊鏈成為時下最熱門的技術領域。在許洪波看來，區塊鏈作為「組合式創新」，將給商業信用體系帶來顛覆式變革。

「舉個例子，我們希望在智慧出行和智慧停車場業務方面，與金融機構包括銀行開展合作。但是停車場經營方提供給銀行的經營數據是否可信？租賃物業無法提供資產做質押，這顯然在傳統信用體系中，是無法從金融機構獲得貸款。」許洪波表示，通過全部數據「上鏈」，大量中小企業特別是商貿類零售企業，能憑藉區塊鏈技術獲得授信融資，徹底解決融資難。

「中小企業得到了授信融資，發展活力就能極大激發，對於廣州這座『千年商都』而言，是激發老城市新活力的關鍵所在。」許洪波表示，目前區塊鏈技術迫切需要完善「合規性」，「一方面線上線下有關區塊鏈應用的法律法規有待統一併完善，另一方面需要界定區塊鏈應用底線，做到『法無禁止即可為，法無授權即禁止』，這樣才能極大地促進創新應用的到來。」

2G時代的簡訊、3G時代的手機QQ，4G時代的微信、抖音、移動支付等APP；互聯網領域，則有信息互聯網的郵件、瀏覽器，商業互聯網時代的阿里巴巴、亞馬遜，社交互聯網時代的Facebook、微博等場景更迭……在與會專家看來，5G、區塊鏈等全新技術尚未迎來「殺手級」應用。

不過，「場景驅動」仍是新技術誕生應用的關鍵所在。「對於粵港澳大灣區而言，最重要的場景就是先進位造業。大灣區的崛起必須依靠先進位造業，數字經濟時代的到來會推動製造業生產組織的變革，帶動大灣區先進位造業高質量發展的同時，也是孕育新技術、新應用的沃土。」亞太創新經濟研究院院長劉江華說。
