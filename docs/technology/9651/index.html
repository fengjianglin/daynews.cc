12月10日主流加密貨幣價格分析：比特幣，以太幣，瑞波，EOS

2019-12-10 768條評論 3,783次閱讀 625人點贊

價格分析

據報道，中國可能會在今年年底之前開始試行央行數字貨幣（CBDC）。文章指出，預期的試點項目「將退出中央銀行系統，並進入運輸，教育和醫療等實際服務場景」，這些最終服務將覆蓋最終用戶。

該項目將由中國中央銀行監督，包括四家主要銀行和大型公司，例如中國電信。這是重要的一步，因為它將為中國提供領先於其他發達經濟體的重要先機。

每日加密貨幣市場表現

今天，Bakkt 在ICE期貨新加坡交易所推出了以現金結算的比特幣期貨，並且是第一個CFTC監管的比特幣期貨合約期權。隨著這些產品的推出，Bakkt的目標是滿足亞洲和美國機構投資者不斷增長的需求。

其他主要的加密貨幣交易所也在擴展其產品，以滿足不斷增長的需求。OKEx還宣布推出將於12月27日開始交易的加密貨幣期權。有了這些產品，機構參與者就可以在加密貨幣領域進行投資。

當前，大多數主要的加密貨幣都是有範圍的，一旦這一階段過去，就可能出現一個明確的方向。讓我們看看圖表上的關鍵水平，這些水平將標誌著新的上升趨勢或下降趨勢的開始。

BTC /美元

比特幣（BTC）面臨接近7,856.76美元的強勁阻力。這表明熊們不打架就不太可能扔毛巾。他們將嘗試將價格下調至即時支撐位7,000美元。

如果該支撐跌破，將會對近期低點6,512.01美元進行重新測試。跌破該水平將是一個巨大的負面因素，可將價格拖至$ 5,533.90。但是，我們認為這種情況發生的可能性很小。

BTC USD每日圖表

儘管在過去四天多頭未能擴大規模並維持在20天均線上方，但積極的跡象表明多頭並沒有放棄太多。我們預計買家將再次嘗試將價格推高至20日均線和上方阻力7,856.76美元上方。

如果成功的話，很可能反彈至下降趨勢線。這是一個主要阻力，但如果突破，將標誌著新的上升趨勢的開始。因此，我們保留較早分析中建議的購買。

ETH /美元

多頭在151.829美元的上方阻力處面臨強勁阻力。如果以太（ETH）從該水平下跌，它可能跌至140美元，然後跌至131.484美元。由於移動平均線都向下傾斜並且RSI處於負數區域，因此空頭具有優勢。跌破131.484美元將是一個巨大的負數，因為下降趨勢將繼續。

ETH USD日線圖

從好的方面來看，如果多頭可以將價格推高至151.829美元以上，則下一個值得關注的水平是157.50美元。我們預計以太坊在突破並維持在157.50美元上方後會繼續上漲。

短期交易者可以在收盤價（UTC時間）上方$ 157.50上方建立多頭頭寸。第一個目標是反彈至173.841美元，並高於其197.750美元。

XRP / USD

空頭正在積極捍衛0.2326美元的阻力位，這表明缺乏更高水平的買主。但是，如果XRP在較小的上升趨勢線獲得支撐，我們預計多頭將再次嘗試突破0.2326美元。

XRP USD每日圖表

我們在製作過程中發現上升三角形圖案的第一個跡象，它將在突破並在0.2326美元上方收盤（UTC時間）時完成。這種看漲設置的目標目標是0.26479美元，接近50天均線。雖然在0.24508美元有一個阻力位，但我們預計它將被突破。

與我們的假設相反，如果價格跌破上升趨勢線，則下一個支撐位是0.20946美元，如果也未能提供支撐，則有可能跌至0.20041美元。如果XRP / USD貨幣對跌破0.20041美元，它將恢復下跌趨勢。

BCH / USD

比特幣現金（BCH）仍然停留在203.36美元至227.01美元之間。這顯示了買賣雙方之間的平衡。從該範圍很難預測突破的方向，因此，我們不建議在當前水平進行交易。

BCH USD每日圖表

突破227.01美元將是一個積極信號，因為這表明多頭已經壓倒了空頭。高於227.01美元，則可能反彈至261.50美元，然後升至306.78美元。因此，我們保留先前分析中給出的購買建議。

然而，如果BCH / USD貨幣對跌破203.36美元，這將表明空頭的優勢。低於該水平，可能會重新測試192.52美元。如果該支撐也破裂，下降趨勢將恢復。

LTC / USD

自12月5日以來，萊特幣（LTC）的日內波動幅度已經縮小，這表明多頭和空頭都在安全地支付這筆費用，並且沒有大筆賭注。低波動時期不太可能持續很長時間。我們預計波動將在未來幾天內回升。

LTC USD每日圖表

如果波幅擴大至下行並令LTC / USD對跌至42.0599美元以下，則下降趨勢將繼續。下行的下一個支撐位是36美元。

但是，如果該貨幣對從支撐位反彈或從當前水平回升並突破上方阻力位50美元，則可能暗示底部42.0599美元。我們將等待價格維持在50美元以上，然後再提議交易。

EOS / USD

多頭未能將EOS推高至20天均線上方的失敗吸引了空頭的拋售。他們將試圖使價格跌破即時支撐位2.5804美元。如果成功，則有可能重新測試關鍵支撐位2.4001美元。

EOS USD日線圖

相反，如果多頭買入逢低買單，並推動EOS / USD突破上方阻力位$ 2.8695，則有可能升至下降趨勢線$ 3.43，然後升至$ 3.69。在價格維持在四個小時的2.8695美元上方之後，短期交易者可以背負這一可能的上漲行情。

BNB /美元

幣安幣（BNB）在過去三天內形成了十字星燭台圖案。這顯示了多頭之間的猶豫不決，有關下一方向的行動。突破上方阻力位16.50美元，將表明多頭又回到了遊戲中。

BNB USD每日圖表

高於16.50美元，我們預計將反彈至21.8美元。儘管50天SMA可能會提供一些阻力，但我們預計它將被突破。在我們建議交易之前，我們將等待價格維持在16.50美元上方。

與我們的假設相反，如果價格跌破支撐位14.2555美元，下降趨勢將恢復。這將是一個巨大的負面影響，因為它將價格拖至11.30美元的下一個支撐位。

BSV / USD

比特幣SV（BSV）過去幾天一直在接近支撐位92.693美元。這表明多頭缺乏在這些水平上進行購買的緊迫性，因為他們不確定是否已經觸底。

BSV USD每日圖表

下滑的20天EMA和RSI在負區域表明空頭佔了上風。如果價格跌破支撐位92.693美元，則可能跌至關鍵支撐位78.506美元。我們預計多頭會對此水平進行有力的防禦。

與我們的假設相反，如果BSV / USD對從當前水平反彈並突破下降趨勢線，則可能升至113.96美元。價格規模擴大後，我們將轉為正面，並維持在50天均線1173美元上方。

XLM / USD

在過去六天里，Stellar（XLM）的交易價格接近其上方阻力位0.056美元。這是一個積極的信號，因為它表明賣方未能將價格下沉至支撐位0.051014美元。

XLM USD每日圖表

如果多頭成功地將價格推高至0.056美元以上，將有可能升至0.06美元。我們預計突破0.06美元將吸引買家。因此，我們建議交易者在收盤價（UTC時間）以上$ 0.06買入，並將止損設在$ 0.051以下。從好的方面來看，要牢記的目標是0.071美元，高於0.088708美元。

如果價格未能突破上行並且空頭將XLM / USD價格跌至關鍵支撐位0.051014美元，我們的看漲觀點將失效。低於此水平，可能會跌至0.041748美元。

XTZ / USD

在過去的幾天里，Tezos（XTZ）迅速攀升，已成為市值排名第十的加密貨幣。雖然大多數主要的加密貨幣要麼停留在一定範圍內，要麼在逐步上升，但Tezos一直在果斷地邁向上升。

XTZ USD每日圖表

自11月初以來，兩個移動均線都在傾斜，價格一直高於移動均線。這表明多頭處於主導地位。在1.60美元處有一個小阻力位，可能升至1.85美元。我們預計1.85美元會有一個堅挺的阻力，但如果多頭突破該水平，XTZ / USD將非常看漲。

但是，如果多頭未能將價格推高至1.85美元上方，則價格可能會再維持幾天的區間波動。儘管看漲，但我們找不到風險回報率高的交易，因此，我們不建議在這些水平上進行交易。
