蘋果前首席設計官艾維等捐50萬美元，幫英國植果樹

2019-12-25 663條評論 4,486次閱讀 2900人點贊

科技訊 12月25日消息，據外媒報道， 蘋果公司前首席設計官喬尼·艾維(Jony Ive)日前向英國植樹活動Be A Tree Angel捐贈了12.9萬美元善款，以幫助英國學校建設果園。


圖：蘋果前首席設計官喬尼·艾維

除了艾維，英國餐館老闆理查德·凱倫(Richard Ccare)、著名商人蘇格勛爵(Lord Sugar)以及一位匿名捐贈者也提供了同等金額的捐款。這四個人提供近50萬美元資金幫助英國各地4000所學校創建果園，並提供維護果園的後續服務。

艾維表示：「我喜歡樹，這些年來，我爬過樹，吃過各種樹上接的水果，也從樹上掉下來過，我種植了儘可能多的樹。」這位設計大師還稱，樹木也「對學習和創造有深遠的好處」。

根據艾維的說法，樹木是蘋果總部「蘋果園」設計的概念基礎，「公園和果園」是其創造的關鍵，「能夠為一個如此貼近我內心的項目做出貢獻，真是太棒了！」

Be A Tree Angel活動由《每日郵報》與英國樹木委員會（The Tree Council）合作推動，他們都在接受讀者和企業的捐贈。到目前為止，這項努力已經籌集到足夠種植3.5萬棵樹的資金。

已故蘋果聯合創始人史蒂夫·喬布斯(Steve Jobs)曾經的夢想就是，模仿他成長的加州來設計蘋果園的景觀。在2013年公開的一段演示視頻中，建築師諾曼·福斯特(Norman Foster)表示，喬布斯認為加州是「美國的水果碗」，希望新園區能反映出他小時候記憶中的風景。

除了點綴在蘋果園中的抗旱樹種，蘋果在其主環結構及其周圍還種植了成片的蘋果樹、杏樹、櫻桃樹、柿子樹和李子樹等。（小小）
