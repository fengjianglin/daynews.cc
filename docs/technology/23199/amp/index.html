三星退出中國製造:將清算崑山三星電機 員工補償N+5

天天要聞
2019-12-18

對於三星來說，他們關閉在中國的工廠反應了在戰略上的調整，其對產業鏈進行重新匹配，首先就是將密集型工廠轉移到用工成本更低的國家，然後將更多中低端手機外包出去，也就是所謂的ODM生產，這樣的好處是，降低成本，確保產品價格有足夠的競爭力。


12月17日消息，據最新消息稱，三星手機正在徹底退出「中國製造」，其後續動作可能是對位於蘇州崑山的三星電機清算，而工廠員工補償高達N+5。

位於崑山的三星電機公司2009年註冊成立，2010年6月份正式量產HDI（智能手機主板），曾經是三星HDI產品的主要生產基地。不過隨著中國公司加入HDI智能手機主板市場，低價競爭導致三星電機的崑山工廠運營困難，此前已經連續5年虧損，被傳關廠清算的消息也早在網路上流傳了。

在三星電機之前，LG旗下的LG Innotek已經宣布退出HDI市場，將重心轉移到高價值的半導體業務上去，而三星電機退出HDI之後也會加強更高端的半導體封裝基板和硬鉛印刷電路板（RF/PCB）領域，提高盈利能力。

有行業知情人士表示，三星手機關閉在中國市場的一系列工廠是情理之中的事情，他們要提升產品的綜合實力，就必須要把相應的工廠轉移到用工成本更低的國家，同時極大手機ODM的外包力度。

已停止手機在華生產

三星電子已經於9月底關閉了惠州的工廠/研發中心（SEHZ），該工廠於1992年正式投產，是三星在中國的主要生產基地，距今已存在了近30年。在這之前，他們還相繼關閉了天津、深圳的工廠。

目前三星手機的生產都是來自印度。去年三星在印度擴建了全球最大的智能手機工廠，擴建完成後，三星新工廠等產量預計將增加一倍，達到近1.2億部，同時三星在越南各地亦設立了8個生產基地，主要生產手機與電子零件等。

對於停止手機在中國生產這件事，三星還曾公開回應稱，這並不是要退出中國市場，而是優化他們產業鏈的行為，其還會繼續在中國智能手機市場深耕，特別是要抓緊已經到來的5G時代機遇。

開始在中國市場戰略調整

關閉在中國工廠的另外一個因素，就是三星手機目前在中國市場表現慘淡。從之前不少調研機構給出的統計數據顯示，三星智能手機單季度在中國市場的出貨量只是在70-80萬台左右，而他們的市場份額在1%左右徘徊，如此成績已經不足以支撐他們在當地的工廠運轉。

如果復盤三星手機在中國市場的衰落，主要是兩個時間點：一個是2014年以後，中低端機型受到國產手機品牌的衝擊，這導致市場份額大幅下滑，另外一個時間就是，Galaxy Note 7的電池爆炸門，這件事讓三星中國承受重創，而通過這兩次轉折點，華為都把握住了機會，搶佔了他們絕大多數市場份額，一躍成為目前中國智能手機市場的領頭羊，並且正在全力向三星手機世界第一的地位發起衝擊。

對於三星來說，他們關閉在中國的工廠反應了在戰略上的調整，其對產業鏈進行重新匹配，首先就是將密集型工廠轉移到用工成本更低的國家，然後將更多中低端手機外包出去，也就是所謂的ODM生產，這個模式在手機圈並不少見，比如小米旗下的紅米手機就是外包給聞泰等生產，這樣的好處是，降低成本，確保產品價格有足夠的競爭力。

據市場調研機構顯示，今年估計由ODM廠商生產的智能手機，在三星智能手機出貨量中的佔比將由2018年的3%增長至8%。具體做法是，三星只是給由ODM廠商生產的智能手機貼上自己的商標即可，這樣可以大大降低開發成本和人工成本。據產業鏈透露，明年三星將有至少6000萬部手機外包出去，而中國市場基本都是ODM機型。

作為全球最大的智能手機市場，三星手機顯然不會輕易退出中國手機市場，現在要看的就是，調整運營策略後，三星手機中國市場接下來的業績會不會有所改善了。
