先別著急買!特斯拉考慮明年將中國產Model 3降價20%

2019-12-18 910條評論 2,086次閱讀 160人點贊

作者：遠洋

IT之家12月18日消息 據彭博社報道，知情人士透露，特斯拉公司正在考慮明年將其中國製造的Model 3轎車的價格下調20%或更多，在中國這一全球最大的電動汽車市場放緩之際，預計此舉將會吸引買家。


知情人士說，特斯拉將通過使用更多的本地零件降低成本，這樣就能進口更少的零件並避免徵收關稅。知情人士還稱，這些汽車將在特斯拉上海新工廠生產，起價為355,800元人民幣，將從2020年下半年開始降低。

隨著電動汽車製造商在前所未有的低迷中掙扎，競爭日趨白熱化。在中國政府縮減補貼之後，電動汽車在中國的銷量已經連續下降了幾個月。

知情人士說，降價的規模和時機可能會根據市場情況而改變。特斯拉在中國的代表拒絕置評。

大部分的成本削減將取決於電池，電池通常是電動汽車最昂貴的部分。知情人士在八月份表示，特斯拉目前完全依靠日本松下公司的電池，但隨著其同意從位於上海西北約320公里的LG Chem Ltd.工廠購買電池，這種情況正在改變。知情人士上個月表示，特斯拉還達成了一項協議，即在明年年初之前使用中國當代安培科技有限公司的電池。

據央視財經報道，走訪上海多家特斯拉體驗中心門店了解到，明年1月起，特斯拉將在中國上調Model 3進口車的價格，但是在1月份之前交兩萬元的意向金，就可以鎖定住現在的價格。
