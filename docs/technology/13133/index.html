11月汽車銷量降幅收窄，新能源汽車或全年負增長

2019-12-11 3,675條評論 4,770次閱讀 2160人點贊

財經網訊 11月汽車銷量數據出爐，同比降幅延續收窄，新能源汽車全年銷量可能出現負增長。

12月10日，中國汽車工業協會發布了「2019年11月汽車工業經濟運行情況」，其中乘用車銷量承接10月份的走勢，繼續有所回升，但整體依然小幅收跌。當月銷量為205.7萬輛，環比增長了6.7%，同比跌幅為5.4%，相比10月份收窄了0.4個百分點。

新能源汽車則延續了2019年7月以來的頹勢，11月銷量為9.5萬輛，環比增長了27.1%，同比跌幅43.7%，比10月份縮減了1.9個百分點。

2019年1-11月，乘用車累計銷量為1923.1萬輛，同比跌幅10.5%，同期新能源汽車累計銷量為104.3萬輛，同比微增了1.3%。

這意味著如果新能源汽車12月份的銷量延續11月超過四成的跌幅，很可能「抹掉」之前累計的正向增長，掉頭向下，全年變成負增長。

對於新能源汽車進入下半年以來增長乏力的原因，中汽協曾經在10月份表示跟新能源補貼大幅下降、市場整體回升幅度有限有關，而且認為在短期內這些因素不會消失，因此整個行業仍將承壓。

根據中國汽車工程學會的推演，在整車購置成本方面，補貼退出後到2030年，新能源車可以實現與同級別傳統燃油車整車成本持平，結合新能源汽車在使用成本方面的優勢，以及可能推出的稅收和路權方面的政策支持，預計2025年即可實現經濟性優勢。

另外，恆大研究院發布的《中國新能源汽車發展報告：2019》顯示，整車企業也可以通過產業鏈上下游分攤降本、技術進步降本、整車輕量化帶來的結構降本以及提高產能的規模降本等，推動新能源汽車購置成本的進一步降低，提振銷量。
