消費電子時代到來！光學鏡頭也要重點關注

2019-12-18 475條評論 1,810次閱讀 63人點贊

近期，消費電子的炒作浪潮中，光學鏡頭是炒作方向之一，韋爾股份是總龍頭。最近幾天炒作的ToF技術，也是用於3D攝像頭的。除了ToF之外，還有Vcsel。
目前在高端手機市場已開始越來越多配置前後VCSEL（改善拍照質量與提供AR應用），2019年主要是華為採用，2020年主要是Apple與華為採用。一旦智能手機市場被撬動，VCSEL的需求量將迅速打開，這就是未來數年VCSEL強勁增長的巨大推動力。
建議留意：
乾照光電（ 300102 ）：2018年出資近16億元投建VCSEL等半導體研發生產項目。
海特高新（ 002023 ）：已經完成三吋和六吋的Vcsel產品工藝開發。
國星光電（ 002449 ）：有VCSEL封裝器件產品。
水晶光電（ 002273 ）：作為窄帶濾光片的全球大型供應商，為蘋果公司產品的3Dsensing功能提供重要光學配件。
福晶科技（ 002222 ）：是全球激光器光學元器件隱形冠軍，2015年宣布自主研發的VCSEL陣列晶元將投入商用。
匯頂科技（ 603160 ）：3Dsensing方面在VCSEL、圖像處理、DOE等晶元都有設計能力。
匯川技術（ 300124 ）：間接持股新亮智能8.38%的股權，新亮智能的VCSEL激光晶元和激光測距感測器曾在第七屆中國電子信息博覽會

關注 002023 海特高新：目前已完成包括砷化鎵、氮化鎵、碳化硅及磷化銦在內的6項工藝產品的開發，可支持製造功率放大器、混頻器、低噪音放大器、開關、光電探測器、激光 器、電力電子等產品，產品廣泛應用於5G移動通信、人工智慧、雷達、汽車電子等領域。截止目前公司部分產品已實現量產，服務客戶數和訂單持續增加。
