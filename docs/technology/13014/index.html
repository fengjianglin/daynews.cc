2019年年度最新編程語言是Java、Python、C、C#？到底花落誰家

2019-12-12 1,128條評論 3,568次閱讀 1703人點贊

TIOBE 近日公布了 12 月份的編程語言指數排行榜，前三分別是：Java、C 語言和 Python。

此外，將在下個月宣布 「2019 年度編程語言」。目前有 4 個候選語言：Java (+1.3%)， C (+1.8%)， Python(+1.9%) 和 C#(+1.4%)。這 4 種語言都在 Top 5。只有 C++ 在 2019 年失去了一些排名分。

Python 是最強勁的候選。雖然它已經是 2018 年度編程語言，但其熱度仍在不斷增長。TIOBE 認為，這主要是因為世界上缺少程序員，而且與其他語言相比，Python 更容易學。

由於物聯網的興起，C公司的業績也不錯。為什麼Java和c#表現良好還不清楚，但這將是 C# 第一次成為年度編程語言。讓我們看看下個月將會發生什麼!
