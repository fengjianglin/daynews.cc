俄羅斯最大暗網正通過ICO籌集資金以支持全球擴張

免責聲明：本文旨在傳遞更多市場信息，不構成任何投資建議。文章僅代表作者觀點，不代表火星財經官方立場。

來源：區塊鏈鉛筆Blockchain

作者：Anna Baydakova 編譯：Penny

Hydra是服務於俄羅斯和鄰國的最大暗網市場，稱其正尋求通過代幣銷售籌集1.46億美元，為全球擴張提供資金。根據該網站上的投資備忘錄，該計劃於12月16日啟動，如果沒有特殊的匿名瀏覽器，將很難訪問。

代幣每個售價為100美元。該備忘錄說，一個由100個代幣組成的「package」將為買家提供0.0093.333％的Hydra利潤份額。它沒有提及代幣將在哪個區塊鏈（如果有）上運行。

考慮到其業務的非法性質， Hydra的代幣發行可能是史無前例的。俄羅斯新聞媒體Forklog警告說，出售代幣可能是騙局，這是普遍現象在暗網市場發生。

Hydra背後的團隊宣稱他們精通安全性和攻擊彈性，它想向世界其他地區介紹其「非接觸式」交易違禁品方法，買方，賣方和快遞商從未見過這種方式。

「這將開啟西方的新時代。規模難以想像。」

ICO將為名為Eternos的新服務的開發提供資金，Eternos被描述為全球暗網市場，結合了加密的Messenger，加密貨幣交易所，類似Tor的匿名瀏覽器，基於AI的爭議解決和非處方葯（OTC）市場。

「沒有KYC，所有內容都是匿名的，而且是最高級別的。公告說，我們不是TON。這是對Telegram消息傳遞應用程序的區塊鏈項目的挖掘，該項目要求其代幣的購買者進行了解您的客戶的篩選。

Hydra計劃發行1,470,000個代幣，佔總額的49％，並承諾如果買家持有超過100個，每個代幣每月將獲得500美元的比特幣分紅。

這意味著Hydra運營商期望Eternos每月產生超過1500萬美元的利潤。公告說：「這基於Hydra的增長指標。」
