年底了還不想買手機再堅持一下吧！明年初即將發布的最強旗艦手機

2019-12-18 1,287條評論 4,359次閱讀 1883人點贊

三星S11

三星Galaxy S11正面將會採用時下熱門的瀑布屏設計，細節上相比現有機型要更為激進一些，曲面可顯示部分幾乎延展到了底部邊框。三星Galaxy S11將採用20：9的 2K+ Dynamic AMOLED屏幕；支持120Hz高刷新率；一億像素主攝，支持5倍混合光學變焦，最高支持100倍變焦；支持45W快充。三星 Galaxy S11+採用的是6.9英寸120HZ屏幕，正面採用了和三星Note 10+一樣的單挖孔屏設計，比之前的雙挖孔屏漂亮太多了。將會採用驍龍864/Exynos 990晶元，全面支持5G網路，三星Galaxy S11將會是一款集大成的作品，對比前代將會有質的提升。

三星Galaxy S11搭載A77+G77 架構處理器以及7nm EUV 5G晶元，採用 LPDDR5 RAM + UFS 3.0 儲存空間，90Hz 或 120Hz 刷新率的屏幕，拍照方面，將使用即比一般手機更大的CMOS 感光元件 + 5 倍光學變焦鏡頭，支持 10 倍無損變焦拍攝，並運行 One UI 2.1 系統。全球最好的顯示屏、頂級的研發實力、強悍的攝影能力，三星手機能夠做到全球第一不是沒有原因的。這些方面，都是國產品牌需要學習的，希望能有一天，能看到國產企業超越三星。

華為P40 Pro

作為華為P系列中即將到來的新成員，華為P40系列的相關信息目前已經在網上曝光了出來。首先在外觀設計上，華為P40 Pro將採用雙挖孔曲面屏設計，手機一體化視覺感極強，機身正面的顏值可以說是相當出眾。華為P40 Pro將配備6.5英寸OLED瀑布屏，支持120Hz屏幕刷新率，前置雙打孔攝像頭，後置五攝，具體參數規格暫不知，不過會保留一顆ToF鏡頭。搭載麒麟990 5G SoC板上釘釘，相機配置有所提升，主攝不再是IMX600，而是向索尼定製的6400萬像素IMX700，RYYB像素排列，然後是2000萬像素廣角鏡頭、1200萬像素潛望式長焦鏡頭、微距、ToF。

華為P40 Pro的主要升級點在於拍照、屏幕等方面。此外，還有爆料稱華為P40 Pro將使用石墨烯電池技術，在電池體積不變的情況下大大提高續航能力，還能增加充電速度。華為P40 Pro內置5500mAh電池，支持50W快充。華為P40 Pro將擁有磨砂+鏡面雙材質的搭配，這一後蓋配色在華為此前發布的P30 Pro特別版上已經出現。因此，這次P40 Pro也算是延續了P30 Pro後蓋的大部分設計元素。值得一提的是，結合此前的爆料消息，這塊「挖孔瀑布屏」還將帶來120Hz的高刷新率。
