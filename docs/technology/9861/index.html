驍龍處理器中人工智慧引擎AIE的名字：是小米林斌幫高通取的

2019-12-09 1,055條評論 2,130次閱讀 1072人點贊

定於明天（12月10日）發布的Redmi K30新機，將全球首發搭載驍龍765G 5G晶元。盧偉冰介紹，驍龍765G 5G集成X52基帶，支持SA/NSA雙模，下行峰值3.7Gbps，同時採用了第五代高通AIE人工智慧引擎，擁有5TOPS的AI算力。

幸好中的「G」更是Gaming的簡稱，指代更強的圖形運算能力。

對於這顆晶元，小米副董事長林斌評價，「這款5G處理器還是很不錯的，不止支持NSA/SA 5G雙模。CPU/GPU/AI性能也非常好」。

他還爆料，AIE這個名字是去年CES我和高通產品經理吃飯時，替他們新的AI引擎起的名字，當時在驍龍845上首發，現在已經第五代了，AI算力也比845強了很多。

據可查資料，驍龍865的AI運算能力（Int8尺度）為15TOPs，是驍龍855（7TOPs）的兩倍多。此次的驍龍765G也幾乎追上了驍龍855，而且超越了845。
