「圖」微軟Ugly Sweater活動再次上線：今年主題是Windows XP

2019-12-12 62條評論 4,063次閱讀 908人點贊

去年微軟在推特上推出了「Ugly Sweater」活動，參與活動的用戶將有機會獲得微軟定製的Windows 95主題「Ugly Sweater」。微軟大約限量生產了100件毛衣送給部分忠實擁躉，由於該活動只送不賣因此有不少人在eBay等平台上高價收購。而今年這項活動再次到來，不過這次是已經壽終正寢的Windows XP，按這樣的節奏明年就應該是已經停止支持的Windows 7了。


同樣，WindowsXP主題的「Ugly Sweater」也是只送不賣，這款毛衣的正面是Windows的LOGO，剩餘部分變成了藍色。除了毛衣之外，今年幸運的用戶還能收到類似於原版Windows XP的包裝盒子，包裝盒子正面印有「Windows XP Pro-ho-ho-fessional」，右上角是開始菜單的LOGO，寫著「Soft-wear Pack 2」。



在盒子的背面羅列了Windows XP的諸多功能，例如易於安裝，用戶界面友好，出色的聯網性能，安全可靠的體系結構，量身定製的體驗以及向後兼容。在盒子內部，則是用薄紙包起來的，而這張薄紙是Windows XP的經典壁紙Bliss。
