今日推薦 | 肖磊：區塊鏈是下一個互聯網+美元的超級系統，美國的對手只有中國

小編：記得關注哦

來源：肖磊看市

最近在反覆思考一個很有意思的問題，僅僅在一百年前，人類還難以快速自如的橫跨各大洲（上個世紀20年代才有了能載運乘客的飛機），但一百年後的今天，地球變得如此之小，以至於各種文明不得不短兵相接，在有增無減的衝突和博弈中尋找新的共識和共生系統。

什麼樣的系統會獲得新的主導權，似乎沒有人能回答得清楚。

前一陣抽空把全球諸多歷史性的考古事件粗略的看了看，受到了一些啟發，所以嘗試著來寫點東西，看能否對人類運行所依賴的系統，尋找到一些端倪或進化規律，供大家參考討論。

我把人類的系統升級，梳理成了一個由關鍵片語成的主線，即：宗教、軍事、法律、美元、互聯網、區塊鏈六個遞進系統，這六個系統可以說組成了從新石器時代到當前，乃至未來的整個人類的系統演化過程。

什麼是系統，舉例來說，就類似於大家現在所用的微信，就是一個比較完整的小系統，你可以在這個系統內完成很多事情，這個系統的功能和規則就是你所有行動的邊界，也是你可以充分利用的工具。

如果站在全人類可依賴的秩序來說，需要的是能裹挾所有人、所有社會行為的大系統。

我們先來說說宗教。

我這裡說的宗教，實際上並不是指我們現在看到的佛教、基督教等，我這裡說的是，從新石器時代的考古當中，就已經存在的一種智人認識世界和影響權力分配，以及主導各種秩序的「神秘」力量。

在人類誕生至石器時代的很長時間裡，可以說始於巫術的神明主導型判罰系統，就是一切人類行動的最高儀式。

祭祀與戰爭是氏族制社會最大的大事，一般都是由最高首領兼祭司（巫師）主持。所以看新石器時代的考古發掘，只要發現墓主人同時是大祭司和武士，那麼很大的可能就是整個氏族的最高首領了。

後來幾大宗教陸續誕生，王權和神權的爭奪貫穿了數十個世紀，但神權依然是保證整個社會秩序化運行的重要底層系統。

軍事

隨著冷兵器時代的發展，人類生產工具的能力不斷提升，在戰勝自然環境方面有了很大的進步，靠武力可以解決很多問題，巫術和宗教的作用就逐步被削弱。族群可以在沒有高山、河流等天險的地方，建起足以抵抗野獸和其他族群攻擊的城堡、城市，人類才出現了文字等重大文明。

但只要有防禦的地方，就會有攻擊，戰爭實力成為新的社會規則，同時，戰爭也成了資源分配和保證社會秩序的主要手段。

為了支撐一次又一次的戰爭，人類發明了各種思想、工具，給科學技術的發展奠定了基礎，軍事運作邏輯取代巫術、「宗教」，成為主導世界秩序的新系統。

法律

軍事能力是一個動態演化的過程，隨著軍事能力的不管擴展，征服者需要管理的土地和人口成倍增加，如果沒有一個較為中立可信的規則，就很難有可持續的產出，征服者也就很難獲得固定的稅收等。

因此，這個時候出現了法律體系的擴張，最終演化成英美法系和大陸法系。兩者看上去殊途同歸，但還是有很大的區別，作為政府手上的工具，在英美法系背景下，政府的權力和威嚴來自司法，而大陸法系背景下，政府權力來源於行政。

我個人的理解是，英美法系更容易激發社會對財富創造的慾望，而大陸法系的強項應該是財富分配，所以英美依然有很強的經濟活力，但並未變成福利社會，而歐洲大陸經濟活力下降，但變成了福利社會。

從長期來看，歐洲大陸和英美的衝突，以及中國跟美國的衝突，本質上是類似的法律體系的衝突，只是這種衝突的發展階段有所不同罷了。

但問題在於，世界依然在高速發展，在兩種法律體系沒有決出勝負或互相融合之前，新的操作系統開始介入，法律雖然還在保持著一種重要的底層規則，但已經難以滿足越來越複雜的全球化趨勢，新的變數已經加入。

美元

到了全球貿易時代，人類需要的不僅僅是靜態的法律條文，而是對各種社會交互形態的整合。

黃金在很長時間裡扮演了貨幣的角色，但黃金缺乏便捷性，同時黃金的產出過於稀缺，很難滿足人類對契約、效率等的高速增長需求。這個時候，由某個單一的國家來提供貨幣系統，就成為一種可能，美元跟黃金脫鉤，更大規模的全球化，實際上就是一種新的社會系統的誕生。

在過去半個世紀里，資本主義和新興經濟體的發展，就是濃縮起來的一部美元分配史。人類的大部分行為都是圍繞著美元來展開，美國所維持的貨幣系統，可以說就是最近半個世紀以來的全球規則，貨幣成了人類最主要的底層系統，以至於時至今日，依然沒有幾個國家有能力凌駕於美元系統之上。

互聯網

進入二十一世紀之後，除了美元，更多的人被互聯網串聯到了一起，世界變得扁平化，信息的不對稱被嚴重削弱，政治、經濟等等領域都因此而不得不做出新的改變。

互聯網不僅能左右美國大選，也能讓中國在很短的時間之內，建立起來比美國更加高效的交易支付系統。互聯網成為人類獲取信息、傳遞政令、維護輿論、產生交易、完成生產的中樞系統，互聯網的規則成為新的人類行為模式，在這個系統之下，無論是個人、組織還是國家，都被無情的席捲，成為互聯網的原住民。

隨著互聯網時代的到來，美元成為互聯網系統之下的一個子系統，美元系統雖然還很重要，但單純的美元（貨幣系統）已經無法處理更加複雜的互聯網時代需求。

區塊鏈（自運行協議）

傳統的互聯網跟國家、軍隊、民族等等類似，都帶有非常明顯的中心化色彩。傳統互聯網聚集起來的人越來越多，但主導權則越來越集中。這種集中效應，甚至超過了宗教。全球有三大宗教，卻只有一個谷歌。

美國之所以把「安全」戰略重心全面轉向中國，其中一個原因就是，中國在未來系統搭建層面，也就是基於互聯網的系統搭建方面，是唯一個有可能超越美國的國家，歐洲和日本等基本歇菜。

互聯網所推動的高效、透明和標準化，在未來很長時間裡，都會是主導整個世界的主流系統和模式。但傳統的互聯網，並沒有解決終極信任問題。

人們一度相信神明，相信暴力，相信法律，相信貨幣，相信互聯網，但這種信任只是從更低級的信任，走向更高級的信任，並沒有脫離對創造信任本身的中心化組織的依賴，這就使得其信任必須要有很強的背書邏輯。傳統互聯網，需要具體的公司來做最後的信用保證，比如昨天我正好趕上了支付寶APP的閃崩，就是因為機房網路出現短暫抖動所致。

傳統互聯網註定是人類系統升值的過渡階段，傳統互聯網的使命是改變人類的交互規模，用信息技術讓本來根本無法聚集到一起來管理的數十億人口，遷徙到了互聯網這個共同的空間，這種距離的拉近，比任何交通工具和電報的效率高無數倍，成本也大幅降低。但也使得其系統十分脆弱，而又影響巨大。

近年來，自動駕駛、智慧機器人、比特幣等的誕生，很多人認為僅僅是一種新時期的工業革新罷了，其實這不是什麼第四次或第五次工業革命，這是一種基於傳統互聯網系統升級而來的人類新的底層生存系統的萌芽。這不僅僅是工業技術問題，還是政治學、經濟學、數學、社會學、哲學等的整體性升級。

空氣、水、黃金，是自運行系統（區塊鏈），宗教、軍事、法律、貨幣、傳統互聯網等都不是。但空氣、水、黃金，可承載的東西太少，它們承載不了透明、契約、法律、權力、分配、安全、民主等等，因為它們嚴重受限於物理世界，它們無法離開現實世界裡中心化物理設備而自行高效運轉，就算是黃金這種金融貨幣屬性極強，全球共識程度最高的自運行系統，其運轉也十分不便，這就限制了它的擴展性，以及締造下一代超級系統的能力。

2009年比特幣誕生，區塊鏈這個自運行系統在地球上出現。區塊鏈幾乎可以承載人類所有階段發明出來的最優秀的系統，它透明不可篡改，可以替代行政審查，以及明確責權；它不需要中心化動力來維持運轉，具有天然的國際屬性，可替代國際各種跨國協議，消除基於宗教、意識形態等的隔閡；它具有極大的可擴展性，可以誕生新的契約和不同屬性的產權，可替代法律合同和證券等；它具有相當強的安全性和抗攻擊能力，可節省大量用來維護安全的防衛性支出。

總之，區塊鏈這套系統，可以承載貨幣、法律、銀行、契約、結算機構、國際協議等等多重角色，而不用擔心它背後的系統的共識性、可靠性。

當然，在新的系統還沒有滲入到全球各個角落之前，人們依然會依賴原有的系統，就像我們目前雖然依賴於互聯網、貨幣、法律等系統，但同時也依賴於宗教、軍事系統。

新的系統的誕生，並不意味著原有系統的失效，這些系統都是人類智慧的結晶，也是人類文明的延續，很可能會長期永久的共存，但每一個系統所發揮的作用，將會呈現明顯的此消彼長。

中國雖然有擴展性極強的移動支付網路，但美國已經逐步放行Facebook發行的基於區塊鏈和美元的國際貨幣Libra，美國財政部長今日已明確表態，只要不用於資助恐怖活動，遵守銀行保密和反洗錢規定，美國將不會反對Facebook創新數字貨幣。這將使得美元成為區塊鏈的子系統，但Facebook屬於美國，未來的世界，依然是美國優先嗎？
