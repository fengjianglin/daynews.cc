百度沸點發布年度榜單，中國人這一年的關注點都在這了，猜猜哪個關鍵詞霸榜？

2019-12-17 119條評論 1,167次閱讀 198人點贊

百度沸點年度榜單今天發布。今年，百度開設了包括年度關鍵詞榜、年度國民驕傲榜、年度事件榜、年度淚點榜、年度流行語榜、年度熱議榜、年度科技榜、年度沸點人物榜、年度電影榜、年度電視劇榜、年度綜藝榜、年度小說榜及年度沸點明星在內的13個榜單，呈現了網民2019年在各個領域中最關注的焦點議題。

縱觀2019年沸點榜單，「新中國成立70周年」佔據了多個榜單的C位，那些值得銘記的國家大事和在各自領域實現了重大突破的人物也成為了網民關注的重點；同時，以AI、5G為首的科技熱詞搜索熱度依舊不減，而文化產業特別是國產電影、電視劇的集中崛起更是推動百度沸點在今年設立了與之相關的單獨榜單。

作為國民獲取知識和信息的最大平台，百度目前全平台累計用戶數超過10億，百度搜索佔據市場份額超過72%，百度信息流日分發量更是超過150億，居全國首位。基於龐大的搜索和信息流數據，也真實、客觀地記錄並還原2019年的中國社會的真實面貌。

「新中國成立70周年」在四大榜單居首

2019年正值新中國成立70周年，人們對於祖國的關注和熱愛也客觀地呈現在了百度沸點榜單中。關鍵詞「新中國成立70周年」成功霸榜，成功躍居「年度關鍵詞」「年度國民驕傲」「年度事件」「年度熱議」四大榜單榜首。

祖國的繁榮發展，社會的日新月異，不斷催生著人們心中的驕傲與自豪感，特別是在新中國70歲華誕這一年，人們更加關心祖國的歷史、變化和進步。全年海量的關注，最終使得「新中國成立70周年」成為了百度沸點多個榜單的絕對C位。

大事件、大人物共譜國民驕傲

站在年終回望2019，它稱得上是讓我們足夠驕傲的一年。嫦娥四號在年初成功著陸月球背面，是首次有人類製造的探測器在月球背面實現軟著陸；年中，長征十一號運載火箭在黃海海域成功實現首次海上發射；9月底，被譽為「新世界七大奇蹟」的大興機場正式投入運營，是全球最大的空港之一……一件又一件國家大事，成為了「年度國民驕傲」。

與此同時，各行各業的傑出人物也在這一年頻頻突破創新，甚至實現了造福於世界的價值。世界雜交水稻之父袁隆平榮登「年度沸點人物榜」榜首，同樣入選榜單的還有在青蒿素領域不斷鑽研並取得突破的屠呦呦。

中國體育同樣在這一年取得了令世界矚目的成績。郎平與中國女排一起再登世界之巔；孫楊達成了世界游泳錦標賽400米自由泳史無前例的四連冠；蘇炳添6秒47奪冠，圍棋國手柯潔一年攬得多個國內外冠軍……「年度沸點人物」「年度國民驕傲」等榜單紛紛記錄下了他們這一年的高光時刻。

科技熱度不減 賦能產業生活

科技給社會帶來的變化同樣被百度沸點年度榜單一一捕捉。人工智慧技術持續落地、5G商用化開啟、智能家居走進千家萬戶……國民生活、各行各業正因科技的力量發生著巨大變化。

如今，人工智慧技術在製造、醫療等行業中已經可以扮演相對重要的角色；5G更是這一年的科技熱詞，其速度快、低時延、穩定性強的特點不僅能進一步提升用戶的網路體驗，甚至還能滿足未來萬物互聯的應用需求。AI尋人的上榜讓我們看到了科技背後的溫暖力量，智能家居、刷臉支付和VR、AR等更是讓我們對於未來的科技生活充滿了憧憬和遐想。

這份「年度科技」榜單，基本可被視為是2019年中國科技發展的縮影。

國產文化集體走向繁榮

2019也是國內文化產業全面爆發並走向繁榮的一年，尤其在電影行業，優秀的國產作品不斷湧現。在百度沸點發布的「年度電影」榜單上，國產電影佔據了八席，既有主題昂揚、氣勢恢宏的《我和我的祖國》，也有《流浪地球》這樣的國產科幻巨作，《哪吒之魔童降世》《少年的你》等電影同樣取得了口碑與票房的雙豐收。

在大熒幕之外，還出現了如《親愛的熱愛的》、《陳情令》、《都挺好》等以質取勝的好劇；《樂隊的夏天》則把人們對於綜藝的審美重新拉回高基準，實現了多方共贏。

作為文化的一種變式，每年都在推陳出新的網路流行語也是觀察社會和大眾的另一個角度。今年的「年度流行語」榜單既收錄了「996」「我太難了」「社畜」等吐槽辭彙，也有「不忘初心」「斷舍離」「好嗨喲」等詞語為積極的生活態度代言。

百度沸點2019中收錄的這些科技、民生、社會、網路等不同領域的熱詞，無一不在呈現著中國社會的千姿百態。憑藉搜索及信息流積累下來的海量數據，百度不僅為全社會提供了真實、客觀的數據參照，成為了折射中國社會的風貌的一面鏡子。
