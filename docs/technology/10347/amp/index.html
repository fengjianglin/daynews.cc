QQ、網易雲、蝦米音樂三分音樂市場後，給咱們這些普通用戶帶來了很多麻煩。

比如你必須同時安裝這三家的 App，才能保證紅心歌單齊全。

但 App 裝多了也不見得有用。

因為各平台都對歌曲進行了加密。

比如網易雲音樂的是.ncm，QQ 音樂是 qmc0、qmc3、qmcflac 等，並非常規的 MP3、flac 等格式。

所以即使你是 VIP，已經花錢買了歌曲。

也不能在其他播放器或設備上聽


加密還是不加密，獨佔還是便攜這個矛盾很難解決。

如果你只是想安安靜靜聽歌的話。

不如試試第三方工具【音樂解鎖】移除歌曲的加密保護。

用音樂解鎖移除加密

音樂解鎖是一個免費開源的加密音樂解密工具。

目前支持網易雲音樂和 QQ 音樂兩個平台。

因為功能簡單，操作起來也很容易。

比如這裡有兩首 .ncm 格式歌曲。

只需將歌曲拖進去，無需等待就能得到常規的 MP3、flac 等格式歌曲。

已經移除加密保護的歌曲，支持下載、試聽和刪除。

音質依然保持原品質。

也可以在其他播放器或設備中播放。

如果是 .ncm 格式還能保留專輯封面信息。

一些注意事項

音樂解鎖本身是一個網頁應用。

所以電腦和 Android 手機打開就能用。

（iOS 由於系統限制無解

）

但需要注意的是：

由於程序編寫使用了ES6語法，過舊的瀏覽器可能無法使用，比如 IE

  • 建議使用 Chrome、Firefox 瀏覽器或微軟 Edge Chromium 版本

  • 目前的解密方式不確保能解鎖所有 mflac 音頻，但不能解鎖的情況應該極少見

當然，也不要使用微信內置瀏覽器，會顯示無法打開。

安卓用戶可以直接在下方兩個文件夾找到加密歌曲。

  • QQ音樂下載的音樂文件默認位於：/qqmusic/song/

  • 網易雲音樂下載的音樂文件默認位於：/netease/cloudmusic/Music/

如果你有幾十上百首歌曲等待解密。

最好在電腦上使用，因為音樂解鎖也支持批量解密。
