揭秘年營收超300億美元的潘世奇物流

2019-12-17 1,993條評論 3,963次閱讀 1273人點贊

導讀

標籤貼標零差錯，成本降低20%，這家認真低調的「富二代「物流企業是如何做到的？

變與不變，從來都是一個「艱難」的選擇。

特別在2019年，無論是對於大平台還是小公司而言，這都可能是一次危、機並存的深層次洗禮。

同樣，進入中國14年的美國物流服務商潘世奇物流(Penske Logistics)也正在進行一場「變革」。潘世奇物流中國區總經理韋奇文（Peter）告訴羅戈網記者，「我們要讓客戶把主要精力放到生產和銷售，物流與供應鏈放心交給我們。 」

潘世奇物流母公司潘世奇集團於1969年在美國成立，年營收超過320億美元，旗下產業涉及汽車零售，卡車租賃，零部件製造，物流運輸，職業賽車隊以及私人股權投資。潘世奇物流每天有超過30萬輛自有卡車在公路上，加上超過兩百萬平方米的倉庫運作，其物流服務覆蓋北美，亞洲，歐洲和南美，是一個勤勤懇懇的「富二代「物流公司。

潘世奇物流自2005年跟隨客戶來到中國市場，至今已深耕國內市場14餘年，有豐富的倉儲、國內外運輸、定製車隊、供應鏈諮詢和規劃等運作和服務經驗。

近日，羅戈網很榮幸能邀請到潘世奇物流中國區總經理韋奇文（Peter）來探討合同物流的數字化變革。以下為專訪實錄：

1. 羅戈網記者：你好，Peter！先簡單介紹一下潘世奇物流的業務特點，比如合同物流和國際貨代分別的規模大概是多少？業務優勢點是什麼？

Peter：我們是一個供應鏈綜合服務企業，能提供國內外運輸、倉儲、定製車隊、諮詢、報關報檢等一系列服務。目前業務形態更偏重合同物流，包括物流運作定製化，我們有能力在整體解決方案中給客戶提供這方面的服務或幫助。

至於服務特點，質量是我們運作的核心。只有堅持做好服務質量，才能有效支持客戶的業務發展，達到雙方共贏，這是我們的目標，是我們和客戶、供應商之間的的一個理念。比如，前段時間我們在北方的客戶有定製化運作的需求，而供應商缺乏資源，那我們就與供應商深化合作，通過資源和資金上的扶持，達到運作的高標準需求，實現共贏。

2. 羅戈網記者：剛才也提到，潘世奇物流是一個供應鏈綜合服務企業，是否能談一下目前客戶的行業分布？針對不同行業的客戶具體有哪些操作？能夠舉一些案例說明嗎？

Peter：目前潘世奇中國區的客戶主要來自於汽車、食品飲料、與工業產品。

比如之前困擾一家國際領先汽車變速箱公司多年的盤點問題 – 上萬個SKU，每月操作幾十萬件產品，要求99.99% 的庫存準確率，每年需要停業4天進行盤點。在我們接手後運用進出庫質量流程把控和循環盤點，把庫存準確率提升到了100%，解決了庫存準確率的後顧之憂，更節省了盤點所付出的時間和成本。

再舉個例子，我們食品飲料行業的客戶，每個月有超過200萬張的標籤需要列印並粘貼，但涉及到的SKU有上千種，同時，需要經過商檢，客戶，市場等多個渠道方面的管控和監督，所以如何確保標籤的準確性？我們通過嚴謹並完善的標籤製作及貼標流程，使貼標正確率從2015年合作至今始終保持在100%，還讓成本降低了20%。

另外，我們幫70%的客戶使用BI工具提供可視化的庫存及報表管理。

3. 羅戈網記者：去年順豐宣布整合DHL在中國大陸、香港和澳門地區的供應鏈業務；馬士基宣布整合丹馬士物流。國內供應鏈物流市場在快速變化，面對這些變化，潘世奇物流未來有哪些戰略規劃？

Peter：市場上的併購對我們影響不會太大，因為我們有五年的戰略方向，然後每年會根據國際形勢和行業內的變化，會進行一些小調整來應對這個市場。我們有成熟的質量體系，以及經驗豐富並且持續學習的團隊，讓潘世奇能夠緊跟客戶的變化與發展，甚至比客戶更快，引導客戶提出想法。

對此，潘世奇是非常嚴肅且認真的。我們在全球有350人的技術與方案工程師分布在國內，亞太，拉美，歐洲和北美，可以分析信息數據策劃和持續改進方案。這350人中，平均在潘世奇的工作時間長達12年，35%以上都有研究生學位，他們每年為客戶提供超過上千個運輸網路優化、合同物流倉庫優化方案，以及供應鏈整體建模項目，使用了目前市場上主流的商業優化軟體譬如Llamasoft系列, JDA Strategist/Transportation Modeler, TMW-Resource Pro以及一系列潘世奇自主開發的優化軟體。值得一提的是，我們的工程師團隊有25%是女性。

4. 羅戈網記者：過去幾年，你認為供應鏈領域甲方的需求和市場競爭在發生哪些變化？目前傳統供應鏈服務的痛點在哪裡？

Peter：對於甲方來講，大家都更加關注成本控制和全程可視化，然後就是業務在轉型，比如全渠道銷售的需求使得當前的供應商裡面有很大一部分的物流體系和承運商資源難以滿足甲方要求。除此之外，後面供應鏈統籌與優化和承運商管理也是甲方在物流及供應鏈管理中面臨的挑戰。

對於甲方關心的問題，我們的想法是這樣的，假如我們每年給到客戶8%的一個指標降幅，我們自己會找一些可以優化的地方。通過我們的系統，通過我們的操作人員，通過我們自己的一些流程改善，我們把這些地方給找出來之後，再看能否在時效上面做一些優化。

比如天津的一個客戶，它開始貨量不多只能走零擔，如何保證時效和成本可控範圍之內，這是一開始我們所面對的問題。後面運用我們的平台，把客戶和我們相應的客戶，通過整合資源配送，路徑規划去幫助客戶提高時效和控制成本。

為了更有效地解決客戶通點，我們正在和客戶嘗試形成一種戰略性的合作。比如客戶的供應商由我們直接管理，參與到客戶整個計劃、採購、庫存管理當中。通過這樣的戰略合作，一方面可以為我們產生一定的利潤空間，另一方面能夠滿足他們的需求同時形成長期的合作關係。

5. 羅戈網記者：從消費互聯網到工業互聯網，供應鏈管理服務會怎樣升級？未來供應鏈管理會怎樣發展？升級後又會給行業、客戶帶來哪些價值？

Peter：從目前的趨勢看，現在整個供應鏈在做數字化的迭代，就是從多渠道、多業態，整體在做數字化的變革。在這個供應鏈數字化變革的過程當中，怎麼樣能夠來整合我們不同的渠道對應的庫存、倉儲、運輸的管理，是個值得深究的問題。合同物流未來的競爭在大部分標準化產品之外如何一些定製化服務方案， 這其中直接決定企業成本和效率的將是運營的精細化。這使得大家都開始重視IT，重視科研力度，要滿足客戶對於時效的要求。潘世奇經過這麼多年運營已經具備一套非常成熟的精益體系的架構，可以從流程標準化和質量穩定性上持續去支撐我們在運營的精益管理，最後來提升我們的客戶體驗度。
