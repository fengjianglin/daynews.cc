有研新材（600206）

公司簡介：

有研新材料股份有限公司是一家主要經營硅單晶拋光片、化合物半導體及鍺系列材料生產銷售。的公司,主要產品有硅單晶拋光片、去絨產品等。該公司是中國最大的具有國際水平的半導體材料研究、開發、生產基地,坐落於北京市高新技術產業雲集的中關村科技園區,佔地3萬平方米。公司擁有符合國際標準的現代化廠房及生產設備,年生產能力達250萬片集成電路級拋光片,是國內半導體材料行業的主導企業。

  • 主營業務：稀土材料、微電子光電子用薄膜材料、生物醫用材料、稀有金屬及貴金屬、紅外光學及光電材料、光纖材料等新材料的研發與生產
  • 產品類型：微電子光電子用薄膜材料、超高純金屬及稀貴金屬材料、高端稀土功能材料、紅外光學及光纖材料、生物醫用材料

    • 概念強弱排名：晶元概念， 5G， 華為概念， OLED， 標普道瓊斯A股， 富時羅素概念股， 滬股通， 軍工， 融資融券， 稀土永磁， 小金屬概念， 新材料概念， 央企國資改革， 醫療器械概念
