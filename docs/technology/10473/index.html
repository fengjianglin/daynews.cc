新股破發，新能源汽車還能跑多遠？

2019-12-04 543條評論 1,926次閱讀 92人點贊

關注美蘭，為你揭秘更多價值投資板塊和市場交易機會！乾貨持續更新，點個關注不迷路！

一、大盤簡述及漲停股分析
復盤前講一下今天科創板上市的建龍維納上市第一天跌破發行價了，基本面無亮點，化工原料分子篩粉，下游是油漆塗料，在科創板發行57倍市盈率，對不起，大家不認可，要是在主板最多20-30倍吧，所以後面新股申購前要考慮一下行業及發行市盈率，同時市場缺乏資金接。
今天市場主要表現的板塊以汽車相關和大科技為主。
新能源汽車板塊比想像中要強，連板股中有6隻都是和汽車相關的，首板助攻也有4隻，不過下午市場並沒有挖掘板塊首板，隨著藍海華騰和渤海汽車的炸板，亞星客車和金冠股份的走弱，預示著明天這個板塊會有較大分歧。
這塊炒作主要方向可能是TPMS，後期看看萬通智控能不能走成ETC板塊中的萬集科技，概念正宗的保隆科技有減持，今天沖高回落。
龍頭是萬通智控，先鋒是威帝股份，這裡明天還是要關注萬通智控有沒有低吸機會，威帝股份上一次炒汽車一字到結束為主，可參與性不強，珈偉新能疊加半導體和鋰電池，也可以關注一下。德宏股份疊加稀土，今天MYZ有消息面刺激，故下午二板回封，相關農業及黃金有所拉升。
科技股誠邁科技和北方華創繼續創出新高，但沖板回落，這塊主要還是以趨勢為主，午後漫步者超預期漲停帶動TWS耳機分支上漲，科技股只要大的趨勢沒有破壞，逢回檔還是可以低吸，科技股主要是在ZMMYZ的背景下、科技興國自主可控的強勁市場需求和政策支撐，機構抱團根深蒂固！
次新股這裡要關注下，看有沒有超預期的走出來，重點看下南華期貨和廣電計量。
另外股權轉讓板塊的明星股麥克奧迪，尾盤搶籌，看一下明天有沒有超預期表現。
二、人氣股分析：
1、萬通智控
萬通智控是汽車及TPMS的龍頭，昨天分析有可能庄會出貨，一致性接盤要謹慎，而昨天晚上受外圍利空影響早盤反而溢價有限，製造了開盤的小分歧，明天繼續看它的表現。板塊明牌龍頭，不用打板，看後期有沒有可能走成萬集科技那樣的趨勢龍。
以今天板塊表現明天又是高溢價，看分歧及承接的機會，重點關註標的。
題材：TPMS+特斯拉+殼資源+長三角+小流通盤；
2、太化股份
這個股今天秒二板，競價是昨天首板中開的最好的一個，股權轉讓加稀土永磁，昨天晚上受MYZ利空影響，稀土明顯是避險板塊，很容易受資金關注，明天三板有一難度，第一，上方平台密集成交區，第二下午德宏股份上二板帶動相關稀土個股沖高，可能會被卡位。
題材：股權轉讓+稀土+低位低價。
3、威帝股份、泉峰汽車、京城股份、珈偉新能、德宏股份
這幾個股放在一起講，除了威帝是先鋒外其它幾個都是純跟風股票，首先威帝昨天第一個上板汽車股票，今天理應享受溢價，昨天晚上復盤提到過會一字，但這個股接力價值不大，一字籌碼斷層，明天可能就會結束，板塊核心龍頭是萬通智控。
泉峰汽車（次新+汽車配件）、京城股份（燃料電池+小盤）、珈偉新能（半導體+鋰電池）這三個，相對看好珈偉新能有半導體屬性，低位低價。
德宏股份（汽車+稀土）下午回封是沾邊了MYZ消息刺激作為稀土的避險屬性，帶動了金力永磁等個股拉升，這個和太化股份放在一起看，同時再看下明天稀土和農業板塊的聯動性。
4、中潛股份
中潛股份，前期庄股，看不懂，不評論！
題材：體育+紡織+小盤。
三、每日一技
說說看到個股競價階段技巧或者是看盤細節。
今天說一下龍星化工，前幾天是新能源龍頭，早上競價有所異動，先看一張圖：

9點18分00秒和9點18分21秒，分別有7530手和10000手大單掛漲停價格買入，但隨後在20分之前全部撤單，這裡大資金在試盤。其實這樣的試盤11月27日在永鼎股份競價階段也發生過，當時資金博弈反包，後失敗。
這裡資金在試盤，我認為大資金一方面看到威帝股份開一字，加上汽車板塊昨天是主流板塊，同時龍星化工作為近幾天新能源小龍頭，熱度還在，很容易吸引資金關注。
那就是人氣尚未消退的龍頭+碰上風口板塊，一拍即合。如果跟隨的話，可以小倉位競價跟一筆。當然打板的也沒必要，今天午後炸板了，明天溢價也有限。
