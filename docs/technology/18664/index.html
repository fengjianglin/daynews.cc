2019年11月22日，2019-2021年中國聯通山東青島智能城域網設備集中採購招標公告：

項目名稱：2019-2021年中國聯通山東青島智能城域網設備集中採購項目

招標內容：本項目計劃在青島城域層面，新建一張以通信雲DC為核心、採用SR+EVPN技術的智能城域網，實現城域範圍內綜合業務的承載。
採購內容：城域網核心匯聚邊緣設備140台，通信雲匯聚邊緣設備50台。

預算金額：人民幣含稅4972萬元。

本項目劃分為2個標段：

標段1為智能城域網核心匯聚邊緣設備，預估數量140台，確定1名中標人；

標段2為通信雲匯聚邊緣設備，預估數量50台，確定1名中標人，兩個標段兼投兼中。

2019年12月16日，2019-2021年中國聯通山東青島智能城域網設備集中採購項目中標候選人公示，思科獨家中標2019-2021年中國聯通山東青島智能城域網設備集中採購項目，中標總金額：39724246.00（元）。
