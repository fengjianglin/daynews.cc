斯坦福大學教授蔣里：擁有人工智慧思維的孩子才不會被替代

2019-12-08 1,634條評論 3,423次閱讀 1327人點贊

12月8日，中國教育三十人論壇第六屆年會在北京舉行，本屆年會的主題為「科技發展與教育變革」。科技發展日新月異，教育到了轉型時刻。在科技的助推下，全世界都在探索教育變革。技術如何改變教育生態？科技發展給教育又帶來哪些挑戰？如何培養適應未來社會的人才？未來學校、人工智慧與教育之間的關係又是怎樣？

斯坦福大學人工智慧、機器人與未來教育中心主任，斯坦福大學全球創新設計聯盟聯席主席蔣里在論壇上發表了《人工智慧——未來教育的機遇與挑戰》的演講，以下是演講內容，原文有改動。

圖為蔣里在中國教育三十人論壇第六屆年會演講

人工智慧給未來教育提出了一個挑戰。人工智慧技術和機器人技術，會推著教育往前發展，最後會擊穿教育，為教育帶來變革。

COMPUTER這個詞相信大家都認識。這個單詞是什麼時候在人類歷史上第一次被使用呢？它是在1613年被第一次使用，這個單詞在當時被創造出來的時候，並不是計算機的意思，而是從事計算的人。到了六七十年代，當IBM的大型機進入到美國航空航天局的時候，就開始用COMPUTER指機器了。

之所以提這個事，是因為經常被問到這個問題：未來真的有很多工作被機器人替代嗎？這是顯而易見的。過去400年都在使用COMPUTER這個詞，有350年用這個詞指人，只有50多年用它指機器，而且大家已經忘記了它曾經是指人。

現在我們處於機器人和人工智慧技術飛速發展的年代，為了更好地理解機器人和人工智慧的進步，我跟大家分享幾個案例：

2015年6月，美國洛杉磯舉行了美國國防部高等計劃署機器人研究挑戰賽，這是當時世界上最高級的比賽。當時機器人必須完成八個項目：開車、下車、開門、進門、打開開關、牆上鑽洞、過網、上台階。當時最先進的一個機器人需要去開門，但機器人看著門就定了2個小時。這是死機了嗎？實際上沒有，它一直在算門把手在哪兒。完成這八個項目，機器人需要很長時間。

8個月後，出現了一個新的機器人，它走路的樣子很像人，可以把自己救起來。

到了2017年7月，這個機器人可以360度後空翻，震驚了世界。

現在一談到阿兒法狗，提到最多的一句話是「它很牛」，因為圍棋下法比整個宇宙的原子數量都要多。這句話是正確但不精確的。為什麼不精確？整個宇宙的原子數量是10的80次方，圍棋能下出多少種下法呢？是10的768次方，如果把整個宇宙中的每一個原子再變成一個宇宙的話，是10的160次方。在圍棋項目上取得的成就，在人工智慧界確實是一個不得了的事情。而且中日韓都認為圍棋是最聰明的人在下，它是智慧皇冠上的明珠，而人工智慧在這一點上戰勝了人。

接下來我將談人工智慧和工作的關係。京東想做完全沒有人的公司，十年之內京東會把公司的人從16萬減到8萬。斯坦福大學去年做了實驗，將人工智慧和20個專業律師放在一塊兒，讓二者同時參閱法律文檔。結果顯示，專業律師的正確率為85%，人工智慧正確率為95%，多了10%的正確率。人參閱法律文檔平均花1.5小時，機器花26秒。前一段時間中國談996，早九點到晚九點，一個星期六天，對機器來說就是007，機器是不休息的。

對於高端領域來說，如果做簡單重複的腦力勞動，確實比體力勞動更容易被替代。2008年瑞士銀行在華爾街的交易大廳人頭攢動，但2016年已經人去樓空。華爾街十年時間的用工人數減掉了一半。花期銀行預計，2025年歐美銀行業將減員30%。人工智慧的威脅是未來10年到20年發生的。根據牛津大學的調查報告，美國現在工作的47%會被代替，中國這個數據是77%。美國勞工部出了一個報告，指出現在在小學階段學習的小孩，以後要從事的工作65%現在還不存在，這給教育行業提出了巨大的挑戰，就是你還不知道靶子在什麼地方。

現在的中小學生有可能成為一進入社會，就被機器人代替。我們如何讓孩子做好準備呢？其實沒那麼複雜。

第一，不要與機器人在它擅長的領域競爭。舉個例子，你不希望自己的孩子跑步特別快，比汽車還快。因為跑得比汽車還快，是沒有意義的。

與此相似，我們訓練孩子的計算能力很重要，但是要追求比計算機還快，也沒有意義，因為終究不可能比計算機算得快。

第二，不要把學生當成機器人訓練。我們處在知識爆炸的年代，1990年前知識和信息的總量100年翻一倍，到了1950年加速到25年翻一倍，到了現在一年翻一倍。什麼概念呢？20年後，我們人類的知識會增長20萬倍。如果不相信，我們可以往歷史看，20年前的1999年，我們用的電腦有一個洞，這個洞用來放軟盤。軟盤的容量是1.44兆。而現在的硬碟已經超過2T，已經超過100萬倍了。

所以我們現在進退兩難，前面是知識爆炸，後面是人工智慧機器人在追。壓力大，給我們帶來一個機遇，斯坦福提出了人工智慧思維，包括三個方面。

第一，了解人工智慧的運作方式。

第二，把自己和人工智慧區別開來。

第三，知道自己和人工智慧如何協作。

具備了人工智慧思維，你才能分得清哪些是人應該培養的能力，哪些是機器培養的能力。

但是，我們目前在中小學還在教很老的知識，甚至很多本科階段還在教幾年前的知識，我們需要一個新的教育系統，將新知識引入到中小學，來激發學生。因此我們重新設計了斯坦福研究生階段的人工智慧科目，把從小學一年級到高三的學生放在一個教室里，讓他們上斯坦福研究生的課程，結果發現三年級以上都能聽懂。我們發現一個問題，我們有必要讓孩子提前10年到15年接觸到世界最前沿的科學技術，包括人工智慧、機器人、設計思維，點燃孩子學習的內在動力他們具備人工智慧思維。這樣他們才能成為引領未來的人才。

我們有一個大膽的嘗試，不光是K12，還把小學、中學和大學生放到一塊兒做項目。令我們驚喜的是，不同年齡段的人做同樣一個項目的時候，學到的東西是完全不一樣的，思考問題的方式也不一樣，互相之間有很多交流。

斯坦福有一個理念，我們必須要付出實踐，拿出來真真實實的測試。我不光去過中國最頂尖的名校，還去過中國最偏遠的地方。去年夏天去到了海拔3400米的地方，這是一個特別窮的地方，周圍沒有任何經濟活動。去到最近的倉管，需要開車一個半小時，去最近的旅館需要兩個小時。

就這樣的地方，學校設施不錯。我在那個地方，給學生上了一個星期的課，這些小孩都能聽懂，而且他們75%是藏民。我到了當地沒有賓館住，住在教室宿舍里，早上五點半被吵醒了。我對老師說，為什麼這麼早就讓小孩讀呢？老師說，是他們自己讀的。這些小孩回家要走一個小時的路。

推未來教育的時候，我去過全世界很多地方，中華民族最牛的一點，所有的人都知道教育是重要的。我去了全世界很多地方，在很多地方首先給大家講了一個問題，為什麼教育是重要，有的時候甚至花一兩天說服不了一個人教育是重要的。中國的教育一定能做好，因為我們的基礎不一樣。

人工智慧思維是未來社會的常識，人工智慧思維是教育在中小學階段和數學一樣重要。我們有必要讓孩子提前10年到15年，接觸到世界最前沿的科學技術，包括人工智慧、機器人、設計思維。用最前沿的科技去點燃孩子學習的內在動力，培養他們成為引領未來的人才。
