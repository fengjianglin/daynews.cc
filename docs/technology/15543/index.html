為青少年提供思維碰撞的平台，法國青少年社交媒體平台「Yubo」獲 1230 萬美元融資

2019-12-14 1,723條評論 2,847次閱讀 1851人點贊

「Yubo」官網

法國青少年社交媒體平台「Yubo」近日宣布獲得了 1230 萬美元融資。本輪融資由 Iris Capital 及 Idinvest Partners 領投，Alven、Sweet Capital、Village Global 參投。

「Yubo」的目標人群為 25 歲以下的青少年群體，希望幫助他們結識新的朋友，創立屬於自己的社交圈，這種相對封閉的圈子也是「Yubo」主打的社交概念。「Yubo」表示，該平台已吸引超過 2500 萬的青少年用戶關注。

目前，「Yubo」的團隊由15名工程師、設計師、數據科學家和營銷人員組成，位於法國巴黎。「Yubo」為用戶提供可信賴且安全的平台，青少年可以在「Yubo」平台輕鬆地結交來自世界各地的新朋友，與他們建立在線友誼。

「Yubo」官網

在「Yubo」App上，用戶可以向左或向右滑動屏幕來尋找想要結交的朋友，類似於 Tinder，但它不是一個約會App，是用於找到志趣相投的夥伴。

「Yubo」最有趣的部分是直播室，用戶可以作為主持人，在平台上直播。所有對直播感興趣的用戶可以隨時加入，並與主持人聊天。他們可以談論最新的 Netflix 電視節目，一起彈吉他或只是放鬆隨意地談論事情。

於2018年10月開始，「Yubo」鼓勵用戶購買平台上的商品或訂閱高級帳戶，從而獲得收益。成為高級用戶後，用戶可以增加個人資料，更改對方加好友請求時自己資料的可見度。據悉，在 2019 年「Yubo」已通過其內購服務創造了 1000 萬的收入。

除了將繼續研究標記工具和實時審核的演算法，來發現平台的不適當內容以外，「Yubo」還將擴大其審核小組的規模。「Yubo」還與 Alex Holmes，Annie Mullins，Travis Bright，Mick Moran，Richard Graham 和 Anne Collier 組成了安全委員會，用來提高平台的安全性。

「Yubo」對青少年用戶非常負責，將安全作為重中之重。「Yubo」限制用戶的最低年齡為13歲，要求所有用戶必須遵守平台的基本規則，這旨在降低年輕人的社交風險，增強他們的安全技能，並為用戶提供最佳的應用體驗。

「Yubo」還將利用本輪融資擴展新市場。目前，「Yubo」的活躍群體主要集中在美國、加拿大、英國、北歐、澳大利亞以及法國，新的融資將推進「Yubo」進軍巴西和日本。此外，「Yubo」也計划進行人員擴張，僱用35名新員工。
