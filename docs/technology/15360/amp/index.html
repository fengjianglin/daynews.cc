黑客帝國屏幕保護設置

1. 打開VS2013或以上版本，新建—>項目,Visual C++，win32，選擇win32控制台應用程序，名稱填寫為”hacker”。

2. 將以下代碼複製到項目源文件中

黑客帝國屏保源碼

3. 編譯項目工程

4. 找到生成的exe文件，將exe後綴改為scr（hacker.exe->hacker.scr）

5. 將文件hacker.scr拷貝到C:\Windows\System32下

6. 滑鼠移到桌面，右擊，選擇個性化，屏幕保護程序，選擇hacker。
