據ZDNet12日報道， 俄羅斯警方突襲NGINX公司莫斯科辦公室，聯合創始人Igor Sysoev被帶走。

事情經過

Rambler集團對NGINX公司提起版權侵犯訴訟，聲稱其擁有nginx Web伺服器代碼所有權，於是警方就帶著搜查令上門拿人了。

Igor Sysoev

事情緣由

Rambler集團聲稱，nignx的作者Igor Sysoev是在Rambler任職系統管理員期間(2000年)開發的nignx，並於2004年開源。Igor Sysoev也在公開場合說過nignx開發經過。

隨後，2009年Igor Sysoev創立NGINX公司。2019年3月，網路安全與設備公司F5以6.7億美金收購NGINX。Rambler集團是看上了這收購分成還是什麼呢？
