東大攜手東升，將研發新能源動力的大型車輛

2019-12-07 1,877條評論 3,353次閱讀 1601人點贊

現代快報訊未來汽車如何向輕量化、電動化、智能化發展？目前國內汽車製造業科研方面已取得哪些進展突破？12月7日，2019國際新能源與智能汽車技術高峰論壇在東南大學舉行。200多位專家學者、產業人員匯聚一堂，圍繞新能源汽車、智能駕駛系統、車路協同等問題展開交流討論。

△活動現場

論壇開幕式上，東大攜手揚州東升汽車零部件有限公司，聯合成立汽車零部件研究中心，致力於研發新能源動力的大型汽車，如大型貨車、工程車等。 「東南大學集聚了汽車製造的多個相關學科，如機械製造、測量工程、電子信息、交通測控等，在新能源與智能汽車技術研究方面有著得天獨厚的優勢。且隨著5G、6G的應用推廣，這種優勢將越來越顯著。」東南大學機械工程學院副院長、教授殷國棟說，所謂智能汽車，就是在普通車輛的基礎上，增加感測器、控制器、執行器等裝置，實現人、車、路等智能信息交換，使車輛具備智能的環境感知能力，能夠自動分析路況與實時環境，並作出反應，解放使用者的雙手，減少交通事故的發生。

△研究中心揭牌儀式

同時，殷國棟還指出，光車輛智能還不夠，路也要變得智能。「很多省份已經開始將原本雙向4車道的高速公路，變為雙向6車道，多出的兩條車道則打造成智能車輛通行的智慧公路，在上面鋪設大量的智能感知設備，實現車路協同，在保證交通安全的同時，提高通行效率，從而形成安全、高效和環保的道路交通系統。」殷國棟說。目前，「車路協調」工程已在四川、山東、浙江、廣東等地建設推行中。

論壇上，江蘇省工業和信息化廳產業轉型升級處發布了《江蘇省新能源汽車的發展報告》，公布了江蘇省智能汽車先進技術產品名錄。數據顯示，依託於高校科研院機構和產業集群發展，江蘇智能汽車製造業水平處於全國前列。

通訊員 唐瑭 實習生 陳志豪 現代快報+/ZAKER南京記者 仲茜 文/攝
