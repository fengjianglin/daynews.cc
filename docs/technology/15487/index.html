「強強聯合」看比亞迪DiLink和百度Apollo如何打造智慧出行

2019-12-14 334條評論 3,222次閱讀 176人點贊

​作為汽車智能網聯行業的引領者，比亞迪DiLink一直秉承開放的原則，其車機不但全面兼容手機生態，還結合用車場景、與國內頂級智能科技企業推出定製版的汽車生態，共同打造技術+內容的服務生態體系。此次比亞迪DiLink與百度強強聯合，通過定製化開發，為車主量身打造定製版百度地圖汽車版，並率先在全新一代唐DM車型上實現OTA，推出更懂你的全新導航應用。

相較於手機版，百度地圖汽車版在保持基礎體驗一致的情況下，增加了橫豎屏自由設置、低油/低電提醒兩大特色功能，同時優化了行程預測、明星語音、慣性導航等多項功能。
