選購新能源車，續航你會考慮多長的？

2019-12-20 715條評論 1,948次閱讀 327人點贊

新能源純電動車的普及是一個趨勢,隨著新能源汽車的保有量增多，車輛的基礎設施不斷的提高，以及車輛技術不斷的提高，在新能源汽車使用上面來說也是越來越方便，購買新能源汽車的人也是越來越多，而車輛的續航里程等在買車的時候是最為關注的，那麼對於購買新能源汽車來說，續航上面你會考慮多長的？

買車續航多長能夠滿足使用，這個需要根據車輛的不同使用場景來決定，基本上在城市用車或短途出行，續航在400km左右的續航就能夠滿足日常的用車需求性，以城市的道路出行來看，車輛續航在400km，按照一天行駛的里程在80km左右，那麼能夠堅持五天的時間不用充電，400公里的續航完全足以。

再者來說的話，從現在充電樁的保有量上面來看，基本上能夠白天用車，晚上用峰谷電進行充電，其次公用充電樁也在不斷的增加，所以對於這樣的續航而言，在市區等短途出行上面，基本上是沒有任何的後顧之憂的。

綜上所述。在選擇購買純電動汽車的時候，對於車輛的用途主要定位日常代步出行的情況，同時又具備充電樁的安裝條件，所在城市公共充電資源又非常豐富的話，那麼車輛的續航里程在400km就真得夠用就行，而且車輛的續航400公里左右就完全可以滿足基本用車需求了。
