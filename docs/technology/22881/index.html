國內要聞

小米武漢總部產業園正式投用

12月18日，小米武漢總部產業園正式投用，小米集團董事長兼CEO雷軍表示，未來十年希望能把武漢建成小米的超大研發總部和人工智慧時代的技術高地，給武漢總部設定的目標是未來要能夠容納1萬名員工。（湖北日報）

聯想控股官宣：柳傳志卸任董事長，CFO寧旻接任

12月18日，聯想控股發公告宣布，按照既定計劃，聯想控股董事長、執行董事、聯想集團創始人柳傳志卸任公司董事長及執行董事，將擔任聯想控股名譽董事長、資深顧問及董事會戰略委員會成員；聯想控股執行董事、總裁朱立南卸任公司總裁之職，將繼續擔任聯想控股董事，並任董事會戰略委員會成員。經聯想控股董事會提名委員會提名，董事會決議通過，聯想控股執行董事、高級副總裁、首席財務官寧旻接任董事長，並任提名委員會及戰略委員會主席。董事會同時通過，公司高級副總裁李蓬出任首席執行官（CEO），董事會亦提名李蓬為公司執行董事。

風行將全面接手暴風TV的系統和廣告平台，目前正做內容整合

暴風TV已經被傳統老牌公司風行網收購，目前正在做內容整合。據內部人士透露，目前內容整合的進展是雙方已經建立微信群對接，在將風行OS接入暴風TV，接入版本軟體迭代已經在執行。未來在暴風電視上，風行將以軟體的形式出現，全面接替暴風的操作系統。風行方面對此表示，「風行將獨家運營暴風的系統和廣告平台」，但強調不是收購形式，沒有股份糾葛。

騰訊正式推出全新視頻相親交友產品「歡遇」

騰訊正式推出全新視頻相親交友產品「歡遇」，產品定位是採用真實視頻聊天的方式相親。通過產品體驗看，「歡遇」核心玩法，是通過創建實時的相親視頻房間，讓男女嘉賓們進行線上交友聊天。

國際要聞

採用缺陷部件組裝 iPhone 銷售，富士康內部人員涉案

蘋果公司正在調查富士康（Foxconn）部分管理人員的一種涉嫌規模欺詐案，這些人員採用不合格的 iPhone 部件組裝 iPhone 出售，三年時間裡涉案金額達到了令人難以置信的 13 億新台幣（約合 4300 萬美元）。據稱，一名台商曾與富士康鄭州工廠的管理人員合作，獲取有缺陷的 iPhone 部件並組裝，然後以中國生產基地生產的 iPhone 產品為幌子銷售。此案大致環節是，負責銷毀有缺陷部件的管理團隊安排將這些部件賣給一個涉案集團，這個團伙將這些部件組裝成 iPhone，然後賣給毫無戒心的買家。

對此，富士康母公司鴻海發表聲明，稱已啟動調查，一旦查實絕不寬恕。

谷歌更新安卓設備間收發簡訊功能

谷歌周二宣布，將通過Messages應用讓Android設備支持RCS（富通信服務）協議。RCS本身並不是由谷歌開發，而谷歌此舉是對蘋果iMessage的回應。谷歌Messages將支持閱讀回執、全尺寸圖片和其他多項功能，但與iMessage不同，谷歌Messages不會支持端到端加密。這種技術在發送方和接收方之間的所有環節將消息加密，使信息無法被竊聽。（新浪科技）

蘋果、谷歌和亞馬遜結盟：打造智能家居統一新標準

據國外媒體報道，亞馬遜、谷歌和蘋果公司周三達成一項罕見的合作夥伴關係，旨在打造智能家居新標準，讓消費者使用起來更方便。這項合作被稱為「基於 IP 的互聯家庭項目」，致力於為智能家居創建一個新的標準，人們在購買產品時就能知道是否可以與家裡的智能系統相兼容，並且這些產品是安全的。

博通擬 100 億美元出售無線晶元業務

據外媒報道，蘋果公司長期晶元供應商博通正考慮出售其無線晶元業務，這筆交易的價值可能高達100億美元，並可能對未來的iPhone和其他蘋果產品產生影響。

作為蘋果供應鏈的長期成員，博通據說正在與瑞士信貸集團(Credit Suisse Group)合作，為其射頻（RF）部門尋找潛在買家。作為其更廣泛無線晶元業務的一部分，RF部門生產薄膜體聲諧振器(FBAR)濾波器來澄清信號，是iPhone等智能手機中使用的常見組件。（騰訊科技）

程序員專區

微軟詳述遷移到新 Microsoft Edge 的計劃

微軟即將在下個月正式推出基於 Chrome 的新 Microsoft Edge 瀏覽器，它將取代現有的 Microsoft Edge 瀏覽器，微軟詳述了新舊更替計劃：舊版本的所有開始菜單固定、磁貼和快捷方式都將替換為新版本；舊版本的任務欄固定和快捷方式替換為新版本；新版本將固定到任務欄，如果任務欄已固定了舊版本，它將被取代；新版本將添加到桌面快捷方式，如果已有舊版本的快捷方式，它將被取代；舊版本默認處理的絕大部分協議將替換為新版本處理；舊版本將在操作系統內深入隱藏，包括設置、應用、文件或協議支持對話框；啟動舊版本的任何嘗試都將重定向到新版本。微軟開始向開發者開放擴展遞交，如果已經有 Chrome 擴展，無需修改就能支持新 Edge。
