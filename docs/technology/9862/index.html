激蕩十年：美國正在衰落和正在興起的25個行業

2019-12-09 1,908條評論 3,114次閱讀 274人點贊

資本實驗室·今日風險觀察

聚焦前沿科技創新與傳統產業升級

張珂

十年，在人類歷史中只是非常短暫的一瞬。但是，十年時間又具備了足夠的力量，能夠對各行業的發展帶來劇烈變革。

在2009—2018這十年中，以互聯網、移動互聯網為依託的各種新興技術層出不窮，不僅改變了傳統的生產、工作、社交方式，更是讓傳統的產業結構今非昔比。

美國財經網站24/7 Wallst每年都會根據美國勞工統計局關於過去十年就業數據的變化，梳理出美國各行業的變化趨勢。近日，24/7 Wallst就此發布了兩份研究報告，反映出過去10年間美國正在衰落的25個行業，以及正在興起的25個行業。

數據清晰地顯示，新技術對各行業的顛覆往往以成千上萬工作崗位的消亡為代價。在過去十年間，在美國正在衰落的25個行業中，有9個行業的勞動力數量減少了一半以上，其中，水力發電、傳統沖印業務和港口航運業務的員工數量減少了七成以上。

有替代就有新生。在傳統的低效工作崗位被自動化技術替代的同時，也誕生了一系列新形式的商品和服務。

在美國正在興起的25個行業中，有20個行業的僱員人數增長了100%以上。其中，啤酒廠、互聯網與網路搜索行業的僱員數量增長了200%以上，移動食品服務、老年人和殘疾人護理服務、健康醫療保險服務、美甲服務等消費類服務則實現了150%以上的增長。

進一步分析上述數據，我們可以看到更多已經到來和即將帶來的變化：

自動化正在加速傳統行業的勞動力替代

自動化技術的普及在大幅提升工作效率的同時，也減少了對傳統重複性工作崗位的需求。過去十年中，水力發電是美國僱員數量減少最多的行業（-82.7%），但這種減少並不是因為市場需求的衰弱，而是自動化軟體普及造成的人員需求下降。

同時，港口與航運業務（-70.1%）、商業和秘書學校（-68.3%）、柑桔種植（-39.4%）、空中交通管制（-36.5%）等行業均受到了自動化技術的影響。

數字媒體與社交網路推動數字信息革命

在日常生活中，我們能感受到的最直接的變化就是互聯網帶來的數字信息革命，特別是移動互聯網與數字媒體的發展。

基於互聯網的數字媒體和流媒體業務直接衝擊了攝影、影像沖印、有線廣播影視、紙制新聞出版、實體書、印刷等傳統行業。隨著智能手機的普及，信息獲取進一步突破了傳統的物理形式與時間的束縛，實現了信息獲取和社交的實時化。

數字化大幅提升了社會效率，並賦予了我們更多的選擇權，也帶來了越來越多的空白市場和商機。

人口結構和消費傾向的變化催生新行業

隨著人口增長及老齡化加劇，以及由此帶來的社會變化已經成為全球各國都面臨的共同問題，人口結構的變化也引發了社會消費需求和行為的變遷。在美國，老年人護理及健康醫療保險服務的社會需求正在變得越來越迫切。例如，2018年美國老年人和殘疾人護理服務行業的僱員人數達到180萬人，比2009年激增了190%。

互聯網在推動數字化社會建設的同時，不僅創造了共享經濟、按需經濟、訂閱式電商、定製化生產等新型商業模式，更是帶動了日常消費傾向和消費行為的一系列變化，對舒適、健康、環保、個性化的消費需求也催生出了更多新的細分行業，如精釀啤酒、美甲、健康服務、室內農業、寵物護理等。

可以確定，除了已經產生顛覆式變革的技術，5G、物聯網、邊緣計算、數字孿生、人工智慧、區塊鏈、量子計算等層出不窮、不斷進化的新技術將進一步推動整個世界的數字化，更多的傳統行業將被數字化所顛覆，並創造出更多的新行業和新市場。

「環球同此涼熱」。即使各國技術發展水平、產業結構、經濟發展水平處於不同的階段，但美國正在衰落和興起的行業就像一面鏡子，映照出全球產業變革的未來趨勢。
