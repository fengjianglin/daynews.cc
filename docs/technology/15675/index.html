全球首個加密紀念幣即將來臨，意味著紀念幣將成虛擬加密幣

2019-12-13 347條評論 2,691次閱讀 1741人點贊

放眼世界

立陶宛共和國

將發行地表上第一枚可收藏的加密貨幣

加密貨幣？還可收藏？

沒錯！這是一枚紀念版加密貨幣

俗稱：網路紀念幣！

根據官方公告：立陶宛銀行將在2020年春季將發行全球首個加密紀念幣。該幣為紀念該國1918年2月宣布獨立及其20個簽署國而發行。

到時會有24000枚加密紀念幣在區塊鏈上發布，它們將被分為六類。

收集者將得到6個隨機的代幣碎片，一旦集齊所有6個類別，將得到一個加密紀念幣，也可以將它們兌換成一枚實物銀幣。

立陶宛官方成員幣表示該項目將為立陶宛銀行提供創建數字貨幣的寶貴經驗和知識。立陶宛希望加入目前由偉大中國牽頭的央行數字貨幣競賽中。

ps：咱們的「央媽」很可能成為地表第一個推出數字貨幣的央行。
