蘋果6K屏幕效果驚人！LED產業升級在即，京東方們迎來新挑戰

天天要聞
2019-12-19

12月11日，蘋果上線了Pro Display XDR，產品很震撼，解析度達到了6016×3384，價格更加震撼，售價為51999元起。

賣那麼貴的Pro Display XDR有多厲害呢？首先，整塊屏幕上具有576個全陣列點亮區域，顯示器解析度達到了6016×3384，能夠實現1000nit的常規亮度和1600nite的峰值亮度，以及1000000:1的超高對比度。這些指標都遠遠超過了現在市場上所有產品。

不過，作為一個科技財經博主，我們要講的不是這款產品，我們要講的是這款產品背後的產業故事。

Pro Display XDR那麼厲害，原因是它採用的是最新的mini LED背板技術。

LED我們聽得多了，mini LED又是啥？其實別說公眾了，對於產業界和資本市場來說，這都是一個新名詞。知勢研究局查閱了相關的專業資料，MiniLED顧名思義，就是尺寸更小的LED產品，具體來說，指晶粒尺寸介於500微米至50微米之間的LED。

但撇開以上這些細節，能賣5萬一台的顯示器，顯然已經指明了產業的方向，全球顯示技術龍頭公司京東方宣布，將停止LCD產能投資，轉向Mini LED。

我們知道，顯示技術近10年有一次重大技術升級換代，上一次是2009年開始LCD換代成了LED，中間歷經長達10年的產業廝殺。結果是今時今日為中國人所熟知的，中國顯示產業鏈上裡面的京東方、深天馬這些企業異軍突起，改變了日韓檯面板企業獨大的產業格局。

相比OLED，MiniLED背光LCD優勢在於能夠以更低的成本實現可比擬OLED面板的顯示效果、輕薄體驗，具備大規模量產能力，而且壽命更長、功耗更低。MiniLED背光可以搭配柔性基板實現類似OLED的曲面顯示，同時在光源可靠性方面相比OLED更有優勢！

這樣的新技術到了換代的時刻，顯然Mini LED 將帶來新需求，甚至會讓整個行業迎來新一輪大浪潮！

故事說到這裡，該說說中國廠商的反應了。

在產業上，中國廠商的動作很快，大幅提升MiniLED產能。三安光電Mini/Micro晶元已批量供應三星等品牌客戶，並已經宣布在湖北葛店投資120億擴產Mini/Micro晶元，封裝環節國星光電已量產4K/8K和背光用MiniLED燈珠產品，華燦光電今年9月26日發布新一代Mini LED晶元產品，也已經批量出貨。

但中國廠商也不是沒有劣勢。根據Digitimes梳理，目前蘋果MiniLED背光模組的供應商以日韓台廠商為主，其中晶元供應商為晶電（三安光電、日亞化學也有望參與），封裝環節供應商為台表科，模組集成環節則為瑞儀、業成和LGDisplay負責。 這裡面，可沒有中國廠商。

這也是這次技術升級如此重要的原因。相比傳統LED，MiniLED晶元尺寸越小，意味著要集成的晶元數量大幅提升，封裝難度就更大；而競爭壁壘抬高，也意味著在這一環節誰能拿下，誰就能在接下來的市場競爭中拿到份額，把對手擠出去。

上一次，中國企業為了能夠拿到市場份額放棄了利潤，得到的是規模；而這一次，中國企業要真正拼的是技術實力，要追上的也是技術上的差距。十年一遇的產業挑戰機遇，從上一代技術升級大浪潮中做到全球第一的京東方們當然能意識到，不過，這個挑戰絕不容易。
