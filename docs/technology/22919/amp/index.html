「運聯研究」快遞櫃離盈利還有多遠？

天天要聞
2019-12-19

2019年12月16日，全國快遞訂單量突破600億件。快遞櫃作為快遞件末端交付的工具，其商業價值得到體現。但目前來看，離盈利還遙遙無期。

來源 | 運聯傳媒(ID:tucmedia)，作者 | 運聯研究院 王偉

核心導讀：

1）快遞櫃的商業價值：快遞櫃已經成為末端交付的重要節點，既能提升派送效率，也解決了客戶末端交付時間差的問題；

2）2018年全國快遞櫃數量突破35萬組，快遞櫃行業競爭從「群雄逐鹿」演變成「雙寡頭」格局；

3）未來快遞櫃想要盈利，至少需要每天每組投放1.15次。

1、快遞櫃商業價值

1.1 快遞櫃已成為末端交付的重要節點

2018年，全國快遞櫃完成投遞的快件量接近44億件，全國主要企業已投入快遞櫃35萬組。快遞櫃已經逐漸成為末端交付的重要節點之一。

在快遞的交付過程中，不同的參與者職責分工各有不同。快遞公司負責中轉分撥，加盟商負責末端派送，快遞員則是最後交付環節的執行者。其中，快遞末端交付環節最為複雜，目前除了快遞員自送，投遞快遞櫃和存放代收點已成為緩解末端派送壓力的重要方式。

1.2 快遞櫃已成為末端交付的剛需

1）解決客戶時間差問題

有數據顯示，目前電商件的收件用戶中，年齡在18到35歲之間的人群的數量佔比為69%。這部分人群基本都是上班族或者學生，白天工作或者上學沒有收件時間；而快遞員又不可能把大量的快遞件壓到晚上派送，這就使雙方的收/派件時間上產生錯位。而快遞櫃的作用，就是用來調和這種時間錯位。快遞員將快遞包裹提前投遞到快遞櫃，收貨人下班後取件，根據公開數據調查顯示，使用快遞櫃取件的用戶取件時間在18點之後的佔73%。

2） 快遞櫃使得派件效率提升28.5%

今年「雙十一」，11月11日至18日全國共產生快遞件28億件，僅「雙十一」當天就處理了5.35億件包裹。這不僅考驗了快遞公司的分撥能力，末端派送方面更是功不可沒。

數據顯示，快遞櫃可以幫助快遞員提升28.5%的效率。一個熟練的快遞員每天送200單，大約需要7個小時；其中200單里大約有80單需要投遞至快遞櫃，由此可以節約2個小時，末端派送效率提升了28.5%。

2、快遞櫃企業運營現狀

到2020年，預計全國智能快遞櫃投放量將達到80萬組。，目前行業現狀已形成豐巢與速遞易的「雙寡頭」格局。

2.1 2018年快遞櫃已達35萬組

國內智能快遞櫃從2010年起步，此後迅速在國內引起熱潮。2012年起快遞櫃進入快速發展期，市場投放量不斷增加。

運聯研究院根據公開數據整理，2014年，國內智能快遞櫃市場投放量只有1.5萬組；2016年，市場投放量已達到10萬組；截止至2017年，市場投放量已增長至27.1萬組。初步測算，2018年中國智能快遞櫃投放量達到35萬組左右。按照這樣的發展速度，預測到2020年全國智能快遞櫃投放量將達到80萬組。

2.2 一、二線城市是投放的戰略要地

對於快遞櫃企業而言，快遞櫃投放城市和位置的選取直接關係到收件量。從這點來看，一、二線城市快遞櫃佔據主要市場，各大快遞櫃企業也是先布局一、二線城市，再逐步深入到三、四線城市的重點區域。

據了解，中郵速遞易目前市場投放量已超10萬組，主要分布在一、二線城市。此外，同一城市快遞櫃企業還會根據流量、位置選取人群密集度高的場所進行投放，如社區、學校、寫字樓、醫院、大型企業等。

2.3 快遞櫃三大派系對比

目前，我國開展快遞櫃業務的企業有三類：電商系、物流平台系和第三方平台。電商系以京東快遞櫃、蘇寧易購為主，主要服務於自身電商平台物流體系；物流平台下的快遞櫃企業，如豐巢科技、中郵速遞易，是其本身物流產業鏈的向下延伸，加強末端派送，以提高自身業務效率；第三方平台則以日日順樂家快遞櫃、江蘇雲櫃、上海富友等為主。

1）物流平台系：中郵速遞易、豐巢

中郵速遞易可以說是快遞櫃行業的「老人」，2012年9月由三泰控股投資成立，2017年中郵資本聯手驛寶網路（菜鳥旗下全資子公司）、亞東北辰（復星集團旗下公司）對速遞易進行戰略投資並更名為中郵速遞易，並與EMS達成合作。其在市場資源、快遞櫃數量上佔有較強優勢，據統計目前市場上快遞櫃數量超10萬組。

豐巢科技於2015年4月由順豐速運牽頭，攜手申通、中通、韻達、普洛斯成立，2018年6月申通、中通、韻達先後撤出。自成立起，豐巢基於自身資金和資源優勢迅速佔領市場，目前市場上豐巢快遞櫃數量超過15萬組，已然是快遞櫃行業巨頭。

2）電商系：京東快遞櫃

京東快遞櫃由京東集團創辦自2013年初入市場，之後不了了之。主要服務自身的電商平台物流體系，也對其他快遞公司開放。2019年2月，基於自身物流體系的服務和物流數據，京東快遞櫃重新進入市場，並且首次投放就在北京、上海、廣州、深圳、成都等城市投放了數千組，其勢頭不可小覷。

3）第三方：日日順樂家快遞櫃

作為三方平台的代表，日日順樂家快遞櫃2014年由海爾集團旗下日日順物流成立，目前市場上快遞櫃數量超過1萬台，分布在96個一、二線城市。值得一提的是，與其他快遞櫃不同，日日順快遞櫃兼具社群交互功能，不再是簡單作為物品交付的柜子。

3、快遞櫃收入成本分析

企業存在的目的是盈利，沒有哪家公司是本著做慈善公益的思想運營的。快遞櫃作為一個新物種，除了提高了貨物的交付率，為消費者和快遞員帶來方便外，還能夠為運營企業帶來什麼？

3.1 快遞櫃企業的收入細分

快遞櫃的收入，一直是外界備受關注的話題。目前，市場上的快遞櫃企業有5種收入來源：向快遞員收費、消費者端收費、廣告費、社區O2O業務及其他收入。

1）派件收入

當快遞員將包裹投遞到快遞櫃時，需要支付的費用。以豐巢為例，小格3.5毛、中格4毛、大格4.5毛；以菜鳥驛站為例，小格3毛、中格和大格3.5毛。

2）寄件收入

這部分收入來自企業向消費者收取的超時費、寄件費：比如包裹在快遞櫃內存放超過24小時，取件時會向消費者收取相應的超時費用；客戶使用快遞櫃寄件時，需要支付手續費。

3）廣告費

廣告載體是快遞櫃企業主要的商業賣點。以豐巢為例，其目前推出的廣告形式有5種，包括品牌櫃貼、液晶顯示屏、線上手機端廣告、線上廣告、樣品派發。

4）社區O2O業務

目前，在各家快遞櫃企業都未盈利的情況下，社區O2O業務最被看好。豐巢聯手順豐曾推出「微倉模式」，據了解，這種模式下消費者完成線上支付後，可直接從快遞櫃內取出所需商品，真正實現「一手交錢，一手提貨」。

運聯研究院認為，未來智能快遞櫃或將會成為社區O2O業務的主要入口，並得到拓展，比如開拓社區金融、醫療、生鮮洗衣、繳費等功能。

5）其他收入

目前部分快遞櫃企業開放了加盟，加盟商需要支付加盟費用、快遞櫃成本、運維費用等。對快遞櫃企業而言，開放加盟能減少成本投入，解決資金占用，同時利用技術和品牌賺取收入。

3.2 快遞櫃的成本細分

自2012年開始投入運營，至今還沒有一家快遞櫃企業盈利。對快遞櫃企業來說，需要持續關注成本項。

快遞櫃成本項主要分為固定成本和變動成本。其中，固定成本包括櫃體成本和場地維護成本；快遞櫃的成本約為1.8萬元至5萬元，場地維護成本約2000元。變動成本主要包括場地租賃成本，以及網費、電費、維修等運維成本；場地租賃成本約0至7000元不等，網電運維成本約2500元。

若按照快遞櫃的使用壽命10年折算，每年的快遞櫃成本約為4000元，場地成本約5000元，場地維護成本約200元，網費、電費等運維成本約2500元，則每組快遞櫃每年成本約11700元。這其中，場地租賃成本佔總費用的43%，因此，快遞櫃企業可以從這方面研究降本的方法。

3.3 每組快遞櫃投放1.15次才能夠盈利

我們可以粗略地算筆賬，假定快遞櫃只靠快遞員投遞賺取收入，單個格口價格按0.35元計算，每組快遞櫃80個格口，一年按365天計算，每組快遞櫃一年固定成本11700元；那麼，需要多少投遞量才能達到盈虧平衡？

計算後可以得出，每天需要投遞92個格子(11700÷365÷0.35≈92)，每組快遞櫃每天需要投遞1.15次(92÷80=1.15)以上，才能盈利。
