特斯拉Model 3車載電腦網關被破解，出現數據泄露

2019-12-23 723條評論 1,651次閱讀 482人點贊

特斯拉於電動汽車行業而言，就像蘋果於智能手機，正是特斯拉首次嘗試使用鋰離子電池作為汽車的動力電池，從而帶來了電動汽車的革命。這個已經將近20歲的電動汽車公司，也終於開始賺錢，實現穩定的收益，甚至把工廠開到了我國上海。在2018和2019這種世界級的車市寒冬背景下，特斯拉卻是一路披荊斬棘，不斷突破和探索。

歷年全球的汽車生產量（中汽協數據）

在全球汽車產量整體下滑的情況下，特斯拉上海工廠建成投產，國產Model 3實現提前交付。上海工廠一期設計產能為15萬輛/年，考慮到產能爬坡，預計2020年內的實際產能約為11.7萬輛。不得不說，特斯拉給中國本土的車企帶來了巨大的壓力，特斯拉到中國建廠，就是典型的「引狼入室」。特斯拉電動汽車業務已實現穩定盈利，其主要業務內容變更

特斯拉上海工廠

特斯拉目前在上海生產的車型為Model 3，根據汽車之家的數據，該款車配備的動力電池為60KWh，而進口版的車型動力電池的容量則為75KWh。

特斯拉 Model 3

不管是60KWh還是75KWh，顯然特斯拉有可能準備在後期將其容量提升到100KWh。一位名為Green的外國黑客破解了特斯拉車載的電腦網關，釋放了一些關於特斯拉Model 3配置相關的信息。在這些未經特斯拉官方確認的信息中，包含了很多微小和重大的信息，其中就有為以後升級100KWh電池的配置信息以及狂暴模式（ Ludicrous mode，外語不好，如果翻譯錯誤請更正）的信息。

特斯拉 Model 3 會有狂暴模式嗎

傳說，特斯拉本來引進了一種新的車載電腦網關用來存儲配置相關的數據。以前，特斯拉的網關電腦都是使用文本文件來儲存這些配置信息的，而新的網關則是通過不透明的二進位數據來配置寄存器，從而配置整車。但是在Model 3的配置中，不知道為啥，又用回來這種文本文件配置的方式，所以我們這位Green老哥就想到了一種方式來破解文本文件中各個參數的值的方法。結果還真對上了：
