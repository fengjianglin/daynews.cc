Linux發行版對比：Red Hat與Ubuntu

2019-12-16 370條評論 1,327次閱讀 52人點贊

Red Hat Enterprise Linux(RHEL)和Ubuntu是企業中最流行的兩種Linux發行版。儘管如此，這兩種Linux版本之間仍然存在一些關鍵差異。當在評估Linux發行版時，功能、用戶體驗、許可和文檔是關鍵考慮因素。

RHEL可以支持幾乎任何應用程序伺服器或基礎架構角色。在其最新版本中，Red Hat似乎側重於安全性和合規性。該公司新增了系統範圍的加密策略、高級審核功能，以及更新的協議，包括傳輸層安全協議(TLS)、IPsec、域名系統安全性擴展協議和Kerberos。

Red Hat還降低了RHEL最新版本的複雜性。RHEL 8使用相同的管理工具來提供一致的用戶體驗，無論伺服器運行在雲端、在VM還是在裸機中。

Ubuntu是一台多功能伺服器，它提供身份驗證、域名伺服器和動態主機配置協議等服務。 Ubuntu也可以配置為Web伺服器、MySQL或PostgreSQL資料庫伺服器、文件伺服器、聊天伺服器或電子郵件伺服器。

Ubuntu還提供下一代功能，包括針對CPU、內存和存儲的服務質量控制，以及安裝快照-使管理員可以輕鬆地安裝應用程序。

Canonical公司是提供Ubuntu服務和管理的公司，該公司是託管OpenStack的頂級提供商。因此，Ubuntu很適合想要運行OpenStack的企業。

架構

Red Hat在多種架構支持RHEL。該公司的主要架構是x86，同時也有針對IBM Power系統和IBM System z的RHEL版本。該公司還提供可在64位ARMv8-A架構運行RHEL版本，該版本目前提供開發者預覽版。

Ubuntu也可用於多種設置。默認支持x86支持，Ubuntu也可用於Arm、IBM Power Systems和IBM System z。

易用性

在對比Linux發行版時，易用性是重要考慮因素。Red Hat似乎做了很大的努力來改善RHEL 8的整體可用性。

在此發行版中，Red Hat引入新的基於Web的管理控制台，以簡化各種伺服器管理和監視任務。RHEL 8還包括Red Hat Insights，該功能旨在檢測並幫助修復伺服器上可能發生的任何內部問題。

眾所周知，Ubuntu是最容易學習的Linux發行版之一。Ubuntu易於安裝，並具有直觀的GUI界面，比命令行界面更易於使用。Ubuntu Linux會自動安裝所有必需的設備驅動程序。 Ubuntu還提供廣泛的社區來解答用戶問題。

許可

Red Hat出售八種不同的許可證類型，企業可以直接從Red Hat網站購買。最便宜的選擇是Red Hat Enterprise Linux Server許可證，單個RHEL實例的售價為349美元。最昂貴的選擇是針對IBM System z的Red Hat Enterprise Linux Server，每個實例的售價為15,000美元。

Ubuntu的Linux發行版不收取許可費，但Canonical公司提供名為Ubuntu Advantage的安全和修復服務，這項服務的伺服器費用為每年225美元，虛擬機費用為每年75美元。該支持計劃的桌面版價格為每年25美元。

文檔

在對比Linux發行版時，另一個重要考慮因素是文檔。Red Hat提供HTML、ePub和PDF格式的在線文檔，並具有針對RHEL的完整培訓和認證計劃。
