打破國外壟斷，國內唯一的X86架構BIOS 和 BMC 固件供應商

公司業務分為雲計算設備核心固件業務、雲服務業務兩塊。公司營收結構如下圖：

其中雲計算設備核心固件業務主要向英特爾，華為、聯想等 CPU、雲計算設備廠商提供固件技術開發服務、及向客戶銷售固件產品（固件授權使用費）。

目前全球主流的 X86 架構 BIOS 固件產品和技術，長期壟斷在美國的 AMI、Phoenix，中國台灣的 Insyde 三家公司手中。在國際廠商處於壟斷地位的情況下，公司不僅是國內唯一的X86架構 BIOS 和 BMC 固件供應商，還是少數同時具備開發 ARM、MIPS、Alpha 等 CPU 架構用 BIOS 產品能力的大陸廠商。在 BIOS 固件產品開發領域居於國內領先地位。2019 年 1 月，公司獲得英特爾全球技術服務商資格，成為繼 AMI、Insyde 後的第三家固件廠商。

自主可控—國產核心固件需求不斷增加，伺服器+PC固件合計市場規模在76億元

啥是核心固件？就是一組固化到計算設備主板上一個存儲晶元中的系統程序。

PC、伺服器和 IoT 等雲計算所需的計算設備均需要安裝 BIOS 固件，而伺服器還需安裝 BMC 固件以配合 BIOS 固件工作。可以說BIOS、BMC 固件為計算設備連接 CPU 和操作系統的橋樑，具備有效降低信息安全風險的能力，在目前 CPU、操作系統等國產化進程相對緩慢的情況下，隨著國家信息安全戰略的推進，使用國產 BIOS、BMC 固件產品的計算設備不斷增加。

按照每台X86 架構伺服器 BIOS 固件及 BMC 固件 200-300 元去估算，伺服器的固件市場規模在37億元；

按照每台 X86 架構 PC 的 BIOS 固件 10-15 元去估算，PC的固件市場規模在39億元；

此外IoT 設備也在快速增長，預計 2020 年全球 IoT 設備數量將達到 204 億台。

2018 年我國 X86 伺服器出貨量為 330 萬台，同增 26.1%，預計到 2023 年出貨量將超過525 萬台，未來 5 年整體市場年複合增長率（CAGR）將達9.7%。

此外搭載海光、兆芯、飛騰、瀾起、申威等國產晶元廠商的計算設備自 2018 年四季度起陸續量產出貨，公司核心參與了以上國產晶元配套 BIOS 的研發。目前公司研發的多架構國產 BIOS 固件的技術性能和產品穩定性已在聯想筆記本和伺服器、曙光伺服器、華為伺服器等項目上得到了充分驗證。

英特爾既是公司第四大股東也是公司主要核心固件客戶 受益國產晶元需求增長公司核心固件營收2019年上半年同比大增55.88%

一般來說首先要為 CPU 廠商提供技術開發服務，只有通過較長時間的服務才能開發出適配 CPU 的成熟固件產品，從而實現向計算設備廠商的批量化銷售。

相比美國的 AMI、Phoenix，中國台灣的 Insyde三家境外老牌廠商，公司成立時間較短，在銷售規模及市場佔有率上還有很大差距，且現階段公司固件業務收入主要是技術開發服務收入，毛利率水平相對較低在45-55%之間，不過固件技術開發在核心固件業務的收入佔比已從2017年的83.76%下滑至2018年的73.23%。而高毛利率的固件產品營收佔比已提升至26.77%，其毛利率超82%。

2018、2018年公司雲計算核心固件營收分別同比增長13.08% 、23.84%，2019年上半年受益於自主可控政策推動國產固件需求增長，公司固件產品支持的龍芯、飛騰、申威等國產晶元陸續被列入中央機關採購名單，從而使得公司雲計算核心固件同比大增55.88%。

此外公司陸續承擔了華為海思 ARM 和 X86 伺服器晶元、上海瀾起的「津逮」 平台、海光「禪定」晶元的 BIOS 和 BMC 固件開發工作。


截止2019年7月底公司固件業務訂單達8,426.62 萬，國產固定產品訂單達4,161.47 萬。下遊客戶上英特爾在公司核心固件業務的營收佔比較高，2016-2018年分別達57.49%、61.27%、45.66%。與此同時英特爾也是公司第四大股東，持股比例為4.8%。


華為為公司固件業務的第二大主要客戶，2016-2018年營收佔比分別為10.52%、10.61%、13.14%。

雲服務規模較小 宜興區域營收佔比超七成

公司雲服務業務收入主要向政企客戶提供政企雲（以縣、區、鎮、科技工業園區為主，業務下沉）及物聯網雲服務（主要聚焦安防、環保領域），公司目前主要專註於江蘇省內，該領域主要同行業公司有萬達信息、南威軟體（公安系統）、華宇軟體（司法、檢察系統信息化）、銀信科技（金融領域）。從雲服務營收規模來看公司與同業仍有較大差距，僅為萬達信息的20分之一

公司雲服務多年來主要面向宜興區域政企客戶， 2016-2018年來自宜興的營收佔比分別為90.39%、88.88%、71.83%，到2019年上半年進一步下降至60.12%。另一方面來自江蘇省外的營收佔比從2016年的0.66%大幅提升至2019年上半年的20.67%，未來將不斷向江蘇省外推廣複製，優先拓展浙江、安徽、上海等華東省市縣域經濟比較發達的地區。截止2019年7月底公司雲服務業務累計訂單為1.7億元，雲服務業務訂單充足，PaaS 平台技術已形成組件化開發能力。2019年上半年公司雲服務營收同比大增67.15%。

四季度營收佔比較高 2019年業績增速較高

公司收入存在季節性波動，因為客戶結構的原因下半年營收佔比較高，2017、2018年公司四季度營收在全年佔比分別高達45.2%、51.95%。

2019年前三季度公司營收為1.33億，同比大增56.58%；扣非凈利潤為2527.92萬，同比大增51.64%。公司預計2019年營收為2.3-2.4億，同比大增30.91%-36.6%；扣非凈利為6200-6800萬，同比增長46.03%-60.16%，主要是因為固件業務的技術水平持續提升和雲服務業務的拓展。
