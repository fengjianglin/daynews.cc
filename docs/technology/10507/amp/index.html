高通實施壟斷被罰款8.73億美元 韓國法院判定合法

天天要聞
2 週 ago

作者:宋星

【TechWeb】12月4日消息，據國外媒體報道，韓國一家法院周三判定，反壟斷執法機構以強迫簽訂不平等合同為由向高通公司徵收8.73億美罰款的監管行為合法。


高通

針對高通向韓國公正交易委員會提起的訴訟，韓國首爾高等法院作出如上判決。

韓國公正交易委員會2016年3月認定，高通方面將數據機晶元組的供應和專利權掛鉤不合理，壟斷專利權，處以罰款並責令改正。

外媒表示，目前還不清楚，高通是否會向韓國最高法院上訴。

今年7月，歐盟反壟斷部門也對高通公司處以2.42億歐元（約合2.7億美元）罰款，稱其阻礙基帶晶元市場競爭。歐盟反壟斷事務負責人Margrethe Vestager周四表示，高通濫用自身的主導地位，導致競爭對手Icera無業務可做，阻礙了基帶晶元市場上的競爭。高通並不認同這種說法。

並且，這還不是高通第一次被歐盟罰款。去年1月，高通被歐盟反壟斷監管機構處以9.97億歐元(約合12.3億美元)罰款，因其向蘋果付費，以換取蘋果獨家使用高通晶元，排擠英特爾等競爭對手。

周二收盤，高通(NASDAQ:QCOM)股價下跌2.27%至80.58美元，總市值約920.10億美元。
