武漢配送員刺死店員：為什麼真相還在穿鞋，謠言已經滿天飛了？

2019-12-24 792條評論 2,625次閱讀 714人點贊

殺人等惡性刑事案件，本來就是容易引人關注的，外賣配送行業更是和網民們息息相關，是廣大網民關注的熱點，幾乎每個人都有和快遞小哥、外賣小哥的故事要講。兩者結合，那就是一個吸引眼球的大新聞。

一美團外賣小哥在街頭送餐（圖文無關）。 （新華社記者 薛宇舸/圖）

2019年12月22日，武漢市內一家商場發生了一起殺人案，一名美團配送員持刀將商場一名工作人員刺死。事發後，許多自媒體紛紛發文，稱事件疑似是因為配送員送外賣遲到，顧客要求賠償並投訴，引發口角，引起配送員不滿後衝突殺人。一時之間，武漢外賣小哥疑因差評殺人的消息在社交媒體瘋傳，還引起了該不該給外賣小哥差評的大討論。甚至有機構媒體的自媒體賬號也發起了什麼情況下會給外賣小哥差評的投票。

但是，武漢警方當天下午發布的警情通報只是確認發生了外賣配送員與人口角而後刺死人的事件，並沒有說明到底是什麼原因導致口角。美團公司23日上午發表聲明稱，事件並非因為送餐遲到被投訴而引發，這一訂單的商戶用戶沒有差評，也沒有電話投訴記錄，而且涉事配送員當時也不是去送餐，而是執行同城配送業務時到超市取貨品與店員發生口角。

那些聲稱「疑似因差評而殺人」的自媒體文章是從哪裡得到的信息呢？即使是「疑似」，也要有一定消息源才行，不能憑空猜測。翻了很久，發現在某社交平台上，一位用戶以涉事配送員同一配送站點的配送員的口吻發布了一條回復，說從站點處了解到的情況，是外賣送慢了8分鐘，客服和送餐員與顧客商量結果是讓送餐員賠償35元餐費，但是送餐員到顧客處後，顧客要求額外賠償50元，不然就投訴，併當面撥打了投訴電話。而被投訴就會罰款500元，送餐員情急之下，抽刀殺人。這一消息看起來有鼻子有眼，像是了解內幕的人發出來的。但是，各個自媒體可能並沒有去認真核實發帖人的真實身份，也沒有向警方和美團核實，就如獲至寶發了出來，「疑似」就越傳越廣了，似乎變成事實了。

自媒體們之所以不認真核實就搶著發疑似消息，原因是要博眼球、引關注。殺人等惡性刑事案件，本來就是容易引人關注的，外賣配送行業更是和網民們息息相關，是廣大網民關注的熱點，幾乎每個人都有和快遞小哥、外賣小哥的故事要講。兩者結合，那就是一個吸引眼球的大新聞。所以，事件一發生，就有大批媒體自媒體搶著蹭熱點，連基本的信息核實都顧不上了。不客氣地說，這就是吃人血饅頭。

無疑，外賣配送行業的管理制度，配送員與顧客間的矛盾衝突，都是配送行業需要解決的問題。但是在事實還不清楚的時候就以「疑似」為標題大發文章，甚至讓網友就站配送員還是撐顧客站隊，只會更加激化二者的對立，對解決問題卻未必有利。

辛省志
