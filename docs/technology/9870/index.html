中國電信啟動總部機構改革序幕 網運部與IT部合併

2019-12-09 119條評論 1,187次閱讀 179人點贊

運營商財經網 康銳 /文

知情人士獨家向運營商財經網透露，中國電信最大部門之一的網路運維部將分拆重組，今天下午宣布。同時，這也意味著中國電信的總部機構改革拉開序幕。


兩個部門今天宣布合併

中國電信的網路運維部、網路與信息安全管理部原來合署辦公，即兩個牌子一套人馬，多達12個處，包括綜合管理處、業務網路維護管理處、數據網路維護管理處等等。

這麼大的部門導致部門領導都比別人多，有一正三副，包括部門總經理李紅斌、部門副總經理諶剛、張強、林睿，還有之前的部門總經理孫小紅，但已經轉資深經理了。而別的部門領導有的只有一正一副。

而此次改革後，新成立網路和信息安全管理部，該部門涉及網路安全的人員劃轉過去；而涉及網路運維的人則與企業信息化事業部合併。相當於網運部一分為二。

企業信息化事業部（數據中心），也是一個規模較大的部門，有10個處或中心，就光數據中心就分北京和上海兩部分，每個數據中心都不少人，

企業信息化事業部總經理目前為馮傑，副總經理有胡志強、黃禮蓮、王桂榮。其中，胡志強的實職是中國電信雲計算公司總經理，他只是同時兼任企業信息化事業部副總經理。


落實「央企總部改革」的一部分

運營商財經網獲悉，中國電信的網路運維部與企業信息化事業部合併後，總經理仍將為馮傑。

此次改革是中國電信「央企總部改革」的一部分，而中國電信已成立相關工作組進行整體規劃和落實，之前，中國電信辦公廳更名為辦公室，其實也是落實「央企總部改革」的舉措之一。

此次中國電信率先將網路運維部門與企業信息化部門合併在一起意義深遠。目前中國聯通總部的內部架構設置基本與中國電信一樣，中國聯通的網路運維部總經理為馬紅兵，中國電信的總部機構改革是否會對中國聯通產生影響，則還不好說。

中國移動的架構設置與中國電信、中國聯通不完全一樣，受影響則小些。


其他人事變動傳聞不一定屬實

另外，近日部分自媒體號的報道稱，「中國電信網發部總經理張新將任電信聯通5G共建共享中心主任。企信部總經理馮傑將任雲網發展部總經理。中國電信系統集成公司總經理劉志勇任雲網運營與信息化部總經理。浙江電信總經理韓臻聰任電信系統集成公司總經理。重慶電信總經理龔勃任浙江電信總經理。中國電信雲公司總經理胡志強任江西電信總經理。網運部總經理李紅斌任集團網信安部總經理，不再兼任網運部職務」。


運營商財經網之前已經澄清沒有做過這方面報道，同時，再次強調，這個人事變動的傳聞有一部分不屬實，真真假假摻雜在一起，不可全信。例如，其中，關於中國電信雲計算公司總經理胡志強將調任江西電信總經理一事，據悉，沒有此事。
