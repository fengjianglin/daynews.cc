浩雲科技（300448）

基本概要

基本面分析：浩雲科技股份有限公司的主營業務為安防業務，包括金融安防系統、安防設備銷售、防盜報警工程、閉路電視監控系統等集成系統以及公共安全大數據的處理服務。

區塊鏈 2019-04-02公司互動平台稱公司和廣州大學共建的公共安全智能研究院也有相關的區塊鏈技術，目前也將逐步應用在公司相關業務中。

國產軟體 國內金融安防業具有較強實力的個性化整體解決方案提供商之一;主營業務為金融安防系統設計、集成和運維服務,以及相關軟、硬體產品的研發、生產和銷售;客戶主要是銀行業金融機構,包括大型商業銀行、股份制商業銀行、城商行、農信社、農商行、農村合作銀行、政策性銀行、郵政儲蓄等金融機構。

華為海思概念 2019年6月19日互動平台回復稱公司核心技術及演算法都基於公司自研或與國內公司合作。應用晶元大部分已在幾年前就已進行國產化布局。核心晶元與海思、君正等國內優秀晶元廠家合作。

K線圖

看點：

1、金融物聯
金融物聯是公司的傳統業務領域，從安防集成發展到綜合物聯再到智慧物聯，公司在金融物聯持續投入、不斷創新，發展到今天已成為金融物聯領域智慧物聯綜合解決方案提供商。公司金融物聯服務主要包括：智慧物聯管理平台、智能感知產品、智能視頻設備、智能控制設備、智能語音設備等。

2、公共安全領域
在公共安全領域公司主要為客戶提供以人、車、物、平台為核心的全套公共安全解決方案，具體主要包括視頻監控智慧大數據運營服務、9 X綜合治理平台、公共安全視頻圖像信息交換共享平台、車輛大數據應用系統、城市監控系統、智能終端以及智慧城市安全綜合管理平台等產品及服務。

3、主力控盤

前十大流通股東機構成本估算 持有1.11億股，占流通盤26.39%，主力控盤度較低。
