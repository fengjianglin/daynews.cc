騰訊愛奇藝修改超前點播規則：單集3元，多看6集

2019-12-18 1,909條評論 4,580次閱讀 3158人點贊

12月17日，在全網爭議之下，騰訊視頻、愛奇藝紛紛修改超前點播規則，騰訊視頻會員可提前看6集權益不變，但可享額外3元每集的超前點播權，不再是支付額外50元觀看6集的設定。

愛奇藝同樣修改了超前點播規則，VIP可在提前觀看6集的基礎上，以每集3元的價格再多看6集。面對修改規則之後，一些網友表示，「3元的價格還是可以接受。」但也有一些網友表示，在VIP會員基礎之上收費還是不合理。



就在今天，兩家視頻網站也針對此事進行回應。騰訊視頻總編輯王娟表示，這次《慶余年》引發的爭議系對會員的告知以及消費心理的把握上「不夠體貼」。她強調未來騰訊視頻將進一步優化並提升會員服務體驗。

愛奇藝副總裁、自製劇開發中心總經理戴瑩今日回應《慶余年》超前點播爭議時表示，視頻平台的內容在越來越多元，用戶的需求也變得更多元。「我們的初衷是想滿足用戶更多元的內容需求，但可能沒太做好，未來希望能更多考慮到用戶的心理，做好排播的設計和告知工作。」

即使如此，超前點播的模式依然引起巨大爭議，在兩家平台不約而同開啟超前點播服務，只要在會員的基礎上再付費50元，就可以提前解鎖6集內容這一規則之後，人民日報更是針對此事評論指出，VIP之外設置VVIP，額外掏錢才能享受超前點播，視頻網站是在製造焦慮誘發用戶消費。

雖然目前視頻網站還處於巨額虧損之中，但兩家的會員已經超過1億，會員模式成為兩家視頻網站的主要收入來源，在解決平台持續虧損、盈利難的問題中，也要兼顧會員的利益。這種情況下，視頻網站就必須要探索下一個新的增長點。此除了出品好的內容之外，視頻網站也需要探索多元化發展，探索出一條良好健康的商業模式才是當務之急。
