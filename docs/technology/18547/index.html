盤點世界上發明專利最多的人擁有最多專利的發明家喬布斯上榜

2019-12-18 86條評論 3,756次閱讀 1326人點贊

專利一般是由政府機關或者代表若干國家的區域性組織根據申請而頒發的一種文件，獲得專利的發明創造在一般情況下他人只有經專利權人許可才能予以實施。那麼世界上發明專利最多的人有哪些？有托馬斯·阿爾瓦·愛迪生、尼古拉·特斯拉、迪恩·卡門、阿爾弗雷德·貝恩哈德·諾貝爾、史蒂夫·喬布斯、侯聖春、中松義郎等等。下面一起來看看吧！

世界上發明專利最多的人榜單，主要根據各發明家以獲得的發明專利數量，並綜合參考互聯網相關排行榜/榜單進行推薦，榜單僅供參考，幫助您了解擁有最多專利的發明家。如有疑問，歡迎評論/批評指正。

世界上專利最多的發明家

1、愛迪生（1000多項）

托馬斯·阿爾瓦·愛迪生，出生於美國俄亥俄州米蘭鎮，逝世於美國新澤西州西奧蘭治，是發明家、企業家。愛迪生是人類歷史上第一個利用大量生產原則和電氣工程研究的實驗室來進行從事發明專利而對世界產生深遠影響的人。他發明的留聲機、電影攝影機、電燈對世界有極大影響。他一生的發明共有兩千多項，擁有專利一千多項。

2、特斯拉（700多項）

尼古拉·特斯拉，塞爾維亞裔美籍發明家、機械工程師、電氣工程師。他被認為是電力商業化的重要推動者之一，並因主持設計了現代交流電系統而最為人知。特斯拉每天只睡2個小時，最終獨自取得700多項發明專利。他的多項相關專利以及電磁學的理論研究工作是現代的無線通信和無線電的基石。除了是一位科學家，他還是詩人、哲學家、音樂鑒賞家、語言學家。

3、迪恩·卡門（400多項）

迪恩·卡門是美國工程師，發明家和商人，以發明賽格威而聞名。1951年狄恩·卡門出生於紐約，5歲時就開始搗鼓一些小玩意兒，曾考入伍斯特理工學院，這可是僅次於麻省理工學院的理工科院校，兩年後退學創立AutoSyringe公司。他被稱為現代版愛迪生，擁有400多項個人專利，預估可以拯救全球10億人的生命。因為這些發明，他的身價高達5億美元，與比爾·蓋茨、喬布斯一起，入選”全球十大輟學億萬富翁”。

4、諾貝爾（355項）

阿爾弗雷德·貝恩哈德·諾貝爾，是瑞典化學家、工程師、發明家、軍工裝備製造商和炸藥的發明者，1833年10月21日出生於斯德哥爾摩，1896年12月10日逝世。諾貝爾一生擁有355項專利發明，並在歐美等五大洲20個國家開設了約100家公司和工廠，積累了巨額財富。1895年，諾貝爾立囑將其遺產的大部分作為基金，將每年所得利息分為5份，設立諾貝爾獎，分為物理學獎、化學獎、生理學或醫學獎、文學獎及和平獎5種獎金，授予世界各國在這些領域對人類作出重大貢獻的人。

5、喬布斯（324項）

史蒂夫·喬布斯，出生於美國加利福尼亞州舊金山，美國發明家、企業家、美國蘋果公司聯合創辦人。喬布斯被認為是計算機業界與娛樂業界的標誌性人物，他經歷了蘋果公司幾十年的起落與興衰，先後領導和推出了麥金塔計算機、iMac、iPod、iPhone、iPad等風靡全球的電子產品，深刻地改變了現代通訊、娛樂、生活方式。據了解，隨著美國專利商標局的批准，喬布斯作為發明人註冊的專利於2016年已經增加到了324項。

6、侯聖春（300多項）

侯聖春，1972年2月出生於中國山東省汶上縣，現居于山東省棗莊市，是中國發明協會會員，濟寧發明協會會員，連年被評為科技創新先進個人，並被評委棗莊市十佳校外科技教育專家。侯聖春擁有600多項發明，已被國家知識產權局授權300餘項專利。成為當地獲得國家專利最多的個人，贏得了民間”發明大王”的稱號。

7、中松義郎（290項）

中松義郎，男，是一名日本的發明家。1928年6月26日，中松義郎出生於一個極有教養的家庭。外祖父曾留學美國，非常喜歡機械。為了培養小中松對機械的興趣，他把家中所有帶機械結構的東西都拆得七零八落，然後再一一組裝起來給小中松看。宣稱擁有3200項發明，遠超過愛迪生；事實上以取得專利數來看，愛迪生有1093件，中松只有290件。
