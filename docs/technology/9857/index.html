文博會上的新技術：中醫+大數據來自機器人的“望聞問切”

2019-12-09 116條評論 1,294次閱讀 55人點贊

市民體驗機器人問診。


生活晨報全媒體記者 張娜 太原報道 在第四屆山西文博會上，有一款有意思的物件，值得老同志去了解了解，那就是智能中醫健康機器人。站在它面前，只用兩分鐘，就可以拿到一份中醫體檢報告。

機器人大夫檢測評估體質

　　　　

由山西慧虎科技有限公司研發的智能中醫健康機器人在山西綜改示範區展區展出。「這是幹什麼的？怎麼弄？」60歲的市民楊天成在智能中醫健康機器人前駐足，想體驗一下機器人如何看病。
　　　　

眼前的這個機器人大夫，有四個「診療方法」。首先是面診。楊天成將臉準確定位在面部區，機器人就採集了他的面色、光澤度、眼神、唇色等信息。
　　　　

然後，機器人大夫要看舌苔，楊天成對著機器人上的攝像頭把舌頭伸出，並使電腦中的影像定位在「舌區」。之後機器人大夫開始為楊天成「把脈」，他把左右手分別放到機器人上的指定位置。
　　　　

待機器人經過「望診」之後，楊天成在工作人員的提示下，掃描二維碼關注「慧虎健康」微信公眾號，不到兩分鐘就拿到了自己的體檢報告。這份體檢報告不僅分析出了楊天成的體質是陽虛質，對體質進行了評估，還對面部、皮膚進行了分析，並提出了一系列的養生方法和養生禁忌。

智能機器人開出的體檢報告

　　　　

見機器人大夫的工作效率如此之高，很多市民對它興趣頗濃，一個接一個地站在它面前「問診」。

將老中醫經驗作為大數據

　　　　

科技已經深深改變了我們的生活，機器人在西醫微創手術中的應用並不鮮見，但在中醫領域，它是如何做到分析體檢報告的？「傳統中醫講究『望、聞、問、切』，我們是將其中的『望診』拿出來做文章。以老中醫豐富的理論和經驗為大數據基礎，將採集到的用戶信息進行全面分析，並對用戶體質做基於中醫理論的全面健康評估，給出個性化調養建議。」工作人員解釋。
　　　　

智能中醫健康機器人創始人王華虎因為自己健康指數偏低，經常生病，通過中醫調理一段時間後，痊癒，因此對中醫充滿崇拜。於是，在不斷研究各種中醫養生知識的同時，王華虎決定將中醫與互聯網、人工智慧相結合，讓傳統中醫科技化、便捷化。
　　　　

王華虎說：「智能中醫健康機器人的誕生解決了現代社會的很多痛點，比如沒時間看中醫，想要調理卻找不到好的老中醫等。」他希望有一天，智能中醫健康機器人能成為現代社會人的健康管家。
　　　　

據了解，慧虎智能中醫健康機器人已取得3項授權專利，18項受理專利，20項授權軟體著作權。目前已投放到省城一些社區醫院、中醫會所，為市民「望診」。
