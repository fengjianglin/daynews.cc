2020年繼續網路提速降費 提速降費有套餐降價嗎？

2019-12-24 528條評論 1,555次閱讀 570人點贊

　　好消息：關注微信公眾號「中研網」，回復「免費報告」，即可獲取一份免費報告，速速來取!

　　5g網路什麼時候能全國覆蓋呢?近日國家相關部門有了答覆。12月23日，全國工業和信息化工作會議在北京召開。對2020年重點工作進行部署時，會議提出，力爭2020年底實現全國所有地級市覆蓋5G網路;繼續做好網路提速降費。會議表示，要著眼融合發展，拓展數字經濟發展新空間。堅持智能製造主攻方向，持續深化人工智慧、區塊鏈、物聯網、大數據等技術創新與產業應用，推動工業化和信息化在更廣範圍、更深程度、更高水平上實現融合發展。

　　2020年優化提升網路供給質量

　　會議要求，2020年，將優化提升網路供給質量，深化電信普遍服務，重點支持邊疆和偏遠地區網路深度覆蓋。繼續做好網路提速降費，在扶貧助殘領域實施精準降費。持續提升IPv6網路質量和服務能力，支持IPv6在5G、工業互聯網、車聯網等領域融合創新發展。

　　實施「5G+工業互聯網」512工程。加快製造業數字化網路化智能化轉型，持續推進車聯網、工業互聯網及人工智慧創新應用先導區建設。高度重視中小企業智能化改造。

　　2019年提速降費成績單

　　預計全年，全國規模以上工業增加值增長5.6%左右，單位工業增加值能耗同比下降3%左右，軟體和信息技術服務業收入增長15%，電信業務總量(上年不變價)增長20%，互聯網行業收入增長20%。

　　《工業和信息化部關於進一步規範電信資費營銷行為的通知》，要求電信企業必須落實資費「清單式」公示等相關工作;二是充分接受社會公眾、媒體和用戶的監督，聽取改進服務的意見建議，更好地規範電信企業營銷行為，提升服務質量。

　　提速降費有套餐降價嗎

　　電信在各地都推出了合適的4G套餐，並且在價格上也嚴格按照國家提出的降費要求，此外對於套餐外流量價格也做出了巨幅下調，最高不會超過0.03元/MB。國內三大運營商已經提前超額完成任務，2018年全年共計讓利超過1200億。對於用戶，最直觀的感受就是流量資費的下降。平均流量資費從2017年的28元每GB，到2018年的不足10元每GB，移動流量降費幅度超過90%。

　　提速降費方案目標

　　中國移動：預計到年底，流量綜合資費同比將下降35%以上，網路能力和網路速率將大幅提升;

　　中國電信：中國電信預計流量資費平均降幅將達30%左右。寬頻免費提速，每Mbps價格降幅近4成;

　　中國聯通：降低全網移動用戶數據流量綜合單價20%以上，有線寬頻套餐降低了近1300元。

　　從三家運營商的提速降費清單總體看來，其中移動流量和有線寬頻資費平均計划下降30%左右，網速全面提升，主要城市城區提升至100兆，基本與工信部公布的提速降費目標相匹配。

　　2018年提速降費發展歷程

　　2018年3月5日，2018年政府工作報告再次敦促提速降費，要求年內取消流量「漫遊」費，且移動資費年內降低至少30%、明顯降低寬頻費用。

　　中國電信、中國移動港股股價分別下跌2.05%、3.17%，中國聯通A股股價跌0.93%、港股跌5.93%。資本市場擔心，持續的提速降費會進一步影響運營商營收。

　　2018年5月17日世界電信與信息社會日，北京聯通宣布啟動第七次寬頻提速降費，家庭寬頻用戶主流速率將達到200M，企業寬頻用戶同步提速。據悉，此次寬頻提速工作將分批實施，預計將於6月30日前完成。

　　2018年9月20日，國務院第八督查組邀請10名網友代表，與三大電信運營商面對面交流，現場反饋問題，對提速降費提出建議。三大運營商當場承諾，將進一步提速降費。

　　2018年11月19日，從中國移動獲悉，中國移動將先行在北京、上海、安徽、江西、河南、湖北、陝西7個省(市)公司試點階梯套餐方案。
