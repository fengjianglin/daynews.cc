到雲南出差旅遊後手機號被封 省公安廳核實後解封

2019-12-08 472條評論 1,792次閱讀 875人點贊

12月7日，雲南省公安廳針對網友新近反映的「到臨滄旅遊時手機被無故封號」一事作出回應，經省公安廳刑偵總隊核實後，已解除網友被禁止的電話號碼。


早前，有網友在人民網領導留言板上介紹，其11月7日和兩個朋友到臨滄旅遊，起初以為卡壞了，後聯繫10086移動說是被停機，手機號碼無故被雲南反詐騙中心停機，給了一個反詐騙中心的電話號碼0883-96110，打了幾天一直佔線，「本人號碼用了十幾年實名綁定了支付寶銀行卡很多信息，從沒發過垃圾簡訊和騷擾電話，現無法接收驗證碼及正常使用已經嚴重影響到了旅遊行程。」

澎湃新聞注意到，近期有多名網友反映手機號被封停的情況。有網友11月30日稱，之前在西雙版納手機號莫名被封，經了解說是雲南公安舉報是電信詐騙，號碼用了八九年了哪裡會是電信詐騙呢？

另有網友同日反映，近期去雲南昆明出差，其間手機除租車的司機外，聯繫的均為北京的同事。11月28日，在活動結束返京的途中手機被停機（為移動號），通過聯繫10086得知，是被雲南省公安關停，並提供了唯一聯繫電話：0871-63054***。經過2天、多人接力式撥打，此電話一直無人接聽。

這名網友稱，目前因手機號被關停，其日常工作生活已受到嚴重影響。手機號碼在今天已不僅是一個通訊工具，也是一種確認個人身份的存在。我們很理解，如今電信詐騙猖獗，此舉也是公安部門對公民的一種保護措施。但是，隨意地「一刀切」關停個人手機，申訴途徑形同虛設，相關部門態度敷衍，與最初的打擊違法犯罪、維護公民合法權益的初衷也背道而馳。

雲南省公安廳對於上述留言均回復道，經省公安廳刑偵總隊核實後，已解除網友被禁止的電話號碼。

這些號碼被關停的理由是什麼？參考臨滄市委辦公室11月27日的答覆，近期，由於公安部集中封停一批在臨滄邊境地區使用的外省及本地重點地段疑似涉嫌電信詐騙的手機號碼，導致大量被封停機號碼無法正常使用。

臨滄市委辦公室介紹，根據公安部及雲南省公安廳要求各（州）市級公安機關在接到請求解封號碼訴求，分兩種情況處理：一是本人親自帶其證明材料到公安機關說明情況的，當地公安機關核實清楚情況後，可直接報公安部五局請求解封；二是對於因個人原因無法提供相關材料或者親自到公安機關說明情況的由（州）市公安機關登記相關信息後報省廳，由省廳核實後報公安部申請解封。

不過，根據網路反饋，有些網友的解封過程較為坎坷，這些投訴言論包括，「營業廳只給了我一個0691-96110號碼給我根本打不進去」、「移動客服告知是臨滄反詐中心停的機，並給一電話96110，連打四天，無人接聽」、「11月9號去雲南旅行，11號手機號就被雲南反詐騙中心停用，通過聯通客服知道0871-63054338這個電話，每天打幾十次，持續一個禮拜，電話也聯繫不上，客服也處理不了」、「移動公司給了我一個電話0871-63054338讓我打，我這一打就是一周，要不就是忙，要不就是通了沒人接」、「臨滄，西雙版納，思茅，德宏反詐中心電話，都是打不通」。還有網友於11月19日反映，0691110說讓郵寄開戶人姓名手持身份證和身份證複印件，快遞上顯示資料已經簽收了很多天了，然後現在又聯繫0691110，他們說也不知道找哪個部門，聯繫114又給了一個0691-2167009同樣的打通無人接聽！這裡里外外都耽誤了兩三個月了，也聯繫不到任何部門，訴說無門。
