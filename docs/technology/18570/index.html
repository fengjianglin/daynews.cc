數字貨幣認知變現：BTC期貨季度交割對市場的影響

以下文章僅為個人觀點，據此投資，後果自負。

BTC即時價格6884美金，還是熟悉的味道，凌晨15分鐘睡夢中毫無緣由的砸盤，然後白天橫盤窄幅震蕩，目前刺破了60周線6900這裡，注意刺破不是有效跌破，所以應有的反彈還是會有的，整體大趨勢依然看空至6000點，昨晚在7140美金止盈多單，而後開空單，今天6884美金，收益70%左右止盈，其他主流幣收益情況也不錯，不是太貪心，直接走掉就好了。

今天主要講期貨合約季度交割為什麼會造成行情的波動，所謂的季度交割，說簡單一點就是三個月結算一次，目前的老季度合約產品是12.27號，新季度是3.27號，先說老季度合約，會提前轉成次周合約，這個季度交割合約時間點為什麼如此重要，因為很簡單，很多機構大戶一般都不會去拿期貨像我們一樣去頻繁操作，包括一些大型礦老闆基本都是用來多期現的對沖，一般拿的都是長線，那麼市場很多散戶也會像發現新大陸一樣，原來他們賺錢是因為拿長線！所以可以理解為很多選手都喜歡做長線，想著比特幣從3000做個多單拿到14000美金，或者從14000美金做個100倍空單，拿到最低點，想法是挺好的，我也想這樣！但是幻想畢竟沒有用，還是要面對現實，基於此，在季度交割之日前後，一些人就會坐不住了，因為假如你不平倉的話，到交割時間點，系統會自動平掉你的多空頭寸，不論虧損或者盈利，都會自動平掉倉位，所以會引起連帶反應，很多人會提前動作，或者選擇讓系統自動平掉，那麼在現階段以期貨為主導的市場，勢必會造成價格瞬時的大幅波動，大家可以回看9.27號前後，遇上了925崩盤，6.27號前後是今年最高點，3.27號前後是今年啟動點，所以季度交割對於現貨行情的影響是不容小覷的。

恰好12.25號又是聖誕節，屬於國外的春節，這些日子都比較特殊，所以要格外注意。對於現貨行情來講，今天最低跌至6830美金左右，目前向上反彈，看了下昨日的成交量，基本是15分鐘的瞬時拋壓，這個拋壓不論是大戶出貨也好，或者是刻意製造恐慌也好，沒什麼用，在這裡會有反彈，關註上方6900-7000第一反彈位，第二反彈位7100-7300！整體市場陰跌向下，6000點指日可待。那麼現貨裝死不動，期貨慎重多空，尤其在交割前後，我是認知變現，感謝閱讀。
