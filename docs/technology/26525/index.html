15歲讀研、18歲讀博，天才少年卻被美國列為潛在危險，讓其回國

2019-12-22 1,128條評論 2,643次閱讀 1206人點贊

「成功在於智商、情商 … 情商有一點很重要的是，我知道我比你好，但我永遠不會說我比你好，我知道我是頂級，但我永遠說我是老二 …」這是馬雲在某一次活動中對一位傲嬌富豪的的刁鑽提問的回答。

我想從馬雲的這句話中，我們足以感受到馬雲的高超情商，反觀馬雲爸爸從一名鄉村教師到現在創立了阿里巴巴的令人矚目的成就，我們不難得出一個結論:一個人要想獲得成功，情商一定不會太差。

小時候的「外濁內清」

真正的天才都不是一蹴而就的，也不是生下來就是天才。謝彥波小的時候是在鄉下和奶奶一起生活的，直到臨近上學年齡，才被父母接回城裡。或許是從小沒有和父母在一起生活的原因，對他們有些陌生，又或許是天生性格內向的原因，他總是不愛開口說話。

從小沒有父母在身邊，在隔輩親的溺愛之下，他連第一人稱也不會用，總是會說「彥波要吃飯，彥波要喝水，彥波要……」這卻讓他的母親感到非常的失望，直嘆氣說:「這孩子，像個榆木疙瘩，長大准沒出息。」

但是他的父親卻不以為然，他覺得自己的孩子很有靈氣，要是加以引導，肯定大有出息。在那個年代，不愛說話時間久了，就會被人視為另類。看著兒子孤僻的性格，爸爸也很著急。

不久，爸爸終於想出了一個辦法，用《小木偶奇遇記》的故事引導他。從那以後，謝彥波開始變得和其他小孩子一樣活躍起來，儘管讓他主動和人說話時還是有點害羞靦腆，但是爸爸認為這已經算是好的開始了，孩子本身性格內向，不能一口吃個大胖子。

最有望拿諾貝爾獎的神童

有一天，爸爸在檢查謝彥波的作業時偶然發現，他一個二年級的學生居然會做五年級的題目。爸爸再三詢問到，才發現他的學習、接受能力特彆強，而且求知慾也很旺盛，便開始指導他自學。既然孩子有天賦，就不在學會的東西上浪費時間。

成功不僅僅是靠在外在因素，更重要的是內因起作用。謝彥波二年級的時候就給自己安排了一個非常緊密的時間規劃表：每天早上6.15起床，然後晨起鍛煉，學習。晚上放學回家先寫學校布置的作業，晚飯後不忘給自己放鬆一下，小玩一會兒，然後開始自學超前數學，晚上8.30睡覺。天天如此，無一例外。

就這樣，他在自己的驚人的堅持下，小學三年級就自學完了初中的課程，四年級就完成了高中數理化等功課，五年級就開始鑽研大學微積分等難題，每天都在題海中奮戰。最令人羨慕的是，他只憑藉小學文憑就獲得的湖南醫學院中學高二數學競賽第二名，這也算是他人生中的一個小轉折點，他出名了。

中國科技大學派專人來對他進行面試，發現現在上小學的他的數學相當於大一水平，各門功課也達到高中畢業水平，他進入了科大的少年班，成為了一個從小學直接進入少年班的傳奇人物。進入少年班後，他也不斷努力，他大學畢業提前考取了本校研究生，成為了國內年齡最小的研究生，1987年就獲得了博士學位。

情商––他成功路上的絆腳石

俗話說，上帝在給你關上一扇門的同時，一定會給你打開一扇窗；世事難兩全，說的就是這個道理。謝彥波雖然智商高於常人，但是情商方面卻不盡如人意。因為他是直接從小學升入少年班，沒有過多生活經驗的他，自理能力極差，連襪子都不會洗。話說，他剛來中國科技大學的時候，手裡還在滾動著一隻鐵環。

但是這似乎並不影響他的學業，他在物理方面的天資一直都得到充分表現，學習路上成績驕人，直到畢業。然而，天資聰穎有好處也有壞處. 在心理健康，人際關係方面，全班同學都落下了，但是謝彥波的問題尤為嚴重。因為他們從小學或是初中各個階段升到大學，生理心理上都不夠成熟，在上學時沒有良好的心態，沒有平常心。

謝彥波為人非常的傲氣，15歲讀研，18歲讀博，被看好在20歲之前獲得博士學位，但這看似他最輝煌的時候，也是唯一一次。他沒能和自己的導師處理好關係，於是轉去美國讀博。

在普林斯頓大學，他看似因禍得福，跟隨大名鼎鼎的菲利普·安德森教授學習，但是他依然處理不好與導師的關係，甚至比導師還傲氣。於是他和導師關係不和，漸漸成為普林斯頓中國圈子裡的熱門話題。

本來，事情並不是毫無轉機，但是似乎上帝也被他的傲氣惹怒了。這時發生了轟動一時的北大留學生殺死美國教授事件，謝彥波被懷疑為潛在的危險。中國科技大學決定讓謝彥波回國，這意味著他的留學生涯徹底的結束。

隕落的天才

而現在的他，僅是中科大的一名普通的物理系的教師，沒有了當初的夢想，沒有了當初天才的光環，甚至也沒有什麼積蓄，早早的結婚被分到一個小房子。

現在的他生活樸素，沒有了當初的張揚與自視清高，不修邊幅，如果看到他現在的樣子，很難將他與那個當初懷揣夢想的少年聯繫在一起。他對於科學的近乎執著和對於純真的不善於打交道，真的很令人惋惜！

其實，我們不難想到他的失敗的一生，時代的影子佔了太多因素。因為建設新中國，為國家培養人才，重視他們的成績，但是卻忽略他們也是一個活生生的人，也是要有自己的生活，沒有情商談何成功？不會合作談何成功？

揠苗助長自古以來就是一個失敗的方法，卻還是有人前赴後繼的以身試險，到頭來也是飛蛾撲火，讓他們失去了本該完美的人生！然而那個時代已經成為過去，一味地追究也毫無意義。還好，最後，謝彥波依然在他喜歡的事業上孜孜不倦的努力，願他的餘生能夠幸福快樂！
