即便9年電池零自燃事故的日產聆風，可能也比不過一台比亞迪

2019-12-15 1,514條評論 4,452次閱讀 1888人點贊

提到日產聆風這款車的時候，很多人都會感到比較陌生，尤其是對於國內的消費者來講。儘管聆風在2011年就進入了國內市場，然而由於它的銷量一直比較低迷，故而不久便退出了中國市場。

不過，即便聆風在國內不受消費者的追捧，但如果放眼世界市場，我們發現這款車的銷量表現一直不錯。根據日產官方公布的數據，聆風自2010年上市至2019年以來，其銷售總量已經超過了40萬輛，榮膺全球範圍內純電車型的銷量之王。

也許40萬的銷量對於一款燃油車型來講不算什麼，但是對於一款純電車型來講就具有不同的意義了。回到聆風最初上市的2010年，那時的國內外汽車市場對於純電動化汽車這種東西還不是特別接受。當然，這其中有多方面的原因，比如「三電」系統的不成熟、配套設施不成熟以及各國政府對於產業發展的政策不明朗等。

然而，如果從現在的汽車產業發展趨勢與聆風車型的熱銷情況來看，日產也許走對了路。作為較早一批投入電動化領域的廠家，日產由於前瞻的意識與技術的積累，使自身在當今電動化發展的道路上具備了了得天獨厚的優勢。

不過，這樣的優勢並沒有在中國市場發揮出可見的作用，起碼目前來看是這樣的。日產在國內的產品矩陣當中，只有軒逸·純電一款純電動車型。也許我們可以將這款車看成是日產聆風再一次進入國內市場的嘗試，然而從目前的銷量來看，軒逸·純電暫時並沒有達到預期的效果。

在國際市場的熱銷，在國內市場的遇冷，聆風這款車似乎經歷了冰火兩重天。如此強烈的對比反差，讓我們不得不思考，為何一款深受好評的純電車型在中國市場如此不吃香呢，難道是國內消費者的胃口太挑剔了嗎？

在筆者看來，也許並不是由於國內消費者對於產品太苛刻了，而是聆風這款車進入國內的時間比較尷尬罷了。

首先，聆風在2011年進入中國市場，而彼時的國內市場對於純電車型來講就好比是一片荒漠，政策不到位，技術不到位，配套設施不到位。就算是存在某些消費者有心為環保出一份力，恐怕也是心有餘而力不足。一旦後續保障不力，車輛「趴窩」，也是比較糟心的事情。

其次，就是它高昂的售價。當時整個汽車行業的電動化趨勢並不明顯，有許多所謂的「專家」都斷言，電動車只是偽需求，只是政策催生下的一種畸形產物。如果政策方面出現變動，特別是對於電動車來講屬於負面調整的時候，整個產業可能無法經受住市場的考驗。

所以在當時不利的市場導向中，售價高達20多萬的聆風，又有幾個人敢接受呢？在這樣的價位，消費者可以買到許多不錯的燃油車型了，何必要做第一個吃螃蟹的人呢？

何況當時的中國市場可能並不需要聆風這種價位的純電車型，因為風險太大。無論是對於生產廠家來說，還是對於終端的消費者來講。於是他們更願意買一些價位相對便宜的純電車型來進行單純的通勤使用，畢竟這樣的選擇風險要相對小一些。

然後，續航里程較短。續航焦慮是純電車型發展到今天仍然沒有很好解決的問題，即便是某些車型已經達到了理論500+km的續航里程，但在實際工況中也會縮水不少。更不必說當時聆風160km左右的續航里程了。續航問題不能夠得到有效的解決，這將會極大制約消費者對於純電車型的最終決定。

再加上國內新能源廠商的競爭，比如比亞迪。那麼這樣一來在有較大價格優勢，以及一定的續航里程也可以保證的情況下，有意向選擇純電車型的消費者也大概率選擇了比亞迪的相關車型。畢竟比亞迪在純電車型的開發方面也有著不錯的積澱。

最後，還要提及的一點就是它只是一台日產。當然筆者在這裡並沒有詆毀日產的意思。其實在純電車型領域有一個比較極端的存在，就是低端車型與高端車型賣得好，而相對處於中端的車型則是一個比較尷尬的存在。

我們可以看到目前純電車型的銷量分布，低端品牌是以北汽新能源為代表的，它的銷量多半來自於出租公司採購、政府採購等，所以在這種情況下買價格高的品牌車型並沒有太大的意義。

而高端品牌則是以特斯拉為代表的。值得一提的是特斯拉旗下的Model 3 車型自2018年上市以來，不到兩年的時間，其銷售總量已經來到了27.6萬輛。

不出意外，按照目前的發展趨勢來看，全球純電車型銷量的寶座易主是早晚的事情。話說特斯拉的勢頭為什麼這麼猛呢？或許它的車就跟它的老闆一樣，因為他們都太酷了。而聆風呢，不可能自降身價，而作為高端品牌，其溢價能力又比較有限。

寫在最後

雖然聆風貴為全球純電車型的銷量霸主，但它在當前全球最大汽車市場的中國不受歡迎也是不爭的事實。伴隨著國內新能源汽車，尤其是純電車型的發展，即便是在技術或者續航等方面都得到升級的聆風再次進入國內市場，其市場表現可能在短時間內也並不會有太大的起色。
