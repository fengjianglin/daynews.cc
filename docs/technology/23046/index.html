鏈圈動態

工行利用區塊鏈技術打造的數字信用憑證，將企業信用延伸到「長尾客戶」

鏈得得（微信號：ChainDD）12月18日訊，據人民網報道，在昨日舉辦的首屆「工行小微客戶節」上，工商銀行行長谷澍發布了「萬家小微成長計劃」。據了解，2018年工商銀行推出普惠金融服務1.0，發布了「一平台、三產品」，一年多來已取得顯著成效。其中，信用類的「經營快貸」已為近百萬客戶授信超5000億元；抵質押類的「e抵快貸」餘額突破1200億元；交易類的數字供應鏈方面，已與3000多家龍頭企業建立合作，並利用區塊鏈技術打造「可拆分、可流轉、防篡改」的數字信用憑證，將核心企業信用延伸到產業鏈末端的「長尾客戶」，以金融力量賦能全產業鏈發展。

杭州互聯網法院上線了浙江首個智能證據分析系統，綜合運用區塊鏈等技術

鏈得得（微信號：ChainDD）12月18日訊，近日，杭州互聯網法院上線了浙江首個智能證據分析系統，綜合運用區塊鏈、人工智慧、大數據、雲計算等前沿技術，將重複的工作全部交給系統完成。法官們一鍵點擊就能獲得證據分析結果，為辦案提供了參考，節約了大量的時間和精力。

巴基斯坦總統：國家工業部門需為應對區塊鏈等顛覆性技術做好準備

鏈得得（微信號：ChainDD）12月18日訊，據Bitcoinist消息，12月17日，巴基斯坦總統Arif Alvi在新興技術任務組會議上表示，該國的工業部門需要為應對人工智慧，區塊鏈和物聯網等顛覆性技術做好準備。

印度政府批准超6000萬美元的技能發展項目，包括區塊鏈培訓

鏈得得（微信號：ChainDD）12月18日訊，印度聯合政府周三批准了一項價值43.6億盧比（約合6114.5萬美元）的技能發展項目，該項目將在未來三年內實施。根據這項名為PRIME的計劃，約有4萬名專業人員將接受物聯網、人工智慧和區塊鏈等最新技術的培訓。印度電子和信息技術部和Nasscom將共同培訓這些專業人才。

幣圈動態

央視再次報道打著 「區塊鏈」旗號進行虛擬貨幣交易和詐騙的行為

鏈得得（微信號：ChainDD）12月18日訊，央視新聞頻道今日下午再對打著「區塊鏈」旗號進行虛擬貨幣交易和詐騙的行為進行報道。報道稱，全國各地近期紛紛出台政策，加大打擊力度，例如關閉大V、網路紅人的微博等。

印度所得稅部門正在培訓官員調查加密貨幣

鏈得得（微信號：ChainDD）12月18日訊，據cointelegraph消息，印度所得稅部門正在秘密訓練官員調查加密貨幣，該部門發行的一本內部指南解釋了加密貨幣、它們的特點、「比特幣的陰暗面」以及稅務官員的最佳調查做法。

據悉，印度政府不認為加密貨幣是法定貨幣，但也沒有宣布它們完全非法，通過加密貨幣賺錢是印度加密社區的灰色地帶。

深圳召開虛擬貨幣非法活動專項整治，8家涉嫌發幣企業被約談

鏈得得（微信號：ChainDD）12月18日訊，根據深圳市地方金融監督管理局發布的《關於召開虛擬貨幣非法活動專項整治會議的通知》文件顯示，深圳市目前要求各區整治辦對轄內涉嫌從事虛擬貨幣非法活動的企業進行初步摸排核查，目前發現8家企業涉嫌從事虛擬貨幣非法活動。
