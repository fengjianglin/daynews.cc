長城郭岩松聊智能化：萬物互聯從一個平台開始

2019-12-10 1,199條評論 4,788次閱讀 1077人點贊

從3G到4G，將人連接起來，世界與生活就發生了這麼大的變化，是當年的我們所不曾設想的。5G時代，萬物互聯，這次我們或多或少會有所期待。而汽車，是一個多場景移動產品，很適合作為萬物互聯的中心載體，至少汽車企業是這麼理解的，所以有實力的企業都在智能化上不遺餘力地探索。這個時候，企業對智能化的理解，對未來競爭格局的判斷決定了他們選擇的路線，也必將決定在這場革命中自己的命運。

長城，作為自主品牌一線企業，正在為更遠大的發展積蓄力量，智能網聯也成為其布局未來的重要一環。此前，長城發布過一個GTO智慧生態戰略，宣布將推出全生命周期在線的汽車，並在將來實現汽車由代步工具到智能出行機器人的轉變。

一個平台，一台車，一個企業，將怎麼樣構建我們的未來汽車生活？我們走進長城，與長城汽車技術中心副主任郭岩松面對面交流，深入了解長城智能網聯戰略的頂層設計脈絡。

一個企業級互聯中心平台

——讓連接萬物成為可能

新車評：對長城來說，如何在智能化這個方向走得更快，獲得更多的資源？

郭岩松：我們希望打造的是一套能保證顧客利益，安全穩固的操作系統。合作方對接我們的時候，不需要做過多的更改，同時也能享有我們固有的安全性。如果通過我們的平台去使用這些APP，那麼用戶的個人信息是可以得到保障的，但是如果你直接去使用這些APP不通我們平台，那我們就保證不了。

新車評：長城在保證用戶隱私安全的情況下，還要去進行更多的創新，怎麼樣才能去趕上競品？

郭岩松：本身跑得快取決於公司的能力，你對這個事情投入多大的資源。長城公司一直秉承這種過度投入的方法，去做研發，所以這一塊我們有信心。另一方面，我們對未來是有一個很清晰的目標，還有志向，我們制定了很嚴密的戰略規劃。而且這個戰略計劃，我們認為是看得比較遠。

為了實現為了這個目標，我們認為平台是非常重要的。我們在今年GTO發布的時候，提到了我們會搭建一套長城的企業級的平台，國內的車企能做到這一點是非常有前瞻性的。所有的信息都要過濾到這個平台上，你才能做到更多的計算的因素。以此來收集大量的用戶的各種數據，成為數據的支撐來做智能化的運算。作為一個車廠，我們現在的第一要務已經變成了搭建一個企業級的平台。

它對一個車企來說的意義也是非常重大的，我們可以通過監控企業內的數據，查看生產效率是否有提升。比如今年的車市低迷，導致所有的4s店的庫存都已經超越了警戒線，給我們經銷商的產業鏈帶來了巨大的壓力，但如果我們有一套數據的平台，就能通過經銷商的銷售情況來制定我們的生產計劃，大幅提升企業運轉效率。無論對產業鏈的任何一方都有很大的效益。

再從售後這個角度來講，如果有一個企業的平台，也能帶來很大的便利。比如車壞了，需要更換部件，那以前的方式就是往往需要等，1~2天的時間甚至更長的零件周期。有了這樣的企業平台後，在確認消費者車輛故障後，我們的系統就能馬上開始準備零件，大大的節省了維修的周期。在壓縮顧客的時間成本同時，又壓縮了經銷商的各種成本。從前中後三個層面都能提升效率。

新車評：物物互聯現在做出來的如小米生態圈，將來長城也會打造一個類似的生態圈嗎？

郭岩松：我們希望是做平台做系統，希望做有自主產權的一套平台，圍繞用我們汽車的用戶來進行服務，這個平台可以和其他的平台進行對接。我們和谷歌比較像，我們先做出來一套東西，你可以和我們不一樣，我們談合作的時候再進行兼容化的設計。

一台全生命周期在線汽車

——給汽車用戶的利益是什麼？

新車評：長城在GTO戰略發布會上提出「汽車全生命周期在線」這個概念，對用戶來說，會帶來什麼樣的利益點？

郭岩松： 我們7月份的GTO全域智慧生態戰略，意思是全時全域生命周期在線，全時就是24小時以至全生命周期，直到產品報廢，客戶都能享受這種智能化的服務，全域是指地域，我們都要提供智能化的服務。這是我們的一個戰略目標。

同時，結合大的歷史發展。包括現代科技發展趨勢，我們認為單純在車上提供服務，是不足夠的。長城作為一個有抱負的中國自主品牌汽車製造商，我們希望通過智能化服務，為用戶全方位全場景地提供智能化服務。

這個怎麼理解呢？比如我現在在車上有這種智能化的服務，通過手機APP共享用車的安排，將我們的服務也能拓展到生活的方方面面。不過現在還在一個起步的階段，我們現在有訂餐，訂酒店，訂機票在車裡實現。

未來我們可以拓展到家裡，在我們的手機APP也會有語音助手，可以通過我們的語音助手來遙控我們車。比如在辦公室著車預熱，未來可能還會直接開到公司樓下，出門直接上車。特斯拉現在也在推出這種預約召喚的功能。

還有我們現在智能家居也在拓展萬物互聯的業務，以後我在車上，沒到家就能先把我們家裡空調打開，空氣凈化系統就會運轉起來。等我到家以後屋裡就可以很溫暖，清潔。甚至未來更先進一點，咱們現在家裡都是電熱水爐，就是24小時開著，涼了再熱，其實是一種能源的浪費，將來可能是我走了，家裡沒人了就可以斷掉，我回家了這個熱水爐就提前熱起來。

但是，我們現在還是認為這種用笨的方法不叫智能，實際上智能這個觀點不斷的發展，我們現在說掏出手機APP點一下遙控那個熱水器，打開那個熱水壺，我想過不了幾年我們不需要點開APP，甚至話都不用說，我們開著車進小區，熱水器就開了。車輛能感知你的位置，他就可以和一些生活的場景關聯起來。

再比如，我們每天早上可能要送孩子，然後再去上班。每天規劃路徑，智能系統知道你起點是家，終點是公司。用目前的導航肯定給你規劃最短的路徑。但是如果我們未來智能化，每天都要送小孩，每天都要經過幼兒園，它自己就會把這些規劃好。實現全時全域的智能化，為了更多是拓展全域的能力。

從賣商品到賣服務的轉變

新車評：智能和互聯會不會改變現在的銷售格局和車企的角色？

郭岩松：會的，現在長城已經逐漸從一個賣商品轉變為賣服務的一個企業，車已經成為我們（車企）和消費者建立聯繫的一個媒介。我們會從一個製造商的身份向一個服務商的身份轉變。

長城現在有租賃公司也有金融公司，貸款買車或者分期付款買車，和一次性付款，我們提供不同的支付方式。長城的出行部門歐拉，為不同的人提供不同的服務方式，也可以租賃，可以按天計按月計。

長城現在整體的規劃是圍繞著出行來進行。共享汽車業務，我們有可能會根據用戶的需求和活動的內容，而會在他需要租的共享汽車中放置一些小物品的小禮物，通過大數據做到更加精細化的服務。

新車評：將來自動駕駛的技術是包含在這個戰略的是嗎？長城自動駕駛方面的規劃怎麼樣？

郭岩松：行業里現在出現了L3了，長城做產品希望腳踏實地，我們不希望給消費者一個錯誤的概念，欺騙消費者產品值得信任，而實際上並不可靠。美國一個大學教授，最近測試了兩個國際領先的自動駕駛系統，其中發現了500多條漏洞，條條都幾乎足以讓人喪命。不靠譜的系統，我們是絕對不會讓消費者去替我們做小白鼠。

對於長城來說，L2.5現在已經實現了，而在2021年會實現L3級的自動駕駛，商業化。

前沿君

長城有35年的積累，今天看來，他正以一個成熟汽車製造商的經驗和心態來應對新四化，穩健老道。穩健的地方，體現在他著眼長遠，從信息底層架構去考慮，沉心做基礎工作。老道的是他明確守護信息安全和自動駕駛可靠性底線，在這個基礎上追求速度，雖然這樣一來顯得保守，但恰恰是這份對安全的敬畏十分可貴。
