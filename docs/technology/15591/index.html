騰訊醫療副總裁張猛：互聯網+精準健康科普，助力價值醫療

2019-12-14 539條評論 4,550次閱讀 516人點贊

12月12日，2019中國價值醫療大會在北京召開。會上，國家衛生健康委中國人口宣傳教育中心與騰訊醫典簽署戰略合作協議，雙方將充分發揮各自專業內容和渠道優勢，合力完善全媒體健康科普知識發布和傳播的機制建設。與此同時，騰訊醫療副總裁張猛發表了《互聯網+精準健康科普，助力價值醫療》主題發言。張猛表示，在健康中國行動等一系列政策推動下，互聯網醫學科普正進入黃金時代。「價值醫療」強調以患者需求為中心，患者不再只是醫療的被動接受者，而是主動參與決策、主動管理自身健康。

騰訊醫典堅持以權威的醫生為作者、前沿的知識為內核、專業的態度為初心，解決用戶在醫療知識方面未被滿足的需求。「面對提升民眾健康素養的重要任務，行業各角色應該從單體作戰，向全社會整體聯動轉變，在個人、醫療機構、政府、社會之間形成知識循環。「張猛表示，通過政府、醫療機構、主流媒體建立開放合作，在診療全鏈條中融入權威可靠的醫學科普，不僅讓患者在醫療機構中獲得疾病診治，而是從健康促進、疾病防控、就醫指導、院後康復、健康管理、臨終關懷等角度，為公眾提供連續性的一體化信息支持，提高整體醫療衛生服務質量。

以下為張猛演講全文：

尊敬的各位嘉賓和媒體朋友：

大家上午好！非常高興今天可以參加此次活動。

價值醫療被稱為最高性價比的醫療，強調以健康為導向，以患者需求為中心。患者不再只是醫療的被動接受者，而是主動參與決策、主動管理自身健康。「每個人都是自己健康的第一責任人」這是健康中國行動核心理念，價值醫療與健康中國的理念不謀而合。

當前，互聯網已經成為大眾獲取醫療信息的第一來源，也是賦能醫療體系，推動服務體系升級的驅動因素。通過互聯網可以面向患者提供健康科普、隨訪管理等一系列服務；可以支持醫療機構改善患者就醫體驗；也可以洞察患者在診療鏈條中未滿足的需求，實現更有價值的創新解決方案。

我們正處在醫學飛速發展的時代，但大眾認知與醫學的發展仍有差距，發病機制的研究、治療方法的演進、疫苗的升級、新藥物的研發，都在突飛猛進。但很多誤解、很多陳舊觀念依舊沒有得到有效轉變。民眾的健康素養從根本上影響人群健康結果，健康意識的缺失使得很多疾病防治工作進展不容樂觀，例如高血壓患者中，不到三分之一的人採取積極治療，也帶來了高昂的醫療成本。

破解醫學高門檻，把普通人帶進醫學的知識大門，解鎖人體複雜的機理和結構, 應該由醫生和醫學從業者主導。但一系列傳播亂象，導致劣幣驅逐良幣。互聯網健康教育、醫學科普，需要進行一次重塑。以權威的醫生為作者，以前沿的知識為內核，以專業的態度為初心，以理性的思想為基石，讓真正的醫學為大眾所知道，增強對健康的自我掌控，改善民眾健康素養，推動醫療效果的整體改善。

首先，讓專業人士取代「小編洗稿」，好的科普應該由醫學專家逐字書寫或審核，從而保證內容的質量。目前，已經有近3000位臨床專家參與了騰訊醫典的內容建設，其中兩院院士、學科帶頭人有80多位。我們與國際領先的、權威醫學健康信息平台WebMD合作，獲得6萬多篇科普文章的獨家授權。截止目前，騰訊醫典覆蓋病種數量超過了1萬個，不僅有常見病，也包括腫瘤等重疾和罕見疾病。

其次，注重醫學科普的體系化，對於重點疾病，覆蓋早期預防、癥狀辨別、初步診斷、治療方法和生活管理，幫助患者在每個環節都做好健康決策。

再次，通過短視頻、漫畫、3D視化等富媒體，讓「硬核知識」看得懂。

最後，到2030年，全國居民健康素養水平要提升一倍，這是一個艱巨的任務，這個過程需要社會各界的共同參與。健康教育要從單體作戰，向全社會整體聯動轉變，在個人、醫療機構、政府、社會之間形成知識循環。

騰訊醫典擁有海量的專業內容，非常期待與各地衛生主管部門、各醫療機構進行授權合作，通過在診療鏈條中融入權威可靠的醫學科普，不僅滿足患者在醫院診治疾病的需求，而是從健康促進、疾病防控、就醫指導、院後康復、健康管理、臨終關懷等服務，為公眾提供連續性的一體化信息支持，提高整體醫療衛生服務質量。

中國人口宣傳教育中心在衛生健康領域有著豐富的積累，在廣播電視欄目製作、影視製作、融媒體傳播、社會倡導與宣傳、圖文宣傳品製作等領域，不斷輸出精品力作。期待雙方合作，可以傳播醫生的專業知識，為患者帶來更多希望和溫暖。共同推動健康科普深入每一個機構、社區和家庭。讓更多人擁有健康理念，享受健康生命帶來的喜悅。謝謝大家！
