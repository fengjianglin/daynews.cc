外媒評谷歌創始人辭任：他們放棄成為科技界巴菲特

天天要聞
2019-12-04

科技訊 12月4日消息，據國外媒體報道，當地時間周二谷歌母公司Alphabet首席執行官拉里·佩奇(Larry Page)宣布辭職，而谷歌首席執行官桑德爾•皮查伊(Sundar Pichai)將接任首席執行官一職。同時谷歌另一位創始人、Alphabet總裁謝爾蓋·布林(Sergey Brin)也宣布辭任。

谷歌創始人早已接受了沃倫•巴菲特(Warren Buffett)不干涉的管理風格，多年來一直試圖將自家公司打造成科技界的伯克希爾·哈撒韋公司。而周二兩位創始人雙雙辭任的決定結束了這一切努力。

佩奇和布林在2015年創建了Alphabet，以便給自己更多時間投資新興科技企業，並將管理谷歌的職責交給了桑德爾·皮查伊(Sundar Pichai)。這一模式的靈感來自巴菲特的做法。這位華爾街大亨將資本分配給不同的企業，並讓各個獨立公司的首席執行官自行決定如何運營。

谷歌創始人通過任命皮查伊為谷歌和Alphabet的首席執行官，從而有效解除了這種結構。皮查伊本已經忙於運營谷歌龐大的數字廣告業務，應對反壟斷調查和員工抗議等棘手問題。現在，自動駕駛汽車、醫療保健、數字城市、送貨無人機和互聯網氣球等諸多項目也成了他的難題。

對於公司的許多人來說，Alphabet自成立以來目的和結構從未真正明確過。谷歌為Alphabet貢獻了超過99%的營收。谷歌一名前高管表示，讓谷歌負責人執掌Alphabet，可能會讓人對Alphabet公司的整個宗旨產生質疑。另一位谷歌前高管說，這一變化將意味著皮查伊面臨的壓力會更小。

從財務角度講，Alphabet向投資者表明，該公司並沒有在雄心勃勃的「登月」項目上投入太多資金，同時也突顯出谷歌主營業務的巨大盈利能力。

但在運營上，該公司架構一直處於近乎持續的動蕩之中，一直難以創建出一項獨立於谷歌的新業務。2018年，該公司包括無人駕駛汽車Waymo和醫療保健Verily等業務在內的「其他賭注」總計虧損34億美元，而上個季度虧損近10億美元。

智能設備製造商Nest最初是一家獨立的Alphabet子公司，但去年被併入了谷歌硬體部門。皮查伊還將DeepMind的許多人工智慧項目納入了谷歌管理的範疇。網路安全項目Chronicle去年剛剛在公眾面前高調亮相，並成為一家獨立的Alphabet子公司，但今年6月就被併入谷歌雲計算部門。而一度備受矚目的光纖通信業務Fiber已不再繼續擴張。

一些谷歌現任和前任員工認為，皮查伊的升遷表明，Alphabet公司的初創業務將獲得更強有力的公司治理。佩奇和布林因其富有遠見的技術理念而備受尊敬，但他們的管理技能卻不甚高明。

風險投資公司Firebolt Ventures合伙人維尼特·布奇(Vineet Buch)，曾在谷歌擔任產品管理總監八年多，直到2018年底才從公司離職。他表示「這一變化可能意味著，其他項目必須開始真正發揮商業功能。不再像之前那樣的雙層體系結構，只有谷歌是企業化運營，而其他項目有無限的時間跨度來實現盈利。」

布奇以Nest為例，說明皮查伊將如何運營Alphabet和谷歌。這家智能家居設備製造商最初被譽為Alphabet獨立業務的典範，擁有自己的首席執行官、營銷部門甚至電子郵件地址。但公司難以按時發布產品，於是首席執行官被迫離職。布奇說:「當Nest被併入谷歌內部時，Nest的發貨速度以及這些產品在市場上的生存能力都上升了。」

佩奇和Alphabet資深律師戴維·德拉蒙德(David Drummond)等活躍在「其他賭注」業務的人一直避免與股東對話，這一事實一度也讓華爾街感到困惑。德拉蒙德現在將向Alphabet負責人皮查伊彙報工作。

投行Evercore ISI分析師在一份報告中寫道:「從長遠來看，事情要簡單得多: 桑德爾是鎮上唯一的治安官。」

「新警長」倒是急於淡化他的重要角色。皮查伊在給員工的電子郵件中寫道:「我想明確表示，這種轉變不會影響Alphabet的公司結構，也不會影響我們日常的工作。」（辰辰）
