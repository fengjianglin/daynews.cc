騰訊發新機，售價1099，配5.2英寸大屏

天天要聞
2019-12-16

在亞馬遜之後，多個科技企業都開始角逐電紙書這一產業。雖然大多數的產品最後都淪為了「泡麵神器」，但是也擋不住電紙書發售的熱情。日前，騰訊就宣布推出新一代口袋電紙書-口袋閱，並正式發售。

該款口袋閱為上一代口袋閱電紙書的升級版本，在外觀上繼承了上一代的設計風格，依然主打的是口袋裝得下的設計理念，因此它的尺寸比例並不大。口袋閱二代的規格尺寸為145×75×8.5mm，重130g，外觀設計形似智能手機。

所不同的是，口袋閱電紙書搭載的水墨屏幕。配備5.2英寸水墨大屏，輔之以2.5D AG工藝玻璃光學技術，使其紙質效果看起來更為逼真。為了增加閱讀體驗，口袋閱還配備有36級閱讀燈，屏幕像素密度達284PPI，具有手機一般的觀感體驗。

系統方面，搭載的是64位操作系統，可支持4G手機卡/流量卡聯網，並支持通話、簡訊和WiFi聯網，還可一鍵切換飛行模式。相比於傳統的電紙書，能力不俗。單就可通話這一點，就值得買買買。

口袋閱二代電紙書內置有2000mAh容量電池，配備Type-C介面，可支持觸控操作以及物理按鍵翻頁等操作。整體設計方向，完全按照手機來設計，唯一不同的是採用了水墨屏幕。

用戶可以通過網路直接在線下載圖書資源，內置有海量資源可供隨時調取，如果你是一個書迷又覺得買紙質書太貴，那麼這款產品可能適合你！
