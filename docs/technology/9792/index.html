中芯國際：與ASML設備出貨問題已解決

2019-12-09 731條評論 4,721次閱讀 1361人點贊

據悉，中芯國際董事長周子學在近日飛往韓國，並單于了韓國工商總會（KCCI）與中國國際經濟交流中心（CCIEE）舉行的第二次中韓商業領袖及前政府官員會議。

「我們目前正在致力於極紫外光（EUV）的相關工作，以及我們在先進位程方面的研究。研究和生產一種在軌道上，我們與客戶及設備供應商目前沒有任何問題。」周子學在會議上說。

目前，在半導體市佔率排行中，中芯國際排名第五，僅次於台積電、三星電子、格芯和聯電。

周子學指出，中芯國際此前與ASML的EUV設備交貨問題已得到解決。此前有報道指出，ASML曾因受到某國壓力，延遲向中芯國際出貨EUV設備。另也有消息人士稱，ASML雖推遲出貨，但並非完全取消訂單。

而中芯國際也有望加速其技術改進的力度，因中芯國際正以接近目前年薪三倍的高額薪酬待遇吸收了台灣和韓國的EUV工程師。儘管中芯國際與三星在技術和生產上仍有約五年的技術差距。

有專家表示，由於中國政府的全力支持，中芯國際對同業的潛在威脅也將迅速成長。

據市調機構9月發布的報告顯示，目前中芯國際的市佔率仍然只有4.4%，在技術方面仍落後於三星。而三星和台積電也正在3.5nm產品的開發上進行激烈競爭，中芯國際的主要製程為14nm鰭式場效應晶體管（FinFET）製程。為了縮小與前兩者的差距，中芯國際在去年4月自AMSL訂購了10億元的EUV設備，計畫用於開發7nm以下的產品。

然而，隨著美中貿易爭端加劇，在美國政府的壓力下，ASML向中芯國際供應EUV設備的計劃受到了阻礙。

對中芯國際來說，此問題有望於近期獲得解決。周子學表示：「儘管ASML闡明了其立場，但EUV設備的供應目前已沒有太大的問題，但我們仍會持續關注AMSL是否會儘快將EUV設備實際出貨」
