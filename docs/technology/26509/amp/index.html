汽車科技的盛會，CES 2020 有什麼看點？

天天要聞
2019-12-22

自動駕駛技術正在重新定義並塑造整個移動出行產業。

它改變的可不只是車輛的運行方式，還有車輛與整個世界的交互方式。舉例來說，讓自動駕駛汽車有了「千里眼」的感測器技術將配備在交通信號燈等基礎設施上，未來這些感測器會成為採集數據的生力軍。

此外，V2X 技術也將成為智慧城市的標配，協助路上的萬物互相「交流」。轉眼間，一年一度的 CES 展會又將在拉斯維加斯拉開帷幕。

近些年來，這個主打消費電子的展會已經成了移動出行技術的新舞台，其風頭甚至要蓋過隨後舉行的底特律車展。

那麼今年，移動出行產業有什麼新趨勢？

1.CES 2019 後的移動出行產業

CES 2019 上，自動駕駛汽車、無人直升機、VR、AR 和機器人等產品爭奇鬥豔。

其中，最令人印象深刻的包括寶馬的自動駕駛摩托車，奧迪與迪士尼的車載 VR 遊戲技術，拜騰電動車中控那塊 48 英寸巨屏以及現代推出的新型步行汽車概念等。

自那之後，移動出行產業的融資春天就來了，自動駕駛、網聯汽車、電動化與智能出行領域的新創公司們都「收穫」頗豐。

麥肯錫最近的報告顯示，過去 24 個月里，移動出行產業至少拿到了 1200 億美元的融資，而且投資者的熱情絲毫沒有消減的意思。

自動駕駛技術，特別是感測器（比如激光雷達和雷達等），是過去一年裡最受投資人青睞的。 不過，即使勢頭良好，移動出行產業也並非一帆風順。

關於自動駕駛汽車的感測器技術，今年業內也掀起了一場大討論。

特斯拉 CEO Elon Musk 就堅信「激光雷達無用論」，但大多數人不買他的賬，包括通用、Waymo 和 Velodyne 在內的業界巨頭紛紛出面發言，闡述自己的立場與看法。

顯然，大多數人依然認為，沒有激光雷達，我們很難實現 Level 4/5 級別的自動駕駛。

此外，業內專家也嗅到了激光雷達行業大洗牌的氣息。

現在百家爭鳴的局面，在未來將不復存在。只有少數幾家公司能生存下來，而投資也會向它們傾斜，因為只有這些頭部公司才能拿出完整的車規解決方案。

需要注意的是，那些吸引了大量投資的頭部公司不會甘心於只做激光雷達，它們還會順勢進入 AI 軟體和數據分析等領域。

激光雷達數據不但要為定位、路徑規劃、避障和地圖繪製服務，還會被用來訓練感知演算法。 因為只有演算法足夠安全可靠，自動駕駛汽車的控制系統才能讓公眾放心。

2.CES 2020 有什麼看點？

CES 2020 上，我們肯定能看到最新的概念車，當然也有各種自動駕駛演示以及面向自動駕駛產業的數據驅動軟體與服務等。

汽車廠商與第三方公司的聯繫也將愈加緊密，因為這些汽車廠商要依賴對方的感知軟體。

此外，激光雷達供應商也會抓住機會，利用此類感知軟體提升自己的價值定位。

因此，AI 軟體肯定會成為 CES 2020 的重頭戲。

同樣地，自動駕駛擺渡車也將迎來大發展，因為未來兩年這類車輛將成為交通運輸市場的重要力量。

這些擺渡車將在固定線路上（有地理圍欄）提供出行服務，成為城市交通的新形式。 如果明年你要是去參加 CES，一定別忘了去體驗試乘這類車輛。CES 期間，賭城拉斯維加斯的路上肯定少不了自動駕駛擺渡車的身影。

對那些城市交通急待改革的城市來說，自動駕駛擺渡車絕對是一劑良藥。

自動駕駛的出現將通勤時間還給了駕駛員，無論是工作還是娛樂，這段時間都會成為服務提供商的新機遇。

因此，CES 2020 上我們肯定能見識到各種依託於自動駕駛汽車的全新商業模式。

當然，5G 必然會成為 CES 2020 的絕對重頭戲。

許多行業專家相信，高帶寬、低時延的 5G 網路能加速 L4/5 自動駕駛汽車的部署。

因此，英特爾與高通等半導體巨頭一定不會錯過網路接入設備這塊大蛋糕，它們在用 5G 網路為自動駕駛汽車獻上高質量的信息娛樂服務的同時，還會提供 V2X 通訊服務，最終將自動駕駛汽車變成移動數據中心。

除此之外，5G 網路還會為自動駕駛汽車提供額外的安全冗餘。藉助低時延的特性，各種遠程操控技術必然會迎來爆髮式發展。當然，5G 帶來的各種用戶體驗提升也會成為 CES 2020 的熱點。

3.「求新」和「求變」成主流

如今全球汽車保有量已高達 13 億輛，汽車巨頭與科技新貴們對自動駕駛的追逐正在改變車輛設計與研發的方式。

顯然，「求新」和「求變」成為主流。除了傳統商業模式，大家還想藉助車輛與乘客數據變現。

CES 2020 上，參會者不但會迎來自動駕駛擺渡車大潮，還將見識到各種 5G 應用，同時感受到感知軟體大爆發。

與此前一樣，CES 依然是展示新技術的最好舞台，希望 2020 年的 CES 展不會讓我們失望。另外，汽車之心也將前往拉斯維加斯帶來第一手的汽車科技報道，敬請關注。
