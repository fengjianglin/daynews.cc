手握31隻獨角獸，造就中國最大跨國民企，未來或將成為亞洲首富

2019-12-06 1,574條評論 4,098次閱讀 1746人點贊

根據今年的胡潤百富榜，阿里巴巴的創始人馬雲以2750億元人民幣的身價再次蟬聯中國首富之位，騰訊的馬化騰以2600億元的財富重返第二位，恆大集團的許家印則以2100億元的財富位列第三。其實，除了馬雲之外，馬化騰和許家印此前也都榮登過中國首富之位，他們多年來一直活躍在各大富豪榜的前列。

（此處已添加小程序，請到今日頭條客戶端查看）

馬雲「我對錢不感興趣」

實際上，每年全球各種富豪榜的數據都不盡相同，所以這些富豪在不同榜單中的排名也會有一些變化。比如在今年的福布斯富豪榜上，中國首富並不是馬雲，而是馬化騰。根據福布斯富豪榜的數據，馬化騰在全球排名第20位，在中國排名第一，馬雲、許家印緊隨其後。不過，如果放眼亞洲地區的話，馬化騰還不是最富有的人。今年的亞洲首富是印度最有錢的富商——穆克什·安巴尼。而馬化騰與穆克什·安巴尼的財富相差112億美元，可見差距還是不小的。

但是要知道，互聯網行業的發展是非常迅猛的，作為互聯網領域大佬的馬化騰，在未來還是很有可能向亞洲首富之位發起衝擊的。為什麼這麼說呢？

首先，騰訊的業績表現非常強勁。到目前為止，騰訊仍然是國內乃至全球範圍內市值較高的科技企業。對於科技公司來說，其股價往往取決於其業務的表現。而專註於移動社交業務的騰訊，僅依靠微信和QQ就獲得了超過11億的用戶。龐大的用戶群體推動了騰訊在網路遊戲、數字內容、金融科技與企業服務、網路廣告等方面的業務增長，帶來了非常亮眼的業績。基於此，騰訊的市值也一直在增長，目前已經達到3.17萬億港元。而馬化騰的身家也會隨著公司市值的增長而增長。

其次，騰訊一直非常注重公司的多元化發展。其擴大業務版圖的方式主要是投資於各種類型的中小企業。騰訊在短短几年內投資了600多家公司。僅去年一年，騰訊就投資了163家企業，近乎每個月投資13.6家企業。因此，騰訊也被外界評價為不是以產品著稱的互聯網企業，只是一家投資企業罷了。根據騰訊的劉熾平的說法，騰訊投資這些企業所獲得的價值，甚至超過了騰訊自身的市值。目前，騰訊的投資已經進入「收穫季節」，它所投資的這些企業實現上市的同時，不僅會給騰訊帶來更多的價值，也會讓馬化騰的個人財富增多。

除此之外，馬化騰還捕獲了31隻獨角獸企業。此前，根據胡潤研究院發布的《2019年一季度胡潤大中華區獨角獸指數》顯示，在上榜的這些企業中，騰訊已成功捕獲了31家。雖然這些上榜的獨角獸企業中的20%都有可能會失敗，但大部分還是會取得成功。這意味著在不久的將來，馬化騰和他的騰訊可能會獲得豐厚的回報。

值得一提的是，騰訊還以5132億元的海外資產，超越了華為、萬達等知名企業，成為中國最大的跨國民營企業。據悉，騰訊已經在海外投資了大量資金，僅去年一年就在160多個國家進行了投資。

所以說，按照這樣的發展態勢，馬化騰在未來是很有可能登頂亞洲首富之位的。未來，馬化騰究竟能否成功摘下亞洲首富的榮譽，就讓我們拭目以待吧。
