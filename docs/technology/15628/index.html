PUA培訓公眾號"浪跡情感"已被封號：影響惡劣

2019-12-13 1,255條評論 2,269次閱讀 665人點贊

科技訊12月13日消息，目前PUA培訓公眾號「浪跡情感」已被屏蔽，頁面顯示，「接相關投訴，因違反《互聯網用戶公眾賬號信息服務管理規定》已被屏蔽所有內容，賬號已被停用。」

PUA全稱是Pick-up Artist，字面上看，是搭訕藝術家，但實際上PUA的定義已從簡單的搭訕擴展到整個兩性交往流程，直到發生親密接觸並確定兩性關係。

騙財騙色、迫害女性，PUA自傳入國內以來就聲名狼藉，廣泛被媒體關注，此前咪蒙在一篇文章中這樣描述PUA：「在這群垃圾眼裡，女生只分為兩種：睡過的，和沒睡過的。」

作者：藍鯨財經

繼去年的「含咪率」之後，「含P率」再度成為今日的話題熱詞，人們紛紛在朋友圈測試有多少好友關注了與PUA相關的賬號，「含P率」越高則朋友圈中渣男越多。

簡而言之，PUA（Pick-up Artist）是一種「搭訕藝術學」。實質上是通過一系列包裝和話術演繹，給目標對象洗腦，並逐步演化成騙色、騙財、誘姦等，最終達到情感操控的目的。

「浪跡情感」創始人劉欣多陣地布局PUA業務

《如何讓富婆愛上你？》

《如何撩內向的乖乖女？》

《浪跡快手直播：實戰聊天之曖昧法則！》

《6w姐：一片難求，身價暴漲，綠帽奴就是賤》

《相親女：「男人30歲連100萬都沒有，打一輩子光棍吧！」》

《結紮男喜極而泣：「昨夜仙人託夢女友，恭喜她懷上了我的孩子，還是個龍種！」》

上述標題出自公眾號「浪跡情感」，今日早間該號的關注好友僅不到10個，截至發稿，關注好友已暴漲到100個。憑藉豆瓣一則「辨識朋友圈PUA男」帖，該公眾號實現了無費用推廣一天漲粉數萬的現象級成就。

「浪跡情感」標榜為「中國最權威的男性情感教育平台」，記者通過查詢發現，該公眾號於2017年4月註冊成功，最初名為「浪跡情感教育」，而後改名為「浪跡教育PUAMAP」，2018年9月公號再度改名為「浪跡情感」。

從2017年至今，公號發文1276篇，平均閱讀超過2w+，最近幾篇推送閱讀更是高達10萬 +，文章內容也多以「教男性如何搭訕、撩妹」為主。除了公眾號上的教學外，該自媒體在快手、YY上上亦有教學直播，預告中稱全明星導師教你如何「撩妹、搭訕、製造新鮮感……」


自媒體推送教學、YY快手直播教學只是冰山一角，該公號所屬公司的法人更有涉及PUA在線教學、機構培訓、軟體開發等業務。


通過查詢主頁發現，「浪跡情感」賬號主體為成都南院科技有限公司，天眼查數據顯示，該公司成立於2017年，註冊資本100萬，法人為劉欣，持股99.5%。

值得注意是，成都南院科技僅是劉欣持股的一家公司，除此之外，其名下還有成都浪巢文化傳播有限公司、成都浪遊科技有限公司、成都南院文化傳播有限公司、成都浪遊影視文化有限公司4家公司，分別持股24%、70%、25%、95%。


以成都浪遊科技為例，該公司涉及遊戲業務，旗下研發有「帝王把妹」這一軟體，據介紹稱，該遊戲是一款可以教授玩家多種戀愛技巧的養成rpg手機遊戲。



但說到5家公司中體量最大，最知名的當屬成都南院文化，該公司主打的是一款名為浪跡教育的app，以男性情感在線教育諮詢為主要服務內容。介紹稱，公司擁有多年男性戀愛培訓服務經驗，旨在幫助男性掌握情感奧秘，開設私教、實戰、全托等課程，通過不斷的專業訓練，幫助用戶最終實現情感自由。




南院文化傳除了劉欣，王環宇才是最大的股東，持股70%，作為「浪跡情感」的創始人、南院文化的CEO，王環宇在PUA圈內被奉為「宗師級」人物。

2015年創辦浪跡教育，3年時間，浪跡教育已成長為PUA領域巨頭，形成一套成熟的組織架構。公司走PUA教學業務增值路線，課程形式包括直播課、免費視頻課程、收費視頻課程、私教課等，收費在幾百元到一萬七千元不等。

然而王環宇、劉欣在PUA的教學之路上也並非一帆風順，記者發現，南院文化此前涉及到多次官司。其中一例，2017年9月，4名學員將「浪跡情感」所屬的PUA公司告上法庭，要求返還學費2.98萬元。今年4月12日，四川省成都市青羊區人民法院一審認定「浪跡情感」的培訓內容違反社會公序良俗，判決雙方合同無效，要求其所屬公司返還學費。


上述幾家公司對應的業務各有不同，但歸根究底其業務始終與PUA有關。成都南院科技旗下設有微信公眾號（浪跡情感）、微博（浪跡情感）、快手（浪跡學長）等自媒體陣地；成都南院文化旗下設有浪跡教育app，負責在線授課；成都浪遊科技旗下研發有「帝王把妹軟體」。

由此可見，劉欣與王環宇圍繞PUA已然形成了一套完整的商業閉環。

抖音快手成套路女生大型授課場所

騙財騙色、迫害女性，PUA自傳入國內以來就聲名狼藉，廣泛被媒體關注，此前咪蒙在一篇文章中這樣描述PUA：「在這群垃圾眼裡，女生只分為兩種：睡過的，和沒睡過的。」

事實「浪跡情感」此類PUA教學的機構、自媒體不在少數。通過微信搜索「搭訕、撩妹」等字眼便可蹦出數十個相關的公眾號。


隨著短視頻行業的飛速發展，抖音快手等平台更是成為套路女生的大型授課場所。例如點開快手發現，相關賬號發布的均是「如何打動學姐、如何打動學妹」等內容。抖音上與搭訕、撩妹相關的賬號更是多達數十個，粉絲超過百萬。



雖然性質上不及PUA惡劣，但不可否認的是為迎合單身男性群體，各個平台正在滋生出越來越多套路女生的所謂教學內容。

業內人士表示，目前針對目前PUA行業缺乏有效監管，挑戰社會倫理道德、侵犯人身、財產權利，此類行為顯然違背公序良俗和法律規定。各平台應該承擔內容審核的職責、自律自查外，執法部門也應加大平台內容監管力度。
