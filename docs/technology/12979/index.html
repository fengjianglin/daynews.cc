理想汽車12月開始交付，但合作銀行被曝不給放款

2019-12-12 1,296條評論 2,511次閱讀 994人點贊

（觀察者網訊 文/潘昱辰 編輯/徐喆）

「這次購車體驗真是糟心！」日前，有自稱造車新勢力理想汽車預定車主的用戶在內部群爆料稱，自己通過銀行貸款購買即將交付的理想ONE，銀行卻沒有放款。車主諮詢銀行方面卻被告知，是理想汽車單方面原因造成銀行不放款，「具體原因由理想汽車負責解釋」。

理想汽車由前汽車之家創始人李想於2015年創立。今年11月，理想汽車宣布首款量產車——2020款理想ONE正式下線，並於12月開始交付。

根據理想汽車提出的金融方案，理想ONE支持交通銀行、中信銀行、江南銀行和招商銀行等有息貸款購車，年利率為3.96%-6%不等，貸款期限可分為12期、24期和36期。


首台量產理想ONE已於11月下旬下線，並於12月上旬開啟交付

業內人士認為，理想汽車在交車的節骨眼上與銀行產生矛盾，或與其收購的力帆汽車的債務糾紛及訴訟有關。

2018年12月，理想汽車以6.5億元收購重慶力帆汽車有限公司100%股權，從而獲得了乘用車生產資質。今年1月，重慶力帆汽車有限公司正式更名為「重慶理想智造汽車有限公司」。

企查查數據顯示，12月2日，重慶理想智造汽車有限公司新增2條被執行人信息，執行標的累計超過人民幣3523萬元。而且，這已經是重慶理想智造（力帆汽車）今年內第三次被法院列為被執行人。此外，重慶理想智造涉及的訴訟多達138起。


圖片來源：企查查

12月3日晚間，理想汽車發布聲明稱，重慶理想智造所涉訴訟均為力帆集團持有並經營力帆汽車期間發生的債務，訴訟案件發生後，均由力帆集團與相關債權人對接並尋求解決方案，後期案件的處理也由力帆集團負責。

理想汽車強調，目前公司資金狀況健康良好，日常業務和生產經營不受訴訟影響。

今年上半年，理想汽車營業收入約為527.76萬元，凈利潤約虧損6.29億元，資產總額約為58.41億元，負債總額為9.31億元。不過在今年8月，理想汽車曾獲得由美團王興領投的5.3億美元(約合人民幣37億元)C輪融資。

然而，力帆汽車自身的處境卻是岌岌可危。根據力帆股份公布的銷量數據，今年10月，力帆汽車乘用車銷量僅為29輛，同比下降99.25%；今年累計銷量為2.2萬輛，同比下降73.5%。 截至目前，力帆汽車尚未公布11月份的銷量數據。


2018年的廣州車展上，力帆展台已頗為冷清；2019年的廣州車展上，力帆品牌早已不見蹤影

財務數據方面，力帆僅今年前三季度的虧損就超過26億元，總負債更是超過了178億元。

今年10月，網曝一份銀行文件稱，力帆汽車將於年底進入破產程序，但遭到力帆方面否認。另據財聯社等媒體報道，為拯救力帆汽車，當月重慶市政府已召集地方金融辦及相關銀行機構債權人等，幫助力帆汽車組織成立債委會，要求各銀行「不抽貸、不壓貸、不斷貸」。

儘管理想汽車試圖與深陷危機的力帆汽車分割關係，但或許在銀行等金融機構看來，力帆的債務與訴訟遺留仍會影響到理想汽車的信譽。截止目前，理想汽車尚未對車主的質疑作出公開回應。

(原標題：曝合作銀行不給理想汽車放款，或因擔憂力帆連帶債務)
