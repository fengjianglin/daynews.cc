人工智慧時代來臨，未來孩子們拿什麼跟機器人拼？

2019-12-21 328條評論 2,802次閱讀 1485人點贊

無人駕駛公交車

9月底，武漢正式發放自動駕駛牌照，首批自動駕駛車輛的商用牌照下發。這意味著無人駕駛汽車技術終於邁出了最關鍵的一步——「正式發放牌照、允許上路」。也許不出幾年，我們就要和駕照Say Goodbye!

人工智慧早已侵入我們生活的方方面面。

無人超市過閘機

2017年，馬雲的第一家無人超市在杭州開業。200平的超市沒有一名員工，掃碼進店，自行選購，直接走人！整個過程簡單流暢，客戶拿了商品後經過大門時自動扣款，無需再進行掃碼支付的操作。

使用「小勤人」前後工作效率對比

還是2017年，一個叫「小勤人」的財務智能機器人震驚了財務圈。只需幾分鐘時間，「小勤人」就能完成財務幾十分鐘才能完成的基礎工作，並且可以7*24不間斷工作！財務人的工作地位岌岌可危！

人工智慧滲透日常工作和生活，不禁想讓人發自內心地問一句：

難道人工智慧和我們搶飯碗的時代就已經來臨了？

這絕不是危言聳聽。麥肯錫的研究表明，到2020年，全球將會有700萬個工作崗位消失。

如果你的工作符合以下特徵，那麼，你被機器人取代的可能性非常大：

l 大量的重複性勞動，只需手熟無需動腦；

l 無需天賦，經由訓練即可掌握的技能；

l 工作環境相對封閉，兩耳不聞天下事。

如果你的工作包含以下三類技能要求，那麼，你被機器人取代的可能性非常小：

l 社交能力、協商能力、以及人情練達的藝術；

l 同情心，以及對他人真心實意的扶助和關切；

l 創意、創造和審美。

人類與機器人最明顯的優勢，就是社交能力、情感交流以及創造力。因為這些都是無法用數據去分析、預測的。

就教師來說，即便機器人擁有更豐富的知識，但教育除了教書還有育人，「以理服人、以德育人」的境界人工智慧很難達到。

機器人可以替代人類完成工作，卻無法像人類一樣進行創造。

作為家長，在這樣的時代，除了讓孩子學習知識，更重要的是從小培養他的學習能力和創造力。時下最流行的編程教育，便是培養孩子學習能力和創造力的一種途徑。

編程俠課堂

少兒編程給孩子帶來的能力和素質的提升是全面性的，除了上面提到的這些重點的好處，學習編程還能加強孩子的邏輯性、創造力和嚴謹性，培養孩子的組織溝通能力、領導力、跨學科能力和敢於試錯的勇氣等等。

編程俠課堂

麻省理工教授雷斯尼克曾說：具有終身創造力的孩子，才能更好地面對不確定的未來。

人工智慧的時代，創造力是保證孩子立於不敗之地的關鍵能力。與其面對不確定的未來而惶惶不安，不如從現在開始投資未來。
