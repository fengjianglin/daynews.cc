蘋果新手機回歸經典，好比iPhone 4S配A13處理器，會有人買單嗎？

天天要聞
2019-12-22

Hello大家好，我是兼容機之家的咖啡。

最近在網上很是流行這樣一張圖片，這一張關於蘋果2020款iPhone的圖片，上面羅列著蘋果未來可能發布的幾款手機，屏幕尺寸從4.7寸到6.7寸的都有。

相比今年的發布的iPhone11系列，只有三種屏幕尺寸，蘋果這次是有意通過屏幕尺寸，來推出多種產品，以此來擴充自己的產品線。

還有傳聞就是蘋果下一代iPhone的設計，會回歸到iPhone 4和iPhone 4s上面的風格，就是金屬中框加上雙面玻璃的方正設計。這也是蘋果最為經典的設計之一，是當年喬老爺子手下誕生的手機，不光是外觀經典，更多的是個人情懷的寄託。

經過那麼多年的發展，iPhone的性能相較之前已經有了很大的提升，現在誰家還有iPhone 4或者iPhone4S，拿出來作為主力機使用一定是卡的要死吧。如果，這裡僅僅說的是如果，iPhone將4S配上全新的A13晶元，重新發布還會有人為情懷買單嗎？

我們知道iPhone 4S是蘋果公司推出的第五代iPhone手機，喬老爺子正好是在iPhone 4s發布的時候去世的，所以這裡的「S」被輿論稱為「iPhone for Steve」，所以在果粉裡面，iPhone 4S的意義非凡。

也正是因為如此，如果iPhone 4S解決了性能落伍這一事實，再次發售，介於這款手機的果粉心目中的地位，答案是肯定不缺人購買的。但是，就算這款手機配上最先進的A13處理器，應該不會有人把這款手機當做主力機來使用。

原因

第一，iPhone 4S的CPU型號是A5，而最新的iPhone 11的CPU型號是A13，A13的性能比A5強太多了。就算攻克了把A13安裝在iPhone 4S上面的技術難題，但是，介於iPhoneSE 4S上面的硬體，是很難真正意義上發揮A13的性能，這裡就有點大材小用的意思了。

第二，屏幕尺寸，雖然很多人喜歡小屏幕，但是iPhone 4S上面搭載的3.5英寸的屏幕，在當年可能不算小，但是放在現在來說，簡直就是「兒童手機」了。

如今安卓手機的屏幕動不動都是6英寸以上，蘋果手機最低的也有4.7英寸，被大屏幕教化過的消費者，真的能接受這麼小的屏幕嗎？

第三，使用習慣。現在的手機都是指紋解鎖，無需操作一秒開機，還有FACE ID更加方便，如果在回歸到蘋果iPhone 4S上面的「滑動解鎖」，先不說麻煩不麻煩的問題，安全性也很難得到保證。

當然還有其他的使用習慣上面的問題，主要也是集中在系統層面，但是如果你說蘋果順便更新了iPhone 4S的系統，就當我沒說。

第四，續航，iPhone 4S手機尺寸：115.2×58.6×9.3mm，手機重量：140g。這麼點大小的機身裡面，只能夠塞下1430mAh的電池，如果換成了A13的處理器，那續航不得分分鐘尿崩。

其他的就是些手機硬體跟不上時代了，比如攝像頭方面。蘋果手機的攝像頭素質一般都是非常好的，領先於同時代的手機，但是如果現在用上iPhone 4S上面的後置800像素的攝像頭，那畫質，真「酸爽」！

其實說了那麼多，人們不是在意iPhone 4S上面的小屏幕，也不是iPhone 4S上面那些落後的硬體。而是iPhone 4S的經典方正設計，再加上雙面玻璃+金屬中框。

其實蘋果有可能也關注到了這個問題，近期關於iPhone 12的種種消息都顯示，蘋果準備給新iPhone換個設計，而最近的放出的iPhone 12的渲染圖都是經典的雙面玻璃加金屬中框的設計。

並且，這次iPhone 12一改之前圓潤的設計風格，回歸到iPhone 4S上面的方正設計。我想這也是眾多消費者所希望的，不過背部依舊是浴霸三攝，至於前面的劉海屏，就是最有爭議的部分了。有人說下一代iPhone會去掉劉海，但是我並不認為蘋果這會這麼激進。

總結

如果蘋果出個小鋼炮，那麼很多小夥伴肯定會願意為之掏腰包的，所以iPhone SE2可能就會應運而生。另外很多消費者喜歡蘋果方正的設計，不知道蘋果的設計師能不能聽進消費的呼聲。
