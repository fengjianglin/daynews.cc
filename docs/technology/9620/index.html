椰子遊戲董事長喻葉：通過認證系統把傳奇私服平台化 幫助小企業合法經營

2019-12-10 326條評論 4,838次閱讀 2237人點贊

椰子遊戲董事長喻葉：通過認證系統把傳奇私服平台化幫助小企業合法經營

2019年12月6日，宜春市宜陽新區管理委員會(以下簡稱「宜陽新區管委會」)、深圳市椰子互娛網路技術有限公司(以下簡稱「椰子遊戲」)、浙江世紀華通創業投資有限公司(以下簡稱「世紀華通」)和盛趣信息技術(上海)有限公司(以下簡稱「盛趣遊戲」)，在宜春市宜陽新區簽署了《宜陽新區招商項目合作協議之補充協議》。

椰子遊戲董事長喻葉女士(左一)代表椰子遊戲與宜陽新區管委會、世紀華通和盛趣遊戲簽約

10月22日，宜陽新區管委會、椰子遊戲和世紀華通三方簽署了《宜陽新區招商項目合作協議》。世紀華通攜旗下盛趣遊戲聯合椰子遊戲在江西省宜春市簽約落地「國民傳奇產業園」並共同發起設立「國民傳奇產業聯盟」。同時，世紀華通和椰子遊戲聯合投資的江西省國民傳奇網路技術有限公司(以下簡稱「國民傳奇」)、江西傳奇創盟科技發展有限公司(以下簡稱「傳奇創盟」)、江西夢回傳奇影視文化有限公司等三家公司均已註冊落地。

此次為了更好地履行該協議，世紀華通及其關聯公司盛趣遊戲與宜陽新區管委會、椰子遊戲協商一致，簽署了補充協議。

簽約儀式後，《環球網》對椰子遊戲董事長、國民傳奇CEO喻葉女士進行了專訪。喻葉稱，「國民傳奇產業園」是目前中國唯一一個基於一款遊戲IP構建的產業園。產業園建立的宗旨和目的，就是為了讓整個產業實現大一統，解決傳奇市場上私服、版權各種亂象以及申請版號等問題。

椰子遊戲董事長喻葉女士(左)和盛趣遊戲董事長王佶先生(右)接受《環球網》專訪

「更為重要的是，地方政府對產業園的支持力度非常大，宜春市政府出了紅頭文件，所有落戶企業通過統一的通道都可以主張自己的權利，獲得司法、安全等方面的保護，相關刑事、民事方面問題也能夠高效解決；另外我們還有一個「國民傳奇產業園」理事會，企業家可以把各種訴求聯名提出來。」喻葉接著說。

據悉，傳奇創盟將聯手宜春大數據中心，建立全方位的版權監測與認證系統，依託高科技手段和規範的運營管理制度，制定授權-認證-運營的標準流程、監控企業侵權違法行為、打擊網路違法犯罪、支持合規企業發展。

關於傳奇遊戲著作權認證服務中心的建立，喻葉介紹稱，我們希望能夠在宜春市宜陽新區傳奇產業園整個私服產業打造過程中，通過認證系統的虛擬認證，能夠把傳奇私服整體平台化，幫助小企業實現認證和合法化，虛擬認證是針對PC的傳奇私服，實名認證是針對傳奇市場的手游合法化。

「長遠來看，認證系統是以傳奇這個IP先做一個案例工程，如果傳奇的IP市場能夠獲得保護，也就意味著未來遊戲產業不論是傳奇IP，還是世紀華通系統的其它IP，甚至大廠包括中國遊戲行業的其它IP，甚至輻射到音樂，輻射到金融，輻射到其他知識產權保護，都能形成一個版權保護的生態。」喻葉進一步講到。

椰子遊戲董事長喻葉女士接受《環球網》專訪

那麼，國民傳奇產業聯盟未來有什麼發展規劃呢？喻葉回應稱：「我們今年的規劃就是先落戶並成立理事會，明年第一件大事肯定是版號，我們希望幫助所有的企業以產業園的形式最高效的獲得版號。只有擁有了版號，企業才能安心去做智力成果開發，發行廠商也能夠去支持你，因此你既有合法的權利又有合法的上線資質，所以明年的規劃實際上就是實現產值。」

最後，喻葉稱：「無論是世紀華通、椰子遊戲、宜陽新區以及盛趣遊戲，我們都非常專註於能夠把國民傳奇產業園腳踏實地建設好，幫助眾多傳奇內容提供商、發行商，從權利的鏈條到內容的製作都能夠高速、高效發展。希望用我們的智力成果來做出經濟貢獻，反哺到支持我們的宜春市政府，反哺到當地的人民。」

椰子遊戲創設於2014年，短短數年就成長為集互聯網內容提供商、運營商和發行商於一體的綜合性泛娛樂產業集團。公司業務涵蓋國內明星IP運作、知識產權保護與增值服務、移動端遊戲開發與運營、影視劇攝製發行、IP衍生品開發等多個板塊，形成較完備的產業鏈生態圈。

以下為專訪實錄：

《環球網》：喻葉總你好！非常感謝在百忙之中接受環球網的專訪，首先介紹下宜陽新區給我們提供哪些支持？

喻葉：宜陽新區建立的國民傳奇產業園，實際上是要往省級甚至國家級產業園的方向發展。首先我們是希望通過產業園的形式，打造中國唯一一個以IP構建產業園的創舉，其次產業園能夠幫助這些企業統一規範首選分發問題。

每一家在宜陽新區落戶的企業，未來版號申請能夠得到統一的規範管理，以宜陽新區國民傳奇產業園的形式向相關主管部門申請，規範性更加嚴謹；其次，每個企業在做互聯網遊戲過程中，都會遇到很多司法訴訟問題，尤其是傳奇產業每年都有很多的官司，但是這些訴訟遍布在全國各地，周期又非常長，其實很多都是基於一個訴訟事實和理由，只是不同主體公司。

盛趣遊戲作為傳奇遊戲著作權權利人，其全資子公司已經落戶在此，進行權利IP的分發。這就意味著所有落戶在宜陽新區的內容開發商、大型廠商以及小廠，他們獲得的這份權利，基礎簽約地是在宜春，意味著宜春能夠統一的為這些中小包括民營經濟企業來進行司法保護。

雖然有統一的訴訟標的、訴訟事實和訴訟理由，但是時間周期要經歷兩到三年，這對於互聯網遊戲等新興行業實際上是折騰不起的，我們的民營經濟非常需要知識產權保護環境下的高效司法環境。因為一個企業對於開發遊戲的周期是兩到三年，如果一個訴訟拖兩到三年，就意味這些創新、民營、實體經濟，特別是新經濟模式企業創業的這些年輕人，沒有獲得最基本權利的保護和認證，導致因為訴訟而破產，因為訴訟而無法融資。

因為現在很多資本對於企業的訴訟比較敏感，企業要併購、上市和資本運作，如果核心產品涉及訴訟會導致大量的問題。這也是為什麼我反覆的提到，我們的內容評估商、發行商，統一到國民傳奇產業園來進行IP的權利人管理，以及我們這些企業能夠獲得司法保護，安全保護，以及相關的刑事、民事等經濟方面的問題，能夠高速有效解決。其實本質還是得力於當地政府的支持，對我們的司法和安全保護，宜陽新區已經成立專項組，不論是司法上面的保護專項，還是公檢法系統的專項小組，宜春市政府出了紅頭文件，就是為所有的落戶這些企業家，不論大廠、中廠還是小廠，只要你是創新創業公司，來到這裡首先你能主張自己的權利，我們有一個統一的通道，還有一個國民傳奇產業園的理事會，這些企業家能夠把自己的訴求聯名的提出來解決。

很多時候一個小民營企業，訴求投到相關的主管部門，就相當於針投到大海里石沉大海。但是當你形成一個產業的聯盟，你代表的是2144億產值裡面600億，甚至加上私服非法的市場近千億的產值。那麼一個產業的管理，不論是在版號規範，還是其他核心訴求，都會得到高效的重視和解決。

因為我相信在中國黨的領導下，以及宜春市政府的幫助下，我們這些年輕創業者年輕的企業家們，因為我們還年輕，代表著未來生產力，也代表著新經濟模式與傳統產業的結合下未來新的生態體。所以這就是我們來到這裡的核心，也是我們獲得宜春市政府保護的最基本的建設這個產業園宗旨和目的。

《環球網》：謝謝，我們這次承諾函裡面提到了，建立國民傳奇的遊戲著作權一個認證服務中心，請您再給我們介紹一下這個認證中心的一個具體職能和範圍？

喻葉：在宜陽新區椰子遊戲和世紀華通已經落戶了三家公司，分別是國民傳奇、夢回傳奇以及傳奇創盟公司，我們的國民傳奇主要是續熱血傳奇和傳奇世界IP以後，椰子遊戲攜手世紀華通成立國民傳奇網路技術有限公司，是想孵化出一個原創IP，大家都知道原創IP包括中國的知識產權保護，還是以新青年以自己的智力成果去創造或打造。

現在到處都在做IP孵化工作，傳奇明年將步入整整20周年。市場上也希望他有新生命的產品和相關聯動的工作出現。所以夢回傳奇是一家以傳奇IP孵化衍生品的公司，包括我們跟湖南台簽的十年投資戰略聯合孵化開發協議，希望在10年規劃的當中能夠孵化四部電視劇、四部電影、四個漫畫、四個遊戲。

三家公司的落地本質上來說，首先從權利鏈條我們清晰化的以一個IP去打造一個國民傳奇產業園的生態；其次保護所有中小以及傳奇產業的企業們，能夠讓他們真切的感受到我們權利鏈條，在未來的傳奇生態中更多的CP能夠專註去把產品打造好，因為針對市場上開發商，發行商，再到併入華通所有這些CP與世紀華通一起，能夠把這個傳奇產業園落地在宜陽新區，是基於宜陽新區政府投資來打造的認證系統。

所以傳奇創盟公司就賦予了這套認證系統的責任和使命，傳奇創盟公司是世紀華通和椰子遊戲聯合成立的一家合資公司，宗旨是希望能夠幫助更多的中小型企業，能夠合法合規的把自己的產品上線。很多產品一直沒有獲得保護，導致市場上大量的侵權，從PC的私服到手游的合法化沒有獲得解決，所以這套認證系統是虛實認證的，私服市場一直以來都是用的虛擬名稱，開了一個私服，沒有公司也沒有組織就是一個作坊形式。

傳奇私服產業最早是基於玩家太熱愛這款遊戲，而變成了一個作坊，在我的理解實際上他們是一些創意新青年，三到五個人因為太熱愛遊戲，不懂法律也不懂得知識產權保護，不懂傳奇IP授權鏈條，所以私設了伺服器。我們希望能夠在宜春市宜陽新區包括創盟公司整個私服產業打造過程中，通過認證系統虛擬認證，能夠把私服整體平台化。傳奇私服市場也幫助很多市場中的中小手游廠商，因為大廠基本盛趣遊戲都在全面的給他們IP授權了，建立這麼多年的這些聯繫包括相關的體制規範，但是很多小廠不論在網路遊戲，客戶端遊戲，手機遊戲，H5，甚至以後衍生作品當中，侵權現象還是非常泛濫。

所以希望組建的創盟公司幫助這些小企業能夠合法和認證，虛擬認證是針對PC的傳奇私服，實名認證是針對傳奇市場的手游合法化。認證系統整體來說是以傳奇這個IP先做一個案例工程，如果傳奇的IP市場能夠獲得保護，也就意味著未來遊戲產業不論是傳奇IP，還是華通系統的其它IP，甚至包括中國遊戲行業的其它IP，甚至輻射到音樂，輻射到金融，輻射到其他等等知識產權保護，都能形成一個版權保護的生態。

《環球網》：好的，我相信我們很多企業願意加入到我們這個聯盟當中來，那麼盛趣遊戲是基於哪些考慮做出承諾函里增加範圍和力度的決策呢？

喻葉：首先盛趣遊戲作為傳奇的權利人，傳奇市場的亂象一直非常的嚴重。那麼國民傳奇產業園首先它的宗旨和目的，就是為了讓整個產業能夠實現大一統，能夠解決傳奇市場上私服、版權等各種亂象問題以及申請版號的問題。

所以盛趣遊戲落戶到宜陽新區，本質上來說是三個宗旨。第一個就是合法合規、把整個產業大一統；其次對於版權的分發，進行大廠管理和小廠維護，能夠把整個傳奇的生態鏈規範化進行治理管理，以及幫助很多傳奇的小廠發展起來。包括做出更多的好的產品，因為現在是一個內容為王的年代，所以手游的產品精品化是我們核心需求。無論是世紀華通、椰子遊戲、宜陽新區以及盛趣遊戲，我們都非常專註於能夠把這個國民傳奇產業園腳踏實地的建立好，能夠幫助很多的傳奇內容提供商、發行商，從權利的鏈條到內容的製作都能夠高速、高效並且能夠長治久安的發展。

《環球網》：最後，我們的國民傳奇產業聯盟的未來發展規劃是什麼？明年的側重點在哪些方面？

喻葉：首先，我們預估是在12月底進行國民傳奇產業園開園發布會和國民傳奇理事會成立，今年的規劃就是先落戶和成立理事會，明年規劃本質上來說第一件大事肯定是版號，希望幫助所有的企業以產業園的形式最高效能夠獲得版號，就像這個資質車檢一樣，因為你只有擁有了版號，你才能安安心心去做智力成果的開發。然後這些發行的廠商也能夠去支持你，既有合法的權利又有合法的上線的資質。所以明年的規劃實際上就是產值，許多到國民傳奇產業園來的企業里有很多上市公司，實際上他們都是當地的納入大戶。

把某一個核心的產品發展到一個產業園的形式，所產生產業園的產值對於宜春新經濟和傳統經濟結合的影響是什麼？我們也是在拭目期待的。所以希望用自己的智力成果來做出經濟貢獻，能夠反哺到支持我們的宜春市政府，反哺到當地的人民。因為一切都是取之於民，還是用之於民的。

《環球網》：非常感謝，我們也祝願國民傳奇產業生態園能夠早日的落成，也希望在盛趣遊戲和椰子遊戲的不斷努力之下，能夠更好發揮我們傳奇IP這一經典的大IP。非常感謝喻葉總接受我們的採訪。
