淘集集今日將宣告破產：支付寶賬戶已被凍結

2019-12-09 924條評論 2,597次閱讀 110人點贊

澎湃新聞記者 吳雨欣

號稱擁有1.3億註冊用戶的社交電商平台淘集集，自救夢碎。

12月8日晚，澎湃新聞（www.thepaper.cn）記者從淘集集處證實，淘集集因資金鏈斷裂而進行的併購重組計劃宣告失敗，將於12月9日正式宣布破產。「簽了SPA協議，但資方打款一直在拖。」該人士告訴澎湃新聞記者。


淘集集隸屬於上海歡獸實業有限公司，主攻下沉市場。2018年8月上線，10月就獲得A輪融資，融資金額為4200萬美元，估值達到2.42億美元。

12月8日晚上，網上流傳出淘集集創始人張正平給員工的公開信。張正平在信中表示，遺憾加抱歉的是，沒有帶著大家把淘集集做成，接下來，淘集集會成立員工善後處理小組，跟進大家的離職辦理、社保問題解決、個人物品拿取等各項事情。從這封公開信中可以看出，淘集集在11月28號支付寶賬戶被凍結，即淘集集所有賬戶都已被凍結，使得員工11月的工資沒有發出，11月社保也無法發出。張正平在公開信中表示：「哪怕最差的結果走到破產清算，資金也是優先結算員工工資。」

對此，淘集集方面人士告訴澎湃新聞記者：「這封公開信不是最終版，可能是創始人和員工做交流的時候流傳出來，但信里提及的內容是真實的。」

值得關注的是，在12月3日，淘集集還曾發布公告稱，已經與國內大型集團順利簽署股權投資協議，目前正處於等待打款的階段，淘集集正在努力完成打款前所需要的前置條件。

據澎湃新聞記者了解，早在今年國慶期間，張正平就表示，談妥國內某大型集團公司進行重組併購，但這家國內大型集團的名字一直未公布。當時，淘集集負債總額為16億元左右，負債主要分為供應商的貨款和廣告代理投放的欠款兩部分。為此，淘集集展開了一系列「自救」行為，如引入重組方、與商戶協議延期還債、把債務變成股份等形式，淘集集的主要經營模式也由商家入駐模式，調整為合伙人自營模式，現有的供應商被轉為淘集集的股東合伙人。

對於淘集集虧損的原因，張正平也曾做過解釋，淘集集在今年6月初啟動B輪融資，進入7月，淘集集銷售額出現停滯，但自己把過多的時間花在融資身上，延誤了黃金自救期，進入9月，融資遲遲得不到確認，現金流下降，危機來臨，供應商集中上門擠兌貨款。

「我和我的家人及雙方直系親屬，不論國內國外都沒有買過車子房子，那錢到哪去了？淘集集有1.3億註冊用戶，市面上獲取一個註冊用戶並不便宜，大家都知道淘集集不收傭金，虧損實際上都虧損在獲客上。」張正平表示，公司重組是最好的方案。

今年10月31日，淘集集發布公告稱，在10月23日，供應商債權人完成51%的債務重組協議簽訂，10月28日，收到資方書面TS，簽訂投資意向書。淘集集方面稱，只有簽約金額超過51%，投資方才會投資淘集集。
