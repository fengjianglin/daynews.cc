人工智慧產業實力究竟如何？“大道之行II”走進深圳三家標杆企業

天天要聞
2019-12-16

每經記者：歐陽凱 每經編輯：張海妮

大道之行II——去深圳對話好公司現場圖

圖片來源：每經記者 張建 攝

放眼全國，深圳在人工智慧和機器人密切相關的智能智造等領域已形成較為完備的產業鏈，是僅次於北京、上海的熱門人工智慧發展城市，這裡既誕生出工業互聯網企業巨頭工業富聯、雲計算的創變者騰訊雲，也孕育出機器人行業「獨角獸」優必選。那麼如今深圳人工智慧產業發展實力到底如何？人機平台之上的未來商業行動路線圖又是怎樣的？

12月12日至14日，由每日經濟新聞主辦、中國上市公司董秘俱樂部提供智庫支持、復旦大學管理學院及變數中國聯合主辦的「大道之行II——去深圳對話好公司」訪學活動在深圳舉行。在復旦大學管理學院信息管理與信息系統系副教授、智慧城市研究中心副主任胥正川的帶領下，來自全國各地的近40位上市公司董秘及其他高管組成的訪學團，訪問了深圳在人工智慧領域的三家標杆企業——工業富聯、騰訊雲和優必選。

激活工業製造新動能

2017年，「人工智慧」第一次出現在兩會政府工作報告中。同年7月，國務院首次印發《新一代人工智慧發展規劃》，明確指出，2020年，中國要跟上全球領先國家的AI技術與應用，2030年更要成為全球頂級的AI創新中心。有數據預測，到2030年，中國GDP中AI將有高達20%的貢獻。

那麼人工智慧究竟能為龐大的製造產業群賦能什麼？製造業在過去很長的一段時間裡，是「勞動密集型」產業的代名詞。作為富士康旗下的工業富聯，便是轉型「智能製造+工業互聯網」的排頭兵。自2018年6月上市至今，工業富聯多次提及「智能製造+工業互聯網」轉型，並要成為這一領域的引領者，以此激活工業製造新動能。

13日上午，訪學團參觀了工業富聯的工業互聯網展廳、柔性裝配作業智能工廠，驚訝地發現，一度被外界誤認為是「代工廠」的工業富聯，其實已成長為全球領先的通信網路設備及高精密機構件、雲服務設備、精密工具及工業機器人專業設計製造服務商。

拿工業富聯的「關燈工廠」來說，便是工業AI及機器學習在工業現場場景下的融合應用。以某電子產品生產為例，該產品全製程包含表面貼裝、組裝、測試、包裝完整生產流程，在不同的製程段導入了各式自動化及AI應用。據介紹，在關燈工廠的運作中，自動化導入是前提，數據採集是基礎，平台構建是核心，工業AI和機器學習是「無憂生產」的關鍵。

工業富聯首席數據官劉宗長向訪學團介紹，在技術不斷革新的當代，在工業互聯網的浪潮下，AI成為製造業的制勝法寶，模型演算法下的機器學習、深度學習、遷移學習，加上智能預測、感測器偵測的技術能力，AI武裝的工廠大腦讓無人化關燈節能工廠成為現實。

傳統工業企業如何升級？

事實上，隨著全球工業加速向網路化、數字化、智能化發展，工業互聯網成為推動製造業高質量發展、數字經濟倍增的新引擎。但是對於傳統工業企業升級來說，面臨的現實問題也很多，例如怎樣把大數據與大機械結合起來，促進物理世界與數字世界的有機融合？如何通過數字化提高全要素生產率，提升人機協作水平？

訪學團在13日下午走訪參觀了騰訊，以騰訊云為基礎，騰訊對外輸出數字化、智能化轉型針對性的解決方案，讓訪學團頗為關注。

《每日經濟新聞》記者注意到，今年10月29日，在騰訊全球數字生態大會成都峰會現場，騰訊雲發布智能製造全新解決方案品牌WeMake，全方位助力工業企業數字化升級。以華星光電為例，通過騰訊雲「工企智能」產品解決方案，實現了人工智慧在AI判片的應用，對比傳統的人力判片方式，AI識別速度提升10倍，縮減人力50%。

騰訊工業雲總經理李向前在分享經驗時，就特別提到上述案例，他指出騰訊在其中投入了很大成本，已經遠遠超出預算，但最終卻沒有叫停。他認為，「（賦能企業）這是門檻太高，一定需要高強度持續投入，可能做一個項目或服務，一個企業是不可承受的，而一旦沉澱出類似產品的東西，便能賦能更多企業，這也是騰訊嘗試在做的」。

但總的來看，不管是工業富聯的「關燈工廠」還是騰訊雲的WeMake，它們的推動和實施都離不開機器人這一載體。作為工業互聯網體系的重要組成部分，機器人承載著大量相關係統、工藝參數、軟體工具、企業業務需求和製造能力，引導匯聚和鏈接著大量工業資源，通過交互協同和迭代優化，為智能的產生提供必要基礎，創造製造業智能化發展的前提條件。

在參觀工業富聯、騰訊雲後，訪學團最後一站來到優必選，從2008年開始投入人形機器人研發，到獲得騰訊領投的C輪融資，投後估值達50億美元，優必選已經成長為服務機器人行業的領導者。

優必選證券事務總監鄧峰坦言，現在的人工智慧還是弱人工智慧，只能在可控場景做有限的智能服務，優必選的策略是先把核心的伺服舵機和運動演算法用在娛樂、教育和消費級機器人上，實現在各個領域內的落地。優必選已經開始布局基礎的核心，如大功率伺服舵機的研發和生產製造，未來搭建好整個機器人平台之後，將會布局非常重要的運動控制演算法。

值得一提的是，此次訪學期間，胥正川還為近40位企業高管圍繞「人機平台與商業未來」主題閉門授課，期間思想碰撞、觀點交鋒、精彩紛呈。
