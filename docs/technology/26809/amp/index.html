安卓授權，說停就停，谷歌：誰說我壟斷，那就壟斷給誰看

天天要聞
2019-12-21

說到移動操作系統，很多人都清楚，在智能手機時代，只有兩種操作系統被大規模部署，谷歌的 android 和蘋果的 ios。 前者被宣傳為開源軟體。 有了谷歌的許可，手機就能正常部署系統和銷售了，而 ios 是 iphone 的專利。

能夠開發操作系統的科技巨頭寥寥無幾，而構建一個完美的軟體生態圈則更加困難，因此谷歌擁有巨大的話語權，許多手機製造商只能傾聽它的聲音。但谷歌也受到了懲罰。例如，它多次因壟斷而受到歐洲聯盟的懲罰。目前已累計被罰94億美元。

雖然谷歌屢屢遭受壟斷控告，但谷歌是否認罰往往是要根據對方的實力。近日，在土耳其，谷歌的Android被懷疑是一個產品壟斷，要求谷歌支付一千六百萬美元罰款，然而谷歌不僅不認罰，反手就是一個聲明，取消了土耳其新手機的安卓授權，意味著在土耳其境內上市的智能手機未來無法載入谷歌的產品和服務。

因為土耳其不僅使用安卓系統，還將各種核心應用程序作為谷歌服務框架的一部分，這意味著土耳其的現有安卓手機也將是是一塊磚頭。谷歌這種對地方政府管轄權的挑戰，實際上已經構成了對別國網路主權的挑釁。

谷歌之所以強大，是因為 android 是很多國家移動網路生態系統的核心，設備製造商和應用程序開發商都離不開它。 隨著智能手機的普及 android 迅速抓住了崛起的機會，以所謂開源、免費的方式贏得了市場。 由於 android 基本上是谷歌公司的產品，理論上它可以被禁用。 此外，谷歌吊銷其許可證的情況並不少見。 今年早些時候，谷歌（google）停止向華為的移動設備提供搜索引擎（search engine）、谷歌應用商店（google app store）和電子郵件等重要服務，因為華為公司已被列入「實體列表」。

當然，並不是所有的Android手機製造商都擔心不能使用它，比如華為。早在很久以前，華為就提高了警惕，居安思危，開始了自己的操作系統計劃。目前，它已經開發了自己的操作系統：鴻蒙。華為表示，鴻蒙系統不僅開放，而且可以適應多種設備，不限於ARM架構，包括手機、電腦、電視、摩托車等。目前，鴻蒙系統已開始用於小型商業用途。例如，比如榮耀的智慧屏幕就已經採用，根據市場用戶的反饋感覺很好。

余承東之前也曾經「暗示」鴻蒙系統可以或許隨時切換得手機上，由於兼容安卓應用，切換系統後原來的數據不會丟失，而且連用戶保留使用習慣都可以得到。現在，華為正在開發與谷歌移動服務（GoogleMobile服務，簡稱GMS）對標但是基於華為自身的的一整套基與華為體系的移動服務HMS，華為還首次向世界各地的黑客發布了一份獎勵公告，，搞大達156萬美元的獎金，獎勵能夠成功攻擊鴻蒙系統內核的黑客。

這裡說一個有趣的故事，華為移動服務英文全稱為Huawei Mobile Services，縮寫為HMS,這個S全是Services——意思是支持軟體應用的數據服務，結果有一些英語不好的媒體誤以為是HongMengSysterm——鴻蒙系統，當時華為自研系統並沒有官方命名，華為覺得這個名字還不錯，就將錯就錯把自己開發的系統暫且定名為鴻蒙（HarmonyOS）。

毫無疑問，華為已經建立了自己的HMS服務，以改善鴻蒙的應用生態，而這出：我打我自己的對黑客獎勵行為，是以證明該系統對外界非常安全。關於流暢度上面，余承東表示鴻蒙比安卓流暢的主要原因是代碼使用效率高，安卓擁有一億行代碼但只有8％能用。據華為提供的信息，鴻蒙將在明年舉行關於使用P40旗艦機，不得不說，越來越多的人期待國產系統。
