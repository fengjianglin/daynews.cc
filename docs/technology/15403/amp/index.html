國內最尷尬的3家快遞，速度慢服務差，還讓同行捎快遞？用過沒？

天天要聞
3 天 ago

眾所周知，中國現在電子商務的發展非常成熟，這還要感謝馬雲，如今他已經退休，行業依舊火爆，其實電子商務發展起來之後，很多周邊行業也跟著迎來了春天，比如說快遞行業，佔據市場主要份額的是順豐和三通一達，可你們知道國內最尷尬的3家快遞，速度慢服務差，還讓同行捎快遞？你用過沒？

1、天天快遞，1994年成立，這個快遞可能還是有人聽過，不過也算是三四線品牌了，天天的快遞的速度那是真的慢，同城快遞還可以，全國範圍就很尷尬了，甚至還有用戶投訴他們本來3天到的快遞卻用了10多天，也真是太尷尬了，用戶取快遞的時候，快遞員態度也很差，甚至有時候還要自己找！

2、佳吉快遞，也是1994年成立的一家快遞公司，不過很多人可能聽都沒有聽過這個快遞，作為快遞行業的「選手之一」也是挺尷尬的，他們大多寄的都是大包裹，口碑也不是很好，所以去寄小包裹的人也很少，快遞做成這麼沒存在感，不知道他們怎麼做得下去，你們用過沒有？

3、優速快遞，是一家成立比較晚的快遞，2009年才成立，也是一家規模比較小的快遞，也是一家很奇葩的快遞，有時候接了你的快遞還會讓其他快遞幫忙「捎帶」，出了事情都不知道該找哪個快遞的那種，實在是有點尷尬呀，據說速度也是很慢，服務也差，這樣竟然還沒有倒閉，也是厲害！

其實快遞行業最看重的就是速度和服務了，想想順豐為什麼能做到行業龍頭的位置，不就是靠這兩個獲得的口碑嗎？要是這些快遞還是這樣，恐怕之後也還是會被淘汰，還是要與時俱進，要不然還怎麼發展呢？不知道你們用過這3家快遞沒有？還知道哪些快遞處於尷尬境地的嗎？
