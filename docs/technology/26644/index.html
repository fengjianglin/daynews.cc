打麻將、看B站，以後這都是特斯拉上的常規操作了

2019-12-21 1,019條評論 2,096次閱讀 333人點贊

近日，特斯拉中國官方微信公眾號發布了軟體更新的預告圖，並表示「線索就在此圖中」。從圖中信息來看，特斯拉將通過車機系統OTA更新，導入歡樂鬥地主、歡樂升級、麻將、嗶哩嗶哩視頻、優酷、空氣質量顯示、天氣預報和娛樂日報等應用，內容覆蓋了視頻、遊戲和資訊三大類，並有望於明年第一季度正式推送。

需要說明的是，特斯拉已於今年5月開始了QQ音樂服務的內測，並在6月發布的2019.16.3版本更新包中推送了QQ音樂軟體。按照計劃，國產版Model 3有望於2020年1月25日開始交付。在車雲菌看來，特斯拉擴充娛樂應用即是為首款國產車型增加吸睛度，也意味著完善本土化服務的更進一步。

本土化早已起步，完善尚需時日

Model 3的國產可以說是特斯拉本土化的關鍵節點，不過伴隨Model S和Model X先後登陸國內市場，特斯拉的本土化早已展開。

在出行領域，特斯拉目前在全國範圍內已建成300座超級充電站，覆蓋140餘個城市，投入運營的充電樁超過2200個。不僅如此，國內銷售的車型將谷歌地圖換裝為騰訊地圖，並在今年6月推送了QQ音樂，意在逐步改善了國內用戶用車體驗。

即便如此，當前的本土化服務仍不完善。比如騰訊地圖的便利性不如百度和高德、語音控制系統進國內後只能說聊勝於無、車機QQ音樂暫不支持下載和隨機播放等等。來到中國的海外造車新勢力，在智能人機交互方面有些水土不服。

所以如果預告圖中的本土化應用都能順利落地，至少能夠優化國內用戶的車機網聯化體驗。可作為「新勢力」的鼻祖，僅實現網聯化還遠遠不夠。諸如語音系統等功能，或許仍需要攜手國內科技公司共同完成，因而特斯拉還需要時間讓國內產品具備與海外版同等的功能體驗。

加速本土化，競爭才會更從容

綜合國產Model 3的相關信息來看，標準續航升級版、雙電機長續航版和雙電機高性能版的國標工況法續航里程分別為445km、590km和595km。相比當前市售的純電動車而言，續航能力仍是國產Model 3的最大亮點，其自身也是國內市場中少數中型純電動車之一。看似直接競品不多，但想發揮自身的全部優勢，本土化仍需加速。

預計2020年第二季度上市的小鵬P7，可以說是國產Model 3最大的潛在對手之一。後驅與四驅版分別擁有650+km和550+km續航能力，而基於國內環境打造的人機交互系統較目前的特斯拉更強。蔚來的首款轎車預計2021年登場，將衍生自第二代平台，續航與交互都值得期待。

事實上，無論國內的新勢力還是傳統車企，在牽手本土科技公司與布局智能化交互方面，近水樓台的速度與完成度較海外車企更具優勢。

因而加速智能交互的本土化進程，對特斯拉而言是構建全面競爭力的關鍵。同時Model Y也將在未來引入中國市場，對於這款中型SUV而言，國內的競爭壓力可想而知。提前實現網聯服務等相關領域的本土化，也能幫特斯拉強化Model Y在國內的競爭力。
