中保創（匯拍天下）與北京鵬龍利恆關於汽車資產處置開展戰略合作

2019-12-12 268條評論 1,246次閱讀 135人點贊

近日，中保創（北京）科技有限公司與北京鵬龍利恆再生資源回收公司正式簽署戰略合作協議，依託中保創旗下平台匯拍天下在拍賣領域的行業資源和技術創新優勢，藉助北京鵬龍在北汽集團獨特的的資源優勢，將就汽車零部件、實驗整車、汽車生產線等物資的詢價、競價、理賠等事項，開展多層次、多維度的戰略合作，共同打造互聯網數字化汽車行業相關物資處置在線一站式服務。

中保創（北京）科技將互聯網大數據和人工智慧技術創新運用於提升傳統產業的生產交易效率，專註於受損保險財產殘值處理和理賠追償服務，是綜合性「互聯網+保險+拍賣」移動垂直交易平台，同時與國內保費收入前四十的財產保險公司基本達成戰略合作，2019年第二季度買家會員即將突破二百萬。

北京鵬龍利恆再生資源回收公司是北汽集團旗下專業的再生資源物資回收公司，專註於汽車生產過程中所產生的再生資源處理服務，擁有國家多項專項許可，是國內再生資源回收領域，尤其汽車行業同類業務的領導者。

此次中保創（北京）科技有限公司與北汽鵬龍強強聯合，是中保創在險後物資業務基礎上向傳統業務新領域拓展的重要標誌，雙方基於信任開展此次合作。可以看到的未來是，雙方還將以本次戰略合作為契機，運用AI、大數據和互聯網的運營經驗提高汽車物資供應鏈的流通效率，合法、合規併合理的處置汽車生產領域產生的閑置、報廢等物資，本次合作將是雙方在汽車資產處置領域布局邁出的重要一步，具有里程碑式的歷史意義。
