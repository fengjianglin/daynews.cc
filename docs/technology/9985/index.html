面向5G應用 中科創達推TurboX邊緣智能平台

2019-12-08 683條評論 4,877次閱讀 3244人點贊

智能訊 12月8日消息，中科創達舉辦Thunder World 2019邊緣智能大會，並宣布推出TurboX Edge Platform（TurboX邊緣智能平台）。

據了解，TurboX邊緣智能平台是面向邊緣計算場景的智能物聯網開發框架平台。採用層級化、模塊化的設計理念以及容器化的部署方式，基礎層包含核心計算模塊（SOM）、操作系統內核、硬體驅動和開發套件（硬體參考設計）；中間件層包含以人工智慧、邊緣計算、安全、人機交互為主的模塊化組件；應用層包含各垂直行業端到端解決方案以及雲服務。層級之間、模塊之間通過標準定義的介面進行互聯互通。

中科創達董事長趙鴻飛表示：「中科創達立足操作系統技術，關注端側智能的前沿趨勢，探索端雲結合的最優方案。隨著物物互聯與智能化需求的升級，通信技術的新一輪迭代，邊緣計算正受到前所未遇的關注」。

他還談到，2019年被稱為5G元年。5G網路的一個重要技術特徵就是多種通信技術的融合，網路切片技術正是基於5G的這一特徵。5G的切片網路保證了不同類型的業務有專有的網路資源，為邊緣計算產業的進一步發展和應用落地提供了重要的保障。邊緣計算具有的高帶寬、低時延以及位置感知等技術特性，將為我們開闢新的商業模式，充滿了豐富的想像空間。

在大會現場，我們還了解到，在中科創達第一代智能大腦平台的基礎上，TurboX Edge針對邊緣計算場景的客戶需求和技術趨勢進行了升級，實現了邊緣與雲端的數據協同、控制協同、管理協同；同時，中間件層和應用層均採用容器化部署技術，邊緣側的模塊化組件具有高內聚，低耦合的特點，更易於定製化服務以及二次開發。

具體來說，中間件層Edge Wares的模塊化組件是TurboX Edge的核心組成部分。其中，邊緣計算組件基於EdgeX Foundry，定義了組件與系統層、組件與應用層、以及組件之間的通信介面標準，採用垂直解耦，水平擴展的設計理念。為邊緣側設備提供南北向的數據通路；而人工智慧、安全、人機交互等組件，則是由中科創達的技術形成通用型組件組成，將為更多的應用場景提供模塊化、定製化服務。與此同時，中科創達也表示擁抱開源社區，與Linaro 96Board等合作，實現硬體設計的標準化、模塊化；軟體服務的開源化。

據介紹，目前，基於TurboX Edge的眾多解決方案及產品，已經應用在智能工業、智能城市、智能零售、智能網聯汽車、機器人等多個垂直行業。

此外，中科創達方面還談到，現在是一個非常寒冷的冬天，非常希望AI、AIoT、5G給大家帶來不一樣火熱的感受，未來5年、10年、20年我們將進入物聯網的時代。到2020年5G的終端會超過1個億，隨著5G和邊緣計算技術的成熟，會進一步加速AI在各個行業的落地，最終會帶來人們在生產生活方式上的巨大變革。（定西）
