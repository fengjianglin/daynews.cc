網路安全+區塊鏈，（300079）放量突發，看多為主

2019-12-11 1,797條評論 4,430次閱讀 1246人點贊

數碼科技（300079）

主營業務：軟體應用與服務、流媒體技術平台、提供雙向網路改造綜合解決方案；移動支付，互聯網及電視第三方支付平台運營，金融IC卡生產銷售；影視製作發行及經紀業務。

經營範圍：第二類增值電信業務中的信息服務業務；製作、發行動畫片、電視綜藝、專題片；經國家密碼管理機構批准的商用密碼產品的開發、生產；銷售經國家密碼管理局審批並通過指定檢測機構產品質量檢測的商用密碼產品；生產非接觸式IC卡；多媒體系統設備、通信交換設備、光電子器件、廣播電視發射及接收設備、計算機軟體及輔助設備的組裝與製造；貨物進出口、技術進出口、代理進出口；技術開發、技術推廣、技術轉讓、技術諮詢、技術服務。

公司簡介：北京數碼視訊科技股份有限公司是一家數字電視軟體及系統提供公司，公司主要從事數字電視軟硬體產品的研發、生產、銷售和技術服務業務，公司主要產品包括數字電視條件接收系統(CA系統)、數字電視前端設備等，其中數字電視前端設備包括編碼器、解碼器、復用器、調製器、加擾器、適配器等；基於公司齊全的產品線、自主研發和持續的技術創新能力，公司面向數字電視運營商提供數字電視整體解決方案。

題材看點

網路安全 在傳統DVB網路安全方面，公司的CAS/DCAS擁有最多的省級網路運營商案例、最高的國密認證等級、最大的單系統承載案例，目前隨著第一批部署CAS系統的老舊換代，CAS市場未來幾年會逐步回暖，市場趨於平穩。

互聯網金融 公司表示，互聯網金融是新業務的重中之重，P2P合作業務量增長較快，後期發展趨勢很好；目前合作方已經近400家，月資金流量15億，勢頭很好；今後在流量的基礎上會考慮開發其他的金融產品或相關的合作。

5G 2019年5月22日互動平台回復稱公司有專門與 5G 網路標準適配的產品（超高清視頻編解碼器及同軸寬頻通信晶元），但不直接參与 5G 無線網路建設。

區塊鏈 2018年年報披露：數碼科技在版權保護與新技術結合方面做了許多前沿探索,在通過對版權內容加密、加水印技術保護同時，研究基於區塊鏈的數字版權管理技術。此研究能夠保證軟體發行的授權,作者對每一個自己著作權的版權控制則能夠變得更加的嚴格。相較於傳統模式而言,基於區塊鏈的版權管理技術有及時確權、點對點通信、歸屬權明晰的優點。

綜述：

數碼科技已連續上漲5天，近期的平均成本為5.64元，股價在成本上方運行。多頭行情中，並且有加速上漲趨勢。今日該股漲停，資金流出量較少，但換手率為12.868%，交易較為活躍。市場最近連續上漲中，短期小心回調。放量突發，多頭有望再次發力，看多為主，支撐在4.47。
