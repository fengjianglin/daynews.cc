無人駕駛能否實現？淺談智能輔助駕駛技術汽車發展及趨勢

2019-12-21 245條評論 1,875次閱讀 176人點贊

說到智能輔助駕駛，可能大部分的人都有點陌生。但要是說到無人駕駛，應該大家都耳目能詳。畢竟無人駕駛一直在汽車界上都是比較熱門的話題，而無人駕駛正是屬於智能輔助駕駛系統的最高級。現在大部分的汽車都配備了自適應巡航，車道偏離警報等這些功能。而這些功能也是屬於智能輔助駕駛系統的其中一種。

簡單來說，智能輔助駕駛系統就是給汽車配備的一項新技術，旨在幫助駕駛員更加安全的駕駛。如今事故車禍頻頻發生，尤其是大貨車盲區範圍大，車身慣性大，司機長時間駕駛容易產生疲勞，這樣一來產生事故的幾率就會增加。因此就衍生了多種多樣輔助駕駛技術，例如有汽車輪胎狀態的檢測，緊急情況幫助駕駛員踩下剎車等等一系列的功能。

一，智能輔助駕駛技術分類

國際汽車協會提出把智能輔助駕駛技術分為5級，這也是如今最被大眾接受的說法。從L1級別到L5級別，從汽車對駕駛員的半干預狀態到完全不需要駕駛員控制的全自動化狀態。

L1級別，對駕駛員有著半干預狀態的輔助駕駛，即當駕駛汽車在某一動作範圍內需要輔助時，系統就會識別並進行輔助。

例如在行駛過程中在直線上需要駕駛輔助，那麼開啟自適應巡航，汽車內安裝的雷達可以檢測前車與本車之間的距離，通過判斷前車的速度以及自己與前車之間的距離，進而調節汽車本身的速度，完成無需人工操作的直線駕駛。

或者是當你需要汽車檢測你自己是否會壓線的狀態下，開啟車道偏離輔助系統，汽車兩後視鏡下方會安裝攝像頭，或者汽車前擋風玻璃正上方安裝攝像頭，通過檢測你兩輪胎之間是否壓線從而對駕駛員進行提醒警告，當駕駛員為做出反應後系統會自動修正方向盤。

L1級別就是只能控制直線速度或者左右方向之中的某一動作，並不能同時控制方向和速度。

L2級別，可以說是可以同時執行兩個動作，當輔助系統在控制汽車直線速度的同時，也可以控制汽車左右的方向盤。

例如現在很有名的自動泊車功能，以及自動變道功能，但駕駛員擁有所有權，可以隨時停止或者控制輔助系統，並且有些系統識別不出的路段，會交接給駕駛員操作。

L3級別，可以在特殊路段實現無人駕駛，第一輛L3級別的量產車車型是奧迪A8。車上有著多個雷達和攝像頭，可以探測到路上多種情況。比如在高速路段，通過雷達和攝像頭共同檢測其他汽車方位，實現自動變道，加減速和超車等功能。

但駕駛員必須全程注意路況，畢竟很多特殊路況機器無法識別，容易造成傷害。

L4級別可以達到部分路況進行無人駕駛，即在特定的路況下，可以完全不需要人工操作，不需要駕駛員握住方向盤，系統會擁有比人工更加精確的路段判斷能力。可能人會有時候把握不好制動距離，但是系統可以非常的準確的把握並且操控。

L5級別就是完完全全的無人駕駛了，無論是任何路段，任何路標指示都能識別並做出判斷。人們只需要在車上好好休息就行了。

智能輔助駕駛從L1到L5，看似一步一步的升級，但是過程中也是非常的不易，想要發展到無人駕駛這個級別，還是需要走很長一段時間。

二，5G時代的到來

用機器幫助人工甚至是代替人工，這個需要到一個眾所周知的一個東西，編程語言。生活中智能化的東西都離不開它，無論是家用的手機，工業上用的機器人，都需要用編程語言命令機器行動。

而智能化汽車也是如此，並且汽車不同於其他智能化產品，汽車結構非常複雜，想要用機器控制汽車，不僅需要控制汽車自身的行走，還需要懂得判斷各種各樣的路標，路況從而改變汽車的行駛狀態。而複雜的點在於第二種，要讓機器學會識別各式各樣的路標，各種各樣的突髮狀況。

這個是不可能的事嗎？當然不是，如今科技的蓬勃發展都是我們意想不到的，而汽車在判斷各式各樣的情況從而執行各種動作這個過程，需要大量的數據傳輸，而如今的4G網路還是不足以快速運輸如此大量的數據。

那麼，現在5G時代的到來，不敢說能實現無人駕駛，但是智能輔助駕駛技術是會取得很大的突破的。預計L3，L4級別智能輔助駕駛技術會成為發展重點。

智能網聯汽車不僅僅是汽車代替人工駕駛的功能，更加有人機之間交互的服務功能。雲服務是一個很好的體現，我們可以通過雲服務知道汽車所在位置，遠程操作汽車等。而這個功能就是更好的服務於人們。

而要更好的服務於人們，如今最流行的就是應用大數據。怎麼理解呢？大數據就是採集各種各樣用戶信息，愛好，並且精準化推薦給用戶。比如我們很多人用頭條的時候，就會發現，我們喜歡什麼，頭條就會推薦什麼給我們。這個就是大數據的應用。

再比如你在網上購物的時候，大數據會通過你平時瀏覽哪些物品比較多，瀏覽哪些和該物品相關的東西，從而給你推薦覺得你需要用到的東西。

5G互聯網的到來也會帶來更加精準的大數據推薦，總的看來，汽車的發展趨勢就是從普通的代步工具慢慢發展成服務於人們的智能化物品。

三，智能輔助駕駛的汽車可靠性

簡單來說，我們可以發現，智能網聯汽車的領頭羊不是汽車公司，而是互聯網公司。而讓一家互聯網公司去造一輛汽車，很多人都會由心的覺得不靠譜。

汽車從發明到現在，經歷了足足百餘年的歷史，一直都是屬於機械工具，包含著各種各樣的物理原理，實在複雜。雖說他們看中趨勢從互聯網轉移到汽車，但是事實也告訴了我們，互聯網公司旗下的汽車質量確實不太可靠。

真正消費於汽車的消費者們，絕大多數人最看重的還是汽車的實用性，而我們擁有百年汽車製造品牌的車企，製造出來的車子質量可靠，故障率低。而新生的車企製造汽車，品質口碑都難以超越老車企們。

當然不排除有些互聯網公司會讓某些汽車生產商品牌代工製造汽車。這種合作當然是最好的了。之前只做汽車的車企現在轉移到互聯網汽車的公司也有，而且做得也非常的不錯。代表有賓士和奧迪。

除此之外，智能網聯汽車發生的事故幾率也是偏高。我們所熟知的特斯拉，也是智能網聯汽車製造的領頭羊，卻發生了多起因系統原因導致的事故發生。這也讓我們更加的擔心智能化的可靠性。

越複雜的東西越容易發生故障，就好比以前的車為什麼說質量那麼好，現在越來越差，其實就是一個道理。

不僅在於汽車，我們的智能電腦，智能手機也偶爾會發生死機的狀態，如果讓它控制汽車，萬一有一天突然「死機」了呢？汽車在高速狀態行駛下，危險率想必也是非常高的。

除去「死機」不說，突然間信號不好而造成的系統延遲造成的危險，幾率也是非常的大。

而且目前對於智能網聯汽車的立法也沒有完全確立，比如純機器駕駛發生的事故，或者人工和機器共同駕駛發生的事故，責任怎麼歸類等等這些也沒有明確。

總而言之，未來汽車的發展趨勢也是非常明顯的，我們平時也不難發現，汽車上控制各種各樣功能的按鍵，慢慢的集中在了汽車的一塊液晶屏上面。這其實就是智能化的體現。想要完全實現L5級別無人駕駛技術，要突破的難度實在是大。如今L1級別的自動緊急剎車系統也經常出現問題，需要解決的問題太多了。即使實現了，面臨的一個明顯的問題就是，如此高科技技術的汽車，配備那麼多感測器，雷達以及電腦晶元等，價格相信不菲。價格的高昂會導致汽車難以大眾化，就像現在的智能化家居，使用的人也是少數。不僅是價格高昂，突然有個新品種進入汽車市場也需要一定時間的融入。

當然科技的發展總是讓人意向不到的，步步突破難關也是遲早的事，我認為實現無人駕駛是可以的，但只是時間的問題。而智能輔助駕駛系統我覺得也是很有必要的，運用到車上也更好的減少駕駛員駕駛汽車的勞累，減少事故發生。360全車景象，ABS防抱死系統，自適應巡航，自動緊急剎車系統等等很多功能，確實也幫助到了駕駛員更好的行車。無人駕駛是它最後的升級版，等到真正那天的到來，大概的情況應該就是手機召喚汽車過來，輸入目的地，你好好坐在汽車裡面就行了。這裡也期待智能輔助駕駛技術的發展到底會如何。
