自主新能源品牌的集體“高端化”之路是否好走？

2019-12-13 85條評論 1,030次閱讀 117人點贊

12月9日，乘聯會披露我國11月狹義乘用車市場的銷量情況：11月國內乘用車表現出季節性回暖，但新能源乘用車並沒有止跌，當月批發量為7.9萬輛，同比下降41.7%。截至目前，我國新能源汽車市場的銷量已經連續5個月下滑。

新能源補貼的退坡成為了五連降背後最大的「推手」，新能源整體市場銷量下滑，正在經歷「陣痛期」，同時也促使新能源汽車市場從政策驅動向市場驅動快速轉變，新能源汽車市場將逐漸向少數、頭部車企集中。

而在這樣的變化下，新能源汽車市場也出現了結構性的變化。一方面，從當前新能源汽車市場的銷量情況來看，結構性升級明顯，高端車型比例升高，低端車型銷量低迷。在這一大趨勢下，自主品牌也開始力求擺脫以往新能源汽車廉價的印象，開始推進品牌、產品高端化的進程。

主流品牌開始全面進軍高端化

隨著賓士、寶馬、奧迪、雷克薩斯等傳統豪華品牌發力新能源汽車，標誌著新能源這一汽車產業趨勢已不可逆轉。特斯拉的國產化也進入到了最後的階段，可以預見的是，伴隨著Model 3等車型的陸續國產，特斯拉在國內銷售有望再創新高。

面對國外對手的咄咄逼人，對國內自主品牌來說，想要提升品牌價值，擴大品牌影響力，除了與海外合資品牌聯手外，打造自己的高端化品牌是最有效的方法之一。在日趨成熟的新能源市場，低價低質的小微型車市場越來越小，消費升級成為了大趨勢，各大主機廠也開始在打造品牌高端化方面逐漸發力。

作為傳統車企來說，趨勢則更加明顯：今年3月，北汽新能源旗下高端智能新能源汽車品牌ARCFOX正式發布，並在日內瓦車展上，帶來了ARCFOX-GT賽道版、ARCFOX-GT街道版，以及一款全新的概念SUV——ARCFOX ECF Concept，開始發力高端市場。

在去年，榮威就推出了MARVEL X，用26.88萬元起的補貼後售價宣布了上汽進軍高端產品的進程；吉利汽車在去年5月發布了技術品牌iNETC，並發展出PMA純電動平台，旗下的獨立新能源品牌幾何也將採用，其首款產品幾何A也用安全、智能等賣點，號稱是「東半球最好的電動車」，對標的產品也瞄準了特斯拉。

廣汽新能源AION S、AION LX兩款車型，同樣成為了邁向高端的產品，AION LX更是喊出了全面碾壓特斯拉的口號，可見其信心之足；比亞迪全新唐EV600的出現，也早已打破了自主品牌30萬元的「天花板」，成功進入到高端新能源車領域。

而在造車新勢力方面，也誕生了不少中高端的產品：身為造車新勢力的頭部企業，蔚來的先期作品ES8甚至直接進入到了50萬元大關，從早期的回饋來看，確實吸引了不少人的關注；同為頭部企業的小鵬與威馬在今年的廣州車展上展示出的小鵬P7與威馬EX6，也開始全面進入到25萬元左右的區間。

電咖汽車同樣在早期就推出了天際汽車這個高端化品牌，儘管在開始階段遭到了一定的質疑，但是天際汽車目前正在穩步推進，預售已經開始，量產也提上日程，相信不久就能在市面上見到產品。

自主品牌正在向著高端化不斷邁進。

進軍高端化能否成功？

高端化，是中國汽車品牌一直力求突破的「天花板」，不僅是在燃油車上，新能源車也是如此。平民化是消費主流，但高端化也同樣能夠影響到平民消費。高端品牌收割低端市場，遠比低端品牌搶佔高端市場更容易。

自主品牌如果不高端化就沒有未來。現在自主新能源品牌尚處於行業底層，合資中層，進口則處於高層位置，在新能源發展初期，用低廉的價格及產品搶佔市場無可厚非。但經過幾年的市場培育，新能源汽車已經有了一定的市場基礎和消費群體，消費者對新能源汽車有了一定認知度，不再將其當做退而求其次的選擇，新能源車也逐漸從代步工具向著具有高品質的高端產品進行轉移。

縱觀已然白熱化的中低端電動車市場，目前高端電動車市場確實算得上是一片藍海。畢竟當下傳統車企及造車新勢力們早已在市場投放或即將推出相當數量的中低端電動車型，如果只做中低端電動車型，那麼短期內勢必無任何優勢可言。反之通過打造高端化來提升品牌溢價能力則更容易錯位搶市甚至獲得資本的青睞，從而為自己發展與未來獲得立足機會。

外資品牌的大舉進入，將持續給自主品牌帶來壓力，如果不能突破高端化的問題，就只能硬生生接下外資品牌砸下的大棒，絲毫沒有還手之力。自主品牌主動高端化，那麼就可以主動佔領行業的主場，主動與合資爭奪陣地，也獲得了還手的能力。而從車企的角度來說，發展高端新能源車型，代表了一個汽車企業的最高製造水平，高端化可以讓企業有更高的品牌價值和溢價能力，目的還是要實現品牌向上。

但是目前自主品牌與外資品牌最大的差距在於品牌自身，哪怕許多新能源汽車的產品力各個方面都不遜色於外資品牌，很多消費者理智上也明白，但是就是覺得開著一輛賓士、奧迪等比開著自主品牌要有面子，這種品牌溢價上的差距是自主品牌最大的劣勢和短板。

自主品牌汽車向上之路並不平坦。事實上，在2008年前後，自主品牌曾集中向高端突圍。但由於根基不穩、品牌溢價能力不足、技術儲備不充分等多種原因，包括華晨中華、奇瑞、江淮等大都鎩羽而歸。前幾年，雖然以國內幾大汽車集團為代表的自主品牌又相繼推出了中高端車型，但其市場反映也不盡如人意，以至於自主品牌一度陷入高端化「魔咒」。而在新能源汽車市場，是否能夠成功突破這股「魔咒」，尚未可知。

市場前景雖撲朔迷離，但可以肯定的是，要想屹立於行業前端，高端新能源汽車品牌必須探索一條創新、可持續發展路徑，為消費者創造更新穎、更全面、更多元的用車體驗。但高端化不是一朝一夕，也不是一款產品之功。想要獲得消費者的認可，在產品品質、服務、品牌建設等方面均不可缺少，但以往自主品牌相對來說比較缺失，如何邁過這些「坎」，才是自主新能源品牌高端化必須要考慮的問題。
