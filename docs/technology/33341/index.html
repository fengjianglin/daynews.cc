員工泄露快手春晚發10億紅包被開除；用戶評滴滴順風車：流程繁瑣

2019-12-24 1,632條評論 3,912次閱讀 937人點贊

曝快手春晚紅包將達10億元，內部員工因泄漏方案被開除

12月23日消息，據鳳凰網科技報道，快手經營分析部商業分析組徐姓員工涉嫌將快手春晚方案等詳細數據透露給媒體而被解除勞動合同，其個人期權也被追回。對此，快手方面暫無回應。

11月20日，快手宣布了與2020央視春晚達成獨家互動合作，開展今年的春晚紅包互動，但並沒有公布具體方案。目前，快手與春晚已經成立專項工作小組。

泄露的方案顯示，快手的央視春晚活動方案已經在12月中旬最終敲定，春晚當天紅包金額將達10億元人民幣。除了現金紅包，快手還將發放大量的電商代金券、實物，並與快手電筒商業務聯動。此次快手的春節活動將從元旦開始預熱，貫穿春晚頂峰至元宵晚會，時間為將近一個月。

滴滴順風車在北京、武漢開啟試運營，用戶反映：流程繁瑣

12月23日，滴滴順風車在北京、武漢等城市重新上線啟動試運營。重新上線後，車主和乘客註冊順風車均需要通過身份證和人臉識別驗證，車主註冊時平台還將以動態拍攝方式採集駕駛證、行駛證和身份證信息，防止證件造假。在行程開始前，車主還需通過人臉識別，避免賬戶被盜用。為了去「社交化」，順風車平台下線了個性化頭像、性別等個人隱私信息。

有用戶體驗後反映，首次使用流程有些繁瑣，乘客發出訂單後司機應答率不高。也有用戶和司機反映了不少問題，比如「如何撤回邀請」「順路程度不準」「誤點到達目的地」「行程前溝通不暢」等，滴滴方面表示，目前正在優化調整。

傳京東物流明年啟動上市，募資金額超80億美元，公司回應：不予置評

據騰訊科技援引外媒報道，京東物流已與銀行就潛在首次公開募股（IPO）進行了初步接觸，上市可能會在2020年下半年進行，募資金額在80億—100億美元。報道還指出，此次IPO京東物流旨在尋求至少300億美元的估值。

對於上市地點，知情人士向媒體透露可能在香港或紐約進行。對此，AI財經社聯繫到京東物流方面，後者對該消息表示「不予置評」。

天眼查資料顯示，京東物流於2017年脫離京東開始獨立運營。2018年2月，京東物流宣布完成25億美元融資，估值達到131.57億美元。在2019年Q2電話會議上，劉強東首次透露京東物流接近盈虧平衡。

中國電信回應「用戶需出天價才能攜號轉網」：按合約辦事

日前，有中國電信用戶爆料，自己於兩年前買的中國電信手機號，並辦理了299元套餐，由於單月話費過高流量低，想攜號轉網。然而中國電信員工回應稱，靚號協議有效期為20年，如堅持攜號轉網，需繳納18388.5元違約金。

此前，還黑龍江某運營商用戶稱被相關工作人員告知，相關靚號的使用期限是20年，月最低消費是399元。如果想攜號轉網，則必須把20年的消費違約金95760元都繳納齊。

中國電信方面向《證券日報》記者表示，每個用戶的使用情況不一，「解約的相關事項我公司將嚴格按照與用戶簽訂的協議條款來執行。」

韓國媒體：華為手機最遲2021年超越三星成全球第一

據韓國媒體報道稱，隨著華為手機出貨量越來越多，其在全球市場的份額也是水漲船高，最終他們可能超越三星。

韓媒指出，目前，華為手機在中國市場已經完全領先，其市場份額快要接近50%（包含榮耀品牌），而三星的份額則是始終在1%上徘徊。來自產業鏈方面的消息稱，如果華為繼續保持著50%以上的增速，那麼最遲2021年就要超越三星。

iPhone 12外形曝光：回歸iPhone 4經典設計，劉海屏仍繼續用

從iPhone產業鏈收集的最新信息顯示，蘋果將於2020年推出iPhone 12系列，該系列屏幕尺寸最大可達6.7英寸。至於屏幕前置攝像頭的設計，據悉將會繼續使用劉海屏造型。

其中，5.4英寸和6.1英寸版本搭載後置雙攝，另外一個6.1英寸和6.7英寸版本將提供三攝，從看到的新iPhone的3D模型中發現，新iPhone的設計是扁平化，這與iPhone 4的外形設計風格類似。

不少行業分析人士都對5G版iPhone表示看好，因為老用戶眾多，想要一步到位直接更換到5G手機的用戶更多，即便售價不會便宜。蘋果股價也因5G版iPhone再創新高，周一報收284.00美元。股價盤中最高至284.25美元，市值達到1.26萬億美元。

馬斯克再開炮：我的百科詞條內容太荒唐

近日，馬斯克又在推特上發文炮轟維基百科上自己的詞條頁面，他表示，這一頁面內容不準確，「幾年來第一次看我的維基百科頁面。太荒唐了。順便說一句，有人能刪除『投資人』的說法嗎，我基本上從來沒有投資。」

維基百科詞條介紹馬斯克為「科技企業家、投資人和工程師」。馬斯克稱，自己有朝一日他會記錄下有關「自己現實狀況的虛構化版本」。

「我的維基百科頁面已經成為一個戰爭地帶（即詞條內容引發爭論），進行了無數次的編輯修改，顯然還沒有完善，有一天，我可能應該寫出一個現實狀況的一個虛構版本。」馬斯克說。

消息稱特斯拉和中國的銀行達成100億元貸款協議

12月23日，據新浪科技援引國外媒體報道，有知情人士透露，中國多家銀行已同意為特斯拉上海汽車工廠提供一筆新的100億元人民幣（約合14億美元）的五年期貸款安排，包括中國建設銀行、中國農業銀行、中國工商銀行和浦發銀行。

知情人士表示，其中一部分將用於展期現有貸款，其餘部分用於發展上海工廠和中國業務。根據特斯拉向美國證券交易委員會（SEC）提交的文件顯示，中資銀行此前已向特斯拉提供了為期12個月、金額為35億元人民幣的貸款，這筆貸款將於2020年3月4日到期。

今年9月，有報道稱特斯拉上海工廠主體已建成，首輛白車身已下線。按照特斯拉的目標，上海工廠在今年年底的周產能將達到至少1000輛Model 3，主要生產特斯拉入門級車型Tesla Model 3和Tesla Model Y，所產車型將全部用於在中國銷售。

共享充電寶品牌怪獸充電完成5億元C輪融資，軟銀願景基金領投

從多個獨立信源獲悉，共享充電寶品牌怪獸充電已完成C輪融資，融資額為5億人民幣，領投方為軟銀願景基金。據接近交易的人士透露，除願景基金之外，高瓴資本、順為資本等老股東將繼續跟投。

據了解，怪獸充電在2017年4月獲得數千萬元的天使輪融資，緊接著三個月後又獲得億元以上A輪融資，然後在2017年11月完成了近2億元新一輪融資。今年4月，怪獸曾宣布完成一輪3000萬美元的融資。

共享充電寶從誕生以來就不斷被各路人士唱衰，下半年以來還出現集體漲價，最高收費甚至可達每小時8元。有投資人向36氪透露，願景基金之所以投資怪獸充電，很大程度是因為「充電寶行業已經被證明擁有較好的造血能力」。
