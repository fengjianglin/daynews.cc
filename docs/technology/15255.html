摘要：加密市場小幅震蕩，比特幣疲軟跌至7100美元附近；富達數字資產考慮在2020年支持以太坊；以太坊核心開發人員提議明年7月30日發布以太坊2.0；萊特幣基金會董事推測LTC近四個月來一直在遭受粉塵攻擊；Monero合規工作組稱FinCEN的資金旅行規則不適用於Monero；Tether為Omni新版本Omni Core 0.7.0開發提供資金支持；Coinbase已經成為XTZ的最大驗證器

【加密市場小幅震蕩，比特幣疲軟跌至7100美元附近】

本周末加密貨幣市場繼續小幅震蕩。市值最高的加密貨幣比特幣跌至7100美元附近。市場情緒目前並不樂觀，其他主要加密貨幣中，Cosmos周六日兩天均有出色表現，本周末漲幅超過10%。

【富達數字資產考慮在2020年支持以太坊】

美國金融服務巨頭富達的加密貨幣交易機構富達數字資產可能在2020年支持以太坊。富達數字資產總裁Tom Jessop在接受加密媒體TheBlock採訪時回答了關於該公司的問題。當他被問及富達數字資產將來支持以太坊的可能性時，他表示如果市場有足夠的需求，該公司可能在2020年增加對以太坊的支持。Jessop說道，「我們在以太坊上做了很多工作。我們打算在來年支持它。我們跟隨客戶的需求。」Jessop還表示，機構應用加密貨幣的主要障礙是價格波動和缺乏監管明確性，更重要的是沒有先例可供參考。不過他表示相信這些問題將隨著時間的推移而解決。

【以太坊核心開發人員提議明年7月30日發布以太坊2.0】

以太坊基金會研究員Justin Drake最近在官方Etherum GitHub發起了一場關於Etherum 2.0規範的討論，他提議2020年7月30日是以太坊區塊鏈五周年，這一天將更適合發布Etherum 2.0。他認為唯一的指標是多客戶端測試網穩定運行三個月，他希望穩定的多客戶端測試網將在明年一季度部署。因此，根據其估計，Etherum 2.0最早可能在二季度推出。

【萊特幣基金會董事推測LTC近四個月來一直在遭受粉塵攻擊】

萊特幣基金會董事Franklyn Richards近日發現，萊特幣網路每天大約有4萬個活躍地址，但是根據Bitinfocharts上的數據，這個值每7天就會升至7萬左右，然後再回到正常。這一行為始於2019年8月20日。此後，每周都有一個峰值和一個急劇下降，並且這種趨勢持續了四個月。此前，2019年8月10日發生了一次非常明顯的粉塵攻擊，他推測這可能是8月10日攻擊的延續。

【Monero合規工作組稱FinCEN的資金旅行規則不適用於Monero】

Monero表明了有關要求貨幣服務企業（MSB）追蹤用戶身份法規的立場。Monero合規工作組近期發表聲明，指出「資金旅行規則（The Funds Travel Rule）」不適用於Monero。該規則適用於受監管的實體，而非實體交易的基礎資產。」美國金融犯罪與執法網路（FinCEN）於5月發布了有關準則，該準則明確規定處理加密貨幣的公司必須保留Monero交易記錄。

【Tether為Omni新版本Omni Core 0.7.0開發提供資金支持】

Tether為新版本的BTC標記化層Omni的開發提供了資金。本周Omni公司宣布發布Omni Core 0.7.0，該開發由Tether贊助。新版本增強了網路性能，並修復了鎖定問題和遠程過程調用。Omni協議是在比特幣網路上運行的系統，該系統在網路上創建代幣。

【Coinbase已經成為XTZ的最大驗證器】

美國加密交易所Coinbase已經成為了Tezos(XTZ)的最大驗證器。驗證是當採礦者(在Pow中)或代幣持有者(在Pos中)驗證特定事務時的過程。在Tezos中，由於它是基於Pos，一個人或一個實體擁有的代幣越多，他擁有的權益就越多。
