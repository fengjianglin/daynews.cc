押注科大國盾，就是押注量子通信的未來

2019-12-05 455條評論 1,476次閱讀 439人點贊

​

港股解碼，香港財華社原創王牌專欄，金融名家齊聚。看完記得訂閱、評論、點贊哦。

高科技股的魔力究竟在哪裡？除了部分所屬產業（譬如軟體）有著較高的行業景氣度之外，還有一個原因：技術壁壘。

道理很簡單，倘若世間有一門生意，這門生意只有唯獨你有能力做到，或者只允許你來做這件事情，那麼就可以說這門生意的操盤者手握著最好的商業模式。

顯然，每一家企業都在努力能夠成為行業獨家解決方案供應商，科大國盾在其領域是否具備這樣的能力與野心，進一步分析。

科大國盾的業務為量子保密通信產品的生產，旨在為各類光纖量子保密通信網路、星地一體廣域量子保密通信地面站的建設系統提供軟硬體產品。

各類光纖量子保密通信網路涉及到骨幹網、城域網和區域網。其中，骨幹網可實現跨省、跨城的連接（包括地面光纖和衛星——地面站兩種實現方式）；城域網則負責城市範圍內不同區域、不同行業機構的連接，上聯骨幹網、下聯區域網；區域網實現的是一個單位或一處地點內多個終端的接入。

量子保密通信網路的建設可以與光纖網路基礎設施融合起來，在光纖網路的機房、站點內部署量子保密通信網路核心設備，隨著集成化程度的提升，還可以與經典通信設備進行融合。通過光纖連接（目前骨幹鏈路使用的是空閑光纖，城域網和區域網已經可以與經典通信設備復用光纖），就可以快速、平滑地形成覆蓋面較為完整的量子保密通信網路，從而實現對基礎信息網路、重要信息系統、重要工業控制系統和政務信息系統等進行有力的安全保護。

科大國盾的量子保密產品包括量子保密通信網路核心設備、量子安全應用產品、核心組件和管理與控制軟體，其中量子保密通信網路核心設備是主要的收入來源。

量子保密通信網路核心設備的產業化是科大國盾基於注入鎖定的光源及編碼技術、qkd偏振編碼調製技術、qkd自穩定強度調製技術、qkd信道波分復用技術、qkd信道自適應等技術轉化的結果。

量子保密通信網路核心設備中，最主要的收入來源為qkd產品，其具體的用途為量子密鑰的分發。

量子密鑰的分發利用的是量子物理原理，可實現遠程密鑰交換或協商並保障密鑰交換的安全性，是最先實用化的量子信息技術。

與公鑰密碼技術利用數學問題求解實現遠程密鑰交換，並利用求解的計算複雜度來保障密鑰分發的安全相比，量子密鑰分發在原理上有根本性的不同。

量子密鑰分發的主要優勢在於安全性，即無論攻擊者有怎樣的計算分析能力，乃至任意的量子計算分析能力，量子密鑰分發都是安全的：

從工作原理圖可以看出，qkd是一個通信雙方協商產生共享密鑰的過程，發送和接收裝置間通過量子信道和經過認證的經典信道相連，量子信道用於傳輸由量子態承載的量子比特信號，可以是光纖、自由空間（包括衛星鏈路）等物理媒介，經典信道則用於發送方和接收方進行基矢比對等數據後處理步驟的信息交互。

儘管量子保密通信技術相對傳統密鑰分發技術存在優勢，但是目前該技術的推廣仍然存在困難，這是幾個方面原因所造成的：

其一，量子保密通信產品與傳統密碼產品相比，在穩定性和價格方面不佔優勢。例如密碼機的平均無故障時間普遍達到3萬小時，但是目前科大國盾產品的平均無故障時間約為2萬小時。值得樂觀的是，隨著量子密鑰分發產品的晶元化發展和規模應用，成本和價格方面的差距會逐步縮小。

其二，量子保密通信產品規模化應用的時間需要待主管部門進行測評和認證才可進行。這是因為高端安全性需求群體對安全產品對資質有嚴格的要求。

其三，量子保密通信產品還在處於標準化體系建設階段，而標準化是該項技術走向產業規模化重要的一環。標準化的建設需要形成支撐大規模qkd組網、運營、應用、認證的完整標準體系，這需要時間。

這意味著量子保密通信技術應用市場行業空間的爆發需要時間，從整個信息安全行業看，密碼技術和產品是受到嚴格管制的高端領域，行業供應商有限但競爭充分，傳統密鑰分發廠商（衛士通、興唐通信、中孚信息、飛天誠信、格爾軟體）的陣地，科大國盾一時半會攻不下來。

從量子保密通信產品直接對標競爭的角度來看，該領域的產品供應商較少，除了科大國盾以外，僅有瑞士的id quantique、安徽問天量子科技公司、浙江九州量子科技公司，目前科大國盾已經是我國量子通信領域市場佔有率最高的企業，這意味著科大國盾未來更多的市場增長主要來源於量子保密通信行業整體性的成長。

整體來看，雖然暫時而言科大國盾沒辦法給股東帶來實在的回報，經營現金流凈額在2016-2018年間均為負值，賺取的利潤亦要不停地流向高比例的研發投入，但這也意味著護城河的建立將阻隔未來的競爭者，所謂前人栽樹後人乘涼，對於科大國盾的發展，不妨樂觀期待。
