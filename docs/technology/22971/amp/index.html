人民網：慶余年付費超前點播，僅僅是"不夠體貼"？

天天要聞
2019-12-19

事件回顧

騰訊視頻得罪全網:《慶余年》預點播50元 還加廣告

騰訊視頻回應點播爭議：對會員不夠體貼

愛奇藝回應點播爭議:我們初衷好 沒太做好

作者：朱昌俊

近日，《慶余年》超前點播事件引起業界和網友的熱切關注，兩家涉事視頻平台均於日前做出回應，並修改了超前點播規則。


超前點播引發的輿論爭議，似乎比《慶余年》這部電視劇本身更火爆，這可能讓各方始料未及。不過，此次事件從另一角度，也測試出了受眾對於會員規則的接受邊界，這無疑有助於相關平台提升改善自己的服務。

點播規則調整後，將此前在會員基礎上再花「50元可再多看6集」，改變為會員基礎上可單集3元進行二次付費點播，相較而言更為靈活，也降低了付費門檻。能夠針對用戶的反饋，及時調整付費方式，也確實是兌現了「探索付費模式創新」的承諾。但是，所謂二次付費點播，依然是在原有會員基礎上的「加更」，屬於會員之外再加「會員」，到底受眾的接受程度怎樣，仍待觀察。

而跳出具體事件看，此次輿論風波的爆發，恐怕很難說是意外。畢竟，在「50元加更」風波之前，有關會員權益的模糊化也早就引髮網友普遍的吐槽。

就以會員免廣告為例，即便視頻開頭的廣告可以免除，在視頻中途也可能加入廣告。近日，浙江消保委針對視頻網站付費製做的一項調查顯示，80%以上的網路視頻用戶購買會員，都是出於「免廣告」目的，但有超70%的用戶表示，購買會員後仍然會遇到廣告。

說到底，會員制，首先得把規則講清楚，用戶對二次付費加更以及購買會員後仍無法免除廣告等做法的反感，並不能簡單理解為對「付錢」的反感，更主要的是對「被套路」的反感。畢竟，既然是會員制，不管是平台還是用戶，都會默認這是一種通過付費而享有的「特殊權益」。如果會員本身被分成很多「等級」，那應該在事前予以充分的告知。在用戶繳費後再「任性」變更規則，衍生出其他模式，其實就破壞了此前用戶對自身權益的基本預期。這次加更的做法，表面看並沒有影響到一般會員的原有權益，但卻破壞了用戶最初的預期，這才是最觸犯眾怒的地方。

在此前的回應中，兩家平台都表示，對會員的告知以及消費心理的把握上還是不夠體貼。嚴格說，這不僅僅是「不夠體貼」的問題，而是規則本身欠缺對用戶足夠的尊重。如果真正把VIP會員當作「非常重要的人」，就不應該冒然「橫生枝節」，在VIP基礎上又推出VVIP。

不乏一些聲音認為，視頻網站的收費爭議，也與部分用戶還沒有形成足夠的付費習慣有關。這樣的觀點其實是似是而非。一方面，引發爭議的並不是視頻網站收費，而是如何收；另一方面，越是在用戶付費習慣仍需要培育的情況下，視頻網站越是要把付費規則做得盡量透明化，否則就可能適得其反。比如，這次就有不少網友表示寧願去尋找盜版資源。由此可以說，用戶的付費習慣不是自動生成的，它同樣需要平台的呵護。

一句話，可以理解視頻平台的盈利需要，付費模式創新也未嘗不可，但真正尊重用戶，首先先得學會敬畏規則、尊重規則。
