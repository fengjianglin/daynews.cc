愛立信承認在五國行賄，拋10億美元和美方達成和解

天天要聞
2019-12-08

瑞典電信運營設備商愛立信於12月7日承認，該公司在近17年來於多國存在行賄行為，為此已與美國檢方達成和解，交向後者交付超過10億美元的和解金。對比該公司近年財報，這筆罰金相當於愛立信單季營收的五分之一。

美方對愛立信的調查自2013年就已經開始。而這家歐洲企業也早料到會簽收高額罰單，因此在今年三季度計提準備金。鑒於本次支付罰單的行為將於12月發生，愛立信方面稱該公司四季度現金流將受到「可控影響」。


圖自美國司法部

美國司法部12月7日公布文件顯示，自2000年至2016年間，愛立信在越南等5個國家內的商業行為涉嫌違反美國《反海外腐敗法（FCPA）》。該公司通過第三方向他國政府官員行賄，並偽造賬簿，以此保持業務增長。

美國證券交易委員會和司法部分別於2013年和2015年對愛立信展開調查。另一方面，愛立信方面也於2018年5月辭退50名員工，疑似回應了美國檢方的壓力。按照美國法律，若愛立信美國檢方起訴定罪，將面臨「吊銷執照」等嚴厲制裁。

但愛立信和美國檢方最終以和解方式結束這項調查。為此，愛立信將支付9.79億美元的和解金，外加8150萬美元稅務，罰款總計10.605億美元。

值得一提，愛立信早料會收到高額罰單。

今年9月底愛立信公布今年三季度財報時，宣布計提120億瑞典克朗（約合12.3億美元）的準備金，以應對美方的調查。而當季愛立信運營凈資產環比大漲13.27億美元。


圖自愛立信Q3財報

12月7日司法部公告發布當天，愛立信官網刊登一則聲明，證實該公司將支付罰金，並稱支付行為將在本月（2019年12月）完成。這意味著愛立信今年四季度的現金流將受到影響，但該公司堅稱「影響可控」。


圖自愛立信聲明

罰款一事將會對試圖在5G領域和華為交手的愛立信造成業績影響。自今年二季度起，伴隨美方加大對華為的壓制力度，《華爾街日報》等媒體曾表示愛立信、諾基亞遇到了「撿漏5G合同」的機遇。然而三季報顯示，愛立信凈虧損達7.11億美元，網路設備業務銷售額同比增長僅為9%。

反觀華為三季報，截至2019年第三季度，華為實現銷售收入6108億人民幣，同比增長24.4%；凈利潤531億，凈利潤率8.7%。按此計算，華為第三季度同比增長率遠遠超過第二季度的12.83%，勢頭向好。
