區塊鏈人才領域供需不平衡：大浪淘沙始見金

2019-12-07 1,744條評論 2,745次閱讀 1732人點贊

免責聲明：本文旨在傳遞更多市場信息，不構成任何投資建議。文章僅代表作者觀點，不代表火星財經官方立場。

小編：記得關注哦

來源：駱駝君有話說

2019年下半年在國家政策利好的帶動下，區塊鏈的求職人數超過招聘人數的7－8倍。人才市場泡沫的不斷加劇，從側面反映出區塊鏈人才市場的盲目跟風比較嚴重。儘管求職人數不斷增多，但契合的上崗人數卻鳳毛麟角，這說明擁有行業相關知識架構和工作經驗的存量人才依舊稀缺。

區塊鏈人才市場現階段需要解決的問題是：專業人才培養和人才標準的制定。

近年來國內一些高校才開始出現區塊鏈課程，由於行業標準未確定，可能專業的區塊鏈課程體系需要3－5年的時間才會形成。一般來說，畢業的學生進入市場有1－2年的一個適應期，這就導致目前市場上專業的區塊鏈人才比較稀少。

區塊鏈真正意義上的發展也不過四年的時間（只從鏈的角度談），那麼按照高校區塊鏈課程結業的時間換算，這個區塊鏈人才市場至少還需要2年左右的時間才會趨於成熟，所以可以預見在這個過程中，區塊鏈人才市場依舊會是需求大於供給。

2019年上半年，一家專註於區塊鏈領域、名為鏈才網的招聘網站試著舉辦了一場行業人才培訓班，但效果並不明顯。歸根結底，主要原因在於區塊鏈1.0的幣圈突出的金融屬性導致人員心態浮躁，另外區塊鏈2.0和區塊鏈3.0的強需求背景下人員專業技能相對薄弱等。

一、人心浮躁

區塊鏈行業是一個名利場，行業的發展背景使然。前幾年宣揚的「李笑來事件」和「財富自由」已經深深的根植在人們心中。金融屬性的無限擴大，也讓從業人員的心態發生了變化，人們都想通過「一夜暴富」來強化區塊鏈的價值，把這種心態稱之為「病態」也一點不為過。

因為這種心態浮躁帶來的價值塑造，並不能代表區塊鏈行業本身的價值，也不能彰顯其內核技術的意義。然而，矛盾在於整個行業近八成的人都浮躁於此的時候，行業很容易被人誤解。

據鏈才網的資深獵頭（化名王小二）告訴駱駝：經常遇到這樣的一類從業人員——需求公司開出20K+的薪資，而他們依舊感覺到很少，入職之後才發現他們帶來的價值遠低於20K，這就導致從業的不穩定性！他們會經常換工作，長時間在多個公司之間周轉。眼高手低，心態浮躁，是這個行業的人才領域普遍存在的現象。

那是否就說明他們自身沒有什麼優點呢？優點肯定有，但鑒於區塊鏈浮誇的金融屬性導致的心態浮躁使他們很難找到自身的定位，只有崗位與合理的薪資水平相匹配，才是區塊鏈人才市場健康發展的趨勢。

所以在物色區塊鏈行業工作機會的時候，首先要明確職業規劃和薪資定位，這樣做的好處是:一來可以讓自己冷靜的看清楚自身的優劣勢，二是通過定位可以讓自己更方便跟需求公司洽談。

二、專業知識薄弱

區塊鏈行業不乏侃侃而談的人，似乎他們無所不精、無所不通，但當專業講師詢問關於區塊鏈技術和相關崗位知識的時候，他們往往就變得思維混亂而答非所問，據行業資深獵頭說這樣的案例每天都在上演。

在駱駝看來，專業技能薄弱的求職人員如果入職成功，那麼又無法勝任該崗位工作的時候，無疑會對需求公司的資源造成浪費，更重要的是對求職人員自身也造成了損耗。基於此，求職人員應不斷地學習專業知識並完善和優化自身行業知識體系，只有這樣才能穩定地在區塊鏈行業深紮下去。

俗話說得好：「不積跬步無以至千里。」當求職人員自身的技能、崗位、薪資三者完美匹配的時候，未來區塊鏈行業必然有你的一席之地——畢竟行業方在發展初期，只要乘風借力，一步領先則步步領先。

人是行業發展最基礎的勞動單元，即便是當下5G、大數據以及人工智慧領域大熱，但是區塊鏈未來的發展仍舊必然與人的因素密不可分。區塊鏈求職人員應放平心態，拋棄浮躁，冷靜地定位自身，讓自身找到一個合理的定位，才能積極健康的投入未來的工作中。

區塊鏈行業的歷史發展尚短，「資歷」尚輕，未來區塊鏈行業必然充滿更多的機遇。當你真正準備好擁抱它的時候，迎接這個行業一切「大浪淘金」的更新迭代，並且堅守下去，區塊鏈行業必然給你帶來驚喜。

畢竟，「念念不忘，必有迴響」。當然，此時更應該明白，「不忘初心，方得始終」。駱駝也希望咱們都能充當區塊鏈行業風口的第一批認知紅利的享有者，成為大浪淘沙後的「真金」。
