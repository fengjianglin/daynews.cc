騰訊視頻被《人民日報》批評為“涸澤而漁、焚林而獵”

2019-12-15 1,175條評論 2,084次閱讀 903人點贊

近日，《人民日報》客戶端關注到某視頻網站，「VIP之外設置VVIP，額外掏錢才能享受超前點播，是在製造焦慮誘發用戶消費」，及時響應了包括我在內的廣大觀眾的關切。

經作者核實，某視頻網站即為業界巨頭「騰訊視頻」，具體見下圖：

打開騰訊視頻

點擊「慶余年」後

VIP後還有「超前點播」，即「VVIP」

《人民日報》同時替我們的權益進行質問：「VIP「專屬廣告」的質疑已沸沸揚揚，一些視頻網站為何還敢於藐視用戶權益？為何一而再再而三地玩套路？面對輿論質疑面不改色？」

並警告某些視頻網站：「抱著涸澤而漁、焚林而獵的心態來運作，長此以往毀掉的恐怕不只是網站前途，還會殃及付費模式」。

文章最後提出建議，也是呼籲有關部門：「對待用戶缺少起碼的尊重，這樣的視頻網站不能慣。完善權益保護，讓侵權者付出代價才是治本之策」。

《人民日報》的分析鞭辟入裡，作為國內龍頭地位的騰訊視頻，這次「吃相如此難看」，惡意利用觀眾心理，刻意營造焦慮氛圍，不擇手段榨取「剩餘時間價值」（這是筆者自己提出的概念，也是能想到的最貼切的詞語，（為首次使用））。

當然，批評不是目的，希望騰訊視頻及時改正錯誤做法，為全國視頻網站樹立良好榜樣，為活躍廣大人民群眾文化生活，負起視頻媒體的責任，我們拭目以待！
