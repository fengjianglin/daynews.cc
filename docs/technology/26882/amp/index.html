12歲上大學，23歲獲博士學位，31歲成最年輕IEEE Fellow

天天要聞
2019-12-20

小德 德先生

‍

現年53歲的張亞勤從加入百度以來，一直是百度的核心人物。5年前的百度，人工智慧剛剛起步。五年後的今天，這一業務已結出百度雲、DuerOS人工智慧系統和Apollo自動駕駛平台三大果實。其中，張亞勤扮演著重要角色。

離開百度之後，張亞勤說要開啟人生的3.0，將把更多的精力投入教學、科研、聯合國慈善項目，以及與家人更多的陪伴。

事件之外，人們似乎更關心他的人生履歷。12歲成為當年中國最年輕的大學生，23歲以優異成績獲得博士學位，31歲成為美國電氣和電子工程師學會（IEEE）百年歷史上最年輕的院士，34歲執掌微軟亞洲研究院，38歲成為微軟全球副總裁……

有人說他是「神童」，但是他的人生軌跡似乎已經超越了「神童」的宿命。

一條新聞，改變了他的人生軌跡

1966年，張亞勤出生在山西太原。在他出生的那一年，文革開始，隨後一年，父親遭受迫害。記憶里，關於父親有兩個面畫：一個是父親莫名其妙地從家中不見了；另一個是5歲的時候，家中愁雲慘淡，他知道父親過世了。

張亞勤的童年，幾乎都是在晉南一個小城的外婆家裡度過的。那時候的張亞勤，已經學會克制自己的慾望，絕對不任性，努力表現自己好的一面，儘力去理解大人們的想法。

張亞勤與西山二中同學

這個毫無優越感的孩子智力超常，用了兩年時間就讀完小學，用一年半讀完初中，又用一年讀完高中，到了1978年，他高中畢業了。

更加幸運的是，在天賦和勤奮之外，社會還為張亞勤這樣的少年提供了機會。1978年3月，全國科學大會召開，科學春天的到來令國人為之一振。就在同一個月，中國科技大學成立了中國第一個大學裡的少年班。

在那個特殊的年份，國人的記憶除了高考恢復之外，最吸引人眼球的事就是中國科大少年班的「神童」，不是一個，而是一個班。當年的少年班裡有30多個學生，從12歲到15歲，來自全國各地。

1977年，在北京參加高等學校入學考試的青年正在認真答卷

13歲的寧鉑考上中國科技大學1978級少年班的故事登上了《光明日報》，成為那個時代的「第一神童」。

看完「神童」的故事之後，張亞勤激動了整整一天，然後又整晚沒有睡覺。受到寧鉑的影響，張亞勤決定也要提前高考。當時距離考試只有半年，為了補上功課，他每天學習20個小時，連走路、課間休息都在背單詞、做數學題。

高考前兩個月，一場突然降臨的肝炎讓張亞勤飽受折磨。面對即將到來的高考，張亞勤對母親說：「讓我去試試，不去考，等於考了0分。」面色蒼白的張亞勤堅持走進了考場，而這份堅持的態度從此改變了他的人生。

高考成績出爐，張亞勤以數學滿分的優異成績順利地成為了一名大學生。當時，他才剛滿12歲。帶著母親的愛和國家的希望，張亞勤走進了大學校園，開始了他本碩近8年的學習生涯。

張亞勤在中科大

30年前的中國科技大學，遠遠沒有今天這麼氣派。張亞勤和少年班的同學7個人一間宿舍，吃5分錢或者一毛錢的萊。

物質生活很艱苦，但張亞勤覺得，眼前的世界一下子大了很多。從太原到合肥，坐火車需要20多個小時，中途要倒兩次車。但是，12歲的張亞勤第一次出遠門上大學的時候，母親不僅沒有送他去學校，甚至連託運行李都是他自己去辦的。

張亞勤選擇了自己嚮往的科大少年班，選擇了自己喜歡的專業。8年時光，少年張亞勤長成了一個青年。此時的中國，一個日益寬鬆而多元的社會也在成長。張亞勤可以更加自由地選擇他的未來。

打破「神童」悖論

張亞勤後來的路順風順水，完美地書寫了標準的「神童」故事。

張亞勤的碩士畢業論文是關於通訊方面的。有一次他讀到了一篇文章，非常困惑，於是給通訊領域的學術權威皮克·霍茲寫信。

張亞勤的碩士學生證

張亞勤沒有收到回信。但兩個月後，霍茲來科大講學，他記得張亞勤的那封信，兩人相談甚歡。回到美國後，霍茲問張亞勤是否願意到美國來上學——給他獎學金。於是，一封美國喬治·華盛頓大學的邀請函發到了張亞勤手裡。

1986年，20歲的張亞勤隻身來到美國，成為皮克·霍茲的一名博士生。

剛到美國兩個星期，皮克·霍茲教授就拿了一堆文獻給張亞勤，讓他在一個月之後做一個評價。看這些文章後，張亞勤頓時就頭大了，因為很多的公式及理論國內沒有學過，而且都是最新的在通訊方面發展的技術。

1987年張亞勤在美國華盛頓

為了完成導師布置的任務，張亞勤借了60本書，查著字典將這60本書看了一遍，不僅完成了導師布置的功課，給所有文章做了一個評價，還把文章中大量的公式推導了一遍。

對於自己認定的目標，張亞勤毫不吝惜地傾注了自己全部的努力，就像當年那個決心用7個月考上大學的少年一樣，勤奮而專註。

1989年，23歲的張亞勤博士畢業，他遞交了華盛頓大學歷史上惟一一個滿分的博士論文。在老師的推薦下，他來到著名的GTE研究中心做研究工作。在接下來5年里，他全身心投入到數字視頻的傳輸和通訊研究領域，成為這個行業最為出類拔萃的專家之一。他在該領域共發表了560多篇學術論文，撰寫了11本專著，取得了60多項美國專利。

鑒於張亞勤的卓越成就，1997年，年僅31歲的張亞勤成為美國電氣電子工程協會院士（IEEE Fellow）。這是電氣和電子學研究領域全世界最高的學術榮譽，他成了這個協會110年來獲此榮譽的最年輕的科學家。

柯林頓在任美國總統時曾給他寫信，說他「是一個靈感的啟示」。美國IEEE院士瑞曼德和張亞勤的博士生導師皮克·霍茲則稱他為「全世界的財富」。

對比同時期的寧鉑、謝彥波，張亞勤可以說是打破了「神童」悖論。2005年，一直承受著巨大壓力的寧鉑遁入空門，出家為僧；與導師不睦的謝彥波，被中科大從普林斯頓召回，以碩士的身份接受了近代物理系教師的工作。

當寧鉑在「神童」的光環里糾結、謝彥波在與導師的緊張關係里找不到定位時，張亞勤成為了「神童」背後真正的贏家。

放棄國外待遇回國，開啟人生新征程

1998年的10月份的某一天，張亞勤忽然接到了李開復的一個電話，說微軟要在中國建立一個基礎研究院，希望他能一起回來創業。

此時的張亞勤，是美國四大科學研究中心之一的桑納福多媒體研究室的總監，這個職位是華人在美國多媒體研究領域做到的最高職位。

放棄優越的科研條件，回國籌建一個前途未卜的研究院，這看起來簡直是不可思議的事情。張亞勤思考了5分鐘，就決定成為這個研發中心的一員，任首席科學家。當時他的妻子已在道瓊斯公司工作，在張亞勤的說服下，他們把美國的房子、車子全部賣掉，舉家遷回中國。

如果1998年，剛剛加盟微軟的李開復沒有給張亞勤打電話，那麼張亞勤的人生會不會是另外一條完全不同的軌跡？這個問題沒有答案，但是可以肯定的是，當年的這個選擇無疑成為張亞勤人生中的第二個轉折點。

按照張亞勤溫和、中庸和略顯被動的性格，如果沒有外部因素，他應該會留在美國繼續搞科研。但是天才總要長大，正是這一轉折，讓張亞勤漸漸擺脫了「神童」的光環，並對其自身價值甚至思考方式完成了徹底的重塑和再造。

加盟微軟中國研究院一年多後，李開復升任微軟全球副總裁，被調回總部主持自然界面部門的研發工作，張亞勤由此走上前台，成為微軟中國研究院第二任院長。

張亞勤與比爾·蓋茨

從首席科學家到院長的角色轉換過程並未遇到阻礙，張亞勤很快就進入了狀態。在他上任後不久，微軟中國研究院更名為微軟亞洲研究院（MSRA），兩個字的差別，氣魄立變。

也就是從張亞勤時期起，中國真正進入了微軟的戰略版圖核心位置，也為MSRA在國際學術領域佔得一席之地奠定了堅實基礎。微軟的金字招牌和張亞勤的個人號召力，使得MSRA在當時中國的各大高校中成為最具吸引力的技術聖殿。

2004年1月，張亞勤被提升為微軟全球副總裁，調回總部負責微軟移動通信及嵌入式系統在全球的開發業務。

沒過多久，張亞勤在總部的平靜生活節奏被李開復的跳槽事件打亂。在蓋茨眼裡，只有張亞勤是最合適的救火隊員。2005年9月，張亞勤被火速調回中國出任微軟大中華副董事長，並開啟了微軟中國研發集團的籌備工作。5個月之後，微軟中國研發集團成立，張亞勤毫無意外地擔任了主席。

張亞勤全家福

自2006年初微軟中國研發集團成立之後，張亞勤一直都在為整合併使微軟在華研發資源最大化而努力著。

在微軟的16年，張亞勤像是一塊磁石，持續吸納了本地和海外的行業精英，培養了一批批人才，微軟在北京中關村的研發團隊儼然成為中國IT產業的黃埔軍校，包括沈向洋、張宏江、林斌、張益肇、李世鵬等人全都彙集於此。

2014年9月3日，在百度世界大會上，張亞勤意外出現在了現場，並和李彥宏等百度高管坐在一起，引起外界猜測。7天後，百度對外宣布：張亞勤加盟百度，擔任總裁，負責新興業務。

對此，張亞勤解釋說：我用16年時間兌現了加入微軟時的承諾……現在，使命達成，是該走的時候了。

談到張亞勤對於百度的貢獻，不得不提的就是其在2016年提出的ABC三位一體概念，即AI（人工智慧）、BIG Data（大數據）和 Cloud Computing（雲計算）三位一體。此外，被寫入政府工作報告的智能+，最早也是由張亞勤提出。

如今，張亞勤在百度負責的業務已經涵蓋雲計算、自動駕駛、人工智慧商業化、教育、安全、國際業務、技術體系、量子計算等多個領域，在其退休後，究竟誰能接任，百度又將走向何處等諸多問題都成為了大眾關注的焦點。
