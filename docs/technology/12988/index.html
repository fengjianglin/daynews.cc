Chainlink近期多場黑客松獲獎項目介紹

2019-12-12 1,343條評論 2,410次閱讀 678人點贊

Chainlink在過去一個月中馬不停蹄地參與舉辦了三場黑客松，其中獲獎項目更是種類繁多，展示了Chainlink安全連接鏈下資源的能力。我們不僅舉辦了首屆Chainlink線上黑客松，還參與了ETHWaterloo黑客松以及Kyber DeFi線上黑客松，並為優勝項目頒發了獎勵。我們非常欣喜地看到參賽項目使用具有外部連接性的新型智能合約開發出了各種具有實際商業價值的應用。

Chainlink線上黑客松

我們在十月份舉辦了首屆Chainlink線上黑客松，並嘉獎了最具實際商業價值的應用，這些應用能將智能合約連接至外部數據、事件和系統。我們頒發了一二三等獎，以及六個評委選擇獎和一個最具創新力獎，優勝項目獲得了Chainlink的獎金。我們與導師團隊一同舉辦了一系列開發者研討會，並藉此機會與智能合約開發者社區加強交流，幫助他們更好地使用Chainlink預言機開發智能合約。

第一名

LinkPal項目拔得頭籌，項目開發者是Vitaly Volozhinov。LinkPal是一個去中心化的託管應用，使用智能合約在ETH和PayPal賬戶之間進行P2P交易，完全規避了對手方風險，也無須通過第三方交易所執行。賣方將以太幣凍結在智能合約中，直到確認買方的Paypal支付信息後才放款，這樣就保證了交易無須信任即可展開。一旦多個Chainlink預言機確認支付信息正確無誤，被凍結的以太幣就會打到買方之前提供的公鑰地址中。LinkPal非常好地展示了如何使用多個Chainlink預言機安全可靠地實現去中心化，並解決鏈下遺留基礎架構中的實際問題。

(LinkPal APP界面，顯示完成交易所需的五個步驟）

第二名

Cerebus Wallet項目獲得了第二名，項目開發者是Mikhail Lazarev。Cerebus Wallet為在以太坊上交易的用戶提供了額外的安全層，通過手機驗證碼實現雙因子驗證。用戶每次用私鑰簽名驗證交易時都會在手機上收到一個2FA碼，必須輸入這個代碼才能完成交易。這個機制可以防止賬戶遭到黑客惡意攻擊，即使是私鑰被盜也可以保障資金安全。Cerebus Wallet接入Chainlink預言機，並接入計時器API驗證用戶是否在一定時間內輸入了2FA碼，同時還接入了Cerebus伺服器確認輸入的2FA碼是否正確。

（用Cerebus Wallet交易的基礎架構）

第三名

獲得第三名的項目是Flyt，項目開發者是Robin Thomas。Flyt是一個去中心化的航班保險應用，用戶可以以無須信任的方式獲得航班保險，並充分保障個人隱私。Flyt繞過了保險公司定價和理賠的環節，用戶可以使用智能合約和Chainlink預言機獲得的web數據（即API）直接計算保費，保費計算也可基於航班和機場評級設置權重。另外，還可以根據航班狀態在24小時內自動理賠。

最具創新力獎

Total Return Swap獲得了最具創新力獎，這支團隊一共五人，均來自Chainlink官方節點Data Links。儘管Total Return Swap的staking功能目前尚未最終完成或在主網上線，但這個項目的設計理念讓人眼前一亮，它讓Chainlink節點運營者可以對沖在服務中抵押的LINK價格波動風險。節點如希望減少LINK價格波動風險，則可以將風險集合打包成資產賣給對手方，合成資產價值與LINK通證價格掛鉤。節點可收取通證貸款利息，而交易對手方則承擔LINK價格上行或下行的風險。

評委選擇獎

Smart Marketing Campaigns項目使用智能合約和預言機自動付款給市場營銷機構，Chainlink預言機可以獲取網站上某類特定用戶的訪問人數，並進行驗證。市場營銷機構經常會在營銷活動投資回報率上與客戶產生分歧，因為很難獲得準確的數據衡量營銷活動的有效性。因此，Smart Marketing Campaigns接入Chainlink預言機輕鬆收集各種可驗證的真實數據（比如新訪客或特定類型訪客人數），並基於這些數據執行服務協議，無須信任。

Steam Trader接入Chainlink預言機和Steam API，為Steam遊戲平台提供安全、去中心化的支付方案。Steam Trader是一個非常有趣且實用的項目，充分顯示了Chainlink技術可以用於任何生活場景中解決人們的日常問題，比如無須信任的所有權交易。

Contractor應用可以根據用戶選擇自行生成完整的智能合約。用戶可選的功能包括智能合約種類、預言機類型和數量、API介面、數據聚集方式等。Contractor讓技術小白也可以輕鬆部署智能合約，並實現各種功能。

（Contractor在以太坊網路中創建智能合約的UI）

Random World是一個賭博遊戲應用平台，所有應用代碼都是開源的，而且保障結果公平公開。平台上的賭博遊戲包括猜硬幣正反面和擲色子，內核是智能合約以及隨機數生成器。平台接入Chainlink外部適配器，連接到Random.org網站，因此在賭博遊戲中可以從外部獲取隨機數，並保證結果無法被篡改。

Fiat Gateway在後端使用智能合約，並接入安全、中心化的Chainlink預言機，進行法幣和虛擬貨幣託管和交易確認，用戶可輕鬆快速兌換虛擬貨幣和法幣。「Makers」為「Takers」（即賣方/買方）提供法幣或虛擬貨幣流動性，並收取一定傭金，預言機確認交易雙方在WePay和PayPal等平台上成功完成法幣支付後才會觸發鏈上交易。

The Anti-Social Challenge激勵用戶減少刷社交媒體APP的時間，並綁定自身利益。用戶需要在智能合約中抵押資金，只有在規定時間內不點開社交媒體APP才能拿回這筆錢。在這段時間內用戶只要一刷社交媒體就會被探測到，並且會通過Chainlink預言機上報智能合約。然後用戶會損失抵押的金額，這筆錢會平均分配給其他成功完成挑戰的用戶。

ETH Waterloo

我們在11月8日到10日去加拿大參加了ETHWaterloo，開發者社區倡導者Dan Forbes舉辦了一場Chainlink開發者研討會，討論如何使用鏈下資源開發具有實際價值的應用。我們還參加了一場黑客松活動，並嘉獎了三個使用Chainlink預言機開發智能合約的優秀項目。

第一名

DeFi-dy獲得第一名殊榮。項目團隊一共四人，成員包括Jean-Baptiste Terrazzoni、Kendrick Tan、Sneh Koul以及Madhur Kumar Sharma。DeFi-dy的策略並不是幫助投資者對沖通證價格波動風險，而是讓用戶主動出擊投資公司的業績。項目打包發行合成資產，資產價值基於具體協議。協議的基本價值根據加權演算法計算，包括受歡迎程度、使用情況、抵押率、借款權、貸款權等。Chainlink預言機從鏈下獲取這些數據，並將數據發送至智能合約，計算合成資產價格。

第二名

CarbonGuilt項目獲得了第二名。這支團隊一共四人，成員包括Jeremías Díaz、Vijay Kandy、Victor Vicente以及Bilal Alghazi。以太坊等POS區塊鏈的共識演算法對環境資源造成了較大負擔，因此CarbonGuilt讓以太坊用戶可以選擇消除其鏈上交易產生的碳足跡。CarbonGuilt接入Chainlink預言機從一系列能源消耗和碳排放數據來源收集數據，並將數據發送至鏈上。接著，智能合約就會用這些數據計算碳排放量，用戶可以選擇向相關可信的環保組織捐款以消除自己的碳足跡。

第三名

WeatherBet獲得了第三名，這支團隊共三人，成員包括 Christina Chen、Christian Agnew和Jake Munsterberg。WeatherBet讓用戶可以用以太幣競猜未來具體某一天某個地點的天氣狀況。項目接入Chainlink預言機從可信API收集準確天氣數據，並聚合數據確定競猜結果。

Kyber DeFi線上黑客松

我們從9月9日到10月28日以技術合作夥伴的身份參加了Kyber DeFi線上黑客松，並為一個接入Chainlink和Kyber的優秀Dapp項目頒發了獎勵。

Chainlink x Kyber獎

EthOptions獲得了Chainlink頒發的獎勵。團隊共四人，分別是Parinaa Anand、Nikhil Yadav、Siddharth Bhalla以及Mohsin Hussain。期權合約是金融系統中的關鍵工具，讓用戶能夠對沖資產價格波動風險。用戶可以按溢價購買期權合約，並有權在未來具體時間點購買合約規定的具體資產。EthOptions是一個去中心化的期權交易平台，全世界各地的用戶都可以在平台上買賣期權合約，將交易門檻大幅降低。項目接入Chainlink預言機，通過CoinMarketCap外部適配器將期權合約與可靠市場價格連通。合約基於外部市場價格決定到期支付金額。

不斷壯大

我們非常欣喜有如此多獨一無二的智能合約項目通過Chainlink預言機連通鏈下資源。未來會有無限可能，而現在只是一個開始。之後會有更多數據提供商、節點運營者、項目和外部系統加入Chainlink生態系統中，並為智能合約提供海量真實可靠的數據。我們期望以後能夠繼續參加各種黑客松活動，不斷培養開發者社區，並將Chainlink生態系統越做越大。
