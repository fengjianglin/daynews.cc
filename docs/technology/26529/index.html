今年斷網93次，“世界斷網之都”印度5G之夢或至少推遲5年

2019-12-22 1,019條評論 3,942次閱讀 1799人點贊

2019年是互聯網問世50周年，也是5G年，不少國家都正緊鑼密鼓地進行5G建設中，特別是東南亞國家。而曾被譽為全球增速最快的經濟體——印度，卻在網路建設上頻頻「摔跟頭」。據統計，今年以來印度實施斷網93次，而這對該國推進5G建設是一大阻礙。與此同時，泰國、新加坡、馬來西亞等國在果斷地選擇華為後，5G建設都有了實質性的進展。

據媒體最新報道，此前8月，印度為國內突然爆發的「緊急活動」而實施局部斷網，截至12月19日，印度已經斷網137天，創下了該國最長的斷網紀錄。根據《今日印度》數據分析部門發布的一份報告稱，自2014年以來，印度共有357次斷網的紀錄。

報告顯示，印度在2014年經歷了六次斷網，到2015年上升到14次。2016年達到31次，2017年達到79次。這一數字在2018年達到峰值，達到134次，截止2019年12月15日，互聯網服務關閉共計93次。而印度時不時就斷網的慣例，對5G運營商在印度推進5G網路部署而言也是個不小的「隱患」。

除此之外，根據印度移動運營商協會表示，由於5G頻譜基礎價格過高，印度5G網路部署至少推遲5年。據了解，在韓國、西班牙、英國和義大利等國家，5G頻段的平均拍賣價格約為每兆赫8.4億印度盧比；而在印度每兆赫的價格是49億印度盧比，購買100兆赫的頻譜，運營商至少需要支付5000億印度盧比。

更關鍵的是，該國多數電信公司都「沒錢」參與5G頻譜拍賣。目前印度電信行業已累計背負著7.5萬億印度盧比的債務，在最近一個季度中，印度沃達豐和印度電信公司虧損金額雙雙創下歷史新高，分別虧損5090億印度盧比和230.45億印度盧比。

按照計劃，印度將在2020年3月前舉行5G頻譜拍賣，並在明年年底前推出5G網路。不過有分析人士稱，印度電信業面臨的資金危機可能會影響到印度推出5G，而愛立信則表示，5G服務預計到2022年才能在印度推出。

值得一提的是，此前印度在是否使用華為5G一事上多次舉棋不定，而為了打消印度最後的顧慮，華為一直大方地向印度伸出橄欖枝，提出願意向印度公開源代碼，甚至是簽署無後門協議。不過，印度多次鬆口後又再次猶豫，這也讓印度難以邁入5G時代。

此外，印度上周通過涉嫌排斥穆斯林的「公民身份修正法案」，引發該國連日的示威和騷亂，已導致至少6人死亡。為防止事態惡化，印度不得不關閉部分地區的互聯網服務。政府主動斷網已成為印度面對緊張局勢的慣常操作。據《今日印度》12月18日公布的統計報告，顯示2014年以來印度政府關閉網路已達357次。

根據數據權益組織「Access Now」的統計，2018年全球有67%的互聯網關閉發生在印度，今年1至7月，全球共關閉網路120次，印度佔80次，佔比同樣達到67%。

如此高頻率的關閉互聯網，讓印媒也自嘲印度已成為「世界斷網之都」。BBC指出，莫迪在2014年當選前提出「數字印度」倡議，旨在加強該國數字基礎建設。他曾表示希望逾10億印度人都能上網，並將廉價高速寬頻網路在農村普及，而如今印度政府的操作卻引發外界質疑：是否這就是莫迪「數字基礎建設」真正的意思？
