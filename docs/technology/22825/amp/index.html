前"鬥魚一姐"馮提莫宣布轉戰B站，12月23日起直播

天天要聞
2019-12-20

還記得鬥魚直播的四大花旦嗎？有的女主播沉寂了，有的被封殺了，堅持到最後、成為鬥魚一姐的馮提莫也在今年9月底跟鬥魚直播結束合約。在今天的生日音樂會之後，馮提莫宣布獨家簽約B站，12月23日晚上8點首播。

據網路資料，馮提莫（本名馮亞男），1991年12月19日出生於重慶市萬州區，中國內地女歌手、網路主播。

2014年9月，馮提莫開始了其網路主播的生涯，並正式成為職業女主播。2016年9月，翻唱歌曲《別找我麻煩》；12月，為電影《28歲未成年》演唱片頭曲《你不懂我》。2017年6月，發行個人首支單曲《識食物者為俊傑》；12月，為電影《前任3：再見前任》演唱同名宣傳曲《再見前任》；12月21日，獲得第六屆搜狐時尚盛典年度最火主播獎。2018年1月，獲得鬥魚年度盛典十大巔峰主播獎；2月，發行個人單曲《佛系少女》。2018年9月30日，發行單曲《青空》。

2019年1月，獲得第一屆Q China 2018年度音樂盛典最佳社交媒體音樂人獎；5月，發布個人首張翻唱專輯《莫須有》；8月，舉辦首場個人演唱會，並發布首張同名原唱實體專輯《馮提莫》。2019年9月30日和鬥魚的直播合約到期。


馮提莫簽約B站讓人意外，但是聯繫到B站最近的動向，其實也很合理，因為B站正在大力加強直播，本月初才跟全球最受歡迎遊戲《英雄聯盟》達成為期三年的國內獨家直播協議。將獲得在2020年至2022年間未來三屆英雄聯盟世界冠軍比賽在中國的獨家轉播權，合同總價值為8億人民幣。

除了遊戲，像馮提莫這樣的頂級流量女主播顯然也是B站拉攏的對象，這次獨家簽約馮提莫，B站付出的簽約費顯然會是個天價，不過雙方並沒有公布具體的金額。
