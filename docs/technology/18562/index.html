外賣巨頭“燒錢”提升市場規模，居然是為了現在的“收割”

2019-12-18 155條評論 1,350次閱讀 136人點贊

自今年以來，美團與餓了么大外賣平台因不正當競爭「行為」被各區域代理商+商家曝光，導火索因外賣抽成比例從最初的15%逐漸上漲至26%，雖然外賣平台的傭金一直都處於上漲狀態，但今年開始上漲的如此之多，不得不讓人深究其原因。

外賣平台最初為了贏得大量市場份額，同補貼燒錢的方式吸引更多商戶入駐到其平台中。隨著市場逐漸趨於飽和，各大外賣平台開始通過提高傭金的方式回收成本。最後成本回收的結果，是由我們商家和消費者共同買單。然而，傭金的提高定將導致一系列連鎖反應的出現，擾亂市場秩序，影響社會健康發展。

外賣傭金的不斷提高，商戶開始用多種方式「消化」成本，其連鎖反應就會像食物鏈一般，會把最大的危害聚集到食物鏈的末端，也就是最終的受害人——消費者。一些商戶為了保證利益，把商品價格提高，將成本向消費者轉嫁;食物質量「打折」，在食品上偷工減料，來降低成本。無論哪一種做法，都會將我們的消費者推向風口，使其成為最終的受害者。

一切規則平台說了算，餐飲商家抱怨重重，規則不透明，市場經理還要給「穿小鞋」。

平台對餐飲商家的扣點，也並沒有明確標準或規則，規定什麼樣的餐廳扣點多少，十分「靈活」。而具體是多少，則掌握在市場經理手中，餐廳通常需要與市場經理口頭協商，沒有文字協議。

規則不透明，直接導致不公平，甚至可能出現「權錢交易」。跟區域市場經理關係好，就能拿到優惠，不搞關係，就要付出比別人多的成本。有很多商家表示：「很多時候，我們都敢怒不敢言，生怕市場經理給我們小鞋穿。」

如果說扣點是餐飲商家使用了平台的服務，必須承擔的成本，那麼在曝光與流量之爭下的「付費推廣」，則讓商家更被動。

數據顯示目前餐廳70%的流量來源於排名，只有30%來源於其他活動入口。然而入駐的商家越來越多，平台排名的好位置是有限的，「僧多肉少」，於是競價推廣、關鍵詞推廣等應運而生。

儘管某外賣平台曾公開回應媒體，「付費推廣與競價排名不同，將儘可能保證排名合理公平」。但一句口頭上的「儘可能保證」，能有多大效力？

曾經就有各地餐飲企業向媒體爆料，某平台收取商家在線支付營業額的5%作為技術服務費，換取餐廳在平台排名星級的提升。如果不加入計劃，就會被該平台市場經理威脅下線關閉其店鋪。

雖然事後該平台公關部門否認「強制收費」的說法，但也暴露了在付費推廣和競價排名面前，餐飲企業的被動地位。

上面的情況僅僅是部分區域的商家的現狀，交錢給平台，正常合理運營也怕市場經理「穿小鞋」，面對被壓榨被清退的風險也不敢有任何「反抗」行為。

而這些行為帶給整個行業的負面影響也是巨大的，在線服務對比傳統服務無法體現出更多價值，讓用戶體驗不到更多的便利與附加值，就算紙面上的規模擴大了，也終不過是含了很多水分，很難禁得起推敲的一個花架子而已。而導致的結果，最終只會像狗熊掰棒子一樣，掰一個掉一個。

未來的外賣平台將何去何從，需要有更多外賣平台可以突破自己的服務，運營效率來提升自己的競爭。或者選擇自主搭建平台的形式來運營，用外賣系統搭建的平台，有多個下單入口，無論是在小程序、微信還是APP、PC等多個入口都可以下單，方便快捷。在平台實際運營中，運營者可以不斷進行業務拓展，為B端用戶提供多元化的生活服務。

相信未來的外賣市場大家越來越能接受新零售形式的運作，大家對於即時的需求會越來越旺盛，也會滲透到更多的領域當中去。遠離了「潛規則」的環境，未來的市場必將會發展得更加廣闊。
