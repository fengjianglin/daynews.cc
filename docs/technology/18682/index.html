花鄉市場探店調查：納智捷、電動車二手車商都不收了

2019-12-17 1,294條評論 3,375次閱讀 1750人點贊

年關將至買啥穩賺不賠？當然茅台啊，這可能是最近在辦公室討論最多的話題。還有什麼穩賺不賠？豐田大霸道、9代EVO，尤其是三菱LANCER 9代EVO，絕對是坐等升值的最佳理財工具。

當然有靠著情懷升值保值的，就有著急出手的「燙手山芋」。近期，根據中國汽車流通協會發布的《2019年11月中國汽車保值率報告》顯示，三年車齡的新能源純電動車平均保值率為36.4%，遠低於傳統燃油車。而在新能源車保值率排行榜上，只有特斯拉Model X車型（三年車齡）的保值率達到63.9%；大量國產品牌新能源車（三年車齡）的保值率均低於50%。

此外，根據中國汽車金融暨保值率研究委員會發布的《2019中國汽車保值率報告》，從新能源汽車的1年保值率情況來看，排在前五位的是特斯拉Model X、特斯拉Model S、別克VELITE 5、沃爾沃S60L和比亞迪宋DM，保值率分別為74.98％、71.42％、60.67％、58.43％、58.05％，但這些數值可能過於樂觀。

花鄉市場E探究竟

花鄉二手車市場作為北京最大的二手車交易市場，這裡不僅有成排大G、稀有豪車，更有眾多二手商家在追求利潤，自然是調研純電動二手車情況的最佳場所。

往常但凡開掛牌燃油車，不管什麼車型，不管是否賣車，總會有三五成群車販子趴窗詢問，哪怕是即將報廢的舊車，在他們眼中也是有利可求，但也有會被冷落的車型–納智捷或電動車。

12月的北京寒風刺骨，若想在佔地500畝的花鄉二手車市場中，找到一台懸掛綠牌的新能源車絕非易事，最好事先全身備好暖寶寶，實際情況不僅凍人可能更寒心。但凡在屋內有展廳的絕對是有實力的二手車商，蠻牛、烈馬、大G，就是找不到一台綠牌車。E車匯在探訪期間，大廳內唯一一家有售新能源車型的商家表示：「是有一台特斯拉 Model X 100D，但店內不讓放，一是不能充電二是怕著火。」

同樣是百萬豪車，特斯拉 Model X 100D只能在戶外感受寒風凜冽，同樣凜冽的是價格。這是一台今年6月的新車，整體並無任何明顯傷痕。當時基礎版本不選裝的前提下，售價就百萬的豪車，時隔6個月僅售79.8萬元，身價跌幅接近25%左右。

當然特斯拉還是好的，畢竟高端車型商家與客戶較為認可，普通家用車就更顯尷尬。例如這台三萬元左右的鎮店之寶–江淮iEV5。即了解走訪，花鄉二手車市場中，經營純電動二手車的商家寥寥無幾，主要原因在於商家手中沒有新能源指標、無法檢測、無法評估、難以脫手。

商家毫無殘值標準

如果你有一台純電動車準備出手，在二手車商眼中絕不會被視為座上賓，甚至少有理睬。究其原因，就是「怕砸手裡」。

在「E車匯」走訪期間，二手商家基本會將純電動二手車劃分為高低兩類車型。首先低端車型主要以江淮新能源為主，作為一個沒有廠家回購政策的品牌，江淮新能源車主在換車之際，只能選擇置換或出售。

專收二手江淮純電動的商家表示，自己手中有設備可以檢測車輛充放電狀態、電池衰減等儀器，但攀談期間反覆詢問車輛使用情況。在未看到「實車」的前提下，已經為「E車匯」口中的「該車」，定價為一萬元左右，而且是一副過了這村沒這店愛答不理的表情。

當然商家也不忘反覆對「E車匯」洗腦：「你的車快到年限了，不賣車換電池更貴，留著續航也不夠用，現在還有點殘值，以後更沒有，一萬我是為你好。」目前市場行情的確如此，二手商家會儘可能壓低收車價格，將風險全部轉移至車主，江淮iEV4、iEV5這類車型，一萬左右價格收、三萬左右價格賣，基本上已是常態。

高端車型基本上也只是以騰勢為主，為何是騰勢？主要是騰勢品牌8年或15萬公里的電池組質保周期，以此為賣點可讓客戶安心購買。即使如此，這台當年售價32.8萬的騰勢500，18年12月上牌僅行駛1.19萬公里，店家出售價格僅為18.98萬元，收車價格基本在16萬左右，跌幅接近50%恰好與騰勢官方回收價格相當。

對於國內新能源市場而言，新車銷量五連降、二手無人要，這樣的情況讓眾多消費者購車顧慮重重。儘管有眾多新能源品牌推出回購政策，但純電動車至今並未推出統一檢測標準。衡量殘值的只有眼觀、手摸、意評，對於純電動車主來說並不公平。

當然，「E車匯」在走訪期間也詢問了部分有回購政策的新能源經銷商，回收本品牌二手車在無結構損壞的前提下，基本可以遵循當時公布的回購政策，若有結構性損傷的車型，則需單獨評估殘值，但大多僅限於換購同品牌新車。

二手車評估標準的缺失，讓消費者在賣出新能源車方面受到了一定的阻力，也就導致了新能源二手車和納智捷一樣，成了車商都不願碰的「燙手山芋」，而想要扭轉這個局面，恐怕還需要很長的一段時間來進行完善。
