國家市場監管總局局長肖亞慶調研騰訊、華為

2019-12-09 20條評論 1,338次閱讀 176人點贊

大河財立方消息據國家市場監管總局官網12月9日消息，近日，市場監管總局局長、黨組書記肖亞慶在深圳市調研，強調要認真學習貫徹習近平新時代中國特色社會主義思想和黨的十九屆四中全會精神，堅持以人民為中心的發展思想，加強網路市場監管，提高網路市場監管效能，引導行業持續健康發展。

肖亞慶前往深圳市環球易購電子商務有限公司、騰訊計算機系統有限公司和華為集團開展調研，觀看了三家企業的重點和特色項目展示，深入了解電商網路平台運作、互聯網消費以及大數據、人工智慧和智能製造等前沿科技發展情況，了解企業發展中遇到的問題，聽取各企業對市場監管工作的建議。

肖亞慶指出，互聯網等新興行業蓬勃發展為經濟建設注入了新動力，也對構建科學高效的市場監管體系提出了新挑戰。網路市場發展關係著人民的獲得感、幸福感和安全感，做好網上監管、維護良好的網路市場秩序，需要持續推動市場監管改革創新，加強監管。

肖亞慶要求，要督促企業嚴格落實主體責任，嚴懲典型違法行為，維護消費者合法權益，要健全相關法律法規體系，推動基礎理論研究，做好知識產權保護，推進商事制度改革，努力營造公平競爭的市場環境。

肖亞慶在深圳市市場監管局調研時，參觀了智能指揮中心，了解「智慧監管」平台建設、商事登記、明廚亮灶、食品藥品風險防控與監管執法等工作。肖亞慶強調，深圳是改革開放前沿，充滿創新與活力。深圳市市場監管局要認真學習領會習近平總書記關於市場監管的重要論述和指示批示精神，以學習貫徹四中全會精神為契機，進一步解放思想，加快推動市場監管工作改革創新，特別是要建設好「智慧監管」平台，嚴守安全底線，防範重大風險；要優化營商環境，不斷激發市場主體活力；要加強幹部隊伍建設，促進全面融合的「化學反應」。

在深圳期間，肖亞慶出席了第三屆中國質量（深圳）大會並主持開幕式；與廣東省及深圳市有關領導就深化市場監管改革工作交換了意見。市場監管總局有關司局負責同志參加調研。
