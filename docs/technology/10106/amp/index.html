5G來了，4G手機會降價嗎？

5G來了，4G手機還會繼續使用很長一段時間，4G手機的需求還會很大，這個不影響4G手機的價格。目前只有一二線城市有5G進行商用階段，但是在這些商用城市裡面，5G基站開通的也不多，基本上也就只有一些商場，交通樞紐等地方。明年4G手機產量會減少，5G手機種類越來越多，尤其是中高端機型今年5G正式商用，同期也有很多5G手機上市，例如華為、vivo等品牌的手機。
