CoRL2019無人駕駛研究成果大總結（含大量論文及項目數據）

2019-12-21 782條評論 1,906次閱讀 533人點贊

機器人學習會議(Conferenceon Robot Learning，CoRL)是一個相對較新的會議，成立於2017年。機器人技術，自主感知和控制領域正在經歷一場機器學習革命，現在正是時候提供一個將機器學習的基本進展與機器人應用和理論的實證研究相結合的場所。舉辦方表示，他們目標是使CoRL成為機器人（機器）學習研究的首選大會。

CoRL簡介


第一次會議將於2017年11月13日至15日在加利福尼亞州山景城舉行，旨在匯聚約250名從事機器人學和機器學習領域的最優秀的研究者參會。經過幾年來的發展壯大，不難發現，CoRL所研究的為人工智慧領域中的細分創新點——機器人與機器學習的交叉領域。對於這一領域的研究者來說，CoRL是一個來展示和交流自己的研究成果的全新會議。值得注意的是，CoRL大會主席由UCBerkeley知名機器人學者KenGoldberg、UC Berkeley學術新星Sergey Levine、Google首席科學家、Google Brain團隊Director Vincent Vanhoucke擔任，均是能引領最新前沿研究的重磅人物，可謂是一個高起點的年度學術會議，也有望成為下一個機器人與機器學習領域的頂級會議。

今年的CoRL2019會議，即第三屆CoRL會議於11月初在日本大阪舉行。本文梳理了一些該會與自動駕駛最相關的論文概述。

1.End-to-End Multi-View Fusion for 3D ObjectDetection in Lidar Point Clouds


https://drive.google.com/open?id=18LymYUkOVkj-xbKQElHzECW1VDbnx4wY

【簡介】這篇論文來自Waymo。作者將兩種用於三維檢測的過程激光雷達讀數方法結合起來:自頂向下鳥瞰圖、以及第一人稱視角。由此產生的網路在Kitti數據集上實現了最先進水平。

2.Identifying Unknown Instances forAutonomous Driving


https://drive.google.com/file/d/1e_kBfHEL9adDWwhuhSrOHz-CUqRbLe2T/view

【簡介】這篇來自Uber ATG的文章提出了一種新的目標識別方法。通過對嵌入空間的聚類，系統可以檢測出一個對象是屬於已知類還是未知類。

3.Worst Cases Policy Gradients


https://drive.google.com/file/d/1v7gGbUtAG58MSahje9OHRymMbr_qunu-/view

【簡介】來自Apple的文章探討了一種強化學習的新方法，即學習不僅使獎勵最大化，而且使安全邊際水平最大化的政策。模擬評估表明，自動駕駛汽車將學習從最危險到最保守的行為。

4.Inferring Task Goals and Constraints usingBayesian Nonparametric Inverse Reinforcement Learning


https://drive.google.com/file/d/1bswpgVJDXp_9vh55_Gz1cAbylhhjQqhS/view

【簡介】本文來自麻省理工。文章描述了運動規劃的約束條件和成本函數設計是運動規劃中的兩個關鍵工程瓶頸，文章詳細展示了如何從人類演示中推斷約束和成本函數。

5.MultiPath: Multiple Probabilistic AnchorTrajectory Hypotheses for Behavior Prediction


https://drive.google.com/file/d/1q-vNE6pN56A-LfdcaRyok94uwEuS9tyN/view

【簡介】來自Waymo的另一篇文章中闡述了預測被觀測車輛未來軌跡的能力。與以往的研究方法不同的是，該方法的分布被建模為混合高斯分布，從而實現快速推斷預測的目的。

6.Learning by Cheating


https://drive.google.com/file/d/1zqKgYuas6-EosH4PNA0CRawAwCSE79U6/view

【簡介】這篇論文投稿自Universityof Texas。本文提出了一種以攝像頭作為輸入的基於人工演示的訓練神經網路驅動汽車的新方法。這是首個在CARLA模擬基準測試上首次達到100%成功率的方法。

7.Discrete Residual Flow for ProbabilisticPedestrian Behavior Prediction


https://drive.google.com/file/d/1LsTuOJOhe7TLjNxSGJNRVDLzlEswCeEy/view

【簡介】來自Uber ATG的另一篇論文描述了一種方法，該方法可以預測未來行人在汽車周圍的姿勢。與以前的方法不同，該網路可以捕獲未來軌跡的多模態分布，並在較長的時間範圍內進行預測。
