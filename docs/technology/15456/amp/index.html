Java 危矣！統治地位已不復存在？

天天要聞
2019-12-15

作者 | loonggg

本文經授權轉自「非著名程序員」

有雲天下，有人江湖，編程的江湖亦是如此。編程的江湖上也是豪強諸起，門派眾多，各足鼎立。雖說沒有什麼武林大會，也沒有華山論劍，但是編程的江湖，也不比他們差，亦是精彩紛呈。

在比較的老派和老牌的門派當中，C 門派最為受人尊敬，輩分較高，C++ 門派都屬於 C 門派之列，後來由於其獨有的氣勢和特點， C 門派中的 C 和 C++ ，都有一種華山門派劍宗和氣宗的意思，但是畢竟同氣連枝，在面對共同敵人的時候，還是師兄弟的關係。

而 Java 門派其弟子更是眾多，自誕生之初，尤其善於後端技術，以其精湛穩定的技術持續火爆了近 20 年，在江湖上也有不減之勢。

在武林排行前十的門派中， PHP 也是一個強勢的存在，對它的讚美和推崇亦是讚不絕口，江湖中畢竟傳言：它才是世界上最好的門派。要論天下武功，唯有 PHP 。作為後起之秀，吸收了 Java 和 C 以及 Perl 等技術特點，很快蠶食了和席捲了 Web 端，讓 Java 在 Web 端損失慘重。

在此期間，Python ，Ruby ，JavaScript 等門派都在涌動的暗流中暗自發力，期待能有一天，與武林中揚名立萬，一統江湖。

這都是在互聯網時代上叱吒風雲的門派了，C 門派在其速度上是其他門派所不及的，可謂穩准狠，軟硬通吃，而其他門派都是吃軟不吃硬，隨著時代的發展，到了移動互聯網時代，Java 竟然在 Android 地界上開闢了一個新天地。

隨著歷史進程的演進，誰先站住終端平台，誰就有先發制人的優勢，不知不覺就在被人追趕超越的時候，Java 在 Android 的地界上火了，已然成為了燎原之勢。一時間讓其他門派望其項背，長期佔據琅琊榜武林門派排行的第一名，其氣勢猶如武林盟主一般。殊不知，在更換了掌門以後得 Java，作為掌門人的 Oracle 喜歡使用陰招，到處往其他門派中送出間諜和卧底，讓其他門派有點措手不及，使用武林名聲要挾其他門派，說其他門派盜其武林絕學，索要各種損失費。

在半夢半醒中，在洋洋得意時，JavaScript 忽然來到了 Java 門派這裡，送上了拜貼，求見 Java 。

JavaScript 行色匆匆，慌慌張張的見到 Java：「Java 師兄，小弟來找你有急事求見。」

「你 TMD 的跟誰攀關係呢？誰是你師兄，你從哪裡論的？我跟你沒毛關係。什麼事？有話快說，有屁快放。」Java 一臉孤傲且有點生氣的說。

JavaScript 心裡暗想：「好吧，我不跟你計較，自己倒霉了，還不知道，傲嬌個屁啊！」

「你知道嗎？據從谷歌那裡傳出來的消息說，在功力較高的弟子中有 35% 的人在 Android 地界都投入到了 Kotlin 門派，而且這個數字正在逐月遞增，越來越多的兄弟要加入了。你損失慘重啊！」JavaScript 暗自慶幸且看笑話的說。

Java：「我知道了，我門派弟子都投向其他門派還不知道嗎？用你來說，這也不能怪我啊，自從換了掌門，在於谷歌打口水仗以來，看似勝了，其實又敗了，我也無能為力。你小子也不是什麼好東西，你來看我笑話的吧？」

JavaScript：「不如咱們開個武林大會吧，召集眾門派，劃清地盤，分清界限，最好井水不犯河水。」

Java 心裡暗想：「劃清地盤，分清界限？就尼瑪 JavaScript 劃不清楚界限，分不清楚地盤，自移動互聯網時代以來，你野心勃勃，到處搶生意。肚子里肯定沒什麼好事，一肚子壞水，我要看看你到底想幹嘛？」

「好，那咱就召開，由 C 門派主持會議，由我光發英雄貼，你負責整個大會的安排，具體你操作。」Java 試探性的說。

JavaScript：「好，馬上干。一溜煙高高興興的馬上去行動了。」

武林大會在 JavaScript 的安排下，如期舉行，看在 C 和 Java 的面上確實來了不少門派。C 老祖出來主持會議，說：「今天召集大家來，是為了我們江湖和諧穩定，不要再各自為戰，造成內耗。」

JavaScript 說：「C 老哥說的對啊，我們要團結，要合作，不要互相搶佔地盤。我們 JavaScript 門派一向都是主張合作，在各領域合作，互利共贏。不管是前端，還是後台，各項方面，只要需要，我們都可以合作，因為哥們活全。」

「卧槽，原來召開大會的目的是這樣啊！JavaScript 想暗度陳倉，通力合作，一統江湖啊！」Java 內心想。

Python：「通力合作？你們 JavaScript 確實活全。但是不精通，想合作行，先把你師弟 Node.js 從後端撤出再說。現在是什麼時代？人工智慧時代，哥們怕你們？」

「PHP，你有什麼要說的？」C 老祖問。

「我沒什麼要說的，我們很專註，我們有自己的能力和業務，作為江湖上傳聞最好的門派，我們一向低調。我們都行。」PHP 說。

OC 說：「我想靜靜，Swift 一出，我家裡都快揭不開鍋了，不是親生的畢竟不行啊！」

Java 這時頻頻點頭。

Kotlin：「對啊，我雖然不是親生，勝似親生，但是畢竟官方首席推薦了，我還是比較幸運的，對吧，Java 老哥？」

Java：「你牛逼，你厲害行了吧？沒有我和 JavaScript 的支持，你屁也不是。在我們的環境下生存，還搶我們的飯碗。」

JavaScript：「大家不要吵嘛，咱們目的是合作，只要能做生意，掙錢，都好說。咱們要不選個盟主？用於調節糾紛和合作。」我感覺選擇個活全的當盟主，有利於領導，畢竟都知道各處的困難嘛！

PHP：「論輩分，論資歷，論能力，論功力，當然……當然不是我，我推薦 C 老哥。」

C 老祖：「不敢當，不敢當。」其實內心想，這破盟主有啥好當的，亂燉。

Java：「要論最近 20 年的流行度和統治力的話，我應該當仁不讓。」

Python：「切，你沒看最近的琅琊榜武林排行嗎？把榜單拿來瞧一瞧，誰是第一？在人工智慧時代，流行度，我第一，排行榜，我第一。你沒落了，心裡沒點逼數嗎？」

Go：「go,go,go，我走了，太吵人了。」Go 暗想：「這這些年都說要推翻 java 的統治地位，也沒見他被推翻，作為後起之秀的我，不想這些，只想努力做好自己。」在大會上一言沒發的 go 走了。

其他門派各自不服，吵聲越來越大，而 JavaScript 在暗喜，我的目的達到了，吵個你死我活，我漁翁得利，等你們兩敗俱傷，我好一統江湖。

總結，以上內容純屬虛構，如有雷同，沒錯是我寫的。關於 Java 危矣的言論，我們就讓 Java 自己現身說法。

Java ：好了，就讓他們吵吧，我來說說我自己的觀點，不可否認 Kotlin 在蠶食我在 Android 端的市場，它有它的優勢，我也有我的特點，而且 Kotlin 現在總體來說確實不錯，但是不能跟我 20 多年的穩定和積累所比。而關於大數據，人工智慧方面，Python 確實增長很快，但是等你真懂了大數據和人工智慧，你都不會去想你用的是 Python 還是 Java 了，在糾結語言的時候，說明你還處在初級階段。另外，我在自誕生以來，就沒錯過互聯網上的任何一個熱點，所以才會這麼持久。JavaScript 不就是因為我 Java 太火太流行，為了蹭我的熱點才起這跟我有關係的名字嗎？其實一點關係沒有。當程序員語言只是工具，不要太在乎用什麼工具，應該在乎如何用好工具，如何使用工具。
