江卓爾：為什麼說現在仍處於長牛市周期

2019-12-22 1,747條評論 3,521次閱讀 2320人點贊

編者按：本文來自獵雲財經，Odaily星球日報經授權轉載。

12月21日，萊比特礦池創始人/比特幣佈道者江卓爾在 「2019國際數字礦業高峰論壇」上發表主題演講。觀點獨到精彩。

針對很多外界的人說數字資產是泡沫，江卓爾表示，在數字資產發展歷程中，經歷過數次下跌，有崩盤跌到93%，2013年崩盤跌到87%，上一輪崩盤跌到85%，「沒有任何一個泡沫可以歷經這麼多次崩盤，歷經這麼長時間的下跌還保持不破的，所以說數字資產不是一個泡沫，數字資產肯定有內在價值。」

數字資產的價值在哪裡？江卓爾認為，數字資產靠去中心化帶來了經濟自由。在數字資產網路的所有節點裡，沒有一個節點是「老大」，不能靠消滅老大來把數字資產的網路消滅掉。因此，江卓爾將數字資產比喻為是升級版的一線城市房產，長期將持續升值。

江卓爾建議：「我們要進行超長期囤數字資產，在十年甚至二十年的基礎上，我們都要一直持有數字資產。」

但恐怕很多人都有同一個問題，沒錢買數字資產怎麼辦？江卓爾以個人經歷建議挖坑可以解決這一痛點。同時，挖坑還具有三大妙處：一是良性負債才是資產；二是挖礦合規、有固定資產、有現金流，容易延伸出融資、合資、賣鏟子等業務；三是被動拿住數字資產，不因上漲拿不住而跳車。

以下為演講實錄（經編輯整理）

大家好！

今天我剛把這個題目提交給組委會的時候，他們還問我，你確定現在是牛市嗎？可能大家都覺得現在熊市來了，或者說處於熊市周期。

但我認為，從長期來看現在還是處於長牛市的周期，就是在四年的周期里，現在還是牛市周期。雖然現在價格跌了不少，但歷史上是發生過這種情況的。

我們看一張歷史上的K線，這裡面有一段小牛市，幣價瘋狂上漲，突然崩盤下來了，一直到最後的時間階段，熊市來了嗎？這實際上是2013年上半年的小牛市，可以看到出現了非常漂亮的V型翻轉，短時間下跌後又迅速開始了一輪上漲。即使有一次比較深的跌幅，但是在周期來看，仍可能處於大的牛市周期中，光從價格上來說，現在價格漲了2倍，所以我覺得現在還處於牛市周期。

價格短期不可預測，長期才可預測。舉個簡單例子，通貨膨脹，鈔票會越印越多，但如果要預測短期的房價是比較困難的，即使是香港這種土地特別少的地方，也出現過房價大幅下跌的情況。但如果拉長一個周期，20年後大排面對人民幣一定漲價。大排面需要材料、人工、店面、房租，這些成本加起來，人民幣一定會漲價，這是毋庸置疑的。這就是我們所說的，越長期的東西越可以預測。我們通常還可以預測一線周期的房產，對大排面來說也一定是漲價的。因為大家都發現，手上的鈔票越來越不值錢，大家怎麼把自己的財富儲存下來？聰明的人會想到不會被通貨膨脹滅奪的資產，比如一線城市的房產，一定會比大排面漲得更快，並且會引入到這個地方。

我們現在處於數字資產歷史全周期的哪個位置？（PPT圖示）這張圖的價格是一個對數坐標，每上升一個價格要乘10倍，上一次減半之前的漲幅，雖然看起來不多，但其實是100倍，前半點到最高點漲了500倍，差不多從10美元漲到1000美元，對比最低點減半的漲了100倍。可以看到，每一個減半點前後都會出現一次對稱性的上漲，減半點之前漲的慢一點，減半點之後漲得快一點。前面其實是一波小牛市，後面是一波大牛市。上次的減半大家更熟悉一點，在減半之前，從最低點開始，差不多是900人民幣，賣了5000美元左右，後來最高漲到13萬人民幣，兩次都形成了非常明顯的周期循環。

為什麼會形成這個明顯的周期循環？因為數字資產減半對供需平衡的破壞是非常直接的。哪怕現在已經第三次減半了，但現在每次減半，都會比以前少生產900個數字資產，這是什麼概念？按現在的價格來看，相當於每天有人買入4500萬元數字資產，並且是持續不斷凈買入的。現在是第三次減半，再往後減半，效果會越來越弱，但是在現在這個周期，我們可以看到減半的效果還是非常強大的。

減半破壞了供需，會開始迅速上漲。從供需來說，假設長期需求是不變的，有些人需要買數字資產，或者需要用，這個長期需求短時間內是不會改變的，那你在供給減半之後，價格必須上漲一倍才能重新打亂供需平衡。但是價格上漲1倍以後，馬上會開始牛市循環。很多人通過新聞知道數字資產漲了，他們進來買，導致價格上漲，最後形成了正循環。

要了解挖礦，一定要了解數字資產。挖礦看起來好像是工業生產行為，可以看到有很多廠房，有幾千台的機器，有大量的工人在那邊維護，看起來好像是工廠。但實際上挖礦本質是金融行為，如果你不能從金融角度進行理解，你就沒法從中賺到大錢。

為什麼數字資產能漲幾萬倍？很多外界的人會說數字資產是泡沫，有哪一個泡沫可以連續漲10年？實際上在數字資產里有非常多崩盤，有崩盤跌到93%，2013年的崩盤跌到87%，上一輪崩盤跌到85%，沒有任何一個泡沫可以歷經這麼多次的崩盤，歷經這麼長時間的下跌還保持不破的，所以說數字資產不是一個泡沫，數字資產肯定有內在價值。那麼價值在哪裡？

數字資產靠去中心化帶來了經濟自由。互聯網就是去中心化的，一開始是美國軍方為了防止核戰爭通訊中樞被打擊，國家通信會癱瘓，所以他們發明了互聯網，互聯網是沒有中心的。我們現在找不到哪個地方叫互聯網中心，也沒有任何一個總控開關可以把互聯網關掉。數字資產也是一樣的，所有節點裡，沒有一個節點是「老大」，你不能靠消滅老大來把數字資產的網路消滅掉，所以說誰都消滅不了數字資產。

什麼叫經濟自由？10月24日中央大力推了一次區塊鏈技術，中央要發行DCEP。為什麼中央要發行DCEP？為什麼中央要大力推薦區塊鏈技術？實際是為了對抗美國可能發生的金融封鎖，因為我們在老二的位置上待了很長時間，並且我們現在還不斷向前進，很有可能我們在幾十年的時間內會取代美國成為老大，美國也反過來對我們進行了大量壓制。經濟自由，指的就是做生意的自由，你可以決定你要跟誰做生意，你有能力決定要把什麼東西賣給別人，或者向別人買東西。

舉個簡單的例子，中國跟伊朗的經濟關係，伊朗有大量的石油，中國很擅長基建，理論來說這兩個國家之間的貿易，中國可以得到石油，伊朗可以得到基建或其他的工業產品。但這時候美國進來插了一腳，說伊朗是邪惡國家，中國不能跟伊朗貿易，世界上任何國家都不能跟伊朗貿易。華為的孟晚舟被扣，也被指控她跟伊朗做生意了。這時候伊朗人就很頭痛，因為他所有對外的交易都被美國封鎖了，所以伊朗開始使用數字資產，他們發現數字資產特別有意思，只要有一個手機就可以使用，並且美國人還封鎖不了。美國人為什麼封鎖不了數字資產？就是因為區塊鏈技術的特性，決定數字資產是沒有一個中心可以打擊的，所以美國政府不能像監管銀行一樣，靠給總行下指令，把伊朗的數字資產地址給封鎖掉。雖然曾經提出一些地址，說這是伊朗的需要封鎖，但實際上他們是沒有能力做到這一點的。

中國從長期來看，也在為美國有可能發生的金融封鎖做準備，剛剛也已經提出來了，大家看新聞的話都知道，美國說要把華為踢出國際結算體系之外，這是很危險的，所以我們要建立自己的結算體系。什麼樣的結算體系？就是基於區塊鏈技術的數字人民幣。有可能銀行是無法在其他國家或在歐美開設分行的，但只要有了數字人民幣，其他國家的人只要有了手機，就可以直接使用人民幣，這就是典型的區塊鏈技術促進了經濟自由，這是一個剛需。剛需使數字資產有著堅實的價值基礎。

所以說，數字資產是升級版的一線城市房產，長期將持續升值。我們要進行超長期囤數字資產，在十年甚至二十年的基礎上，我們都要一直持有數字資產。

但恐怕很多人都有同一個問題，沒錢買數字資產怎麼辦？2015年底的時候，實際我也碰到過這個問題，當時我在知乎寫了「數字資產是什麼？」，之前每次數字資產價格波都是特別劇烈的，每次都上竄下跳，但如果看另一個數據，數字資產的活躍地址標識出了數字資產用戶數的使用情況，不管價格怎麼上竄下跳，數字資產使用的用戶數是一直在快速增長的。他們得出一個結論，數字資產牛市要來了，但我已經把所有的錢都買進去了，怎麼辦？

所以我那個時候就開始挖礦，挖礦有很多妙處：

一是良性負債才是資產。比如你十年前買了一棟房子，花了100萬，十年後這套房子漲到1000萬，請問你的資產增加了嗎？實際是沒有增加的，因為你總共只有一套房子，現在你還是只有一套房子，並且房子里的裝修變舊了理論來說，你的資產是上漲的，什麼時候你的資產可以增加？就是十年前的時候，你不是花100萬去買房子，而是把100萬拆成3個30萬，按揭去買了3套房子，這時候你看起來凈資產是一套房子，但是過了十年以後，你發現這套房子漲到100萬，剩下70萬你還貸款，剩下的60萬或50萬的負債對1000萬的房價來說已經微不足道了。這種情況下，你的資產才是增加的，所以要深刻理解這一點，你的負債才是你的資產。

在快速通道的世界裡，你的債務利率要低於通脹速度，你的良性負債才是你的資產。你欠了一些錢，比如最良心的負債就是房貸，房貸利率只有4%-5%，膨脹10%左右，房價平均上漲速度是20%以上的，所以過了十年以後，你當時用負債買東西，剩下的就成為了你的資產。

挖礦也是一樣的，2015年底、2016年初的時候，我借了錢，在價格乘10倍、100倍上漲之後，基本都歸零了，這是微不足道的，跟剛才講的房子的道理是一樣的，剩下的東西就變成了我的資產。

二是挖礦合規、有固定資產、有現金流，容易延伸出融資、合資、賣鏟子等業務。

三是被動拿住數字資產，不因上漲拿不住而跳車。這對很多普通人來說是有好處的，對礦工來說，在牛市周期里，礦機是「會下金蛋的金雞」，礦工這時候不會把賺錢的礦機賣掉，他們會從一開始拿到最後，反而賺的是最多的，比很多持數字資產者賺的多。

但反過來說，挖礦大概率挖不回數字資產。（PPT圖示）這是上一輪S9的統計圖，橙色的線是S9的靜態回本數，牛市前半段的回本天數在259天左右，後半段下降到100天，也就是說整個牛市周期差不多是720天，就算按照平均200天來算，挖礦差不多能回本4次，當然復投收益會更高，但比不上持數字資產，持數字資產從頭到尾是100倍，挖礦是5倍，復投大概在10-20倍左右，這要看具體的市場情況。下面這條線是數字資產的挖礦產出，大概在5塊錢左右。為什麼會這麼穩定？實際那個時候價格已經漲了幾十倍，但算力對應也漲了幾十倍，總能跟上價格的漲幅，所以最後會導致數字資產挖礦從人民幣來算，是一個好生意，但是從數字資產來算，不一定能把數字資產挖回來，這是我們在做一些投資計劃，尤其是引入一些投資的時候，儘可能把人民幣用來買礦機、交電費的原因。因為你用它交電費，基本也能拿到票，這些支出全部都合規。儘可能把這些人民幣都轉化為數字資產的形式囤積下來。

這就是我們說的，挖礦不是目的，囤數字資產才是目的，挖礦一定要挖，但數字資產一定要囤住。

最後以我很喜歡的一句話結束演講：數字資產，多一種選擇，多一份自由。
