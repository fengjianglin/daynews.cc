沒想到大多數Google員工都出自這些學校！他們都是如何準備面試的

天天要聞
2019-12-18

你是不是也曾好奇過，那些進入谷歌、facebook、高盛、uber的人都是什麼來路。

哪些學校，在大公司眼裡更具有競爭力？

以谷歌為例，Google在全球擁有超過6萬名全職員工，並且常年穩居最受歡迎工作地點的「上位圈」內，因起薪高，福利好，成為無數留學生心目中的dream company。

然而，每年約有250萬至350萬人申請Google的職位，這之中，平均只有0.13%（約4000人）最終拿到offer。

那麼，Google員工大多數都來自哪些學校呢？

從LinkedIn的提供的數據中我們發現，斯坦福大學，加州大學伯克利分校和卡耐基梅隆大學分別是前三名在Google擁有最多校友的高校。而前20名高校中，1/3的學校都來自加州。

值得一提的是，清華大學名列第13位，成為了榜上唯二的非美國本土的學校！

另外在這個排行榜中，你可能還會發現一個不那麼眼熟的名字：San Jose State University（簡稱「SJSU」）。它的名氣在中、東部可能不那麼大，但誕生於「科技大本營」矽谷的它，卻是為Google源源不斷地輸送新生力量的黑馬。
