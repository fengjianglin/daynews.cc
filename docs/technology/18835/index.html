HUAWEI DevEco Studio Beta全面開放申請

2019-12-16 1,460條評論 2,607次閱讀 881人點贊

近日，華為開發者聯盟開放了HUAWEI DevEco Studio「試用申請」，廣大開發者終可以參與到這個體驗活動中。

HUAWEI DevEco Studio(簡稱DevEco Studio)是華為消費者業務部為開發者提供的集成開發環境，旨在幫助開發者快捷、方便、高效地使用華為終端開放能力。DevEco Studio具備工程管理、代碼編輯、編譯構建、調試模擬等基礎功能，並支持華為開放能力包(例如互聯互通、HiAI、安全、HiCar、HMS)等特色功能。

2019年8月的華為開發者大會上，華為發布了這個IDE平台，並計劃於2019年底提供一個友好試用版本，現在看來華為確實在按原計劃有條不紊地執行。開發者們可以搜索華為開發者聯盟，登陸華為開發者聯盟官網，進入【開發】—【開發工具】—【HUAWEI DevEco Studio】，點擊【試用申請】來開啟體驗之旅。

這次試用的版本有哪些功能?

「本次開放邀請Beta版本基於IntelliJ IDEA Community開源版本定製開發，除提供基礎的應用開發功能外，更集成了華為開放能力Kits、遠程真機調試和應用雲測試等功能，在後續的版本中將進一步集成華為分散式能力、一次開發多端部署等功能…」

從官方放出的說明和簡單上手後得出的體驗來看，有如下幾點值得一說：

1.基於IntelliJ IDEA開源版本定製，開發習慣不需要改變

DevEco Studio跟Android Studio一樣，也是基於IntelliJ IDEA Community開源社區版開發。因此，基本的開發和調試功能，如新建工程模板、終端模擬器、布局編輯器，可拖拽式UI控制項並預覽、調測、代碼庫管理等等，DevEco Studio使用起來不需要新的學習和改變。如果之前是Android Studio的用戶，可快速平滑地過渡到DevEco Studio。

2.內置了EMUI開放能力一站式開發和管理

DevEco Studio深度集成了華為EMUI的開放能力，包括HiAI、多媒體、互聯互通等，提供了」Kit Manager」進行一站式的管理。開發過程非常簡單，選擇對應的Kit就可以引入工程，省去很多依賴手工添加的工作。

在諸多功能中，特別推薦一個叫「Kit Assistant」的功能，其可以對主要功能的API提供了參考和代碼樣例，支持一鍵拖拽的方式將代碼樣例插入到源碼文件中直接調用，功能效果很不錯。

3.All-in-one的測試和質量分析能力

Remote Devices功能提供5000+華為全系列遠程真機測試的能力，7*24小時免費在線，隨時隨地響應，能夠充分滿足開發者真機測試資源訴求，不再只限於模擬器測試;

App CloudTest支持原生應用、混合應用和遊戲類應用的自動化測試，覆蓋兼容性、穩定性、性能、功耗、安全、多語言等多種測試場景。

DFX測試可實現精準檢測，同時提供非侵入式APP的質量分析服務，涵蓋Crash、ANR、啟動時長、丟幀、啟動卡頓等指標，所有服務在一個平台即可完成。

離「一次開發、多端部署」有多遠?

當前，行業通常做法是為每種設備單獨設立開發團隊，對不同終端單獨的開發適配。但是在8月華為開發者大會上，DevEco Studio發布的一個重要特性是支持1+8+N全場景多終端的應用和軟體開發，實現一次開發、多端部署。提供統一的UI編程框架和分散式應用編程框架，只需要開發和維護一套代碼就能解決多設備適配的問題，有效的降低了開發難度和技術門檻。

不過從當前這個Beta版本看，還沒有支持這個功能，應該會在後續Beta版本中提供。

寫在最後

從此次Beta版本看，HUAWEI DevEco Studio基本思路是站在IntelliJ IDEA的巨人肩膀上，增加華為定製化的服務，如EMUI開放能力、遠程真機、雲端測試等增強功能。後續還將逐步擴展一次開發多端部署等重點特性，應該是走一條非常務實的道路。

這條路讓人想起華為消費業務軟體總裁王成錄博士在發布時說的一句話：「軟體工程技術它特別像硬體領域裡的生產線，如果沒有強大的生產線，靠人工拿鎚子是一定敲不出來一艘航母的。我希望華為DevEco Studio可以像一個巨大的容器，把所有優秀的軟體編碼人員的經驗都沉澱其中，為開發者提供越來越強大的工具。」

如果你也愛移動應用開發，不妨下載HUAWEI DevEco Studio一試
