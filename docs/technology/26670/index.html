有趣的Python小項目，利用Python製作一款截圖識別軟體

先給大家推薦一款截圖軟體，非常方便，可以把截出的圖片放置到窗口上。

並且可以隨意移動，這個是微軟開發的一款工具：Snipaste。

私信小編01 領取完整項目代碼

支持各類電腦系統，先簡單介紹一下它的用法，F1截圖，ctrl+c把圖片複製到剪貼板，方便下次使用（ctrl+v）。

我們要做一款截圖工具，也就是模仿這款軟體的功能。

利用到的python模塊有：

  • import keyboard
  • from PIL import ImageGrab
  • from time import sleep

keyboard為我們的鍵盤記錄模塊，判斷我們是否按下了F1鍵，PIL為我們的圖片處理庫，導入了其中的剪貼板模塊。

若未按下F1那麼程序處於阻塞狀態，只有按下後代碼才會向下執行。先F1截圖，然後Ctrl+c保存我們的圖片到剪貼板。

最後把圖片從剪貼板中取出來保存為 test.png。
