新版汽車三包規定即將出台 罰金暴增十幾倍 車企4S店集體顫抖

2019-12-21 156條評論 1,489次閱讀 375人點贊

據國內媒體報道，12月20日，在2019中國車商高峰論壇上，國家缺陷產品管理中心三包部主任賀興在發言中稱，新版汽車三包規定即將發布，主要針對新能源汽車、退換車條件、處罰額度等方面進行調整。

賀興介紹，我國汽車三包規定自2013年10月1日起在全國實施，6年以來，汽車三包規定在保護消費者合法權益、促進汽車產品質量提升方面都發揮了重要作用。

而隨著新技術的發展及產業的升級，汽車三包規定也面臨新問題、新挑戰。比如，針對新能源汽車搭載的動力電池，在現行的汽車三包規定里並沒有特別詳細的規定。

早在今年3月份，國家市場監督管理總局發布消息稱，市場監管總局組織對《家用汽車產品修理、更換、退貨責任規定》進行了修訂，現向社會公開徵求意見。

經過數個月的意見收集整理與研討，目前，新版汽車三包規定已經形成了基本的制定思路。新版汽車三包規定將新能源汽車動力電池、電機納入到退換車條件里。

在國家規定的2年/5萬公里的三包期和3年/6萬公里的保修期期限的要求下，還單獨對動力電池提出了要求，要求純電動車型必須明確標註動力電池的衰減限制和相關的測試方法。

在消費者最為關注的新車退換方面，新版三包規定也做出了相應調整。在現版的汽車三包規定中，有效期內且因產品質量問題維修時間累計超過35日，或同一產品累計修理超過5次的，消費者可以憑三包憑證、購車發票要求換車。

而在新版三包規定中，賀興介紹，這兩個數字分別縮減為了30天和4次，向著消費者更有利的方向進行了調整。

此外，為了更好的保護消費者的權益及對商家企業或廠商產生更大的威懾力。新版汽車三包規定將拒不履行三包規定行為的企業處罰金額由之前的3萬元，提升到了50萬元，同時要求汽車主機廠支持和配合汽車經銷商實施工作。
