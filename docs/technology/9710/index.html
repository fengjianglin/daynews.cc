豐巢回應公司註冊資本減少13億：已註冊新主體公司

2019-12-10 894條評論 2,486次閱讀 608人點贊

記者|楊霞


豐巢快遞櫃資料圖（拍攝：楊霞）

日前，豐巢快遞櫃的主體公司豐巢科技發生重大工商變更，引發外界關注。

據天眼查顯示，深圳豐巢科技有限公司12月6日發生工商變更，註冊資本由原來約24.50億元變更為約11.67億元，減少了大約13億元人民幣，減少幅度約52%。

對此，豐巢CMO李文青回應界面新聞稱：「公司經營發展需要設立了新的主體和組織，對應有相關的調整，新的主體名稱為豐巢網路。」

界面新聞查詢天眼查發現，深圳市豐巢網路技術有限公司（即豐巢網路）已經於2019年7月24日成立，法定代表人為豐巢創始人及CEO徐育斌，註冊資本為30.49億元。


天眼查顯示，豐巢網路在2019年11月1日完成了註冊資本、投資總額、投資人等多項工商變更。其中，註冊資本由1000萬元變為30.49億元，投資總額由1428萬元增至90億元。

由此看來，較原來豐巢科技24.50億元的註冊資本，豐巢新的主體公司註冊資本實際有增無減。

豐巢是一家智能快件櫃服務提供商，主打產品是24小時快遞自助開放平台——豐巢智能快遞櫃，以提平台化快遞收寄交互業務，主要提供自助寄件、收件等服務。公司產品設計覆蓋物流快遞、社區服務、廣告媒介等領域。

公開資料顯示，2015年6月7日，順豐聯合申通、中通、韻達以及普洛斯集團共同宣布創建豐巢科技，總投資規模為5億元，其中順豐占股35%，申通、中通和韻達各佔20%，普洛斯則佔5%。

此後，中通快遞、申通快遞、韻達股份相繼宣布轉讓持有的豐巢科技全部股權。至2018年6月14日，通達系快遞公司全部從豐巢股東名單中退出。

12月7日，豐巢科技的公司還發生了股東變更。其中，值得注意的是普洛斯也從豐巢科技的股東名單中退出。

目前，豐巢科技共有七大股東，其中大股東深圳瑋榮企業發展有限公司（屬於順豐集團）持股比例為46.07%，深圳順豐控股有限公司持股30.29%，深圳明德控股發展有限公司，持股比例為12.28%。豐巢科技的法定代表人為豐巢創始人兼CEO徐育斌，但實際控股人為王衛。

對比豐巢科技與豐巢網路，儘管兩家公司的法定代表人均為豐巢創始人及CEO徐育斌，但是從現有工商信息來看，在股權關係上，新公司主體豐巢網路暫時無法體現出與順豐集團的關係。

目前，豐巢網路的股東為豐巢（香港）有限公司（HIVE BOX (HK) LIMITED），持股比例為100%。後者是一家在香港註冊的私人股份有限公司，成立日期為2019年8月6日。
