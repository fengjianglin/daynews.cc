拯救愛奇藝：漸成雞肋的視頻VIP會員

天天要聞
2019-12-12

這兩天，因為騰訊視頻播放《慶余年》時，需要VIP會員也付費觀看，網上吐槽一片。

更早之前，騰訊視頻的《陳情令》就已經如此這般的操作了一番，這次只不過是有一次故伎重演。

不約而同的是，愛奇藝對剛上線的《中國機長》，也是給出了VIP會員需要另外再支付6元的點播費操作，XXX！

一、為什麼要做VIP會員制

今年以來，我先後撰寫了《互聯網平台的2019：押注內生廣告、內容電商和會員制》《三種內容付費方式，哪種更容易獲得成功》《全民VIP：為什麼要做付費會員制？怎樣做付費會員制？》《張棟偉：怎樣正確的設計付費會員制？》等文章，探討了目前付費會員制的原理和玩法。

在用戶增長紅利不再爆髮式增長的今天，通過內生廣告發掘存量市場，通過內容電商開發增量市場，以及通過會員制來保障持續增長，已經是互聯網平台2019年的必需戰略。

為什麼要做會員制？

因為營銷本質正在從「以商品為紐帶」，過渡到「以人為紐帶」。

市場營銷的本質，從售賣商品，變成了經營用戶。

從前的產業鏈「產品–渠道–銷售–服務」，變成了「拉新–留存–促活–轉化」。

以愛奇藝為例，目前愛奇藝的總訂閱會員人數為1.058億人，與去年同期的8070萬人相比增長31%。截至2019年9月30日，愛奇藝的總訂閱會員人數中99.2%都是付費訂閱會員。愛奇藝曾把自己的會員數達到了億+級作為重點公關宣傳事件，進而引發了騰訊視頻、優酷土豆誰會成為下一個億+俱樂部成員的討論。

如果不是媒體對COSTCO的廣泛報道，恐怕我們還很難接受「在購物之前先付一筆無關的支出」這種消費習慣。但事實上，國人的學習能力和接受能力很強，數年之間已經在國內的互聯網平台上找不到完全免費的VIP服務了。

二、殺雞取卵還是飲鴆止渴

預先付費的VIP會員制，是為了完成付費形成的「等級優越感」「行為活躍度」「排他性競爭」。

而錯誤的付費會員制設計，則是「把會員製作為拉新手段」「把會員制當成收入手段」「把會員制運營作為獨立職責」。

沒有哪個手段是全能的。會員制也一樣。

回顧上面一節的那句內容：「拉新–留存–促活–轉化」，只有「留存」是會員制的訴求核心。

在開展會員制的時候，需要兄弟部門為「留存」這個動作做出非常多的支持，從流量導入到權益獲取，特別是跨企業跨行業的相關權益，遠非單獨的用戶運營部門所能獨立開展。比如從財報上看，視頻網站實際上是把把原本應該由廣告來表現的收入，轉成了內容訂閱收入。

也就是說，付費會員制是一個業務戰略，而不是業務戰術。用戶在A平台成為付費會員之後，就會把消費頻次重點沉澱在A平台，從而換取積分增長和其他留存權益，繼而降低使用類似平台和服務的機會。

付費會員制度本身還是一種進攻戰略，是成為運營式市場營銷的核心手段之一。

但是從騰訊視頻、愛奇藝不約而同的開始向會員單獨收費，此外VIP會員還能享受自己獨特的「推薦廣告」來看，視頻網站玩家們並沒有深刻理解會員制的意義，為了快速降低虧損困局，採用了飲鴆止渴的模式，要把VIP會員作為「收入」而不是「留存」來運營了。

比如愛奇藝會員及海外業務群總裁楊向華就公開表示：「愛奇藝正在醞釀會員費用上漲，不排除會員率先提價，但並無時間表。」也就是說，愛奇藝的會員有可能要漲價了，只是時間尚未確定。愛奇藝創始人兼CEO龔宇也曾抱怨，「20塊的會員價定低了。」

對此，網友普遍性的回答是：召喚迅雷，回歸D版！

—當下載一部電影只需要幾秒鐘的5G時代即將來臨的前夜，視頻網站竟然要用額外收費或者漲價的方式來把用戶趕回到D版陣營，實在匪夷所思。

三、以產業互聯網的高度破局

無論是愛奇藝、騰訊視頻還是優酷，實際上都還是電子商務邏輯，即「賣內容的網店」。

在當前消費互聯網已經觸碰到天花板的今天，視頻網站單純依靠選品（SKU）、提價（ARPU）、價值組合（如88會員）都已經不能從根本上破局。其實只要看一眼財報數據，都不需要大腦分析，就能明白視頻網站的營收虧損主要是來自於無窮盡的內容購買黑洞。

同樣對比一下在2003年前後成立的淘寶和京東，雙方最終走向和價值發生了巨大變化，原因就是淘寶（阿里巴巴）始終堅持走平台道路，始終圍繞為商家和買家搭建連接，並向上游商家賦能，最終成為了一個超級生態平台——產業互聯網的雛形。

同樣的模型也發生在拼多多，通過運營海量用戶向上游廠商進行C2M賦能改造，向農品原產地進行定製賦能，表面上是2C（對消費者）的生意，實際上在做2B（對企業商家）的生意。

說回到視頻網站，繼續以愛奇藝為例，其第三季度會員收入已經超過了50%（第二季度是超過40%），會員服務收入為37億元，廣告收入為21億元，這兩個項目是字核心的兩大收入，總計58億元。

為了便於舉例，我們就當是愛奇藝核心業務營收是60億元/季度，全年240億元。騰訊視頻和優酷等合起來乘以 3 倍，國內在線視頻業務總盤子在720億元左右。

而傳統的影視內容市場有多大盤子呢？

1、電影：

僅2019年7月01日-07月10日（上旬）10天時間裡,全國處理備案故事影片共95部，其中現代題材(49年以後)77部,佔81.05%，歷史題材(49年以前)18部,佔18.95%。

到08月01日-08月10日（上旬)10天時間裡，電影局處理備案故事影片更是達到124部，其中現代題材(49年以後)102部,佔82.26%，歷史題材(49年以前)22部,佔17.74%。

以100部均值，全年35個周期（去掉放假），每年備案製作的電影數量在3500部左右。

2、電視劇

廣電總局發布關於2019年7月全國拍攝製作電視劇備案公示的通知，通知提到：2019年7月，全國電視劇拍攝製作備案公示的劇目共74部、3030集。

2019年9月，全國電視劇拍攝製作備案公示的劇目共73部、2648集。

以每月70部最低值，全年備案製作的電視劇數量在840部左右以上，近30000集以上。

3、網路影視劇

僅僅2019年10月一個月內，在廣電總局「重點網路影視劇信息備案系統」中登記並符合重點網路原創視聽節目製作相關規定的片目就共有469部，其中，網路劇94部2404集，現實題材佔比為76%；網路電影357部，現實題材佔比為61%；網路動畫片18部574集。

以此推算，全年網路劇1000部，2500集；網路電影4000部；網路動畫片200部。

以上是直接的影視核心內容，外圍還有動畫片、紀錄片、衍生品、現場活動等等，目測應該以萬億計算吧。

如果我們將互聯網發展分為上下兩個半場，那麼可以說，上半場所依賴的技術能力主要為連接技術，推動了消費互聯網領域的蓬勃發展。而隨著技術能力迭代和資源要素相對價格的變動，下半場無疑將會成為以ABCD技術為核心的產業互聯網舞台。

如果說消費互聯網更多是和 C 端直接關聯的營銷、銷售、服務環節的互聯網化，那麼產業互聯網就是對銷供產全產業鏈的組織與組織之間的互聯網化。消費互聯網是直接to C，而產業互聯網則是對 B 賦能和加持的邏輯。

從創意、劇本、組織、劇組、製作、排期、發行、營銷、衍生、服務等等全產業鏈，當前都沒有進行哪怕初級的互聯網化，整個改造和升級的價值何以萬億！

愛奇藝、騰訊視頻、優酷，三個寡頭手持最先進的技術基礎，擁有上億的正版意識死忠用戶，守著萬億的影視產業，還死盯著消費互聯網裡的那點兒用戶摳錢，有意思嗎？

寄語：

盛景網聯董事長、盛景嘉成母基金創始合伙人彭志強認為，流量端和供給端都要碎片化，這個碎片化的標準有兩個：市場前十名的市場佔有率不超過20%，千億級總規模。

對比一下影視產業，不管是供給端的影視公司、編劇、導演、演員、製作等要素，還是流量端的用戶群，都是碎片化分配。所謂市場前十名不僅僅市佔率低，而且簡直是靠天吃飯，十分不穩定，新秀迭起，黑馬頻出，這種亂局才剛剛進化到互聯網時代初期階段，影視產業互聯網大有可為！
