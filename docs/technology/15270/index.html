華為仍將參與建設挪威5G網路，不過不涉及核心網路

2019-12-16 489條評論 2,177次閱讀 63人點贊

科技訊 12月16日消息，據路透社報道，跨國電信運營商Telenor周日重申，華為將與愛立信一起參與建設挪威的5G網路。至此，Telenor成為不顧美國壓力，繼續與華為合作的幾家公司之一。


不過，Telenor挪威業務負責人弗伯格(Petter-Boerre Furberg)上周五對路透表示，該公司將在為期4至5年的升級期間，在挪威的電信網路中逐步淘汰華為的網路組件。

Telenor負責通信業務的副總裁漢妮·克努森（Hanne Knudsen）對路透社表示，華為將繼續在該公司的基礎設施升級過程中發揮作用。在4G網路時代，Telenor與華為進行了長達10年的合作。

克努森在回答路透社書面提問時表示：「愛立信已被該公司引入作為新的5G無線接入網(RAN)供應商，但我們也將與華為合作，對現有4G網路進行維護，並在挪威等部分地區將4G網路升級至5G網路。」

她說：「華為已為4G無線接入網提供了硬體，但沒有為核心網路提供硬體。當他們將在部分地區參與5G網路建設時，也是提供無線接入網的硬體，而不是核心網路的硬體。」

克努森還表示，Telenor芬蘭子公司DNA也將華為列為5G無線接入網的幾家供應商之一。

無線接入網(RAN)將智能手機連接到移動網路，佔5G網路成本的大部分。但它不是5G網路的核心部分。

華為表示，歡迎5G領域的競爭，並期待繼續與Telenor合作。

Telenor正採用諾基亞和愛立信的設備建設其核心網路。上周，德國電信宣布選擇華為和諾基亞的設備建設其5G網路。（劉春）
