全球央行數字貨幣（CBDC）競爭已經開啟，2020年將是關鍵一年 | 火星號精選

免責聲明：本文旨在傳遞更多市場信息，不構成任何投資建議。文章僅代表作者觀點，不代表火星財經官方立場。

小編：記得關注哦

來源：52CBDC

​全球央行數字貨幣（CBDC）競爭已經開啟，2020年將是關鍵一年。

2019年12月18日，新華社發布了一篇名為《財經觀察：「央行數字貨幣」破繭還有多久》，在文章中提到，目前越來越多的央行開始重視數字貨幣的研發、應用和監管前景，更吸引不少央行主動加入到數字貨幣研發隊伍中。多方研究顯示，全球多家央行已在數字貨幣研發上取得進展，「央行數字貨幣」或將在不久的未來破繭而出。

​可以預料的是，在即將到來的2020年，會有越來越多的央行會關注央行數字貨幣，加上中國、法國等國央行數字貨幣的加快、Libra的落地。2020年可以被稱作央行數字貨幣元年。

目前有多個國家在加速央行數字貨幣的研發與落地：

中國：即將進行試點

此前有報道稱,由人民銀行牽頭,工、農、中、建四大國有商業銀行,中國移動、中國電信、中國聯通三大電信運營商共同參與的央行法定數字貨幣試點項目有望在深圳、蘇州等地落地。該試點項目由央行貨幣金銀局牽頭,數字貨幣研究所具體落實。

法國：明年一季度開啟央行數字貨幣測試

據The Block 12月4日消息，在法國審慎監管與決議局（ACPR）主持的周三會議上，法國中央銀行行長FrançoisVilleroy de Galhau表示，央行將很快開始測試數字貨幣，並「將在2020年第一季度末之前啟動項目」。此前消息，法國央行將重新組織監管體系，把數字貨幣納入監管。

瑞典：將與埃森哲合作推出央行數字貨幣

據路透社12月13日報道，儘管瑞士早些時候對中央銀行數字貨幣（CBDC）持消極態度，並聲稱這樣做弊大於利，但最近瑞典卻開始探索電子克朗的潛在優勢。

瑞典中央銀行Riksbank表示，它將與愛爾蘭專業服務公司埃森哲合作，為電子克朗的數字貨幣創建一個試驗平台，以增強其對數字貨幣潛力的了解。該銀行表示：「電子克朗試點項目的主要是加強銀行對克朗數字貨幣技術可能性的理解。」

歐盟：加速對央行數字貨幣的研究

據路透社報道，歐盟輪值主席國芬蘭已提出一份文件草案，建議歐洲央行和歐盟國家央行採取具體措施推進「央行數字貨幣」研究和發行。此前，法國央行已宣布將於2020年第一季度啟動數字貨幣項目，並敦促歐洲央行儘快發行數字貨幣。

歐洲央行行長拉加德表示，歐洲央行設立央行數字貨幣專門委員會，預計將在2020年中期獲取關於數字貨幣的結果。他還表示，歐洲央行的數字貨幣要保持領先優勢。

​歐洲中央銀行（ECB）最近的研究聲稱，正在開發保護用戶隱私的中央銀行數字貨幣（CBDC）付款系統。根據這份名為《探索央行數字貨幣的匿名性》的報告，歐洲央行系統建立了一個CBDC中匿名的概念驗證(PoC)，這是其正在進行的CBDC研究及其對公眾潛在利益的一部分。

伊朗：也表示應該推出一種央行數字貨幣代替美元

2019年12月20日，據美聯社報道，Rouhani周四在馬來西亞峰會上表示，美國已將經濟制裁作為對其他國家進行「霸權和霸凌的主要工具」，因此他提議創建一種加密貨幣以替代美元在穆斯林國家之間進行貿易。

除了這些主要國家以外，各個國際組織也開始對央行數字貨幣密切關注。

國際清算銀行：敦促各國央行採用數字貨幣

據Cryptopotato消息，國際清算銀行(BIS)總經理Agustin Carstens在最近的一次講話中說，敦促各國央行採用數字貨幣。他補充說，金融世界目前正在經歷一場革命，銀行作為值得信賴的機構，應該掌握控制權。但他只是支持專業數字貨幣，仍然不贊成比特幣和大型科技公司的其他項目，例如Facebook的Libra。

金磚國家組織：討論建立加密貨幣的可能性

據Cointelegraph援引俄羅斯媒體RBC報道稱，金磚五國（巴西、俄羅斯、印度、中國、南非）在近日舉行的金磚國家論壇上討論了為統一支付系統建立一種新加密貨幣的可能性。俄羅斯直接投資基金（RDIF）總幹事Dmitriev表示，金磚五國計劃在成員國之間開發單一支付交易系統，並討論了將來可以通過該系統以加密貨幣「BRICS」進行交易的可能性。其指出，金磚國家支付系統能夠刺激本國貨幣結算服務，並確保成員國之間的結算和投資穩定。

​而Facebook的Libra也在加快進度，據路透社消息，Libra協會成員Patrick Ellis今日表示，Libra項目對於在2020年如何推出或者在哪裡推出目前尚沒有清晰的戰略。儘管戰略並未敲定，但Libra將在2020年推出。
