Mac Pro/Pro Display XDR 12月10日開售:頂配超30萬

2019-12-08 135條評論 1,106次閱讀 64人點贊

近日，據外媒報道，蘋果已經通知部分客戶，將於12月10日開啟Mac Pro和Pro Display XDR兩款產品的銷售工作，發布近半年時間後，這款蘋果性能最強的台式電腦與顯示器終於要上架了！

價格上，Mac Pro售價5999美元起步（約合人民幣41000元），Pro Display XDR售價4999美元起步（約合人民幣34000元），顯示器支架999美元（約合人民幣7000元）。


換句話說，如果你看上了新的Mac Pro主機、Pro Display XDR顯示器，即便是最低配置，也需要10998美元，也就是7.6萬元人民幣，頂配價格可能會突破30萬元人民幣。

不過蘋果並未透露下訂單後兩款產品何時發貨。

配置上，低配版Mac Pro搭載了8核16線程的Intel至強W系列處理器（最高28核56線程），32GB內存（最高1.5TB），256GB SSD（最高2TBx2），Radeon Pro 580X顯卡（最高4塊Vega顯卡）。

Pro Display XDR顯示器採用了一塊32英寸的IPS LCD液晶面板，Oxide TFT工藝，解析度6016×3384，長寬比16:9，總計2040萬像素，密度為218PPI，刷新率最高60Hz。XDR的意思是「Extreme Dynamic Range」，也就是極限動態範圍，包括全屏穩定亮度1000hits、峰值亮度1600nits、SDR亮度500nits、對比度1000000:1、典型反射率1.65％、P3寬色域、10-bit色深(10.73億色)。


(原標題：蘋果通知Mac Pro/Pro Display XDR 12月10日開售：頂配超30萬)
