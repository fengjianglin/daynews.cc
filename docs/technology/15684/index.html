Facebook前瞻：初步禁令浮出水面，Facebook遇“大劫”？

2019-12-13 1,672條評論 3,234次閱讀 87人點贊

【股價變動】

12月12日美股收盤，Facebook收跌2.72%，報196.75，本周累計跌幅2.14%。

【公司動態】

1.Facebook承諾將斥資1.3億美元成立一個獨立委員會，負責審查該公司如何調整其審核規則，為其更好地監管平台內容提供長期支持。

2. 據外媒報道，美國聯邦貿易委員會（FTC）正在考慮以反壟斷為由尋求對Facebook頒布一項初步禁令。這項禁令很可能會尋求禁止Facebook執行一些公司政策，甚至有可能阻止Facebook推進整合旗下三大即時通信服務的計劃，以備聯邦貿易委員會隨後可能會試圖撤銷Facebook此前收購Instagram和WhatsApp的交易。

【前景展望】

，據華爾街日報報道，美國聯邦貿易委員會（FTC）正在考慮以反壟斷為由尋求對Facebook頒布一項初步禁令。FTC的重點在於Facebook如何整合應用程序以及Facebook與潛在競爭對手合作的政策上，是否涉及壟斷。除了其核心社交網路，Facebook的主要產品還包括Instagram、Messenger和WhatsApp。此前，Facebook此前已公開過整合三款消息服務APP的計劃，並從端到端對信息進行加密。檢察長William Barr等執法人員對Facebook的計劃表示關注。

Facebook周四承諾，將斥資1.3億美元成立一個獨立委員會，負責審查該公司如何調整其審核規則，為其更好地監管平台內容提供長期支持。這筆錢標誌著對一個尚不存在組織的重大投資，但它可能會為公司的一些最棘手的決定承擔責任。
