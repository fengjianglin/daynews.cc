“新零售”風口，蘇寧已經拔得頭籌，黃光裕是否可以搭上末班車

天天要聞
2019-12-14

最近幾年，網路科技快速發展下，伴隨著互聯網在行業內應用場景市場需求不斷的擴大，也是成就了一批身價上千億的互聯網大佬，其中尤為耀眼的是中國BAT這三家巨頭公司。這些互聯網大佬憑藉超前的思想意識站在了互聯網迅猛發展的這一風口，迅速成為了行業的佼佼者。其中，阿里巴巴的馬雲更是成為了中國互聯網的「弄潮兒」即便現在的馬雲已經卸任阿里巴巴董事會主席，但他給中國帶來的影響力是一直存在的。卸任後的馬雲最近更是在「演唱會」「樂團指揮家」領域玩出了新的境界，讓大家不禁感嘆有錢人是真的會「玩兒」。

馬雲在任阿里巴巴董事會主席的期間，也是預言過未來三個不可估量的潛在領域。其中就有「新零售」這一領域。但當大家都認為電商可以取代實體經濟的時候，馬雲又再次預言了「未來的十年電商會被淘汰，而取代電商的就是「新零售」。至此各大互聯網大佬商業觸手伸向「新零售。（小編覺得，「男人的嘴，說謊的鬼」。但是馬雲的嘴，卻不見得是「說謊的鬼」。）因為馬雲自己預測了下一個風口，自己也是提前布局先是收購了大潤發超市，緊接著推出了盒馬鮮生等項目。開啟了線上線下結合的銷售服務，在現在的新零售領域裡，做的也是風生水起。

雖然馬雲會預言但是沒預到，今年蘇寧憑藉年入賬6000億元率先拔得「新零售」頭籌。（小編詭異」第三人格「說：「其實也間接證明了，馬雲沒有「先知」超能力，不是從未來穿越過來的人，他是完全憑藉著自己的能力和智慧取得成功的。」）據媒體日前發布的《2019年中國民營企業500強名單》，蘇寧控股集團營收排名第三，新零售行業排名第一，第二則是國美電器，但是相比蘇寧，國美卻差了2675億。蘇寧的成功雖是意料之外，但也在情理之中。從蘇寧整體可以看出，現在的蘇寧商業觸角已經不局限於家電，近幾年更是布局各大領域，列如超市、母嬰、百貨連鎖。智能傢具，汽車等大型物品也在蘇寧「涉獵名單」。也是在張近東這樣大手筆的操作下，使得整個蘇寧得到全面迅速的發展。

正因為依託科技時代高速發展的背景，利用大數據和人工智慧，蘇寧將線上線下完美結合。實現了「新零售」這一概念。也是通過這種「新零售」模式，在19年全球500強企業中，排名迅速提升94位，由18年的427名上升到333名。通過這些數據，可以感受到「新零售」時代風口即將來臨，而已經入局的企業也將會面臨著新一輪的廝殺。

這裡不得不提，新零售排名第二的國美電器，想必大家也應該或多或少的對它有所了解。也是有消息稱國美創始人，黃光裕會在2021年2月16日出獄。曾經黃光裕的妻子杜鵑也是對外表示說：「等黃光裕出來，我會給他一個更好的國美。」

提起黃光裕，這是中國商業界歷史上無法避開的名字，他曾三次登頂中國首富，風頭不亞於現在的馬雲、馬化騰。當時也是創業青年備受追捧的商業明星，因其經營手段雷厲風行，擅長打「價格戰」。被許多人稱為「價格屠夫」。國美在黃光裕的帶領下，迅速收購同行擴大經營場所。一句「商者無域，相融共生」更是成為了無數青年創業者堅信的信條。如果2008年的黃光裕沒有入獄，現在的蘇寧極有可能已經被國美收購。在入獄期間有人爆料黃光裕提前出獄，國美的股票都能受到「效應」立馬暴漲。，可見當時的黃光裕就是一個商界的傳奇人物。

此次蘇寧「新零售」拔得頭籌，我們不難看出風口已經來臨，但是不知道2021年的黃光裕出獄後，是否還可以搭上「新零售」這一風口的末班車。現在看國美雖然失去了十年的時間，但是現在的國美還是很有實力的企業。所以，黃光裕出獄後極有可能會在「新零售」的風口下，重振旗鼓去收復失地。歸來後的黃光裕，還能再次迎來屬於他的時代嗎？這個誰也不無法預知，但有一點可以肯定，以黃光裕的性格，必然不會就此沉默，是肯定會在行業里再掀風雲的。
