新一批國產遊戲版號下發:42款遊戲獲批 騰訊網易在列

2019-12-10 651條評論 2,222次閱讀 215人點贊

新京報訊（記者 陸一夫）12月10日，原國家新聞出版廣電總局官方顯示，新一批國產遊戲版號名單更新，共42款遊戲獲批，審批時間為12月9日。其中騰訊的《我叫MT自走棋》、的《有殺氣童話2》、趣丸網路的《長安十二時辰》、中手游的《魔性酷跑》、創夢天地的《鎮魂街：武神軀》等遊戲獲得版號。
