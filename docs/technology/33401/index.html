58歲前央視名嘴，被女兒斥家暴出軌棄養，今東山再起梳大背頭演講

2019-12-24 1,696條評論 3,297次閱讀 1923人點贊

12月22日，前央視主持人方宏進分享了一張發表演講的照片，配文稱他正在參加2019中國通證數字經濟峰會。

方宏進頭髮花白一身黑衣，梳著大背頭總裁范兒十足，在演講台上侃侃而談，整體狀態來說非常的不錯。

另外從名牌來看，方宏進是在前排就坐，可見在這場峰會中還是很受重視的。有網友爆料，方宏進目前的身份是「數碼動力（香港）有限公司創始人、香港區塊鏈協會共同主席」。

人生幾次起落，方宏進在商場跌倒，58歲又在商海出發，東山再起，其不屈精神讓人敬佩。

央視元老，《焦點訪談》創辦人之一

前幾天，方宏進還在個人社交平台曬出了一張與敬一丹的合照，引發一波兒回憶殺。

方宏進是物理系畢業，因為母親是報社記者，所以對新聞從小耳濡目染，十分敏感。

30歲參與北北京衛視製作節目，31歲進入中央電視台《觀察思考》，後來主持《東方時空》，可以說是白岩松和水均益的前輩。

參與創辦和長期主持的《焦點訪談》是一代人的記憶，在央視「三峽截流」、「香港回歸」、「澳門回歸」等大型直播報道中也少不了他的身影。

五次獲得中國新聞獎一等獎的方宏進，當時有多紅？

央視著名主持人白岩松曾在《痛並快樂著》中透露這樣一段往事，在一年春節內部的聯歡會上，有一個小品里有這樣的台詞，是幾十年後方宏進的孩子說給崔永元孩子聽的：

“哼，當初我爸出名的時候，你爸還嗑瓜子練嘴皮了呢!”

跳槽離開，下海經商虧損1600萬

可是不知道什麼原因，2003年，42歲的方宏進跳槽離開了央視，加入了剛剛創辦不久的東方衛視，一手扶持了東方衛視的成長。

逐漸轉戰幕後的方宏進幾乎消失在了人們的視線，再起引起人們關注是因為他下海經商，與人發生的經濟糾紛案件，甚至有人說他是「詐騙」，方宏進還曾因「詐騙罪」被警局逮捕。

好在最後無罪釋放，他也發表了聲明，

方宏進聲明全文如下：我在創業過程中得到若干朋友在各方面的鼎力支持，他們至今仍然對我信任有加，對此我表示由衷的感謝，並一定會以實際行動不辜負他們的理解、支持和期待。

據悉，這次糾紛是因為方宏進投資1600萬拍攝了情景劇《候機大廳》，原擬在央視等電視台播出，後因故播出計劃流產，他虧的血本無歸。

被女兒斥家暴出軌棄養

福無雙至禍不單行！方宏進與妻子於紅偉恩愛20年，還有一個17歲的女兒，一家三口也曾被受羨慕，可是自從方宏進下海經商後，夫妻感情產生了裂痕。

2010年，於紅偉起訴離婚，隨後爆料方宏進2007年離家出走，杳無音信，才導致兩人離婚。

更加讓人意外的是，2011年12月，一位自稱是方宏進女兒方貞的人在微博上控訴父親方宏進家暴出軌，還不給生活費、贍養費。

對此方宏進表示：

「這個微博還有幾個馬甲號。是不是我女兒本人值得推敲。我不會和她理論什麼。」

至於撫養費一事，方宏進則坦言，他是凈身出戶，留了很多財產給母女倆，

「我女兒的母親沒有任何收入。如果我不給錢，她們這4年來如何住豪華公寓、開轎車。我女兒如何讀貴族學校？」

2012年，方宏進和於紅偉正式離婚，女兒歸前妻撫養，這樣鬧劇才宣告結束。

現在方貞早已長成一個才貌兼備的美人，曾代表上海賽區參加央視「希望之星」英語比賽獲獎，喜歡唱歌且頗有音樂天賦。

58歲再出發，鬥志昂揚

離婚事件後，方宏進消失在了人們的視線中，是整理心情，也會積累能量。

如今他58歲再戰商海，鬥志昂揚，從哪裡跌倒就從哪裡站起來，緊跟時代潮流的他，不知道會再創什麼樣的輝煌呢？
