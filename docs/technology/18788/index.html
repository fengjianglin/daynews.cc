超過200輛換電重卡將用來運砂石！首批30輛交付！誰家車這麼牛？

2019-12-17 465條評論 3,818次閱讀 258人點贊

隨著「打贏藍天保衛戰」政策的實施，推進大宗貨物運輸「公轉鐵」，進一步加快物流運輸結構調整，構築高效、綠色、環保的物流運輸體系成為運輸行業發展的趨勢。

​

針對北京砂石公鐵聯運市場，為實現礦山開採、加工、運輸、廢料回收等所有環節的綠色能源（電動）化，建材綠色供應鏈「綠色聯盟」於12月12日在北京正式成立。

​

融和電科以「綠色聯盟」戰略合作方身份，聯合新能源生態圈合作夥伴特百佳、華菱星馬、玖行能源共同研發的全球首款智能換電重卡，為北京密雲礦山砂石骨料公路運輸提供「車輛運輸、能源補給、運營管理、金融服務」一體化綠色物流解決方案，助力北京公鐵綠鏈公司實現整個砂石骨料開採、運輸、廢料回收全綠色產業鏈的打造。

​

在活動當天的交車儀式環節，30台換電重卡和玖行能源第四代最新緊湊型換電站順利完成交付，標誌著融和電科在北京首個智能換電重卡市場化示範項目順利落地。

​

融和電科此次交付的換電重卡為6*4換電型純電動牽引車，搭載360kW超強馬力電機，續航里程可達200km，充分滿足砂石短途倒運的物流特點。礦山內配置第四代全自動無人值守換電站，最快可實現3分鐘「極速」換電，現階段可徹底解決電動重卡的「里程焦慮」，極大地提升了車輛運營效率。

​

為確保運營可靠性，融和電科所投放車型在多地的港口、鋼廠等複雜實際場景和專業實驗場地經過了高強度耐久性能測試，累計行駛里程超過一百萬公里。

​

2020年3月份，根據計劃將投放超過200輛換電重卡在密雲礦山穿梭，按照每輛車每天400公里行駛里程計算，預計每年可減少約28000噸二氧化碳的排放與2500噸污染物排放，經濟性方面電費花費較之燃油車可節省超過4000萬元。

此次30台換電重卡的首次市場化投放，是融和電科「智能換電重卡」創新項目的一個里程碑。在重卡向新能源轉型的大背景下，融和電科將以北京項目為區域化切入點，在京、津、冀及周邊地區「2+26」城市進行複製推廣，開啟綠色清潔運輸新模式，攜手共築美好藍天夢！

（來源：融和電科）
