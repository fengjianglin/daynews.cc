特斯拉電動汽車業務已實現穩定盈利，其主要業務內容變更

2019-12-16 303條評論 1,422次閱讀 375人點贊

2019年10月，特斯拉公布了其第三季度的財報，披露了其破紀錄的交付量，引起了其股票市值的飆升。作為一個非常關鍵的盈利指標，特斯拉的毛利潤遠超過投資者的預期，而這主要來源於特斯拉全新電動汽車車型的成本削減和產能提升。該消息紕漏一個小時以後，特斯拉的股價即上漲了近17%，交出了一個每股1.86美元收益的答卷，遠超過早期分析師預計的每股損失42美分。

特斯拉第三季度的財報也許預示著特斯拉至2003年成立以來，終於實現了穩定的盈利，這也許是來源於特斯拉Model 3的大好前景。最新的消息表明，隨著特斯拉電動汽車生產終於走上正軌，其盈利能力也逐步穩定，馬斯克開始將特斯拉的主要業務轉向能源業務，包含太陽能和儲能兩大模塊。

特斯拉 model 3

在十一月的電話會議中，特斯拉的CEO馬斯克表示公司最近兩年已經轉移了大量的資源，隨著公司的能源業務成為特斯拉的主要經營業務，很難想像特斯拉會進入怎樣的一個瘋狂擴張期。馬斯克從來不是一個喜歡隱藏自己野心的人，他認為特斯拉的能源業務會變成和特斯拉的電動汽車業務一樣的規模，其中太陽能業務將會按照最快的百分比增長，而儲能業務將會緊隨其後。在馬斯克給員工的一封內部郵件中，其列出了年底的兩項優先事項，交付完成所有的電動汽車和大大提高太陽能的部署速度。

特斯拉的太陽能儲能業務

特斯拉將太陽能業務與儲能業務相結合，對家庭用戶和企業用戶提供能源服務。以家庭用戶為例，特斯拉提供太陽能光伏屋頂，將普通的屋頂替換為能夠將太陽能轉換為電能的光伏板，並將這些電能存儲起來，作為家庭用電和電動汽車用電的來源。該項目也可以在電費較低時，將電能儲存起來，而當電價升高時，放出電能，以實現梯度用電，移峰填谷的目的。

特斯拉光伏儲能業務

在之前的文章中，（電動汽車骯髒的秘密！不只是有條件的「零排放」，還涉及童工）我們曾提到新能源電動汽車的零排放其實是有條件的零排放，即電動汽車若要實現零排放，得從其根源上實現零排放，這其中就包括電的來源，如果還是以火電等進行充電，那麼很大程度上來說，不是真正的零排放。而特斯拉的該業務，將太陽能儲存起來，再充給電動汽車充電，可以說離電動汽車真正實現零排放又近了一步。

光伏屋頂的綠色電能

之前另外一篇文章中（在電動汽車車頂，加太陽能光伏板，增加續航能否行得通），我們也提出了「利用太陽能光伏板增加電動汽車續航是否靠譜」的問題，當時給出的結論是，因為電動汽車車頂的空間有限，且大多數時間車輛停在沒有光照的區域，所以增加續航的效果不是那麼理想。而特斯拉的該方案，不是將光伏板鋪在車頂，而是鋪在樓頂，樓頂的面積足夠大，光照時間充足，每天可儲存的電量可達到50度以上，夜間可用於家庭使用和電動汽車充電，再配上特斯拉本身超長的續航能力，一日一充基本可以解決用戶焦慮。

特斯拉家庭光伏屋頂可為電動汽車充電

特斯拉自家的電動汽車業務和能源業務互相補充，讓特斯拉電動汽車真正朝著零排放前進，同時解決用戶的焦慮問題，但是在中國，我還是想要提出兩個疑問：

①北上廣深幾百幾千萬買的房子真的有樓頂嗎？-沒有樓頂怎麼鋪光伏板？

②農村的我屋頂倒是有，可是買不起特斯拉的車，更買不起這套光伏儲能系統啊，怎麼辦？
