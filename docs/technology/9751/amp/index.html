中國半導體自給率嚴重不足，如何突破重圍，走出自主創新之路

天天要聞
2019-12-10
  • 中國是世界上最大的半導體進口國和消費國，受各種條件限制，目前國內半導體自給率只有約15%，一直希望半導體產業能夠實現自給自足，貿易摩擦的關鍵點也是在於半導體技術，不掌握核心技術就容易導致被動；
  • 雖然國家給以高度重視和支持，但想要有所突破還是困難重重，究竟哪些方面存在不足，如何才能突破發展瓶頸呢？

一、全球15大半導體公司無國內企業

  • 2019年上半年全球15大半導體公司全部為歐美、日韓和台灣公司，中國大陸沒有公司入圍；

  • 存儲以三星、SK海力士、美光為代表；
  • 邏輯電路以Intel、博通、高通為代表；
  • 晶圓代工以台積電為代表；

二、IC設計有望突破，但華為海思孤掌難鳴

  • 按照營收排名，華為海思目前在晶元設計領域排名第5，但除了華為海思，沒有其他更有競爭力的國內知名企業；

  • 總體來看，設計行業的核心技術仍然在美國，2018年美國佔了全球IC設計份額的53%，中國佔比為11%；

三、IC設計面臨兩大制約因素

  • EDA設計軟體方面，美國的三家公司（Synopsys、Cadence、Mentor）壟斷了國內96%以上的市場份額；
  • 底層架構方面，以intel、AMD為首的X86架構，在個人PC端占絕對主導；

  • 以IBM、ARM為首的精簡指令集ARM/MIPS/Power，在移動設備和物聯網設備晶元中占絕對主導；

四、晶圓代工被海外壟斷

  • 晶圓代工方面如果沒有全球領先的技術，成本投入將會是非常巨大的；

  • 台積電佔全球晶圓代工的市場份額超過50%，其次為三星、格芯，三家市場份額接近80%；

  • 國內僅有中芯國際一家，市場佔比約5%；

五、刻蝕機、光刻機被歐美日壟斷

  • LAM、TEL、AMAT三家佔據了刻蝕機91%的市場；

  • ASML是全球光刻機的龍頭，佔全球75%的市場份額；

  • 再加上尼康、佳能，三家佔據了93%的市場；

六、國內設備企業規模小，技術差距較大

  • 國內排名第一的設備企業北方華創，2018年營業收入為約4.75億美元，距離排名第一的公司140億美元的營收差距很大，技術節點多數都還比較落後，高端光刻機等核心設備仍依賴進口；

七、IC封測門檻相對較低，本土廠商逐漸崛起

  • 目前國內已有3家企業進入封測產業全球前10，分別是長電科技（第3）、華天科技（第6）、通富微電（第7）；

  • 由於封測產業對規模化要求較高，相對於設計與代工，國內封測企業目前排名相對靠前，主要採用的方式是加大研發投入以及併購整合；

結語

  • 雖然國內半導體產業仍然落後國際水平，但相信在國家政策的大力支持和各方投資下，國內雲計算、物聯網、大數據等產業生態系統的高速發展下，以及業內人士的共同努力下，2020年，按照《中國製造2025》的規劃，半導體自給率達到40%，這個目標一定能夠實現。
