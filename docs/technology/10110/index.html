賈躍亭在聽證會否認隱匿資產，考慮回國量產FF91

2019-12-07 247條評論 1,636次閱讀 267人點贊

澎湃新聞記者 陳宇曦

賈躍亭個人破產重組案件有了最新進展。

賈躍亭債務處理小組微博在12月6日晚間發布消息稱，美國東部時間12月6日上午10時，賈躍亭破產重組聽證會在美國特拉華州舉行，該會議由美國司法部下設的債務信託受託人辦公室主持，賈躍亭將出席並依照法律程序如實解答所有債權人提出的問題。

這場聽證會開了3個多小時。澎湃新聞記者從相關債權人處獲悉，賈躍亭針對債權人提出的個人資產問題、FF發展前景問題，以及FF全球合伙人計劃等問題做出了解答。

據悉，賈躍亭現場表示，聽取了債權人建議，將認真考慮FF 91在中美兩地大規模量產，這也是FF首次提出FF 91的回國大規模量產計劃。


自宣布在美申請個人破產重組計劃以來，賈躍亭個人是否隱匿資產、FF的發展前景是否樂觀，以及FF全球合伙人計劃的合理性等等，成為備受關注的三大焦點。

賈躍亭在聽證會上表示，希望所有債權人能夠支持其個人破產重組方案，支持FF全球合伙人計劃，助力FF儘快實現B輪融資，從而同時實現所有債權人利益的最大化。

賈躍亭回應隱匿資產質疑

針對隱藏資產的質疑，賈躍亭當庭宣誓後強調其唯一資產就是FF的股權，除了之前已經分給FF全球合伙人的股權之外，所有的股權及其收益都將在破產重組方案通過後轉入債權人信託，其個人也將不再持有任何FF股權。

關於賈躍亭曾經控制的OceanView公司擁有的房產以及美國另一家電動車初創企業Lucid的股權，賈躍亭表示，OceanView的確曾經是他100%擁有的公司，並且分別在2014年和2015年在洛杉磯Palos Verdes市購買過幾處房產，但這些房產此前均用於FF高管的臨時住宿場所和周末開會所用，在資金鏈陷入危機之後，被反覆抵押貸款並輸血FF以支持公司運營，從資產價值看，這些房產在被反覆質押後目前幾乎為零，該公司也已經於2017年被第三方接手，並不屬於賈躍亭的資產，只是其中一套房產租給了賈躍亭居住。

關於Lucid的股權，賈躍亭稱，Lucid是樂視控股以公司名義投資的公司，他個人此前就沒有任何處理權，而且目前Lucid股權也已經轉讓給了第三方公司，目前他個人並不持有該公司任何股權。

賈躍亭首次提出FF 91或將回國量產

除了隱匿資產的質疑之外，債權人最為擔心的是FF公司的價值及未來發展。

在聽證會上，賈躍亭稱，FF是目前行業公認的在智能互聯網電動車領域擁有世界領先產品和技術的公司，目前，來自中東、亞太和美國的多家重量級投資方都表達了投資FF的興趣，因此希望全體債權人能夠支持其個人破產重組計劃，推動FF的B輪融資。

此前，賈躍亭將股權分給FF合伙人的做法遭到部分債權人質疑。

對此，賈躍亭在聽證會上回應稱，在與恆大和解以及建立FF員工期權池之後，他個人所持FF的股權比例在40%左右。在制定全球合伙人計劃以及提交個人破產重組申請之前，已經對他個人所持的這部分40%左右的FF股權及其收益權的分配做了細緻測算：即將轉入債權人信託的FF股權擁有特殊股份收益權，該部分股權的總收益額遠超過FF全球合伙人將獲得的收益，預計佔到賈躍亭原有所持FF股權收益額的七八成左右。與此同時，賈躍亭強調，全球合伙人計劃是保障FF繼續健康前行的關鍵，也是最大限度確保債權人利益的重要舉措之一。

賈躍亭還在聽證會上稱，儘管這兩年因為資金問題減緩了FF 91的量產進程，但是，FF核心團隊不斷對FF 91進行了產品技術迭代和進化，確保了FF 91至今在全球智能汽車領域領先。

據現場債權人透露，賈躍亭首次明確提出FF 91有可能實現國內量產，稱FF 91如果實現中美兩地大規模量產，將有望實現成本大幅下降，這在全球最大的汽車市場中國將有極大的市場潛力。

據稱，促使賈躍亭作出這一改變的，應該是11月25日債權人大會上眾多債權人提出了FF 91在中國量產的建議。
