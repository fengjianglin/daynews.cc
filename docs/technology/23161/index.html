自動駕駛還遠嗎？核心感測器都被安森美半導體布局齊全了

2019-12-18 1,826條評論 4,601次閱讀 1999人點贊

微訪談：安森美半導體智能感知部市場總監郗蘊俠

採訪背景：自動駕駛是最具影響力的技術之一。特斯拉（Tesla）、英偉達（NVIDIA）、英特爾（Intel）旗下的Mobileye等企業或許是大眾首先想到的與自動駕駛相關的明星企業。但是，這個領域還有一家值得關注的企業，為自動駕駛提供零部件，它正通過積極的布局和穩健的成長，成為自動駕駛智能感知領域的領導者，它也是市場上唯一一家能夠提供激光雷達、毫米波雷達，攝像頭和超聲波雷達四種核心感測器晶元的廠商。它就是安森美半導體（ON Semiconductor）。

今年12月19日將在深圳舉辦的『第二十八屆「微言大義」研討會：激光雷達技術及應用』，安森美半導體智能感知部市場總監郗蘊俠博士將出席會議，並為大家做題目為《通向量產之路的激光雷達產品及解決方案》的演講。在此之前，我們專訪了郗蘊俠博士，通過本次採訪，我們對安森美半導體在自動駕駛領域的思考和布局有了進一步的了解。以下是採訪內容的整理。

安森美半導體智能感知部市場總監郗蘊俠

麥姆斯諮詢：非常感謝郗博士給予的此次採訪機會。請您向大家介紹下您在安森美半導體的主要工作職責，謝謝！

郗蘊俠：我現在就職於安森美半導體，負責大中華地區智能感知產品市場開發工作，包括攝像頭CMOS圖像感測器、激光雷達核心器件和毫米波雷達晶元在汽車ADAS（高級駕駛輔助系統）和自動駕駛領域的拓展工作。

麥姆斯諮詢：經過這幾年的布局，安森美半導體已經成為自動駕駛應用的一站式感測器供應商。請您談談安森美半導體的汽車CMOS圖像感測器、雷達晶元和激光雷達核心器件業務情況。

郗蘊俠：安森美半導體關注與ADAS和自動駕駛有關的所有應用，並提供汽車全線的智能感知器件，包括超聲波雷達、攝像頭、77GHz毫米波雷達和激光雷達的核心器件。其中安森美半導體的超聲波雷達SoC和CMOS圖像感測器產品在全球車載市場佔有率位居第一。即將推出的符合車規的激光雷達核心器件（光電探測器）為未來車規化激光雷達量產做好充分準備，將成為全球知名激光雷達系統客戶的首選。第一款毫米波雷達晶元將在2020年首發，為適應L2以上自動駕駛系統而設計。安森美半導體是現今全球唯一一家可以提供以上全部感測器晶元的半導體公司，以後可延伸業務，提供物端感測器融合方案。

麥姆斯諮詢：車用攝像頭、毫米波雷達、激光雷達的技術成熟度發展不同，相應地，其所用的核心晶元技術成熟度也處於不同階段。可以介紹下安森美半導體在上述三種感測器晶元方面的研發戰略嗎？

郗蘊俠：安森美半導體是汽車攝像頭CMOS圖像感測器的領導者，在ADAS和自動駕駛CMOS圖像感測器市場佔80%的份額，具有領先業界的最佳的汽車像素技術、微光性能、高動態範圍、多種色彩濾波陣列(CFA)、解決LED閃爍問題，有非常完善的質量監督和控制體系，符合車規安全認證的產品，並同時實現功能性安全和業界首創的網路安全，可擴展平台等加快了設計新一代ADAS和自動駕駛系統，降低整體開發成本。

安森美半導體於2017年收購IBM毫米波雷達技術，主要開發針對L2以上自動駕駛應用所需的77GHz毫米波雷達晶元，可以滿足長距離（大於200米），角解析度高，設計獨特的片上MIMO（多輸入多輸出）方式用以優化系統架構，可實現長短距離切換，為整個ECU（電子控制單元）節約成本。

安森美半導體於2018年收購專門做SiPM（硅光電倍增管）和SPAD（單光子雪崩二極體）探測器的公司——SensL，成為擁有此技術的市場領袖，這些技術是實現激光雷達系統的核心器件。這兩種光電探測器類型均基於蓋革模式。因此，安森美半導體的目標是實現符合車規、高增益、低成本、尺寸緊湊的光電探測器，極其適用於解決汽車長距離激光雷達的微光探測挑戰，提供業界最高靈敏度、最佳一致性和低雜訊的產品。

安森美半導體在以上三種感測器晶元整體研發戰略就是持續投入，持續創新，持續開發新產品適應市場的需求。

麥姆斯諮詢：請談談您對應用於激光雷達的核心光電探測器的理解，探測器的技術成熟度如何？哪些核心指標是用戶最為關心的？

郗蘊俠：激光雷達探測器因為有SiPM和SPAD技術的引入，相比傳統的APD（雪崩二極體）性能指標有了巨大的提升，因此成熟度越來越高。現在市場就是看哪一家可以同時研製出符合車規的產品，在激光雷達產品量產落地時保證質量和可靠性。

汽車激光雷達客戶最為關心的指標包括PDE(Photon detection efficiency，光子探測效率)、DCR(Dark Count Rate，暗計數率)、Crosstalk（串擾）、Gain（增益）、供電電壓、一致性等。安森美半導體的SiPM和SPAD感測器一直在不斷提高PDE，同時降低探測器雜訊。由於其增益為傳統APD的一萬倍，具有遠距離探測的優勢。供電電壓為APD的五分之一，大大降低了功耗。另外，最佳的「一致性」特性是激光雷達產品量產的重要指標之一。

麥姆斯諮詢：去年5月，安森美半導體收購了愛爾蘭SiPM（硅光電倍增管）和SPAD（單光子雪崩二極體）供應商SensL。SiPM和SPAD被視為激光雷達的核心光電探測器，其技術的獨特優勢有望取代現有主要光電探測器——APD（雪崩二極體）的地位。請談談收購SensL以來，安森美半導體已經為激光雷達推出了哪些新款SiPM和SPAD產品？市場反應如何？

郗蘊俠：現有客戶都在使用安森美半導體SiPM的RA或RB系列開發激光雷達產品。安森美半導體即將發布SiPM的R系列新產品，這是我們結合了較高PDE和較低雜訊方面的開發成果。這些光電探測器的緊湊封裝使客戶能靈活地布局多個元器件以配合其應用，且符合AEC-Q102汽車認證。我們還在開發多種單晶元線陣產品，將結合最新性能，採用經證實的格式用於廣泛的激光雷達應用。此外，安森美半導體推出首個SPAD面陣陣列：Pandion晶元。這是採用捲簾快門讀出的400 x 100像素SPAD陣列。市場對新的車規級R系列新產品以及面陣Pandion晶元非常感興趣，很多國內外知名激光雷達客戶已經用這些探測器進行系統開發。

麥姆斯諮詢：安森美半導體作為汽車CMOS圖像感測器領域的霸主，已經成功打入百度Apollo等自動駕駛平台。安森美半導體擅於與汽車行業最大且最具影響力的夥伴合作，構建自動駕駛生態系統。那麼，安森美半導體的SiPM和SPAD產品會選擇怎樣的合作平台？目前進展如何？另外可以透露已經服務的典型激光雷達客戶嗎？

郗蘊俠：我們會像CMOS圖像感測器一樣構建完整的生態系統，和激光雷達的各種部件供應商合作，包括光學系統、激光器、讀出電路、SoC（系統級晶元）等，做成參考模組提供給終端客戶。

我們和國內外知名的激光雷達客戶都有合作，暫時不方便透露細節。

麥姆斯諮詢：可以談談您在開拓國內SiPM和SPAD市場過程中，遇到的主要競爭對手有哪些企業嗎？您如何評價安森美半導體與競爭對手之間的差異和產品優勢？

郗蘊俠：安森美半導體是SiPM和SPAD技術的市場領袖，與已有的APD產品相比，具有高靈敏度（2000倍），高增益（1萬倍），供電電壓低（~32V），及最佳一致性等優勢。另一個明顯優勢就是安森美半導體從一開始就考慮汽車認證去設計光電探測器及其封裝，並採用CMOS工藝。我們將很快推出首款符合AEC-Q102汽車標準的光電探測器，為激光雷達量產化鋪平道路。除了光電探測器以外，安森美半導體還提供激光系統中符合車規的驅動晶元、電源晶元、放大器和讀出晶元等整體系統硬體解決方案。安森美半導體為客戶提供快速全面的服務，通過廣泛的現場應用工程師網路、激光雷達相關的應用文件和視頻庫、產品演示系統、經驗證的激光雷達模型的模擬數據等，有助於客戶掌握新技術並快速將產品落地。

麥姆斯諮詢：目前安森美半導體推出的SiPM和SPAD主要基於CMOS工藝，在硅晶圓上實現。是否有計劃推出基於III-V族化合半導體工藝的短波紅外（SWIR）光電探測器？

郗蘊俠：暫時沒有計劃。

麥姆斯諮詢：請暢談下你們對激光雷達技術發展趨勢的理解。

郗蘊俠：因為機械式激光雷達的質量和可靠性不能完全符合車規標準，因此激光雷達真正大批量生產應該是走向固態激光雷達。在固態激光雷達中，MEMS激光雷達擁有最快落地的可能性。主要原因：MEMS微振鏡幫助激光雷達擺脫了笨重的馬達、多稜鏡等機械運動裝置，毫米級尺寸的MEMS微振鏡大大減少了激光雷達的尺寸，集成度高；其次MEMS微振鏡的引入可以減少激光器和探測器數量，極大地降低成本，成本方面具有優勢；最後，MEMS微振鏡在投影顯示領域商用化應用多年，有良好的市場基礎，可以吸取以往的經驗。

麥姆斯諮詢：針對車內應用，安森美半導體有哪些感測器解決方案？如駕駛員監控、手勢識別……

郗蘊俠：艙內監控是增長率非常高的一個應用領域，艙內監控包含駕駛員監控和乘客監控。相機要捕捉司機眼睛和頭部動作，全局快門圖像感測器是駕駛員監控最好的方式，可以捕捉快速運動，無拖影。乘客監控則採用RGB-IR圖像感測器，以達到白天和夜晚同時可以看到清晰圖像的效果。安森美半導體提供全面的感測器產品線：從標清至2MP，CMOS圖像感測器具有最佳的紅外(IR)響應，可以在紅外條件下清晰地看到駕駛員眼睛，具有最佳的全局快門效率，明亮環境下的圖像受控。另外強大的軟體夥伴提供針對安森美半導體CMOS圖像感測器的應用軟體。駕駛員監控主要推薦的全局快門圖像感測器包括AR0144AT、AR0135AT和AR0234AT。而乘客監控則推薦採用RGB-IR圖像感測器AR0239AT。

麥姆斯諮詢：2019年，全球汽車銷量行業表現低迷，對安森美半導體的汽車板塊業務有怎樣的衝擊？也請談談你們的應對方案。

郗蘊俠：雖然全球汽車銷量行業表現低迷，但是汽車的電氣化和智能化一直在不斷發展，需要更多的和先進的半導體元器件。比如隨汽車自動駕駛級別的提高，需要越來越多的感知器件，每輛汽車的攝像頭數量從1~5個增加到10~15個，甚至更多，而且功能要根據新的要求而不斷提升。安森美半導體是汽車半導體零部件領軍企業，會繼續在汽車半導體零部件領域投資，尤其在新技術和新應用投入大量研發，設計出更加適合於汽車行業發展的新產品，提供世界一流的新方案。
