2019創業死亡名單：湖北陣亡6家，3家來自光谷

2019-12-19 28條評論 4,210次閱讀 993人點贊

2019行將落幕。

這一年中，互聯網發展依舊潮起潮落，一邊是拼多多和位元組跳動突出重圍，巨頭阿里港股上市；另一邊是上市公司暴風倒下，創業公司淘集集也失血而亡。

寒冬年年有，今年特別冷。

數據顯示，2019年倒閉的創業公司之中，有55%的企業直到最後一刻也未能獲得投資。「抱歉，公司融資失敗，已倒閉」一時成為創投圈的咒語。

回顧即將過去的2019，300多家企業在經歷裁員、欠款、跑路後，最終倒下。有的曾經死得轟轟烈烈，而有的則消失得悄然無息。

2019創業死亡名單

千里馬、獨角獸相繼陣亡 金融、電子商務成「死亡賽道」

回顧2019年，似乎未能出現一個創投兩熱的風口，反倒一些以往的風口正處在泡沫消退階段，成為典型的「死亡賽道」。

據IT桔子發布的《死亡公司公墓——新經濟死亡公司資料庫》統計，截止 2019 年12月16日，共有336家新經濟公司宣告「死亡」。「死神」活躍在各個領域，社交電商、生鮮、電子煙、教育等無一倖免。

2019新經濟死亡賽道及公司數量（數據來源：IT桔子）

從行業分布上看，2019年死亡公司排名前五的行業，分別是金融（62家）、電子商務（38家）、企業服務（34家）、汽車交通（32家）、本地生活（30家）。緊隨其後的行業為教育及文娛傳媒，均達到20家以上，醫療健康行業死亡公司數量也達到了19家。

從2018年開始，P2P公司便開始集中爆雷，因此金融成了今年最慘烈的創業賽道。2019年關閉的公司中，就有將近18%來自金融行業。其中，獲得融資金額最高的10家死亡公司中，就有3家是金融行業公司，加起來融資超40億。可以說，金融行業當之無愧是最燒錢的行業。互聯網金融行業不斷暴雷導致監管迫近，行業大洗牌成為共識，2019年無論是成交量還是平台數量都在逐月減少。金融行業利潤大，但承擔的同樣是高風險，一旦遇上暴雷，投資者可能血本無歸。未來，金融行業仍然面臨生死存活問題，洗牌也將繼續。

2019死亡公司燒錢榜前十（數據來源：IT桔子）

除了金融行業之外，電子商務領域也是傷亡慘重。事實上，隨著電商「紅利」時代的結束，沒有後台強力支持的垂直電商在各大電商巨頭的夾縫中根本難以生存，年末的生鮮電商倒閉潮就是一個很好的註解。自2013年以來，每日優鮮、天貓生鮮、盒馬鮮生、京東到家數十家生鮮電商一個接一個地成立，行業競爭極為激烈。在這樣的背景下，頭部之外的玩家日子都不好過。融資遇阻，裁員調整等消息不時傳出，有一些則最終停運倒閉，進入死亡名單。巨大的市場規模，使得創業者們不易快速摸到天花板，但通常情況下，賽道越火爆，就意味著競爭將更加激烈，投資者則更容易因投機而死。

在這些關閉的公司中，不乏「站在風口上的豬」，比如淘集集、鮮來多等知名電商；也不乏影響了萬千讀者的自媒體，比如引發眾怒的咪蒙；甚至曾經站在電競直播頂端，與鬥魚、虎牙一較高下的熊貓直播也難逃破產命運。估值曾高達100億元的團貸網、互聯網房產黑馬公司愛屋及烏、自動駕駛的明星公司Roadstar.ai……千里馬甚至獨角獸企業相繼倒下，新經濟公司在2019年的發展狀況並不十分樂觀。

這還不是最壞的時代 2019年或成由盛變衰轉折點

根據數據統計顯示，陣亡公司最多的是在2017年，有2056家新經濟公司宣告死亡。2015年陣亡1140家，2016年陣亡1446家。

湖北地區死亡公司數據與這個趨勢基本相符，2017年，湖北共有42家新經濟公司宣告死亡，2016年陣亡24家，2015年陣亡16家。數據顯示，湖北今年共有6家新經濟公司倒在了寒冬里，其中3家來自光谷，分別是夢想家Uhoem、十一平米車養護、愛招聘。

2019湖北死亡公司名單（數據來源：IT桔子）

但光谷夢想查詢發現，這三家公司中只有夢想家Uhoem的天眼查信息顯示「已註銷」，其他兩家公司仍是存續狀態。其中，愛招聘項目的母公司武漢智尋天下科技有限公司已經掉轉方向，專註公司的另一產品「簡尋」，並不能算是「死亡」。

值得注意的是，陣亡公司最多的三年正處於創投熱潮的泡沫期，資本鼓吹的失敗率較高，造成了「死亡率」較高。2018年起，資本越發冷靜、新興風口行業泡沫破滅，行業洗牌整合加速，有媒體分析，2019年才是新經濟由盛變衰的轉折點。

一家企業成功需同時兼具天時地利人和，同樣，企業的消亡也是由多種問題迸發而導致。在數據整理中，2019新經濟公司死亡原因大致歸納為市場、團隊、產品、資金、運營、環境六種。

而市場定位不準是新經濟死亡公司的最主要原因，從IT桔子統計的數據中看，死亡的新經濟企業超過半數擁有市場方面的問題，產品以及運營問題也是基於對市場的把控出現偏差，從而影響變現能力出現資金問題，再加之外部環境催逼，公司最終落寞。

融資失利，似乎成為了壓死創業公司的最後一根稻草。

寒冬之下，投資人出手越來越謹慎。

2019前11個月，VC市場投資總額約7300億人民幣，同比下降29.5%；投資案例數量約7800起，同比下降18.7%。而細分來看，各行業的投資數量同比去年均下降的十分厲害，其中金融行業滑坡最為明顯，投資數量下降了85.5%，娛樂傳媒行業下降了78.7%，物流行業也下降了73.4%，側面反映出企業融資難度加大。

隨著風口變冷，資本回歸理性，失去輸血能力的創業公司很難堅持下去，即使未來也許有機會成功。

經營公司不能一蹴而就，走上倒閉也絕非頃刻之間，在紅海泛泛、浮屍遍野的2019年，倖存的新經濟公司需要再度審時，謹慎邁步。

寒冬之下，創業公司應該如何度過？

據光谷夢想初步統計，截至2019年12月16日，光谷共計發生68筆融資交易，總計交易金額接近10億元人民幣，融資數量較去年有小幅回落，但相差不大，逐漸呈現出平穩的態勢。

處於雙周期轉折點的2019年，轉眼間已過去一半，寒冬散去後的初春，新經濟公司需要採用新招式，才能逆轉2019年的頹勢，在未來發展中佔據先機：

1、建立業務護城河

2019年資本向頭部主流公司聚集的現象更加明顯，新經濟公司建立護城河的重要方式之一，就是建立行業規模壁壘。如今業務跑進前三已經並不穩妥，前二甚至前一才有可能業務面基本穩定。尤其在行業壁壘不高的賽道，典型諸如近幾年的電子商務類賽道，2019年死亡公司數量高達38家，排列新經濟公司死亡賽道第2名。要想在激烈的戰場中殺出一條血路，建立護城河可以說是新經濟公司的不二選擇。

2、賺辛苦錢壁壘更高

2019年消費互聯網紅利逐漸消失，產業互聯網的價值逐漸體現，AI、消費、教育等領域的投資熱度也正在以肉眼可見的速度下降。過去資產優化、供應鏈提效等領域賺取的是辛苦錢，並不被新經濟公司看好，如今在產業互聯網時代則有飆紅跡象。2019年中，重複獲得融資的企業近乎都是產業互聯網企業，比如貨拉拉、水滴互助、準時達等企業。他們在下沉市場或者物流/醫療等供給側做生意，逆市取得較好的資本成績。

3、考核標準從增速換為現金流

2019年的一個明顯大趨勢，是新經濟公司正從注重增速轉換為增加現金流。商業化或者盈利是2019年新經濟公司的關鍵詞。快手2019年營收目標增加50%，抖音目標營收達到500億元，知乎全面提升社區的商業化程度，一些以「慢公司」著稱的公司都開始在行動。除去上述公司外，2019年準備加速商業化的公司，還可以列出長長的名單。

結語

去年年底，美團總裁王興在社交媒體上發了一段若有所指的話：「2019可能會是過去10年最差的一年，但卻是未來10年最好的一年。」雖然市場並非王興所言這般悲觀，但市場情況確實正在轉變。

隨風而起、風停霧散。資本寒冬的冷風從2018年底便猛烈颳起，在風口論、泡沫論的籠罩下，2019年新經濟市場有喜有憂。但是能夠看到，近兩年死亡公司的數量在逐年下降，投資者、創業者在試錯中不斷地調整著策略。

希望這個冬天早點過去，期待新的產業互聯網風口帶來下一個美好明天。
