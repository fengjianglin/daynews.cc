高曉松加入民主黨派九三學社，網友鼓勵“積極參政議政”

2019-12-17 1,634條評論 4,336次閱讀 2896人點贊

東方網·縱相新聞記者 陳浩洲

今天（12月16日）下午，著名音樂人、阿里娛樂戰略委員會主席高曉松在個人微博曬出照片，證實自己已加入民主黨派九三學社。他寫道，「感謝九三學社吸收我加入這個知識分子大家庭的民主黨派，願與全國十八萬學社同志一起，為學社『民主、科學』的宗旨奮鬥。」

九三學社北京市委員會組織部發給中共阿里巴巴（中國）有限公司委員會的函件顯示，經九三學社北京市委員會研究決定，同意吸收高曉松同志為成員，批準時間為2019年11月25日。另一份函件顯示，高曉松被編入九三學社商務印書館支社參加活動。

公開資料顯示，高曉松1969年11月14日生於北京，是著名音樂人、導演、脫口秀節目主持人。2015年7月，他加盟阿里音樂出任董事長，2016年9月出任阿里娛樂戰略委員會主席。

高曉松在微博曬出照片後，也受到了粉絲和網友的關注。網友「這裡是小岳」評論說，高曉松在50歲生日之際加入九三學社，很有紀念意義。網友「陽陽要當麵包干」則表示，希望高曉松積極參政議政，充分發揚社會主義民主制度的優越性。

高曉松加入的九三學社，是以科學技術界高、中級知識分子為主的具有政治聯盟特點的政黨。官網顯示，截止到2018年12月31日，九三學社共有地方組織340個，社員總數181262人。九三學社是「院士大戶」，先後擁有180餘位中國科學院院士（學部委員）和中國工程院院士。社員王淦昌、鄧稼先、趙九章、陳芳允、程開甲榮獲「兩彈一星功勛獎章」，社員王選、黃昆、師昌緒、謝家麟、程開甲是國家最高科學技術獎得主。
