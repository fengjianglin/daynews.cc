百度員工想跳槽華為，曝出年薪60萬變90萬，網友說他：瞎折騰

2019-12-22 1,215條評論 3,989次閱讀 1389人點贊

原創:丁香學姐 來源:大魚號

近日，在互聯網職場社區，一名百度的員工覺得工作太安逸，想跳槽去華為奮鬥，也順利拿到了入職offer，但還是有點糾結前途問題，於是發帖徵求大家的看法。

樓主在百度工作四年了，現在是T5級別，一年總收入60萬多一點，但樓主今年才26歲，想著年輕可以再拼一拼。去華為面試，華為給出17級的待遇，一年下來基礎工資加年終獎將近90萬，雖然第一年沒有股票，但HR說一年後就能配股。樓主說其實百度領導挺器重自己的，但覺得三年之內想升到T7太難了，工資漲幅也不大，樓主拿不準主意，想問問大家：在華為是不是比百度更有前途？華為三年能晉陞到18級嗎？第二年什麼情況可以給配股呢？

百度T5級別可以說是員工等級薪資表的分水嶺，T5及以上級別才有股權，百度T4、T5人數佔比也是最大的，基本上只要認真工作，達到T5還是比較容易的，年薪可以輕鬆達到50萬。華為社招工作八年的普通員工一般會給予16或者17級，而18級往下的人數也是最多的，18級往上就要重新考評，技術崗的話19級就是頂配了。樓主面試的崗位，和自己在百度的工作職能也比較匹配，工作內容上應該上手很快。

有網友說，華為的企業文化和百度確實存在差別，華為員工普遍都很拼，有人覺得在華為壓力大，但就像樓主說的年輕人可以拼一拼。華為國際化的視野也是其他公司所不具備的，華為各方面條件都很不錯，互聯網企業競爭這麼激烈，選擇一個大的平台，更能抵禦各類風險。反觀百度，近幾年有點走下坡路，既然樓主想拼搏，那在華為的前途是無可限量的，及時跳槽是正確的決定。

據華為員工透露，在華為晉陞18級還是有點困難的，公司里18級的基本都是領導級別，如果實力確實優秀，得到領導認可，兩三年升到18級也是可行的，但再往後就不好晉陞了，一來沒有職位欠缺，二來可能年紀大了，沒晉陞前就被優化淘汰掉了。網友說樓主現在在百度的職位也不低，晉陞T6也很快，沒必要瞎折騰，畢竟很多從輕鬆的互聯網大廠去華為的人，都說落差會比較大，一畢業就入職華為的還好，跳槽過去的都表示不太適應。

對於樓主問的配股問題，部分網友覺得有點懸，年薪90萬應該是已經加上股票的數額了，但17級社招進來的員工一般沒有股票，薪資方面還是跟HR詳細核對一下較為慎重。想跳槽肯定有一部分原因想漲薪，如果華為薪資水平確實比百度給的高，職位方面又比較匹配，那完全可以去華為拼搏試試。
