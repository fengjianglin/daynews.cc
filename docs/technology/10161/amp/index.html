帶你了解電動汽車，電池與汽油能量密度對比，季節對續航影響

天天要聞
2019-12-06
  • 電動汽車的結構比燃油車簡單

    • 從機械結構上來說，電動汽車要遠比傳統的燃油車簡單得多。電動汽車只需要非常少量的傳動部件就可以把電動機的動力傳遞到車輪上，而且這些傳動結構較為簡單。

    以變速箱為例，傳統汽車中的變速箱需要多檔變速，這就導致其內部傳動系統複雜，整體重量和體積大，在車輛中的布置非常苛刻。而電動汽車的呢，由於其電動機的寬轉速範圍，無需多檔變速，因此，大多數新能源汽車都只採用了單極變速，較小的變速箱體積，甚至可以做到與電機合為一體。除此之外，內燃機和電動機的複雜程度對比也是天差地別。正是因為系統的精簡化和集成化，電動汽車零部件失效的可能性遠低於燃油車。但當提到動力電池，這可能會讓人很焦慮。

    • 動力電池

    電動汽車上使用的動力電池與手機和筆記本電腦上使用的電池是非常相似的，只不過車上的要大得多。鋰電池與傳統汽車上用的鉛酸電池和鎳金屬電池有著本質的不同，其中一個最主要的特點就是新能源鋰電池的能量密度更高。而且鋰電池的靜置容量損失更少，鉛酸電池擱置一段時間後往往會虧電，正是因此燃油車停得太久之後，需要搭電救援。新能源的動力電池並非直接就是一個很大的電池，而是用很多小的單節電芯組成的。特斯拉有一款車就使用了7000多節18650電芯組成，不過值得一提的是，他們並不是一次就組成動力電池的，而是先由幾十或幾百節的小的電芯先組成模組，再組成動力電池包的（當然，最近很多廠家為了降低成本，提升能量密度，在進行去模組化的電池研發，提升電動汽車續航里程。CTP技術提升動力電池能量密度，增加電動汽車續航里程）。

    • 動力電池的電量和重量

    動力電池所能容納的能量的多少，衡量方法是使用千瓦時（KWh)單位，也就是我們常說的多少度（電）。從這個角度來說，買一個裝電能力更強的電動汽車，就相當於買一個油箱更大的燃油車，這樣，不管是電動汽車還是燃油汽車，加滿一次，都能開得更遠一些。不過在實際使用中，動力電池的電池管理系統（BMS）會確保動力電池的電量不會100%的被抽干，以確保電池不會很快的損耗，以延長其使用壽命。另一方面，動力電池的電量也不是越多越好，一個燃油車，油箱容量的增加對整車質量的增加微乎其微，而對電動汽車而言，每一度電的增加，可能會導致整車重量增加10kg左右。我們常說的能量密度，就是電池的能量除以電池的重量所得出的比值，也即每單位電池所能容納的電量，目前動力電池包的能量密度主要分布在100-200wh/Kg範圍。

    • 電動汽車的續航里程

    幾乎所有的汽車廠都會使用一個叫做續航里程的參數來評估電動汽車的單次加電里程。但是實際情況是，不同的駕駛速度下，續航里程可能會完全不同（比如國內比較喜歡使用60公里等速續航）。除了速度，對電動汽車裡程影響的因素還非常多，比如當前的環境溫度，個人駕駛習慣等等。因此，如果你發現你的電動汽車續航里程和實際廠家標註不一樣，一定要先看看廠家是否標了恆速、不開空調等等條件。可以說溫度是一個非常關鍵的因素，特別是北方寒冷地區，可能會導致電動汽車既不能充電也不能放電，也就是我們說趴窩了。一項由AAA found 做的研究表明，當環境溫度低於大約6.5℃時，電動汽車就會啟動電池加熱器，電動汽車的續航可能會達到41%的降幅。而到了夏天，當室外溫度達到35℃時，用戶會打開車內空調，這可能會導致電動汽車的續航下降17%。

    低溫導致電動車續航降低

    電動汽車最為核心的問題就是續航和充電的問題，能力密度是影響續航的主要內在原因，而環境溫度則是影響續航的外在原因，除此之外，還有許許多多的因素也會影響電動汽車的續航能力。

    如果電動汽車的續航與官方標註不一樣，一定要先看看廠家的測試條件再生氣。

    如果實在接受不了電動汽車的續航，也可以考慮一下可換電電動汽車，三分鐘更換電池，滿電即刻出發。
