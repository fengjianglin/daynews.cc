Postman收費太貴了，我決定用Postwoman...

2019-12-19 3,307條評論 4,279次閱讀 2354人點贊

介紹

Postwoman是一款開源的 Postman 替代品：

  • 輕盈，可直接在線訪問；
  • 簡約，採用簡約的 UI 設計精心打造；
  • 支持 GET, HEAD, POST, PUT, DELETE, OPTIONS, PATCH 方法；
  • 支持驗證；
  • 實時，發送請求即可獲取響應。

體驗demo：https://postwoman.io

Github

https://github.com/liyasthomas/postwoman

開發

先來看看它支持的功能列表：

  • 開源
  • 運行在瀏覽器端
  • 支持多平台、多設備
  • 支持PWA
  • WebSocket 測試
  • 定製化
  • 歷史記錄
  • 集合
  • 代理
  • 請求前腳本和環境變數
  • Docker

使用步驟

git clone https://github.com/liyasthomas/postwoman.gitnpm installnpm run dev//打開瀏覽器即可//或者，使用docker-compose：#pulldocker pull liyasthomas/postwoman#rundocker run -p 3000:3000 liyasthomas/postwoman:latest#builddocker build -t postwoman:latest

界面截圖

幾個常用的快捷鍵：

發送請求：ctrl+G

保存到收藏夾：ctrl+S

複製請求鏈接：ctrl+K

重置請求鏈接：ctrl+L

其實它的顏值真的高，隨手截圖一下，供大家參考^_^

總結

個人覺得，運行在瀏覽器端這一點比較實用，畢竟我們都是 Web 開發人員，瀏覽器跨平台的便利性早已深入人心。無需安裝，隨時隨地可以測試介面。按照作者自己的說法，他就是因為自己的破機器無法再忍受安裝一個這麼大的 Postman 客戶端，才動了這個邪念開發出了 Postwoman 的。

最重要的一點是它是開源的，且免費。
