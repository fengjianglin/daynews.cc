騰訊年終獎刷屏後，阿里百度京東美團要咋辦？

天天要聞
2019-12-21

12月19日，騰訊發年終獎的消息還是爆屏了。先是騰訊雲的陽光普照，騰訊雲今年完成了100億營收目標，全體騰訊雲員工人手一台iPhone 11 Pro，總獎額超過一億元。

是的，你沒聽錯，是1個億！

這顯然不是騰訊的年終獎重點，據說微信支付團隊拿到了騰訊2019年的公司創始人獎，將會獎勵給微信支付團隊2億元，按照一千人算，人均就是20萬。然而，這2億還不算是年終獎，據說微信支付團隊的年終獎都是10個月工資起算。

消息傳開，不只是互聯網各大名廠員工們瘋了，我的票圈也都瘋了，都整齊劃一地表示要去企鵝廠上班，簡直是：白日做夢！

以前，阿里的馬老闆說996是福報，很多不明就裡的人衝上來就罵，一想到各個大廠在年終獎神馬上的投入，我有一點相信福報了……

今年騰訊的年終獎一經放出，吃瓜群眾一臉羨慕外，各個互聯網大廠的HR可能要一臉懵逼！迎來內心三連問，我們咋辦？發多少？老闆會批嗎？在各大廠還沒有放出來消息之前，扒老師大膽預測一下吧！

阿里

今年阿里在香港又又上市了，年終獎自然也成為更多數人所關心，但歷年來阿里的年終獎算得還是釘釘鐵鐵，代表其企業高度價值觀的一句話是：在阿里上班還想年終獎是沒出息的。

按照慣例，阿里巴巴每年四月公布發放上一年度年終獎。這個時間點很雞賊，想要拿完年終獎，過完年不來了？不存在的！

但是阿里巴巴的年終獎還是很不錯的，據了解普通員工，一般可以拿到四個月到九個月月薪的年終獎。所以阿里的員工可以根據這個慣例算算自己能到到的年終獎範圍。

華為

和騰訊為同城的華為，早在11月份時就發過一次總額為20億的獎金，前期是爆出胡玲事件，獎金髮完後，網上再次又爆出251事件。華為的獎金也有的說是按照季度發放，之前統計的略少，但今年兩大事件再加上毛衣戰，華為員工壓力不小，年終獎上應該會「安慰」到員工吧！

下面這個「華為員工」的爆料，也可以供華為人參考，坐標成都，在華為工作三年，年終獎發了32萬，稅繳了8萬。

百度

百度還沒有年終獎的消息，但演算法已經在網上有所披露，結合百度今年的表現來看，也是一波三折，大跌之後再沒回到三甲之位。百度的現金奶牛還是要大量投放到AI項目上，此一時的縮手腳顯然是為了積蓄更多未來的力量。

按照往年的慣例，百度的年終獎一般是2個工資，當然也有能拿到更多的。今年如果不減少的話，百度的員工就應該感謝李帥哥了。

美團

美團一般在年終獎上並不是很突出，但今年可能有所不同，今年美團的股票業績不錯，照理說應該大方一回了。但是，也不能太樂觀，因為畢竟說起來，美團仍然是送外賣的企業，大手大腳的風格不太適合其營生。

京東

往年京東的年終獎都不會作為重頭話題，京東年底的大戲一般在「年會」上。但是隨著老劉退居幕後，今年的年會能否成為「爆款年會」還不知道。今年京東有所不同的是，京東市值終於超過了百度，Q3業績也扶搖直上，年終獎亮一次豹子也大有可能，況且之前老劉對員工也是真性情付出，值得期待。

保守預估，京東員工的年終獎為1—2個月工資,績優員工可以在正常年終獎勵外獲得額外獎金。

小米

小米衝擊IPO以來，屢創佳績，未上市跟上市能否不同？今年也是極為不同的一年，去年小米也是被垢病「扣門」，而在年會抽獎上，畢竟也是有電動汽車的，也算很可以了。今年包括相機在內的很多部門曾害怕拿不到年終獎，現在看來，拿不到其實也比較正常。

滴滴

各大互聯網公司里，滴滴從創立以來，印象里只搞過一次轟轟烈烈的年會，且獎品豐富，令人遐想萬千。這兩年來，滴滴每一年都會出點事情，也搞得今年的年終獎終於不需要有什麼懸念了，去年就減了半，今年對有些在職人來說，有更好，沒有，也沒辦法。

總結：

對各大互聯網公司的年終獎做了一下掃射，每年這個時節，往往是一些企業領導人巴不得出去躲幾天的時候，這兩年也是經濟拐角太明顯，很多企業壓力巨大。年關將近，很多裁員之聲四起。真正不裁員還能堅持發獎金的公司真的是不多了，希望大家都能且行且珍惜。

你們公司的年終獎是多少呢？歡迎大家在評論區踴躍留言~
