創新嘗試：華東政法大學“一帶一路”法律大數據平台上線

2019-12-05 1,260條評論 3,501次閱讀 2045人點贊

「一帶一路」法律大數據平台4日正式上線。 芊燁 攝

「一帶一路」法律大數據平台4日正式上線。 芊燁 攝

中新網上海12月4日電 (記者 陳靜)由華東政法大學師生共同打造的、全開放「一帶一路」法律數據信息檢索系統——「一帶一路」法律大數據平台4日正式上線。

該平台集結了「一帶一路」沿線66個國家，包括憲法，民商，刑事等在內的多類別法律法規以及與中國的雙邊條約協定，並專門專門針對航天法律、海事海商、爭端解決、投資措施設立討論專題。

據悉，華東政法大學是中華人民共和國創辦的第一批高等政法院校之一。當日上線的平檯面向全社會公眾，致力於為專家研究、投資合作者諮詢以及相關企業和政府部門決策打造全方位、一站式的「一帶一路」法律信息服務。

據華東政法大學「互聯網+法律」大數據平台常務副主任童瀟介紹，華政「一帶一路」法律大數據平台是中國國內「一帶一路」法律服務網站的創新嘗試。他告訴記者，目前已建的「一帶一路」法律服務網站大致分為新聞集成型和「律所依託」型。

童瀟分析，新聞集成型網站對外部新聞來源的依賴性較強，同時，此類網站同質化嚴重；「律所依託型」網站實質上是另一種方式的雲上「面對面」諮詢，付費諮詢對於學生和研究人員深度獲取信息絕非最佳使用途徑。

華政「一帶一路」法律大數據平台則充分利用華政法學學科優勢，力求實現網站外文資料的本土化。該平台下設5大版塊，包含重要時訊、觀點報告、沿線國法、條約協定、法制專題。

據介紹，重要時訊版塊充分展現「一帶一路」國家法律法規發展最新動向和中國各省市縣「一帶一路」建設最新資訊。觀點報告版塊收入專家學者對中國和「一帶一路」沿線國家的法律外交觀點、時評及文章。條約協定版塊收入中國與俄羅斯、新加坡、韓國、日本、印度等簽署的雙邊條約及多邊條約，為中國與「一帶一路」沿線國家保持發展良好合作保駕護航。

記者看到，「爭端解決」專題深度聚焦「一帶一路」國家相關爭端解決方式，提供調解案例、法律法規、聯合國調解公約、仲裁機構、仲裁案例等；「投資措施」專題，雙向囊括中國與「一帶一路」沿線國家投資措施，為進一步促進中國在國際層面投資提供政策支持和案例參考。(完)
