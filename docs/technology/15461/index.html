中電啟明星：操作系統快速部署平台 解決IT運維人員之苦

2019-12-15 2,036條評論 4,564次閱讀 1789人點贊

【能源人都在看，點擊右上角加’關注’】

北極星輸配電網訊:近日，中電啟明星操作系統快速部署平台在國網安徽電力持續擴大試用範圍，進一步推動國網安徽電力操作系統的部署流程規範化，提升工作效率，簡化運維人員工作內容。

隨著信息化工作的逐步深入，物理機環境的運維工作也變得極為複雜，運維人員面對日益增長的運維規模與有限的人力，工作負擔愈發沉重。當前，現有的人工運維方式無法滿足運維工作的實際需要成為普遍現象，主要體現在傳統的安裝部署及配置調試均採用手工方式，而手工方式存在易出錯、出現問題難排查、大量重複性操作等缺點。

針對上述問題，中電啟明星採用預啟動執行環境、動態主機配置協議、超文本傳輸協議、簡單文件傳輸協議、YUM倉庫等技術進行統一管理，並最終實現了不同用戶需求的可定製操作系統部署方案，即操作系統快速部署平台。

該平台具備「一鍵式操作系統交付」功能，能夠將繁瑣的機器開機、重啟、部署等環節智能化，實現「一鍵式」整體交付，過程中無需人工干預，整體效率較傳統交付方式提升近5倍，而面對企業級大批量物理機同時上線的情況，平台所帶來的效率提升逾加明顯。

（圖：一鍵部署）

同時，平台對所有接入交換機下的物理伺服器均可進行管理，可輕鬆部署常見的Windows、Linux等操作系統。不僅如此，平台通過自動部署功能，還可設置SSH埠號進行自動更改、OpenSSH自動升級和符合國家電網公司要求的安全加固項，以此來實現一鍵部署操作系統的IT運維需求。

（圖：離線鏡像部署）

目前，中電啟明星操作系統快速部署平台已經在國網安徽電力等網省公司開展了試點應用。試用證明，通過該平台，各企業進一步規範了操作系統的部署流程，將傳統的系統安裝方式替換為網路安裝，並將系統鏡像統一化、標準化。運維人員通過「一鍵部署」的方式，輕鬆保證了所有物理伺服器環境的一致性，並最終實現了自動註冊、一鍵部署、一鍵恢復等功能。

下一步，中電啟明星還將繼續深挖當前運維工作的難點和痛點，持續完善操作系統快速部署平台的功能，打造平台的技術優勢，為基層運維人員減負。
