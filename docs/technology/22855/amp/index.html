從老中醫購得配方，靠這“擄獲”女人心，他用“小秘方”賺279億

天天要聞
2019-12-20

一個企業想要成功，需要有哪些要素？有的人會說，機會十分重要。雷軍也曾說過「站在風口上，豬都能飛起來」這麼一句通俗但是極具內涵的話語。許多成功的企業家，其發家的背後也與機遇密不可分。馬雲抓住了互聯網興起的機遇，成就阿里巴巴帝國；王健林抓住房地產大熱的機遇，成就萬達集團；雷軍抓住手機市場的機遇，成就小米的大火。如今，這些抓住機遇的企業家都成就非凡，更是成為了業界標杆之一。

但在機遇之外，一個企業要想成功，作為發展基礎的技術力是必須的。雷軍如果對於手機一竅不通，那麼也不可能打造出國民品牌小米。還有許多企業，憑藉收購技術、傳承秘方成就商業帝國。老乾媽從一個小攤的秘制辣醬做成了全球暢銷的辣椒醬；娃哈哈更是花費了5萬元買來醫科大學的獨門秘方，依靠「娃哈哈口服液」迅速征服市場，進而賺得數十億身家。

有一人，也像娃哈哈的創始人宗慶後一樣花費重金買下配方，依靠這個配方，現在已經賺到了279億身家，他就是創立健康元的朱保國。

對於健康元，可能許多人第一印象對他並不了解，但是如果提起它旗下的產品「太太口服液」，相信一定有許多人都有印象,曾經林志玲與毛阿敏還曾為太太口服液代言。

在幾十年前，保健品行業曾經十分火爆，許多企業依靠保健品在一夜之間身價暴漲。但是，由於缺乏監管，保健品行業當時也是十分混亂，許多企業在其中魚目混珠，做了許多虛假宣傳。不過，消費者的眼睛總歸是雪亮的，虛假的產品不可能在市場上長久地存活下去。終於，這個讓人暴富的時代在幾年時間後便逐漸冷靜下來，許多保健品企業也就此銷聲匿跡，消失在眾人眼中。

而健康元企業卻在這場冷卻中堅持了下來，並且成功轉型，這其中離不開健康元的創始人朱保國。

從老中醫手中購入配方，在保健品行業中殺出一片天地

1962年，朱保國出生於河南新鄉，從河南師範大學化學系畢業後，朱保國被分配到家鄉焦作的一家國企中擔任技術員，後來又被調往一家瀕臨破產的國營化工廠擔任廠長。

朱保國的經營能力十分出色，儘管這家化工廠在他上任時便千瘡百孔，但是在他的帶領之下，2年內竟然起死回生，實現了營收4000萬的業績。

經此一戰，朱保國對於自己的經營頭腦有了一定信心，創業的念頭隨即誕生。正好，當時保健品行業正處於蓬勃發展的時期，朱保國盯准了這個機會，在朋友的介紹下，花費了9萬元，從新鄉的一名老中醫手裡買來了其獨家發明的女性保健品配方，並且和幾個朋友共同籌資近2000萬，在深圳創辦「太太葯業」，後來發展為「健康元」股份。

但是天不隨人願，朱保國的工廠在建廠的時候碰上了颱風天氣，原本4個月就能完工的工廠，硬是花費了8個月才完成，這導致朱保國手中的流動資金十分緊缺。

為了公司的發展，朱保國不惜將工廠拿去抵押，借來了500萬的資金，朱保國也深知這是一次豪賭，如果成功，他將會富有，如果失敗了，那麼他身上將會負上難以想像的巨額債務。

但是這一次，朱保國賭贏了。當時市場缺少針對女性的保健品，而由於太太口服液的配方獨特，並且主打女性市場，迅速吸引了大量消費者的青睞，一時之間成為了國內女性保健品品牌當中的領軍者。2000年，太太口服液的銷售額更是突破了4個億。

保健品行業從讓人「暴富」到「暴負」，而他卻成功轉型存活

趁著太太口服液的紅火，朱保國還推出了靜心助眠口服液，還斥重金請來了林志玲和毛阿敏為其代言，太太葯業可以說風光無限。

2001年6月，太太葯業在上交所正式上市，募資17億元，這為朱保國在後來為太太葯業轉型提供了足夠的資本。

2002年3月，也許是看到了保健品行業發展空間的局限性，又或者是想要轉型，朱保國以每股4.27元的單價收購了麗珠的股份，旨在進軍醫藥行業。

果然保健品熱很快就冷卻下來了，保健品的效果逐漸被人所質疑，更有人認為太太口服液是「忽悠」人的產品，而許多保健品品牌諸如太陽神、三株等也在這次行業震蕩中敗下陣來，逐漸消失在人們的視野當中。而太太葯業卻在朱保國的帶領下存活了下來，並且更名為健康元集團。

健康元集團的市值不斷提升，而朱保國的身價也隨之節節高盛，達到了279億元，登上了福布斯排行榜。

在當時保健品行業如此紅火的情況下，朱保國能保持如此敏銳的眼光及時轉型，在那場「暴負」風暴中全身而退，轉向醫藥領域，不得不說，朱保國的商業眼光確實準確。
