亞馬遜想發射數千衛星卻不想申請頻譜 SpaceX不幹了

天天要聞
2019-12-21

科技訊 12月21日消息，據外媒報道，任何衛星互聯網項目發射之前，都必須得到美國通信委員會（FCC）的批准，才能在特定頻譜範圍內運行。但亞馬遜現在正尋求豁免這項規則，此舉遭到SpaceX、OneWeb等公司極力反對，聲稱這損害了競爭。


亞馬遜已經證實，該公司計劃將3236顆寬頻互聯網衛星發射到近地軌道上。這個名為Project Kuiper的龐大衛星群，將來可為全球數千萬人提供低延遲、高速寬頻，同時也將把他們與亞馬遜提供的服務連接起來。但Project Kuiper迄今尚未獲得FCC批准。

在今年7月提交的申請中，亞馬遜希望能豁免FCC的規定，從而獲得必要的發射許可。然而問題在於，FCC早在幾年前就已經通過複雜的審批過程，向其他九家衛星互聯網公司發放了頻譜許可證。

FCC的記錄顯示，包括SpaceX和OneWeb在內的這些公司，正在遊說FCC否決亞馬遜的豁免請求。如果遊說成功，這將顯著降低Project Kuiper在本就相當擁擠的衛星互聯網市場上的生存能力。

在FCC的記錄中，SpaceX高管已至少三次與FCC主席阿吉特·帕伊(Ajit Pai)以及該機構其他工作人員會面，就亞馬遜的申請提出當面投訴。第一次會面是在亞馬遜提交申請幾周後，最近一次是在12月2日和3日。

SpaceX的律師在11月25日的文件中寫道:「亞馬遜公然試圖推翻長期存在的規定，這將破壞人們對審批程序的信心，損害競爭，並減少消費者的寬頻選擇。Project Kuiper將對SpaceX造成重大不利影響，亞馬遜有缺陷的分析得出的結果有違常理。」

業內專家認為，亞馬遜的請求不符合規則，但該公司試圖通過「走後門」的方式進入這個令人垂涎市場的渴望可以理解。如果Project Kuiper被迫等待第二輪ka波段頻譜授權，它可能會被要求在頻率上以「二等公民身份」運行，當其衛星干擾了之前建立的衛星互聯網時，就會被迫停止發射。

FCC官員表示，該機構必須考慮SpaceX、OneWeb和其他第一輪授權方已經獲得的權利。這位官員稱:「儘管我們認為，我們沒有理由讓亞馬遜遵循同樣的程序。畢竟，並非所有原來授權方制定的計劃都能成為現實，這個領域還為其他系統留有空間。」

主導衛星互聯網領域的競爭愈演愈烈，這為監管進入太空的主體帶來了前所未有的挑戰。人類總共向太空發射了不到9000個人造物體，目前有5400顆衛星仍在繞地球運行，而且只有半數處於激活狀態。

SpaceX、亞馬遜和其他公司提出的大規模衛星互聯網計劃將大幅增加太空中的衛星數量，以至於天文學家擔心衛星會遮住夜空。將大量衛星送入近地軌道也會使太空垃圾更有可能發生碰撞，並可能造成更大的破壞。

雖然太空很大，所以所有計劃中的衛星都有進入軌道的餘地。但傳輸數據所需的寶貴頻率受到的限制要大得多。行業專家表示，衛星寬頻互聯網的經濟效益非常低，可能只有少數早期進入者能夠生存下來。

TelAstra公司衛星和電信顧問羅傑·拉施（Roger Rusch）指出:「我曾認為亞馬遜無法生存，只是因為他們的計劃還沒有起步，而SpaceX、OneWeb已經在發射衛星了。到亞馬遜執行計劃的時候，他們可能已經落後好幾年了。」

SpaceX已經發射了1.2萬顆低地軌道衛星中的180顆，該公司已獲得FCC的運營許可，並申請再發射3萬顆衛星。SpaceX計劃在2020年再發射1440顆衛星，並希望在明年年底之前開始在全球提供互聯網服務。

SpaceX在Kuiper案中的行動表明，它急於保護自己的投資。SpaceX拒絕公開置評。除此之外，SpaceX還有很多盟友。OneWeb已經獲得了FCC發射720顆衛星的批准，並申請批准另外1260顆衛星。

FCC的記錄顯示，OneWeb已經提交了請願書，要求FCC拒絕亞馬遜的申請。其他公司，包括加拿大電信公司、Theia控股公司和銥星通信公司，也參與了反對亞馬遜的行動。

OneWeb此前曾抗議FCC允許SpaceX在當前頻段添加衛星的決定，但亞馬遜的請求要複雜得多。FCC從2016年7月開始接受ka波段的許可申請，截止日期為當年11月。12家公司提出了申請，其中兩家後來退出。該機構向九家公司頒發了許可證。

亞馬遜不是最初的申請者之一，該公司在那三年後才提交了申請，同時還提出了獲得與首批授權方同等待遇的特殊豁免申請。在申請中，亞馬遜辯稱，豁免是適當的，因為它的衛星落網計劃符合公眾利益，而技術進步將更好地使其衛星能夠共享已授權給其他公司的頻率。

對於亞馬遜的申請，SpaceX、OneWeb和其他公司認為，Project Kuiper會嚴重干擾他們正在談判的謹慎合作協議，就像在表演中途向變戲法的人扔了3236個不同形狀的新球一樣，這成倍地增加了複雜性。

亞馬遜發言人表示：「我們期待就許可申請事宜與FCC繼續接觸。亞馬遜專註於為我們的客戶進行創新，包括實現我們對Project Kuiper的願景，即為全世界數千萬生活在沒有網路服務或服務不足地方的人們提供低成本、高速寬頻服務。」（小小）
