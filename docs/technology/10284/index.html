當“共享”成為一種消費理念時，王思聰還敢說“不看好”嗎？

2019-12-06 215條評論 1,229次閱讀 282人點贊

共享單車

共享單車ofo

共享電動三輪車

第三方數據研究機構比達諮詢日前發布的《2016中國共享單車市場研究報告》顯示，截至2016年底，中國共享單車市場整體用戶數量已達到1886萬，預計2017年，共享單車市場用戶規模將繼續保持大幅增長，年底將達5000萬用戶規模。

報告指出，中國共享單車市場已經歷了三個發展階段。2007年—2010年為第一階段，由國外興起的公共單車模式開始引進國內，由政府主導分城市管理，多為有樁單車。2010年—2014年為第二階段，專門經營單車市場的企業開始出現，但公共單車仍以有樁單車為主。2014年至2018年為第三階段，隨著移動互聯網的快速發展，以摩拜為首的互聯網共享單車應運而生，更加便捷的無樁單車開始取代有樁單車。

報告還顯示，共享單車更受年輕男性歡迎。中國共享單車用戶中男性佔比54．2%，女性佔比45．8%。用戶年齡分布中，25歲—35歲人群使用最多，其次是25歲以下人群。使用頻率中，每周使用3次—4次的用戶最多。

繼共享單車、共享汽車後，共享運營的時代也催化了另外一種交通工具的流行，那就是共享電動車。

其實共享電動車的起步並不比摩拜和ofo晚，只是太多的運營商選擇校園或者是景區作為運營範圍，不受人們的關注。國內已經出現了包括獵吧、租八戒、小鹿單車、電斑馬、ebike、八點到、7號電單車、萌小明等一系列從事電動車租賃的公司，並且還有一些傳統電動車製造商也在考慮以電動車切入到共享出行領域。

2017年3月，某品牌共享電動車出行首現北京，收費模式0.5元每公里+0.1元每分鐘。

這款新型電動車也是通過掃碼開鎖，尾部擋泥瓦的黑黃相間條紋看起來很像小蜜蜂的腹部，前面裝有車筐。

共享充電寶

來電科技共享充電寶

王思聰不看好陳歐投資共享充電寶

共享充電寶是在「共享」風口上出現的全新細分行業。2014年8月，來電科技成立 [4] ，隨後，才有了接二連三的入局者。

2017年以來，隨著共享經濟概念的火爆，共享充電寶也被資本推上風口。而除了融資大戰之外，專利大戰早在2016年就已爆發 。

2016年7月22日，來電科技以侵犯其3件實用新型專利權為由將雲充吧告上法庭。2016年10月，深圳市中級人民法院一審判決來電科技勝訴，要求雲充吧立即停止製造、使用、銷售、許諾銷售等侵犯來電科技專利權的行為，並賠償來電科技人民幣60萬元。雲充吧不服判決並隨即向廣東省高級人民法院提起上訴 。

實際上繼雲充吧後，來電科技後來又相繼起訴街電、友電等侵權。其中街電科技因聚美優品的3億元投資、陳歐親自出任董事長而被廣泛關注 。2017年5月，聚美優品CEO陳歐3億元投資共享充電寶企業，資本在搶時間，其布局在未來最晚不會超過5個 。

2017年7月，廣東省高級人民法院宣布二審判決，駁回了雲充吧的上訴，並維持原判。這場長達一年的專利糾紛告一段落 。

2017年7月27日，智能共享充電公司怪獸充電宣布獲得高瓴資本，清流資本、藍弛創投、順為資本等機構億元人民幣A輪融資 。

2017年10月11日，共享充電寶首現出局者， 樂電官方微信宣布停止運營 ，回收所有共享設備 。截至2017年10月11日，該公司已回收所有充電寶設備，並提醒用戶將未提現的押金提現，提現後餘額會在1至2個工作日退回。該公司的客服電話一直處於佔線或者無人接聽的狀態。樂電於2017年3月份上線，在杭州鋪設了近200台設備，覆蓋公交站、商場、KTV、酒店等場所。樂電與街電的類型相似，均是固定場所下的可移動型充電寶 。

隨著共享概念橫掃各個行業，在資本推動下，2017年上半年共享充電寶開始跑馬圈地，出現數家運營企業。最瘋狂的時候，曾經40天有12億資本入局。但近期有公司曝出裁員、資金緊張等消息，業界在關注這個新生態將如何演進。

共享服裝

共享服裝

2015年，共享服裝租賃行業曾經站上了風口，但受資本寒冬和用戶接受度的影響，行業充斥著「市場小眾」、「不看好」的聲音。

2017年，共享服裝租賃行業的融資通道再度開啟。目前，共享服裝平台主要有女神派、衣二三、多啦衣夢，但與從禮服租賃裝切入服裝租賃的女神派，擁有時尚圈資源的衣二三不同，多啦衣夢的目標用戶是收入在5000元上下、二三線城市的愛美女性。平台上的服裝主要集中在設計師時尚品牌，平均價格為600元至1000元。

隨著共享產業的擴展，興起共享服裝熱潮。顧客每月支付幾百元租金，便可無限次數租借服裝，每次可租3件。不過，有用戶對服裝衛生問題感到擔憂。儘管各大平台都表示，自己的清潔衛生標準比酒店的級別更高，但要得到用戶的信賴還需一定的時間。

共享手機

共享手機

2017年09月，共享手機業務現身昆明國美順城店，據該店負責人介紹稱，除了iPhone7手機的租賃，還支持iPhone7Plus、華為P10、魅族PRO7、魅族PRO7Plus、三星Galaxy S8 Plus等收集租賃，而將於9月12日上市新款iPhone作為共享租賃業務的主要推廣手機。

共享手機基本租期為1年，想要得到共享手機需要提供身份證還有銀行卡，還有該公司還要用提供專業軟體檢測消費者的信用卡是否存在還款逾期等不良記錄，如果有就不能申請，具體來說，以iPhone7 32G版為例，以信用卡方式租賃，信用卡剩餘額度必須5388元，租金是264元/期，每期25天，合約周期為15期，以芝麻信用方式租賃芝麻信用分必須650分、螞蟻花唄餘額>6131元，芝麻信用模式租金為330元/期，每期30天，合約周期為12期，一年下來租金為3960元。

當這種”共享經濟”成為一種普遍的現象時，你還能說共享沒有前景嗎?
