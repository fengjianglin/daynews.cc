蘋果16寸MacBook Pro爆音問題:將通過軟體更新修復

2019-12-08 878條評論 2,055次閱讀 820人點贊

蘋果日前推出的16英寸MacBook Pro號稱是Mac筆記本的巔峰之作，其中音質是官方宣傳的亮點之一。

然而好景不長，一些早期用戶已經開始反饋MacBook Pro的爆音問題。

對此，蘋果已經確認，表示將通過軟體更新的方式予以修復。在一份內部備忘錄中，蘋果提示維修點和售後人員，不要接受預約服務、也不要直接換機。

如果你在使用Final Cut Pro X、Logic Pro X、QuickTime Player、Music、Movies等軟體，尤其是暫停和清理音視頻時聽到噪音，那大概率就是中招了。


資料顯示，這並非蘋果筆記本第一次曝出聲音問題，2016年的Macbook Pro(TouchBar)同樣有類似遭遇，更早的甚至可以追溯到2007年，當時蘋果被迫對OS X Tiger進行大範圍升級更新。

按照此前蘋果的宣傳，16寸MacBook Pro聲音格外洪亮。六揚聲器音響系統可營造雄厚的立體聲音效，縈繞整個房間，大大提升筆記本電腦的聽覺體驗。雙振動抵消低音單元不但能有效減少系統振動帶來的影響，使聲音更清晰更自然，還使低音的音域再低半個八度。同時，錄音棚級三麥克風陣列幾乎可媲美專業的第三方麥克風。
