全球APP下載排行榜TOP10 中國有兩款上榜

2019-12-18 699條評論 2,243次閱讀 625人點贊

自打手機進入智能時代以後，基於APP的加持，手機已經不在是傳統意義上打電話發簡訊的工具了，而是進化成一個能拍照、能導航、能點外賣、能打遊戲的綜合體，當然這些功能都是由用戶自由下載的APP來實現的，所以近10年來，APP的發展大大提升了手機的軟實力。

近日，一份來自應用分析公司App Annie的數據報告顯示，2019年全球用戶通過iOS和Android設備下載的APP數量超過1200億，而在應用上的消費金額也超過900億美元，比去年增長了15%

除此之外，該公司還統計了一份近十年來APP下載量的榜單，除了國外比較普及的Facebook、Facebook Messenger、WhatsApp Messenger、Instagram佔據前四外，中國也有兩款APP上榜。分別是：TikTok（海外版抖音）、UC瀏覽器
