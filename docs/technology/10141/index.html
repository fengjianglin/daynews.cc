“廣協庫”平台開發應用，為廣告發布者帶來福音

2019-12-07 1,164條評論 2,701次閱讀 335人點贊

隨著國家監管部門對企業廣告發布行為由「前置審核」改為「後置監管」後，全國被查處的各類違法廣告案件不斷上升，已經對廣告經營者和廣告發布者產生了較大的風險。研究開發一個廣告內容風險防控諮詢平台，能夠對廣告內容在發布前進行審查和預警，顯得尤為重要。

由上海市廣告協會牽頭、多家省級廣告協會參與開發建立的「廣協庫」廣告內容諮詢專業服務平台，於2018年末已上線投入使用，這是一個由用戶對廣告發布內容通過網路平台自行檢測和專業諮詢的智能化開放式服務平台。開發和推廣「廣協庫」平台，旨在為用戶防範廣告發布內容和形式的違規風險，有效提升廣告行業形象，是近兩年來上海市廣告協會加強行業規範建設工作的重要組成部分。

「廣協庫」平台本著嚴謹和專業，由各廣告協會派出廣告專業人士對廣告內容進行審核諮詢。平台還不斷利用新的技術更方便用戶使用。在新的2.0版本中自動審核功能又增加了違禁廣告語提示。自2018年平台正式上線以來，「廣協庫」就已經完成多家大型傳媒巨頭旗下子公司以及數十家中小型廣告和傳播公司的創意類廣告內容審核，得到了用戶的廣泛認同。「廣協庫」平台的開發應用，為廣告經營者和廣告發布者帶來更多便利。

入口：微信公眾號、網站www.guangxieku.com

方式：直接遞交進行審核，自動檢測免費，需要轉人工由專家諮詢的，視審核材料的難度酌情適當收費。
