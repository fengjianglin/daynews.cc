行業聚焦│特斯拉反超再成全球電動車銷售冠軍 我們該思考些什麼

天天要聞
2019-12-22

截至2019年10月，特斯拉累計銷售807954輛電動車，相較之下，比亞迪則累計售出787150輛新能源汽車，由於比亞迪的銷量里包括純電動汽車和插電式混合動力汽車，因此，從這一統計結果來看，特斯拉再次超過比亞迪，成為了全球純電動汽車銷量的冠軍。

當新能源汽車補貼的大潮退去，才能顯現出誰在全球市場中競爭力更強。近日，美國數據研究者Kevin Rooke公布了一項統計數據：截至2019年10月，特斯拉累計銷售807954輛電動車，相較之下，比亞迪則累計售出787150輛新能源汽車，由於比亞迪的銷量里包括純電動汽車和插電式混合動力汽車，因此，從這一統計結果來看，特斯拉再次超過比亞迪，成為了全球純電動汽車銷量的冠軍。

為什麼說是再次呢？

根據Kevin Rooke的統計，與比亞迪同樣成立於2003年的特斯拉，在2016年前，銷量一直都在比亞迪之上，直到2016年1月左右，比亞迪整體銷量才超過了特斯拉，並在長達三年的時間裡始終保持全球新能源汽車銷量第一。但隨著 Model 3 產能的釋放，截至今年10月，特斯拉再次實現了反超。

事實上，在單車型銷售方面，特斯拉領先更多。據統計，今年1～10月，特斯拉Model 3全球銷量為22.13萬輛，位居第二的北汽EU系列售出8.2萬輛，比亞迪的元車型緊隨其後，銷售6.4萬輛，日產的聆風售出5.9萬輛，排名第四。可見，即使把後三名的銷量加起來，也不如特斯拉多。

那麼，為何在過去的三年時間裡，比亞迪能佔據新能源汽車全球霸主的位置？但從今年下半年開始，銷量增長勢頭依然迅猛的比亞迪卻逐漸被特斯拉反超？

這或許與中國新能源汽車市場高速發展的時期密不可分。

根據中國汽車工業協會統計，2014年中國新能源汽車銷量為7.5萬輛，而到了2015年，這一數字迅速提升至33.1萬輛，同比增長也驚人地達到了342.9%，創下歷史新高。此後，我國新能源汽車市場的增長率都保持在50%以上的高位。2018年，中國新能源汽車年銷量更是首次超過100萬輛，達到了125.6萬輛。而在今年前11個月，我國新能源汽車銷量為104.3萬輛，同比微增1.3%。

「比亞迪和特斯拉銷量之間的比較毫無意義。」一位業內人士指出，首先，從雙方的銷售數據來看，特斯拉的累計銷量全部為純電動乘用車，但比亞迪的銷量還包含了插電式混合動力和一部分商用車；其次，特斯拉麵向的幾乎為私人消費者，也就是C端用戶，而比亞迪則有相當一部分的汽車銷售給了B端用戶，包括計程車以及移動出行公司等；最後，也是不容迴避的重要一點，特斯拉產品的平均售價遠高於比亞迪，國內能在單車價格上與特斯拉同台競技的新能源汽車品牌少之又少。

目前，隨著新能源汽車市場競爭的進一步加劇，距離《外商投資法》的實施也還有不到十天的時間了，其中規定的外商投資的股比限制、國民待遇、政府採購、產權保護等條款，都將進一步提高我國新能源汽車市場競爭的激烈程度。到2020年年底，新能源汽車補貼也將正式退出，大眾、寶馬、賓士等外資品牌已經看準了中國新能源汽車市場的發展前景，準備好了一系列的新能源汽車產品……

當然，經過了這麼多年的發展，國內新能源汽車企業也積累了一定的先發優勢，例如形成了一定的品牌認可度、更加了解中國消費者的喜好等，此外，在智能網聯技術領域，我國汽車產業也具備更好的發展基礎。

在挑戰和機遇並存的未來，誰能更勝一籌，讓我們拭目以待。（中國汽車報網/施芸芸）
