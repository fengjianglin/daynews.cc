從滬電股份年報預測挖信息：華為基站明年翻倍 汽車行業仍未復甦

2019-12-21 735條評論 4,567次閱讀 973人點贊

作者：有魚的魚

12月還沒結束，滬電就很高調地發了業績預測，有說超預期的，有說低於預期的。

我是在11月做的業績預測，營收是72億、業績12億（之前有個帖子提過，但沒放過程），恰好是公司預測的中值。當時推算主要基於幾個假設依據，現在既然我的預測和公司相對吻合，反過來看，說不定就意味著這幾個假設就是成立的。

這幾個假設同時也是未來可能影響業績的主要因素，有必要說一下。

1、汽車板收入佔比和增速

半年度汽車板收入同比略低於18年。測算時假設汽車板收入每季度變化不大，且19年下半年汽車板營收下降幅度與半年度一致。

根據這個假設，可以計算出三、四季度汽車板營收和凈利潤。

結論1：全年汽車板營收略微下降（預計12.46億，增速-2.8%），毛利率持續上升（預計25%）。凈利潤佔比進一步減小。

結論2：說明汽車電子市場仍然沒有復甦，特斯拉產出對滬電業績貢獻尚未體現。

2、通信板（辦公部和企業通信板合為整體）收入和增幅

測試時第二個依據是把公司分的辦公板和企業通信板合併在一起看。根據總收入-汽車板收入，可以計算出三季度上述所謂通信板收入和增速。半年度通信板營收增速為36.59%，三季度是41.1%。

72億收入、12個億凈利潤對應的通信板增速是45%。公司整體毛利率也將有所提升（預計29.88%）。

結論3：19年預計營收達到72億（中值）、增幅為31%。相比半年度26.7%的增幅是在加快的。

結論4：由於5G拉動作用，通信板的增速呈現加速狀態，進一步改善毛利率。

3、要做個澄清，四季度凈利潤增幅下降正常嗎

有些人說，四季度是旺季，應該比3季度高。我們看看歷史的情況：

可以看出歷史上三四季度的確是旺季，但對比生益科技和深南電路，3、4季度凈利潤相對比較接近，高低都出現過。

另外一點，可以看出18年的幾個季度業績是太厲害了，尤其是4季度凈利潤增幅達到345%（17年4季度相對較少）。這個之前帖子也解釋過，主要是新廠產能爬坡和良率提升，以及費用控制較好，公司毛利率不斷提升的原因。

結論5：所以，今年四季度單個季度凈利潤增幅下降很正常，一個是三四季度凈利潤本來就相差不大，加上產能爬坡和良率提升的增效已經過去，另一個則是18年四季度基數太大。

順便說一聲，看得出來，深南電路產能爬坡、良率提升的效應在三季度已經體現出來了。

4、明年一季度或全年業績收入增幅

今年10月前華為已生產40萬5G基站，明年預計100-150萬，通信板增幅會進一步加大。特斯拉如果周產3000量，汽車板業務將開始復甦，但這面臨不良率提升的產能爬坡過程，毛利率會下降。

這裡就有很多假設量了，例如究竟新能源車裡汽車板價值是單車多少元（有說1萬的，有說2000的，現在普通汽車板才800每平米）。再比如明年華為到底能生產多少5G基站，從目前來看，國內建設進度明顯超出預期，以中移動為例，原計劃5萬，現在至少7、8萬了。再有公司產能能否跟得上，華為會不會給其他廠家份額？

我毛估估明年一季度38%以上，凈利潤100%以上，所以我才在上一個帖子里提到看好明年一季報。這個純屬個人預測，看看就行。
