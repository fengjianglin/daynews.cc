馬云：考試不能是教育唯一KPI 文憑只是學費的收據

2019-12-04 3,094條評論 4,568次閱讀 2069人點贊
精彩彈幕，盡在客戶端


澎湃新聞記者 吳雨欣

當地時間12月3日，經濟合作與發展組織(OECD)舉辦的「世界教育論壇」在巴黎召開，馬雲受邀分享對未來教育的思考。馬雲表示，今天討論教育的出發點，是希望孩子成為什麼樣的人。馬雲說：「我的第一個工作是老師，我希望我的最後一個工作也是教育。」

阿里巴巴方面人士表示，在現場，馬雲回憶自己高考多次失敗，最後大學只上了「四本」，但也很慶幸在社會這所大學裡從沒停止學習。對於教育，馬雲的建議是改變教育的KPI。

「如果考試是唯一的KPI，教育就不會是正確的方向。文憑只是學費的收據而已，真正的文憑是你為自己生活奮鬥中得來的，在自己七八十歲的時候得到證明。」馬雲在演講中指出，很多發展中國家，喜歡比大學的排名是多少。馬雲認為，應該比的是幼兒園、小學、中學、排名，請大家把更多的教育資源投到前方去。

很多高中生想讀大學，唯一的原因是大學是找到工作的保證。馬雲的經驗是： 「大學不保證你肯定有工作。我們招人不取決於你是不是哈佛，而是你願不願意學習。我們需要有那些真正思考的人，社會是真正的學校。」

在分享中，馬雲多次強調了老師的重要性。「我的英語是數學老師教的，那時候中國根本沒有足夠的英語老師，她上午去學ABC，下午就給我們上課。但是她跟我說，馬雲，你的發音很好，我倍受鼓舞。一個比我早半天學習英語的數學老師，她靠的不是知識，是智慧。」

馬雲說，教育就是未來，我們應該給老師最高的認可、最好的待遇，這樣他們就會尊重工作、尊重孩子。「好老師可以一輩子至少幫助200名學生，一個校長可以支持100個老師。我的第一份工作是老師，我希望我的最後一份工作仍然是從事教育。希望我可以和大家一起為教育的變革而努力。」馬雲說。

(原標題：馬云：應該給老師最高認可最好待遇，希望最後一份工作是教育)
