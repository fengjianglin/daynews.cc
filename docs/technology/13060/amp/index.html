狀元論幣：12月11日行情分析，陪我走過牛市的交易所關停了

天天要聞
2019-12-11

今天的要講的是陪我走過牛市的一家交易所，這家交易所叫做OTCBTC，它本月底即將關停。

我進入幣圈使用的第一家交易所是李笑來的雲幣網，後面由於「94」事件，雲幣網關停，用戶被李笑來推薦到OTCBTC。這家在台灣的交易所裡面巨大的用戶流量幾乎全部來自於中國大陸。

這家交易所的創始人叫鄭伊廷，人稱Xdite，她過去的技術經歷非常耀眼，2008年設計並搭建veryxd網站，2012年在黑客松比賽中獲獎，2015年開辦線上編程課程，2016年開始與李笑來合作並獲得大量關注。

2017年開始創辦OTCBTC網站，於10月26日上線，用戶流量大量來自於李笑來的粉絲，完成了初期的引流。

這家交易所主打的是場外交易，直接通過法幣完成對加密貨幣的撮合交易，也是當時我們作為小白用戶購買加密貨幣的最主要方式之一。

由於正處於大牛市期間，上線首日成交額就達到了38萬人民幣，三個月後日成交額最高達到1億人民幣。場外一年的交易額超過了100億人民幣。

正處於大牛市巔峰時期，怎麼能不割一波韭菜？不，是怎麼能不有所作為呢？

於是，Xdite發行了平台幣OTB，這個幣也是我的心頭之痛。

OTB是該平台的平台幣，發行量是2億個，僅僅只拿出2000萬個來分給早期的投資者，OTB只在該平台流通。

在發幣通告出來後，因為一些利益關係，李笑來公開發表微博宣稱：Xdite是個完全沒有信用的人。

Xdite也不甘示弱，撕逼李笑來，各種恩怨瓜葛、利益糾纏全部公佈於眾，一時間，看客也不知道孰真孰假，善惡難辨。

在OTB眾籌的時候，幣價大概是1.7元到2元左右，在公開交易之前，幣價已經被炒到了6元，這個時候OTB可以在站內互轉，像積分一樣。

開放交易當天，幣價直接衝到了10元，此時的各大群里集體狂歡，大佬們在群里振臂高呼：再不進場，我可要買光了喔！

誠如大佬們所說的那樣，大佬們瘋狂收貨，幣價每天上漲，數天以後，OTB達到了歷史最高價28元。

賣早了的韭菜紛紛拍大腿，只恨自己當初沒管住手。這個時候，群里再一次集體高潮：OTB上50元，成都吃火鍋！

我也想去成都吃火鍋，當然沒有賣出，幻想著跟群友在成都火鍋店觥籌交錯的場景。

幣價終於開始回落，跌到20元的時候，群里的韭菜們互相打氣：絕佳的建倉時期，再不買就沒機會了！

幣價跌到15元了，韭菜們更加瘋狂：梭哈！抄底！這個時候整個市場都在跌，沒人覺得這有什麼問題。而前不久振臂高呼的大佬們開始集中割肉，韭菜們開心「抄底」。

幣價跌到了7元，再一次腰斬，都已經跌了這麼多了，應該不會再跌了吧？也有人想著：哪怕跌得再狠也不賣！沒有信仰，如何賺錢？

然後幣價就從7元跌到3元，再跌到0.3元，繼續跌到如今的1分錢。

可悲的是，在下跌的過程中，Xdite在群里還不停給韭菜們打雞血：OTB年底800！團隊在憋大招，拿住別動！

無論怎麼喊韭菜去接盤，拿住不動，幣價還是從28元跌到1分錢，這就是OTB的歸零路。

而我的許多朋友，OTB的倉位高達數萬個甚至10萬個，按5萬個OTB算，最高持倉價格140萬人民幣，而如今的持倉價格是650元。

在幣圈，650元變成140萬非常困難，但是140萬變成650元卻非常容易，只要你選擇一個歸零幣大量買入，相信團隊在做事，拿住不動，別說變成650元了，就算是變成0元，也只是時間問題。

這次血的教訓告訴我：

不要相信某個項目方創始人的喊單，因為你跟他不在同一利益立場。

真正的好項目不需要喊單。你見過中本聰和V神喊單嗎？

不要對投資品種產生信仰和感情，信仰和感情只會讓你虧錢，成功的投資家對待投資品種都沒有感情。
