谷歌兩位創始人同時卸任 美媒：結束不平凡的20年

2019-12-05 260條評論 2,490次閱讀 824人點贊

參考消息網12月5日報道 美媒稱，美國谷歌公司兩位共同創始人卸任其母公司「字母表」公司的首席執行官（CEO）和總裁職務，從而結束了一個不平凡的20年。其間，拉里·佩奇和瑟吉·布林把在矽谷的一個車庫裡創建的一家初創公司打造成了全球最大、最強且最令人生畏的公司之一。

據美聯社12月3日報道，一直作為CEO領導了谷歌公司四年多的孫達爾·皮柴將兼任谷歌母公司「字母表」的CEO。這個職務此前一直由佩奇擔任。「字母表」尚未任命誰來填補布林的總裁職務空缺。

1995年，同為斯坦福大學研究生的佩奇和布林在相識後不久就創辦了谷歌公司。該公司最初只是對日益發展的互聯網業務進行編目分類，現在已成為全球最有影響力的公司之一。谷歌佔據網上搜索和數字廣告的主導地位，並開發了全球使用最廣泛的智能手機操作系統安卓。

但谷歌公司一直面臨來自主張隱私保護的組織和人士的壓力，他們指責谷歌收集和利用個人信息來投放廣告。它還面臨有關其濫用搜索和網上廣告的優勢地位排擠競爭對手的指控。

創新戰略諮詢公司的長期技術分析師蒂姆·巴賈林認為，該公司高管的人事變動不會帶來太大改變。

(原標題：谷歌兩位創始人同時卸任 美媒：結束不平凡的20年)
