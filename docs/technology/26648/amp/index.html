一男子詐騙FB、谷歌1.2億美元：被判5年

12月19日，美國司法部紐約南區檢察官辦公室發布消息，表示此前因詐騙Facebook與Google兩家公司1.2億美元（約合人民幣8.4億元）的立陶宛男子被判處5年有期徒刑，並處沒收罰款5000萬美元以及支付2600萬美元賠償金。該男子名為Evaldas Rimasauskas，他於2013年-2015年期間精心策划了一場詐騙。

他在拉脫維亞成立了一加公司，該公司與名稱與硬體製造商廣達電腦（Quanta Computer）相似。隨後該男子以廣達名義向大量公司與代理商發送郵件，要求這些公司匯款至全新銀行賬戶。因為Facebook與Google與廣達電腦定期會有數百萬美元的交易，所以這兩家公司就真的上當受騙，將多筆資金轉入該男子賬戶。

騰訊愛奇藝回應”慶余年遭非法銷售”：公安機關已立案

針對12月9日發現網上有人未經權利人許可，擅自通過互聯網非法銷售、傳播電視劇《慶余年》（第一季）盜版內容，大肆實施侵權盜版一事，騰訊影業公司回應稱，權利人已經向公安機關報案，要求追究相關責任人的刑事責任，公安機關已在立案偵查中。

2019最糟糕密碼榜單出爐：「123456」繼續霸榜

2019年最差的密碼排名出爐，還是由” 123456 “這個老面孔持續”霸榜”。據悉，SplashData 公布的最差密碼 100 強，是從密碼管理服務 SplashID、TeaMSID 和 Gpass 收集得來。今年排名前五位的最差密碼，分別是” 123456 “、” 123456789 “、” qwerty “、” password “和” 1234567 “。

雷軍：小米明年研發投入超百億

在第四屆（全球）楚商大會上，小米創始人、董事長兼CEO雷軍發表演講。他表示今年小米的研發投入預計會超過70億元，明年會超過100億元。

「我們堅持投入，因為這是在投資未來。有了人才才有創新和未來。」雷軍說。

阿里巴巴：穩定價格行動及穩定價格期結束

阿里巴巴20日對外發布公告稱，與全球發售有關的穩定價格期已於2019年12月20日（即遞交香港公開發售申請截止日期後第30日）結束。

另外，阿里巴巴表示，穩定價格操作人於穩定價格期內概無就穩定價格而在市場上買賣任何股份。

新一批國產遊戲版號下發：共53款，騰訊、網易在列

原國家新聞出版廣電總局官方顯示，2019年12月份最新一批國產網路遊戲審批信息更新。

本次新過審53款遊戲，騰訊《彩虹墜入》、網易《風雲島行動》等遊戲過審。
