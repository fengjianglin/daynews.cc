大病籌遭遇信任危機，那些“剛需”怎麼辦？

2019-12-20 1,692條評論 2,703次閱讀 1649人點贊

圖源網路

從一開始看到相關信息拋灑熱淚、慷慨解囊，到現在根本就不會點開，互聯網大病籌款經歷了「過山車」般的跌落。

據媒體報道，5年前，河南洛陽欒川縣小男孩程智斌查出罹患重型再生障礙性貧血，從那以後，全家過上了四處籌錢，四處求醫的日子。然而，不久前爆發的一則「水滴籌」負面新聞引爆了公眾信任危機，危機直接導致程父籌款半月，只籌到442元。

其實，類似「水滴籌」等大病籌款平台被公眾厭棄，並非自今日始，而是早有跡象。

從2017年網路籌款平台勃興，兩年時間，大病籌做得風生水起。2018年，民政部指定的20家互聯網募捐信息平台，募集善款總額超過31.7億元，同比2017年增長近三成。

不必否認，不少生活陷入絕境的民眾，從這種「眾人拾柴火焰高」的互聯網救助中得到救贖。但諸如信息失真、編故事乃至詐捐等問題也由此而生，籌款高頻次出現，某種程度上也是社會公眾愛心、同情心的透支。

魚龍混雜，泥沙俱下。眾多求助者湧向各種互聯網籌款平台，一方面是確有需要，就像欒川的程智斌，動輒幾十萬的醫療費用，這個農家不可能拿的出來，網路籌款實屬不得已為之。但另外也不排除跟風的，比如原本有支付能力，但看到網籌這麼方便，就跟著湊熱鬧。

尤其是當愛心救助與平台的籌款指標，與「地推」人員的績效考核掛鉤時，則公益和商業的界限必然變得模糊起來。

目前，整治、規範、重新設定規則等等，都沒有問題，只是經過一次次的信任退潮，愛心也會變得越來越微茫，那些真正有需要的大病家庭究竟該怎麼辦？

這一問題並非僅僅出現在「大病籌」，也出現在其他方式的籌款行為中。可以說，時至今日，我們都沒有解決好籌款中的信任問題。信任稀缺是一個社會問題，也是一個時代性難題，並非僅僅指向新業態、新經濟。

很多網友呼籲政府監管，幾乎每一次遭遇信任危機都會這麼說。事實上，通過行政力量「管」起來，未必會比社會互助更讓人放心。

或者說，大病籌為什麼會迅速擴展開來？除了互聯網本身的凝聚力外，也與社會公眾求助無門、官辦機構力有不逮有很大關係。

也因此，當下的急務，不是一竿子把各種籌款平台打死，而是要充分發揮平台的自救功能。要知道，互聯網的最大好處恰恰在於它具有自行甄別的功能。信息失真？好吧，那就擴大信息公開，讓網友參與監督；詐捐？也好辦，既然能獲得救助，也就能辯明真相。

那些在幽暗處唉聲嘆息的大病患者，他們是大病籌的「剛需」，他們需要網聚起來的「正能量」，需要眾人都來「添柴」。江湖救急，一根火柴能夠點亮一個生命。

當然，從長遠看，還是要通過深化醫療體制改革，完善大病保障，讓每一個國民都能需求到制度性的救助，這才是根本之道。

(來源:澎湃新聞)
