中國移動專家：5G天線更多融合、更深覆蓋、更加智能

2019-12-05 201條評論 1,266次閱讀 258人點贊

C114訊 12月5日消息（九九）在日前舉行的2019中國信息通信大會上，中國移動通信研究院專家曹景陽表示，5G商用牌照頒發後，中國移動宣布將於年底在超過50個城市實現5G商用服務；2020年將進一步擴大網路覆蓋範圍，在全國所有地級以上城市提供5G商用服務。SA將成為5G網路目標架構，2020年將面向智能手機用戶和行業用戶實現SA商用。

曹景陽指出，5G已成為當前基站天面建設的主要驅動因素。一段時間內，4G網路流量仍將保持快速增長，但後續主要依靠5G反向開通來滿足；為支持5G天線建設，中國移動近年來天面整合的力度不斷加大，4488天線、4+4天線等多頻天線獲得了大量應用，與之對應的是寬頻非電調天線應用大幅減少，天線多頻化、電調化的進程明顯加快。

天面整合替換將是後續傳統基站天線的重要應用方式。現網中大量存量的單頻天線，如FAD智能天線等，為天面整合提供了條件；經過NB/FDD的建設，900/1800頻段也已快速形成了應用規模。此外，天面的整合策略與鐵塔租金政策緊密相關，運營商需要基於網路實際需要選擇最優的整合方式。

在5G天線需求方向上，曹景陽重點提到以下幾點：

更多頻率，更高融合

D頻段（2.6G）擴展至160MHz，目前產業能力已支持，天線成本基本不變或略增加；新頻段4.9G可用於容量補充等，受限於天面空間無法獨立建設時，可考慮與其他AAU或無源天線融合部署；AAU還可與傳統無源天線進一步融合，但如何滿足迎風面積和重量的限制將面臨不小的挑戰。現有天面將持續整合，目標是在天面數不增加的前提下建設5G。對4488等多頻天線，還需要在保持性能指標的前提下提高集成度，縮短天線長度以便利工程安裝。

更多場景，更深覆蓋

對於城區需要兼顧覆蓋和容量，初期將主要採用MassiveMIMO形式，其創新方向主要是降低重量和成本；對於農村以廣覆蓋低容量場景為主，仍可採用傳統多通道天線；對於新建高鐵，可引入8通道天線，通過掃描能力解決水平或垂直零點問題；對於公路可採用寬波束天線，通過解決垂直零點問題提升線路覆蓋；對於地鐵隧道，可考慮漏纜、對周和貼壁天線等方式滿足容量需求。

更多功能，更加智能

對於宏站場景，AAU可支持工參測量，為智能化網優創造條件；對於大型公共場所等室分場景，天線可與藍牙技術結合，在滿足覆蓋要求的同時提供精準定位和IoT相關業務能力，滿足5G+X發展戰略。曹景陽進一步指出，目前5G高性能室內產品成本較高，運營商可基於4G的建設經驗及產業實際，考慮規劃系列化站型，進行精細化室內建設。
