12月最大地震在陸地“顯現”，播報震級6.8，1個半小時內餘震6次

2019-12-15 1,735條評論 4,042次閱讀 603人點贊

趣味探索訊 毋庸置疑，21世紀的每一天，地球人類都在進步，尤其是在人工智慧和航天科技領域，可以用「日新月異」來形容最為確切。日本一家好公司已經生產出能陪伴超級宅男的「女性機器人」，波士頓機器人學會「左右翻飛」等多種雜技，美國旅行者1號飛船來到了212億公里外，中國玉兔2號已在月球馮卡門撞擊坑行走了345.059米。

這些都證明人類科技水平已經達到了一個新高度水平，似乎不日將會殖民火星、甚至未來離開太陽系，然而事實上一些經常發生在我們身邊的突發事情卻一直無法做到預測，比如地震，沒有人能預測地震發生的準確時間，也沒有人能預測地震發生的精確位置。地震記錄數據顯示，地球每一天都有地震發生。就在2019年12月15日，12月最大地震顯現了。

根據中國地震台網最新報道顯示，北京時間2019年12月15日14:11:51，菲律賓棉蘭老島陸地顯現一個大地震，震級為6.8級，具體位置為北緯6.55度，東經125.3度，震深為30公里。同時美國地質調查局（USGS）對該地震進行了報道，不過跟中國地震台網播報數據略有不同，該地震震級為6.8級，深度22.4公里，位置是在菲律賓麥格塞塞以南6公里，坐標為北緯6.708°、東經125.188°。

雖然這並不是2019年12月份首個地震，但卻是本月最大一個地震，並且發生菲律賓以南島嶼陸地上，所以USGS對此尤為重視。USGS報道，距離震中約6.6公里的達沃省麥格塞塞居住著16592人，距離震中位置約9.1公里的班薩蘭居住著21391人，距離震中位置約18.3公里的歸興泊泊居住著12466人，距離震中約18.8公里的帕達達住著11439人。

慶幸的是，距離震中位置10公里以內範圍內只居住著3萬多人，人口還不算太密集，所以該地震影響應該不是很大。此外，USGS還對該地震是否產生海嘯、經濟損失情況、搖動強度等進行估算和跟蹤。USGS還記錄了，6.8級大地震後，接下來的1個半小時內出現了6次餘震，最大餘震級別為5.7級，震深為10公里，具體坐標位置為北緯6.527°、東經125.299°。

據了解，菲律賓海板塊東接壤好太平洋板塊，北接接壤歐亞板塊，還其它較小板塊，所以在科學中一直被看成是與眾不同的，並且又處在環太平洋火山帶上，所以是一個地震多髮帶。有趣的是，2019年12月份一共出現了4個6.0級以上大地震，根據我國地震網統計數據顯示，分別出現在安德烈亞諾夫群島，智利北部沿岸近海，湯加群島和萬那杜群島。
