特斯拉直接被砍30萬元 二手電筒動汽車為何賣不動？

2019-12-16 668條評論 2,378次閱讀 1442人點贊

「賣家不想賣，車商不願收，買家不敢買」，新能源二手車正在陷入這樣的怪圈。

「打算買新能源二手車的客戶大多屬於定向尋找，像特斯拉這樣的高端品牌是他們的首選，但低端電動車型我們都不敢收，因為根本賣不出去。」日前，北京花鄉二手車交易市場的一位商戶告訴每日經濟新聞(博客,微博)（微信號：nbdnews）記者，現在買新能源二手車的人還是少數，收太多車他們的資金也承受不了。

儘管國內新能源汽車的保有量早已突破百萬輛，但在二手車市場，新能源汽車仍以「小眾選擇」的身份存在著。日前，記者在調查走訪中發現，「有賣方沒買方」仍是新能源二手車市場的常態。而對二手車商來說，現金流就是「生命線」，即便是低價出售，也不能把車砸手裡。在這種情況下，新能源二手車的保值率令人堪憂。

「半價收的電動車成了鎮店之寶」

公開數據顯示，二手燃油車「上架」後，當月實現成交的比例超過40%，絕大多數二手燃油車都可以在兩個月內完成交易，但新能源二手車「上架」後，當月出售的比例還不足10%。

「今年年初我收了一輛電動車，當時是半價收購，結果8個月都沒賣出去，快成『鎮店之寶』了，最後賠了幾萬塊錢才出手。」北京花鄉二手車交易市場的一位商戶向記者抱怨。

該商戶坦言，新能源二手車之所以難賣，主要是因為其續航里程不好把握，消費者無從檢驗。再加上新能源汽車在續航和技術方面的更新迭代速度較快，這也讓更多的消費者把目光投向了新車。

圖片來源：每經記者 趙成 攝

「對二手燃油車來說，客戶往往可以通過行駛里程來判斷動力系統、底盤等車輛情況，但對新能源二手車來說，普通消費者如果僅通過行駛里程很難判斷出車輛動力電池的衰減程度。」上述二手車商戶對記者說。

一位出售二手新能源汽車的商戶告訴記者：「我們收購車輛也有標準，行駛里程在6萬公里以上的車輛，一般不收。因為二手新能源汽車的利潤很薄，買車的客戶多數是用來跑網約車的，我們要向用戶承諾車輛的續航里程，而行駛里程在6萬公里以上的電動車，動力電池衰減太嚴重，無法保證日常的正常使用，如果重新更換動力電池，價格要高出車輛本身的收購價，得不償失。」

據記者了解，一輛新能源汽車過質保期後，如果要更換電池，費用預計在4萬元左右，這與二手車市場上部分新能源車的價格基本一致。

二手特斯拉先降30萬再收

中國汽車流通協會發布的《2019年11月中國汽車保值率報告》顯示，三年車齡的純電動新能源車平均保值率為36.4%，大大低於傳統燃油車。而在新能源車保值率排行榜上，只有特斯拉Model X車型（三年車齡）的保值率達到63.9%；大量國產品牌新能源車（三年車齡）的保值率均低於50%。

另據中國汽車金融暨保值率研究委員會發布的《2019中國汽車保值率報告》，從新能源汽車的1年保值率情況來看，排在前五位的是特斯拉Model X、特斯拉Model S、別克VELITE 5、沃爾沃S60L和比亞迪宋DM，保值率分別為74.98％、71.42％、60.67％、58.43％、58.05％。這意味著，除了特斯拉以外，其餘新能源車型的保值率普遍在60%以下。

圖片來源：每經記者 蘧毛毛 攝

有二手車商戶向記者稱：「我們在收車時即便是像特斯拉這樣的高端電動車品牌，也不會先看車輛使用年限和行駛里程，直接將收購價在官方指導價上降30萬，再根據車況進行價格調整。」

通常來說，影響車輛保值率的因素主要包括新車價格浮動、品牌口碑、保有量、車輛的使用性能、實用性、養車維護費用和便利程度等。但對於二手新能源汽車來說，品牌溢價和技術實力決定著車型的保值率。

「一般情況下，品牌越有歷史底蘊和群眾基礎，品牌的溢價能力就越高，同時車輛性能和新車終端價格也影響著二手車的價格走勢。」中國汽車流通協會副秘書長羅磊告訴記者，隨著續航里程、智能化程度的提升，二手新能源汽車的殘值將下降。

眼下，新能源二手車已陷入「賣家不想賣，車商不願收，買家不敢買」的怪圈。也正因如此，目前推動二手新能源汽車交易的主體實際上是汽車企業。

據了解，北汽新能源2016年就提出以市場售價的45%—55%回購某一時期的新能源汽車。雲度汽車也於2018年提出購買3年、行駛6萬公里以內的新能源二手車，回購價格為補貼後車價的50%。同時，比亞迪、東風日產等也都有對自家新能源車型的回購計劃。特斯拉、蔚來、威馬更是直接在官網上進行認證二手車的銷售。

有業內人士表示：「車企通過此舉，無非是希望用戶繼續購買同一品牌的車升級換代，成為『回頭客』。另一方面，殘值率目前仍是車企品牌力的客觀反映，車企不希望因殘值問題影響到消費者對本品牌新車的購買決定。」

評估標準有望明年初落地

根據《汽車產業中長期發展規劃八大重點工程實施方案》，到2025年，我國新能源汽車年銷量將達700萬輛，新能源汽車保有量將達2000萬輛。如果能實現如此大規模的保有量，意味著將會有更多的新能源二手車流入市場。

記者了解到，目前針對二手新能源汽車的評估體系幾乎還是空白，車商收車基本是按照二手燃油車適用的《二手車流通管理辦法》和《二手車鑒定評估技術規範》對二手新能源汽車進行殘值評估。這難免會讓二手車商缺少可參考的估價案例，導致車商對於二手新能源汽車胡亂報價或拒收現象頻發。

圖片來源：視覺中國(000681,股吧)

「2018年以來，中國汽車流通協會開始推動《新能源乘用車二手車鑒定評估技術規範》團體標準的制定，其中包括新能源二手車鑒定評估機構條件要求、鑒定評估程序、技術狀況鑒定要求等，並在各相關單位以及科研、學術機構的支持下，經過三次討論，形成了第三版討論稿。」羅磊向記者透露，「《新能源乘用車二手車鑒定評估技術規範》有望在明年初正式出台。」

在二手車行業專家王萌看來，上述評估技術規範的出台，只是在政策層面對二手新能源汽車的產品質量進行了約束，新能源二手車市場目前還是待拓荒狀態，歸根結底是由於新能源車的保有量還不夠。

從今年前11個月的銷量數據來看，我國新能源汽車產銷量正在快速增長。中汽協數據顯示，1-11月，我國新能源汽車產銷分別為109.3萬輛和104.3萬輛，同比分別增長3.6%和1.3%。

「目前新能源車的保有量近300萬輛，而二手燃油車市場爆發時，保有量已達8000萬輛，什麼時候新能源車的市場保有量比例達到20%，二手新能源車市場才有望發展起來。」王萌說。
