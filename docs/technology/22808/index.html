阿里巴巴在市值沒有超過亞馬遜，利潤反而比亞馬遜高很多？

2019-12-20 2,121條評論 4,532次閱讀 75人點贊

阿里巴巴和亞馬遜相當於完全不同的模式，國內京東倒是特別像亞馬遜的方式，亞馬遜售賣的產品幾乎都是自家代理，相當於亞馬遜平台屬於售賣產品的品牌，大部分的品牌都屬於自家代理，這樣意味著作為平台一方需要做的事情非常巨大，需要對應產品的質量以及價格都要管控到位，而且還要安排相應的人員提供支持，這種模式有點類似於線下的門店方式；阿里巴巴構建的電商平台雖然也是在網路上售賣東西，但是沒有一家品牌屬於阿里巴巴自家的，阿里巴巴只是負責搭建一個平台，只要符合標準的商家都可以入駐到這個平台。

阿里巴巴更像是一個純正意義的商場，至於在裡面開店，阿里巴巴就要收取租金，由於屬於虛擬的商店，最大的成本在於後台技術的運維人員以及相應的產品運營團隊，但是商品店可以低級本的增加，只要伺服器的負載能力足夠，所以錢來的就會特別容易，相當於搭建了一個流量平台，吸引眾多的商家入駐，商家想要自己的產品能夠出現在排行榜中，就可以給阿里平台一定的費用進行前排的宣傳，同時一個不是很出名的品牌如果在阿里巴巴平台銷售非常給力，那麼就會贏得高的推薦，同時還能把自己的品牌給宣傳出去。

阿里巴巴當初的夢想是讓天下沒有難做的生意，從這個角度上阿里巴巴的發展潛力也是非常巨大，亞馬遜屬於前期的投入比較大，後期的收益也會比較穩定，兩種不同的方式直接決定了利潤的差異，阿里巴巴主推的雙11活動最大的贏家不是各個商家還是阿里巴巴，很多商家都是接近成本的方式在銷售，但為了銷量還是給阿里巴巴的平台一定的宣傳費用，這種輕量級的平台投入方式放眼在全球全面鋪開的情況下，會比亞馬遜更加有潛力。

目前無論是阿里巴巴還是亞馬遜企業都不是單一的電商公司了，旗下有多種業務摻雜在一起，由於模式的不同直接決定了利潤的取向，阿里巴巴的劣勢在於推向全球的過程中，由於很多國家對於中國的科技企業還是存在一定的誤解，現在的阿里巴巴主要業務還是在國內， 只是做到了讓國內的廠家在這個平台上開花結果，但從全球範圍內推廣力度看，差距還是非常大，可以想像如果阿里巴巴做到亞馬遜在全球的範圍，市值上一定會超過亞馬遜，這都需要時間來積累。

雖然模式不同，但在很多業務上有著很多的相似之處，雲計算在兩家企業中都佔據著非常重要的戰略地位，而且兩家企業在技術上都有大量的投入，目前都是技術厚重感極強的企業，團隊的戰鬥力都很強，只要是亞馬遜接手的產品或者項目都是能迸發出很強的戰鬥力。阿里巴巴在國內互聯網公司裡面的薪資和加班強度在業內也是出名的，由於戰略目標不一致造成利潤點不一致，從發展的角度看現在的亞馬遜領先於阿里巴巴，阿里巴巴還要為出海進一步努力奮鬥，希望能幫到你。
