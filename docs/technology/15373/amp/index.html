攜手探索星辰大海 共同邁向智能互聯——2019年全球科技發展呼喚開放合作

天天要聞
3 天 ago

新華社北京12月15日電（記者張瑩）人類首張黑洞圖像面世，全球開始進入5G商用部署階段，嫦娥四號首次實現人類探測器在月球背面軟著陸……歲末回望，2019年全球科技發展亮點頻現。

不僅僅是重大科技進展，各國科研人員展現的合作精神同樣引人注目。他們攜手探索星辰大海，共同邁向智能互聯，為全球科技合作寫下生動註腳；他們重申開放合作的重要性，有力回應科技領域「脫鉤」「築牆」等雜音。

攜手實現共同夢想

在今年多項科技突破與進展中，國際合作發揮了不可或缺的作用。比如，4月面世的人類首張黑洞圖像，來自「事件視界望遠鏡」項目。該項目利用分布世界各地的8個射電望遠鏡，彙集了全球多國超過200名科研人員的努力，終於完成「上一代人認為不可能做到的事情」。

人類史上首張黑洞照片。（新華社發/「事件視界望遠鏡」項目組供圖）

「不同文化、不同機構、不同國家和大洲（的科研人員）走到一起合作並不容易，但如果有共同願景的驅動，有首次看到黑洞的共同夢想，合作就變得可能。」項目科學委員會主席、荷蘭奈梅亨大學教授海諾·法爾克說。

2019年適逢人類首次登月50周年。50年前的登月將人類太空探索推向一個高峰，50年後人類航天探索迎來新變局：商業航天興起、中國日益成為世界航天合作的重要力量……

中國嫦娥四號在1月實現了人類探測器首次月背軟著陸，這個重大項目沒有忽視國際合作：嫦娥四號搭載了來自荷蘭、德國、瑞典和沙特等國的科學載荷；美國航天局根據中方提供的信息，使用其月球勘測軌道飛行器對著陸點成像。

圖為嫦娥四號著陸器彩色全景圖。（新華社發/中國國家航天局供圖）

這一年的中國貢獻還體現在助力全球進入「5G商用元年」。全球移動設備供應商協會數據顯示，到10月底全球已有超過20個國家開通5G商用服務。其中，英國、西班牙等國首次開通的5G服務都部分使用了中國企業的設備，地中海畔的「袖珍國」摩納哥還在中國企業成套設備幫助下成為世界首個全境覆蓋5G的國家。

2019年7月23日在摩納哥電信公司營業廳里展示的支持5G的智能手機華為Mate20X（左）和小米MIX3。（新華社記者高靜攝）

呼籲破除共同障礙

世界知識產權組織11月發布的2019年《世界知識產權報告》指出，全球創新活動呈現出合作日益緊密、更加國際化的特點。知識產權組織總幹事弗朗西斯·高銳表示，當今的創新格局在全球範圍內緊密關聯，面對共同的全球挑戰，各經濟體必須在追求創新方面保持開放。

然而，個別國家卻發出單邊主義、保護主義和「反全球化」的雜音。面對試圖阻撓開放合作大勢的言行，世界越來越多有識之士發出客觀、理性的聲音，予以有力回應。

在全球5G建設中，個別國家針對通信設備企業所在國貼標籤、設限制，還試圖干擾其他國家在5G設備上的選擇，把技術問題政治化，給各國築起藩籬，阻礙5G技術聯通全球、造福人類。

10月，在美國華盛頓舉行的第70屆國際宇航大會上，中國代表團因簽證受阻而缺席開幕式。現場聽眾通過手機投票將一個問題在會場大屏幕上置頂：「在這個會議上，我想念一個重要的航天局。中國（航天局）去哪兒了？」

科學技術是世界性的，打破「圍牆」、集合全球智力才能更好推動科學發展。俄羅斯齊奧爾科夫斯基航天研究院院士亞歷山大·熱列茲尼亞科夫認為：「國際航天合作可顯著降低『孤軍』探索的投入，多國聯合實驗所取得的成果有助在更廣泛領域催生重要發明。」

個別國家在國際科技合作領域頻繁製造障礙，其源頭是不能接受別國科技快速發展，試圖以「脫鉤」來保持領先。這與開放合作的時代大潮逆勢而行，引發學界憂慮。美國60家科學組織在9月聯名發表公開信，呼籲美國政府停止打壓外國科研人員和阻礙國際科學合作，認為這將損害科學事業。

合作為了共同命運

仰望星空，加強國際合作探索星辰大海，關係人類未來命運。中國空間站計劃在2022年前後完成建造，今年6月，中國就宣布17個國家的9個項目成為中國空間站科學實驗首批入選項目。這生動詮釋了多邊主義，充分體現了開放包容，是推動構建外空命運共同體的鮮活寫照。聯合國外層空間事務辦公室主任西莫內塔·迪皮波說，這是「一個偉大範例」。

腳踏實地，人類已可通過5G等通信技術智能互聯，形成更加緊密的整體。回想3G時代，全球移動通信擁有3個標準，4G時代變為2個，而5G時代全球形成了統一的國際標準。中國工業和信息化部部長苗圩在2019世界5G大會上表示，未來開放合作仍然是推動移動通信發展的全球共識，中國歡迎全球企業、研究機構共同建設好5G網路，共享5G發展成果。

己亥秋末，第二屆世界頂尖科學家論壇在上海舉行。全球近70位諾貝爾獎、圖靈獎、沃爾夫獎、拉斯克獎、菲爾茲獎得主，10餘位中國兩院院士，與來自全球的100多名優秀青年科學家，發出《上海倡議》，表示要不懈努力以期共同實現「科技，為了人類共同命運」的願景。

這份倡議說：「開放、合作和科技共同體的觀念，仍然是支撐科學發展的決定性基礎。……跨國界、跨地域、跨學科、跨文化的開放性合作，這在過去、現在和將來都將是一面高高舉起的旗幟。」（參與記者：欒海、劉芳、劉曲、周舟）
