阿里系的菜划算，為啥“投靠”騰訊？

天天要聞
2019-12-18

你能想像么，一款阿里系的產品居然只上線了微信小程序，而沒有上線支付寶小程序？

這款產品就是最近出現在大家視野的生鮮電商「菜划算」，「菜划算」在今年5月獲得了熊貓資本投資後，7月默默推出了APP和微信小程序，雖然表面上看與阿里並無關係，但是曉程序觀察（yinghoo-tech）還是在菜划算的股權結構中，發現了一些蛛絲馬跡，它背後實際的控股方，其實就是阿里。

  • 在菜划算的股權結構中，杭州短趣網路傳媒技術有限公司占股70%，而杭州短趣網路傳媒技術有限公司由阿里和天貓全權控股，為阿里旗下產業，曾運營阿里趣拍雲相關業務。

  • 占股20%的方傑，曾在阿里任職，任美菜網COO。
  • 此外，菜划算官網顯示的公司地址為杭州市濱江區阿里中心。

種種跡象表明，這就是阿里親自下場推出的社區生鮮業務。

這不禁讓人疑惑，為何阿里的「兒子」，竟去給騰訊「打工」了？我們不妨就社區生鮮電商的應用場景，看看「菜划算」為何選擇了微信小程序，而不是支付寶小程序。

前置倉的弱點只有微信小程序可解

想要找到這個問題的答案，還是要從「菜划算」產品中，去一探究竟。

菜划算採用前置倉模式入場，即根據社區選點，建立倉庫，覆蓋周圍社區，由總倉配送至前置倉，再由前置倉直接配送到家。與每日優鮮和叮咚買菜一樣，菜划算並沒有設立線下門店，而是通過在社區設立前置倉儲站點，覆蓋站點附近社區1-3公里，每當覆蓋區域用戶下單，工作人員在前置倉進行配貨，騎手在29分鐘內完成送貨到家服務。

由於沒有線下門店，每個前置倉的單店成本非常低，又能夠快速方便的送達到用戶手裡，對於社區生鮮電商來說，確實是個不錯的選擇。但也正因為沒有線下門店，用戶無法通過門店觸達，線上成為主要的拉新方式。

「菜划算」也正在通過各種線上渠道進行前期的用戶積累。目前，菜划算通過邀請獎勵、限時搶購、爆品推薦等增加日活和拉新效率，主打29分鐘新鮮配送到家的便利性，滿足用戶做飯時即時性的需求。

對於「菜划算」來說，拉新最重要的一點，就是基於LBS定位，快捷、精準找到周邊用戶。

對比微信小程序和支付寶小程序，我們發現，微信小程序中的「附近的小程序」能力可以向用戶推薦周圍5公里內的小程序，剛好能夠滿足「菜划算」的拉新需求。隨著微信小程序的不斷發展，「附近的小程序」也逐漸形成了一個更加開放的本地化生活平台，更契合「菜划算」類的本地服務商家的定位，幫助它們快速、精準的找到用戶。

當然，對於「附近的小程序」來說，菜划算也是為它「正名」的一個案例。「附近的小程序」作為微信小程序中與「我的小程序」並肩的板塊，剛推出的時候便受到很大爭議，一度被認為是為線下門店所打造的，如果沒有線下門店的小程序，便不會被推薦。

菜划算的案例證明了，「附近的小程序」之所以能夠被放在如此重要的戰略地位，其原因就在於隨著服務的不斷線上化，平台推薦不再以商家為主體而是以用戶為主體。從「人找服務」不斷升級為「服務找人」，人在哪裡，服務就在哪裡。

而支付寶小程序中卻沒有基於位置服務推薦的能力。雖然支付寶小程序的的拉新能力也非常強，設立了首頁Banner和平台推薦等板塊幫助商家拉新，但對於「菜划算」類的前置倉玩家來說，即便獲得了首頁推薦，恐怕也無法精準觸達到服務區域內的用戶。

不過對於支付寶來說，位置服務應該不是什麼難事，據曉程序觀察（yinghoo-tech）了解，目前支付寶與高德地圖已經實現了端對端的打通，而高德地圖一直是一個深耕LBS位置服務的玩家，甚至在阿里經濟中，支付寶與高德地圖的打通，也被稱為是一個典型樣本，但至於為什麼支付寶小程序沒有推出類似的位置服務，我們就不得而知了。

高復購、高轉化的熟人經濟

除了「附近的小程序」的拉新能力之外，對於「菜划算」來說，微信中熟人經濟背後帶來的高轉化與高復購也是一個強大的吸引力。

事實上，已經有不少生鮮電商玩家，在微信生態里，利用微信小程序工具，完成了低成本獲客和極高的復購率。以社區團購最早的玩家之一：「你我您」舉例，其便是通過主打社區中的熟人社交，3年時間實現覆蓋30個城市，月復購率和月留存數據已經做到了83%和67%。而一般傳統電商這兩項數據僅為15%和8%，數據遠遠高於其他賽道上的同行。

其實「你我您」的玩法並不複雜，簡單的來說，就是團長負責自己居住的小區，通過微信群和微信小程序向用戶推薦商品。「你我您」負責源頭集採集配，省略所有中間環節成本，直接配送到小區。

重要的是，在微信生態中，「你我您」良好的利用了微信小程序和微信群兩個工具：

  • 微信小程序，可以流程標準化，採購、宣傳、購買、配送、售後等全鏈條的效率和成本得到極大提升。加上整體環境因素，在一年內實現了滾雪球式的增長，最終由鬆散的小生意升級為一個大商業機會。
  • 通過微信群，與小區住戶快速建立連接和互動，利用特定場景下的熟人關係來提供實惠、便利、到家的社區團購服務。

利用微信小程序通過微信群滲入鄰里鄰居組成的碎片化關係網路，在微信群中完成分享、拉新、高轉化、高復購。

相比於社區團購，前置倉有更無法代替的優勢，那就是能夠滿足用戶的及時性需求。「你我您」最快的配送速度為「次日達」，尚且能夠完成如此高的效果，對於「菜划算」來說，未來的想像力會更大。

事實上，在阿里生鮮盤子中，除了外部扶持「菜划算」進入賽道，內部還在不斷培養「盒馬鮮生」、「淘鮮達」等玩家不斷增加自己的實力，可見阿里對於生鮮賽道的重視。但是目前「菜划算」還需要藉助微信小程序的力量，完成最重要的用戶閉環。

而這些在支付寶小程序中就一定做不成么？也未必。對於支付寶來說，雖然目前已經實現與高德地圖的端對端打通，但在支付寶小程序中，卻連高德最擅長的位置服務都沒有。打通後如何進行有效聯動，依舊是支付寶甚至是整個阿里經濟體，未來要面臨的命題。
