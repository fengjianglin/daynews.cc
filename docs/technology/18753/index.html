保值率最高的3款新能源車，不怕二手車販子不要

2019-12-17 1,184條評論 4,999次閱讀 3459人點贊

和盛行了百年的燃油車相比，新能源汽車在保值率方面一直是令人詬病的存在，很多人不買新能源汽車，很大一部分顧慮就在保值率上面，大部分新能源汽車保值率低到連二手車販子都不敢接手。

雖說新能源汽車的保值率普遍低，但並不是百分之百。在二手新能源車裡面，還是有幾個能拿得出手的好貨色的，比如以下這3款，屬於新能源車裡邊，保值率TOP前3的。所以買新能源車除了膽大嘗鮮，也要有個好眼神。

TOP 3：吉利博瑞GE

三年保值率：43.5%

博瑞GE是吉利旗下的混動旗艦車型。近日，2020款吉利博瑞GE車型正式上市。新車共推出了搭載三種動力的8款車型，售價區間為13.68-20.98萬元。作為年度改款車型，2020款博瑞GE保持了現有車型的大部分設計，但針對外部細節、動力和配置等進行了優化升級。

2020款博瑞GE新增了搭載1.8T直噴發動機與6AT變速箱的1.8T自動領航版，以及與之對應的插電混動車型——PHEV領航版，進一步豐富了消費者的選擇，同時全系車型也均滿足國六b排放標準。

配置上，2020款博瑞GE提供全景天窗、主駕電動座椅、LED大燈、博世ESP 9.3車身穩定系統、GKUI吉客智能生態系統、倒車影像、無鑰匙進入+一鍵啟動等，同時還配備有L2級自動輔助駕駛。

TOP 2：比亞迪唐DM

三年保值率：51.5%

如果要從國產新能源品牌找一個比較「能打」的選手，那比亞迪是首當其衝。這其中，比亞迪旗下的唐DM是個狠角色。從它三年保值率51.5%的成績說明市場對它還是很認可的，而且這個成績排在所有新能源車型的第二名！

唐DM頂配車型百公里加速4.3秒，其他車型為4.5秒。頂配和次頂配車型純電續航為100公里，其餘三個車型為80公里。其性能數據，不僅是同級最強，在目前國內插混車型中也是頂級的，且可能將在相當長時間內具有領先優勢。

TOP 1：特斯拉 Model S

三年保值率：60.3%

在新能源領域裡，特斯拉的說一不二的老大。雖然特斯拉的車型最便宜也要40-50萬，很多甚至都是百萬級別以上，而且，特斯拉發展至今已經十幾年了，品牌已經都打出去了。加上其電動車一哥的地位，理所當然，特斯拉旗下車型保值率一直都很高，特別是Model S。

Model S的三年保值率基本還能達到60.3%左右。這輛在2009年正式亮相，2012年開始交付的純電車型，至今也是走過了7-8年的時間，目前特斯拉的車在北美的保有量最大，而在國內，其實也有一定的「粉絲」基礎了，保有量也還算可觀。再加其品牌溢價，故Model S的保值率一直處於特斯拉車型中的No.1。當然了，也是所有新能源車中最高的。
