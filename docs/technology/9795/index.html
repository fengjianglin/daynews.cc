11月燃油車銷量大漲44% 比亞迪已經觸底反彈？

2019-12-09 543條評論 1,997次閱讀 802人點贊

驅動中國2019年12月9日消息 日前，比亞迪官方發布了11月旗下汽車銷量數據。總體上來說，新能源汽車銷量下滑明顯，儘管燃油車銷量有所上升，但還是被新能源拖累。11月比亞迪公司整體銷量4.13萬輛，環比增長0.4%，同比下滑19%。1-11月累計銷量41.82萬輛，同比下滑7.3%。

主要的問題出在新能源汽車部分的銷量。11月新能源汽車銷量1.12萬輛，同比下滑62.7%，環比下滑8.1%說明現在比亞迪新能源車銷量下滑幅度正在擴大。1-11月累計銷量21.64萬輛，累計同比增長7.6%。其中新能源乘用車銷量1.07萬輛，銷量同比下滑62.9%，下滑幅度環比下滑9.1%。新能源商用車銷量545輛，銷量同比下滑59.2%。純電動乘用車銷量8000輛，同比下滑47.9%；插電式混合動力乘用車銷量2,675輛，同比下滑80%。

不過比亞迪的燃油車部分增漲喜人。11月比亞迪燃油汽車銷量3.01萬輛，同比增長43.9%。轎車和SUV銷量均實現高增長。具體車型上，11月轎車銷量6,869輛，同比大幅增長81.7%；SUV銷量2.05萬輛，同比大幅增長188.7%；MPV銷量2,727輛，同比下滑72.8%。隨著汽車行業整體銷量觸底反彈，比亞迪燃油車銷量有望進一步提升。

上個月比亞迪與豐田簽署協議合資成立純電動車研發公司，雙方展開深度合作，未來產品將進一步融合豐田在品質及安全等方面的技術優勢。廣州車展上，比亞迪最新一代雙模DM3.0和智能網聯車機DiLink2.0亮相。隨著補貼退坡的影響逐漸減弱，憑藉領先的新能源車技術和完整的產品體系，比亞迪新能源車銷量有望保持行業領先。
