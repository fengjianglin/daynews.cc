2019全球超級計算機排名TOP500完整版中國神威“太湖之光”第三

2019-12-23 1,013條評論 3,990次閱讀 2340人點贊

2019年11月全球超級計算機排名最新榜單，之前四屆top500排名中，我國的神威 「太湖之光」超級計算機都是冠軍，2019全球超級計算機排名中國神威 「太湖之光」第三。

2019年11月全球超級計算機排名最新榜單前10名

No.1 Summit（美國）

美國 製造商：IBM

處理器：2,397,824 個；峰值速度：200795 TFlop/s

簡介：

Summit是IBM和美國能源部橡樹嶺國家實驗室（ORNL）推出的新超級計算機，Summit 要比神威·太湖之光快 60%，比同在橡樹嶺實驗室的 Titan——前美國超算記錄保持者要快接近 8 倍。而在其之下，近 28,000 塊英偉達 Volta GPU 提供了 95% 的算力。

No.2 Sierra（美國）

美國 製造商：IBM

處理器：1,572,480 個；峰值速度：125,712 TFlop/s

簡介：

Sierra超級計算機美國國家能源局橡樹嶺國家實驗室已經給它定下來要做的事情，助力科學家在高能物理、材料發現、醫療保健等領域的研究探索。其中在癌症研究方面將用於名為「CANcer分散式學習環境（CANDLE）」的項目。

No.3 神威 太湖之光（Sunway TaihuLight）中國

中國 製造商：國家並行計算機工程技術研究中心

處理器：10,649,600 個；峰值速度： 125,436 TFlop/s

簡介：

之前四屆top500排名中，我國的神威 「太湖之光」超級計算機都是冠軍，全部使用中國自主知識產權的晶元。

No.4 TH—2 天河二號（中國）

中國 製造商：國防科大

處理器：4,981,760個；峰值速度：100,679 TFlop/s

簡介：

天河二號曾經6次蟬聯冠軍，採用麒麟操作系統，目前使用英特爾處理器，將來計劃用國產處理器替換，不僅應用於助力探月工程、載人航天等政府科研項目，還在石油勘探、汽車飛機的設計製造、基因測序等民用方面大展身手。

No.5 Frontera（美國）

瑞士 製造商：Cray

處理器：448,448 個；峰值速度：38746 TFlop/s

簡介：

由DELL EMC公司為德克薩斯高級計算中心（TACC）打造，將用於多領域科研計算。

No.6 Piz Daint 代恩特峰（瑞士）

瑞士 製造商：Cray

處理器：387,872 個；峰值速度：27154 TFlop/s

簡介：

由Cray公司為瑞士的國家超算中心打造，該超算同時也是Green500的第一名，可見其節能效果非常明顯。

No.7 Trinity 三一（美國）

美國 製造商：Cray

處理器：979,968 個；峰值速度：41,461 TFlop/s

簡介：

Trinity超級計算機,由Cray公司建造,是由美國能源部洛斯阿拉莫斯國家實驗室和桑迪亞國家實驗室部聯合部署的。

No.8 ABCI（日本）

日本 製造商：富士通

處理器：391,680 個；峰值速度：32,576 TFlop/s

簡介：

AI Bridging Cloud Infrastructure簡稱ABCI，注於AI應用的大型雲平台，由1,088個富士通伺服器PRIMERGY CX2570 M4 x86伺服器組成，每台伺服器配備兩個Intel Xeon Scalable家族處理器和四個NVIDIA Tesla V100加速器。

No.9 SuperMUC-NG（德國）

德國 製造商：聯想Lenovo

處理器：305,856個；峰值速度：26873 TFlop/s

簡介：

德國 SuperMUC-NG超算由聯想公司打造，使用聯想的ThinkSystem SD530伺服器組成，該伺服器在SPECpower_ssj 2008 2插槽（4節點）測試中取得性能世界紀錄。

No.10 Lassen（美國）

美國 製造商：IBM

處理器：288,288 個；峰值速度：23047 TFlop/s

簡介：

Lassen超級計算機位於德克薩斯高級計算中心（TACC），異構的IBM / NVIDIA系統，具有與Sierra相同的架構。
