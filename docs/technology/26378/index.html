2019-12-23 2,615條評論 4,447次閱讀 3449人點贊

相信不少讀者都有過這樣的經歷：「開了視頻會員，結果發現還有專屬廣告」，而現在一些視頻平台推出了超前點播的服務，引起了網民熱議。

「會員免廣告、花錢預點播、超前點播、好片另付費……」APP會員制收費套路層出不窮，也暴露出了越來越多的問題。

用戶對視頻平台的訴求點都有哪些？視頻平台為什麼會出此舉措？今後，視頻平台又該如何去實現營收與內容質量雙增長？

視頻會員費大概率漲價？近9成網民不接受

近日，愛奇藝會員及海外業務群總裁楊向華在接受採訪時坦言，正在醞釀將愛奇藝會員費用上漲，不排除率先提價，而另一家影視上市公司高管也透露，目前視頻平台會員費上漲是「幾乎確定」的事情。

視頻平台會員費大概率上漲，對此21世紀經濟報道發起了一項調查，結果顯示：

  • 89.4%的網友為此表示：「不接受，吃相有點難看」，並在留言中表達了對視頻平台上調會員付費金額的做法表示不理解。

  • 只有5.2%的網民對此表示「接受，漲價可以理解」，這部分網民認為視頻平台長期面臨著營收虧損的狀態，也可以理解這麼做的原因；

  • 而5.4%的網民則對此表示「無所謂」，漲不漲價都不影響他們觀看視頻。

視頻會員服務、內容質量成吐槽「重災區」

從調查結果可以看出，大部分網民對視頻平台的漲價做法並不買單，有網友表示，「開了會員還給我放廣告」，存在二次收費的嫌疑，而且會員視頻和服務品質也沒有跟上來，拉低了用戶的觀看體驗。

但也有網友認為：「漲價可以理解，問題是現在VIP權益得不到保證，依然有廣告，依然需要付費點播。尊重版權很重要，但是別忽視了用戶權益。」


還有的網友認為，視頻平台為彌補營收虧損，「上調視頻的廣告費」入手，而不是通過上漲視頻會員的價格來解決問題。

根據浙江省消保委調查顯示，71.04%用戶購買會員後，還是會經常看到廣告；76.15%的用戶成為會員後，還遇到需另外付費購買視頻等情況。

24款知名APP付費會員調查：消費者「被自動續費」現象普遍

近期，江蘇省消費者權益保護委員會開展手機APP付費會員制消費情況調查，選取了6類、24個日常使用頻率較高的APP作為調查樣本。

調查結果顯示:

  • 消費者充值APP會員種類多、開通數在2個左右，以月付費為主。

  • 註冊付費會員的目的主要為享有去除廣告、使用更多功能和享有優先權。

  • 超5成的消費者表示充值之後的預期效果一般或不滿意，不滿意的原因主要為付費後部分內容仍需再次付費，其次為成為VIP後仍有廣告。

對於APP平台付（續）費的清晰透明程度和產品服務的知情告知程度，線上調查結果顯示滿意度較低：73.51%消費者表示APP付（續）費套路多，希望有關部門介入督促企業改進並加強監管。

線上調查顯示，69.30%的消費者表示遭遇系統默認自動續費，但僅有5.58%的消費者表示會選擇自動續費。

從樣本體驗情況來看，24個樣本均有自動續費的功能。其中，21個樣本將自動續費設置為優先選項。僅有3個未將自動續費設置為優先選項，分別是：蝦米音樂、蜻蜓FM、騰訊微雲。

另外，24個樣本中，有12個設有試用功能，其中有8個樣本一旦用戶選擇試用，就等於默認選擇了自動續費，須在試用期屆滿前一定時間內手動取消，否則即成為自動續費用戶。

對於自動續費，多數樣本傾向於引導用戶選擇自動續費功能，在沒有盡到充分告知甚至根本沒有提醒告知的情況下，使用戶在記不得甚至不知情的情況下延續繳費，同時取消途徑也很隱蔽和繁瑣，不同程度侵犯了用戶知情權、自主選擇權。

在線上問卷調查中，73.33%的消費者表示充值成為會員後部分產品仍需額外付費，主要集中在視頻類、音樂類和音頻類的APP。

在體驗調查的24個APP樣本中，有18個樣本在消費者支付會員費之後，仍有部分產品或者服務需額外付費。對於18個涉及額外付費的樣本，未履行相關告知義務或者未充分履行告知義務的樣本佔61.11%。

視頻平台的內憂與外患

  • 內憂：平台營收壓力巨大

在線視頻經過多年廝殺後，倒下的倒下，整合的整合，最終形成了騰訊視頻、優酷和愛奇藝三家鼎立的市場格局，但在這場大戰中各家基本都燒了近千億資金。

比如，愛奇藝發布的2019年第三季度財報顯示，其第三季度營收約為74億元，會員渠道收入37億元，佔比過半，雖營收同比增長6.98%，但虧損幅度在不斷拉大；而其他視頻平台也同樣存在虧損狀態。

  • 外患：短視頻、直播蠶食市場份額

抖音、快手為代表的短視頻還有虎牙、鬥魚為代表的直播平台，這兩類新勢力都在不斷地蠶食傳統視頻平台的用戶市場。

根據艾媒諮詢發布的《2018-2019中國短視頻行業專題調查分析報告》和《2018-2019中國在線直播行業研究報告》顯示，2018年中國短視頻用戶規模達5.01億人，預計2019年短視頻市場規模或達到233.5億元，並且未來中國短視頻行業用戶規模仍將保持穩定增長態勢。

數據來源：艾媒諮詢

據大河報梳理，短視頻之所以快速興起，主要有三個原因：

  • 其一，4G迅速普及、5G即將到來，網度提上去了、費用降下來了，越來越多的用戶捨得花流量觀看、上傳短視頻。

  • 其二，是因為各類性能強悍的手機陸續發布，手機視頻剪輯APP層出不窮，極大降低了對視頻創作者的門檻要求；

  • 其三，短視頻極大滿足了人們對短、平、快產品的需求，利用碎片化的方式為用戶提供便捷的信息消費體驗。

報告還顯示，2018年中國在線直播用戶規模達4.56億人，增長率為14.6%，預測2019年在線直播用戶規模達到5.01億人，到2020年將達到5.24億用戶規模。

數據來源：艾媒諮詢

儘管有分析師認為，直播行業正處於降溫時期，用戶增長的速度也在不斷放緩，但是很多頭部直播平台開始在內容生產、直播培育和吸引流量等方面開始加大投入力度，對傳統視頻網站構成直接的競爭關係，值得愛奇藝、騰訊和優酷引起注意。

在行業紅利逐漸消失的當下，任何一種爆款影視IP都有可能成為拉起其他競爭對手的制勝法寶。正如，愛奇藝創始人龔宇所言：「市場跑馬圈地的時代過了，每一個縫隙都要爭，才能持續增長。」

簡言之，今後很長一段時間裡，各視頻網站仍需繼續堅持深耕優質IP開發和提升用戶服務體驗等。
