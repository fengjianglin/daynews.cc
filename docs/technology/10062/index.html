去年出3000多起性侵事件 Uber市值蒸發14億美元

2019-12-07 1,257條評論 2,353次閱讀 1236人點贊

中新社舊金山12月6日電 美國優步公司當地時間5日發布的一份安全報告顯示，其美國市場2018年發生了3000多起性侵事件。受此影響，公司股票市值6日下跌了14億美元。

《聖荷西水星報》報道稱，優步股票5日收盤價為28.65美元，市值為489億美元。6日，其股價下跌2.7%，收盤於27.88美元，市值跌至475億美元。


5日晚間，優步發布了安全報告。報告顯示，該公司在2018年收到3045起性侵犯報告，2017年的同類事件為2936起。2018年，優步收到235起強姦案件的報告，高於2017年的229起。報告還公布了數以千計的其他類型的侵犯，包括強姦未遂以及違背意願的觸摸、接吻。優步表示，2017年至2018年之間，乘客平均每天在美國搭乘優步出行310萬次。

發布安全報告後，優步首席執行官達拉·科斯羅薩西在社交網路上承諾，將採取進一步措施，以保護乘客和駕駛員的安全。他寫道：「我的內心與每一位受害者在一起，優步今天邁出了重要的一步，我們的工作將永不停止。」

《聖荷西水星報》報道稱，對於近年來一直被一系列問題困擾的優步來說，該報告成為其另一個污點。

美聯社報道稱，在過去的兩年中，優步一直在努力提高其出行安全性，比如，在其應用軟體里設置緊急按鈕，以及推出允許乘客或駕駛員與親友分享位置的功能。在安全報告中，該公司還介紹了即將採取的其他安全措施。

批評人士說，優步應該採取更多措施，尤其是進行背景調查，以排除潛在危險的駕駛員。與許多計程車公司不同，優步及其美國主要競爭對手Lyft公司不會根據國家資料庫檢查駕駛員的指紋。

(原標題：安全報告發布之後 優步市值一天蒸發14億美元)
