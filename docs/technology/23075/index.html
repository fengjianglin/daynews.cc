這個和“買買買”有關的數字，我們終將成為全球第一

2019-12-19 1,851條評論 4,108次閱讀 1539人點贊

12月18日，阿里發布《重構增長力量：2019企業數智化轉型發展報告》。報告中提到：

「2008年中國商品零售額僅相當於美國的一半左右，2019年中國這一數字預計將首次超過美國」

數據還透露了哪些信息？小編為您貼心準備了一份數據圖析：

化妝品、日用品、食品飲料等零售額增速最快↓

網路零售比重持續提高↓

零售企業數量持續增長↓

目前我國消費品市場運行整體都有所增長，吃用類商品增勢平穩，餐飲服務市場增速有所加快，文娛類商品增速明顯加快，網上零售更是呈快速增長的態勢。

但無論是零售業的快速發展，還是消費額的飛速增長，都離不開它們背後共同的力量：消費者。下面，我們特邀收到年度賬單的兩位央視小編，來談談他們的「消費感悟」↓

回顧我國零售發展的40年，每一步都踏出了堅實而穩定的步伐。一起追尋那些年，企業零售走過的路。
