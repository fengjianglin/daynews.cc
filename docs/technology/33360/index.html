懟完蔚來後，馬斯克又開懟小鵬汽車竊取技術

2019-12-24 116條評論 1,599次閱讀 447人點贊

圍繞小鵬是否剽竊特斯拉自動駕駛技術的爭論還沒有塵埃落定，馬斯克在推特上已經開懟了。

36氪獲悉，事情圍繞一則關於小鵬汽車的報道展開。12月22日，CleanTechnica發表題為「小鵬汽車自主研發自動駕駛技術讓其維持低價」的文章，馬斯克以揶揄口吻評論此文「自主研發」，並附上兩個笑哭的表情。


圖片來自推特截圖

間隔不到1小時，馬斯克在回復一名網友時表明了立場，「（小鵬的自動駕駛軟體）確實是我們的靈感，但這是一個舊款的特斯拉軟體版本，所以效果有限」。


今年年初，特斯拉就已經和小鵬汽車結下樑子。彼時，特斯拉狀告小鵬汽車剽竊Autopilot的代碼，將曾任特斯拉高級工程師、後來跳槽小鵬汽車任感知主管的曹光植等人告上法庭。

曹光植曾承認，在2018年底曾向自己的iCloud雲賬戶備份了包含AutoPilot源代碼的壓縮文件，但文件在後來離職時已經被刪除，小鵬汽車當時也主動提供了曹光植工作筆記本的相關所需信息。這一案件尚無最新進展。

對於特斯拉的做法，小鵬汽車董事長何小鵬此前也罕見嗆聲特斯拉，他認為，特斯拉是以民事訴訟為名，行打擊競爭對手之實，「不會有任何畏懼和放棄，還將繼續引入更多領域的高科技人才」。

目前，車企以及創業公司對於自動駕駛技術人才的需求很高，公司之間的人員流動、挖牆腳的現象非常普遍。技術是跟著人才團隊流動的，由此帶來的技術產權問題也很令科技公司頭疼。

小鵬汽車從特斯拉挖走高管最早是發生在2017年10月。當時，Autopilot 2.0的核心打造者之一、特斯拉機器學習團隊負責人谷俊麗加盟小鵬汽車，任職自動駕駛研發副總裁，向何小鵬彙報。除了小鵬，特斯拉自動駕駛方面的員工還經常被蘋果以及各類初創公司挖走。馬斯克曾經公開宣稱蘋果是特斯拉的「人才回收站」。

除了狀告小鵬，特斯拉今年年中還順帶告了跳槽到美國無人車獨角獸Zoox的4名員工，指責其盜竊特斯拉倉儲物流和庫存控制信息。

在矽谷，這類型的訴訟其實非常常見，但以往都是特斯拉以公司層面提起訴訟，馬斯克儘管經常在推特上放飛自我，但圍繞這一問題、特別是中國新造車企業則鮮少發表評論。

除了小鵬汽車，蔚來汽車也早在今年3月份被馬斯克關注到。有網友在轉發CBS一期關於蔚來汽車的報道後，艾特馬斯克並詢問其對蔚來汽車的看法，馬斯克隨後稱「peace was never an option」。有人將這句話解讀為，蔚來和特斯拉未來將有一戰。

不過，馬斯克再怎麼語出驚人，網友們大多已經習以為常。小鵬汽車以及zoom是否真的剽竊了特斯拉的技術，很難判斷，但從另一個角度或許也可以表明，這些中國的新造車企可能已經進入馬斯克的視野中了。
