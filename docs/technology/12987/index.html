黑洞投資合伙人楊蓉：新常態下，心態和現金流要“兩手抓”

2019-12-12 3,229條評論 4,250次閱讀 3055人點贊

【獵雲網北京】12月11日報道

12月10日，2019年度CEO峰會暨獵雲網創投頒獎盛典在北京望京凱悅酒店隆重舉行，近百位知名資本大咖，獨角獸創始人、創業風雲人物及近千位投資人與創業者共聚「新勢力·2019年度CEO峰會暨獵雲網創投頒獎盛典」。

在由黑洞投資合伙人楊蓉主持的《新常態下創投發展新趨勢》高峰論壇上，五嶽資本創始合伙人蔣毅威、愉悅資本創始人合伙人戴汨、渶策資本創始合伙人胡斌、德迅投資執行董事齊海、雲天使創始人秦捷、芳晟基金合伙人方芳，就論壇議題發表了精彩觀點。

楊蓉認為，在新經濟常態下，創業者應聚焦擅長的賽道，重點耕耘。既要有節省現金流的準備，同時要有樂觀的心態，兩手都要抓。

而說到下沉市場，難免會回到消費升級、消費降級的討論。她認為，大家對於美好生活的追求都是一致的，最終還是消費升級。關鍵在於是否了解市場，了解市場才可以更好創業，

黑洞投資成立於2014年，從成立至今的五年時間裡，黑洞投資一直致力於尋找和孵化能對未來產生巨大變革的企業和創業者。重點聚焦於地產+、人居場景、人工智慧等前沿科技領域，在多個細分領域均有布局。先後投資與孵化出了桉樹空間、37度智能傢具、織點智能、貨拉拉、好租、小狗機器人等多家在行業內領先的企業。

為了幫助創業者和投資人重新蓄力，2019年，獵雲網攜全新品牌「新勢力（New Force Summit）」亮相。本次峰會由獵雲網主辦，銳視角、獵雲資本、獵雲財經、企業管家協辦。

此次盛典上，獵雲網將通過六個版塊分享創業者和投資人在智能製造、文娛、零售、醫療、教育、汽車等領域的啟發性的觀點和行業前瞻，圍繞多個維度，分享科技和產業前沿觀點，探討創新潮流趨勢、把握未來新方向。

以下為楊蓉的演講實錄，獵雲網整理刪改：

大家好，我是黑洞投資的楊蓉，非常高興擔任本次圓桌的主持人。

在場的各位有美元基金，也有人民幣基金。這兩年，整個資本環境非常嚴峻，大家提到資本寒冬，所以不僅是投資機構不太好募資，好項目也非常少。

說到寒冬，還有一個詞是周期。有一些項目面臨周期的時候會越做越好，有一些創業項目就消亡了。事實上，周期對創業者來說影響不會特別大，本質還是在於商業模式怎麼樣，現金流是不是健康，以及心態是不是很好。

既要有節省現金流的準備，同時要有樂觀的心態，兩手都要抓。總體來講就是有所為、有所不為，一樣可以度過難關。另一方面，創業者了解投資環境，這對融資包括定公司的戰略上是非常有幫助的。而對於投資方來說，正是因為資本寒冬，篩選出來的才是真正有實力的投資人。

通過幾位嘉賓的分享，大家可以了解到資本寒冬里，投資圈與創業圈一樣面臨著嚴峻的挑戰。頭部基金越做越好，有些在練內功，並不是大家不投項目，而是在尋找好的標的，要把項目做好，好的項目是不缺投資的。

說到下沉市場，難免會回到消費升級、消費降級的討論，但是大家對於美好生活的追求都是一致的，最終還是消費升級。關鍵在於是否了解市場，了解市場才可以更好創業，

說實話，寒冬對於聚焦在自己擅長的領域、能發揮出創始團隊獨特優勢的創業項目來說，影響還是比較小的。所以，大家創業的時候聚焦擅長的賽道，重點耕耘，才有可能有機會成為老大。

我是創業者出身，我在看投資項目的時候更看重產品和服務能真正滿足消費者需求的項目。只有這樣的項目才可以穿越寒冬，迎來春天。
