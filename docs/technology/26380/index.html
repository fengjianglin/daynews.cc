鏈圈動態

工商銀行行長谷澍：擴大區塊鏈、大數據等金融科技在普惠金融領域的應用

據中國網財經消息，12月21日，在2019中國金融學會學術年會暨中國金融論壇年會上，工商銀行行長谷澍表示，金融供給側改革要「補短板」，重點是要更好的發展普惠金融。

商業銀行搞普惠，必須真正實現業務、服務的下沉，創新構建普惠金融生態圈，擴大區塊鏈、大數據等金融科技技術在普惠金融領域的應用，為普惠金融提供生態化、場景化、智慧化服務，切實提高服務效率。

央行科技司副司長：區塊鏈等技術正在重塑金融業態

據中國金融新聞網消息，在近日舉行的「2019中國金融科技年會暨第十屆金融科技及服務優秀創新獎頒獎典禮」上，中國人民銀行科技司副司長羅永忠表示，當前，金融科技在全球範圍內蓬勃興起，以大數據、人工智慧、雲計算、區塊鏈等為代表的新一代信息技術全面滲透到諸多金融細分領域，正在改變和重塑金融業態。

面向未來，金融業要切實抓好科技在金融領域規範應用，推動金融科技高質量發展，增強人民群眾對數字化、網路化、智能化金融產品和服務的滿意度。

「浙商產業區塊鏈促進聯盟」成立，將推動區塊鏈知識普及和產業落地

據證券日報消息，12月20日，在2019第十二屆《浙商》年會之第二屆中國產業區塊鏈發展高峰論壇上，「浙商產業區塊鏈促進聯盟」正式成立。

據報道，聯盟是由國內區塊鏈行業頭部企業及行業專家共同發起的非營利性社會自治組織，旨在團結區塊鏈領域的實幹者，共同推動區塊鏈基礎知識普及以及產業區塊鏈在國內的落地發展。

接下來，浙商產業區塊鏈促進聯盟將基於自身優勢，開展區塊鏈知識傳播，區塊鏈技術、服務對接，搭建國內外技術交流平台等工作。

長江大數據中心正式上線，將利用區塊鏈技術打通信息壁壘

據新華社消息，為長江水利建設提供大數據支撐服務的長江大數據中心21日在長江勘測規劃設計研究院正式上線。這也是長江勘測設計行業建立的首個共享的水利大數據中心，對保護治理長江具有重要意義。

長江勘測規劃設計研究院總工程師王小毛介紹，建設長江大數據中心主要是基於水利勘測規劃、水行政管理、水利水務工作以及水利現代化發展的需要。當前，長江雖有各種資料庫，但彼此並不互通。長江大數據中心利用區塊鏈技術，試圖打通長江各類數據的信息壁壘。

下一步，長江大數據中心將逐步聯合行業內外單位、機構，打造一個為我國水利工作科學決策、水利工程綜合效益充分發揮、水資源最優管理、最大化服務公眾社會的大數據應用平台。

幣圈動態

央行副行長：央行數字貨幣是為了應對全球穩定幣帶來的挑戰

據中國證券網消息，在昨日舉行的「2019中國金融學會學術年會曁中國金融論壇年會」上，中國人民銀行副行長陳雨露在會上表示，穩定幣會對支付體系、金融穩定、貨幣政策和國際貨幣體系等四方面產生影響。

此外，五年前，人民銀行啟動了數字貨幣的研究和法定數字貨幣（DC/EP）的研發。一方面，要增強法定貨幣體系對未來數字經濟生態的適應性；另一方面，也是要應對全球穩定幣的挑戰。

當然，主權數字貨幣的發行也存在著「狹義銀行」效應，需要我們供給相應的理論創新成果和應對政策。

分析：涉及加密貨幣和區塊鏈的三個公共政策領域或將成為美國國會明年關注重點

據福布斯發布的分析文章，截至2019年底，美國國會已經提出21項針對加密貨幣和區塊鏈政策的法案，有望在2020年審議。事實上，美國立法者一直忙於研究這項新技術對企業、消費者和整個社會的影響。儘管國會提出了若干涉及加密貨幣或區塊鏈技術的法案，但仍有三個主要的公共政策領域可能成為國會2020年的持續關注重點。

文章表示，第一個主要的公共政策問題涉及加密貨幣如何被廣泛用於各種非常危險的活動，如逃避美國制裁、人口販運和恐怖主義活動。除了這些擔憂外，許多立法者還希望美國探索加密貨幣的獨特跟蹤能力以及區塊鏈技術如何幫助美國政府機構追查上述活動中的不良行為。

第二個也是最常被報道的公共政策問題是，在當前的監管框架下，公司如何在業務模型中使用加密貨幣和區塊鏈。美國經濟的規模及其聯邦和州一級監管結構的複雜性可能會抑制私營部門的創新。

最後，美國政府本身如何利用分散式賬本技術的政策問題通過迄今為止引入的立法得到解決，特別是在其他國家已經將大量時間、精力和資金集中在加密貨幣和區塊鏈技術上的情況下。

詹克團再卸任一公司法人，曾獲比特大陸投資

據天眼查顯示，12月20日，詹克團卸任福建雋智科技有限公司的執行董事兼總經理及法人。福建雋智科技有限公司於2018年3月成立，曾獲比特大陸投資。吳忌寒曾任公司監事，詹克團均曾為公司法人。目前二人均以退出公司。不過，詹克團並未完全與福建雋智科技有限公司斷開關係，目前該公司第二大股東為比特大陸全資持股的福建湛華智能科技有限公司，詹克團為法人及執行董事兼總經理，同時吳忌寒為該公司監事。

據統計，近段時間以來，詹克團已經卸任了多家公司的法人/高管一職，其中包括算豐科技（北京）有限公司（卸任法人、執行董事、經理）、天津迪未數視科技有限公司（法人、經理、執行董事、主要成員）、北京比特大陸科技有限公司（卸任法人、主要成員、執行董事、經理）、北京比特大陸科技有限公司五原縣分公司（負責人）等。

央視新聞：打擊新型竊電犯罪，河北警方收繳比特幣礦機6890台

據央視新聞消息，前不久，全國多地公安機關會同國家電力部門，持續開展打擊竊電犯罪的專項行動，並取得階段性成果。

河北唐山警方在行動中發現，借用比特幣礦機盜電，成為近年來新型的竊電犯罪手段。截至目前，河北唐山警方扣押大功率變壓器52台，收繳比特幣礦機6890台。警方表示，打擊竊電犯罪行動還將持續進行。
