FTP伺服器端 Wing FTP Server 6.2.2 完美破解企業版本

2019-12-14 703條評論 2,021次閱讀 639人點贊

Wing FTP Server是一款專業的跨平台FTP伺服器軟體, 支持可伸縮的處理器架構並採用非同步IO, 所以在速度和效率方面遙遙領先於其他同類產品，當然他還非常穩定可靠, 在高負載的情況下也能持續地正常運行, 非常適合企業的文件傳輸。 除了基本功能外, 他還提供一個基於Web的管理端和客戶端, 何時何地都能管理你的伺服器. 它還支持可編程的事件, 計劃任務, Lua腳本擴展, 虛擬文件夾, 上傳下載比率分配, 磁碟容量分配, ODBC/Mysql存儲賬戶, 多國語言等特性。

Wing FTP Server 主要特點

跨平台：可在 Windows, Linux, Mac OS X, Solaris等操作系統上運行。

多種傳輸協議 ：支持 FTP, FTPS(帶SSL的FTP), HTTP, HTTPS, 以及 SFTP(基於SSH的FTP)。

FIPS 140-2 ：支持更加安全的FIPS 140-2 驗證加密模塊（證書編號:1051)。

基於web的管理端 ：何時何地管理你的FTP伺服器。

基於web的客戶端：何時何地上傳或下載你的文件。

支持多個域：相當於在一個IP地址上運行多台虛擬FTP伺服器。可編程的計劃任務：使用Lua腳本來完成你的計劃任務。

可編程的事件管理器：事件觸發時, 可執行Lua腳本, 發送電子郵件 或 執行第三方的應用程序。

多類型的用戶驗證 ：通過 XML文件，ODBC 或 MySQL資料庫來存儲大量的用戶數據. 還可以用LDAP或活動目錄來做用戶驗證。

應用程序介面：提供了豐富的API供Lua腳本調用。

虛擬目錄：可以映射虛擬目錄到物理路徑, 當然你可以使用Windows下的UNC資源。

實時信息：你可以實時的監控你的FTP伺服器, 當然還可以查看單個會話的活動情況。

磁碟配額及比例：每個用戶都可以分配到獨立的磁碟空間, 上傳/下載比例, 以及網路帶寬。

多種語言：目前支持14種語言 – English, Français, Deutsch, Italiano, Türkçe, Český, Română, Nederlands, Português, Español, 簡體中文, 繁體中文, 日本語, 한국어。

WEB客戶端包含了類似Windows資源管理器的功能, 並帶來以下主要特點

支持iPhone/Andriod. – iPhone,Andriod用戶可以方便使用手機來訪問web客戶端。

多文件同時上傳或下載 – 利用web客戶端自帶的控制項,你可以輕易地同時上傳或下載多個文件。

支持Web外鏈下載 – 你可以通過Web外鏈來分享你的文件, 你還可以給外鏈設置下載次數和過期時間。

在線文本編輯器 – 你可以用在線文本編輯器輕鬆地查看或修改伺服器上的文檔。

壓縮/解壓縮文件 – 直接在伺服器上壓縮/解壓縮文件,節省你的上傳/下載時間。
