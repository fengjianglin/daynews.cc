紫金新評：以積極態度應對外界質疑，編程貓才能走的更遠

2019-12-15 1,589條評論 3,424次閱讀 1327人點贊

紫金財經12月15日消息 少兒編程，是一種通過編程遊戲啟蒙、可視化圖形編程等課程，培養學生的計算思維和創新解難能力的課程。近年來，國內對於編程的利好政策不斷加碼，2017年，國務院出台《新一代人工智慧發展規劃》，要求在中小學階段設置人工智慧相關課程、逐步推廣編程教育。之後，浙江、北京、重慶等省市還將少兒編程納入教學體系之中，於是少兒編程啟蒙教育迅速躥紅。

少兒編程教育方興未艾，根據《中國互聯網少兒編程教育市場分析2019》的報告，2019年，中國互聯網少兒編程市場交易規模或達258.4億元，至2020年預計可達377.1億元。另據不完全統計，目前至少有200家公司投身少兒編程教育，新東方、好未來和高思三大教育巨頭。

而作為獨立的創業機構，少兒編程的賽道也有不少可圈可點的公司出現，比如編程貓、西瓜創客等。日前，在線少兒編程教育品牌編程貓宣布成功完成一輪4億元融資，累計融資額達到10億。8月份，在線少兒編程教育品牌西瓜創客宣布獲得B輪價值1.5億人民幣的美元融資，融資由新東方、賽富中國、泛海創投、紅杉中國、經緯中國、百詞斬投資。在整個互聯網的「資本嚴冬」之下，少兒編程還能獲得大筆融資，自然獲得了更多人的關注。

未來前景一片大好，不過近期網上一直對編程貓存在一些質疑，這些質疑如果不重視，逐步積累，反而不利於公司的進一步發展。

比較突出的質疑包括如下兩點：首先是加盟商的抱怨。據自媒體《磐石之心》的一篇文章，一位加盟商在文中表示，自加盟之後，編程貓不僅沒有履行應盡的義務，向他提供經營指導、技術支持與業務培訓。而且他曾按照正常指導流程，將線下招生信息編入編程貓後台系統，卻發現編程貓將他的學生信息在向線上教育模塊做導流。

有知情人士表示，編程貓此舉是為了融資和擴張，想盡一切辦法做高線上學員數量，但是，結果卻在損害加盟商的利益。按理說，編程貓在幾輪融資後應該有巨大的資金和力量去研發教材，教程等。但是，對於加盟商在教研支持方面卻做的很差。

網上的信息顯示，從2017年開始，編程貓即開展與線下中小培訓機構的合作。按照公開的加盟政策要求，加盟商除了自己承擔裝修費外，每個縣級城市到省級城市的合作機構的還必須繳納3.5萬元至4.5萬元不等的首批進貨費。業內人士分析，這種加盟模式可以幫助編程貓快速回籠資金，避免重資產運營，但如此一來，維持線下正常運營的壓力就全部壓在了加盟商身上。

加盟商的前期大筆投入、說不清楚的線上導流、缺失的經營指導以及經營壓力全部在加盟商身上等問題，造成了加盟商的普遍不滿。

第二個質疑，關於編程貓是否存在加盟資質的質疑。按照《商業特許經營備案管理辦法》的相關規定，特許人未按條例規定向商務主管部門備案的，由商務主管部門責令限期備案，逾期仍不備案的，處5萬元以上10萬元以下的罰款，並予以公告。通過商務部的業務系統統一平台進行查詢，的確無法查到編程貓的相關備案信息。

據了解，不僅僅是編程貓，有相當一部分創業企業，在上述官方平台上，都無法查到相關備案信息。違規經營、在加盟上走鋼絲，幾乎成為業內眾所周知的事實。

公開資料顯示，編程貓由李天馳、孫悅於2015年創立，2018年，獲得前央視主播張泉靈旗下的紫牛基金投資。據編程貓的數據，迄今為止，學員人數已超3000萬人，並且入駐11500所公立學校。目前編程貓社區已獲得小羊肖恩、阿狸、故宮、功夫熊貓、鹿精靈等多個知名IP免費授權，為用戶提供更多創作的可能性，目前平台作品已突破700萬。

編程貓的業務不止步於中小學，也深入清華、北大等高等學校計算機核心課程建設，組建計算機領域專家團隊，培養產業人才。目前，編程貓已建立完整的課程研發體系，累計研發上千節課程。

編程貓CFO張煒在今年年初則表示，基於公司目前的發展速度，我們將於今年開始上市規劃，2年內進行IPO。 甚至透露出確實已經有券商的保薦人來做編程貓的工作，編程貓會做好自身的業務，相信在科創板上一定會有很好的發展，對編程貓來說，港交所和納斯達克也會進行考慮。

4年8輪融資，編程貓成為全球編程教育領域首個單筆融資上億元的創業公司。這樣一家前景廣闊的公司，如何平衡好加盟商的利益，在賦能加盟商的過程中實現共贏，依舊是編程貓需要應對的問題。而與加盟商加強溝通，合理回應外界關切，也是一家致力於長久發展的企業必須要做的。

作為少兒編程教育的頭部企業，特別是有志於通過上市，成為公眾公司來尋求更大發展的企業來說，合法經營、合規經營更應該是應有之意，而不應在行業亂象中迷失自己。迅速補足法律法規上的短板，持證合規經營，才能在少兒教育的賽道上走的更遠。

最後仍然值得一提的是，如果選擇在國內科創板上市，鑒於科創企業的特點，關於知識產權的訴訟一直是尋求科創板上市企業繞不過去的坎兒，關於這方面的問題，編程貓也應該及早準備，未雨綢繆！（紫金財經綜合）
