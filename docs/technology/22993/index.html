成立僅3年，這家公司憑什麼引得英特爾拿20億美元收購

2019-12-19 1,341條評論 4,110次閱讀 89人點贊

傳了半個月的收購案，終於官宣了。

當地時間周一，英特爾宣布以約20億美元的價格收購AI晶元製造商Habana Labs，這也是繼2017年收購了自動駕駛晶元公司Mobileye之後，英特爾又一次斥巨資收購以色列AI晶元公司。它將成為英特爾在以色列的第二大收購。

英特爾方面稱，收購完成後，Habana Labs將繼續在以色列以一個獨立公司的身份運營，也依舊由目前的管理團隊領導，並向英特爾執行副總裁兼數據平台事業部總經理Navin Shenoy彙報。

英特爾為什麼選擇了Habana Labs？

去年，英特爾的投資部門對這家AI晶元公司Habana Labs進行了投資，也是該公司獲得的7500萬美元B輪融資的一部分。追蹤以色列科技業的Start-Up Nation Central稱，迄今為止，這家以色列公司已經籌集了約1.2億美元。

Habana Labs成立於2016年，最初的業務是開發專為深度神經網路訓練和生產環境中的推理部署而優化的處理器平台。2018年，Habana Labs發布了第一款產品，即AI推理處理器「Goya」，官方稱該晶元在ResNet-50模型的推理測試中吞吐量較英偉達T4提升了1.67-2.06倍，整體性能提升了4倍，低延遲方面也有大幅提升。9個月後，Habana Labs又宣布推出AI訓練處理器「Gaudi」，該公司首席商務官Eitan Medina曾在介紹這款晶元時表示，「Gaudi擁有比GPU還要高的能力和處理性能，以及更低的功耗。除此之外，通過內置RDMA乙太網處理單元，該晶元擁有『創記錄性』的可擴展能力，可訓練非常大的AI模型。」

具體到性能上，Gaudi雙向數據通量達2Tbps，這意味著該晶元可以直接訪問任何其他晶元的內存。

推理、訓練雙芯在手，也讓該公司在營收上有了不錯的成績，Habana Labs給出的官方預測是，「預計今年我們的AI產品將創造35億美元的營收，較去年同比增長250%。」

Habana Labs首席商務官Eitan Medina

據Habana Labs首席商務官Eitan Medina介紹，其團隊成員主要是來自處理器、DSP、系統設計以及網路設計等領域全球知名企業的精英，能夠支持Habana Labs從軟體到硬體的研發需求。同時，他還直接「放話」—— 要做人工智慧晶元產業的領導者。而其底氣，就是Goya和Gaudi兩款產品。

軟體方面，Habana Labs還擁有一個開發和執行環境SynapseAI，該環境具有庫和JIT編譯器，支持所有標準的AI和機器學習框架，以及Open Neural Network Exchange格式。

而除公司實力強勁之外，Habana Labs的創始團隊也是人才濟濟，創始人David Dahan和Ran Halutz曾是PrimeSense Limited（專註於3D圖像感測技術及產品研發）的高管，該公司曾在2013年被蘋果公司以3.6億美元的價格收購。

根據官方的信息，目前該公司全球擁有150名員工，在特拉維夫、加利福尼亞聖何塞、北京、波蘭格但斯克設有辦事處。

投資者方面，Habana Labs的第一位投資者及其董事長Avigdor Willenz，曾有過多次成功的創業經歷，如創立的Galileo Technologies曾被Marvell以27億美元的價格收購，聯合創辦的Annapurna Labs則在2015年被亞馬遜以3.7億美元的價格收購。

此次Habana Labs被英特爾收購，也是該創始團隊的又一成功創業經歷，Avigdor Willenz則將進入英特爾數據平台事業群並擔任資深顧問。而更有意思的是，有關Habana Labs的一切事物將直接向數據平台事業群總經理Navin Shenoy彙報，而非大多數人此前所猜測的英特爾AI產品負責人Naveen Rao，這也直接反應了Habana Labs在英特爾AI晶元發展戰略中的重要性，因為數據平台事業群才是英特爾廣泛的AI技術大本營。

英特爾的AI野心，訓練、推理雙晶元布局正在展開

英特爾執行副總裁Navin Shenoy曾公開表示，「隨著英特爾轉變為一家以數據為中心的公司，AI晶元將成為公司未來戰略中極為重要的一環，且由該業務所帶來的營收佔比正在逐年上升。」

而此次收購Habana Labs，便是英特爾數據中心AI計劃的戰略舉措之一，也可以看做是對Nervana的一種補足。
