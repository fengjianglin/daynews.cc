供不應求？摩托羅拉推遲可摺疊手機Razr上市時間

2019-12-21 249條評論 2,441次閱讀 999人點贊

科技訊 12月21日消息，據外媒報道，摩托羅拉宣布推遲可摺疊智能手機Razr在北美地區的上市時間，理由是這款售價1500美元的手機需求超出了其供應極限。

摩托羅拉於今年11月發布Razr，原計劃在12月份接受預定，並於2020年1月份開始發貨。摩托羅拉在最新聲明中表示:「摩托羅拉已經決定調整Razr的預售和上市時間，以更好地滿足消費者的需求。我們正在努力確定合適的數量和時間表，以確保更多的消費者能夠在Razr上市時就能買到它。」

摩托羅拉拒絕透露這款手機何時上市，但表示，預計不會與最初的上市時間有「太大變化」。在今年早些時候接受採訪時，摩托羅拉總裁塞爾吉奧·布尼亞克（Sergio Buniac）表示，他希望需求比供應「多一點兒」。

摩托羅拉沒有將Razr推遲上市歸咎於技術問題。此前，三星電子等其他公司始終難以按時推出可摺疊設備，原因是屏幕出現故障。當摩托羅拉發布這款手機時，其設計總監魯本·卡斯塔諾（Ruben Castano）說:「我們覺得我們已經開發出了一個強大的解決方案。」

可摺疊的Razr與21世紀初的原型機相似，但運行的是谷歌安卓操作系統，內外都有觸摸屏。摩托羅拉此前曾表示，這款手機還將在歐洲、拉丁美洲、亞洲和澳大利亞發售，但沒有透露這些地區的最新發售信息。 （小小）
