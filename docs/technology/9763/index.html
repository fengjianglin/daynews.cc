業界最牛逼創客節，竟然不是華為和阿里辦的

2019-12-09 3,080條評論 4,118次閱讀 1909人點贊

怎麼理解「創客」？

作為數字經濟時代，行業創新的最小因子，年輕、睿智、真我的開發者們，擁有強烈的自我認知，不妥協於現狀，不拘泥於現在的遊戲規則，打破常規，創造別人眼中的不可能。這是創客的常態。

但是，隨著不同產業數字化應用場景的複雜度與日俱增，儘管各大開源技術生態逐漸穩定， 但「技術門檻高、交流渠道狹窄、學習機會匱乏、優質開發工具難覓……」依然是這個群體所面臨的困境。

所以，如果把開發者看作一顆又一顆的「種子」，如何給予他們技術創新的土壤使其成長為參天大樹，才是今天問題的關鍵所在。

正是為了解決這些問題，去年10月，首屆NAVIGATE創客節本著打造技術生態的理念在紫光旗下新華三集團杭州總部揭幕。本年度11月17日，2019 NAVIGATE創客節暨中國智谷·環重郵創新創業大賽又在重慶落幕。時隔一年，從杭州到重慶，創客節的規模更大，覆蓋的人群更多，內容更豐富。但在這些表象之後，我們還能夠從NAVIGATE創客節上看到什麼？

這是開發者最好的時代，還是最壞的時代？

「這是最好的時代，這是最壞的時代。」在狄更斯的這句耳熟能詳的名言之後，還有一句：「這是理性的時代，這是困惑的時代，這是希望之春，這是失望之冬。」我認為，這句話從某種意義上代表了開發者的現狀。

一方面，開發者是各大互聯網公司、科技公司技術生態大會的「香餑餑」。針對開發者的大小會議一個接一個，形式多種多樣，涉及的行業、領域也多種多樣，它們提供了將開發者技術能力向最終創新產品形態轉化的催化劑，是創客的「點將台」。

另一方面，我們也要看到，在這些大大小小的技術生態活動之外，常態化的適合開發者成長為創客的土壤還不夠肥沃。人才培養的體系與實際的行業需求之間，存在巨大的落差。在基礎人才培養最末端的高校，產學研用一體化建設還有很大的空間需要挖掘。

我認為，連續兩年舉辦NAVIGATE創客節背後，新華三集團已經構建好一個技術生態，通過與政府和高校合作，建立了產學研用為主導思想的人才機制。

新華三集團聯席總裁王景頗表示，未來的世界屬於創造者，屬於創客們，希望通過NAVIGATE創客節活動所取得的創新成果，未來都能有機會轉化為創新生產力，為各行各業創造價值。

客觀來說，創客群體也是需要被賦能的。數字化技術必須結合行業實踐才能發揮最大效能，所以，好的技術生態就是能夠為創客們提供多樣化的數字化工具，提供實戰的舞台，從實踐的角度，增加他們與企業數字化需求匹配的機會。

那麼，從這樣的標準來看， NAVIGATE創客節是如何扮演好賦能者這一角色的？

以賽代練的實戰賽，會創造何種價值？

從本屆NAVIGATE創客節的賽事來看，通過黑客松、解決方案合作夥伴邀請賽和安全對抗賽，以賽代練，真正起了催化創造的「化學反應」。

首先，黑客松代表了一種創新的極限，創客就要挑戰各種不可能。

本屆NAVIGATE創客節黑客松比賽在一個極具「創客屬性」的場地里拉開了序幕，來自全國頂尖高校的近30支參賽隊伍，聚精會神、爭分奪秒，在36小時內不斷的頭腦風暴，創造和創新。

第二，解決方案合作夥伴邀請賽代表了一種實踐的精神。開發者不僅需要豐富的技術賦能，也需要行業的場景來獲得實戰的經驗。

這次解決方案合作夥伴邀請賽決賽，針對智能識別、圖形數據分析、數據檢索等賽題，每隊進行了現場40分鐘的演講和答辯，充分展示了團隊在視覺領域的軟體設計與編程的能力。

第三，安全對抗賽代表了對技術的敬畏之心，任何技術和行業解決方案的打造都離不開安全的要素。攻防對抗是安全技術高水平的展現，在近4小時的攻防大賽中，選手們以熟練的技巧操作，默契無間的配合，進行了這場「矛與盾」的比拼。

NAVIGATE創客節，給整個行業帶來哪些啟示？

一個成功的創客節背後，我們能夠讀出哪些對行業的啟示？

第一，數字經濟正在呼嘯而至，而最大的矛盾就是全行業對ICT人才的渴求，與ICT人才短缺造成的供求失衡，和人才錯位。據權威調研報告顯示，僅中國人工智慧人才缺口，就至少在100萬以上。預計到2020年，新一代信息技術人才缺口將達750萬。

解決問題的方式，單純依靠某一家企業的能力顯然是不夠的，所以NAVIGATE創客節創新性地開創企業與政府、高校合作，緊密結合了產學研用的路線，做到不拘泥於創客節本身，也為將來的技術人才培養鋪平了道路。

技術人才生態的培育註定是一個長期的過程，任何企業都無法在短期內產生收穫。本屆NAVIGATE創客節為例，從今年4月的領航者峰會之後就開始籌劃，在9月份正式開始啟動，歷經兩個月的初賽、複賽，才走到了如今的決賽。可見，承辦方新華三集團對NAVIGATE創客節的用心是持續的。

第二，技術生態的外延在不斷擴展。技術的融合與生態的融合，打破了原來信息化時代的應用邊界，而賦予了不同的角色更多維度的定位。這不僅使得開發者與企業之間的距離變得更近，生態合作企業之間的關係也越來越緊密。

在技術賦能方面，新華三集團今年提出數字大腦計劃，通過自身提供的智能數字平台支撐合作夥伴的智慧應用，形成智慧解決方案，服務百行百業客戶的數字化轉型，而NAVIGATE創客節解決方案合作夥伴邀請賽充分驗證了這一賦能模式。通過NAVIGATE創客節這樣一個持續的活動，新華三集團的技術生態也得到了擴展。

事實上，在新華三集團身上其實有一種珍貴的品質，就是不以短期的利益為目標，而是著眼於未來，著眼於大局。NAVIGATE創客節的成功，其實也印證筆者的這個判斷，技術生態的建設，兼具了企業發展和社會發展的雙重責任，是新華三大局觀的直接體現。
