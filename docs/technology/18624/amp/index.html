孫正義投資失利 引發銀行巨頭不滿！軟銀還有哪些王牌？

天天要聞
2019-12-17

外匯天眼APP訊 : 近日，軟銀創始人孫正義在東京大學與馬雲對話時表示，自己並不是5分鐘決定投資馬雲，實際上應該聊了10分鐘左右。馬雲當時讓他印象深刻，是唯一一個沒向自己要錢的人。

孫正義當年慧眼識珠，也正是因為投資了阿里巴巴而在全球舞台上聲名鵲起！當然，除了阿里巴巴，孫正義還投資過許許多多家科技企業，並獲得了令無數投資人眼紅的高回報。然而，人有失手，馬有失蹄，再厲害的投資人，也有失敗的時候，孫正義也不例外！

據海外媒 體報道，多年來始終堅持支持軟銀集團的日本銀行家們，正在認真審視這位「客戶」。在對辦公空間共享初創企業WeWork代價高昂的紓困以及遭遇其他一些挫折之後，日本銀行集的高管們表示，他們對這位億萬富翁管理軟銀及1000億美元願景基金的做法產生了疑慮。

有銀行高管表示，希望在借給軟銀更多錢之前看到一個令人信服的轉型計劃。另一家銀行的高管也表示，銀行正對軟銀採取謹慎態度，對孫正義向高估值的初創企業投入大量資金的戰略表示懷疑。

日本銀行業在近40年時間裡一直在幫助孫正義，截至目前給軟銀和願景基金提供的貸款餘額至少達到150億美元。此外，他們還與孫正義建立了廣泛的投資銀行業務關係，向軟銀提供交易顧問，幫助軟銀從日本的企業債券市場籌集更多資金。市場研究機構Freeman&Co的數據顯示，自2015年以來，軟銀已經向全球銀行支付了超過19億美元的費用，其中大部分流向了日本的銀行。

軟銀接洽的日本大型銀行包括三菱日聯金融集團、三井住友金融集團以及瑞穗金融集團。大約37年前，瑞穗為孫正義提供了第一筆貸款。截至今年3月，瑞穗依然是軟銀最大的貸款人，未償還貸款總額為55億美元。

軟銀此前對外公布了願景基金二號計劃，該基金將專註於投資人工智慧（AI）領域，加速AI革命。軟銀集團表示，根據已簽訂的一系列諒解備忘錄，預計籌資約1080億美元。軟銀集團在一份公告中表示，該公司計劃向該基金先行投資380億美元，預計參與該基金的其他公司包括富士康科技集團、蘋果、微軟、瑞穗銀行、三井住友銀行和三菱UFG銀行等。還有消息指出，中亞國家哈薩克政府也將參加這一基金的投資，預計拿出30億美元。

孫正義最近的一連串失誤考驗了銀行對其新模式的容忍程度，包括WeWork和Uber在內的知名創企估值暴跌，導致軟銀在第二財季報告運營虧損近65億美元，這是14年來的首次。孫正義在此前舉行的財報發布會上承認：「我在投資判斷上出了問題，目前正在深刻反省自己。」

除了WeWork和Uber，願景基金在近日計劃將其在寵物護理軟體Wag Labs近50%股份出售，此舉正值這家初創企業大幅裁員之際。作為協議的一部分，Wag Labs將重新控制屬於軟銀的兩個董事會席位。去年年初，願景基金向Wag Labs投資3億美元。在公司裁員之際做出這樣的選擇，軟銀似乎又「敗了一局」。

可以預見的是，如果銀行業收緊錢包，可能會使軟銀的融資計劃變得更加複雜化。未來幾周，軟銀可能會迎來一個關鍵考驗，因為其正在努力敲定高達3000億日元(約合27億美元)的新融資條款。此外，孫正義可能會轉向其他融資渠道，包括利用軟銀390億美元的現金和等價物，該公司還可以發行更多債券，儘管這些債券的利率高於銀行貸款。接下來可能還有最大的王牌：截至上周五，軟銀持有的阿里巴巴股份價值1290億美元。
