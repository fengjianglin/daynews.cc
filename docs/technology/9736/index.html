Q3全球半導體公司排名：英特爾第一，日本企業重返前十

英國市場趨勢研究公司IHS Markit於12月5日（歐洲時間）發布了2019年第三季度的十大半導體公司排名。英特爾第一，索尼首次進榜。

第三季度市場增長同比下降14.7％

2019年第三季度的半導體市場為1113億美元，同比下降14.7％。此外，第二季度下降了15.3％，第一季度也下降了12.4％，內存市場在2019年1月至2019年9月期間比去年同期下降了34.4％。在第四季度，儘管存儲器市場一直處於復甦趨勢，但整個半導體市場一直停滯不前，因此，市場規模預計將同比下降7％，並將繼續負增長。因此，IHS預測，全年半導體市場將同比下降12.4％，下降兩位數。儘管自第三季度以來內存領域開始出現復甦的跡象，但據說內存價格大幅下降和其他因素的影響在全年中是巨大的。IHS Markit的高級研究分析師Ron Ellwanger說：「由於庫存過多以及雲計算和企業伺服器業務的需求疲軟，第三季度內存銷售為負。但是，與上一季度相比，它正增長，可以說存儲器市場正在逐步恢復。」

半導體市場的季度趨勢。儘管從2019年第二季度開始有輕微復甦的跡象，但與去年同期相比仍繼續以兩位數下降（來源：Informa Tech的IHS Markit）

日本企業重返前十

第三季度排名第一的是英特爾，比上一季度增長了16.3％。SSD和內存市場也在增長，該領域的環比增長37.2％。

2019年第三季度排名前10位的公司（來源：

Informa Tech / IHS Markit）

IHS Markit首席分析師弗拉基米爾·加拉博夫（Vladimir Galabov）表示：「英特爾在所有應用程序中，特別是在無線領域，都具有強大的性能。英特爾是設備的設備數據機。」 「我們正在為蜂窩基站提供FPGA和ASSP，該公司正在受益於該領域不斷增長的需求。」

索尼公司的子公司索尼半導體解決方案公司是前十名中唯一的日本公司。

該公司第三季度業績同比增長27.1％，比上一季度增長41.5％。結果，排名從上一季度的第15位躍升至第6位，並排名前10位。這是索尼首次進入前十名，IHS稱該公司為今年半導體市場的「閃亮之星」。

索尼的CMOS圖像感測器業務蓬勃發展，這歸因於智能手機本身的銷售增長以及安裝多台攝像頭（主要是高端設備）的趨勢。此外，該公司目前正進行資本投資以實現進一步的產量增長，並計劃在長崎技術中心附近地區開始建造新的大規模生產工廠，由於產量增加，銷售額將進一步增加。
