定投指南

01 如何選擇定投標的

在肖恩看來，在定投加密資產的過程中，如果選對了適合定投的標的，那麼你的定投策略應該成功了一大半，長期增值的優良資產，是大家定投成功的基礎。

目前幣圈有上千個加密貨幣，但並不是每一個幣種都適合定投，以肖恩多年的定投經驗來看，大家應當選擇長期看好、未來確定性大、存活率最高、存活時間最長的幣種。這樣的標的才最有穩定性，也最適合定投。就目前來看，比特幣應該是最佳定投標的了；其次是三大交易所平台幣。

大家要知道，比特幣是區塊鏈技術的首個成功應用，在一定程度上象徵著區塊鏈的風向標，如果連比特幣都不行的話，那其他項目估計就更不行了。不過就目前來看，比特幣有著龐大的用戶群體，其市值占整個加密市場50%以上，崩盤概率極小，因此風險也相對較小；肖恩相信，隨著區塊鏈行業的發展，更多用戶將會接觸到比特幣，比特幣毫無疑問將會是最大受益者，其未來增長空間較大，所以我認為定投比特幣是最佳選擇。

02 合理的定投方式

對於幣圈定投的方式，肖恩簡單分為兩種，一是不定期不定額定投；二是定期定額或定期不定額定投。

不定期不定額就是，投資者有閑錢就加倉，不論時間，不論數量，零散投資，日積月累，達到定投目的。這種方式適合收入方式、金額不固定的投資者。

定期定額/不定額就很好理解了，確定你的投資周期，是每周，還是每月，最好固定下日期，比如每周六，每月1日等等。確定你每期的投資金額，根據個人情況制定方案，當然也可以每期不固定金額進行定投；最好每次記錄下自己的投資時間、投資金額、投資數量。這種方式更適合工薪族。肖恩建議大家，如果對於項目的趨勢長期看好的話，項目價格越低，可以適當增加定投額度。

大家可以根據自身情況來選擇適合自己的定投方式。

03 怎樣定投最划算

在肖恩看來，在定投的同時，各大交易所的交易手續費也不容忽視；因為我們是長期定投，日積月累下來，交易費數目也不小。

假設每期定投1000元，一年52周，一共5.2萬，那麼通過哪家交易所購買幣種划算呢？

假定交易平台 A 的手續費 0.2%, 交易平台 B 的手續費 0.15%，每周定投1000元，一年52周，定投7年。

0.2% x 1000 x 52 =104元，7年是104 x 7=728元；

0.15% x 1000 x 52 =78元，7年是78 x 7=546元；

摺合成人民幣，只差182元，好像不多；但如果摺合成 BTC，且BTC在未來大漲的話，那麼最終差額可能會更大。

04 如何選擇定投日

如果總想追求最低價，可能需要花很多時間盯盤。這樣的話，原本簡單的定投動作就變得複雜，會增加定投的難度。那麼如何讓定投大概率買在最低價，或相對最低價，但又不花大量的時間盯盤呢？對此，肖恩通過BTC的歷史數據來告訴大家，究竟何時定投較為合適。

肖恩通過技術手段在www.coinmarketcap.com 網站爬取了比特幣從2013年4月29日以來的所有歷史數據。接下來我們再詳細分析一下這些數據。

經過統計後，肖恩得出以下數據：

星期二出現最低開盤價次數共46次；

由數據可知，如果定投比特幣，星期一出現一周最低開盤價的次數最多，高達105次，比第二名還要高出2倍之多。可見，比特幣的定投日選在周一更為合理。還有一點要說的是，本文是以比特幣開盤價進行的數據分析，所以在周一的凌晨進行定投操作最好。
