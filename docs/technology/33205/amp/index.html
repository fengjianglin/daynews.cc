【烏茲別克當局修訂加密監管制度，公民只能出售代幣】

據news.bitcoin12月24日報道，烏茲別克國家項目管理局(National Agency for Project Management)最近通過了對監管制度的修訂，大幅限制了本地個人使用加密貨幣的選擇。修訂製度已禁止居民在該國許可的數字資產交易所購買加密貨幣。加密貨幣持有人只能出售他們的代幣，前提是這些代幣不是通過匿名交易獲得的。

據《經濟日報》12月24日消息，中國銀保監會在近期重點工作通報會上發布了發展普惠金融、支持小微企業等的最新成果和主要舉措。銀保監會有關部門負責人表示，銀保監會將引導各類商業銀行細分客戶群體，避免「壘小戶」，形成既相互競爭,又各有側重小微金融市場格局。支持政策性銀行通過轉貸款方式與中小商業銀行合作，加大小微企業融資支持力度。鼓勵商業銀行利用互聯網、大數據、人工智慧、區塊鏈等技術創新產品和服務方式，鞏固已經初步形成的「能貸、會貸」局面。

【鎮江特大「礦機」盜電挖比特幣案開庭，主犯最高獲刑13年】

據法制網12月24日消息，江蘇省鎮江市丹徒區人民法院依法公開開庭審理了鎮江特大「礦機」盜電挖比特幣一案，併當庭宣判。據悉，此案是江蘇省有史以來涉案金額最大的一起盜電案件。法院經審理查明，為降低運行比特幣「挖礦機」高額的電力成本，2017年3月至2019年5月間，被告人蘭某鋒、李某、毛某海共謀在鎮江市丹徒區等地租用場地，通過互感器短接等方式，盜竊國家電力挖取比特幣。其中，主犯蘭某鋒涉案金額超1378萬元。庭審中，蘭某鋒等人自願將公安機關在偵查階段扣押的近4000台「挖礦機」以及賬戶中持有的14枚比特幣作為違法所得退賠給供電公司。根據各被告人犯罪的事實、犯罪的性質、情節和對於社會的危害程度，合議庭經評議後，分別依法判處10名被告人3年至13年不等刑罰，並處罰金。

【因虛擬幣支付密碼錯誤無法轉移資金，用戶將虛擬貨幣網站訴至法院遭駁回】

據新京報12月24日消息，海淀法院一審判決駁回一虛擬貨幣貨幣支付案件原告。據悉，在該案件中，由於多次輸入虛擬「萊特幣」支付密碼未果，且同時不能提供重置密鑰憑證，錢包賬戶無法轉出。原告李先生將虛擬貨幣網站公司訴至法院，要求網站協助其在網站重設支付密碼，並保證儲存在該賬戶的虛擬資產在該平台安全轉移。法院認為，根據現有情況，網站已經表示無法重置支付密碼，且李先生也無法證明，在沒有提示符的情況下，網站具有為客戶重置密碼的義務及能力。因此，李先生的訴訟請求缺乏事實依據，法院不予支持，故判決駁回原告李先生的訴訟請求。

【普華永道加密主管：預計更多機構將從2020年起開始入場】

據Bitcoinist12月24日消息，普華永道亞洲金融科技及加密主管、香港金融科技協會主席Henry Arslanian最近在接受彭博社採訪時表示，Libra是導致全球央行對加密更感興趣的原因，起初銀行對虛擬貨幣的反應非常緩慢。然而，如今世界上超過70%的央行都在研究加密。他相信，在接下來的幾個月里，當大型跨國組織對加密技術表現出興趣時，將會出現重大的變化。Arslanian稱：「比特幣仍然是所有加密貨幣之母。而Libra仍依賴法定貨幣作為基礎資產。另一方面，比特幣是去中心化的。至於其他加密資產，它們都有自己的用例和用途，這就是為什麼它們可能同時存在的原因。」此外，Arslanian預計更多的機構將從2020年起開始入場。原因之一是機構客戶對加密產生了更大的興趣。數字資產領域得到了許多規範的解決方案，如加密託管、基金和規範的工具，如期貨。這些都將允許傳統的玩家以一種他們更熟悉的方式接觸加密生態系統。與此同時，Arslanian還補充道，大量的數字和虛擬銀行正在湧現。

【李銀科：2020年是區塊鏈的轉折之年，供應鏈金融、數字資產治理領域具有發展潛力】

工信部中國電子商會區塊鏈專委會秘書長李銀科12月24日在接受採訪時表示，就技術而言，區塊鏈是一個去中心化的分散式賬本，也可以理解為資料庫。若不和大數據、人工智慧、物聯網、雲計算等技術融合應用，單一解決問題的能力非常有限。李銀科認為，2020年是區塊鏈的轉折之年。當一項技術上升到國家戰略層面，充分說明了區塊鏈在未來信息技術領域的重要性，但機遇和挑戰並存，區塊鏈的從業者應找准自身的角色定位。近年來打著區塊鏈名號的風險事件比較多，在金融領域尤其要注意。區塊鏈從業者可以在供應鏈金融領域或是數字資產治理領域開拓業務。

【賽迪區塊鏈研究院院長：區塊鏈帶來了資料庫結構的創新】

12月24日，新華網主辦的2019第六屆金融企業社會責任論壇在北京舉辦。賽迪區塊鏈研究院院長劉權在發言中表示，區塊鏈帶來了資料庫結構的創新。傳統的資料庫結構是樹狀，而區塊鏈是鏈狀，可實現數據全生命周期管理可追溯。劉權說，「區塊鏈建立了非常複雜的關係網路，再加上資料庫，這就是一個完整的區塊鏈。關係網路能夠實現兩個節點之間的數據互通，而資料庫則圍繞數據對象，記錄其整個生命周期。」 劉權表示，區塊鏈具有去中心化、高度透明、可靠資料庫等特點。區塊鏈技術的價值體現在為信息防偽與數據追蹤提供了革新手段，有效解決目前信息化應用突出的信息保護和授權訪問問題，有助於實現用戶對數據的控制權等多個方面的難題。
