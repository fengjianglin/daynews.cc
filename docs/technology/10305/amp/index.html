“plus token”涉傳銷！南寧上千名投資者損失慘重，卻有多人不願報警

天天要聞
2019-12-05

近日，有多名市民向南國早報客戶端記者反映，他們參與投資的一個「plus token區塊鏈錢包」項目疑似跑路。據估算，南寧有上千人參與投資，每人投入數千元至上百萬元不等，損失慘重。然而，不少受害人還心存幻想，不願報案。

心動：抵不住誘惑，先後投入60萬元

今年2月底，雷女士通過老同學雷某介紹，開始接觸「plus token區塊鏈搬磚套利錢包」，該項目稱，只要通過數字貨幣交易市場買入比特幣等七種數字貨幣，存入錢包，該平台就會在各大數字貨幣交易市場進行高賣低買，通過高頻量化交易實現搬磚套利，每月有8%至20%的收益。


「plus token區塊鏈錢包」之前打出的廣告。

今年3月，雷女士被高收益打動，嘗試投入了4萬元，剛開始，收益每天到賬還能隨時提現。

「plus token區塊鏈錢包」之前打出的廣告。

後來，投資群里又發布了更「誘人」的福利信息，嘗到甜頭的雷女士便加大了投資力度，先後投入了60萬元。

調查：網站留言無回應，APP已無法下載

然而，6月27日，該平台突然無法提現。7月10日，平台APP也無法登錄，所有交易停止。據雷女士介紹，從今年3月至今，她先後獲益4萬元，相當於損失了56萬元。

同為該項目投資人的市民潘女士也損失了20多萬元。據她介紹，該項目分別普通用戶、大戶、創世等等級，發展下線便有層層獎勵。原本負責南寧市場推廣宣傳的一名創世用戶（即最高等級）是一名湖南籍女子。如今，該女子已聯繫不上。

12月4日，雷女士告訴南國早報客戶端記者，事發後，她本想讓「大咖」級的老同學雷某帶她們到公安機關報案維權，卻發現自己已被拉黑。5日上午8時23分，記者向雷某發送了採訪簡訊，但目前沒有得到任何回應。

「plus token」全球中文社區的網站首頁截圖。

同日，南國早報客戶端記者登錄「plus token」全球中文社區的網站，發現首頁只顯示「開網倒計時中……」。於是，記者通過在該網站留言，也並未收到回應。記者又試圖下載該平台APP，發現已無法下載。

「plus token」 APP還能登錄時的截屏。受訪者供圖

深思：有人仍心存幻想，不願報案

「目前，南寧才有10人左右去報案。」據潘女士介紹，今年7月發現被騙後，她是首個站出來維權的，還一度被人威脅說要把她人肉出來打。

雷女士也表示，事發後，該平台有些「大咖」級以上的人員主動跳出來「維穩」，聲稱如果報警，「開網」後賬戶要被封掉。因此，很多投資者擔心被報復不敢報警，還有人仍對「開網」心存幻想不願報警。前段時間，她加入了廣西的一個維權群，截至目前只有80多人加入。

關於該項目的網上檢索結果。

12月5日，南國早報客戶端記者上網搜索，發現有外地警方已認定該項目屬於傳銷組織，並抓獲了相關嫌疑人。

受害人的報案回執單。

目前，潘女士、雷女士等人已報案，警方仍在進一步調查中。
