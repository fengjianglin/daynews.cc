新能源物流車市場前11月大跌近4成 全年還能有盼頭嗎？

天天要聞
2019-12-16

第一商用車網 謝光耀

正當重卡、輕卡市場四季度表現連續超預期的同時，商用車市場的另外一個細分領域——新能源物流車市場，卻已經一路跌至低谷。

根據第一商用車網最新掌握的數據，今年11月，我國新能源專用車市場（以新能源物流車為絕對主力）共計生產各類車型10565輛，比上年同期的2.24萬輛大幅下滑53%；今年1-11月，我國新能源物流車市場僅僅累計生產4.73萬輛，比上年同期的7.66萬輛大幅下滑38.2%。

市場已經低迷至此，全年還能有盼頭嗎？

2014年-2019年11月新能源專用車市場月度走勢圖（單位：輛）

從上圖可以看到，新能源專用車市場今年從1月到11月，有8個月都處於深度下滑，只有3個月是同比上升的。根據第一商用車網掌握的數據，同比上漲的月份分別是3月份，同比增長12.5%；5月份，同比上漲53.3%；6月份，同比上升127.1%。5月和6月大幅增長的原因，主要是補貼過渡期的設置。根據規定，在3月26日至2019年6月25日的過渡期內，符合2018年技術指標要求但不符合2019年技術指標要求的銷售上牌車輛，按照《財政部 科技部 工業和信息化部 發展改革委關於調整完善新能源汽車推廣應用財政補貼政策的通知》（財建〔2018〕18號）對應標準的0.1倍補貼，符合2019年技術指標要求的銷售上牌車輛按2018年對應標準的0.6倍補貼。而過渡期一旦結束，針對新能源專用車的補貼金額便瞬間下降超過50%，如此一來，客戶在5月和6月多下訂單、多買車，對這兩個月市場的產銷量大漲起到了推波助瀾的作用。

北汽新能源物流車

下滑的月份則包括：1月份，同比下降86.2%；2月份，同比下滑80.8%；4月份，同比下滑52.4%；7月份，同比下滑79.5%；8月份，同比下滑73.4%；9月份，同比下滑69.5%；10月份，同比下滑31.2%；11月份，同比下滑52.9%。下滑的月份數量之多和下降的幅度之大，令人揪心！

2013年至2019年11月新能源專用車市場年度走勢圖（單位：輛）

從上圖可以看出，新能源物流車市場從2015年進入快速發展通道，2017年達到發展的巔峰，當年產量超過15萬輛，達到15.45萬輛。2018年，新能源物流車市場首次下滑，同比下滑達到25.3%，跌至11.54萬輛；2019年1-11月，市場再度下滑38.2%，降至不足5萬輛，市場持續進入深度調整期，全年產銷量預計難以達到7.5萬輛以上的規模。

第一商用車網分析認為，市場降幅如此慘烈的原因，主要就是補貼大幅退坡。根據今年3月26日財政部聯合工信部、科技部和發改委發布的「關於進一步完善新能源汽車推廣應用財政補貼政策的通知」，3月26日至6月25日的過渡期結束後，按照2019年新能源汽車補貼新政，新能源貨車可享受的中央財政單車補貼上限只有5.5萬元，相比2018年退坡45%；每度電的補貼標準從2018年的最高850元下降至2019年的350元，退坡幅度最高達58.82%。

2019年1-11月，新能源專用車的市場結構發生新的變化，總質量3.5噸及以下的微型新能源物流車佔比從2018年的62.58%增長至今年的71.38%，市場佔比進一步擴大。其中原因，主要是補貼的大幅退坡，而微卡型物流車和微面型物流車售價相對較低，因此相對較受平台型客戶的歡迎。3.5-4.5噸的輕型新能源物流車（輕卡和輕客型）佔比從去年的35.29%下滑至今年1-11月的15.27%，也是政策退坡進而導致產品售價上漲的結果。

比亞迪T10ZT純電動智能渣土車

值得一提的是16噸及以上的重型新能源專用車，其市場佔比從去年的0.43%迅速擴大至今年1-11月的近10%，究其原因，主要是比亞迪純電動重型自卸車今年以來在深圳開花結果，為整個市場貢獻了接近3000輛的數量。

2016-2019年氫燃料電池物流車市場走勢圖

上圖顯示，雖然新能源物流車整體市場表現不理想，但氫燃料電池物流車走出了自己的風景線。今年1-11月，氫燃料電池物流車共計生產846輛，同比增長近25倍，全年破千在望。其中原因，跟氫燃料電池商用車財政補貼不退坡、氫能產業鏈投資增多、試運行地區越來越多等因素有直接關係。
