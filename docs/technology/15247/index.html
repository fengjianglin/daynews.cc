小米盧偉冰回應"低價誤國"：華為是"低價"戰略榜樣

2019-12-16 950條評論 1,994次閱讀 325人點贊

一直以來，小米都堅持「極致性價比」的路線，在市場上廣受認可。不過最近有人卻對小米的低價模式進行攻擊，認為小米的低價後果嚴重，甚至會成為拖累國家往世界級水平升級的絆腳石，最後已經上升到「低價誤國」。


針對此事，小米中國區總裁、紅米Redmi品牌總經理盧偉冰在微博回應稱，華為是中國企業「低價」戰略的榜樣，並且發達國家也有諸如宜家、Costco等品牌，小米有沒有錢做研發，看看小米財報就可以（關鍵要會讀）。


小米發布的Q3財報顯示，小米正在全力推進5G手機研發，研發投入不斷加大，2019年第三季度研發開支人民幣20億元，同比提升32.5%。

以下為回應全文：

今天很多人@我希望我評論一下，簡單說幾句:

1.當一個人認為是中國的社會文化和結構出了「問題」的時候，一定是他自己腦袋出了「問題；

2.以發達國家來「例證」本身就是缺乏民族自信的表現，況且「發達”國家出現了沃爾瑪，宜家，Costco, 優衣庫這些公司和品牌，全球受益；

3.小米有沒有錢做研發，看看小米財報就可以（關鍵要會讀）。小米的員工很多來自全球一流高校的人才，沒有高薪吸引不來；

4. 我深度研究了華為的發展路徑，熟讀任總講話以及任總在不同階段關於手機終端的戰略講話，華為是中國企業「低價」戰略的榜樣，並且雙11智能手機最低價399也是友商某型號。

最後提醒，「謬論」通篇全部是逗號，只有一個問號，連句號都沒有。建議還是先回小學學好語文，回中學學學邏輯再出來發表「大論」為宜，否則就有點丟人現眼了。
