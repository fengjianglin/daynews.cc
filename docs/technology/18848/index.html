雷軍50歲生日，小米卻被指“低價誤國”；柯潔稱這輩子都下不過AI

2019-12-16 594條評論 2,176次閱讀 664人點贊

中興活過來了！已恢復到2017年營收水平

近日，中興通訊高級副總裁劉健在中國經濟年會上透露，中興已經恢復到2017年的收入水平，毛利率超過2017年的32%。中興方面表示，2019年全年預計盈利43億元至53億元，而2018年同期，公司虧損69.83億元。

隨著盈利水平提高，中興也加大了創新力度，5G時代的到來，對於中興而言是一個重大機遇。劉健表示，中國的5G技術水平已經走到世界前列，華為和中興的5G技術水平領先愛立信和諾基亞10~11個月。

種種跡象都表明，遭遇重重難關的中興活過來了。

雷軍50歲生日淡定又從容，小米卻被指「低價誤國」

12月16日，小米創始人、小米集團董事長兼CEO雷軍迎來50歲生日，小米CEO助理陳曦祝福道，「50歲，人生剛剛開始」，淡定又從容。

但紅米品牌鍾經理盧偉冰卻不怎麼淡定，昨日有網友發布「小米低價誤國」言論，聲稱小米這種公司在中國能擁有打臉給粉絲和市場，本身就是有問題的。對此，盧偉冰回應稱，以發達國家來「例證」本身就是缺乏民族自信的表現，況且「發達」國家出現了沃爾瑪、宜家、Costco、優衣庫這些公司和品牌，全球收益。至於小米有沒有錢做研發，看看小米財報就可以。最後盧偉冰還反擊稱，華為是中國企業「低價」戰略的榜樣，並且雙11智能手機最低價399也是某友商型號。

華為Mate X 5G總是秒售罄，官方回應：用戶太捧場

售價16999元的華為摺疊屏手機Mate X 5G已經在官方商城經歷多輪銷售，卻還是有很多花粉沒有搶到，幾乎每一輪開售都是秒售罄。開售一個月了，卻還是如此難買，網友不免懷疑華為供貨能力不足，或是在效仿小米搞飢餓營銷。

面對質疑，華為方面回應稱，華為的產品外觀及配置功能等十分突出，當前收到許多用戶捧場，望請體量。也就是說，華為Mate X 5G粉絲太多，才導致了供不應求的情況。

不過，目前華為官方商城的Mate X 5G僅有446條評價，產量的確趕不上用戶需求。

柯潔稱這輩子都下不過AI了，但仍會堅持下去

柯潔在和R1SE交流的過程中坦言，自己和人工智慧的實力實在是相差太多，在和AI的對戰過程中，始終無法看破對手後半程的棋路。以前都是跟人類訓練，現在都是用電腦練，因為下不過AI，所以只能跟AI練。

現場有人提問，沒人下得過AI了嗎？柯潔回答：我只能說我這輩子是做不到了。

即使可能是 「下一百盤輸一百盤」，即使「知道不可能贏」，柯潔也還是會一直堅持下去，因為下棋是他最喜歡的事情。

人生總是有些明知不會成功，卻還要去做的事……
