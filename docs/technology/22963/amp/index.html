隨銳發布全國產化視頻通信解決方案 助力政企實現高清視頻通信

天天要聞
2019-12-19

隨著科技不斷進步和發展，人類將迎來信息化時代。然而，人們在享受先進的信息通信技術帶來的方便與快捷的同時，也面臨著信息安全的威脅。近年來，全球信息泄露事件頻發，為許多國家、企業和個人造成了無法挽回的損失。如何加快推進國產信息化替代，構建安全可信賴的溝通協同體系成為重要課題。

隨銳會見在視頻通信領域率先推出全國產化視頻通信解決方案，構建從視頻通信平台、會議室專用的視頻通信硬體終端，到個人桌面件客戶端、集成定製開發的SDK/API的全國產化產品體系，只要網路可達信息溝通即可達，視頻通信不再局限於固定會議室，不再局限於固定網路，真正實現了隨時隨地溝通無限。同時隨銳完全依靠自身研發團隊，全面掌握視頻通信底層核心技術，從架構設計、編碼實現、升級維護完全自主研發，擁有所有源代碼與自主知識產權。

1.全國產化視頻通信服務，採用雲轉發架構，國密演算法加密，構建自主可控安全通信基礎

傳統視頻通信解決方案在服務端一般採用全編全解模式為各終端發過來的音視頻數據流進行編碼和解碼，因此對伺服器的性能要求較高，佔用伺服器資源較多，部署成本比較高。所以傳統視頻通信解決方案沒有在自主可控服務平台得到大規模商業化普及。

隨銳會見基於雲架構及服務模塊化設計，採用轉發架構，可根據不同客戶端、不同網路，智能選擇發送不同解析度和幀率視頻流程，使對視頻、速率的處理機制可被運用到每個節點，對每個節點的視頻速率、編碼格式進行獨立處理，實現系統對視頻的處理優化到每個節點，真正達到以點為核心，從而降低對伺服器硬體的要求；基於私有雲、混合雲、雲端部署，可部署在全國產飛騰FT1500A、飛騰2000+伺服器上，適配銀河麒麟操作系統、神通資料庫、TongWeb中間件，從而實現了1080P全國產化多人實時高清音視頻通信。以單台飛騰2000+伺服器為例，會見可實現300方1080P多人音視頻實時通信，大大降低了部署成本，適合大規模普及化部署。

隨銳會見在業內率先通過了」PK體系」（「飛騰—麒麟國產化體系」）認證。針對視頻通信安全要求，會見媒體流和數據流採用國密SM4加密，媒體流經過編碼打包處理，再進行數據加密和傳輸，客戶端收到後再進行解密；數據傳輸全程加密，任何第三方拿到數據均無法使用，確保數據在傳輸過程中達到高可靠安全級別。

2.全平台融合互聯，多種類型設備接入

不同行業採用的視頻通信終端類型不同，如何解決用戶統一管理、設備統一接入、統一控制等需求，是視頻通信行業的存在已久的難題。會見視頻通信服務內置融合網關模塊，該模塊可實現不同類型協議、碼流的轉換，在會議室、桌面機接入的同時，還可以將執法儀、手機、無人機、監控設備、AR穿戴設備、第三方品牌會議室設備等統一接入，同時提供多種許可權的統一管理平台，可實現不同類型設備的音視頻交互，解決了一直以來困擾政企客戶端的難題，為客戶帶來統一通信、融合互聯的全國產化視頻通信新體驗。

同時隨銳會見支持政企內網（專網）與外網融合通信，通過內外網部署、分散式部署等多種部署模式，解決網路帶來的隔絕問題，無論客戶身處何種環境，只要部署了隨銳會見產品，就能隨時隨地開展高清流暢的視頻通信。

3.1080P高清音視頻通信終端，助力國產化視頻通信

2019年10月，隨銳研發了業內第一款基於飛騰晶元的1080P高清視頻通信終端產品。該產品採用 SVC 視頻分層編碼技術與 OPUS 音頻編碼技術,是目前基於飛騰FT-2000 晶元、銀河麒麟操作系統、自主研發的視頻通信終端產品，面向政企黨政軍客戶，該產品實現了點對點、多人實時高清音視頻通信、屏幕共享、會場控制、布局切換等功能，可滿足日常視頻會議、視頻指揮、視頻監管、多方視頻協作等要求，完全具備與國內外同類產品對標競爭的實力。

隨銳會見國產化高清視頻通信終端

同時，隨銳擁有低延遲網路自適應技術，可根據各個終端呼叫上下行帶寬的變化，動態調整傳輸給終端的解析度、幀率，從而保證各方視頻通信穩定流暢進行。採用前向糾錯傳輸演算法，NACK重傳機制，超強丟包補償演算法，以及最優化音視頻帶寬分配演算法，使得在較高丟包率的惡劣網路條件下，依然保證音視頻流程。

4.高清國產化桌面軟體客戶端，視頻通信高效便捷

針對國產化台式機等桌面終端用戶，隨銳推出了PK（飛騰—麒麟）軟體客戶端，用戶只需要在桌面上安裝隨銳客戶端軟體，即可開啟多人視頻通信。一鍵開啟會議，即可開啟音視頻會議；多種方式入會，例如會議ID、鏈接、會議列表等一鍵入會；會議中可共享文檔、視頻，發起共享方可以標註；通訊錄邀請、控制其他參會方靜音、視頻顯示布局切換、錄製等多種功能更，滿足客戶多場景的需求，視頻通信不再受時間、地點的限制。

隨銳會見全國產化視頻通信解決方案，實現了真正的高清視頻通信，可適應大並發、超大用戶量多方會議，容錯能力強、帶寬和計算能力成本可控，支持全虛擬化部署，用戶通過硬體終端、PC、Pad和手機即可實現隨時隨地入會。可應用於視頻會議、指揮調度、視頻監控協同等重要應用，還可以實現遠程培訓、學習交流等服務功能，無需會議室，只需在辦公室就可以隨時安全高效地進行溝通協同。

隨銳科技集團是業內領先的人工智慧與通信雲產品運營商，始終聚焦「自主品牌、自主研發」，每年為50餘萬家政企客戶提供優質的人工智慧與通信雲產品及服務。未來，隨銳會見在國產信息化替代浪潮中，將繼續加大對視頻通信底層技術研發與產品創新，為黨政軍、公檢法、企事業單位提供標準化、人性化、智能化、高可靠、高安全的視頻通信產品與服務。
