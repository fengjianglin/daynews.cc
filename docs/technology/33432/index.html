向“GPS”宣戰！Fujifilm將以1790億日元收購日立影像相關業務

2019-12-23 3,279條評論 4,447次閱讀 3427人點贊

前言
隨著科技的發展，臨床診斷的70%都依賴於醫學影像。然而，全球醫學影像市場60%的市場份額都幾乎被「GPS」（GE醫療、飛利浦醫療、西門子醫療）這三大巨頭壟斷。近日，Fujifilm宣布將以1790億日元（約16.3億美元）收購日立影像診斷相關業務，向「GPS」宣戰！

一、日立將成立全資子公司，並把股份轉讓給Fujifilm

Fujifilm在新聞稿中稱：日立將拆分其影像診斷業務，成立全資子公司，並把子公司的全部股份轉讓給Fujifilm。根據12月18日披露的購股協議，此次收購價格預計約為1790億日元。日立在1953年就推出了X線機，並提供CT、MRI、X線機和超聲在內的全套影像產品。通過收購日立的影像診斷相關業務，Fujifilm將提供包括CT、MRI、醫療IT、體外診斷和內窺鏡檢查在內的一站式整體解決方案。這將增強Fujifilm為醫療機構提供全面解決方案的能力。此外，覆蓋「診斷」和「治療」的綜合醫療保健公司Fujifilm將能夠開闢新業務，從而使其醫療系統業務與醫療保健領域其他業務（例如藥品和再生醫學）產生協同作用。Fujifilm還在新聞稿中透露，Fujifilm將利用兩家公司廣泛的銷售渠道，通過交叉銷售進一步擴大全球市場的業務。

（日立的影像設備）

二、日立接下來的路會怎麼走？

日立稱剝離了影像診斷業務之後，將把重點放在放療上。日立的質子治療設備迄今為止已經服務了6萬多名患者。日立的離散點掃描質子治療技術是首個獲得美國FDA批准上市的質子治療技術並在日本也獲得了藥品和醫療器械局（PMDA）的批准。日立的離散點掃描技術可實現PBT（質子束治療）的最先進治療應用，例如強度調製質子治療（IMPT）。日立還將加強其體外診斷系統業務，使疾病得以儘早發現，並為未來在再生醫學領域擴展細胞治療解決方案奠定基礎。日立稱，將繼續在IT領域提供高增值服務，利用其在數字技術方面的優勢，拓展醫療保健業務。

三、「AI+醫學影像設備」會是Fujifilm的出路嗎？

Fujifilm是日本精密化學製造、膠片、存儲媒體和相機生產商，在亞洲、歐洲和北美洲擁有超過220家研發、生產以及銷售的機構。膠片業務受到衝擊後，Fujifilm轉而研發數字相機業務，並憑藉多年來涉足化工產品的經驗，發展起藥物和化妝品等多元化業務。

在2019ECR（歐洲放射學年會）上，Fujifilm曾展示過其REiLI醫學影像與信息學AI。當時Fujifilm透露正在開發一系列AI技術，如智能勾畫靶區。REiLI的其他正在開發的應用包括計算機輔助診斷。Fujifilm AI的另一個新應用：FCT Pixel Shine則提供了低劑量CT影像的圖像處理技術。Fujifilm曾展示過一台64排CT，它配有嵌入Synapse 3d重建模塊的控制台。Fujifilm的FCT Pixel Shine技術，它能使用AI演算法在極低輻射劑量下實現高效的圖像處理。

此次收購後，Fujifilm將把獨特的AI圖像處理技術應用在日立的影像設備上。通過廣泛的產品組合，Fujifilm將加強與醫生及醫療機構的關係，並獲得高質量的影像和操作數據。通過利用這些數據，Fujifilm將能夠擴展到新的領域，包括「 AI輔助診斷」和「 AI輔助維護」。

結語

去年，Fujifilm在420億美元的醫學影像市場中佔據了5.5%的份額，日立佔了2.9%的份額。而西門子醫療以23.1％的市場份額領先市場，隨後是GE醫療和飛利浦醫療，分別佔據了21.8%和20.5%的市場份額。Fujifilm此次重金收購日立的影像診斷業務能否與GPS抗衡，還需時間來證明。
