一、曉峰視點

今天，首先來看看好的，特斯拉股價飆升：

和19年年中的180美元相比，特斯拉股價創了歷史新高，當時特斯拉還受到輿論普遍的質疑，覺得他第二天就要關門了。但事實上，是特斯拉搞了一個關門打狗，據說空特斯拉的空頭，虧損已經超過了20億美元。

而特斯拉的成功大家都知道，卡車、SUV，還有中國廠的竣工，對特斯拉而言都是一片凱歌奏響，當然也帶動了其整個下遊行業的走強。

另一個類似的亮點是蘋果，蘋果在今年給大家帶來了一個驚喜，一不小心把一個老產品賣成了爆款，就是他的applepods pro的降噪耳機。在美國市場，聖誕還沒有到，已經賣脫銷了。

我的朋友在我生日前幫我定了一副做生日禮物，結果要在我生日過後20天才能拿到貨，真的是非常的誇張。看著蘋果耳機大賣，我覺得最傷心的是BOSS和SONY，他們把市場培育好了，果子被蘋果摘了。

從上面兩個公司的事件可以看出，高科技路線的消費品容易出爆品，這似乎是情理之中的事情。當然，沉澱和運氣也佔了很大的比重。

這個時候，再來看我標題提到的這個新聞和行業，對比就強烈了。

阿里巴巴這個名字在當今中國可謂是成功的代名詞，具備了點石成金的功能。但是在影視行業，它也是連續虧損。

實際上，不僅一直在虧，而且越虧越多。

阿里影視的這個新聞，還提到公司目前收入的80%來自互聯網發行。這不由得讓我想到還有一家美股，也就是看盜版美劇一定會碰到的一家公司：

這家公司在今年的7月到9月經歷了一波大跌，股價從近400美元跌到250美元。原因似乎也很簡單，就是在他的傳統競爭對手，比如HBO之外，又加入了新的對手，蘋果！

邏輯就不用多推演了，既然能夠嚇得住美國投資者，那中國投資者也能輕易想明白其中的邏輯。問題是，最近股價又漲回到325的水平了，比美股大盤要漲的好。

這又是為什麼呢？

其實，原因並不重要。我們關心的是中國的影視類股票，以及做影視發行的公司，未來是不是存在兩方面的難題。

一個行業本身。很多影視的想法是，反正業績不好，理由問市場就好了。

比如，我過去多少年一直都是影院的常客，大片幾乎一部不漏。但現在去得少了，當然不是缺票錢，而是宅在電腦和手機上，已經把大部分片子給消滅掉了。這是普遍現象嗎？

其實我現在越來越覺得一個人的關注力，或者說消費力，確實是最大的生產資料吧。消費者只有首先關注了什麼，才有可能產生消費。

那這個行業的第二個問題就是，隨著中美貿易協議的簽訂，如果柰飛，蘋果、HBO，都被允許進入中國市場呢？本國影視公司是不是會更加舉步維艱？

今天直到收盤，市場表現平穩。

指數先抑後揚。

正如我昨天預計的，大盤在這裡要做個小中樞。只有做成了，才能繼續上漲。

15分鐘，跌得有點多，明天就能看出方向了。雖然短期會有起伏，但大概率還是漲的。

今天工業大麻板塊大漲，但是美國大麻股都跌得稀里嘩啦的。兩地的邏輯是不一樣的，不過還是應該參考一下。
