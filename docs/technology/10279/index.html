郭明錤預測蘋果明年發5款iPhone，4款支持5G

2019-12-06 2,229條評論 4,513次閱讀 3509人點贊

科技訊 12月6日消息，據外媒報道，知名蘋果分析師郭明錤(Ming-Chi Kuo)發布最新報告預測，蘋果將在2020年發布四款5G iPhone 12和「iPhone SE 2」，並在2021年推出「iPhone SE 2 Plus」，這些手機可能引發分析師們期待已久的「超級周期」。


圖1：郭明錤預計蘋果2020年和2021年將發布的iPhone機型

郭明錤表示：「5G支持將是2020年款iPhone機型最關鍵的技術升級。」他預測，到2020年秋季，所有「iPhone 12」機型都將配備高通X55數據機，並支持5G，具體支持頻段取決於不同國家。

他預計Sub-6G頻段+毫米波iPhone將在包括美國、加拿大、日本、韓國和英國在內的五個市場上市，並在2020年下半年佔新iPhone總出貨量的20%。

除了5G支持之外，2020年版iPhone後置攝像頭預計增加3D深度感測技術。郭明錤還稱，蘋果公司可能通過利用聲學指紋技術將整個屏幕變成指紋識別器，重新採用Touch ID，這已經在該公司的幾項專利申請中得到了體現。

郭明錤進一步重申了之前的預測，即「iPhone12」將有四種型號，包括配有後置雙攝像頭的5.4英寸OLED屏幕iPhone和6.1英寸OLED屏幕iPhone、配有後置三攝像頭加飛行時間感測器的6.1英寸OLED屏幕iPhone和6.7英寸OLED屏幕iPhone。他預計，蘋果將在2020年早些時候發布4.7英寸LCD屏幕「iPhone SE 2」，外形尺寸與iPhone 8相似。


圖2：新款iPhone機型的規格和部件供應商

不過，郭明錤首次預計，蘋果將於2021年上半年推出「iPhone SE 2 Plus」，尺寸可能為5.5英寸或6.1英寸，採用全屏設計。這款手機將有很小的劉海，因為它不支持Face ID，並且在側面的電源按鈕中集成了Touch ID。

備受爭議的是，郭明錤預計2021年秋季發布的「iPhone 13」將不會有Lightning埠，進而提供一種「完全無線體驗」。

郭明錤預計，iPhone出貨量將同比增長約6%和8%，到2020年和2021年分別達到2.1億和2.25億部左右。這特別要歸功於5G支持、新的外形設計和更好的產品組合。

最新銷量預測增幅高於郭明錤此前的預計，他認為2020年出貨量可達2億部。新的預測更符合2015年iPhone銷量全盛時期的趨勢，當年蘋果賣出了2.312億部iPhone。該公司在2016、2017和2018財年分別售出2.118億部、2.168億部和2.177億部iPhone。（小小）
