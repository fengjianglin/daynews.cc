《慶余年》全集遭泄露，版權監測中心：侵權鏈接近4萬條，與平台加播收費無必然關聯

近日，網路上開始出現《慶余年》全集盜版資源。隨後，包括騰訊視頻等平台均已發表聲明稱已交由有關部門處理。對此，中國版權協會版權監測中心主任吳冠勇稱，目前《慶余年》侵權鏈接已近4萬條，這與片方、發行方、平台方把控不嚴有關。

新一批國產遊戲版號下發 騰訊網易在列

原國家新聞出版廣電總局官方信息顯示，新一批國產遊戲版號名單更新，共53款遊戲獲批，審批時間為12月20日。其中騰訊的《彩虹墜入》、網易的《風雲島行動》、三七互娛《笑傲江湖頁游》、多益網路的《神武4》、電魂網路的《我的俠客》、盛趣遊戲的《創意甜品屋》等遊戲獲得版號。

滴滴回應擬明年在日本推出送餐服務：無確定計劃

日前據外媒報道，滴滴出行將於2020年在日本推出網路送餐服務。對此，滴滴回應稱，滴滴目前在對若干市場外賣服務進行調研，還沒有確定計劃。

自2018年9月在大阪上線計程車服務以來，滴滴在日本業務進展良好，目前我們和超過400家本地優秀計程車公司合作，在21個地區提供計程車叫車服務，服務覆蓋超過50%日本人口。

前員工發文揭Keep困境，高管發聲：職場背叛是一生污點

日前一篇《Keep的困頓與終局》的文章讓Keep的發展引發關注。該文章指出，Keep遭遇困境源於兩個核心問題：業務增長遭遇瓶頸和變現渠道模糊不清，並貼出了眾多內部數據。後來該文章的作者被發現是Keep前員工。Keep官方對此表態稱，文章眾多數據不實，將保留法律手段追責。Keep政府事務總經理在社交平台稱，一旦有了職場背叛記錄，污點將伴隨一生。

雷軍手機屏保曝光：「能不管的事情一定不管」

由湖北省政府主辦的第四屆楚商大會於2019年12月20日在武漢召開，小米科技有限責任公司董事長兼CEO雷軍出席並演講。有人在楚商大會歡迎晚宴現場拍到了雷軍手機屏保，上面寫有幾個大字：能不管的事情一定不管。

任正非：華為十月增長幅度下降至17%，預計2020年在10%左右

任正非近日在《洛杉磯時報》中表示，預計華為業績在2020年還會增長，但增長幅度不會太高。華為今年十月增長幅度已經下降到17%，預計2020年會在10%左右。

任正非指出，他認為華為2021年可能會開始規模性增長，但高層團隊認為恢復規模化增長在2022年，這點他個人與高層團隊是有分歧的。他認為高層團隊考慮得更加穩妥一些，所以他簽發的文件是預測2022年開始規模化增長。
