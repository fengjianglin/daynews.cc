中國聯通用戶數兩連跌：上次還是4年前

2019-12-21 735條評論 1,793次閱讀 332人點贊

近日，三大運營商陸續公布了2019年11月份的運營數據，其中中國聯通情況不妙，總用戶數連續兩個月出現下滑。

截止2019年11月底，聯通的移動出賬總用戶數為3.21億，當月凈減85.6萬，而此前10月份已經凈減261.0萬人——自從2015年12月以來，這是聯通用戶數第一次兩連跌，事實上也是第一次出現下跌。

相比之下，移動的總用戶數已達9.47億，上月凈增295.5萬；電信的總用戶數則是3.34億，上月凈增186萬。

今年以來，移動、聯通、電信用戶分別累計凈增2147.6萬、622.7萬、313.9萬。

很顯然，攜號轉網給聯通帶來了空前的壓力，很多聯通用戶都轉投了移動和電信。

4G用戶方面，移動、電信、聯通目前分別為7.53億、2.80億、2.54億，聯通仍是最少的。

固網寬頻業務方面，聯通當前用戶只剩8445.3萬，上月凈減10萬，是去年12月以後一年來首次下滑，今年凈增434.1萬，同樣是最差的。

移動當前用戶1.88億，上月凈增69.4萬，今年凈增3096萬。電信為1.54億，上月凈增50萬，今年凈增796萬。
