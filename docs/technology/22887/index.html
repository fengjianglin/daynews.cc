年底很忙 科技巨頭紛紛宣布大事件（附股）

2019-12-20 3,160條評論 4,731次閱讀 2004人點贊

科技巨頭的動向不僅可以引領行業的發展，同樣對A股市場也有著重要影響，快來看看近期科技巨頭們都在忙些什麼？

騰訊控股

12月18日，騰訊遊戲與英偉達簽署戰略合作備忘錄，共同宣布成立聯合創新實驗室，共同探索未來新的產品形態和應用場景，一起為遊戲用戶創造更加卓越的遊戲體驗。近期部分有基本面支撐的遊戲股也在走趨勢，如完美世界、三七互娛、吉比特、游族網路等。

而雲遊戲概念在前不久也被市場炒作，預計雲遊戲技術下，手游市場規模有望從原來的1400億元增長至3500億元左右。目前在國外廠商中，微軟、谷歌、英偉達、索尼四大巨頭具備競爭優勢，國內廠商也在積極布局，後面還會有相關消息出來。

特斯拉

特斯拉股價於美國當地時間周三收盤上漲近4%，創下每股393.15美元的歷史收盤價新高。有媒體報道稱，特斯拉正在考慮將其中國製造的Model 3轎車價格下調20%或更多，但之後特斯拉中國公司的相關負責人表示上述說法「不是官方信息」。

此前產能一直是困擾特斯拉發展的問題，但今年上海工廠的竣工並投產，對於一家歷史上始終未能達到投產預期的公司來說，有望迎來新的發展契機。產量的增加也意味著生產成本的下降，所以降價並不代表利潤率會降。

阿里巴巴

12月19日午間消息，阿里巴巴宣布新一輪面向未來的升級，集中發力推進全球化、內需、大數據和雲計算三大戰略。目前，阿里已實現核心系統100%上雲，同時在人工智慧、大數據、區塊鏈等技術上取得突破性進展。

說到雲計算，幾家巨頭均有布局。12月11日，華為註冊雲計算公司，加大在雲計算相關領域投入力度；12月11日，騰訊於AI大數據新品發布會上全面更新騰訊雲數據智能服務；12月10日，百度智能雲正式上線崑崙雲伺服器，採用基於百度自主研發的AI晶元。綜合來看，近期國內各大雲計算廠商動作頻繁，反映出雲計算產業目前熱度持續攀升，整個行業景氣度處於較高水準。

雲計算產業鏈

華為

12月19日，華為VR眼鏡將正式開售。這款VR眼鏡擁有IMAX大片、熱門電視劇、動畫片等海量優質片源，同時引入包括愛奇藝、VeeR、GVR等多家優秀平台資源，號稱「足不出戶，便可享受私人專屬影院」。有分析稱該款眼鏡有望引爆VR行業。

VR/AR：2019年第一季度，中國 AR/VR頭顯設備出貨量接近27.5萬台，同比增長15.1%。5G商用化帶來運營商渠道對頭顯設備的需求大幅上升。預計2023 年，中國VR頭顯設備出貨量將突破1000萬台，AR設備出貨量將超過800萬台。

華為VR Glass還支持基於5G網路的直播觀看、VR划船、100餘款VR遊戲等功能，有望真正實現手機顯示屏元件的外部化，豐富人機交互的方式。VR產業鏈相關公司有歌爾股份、水晶光電、京東方A、利亞德等。

百度

12月18日，百度在長沙舉辦首屆Apollo生態大會，宣布將自動駕駛平台升級到了Apollo5.5版本，此外還新發布了車路協同、智能車聯兩大平台。同時還發布新一代智能交通解決方案、小度車載2020、鴻鵠晶元等15大新品及解決方案。百度在國內AI領域位居前列，此次大會也可被視為國內無人駕駛領域的生態大會，對業內相關公司的影響甚大。

麥肯錫研究報告顯示，中國未來很可能成為全球最大的自動駕駛市場，至2030年，自動駕駛相關的新車銷售及出行服務將超過5000億美元。在國內無人駕駛高速發展的大背景下，關鍵零部件供應商、汽車電子解決方案供應商、智能駕駛整車龍頭有望受益。

A股多家上市公司也很早就開始布局無人駕駛。如四維圖新的高精度地圖在智能駕駛領域具備一定優勢，不過從估值角度來說，目前還是偏貴。在以後的文章中，我們也會詳細梳理無人駕駛概念。
