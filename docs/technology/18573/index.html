“產能地獄”魔咒，傳統車企巨頭也不能倖免？

2019-12-18 719條評論 2,024次閱讀 347人點贊

「產能地獄」一詞來自於特斯拉，特斯拉的每款電動車（雖然目前量產的只有3款）都經歷過「產能地獄」，無法按時交付用戶。此前，有人一直嘲笑說特斯拉不行，有產能地獄，造不出車。但是現在，就連傳統車企巨頭們也都出現了這個問題。

Model 3產能地獄

在說傳統車企產能地獄之前，我們首先來看看這位電動車領域的先行者的產能爬坡之痛。

自2016年4月Model 3發布，狂砍50萬訂單，直到2017年7月才交付了首批30台，之後開始產能爬坡。

馬斯克曾立下flag，要在2017年年底周產5K的目標，但是因為各種延誤，良品率問題，馬斯克將該目標推遲到2018年第二季度末。

2018年6月底，終於完成立下的周產5K的flag，但為此馬斯克也是出了很多代價，親自赴工廠督戰、睡工廠、精神壓力大到吃藥才能睡著。

2018年年底，Model 3試驗性的實現了周產破7K，而現在，Model 3的周產能已經可以做到破8K…..這也就意味著，特斯拉走出了產能地獄。

根據外媒消息，特斯拉中國工廠的周產能已經突破1K，按照馬斯克此前規劃，年底Model 3要實現周產1萬台，也就是說，上海工廠正在往周產2K這個目標努力。

年底，特斯拉將衝擊周產10000台Model 3的目標。

很多人嘲笑，能陷入產能地獄，還是說明特斯拉這家企業太新了，沒有造車經驗，但是現在來看，傳統車企也陷入了這個地獄之中。

傳統車企也逃不過的產能地獄

捷豹算是傳統車企陣營中比較早宣布要電動化的一位。2018年4月，在首款純電動車I-Pace發布，在交貨的時候捷豹宣布推遲I-Pace的交付，經銷商表示是因為「製造問題」引起的，而捷豹官方給出的解釋是「對訂單進行了優化排序」。

接下來要說的是，德系三巨頭之一的奧迪，首款純電動車e-tron發布之後，在2018年年底奧迪推遲了一次e-tron的交付時間，當時給出的說辭是為了完善用戶體驗，但是在2019年2月，奧迪在此推遲交付，將歐洲交付時間推遲到2019年3月，美國市場推遲到2019年第二季度，理由要麼是LG化學的電池供應出了些問題，或者是軟體出現問題，看來不靠譜的不只是特斯拉，傳統車企同樣如是。

同樣是大眾集團旗下的另一個品牌，保時捷的首款純電動車型Taycan也面臨了這個很尷尬的問題。今年11月，在給預訂車主的郵件中寫道，因為生產的複雜性，Taycan交付時間推遲到明年3月末或4月初，而此前說定的時間是2020年1月。

無獨有偶，就在前幾天，梅賽德斯-賓士推遲純電動車EQC在美國的交付，時間延遲到2021年。對此，賓士給出的回應是：「這是一項戰略決策，首先要支持歐洲客戶對EQC的需求」。要說明的是，這並不是賓士第一次推遲EQC的交付，早在今年2月，賓士就將交付時間表推遲到11月。

與此同時，起亞推遲了在美國上市的SoulEV，理由是電池供應出現了問題……

小結

到這裡，我想很多讀者已經看清楚了，即便是這個相比於傳統燃油車結構已經變得很簡單的電動車，同樣有它的准入門檻。

先入場的特斯拉已經佔了起手優勢，超過這些傳統巨頭們很多了，要想追平甚至超越特斯拉，這些巨頭們要付出不小的心力啊。
