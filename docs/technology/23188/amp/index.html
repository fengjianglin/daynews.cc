找到"救命錢"！WeWork獲高盛等17.5億美元貸款

天天要聞
2019-12-18

作者|張超 編輯|安心

IPO夭折後的WeWork正在進行大規模自救，如今終於等來了「救命貸款」。

12月18日，媒體報道稱，在高盛集團牽頭的一輪融資交易，WeWork獲得了17.5億美元的新融資，為這家苦苦掙扎的辦公室共享公司帶來了急需的大量現金。這一輪融資將以信貸額度的方式提供給WeWork。


「我們很高興WeWork、軟銀集團公司與高盛簽署了承諾書，」WeWork發言人艾琳·克拉克(Erin Clark)在一份電子郵件聲明中表示：「根據新貸款協議，WeWork不會被要求進行任何現金抵押。「WeWork公司和軟銀集團分別是高級擔保和無擔保貸款的共同債務人。」

兩名知情人士還透露，高盛集團提供的貸款將會釋放WeWork公司約8億美元現金，這是WeWork公司為滿足之前信用額度的約定而預留的。

據悉，該筆貸款也是軟銀集團承諾為WeWork籌集50億美元債務融資計劃的第一個障礙，如今障礙已經被克服。

今年10月，WeWork和軟銀共同宣布達成一項協議，軟銀承諾向WeWork提供一筆數額巨大的融資，包括50億美元的新融資以及向現有股東發出最多30億美元的收購要約。此外，軟銀還表示會加快現有的一筆15億美元融資承諾。

截至目前，已經實施的債務融資救助計劃包括：11億美元的有限擔保債券，22億美元的無擔保債券以及17.5億美元的新融資。這也在很大程度上提升了WeWork的資金周轉能力。

不過，有說法認為，軟銀對WeWork的持續援助是一個不明智的舉動，它是把現金投入到一個失敗的商業模式和公司中。

公開資料顯示，截至2019年9月，WeWork在過去一年中投入了數十億美元用於擴張，估值也從年初的470億美元縮水到80億美元。WeWork的過度擴張和不佳表現也拖累了軟銀的業績，今年三季度，軟銀營業損失7044億日元（約合65億美元），凈虧損7000億日元（64億美元），創下成立以來最差季度表現。

但在軟銀CEO孫正義看來，長期看WeWork項目將帶來盈利，軟銀有足夠資金解決WeWork面臨的問題。
