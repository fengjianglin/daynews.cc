全球首創V型雙旋翼無人機發布：造型科幻 續航50分鐘

天天要聞
2019-12-19

12月19日消息，零零科技在杭州發布了全球首創V型雙旋翼無人機V-Coptr系列，第一款產品名為Falcon（「獵鷹」）。在消費級無人機普遍採用傳統四旋翼機構的背景下，V-Coptr Falcon的出現實現了無人機行業的一次革命性突破。

續航問題始終是無人機產品的一大痛點。在大多數消費級無人機只有30分鐘極限飛行航時的情況下，V-Coptr Falcon將其續航能力提升至50分鐘。

這得益於V-Coptr精簡的V型雙旋翼結構設計（多項零零自有專利），結合其自主研發的舵面傾轉技術和非線性動力模型設計及控制演算法，讓V-Coptr的動力系統在相同負載、同樣電池動力的情況下，較同等尺寸的傳統四旋翼飛行器，力效大幅度提升。

V型雙旋翼構型使得Falcon以更高力效提供升力的同時，旋翼葉尖轉速也更慢，進而大幅降低飛行器噪音，獲得更安靜的飛行體驗。不僅如此，零零在槳葉設計上擁有多項創新專利，得到更優的降噪和力效提升的組合, 由此進一步提升了槳葉力效，達到雙重降噪效果。

Falcon的外形設計也是一大亮點，V型雙旋翼去除冗餘，精緻潔簡。減小的機頭受風面積有助於更加穩定高效地懸停及高速飛行，可摺疊式設計更是大大減少了收納所需的空間。


當機臂展開，電機座通過機臂內嵌的高效能舵機自動旋轉至待飛模式位置，整體造型充滿了未來科技感。機臂展開狀態下的Falcon外形酷似張開羽翼待飛的大鳥，高速飛行狀態下的Falcon更如其名「獵鷹」一般，展翅高飛。

為了保證更穩定的航拍效果，Falcon配備了三軸機械增穩雲台，Yaw方向轉動可達到±80度。在此基礎上配合支持4K超高清視頻和1200萬像素照片的攝像頭，拍攝穩定、清晰的畫面不在話下。

成像系統方面，Falcon採用了SONY的1/2.3英寸感光器件和QUALCOMM（高通）的影像編解碼晶元（ISP），在影像畫質方面表現出色，達到了一流航拍器的水準。

V-Coptr Falcon還配備了先進的VIO（視覺慣性里程計）技術和視覺避障系統，讓飛行器在飛行過程中能夠主動感知周圍環境，實時檢測障礙物，實現穩定懸停和飛行，避免發生撞擊。


全新升級的視覺跟蹤演算法讓V-Coptr Falcon 可以精準穩定地跟拍畫面中的人物或其他移動物體，軟體中預設的經典拍攝軌跡，可一鍵拍出大片級的運鏡效果。

目前，V-Coptr Falcon公布的海外市場零售價為999美元，並在北美市場啟動預售，2月中旬發貨。相信不久，這款產品也會在國內市場上與大家相見。

本次發布會上零零科技還展示了「One More Thing」—— Project D。其造型同樣基於V型雙旋翼動力系統，只不過體積更小，相比Falcon的730g，Project D總重量只有249g，同樣實現了50分鐘的超長續航。
