比特幣暴富的90後！幣圈孫宇晨

2019-12-06 1,747條評論 4,623次閱讀 949人點贊

孫宇晨是幣圈一個頗有爭議的人物，他的頭銜很多，比如「北大高材生」、「幣圈賈躍亭」、「90後創業領袖」、「馬雲門徒」。他眼中的世界，除了成功，就是失敗，沒有中間地帶。這個90年出生的偏執狂曾放豪言：「我衡量人的標準是看你賺多少錢。」

叛逆但有才

孫宇晨從小是一個極具批判意識的人，上學期間他通過實際行動嚮應試教育挑釁：語文考試只寫作文；英語考試用中文答題；歷史考試填空時，反面人物一律填上班主任的姓名，英雄的人物則填上自己的名字。

進入高中後特別喜歡看小說，整個高中生涯，孫宇晨能逃的課都逃了，大部分時間都泡在了圖書館裡看小說。因此成績也一落千丈，成績排在了全班墊底的位置。到2007年的秋天，為了逃離「城鄉結合部」般的惠州老家，小城少年孫宇晨，憑藉新概念作文一等獎20分加分，以及高三一年的瘋狂惡補，實現了「從三本到北大」的跨越，被北大中文系錄取。

2010年底，孫宇晨申請提前一年從北大畢業，並於2011年秋入讀賓夕法尼亞大學的東亞研究專業。到達美國後，他自稱在美國讀到女作家安·蘭德的著作，經受了「一場價值觀的洗禮」，認為企業家才是推動社會進步的動力，遂嘗試各種門徑邁進商界。為了進入商圈，孫宇晨開始選修沃頓商學院的課程，並加入賓大投資協會。

ALL in 比特幣

在2013年4月11號這天，孫宇晨第一次從《紐約時報》的一篇文章里知道比特幣。這篇文章里的溫克萊沃 斯兄弟，在美國是有名的「上訪專業戶」，因為他們纏著扎克伯格不放，扎克伯格被迫賠了他們一大筆錢。他們拿著這筆錢做的第一件事，就是買了很多比特幣。

當時比特幣價格經歷了一輪漲幅，雖然和現在相比微不足道，還是引起很多人重視。全世界除了比特幣社區中的一些極客在探討比特幣的學術問題，絕大多數人還是認為這是一個 專門圈錢的東西，就像龐氏騙局或者鬱金香泡沫。所以孫禹城雖然參與到美國比特幣的社區中，但在周圍的人看來，還是在做一件非常孤獨、非常不靠譜的事情。因為你告訴他你是搞比特幣的，他就會有「先生您聽過安利嗎」這種感覺。

當時他就面臨著兩種選擇：選擇一：押注比特幣，做一些與比特幣社區有關的事情；選擇二：在我的母校賓夕法尼亞大學繼續我的求學之路。面臨這兩個選擇孫宇晨很糾結，因為老被人當成是安利推銷員實在不是很好的體驗。但最後還是選擇了比特幣。

孫宇晨表示，比特幣是第一次實現了在金融清算中不用通過中心節點。 這可能非常抽象，但他當時接觸到這個概念時感到非常震撼。因為一直以來，人類在貨幣清結算領域，都沒有辦法擺脫中心化的節點。我們日常生活中互相之間轉 帳，都有一個中心節點為我們記賬：人與人之間發送貨幣是商業銀行為我們記賬，商業銀行與商業銀行之間有人民銀行的清算中心為它們記帳。我們長期的習慣思維 是：任何貨幣之所以有價值，是因為有主權國家在後面為它背書;任何貨幣之所以可以被清算，是因為有可信的機構在那裡對它進行清算。

孫宇晨剛接觸比特幣 時，它大概五六十美元，有時漲到六七十就已經把他高興壞了，比特幣整體市值在5億至10億美元之間徘徊。當時他很簡單算了一筆賬，全世界黑暗互聯網每年的 估值大概2萬億美元左右，比特幣的匿名性很好地為它解決了金融清算問題，他就想比特幣的市值完全可能因此隨便就能增長100倍。於是，我就把上賓夕法尼亞的學費全部都用來買比特幣了。之後沒多久，比特幣瘋狂增長，這是他當初沒想到的，比特幣讓他擁有了第一個一千萬。

獲千萬融資，馬雲得意門徒

2013年底，加入位於矽谷的互聯網金融公司RiPPleLabs。他反覆向媒體描述，Ripple 在他眼中比比特幣還要神奇，「這是一套由價值網路支持的去中心化的支付體系，可以讓不同貨幣自由、免費、零延時地匯兌」。一個多月後，孫宇晨以RippleLabs大中華區首席代表的身份回國，拿到RippleLabs股東之一IDG的千萬投資，成立銳波科技，擔任創始人兼CEO。

在向一家創投基金尋求投資受挫後，他迅速找到著名的 IDG 資本，成立銳波科技。他認為之所以能搞定 IDG，一是 IDG 是 Ripple Labs 的股東之一，投他順理成章；二是 IDG 當時在投資圈裡率先打出了「90後創業者」的概念。拿到投資後，他被拉入 IDG 的90後創業者微信群，起初只有三四個人，後來很快增加到三四十人。這讓他意識到，一場「90後創業熱」即將颳起，自己應當站在浪潮最前沿。為此，他聘請一家著名財經媒體的高管擔任市場副總裁，專門為他打理公關事務，主打「90後創業領袖」牌。拜會各大商業媒體，尋找論壇演講機會，成為他當時的頭等大事。不久之後，IDG 宣布設立「IDG 90後基金」，規模1億美元。面對媒體「這是意在炒作」的質疑，IDG 資本創始合伙人熊曉鴿稱，90後創業者的時代已經到來，投資和支持他們是搶佔行業先機和制高點。在 IDG 力推下，「90後創業」的概念迅速攻佔各大報刊，孫宇晨是最耀眼的明星之一。「宇晨的 PR（公關）能力簡直強爆了。」現在已是他助理的林坤說。他在場合高調宣講：「90後是移動互聯網的原住民」、「90後必將顛覆世界」、「我們不跟對手在同一維度競爭，戰勝你，但與你無關。」一位90後創業者將孫宇晨的自我推廣形容為「滾雪球」。「IDG 的投資，媒體的不停報道，其實都起到為他背書的作用。這種背書越多，人們越願意關注他、相信他。他就能獲得越來越多的資源。」

他最自豪的是入選馬雲創辦的創業者培訓營「湖畔大學」，是其中唯一的90後。剛一入選，「馬雲最年輕的門徒」就出現在他的百度百科詞條里。在一檔視頻訪談中，他稱自己與馬雲「相見恨晚」。「我跟馬雲一聊，就感覺很多共同話題，哎呀，大家一下感覺就很鐵。」

幣圈賈躍亭

2017年下半年，孫宇晨創建了第二個項目「波場TRON」，號稱利用區塊鏈技術，構建全球去中心化的自由內容娛樂體系，商業模式就是發幣，俗稱1CO。知名投資人李豐和薛蠻子是這個項目的投資人，孫宇晨還另外請了一堆知名人士站台，為他背書。這種背書帶來的價值是，人們能夠相信他。截止發文在超過2000多種數字貨幣中，市值95億RMB排第13名，年初市值最高時達到900億RMB。

2017年8月22日12：00，全球最大的數字貨幣交易平台幣安進行5億枚波場幣的搶購活動，53秒內所有份額隨即被搶購一空。為了實現這一目標，波場幣1CO前宣傳中多次提及，波場TRON由前Ripple大中華區首席代表、馬雲湖畔大學一期學員、「馬雲門徒」孫宇晨所創立，其投資人包括比特大陸CEO吳忌寒，信中利資本董事長汪潮湧，OFO小黃車創始人戴威，RippleCoinbase投資人，峰瑞資本合伙人李豐，FBG資本合伙人周碩基，量子鏈創始人帥初，天使投資人薛蠻子等。

隨著1CO亂象的不斷曝光，監管層面不斷施壓，原本定於9月9日進行「波場幣」ICO，緊急宣布提前一周至9月2日進行。9月3日，孫宇晨於個人微博宣布，「波場幣」正式完成ICO。果不其然，就在「波場幣」完成ICO的第二天，監管就下發《關於防範代幣發行融資風險的公告》，叫停各類1CO活動，並要求退幣。當此時的孫宇晨已經靠1CO募集了大約4個多億的資金，為了安全考慮孫宇晨還是將1CO募集到的這些幣都退還了。

俗話說，「上有政策，下有政策」。剛在國內退完幣的孫宇晨馬不停蹄的趕往了美國，在美國繼續開展他波場幣的宣傳活動。同年十月份，波場在海外登陸了交易所開始公開交易。剛開始的時候一枚波場幣的市價僅為1分。但從11月底開始隨著比特幣價格連續走高，市場開始逐漸回暖，到了12月份，波場幣的價格被拉升到2塊錢。隨著波場幣的持續走高，相關人士發現至少有超過50%的「波場幣」存儲在同一個錢包中。市面流通的「散戶韭菜」的比例僅為9%，流通市值不足4500萬元，完全與波場白皮書分配計劃對應不上。也就是說，「波場幣」被某個或某幾個莊家高度控盤。孫宇晨割散戶的架勢早早就擺好了。並且孫宇晨的錢包交易記錄顯示，其錢包每天發送2億個波場幣至幣安等交易平台換成以太坊，這樣的行為持續了19天，也就是說，孫宇晨換掉了60億的波場幣，按照當時的幣價來算，孫宇晨套現了120億。

曾有一位投資機構人士將他形容為「一個成功的創業演員」。「比方說他本來是100分，精心包裝成1000分的樣子，只要這個1000分的泡沫不戳破，他就可以在市場上找來1000分對應的資本和行業地位。一直這樣玩兒下去，等泡沫吹得足夠大，圈到足夠多的錢，再去市場上收購一個真正靠譜的公司，這個資本遊戲就算玩兒成了。」

「這位北大90後精英做事的標準，與賈躍亭造車的路子如出一轍。「高調從事公關，公司經營卻缺乏進展。」一些互聯網金融人士表示。

孫宇晨從不掩飾對名利的渴求，頻繁接受媒體採訪。可以說，那時他的名氣遠勝過他公司的名氣。

不可否認，90後孫宇晨資本玩轉的套路水平不亞於頂級投資者。從差等生到北大天之驕子；果斷休學押注比特幣；以Ripple Labs大中華區首席代表身份（當時是光桿司令）斬獲IDG上千萬融資；用波場區塊鏈項目巧取豪奪上千億。孫宇晨通往財富自由之路，似乎和大多數富豪的成功學截然不同。另闢蹊徑，飽受質疑，但和大多數人想法是一樣的，”錢或許是唯一能帶來安全感的東西”
