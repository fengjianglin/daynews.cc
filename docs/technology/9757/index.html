小米集團顏克勝：互聯網賦能質量管理

2019-12-10 1,236條評論 4,484次閱讀 1063人點贊

12月5日-6日，中國國家市場監督管理總局與深圳市人民政府共同舉辦以「質量 變革 共享」為主題的第三屆中國質量（深圳）大會。

據了解，本次大會圍繞大灣區與區域高質量發展、智能製造質量管理與創新等議題進行交流研討，搭建世界更好了解中國質量發展的平台，提供中國學習借鑒世界經驗的機會，推動經濟社會實現高質量發展。宜家、戴勒姆、中國平安保險、雀巢、施耐德、西門子等企業參與本次會議。此外，包括騰訊、阿里、小米等在內的互聯網公司也受邀參加。質量領域新一級「ATM」引領該行業品質持續升級。

在會議上發表講話的小米集團副總裁、集團質量委員會負責人顏克勝說道，「小米的初心就是以互聯網方式推動中國製造業轉型升級，用質量和創新，推動時代變革。」

目前，產品質高價優已成為新一代消費群體的最新消費趨勢。顏克勝表示，企業需要從用戶角度出發，用互聯網思維賦能品質，藉助互聯網工具做到「從群眾中來，到群眾中去」，讓用戶全程深度參與產品的研發過程，實時收集他們的反饋，並形成良性互動。小米的品質工作就是要百分百把用戶當朋友，把用戶滿意度放在最重要的位置。

「質量工作沒有捷徑可走，是一場硬仗，也是一場持久戰，需要有死磕到底的決心」，顏克勝說，「經過過去幾年所做的持續改善工作，小米的產品質量取得了長足進步。有了質量的保證，小米業務也得到了蓬勃發展。」

小米在質量上的努力與投入贏得了消費者的讚許的同時，也榮獲了業界廣泛認可。近兩年小米榮獲了中國質量技術獎一等獎、全國質量標杆獎、亞洲服務獎等獎項。今年年初推出的Redmi Note 7小金剛系列打破了行業慣例，成為國內首款18個月超長質保的智能手機，開售僅7個月，Redmi Note7系列全球銷量突破2000萬台。

顏克勝指出，小米最核心的理念是將質量與創新並舉。放眼未來，5G時代的到來，不僅帶動智能終端進入升級發展的新周期，更開啟了人工智慧與物聯網有機結合的新篇章，成就下一代由5G+AIoT組成的超級互聯網。AIoT在5G的加持下，網路空間與設備的連接將更為緊密，需要解決好單點設備的品質保障、設備間互通互聯的穩定性、以及整個超級互聯網的體驗，才能形成未來5G生活的有力保障，這也是超級互聯網在未來要面臨的質量挑戰，需要大家一同推動建設。

上游新聞·重慶晚報 記者 顧立 實習生 許效溪
