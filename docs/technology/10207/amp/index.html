火星UP主招募計劃正式啟動，10大權益賦能短視頻創作者

天天要聞
2019-12-06

免責聲明：本文旨在傳遞更多市場信息，不構成任何投資建議。文章僅代表作者觀點，不代表火星財經官方立場。

小編：記得關注哦

來源：火星一線

12月6日14時，火星UP主招募計劃線上發布會在火星社群展開。火星財經總編輯猛小蛇擔任本次AMA主持人，對話火星財經流量運營負責人沈溪，全面講解「火星UP主招募計劃」10大權益。

「火星UP主招募計劃」由火星財經為區塊鏈、5G、人工智慧等科技領域以及財經領域的視頻創作者推出扶持計劃，旨在通過億級品牌推廣展現以及生態級全領域流量支持，幫助創作者打造自己的視頻節目，推進創作者個人IP的打造與升級。

對話中，沈溪表示，「從今天起，火星財經團隊將通過在品牌、技術、運營、產品、推廣等方面的沉澱，助力每位視頻達人成為區塊鏈領域的流量紅人。」

他同時透露，火星財經團隊將為UP主在渠道宣傳、原創展示、運營指導方面提供10大權益。

權益一：覆蓋全球4000+核心互聯網傳播渠道

對於UP主在平台發布的原創視頻，火星財經團隊將第一時間push到各大流量與生態節點，不同於傳統的視頻媒體，火星財經的傳播渠道更加聚焦於區塊鏈行業，將為您的視頻提供免費及全面的行業用戶覆蓋。

權益二：億級品牌推廣展現

依託自身現有的生態流量資源和品牌流量，火星財經團隊將為每一位視頻UP主提供符合區塊鏈行業特徵的一站式的視頻推廣服務，針對傳播性較強的優質原創視頻，為UP主免費提供全網營銷推廣的視頻展現支持。

權益三：生態級全領域流量支持

以核心平台流量為支撐，以4000+推廣渠道流量為依託，以300萬+社群、平台用戶為落地的生態級全領域流量，覆蓋用戶從感興趣，點擊，觀看，成為粉絲，最終轉化的全流程。

權益四：覆蓋行業生態社群10000+

火星財經團隊將通過旗下10000+個微信社群、海外電報社群、6500+自營社群，以及3500+外部合作社群，直接覆蓋1500000+行業用戶、從業者，實時傳播優質的視頻內容，為優質up主建立專屬的粉絲社群，提供全維度支持。

權益五：直接覆蓋區塊鏈愛好者3000000+

目前，火星財經在區塊鏈領域的全平台用戶覆蓋用戶量已突破300萬，團隊可將UP主視頻內容快速傳播，可助力視頻內容全面孵化和推廣。

權益六：1V1運營指導等多方面支持

火星財經團隊將組建一支內容推廣與創新運營團隊為來為up主提供1V1的服務，在保證專業性的同時，也將以求真務實的態度與持續創新的精神，為UP主的內容創作與推廣保駕護航。

權益七：參與「火星UP主招募計劃」每月單獨發布榜單

火星UP主的招募計劃從開放之日起，將每月推出「火星UP主招募計劃」排行榜單，該排行榜由視頻質量，社群傳播數，觀看擊量，觀看用戶數，觀看時長等多維指標計算得出，可反映上榜的在品牌知名度、內容質量、傳播水準、社群能力等方面的整體實力。

權益八：優質視頻內容均有機會獲得以下推薦資源，觸達數百萬粉絲

針對優質UP主視頻，火星財經團隊將會在將會在平台、APP的核心位置，為夥伴提供免費的推薦資源，包括但不限於：

1）火星財經網站首頁置頂推薦；

2）火星財經快訊推薦；

3）火星財經APP彈窗推送；

4）火星財經APP火星號頁面推薦；

5）火星財經公眾號、頭條號編輯推薦；

6）合作視頻網站首頁推薦位等。

權益九：原創視頻欄目logo展示機會

火星財經團會篩選優質原創視頻，第一時間將視頻展示在欄目頂部。

權益十：有機會參與火星財經「POW’ER系列峰會」
