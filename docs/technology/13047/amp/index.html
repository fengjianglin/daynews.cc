數百萬美元投物流人工智慧，為何又是“懂調度”？

天天要聞
2019-12-11

物流行業的整體融資趨勢變了，資本方越來越重視能真正為物流行業提升效率和生產力、解決痛點的物流科技，關注已經落地應用的智能化帶來的降本增效。

12月11日，上海訊輕信息科技有限公司（以下簡稱「訊輕」）宣布獲得了藍湖資本、光速中國的數百萬美元A輪投資，毅仁資本擔任本輪融資獨家財務顧問。此前，訊輕於在2018年11月獲得了梅花創投和銘笙資本的天使輪投資；2018年12月獲得了銀河系創投的Pre-A輪投資。訊輕成立於2017年，是人工智慧驅動的物流科技服務商，以核心動態蜂窩演算法+凹凸演算法+智能優化引擎+人工智慧，提供智能調度和路徑優化的配送優化整體方案。

如今全球經濟下行壓力加大，物流賽道呈降溫態勢，為何年輕的訊輕能用一年時間逆勢完成3輪融資？經過分析，羅戈網.物流沙龍認為有幾個關鍵因素：

  • 物流行業的整體融資趨勢變了，曾備受青睞的各種創新商業模式開始褪去光環，資本方越來越重視能真正為物流行業提升效率和生產力、解決痛點的物流科技，關注已經落地應用的智能化帶來的降本增效，智能化已經成為物流企業的「硬核能力」；
  • 配送行業是非常巨大且發展快速的傳統行業，技術有很大的滲透空間，智能調度是其運轉效率的核心環節，行業急需真正具備落地應用能力的企業出現；
  • 最為重要的是，訊輕擁有過硬的人工智慧實力、演算法以及智能調度SaaS「懂調度」這樣的「硬核能力」，「懂調度」可以優化成本、效率和管理，平均降低物流成本10%～15%；團隊對配送調度的業務場景有很深的理解，已經積累了眾多成功落地應用案例。

訊輕的獨特優勢

配送的複雜程度遠高於運輸管理和倉儲管理等環節，而且終端客戶的要求越來越高，需求個性化、定製化、碎片化、高頻化趨勢明顯，交貨周期要求更短，而且配送在不同行業應用場景中的服務需求各有差異。面對現狀，人工調度根本無法滿足行業需求，以「AI+演算法+大數據」的智能系統代替傳統的人工排線和調度是大勢所趨。

針對行業的迫切需求，訊輕自主研發了國內首款智能調度SaaS——「懂調度」系統，以核心動態蜂窩演算法+凹凸演算法+智能優化引擎+人工智慧為技術核心，可代替傳統的人工排線和調度，平均降低物流成本10%～15%。「懂調度」的定位是路徑優化和智能調度SaaS，讓流通更輕鬆，一般人工調度排線需耗時2～3小時，「懂調度」幾秒鐘即可完成複雜的計算，幫助客戶實現從商流到物流的全鏈條優化。「懂調度」還能進一步提升客戶的信息化及數字化程度。

「懂調度」的功能包括：訂單合併、地址清洗、路徑優化、運單地圖展示和規劃、調度管理、司機與配送員管理、計費模塊、限行模塊、站點模塊、用戶模塊、智能調度控制塔。

據訊輕介紹，其中有幾個突出的亮點：

  • 「懂調度」實現了地址清洗，一方面自建了九級地址庫，構建了全國最全的貨車限行限高數據；另一方面，實現了地址標準化，避免因地址不準確影響銷售效率，解決了行業一大痛點。
  • 「懂調度」的智能調度控制塔可以洞悉和留存各區域或業務單元配送業務運營數據，以及其相關需要的承運商、車輛資源，提升全局數據可視化水平；可以實現中央調度，加速調度人員經驗線上化及新區域新調度培訓；排線計劃可以作為執行數據反饋的績效參照，並支持基於項目大數據的管理決策。
  • 「懂調度」首創了「凹凸演算法」，在保證原有配送區域的情況下，動態計算最優配載率，訂單密度低的向外「凸」一點，其核心是：既幫助了原來運營的「區域化」，在保證司機交付和終端配送服務的同時，又能提升配載率，從而從整體上優化運營。
  • 「懂調度」的智能優化引擎支持40+個計算維度，比如訂單信息、區域劃分、車型條件、車輛裝載率、交付時間要求、道路種類、禁行限制、司機工作時長、實時訂單、調線機器學習等。

業內同樣做城配物流SaaS服務的供應商有不少，國外知名品牌也在中國開拓市場，訊輕脫穎而出靠的是以人工智慧演算法推動企業精細化運作的能力、更專業和務實的團隊，「接地氣、更實用、更智能、更精準」的解決方案，以及深入客戶第一線，了解客戶的運營場景和痛點。訊輕是目前國內市場上唯一可以實現同城配送領域全智能調度併產品化大規模交付的公司，已服務多行業標杆企業，進入垂直複製和快速拓展階段。要知道，城配在不同行業應用場景中的服務需求各有差異，能做到這一點很不容易。

如今，訊輕產品的落地性已不斷被證實，在醫藥物流、快消、生鮮等行業擁有眾多客戶，如國藥物流、易果生鮮、正廣通等行業標杆企業。

資本青睞：數百萬美元A輪投資

藍湖資本和光速中國是一線美元基金，在企業服務和供應鏈升級等多個領域有非常多的成功案例，兩家投資方均對訊輕充滿信心並有著很高的期望。

藍湖資本主要投資以科技創新為導向的新興行業和傳統行業的轉型升級，重點關注包括企業軟體及服務、消費互聯網、智能製造等領域在內的早期成長型企業。SaaS是藍湖資本重點布局的賽道，此前已經布局了聚水潭（電商ERP）、匯聯易（報銷費控SaaS）等優秀的SaaS公司。「目前城配行業迫切需要一種適應城配場景的輕量級軟體產品，用合適的價格切實解決傳統調度的痛點。」藍湖資本合伙人魏海濤表示，「訊輕在一個巨大的存量市場裡面，用技術和SaaS的方式解決垂直行業的痛點，降本增效的價值點非常清晰，我們看好訊輕以一種相對標準的產品和服務收穫成功。」

對早期投資而言，大賽道和對行業具備洞察力的CEO是光速中國判斷項目時非常看重的兩點。「城配領域一直以來都是個大市場，並隨著終端消費形態的變化在不斷的演進並變得日益複雜，智能調度是城配運轉效率的核心環節，也是數據智能化應用的最佳場景，行業急需真正具備落地應用能力的企業出現。」光速中國助理合伙人王國棟表示，「智能調度是一個極具門檻和延展性的切入點，看好訊輕產品落地應用的前景，以及團隊對行業的洞察力和務實的作風，訊輕已經積累的眾多落地應用案例會有助於公司進一步打開更大的市場。」

2019年12月，訊輕通過國家高新技術企業認定。目前訊輕擁有員工70餘人，80%為研發人才，核心團隊幾乎全部深耕在客戶的一線現場，具備紮實且豐富的物聯網、互聯網、智能演算法和大數據的行業經驗。訊輕創始人&CEO董凱是演算法科學家，擁有中微半導體8年軟硬一體化經驗，2016年任EGoBus共享巴士CEO，主導全動態運營。董凱表示，此次融資資金將主要用於團隊和產品：吸引更多有共同願景的人才，從多維度將團隊的廣度和深度增強，致力於打造最好的產品和最好的服務。

結語

如果說傳統物流時代運營是核心，互聯網物流時代重點在於網路效應，智慧物流時代技術就是最大的壁壘。物流企業越來越重視數字化改造，運營方式也將從傳統的人工決策人工運營，轉變成數據渠道數據決策的智能化運營模式，調度和路徑優化是企業物流提升的最佳切入口。雖然2019年投融資熱度降至近年來最低點，不過業內人也表示，「資本寒冬來襲能讓隨波追流者退出賽道，讓身強體壯者茁壯成長。」像訊輕這種擁有過硬的人工智慧實力、演算法以及智能調度SaaS「懂調度」的硬實力，又具備出眾核心團隊、可持續發展的業務模式的軟實力的公司，還是會受到資本的青睞。我們也期待行業內出現更多像訊輕這樣的智能化企業，通過AI+演算法+大數據的智能系統，幫助城配行業升級，進而改變整個行業的競爭格局。
