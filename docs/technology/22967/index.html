劉強東千杯不醉！馬雲半斤就倒，中國頂級富豪們的酒量你知道嗎？

2019-12-19 452條評論 2,278次閱讀 1191人點贊

咱們國家的應酬，一直跟酒離不開關係。

酒場如戰場，多少大生意都是在酒桌上談成的。

那麼作為商業大佬，肯定也逃不開「酒局」，

今天，我們就來盤點一下站在中國財富頂端的那些人，他們的酒量和酒品如何呢？

是不是和他們的賺錢能力一樣強？

馬雲白酒只喝茅台，酒量三兩，卻十分豪爽

馬雲很喜歡武俠，金庸的武俠世界是他最嚮往的地方，馬雲甚至還給自己取了「風清揚」的花名，阿里巴巴的每一位員工，入職後都被要求取一個花名。

馬雲的性格雖然很「風清揚」，但他的酒量卻沒有達到風清揚的水平。作為一個武俠小說愛好者，馬雲的酒量只停留在小嘍啰的地步。據馬雲身邊的人說，馬雲愛喝酒，尤其鍾愛茅台，但酒量不高，也就半斤的量，超過這個量，就不行了。

不過碰到欣賞的人馬雲是非常豪氣的，近日，拳王帕奎奧到訪杭州，就流出一段馬雲喝酒的視頻，視頻中的馬雲拿起酒壺一口乾，十分爽快！

其實對馬雲來說，酒更多的是一種情懷。今年9月10日，馬雲正式卸任阿里巴巴董事局主席。他送給每位阿里員工一份阿里20周年的禮物：一瓶「會說話的起泡酒」。

馬雲說：「20周年，我送大家每人一瓶酒，我在學習釀酒的時候，悟出了一個道理，工作是水，生活是酒，水會決定酒的品質，但是生活要過得向酒一樣，不能像水一樣沒有味道，祝願大家，認真生活，快樂工作，把幸福握在手中」。

馬化騰很會勸酒，自身酒量也很驚人，頗具潮汕人的氣質

小馬哥作為國內最知名的企業家之一，為人非常的低調，幾乎很少在公眾場合拋頭露面。

但低調的小馬哥酒量可不一般，作為潮汕人，小馬哥能喝一斤多的白酒，對於XO，威士忌這些洋酒，更是輕輕鬆鬆幹完一瓶。而且小馬哥特別會勸酒，對於酒桌上的禮儀非常了解，也很能活躍酒桌氣氛。

和面對外界的低調比起來，小馬哥面對朋友時還是很幽默風趣的。2016年的烏鎮互聯網峰會，在丁磊組的酒局上。小馬哥憑藉自己出色的勸酒功夫，將張朝陽，楊元慶等人灌得不輕，事後引得楊元慶發微博「抱怨」：「馬化騰最能喝最能勸酒！」

周鴻禕外號酒仙，酒量自然不是一般高

周鴻禕能說會道，被稱為是僅次於馬雲的嘴皮子，而這張嘴，除了能說，一樣能喝。

在大學時，周鴻禕就是班級里的酒仙，每逢聚會，他都是喝到最後才倒的那個，從來都是他攙醉酒的同學回宿舍。創業之後，周鴻禕的酒量可以說更上一層樓，白酒兩三斤的量，紅酒喝個四五瓶也很輕鬆。

周鴻禕早年擔任雅虎中國總裁的時候，有一次在海南三亞做活動，和手底下的員工晚上一起吃飯，喝了很多的酒。因為喝得太多，回酒店的途中不慎跌入泳池，把自己的門牙磕掉了兩顆。

劉強東千杯不醉，卻不料酒後亂性，差點毀了京東

劉強東的老家宿遷是中國著名的酒城，洋河，雙溝兩大白酒品牌皆出自這裡。在宿遷有句俗語：「麻雀都能喝二兩」，劉強東的酒量稱得上中國企業家的佼佼者，他曾經對下屬說過：「如果你連酒都不會喝，那你還能做什麼事！」

劉強東能喝，卻也因酒誤事。2018年9月2日，劉強東在美國明尼蘇達酒後性侵女大學生，被美國警方逮捕。此事導致京東股價腰斬，至今沒有恢復！而劉強東本人苦心多年經營的好形象，也因此毀於一旦。如今的劉強東，已經很少拋頭露面。

李彥宏偏愛紅酒，家中木桐數百瓶

李彥宏曾留學美國，深受歐美思想浸潤，對於歐洲的紅酒文化非常感興趣。他在法國有一處私人酒庄，收藏了很多陳年的紅酒，其中知名的拉圖，木桶年份紅酒多達數百瓶。

李彥宏愛收藏紅酒，但並不愛喝酒。他曾經對自己的下屬說過：「要時刻保持清醒的頭腦，應對隨時可能發生的情況，所以絕對不能喝醉！」

雷軍喝酒不耍猴，最大愛好是吹牛！

雷軍的手機雖然性價比高，但總是缺貨，一款新機發布，想要買到基本得排幾個月的時間，因此惹得不少人怒罵雷軍是耍猴的！

雷軍賣手機愛耍猴，喝酒之後就改行了，他在酒桌上最大的愛好，就是吹牛。雷軍好友王川曾經爆料過，雷軍經常找他喝酒，一喝多了就開始大談特談自己年輕時的輝煌成績，以及小米未來將會如何的偉大。

許家印為謀生酒桌死拼，一斤白酒天天喝

許家印出身貧寒，從小就沒了母親。年輕時，為了生計，他經常主動請客戶喝酒，爛醉如泥是常態。據他自己透露，那時候的自己，一個星期至少有4天是在酒桌上度過，平均每天至少喝一斤高度白酒，從沒有一天是完全清醒的狀態。

不過多年的酒場磨鍊，令許家印學到了很多東西，也正是憑藉這些經驗知識，許家印一步一步將恆大做大做強。如今的許總，身處高位，不再需要靠喝酒來掙錢謀生。
