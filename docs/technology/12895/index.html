最美斯柯達正式亮相，性能直指比亞迪，價格可能成為百姓最愛

2019-12-12 1,150條評論 2,924次閱讀 405人點贊

在我國新能源技術日益提高下，國內的新能源車型銷量呈現出爆髮式的增長，得到了消費者的支持與信任，不過這些銷量高的車型大多都是國產車，因為很多合資車企的混動版與純電版車型相對較少，在新能源這一領域中幾乎沒有任何優勢可言，這就讓類似比亞迪這類電機技術高超的車企獨佔鰲頭。為了能夠在新能源領域占的一席之地，許多車企都紛紛發力，就例如斯柯達近期亮相的VISION IV，作為斯柯達第一款純電動車型，它幾乎整個了企業所有的資源技術，進行全力打造，並且定價可能會是最大的亮點，下面就搶先為大家帶來斯柯達VISION IV的全面評測，為大家購車做出參考。

斯柯達 : VISION IV

外觀 :

從外觀方面來看，VISION IV的設計風格在當下市場中絕對有著強有力的競爭力，整車採用了未來的設計語言，雖然是電動車型，但前臉依舊搭載了熏黑式的中網進行裝飾，並且以貫穿式的日間行車燈完成切割，黑白相間的視覺效果衝擊力十足，下方狼爪式的黑塑料裝飾品也為整車帶來了一定的運動感。車側的設計科技感十足，摺疊式的流媒體後視鏡，隱藏式的感應車把手，隱私玻璃的設計，科技感與豪華感並存，不過流媒體後視鏡與隱藏式門把手的設計應該不會在量產車上出現。車尾的設計更加亮眼，迴旋鏢式的尾燈十分動感，下方紅色的燈帶與前臉形成一定的呼應，就連LOGO都會發光，夜晚點亮之後整個車尾非常美觀。

內飾：

目前VISION IV的車內並沒有全方位多角度呈現出來，只有一個大概的布局，從大概的輪廓可以看出，VISION IV採用了雙幅式多功能方向盤，懸浮式多媒體液晶屏的尺寸也很大，另外，中控台的功能配置都會集成到一塊屏幕當中，檔桿也可能會採用旋轉換擋方式。概念車的車內配備了各種激光掃描儀、雷達感測器和攝像頭，這也表明了它可能具備人臉識別與駕駛記憶等先進的功能，使我們在日常出行時能夠更加便捷。雖然我們沒有進車內來感受用料部分，但VISION IV作為斯柯達的核心車型，在用料方面絕對會非常可觀，這點大可不必擔心，下面我們主要來了解一下它的動力系統。

動力 :

斯柯達VISION IV混動車型的搭配形式還是未知，但純電動車型採用了雙電機驅動並實現全時四驅，最大可以迸發出310匹馬力，總功率達到了225千瓦，目前沒有關於扭矩的參數信息，但根據馬力與功率的推算，最低預計不會低於430牛米。值得一提的是，在WLTP循環下，VISION IV的綜合續航不低於500km，同時，斯柯達的快速閃充系統進行再次升級，將電量充至80%的時間縮短到了30分鐘以內，在所有純電動的新能源汽車中，這對消費者來說最為關鍵。

總結：

從各種角度來看VISION IV具有很強的競爭力，不管是顏值配置，還是動力系統，都非常可觀，不過由於我國比亞迪在新能源領域的技術過硬，推出的車型價格又非常低廉，所以VISION IV就算是發售，能不能在新能源這塊領域分得一塊蛋糕還是個未知數。不過，VISION IV主打性價比，可能在配置與用料方面有所欠缺，但定價絕不會離譜，如果能與比亞迪持平，可能會在市場中脫穎而出。大家對於這款車型有什麼看法呢？歡迎在下方留言評論，我們一起交流學習。
