全球高端手機蘋果份額佔一半，在中國華為佔80%

天天要聞
2019-12-10

錢童心

今年第三季度，蘋果公司在400美元以上高端機市場佔據了超過一半份額，佔比達52%，排名第二和第三的三星和華為的市場佔比分別僅25%和12%。

儘管iPhone銷量放緩，但蘋果公司仍然主導著全球高端智能手機市場。

根據市場研究機構Counterpoint Research的最新報告，今年第三季度，蘋果公司在400美元以上高端機市場佔據了超過一半份額，佔比達52%，比去年同期48%的佔比有所上升。而排名第二和第三的三星和華為的市場佔比分別為25%和12%。

不過在中國高端智能手機市場，華為遠超蘋果「一統天下」。市場數據顯示，受益於華為Mate 30和P30等機型，華為佔據了中國超過八成的高端智能手機市場份額，高於去年同期的73%。


Counterpoint的統計數據還表明，用戶對於價格在600美元至800美元區間的智能手機的需求顯著上升，由去年第三季度占高端機31%的份額激增到今年的43%，主要推動力量是蘋果的iPhone XR；而價格區間在400美元至600美元的智能手機的需求則由去年同期的42%大幅下滑至24%。

此外，價格在800美元至1000美元區間的智能手機的需求也大幅增長了60%，占高端智能手機市場整體銷量的21%，主要的推動因素是三星的Galaxy S10和Galaxy Note 10系列手機。

報告還顯示，第三季度智能手機的市場銷售價格平均提高了5%，這與目前5G智能手機的推出密不可分。在5G智能手機市場上，三星、LG和vivo排名分列前三，佔比分別為74%、11%和5%。小米創始人雷軍也在早些時候表示，明年小米將推出至少十款5G智能手機。

目前蘋果公司尚未發布任何5G手機，但已經有分析數據預測蘋果今年秋季發布的iPhone 11將帶動公司在第四季度的全球銷量取得突破，甚至反超華為。

根據數據統計公司TrendForce的預測數據，蘋果有望在第四季度重返全球銷量第二，市場份額將增長至18.7%。該公司統計的第三季度銷售數據顯示，在全球智能手機市場出貨量排名中，三星、華為和蘋果分別佔據前三席，份額為20.8%、18%和12.4%。其中，iPhone 11第三季度產量環比增長42%，預計第四季度將達到6900萬台。

不過第三季度蘋果在中國市場的表現出現下滑，根據行業市場分析，iPhone中國市場用戶在過去一年的流失率增長了一倍多，今年三季度，蘋果中國智能手機的出貨量僅820萬台，同期下滑14%，市場份額進一步跌至7.9%。

蘋果在中國的「遇冷」一方面是受到華為手機突起的衝擊，另一方面也是受到價格的影響。為此，蘋果公司也在價格方面做出調整，比如iPhone 11的售價相比iPhone XR發布之初就直接下降了近千元，而「雙十一」期間，蘋果iPhone 11的銷售預計達到了數萬台，也意味著降價策略在中國市場是有用的。
