日本公司看好面部識別技術 想要與中國企業爭市場

天天要聞
2019-12-20

科技訊 12月20日消息，據國外媒體報道，從購物到銀行再到登機，一種基於面部識別的經濟正在日本紮根。NEC等日本公司在機場、商店和主題公園部署安全加密的面部識別技術，與中國企業進行競爭，希望讓消費者過上無現金支付的生活。


圖示：一名乘客正通過NEC的面部識別技術登機。

當遊客在日本西部和歌山縣南紀白濱機場著陸時，首先迎接他們是一個顯示「歡迎先生/女士(遊客名)」的電子屏幕。

而在當地遊覽時，遊客可以在商店裡刷臉購買口香糖，也可以刷臉進入酒店房間。當他們前往和歌山冒險大世界主題公園時，可以通過刷臉進入，觀看熊貓啃竹子和海豚表演特技。

這是日本信息技術公司NEC上個月在白濱進行的面部識別實驗。NEC押注，在面部識別技術市場上，便利性加上在數據安全方面為用戶帶來的安心將是制勝法寶。


圖示：遊客現在可以刷臉進入日本和歌山冒險大世界主題公園。

印度研究公司Mordor Intelligence的數據顯示，隨著面部識別技術擴展到自動駕駛、金融和醫療領域，全球面部識別市場將在2018年至2024年期間擴大一倍，達到91億美元。

毫無疑問，目前中國企業在這一領域佔據主導地位，因為它們在規模和面部識別準確性方面的排名迅速上升。但日本企業也看到了其中的市場機遇。

NEC的面部識別技術世界領先。1989年，該公司就開始研究面部識別，早期的誤認率約為30%。

但NEC採用了人工智慧並自行開發了一種原始演算法，一直在穩步提高其面部識別技術的準確性和識別速度。

NEC面部識別技術的誤認率現在降到了0.5%，系統的運算速度高達每秒2.3億次。速度和容量遠遠超過其他公司，也使得NEC的技術在美國國家標準與技術協會(NIST)的基準測試中排名第一。

儘管NEC擁有技術上的優勢，但開發面部識別系統的中國供應商已經在搶佔全球市場的大部分份額。

海康威視是中國企業中獲利頗豐的一家，主要原因是其產品價格不到發達國家同類供應商的一半。

從技術層面講，中國的面部識別技術並不落後。在2018年NIST基準測試排名前五的公司中，排在NEC之後的三家機構都來自中國，它們分別是依圖科技、中科院深圳先進技術研究院和商湯科技。

與2013年相比，這是一個驚人的進步，當時排名前五的機構中沒有一個來自中國。

NEC和日本國家信息通信技術研究所聯合開發了一種面部識別系統。該系統基於量子密碼技術，具有高度安全性。系統使用一種稱為「秘密共享」的方法對加密身份驗證數據進行分割和存儲，幾乎不可能被破解。

NEC希望通過強調該系統可以降低泄露風險來推廣該系統。但與此同時，中國企業正在以較低價格打入新興市場，這意味著日本和其它競爭對手也需要提供負擔得起的產品價格以保持競爭力。

松下今年11月份推出了一款基於雲計算的面部認證系統，採用了按需付費的模式。松下對每一次身份認證收費1日元(不到1美分)，其願意以低價來推廣這項服務。（辰辰）
