全球超過比亞迪，特斯拉國內也要稱王？四款自主電動車不服

2019-12-18 1,403條評論 3,739次閱讀 95人點贊

不久前，「比亞迪全球總銷量被特斯拉反超」的消息，讓國內汽車圈兒瀰漫了悲觀的情緒。統計數據顯示，截止今年10月底，特斯拉已經交付了807,954輛，超過比亞迪的787,150輛。特斯拉全是高端豪華純電動，而比亞迪是中低端純電加插混，這樣一比差距更大。

當然，特斯拉的優勢是全球銷售，而比亞迪主要銷量還在國內，但比亞迪開拓國際市場並不容易，而國產特斯拉卻即將交車。那麼，面對即將交車的國產Model 3和Y，一直主打中低端的自主品牌有能應戰的車型嗎？三個司機認為，如下四款車型最有希望。

蔚來ES6：月銷量穩超2000台

蔚來汽車從成立開始就飽受爭議，原因竟是因為「定位太高端」，當然主要還是不相信蔚來的決心和實力。蔚來從成立到現在，也確實走得跌跌撞撞，現在仍沒有徹底走出財務困境，但產品目前確實最靠譜。

雖然蔚來的車價格不低，但是賣得還不錯，尤其是ES6月銷量已經穩定在2000台左右，是目前唯一能與特斯拉PK的量產車。中汽研的上險數據，11月蔚來上險數2619台，其中ES6達2151台，無論在國內豪華車領域，還是在純電動車領域，這個數字都可圈可點。

蔚來ES6於2018年的蔚來日（12月15日）上市，定位為中型純電車SUV，採用五座布局，標配70KWh三元鋰電池組，可選裝容量為84KWh的鎳鈷錳NCM811電芯，共有基準版、性能版以及首發創始版三個版本共6款車型，百公里加速最快4.7秒，NEDC綜合工況續航最高510公里，補貼前裸車售價在35.80萬-54.80萬元之間，妥妥的豪華車范兒。

小鵬P7：續航里程超過Model 3

國內造車新勢力沖在前面的，是蔚來、小鵬和威馬，其中威馬定位偏大眾化，所以在高端市場，另一個值得期待的是小鵬。目前，定位高端跑車的小鵬P7明年即將上市，算是可以和特斯拉Model 3直接競爭的車型。

據悉，小鵬P7將提供後驅超長續航版和四驅高性能版兩種版本的車型，其中後驅超長續航版搭載最大功率196kW（267Ps）的電機，續航里程超650km。四驅高性能版最大功率為316kW（430Ps），續航里程超550km。作為對比，特斯拉Model 3的3款車型續航分別為445km、590km以及595km。可以看到，小鵬P7在續航方面的優勢還是較為明顯的。

從外觀設計上看，小鵬P7氣場並不輸特斯拉，同樣非常炫酷，在不久前的廣州車展上，小鵬P7給出了補貼後的預售價為27萬-37萬元，一般上市後的實際價格都要低於預售價，所以小鵬P7的價格還是很親民的，相對於特斯拉Model 3有價格優勢。

比亞迪漢：百公里加速2.9秒

比亞迪老大王傳福曾說，「可以分分鐘造出特斯拉」，很多人認為在吹牛，甚至馬斯克本人也對比亞迪持不屑一顧的態度，認為比亞迪「應首先考慮在中國怎麼活下去」的問題。

比亞迪漢的出現，將改變大家對BYD的低端印象。可以說，這款車對比亞迪也是至關重要的，所以上市時間一拖再拖，早在2013年做王朝系列規劃的時間，就已經有這款車型了，但今年以前一直是傳說。「漢」其實是比亞迪542工程中的最後一款車型，542工程包含MPV、轎車、SUV、跑車，其中前三款車都已經打造出來了。按計劃，比亞迪漢將於2020年上市，是王朝系列的壓軸車型。

這款車的車身長寬高分別為4950/1890/1490mm，軸距為2940mm。DM版搭載一台2.0T發動機+前置110kW電機和後驅350kW電機所組成，百公里提速僅2.9秒。而純電版本的續航或將超過700公里，而風阻係數僅0.233Cd。

廣汽EN0.146：續航里程1000公里

比亞迪漢的風阻係數已經很強了，但是看完廣汽的數字，你會覺得更不可思議，這款車的名字就叫EN0.146，0.146Cd絕對全球第一了。眾所周知，電動車在高速行駛的時候，有近乎於70%的動力都是用於克服風阻的。

所以在電動化的時代，與其盲目疊加大容量電池增加續航，倒不如想辦法降低能耗，從而增加續航，這才是良性的發展。據廣汽研究院副院長張帆的介紹，EN0.146僅憑100kWh的容量電池，就能實現高達1000公里的續航。如果真能實現電動車確實可以取代燃油車了。

另外，EN0.146還創新般的推出了2＋1＋2＋1的六座布局，它一共有四排座椅，其中第二排和第四排設置了一個反向座椅，以供兒童使用。這種排列方法可以說是為二胎家庭量身定做的，每一個座位都是獨立的座椅，能最大程度保證乘坐的舒適性。

[總結]相對於特斯拉，自主品牌給人的感覺就是「太普通」，雖然可以解釋為「親民」，讓更多人都享受科技的樂趣，但無形中也拉低了品牌形象。從比亞迪全球銷量被特斯拉超過，可以看出品牌的重要性，好在自主品牌現在也在努力往「高大上」方向發展，當然實力如何還要等正式上市才見分曉。或許，國產後的特斯拉並不是終結者，而是一個新時代的開始。
