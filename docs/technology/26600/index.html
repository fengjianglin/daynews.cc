最新版微信脫胎換骨：有太多變化

2019-12-22 190條評論 4,462次閱讀 3297人點贊

近日微信發布了7.0.10內測版，新版的一大看點是可跟隨系統自動切換的「黑暗模式」，太平洋電腦也在第一時間對這項功能進行了評測。

不過經過這兩天試用，小編髮現7.0.10版遠不止黑暗模式這一處改進，很多細節上也都進行了調整。

變化一、消息引用回復

在聊天記錄上長按，可以看到快捷菜單里多出了一項「引用」功能。

這項功能最早出現於PC端，7.0.10版首次將它引入到移動平台。

顧名思義，「消息引用」是當你對會話里某條消息做回復時，微信自動將消息原文附加到回復語之下，以便接收者能夠快速區分你的問題。

尤其在群聊中，這項功能非常實用。

7.0.10版在引用消息的顯示效果上相比PC版更好，原文和回復語之間加入了明顯的色彩區分，而不再是PC版那個簡單的分隔線。

當然這也需要看接收者的微信版本了，如果版本號低於7.0.10，那麼顯示出來還是和PC版一樣。

變化二、「掃一掃」改版

新版掃一掃有了一點點改版，除了將底部按鈕精簡為「掃碼」、「識物」、「翻譯」三個之後，中間的掃描框也取消了，看上去更像是系統相機的調調。

新增加了三個按鈕，分別是「我的二維碼」、「手電筒」、「掃描已有圖片」。

當然這三個也都是老功能了，只不過變成按鈕後，用起來更方便了。

變化三、「識物」

「識物」是掃一掃中新增加的一項功能，說白了就是通過搜索引擎，直接告訴你當前掃描的是個什麼東東。

當然這樣的流量入口，騰訊是不會白白浪費的。

如果是日用品、食品之類，除了會有相關內容介紹外，還會附帶電商鏈接，點擊後可以直接跳轉到電商小程序完成購買。

「識物掃」挺好玩的，識別率挺高，感興趣的小夥伴可以自己試試

附有網店鏈接，感興趣的話可以直接買下它

新版微信在掃碼成功後，會彈出一個定位點，這也是老版中所沒有的。

不過這個不像前面那些模塊一樣，有什麼具體的用途，僅僅是UI方面的調整。

變化四、查看原圖

「查看原圖」與老版有所區別，「下載」和「會話中其他圖片」被挪到界面右下角，「查看原圖」則放置到左下角，應該是為方便用戶下載而做的調整。

畢竟多數網友都是左手持手機，相比之前的居中設計，新版「查看原圖」更加好按一點。

唯一美中不足的，就是按鈕小了一點，不知大家是何感受！

變化五、新版變成64位了

新版微信已經默認使用64位了，重複一遍，64位可不代表速度更快喲！

變化六、直接卡券轉贈

7.0.10版支持一鍵轉贈自己手中的微信卡券，打開會話窗口「+」，滑到第二個頁面，這裡多出了一個「卡券」按鈕。

點擊進入後，如果你的微信里有可以轉贈的卡券，就會直接顯示出來，點擊即可直接贈給好友。

變化七、「好物圈」→「圈子」

隨著「搜一搜」的升級，原本的好物圈也在新版中變身為「圈子」。

和熟悉的朋友圈不同，「圈子」更像是一個個興趣貼吧，在這裡你可以根據興趣、地域、愛好，尋找適合自己的圈子。

在圈子裡結交朋友，發表文章或視頻，或者評論一下其他好友的文章等等。

如果說朋友圈是一個熟人平台的話，那麼圈子正像是微信的陌生人平台，甚至可以看作是微信公眾號的2.0版本。

不過從目前情況來看，圈子的活躍度還是一個問題，儘管很多圈子動輒都是幾千上萬人，作品的數量也是以千來計，但每篇文章下面的點贊數和評論量寥寥無幾。

當然對於微信這樣的流量巨頭，產品定位與後續的扶持力度，往往會決定最終的結果。這個不再多說，慢慢觀察吧。

寫在最後

總體來講，這一版帶給我們的驚喜還是比較大的，除了上面這些小變化外，「黑暗模式」依然是本季更新中最靚的仔。

好了，這就是本期微信7.0.10帶給我們的小變化，你感興趣嗎？
