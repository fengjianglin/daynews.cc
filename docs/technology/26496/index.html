如何抓住5G風口推動AR和VR發展？專家在深圳這麼說

2019-12-22 785條評論 2,884次閱讀 1793人點贊

12月20日，由深圳市增強現實技術應用協會主辦的第三屆賦能/AR/VR跨界融合創新論壇在深圳舉行。

此次論壇以「5G賦能，預見未來」為主題，旨在進一步推進中國虛擬現實與增強現實技術在行業領域的應用交流，加快行業+AR/VR的產業化推進。論壇現場還發布由電子工業出版社出版的新書《互聯網下一站 5G與ARVR的融合》及深圳市AR協會整理的《2020深圳市增強現實技術應用協會會刊暨AR/VR行業目錄》，彙集5G+、行業+以及AR/VR+等解決方案。

論壇上，企業家們共同探討5G時代AR/VR技術在各領域的最新理論、技術成果與行業應用。多名行業專家在論壇上發表主題演講，對5G技術優勢、AR文旅落地、全景應用創新、5G Cloud VR、IMAX沉浸式線下娛樂新機會等多領域進行深入分析。

「5G對AR的發展有非常大的貢獻：讓AR眼鏡使用更加便捷；讓我們的產品實現規模性的量產；可以讓遊客和產品的互動性更強。」 太平洋未來科技CEO李建億表示，遊戲是AR發展的重要場景，想要得到市場的肯定，產品就要兼顧性能、功耗和體積AR為用戶帶來沉浸式的使用體驗，技術突破才是發展的原動力。

圓周率科技營銷VP田明岩表示，圓周率科技不是在生產全景相機，而是在做機器人眼、空間眼，相機只是這隻「眼睛」的其中一款應用產物。在5G驅動下，「5G+VR+8K」讓全景技術不僅僅是變成把玩的消費產品，而是要成為我們的客戶新的生產力，為我們的客戶和社會提供生產力和更高的效益。

據了解，深圳市AR協會是目前國內AR/VR行業最為活躍的行業協會。協會成立兩年來，累計對接項目超200個，舉辦大小論壇及沙龍超50場。

未來，協會將加強與各行業相關機構合作，在AR/VR產業發展、技術創新、標準制定、應用拓展、投資促進、品牌宣傳、人才培養、會展活動等方面加強合作交流，推進技術、人才、資金等資源互動。充分發揮科技社團組織的資源優勢、學術優勢和平台優勢，通過集聚資源、搭建平台，推動虛擬現實行業發展機遇，整合全產業鏈資源，幫助行業內企業突破產業壁壘，和行業共同探索AR/VR服務提供模式。
