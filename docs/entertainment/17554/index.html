王晶的眼光真是好，童菲成新一代“晶女郎”

2019-12-18 1,513條評論 3,541次閱讀 2392人點贊

相信很多讀者朋友們都認識王晶吧，就算不認識他，也一定看過他所拍攝的電影。其實他是來自香港的一個知名導演。而且他不僅在香港，甚至在全國都有很高的名氣，他也拍出了很多優秀的經典電影作品，其中最為出名的就屬《賭神》了，而且這部電影是他與周星馳和天王巨星劉德華一起拍攝的，他也因為這部電影成為了非常知名的導演，後來他又攜手周星馳拍過《賭俠》，不過還是沒能超越過《賭神》這部電影。

而對他比較了解的觀眾朋友們一定都知道，他拍電影都會找一些非常漂亮的女演員來擔當主角，因此這些在他的電影中擔當過主角的女演員被觀眾朋友們統稱為「晶女郎」。而今天小編要給大家介紹的這位女演員就是王晶作品中最新的「晶女郎」，那麼她到底是誰呢？想知道的讀者朋友們就跟著小編我的腳步一起來看一看吧！

這位新一代「晶女郎」就是童菲，她也是因為出演了王晶拍攝的電影《大上海》，並且在其中擔當女主角的角色，因此被觀眾們所認識。但是不得不說童菲真的是所有「晶女郎」里最漂亮的一個，她不僅長相十分的精緻漂亮，而且非常的獨特，給人耳目一新的感覺。而且她的美不是那種千篇一律的網紅臉，美則美矣，但是給人的感覺都長的差不多，不會給人留下太深刻的印象，童菲的美就美在她美得十分獨特，十分精緻。而且在她通過這部電影出名之後，童菲的資源也是特別的好，還和舒淇，謝霆鋒等大牌明星合作了，甚至還參加了綜藝《我們的法則》，她在其中的真實不做作的表現也圈粉了一大波的網友。

對此，就有很多網友紛紛發表了自己的觀點。有的網友表示，童菲真的是長的特別的漂亮，還很有特點，最讓人羨慕的是她的身材還特別好，但是最圈粉我的還是她大膽率真的性格。還有的網友表示，雖然我不是太了解童菲，但通過她的那些電影作品，我覺得她真的是一個很努力，也很上進的女孩子。

小編也非常贊同網友們的觀點，也希望童菲能在今後為我們帶來更多更優秀的作品，那麼讀者朋友們，你們喜歡童菲嗎？歡迎留言評論告訴我們。
