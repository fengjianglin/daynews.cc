收 藏

謝霆鋒和王菲分分合合幾次，現在還在一起，那麼為什麼兩個人好了那麼久，王菲就是進不了謝家門呢，網友分析有幾個原因。

1、原來當初王菲和李亞鵬離婚時，曾經定下協議，在女兒李嫣還沒有成年的時候，王菲和李亞鵬都不能結婚，他們一直堅守著這個協定。

2、謝霆鋒老爸謝賢不太認同王菲作為自己的兒媳婦，四哥謝賢只承認張柏芝是謝家的兒媳婦，畢竟，張柏芝為謝家生了兩個兒子，雖然謝霆鋒和張柏芝已經離婚6年，張柏芝獨力撫養著2個孩子長大，還為了養家重出江湖進軍影視圈，看著非常辛苦，但是四哥謝賢和張柏芝關係非常好，非常關照這個前兒媳婦，謝霆鋒在一次訪問中透露了箇中原因，原來謝賢目的是希望多探望自己兩個乖孫子，並決定以後的遺產由張柏芝和兩個孫子來繼承。

3、前段時間，謝霆鋒曾經和張柏芝提出過，希望能和兒子生活在一起，張柏芝當然一口回絕，並表示應該好好和王菲過生活，不要節外生枝，王菲得知這個消息後，也不太樂意，因此傳出他們鬧矛盾，所以近期結婚就更不太可能了。

4、謝霆鋒大兒子Lucas已經漸漸長大，謝振軒Lucas現年11歲了，身高可長得不錯，個子直追媽媽張柏芝，這樣看來將來也會是一個長腿歐巴，那娛樂圈真得好期待他快高長大了，若出道必然也是一個萬人關注的明星。因此張柏芝對他愛不惜手，有記者問Lucas，最近有見過爸爸謝霆鋒嗎？他說已經很久沒有見了，因為爸爸都非常忙，所以媽媽是最疼愛我們的了。

為此很多網友看了都覺得心酸，如果張柏芝和謝霆鋒在一起該有多美滿，所以無論謝賢、媒體、兒子、網友、影迷各方面都希望謝霆鋒張柏芝複合一起，這樣能為孩子創造一個溫暖的家庭環境，各位觀眾，你們認為有機會嗎？請留言發表你的寶貴意見。
