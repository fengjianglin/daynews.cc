女生提問如何嫁給易烊千璽 老師竟然畫了個魚骨圖

2019-12-10 92條評論 2,338次閱讀 78人點贊

帥氣的面龐加上略微迷人的演技，如今的小鮮肉被很多人愛到不行。當下最火的肯定是四字弟弟易烊千璽了，《少年的你》讓他火的一塌糊塗。

能從小鮮肉凈化成演技派是不容易的，四字弟弟肯定付出了很大的努力，如此優秀的苗子誰不愛呢。近日，女生提問如何嫁給易烊千璽，老師竟然畫出了魚骨圖。

看了之後你會發現，這說的都是啥，完完全全就是雞湯呢，老師終究是老師，跳不出教書育人的人設和圈子。

既然如此，小編就去搜羅點有用的東西。在某乎上看到了熱心網友給大家的想到了解決的辦法。

接下來我們看看如何嫁給偶像的正確發開方式。

如果想和愛豆談戀愛，就一定不要做他粉絲。

如果是粉絲的話，頂多也就是和愛豆曖昧一下吧…首先，作為一名粉絲，一定得花錢。要花錢去見面會握手會啊簽售會啊這種可以注意到你的地方，刷臉。這種場合要保證自己是美的，一定要打扮自己。

然後就是臉刷夠了錢也花夠了，付出夠多了，相信就能打動愛豆（不是），得到聯繫方式。

這裡我想強調的是，如何和愛豆保持曖昧。有兩點要切記。

1.得到聯繫方式不要鬆懈，要繼續刷臉。不要頻繁聯繫愛豆，人總是喜歡若即若離的感覺。就算我是你的粉絲，我也不會放下我的尊嚴（屁嘞）。

2.千萬千萬千萬不能跟愛豆發脾氣。相信接觸久了，兩個人熟悉了，是會產生誤會的。產生誤會了之後，如果發了脾氣，就會和愛豆產生隔閡。當然也不是說沒有原則和底線，說出自己的不滿即可（但最好別說，真的）

總之就是，花很多錢，累的要死要活，還得忍氣吞聲，換來的就是微不足道的一點糖。

他一旦寵了你一次，你就想讓他永遠寵你。但是他畢竟是個愛豆，粉他的姑娘千千萬，那麼多姑娘里好看的人不少，不可能沒有他動心的。你最多只能做到他現階段的曖昧對象，過一段時間，會有新的，更美的更有錢的粉絲代替你。到最後，就是人財兩空。

所以要佛系追星，別整天想怎麼跟愛豆談戀愛這些有的沒的了。真正談了戀愛，後悔都來不及。

再配上一段撒老師的名言。

撒貝南一針見血地說到：「追星，其實你是在追自己，你其實是在為自己設計一個你理想中的生活人設狀態和你想成為什麼樣的人。你其實最終追來追去，你追的都是你自己的影子。」用幹練的話語道出了追星的真諦。如果按照這條路走下去，最終，就會一步一步地接近自己喜歡的人，想要的東西。在不知不覺中，也變成了自己喜歡的樣子。

偶像身上有正向的、積極的、勵志的正能量，也正是你需要的。偶像是鏡子，折射的是你希望自己成為的樣子。
