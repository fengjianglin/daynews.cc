她為事業丁克10多年，終被丈夫感動41歲生子，現成了這般

2019-12-21 54條評論 2,155次閱讀 521人點贊

娛樂圈當中的眾多明星們的感情一直都被人們所高度關注，我們都知道娛樂圈是一個大染缸，在這個圈子當中對於女性的要求也是比較高而殘酷的，丁克這樣的事情也出現的比較多了，有些人是因為自己理念的原因，而有一些則是因為自己身體的原因，很多的明星在自己事業的高峰時期都不選擇生孩子，在娛樂圈當中有一位女明星就是這樣選擇的。

謝蘭，或許大家對於這個演員的名字不是特別熟悉，畢竟在上個世紀的一些老戲骨都不被人們所知道名字，她就是曾經搭檔國家一級演員陳寶國和斯琴高娃出演了《大宅門》，當時憑藉這一部作品也開始讓她走紅，不管是從演技方面來看還是從人際關係上面看，她一直都有著十分好的口碑。

結婚之後的謝蘭一直到現如今也已經有二十多年的時間了，可以說她也是在感情方面比較順利的一位女明星了，畢竟在圈子當中我們經常能夠看到離過很多次婚的明星，他們二十多年的婚姻也是非常幸福的，丈夫也是一個非常具有責任心和擔當的男人。

其實在剛剛步入婚姻殿堂的時候，謝蘭就已經有了為了工作不要孩子的決定，剛剛開始的時候事業確實是比較穩定，但是到了後期她的事業就開始越來越不順利了，在演藝圈當中也是經常碰壁，這樣的一位女明星卻堅持丁克了十多年的時間，好在在丈夫的疏導之下結婚第十年的時候全身心回到了家庭當中，懷孕的她也被丈夫所感動41歲生下了孩子，現如今的謝蘭也已經四十五歲了，但是一家三口卻非常幸福，也希望她能一直幸福下去。
