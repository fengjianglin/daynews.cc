另外，恭喜#楊紫#憑《親愛的熱愛的》佟年一角獲當代題材電視劇最佳女演員！可是啊可是，不知道是不是和肖戰提前約好了，楊紫也沒來，不過她的原因是天氣導致的無法到場，所以說，想​來卻來不了的楊紫獲獎了，檔期衝突的肖戰獎項取消了。

在頒發中國百強電視劇最佳男主角時，#江珊摔跤# #最佳摔跤手江珊# 頒獎嘉賓江珊在上台時因台上燈光原因差點摔倒，江珊在做自我介紹時，高情商化解尷尬「我是最佳摔跤手—江珊」。 其實，江珊是倒時差沒倒好啊，哈哈#江珊倒時差沒倒好#

雖然陳寶國和惠英紅分別獲得了中國百強電視劇最佳男女主角獎，即華鼎獎視帝視後，可是都沒有江珊那麼高光，現場觀眾的眼光都被江珊吸引過去了。
