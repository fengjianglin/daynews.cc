陶玉玲：為找她出演《柳堡的故事》，導演組追著她跑了大半個江南

2019-12-16 615條評論 3,791次閱讀 1362人點贊

1957年，原著作者石言和著名編劇黃宗江將《柳堡的故事》劇本改編完成。八一電影製片廠決定立即投入拍攝，並指定由新中國第一位女導演王蘋來執導這部影片。

新中國第一位女導演 王蘋

王蘋，原名王光珍，南京人。1935年，身為小學教師的王蘋參加了磨風藝社排演的易卜生名劇《娜拉》。這是一出反壓迫的進步戲劇，導演選中了性格剛強的王蘋扮演娜拉，她的演出非常成功，轟動一時。從此王蘋投身到滾滾的革命洪流之中，出演了《一江春水向東流》、《麗人行》等影片。

年輕時的王蘋

不僅如此，王蘋還是新中國第一位女導演，她後來還曾執導過《永不消逝的電波》、《霓虹燈下的哨兵》等經典影片。

王蘋接手《柳堡的故事》後，又組織人按劇情需要對劇本作了適當的修改。但卻一直沒有物色到適合演「二妹子」的人選。他們去各大劇團看演出，挑演員，也有知名導演進行推薦，這些候選人都不太符合二妹子的感覺。有的雖漂亮，卻不像一個羞澀質樸的農村姑娘，有的模樣也標誌，卻差了一股子甜美的感覺。這可急壞了編劇和導演。直到有一天，一個吃著冰棍的姑娘闖入他們的眼帘。

黃宗江

一天，編劇黃宗江和石言，來到南京軍區前線話劇團看話劇，也想藉此機會尋找合適的演員。而陶玉玲正在舞台上參演一部話劇，當時她並不知道八一電影製片廠正為拍攝《柳堡的故事》尋找「二妹子」的扮演者，更不知道這部影片的編劇胡石言與黃宗江就在台下觀看她的表演。

少年時期的陶玉玲

黃宗江和石言對陶玉玲的表演印象挺深的，覺得這姑娘很有靈氣。巧的是，演出結束後，二人在劇院門口，又撞上了陶玉玲，她剛表演完，正買了一個冰棍開心地邊吃邊迎面走來，這姑娘一抬眼，看到黃宗江和石言，害羞地笑了笑，喊了一聲「老師」，並給他們讓了一下道，就走了。這時，黃宗江和石言很有默契地相視一笑，都覺得一下子找到了二妹子的感覺，這位姑娘很清純，就是她了！他們立刻把她推薦給導演王蘋。

陶玉玲

陶玉玲當時正跟隨南京軍區前線話劇團在各省演出，當導演組趕到南京的時候，陶玉玲已經去了浙江了。劇組又馬不停蹄趕到浙江，可這時陶玉玲又已經去了舟山群島。就這樣，劇組為了找到陶玉玲，追了足足大半個江南。好不容易找到了陶玉玲，沒想到卻她卻連說不行不行，這是怎麼回事呢？

原來啊，陶玉玲那段時間一直跟隨南京軍區前線話劇團，在島上沒日沒夜地演出，大概是化妝感染的緣故，眼睛腫得像個大核桃。而導演組一來就要給她拍照，這樣拍那樣拍的，她覺得自己現在的樣子怎麼能夠去選了當電影演員呢？導演組好不容易找到她，哪肯輕易放棄，他們堅持說只是拍個照片讓廠里的領導們看看，最後還以試鏡的形象為準。陶玉玲這才勉強同意拍照，但是她壓根沒想到自己能選上。

照片上的陶玉玲眼睛確實有些腫，不是她的最佳狀態，因此，有人認為陶玉玲不是二妹子的最佳人選。可是最後，王蘋導演還是在一片反對聲中定下了陶玉玲，她認為二妹子應該很甜，而陶玉玲的酒窩比眼睛還要迷人，她平時又愛笑，長倆小酒窩更是甜美可人。
