01、人生如戲，戲如人生

藝人進入到娛樂圈當中一般有這樣的三種情況，一種是家庭環境好，進入到娛樂圈當中只不是玩玩，而且背後又家庭支撐所以不會發生什麼事情。

有的人可能是自身條件優秀，比方說有演技，歌喉好之類的。但還有一種就是為了謀生活了，進入娛樂圈完全就是為了掙錢養家。

而今天跟大家說的這個藝人完全是第三種情況，身世極其悲慘，很多人都把她比作泰國的張韶涵。

pat全名PatNapapaThantrakul是泰國娛樂圈當中十分出名的小清新女藝人，但是在光鮮亮麗的外表之下，pat卻有著電視劇都不敢寫的撒狗血經歷。

作為泰國當中的女藝人，接受媒體採訪時非常正常的事情，而在一檔節目當中她卻意外的爆料，自己一直稱呼為各個的人是自己的生父，而自己的母親確實奶奶。而她自己也是在十年前才知道這個事情。

當時她說：撫養自己的爸爸媽媽，按照輩分來講應該是爺爺奶奶，而生父是奶奶的兒子，但是因為之前被撫養的原因，所以一直叫生父哥哥。

如今哥哥已經有了自己的家庭，並且兩人之間沒有過多的交集。

很多觀眾在聽到這則新聞的時候，頓時就傻眼了。他們家的關係真得是亂如麻啊，沒有想到電視劇裡面看過的劇情在現實的生活中也會發生。看來真的是人生如戲，戲如人生。

02、面臨困境，從來不怨天尤人

其實pat出名很早，小時候的她，長相甜美，清純可人，被很多同齡小朋友嫉妒。

在她還在念書的時候，她跟母親兩人在逛商場的時候被星探發現。當時的那個人就建議pat，希望他能夠從事演藝行業，能夠進入到娛樂圈當中。

只是當時年紀尚小，覺得這個讓她進入娛樂圈的人是個騙子，所以並沒有過多的關注。

但是星探卻不願意放棄任何一個機會，一次次的用誠意打動pat，最終她接受了星探的邀約拍攝了自己人生的第一部廣告。

出道之後的pat因為出眾的樣貌獲得了很多觀眾注意，很多人都說她的長相太過於青春了吧，感覺笑起來的樣子，什麼的甜美無害。

年少的她對於娛樂圈沒有過多的眷念，當時她只想要完成自己的學業，一戰成名之後。她還是回到了校園之中，當時的她決定在完成學業之後，再重返娛樂圈。

在學校的那段時間，她努力學習，培養自己的專業知識。畢業之後，他被泰國三立所看重，成為他們旗下的一個藝人。

再次出發，公司並沒有給pat特別好的資源。只是讓她出演了一些電視劇的配角，而且這些配角都是反面的形象。

雖然說角色都是反面的，但是pat並不在意，她重視每個角色，珍視每一次機會。因為她不僅僅是一個藝人，更多的是還要負擔家庭的責任。

pat的家境並不是很好，為她的母親還患有阿茲海默症。為了能夠延遲母親的生命，她每個月都需要付高昂的醫藥費。但是面臨這樣的困境，她從來不怨天尤人，而是一直努力。

03、梅花香自苦寒來
