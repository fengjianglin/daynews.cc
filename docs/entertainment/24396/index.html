72歲鄭少秋罕現身，精壯不輸年輕小伙！對身旁女生甜言蜜語超會撩

2019-12-18 834條評論 2,243次閱讀 783人點贊

戳右邊關注我們，每天都有最新娛樂八卦等你批閱！

12月16日，有網友曬出在機場偶遇鄭少秋的視頻。當天鄭少秋穿著厚厚的外套搭配牛仔褲，系著藍色印花的圍巾，單肩挎著背包，休閑又帥氣，挺拔的身姿看起來絲毫不像是72歲的老年人，精壯完全不輸年輕小伙。

​視頻中鄭少秋在走動中與旁邊的人說話，據上傳視頻的網友表示，當天他一直誇獎女粉絲好靚，不僅頭髮和皮膚狀態好，甚至連妝容都很美麗，把粉絲誇得心花怒放，不得不說如今70多歲的秋官依舊很會「撩」。

​不過這樣會「撩」的鄭少秋在感情上卻並不是一帆風順的。

鄭少秋一共有三段婚姻。他本人曾爆料過自己的第一任妻子，稱兩人在演話劇時相戀，育有一女，不過很快因為性格不合便離婚，而女兒則被前妻抱走。

​他的第二段婚姻是和肥姐沈殿霞，兩人在同居十年後才得以結婚，也生有一女。當時兩人在一起的時候，鄭少秋已經成名。不過因為外貌差距過大的原因，有不少人認為他靠女方「上位」，還曾被曝「出軌」官晶華。

而後鄭少秋和肥姐離婚，更是讓他背上「渣男」的稱號，直到現在還依舊有人稱他是「現代陳世美」。

​鄭少秋在1989年時和官晶華結婚，雖然兩人的婚姻並不被大多數人祝福，官晶華也被說「小三上位」。不過兩人從結婚至今，已相伴30年，一直都很恩愛，並且在婚後生有兩女，一家人過得很幸福。

​相比於婚姻上的一波三折，鄭少秋的事業還算順利。20歲踏入演藝圈，鄭少秋塑造了很多經典角色，比如風度翩翩楚留香，仿若盜帥本人；亦或是丁蟹這個矛盾十足的「壞人」。

​如今他入行奮鬥五十多年，一直沒有放棄演戲，2018年時他參與拍攝《將夜》備受好評，2019年參與拍攝的《詭探前傳》也已經播出，演技依舊在線。

​當然秋官不僅僅沒有放棄演戲，同樣沒有放棄音樂，如今已經70多歲的他依舊在各地開演唱會，還時不時與汪明荃老師同台演出。

​12月12日時，汪明荃老師曬出兩人在吉隆坡開演唱會時拍攝的照片。照片中的兩位老師都不再年輕，看起來卻仍像四五十歲一樣。

​看到照片，網友在感嘆他們的高顏值之外，還調侃這兩人是否都吃了「防腐劑」。說起來，72歲的秋官之所以能保持年輕，最大的原因是他的自律。他曾在接受採訪時表示過自己這麼多年一直堅持健身，並且拒絕抽煙喝酒等不良嗜好，保持良好的心態。

​這種心態不僅僅是保持開心，還有面對風雨時的淡定，比如他對待二女兒的態度。雖然當初有人罵鄭少秋和肥姐離婚，拋棄二女兒，而鄭少秋對於女兒愧疚不忘彌補她。

​比如鄭欣宜在紅館開演唱會時，父親鄭少秋便去為女兒捧場；鄭欣宜參加綜藝，鄭少秋也登台演唱為女兒助陣。

​而近日汪明荃曬出照片後，鄭欣宜還調侃希望她能教自己的父親開社交賬號，可見兩人的互動已不受母親的影響，如今的父女關係並不差。

​出道這麼多年，鄭少秋的心似乎不曾隨著年齡老去，也沒有改變過自己的初心，這大概是他能保持家庭和睦、以及粉絲們為什麼一直喜歡他的原因之一吧。
