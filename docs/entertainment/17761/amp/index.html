愛優騰2020年四個季度劇目（部分）預排

天天要聞
2019-12-16

作者 / 周燚

影視行業區別其它行業的一個最大特點就是「時間差」：從開機到播出存在一個製作周期，這個周期少則半年，長則長達兩年。很多頭部大劇從開機到播出就有近2年的時間差。

而排播又是受多重因素的影響，尤其是在總局宏觀政策趨嚴的情況下，平台往往也是等通知，更不用說製作方的被動。

今年的暑期檔和年底檔，就是古裝劇集中釋放的缺口，多部古裝劇爭相播出。

去年年底，《大江大河》和《知否知否應是綠肥紅瘦》撞了檔期，2018年12月10日《大江大河》在東方衛視、北京衛視播出，隨後2018年12月25日《知否知否應是綠肥紅瘦》在湖南衛視金鷹獨播劇場播出，兩個劇存在一個10多天的「重疊期」，這期間就相當於正午陽光「自家的劇打自家的劇」。

《知否》看片會上，侯鴻亮就說：從製作方的角度，這肯定不是最理想的排播方式，但是這不是製作方決定的，是平台的排播戰略和大方向的排播計劃決定的。

除了宏觀調控政策因素，特殊題材、特殊內容的嚴苛審查導致的過審周期過長也是影響劇集在預定時間上線的重要因素。

比如正午陽光的《尉官正年輕》後改名的《我們正年輕》就沒有在70周年作為獻禮劇開播，至今仍無確切的播出檔期。坊間流傳該劇延緩播出是因為題材特殊，軍方的審核流程複雜、審查細節繁瑣所致。

另一部大劇《人民的財產》由《人民的名義》編劇周梅森擔任編劇，因為題材是講述國企改革和反腐，受到業內人和觀眾的極大期待。《人民的財產》於2018年12月殺青，目前暫無確切的播出時間。

總結起來，劇目播出主要受三個要素的影響：

第一：總局的宏觀調控政策，比如對古裝劇的調控。

還比如任務劇、獻禮劇。中央廣播電視總台出品的獻禮劇《激情的歲月》2019年7月16日殺青，2019年9月23日在央視一套黃金檔播出，從殺青到開播前後僅相差2個月。

第二：特殊題材、特殊內容劇目會受到官方的細節化審查。

第三：假期形成的大檔期，比如每年暑期檔、跨年檔、春節檔平台都紛紛上線「殺手鐧大劇」，重要檔期打得火熱，造福觀眾。

比如《少年派》考慮在暑期檔金鷹劇場開播，就可能與湖南衛視暑期受眾有關（學生放假）。

上線時間以實際播出為準

愛奇藝投資，克頓辛迪加製作的《愛情公寓5》是2020年開年大劇，改劇也將會同步登陸東方衛視。

回溯《愛情公寓》系列劇的播出時間：第一季於2009年8月5日在江西衛視獨家首播；第二季於2011年1月24日在東方衛視、廣東衛視、寧夏衛視、山東教育電視台首播；第三季於2012年7月30日在安徽衛視、湖北衛視、陝西衛視首播；第四季於2014年1月17日於安徽衛視、東方衛視、黑龍江衛視、湖北衛視聯合首播。

暑期和開年，均是《愛情公寓》播出的合適檔期，《愛情公寓》第二季、第三季、第五季均在開年檔播出。

愛奇藝和歡娛影視合作的自製劇《鬢邊不是海棠紅》是愛奇藝一季度的重點項目。在此之前的成功案例就有愛奇藝和歡娛影視締造的暑期爆款《延禧攻略》。

涉案劇是愛奇藝的特色和長項，預計Q1季度愛奇藝會上線品質上佳的《唐人街探案》，網劇版《唐人街探案》一共只有12集。從內容儲備的題材比例來看，Q2、Q3、Q4不同季度愛奇藝2020將有多部涉案劇上線，比如秦昊、王景春出演《隱秘的角落》，廖凡、白宇主演的《沉默的真相》，朱亞文、金晨飾演的《十日遊戲》，王千源、鹿晗主演的《在劫難逃》等。

另外興格傳媒製作、愛奇藝投資的電視劇《誰說我結不了婚》大概率會Q1季度在愛奇藝獨播。

表格也顯示了一些台網劇的大致檔期，這些台網劇也將會在騰訊視頻、優酷等平台播出。

《大宋宮詞》《如果歲月可回頭》《風暴舞》《新世界》《完美關係》在Q1季度；

《我們正年輕》《半生緣》《特戰榮耀》在Q2季度；

《人民的財產》《暴風眼》《青春創世紀》在Q3季度；

《大江大河2》在Q4季度。

上線時間以實際播出為準

愛奇藝的《大主宰》將會在12月26號上線，騰訊視頻的《三生三世枕上書》最遲最遲會在Q1季度播出，也可能在12月末跨年播出。

本山傳媒的《劉老根3》預計會春節檔在騰訊視頻獨家播出，《鄉村愛情浪漫曲》（第8部）2015年12月11日在騰訊視頻獨播；《鄉村愛情進行曲》（第9部）2017年2月1日大年初五在騰訊視頻獨播；《鄉村愛情協奏曲》（第10部）2018年2月5日在騰訊視頻獨家首播。

騰訊視頻自製劇《致我們甜甜的小美滿》預排顯示在Q1季度，《致我們單純的小美好》《致我們暖暖的小時光》都在騰訊視頻取得較好的口碑。

《鬼吹燈之龍嶺迷窟》預排也顯示在Q1季度，先前的《怒晴湘西》於2019年1月21日在騰訊視頻獨播，憑藉精良的製作獲得一眾好評，播放數據較好。

大IP改編劇目《斗羅大陸》預計會在暑期或者暑期前播出，該劇由新麗承製，《慶余年》編劇王倦參與劇本把關。

電競題材超級網劇《穿越火線》適合在2020年暑期上線，2019年暑期上線的《全職高手》已經做出了好的表率。

企鵝影視自製，唐嫣、竇驍主演的大型古裝劇《燕雲台》目前顯示在Q3季度播出。

表格所列劇目只是騰訊視頻2020部分待播自製劇，更多騰訊視頻主投、參投、獨家買斷的獨播版權劇，比如《有翡》《三十而已》《賣房子的人》等大劇也會在2020年紛紛開播。

上線時間以實際播出為準

2019年優酷播出的劇目《長安十二時辰》《鶴唳華亭》《大明風華》等劇，大多是2017年、2018年開機劇目，算是優酷人事組織大換血前的「遺留項目」。

2019年，優酷有多部自製劇開機，同時又採購了多部版權劇，這些操作顯示了優酷在做好內容上的決心。這些新開機劇目或採購劇目預計會在2020年Q3季度、Q4季度上線。

《鄉村愛情12》預計2020開年會在優酷播出，《鄉村愛情11》於2019年1月26日在優酷視頻獨家播出，市場反饋效果較好。優酷的《鄉村愛情12》可與騰訊視頻的《劉老根3》對打。

《冰糖燉雪梨》也是優酷的重點項目，預計會在Q1季度播出，該劇製作班底是《香蜜沉沉燼如霜》的原班人馬。

優酷採購的版權劇，《三生有幸遇見你》《最初的相遇最後的別離》預排顯示在Q2、Q3季度播出。

而優酷2019年下半年開機的定製劇、自製劇，比如《司藤》《十二譚》《危機先生》等劇，預計會排到2020年Q4季度或者之後。
