被稱為“渣男”，卻屢屢收穫愛情，李晨為什麼還是那麼紅

2019-12-23 1,004條評論 1,939次閱讀 357人點贊

感情真的是一個玄乎的東西，面對感情，總想討論出孰對孰錯，總想得到一個自己認為心安理得的結果，可是誰又真的得到呢。感情就是一段剪不斷理還亂的糾葛，對於真正愛過的人來說，感情重要的不是誰對誰錯，而是在一起的時光。

那些美好的，傷感的，都會成為你這一輩子最難忘懷的回憶。只要你在那一刻是真心的，只要你曾無怨無悔過，那麼這何嘗不是一段美妙的旅程。因為愛情，李晨被無數人詆毀，網路上罵起「渣男」的人不計其數，可是大家想過沒有，為什麼人們口中的渣男，還是得到眾多美女的垂青。

李晨變為「渣男」是有幾個方面的原因的。

首當其衝的當然是其的感情生活，他交往過多任女友，個個都是大美女。從初戀女友牛萌萌到大美女范冰冰，他的感情生活真的像個傳奇。作為男性，我就是其中羨慕嫉妒恨中的一員，為什麼便宜都被他佔了呢？真的是堪稱娛樂圈的「女神收割機」。

在國內，一夫一妻的傳統觀念還根深蒂固，畢竟咋們沒有人家「歪果仁」開放啊。你要是三天兩頭換朋友，在人們的眼中你不是渣男也變渣男。這個渣男便是人們思想中給你強加的人設，你不服不行。

作為對比，我們隆重介紹國外男神萊昂納多。據不完全統計20多年一共換了近57位漂亮女友，其中大部分還都是超級模特。如此美艷的生活，卻似乎沒有粉絲吐槽過小李子是個渣男。而且更讓人吃驚的是從不跟女友談結婚，合則來不合則散，那為什麼沒有人叫他「渣男」呢？首先他是一個很有正面形象意識的男人，平時還喜歡做一些慈善公益活動，總會給人一種親和力。其次呢小李子為人很懂得低調，儘管頻頻換女朋友也不會拿出來顯擺，招惹一些人吃醋。第三，人家分手了不會再與前任有任何瓜葛，任何交集。

對比李晨，他與女友的感情經常讓他登上熱搜，雖然你沒有顯擺的成分，但是觀眾的潛意識裡你的顯擺意味就很足，所以必然招惹到一部分的群體。另外，與張姓女星的互撕更是成為了眾多人口中的把柄。李晨是不是渣男我們先不去評論，但是做事不低調，這個事情上你確實得認輸。

感情生活的多變以及與兄弟印小天「插刀門」事件，讓「渣男」人設漸漸成立。李晨曾與印小天十分要好，當時曝出印小天毆打女友邊瀟瀟，李晨卻站在邊瀟瀟一邊，並要求印小天道歉。後來由視頻證明印小天是冤枉的，李晨還拒絕道歉。此事被稱為「插刀門」。

其實從李晨頻繁換女友還有維護邊瀟瀟的事件，我們不能草率的認為他就是一個不仗義不重感情的「渣男」，畢竟我們是局外人，我們並不知道局裡的世界。

張譯曾經坦言：李晨究竟是什麼樣的人，圈裡面的人都知曉。李晨雖然不能用實在這個詞來形容他。但他確實是一個老好人。對待身邊的朋友，誰有事情的時候找他都會幫忙。對待每一任戀人，他都送上了自己認為意義深刻的「心形石頭」。

為什麼說意義深刻呢？現在的李晨很少被談到他的家庭，很少的人知道他童年的時候生活在一個不太幸福的家庭當中，在他很小的時候，他的父母就已經離異。11歲時李晨為了想讓媽媽開心就拿磚頭磨成心形，直到雙手都是鮮血，晨媽媽感動的眼淚嘩嘩！所以，從小懂事的李晨送心形石頭來表達對媽媽的愛，這種愛也是發至內心的愛。而戀愛時送女友心形石頭也一定是認定了這份愛情，不然不會把這麼具體意義的石頭送給其他的女人。

李晨並不是一個很善於偽裝的人，簡單認真低調，不世故不野心，不會想太多，他給人的感覺就是老實憨厚。他對待女友體貼入微，這可能也是女神拜倒在他的牛仔褲下的原因。

有一期李晨與姓范女友參加《康熙來了》，小S直觸試探他們的的房事生活，而憨厚的李晨則直接答道：我從澳洲回來的第一天吧！姓范女友看到這麼可愛的李晨，真的讓人恨不起來。還有一段視頻，李晨和姓范女友兩個人在一塊坐著，參與綜藝節目，也許是那個房間裡面太熱，范的面前放了一個小風扇，後來范把小風扇放到李晨面前，很明顯范的意思是想讓李晨吹一吹，但是剛放過去馬上被李晨拒絕了，李晨把風扇放在范面前，從這個細微的小動作來看，我覺得李晨很暖，他懂得照顧女生！

在綜藝節目《跑男》裡面李晨的表現很突出，在「撕名牌」的時候很少撕女生，很多人都說李晨很紳士，對別人也很有禮貌，無論是誰出來評價，大家都會說李晨很老實。

憨厚老實，對女友體貼入微，為人熱情這些都是李晨的標籤，能抱得美人歸一定是他身上持有的某種特質。感情的事誰也說不清楚，娛樂圈複雜的很，干擾因素太多，誰也說不清一個所以然。
