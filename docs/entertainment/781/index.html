第一位：蘇醒，說到他，我想大家都是不陌生的，他憑藉《快樂男聲》步入演藝圈，而且還認識了李煒，而蘇醒也是經常會帶著他一起同台演出，但是結果，李煒卻是在微博上發了跟蘇醒的女朋友的親密照片，而蘇醒知道後很生氣，因此還對蘇醒大打出手。

第二位：藍正龍，說到他，我想大家都是不陌生的吧，他當時跟大S拍《流星花園》而因戲生情，在當時，大家是非常看好他們的，但是後來大S和藍正龍的兄弟周渝民在一起了，而兩人最後也是沒有什麼聯繫了！

第三位：古天樂，說到他，我想大家一定是不陌生了，而他在20歲那年，替一些不務正業的「兄弟」扛下罪名入獄兩年，當時他拜託自己的兄弟照顧自己的女朋友，可沒想到的是，他的「好兄弟」卻直接撬牆角，和他女朋友在一起了，出獄後的古天樂浪子回頭，如今也是成為了大明星，而且還到處捐學校。

第五位：歐弟，說到歐弟，我想大家一定是不陌生的吧，歐弟跟羅志祥是好兄弟，但是歐弟要去服兵役，而一去就是兩年，於是歐弟就讓羅志祥照顧好自己的女友，但是讓人尷尬的是，後來羅志祥跟歐弟的女友產生了感情，而兩人也是經歷了一段尷尬的時期，但是後來歐弟也是原諒了羅志祥。
