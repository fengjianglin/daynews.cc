行業寒冬？量少質優也不錯

都市話題勵志劇《第二次也很美》正在東方衛視熱播，天津籍演員于小偉（上圖）在劇中飾演暢銷書作家俞非凡。日前該劇在津舉辦觀眾見面會，于小偉回到家鄉與觀眾見面交流互動。在接受記者採訪時，他講述了自己對於角色、行業等方面的看法。

多部作品近期受關注

這段時間，于小偉參演的多部電視劇連續播出。從之前在央視播出的年代劇《靈與肉》，到湖南衛視播出的諜戰劇《驚蟄》，再到目前東方衛視播出的都市情感劇《第二次也很美》、正在騰訊視頻播出的古裝大劇《慶余年》，于小偉可謂「霸屏」。在四部類型完全不同的作品中，于小偉飾演四個不同時代、不同時空的人物遊刃有餘，演技展露無遺。

在《第二次也很美》中，于小偉飾演的著名作家俞非凡是一個典型的高富帥。談及這個角色，于小偉說：「俞非凡是一個性格模糊的角色，他要對很多人和事負責，但在感情上卻優柔寡斷。這個角色距離我自己很遠，我也不是很喜歡俞非凡這個人物的性格，但一部劇中的每個角色都有自己的作用，所以我很認真地按照劇本的要求把他演好。」

更加偏愛古裝劇角色

在近期的這些作品中，于小偉直言最喜歡《慶余年》中的角色。他說：「在《慶余年》的故事裡，有慶國和北齊兩個國家，我飾演的這個角色會在後半部分出現。他是北齊的權臣，位高權重、工於心計，是個大反派。很多男演員都會比較喜歡權謀劇，我也一樣，對這個角色投入的心血更多一些，演得也比較過癮。」

現代劇和古裝劇，哪類角色塑造起來更容易？他說：「現代劇雖然距離我們的生活更近，但角色塑造上並不一定更容易。當飾演角色與自身距離較遠時，表演上我得努力去找那個感覺。」而在《慶余年》這類架空歷史的古裝劇中，對角色的塑造反而可以更自由。

調整心態面對行業寒冬

近兩年來受多方因素影響，影視行業進入寒冬。受此影響，很多演員的工作機會減少。于小偉表示自己也有這種感受，但心態方面受到的影響並不大。他說：「的確，影視項目開機的少了，對於我們演員來講，選擇角色的機會也沒有那麼多了。但從另一個角度來看，現在的影視作品質量也越來越好，特別是一些在網路播出的作品也不錯。所以我在選擇作品的時候，只去想怎麼拍出好作品，並不會刻意區分它是網路劇還是電視台播齣劇。整個行業內，大家都在努力追求作品的品質，這對觀眾、對演員都是一件好事。」

于小偉認為，面對這種行業狀態，調整好心態很重要。「不工作的時候，我就看書、跟家人一起旅遊，生活也是對表演的一種積累。」由於工作的原因，于小偉並不能常常回天津。「一年大約回來一兩次吧，我把媽媽接到北京去一起生活，這樣對我來講更方便一些。」儘管回來的少，但每次回來都會明顯地感覺到天津城市面貌的變化。「每次回來都很高興，感覺都很親切。咱天津的快速路四通八達，我每次都必須要原路來、原路回，不然會很容易迷路。」于小偉說。（記者 劉桂芳 記者 王津攝）
