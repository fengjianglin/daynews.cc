陳嘉桓搬離黃宗澤豪宅，浪子本性難改，左擁右抱與兩女同時拍拖？

2019-12-24 2,020條評論 4,700次閱讀 326人點贊

在感情的世界裡，婚姻從來不是唯一的保障。但連一紙婚書、一個名分、一句承諾都沒，那這段關係可能會變得岌岌可危。尤其是當你愛上了一個浪子，就要做好隨時會被別人挑戰「主權」的準備。

在剛過去的冬至，香港娛樂圈突然爆出了一單三角戀事件。事緣就是一直被傳和黃宗澤秘密同居的陳嘉桓，最近突然搬離了黃宗澤的「黃宮」豪宅。

而同一時間，一位叫張沙沙的內地女星突然襲港，趁著黃宗澤生日當天去了他的片場探班。於是乎，一出「兩女爭一男，誰才是正印」的大戲就猛然開始了。

雖然說，在這個新電影拍攝的階段，這位張沙沙突然出現，又是和黃宗澤爆出戀情，又是和鄭嘉穎曬出合照，大有一番為新電影宣傳搏噱頭的嫌疑。

但是俗話說得好，蒼蠅不叮無縫的蛋。黃宗澤也不會無緣無故地就和這位張沙沙傳出緋聞。

不然男明星那麼多，在拍的電影也那麼多。為何傳緋聞的就不是張家輝，劉青雲，又或者古天樂呢？所以空穴來風，未必無因。

不過說到這裡，還是要先介紹一下這位張沙沙到底何許人也。話說這位出生於黑龍江的張小姐，長得是貌美高挑，身材姣好。只可惜入行多年一直不紅，作品也沒拍幾部，倒是花邊新聞最為大家津津樂道。

16年的時候，也是一則出軌新聞把張小姐帶到了大家面前，當時一個普通網友突然哭哭啼啼地爆料給媒體：「嗚嗚嗚，我的男朋友和女明星偷情了，他是做投行的，我們倆因為工作原因是異地戀，我最近察覺到男朋友有點變心，於是我打開了他的電腦，看到裡面居然有不雅照！而且，和他約的那個女人，居然還是個十八線小明星！」

一時間，這單爆料引爆了大家的好奇，吃瓜群眾們紛紛都去查證，這個十八線小明星到底是誰。一查才知道，這位小明星就是張小姐，而更奇葩的是，當時張小姐已經有個正牌男朋友江奇霖。

那段時間，江奇霖正憑藉著在《太子妃升職記》裡面演「綠王」而稍有名氣。可誰能想到，在自己終於要迎來事業春天時，女朋友卻在和別的男人曖昧出軌。雖然事後他們都發聲明否認了「綠帽事件」，但是毫不意外地，這倆人最後就是分手收場。

然而，即使是經歷了這麼一單「紅男綠女」的桃色新聞，即使是身處十八線，怎麼都捧不紅。但似乎也並沒有影響到張小姐在行內的異性緣，而她和黃宗澤據說就是前兩年一起拍《我的真命天女》時擦出愛火花的。

大家都知道，出了名愛玩的Bosco一直酷愛都市麗人類型的窈窕美女。而十八線的張小姐能得到浪子垂青，除了外形條件以外或者還是因為她特別了解「渣男需求」。

我不需要你給我名分，我也不需要你常常陪在身邊。你要工作，你要去玩，我都可以理解，只要我們在一起的時候開開心心就夠了。

看吧，擁有一個長得美、想得開，還不會老是纏著你的女朋友，不正是每個浪子都渴望的嗎？

就這樣，傳說這位張小姐就是靠著格外討好的女友作風，和黃宗澤秘密拍拖了兩三年，一直都沒有透出半點風聲。

而與此同時，在香港這邊黃宗澤和陳嘉桓的緋聞也是傳得沸沸揚揚。前段時間，陳嘉桓的弟弟過生日，陳嘉桓就帶著家人去了黃宗澤的餐廳幫弟弟慶生，此番舉動，宛若女主人。

或者在嘉桓妹的內心深處，也早就把自己當做了黃家女主人看待。因為自從黃宗澤因為拍戲受傷，行動不便之後。就有傳聞說，陳嘉桓為了照顧宗澤搬進了他家的豪宅，溫柔服侍，照顧周到。

而享盡了「齊人之福」的宗澤，其浪子本性也都昭然若揭了。一邊是漂亮甜美的張沙沙，一邊是溫婉賢惠的陳嘉桓，兩邊甚至都不知道對方的存在。

而對於黃浪子來說，他既沒有給出承諾，也沒有要負責任的打算，只是單純地覺得：「我沒有要求你做牛做馬、忠心耿耿地待在我身邊啊，我也肯定不會公開我們的關係，你要願意做我背後的女人，我也會和你像男女朋友一樣相處，你要不願意離開，我也會祝福你開心快樂！」

這番浪子心聲，在我們局外人看來當然是渣男本渣的言論。但是對於陷入感情的陳嘉桓而言，可能還是抱著憧憬地希冀：「沒關係，我可以等你，我相信我能感化你，只要我忍辱負重，只要我有耐心，你一定會看到我的好，你會習慣我在你身邊，到時候我就能像三嫂一樣，你也會三哥一樣收起花心，為了我放棄那些狂蜂浪蝶，鶯鶯燕燕。」

雖然嘉桓妹妹勇氣可嘉，但是說到底宗澤不是三哥，她也不是三嫂。三哥和三嫂算得上是金童玉女，勢均力敵。

而嘉桓這幾年，事業不振，星途堪憂，名氣寥寥。只是一心撲在感情上，家庭上，和宗澤談戀愛似乎也成了她全部生活。殊不知在宗澤看來，這也不過是生活中的餐後甜點，錦上添花而已。

所以，得不到承諾和保障的感情只會迎來一次次的挑釁。張沙沙的出現並不是偶然，也不是第一次。在她之前，還有一位叫孫驍驍的內地女明星，當時她和黃宗澤一起拍《少年神探狄仁傑》。

兩個人朝夕相對，你來我往之下就成了「好哥哥」和「乾妹妹」。不時會在網上兄妹相稱互動，黃宗澤還會帶上她見自己的媽。傳聞黃媽媽還特別喜歡這個「乾女兒」，甚至想撮合她和宗澤。

看到這裡，真不能想像作為陳嘉桓會是何種心情。看著自己所謂的男朋友，今天帶著這個「乾妹妹」回家吃飯見媽媽，明天帶著這個「好朋友」去片場探班見兄弟。還會是一副理直氣壯，名正言順的樣子。

你要鬧脾氣，你要不舒服，那就是你不懂事，你想多了。所以，不懂事的陳嘉桓也被宗澤放逐出了黃家豪宅，無可奈何只能住到酒店去。

而儘管這樣，我想嘉桓妹的內心還是無法放棄宗澤。畢竟對於她來說，牢牢抓住這個男人就等於抓住了人生的轉機，事業的轉機。於是她採取「忍字訣」，以為只要她能等，她能忍，她就能等到浪子回頭。

然而，其實我們都能想像，浪子之所以遲遲不肯給出承諾，給出名分。不就是因為他還沒有玩夠，也還沒遇到所謂的真命天女嗎？只要那個「對的人」出現，浪子也被吃得死死，也會恨不得馬上結婚，把對方留在身邊。

所以，也許當年的胡杏兒也是有過「我能等、我能忍」的想法，才會一等等了七八年。但最後她也是看清了花心浪子的本質，才會毅然決然地含著淚喊分手，因為及時止損總比浪費年華來得划算。

所以，不僅僅是陳嘉桓，也想奉勸各位在和渣男浪子周旋的女生們。與其費盡心思地百般討好，不如跳出「浪子圈套」去尋找真正值得的人。
