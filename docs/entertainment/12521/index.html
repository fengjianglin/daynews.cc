娛樂圈六個“禁忌”, 不要和劉德華演情侶，不要和張耀揚同桌吃飯

2019-12-12 2,271條評論 3,378次閱讀 529人點贊

明星們在工作時，往往不經意間就會流露出一些有趣好玩的事情。有時候是某個影視劇的橋段，有時候是一兩句對白，都是一些無心插柳之舉，但卻因此而風靡，這就很有趣了。下面就盤點了娛樂圈的六個「禁忌」，快來看看都有誰哦！

一.不要和張學友演對手戲

張學友唱歌厲害是眾人皆知的，但他其實演戲也是一流，雖然還缺少一個影帝為其正名，但他的演技實力卻是毋庸置疑。他演過很多電影，和很多明星都有過合作，這其中不乏劉德華、李連杰、周星馳、黎明這樣的大腕。他的演技自成一派、不按常理出牌，往往令對手無所適從，因此即使他是配角，也仍然搶光了所有的風頭。

最為人熟知的，還是《旺角卡門》了，他的表情包直到現在仍然在網路上風靡一時。

二.不要和劉德華演情侶

早期的劉德華在電影中總是以爛仔小混混的形象示人，通常他演的這種角色，結局都很悲催，也連累了他在片中的女朋友。兩部《至尊無上》《天與地》《龍在江湖》《獄中龍》《殺手的童話》等等，關之琳、吳倩蓮、陳少霞、黎姿、袁詠儀，基本上和他演情侶的都很悲慘，不是被亂槍掃死，就是被亂棍打死，一個比一個慘，這才有了不要和劉德華演情侶的說法。

三.不要和伍佰喝酒

伍佰是台灣省著名的搖滾歌手，他的《挪威森林》《浪人情歌》《痛哭的人》等都曾是廣為傳唱的經典，直到今天都在KTV擁有很高的點擊率。它曾唱過一首歌《突然的自我》，裡面有句對白「來來來，喝完這杯，還有三杯」。因此很多人說不要和伍佰喝酒，因為他總是勸酒，一杯接一杯，恐怕沒兩下，就醉倒了。

四.不要請劉德華當演唱會嘉賓

劉德華是四大天王之一，不僅在影壇，在歌壇他也是屢有建樹。出道多年，他的經典歌曲數不勝數，傳唱程度之廣，流傳曲目之多，絕對無人能及。

正因為他的名氣太過響亮，很多歌手辦演唱會都以邀請他做嘉賓為榮。但往往他一登場，就出盡了所有風頭，一首《我恨我痴心》嗨爆全場，足以喧賓奪主，活生生的將別人的演唱會變成了自己的主場，這就很尷尬了。

五.不要和黃日華打架

迄今為止，《天龍八部》已被改編了N多個版本，但要論最經典的，無疑非97版莫屬。黃日華的喬峰慷慨激昂、豪邁大氣，從某種程度上，滿足了人們對喬幫主的所有幻想。這版喬峰的造型也與眾不同，全程戴著一頂氈帽，有人調侃裡面藏著一個音響。

伴著雄渾激昂的音樂，每次出場都能力挽狂瀾，一出手就是十八條小金龍。以至於網上一直流傳著「欲敗喬峰，必先毀其音響，拔其插頭」的段子，這才有了不要和黃日華打架的說法，畢竟自帶音響的男人惹不起。

六.不要和張耀揚同桌吃飯

相信看過古惑仔的，對烏鴉應該印象深刻，憑著殺馬特的髮型、不羈的外表、略帶神經質的氣質，其扮演者張耀揚成就了這個最為經典的反派。尤其是烏鴉掀桌子的經典鏡頭，更是在網路上風靡一時，也因此才有了不要和張耀揚同桌吃飯的老梗，因為他動不動就掀桌子，和他吃飯總是吃不飽。

當然，這些也純屬娛樂好玩而已，只為博大家一笑，這也間接的證明了這些明星受歡迎的程度，你們又還記得哪些好玩有趣的段子呢？快來分享一下吧！
