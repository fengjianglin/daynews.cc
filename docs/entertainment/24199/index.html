金晨角色被換成楊采鈺，比她更慘的劉亦菲，所有資源都被截胡了

2019-12-19 260條評論 2,492次閱讀 1262人點贊

今天，去年播出大火的《大江大河》續作《大江大河2》，公布女主「梁思申」人選為出演過《芳華》的女演員楊采鈺。

EXO ME？

如果沒記錯，之前聽說跟王凱搭檔女主版本是金晨啊！

早在第一部未播之前，劇方就放出了成年梁思申的劇照，雖然金晨在第一部中並未出場，但很多觀眾都認定了她是第二部的女主角。

金晨也是命苦，變故來得太快，就像龍捲風。

當然，命苦的人肯定不止金小花一位，比她更大的咖劉亦菲也遭遇過。

今年3月，籌備多年的好萊塢大片《重力反轉》開了發布會，女主角定了楊采鈺。

可是這部電影之前一直對外宣布的女主角，是劉亦菲啊。

結果到了開拍的時候突然換人，這就很微妙了。

無獨有偶，之前Dior官宣了楊采鈺「香氛世家大使」的title，還一起去了巴黎時裝周。

無獨有偶，仔細觀察就知道，不聲不響的，楊采鈺就把好萊塢大製作和藍血大牌，一口氣都拿到手了。

要知道前幾年和Dior密切互動的，可是劉亦菲啊。

一直到2017年，劉亦菲還在為Dior站台，去了時裝周。

結果僅僅一個月後，Dior就官宣了Angelababy為中國區品牌大使了。

雖然Dior也曾給過劉亦菲一個花蜜系列的宣傳大使稱號，但當時還是有很多網友為劉亦菲打抱不平。

劉亦菲本人也在那之後，再也沒穿過Dior的衣服了。

沒想到，僅僅一年的時間，楊采鈺就以一個影視新人的身份，得到了Dior的青睞，官博宣傳次數，可比當年的劉亦菲多多了。

時裝周的時候，還直接把官博都交給楊采鈺了。

之前，我們也寫過楊采鈺和劉亦菲之間，盤根錯節的過往。

簡單來說就是楊采鈺的現男友，是劉亦菲的教父，所以坊間也一直有說法是，楊采鈺現在的資源都是劉亦菲的。

而且兩人形象氣質，確實乍一看是像的。

之前辛苦經營時尚資源的劉亦菲，沒想到現在被「小劉亦菲」給撿了個漏，也是很有趣了。

一直以來，楊采鈺和劉亦菲就因為外形和生活上的相似，被各種拿來比較。

就連楊采鈺之前上《天天向上》公開自己的學歷，還意外踢爆了劉亦菲疑似學歷造假的新聞。

楊采鈺在節目里介紹了自己哥大研究生的背景，還順帶科普了下這專業有多牛逼。

劉亦菲粉絲不服，將早年劉亦菲說自己被耶魯錄取，但因為要拍戲沒去上的事迹拿來比。

沒想到意外被較真的網友們查到，耶魯無論任何系，都沒有錄取過劉亦菲……

這麼一看，早早曬出哥倫比亞大學offer的楊采鈺，在塑造學霸形象這一局，確實贏了不少。

除了都走高知女神路線，楊采鈺私下裡的畫風，也和劉亦菲挺像的，走清淡文藝氣質路線。

微博里隨便翻翻，都是又高級又好看的美照。

不過往前翻翻楊采鈺的照片，發現她的臉和現在還是有一點變化的。

楊采鈺是北電2011級的學生，和周冬雨、古力娜扎是一個班的同學。

這張畢業照上能明顯看出，當時的楊采鈺和現在這個氣質脫俗的美少女，還是有不小區別的。

2015年陳金飛與《芳華》女主角楊采鈺電梯熱吻戀情曝光。

2017年7月，楊采鈺發布並置頂兩人合照，承認戀情。

當時還借用《這個殺手不太冷》裡面的經典台詞——「里昂，看，愛是無法阻擋的」。

兩人相差30歲的戀情還引起不小轟動和爭議，楊采鈺還在評論區怒懟網友：「從戀愛開始，我就從未覺得這和其他感情有任何不同。」

今天又添了一記猛料：知名博主爆料楊采鈺已經和劉亦菲乾爹陳金飛領證結婚。

後有媒體聯繫了陳金飛求證此事，陳金飛並未否認，只是表示：「不需要回應這個，這只是我們生活上的事情」。

看樣子，戀情確實是修成正果了。

有網友擔心，「劉亦菲義父娶了一個跟自己長得這麼像的人會不會感到尷尬」，大家覺得呢？
