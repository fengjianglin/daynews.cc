6歲嗯哼被同學起外號，卻和杜江當年的一模一樣，這也能繼承？

2019-12-18 1,212條評論 2,238次閱讀 1012人點贊

6歲嗯哼被同學起外號，卻和杜江當年的一模一樣，這也能繼承？

說起近段時間，娛樂圈內最火的明星家庭，當屬杜江霍思燕一家了，杜江和霍思燕兩人參加的《妻子3》這檔綜藝，可謂是讓很多觀眾更加了解這對甜蜜的明星夫妻，和節目中其他幾組家庭有著不小的差距，大家看到霍思燕被寵成公主的樣子都有些羨慕了！沒想到近日，一家三口又參加了一檔節目的錄製，這回在節目中最搶鏡的當屬才剛剛6歲的嗯哼，被網友們稱為「梗王」！

在節目中，嗯哼和主持人的互動逗笑了在場的觀眾，在主持人問到：你覺得爸爸媽媽應該打小孩嗎？嗯哼回答：不應該！主持人接著問道嗯哼：將來你有孩子，你會打他嗎？嗯哼下一秒的回答真的是讓網友哈哈大笑。直言：我不會有孩子的，因為我是男生，不會生孩子！這呆萌的腦洞真的是逗笑了小編，就連杜江和霍思燕聽到兒子的回答都撲哧的笑出聲來！

之後6歲的嗯哼還在節目中自曝了校園的一些不愉快經歷，讓網友們聽後有些同情，嗯哼雖然是星二代身份，但是在校園內也和其他同學一樣，沒有受到特殊的待遇，也會像其他同學一樣，有著被別人起外號的經歷，對於這一點想必每個上學的小夥伴都有同樣的經歷，只是有些是可以接受的，有些卻讓自己覺得非常的苦惱，就像嗯哼被同學叫「肚臍眼」就不是很開心，明顯對於這個外號並不接受！

還在節目中坦言自己不喜歡這樣，但是其他同學還是這樣的開玩笑，嗯哼的表情也是很無奈的，而對於兒子的這一稱號，身為家長的霍思燕和杜江也並不知情，直言也是在這次節目中才知道這一情況的，不過老媽為了緩解兒子的情緒，故意將話題引到了杜江的身上，問他小時候的外號叫什麼！

沒想到這一問，還問出了一個「笑話」來，杜江坦言自己上學時的外號也叫「肚臍眼」，妻子聽到後都露出驚訝的表情，這時嗯哼呆萌的說道：他也姓杜！可能在孩子的心裡一個姓的人都是一個外號吧，真的是太可愛了！但是網友們聽到後真的是笑出了豬叫聲，沒想到嗯哼不僅繼承了爸爸優秀的基因，還繼承了爸爸的外號，真的是太歡樂了！想必大家也是第一次聽到這樣有趣的事情吧，大家喜歡杜江他們一家嗎？
