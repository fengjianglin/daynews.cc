讓人恨不起來的反派，潤玉、岳綺羅上榜！看到圖四：讓我做你的糖

2019-12-22 229條評論 1,338次閱讀 308人點贊

在影視劇中，總有正派和反派之分，好像只有這樣才能增加劇情的生動性，讓關注更喜歡看。而劇中的反派總是讓我們恨得牙痒痒，不過還是有很多反派是讓我們恨不起來，反而很心疼的。今天我們就一起來看看影視劇中讓人恨不起來的反派吧！

潤玉

這是《香蜜》中的小魚仙倌，也是很多人心目中的白月光吧。前期的潤玉還是很暖心的，不過後期因為對錦覓的愛而不得，和得知生母的遭遇之後，潤玉就黑化了。黑化之後的潤玉雖然壞事做盡，但還是很讓人心疼的，完全討厭不起來啊。

岳綺羅

《無心法師》中最受歡迎的一個角色了，岳綺羅雖然外表看上去只有十七八歲，但她的靈魂卻經歷了幾世了。雖然在劇中嶽綺羅經常和月牙、無心作對，但觀眾對她就是討厭不起來，反而覺得她很可愛呢。

張顯宗

這也是《無心法師》中的一個角色，張顯宗所有的壞都是因為岳綺羅。為了岳綺羅他可以做任何事情，不管對錯，也不管好壞。不過最後張顯宗卻為了救岳綺羅而死，看到岳綺羅最後的一句：張顯宗，我牙疼。直接淚奔啊！

薛洋

最後一個就是《魔道祖師》中的薛洋，他的壞並不是天生的，而是早期的各種磨難讓他變成這樣的。而薛洋很有個很大反差萌的愛好，就是嗜甜如命，就算最後左手被砍下，都沒有鬆開手中的那顆糖，很想對薛洋說：讓我做你的糖吧！

那你還知道劇中哪些恨不起來的反派呢？
