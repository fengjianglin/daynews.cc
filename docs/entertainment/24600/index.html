從開年檔到春節檔，“劇集大戰”即將拉開

2019-12-23 2,093條評論 4,491次閱讀 2703人點贊

開年檔到春節檔橫跨元旦、寒假、春節等多個重要節點（時段），也成為視頻平台爭奪觀眾注意力的重要時段，視頻平台在劇集編排上也更加靈活。

來源：傳媒內參-傳媒大眼綜合

開年檔到春節檔向來都是各大平台排播的重頭戲，隨著2020年臨近，各大平台也在劇集編排上下大功夫，一批重點劇目也有望在該期間播出：

愛奇藝：《唐人街探案》、《新絕代雙驕》、《愛情公寓5》、《兩世歡》、《大主宰》、《獵心者》、《看不清的真相》、《鬢邊不是海棠紅》、《誰都渴望遇見你》；

騰訊視頻：《蓬萊間》、《長安少年行》、《大秦帝國之天下》、《三生三世枕上書》、《劉老根3》、《古董局中局2》、《你是我的命中注定》；

優酷：《冰糖燉雪梨》、《淑女飄飄拳》、《鄉村愛情12》、《夢醫甜妻》、《人間煙火花小廚》；

芒果TV：《錦衣之下》、《三千鴉殺》、《拜託拜託請你愛我》、《時光與你都很甜》、《下一站是幸福》、《我不是購物狂》。

開年檔到春節檔橫跨元旦、寒假、春節等多個重要節點（時段），也成為視頻平台爭奪觀眾注意力的重要時段，不同於衛視主打懷舊、合家歡，視頻平台在劇集編排上則更加靈活，可以預見，從開年檔到春節檔，一場「熒屏大戰」也即將拉開！

隨著國家廣電總局電視劇司部署的「優秀電視劇百日展播」工作的結束，古裝劇、偶像劇正陸續回歸大眾視野。從現有劇目來看，除了《新絕代雙驕》《兩世歡》《大主宰》《錦衣之下》《三千鴉殺》等一批古裝劇再次來襲，《愛情公寓5》《劉老根3》等主打合家歡的劇目也成為編排新亮點。傳媒內參——傳媒大眼梳理十部有望在開年檔到春節檔播出的劇目信息：

《大主宰》

導演:張萌 / 澄豐

編劇: 天蠶土豆 / 白一驄

主演: 王源 / 歐陽娜娜 / 駱明劼 / 張誠航 / 夏楠

類型: 劇情 / 動作 / 奇幻 / 冒險

劇目看點：《大主宰》是由張萌、澄豐執導，王源、歐陽娜娜、 駱明劼領銜主演的古裝青春熱血劇。

該劇根據天蠶土豆的同名玄幻小說改編，講述了資質卓絕的少年牧塵從草根少年成長為撼動大陸、為善除惡的大英雄，同時收穫了友情和愛情的故事 。

《愛情公寓5》

導演: 韋正

編劇: 韋正 / 鄒傑

主演:婁藝瀟 / 孫藝洲 / 李佳航 / 李金銘 / 陳赫

類型:喜劇 / 愛情

劇目看點：《愛情公寓5》是由韋正執導，韋正、鄒傑編劇，婁藝瀟、孫藝洲、李佳航、李金銘等主演，陳赫特別出演，趙文琪、劉萌萌、榕榕、三浦研一等友情出演的都市輕喜劇 。

劇中老住客們有人領證結婚，還有神秘的新住客入住。鐵打的公寓，流水的愛情，新的浪漫故事在這裡不斷上演。

《劉老根3》

導演:趙本山

主演:趙本山 / 范偉 / 李靜 / 閆學晶

類型: 喜劇

劇目看點：《劉老根3》是由趙本山、李春嘯聯合執導 ，何慶魁擔任編劇 ，趙本山、范偉領銜主演 ，閆學晶、李靜、王娟、王小寶、張小飛、孫小飛等聯合主演的現代鄉村題材電視連續劇。

該劇主要講述了因龍泉山莊經營不善，在家頤養天年的劉老根決定時隔15年後重返山莊，再次主持大局，將山莊重整旗鼓，從而發生的一系列啼笑皆非的故事 。

《淑女飄飄拳 》

導演:羅安得

編劇:天衣有風

主演: 畢雯珺 / 孫千 / 丁澤仁 / 賈征宇 / 張馨月

類型: 愛情

劇目看點：《淑女飄飄拳》是由羅安得執導，畢雯珺、孫千、丁澤仁、賈征宇等主演的新古風青偶劇 。

《淑女飄飄拳》講述的是「尚香淑女」風飄飄（孫千飾）在一次意外中，出手救下太極「奇才」衛楚（畢雯珺飾），伴隨著一系列有甜有燃有蘇有虐的劇情，一場青春鮮甜的純愛故事由此展開。

《時光與你都很甜》

導演: 西島

編劇: 禾白三少

主演:呂小雨 / 孫澤源 / 王晴 / 林昕宜 / 孫鎮洋

類型: 劇情

劇目看點：該劇根據藍色獅的同名小說改編，講述了奸臣當道的明朝嘉靖末年，錦衣衛陸繹奉命調查揚州修河款失蹤案，得到六扇門女捕快袁今夏協助，二人就此結為甜蜜冤家，卻意外捲入重重陰謀的故事 。

《時光與你都很甜》是由芒果TV、晨星盛世聯合出品，呂小雨、孫澤源主演的青春校園劇 ，該劇講述「95後」女孩林星辰和男生陸一白及身邊同學在高中時期一起成長、一起追夢、相互治癒的故事。

《冰糖燉雪梨》

導演: 朱銳斌

編劇: 馬佳

主演: 吳倩 / 張新成 / 周曆傑 / 楚月 / 曹博

類型: 劇情 / 運動

劇目看點：《冰糖燉雪梨》是由朱銳斌執導，吳倩、張新成、周曆傑、楚月、曹博、何宣林、魏天浩、韓玖諾、秦天宇主演的青春愛情競技成長劇。

該劇根據酒小七同名小說改編，講述了「速滑少女」棠雪在大學校園裡重逢「冰球男神」黎語冰，兩人在追逐冰上夢想的過程中，收穫真摯愛情的故事 。

《錦衣之下》

導演: 尹濤

主演: 任嘉倫 / 譚松韻 / 韓棟 / 葉青 / 姚奕辰

類型:劇情 / 古裝

劇目看點：《錦衣之下》是由尹濤執導，任嘉倫、譚松韻、韓棟、葉青領銜主演的古裝愛情懸疑劇 。

該劇根據藍色獅的同名小說改編，講述了奸臣當道的明朝嘉靖末年，錦衣衛陸繹奉命調查揚州修河款失蹤案，得到六扇門女捕快袁今夏協助，二人就此結為甜蜜冤家，卻意外捲入重重陰謀的故事 。

《三千鴉殺》

導演: 回宇

編劇: 趙天佑

主演:鄭業成 / 趙露思 / 王萌黎 / 劉怡潼 / 代雲帆

類型: 古裝

劇目看點：《三千鴉殺》是由回宇執導，鄭業成、趙露思、劉怡潼、王萌黎等主演的仙俠玄幻劇。

該劇根據十四郎同名暢銷小說改編，講述了燕國的熱血公主覃川為復國，改名字、練仙法，被傅九雲守候千年更捨身成全的愛情故事 。

《兩世歡》

導演:余翠華 / 劉鎮明

主演:於朦朧 / 陳鈺琪 / 劉帥

類型:愛情 / 古裝

劇目看點：《兩世歡》是由北京愛奇藝科技有限公司出品的輕懸疑古裝言情探案劇，余翠華執導，於朦朧、陳鈺琪領銜主演。

該劇根據「寂月皎皎」的同名小說改編，講述了因仇恨不能相守的一對青梅竹馬，各換一重身份，攜手連破大案，共保一方太平的故事。

《長安少年行》

導演: 周家文

編劇: 熊誠 / 宋晉川 / 郭琪 / 張漢 / 孟媛

主演: 王玉雯 / 吳希澤 / 劉奕暢 / 謝彬彬 / 漆培鑫

類型:愛情 / 古裝

劇目看點：《長安少年行》是由周家文執導，王玉雯、吳希澤、劉奕暢、謝彬彬、漆培鑫領銜主演，李博洋、金鐘、溫晴、謝詩煜等主演的古裝愛情劇。

該劇講述了市井廚娘沈依依為報答好友相知之恩，替嫁長安履行婚約，誤打誤撞進入尚藝館，開啟了一段破奇案、打貪官、尋真愛的青春之旅的故事 。
