參演《一起同過窗》，顧一心扮演者桑硯，這幾年為何銷聲匿跡？

2019-12-21 2,512條評論 3,976次閱讀 858人點贊

大家好，我是慕白。相信很多人都看過這部豆瓣評分高達9.0評分的青春喜劇《一起同過窗》。距離這部青春劇已經過去了3年多了。劇中的角色深入人心，也帶火了劇中多個角色。劇中的顧一心和畢十三在一起的呼聲也是越來越高。這幾年她在上大學，不知道第三部她是否會回歸呢，顧一心的扮演者叫做桑硯，是一個很漂亮的女孩。

桑硯，1995年7月2日出生於浙江杭州，今年24歲。就讀於北京電影學院，影視表演專業。演員兼模特。她曾經參演過《最美韓國女孩北京遇宅男》《梁祝：跨時代幻想》《一起同過窗》三部影視劇。

現在的桑硯在上大學，生活嘛，肯定是多姿多彩的。今年4月份去了日本東京~18年9月份去了西藏，去過多次西藏。是一個喜歡旅遊，大海的女孩呢。最近的她除了在大學念書，旅遊之外，還會經常在微博曬一些古裝造型。看一看這個一身紅衣，騎馬女孩回眸的瞬間，是不是喜歡上了呢，哈哈。

至於很多朋友擔心桑硯好久沒拍戲了啊，是不是不再拍戲了，因為從17年的《一起同過窗》以後，桑硯就沒有作品再出來過了。這個請朋友們不要擔心，因為桑硯的微博明確表示會繼續拍戲，那可是愛好。也許是因為學業太忙了吧，請朋友耐心等待，期待桑硯的新作品，當然，慕白和大家一樣，更希望桑硯能參演《一起同過窗》第三部。特別想看到你和畢十三在一起。感覺會很暖的哦。聽說《一起同過窗》第三部已經在寫了。最後祝桑硯在演戲生涯越來越好。期待你的作品。
