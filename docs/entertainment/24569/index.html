《封神三部曲》黃渤陳坤兩位仙人率先登場，片中美術設計和建築引關注

2019-12-23 391條評論 1,904次閱讀 127人點贊

由烏爾善執導，黃渤、費翔等主演的《封神三部曲》日前發布先導預告片。該預告片時長為30秒，共呈現14個影片中的精彩片段，眾多經典角色悉數登場，揭開了這個「三部曲」系列的神秘面紗。此外，影片中的美術設計和建築特點，也受到了廣大影迷們的關注。

專門開設「封神演藝訓練營」

先導預告片中，由黃渤飾演的姜子牙以白髮老者的形象率先亮相，手持封神榜立於殿上，神情凝重面有疑色；由陳坤飾演的元始天尊緊隨其後，一語不發卻肅穆威嚴。預告片中兩位仙人相繼登場，預示「昆崙山子牙下山」這一經典情節由此展開。

商王殷壽（費翔飾）則以不同形象亮相三次。殷壽的第三次亮相則是與妲己共浴，赤裸的紂王與只露背影的妲己含情對望，引發無盡想像。而另一重要女性角色姜王后（袁泉飾），賢良端莊、宅心仁厚，勇敢勸諫紂王卻遭狐妖陷害慘死。預告中，袁泉手執利刃求賜一死的場景，同是深藏悲慟與絕望。

西伯侯姬昌（李雪健飾）此次出場，則演繹出截然不同的王者風範。在荒郊野嶺中懷抱嬰兒的神秘姿態，正印證了《封神演義》第十回中「姬伯燕山收雷震」的原著典故。此外，夏雨飾演的申公豹，由姬發（於適飾）、殷郊（海亮飾）領銜的「質子旅」首次以齊整陣容躍入沙場，成為貫穿預告片的一條重要線索。

烏爾善表示，早在《封神三部曲》籌備之初，自己就已經下定決心，不選用流量演員，而是開設「封神演藝訓練營」，從無到有潛心打造全新的「質子旅」角色。為此劇組自2017年2月開始在全球華人範圍內進行了大規模的演員招募。從15000人中甄選出1400人由導演親自面試，最終篩選出二十多位年輕演員進入「封神演藝訓練營」。最後有六個重要的角色都是由新人演出。

電影美學源自青銅器美學元素

此次《封神三部曲》預告片中選取的場景，有商朝王宮中的「龍德殿」，作為朝歌城中象徵著至上王權的最高等級建築，「龍德殿」無疑是該片展現創作美學概念的集大成所在。此外還有代表紂王張狂個性的重要場景——鹿台。由七層建築組成，於各處細節盡顯奢華嫵媚。

在《封神演義》原著中，「封神榜」從未以實體的面貌呈現出來，而在預告片中，它成了姜子牙手中一件白如脂玉的寶物，圍繞這件寶物會衍生出怎樣的故事，還是一個懸念。

導演烏爾善在採訪中解釋了這個系列在美學上的特點。他把《封神三部曲》定義為神話史詩電影，這樣一個電影類型，一定要把想像和神話作為一個主體設計，不能完全走純歷史電影的路線。《封神三部曲》中的美學風格可以分為三個部分：一部分元素來自於元明時期的道教藝術，尤其是道教中的水陸畫藝術；第二個元素來自於商周時期的青銅器、青銅畫，尤其是片中「龍德殿」的設計，就參考了商朝的青銅器美學。烏爾善導演認為，商朝的文化比較「噬血」，因為它是當時部落聯盟的文化，很少出現植物花卉，圖案紋飾幾乎全是饕餮文、磐石文，看起來有點恐怖猙獰，但很貼合《封神三部曲》中的氛圍；還有一部分是來自於宋代的山水畫。基本上電影中的服裝、道具、建築、布景等都是根據參考這三個元素製作而成的。

故事背後的當代心理學共性

導演烏爾善始終強調「傳承經典，守正創新」的精神信念。導演與編劇團隊曾用五年時間打磨劇本，故事改編主要基於明代神魔小說《封神演義》及南宋話本《武王伐紂平話》的內容，力圖講述一場人、仙、妖之間曠日持久的神話戰爭。

烏爾善認為，《封神演義》小說裡面有很多經典劇情、經典的角色和經典的設定，一定要保留，「這裡面涵蓋了整個中國傳統文化里的標誌性人格，在人物原型方面非常有代表作用」。

烏爾善導演將《封神三部曲》定義為神話史詩電影。《封神演義》在15世紀形成，綜合了很多民間傳說，還有中國的一部分歷史，創作出一個道教的神話系統,比較市民化，可以把《封神演義》理解成是民間藝人說書的文本。因此，在改編的時候，最主要的是從當代的角度重新理解這些故事，找到它的心理學共性，「這可能會是一個新的角度，也是能夠跟其他文化交流的一種方式」。

《封神三部曲》是華語電影史上首部採取三部連拍模式進行製作的電影作品，預計2020年、2021年、2022年連續三年上映，第一部將於2020年暑期登陸大銀幕。
