這5部末日大片全程高能，看似很硬的外殼下卻有著飽滿的情感

天天要聞
3 天 ago

關於末日題材的影片，一直是廣大影迷的心中所愛，今天，我就跟大家分享5部全程高能的末日大片，在這幾部電影中，一邊是末日來臨的恐懼，一邊是通俗的情感戲，無論外表多硬多科幻，最核心的仍是親情與愛，看似很硬的外殼下卻有著飽滿的情感。

1、《未來水世界》

這是一部商業巨制，劇情、角色、視效、音效等都極其出色，電影於1995年7月在美國上映後，立刻就引起了巨大的轟動。它是世界電影史上耗資最大的影片之一（在泰坦尼克號之前是影史之最）。影片的前期製作費用高達二億美元。《未來水世界》在2007年時依然位居洛杉磯時報的「影史十大最難拍電影」之列，從籌拍、開機，到後期的製作、上映，這片都堪稱磕磕碰碰、多災多難，投入的成本一加再加，堪稱影史奇觀。

雖然有些評論家認為影片的內容缺乏深度，但影片卻以其華麗的場面，巨大的布景和眾多的特技效果令觀眾們驚嘆不已。而幾位影星的出色表演也令人大飽眼福。影片猶如一個成年人的華麗童話，把觀眾們引入到一個奇妙的電影世界之中。

未來水世界是一部當初被嚴重低估的電影，無論從創意還是視覺效果，它都讓我感到過目難忘，甚至足以媲美另一部經典末日片《瘋狂的麥克斯4》，好在它如今的豆瓣評分正逐年升高，好片子，終究不會被埋沒！

2、《撕裂的末日》

世界核戰爭後，政府給每個人注射一種麻痹情感神經的藥物，使人們封閉感情，變成行屍走肉。 約翰·普雷斯頓是政府軍事部門高級官員。他摧毀一切藝術品、工藝品，並將一切抗令不拋棄自己感情的人殺死，同事艾洛·帕特傑因藏有一本違禁書籍也未曾倖免。一次行動中，約翰遇到瑪麗·歐布萊恩。她使得約翰開始斷絕藥物，試圖體會有感情的生活究竟是一種什麼樣的滋味。

不得不說的一部經典，炫酷的場面，槍斗術的起源，簡單而深刻的反烏托邦世界觀。強制的收斂和抑制的爆發，人的情感和自由意志豈能被抹殺？貝爾顏值太高，全程開掛，表示還沒有看過這麼精彩的槍斗術，太炫酷簡直帥炸。結尾乾脆利落，絲毫沒有拖節奏。

3、《世界末日》

與其說這是部災難片，倒不如說它是一部讓你笑到噴飯的惡搞喜劇。用豆瓣網友的評價就是：連我七十歲的大爺都笑抽了。影片一開始簡直讓人輕鬆到極致，貼心的布置、嗨到爆的party，更佩服的是那麼多明星的客串。但180度大轉彎 ，噩夢來得太突然，輕鬆的喜劇片好似一秒鐘變災難片，倒是其中的笑料百出還在提醒著我這是喜劇片。 但這不是不含養分的無厘頭搞笑，在歡樂中或許應該悟出些什麼，我們現在有舒服的房子，潮流遍地，沒準還能和明星拍拖，但刨去自我的滿足是否想過共同生活的家園正一天天的垮塌下去，更喜歡看外國電影的原因正是因為他可以在滿足視聽享受的同時還會帶來更深的東西，倘若有一天世界末日真的到來，我還真要謝謝那些外國影片教會我的求生招數。

最近看了很多災難片，在想難道只有電影人是關注地球安危的？也許他們只是抓住賣點製造噱頭罷了。可是有時真該思考一下，我們的家園還禁得住我們破壞多久？電影最終還是善有善報惡有惡報，只要你肯懺悔，虔誠的面對身邊的朋友，善良的對待所有人，什麼時候都不晚。最後的超級男孩是我最感動的地方，我不知道電影用《everybody》做片尾的用意何在，但超級男孩陪伴了我長大，好結局加上喜愛的音樂，這部片子確實讓我喜愛。

4、《星際穿越》

影片發生在不遠的未來，地球氣候已經不適合糧食生長，水資源枯竭，饑荒肆掠，人類面臨著滅絕的威脅。這時科學家們發現了一個神秘的「時空裂口」，通過它可以到外太空尋找延續生命希望的機會。一個探險小組穿越到太陽系之外，他們的目標是找到一顆適合人類移民的星球。在這艘名叫做「Endurance」的飛船上，探險隊員著面臨著前所未有，人類思想前所未及的巨大挑戰。然而，通過蟲洞的時候，他們發現飛船上的一個小時相當於地球上的七年時間，即使探險小組的任務能夠完成，他們能提供的救贖對於對地球上活著的人來說已經是太晚。飛行員庫珀必須在與自己的兒女重逢以及拯救人類的未來之間做出抉擇。

本片獲得奧斯卡金像獎(2015年第87屆)，這是諾蘭所有作品中時長最長的一部，總共169分鐘，拍攝成本為1.65億美元，這意味著裡面每一分鐘的鏡頭造價為97.6萬美元。本片使用35mm膠片和70mm膠片IMAX兩種格式拍攝，並且以膠片和數字兩種格式分別在影院上映，是最後一部使用70mm膠片拍攝的IMAX商業電影。

整場基本是屏住呼吸，瞠目結舌，無話可說。就連名導昆汀看完本片後也是拚命點贊，稱讚諾蘭帶來了非常不一樣的東西，「那種人們通常會想要從俄羅斯導演安德烈·塔科夫斯基和美國導演泰倫斯·馬力克影片能夠找到的東西，而不是在一部科幻影片中。」

我還記得在電影院看這部電影時，四周響起的啜泣聲，沒有人不為之動容，而那句「宇宙的終極答案，是愛」現在回想起來，心中也仍有無限感動。

5、《我是傳奇》

影片改編自美國20世紀最偉大的作家理查德·馬瑟森(Richard Matheson)創作的同名小說……作為黑人演員中最有影響力的面孔，威爾·史密斯出演過一系列既賣作又受到好評的科幻動作電影，例如《我，機器人》、《獨立日》、《黑衣人》和今年的《雙子殺手》等等，所以對於他來說，這類影片絕對是他的「看家本領」，再加上他本人也是馬瑟森的忠實擁躉，貌似沒有比他更適全的演員人選了，史密斯說：「原著中包含了很多流派，恐怖和科幻也位列其中。當你成為全世界最後一個人類的時候，那種刺激的感覺源於心理，對我有著致命的吸引力，無論是從身體、情感還是精神上。故事中的主角羅伯特·奈維爾唯一的目的就是存活下去，這樣的狀態有可能創造出一個美麗的機會，來講述一下關於人類天性的世界觀。」

這部科幻動作驚悚片的導演則是弗朗西斯·勞倫斯，他的處女作就是那部同樣具有「末世情結」的《地獄神探》，勞倫斯說：「這類關於一個男人與世隔絕地生活以及在現代的城市環境中孤獨的生存的想法，讓我感到無比地著迷，所以才會產生用影像將這個故事表現出來的想法。《我是傳奇》中的精華部分，就在於這裡有一個要和整個世界進行對抗的男人，它雖然誕生於半個世紀前，可是到了現在仍然能夠激起人們無限的想像力。」

這部發行於1954年的小說，以現代人的觀點看，仍然會有一種非常複雜的感覺且極具有煽動性，已經被廣泛認為是邁入現代恐怖科幻領域的入門教材，不但影響了幾代人，還刺激了無數個爭相效仿的後來人，以及兩部電影改編作品：1964年的《地球上最後一人》，由文森特·普雷斯領銜；1971年的《最後一個人》，查爾登·海斯頓是這裡的主演……到了這部最新版本的影片，其編劇兼製片人阿齊瓦·高斯曼不但曾因《美麗心靈》而獲得了奧斯卡最佳改編劇本獎，他自己也是小說的作者理查德·馬瑟森一生的追隨者，所以對於高斯曼來說，能夠主筆將《我是傳奇》再度帶上大銀幕，確實是一件非比尋常的大事件，他表示：「馬瑟森對於我們這些為科幻故事神魂顛倒的怪胎來說，就好比是神。在嚴肅的科幻世界裡，有一個約定俗成的傳統，那就是利用角色性格的深層次根源，來釋放出強烈的主題–這一點和寓言故事很像，而這部影片也沒有脫離開這個潛規則。」

當地球陷入末日災難時，有一種比在災難中死亡更可怕，那就是：你身邊的人都死了，只有你活著，《我是傳奇》中的科學家羅伯特就面臨這樣的困境，致命的病毒不幸在全球範圍散布，人們死的死，瘋的瘋，而羅伯特卻因為有獨特的免疫力而活了下來，除了一隻狗作伴，他只能孤零零一人，此外，他還要抵抗那群發了瘋人類的攻擊。

孤獨，恐懼，絕望，沒有盡頭一般……威爾·史密斯奉獻出了一場驚人的「獨角秀」，值得一座奧斯卡小金人的嘉獎，作為觀眾的我，也彷彿被吸進了那種絕望至沒有盡頭的生活里。
