48歲男星自曝錄綜藝辛酸史：被逼與餓犬搶食，大雨中狂奔險遭雷劈

2019-12-21 567條評論 1,472次閱讀 131人點贊

近年來，大量綜藝節目的興起導致越來越多的藝人報名參加綜藝，一來可以在一定程度上提高自己的曝光率，二來面臨影視市場的寒冬，藝人不得不另謀出路。而各大製作組為了博取觀眾的眼球，往往會製作一些毫無下限，安全措施也不到位的項目來讓嘉賓完成。就算項目千奇百怪，藝人也會努力完成。近日，台灣藝人小鍾透露，之前的一檔節目中，被逼與餓了三天的餓犬搶食，險些喪命。

小鍾在《小明星大跟班》中自曝錄製綜藝時的辛酸史，他表示在之前的節目因為玩一個遊戲失敗了，所以要接受製作組的懲罰。

沒想到製作單位這時候牽來了一隻流口水的大狼狗。

並告知小鍾這隻狗已經三天沒吃飯了。

甚至在大狼狗的碗里放一個雞腿，讓小鍾在大狼狗面前搶它的食物。

小鍾當即表示太危險了，製作人還好心安慰他：你去拿的時候，它要去搶，它要咬到你的瞬間，我會叫狗主人把它拉開。雖然小鍾當時心裡不太相信製作人說的話，但是為了節目效果還是硬著頭皮上了。

由於狗主人的拉繩時間並不能算的十分精準，在小鍾伸手搶雞腿的時候手還是不可避免的被狼狗咬了一口。

對於這件意外的發生，製作人回應：和狗搶飯碗壓迫性才強嘛，重點是觀眾愛看。這番荒唐言論驚呆了在場的嘉賓。紛紛表示可能會鬧出人命。主持人吳宗憲也直呼：你跟一隻狼狗爭什麼？

小鍾作為台灣的脫口秀主持人，今年48歲。他在1997年出道，活躍在台灣各大綜藝平台。到現在也有20餘年的時間，但是據他描述這也不是第一次參加這麼危險的綜藝項目。

有次出外景時，被製作方要求大雨中狂奔，險遭雷劈。當時小鍾穿上超人裝在雨中奔跑。那天雨下的很大，小鍾剛剛跑出去，下一秒剛好一個落雷就打在他的腳邊，差點就發生意外。

綜藝節目千千萬，而節目組的最終目的也是為了收視率。為了達到目的，運用各種非常規手段，瘋狂和獵奇成為主軸。在一定程度上，觀眾和製作方都參與了這件事，雙方都需要反思。該如何營造一個健康的影視環境。
