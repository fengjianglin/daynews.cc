鄭爽跟男友分手後開始搞事業？與成龍馮小剛私下聚會，疑打入京圈

2019-12-19 391條評論 1,762次閱讀 502人點贊

12月18日，有知情人稱鄭爽與男友張恆已經分手，只等待兩人正式承認，目前，有媒體聯繫了兩方的團隊，暫時還沒有得到確切的回復。

不過鄭爽和張恆分手也不是第一次被傳出來了，從綜藝節目《女兒們的戀愛》里，大家能看出鄭爽和張恆之間的問題還是比較大的，他們的問題不解決，分手是必然的結局。

從張翰再到張恆，不得不說，鄭爽的每一段戀情都很轟轟烈烈，不過感情裡面的事情，冷暖自知，旁人沒有資格評論太多。

鄭爽可以說是很多人看著成長的，十幾歲的她就因為《一起來看流星雨》而走紅，雖然以現在的目光再來評價這部劇，很多人說太雷了，但誰的青春里沒有慕容雲海與楚雨蕁呢？

不過鄭爽這幾年的事業發展算不上好，以前她是90後花旦里獨一無二的流量花旦，這幾年，隨著周冬雨，楊紫，迪麗熱巴等人的走紅，鄭爽90後第一小花的位置早已經不保。

這次鄭爽被爆與張恆分手，她的不少粉絲都在替她開心，相比較看她談戀愛，網友們更希望她能認真搞事業，仔細想想，這樣子的人設才更帶感。

日前，還有人曬出了鄭爽與成龍，馮小剛，張國立等人私下聚會的照片，素顏的鄭爽在其中還是很漂亮的。

更令大家好奇的是鄭爽怎麼搭上這些知名前輩藝人，有人稱是因為趙寶剛導演很賞識鄭爽，而他正好是京圈的人，往後鄭爽的資源會越來越好。

這一說法並沒有得到認可，只是網友們的爆料而已，不過從照片能看出，如果鄭爽真的跟馮小剛他們合作，未來發展勢必不會差。

而鄭爽和趙寶剛的緣分還要從他們合作《青春斗》開始，雖說這部劇頗受爭議，趙寶剛對鄭爽還是很賞識的。

這同時帶來了另外一個問題，就是鄭爽的演技，儘管小小年紀就已經出道，第一部作品就是女主角，但鄭爽這幾年的演技卻始終進步不大，包括她和趙寶剛合作的《青春斗》，很多人說她在劇中的演技還需要打磨。

而且這幾年，因為暴瘦，「變臉」的話題，鄭爽的觀眾緣已不如當年了，這些事情也不是僅僅是有好資源就能改善的。

出道幾年，鄭爽一直都是圈內的流量，不管怎麼說，網友們還是希望她好的，大家也期待鄭爽能發現自己身上的不足點，好好提高業務能力，這樣子才能夠不辜負網友們的喜愛。

分手的鄭爽到底是拿著大女主的劇本，還是拿著「頹喪」劇本，這一切還需要等待時間來驗證。
