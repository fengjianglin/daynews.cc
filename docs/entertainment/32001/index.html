《多功能老婆》進入結局篇，這會還有多少人在追劇？

2019-12-25 832條評論 2,265次閱讀 435人點贊

這段時間在追的新劇有點多，感覺時間都快不夠用了，不過TVB的劇，還是習慣性地追著，即使在播的這部《多功能老婆》不是我喜歡的類型。

作為TVB今年最後一部台慶劇，《多功能老婆》只靠楊千嬅，便已經足夠吸睛，更何況這部劇TVB的當家小生花旦來了不少，黃浩然、陳煒、朱晨麗、洪永城等。只看牌面實力，《多功能老婆》在TVB劇中已經是大製作，這一點從這部劇的音樂原聲也能看出來，一部劇四首歌，要知道大製作如《再創世紀》都沒這待遇！

不過，《多功能老婆》的起點雖然高，收視也還不錯，但是成績對於這部劇來說卻不算很亮眼，畢竟豪華陣容加台慶檔，總的來說，跟今年TVB的其他幾部熱劇相比還是有一些差距。不知不覺間，《多功能老婆》也在本周進入結局篇，不知道這會還有多少人在追劇？

我想喜歡楊千嬅的觀眾肯定不會錯過這部劇，當然對於看習慣TVB劇的觀眾而言，這部劇算是比較特別，濾鏡看不大慣，劇情比較離地，模式也比較老套，唯一一點比較特別的，也是上次說到的，這部劇的感情線真的多且亂，上次寫完這部劇的多角戀之後，當晚更新的那集，居然在鋪墊萬風華跟渣男尚劍雄有過一段「情」。不知道大家看到這段時感覺如何，反正我看到這裡真的挺佩服編劇的，而且也能猜到這個鋪墊，在結局篇會帶來一波高潮！

說到這部劇的感情戲，各種CP真的很多，最後只剩5集了，大家猜猜這幾對CP最後能成多少？如果這部劇走大圓滿路線的話，那麼最後能成的CP應該會有7對，讓尚劍雄跟那顏爺落單。如果是一半HE一半BE的話，那麼思蕾跟宏立應該能走到一起，然後邢老闆跟呂老師，還有那兩對中年CP。

當然，還有另一種可能，那就是其他能湊得上的CP全在一起了，唯獨留下女一藍飛跟男一榮浩田沒在一起，然後來個開放式結局，這樣既有遺憾，又充滿想像空間，然後再插播回憶，配上這部劇4首歌作為Ending，似乎也蠻符合這部劇的風格！當然，具體如何的，還得看完大結局才知道，大家猜哪幾對最後能走到一起呢？
