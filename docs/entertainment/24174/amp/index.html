不愧是“瑤女郎”，年輕的時候比如金的當紅明星還要美

天天要聞
2019-12-19

作為1959年出生的女演員，劉雪華一直也都有著十分豐富的經歷。是我們大家比較敬重的表演藝術家，她們雖然現在已經不再年輕，但是留下來的作品還是非常之多，讓很多後輩都值得學習。對於劉雪華，大家對於她應該也是非常的熟悉。在娛樂圈裡面劉雪華老師也應該是一個實力派演員，她出演過的角色也是非常的多，很多劇也都是非常的經典。

很多人都無法想像，原來劉雪華年輕的時候也堪稱是一代美人，都說當年的演藝圈中美人輩出，記憶中也只是我們熟悉的朱茵、黎姿、王祖賢、林青霞這些大家所熟知的，卻不曾想原來當年的劉雪華，也是一個驚艷過時光的美人。很多人都無法想像，原來劉雪華年輕的時候也堪稱是一代美人，都說當年的演藝圈中美人輩出，記憶中也只是我們熟悉的朱茵、黎姿、王祖賢、林青霞這些大家所熟知的，卻不曾想原來當年的劉雪華，也是一個驚艷過時光的美人。

「傾國傾城」這個成語，大概就是來形容劉雪華年輕時候的樣子，來看看這組照片，你就知道她有多美了，完全不輸現在任何一個流量小花。她那修長的天鵝頸，彎彎的眉目，炯炯有神的大眼睛，簡直不要太迷人，可以說她天生就是一位大美女。

劉雪華是最早的「瓊女郎」，因為在瓊瑤劇中她多數飾演的是命運悲慘的苦命女子，所以有大量的哭戲，也因此被封為「眼淚皇后」。雖然劉雪華現在演的都是刻薄婆婆之類的角色，但她年輕的時候可是又漂亮、又柔弱，美到無話可說。

不得不承認，演技超群的劉雪華，當年的美貌真的是驚艷了時光，有一種說不出來，也難以被複制的古典美的氣息在其中，看完了劉雪華的這一組年輕時候的美照，如今也終於知道，年輕時候的劉雪華，這顏值到底是有多能打了。

她的五官也很精緻，又很立體，有一種東方女性的特有的美，她帶著一個綠色的髮帶，留著那時候最潮的頭髮，都不用怎麼精心打扮，毫不遜色當紅女星，現在雖然年紀大了，但是底子還是在的。
