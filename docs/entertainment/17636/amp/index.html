龐麥郎再髮長文反擊華晨宇，大吐苦水自稱弱勢，反問自己錯在哪兒

天天要聞
1 天 ago

最近幾年關於音樂版權的紛爭一直都不少，12月13號天才創作歌手華晨宇就因為音樂版權的問題被Cue了，而cue他的人很早就淡出大家的視野了，他就是龐麥郎，那個以一首《我的滑板鞋》血洗過全網的男人。他發布一條微博稱「只授權了改編，從未授權商業演出。」

但是在今日（12月16日），就被華晨宇的工作室發布了聲明，證實了這首歌的改編權和表演權華晨宇都有。後來又有網友扒出龐麥郎的《我的滑板鞋》和華晨宇的《我的滑板鞋2016》其實是兩首歌，因為龐麥郎把《我的滑板鞋》版權賣給了前公司華數，華晨宇先生又從華數買下了這首歌的歌詞版權，在作曲方面完全跳出改編思維，全部重新創作，形成一首新歌《我的滑板鞋2016》，並在歌曲中還標註了龐麥郎的名字。這都是好幾年前的事情了，今年都要到年尾了，龐麥郎突然在社交平台上提起 ，真的讓人摸不著頭腦。

現在的龐麥郎的一頓操作真的很難讓人不懷疑他不是在蹭熱度。2014年，《我的滑板鞋》莫名其妙的火了，在嗶哩嗶哩上到處都傳著他的鬼畜視頻，也是那一年，他和華數唱片簽約了，華數唱片陸陸續續和他預定了30多場商演，按照原計劃他應該會參加一些節目，一次來擴大名氣。龐麥郎紅了之後被媒體捕捉到了他一系列奇怪的舉動，明明是80後的他一口咬定自己是90後，明明有一口陝西口音卻堅稱自己是台灣人。更嚴重的是他直接違約逃跑。

現在的龐麥郎在大眾的眼中也許就是個「傻子」，傻得又有點讓人心疼。明明是小村子裡出來的，他給自己改了名字，把自己的家鄉也改了，急切地給自己換個身份，想要貼近這個不屬於他的大城市，而且還要始終如一的走出一股國際范，看起來十分尷尬。龐麥郎作為一個有夢想但是實力和背景都不那麼雄厚的追夢人已經夠幸運了。

而在隨後龐麥郎再次髮長文，道出這幾年來的酸楚和在追求音樂道路上所遭受的挫折，自稱遭營銷號詆毀，自己僅僅是一個想好好唱歌的普通人，並反問讀者自己錯在哪裡，難道錯在自己沒有強大的實力？自己真的就不該發聲嗎？其實在外界看來實力並不出眾的龐麥郎，在努力了幾年之後被人看重給了他實現夢想的機會，但是成名帶來的那些關注讓他接受不了原本的自己和那些冷言冷語給他前所未有的恐懼。於是身份作假，假唱，違約，導致現在人們都不認為他是一個歌手。

華晨宇對《我的滑板鞋》的改編，確實要比龐麥郎版本的高了不止一個檔次。但是現在重新聽聽龐麥郎的版本，彷彿有種強烈的渴望被這個自己嚮往卻不可得的世界包容的感覺，一種想被人記住的感覺，這也是很多網友支持華晨宇的原因之一。與其「碰瓷」華晨宇，倒不如龐麥郎再拿出一部好作品讓網友真正的對其刮目相看！
