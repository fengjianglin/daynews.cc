網曝鍾漢良疑患焦慮症自殘，粉絲闢謠：無中生有，請大家善良一點

2019-12-17 244條評論 1,813次閱讀 730人點贊

12月17日，某娛樂八卦博主的一則爆料引起了大家的注意，該博主稱曾和AngelaBaby一起拍戲被質疑扎戲的男星患上了焦慮症。

知情人說得有板有眼，稱他近日焦慮症複發，被助理髮現自殘過好幾次，有一次還鬧到醫院，團隊跟醫院高層簽了保密協議，還說他的電影要進入宣傳期，沒時間治病只能硬撐。

他所說的這些特徵都指向了男星鍾漢良，鍾漢良和AngelaBaby拍攝過《孤芳不自賞》，當時這部劇因為摳圖等原因口碑很差，對鍾漢良也造成很大影響。

而鍾漢良的電影《寵愛》定檔在12月31日，近期確實是在宣傳期內。

這則消息出來後，一些媒體為了引人注目，甚至起了很驚悚的標題，用上「一路走好」等詞。

隨後，鍾漢良的粉絲看到後則十分氣憤，因為他前段時間剛上了綜藝節目《漫遊記》，從中我們能看出鍾漢良是一個很溫柔的人。

他熱愛生活，堅持鍛煉，心態樂觀，正如他的昵稱「小太陽」一樣，一直是個溫暖的人，和焦慮症的癥狀沒有一點點相似。

或許大家可以造謠鍾漢良團隊隱瞞他的病史，但一個人健不健康，尤其是在精神方面，從他的一舉一動細節中能感受到的。

而鍾漢良的生活如此有規律性，絕對不像是患有生理性疾病的人。

粉絲也呼籲網友們能夠善良一點，不要為了滿足自己的好奇心，就來隨意揣測別人，還有一些娛樂博主，為了追求熱度隨意造謠藝人，這種做法更是可恥！

為了讓這些負面辭彙不出現在鍾漢良的微博廣場，很多粉絲自發性地開始安利鍾漢良的作品，他的每一張照片都傾注了粉絲的愛，也希望廣大路人們能停止猜測，別真等鍾漢良工作室追究起來，一切都晚了。

今年鍾漢良已經45歲了，有人說他是娛樂圈裡的一個bug，四十多歲的他看起來依舊年輕帥氣，不熱衷炒作，卻給觀眾留下過很多優秀作品。

這幾年，因為年齡差太大，鍾漢良還在尋求轉型，作品沒有之前多，以前他可是偶像劇男主角的不二人選，他穿著西裝，風度翩翩出現在鏡頭前，確實讓很多言情劇都有了臉。

在《來不及說我愛你》中，鍾漢良合作李小冉飾演了慕容灃，如今8年過去，很少再有民國劇能拍得讓觀眾又愛又恨，而鍾漢良穿著軍裝的樣子依然讓人歷歷在目。

還有他和唐嫣主演的《何以笙簫默》，這部劇當時熱度也特別高，是根據同名小說改編而成，因小說熱度很高，在作品剛開始拍的時候，很多人都對鍾漢良飾演何以琛產生過懷疑。

事實不過大眾又被打臉了，鍾漢良與何以琛融為一體，在劇中，他就是那個驚艷了時光的偶像劇男主角。

這麼一位優秀的男演員，大家不關注他的作品，卻屢次造謠他的個人生活，實在是令人可嘆。
