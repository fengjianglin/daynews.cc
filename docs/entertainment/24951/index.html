王子文與塗磊同樣是道歉，為何反響截然不同？對比之後高下立判

2019-12-20 415條評論 1,410次閱讀 339人點贊

估計王子文腸子都悔青了，如果時間能夠倒回，她應該絕對不會發那張照片。

近日，女星王子文在其社交平台賬戶，曬出一張照片，照片上一雙腳和一個包。打眼一看只知道那雙鞋是某大牌爆款，而那個包也價值不菲，除此之外，貌似並無特別之處。

畢竟，普通女孩都愛秀曬炫，更何況是個女明星，所以她這種行為都能理解。可就在大家調侃王子文炫耀之時，細心網友卻發現了問題，王子文的那雙腳，踩的正是高鐵的餐桌。

於是，王子文成功地曝光了自己的不當行為。

隨後，12月19日，在網友的指責和質疑聲中，王子文刪除照片，並發文道歉，她表示給大家添麻煩了，對這次的不當行為，感到十分愧疚，以後必將嚴於律己加以改正。

如果沒記錯的話，娛樂圈上一個和王子文有過類似舉動的，應該是情感導師塗磊，只不過人家塗磊老師是被別人曝光的，而王子文是自己「坑」自己。

還記得，當時塗磊乘機視頻，在網上傳得鋪天蓋地。很多人調侃他是一腳蹬出N個熱搜，網友口誅筆伐的陣勢，比現在的王子文有過之而無不及。

不過，王子文和塗磊一樣的開始，一樣的經過，結局卻大不相同。

視頻被曝光後，塗磊也曾發文道歉，可不僅沒有平息網友的怒火，反而質疑聲四起。甚至，最後連央視主持人元元都被牽連在內。

再反觀王子文，道歉之後，網友紛紛力挺，表示人非聖賢孰能無過，只要知錯能改，就是好同志。

看看，王子文和塗磊同樣是道歉，反響卻截然不同。

這就奇怪了，明明不雅的行為類似，道歉的結果為何不一樣呢？難道就因為王子文是女性，就因為她長得美？

當然，這只是玩笑話，網友絕對沒有那麼膚淺，更不會以貌取人。其實，大家對兩個人的態度不同，還是他們自身的問題，咱們往下分析

先說塗磊，當初視頻被曝光後，他並沒有馬上做出回應，輿論發酵多日，他才迫不得已出面道歉。

再說王子文，受到網友譴責後，馬上發文承認錯誤，未有任何的拖延。

眾所周知，明星藝人出現負面消息，考驗的是公關能力，網友最討厭的就是「裝死」、保持沉默、和拖延，在這一點上，顯然王子文做得更好一些。

說完道歉速度，再說一下道歉態度

還是先說塗磊，他的道歉發文，洋洋洒洒上幾百字。但是，總結下來就兩個重點

第一：之所以腳踩機艙壁，把腿放高，是因為靜脈曲張，而且自認為並沒妨礙別人。

第二：把網友分成三類人，對質疑自己的人，雖用詞婉轉，卻極盡嘲諷。

要說，這塗磊老師也很有趣，道歉就道歉唄，非要咬文嚼字。可能在他看來，自己的發文，並無任何不妥。可在網友看來，他字裡行間都透露著不誠懇。

要知道，道歉這種事也是有學問的，相對文采，大家更在乎的是你的態度。就這點，塗磊老師真應該向王子文學習。

確實，論文采王子文處於下風，論字數她也遠不及塗磊，但關鍵是誠懇啊。沒有找借口、沒有分辨，完全就是一副「躺平任嘲」的姿態。試問，人家都已經這樣了，誰還忍心再苛責？

這樣對比之後，就會發現兩個人的道歉水平，高下立判。所以，也就難怪王子文能過關，塗磊遭受非議了，虧他還是情感導師，被王子文甩了半條街。

說到底，還是大家對塗磊期望過高。他平時經常在節目里說教別人，儼然就是道德的典範，這樣的人設，犯錯已是不該，更何況態度還不端正。

王子文不一樣，她沒有人設壓力，又不經常說教別人，犯錯雖是不該，但認錯態度良好，大家自然會給予理解和支持。

可即使王子文態度再誠懇，她還是做錯了。身為公眾人物，理應謹言慎行，為大眾做出好的表率。

或許在有些人眼裡，這樣小小的錯處，根本不值一提。可是，大家要知道，作為禮儀之邦的一員，無論明星藝人還是普通人，都應該懂得文明舉止，而這些往往就體現在生活的細節里。

我倒認為，與其像塗磊和王子文這樣事後道歉，懇求大眾諒解，倒不如一開始就嚴格要求自己，避免類似事件發生，讓素質堪憂變成人人可見的優點，豈不更好！
