《武林外傳》當中的李大嘴、佟大爺、錢掌柜竟與四大名著緣分匪淺

2019-12-24 660條評論 1,838次閱讀 892人點贊

由尚敬執導，閆妮、沙溢、姚晨、喻恩泰、姜超、王莎莎領銜主演的情景喜劇《武林外傳》被很多人視為國產情景喜劇的巔峰之作。該劇在2006年1月2日央視電視劇頻道黃金檔率先播出之後，立刻在全國範圍內掀起了一股追捧《武林外傳》的熱潮，而因為該劇播出之後收視率異常火爆，因此當《武林外傳》的大結局在除夕夜前後播出時，該劇的收視率甚至一度超越當年央視春節聯歡晚會的收視率。

這部電視劇熱播之後，不僅讓閆妮、沙溢、姚晨三位領銜主演一炮走紅，而且也讓劇中的很多配角走進公眾視野當中。我們都知道，作為一部情景喜劇，《武林外傳》每集前後之間的劇情沒有太大的關聯，這就給了眾多配角粉墨登場的機會，在《武林外傳》的每一集當中幾乎都會出現一位配角，而這些配角的精彩表現為該劇增色不少。

所謂好花還需綠葉配，如果沒有那些配角的襯托，主角們的形象肯定就顯得不夠豐富和飽滿，也正是在那些配角們的襯托下，佟湘玉、白展堂、呂秀才、郭芙蓉等一個個人物才顯得活靈活現，讓觀眾記憶深刻，而儘管如今距離《武林外傳》首播之日已經過去了10年有餘，但是現在這部電視劇在網路上的點播率依然非常之高。

能夠經得起時間考驗的電視劇才能被稱作經典，而如今距離《武林外傳》首播之日已經過去了10年有餘，但是時至今日該劇在網路上的點播率依然非常之高，這足以證明《武林外傳》是一部經得起時間考驗的經典之作，而因為《武林外傳》深受觀眾朋友的喜愛，因此該劇熱播之後，劇中的很多主角和配角都受到網友熱烈關注。

《武林外傳》是一部情景喜劇，但很多人可能不知道，《武林外傳》當中的三位演員與四大名著改編而成的電視劇卻擁有一段特殊的緣分，這三位演員分別是「李大嘴」的扮演者姜超，「佟大爺」的扮演者朱龍廣，以及「錢掌柜」的扮演者洪劍濤，姜超曾參演過94版《三國演義》，朱龍廣曾參演過86版《西遊記》，而洪劍濤雖然參演了98版《水滸傳》，而沒有參演過87版《紅樓夢》，但是他與該劇卻也有一段特殊的緣分。

在《武林外傳》當中姜超飾演了「李大嘴」這一見利忘義的角色，但是很多人可能不知道，姜超還曾在94版《三國演義》當中飾演了「魏續」這一角色，而值得注意的是，姜超飾演的「魏續」出鏡時站在他身邊的大將軍「宋憲」則由高亞麟飾演，而高亞麟則在另一部情景喜劇《家有兒女》當中飾演了「夏東海」這一角色。

而更讓人感到不可思議的是，在《武林外傳》當中，飾演「佟大爺」這一角色的是朱龍廣老師，而朱龍廣老師是86版《西遊記》當中如來佛祖的扮演者，而因為在86版《西遊記》當中朱龍廣老師飾演的如來佛祖一角被化妝師化得「面目全非」，因此當他出現在《武林外傳》當中時，網友很難相信他竟然就是《西遊記》當中如來佛祖的扮演者。

姜超和朱龍廣老師因為都在四大名著改編而成的電視劇當中出現過，因此當網友稍加點撥之後，很多人似乎還有一些回憶，而洪劍濤雖然在98版《水滸傳》中出現過，但是在87版《紅樓夢》當中他自始至終都沒有出現過，因此網友很難想像，他與87版《紅樓夢》會有怎樣一段緣分？

在98版《水滸傳》當中洪劍濤飾演了高俅的義子「高衙內」這一角色，但很多人可能不知道，洪劍濤曾試鏡過87版《紅樓夢》「賈寶玉」這一角色，而因為當時他的額頭上有一塊疤痕，他因此落選，而就在2016年，洪劍濤還曬出了一張當年他試鏡「賈寶玉」的照片，而如果不是我們介紹，網友大概很難想像《武林外傳》當中的李大嘴、佟大爺、錢掌柜竟然與四大名著改編而成的電視劇擁有如此奇特的一段緣分。
