甄嬛傳：皇帝對甄嬛的感覺甚至超過了純元

2019-12-14 1,167條評論 2,444次閱讀 146人點贊

其實，皇帝是真的喜歡上了甄嬛。

一開始他以「曾經滄海難為水，除卻巫山不是雲」，來說明甄嬛是替身。其實換個角度來想，完全就是另一種解釋。

只有你在試圖說服自己的時候，才會做出某種特定的行為，來提示自己。

皇帝作為一個夫君，在劇中給他的人設，是十分思念純元的。可是為什麼偏偏是在甄嬛穿錯衣服被禁足後，他才會專門寫詩來思念亡妻呢。雖然這句詩裡面，主要強調的是除去巫山不是雲。但很明顯，他還是把純元和甄嬛做對比了。如果真的純元在他心中無可撼動，那為什麼要寫詩來專門強調自己有多深愛呢，為什麼還非要把甄嬛和純元比較一下呢？答案是：皇帝在說服自己，他一直愛的是純元。

這從側面說明了，甄嬛動搖了他。甚至可以說甄嬛動搖了他心中那份對於亡妻深愛的忠貞，他身上所被強加的責任感和道德感，讓他不能接受，不能允許自己背叛對純元的守護。即使要對甄嬛說假話。

有一個細節，陳建斌演繹的非常到位。當甄嬛質問他莞莞類卿的時候，皇帝的眼神閃躲了，他不敢正面回應甄嬛。他如果真的問心無愧，又怎麼可能不敢與甄嬛眼神交流呢。

在甄嬛生下孩子後，皇帝高興得都坐不住了，說公主也很好。當甄嬛執意離宮時，他都說，你要是喜歡，朕立刻可以封你為妃。你們聽懂這句話的意思嗎？皇帝是因為甄嬛穿純元故衣而被取消封妃，皇帝現在怕失去她，已經顧不得堅守純元那條底線了，直接要封她為妃。

至於後來，皇帝在睡夢中呼喚甄嬛。知道她和果子狸有染，仍然只是將她禁足，還願意相信她。（對比乾隆懷疑如懿和大師有染時，拷打四心。懷疑和凌雲徹有染時，就直接上宮刑）看到梅花，想起來的是甄嬛，而不是純元。原諒甄嬛時，哽咽得說，孩子們都很想你。這些都印證了，皇帝早已愛上了甄嬛。

皇帝對甄嬛的感覺甚至超過了純元。
