《懸崖之上》《陰陽師》《七人樂隊》這份電影備案看點太多

2019-12-19 1,950條評論 3,542次閱讀 979人點贊

2019年的最後一個月，電影檔依舊精彩。從點映開始就刷爆好評的《誤殺》，期待已經的《南方車站的聚會》終於和觀眾見面，馮小剛、黃渤新作扎堆上映……

最近，國家電影局備案公示已經立項的幾部電影刷爆了大家的屏，其中不乏許多大IP，大製作的身影，不由得引起了大家對2020年上映電影的期待。

《七人樂隊》

千呼萬喚始出來，《七人樂隊》（原名《八部半》）終於正式立項，由寰亞備案。

《新京報》曾在報道中寫道：《七人樂隊》的神仙陣容可以說是華語影史上不曾有的。這句話一點不誇張，《七人樂隊》原名《八部半》多位中國香港的著名導演將參與到這部影片中，包括徐克、洪金寶、許鞍華、譚家明、袁和平、杜琪峰、林嶺東。原計劃吳宇森導演也將參與到影片中，最終因為身體原因而退出。

電影將以上個世紀40年代的香港為背景，每位導演選擇一個年代進行創作和發揮。值得一提的是，在電影的數碼化已經很成熟的年代，《七人樂隊》選擇了用膠捲拍攝，表達了向菲林致敬的含義。

《七人樂隊》獲准立項的消息一公布，不少影迷朋友表示非常期待。今年的國慶，7位著名導演為全國觀眾獻上的《我和我的祖國》，再現了新中國的誕生和發展。同樣都是7位導演合作，每人知道一個片段，不知道這支《七人樂隊》為我們帶來怎樣的視聽盛宴？

《晴雅集》

昨天，郭敬明版《陰陽師》立項一公布，立即登上熱搜：

微博轉發人數也達到65.9萬

一個Tip，郭敬明導演的這部《晴雅集》是改編自夢枕貘的小說《陰陽師》，而非網易遊戲《陰陽師》。電影從16年籌備至今，終於完成了立項。

原著小說豆瓣評分8.6，以幽暗遙遠的平安時代為背景，講述了陰陽師安倍晴明和好友武士源博雅遊走於陰陽兩界，破解離奇事件，為人鬼解憂的故事。

電影《晴雅集》主要講述晴明承師傅遺命，前往天都參加祭天大典，與博雅從對手成為搭檔，一同破解離奇案件的故事。從劇情梗概上看，對原著的改動並不大，不過想要在短短几個小時內講述體量那樣大的故事，應該還是需要進行一定的改編的。

不愧是郭導，電影立項和官宣陣容只差了幾天，這樣的劇透效率觀眾表示real喜歡：

趙又廷飾演晴明

鄧倫飾演博雅

王子文飾演公主

春夏飾演瀧夜

汪鐸飾演鶴守月

人物海報融入了傳統的浮世繪元素，質感max，小四的審美超棒！

《瀧夜曲》

如果說《晴雅集》著重講述的是兩人相識相知的故事，《瀧夜曲》則更側重講述晴明和博雅聯手破解怪譚的故事。

《瀧夜曲》中，晴明和博雅臨危受命調查天都怪事，他們發現這些怪事之間存在著說不清的聯繫，而它們的背後有著更大的陰謀，直指十年前南疆舉兵進犯，當時參與平亂的四位有功之臣……當時的重臣如今卻紛紛遭遇生命危險，二人必須在有限的時間內破解陰謀，才能力挽狂瀾……

看完劇情簡介，有一種遊戲中大家爭分奪秒齊心合力打鬼王的趕腳。

《瀧夜曲》中春夏將飾演女主角瀧夜。春夏給人的感覺像是一隻小鹿，倔強，靈氣中還有些嫵媚，和瀧夜有幾分相似。

此前，夢枕貘曾出版過小說《陰陽師.瀧夜叉姬》，小說中瀧夜叉姬是一個頭戴斗笠的曼妙女子。傳說中，世上有一把寶刀名叫黃金丸，被它所傷的傷口二十年內都不會癒合。驍勇善戰的藤原秀鄉曾經用這把黃金丸殺了四處作亂蜈蚣怪。而藤原秀鄉和黃金丸在夜裡遭遇了瀧夜叉姬的襲擊。

襲擊案的背後，是一場驚天陰謀。此時的平安京暗流涌動，陰陽兩界的決戰一觸即發。書中關於這場爭鬥的描寫引人入勝，不禁然後讓人期待，如果將這場戰爭影視化，將會是多麼壯觀的畫面。

回看《陰陽師》改編的兩部電影，演員陣容沒得說，海報也是美到讓人感嘆，接下來就該期待人物定妝照和預告片了，不知道郭敬明為觀眾創造的平安時代會是怎樣的？

《懸崖之上》

《懸崖之上》是張藝謀執導的首部諜戰片，講述了上世紀三十年代，為了獲取日本展開反人類試驗的證據，由張憲臣、張蘭、王郁、王楚良四人組成名為烏特拉的小組被派往東北。特務科長高彬已經盯上了四人，而卧底在特務科的周乙必須在保全自己的情況下，為四人保駕護航……

目前，電影《懸崖之上》仍未官宣人員陣容，導演張藝謀表示：「演員很好，故事有很大的懸疑感」。一向在演員的選擇上有想法有眼光的他，這一回又會給觀眾帶來怎樣的驚喜呢？

《鳥鳴嚶嚶》

闊別導演這個角色十年，田壯壯老師終於要帶著《鳥鳴嚶嚶》回來了。

《鳥鳴嚶嚶》改編自阿城的中篇《樹王》，《樹王、棋王、孩子王》是阿城最著名的小說。1987年，小說《孩子王》改編為同名電影，由阿城擔任編劇，1991年阿城再次擔任電影《棋王》的編劇，不過在《鳥鳴嚶嚶》中，阿城不會擔任編劇。

《鳥鳴嚶嚶》的背景發生在上個世紀60年代末，一批知識青年胸懷抱負和理想來到了雲南，他們為了種植「有用」的橡膠樹，而大肆砍伐「沒用」的原始森林。砍伐的行動愈演愈烈，人們甚至把砍刀舉向村民時代敬畏的「樹王」。當地村民「肖疙瘩」是山林的守衛者，他懂山懂森林，懂得森林是山的皮膚。他和堅持要砍樹的知青們開展了一張較量……

《鳥鳴嚶嚶》的片名出自《詩經·小雅·伐木》中的「伐木丁丁，鳥鳴嚶嚶」，結合《樹王》要表達的主旨來看，有保護生態，維護自然的含義。

首發的人物預告可以看到周奇、李宛妲的身影，換上「知青裝扮」的他們青春洋溢，一下子就讓人回到那個時代。

目前其他主演的名單尚未透露，希望通過這部電影能夠讓人們再一次反思我們和自然該如何相處。

《刀尖》

如果要評選最佳諜戰片，相信不少人會把寶貴的一票毫不猶豫地投給《風聲》。在《風聲》的海報上寫著這樣一句「風聲之後，世間再無傳奇。」

的確，《風聲》是國產諜戰片的一座大山。如果當年製造這部傳奇的人再度聯手，會誕生一個新的傳奇嗎？沒錯，我要講的就是《刀尖》。

《刀尖》改編自麥家的同名小說，《風聲》、《潛伏》等經典之作正是出自他之手。

導演高群書是電影《風聲》的導演之一，此次高群書和麥家二度合作，讓不少影迷感到興奮。

再看演員，張譯、黃志忠、成泰燊、郎月婷、黃璐……都是表現力十足的實力派。這一點，在預告片中體現得淋漓盡致。

首發預告長達9分鐘，卻只有4個長鏡頭，記錄下四個主演的表情。

他們或沉默

或哭泣

或大笑

奉獻了極具張力的表演。

1940年南京，各方勢力盤根錯節，身負家愁的金深水與林嬰嬰，為了生存共同聯手。他們發現了日軍慘無人寰的「春雷計劃」，為保家衛國，二人抱著必死之心，行走在刀尖之上，置之死地而後生。

從原著到主創陣容再到劇情，無一不是重磅級，期待它能續寫《風聲》的傳奇。

梳理完這些已立項的電影，小戲不由得開始期待它們能早日和觀眾見面，以上這些電影中，你最pick哪一部，趕緊參與我們的投票吧！
