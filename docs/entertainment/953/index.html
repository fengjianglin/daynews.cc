12月3日由漫威影業出品，凱特·紹特蘭執導，斯嘉麗·約翰遜、塞巴斯蒂安·斯坦等主演的電影《黑寡婦》全球首支先導預告片出爐。

預告片時長2分11秒，預告片一開頭是一個遠景鏡頭，然後就是寡姐的臉部特寫，不得不說寡姐顏值是真的高，即使素顏出境也十分扛打。

故事的背景發生在蘇聯解體後15年的美國，大約是2006年左右，《鋼鐵俠》故事發生之前。預告開始，黑寡婦有一段獨白”我曾經一無所有，直到我加入了復仇者聯盟，這個大家庭”鏡頭隨著寡姐的自白不停的切換。

在《復聯4》時寡姐和美隊說過類似的話，她真覺得復仇者聯盟是一個家一樣的存在。

停頓後，寡姐繼續說”但沒有什麼是永遠不變的，”然後鏡頭出現了”漫威影業”黑紅色標識。

然後傳來一個富有磁性的男聲”聽說你當時不得不匆忙離開”寡姐的聲音再次響起”現在的日子並不容易”。

接著就是一段直白的男女對話，交代了一部分背景，寡姐說自己曾經有著多重身份，但是已經不願意再迴避過去。

預告片中還有一段寡姐與二代黑寡婦葉蓮娜·貝洛娃同框的畫面，寡姐舉著槍說”我知道你在那裡，你肯定也知道我來了”然後一個閃身她與二代黑寡婦面對面舉槍見面了。

之後二人展開了一頓激烈的近身博弈，然後寡姐說”我也很高興見到你，妹妹”然後二人和平相處，一同騎機車離開。

在預告片中還提前透露了多個精彩的動作場景，寡姐的身手依然矯健，她的爸媽也同框出境”再次相聚”。
