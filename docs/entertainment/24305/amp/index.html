知情人證實鄭爽、張恆在戀愛秀交惡：最後一期節目拒絕同台

天天要聞
2019-12-19

傳媒櫻桃派系【頭條理娛君】特約作者，看娛樂熱點深度解讀，認準理娛君！

12月18日，汪峰推出全新單曲，寡淡的娛樂圈風平浪靜。誰也沒成想，鄭爽和張恆的分手又出新番，據媒體報道，多位知情人確認兩人已經正式分手，目前張恆已經不再負責鄭爽的相關工作。

不僅如此，兩人的分手也並不友好，疑有經濟糾紛，而兩人合資的公司已經分給了男方，股份已經拆分，將可能會訴諸法律。

其中，還提及10月17日，兩人以「當期不合適」為由拒絕了某節目的錄製。

對此，傳媒櫻桃派從參與過《女兒們的戀愛》錄製的知情人處了解到，在節目的最後一期，兩人確實婉拒一同路面，連感情問題都成了「禁區」。

一檔戀愛秀錄到最後變分手見證，說來也是傳奇。

1、 鄭爽分手的線索脈絡

鄭爽與張翰、胡彥斌的兩段戀情結束後，高調宣布與圈外人張恆墜入愛河，因為她對於婚姻的渴望，也被認定好事將近。

在錄製戀愛秀之前，鄭爽就委託男友擔任經紀人，並且開始聯手創業打造網路平台。

一切都顯得其樂融融時，鄭爽就曾做客好友馬天宇的綜藝，直言想要生三個孩子，亦被認為好事將近。

2019年夏天，兩人答應一同錄製戀愛秀，也被認為這段戀情將借著綜藝瓜熟蒂落。第一期節目中，鄭爽曾感慨遇見張恆之後不僅體重反彈、心情也變得開朗。

而鄭爽媽媽們看著張恆也有如岳母看待准女婿，直接問起了對於婚期的想法。

然而，這一切隨著節目推進就變了，張恆不善於溝通，每逢爭執就開始「冷暴力」。他的心智並未如外界與鄭爽父母預想的成熟，在男友、經紀人、合伙人三重身份疊加中顯得疲於分身。

因為分歧，鄭爽數次灑淚後又怯懦的哄男友重歸於好，兩人的愛情模式看得鄭爽爸爸隔空眉頭緊鎖。

比起隔壁的郭碧婷、向佐和徐璐、張銘恩日趨成熟，鄭爽和張恆的戀情更像小朋友過家家，不成熟的相處方式讓看客們集體捏汗。

在終站錄製時，兩人的矛盾再度爆發，恰逢九月底的酷暑餘溫，兩人一同前往千島湖，關於該出去玩還是去採摘，意見頻頻發生分歧。鄭爽忍耐暈車不適陪張恆看湖景，結果男友呼呼大睡，連紀念照都沒有拍就趕緊返程。

雖然兩人的矛盾以鄭爽故作生氣，其實是給男友準備驚喜作為結局，但是關於兩人隨著節目尾聲告吹的消息也不絕於耳。

這其中，張恆悄然清空了自己的社交平台，而鄭爽則開始在網路上出手自己的二手情侶服飾。對於一個戀愛大過天的女孩，拿出自己的情侶款出售，這讓眾多粉絲也認準自家愛豆新戀情又受阻了。

鄭爽與男友共同投資的網路公司被員工在網路爆料已停止運營，並暗示兩個老闆將分家。此後，有媒體探訪鄭爽公司的所在地，發現公司已經處在停止運營的狀態。

不僅如此，張恆也被翻出已經註冊新公司，而股東名單里恰巧沒有了鄭爽。種種指向里，曾經被認為是鄭爽單身終結曲的戀情被認定將謝幕。

2、 知情人曝他們確實遭遇危機

在張恆與鄭爽的戀情被認為亮紅燈的最為「實錘」的一點是他們戀愛秀的收官，節目10月17日在長沙錄製，陳喬恩帶著節目里擦燃愛火的艾倫出現，郭碧婷和向佐，徐璐和張銘恩都雙雙見家長，其中唯獨鄭爽是單獨一人。

而鄭爽全程對於戀情閉口不談，對於戀愛秀的總結是，這是和爸爸難忘的一次綜藝錄製。這個信息點讓外界都感覺有點微妙。

鄭爽和張恆到底怎麼樣了？傳媒櫻桃派從一位參與過該節目錄製共走的知情人士了解到，兩人在綜藝中的爭吵從來都不是劇本：「他們最開始確實很好，但是節目里的爭吵幾次下來，可能兩邊都有反思，這些矛盾對於戀人來說都是很傷的。但是我可以肯定的是，他們在千島湖那一次的錄製結束還沒有分手，只是後來發生了什麼就不知道了。」

該知情人士稱，鄭爽的父母看完兩人的相處後對女兒的戀情也有了新的態度：「爽爸其實是很喜歡張恆的，但是作為女孩子的爸爸，看見張恆的許多事情處理不成熟還是會著急。而且他也認為他們不適合再繼續把生活和工作全部都捆在一起了，但是拆解的這個過程其實很容易發生矛盾，會有信任度的懷疑等等。」

該知情人士透露，10月17日的最後一期節目錄製時也有一段插曲。原本節目組希望所有情侶都能抵達現場，但是張恆方面一直都以「時間合不上」為由拒絕了錄製，但是節目組有人了解到其實那段時間他是在上海的，所以大家就覺得很微妙。

而鄭爽抵達錄製後，她的跟拍導演也要求節目組現場不再回味她的這段戀愛秀，當時也讓許多人都覺得有些意外。「所以大家最終看到的播出版沒有提及感情，我沒有參與具體的對接，但是氣氛確實很微妙。」

在許多與鄭爽、張恆相處過的人來看，他們其實互相都很疼惜對方，但是溝通錯位才是主要矛盾：「愛情的經營本來就不容易，要做好戀人又要做好經紀人、合伙人，不是人精做不好這些，大家可以看到鄭爽和張恆肯定都不是。他們都是個性很單純的人。」

3、 學會獨立的相愛才是28歲的鄭爽該學會的成熟

在鄭爽帶著張恆錄製這檔戀愛秀初時，她在上海發布會的後台接受傳媒櫻桃派採訪時曾經感慨28歲的年齡已經成熟了許多，不再是過往的小女孩。

然而隨著戀愛秀深入，她與男友相處時面對分歧的溝通錯位、爭吵後的示弱都讓人看得很窒息。

童年曾經遭遇同學的排擠，取悅似乎變成了她的本領。同時，她對於另一半的需求也不僅僅是感情的陪伴，還有事業的合作，她期待和男友創業，兩人不僅是戀人還是黃金拍檔，從中尋求成就感。

但是換個面來看，這難道不像《雞毛飛上天》里的駱玉珠和陳江河么？這種集情路鴛鴦與商場搭檔於一身的愛情在影視劇里都堪稱佳話，生活里要做到有多難？

如果說鄭爽曾經為張翰拒絕拍吻戲，甚至要停工求愛是王子和公主的童話。她在這段戀情里曾經自詡的成熟應該是渴望的模式從偶像劇變成了商戰偶像劇。共同的特質依然是夢幻。

她和張恆如今被媒體報道已經分手，還因為經濟糾紛或訴諸法律。這像極了童話故事在現實里的四處碰壁。

而張恆的社交平台連過往自我簡介里關於他與鄭爽共同創建的網路平台介紹都已經默默刪除。

如果鄭爽只把張恆當男友，他們就這麼各有各路的奔跑，結局會不會好一點呢？答案未可知，但是可以肯定的是，這比她認定的成熟會更理性，也會減少矛盾。

諸多指向都讓這對才修完恩愛的情侶成了分飛燕，我們依然期待既然彼此心疼過對方的兩人不需要鬧到魚死網破，好就好散是一種圓滿。對於鄭爽來說，要獲得所謂完美的愛情，更應該懂得看得起自己。
