郭德綱前妻嫌窮離婚，現在兒子成國民老公，前夫賺大錢，悔恨大哭

2019-12-18 1,212條評論 3,709次閱讀 2651人點贊

作為著名相聲大師的兒子，郭麒麟原本被觀眾看成是”搞笑”一卦的人物，有了這麼個會說相聲的老爸，兒子或許也帥不到哪裡去，可誰知道他居然憑藉《歡樂喜劇人》這檔節目一夜火了起來，成為萬千少女的”新牆頭”，播放時的全網討論度趕超李易峰，雖說繼承了父親的相聲事業，行當和”男神”這個詞看起來差別有點遠，可是郭麒麟論說人品和業務能力，卻是實打實的男神本人，還被不少少女笑言是”國民老公”，有著小鮮肉的長相和年齡，卻有遠超過他們的沉穩大氣。

這個表面看起來陽光開朗、做人暖心體貼的少年，其實之前曾在節目上透露過對於生母的怨恨和懷念，這些讓人疼惜和哀傷的經歷，全都起源於郭德綱的前妻。在認識現任妻子結婚之前，郭麒麟在天津學習相聲的過程中意外認識了一名名叫”胡中惠”的姑娘，當年都是少年，兩人情愫暗動間也決定從此攜手一生。

只不過婚姻生活遠遠沒有兩個人想像的這樣美好，那個時候郭德綱還只是相聲界的學徒，不像現在這樣有名有勢，每日輾轉於各種相聲場，但總也受不到提拔的機會。在這段有些艱難的婚後生活中，胡中惠生下了郭麒麟。

原本產後精神狀態就十分緊繃的她，對於這丈夫沒錢供養家庭，沒精力照顧她的現狀感到極度失望。據傳夫妻之間因為”錢”的問題曾爭吵過不下百回，胡中惠在家是備受疼愛的公主，和郭德綱結婚後卻得處處節儉，由奢入儉難，最後實在難以忍受這樣的生活，選擇和郭德綱離婚，遠走日本。

隨後郭德綱則帶著兒子開始在相聲界的打拚，後來又認識了自己的現任妻子王惠，應該說郭德綱能取得現在的成就，和這位不嫌貧愛富的現任妻子有很大關係，在面臨離婚、孩子撫養、事業打拚的多重壓力下，王惠的到來解救了他，也解救了這個家庭。

因為賢惠能幹的現任妻子，郭德綱的事業很快就發展起來，成為了圈中人人稱羨的相聲大師，一手開辦德雲社，打造了相聲界完整的產業鏈條，從之前不值一文的毛頭小子到現在坐擁金山銀山的老闆。他的兒子郭麒麟也頗為爭氣，通過一檔相聲類節目大火後，迅速在迷妹圈子裡瘋狂圈粉，成為了名副其實的”國民男神”，也不知道胡中惠是不是悔不當初。

之前曾有日本媒體在機場意外捕捉到前往國內探親的胡中惠，提起如今丈夫和兒子的現狀，她看起來也委屈、後悔不已，曾落淚袒露了自己如今的境遇：早知道當年應該再等一會。但其實這個假設並不成立，她原本就是個趨利的女人，不會陪伴郭德綱度過那段艱難的日子，也無法見證郭麒麟今日的成功。那麼對於郭德綱前妻如今的現狀，你有什麼想說的嗎？
