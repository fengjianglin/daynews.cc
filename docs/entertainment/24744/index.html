沈騰不夠芭莎來湊，情侶刊被提上日程，楊冪魏大勛戀情藏不住？

2019-12-21 996條評論 3,592次閱讀 349人點贊

楊冪魏大勛這對緋聞男女友，最近可是賺足了大眾眼球，戀情傳聞一波接著一波，兩人卻是能夠耐得住性子，以保持沉默對待，不回應的態度更讓人浮想聯翩。

繼楊冪魏大勛住同一酒店被拍後，兩人過往的點點滴滴都被挖了出來，魏大勛愛慕楊冪許久這件事被曝，無疑加重了戀情傳聞的可信度，儘管兩位當事人久久未回應，但在外界看來早已八九不離十了。

12月20日，有吃瓜群眾在芭莎主編留言區評論，提議為可以為楊冪魏大勛來一本情侶刊。沒想到該網友竟然被主編翻牌，時尚芭莎的主編衛甜回應：「確實有編輯報了這個話題」。

該主編的一句話引來許多人圍觀，一幫吃瓜群眾好生期待，並調侃芭莎為「紅娘刊」。

網友因此事而熱議，不過要注意的是，芭莎主編回應只是將話題報了上去，也就是該項目正在籌劃當中，但這件事是否能夠最終得以實施就很難說了，畢竟紅娘也有難拉成的媒啊！楊冪魏大勛本就是大眾熱議的焦點人物，他們避還來不及，拋開戀情緋聞合作的幾率有點小。

如果兩人戀情屬實的話，那麼這確實是個公開的絕佳好機會，能與楊冪再次合作，想必也是魏大勛心之所願吧！如今有芭莎主編親自回應情侶刊一事，對於大眾而言，無疑是對楊冪魏大勛戀情的再一次實錘，這也是繼沈騰吃瓜後的再次實錘。

12月19日晚，沈騰跑到楊冪留言區評價，艾特她的緋聞男友魏大勛，完全是一副吃瓜群眾的模樣，他的舉動引起不小的影響力，更讓網友一頓熱鬧。沈騰的舉動並未對當事人的行為造成什麼影響，兩人仍默默無聞，如今沈騰不夠芭莎來湊，不知楊冪魏大勛是否有話要說。

不過不管楊冪魏大勛會怎樣，可以肯定的是如今兩人的情侶刊已經被提上了日程，至於他們答應還是拒絕，只能讓大家拭目以待了。

楊冪魏大勛的戀情頻頻被實錘，尤其是繼沈騰、芭莎主編公開言論此事後，戀情更藏不住了。關於楊冪魏大勛久久不承認戀情的原因大家議論紛紛，其實離不開這幾點。

其一，楊冪魏大勛確確實實沒戀愛，兩人只是關係友好而已。其二，他們遲遲不予承認，無非是想要讓熱度維持的時間更久一些，畢竟明星戀愛之事與本人事業發展息息相關，不回應反而吊人胃口，總讓人多加猜想，對於兩人而言都是話題度。

其三，有媒體猜測楊冪魏大勛之所以不承認戀情，或許是為了給自己留後路，避免今後兩人感情有了變故而尷尬。其實明星有這樣的憂患意識實屬正常，畢竟他們是公眾人物，許多時候會因為戀情私事而影響到事業，戀愛確實要謹慎，公開戀情更要三思而後行。

尤其楊冪與劉愷威已經有過一段失敗的婚姻，她對待感情會更慎重一些，即使外界看好她與魏大勛這對組合成真，但她也不會輕易將新戀情公佈於眾，想必她十分在意新戀情會影響到自己的星途發展。

楊冪與劉愷威已經離婚一年了，兩人好聚好散、各自安好，無論哪一方都有權利追逐新的感情，被過去牽絆影響，並非是一件好事。

楊冪魏大勛雙方對戀情傳聞都不予回應，其實大夥完全不必著急，事情到底是怎樣一回事，隨著時間流逝自然會見知曉。當然了，大家都希望楊冪魏大勛的戀情傳聞是真，而非過眼雲煙，而且十分期待他們的「情侶刊」，如果出版的話，應該會大賣吧！
