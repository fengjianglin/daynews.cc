張鐵林攜愛魚“乾隆”奪冠，網友：“皇阿瑪”是個懂魚愛魚之人

2019-12-24 616條評論 4,212次閱讀 3221人點贊

張鐵林是我國著名的影視演員，1998年，一部《還珠格格》火遍兩岸三地，一時萬人空巷。它開創了中國電視劇至今無人超越的收視紀錄，最高點突破65%，在此後的二十年里被重播無數次。飾演乾隆的張鐵林，也成為了觀眾們熟悉喜愛的「皇阿瑪」。

還珠之後，張鐵林似乎就和龍袍結了緣，大家喜歡稱他為：皇帝專業戶。《漢宮飛燕》里的漢成帝；《南少林三十六房》里的雍正帝；《滿漢全席》里的康熙；《火帥》里的宋仁宗 ……他塑造的皇帝威嚴霸氣、不怒自威，笑起來慈祥親切，既有威懾力又不失距離感。

鮮為人知的是，在工作的閑暇之餘，這位皇帝專業戶還以魚為樂，在家裡造了個大魚池，養了十幾條錦鯉，都非常漂亮健康。其中的一條名為「乾隆」的錦鯉，更是在剛剛結束的2019年中國錦鯉大賽中奪得75部紅白冠軍！

圖片來源公眾號愛鱗說

據報道，這條「乾隆」是2017年5月出生的三段紅白錦鯉，有著著名的XJR血統，精品中的精品。圍觀網友讚歎：「皇阿瑪」不僅是皇帝專業戶，還是個懂魚愛魚之人！

圖片來源公眾號愛鱗說

其實除了張鐵林，另一位皇帝專業戶張國立也是一位錦鯉愛好者。據報道，在2015年舉行的第15屆香港若鯉比賽，得第一的竟是張國立，對，沒錯，就是那個演康熙微服私訪記的張國立，拍戲這麼忙竟有時間參加比賽，不過這一下也成了錦鯉的免費代言人了吧。

張國立斬獲香港錦鯉界「金像獎」

在國人眼中，養魚不僅可以作為興趣愛好，閑暇之餘還可以放鬆心情，陶冶情操。現在養魚不僅是一種文化傳承，也是一種時尚流行。一起來看看作為公眾人物的明星大佬們都是如何養魚的。

馬雲

據傳讓首富最初入坑的是一缸紅財神。

譚詠麟

華語歌壇唯一「校長」譚詠麟，經常逛香港的金魚街，坦言養魚能用來減壓，工作繁忙時，會坐在魚缸前發獃，通過觀魚能把碰到的困難、難題忘卻。

比爾蓋茨

這位世界級的首富，他也喜歡養魚。家裡的魚缸大的，全地球人都知道。蓋茨從1990年開始，花了七年時間、6000萬美元，建成的豪宅里，建有一個巨大的魚缸。

據說家裡養的是鯨鯊~或者說他的那不是魚缸，應該叫水族箱~

阿杜

沙啞天王阿杜，愛好竟然是養龍魚，並且在自己新加坡的老家投資過千萬建設了一個大型的養殖場。

成龍

成龍養魚，應該是大家沒有想到的吧~看他逗心愛的魚，美哉美哉~

之前他還做了一個視頻，逗他心愛的成吉思汗鯊。

劉亦菲

原來不止有男明星喜歡養魚，女明星也不例外哦~我們的神仙姐姐劉亦菲不僅是個愛狗人士，對於養魚也是一個忠實的愛好者。看這位美女家的魚缸，真是霸氣十足呢！

還等什麼，趕緊轉起來，給愛養魚的朋友們一起看看！
