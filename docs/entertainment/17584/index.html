李宇春曾被父親勸退出娛樂圈，肖戰、高以翔、藍盈瑩也有同樣經歷

2019-12-17 61條評論 1,057次閱讀 63人點贊

作為歷屆最紅的一代超女，李宇春21歲就一唱成名，紅透中國登上美國《時代周刊》亞洲版封面，創造了人生奇蹟。不僅如此，李宇春還逆轉了流量只是曇花一現的魔咒，多年來在音樂、時尚、表演等領域不斷突破自己，給觀眾和粉絲帶來驚喜，在業內也越來越有地位。

然而雖然李宇春取得了諸多令人艷羨的成績，但她的家人更多地還是看到了成績背後的心酸。尤其是身為警察的父親，可能一直都未能真正理解李宇春的演藝工作。由於一直對演藝圈抱有戒心，李宇春父親在其參加超女比賽時就沒有去現場支持過，他認為這不過是一場唱歌比賽而已，等淘汰了女兒就回來了，卻沒料到李宇春竟因此徹底改變了人生軌跡。

2013年，李宇春在長沙機場電梯被壓倒，更讓父親覺得演藝圈不安全，萌生了讓女兒退出一線的想法。等到2018年李宇春因滑膜炎發作，選擇打封閉後拄拐上台完成演唱會巡演時，李宇春父親更是直接找到了經紀人要求對方不要再讓女兒當藝人了。

李宇春的父親也是愛惜和心疼女兒，但對於李宇春來書舞台和歌手就是她最摯愛的事業。

說起來，娛樂圈反對子女出道的父母確實不少。前不久因錄製節目發生意外去世的高以翔，早前進入娛樂圈時，也曾遭遇過父親的激烈反對。

與李宇春不同，高以翔父親是因為當時兒子頻頻被指是假豪門，妻子被造謠不是選美小姐、到處借錢等感到氣憤，才透露因為環境太複雜，自己曾堅決反對高以翔進娛樂圈，但沒想到最後兩個兒子都選擇了這條路，不過他也表示既然孩子已經選擇這行，那做父母的也只有力挺。

要說娛樂圈的環境複雜，同為綜藝天王的吳宗憲更是深有體會。2005年，吳宗憲曾被不明人士圍毆，頭部和眼部皆受重創，父母因此心疼不已，力勸他乾脆離開演藝圈。

除了環境複雜，工作壓力又大又不穩定，也是不少明星父母不希望子女進娛樂圈的原因。憑藉《陳情令》大火的肖戰，如今是娛樂圈當之無愧的頂流之一，但作為頂流，就一定有頂流需要付出的代價。自從爆火以後，肖戰就如同空中飛人一般，馬不停蹄地參加各種活動和拍攝，身體疲憊和精神壓力也隨之而來。

雖然肖戰自己一直非常自律，但其父母還是免不了覺得這樣的生活太累太不穩定，多次勸他要是做不下去就回來。

因飾演沈眉庄被觀眾熟知的斕曦，最初進入娛樂圈時也遭到了父母的反對。和肖戰父母不同，斕曦父母本來就是黃梅戲演員，深知做演員的苦，所以一直不同意女兒繼承自己的事業。

不過相對同劇演員藍盈瑩的父母，斕曦父母的反對已經溫和了很多。由於家裡人都在經商，藍盈瑩父母一直希望她大學能夠學習工商管理，等到畢業後回去接管家裡生意。

但無奈藍盈瑩從小就喜歡演戲不喜歡經商，所以為了完成夢想她就背著父母報考了上戲附中，其父母發現後表現也是決絕，直接撕碎了她的錄取通知書。

但無論是像藍盈瑩父母這般激烈反對，還是像吳宗憲父母那般苦勸，最終這些明星都還是選擇在娛樂圈繼續闖蕩，只是接受了台前的光彩奪目，也就必須得承受幕後的辛酸冷眼。說到底，無論干哪一行都是如此，明星自然也不能例外。
