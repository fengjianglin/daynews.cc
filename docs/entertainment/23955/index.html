慶余年劇透：范思哲的父親其實不是范建，而是郭德綱

2019-12-20 2,363條評論 4,924次閱讀 2164人點贊

　「沒錢？沒錢鬥不過，鬥不過。」「姑娘，可否與小生共推牌九？」「斷我財路，如同殺我父母。」

　　在如今大火的《慶余年》中，范思哲金句頗出，表面是個憨憨，實則小算盤比誰都精明。畢竟是和財務大臣推牌九都能把把吃雞的人。

　　讓人驚奇地構思，金句狂出的劇本台詞，讓古裝劇《慶余年》大火，牢牢佔據劇榜第一名，而劇中人物范思哲，也讓觀眾喜歡的不得了。收到不少粉絲的追捧。

　　然而隨著范思哲的爆火，愛錢如命的「地主家傻兒子」卻被扒出了現實中的真實身份。郭德綱之子、德雲社「少班主」。

　　然而對於郭麒麟而言，並不想讓大家知道他是郭德綱的兒子。不想借著父親的名號行事。要不是網友曝出，好多人都以為他的「父親」是范建。

　　然而就在今天，一則消息登上了熱搜第一，郭德綱不知道兒子出演《慶余年》。於是網友們調侃說： 「郭德綱：連郭麒麟是我兒子都不知道就想騙我看劇」。

　　事實上，郭麒麟早就參加《我就是演員》進行學習，磨練演技。想要轉行進入演員行列。而從目前《慶余年》的表現來看。似乎學有所成。

　　郭麒麟曾在節目中說過，「我可以一輩子都不如我爸，但是我要一輩子都為我喜歡的事業而奮鬥，就是想讓觀眾朋友們認識認識，他不僅是個相聲演員，他更是一個演員。」
