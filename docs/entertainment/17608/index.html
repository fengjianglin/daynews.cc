《雪中悍刀行》確認由王倦改編，劇本可以放心，男主選角卻辣眼？

2019-12-17 1,496條評論 3,064次閱讀 1979人點贊

近幾年來，網友大IP改編劇已經成為了現在電視劇的流行。這樣的益處其實很大，一來大IP本身就帶有巨大的流量收益，以及粉絲群體。所以作為一部改編劇，不管做的怎麼樣，總有網友沖著IP過去看。

二來是因為有優秀的原著的基礎，很多國產劇的編劇就輕鬆了不少，只需要在原著主線上進行一些適當的修改就好了。

這樣又省編劇心力，有本身帶有流量的作品，能成為主流也不意外。不過照IP改編劇這些年的發展趨勢來看，越來越多的劇組仗著原著粉絲多，就開始隨便對自己的劇本動手。

《雪中悍刀行》主演曝光

什麼魔改情節啊，換人設啊，或者將主線改的一團亂遭。

本來原著是很優秀的作品，但是在那些編劇的手裡，就統統面目全非了。

所以現在大家對於改編劇，真的口碑也是褒貶不一。要是遇上好編劇了，那就皆大歡喜。要是沒遇上好的編劇，那完全就是毀原著。除此之外，演員也是很重要的一點，要是演員不好，那估計也得毀原著。

而最近，就有人曝出了《雪中悍刀行》這部IP改編劇的演員表了。

關於《雪中悍刀行》這部小說，或許有的書粉應該知道，這部小說是烽火戲諸侯的經典之作，也是最好看的一部小說。而這個大IP，早在2016年的時候就開始備案了，據說製作周期兩年。

不過備案上雖然是這麼寫，但是直到現在19年年底了，也沒有看見官宣。但是還是有很多人期待，其中大家最期待飾演主演的幾位演員，大概就是陳坤、張若昀、胡歌等人。

李光潔將飾演徐鳳年

然而最近，有網友卻發現，在《雪中悍刀行》的豆瓣頁面上，可以看到演員表上寫著白冰和李光潔。照這麼說，那這部劇的主演，很可能是白冰和李光潔了。

說實話……

這個陣容，真的不太行了。先不說女主白冰，就說這個男主李光潔，是認真的嗎？

在這部劇的里，男主徐鳳年的人設是少年世子，李光潔都38了，到底哪裡少年了？雖然說李光潔年輕的時候很帥，現在顏值也一直不錯，但是真的和少年搭不上邊啊。

我估計書粉們看見和這個選角，估計都要崩潰了。

唯一的安慰，可能就是這部劇的編劇是王倦大大了吧。王倦這個名字應該也有好一部分的人認識，不認識的也不用著急，小編列舉幾個他近年來的作品，大家就知道了。

《雪中悍刀行》編劇王倦

《大宋少年志》、《從前有座靈劍山》、《慶余年》……

這些都是王倦改編的作品。看到這些作品，是不是覺得《雪中悍刀行》的劇本，至少可以放心了？

講道理這也就是唯一的安慰了，要是男主真的是李光潔來演，那原著粉估計真的要爆炸了。不是說他的演技不少，但是老黃瓜刷綠漆什麼的，真的不行啊。

今年的39歲的董潔因為飾演少女被網友瘋狂吐槽，38歲的李光潔難道也要走後路嗎？他要是頂著一張中年感滿滿的臉來飾演少年，那真的估計都沒人能注意到劇本了。

光吐槽李光潔了。

小編還是希望，這個消息是假的吧，畢竟還沒有官宣。

可是小編看不僅豆瓣，哪怕是271上，演職員表也是寫著李光潔……

這真的有點虐。

明明娛樂圈裡有很多更適合的演員，為什麼非要找一個38歲的中年大叔。李光潔要是飾演一個更適合他年紀和形象的角色，可能對他來說，才能出彩。

不知道你們怎麼覺得？

還有，你們心目中，徐鳳年應該由誰來飾演呢？
