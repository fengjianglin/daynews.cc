重溫｜12000小時製作女主一件禮服，它提名奧斯卡獲全球票房TOP16

2019-12-23 115條評論 1,027次閱讀 84人點贊

流傳了3個世紀，為許多人所耳熟能詳，這個美麗的童話故事從圖書到被製作成電影，被多次搬上熒幕展示給大眾。

2017年，真人版電影的華麗上映更是引發一片好評，它就是提名多個獎項的《美女與野獸》。

影片上映後，3天票房破3億，如今的全球票房已達12.6億美元，在2019年全球電影票房排行榜中位居第16名。

這是一部由動畫改編而來的電影，披著野獸外表的王子和一個美麗善良的姑娘相愛，這樣的情節堪稱經典。

2017年版的《美女與野獸》不僅請來了《哈利波特》里的主演赫敏——艾瑪·沃特森擔當女主角，加入的3D特效更是讓這部影片的場景美得如夢似幻。

製作精良以及影片在人物情感處理上的細膩，讓人明白這部電影為什麼有能力讓人眼前一亮。

看這樣的電影是一種美的享受，不僅讓人產生更深的沉浸式體驗感，對故事中正反派人物的三觀也會有更豐富的理解。

王子居住的古城堡宏偉壯觀，高高聳立在雪白的天地間，隱藏在森林深處。因為被施了魔咒，雪花和寒冷光臨了這裡，城堡前是沿著樹藤蜿蜒生長的玫瑰，潔白、清冷而優雅。

圖片來自《美女與野獸》預告片

城堡厚重的木製大門隔開了外面的天地，寬敞的大廳，精緻的傢具，無一不透露出古堡的莊重和華貴。

真是該美的地方美，片方毫不吝嗇地用高科技為故事打造了炫目的場景秀。除此之外，電影中還有兩個片段讓人印象深刻。

女主角貝兒為了救父親甘願留在古堡再也無法離開，王子僕人變換成的傢具們為貝兒精心準備了一頓晚餐歡迎她的入駐。

古代貴族宴會裡常出場的長條式餐桌被鋪上了桌布，貝兒被邀請在一側入座。

燭台僕人揮舞著兩手燭光蹦跳在長桌之上，鋼琴僕人彈起叮咚的旋律……杯盤羹匙成為餐桌歌舞劇的主角，碟子在空中快速旋轉組合變換出不同的造型，明亮耀眼的煙花在空中綻放，各色精美食材被一一展示。

整個場景絢麗到令人眼花繚亂，猶如一個巨大的萬花筒在熒屏上被立體放大了數倍，場面之熱鬧讓觀影之人目不暇接，嘆為觀止。

影片第二個令人難忘的場景發生在王子和貝兒的情感升溫之時，野獸王子邀請貝兒參加舞會，衣櫃僕人這回非常認真地為貝兒準備了一套禮服。

溫軟的鵝黃色露肩禮服襯出貝兒的優雅與美麗，衣櫃僕人想要為貝兒的裙角添上星光，於是廳堂之上鎏金木枝的金緩緩脫落，在空中點點匯聚成線，拼接成花紋落在了貝兒的裙角，轉動裙擺便見光華流轉，低調而華美。

其實貝兒的這件禮服真的花了大手筆，180英尺羽毛般輕盈的綢緞歐根紗，2100多顆施華洛世奇水晶，共耗時12000小時才得以完成，也因此才能打造出如此絢麗的效果。

毫不意外的，這部電影被提名90屆奧斯卡金像獎#最佳服裝設計#獎項。

貝兒本身就是一個喜愛讀書的姑娘，而王子擁有一個巨大無比的書房，藏書豐富甚至需要借用移動的木梯才能夠得著高處的書籍。

當他帶貝兒走進偌大的書房時，貝兒既震驚又欣喜。野獸王子酷酷地說：「你喜歡的話，那送你好了」，恩，無意間又讓貝兒對野獸王子的好感倍增。

此後他們在不同地方一同看書，在溫暖的室內、在潔白的玫瑰花群之下、在小拱橋之上，貝兒在冰天雪地的美景里一句句為野獸王子念出那些令人心動的話。

他們擁有相同的愛好，一如古人「賭書消得潑茶香」的生活，讓內心平靜而歲月靜好，我想他們就算就此在這裡生活下去，也沒什麼不可以。

故事中對野獸王子的設定是不懂愛的人，所以在一位貌丑的老婆婆投宿時拒絕提供幫助，才被施魔咒成為了野獸，而僕人皆成了會說話的傢具。

野獸王子擁有著很鮮明的性格色彩，察覺貝兒父親摘了玫瑰花時異常暴怒，在看到貝兒願意就此被束縛而不願父親受苦時的驚訝與心起波瀾，再到與貝兒相處之後顯現出的溫柔，可愛，純真，剋制，真誠，讓觀者感受到了他的特別。

也真的是一物降一物，野獸王子在僕人的印象里可並不熱絡，至少表象上不是一個容易親近的人，但貝兒可不怕他，她吼他！懟他！凶他！可以說毫不客氣，可是野獸王子拿他沒辦法，在貝兒有理有據的「怒吼」之下，他不服氣也只能是不服氣。

野獸的外貌很是彪悍高大，撇嘴的小動作卻分外可愛，還會做些常人會做的搞怪小表情，實在讓人忍俊不禁。

捧著一碗湯，埋頭就搖頭晃腦地呼嚕呼嚕喝得專心，待到一抬頭嘴唇上沾滿了湯汁，卻愣是毫不在意，不以為然的萌蠢萌蠢感，將人物純真的形象刻畫地自然又飽滿。

故事後半段，貝兒要暫時返回村子解救父親，野獸王子以為她將永遠離開他的世界，雖然萬分不舍卻還是隱忍地看著心愛的女孩離開，沒有請求她回來，只難過地在高樓之上以歌聲表示願意永遠等候的心情。

故事最大的矛盾爆發，村民在狠辣的大壞蛋加斯頓有意誤導之下，相信森林深處的野獸王子會危及個人利益，於是群情激憤紛紛拿好武器想要消滅腦海中錯誤放大的假想敵。

但他們忘了，在此之前他們甚至不知道森林裡有古堡，更不用提及真實存在的野獸從來就不曾攻擊過村民。

這也側面反映出一個人性弱點：

大部分人在正常情況下對周遭的態度都相當冷漠，而一旦事情關乎自身利益時腦子就容易短路，被有心之人稍作引導一言不合就大動干戈，就怕其他人的言行會損害到個人利益。

矛盾的高潮，加斯頓趁機向野獸王子暗下黑手使他身受重傷，野獸王子禁不住發怒了，加斯頓連聲求饒。

野獸王子最終選擇放了他，但一個壞蛋既然拋棄良善，他的本性也沒那麼容易在瞬時就發生改變，所以毫不意外地他趁野獸王子不注意時還是朝他補了一槍，最後自食其果墜下古堡。

野獸王子和勇敢善良的貝兒幸福地生活在了一起，作為看客的我又被無情地灑了一把狗糧。

「王子和公主幸福的生活在了一起」，《美女與野獸》的結局和許多童話的結尾一樣，他們在一起之後的生活到底會什麼樣子，日常相處會和其它情侶一樣嗎？

這可能不是大多數人關心的重點，或者說，其實通過那些跌宕起伏的情節，我們其實也知道，每一份想要收穫幸福的愛情，都需要雙方充分了解，通過相處知曉各自的喜好、品性、脾氣……經受住磨合期，也才有可能遇見真正的幸福。

希望更多人能遇到生命中「王子」和「公主」。
