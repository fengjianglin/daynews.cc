2019娛樂圈十大喜事：肖戰王一博上榜，奚夢瑤趙麗穎“母因子貴”

2019-12-22 921條評論 2,865次閱讀 335人點贊

文/殷素素
2019年已經接近尾聲，今年國內發生的大喜事似乎比往常更多。

從85後一眾小花順利孕育了新的生命，迎來了新的頭銜，成功獲得”新手媽媽”的稱號，到娛樂圈中不同藝人感情的分分合合。

儘管許多網友都在吐槽”2019年到底為什麼這麼不順”，但其實娛樂圈中的大喜事依舊不少，除了每件喜事能夠讓許多網友”同樂”，更希望每件事情在發生後都能給網友們感悟和思考。

「伍脊六獸編輯部」通過對2019全年大事件進行了梳理，整理出一份「華語娛樂圈2019十大喜事」，敬請小夥伴們查收。

NO.1 張若昀唐藝昕舉辦婚禮

娛樂圈中天作之合的情侶不少，但是像張若昀和唐藝昕這樣甜蜜戀愛許久，在經歷了九年戀愛長跑後修成正果的其實不多。像兩人低調戀愛一樣，張若昀和唐藝昕的婚禮也十分低調，只是邀請了幾位圈中好友作為伴娘伴郎，並未邀請任何媒體前往現場。兩人的戀情可以說是”全網皆知”，從改名字到情侶紋身，兩人的戀情從只有粉絲知道到全網皆知，儘管低調卻發了許多”暗糖”。

事實上，張若昀和唐藝昕能夠獲得祝福的很大一部分原因還是與兩人在大家眼裡是一對金童玉女有關。張若昀終於娶到了當初動態中提到的”便利貼”女孩，唐藝昕也終於嫁給了自己心愛的那個男孩。在唐藝昕參與《我們來了》前不久，張若昀霸氣公布兩人關係，既體諒到唐藝昕在節目中很有可能被問及兩人感情不好作答，又考慮到女生都希望獲得的安全感。”時光賜給我們盜不走的愛人，而你賜給我時光”和”就這樣笑一輩子吧”也被評為公布戀情和婚訊的最佳文案，希望張若昀和唐藝昕能夠在未來也攜手共度，就這樣相互依偎一輩子。

NO.2 應采兒二胎

就在12月7號，陳小春在自己的報道演唱會上親自公布了應采兒懷孕的消息，他的激動之情溢於言表：”我有第二個孩子了，但今天剛好是三個月，是男是女我還不知道，感謝我老婆。”其實早在之前就有許多傳言稱應采兒已經懷孕，但此次陳小春親自”闢謠”的同時公布了這個喜訊，大家也都紛紛恭喜。除了陳小春和應采兒即將迎接一個新生命外，Jasper也有了新的身份，相信懂事貼心的Jasper能夠在未來充分詮釋”好哥哥”這個詞。

在公布了這個喜訊後沒多久應采兒就發布了”一家四口”的合照，儘管照片中只有三個人，但溫馨的氛圍透過照片也能夠讓人感受到。應采兒在配文中稱：”台北演唱會，最終還是順利完成了。”似乎既是對公布二胎作了進一步的說明，同時也是安慰許多擔心的粉絲們。陳小春對於Jasper保護得很好，儘管兩父子因為出演親子綜藝走紅，但是陳小春卻並無讓Jasper進入娛樂圈的意思。希望應采兒能夠順利完成生產，也期待Jasper第一次”扮演”哥哥這個角色。

NO.3 撒貝南喜獲龍鳳胎

2019年娛樂圈中懷孕生子的明星確實不少，撒貝南與妻子李白共同孕育的結晶卻受到廣泛關注。雖然撒貝南並未完全進入娛樂圈，但事實上，因為出演《明星大偵探》撒貝南”吸粉”無數，也因此有了”芳心縱火犯”這個稱號。此次喜獲龍鳳胎，幾乎全網都在向撒老師獻上自己的祝福，這也從側面展現了撒貝南的獨有的人格魅力。

提到撒貝南不得不提的另一個女藝人就是章子怡，兩人曾經結緣並戀愛，儘管兩人最終選擇了分開，但是最後雙方各自找到了新的歸宿，也收穫了新的幸福。當被問及為什麼沒有向大眾公布這個好消息時，撒貝南表示，這只是自己的家事，不希望佔用太多公共資源，所以只是享受著屬於自己的小幸福。儘管因為綜藝走紅，撒貝南卻並沒有忘記自己作為主持人的本分，比起一些因為小事頻登熱搜的明星，撒貝南確實值得點贊。

NO.4 林志玲黑澤良平結婚

2019年的喜事一樁接一樁，林志玲作為著名主持人其實感情方面一直沒有什麼消息，但就在今年6月，林志玲直接在個人社交平台丟出一顆”重磅炸彈”，直稱自己已結婚，並曬出了兩人的合影。似乎是因為男方的顏值有些遜色，所以對於林志玲的婚姻，很多人都抱著不看好的態度，儘管收到了圈內許多明星的祝福，但是在網友眼中，林志玲與丈夫黑澤良平結婚是”下嫁”，兩人遲早會有分開的那一天。

但事實上兩人不僅沒有任何分開的跡象，反而如膠似漆。值得一提的是，在今年11月上旬，兩人簡單的完成了重要的婚禮儀式，比起黃曉明和baby花費數億、周杰倫和昆凌寶下古堡，林志玲與丈夫的婚禮確實”寒酸”得多。兩人低調結婚並舉辦婚禮，雖然沒有大張旗鼓，但是婚禮現場卻依舊很久質感。作為夫妻，重要的並不是婚禮價錢的昂貴，而是兩個人的真心。低調也未嘗不是一件好事。只要兩個人能”白首不相離”，無論是一擲千金還是平平淡淡，兩個人都能夠收穫屬於自己的幸福。

NO.5 奚夢瑤生子

10月24日下午，嫁入豪門的超模奚夢瑤誕下一子，何猷君發文宣布了這個喜訊，同時也公布了孩子的名字”何廣燊”。作為2019「母因子貴」的典型，從何猷君的文字中不難看出，他對於自己的妻子奚夢瑤其實十分感謝，也體會到了世界上所有媽媽生子的辛苦。何猷君還同時解釋了自己兒子姓名中所包含的深意：”希望他能夠像爺爺一樣，成為一個有益於社會的傳奇人物”，同時他還對自己的兒子寄予厚望，這確實是初為人父的難以抑制的喜悅。

隨後奚夢瑤也幸福的轉發了何猷君發布的動態，甜蜜溫暖的稱自己的孩子為”小王子”，可以說不論是奚夢瑤還是何猷君，對於兩人愛情的第一個結晶都十分重視，對孩子獻上了自己全部的愛並寄予了自己的厚望。何廣燊作為長孫，自然會受盡萬千寵愛，所以就更加希望他能夠在好的教育環境下成為一個對社會有益的人。

不過說起何猷君和奚夢瑤這樁婚姻卻是也很巧，兩人因為一檔綜藝結緣，雖然第一次見面略有些許尷尬，但是兩人卻依舊因為有著相同的三觀走到了一起，儘管外界對於兩人的結合評價頗多。但只要能夠守住自己的小幸福，無論外界再多風言風語都可以不必理會。

NO.6 趙麗穎馮紹峰生子

馮紹峰和趙麗穎兩人在去年女方生日10月16日10時07分”官宣”結婚，兩個字一張合照就充分展現了兩人的甜蜜。因為之前有過多次作品上的合作，所以評論里也有很多網友調侃稱：”唐僧終於還是娶了女兒國國王啊”，”二叔和明蘭要一直白頭偕老”，雖然網友們調侃不停，但是對於兩人的愛情大家還是十分羨慕，直稱兩人喜結連理是”天作之合”。

在今年1月1日，馮紹峰”官宣”結婚後首次暗示愛妻趙麗穎已經有孕，一條文案：”知否知否，應是一家三口”一下引發許多猜測，趙麗穎也立刻轉發回應道：”等候等候，恭迎2019″，雖然兩人用十分輕鬆的文字分享了自己的快樂，但是作為孕婦，趙麗穎應該也很有壓力。好在3月初馮紹峰就再次公布了喜訊，曬齣兒子的小腳印。趙麗穎順利產子，家裡又多了一位男子漢一起守護偉大的媽媽。

在大家印象中，趙麗穎似乎還是青澀可愛的模樣，不過短短几年卻已經為人妻為人母，肩上的責任又更重了，所以儘管很多人期待趙麗穎復工，但是卻沒有人催促，只是希望趙麗穎能夠在產後享受一段輕鬆的時光，也給趙麗穎調整狀態回歸做足準備。所以趙麗穎不負眾望，回歸後立刻開工投入拍攝，期待有了新身份的趙麗穎在演員這條路上也能越走越順，越走越遠。

NO.7 郎朗吉娜結婚

今年除了林志玲何猷君外，還有一對新人受到廣泛的關注。事實上郎朗與吉娜在最開始並不是因為婚姻甜蜜被人羨慕，反倒有許多網友看見機場吉娜大包小包而郎朗兩手空空照片，對郎朗大肆批判。儘管郎朗在動態中不斷用調侃的方式回應，但依舊沒能獲得網友們的”原諒”，網友反倒認為郎朗是在作秀。

不過在兩人登上了一檔綜藝後，風評逆風翻盤，在《幸福三重奏》中，郎朗和吉娜迎來了獨處的時間和空間，同樣兩人的感情升溫，受到的羨慕之前也越來越多。因為相處十分自然，兩人撒糖也不在乎場地和時間，郎朗撅著嘴向吉娜撒嬌的樣子也成了大家心中最羨慕的模樣，從不受認可到全網羨慕，兩人用細節互相溫暖，也用細節讓大家看到了愛情最真實也最美的樣子。

NO.8 楊丞琳承認與李榮浩領證結婚

李榮浩和楊丞琳其實也是圈內經歷了愛情長跑的一對戀人，最終能夠步入婚姻殿堂也是兩人愛情最美好的結局。兩人自2015年首度公開戀情後就十分”高調”，演唱會上熱吻也十分常見，對於表達對楊丞琳的愛，李榮浩從不吝嗇，反倒大方自然。2019年明星們公布結婚的方式似乎更加直接，曬出結婚證合照，選擇合適的時間點就已經足夠。楊丞琳與李榮浩官宣的時間為”9191520″，似乎也是取了”就要就要我愛你”的諧音，可以說是別出心裁。

兩人既然已經領證，關於婚禮的安排大家應該也十分好奇，只不過據楊丞琳透露，婚禮並不會在寶島舉行，也不會有世紀婚禮和海島婚禮，似乎是想獨享自己的幸福。明星們在公布戀情後容易遭受網友催婚，而在公布婚情後又會被網友們安排”生子”。但楊丞琳透露稱沒有生孩子的計劃，所以真心的祝福楊丞琳和李榮浩能夠在在未來的每一天中都甜甜蜜蜜的度過。

NO.9 肖戰王一博《陳情令》入圍全球50大電視劇

某知名社交平台2019年的全球50大電視劇榜單中，《陳情令》成功上榜，作為在榜唯一的一部國劇確實讓人驚喜。儘管是網劇、受眾範圍略小，但《陳情令》依舊憑藉演員的精湛演技和快節奏的劇情”出圈”，成功領跑2019年的夏天。事實上，在電視劇播出前，大家對於這樣一部全憑新人演員演繹的影視劇並不看好，讓人沒想到的是，《陳情令》一舉突圍，餘韻悠長。國劇衝上全球榜單，著實可喜可賀，我們將肖戰王一博列入這份榜單也正因為如此。

《陳情令》的兩位主演肖戰和王一博都斬獲了泰國頭條新聞年度風雲人物，電視劇更實在各個國家熱播，可以說是走出國門，成功的對外輸出了中國的文化和氣韻。雖然王一博和肖戰兩人都是第一次與對方合作，因為兩人之間默契很足的緣故，所以也讓觀眾產生了很強的代入感。能夠用小成本精準”出圈”甚至走出國門，這確實是值得欣喜的一件大事。

NO.10 向佐郭碧婷結婚

2019年娛樂圈中喜結連理的新人確實不少，經歷了多年戀愛長跑最終修成正果的也很多，但郭碧婷和向佐兩人從戀愛開始就十分美好。向佐在接受採訪時回憶兩人相遇的情景也十分碰巧，因為紳士的扶住了差點摔倒的郭碧婷，兩人的緣分也就此展開。因為郭碧婷從小就很怕生，所以也更喜歡一個人呆著，向佐能夠走進郭碧婷的心對於郭碧婷來說也不是一件壞事，從此之後身邊就有一個人可以依靠，自己也不用一個人承擔太大的壓力。

從《女兒們的戀愛》到《我家小兩口》，向佐和郭碧婷之間的相處模式越來越契合，儘管仍然會有觀念不合導致的矛盾，但是兩人也能夠共同面對。在向佐和郭碧婷的戀愛和婚姻中，向太也是十分重要的一個”助推力”，為了減輕郭碧婷的壓力，向太對郭碧婷就像對待自己的親生女兒，”婆媳矛盾”就更是不存在了。郭碧婷婚禮當天頭戴的王冠價值不菲，向太送給郭碧婷的訂婚戒指價值更是在五百萬以上，這樣的小細節應該更能讓郭碧婷安心，也更能讓郭碧婷更好的融入向家的生活。沒有婆媳矛盾，沒有夫妻吵架，郭碧婷和向佐兩人的婚姻確實是大家心目中最美好的樣子。

​2019即將結束，雖然遺憾不少但是欣喜和收穫更多。希望在2019年收穫幸福的各位繼續前行，也希望稍有遺憾的大家能夠重燃對2020的期許。
