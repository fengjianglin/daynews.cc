除了明家大姐，還有哪個女人壓得住靳東

2019-12-24 2,971條評論 4,197次閱讀 525人點贊

《精英律師》殺青那天，靳東發了條微博，一口氣艾特了50位圈內朋友，陣仗頗大。

這一次，靳東不僅是主演，還擔任了聯合出品人，為了這部劇沒少下血本，還親自出面找了一大票朋友來友情出演。

發布會那天，靳東站在三個女人面前，喜滋滋地講述著羅大律師的煩惱，「有一個不聽話的助理菜鳥律師，一個生命時時刻刻都在等待羅律的秘書，還有一個永遠都虧欠她的前女友，三個女人一台戲，就好好看戲吧。」

這時候王鷗打斷他，「最重要的是，你還不止三個女人啊。」靳東趕緊補了一句，「在這還要特別感謝沒來的袁泉和蔣欣，也是和我有一點點感情糾葛的兩位女演員。」

怎麼說呢，靳東的女人緣是真的好。一招手，就招來了好多個女演員給自己作配，藍盈瑩朱珠王鷗袁泉蔣欣劉敏濤左小青吳越，個個都是盤亮條順的大美女，全都圍著他團團轉，然後靳東還要假裝苦惱地揮揮手，哎呀，女人就是事多，愁人！

嚯，好深沉的一個中年男子瑪麗蘇夢想，令我懷疑這劇到底講的是精英律師，還是精英PUA。也許普通男人與精英男人的區別，就是你能同時吸引多少個女人？

當然，靳東會告訴你，戀愛戲只是前菜點綴，律法大戲才是正餐，既然title是律師，怎麼也得講點事業心。據導演劉進說，《精英律師》在司法部拿了將近1000個結過案的案子，經過專業律師團隊的甄選和討論，將在故事裡以單元案件敘述方式逐一呈現。

這說明，劇組多多少少也是做了功課的。本著不埋沒一部好劇，也不放過一部爛劇的初衷，我打著眼藥水點滴追了五集劇。怎麼港呢，案子沒看出有什麼新意，倒是這故事結構，頗像美劇《金裝律師》（《Suits》）啊（不確定到底有沒有買版權）。

《金裝律師》里有一個很經典的CP搭配。一個自負聰明的精英律師Harvey，偶然錄用了一個法律系輟學生Mike，兩個人組成了老炮兒帶菜鳥的黃金搭檔，在律師事務所中無往不勝。同時，律所里還有橫亘在中間搞事情的心機律師Louis，美貌又善解人意的律師助理Donna，男男女女的感情線穿插其中。

再看《精英律師》，靳東演的羅彬，人設跟Harvey差不多，作為律所的王牌大狀，話語權相當高，所有大案子都得交給他打理。《慶余年》里穿過來的王啟年，演的何賽，其實就是Louis那個角色，作為律所老二，嫉妒羅檳嫉妒得發瘋。再來，朱珠演的女秘書栗娜，跟Donna簡直就是一個模子刻出來的。

所有人連名字都透出一股譯制腔的味道，律所辦公室的布景都差不了太多，唯一的不同是，美劇里的小菜鳥Mike，到了這部劇里，性轉成了藍盈瑩演的戴曦。也是一個精通法律條文卻入錯了行的新人。

如果是老老實實的翻拍也就算了，日本和韓國都翻拍過《金裝律師》，拍出來也很好看。偏偏到了國內這一版，硬是把男男cp拆成了男女cp，整個都走樣了。

第一集開頭靳東出場，還是那股熟悉的逼味，逼格的逼。只見他演的羅檳律師，梳著大背頭戴著墨鏡，穿著一身筆挺西裝從豪車下來，邁著從容自信的步伐緩緩走入上海CBD的摩天高樓。

在電梯里，他撞見了藍盈瑩演的咖啡店外賣員戴曦。因為人員超載，羅檳勸戴曦乘一班電梯，戴曦不願意。這時候，羅檳遞給她一張名片說，「如果你因此丟了工作，你可以來找我，我可以讓你老闆賠你一大筆錢。」

戴曦一頭霧水地走下了這班高貴的電梯。我懷疑羅檳這種精英，可能都是堂食，從來沒有叫過外賣。一單外賣超時最多是打個差評影響績效，能找誰打官司去？

彈幕都在吐槽，明明羅律師最後一個上電梯，卻讓送外賣的先下去，這是不是赤裸裸的職業歧視？

這個戴曦，也很讓人無語。雖然沒有參加司法考試，但是她喜歡替所有人寫狀紙。

外賣小哥父親受了工傷她要管，朋友設計作品被侵權她要管，羅檳姐姐被公司辭退她要管，自己出不了庭就讓律師男朋友免費出面打官司。太感人了，這是專門替人申冤的狀王宋世傑轉世啊，宋世傑都沒你這麼無私啊。

作為高級律所的頭牌，羅檳願意聘用這個只有本科學歷也沒有通過司法考試的宋世傑，根本說不過去。

戴曦有什麼過人之處嗎？劇里展示出來的天賦，就只有一個，那就是背書，無窮無盡地背書,替人起訴要背書，威脅領導要背書，爭取職位還是要背書。

何賽考戴曦行政訴訟法那一段，戴曦就端端正正站在那，一字不差地複述著課文，這像不像小學班主任辦公室里守著背課文的你？請問這是應試教育的宣傳片嗎？

藍盈瑩也是可憐啊，每次背台詞就要背兩個禮拜，吃飯背、睡覺背、走路背，有一次已經背到倒數第二句了，腦海里突然閃過一句「藍盈瑩你好棒」，結果最後一句就忘了。確實是忘了，戴曦背訴訟法這一段，明顯能看出是配音，演員跟台詞的口型都對不上。

感覺我國編劇始終對職場劇有什麼誤解，每次要展現職業素養的時候，就只能大段大段地掉書袋，還是那種極其粗糙原始冗長的書袋，演員背得奄奄一息，觀眾也聽得奄奄一息。

看這劇的時候，最怕就是空氣突然安靜，藍盈瑩突然開始背書，她簡直就像一個永不斷電的天貓精靈，一聽到關鍵詞就會毫無預警地開始嘚啵嘚。

細節才是建立一個故事可信度的關鍵。

美劇里，Mike闖入律所面試那一段就拍得很有意思。自大的Harvey想招一個助理，沒有別的條件，就是「像我一樣聰明的」，讓Donna替他先酒店面試一輪。

另一邊，法律肄業生Mike，為了給奶奶買葯，鋌而走險去販毒。他穿了一身西裝拎著一箱毒品來到同一間酒店，卻發現交易的房間門口蹲著奇怪的服務生和經理。

Mike注意到，服務生沒有戴手套，於是走過去詢問他，「我想去游泳，這間酒店的游泳設施還可以吧？」服務員回答他，當然。這時候Mike腦里閃過他看到的告示牌，今天游泳池關閉。

他又轉頭問經理，「請問現在幾點了？」經理一抬手，露出了腰間的槍。這時候Mike確信，這兩個人是警察。他腦里又閃過看到的有關面試的信息，於是闖入了Harvey的面試房間。

Harvey知道原委後，先是婉拒了他，說我們只招哈佛畢業生。Mike讓Harvey考了他一題，又反過來考了Harvey一題，Harvey漏掉了一個重要法案，Mike一邊玩著紅心接龍一邊告訴他，你輸了。

沒有大段大段地背書，也沒有亂七八糟的感情線，就這麼幾個細節，展現出了Mike的特質：過目不忘的記憶力，機警的分析能力，超高的智商還有過人的膽識。

而Harvey願意冒險錄用一個沒資格的菜鳥，也說得過去，自負聰明的他，很難接受這世界上還有比自己更聰明的人。後來Harvey也想過要解僱Mike，因為他是個人利益至上的精明人，但Mike也反過來威脅他，你解僱了我，全世界都會知道這件事。

再回過頭來看，羅檳與戴曦這一對律師搭檔，就非常奇怪了。

如果說羅檳當初是為了官司和解才僱傭了戴曦，但後來他居然為了成全戴曦的聖母心不惜得罪公司大客戶，這很不符合律師的職業邏輯。你要說他們是惺惺相惜互相欣賞的知己吧，戴曦到處管閑事還丟掉公司大單這德性，也把羅檳氣得夠嗆。

兩個人唯一的紐帶，大概就是劇里一直強調的，假大空的正義和善良。

為什麼非要把Mike這個開掛的男性角色改成一個傻白甜的女性呢？金裝律師這一對搭檔之所以好看，是因為兩個人存有新鮮的戲劇張力，一個老世故和一個小滑頭，彼此之間既有同性老少之間的智慧角力，也有男人與男孩之間的相互關照。

早在幾年前，有B站阿婆主提名過國內《金裝律師》的理想人選，靳東可以演Harvey，而王凱可以演Mike，想一想，還真的挺帶感的。

就在靳東說出「一切不男不女、非爺們的人事,都無意接觸」之後，大概也是徹底拒了男男cp這一條路。寧願往劇里添無數個女配角，也不願意再多一個男主角。

靳東也演慣了一個絕對強勢的男主角，明樓也好，老譚也好，賀涵也好，庄恕也好，這些角色都掛著同一張仿若洞悉一切的精英面孔，動輒就要語重心長給女主講大道理，頓句、點頭之間，都有一種「長兄如父」的大家長感覺。

馬伊琍、江疏影這樣的大女主，跟靳東搭戲的時候，都變成了乖乖被牽著走的小女孩，找工作要他幫忙，情感問題要他幫忙，在他面前總是無知又無助，點頭如搗蒜。

白百何白姐要勇敢一些，至少在他講道理的時候，敢於衝上去一吻堵住他的嘴。

這部戲的藍盈瑩也是一樣，甭管記憶再好天賦再高，被靳東一吼，就一哆嗦，躲廁所哭去了。估計後面兩個人打著打著官司，就會開始談戀愛。畢竟靳東真的是太愛那些乖乖聽他佈道的小女孩了。

稍微能跟靳東抗衡一下的，也就是大姐劉敏濤了。要扇巴掌就扇巴掌，要他跪下也只能噗通一聲跪下，《精英律師》里演的也是靳東的上級。但大姐代表的，始終是大家長的集權，而不是平等的伴侶關係。

《偽裝者》之後，王凱演了土裡土氣的農村學生宋運濤，胡歌演了狼狽逃命的悍匪周澤農只有靳東，依然脫不下那身西裝，拋棄不了瀟洒的派頭。看膩了靳東千篇一律的精英嘴臉，倒是有點懷念他當年在《闖關東》里演的那個龜田一郎，害羞，敏感，還有點悲情，心裡惦記著嫂嫂，那蠢蠢欲動的愛與欲啊，可比一絲不苟的刻板西裝好看多了。

只是不知道現在這個強勢慣了的靳東，還回得去嗎？
