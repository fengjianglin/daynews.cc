湖南衛視寶貝有戲《搭錯車》何羿霏再現經典《酒干倘賣無》

2019-12-25 170條評論 2,435次閱讀 819人點贊

國內一檔兒童競演類綜藝節目、每周五晚8點湖南衛視芒果TV獨播的 「寶貝有戲」火熱播出中，第10期劇目里12歲的劉銘澤飾演啞父、11歲的何羿霏飾演阿美，共同聯手上演的《搭錯車》令明星導師現場熱淚盈眶。

何羿霏《搭錯車》 飾演阿美

上世紀70年代，天性善良的啞父收養了撿來的女嬰，取名阿美，啞父以收破爛為生，獨自撫養阿美長大含辛茹苦養育撿來的女兒阿美，轉眼阿美長大有機會去成為一名歌星，啞父不同意，父女爭吵之下，啞父動手打了阿美……

何羿霏劉銘澤《搭錯車》

《搭錯車》無法忘記那一幕幕:靠收舊酒瓶、傾注一生心血拉扯大養女的啞叔，帶著無比自豪的笑容，傻傻穿上一輩子唯一的一套西裝，去參加女兒成為明星的慶功宴，他只希望分享那份喜悅，卻讓殘酷的現實擊得遍體鱗傷，保安將他趕了出去。是啊，一個收舊酒瓶、一個只會用玻璃瓶敲「酒干倘賣無」調兒的啞巴。《酒干倘賣無》震撼人心的弦律感人至深的故事情景，隱藏不住眼中心中感動的淚水。

何羿霏演唱《酒干倘賣無》

多麼熟悉的聲音，陪我多少年風和雨，從來不需要想起，永遠也不會忘記……沒有天哪有地?沒有地哪有家?沒有家哪有你?沒有你哪有我……何羿霏現場深情演唱《酒干倘賣無》空靈的聲音，看到這種心碎的畫面融入其中，在聽覺與視覺的細膩結合中重現故事中感動揪心的真情，詮釋毫無保留的人間父愛。

何羿霏演唱《酒干倘賣無》

劉銘澤何羿霏《搭錯車》

《搭錯車》劇目排練時間2,3次，劇中啞父生氣動手打啊美，演員為真情融入劇情演繹也是狠下心真打，看得網友直喊心疼。

劉銘澤《搭錯車》飾啞父

寶貝有戲兒童版《搭錯車》劇目唱演結合，這份滲透在骨血里的親情悄悄抵觸我們心底最柔軟的地方，演繹牽動心弦，首部兒童版《搭錯車》再現經典摯愛人間平凡的親情，看了第10期芒果TV 兒童版《搭錯車》感動到你了嗎？
