造型師是跟她有仇嗎？還我文藝女神啊

2019-12-20 94條評論 1,754次閱讀 624人點贊

最近娛樂圈的迷惑行為實在有點多。

鄭爽張恆上個戀愛節目之後分手了，遭殃的卻是張翰。鄭爽的粉絲去張翰上周發的微博下刷屏留言「快去把小仙女追回來！」，還長篇大論教育張翰，人這輩子還是要和初戀在一起……

《一起來看流星雨》都過去十年，兩人分手各自開始新戀情也都幾輪了，當年是小學生就算了，現在怎麼著也都接受完九年義務教育了吧，為什麼還有這麼幼稚的留言……

小學生沒長大，行為幼稚可以理解，但是作為成熟的演員，擁有成功的代表作，卻在新劇里還能展示出幼稚的演技，這就讓人更加迷惑了。

許多人都沒想到，被奉為文藝女神的湯唯，有一天會跟迷惑行為聯繫在一起。

早在《大明風華》預告片出來看見湯唯的造型後，大家頭皮和心臟都一緊。正片播出後，果然讓人眼前一黑：女神的台詞怎麼會介樣！原聲配音聽起來奇奇怪怪，而且還有許多迷之小動作？？

今年，湯唯有一部電影《吹哨人》上映，一部《大明風華》播出，但就目前看來，吹哨人口碑票房雙雙遭遇滑鐵盧，《大明風華》讓觀眾直觀看到了湯唯的演技短板。

所以說啊，讓演員保持神秘感，也是有道理的。在這兩部作品接連上映之前，湯唯並沒有連續產出過優秀的作品，可她也不經常出現在公眾視野，讓觀眾對她充滿了好奇。

但今年的連續曝光，卻接連被吐槽，不得不讓吃瓜群眾重新思考：曾經對女神演技的那些堅定與熱愛，終究是錯付了嗎？

——我是演技失望的分割線——

《大明風華》這部劇被明朝歷史愛好者們稱為「大明瘋話」，因為劇組雖號稱是「歷史正劇」，但才播出了短短四集，就被八出有許多地方與真實歷史出入太大。

劇中仿照被醜化的朱元璋畫像做了倒模，硬生生把一部歷史片變成了鬼片。

飾演朱元璋的演員明明是清清秀秀帥哥一枚，劇組你們在想什麼啊！

‍更不用說第二集就上演「女官相親記」，以及讓朱棣親口說出「靠在豬圈吃豬屎打天下」這樣奇奇怪怪的話。

這趴的問題該由歷史顧問負責，咱們就不多贅述了，還是來看看被稱為下凡演電視劇的湯唯，在其中表現如何吧。

當年拍《色戒》時，李安曾把章子怡和湯唯對比，說章子怡一看就很上相，怎麼拍都好看，但拍湯唯就需要一點技術。可惜的是，《大明風華》的導演和造型師，恰恰欠缺了這一點技術。

要麼是邋裡邋遢暴露髮際線缺點的流浪漢造型

要麼是讓人頭皮一緊巨顯臉大的戴帽「女扮男裝」造型。

加上時不時正面鏡頭直接懟臉

以及災難打光……

and迷之角度

朱瞻基（朱亞文飾）對孫若微（湯唯飾）說的這句話，也是觀眾的內心吶喊：這樣的湯唯，誰不是第一次見呢！！

造型敗了，可以靠演技挽回一局嗎？就目前看來，似乎不太可。

孫若微這個角色，目前呈現出來的性格相當分裂。參與刺殺行動時，是有勇氣有膽識的女子，可真到關鍵時刻，畏畏縮縮又變成了傻白甜。

拿著毒酒赴約，被朱瞻基三兩句話就給嚇得不輕。

湯唯表演「害怕」的方式也讓人一頭霧水，五官擰在一起帶著猙獰感，縮脖子可以理解，但是迷之頸前伸是為了表現啥啊？

孫若微和朱瞻基一起練習射箭，朱瞻基手扶著她的腰開始耍帥撩妹。換在其他男俊女美的劇里，觀眾早就嗷嗷大喊「我可以」。

但把皇子演成邪魅東廠太監的朱亞文and用力過猛把緊張演成痛苦的湯唯，就只能讓觀眾大呼：My eyes！my eyes！

湯唯在《大明風華》里的演技被說像《落跑甜心》的女主，是因為她呈現的方式似乎總像新人一樣僵硬，且與情緒不同步。

就拿射箭這段來說，能從表情看出來她內心對朱瞻基的貼近想法有很多，肢體也能感受到僵硬，但是台詞和眼神卻跟不上，所以導致整體情緒的呈現有一種脫節的怪異感。

和人理論要去救那些刺殺行動失敗後被捕的兄弟，扭曲的五官和眼神讓人感受到她的著急，但是肢體動作卻又放不開了。

湯唯自己彷彿都沒入戲，如何能讓觀眾跟著她一起入戲呢？

當然，這是湯唯第一次擔任女主角的電視劇，畢竟電視劇和電影的表演方式不同，在最近的採訪里，湯唯也認了這一點，稱感謝劇組對她的包容。

《大明風華》演技被吐槽可以說是水土不服，可今年湯唯主演的電影《吹哨人》也收穫了口碑票房雙撲街，豆瓣5.7分的成績連及格線都沒達到。

客觀來說，這電影是從劇本到拍攝手法都很爛，湯唯演的人設也很不討喜，不該她一個人背鍋。但她在其中的演技，也確實平平。

電影院的大屏幕，會放大演員的情緒，但同時也會放大缺點。湯唯在《吹哨人》中的表演，被吐槽最多的是她的台詞，很難讓人跟她一起感受到角色的情緒起伏。

在不熟悉的領域跌倒，可以理解，但是在自己擅長的領域裡，為什麼又沒能收穫預期的成績呢？

湯唯今天在談起《大明風華》演技被吐槽時說：「我是一個菜鳥。」

而這話，也不是她第一次說了。

——我是體驗派演技的分割線——

不造是不是飯圈文化弘揚太廣，現在評價明星也漸漸出現飯圈的「回踩趨勢」。一旦某位明星風評口碑變差一點點，過往的一切光環都會被推翻，這點其實duck不必啦！

對湯唯來說，她在《大明風華》中的表演目前來看不算太理想，但並不能以此就推翻她過往一切成就啊。她是一位優秀的演員，只是並非全面型選手，但仍然在她適合的領域，曾貢獻了卓越的表演。

紐約時報曾在大約七八年前找幾位當紅的好萊塢明星問過一個問題，是「十年來你最喜歡的電影表演」，大部分人當然都答的是外國電影，但《阿凡達》女主Zoe Saldana的回答，是《色戒》里的湯唯。

「王佳芝」的成功，首先是湯唯的形象與角色貼合度極高。不止外貌符合原著中描寫的「六角臉」，她整體氣質也很獨特，有一種未經雕琢的誘人，所以李安說，看到她的第一眼，就覺得王佳芝的故事會發生在她身上。

湯唯在《色戒》裡帶來的驚艷，即使已經過去了十幾年，也依然經得起時光檢驗。

坊間評價湯唯最著名的一句話是：「湯唯可能是中國大陸女演員裡面，唯一看上去像是念過書的人。 」當然這話有些誇張啦，但湯唯身上強烈的書卷感和內斂的文藝氣息，形成了她無可替代的個人特色。

這種特色，遇到了適合的角色，就會大放異彩。比如《晚秋》里美得極有風情的安娜。在一片天地蒼茫里和玄彬接吻，在遊樂場里旋轉起舞，她的氣質與角色融合度極高，甚至只用站在那裡，就足夠動人。

但同時，也是一種局限，她的個人感太強，所以很難演好「別人」，尤其是一個沒什麼文藝氣質的普通人。

比如《北京遇上西雅圖》，雖然是一部很成功的商業片，但題材與劇本的功勞，大於湯唯的演技。

前半段演的拜金女咋咋呼呼，雖然她本身的文藝氣質給這個並不討喜的第三者角色扳回了一點印象分，但在「拜金」的呈現上，跟《大明風華》里迷之相似：搖頭，五官用力，不協調感real強烈。

直到電影的後半段，「拜金」人設消失，湯唯才找到自己熟悉的方式，回到最合適的領域。

其實吧，演員強烈的個人特色造成的局限很常見，但特色也是優點嘛，能聰明地揚長避短，對演員來說也很重要。

但是湯唯呢，她真的不是一個能用「聰明」去形容的演員。

陳可辛轉發過音樂人許願評價湯唯的這段話:「湯唯，努力把自己從一個平凡人變成一個有國際視野的成功灰姑娘。」

灰姑娘的特色不是聰明，而是幸運加努力。

湯唯在演完《黃金時代》里的蕭紅後曾對媒體說：「我真的覺得自己沒演技，我感受到的東西就能演到，感受不到的東西，就真的演不到。」

而「感受」，是一種很虛無縹緲的東西，沒辦法量化，也沒辦法用技巧，只能去體驗。因此，湯唯說，「我不是那種理論方法派，幾乎全靠老天給我什麼，我就做什麼。」

老天給了她文藝的臉和獨特的氣質，但在這之外的角色，都要靠著她辛苦地去探索，去找共情，對演員來說，也是一種巨大的內耗。

拍《北京遇上西雅圖》，她得每天扛著二十多公斤包，裡面裝滿了米和鉛球，把自己都練成八字腳了，才找到孕婦的感覺；

拍《黃金時代》，她用五個月抄《呼蘭河傳》，還抄的是繁體字，去感受當時蕭紅寫作的心情。可最終出來的結果，也備受爭議。

其實讓人欣慰的是，湯唯一直信奉並且堅持著「勤能補拙」的道理，這種靠努力彌補不足的心氣兒一直都保持著，甚至《大明風華》演技被吐槽，但拍戲時依然努力還為此得了肺炎，由此才引發雷佳音抱怨她老愛拍戲上廁所。

歪樓多講兩句：雷佳音的吐槽曾引發了一輪關於他不尊重女性的批判，他公開場合這樣說確實失禮。不過湯唯當時的狀態並非像網上所說是因為剛生完娃，實際上她是2016年當媽媽的。由於這事跟今天的主題無關，所以就此打住。

但同時讓人遺憾的是，她的局限確實是存在且很難被這種努力克服的。因為缺失了那麼一點點悟性與靈氣，所以導致張力不太夠。想要找到能超越「王佳芝」的起步符號，並不容易。

這是一個特別無解的題。想要做好且真的為此努力了，可最後的結果還是不夠好。尊重這份努力，但也得客觀評價最終的結果嘛。

在她合適且能遊刃有餘駕馭的角色里，大家還是愛湯唯的。但《大明風華》里的……就，等大家多看幾集，觀望一下，再決定吧。
