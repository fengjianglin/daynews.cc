沙和尚和容嬤嬤合影了！72歲劉大剛變老明顯肚子搶眼，還打錯字？

2019-12-18 375條評論 2,721次閱讀 356人點贊

12月15日，曾飾演《西遊記》里沙和尚的國家一級演員劉大剛在個人賬號上曬出了和演員方青卓的合照，方青卓曾在新版《還珠格格》中飾演容嬤嬤一角，兩人的同框讓人驚喜。

圖中劉大剛穿了一件深灰色的外套，下身一件杏色長褲，頭戴鴨舌帽，打扮很是舒適簡單，方青卓則穿了一件藍色的高領毛衣，外面是一件大紅色的毛呢大衣，顯得非常精神抖擻。

劉大剛老師在曬照時，配的文是：與方合影青卓，不知道是不小心打錯了還是故意調皮的這樣打字，將方青卓的名字分開來，有網友則在評論區善意的提醒劉大剛老師句子的錯誤。

前段時間，86版西遊記唐僧的扮演者遲重瑞在微博上曬出了師徒四人參觀紫檀博物館的照片，照片中西遊記師徒四人遲重瑞、六小齡童、馬德華、劉大剛四位老藝術家久違同框。

此前劉大剛、六小齡童、《遲重瑞、馬德華、也曾合體為祖國慶生的視頻，四位老師可以很明顯的看出有不同程度的發福，但臉上依舊帶著溫和的笑容，彷彿回到了當年的西遊記。

四個人很認真的參觀著博物館，孫悟空的扮演者六小齡童在看到一件物品的時候還模仿起了孫悟空的經典手勢，臉上帶著開心的笑容，其他幾個人在一旁看著，臉上也露出了笑容。

在近日結束的第二屆科普科幻電影周閉幕式上，86版西遊記主創再聚首，六小齡童、馬德華、左大玢、汪粵、劉大剛紛紛出席，幾個人也讚歎中國科幻電影的巨大變化。

劉大剛老師隨著年紀的增大，已經很少演戲也很少出現在觀眾的視野，但是每一次出現都會把人們帶回看西遊記的小時候。
