王子文腳踩高鐵桌板為曬“假鞋”？明星曬貨的正確姿勢了解一下

2019-12-21 59條評論 1,013次閱讀 52人點贊

盛裝打扮也未能在群星薈萃的國劇盛典艷壓別的女明星一頭的王子文，轉身曬出了高鐵上的私服造型，終於喜提熱搜。

只不過這個熱搜並不是正向的，大家都在吐槽她腳踩小桌板，非常沒素質。

沒多久她就被輿論逼得刪除了照片，公開道歉。

F覺得整個事件非常匪夷所思。

明星在公眾交通工具上，舉止不文雅，引發口水大戰的案例，之前有過太多太多。

多到F一度以為現在每家經紀公司都會準備好《高鐵、飛機擺拍十要十不要》的詳細指引，派發給旗下藝人。

怎麼還會有翻車的？

仔細看王子文的發照內容，焦點全在腳上：來看，我穿的鞋。

這是一雙怎麼樣的鞋，值得她甘冒被吐槽的風險，也要擱在高鐵桌板上，用大特寫去曬？看鞋型+設計元素，像是權志龍的個人品牌PMO跟CONVERSE的聯名款？

但，PMO跟CONVERSE是什麼時候開始的合作？

被稱為「潮圈最帶貨的男人」的權志龍，剛退伍就成為各大品牌爭搶的合作對象。

大部分潮人們更新到的情報是：前陣子他跟NIKE推出過一款聯名Air Force 1。經典鞋型+「刮開有驚喜的」的黑色塗層+獨家小雛菊標誌，長得就是一副爆款相。

果不其然，這一雙發售價1000+，卻在預售階段就被炒成了4000+。天貓旗艦店開售前，一些做代拍業務的賣家，紛紛開出了1000+的代拍費，而且不保證成功。

▲旗艦店兩次補貨共15000雙，都是秒沒，F不自量力試過用人手對抗代拍系統，被羞辱。

因為難得，那些通過內部渠道拿到貨的明星都迫不及待地曬出來，顧不得矜持。

而當圈內同事還在追NIKE聯名款，王子文卻搞到了「更前沿的CONVERSE聯名款」，這種壓抑不住想要曬的心情，似乎也不難理解。

問題是，不但一般潮人不知道權志龍跟CONVERSE出了這麼一款聯名鞋。連CONVERSE的高層也不知道？

而王子文搞到鞋了，身為代言人歐陽娜娜還沒搞到，仍在曬NIKE聯名款，這情況也很可疑。

事實上，這個所謂的聯名款，起初只是某些球鞋愛好者開腦洞設計出來，PMO方面一早就澄清過，沒有授權生產。

現在市面上能買到，都是一些DIY出品，價格也參差不齊。某寶上當山寨品明著賣是三位數。朋友圈上代購當正品騙著賣是四位數。

不知道王子文是花了哪一檔的價錢？從她曬鞋的迫切度來看，F猜是後者……

現在王子文的道歉微博，底下的評論已經從嘲她沒有公德心，發酵為嘲她穿「假鞋」，不尊重品牌知識產權。

一張擺拍照，被嘲了兩重，這種苦與誰訴說。

藉由這個事件，F在這裡也想稍微探討一下，明星「曬貨」的正確姿勢。

是的，就算是見多識廣的明星，也會在覓得「尖貨」後，壓抑不住想要跟全世界分（xuan)享(yao)的衝動。

表現看曬的是貨，實際上曬的是自身的時尚嗅覺、圈內資源和頂級購買力：大家還不知道，知道也買不到，能買到也買不起的寶貝，我拿到手了！

然而明星被公眾用放大鏡盯著看，就連曬個貨，也不是買、曬、贊，就完事。裡面的門路彎彎繞繞。

首先是曬什麼？都想曬「獨家」，但過於追求「人沒我有」，可能會像王子文一樣，被山寨賣家用「腦洞設計」騙個正著。

對於一些沒聽過的聯名設計，太過稀罕的新奇單品，最好問過自家造型師再曬，畢竟他們更專業。

▲唐藝昕之前穿過一雙「不存在」的SupremeX Yeezy聯名款球鞋。雖然事後粉絲辯解不算假鞋，是真鞋定製改裝，但涉及兩個品牌的版權還是無法平息爭議。

曬大牌爆款會安全一點？分分鐘被網友先自己一步發現：買到了山寨！

最典型的翻車個案莫過於沈夢辰。熱衷於在微博上曬私服細節的她，三不五時就被網友質疑：為什麼你穿的Chanel、Gucci等爆款，跟別的女明星穿的總存在這樣或那樣的細節差？

老實說，以明星的購買力，刻意買假還秀出來的，在少數。但依賴網購的她們，很容易被不靠譜的代購欺騙。

最安全的做法是，非專櫃買入不曬，非品牌贈送不曬。

具有爭議性的時尚單品譬如真皮草，切記謹慎曬。

名氣已經類比二線明星的網紅晚晚，前陣子一邊誇著龍貓可愛，一邊曬出了龍貓皮製作而成的皮草大衣，就被網友吐槽慘了。

在規避風險，保證不翻車的前提下，還要曬出個性。在選品方面，F就建議拓寬思路，不拘泥於常規的鞋、包、衣。

高手如劉嘉玲，雲淡風輕曬出一張看花圖，手上毛絨玩具，彰顯了童心。回頭懂行的潮人一科普，才發現是KAWS和Dior的聯名款，每隻摺合人民幣5萬多（炒價達百萬），而且有錢也買不到。

嘉玲姐跟品牌的深厚友誼，也被巧妙地曬到了，這跟曬個5萬多的包包，引發的關注度完全不一樣。

選好品，還要看怎麼曬。

王子文大特寫拍鞋的曬法，看起來老派，但其實很多新生代的女明星還在用：來，鏡頭鎖定我的大牌限量新品。

這樣赤裸裸地直曬，炫富感過強，可能會激起吃瓜群眾的逆反心理。

不如將單品融入造型之中，用整體的時尚感帶動大家回過頭來發掘各種細節。

▲很少大特寫單品的大冪冪卻總是被問同款。

還有一種更高端的曬法是「不經意曬到」。之前易烊千璽高考後曬出一組「出關圖」，

眼尖的網友從他身後的一堆雜物中，發現了一隻Supreme X Rimowa限量款行李箱，裡面裝滿了書。

無心插柳的易烊千璽因此喜提”最潮高考生」的熱搜。

再時髦的單品也是拿來穿、用的，這樣的分享方向更容易博得大眾好感。

不過也不一定，曬貨的平台也有講究，在小紅書這樣的「種草」平台上，直曬模式是主流，明星們也無需過分遮遮掩掩。

回過頭來說，王子文，這次曬貨翻了車，但也不全然是壞事一樁。起碼促成了品牌方將打假提上日程，也避免了更多後來者被騙被嘲了。
