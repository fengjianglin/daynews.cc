高以翔“哥哥”正面剛藍台，曾被偷刪評論，獲百萬粉絲力挺

天天要聞
7 天 ago

高以翔突然去世已經十天多了，但網民們仍然堅持為高以翔「討一個公道」。浙江衛視已經三次發表聲明，但每次都含糊其辭，不願向公眾透露真相。現在聲討浙江衛視的不僅是粉絲和網友，還有很多公眾人物。

連凱是與浙江衛視抗爭最活躍的明星。連凱是一位來自台灣的演員，連凱之前演過許多受歡迎的電視劇。連凱是個好演員，演技高，生活低調，所以大家會很熟悉連凱演過的角色，但對連凱本人有甚多的不了解。

連凱與高以翔、焦俊艷一起出演過《遇見王瀝川》，三個人因這齣戲而相熟。連凱在劇中飾演王瀝川的哥哥，兩人因此劇結下了深厚的友誼。

這一次高以翔意外死亡，連凱是最先發聲怒懟節目組的首批明星之一。連凱寫道，現在娛樂圈的氣氛很不好，所有的演員都不務正業跑去錄製綜藝節目了。連凱本人也參演過許多影視劇，但連凱幾乎不參加真人秀。連凱覺得演員們不應該過度曝光自己。

連凱還直接斥責浙江衛視刪除微博上的評論。

當越來越多的人開始加入到這場討伐大戰時，連凱說，打倒城牆並不是一夜之間的事，這似乎表明他已經準備好與浙江衛視進行長期的抗戰準備了。連凱的態度得到了網民的大力支持，高以翔的粉絲也感動了。

此後，連凱還直接嘲諷浙江衛視是縮頭烏龜，並要求大家抵制浙江衛視。連凱已經完全站在浙江衛視的對立面了。

就連《遇見王瀝川》片方也加入進來，在微博上說，要是善良的人不可以再發聲，這個世界將失去所有光明。。官方微博所代表的並不是一兩個人的意志，而是整個片方兼權尚計的結果。他們希望為高以翔說話，為他伸張正義。
