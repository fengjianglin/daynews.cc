不是“廠花”是愛玩！聽朱亞文解讀，明朝“皇太孫”為何這麼撩？

2019-12-22 107條評論 1,752次閱讀 219人點贊

最近，朱亞文有點太難了。

難得演部古裝劇，居然被觀眾看出了「東廠氣質」；

為新劇微博營業，艱難求助要怎麼給自己換上「我是孫子」的頭像……

原因無他，正在湖南衛視熱播的《大明風華》確實火！而朱亞文扮演的「皇太孫」更是「明朝一家人」中格外特別的一位。日前，在接受湖南衛視採訪時，朱亞文直面外界對角色的討論：原來，只看了兩集劇本，他就定下了這個角色；為了演好這位玩世不恭的人物，他還換了三種聲線表演，「這一次，我真的感到非常滿足！」

「不著調」的皇太孫

《大明風華》是朱亞文近年罕有的古裝作品，他自己也承認，原本對此類作品非常謹慎，但卻在看了短短兩集劇本以後就決定參演，「可以同時和這麼多優秀的演員在一起創作，是一件非常難得也很難再複製的事情。」

而對於如今引發熱議的「皇太孫」朱瞻基一角，朱亞文也有自己的理解。比如劇集初期他以錦衣衛形象示人，不只是使命在身，也是玩性使然，「我之前對於朱瞻基做了一些了解。這個皇帝很會玩兒，也很愛玩。導演是把這種玩兒性巧妙的融入到劇作當中，才有了大家現在看到的這個表面泡妞、實則查案的皇帝形象。」

在朱亞文看來，以往影視作品對朱瞻基的描繪不多，這也給了他一次自由創作的機會，比如在創作過程中，他使用了三種不同的聲調描繪人物前期翩翩公子、中期英朗將帥、末期霸氣帝王的狀態，「前期大家會看到他好像故意地有一些不著調，一些搔首弄姿等等。這其實是人物身上的保護色。在家族的關係中，他需要生存下來，需要變換身上的顏色、為人處事的方式。」說著，他也並不諱言被外界調侃「有點娘」，「說什麼廠花，廠哥的，都沒關係。大家慢慢往後看，這個人物會越來越豐富，越來越堅挺。」

「明朝一家人」很有趣

不同於以往許多古裝戲，《大明風華》開篇雖有男女主角的相識相遇，但更多著墨於朱棣（王學圻飾）、朱高熾（梁冠華飾）、朱高煦（俞灝明飾）和朱瞻基這「明朝一家人」的日常相處。從朱棣發動靖難之役並取得帝位，到不受寵的「太子」朱高熾的艱難處境，再到朱高煦對哥哥朱高熾的虎視眈眈，乃至於「皇太孫」朱瞻基的夾縫生存，一家人夾雜著權力與親情的關係頗具看點。

朱亞文坦言，「明朝一家人」情節之所以好看，得益於導演和編劇的巧妙處理。「雖然我們講的是宮牆大院里的事情，但導演和編劇更多的是希望讓觀眾看到人和人之間真實的相處和摩擦。用風趣幽默的方式來寫，更容易讓大家有帶入感。」

這樣的設定之下，演員們之間的對手戲也非常過癮。以與王學圻的對手戲為例，朱亞文直言自己是又享受又幸運，「跟學圻老師的這一次合作，讓我重新對演員的標準有了一個新的定義。一個70多歲的演員，在現場完全投入、精湛演繹，這整個過程讓我折服。我非常有幸能夠在角色當中跟他有這樣的爺孫關係。後期我跟王學圻老師很多的戲，都是賴在他懷裡撒嬌的一個狀態，非常享受。」

不只是「會撩」女主角

明明隔著世仇，卻誤打誤撞相識，朱瞻基與孫若微（湯唯飾）的感情線索也是《大明風華》的看點。朱亞文說，朱瞻基可能一輩子都在猜測孫若微對自己的愛。

「一開頭在酒館互相試探的過程當中，朱瞻基喜歡上了孫若微。但是那個時候的喜歡，還是有著階層的差異、強弱的區別。正是因為孫若微身上有著和朱瞻基比較類似的成長經歷，讓朱瞻基找到了一種久違的或者從來沒有過的共鳴。」朱亞文解讀說。

至於朱瞻基無論是對女主、爺爺的相處模式上都被評價「會撩」，朱亞文笑說，「你這麼一說，朱瞻基還確實是挺會撩的，而且他針對不同的人呢，他是軟硬兼施，然後能夠低得下來。我覺得『撩』這個東西呢，不是一個特別生硬的行為啊，它是一種柔軟有彈性的。那得知己知彼，只有懂得這些才能對症下藥。」
