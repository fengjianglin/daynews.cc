12月4日，一年一度的Mnet亞洲音樂大獎（簡稱MAMA）再次拉開帷幕，雖然頒獎儀式已經來到20周年的日子，但劇本方此次並未給韓國本土的觀眾帶來太多驚喜，而是按照順序將此次頒獎典禮安排在了日本名古屋。

雖然作為音樂盛典，但作為介紹環節的紅毯儀式自然也是必備，可正當日本觀眾們還在期待韓國女愛豆們會如何驚艷時，猝不及防就被咱們唯一的中國女星刷了屏。

據悉，古力娜扎作為此次MAMA上唯一受邀的中國女星，雖然是擔任頒獎嘉賓，但在紅毯環節一亮相，就妥妥引起了各界網友的轟動。

此次亮相MAMA紅毯，古力娜扎身穿一席銀色亮片拖尾禮服，大波浪捲髮披肩的同時，也把長裙露肩的部位襯得無比顯眼，而不少網友在看到古力娜扎的造型後，也是紛紛感慨其如同仙女下凡，美麗迷人。

當然，不僅僅是高清鏡頭下能抗能打，即使在直播鏡頭中，古力娜扎依然顯得笑靨如花，身邊兩位韓國美女主持，在這一刻也是被襯得如同「村姑」，無論是面容還是形體，甚至是在氣質這一檔上，完全都被古力娜扎給甩到了天邊。

而相比較娜扎的驚艷亮相，後續出場的韓國女愛豆們就顯得平淡了太多太多，無論是一身白衣套裝亮相的ITZY組合成員

還是全員黑衣all black的「MAMAMOO」組合，

又或者是韓國目前最火爆的兔瓦斯女子組合（TWICE），她們無論在禮服的搭配上，還是妝容的精緻度上，甚至就連在舞台上的氣場，都完全不可與古力娜扎相提並論。

而SOLO歌手金請夏，即使也是一人在舞台上亮相，但多少仍然顯得黯淡無光，平淡過了頭，簡而言之，也不知道是否有著其他原因，韓國女愛豆們此次紅毯表現總體給人一種太過小家子氣的感覺，對比娜扎的大方自信，完全一眼就能高下立判。

當然，其實真要說起來，MAMA紅毯出現這種情況也不是第一次了，就拿去年的紅毯儀式來說，當時，「南韓腿精」宣美和「收視女王」徐玄振等韓國女星都有亮相。

在紅毯上的她們雖然較這屆的後輩好了不少，但在angelababy登場之後，也是讓人感受到了巨大的差距，而且與古力娜扎相同的是，angelababy在去年的MAMA紅毯典禮上，也是艷壓全場，憑一己之力完勝所有同場女嘉賓，而當時，就有網友點評實在是太長臉了。

不過，在連續兩次MAMA紅毯被我國女藝人美成「主場」之後，不由得也是讓人產生了些許對比，比如此次古力娜扎的眼壓全場，和去年angelababy的一枝獨秀，倘若兩人展開對比，又不知道誰會更勝一籌呢？
