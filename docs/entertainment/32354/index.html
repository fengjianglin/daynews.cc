朴樹的耿直並不是裝的，還記得他在金馬獎的霸氣發言嗎？

2019-12-23 664條評論 4,191次閱讀 2101人點贊

在這世間生活，免不了被條條框框的規則所限制，所以有些人世故圓滑、左右逢源；有些人生活所迫，「為五斗米折腰」。然而，總有那麼一些人，不在乎外界的看法，活出自己想成為的樣子。被稱為「人間真實」的朴樹就是一位真實耿直、保持獨立的音樂人。

一向低調的朴樹在綜藝節目中罷演！在綜藝《樂隊的夏天》里，節目錄到一半朴樹突然起身離開，稱:「時間到了，我要回家睡覺了」，驚呆了在場觀眾和工作人員。有些網友質疑朴樹在裝耿直，畢竟哪有錄節目到一半離場的？

但朴樹的驚人之語可不止這一次，2014年朴樹因《平凡之路》拿下金馬獎最佳原創電影歌曲，在一片盛裝出席的明星的圍繞中，他一身隨意的休閑服上台領獎並霸氣發言:「我一直特別抵觸頒獎典禮，之前經紀人動員了我好久，我才答應來參加，很多年來，我對華語的電影也有成見，我認為這好像只是錢的遊戲」，隨後又表示了對金馬獎的敬意和對音樂的堅持。一番大膽的發言贏得了全場喝彩，不禁讓人感嘆真是個耿直boy。

一些網友認為，朴樹這樣是在作秀，對舞台缺乏敬意，也沒考慮節目組和觀眾，實在是太任性了。而另一些網友則認為，在這個契約社會裡，真實是難能可貴的，高情商不是圓滑世故，而且而是可以做自己。朴樹身為一個歌手一直本分踏實地專註於自己的本職工作，耿直真實不妥協正是他性格的閃光點。

小編認為，千人千面，每個人對生活都有自己的理解。有些人認為生活更重要，有些人終生都在追尋理想，如何在現實和夢想中找到一個平衡點是我們需要終身學習的課題。如果可以選擇，你認為左右逢源和保持真實哪個更重要呢？
