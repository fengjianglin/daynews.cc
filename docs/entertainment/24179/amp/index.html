她是學霸港姐，被渣男欺騙後單身至今，今年52歲生活瀟洒

天天要聞
2019-12-19

說起陳法蓉，我們非常熟悉，她是港姐出身，作為一名女星，陳法蓉在娛樂圈的知名度是非常高的。因為長得好看，外形條件比較好，她得到的表演機會比較多。在上個世紀的香港，她非常受衛視的喜歡和熱捧。

陳法蓉長著一雙大大的眼睛，一雙大眼睛非常有神采。年輕時候的陳法蓉迷倒了很多人，和她合作過的演員都誇她長相俊美。現在的陳法蓉雖然不再年輕，看起來還是非常優雅。她的性格非常開朗，人活得積極樂觀，現在的她雖然沒有成立自己的家庭，但是活出了瀟洒自在的人生！

平時的她熱愛旅行，她經常獨自一人去自己喜歡的地方旅行，生活中的陳法蓉特別具有童心，她經常和不同年齡段的朋友一起出去玩聚會。她的童心讓她保持非常年輕的狀態。也讓她活得非常瀟洒。

平時的陳法蓉還非常喜歡瑜伽等輕運動，在呼吸之間把身材保持地很好。

在演戲方面，她非常有天賦。扮演的很多角色都給大家留下了深刻的隱形。陳法蓉非常適合飾演古裝影視劇，她的古裝照片都非常美，大家看到她的古裝照覺得驚為天人。她在當時非常火的影視劇《水月洞天》中有扮相，她和當時在劇中扮演童戰的楊俊毅因戲結緣成了情侶。但是楊俊毅非常花心，他並沒有像表面看到的那麼踏實，在倆人談了4年戀愛後，最終選擇了分手。

現在的陳法蓉52歲了，但是身段皮膚還和年輕時候差不多，讓人感慨時間沒在她身上留下痕迹。現在的她依然孤身一人，但是生活過得非常充實。
