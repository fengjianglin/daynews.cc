《客棧3》李蘭迪睡衣土氣，林心如少女，再看闞清子：我沒眼花吧

2019-12-18 442條評論 2,097次閱讀 816人點贊

一說到最近熱度最高的綜藝，大家都會想到《親愛的客棧》，節目的形式就像是模擬了一個「小型社會」，讓網友看到了一個職場圈子，因為客棧里有老闆、員工、管家以及助理等人，每個人的分工都很明確。不過幾個男孩子的性格都很好，馬天宇很佛系，張翰不爭不搶，吳磊孩子氣，都是很典型的代表。

而女生的話雖然年齡跨度比較大，但她們的共同話題還是有不少，像闞清子、李蘭迪以及林心如是住在同一個宿舍里，每天朝夕相處，三人之間會有抱怨，也會有分享。而站在觀眾的角度，更希望看到的是她們團結協作。但現實是很殘酷的，她們的表現也很真實，其中闞清子大清早賴床的那一段，也是說出了無數女生的心聲。

當時林心如和李蘭迪正在緊張地化妝，因為很快就要去開早會了，但闞清子依舊在床上賴著，好不容易坐起來了，接著又倒下去了，看得觀眾也十分著急。但林心如和李蘭迪都沒有叫她的意思，畢竟很多人都有起床氣，貿然去叫也是不禮貌的。而等到闞清子起床後，導演也是給了三個女生睡衣一個大大的特寫。

我們能看到當林心如起身後，李蘭迪的睡衣出現在了我們的面前，她選擇的顏色有紅有綠，也是被網友吐槽土氣，不像是個20出頭的女生穿的。而當林心如準備回來坐到梳妝台上的時候，網友也看到了林心如的睡衣，她的粉色條紋睡衣看起來質感很好，並且也是很少女，顏色選的很襯皮膚，看起來很白，

不得不說居家型的女明星就是不一樣，連睡衣都穿得那麼洋氣，真的是讓人很佩服。而最讓人期待就是闞清子的睡衣，當時她從床上站起來的時候，睡衣也是曝光了，雖然她外面穿了一件超厚的外套，但網友還是看到了闞清子的睡衣是低領，和李蘭迪、林心如比起來也是要更高檔，顯得更有氣質，並且這套睡衣在上一季《親愛的客棧》中也出現過，還是情侶款，闞清子在第二季中仍然在穿，可見她有多喜歡這套睡衣，或許也是懷舊吧，網友看完也表示我沒眼花吧！那你們怎麼看？
