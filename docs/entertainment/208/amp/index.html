首先是楊冪在劇扮演的是一個非常痴情的一個女角色，在劇中的結局是讓人很是心疼，來看看現如今的她還是想當年是一樣，她的公司也是培育出了很多演員，也是自己當老闆了。

​

李易峰的顏值是很高的，當年在劇中扮演的角色身世是那麼的悲慘，讓我們大家紛紛側目。都認為是走偶像派的其實人家走的是實力派，如今也是出演了一些非常不錯的電視劇。

鄭爽也是出演了這部劇，不知道當時你們看出來了嗎？她扮演的是那隻可愛的小狐狸，那可愛單純的模樣有沒有迷到你，現如今也是越發的漂亮時尚。她是一個很努力的人，期待她能給我們帶來更多精彩的作品。古劍奇譚5年前VS後，風晴雪逆生長像18，方蘭生變化多端！

馬天宇在劇中扮演是一個給人就是頑皮的感覺，他一直想要學法術，直到遇到屠蘇就一直要學法術。他與小狐狸的感情讓我們看的揪心不已。如今變得很是成熟帥氣，期待他的新作品。
