他在不知情的情況下，“演”了一部大片，網友：最適合徐鳳年的人

2019-12-19 462條評論 2,624次閱讀 691人點贊

他在不知情的情況下，「演」了一部大片，網友：最適合徐鳳年的人

（以下圖片均來自網路，若有侵權，告之則刪！）

近幾年，很多熱門小說都被改編成電視劇了，像貓膩的《將夜》、《擇天記》，天蠶土豆的《斗破蒼穹》、《武動乾坤》，還有就是蕭鼎的《誅仙》。

這些曾經風靡一時的網路小說，曾經因為影視技術的限制，無法被改編成電視劇。不過隨著國內特效技術的不斷成熟，這些熱門的玄幻仙俠小說，也紛紛走上了被改編的道路。

不過這些小說被改編成電視劇之後，口碑卻出現了明顯的兩極分化，像貓膩的《將夜》，就受到了很多觀眾的肯定，而天蠶土豆的《斗破蒼穹》卻被嘲諷的一無是處，甚至有不少的網友吐槽，好好的一部玄幻劇硬是被拍成了武俠言情。

當然，這些玄幻仙俠小說能夠被改編成良心劇，除了技術特效之外，還有非常重要的一點，就是演員的演技水平。就像最近熱播的電視劇《慶余年》，整部劇的特效水平，幾乎顯示不出來，但演員的表現和演技，卻受到了觀眾的一致好評。

像陳道明的慶帝、王陽的滕梓荊、田雨的王啟年、吳剛的陳萍萍等，哪一個不是將人物的精髓演到了極致。就連從未演過電視劇的郭麒麟，都完美地駕馭了范思轍這個角色，這樣的表現，怎麼可能不受到觀眾與原著粉絲的認可呢？

當然，除了演技之外，演員自身的形象與角色的契合度，也是非常重要的。

就像貓膩說的那樣，慶帝這個角色是按照陳道明量來寫的，所以他來出演慶帝，能最大程度上還原這個角色。還有就是范思轍，郭麒麟出演這個角色的時候說過，雖然他沒演過太多的電視劇，但他感覺他能夠駕馭范思轍這個角色，因為形象比較符合。所以最終我們看到，他的范思轍確實給了觀眾一個驚喜……

而說到這裡，就不得不提另一部網路小說《雪中悍刀行》。網傳這部玄俠小說也將要拍成電視劇了，但關於演員的選擇，卻是一個非常頭疼的問題。一方面是演藝圈年輕一輩的演技，恐怕難以駕馭男主徐鳳年這個角色。但年齡過大的演員，又和徐鳳年的形象不符合。

不過，如今這個顧慮，似乎可以打消了。因為網友已經找到了一個最適合徐鳳年的演員，他就是陳坤。

陳坤的外表俊美，能夠駕馭多種演繹風格，無論是陰柔狠辣的西廠廠公，還是心機深沉，深陷權力傾輒的六皇子，他都完美地演繹了出來。尤其是他在《天盛長歌》中的表現，舉手投足間散發出來的氣質，簡直和《雪中》里的徐鳳年如出一轍。

六皇子的隱忍、謀略、大義、對將士的態度，在陳坤的演繹之下，簡直就是另一個翻版的徐鳳年。

基於此，很多《雪中》的書迷，都以陳坤為原型，剪輯了很多經典的視頻橋段。像B站和某音上，搜索一下徐鳳年，你能看到一大堆關於陳坤的剪輯。

而陳坤，也在不知情的情況下，「演」了一部大片。

在這些視頻播出之後，不少網友紛紛反映，陳坤真的是最適合徐鳳年的人。他的陰柔、他的狠辣、他的城府和情義，全能從視頻中體現出來……
