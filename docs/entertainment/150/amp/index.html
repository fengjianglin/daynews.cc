大家好，我是yaya.

自十二月以來，各平台紛紛上新了我們期待已久的「電視劇」。接下來yaya就帶大家盤點一下個平台已播出的劇，並簡單的介紹及個人觀看後的感受。

1.《從前有座靈劍山》

這部劇是從11月12日開始上映的，全劇共37集，現在已經全部更完，會員可看全集，非會員可看到32集。這部劇小編對它可以說是感情深厚啊，記得15、16年的時候小編就開始追《從前有座靈劍山》的動漫，可惜的是動漫只有兩季，就沒有再更了，不過最近第三季又再次來襲了，不過小編已經看完電視劇了，所以不會去追第三季了。這部電視劇總的來說還是比較符合原著的，沒有魔改，人物設定也都非常符合原著，這樣講吧：這是除了胡歌仙劍系列，我唯一覺得不錯的仙俠劇，服裝、演員、道具、特效都是可圈可點的。這部劇是穿越劇，所以男主的辭彙比較現代，不過這也是一大特點。唯一有點遺憾的是感覺這部劇後半部分節奏有點快。

2.《心靈法醫》

這部劇是由聶遠和宋軼主演的懸疑劇。懸疑劇作為近年來市場熱門的題材，基本是播一部火一部（粗製濫造的不算）。這部劇也是11月開播，共有36集，現在更新至32集。這部劇總體來說不能稱 為一部優秀的作品，主要原因是劇本的人物設定及拍攝、影視後期等方面實在太爛。宋軼主演的女一人物設定不對，不像是一個刑事隊長，經驗不足，這是槽點一；整部劇的畫面一直在抖動，我不知道拍攝的團隊有沒有發現這個問題，這太影響觀感了，這是槽點二；整部劇的背景音樂實在是太吵了，有點噪；這是槽點三。說完槽點，也說下優點：一是演員演技在線，二是劇情內容尚可。也正因為優點小編才能追下去。

3.《天機十二宮》

這部劇來自王超所著的小說《天機十二宮之詭局》。這本小說非常好看啊，這部劇總的來說也是非常不錯的懸疑劇。十二月上映，現第一季第二季已更完共24集。電影畫質、服裝精良、演員演技在線。這部劇相信很多小夥伴都不清楚這部劇，因為這部劇的演員陣容確實很弱，都是些名不見經傳的青年演員主演，並且上映的時機也不好，放在了十二月上映，與很多大劇衝撞，上映幾日愛奇藝首頁都沒給推薦，實在是有些可惜。這部劇值得一追。

4.《慶余年》

這部劇來自貓膩大大的《慶余年》小說。第一季現已更新至17集。從目前更新的內容看還是比較貼合原著的，不過原著的滕梓荊沒死，而電視劇里卻死了，這部劇一上映便被大家所矚目，畢竟IP大、演員陣容大。這部劇由青年演員張若昀、李沁主演，陳道明、吳剛等老戲骨鼎力加盟。這部劇無論是從哪個方面看都是一部製作精良的優秀古裝電視劇。但這部劇唯一讓我遺憾的是劇作的節奏有點慢，可能是投資過大，所以集數需要拉長。

5.《劍王朝》

這部電視劇是來自無罪大大的《劍王朝》小說，和《慶余年》一樣都是大IP改編的，驚喜的是這部劇是一部玄幻劇，相信大家都知道近年來的一些玄幻劇出一部撲街一部，具體劇名我就不說了，喜歡追劇的朋友心中都有數。為什麼說是驚喜呢？應為這部戲和往日的玄幻劇確實大不一樣，首先從製作陣容來說，這部劇是馮小剛監製，馮小剛是搞攝影美術出身的，可能因為這個原因，這部劇的畫面確實很不一般。尤其是山水方面，可以說每一幀都是高清截圖啊。演員陣容也是非比尋常由李現、李一桐主演，眾多實力派演員傾情加盟。最驚喜的還要數特效及打鬥畫面，玄幻劇的槽點就在這，但《劍王朝》在這方面可以說是絲毫不慫啊。這部劇的台詞也有驚喜，尤其是丁 寧和小姨的虎狼之詞，真是笑到我了。這部劇也有槽點，就是李現的劉海前幾集看真的想笑。不過後面看習慣了。

6.《鶴唳華亭》

這部劇的定位是正劇，看到現在也確實沒什麼大的槽點。不過全劇都在圍繞著一個主題「太子哭唧唧、太子太南了」（玩笑話）。這部劇目前更新到23集了。《鶴唳華亭》演員陣容也是蠻好的，由羅晉、李一桐領銜主演，王勁松、黃志忠等戲骨加盟。演技方面完全沒槽點。道具、服裝也是良心。個人最喜歡的一幕是，太子與陸文惜在盧尚書家中相遇，太子始終沒有跨越屏風，而是給了陸文惜尊重，並且整部劇中的這種尊重含蓄都在，個人很喜歡這種劇情，而不是千篇一律的霸道總裁既視感。

你以為這就是全部了？不，接下來還會有《大明風華》、《山海經之上古密約》、《精英律師》等劇將在十二月與大家相見。都在一起播，就會產生競爭，競爭就會加更，加更什麼的我最喜歡了。哈哈
