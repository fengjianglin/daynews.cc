楊千嬅一兩句唱不好就很不開心，唐一菲一句話，專業與業餘的區別

天天要聞
5 天 ago

《妻子的浪漫旅行》第三季已經播出有一段時間了，現在妻子們已經從泰國的曼谷來到了美麗的蘇梅島，在這裡她們將展開新的旅行，那麼接下來她們會帶給我們什麼樣的精彩看點呢？讓我們一起來看一下吧。

一大早魏大勛就被謝娜叫到廚房幫忙做早餐了，而其他的妻子們則是在餐桌上等待美味的早餐，等待的過程是比較漫長又無聊的，所以她們便聊起了楊千嬅的演唱會。

楊千嬅表示自己每次開完一場演唱會一下台就會把腳插進冰水裡消腫，因為在台上站的時間太久了這樣能讓自己很快得到緩解，這也是很令人驚訝地了，如果不是在節目中聽到還真是不知道原來歌手每開一次演唱會是這麼的辛苦。

楊千嬅還表示在開演唱會的時候，如果一首歌中有那麼一兩句自己唱得不好就會很生氣，對於這一點唐一菲表示很不理解，她說：可是那麼多首歌總有一兩句唱得不那麼好，而且觀眾又聽不出來，唐一菲的這句話，立馬看得出專業與業餘的區別了。

楊千嬅卻表示必須要保持不讓自己出一點的錯，雖然觀眾們可能聽不出來，但自己是騙不了自己，這真的是一位好的歌手所具備的素養，真的是一個完美主義者，紅也是有道理的，網友們對此有什麼看法呢？歡迎大家來留言、點贊哦！
