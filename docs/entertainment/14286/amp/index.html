這是【我的私人榜單】的第二期，將為大家分享我最喜愛的5部經典歌舞片。

無論是所有人都聽說過但並非所有人都真的看過的《雨中曲》和《音樂之聲》，還是近幾年頗受矚目的《愛樂之城》和《馬戲之王》。

《馬戲之王》

無論是迪士尼動畫《美女與野獸》《獅子王》，還是一言不合就尬舞的印度電影《三傻大鬧寶萊塢》。

《三傻大鬧寶萊塢》

歌舞片——作為一種古老而傳統的影片類型——始終非常受觀眾歡迎。

1927年，史上第一部歌舞片《爵士歌王》問世，同時也標誌著有聲電影時代的到來。

如今，過去了92年，歷久彌新的歌舞片依舊屹立在銀幕之上，經典頻出。

《爵士歌王》

愛它的人愛到痴迷，厭它的人則往往不能理解為什麼好好的電影非要唱唱跳跳的。

然而這正是歌舞片的魅力所在——

以動人的音樂與優雅的舞蹈，去替代用嘴說出的普通台詞，它所能給人帶來的視聽享受和愉悅情緒，是無法比擬的。

《歡樂滿人間2》

接下來，我將為大家推薦5部我N刷不厭的優秀歌舞片。在我看來，這些電影隨便哪部都值得被稱之為「人生必看」。

老樣子，排名分先後。純屬個人口味，不喜還望輕噴。

  • No.1 《雨中曲》

導演：斯坦利·多南、吉恩·凱利

主演：吉恩·凱利、黛比·雷諾斯、唐納德·奧康納

年份：1952年

片長：1小時43分鐘

無聲電影時代，演員的形體遠遠比聲音重要。琳娜就是憑著漂亮的臉蛋和豐滿的身材，和唐成為了大受歡迎的銀幕搭檔。

然而，隨著技術的發展，有聲電影問世，這就難倒了聲音極其難聽的琳娜。無論如何訓練，她都不能把台詞念得漂亮。

於是，唐請來了聲音甜美的凱茜，讓她做琳娜的幕後配音者。

凱茜的聲音，使得唐和琳娜的有聲電影大獲成功。更加走紅的琳娜，卻反而變得目中無人，不承認凱茜的貢獻。

在唐與凱茜相處的過程中，兩人彼此互相傾慕，擦出了愛的火花。一場大雨給了唐無限的靈感，他決定要讓才藝俱佳的凱茜得到應有的名譽和地位。

上榜理由：《雨中曲》是影史最佳歌舞片，沒有之一。

它早已逾越了「美國國寶級影片」的頭銜，成為了全世界影壇的一份無上瑰寶。

毫不誇張地說，《雨中曲》所呈現出的那種舞蹈與劇情的銜接、對白與歌曲的交融，迄今為止都還在被模仿，卻未曾被超越。

這部影片最了不起的地方，在於其電影手法的高超運用。

從整體構思來看，該片可以說是對電影套層結構的一次精巧實驗，而它自然流暢的場景過渡也堪稱是教科書級別的。

《雨中曲》輕鬆幽默的情節能吸引各個年齡層的觀眾。片中所體現出的對默片時代的致敬以及對歌舞片類型的熱忱，也都讓這部電影成為可以傳代的精品。

畢竟，那麼多電影對它的各種致敬，可都不是沒來由的。

  • No.2 《芝加哥》

導演：羅伯·馬歇爾

主演：蕾妮·齊薇格、凱瑟琳·澤塔-瓊斯、理查·基爾

年份：2002年

片長：1小時53分鐘

維爾瑪是一個夜總會舞女，本來名氣不是很大。當她在發現自己的妹妹和花心的丈夫睡在了同一張床上後，一怒之下射殺了兩人。

比利是芝加哥最能顛倒黑白的律師，靠著打謀殺案的官司而提升知名度。他受理了這樁案件，心中有著自己的打算。

他藉助媒體炒作維爾瑪的案子，一方面令維爾瑪瞬間成為紅人，而他自己也隨之聲名大振。

然而，另一件相似的謀殺案碰巧在此時發生了。一心夢想成為歌舞演員卻被男友欺騙的洛茜也一氣之下射殺了男友，被關進監獄。

比利發現洛茜的案件更能吸引公眾目光，於是他準備推遲維爾瑪的案件，來藉助羅克茜一案贏得更大的聲名。

他巧舌如簧，把殺人犯洛茜包裝成楚楚可憐的受害者，令媒體瘋狂地大肆報道。

維爾瑪覺察形勢不妙，為了保全自己的利益，她與洛茜之間的明爭暗鬥不可避免的開始了。

上榜理由：改編自百老匯同名經典音樂劇的《芝加哥》，堪稱是歌舞片中的典範。

前幾年《愛樂之城》大熱時，我曾無數次向身邊的朋友安利《芝加哥》。

我說：「《愛樂之城》固然是一部優秀的佳作，但其最大的問題，或許就是歌舞編排上的無力感。而《芝加哥》則最為完美地詮釋了什麼叫『歌舞推動劇情』」。

《芝加哥》解決了歌舞片長久以來的兩項短板——

首先，影片深度往往不是歌舞片的特長，因此《芝加哥》能有如此針砭時弊的主題，的確是鶴立雞群了。

其次，《芝加哥》中的歌舞非常之多，信息量又極大，幾乎全程用在用歌舞敘事。而它非但沒有拖慢劇情節奏，反而像一列火車將劇情飛快地駛向前方。

整部影片的歌舞幾乎全部是女主角洛茜的精神幻覺。在幻想與現實間巧妙穿插的演繹方式，將這部歌舞片的剪輯理念、視覺創意和藝術張力都推至了前所未有的高度。

難怪《芝加哥》能在第75屆奧斯卡上以13次提名、6次獲獎 (包括最佳影片) 的成績成為當之無愧的「全場最佳」。

  • No.3 《音樂之聲》

導演：羅伯特·懷斯

主演：朱莉·安德魯斯、克里斯托弗·普盧默

年份：1965年

片長：2小時54分鐘

瑪利亞是一個年輕活潑的修女，喜歡在大自然下高聲歌唱，因此常常忘記修道院里的規矩。

院長認為瑪利亞並不屬於規矩嚴格的修道院。於是她決定安排瑪利亞到一位名叫特拉普的上校家中做家庭教師。

上校的妻子去世多年，留下了7個孩子。他對孩子們的管教十分嚴格。他告訴瑪利亞，因為孩子們的種種惡作劇，來他家的家庭教師都做不長久。

瑪利亞果然也遭到了惡作劇，可善良的她並沒有告訴他們的父親，而是像個母親一樣照顧與包容這些孩子。

瑪莉亞很快跟孩子打成了一片，而上校也在瑪利亞的引導下，漸漸改變了對孩子們的態度。

上校與瑪利亞之間滋生了感情，他們完婚後回到了奧地利，卻發現自己的家鄉已被納粹佔領。

並不想為納粹效力的上校，在與家人一起演唱完一首飽含對祖國熱愛之情的《雪絨花》後，準備逃亡出奧地利，而納粹也展開了對他們的追捕。

上榜理由：第38屆奧斯卡上，《音樂之聲》被提名10次、獲獎5次 (包括最佳影片)。

而這些獎項對它來說，甚至已經稱不上「談資」了。即便無視它獲的獎，《音樂之聲》依舊是名聲最大的一部歌舞片。

在這一點上，即便連《雨中曲》也是不服不行的。

看看那些歌曲吧——

《哆來咪》《孤獨的牧羊人》《雪絨花》《音樂之聲》《晚安，再見》。有太多的人從小就是聽著這些歌長大的，我也一樣。

美好而溫暖的情感，以及無數雅俗共賞、膾炙人口的歌曲，無疑是該片取得空前成功的最關鍵因素。

如果你早就聽過上面這些經典歌曲，卻始終沒有好好看過這部電影 (或是看得太早，已經根本沒印象了)，那麼，答應我，去看它！

  • No.4 《西區故事》

導演：羅伯特·懷斯、傑羅姆·羅賓斯

主演：娜塔莉·伍德、理查德·貝梅爾、喬治·查金思

年份：1961年

片長：2小時32分鐘

曼哈頓西部貧民區里，有兩幫勢不兩立的流氓團伙。

一個是由白人里弗領導的「火箭幫」，另一個是由波多黎各人貝納爾多帶頭的「鯊魚幫」。兩幫相互仇視，經常鬥毆生事。

某次，西區舉行舞會，兩個幫派相互拼舞，里弗的朋友托尼與貝納爾多的妹妹瑪麗亞一見鍾情。

貝納爾多發覺此事後，強行讓手下帶走了妹妹。當夜，托尼站在瑪麗亞的窗前呼喚她，兩人還是堅定地相愛了。

第二天，瑪麗亞聽說「鯊魚幫」將和「火箭幫」決鬥，便讓托尼前去阻止。

托尼匆匆趕到現場，試圖阻止鬥毆的發生。在混亂中好友里弗被殺，他出於自衛而失手殺死了貝納爾多。

這讓瑪麗亞陷入矛盾和痛苦中，她忍受著家人的責罵與侮辱，但心中卻無法放下托尼。

貝納爾多的好友持槍尋找托尼算帳，得知消息的瑪麗亞匆匆去告知托尼，然而，悲劇還是趕在她之前發生了。

上榜理由：11提10獲，讓《西區故事》成為了奧斯卡史上獲獎最多的一部歌舞片。

它的藝術性的確是太高了，看完後你甚至會想不明白它是如何將歌舞融合進那些場景中去的。小流氓的街頭械鬥都能用舞蹈給跳出來？

不得不說，強調大場面群舞設計的《西區故事》，確實具備著大多數歌舞片所沒有的那種獨特活力與磅礴感。

《西區故事》也是羅伯特·懷斯的導演作品，而他在本片中所顯露出野心，可比他後來的《音樂之聲》要大得多。

他請來了能與卡拉揚一較高下的著名指揮家、作曲家伯恩斯坦來為本片編曲，使得這部改編自莎士比亞戲劇《羅密歐與朱麗葉》的歌舞片的質感瞬間拔高了好幾個層級。

《西區故事》作為史蒂文·斯皮爾伯格最愛的歌舞片，早在十幾年前他就已拿下了改編權，並一直計劃將其翻拍。

斯皮爾伯格與他的新《西區故事》主演們

目前已確定了男主角將由《極盜車神》《星運里的錯》中的男主角安塞爾·埃爾格特擔當，而新版的《西區故事》也將於2020年上映。

新《西區故事》劇照預覽

大導演斯皮爾伯格將如何延續經典？

那麼首先，在期待明年的新版之前，我們還是要先去領略舊版《西區故事》到底經典在哪。

  • No.5 《理髮師陶德》

導演：蒂姆·波頓

主演：約翰尼·德普、海倫娜·伯翰·卡特、艾倫·里克曼

年份：2007年

片長：1小時56分鐘

受法官塔平的陷害而被迫流放他鄉15年的理髮師陶德，終於又回到了倫敦。他懷著滿腔怒火，一心只想復仇。

他的妻子露西已經不知所蹤，而女兒喬安娜則被塔平收為了養女，軟禁在深宅大院中。

一次偶然，陶德遇見了餡餅店的老闆娘洛薇特夫人，後者對他心生愛戀。於是，在餡餅店的樓上，陶德重新開張了他的理髮店。

此後，原本已瀕臨倒閉的餡餅店，生意竟漸漸好了起來，只是煙囪里不斷冒出惡臭的黑煙。

而陶德的理髮店客人也很多，卻很少能見到有人從裡面走出來。陶德一直在耐心地等著法官塔平的到來……

上榜理由：在我看來，蒂姆·波頓始終是一位庸才，他在國內觀眾心中完全被過譽了。

就事論事地說，除了早期的《剪刀手愛德華》《聖誕夜驚魂》和《艾德·伍德》以外，蒂姆·波頓長久以來為觀眾帶來都是些看似古靈精怪、實則毫無亮點的平庸之作。

他和德普、海倫娜所組成的「哥特鐵三角」，也逐漸令人覺得缺乏新意。

但我願意將《理髮師陶德》與他的上述三部作品相提並論，而這也是蒂姆·波頓最後的「迴光返照」。在此之後，他再沒能拍出像樣的電影。

這是一個「基督山伯爵」式的悲情故事，充斥著關於迷戀、愛、痛苦、恐懼、失落的複雜情感。

波頓依舊選擇用他所 (唯一) 擅長的哥特風來呈現，是種正確的選擇。那種陰鬱而殘忍的氣質，令觀眾心生恐懼、脊背發涼。

作為一個誕生於1847年的歷史悠久的經典音樂劇，蒂姆·波頓要將其搬上電影銀幕，堪稱是一種勇敢的行為。

畢竟，歌舞片和驚悚恐怖片，聽上去就是兩種調性不搭的東西。或許也正因如此，才讓蒂姆·波頓這種古怪的導演發揮了他的天賦。

於是乎，充斥著血漿與殺戮的一部R級歌舞片，就這樣誕生了。

以上就是我最愛的5部歌舞片了。

然而優秀的歌舞片很多，下面的這些也都是佳作。只不過蘿蔔白菜、各有所愛，它們沒能擠進我心目中的Top 5罷了。

比如《愛樂之城》。如果要列6部，那麼第6名肯定是它。

它代表著一部歌舞片所能做到的與時俱進、風格多元的極致。唯一的美中不足是：歌雖然足夠好聽了，可舞蹈終究是有些差強人意。

《愛樂之城》

再比如《悲慘世界》。一部恢弘磅礴、敘事精妙的傳統歌舞片，一眾明星大咖也是唱得非常賣力。

《悲慘世界》

還有華麗美好的愛情故事《紅磨坊》，和幽默搞笑的勵志故事《髮膠》。
