《小歡喜》英子《少年派》林妙妙走紅毯 一個老氣一個粗壯

天天要聞
3 天 ago

最近幾年的都市家庭教育劇捧紅了不少00後演員，比如2002年出生的趙今麥，2000年出生的李庚希。這兩位的代表作品角色分別是《少年派》林妙妙、《小歡喜》英子。在代表作品中，兩位青春無限的演員都把角色演繹得入木三分。

作為娛樂圈備受關注的00後新人，趙今麥和李庚希的口碑一直都不錯。兩位顏值不低，演技更是在新人中特別突出。

只是這兩位青春少女背後的團隊不是太給力啊，尤其是造型團隊，紅毯造型總是叫人失望。

19歲的李庚希紅毯上穿成這樣，完全像是在刻意扮成熟，但又沒有成熟的風韻。

今晚國劇盛典紅毯上的李庚希，造型依舊不被看好，甚至被吐槽為像大媽。裙子本身沒什麼問題，但請問髮型是什麼鬼？整個人瞬間老了十歲不止！

此外，趙今麥這次在國劇盛典紅毯上的造型也有些雷人，整個人顯得非常粗壯。真心搞不懂，現在的造型團隊是怎麼回事，年輕輕的姑娘，非要打扮得暮氣沉沉。

看到這樣的趙今麥和李庚希，不由得想到童星出道的林妙可。林妙可日常生活照還挺過得去，但一旦走上紅毯就槽點滿滿，完全是紅毯絕緣體。

或許，這些00後演員的造型團隊真的該好好考慮下了，自家的演員到底適合什麼樣的風格。別再讓她們穿一些亂七八糟的衣服出現在大眾面前，這樣真的很敗好感啊。
