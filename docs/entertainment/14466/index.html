編劇透露慶余年大結局，身邊好友反轉成阻礙，三詞預警讓觀眾擔憂

2019-12-14 1,236條評論 3,623次閱讀 78人點贊

《慶余年》已經播出一段時間，收視率十分可觀，觀眾好評不斷，劇中無論主角還是配角演技都十分出彩，陳道明老師不愧是老戲骨，一出場就有讓人想要膜拜的氣場，郭麒麟等配角給劇情添彩不少，每次出來都像是在說相聲，讓人笑到不行，張若昀的專業實力也可圈可點，再加上李沁、宋軼等人，整部劇可以說是顏值與實力兼備。

當然最吸引人的還是劇情，范閑本身是一個現代人，穿越到古代之後擁有著無數現代的知識，這讓他從一開始就表現出與他人不一樣的特點。而且范閑的身份也比較特殊，他的母親同樣是從現代穿越時空之後到了那個世界，她利用自己的所學，造出很多古代沒有的物品，創立了最大的商號，打造出自己的商業帝國。

還與如今的慶帝一同創立了這個國家，為陳萍萍建立起了監察員，還讓它擁有著相對獨立的權利能夠監督帝王，但是監察員慢慢成為慶帝手中的工具，為他監察百官，完成暗探和刺殺等無法擺上檯面的事情。

可以說慶國是范閑母親一手創立起來，全劇圍繞范閑穿越之後尋找世界之謎、身世之謎的過程展開，他還在這個過程中將慶國的各方面發展又完成了進一步推動，最終以自己的力量殺掉慶帝，培養出真正優秀的一代帝王。

他能夠順利了完成這一系列操作，與身邊的助力脫不開關係，這這些人也可以看作是范閑的主角光環，慶帝是他的親生父親，監察院院長與她母親是生死之交，戶部尚書是他的養父，宗師五竹隨時在旁保護著他，當朝宰相還是他的岳父。

利用這樣的身份，在慶國基本上范閑可以沒有任何阻礙，所以劇情前期的發展是范閑憑藉著自己的主角光環一路綠燈，如果按照這樣發展下去，結局也不會有什麼意外。可是根據編劇的透露，似乎並不是這樣，應該後期還會有比較大的轉折。

對於范閑的主角光環，編劇表示前期看來他有很多助力，但是會慢慢變成他的阻礙，這些幫助他的人會成為他最大的困難，這是一個讓人比較意外的地方。對於結局他用了三個詞來形容”置之死地而後生””特別意外””刀片批發”，看來最後在主角身上會發生一些特殊的事情，而且是觀眾並不願意看到的，這屬實讓觀眾十分擔憂！

編劇還說觀眾可能看完之後不想再用詞語來形容自己的感受。雖然沒有對具體內容進行劇透，但是看來結局並不如大家所期待的那樣皆大歡喜，或許還會猛賺觀眾一波眼淚，但是劇情一路平淡就失去了看點，情節波瀾起伏中帶著歡笑與悲傷才算是真實，追劇女孩們要做好心理準備了，刀片最好提前預備一下，畢竟編劇已經發話了，那最後的大招就不會放過大家的眼淚了。
