今年的韓劇反派都太帥，怎麼一笑臉就“崩”

2019-12-20 939條評論 3,064次閱讀 1176人點贊

冬天離不開各種劇，這不，一入冬就有了能「煲」的韓劇。近期播出的《精神變態日記》，可謂是全劇「鐵憨憨」，打著懸疑的名號演著正兒八經的喜劇。

工作照（右男二徐仁宇飾演者朴成勛）

男二徐仁宇「變態殺手」這個角色因為長得太帥不僅讓人恨不起來，還反而吸引很多人喜歡他，真是顏值即「正義」 。

男二劇照

男二劇照

徐仁宇的飾演者是85年出生的朴成勛，今年已經34歲的他可以說是「大器晚成」，也可能是時光的歷練使得他能時刻散發出成熟男人的魅力。

朴成勛私下工作照

年底的這部韓劇《精神變態日記》在豆瓣評分上已經超過了8分，很多追劇的網友都希望千萬不要爛尾！

同樣在今年中旬，漫改韓劇《他人即地獄》也受到廣大好評，劇情扣人心弦，結尾稱得上是高能不斷。劇中李棟旭飾演的反派牙科醫生徐文祖帥到「掉渣」，不怕反派壞，就怕反派長得帥！

劇照（左反派徐文祖）

劇照（右反派徐文祖）

雖說反派長得帥，但是這兩反派一笑就「崩」，還是「冷臉」有感覺。

要說《精神變態日記》中徐仁宇笑，那可是，不笑是「變態殺手」，一笑果斷成「憨憨」。
