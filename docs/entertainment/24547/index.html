68歲張紀中發飆，連扇成龍24歲外甥三個耳光，直言：我這是為他好

2019-12-23 321條評論 1,823次閱讀 861人點贊

近日，68歲的張紀中因為扇男演員耳光引起了熱議。原來，張紀中是在錄製《巔峰對決》，而他準備的劇本是《漂亮媽媽》。

關曉彤因為學業問題不得不離開舞台，女主角臨時換上了來救場的牛莉，好在牛莉有著豐富的表演經驗，所以她很快就入戲了。

可是，與牛莉搭戲的學徒董又霖卻表現得不盡人意，讓人感覺不到他的情緒。要知道，張紀中是出了名的嚴厲，董又霖「敷衍」的演技根本入不了張紀中的眼。

短片中有一個片段，董又霖與牛莉發生了爭執，他的情緒應該是憤怒而激動的，可想而知董又霖的表現差強人意，他還辯解稱：「自己從未有過憤怒的情緒，不知道怎麼演。」

看到董又霖不入戲，張紀中親自示範，可是董又霖卻並不認真，一直嘻嘻哈哈的。突然間，張紀中直接發飆，一巴掌拍在董又霖臉上，把他打懵了，隨後張紀中又扇了董又霖兩巴掌。

看到三個巴掌下去以後，把網友們都嚇壞了，而董又霖的表情也變了，他緊抿著嘴唇，眼睛裡看得出滿是怒氣。看到董又霖變臉，張紀中才收回手，還告訴他自己這是為他好，幫助他找到真實的憤怒情緒。

直到這是，一臉懵的觀眾們才明白，張紀中是用這樣的方式幫助董又霖入戲。顯然，張紀中的「發飆式教學」很有效，董又霖的情緒明顯激動了起來。不過，張紀中導演的幫助方式還是嚇壞了不少網友。

可惜，張紀中的幫助只起到了一會兒的作用，表演開始之後，董又霖與牛莉對戲的過程中，絲毫跟不上牛莉的節奏，就連搭戲的小演員都在演技上「碾壓」了董又霖。

表演結束之後，董又霖直言自己感覺張紀中導演很恐怖，三個巴掌令自己印象深刻。聽到董又霖的這番話之後，張紀中一直笑而不語，看得出來，他最終對董又霖的表現依然不滿意，而董又霖也並沒有感激他的「幫助」。

24歲的董又霖是一位新生代歌手、演員，因為參加《偶像練習生》而進入大眾的視野中，年輕帥氣的外表，很快就受到了很多粉絲的喜愛。

在參演《偶像練習生》之前，董又霖也演唱過一些歌曲，不過人氣並不高，他還曾演唱過電視劇《因為遇見你》的插曲。

張紀中連扇董又霖三個巴掌的視頻曝光後，立刻引起了粉絲的不滿，很多人開始質疑張紀中的職業操守。不少人覺得，雖然張紀中這樣確實能夠幫助演員入戲，可是這樣的做法卻不太妥當，有些不尊重演員。

還有很多不是粉絲的網友忍不住為張紀中捏一把冷汗，因為有消息稱董又霖的身家背景不簡單。除了是歌手、演員之外，董又霖還有一個身份，他的姨媽是成龍大哥的妻子林鳳嬌，也就是說，他是成龍的外甥。

雖然有這樣一個強大的靠山，不過董又霖並未利用過成龍的名氣，或者靠著成龍來幫自己拉資源，一直低調的打拚自己的事業，這一點讓很多網友佩服不已。

​從這一點可以看出，董又霖其實是一個非常謙虛的年輕人，相信他一定能夠理解張紀中的這「三個巴掌」，也希望這三個巴掌能夠給他以後的演藝事業帶來幫助。
