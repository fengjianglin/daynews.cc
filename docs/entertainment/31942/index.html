慶帝曰：十大老戲骨，歷史劇就服你們

2019-12-25 1,337條評論 4,768次閱讀 2498人點贊

《慶余年》可謂是近幾年最火爆的的國產劇，當然也是最火爆的歷史權謀劇。

張若昀、李沁、宋軼等一批中青年均獲得了搞對讚揚，就連路人甲乙丙丁都情商在線

首先離不了原著的非常成功，以及編導的駕馭能力，當然，還有陳道明、吳剛等幾個老戲骨的鎮台

說道老戲骨，慶帝的扮演著陳道明說：歷史劇中的老戲骨，就服他們！

第十名：馬少驊

馬少驊大人是為數不多的讓我在短時間內從路人變粉的演員，除了演技好之外，最重要的原因還是他特別適合古裝戲，馬少驊演繹的古代人物特別具有古人的氣質，充滿了古典美的感覺。尤其其對細節演繹特別到位，結合到古裝劇中，可以感受到馬大人對於劇本和角色還是用心思考，吃透了的。舉兩栗子：

《貞觀之治》的長孫無忌：無忌欣賞晉王書法，發現其字有帝王氣（脫胎於幼年高宗草劃成敕的故事），那一刻的神情，簡直絕了！個人理解歷史上的長孫無忌是一個忠於皇室，深有城府，行事頗陰鷙狠辣之人。馬大人那一刻的表演，把無忌的陰鷙城府瞬間表現得一覽無遺，至今沒有看到哪位扮演無忌的演員有過類似的演繹。

《漢武大帝》竇嬰：被囚禁牢獄中的竇嬰，對來看望他的武帝，像抓住了最後一根救命稻草一樣，眼露驚恐與希望地高速武帝，金匱之中有景帝遺詔。饒是如此，除了眼神中的流露，竇嬰的言行舉止並未顯出更多驚慌失態，仔細想想不得不為馬大人的演繹叫絕！

馬大帥需要一部有自己深深烙印的電視劇，來將自己的歷史劇地位進一步提高！

代表作品：《走向共和》、《漢武大帝》、《貞觀之治》、《北平無戰事》

第九名：鮑國安

即便在老版三國這種文戲水準逆天的劇中，鮑國安的表演也是最璀璨的，沒有之一。一般來說老版三國公認的塑造的最好角色為鮑國安的曹操，鮑國安飾演的曹操，是從整體上是真正的保持穩定的神級表演，鮑國安自己之前和後來飾演的角色也都沒有超越曹操。鮑國安的曹操緊緊圍繞的奸雄二字飾演，難得的是將奸和雄很好的融合起來，6分雄而4分奸，渾然一體。以煮酒論英雄一段，評點其天下英豪那叫一暢快淋漓，當真是藐視天下，是令人心折的豪傑，但描述之際斜眼窺視劉備，奸雄之相躍然熒屏，這一雄一奸轉化的尤為自然，可謂鬼斧神工，功力之深自不待言，這也是在我看來是其它各版曹操均不及的地方。

其他歷史劇：《鴉片戰爭》中飾演林則徐，《帝師劉伯溫》飾演劉伯溫，《西風烈》飾演秦昭襄王，《江山風雨情》中洪承疇飾演，《開創盛世》飾演隋煬帝，《越王勾踐》中飾演伍子胥，電影《趙氏孤兒》中飾演趙盾。

第八名：焦晃

從《雍正王朝》、《乾隆王朝》到《漢武大帝》焦晃飾演了三個不同時代、不同個性、不同經歷的「老皇帝」，受到觀眾的認可。其實早在「觸電」影視圈前，他在話劇界的聲望就數一數二——話劇界歷來有「南焦北於」的說法，「南焦」指的就是焦晃。

焦晃老師在《雍正王朝》里扮演的康熙這個角色，可以算得上是中國電視劇史上幾個的巔峰表演之一，幾乎每一個看過《雍正王朝》的觀眾都激動得表示，焦晃老師不是在演康熙，他就是康熙，這基本可以算是對演員演技得最高評價，我記得當時央視首播《雍正王朝》得時候，那會兒我剛剛小學畢業，這種複雜的政治戲根本看不懂，很多場景看完就忘，但有一幕我記得特別清楚，就是從木蘭圍場回京的那一幕，康熙從車裡走出來，接受百官覲見

第七名：王勁松

王老師處理角色，不僅每個角色細微處的表情動作可以大不相同，不同的角色聲線也可以天差地別。楊金水的聲音有一種恰到好處的陰柔，京腔圓潤如珠玉，當得起華麗二字；荀彧的聲音則持重而沉穩，是典型的老生。不僅如此，他在處理每個角色的時候，乃至於台詞重音都會都不一樣。特別提到台詞重音，是因為有許多很好乃至於非常厲害的演員，在處理台詞的時候都有一定的重音習慣，角色轉換之間，這種習慣不會變。無可厚非。

與明星相比，王老師像個手藝人。一部戲接了演了，事了拂衣去，深藏功與名。身後的一個個角色卻千姿百態，如觀世音三十三應化身，紮實得水潑不進。他演歷史人物，你會覺得那個人物就是他這個樣子（荀彧，湯薌茗，王蒲臣）；他演虛構的人物，你會覺得歷史上平白多出來這麼一個人。比如楊金水。

王勁松老師與馬少驊一樣，缺德就是一部完全屬於自己的歷史劇，論劇會想到他，論他會想到這部電視劇。

代表作品：《大明王朝1566》、《琅琊榜》、《軍師聯盟》、《北平無戰事》

第六名：黃志忠

黃志忠最出彩的兩部片子，個人認為是《大明王朝1566》中的海瑞和《人間正道是滄桑》中的楊立仁。黃志忠喜歡詮釋深沉厚重的人物，《大明王朝1566》他把一身風骨的海瑞演繹得入木三分。

在人間正道是滄桑中的表演超過了本應為主角的孫紅雷所飾演的角色。他對楊立仁準確到位的詮釋又迎來了多方喝彩，而正是飾演這個在信仰、感情和命運中糾結的角色，體現出來的精湛演技，征服觀眾。

尤其是《大明王朝1566》中的海瑞，成了海瑞這個角色的標杆，後來者要超越，無論是劇本、演技及導演都要開組火力，難！

第五名：于和偉

許多觀眾都是從《歷史的天空》裡面那個壞人「萬古碑」開始認識他的，此後他又相繼出演了一些或卑鄙或小人的反面人物，還有更多或鐵血或俠義的正面角色。而這些亦正亦邪的熒幕形象，在於和偉精心塑造和富有張力的表演之下，都讓觀眾記憶猶新：壞人也可以贏得淚水，好人更能贏得滿堂喝彩。尤其是先後出演三國兩個代表人物劉備、曹操，讓于和偉走上歷史劇老戲骨的殿堂，以至於很多劇迷認為于和偉的曹操已經超越了鮑國安版。

分別飾演了三國劉備和曹操，大家很感興趣于和偉何時穿上孫權的行套，玩一把江東霸主，非常期待！

代表作品：《三國》、《新三國》、《楚漢傳奇》、《軍師聯盟》

第四名：陳道明

在我的記憶中，陳老師給我印象最深的三個角色分別是

《圍城》方鴻漸——留洋知識分子慢悠悠搖擺踱步，愛把手斜插在吊帶褲的口袋裡。

《康熙王朝》康熙——給人留下太大印記，更像是傳統話劇舞台藝術在大眾傳媒的展示。

《冬至》中陳一平——腳步虛浮、外八字腳、微駝著背的陳老師，堪稱是陳道明從藝至今最為卓越的演出。

順便也說一說《二馬》里馬則仁，腆著肚子兩手叉開神氣活現。吃喝玩樂無所不會就是不會置產營生、酷愛裝大尤其是在五姨太等他喜歡的女人跟前——那腆著肚子的形態神氣簡直神了。

第三名：倪大紅

倪大紅他們這代演員的表演方式深受蘇聯派的影響，倪大紅在文革下農村的時候帶著的表演課本就是《演員的自我修養》，方法派對於身體和環境對人物內心的表現力非常看重，所以倪大紅的角色細節設計能力及其出色，他站在那兒或坐在那兒，像平時一樣，有他在，你覺得戲有了味道，畫面有了故事。

07年和張黎合作的《大明王朝》，扮演內閣首輔嚴嵩，這是倪大紅老師最經典的一個角色，同時也是中國電視劇史上的巔峰表演之一，嚴嵩在劇中已經是80歲的老人了，所以張黎當初選角的時候定位是60歲以上的老演員，而倪大紅當時只有47歲，張黎還有點不放心，但是倪大紅試完妝第一場戲後張黎就拍板了，真是神形具備，讓人嘆為觀止，倪大紅也給嚴嵩設計了稍微耳背同時又手發抖的特點，一方面這是老人常見的毛病，但另一方面又是暗合了人物的處境和內心，尤其是配合劇中嚴嵩最經典的幾場戲：《祭十二郎文》和最後的六必居牌匾，這種環境和內心的互動搭配的天衣無縫，讓人回味無窮。

第二名：陳寶國

國內電視劇專業領域再沒哪位比陳寶國的風頭更足了。2015年年末，這位今年步入花甲的大叔，在小鮮肉橫行的時代，第五次拿到中國電視劇飛天獎優秀男演員獎。

很多男演員一旦上了一線二線，常常掛在嘴邊的話是：不重複角色。但是真正對演技自信的演員，永遠不可能演出重複的角色。

比如皇帝戲，陳寶國演過《漢武大帝》的漢武帝劉徹、又演過朱元璋、還演過《大明王朝1566》嘉靖帝，但每個帝王既不會重複，也不會讓觀眾感到隔閡。劉徹那句擲地有聲的「寇可往，我亦可往」聽得人血脈僨張！

作為純觀眾的看法：他就是劉徹他就是白景琦他就是嘉靖他就是玻璃花徐鐵英牛大膽，完全感覺不到是在演戲，而是覺得那些角色真的活在這世間。
