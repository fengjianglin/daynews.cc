演藝圈金角大王謝娜，銀角大王趙麗穎，撞衫撞出來的友誼

2019-12-16 184條評論 1,579次閱讀 450人點贊

人們常說，撞衫不可怕，誰丑誰尷尬。娛樂圈有這麼兩個人，不僅撞衫撞出了友誼，還成為了很好的閨蜜。這兩個人就是謝娜和趙麗穎。人稱金角大王，銀角大王。

趙麗穎和謝娜是從一部綜藝節目，《偶像來了》認識的，《偶像來了》新聞發布會上，兩人撞衫了。不知道是不是節目組故意安排的。

後來的節目中因為兩人都穿藍白碎花裙，而很自然地坐到了一起，之後的了解當中兩人有很多相同的興趣愛好，一會就無話不談了。

之後的節目中，可能是特別的緣分吧，兩人每期穿的都差不多。趙麗穎對謝娜說：娜姐我感覺我每次都穿得不好看，但是還好有你陪我。謝娜當時就蒙圈了，哈哈。 好不容易不撞衫了，竟然選擇是同一家品牌。

兩人因為撞衫結緣，後來謝娜又邀請趙麗穎做客《嚮往的生活》，謝娜和趙麗穎兩個人就好像兩個沒長大的小女孩一樣。在節目中兩人還自封金角大王，銀角大王。

節目中趙麗穎親切的稱呼謝娜為「大哥」，謝娜則稱呼趙麗穎為「二弟」，謝娜在節目中透露：趙麗穎一年360天都在拍戲，僅有的幾天假期也會來找她玩兩天，可見兩人的關係私下裡也是非常好的。

大家身邊肯定都會有各種形形色色的朋友，可是真正交心的朋友又有幾個呢？大家羨慕這樣的友誼嗎？好了今天的分享就到這裡了。感謝大家的觀看，喜歡的點個關注，點關注，不迷路。
