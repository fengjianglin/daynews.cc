作品被豆瓣評分低，新人男星疑帶粉沒刷上，微博再diss豆瓣

天天要聞
3 天 ago

​對於一部電視電影是否好看，很多人都習慣去豆瓣看看網友的打分。原本以為這種自發的評價真實性應該很高，沒想到有網友爆料某網劇電視劇主角之一公開DISS豆瓣分數評分太低。

並且有網友晒圖爆料疑似本人在粉絲群內抱怨豆瓣評分太低，應該是希望大家在努力刷分數。

不僅如此，該位男藝人在微博上活躍連連，鬧騰的宣傳該劇，還截圖了自己和該劇的排行，微博上配文「狗頭大軍真的敲級猛」。可以說為宣傳自己作品也是盡心儘力了。

雖然最終刪掉了抱怨的微博，但看到的粉絲們在下面評論憤憤不平，豆瓣瞬間被黑哥體無完膚，讓豆瓣方表示這個鍋莫名其妙啊~！甚至袁昊本人在賬號下說「無法控制的豆瓣，在下服了」

果然豆瓣的網友都不是好惹的，看來這作死的袁昊和他狗頭大軍的發言，在下面留言毫不客氣。稱「一部完全沒腦子的瑪麗蘇劇」。

男主角袁昊鬧騰了這麼久這麼歡快，我們來看看這部劇現在的得分，4.7分！分數怎麼越鬧騰越低，不知道該劇的導演和公司會不會感覺到太陽穴隱隱作痛？

緊跟著剛剛，在豆瓣上有一個帖子疑似將刷分的事件從袁昊頭上轉移到該劇陳姓製片人身上，帖子名「關於豆瓣評分的事情要還袁昊一個清白」。

下面的網友評論不一，該事件雲里霧裡的無法理清一個真實的線索。不過也有大度的網友表明了這部劇都是新人，覺得新人難免說話過於真性情。

其實不管怎麼說，演員做好自己該做的事情就好，為不必要的事情會顯得過於輕浮，一部劇的爆紅牽涉到很多因素，不是找人刷分就能排名第一，這樣的第一拿到手也沒有意思，不如把精力放到琢磨演技上面去。
