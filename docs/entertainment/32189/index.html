有種“逆生長”叫瞿穎，雖然曾發福但已經恢復原本狀態

2019-12-24 679條評論 1,603次閱讀 694人點贊

不知道大家認不認識瞿穎，瞿穎1971年出生在湖南，在1991年參加超級模特大賽中國賽區獲得亞軍，並且還出演了電影《不要問我從哪裡來》，從此正式出道，雖然這些年來一直混跡在娛樂圈，但是卻一直不溫不火。

大家對瞿穎印象最深刻的時候應該是她和賈玲合作的小品《女神和女漢子》吧。在這個小品里，她的身材極其高挑，凹凸有致，與賈玲的形象形成了很大的反差。但是後來幾年不知怎麼回事，她似乎不怎麼注意自己的形象了，發福了很多，這下讓很多網友接受不了，瞿穎也受到了很多爭議。

但是瞿穎不愧是模特出身，似乎注意到了自己的形象不被喜歡，也或許迫於外界的壓力，很快就回到了之前的狀態，女神又回來了。通過看瞿穎在社交平台上發布的個人動態來看，她的身材越來越好，絲毫沒有受到年齡的影響。前段時間發布的兩張照片讓很多網友直呼受不了，照片中的她身穿比基尼，沐浴在水池中，一顰一笑堪稱小妖精，說是出水芙蓉一點都不為過。真是沒想到她的身材不但回到了以前的狀態，更是比以前猶有過之，給人的感覺就像是逆生長了。

有的網友說：「瞿穎不愧是當過模特的人，光看這身材怎麼也不像接近半百的人，而是一個二三十歲的小姑娘」。還有的網友說：「前一段時間到底是怎麼回事呢，為什麼突然發福了，每個人都有懶惰心理，或許是慵懶了」。小編想說的是：「不管瞿穎那點時間經歷了什麼，但現在至少是好的，身材出現了讓人心動的逆生長，希望她能夠一直這麼美下去」。網友們還有什麼想法呢？歡迎點贊評論。
