辨識度超高的韓國女星，主演韓劇百看不厭

2019-12-19 705條評論 4,869次閱讀 925人點贊

小編將會分兩期給大家介紹12位辨識度超高的韓國女星，基本都是想當年國內熱播過的韓劇女主，喜歡想重溫的可以當成熱播韓劇推薦單哦！

1. 李英愛

李英愛曾憑藉古裝女性勵志劇《大長今》中的徐長今一角火遍大江南北，李英愛在韓國也是家喻戶曉的女明星，尤其是古裝韓服的裝扮更是被稱讚為古典長相第一人。

2. 宋慧喬

宋慧喬的長相頗受內地觀眾的喜愛，主演的多部熱播劇更是收視口碑雙豐收，代表作如《藍色生死戀》《情定大飯店》《浪漫滿屋》《那年冬天風在吹》《太陽的後裔》等。

3. 全智賢

全智賢之前憑藉電影《我的野蠻女友》火遍亞洲大街小巷，2012年由其主演的愛情喜劇《來自星星的你》又徹底大火了一次，刁蠻任性的千頌伊真的是可愛迷人。跟李敏鎬主演的《藍色大海的傳說》又展示了一次美妙身姿。

4. 李多海

左一李多海

李多海的中文說的是相當不錯呀！《我的女孩》真的是百看不厭，堪稱經典。《你好小姐》《綠薔薇》等也是好看到不要不要的。

5. 韓佳人

「媚鼻女王」韓佳人真的不僅有辨識度還美的氣質耐看。韓佳人主演的《魔女幼熙》《黃手帕》《擁抱太陽的月亮》也是韓劇經典，短髮長發古裝現代都是絕對出挑的大美人。

6. 韓彩英

韓彩英自從嫁入豪門之後，鮮有影視作品了。之前的愛情三部曲《火花遊戲》《威尼斯戀人》《豪傑春香》都在芒果台熱播過，小編本人最喜歡《豪傑春香》了。

下期將會為大家介紹剩下的6位韓劇虐戀女主，大家可以點進主頁觀看哦！大家也可以在評論區留言長得好看的韓國女星，謝謝大家的關注哦！
