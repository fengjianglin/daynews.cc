今天是12月5日，李榮浩的新歌《麻雀》「如願」上線了。

為什麼說「如願」？因為就在上個月，他在社交平台上透露，新專輯從今年5月開始已經製作得差不多了，但是發表日期遲遲不能被提上日程，「一直在等公司安排發行，我想問一下各大偉大的音樂平台和我偉大的經紀人，我想上一首新歌，就那麼難嗎？」

李榮浩還說，從9月開始一直等到現在，一問就被告知「需要流程」。「天天用流程搪塞我……要是12月5日發不了，我就找個電動車維修論壇，發個帖子，把歌發了。」

一條「發歌難」的動態，獲得「不明群眾」兩百多萬的點贊和十幾萬的轉發……有業內人士含沙射影地說：呵呵，李榮浩真是個自媒體營銷奇才。

像李榮浩這樣的歌手發歌，在音樂平台上發歌難不難？

人家音樂平台的人說了：根本就不難！作為頭部藝人，任何平台都會搶著要他的歌。

何況，李榮浩是自己運作的公司，所有安排他說了算。而他埋怨的「偉大經紀人」呢？他們依然「恩愛有加」呀，經紀人也沒有引咎辭職，還在朋友圈裡為李榮浩賣力推新歌呢。

種種跡象表明，這是一場有計劃的炒作事件。

別看李榮浩眼睛小小的，卻裝得下雞毛蒜皮的事：他和汪峰老師一樣，會親自審核稿件，誰寫他不好，他會牢牢記住。

誠如業內人士所言，李榮浩的確玩得轉互聯網，你看他熱搜不斷：什麼手機欠費停機事件，什麼楊丞琳演唱會激吻事件……李老師算得上經驗老道。

這一點，也深深地影響了他的妻子楊丞琳。

就在上個月4日，楊丞琳突然刪光所有社交平台的內容，包括李榮浩向她求婚的照片，並且將頭像換成了空白圖，還在動態里接連發問：你曾經被親近的人背叛嗎？你在愛中曾變得不堪嗎？你想努力忘掉某段記憶嗎？

網友還以為她和李榮浩情變，在一片扼腕聲中將楊丞琳送上熱搜。事件發酵幾天後，她才慢悠悠地解釋說：哎喲，是你們想多了。隨後，丟出了新專輯《刪·拾 以後》。

有些網友實在不解：為啥要用這種方法吸睛？用作品說話不行嗎？

對啊，就是因為作品不行，這兩口子才玩這招嘛！

李榮浩和楊丞琳的新作品並沒有獲得認可，網友毀譽參半，看幾條這兩口子想舉報的評論吧：

李榮浩的《麻雀》推出後，有網友就說了：炒作低級，無聊歌曲。


楊丞琳的專輯《刪·拾 以後》推出後，網友尖刻評論：正因為專輯乏善可陳，才需要炒話題吧！

又要說一句老生常談的話：歌手還是應該用作品說話。作品獲得認可，自然會受到網友的擁戴，靠小聰明來博取關注，終有一天會嘗到作繭自縛的苦頭。
