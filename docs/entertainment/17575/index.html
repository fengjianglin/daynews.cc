小熒屏年終盤點，甜寵教育仍舊火爆，古裝少而精緻，宮斗退出競爭

2019-12-17 1,584條評論 3,469次閱讀 667人點贊

2019餘額告急，年終總結正在進行，各大平台盤點的年度電視劇10強榜單也將全數面世。以大數據為支撐的熱度榜與粉絲投票而成的人氣榜單雖有些許不同，但名單中的每部作品，都在2019引發過熱議。

右為百度沸點十強名單，左為電視劇大賞投票名單（截止至12月16日）

基本上，必有一部是讓觀眾冒著「禿頭」風險狂追過的作品。

但新的一年，新的數據，新的劇名，也創造了新的格局。從各榜單處前位的作品中，可以發現一些微妙的變化，有些意料之中，也有意外之喜。

甜寵仍是收視良藥，教育最能引發話題

2019有神人總結了娛樂圈的頂級人氣男星，用了「四大三小雙頂流三爆爆」來形容，其中「三爆爆」李現、肖戰、王一博皆是本年度人氣飛升的藝人。這三位能成人氣頂流，與年度爆劇《親愛的，熱愛的》、《陳情令》息息相關。

而這兩部作品，雖然題材完全不同，卻皆以「甜」獲勝。無論是男歡女愛還是兄友弟恭，劇中主角們友愛又甜蜜的互動，給予觀眾無限遐想空間，是忙碌生活的中高糖調味劑，很難不去品嘗。

除此之外，仙俠甜寵《招搖》、校園純情《初戀那件小事》以及發一顆糖後瘋狂虐到結尾的《東宮》，主打看點都基於甜寵之上。年度電視劇人氣榜中甜寵或偽甜寵占居多席，可見甜寵長盛不衰，但產量最大的甜寵脫穎而出概率並不高。

從《陳情令》、《親愛的，熱愛的》的走紅便可以看出，甜寵雖討喜，但量與新鮮感才是取勝王道。

甜寵是收視良藥，家庭教育類題材則稱霸話題，若提及本年度討論最為持久深入的關鍵詞，非「原生家庭」莫屬。正午陽光用一部《都挺好》向觀眾詮釋何為「不幸的人用一生治癒童年」，原生家庭缺失的成年人，崩潰只在一瞬間。

《都挺好》的出現，給無限憋屈的成年人一個發泄出口，也讓觀眾看到更豐滿的世界觀。有時候孝道與教育需要雙向考慮，當我們真正意識到原生家庭的影響性時，才能將悲劇不再延續下去。

原生家庭越刺痛成年人，親子教育便越會受關注，黃磊海清兩位育兒專業戶再次出現，張嘉譯、閆妮也加入高考家長陣容。《小歡喜》、《少年派》兩部以家庭教育、高考為題的電視劇，不斷稱霸熱搜榜。

磕磕碰碰的教育困局，總是大眾觀眾的焦點，也能引發各式共鳴。編劇只要能窺視到生活中的觸點，便可以輕鬆收穫熱議，但聚集問題之時還需給出對「小歡喜」的思考，否則只能引爆話題卻難拿優秀口碑。

以甜寵與教育類題材的話題度，2020仍舊少不了它們的身影。

宮斗退出競爭，宅斗、權謀重新出發

2019受限古令影響，古裝劇上新相對下降，但人氣作品卻並不缺失。年初正午陽光靠《知否》完美開局，年未張若昀又帶來老戲骨雲集的《慶余年》，加上年度口碑力作《長安十二時辰》與幾部古裝甜寵，成績相當突出。

然而，2019年的古裝劇與往年相比也出現了明顯的格局變化，其中之一便是宮斗產量的壓縮，這一年幾乎沒有出現人氣宮斗劇。雖與市場調節有很大關聯，但宅斗題材《知否》、權謀劇《慶余年》的受寵，已經暗示觀眾口味的變化。

熱絡多年的宮斗，明顯已經在不斷積累的產量中讓觀眾審美疲勞，在2018年其實已經出現預兆。彼時正統宮斗《如懿傳》因延續了老式風格讓觀眾逐漸審美疲勞，雖然質量頗佳卻無法做到現象成績，反而是《延禧攻略》靠反套路爆紅。

依靠「獵奇」獲寵的宮斗題材，很難再重回受寵榜，不過這並不算一件壞事。對於劇粉來說，多元化的題材，顯然能夠獲取更多樂趣。至少，披著搞笑外殼的《慶余年》便讓觀眾眼前一亮，帶著古風古韻的《知否》，更具文化內涵。

一個泛濫題材的縮減若能讓更多題材登上舞台，委實是件樂事。期待2020古裝題材，能夠繼續放棄宮斗，挖掘出更多新鮮的切入點。

大男主IP繼續撲街，但也找到改編曙光

2017-2018年是大男主IP劇化的狂熱年份，從《擇天記》到《武動乾坤》再到《斗破蒼穹》，一個比一個卡司華麗，但一個比一個撲的慘烈。對於這一類題材，製作方們熱衷粉絲也期待，但成品難受認可是一大難題。

而至2019，大男主IP劇化仍舊存在困局，由口碑人氣兼具的95後演員劉昊然出演的《九州縹緲錄》，製作相當用心，但仍舊沒有逃過「雷聲大雨點小」的局面。觀眾認可劇方的誠意，也點贊演員的表現，卻實在難被冗沉枯燥的劇情吸引。

最終，《九州縹緲錄》步入《九州牧雲記》同樣的困局之中。

然而，同樣在2019，大男主題材也終於找到了一絲成功的曙光，出現了《長安十二時辰》、《慶余年》、《全職高手》三部人氣與口碑兼具的優秀作品。它們的成功，給大男主IP劇化增添一支強心針，也證明這一類型作品並不缺觀眾。

這三部大男主作品的成功給了製作方們一個破局思路，用心的改編非常重要。劇情呈現一定要尊重影視化習慣，不能原封不動照搬文字的節奏，否則很容易讓觀眾失去耐心，《慶余年》編劇突出搞笑基調便是不斷拔起觀眾興趣的重要助力。

同樣的，在影視化過程，一定不能失去原作的核心魅力，要對觀眾保有信心。很多大男主題材改編會突顯戀愛線，甚至是強組CP，一定程度上是對觀眾的信心不足，認為只有甜蜜的戀愛戲才能俘獲觀眾芳心。

然而，專註懸疑燒腦的《長安十二時辰》，成為本年度古裝口碑NO.1，熱度也相當出色，甚至讓人氣偶像易烊千璽成功轉型。而由楊洋出演的《全職高手》完全保留原著熱血電競風，拋棄了增添戀愛的慣用套用，反而口碑熱度皆獲成功。

幾部大男主IP優秀的改編，是成功的基石，也給未來大男主IP一條不錯的指引。因為這同部作品的成功，相信在2020大男主IP劇化會更加火熱，但最終誰能脫穎而出，便看劇化夠不夠有魅力了。

類型片出彩，製作方突破安全區

2019年新劇，最讓七話驚喜的莫過於類型片的出現，製作方終於不再把焦點放在曾經泛濫的宮斗、瑪麗蘇言情、諜戰類作品之上，而是開始尋找一些國產劇罕見涉及的題材，嘗試突破他們的安全區。

懸疑古裝《長安十二時辰》、不談戀愛的《全職高手》甚至變相版的「都市妖精」《動物管理局》，都讓觀眾喝彩，這些作品製作也相當精良。即使在青春作品圈，也出現了真正接地氣的《我在未來等你》，挖掘戀愛以外的青春記憶，非常難得。

觀眾其實早已看膩了一些跟風製作的常規題材，在類型片的需要上非常之高，這一點大熒屏中《戰狼》、《流浪地球》、《哪吒》三兄弟受追捧，便足以證明觀眾對類型片的迫切渴求。只是曾經很多製作方不敢嘗試，總是把重心放在易出爆款的言情的題材之中，慢慢局限了創作力。

當下，受大熒屏影響的小熒屏，也開始陸續出現吳京、郭帆這樣的「出頭羊」，這是本年度最值得點贊的現象。若能保持住類型片的深挖、國產劇題材上的突破，一定會給到觀眾更多驚喜，也會促成國產劇的良性發展。

2020，我們拭目以待。
