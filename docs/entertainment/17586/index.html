TVB三部劇集收視回升卻不值得慶祝 這部劇集或許將打破收視新低

2019-12-17 1,116條評論 2,190次閱讀 845人點贊

本文編輯劇透社：3gor47

未經授權嚴禁轉載，發現抄襲者將進行全網投訴

熱播劇《鳳弈》《多功能老婆》《愛回家之開心速遞》近日成為了網友的熱議對象。皆因收視滯後報告中三部劇集的收視點都有所上升，但不少網友對此僅評論為「在家看電視的人多了」而已。

從收視的滯後報告看出，收視的日期是12月2日至12月8日，這次的收視報告用平平無奇四個字就可以概括，令人關注到的是《鳳弈》《愛回家之開心速遞》及《多功能老婆》收視點都有所回升。

《愛回家之開心速遞》是一部長篇的都市生活情景劇，劇集現七百多集，隨著之前宣布從明年起一個星期會播出七集，每天不間斷，讓不少本身劇迷都十分興奮之外，從12月2日到8日的收視報告中看出，該劇集一個星期的平均收視僅有25.7點，比前一個星期要高一點。

但是這部劇集對很多沒看過的網友來說，想要從頭追起估計是一件難事，畢竟劇集集數太多。收視有所回升，的確是件好事，但似乎並不值得慶賀，不少網友則認為《愛回家之開心速遞》吸引的多數都是老劇迷，而作為處境劇，劇情的發展始終有限。

亦有網友認為，明年將繼續播出的該劇，或許在口碑上會有所提高，畢竟處境劇的劇情容易入戲，加上之後都會加入不少新面孔，相信都會為該劇帶來不錯的收視。

《多功能老婆》的平均收視在26.2點，是三部劇集中最高的一部，但仍有不少網友不買賬。

劇情講述的是擁有超多「技能」老婆，面對出軌丈夫時身邊的閨蜜一直支持、亦出現了暖男幫助「多功能老婆」走出情傷，開展自己的新生活。

雖然演員陣容上有看點，朱晨麗作為新一代被力捧得小花在劇中的笑點、看點都勝過其他主演，新劇中的表現都被不少網友誇讚有進步。

陳煒的演技一如既往的出色，飾演女強人似乎都挺配角色的人物性格，而演小三的於淼也是此次劇中比較有看頭的演員，於淼飾演的小三被網友稱為「霸道小三」，讓不少網友紛紛表示演的很入戲，導致有網民都好「討厭」她。

但是由於劇情題材的局限性，似乎沒有太大的亮點，劇情的發展也是大眾心中的發展路線，極其平淡，沒有驚喜之處，收視的有所回升或許只能表達出觀眾想要看演員的演技多過劇情的發展。

三部劇集中最慘的就是《鳳弈》，《鳳弈》雖有視帝黎耀祥的加入，卻依舊無法拯救收視，上周的收視僅18.1點，劇集放在近年高峰位劇集上，這個收視點絕對不是一個理想的數字。不少網友猜測《鳳弈》這部劇集將會創「新低」，而且是近十年來的新低。

綜上分析三部劇集，雖然收視有所回升，卻並不值得慶賀。你是怎麼看待這三部劇集的呢？在你心目中這三部劇集有沒有吸引之處呢？
