娛樂你問我答之：胡歌新片不佳影響不大？劉詩詩吳奇隆恩愛如初？

2019-12-14 1,409條評論 2,863次閱讀 226人點贊

1、爆客，胡歌新片不佳影響資源？

最近有粉絲來問胡歌、桂綸鎂主演的新片《南方車站的聚會》，排片場次並不算很好，如果票房不大爆會不會影響到他後續的電影資源。爆客在這裡給擔心的粉絲回答一下，目前這個情況對於胡歌來說其實還好，雖然這次新片的類型確實不屬於能獨攬票房那種，但新片口碑是穩穩的拿下了，而且入圍戛納主競賽單元電影的《南方車站的聚會》已經算是對主演們和作品的認可。以胡歌現在的狀態，有演技實力傍身和圈內的認可，就算新戲沒有大爆也不會太影響他後續的電影資源，這個不用多想。

2、爆客，劉詩詩配不上吳奇隆？

自從劉詩詩不再人淡如菊之後，她那從來都不人淡如菊的粉絲，就開始舞了起來，從她復出開始，就一會兒撕這個，一會兒撕那個。其中最被劉詩詩粉絲厭惡的，就是劉詩詩的老公吳奇隆，所以近來劉詩詩粉絲可以說是一直在揶揄抹黑吳奇隆。

面對這種情況，吳奇隆粉絲現下那也是是可忍孰不可忍，所以已經忍夠了的吳奇隆粉絲，也開始有組織有目標的狙擊劉詩詩，把吳奇隆的風光實績一一列出，表示就劉詩詩這種靠碰瓷上位的貨色，能嫁給二婚的吳奇隆簡直就是高攀，畢竟吳奇隆的前女友可是蔡少芬、楊采妮這樣的女神，劉詩詩放到她們跟前，撿鞋子都不夠格。面對吳奇隆粉絲有理有據的開黑，當下不僅是劉詩詩粉絲，連她的團隊都是不勝其煩，要知道復出之後，劉詩詩就一直沒有新代言入手，這個時候她的風評要是再不好，那她後續的發展可就真的會被唱衰了。

比起吳奇隆的實績，劉詩詩的那點「家底」確實不夠看，但好漢不提當年勇，就當下來看，劉詩詩和吳奇隆的婚姻關係還真是女強男弱，要不是吳奇隆手上還有中花在握，內娛的資本們怎麼會任由他招搖撞騙，四處空手套白狼呢。當然了，不管外間如何流言紛擾，吳奇隆和劉詩詩倒是恩愛如昔，吳奇隆還頻繁探班劉詩詩劇組，現下劉詩詩已成花旦里拖家帶口拍戲的第一人了。

3、爆客，鞠婧禕碰瓷被黑？

之前鞠婧禕那套黑色禮服裙走紅毯圖出了圈，她那次的紅毯表現力確實不錯，不光艷壓了好幾位小花，還把前同事李藝彤壓得毫無存在感，這次出圈讓她得到了很大的曝光度，也讓其擠進了古力娜扎、baby一列擁有高顏值一掛的藝人群體，這對她一個小作坊出來的藝人而言，算是提了咖位。但她的顏值出圈，也讓不少同行心生妒忌，最近有關鞠婧禕小網紅時期的照片頻繁被人挖出來，對方的目的就是給觀眾洗洗眼睛，讓大家看看曾經的她是那種摸樣，同時也打壓一下鞠婧禕團隊，讓其別太囂張。鞠婧禕的舊圖也挺多的，和以往顏值相差甚遠的她經不住挖，鞠婧禕團隊看到這些圖片也很著急，只能慢慢公關慢慢刪帖，降低圖片傳播量。

4、爆客爆客！林更新在幹嘛呢？

林更新前段時間忙著拍戲，就是那部現代戲《最初的相遇最後的別離》，10月剛殺青，預計明年播。他拍戲屬於間歇性的那種，會給自己休息期，現在正在看新劇本，下一部應該還是現代戲，打算年後開機進組。所以在拍戲的那段時間他偶爾上些熱搜保持下熱度就可以了，比如前一陣「難道我不該在1那行嗎」還有「喜歡我的人都不是少女了」的熱搜，內容一如既往的懟粉絲and十分符合他的沙雕風格，這樣就夠了。另外他現在還做副業了，開了烤肉店，所以事情也不算少。

5、爆客，陳赫的前妻，許婧，最近怎麼那麼多的負面新聞，好像有點不正常呢？

陳赫的前妻許婧，憑空出來N多的負面，不厚道拿了陳赫錢、bao養小鮮肉、男朋友XD等等，反正各種亂七八糟的都出來了，而與此同時，不知道大家發現沒有，陳赫和現任老婆，張子萱的正面新聞，多了不少，恩愛有加，好爸爸好媽媽，萱萱身材好，時尚感好，要進軍娛樂圈等等。說白了，現任不就是想復出進娛樂圈么？真的沒有必要去對前任各種說三道四，婚內出軌，沒得洗。
