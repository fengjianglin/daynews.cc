可憐的伍宇娟：曾與鞏俐齊名的影后，離婚破產沒戲演，54歲仍單身

2019-12-10 350條評論 1,716次閱讀 525人點贊

中戲96級明星班走出了趙薇、黃曉明、陳坤等不少家喻戶曉的大咖巨星，其實中戲85級也曾名震一方，這個班裡誕生了包括鞏俐、史可等明星，伍宇娟也是其中備受矚目的一位。

伍宇曾與鞏俐、史可等並稱中戲五朵金花，23歲一脫成名，獲得金雞百花最佳女主角提名還一舉拿下「城市獎」最佳女主角，27歲又摘得金熊獎，躍升為電影界最受矚目的新星。

誰料感情坎坷，一婚嫁給初戀賈宏聲（後成周迅男友），老公卻染上毒癮，她曾下跪求男友戒毒，5年感情終成空；選擇美籍華人結婚，生下兒子，卻被誣陷給丈夫戴綠帽；婚姻破裂後，她欠債無數，忙於掙錢養家，放在妹妹家養的兒子喊了自己4年半姨媽，

如今54歲的她，已經當了18年單親媽媽。伍宇娟一生命運坎坷，屢次遇人不淑，走過半生，她終於找到了想要的生活。

初戀吸毒早逝

伍宇娟生自湖南耒陽，自幼家境貧寒，14歲加入耒陽花鼓戲團，20歲順利考上中戲。

在大學時，伍宇娟就已經艷名遠揚，男同學將她和同為85級的鞏俐、史可、金莉莉、陳煒並稱為「五朵金花」，五位芳華絕代的美人中，伍宇娟是最清純靚麗、氣質脫俗的那一個。

美得不可方物的伍宇娟，還時常登上掛曆封面，她是整個80年代人氣最高的掛曆美人之一，也是萬千男人心中的夢中女神。

伍宇娟可不是毫無靈魂的花瓶，大三那年，她就展現出了一個演員的決心和魄力。

1986年，導演周曉文正在為《瘋狂的代價》選角，女主角需要有裸露鏡頭，很多女演員忌憚於保守的社會風氣，知難而退，伍宇娟卻迎頭直上，出演了這部大陸首個出現裸露鏡頭的影片。

憑藉精湛的演技，伍宇娟一舉斬獲百花獎和金雞獎最佳女主角提名。雖然這部電影為她招來不少非議和罵名，但她也迅速躥紅，一位電影界的明日之星就此誕生。

1990年，伍宇娟參演電影《龍年警官》，再次拿下百花獎最佳女配角，此後事業一路高飛。

1991年飾演《雪山飛狐》袁紫衣一角，成為一代熒幕經典，92年又與劉曉慶聯合出演《香魂女》，摘得第43屆金熊獎。

事業得意的伍宇娟，情場卻屢遭磨難。彼時，她的男友是中戲時的同班同學賈宏聲，也是當時最具偶像氣質的青年演員。

在中戲時，賈宏聲已是風靡全校的帥哥，導演張揚評價他「賈宏聲那時又高又帥，又挺高傲的，非常受女人喜歡。」

賈宏聲成名很早，大三便一炮走紅，此後又拍攝多部電影、話劇，可謂有才有貌，年輕多金。

伍宇娟喜歡活潑陽光的男人，而賈宏聲正好是陽光男孩的類型，伍宇娟因此愛得痴迷，「不知為什麼，我一見到他的眼神，就會產生一種異樣的情感」。

而賈宏聲也對風華正茂的伍宇娟情有獨鍾，為了追求她，不惜得罪班上其他男生。

兩人一片濃情蜜意，相戀不久，就結成夫妻，眾人眼中，他們郎才女貌，性情相投，是最幸福美滿的情人。

誰知，不幸突降，賈宏聲染上毒癮，性情大變，越來越瘋癲和偏執。

為了勸阻丈夫吸毒，伍宇娟不惜下跪哀求他，但賈宏聲依舊沉溺毒品，徹底看不到希望後，伍宇娟揮淚告別了這場長達5年的愛情。

2010年，前夫賈宏聲從17樓一躍而下，43歲便英年早逝。

回憶起昔日戀人，伍宇娟忍痛道，「賈宏聲是一個不願意帶著各式各樣的沉重面具而生活的人，其實我也不想戴，但我還是給自己找了些理由，把這些面具一個又要給地套在了自己臉上，這就是我自己的悲哀！。」

美麗的童話故事，終以生離死別收梢，人間獨留她一個千古傷心人。

伍宇娟畢生的夢想，莫過於執子之手，與子偕老，容貌也好，演技也罷，對她而言不過身外之。

為了家庭幸福，伍宇娟甘心放棄自己的事業。然而她想要的，偏偏得不到。

與兒子相依為命

1994年，一次機緣巧合，伍宇娟偶遇了美籍華人李慕華，李祖籍上海，在洛杉磯大學擔任經濟學教授，不僅才貌雙全，而且活潑幽默，李慕華對伍宇娟一見鍾情，就此展開猛烈攻勢。

伍宇娟回家探親，李慕華也一同隨行，還為伍宇娟就讀過的小學捐資搭建新校舍。李慕華的真誠追求打動了伍宇娟，不久她便以身相許，婚後兩年誕下一子。

可伍宇娟和賈宏聲的戀情早已人盡皆知，不知從何時起，外界便流傳風言風語，稱伍宇娟的兒子是賈宏聲的私生子，李慕華被戴綠帽子了。

李慕華卻信以為真，不斷懷疑伍宇娟，加上語言和文化差異，兩人爭吵不斷，感情裂痕越來越大。

1997年，伍宇娟自編自導自演電影《山茶花開》，不料賠得血本無歸，時常有人打電話恐嚇她，還有人把她堵在飯店門口。

走投無路之際，伍宇娟向丈夫尋求幫助，對方卻冷漠道，「放著好好的演員不做，出什麼風頭」，「我沒錢，你另想辦法吧」。

伍宇娟對這個男人失去了最後一絲信任和愛意，兩年後，她選擇了和平離婚。

背負巨大債務壓力，兩度遇人不淑，伍宇娟早已傷痕纍纍，但為了年僅1歲半的孩子，她必須強大起來。

於是伍宇娟拚命拍戲，先後參演電影《尋槍》和電視劇《沒有硝煙的戰場》，為了早日還清債務，她把年幼的兒子交託給遠在長沙的妹妹和妹夫。

由於陪伴不多，兒子多年以來，一直誤把伍宇娟叫做姨媽，十歲那年，才得知伍宇娟的真實身份。可憐天下父母心，個中苦楚，怕是難以言表。

離婚後，伍宇娟曾遇到過合適的姻緣，拍攝電視劇《用我的生命去愛你》時，她與同劇組的路羽譜寫過一段戀情。

生日那天，伍宇娟收到了路羽的鮮花和祝福，兩人從此步入戀愛階段。

「其實，具體要說她身上什麼東西吸引我，我還真說不出來，可能就是一種很微妙的感覺，提及伍宇娟時，路羽常常透露著幾分甜蜜「沒拍戲時，我們經常不自覺地就會在一起聊天，慢慢的，就喜歡上了對方。」

可這段戀情最終還是無疾而終，個中緣由或和伍宇娟的兒子有關。

伍宇娟曾對好友坦言「不是我不願意找對象，我只是擔心虧待了孩子，我沒能給嘛香一個完整的家，如果再讓他受到繼父的虐待，我一輩子都不會心安！」

出於對兒子的虧欠感，伍宇娟逐漸隱退影壇，一心陪伴兒子成長，給兒子送飯，陪兒子上學，不再減肥、應酬、上節目，日益遠離鎂光燈下的生活。

回憶起自己前半生的演員生涯，伍宇娟言談中透露著些許疲憊：「我一直過著動蕩不安的生活，很小就離開家唱戲練功，後來到處拍戲，干這行差不多30年，有點厭倦了。每天早上我睜開眼就要先想想自己在哪個城市的哪家旅館」。

如今，伍宇娟的昔日同窗，或揚名立萬，載入史冊，或嫁入豪門，享受天倫之樂。而伍宇娟奔波半身，人財兩空，年過半百，仍是孑然一身，昔日當紅美女明星，落魄至此，實在令人唏噓。

結語

「我所有的情感，我都會特別真心。特別努力去珍惜我所有的情意。即使年輕時因為對方或是我的問題有矛盾，但是過後回憶的時候，還是覺得挺美好的。」

如今的伍宇娟已經對一切釋懷，或許，生命不只是在於結果，經歷過其中的酸甜苦辣，便無需後悔。

或許伍宇娟的事業談不上成功，情路也算不上幸運，但人生無需取悅他人，自己覺得幸福的人生，便是最好的人生。

歷經磨難，仍然心懷感恩，笑對人生，伍宇娟令世人敬佩，也值得我們祝福。
