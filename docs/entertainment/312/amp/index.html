收 藏

由劉亦菲、鞏俐、甄子丹、李連杰等群星主演的《花木蘭》前不久發布預告片，和國內版預告不同，北美版時長多了幾十秒，有不少新的畫面，對故事介紹也更加全面。

木蘭家族的吉祥物鳳凰出現（貌似木須龍是不會有了），木蘭的父親也點出其象徵意義：「她是皇帝的保護者，最為美麗、強大」。以及木蘭上女紅妝的過程，也介紹了反派——李截飾演的可汗。

影片將於明年3月27日上映，改編自1998年同名動畫片，和動畫片歌舞片模式、歡樂的氣氛不同。這次在真人版似乎想走嚴肅路線，古裝史詩的感覺。我們看看預告給出的畫面和《木蘭辭》是如何對應的吧！

唧唧復唧唧，木蘭當戶織

不聞機杼聲，唯聞女嘆息

軍書十二卷，卷卷有爺名

阿爺無大兒，木蘭無長兄

願為市鞍馬，從此替爺征

朝辭爺娘去，暮宿黃河邊

旦辭黃河去，暮至黑山頭

不聞爺娘喚女聲，但聞燕山胡騎聲啾啾

萬里赴戎機，關山度若飛

朔氣傳金柝，寒光照鐵衣

將軍百戰死，壯士十年歸

鳳凰替代了木須龍

李截扮演的反派可汗

鞏俐扮演的反派

歸來見天子，天子坐明堂

當窗理雲鬢，對鏡貼花黃

2020年3月27日上映
