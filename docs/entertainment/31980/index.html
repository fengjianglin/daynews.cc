吳磊和張子楓換話筒，兩人商業互誇，張子楓稱他是暖心的大哥哥

2019-12-25 978條評論 4,756次閱讀 2381人點贊

近日，電影《寵愛》在北京舉行首映禮，吳磊、張子楓、李蘭迪、郭麒麟、于和偉、陳偉霆、鍾楚曦、鍾漢良、楊子姍、檀健次、闞清子等各位主演們亮相現場。

在電影《寵愛》首映的見面會現場，各位主創們的友愛互動吸引了眾多網友們的關注。

在交流的過程中，張子楓話說到一半，話筒突然沒聲了，一旁的吳磊很迅速地把自己的話筒遞給了子楓妹妹，並接過了子楓妹妹手中壞掉的話筒。

不得不說，三石弟弟反應實在太迅速了，不愧是以暖心著稱的紳士磊。

眾所周知，吳磊是個情商極高的小暖男，他總是能讓身邊的人因為他的存在感到舒服，看來這一換話筒事件又可以記錄到他的高情商檔案中去了。

在此期間，吳磊和張子楓還進行了商業互誇，不得不說，兩人的互誇真的就是「商業互誇」了，一直在說對方特別好，除了好還是好，沒有任何吐槽，還被徐崢調侃「兩人太官方了，看來真不熟」。

張子楓還提及，在拍戲的過程中，吳磊一直很照顧自己，並誇他是一個暖心的大哥哥。

雖然在眾多網友眼裡，吳磊和張子楓這對少男少女之間的互動一直很甜，並站了兩人的cp。

吳磊和張子楓，一個是國民弟弟，一個是國民妹妹，兩人都是童星出道有顏值有實力的老戲骨，就很身高差都很搭，不過小編覺得，兩人不管是在戲裡還是戲外，應該都只是純純的革命友誼，以及為了拍好電影而合作的工作搭檔，畢竟兩人在新生代演員中都十分敬業，口碑極好。

在《寵愛》主創們最後大合照時，吳磊弟弟男友力MAX，一把將郭麒麟摟到自己的懷裡合影，就連旁邊的子楓妹妹也忍不住默默為他們讓出了道。看來我們的三石弟弟無論對男生對女生都有男友力，男女通吃啊哈哈。

吳磊在電影《寵愛》中飾演一個盲人小哥哥，在電影交流會中，吳磊提及剛開始演這個角色對自己很沒信心，但後來他幾乎看完了所有關於盲人的電影，硬生生地把自己變成了「盲人專家」，最後如此充分的準備程度讓導演和劇組工作人員都感到震驚。

其實一直以來，吳磊都以「拚命三石」著稱，無論是在工作中還是學習中抑或是生活中他都是一個很認真的人，總之很期待吳磊的在電影《寵愛》中的表現，期待他的陳樂雲。
