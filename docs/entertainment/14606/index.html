青年演員胡一天，高大英俊帥氣，他的未來不可估量

2019-12-13 1,037條評論 2,125次閱讀 795人點贊

胡一天是青年演員，出生在1993年12月。他長相帥氣，1米88的個子。天生的演員。我第一次注意他的時候是他出演【致我們單純的小美好】里的男主角江辰。在劇里從學生時代演到參加工作。【致我們單純的小美好】主要講述了傲嬌腹黑的江辰和活潑的元氣少女之間發生的愛情故事，從學生時代的青梅竹馬到最後參加工作後的再次相遇，再到最後走到一起經歷了19年，江辰才認清自己內心，終於直面陳小希了。兩個人結婚後的甜蜜生活也是一度甜到讓人羨慕。

後來在今年夏天又看到胡一天在熱播劇【親愛的熱愛的】裡面出演了電競高手吳白，大家都叫他小白。這部劇的收視率非常高。楊紫捧紅了李現，也帶火了胡一天。雖然胡一天在劇中的戲份比較少，可是每次出場都能吸引大批粉絲，在劇里他是吳白是世界第一的高手，粉絲不計其數。戲外他的粉絲更是多如牛毛。在劇中他的智商情商都十分高，雖然話不多，但每次說話都恰到好處。韓商言和佟年的結合有小白大大的功勞。每次韓商言沒錢的時候都是小白出錢。而且還在關鍵時候幫韓商言說話。這是目前為止我看過的電視劇中，出鏡率少可是給大家留的印象最好的男明星了。一個演員的演技好不好這時候就能體現出來。當然胡一天的外形也是自己的優勢。我很看好胡一天，期待他的新作帶給我們不一樣的感覺！
