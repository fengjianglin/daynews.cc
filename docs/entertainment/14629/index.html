吳昕：“我一哭就被黑”——女孩子愛哭，真不是矯情

2019-12-13 1,711條評論 2,961次閱讀 497人點贊

昨天，吳昕的新綜藝《你怎麼這麼好看》開播，吳昕聊到自己的獨居生活時忍不住又哭了，但這次她強忍住眼淚說「我不能哭，我一哭就被黑」。

確實，這已經不是我們第一次看到她在節目里哭了。

每一次都有網友指責：明星是很辛苦，但是收入也很高啊；為什麼沒見別人哭呢，就她矯情；為什麼總是這麼喪，不傳遞一些積極的生活觀念？

她多辛苦我就不說了，因為畢竟沒有誰的生活不辛苦。

不過我想說，每個人的性格都不一樣，每個人也都有情緒表達的時候，只不過剛好她的被大家看到了。

而且，吳昕也不是故意在賣慘，她最終還是想給大家傳遞一種積極的生活觀念。

她在後面有說：「但我覺得，其實我們有最堅強、最洒脫、最自由的靈魂，然後我們有這個世界上，最理解我們的爸爸媽媽。」

我倒覺得，她總是把自己的最真實的情感、最脆弱的一面在人前暴露出來，恰好說明她是一個簡簡單單、沒有防備的人。

其實這次黑她的人比較少了，有些小可愛表示特別能理解，說她們也屬於這種「易哭體質」。

動不動就哭確實很沒必要，但是有的人就是沒辦法，控制不了啊。

這一點我深有體會。

一直以來，我特別害怕跟別人「深入交談」。

就是有些事情不管多辛苦多難過，自己悶在心裡一遍遍回想，都不覺得有什麼。

最怕的是跟別人傾訴，即使是一點點小事都會開始流眼淚。

最最怕的是，當你故作堅強跟自己說沒事，這時候旁邊有個人開始安慰你，輕輕撫摸你，說沒事兒……

那完了，怕是要演變成號啕大哭。

還有一些時候，跟別人好好說著話，也不知道對方哪句話就觸動了自己敏感的神經，就開始鼻子酸酸眼淚打轉。

一方面很尷尬，一方面也在心裡責怪自己怎麼這麼沒用。

於是只能低頭或裝作不經意轉身，或借口走開。

最搞笑的是吵架的時候，根本吵不起來，欲語淚先流，腦子裡沒詞，氣勢上也已經輸了好嘛。

但是總能在事後想到一堆反駁的理由，嗯，這樣吵我肯定能贏。

仔細想想，其實我小時候就是這樣了。

大人一說我什麼，或者拿我在大家面前開開玩笑，我就會默默低頭強忍眼淚。

我自己都不知道為什麼。

後來我還去百度了一下，結果你猜我發現了啥？

一個好高大上的辭彙：淚失禁體質。

（以下內容來源：科普中國官方平台）

一、哪些人容易「淚失禁」？

從心理學角度來講，人的氣質分類中的抑鬱質更接近於上面所說的「淚失禁體質」。

這一類型的人思維敏捷、心思細膩、有豐富的想像力、情緒體驗深刻、缺乏安全感、比較悲觀、偏內向、不善交際。

說起抑鬱質的代表人物，非林黛玉莫屬。

哭是林黛玉的特性，有含淚低泣，也有淚如雨下，可以說是典型的「淚失禁體質」。

二、「淚失禁體質」真是心理疾病嗎？

判斷「淚失禁體質」是否為心理疾病，可以從以下三個方面來考慮：

第一，這個哭是否事出有因。

如果這個哭是由客觀事實引起的，且自己對這個情緒很清楚，哭的原因符合大多數人的邏輯，就不是心理疾病。

第二，哭的程度與持續時間。

如果哭的時間和程度與大多數人一樣，那就不是心理疾病。

第三，是否影響社會功能。

如果不影響個人正常的生活、工作，那麼也不是心理疾病。

一句話，如果哭得合情合理，不影響正常生活工作，就不是心理疾病。反之，則考慮到正規心理機構去做檢查。

三、「淚失禁體質」需要改變嗎？

哭是人類的一種生理情緒的表達。

心理學研究認為約一半的哭是由於悲傷，當然生氣、同情、恐懼甚至高興也都會讓人哭泣。

所以，從某些角度來說，「淚失禁體質」也是有一定好處的。

1、哭可以發泄情緒，對維持心理健康有一定的好處。

當人們悲傷、緊張、憤怒、恐懼時，人的心理和生理都處於高度緊張的狀態，而當這種緊張狀態得不到釋放時，就會在體內積聚起來，長此以往，會影響身心健康，容易引起疾病。

而哭泣，可以將這種能量釋放出來，解除了這種情緒壓力，對人的心理健康起到一定的保護作用。

2、眼淚有「排毒」的作用。

這裡所說的「排毒」，不是真正意義上的毒素，而是說人在應激（遭受壓力、委屈或危險事件時）的過程中體內會有應激反應，而眼淚可以將體內的應激激素排除體外。

所以，很多時候哭完之後，會感覺好多了。

3、哭可以傳遞信息。

人類情緒本身就具有信號作用，哭也不例外。剛出生的嬰兒就是通過哭來跟父母產生互動，從而保證基本的生存。

對於成年人而言，哭依然是一種互動交流方式，通過哭可以告訴對方你的底線、要求和目前的心理狀態。

但如果還是想改變這種「淚失禁體質」，其實也是有辦法的。

四、如何緩解「淚失禁」？

1、先不要嫌棄愛哭的這個特質。

把它作為自己的一部分先接納，與其和平相處。

2、了解自己每次哭的原因，尤其是深層次原因。

比如是因為恐懼、擔心還是憤怒才會「淚失禁」，究其原因，才能找到解決的辦法。

3、針對自己找到的原因來鍛煉自己的承受能力。

4、要學會把這些情緒說出來或是寫出來。

看到這裡，是不是覺得「淚失禁」還蠻可愛的？

其實，我倒是不覺得這是個需要改變的問題。

我相信很多小可愛跟我一樣，除了有些尷尬，吵架吵(zhen)不(diu)贏(ren)，對自己的生活也沒什麼影響和困擾。

我們依然很樂觀、很開心呀。

所以說愛哭的女孩子真不是矯情。如果你遇到一個「林妹妹體質」的女孩，請多給她一些包容和理解吧。
