《新喜劇之王》後再無作品？鄂靖雯：也有劇本找，但星爺覺得不行

2019-12-16 241條評論 1,671次閱讀 553人點贊

在影視圈有個很具備年代感傳奇感的詞語——星女郎。

上世紀九十年代，是周星馳無敵的年代，稍微誇張點可以說那是屬於周星馳的年代，從爆紅到票房統治者，他身上的傳奇相信大家都有所耳聞，特別是1992年，周星馳電影包攬年度票房前五，這樣的影壇傳奇可謂前無古人後無來者。

而周星馳的爆紅，也帶動了身邊演員的身價起伏，「星女郎」、「周星馳黃金配角」等都成為了屬於那個年代的特殊辭彙。

《喜劇之王》的張柏芝、《功夫》的黃聖依、《長江七號》的徐嬌、張雨綺，《美人魚》的林允等，無疑不是通過星爺的電影，被廣大觀眾所熟知，身價倍漲。

但也偶爾有例外，那就是演員鄂靖雯。

2019年春節，周星馳導演、鄂靖雯主演的《新喜劇之王》上映，雖然6.2億的票房在《流浪地球》、《瘋狂的外星人》、《飛馳人生》面前不算多，但僅憑「周星馳」的名號也為這部電影帶來了無法估量的熱度。

但令人沒想到的是，主演鄂靖雯卻在網路上遭到了一波又一波的質疑，從形象到氣質再到實力，不少觀眾不禁懷疑星爺這次的選角眼光。而當電影下映，熱度褪去後，主演鄂靖雯也似乎消失了。這樣的現象在周星馳電影中確實少見，作為「星女郎」，鄂靖雯沒有被大家歡呼「驚艷」，也沒有因參演周星馳電影而流量倍增、事業平步青雲，反倒是銷聲匿跡，持續不溫不火的狀態。

這種現象的出現，也讓不少網友無情調侃「星爺也碰到了捧不紅的人」，甚至將鄂靖雯稱為「最失敗星女郎」。

將近一年過去了，我們才偶然在綜藝節目《演員請就位》中再度看到了鄂靖雯。她在第一期挑戰了《無名之輩》中「馬嘉旗」一角，讓郭敬明大加讚賞，也讓郭敬明與趙薇上演了搶人大戰；遺憾的是，在節目第七期，在挑戰完《大話西遊》中「白晶晶」一角後，遭到了淘汰。

不過，對於這次的淘汰，鄂靖雯卻表示並沒有覺得感傷，因為節目錄製過程太緊張太高壓，淘汰反而讓她鬆了一口氣。在這之後，周星馳還安慰了她，並且指出她不太為自己爭取的缺點。就像她在採訪中回憶的那樣，當初《新喜劇之王》的項目找來，她的第一反應是「不去」：

「我是被逼著去的，幸運砸到了我的頭上。」不想去，是因為她自知機會渺茫，但畢竟星爺發話了，在身邊人的敦促下，她「被逼」參加了試鏡，沒想到就被選上了。

面對網路上種種質疑，鄂靖雯卻不太在意，演過《新喜劇之王》後，她基本就消失在大眾視野了，其實在這之後，她簽了周星馳的公司，但並沒有急著演影視作品，而是回到了話劇領域。

當然，很多人覺得演過《新喜劇之王》後，她的流量和名氣也沒有提升多少，但實際情況是，她也接到了不少影視劇本，但是一直沒有碰到合適的，反而接了自己比較喜歡的開心麻花話劇。

有意思的是，當有影視劇本找來，星爺覺得不行時也會建議她放棄。

「我們覺得有意向的劇本，經紀人都會給他（星爺）看一下。有的時候他會說，你覺得這個好嗎？ 星爺說，可以再放一放。我覺得老闆都不著急，我也就別著急了。從這一點來講，他很保護我，覺得很溫暖，他沒有說你是我的藝人，我簽了你，趕緊出去給我賺錢。他能允許我有沉澱的機會。」鄂靖雯在採訪中這樣說。

可能每位演員都有自己的選擇，很多演員在抱怨影視業寒冬，抱怨失業危機，但鄂靖雯卻拒絕了找來的很多劇本，反而回去演話劇去了，這樣的選擇是對是錯暫且不談，但這其中不驕不躁保持質量與初心的態度，確實值得不少演員深思。
