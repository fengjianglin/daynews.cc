演員鄧莎按摩店出意外！拔罐時背上著火，細節照曝光身體面目全非

2019-12-14 522條評論 3,603次閱讀 2118人點贊

戳右邊關注我們，每天都有最新娛樂八卦等你批閱！

12月14日上午，演員鄧莎在個人社交上更新動態，稱自己在拔火罐時被燒傷，並曬出了燒傷的圖片，該消息一出，引起了很多網友的議論。

​據鄧莎自己透露，她在12月9日結束活動行程回北京，到達北京已經是凌晨，身體比較難受就找了個spa店做按摩，後來順帶拔了個火罐，沒想到酒精起火背部嚴重燒傷。從鄧莎曬出的照片來看，燒傷部位有很多火泡，並且有些部位的傷口血肉模糊，觸目驚心。

​鄧莎還在文中告誡大家在遇到這種事該如何處理，把頭髮紮起來，要及時用濕毛巾撲滅火焰，先用燙傷膏處理之後再去醫院就醫，並之後要好好護理。

​據了解，鄧莎是內地小有名氣的青年演員，曾參演過《美人心計》、《孤芳不自賞》、《笑傲江湖》等多部電視劇，憑藉甜美長相和不錯的演技收穫了不少觀眾的喜歡。

​不少粉絲看到鄧莎的情況後都紛紛留言表示擔心，也有人在關心之餘表示，自己經常拔火罐，從來沒遇到過這種情況，鄧莎是不是遇到黑店了。

​還有人說，拔火罐是從古到今的手藝活，不是專業的就不要開店攬客，很容易出意外。

​此前女演員徐冬冬在某整形醫院做粉瘤切除手術，但因為手術處離動脈太近出了意外，之後被緊急送往北京某醫院急診出進行緊急搶救。

​本來粉瘤是一種皮脂腺囊腫炎症，切除手術十分簡單，一般情況下在醫院門診就能解決。但這次徐冬冬在整形醫院卻出現了危險，確實非常令人意外。

​鄧莎也表示，會不會留疤要看接下來的護理，希望鄧莎能早日康復。不管是拔火罐還是切除粉瘤，涉及到自己身體健康時，大家還是要選擇專業的地方進行處理，為自己的安全多多注意。
