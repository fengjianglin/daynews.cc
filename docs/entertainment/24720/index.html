真正的“80後”老戲骨，你知道她們的名字嗎？

2019-12-22 1,148條評論 3,223次閱讀 1595人點贊

如果一個演員演繹的同一個角色演多了，就會被冠上專業戶的稱號，老戲骨韓童生被稱為國民爸爸專業戶，閆妮和張嘉譯合作了很多部劇被稱為國民劇夫妻專業戶。婆婆專業戶該屬於誰？太后專業戶該屬於誰？

戲如人生，人生如戲！

★83歲高齡老戲骨鄭毓芝

她是在中國舞台上第一個扮演宋慶齡的人，從1954年到2019年從配音作品到話劇作品，再到個人著作，電視劇，電影。似乎和影視相關的都努力而認真去做過。

印象中關於她的第一部戲是《婆婆媳婦小姑》這部劇獲得了大眾電視劇「金鷹獎」最佳電視劇，這也應該算是婆媳劇的鼻祖了。鄭毓芝老師演繹了上海弄堂里萬家燈火中最平凡普通的一位婆婆，好與壞，對與錯雜糅在一起的家庭生活。

出演這部劇之後讓鄭老師感受到了電視的影響力，她第一次被觀眾在生活中認出來。

最近在大火的《慶余年》裡面飾演南慶太后，皇家利益大於一切，不怒自威的南慶太后，雖然出場次數不多但每次說話的分量都是在捍衛皇權不容置喙的！

《琅琊榜》里的太皇太后，那個死前最後一口氣嘴裡還念著「小殊」的太婆。林殊是太皇太后最疼愛的孩子，鄭毓芝老師把那種見到小殊既看錯又沒看錯的那種真真假假的心態演繹很能讓人猜測一翻。

《上錯花轎嫁對郎》里的老太君，《黛玉傳》裡面的賈母，《傾城之戀》的白老太……演了許許多多的角色，大家都說她是名副其實的「老太太專業戶」，在一次娛樂拜訪中，她曾說「我如果不給別人添麻煩，而劇組恰好缺少一個這樣的老太太，我就會去。」很樸實的話，卻發揮著力所能及的光和熱。

★「百變老太」李明啟

她和鄭毓芝老師是同齡人，李明啟老師的性格洒脫，說話做事乾脆利落。相對鄭毓芝老師的低調，李明啟老師在娛樂圈更為活躍些，可能是性格使然吧！向來都是人未到，聲先到！

參加綜藝，上春晚，在各大頒獎典禮上也會常常看到她的身影。這不最近剛剛參加了中國電視好演員盛典和育良書記一起走紅毯。

從《水滸傳》裡面那個頭帶紅花壞壞的王婆開始，似乎一些反面角色特別與她投緣，後來火遍大江南北的《還珠格格》裡面的容嬤嬤，更加奠定了在觀眾朋友們心中的惡人形象！

拍《水滸傳》時她是劇組裡面年齡大的演員，有一檔節目採訪她時，李明啟老師說有一場戲她被送刑場，當時她的腿燙傷了，但是仍然堅持把戲拍完，結果傷口裂開褲子全是血，但是她卻對觀眾朋友說「無論做什麼工作一定要吃得起苦，受得起罪，認真的去工作」！

連續接拍《神鵰俠侶》、《雙面膠》等多部反面角色的戲，她被冠上了「惡婆婆」的代名詞。

但也出演過偉大母親的角色，比如《家有九鳳》中初媽媽，一位用真心去對待每一個女兒的媽媽，無論她們貧窮還是富有，無論她們是聰明還是痴傻，無論她們幸福還是不幸，她都會對女兒們說要團結，因為她們是一家人！

李老師說，「表演藝術是體驗的藝術，沒有體驗，哪有體現？這是已故演員金山的話。有了體驗你才能塑造出人物的魂來。你在那個生活中待一天，與在那裡待十天，感覺肯定不一樣，要把人物形象掰開了，揉碎了，化在心裡，才能做到神似。」

在《中國文藝》節目中她曾寫到「演戲不是就是論事，而是體驗人生！」

★國民母親、苦命媽媽張少華

10多歲開始演戲，60歲才等到了大獎的肯定！

成功有時是一場漫長的等待！

第十一屆上海電視節 「白玉蘭獎」最佳女主角的獲得者。

獲得《超級訪問》8周年頒獎盛典「超級藝術成就獎」。

獲得第四屆中國演藝界十大孝子。

《大宅門》中飾李香秀的母親馬立秋、《勇敢的心》中的霍白氏、《鐵梨花》中的老太太、《武林外傳》中的斷指軒轅……

然而讓人印象深刻的還是那部《我的丑娘》，不知道哭過多少回？母愛的無私和偉大！

張少華老師總說「母親是世界上最偉大的人」

因為她的母親是她的忠實粉絲，是她在演戲事業中堅強的後盾，而《我的丑娘》似乎是特意為張少華老師而來，她需要這樣一部戲去感受母愛的偉大，去報答自己的母親！

《我的丑娘》也是張少華老師首次出演女主，因為這部劇流淚的不只是觀眾，做為主演的她也流出了一茶缸的眼淚！

我不盡心儘力，我不配演員二字——張少華

80多歲的老戲骨似乎骨子裡都有一股韌勁，都有一種叫忠誠的東西！她們都忠於自己的事業，都有一種敬業、認真、踏實不服輸的精神！

屬於她們的朝氣蓬勃一直在的心裡！
