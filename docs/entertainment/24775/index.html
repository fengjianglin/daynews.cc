票房狂轟4.11億隻用8天，投資人賺翻了，肖央終於為自己正名了

2019-12-21 477條評論 3,514次閱讀 2608人點贊

今天市場又新開畫了多部大片，原以為本輪大片會壟斷影市，但沒想到票房又啞火了。除了《葉問4》其他兩部大片的開畫都不佳，《只有芸知道》表現還算中規中矩，好萊塢巨制《星球大戰9》更是直接撲街了。柯汶利《誤殺》雖然已上映多日，但憑藉炸裂的口碑，以及爆棚的上座率守住自己的票房份額。

《誤殺》是今年好評度最高的國產片之一，很多電影的口碑都是高開低走，但它卻是逆襲跌一路走高，柯汶利不愧是獲得過奧斯卡短片提名的導演。本片故事講述李維傑與妻子阿玉打拚多年，膝下育有兩個女兒。一個雨夜，一場意外，打破了這個家庭的寧靜。而李維傑作為一個父親，為了保全自己的家人，他不顧一切地決定瞞天過海……

《誤殺》能夠成為賀歲檔票房大黑馬除了故事精彩外，影片的台前幕陣容強大也起到了保駕護航的作用。比如監製陳思誠，那可是影壇響噹噹的票房王，男主角肖央是後起之秀，陳冲是老牌影后，姜皓文，秦沛等都是老戲骨。或許有人會說它前面票房不俗是因為對手不強，那今天受到新一輪大片沖它依舊大豐收這下能證明自己實力了吧？

《誤殺》受到新片衝擊影響場次縮水了一半，但即便如此它依舊拿下亞軍位置足見其實力。截止21點40，實時票房收3568萬，上映8天累計4.11億元。放映4.9萬場次，排片量13.6%，場均22人次上座率爆棚，它周末逆襲贏回場次還有懸念嗎？按分賬比例算，目前片方分賬到手1.41億元，片商收回成本盈利了，該片最終預計能收7億以上，投資人要賺翻了。

男主角肖央終於為自己正名了，映前有多少人忽視他甚至不正面瞧一眼，但現在票房笑到最後的卻是他。肖央在影壇基礎比較薄弱，雖然他演戲很有天賦，但早年沒什麼好資源，直到遇上陳思誠才逐漸走上事業正軌。去年他主演的《唐人街2》票房大爆了33.97億元，打破喜劇片歷史紀錄，但他的貢獻一直被人忽略，他急需一部獨挑大樑作品來證明自己，現在他終於做到了。

《誤殺》在外人看來紙面實力一般，演員沒有票房巨星，導演還是個新人，但上映後才發現大家小看它了，截止目前很多部大片都被它擊敗了。導演柯汶利一鳴驚人了，肖央票房福星體質再一次得到印證。賀歲檔票房開打到現在大致上情況很明朗了，本周新片中只有《葉問4》是真正的對手，這段時間這兩部影片將聯手壟斷市場瓜分票房。

《誤殺》票房狂轟4.11億隻用8天，成為黑馬的背後是它各方面細節都做到了極致。片中的每個人物角色演員塑造得很到位，演員的演技精湛張弛有度，導演對節奏的拿捏恰當好處，過硬的劇本，走心的劇情，它票房大賣是必然的。肖央這次給觀眾的驚喜太震撼了，很多人都以為他只能演喜劇片，沒想到其他題材表現同樣這麼炸裂，他果然是巨星潛質藝人！

《誤殺》票房一路飄紅最開心的就片商了，投資人現在都賺翻了，這還只是票房上的利潤，算上後續的碟片視頻電視版權等收益片方賺到手軟了。柯汶利和肖央是最大贏家，一個在影壇打響招牌，一個證明了挑大樑能力。陳思誠只擔任監製也是在幫扶新人導演，這是一部大家都共贏的精品佳片！
