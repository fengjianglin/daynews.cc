小八卦：羅雲熙喜歡現代劇？王一博緊張時咬手指？周冬雨性格直？

2019-12-20 1,509條評論 2,484次閱讀 1379人點贊

1、相比於古裝劇，羅雲熙更喜歡現代劇，他是古裝劇走紅的，這個印象也太深了，他想改變一下。

2、王一博緊張的時候就咬手指，手指甲都咬沒了。

3、周冬雨說話比較直，這樣的性格上綜藝很容易翻車，被嘲，所以周很少參加綜藝，即使參加了，團隊也會叮囑她說話注意一些。

4、楊天真開始培養自己的新人，簽了很多藝人，也會通過旗下藝人招商引資開闢自製劇。

5、金晨是自己和正午鬧掰了才錯失角色，現在團隊開始賣慘聲稱自己再次被截胡。金晨不是第一次賣慘角色被截胡了，但第二部並沒有接觸過她，她這種營銷手段很容易得罪人，楊采鈺也不是好惹的，團隊已經採取措施反擊了。

6、郭敬明現在就是想靠著陰陽師去翻盤，上綜藝也是為了回一些熱度，這部劇他特別上心，每個演員他都會親自把關。

7、王寶強最近給他新女朋友拉了不少資源，帶著出席各種酒局，她女朋友打算進娛樂圈發展了，說是想自力更生，希望寶寶這次是真的遇到愛情了吧。

8、毛不易很喜歡吃肉，拒絕肉絲，無條件是大塊大塊的肉，他還喜歡吃豬腦。

9、牛駿峰演技確實不錯，但是他情商也確實挺感人的。因為在某綜藝比賽上獲得的榮譽讓他飄了不少，演戲真的用力過猛，而且還把自己接不到好劇本的鍋都扣給了團隊，現在團隊里也有人對他頗有怨言。

10、楊采鈺不止演唱過《少年英雄小哪吒》，她小的時候家裡就很注重培養她的才藝，從小參加過很多比賽，拿過不少獎。現在她背靠大樹好乘涼，《大江大河》中梁思申一角已經確定是她了，這個角色人設非常好，如果她演技不出大錯，未來播出後一定會很圈粉。

11、L湘不允許媒體說她的女兒胖，要用「可愛」這個詞，否則就要被威脅去法院。

12、范冰冰現在已經拍了電影了，算是復出了，只是還在慢慢試探，看看大眾對她的態度。

13、李蘭迪這次口碑又下滑了一波，本來年輕加上唐人最捧她，公司的有些藝人自然不舒服，這次也是想逮住機會黑一波。

14、程瀟最近沒什麼大的動靜，這段時間整體熱度不高，但她一直都沒閑著，這段時間都在劇組拍戲，尤其是接下來要和許凱合作的《你微笑時很美》，還沒拍就有不少粉絲很期待。她現在一心要往演員的道路上發展，畢竟女愛豆的青春飯吃不長久。

15、張雨綺這些年基本上沒攢到什麼錢，除了被坑了以外，她的大部分都花在內容保養上面，開銷特別特別大。
