《夢回》上線一天，已被認定為“爛片”，胡歌、娜扎都救不回來

2019-12-16 618條評論 1,848次閱讀 304人點贊

12月15日，小熒屏又迎來了一部古裝劇，也是根據小說改編的，名字叫做《夢回》。因為有著當紅小花李蘭迪參演，所以很多觀眾還是比較期待的。在上線一天後，《夢回》也確實引起了不小的熱度，同時也跟觀眾引起了不小的「不適」！

、《夢回》上線第一天，就已經快要被追劇的觀眾給吐槽爛了，更是已經被很多觀眾認定為「爛片」。同樣是根據IP改編的，作品，為什麼質量差距就這麼大呢？看看隔壁家的《慶余年》、《鶴唳華亭》，為何就能被觀眾如此的青睞！

根據已經觀看過《夢回》的觀眾評價，《夢回》主要存在兩個問題：劇情、演員。

既然是根據同名小說改編的，自然是要最大程度地按照原著中的內容來，而不是往裡面亂加東西。《夢回大清》這本小說在連載的時候，是非常受讀者們青睞的，但被影視化之後，竟如此被觀眾嫌棄，只有一個可能，就是編劇的問題了。

還有就是《夢回》主要講述的是穿越到清朝的故事，又是一個穿越梗，而且又是清朝，還是那個九子奪嫡的一代。關於康熙到雍正的故事，觀眾們都快要倒背如流了，而《夢回》偏偏要再試一次，好了，被觀眾們嫌棄就是結果。

當然，要是故事足夠精彩，這樣的嘗試也未嘗不可，《延禧攻略》就是靠著劇中的內容崛起的。但《夢回》製作方好像並沒有靠劇情讓劇爆紅的打算，畢竟之前的幾次穿越宮廷劇都挺成功的，這次應該也不例外。

很可惜，這次是完完全全的想錯了。以前的穿越劇是新奇，現在的穿越劇是老梗，甚至還有些審美疲勞。所以想要複製《宮鎖心玉》、《步步驚心》這樣的現象級劇集，就不要痴心妄想了。

說完劇情再說說演員，《夢回》這次的主演都是青年演員，名氣比較大的也就是女主李蘭迪了。李蘭迪的校園青春劇確實演得好，這個沒法反駁。而且《夢回》也是她的第一部古裝劇，所以《夢回》在開播的時候能夠如此被大家關注，這也是一個重要原因。

不過，清純可人的李蘭迪的古裝，好像並沒有讓大家眼前一亮的感覺，相反還有一點眼前一黑的危險。拍《夢回》的時候，李蘭迪可能沒有注意身材的保持，所以把有些圓潤的李蘭迪困在旗袍裡面，感覺頗有些怪異。

再加上其他一些主演的顏值也是忽高忽低，更是讓觀眾們沒有看下去的慾望。而因為主演們都是青年演員，所以現在《夢回》，很有是為了捧人才拍的嫌疑。

因為《夢回》是唐人製作的，作為唐人藝人的胡歌、古力娜扎自然也是要為這部劇宣傳一下。不過，胡歌與古力娜扎的粉絲們好像並不買賬。對於胡歌大家依舊是把注意力放在了《南方車站的聚會》，而娜扎的微博評論被限制了，所以看不到粉絲們是非常態度。

不過按照已經看過《夢回》的觀眾評論來看，《夢回》這次的「爛片」標誌是跑不掉了。胡歌、娜扎也救不回來！
