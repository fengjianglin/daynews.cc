從爆款劇到爆款綜藝，這家老牌影視公司選擇這樣做

2019-12-24 622條評論 1,941次閱讀 478人點贊

都說「影視寒冬」，許多大牌製作公司都幹什麼去了？

翻看新聞，減產甚至停產「禦寒」似乎是常態，即便開工也鮮有大製作。

不過，也有一些老牌公司找到了自己的新方向——

曾經打造了《花千骨》《老九門》《楚喬傳》《涼生，我們可不可以不憂傷》的慈文傳媒，如今不僅僅是「爆款劇締造者」，更成為爆款綜藝的幕後推手。其中，慈文傳媒與東方衛視聯合出品熱播綜藝《我們的歌》一直居周末綜藝收視率榜首。最近一段時間，慈文也連續宣布新計劃，從綜藝到科創到新劇儲備，「禦寒」也有新方法！

大戲不斷，融合有成效

作為老牌影視公司，慈文傳媒今年的動作不小：2月，其引入新的控股股東——江西省出版集團全資子公司華章天地傳媒投資控股集團有限公司，成為首個依託國資賦能的影視公司。

而從大半年的發展看，這一融合初見成效，爆款頻出：

由曹衛宇、張晞臨主演的《大地震》，根據唐山大地震中開灤煤礦萬餘名礦工礦井自救真實事件改編，按照院線電影的標準拍攝，為網路電影樹立新標杆；

由張譯、黃志忠主演的反特刑偵劇《光榮時代》播出後獲得收視與口碑雙豐收，在獻禮劇中獨樹一幟；

而正在熱播的《中國夢之聲·我們的歌》，憑藉最高2.4%的收視率，持續穩居榜首的播出成績單，熱門話題熱搜榜持續霸屏的現狀。

從某種角度上說，國資控股後的慈文傳媒步伐穩定，持續保持對優質內容的把握力，在影視寒冬之下堅持投入挖掘優秀項目。優質內容的價值提升在於其延伸，是能否成為IP，進而成為IP集群。慈文傳媒董事長吳衛東透露，如何從內容出發進行聯動，是實現內容企業價值提升的關鍵。在《我們的歌》取得「內容+平台」的成功探索後，未來，內容與平台的融合將成為新的運營模式，亦是市場新的盈利爆發點。

科創助力，優質IP再發力

慈文傳媒引入控股股東江西出版集團旗下華章投資集團以來，市場一直在關註：於行業洗牌期主動調整的慈文傳媒將會有什麼樣的動作。

如今看來，老牌影視公司無疑正在探尋以新的盈利模式，新的業態組成，新的產業集群打破行業瓶頸。據透露，慈文傳媒將與清控科創智運成立合資公司，以共建文化科技創新產業園區的方式，探索5G、人工智慧、區塊鏈、大數據等尖端技術在文化產業的應用場景和商業模式，同步開發資源導入和生態搭建的創新模型，締造涵蓋影視、動漫、遊戲、科技、文旅、教育等領域的泛娛樂產業高地。

「科技創新幫助文化產業不斷地突破傳統邊界，文化創新反過來又能激發科技的進一步演進，在這種交互過程之中，誕生了不少新產品、新業態，其中相當多的新業態已經實現了國際領先地位。」吳衛東表示，在這樣的宏觀背景下，慈文傳媒積極擁抱科技浪潮，不僅僅出於延續精品意識和創新底色的考量，也是身為文化企業，需要擔當起提升思想內涵、展現國家軟實力的使命。

目前，背靠江西出版集團的華章投資為慈文傳媒提供了不少內容上的共享和協同，輸送的優質IP更支持其開拓創作。據悉，以保護長江為題材的電視劇《一江水》，陳建斌、董勇主演的刑偵劇《三叉戟》正如期拍攝之中，《紫川》、《彈痕》、《空港》、《婚姻的兩種猜想》、《密使一號》、《捕蟬》、《天涯客》等優質大戲已處於籌備階段，未來，慈文還將試水古裝互動劇製作及古裝劇線下互動，並與平台打造古裝短劇等。
