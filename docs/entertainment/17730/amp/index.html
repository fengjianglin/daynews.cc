探班《再見啦！母親大人》，董潔無懼“變老”演母親

天天要聞
2019-12-16

南都訊 記者蔡麗怡 實習生肖綺文 發自寧波由優酷、魔瞳影業聯合出品，現實主義題材、溫暖治癒的超級劇集《再見啦！母親大人》正在寧波象山熱拍中。該劇改編自不良生以個人真實故事寫成的散文集《雲上：與母親的99件小事》，豆瓣評分8.4分，曾令萬千讀者動容落淚，書里描寫了一個平凡而不幸失婚的50後母親，以最純粹的母愛，奮不顧身地愛孩子，直到奉獻了所有，把孩子撫養成人，而自己卻查出患癌，母子二人相伴度過艱難但溫暖的抗癌生活。電視劇版由今年剛奉上全民爆款劇《長安十二時辰》的著名導演曹盾與荊沖聯合執導，鞏雪編劇，以突破傳統的敘事結構，雙線並行勾勒母子二人分合交織的半生軌跡，完成對中國式母親的深情獻禮。

一向以清冷高貴形象出現在屏幕上的董潔，此次將塑造母親丁碧雲一角，角色年齡跨度長達三十年，董潔挑戰自我詮釋一位女性從少女、到中年、到老年的歲月轉變；以《藍色骨頭》《少年的你》等電影受到矚目的文藝小生尹昉，出演少年與成年版丁小軍一角，他與年齡僅相差6歲的董潔不演CP反而演母子，如何讓人信服地演繹出這對母子間溫暖緊密、生死相伴、長久不變的美好感情，勢必成為播出後的熱點，令觀眾充滿好奇與期待。

12月14日，南都記者來到寧波東錢湖上一條名為建設村的小村落中，探班正在此實景拍攝的《再見啦！母親大人》劇組，主演董潔、尹昉、蔡鷺、陳泇文及原著作者不良生、編劇鞏雪齊聚在小村的一座老房子中，接受媒體專訪，分享拍攝的感受和趣事。

董潔演媽媽？她說：「相信是很重要一件事情！」

探班當日恰逢拍攝年夜飯場景，陳設和老道具的細緻還原，讓大家恍惚穿越回十幾年前的小縣城，農曆年紅紅火火、熱熱鬧鬧的氣氛中。走戲時，一眾演員圍成一桌一邊互懟吐槽，一邊討論劇本，丁碧雲「一家人」輕鬆歡樂、笑聲不斷的拍攝氛圍，給媒體記者留下了深刻的印象。雖然劇組一直拍攝至第二天凌晨，氣溫驟降，但無阻演員們一入戲便投入故事中去，細膩深情的表演，令人愈發期待他們將來在劇中的呈現。

董潔飾演「母親大人」丁碧雲，當天上的是40歲~50歲之間的中年母親妝容，南都記者在她的休息車上親見她的「老妝」，略顯灰黯的臉色，蓬鬆捲曲的短髮，瘦削的臉頰,容顏樸素的「母親大人」雖顯歲月痕迹，卻因為董潔一貫的優雅而顯得氣質大方得體。來到群訪間面對大家時，董潔又化上了青年丁碧雲的妝容，問她會被剛才的「老妝」嚇到自己嗎？董潔大笑說：「那還不是我最老的妝容，只是40、50歲時候的，後面還會化到60歲的，這是我第一次塑造這麼大歲數的角色！」

怕嗎？怕演完這個戲後，觀眾覺得：董潔都可以演媽啦！？董潔卻說：「我很開心的是，這部戲的導演和攝影太好了，把我們拍得很漂亮！因為我的角色有一個從20多歲演到60歲的跨度，導演的鏡頭紀錄了母親所有的美好，它不只是單方面告訴大家母親的衰老，也在告訴大家她與兒子相處的美好時光。」

成熟演員總會面臨大跨度角色帶來的挑戰，無論演20歲強凹少女感，還是演60歲強凹老婦妝，都勢必會引起爭議，董潔對此怎麼看？董潔說：「演員要怎麼讓自己相信角色？這是一件很重要的事情。所以我們在定妝的時候，都是反反覆復地嘗試很久，讓每一個階段的妝都更自然真實。我第一天開工，就很緊張地追上去看鏡頭、看劇組人員給我拍的照片，一看，我的自我認同感就出來了：那就是四五十歲的丁碧雲的樣子，這戲的造型真的讓我很驚訝。如果我不相信這個，我怎麼能和尹昉搭戲演母子呢？

尹昉叫媽自然嗎？他說：「很自然，我戲外都叫她母親大人！」

從男女主角的資料看，董潔僅僅比尹昉大6歲，兩人在劇中扮演的並不是CP，而是母子關係。真實年齡差讓兩人在戲外就如姐弟倆，怎麼默契演好母子倆？

尹昉被問到「劇里叫媽，自然嗎？」時，董潔搶答：「我從不去想他真實年齡有多少！」 尹昉則毫不猶豫道：「在戲裡叫媽很自然，平時我也叫她（董潔）母親大人。」董潔透露，自己平時給尹昉發信息，都叫他「大兒子」，怎麼評價這位「兒子」？她說：「他特別認真，每場戲都要說通自己才去演，是一個每場戲都不放過的演員。因為每場戲都有充足依據，所以他演起來也都是用真情實感去完成的。」

人生冥冥中有註定，尹昉自己也經歷過如丁小軍一般在外飄泊闖蕩的日子，他的母親也因為患癌去世，這段傷痛，劇組和媒體記者都小心翼翼地不多提，尹昉自己卻坦然地說了：「因為我的母親也是患癌去世的，但具體的事與劇中故事並不相似。拍這部戲也是我對母親的思念和寄託。我是從丁小軍的高中演到30多歲，通過我與母親關係的變化，從小時候我對她的依賴，長大後我的叛逆，到後來得知母親患癌時的愧疚和害怕，失去後的思念，是一個成長的過程。」

丁小軍簡簡單單的一句台詞：「媽媽，我愛你！」，或者母子間的一段對話：「今晚的月色真美」，董潔和尹昉都覺得，越是簡單、自然、真實的戲，就越是拍出了世間母親與孩子最大的浪漫。

董潔說：「我們倆很相信這個關係，我相信他就是長大的丁小軍，我就是他的母親，這不需要刻意去創作、去設計，一切都是自然的、由衷發生的。」她甚至透露，「看到尹昉會讓我想像，如果有一天頂頂長得像尹昉這麼高的時候，是什麼感覺？哎，如果有一天要仰著頭看頂頂的時候，我想一定會是很欣慰、很幸福的感覺。」但如果頂頂也會叛逆，也會讓媽媽傷心呢？董潔說：「這是很自然的過程，每個母親都會遇到，我們要在這個成長的過程里，學會接受和包容。」

聚焦現實題材 溫暖療愈告別之殤

近年來，優秀的現實題材劇集不斷湧現，以不俗的口碑表現迅速佔領市場。《再見啦！母親大人》是一部深刻聚焦母子關係的作品。

原著作者不良生談到自己寫作《雲上：與母親的99件小事》這部書，是從自己個人情感的角度出發，對自己的母親作一次懷念，「很多人在面對親愛之人的離去時，都會有一種懊惱和後悔，如何面對和處理這種情緒，我在寫作中得到了自我療愈和救贖。如果說與親人的告別是人生的必修課，看這本書、看這部劇，是一種練習告別，只有練習告別，我們才能更好的珍惜當下。」談到當下網路部分群體認為「父母皆禍害」，不良生不認同這種說法，他希望通過《再見啦！母親大人》能提醒大家珍惜與家人相處的美好時光。

《再見啦！母親大人》歷時三年打磨，完成了從原著散文體到劇本「故事體」的改編，編劇鞏雪談到創作時最大的難點，「原著只有兩個主要人物，我要創作出更多的人物，如何讓他們具有真實感，不懸浮，是最大的挑戰」。

總製片人張阿瞳也談到，該劇直面中國家庭教育中常常規避的「告別教育」，試圖以遞進的方式，通過主角的經歷呈現出人生不同階段所面對的不同告別，用溫暖去撫平傷痛，剋制地闡釋如何與「成長之痛」和諧共處，集笑點、淚點、燃點於一身，希望將現實生活的五味雜陳盡數呈現，給觀眾帶來更深刻的情感共鳴。

據悉，《再見啦！母親大人》將於2020年1月殺青，有望明年登陸優酷視頻。製作方以真誠的態度打造落地的、有真情實感、有溫度的作品，希望打造成為一部「中國版的《東京塔》」。
