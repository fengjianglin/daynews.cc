利刃真的出鞘么？只能說期待太多，失望也太多

收 藏

令人非常期待的《利刃出鞘》終於來了，吸引人的是其爛番茄新鮮度高達97%，MTC評分高達84，而上映時豆瓣評分高達8.6（如今降至8.3）。甚至魯豫也已經提前二刷，認真看清楚裡面更多的細節來與大家分享和描述亮點所在。

另外在演員方面，都是好萊塢頂級陣容，特別是多位奧斯卡老戲骨的出演，更是讓此片增色不少。而更為閃耀的還是這兩位”大人物”：飾演”007″的丹尼爾·克雷格，飾演”美隊”的克里斯·埃文斯。

可以說，《利刃出鞘》是帶上超高評價、超強陣容、超帥男主等來和我們”見面”的。綜合來看，就讓我們都對這部年度懸疑大片充滿了期待，而在觀看電影時，我們也能感受到那超贊的音樂氛圍，還有極具藝術感的鏡頭和細節。

只因整個故事就是發生在古堡式的莊園豪宅里，無論室內室外都充滿著高雅感，而配樂基本都是以古典交響樂為主，再加上恰到好處的音效而帶起的觀影氣氛，時而輕鬆歡笑、時而緊張刺激，都是由精妙的配樂而帶動的。

但是，上映兩天後，很多人在朋友圈上吐槽對《利刃出鞘》有點失望，為什麼呢？

那就是劇情上的問題，其實《利刃出鞘》整個故事是邏輯嚴謹的，導演從頭到尾用心的設置而來很多前後呼應，而且案件的懸念是逐步揭開，當你以為答案就是這樣了，卻來了一個大反轉。

然而劇情很明顯，在懸疑片中並沒有”太燒腦”，對於我們中國觀眾來說，就是按著導演的思路去看，基本就能在輕鬆歡樂的劇情中得到最終的答案。

“很不幸”，讓我們感覺”不燒腦”的原因就是其針對美國的政治、社會問題等來展現劇情，所以對於這些方面，我們是看不懂，也無需去看懂。因此，我們也無法在劇情中有身同感受，就是看了這一班奧斯卡大咖們精湛的演技、表情、動作等等。

尤其是故事的開端，角色們的審問式對話可是充滿著各種諷刺、各種對美國現今社會問題的批判等等，也是對如今美國人的一種告誡。但我們確實是看不懂的，因此也有觀眾開始感覺非常乏味，甚至會”睡著了”。

或許這就是一個被扣分的位置吧，但另外還有一個位置，我覺得也會比不少觀眾所扣分的。

那就是”美隊”克里斯·埃文斯所演繹的角色，為了盡量不過於劇透，我就用花花公子來形容其在片里的角色吧。但美隊在大家心目中肯定已經是定型了，就算是復聯里不再出現，但他在我們心目中的形象是無法改變的。如今導演卻讓他扮演一位令人討厭的花花公子，實在是讓我們有點”難以接受”。

而更令人”難以接受”的還有”美隊”在此片里的”垃圾對白”，不但讓我們聽起來很生厭，而且能感覺到克里斯·埃文斯說這些對白時”很不自然”。

所以，個人感覺這裡就是《利刃出鞘》的最大敗筆。

因此，本來這部在我心目中的滿分好片，如今卻只能被無情的扣去一星了！

其實，《利刃出鞘》是本年度值得一看的好片，只要大家熬過了前期的沉悶，後期可是越來越精彩的。前期要認住一個杯子：MY HOUSE，MY RULES，MY COFFEE，最後也會前後呼應的再出現。相信這個”悲劇”，也是大家看片後所追求的。

對於看過此片的你，是否已經入手了這個杯子呢？
