24年前的《蓮花爭霸》：主角境況迥異，有人出家有人退圈有人離婚

2019-12-20 40條評論 1,438次閱讀 230人點贊

很多時候我們在談論電視劇都只會想到國產劇、韓劇、泰劇、歐美劇等等，但其實有很多新加坡電視劇也曾經陪伴著一大批70、80、90後成長，像是《蓮花爭霸》、《真命小和尚》、《蓮花童子哪吒》、《東遊記》、《神鵰俠侶》等等，通過這些劇很多劇迷認識了像曹駿、范文芳、李銘順、李南星、陳文華等一批優秀演員，這些劇也陪伴著很多人成長。今天要說這部劇可以說是經典中的經典，那就是在94年首播的《蓮花爭霸》。這部劇改編自古龍的《流星蝴蝶劍》，講述江湖中最優秀的殺手沈沖在一次刺殺行動中邂逅了奇女子南宮蝶，兩人情深意重，生死纏綿的故事，這部劇捧紅了很多演員，24年過去了，我們一起來回顧一下吧。

劇中飾演男主角沈沖的李南星相信大家都不陌生，他被稱為是新加坡周潤發，他在新加坡享有很高的知名度，從《蓮花爭霸》、《傷城記》到《雙天至尊》、《豹子膽》再到《迷雲二十天》，李南星詮釋著多面的不同角色，深受廣大觀眾的喜愛和好評。中國觀眾對他最早的關注還是在《蓮花爭霸》這部劇中，沈沖成就了他。之後他演出了超過50部作品，直到2015年他仍舊極具號召力，不過16年之後他就人氣滑落。李南星的事業上還算一直順風順水，但婚姻愛情卻沒有那麼圓滿。他和同為演員的楊莉冰，經過八年的愛情長跑後結婚，但最後離婚收場。兩人結婚多年沒有小孩，如今的李南星依舊帥氣，只是有些孤獨。

劇中的女主角則是由朱樂玲飾演的南宮蝶，大家對她當時應該是選沈沖還是白玉川有過糾結，但最後還是祝福她能夠和沈沖有一個美好的結局。朱樂玲在娛樂圈的時間並不算長，未踏入演藝圈之前朱樂玲還從事過售貨員、模特兒、補習教師等職業。在進入娛樂圈的8年時間裡，朱樂玲的作品也不多，她一度進軍商界大獲成功，之後因為結婚生子而隱退，中間有很多人邀請她復出拍戲，但她都拒絕了。結婚之後她就全身心投入到了家庭裡面，相夫教子也是相當幸福的。

劇中還有一個女主角那就是劉秋蓮飾演的秋若蘭和風二娘。劉秋蓮在娛樂圈的時間也不長，當年憑藉《蓮花爭霸》讓她人氣大漲，她被稱為是新加坡的林青霞。她不僅在演戲上面很強，唱歌也是不遑多讓。有人曾用詩一般的語言描繪劉秋蓮：楚楚動人，柔美的聲線頗為攝人。不過遺憾的是，《蓮花爭霸》一劇成名之後，劉秋蓮到了馬來西亞報讀工商管理課程，之後投身商界做起了生意。之後劉秋蓮有重回過娛樂圈一段時間，不過人氣卻遠不如當年。如今的劉秋蓮，應該還在忙她的生意，同時照顧她的一雙子女。

劇中有一個反串的角色讓大家印象深刻，那就是飾演白玉川和段素素的塔琳托婭。塔林托婭把男人的風流倜儻和變成段素素後作為女人的風情萬種詮釋得淋漓盡致，塔琳托婭接拍《蓮花爭霸》時，年僅23歲。年紀雖輕，卻能把老成持重、陰險腹黑、城府極深、文武雙絕的風流公子白玉川演繹得出神入化，讓萬千女性如痴如醉。她塑造的反串形象經典程度堪比林青霞的東方不敗。即使過了這麼多年，無數觀眾依舊對《蓮花爭霸》里那個風流倜儻的白玉川念念不忘。多年後塔琳托婭和多數主演一樣也隱退了，不過卻嫁入豪門過得十分幸福。據悉，塔琳托婭的丈夫是新加坡富商，過得也是相當地好了。

最後再來說一下，葉群的扮演者謝韶光，大家對他不會陌生，大家最熟悉的一個角色莫過於《東遊記》里的穿山甲 了。謝韶光當年憑藉《蓮花爭霸》被觀眾熟知，短短一年後他就搭檔范文芳主演了另一部大火的劇《緣盡今生》，之後演藝事業扶搖直上。從《出路》里飾演遊手好閒到奮發圖強的大少爺「張家富」；到《荷蘭村》里飾演「娘娘腔」的少爺「楊雄」；再到《喜臨門》里他又變成了不務正業的「羅家龍」。憑藉在這三部劇的出色表演，謝韶光的演技得到了認可。他也獲得了很多獎項的肯定，不過他隨後退出娛樂圈並且在2013年正式剃度出家。3年後他還俗，並致力於培養年輕一代的演藝能力。
