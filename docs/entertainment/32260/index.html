吳昕給何炅抹「髮際線粉」，當看到最後效果那一刻，笑出豬叫聲

大家都看過最新一期的《快樂大本營》嗎？節目組安排了這期主題是「精緻」，因此嘉賓們都需要曬出自己的精緻好物，想知道他們會有什麼樣子的表現呢？一起看！

各位嘉賓為了顯示自己的精緻當然是大展神通，陳偉霆的吸油紙，闞清子的防皺紋枕頭，這些東西真是讓人大開眼界呢！但是吳昕作為養生達人卻直接曬出了解決當代人最為困擾的難題的物品，想知道是什麼呢？

隨著社會壓力的增大，很多人都會有了脫髮的困擾，因此髮際線上移也讓他們擔憂不已，而吳昕就曬出了一款「髮際線粉」，後來何炅還主動讓吳昕給字嘗試，誰注意到最後效果是怎樣的呢？

原本意外在吳昕的塗抹下何炅的髮際線會呈現讓人驚艷的效果，但是令人意外的是卻「翻車」了，看這髮際線粉竟然直接塗到了何炅的太陽穴位置，鏡頭拉近還給人一種髒亂的感覺，這簡直就沒眼看啊！網友們看到這樣的何炅也是笑出了豬叫聲呢！
