《旺達幻視》首波片場照放出，天劍局登場

2019-12-17 1,657條評論 4,847次閱讀 854人點贊

今天，正在拍攝中的《旺達幻視》首波片場照被公布。證實了此前從未露面的天劍局將在MCU中出現，並揭示了泰柔娜·派麗絲飾演的成年版莫妮卡·蘭博。

先來看一波旺達的片場照，看起來應該是在休息期間所拍攝到的。奧妹兒劇中造型和之前反差不算大，和誰在聊電話呢那麼開心。無論平時街拍還是片場照，小奧總是水瓶不離手哦。

接下來是由泰柔娜·派麗絲飾演的成年版莫妮卡·蘭博。

該角色並不是第一次出現，青少年版的她曾出現在《驚奇隊長》中，她就是卡羅爾好姬友瑪利亞·蘭博的女兒。

《驚奇隊長》的時間線在1995年，終局之戰發生在2023年，而《旺達幻視》正常時間線是在終局之戰後。所以這時候的莫妮卡·蘭博，至少應該有35歲左右了。

漫畫中的莫妮卡·蘭博也是一位超級英雄。在一次意外中，她被能量裝置擊中後，從而獲得了超能力。

不知道MCU版會改編成怎樣，但從片場照來看，這時候的莫妮卡應該是天劍局的一名探員。

在今天片場照中，信息量最大的就要算確認天劍局的出現了。

漫畫中的天劍局，和神盾局一樣，有著超級長的一個名字，叫做知性世界觀測與應對局。該局性質和神盾局類似，只不過神盾局主要負責處理地球事務，而天劍局負責處理外星事務糾紛。

天劍局成員由地球人和外星人組員，由局長阿比蓋爾·布蘭德領導。

該局的成員包括：Agent Deems、Agent Paulletz、野獸和Sydren等。

在之前的MCU電影中，還沒有明確展示天劍局的存在。不過從《蜘蛛俠：英雄遠征》的彩蛋來看，尼克·弗瑞和一大群斯庫魯人的場景，可能就是在為天劍局的出場作準備，而且驚奇隊長本來也很天劍局有著聯繫。

之前更有傳聞，在《神盾局特工》結束後，漫威會製作天劍局的電視劇。
