《我們的歌》讓“寶麗金雙子星”闊別17年再同台！

天天要聞
2019-12-16

有網友說，《我們的歌》真的是一檔非常有特點的節目，不管是它獨一無二的「代際音樂融合」概念，還是盲選配對的創新賽制。但更讓人點贊的就是這檔節目真的太 「寵粉」了。

從歌手們來說，它讓許魏洲成為了費玉清的關門弟子，讓李紫婷能夠和前輩蔡健雅同台唱歌。從觀眾和歌迷來說，《我們的歌》滿足了粉絲們多聽幾首小哥經典歌曲的心，這次團戰，它又讓李克勤、黃凱芹這對「寶麗金雙子星」時隔17年後再同框，簡直是「活久見」！

兩個人除了分別在台上帶來了《卡門》、《另一種鄉愁》、《愛是永恆》、《DISCO MEDLEY》等十分驚艷的金曲改編，讓歌迷們重溫了兩位香港歌壇唱將的實力。終於在團戰台上重聚的李克勤和黃凱芹也非常感動能在節目中有這次同台的機會。

李克勤還向黃凱芹發出了演唱會邀請。

不止如此，在大家的共同期待下，李克勤和黃凱芹在時隔17年後，沒有任何綵排的演唱了曾經橫掃香港樂壇的經典歌曲《傷感的戀人》。一時間節目現場宛如回到了香港金曲年代，充滿回憶感的歌聲引發全場熱烈掌聲，兩人對唱的畫面不要太感人。

沒想到節目播出之後還有意外驚喜，在李克勤的演唱會上，黃凱芹真的到場，共同為歌迷們帶來了經典歌曲重唱。

許多粉絲都十分感激節目組能夠讓兩人重聚，促成他們同框合唱。

不得不說，節目組為了能讓觀眾聽到更多金曲改編，滿足大眾對代際音樂的期待可以說是十分努力了。原本節目對金曲的創意改編就收穫好評無數，但是節目組似乎覺得還不夠，第七期就再邀請了一支陣容十分強大的樂隊加入到節目音樂製作團隊里。

樂隊總監章謀聖為張信哲、王菲、蔡依林等一線歌手製作過專輯、演唱會；鍵盤手屠穎是華語樂壇編曲大師、華人演唱會頂級音樂總監；吉他手Mike Mclaughlin也是張學友演唱會的專屬吉他手；鼓手姜永正參與過張雨生所有專輯的錄製；貝斯手邱培榮也為林憶蓮、費玉清、張惠妹等歌手的演唱會提供音樂助力。

《我們的歌》用電視綜藝史上前所未有的強大音樂陣容來打造每一首金曲，可見其對音樂的極致追求。這樣的好節目值得更多的掌聲。讓我們期待下一期節目能帶來更多的精彩表演吧！
