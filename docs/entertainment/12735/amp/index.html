楊紫在節目中表示，如果37歲的他追求自己，她會毫不猶豫地同意

天天要聞
2019-12-11

今年很火的戀愛甜劇《親愛的熱愛的》，「看了看了，太好看了。」大家都沉浸在兩人甜甜的愛情當中，楊紫把佟年的人物形象，塑造得入木三分。尤其尤其是分手的哭戲，哭得太逼真了，楊紫的演技杠杠的。

楊紫的演技毋容置疑，這部劇不僅僅是她火的一塌糊塗，她還把李現給帶火了，李現從默默無聞的三線演員變成了如今席捲各大榜單的「現男友」。去年的鄧倫，今年的李現，「旺夫」不是隨便說說的。楊紫真是旺夫女神啊。

雖然楊紫不是擁有頂級顏值和身材的小花，但靠越來越穩的演技， 獲得很多觀眾認可，就像彭于晏在快本里他表示楊紫在以後拍戲會越來越多，人氣也會很高，雖然當時只是客套話而已，但楊紫做到了。

沒想到如今的楊紫全都實現了，偶像彭于晏的寄語簡直像開過光啊，楊紫可是彭于晏的鐵粉啊，男神的話她肯定是記在心裡的，當初兩人第一次見面是在快本，楊紫一看到彭于晏就像花痴一樣，楊紫看到彭于晏心跳加速。

之前楊紫還在《親愛的客棧》里楊紫也是談到了自己的男神，說自己特別喜歡像彭于晏那樣子的肌肉男，感覺線條十分完美，還說彭于晏的臉是那種娃娃臉，看上去真的想去親一口。哈哈，楊紫是彭于晏的迷妹絕對沒錯了。

楊紫曾多次表示彭于晏就是自己最理想的類型。 借用楊紫的那句話說：彭于晏就是我的菜！楊紫在節目中承認，如果37歲的他追求自己，她會毫不猶豫地同意。如今兩人都是單身，要是兩人在一起也應該是甜甜的愛情。

其實，無論娛樂圈的明星還是生活中的大眾，內心有個人喜歡還真是件幸福的事情！
