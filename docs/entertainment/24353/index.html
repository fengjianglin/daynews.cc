她與第一任丈夫閃婚閃離，與第二任丈夫離婚開撕，40歲開啟新人生

2019-12-18 372條評論 2,850次閱讀 935人點贊

關注小魚，帶你吃瓜帶你嗨

近日，黃奕在社交平台上轉發了自己在新節目中訪談的視頻，並配文稱：希望可以讓自己的表演讓孩子了解到更多的科幻知識。

粉絲們紛紛點贊，並為她鼓勵加油。

看來黃奕還是非常受大家的歡迎，而這個曾經的女神，卻一直伴隨著爭議。

黃奕出生在上海，小的時候父母就已經離婚了。

從小的時候就和爺爺奶奶住在一起。

因為家庭的特殊性，她從小就非常的懂事，上中學開始，黃奕就會利用寒暑假去打工。

之後也一直勤工儉學，曾經在專賣店賣過衣服，在餐廳做過服務員。

在一次偶然的機會之下，黃奕被找到，請她參與拍攝一個廣告，一舉成名。

也開始了自己的演藝生涯。

2000年，23歲的黃奕接到一個角色，那就是《上錯花轎嫁對郎》的「李玉湖」。

也是這部電視劇，她紅遍了大江南北，成了她的成名之作。

三年之後，她又與古巨基、周傑、馬伊琍、黃曉明等演員合作主演電視劇《還珠格格第三部天上人間》。

這個時候的黃奕已經有著讓人羨慕的成就。

而說到《上錯花轎嫁對郎》，想到男主角：聶遠。

沒錯，兩人其實不僅僅在劇中是PC，在生活中，還是一對。

在之前的一個節目中，聶遠更是當作老婆的面大方說出來。

在老婆面前提前女友？

看來在聶遠心裡，還真是真心愛過黃奕，不然不會說出這樣低情商的話。

可讓人遺憾的是，最後兩人卻沒能走到最後。

不知道在黃奕的心裡，是不是會為這段純真的感情感到遺憾。

可最後兩人還是走向了自己的人生軌道。

2010年5月，黃奕和姜凱在芝加哥的結婚照被曝光。

原來兩人認識41天之後於2009年閃婚了。

可誰曾想，就在結婚的消息曝光不過一個月，黃奕就宣布和丈夫離婚並且分居。

離婚之後的黃奕一度陷入低谷，只有不斷的工作才能讓她的心情稍微恢復。

直到兩年後，她遇到了第二個男人：黃毅清。

也就是這個男人，讓她痛苦了幾年。

兩人公開承認戀情之後，很快就結婚。

2013年，黃奕在美國生下一個可愛的女兒。

當時的黃奕36歲，她沒有想到，自己的婚姻很快就走到了盡頭。

在一片風平浪靜之下，暗潮洶湧。

2014年，黃奕老公黃毅清在網上發布了一篇長文，稱已經和黃奕分居，並且指出黃奕出軌。

稱手上有證據，還說黃奕是虛偽的女人。

文章一發出，網上一片嘩然。

沒過幾天，黃奕這邊又發出律師函，稱已經和丈夫提出離婚，並且指出丈夫造謠以及遭其家暴的事實。

後來兩人協議離婚，女兒撫養權歸黃奕，這件事情也就告一段落。

2017年，隨著案件的審判，黃奕勝訴，黃毅清公開致歉並賠償精神損失費。

這段長達6年的感情糾葛隨著案件落下帷幕，也走到了終點。

和丈夫離婚之後的黃奕選擇一個人撫養女兒，並且重新復出拍戲。

也參加了很多的綜藝節目，其中包括《超級演說家2018》。

在節目中，她談到那段往事，稱那段時間是人生的低谷，幾個月失眠、掉頭髮、不敢上街、不敢上網。

命運坎坷如黃奕，不也一樣走出來了。

不管此時此刻你正在經歷什麼，你要相信，你一定會走出來的！

就像她在節目中說的：

命運一定是掌握在你自己的手裡的！
