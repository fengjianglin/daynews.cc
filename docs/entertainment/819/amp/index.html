收 藏

86版《西遊記》是許多人的兒時回憶，我記得每次暑假都會播放，以前大多數人沒有手機，也沒有安卓的出現，電視陪過了我們的童年！《西遊記》一播出就創造了89.4%的收視神話，每到寒暑假，央視必然重播《西遊記》，重播次數超過4000次。成為中國電視劇史上不可超越的經典，陪伴了一代又一代孩子的童年。今天我們一起來看看當時那些演員的現狀吧！

六小齡童是《西遊記》孫悟空，據了解，六小齡童是最低調的一位，座駕只是一輛價值幾十萬的商務用車除了座駕低調，他目前的居所也是在普通的居民樓中，不是所謂的別墅豪宅。而且，六小齡童都沒有配備專門的造型師，連平時的理髮也會像普通百姓一樣，出入理髮廳。除了平時做事低調，六小齡童還是特別接地氣，近幾年，六小齡童經常出現在公共場合，以自己對西遊記的感情和領悟，向身邊的人們傳播著與《西遊記》有關的西遊文化。還有近幾年網上對他爭議不斷，他曾經諷刺過其它西遊題材里的孫悟空人物形象頗受爭議！

遲重瑞在《西遊記》，飾演唐僧， ，播出後人氣一路上升，被譽為是比較接近原著的唐僧扮演者 。之後他就在1990年遲重瑞與大他十一歲的中國女富商陳麗華女士結婚，身邊帶著三個孩子（一個兒子兩個女兒），二人已攜手走過二十餘載，現在的他幾乎沒有什麼影視作品了！

中國老先生認為遲重瑞對角色的成功塑造，「重瑞新體，二聖千秋。玄奘西去，堅毅之尤。鑒真東渡，碧海安流。雪山皚皚，鷲嶺悠悠。」這不僅是對遲重瑞一生演藝事業最好的總結，也是對他與夫人共同傳承紫檀文化最好的詮釋。

他有著俊朗的外形，深厚的藝術功底，加上家庭戲劇方面的熏陶，以及祖上一直信佛的影響，所以他演起唐僧來，得心應手，揮灑自如，並在當時的中國引起了很大的轟動，觀眾好評如潮，給人們留下了深刻的印象。

者閆懷在《西遊記》飾演沙僧，他1982年，步入電視劇的舞台。1986年，因出演電視劇《西遊記》中的沙僧角色而被廣大觀眾熟悉，同時在劇中兼飾太上老君等10多個角色。

1991年，閆懷禮與吳曉東、唐國強合作參演由王扶林執導的歷史電視劇《三國演義》，飾演程普一角 。

1993年，閆懷禮與夏宗學、黃國強合作，參演由霍庄執導的電影《痴男狂女兩世情》，在影中飾演鐵拐李一角；同年與李連杰合作參演電影《倚天屠龍記之魔教教主》，在該影片中飾演金毛獅王謝遜一角 。

1996年，再度和唐國強合作參演沈好放導演執導的歷史電視劇《東周列國》，在該劇中飾演鄭文公、齊莊公 。

1999年，又與歸亞蕾、周迅合作參演歷史電視劇《大明宮詞》，在該劇中飾演宰相裴炎[3] 。

2000年，在電視劇《鐵齒銅牙紀曉嵐Ⅰ》中，飾演陳輝祖一角。

1993年，閆懷禮因嗅覺不好誤吸入過量敵敵畏，再加上多年拍戲工作太過勞累，醫生給出的檢查結果是肺纖維化 。2009年4月12日上午，閆懷禮因肺部感染在北京世紀壇醫院去世，享年73 歲！

馬德華在《西遊記》飾演豬八戒，1986年，其與六小齡童、閆懷禮、遲重瑞等人領銜主演的由楊潔執導的古裝神話劇《西遊記》首播，馬德華在劇中飾演好吃懶做、法力高強的豬八戒，此外還出演了白毛猴、強盜、平頂山山神等其他小角色，並為王忠信所飾演的太白金星配音 。1987年，參與了《「齊天樂」春節聯歡晚會》的節目錄製。1988年，與六小齡童、遲重瑞、閆懷禮一同登上中央電視台春節聯歡晚會表演 ，此後馬德華息影了一段時間。

2006年，與張炬、謝芳等人合作主演喜劇《老人的故事》，飾演退休京劇名角施老頭；同年，與王秀強合作主演喜劇《瞧這一擔挑兒》，在劇中飾演想要發家致富的孔山。

2007年，與六小齡童、石筱群等人合作出演古裝劇《吳承恩與西遊記》，在劇中飾演好吃懶做、性格憨厚的豬八戒以及劇中吳承恩的好友姚老大 。2011年，主演由康寧執導的喜劇電影《冒牌董事長》，在片中飾演公司總經理廖總。

2012年，出演了由韓兆執導的喜劇《喜事連連：剩男相親記》，在劇中飾演小月性格豪爽的父親 。2013年，出演了由高曉松執導的環保公益短片《地球來信了》。

2014年2月，其出演的由森島執導的喜劇勵志電影《80後的獨立宣言》上映，在片中飾演一位鄉村書記 ；同年，與李玲玉在浙江衛視節目《我不是明星第二季》中為演員宋佳之女張楚楚助陣 。2015年4月，與嚴曉頻等人合作出演電視劇《相親相愛一家人》，在片中飾演陳玉的老公郝志國；8月，其出演的由吳錦源執導的抗戰劇《踏破硝煙》首播，在片中飾演大戶人家錢老摳 。

2016年1月，參加「大好河山 ·夕陽紅鄭州新春音樂會」，在會上演唱了歌曲《敢問路在何方》、《豬八戒之歌》 ；2月，其與馬精武、管宗祥等人合作出演的喜劇電影《過年好》正式上映，馬德華在電影中飾演一位打麻將的老人；4月，與包美美合作主演由楊亞洲執導的魔幻喜劇電影《封魔錄後傳》，在電影中飾演一位企業老總 ；5月，參演了由沈火新執導的古裝喜劇電影《滾出來，兇手》 ；6月，確定與六小齡童搭檔出演中美合拍3D電影《敢問路在何方》 ；7月，出演了由黃建勛執導的喜劇電影《飛來神石》，在電影中飾演想要搶奪七彩神石的龍霸天 ；8月，與桑娜合作出演的古裝魔幻電影《孫悟空七打九尾狐》正式上線，馬德華在電影中飾演菩提老祖 ；11月，與六小齡童、劉大剛等人合作主演電影《財迷》，在電影中飾演想得到郝先生的錢的朱首富 。

2017年5月，出演了由谷傳民執導的電影《招猴》；8月，主演公益教育電影《慈愛的心》，在片中飾演一位爺爺 [；9月1日，與姚銳、惠祥意等人合作出演神話電影《笑八仙前傳之馴妖記》，在片中飾演與其他神仙一同斬妖除魔的張果老 ；隨後，出演由劉愛金執導的警匪電影《天黑之前》，在電影中飾演因當幹警的兒子犧牲從而變成傻子的父親劉志 ；10月，其與傅泓霖、姚銳等人合作主演文藝電影《燦爛光輝》，馬德華在電影中飾演一位老村長 ；12月17日，其參演的由意紫導演的古裝電影《魔幻西遊2之伏魔篇》上線，馬德華在電影中飾演佛祖 ；12月19日，其出演的由陳奕銘執導的古裝神話電影《七娃降魔篇》正式上線，在電影中飾演兼濟天下的北斗尊者 。

2018年2月，參與綜藝節目《王牌對王牌第三季》的錄製，與1986版《西遊記》的部分演職人員聚首 ；3月，與部分86版《西遊記》的演員參與節目《天天向上》的錄製 ；4月，與李世宏、徐少華等人合作主演電影《誤入青春》，在電影中飾演身懷絕技的老人江海；隨後，其參演的由席東緒執導的玄幻電影《白骨夫人大戰狼妖》上線，其在電影中飾演女主角花兒的父親；8月，其與石玉鳳、李慧佳等人合作主演的慈善電影《追光少女》正式上線，在電影中飾演帶著學生去農村體驗生活的李老師 ；11月，其出演的由王軍執導的災難愛情片《在天堂等我》上映，他在片中飾演一個門衛 。

2019年1月，其編寫的自傳《悟能》正式出版 ；1月23日，參與2019年北京春節聯歡晚會錄製 ；2月，參與2019年山東衛視春節聯歡晚會的錄製，表演節目《二師兄回來了》以及演唱歌曲《卡路里》 8月23日，其出演的由姜大偉執導的電影《格桑花之愛在前行》上映，馬德華在電影中飾演郭爺爺 。12月，以動作捕捉表演的形式參演由韓延執導的電影《西遊記真假美猴王》（曾用名《敢問路在何方》 。

王伯昭在《西遊記》飾演白龍馬，1984年，王伯昭與演員六小齡童、馬德華合作出演了經典電視劇《西遊記》，因在劇中飾演白龍馬人型被大多數觀眾所認識！王伯昭不僅在演技方面表現出色，而且還擅長書法。 閑暇之餘，潛心筆墨，頗有成就。在自己首部電影《筆中情》中扮演男主角趙旭之，將自己書寫的「飛舞」二字，呈現在觀眾面前，也展示了他在書法領域的造詣。其書寫的大字渾厚有力、一擲千鈞；小字秀麗挺拔、氣韻流暢，有不錯的藝術價值，他還喜歡美食，也是開了一家滷味店賣著自己愛吃的滷菜！

歲月不饒人，一轉眼三十多年過去了，物是人非啊，回憶當初看《西遊記》的時候真的很開心，師徒四人經歷的種種磨難才能到達西天取到真經，曾經的回憶也只能在心裡了！
