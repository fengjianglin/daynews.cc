27歲的英國女演員娜奧米·阿奇正式成為《星球大戰》的一部分

2019-12-18 120條評論 4,725次閱讀 2639人點贊

娜奧米·阿奇的生活即將改變，她知道這一點。憑藉在《天行者》中的角色，這位27歲的英國女演員將正式成為《星球大戰》宇宙的一部分。儘管關於這部電影的細節在12月20日上映之前都是保密的，但是Ackie對於這部電影對她事業的重要性是坦率的。「我感覺我的生活真的發生了轉變，幾乎是立刻的，我知道這將標誌著我下一段旅程的開始，」當她發現自己在j·j·艾布拉姆斯(J.J. Abrams)的最新電影《星戰》(the latest film)中出演角色時，她這樣說。

你必須看這部電影才能了解阿奇飾演的叛逆角色Jannah的全部背景故事，但只要看一眼預告片，你就會知道她是個壞蛋。作為獨立電影和英國電視的常客，Ackie需要開發一套全新的技能來進入星戰的科幻世界。伴隨著光劍之戰、宇宙之戰和肉搏戰，Ackie的工作被削減了，但是她接受了挑戰。「整個經歷中最有意義的部分可能是，我知道自己能有多強大，我如何應對壓力，」她說。她認為片場的同事情誼幫助她做到了最好。「有太多值得感激的事情了，」她說。「我認識的朋友，我在片場學到的東西，還有和我一起工作的人。」

經過數月的拍攝和保密，這部備受期待的電影終於在昨晚的好萊塢世界首映式上得到了一絲喘息，感謝大家的辛勤工作。她穿著斯特拉·麥卡特尼的柱形禮服，看起來美極了。在紅毯上，她與劇組成員黛西·雷德利、約翰·波耶加、奧斯卡·艾薩克和凱莉·瑪麗-特蘭一起，讓人們深刻體會到這部世界上最重要的系列電影的故事。她說:「對我來說，那一刻就像是被凍結在了時間裡。」「當我長大一點再回頭看的時候，我會想『哇，我做到了。
