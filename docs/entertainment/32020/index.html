《葉問》系列落幕，終於來了一部能打的賀歲片

2019-12-25 2,070條評論 3,226次閱讀 281人點贊

賀歲片的概念雖然沒有以往那麼著重強調了，但每年11月下旬到12月末，依然是很多影片扎堆，很多一年沒看電影憋著要在年底消費一次的人期待的。

中國電影2019年的大盤子雖然超越了去年的600億，但是用《葉問》中的台詞來說，「一個能打的都沒有」。

中外電影呼呼啦啦上映了二十幾部，無論是打分高的，還是視覺爽的，沒有一個能再現國慶檔時的觀影盛況。

責任終於落到了《葉問4》身上。

其實好多人都不看好這部系列的終極篇。打日本人、打英國人、打泰國人，打古惑仔，打拳手，打殺手，打拳王，這次還打誰？

12月17日在佛山開幕的「功夫電影周」上，活動代言人甄子丹在見面會與開幕式晚會上，兩度提起《葉問4》是他的最後一部功夫片，確定與功夫片告別。

在之前10天播出的第四季《吐槽大會》上，罕上綜藝的甄子丹登上主咖位任由「吐槽」。

戲霸問題，和趙文卓、吳京的恩怨問題等等都在各個吐槽嘉賓口中不斷噴出，節目看點十足。

當然就像李誕說的「（吐槽大會）這破玩意兒沒想到拍到了第四季」，吐槽大會在吐槽效果的渲染下反倒成為一種重新聚攏人氣的焦點，因此該節目又被稱作「洗白大會」。

不管洗白也罷，告別功夫片也罷，甄子丹這麼高光地出現在大眾面前，在前幾部《葉問》上映時並不多見。

讓人倒是對《葉問4》的質量產生了一定的懷疑。尤其是劇情公布後，走出國門到大洋彼岸打老美，似乎不僅老套，而且沒啥看點。

讓人意外的是，《葉問4》的口碑與票房同時起飛。

儘管有販賣民族情緒之嫌，有蹭中美貿易摩擦之舉，一個周末三天三億多的票房，雖然還不是現象級的賣座，但在蕭瑟秋風今又是的2019年賀歲檔，已經是成績最好的一個。

和它同期競爭票房的是曾經的賀歲片之王馮小剛，他的新作純愛片《如果芸知道》，上映了三四天，依然沒有過億，貓眼預計最終可能僅為2億。馮氏票房神話不再。

它前面是口碑還不錯，但上映兩周票房還沒超過6億的《誤殺》。

在整個賀歲檔票房成績最高的還是《冰雪奇緣2》，也就是剛剛過了8億，但是它已經上映了一個多月了，已是強弩之末。

種種跡象表明，《葉問4》可能會是整個賀歲檔的票房冠軍。

低迷了一個月的賀歲檔，終於來了一部能「打」的了。

《葉問4》的動作戲依然打得好看。尤其是最後一場高潮戲，葉問對陣美軍教官，比武規則是「沒有規則，倒下為止」。

這是一場突破規則和底限的較量，為了勝利可以使用任何手段。對手更加殘暴，葉問也多了一些更加實用也更加兇狠的招式。

當然勝利者依然是葉問。

有人說這是中式意淫的範式。

個人覺得和一些強行表達情緒和意願的片子還是不一樣。

首先本片的切入點更加普通。葉問無法馴服處在叛逆期的小兒子，於是去美國給他找海外留學的學校，經過一番風波，最終和兒子達成和解。

影片視角不再沖著民族大義的高大上，而是小處著眼的更加平民化，出發點也更容易引起普通觀眾共鳴。

其次是本片中的葉問，已經身患癌症，一下子又拉近了和普通觀眾的距離——超級英雄也有打不過的對手。

教訓小流氓還行，到了和真正的高手對打時就開始不時地氣喘，體力不支。東西方不敗的葉問也終於被還原成一個普通人。

《葉問》系列不是抗日神劇，飛檐走壁、手撕鬼子、三八大蓋擊落飛機等違反自然科學的噱頭從沒有過。

詠春拳起於嚴詠春，衍於梁贊，盛於葉問。葉問的徒弟李小龍則讓全世界知道了不僅是詠春拳，而是真正的中國功夫。世所公認。

再者，《葉問》系列一貫的對打著弘揚中華武術旗號的某些欺世盜名的人予以「曝光」，《葉問4》也不例外。

這裡面還有個笑點的包袱。那個在舊金山混跡唐人街會些三腳貓功夫的羅師傅，被美軍教官一兩個照面就踹下舞台，還找借口嘟囔著「檯子那麼小」。

這位由當年邵氏小生羅莽飾演的羅師傅也曾在《葉問2》《葉問3》中被狠K，每次挨打也都會找出「桌面太滑」之類的理由。

而且羅莽不僅被甄子丹的葉問打，在梁朝偉出演的《一代宗師》中，羅莽還被影帝暴揍。

幸好，這是《葉問》系列的最後一集，曾經多年的TVB黃金綠葉，這回終於熬到頭了。

當然本片也不是簡單對江湖騙子的拆穿，還是把目光放到了李小龍在美國傳播中國武術所引起的在美中華武術界的衝突上。

傳統的門派觀念，與強身健體服務全人類的新觀念之間的對抗與消解，通過葉問利用功夫來做傳達。

此外，葉問的「打」更師出有名，卻又不說教：遇到不平的事情就要站出來，這就是學武之人的初心。讓對抗洋人歧視的「傳統」套路有了一些新變化。

甄子丹的葉問，表達依然含蓄溫婉，盡顯儒家武學精神。

《Ip Man》的最後一集，也讓葉問這個大IP終於盛裝落幕。

杜宇航主演的網路大電影《宗師葉問》也趁機在視頻平台上蹭IP播出。

杜宇航此次是繼《葉問前傳》後二度飾演葉問。

《葉問前傳》是2010年上映的，杜宇航飾演少年葉問，由香港著名快手導演邱禮濤執導，有樊少皇、元彪、陳嘉恆、黃奕、洪金寶、林雪、徐嬌、張繼聰、張繼軒、李力持等眾星捧月。

葉問長子葉准特別出演，而杜宇航正是葉準的徒孫。

他不但主演了《葉問前傳》，也在《葉問》和《葉問2》中都有角色出演，而且憑藉《葉問2》和《葉問前傳》中的表現，獲得了2011年香港金像獎最佳新演員的提名。

《葉問》系列其實對甄子丹來說是個挑戰。

進入新世紀後，他連續出演了葉偉信導演的《殺破狼》《龍虎門》《導火線》等動作片，甄子丹已經成為了功夫片的唯一代言人。

不過這幾部動作片雖然也火爆異常，但是裡面的甄子丹除了動作戲更加寫實迅猛外，角色和《精武門》時期的陳真沒有太大區別，性格衝動，動作兇狠凌厲。

但是《葉問》不一樣。

從獲得的榮譽就能看出來，「第28屆香港電影金像獎，提名最佳男主角」，雖然沒拿到影帝，但是從影二十多年後他的表演終於獲得了行業認可。

葉問對甄子丹來說，最大的不同，就是他收斂了眼神。

就這麼一下收斂，他身上爆裂的戾氣一下子蹤影全無，讓他從一個動作演員真正進化成為了一名演員。

拍了11年的《葉問》也終於有了一首可以讓人循環播放的片尾曲。

之前楊坤為《葉問2》唱作過片尾曲《信仰》，王蓉也唱過《葉問3》的插曲《葉問》。

但都沒有這首李宇春和方文山時隔七年後合作的這首《詠春》這麼能「打」。

李宇春當年首次觸電，就是與甄子丹一起拍攝動作電影《十月圍城》，李宇春的處女作便奉獻了吃重的打戲，且表現不俗，同時入圍了第二年的金像獎最佳女配角和最佳新演員等獎項。

十年後，李宇春以演唱的方式再次與甄子丹合作，也讓《葉問4》又多了一張情懷牌。

12年經典系列虎頭出演，豹尾結束，意猶未盡。
