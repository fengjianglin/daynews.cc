電影《解放·終局營救》女導演李少紅：最想拍戰爭中的人

2019-12-19 1,727條評論 4,255次閱讀 2999人點贊

近日，由韓三平任總策劃、李少紅任總監製兼總導演、常曉陽任導演的戰爭電影《解放·終局營救》發布了導演特輯。該片以平津戰役總攻前夕為背景，聚焦了戰火硝煙下的人性與真情。特輯中，常曉陽坦言了自己參與拍攝的初衷。李少紅也表示戰爭片是自己沒觸及過的領域，有極強參與感。兩人的「王炸」組合為該片上了「雙保險」，演員們稱讚不絕，影片將於12月27日全國上映。

「王炸」組合「一步上膛」 常曉陽：不能讓父輩人笑話

電影《解放·終局營救》的高標準效果呈現和順利拍攝離不開導演常曉陽與總監製兼總導演李少紅兩人全方位的準備和對內容的精打細磨。攝影指導李強坦言，在確認拍攝畫面時他們心裡已經「非常有數」了。導演常曉陽表示，自己拍攝這部影片是想讓現在的人知道70年前曾經發生過平津戰役這樣一件驚心動魄的大事。演員鍾漢良和王鏘透露，常曉陽對軍備細節非常注重，軍人出身的他經常在片場和大家分享飾演軍人的要領，其膽大心細的做事風格令演員傅亨稱讚不已。製片人沈羽倩表示，每個導演都有自己的風格，對的人做對的事是大家對這部作品一致的追求目標。

電影《解放·終局營救》聚集眾多演員傾情出演，解放軍炮兵偵察連連長蔡興福的扮演者周一圍，國民黨軍需官姚哲的扮演者鍾漢良，擁有雙重身份的神秘歌女梅艷扮演者鍾楚曦，解放軍炮兵偵察兵馬寶樹的扮演者王迅，解放軍炮兵見習參謀葛貴忱的扮演者王鏘，國民黨天津警備司令部稽查處科長錢卓群的扮演者姜皓文，中共天津市地下黨黨員、蔡興福妻子何秀萍的扮演者楊冪，國民黨兵站副官萬三的扮演者潘斌龍，姚哲之女姚君蘭的扮演者烏蘭托雅·朵，中共天津市地下黨黨員曹春生的扮演者傅亨，解放軍炮兵報務員廖楓的扮演者郭麒麟，國民黨軍迫擊炮連連長李梹的扮演者尹子維，小五子的扮演者魏子淇，梅艷弟弟大包牙的扮演者季路桐，林默雪的扮演者艾米，蔡興福何秀萍之子蔡解放的扮演者高銘昊，解放軍炮兵團團長的扮演者邵兵皆將在影片中有著重要表現。

片場「拚命三郎」 李少紅：戰爭片讓我有極強參與感

電影《解放·終局營救》的故事發生在平津戰役總攻打響前夕，周一圍飾演的解放軍炮兵偵察連連長蔡興福所帶領的解放炮兵偵察別動隊將要執行一個特殊的火線任務——不僅要將國民黨重兵看守的地下兵工廠一舉摧毀，還要救出被關押在監獄中的人質。一場以寡敵眾的「不可能完成」行動在中共地下黨的協助下迅速展開。此次發布的特輯中，總監製兼總導演李少紅表示，戰爭片是自己沒觸及過的領域，這使得自己有極強的參與感。該片對自己意義非凡，最重要的一點是拍了戰爭中的人，不論是普通戰士、老百姓，還是孩子，他們都是平凡人的縮影，也是大眾對於父輩的記憶。

正是源自這樣的拍攝初衷，李少紅在片場既是一個大家長的形象，又是一名名副其實的「拚命三郎」。演員王迅和潘斌龍表示，李少紅潤物細無聲一樣地指導著演員們的表演，大到歷史時代感小到角色做事的前因後果，事無巨細。周一圍也坦言李少紅是一個讓自己安心的導演。鍾楚曦認為，一天的爆破戲下來，片場所有人又累又困，李少紅對戰爭片創作的熱情令她依然精神奕奕地穿梭於片場，指揮著各部門行事。
