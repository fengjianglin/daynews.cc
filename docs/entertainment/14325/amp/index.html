他曾將趙本山弟子「請」下央視舞台，今再說相聲，實力猶在

12月13日晚，首屆「非遺」相聲大會閉幕式演出，在北京民族文化宮大劇院舉行。

自8號舉行以來，在首屆「非遺」相聲大會舉行期間，包括姜昆、戴志誠、王謙祥、李增瑞、李國勝、苗阜、高曉攀、何雲偉、武賓、高曉攀、尤憲超、金霏、陳曦等在內的老中青三代相聲名家紛紛登台獻藝，為觀眾帶來了歡聲笑語。

這其中，還有一位久違的相聲演員，那就是鄒德江。

在此次「非遺」相聲大會舉辦期間，他也曾搭檔相聲名家劉惠合說了一段相聲，叫《說與唱》。儘管這段節目的時間不長，卻將相聲演員說學逗唱四項基本功中的說與唱，較好的表現了出來。

鄒德江不愧是師勝傑的徒弟，相聲基本功還是比較紮實的。儘管已經50多歲了，但聲音依舊宏亮、吐字依舊清晰，學唱也是有模有樣，一段繞口令也表演的乾淨利落。

《說與唱》中，鄒德江不僅說了繞口令，還學唱了歌曲、京劇，不僅如此，還將繞口令與歌曲、京劇融在了一起。儘管包袱不是特別密集，但笑聲也是源源不斷，總體而言，效果還是不錯的。

這很容易想起八九十年代電視相聲最輝煌也最火爆的階段。

在相聲舞台上「消失」了那麼久，再次看到鄒德江登台說相聲，作為一名相聲愛好者，還是蠻感慨的。

鄒德江與相聲門師父師勝傑在央視節目的錄製過程中。

師從師勝傑的鄒德江，最早被觀眾熟知還是在上世紀90年代中期。當年，央視最火爆的電視節目之一有《曲苑雜壇》。其中有一個欄目叫《聰明的劇務》，鄒德江就是通過這個欄目被大家所熟知的。

《曲苑雜壇》是當年收視率非常高的央視綜藝節目，發現、挖掘、推廣曲藝新人，傳承、弘揚傳統民族文化，是這個節目的宗旨之一。洛桑、買紅妹當年也都是通過這個節目紅遍大江南北的。

​後來，鄒德江成了中央電視台名牌欄目《周末喜相逢》的主持人之一。《周末喜相逢》是一個喜劇類綜藝節目，每周播出一期，每期都有相聲、小品什麼的，也是電視相聲最紅火的見證者之一。而很多觀眾認識、了解、喜愛相聲，也都是通過《曲苑雜壇》《周末喜相逢》等央視綜藝節目的。

作為相聲演員，鄒德江的主要工作自然還是以相聲為主，也因此，在不少的晚會、活動、電視欄目中，都能聽到看見鄒德江的相聲作品。

《周末喜相逢》欄目停播之後，鄒德江也漸漸「消失」在了大眾面前。這個階段，除了繼續說相聲之外，鄒德江也參加了一些影視作品的拍攝，也因此，大家偶爾會在一些電影、電視劇中，或在一些活動上、欄目里看到鄒德江。

鄒德江雖然不是特別有名的相聲演員、節目主持人，但他有一件事情卻做得讓人至今都記憶猶新、歷歷在目。

那還是15年前的事情了。

2004年1月4日晚，在首屆CCTV喜劇小品大賽頒獎晚會《今夜小品燦爛》的直播現場，趙本山的徒弟在演出小品時，因發生「意外」，被節目組緊急叫停。兩人節目還沒演完，就被當時的節目主持人「請」下了台。

而這個節目主持人就是鄒德江。

在台上表演節目的兩個演員則是在《劉老根》中出演「二柱子」的張小飛和「大胖」的扮演者賀美齡。
