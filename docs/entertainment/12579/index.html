男星也追女神，肖戰追孫燕姿，張藝興追唐嫣，楊冪成男神收割機

2019-12-12 180條評論 4,409次閱讀 1745人點贊

在很多人心中，都有自己喜歡的明星偶像，可能因為他們的才華、長相，亦或是性格，被深深吸引。其實，明星同樣會追星，他們見到自己的偶像也會秒變迷弟迷妹。今天，小喵就帶大家看看當紅男星追的女神，你覺得誰的眼光更好呢？

肖戰是今年最大的黑馬男星，在古裝劇《陳情令》中將「魏無羨」這個角色演活了，不管是演技還是古裝形象都備受認可，成為了最火的男星。在被問到自己的偶像時，肖戰秒回答是孫燕姿，表示未出道前就喜歡她了，喜歡她的歌，喜歡她的性格。

而在某一節目後台，肖戰終於見到了自己的偶像孫燕姿，還一起合影了，表情很靦腆可愛，有木有小迷弟的感覺啊？

鄧倫這兩年也是紅得發紫，上一年在奇幻愛情劇《香蜜沉沉燼如霜》爆紅後，人氣勢如破竹，今年主演的三部劇更是接連上線，成為霸屏男神。而他的女神是張韶涵，看他跟女神的合影笑得多開心，也是追星成功。

白敬亭，長相白白凈凈的實力派演員，觀眾緣很好，他的女神是韓國著名女星林允兒。據悉，他在大學期間就很著迷林允兒，多次在網路上公開表白，女神的作品也都會追，幫助宣傳，標準的小迷弟。

馬天宇是很多人喜歡的男歌手，唱歌甜、長相暖暖的，沒想到坐擁幾千萬粉絲的他也追星，女神是王菲，還自稱自己是腦殘粉，每次出新歌一定會轉發，生日也會準時送上祝福，多次去她的演唱會捧場，甚至為了看演唱會還會推掉工作，真正的鐵杆粉絲。

超人氣男星張藝興很討觀眾喜愛，性格溫順可愛，在舞台上火力全開、魅力四射，也一直很敬業的拍戲，從來沒有鬧過緋聞或什麼不好的傳聞，口碑好，人氣高。他曾在節目中自曝他的女神是唐嫣，看來張藝興也是喜歡甜美可愛系的女生。

王鶴棣是98年的年輕藝人，這兩天發展還不錯，已經主演了不少的電視劇，人氣知名度高漲，他的女神是迪麗熱巴。

說道男星追女神，不得不說一下楊冪，她在圈內很多男星都表白過，堪稱男神收割機。頭號迷弟就是魏大勛，魏大勛在節目中多次表白楊冪，稱是死忠粉，已經追了十多年，還一起上了節目，跟女神零距離互動，還因此鬧出了緋聞，可以說是追星大贏家。

超人氣男團tfboys的隊長王俊凱也是楊冪的頭號迷弟，他可是坐擁7千多萬粉絲的超人氣偶像，沒想到還是被大冪冪「征服」了，大冪冪的魅力太大了。

還有影響力很大的「王大校長」王思聰也是楊冪的鐵杆粉絲一枚，多次公開力挺楊冪。大冪冪在普通人中就是萬人迷，沒想到在圈內也這麼受歡迎，魅力太大了！看了這麼多男星追的女神，你覺得誰的眼光最好呢？
