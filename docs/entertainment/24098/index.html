青年演員譚垚君參演《大明風華》演技爆表詮釋廢帝

2019-12-19 881條評論 3,246次閱讀 159人點贊

跨年大型古裝巨制《大明風華》已於12月17日起開播，該劇由張挺執導，湯唯、朱亞文、鄧家佳、喬振宇等主演。《大明風華》的故事橫跨明朝永樂至天順五帝六朝，講述了自永樂元年靖難之役後，明朝君臣上下一心，國家開創盛世的歷史故事。善易影視旗下藝人譚垚君在劇中飾演被朱棣推翻的建文皇帝朱允炆。

在劇集中，建文皇帝朱允炆一開場就披頭散髮，衣衫不整，他不停的質問身邊的大臣到底能不能抵擋住燕王朱棣的進攻。隨即女主孫若微的父親告訴朱允炆，劉伯溫曾經給他留下兩件寶物，能保住性命。這寶物正是剃刀和僧袍，而後朱允炆就剃成光頭，化作僧侶逃出宮去了。

在這裡，編劇採用的是民間傳說，讓電視劇更加具有傳奇意味。劇中譚垚君十分出色的演繹了朱允炆，把這個角色在絕望之際的歇斯底里、面對朱元璋畫像時的痛苦，以及回望之際的痛恨都表現的淋漓盡致，雖然戲份不多，但留給觀眾的印象卻十分深刻。

從目前劇情走勢來看，刺殺朱棣的幕後黑手很有可能就是朱允炆，所以朱允炆如何應對朱棣，是本劇之後劇情的一大看點。更為重要的是，孫若微註定是朱瞻基的皇后，但目前兩人有著不可調和的仇恨，所以最終兩人如何化解仇恨，相信關鍵也是落在朱允炆身上，後續劇情中譚垚君與演技大神王學圻和湯唯的兩場對手戲令人十分期待。欲知今後劇情，請每晚20:00鎖定湖南衛視金鷹獨播劇場《大明風華》，22:00優酷同步獨播哦！

譚垚君，1990年5月23日出生於黑龍江，善易影視男演員，畢業於上海戲劇學院表演系10級本科。曾出演《武神趙子龍》《女管家》《火爆天王》等電視劇。
