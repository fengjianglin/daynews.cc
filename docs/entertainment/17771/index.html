191216 我們不知道的太多太多，工作人員講述隱藏在新曲《外婆》背後的故事

2019-12-16 654條評論 2,723次閱讀 1556人點贊

演唱會，排練，健身，寫歌…… 照常執行著每個行程，演唱會上和粉絲溫柔互動，你把你的悲傷埋藏在心底最深處，直到被工作人員撞到，直到現在才發現我們不知道的你曾那麼痛苦……

結束15日晚在深圳舉辦的巡演最後一站，工作人員分享了在陪伴藝興工作這段時間的點滴細節，以及新曲《外婆》誕生的背後是怎樣的心境，公開的全文里寫道：

2019年份額的大航海演唱會今晚在深圳就徹底宣布告一段落了，12月的熱鬧餘溫里，我們和所有演唱會現場的Xback們在場館聽到了老闆寫給外婆的同名新歌。好像一下子也不知道該怎麼形容這種感受：

明明決定好要在2019年最後一場演唱會上首唱《外婆》並播放僅此一次的、有外婆影像的MV已經有一段時間了，工作人員們也都大多親眼見證，親身參與了老闆拍攝這支MV的全過程——甚至我們看見那段外婆的加油視頻遠比張藝興本人早更多更多。原本以為，從老闆到我們，每個人也都不會再有太大的情緒波動，可當場館裡響起外婆的聲音，響起老闆唱月亮粑粑的聲音時，大家還是莫名其妙在內場，在後台，在化妝間，在服裝準備間，在鏡頭後···在任何場館裡能聽得見歌聲的地方，突然就淚流滿面。

今年7月6號，張藝興大航海演唱會上海首唱。

此前一個月，外婆已經病得很重了。家人們也從最開始聽到「做好準備」時的情緒激烈變到後來的理智平靜。關於外婆的病情，大家都是瞞著老闆的，但老闆也從沒追問，只是後一段時間他頻繁回長沙，經常在病房裡一陪護就是一整晚，他跟外婆之間維持著一種揪心的和諧：你不說，我不問，你病著，我陪著。

上海場開始前半個月，外婆特別想要來看看老闆的演唱會。雖然所有人都知道，她的身體狀況根本不能允許這種輾轉折騰和旅途勞頓，但還是沒有一個人試圖去阻攔過她。從17歲離開家鄉，到27歲站上自己的個人演唱會舞台，都說張藝興努力努力再努力，可他到底是怎麼度過這十年，他一路走過來到底都經歷了什麼——好像沒有人會比陪他長大的外婆更了解。所以整個團隊、主辦方、醫院、家人，所有人都全力去計劃安排最合適的路線，最完備的醫療，以及應對各種突髮狀況的措施等等。

但由於外婆的病情還在不斷加重，最後，關於老闆的上海演唱會她還是未能成行。

我們第一次看見外婆的加油視頻就是在上海演唱會的後台，老闆就在一牆之隔的化妝間里。好像每一個人都沒辦法在他上台前給他看這個，因為誰也不知道，他會有什麼樣的反應，有怎樣的情緒波動。當時我們想著，就拖一拖吧，等一個合適的時機。

這一等，7月末，外婆過世了。

老闆連夜趕回長沙。之前之後，都是演唱會。他照常排練、綵排、健身、寫歌，還是照常喊大家吃飯，照常十幾個小時排練，照常工作，學習，飛來飛去。最多就是南京演唱會結束，他沒去慶功宴，他說累了想回去休息，說希望大家吃得開心，他來付錢。然後在萬千熱鬧和燈火輝煌里，自己一個人回了酒店。

再到後來，我們都以為他大概是已經過了最難的那一段兒了，直到有天，他輕描淡寫地發來一首歌，跟我們說，它叫《外婆》。

MV里有他從小到大從家裡到學校的路，也有外婆騎著二八大杠帶他回家時經過的大街小巷，有一起等過的公交車，也有最熟悉的，家鄉的日與夜。拍攝MV時，做記錄影像的攝影老師沒開機前一直站在那兒遠遠看著老闆，我們問他看什麼呢，他沉默了好一會兒，然後跟我們說：張藝興寫這首歌的時候我正好看見。當時車停在路邊兒，他坐在副駕，大家都下車買吃的去了，我忘帶了東西，回去一開車門，看他在邊寫邊哭。

那時候，我們才真切感知到有一句話雖然年代久了些，卻越長大越覺得理解：那一刻，他心裡有一場海嘯，可靜靜的，只有他一個人知道。

很多事好像過去了，也好像永遠都過不去了。

2019年12月15日，張藝興大航海演唱會一巡最後一場深圳站，安可曲的最後一首歌是《外婆》——他終於以自己的方式，帶外婆來到了他的演唱會現場，以自己的方式，讓她見證了他站在燈光匯聚的舞台上。

2019年，我們在還沒怎麼經歷過真正的失去時，見證了張藝興的失去，也明白了，成長本身就是由一場又一場沒有盡頭的失去組成的，但所有失去也會組成新的獲得：愛你的人，終究會一路陪著你，以不同的方式，在不同的故事。

挽不回已失去，就珍惜眼前擁有，希望聽完《外婆》，你可以給那些雖然越長大越覺得嘮叨，越長大越覺得無聊，卻始終深愛著你的人一個擁抱：正因為有他們，故鄉才永遠是故鄉，你才永遠被放在心頭上。

明天十點，我們，故鄉見。
