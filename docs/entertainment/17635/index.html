《慶余年》之後是《大主宰》，古裝劇扎堆，《枕上書》要等到何時

2019-12-17 2,011條評論 3,501次閱讀 144人點贊

喜歡看古裝劇的網友們今年一定過得很艱難，在11月份之前播出的古裝劇實在是屈指可數，小編有印象的大抵就是《長安十二時辰》《陳情令》《封神演義》等，而《慶余年》《枕上書》等讓人苦等了一年的古裝劇的檔期也是晃了大家一次又一次，當大家都不抱希望的時候，這些劇突然就扎堆開播了，簡直毫無預兆，小編最近真的是追劇追到上頭，周一到周四《慶余年》《鶴唳華亭》交替著看，周五到周日還有《劍王朝》，這日子過得不要太美好，而更美好的事情是，又有一大波優質古裝劇定檔將播。

其實古裝劇扎堆播雖然很好，但好劇多了，彼此之間難免有所競爭，就比如說最近正在熱播的《慶余年》《劍王朝》和《鶴唳華亭》，這三部劇都是非常不錯的，如果不是撞在了一起播出，相信它們都會成為爆款劇，而接下來的這些劇其實也都還是很受期待的，比如《大明風華》《大主宰》《錦衣夜行》等。

在《大明風華》之前不得不提的一部劇就是《夢回》了，《夢回》原名《夢回大清》，由李蘭迪主演，講述了一個現代女孩在遊覽故宮的時候意外穿越到大清，並且卷進了皇室的權利之爭中，這個故事梗概是不是很熟悉，跟楊冪主演的《宮》十分相似，如果這部劇早幾年播出的話，也許會很火，現在在《宮》和《步步驚心》之後，清穿劇的熱度不再，而李蘭迪的知名度又不夠，估計這部劇不會大熱，跟《大明風華》競爭的話，只怕會輸。說句題外話，清穿劇小編只認《步步驚心》，劉詩詩的馬爾泰若曦深入我心，當年追這部劇的時候，小編也是貢獻了不少眼淚呀。

《夢回》已經在12月14日播出了，而《大明風華》則是定檔12月17日，由湯唯、朱亞文主演，講述了孫若微為雙親報仇刺殺朱棣，失敗後嫁給了皇太子朱瞻基，最終放棄執念輔佐朱瞻基治理天下。以大明王朝為背景的電視劇不多，最出名的當屬《大明王朝》了，那是一部實打實的權謀劇，豆瓣評分高達9.7分，主創陣容也是超級強大。小編知道的另一部跟朱棣有關的電視劇還是穿越劇的鼻祖，也就是張庭主演的《穿越時空的愛戀》，那就是一部簡單的古偶，所以相比起來，小編還是很期待《大明風華》的，湯唯和朱亞文都是演技顏值雙重在線優質演員，而這部劇的配角陣容也很厲害，鄧家佳喬振宇張藝興俞灝明都是口碑很好的演員，相信《大明風華》會大爆，雖然小編並不喜歡湖南台。

和《夢回》《大明風華》比起來，小編其實最看好的還是由王源和歐陽娜娜主演的《大主宰》，原因也很簡單，因為易烊千璽的《長安十二時辰》爆了，所以小編相信跟易烊千璽一起出來的王源也會同樣優秀，而且TFBOY的粉絲眾多，王源也算是自帶流量的明星了，再加上《大主宰》原著粉絲的加持，相信這部劇一定會大火吧。目前豆瓣上顯示《大主宰》定檔12月26日，與之同一時間播出的還有《錦衣之下》。

《錦衣之下》由任嘉倫譚松韻主演，定檔12月28日，講述了天賦異稟的六扇門女捕快袁今夏因為一樁案件和性情狠辣的錦衣衛陸繹結下樑子，上演了一出歡喜冤家的戲碼，譚松韻飾演的袁今夏活潑開朗，而任嘉倫飾演的陸繹則是冷麵帥哥，這樣的人設讓小編不得不聯想到《》，楊紫和任嘉倫的表演深入人心，不知道賺取了多少眼淚，楊紫的演技也是超級棒，作為一名紫米，小編私心還是想看楊紫的作品的，不過譚松韻也很可愛，相信《錦衣之下》也是可以與《大主宰》較量一番的。

最後不得不提一下《枕上書》，相信很多人都跟小編一樣，期待《枕上書》好幾年了吧，記得播《三生三世十里桃花》的時候，小編剛剛懷上朵朵，如今朵朵都快要上幼兒園了，但《枕上書》卻還沒有播出，真是讓人等的很憔悴，不過如今古裝劇扎堆播出，小編總覺得《枕上書》應該也快了，畢竟馬上就2020了，而且在某平台上的近期播齣劇單中，《枕上書》也是比較靠前的，相信大家很快就能看到東華帝君和白鳳九啦。
