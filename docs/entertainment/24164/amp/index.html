近10部古裝劇“泄洪式”上線，寒冬中何以“富貴險中求”？

天天要聞
2019-12-19

說了一年的古裝劇寒冬，卻在真正的寒冬時節逐漸「回暖」。

《鶴唳華亭》《從前有座靈劍山》《慶余年》《劍王朝》《一夜新娘》《惹不起的殿下大人》《夢回》《天機十二宮》……近一個月來，各大視頻網站先後扎推上線了近10部古裝劇。

此外，湯唯與朱亞文主演的《大明風華》目前已官宣定檔，將於12月17日起在湖南衛視和優酷上播出，以及王源與歐陽娜娜主演的《大主宰》、譚松韻、任嘉倫主演的《錦衣之下》傳言也將在年末或者2020年年初播出。

不僅數量上回潮明顯，在聲量上，古裝劇也掀起了最近劇集市場的觀看狂潮。

貓眼全網熱度榜顯示，《慶余年》《劍王朝》《鶴唳華亭》三部古裝劇始終位居榜單前列。其中《慶余年》已成為年末的爆款之作，豆瓣評分達到8分，其也是迄今為止，唯一一部播出前期就敢在會員基礎上再度收費的一部劇，火爆程度可見一斑。

捉摸不透的2019年古裝劇

一系列數據證明，經歷了各種獻禮劇、都市劇霸屏之後，觀眾對古裝劇的需求十分強烈。尤其是《慶余年》《鶴唳華亭》等作品的出色表現，讓今年各種受限制的古裝劇看似「前途一片光明」，但與此同時也有不少業內人士認為，目前播出的古裝劇大部分都是已經積壓了很久的劇，實際上只是在去庫存、消化堰塞湖的過程。

觀察幾部劇的出品時間，《慶余年》影視化的消息最早出現在2017企鵝影視電視劇年度發布會上，去年8月發布首款預告片，10月又出現在東方衛視的開放大會上，直到一年後才低調在網路平台播出。《劍王朝》也歷經了較長的等待時間，2017年12月，該劇在浙江橫店影視城開機拍攝 ，2018年4月便已殺青。

不過《從前有座靈劍山》《一夜新娘》等中小體量網劇的出品時間均在2019年，斷言目前大部分的古裝作品是積壓劇，似乎有點不太恰當。但種種現象表明，2019年下半年古裝劇的市場環境確實讓人捉摸不透。

首先是「零宣發」播出成為常態。早在暑期檔的《陳情令》《長安十二時辰》悄悄播出之前，《九州縹緲錄》在開播前半小時就被突然撤檔，接著《封神演義》臨近大結局卻被突然停播，再到後來的《陳情令》《長安十二時辰》《鶴唳華亭》《慶余年》等，古裝劇零宣傳開播已是常態。

除了「零宣發」播出，轉網播出是今年古裝劇的另一「求生法門」。《第二次很美》《鱷魚與牙籤島》《熱愛》《極速救援》《河山》《澳門人家》……觀察最近上星的電視劇，古裝劇數量居然為零。網劇這邊，古裝劇則霸佔了半壁江山。

事實上，從今年年初到12月中旬，成功在衛視播出的僅有《知否》《小女花不棄》《九州縹緲錄》《大宋少年志》《招搖》《封神演義》《重耳傳奇》等7部古裝劇，而去年有15部。

當然，從《長安十二時辰》《慶余年》等熱播網劇來看，今年影視劇「由台轉網」的這一過程中，古裝劇能否成功」上星「已經不足以對該劇的成績造成任何影響。

而隨著優愛騰開始佔據主導地位，各大視頻平台也越來越多的參與到作品的投資出品中。《慶余年》《從前有座靈劍山》背後是騰訊影業，《劍王朝》背後是愛奇藝，《鶴唳華亭》背後是優酷，《一夜新娘》背後是芒果TV。

視頻平台的強勢表現正在改變電視檯面向製片方購劇的2B模式，將整個劇集推向了2C階段，其影響主要表現在會員收入佔比上漲。

另值得一提的是，有著粉絲基礎的IP改編劇仍然是古裝劇市場的主流。在近期播出的作品中，《鶴唳華亭》《從前有座靈劍山》《慶余年》《劍王朝》《一夜新娘》《夢回》《天機十二宮》均根據小說改編。

去年《扶搖》《武動乾坤》《斗破蒼穹》等大IP劇播出成績不達預期，業內紛紛直言大IP已失靈。

大製作古裝劇為何一度成為眾矢之的？第10放映室對《武媚娘傳奇》的官方「吐槽」或許能夠引起創作者們的重視和反思。

「憑藉大明星、大製作、大宣傳名聲大噪，極盡奢華的服裝和演員的美貌相得益彰…從劇本到選角到表演，極盡所能地奉獻了自己的全部藝術造詣：高顏值、高顏值和高顏值，舍此之外別無其他。

「將盛唐氣象簡化為風花雪月的故事，將王朝演進簡化為情慾戰爭的宮廷，將歷史傳奇劇簡化為青春偶像劇。對歷史人物肆意解構的精神，正應徵了那句至理名言：無知者無畏…”

曾幾何時，大IP古裝劇總是極致奢華，極致高調。但在今年，大IP古裝劇主打「低調內斂」路線，反而重新站穩了腳步。可見，IP不是古裝劇的原罪，口碑傳播+題材紅利才是作品脫穎而出的關鍵。

古裝劇回暖了嗎？

2019古裝劇大盤整體遇冷，直到暑假檔和年末才有所發力，並且誕生了《長安十二時辰》《慶余年》等爆款作品，這些作品無疑如寒冬中的暖陽，多少帶來了一點希望。

不過需要注意的是，《大宋宮詞》《帝凰業》《天下長安》等大熱劇都沒有如期播出。更有甚者不得不面對等了兩三年也無法播出的命運，包括《愛無痕》《謀聖鬼谷子》《蔓蔓青蘿》等劇，時間最長的一部《謀聖鬼谷子》已經積壓6年難以播出。

古裝劇遭到「積壓」一方面與作品本身的內容有關，一方面也受當下市場環境和政策風向的影響。

今年是「新中國成立70周年」開始，2020年要「全面建設小康社會」，2021年是「中國共產黨成立100周年」。可以預見，在未來兩三年，古裝劇依然是「富貴險中求」。

大量古裝劇在不斷積壓，可每年大概也就兩個出口期，這讓製作方和出品方面對這一類型的態度越發謹慎。

備案信息顯示，2019年前三季度，全國拍攝製作電視劇備案共646部，比去年同期的886部減少27%；拍攝製作電視劇備案共24617集，比去年同期的35209集減少30%。

其中古裝劇每月備案數量與2018年同期相比呈斷崖式下降，從三月開始至九月，連續出現備案數為個位數情況，且五月僅為1部，七月為2部。最新公布的10月份備案中，古裝劇也只有3部。

從備案劇細分題材來看，截止到目前，宮廷劇持續遇冷，還未有一部備案。而在為數不多的古裝劇備案中，細分題材主要集中在「傳記、傳奇」類，普遍在創作上體現出正劇化走向，以反映歷史人物、歷史事件為架構，或以小人物成長奮鬥為核心。

備案數斷崖式下跌是否意味著明後兩年「古裝劇不行了」？對此也需要理性看待，行業人士透露，備案信息只能體現當下的狀況，不少製作公司會選擇臨開拍前再上備案。

也就是說，備案數下降並不驗證古裝劇市場不復往日風光。作為中國電視劇一大重要類型，古裝劇某種程度而言是「長盛不衰」的，有變化的只是排播定檔的方式、細分題材的革新，或是收費方式上的「活久見」。

知乎上有一個話題「為什麼大火劇大部分都是古裝劇？」其中一個回答是——好奇心和新鮮感：古裝劇在編劇上可以不受現實限制，造型上大有文章可做，在具體情節上都是現代人很少或從未見到或接觸到的，可以滿足人們的好奇心和新鮮感。

正所謂禁限令限制有多狠，反彈就有多猛。對於好內容而言，正是「超級加倍」獲得收益的好時機。
