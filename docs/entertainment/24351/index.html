明星課堂：白宇被催婚？郭碧婷處處受豪門管制？張佳寧逃出泰川？

2019-12-18 346條評論 4,252次閱讀 1296人點贊

1、白宇

白宇現在就是拍戲呢，吻戲路透圖不還傳出來了嘛，之前這部戲最初找的人就不是他們倆，剛好那邊都有事沒接，他就直接談下來了，現在和女朋友的感情很穩定，兩人也並沒有用感情這塊做文章，之前女方團隊是有想法的，但男方這邊不同意，就沒有干，而且據說好事將近，雙方家長還是有催的。

2、郭碧婷

16日晚上，向佐曬出了一家四口的合照為爸爸向華強慶生，並配文今年第一次有兒媳婦陪你，希望你和媽媽繼續恩恩愛愛開開心心健健康康，隨後郭碧婷也轉發表示開心，這是郭碧婷首次為公公向華強慶生，只是「開心開心」這個字，卻被拿來和奚夢瑤作比較了，不少網友都說在後者的比較之下，顯得郭碧婷不是那麼嘴甜會來事。

所以大家有沒有感受到，自從雙雙嫁入豪門之後，郭碧婷跟奚夢瑤就互相較上勁了一樣，兩人稍有動作就會被拉出來做比較，特別是現在奚夢瑤正在籌謀復工的階段，她最近的通稿很大部分都自動捆綁了郭碧婷，她沒少暗戳戳地散布消息說自己過得比郭碧婷好，自己夫家更有錢之類的拉踩郭碧婷，甚至還直接盯上了郭碧婷的資源。

而郭碧婷這邊，倒是不太在意奚夢瑤這邊的動作，因為她現在有更重要的事情要做！看過她的綜藝的觀眾應該都知道，郭碧婷原本是比較獨立的內斂的一個人，只是自從與向佐結婚之後，她的獨立受到了諸多限制，其中最大的影響就是來自她強大的婆婆，現在她婆婆十分渴望抱孫子，還特意拿齣電影資源作為獎勵，一心激勵郭碧婷為向家開枝散葉，而且其他生活方面，郭碧婷也是處處受到管制，現在是想換個髮型，都要問問婆婆的意見了。

3、張佳寧

雖然頗受baby恩遇，自入泰川之後，就成為了被力捧的對象，不管是商務資源，還是影視項目都到手了不少，但當下張佳寧還是義無反顧的棄baby和泰川於不顧，轉頭去了王凱的新公司。

張佳寧之所以會投奔王凱，完全是因為她而今在泰川的立場著實尷尬，她之所以能進泰川並且得到力捧，開始全然是因為她的舅舅張曉龍是教主的狗腿子，可是加入泰川之後，隨著教主夫婦的失和，她卻莫名其妙的被站隊到了baby的身邊，在這種情況下，她這一天天都過得如履薄冰，因此眼下她只能選擇及時抽身，遠離泰川這個是非之地。

當然了，baby固然心狠手辣，但是對張佳寧的出走，她卻並不敢有什麼明面上的還擊，一來，張曉龍不僅是教主的狗腿子，更是任泉的意難忘，既然他把張佳寧看作是掌上明珠，那baby再這麼囂張，也得投鼠忌器；二來，王凱背後站著正午，張佳寧現在去給王凱錦上添花，就是正午的人，baby當下要是表露不悅，就是開罪了正午，為了大局著想，baby這頭自然也不敢挑釁太甚。不過，目前baby仍舊資本傲人，所以正午這邊也不敢「欺人太甚」，為了讓baby心平氣和的吃下這個啞巴虧，金晨那個已經被搶走的正午角色，稍後就又會回到金晨手上了。

自今日開始，教主正式榮膺中國電影家協會青年和新文藝群體委員會會長一職，其在貴圈地位之隆只怕已然少有人敵，後續baby這邊只怕不管怎麼掙扎，到最後都是要服軟的。

4、張若昀

張若昀現在主演的《慶余年》追著看的人非常多，對他來說也是挺好的，加上劇里有好多老戲骨，他演技也沒有被比下去，反而算是撐了起來。現在自己感情方面也是甜甜蜜蜜，而且女方也懷孕了，彼此都挺期待新生命的降臨的。他原生家庭其實還挺複雜的，所以有時候會過分成熟。也多多少少對他現在的性格造成了一定的影響。

5、李蘭迪

蘭迪妹妹也是慘，前一段時間剛因為綜藝被罵，好不容易新戲上線了，結果又被吐槽，不過也不能怪觀眾，前幾集的劇情實在槽點太多了，而且確定這是步步的原班底？至於說李蘭迪的身材和形體，一直都是很大的問題，恰好拍這部戲的時候是她最胖的時候，不過最近一直在努力減肥，現在已經瘦了很多了。
