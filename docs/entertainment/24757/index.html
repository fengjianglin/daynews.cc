成龍辛酸史，當年被古龍當猴耍，躲進廁所痛哭：莫欺少年窮

2019-12-21 571條評論 2,191次閱讀 832人點贊

成龍辛酸史，當年被古龍當猴耍，躲進廁所痛哭，網友:莫欺少年窮!

相信大家對古龍都不陌生，他的武俠小說引進了經典的文學殿堂，開創了近代武俠小說新紀元，將武俠文學推上了一個新的高峰。

古龍是一個特別有才的流氓，而且還具有浪子的個性，因為他特別的喜歡喝酒，所以經常誤事，並且難免會出現代筆什麼的，導致他前後的水準真的是參差不齊，也是讓人感到特別的惋惜。

還有很多的人一直都認為，古龍和金庸的地位，是可以相提並論的。

眾所周知，娛樂圈如果說有後台支撐的話，一路上就會走得非常順利，但這種情況比較少見，有很多的大牌明星都是從跑龍套做起的，並且全靠往日的辛酸和辛苦的打拚，才能累積起今天的地位。

而我們今天要說的這個人就是成龍，成龍曾經在走紅的之前也是一個無名小卒，他當時為了讓古龍先生給他寫劇本，曾經陪酒陪到躲到廁所里痛哭，被古龍先生當猴耍，最後還被古龍先生罵他丑，但是他卻還是要陪笑臉。

古龍先生這個人說話非常直來直去，根本不顧忌成龍的面子，直接當著他的面，說他長得實在是太丑了，根本就不會給他寫劇本。

這件事情對成龍的打擊很大，成龍曾經在採訪中還說道：作為一個演員是沒有話語權的，只能聽從導演的安排。

幸運的是，後來成龍也遇到了自己的貴人，那就是導演羅維，羅維打算把成龍培養成「李小龍第二」，所以讓他出演了《風雨流星》等一系列武打劇。

這些都是從古龍的小說中改編的，但是成龍還是被古龍先生給批判了，並且他的作品不是給成龍拍的，是給狄龍和姜大衛拍的。

其實當時羅維也是因為成龍年輕身手靈活，勇於嘗試，並且他敢於挑戰高難度的動作，所以羅威對他抱了很大的希望，也有心要捧紅他。

所以對成龍說:你不是想要成為李小龍嗎?我看你的藝名就叫成龍好了。當時成龍的名字叫「陳元樓」，而「成龍」兩個字的意思就是「成為李小龍」。

而成龍正是憑藉著自己的實力和忍耐，開創出了一個和李小龍截然不同的風格，一個屬於自己的喜劇武打風格，得到了觀眾們的認可，獲得成功。

可惜的是，古龍在85年的時候去世了，如果古龍能活到現在，見到成龍現在的成就，他不知道有何感想。

很多的網友也都說古龍當時是真的看不起成龍，而成龍也是用自己的實力來證明了自己，所以說:莫欺少年窮。這是一個道理的!
