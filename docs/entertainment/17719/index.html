那些根據真實故事改編的電影，你都看過嗎？勵志感人值得深思

2019-12-16 31條評論 1,659次閱讀 520人點贊

現在有很多電視都是根據真實故事改編的，那麼你看過這些電影嗎？你都知道是那些原型改編的嗎？下面就由小編給您盤點一下。

1.我不是葯神

是由文牧野執導，寧浩、徐崢共同監製，徐崢、周一圍、王傳君、譚卓、章宇、楊新鳴等主演的劇情片 。該片於2018年7月5日在中國上映。該片講述了神油店老闆程勇從一個交不起房租的男性保健品商販，一躍成為印度仿製葯「格列寧」獨家代理商的故事。

原型是無錫人陸勇是名白血病患者，也是一位企業家，許多白血病患者稱他為「葯神」。2002年，陸勇被查出患有白血病，隨後開啟他漫漫尋葯之路，為自己的同時也幫助病友，他也通過網購的信用卡為很多病友代購了這種藥物，被稱為抗癌藥「代購第一人」。

2.當幸福來敲門

由加布里爾·穆奇諾執導，威爾·史密斯、賈登·史密斯、桑迪·牛頓等主演的美國電影。影片取材真實故事，主角是美國黑人投資專家克里斯·加德納。影片講述了一位瀕臨破產、老婆離家的落魄業務員，如何刻苦耐勞的善盡單親責任，奮發向上成為股市交易員，最後成為知名的金融投資家的勵志故事。

是一部改編於美國百萬富翁克里斯·加德納個人的真實事迹的故事。

3.戰爭之王

是由安德魯·尼科爾編劇並執導，尼古拉斯·凱奇、傑瑞德·萊托、布麗姬·穆娜、伊恩·霍姆等主演的犯罪題材類電影，於2005年9月16日（北美）上映。該片是一部關於描繪武器交易世界的電影，講述了一個發生在烏克蘭軍火商人尤瑞·奧洛夫身上，充滿火爆動作場面的故事。

原型俄羅斯籍軍火商維克托·布特。 布特,先前是蘇聯空軍軍官,能說六國語言,涉嫌在非洲、南美、中東等地大量販賣軍火,綽號「死亡販子」。他的公司職員超過300人，擁有約40架貨機，他所擁有的「安」式運輸機的數量是世界上最多的。

4.熔爐

影片由黃東赫執導，孔劉、鄭裕美、金賢秀、金志映等主演。該影片以2000年至2004年間發生於光州一所聾啞障礙人學校中性暴力事件為藍本，描述該時間所引發的悲劇以及學校的教師和人權運動者一起力圖揭開背後黑幕的故事。2011年9月22日在韓國上映。由於該影片所反映出的社會問題受到韓國國家高度關注，故該電影亦被稱為「改變韓國國家的影片」。

是根據真實事件韓國作家孔枝泳根據2000年到2004年光州某聾啞學校學生被教職工性暴力的事情改編的。

5.綠皮書

是由彼得·法拉利執導，維果·莫特森、馬赫沙拉·阿里主演的劇情片，講述了意裔美國人保鏢托尼，他被聘用為世界上優秀的爵士鋼琴家唐開車。鋼琴家將從紐約開始舉辦巡迴演奏，倆人之間一段跨越種族、階級的友誼的故事。

是根據真實故事改編的，它發生在上個世紀六十年代，當時美國的種族歧視特別嚴重，黑人在美國的地位很低下，很多地方都是禁止黑人出入的，於是在當時「綠皮書」盛行。那本小冊子算是當時黑人的出行指南，裡面會記載著黑人可以出入的旅店、飯店、商店等公眾場所。而電影《綠皮書》講述的就是那個時代背景下，一個「不算太白」的白人和一個「不算太黑」的黑人之間的故事。

6.中國合伙人
該劇由香港導演陳可辛執導，黃曉明、鄧超、佟大為主演。該片於2013年5月17日在中國上映。該片講述了「土鱉」成東青、「海龜」孟曉駿和「憤青」王陽從20世紀80年代到21世紀，大時代下三個年輕人從學生年代相遇、相識，共同創辦英語培訓學校，最終實現「中國式夢想」的故事。

是以新東方教育集團俞敏洪和王強等人的經歷為故事藍本,進行藝術改編和誇張。

7.親愛的

由陳可辛執導，張冀編劇，趙薇、黃渤、佟大為、郝蕾、張譯等主演。電影主演講述以田文軍為首的一群失去孩子的父母去尋孩子以及養育被拐孩子的農村婦女李紅琴如何為奪取孩子做抗爭的故事

是根據央視早年的一則「打拐」新聞改編而成，講述一對夫妻因為關係不和睦，兒子成為他們唯一的聯繫。

8.敦刻爾克

由克里斯托弗·諾蘭執導，菲恩·懷特海德、湯姆·格林-卡尼、傑克·勞登、哈里·斯泰爾斯、阿紐林·巴納德、詹姆斯·達西等領銜主演。

該片改編自二戰歷史事件「敦刻爾克大撤退」，當時40萬英法聯軍被敵軍包圍在敦刻爾克的海灘上，面對敵軍步步逼近的絕境，他們不得不為自己的命運背水一戰。

你們還看過什麼真人改編的電影，可以推薦給小編我，歡迎在評論區留言。
