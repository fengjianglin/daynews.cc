眾明星平安夜曬美食 林心如自製烤雞誘人 袁姍姍的菜重口味嚇人

天天要聞
2019-12-25

12月24日是一年一度的平安夜，平安夜是聖誕節前夕，雖然是外國的節日，但由於中西文化融合，平安夜和聖誕節已經是一個世界性的節日了，在中國大家慶祝很多節日都離不開美食，那麼平安夜也不例外，很多明星在這一天也通過曬美食來慶祝這個節日。

林心如在個人社交平台曬出了自己親手做的烤雞大餐，並配文「大成功」。哇，從照片中看，心如版的烤雞色澤飽滿，十分誘人，光看了都想吃。

看來通過在《親愛的客棧》的歷練，林心如的廚藝也是有所提升啊，除了煲湯外，我們的心如女神又多了一道拿手菜。愛生活又愛美食，會做飯又會演戲，也是全能藝人了~~

小可愛楊紫在平安夜也奮戰在拍戲崗位上，十分辛苦，不過在收工後還是曬出了聖誕樹美食來慶祝節日，楊紫真的很有生活情趣了，用各種水果，蔬菜還有麵包堆積拼成的聖誕樹，看起來十分精緻可愛，這個聖誕樹可是很「健康」啊，全是低脂低能量的食材，不知道是不是楊紫的減肥食譜呢？哈哈~~

除了以上兩位外，袁姍姍也在平安夜曬出了一組美食，還配文「平安夜，隨便吃點」。其實袁姍姍的美食一點都不隨便，還很重口味呢。

照片中除了水果榴槤外，剩下的居然全是「蟲子」，有炸螞蚱，炸竹蟲，炸蠶蛹，各種「蟲子」琳琅滿目，很多網友看到表示：太嚇人了，還有網友表示剛吃的東西已經開始反胃了。看來袁姍姍曬的美食大家都接受不了啊，雖然這些東西很好吃，但真不是一般人能享用的，袁姍姍真的很膽子大了。

雖然看的人很害怕，但袁姍姍吃的卻很開心，在曬的一組美食中，還藏著一張自己的照片，扎著丸子頭，側靠在手臂上，看起來十分舒適休閑，尖下巴明顯，看起來保養的非常好，不知道是不是吃這些「重口味食物」的原因呢？

平安夜各明星紛紛曬美食慶祝，那你是怎麼度過平安夜的呢？
