她為養家拚命拍戲，一生未嫁累死片場，死後捐器官遺愛人間救8人

2019-12-11 236條評論 1,451次閱讀 240人點贊

吃瓜的娛樂圈，打娛帶你看沒有綵排的娛圈現場！

在娛樂圈中。

其實也是有很多是星二代，富二代。

在演藝圈中，他們的起點都比別人高。

成名，雖然不是全靠運氣和有著比別人高的起點。

但多少還是占著一部分的。

而就是有那麼些人還是欠缺一點運氣的，遲遲不紅。

她沒有背景，演藝圈中一路走來，全靠自己的努力。

她的努力和付出不成正比。

然而她便是因著努力拍戲過度勞累。

永遠的離開了我們，令人惋惜。

她便是夢庭麗。

生於1966年的夢庭麗，自幼便在台灣一個普通家庭。

因著家裡的經濟狀況並不是很好。

從小，夢庭麗便很懂事乖巧，不給家裡增添麻煩。

身為長姐的夢庭麗，下面還有3個弟弟和1個妹妹。

或許便是因著這個原因，夢庭麗便很懂事，很自強。

在初中畢業後，夢庭麗選擇去衛校。

以便畢業後，能早點出來換來一份能養活家裡人的工作。

從衛校畢業後的夢庭麗，成為了一名護士。

但因著家裡的開支來源幾乎都源於夢庭麗。

所以她只能拚命工作。

在一次機緣巧合下，夢庭麗因著出眾的長相。

被廣告公司的工作人員相中。

隨著廣告的發酵，夢庭麗被演藝圈導演相中。

1990年，夢庭麗出演了首部電影作品《中華警花》正式的踏入了演藝圈。

隨後幾年，夢庭麗先後出演了多部作品。

但都沒什麼水花，且都是些龍套的角色。

1994年，夢庭麗便轉戰到了影視圈。

她先後出演了多部作品。

比如《包青天》、《嫁妝》、《女人花》等，這些經典的作品中都可以看見她的身影。

其中，夢庭麗最讓人印象深刻的作品便是《香帥傳奇》了吧。

這部作品在1995年播出之後，受到了觀眾的一直追捧。

更是創下了收視傳奇，成為了台灣經典的電視劇之一。

這部作品不僅成就了鄭少秋一副「俠義」的面龐。

也成為少女心中的夢中情人。

同樣，飾演「練霓裳」的孟庭麗也因《香帥傳奇》而一夜成名。

通過這部作品，也讓夢庭麗對演繹藝術有著極大的樂趣。

她自知不是科班出身，演技相比較其他演員有許多弱勢。

但她在不斷的接戲拍戲中，磨鍊自己的演技。

不管是古裝還是現代劇，她都可以拿捏的很好。

通過自己的努力補足了自己的缺點。

或許，很多人都認為孟庭麗如此頻繁接戲一定身價不菲。

即便不是主演但是這些電視劇的知名度都不低，她的片酬應該還是很客觀的。

但她拍戲換來的片酬大部分都給了家裡。

而孟庭麗本人得生活過得也十分簡單。

住的是租來的房子，每餐都是將就著吃。

甚至，為了養家，她不得不為了自己的事業去打拚。

為了家她犧牲了自己的愛情，一輩子沒嫁人。

出道多年來，夢庭麗雖然並沒有躍身為一線演員。

但她的演技還是獲得大眾的認可的。

而夢庭麗不僅在演藝圈有成就，在綜藝主持也是有成就的。

或許就是因者她「拚命三娘」的個性，為她之後埋下了病因。

也是這個隱患，讓意外便這麼悄無聲息的來。

孟庭麗當時因得了流感引發肺炎。

她卻仍堅持繼續工作。

最後在長時間的勞累之下， 孟庭麗因勞累過度累垮了。

導致在拍攝《加油！美玲》過程中，無預警休克、心臟及呼吸停止。

去到醫院被診斷出是腦溢血。

雖然有持續10多分鐘的搶救，但是孟庭麗還是永遠離開了我們。

為了養活家裡人，為了心中的理想，她活活的把自己累死了。

2016年2月5日，孟庭麗因腦幹功能嚴重衰竭後在當天宣告死亡，享年50歲。

出道拍戲26年，卻永遠的離開我們，讓很多人都惋惜。

經家屬同意，她的器官被捐獻，表示遺愛人間。

後來醫生透露，孟庭麗共捐出心臟、胰臟、肝和2顆腎。

還包括兩邊眼角膜，共5項器官，算起來有8人以上受惠。

如再包括骨骼，那更是算不清了。

「春蠶到死絲方盡，蠟炬塵灰淚始干。」

一個為別人活一輩子的人，到了臨死了還把自己全部都奉獻給別人。

是多麼偉大的一個人。

這個女人的一輩子很短暫，她的一生都在為別人而活。

或者說她沒有真正為自己活過。

最後還是希望，夢庭麗在天堂上幸福快樂，沒有病痛。
