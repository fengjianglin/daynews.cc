《有愛就有家》上演親情救贖，薩日娜、王琳二度合作成情敵

2019-12-25 1,340條評論 3,139次閱讀 1653人點贊

由CTMG泰和傳媒集團投資出品的都市濃情大劇《有愛就有家》將於12月25日在江蘇城市頻道第一劇場溫情開播。該劇由實力派演員薩日娜、畢彥君、王琳攜手青年演員王陽、是安、張佳寧等傾情演繹。《有愛就有家》以一貧一富的兩個現代家庭為故事發展背景，講述三代人之間的愛恨、恩怨、離合、成長及諒解的家庭情感故事。

薩日娜、王琳二度合作成情敵

劇中，薩日娜飾演的慈母李紅梅作為貫穿始終的主線，是兩個家庭、三代人之間從怨恨到和睦、從冷漠到溫暖的精神橋樑。李紅梅可謂「兒女成群」，然而自己的親生孩子卻只有一個。可她並未因此對其他孩子有所怠慢，對妹妹的兒子、哥哥的女兒以及「情敵」沈雪的孩子，她都一視同仁，用愛與寬容時時刻刻溫暖著每一個孩子的心。

王琳飾演精明幹練的企業「辣媽」沈雪，不僅輔助丈夫經商，還將丈夫與前妻（薩日娜飾演）的孩子視為己出，可謂十足「賢妻良母」。後來因為自己對待愛情的敏感性，漸漸與昔日的好友李紅梅為敵。猜忌、懷疑吞噬了沈雪對待家庭的單純情感，將全劇推向矛盾的高潮。

李紅梅面對沈雪的刁難，依然用真情軟化仇意，用仁愛消融了親生兒子的怨恨，用寬厚包容了另一個女人的自私，更用細膩溫暖的情義喚回了整個家庭的幸福。

深度挖掘 「家」文化

隨著家庭生活劇成為近年觀眾關注的焦點，蜂擁而至的類似題材使市場再次出現了盲目跟風的創作現象。《有愛就有家》用獨樹一幟的創作理念，贏得了市場認同。

出品人李洪在接受採訪時，表示：「《有愛就有家》選取了兩個背景截然不同的家庭為故事源頭，通過人物主線的情感串聯，展現出無論在什麼樣的家庭里，生活本質上都是普通的。家的文化精髓就在於平凡的幸福。在故事中我們加大了對社會生活深度的開掘，弘揚生活重壓下的社會責任感，開掘潛隱其中的更複雜、更本質、更具震撼力和感染力的人性內涵。」該劇帶有強烈的人文關懷，有一定的針砭社會現實的力度，能夠給觀眾帶來一些超乎一般影視劇觀賞審美的理性思考，從而使劇中人物和情節成為更具力度的載體。

揚子晚報/紫牛新聞記者 張楠
