為公司賣命十幾年，他們最後連姓名也不配擁有？

2019-12-13 198條評論 1,369次閱讀 350人點贊

正能量追星，陪你飯愛豆！

Q：豆姐，今天看到熱搜權志龍太陽不續約將無法使用藝名，公司的條件是不是也太苛刻了？這是不是他們逼迫藝人續約的手段？

沒辦法，藝人不續約，公司肯定會有所行動，但這一招似乎對他們倆也沒什麼用，因為就算不叫GD和太陽，權志龍和東永裴依舊是閃閃發光的巨星啊！

今天有媒體曝出權志龍和太陽不續約的話則將無法使用藝名，兩人所屬的公司YG在2003年和2015年相繼註冊取得了「지드래곤」、「G- DRAGON」、「TAEYANG」、「GD X TAEYANG」的商標權。而且不僅在韓國，YG在美國也已經登記。

儘管不少人都是通過G-DRAGON開始認識權志龍的，但是就如粉絲所說，「他們不需要藝名加持，就以權志龍、東永裴活動一樣是王者。」

最近剛發行新歌的鄧紫棋，在和前公司解除合約的時候也遇到過一樣的問題。由於公司總是背著鄧紫棋私自接活動，多次協商與讓步依舊得不到解決，忍無可忍之後終於下定決心解約。

而解約之後，鄧紫棋也將面臨未來可能不能再使用「鄧紫棋」這個名字，原因是早在2014年9月，公司就已經將「鄧紫棋」和「G.E.M」申請商標註冊，還在15年發布公告，產權使用到2025年結束。

而且這個問題在最近發行新歌的時候，得到了驗證。新歌無法在各大音樂平台上線，新專輯也無法在內地發行。

相同問題的還有S.H.E，合約到期之後沒有續約，組合名字也不能繼續使用了。

雖說S.H.E組合跟公司關係像是「一家親」，但談到解約牽扯到利益的時候，公司的「商人」嘴臉就徹底暴露，就算有著這麼多年的情誼，但卻連組合的名字都不願意給，這也造成了S.H.E今年的成團18周年紀念日也無法合體。

但公司沒有想過，愛豆讓我們著迷的從來都不是名字，而是無人可以替代的顏值與才華，這再膚淺不過的名字，就留給公司自娛自樂吧！

大家有任何對飯圈對娛樂圈的疑問都可以向豆姐提問~下一個被抽中解答的可能就是你啦！
