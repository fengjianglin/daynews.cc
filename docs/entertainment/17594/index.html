離開宋慧喬之後，宋仲基的“中年危機”來了？

2019-12-17 1,560條評論 3,132次閱讀 1525人點贊

12月17日，韓國媒體稱宋仲基與所屬社Blossom Entertainment（BLOSSOM娛樂）的合約即將到期，而宋仲基本人決定不再續約。

也有相關人士透露，宋仲基將成立個人公司。

韓國經紀公司與藝人簽訂的合約周期一般較長，宋仲基2013年與經紀公司簽訂了7年合約，期間拍攝了不少作品，但是唯獨與前妻合作的《太陽的後裔》熱度比較高，其他作品的水花不大。

宋仲基正式加盟BLOSSOM娛樂大約半年時間就入伍服役暫別演藝圈了，事業暫緩，資源不如從前，實話講與這家公司的感情其實沒有多深。

復出之後他似乎一直在挑劇本，最終以KBS水木劇《太陽的後裔》作為重回熒幕的首秀。

不得不說，宋仲基的眼光很准，這部劇一經播出就備受好評，數據也很好看。

全劇平均收視率達28.6%，最高收視率達38.8%，摘得年度收視冠軍，並獲得第52屆百想藝術大賞大賞。

爆紅的「雙宋CP」（宋仲基、宋慧喬）還分別獲得了由中國網友票選出的最具人氣男演員獎、女演員獎。

《太陽的後裔》或許是宋仲基簽約BLOSSOM娛樂最成功的作品了，他在這部劇中和宋慧喬配合默契，將柳時鎮的一身正氣與對愛人姜暮煙的專一表現得淋漓盡致。

因戲生情，宋仲基在當紅時娶了宋慧喬，收穫愛情的同時其實讓不少粉絲傷了心。

不過，婚後二人所表現出的狀態完全是「找到真愛」的感覺，起初不能接受他倆在一起的粉絲也真心祝福這對小夫妻。

愛情總是稍縱即逝，結婚不到兩年時間，宋仲基就向宋慧喬提出協議離婚，宣告了這段被網友蓋章為「神仙愛情」的婚戀關係告終。

宋仲基與宋慧喬分手乾乾脆脆，沒有過多糾纏。

熱心的吃瓜群眾還是像當初不能接受「雙宋」在一起那般，對他們突然分手這件事難以釋懷。

一時之間，眾說紛紜，有說夫妻中有一方出軌的、也有稱宋仲基與宋慧喬在一起「女強男弱」壓力太大而不能持續這段關係的、也有說男方家庭希望女方能夠續弦，但宋慧喬事業心太強沒有生娃意向而與婆家產生隔閡的……

宋慧喬、宋仲基的愛情無法挽回，他倆也沒有對為何離婚做太多解釋。

離婚這件事對藝人產生了一定的影響，對宋仲基來說最直接的影響就是人氣下滑。

1985年出生的宋仲基已經不再是那個鮮衣怒馬的少年。

他也有事業心，實話說簽約BLOSSOM娛樂之後，宋仲基的戲路沒有想像中寬廣，雖然嘗試了不少的角色，但《軍艦島》、《阿斯達編年史》沒能讓宋仲基塑造出能夠替換掉「柳時鎮」的新形象。

這對一位有追求的演員來說，是種失敗，但也無奈。

合約到期，考慮到這家經紀公司給自己的規劃不如預期，接近7年吊在這棵樹上沒能結出香甜的果實，宋仲基不再續約也在情理之中。

無論未來宋仲基是另覓新東家或是成立個人工作室，都需斟酌。

宋仲基是個聰明人，也是有「感情潔癖」的典型處女座男生，對他來說，換個環境，開始一段新的人生，離婚這件事給他帶來的瘡傷或許才能真正癒合，與「姐姐」的這段源於激情，終於一地雞毛的失敗婚姻才能真正翻篇……

34歲的宋仲基，還有很長的路要走……

漫漫長路中，家人與粉絲才是他最強大的依靠！
