鍾漢良陳偉霆張子楓已經愛上他們

2019-12-24 562條評論 2,100次閱讀 1073人點贊

信息時報訊（記者 陳慧）徐崢監製的賀歲電影《寵愛》，雲集眾多明星，但話題最多的還是出演電影的寵物「演員」們。在日前舉行的首映禮上，陳偉霆說拍攝時所有動物都被保護的很好，鍾漢良則自曝跟粉紅小豬拉緊距離的秘訣，郭麒麟則因為對手戲都是狗狗，拍著拍著就瘦了。

楊子執導的電影由六段風格不同的故事組成，但相比出演的明星，每個片段里出現的寵物更有故事。像陳偉霆與鍾楚曦「新婚」甜蜜蜜，卻總在關鍵時刻被寵物狗攪和了「好事」，于和偉與李蘭迪是對關係緊張的「父女」，百愁莫展下「喵主子」的出現讓兩人有了和解的契機，還有對鍾漢良「情有獨鍾」的楊子姍，遇上粉紅寵物豬爭寵也是無計可施。

北京首映禮上，導演楊子透露，片中這些動物演員來頭不小，全是遠道而來的好萊塢知名「演員」，拍攝比想像中難度大。但他強調自己想講的還是人和人的故事，「小動物其實是愛的紐帶，串聯起父女、朋友、情侶之間的情感。」

到場明星則分享了自己的寵物情緣，鍾漢良談到自己飼養的這頭「膚紅貌美」的小萌豬，瞬間角色附身，笑言如果不是人與豬之間的物種差異，自己絕對已經愛上的它了。至於他「養豬秘籍」則是這些：「抱著它猛親，和它吃一樣的食物（玩笑）。其實就是要多陪陪小豬，不拍戲的時候如果它願意，也一直陪著它。」

陳偉霆印象深刻的戲是抱著狗狗豪七跑，拍攝時工作人員擔心他抱不動，始終抬著凳子跟在旁邊跑。陳偉霆特別跟觀眾喊話，稱拍攝時所有動物都被保護得很好，請大家放心。演技有口皆碑的張子楓就說這次狗狗演員太會演，自己有些 「壓力很大」。原來有一場訓練狗狗過障礙物摔倒的戲，「它看到我摔傷後演出的疼痛表情，以為是真的，之後看到障礙物就死活停下不動了，怕我受傷。」輪到郭麒麟時，「單身狗」的他被楊子爆料，稱郭麒麟對手戲都是跟狗，又幾乎在室外戲，因為要一直追著狗跑，所以拍完後瘦了，甚至還變成了短跑運動員。

《寵愛》將於12月31日全國公映。
