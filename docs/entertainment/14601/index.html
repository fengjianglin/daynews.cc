50歲仍然美的驚艷的她，曾是馮小剛的夢中情人

2019-12-13 531條評論 2,597次閱讀 1314人點贊

大家好，我是小月

很多女星都憑藉驚艷的顏值，受到眾多男人的追捧，還有很多女星堪稱是「尤物」，在她們身上更是把優雅嫵媚展現得淋漓盡致！很多明星也是因此在娛樂圈的發展非常好，受到了眾多人的關注！

接下來要說的這位女星，她就一直被稱為「尤物」，更是馮小剛的夢中情人，她出道至今39年，一直都不爭不搶也不領獎，如今她已經50歲了，仍然還是美得驚艷！

這位女星就是許晴，不得不說許晴的確是非常驚艷，馮小剛曾直言：她是我們一代人的夢中情人，就連姜文導演都曾表示：她已經達到了瘋魔的狀態，彷彿渾身都沒骨頭！一直以來許晴都把自己身上的美詮釋得非常好！

許晴還是一位顏值與才華並存的女星，她可以說是一位被演藝耽誤的外交官！許晴出生書香門第，她的姥姥、小姨、姨夫都是外交官，而她的媽媽是總政歌舞團舞蹈隊隊長！許晴在11歲的時候就曾出演電影《鐵架008》，之後她就一直都有著一個表演夢！

在許晴19歲的時候就拿到了北京電影學院和外交學院的錄取通知書，最後許晴選擇了北京電影學院，走上了表演道路！說起來許晴出道至今已經39年了，一直以來她都不爭不搶不領獎，更是堪稱是不食人間煙火！

許晴22歲的時候就出演了電影《狂》，在電影中跨齡出演風雲開朗的蔡大嫂，在戲中還有大尺度的「哺乳戲」，許晴真的是為藝術獻身！許晴一直都有著驚艷的外表，笑起來有著甜美的酒窩，更是成為了觀眾們喜歡的演員！

如今許晴也已經50歲了，她依然還是保養得非常好，仍然還是美得驚艷！許晴至今都還沒有結婚，其實她並不是不婚主義，只是她沒有遇到對的人，因此祝福許晴還能遇到對的人！
