小料：李易峰、袁姍姍、劉詩詩、趙文卓、白敬亭、董璇、張若昀

2019-12-16 2,685條評論 3,596次閱讀 381人點贊

李易峰正面小生的形象現在營造的很成功，很多時尚資源都搶著要他，但是因為他現在以影視為主，不想因為其他事情耽誤拍攝，所以回絕不少邀請，2020年3月份他會陸續接代言的

袁姍姍私下經常把身邊的工作人員叫到家裡來，讓他們給自己收拾房間，有好幾次工作人員還在屋子裡翻出好幾天沒洗的內褲，她也毫不在意自己的邋遢，還讓工作人員隨手幫她洗一下

王迅私下很鐘意那種年齡小的女孩，越小越好的那種，之前他要碰一個未成年女孩，不過沒有成功，被身邊的兄弟攔下來了，讓他別因小失大，他這才沒有做出什麼事情

劉詩詩目前還是以拍戲為主，但是也沒有以前那麼拒絕綜藝類節目了，短暫的露臉是可以接受的，但是還是不喜歡真人秀之類的綜藝，她認為演員還是要專註自己的作品

趙文卓也要當導演了，是個動作電影，男主角也是他自己，大概明年3月份開。這個戲的攝像還挺厲害的，就是《春光乍泄》的攝影

白敬亭在上學的時候，有個女生對她有意思想追他，經常找理由找小白幫忙，結果小白居然說你咋天天這麼多事

宋繼揚就目前來看發展是算是很不錯的。因為品牌推廣挺多的，主要看中他的帶貨能力

董璇現在走的也是獨立大女人人設，上綜藝上節目都會表達自己的獨立，還是有不少節目邀請她的

張若昀在新劇里被吐槽是不是胖了，其實是因為他拍戲的時候影業的領導在片場，他經常下戲後帶著領導去吃宵夜，所以在鏡頭前會感覺到胖了
