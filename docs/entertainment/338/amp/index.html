演員被逼哭？吳鎮宇還全程黑臉，原來做演員沒有那麼簡單

收 藏

之前被大家稱作「吳三歲」的吳鎮宇，原本是溫柔的，這樣的暴躁的吳鎮宇你見過嗎！全程黑臉，面對女生哭，沒有安慰，反而更去刺激對方。

《演技派》已經播出到第五期，原本節目就打出了「演員片場生存真人秀」的標籤，在現實片場如何生存下來、晉級就成為了難題，沒想到最新一期的《演技派》又玩出了新高度。

參加《演技派》的演員要在「生存」的前提下，一起競爭於正原創劇本《紫禁城裡的小食光》里的人物角色人物，每次於正都會下達相應的人物的創作表演來考驗演員們，之前播出的幾集，已經定出幾個角色人物。

王玉雯、周陸啦、趙天宇等人都已經定了相應的主角角色，而剩餘人還要再次競爭其他主演角色。而這次於正等演技導師更是給出更加高難度的題目。

難題一：生活在戲中

生活的自己與所拍攝的影視劇的角色如何分開，對於很多演員來說是一個挑戰。

就像是吳鎮宇所說，很多好的演員為了所飾演的角色，在戲外的生活中要依然保持劇中的人物性格，或許暴躁或許陰鬱。這樣做讓所飾演的人物角色更加的豐滿，但是或許也會造成別人在生活中對這個人產生極度的反感。

而為了讓這些年輕演員們能真正的能明白這些，吳鎮宇也給他們上了一節「特殊」的課程。

而因為這次的課程，還有女生在現場崩潰。

吳鎮宇讓所有演員們演繹「黑臉的一天」，就是所有人都要展示一個人不開心的時候，這一天是怎麼過的。吳鎮宇本人更是一上來就是「黑臉」，只要有演員笑了，吳鎮宇馬上就會出現。

甚至為了展示自己的「黑臉」，特意讓節目組準備了能摔的碗、盤子。

原本演員們還有點笑臉，但是一下子就被吳鎮宇帶的情緒開始下行，王玉雯和張楠開始「吵架」，張彬彬彈鋼琴劇情被吳鎮宇吐槽。

而只有王鶴潤被吳鎮宇表揚，就是因為她在表演的時候注重了很多的細節。黑臉不開心的一天，連自己手中的快板好像都在和自己作對，線頭打亂了表演的節奏。

所有演員都展示出了壓抑的內心，最後為了結束這堂課，吳鎮宇還是以摔盤子結束。

只是這次盤子沒有摔碎，吳鎮宇用這個例子告訴學生們，當你的壓抑的情緒不在的時候，連摔盤子地聲音都沒有在情緒還在的時候「狠」。

其實吳鎮宇這麼做也只是想讓這些演員知道，情緒的重要性，你的一舉一動都會被影響，就像王鶴潤手中打不好的快板、就像自己手中無法摔壞的盤子。

而生活在戲中這堂課，於正也用了一個特殊的方式。就是「整蠱」明星嘉賓尹正。

於正讓上課的學生在一開始就「演繹」一個和自己完全不同性格的人。話癆的男孩變羞、活潑的鄭湫泓飾演一個高冷的女生，肌肉男朱元冰飾演一個呆呆傻傻的年輕男孩，王玉雯飾演一個溫柔害羞的女生。

四個人一出現就把尹正給搞蒙了。尤其是鄭湫泓飾演的高冷女生一開始就高傲的的揚著頭，連請教的時候，都顯得「不尊重」別人。

當尹正得知真相的時候，也吐槽節目是在「溜」嘉賓，同時也表揚幾個人演技都不錯，連自己都騙了過去。

但演員就是「人生如戲」，在生活中演戲，用經驗詮釋角色。一個好的演員就是能隨時進入角色當中，就如尹正所說「要不然就是我就是角色，要不然角色就是我。」

難題二：迅速減肥

製片人於正為了短劇《紫禁城裡的小食光》繼續選擇角色，這次給演員們又出了一個難題：要在限定時間內創作一個簡單的原創劇本，而且女生還要在限定的時間內增肥，男生要完成增肌的任務。

而這個時間只有兩天，男生增肌可能還算容易些，多加一些鍛煉，上鏡還會看起來更帥氣。而增肥對於任何一個女生來說都是滅絕性打擊，更別說一個女演員了，面對鏡頭哪怕胖一點都會對自己的顏值大打折扣。

但是參與考核的演員們還是在盡自己最大的努力，最後只有鄭湫泓一個女生順利的完成了增肥任務，與搭檔朱元冰獲得了角色。

但其實對於這種迅速減肥、增肥的人物，對於任何人的體質都是有極度消耗的，導師吳鎮宇和張頌文都曾經有過這種經驗，而張頌文曾經在20天內增肥31斤，就是為了角色能詮釋的更好，每天就是吃和睡，但是也就想張頌文所說

但是這就是演員的職責，演繹的是別人，從來都不是自己，這個職業外表看起來光鮮亮麗，可是受的苦誰又知道呢！

但是也就是通過《演技派》好像讓我們對於演員這個角色好像又有了新的認識。而下一期又將迎來新的嘉賓導師——老戲骨李立群，而他對於「演員」與「演技」又會做出什麼新的解釋呢，我們拭目以待！
