防彈少年團將於明年2月回歸，或是入伍前最後一張專輯

2019-12-10 1,590條評論 4,243次閱讀 1999人點贊

12月10日，據韓國媒體報道，防彈少年團確定於2020年2月中旬回歸，目前正在進行新專輯的製作。

在報道出來之前，防彈少年團的成員多次有意無意提到回歸的消息。

朴智旻在4日舉行的MAMA頒獎典禮上發表獲獎感言時表示：”現在我們正在準備下一張專輯，這張專輯可能比大家期待中的還要好。希望大家多多期待”，直接表示現在正在認真準備新專輯。

在美國加利福尼亞州舉行的Variety’s 2019 Hitmakers頒獎典禮，防彈少年團成為首組獲得該頒獎典禮”年度組合”獎項的韓國歌手後，金南俊：”正在進行新專輯製作。希望能儘快給大家聽，請期待2020年的防彈少年團”，提高了人們對新專輯的期待。

防彈少年團此次新專輯很可能是成員們入伍前的最後一張專輯。因為隊內大哥金碩珍1992年出生，最遲也要在明年12月入伍履行兵役義務。

Bighit目前靠防彈少年團支撐，如果防彈少年團全員入伍，公司會減少百分之九十的收入，按公司的角度，不會讓防彈少年團全員入伍，在有成員入伍後，會準備其他成員的solo或小分隊。但全員同時入伍，可以大大縮短空白期。
