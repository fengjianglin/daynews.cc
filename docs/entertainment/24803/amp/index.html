59歲費翔片場“共浴”，身材健碩太性感，網友：收割了兩代少女心

天天要聞
2019-12-21

費翔復出飾演紂王，預告片爆出好身材，引起大家熱議 近日《封神三部曲》放出預告片，在預告片中我們看到了一個非常熟悉的人，他就是費翔。

費翔在《封神三部曲》中飾演男二號殷紂王。預告片一出，費翔就因超棒的身材引起熱議。在預告片中我們可以看出，費翔飾演的殷紂王，臉上有著很多性感的鬍鬚，身材健碩，肌肉飽滿，完全不像要60歲的樣子。

據說為了更好地飾演紂王這個角色，費翔苦練身材。他自己透露”進了劇組，就完全沒有自己的生活了！每天不是拍戲，就是排練或訓練（馬術、武術、塑身）……”

看到如此有魅力的費翔，網友們大佳讚賞並感嘆費翔不愧是媽媽時代唯一的偶像。

也有網友表示，憑費翔的顏值，在哪個時代都會火吧！

說起費翔，我們對他最深刻的印象還是那首曾經火遍大江南北的歌曲《冬天裡的一把火》，當時的費翔一亮相就成為了萬千少女的夢中情人。

不得不說，作為美籍華裔的費翔年輕時真的是非常帥氣有魅力，在華語樂壇中也有著不可替代的重要地位。

當年在國內大火之後，費翔選擇去美國發展，參加美國”百老匯”的演出，所以他在中國娛樂圈中並不活躍，在自己的社交平台上也大多是分享生活和一些感悟。

決定參演《封神三部曲》之後，他的個人社交平台上開始有了更多對於新劇的宣傳以及分享自己在劇組中的一些體會。

除此之外，費翔經常分享自己在國外觀看網球比賽的日常，看得出來費翔是一個網球迷。

而且費翔是一個非常有生活情趣，懂得享受生活，享受閑適的人。他更喜歡每天游泳、看書、聽著海浪睡覺的悠閑生活。

對於《封神三部曲》這部作品，名演員雲集且個個演技在線，開播後大家可以追一波劇啦。
