《一夜新娘》反殺《慶余年》，李現拿下最佳主創、郭麒麟排名第三

2019-12-25 2,802條評論 4,093次閱讀 1885人點贊

「電視勢力榜」第112期發布，本期榜單最值得的關注就是網劇《一夜新娘》，這部劇也成為了本期榜單的大黑馬。

收視率方面，《鱷魚與牙籤鳥》以1.66%的收視率排名第一。第二名是《第二次也很美》，收視率為1.50%，也是收視率榜單上唯一的熱門話題劇。如今的收視率榜單與話題量榜單重合度極小，至於哪張榜單更有說服力一些，可能不同人群會給出不同的答案。依然選擇看電視的人，一定會認為收視率更為權威一些。但對於經常上網的人來說，必然會認定話題量榜單更具有說服力一些。

話題量方面，原本在大家心目中一定會是《慶余年》，但《慶余年》以5.1億的話題量屈居第二位。第一名是網劇《一夜新娘》，話題量為7.3億。估計很多人都沒有聽說過這部劇，查看了一下《一夜新娘》的百科。主要演員有袁昊、易柏辰、趙昭儀、高基才、王珮寒、余凱寧、黃千碩、馬超，可以說沒有一個人是絕對的大牌。倒是導演高林豹的履歷很有料，指導過《畫皮》、《畫壁》、《花千骨》、《無心法師》、《青丘狐傳說》和《重返二十歲》等熱門影視作品。另外《一夜新娘》屬於古裝甜寵劇，眼下這種甜寵劇正流行，或許因此才在話題量上一鳴驚人反殺了《慶余年》。

主創影響力方面，儘管《劍王朝》表現不佳，但李現還是憑藉此劇拿下了最佳主創。但以劇為單位來看的話，最大的贏家依然是《慶余年》。李沁排名第二、郭麒麟排名第三、張若昀排名第五、宋軼排名第十，總共有4人上榜。這其中要重點提一下郭麒麟，因為在《慶余年》中郭麒麟恐怕都排不進男性角色的前五。但即便是這樣，郭麒麟還是獲得了如此之高的排名，說明這個范思轍這個角色塑造的成功。郭麒麟的家庭背景大家都清楚，如果郭麒麟在影視界有了不錯的發展，將來有沒有可能以演員為主業呢？
