李艾為瘦身屁股打石膏，郭麒麟曾一次性減肥70斤，蔡明的話又應驗

2019-12-19 361條評論 1,457次閱讀 519人點贊

很多女藝人都擔心自己結婚生子之後，事業會受到影響。例如謝娜、吳昕等，都曾在不同場合公開表露過類似觀點。而同樣是主持人出身的李艾，在懷孕生子後，經過細心調養，貌似又迎來了自己事業新的春天。

李艾的回歸，莫過於得以主持芒果的經典綜藝之一，《妻子的浪漫旅行》。這個節目陶晶瑩主持過、李靜主持過，如今再次「易主」，不難看出李艾開始在事業上正式發力。而能拿到這樣的資源，除了團隊的功勞，李艾自身的努力也是不可或缺的。

為母不易！李艾最近在個人社交賬號晒圖發文：我的臀，不僅瘦了，要圓乎了。如果不是一直放在家裡，我會拒絕承認9月份的那個石膏和我又關係。雖然是短短的幾句話，但是字裡行間不難體會李艾的付出。為了瘦身選擇在臀部打上石膏！這股勁，也著實夠拼了。

從石膏的細節上看，我們不難體會李艾為何開心到「特別發文」。從6月11日起李艾就嘗試為臀部打上石膏，當時也是體態最「寬」的時候。之後，每隔三個月左右的時候，李艾會換一次石膏。直到第二階段，9月25日到12月15日，李艾的瘦身效果最為明顯。相比剛帶上石膏的時候，說瘦了一半也不為過，難怪李艾的事業狀態現在是越來越好。

長達半年的石膏經歷，不得不讓人佩服李艾的毅力，但側面也反映了一個不爭的事實，什麼呢？就是：身材管理對藝人來說實在太重要。另一個很類似的例子，是發生在郭麒麟身上。

郭麒麟最近因為《慶余年》的熱播，關注度是越來越高了。當然，也許會有人說，從前郭麒麟的關注度也不低呀！確實如此，但從前的關注，本質上看和現在可不一樣。從前郭麒麟是盯著「德雲少班主」的頭銜，不得不受關注；如今，郭麒麟是憑藉自己的努力，在演藝界闖出一片新天地，是值得關注，也值得老郭欣慰。

先不論郭麒麟的演技如何，至少從決定走上演藝之路的那天起，郭麒麟的努力真是肉眼可見的。首先要過的一關就是體重！在最近一期《漫遊記》中，郭麒麟的證件照曝光，包括鍾漢良在內的所有嘉賓，都大為吃驚。而郭麒麟則淡定回復一句：「當時是瘦了70多斤」。

70多斤什麼概念呢？相當於從自己身上剝了一個小孩出來！這樣的改變，也可以說是「脫胎換骨」了。惹得郭麒麟每每過海關都有了障礙，因為，郭麒麟自稱：每次過海關都被要求背出證件上的信息，不然不給過。看看這變化的程度！

不管是郭麒麟還是李艾，都以實際行動證明了：減肥瘦身對藝人來說何其重要。不管是天生的肥胖，還是後期客觀原因造成的孕胖，藝人這個行業，不會聽任何理由。這不禁又應驗了蔡明曾經說的那句話：你要想吃這碗飯，就別吃那晚飯。對於這個原則，多年來蔡明自己是嚴格遵守的。而曾經被大家擔心的倪萍老師，也是在這句話的鼓勵下成功減肥。倪萍老師後來分享了自己瘦身的方式：很殘酷，是餓出來的！可見，「藝人」這碗飯，還真是不好吃呢。

每一行有每一行的不易，在演藝圈，或許真的有人一出道，就能靠顏值和粉絲的加持一路登高，但從後期的結果來看，如果只是靠「皮囊」走天下，往往總是登得越高摔得越慘。所以，要想走得長遠，還是要心懷敬畏之心、穩紮穩打才行。像李艾、郭麒麟、倪萍等，包括蔡明自己，所有敬畏「藝人」這碗飯的人，不管他們事業現在是高低是低，不管是一直輝煌還是逐漸過氣，就憑這份敬業的態度，就很值得尊敬，祝福他們！
