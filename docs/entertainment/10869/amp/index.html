這部武俠片被稱為爛片 我是真的不服

天天要聞
2019-12-10

華語電影圈已經很久沒有武俠片上影了，現在的電影江湖早已不是90年代的江湖，現在是漫威和迪士尼的江湖，舞刀弄劍的大俠們早就被歷史車輪碾過，相忘於江湖。
作為一個武俠片愛好者，我總想起一部電影，它在2010年上映時非常不受待見，沒有得到相應的榮譽不說，被各種噴到自閉，票房慘敗，只收到4800萬，被同期上映的《狄仁傑之通天帝國》（2.9億）按在地上摩擦。寫著吳宇森作品，其實吳宇森只是監製，導演和編劇是台灣導演蘇照彬，也是電影圈一大奇談。

快十年過去了，它可能已經被遺忘了，我還是特別想向喜歡武俠片的朋友安利這部《劍雨》。
《劍雨》講的是因爭奪羅摩遺體引起江湖大亂，故事主線雖然比較老套，不過它的敘述手法有不少創新。這是古龍式的武俠江湖，一個詭異而略有邪氣的探案故事，各色人物粉墨登場，各懷絕技，同時也各懷鬼胎。夫妻可以是仇人，同盟也可以自相殘殺，主人和下屬可能一秒轉換…..《劍雨》里的江湖一點都不高尚，為了爭奪資源，為了獲取經濟利益，為了滿足自己慾望，這是個蠅營狗苟的江湖，最大的反派王學圻扮演的轉輪王幾十載千辛萬苦地想得到「達摩遺體」，只為做一個真正的男人——就像豆瓣上有人調侃：「沒有性生活的苦逼總是會催生邪念，提前步入報復社會的歧途」，你可以吐槽，但是你不得不承認這個故事的邏輯很老派，但也算能夠自洽。

在設計人物上，不是簡單地基於正邪兩立。複雜的利益決定了複雜的人物關係，人物從自身的利益考慮問題，使他們的行為邏輯合理，從而決定了劇情的張力，讓整部電影的可看性提升不少。楊紫瓊和鄭雨盛兩位主角的感情線也不俗套，鄭雨盛也許不知道楊紫瓊的身份，也或許剛開始有一方認為是戲，可是相處過程中情是真的，其他真的假的都不重要了。

作為武俠片，《劍雨》沒有丟掉它的看家本領，就是武打戲。動作場面並非靠電腦特效完成，而是港式武打特效，每個人的武功都設計得非常有個人特點：細雨的四十二路辟水劍法，彩戲師雜耍般的神仙索，雷彬的飛針，既新穎又與角色個性貼近，飄逸，又不失現實感。不同人物，動作風格迥異，即便是一個次要人物，也會以一種特別的方式死去。

這部電影優點與缺點都很明顯，最大的槽點我覺得類似《赤壁》的無厘頭台詞，這或許是吳宇森一直的毛病。還有演員的表現參差不齊，大S幾乎每一次出場，無論什麼場景內容，觀眾們都會想笑，她成功地將一名冷血女殺手演繹成腦殘，很有女諧星的潛力。特別是那句名台詞「他是不行，你是沒有」，反正我記了好多年。

總體來說，《劍雨》是一部中上水平的武俠片，怪只怪它生不逢時。2000年李安拍了《卧虎藏龍》以後，華語電影掀起一波文藝古裝片的跟風潮，張藝謀古裝片三連發：《英雄》（2002）、《十面埋伏》（2004）、《滿城盡帶黃金甲》（2006）2005年陳凱歌的《無極》、2006年馮小剛的《夜宴》等等，名導們相繼進場，拍了無數大製作古裝片，這個十年完全消費了古裝片影迷的耐心和審美，所以徐克轉型玄幻+武俠，而吳宇森和蘇照彬還在老套路上下功夫，不管你做得多精緻，觀眾是真的累了，從大環境來說，對應主流觀眾群的懷舊武俠片迷也畢竟是少數，這個市場沒那麼大。

《劍雨》最衰的地方還在於同檔期的電影都非常能打，除了題材相近的狄仁傑，還有神片《盜夢空間》，張藝謀的山楂樹之戀，在商業上失敗也是合情合理。
可惜了這一腔傳統武俠的情懷啊，只願時間能給《劍雨》一個公正的評價了。
