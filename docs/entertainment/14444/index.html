奧斯卡獲獎卻被罵，一舉一動受爭議，安妮·海瑟薇怎麼得罪了觀眾

2019-12-14 256條評論 2,206次閱讀 206人點贊

近日，安妮·海瑟薇被拍到與丈夫帶著三歲兒子和小Baby現身街頭。這也是自今年7月安妮·海瑟薇宣布懷上二胎後，首次拍到一家四口共同亮相。

提到安妮·海瑟薇，絕大多數人第一時間想到她的處女作電影《公主日記》。彼時剛滿19歲的安妮，2001年首次出演迪士尼出品的《公主日記》便一炮而紅，名聲大噪。

安妮·海瑟薇更憑藉此劇，被《人物》雜誌評為”2001年最佳新星”，可謂出道即巔峰。”米婭公主”也成了安妮·海瑟薇的代名詞，同時這也將她定位在了”偶像演員”的戲路上。

不想尋求轉型的偶像演員，不是一個好演員。深知此道理的安妮·海瑟薇更是如此。

2005年，安妮·海瑟薇出演了李安導演的《斷背山》。在這部男性電影中，安妮的表現同樣不落下風，並自此肅清了偶像形象，逐步向成熟女性角色轉型。

2007年的一部《穿Prada的女魔頭》將安妮的演藝事業推向高潮，這部集時尚與勵志的電影，被奉為”女性必看十部電影之一”。而安妮·海瑟薇在劇中的穿搭更被奉為時尚經典。

而在2013年，剪去一頭長髮的安妮·海瑟薇，以寸頭、暴瘦的形象出演了《悲慘世界》中的”芳汀”一角，並因此獲得”奧斯卡最佳女配角”獎，捧得小金人。

然而奧斯卡帶給安妮·海瑟薇的並不是榮耀，反倒成了美國媒體和民眾嘲諷她的理由。

安妮·海瑟薇在奧斯卡上的獲獎致辭，被批評做作、誇張，推特上19685條留言都是在罵她。

《紐約時報》曾發布一篇文章：你們為什麼喜歡黑安妮·海瑟薇？而美國民眾更是將她評為”最討厭的名人之一”。

無論是她的穿搭、作品，甚至是眼角的魚尾紋，都成為了大家抨擊的話題。

安妮·海瑟薇從”全民捧”到”全民黑”，彷彿只是一夜之間發生的事情。可是究其”被黑”原因，又似乎尋不到具體的理由。就像某位美國主持人說道的：“我都不知道自己為什麼討厭她。”

如果非要說一個原因的話，可能就是因為安妮·海瑟薇“太完美”了。

當一個人完美到無可挑剔的時候，出於嫉妒的心理，大家就開始瘋狂地diss她。大概這就是所謂的”物極必反”。

安妮·海瑟薇的美幾乎無可挑剔，皮膚白皙、明眸皓齒、性感紅唇，凹凸有致的身材又融合了四國混血的特色。一顰一笑、一舉一動，都足夠動人心魄。

而在演藝道路上，即使質疑聲從未間斷，安妮·海瑟薇依舊默默地磨鍊演技。

她既是《穿Prada的女魔頭》中幹練、時尚的實習生助理；也可以一身黑色皮衣，化身酷炫十足的”貓女”；也能是為了角色暴瘦14斤，義無反顧減掉一頭性感捲髮的安妮·海瑟薇。

如果說你以為她只是一隻美麗的”花瓶”的話，那就大錯特錯了。

除了演員，安妮·海瑟薇還是聯合國”親善大使”，致力於促進女性平等，為女權發聲。並在聯合國演講中，霸氣宣言：”我不做公主，要做自己的女王”。

長得美、肯努力，又有實力的安妮·海瑟薇真的”討人厭”嗎？或許只是因為她太完美，所以大家只能憑空去找些痛點，刻意”黑”她。

相信隨著時間的沉澱，終究有一天，安妮·海瑟薇能夠用實力堵上”黑粉”們的嘴！
