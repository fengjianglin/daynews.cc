PRODUCE101涉事公司曝光！星船武林皆在列，堂堂正正變醜聞纏身

​文/雙兒

近日，據韓媒的最新消息，幾家招待PRODUCE節目組的經紀公司被扒了出來，這幾家公司分別是STARSHIP、WOOLLIM以及8D Creative，據悉，這幾家公司為安PD在娛樂場所等地點的消費提供了至少1千萬韓元，其中最高的金額則高達了5千萬韓元。至於這幾家公司旗下的練習生的出道情況有沒有受到此事的影響，現在我們還不得而知。

據警方此前通過對安某的調查，判斷出Produce第3季(Produce48)以及第4季(Produce X101)投票造假的可能性較大，另外警方還掌握了賽季前後竟出現了陪酒招待的情況。

先說第一家涉及此事的公司星船（STARSHIP），這家娛樂公司在2008年成立，而公司旗下的藝人眾多。很多人一看到這個公司的練習生，就覺得他肯定能出道，畢竟星船的練習生從播出開始分量就與他人不同，而練習生的排名突然急速上升，也可想而知這中間的水有多深。

而通過《Produce X 101》出道的練習生薑敏熙擁有著很不錯的臉蛋，身上的少年感很足，他清純的外形給人留下了深刻的印象。他最終是以第10名的成績獲得的出道資格，但是他這一成績如今也是備受質疑，票數造假風波之後，有很多人都認為他第十名的成績是頂替了別人的位置才得以實現的。

宋亨俊其實在首次舞台里的狀態也並不佳，在舞台上沒能發揮出他應有的水平。後來，他憑藉自己出色的舞蹈實力以及良好領導能力給觀眾留下了深刻的印象，原以為他是堂堂正正贏得出道位置，現在此事一出，真相變得撲朔迷離。

而武林（WOOLLIM）這家公司掀起的風波似乎最大，這家公司在該節目里，推送出了許多練習生出道，據網友猜測，也許是公司前幾季的成功嘗到了甜頭，因此他們在第三、四季就迫切地去找安俊英內定選手。據WOOLLIM的一位練習生透露，他在決賽當天就知道自己不是出道人選，因為他早就聽說WOOLLIM只能出一個人，而這個可以出道的練習生應該就是車俊昊。

車俊昊在《PRODUCE X 101》初期，給人的感覺就很「面癱」，因為他在節目里的表情基本沒有什麼變化，所以看起來似乎是機器人一般，他最大的特點就是酷似金明洙，吸引了很多關注度，但是他在節目中並沒有很多鏡頭，但仍然能夠取得相對較高的排名，著實讓人有些詫異。

而8D creative所屬的姜惠元在PD48的順位變化就有一些迷惑了，雖然這個妹妹好看又優秀，不管從哪個角度看都毫不遜色，可在節目裡面，她一直都默默無聞，她似乎只想一門心思地跟小夥伴們在一旁玩耍，而最終成績卻突然up，排名變化如此大，確實讓人意想不到，原以為是她突然抓住了國民製作人的心，不過現在看來可有點玄了。

不管怎樣，公司的暗箱操作卻讓輿論的壓力上升到練習生本人，似乎也有些冤枉，但是現在已經曝光的事實當中，觀眾也很直觀地感受到了各家娛樂公司與安PD之間的暗箱操作有多麼恐怖，充滿著多少不公。大家現在所看到的最終的出道順位，只是各家公司聯合節目組安排的一場欺騙觀眾的大戲，似乎絲毫沒有真實性可言。

雖然這些牽扯進造假風波的練習生也很努力，但是如果用不正當的手段，把別人原本的出道位搶了過來，未免也讓人十分心寒，希望這些涉事公司能主動承擔責任，妥善處理這件事情，給予各方一個合理的交代，這才是一個大企業應有的作為。
