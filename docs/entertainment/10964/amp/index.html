“港片已死”的年代，這部TVB出品給了劇迷意外之喜？

天天要聞
2019-12-09

「港片已死」這句話說了很多年了。上世紀八十年代是港片最輝煌的時期，經典韓劇《請回答1988》中就不止一次出現了《英雄本色》的元素，而當時香港電影對好萊塢的影響也非常大，暴力美學大師吳宇森影響了包括昆汀在內的不少導演。但是進入21世紀之後，各大電影公司比如邵氏、電視台比如TVB熱度紛紛減退，香港電影電視劇的質量也越來越差，很少能見到好作品在國內出現了。不過前不久有部港劇引起了不少電視劇愛好者的關注，那就是TVB的《今宵大廈》。

這部劇是懸疑奇幻題材的，要知道香港神神鬼鬼的傳說很多，前世今生輪迴的故事也很多，比如《胭脂扣》。《今宵大廈》的背景也是一對主人公的前世今生，男主阿蕭一直能夢見一個美艷的旗袍女，但做的都是噩夢。後來阿蕭進入今宵大廈做保安，竟然在樓中遇到了跟旗袍女長得一樣的空姐，叫Alex。而前世他們兩個人在今宵大廈相處的片段總是莫名其妙地出現在阿蕭記憶里。

​《今宵大廈》的主線是兩人前世今生的故事，中間還穿插著各種懸疑的小故事，兩集一個。比如住在今宵大廈704的何太太，每天用嬰兒車推著自己兒子，但從來沒人見過她兒子長什麼樣。何太太在醫院認識了一位姓徐的太太，因為這位徐太太是職業女性，無法兼顧家庭和工作，於是主動提出去徐家給她照顧孩子。

何太太完全登堂入室，把孩子、老公和婆婆都當成自己的照顧，還把徐太太折磨成了產後抑鬱，整天疑神疑鬼。原來，何太太早年經歷非常不幸，兒子出生沒多久就夭折了。徐太太的兒子跟自己兒子是同一天出生，她下意識認為這個孩子就是自己孩子投胎轉世，於是她決定奪走徐太太的家庭。

這個故事最後以徐太太被送入精神病院為結局，這樣的悲劇是香港職業女性群體的一個縮影。香港社會壓力大，一邊是職場上的性別歧視、待遇機會等方面的不公平，另一邊是根深蒂固的傳統女主內思想讓家人產生的不滿，職場女性的壓力可想而知。《今宵大廈》中何太太的故事還只是其中尺度不算大的一個，有不少單元故事都很獵奇，但反映的問題卻很沉重。這樣好的港劇可以說是近年少見，你對它有興趣嗎？
