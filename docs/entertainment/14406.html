網評超好的六部韓國犯罪懸疑電影，噓！別說話，兇手在看

2019-12-14 3,325條評論 4,371次閱讀 1173人點贊

一、《追擊者》（2008）

主演: 河正宇 / 金允石 / 徐英姬 / 金裕貞 / 鄭仁基

豆瓣評分：8.4

故事根據轟動整個韓國歷代最殘忍的殺人魔柳永哲真實個案改編。

男主忠浩在刑警改行後就經營者=著一家按摩院，可最近女郎們頻頻失蹤，在調查最後一位女郎美珍的過程中，他遇到了一個叫英民的男子，於是，一場追擊戰就這樣開始了。這部電影在劇情中沒有拖泥帶水，讓我的神經一直處於緊繃狀態，既殘忍又讓人絕望，最後的最後，還是沒能挽回。

二、《不可饒恕》（2010）

主演：薛景求/韓惠珍/柳承范

豆瓣評分：7.8

在調查一起連環殺人案件時，女兒卻意外失蹤了，為了救出女兒，法醫姜民浩開始與殺人魔爭分奪秒的作戰。雖然這是由金亨俊執導的第一部長篇電影，但卻不顯得青澀，獲得了諸多好評。演員也都是很有實力的老牌演員，結局真的是很變態啊。

三、《暗樹殺人》(2018)

主演: 金允石 / 朱智勛

豆瓣評分：6.8

也是根據真實案件改編的電影。講述了一個變態殺人狂被抓進監獄後，竟然自己坦白了七個罪行，卻是真假參半，企圖減輕罪行，即使證據不足，刑警仍然沒有放棄調查。這是一個罪犯主動出題給警察的故事。整部影片還是很驚喜的，演的真的很好啊。

四、《老男孩》（2003）

主演：崔岷植/劉智泰/姜惠貞

豆瓣評分8.2

影片講述了一個復仇的故事，主人公吳大修在一次醉酒回家的路上突遭綁架，被囚禁在一個私人監牢中長達15年之久，在最終得以重見天日之後，他不顧一切的要找到自己遭到綁架的原因，並展開了瘋狂的復仇計劃。這部電影真的是很絕了，父女之間的亂倫之戀，絕望又瘋狂。戲劇化的結尾引發了很多的爭論。

五、《恐怖直播》（2013）

主演: 河正宇 / 李璟榮

豆瓣評分：8.8

廣播電台的主持人尹英華意外接到威脅電話，想藉此為自己打一個翻身仗，於是，針對恐怖分子的直播連線據此展開，可事情的發展卻遠超他的想像。河正宇一個人就能撐起一部戲，全程都是緊繃的狀態，懸念感隨之而生，真的很刺激啊，真實又恐怖。

六、《殺人回憶》（2003）

主演：宋康昊/金相慶/金羅河/鄭仁善

豆瓣評分：8.8

《殺人回憶》是根據韓國的三大未破解的奇案改編而成的。這部電影講述了韓國的一個小鎮上，出現了一系列的連環殺人案，小鎮警察和從漢城調來的蘇警察共同辦案的故事。電影的節奏把握的非常好，既給了觀眾緩衝期，卻又不失沉重與陰鬱的氣氛，結局給了我們自由想像的空間，並沒有一錘定音。整個電影情節都是錯綜複雜到難以想像。也是難怪網上一致好評啊。
