大張偉和劉迎

大張偉給人的感覺就像是一個長不大的孩子，很多人都以為他還是單身，但是他早就結婚了，對象正是自己的經紀人劉迎。雖說劉迎的外表並不是特別出眾，但是一路走來她幫助和鼓勵了大張偉很多，簡直就是現實生活中的福子郝澤宇。而大張偉也特別好，這麼多年都是零緋聞，給夠了女方安全感

2.黃聖依和楊子

黃聖依大火之後選擇與星爺的公司解約，轉身去了自己老公楊子的公司，後來楊子也成為了她的經紀人。

3.袁成傑和陳芊芊

袁成傑和陳芊芊因話劇《奮鬥》相識，相戀4年後，在13年袁成傑和陳芊芊求婚，在14年就曬出了自己孩子的靚照。

4. 鍾鎮濤和范姜

鍾鎮濤和范姜也算是患難夫妻了，鍾鎮濤因為生意失利而欠下一大筆債，在這人生困難的時候，髮妻決定離開他。後來他就遇到了范姜，她不怕他窮困，她和他一起走過人生最難的時候，好在熬過了那段時光，現在兩個人非常幸福的生活在一起了。

5.古巨基和陳韻晴

94年古巨基發第一張專輯時，陳韻晴就是他的經紀人了，08年的時候古巨基第一次公布了兩人的戀情，受到了大家的祝福，畢竟兩個人一起風風雨雨經歷了那麼多，14年兩個人20年的感情終於修成正果。

雖然有情人終成眷屬，是值得祝福的事情。不過對於粉絲來說，是很心碎吧，自己的愛豆和朝夕相處的經紀人在一起是怎樣的感受呢？
