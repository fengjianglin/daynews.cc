簽約王家衛、合作田壯壯，16歲就演大片，她是被強捧還是真實力？

2019-12-22 1,201條評論 2,176次閱讀 786人點贊

現如今，娛樂圈藝人的更替越來越頻繁，幾乎每隔一段時間便有新人冒頭。

近日，16歲的李宛妲就因為電影《葉問4》備受關注。

​這位出道沒多久的影視新人，儘管在《葉問4》中只是特別出演，但在商業代言方面已經有著相當高的待遇。

不僅先後為國際美妝品牌拍廣告片，還在不斷攻克雜誌封面。

​極具辨識度的長相讓李宛妲獲得了「高級臉」的讚譽，拍起時尚大片好評度高，參演電影認可度挺不錯。

和木村光希某些角度神似的長相，同時又頗為印度風，也為她增添了許多話題度。

​對比李宛妲和木村光希的出道經歷來看，二者在職業發展方面還是有一定相似的。

都以時尚攝影作為起點，一個專註於電影圈，一個則打算走模特路線。

兩人的不同則在於，比起木村光希備受矚目的星二代身份，李宛妲的成長經歷更具傳奇性。

​李宛妲的父親是德國生態學家，母親是中國環保工作者，李宛妲和姐姐從小生長於西雙版納的雨林中，她和姐姐因此被稱之為「雨林精靈」。

​可能是成長環境不同養成的獨特氣質，李宛妲在演繹一些自然元素的時尚作品時格外有靈氣。

​也因為父母工作的浪漫傳奇性，李宛妲很小的時候就拍攝過相關紀錄片。

​因此，拍電影之前，熟悉「李宛妲」這個名字的人其實不少。

等到簽約王家衛的影視公司，出演《葉問4》之後，她便在娛樂圈正式有了名號。

​後續的作品則是第五代導演田壯壯的新作《鳥鳴嚶嚶》，這部電影還在拍攝中，李宛妲是女主角。

​李宛妲出道的時間並不長，短時間內卻迅速在影視領域和時尚領域達到了雙雙爆發的程度。

這種刷屏式的出現無疑讓大眾迅速了解了她，但也不免有「資源咖」的嫌疑。

那麼問題來了，手握眾多籌碼的李宛妲，能否在短時間內迅速走紅，讓她的資源「變現」？

​以現階段女星們的走紅史來看，她們獲得認可的原因有兩種。

一是因廣為人知的「個人標籤」成名，典型代表如「音樂才女」歐陽娜娜和「童年回憶」宋祖兒。

二則是因為足夠出色的影視作品，代表人物是張子楓。

​比起因作品走紅，因「個人標籤」成名的過程似乎要快速一些。

短期的資源堆積之後，通過多方宣傳的方式打造鮮明的個人形象，大眾了解之後，這些新人們便有了一定的熱度。

而如果打算靠作品穩定咖位，不但要有代表性的作品，還需要長時間的堅持。

以廣受認可的演技派小花張子楓為例，她的成名史便是一部部經典影視劇的產出史。

無論是早期的《唐山大地震》，還是成名作《唐人街探案》，以及近期的《寵愛》，她都從質量和持久性上證明了自己作為演員的實力。

​「標籤派」和「作品派」女星的區別在於，前者熱度偏高，後者更受大眾認可。而長期積累之後，「作品派」女星的口碑會越來越好，達到真正意義上的大火。

因此可以看到，儘管00小花之中因為「個人標籤」成名的不少，但大多數人已經轉向「作品派」，開始追求影視作品方面的硬實力。

​而以李宛妲近期的工作安排來看，她本人的發展路線也是如此。

時尚領域的成就只是星途中的調劑品，拍電影才是主要工作。

但簽約王家衛影視公司後的李宛妲，能否拿到諸多優秀電影資源，讓她實打實的向「作品派」發展呢？

​眾所周知，王家衛拍電影的期限特別長。

他的影視公司也遵循了這一習慣，平均每一年半完成一部電影。

公司旗下藝人，鞏俐、梁朝偉、張曼玉等人也都屬於「慢工出細活」的代表。

因此短時間內，李宛妲拿到王家衛的電影資源的幾率不大。

​但只要完美完成已有的影視資源，獲得一定演技口碑，和王家衛自然有了合作的可能。

可以說，李宛妲簽約王家衛，也許不能短期內迅速大火，但卻有著接觸頂級影視資源，讓自己轉為實力派的機會。

​翻看00後小花們近期動態不難發現，當下通過營銷炒作快速建立「個人標籤」的行為已經很難為女星們帶來足夠穩固的地位了，反而會招來非議。

憑藉作品獎項走紅的女星才能達到真正的大火。

​因此，《葉問4》上映後，即便李宛妲的表演獲得觀眾認可，卻也只是她邁向娛樂圈的第一步。

未來如何，取決於後續發展的方向與厚度。
