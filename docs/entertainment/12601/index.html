十年前這部劇美女如雲，沒想到最後這位不起眼的配角卻嫁給了男主

2019-12-12 136條評論 4,994次閱讀 1223人點贊

2009年上映的一部電視劇《鎖清秋》，裡面美女如雲，男主角是現在當紅影星馮紹峰，他在裡面艷福不淺，跟四個女人都有感情瓜葛，沒想到現實生活中最後娶了裡面不起眼的小丫鬟。

第一位要說的美女是我們熟悉的伊能靜，在劇中飾演李月娥，當時的伊能靜狀態還是非常不錯的，演技也在線，給人留下很深刻的印象。

第二位是安以軒，是劇中的女主角，當時正是安義軒非常火的時候，她在劇中飾演生活經歷波折不斷的杜蘭嫣，性格是那種柔柔弱弱的，但是總感覺安義軒的臉太高級，不像會受欺負的，呵呵。

第三位是李菲兒，在劇中飾演蘇心禾，剛開始沒看出來這個是李菲兒，這麼多年菲兒的變化還是挺大的，你們說呢？

第四位是鄧家佳，我們對她應該不會陌生，她是《愛情公寓》里那個討人喜歡的唐悠悠，不過在這部劇里她飾演反派角色況春泥，讓人恨得咬牙。

第五位是李彩樺，這個名字大家可能不太熟悉，但是說過她出演的電視劇《回家的誘惑》大家應該就知道是哪位了，她在劇中飾演風情萬種的大奶奶江雨萍。

第六位是肖雨雨，在劇中飾演陸無雙，我最喜歡的角色，肖雨雨長得很有靈氣，屬於很耐看的那種，但就是不知道這麼多年為什麼一直不火。

最後這位是你們喜愛的趙麗穎，麗穎在裡面的飾演一個小丫鬟，不注意看還真找不到她的影子，她在劇中和馮紹峰沒有對手戲，但是有出現在一個屏幕中，當時的麗穎肯定沒有想到多年後會嫁給劇中的男主，哈哈。
