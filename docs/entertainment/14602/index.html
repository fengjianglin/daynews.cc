電影《呂建江》亮相“美麗中國”微電影盛典

2019-12-13 73條評論 2,225次閱讀 497人點贊

12月12日，「美麗中國」微電影盛典暨第五屆「善美石家莊」微電影大賽在石家莊舉辦，此次盛會增設特別放映單元，電影《呂建江》第一次正式和觀影代表及媒體代表見面。觀影開始前進行了主創見面會，劇組主創人員與現場觀眾進行了交流互動。

電影《呂建江》自籌拍以來就受到了社會各界的廣泛關注和支持，這次以善美為主題的微電影盛典更是特意邀請電影《呂建江》片方出席並設置了特別放映環節，同時為《呂建江》電影頒發了大賽特別放映單元長篇大獎。電影《呂建江》取材於石家莊安建橋綜合警務服務站原主任呂建江的事迹，由著名導演遠澮執導，著名演員董勇、涓子主演。電影於今年4月29日在石家莊開機拍攝，11月份取得公映許可證。

著名演員董勇擔任主演

呂建江的事迹很多，而且都是生活中的瑣事，因此在整個電影的策劃、拍攝、後期製作各個創作階段，電影主創團隊遵循生活化、接地氣的原則，精心提煉呂建江身上的閃光點。製片人孫素坤說：「在創作過程中時刻都被呂建江的精神所感召，因此創作出一部真正體現呂建江精神、體現基層幹警為民服務的好作品成為一種光榮的使命和責任」。整個影片放映過程中，觀影代表們感動不已。觀影代表們紛紛表示，影片拍攝得很成功，把呂建江這個平凡的人物表現得有血有肉。
