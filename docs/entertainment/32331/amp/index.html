​揭秘！一檔明星探案類綜藝，何以連火五季？

天天要聞
2019-12-23

目前綜藝連季並不鮮見，但能夠一直保持高製作水平、高口碑的卻不多，而《明星大偵探》以真人角色扮演來探案抓「兇手」的遊戲邏輯連火五季，不可謂不是綜藝界的一朵「奇葩」。今天，小諜一起來詳細解密下這背後都有哪些技（tao）巧（lu）！

套路一：劇本風格=燒腦+細思極恐+無厘頭

玩過「我是謎」的朋友都知道，這類探案遊戲最重要的要素之一就是劇本，而合理的劇情設置、縝密的判斷邏輯、細緻的案件線索都是構成一個好劇本的必備要素。《明星大偵探》就是一檔「我是謎」遊戲的真人版節目，其劇本可謂精彩絕倫。從第一季第一案的「網紅校花的墜落」到新一季的「天台上的罪惡」，每一案劇本的複雜和燒腦程度都遠遠超乎觀眾們的想像。

其中讓小諜印象最為深刻的就是「恐怖童謠」的劇本，該劇本分上下兩卷，當第一案告破，白郵差被指兇手，所有人都以為這是一樁見財起意的「殺人」案，然而隨著白郵差夜裡被殺，故事線再一次延伸，新玩家薛獵人上線、每個人吃的葯突然變成了同一款、還有孤兒院場地的解鎖……原來劇本中的每一個人都不是真實存在的，而僅僅只是甄公爵因為目睹小夥伴被殺後分裂出的多重人格。所有的「兇殺」源頭指向了人格的覺醒，故事也從一開始的《無人生還》走向了《致命ID》。

除了「恐怖童謠」以外，《明星大偵探》其他案件的劇本都十分高能，不僅劇情反轉頻繁，而且往往需要極大的腦洞才能破解，每一處細節都常常令人細思極恐。換臉、《楚門的世界》式直播……每一個劇本都能在各類外國文學和影視著作中覓得蹤跡，但因為添加了新的創意表達，而讓人驚嘆不已。

不過，作為一檔真人秀綜藝，其娛樂性還是本位，所以劇本核心無論多麼的高能燒腦，都還需要增添一些娛樂元素來調和。就《明星大偵探》而言，其五季劇本都有無厘頭的成分在，而且往往是季季相承的，例如NZND組合的前後故事，還有名場面「認親」。新一季第一案「海上鋼琴師」中，所有的犯罪嫌疑人都是父子，因為一個鼓浪嶼傳說而齊聚在一起，再一次做到了「認親也許會遲到，但永遠不會缺席」。燒腦讓探案劇本保持了嚴謹高能性，滿足偵探迷的需求，而無厘頭的名場面則使得劇本更加幽默，更加適應大眾的娛樂需求。

套路二：嘉賓形象=顛覆+出乎意料+有梗

作為「明偵」的五季老粉，何炅、撒貝南、白敬亭、鬼鬼（吳映潔）、王鷗可以說是小諜的初心所在，但隨著越來越多飛行嘉賓的加入，《明星大偵探》讓我們看到了每個嘉賓不一樣的熒屏形象，其中多有顛覆者。

首先要提到的就是撒貝南撒老師了，當我們對他的印象還停留在《今日說法》節目中嚴肅正經的普法主持人，他卻在「明偵」中金句頻出，上一秒可能還在認真分析案情，下一秒投票就開始一路跑偏，奔著明燈的方向而去，甚至還喜獲「狗頭偵探」「明燈」等諸項稱號，從央視主持人的「神壇」走下，撒貝南在《明星大偵探》中成了公認的「芳心縱火犯」。

又如最近因為《慶余年》小范大人一角而大火的張若昀，在節目中的表現亦可稱作是「高能玩家」，從「玫瑰酒店」的張經理、到「逃出無人島」的張醫生、到「天堂公寓」的張高級、再到「海上鋼琴師」中的張二副，不論是搜證，還是線索推理；不論是拿平民牌，還是握著兇手牌，他都體現出與其「二哈」之名不符的沉著與冷靜，分析起來亦是邏輯嚴謹。

此外喬振宇、劉昊然、白敬亭、楊蓉、魏晨等也一樣有著亮眼的表現。縱觀「明偵」的所有嘉賓，除了形象顛覆外，他們所擁有的一個共同特徵就是有梗。這其中又以魏晨為代表，他在五季《明星大偵探》先後扮演過晨歌手、魏國王、魏來、晨唱跳、魏高管、魏了睡、晨老大等十多個角色，如「夠夠了」「叔不知嬸知道」等晨氏梗信手拈來，被評為「明偵」金梗王。當然，在這樣一群明星玩家的加持下，《明星大偵探》也獲得了最基本的流量保證。

套路三：節目風格=娛樂+隨意切換+有格調

如前所提，《明星大偵探》在節目劇本和嘉賓配置上都獨具特色，但究其根本仍然是一檔娛樂性綜藝，所以在節目風格上仍然保持了較強的娛樂特性，搞笑的段子、誇張的表演、甚至是稀奇古怪的道具都讓人還未沉浸於緊張氛圍就先破功大笑。但是如果僅僅只是逗人一笑，想要連火五季就並不容易了。

「明偵」的成功之處還在於較大程度上做到了寓教於樂，從第一季開始，節目就有意識的針對時下的一些熱點問題予以表態，並將觀點融於每一個故事的劇本之中。校園霸凌、網路暴力、抵制PUA欺詐、人工智慧等熱點話題都在節目中有所反映，例如第一季第六案「2046」就隱含著對人類與人工智慧間的關係討論，何炅在這一案結束時強調，「其實人工智慧的善最重要的還是人的善，你怎麼對待它，它就會怎麼對待你。」正因為沒有局限於用跌宕起伏的劇情、意想不到的環節設置和笑料十足的節目效果來奪人眼球，並且利用正能量敘事來傳達正確的價值觀，「明偵」才得以在一眾綜藝之中獨具格調。

從第一季走到現在的第五季，「明偵」呈現了將近五十個精彩案件，儘管會被人調侃成「甄氏滅門慘案」，但真正吸引人的還是這份節目始終堅守著自身的節目定位，做到了「偵心不改」。對於任何一檔綜藝節目，乃至影視劇作品，自身的定位問題都是關鍵，對於《明星大偵探》來說，它要做的就是用燒腦的劇情「碾壓」觀眾，用搞笑的金句逗笑觀眾，用正確的價值觀引領觀眾。好了，小諜現在要下班去接受「明偵」的智力考驗了，我們下回見！
