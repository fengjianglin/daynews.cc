12月上半月的院線有8大新片！黃渤胡歌新片杠上布拉德皮特和強森

2019-12-03 0條評論 10次閱讀 0人點贊

摘要

很快，2019年的只剩最後一個月不到了。回望即將過去的這一年，國產電影屢破票房紀錄、各類型電影黑馬頻出，讓我們看到了國產電影的巨大潛力。而進口大片也一點沒放鬆，從年初到11月都有不少吸引我們走進影院。

收 藏

很快，2019年的只剩最後一個月不到了。

回望即將過去的這一年，國產電影屢破票房紀錄、各類型電影黑馬頻出，讓我們看到了國產電影的巨大潛力。而進口大片也一點沒放鬆，從年初到11月都有不少吸引我們走進影院。

當然，12月依然還有不少讓我們翹首以盼的年度佳作，院線電影的最後一波發力一點也不敷衍，還是大片續作、知名導演新作和新人導演處女作齊發。

誰將會成為2019年又一匹的黑馬，誰又將在臨近跨年之際再創佳績？

由於12月新片太猛，這次我們分為上半月和下半月來細聊：

《南方車站的聚會》

上映時間：12月6日

主演：胡歌 / 桂綸鎂 / 廖凡 / 萬茜 / 奇道 / 黃覺 / 曾美慧孜

導演：刁亦男

期待值：5星

從今年戛納電影節開始就備受期待的電影，導演刁亦男繼《白日焰火》登頂金熊五年後的新作，胡歌作為電影新人首次主演的電影，桂綸鎂、廖凡《白日焰火》後再次出演，再加上萬茜、黃覺、曾美慧孜、奇道的加盟。

無論從什麼方面來說，《南方車站的聚會》都是十二月必看的電影之一。

從電影類型上看，影片依然是懸疑犯罪題材，而且是典型的黑色類型片。

胡歌飾演的周澤農是偷車團伙中的一個領頭大哥，由於意外殺了警察而被重案隊長（廖凡飾）追捕，深陷絕境。

逃亡路上，桂綸鎂飾的陪泳女、奇道飾的曾經的好友、萬茜飾的五年未見的妻子，各色人等各懷心事，相繼被捲入這場罪與罰的追擊旋渦，這場特別的「聚會」該如何收場？

從曝光的劇照和預告片來看，電影的視聽語言和畫面細節都很考究，甚至很多畫面都有一種頹廢的美感。

而且導演一直以來都很擅長人物的雕琢，隨便一張劇照看都是故事，反正看是看定了，具體評價，還是等看完再聊吧。

《星際探索》

上映時間：12月6日

主演：布拉德·皮特 / 湯米·李·瓊斯 / 魯絲·內伽 / 唐納德·薩瑟蘭

導演：詹姆斯·格雷

期待值：4星

一部從卡司和題材上看都很值得期待的電影，布拉德·皮特和湯米·李·瓊斯主演的科幻題材電影。

不過影片在國外上映，只拿了個「叫好不叫座」的評價，影片曾入圍威尼斯電影節主競賽單元，爛番茄上媒體好評率也高達84%，但觀眾好評僅有40%。

綜合來看，影片所營造的視覺效果上還是很震撼的，但在敘事上卻很容易讓人乏味。

《星際探索》雖然還是有將太空冒險納入影片中，但在本質上還是回歸到情感上，電影的主題最後還是落在了最常出現在電影里的父子重逢。

所以與其說這是科幻電影，更像是披著科幻外殼的「情感片」。

不過其實能在大銀幕上看到布拉德·皮特，對於他的顏粉來說，也就夠了。

《吹哨人》

上映時間：12月6日

主演：湯唯 / 雷佳音

導演：薛曉路

期待值：3星半

原定於中秋檔的電影，改檔後終於有了要上映的意思。

雖然大頭雷佳音搭檔女神湯唯很讓人期待會產生什麼樣的火花，但擅長都市愛情的導演首次涉獵犯罪懸疑題材，還是挺讓人擔心的。

片名「吹哨人」指的是所在企業或組織存在重大問題、嚴重威脅公眾利益時，不惜面臨巨大危險勇敢揭露的人。

影片中最大的矛盾衝突也在於是做個「吹哨人」向社會公布真相？還是視而不見明哲保身？

雖然主題挺有新意的，但海報給人的感覺卻不是那麼好，希望影片上映能給到我們驚喜。

《勇敢者遊戲2：再戰巔峰》

上映時間：12月6日

主演：道恩·強森 / 凱倫·吉蘭 / 傑克·布萊克

導演：傑克·卡斯丹

期待值：4星

和《星際探索》同一天上映的《勇敢者遊戲2：再戰巔峰》是一部典型的合家歡電影。

2年前的《勇敢者遊戲》以低成本高票房的表現讓索尼哥倫比亞公司賺得盆滿缽滿，作為一款動作冒險解密類遊戲，動作戲的看點有了，而主人公的角色錯位也帶來了不少笑點，是一部合格的爆米花電影。

作為續篇，依舊是原班人馬，不過這次有了新的變化，冒險場景的升級加上幾個人的角色又一次的錯位，看來新的笑點肯定是也是要升級了。

預告片里出現的幾個冒險場景，明顯看出第二部真的是有錢了，應該是可以彌補第一部經費不足帶來的遺憾。

而且，此次《勇敢者遊戲2：再戰巔峰》提早北美一周率先登陸內地大銀幕，看來電影這次信心十足啊。

《蘭心大劇院》

上映時間：12月7日

主演：鞏俐 / 趙又廷 / 小田切讓 / 帕斯卡爾·格雷戈里

導演：婁燁

期待值：5星

最後要聊的是早就定檔但一直沒怎麼宣傳，很可能會撤檔的《蘭心大劇院》。

影片定檔於12月7日，主演陣容都是神級，鞏俐、趙又廷、小田切讓、帕斯卡爾•格雷戈里，當然還有婁燁導演。

電影改編自虹影《上海之死》和橫光利一《上海》，是以1941年的上海為背景，著名演員於堇返回上海，表面上是為了出演她的舊愛執導的話劇《禮拜六小說》，但是她真正的目的誰也不知道。

是為了救出她的前夫？為了給盟軍搜集情報？為了給養父工作？還是為了和自己的愛人一起逃離戰爭？

從故事來看，影片可以被歸類到諜戰片的題材里，這是我們沒見過的婁燁，婁燁的藝術性和諜戰片的真實碰撞，無疑也加大了我們對影片的期待。

其實一年內能夠看到兩部婁燁導演的作品，對影迷來說就相當於是狂歡了，什麼也不必多說，先是5星預定！

《若能與你共乘海浪之上》

上映時間：12月7日

配音：片寄涼太 / 川榮李奈 / 松本穗香 / 伊藤健太郎

導演：湯淺政明

期待值：3星半

作為日本動畫鬼才大師的湯淺政明，曾被宮崎駿稱作是「百年一遇的天才動畫師」，他以往的作品留給我們的印象大多是用色誇張強烈，題材暗黑大膽。

可這些在《若能與你工廠海浪之上》里完全看不到，和以往作品不同的是，新作走的是小清新的畫風，線條更為規整，用色也更加溫柔。

這個新的嘗試也出手不凡，電影未公映就已經入圍法國昂西國際動畫電影節長篇單元，還拿下了第22屆上海國際電影節金爵獎。

影片講述的是兩個看似命中注定會在一起的人，轉瞬又歷經生離死別的動人故事。

雖然故事聽起來奇幻且帶著悲傷的味道，但在細節上其實處處都是戀愛的酸臭味，再加上片寄涼太的配音，真的是分分鐘淪陷。

《被光抓走的人》

上映時間：12月13日

主演：黃渤 / 王珞丹 / 譚卓 / 白客 / 黃璐 / 文淇

導演：董潤年

期待值：4星

又是一部都是大咖級演員的電影，影片走的是輕科幻路線，講述的是突然有一天一道巨大的光幕籠罩城市，很多人都失蹤了，而失蹤的人被發現是相愛的人，那麼剩下的人又該怎麼辦？

顯然，影片是用科幻的外殼來探索愛情的真相，通過這道光我們看到了城市裡的人的真實的愛情故事，究竟他們的愛情都藏著怎樣的秘密？

導演董潤年是編劇出身，在這部電影之前，他就是有多部知名作品傍身的金牌編劇，此次帶來的處女作的故事也是源於他的一個腦洞。

所以演員和故事看來是沒問題了，就看這位新人導演這次會交出怎樣的答卷了。

《誤殺》

上映時間：12月13日

主演：肖央 / 譚卓 / 陳冲 / 姜皓文 / 秦沛等

導演：柯汶利 監製：陳思誠

期待值：4星

又是一部新人導演作品，翻拍自印度高分懸疑電影《誤殺瞞天記》，原作本身故事就非常精彩，劇情精妙，從影片1/3開始全程高能，直到結尾。

影片講述的是一位平凡的父親在女兒失手誤殺了對自己圖謀不軌的壞人之後，為了守護家人，如何通過精心的設計，最終逃脫被定罪懲罰的故事。

從劇照來看，影片無論是在演技還是故事，都可圈可點。

懸疑劇翻拍最忌諱的就是懸疑感已經提前知曉，就看這部電影能不能帶給我們點不一樣的驚喜。

總體而言，這8部新片各有各的看點，內地有胡歌鞏俐黃渤湯唯的新片，但基本以文藝片為主，娛樂元素比較強的也就是陳思誠監製的《誤殺》了；和他們對杠的則是，好萊塢布拉德·皮特的科幻片，還有巨石強森的爽片。

最後誰能在口碑和票房上勝出，咱們拭目以待！
