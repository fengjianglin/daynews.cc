前幾日張震在出席活動時暴瘦引髮網友熱議，原因是張震在拍攝新戲，為了角色暴瘦了二十斤，頭髮鬍子也剃了，本來他就很瘦只有130多斤，現在已經瘦成皮包骨了…這才是演員。

歐陽娜娜被拍到和她媽媽一塊現身北京的售樓處準備買房，不出意料的又被嘲了一波。其實，歐陽娜娜選擇在北京買房，也是覺得自己現在的口碑確實不好，之前又因為lc問題鬧了一次風波，所以她們家現在就想通過在北京買房這事兒，把lc的事兒穩住，順便刷一波好感。

《蘭心大劇院》原本的上映時間是12月5日，已經改檔。而這次之所以會改檔是 因為宣發問題，在宣發拼不過其他同檔期的電影，現在片方已經換了新的宣發公司。

袁姍姍作品還是不少的，只不過她現在很多作品都是在地方台播出或者網劇，她跟於正沒啥聯繫了。

關曉彤在團隊的話語權還是挺大的，造型這方面她也特別有主見，不管是選禮服還是拍照時的一些姿勢什麼的，很多都是她自己決定的。但是每次造型或者硬照被罵之後，她又立馬甩鍋給造型師和攝影師。

李宇春的長相符合西方對中國人的審美，所以各個品牌方一直很青睞於她，李宇春在歐美時尚圈一直很吃得開，甚至不需要她帶貨，只要平面走秀就可以。

婁藝瀟私下整容整得太頻繁，導致臉有點變僵，在拍戲的時候很多微表情都沒辦法做出來，導演都快急壞了，可她自己還不以為然，有戲拍就拍，沒戲的話就出門旅行。

李沁和唐藝昕是關係很好的小姐妹，所以張若昀才敢這麼不避嫌曬齣劇中的cp照，李沁倒也不是擔心唐藝昕誤會，其實是怕網友代入感太強，被有心人士帶節奏，引起粉絲互撕。

陳立農今年發了新歌，明年會籌備演唱會，出道時間不久能舉辦個人演唱會是很厲害的了。

朱正廷雖然沒有剛出道時那麼火，但也是很忙的，他的綜藝輪番上陣，新電影也快上了。

張天愛還有兩部大IP存貨沒有播，她今年播出的劇口碑都不咋地，頻繁上熱搜還是因為綜藝cp。

小宋佳以前還帶著角色光環觀眾緣不錯，後來在綜藝里直來直往又搞了這次的言論，也是好牌被自己打爛的典型。

黃渤老師靠著出神入化的綜藝感，成了收視率的保障，在綜藝節目身價里算是數一數二的。
