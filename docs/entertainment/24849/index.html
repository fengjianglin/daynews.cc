TVB這些腦洞大開的懸疑劇，你看過幾部

2019-12-21 1,172條評論 2,602次閱讀 258人點贊

一說到TVB懸疑類的劇集，觀眾首先想到的是《刑事偵緝檔案》《法證先鋒》和《鑒證實錄》等不朽經典。一說劇集的名字，郭可盈、陶大宇、歐陽震華、陳慧珊、林保怡的面孔，就自動在眼前浮現。

並且，眾所周知，《法證先鋒4》已經在拍攝完畢，將在不久的將來和大家見面。

可是，說起TVB懸疑劇，風格和類型簡直百花齊放，可搞笑可治癒可穿越。

大家應該還記得不久前被《金宵大廈》驚艷到的感覺吧。

在《刑事偵緝檔案》之後，TVB於2002年推出了名字僅一字之差的《情事偵緝檔案》，因此有些盜版的印象製品，會為該片印上「刑事偵緝檔案5」的字樣。

《情事偵緝檔案》

既然以「情」字開頭，自然每起案件都是一筆桃花債。

私家偵探何奇通（郭藹明飾）以幫助他人尋找出軌、通姦證據為業，在一次辦案過程中，與警察周耀祖（郭晉安飾）狹路相逢並鬧得不歡而散，後來失業、失戀的周耀祖被警隊辭退，進入何奇通的偵探社，偵破一樁樁峰迴路轉的情感懸案。雖然與情感糾紛有關，卻充滿了本格推理的懸疑和詭計。

兩位主演的表演相得益彰，郭晉安是TVB的懸疑劇大戶，TVB大概看中的就是他這股咄咄逼人的神經質，他的這種風格，一直延續到2014年為他三度榮膺視帝的《忠奸人》以及2016年的壓軸台慶劇《致命復活》。

《情事偵緝檔案》

除了《情事偵緝檔案》以外，TVB另外幾部懸疑劇遺珠如《隔世追兇》《謎情家族》《與敵同行》均為郭晉安主演。

《隔世追兇》里，出生於警察世家的29歲高級督查何天光（郭晉安飾）在年幼時，因為當警察的父親在一次查案中死在一名妓女的床上，這讓他的整個童年都蒙上了恥辱的陰影。

《隔世追兇》

既然是「隔世」，說明劇中的案子都是多年未能偵破的陳年舊案，且經歷了二十年的時間沖刷，案中的關鍵證據早被洗刷得一乾二淨。

多虧了可以溝通時空鴻溝的大哥大，讓何天光在2004年破案理不出頭緒之時，可以通過撥打大哥大的方式求助1984年的父親為他去案發現場搜集線索和證據。

《隔世追兇》無論在案情還是人物塑造上，都用心雕琢。第一起女性分屍案中，兇手每次殺完人，都要為死者悉心打扮，並且砍下死者的大腿包裝在精美的禮盒裡；第二起富商為亡妻翻案，妻子的遺書以及殘敗的玫瑰花園都營造出凄美的氛圍。

出演女一號的陳慧珊，一改往常幹練的白領形象，在劇中的前半部分飾演一位身患殘疾卻樂觀堅強的偵探小說作家高珊，後半段因為大哥大改變歷史的原因，她則從感性的作家搖身一變成了冷血凌厲的黑社會女老大。

陳慧珊在劇中飾演身患殘疾的偵探小說作家。

《隔世追兇》次年的2005年，TVB推出了一部私家偵探題材的懸疑劇《人間蒸發》，男女主角分別是息影多年剛剛復出的苗僑偉，以及當時剛升做女一號而今已經息影多年的吳美珩，該劇的案件涉及範圍比較廣泛，大至連環兇殺，小至尋找失物，都在該劇的範圍內。

吳美珩還一人分飾兩位性格迥異的雙胞胎姐妹，與苗僑偉有一段糾纏不清的情緣，劇中細膩浪漫的氛圍，與《隔世追兇》很有異曲同工之妙。

《人間蒸發》

還有一類懸疑劇，關注的重點不是取證、推理、破案，而是罪案背後，涉案人物的心理動機。

由方中信和徐子珊聯袂主演的《仁心解碼》最早於2009年在馬來西亞首播，直至2011年才在香港首播，並於2013年推出續集，因為首部女主角徐子珊在劇中的角色因病早逝，所以第二部由楊怡和蒙嘉慧接任女主角。

該片里的罪犯均有反社會人格傾向，他們的成長過程中，都曾被嚴重傷害過，所造成的心理瘡疤沒予以及時處理，而造成知名後果。因此在這個系列的劇集里，擔任破案工作的，主要是精神科醫生，而非警察和法醫。

《仁心解碼2》

將心理因素與解謎因素結合在一起的，還有2005年陳錦鴻與滕麗名聯袂主演的《心理心裡有個謎》。但該劇里的謎題，沒有《仁心解碼》那麼尖銳與陰暗，它更像是一部「怪誕心理學」，解釋平常人日常生活中一些讓人無法理解的奇異行為。

陳錦鴻在劇中飾演一名能醫人卻不能自醫的心理治療師，信任科學與數據，而滕麗名則飾演一名半吊子的風水師，因誤讀命理而鬧出一連串笑話，並和陳錦鴻成為一對歡喜冤家。

《心理心裡有個謎》

還有一類懸疑劇，演員表現出位、娛樂效果十足，這一類劇集通常搞笑成分大於推理成分，因此由喜劇演員來擔任絕對主角。

這系列劇集，有黃子華主演的《棟篤神探》、陳百祥主演的《窈窕熟女》、李思捷主演的《八卦神探》、郭晉安主演的《古靈精探》以及汪明荃協同「福祿壽」三人組（「福祿壽」三個字依次代表王祖藍、李思捷、阮兆祥）領銜主演的《荃加福祿壽探案》。

《荃加福祿壽探案》

比如《棟篤神探》的看點，也不在於案子的偵破，幾乎每個案子都只有一個嫌疑人，用大拇指想想都能知道兇手是誰。此劇最大的看點，正是黃子華的脫口秀一般地對案情和人性的抽絲剝繭。

《棟篤神探》最後一幕

首播於2005年的《窈窕熟女》其實是一部情景喜劇，當時已前衛地觸碰了「剩女」這個敏感題材。滕麗名、苑瓊丹、韓君婷和伍詠薇所飾演的四位女主角在劇中是中學同學，因為感情上的困擾而常常進行閨蜜聚會。並且每當劇中出現一個男性角色，不管是老同學、舊上司，哪怕是家裡爆水管，上來了一個水電小哥，都會被四位饑渴的女主角投射愛欲。

盛開的爛桃花，一不小心就會演變成血案，四位女主角還常常陰差陽錯地成為犯罪嫌疑人。

這時，「叻哥」陳百祥飾演的神探畢奇理就會出現破案，而且每次公布真相前，都會像古畑任三郎一樣，對著鏡頭、儀式感十足地宣布偵破的開始。不過老港劇迷們倒是可以找來看看，有些橋段就算現在看來也還是挺搞笑的，而且還能看到一堆如今已經很難看到的熟悉面孔。

《窈窕熟女》

《八卦神探》原本屬意的男主角是歐陽震華，而後歐陽震華檔期不合而換做李思捷頂上，可是案子本身乏善可陳，《荃加福祿壽探案》則是完全以當時憑藉綜藝節目紅遍香江的「福祿壽」組合為中心，加上「阿姐」汪明荃的鼎力加持，可見TVB當時對「福祿壽」的重視程度，而劇中的搞笑方式，也與綜藝節目的搞怪方式如出一轍。

即便新世紀以來，TVB對民心籠絡的程度每況愈下，可是從這些遺珠還是能看出TVB企圖突破、用心創作的精神。隨著TVB與內地視頻網站合作的幾部劇集逐漸上線，儘管毀譽參半，但還是希望TVB在未來的日子裡，獲得更多的資源及資本注入後，能將那些優秀的創意更好地執行並呈現給觀眾。
