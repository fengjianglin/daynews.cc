新天涯古風“四美四丑”紛紛出爐 肖戰、張若昀、李現、上榜

2019-12-18 2,347條評論 3,718次閱讀 72人點贊

古裝劇一直是老王的熱愛，限古令解封過後愛古裝的朋友相信都與老王一樣眼花繚亂。

2019最驚喜的古裝劇莫過於《陳情令》與《東宮》，陳情劇組火遍海內外，演唱會都開到泰國，東宮女孩虐到心肝脾肺疼。

作為八卦雜談起家的天涯論壇在以前是相當的火，近些年雖然被豆瓣討論組搶過風頭，但評選古裝四美這一盛典仍在繼續中。

說到古裝 好的服化道讓人賞心悅目，想回到那個年代沉浸在劇中無法自拔；差一點的服化道演技好 長相俊美也能說得過去。

舊「天涯四美」

鍾漢良-顧惜朝、喬振宇—歐陽明月、嚴屹寬飾演的李建成、霍建華飾演的徐長卿。

這個版本經過過濾到定型 最終達到共識全網認可。

2019新「天涯四美」

肖戰—魏無羨、羅雲熙—潤玉、朱一龍—齊衡、徐凱—墨青

前三位都名副其實，造型完美演技可圈可點最後一位徐凱頗受爭議，雖有《延禧攻略》與《招搖》兩大古風造型做底，但爆出家暴醜聞之後更是人氣下滑。

這一屆的網友非常不好帶，影視劇中不是期盼素素跳誅仙台，就是小楓跳忘川。而現在大家不止是沉迷於帥氣，丑也要大膽說出來。

2019 「古裝四丑」

12月古風強勢月，古風三劍客劇情 預告基本都是蟬聯熱搜榜，就連這「四丑」都容納了三位主演，有點辣眼睛。

張若昀—慶余年、羅晉—《鶴唳華亭》、李現—《劍王朝》、朱亞文—《大明風華》

個人覺得除了朱亞文造型有點像東廠，其他都還說得過去。

蘿蔔青菜各有所愛，只要劇情夠好，演技其他也可以忽略 畢竟顏值+演技都好的也有限。

你們覺得這一屆的四美 四丑評選怎麼樣？
