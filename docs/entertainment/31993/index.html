周杰倫關注吳亦凡另有所圖？網友調侃：肯定是奶茶的魔力

2019-12-25 172條評論 1,141次閱讀 151人點贊

23日，吳亦凡在個人社交平台中曬出了一張照片，身穿長款羽絨服，頭戴鴨舌帽。對於吳亦凡而言，不管是工作還是生活，都會穿著Vans品牌的衣服，透露出濃濃的潮流氣息。

吳亦凡與周杰倫互動

不過，吳亦凡的穿搭並不是亮點，在留言區內，粉絲赫然發現了一個熟悉的ID，沒錯，他就是周杰倫。

周杰倫在吳亦凡帥照下評論”唉喲”，而吳亦凡則回復道”不錯哦”，並且還配上了開心的表情。顯然，兩人有著親密的交集。

看到周杰倫與吳亦凡的互動，粉絲紛紛不淡定了，許多人猜測，兩人或許將會有新合作。熟悉周杰倫的歌迷都知道，他一向愛玩神秘，”哎喲不錯哦”正是周杰倫的口頭禪。或許，兩人即將在音樂上合作，兩人的實力有目共睹，此番合作必然是強強聯合。

(圖片 : 個人微博)吳亦凡

其實，周杰倫與吳亦凡並沒有一起上過綜藝，不過在2017年時，曾參加了LOL六周年活動。面對媒體採訪，吳亦凡表達了對周杰倫的欣賞，坦言周杰倫是自己的偶像，和周杰倫在一起玩遊戲很開心。

不僅僅如此，吳亦凡與周杰倫還參加了歐萊雅集團年會。在音樂上，周杰倫是吳亦凡的前輩，而吳亦凡對周杰倫十分尊敬，私底下也是好朋友。

周杰倫

對於吳亦凡與周杰倫的互動，粉絲卻調侃道，”周杰倫是不是知道吳亦凡會做奶茶？”前段時間，在《心動餐廳》中，吳亦凡化身小廚師，親自製作奶茶，從嫻熟的操作中可以看出，吳亦凡必然是練家子。

想必，周杰倫是盯上了吳亦凡的奶茶，不得不說，奶茶的魔力就是大。
