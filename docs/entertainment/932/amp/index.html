本報訊（記者 楊文傑）都市劇《他其實沒有那麼愛你》正式殺青，並發布一組主演殺青照及殺青特輯。該劇由宋茜、鄭愷、盧靖姍、張佳寧、李純領銜主演，楊祐寧特約出演，王耀慶友情出演。劇中講述四位女性，在她們認為人生最美好和最性感的年齡，在她們認為最華美和最便利的城市上海，繼續尋求自身的成長，尋求愛情和友情、自我和獨立的故事。是《我的前半生》原創班底致力打造的治癒系劇集。

此次發布的殺青劇照及特輯中，四姐妹熱愛生活，有為之努力奮鬥的事業，並在遇到了心愛的他之後開始踏上愛情征程。她們在工作中降龍伏虎，在情路上撲倒再戰，最終明白了「愛人七分足矣，剩下三分，愛自己」。據該劇總製片人、愛奇藝高級總監、奇星戲劇工作室總經理李蒞櫻介紹，「這部劇不僅會給觀眾帶來一場尋找真愛、發現自我之旅，還能打破以往一些影視作品中對女性友誼的認知偏見」。

該劇在人設、故事、台詞上做了精準的目標定位，核心受眾瞄準25-40歲女性人群，「我之前很抗拒說自己在做一部治癒系劇集，因為治癒不是生硬的概念，而是融入在劇情中的觸動和共鳴，隨著製作的推進，我發現自己可能真的在做一部治癒系的作品。如果這部劇明年真的成為話題劇，我會勇敢地說，『我其實做了一部治癒系的戲』。」李蒞櫻說。

在電視劇《他其實沒有那麼愛你》中，宋茜飾演的孫藝荷是一名奇異果TV製片人，她精力無限，還是個女漢子。

鄭愷飾演的於天是一名奇異果TV的攝像師，也是孫藝荷的同事兼死黨，平時經常和她插科打諢。他一直很好地控制自己的人生，堅定不移地行走在自己渴望的道路上，直到遇到真愛發現自己整個世界都變得不受控制了起來。在愛情里他變得更有責任感，也在短時間裡不斷成長。

盧靖姍飾演的Karen是健身房合伙人，也是一名健身教練，同時是個單親媽媽，年輕時早早結婚生子，卻又因前夫的不安定而離婚，從此不想再婚。她能享受戀愛，也敢直面孤獨，遇到愛情時會遵從內心，從來不問愛一個人值不值得，只是單純去愛。

張佳寧飾演的丁丁是一名練習生經紀人，也是一個家境良好的小公主，她有一副萌妹子的長相，看上去沒心沒肺，其實比誰都看得透。她認為每一段感情都有保質期，每一個決定風險自擔，但當她遇到真愛時，也不禁迷茫了起來。

李純飾演的任染是DMA公司副總裁，她在事業上殺伐果斷，是個十足的女強人，但她所有的理智都在遇到愛情時被撕裂。她渴望被愛，卻又極度敏感，直到後來才明白真正的安全感不是來自別人，而是來自自己。
