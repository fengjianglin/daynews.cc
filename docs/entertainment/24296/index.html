哇噢！44歲米拉.喬沃維奇又懷孕啦！她主演的《生化危機》太帥了

2019-12-19 365條評論 4,548次閱讀 3236人點贊

最近米拉.喬沃維奇經常在社交媒體上曬出她懷孕的照片，並說還是個女兒，加上之前已經有了兩個女兒，她這是要把家裡變成女兒國的節奏嗎？44歲做孕婦肯定很辛苦吧？幸虧米拉.喬沃維奇身體素質不一般，要不然普通女人的身體不一定吃得消。

可能大家對米拉.喬沃維奇不是很熟悉，我簡單的介紹下吧，她出生在烏克蘭，母親是俄羅斯人，父親是塞爾維亞人，所以米拉.喬沃維奇是妥妥的純種斯拉夫人。

讓米拉.喬沃維奇出名的電影是主演由法國著名導演呂克.貝松執導的科幻電影《第五元素》，該片中的男主角是當時好萊塢一線男星布魯斯.威利斯；米拉.喬沃維奇和導演呂克.貝松在《第五元素》拍攝完成後就結婚了，但是這段婚姻並不長久，只維持了兩年。

米拉.喬沃維奇還和呂克.貝松合作了歷史電影《聖女貞德》，米拉扮演了法國抗英民族英雄聖女貞德，但是觀眾對這部電影褒貶不一，沒有《第五元素》受歡迎，而且米拉.喬沃維奇被評為最差女演員，這也可能是她和呂克.貝松感情出現裂痕的原因之一吧。

真正讓米拉.喬沃維奇登上事業巔峰的是她重返好萊塢之後，她主演了由日本開發的熱門遊戲《生化危機》改編的同名電影，她在影片中扮演試圖探尋真相的高科技公司的安保主管；她在電影里伸手矯健的打鬥戲讓人印象非常深刻，並且在《生化危機5：懲罰》里和中國女演員李冰冰深度合作，這一中一外、一高一矮的搭配相得益彰，把《生化危機》系列推到了一個嶄新的高度。

《生化危機》拍完最終章後，米拉.喬沃維奇拍電影的節奏明顯放慢了下來，最近3年她僅拍攝了2部電影，值得一提的是今年她主演的新片《地獄男爵:血皇后崛起》上映了，這預示著米拉很可能是《地獄男爵》系列的常駐女主角。

《生化危機》系列沒有了後繼作品確實很可惜，但是《地獄男爵》系列又讓米拉.喬沃維奇有了再次成為一線女明星的希望；希望她們母女平安，保重身體，身體恢復後再給我們觀眾一展身手，把妖魔邪怪通通都收拾乾淨！
