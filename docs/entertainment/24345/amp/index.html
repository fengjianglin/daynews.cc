偶遇王思聰日本吃平價拉麵，戴綠色帽乖巧排隊不見女友

天天要聞
2019-12-18

12月18日，王思聰的最新動態曝光，他被網友偶遇在日本的拉麵館吃拉麵，而且有意思的是，王校長今天還和友人一起戴著同款綠帽出現，兩人乖巧的排隊點單，一點都沒有闊少的氣場。

在此之前，王思聰還被網友拍到一起到日本滑雪的畫面，當天他和女友在一起完全沒有被限制高消費的神情，依然在多個助手的幫助下，輕鬆的前往日本滑雪。

在日本滑完雪之後，自然也是不能錯過日本的美食，所以沒隔兩天，王校長又被網友在拉麵館偶遇了，不過這次他似乎沒有帶女友同行，而是和一名男性友人一起出現。

當天王思聰和友人來到一家拉麵館，從排隊的人數看，應該還是比較火的一家拉麵館了，王校長這次也是非常低調的跟隨人群排起了隊，並沒有使用特權直接享用美食。

而這次王校長戴的帽子就非常有意思了，頭上戴著乖巧的綠帽在默默低頭玩手機，這次恰恰身邊又沒有女性的同伴，瞬間給人被「戴綠帽」的感覺。

王校長這次能夠和友人一起排隊吃的拉麵，其實從價格表上看，也並不是特別昂貴，價格為1000日元一份，其實合到人民幣也就64元，這在王校長的消費歷史裡，算非常廉價的了。

之前還傳出王思聰在酒店點一份58元宮保雞丁的消息，在吃東西這方面，似乎他已經不再高調了。

不過王校長這次低調出現，網友還是有了不少神回復，特別是因為之前限制高消費的消息，成為眾多網友吐槽的梗。

如今王思聰還是和平常一樣，坐飛機旅遊滑雪，一點都沒有被限制的樣子，難道之前的限制令僅僅是說說而已嗎？
