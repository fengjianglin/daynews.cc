12月的電影院，有一部隱藏的“超級英雄”片

天天要聞
2019-12-10

一個肉眼可見的事實是，這兩年現實主義題材的作品越來越受到大眾的歡迎。

《無名之輩》《我不是葯神》的成功都在向市場傳遞這樣一個信息：國內的影迷們需要更多優秀的現實主義題材影片。

《無名之輩》斬獲近8億票房

在這樣的背景下，今天要介紹的這部影片誕生了。

吹哨人(2019)

導演 / 編劇: 薛曉路

主演: 雷佳音 / 湯唯 / 齊溪 / 吳彥姝 / 楊立新 / 王策 / 邢岷山

類型: 劇情 / 懸疑 / 犯罪

製片國家/地區: 中國大陸

上映日期: 2019-12-06(中國大陸)

片長: 139分鐘

又名: The Whistleblower

和先前的現實主義題材略有不同的是，《吹哨人》聚焦的不是底層民眾，不是赤裸裸的社會慘狀。它關注的是今年九月剛出爐的一條法案——吹哨人制度。

「所謂吹哨人制度（也叫吹哨人法案），就是知情人士的爆料制度，而知情人往往又是內部人，作為知情人能夠儘早發現問題，吹響哨聲，大幅度降低監管成本。而對不遵紀守法的企業則可起到堡壘從內部攻破的作用。」

對絕大多數的觀眾來說，這是一個非常陌生的領域。同時，影片也很有野心，在裡面投擲了貪腐、黑幫、能源工程、跨國合作等更大的格局和現實命題。

如何將這麼多宏大的命題糅合進兩個小時的影片里，非常考驗導演及其他主創的能力。

這一次，薛曉路又請來了湯唯主演，之前她們合作過《北京遇上西雅圖》系列。不論是票房還是口碑，表現都算不錯。

不僅如此，薛曉路還請來了焦華靜操刀劇本。焦華靜此前曾經憑藉《烈日灼心》獲得第33屆大眾電影百花獎最佳編劇獎。

早在三年前，她和薛曉路就合作過。兩人作為愛情喜劇電影《北京遇上西雅圖之不二情書》的編劇，一同入圍了第53屆台灣電影金馬獎最佳原著劇本獎。

《北京遇上西雅圖之不二情書》(2016)

依舊是三位女性主創，因為彼此都有過合作的經歷，從影片的流暢程度也能看出來，她們這一次在《吹哨人》的合作中已經非常熟悉了。

提到這部影片的男性主創，就不得不提大家都非常喜歡的「前夫哥」雷佳音。

作為中年竄紅的「流量代表」，雷佳音在電影領域的表現也可圈可點。尤其是在《綉春刀II：修羅戰場》和《黃金大劫案》中，雷佳音都貢獻了不錯的表現。

所以這一次，雷佳音的加盟也是影片的一個看點。

《綉春刀II：修羅戰場》(2017)

話說回《吹哨人》本身，這部電影到底講了一個什麼樣的故事呢？

澳洲華人職員馬珂（雷佳音飾）在接待一個來自公司的訪問時，意外遇見了前女友周雯（湯唯飾），並發生了一夜情。馬珂這邊還沉浸在出軌帶來的愧疚中，那邊周雯的飛機失事。

處理完周雯的後事、解決掉公司的爛攤子之後，剛開始平靜生活的馬珂又添意外。周雯意外「復活」，神秘殺手接踵而至，越來越多的證據顯示公司即將啟動的能源項目有新的危機。

面對這樣的現狀，馬珂和周雯遠走非洲尋找真相。決定以一己之力舉報公司的馬珂最後發現一直跟在身邊的前女友周雯也並不簡單……

乍看《吹哨人》的題材其實非常生猛，不僅有勇氣觸碰「吹哨人」這樣的題材，還在其中注入了犯罪、動作、懸念等生猛的類型。

這個時候，薛曉路作為女性導演的長處就體現出來了。雖然這一次挑戰了製作類型複雜的商業片，但影片完成度依舊很高，還有女性導演的細膩。

薛曉路導演沒有像一般的男性導演一樣生猛地處理，而是選用了她擅長處理的感情方向。

《吹哨人》的懸疑感體現在兩方面，一是劇情層面的，馬珂所在的公司這次能源項目背後到底隱藏著什麼樣的危機？

二就是更深層的，關於情感的懸疑。

《吹哨人》中，雷佳音再次挑戰了一個立體的、有瑕疵的人物。和他過往的那些作品一樣，「馬珂」這個角色並不能用單一的好、壞二元對立來評價。

他不惜冒險也要做「吹哨人」舉報公司違法行為的舉動是正義的，但情感上，作為一個父親、一個丈夫，他在家庭生活中長期缺位，的確算不上一個合格的丈夫和父親。

「馬珂」這個角色理性和感性交織，塑造起來相當有難度，但雷佳音的完成度很高。

依舊和《我的前半生》里的陳俊生一樣，「馬珂」這個角色並不完全正面，但觀眾對雷佳音恨不起來。

觀眾理解他的困難，理解他的種種不容易和這個小人物的搖擺。即使他在澳洲頂級的能源公司工作，深受上級的重視，但他仍然要很努力地在這個環境中掙扎存活。

處境同樣艱難，而且不能用單一的好壞標準來評價的還有湯唯飾演的周雯。

她不僅是闖入馬珂婚姻生活中的人，還身負經濟案件，她是整個故事裡的反派。但她後來還有主動協助馬珂作證舉報公司的舉動，因此周雯的「壞」也不是徹底的。

「周雯」的人物深度也決定了最後她的自我救贖是最徹底的。

一個男人夾在前女友和妻子之間的倫理故事並不少見，但是將它置於能源危機、挽救城市這樣的大背景之下，會產生什麼樣的化學效應？

《吹哨人》的懸疑或者反轉不僅僅體現在具體的劇情上，它隱晦地指向另一個角度：愛情才是我們日常生活中最大的懸疑。

馬珂、周雯和馬珂的妻子，三個人的情感線也是整個影片中的一重懸疑。

馬珂到底更愛誰？他會不會為了這個突然出現的前女友拋棄家庭、拋棄妻子？如果他真的完全對周雯無意，為什麼還要一而再再而三冒著生命風險幫周雯？

在十分緊湊的觀影氛圍中，這些問題一直縈繞在觀眾的心裡。

同時，《吹哨人》藉此也在叩問每一個觀眾，關於你自己，關於自己的愛情。

情感版海報

必須要承認的是，雖然《吹哨人》能在這麼有限的時間內將能源、犯罪、愛情這些宏大複雜的命題糅合，還做到了極高的完成度，它其實還有進步的空間，它可以做得更好。

只是姿勢水平都可以提高，影片當然有可以再精進完善的空間。但影片對「吹哨人」題材的關注和前瞻性都值得一句肯定。

如果在這之前，你對「吹哨人」的了解同樣不多，那麼《吹哨人》這部影片不應該被錯過，值得你進影院一看。
