做過服務員，當過粉刷匠，出道為父親還千萬賭債，流產4次生3胎

2019-12-25 712條評論 2,898次閱讀 1340人點贊

溫小俠吃瓜指南：你蹲著就完事了！

近日，有網友曝出了張柏芝曾經的一張證件照。

照片中的張柏芝甜笑可人，眉眼清秀，網友們紛紛誇讚是羨慕不來的神仙顏值了！

張柏芝如今可是「辣媽」一枚，39歲的她顏值依舊耐打，保養的特別好。

現在有著3個孩子，過著平淡美好的生活，只是感情史卻坎坷不斷。

不少人都感慨，曾經風華絕代的一代女神張柏芝，現在的人生卻只剩下「生孩子」。

那曾想，現在出現在爛片和綜藝里的她，曾經是香港電影中最具靈氣的演員。

時間重回到1980年，那一年張柏芝出生在香港。

父親被香港媒體稱之為「江湖猛人」，她就跟著父親從小在麻油地一片長大。

從小就喜歡照顧人的她，夢想是做一個護士。

但是現實是現實，後來她的父母離婚了，

母親就帶著張柏芝去了澳洲生活。

年紀輕輕的張柏芝，就去餐廳上班，晚上去酒樓做服務員。

後來為了掙到更多錢，她還在KTV做過女招待，甚至還做過粉刷工。

真是心酸坎坷啊。

但是在一次逛街的時候，她被星探發掘，陰差陽錯地踏進了娛樂圈。

拍攝了第一支檸檬茶的廣告。

後來，遇上了一個貴人周星馳，出演了《喜劇之王》這部電影，飾演柳飄飄。

隨即紅遍了大江南北。

不過，張柏芝的家庭一直是拖油瓶一樣的存在。

在這部電影上映之後，父親欠下了一大筆賭債逃走了。

19歲的張柏芝不得不替父還債，沒想到父親又欠下了千萬港幣，跑到了國外。

為了給父親還債，她拚命的想辦法，接了不少戲。

不過也成就了很多的經典。

在《星願》中，一雙靈氣的眼睛再次吸引了無數觀眾對她的喜歡。

還演唱了那首《星語心愿》。

憑藉著此片，獲得了金像獎最佳女主，更是當年的最佳新人。

2003年，23歲的張柏芝獲得了香港金像獎最佳女主。

因為這部片子——《忘不了》。

接著她的星途越來越璀璨，與許多大牌明星都合作過。

不得不說年輕時候的張柏芝是真的非常漂亮，十分有靈氣。

而她的感情卻十分坎坷。

傳過的緋聞男友有黎明、陳曉東、周星馳……

最為人熟知的是與謝霆鋒的感情，兩人6年的婚姻最終還是走向了終點。

2007年她與謝霆鋒生下第一個兒子，

2010年生下第二個兒子。

今年又有了第三個兒子。

不過三胎爸爸的身份一直沒有公布，至今還是個謎。

也能看出張柏芝是一個十足的「兒控」。

曾經，在一個節目中，張柏芝自曝自曝自己曾經流產4次。

甚至還說，自己13歲開始就想結婚了，想當媽媽的那種。

如今，39歲的她，作為單親媽媽，在外人看來亞歷山大的她，卻展現了她超強的一面。

而她之前塑造的那些經典角色，也會成為我們心中的難忘的回憶。

不關注溫小俠，你的娛圈還想爆炸？

筆者力薦：

他因病離世後，葛優為他養子，馮小剛為其還債，張國立接濟他老婆

他演技不輸張嘉譯，卻因得罪宋佳被雪藏10年，後給胡歌當配角爆紅

國家級演員摘2肋骨，20年不吃米不生子豪宅被封，今穿龍袍上央視
