2019年底，良心國產劇推薦：又一次被國劇感染

2019-12-21 197條評論 3,241次閱讀 205人點贊

小冒堅持原創的413天 追劇就是娛樂，保持自己獨立思考，求同存異地歡樂討論，各抒己見雲互動，真的很好。

自從2015年真情實感追《琅琊榜》之後，我今年幾次海外短暫停留，隨身必備就是下載好的口碑國產劇——以慰思鄉之情。（其實是英語用到想哭）

這才開始認真追了既有口碑，又有熱度的國產劇。

2019火熱國產劇盤點：重新燃起追國產劇興趣，充滿期待

我從來不覺得，自己比只看國產劇的人，有優越感。也承認每個國家的影視作品都是好壞兼有，更沒必要故意抹黑國劇，踩高捧低。

電視劇嘛，只要喜歡，看得開心，稀里糊塗地看熱鬧和精細精緻地深入分析，都可以，並不是特別複雜和絕對對立。

只不過，可能我自己定義的好看的標準，更傾向于海外劇。

主要是連續好幾年的比較火的國產劇，幾乎都以我當年刷過的小說IP改編為主，對於已經知道情節的故事，好奇心減少了不少，何況經常覺得選角不符合「原著黨」的審美，就更難提起興趣，追超過20集的電視劇。

這次推薦的三部在熱播新劇，都是古裝劇。

中華上下幾千年，認真拍起來，就服化道具和歷史文化底蘊這一塊，都是吊打。更別說，其實我們有顏值有態度的新人演員，以及演技精湛的老戲骨，真的不輸其他國家。尤其是已經完結的幾部熱播古裝劇，也可以文化輸出了，口碑很好，非常爭氣。

要製作良心的影視劇作品，道理就是這樣，觀眾不是崇洋媚外，只是想看自己更喜歡的作品。不能人為強制大家降低審美標準，你不去生產好的作品，別的國家就會佔領高地，文化輸入。

我原以為今年的國產劇，應該就此收官，沒想到，年底突然發現三部不錯的國產劇，其實已經很火了，順帶一推。

《鶴唳華亭》

1.劇情和人設還是很走心，沒有敷衍，認真講好一個故事的起承轉合，沒有「無緣無故的愛恨情仇」，邏輯可以自洽。

我看官方介紹的時候，非常詳細，但也非常平淡，沒啥特別感覺。反而是認真追了劇，才發現可以看進去。

就比如：

  • 太子有種孤立無援，危機四伏的進退兩難的無奈；一個深受儒家思想熏陶，想要無爭，但不得不為保全自己，保護身邊人而進入權力的鬥爭。

官方介紹：皇太子蕭定權（羅晉飾）少年喪母喪妹不與父親親近，早年的經歷讓他對親情極度渴望，總是想竭力留住身邊僅有的親人。對於父親，蕭定權又敬愛又畏懼，既期望獲得父親的肯定，又害怕面對希望落空時的窘境。而太子的大哥，齊王及其岳父中書令李柏舟屢屢製造禍端，不但迫害了蕭定權身邊僅有的珍視之人，還令蕭定權與父親間的隔閡日漸加深。

——那一段太子因為父親的偏愛，忍無可忍，只能一己之力維護清白的隱忍，真的讓人憋屈。但另一個側面，也許太子也有不懂自己父親的一面。父子之前，先是君臣。尤其是對於儲君，要求更高，不然怎麼能坐得穩江山。但偏偏父子之間，隔著君臣關係，很多話，很多事，自然不能如同尋常人家的父子關係。

相較之下，太子與高居將軍的舅舅之間的親情與信任，性格開朗，正直的的表弟，也足以看出太子對那一份親情的渴望和羨慕。但站在皇帝的角度，對任何臣子的過度信任，都有可能是禍端。他自然希望太子能保持距離。

就是這樣，所有的情節，人物的立場，都能給觀眾很多思考空間，但又不至於太隱秘晦澀。

我非常喜歡這種不言而喻，但需要認真看進去，思考的追劇體驗。

2.反轉不斷，情節豐富。

國產劇我沒有耐心追完的一個原因，也是因為節奏慢，劇集長。我曾經看過一個圈內編劇的採訪，這不是因為編劇寫不出短小精悍的作品，而是劇集多，時長長，才能賣出更好的價格。

所以國產劇，我經常會倍速或快進。

但這一部《鶴唳華亭》，真的很特別。

節奏是真的可以慢下來，事無巨細地演細節，演神態和心理，

但又不顯得拖沓，因為反轉來得很快，一環扣一環。雖然慢，但是真要提前預知劇情走向，還是很需要一定的深入分析才行，有時候真是沒反應過來。

今年給我這種停不下來的反轉驚喜的劇，還是隔壁的《輔佐官》兩季和《囚犯醫生》。國產劇，終於願意在權謀上，劇本上，用心了。可喜可賀。

3.演員配置上：

像王勁松老師，郝蕾這樣的演員做配角，真的眼角眉梢，都是戲。老戲骨們，真是劇拋臉，演什麼像什麼。

就說一點，男主被人吐槽，太軟弱，太愛哭，我個人倒是覺得站在一個有著這樣背景的太子，不會主動迫害和爭取什麼，這必然+要有個撕裂成長的過程。

一個人的本性就是如此，如果不是什麼大的變故，怎麼可能說變就變，立馬黑化？

可能對於追求「短平快」，趕緊逆襲的觀眾來說，太拖沓；我這樣的觀眾，也會覺得合情合理地慢慢變化，是可以接受的。

《慶余年》

我之前寫過氣場碾壓的老戲骨「陳道明，在這部劇里，角色是分量並不算太重的慶帝。但不管是不是主演，出場多久，和誰同台，都能閃閃發光，氣場大開，碾壓眾人。

作為一個架空朝代里的梟雄，即使沒有時時刻刻不威自怒，哪怕衣衫隨意凌亂，作態放鬆隨和，上一秒輕描淡寫嘮家常，雲淡風輕，下一秒立馬切中要害，令對方膽戰心寒，該心狠手辣，絕不手軟；

展示帝王之威，根本不在於狂吼發怒或者施加命令，一個抬眉側目，一個聲量提高，就足以讓臣子們戰戰兢兢。

畢竟是全劇的最運籌帷幄，掌控全劇，深藏不露的人呀。

每個人都被他看得明白透徹，所有人也在他的棋局裡，需要誰攪局就助他一臂之力；面對那些迷糊不清，利欲熏心，不明聖意的人，該敲打點撥，該手段凌厲，乾脆利落毫不猶豫。

他不是第一次飾演帝王，但這一次的慶帝，非常有辨識度，絕對不是雷同角色。相似的扮相，不需要眼線黑化和服化加持，就是另外一種霸氣帝王。

這也符合他心目中對一個合格演員的定位：

這不雷同的帝王角色，足以看出陳道明的用心，作為一個演員的自我修養。

其實也就是一份演員的專業態度和信念感——在名利浮躁之外，仍願意花時間精力，打磨角色，高標準嚴要求，即使不是主演，即使簡單應付也可以交差——

換種說法，這就是種匠人匠心。

雖然這種雜糅了古代和現代的腦洞大開的設定，褒貶不一。我其實很喜歡這種，輕鬆化，又充滿想像力的主角開掛視角，就像我小時候看劇時，總是會做夢，以一種無所不能的狀態，作為故事的主角，維護世界和平和內心秩序。

《一夜新娘》

講述了狂傲的海上霸王秦尚城和將門俠女花溶之間「高糖甜寵」的浪漫愛情，少女花溶因懷有一個「俠女夢」而離家出走，陰差陽錯差點成為「壓島夫人」。歷經一番艱難，花溶終於逃離海島，但早已情根深種的秦尚城不願輕易放棄，就此踏上了「漫漫追妻路」。劇中融合江湖奇聞、先婚後愛等多種元素， 展開一場「天使與惡魔情歸何處」的新古裝甜寵故事。

我能說，雖然就是典型的網劇甜寵套路，但是很甜很有毒嗎？

彷彿當年看到又窮又覺得高性價比的《太子妃升職記》的感覺。

看劇前我沒啥期待值：新人，網劇，不算出名的IP小說，傻白甜的劇情，想不出亮點。

沒想到，一口氣看完不費勁。還挺喜歡這種「小而甜「的簡單粗暴的快樂。

演員們雖然是新人，但撇開誇張人設，表現的還不錯。

男主秦尚城的扮演者，袁昊，一開始並不算戳中我審美的長相，莫名因為還挺穩定的演技和自然的表現，越看越順眼。才是96年的新人，表現在這個劇組還挺突出的。看過幾個直播和採訪cut，莫名覺得也是比較清醒自知有分寸的思想成熟的人，很難得，我是特別容易變身訪談路人粉的。

上一個讓我有種，越看劇越帥的非典型男主長相的男演員，還是隔壁國家因為《金秘書為何這樣》紅紅火火 朴敘俊。別說，兩人從長相類型，到氣質和感覺，還真有點像。

中間還有刻意模仿李佳琦帶貨的片段，也是可可愛愛。

雖然情節會幼稚，有硬傷，但有些堅持表達的點，還是挺正面的。

比如女主咋咋呼呼，但是全程堅持「真善美」的俠義心腸；比如男主雖然霸氣側漏，後期反而變身尊重愛護女主的貼心會撩的暖男；男女主之間的小鬥嘴小矛盾，主要還是在強調，情侶和夫妻之間應該有的理解，尊重，平等。

畢竟大製作名演員和流量們出演的巨制，還有可能非常尷尬，讓人難以繼續；這部劇能讓我歡樂地看進去，就足夠了。

這類劇，不求感受完全一致，大家都圖自己開心就好。簡單一點，喜歡就看，再幼稚不丟人；受不了也別勉強，再出名不看也罷，不要為了和大家有話題勉強自己。

對於每部劇我的評價和感受，包括喜好都很主觀，我分享也是因為，希望更多和我一樣愛好和取向的人，可以欣賞到這些劇，但從來都不想強加自己的觀點給別人。

追劇就是娛樂，也能保持自己的獨立思考，求同存異地歡樂討論，各抒己見雲互動，真的很好。

另外，盡自己綿薄之力，推薦一下，靠導演賣掉北京房子，來拍那些中國手藝的紀錄片。很多很多的民間手藝，可能不曾被人知道，就在這個世界上消失了——最後一個傳人去世。

導演，用自己的理想，讓這些手藝，還能永遠活在紀錄片里。

我沒勇氣，做堅守傳統手藝的匠人，或者這個有情懷的導演，就希望大家都去多看看，這個紀錄片——

《尋找手藝》1和2.

尤其是新上的第二部，太窮了，請不起宣發，也沒有電視台和主流網站購買播放，所以只能在網站bilibili上播放。三個月，悄無聲息，1.7萬觀看量。

至少也用點擊和觀看，來支持一下，這個局促不安但仍然在努力剪輯第三部的導演張景吧。不管是內容還是背後的故事，都能讓人感動到淚流滿面。

娛樂的時代，也不要忘記看看自己的身邊，關心現實和當下。
