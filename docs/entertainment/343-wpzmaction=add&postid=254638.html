《陳情令》又出新綜藝，這哪是“割韭菜”啊，分明是把地都刨了

收 藏

《陳情令》確實很優秀

如果要評選本年度最火爆的電視劇，那就不得不提《陳情令》。

雖然現在離過年還有段時間，可是應該也不會有新劇能在人氣上碾壓它了 。

這部劇不僅帶火了肖戰和王一博兩位主角，讓他們成為娛樂圈最頂級的流量明星。

而且還在國外被評為全球五十大熱播劇，《陳情令》在這份榜單上位列第36。

創造了國內電視劇的新記錄。

劇沒播完，「套路」就開始了

這部電視劇可以走到今天，完全離不開粉絲的支持。

在電視劇熱播的過程中，肖戰和王一博分別飾演的魏無羨和藍忘機就吸粉無數。

或許是因為看到了這其中的商機，劇組跟視頻平台在電視劇還沒播完的情況下就開始”割韭菜”。

本身這部劇在某視頻平台上就需要會員才可以觀看，但是在電視劇即將結束的時候，又搞了個”提前點播”的幺蛾子。

在開通會員的情況下，再花30塊錢就可以提前看到大結局。

很多沉浸在劇中的影迷朋友們，在追劇根本停不下來的情況下，沒辦法紛紛掏腰包買單。

剛看完劇，遊戲就安排上了

而這件事情在網上也引起了廣大網友的不滿，很多人都紛紛吐劇組的”吃相”太難看。

可是一邊吐槽，一邊掏錢倒是”真香”了。

或許是劇組嘗到了甜頭，他們遠遠不滿足只用這樣的方式賺觀眾的錢。

他們還推出了各種跟電視劇掛鉤的周邊手辦，還跟遊戲公司一起開發了一款跟電視劇情差不多的遊戲。

要知道，這部劇是根據小說《魔道祖師》而拍攝的。

但是遊戲宣傳的時候遊戲公司表示會遵循《陳情令》的劇情來製作，想蹭熱度的心思，完全就是擺在檯面上。

遊戲剛安排上，演唱會就來了

除了打造同款遊戲之外，他們還全員開動組織了一次演唱會。

演唱會的名字就叫《陳情令告別演唱會》，看起來有點”劇終人散”的味道。

因此粉絲們為了珍惜一幫主演在一起的美好時光，紛紛慷慨解囊。

這場演唱會上的門票，最高的被炒到十幾萬，除了現場版的要錢，就連在網上看網路直播也是要花錢的。

這一場直播經過有心人的統計，差不多有320萬人在線觀看，網路直播的觀看價格是50元。

僅僅通過網路直播，就收入上億，可見當初是有多少”韭菜”自己送上門去被割掉。

演唱會剛結束，大電影就安排上了

如果說上述的這些活動，因為跟肖戰和王一博有關，粉絲願意為此買單還是情有可原。

可是劇組在越來越多的周邊上，慢慢的出現誠意的下滑。

就好像一直都在宣傳的《陳情令》大電影，宣傳時所說的原班人馬，幾乎就是騙人的。

這部電影里根本就沒有肖戰和王一博的身影，很多網友都在吐槽電影方是虛假宣傳。

一部沒有主演的電影，能叫原班人馬嗎？

除了蹭肖戰和王一搏的熱度，還能有其他的理由可以解釋嗎？

大電影餘音剛落，綜藝節目立馬就跟上了

如果說大電影只是蹭熱度，那麼新上映的《陳情令》綜藝就顯得有些”無恥”了。

這檔綜藝節目，依然是打著肖戰和王一博的名號，雖然不像以前看電視劇那樣需要額外花錢購買，但是也是需要VIP才能觀看。

節目名單上雖然有肖戰和王一博，但是所有的內容跟他們毫無關係。

而且綜藝的內容，其實就是一些採訪剪輯下來的片段。

對於那些肖戰和王一博的CP粉來說，這檔節目簡直不要讓人太失望。

「韭菜」慢慢長，先把地刨了，方便以後繼續種韭菜

不得不說，《陳情令》劇組真的把韭菜擱到了極致。

他們一遍又一遍的收割著劇迷對這部電視劇的好感，他們一次又一次的利用肖戰和王一博的名氣來為自己賺錢。

一部電視劇，可以衍生出周邊手辦，遊戲，演唱會，大電影，綜藝，簡直聞所未聞。

與其說他們是在”割韭菜”，不如說他們把”韭菜地”都刨了好幾遍。

只是讓人費解的是，為什麼還有這麼多觀眾，願意為此買單呢？
