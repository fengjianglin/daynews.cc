「剖析篇」還珠中乾隆和小燕子更有父女感的原因

寫在前面：

最近鴿了很久，除了工作比較忙之外，也是因為我的創作陷入了瓶頸！！之前我說過，寫《情深深雨濛濛》系列完全是因為國慶在家閑的無聊自娛自樂，結果漲了一波粉絲。我嘟嘟嘟嘟寫了一個月，感覺能掰扯的都掰扯完了。而且老有杠精來杠我，清晨起床看到幾個惡評，一天都不太好，特別打消我的積極性。現在就是《情深深雨濛濛》暫時審美疲勞了，嘗試寫一下《還珠格格》系列吧，正好最近又重新看了一遍……

有圖為證！

最近買了一個40寸4:3的幕布，特別適合看老劇！！一瞬間感覺自己回到了小時候，而且在這上頭看《還珠格格》的感覺和在手機上看是完全不一樣的，特別容易被劇情帶進去……

其實我本來就是想試一下效果，結果直接從《還珠》第一部看到了第二部，每天晚上下班準時播放……也是這高清大屏，讓我窺探到了《還珠》不為人知的秘密～

為什麼乾隆和紫薇沒有父女感？

縱觀兩部《還珠》，小燕子對皇阿瑪的膝下承歡貫穿了整個系列。他們的對手戲經常是皇阿瑪很威嚴地坐在那裡，小燕子真·承歡膝下，仰著頭對著皇阿瑪各種撒嬌，掛在嘴邊的口頭禪就是，「皇阿瑪，你對我這麼好！我簡直幸福得要死掉了！！」

這時候紫薇則因為過於端莊而顯得格格不入。第一部還情有可原，畢竟在乾隆眼裡，小燕子是自己的親閨女，第二部還這樣是有點奇怪的。

這也是我更喜歡剖析《情深深雨濛濛》的原因，因為它的劇本更成熟，人物邏輯更完善、更自洽。而《還珠》就會出現一些盲區，不止是乾隆和紫薇沒有父女感，永琪和紫薇也沒有兄妹感啊！永琪全劇幾乎沒有展現過兄長的溫情，尤其是紫薇被小燕子弄丟了那段戲，永琪全程忙著安慰小燕子，甚至責怪爾康，「你怎麼自己不看好紫薇呢？「。

卧槽，紫薇是你親妹妹啊！（雖然那時候紫薇的身世已經被懷疑了，但我想永琪作為主角不會那麼狹隘地聽信一面之詞，而不願意相信紫薇吧。）

之所以永琪和紫薇沒有兄妹感，乾隆和紫薇沒有父女感，純粹是劇本層面的問題。全劇的核心是愛情，永琪這個角色基本是為了小燕子而存在的，所以他和紫薇的親情線是可有可無的。

而乾隆這個角色就是主角團要攻克的一個NPC，第一部的時候需要紫薇用才情和美貌投其所好，嶄露頭角；第二部里紫薇已經認了爹了，她就只剩下替小燕子講道理、圓謊這些比較零散的小任務。

為什麼乾隆和小燕子更有CP感？

小燕子就不一樣了，她不是乾隆的真女兒。所以在第一部里，編劇需要塑造出她和乾隆有很強的父女感，不然無法解釋真相大白後乾隆為什麼還願意將錯就錯，整部劇也會顯得非常low。觀眾看完就會覺得說，哦，《還珠格格》講了一個紫薇和小燕子身份弄錯了，最後乾隆發現紫薇才是格格，對她疼愛有加，然後把撒謊的小燕子趕出皇宮的故事。

這還有什麼看頭！《還珠格格》好看就好看在那些誤會、巧合帶來的戲劇性，還有角色內心的掙扎。

而第二部我覺得純粹是編劇在偷懶，小燕子和乾隆的對手戲基本是照著第一部依樣畫葫蘆。第一部里，小燕子因為乾隆錯把自己當女兒，每次看到乾隆這麼寵愛自己，又是感動又是內疚，所以才會有「快要死掉了」「你一對我好，我就是不知道怎麼辦了「這些非常感性的言論。

可是第二部里，小燕子搖身一變成為乾隆的準兒媳婦了。但是人物關係的變化，居然沒有對人物的行為產生任何影響……在我看到有一段小燕子承歡膝下，還是念著和第一部差不多的台詞時，我第一次沒有入戲（趙薇的表演一直很有感染力），甚至覺得小燕子有點那啥，內心湧現出一句話——會哭的孩子有奶吃。

小燕子啊，認清楚自己的位置好不好啦。你哇啦哇啦撒嬌的時候，人家的親女兒紫薇還杵在那裡耶。我先申明啊（被懟怕了真的），我沒有任何陰謀論，或者覺得小燕子心機的意思！只是吐槽下編劇這裡不應該簡單粗暴地複製粘貼，完全不符合人物身份和行為邏輯，很不合時宜。

偷懶！偷懶！

小燕子對皇阿瑪的態度在第二部完全崩壞！

除了小燕子身為準兒媳，卻對皇阿瑪強行撒嬌（和第一部比動機並不充分），第二部里小燕子對乾隆的態度也變得很「白眼狼」。

好喜歡第一部里的小燕子啊。大結局的時候大家一起逃亡，永琪要帶小燕子遠走高飛，小燕子想都沒想就拒絕了，不是為了永琪，而是為了她的皇阿瑪。只因為她不能帶走皇阿瑪最寵愛的兒子，哪怕這時候她不會拐彎的小腦袋瓜還認為是皇阿瑪毆打了自己，她還是一心一意在為皇阿瑪著想。

鼓掌鼓掌，不愧皇阿瑪一直這麼寵愛她，沒有白疼！

然鵝，故事到了第二部，小燕子前期對皇阿瑪還是像第一部一樣各種撒嬌承歡，動不動就是「皇阿瑪你好偉大！」，後期在逃亡路上卻開始對乾隆各種不尊敬。按她的話說，乾隆要砍她的頭，她氣都氣死了，所以要拐走他最疼愛的皇子，還給他起了瞌睡龍的綽號。永琪不好意思叫，叫了聲「皇龍「，還被她嘲諷了一番。

小燕子啊，你還記得你第一部說的話嗎？真特么翻臉無情。其實兩部里，小燕子都知道自己犯錯了，第一部錯認格格，第二部拐跑妃子，然而第二部卻從來沒有反省過自己的錯誤，只記得乾隆要砍她的頭。人家好端端地為啥要砍你的頭，你咋不想想呢？

最後乾隆跑到了南寧，要小燕子和紫薇跟他回去。最後也是紫薇先點了頭，小燕子才說，紫薇要回去，那我也要回去！

其實我也能明白編劇為什麼把小燕子寫成這樣，就是為了襯托紫薇、永琪和爾康這些智慧型角色唄，和他們有區分度唄。但是這種刻意降低我燕子智商情商，把我們小燕子快塑造成「養不熟的白眼狼」的行為，說實話，我很反感。
