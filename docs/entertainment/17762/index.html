王珞丹黃璐“大打出手”，導演直呼：動作指導都白請了

2019-12-16 1,481條評論 4,653次閱讀 203人點贊

現代快報訊（見習記者 徐可 記者 鄭文靜 文/攝）寧靜的城市，突然閃過一道光。之後，相愛的人都消失了，剩下的人……都已沒有愛情了嗎？這個有點殘忍的腦洞，被拍成了電影《被光抓走的人》。12月15日，該片導演董潤年、主演譚卓、黃璐來到南京。面對電影里那道神秘的光，他們會有什麼樣的反應？

如果有一道光可以測伴侶是否愛你，你願意接受嗎？

《被光抓走的人》中，隨著一道白光閃過，城市裡的一些人消失。接著，科學家宣布，消失的人都是真正相愛的人，而留下的人則沒有愛情。這時，尷尬的一幕出現了，黃渤與譚卓飾演的一對夫妻都沒消失。他們之間還有愛嗎？

△譚卓

譚卓非常篤定地說：「我們之間一定有愛情。戲裡出現了黃覺這麼帥的情敵，我也沒有離開黃渤。」電影中，黃渤和譚卓的故事非常接近日常生活，不狗血，也沒有強烈衝突，但譚卓卻表示，非常喜歡這些戲份。「有沒有那道光我覺得不重要，重要的是，這部電影讓我有了很深的自省和反思，感情對我來說意味著什麼？尤其是一對伴侶彼此意味著什麼？」

主持人向另一位主演黃璐發問，如果這道光可以測試真愛，你願意接受測試嗎？黃璐笑著拒絕。「不用做，內心的感受更重要。」黃璐在這部電影里扮演了一個「悲催」的「小三」，她的情人消失了，顯然，這個男人還有「小四」「小五」……於是，她與王珞丹飾演的「正室」一路「相愛相殺」，踏上尋找「小四」「小五」之路。

△黃璐

首次掌鏡遇上全員「戲精」，導演：他們給我許多安全感

這部電影，講述的是好幾對伴侶或情人之間的故事，所以，演員之間的配合和默契十分重要。

黃渤與譚卓的搭檔，是大家好奇的焦點。譚卓坦言，黃渤是個極具人格魅力的人，無論是他的敬業或是高素養，都讓她感到敬佩。「有一次，這部劇已經拍完了，但是黃渤找到我說，有一場戲希望能夠重拍。後來我們又重拍了這場戲。能夠在工作上遇到這樣的夥伴是非常幸運的。」

黃璐分享了一則跟王珞丹合作時的趣事。「有一場戲我和王珞丹扭打在一起。本來是有一個動作指導在現場的，但真演起來就開始抓頭髮，甚至滾到地上，顧不上那麼多了。」導演董潤年在一旁補刀：「動作指導都白請了！」

黃渤、譚卓、黃璐……這些公認的高口碑「戲精」們的加盟，會不會給首次指導長片的導演壓力呢？其實，他們對董潤年而言，更像是「定海神針」。「不僅不會有壓力，他們的存在反而會更讓我放心，有安全感。」

董潤年雖是初次掌鏡，但之前已是一位頗有名氣的編劇，《心花怒放》《老炮兒》皆出自他之手。對於這次創作，他自然有想法要表達。為什麼他要創造這道白光？「我就想讓觀眾跟隨主人公去尋找真正的自己。現在的時代是紛繁的，浮躁的，已經很少有人跟自己對話，問自己最真實的恐懼和慾望是什麼。其實，只有面對真實的自己，和自己和解，才能夠得到真正幸福和內在的平靜。」
