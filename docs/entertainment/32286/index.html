盤點2019年主流純動作片，全年最刺激感官的都在這了

2019-12-23 2,124條評論 4,543次閱讀 2432人點贊

還有不到10天，我們就要告別21世紀的10年代，踏入曾經感覺離我們很遙遠的2020年，那個經常出現在科幻片里的年代。

到了這個時候，今年的動作類型新片也都基本上完了，目前就只剩備受期待的「一鏡到底」戰爭片《1917》（羅傑·狄金斯掌鏡，北美12月25日上映）了。

年度戰爭大片《1917》海報

在今年上映的這些動作片里，亮點和驚喜並不多，整體質量也遜於去年這8部「今年這8部非院線的R級動作爽片，你可絕對不要錯過了」，畢竟現在要製作一部好的動作片真的不易，不論製作規模大小，製片公司總得在題材和尺度上做權衡，誰都不想做虧本生意，如果還要兼顧創意和驚喜，就更難上加難了。

本文盤點的，都是今年在全球範圍內（覆蓋亞洲和歐美）上映的純動作片，這個「純」，指的是類型上以動作為主，並且其它類型元素的比重較少，所有不要問為什麼沒XXX或XXX，例如漫威、DC這些漫改或IP作品，還有就是爛到家的（比如豆瓣3.8分的《金蟬脫殼3：惡魔車站》），都不在選擇之內。

《金蟬脫殼3：惡魔車站》劇照

而這些影片基本都是偏主流的商業動作片，共13部，9部為R級，6部在內地上映，故事類型和題材比較多元，不全是高口碑作品，但應該能夠滿足不同人的口味。

（以下排名不分先後，按上映時間排序）

極線殺手

導演: 喬納斯·阿克倫德

主演: 麥斯·米科爾森 / 凡妮莎·哈金斯 / 凱瑟琳·溫妮克

類型: 動作

製片國家/地區: 美國 / 德國

上映日期: 2019-01-25(美國)

豆瓣：7.0 IMDb：6.3 爛番茄：20%

今年的第一部黃暴動作大爽片，由Netflix聯手康斯坦丁影業打造，改編自黑馬漫畫《Polar:CameWiththeCold》，講述一個臨退休的世界最強殺手「黑凱撒」，被僱主派人追殺後反殺的故事。影片顯然是Netflix通過大數據分析下的產物，從題材到類型都是《疾速》系列的高仿，只不過引發血案的主因，從一隻狗變成了一份退休金。戛納影帝麥斯·米科爾森的加盟，讓電影增色不少，「拔叔」也沒讓我們失望，酷得沒話說（手控加特林掃射簡直炸裂），槍戰肉搏床戲一個不落，尺度非常可觀。專拍麥當娜、U2、羅比·威廉斯、Lady Gaga等巨星歌手MV的導演，也為影片的影像注入了花俏、動感的漫畫風，色調濃重，反派人物盡顯浮誇、中二的氣質。就結局來看，很可能會有續集。

二鳳

導演: 黎文傑

主演: 吳青芸 / 范英科 / 璇蜜

類型: 劇情 / 動作

製片國家/地區: 越南

上映日期: 2019-09-20(中國大陸) / 2019-02-22(越南)

豆瓣：5.7 IMDb：6.3 爛番茄：95%

Netflix出品，難得一見的越南電影，更意外的是還能在內地上映（票房僅63.6萬）。影片講述從黑道金盆洗手後，二鳳靠著替人收債與女兒相依為命，在女兒被人擄走後，不得不獨自走上尋回女兒的營救之路。飾演女主的吳青芸，有著「越南章子怡」之稱，舞蹈家、歌星、模特和演員的多重身份，在越南是是家喻戶曉的超級巨星，參演過《星戰8》、《光靈》等好萊塢作品。故事類型上，電影很容易被形容為「女版XX」之類，基本靠著一場接一場的動作戲來推進劇情，忽略女主忽高忽低戰力的話，打鬥場面尚可，尺度關係，血漿用得比較節儉，整體不太對得起「票房冠軍」和「首登美國」這樣的宣傳噱頭。

三重威脅之跨國大營救

導演: 傑西·約翰遜

主演: 陳虎 / 盧靖姍 / 托尼·賈 / 伊科·烏艾斯 / 斯科特·阿金斯

類型: 動作

製片國家/地區: 泰國 / 中國大陸 / 中國香港 / 美國

上映日期: 2019-03-29(中國大陸) / 2019-03-19(美國)

豆瓣：4.6 IMDb：5.5 爛番茄：70%

有著陳虎+托尼·賈+伊科·烏艾斯這樣的中泰印打星陣容，可以算是目前亞洲最頂級的配置了。影片講述一個億萬富翁的女兒成為犯罪集團的目標，陳虎三人將聯手對抗並保護她。從去年年底知道這片子，也沒期望能有多好，只求奉獻一場能看爽的動作盛宴就足矣，但最終成片還是讓人不忍吐槽（內地上映僅152.8萬票房），劇情和槍戰場面簡直太弱智，很多支線和人物都是廢的，三大打星中也只有托尼·賈的演技和動作戲在線，陳虎愧對自己「全美空手道大賽60KG級冠軍」的頭銜，動作毫無力量感，不知道爛番茄70%新鮮度怎麼刷出來的。

巴爾幹邊界

導演: 安德烈·沃金

主演: 安東·龐布施尼 / 尤里·庫琴科 / 米洛斯·比柯維奇 / 米萊納拉·德羅維奇

類型: 動作 / 戰爭

製片國家/地區: 俄羅斯 / 塞爾維亞

上映日期: 2019(中國大陸) / 2019-03-21(俄羅斯)

給這張，不信你們不去找來看

豆瓣：7.2 IMDb：6.6 爛番茄：N/A

一部由俄塞聯制的戰爭大片。影片以當年南聯盟被北約轟炸後，被迫妥協撤出科索沃地區，俄軍維和部隊在短暫的權力真空期迅速派兵佔領重要據點-普里什蒂納機場為故事背景，虛構了一支8人小隊與百名「科解」分子進行的機場爭奪戰的故事。如果不熟悉這段歷史背景也不要緊，把它當成一部戰爭爽片看也是及格的，最後一段近40分鐘以8敵百的機場大戰拍得還是挺猛的，尺度可觀看著過癮，對於僅2.3億盧布（約370萬美元）的製作成本來說，場面已經很不錯了。而8人小隊中，由俄羅斯人、韃靼人、印古什人等不同身份成員的組成也很有意思。要說缺點，就是強加的愛情戲和151分鐘片長有點冗長了，不過女主的顏值和身材真的很搶眼。

惡人傳

導演: 李元泰

主演: 馬東錫 / 金武烈 / 金成圭 / 劉承睦

類型: 劇情 / 動作 / 犯罪

製片國家/地區: 韓國

上映日期: 2019-05-15(韓國)

豆瓣：7.7 IMDb：6.9 爛番茄：96%

一部沒什麼野心，只求娛樂性的韓國動作片。黑警和黑幫大佬聯手抓捕連環殺手的設定挺新穎，兩個壞人以暴制暴對付一個更壞的人，一個為業績，一個為報仇，最後也算是個兩贏的局面。根據真實事件改編的故事，影片有意弱化了探案和庭審部分，少了些角色內心戲的描繪，深度不足，但都是為了最大化動作場面的效果，打鬥和飆車戲都算精彩。馬東錫雖然壯實又大隻，外型很適合演黑幫大佬，但在《釜山行》和《與神同行》塑造的角色，總給人他有點暖男、又喜感的形象，這次就算滿身的刺青，還是感覺萌萌的，很討喜。

疾速備戰

導演: 查德·斯塔赫斯基

主演: 基努·里維斯 / 哈莉·貝瑞 / 勞倫斯·菲什伯恩

類型: 動作 / 驚悚 / 犯罪

製片國家/地區: 美國

上映日期: 2019-05-17(美國)

豆瓣：7.8 IMDb：7.5 爛番茄：90%

絕對是19年最受期待的動作片，最終的成片質量也對得起大家的等待。這一部基本把電影的背景、世界觀、John Wick的身世補充完整。當把一個類型做到極致，大概就是這個樣子，所有設計都圍繞著動作場面，肉搏、械鬥、飆車，一場接一場堪稱過癮，殺掉的人數也來到新高的167人。但是，已經55歲的里維斯在動作戲呈現上，速度和力量感確實是肉眼可見的吃力。第3部之後，這個系列可能會先開發一部衍生項目《大陸酒店》，里維斯不會是主演。至於第4部，稍早之前已經宣布檔期，將和他的《黑客帝國4》在2021年5月21日同天上映，著實令人激動，不過還有1年半，也存在變數。

安娜

導演: 呂克·貝松

主演: 薩莎·露絲 / 海倫·米倫 / 盧克·伊萬斯 / 基里安·墨菲

類型: 動作 / 驚悚

製片國家/地區: 法國 / 美國

上映日期: 2019-06-21(美國) / 2019-07-10(法國)

豆瓣：7.7 IMDb：6.6 爛番茄：36%

一直嚷嚷「拍完10部就退休」的呂克·貝松的新作，一部擦邊冷戰題材的R級動作片。影片講述了一個「螳螂捕蟬，黃雀在後」的雙面間諜故事，對於呂貝來說就是老瓶新裝。敘事上，各種玩弄時間線、反轉和解釋的手法，才讓劇情看起來顯得厚實一點。不過，這片子的最大看點，絕對是飾演女主Anna的性感超模薩莎·露絲，顏值和身材自不必說，動作戲也非常有看頭，實在是又美又帥，單論這點，影片給人在感官上是非常刺激的。而作為一個法國人，讓一個雙面女間諜遊走於KGB和CIA兩個男人之間，呂貝這次把俄羅斯和美國黑得夠可以的。（就是80年代末的背景，片中出現各種手機、U盤、wifi和筆電，還能再扯不？）

速度與激情：特別行動

導演: 大衛·雷奇

主演: 道恩·強森 / 傑森·斯坦森 / 伊德里斯·艾爾巴 / 凡妮莎·柯比 / 艾莎·岡薩雷斯

類型: 動作 / 犯罪

製片國家/地區: 美國

上映日期: 2019-08-23(中國大陸) / 2019-08-02(美國)

豆瓣：6.3 IMDb：6.5 爛番茄：67%

「速激」系列的外傳電影，「兩大光頭」聯手的無腦爆米花大片，劇情約等於無。倆光頭全程開啟嘴炮互懟模式，「賤賤」和凱文·哈特的強入也讓影片變得更有趣。但這次在動作戲上，不只是突破極限和想像力，而是已經嚴重超綱，基本是往超英方向發展，畢竟大反派是個經過基因強化的生化武器。女卡司方面，顏值非常可觀，《碟中諜6》中驚艷亮相的「白寡婦」凡妮莎·柯比依舊搶眼，貢獻了不少動作戲，就是新晉「拉丁女神」艾莎·岡薩雷斯戲份略少有些遺憾。與賽車主題漸行漸遠的這部外傳，就只剩強點的family主題了。

天使陷落

導演: 里克·羅曼·沃夫

主演: 傑拉德·巴特勒 / 摩根·弗里曼 / 派珀·佩拉博 / 蘭斯·萊迪克

類型: 動作 / 驚悚 / 犯罪

製片國家/地區: 美國

上映日期: 2019-12-31(中國大陸) / 2019-08-23(美國)

豆瓣：6.1 IMDb：6.4 爛番茄：39%

《奧林匹斯的陷落》和《倫敦陷落》後，傑拉德·巴特勒的「Fallen」系列第3部，依然是拯救美國總統。這個系列雖然一直是中成本的行貨製作，故事只算勉強說得過去，但動作場面卻都維持了不錯的水準。這次即使是系列最低的4000萬美元製作成本，還是把錢用在刀刃上，幾處動作戲比如無人機滅團、老爹轟平山頭和最後的醫院大戰，都拍得乾脆利落的，至少看得過癮。前陣子影片已經定檔12月31日，還沒看的到時可以去影院感受下。

第一滴血5：最後的血

導演: 阿德里安·格魯伯格

主演: 西爾維斯特·史泰龍 / 帕斯·貝加 / 塞爾吉奧·佩里斯-門切塔

類型: 動作 / 驚悚 / 冒險

製片國家/地區: 美國 / 西班牙 / 保加利亞

上映日期: 2019-09-20(美國)

豆瓣：7.5 IMDb：6.3 爛番茄：27%

不出意外的話，跟阿諾·施瓦辛格的《終結者：黑暗命運》一樣，這部也會是史泰龍「第一滴血」系列的最後一部。影片講述Rambo為救回朋友之女，與墨西哥毒梟展開殊死搏鬥的故事。故事基本簡化到成白開水，就是隨便給個理由讓Rambo開殺戒，這也是影片的觀眾評分和專業評分呈兩極化的原因。而這次在尺度的開放程度上，堪稱系列之最，最後硬核、殘暴至極的地道花式殺人，開膛後徒手掏心都非常血腥，不過看著已經73歲的史泰龍飾演的Rambo手刃毒梟仇人，還真的蠻爽的。另外，片尾有一段「犯規」式的前作片段混剪，很是煽情。

雙子殺手

導演: 李安

主演: 威爾·史密斯 / 瑪麗·伊麗莎白·溫斯特德 / 克里夫·歐文 / 本尼迪克特·王

類型: 劇情 / 動作 / 科幻

製片國家/地區: 美國 / 中國大陸

上映日期: 2019-10-18(中國大陸) / 2019-10-11(美國)

豆瓣：7.1 IMDb：5.7 爛番茄：26%

今年最特別的一部動作片，李安的第二部「120幀」作品。如果說《比利·林恩的中場戰事》還有口碑支撐，那這次口碑和票房都雙雙摺戟了。與3年前受限於影院放映設備不同，這次內地已經有20多家影院能放映頂配版本，60幀的也隨處可見，只看60幀的版本，已經能體驗到完全不同於傳統24幀電影的觀感，有著不可比擬的畫面清晰度和順滑度，能在高速運動的動作場面里看到更多的細節，這在片中摩托車追逐、墓穴打鬥和商店機槍掃射幾場戲中都體現得淋漓盡致，而全CG打造的「年輕版史皇」，在大多時候也是真假難辨。可以說，這是部必須在電影院觀看的電影。希望李安下一部「120幀」、關於拳王阿里故事的《馬尼拉之戰》能早日到來。

鬼影特攻：以暴制暴

導演: 邁克爾·貝

主演: 瑞恩·雷諾茲 / 本·哈迪 / 梅拉尼·羅蘭 / 亞德里亞·霍納 / 戴夫·弗蘭科

類型: 動作 / 驚悚

製片國家/地區: 美國

上映日期: 2019-12-13(美國)

豆瓣：6.2 IMDb：6.2 爛番茄：37%

Netflix今年投資最大的兩部電影之一（另一部是1.6億美元的《愛爾蘭人》），《變形金剛》導演邁克爾·貝執導。看完後真是一言難盡，論糟蹋錢，賣拷貝真的無人出其左右，真替Netflix心疼那1.5億美元。明知賣拷貝不會講故事，但那一塌糊塗的剪輯和敘事還是太令人無法忍受，演什麼都像死侍的「賤賤」，嘴炮貫穿全片。動作場面還是爆炸貝那熟悉的配方（快剪+慢鏡），開頭佛羅倫薩飆車看完後可以直接拉到1小時後，最過分的是，一些飆車和碰撞後的音效，明顯是直接拿《變形金剛》的素材來用的，比如汽車人變形的聲音，簡直一模一樣。花同樣的錢，既敢拍這樣的爆米花爛片，也敢拍大概率收不回本的黑幫史詩<愛爾蘭人>，Netflix也真是有錢任性。
