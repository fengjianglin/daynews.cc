35歲星二代稱恨父親入骨！黑歷史被扒他欠父母太多，本人這樣回復

2019-12-19 2,247條評論 4,753次閱讀 1006人點贊

12月17日，台灣媒體報道，知名藝人余祥銓在綜藝節目中疑似意外透露心聲。

​

余祥銓在節目中出演小短劇，在短劇中他所扮演的就是自己，因此開始了一段自嘲之路。

短劇一開始，他就對著電腦發獃，不知道該怎麼寫簡歷，從小就被貼上「星二代」標籤的他，抱怨道「不喜歡我爸是余天的關係，做什麼事情都被無限放大、無限檢視，好難過啊！」

說完這些話後，他下定決心要拋棄「星二代」身份，開始獨立，因為和爸爸同住，所以不願寫地址，因為手機號和爸爸只差一位數，所以也不填，甚至因為跟爸爸是同一所學校畢業，連學歷一欄都空著不寫。

​

不僅如此，余祥銓還向主持人傾訴，稱自己想要整形，「把和爸爸相似的雙眼皮縫成單眼皮」。甚至說，如果要選「靠爸」還是「媽寶」的話，比較不介意當媽寶。

最後寫道生平經歷時，余祥銓思考許久，只寫下了一個「無」字，並且解釋道「過去的點點滴滴都是我爸給的，不想靠爸，所以過去就是空白的」。旁邊的主持人聽完後一臉震驚，甚至不知道該怎麼接這個話。

​

從余祥銓的語言中可以看出，他對於自己「星二代」的身份、對於自己身為余天兒子的身份似乎恨之入骨。

不過18日凌晨，余祥銓也在個人社交平台上發文，稱「這些內容不是我的本意」，在記者諮詢時，他還透露，「錄影前節目組說可以隨意發揮，自己講了又被喊卡，被逼迫按照腳本講。」對此，網友紛紛猜測，他是作秀還是意外吐露心聲不承認。

據了解，余祥銓是台灣著名男歌手、主持人余天與藝人李亞萍之子，從小備受父母疼愛。余天夫妻還曾公開表示已經為子女在巴厘島度假勝地，購買了私人別墅。

​

父母的溺愛讓余祥銓成了一個熱衷於惹事的人。

有媒體曾統計，從2002年至2009年間，余天至少七次為了兒子吸毒、打架等事公開上新聞媒體道歉或解釋，走過六個警察局、一個觀護所。

而余祥銓身為余天之子，從加拿大留學回來後，他想闖蕩娛樂圈，首次登台就被批得體無完膚，還讓他因此患上了憂鬱症及恐慌症等精神疾病。最嚴重時，他不吃不喝、經常失控哭喊，甚至連父母是誰都不認識。

​

為此，余天夫妻把矛頭直指綜藝節目。此事件上遍各大新聞媒體，使余祥銓一時間聲名大噪。

後來的幾年，余祥銓仍然頻頻闖禍，不是與別人發生停車糾紛遭老翁痛毆，就是酒駕遭罰，余天也在不斷地出面道歉、賠償，護著自己的孩子。

​

可以說，余祥銓的前半段人生，黑歷史不斷，而這些其實除了本身的惡劣本性外，還和他父母的縱容有關。

父母的溺愛，讓這個男人產生了無論怎麼樣，都會被原諒的錯覺。很多人也認為余祥銓不管是否真的對父母不滿，他都欠了他們太多。

眼看著他已經35歲了，也不知道會不會幡然醒悟，後悔曾經所做的一切。
