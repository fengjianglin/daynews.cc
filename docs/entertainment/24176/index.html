周星馳當年最受侮辱的兩件事，放咱平常人都受不了

2019-12-19 1,075條評論 2,652次閱讀 700人點贊

周星馳——這位充滿傳奇色彩又伴隨不少爭議的喜劇演員和導演。

相信有不少人看過83版的《射鵰英雄傳》儘管這部電視劇被翻拍了一遍又一遍，但是經典還是以前的。那裡面就有周星馳，而當時的他就是個跑龍套的，演什麼呢？他在裡邊演過精兵，也演過文武的小兵，有那麼一段就是梅超風練九陰白骨爪，要用人腦袋練，結果一伸手就把蒙古的小兵給抓死了。周星馳就是這個蒙古小兵。

本來簡單的一摁，啊的一聲往這一躺就完了。周星馳卻不幹，跑去跟導演說：我覺得我應該有個動作，她一伸出這個手來我應該這麼的擋一下（便說邊做動作）。導演怒吼幹嘛啊，周星馳絲毫不心虛：我給你分析啊，這個小兵他不想死啊！所以梅超風的九陰白骨爪下來後他這麼擋一下，表達他不想死的心情。導演一聽樂了，還真欣賞他的創意，送給他一個字「滾」！電影《喜劇之王》中跑龍套一段便是他的親身經歷寫照。

所以那個時候周星馳飽嘗了這些白眼，不光這個，有些時候他還得跟一幫人一味地拍馬屁，十分難捱。周星馳為了上戲，他托熟人找一部電影的導演，不斷的給導演拍馬屁：我看過你好多片了，什麼什麼的都好到一絕了。其實那導演凈拍爛片，給導演拍得挺舒服，以後有戲就找你！（謝謝導演，謝謝導演，謝謝導演）。

就這樣，千恩萬謝的出了門，出門周星馳沒走站那點根煙的功夫，就聽屋裡的導演說話了，跟身邊的隨從說：剛出去的那個叫什麼？周星馳？怎麼長得像條gou似的。你想誰聽這話放心裡能不被刺傷。

《大話西遊》中有這麼一段，在城牆頂上他和朱茵兩個人擁抱，然後這個時候下邊孫悟空一看，兩人好了。便拖著一個金箍棒走了，這時城牆上的兩人卻說了一句很多反應不過來的台詞：那人看起來像條狗！這其實就是周星馳對自己當年聽到的說這句話的一種讓人心酸的表現。
