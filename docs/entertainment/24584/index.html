《花木蘭》還沒上映火成這樣，但卻令網友"不太滿意"

2019-12-23 45條評論 1,706次閱讀 493人點贊

真人版《花木蘭》的首款預告片發布以後，坊間關於這部作品的討論聲一直就沒消停過，迪士尼大筆一揮斥巨資翻拍出來的”中國經典故事”，一經曝光瞬間點燃互聯網的聲浪。

劉亦菲出演的花木蘭也好，真人版和迪士尼動畫版之間的差距和改動也好，這些變化都成為了咱們中國觀眾茶餘飯後的談資。

畢竟迪佬拍一部中國古代題材的故事，也是對咱中國文化的承認和喜歡，但是看完預告片，很多觀眾對這個電影里出現的很多bug問題都不太理解，編劇和導演到底為什麼要這樣處理？

第一：角色刪改大動刀，觀眾：這樣還有原來那味兒嗎？

《花木蘭》這個故事大家肯定都聽過，木蘭男扮女裝替父從軍十二載，也被譽為咱中國古代孝順的最高境界。

二十年前被迪士尼老大哥買走版權以後變成了一個徹徹底底的神話故事，不僅形象化了李翔這個角色，還特地捏造了一個木須龍來增加花木蘭的神秘感。

但真人版的花木蘭主要受眾肯定是中國的廣大影迷們，這個故事每個人都聽說過，原本一聽迪士尼要翻拍，那肯定還是得興奮搓手暗自期待一番的。

可是電影本身針對動畫版中李翔、木須龍這些主要角色做了改動，製片方想要把這部影片打造成帶有鮮明粗糲感的真人戰爭片，所以自然要削弱一些”公主光環”。

既然有期待就肯定會有比較，原本的動畫電影里這些小小的改變放在動畫片里其實還是比較出彩的，畢竟也是一部動畫作品，以吸引低幼齡的小朋友為主，真的拍成一個中規中矩的傳統故事恐怕也不會大受歡迎。

所以在真人版電影里王子和寵物這樣的設定基本上不需要出現，浪漫元素也會少很多。本來木蘭也是迪士尼公主系列裡的一員，不少觀眾都有點擔心，這部電影拍完以後，咱中國的木蘭到底還能不能被稱之為公主？

第二：中國演員，中國故事，拍戲卻要說英文？

如果說能不能當公主是一件小事？那麼整部電影的語言基調就是一樁大事了，迪士尼啟動這個項目的時候大家就已經看出來了。

這部影片的啟動很顯然，就是沖著中國人來的，故事背景發生在中國，這個故事本身也是中國的經典故事，選用的演員從主角到配角都有中國背景。

劉亦菲，甄子丹這些華人演員自然不必說，就連在節目里戲份沒有那麼多的小配角都是亞裔面孔。因此要說這部作品不是拍給中國人看的，那恐怕誰都不會信。

但是很奇怪的是，明明基本上所有有台詞的演員都會說中文，但是拍戲的時候所有的演員還必須要說英文，這是什麼道理？

雖然電影里的演員們都有英文背景，說起外語也都挺溜的，但是想想就覺得奇怪，中國人拍中國故事卻要說英文，這到底算怎麼回事啊？

我們身為觀眾，有時候也要考慮一下片方的困難，中文這種語言確實是世界上第一難學的語言，雖然眾位主演都可以用中文來演，但是導演和工作人員卻不見得都能聽得懂。

拍的時候當然可以硬撐著用中文來拍，但是後期剪輯的時候，這背後的工作量會因為語言不通翻好幾倍，而且到時候剪出來的效果會沒有保證，很可能還不如英文拍出來的更好一些，所以在語言問題上這樣處理也是沒辦法的辦法啊！

第三：為中國市場考慮，歌舞全刪，真的有必要嗎？

除了這些，《花木蘭》這部電影里還有另外一件讓人不得不在意的事情，那就是製片方刪除了所有歌舞片段。

在電影市場上，除了印度電影喜歡用歌舞劇的形式呈現以外，迪士尼也很喜歡在電影中插入一些歌舞片段，就是一段對白演著演著女主角突然開始唱歌跳舞。歌詞的內容就是台詞的內容，這樣處理既能讓整個故事顯得活潑風趣，又能把觀眾情緒帶動起來，

在很多國外大片中經常會出現類似的橋段。

比如最近大火的《冰雪奇緣》就是一部非常典型的”音樂劇影片”，女主角愛莎在影片中一言不合就開始唱歌。

心情好了要唱歌，心情不好也要唱歌，表達情緒不靠台詞、不靠表情、不靠細節，就靠唱歌。還有之前《美女與野獸》、《衝出康普頓》等大熱的作品裡也有典型的音樂表演。

除此之外，同樣自成體系的還包括由印度出品的寶萊塢歌舞片，這些都是比較有意思的”歌舞代表”。

在華語電影市場上，這種歌舞形式幾乎是從來沒有出現在電影里過，一方面是因為咱們中國文化博大精深，只要編劇把台詞寫得足夠深入人心，完全可以做到不唱歌的煽情。

另一方面，也是因為我們實在是找不到既會演戲、又能唱歌的演員。舉個例子，老戲骨陳道明在演戲的時候突然開始唱歌，這畫面誰能頂得住？

劉亦菲演歌舞，想想就覺得激動，結果刪了！

但也恰恰是因為在華語電影拍不出這種效果來，我們才更格外期待迪士尼翻拍花木蘭的時候保留歌舞部分，在動畫版《花木蘭》中。

包括主題曲《自己》（中文版本由李玟演唱）在內的多首經典歌曲廣為傳唱，如果仙女姐姐劉亦菲在電影里一言不合開始唱歌跳舞演歌劇的畫面，誰不想看？想想就覺得非常期待啊！

但是很可惜這件事情恐怕不會變成現實了，因為人家片方已經公開宣布，基本上不會保留任何歌舞部分，也就是說，人家確實是拍了歌舞的部分，但是在後期剪輯的時候都給剪掉了。

考慮到華語電影市場的觀影喜好，覺得中國的觀眾大概不會喜歡看這種歌舞的劇情，所以乾脆一刀剪掉了。看樣子迪士尼為了中國市場的票房，確實是下了不少小心思，但是說真的，我們可以不要這種特殊照顧的啊！

不過不管怎麼說，《花木蘭》這部全新的迪士尼真人電影，絕大多數中國觀眾還是很期待的，各位小夥伴，不知道大家對這部影片有什麼樣的期待和擔憂呢？快在評論下方寫下你的想法，跟大家一起討論吧！
