收 藏

《金婚》這部電視劇不知道現在年輕人看過沒有？但是父母輩他們應該都看過。

小編對劇中飾演庄玉心的林永健印象深刻，眼睛小小的，也不帥，但是就是給人一種討厭不起來的感覺。其實這就是所謂的耐看吧，他幽默的風格，讓人過目不忘，這應該也是他在娛樂圈的一種特點吧！

記得在《金婚》中庄玉心平時吊兒啷噹，喜歡逗個姑娘、媳婦樂呵樂呵，時不時搭訕一下解解悶，要不就是揪人家一把、拍人家一下，其實他是一個重情重義的好男人。雖然年輕時有些拈花惹草的毛病，但他對庄嫂的愛是最真摯的。

再次關注林永健是在《爸爸去哪兒》中帶著兒子一起去的，讓我們看到了一個生活中的林永健，樂觀、對孩子教育以身作則，是一個慈父也是一個嚴父。

前兩天看了《媽媽像花兒一樣》林永健飾演馬三強，小編感覺這個角色很貼近生活。

現在有些男人有點小成就，就感覺自己了不起，回到家葛優躺，對妻子、孩子呼來喚去的，自己犯錯了回家對妻子還發火動脾氣。馬三強這個人物他就是這樣，內心還有自己的焉主意，但是經過一系列的事情，他蛻變了，內心堅強成熟了，讓自己成為了真正有當擔的大男人。希望大家對號入座，及時「成長」哦！

林永健飾演的電視劇都挺有特點的，長著一雙小眼睛的林永健似乎有一種魔力，無論出演好人，壞人都能讓觀眾印象深刻。

林永健的表演功底很紮實，懂得揚長避短。林永健雖然眼睛小，可是妻子很漂亮哦，妻子是周冬齊也是一位演員，婚後有個兒子林大竣，也很帥氣哦，比爸爸眼睛大。

林永健也是個生活和事業都很低調的藝人，老婆重心在家裡，讓他無後顧之憂，好好奮鬥事業。

你們對這個眼小聚光的林永健大哥還喜歡么？
