圈內最任性男星，不想當主角因為戲份太多耽誤旅遊，難怪一直不火

天天要聞
2019-12-17

娛樂圈這個地方，真的是普通人爭破腦袋也想在這裡面爭出個所以然來。很多進了娛樂圈的明星，都會選擇瘋狂的工作，一心想要在圈內發展成為一線藝人。

特比是圈內的演員們，為了能夠火或者維持熱度，經常全年無休的在劇組拍戲。不過不管是在哪個職業，哪個圈子裡，都會有一些比較佛系的人。他們把名利和金錢看的很淡，在圈內不爭不搶，也不在乎自己是不是紅。

張智堯拍戲太佛系

比如張智堯就是其中一個。

在早年間，張智堯可是古裝美男的天花板啊。他的古裝造型，真的是每一個都驚艷眾人，隨便放一張照片都可以俘獲一大群少女的心。比如《楊門女將》里的楊宗保，《陸小鳳傳奇》裡面的花滿樓。

這些他年輕時接拍的角色，雖然都不是主角，但是卻是大家心裡的經典。因為張智堯的出演，甚至直接讓一個配角的風頭，壓過了主角。

而近兩年的張智堯雖然年紀大了一些，不及當年顏值的巔峰時期了，但是顏值依舊很穩定。比如《古劍奇譚》里的紫胤真人，還有《三生三世》里的折顏上神。

這兩個造型，都是活脫脫的美男子啊。

但是大家仔細一想想，就會發現，張智堯明明經典角色這麼多，但是卻好像沒有幾部做主演的戲？好像在張智堯的演戲生涯里，他大多數都是在當配角，而且一直沒有大火過。

張智堯為旅遊讓編劇把自己寫死？

確實是這樣，因為張智堯真的太佛系了。

之前據說有記者採訪張智堯的時候，張智堯表示說主角的戲份太多了，拍戲要拍很長的時間，他就喜歡配角，因為戲份不多很快就很殺青。

是不是很任性？

別人都爭著搶著想要主角，但是只有張智堯一點也不想要主角。甚至還有網傳說，張智堯在出演《楊家女將》的時候，因為想要去外面旅遊，就讓編劇把自己寫死殺青了。

這個操作真的是讓小編意外至極了。當初不少人看《楊家女將》的時候，還對楊宗保去世這件事耿耿於懷，沒想到居然是因為這種原因。

是不是有點啼笑皆非？

確實，張智堯也太佛系了一點。要是張智堯不這麼淡泊名利，說不定他早就在圈內大火了。畢竟當年正值青春的他，在圈內勢頭真的很大。

這一點，和現在圈內的楊蓉真的很像了。

楊蓉在圈內同樣佛系

和張智堯一樣，楊蓉也是演了好多個出彩的配角角色，但是卻一直沒有大火。主要原因，也是因為楊蓉太懶了。之前參加天天向上的時候，汪涵就爆料說楊蓉在拍戲方面真的很佛系。

有多佛系呢？佛繫到楊蓉在不拍戲的時候，能夠半個月不聯繫自己的經紀人，也不接新工作，整天就在家裡待著，一點也不求上進。因為楊蓉每次拍完一部戲，都要這麼休息很長一段時間。所以楊蓉根本就是繼張智堯之後，第二個”佛系大王”啊。

甚至於正還說過，曾經趙寶剛導演找楊蓉去試鏡，結果楊蓉一看天色太晚了，就直接不去了。

看到是真很懶了。

要是楊蓉不這麼懶，多珍惜一下資源，可能也早就紅了吧？不過咱們也不用替他們惋惜什麼，因為不管是楊蓉還是張智堯，他們都是在按照自己的喜好生活和工作。

雖然在我們看來，他們明明有很好的條件卻沒有大火很可惜。但是也許在張智堯看來，自己這樣的人生很充實，很開心呢？畢竟張智堯從小就旅行全國，旅行應該是他最愛乾的一件事了。

而楊蓉這樣又宅又懶的生活，也正好是她想要的，她過的舒舒服服，這樣不挺好的嗎？

不知道你們怎麼覺得呢？
