3. 可以問一下陳曉到底離婚了嗎？感覺之前好多人再傳他已經離婚了 但是最近風聲小了很多 還有就是陳曉真的會跟wyb合作嗎？感覺他們倆個不同定位吧？？


沒離婚，現在陳曉有的時候去參加活動，陳妍希都會跟著，就是為了破離婚傳聞，而且現在夫婦兩個人在積極備孕二胎中。

不過兩個人之前感情確實出過問題，現在都解決了，還有採訪的時候陳曉很排斥問他的婚姻感情生活，希望大家還是多關注他的作品吧。

《冰雨火》基本上已經定了他倆，我之前也說過，這個劇偏主旋律，是緝毒題材，導演是《破冰行動》的導演，在目前這個寒冬環境下，是很好的餅。

4. 小扒小扒？！我真的超級想知道趙麗穎的下一部電影有沒有戲？！小扒快翻我的牌？！啊啊啊，真的特別想知道？！


電影你說的《援軍》嗎？現在這個環境好多項目都開不了，這個電影目前還沒有消息，演員還沒最終定下來，這個瓜可以先不要吃了。

趙麗穎拍《有翡》預計明年2月份殺青，之後不出意外就是跟鄭曉龍合作了，雙方合作意向還挺明顯的，到目前為止沒什麼變動。

對了最近網傳的趙麗穎和楊冪要合作，這個消息溜粉兒居多，現在是擬邀階段，把她倆拿出來是忽悠投資商掏錢用的。

5. 小扒，肖戰和王一博現在單身嗎？

小扒，肖戰和王一博關係怎麼樣？

小扒，你是不是不喜歡王一博，感覺你回答他的問題都特別簡短


這三個問題都是問肖戰和王一博的就一起回答了。

他倆現在都單身。

說實話，男生的關係其實相對來說單純很多，肖戰和王一博都不是矯情的人，在劇組拍戲的時候兩個人打打鬧鬧，關係還是很好的。

但是拍完戲之後，由於工作比較忙，而且他倆私下裡玩的東西也不一樣，聯繫也就少了，給你打個比方就明白了，想想你關係的比較好的前同事，離職之後，不在一個地方上班，聯繫是不是也慢慢變少了。

我沒有不喜歡王一博，因為覺得他滑滑板很帥，我還專門去買了滑板好吧，雖然沒幾天就閑置了，我回答簡短不是為了迎合酷蓋的氣質嘛。
