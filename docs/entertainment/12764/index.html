蔣依依去中戲食堂吃飯，意外暴露出價格，明星學校就是好！

2019-12-11 1,522條評論 3,592次閱讀 1707人點贊

說到蔣依依，她是很年輕的一位演員。憑藉著自己的演技，在娛樂圈發展的還不錯。別看她年紀小，可已經參演了很多不錯的作品。比如說《旋風少女》，還有今年播出的《帶著爸爸去留學》。她在這些劇中都有參演，而且也給大家留下印象。

在今年蔣依依是非常火的，大家很喜歡她在《帶著爸爸去留學》中飾演的武丹丹。而且今年她還參加了高考，也憑著自己的實力被中央戲劇學院表演系錄取。現在娛樂圈中很少看到蔣依依的活動，應該是在學校里好好學習呢。

不過蔣依依會在微博上，跟大家分享學校的一些事情。最近小編就看到蔣依依去中戲食堂吃飯，對於這個學校的學生，知道中戲食堂是什麼樣。像是我們都沒有去過，看到蔣依依發出來還有點好奇，想看看明星學校食堂是什麼樣子的。

在蔣依依發的視頻中，中戲的食堂真是很好，感覺就像是高檔餐廳一樣，而且吃的東西也都挺好的。倒是她也意外暴露出價格，沒想到價格卻不是很貴，跟我們正常學校一個價。肉菜差不多是4元，素菜是在3元，這是很便宜的價格。

看來中戲食堂表面看著很高大上，價格是很親民，怪不得這麼多人喜歡吃。很多網友得知價格後，都覺得明星學校就是好。環境好教學也好，連食堂的飯菜都怎麼便宜，真的讓人心動。你想要去中戲食堂吃飯，就好好學習，爭取考進中戲。你們有什麼想要說的？
