1998年，洪金寶、成龍、李連杰同時進入好萊塢，但境遇卻截然不同

2019-12-11 1,642條評論 3,255次閱讀 733人點贊

豐滿的體型、靈活的身手，讓洪金寶受到了許多影迷的喜愛；大膽、搏命的動作特技，淋漓盡致的道具應用，也讓成龍獲得了不少觀眾的好評；而飄逸、洒脫的身手，沉穩內斂的氣質，則為李連杰打下了良好的銀幕基礎。

在華語動作影壇之上，洪金寶、成龍、李連杰的動作風格雖然略有不同，但三人的事業發展軌跡卻是驚人的相似，都是「因為做打星成名——轉型導演——成立電影公司——走入好萊塢」。

因為在功夫片中的出色表現，洪金寶、成龍、李連杰在70年代末、80年代初的華語大銀幕上備受關注。而隨著打星之路的穩固，三人也先後坐上了導演的位置，開始對自導自演的電影之路進行嘗試，並留下了《奇謀妙計五福星》、《警察故事》、《中華英雄》這樣的作品。

80年代末、90年代初的「動作港片洪流」席捲國際，而「德寶電影」、「威禾電影」、「正東電影」三家公司的名號，也成為了此時動作港片的品質保證。

90年代末，隨著港片市場的快速衰落，洪金寶、成龍、李連杰三人不約而同地在1998年踏入好萊塢，開始尋求新的發展機遇。在動作港片的銀幕之上，三人雖然平分秋生、互不相讓，但在北美電影市場之上，三人卻面臨了截然不同的境遇。

1998年的好萊塢大銀幕上，成龍應該是三人之中，事業最為順利的一個了。

1995年，「嘉禾」為成龍打造了電影《紅番區》，而好萊塢「新線電影」也為該片擔任了北美髮行商。《紅番區》在北美取得了不錯的票房成績，成龍也憑藉該片成功開啟了自己的好萊塢之路。

《紅番區》之後，「嘉禾」又相繼為成龍拍攝了《警察故事4》、《一個好人》兩部作品，而「新線電影」再度以北美髮行商的身份，為這些作品在北美的宣發保駕護航。

《紅番區》、《警察故事4》、《一個好人》在北美都取得了不錯的票房成績，其中《紅番區》和《警察故事4》還連續兩次獲得了「美國MTV電影獎·最佳打鬥場面」的提名。這樣的榮譽，無疑成為了成龍好萊塢打星之路上的堅實保證。

1998年，成龍在完成了《我是誰》的拍攝之後，正式赴美髮展，並在「新線電影」的邀請下，出演了《尖峰時刻》。

之前的《紅番區》、《警察故事4》、《一個好人》，已經在北美動作喜劇市場上，為成龍積累了良好的觀眾基礎。而《尖峰時刻》里出色的喜劇、動作表現，也讓他受到了更多北美觀眾的喜愛。

《尖峰時刻》的首戰告捷，迅速穩定了成龍在好萊塢的地位，隨後的《尖峰時刻2》、《尖峰時刻3》、《上海正午》、《英倫對決》，也讓成龍的好萊塢打星之路，走上了新的高峰，並讓他成為了首位獲得「奧斯卡終身成就獎」的華人動作演員。

一樣是1998年走入好萊塢大銀幕，李連杰在北美的境遇，較之成龍就遜色了許多。

1996年，李連杰因為出演《黑俠》，受到了不少北美片商的注意。1998年，好萊塢「華納影業」拿到了《黑俠》在北美的發行權。同時，李連杰也受到了「華納影業」邀請，出演了電影《致命武器4》。

在這部1998年的《致命武器4》里，李連杰沒有得到出演主角的機會，只是在片中擔任了大反派。而《致命武器4》上映後慘淡的票房成績，也沒能為李連杰的好萊塢打星之路，帶來太多的幫助。

1999年，華納影業在北美地區，對李連杰的《黑俠》進行了發行。該片上映後，獲得了不錯的成績。華納影業也趁熱打鐵，為李連杰策划了《致命羅密歐》。

而李連杰的好萊塢打星之路，也隨著這部《致命羅密歐》的熱映，迎來轉折，之後更是拍攝了《龍之吻》、《龍潭虎穴》等經典作品。

1998年，洪金寶、成龍、李連杰都走入了好萊塢的電影市場。然而三人之中，洪金寶無疑是處境最為凄慘的一位。

《一個好人》在北美市場上映後，取得了不錯的成績。於是在1998年，信心滿滿的洪金寶也來到了好萊塢，並希望在這裡發揮自己的導演才能。然而，在好萊塢洪金寶沒有接到電影邀約。只受到美國CBS電視台邀請，出演了美劇《過江龍》。

而這部美劇，也是洪金寶在好萊塢參與拍攝的唯一一部影視作品。

1998年，成龍、洪金寶、李連杰三人同時走入了好萊塢。2000年的成龍，因為《尖峰時刻》、《上海正午》春風得意。此時的李連杰，也因為《致命羅密歐》逆風翻盤，迎來事業轉折。而這一年的洪金寶，卻對好萊塢心灰意冷，並離開北美，重返華語影壇。

昔日在港片銀幕上備受喜愛的三位動作大咖，此時卻在好萊塢遭遇了天差地別的境遇。同樣都是在1998年來到北美，而洪金寶卻在2000年，率先結束了自己的好萊塢征戰之路。
