娛樂圈那些分手後的情侶如今都怎麼樣了？黃宗澤後悔，周星馳孤獨

天天要聞
3 天 ago

娛樂圈好多情侶分手後，有的是雙方都情傷，有的是一方混的風生水起，而總有幾段感情是觀眾都為他們可惜，下面看看有沒有同感？

黃宗澤是港星中有顏值有演技的男明星，出道多年戀情緋聞也不少，唯獨與胡杏兒的戀情讓人遺憾，7年都沒有修成正果。2012年，胡杏兒承認與黃宗澤分手，與黃宗澤情斷後，胡杏兒遇到李乘德就迅速步入了戀愛，並很快步入了婚姻，成了幸福的人妻。至今，胡杏兒結婚才4年，就生了2個兒子，事業家庭孩子齊齊豐收。

在綜藝節目《各位遊客請注意》綜藝節目上，「大冒險」之後，黃宗澤在第二輪遊戲中輸了，黃宗澤接受女嘉賓的真心話大冒險拷問，一位膽子非常大的女嘉賓Seeya_L 激動地發問黃宗澤：「你現在對胡杏兒什麼感覺」。黃宗澤首先大方地對胡杏兒送上自己的祝福，隨後黃宗澤還補充地說：「我也替她現在的生活感覺到很開心，真的。」

看看他們的曾經，感覺真的很可惜，自從分手後，黃宗澤總是感覺形單影隻，鬱鬱寡歡！

周星馳出演的電影大多都十分經典，即使放到現在來看依舊十分精彩，他在電影中飾演的角色彷彿自帶一種吸引力，能夠讓許多人真切地感受到那種情緒。他的笑中夾雜著酸楚、苦澀、無奈，相信每一個歷經社會磨練的成年人都能真切感受到那種情緒，

這也正是他演技的強大之處，明明是在演戲，卻總是能讓觀眾將自己代入到劇中，說不出的酸甜苦辣，五味雜陳。這就是星爺電影為什麼能讓人回味，經久不衰的原因，現在多數電影都是一笑而過，沒有深意！

他與朱茵曾有過一段地下情，分手後大家才知道，朱茵已經結婚生有一子了，而周星馳事業很好，但是婚姻卻至今單身，有次綜藝採訪問他為什麼單身？他笑了笑說：「我運氣不好啊」！不是別的，當時感覺星爺好滄桑，好希望他可以幸福啊，我覺得是星爺也好，黃宗澤也罷，不是多留戀前任，是他們可能始終沒有碰到那個讓他們再次心動的女孩子了！也許曾經那段感情就是最該珍惜的，可是，往事不可追，沒辦法，希望他們在次獲得幸福吧！
