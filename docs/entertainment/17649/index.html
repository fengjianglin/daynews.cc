印小天插刀門，韓寒站高曉松：時間就是一把撕裂虛偽的尖刀

2019-12-17 399條評論 1,306次閱讀 375人點贊

2012年演員邊瀟瀟發微博說自己被同組的演員印小天打了，大罵印小天打女人，不是個男人。

隨後，娛樂圈裡不少演員都相繼為邊瀟瀟發聲，控訴印小天是個打女人的卑鄙男人。

在那些發聲的演員中，有不少人都和印小天合作過，甚至還有幾個在出事前一直和印小天是「好兄弟」關係。

那時，星途璀璨的印小天迅速被一眾明星推上風口浪尖，哪怕後續他放出了許多能夠證明邊瀟瀟說謊的證據，大家都不信。

沒多久，網友、明星一手把這個當紅小生拉下了泥地里，徹底給他標上一個「打女人」的標籤。

直到後續媒體採訪了更多當時在現場的工作人員，他們都一致的說印小天沒有打人，這才洗清了印小天冤屈。

曾罵過印小天的網友在微博道歉，也要那些不顧事實替邊瀟瀟發聲的明星們道歉，可是那些人除了默默刪博什麼都沒做。

多年後，有媒體採訪印小天問他對當年的「插刀門」是什麼看法，他說：「我看人，確實很不準」。

當年一場「插刀門」，將印小天拉入了事業低谷，此後他的事業再沒什麼起色。不過好在，經此一事他算看清了那些虛假的面孔。

在娛樂圈裡，高曉松和韓寒的友誼也是一段佳話。

在2006年，那時微博還沒有出現，許多熱愛上網的人都聚集在博客上，那時因為言語犀利，韓寒成了博客時期小有名氣的網紅。

眾所周知，在圈裡要說誰脾氣怪、說話直接，那肯定很多人投高曉松一票。就因為幫朋友與韓寒對罵，高曉松和韓寒有了最初的交集。

俗話說不打不相識，因為博客上的一場罵戰，高曉松和韓寒成了朋友，倆人時不時還出來喝點小酒、交流交流文學。

2011年，高曉松醉駕造成了交通事故，被法院判處拘留六個月，就是在牢獄的時候，高曉松自編自導的電影《大武生》上映了。

那時，還在牢獄中的高曉松打電話給自己在上海的幾個大腕兒朋友，希望他們去幫自己的電影站台宣傳宣傳。

結果，在所有打過電話的腕兒里，就只有韓寒一個人來了，他還是其中最不愛社交的人。

那時，韓寒自己在山東有一場賽車比賽，為了給高曉松宣傳，他比完賽立馬就轉機到上海來了，說：「曉松不在，我們幫他把這事給辦了吧。」

韓寒這件事讓高曉松非常的感動，曾經在很多場合里都誇韓寒是個真朋友，常感慨：「人生有這樣一段經歷，你才能看出人心。」

網友葉揚分享過自己一個故事，他說：「相識10年的朋友，一個晚上我就決定了拉黑他」。

那時，葉揚的二寶剛出生不久，家裡各項開銷緊張，他一直想著出去做點兼職補貼家用。

剛好這時，一個認識了十年的好朋友徐立言跑上他家來，說是有一個發財的好項目。

他說他手頭上有一個電商項目，現在正是紅利期，如果現在拉人加入就可以實現共贏，每月至少多5000的額外收入。

聽著徐立言描繪的偉大宏圖，看著妻子在房間里哄著兩個孩子睡覺，葉揚非常的心動，想著明天就把定存的那筆錢拿出來做投資。

可是，在他還沒有把錢取出來的時候，他就接到了另一個大學同學的電話，他說他被騙了10萬。

接著葉揚就問了他那個同學的被騙經歷，結果發現，他被騙錢的電商項目和昨天徐立言給他描繪的一模一樣！

一時間，葉揚又氣又震驚，急忙跑到徐立言家裡，逼著他必須說實話，不然就捅到徐家爸媽面前去。

這時，徐立言才坦白說，其實自己也是被騙了，他已經陷進去了十幾萬，如果現在不再找幾個代理進來做冤大頭他賠的就更多了。

顯而易見，徐立言就是打算拉葉揚進來當冤大頭的。為了自己的利益寧願拉朋友墊背，十年的友情就被他毀了。

當天晚上，葉揚就拉黑了徐立言，刪掉了他所有的聯繫方式。朋友，果然是日久見人心。

有人說，時間就像是一張網，好的壞的，會通過這一張網，一點一點的被挑揀出來。

虛偽的感情總會隨著時間的推移，被時光的網磨掉外在的稜角，露出真實的內里。

這時候，誰是真，誰是假，一目了然。

感情經歷的時間越長、遇到的事兒越多，你就越能分清哪些人對你是真心的，日久見人心、患難見真情就是這個道理。

作者：阮鳳兮，一個兼具理性與感性的江南姑娘。
