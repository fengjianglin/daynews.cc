追劇方式又變了，角色比劇情更好追？

2019-12-23 374條評論 2,233次閱讀 467人點贊

12月以來，古裝劇「泄洪」般集中上線，《鶴唳華亭》、《慶余年》、《劍王朝》等劇可謂「神仙打架」，掀起觀眾新一輪的追劇熱潮。但這背後，幾大古裝劇風格的演變，似乎也反映出了時下觀眾觀劇審美的迭代。

那麼，究竟什麼樣的劇集風格、什麼樣的角色人設更能刺激大眾的觀劇慾望呢？

配角亮眼，各具看點

觀眾有偏好

不管是開年大劇《知否》，還是暑期檔熱劇《長安十二時辰》與《陳情令》，亦或是當下熱播的《慶余年》、《鶴唳華亭》等大劇，皆口碑與熱度兼具，而又都是古裝群像戲，配角對劇集熱度的帶動作用不可小覷。

《慶余年》開播之際，就有統計這部劇的熱度帶動最大的是據說20番開外的肖戰，而實際上，開播之後，的確是配角更精彩，小范閑、范思轍、滕梓荊、費介、五竹、王啟年、二皇子、辛其物等配角討論度頗高，並且屢上熱搜。

而《鶴唳華亭》中戲份加起來不到一集的「張內人」（郝蕾扮演），八番配角太子妃等角色的相繼下線，也都引起熱搜話題。此外，還有王勁松、張志堅、王建國、等多位老戲骨加盟，保證了劇集的質感。開播不久的《劍王朝》，亦有劉奕君、李光復等實力派做配。

幾部劇配角陣容強大，演員們的演繹也頗受認可，正反派人物群像看點多多。《慶余年》中郭麒麟成為觀眾新的快樂源泉，反派林珙的妹控人設，戳中不少女性觀眾的蘇點，讓人很難討厭起來。《鶴唳華亭》中，郝蕾將人物的猶豫與抉擇詮釋得淋漓盡致，而太子妃的「白月光」人設又令觀眾心疼不已。

配角刻畫細緻多樣，但若要總結起特點，劇中人物又略顯趨同，反派人物「集體可愛」。《慶余年》的人物呈現萌化傾向，范思轍、王啟年、二皇子、辛其物為劇中「四憨」，而小范閑、費介、五竹、陳萍萍、慶帝等角色又屬「腹黑型」人設，正反派都有其可愛的一面。《鶴唳華亭》中善如太子妃、盧世瑜等「白月光」般存在，惡有奸角李柏舟處處針對，亦有蕭定棠、趙貴妃等「可愛」反派令觀眾又愛又恨。

如果說兩部劇中的配角都十分出彩，但喜劇性人物似乎更容易博得觀眾喜歡，出圈指數更高，已經成為2019年群像戲的常態。《都挺好》中的蘇大強、蘇明成，《小歡喜》中的方一凡、林磊兒，都是自帶萌點的人物類型，一舉一動都容易引發社交話題。

《慶余年》光最近幾天，就有#陳萍萍是個狠人#、#二皇子可愛#、#五竹笑了#、#慶帝梳頭了#等喜劇角色或笑點頻上熱搜，而隔壁《鶴唳華亭》近日僅有#太子妃下線#成為熱搜話題，而太子妃的「白月光」人設，也是觀眾喜愛的設定。

兩部劇出圈「指數」有所不同，或許和兩劇的基調是掛鉤的。《慶余年》歡脫有度，喜劇、權謀、甚至是科幻元素都徐徐展開；《鶴唳華亭》則過於悲涼，論感情線，男女主虐戀情深，論權謀線，正派人物又幾乎沒有真正贏過。截止成稿，《慶余年》在豆瓣上超10萬人評分為7.9分，而《鶴唳華亭》3萬餘人打出7.5分，觀眾的偏好或可管窺。

主角存在感被削弱？

無論是一喜一悲的劇集基調，還是更具看點的配角表現，但對配角身世、性格的豐富的同時，一定程度上也讓兩部劇主角的存在感被「削弱」。

首先，與劇集基調相對應的兩個男主都很「弱」。相較原著，范閑更加溫和善良，雖然一路金手指大開，但多是關係類外掛，並非男主自身實力。而在更為嚴肅的《鶴唳華亭》中，「哭包」蕭定權在權謀之爭中顯得氣場太弱，給了觀眾太過「憋屈」的觀劇體驗。

其次，今年的古裝劇普遍採用群像戲架構，「青年主演+實力派做配」成劇集標配，但大波老戲骨實力太強，青年主演們自然相形見絀。開年大劇《知否》中「硬核祖母團」戲份不多、但熱議不斷；《鶴唳華亭》中張志堅、王勁松、邱心志、鮑大志、王建國幾位戲骨更是接受了芭莎電影的採訪；《慶余年》中諸位配角更是「常駐」熱搜。值得注意的是，最近幾部古裝劇男主們的扮相不佳或許也影響了其「出圈」指數。

男主存在感尚且不足，女主們更是大多淪為「鑲邊人設」，女性角色越發單薄化、工具化。《慶余年》兩大女性角色范若若與林宛兒傻白甜化，而辛芷蕾、李純、劉美彤等實力派小花集中亮相，在劇中的作用卻大多止步范閑的「紅顏知己」。而同時在《鶴唳華亭》、《劍王朝》出演女主的李一桐，也因為人設不精彩被網友吐槽「帶誰不火系列」，李一桐則一語雙關回復「我體寒」。

三部男頻劇同期熱播，幾位男女主的表演也都在水準之上，但是相較於表演嫻熟、人設更為豐滿的配角們，男女主角的存在感卻不斷被擠壓。但不得不說，「青年主演+實力派做配」的打法一定程度上保障了劇集的品質，而未來，業務能力才是角色「存在感」最佳證明。

故事先行，還是人物先行？

過去很長一段時間，影視行業都表現出對好劇本、好故事的渴求，今年《都挺好》、《小歡喜》、《陳情令》等口碑熱劇，在講好故事的同時，也因「人設紅利」帶火了不少演員。

當下，《鶴唳華亭》、《慶余年》、《劍王朝》三部男頻劇同台競爭，或許能從編劇/導演對劇集的理解與表現入手，推測主創人員之於故事與人物的態度。

《慶余年》編劇王倦採訪時曾表示，「當你筆下的人物被觀眾喜歡之後，你會發現一個很有趣的現象：當觀眾喜歡劇中的人物時，即使人物沒有劇情發展，他們在說閑話，哪怕說一個小時的閑話，觀眾也願意看。」在具體的情節鋪展中，《慶余年》不斷反轉也是更日常、生活向的，看點都多在人物閑話時冒出的「金句」之中。

該劇導演孫皓同樣表示，劇集在籌備時便認定了「大題材幽默做，沉重題材輕鬆做」的拍攝方式，而播出至今不難看到，《慶余年》的幽默與輕鬆主要靠趣味性的人物設置，達到解構嚴肅、增添笑料的雙重作用。

而官方主打「高能反轉」的《鶴唳華亭》，或許是出於對強情節爽劇的青睞，過於注重反轉的次數和形式，正如導演楊文軍的原話，「60集的劇集體量中，幾乎每集都有反轉」。但「反轉」過多除了令觀眾眩暈，卻沒有因此凸顯更具看點的人物形象。沒有討喜人設支撐的權謀劇，對於觀眾來說無異於看俄羅斯套娃，很容易幾集棄劇。

《劍王朝》開篇以復仇為主線，先後加入宮廷、江湖、修行等元素，但人物的出場、背景卻沒有給出足夠的表現空間，僅僅通過字幕來展示。同時，該劇世界觀龐大、人物派系眾多，即便有不少熟面孔的老戲骨做配，但觀眾直到下線也沒能記住劇中人物的名字和幫派。

與《劍王朝》人物名字拗口難記相反的是，《慶余年》早在播出之前就因劇中角色名字太過簡單而出圈，陳萍萍、司理理、范若若、戰豆豆、范閑、范建等「ABB」式或帶有諧音梗的名字，觀眾笑過了也記住了。

或許是這屆觀眾太沒「耐心」，於是《慶余年》憑藉可愛的角色們就俘獲了觀眾的心，而「正經」的《鶴唳華亭》與《劍王朝》，前者角色太過悲情，後者角色關係繁雜，似乎很難令觀眾沉下來看劇。

誠然，好故事永遠是優質作品的基石，但好的人物則是作品迅速出圈的保障。對於優質作品而言，「故事」和「人物」不能單行，只有並行才能兼具口碑與熱度。
