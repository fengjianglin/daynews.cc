《吹哨人》：揭秘海外行賄驚天操作！王策每分鐘都是戲，懸念感足

天天要聞
2019-12-13

電影《吹哨人》上映以後彙集了各種稱讚和質疑的聲音，各路人馬褒貶不一，使得這一部電影的命運多舛充滿了豐富的懸念色彩。

不可否認的是，不管片中男女主角出於什麼出發點做了吹哨人，他們充滿艱辛的舉報之路還是喚醒了觀眾內心滿滿的正義感。

對於很多觀眾來說，「吹哨人」是一個陌生的辭彙。1986年牛津英語辭典增補本第一次對「它」有了解釋。

1972年的時候，英國就已經將「吹哨人」定義為內部舉報人，我國目前有一些地區在試點「吹哨人」制度。

其實百姓多見的這種角色叫「知情人」。《吹哨人》中的Peter Wu其實就是整劇的核心知情人，他的角色串聯起了整個高層的利益鏈

當然，海外的生活並不是我們想像的那麼精緻，就如Peter Wu（王策飾演）和Mark(雷佳音飾演馬珂)。

影片一開始為我們展現的是兩個華人在中國人遇到的職場矛盾，Peter Wu玩笑帶著一絲絲揭短的成分，語氣咄咄逼人，態度不甘心。一個是職場老手連棍帶槍的為自己鋪路，一個是傻白甜的高層新星強裝淡定，它在兩者張力上的設計帶觀眾跳入了劇情的陷井。

可以說王策把這個角色的雷挖的特別成功，從你看到他的每一秒都是戲點，環環相扣的人物關係和線據都在懸念中讓你內心想推敲下去。

Peter Wu和公司一直是相互利用的關係，Peter Wu利用公司觸到華人職場的天花板，想成為海外核心圈的管理層之一，而公司想利用Peter Wu達到自己的商業目的。

在慾望的促使下，公司制定一連串可怕的海外行賄行為，但Peter 卻遲遲沒有能把項目推進下去。影片在播的過程中，一直在埋包袱，前期一直圍繞著一個問題：「Peter 到底怎麼回事？

這個Peter 顯然是一個有生活痕迹的，在一連串真相爆發之前，他曾經經歷了善與惡的衡量。

一邊是高層苦苦相逼儘快達到合同，他的一次又一次借口逃避；另一邊是技術漏洞將會導致的可怕真相，技術員一次又一次拒絕他的賄賂；我們從客觀條件上能看到他的掙扎，進而也就明白為什麼他一看到馬珂就會很激動，甚至用語言有預設性的攻擊他。

也正是前面這些古怪的騷操作，馬珂從Peter的提示里，找到了線索的蛛絲馬跡。王策通過人物表達的關鍵點：救贖、隱忍、反抗是，抓住角色的擰巴與變數，這一系列的刻畫將Peter短暫的瞬間發揮到了最厚重的解構性。進而，你會在後面的片段你會看到馬珂不斷回憶Peter高喊「Check the Gate 」的鏡頭

其實Peter 高喊「Check the Gate 」有兩層意思。

一層是馬珂去非洲發現的關鍵問題，從 Gate 9進去，馬珂終於發現這家公司地下管道埋著一個是非天然因素造成的地質坍塌現場，說明公司技術存在了很大的風險漏洞。

而另一層意思影片的梗：「不要打開埋下禍根的大門」，即「Check the Gate 」。

這也就是Peter 為什麼使用了和他身份極其不符的語言，那種焦灼感預警了這個驚天的大秘密

商業競爭是殘忍的，並非所有的人都是可以合乎道德合乎法律的選擇做自己。Peter 到底是怎麼死掉的，警方一直沒有一個說明，但結局已經說明一切了。

或者說劇本不允許Peter 還活著，至少有過明確犯罪事實註定要受到懲罰。於是，我們能看到導演特別糾結的一面，一面是同情，一面是批判。任何人都沒有道德豁免權，即使是主角。

拋去感情戲和動作戲的外套，《吹哨人》是一個略帶深刻主題的電影。他們三個主角雖然在情感和道德上被大眾批判，但並不意味著與惡有關。站在至高視角下，看中國人在海外艱辛的「吹哨」無不為之動容

Peter不像馬珂有著憨厚的執著，也不似周雯有著迷一樣的心思，更與貪婪的高層有著完全不一樣的風格，他這個人藏了很多事。在眼神和語氣中，王策在限的台詞情景中賦予了人物最多的情緒語言。透過他短短的信息，你會發現職圈埋藏著太多太多的複雜。

與馬珂的對話鏡頭裡，Peter是強勢的上一位又是弱勢的呼救者；在與非洲工程師的對話中，他的眼神訴求了無限的狼狽與無奈，一身褶皺的西裝無時不提醒觀眾這個人處在困難中。他是夾雜在兩個極端的夾心人物，也是一個多面化的人物 相比於雷佳音的城市英雄人設，大家對身不由已的王策更有代入感。或許你會覺得這是演員的本色演出。的確，有著專業擊劍訓練和模特功底的王策駕馭這種角色輕車熟路。

但無疑王策也是實力派演員。從第一個鏡頭開始，Peter的每個行為都有著張力和懸疑感，正是演員擁有這種與角色需要的沉穩重疊，才使得人物有了更厚重的層次感

王策飾演過的角色，無論是海歸精英或是國民老逗，都給人留下了深刻的印象，比如今夏最火最火的偶像劇《親愛的熱愛的》。

王策在該劇中飾演佟年的爸爸，他身上結合了高知與暖萌的喜劇色彩，圈粉無數。從風格演繹上來說，我更喜歡《天盛長歌》的顧指揮使，一身鎧甲，錚錚鐵骨，千斤重擔。

不管是什麼樣的角色，他都能賦予人物最精準的情緒和狀態，在起伏之間塑造出人物的底韻。這需要極高的情商與演技，兩者缺一不可！
