有著一個不錯的構思，卻輸給了莫測高深的情節

2019-12-23 740條評論 1,899次閱讀 322人點贊

官人說：

好萊塢懸疑片備受國內影迷喜愛。

最近又上映了一部懸疑片。

那就是由好萊塢影帝馬修.麥康納和好萊塢甜心安妮.海瑟薇主演的《驚濤迷局》。

《驚濤迷局》講述的是一個關於真相與救贖的故事，兩位主演的情感關係非常撲朔迷離。

今天首席影評官發表的是入駐影評人王行勤的文章—《驚濤迷局》。

安妮.海瑟薇和馬修.麥康納自從2013、2014年相繼得到奧斯卡肯定後，就陷入一個怪圈。

兩人主演的影片，除了14年的《星際穿越》票房口碑雙豐收外，其餘全部表現平平，更糟糕的是，由於給製片公司造成不小的虧損，二人都不幸邁入「票房毒藥」的行列。

也許《達拉斯買傢俱樂部》和《悲慘世界》兩部優秀影片已經耗光了他們對表演的激情，再度合作的這部《驚濤迷局》也沒有挽回越來越低迷的職業生涯。

從設定上看，其實《驚濤迷局》有著一個不錯的構思。

一開始，影片就製造各種懸念，圍繞著男女主角撲朔迷離的身份和背景，不斷拋出謎團，讓觀眾想一探究竟。

然而，後期雜亂無章的解謎過程，無節制的故弄玄虛和莫名其妙的反轉，徹底浪費了電影前期所有的鋪墊。

同時，導演完全不顧觀眾的需求，將一個簡單的故事拍的又臭又長，在觀感上也將影片推入萬劫不復的深淵。

1.主題風格搖擺不定

對於一部電影，要想讓觀眾記住並喜愛，一定要有自己獨特的風格。

很顯然，《驚濤迷局》完全沒有找准自己的定位。

剛開始的時候，影片主要朝著科幻片的風格去發展。不斷出現的推銷員，父親與兒子的心靈感應，讓人以為這又是一部類似時間旅行的腦洞大開電影。

然而，隨著故事的推進，尤其是女主角卡倫的出現，影片又迅速轉入懸疑片的基調。精心布局的謀殺，人性的拷問和掙扎，似乎正在醞釀一場暴風驟雨般的巨變。

可是，當真相揭曉後，觀眾才驚覺，原來整個故事都不過是一個小男孩的意淫和想像，影片在荒誕中最終淪為神經錯亂的精分片。

一部電影糅合多種元素並不是壞事，如果處理好的話，甚至會錦上添花。本片想嘗試這樣做，但完全失敗。

它有野心，想通過多種敘事方式，去探討諸多的社會問題，諸如教育，諸如社會對普通人的關注，甚至還有伊拉克戰爭的創傷。

但是由於導演的能力有限，又犯了貪多嚼不爛的毛病，導致全片沒有任何一個觀點表述到位，從而造成現在極為詭異的風格。觀眾看完不僅無法獲得滿足感，反而會更加一頭霧水。

2.人物邏輯關係混亂

由於有了兩位大牌演員的加盟，影片基本把所有的重心都放在貝克和卡倫兩個角色的塑造。

然而，劇本的先天不足，以及導演的刻意為之，讓兩人從頭到尾都透著各種不協調。

貝克對金槍魚的執念，卡倫對現任丈夫的仇恨，儘管影片在後期交待是他們兒子帕特里克設計的情節，卻仍然無法讓觀眾認同這種邏輯關係。

因為二人始終沒有擦出任何火花，他們更像是為完成這個局而強行放在一起的布景板，所有的互動都經不起推敲。

於是，電影就用不斷的露肉和激情戲去衝擊觀眾的眼球，卻尷尬的發現，不僅馬修身材一般，安妮也只露個後背，所有裸露除了展現人類最原始的慾望，毫無美感可言。

3.節奏把握完全失控

不得不說，如果電影認認真真去講述一個關於殺人的故事，可能都比現在要好很多。

前期犯罪的動機、執行的猶豫、過程的多變、旁人的阻擾，都能夠一一進行展現。

尤其是男主角貝克對這件事情態度的轉變，透過他和兒子感情的閃回，很容易讓觀眾主觀去接受這場犯罪，而不需要考慮法律層面的問題。

可是這種緊張氛圍的營造，卻沒有一直堅持下去，電影后期突然變得極為鬆散

也許因為整個故事都是男孩設計的遊戲，讓影片之後的發展變得很不確定。

作為一個未成年人，就算再聰明，智商再高，缺乏生活經驗的累積，還是很容易把自己臆想的社會和現實脫節，讓成年人看後有種說不出的難受。

電影最大的問題，就是把真實和虛幻輕易的進行割裂，用簡單粗暴的劇情和台詞強行打破觀眾的幻想，不留一點餘地。

所以當結局告訴觀眾真相時，不僅沒有恍然大悟的感覺，反而讓人倍感失落。因為電影用了太多筆墨去刻意強調的細節，到頭來卻發現其實根本沒有多大用處。

而作為Npc出場的配角，除了說些神神叨叨的話，讓情節莫測高深外，實在是看不出意義何在。

電影成功把一部充滿著諸多元素的倫理道德片講述的沉悶無比，並在兩位大牌演員的精心演繹下未能激起一絲浪花。

很奇怪的是影片片名直接翻譯應該叫做《寧靜》，然而全片和這個名字卻一點關係都沒有。
