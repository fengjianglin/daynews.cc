董晨：以“最帥警察”，打破“校草”桎梏

2019-12-19 1,118條評論 2,257次閱讀 1290人點贊

1999年12月20日零時，澳門回歸的歷史時刻被銘記在了華人的心中。轉眼20年過去，我們共同見證了澳門的磅礴變遷。為了慶祝澳門回歸20周年，以澳門普通民眾為主角，歌頌澳門人民艱苦奮鬥精神的獻禮劇《澳門人家》應運而生。

自該劇在央視一套播出以來，不僅在開播首周取得了酷雲指數第一的成績，穩居全國黃金檔電視劇榜首，更讓在劇中飾演「澳門最帥警察」陳小帥的董晨猝不及防的火了一把。

最近，金牌經紀人有幸與這位新生代演員來了一次面對面的對話。在採訪中，他聊表演、聊角色，思維清奇、獨具匠心，展現出了與憨厚、呆萌的陳小帥截然不同的一面。

//

首演獻禮劇

//

近年來，古裝劇市場遇冷，越來越多的觀眾開始關注以「小人物、主旋律」為主題的現實題材電視劇。作為獻禮澳門回歸祖國20周年的現實主義力作，《澳門人家》聚焦澳門三灣斜街上的百年老店「梁記餅店」，書寫了三代人如何將「梁記餅店」發揚為「澳門良心」的故事，全景展示了澳門回歸祖國前後的發展變遷。

劇中，董晨飾演的陳小帥是一名澳門警察，在澳門著名的觀光區三灣斜街負責居民和商戶的生活安全，崗位雖小，卻有著崇高的使命。他來自一個單親家庭，10歲的時候跟隨媽媽來到澳門投奔舅舅，所以嚴格意義上並不是一個純粹的澳門人。但也正因如此，澳門對他來說有著特別的意義。在董晨看來，陳小帥是一個柔軟且複雜的角色。

「雖然他外表陽光、開朗，但我不認為他像看起來的那麼快樂。他骨子裡是有一種自卑感的，這種自卑感來自於他的原生家庭。他的父親在他很小的時候就去世了，他從廣東到澳門，又要很快的去適應環境，可能還會被當地的小朋友欺負。種種這些加之在他的身上，造就了他的多面性。」

其實，《澳門人家》是董晨第一次出演獻禮劇，而陳小帥也是他第一次飾演「乖男孩」。因為在出道第一年就演了5個富二代，又自帶一股「公子氣質」，他給觀眾留下的印象大多是富家公子。

正因如此，剛接到這個角色的時候，就連董晨自己都是忐忑的。但是真演下來之後，他發現自己與角色真正做到了「合二為一」，這是這部戲帶給他最大的驚喜。

現在重新看，真的覺得自己演得比誰都乖，完全被帶進去了。就連我爸媽都說，這個角色身上完全沒有我的影子，包括在現場，柯藍開玩笑管我叫傻兒子，董潔也覺得我傻傻的，他們真的覺得我在現實中是小帥那樣的性格，這讓我覺得挺有成就感的。」

當然，在拍攝過程中，任達華等實力派演員也給予了董晨很大的幫助。

「《澳門人家》里的酒戲特別多，導演對於演員喝完酒之後一剎那的反應要求很高，但是現場不會給你真的酒，我又是一個不喝酒的人，這件事就讓我很苦惱。後來，華哥教給我一個好方法，他說喝二鍋頭就像你在喝葯或者猛的被酸到的感覺，我覺得特別形象。」

//

以細節打動人心

//

董晨是一個「感受型」的演員，在表演上，他一直有一個原則——將自己的每一個角色都演成「人」。在他看來，演戲也是演人，想要演好人首先要學會「落地」，無論是哪個年齡段、哪個職業的人。或許正是因為這種原則，他才能真正做到細節取勝，以情動人。

「他演得非常好，他在戲中有很多不同的笑容，這種笑容真的很真實、很單純，我們演不到。」在《澳門人家》的開播發布會上，任達華曾如此評價董晨。

任達華提到的，是演員的一種「下意識」。很多時候，這是「演」不出來的，因為未經雕琢，所以更加真實。比起很多「技巧型」的演員，董晨的表演趨於生活化，非常接地氣，幾乎沒有表演痕迹。這是一種更高級的表演，他用一種明朗且克制的方式將自己「裝」進了角色的容器。

「出演這樣的一個角色，我覺得表演必須要很克制，那個度不能太高也不能太低。」

至於怎麼才能完成「克制」的表演，在董晨看來，秘訣在於細節。為了將人物還原得更真實，包括澳門警察在巡街時銳利的「鷹眼」，敬禮時右手從褲線划過額頭的弧度，甚至是他們平時走路的速度和說話的狀態，他都有仔細研究。對他來說，想要演好一個角色，要精準到毫釐。當一個演員做足了準備工作，之後的一切都是水到渠成的。

「像是小帥過生日我在等梁叔和家棟的時候把蛋糕帶子捋直，還有我在戀愛巷拿花等梁舒出來的時候把花剝開，這些動作我真的沒有設計過，都是下意識的。」

同樣，在拍攝過程中，還有一場戲令董晨印象深刻。那是前幾天播出的一集，全家人圍坐在一起看奧運會，此時國歌響起，所有人都不由自主的站了起來。

其實在實拍的時候，電視上並沒有播東西，演員們盯著的只是綠屏。然而，就在工作人員奏響國歌的一剎那，他和任達華率先站了起來，跟著音樂一同合唱。誰能想到？這令人心潮澎湃的情景竟然是臨場發揮。這是演員內心的一種本能反映，也是所有演員共同的默契。

「所以其實我覺得這樣的表演叫本能反應，就真的是你由衷的去溢出一種表演，這個表演是難能可貴的，這是我覺得是最好的表演。」董晨激動的說。

//

不只是「校草」

//

作為演員，董晨有著令人艷羨的高顏值。

因為面部五官比例精準堪稱完美，他曾被整容博士陳煥然稱為中國五官最完美的男生。後來，因為一張照片在網路上流傳，得到了很多人的轉發，他又得到了一個「第三代國民校草」的稱號。今年，他還因為高顏值被TCCAscia評為亞太區最帥100張面孔第89名。

然而，很多人不知道的是，雖然是以顏值出圈，但生活中的董晨卻是個不折不扣的體育和英語特長生。

董晨從小就是英語特長生，從初中開始，他就經常以北京地區代表的身份參加演講比賽。不只如此，因為自己的英語特長，他從很小的時候就開始看英文電影、聽英文歌。時至今日，已經累積了近千部的閱片量。他預測的奧斯卡獲獎電影，精準度達80%。

對於年輕演員來說，大量閱片的好處不言而喻。如此數量的優秀電影，不僅磨練了董晨的審美，更讓他對表演有自己獨特的見解。他欣賞Jennifer Lawrence在《烏雲背後的幸福線》中的爆髮式演技，也喜歡金敏喜在《獨自在夜晚的海邊》中那種潤物細無聲的表演。在他看來，好的表演要像一潭清水——慢慢划過的時候，水面上的冰碴一點一點的融化。

「《烏雲背後的幸福線》中有一幕，是Jennifer Lawrence教Bradley Cooper跳舞，告訴他什麼叫『情緒』。當我看了五六遍之後，突然明白了這個詞的意義。」

同樣，聽歐美音樂也讓董晨很有「態度」。他喜歡說唱歌手Nicki Minaj在舞台上的侵略性和遊刃有餘，這讓他聯想到表演中的聲台形表。

「因為她要說出態度，把歌詞中隱含的內容說出來給別人聽，這和表演是一樣的，我覺得很多藝術表現形式都能輻射到表演。」

《烏雲背後的幸福線》

在董晨看來，一個好角色一定能是反映人性的。作為演員，應該通過自己角色讓觀眾產生反思。

「首先，這個角色的三觀我要認可，不說全部認可，至少也要認可一半。其次，這個角色要跟我有一定的共性，就像陳小帥，雖然乍看之下與我不同，但仔細去想，我是可以從他身上找到共鳴的。最後，我偏愛現實題材的角色類型，像是有原生家庭影響的角色，他的種種行為，能讓觀眾對自己生活產生反思。」

雖然是初出茅廬，但董晨顯然對自己的未來十分清醒。這樣的他依然在不斷成長，不得不讓人感嘆後生可畏。

「演員對我來說就是一件工作，或許性質和其他工作不太一樣，但並沒有本質區別。如果說我有什麼原則，第一是熱愛，第二是專註，第三是不忘初心，這是亘古不變的真理。」

劉昊然
