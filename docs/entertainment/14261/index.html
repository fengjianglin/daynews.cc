排片不到3%，這部經典不該被忽略

2019-12-16 139條評論 3,289次閱讀 71人點贊

今年10月，《續命之徒：絕命毒師電影》上線，瞬間引爆全網熱度。

距離劇集完結已有7年，主創們彷彿在以這種方式告訴粉絲：儘管天下沒有不散的筵席，但戲中人的故事仍在繼續……

幸運的是，除了《絕命毒師》，還有一部經典劇集在今年迎來了電影版。

2010年首播，2015年完結，每季平均分9分以上，如果盤點屬於2010s的經典劇集，一定有它一席之地。

劇集落幕四年後，萬千觀眾期待已久的大電影，終於緩緩而來——

唐頓莊園（2019）

導演: 邁克爾·恩格勒

主演: 米歇爾·道克瑞 / 休·博納維爾 / 伊麗莎白·麥戈文 / 瑪吉·史密斯 / 艾美達·斯丹頓

類型: 劇情

製片國家/地區: 英國 / 美國

上映日期: 2019-12-13(中國大陸) / 2019-09-13(英國)

片長: 122分鐘 / 120分鐘(中國大陸)

又名: 唐頓莊園電影版

《唐頓莊園》電影版有一個英式又含蓄的開場，觀眾藉由一封來自倫敦的信，伴著火車壯觀的蒸汽和鳴笛聲穿行在茂密的林地里。

直到郵遞員親手將信件送入唐頓莊園，熟悉的開場樂正式響起，我才意識到，這封信件走過的長路無非是劇集離開這四年的隱喻，可能編劇和導演早就寄出了這封信，而我們也終於在影院收到了它。

《唐頓莊園》擁有六季的劇集版，同時它也是PBS有史以來收視率最高的系列劇，不僅製作精良，還有穩定出色的核心演員作為支撐。

不得不說，由劇集到院線的過渡是個水到渠成又令人驚喜的結果。

由於演員陣容龐大，故事複雜而密集，電影里唐頓莊園的每個側顏均被迅速滑現，有些場景甚至只持續了15秒，簡直速戰速決。

不過巧妙的是，這樣的節省並不影響觀眾的感知，導演和編劇實在是老練，或者說他們太了解《唐頓莊園》的肌理，他們不僅知道觀眾要什麼，也知道該如何去「給」。

在電影版中，所有重要角色都獲得了更好的未來，尤其是湯姆·布蘭森，雖然他關於政治和舊貴族體制的矛盾仍未解除，卻也做出了正確的選擇，並重新擁有了一份感情和未來。

而管家托馬斯·巴羅，在那樣的年代背景下也被編劇給予了美好的期盼，同性情節被如此寬容以待，真的很暖。

當然，「美好」並不代表會墜入輕薄的敘述，編劇費羅斯不會因為時間不充裕而去走不負責任的過場。

一次英國皇室的造訪是精明的設置，它重新激活了所有角色，無論是貴族還是僕人又回到了老樣子，他們的缺陷仍舊存在，善意也仍舊真摯。

延續著整六季的劇情，電影版聚集了所有的主次人物，貴族和僕人執行著古老調性的宏大任務，並讓各種流言蜚語充斥耳畔。

時間有限，巨大的人物群像需要被合理地輕拿輕放，著實不是一件容易的事。

但在這龐雜的人物穿插里，我們依然可以看見《唐頓莊園》的靈魂，即是那事無巨細的細節，「盤子、器皿、餐具、修理鍋爐、匆忙趕製的衣物」，是豐滿的流程讓這幢唐頓莊園鮮活起來。

一群人忠於這幢莊園，並用莊園反哺他們的精神歸宿。

編劇費羅斯早在為劇集版寫結局時就想著「我最好不要把這扇門關得太緊」，在這部電影里他也這樣做了。

費羅斯懂得如何達成最溫柔的歡迎儀式和告別。

為什麼《唐頓莊園》能夠對觀眾產生持久的吸引力呢？

從劇集到電影，《唐頓莊園》之所以能捕獲一大群忠實觀眾，大抵是因為我們難以抗拒那樣一個特殊的時代氛圍。

就像我至今熱愛的國產劇集《大明宮詞》，熱愛盛唐，並非因其昌盛，而是那個時代的人令人欣賞，他們身上存放著現代人業已不具備的美好嚮往。

老劇迷們是陪伴著人物成長的，大家追隨了劇中的14年光景(1912~1926)，觀察了一群豐滿角色在整整52集里的成長和變化，所以這註定是一部越了解越動情的電影，老劇迷淚灑影院是完全有可能的。

如果說它還有什麼令老劇粉感到遺憾的地方，大概就是它實在無法顧及到所有的偏愛，比如貝茨先生的粉絲們可能真的會有些難過，他並沒有在影片里出現太多。

此外，電影里不得不捨棄一些比較重要的情節，包括「暗殺計劃」，這麼個可以抻長至三集的分支很快被轉移到了其他事上，可真是令人困擾的倉促。

但從另一角度去看，「面面俱到」反而凸顯了一抹溫柔的觸感，就像為竣工的房屋貼了一層賞心悅目的壁紙，讓一份愉悅映襯另一份愉悅。

畢竟，對大多數人來說，最珍貴的回憶正是對過去的美好回憶，電影雖然結束於一場晚宴，卻給每個人物的未來留下了美好。

哪怕二戰就要開始，哪怕貴族階級的衰落已經既定事實，這種美好仍舊撫慰人心，我們熱愛觀看人們期待而充滿動容的目光，這也正是電影版《唐頓莊園》的無私饋贈。

如果我們會去影院尋求幻想，尋求歡笑與釋放，那麼這樣一部穿著考究、精緻內斂的《唐頓莊園》是不是欠缺魅力呢？

國內院線過少的排片如實展現了這一偏見，會有誰愛看這樣一個陌生的時代呢？除了念念不忘的老劇迷，還有誰會奔向這麼一部四平八穩的電影呢？

不是劇迷錯了，也不是《唐頓莊園》沒有匹配高票房的能力，是市場還不夠了解成年人的世界，是市場還沒有真正了解到《唐頓莊園》的核心吸引力。

宴會、舞蹈、家庭陰謀、禮服、調情、純潔的愛情聖潔的婚禮，我們就是被這些東西所吸引的嗎？

其實並不是。

喜愛《唐頓莊園》的觀眾大概最能體會這樣的感受，在缺乏安全感的生活里，勞碌的成年人真正需要的不是探險故事和避世的荒島，而是這麼一座「唐頓莊園」。

每個角色都安全地蜷縮在他們的生活中，他們經歷磨難，總有惶恐，卻最終安全，就是這樣一個簡單的理想將唐頓莊園從一幢建築物升華成避難所，這是成年人最現實的夢想。

在該告別的時候離開，在最好的時機回來，多麼美好的際遇啊！

鑒於排片量過少，大家趕緊去影院看看吧，不僅是《唐頓莊園》的老劇粉，每個人都值得這樣一份幸福的眷顧。

可能你會看到老劇迷無法自控淚灑影院，沒關係的，相信你的那份眼淚也已為《唐頓莊園》寄存，我們的感動就在未來的路上。
