電影《天機之九幽業火》正式定檔12.26，熱血江湖混搭妖怪奇談

2019-12-18 30條評論 1,010次閱讀 95人點贊

近日，由導演周敬峰執導，淮文、陳偌汐、程琢、張楷依等新銳實力派明星聯袂主演，雷牧特別出演的古裝奇幻電影《天機之九幽業火》發布精彩預告片，正式宣布定檔12月26日。

影片故事講述在架空世界的暗夜王城，深邃幽玄的恐怖氛圍中，三起令人毛骨悚然的「業火殺人」案件，拉開了故事的序幕。來自天照府的破案天才時浩軒與千年城隍廟的道術奇才玄坤，兩個天賦異稟的少年為參透案中玄機而身陷謎團之中。每一件超自然懸案，最終都指向了一部名為《業火五災》的神秘古卷。它不但是破解主人公身世之謎的鑰匙，更是偵結幕後元兇的關鍵所在。

傳說中源自古倭桑國山鬼物語的《業火五災》，記載了叢原火、拔舌火、皿數火、姥姥火、釣瓶火五種業火，分別代表著人的五種罪業。傳說中，世間不仁不義不忠不孝者都難逃業火之劫，叢原火代表不忠，姥姥火代表不孝，釣瓶火代表不仁，拔舌火代表不義，皿數火代表不智。在電影《天機之九幽業火》中，被害者詭異而慘烈的案發現場，與古卷中有關於「業火五災」的傳說不謀而合。這到底是無意的巧合？還是真兇的精心構置？

例如《七宗罪》《雙瞳》等經典驚悚懸疑片，都巧妙地利用了本土神秘文化來編織恐怖兇殺的故事情節。而影片《天機之九幽業火》將東瀛妖怪奇談混搭古代的江湖恩仇，用以展現面對困境如何尋找救贖的主題內核。業火為何能在暗夜肆虐殺戮於無形？主角的身世之謎如何反轉案情？幕後元兇到底是人為操縱還是鬼神指引？神秘古卷中匪夷所思的奧義又能否揭開？諸多未解之謎，亟需影迷追隨破案高手時浩軒與道術奇才玄坤，聯手去逐一破解。一切真相，都將於12月26日大白於天下。

影片《天機之九幽業火》繼承了出品方以往作品以強情節為驅動、強視效為硬核等優良基因，同時混搭武俠歷險、懸疑推理、愛情搞笑等流行元素，強化奇幻電影的類型立體感，從而成功穿刺受眾圈層。作為影片出品方之一的奇月星空影業，近年來深耕中國奇幻電影領域，投資、製作、宣發一體化的全產業鏈勢能全面爆發，出品了《畫皮師》、《極道校園》等一批重磅爆款，成為本片的強有力背書。本片由北京奇月星空影業、上海華錄百納、深圳海川傳媒聯合出品，將於12月26日獨家登陸愛奇藝，敬請期待。
