47歲黎姿近照曝光，身材曼妙素顏似少女

2019-12-21 1,294條評論 2,968次閱讀 1272人點贊

黎姿相信很多網友都認識，作為一代女神，她真的是驚艷了時光，2019年7月15日，黎姿在社交網站上面發出了自己外出遊玩的照片，照片中的黎姿應該是在某個地方度假，穿著泳衣，看起來非常清涼。也正是因為黎姿穿著泳衣，所以將她的好身材完全凸顯出來了。黎姿雖然已經四十多歲了，但是她的身材依舊凹凸有致，非常完美，完全不像是一位48歲的人，反而像一位二十多歲的少女。

不僅如此，照片中的黎姿看起來狀態非常好。面對鏡頭的時候，黎姿笑的非常甜蜜，臉上的小梨渦看起來非常可愛、機靈，脖子纖細修長，肩部也非常直挺，看來平時真的沒少鍛煉。當然，黎姿平時確實是經常鍛煉，喜歡她的網友都能夠發現黎姿經常在社交網站上面發出自己鍛煉瑜伽的照片，所以想要好身材肯定是需要努力鍛煉的。

網友們看到了黎姿的顏值和身材之後，都表示非常羨慕。也有網友都表示現在的黎姿和以前的黎姿真的是一模一樣，完全沒有變化，果然女神就是女神，就算是時光也格外憐惜，沒有在她的臉上留下任何的痕迹。黎姿既是一位優秀的歌手，也是一位優秀的演員。在黎姿剛剛開始出道的時候，黎姿就在出演電視劇和電影的同時發行了唱片，特別是在唱歌這一方面，黎姿取得了非常好的成績，她發行的第一張唱片就獲得了金唱片銷量獎。

隨著聖誕節的悄然臨近，已轉戰商場多年的黎姿，專門抽時間陪伴女兒選購聖誕禮物。一個人似乎有些忙不過來，為了更好的照看三個女兒，黎姿還攜帶了兩位傭人姐姐一同出行非常周全。一行6人進入商店，三個女兒開始挑選玩具，黎姿則負責挑選大件的裝飾物為聖誕節做準備。

她7歲的女兒Pegella非常愛粘著媽媽，她挑選了喜歡的玩具後，就湊到媽媽身邊，時不時的撒嬌，非常的可愛俏皮。Pegella也是公認”小黎姿”，那雙招牌一字眉，簡直和黎姿小時候的照片是一模一樣的!

別看如今的黎姿家庭非常幸福溫馨，事業也做的風生水起，是妥妥的人生贏家。但是她的人生經歷是非常坎坷的。黎姿是家中長女，她幼時家境優渥，在她才9歲的時候，因家族鬥爭，一家四口一下跌落到住進20多平的貧民窟。父親因為身體不好，只能找到很簡單的工作。全家的生活重擔就落到了黎姿母親的身上。黎姿的母親靠做貨車司機和打雜工養活了一家人。
