精英律師翻牌美劇？王歐靳東cp感十足，朱珠剛開始就頻繁換衣服

2019-12-24 39條評論 1,122次閱讀 65人點贊

剛播出的《精英律師》怎麼有股美劇的畫風，像人物造型和畫面感，就連對話，總有種看美劇的即視感，不少網友表示和美劇《金裝律師》有些相似，相似到場景布置甚至人物關係都很相似，幾乎完全翻牌。

剛看見栗娜和何賽於是馬上就想到了Donna和Litt，幾乎人設一模一樣，而羅檳就是Harvey的翻版，麥飛約等於Mike吧，但是又加了一個戴曦的角色給我感覺更像Mike，尤其到第二集戴曦和Mike感覺一模一樣，運用自己的聰明和精明的法律喜歡幫助別人，但是因為種種原因無法參加司法考試，但是還是被招進了律所，羅檳和何賽的關係和Harvey跟Litt的關係一模一樣，而且律所的整個結構布置，前台和電梯一模一樣，辦公室感覺也差不多。

《精英律師》延續了靳東都市成功人士的一貫風格，西裝革履、戴名表、開豪車，帥氣男、毒舌哥，奔波高檔寫字樓、身邊女秘書多的美好人生。一口氣看了五集，個人感覺靳東很適合這種精英人才的角色，尤其他穿西裝的時候，立馬成功男士的感覺湧現出來，不少網友表示：請停止散發你的魅力！

在劇中羅檳和何賽是一對歡喜冤家，兩人是大學同學，從前三集來看何賽是出了名的講原則，身上有一些理想主義色彩，而羅檳做事不僅講效率，身上也是滿滿的銅臭味，兩人所堅持的東西不一樣，所以會經常發生衝突，他倆嗆起來那看點相當有意思，「咬文嚼字」的模樣，讓我來來回回看好幾遍。

其實這部劇可以改改名，應該改名為《偽裝者的前半生》，因為在《精英律師》這部劇裡面，不僅有靳東，還有《偽裝者》中的王鷗，劉敏濤，還有《我的前半生》裡面的袁泉、雷佳音、凌玲，劇中靳東飾演羅賓，王鷗飾演藍紅，正好羅賓是藍紅的前男友，這不就和《偽裝者》的劇情對上了嗎，不過他倆看起來CP感十足，不過最後是不是靳東跟戴曦在一起了？

其實最吸引我的是羅檳精明強幹的女秘書栗娜，這個角色給劇中添了一抹靚麗的色彩，栗娜是朱珠飾演，剛一開播就有了「職場白骨精」和「金牌秘書」的稱號，工作不僅處理的僅僅有條，而且還超極有女人味，剛開播如今到了第五集，就換了五套衣服，感覺有穿不完的裙子。

栗娜的扮演者朱珠是一位公認的才女，她爺爺是一位老紅軍，爸爸是一位成功的商人，良好的家庭氛圍，自幼磨練了她的意志，她以優異的成績畢業於北京工商大學，畢業後因為她的藝術天賦，她先後做過音樂人，電視台節目的主持人，後來進軍影視界，拍了多部的影視作品，特別一提的是，在2011年朱珠簽約美國創新精英文化經紀公司，在美國拍了幾部很有影響力的電影，成為了美國雜誌發表的全球100張美麗面孔中的人物之一。

嫵媚的眼睛和俏皮的梨渦，是朱珠標誌性的特點，身穿V領印花緊身連衣裙，身材凹凸有致，最喜歡就是她的髮型，整整齊齊的波浪長發，女人味十足。

白色連衣裙，走起路來婀娜多姿，白色最考驗人的身材，而朱珠穿起來卻非常好看，破洞的剪裁突出與眾不同的魅力。

栗娜在戴曦沒有來之前，就和羅檳在一起工作多年了，栗娜高高的個子、精緻的五官，嫵媚動人，身材凹凸有致，是一個讓人心動的女孩，估計他背地裡暗戀羅檳，而羅檳應該對她只有工作方面的交流，並沒有其他的意思，不知道在以後的劇情會和戴曦有怎樣的關係？

不過栗娜在律師事務里像模特一樣的穿梭走動，成為了一道美麗的風景。
