熱搜“第一位”！“摳門、不營業”的郭麒麟，終於紅了

2019-12-20 80條評論 3,127次閱讀 1834人點贊

#郭德綱不知郭麒麟參演慶余年#

郭德綱、郭麒麟父子罕見的一同登上熱搜，而且還是第一位！

12月19日，郭麒麟成功用「父親不負責」的言論，將父親大人與自己送上熱搜。

事件的起因，郭麒麟參加活動被問及

記者：在《慶余年》中的精彩表現，有沒有得到父親的認可。

郭麒麟：他都不知道我演過這部戲，因為他從來不看電視劇。

正是這番言論，讓廣大網友朋友開始懷疑「郭麒麟到底是不是郭德綱的親兒子」。（手動捂臉）

雖然大家都知道這是一個「梗」，但是對於這種「包袱」還是大家喜聞樂見的事情。

畢竟，父親與師父是「老搭檔」，且這兩個詞語還都有「相互包含」的意思所在，也難怪網友們翻「包袱」。

其實所言這麼多，真正的含義就是郭麒麟紅了。

這一次不是單純的相聲界，而是成功的紅出了「圈外」，德雲社少班主一躍成為德雲社最具有「演技」的相聲演員。

雖然說有一些繞口，但是大家都明白。

德雲社相聲說得好那是公認的硬實力，但是德雲社演員在影視劇中的地位，基本與趙家班平行，但凡有他們參與過的電影，大抵逃不開「爛片」的命運。

細數有德雲社演員主演的電影，基本都是「撲街」了。

《祖宗十九代》的電影，用全明星陣容也沒有抬起來票房，而德雲社上一部最賣座的電影是于謙的《老師好》，你品、你細品。（細思極恐）

其實德雲社今年算是「黑料連連」，從吳鶴臣的「眾籌」到張雲雷的口誤，從王鶴宇的「被勸離隊」到張番劉銓淼的「被現在叫停」，從秦霄賢「懟觀眾」到張九齡、王九龍大粉涉嫌「坑票錢」，德雲社這一整年都在風雨飄搖中度過。

如何面對粉絲蜂擁而來的「現象級」狀況，德雲社要走的路還有很長很長。

但是，細觀察之下郭麒麟從始至終都巧妙的避開在這些狀況之外。

年初開始，郭麒麟就已經開始走起來「另一路」，憑藉德雲社的資源開始闖蕩影視圈。

因為現如今德雲社並不缺少相聲舞台的號召力，有沒有郭麒麟的參與都顯得並不重要。

德雲社如今開始「捧」徒弟的順序，已經到達了「九字科」，從張九齡與王九龍到尚九熙何九華，就是最好的證明。

而已經成名的弟子也是都有很好的發展，岳雲鵬雖然被調侃成「德雲社前兩年最紅的演員」，但是如今岳雲鵬依舊是德雲社資源最好的徒弟，沒有之一。

張雲雷雖然屢次被全網「大家來找茬」，但是從專場票房到電子雜誌，從代言產品到新歌專輯，成績依舊很可觀。

從此之外，孟鶴堂、張鶴倫、燒餅、「德雲三寶」等專場也都是在如火如荼地進行著，顯然有無郭麒麟的加入，都不算太打緊。

郭麒麟自身年紀也是黃金階段，有著更廣闊的的未來可以去嘗試，且郭麒麟自己活得又很通透，看出相聲就是一場「虛假繁榮」。

所以他選擇去更廣闊的未來，去尋找更好的自己。

這樣的少班主卻是得到了粉絲的喜歡，但是郭麒麟身上同樣有著缺點。

例如「摳門」這一特點，讓很多人懷疑《慶余年》中的「范思轍」就是郭麒麟的本色出演。

摳門就摳門吧，畢竟這是德雲社的歷史遺留問題，每個人都是「摳門、小氣鬼」。

「錢鑲在了腎上面」，是所有粉絲對於德雲社演員的形容。

如果「摳門」算是郭麒麟的第一個缺點，那麼第二個缺點一定就是「不營業」了。

極度不愛更新動態的郭麒麟，很長一段時間都是被迫營業的狀態。

「三分鐘無聲攪餃子餡」、「無茶葉表演泡茶技術」、「只讀問題，不做回答的樹洞問題」、「看嘴型猜大意的無聲演講」。

雖然屢次坑粉絲，但是大家已經被這樣的郭麒麟「洗腦」，除了能完整的觀賞整段視頻，還會對以後出現的視頻有所期待。

《慶余年》一部戲開播，郭麒麟上熱搜不下於五次，這時候大家才真正明白，這為德雲社的少班主他終於紅了，靠著自己精湛的演技，終於成功「出圈」。

人生路漫漫，郭麒麟註定再走一條飛翔之路。

曾經那位讓我們「拱手叫一聲少班主的郭麒麟」，從今往後他的標籤再一次加大。

「德雲社少班主、相聲舞台優秀逗哏、影視劇男演員」
