貧嘴張大民化身明朝太子，蠢笨的樣子讓皇帝王學圻連爆兩次粗口

天天要聞
2019-12-19

昨晚演出的《大明風華》前兩集，看到太子與皇帝在御書房的一段對手戲，覺得非常好笑。

太子和皇帝兩位老戲骨飆演技，一個把太子的蠢笨表演的惟妙惟肖，一個把皇帝怒其不爭的樣子演的十分到位。

皇帝朱棣御駕親征期間，由太子朱高熾監國。皇帝剛一進城，便受到伏擊，又因二皇子在中間挑撥，皇帝決定收回監國之權，還順便檢查一下太子監國期間所有的奏摺。

這把太子嚇的，連連稱不想當太子了，要捲鋪蓋走人。別人勸了半天，也不頂事。如此懦弱的一個人，竟然真的去找皇帝了。

到了皇帝寢宮，兩位老戲骨的對手戲才真正開始。

皇帝讓太子評價一副大老虎和二個老虎崽子的畫，太子仔細看了一眼，說好畫。皇帝一聽太子喜歡，就把畫送給太子了。太子誠惶誠恐，還來不及感謝，皇帝又讓他給畫題上字。太子本來以為收禮是好事，沒想到還收到作業了。

皇帝還是在乎太子感受的，專門和顏悅色的解釋看他奏摺的意思，就是怕太子多想。

誰知道朱高熾打定主意不想當太子了，自說自話，也不管皇帝此時正在說話。倆人你說你的、我說我的，直到皇帝發現不對停了下來，太子才意識到自己搶話了。

得了，皇帝不說了，讓太子說，太子又稱不敢。皇帝嘆了口氣，沒轍，誰讓是自己兒子呢，讓兒子說算了。太子這才拿出奏摺給皇帝看，皇帝接過奏摺時的表情就像是說：「這不會又是來給我添堵的吧！」

果然是。

皇帝看了以後，面上不動聲色，其實已經氣得不行了，發出了靈魂三問：

  • 「這是你的意思，還是老二這個王八蛋的意思？」
  • 「天下是你的嗎？朕要立誰做太子何須你讓位？」
  • 「你是要回順天，躲開朕，培植自己的力量，起兵勤王、把朕逼走。你就這麼盼朕死？！」

皇帝真的氣急了，太子唯唯諾諾不敢搭腔，皇帝又爆出一句粗口：「你他娘的給我說話！」

可惜太子嘴笨，「這個、哪個」了半天、一個字也說不出來，皇帝深呼吸一口氣：「滾！」太子麻溜的準備滾。

走之前還忘了拿畫，又惹皇帝生氣的說：「畫是誰賜你的，就這麼扔在地上？就這麼藐視朕？」

感覺皇帝遲早要被朱高熾氣出毛病來。

這倆人是演的真好。太子是由梁冠華飾演，皇帝是由王學圻飾演。梁冠華面部表情配合肢體語言，把太子小心的樣子、執著的樣子、害怕的樣子、扶不起牆的樣子演活了。

王學圻則利用小動作、小表情，把皇帝深沉的樣子、慈父的樣子、恨鐵不成鋼的樣子、喜怒不形於色的樣子、氣得口吐「蓮花」的樣子，也表現了出來。兩位老戲骨的對手戲實在精彩。

王學圻我們不多介紹了，我們介紹一下不常演戲的梁冠華。

梁冠華很多80後應該都眼熟，他是電視劇《貧嘴張大民的幸福生活》里張大民的扮演者。他是國家一級演員，演市井生活的戲比較多一些，古裝劇有名的要數《神探狄仁傑》系列，塑造了一位明察秋毫的斷案神探。

此次扮演的是在位只有九個多月就突然駕崩的皇帝明仁宗朱高熾。史書記載朱高熾端莊沉靜、喜好讀書，卻是個胖子，重達300多斤。因為肥胖導致身體健康出了問題，在47歲時便駕崩了。他在位時施以仁政，為「仁宣之治」打下了基礎。

形象上，梁冠華和朱高熾非常接近，而要把朱高熾羸弱的狀態演出來，不是一件容易的事。眼神、肢體、做派都要呈現出羸弱的樣子。要知道，因為朱高熾這種狀態，明成祖朱棣還動過換太子的念頭。如今，梁冠華呈現出的太子模樣，確實性格上非常軟弱，耳根子也軟，經不得人說。平時在家不是躺著就是卧著，真挺羸弱的。

這樣的太子，怪不得皇帝會生氣。誰家有這樣扶不上牆的孩子，都得生氣。可他是太子，罵不得、打不得，只能慢慢教導。估計王學圻會因此被太子氣得吐好幾回血吧！

劇中還有很多實力派的演員，期待給我們更多的驚喜。
