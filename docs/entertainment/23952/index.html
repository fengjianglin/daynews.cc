前TVB女星患皮膚癌和乳腺癌獲外籍前夫幫助 因飾演山雞哥女友走紅

2019-12-20 607條評論 4,240次閱讀 1882人點贊

本文編輯劇透社：issac

未經授權嚴禁轉載，發現抄襲者將進行全網投訴

在上個世紀的80年代和90年代，可謂是香港電影的鼎盛時期，因此也有各種類型的電影如同雨後春筍般的湧現出來。像殭屍片、警匪片、喜劇片等類型的電影都受到廣大觀眾的喜愛。不得不說，那個年代的香港電影和香港演員也成為了一代人最為美好的青春回憶。

在這麼多優秀電影中，《古惑仔》系列一定是最為成功和最受歡迎的電影之一。據悉這個系列的電影一共拍攝了14部，由鄭伊健、陳小春、謝天華、錢嘉樂、林曉峰領銜主演。而這其中《古惑仔之人在江湖》這部電影更是經典中的經典。在電影中飾演山雞哥女友可恩的陳秀茹想必觀眾對她一定還有深刻印象。

雖然陳秀茹已經淡出娛樂圈多年，而且已經遠赴美國定居生活，鮮少在公眾面前露面。但期間她還是會時不時的回到曾經熟悉的香港看看家人、看看朋友，所以偶爾也能在陳秀茹圈中好友的社交媒體上看到她的近況。近日她就再一次回到了香港，同一幫好友馬國明、葉暐等人聚餐。只見陳秀茹笑容滿面，狀態極好！

其實陳秀茹當年僅僅只有17歲就報名參加了亞洲小姐的競選，雖然在比賽中輸給了楊恭如等人，三甲都未入，但是還是憑藉她甜美的長相和清純的氣質獲得了高層賞識，順利出道。入行初期陳秀茹更是被外界封為「翻版鍾楚紅」，但經常被公司安排出演一些反叛少女的角色，直到後來加入TVB才有了更好的發展空間。但是在2007年，陳秀茹為愛放棄了自己心愛的演藝事業，與外籍丈夫結婚後一同定居美國。

本以為婚後可以幸福美滿，但因為和外籍丈夫性格不合，因此陳秀茹也最終只能與其分道揚鑣。正所謂禍不單行，正當陳秀茹走出離婚的陰霾時，命運再一次給她開了兩個小小的玩笑。在2015年陳秀茹回到香港探親時，發現自己左胸有個良性水瘤，隨即便到醫院接受了治療。但是在第二年，陳秀茹再次發現自己的乳房處有硬塊，於是再到醫院檢查，被確診為乳腺癌。不僅如此醫生髮現她下巴位置還有皮膚癌，這個結果讓陳秀茹無法接受，一度當場崩潰。

面對病魔，孤苦伶仃的陳秀茹只有勇敢面對。她在美國接受了胸部癌細胞的切除手術，而下巴的皮膚癌醫生則採用了激光將其颳走，所幸也沒留下任何傷疤。手術雖然成功了，但過後的康復生活可謂相當艱苦。早前陳秀茹在接受傳媒採訪時就表示藥物給她帶來的巨大副作用讓她一度狂嘔，可謂非常辛苦。

而抗癌需要的醫療費和康復費對於孤身一人的陳秀茹來說可謂是一筆巨大的開支。但是這個時候陳秀茹的外籍丈夫主動站了出來，為陳秀茹提供了不少經濟上的幫助，前夫的無私幫助也讓陳秀茹在抗癌的道路上得到了心靈的安慰。經過幾年的努力抗癌後，如今陳秀茹的身體也得到了基本的康復，整個狀態恢復得相當理想。

患癌後的陳秀茹整個人也變得更加樂觀，面對病魔她積極面對，不逃避。她更是在康復後將自己的抗癌經歷代入到自己的寫作中，以此來鼓勵更多患癌的朋友。最後小編也祝福堅強、樂觀的她一切都好！早日找到屬於自己的幸福。
