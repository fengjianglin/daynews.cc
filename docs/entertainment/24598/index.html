娛樂圈明星的暗戀史：林俊傑鍾情她15年，易烊千璽的女神想不到

2019-12-23 645條評論 2,711次閱讀 1013人點贊

暗戀其實也是一種愛情，只是很多人愛而不得罷了。暗戀不分老少，除了普通人，明星其實也會在某個階段暗戀某個人。今天小編就來盤點一下明星們曾經或現在的暗戀對象。

白敬亭說過自己的女神是林允兒，來自韓國的一位美女，稱自己在大學期間非常的著迷，在微博上瘋狂粉允兒，是不是這樣的白敬亭非常的搞笑呢？

劉亦菲是無數宅男仰慕的女神，她暗戀的對象就是多才多藝，長相英俊的王力宏。兩人共同出演《戀愛通告》飾演了一回情侶，也算是單方面滿足了劉亦菲的這個遐想。

身為娛樂圈中白馬王子的林俊傑「暗戀」著田馥甄，這事幾乎是眾所周知的了，這麼多年過去，雖然不斷有緋聞，但他15年來心目中的女神都只有一個：田馥甄。真的是好痴情啊！

「國民閨女」關曉彤一直暗戀的對象是李易峰，她曾在多個公共場合，表示自己對李易峰的好感，奈何落花有情，流水無意。

天后張韶涵，出道以來就經歷了很多坎坷，不過好在她一直堅持，最終取得了巨大的成功。在張韶涵年輕的時候，曾經表示過很喜歡彭于晏。

蔡徐坤在節目中曾說過謝娜是他心目中的女神，《快樂大本營》的時候曾說過自己在小的時候就想拜謝娜為師，而現在作為女神的徒弟，相必他很開心吧！

黃曉明的女神是趙薇，而且還公開承認是單方面的暗戀。據說大學期間第一次見面黃曉明就被趙薇的大眼睛長頭髮吸引住了。傳言黃曉明曾經給趙薇告白過三次，但都因為各種原因失敗了。

楊冪在事業起步時曾坦陳很喜歡謝霆鋒，每次看到他都不由自主的心情激動，臉紅到耳根。而且還渾身發抖，不敢與他過多言語。

楊冪暗戀謝霆鋒，也有人也在悄悄暗戀著楊冪，馮紹峰曾在接受媒體採訪時坦言，合作《宮》裡面有點分不清戲裡和戲外，曾經愛過楊冪，對她很有好感。

享有「台灣第一美女」稱號的林志玲曾在多個節目中都示愛黃渤，不過她內心深處真正所期待的男人是劉德華，與劉德華同行的林志玲，內心也會有些小激動。

易烊千璽的女神是朴山多拉，她是2NE1的童顏擔當以及宣傳部長，她有著清爽、平易近人的性格，愛搞笑的她是隊中的開心果，還真是讓人想不到！

吳昕在節目中爆出喜歡的男神非常多，其中暗戀的對象便是李晨，只要是李晨來她的節目，吳昕就會露出害羞的一面。

看到女明星就害羞的張藝興心中的女神是唐嫣。在一次記者採訪中，被問到覺得哪位女明星最漂亮，張藝興一下子就脫口而出：「唐嫣」，笑得很害羞。

上面的明星都沒有和自己心目中的女神（男神）在一起，希望他們以後都可以找到屬於自己的幸福。大家還知道哪些？歡迎留言評論！
