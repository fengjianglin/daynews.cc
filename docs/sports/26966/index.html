德國賽中國選手爆出兩場冷門！年輕小將險勝衛冕冠軍，艾倫被淘汰

2019-12-23 122條評論 1,225次閱讀 70人點贊

2019年斯諾克德國大師賽資格賽在今日開始最終的較量，11位中國選手將陸續出戰對32強的席位發起衝擊，能夠晉級的球員將在一月底前往德國柏林的泰普穹劇院參加正賽，丁俊暉也將和兩屆世錦賽亞軍得主馬修.史蒂文斯爭奪一個正賽席位，這場比賽受到了外界球迷的高度關注。

丁俊暉職業生涯和史蒂文斯交手10次8勝2負，不過在上一次交手中史蒂文斯也是以6-5險勝，但現在的丁俊暉已經找回曾經的狀態，比賽開始後他打出單桿80分和55分連贏四局以4-0率先拿到了賽點，隨後史蒂文斯轟出單桿104分和67分連追兩局，第七局他打出單桿51分以60-5鎖定勝利。

最終丁俊暉拒絕逆轉以5-2戰勝馬修.史蒂文斯晉級正賽，將在1月份參加大師賽、歐洲大師賽和德國大師賽，丁俊暉確定會在國外過年將參加世界台聯在明年1月舉辦的三站賽事。

田鵬飛的對手是世界排名第15位的馬奎爾，之前的歐洲大師賽資格賽田鵬飛擊敗了世界排名第16位的喬佩里，這次面對馬奎爾他也是再次爆出冷門，打出單桿96分、63分和57分以5-3順利晉級，在連續兩站排名賽中都打進了32強，將和丁俊暉一起在國外過年。

年輕小將趙心童挑戰衛冕冠軍凱倫.威爾遜，比賽開始後凱倫.威爾遜打出單桿82分、75分、60分和58分同轟出單桿109分和66分的趙心童戰至4平，決勝局趙心童轟出單桿127分直接單桿制勝，最終以5-4爆冷擊敗威爾遜在連續兩站排名賽都打進32強，袁思俊打出單桿95分以5-3擊敗周躍龍。

李行的對手是四次世錦賽冠軍得主世界排名第六位的希金斯，比賽開始後李行轟出單桿113分、74分和63分連贏3局，隨後希金斯打出單桿68分和64分將比分追至3平，關鍵的第七局李行以66-32贏得纏鬥率先拿到了賽點，隨後希金斯追平比分，決勝局李行在49-4領先的情況下選擇了主動防守，希金斯抓住機會直接清台上演單局逆轉，最終李行功虧一簣4-5不敵希金斯無緣正賽。
