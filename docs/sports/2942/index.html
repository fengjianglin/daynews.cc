北京時間12月2日，國際乒聯世界盃賽已經正式的落下帷幕，來自中國台北18歲的天才少年林昀儒，在比賽當中發揮出色，最終戰勝了中國的奧運冠軍馬龍拿下了季軍。這是林昀儒第1次參加世界盃賽，居然就有如此出色的表現。

林昀儒也成功的創造了全新的歷史紀錄，他成為了歷史上第1次參加世界盃賽，就打進半決賽的最年輕參賽選手。2015年在瑞典的時候，樊振東代表中國隊參加世界盃賽打進半決賽，當時樊振東的年紀是18歲零268天，而林昀儒則是在18歲105天的時候創下了新的紀錄，刷新了樊振東保持的紀錄。張本智和雖然比林昀儒和樊振東都年輕，今年不過才16歲，也打進了世界盃的4強，但他不是第1次參加世界盃了，所以他就無緣創造這樣的紀錄。張本智和在去年就拿到了參加世界盃賽的資格，但最終在1/4決賽當中，總比分1:0領先的情況下被當時37歲的世界冠軍波爾連扳4局4:1淘汰，最終無緣打進四強，不然的話張本智和當時也能夠創下紀錄。

而這也是世界盃賽歷史上，39年來的全新紀錄，林昀儒也成為了中國台北歷史上打進半決賽最年輕的球員。此前中國台北名將庄智淵嘗試了15次，但是從來沒有能夠打進過世界盃賽的八強，蔣鵬龍則在2001年的世乒賽當中打進過四強，林昀儒成了二個打進4強的中國台北球員，但他卻是中國寶島最為年輕的半決賽球員。

對於這位只有18歲的年輕天才，國際乒聯在頭條文章當中對他大加讚賞，稱讚他擁有一隻金手，更確切的說應該是金左手。這是因為林昀儒和中國名將許昕是一樣的，兩個人都是金左手，許昕是左手直板，林昀儒是左手橫板，但是兩個人的技術能力都非常全面。

在短短6天時間裡，林昀儒先是4:0戰勝了中國的世界冠軍林高遠，隨後又以4:3的比分絕殺了奧運冠軍馬龍，這樣的表現足以證明他現在具備了和中國隊分庭抗禮的實力，中國隊絕對不能夠對這名年輕天才掉以輕心了。中國名宿鄧亞萍在t2聯賽期間就對林昀儒大加讚賞，並且提醒劉國梁，一定要重視這名年輕天才，此人會在未來10年內，成為中國隊的心腹大患，劉國梁下一個應該重點研究和針對的就是此人，他比張本智和還全面。而這次比賽鄧亞萍更是呼籲國乒上下要高度重視他，你很少能看到鄧亞萍會對一位來自對手的年輕天才，給予如此高度的評價，看來劉國梁急需要制定方案，好好研究下這位天才少年了。
