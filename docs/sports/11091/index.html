曼聯1500萬菜鳥征服球迷！傳奇門神贊超值簽約，紅魔有他真幸運

2019-12-11 3,453條評論 4,401次閱讀 1186人點贊

對於22歲的丹尼爾-詹姆斯而言，2019年必然是他職業生涯值得銘記的一年。這一年，無論是在曼聯還是威爾士國家隊，詹姆斯一步一個腳印，走得踏踏實實。

當時間指針撥回今年的3月24日，在歐預賽威爾士與斯洛伐克的比賽中，第二次代表紅龍出戰的詹姆斯打進全場唯一進球，助威爾士1-0小勝。

只是在當時，包括曼聯球迷在內絕大多數人，都沒怎麼聽過丹尼爾-詹姆斯的名字。彼時在英冠斯旺西效力的詹姆斯尚未進入更大眾的視線，但他的恩師，威爾士主帥吉格斯相信他是一塊未來璞玉。

在吉格斯的舉薦下，3個月後，曼聯出人意料的以1500萬英鎊簽下丹尼爾-詹姆斯，成為索爾斯克亞入主曼聯後的第一筆簽約。而在簽約之前，詹姆斯的父親不幸因病離世，未能親眼看到兒子身披紅魔戰袍馳騁賽場。

帶著父親未竟的夢想，曼聯新21號主動縮短假期，全力投入到紅魔的新賽季備戰。

8月11日，曼聯第一輪碰上切爾西，比賽第74分鐘詹姆斯替補登場換下佩雷拉，完成加盟曼聯的正賽處子秀。81分鐘詹姆斯接到博格巴傳球抽射破門，首秀7分鐘就敲開進球賬戶，追平主帥索爾斯克亞保持的紀錄。

進球之後詹姆斯雙手指天，告慰父親的在天之靈。

詹姆斯以最快速度融入適應英超，在代表曼聯的前4場比賽中，詹姆斯打進3球。

速度是DJ的最大利器，但詹姆斯也不斷開發自己的新技能，他在反擊中總能送出極有威脅的傳球。對陣利物浦、諾維奇、謝菲聯，詹姆斯用了長傳、傳中等方式為拉什福德送出助攻。若非拉師傅浪費太多機會，DJ助攻數還會更多。另外，詹姆斯的突破還為球隊創造了3個點球。

而在他參加的第一場曼市德比，詹姆斯和馬夏爾右路配合助馬球王進球，完成本賽季個人的第4次助攻。

完場技術統計顯示，詹姆斯打滿90分鐘，21次衝刺全場最高，送出4次威脅傳球本隊最多，8次奪回球權全場最多，其中5次是在防守三區完成。

詹姆斯的比賽態度一如既往的積極和專註，有球時快速推進前場，無球時為隊友扯開空檔，或者不斷騷擾對手持球球員。

曼城主帥瓜迪奧拉賽後表示：「曼聯球員速度太快了，有時你丟了球權就很困難。我們上半場被打了3-4次反擊，他們進攻很有針對性。我們嘗試過壓迫對方，往進攻三區傳球，但他們太能跑了，可能比平常還跑得更多。」

曼聯全隊的優秀表現，詹姆斯的帶動作用不可或缺。

曼聯傳奇門神彼得-舒梅切爾在接受英超官網採訪中盛讚詹姆斯表現「難以置信」，絕對是曼聯今夏的夢幻簽約。

老舒表示：「丹尼爾不會上新聞頭條，但他的表現真的令人難以置信，每球必爭，而且每一次反擊都參與到其中。在曼聯右路他和萬-比薩卡的組合很有威脅，他參加了曼聯每一場比賽，這真的是一筆很棒的簽約。」

而詹姆斯的恩師吉格斯則在解說中開玩笑：「賽前我跟丹尼爾講了， 如果他不能贏下曼市德比，下一次國家隊我就不招他了。」

當然這只是一種激將法，事實上今年威爾士的所有集訓詹姆斯都參加了，明年歐洲杯詹姆斯必然也是主力人選。

詹姆斯已經征服了很多球迷，他現在的價值絕非1500萬英鎊，不過他仍然要繼續證明自己，要繼續逼出自己的潛力。為天堂的父親而戰，為曼聯重回巔峰而戰，DJ加油！
