拒絕大逆轉！陳夢4比3險勝劉詩雯，決勝局打11比1，將對決伊藤

2019-12-13 662條評論 2,360次閱讀 810人點贊

北京時間12月13日，2019年國際乒聯總決賽在鄭州繼續進行，女單1/4決賽，國乒世界第一陳夢對陣世乒賽冠軍劉詩雯，這幾乎是提前上演的決賽。在2比0領先的情況下，陳夢連續輸了3局，第六局，陳夢11比5勝出，兩人進入決勝局。決勝局，陳夢拿下，總比分4比3進入四強，半決賽對陣伊藤美誠。

本次總決賽，陳夢和劉詩雯都是奪冠熱門，不過兩人分在了一個1/4區，簽運並不好。首輪比賽，劉詩雯和陳夢都是順利晉級，8進4的比賽，兩人爭奪一個四強名額。在這個半區，伊藤美誠4比0橫掃佐藤瞳，率先進入四強。

首局比賽，陳夢打得更為主動，11比7拿下。第二局，劉詩雯還是沒能挽回局面，又是一個11比7，陳夢拿到了2比0的領先。從第三局開始，劉詩雯終於找回了節奏，11比9、11比6和11比9，劉詩雯連續贏下了三局，總比分3比2反超。

關鍵的第六局，陳夢開局5比0領先，劉詩雯努力追分到了4比7，陳夢連續拿到3分，10比4拿到了局點，11比5拿下了這一局，總比分3比3平，兩人進入了決勝局。

決勝局的爭奪，陳夢4比0開局，劉詩雯叫暫停。暫停後，陳夢連續得分，9比0領先，劉詩雯追了1分，陳夢11比1拿下這一局，4比3險勝進四強，將對陣伊藤美誠。
