洛馬琴科遇勁敵！22歲天才拳手2回合KO衛冕冠軍，明年或打統一戰

天天要聞
3 天 ago

【文/搏擊江湖】剛剛在美國紐約麥迪遜廣場花園上演的一場12回合IBF輕量級拳王爭霸戰當中，來自美國紐約布魯克林的天才拳手特奧非莫-洛佩茲（15-0，12KO）不負眾望，僅僅2回合就殘暴KO了前IBF輕量級拳王迦納人理查德-康梅，成為新科IBF冠軍。洛佩茲職業戰績提升至15戰全勝12場KO獲勝，賽後他興奮不已，和嬌妻擁吻，而且上演標誌性的後空翻，而且WBC/WBA/WBO輕量級三冠王烏克蘭拳擊天才瓦西里-洛馬琴科也登上拳台，和推廣人鮑勃-阿魯姆一起，站在洛佩茲身旁合影留念，兩位頂級拳王的終極統一戰，或在明年上半年正式打響。從洛佩茲的表現來看，洛馬琴科有些懸了，起碼來說遭遇了一位勁敵。

比賽一開始，兩位重炮手就展開激戰，誰也不吝嗇重拳，第一回合打得很激烈，可以說難分伯仲，洛佩茲的刺拳不俗，衛冕冠軍康梅的反擊也很犀利。比賽似乎隨時都可能出現擊倒場面。

就在比賽的第2回合，洛佩茲和康梅在換拳過程當中，拳速更快的洛佩茲搶得先機，一記兇狠的右手重拳擊中了康梅的下巴，康梅應聲倒地，這一次，康梅遭受了重創。但是衛冕冠軍並不甘心被KO，他被讀秒後頑強再戰，洛佩茲豈能錯過機會，瘋狂的組合拳暴打康梅，康梅只能疲於應付，無法還擊。

康梅幾乎無法打出反擊拳，被洛佩茲的組合拳打得喘不過氣來，而且洛佩茲的拳頭非常重，裁判終止了一邊倒的殺戮。就這樣，22歲的洛佩茲強勢KO了衛冕冠軍康梅，將IBF金腰帶收入囊中。賽後的洛佩茲激動得流下眼淚，他的妻子也哭泣。

上演完後空翻的洛佩茲難以抑制激動的心情，接受採訪時表示願意和洛馬琴科上演終極統一戰，如果他能擊敗洛馬琴科，那麼將成為135磅無可爭辯的霸主。筆者認為，洛佩茲和洛馬琴科的比賽很有懸念，儘管洛馬琴科素以技術精湛著稱，但是洛馬琴科在135磅的體格吃虧，洛佩茲則是天生的135磅重炮手，一旦洛馬琴科挨到洛佩茲的重擊，他還能施展自己的精湛拳法嗎？

可以說洛佩茲是洛馬琴科明年統一戰的勁敵，洛馬琴科完全沒有把握獲勝，這是一場懸念極大的賽事，不排除洛馬琴科遭遇職業生涯第二場失敗。到底明年的這場統一戰何時打響，讓我們期待早日敲定。因為兩人都是頂級公司的拳手，洽談比賽非常容易，而且兩人都渴望一戰。
