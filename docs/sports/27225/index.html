足壇韻事！蘭帕德有多像穆里尼奧？切爾西老闆還喜歡他嗎？

2019-12-21 1,653條評論 4,048次閱讀 1805人點贊

周日熱刺對切爾西的比賽意味著英超聯賽前四名的爭奪將不止三分，而在教練席上發生的事情可能和球場上發生的事情一樣引人入勝。

熱刺在積分榜上排名第五，但只比他們的倫敦對手落後三分，而且還有一個更大的凈勝球差，這意味著何塞·穆里尼奧的球隊可以在擊敗他的一家前俱樂部後進入歐洲冠軍聯賽的席位，而這傢俱樂部現在是由他幫助培養成世界級球員的一位藍軍傳奇人物管理的。

這位葡萄牙人在切爾西的歷史上有著顯赫的地位，在斯坦福橋的兩段時間裡，他帶領球隊獲得了三次聯賽冠軍。

對穆里尼奧來說，與球員的情感紐帶是帶領他們進入下一個階段的關鍵。蘭帕德透露穆里尼奧在2004年和他的第一次季前賽中告訴他他是「世界上最好的球員」。

一年後，蘭帕德在金球獎的排名中僅次於羅納爾迪尼奧，位居第二，這在很大程度上要歸功於他在幫助切爾西結束50年之久的等待中所發揮的作用。

蘭帕德沒有回頭。穆里尼奧灌輸給他的信念是不可動搖的，他帶領切爾西在2012年的冠軍聯賽決賽中戰勝了拜仁慕尼黑。

作為一名主教練，蘭帕德現在正試圖給他的年輕切爾西球員同樣的信心。和穆里尼奧一樣，蘭帕德的大部分戰術都集中在人員管理上，儘管他給人的印象更富有同情心。他和穆里尼奧一樣憎恨輸球，同時他也喜歡在戰術體系多樣化的情況下設置來抵消對手的優勢。

然而，儘管蘭帕德承認他已經從這位兩屆歐冠冠軍身上學到了很多，但每當這個話題出現時，他總是堅持說他是一個獨立的人。

儘管如此，穆里尼奧仍然和他在切爾西的教練們保持著緊密的聯繫。蘭帕德從未忘記這樣一個事實:即使在穆里尼奧離開切爾西加盟國米之後，教練也經常在他母親帕特里夏2008年去世後給他打電話。

穆里尼奧也打電話祝賀蘭帕德的女兒的出生，就在幾個月前，德比在他們作為主教練的第一次會面中把他的曼聯踢出了卡拉巴杯。

不過，這兩個人也有過起起落落。事實上，是穆里尼奧在2014年告訴蘭帕德他在斯坦福橋的日子已經結束了，因為切爾西拒絕延長他的合同一年。

這並不是他在藍軍生涯的終點，蘭帕德認為他是罪有應得的，他準備轉會到紐約城市足球俱樂部，後者隨後選擇將他租借給他們的合作夥伴曼城，他在對陣切爾西的比賽中打進了著名的一球。

這感覺像是一種報復，尤其是當穆里尼奧說到蘭帕德轉會到阿提哈德時:「當他決定去一個直接的競爭對手，那麼愛情故事就結束了。」

具有諷刺意味的是，穆里尼奧將會簽約另外兩個對手，曼聯和熱刺，這兩個對手已經玷污了他與切爾西球迷的關係。

蘭帕德對穆里尼奧在賽季第一天在老特拉福德0-4輸給曼聯的表現也不滿意。作為天空體育的權威，穆里尼奧認為藍軍「太軟弱了」，並批評了他的首次亮相。

這兩者之間還有另一個相似之處:在保護球員方面，蘭帕德帶有一些穆里尼奧著名的爭強好勝和他的厚顏無恥的幽默感。

蘭帕德開玩笑說，他很高興穆里尼奧一個月前接手了熱刺隊的工作。對蘭帕德和穆里尼奧關係的下一個考驗將在北倫敦的電視攝像機和62000名球迷面前上演。這兩人都是競爭對手，他們試圖在過去的感情和現在渴望在英國最激烈的德比比賽中獲勝之間找到平衡。。他們的邊線之戰會讓觀眾欲罷不能。
