又黑國足！韓媒：中國球員不是來踢球的，他們想殺人

2019-12-11 285條評論 3,051次閱讀 438人點贊

東亞杯首場比賽，中國國足1-2不敵日本，取得了「開門黑」。除了輸球，國足後衛姜至鵬因為在比賽中飛踢橋岡大樹的頭部，在賽後也引起了軒然大波。不僅日本媒體對姜至鵬口誅筆伐，甚至連看熱鬧不嫌事大的韓國媒體，也開始趁機又「黑」了國足一把。

​

韓國媒體MT撰文表示：「中國和日本之間上演了一場激烈的比賽，從一開始就犯規不斷。不過隨著比賽的進程，日本人憑藉更出色的技戰術能力佔據了主動，而中國球員別無他法，只能全力防守。從中國隊的立場來看，這樣的場面顯然讓人非常失望。」

​

「就在這時候，中國球員開始頻繁上演像是殺人一般的鏟球動作。他們的目標不是皮球，而是直奔對手的腳踝而去。尤其是姜至鵬，他在一次粗野犯規後又猛踢橋岡大樹的頭部，這樣的動作一不小心就會造成嚴重的傷害，難怪日本替補席上會爆發出猛烈地抗議。隨後，中國球員明天又上演了一次粗暴鏟球，甚至比賽現場都因為這個犯規爆發出了一陣嘆息聲。」

​

韓國媒體強調，這支中國隊的平均年齡，比日本隊大了整整五歲，但是體現在比賽中，卻沒有展現出任何大齡球員應有的優勢。「中國隊由年齡比日本大的選手組成，因此預計，在比賽開始前，中國隊和日本應該是旗鼓相當的。但實際開戰後發現卻有所不同。中國只能用不斷的犯規阻止對手，觀眾們看的也是大皺眉頭。」

從慢鏡頭回放來看，姜至鵬對橋岡大樹的犯規動作確實夠狠，主裁判僅出示黃牌，顯然是手下留情了。不過韓國媒體不需要幸災樂禍，因為很快，作為東亞杯東道主的韓國國家隊就能與中國國足交手，屆時，我們期待國足能讓韓國人感受到我們的怒火。
