不愧巴薩大佬！1手勢為皇馬隊長仗義解圍 9萬人心服口服

天天要聞
2019-12-20

西甲國家德比首回合，巴薩和皇馬踢成0-0，雙方各取一分。儘管這場比賽已經過去，但許多有意思的細節也逐漸被翻出，據《世界體育報》透露，巴薩9萬多名球迷本來狂噓拉莫斯，但巴薩大佬皮克仗義出手，示意球迷不準辱罵皇馬隊長。

眾所周知，皮克是堅定的「反皇馬鬥士」，過去不僅多次出言嘲諷，並且還用手勢羞辱過死敵。10/11賽季穆里尼奧執教皇馬的首場國家德比，巴薩在主場5-0狂勝皇馬，賽後皮克向諾坎普的觀眾張開巴掌，亮出了大名鼎鼎的「五指山」，示意巴薩進了皇馬5個！

上賽季國家德比，巴薩時隔9年再次在諾坎普吊打皇馬，一場5-1讓銀河戰艦顏面掃地。當比達爾打進第5球的時候，皮克再次向諾坎普亮出了經典五指山，這是對皇馬的又一次狠狠的羞辱。只要有機會，皮克總會在場上不留情面地打擊皇馬。

不過，皮克同樣是一個有情有義之人，剛剛結束的國家德比，拉莫斯獲得諾坎普9.3萬名主場觀眾特殊「照顧」，每當皇馬隊長觸球時，巴薩球迷總會對他報以猛烈的噓聲。

隨著比賽深入，球迷的行為也變得過分。據《世界體育報》透露，拉莫斯聽到了巴薩球迷對他的辱罵，關鍵時刻皮克站了出來，他張開了手，並且要求球迷停下來，這一幕正好被攝像機完整拍攝下來。

《世界體育報》還透露，皮克不是第一次阻止巴薩球迷辱罵拉莫斯，去年10月的國家德比，皮克曾經伸出食指，指著那些對拉莫斯進行辱罵的球迷說不。

皮克曾經表示，他和拉莫斯各為其主，一個代表巴薩，一個代表皇馬，自然是有不同的立場。但是，兩人始終保持著對彼此足夠的尊重，在國家隊合作的時候也相當愉快。皮克還透露，自己跟部分皇馬球員關係不錯，甚至和皇馬主席弗洛倫蒂諾都有交情。

對於皮克的仗義之舉，球迷表示巴薩大佬好樣的，關鍵時刻證明了自己的人品他不僅為皇馬隊長拉莫斯解了圍，更重要的是展示了巴薩作為一家百年豪門應有的風度和底蘊。
