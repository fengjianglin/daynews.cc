紅軍官宣世俱杯大名單！世界盃成員意外落選，主力鐵衛傷情嚴峻

2019-12-07 0條評論 23次閱讀 0人點贊

摘要

昨天下午的時候，紅軍在俱樂部官方網站上公布了球隊參加世俱杯的比賽大名單。有傷在身的馬蒂普和法比尼奧都落選了球隊前往卡達的大名單，原本媒體認為馬蒂普只是6周左右的傷病，但是從這次落選大名單來看，馬蒂普的傷情可能比預想中的要更加糟糕。

昨天下午的時候，紅軍在俱樂部官方網站上公布了球隊參加世俱杯的比賽大名單。

有傷在身的馬蒂普和法比尼奧都落選了球隊前往卡達的大名單，原本媒體認為馬蒂普只是6周左右的傷病，但是從這次落選大名單來看，馬蒂普的傷情可能比預想中的要更加糟糕。

進入世俱杯名單的23名球員分別是：

門將：阿利松、阿德里安、洛納根；

後衛：范迪克、洛夫倫、喬-戈麥斯、阿諾德、羅伯遜、威廉姆斯；

中場：維納爾杜姆、米爾納、凱塔、亨德森、拉拉納、張伯倫、沙奇里、柯蒂斯-瓊斯；

前鋒：菲爾米諾、馬內、薩拉赫、布魯斯特、奧里吉、埃利奧特。

一線隊名單中落選的球員有4名：克萊因因為長期傷病的原因新賽季甚至還沒有進入任何一場比賽的大名單，目前沒有關於他傷病情況的最新消息；門將方面渣叔選擇了老將洛納根而不是年輕的凱勒赫，可能更多是出於讓小將在聯賽杯當中得到更多鍛煉機會的原因。

被視為左後衛頭號替補的拉魯西也未能入選世俱杯比賽大名單，今年因傷缺席了一段時間的年輕人已經在預備隊中逐漸找回了自己的首發位置，此番渣叔也選擇將他留在國內征戰聯賽杯的比賽。

這樣一來，羅伯遜在世俱杯的比賽中將沒有主要踢這個位置的替補，看來老將米爾納在需要人手頂上的時候又要客串到這個位置上了。世俱杯的比賽強度比起英超和歐冠總體來說不算太大，不排除首場比賽渣叔會先用一套偏輪換球員為主的陣容出賽，給到大部分主力休養以及減緩飛行疲勞的機會。

右後衛位置的替補人選渣叔啟用了威廉姆斯，這位年輕球員在個人代表紅軍的首場一線隊正式比賽就是聯賽杯對陣阿森納，他在最後時刻送出絕平助攻，幫助奧里吉完成凌空側勾的精彩進球，進而將比賽拖進點球大戰並最終淘汰了阿森納。

這倒是有點出乎球迷們的意料，荷蘭U17世界盃主力成員赫維爾的落選讓人感到有點驚訝，不過仔細思考這個選擇其實也很在理：右後衛的位置上已經有了阿諾德和戈麥斯兩名球員，赫維爾能夠競爭上崗的機會其實並不大，還不如將他留在英國國內在聯賽杯與維拉的比賽中鍛煉一下來得更加有價值。

今夏紅軍的首筆引援范登博格也沒能進入世俱杯比賽大名單，對比同樣是今夏加盟的埃利奧特來說，范登博格至今的表現顯然還沒有達到球迷們的預期，好在他還年輕，趁著這次聯賽杯的機會，他會為自己打出優秀表現，爭取在一線隊得到更多的鍛煉時間嗎？

世俱杯參賽名單人選並沒有多大的意外，倒是相比之下紅軍球迷可能更加關注聯賽杯的比賽大名單，預備隊中除了埃利奧特、瓊斯這兩名常規主力被徵召到一線隊參加世俱杯之外其實並沒有其他更多的人員空缺。

現在聯賽杯對戰維拉的可選擇球員除了上面提到的幾位一線隊落選球員之外，還有像奇日維亞、傑克-凱恩、加拉徹爾、克雷頓、邦納爾、朗斯塔夫等一眾年輕好手，因此在湊足比賽大名單和上場球員方面完全不成問題。

至於大家很關注的青年鋒霸保羅-格拉策爾，因為在季前賽中受傷，至今仍然無法參加球隊的正常訓練，他的傷勢最快也要等到明年才有最新的進展。

最後還要補充一句，在比賽開始之前的24小時，世俱杯參賽球隊還可以更換現有大名單中的球員。但是還是不希望看到紅軍被迫更換球員的消息，因為這通常意味著傷病，世俱杯回來還有聖誕新年快車期的比賽，遠離主場作戰的紅軍球員千萬要防止傷病啊。
