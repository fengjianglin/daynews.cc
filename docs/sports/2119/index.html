場均得分7.6，美國二傳不簡單發扣攔樣樣俱佳，基拉里一定後悔了

2019-12-09 0條評論 28次閱讀 0人點贊

摘要

北京時間12月9日訊，女排世俱杯昨晚落幕，來自義大利的科內利亞諾女排創造新隊史，完成世俱杯的登頂任務。同樣來自意甲的諾瓦拉，在季軍爭奪戰中，不敵朱婷老東家瓦基弗銀行，無緣領獎台。但是經過這一周的比賽，諾瓦拉隊也給球迷們留下了深刻的印象。

北京時間12月9日訊，女排世俱杯昨晚落幕，來自義大利的科內利亞諾女排創造新隊史，完成世俱杯的登頂任務。同樣來自意甲的諾瓦拉，在季軍爭奪戰中，不敵朱婷老東家瓦基弗銀行，無緣領獎台。但是經過這一周的比賽，諾瓦拉隊也給球迷們留下了深刻的印象。如果要說這個隊伍當中，誰的表現最驚艷，那麼一定是來自美國的二傳漢考克。

這個奧運周期，漢考克一直有入圍美國女排大名單。不過距離主二傳的位置一直差一口氣。從去年的世錦賽到今年的國家聯賽，她是勞埃德和波爾特的替補。而到了女排世界盃之前，卻被主教練基拉里移出了參賽名單，遺憾落選。這次世俱杯，可以說是漢考克的正名之戰。和中國女排的丁霞一樣，她也是一個左手將，善於二次暴扣進攻。

作為一名二傳手，漢考克在本屆世俱杯所有五場比賽當中一共砍下38分，場均高達7.6分。發扣攔三大得分手段，一樣不差場場有分入賬。特別是小組賽對陣瓦基弗銀行，諾瓦拉最終鏖戰五局取勝，漢考克功不可沒。那場比賽，她的個人得分為11分，並且榮膺單場MVP。更不得了的是，五場比賽中，她有三場得分超過了副攻兼隊長齊里切拉。這樣攻擊屬性值爆表的球員，居然會被國家隊遺棄，實在是捉摸不透基拉里的心思。

有球迷說，漢考克雖然發扣攔樣樣俱佳，但是在二傳的本職工作方面並不出色。這話看來也是有點道理，對比科內的沃洛茲，漢考克的傳球風格不是很符合美國女排的「光速體系」要求。可如果對比美國女排主打世界盃的兩名二傳，漢考克也不見得差距特別大。尤其是和卡爾利尼比較，漢考克「立高球」的能力還是更勝一籌。同一個位置選用不同風格的球員，是一個比較好的選擇。

今年的女排世界盃，美國隊被中國女排以3-0橫掃，完成了世界三大賽對陣中國的三連敗。那場比賽，美國女排輸在哪裡？輸在發接發環節。美國女排的戰術體系對於一傳的依賴度非常高，一旦一傳崩潰，網口被重點施壓，就會變得十分被動。假設基拉里當時把漢考克帶上，哪怕作為一名發球替補，利用她的大力跳發，也會給中國女排造成不小的麻煩。那麼至於那場比賽的結果，至少不太可能是脆敗。一想到這裡，基拉里可能已經後悔當初的決定了。

作為國家隊的主教練，基拉里一定會關注球員在俱樂部的表現。今年新提拔的主二傳波爾特，還只是22歲的小將，國際比賽經驗不多。她能不能在頂級的奧運會賽場經受住考驗，基拉里心裡肯定也會打鼓。漢考克作為一名27歲的成熟球員，或許更能在關鍵時刻頂得住。那麼對於中國女排來說，也不得不提防一下這個左手二傳的發球和進攻了。
