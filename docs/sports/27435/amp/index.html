列維：穆里尼奧不會進攻？看看熱刺進球數 不會給他下達奪冠任務

天天要聞
2019-12-20

倫敦標準旗幟晚報繼續刊登了對熱刺主席列維的專訪，這一次列維談到了主帥的問題。列維說穆里尼奧一直都是熱刺新新主帥頭號人選，列維還反駁了對外界對穆里尼奧不會進攻的質疑。談到冠軍問題，列維強調，熱刺一定會全力爭取冠軍，但不會以是否獲得冠軍來衡量穆里尼奧的工作，也不會給穆里尼奧下達任何具體奪冠指標。

倫敦標準旗幟晚報報道截屏

在穆里尼奧接受熱刺之後，英超和歐冠踢了7場比賽，其中英超4勝1負，將落後英超前四11分的情況下，已經追到距離第四名切爾西只有3分的差距。在這周末的比賽中，只要熱刺擊敗切爾西，那麼就將壓倒對手進入前四。在英超的5場比賽中，穆里尼奧的熱刺打入了14球。

列維說外界對於穆里尼奧不會踢進攻足球的說法進行了駁斥，「他不會踢進攻足球？我們這段時間的進球數量就是最好的證明，我們現在的足球風格還是讓人看得非常激動的。」其次，外界一直認為穆里尼奧不會使用年輕人，壓制球隊青訓。對此，列維同樣進行了反駁。「穆里尼奧不重視青訓？外界一直有這個說法，但我認為這不是事實，隨著賽季的進入，我相信熱刺很多年輕球員都會得到出場的機會。」

談到穆里尼奧執教一支球隊總是以三年作為節點的說法，列維表示穆里尼奧是一個非常頂級的主教練，熱刺同樣也是一家大俱樂部。「對的時間來到對的地點，遇見對的人，我們彼此都是合適的。」列維說。

熱刺已經11年沒有拿過任何賽事的冠軍，國內最重要的獎盃獎項更是要追溯到1991年的足總杯，列維堅持說，在穆里尼奧的帶領下，他不會以是否獲得冠軍來決定穆里尼奧工作是否成功。「我從來沒有對穆里尼奧說過，我們需要在某一個時間節點，取得什麼樣的成績。我們需要發揮出現有球員的最大實力，當然我們都希望贏。但是英超競爭實在太激烈了，不管我們多麼努力，我們都無法保證一定會勝利。但是勝利和冠軍永遠是我們的目標，但我不會給任何人下達一個具體的目標。」列維說。
