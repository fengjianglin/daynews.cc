男羽年終排名：諶龍升第3石宇奇跌第8 林丹僅第17擺脫TOP16禁令

2019-12-17 487條評論 4,486次閱讀 2054人點贊

伴隨2019年世界羽聯年終總決賽落幕，羽壇男單的年終排名也是出爐，國羽排名最高的是奧運冠軍諶龍升至第3名，無緣總決賽的石宇奇跌至第8名，36歲的林丹僅第17名，也意味著他明年徹底擺脫TOP16禁令，可以無限制參加低級別賽事搶奪奧運積分。

男單年終世界第1的排名沒有任何懸念，日本一哥桃田賢斗以絕對優勢繼續霸佔榜首位置。桃田賢斗在總決賽時隔4年再次奪冠，也是他今年第11冠，從而超越李宗偉保持的單賽季10冠紀錄，還在今年成為日本首位奪得世界羽聯年度最佳男球員獎項的選手。桃田賢斗以111918分強勢領跑，繼續穩坐年終第1的寶座，也是領先第2名的周天成超過3萬分。

世界第2依然是中國台北一哥周天成，但他僅僅只領先身後的諶龍不到1000分的微弱差距，周天成在總決賽因為安賽龍的退賽，導致無奈以2勝1負戰績無緣晉級四強。

國羽奧運冠軍諶龍哪怕在總決賽止步四強，他在半決賽被印尼一哥金廷橫掃淘汰，但諶龍的積分依然提升到79640分，也是超越丹麥名將安東森，升至世界第3名。只是諶龍本賽季僅奪得法國賽一站男單冠軍，無疑他需要在2020年奧運年繼續努力。

丹麥雙星安東森與安賽龍分列第4和第5名，他們均是在總決賽止步小組賽。再往後，則是印尼雙星喬納坦與金廷分列第6和第7名，其中金廷在總決賽收穫男單亞軍，險些就擊敗桃田賢斗，但最終慘遭大逆轉丟冠。

國羽新一哥石宇奇下跌至第8名，他在受傷前曾經高居世界第2名，但因傷缺席長達3個多月的比賽，外加復出後整體戰績不佳，並且尚未完全康復，讓石宇奇已經跌至世界第8名。石宇奇能否在奧運年殺回來，自然也是令人期待之事。

第9名與第10名分別是中國香港一哥伍家朗與日本2號男單主力常山干太。

國羽老牌天王林丹則是已經跌至世界第17名，他目前的積分是49470分，距離世界第10的常山干太相差7628分。當然，林丹早就已經跌出世界前十，且如今還跌出世界第前16，對於林丹無疑也算是好事。

林丹在2018年受限於TOP16禁令，按照世界羽聯規定排名前16的選手只能參加3站超級500以下級別的賽事，多出來的參賽再參加將不能得到積分。正是基於如此因素，林丹在韓國大師賽拿到男單亞軍，卻也沒有能夠收穫任何積分。

林丹在2020年擺脫TOP16禁令後，他可以無限制參加低級別的賽事，無疑有助於林丹去搶奪奧運積分，從而去衝擊生涯第5次奧運之旅。
