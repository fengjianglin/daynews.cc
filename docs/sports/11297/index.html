中國拳手世界頭銜戰從未贏過名將？楊永強望破魔咒，他實力被低估

2019-12-10 828條評論 2,115次閱讀 766人點贊

有人提出中國職業拳擊在世界頭銜戰從未贏過真正的世界名將，對此我竟很難反駁，實際情況確實如此，儘管我並不願意承認。不過，這個尷尬紀錄很有希望被楊永強打破，他是真正的實力派拳手，明年2月他將在海口挑戰美國海軍陸戰隊傳奇拳王赫林。

我國只有三位拳手在世界頭銜戰里贏過，他們的世界戰對手網友普遍都沒聽說過（專業拳迷除外）。日本拳王木村翔是因為KO鄒市明才出名的，儘管他確實體能變態，拳頭很重，但在賽前幾乎沒有名氣。徐燦兩次衛冕戰對手較弱，比如羅伯斯四大組織綜合世界排名第72，還被噴子質疑。熊朝忠其實打過一個實力很不錯的對手奎洛，但由於是體重最低的草量級，這個級別幾乎所有拳手都沒有什麼名氣。

接下來，中國兩位拳擊勇士要對陣真正名望很高的世界名將，先是M23戰隊的卓越拳手烏蘭對陣日本名將田中恆成，接著就是楊永強對陣美國海軍傳奇赫林。田中恆成在日本雖然不是家喻戶曉，但也積累了不少粉絲，絕對算得上超人氣拳王，烏蘭勇氣可嘉，但客場挑戰田中勝算微乎其微。而楊永強死磕赫林倒是有至少四成把握。

對於楊永強，體育專家趙佳興給予高度評價：「他被拳迷們嚴重低估了。不少中國拳擊與格鬥迷們其實非常膚淺，只會人云亦云，他們沒有獨立的觀察力和判斷力，完全是一犬吠影百犬吠聲。比如張偉麗拿UFC世界冠軍了，都在盛讚張偉麗厲害。但張偉麗前兩年打崑崙決的時候，又有多少人看好她呢？當時很多網友還在信誓旦旦的說UFC如何如何難打，在崑崙決無敵的張偉麗不可能在UFC取得突破。所以，現在拳迷們對楊永強知之甚少，並不看好他，但用不了多久楊永強贏得金腰帶，他們就會膜拜楊永強，成為楊永強鐵粉，假裝很久以前就已經支持楊永強了。」

「我現在不好說楊永強比誰誰誰厲害，因為那樣肯定會被噴，因為錦標主義深入人心，你拿一個沒有世界頭銜的拳手去跟世界冠軍比，那隻會招來謾罵。因為網友們看不出誰強誰弱，只能通過金腰帶來判斷。我要說的是，楊永強的拳真的非常重，他和帕奎奧曾經長期一起訓練，風格有一點兒像帕奎奧，這在國內拳手裡是非常難得的。」趙佳興補充道。

那麼楊永強是現役最出色的中國職業拳手嗎？趙佳興不這麼認為：「可不能這麼說，其實這些職業拳手都知道，職業拳手和體制內拳手是完全不同的。職業拳擊是可以通過推廣公司合法運作的，跟誰打，什麼時候打，在哪打，都可以談，甚至還可以花讓路費讓別人讓路。奧運會、世錦賽可以運作嗎？一名拳手贏得世界金腰帶有多種因素，還是讓我們祝福楊永強吧。」
