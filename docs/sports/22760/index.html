巴薩VS皇馬前瞻：皇馬全靠一人抗衡梅西 聯賽榜首之爭+金靴之爭

2019-12-18 2,422條評論 3,595次閱讀 1249人點贊

北京時間12月19日凌晨03時，2019-2020賽季西甲聯賽第10輪迎來一場補賽，巴塞羅那坐鎮諾坎普球場迎戰皇馬。本場比賽，將是西甲榜首爭奪戰和金靴爭奪戰，國家德比勝者將佔據榜首。
雙方最近6次交鋒，巴塞羅那4勝2平保持不敗，皇馬上一次在國家德比戰中取勝，還要追溯到2017年8月西班牙超級盃，當時主場2-0完勝，並對巴塞羅那完成雙殺，總比分5-1奪冠，需要指出的是，這也是皇馬最近10次國家德比戰中唯有的2場勝利。不過，在諾坎普球場，皇馬近年來則是勝多負少，最近6次做客巴塞羅那僅輸1場比賽，其餘5戰取得2勝3平。事實上，最近10次在諾坎普球場進行的國家德比，巴塞羅那和皇馬的戰績平分秋色，雙方各贏了3場比賽。
這場國家德比，本來是在10月底進行，由於加泰羅尼亞地區的政治局勢不穩定，被迫延期，進而讓中國球迷受到影響，本來是在晚上的黃金時間進行，如今變成在深夜凌晨展開較量。上周末，兩支球隊都迎來了一場西甲賽事，戰績都非常一般，巴塞羅那客場2-2艱難戰平皇家社會，正賽6連勝被終結，但過去8場6勝2平的不敗戰績還在延續。巴塞羅那上一次輸球是在11月初，當時聯賽客場1-3不敵萊萬特，這也是加泰羅尼亞豪門過去16場正賽唯一一場失利的賽事。

皇馬上周末聯賽是依靠著補時階段絕平，客場1-1險平瓦倫西亞，正賽三連勝被終結，但11場8勝3平不敗戰績還在延續，上一次輸球是在10月下旬，當時客場0-1不敵馬略卡，這也是銀河戰艦過去17場正賽唯一一場失利。從近期戰績來看，無論是巴塞羅那，還是皇馬，競技狀態都很不錯。
在諾坎普球場，巴塞羅那已經保持一年不敗，上一次輸球是在去年11月，當時主場3-4負於貝蒂斯，此後27個主場取得23勝4平不敗，期間一度豪奪16連勝。由此可見，巴塞羅那在諾坎普球場的統治力。不過，皇馬近期的客場成績也不錯，過去5場比賽取得4勝1平，最近8場僅輸1場比賽。
本場比賽，巴塞羅那中場阿圖爾和前鋒登貝萊繼續因傷缺陣，塞梅多如果出場，那麼將迎來紅藍軍團生涯第100場比賽，若無意外，蘇亞雷斯、梅西和格列茲曼組成的MSG三叉戟將繼續先發。需要指出的是，蘇亞雷斯聯賽主場對陣皇馬，經常有不錯的發揮，5戰打進6球，其中包括上賽季5-1橫掃取勝，那場比賽的結果，直接讓洛佩特吉下課。算上其他賽事的進球，蘇亞雷斯在國家德比賽場上進球數達到12個，排在這項賽事歷史射手榜第八位。不過，梅西在國家德比進球數更多，自出道以來，踢皇馬打進了26球，排在國家德比歷史射手榜第一位，比第二名C羅高出8球。本賽季，梅西延續著出色的競技狀態，各項賽事出場16次，打進14球助攻9次。

相比起巴塞羅那，皇馬傷病嚴重，阿扎爾、馬塞洛、阿森西奧和巴斯克斯都將因傷缺陣，若無意外，納喬將頂替馬塞洛的位置繼續首發，中後衛的位置由拉莫斯搭檔瓦拉內，貝爾有望獲得先發機會，本澤馬則是銀河戰艦進球的保證，本賽季他各項賽事出場21次，打進16球助攻7次，是隊內頭號得分手。
這場國家德比，將是榜首爭奪戰，17輪戰罷，兩支球隊少賽一場的情況下，均拿到35分，巴塞羅那取得11勝2平3負，皇馬取得10勝5平1負，加泰羅尼亞豪門暫時以凈勝球的優勢排在榜首。需要指出的是，巴塞羅那和皇馬上一次在國家德比前同分是在2009-2010賽季，當時紅藍軍團客場2-0取勝，並最終奪得西甲冠軍。時隔9年，巴塞羅那再次與皇馬在西甲賽場同分較量。此外，這還是西甲金靴之爭，目前梅西和本澤馬均以12球，排在聯賽射手榜首位。
賠率方面。巴塞羅那主場贏球的賠率為1賠1.75，皇馬客場贏球的賠率為1賠4.20，雙方戰平的賠率為1賠4.30；亞盤方面，博彩公司開出了主隊讓半一的盤口，巴塞羅那1.96水位，皇馬1.97水位。波膽賠率，巴塞羅那主場贏球最有可能的比分是2-1，賠率為1賠8.5，2-0和1-0的賠率皆為1賠11，3-1的賠率為1賠12.5，3-0的賠率為1賠16.5，3-2的賠率為1賠18.5；皇馬客場贏球最有可能的比分是2-1，賠率為1賠14.5，1-0的賠率為1賠18.5；平局方面，1-1的賠率為1賠8.8，2-2的賠率為1賠13。球員方面，梅西的進球賠率為1賠1.62，蘇亞雷斯的進球賠率為1賠2.2，法蒂和格列茲曼的進球賠率皆為1賠2.9，本澤馬的進球賠率為1賠2.3，約維奇的進球賠率為1賠2.8，貝爾和羅德里戈的進球賠率為1賠3.3。
