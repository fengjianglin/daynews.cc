逆天彈跳堪比喬丹 外媒曝光C羅越老越妖的“秘密武器”

2019-12-22 462條評論 2,354次閱讀 258人點贊

參考消息網12月21日報道 外媒稱，在18日進行的意甲尤文圖斯對陣桑普多利亞的比賽中，C羅打進了一枚驚天「高空」頭槌讓這場普通的比賽變得不平凡。這個離地2米56的滯空頭槌的背後到底有什麼秘密？

據英國《太陽報》12月19日報道，隊友在邊路起球後，尤文圖斯前鋒C羅縱身彈跳至2米56的高空，以「高人一等」的方式打進了一枚漂亮的頭槌。人們甚至將C羅的這一「飛」與NBA傳奇人物邁克爾·喬丹相提並論。

報道稱，在2018年的採訪中C羅曾說過自己的「生理年齡是23歲」，當時實際年齡33歲的他說：「我還有很長的路要走，我可以一直打到41歲。」

C羅保持如此狀態的方式不是什麼秘密。健身、休養結合、嚴格的控制飲食……都是他的「秘密武器」之一。

作為健身房常客，C羅熱衷於力量訓練。比賽結束後他通常直接走進泳池，游泳不僅能幫助他從一場比賽的體力消耗中解脫出來，還能再次鍛煉他的手臂、核心肌肉和腿部肌肉。

除了健身房，保證肌肉狀態甚至肌肉再生的好方法是冷凍治療儀。據悉，這件價值5萬英鎊的裝備的工作原理是站在艙內大約3分鐘，液氮會將人體置於零下160到200攝氏度之間。

冷凍療法可以消除肌肉酸痛，減少腫脹、炎症或扭傷——讓C羅更快地為下一場比賽或訓練做好準備。

C羅的日常訓練一部分是關於肌肉爆發力的，他經常在家完成普拉提訓練，在晚上睡前或清早醒來後進行腹肌鍛煉。「如果你養成了一種習慣，就會變得更容易，」C羅補充道。

但這並不是說C羅的一天都在訓練或者訓練的路上，實際上，休息對他來講也十分重要。C羅稱他每天要保證8個小時睡眠時間。此外，和家人的休閑度假時光更是不會減少，C羅解釋說，「精神力量和身體力量一樣重要。」

除了這些，飲食上的自律更為關鍵。不以蔬菜為主，但也不攝取更多的肉類，魚是日常飲食主要組成部分，少食多餐。C羅曾說過：「我吃高蛋白的飲食，含有大量的全穀物碳水化合物，水果和蔬菜，避免含糖的食物。經常這樣吃，如果你經常訓練，保證能量很重要，這樣可以有更好的表現力。為了確保有足夠的能量來達到最高訓練水平，我會選擇少食多餐的飲食方式，有時我會一天吃六頓飯。」

此外，從2005年起，C羅就滴酒不沾了，這不僅是因為戒酒是飲食習慣中的一環，還因為C羅的父親死於與酒精有關的疾病。
