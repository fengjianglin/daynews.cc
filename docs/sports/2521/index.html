巴薩內部出亂子，球員隊醫之間信任崩塌，巴爾韋德也成傀儡

2019-12-06 0條評論 21次閱讀 0人點贊

摘要

本賽季大家發現，巴薩的傷病狀況比以前真的多了不少。一個球隊出現傷病是很正常的事情，但是頻繁出現，只能說明這個隊伍的訓練有問題。現在不只是登貝萊這樣的新銳，就是梅西和蘇亞雷斯這樣的老球員也出現這樣的情況，此時是不是該讓巴爾韋德站出來負責呢？

尷尬的是，最近受傷的登貝萊和烏姆蒂蒂，他們2個人竟然直接回到法國尋求醫療。對於球員們的這個舉動，加泰當地的媒體相當的不滿，因為這看起來就是名正言順的宣布巴薩的醫療團隊水平欠佳。

登貝萊雖然場外的麻煩不少，但是其實在球場上，他的發揮還是信得過的。但前不久在球隊對陣多特的歐冠比賽中他再次出現了受傷的情況，受傷之後他直接回到了法國，顯然他的態度說明了一切。

和外界一貫的印象不一樣，其實本賽季的巴薩到現在不管是歐冠還是聯賽，成績的底線還是很好地守住了，並且排名和小組名次都是相當的理想。但是現在這支球隊也給人一種感覺，那就是還是在吃老本，更多的還是指望著梅西和蘇亞雷斯這2個人的努力發揮。

這個時候真的需要點評一下巴爾韋德這位球隊主帥了，實際上，這幾年在他的執教歲月里，巴薩真的沒少花錢，但不管是登貝萊，還是庫蒂尼奧亦或是現在的格列茲曼，他都沒有用好，不清楚這裡的原因到底是他不會用，還是說球員不適應巴薩的技戰術。但總而言之，執教到現在，大家看到，其實巴爾韋德還是在依賴著那一群老將，並沒有本質上的更新換代。

可以說現在的巴薩，甚至是巴爾韋德，更多都是在靠著梅西去續命。言外之意，這個球隊其實有沒有巴爾韋德似乎都沒什麼區別，而他也沒有給這個球隊帶來什麼創新和實際性的改變挺好，他也只是在依靠著梅西。

所以說巴薩如果真的打算在歐冠層面有所突破，那麼現在真的該考慮是不是拿下巴爾韋德了。這個教練目前可以下這樣的結論，能力是真的不怎麼樣。他更多的只是在靠著老將和頂級巨星去發揮給自己拖延時間，而他面對那些本來是能讓球隊更上一層樓的球員，卻始終釋放不出來價值，不知道巴薩高層現在還在等什麼，還不將其趕緊解僱。
