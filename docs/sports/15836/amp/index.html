前敘利亞隊主帥：並沒發現歸化球員對中國隊有何貢獻

天天要聞
3 天 ago

直播吧12月15日訊 敘利亞足球近年來成為了中國足球的苦主，上屆世預賽12強賽，敘利亞隊在西安1-0取勝，在馬六甲則憑藉最後時刻的破門2-2戰平，直接導致國足出局。

本屆世預賽40強賽，敘利亞2-1戰勝國足，送走了里皮的同時也幾乎帶走了國足爭奪小組第1的希望。而最近在珠海舉行的四國邀請賽，中國國奧也0-1輸給了敘利亞國奧。

敘利亞國家隊前教練、敘利亞國奧主帥哈基姆在接受《紅星新聞》專訪時表示雖然中國隊有歸化球員，但這支中國隊要遠遠弱於兩年前的那支中國隊。

「中國足球在過去幾年有了很大的發展，但很難說中國各級別國家隊輸給敘利亞是一個巧合，我們對中國隊的研究非常深入，我們看過中國隊的許多錄像，研究過中國隊的許多球員，所以從教練的角度看，敘利亞贏了中國隊並不是巧合。」

「這支中國隊球員的技術水平比敘利亞球員要差一些，而且我們的球員都很年輕，中國隊的烏龍球出現在離比賽結束僅10多分鐘，球員體力不支動作變形有很大的因素，所以我不認為戰勝中國隊是運氣，這和球隊的球員構成有關係。」

「我認為上屆世預賽的中國隊要遠遠強於這屆世預賽的中國隊，我並沒有發現歸化球員對中國隊做出了任何貢獻。我覺得在中國隊戰平菲律賓後，里皮可能就在考慮辭職的相關事宜。」

「里皮之所以在輸給敘利亞後就宣布辭職，我認為他可能覺得自己有一個非常良好的執教經歷，不想破壞自己在國際足壇的形象，他也不想在這樣一支柔弱的國家隊之中執教。」

（11區電台）
