今年樊振東共拿了15個冠軍，但你知道他總共才獲得了多少獎金嗎？

2019-12-20 708條評論 2,284次閱讀 588人點贊

在此前結束的2019年國際乒聯總決賽上，兩位中國選手樊振東與馬龍最終成功會師決賽，在這場「中國德比」之中，近兩個月逐漸找回狀態的小胖樊振東，以4-1戰勝了隊友馬龍，從而在這場年終大戰里，贏得了最終的冠軍，而這一個冠軍，也將樊振東2019年的冠軍數達到了15個，相當出色！

總決賽男單項目的冠軍，這也是繼此前世界盃男單冠軍之後，樊振東又一項世界大賽奪冠。事實上，在10月份逐漸找到了狀態之後，樊振東也先後贏得了德國公開賽、世界盃團體項目冠軍、奧地利公開賽、世界盃男單、總決賽男單五個項目的冠軍，成為了下半年度最風光的球員。

整個2019年，樊振東獲得了乒乓球世界盃的男單冠軍、男團冠軍；國際乒聯巡迴賽總決賽的男單冠軍、男雙冠軍；奧地利乒乓球公開賽男單冠軍、德國乒乓球公開賽男單冠軍、瑞典乒乓球公開賽男雙冠軍、韓國乒乓球公開賽男雙冠軍、日本乒乓球公開賽男雙冠軍；2019乒乓球亞洲杯男單冠軍、第24屆亞洲乒乓球錦標賽男團冠軍、2019世界軍運會男單冠軍、男雙冠軍、混雙冠軍、男團冠軍

全年度，樊振東一共贏得了15塊金牌，很多人也都在關心，樊振東因此將會贏得多少獎金呢？那麼來詳細計算一番，3月份拿到亞洲杯男單冠軍之後，樊振東贏得了1.4萬美元獎金，之後10月份的德國公開賽樊振東贏得了3萬美元獎金，10月下旬的軍運會上，樊振東一人連拿多塊金牌，但是獎金卻可以忽略不計，11月份的奧地利公開賽，男單冠軍大約3萬美元，之後的世界盃男團冠軍，總獎金一共5萬美金，而世界盃男單冠軍，樊振東贏得了大約6萬美元，加上這次總決賽男單冠軍，獎金大約在10萬美金。

全年度樊振東雖然一共贏得了15塊金牌，總獎金卻只有270萬元人民幣左右，這還是按照人民幣1-7兌換美元的價格兌換，當網友看到這樣的獎金，也是忍不住調侃，樊振東辛苦了一年的獎金，也算是可以在北京三環以內買一個卧室了。而270萬對於中超球員來說，就太低了，據《南華早報》之前報道，中超球員薪水相當於全國平均工資160倍，平均年收入約為743萬人民幣。樊振東一年拿了15個冠軍，還不到中超球員平均收入的一半

可以說乒乓球運動員的收入，其實非常微薄的，與足球、籃球甚至田徑運動員都無法相比，國內中超聯賽運動員，基本上一年的收入早已突破百萬千萬，因此對於這些為國爭光的乒乓球運動員而言，這樣的收入確實有些太低了。
