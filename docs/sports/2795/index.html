限薪令讓他成恆大最大受害者 最佳難拿頂薪 替補卻能拿1200萬？

2019-12-04 0條評論 25次閱讀 0人點贊

摘要

雖然中國足協還未正式出台針對本土球員的限薪令，但限薪令裡面的條條框框已經被各方媒體曝光的八九不離十，基本內容就是本土球員的頂薪將控制在1000萬人民幣以下，若是國腳則能上浮20%，也就是1200萬人民幣。

雖然中國足協還未正式出台針對本土球員的限薪令，但限薪令裡面的條條框框已經被各方媒體曝光的八九不離十，基本內容就是本土球員的頂薪將控制在1000萬人民幣以下，若是國腳則能上浮20%，也就是1200萬人民幣。

對於這個限薪令的出台，一些中小球隊受到的影響比較小，畢竟他們球員的薪水本身就沒那麼高，但對於一些豪門球隊來說，他們受到的影響就比較大了。拋開一些國腳球員，即便是一些年輕小將，他們動輒也能拿到千萬年薪，比如說恆大。當然如果球員與球隊合同還有幾年，那這個政策暫時對他們還沒影響，可對於那些到合同期的球員，原來拿高薪的球員勢必會減薪，而一些原本想靠出色表現贏得大合同的球員，那這個願望可能要落空。

根據德國轉會市場顯示，恆大目前有6名球員面臨合同年，他們分別是曾誠、劉殿座、馮瀟霆、鄭智、徐新和郜林。曾誠、馮瀟霆、鄭智以及郜林都是年過三十的老將，本賽季四人都已不是球隊的絕對主力。鄭智下賽季應該會慢慢從球員轉型進入恆大管理層；郜林以後更多會承擔板凳奇兵的角色；曾誠地位已被劉殿座動搖；至於馮瀟霆，他的位置慢慢也會被高准翼、梅方、布朗寧等人所取代。所以這四人即便都續約，但其中只有曾誠還可能繼續成為國腳拿頂薪，至於其他三人，薪水肯定會有所下降。

年輕的徐新這幾年在恆大始終沒有展現出高人一等的實力，在恆大他只有替補的命，可即便這樣，徐新卻還有希望在新合同下拿到頂薪。在國內足壇，後腰這個位置比較缺人，所以即使徐新在恆大並非主力，但他還是能屢次被召入國家隊，因此還貼有國腳標籤的徐新，依舊有希望拿到1200萬人民幣的頂薪。

與以上5名球員相比，本賽季為恆大奪得聯賽冠軍立下赫赫戰功的劉殿座恐會成為限薪令下的最大受害者。他與恆大的合同也是明年底到期，如果沒有限薪令，那慢慢坐穩恆大主力門將位置的劉殿座有望在他職業生涯巔峰時期收穫一份大合同，可如今他不僅這個希望落空，甚至連拿1200萬頂薪的機會都沒。

劉殿座本賽季的發揮有目共睹，但在張鷺被開除出國家隊後，國足也未曾徵召劉殿座，所以在顏駿凌、王大雷還處於當打之年，曾誠還受足協青睞的情況下，作為本賽季最佳守門員之一的劉殿座說不定未來還沒法成為一名國腳，這對於他來說無疑是非常悲劇的。
