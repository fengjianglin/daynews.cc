猶他爵士隊上賽季是季後賽強隊，本賽季開局階段也打的很完美，一度衝到西部前三，不過在11月中旬後戰績開始下滑，最近8場比賽輸掉了5場，12月3日客場94-103負於費城76人後遭遇了二連敗，爵士隊是西部的傳統強隊，他們多年來都是以打防守體系為主，魯迪-戈伯特更是兩屆DPOY，主帥斯萊德有鬼才之能，可為何在11月中旬開始陷入泥沼呢？關鍵還是防守的下滑，爵士隊最強之技是防守，可在打出優異的開局後球員們有些自滿開始不按照斯萊德的戰術進行，很多時候竟然去跟強隊拼進行，比如118-122負於雄鹿，要知道雄鹿是進攻效率聯盟第三的球隊，跟雄鹿拼進攻無異於自掘墳墓，雖然全隊手感也不錯，可最終還是輸球了。

爵士隊本賽季的平均進攻效率105.3聯盟第23，在這8場比賽之前106.1聯盟第26，幾乎就是墊底的水準，可這8場比賽爵士在某些比賽去跟對手拼進攻，讓他們的進攻效率有所提升，可這種提升換來的是防守效率的暴跌，11月13日爵士擊敗籃網時防守效率還是聯盟第一，但在這一波8勝5負後防守效率已經下滑到聯盟第11，這個下滑幅度相當驚人。作為一支以防守建隊的球隊，防守只能算聯盟中上，而進攻還在聯盟倒數10位徘徊，能贏三場比賽著實不容易了。

爵士隊防守效率已經只有104.1聯盟第11位了，湖人之前跟爵士隊競爭聯盟防守第一的球隊，他們也是因為一波10連勝導致球隊在防守端專註度有些下滑，從第二下滑到第五，這個下滑是能接受的，只要防守效率進入前五都是頂級防守球員，可爵士隊這個下滑幅度實在是太大了。

這8場比賽輸掉的五場：94-103不敵費城、110-130不敵猛龍、102-121不敵步行者、102-112不敵森林狼、106-107不敵灰熊，五場比賽有三場失分超過了110分，兩場超過120分，要知道爵士隊本賽季場均失分只有105.1分，爵士隊如此打下去，將成為一支進攻依舊稀爛，防守也稀爛的球隊，斯萊德教練或許該改變一下陣容了，博格丹諾維奇進攻端的確打的出色，可他在防守端的專註度真的很差，各個球員的戰術地位也該進行一下改變。

邁克-康利作為拿著1.53億巨額合同的准全明星控衛，在經歷了一波低潮期的開局後好不容易回暖，可在這8場比賽中，他的戰術地位又變得很突兀，他淪為了一個定點投手，這8場比賽他場均28.6分鐘，拿下12.5分4.3籃板5.4助攻0.6搶斷0.3蓋帽，命中率38.6%，三分命中率48.6%，場均1.9個三分球，康利場均11次出手有3.9次三分出手，他在關鍵時刻幾乎不拿球，就蹲在兩個底角等米切爾與博揚的分球，他的突破是一點都沒打出來，斯萊德安排給他的擋拆戰術也不多，幾乎要他硬打，1V1對於一個身高1.85米的球員來說是相當消耗體能的，這就讓康利在防守端的付出減少了。

這八場比賽，米切爾防守效率115，康利防守效率112.7，博揚防守效率111.1，傑夫-格林防守效率112.4，英格爾斯防守效率109.6，戈伯特防守效率109.3，這都遠遠低於聯盟中等水準，當爵士隊丟掉了自己的最強武器，想用進攻跟對手一較高下時這支爵士也就只能擊敗一下勇士、灰熊這樣的球隊了，斯萊德接下來該如何調整呢？
