CBA爭冠隊強勢復甦，遼寧擊潰大黑馬，國產保羅狂轟5三分創新高

2019-12-15 1,439條評論 3,145次閱讀 94人點贊

北京時間12月14日，CBA第17輪的4場比賽率先開戰，遼寧主場對陣吉林迎來東北德比。遼寧首節就火力全開，單節領先了16分。吉林一直苦苦追趕，幾次將分差追到個位數，不過遼寧牢牢掌控了比賽節奏，最終主場115-105戰勝吉林，豪取4連勝，戰績變為12勝5負，升至積分榜第三。

兩隊最近的狀態都非常不錯，吉林取得4連勝，11勝5負的戰績排名第三，本賽季他們曾經戰勝過廣東和新疆，是CBA有名的巨人殺手。遼寧此前取得3連勝，戰績同樣是11勝5負，暫時排名第5。兩隊淵源頗深，鍾誠、高詩岩、柴長易等球員都曾在兩隊效力，彼此非常熟悉。

首節比賽，趙繼偉在進攻端爆發，韓德君和史蒂芬森也內外開花，而吉林只能依靠瓊斯一人得分，遼寧首節就轟下35分，領先了16分，給了吉林一個下馬威。吉林在第二節開始追分，一度追到只差7分，半場結束時吉林只落後10分。吉林只有瓊斯一人拿到了21分，而遼寧全面開花，5人的得分在7分以上。

易邊再戰，遼寧再度爆發，趙繼偉、巴斯等都延續了火熱狀態，吉林除了兩名外援之外，本土球員都手感冰涼，遼寧一度領先了21分。不過本節末段，吉林打出一波小高潮，一度將分差追到個位數。遼寧帶著12分優勢進入最後一節。

最後一節一上來，吉林再度開啟追分模式，分差來到個位數。不過遼寧總能及時給予回擊，分差始終在10分左右徘徊。最後5分鐘，遼寧逐漸拉開分差，最終主場115-105戰勝吉林，豪取4連勝的同時，終結了對手的4連勝，遼寧的戰績變為12勝5負，一躍升至第三。

本場比賽，遼寧有5人得分上雙，4人上20 ，史蒂芬森23分6籃板，郭艾倫20分6助攻，趙繼偉投中5個三分，得到23分10次助攻，巴斯26分，韓德君也有17分13個籃板。吉林的兩名外援聯手得到50分，不過本土球員中只有一人得分上雙。

拿到本場勝利之後，遼寧的戰績變為12勝5負，一躍升至第3，這支CBA老牌勁旅強勢爆發。郭艾倫表示，比賽主要贏在防守，讓瓊斯只得到30分，整體協防控制很好。
