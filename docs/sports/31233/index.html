“女象第一人”唐丹膽識驚人，棄炮狂殺，北京反超廣東居首

2019-12-25 970條評論 2,952次閱讀 151人點贊

12月23日，「樂昌桃花杯」2019-2020年全國象棋女子甲級聯賽「碧桂園」第二階段賽會制結束第九輪和第十輪戰鬥。女象第一人「八冠王」北京隊唐丹戰功卓絕，先後兩次殺敗00後江蘇年輕女將李沁，北京九重匯兩連勝江蘇棋院隊，反超廣東居首。而廣東樂昌隊被上海九城集團隊連續頂和兩場，屈居第二。

「女象第一人」唐丹兩連勝江蘇李沁

第九輪，「女象第一人」北京九重匯唐丹執黑棋後行對陣江蘇棋院隊00後年輕棋手李沁，以中炮右橫車對屏風馬開局，雙方戰鬥激烈異常，唐丹抓住紅方布局漏洞，成功破掉紅底相。戰至殘局，雙方子力相等，都是車馬炮雙兵（卒），但紅缺相。唐丹殘局功力深厚，車占要道，炮轟邊兵，步步緊逼，紅方已經是疲於應付。最終，唐丹以空頭炮之勢，車馬擒士獲勝。

第十輪，唐丹執紅棋先手布下仙人指路陣，李沁挺卒以對。開局不久唐丹先發制人，調動車雙炮於一側發起猛烈攻擊，三路兵也趁機過河參戰，逼退黑馬回窩心，紅方局勢一片大好。面對紅方如此強大攻勢，黑方壓力山大，揮炮敲掉邊馬並退車捉炮，謀求緩和。此刻，唐丹竟然棄炮，挺兵封住車路，真乃膽識驚人！

圖一

紅方棄炮後，炮二平五轟掉中卒，形成空頭，並取得多兵優勢。殘局階段，紅方乘勝追擊，捉死黑邊馬。雙過河兵連線，策馬掛角追殺，殺得黑方子力分散，陣型零亂不堪。黑方難以抵擋，102手認負。

這樣，唐丹兩戰兩勝，取得關鍵性勝利，幫助北京九重匯連克江蘇棋院隊，反超廣東居首。另一場北京隊劉歡對陣江蘇董毓男，雙方糾纏良久，第九輪和第十輪均以和棋收場。

值得一提的是，女子象甲賽場上，多位00後棋手展現了強大的鬥志，令人眼前一亮。上海九城集團隊董嘉琦在最近四輪比賽中有三輪遇到頂尖名將，都沒有失分，22日戰和北京隊劉歡，此番兩戰廣東隊王琳娜也都取得和棋，賽後這位靦腆的姑娘說，和王琳娜第一局自己布局準備到了，第二盤王琳娜後半盤沒有下好。

成都錦江隊「象棋一枝花」梁妍婷兩輪連和

其他對陣，成都錦江隊未能佔到浙江泰順隊便宜，取得一和一負戰績，仍然位列第三，但與榜首兩強積分差距有所拉大。金環建設河北隊與廣州隊、浙江秀州建設與杭州燃氣集團隊均拼殺激烈，各勝一場，不分高下。

24日上午女子象甲進行第十一輪比賽，下午進行賽事推廣活動，唐丹等多位大師走進羅定第一小學與愛好者們交流。
