名宿：布馮是義大利最佳門將 漢達應是世界前五門將

2019-12-19 771條評論 2,626次閱讀 601人點贊

直播吧12月19日訊 據《足球市場》網站報道，前AC米蘭、拉齊奧和卡利亞里門將伊爾波在接受電台採訪時，談到了尤文圖斯門將布馮、國際米蘭門將漢達諾維奇和拉齊奧本賽季的情況。

現年56歲的伊爾波在上世紀80年代和90年代效力於多支意甲和意乙球隊，並在1993-94賽季以替補門將身份跟隨AC米蘭獲得歐洲冠軍杯和意甲的雙冠王。

伊爾波在接受Radio 24電台的《全員集合》（Tutti Convocati）節目採訪時，說道：「拉齊奧進入爭冠行列？但願如此吧，但願尤文會暫時不在狀態，我也希望國米和拉齊奧能有所改變，雖然我不太清楚『藍鷹』是否能撐到最後。」

41歲的布馮目前在意甲出場646次，距離AC米蘭傳奇隊長馬爾蒂尼保持的意甲最多出場紀錄僅差一場。對於布馮可能打破這個紀錄，伊爾波表示：「這是個有點搶手、有點強迫式的紀錄，不過他一直都是義大利足球界最優秀的門將。」

而對於漢達諾維奇，伊爾波表示：「漢達諾維奇的100場零封是很有意義的。他是一位被低估了的門將，可能是因為在這些年裡他跟隨國米沒有贏得太多榮譽。他是世界前五的門將之一，作為一名門將我非常喜歡他。」
