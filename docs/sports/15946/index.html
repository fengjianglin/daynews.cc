兩隊戰績再次持平，湖人戰績超過雄鹿，老詹在MVP榜會超字母歌么

2019-12-14 710條評論 4,519次閱讀 1583人點贊

湖人和雄鹿在都在客場取得了勝利。湖人客場擊敗了熱火，打破了熱火主場不敗的金身，也迎來自己客場13連勝的記錄。雄鹿客場擊敗了灰熊，繼續刷新連勝紀錄，現在已經是17連勝。值得一提的是兩隊的戰績均是23勝3負，分別東西部的領頭羊。

今天兩隊的核心同時迎來爆發，字母歌在對陣灰熊的比賽中23投12中三分球7投3中，得到37分11籃板2助攻1搶斷，字母歌在第四節得到17分直接帶走比賽。本場比賽字母歌13罰10中，本賽季字母歌的投籃和罰球得到很大的提升，特別是三分球上賽季命中率只有25.6%，本賽季至今三分球命中率32.5%。米德爾頓19投10中三分球6中3，得到26分3籃板3助攻1搶斷，來人合砍63分。

說來也是怪，雄鹿的字母歌今天三分球准，詹眉也不落下風。詹姆斯22投11中三分球8中4得到28分9籃板12助攻准三雙數據；戴維斯20投11中三分球9中4，得到33分10籃板1助攻3蓋帽，戴維斯多次命中關鍵三分幫助球隊穩住局勢，不然還沒準真的被熱火青年隊翻車，詹眉合砍61分。

在最近一期MVP榜單中，字母歌繼續霸榜排在第一，然而詹姆斯卻被東契奇超越來到了第三位。就目前詹姆斯和字母歌的表現來看：字母歌場均31.1分13.1籃板5.4助攻1.28搶斷1.24搶斷；詹姆斯場均25.9分7.1籃板10.8助攻1.31搶斷0.5蓋帽。畢竟詹姆斯本賽季打組織後衛，進攻方面的數據比不了字母歌。但是老詹的RPM值9.7聯盟第一，而字母歌6.72排在了第三位。RPM是真實正負值反映了球員在每100回合攻防兩端為球隊做出的貢獻。

目前的雙方的戰績持平，還有兩場比賽雙方就要見面了。湖人的對手是老鷹和步行者，雄鹿的兩場分別是獨行俠和騎士，相比之下獨行俠和步行者都是難啃的骨頭。獨行俠還曾擊敗過湖人，終結了湖人的10連勝。雙方能見面前繼續保持戰績持平在見面的時候再分高低，那想必是一場火星撞地球的比賽。如果湖人的戰績超越了雄鹿，下一期的mvp榜中詹姆斯名次會提升么，會超越字母歌？

雄鹿和湖人的大戰，你更看好誰？歡迎大家留言評論。
