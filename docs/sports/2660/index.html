超蠢的！騙子謊稱被中超看上可以賺轉會費，歐洲某隊就這麼上當了

2019-12-05 0條評論 20次閱讀 0人點贊

摘要

相信不少人應該都收到過詐騙簡訊、郵件，這些騙子會編造一些極度荒謬的理由並承諾高額回報以此騙取大家的錢財。這種爛大街的套路，如今應該沒幾個人會信了，不過一家歐洲的足球俱樂部還是一愣一愣地上鉤了。丹麥甲級聯賽的維堡隊，在今年夏天收到了一封電子郵件。

相信不少人應該都收到過詐騙簡訊、郵件，這些騙子會編造一些極度荒謬的理由並承諾高額回報以此騙取大家的錢財。

這種爛大街的套路，如今應該沒幾個人會信了，不過一家歐洲的足球俱樂部還是一愣一愣地上鉤了。

丹麥甲級聯賽的維堡隊，在今年夏天收到了一封電子郵件。對方表示自己是知名足球經紀公司恆星集團，他們旗下有一位名叫維哈根的球員已被中超球隊看中，即將於明年1月份加盟中超，現在維堡有一個在此之前免簽該球員的機會。

中超球隊的財大氣粗在世界足壇都是出了名的，按照這封郵件的意思，維堡只用先將維哈根簽下，半年後便可大賺一筆轉會費。雖然，維堡此後給恆星集團發郵件確認一直未能得到回復，但他們還是找到維哈根，並簽下了他。

就當維堡隊滿心歡喜以為撿到了大便宜，在第一堂訓練課上他們就發現自己上當了，因為這位號稱將要加盟中超的球員根本不會踢球。但合同已經簽了，簽字費也給了，維堡隊目前只能寄希望於通過法律途徑維護自己的權益。

其實類似的騙子，在過去的世界足壇也並不少見。1996年，英超南安普頓隊的主帥格雷姆·索內斯接到了一通電話，對方聲稱自己是剛拿下金球獎和世界足球先生的賴比瑞亞球星喬治·維阿，他向索內斯推薦了自己的弟弟，稱其之前在巴黎聖日耳曼效力並且是塞內加爾的國腳。

來電的當然不是維阿，但索內斯偏偏就信了，他馬上就用200萬英鎊的簽字費簽下了這位名叫阿里·迪亞的弟弟。此後，迪亞開始炸傷，直到該年的11月他才在索內斯的不斷催促之下第一次在英超替補登場，並馬上被識破。

而世界足壇最著名的騙子，則非巴西人卡洛斯·凱撒莫屬了。這位根本不會踢球的球員，其職業生涯長達十幾年，並且效力的都是博塔弗戈、弗拉門戈、弗魯米嫩塞這樣赫赫有名的豪門球隊。但在這漫長的職業生涯中，凱撒只登場了十來次，並且他雖然號稱前鋒卻從未斬獲過進球。

凱撒之所以能混的這麼開，主要是因為他點滿了社交技能，他與一眾巴西球星私交頗深，因此總是會被自己的朋友們推薦給這些豪門球隊，甚至還實現了許多巴西球員一直夢想的去歐洲踢球。與迪亞一樣，凱撒也通過炸傷來掩蓋自己的真實實力，但與前者不同的是，他有著一副魁梧的身材以及更多的套路，比如上場後主動申請紅牌、與敵對球迷爆發衝突等等。

阿里·迪亞與卡洛斯·凱撒所處的那個年代消息相對閉塞，他們主要也是利用了各傢俱樂部的信息不對稱。

然而，在如今這個資訊便捷的年代，維堡隊的上當受騙就非常不應該了。先不說這個騙術有多低劣，恆星集團沒有給他們回復就已經是個很大警訓了，而他們卻在沒有試訓的情況下直接就簽下了這位來路不明的球員。天上不會掉餡餅，維堡隊的這一系列操作，恐怕是有些利欲熏心了。
