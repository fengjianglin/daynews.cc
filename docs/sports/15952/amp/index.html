61分、77分、88分，詹眉、威登和卡椒，這3對雙人組到底誰更強

天天要聞
2019-12-14

本賽季，「最強雙人組」的名號到底該給誰呢？湖人的詹眉、火箭的威登、快船的卡椒，從球隊成績上來看，詹眉更勝一籌，但從球員個人實力來看，似乎並沒有準確的答案。就在今天，這三對組合給我們帶來了一場接一場的神仙打架一般的比賽。

首先，在湖人客場險勝熱火的比賽中，詹姆斯全場拿到28分9籃板12助攻的准三雙表現，濃眉哥則砍下33分10籃板3高帽的出色數據，兩人合砍61分19籃板12助攻率隊從東部第二熱火的客場全身而退。

可以說，詹姆斯和濃眉哥的組合是本賽季成績最出色的一對組合，沒有之一。如果詹姆斯、濃眉哥本賽季一直保持健康，那麼湖人將會成為最大的爭冠熱門，目前球隊23勝3負的戰績已經說明了湖人全力爭冠的野心。

其次，火箭今天同樣在客場擊敗對手，在對陣魔術的比賽中哈登又一次暴走了，他全場31投19中，三分球15投10中，一人砍下54分5籃板7助攻2搶斷3蓋帽的全能數據。值得一提的是，哈登只出手31次就拿到54分，且只有6次罰球，這也是哈登連續第二場比賽命中10記三分，成為NBA歷史上第二位背靠背命中10記三分以上的球員，此前只有庫里做到過。

威少在這場比賽中手感同樣不錯，他18投9中得到23分7籃板6助攻，威登組合合砍77分12籃板13助攻。從數據上，威登組合似乎比詹眉更強一些。

最後，快船和森林狼也在今天上演了一場「神仙打架」的比賽，快船客場險勝森林狼，萊昂納德全場24投10中得到42分11籃板2助攻，喬治則是31投17中砍下46分5籃板7助攻，卡椒組合砍下88分16籃板9助攻的超級數據。

反觀森林狼這邊，維金斯扔掉「枸杞杯」之後得到34分，唐斯更是打爆快船內線，全場砍下39分12籃板的數據。森林狼的兩位當家也合力拿到73分，儘管森林狼主場輸球，但輸的可歌可泣，維金斯和唐斯更是讓球隊管理層看到了未來。

那麼，詹眉、威登、卡椒，再加上狼隊雙星，這四對組合到底誰更強呢？是成績穩定的詹眉，還是連續暴走的威登組合？亦或是攻防兼備的卡椒，還是年輕的狼隊雙星？
