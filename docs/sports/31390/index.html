萊萬、維爾納未能更進一步！記錄還是屬於“那個男人”

2019-12-24 1,003條評論 2,368次閱讀 885人點贊

冬歇期前最後一輪德甲聯賽在今天凌晨完成的大部分場次的較量，其中拜仁與萊比錫兩支勁旅頗為引人關注，一方面是因為兩支球隊都是本賽季冠軍的有力爭奪者，另一方面則是兩大射手萊萬和蒂莫.維爾納在本輪之前都有希望衝擊德甲半程20球的歷史記錄。

本賽季至今，火力全開的萊萬是拜仁的進攻保障，他在禁區中呼風喚雨，如入無人之境，在本輪之前，波蘭神鋒共取得19個聯賽進球。萊比錫紅牛高歌猛進，維爾納的攻勢摧枯拉朽，18個進球使維爾納在射手榜上緊追萊萬腳步。

萊萬和維爾納帶著期許迎來了本輪比賽的開場哨，在球場上二人也都全力以赴渴望將自己的進球數字進一步擴大。事與願違的是雖然二人非常努力，他們所在的球隊也得以拿下對手，不過兩位優秀的前鋒在本輪並未將火力轉化為進球。

後人挑戰的腳步未能企及，塵封的記錄被再次銘記，多年之前，一個名叫蓋德.穆勒的男人曾在德甲賽場和世界足壇昂首挺立，他強大的射術讓所有防守球員談之色變，在那個年代中蓋德.穆勒的名字就是進球的代名詞。

蓋德.穆勒的職業生涯太過傳奇，一一敘述哪怕三天三夜也講不完其中精彩，挑選一些重要的節點簡要回顧這位傳奇前鋒，在您看到以下的進球數字後一定會大為驚嘆。

德國慕尼黑北部小鎮諾林根是蓋德.穆勒的出生地，幼年時過人的足球天賦使他來到諾林根當地的TSV足球俱樂部，並在這成為一名前鋒球員，渡過了自己的少年時期。

1962-63賽季，諾林根TSV征戰巴伐利亞地區聯賽，在那一年，蓋德.穆勒作為球隊首發前鋒共計打入188粒進球，這樣鋒芒畢露的表現也吸引了拜仁的注意。1964年，在拜仁主席努德克的運作下，蓋德.穆勒披上拜仁戰袍，從此，「轟炸機」的美名名揚世界，讓綠茵江湖為之震顫。

從1964-1979，蓋德.穆勒將自己職業生涯最好的時光都奉獻給了拜仁慕尼黑，而拜仁也在穆勒火力的加持下逐步從德甲脫穎而出，坐上王者之位，並在歐冠賽場開創了屬於自己的時代。十五年拜仁光景，蓋德.穆勒共為拜仁出場427次，打入365球，這般進球效率實在太過驚人，上文我們所說的半賽季20球的記錄正是蓋德.穆勒在其最為巔峰的1968-69賽季所創造。

除去在拜仁所向披靡之外，蓋德.穆勒在國家隊的成績同樣耀眼，他在1970、1974兩屆世界盃中為西德隊出戰13場打入14球，在1972年歐洲杯中6場打入11球，其職業生涯總共為西德出戰62次，進球數則為68個。

一系列進球數字彰顯了那個年代蓋德.穆勒的鋒線統治力，因為其獨一無二的進球能力，2000年蓋德.穆勒被FIFA國際足聯封為「史上第一射手」。

前人的故事高懸空中令後人感懷凝望，塵封的紀錄總是與追逐者的身姿相伴。當足夠出色的萊萬多夫斯基和蒂莫.維爾納攜手並肩向蓋德.穆勒的記錄衝擊時，「轟炸機」的故事和他傳奇的職業生涯理應被我們再次講起。
