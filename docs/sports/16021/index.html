要臉嗎？東亞杯國足以大打小還輸球，教練組：沒慘敗，符合預期

2019-12-14 994條評論 2,115次閱讀 596人點贊

據《天津日報》報道，國足選拔隊隊內人士表示，東亞杯首戰輸給日本的結果屬於比較正常，選拔隊的表現總體上符合預期。對於首輪失利的結果，國足選拔隊隊內人士表示：「輸球是有心理準備的，但教練組擔心的是輸得難看。這場比賽隊員從精神面貌、技戰術發揮上，還是比較正常。」

從天津媒體這番報道來分析，國足選拔隊踢日本比賽時，教練組心態是不對的。球迷認為國足選拔隊踢日本，本身是以大打小，應該要有必勝的信念。如果國足選拔隊在比賽前，教練組都沒有給球員傳遞必勝的信心，那麼如何保證國腳能在比賽中全力以赴，贏下實力不俗的日本隊？國足選拔隊雖然不是以主力陣容出戰，但是像金敬道、張稀哲、姜至鵬都是多次入選國足的老牌國腳。

所以，國足選拔隊面對以年輕球員為主的日本三隊，應該是可以有必勝的信心。部分球迷對於教練組這樣的預期目標略感遺憾，認為為國出戰應該抱著必勝的信念，甚至一些極端球迷回擊稱呼奇葩一朵。仔細分析，教練組沒有在賽前跟國腳們下達必勝的指令，這可能是為了給國腳減輕心理壓力。試想一下要是國足教練組給國腳賽前傳遞了必勝的比賽目標，那麼國足選拔隊場上球員，就會背上想贏怕輸的思想包袱。

國足教練組對於國足對陣日本隊的賽前預期，就是不要輸得太難看。這就等於是給國腳們在場上減壓，以便能在比賽中讓國腳們有理想中的發揮。事實證明教練組賽前減壓起到了不錯效果，國足與日本隊的比賽中，大部分國腳都是輕裝上陣。雖然國足沒有最終戰勝日本隊，但是球隊在落後兩球的情況下，依然是拿出了與日本死磕到底的精神，從而追回了一球。這是讓很多球迷都感到非常欣慰的事情，紛紛為國腳這場比賽雖敗猶榮的表現較好。

另外，著名足球記者馬德興在對國足和韓國比賽做前瞻分析時，也是表示球隊只要在這場比賽不要輸得太難看就行。馬德興認為韓國隊在家門口作戰，肯定是會對這場比賽非常重視。韓國首戰對陣香港隊，沒有發揮出應有的狀態，就是要留力踢國足。所以國足要想在這場比賽贏下韓國，這是非常困難的事情。馬德興認為國足備戰與韓國的比賽，就是要少輸當贏。希望國足能放下思想包袱，在對陣東道主韓國的比賽中，繼續給球迷帶來精彩表現。

（羅掌柜）
