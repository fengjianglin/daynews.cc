國羽第2冠！凡晨2-0橫掃前世界第1 斬獲賽季第6冠，次局僅丟10分

2019-12-15 1,345條評論 3,363次閱讀 124人點贊

北京時間12月15日，羽聯年終總決賽最後的決賽日。國羽混雙包攬冠亞軍，鄭思維/黃雅瓊2-0戰勝王懿律/黃東萍，2組合分獲冠亞軍；女雙方面，重返世界第1的陳清晨/賈一凡與前世界第1的松本麻佑/永原和可那展開角逐，經過42分鐘交手，凡晨組合2-0橫掃松永組合，斬獲冠軍，本賽季已奪6冠；國羽也收穫總決賽第2冠！這也是凡晨組合第2次奪總決賽冠軍（2016年第一次奪冠），同時也實現對松永組合第4連勝！

陳清晨/賈一凡在2016年就先後擊敗張藝娜/李紹希、松友美佐紀/高橋禮華，斬獲年終總決賽冠軍；不過在2017、2018兩次總決賽中，凡晨組合都止步小組賽，沒能晉級4強。在2019賽季，凡晨組合的狀態逐步回暖，已經斬獲5站冠軍（全英賽、大馬賽、亞錦賽、中國賽、香港賽），世界排名也重返第1位。在本次總決賽中，凡晨組合整體狀態並不理想，小組賽也打的很辛苦，2場勝利 均是和對手大戰3局；不過在昨天的半決賽中，凡晨組合直落2局橫掃韓國的李紹希/申昇瓚，闖進決賽。

松本麻佑/永原和可那在2018、2019連續2個賽季斬獲世錦賽冠軍。她們在外戰方面成績不算好，內戰卻屢戰屢勝，2次世錦賽冠軍都是擊敗了隊友，被球迷稱為「日本內戰王」。凡晨組合與松永組合交手6次，4勝2負；今天再次交手，凡晨組合又一次取勝，本賽季已經4連勝對手。

首局，凡晨組合開局5-0佔優，隨後就一直保持領先狀態；松永組合不斷的進攻衝擊，但凡晨組合的防守嚴密，不給她們連續追分的機會；17-10時，凡晨組合連丟3分，但很快調整狀態又連得3分拿到局點；松永組合無計可施，最後，凡晨以21-14拿下首局。

次局，凡晨組合氣勢提升，日本組合沒能頂住；2平之後凡晨組合就開始領先，在7-4時又打出一波7-0的高潮，將比分擴大至14-4，領先10分；之後的比賽凡晨組合控制住比賽節奏，日本組合沒有太多的反擊，最終凡晨組合以21-10再贏一局，奪得冠軍！
