哈德森用41分回應老東家 為了芬森換掉他，遼寧隊後悔了嗎？

2019-12-20 2,467條評論 4,179次閱讀 2217人點贊

時隔2133天，哈德森又一次站在了遼寧隊的對面。

過去的5年，他都在那支球隊當中，與郭艾倫、韓德君們並肩作戰。2018年，哈德森更是帶領著遼寧隊拿到了隊史上的第一座總冠軍獎盃。只不過在上個賽季結束之後，遼寧隊選擇不再與哈德森續約。在這個夜晚，哈德森本賽季首次與舊主碰面。在比賽開始之前，他站在球員通道入口處與每一位昔日的隊友打招呼。但在比賽開始之後，一切都變成了另外一副樣子。

雖說哈德森與遼寧隊是和平分手，可任何一個運動員在面對舊主的時候，都會有著不一樣的心態。哈德森自然也是如此，此役他打出了本賽季至今最為精彩的一役。全場比賽轟下41分不說，哈德森在末節更是瘋狂的砍下18分，他的每一個進球，都像匕首一樣直插遼寧隊的心臟。最終，山東隊也以20分的優勢大勝對手。

此役開局階段，哈德森在場上還顯得相對低調。尤其是在第二節，他的發揮很令人失望，連續投籃不中。不過在下半場易地再戰之後，哈德森立刻變身成為了另外一個自己。第三節，他便單節轟下16分，帶領球隊打出一波37比19。而在第四節，哈德森更是像穿越了時光一般，打出了2015年前後的水準。

在那時，遼寧隊是一支第四節惹不起的球隊。他們之所以總是能夠在末節掀起得分風暴，原因就在於隊中有哈德森這樣一個大殺器。此役，遼寧隊還想像往常那樣在第四節製造波瀾，但卻被哈德森的狂濤徹底壓制。三分球，中距離，突破，昔日的總決賽MVP在這個夜晚幾乎無所不能。遼寧隊分別使用趙繼偉、郭艾倫來盯防哈德森，甚至有過包夾老哈的回合。可不論他面對怎樣的防守，眼前是誰，哈德森總能將球投進籃筐。

尤其是在比賽還剩下3分鐘左右的時候，手握巨大領先優勢的山東隊本應該穩住局面。但哈德森卻違反常規的出手了一記高難度的追身三分，這一球同樣空心入筐。遼寧隊的球員們在看到這一幕之後無不感到沮喪，哈德森的這記三分可謂殺人誅心，徹底摧毀了比賽的懸念。

末節18分，全場41分，同時還有10籃板外加7助攻。在35歲的年紀上，哈德森依舊能夠拿出這樣的發揮，的確令人感到無比動容。反倒是他對面29歲的史蒂芬森，又一次如同夢遊一般，在這場「現任」VS「前任」的對決當中徹底敗下了陣來！

此役過後，山東隊已經完成了一波8連勝。就像當年扛著遼寧隊前進一樣，如今哈德森又成為了山東的英雄。即便他已經35歲，各項能力都與巔峰期存在差距。可老哈對待比賽的態度，對於勝利的執著依舊沒有變。與此同時，他還多了一分成熟，能夠給球隊帶來更多的幫助。

不知道在看到哈德森這樣的發揮之後，遼寧隊球員們會作何感想。更不知道郭士強以及球隊的管理層，是否會後悔今夏的選擇。可不論如何，遼寧隊都已經做出了自己的決定，且時光不可能重新來過。遼寧選擇史蒂芬森替換哈德森，目的是為了增強球隊的活力，從而在新賽季再度衝擊總冠軍。

但就目前來看，史蒂芬森似乎並沒有顯示出什麼令遼寧隊感到驚喜的特質。反倒是哈德森在面對舊主的時候給了老夥計們一記重擊，就像有首歌里唱的那樣：「有的人說不清哪裡好，但就是誰都替代不了。」
