曝熱刺放棄留住核心，1億身價如今賣2500萬鎊，三大豪門爭相報價

天天要聞
2019-12-21

一月份的冬季轉會窗大門即將打開，熱刺很可能將清洗昔日中場核心埃里克森，減少球隊損失也避免人財兩空。《每日鏡報》報道稱，熱刺將會在一月份清洗埃里克森，這位昔日熱刺中場核心一直決絕和熱刺續約，他的合同將會在賽季結束後到期，熱刺為了不想在夏天讓埃里克森人財兩空，因此他準備在一月份賣掉埃里克森。

2013年5月，埃里克森從阿賈克斯轉會到熱刺，他就一直是熱刺的中場核心，本賽季，進入到合同年的他只為熱刺出場19次，其中只有9場首發，獲得2球3助攻的成績。穆里尼奧上任以來，一直在淡化埃里克森在球隊的作用，他並沒有給埃里克森多少比賽的機會，看起來狂人已做好失去埃里克森的準備。

《每日電訊報》稱，穆里尼奧希望在明年1月份的轉會窗口上調整球隊的架構，使得球隊的陣容達到理想的規模。，穆帥理想中的球隊陣容是由21名非門將球員組成，每一個位置都有合適的替補，每個球員都能參與到一線隊的比賽，如今熱刺的陣容達到了25人，他希望陣容可以得到縮減，埃里克森就是其中之一。

熱刺主席列維相信他已經失去埃里克森，日前他在接受《每日鏡報》採訪時說到：「說實話，我們不怕和競爭對手做交易。我的觀點是一個正處於續約時期的球員來說，他是會先想和其他球隊談談的。球員續不續約，這是他自己決定的。」

熱刺為了能順利出售埃里克森及時止損，並沒有對埃里克森標價太高，上個夏窗埃里克森只剩下一年合同的情況下被標註一個億，但是如今他只被標價2500萬鎊，列維只想挽回一些損失。目前對埃里克森有意的球隊並不少，曼聯、皇馬和國米都有意這位中場天王。
