國安外援太秀！踢中超1年沒進球 代表太極虎卻虐了上港恆大門神

2019-12-15 1,209條評論 4,253次閱讀 3022人點贊

年初的亞洲杯，金玟哉的進球幫助韓國隊2-0擊敗國足，拿到小組頭名出線。而年底的東亞杯，金玟哉面對張稀哲、于大寶等俱樂部隊友，再次在中國隊身上撒了一把鹽。值得一提的是，金玟哉加盟國安一年時間，在中超、足協杯和亞冠三線，沒有取得1粒進球，全年僅有的2球全部送入了中國男足的城池。

年初的亞洲杯小組賽末輪，中韓大戰中，當時與中超傳出緋聞的金玟哉，接孫興慜的助攻幫助韓國隊2-0鎖定勝局。隨著亞洲杯結束，國安很快官宣金玟哉加盟，值得一提的是，金玟哉原本更加接近與天海走到一起，因為天海出事，最終金玟哉與國安牽手。

防守不錯，又在定位進攻中有一手的金玟哉，原本被認為會在中超大殺四方，但2019賽季的中超，金玟哉首發25次替補登場1次，登場時間排在國安隊內的第5位，不過未能取得任何一粒進球。在足協杯和亞冠戰線，金玟哉同樣顆粒無收。

不過年初亞洲杯進球，年末的東亞杯，金玟哉連續2輪代表太極虎首發登場，面對中國香港毫無建樹，與中國男足的較量，金玟哉開場13分鐘就幫助韓國隊取得領先。

在中超面對恆大、上港時，國安僅取得1勝3負的成績，不過面對上港的主力門將顏駿凌、恆大本賽季上位的門將劉殿座，金玟哉年初亞洲杯和年末東亞杯，分別讓顏駿凌和劉殿座的城池失守，彌補了中超賽場的遺憾。
