讀秒絕平！一場1-1讓皇馬失去登頂良機，世紀大戰前仍落後巴薩

2019-12-16 308條評論 3,515次閱讀 1727人點贊

巴薩被意外逼平，讓身後的皇馬看到了登頂的機會。可佔盡優勢的「白衣軍團」，並未取得令人滿意的結果。西甲第17輪，皇馬客赴梅斯塔利亞迎戰巴倫西亞。前45分鐘，皇馬佔據明顯優勢，一共完成7腳射門，但由本澤馬領銜的攻擊線，沒有打破僵局。下半場易邊再戰，久攻不下的皇馬，反而被巴倫西亞洞穿球門。第78分鐘，索萊爾近距離抽射破網。傷停補時最後1分鐘，本澤馬扳平比分。90分鐘戰罷，皇馬以1-1戰平巴倫西亞，在客赴諾坎普之前，皇馬仍因凈勝球上的劣勢，位列積分榜次席。

聯賽前16輪戰罷，巴倫西亞以26分位列積分榜第8，少賽一場的皇馬，則以34分高居次席。因為率先出戰的巴薩，在客場被皇家社會逼平，皇馬迎來絕佳的登頂機會。

此役，皇馬的阿扎爾、馬塞洛和阿森西奧均因傷缺陣，主帥齊達內排出常規的4-3-3陣型，門將是庫爾圖瓦，傷愈復出的納喬出任左後衛，巴爾韋德、莫德里奇和克羅斯組成三中場，連續作戰且面臨停賽風險的卡塞米羅，終於獲得輪休的機會。伊斯科、本澤馬和羅德里戈在鋒線上活動。

主隊巴倫西亞的傷情，要更加嚴重，包括格德斯、李剛仁、加梅羅、皮奇尼、馬克西·戈麥斯和西萊森在內的多名球員，均高掛免戰牌。近7場比賽打進4球的費蘭·托雷斯和西班牙國腳羅德里戈，是「蝙蝠軍團」最為倚重的兩大鋒線尖刀。球員時代，齊達內和巴倫西亞主帥塞拉德斯曾在皇馬一起效力，上賽季，塞拉德斯甚至擔任過皇馬助教。

幾乎沒有經過任何試探，皇馬就迅速進入狀態。第5分鐘，本澤馬在禁區前沿做球，巴爾韋德迎球抽射，被巴倫西亞門將豪梅封出。4分鐘後，莫德里奇在失去重心的情況下倒地傳球，本澤馬稍作調整，起右腳射門，豪梅把球摁在身下。在這之後，克羅斯和本澤馬又連續製造威脅，主打防守反擊的巴倫西亞，則由費蘭·托雷斯完成一腳高質量的射門。

前30分鐘，皇馬的優勢非常明顯，但伸縮自如的巴倫西亞，沒有被對手拖垮。第36分鐘，主隊完成極具威脅的一次進攻，加亞從左路送出高質量傳中，托雷斯甩開瓦拉內，並趕在庫爾圖瓦出擊前甩頭攻門，皮球稍稍高出。因為技術能力上處於劣勢，主隊球員付出黃牌代價，也在所難免。第37分鐘和第41分鐘，中場索萊爾和右前衛瓦斯，分別被出示黃牌警告。

下半場易邊再戰，兩隊均未換人。第46分鐘，拉莫斯意外倒地，托雷斯快速殺進禁區形成單刀，庫爾圖瓦出擊將球封出。眼看對手來勢洶洶，皇馬又再次祭出高位逼搶，不斷消磨對手的耐心。

第60分鐘，塞拉德斯率先做出換人調整，馬努·巴列霍換下豪梅·科斯塔。這樣一來，瓦斯就回撤到右後衛的位置上，費蘭·托雷斯則頂替瓦斯出任右前衛，巴列霍和羅德里戈組成鋒線搭檔。第69分鐘，齊達內同時做出2處換人調整，維尼修斯和貝爾分別替換伊斯科和貝爾登場。第72分鐘，本澤馬把球做給克羅斯，後者半轉身抽射高出。

第78分鐘，皇馬的左路防守被打穿，瓦斯下底傳中，瓦拉內試圖解圍沒有碰到皮球，索萊爾近距離抽射破門，巴倫西亞1-0領先。傷停補時最後1分鐘，本澤馬扳平比分。90分鐘戰罷，皇馬以1-1戰平巴倫西亞，在客赴諾坎普之前，皇馬仍因凈勝球上的劣勢，位列積分榜次席。
