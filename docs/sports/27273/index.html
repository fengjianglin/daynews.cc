4.2億豪賭！高拉特竟是國足1緋聞主帥年薪54倍，國足給自己埋1雷

2019-12-21 1,651條評論 4,668次閱讀 1339人點贊

北京時間12月21日消息，早些時候中國足協已經開始著手準備之後世預賽亞洲區40強賽的相關事宜，首先要解決的就是歸化球員和國足主帥的問題。4.2億豪賭！高拉特竟是國足1緋聞主帥年薪54倍，國足給自己埋1雷。

此前高拉特自曝已經被歸化，也就是說接下來他已經具備了為國足出戰的可能，根據《東方體育日報》的消息，為了歸化高拉特廣州恆大簽下了年薪1080萬歐的5年合同，換句話說也就是5400萬歐，摺合人民幣近4.2億。

廣州恆大這次歸化高拉特是一場4.2億的豪賭，但是高拉特如此之高的年薪也是國足1緋聞主帥年薪的54倍，今年率隊拿到中超第5，足協杯亞軍的山東魯能主帥李霄鵬年薪僅為20萬歐，高拉特1080萬歐是20萬歐的54倍！

李霄鵬此前已經進入世界教練排行榜的前30，接下來假如要成為國足主帥的話，李霄鵬假如年薪和俱樂部持平的話，那麼他很可能是球隊中薪資最低的人！國足這次開始歸化球員，大部分歸化球員都可以得到高額的年薪，雖說他們的能力很強，但是國足也給自己埋1雷。

假如國足真的使用李霄鵬和李鐵等國產教練，讓他們指揮動輒年薪上千萬歐的歸化球員，不管教練水平如何，這些大牌球員能否願意聽從教練的指揮還不好說。里皮之前年薪1.8億，是李霄鵬目前年薪的100多倍，一旦在世預賽的關鍵時刻大牌外援不願意聽從教練的指揮，球隊可就徹底亂了套，假如足協使用本土教練的話，希望提前有所預防。
