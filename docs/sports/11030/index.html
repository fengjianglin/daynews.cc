贏了比賽還道歉？王寶泉賽後先認錯，還說功勞是大家。談朱婷傷病

2019-12-12 753條評論 4,582次閱讀 949人點贊

12月11日昨晚排超聯賽第二階段第二輪的比賽已經結束了全部的爭奪，損兵折將又勞師路遠的天津女排客場挑戰江蘇女排。最終津女在先輸一局的情況下連扳三局，逆轉戰勝江蘇女排。這場比賽中李盈瑩等球員的表現非常不錯，可如果要說本場比賽的最大功臣，還應該給臨時主帥王寶泉。而王寶泉在賽後卻先認錯，再推功。還主動透露朱婷傷勢，讓不少球迷為之點贊！

王寶泉贏了比賽還道歉

王寶泉是天津女排的功勛主帥，也是排超有名的名帥。在世俱杯結束後天津女排的主帥陳友泉因生病提前返回天津，所以王寶泉臨時作為救火教練來常州指導天津女排。可以說是倉促上陣的，可是這一仗打下來球迷忍不住為王導舉起了大拇指。執教水平真的是太厲害了，完爆對面被球迷稱作小諸葛的蔡斌！戰術靈活多變，根據場上形勢死掐江蘇痛點。把江蘇打得毫無脾氣。

天津女排贏下比賽

可是比賽結束後王寶泉指導卻首先向球員們認錯，他表示第一局的指揮有誤，不應該讓胡克爾去打主攻。因為球隊的主攻線短缺，所以讓胡克爾去打。結果胡克爾卻因此發揮不佳！所以第二局他做了調整，讓胡克爾回到自己喜歡的位置。所以後三局的進攻恢復正常！而且他認為本場比賽能贏球是大家一起努力的結果，感覺我們的隊伍非常團結，特別是教練班子對我，隊員對我都是非常支持的，贏了這場球是靠大家去拼、去搶，她們起到主要作用。

王寶泉完勝蔡斌

不得不說王寶泉教練果然是老手，要比對面的蔡斌好上不少。在比賽中迅速變更戰術，戰術靈活多變。賽後還主動認錯、攬責，要比陳友泉強上太多了。而且他還在賽後主動提到朱婷的傷病。說道：朱婷什麼時候能打上球還沒有時間表，我們時刻都準備著。在缺少朱婷的時候我們會努力拚搏，也希望朱婷能早早康復，參加我們的比賽。為王寶泉指導點贊！
