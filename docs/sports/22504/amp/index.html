真殘暴！趙睿生日遭阿聯“雙手暴扣”！而杜鋒的一動作太暖心了

天天要聞
2019-12-20

北京時間12月19日，經歷了魔鬼二連客之後，廣東男籃回到宏遠訓練基地後，進行了正常的訓練，積極備戰明天晚上主場迎戰天津男籃比賽。

今天恰是趙睿的24歲生日，今天下午廣東宏遠全隊「精心」為他準備了一個生日慶祝儀式，這儀式最重要、最精彩的環節當然就是拍蛋糕了。趙睿歷來都是隊友們過生日時拍蛋糕的主謀，包括大哥易建聯在內都深受他的「陷害」，現在輪到他本人過生日了，「仇家們」終於找上門了。

在休息室了，宏遠全體將士們把壽星趙睿早已圍成一圈，易建聯站在他的左邊，在大家為趙睿唱生日祝福歌時阿聯就開始活動雙臂進行熱身，看來他這次要親自出馬操辦這「隆重」的儀式了。果不其然，趙睿唱完生日歌之後進行了短暫的許願，但還沒等他反應過來阿聯就雙手緊緊按著趙睿的頭顱扣在蛋糕之上，趙睿頃刻變成一個蛋糕人。

杜鋒指導一直在背後欣賞這精彩一瞬間，當看到趙睿秒變蛋糕人，他也露出了無奈的笑容，並且很是憐憫愛徒趙睿，不斷拍打他的左肩膀以示安慰。

趙睿是杜鋒當年親自從柏寧籃球俱樂部挖來的愛將，是宏遠後衛線的最大依仗，本賽季周鵬隊長因傷缺陣，杜鋒更加需要趙睿承擔起更多的得分重任，趙睿也不負杜指導的期望，本賽季場均能砍下12.7分5籃板3.9助攻2.2搶斷的全面數據，兩分球命中率54%，三分球命中率34%，他的個人數據全面超越上賽季，杜鋒也自然對他疼愛有加。

目前廣東男籃正處新老交替時期，隨著易建聯、周鵬等大佬的慢慢老去，杜鋒迫切需要找一位新核心、新領袖來引領廣東宏遠繼續前進，而趙睿是目前九冠王新生代球員中最具領袖氣質的球員，他不僅球技越來越精湛，而且交際能力非常出色，號召力強，在阿聯和周鵬退役之後，他最有能力成為廣東男籃的新領袖，也難怪杜鋒對他疼愛有加。
