TWT騰訊圍棋大賽韓國獨苗出局 中國棋手包攬冠亞軍

2019-12-15 409條評論 3,499次閱讀 402人點贊

特約記者古柯報道 12月14日，由中國圍棋協會指導，騰訊棋牌主辦的2019TWT騰訊圍棋錦標賽半決賽在海口結束，倖存的韓國獨苗卞相壹九段（miracle97）不敵辜梓豪九段（印城之霸）；另一盤中國棋手內戰，范蘊若八段（北海的早晨）淘汰唐韋星九段（諸神的榮耀），晉級決賽。從明天開始，由辜梓豪對范蘊若的三番棋決戰以一天一盤的節奏進行。

本屆TWT大賽，共有6000多名棋手參賽，賽選手中脫穎而出的32名棋手經過單敗淘汰制的複賽爭奪戰，來自中國的唐韋星九段、辜梓豪九段、范蘊若八段和來自韓國的卞相壹九段獲得四張半決賽門票。

辜梓豪執黑對韓國排名第三的卞相壹，雙方相互點三三後，白棋靠入右下黑角拉開激戰序幕，但從結果來看，白棋所得甚微。卞相壹輕易棄掉兩子，讓黑角得以夯實，而原本黑小目高拆二守角本質上並非實空，實戰等於白棋幫黑棋完成了勢與地的轉換；不僅如此，因為白棋自右下生出一塊孤棋，在進攻這塊孤棋的過程中，黑棋順勢守住右邊盤實地，正好與右上黑棋外勢相呼應，實為黑棋求之不得的結果。

這之後，卞相壹繼續弈出疑問手，在右上三路沖、夾落了後手，辜梓豪機敏地脫先不理，及時於左上點角，鳩佔鵲巢，將白角據為己有，從而在實地對比上明顯佔優。

劣勢下的卞相壹發力猛攻左上黑兩子，這也是白棋唯一可能逆轉之處。辜梓豪在往外出逃的途中，還不忘在上邊二路上斷一手再撈一票，簡直藐視對手的攻擊力。待黑大塊孤棋在中腹走暢後，辜梓豪揮戈殺向右下尚未活凈的白大塊棋，完全掌控局面。

在黑棋的百般欺凌搜刮中，卞相壹忍無可忍，怒而開劫，但此劫對於白棋而言實為難以承受之重。最後不得不忍痛棄掉外圍三子，這才勉強打贏劫爭，但黑棋亦藉此全盤走厚，並將左下白棋厚勢成空潛力蕩滌一空，優勢已不可動搖。續弈數手後，卞相壹投子認輸。

唐韋星九段執黑對范蘊若八段，右上角兩人比拼久違的大斜定式，唐韋星沒有按照超級AI絕藝推薦的著法進行，導致勝率下降不少。白棋內外皆走到，頗為成功，勝率超過60%。不過，隨後范蘊若在右邊盤的著法略有過分之嫌，黑棋出頭後，白棋外勢並不完整，存有分斷的瑕疵，勝率掉至50%以下。

擅長利用厚勢作戰的范蘊若對把握外勢頗有心得，趁著黑棋在左邊盤優哉游哉地拆二，白棋在上邊盤壓迫黑棋二路渡過，此役白棋再次得分，勝率接近70%。

​

戰鬥轉至右下角，黑棋「實地為王」，在角上吃住白三子，守住角地的同時，亦將外圍數子置於白棋的攻擊射程內。一向性格和棋風溫和的范蘊若突然使出左邊盤四路碰的一手，這者棋意在略加借用換取外勢後，對下邊盤黑孤棋發起猛攻。

唐韋星不堪被利，奮起還擊，但此舉虎頭蛇尾，黑棋只是單方出逃，就連打斷白棋的拼一把著法都不曾使出。實戰白棋獲利超於預期，黑棋幾無所獲。之後唐韋星四處開攪，此為他成名絕技，但范蘊若不予糾纏，在大局上化解，在中腹圍出大空後，鎖定勝局。

辜梓豪對范蘊若的三番棋決戰從明天開始，本次冠亞軍獎金分別為70萬、20萬元。
