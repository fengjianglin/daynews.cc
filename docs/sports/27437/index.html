46歲李楠新工作曝光，薪資大幅下跌，與17歲兒子同隊

2019-12-20 389條評論 1,619次閱讀 324人點贊

今年夏天，男籃世界盃在中國舉行，而作為主教練的李楠被球迷們寄以厚望，他所帶領的男籃「紅隊」，去年在雅加達亞運會上發揮神勇，斬獲男籃金牌，給中國籃球注入了一劑強心劑，李楠也得到了大量網友的讚歎，大家對他充滿信心。但事與願違，男籃的接連輸球，讓球迷感到十分失望，與委內瑞拉的生死戰再次告負時，北京五棵松籃球館幾乎所有的中國球迷都在高呼「李楠，下課！」，網友們也對他進行了大量的討伐，而中國男籃則結束了自己在世界盃的征程。

世界盃過去後，李楠卸任國家隊主帥的位置，在離任的一個小時後，他也在自己的個人社交媒體上發布了一段長文向廣大球迷們道歉，「在離任之際，我誠懇地向所有為中國男籃失望過傷心過的人說一聲：對不起」，而他的繼任者是CBA廣東宏遠隊主教練杜峰，也給這次「鬧劇」畫上了句號，不過生活還得要繼續，離任後的李楠，如今去了哪裡？

離任後的李楠沒有接受籃協的安排，沒有選擇前往CBA執教，也沒有放棄自己的籃球事業，就如同他所說的一樣，「這些挫折不會讓我減少一分對籃球的熱愛，我會更努力的思考，去學習，去提高自己！」，如今他的新工作是在江蘇省青年隊做助理教練，這是他的老本行了，在沒有接任國家隊主教練時，他一直在八一青年隊和國家隊做助教，這也許是他最擅長，最熟悉的工作了。

不過大家對於李楠的選擇很是好奇，為什麼李楠沒有前去CBA執教？有人說，李楠的臨場反應能力不行，CBA的球隊都看到了，估計在質疑李楠的執教才能。也有人說，估計是壓力太大的問題，李楠不想去CBA指教。的確，在這幾個月里，李楠承受了很大的壓力，如今當青年隊助理教練，壓力會減輕很多，但同樣，他的薪資也會下降得很多，畢竟這只是省隊的青年隊，不過更主要的是為了他的兒子。

​大家都知道，李楠以前是國家隊的神射手，綽號「小李飛刀」，在CBA聯賽中的三分球總數達到1000個，排在歷史第一，是一個實力很強的外線球員，北京奧運會上，國家隊能進八強也有他一份的功勞。而子承父業，他的兒子李祿瞳，現在也是一名籃球運動員，今年17歲，也在江蘇省青年隊打球，名義上來說與李楠是「同事」，如今，李楠接任省隊助教就親自指導兒子，還可以更好的陪伴他，相信在未來的國家隊里，我們還可以看到當年的那個「小李飛刀」。

如今中國男籃已換主帥，而李楠也找到了自己的新工作，我們也祝福他能夠東山再起，而中國籃球能也能涅磐重生！
