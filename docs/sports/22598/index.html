瓜帥執教大巴黎實錘！巴薩功勛竟追隨加盟 重金請他解決一難題

2019-12-19 1,740條評論 3,300次閱讀 640人點贊

北京時間12月19日，《阿斯報》援引法國媒體消息，大巴黎計劃明年夏天邀請前巴薩主帥瓜迪奧拉入主王子公園球場。
《阿斯報》稱，由於和巴黎的兩大天王，內馬爾和姆巴佩不和，現任大巴黎主帥圖赫爾的帥位已經岌岌可危，法國媒體傳言稱，德國教頭保住帥位的方法只有一個，那就是拿到歐冠冠軍，否則明年夏天，巴黎主帥的位置肯定會易主，而最有可能的人選就是瓜迪奧拉。

本賽季歐冠淘汰賽的抽籤剛剛結束不久，大巴黎在淘汰賽的對手是德甲勁旅多特蒙德，這是圖赫爾的老東家，2015-17賽季，圖赫爾曾擔任多特蒙德主帥。對於圖赫爾來說，對陣自己的老東家是一把雙刃劍，一方面，德國人很熟悉多特，且目前多特主帥法夫爾帥位不穩，這很可能會影響到大黃蜂的作戰狀態。另一方面，這也對圖赫爾提出了更高的要求，歐冠淘汰賽是巴黎近年來的魔咒，被逆轉的噩夢一直困擾著法甲霸主。《阿斯報》稱，如果圖赫爾不能在兩回合的比賽中取得勝利，那麼這將是德國人在法甲的謝幕。
法國媒體稱，巴黎高層已經開始謀劃夏天換帥，大巴黎方面想邀請前巴薩主帥瓜迪奧拉，同時邀請前巴薩隊長哈維作為助手加入教練團隊。巴黎高層認為，這對曾帶領巴薩所向披靡的師徒，是能夠幫助自己登頂歐洲之巔的最佳組合。此外，這樣強大的主教練團隊，也能更好的留住內馬爾和歐巴佩。

內馬爾和姆巴佩的合同都將在2022年到期，內馬爾一直被傳希望返回巴薩，而姆巴佩則長期以來都是皇馬的目標。由於在歐冠賽場上戰績不佳，如何留住內馬爾和歐巴佩一直是困擾巴黎高層的難題，但是現在大巴黎的管理層相信，邀請到瓜迪奧拉和哈維，足以讓內馬爾和姆巴佩考慮留下。目前瓜迪奧拉在曼城擔任主教練，哈維則遠在卡達。值得一提的是，此前英國媒體曝光過瓜迪奧拉的合同細節，據英媒的說法，瓜迪奧拉和曼城簽有特殊條款，即瓜帥在滿足某種條件的情況下，可以自由離開藍月亮。
《阿斯報》稱，儘管很多人認為巴黎邀請瓜迪奧拉執教並讓哈維擔任助手的想法是痴人說夢，但是大巴黎也有可能做到這一點：外界認為，瓜迪奧拉在執教曼城三個半賽季後，曼城已然停滯不前，按照瓜迪奧拉此前的作風，他有可能離開曼城。而哈維的首次執教經歷也並不完全順利，薩德球迷甚至要求哈維下課。
