天津女排逆轉江蘇：王寶泉三句話扭轉敗局，蔡斌失策輸球被人砸

2019-12-11 88條評論 3,094次閱讀 1525人點贊

2019年女排超級聯賽繼續進行八進四的爭奪，在昨晚一場焦點大戰中，天津女排在立足未穩痛失首局的情況下，最終連扳三城實現了超級大逆轉，客場擊敗江蘇取得勝利。天津女排之所以能夠快速止住頹勢，主教練王寶泉的三句話至關重要，活得廣泛盛讚；相比之下江蘇女排的蔡斌主教練指揮乏術，遭到球迷不滿，甚至被主場的球迷憤怒地丟了礦泉水瓶，真是讓人哭笑不得！

首先看天津女排3-1（16-25，25-19，25-22，25-20）江蘇技術統計：李盈瑩41扣21中21分，胡克爾34扣17中17分，於鋆煒、王媛媛8分，李雅楠、姚迪6分，王藝竹2分。張常寧37扣15中1攔4發20分，龔翔宇34扣15中1攔1發17分，吳晗46扣14中1發15分，周馨憶、王辰玥、刁琳宇5分，許璐瑤2分。技術環節對比：進攻56-55，發球6-10，攔網6-4，自失13-19。而通過數據來看，江蘇表面上是輸在自失環節，其實並不是這樣，而是兩隊教練起到了決定性作用。

其二來看在第一局輸球後王寶泉說了什麼–在江蘇女排發球攻擊李盈瑩比較強勢，一路大比分橫掃天津，第二局天津依然落後時，王寶泉在暫停時對姚迪說了三句話扭轉了整個戰局：1.要不傳外援要不傳李盈瑩，不然這局就又沒了；2.你再不傳給她（胡克爾）我就換楊藝上場了；3.調整攻你就別傳於鋆煒了！

這三句話非常關鍵，姚迪隨後把主要進攻球權給了李盈瑩和胡克爾，結果李盈瑩51%的進攻成功率，21分全場得分最高，胡克爾50%的進攻成功率，拿到17分撐起天津右翼，而於鋆煒、王媛媛、李雅楠則適當投機，拿到錦上添花的分數。當然，王寶泉不止於戰略決策，他還敏銳地在後排替下經常發球失誤的王媛媛，在賽後給求勝欲強的胡克爾道歉，當然暫停時機的選擇和戰術運用也是得當的。

其三，來看球迷如何評價王寶泉：「王寶泉不愧帶過國家隊（儘管不成功），還是天津總教練，他指揮天津多年戰江蘇，蠻有心得的，僅憑他對姚迪明確分配球的思路清晰，就比天津女排教練陳友泉、甚至比蔡斌技高一籌」、「天津女排第二局暫停到後來扭轉戰局特別重要，當然天津女排後面的一傳相對好了，球好打了，但王教頭的觀察力和指揮能力，確實發揮了重要作用。」

其四，蔡斌表現哪裡失策，球迷為何丟他礦泉水瓶呢？1.變化不足戰術失策，當李盈瑩第二局一傳穩了還全場追發她，而不追發整場一傳差的孟子璇和於鋆煒就是失策，給對手穩定後方，進攻下球的機會。2.訓練培養了幾年，江蘇的副攻一塌糊塗，都是三無人員，場上完全不能分擔邊工的攔防壓力，真的不知平時道練了啥！

3.用張常寧龔翔宇主接一傳養吳晗戰略失策，吳晗全場46扣最多，但是進攻成功率只有30%，效率還不到20%，並且吳晗沒有攔網、一傳、防守都很糙，全場表現蠻賠錢的，可以說她完全撐不起張常寧的對角。而張常寧在後兩局打出60%和55%的進攻成功率，二傳並沒有多分配球權給重點人，即使關鍵分也隨意傳，這裡蔡斌的調度，就看出不及王寶泉精明了！比賽結束，有球迷第一時間用礦泉水瓶砸中蔡斌來發泄憤怒，雖然主教練做得不夠好，但球迷泄憤的做法也真欠妥！

另外，就兩隊王牌對比，李盈瑩接發球55次，承擔71%的一傳，進攻也是天津最多的41次，負擔較重也頂住了，非常不容易。張常寧同樣是全隊最多一傳31次，且在攔網、發球和防守端都表現全面，37扣15中1攔4發20分也不遜色，但遺憾的是她後面狀態火了，江蘇分配球權卻不合理，導致最終遺憾地輸掉了比賽。

而真正分出勝負的是王寶泉完勝蔡斌，才能客場大逆轉帶哦組勝利；可見教練的決策和洞察力對比賽勝負影響很重要！當然，這一戰對天津和江蘇都很重要，天津基本鎖定小組頭名，接下來穩紮穩打即可保持優勢；而江蘇則面臨著出局的危險，後面每一場比賽都很重要，需要抓緊找出問題彌補不足，在爭奪四強的道路上，給我們奉獻更多精彩的比賽！
