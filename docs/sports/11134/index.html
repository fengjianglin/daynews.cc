晉疆之戰！賽後王非和阿的江只握手不說話，發布會上卻又惺惺相惜

2019-12-11 1,530條評論 2,936次閱讀 436人點贊

114-110，當比賽結束的哨聲響起，王非和阿的江來到中場，贏輸和輸球心情自然不同，面對新疆隊主帥阿的江伸出的雙手，山西主帥王非只是禮節性的輕輕握了一下。兩人之間沒有任何語言交談，甚至連眼神交流都沒有……

要說本賽季的CBA聯賽，著實是好看。晉疆之戰，本賽季最大「黑馬」對陣CBA老牌勁旅，汾酒與伊犁特酒對碰，前八一隊兩位名帥王非與阿的江過招，顯然是充滿噱頭和故事。整場比賽，兩隊殺得難解難分。山西隊憑藉「三雙王」富蘭克林的四雙表現，給新疆隊製造了足夠大的麻煩，而單外援出戰的新疆隊，則依靠國內球員的出色發揮，將比賽懸念拖到最後一刻，並最終在客場帶走勝利。

此役，王非和阿的江的場上鬥法，同樣十分精彩。由於眾所周知的原因，兩人都不希望在這場較量中落入下風，戰術布置，臨場應變，調兵遣將，可謂是針尖對麥芒。關鍵時刻兩人見招拆，通過暫停鬥智斗勇，令比賽場面十分精彩。而兩人在場邊的激情指揮，更是成為球場上的另一道風景線……

儘管王非和阿的江在現場沒有任何語言交流，但賽後的新聞發布會上，兩位主帥在總結這場比賽時，依然表現出對對方的欽佩和讚賞。王非表示自己的球隊犯了一些不該犯的錯，同時說道：「不一樣的比賽，面臨對手不同，所以不管在進攻端還是防守端，對方比我們打得更有經驗。」；「作為我們球隊來講，打關鍵球的經驗需要提高，特別是面對新疆隊這麼多的國手，通過這場比賽學習了。」。

而阿的江也對山西隊大加讚賞，認為：「今年山西隊開賽以來整個表現非常好，從數據上也能看出來打的很團隊。」；「雙方都在不同階段打出了漂亮的球，但同時也都犯了錯誤，重要的比賽就在錯誤和正確當中進行，甚至最後我們漏了山西隊最重要的投手原帥，但是反過來他在防守中也漏了我們的投手阿不都，比賽確實很好看，驚心動魄……同時也為山西男籃這賽季的變化而祝賀，希望他們也打好接下來的每場比賽。」。

場上針鋒相對，場下英雄相惜，也許，這就是競爭運動的真正魅力所在……
