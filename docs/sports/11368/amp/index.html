大烏龍！魯媒報道國足大名單紕漏百出 球迷調侃：記者喝多了？

天天要聞
2019-12-09

2019年東亞杯即將打響，國內媒體十分關注國足表現，但是魯媒《半島都市報》的報道紕漏百出，尤其是國腳的名字，錯誤太多了。球迷調侃，「記者是不是喝多了？」還有球迷吐槽，這是故意黑國足吧，不然錯的太離譜了。

該篇報道的標題是《厲兵東亞 東亞杯明日打響,李鐵昨率隊出征》，其中詳細介紹了國足大名單，結果，很多名字都弄錯了。該媒體給出的大名單是門將：劉殿柱、董春雨、鄒德海；後衛：王松超、于大寶、梅芳、蔣志鵬、李昂、紀翔、楊帆、王上源、明天；中場：蔡惠康、張賢哲、康健、曹雪丁、金景道、穆亨特江-莫扎帕；前鋒：武士超、魏士豪、董雪生、馮金、譚龍。

其中，正確的名字只有8個，分別是董春雨、鄒德海、于大寶、李昂、楊帆、王上源、明天和譚龍，其餘名字全錯。如此紕漏百出，讓人大跌眼鏡。資料顯示，《半島都市報》是山東省第一份都市類晨報，創刊於1999年8月9日，距離已有20年歷史，2012年時，他們的發行量突破百萬。

可如今，這份報紙在體育報道上出現如此業餘的失誤，讓人唏噓不已。作為記者，不確定球員姓名，應該核實才對，名字基本全錯，那隻能說太不負責任了。

球迷吐槽，「寫文章時，記者是不是喝多了，就這水平的媒體，配不上現階段的中國足球隊，遑論還要點評一二，豈不是笑死個人。」

還有球迷認為是報紙故意的，「你說錯一個能理解，錯那麼多是故意的吧，惡搞國足！這張報紙值得珍藏。」

北京時間12月10日，國足首場東亞杯將打響，對手是日本隊。
