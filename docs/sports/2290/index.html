1.5億鎊轉會預算，切爾西劍指兩大引援目標！曼聯冬窗或受其影響

近日，切爾西在他們的轉會禁令上訴中獲得了成功，這意味著藍軍可以在冬窗簽下球員了。而這對於仍然需要時間和好球員重建的曼聯來說，可能是一個壞消息。

作為教練，球隊戰績和陣容搭建同樣重要，雖然青年風暴颳得猛，但是藍軍對強隊戰績不佳，可以看作蘭帕德平時成績優秀但是月考不及格。據外媒報道，蘭帕德冬窗將有1.5億英鎊的轉會費可用於球隊買人，而這筆錢怎麼花將成為蘭帕德執教半年來的一次大考。

萊斯特左後衛奇爾威爾是切爾西的頭號目標，阿隆索已經被棄用，西班牙人被藍軍掃地出門只是時間問題。

不過，奇爾威爾也在曼聯高層伍德沃德的視線之內，冬窗紅魔可能會在左後衛位置上有所加強，因為盧克·肖在上半賽季一直在與傷病作鬥爭，而小將威廉姆斯則缺乏經驗。

還有一名兩傢俱樂部都在關注的球員是多特蒙德的邊鋒桑喬。

這名19歲的球員已經和法爾夫發生了不小的矛盾，他有可能在一月份離開德甲。多特蒙德希望能從桑喬身上能得到1億英鎊的轉會費。

切爾西目前在英超聯賽中排名第四，以8分的優勢領先曼聯，但是兩傢俱樂部都希望能進入前四，並確保冠軍聯賽的參賽資格。橫空殺出的切爾西與曼聯引援目標重合，這使得伍德沃德和索爾斯克亞的計劃落空。(Qfwfq)
