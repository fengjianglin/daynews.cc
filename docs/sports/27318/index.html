不解決劫富濟貧的問題，索帥的曼聯很難走得更遠

2019-12-21 304條評論 2,214次閱讀 414人點贊

劫富濟貧是俠士之舉，也是本賽季曼聯之舉。現在的曼聯，就像是兩年前的利物浦，是英超著名的羅賓漢。索爾斯克亞本賽季專門偷走強隊積分，再給弱隊雪中送炭。比起到利物浦的安菲爾德球場，還是曼城的伊蒂哈德球場，曼聯的老特拉福德可說是中下游球隊的最愛，因為本賽季來到這裡通常都不至於空手而歸，曼聯可真是大氣啊！大氣啊！

遇強則強，遇弱則弱，這句話完美體現了現在曼聯的尷尬處境。面對積分比自家高的球隊，曼聯出戰5場取得4勝1平。平利物浦、勝萊斯特城、勝曼城、勝切爾西、勝熱刺。不管是曼城，還是切爾西，甚至是穆帥領軍的熱刺，都得老實地將3分拱手相讓。既然對強隊都能交出漂亮成績單，那為什麼還是無法躋身前四爭霸戰呢？就讓我們一起來回顧曼聯尷尬的對戰紀錄！

面對積分低的對手，曼聯平狼隊、平南安普頓、平阿森納、平謝菲聯、平維拉、平埃弗頓；輸水晶宮、輸紐卡斯爾、輸伯恩茅斯；只贏諾維奇、贏布萊頓。出戰11場，取得2勝6平3負的悲慘戰績。曼聯啊曼聯，紅魔鬼也確實將劫富濟貧的精神發揮到淋漓盡致，更勝兩年前克洛普領軍的利物浦。

  • 跑傳戰術不成熟，中前場沒創意來源

老鐵們，英超中下游最會打的戰術陣型，你知道是哪個嗎？就是鐵桶陣。打不過嘛，就別學西甲中游硬攻，最後沒破門，自家球門還得遭殃，賠了夫人又折兵何苦呢？面對強敵，堅守陣型等待機會打反擊，從古至今都是英超中游主帥的保命符。這難不倒利物浦的高壓傳控，對曼城的攻勢足球也沒太大效果，但曼聯呢？沒有足夠的板凳深度支撐，沒有成熟的跑傳戰術控場，索爾斯克亞的曼聯軍在面對鐵桶陣就顯得有些雞肋。

跑傳戰術就是注重跑動和傳球，用跑位互相掩蓋空洞，再用傳球控場將球權守在腳下。簡單的戰術往往最難，跑傳最大的考驗就是默契和速度。沒有默契和速度，跑傳很容易就變成定點傳球，放棄跑動來確保不丟掉球權。這也是為何索帥寧願用麥克托米奈，都不願用老將馬蒂奇，就是因為他缺乏速度。得到羅賓漢的稱號，和這套戰術有很大的關係。看見曼聯，強隊都想要從他們身上佔便宜，所以都會拉開陣型積極進攻，讓曼聯有機會伺機而動，跑傳也有更多空間能施展。

硬碰強隊好打，遇到弱隊卻難發揮效用。弱隊擺鐵桶收縮陣型，曼聯跑傳就是在自家半場傳傳球，根本打不進對方禁區。那跑傳遇鐵桶何解？創意，跑傳很吃中前場創意，很要求中前場是否能在鎖局殺出血路。要就是你挑傳很強，要就是搭配撞牆製造射門機會。說道曼聯創意來源，就不得不提起博格巴，就算他再怎麼被批評，你都不能否認他中前場的串聯能力。他的那腳傳球很關鍵，因為那是能破門的傳球。索爾斯克亞近期自己也說：「曼聯也許就缺乏一腳具有創造力的傳球，就能拿下最終的勝利。」

  • 缺乏板凳深度，場場都得重挑陣容

索帥清楚，曼聯需要面對英超，還得扛起歐聯賽事，靠著曼聯現在的板凳深度，自然不能場場都精銳盡出。本賽季的英超賽場，曼聯之所以遇強則強，遇弱則弱，和陣容無法脫鉤。想想，拉什福德和馬夏爾需要將體能維持在正常水平，就不得不犧牲掉某些比賽強度。

就像是利物浦的薩拉赫，在球隊贏球的下半場時，總是會在場中偷閑，趁機在前場降低跑動次數，藉此來保留體力。拉什福德和馬夏爾也相同，大家都清楚他們經常會互換位置，而每當曼聯面對弱隊，他們在半場後的跑動就會明顯減緩。反觀碰到焦點戰，他們卻絲毫沒有放鬆腳步，直到90分鐘還是持續跑動。索帥硬碰硬，弱碰弱的安排真的可行嗎？

當然！就在外界批評聲浪不斷時，索爾斯克亞悄悄帶著曼聯軍暫時重返積分榜前六名。單周連續打敗熱刺和切爾西，上周再戰平埃弗頓，被稱為英超羅賓漢的曼聯狀態正勇。但索帥得謹記，賽季才剛剛過半，後續還需要面對很多比賽。劫富濟貧固然可行，但始終不是長遠之策，也不是渴望爭冠的豪門球隊該有的態度。碰到中下游球隊就屢屢丟分的尷尬需要儘快解決，索帥的重建之路才能走得更順遂、更輕鬆。
