西甲今日頭版：極其清楚的憤怒 那一年非常偉大

2019-12-20 551條評論 1,538次閱讀 242人點贊

熱愛西甲，關注伊比利亞半島頭條，一起來瀏覽今日各大西班牙報紙的頭版消息。

《馬卡報》：

1、「皇馬認為VAR不會被黑手消滅」，皇馬對於國家德比的判罰非常不滿，他們要求得到裁判之間關於瓦拉內疑似點球的錄音，「沒有什麼可以做的」，評論聲相當失望。自從范尼斯特魯伊罰進點球後，皇馬已經12年沒有在諾坎普獲得過點球了。

2、西甲球隊對於視頻裁判不滿意。

3、畢爾巴鄂競技前鋒威廉姆斯接受了《馬卡報》的採訪，他表示，「我的母親是家裡的頭，她讓我有了今天的表現。」

4、前馬競隊長加比接受了《馬卡報》的採訪，他談道，「這是馬競過渡的一年？我能理解西蒙尼。」

5、遭受巴列卡諾球迷辱罵的阿爾瓦塞特球員祖祖利亞坦言，「沒有一個國家的主席支持納粹。」

《阿斯報》：

1、「極其清楚的憤怒」，皇馬將要求得到國家德比主裁判與VAR裁判關於瓦拉內疑似點球的錄音，他們認為自己是這場比賽的受害者，諾坎普球場已經12年沒有給皇馬點球了。

2、在布斯克茨生病的情況下，巴薩踢得有些不知所措。

《世界體育報》：

1、「那一年非常偉大」，前巴薩隊長普約爾回憶了夢幻般的2009年，那一年巴薩贏得了六冠王。

2、對於國家德比的分析，巴薩加強了防守，更好地應對壓力。

3、在國王杯的一場比賽中，武磊梅開二度，西班牙人客場2-0擊敗萊里達晉級下一輪。

《每日體育報》：

1、「範例」，巴薩在國家德比的比賽組織工作取得了出色的成績，僅有個別的場外發生衝突，諾坎普球場幾乎坐滿，出售了創紀錄的22000張主場球票（不含年票持有者）。巴薩更衣室在比賽中進行了自我批評，德容表示，「巴薩始終希望在主場贏球。」布斯克茨和法蒂能否出戰本周末與阿拉維斯的聯賽還存在疑問，皇馬方面則為了沒有判罰的點球而哭泣。

2、曼聯想要托迪博，巴薩認為可以從這位法國中衛身上得到2000萬歐元轉會費。

3、博揚在《每日體育報》的最新專欄中寫道，「前鋒越接近禁區越好。」

4、武磊用兩個進球帶領西班牙人在國王杯比賽中晉級。

（Paul）
