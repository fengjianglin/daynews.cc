韓國主帥：中國新男足值得重視！歸化球員+中國球員必會產生變化

2019-12-24 767條評論 4,803次閱讀 1634人點贊

高拉特成功入籍中國

近期隨著前巴西籍球員高拉特宣布成功加入中國國籍並可代表中國男足出戰的消息之後，迅速吸引了韓國媒體的廣泛關注，顯然高拉特的名氣與實力遠超艾克森、李可等歸化球員。據韓國媒體報道，高拉特曾經獲得2014年巴西金球獎、2015、2016年中國足球先生、2015年亞足聯年度最佳外籍球員、2015年亞洲冠軍聯賽金靴獎、最佳球員獎、2016年中國足球協會超級聯賽金靴獎等，球員實力強勁。

保羅·本托的執教能力已經得到韓國足協的認可

保羅·本托：高拉特絕對值得重視

近期韓國主教練保羅·本托率領韓國男足在東亞四強賽取得3場連勝最終拿到了東亞四強賽冠軍，保羅·本托的執教水平已經得到了韓國足協的認可，近期保羅·本托也接受了韓國媒體的專訪，對於中國男足保留·本托也進行了評價，保羅·本托表示此次來到韓國參加比賽的中國男足球員大部分都不是里皮執教時的主力球員，尤其是隊中多名歸化球員沒有參加比賽，這樣的一支中國男足並不是最強的，所以韓國雖然在東亞杯取勝，但不足以認為在未來亞洲區12強賽、亞洲杯等大賽中也可以輕易戰勝對手。

保羅·本托曾經在中超聯賽執教，對於中國球員了如指掌

保羅·本托表示，自己之前曾經在中超聯賽執教過，對於高拉特非常熟悉，高拉特在亞洲絕對是一流的前鋒或前腰球員，任何球隊都會想得到這樣的球員，之前中國男足在前腰、中場位置一直存在著問題，高拉特的加入完全可以彌補這個問題，另外高拉特在中前場的組織能力也極強，未來絕對是中國男足的進攻核心。

艾克森與中國球員一起慶祝進球

保羅·本托：卡達世界盃亞洲區12強賽碰到中國男足的概率不小！

另外保羅·本托表示，在未來卡達亞洲區12強賽中，與中國男足相遇的概率不小，外界一直認為中國男足就算有歸化球員加入也無法提升中國男足的實力，但我並不這麼認為，中超聯賽靠的是超級外援+中國球員，所以中國俱樂部很強，因為幾個關鍵位置都是外援在踢，中國國內球員只需要磨合好陣容即可，所以未來這支中國男足很像是一支加強版的中國俱樂部，只要給予足夠的磨合時間，相信這支中國男足會發生較大的變化！
