連續10年奪大滿貫冠軍有多難？只有一人做到，不是費德勒和德約

2019-12-23 3,607條評論 4,619次閱讀 682人點贊

在網球比賽中地位和含金量最高的當屬四大滿貫賽事，一個網球運動員如果能夠在大滿貫賽場上奪得冠軍，那他的歷史地位足以進入到網球名人堂。如果多次在大滿貫上都有奪冠的話，這位運動員一定是歷史級別的。如果一個人能夠連續10年在大滿貫至少獲得一座冠軍，那麼這個人會是誰呢？

這一紀錄的保持者是西班牙紅土之神納達爾，在2005年納達爾在法網奪得了自己職業生涯的首座大滿貫冠軍，在此後的10年時間裡，納達爾每一年都至少有一座大滿貫冠軍進賬。從2005年到2008年納達爾在法網實現四連冠，2008年他還在溫網成功奪冠，奪得了自己在紅土之外的首座大滿貫冠軍。

2009年納達爾雖然在法網爆冷輸給了索德林，但是在年初的澳網他卻首次在硬地賽場拿到大滿貫冠軍。從2010年開始納達爾在法網再次實現了一波五連冠，他連續奪得大滿貫的年份定格在了10年。在2015年傷病問題的困擾讓納達爾陷入低迷，他連續10年奪得大滿貫的紀錄也被終結。

納達爾的老對手費德勒在這一紀錄上的連續年份是8年，從2003年溫網奪冠開始，直到2010年澳網奪冠，他在連續8年的時間裡都有冠軍進賬。不過在2011年費德勒的這一紀錄戛然而止，在2011年四大滿貫費德勒只在法網打入決賽，最終輸給了納達爾無緣冠軍。

德約科維奇的連續奪大滿貫年數是6年，從2011賽季德約科維奇開始崛起，網壇三巨頭格局正式形成。從2011年到2016年這6年的時間裡，德約科維奇每一年都至少有一座大滿貫冠軍進賬。但是在2017年因為傷病和冥想的緣故，德約陷入到了職業生涯的低谷，他未能連續7年奪得大滿貫冠軍。

對比一下費德勒和德約科維奇的成績，就更能顯示出納達爾這一紀錄的偉大。這位紅土起家的馬洛卡少年，如今已經是19座大滿貫冠軍得主了。在下賽季納達爾將會向著費德勒歷史最佳的地位發起強力衝擊，或許這一次他真的能夠做到！
