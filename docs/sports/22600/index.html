德甲綜合｜小將替補成“關鍵先生”拜仁客勝弗賴堡

2019-12-19 1,536條評論 4,010次閱讀 413人點贊

12月18日，拜仁慕尼黑隊球員齊爾克澤（右）在進球後慶祝。新華社/路透

新華社柏林12月18日電（魏頎）德國足球甲級聯賽第16輪18日進行了五場較量，其中拜仁慕尼黑在補時階段連入兩球，3:1客場戰勝弗賴堡，排名超越沙爾克04和多特蒙德，升至第三位。

拜仁開場後就不斷威脅主隊大門。比賽剛開始不到2分鐘，庫蒂尼奧接阿拉巴中場直塞，單刀赴會挑射越過弗賴堡門將，但後衛古爾德在球門線前將球救出。5分鐘後，穆勒的一腳補射打在立柱上彈出。第16分鐘，左後衛戴維斯在邊路「千里走單騎」殺到前場，左腳送出精準傳中，助攻萊萬多夫斯基攻入賽季第19顆進球。

12月18日，拜仁慕尼黑隊球員齊爾克澤（左）在比賽中破門。新華社/路透

上半場進入下半段後，弗賴堡逐漸站穩腳跟並在進攻上大有起色。但門將諾伊爾表現出色，力保城池不失，拜仁1:0領先進入中場休息。下半場弗賴堡和拜仁不相上下，雙方你來我往互有攻守。第59分鐘，弗賴堡的韓國球員權昶勛在右路策動進攻，哈勃勒傳中至球門後點，格里福無人看防，凌空推射將比分扳平。

第90分鐘，拜仁18歲的荷蘭小將齊爾克澤替補出場上演德甲首秀，旋即成為本場比賽的「關鍵先生」。上場不到2分鐘，齊爾克澤接格納布里傳球，將球從弗賴堡門將兩腿之間送進球門。隨後，格納布里又攻入一球，最終拜仁3:1取得勝利。

同日，門興格拉德巴赫主場2:0戰勝帕德博恩，因凈勝球少於萊比錫而排在積分榜第二位。克林斯曼率隊取得聯賽兩連勝，柏林赫塔客場1:0戰勝勒沃庫森。科隆在0:2落後的情況下，連扳4球以4:2逆轉法蘭克福，逃離降級區。沙爾克04和沃爾夫斯堡1:1握手言和。
