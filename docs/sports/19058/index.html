東亞杯女足收官！女國足3輪僅進1球，1-0首勝取得東亞杯女足季軍

2019-12-17 737條評論 4,959次閱讀 3151人點贊

北京時間12月17日下午15點15分，2019年東亞杯最後一輪正式打響，中國女足隊客場挑戰中國台北女足隊！

此前兩場比賽，中國女足0-0戰平韓國女足，次輪0-3不敵日本女足。中國台北女足則是0-9不敵日本女足，0-3不敵韓國女足。

中國台北女足首發陣容

中國女足首發陣容

中國台北女足0-1中國女足

比賽時間開始，第29分鐘，中國女足獲得了右側角球的機會，王霜將角球開到中國台北女足禁區後點，隊友吳海燕在無人盯防下插上，輕鬆將球頂入大門得分，幫助中國女足客場1-0取得領先，同時中國女足在東亞杯斬獲首球！

隨著裁判一聲哨響，上半場比賽結束，目前在比分上，中國女足客場1-0暫時領先中國台北女足。上半場中國女足佔據優勢，吳海燕頭槌破門幫助中國女足取得領先，翟晴射門擊中了橫樑。

下半場易邊再戰，第54分鐘，中國女足在中場位置斷球後李影回敲了一下，將球分到左側的隊友古雅沙，隨後古雅沙再將球傳到中國台北女禁區內，李影接球後形成了單刀，面對中國台北女足門將射門，可惜的是打得太正了，被擋了出去，李影也沒能調整好步伐再射門！

隨著裁判一聲哨響，全場比賽結束，最終中國女足客場1-0中國台北女足，東亞杯女足首次取得勝利，積4分居於積分榜第三，取得東亞杯女足季軍；而作為主場的中國台北未能取得勝利，也在3輪比賽中難求一勝，居於積分榜第四。
