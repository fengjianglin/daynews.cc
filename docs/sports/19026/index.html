女排四強出爐！恆大晉級，老牌勁旅出局，六大主力國手無緣爭冠

2019-12-18 1,025條評論 2,994次閱讀 400人點贊

北京時間12月17日，中國女排超級聯賽第二階段全部結束，四強正式出爐，上海、北京、天津、恆大將爭奪本賽季冠軍。而隨著江蘇隊無緣四強，郎平手下的六大主力國手無緣爭冠，分別是江蘇隊的張常寧和龔翔宇、遼寧隊的丁霞和顏妮、八一隊的袁心玥、山東隊的王夢潔，加上朱婷舊傷複發，這種劇情實在少見。

本賽季，排超聯賽為國家隊奧運備戰讓路，根據全新的賽制，如果能在第一階段拿到足夠的優勢，晉級四強的希望就非常大。第一階段結束後，北京、天津、上海、江蘇都帶著兩個勝場的優勢進入第二階段，經過一番爭奪，北京、天津、上海把握住了機會，提前晉級四強。江蘇則慘遭連敗，被恆大反超，兩支球隊將四強爭奪懸念延續到了最後一輪。

因為恆大有勝場優勢，只要在最後一輪獲勝，她們就可以晉級四強，而江蘇女排不僅要在客場戰勝天津，還要寄希望於遼寧阻擊恆大，晉級四強的難度可想而知。對江蘇女排更不利的是，天津女排為了給之後的比賽打好基礎，仍然派出了除朱婷外的全部主力球員，兩大核心張常寧和龔翔宇就要率隊放手一搏。

開場後，天津女排馬上佔據主動，李盈瑩延續出色的競技狀態，胡克爾也保持不錯的進攻手感，兩大攻擊手讓江蘇隊吃盡了苦頭，天津女排很快以25-16拿下第一局。進入第二局，陷入絕境的江蘇女排打出了極高的競技水準，她們始終保持領先，後半局也頂住了天津隊的反撲，25-19扳平比分。士氣大振的江蘇在第三局前半段保持領先，但局中被天津反超，李盈瑩再次接管比賽，天津女排關鍵分表現出色，25-23再贏一局。

第四局，或許是恆大輸球的消息刺激到了江蘇女排，在張常寧和龔翔宇的帶領下，江蘇女排在這一局後半段取得兩位數領先優勢，連續失誤的天津女排15-25丟掉這一局。決勝局，李盈瑩發揮相當出色，率天津拿下比賽，2-3輸球的江蘇女排遺憾出局。

另一場比賽，恆大和遼寧爭奪相當激烈，恆大在第一局有些慢熱，無謂失誤過多、一攻成功率偏低，遼寧隊25-17拿下這一局。從第二局開始，恆大狀態回升，拉巴德捷耶娃、科舍列娃、鄭益昕、栗垚都發揮非常不錯，恆大很快以25-17扳平比分。第三局，遼寧強勢反撲，雖然早早無緣四強，但遼寧隊也要為榮譽而戰，她們在這一局後半段打出高潮，25-19再贏一局。恆大隊長栗垚在比賽中受傷，這讓恆大壓力相當大。

之後，遼寧女排完全佔據主動，她們在丁霞、段放及外援的帶領下打出了精彩的進攻，局中16-8遙遙領先。恆大女排無力反撲，14-25丟掉這一局，總分1-3完敗遼寧。雖然輸球，但因為江蘇隊也輸給了天津，恆大還是壓哨晉級四強。

其他比賽，上海0-3慘敗北京、山東3-1完勝八一、浙江3-0橫掃福建、雲南3-0橫掃河南。半決賽，天津迎戰北京，上海對陣恆大，天津和上海拿到主場優勢。整體來看，天津、北京、上海晉級四強符合預期，而老牌勁旅江蘇無緣四強有些意外，恆大則帶來了驚喜，經過方岩指導的調整，這支球隊的未來更值得期待。
