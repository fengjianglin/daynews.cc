英媒：阿爾特塔想挖走“巴薩教父”，被瓜迪奧拉果斷提為曼城助教

2019-12-21 662條評論 2,004次閱讀 741人點贊

當地時間12月20日下午，英超球隊阿森納宣布37歲的前曼城助教阿爾特塔為球隊主教練，以取代臨時助教永貝里，成為繼埃梅里之後的又一位阿森納主帥。根據英國《天空體育》的消息，阿爾特塔在與阿森納簽約之後，還想在瓜迪奧拉的曼城教練團隊挖人，但是遭到瓜迪奧拉的果斷拒絕。

報道說，阿爾特塔想帶走曼城教練團隊的一線教練博雷爾去當自己的助教。不過，瓜迪奧拉確認他已經當即任命博雷爾為自己的助手，以接替離開的阿爾特塔。值得注意的是，博雷爾雖然是曼城助教，卻是巴薩教父級的人物，他在1996年至2009年的長達13年時間裡負責巴薩青訓。幾乎執掌過巴薩所有年齡組的青訓梯隊，以梅西、皮克和法布雷加斯為代表的「87黃金一代」便是出自博雷爾之手，他還是梅西加盟巴薩之後接觸的第一主教練。

對此，瓜迪奧拉說：「從現在起[博雷爾]將和我呆在一起。在夏天之前，我們將以這種方式繼續下去，與在座的人和我們三年前開始交往的人在一起。到了夏天，我們會看到我們能改進什麼，不能改進什麼。」

而對於自己的去留，瓜迪奧拉則表示他希望留在俱樂部，儘管外界猜測他將在2021年合同到期後離開。2016年，瓜迪奧拉取代戰績不佳的佩萊格里尼執教曼城，並在2018年同意與球隊續約。目前，目前他和球隊的合同還有18個月的時間。

瓜迪奧拉重申了自己在合同期滿後留在曼城的願望，但承認「並不急於」達成新的協議。他說：「我認為在足球界一年半的時間對於一個教練來說是很長的。我們必須徹底看看它是如何工作的，發生了什麼。我已經說過很多次了，在這裡對我來說有多好，當然我也希望儘可能多的留下來。這是我的願望，但我想確定這對俱樂部來說是最好的，他們希望我能呆得更久。」

​「在一傢俱樂部待上5年，在這場競爭中要待上很多年，所以最好確保你的決定對雙方都是完美的。不著急。三四個月內還沒有結束，俱樂部在各個部門都運轉得很好，所以說實話，這不是問題。」
