國足進攻新核心人選浮出水面，東亞杯發揮搶眼，曾留洋德甲豪門

2019-12-12 24條評論 1,268次閱讀 285人點贊

張稀哲在國內中場球員中，一直被視為是技術和意識都非常出色的球員。著名足球解說員董路，就對張稀哲能力非常欣賞。董路認為張稀哲最大優點，就是在於他的一腳停球。這位國安中場核心一腳停球下來，總能讓防守球員一籌莫展，然後給國安隊友傳出好球。只是張稀哲有一個問題，就是無法將在國安的出色狀態，延續到國足比賽。

張稀哲屬於年少成名的球員，他很早就入選了國足被寄予厚望。張稀哲的天賦和能力，非常受大牌名帥里皮賞識。一度里皮想把張稀哲打造成國足中場核心，只是讓球迷感到遺憾的是張稀哲在國家隊表現，始終不如在國安搶眼。所以里皮在俄羅斯世界盃亞洲區十二強賽，給了張稀哲多次上場機會，沒有收到應有的回報。這讓里皮直接將這位曾經在德甲豪門沃爾斯堡效力的球員，直接打入冷宮，沒有讓他參加阿聯酋亞洲杯。

雖然里皮在第二次接手國足之後，多次招入了張稀哲，但是里皮沒有在世預賽上重用這位國安核心。直到李鐵開始出任國足選拔隊主帥，張稀哲才被委以重任，成為了國足選拔隊10號核心球員。國足東亞杯首戰對陣日本的比賽，身披10號戰袍出戰的張稀哲，在比賽中發揮非常搶眼。張稀哲在這場比賽最大改變，就是球風變得硬朗。

之前張稀哲球風偏軟讓很多球迷都對他表現不滿意，甚至國足主帥佩蘭在圈定國家隊出戰2015年澳大利亞亞洲杯23人名單時，以張稀哲球風偏軟為由將他徹底棄用。這一次在國足與日本的東亞杯首戰中，我們看到了一個不一樣的張稀哲。張稀哲在拼搶中展現出了自己兇悍的一面，給球迷帶來了眼前一亮的表現。張稀哲球風變得硬朗，讓他在比賽中逐漸顯露出捨我其誰的領袖氣質。

以張稀哲在國足對陣日本的東亞杯首戰可圈可點的表現，他未來很有希望成為這支球隊的新核心。張稀哲為了在國足站穩腳跟，他主動改變自己的球風，這是讓很多球迷都感到非常欣慰的事情。唯一讓球迷感到遺憾的是張稀哲這場比賽，並沒有給國足帶來助攻和進球。希望國足對陣韓國隊的比賽，張稀哲身為國足新核心可以為球隊繼續奉獻精彩表現，同時為國足打入進球，上演力挽狂瀾的好戲。
