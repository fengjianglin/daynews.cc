國足隊員的豪車陣容一個比一個豪！簡直是輸球不輸車

2019-12-10 115條評論 3,831次閱讀 1173人點贊

當輸球成為常態，當失望成為習慣，當打進世界盃越來越天方夜譚，國足還有什麼亮眼的呢？答案是，他們太太都很漂亮，他們的衣服都很奢以及他們開的車都很壕。

作為12強賽中國男足對陣敘利亞發揮最好的幾名球員之一，趙明劍的愛車選擇非常霸氣，價值400多萬的法拉利458估計在眾多國足隊員中也算比較高調的了。 據說這還是給寶貝兒子準備的法拉利，自己作為老爸開起來也是沒毛病。

楊旭，GTR與卡宴的組合，完全不遜色任何人。

至於如今炙手可熱的張玉寧，他其實還沒有自己的專屬座駕。不過，此前他從荷蘭回國時，接機的福特金牛座倒是非常搶眼。

張琳芃作為本土球員身價最高的球員之一，座駕當然不能落後，他的座駕同樣是保時捷，不過與鄭智不同的是輛豪華SUV MacanS。

作為國腳的朴成選擇了低調的小眾品牌邁凱輪,這可一點不低調。

黃博文，司職中場，現效力於中國廣州恆大足球俱樂部。把小奇瑞換成百萬級別保時捷帕梅拉，車的級別也見證他從小鮮肉長成大叔。其座駕乃是一輛價值百萬的豪車保時捷帕拉梅拉，搭載4.0T 雙渦輪增壓發動機，百公里加速3.9秒即可完成。

鄭智可謂是保時捷的忠實粉絲，在歐洲開的是一輛保時捷卡宴，回到國內以后座駕換成了保時捷Panamer頂配版價值233.80萬，這輛車可謂是最完美的轎車。這麼看來，國足隊員對於保時捷這個品牌還真是情有獨鍾。要是它們的表現能像保時捷在汽車領域的表現一樣優秀就好了！

前鋒郜林的保時捷Boxter，官方的報價是69.70-101.50萬元，據說這還是買來送給女友開的。

曾誠寶馬3系，他還有一輛寶馬X5,看來比較偏愛寶馬。

蒿俊閔瑪莎拉蒂GranTurismo,國內售價：159.80-248.80萬，這車原本算是很個性的一款GT跑車，只不過因為某美美成了網紅，作為海歸的蒿俊閔選擇這車還是挺有品味的。

國足球員張鷺醉駕被拘留的消息也是人盡皆知的，網路上被抓的的圖片也是被曝光，意外的讓我們看到了他的座駕是一輛路虎，這車的指導價格在122.5萬元到321.3萬元。當然對於年薪千萬的他們買一輛百萬的豪車，就是九牛一毛的事情。

王大雷是一個守門員，堪稱守門員的戰鬥機，在足球界的水平還是很高的，同時王大雷也有商業頭腦，擁有自己的潮牌店鋪。有球迷曾經看到王大雷的座駕是一輛邁巴赫，價值在139.88到309.88萬元，也是很豪華了。

趙旭日座駕是一台奧迪A4，價格在50萬左右，這個在國足裡面還算是比較低調的。

另外一位隊中的老資格，曾在凱爾特人留下過一筆的杜威，座駕是一輛紅色的牧馬人，彷彿就是他的汗血寶馬。

而備受爭議的任航，也是一輛保時捷加一輛凱迪拉克，派頭十足。
