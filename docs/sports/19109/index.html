唏噓！德佩重傷荷蘭鋒線告急 五大聯賽僅剩10名荷蘭籍前鋒

2019-12-17 978條評論 2,066次閱讀 1097人點贊

歷代荷蘭國家隊，從來都不缺少頂級前鋒。從克魯伊夫、范巴斯滕、古利特、里傑卡爾德到羅本、范佩西，各個都是一把好手。不過，如今的這支荷蘭國家隊，鋒線人才儲備卻非常尷尬。唯一能拿得出手的前鋒只有孟菲斯-德佩一人。但如今，里昂前鋒十字韌帶撕裂，基本確定將缺席明年6月份的歐洲杯。

本輪法甲比賽，里昂遭遇雙重打擊。球隊0-1輸給雷恩的同時，隊內核心孟菲斯-德佩也出現重傷情況。賽後，德佩被被診斷為十字韌帶撕裂。這樣的重傷，至少需要休養6個月以上。德佩本賽季已經提前宣布報銷，而明年6月舉辦的歐洲杯，他也很難能趕上。

德佩本賽季狀態相當出色，32場俱樂部和國家隊比賽中已經打進了23球，平均74分鐘就能貢獻1個進球或助攻。他的這次受傷，不管是對里昂還是荷蘭國家隊，都是一個十足的壞消息。尤其是荷蘭隊，前鋒線上本來就無人可用。德佩受傷，也讓橙衣軍團的歐洲杯前景蒙上了一層陰影。

在不少球迷的印象里，荷蘭國家隊全攻全守，從不缺少頂級前鋒。實際上，如今的這支荷蘭國家隊並非如此。他們在中場擁有著弗蘭基-德容、維納爾杜姆這樣的球星，後場也有如今世界上的第一中後衛范戴克以及新星德里赫特。唯獨在鋒線上，荷蘭國家隊前場缺乏足夠的爆炸球星。

在此前結束的歐預賽上，維納爾杜姆打入8球，是荷蘭隊的第一射手。德佩打入6球7助攻，是球隊的第二射手。再往後，則是打入2球的巴貝爾。這份數據，足以證明荷蘭鋒線缺人的現實。

縱觀如今的五大聯賽，其實豪門之中鮮有荷蘭前鋒的身影。除了德佩之外，五大聯賽只剩10位荷蘭籍前鋒。分別是英超曼聯的鐘塔西（0球0助攻）、維拉的艾爾-加齊（4球5助攻）；西甲塞維利亞陣中的盧克-德容（2球）；意甲羅馬陣中的小克魯伊維特（5球）；德甲沃爾夫斯堡陣中的韋格霍斯特（9球3助攻）、霍芬海姆陣中的洛卡迪亞（4球）、柏林聯合的貝克爾（0球2助攻）、法蘭克福多斯特（3球）、柏林赫塔的迪爾羅桑（2球2助攻）以及美因茨的博埃蒂烏斯（3球3助攻）。

這些球員中，除了小克魯伊維特、盧克-德容、韋格霍斯特小有名氣之外，其餘球員都比較平庸。在缺少優秀前鋒的情況下，荷蘭國家隊縱使中後場再強，也很難讓人看好他們明年的歐洲杯之旅！
