一天3消息！國內媒體怒批國足，李鐵正式表態！現場球迷噓聲四起

天天要聞
2019-12-19

中國隊終於贏球了！在東亞杯最後一場比賽中，中國隊2比0擊敗了中國香港，結束了本次杯賽的兩連敗。3場比賽，打入3球丟了3球，最終排名小組第三。國足的這個成績在球迷和媒體的預料之中，拿下香港也讓所有人長舒一口氣。畢竟，這避免了中國隊創造參加東亞杯以來的最差戰績。不過，國內媒體和球迷對於國足贏球一事並不買賬。直言中國足球發展這麼多年，根本就是個四不像！賽後，主帥李鐵也正是表態，並談到了自己的未來！國足贏球，媒體怒批中國隊四不像，李鐵正式表態！球迷：沒啥可喜。

中國隊主帥李鐵

雖然2-0擊敗對手，但比賽場面卻不像比分顯示的那麼輕鬆。從賽後數據來看，國足只不過在控球率方面優勢較大。射門數雙方都是8腳，射正數國足也只比對手多2次。賽後，獲得勝利的國腳們臉上露出了微笑，但球迷和媒體可笑不起來。今天晚上，國內某著名媒體就國足的表現做出了評價。

中國男足擊敗香港

在這篇文章中，他們毫不留情的表示：中國足球目前就是一個四不像！在本屆杯賽中，中國男足和女足都沒能戰勝日韓。而且，我們和他們的差距已經越來越大！現在，有很多日韓球員已經在國外打出來名聲。而反觀我們這幾年，從西班牙到德國，再到義大利，再到法國等。各國的青訓體系都被引入到了國內，也有不少青少年球員被送往歐洲，但中國足球卻依然找不到方向。歸根到底，中國足球的問題在於太過浮躁！

國內媒體怒批男足

這樣的評論對於中國足球來說一陣見血。可問題在於，中國足球那麼多年不缺少指出問題的人，而是缺乏能夠真正去指正和改變的方法。面對媒體，中國隊主帥李鐵也在賽後表態。他說到的第一句話就是：「這支球隊很不容易。」李鐵肯定是希望保護自己的球員，畢竟他們很多人剛剛結束漫長的賽季，又馬上投入到東亞杯的備戰中。而談到未來的時候，李鐵表示：「我非常驕傲，因為自己完成了兒時的夢想（成為國家隊主帥）。不過，未來的事情沒人知道，我只會把每一天的工作做好。」

張稀哲

可以說，從本次杯賽來看，李鐵指導的態度沒有問題。他真的是用心在對待這支球隊，這從他的每一次採訪中都可以感覺到，他也很期待給中國男足帶來變化。但是，冰凍三尺非一日之寒。中國足球的問題，遠不止1-2、0-1輸給日韓那麼簡單，也不是贏下一場比賽就可以解決的。正如球迷們所言，贏下香港根本不足為喜。因為中國足球未來的方向，依然迷茫。
