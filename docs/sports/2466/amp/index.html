穆氏熱刺初具規模，中場組合還需確立

天天要聞
2 週 ago

摘要

穆里尼奧上任四場比賽，前三場比賽進攻時左後防內收，右後防前插成為邊鋒，形成一個3241的進攻模式，這種向對方禁區施壓的打法，收到了不錯的進攻效果，雖然沒有中場中路的過程，但直接衝擊球門可以掩蓋熱刺球員組織細膩度不足的問題，發揮這群“10號半”球員在前場的衝擊力。

穆里尼奧上任四場比賽，前三場比賽進攻時左後防內收，右後防前插成為邊鋒，形成一個3241的進攻模式，這種向對方禁區施壓的打法，收到了不錯的進攻效果，雖然沒有中場中路的過程，但直接衝擊球門可以掩蓋熱刺球員組織細膩度不足的問題，發揮這群「10號半」球員在前場的衝擊力。

三場比賽在進攻上都有不少的亮點，但同時也出現了相同的隱憂，主要問題在於熱刺在陣地戰圍攻中的效率太低，手段不足，部份球員在禁區前沿的威脅性偏弱，而且「綠葉」球員的狀態起伏巨大，導致熱刺在整體發揮上，仍然存在不少的漏洞。

防守上，雖然3+2的後防有效地阻擋對手的快速反擊，但熱刺球員本身在陣地防守意識的不足，對兩側傳中處理的失誤偏多，加上門將不穩的情況下，穆里尼奧在上任前三場比賽場均丟掉兩球（第四場同樣丟了兩球），這顯然不是防守起家的穆里尼奧樂於看到的。

對陣曼聯的比賽，穆里尼奧嘴裡說要封鎖空間，減少曼聯的反擊威脅，但實際上在布置上並不保守，費爾通亨沒有像以往一樣內收成為三中衛、奧里耶也沒有前插成為右邊鋒，球隊執行的是一種常規的442打法，穆里尼奧削弱中路防守加厚邊路，並在邊路設置「驛站」策動反擊的意圖很明顯，但卻被球員的防守能力問題而在兩個開場階段令自己陷入被動的狀態。

兩個失球與西索科、達文森·桑切斯、奧里耶三位球員的防守失誤有關，第一個失球顯示出熱刺球員在防守時的心態、意識和整體協防、換防的判斷上出現困難，達文森·桑切斯沒有站住位置、奧里耶與西索科沒有適時補位，導致達文森·桑切斯連續受到曼聯球員的三次衝擊而崩潰，門將的失誤更加為後防球員的過失「買單」，這一點，是穆里尼奧需要好好打磨的部分。

上任僅僅四場比賽，而且是密集式的賽程，要求穆里尼奧馬上將波切蒂諾遺留下來的問題全部理清，並且高歌猛進並不現實，目前來看，穆里尼奧已經確立了球隊的運作框架：

孫興慜：戰術作用最大的一個人，進攻需要作為邊路推進器持球衝擊，也需要跨區作戰完成最後一傳、一射的任務，除此之外，他還承擔了不少的防守任務。

奧里耶：右路進攻的核心人物，主要負責球場寬度利用以及最後一傳，甚至需要壓入禁區包抄後柱，當然也有不少的防守任務，目前來說是穆里尼奧整個想法的「二號人物」。

阿爾德韋雷爾德：防線的領軍人物，不僅承擔了拖後包抄和指揮防線的任務，更是球隊轉守為攻的重點，對伯恩茅斯的兩次長傳完美地演繹穆里尼奧的「高空走廊」打法。

凱恩：相對於過往，他更加專註地與對方中衛糾纏和博斗，儘可能拉扯禁區的空間，相較於波切蒂諾時代經常要在中場區域策應，穆里尼奧需要他更靠近禁區活動。

阿里：穆里尼奧體系的最大受益者，完全削去了外圍任務，專註衝擊球門和反搶，他插上的把握能力充份展現，也掩蓋了組織能力的低下。

同樣是每場打滿的達文森·桑切斯在戰術上作用比較小，暫且不談。

但從以上的人員構成可以看得出穆里尼奧已經搭建了一個以兩翼為重心，中路接應為配合的大方向，糾正波切蒂諾離任前執迷於中路組織的思路，儘可能以邊路球員帶動中路的跑法，這一點或許是未來整體的發展方向。

不過，四場比賽穆里尼奧用過的中場組合：

對西漢姆聯：溫克斯 + 戴爾。

對奧林匹亞科斯：溫克斯+戴爾 → 溫克斯+埃里克森 → 溫克斯+恩東貝萊。

對伯恩茅斯：戴爾+恩東貝萊 → 戴爾 + 西索科。

對曼聯：溫克斯 + 西索科 → 恩東貝萊+西索科 → 恩東貝萊+洛塞爾索。

中場球員的出場時間方面：

溫克斯：251分鐘。

西索科：212分鐘（超過一半時間擔任右中場）。

戴爾：209分鐘。

恩東貝萊：101分鐘。

洛塞爾索：7分鐘。

埃里克森：98分鐘（只有少部分時間在防守中場區域）。

從以上的數據可以看出，在關鍵的防中組合上，穆里尼奧還沒能找到一對讓他完全滿意，或是風格上比較協調的組合，很多人的誤區在於，穆里尼奧要求的中場必須是「黑又硬」的掃蕩者，但實際上從坎比亞索+蒂亞戈·莫塔、阿隆索+莫德里奇，以至於馬蒂奇與法布雷加斯，穆里尼奧的防中組合當中，對於球員的組織能力要求非常高（否則也不會極力引進博格巴）。

目前的中場球員優缺點分明，而且風格沒能完全調和，擅長亂戰而缺乏組織思維、喜好持球突進而忽略失誤的風險，穆里尼奧無論選擇任何一個組合，都必然要承受另一端的缺點，能否在12月內儘快找出解決的方法，或許是賽季走勢的關鍵因素。

從紙面上的實力而言，能推進突破、能組織分球、也能身體對抗防守的恩東貝萊貌似是與戴爾搭檔的最佳人選，但是恩東貝萊有著法甲球員天然的失誤率，加上兩人偏弱的覆蓋範圍，會令第二落點的爭搶效率降低。

溫克斯有著對球路辨識的頭腦、出色的跑動能力，但卻在防守技巧和意識上有不少的缺陷，他與戴爾的組合防守性較強但進攻能力偏弱、與恩東貝萊搭檔則防守能力過分低下。

西索科沉迷盤帶、對防線保護意識不足，他的最佳位置似乎是與盧卡斯輪換右中場的位置，成為無球插上的棋子（伯恩茅斯一戰）多於雙防中的角色。洛塞爾索尚未完全適應英超的節奏，風格上更貼近中場區域的推進器，這一點與穆里尼奧目前的走向有著不同的思路，或許需要在適應期過後對自身的打法做出一定的轉變。

穆里尼奧能否在冬窗要求在防中位置進行引援，又是否會通過拿下盧卡斯增加一名中場中球員來提升防中位置的實力，或者能夠通過內部挖潛和改造完成戰術框架的搭建呢？我們拭目以待！
