穆里尼奧：這次是我足球生涯中最慫的一次

2019-12-22 817條評論 4,730次閱讀 498人點贊

黑幫教父杜月笙曾說過這麼一句話：人生有三碗面不好吃，一是體面，二是場面，三是情面。活得要體面，撐得起場面，顧得好情面。球場上亦是如此。

穆里尼奧和溫格的19次交鋒中，19戰10勝7平2負，戰績完全碾壓溫格，從此兩人勢如水火，場上場下彼此的相互攻擊對方，兩人經常在賽前發布會相互鬥嘴，球場上甚至都出現了動手的情況。那麼兩人為什麼會有如此大的恩怨呢？

不主動的球隊是對足球的侮辱

05賽季足球狂人幫助切爾西拿到聯賽的冠軍，但穆里尼奧的比賽方式引來眾多的批評和反感，而當中反應最強烈的當數教授溫格了，對於穆里尼奧的保守足球戰術直接就開炮了：我們清楚的知道，生活中只有贏家和輸家，當體育運動鼓勵不去爭取主動的球隊，這項運動就處在危險之中。暗諷穆里尼奧的防守足球是對足球的侮辱

偷窺狂魔

溫格的溫文爾雅在英超賽場是出了名的，有一股天生自帶的教授書生氣質。讓其在英超聯賽里有頗強的人緣。可但是初來乍到的愣頭青穆里尼奧可不慣著他，直接就瞄著溫格開炮：我想他是一個有偷窺癖的人，他總喜歡偷看別人，他對切爾西糾纏不清。有那麼一種人，在家的時候他總會拿望遠鏡看別人家的故事。穆里尼奧用自己的三寸不爛之舌，把溫格直接懟蒙了，溫格完全沒想到眼前這個年輕人會如此口無遮攔，張狂至極。絲毫沒有給溫格留一絲餘地。

愚蠢的人

溫格對穆里尼奧的反擊非常文明：穆里尼奧歪曲了事實，並且侮辱了他的人格，並認為有些時候，愚蠢的人成功不會讓他們變得更聰明，相反會讓他們變得更愚蠢。但穆里尼奧依然不依不饒的對溫格進行諷刺：在過去12個月里，我們已經從溫格的發言中總結出一份檔案，這不是一份只有5頁的檔案，他應該有120頁之多。溫格和穆里尼奧一連串的交鋒當中，沒有討到一丁點的好處，處處落於下風，c處處受制於穆里尼奧。估計溫格非常後悔的和這個不按套路出牌的年輕人挑起無謂的口水戰了。但為時已晚。

真刀真槍還真干不過教授

球場大打出手

真正的打鬥開始了，年邁而又身材高大的溫格迅速的佔據上風，而穆里尼奧滿臉通紅的只有招架之功，毫無還手之力。哈哈。教授終於扳回一局。賽後不甘的穆里尼奧迅速做出反擊：對於場上發生的這些我並不感到驚訝，而他卻沒有被指控？如果換成是我，我不知道將被禁賽幾場。這次場上的真刀真槍的交鋒，想必是穆里尼奧職業生涯最「慫」的一次了。

法布雷加斯再次點燃「戰火」

法布雷加斯從巴薩加盟穆里尼奧的切爾西，穆里尼奧當然不會放過刺激溫格的絕佳機會。當即向媒體暗示是法布雷加斯自己選擇加盟的切爾西，而且現在無比愛自己的新家。溫格在面對穆里尼奧的挑釁，自然不甘示弱：有人站出來說小法想加盟切爾西，但我不知道他說的是不是真相。穆里尼奧：不要管真相是什麼，小法現在切爾西的陣容中就是事實。自己一手培養的球員被自己最大的對手所用，溫格的內心想必比吃個蒼蠅還難受。

如今的英超賽場穆里尼奧依然在帶隊廝殺，但那個口水戰中屢戰屢敗的教授溫格卻早已不在，想必穆里尼奧現在無時無刻的不在思念著教授。

大家好我是22年的老球迷@足壇一鍋燴，歡迎大家一起來談論足球、分享球路上的點點滴滴，會持續的更新自己的原創文章，喜歡的點個關注，我們評論區見。謝謝大家。
