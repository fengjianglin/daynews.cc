曝曼聯接近7600萬鎊簽下哈蘭德 先回租半年明夏正式加盟

2019-12-18 1,277條評論 3,449次閱讀 1424人點贊

據《每日郵報》援引《太陽報》消息，曼聯將在一月份以7600萬英鎊的價格簽下薩爾茨堡紅牛前鋒哈蘭德。不過，曼聯還無法立即獲得哈蘭德的使用權，本賽季剩餘的時間要將哈蘭德租借給紅牛。

本賽季哈蘭德的發揮十分驚艷，22場比賽中打進了28球，外加送出8次助攻，這樣的表現也迅速引發了歐洲豪門的關注和哄搶。在這其中，曼聯對哈蘭德格外關注，索爾斯克亞想與昔日弟子完成再度進行合作。

報道稱，如今紅魔在哈蘭德的爭奪中處於領先地位，雙方將在近期達成協議。當被問及哈蘭德是否加盟一事時，索帥表示：「我不對猜測發表任何言論，就是這樣，他知道自己想做什麼，也知道自己要做什麼。」

曼聯已經確定了冬窗的引援目標，為進一步增強陣容，將在一月份引入一名前鋒和兩名中場球員，而這名前鋒首選就是哈蘭德。不過，曼聯還要提防尤文、多特，尤其是紅牛姐妹俱樂部萊比錫的競爭，據說萊比錫有2000萬英鎊的特殊條款可以讓哈蘭德為德國球隊服務，在本周二萊比錫紅牛主管馬庫斯-克魯什表示已經提供報價，正在等待著薩爾茨堡方面的回應。
