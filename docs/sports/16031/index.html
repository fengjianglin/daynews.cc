棋逢對手！奧沙利文轟兩桿過百追平塞爾比！高手對決太精彩！

2019-12-14 2,392條評論 4,067次閱讀 348人點贊

北京時間12月13日晚，2019蘇格蘭公開賽8進4比賽展開爭奪，在率先進行的一場比賽中，衛冕冠軍馬克艾倫5-1大勝對手成功晉級4強。緊接著就是火箭奧沙利文和塞爾比的比賽。攻防大戰，讓人慾罷不能。

奧沙利文-塞爾比

目前在最新的世界排名上奧沙利文排名世界第三，塞爾比排在第六。前四輪比賽，奧沙利文除了在首輪4-3險勝戴爾之外，其他三輪都是強勢晉級，其中兩輪完成橫掃。而塞比爾這便同樣出色，最近兩輪都是4-0橫掃，狀態也越來越熱。此番再次交手，很讓人期待。

首局比賽，塞爾比開球漏出機會，火箭打進紅球上手，可惜由於控球不到位只拿到11分進攻中斷。雙方在防守幾桿後，塞爾比抓住機會成功上手後一改慢熱單桿轟出120分完成清台。塞爾比取得開門紅。火箭打的有些隨意第一局。

第二局，塞爾比上手打綵球不進，直接將紅球堆打掃給了奧沙利文絕佳機會。火箭當然不放過這等良機，又一次向球迷展示了一把自己圍球的能力，單桿123分清台！奧沙利文也完勝單桿過百追平比分。

第三局，比賽進入了塞爾比喜歡的節奏，兩人開始陷入僵局，此時有意思的一幕發生了塞爾比防守時輕貼咖啡球沒有貼到形成自由球，火箭奧沙利文成功上手打破僵局，可惜只得到24分出現失誤，塞爾比上手後拿到53分面對一顆難度較大的紅球沒能打進停在袋口，但是奧沙利文將紅球打進後卻將粉球打丟。塞爾比再次上手並最終拿下第三局勝利再次領先！

第四局，奧沙利文白球沉底漏出機會，塞爾比上手拿到16分後選台紅球不進留下機會，奧沙利文上手後選擇K球將紅球堆帶開，並且他也做到了，取得不錯效果，比賽又進入火箭節奏，隨後奧沙利文向我們展現了他流暢的進攻以及清晰的解球思路。單桿113分！奧沙利文再次單桿過百！兩人2-2再次打平，回到中長休息！

從前四局來看，雙方的狀態都十分不錯。一有機會就能一桿打死對手。真的是棋逢對手啊！
