弗格26分孫悅22分湯普森14+24 北控主場23分大勝江蘇

2019-12-22 1,361條評論 2,449次閱讀 243人點贊

北京時間12月21日，CBA常規賽，北控主場以118-95戰勝江蘇，如願捍衛主場。

北控隊弗格26分6籃板6助攻4搶斷，孫悅22分2蓋帽，桑尼14分7籃板，湯普森14分24籃板，王子瑞13分4助攻，王少傑10分2蓋帽；江蘇隊吉布森36分5籃板4助攻，拉杜利察18分8籃板，王洪11分4助攻。

北控以11勝8負的戰績排在第8位，球隊也是期待著能夠脫胎換骨，獲得季後賽的資格；江蘇隊以9勝10負排在第12位，球隊也是希望能夠挺進季後賽。

首節比賽，王子瑞和弗格相繼進攻打成，王少傑中投得手，助北控率先佔得先機。王子瑞跳投打進，桑尼衝擊籃下打成，助北控以14-6領先。王少傑上籃打進，隨後他籃下又貢獻暴扣，助北控仍然牢牢保持優勢。孫鑫籃下進攻打成，吉布森命中空位三分，王洪上籃打進，助江蘇展開追擊。該節戰罷，北控以22-16領先。

次節比賽，弗格外線遠投命中，孫桐林和湯普森相繼進攻打成，孫悅和王子瑞相繼遠投命中，助北控以38-26取得兩位數的優勢。吉布森進攻打成，王洪又投中三分球，江蘇將比分追成32-38。湯普森和孫桐林相繼進攻打成，孫悅和弗格連進三分球，弗格拋投打進，助北控又將場上比分拉開。半場戰罷，北控以54-39領先。

第三節比賽，孫悅外線相繼命中2記三分球，弗格也遠投命中，北控以70-50取得20分的優勢。孫悅發揮神勇又進三分球，湯普森和桑尼相繼進攻打成，孫悅扣籃打成，湯普森籃下又輕鬆吃餅，北控在場上愈戰愈勇，牢牢的保持著自身的優勢。該節戰罷，北控以89-67領先。

末節比賽，孫桐林籃下貢獻暴扣，王少傑快攻上籃打進，助北控仍然保持著優勢。吉布森命中超遠三分球，劉宇軒和孫鑫相繼進攻打成，江蘇將場上比分追成84-99。弗格和孫桐林的快攻打進，北控在場上愈戰愈勇，以109-87拉開分差，這也令比賽再無懸念。接下來的比賽里，北控牢牢的保持著場上，以118-95勝出。

首發陣容

北控：桑尼、弗格、王子瑞、王少傑、孫悅

江蘇：史鴻飛、黃榮奇、易立、柳偉、拉杜利察
