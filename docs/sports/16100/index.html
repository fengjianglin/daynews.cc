樊振東全力一戰！國乒兄弟隊伍實力不足？鐵人許昕輕鬆晉級比賽

2019-12-13 61條評論 1,014次閱讀 94人點贊

隨著時間的推進，2019年國際乒聯巡迴賽總決賽首個比賽日已經結束了！在一場男雙四分之一決賽中，國乒組合樊振東/許昕以3-0輕鬆擊敗中國香港組合黃鎮挺/何均傑，晉級4強，但接下來將迎戰國乒組合林高遠/梁靖昆，屆時孰強孰弱，一戰分高下！

據悉，本場比賽是總決賽男雙四分之一決賽第二場，由國乒組合樊振東/許昕對戰中國香港組合黃鎮挺/何均傑，作為中國香港老牌組合，何均傑與黃鎮挺配合默契極高，國乒想要拿下比賽並不簡單，要知道黃鎮挺曾在單打上面擊敗過樊振東，實力不容小覷！在今年的卡達公開賽上，兩人配合更是一舉擊敗波爾/弗朗西斯卡組合奪得冠軍。

首局，黃鎮挺率先發球，但沒有取得很好成效，倒是被樊振東反制了一手，雙方比分以2-2開局，而後樊振東擺短被何均傑上手調大得分，許昕主動拉球也沒有取得很好的效果，但即使如此，兩部比分依舊以7-5領先，而後樊振東再次利用兩個發搶配合許昕正手得分，最終以11-7拿下首勝。

第二局，國乒組合越戰越勇，開局以6-0領先，並且一直保持領先狀態，後續何均傑/黃鎮挺奮力追分未果，在沒有任何機會的情況下，國乒組合以11-5輕鬆取勝，第三局，兩戰連敗，黃鎮挺/何均傑放手一搏，畢竟保守打法顯然是行不通的，要知道許昕是出了名的「纏」球選手，但防守一波的後果就是一國乒11-3懸賞比分結束。
