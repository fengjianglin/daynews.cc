英超西甲對撞！瓜帥再殺皇馬齊達內不答應 克洛普遇到更鐵血對手

2019-12-16 2,090條評論 3,731次閱讀 1479人點贊

北京時間12月16日晚，歐冠小組賽抽籤已經全部結束，由於本次16強球隊全部來自五大聯賽，強強對話成為了這次淘汰賽的主題。

我們知道，英超和西甲作為現在歐洲五大聯賽的兩大巨頭，這兩個聯賽之間的競爭可謂是十分激烈，上賽季英超球隊壟斷了歐冠和歐聯的冠軍，而這一次，西甲和英超又將進行直接對話。皇馬VS曼城、馬競VS利物浦，究竟是英超繼續獨領風騷，還是西甲發起反擊呢？

皇家馬德里VS曼徹斯特城

齊達內率領的白衣軍團在上個賽季簽運不錯，但這一次他們在16強中就遇到了英超豪門曼城。從實力上來看，兩傢俱樂部都是超級豪門，巨星雲集，比賽場面必然是火星撞地球，十分精彩。

但從過往戰績來看，皇馬可謂是佔據絕對優勢，在過往的4次交鋒中，皇馬對陣曼城保持著2勝2平的不敗戰績，在205-16賽季的歐冠半決賽中，皇馬以總比分1-0淘汰了當時不可一世的曼城，奪得了當季的歐冠冠軍。

因此，對於曼城來說，這一次的對決也是他們復仇的最好機會。而比賽的另一個看點是瓜迪奧拉與皇馬的恩怨情仇，在職業生涯17次率隊面對皇馬的比賽中，瓜迪奧拉的球隊拿下了9場勝利，這是瓜帥的曼城第一次對陣皇馬，他有望打破曼城歐冠對皇馬的不敗紀錄，而皇馬也將全力一戰，這組對決雙方實力十分接近。

馬德里競技VS利物浦

利物浦連續進入兩年歐冠決賽，並在上賽季捧杯，這支利物浦毫無疑問是現在全歐洲最耀眼的球隊。在豪門雲集的英超中，紅軍16勝1平0負積49分，這樣的戰績將時每一支球隊都望而退卻。對於誓在衛冕的紅軍來說，馬競是他們必須戰勝的對手。

而對於馬競來說，這也是西蒙尼第一次與克洛普交鋒，兩隊之前的交鋒記錄也寥寥無幾，只有1場較量，還是歐聯杯的比賽。然而，兩支球隊如今的狀態可謂是天壤之別，馬競目前只排在西甲第5位，格列茲曼走後球隊的進攻也不如從前。

從球隊實力上來看，紅軍顯然要高於馬競，但西蒙尼麾下的馬競最不缺的就是不懼強敵、勇往直前的精神，堅硬的盾遇上看上去更鋒利的矛，西蒙尼的球隊將受到來自對手的狂轟濫炸，而面對這樣一支鐵血馬競，紅軍也不可輕敵。
