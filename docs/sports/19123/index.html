陳羽菲或將成為2020年東京奧運會羽毛球女單的最大贏家？

2019-12-17 1,776條評論 4,098次閱讀 1534人點贊

陳羽菲或將成為2020年東京奧運會羽毛球女單的最大贏家？

  • 隨著2019年世界羽聯巡迴總決賽的結束，至此2019全面的系列賽事全部結束，接下來就是全新的2020年。

  • 2020年對國羽來說至關重要的一年，也是極具意義的一年；我們都看到近兩年，國羽因為正處於老一代運動員和年輕運動員進行接替更新的一個時期，成績一直都起起伏伏，不夠穩定，相比2012年倫敦奧運會時期的包攬五項冠軍，現在可以說現在的成績根本不夠以前的份量，國羽急需在短期內拿出一些成績來激勵運動員，和給廣大球迷一個滿意的答卷。

  • 在剛結束的2019年總決賽上，女單項目陳雨菲表現得可圈可點，她在先輸一局的情況下，憑藉強大的心理和嫻熟的技術逆勢而上，連贏兩局，以2：1擊敗中國台灣的戴資穎，奪得2019年度女單總冠軍，這樣也是國羽時隔六年再次女單總決賽冠軍。

  • 在2019年的世界羽聯巡迴系列賽事中，陳雨菲7次打進決賽並且7次奪冠，保持決賽百分之百的勝率。

  • 還是在2019年，陳雨菲以敗頭號天敵戴資穎的方式奪冠，並且同時還擊敗了因達農、奧園希望、山口茜等諸多排名靠前的選手。

  • 依舊是在2019年，陳雨菲登頂世界羽聯積分排名世界第一，結束了國羽女單對世界第一排名長達56個月的漫長等待。
  • 隨著上一屆奧運會結束，國羽老將李雪芮傷退，王儀涵、王適嫻相繼退役我們經過漫長的等待，終於等來了陳雨菲。

  • 陳雨菲在2019年所表現出來的嫻熟的技術，明智的戰術，強大的心理素質，讓我們對2020年東京奧運會羽毛球女單奪冠更有信心，讓球迷看到國羽女單重新崛起的希望。

加油，陳雨菲！！！
