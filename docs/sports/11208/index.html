一聲嘆息！上港掉進亞冠死亡之組，中超四強僅恆大有望好籤

2019-12-10 1,375條評論 2,689次閱讀 77人點贊

2020賽季亞冠小組賽抽籤儀式，已經在今天宣告結束。最終，廣州恆大和北京國安的抽籤運氣相對不錯。而在中超四強當中，上海上港可謂是簽運最差的球隊，他們必須在小組當中面對來自日、韓、澳球隊的激烈競爭，出線將面臨比較大的考驗。

​

E組的北京國安抽到了清萊聯，考慮到國安過去對陣泰國球隊的成績還算不錯，因此清萊聯應該不會對國安造成什麼障礙。該組另一支球隊應該是韓國的首爾FC，儘管曾在亞冠賽場上數次狙擊中超球隊，但是首爾最近幾個賽季實力有一定程度下滑，目前已不屬於韓國聯賽的頂尖隊伍。最後一個對手將在日本球隊鹿島鹿角和川崎前鋒之間產生，考慮到日本球隊近年來在亞冠上的強勢，無論誰進入小組都不好對付；唯一值得欣慰的是，國安對抗日本球隊一直不算吃虧。

​

F組的上海申花，已經確定與K聯賽第二名蔚山現代同組，對手當中還有珀斯光榮，以及一支資格賽球隊（很可能是日本球隊東京FC）。客觀來看，除了蔚山現代之外，申花這個小組的其他對手都算不上強勢，但是考慮到申花的實力，出線前景還是要打一個問號。當然，下賽季申花亞冠前景如何，還是要看崔康熙這位名帥在冬天的調教。

​

G組的廣州恆大，與水原三星和馬來西亞球隊柔佛同組。水原三星是上賽季韓國足協杯冠軍，而柔佛本身實力並不出眾，恆大如果正常發揮，這兩個對手應該不會構成太大威脅。比較值得關注的是，恆大還將在這個小組中面對日本天皇杯冠軍。目前天皇杯剛剛打到半決賽，奪冠最大的熱門是鹿島鹿角，但如果鹿島「陰溝翻船」，天皇杯被清水心跳、神戶勝利船或長崎奪得，那對恆大而言無疑是巨大利好。因此就目前而言，恆大是中超四強中最有希望斬獲上上籤的球隊。

​

中超四強里，簽運最差的當屬H組的上海上港，他們將在小組賽面對日本冠軍橫濱水手，以及韓國冠軍全北現代。全北的實力有多強不需要多做解釋，橫濱水手則是當今日本進攻火力最強的球隊，也不容小覷。該小組的另一個對手是悉尼FC，而上港今年與該隊兩次交手，僅獲得平局。從這個角度來看，上港的出線形勢不容樂觀。
