世界搏擊12月份排名出爐，中國11人上榜，傳奇泰拳王跌落前十

2019-12-18 788條評論 2,391次閱讀 1385人點贊

根據世界權威機構Combat Press公布的最新（12月份）世界自由搏擊排名來看，中國搏擊總計11人榜上有名，8男3女，展現出了中國搏擊從未有過的輝煌。其中羽量級邱建量仍高居第一、賈奧奇與鐵英華分別位居第八、第九緊隨其後；接著雛量級的名將魏銳排名第三，王文峰與朱帥分別位居第六、第十；然後是蠅量級的趙崇陽排名第八，草量級的王俊光排名第八；女子組P-4-P排名，汪柯菡、龔艷麗、王聰分別排名第五、第六、第七，大有巾幗不讓鬚眉之勢。當然幾家歡喜就有幾家憂愁，泰國拳迷在12月份就迎來了一個壞消息，他們舉國稱讚的傳奇拳王跌落世界前十。

該拳王相必拳迷已經知道，此人正是雅桑克萊，曾經連續7年未嘗一敗，其也是真正在泰國能與泰拳王子播求齊名之人，只不過2019年雅桑克萊的運氣實在太差了，說是霉運當頭一點都不為過，先是在年初ONE冠軍賽魔鬼八人賽上的首輪被法國小輩薩納爆冷KO，從世界第五直接跌倒世界第九，上個月同樣在ONE冠軍上，雅桑克萊迎戰36歲俄國老將賈馬爾.尤蘇波夫，都以為這位比雅桑克萊還要大兩歲的老將會被打穿，結果沒想到賈馬爾.尤蘇波夫直接爆冷KO了雅桑克萊，令人驚嘆，拳迷也直接看傻眼了，感慨道，昔日的世界王者估計要進入了退役倒計時，現在真是人見人打啊。

令人稀罕的是邱建了錯過了一個絕佳進入輕量級前十的機會，熟悉的拳迷都知道邱建良一直在找機會升重打輕量級，上個月還有提到其如果先對決雅桑克萊，打響自己升重輕量級第一戰，或許能夠順利通過雅桑克萊打進輕量級前十，直接創造中國搏擊歷史。如今看來，雅桑克萊跌出前十，輕量級最低的入場券就是目前排名第十的播求了，但從狀態來看，生涯末年的播求，彷彿重新找回了巔峰的自己，有點越老越妖的味道，近期打的幾場比賽都吊打對手，所以邱建良挑戰播求的勝算不大，錯過了雅桑克萊這個好機會，只能另擇良機了，我們也希望雅桑克萊能夠重新調整自己，找回那個傳奇泰拳王。
