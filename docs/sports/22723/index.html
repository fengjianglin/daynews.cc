東亞杯-東道主韓國1-0勝日本勇奪三連冠，黃仁范轟入制勝球

2019-12-18 654條評論 3,580次閱讀 1001人點贊

北京時間12月18日18時30分，2019年東亞杯末輪，東道主韓國隊對陣日本隊。歷經一番角逐，韓國隊憑藉黃仁范的進球1-0戰勝日本隊，從而以3連勝且零丟球的戰績鎖定本屆東亞杯冠軍。同時，韓國隊還憑藉此冠實現東亞杯三連冠且在此期間未嘗一敗。反觀日本隊則憑藉2勝1負獲得本屆賽事亞軍。

該場對決賽前，兩支球隊均取得兩連勝，其中日本隊凈勝球佔優。雙方此前在東亞杯賽場累計交鋒7次，各自拿到2勝3平2負。本屆東亞杯，韓國隊並未徵召旅歐主力，反觀日本隊則以奧運會適齡球員為建隊班底。另外值得一提的是，韓國隊此前兩屆東亞杯均奪取冠軍。

開局第9分鐘，朱世鍾主罰右路角球起右腳傳中，埋伏在小禁區內的金玟哉高高躍起搶點甩頭轟門，皮球擊中球門橫樑彈出，日本隊逃過一劫。第15分鐘，三浦弦太后場中路接應隊友分球送出過頂長傳球，鈴木武藏前插到禁區邊緣將球卸下，鈴木武藏隨即帶球突入禁區晃過金太煥後起右腳抽射，皮球彈地後擦柱而出。

第21分鐘，韓國隊前場中路獲得自由球，黃仁范主罰起右腳射門將球打飛。賽至第25分鐘，朱世鍾主罰右路角球起右腳傳中，畠中槙之輔門前高高躍起與金英權爭頂高空球，皮球砸中畠中槙之輔頭部反彈直奔球門方向，皮球擊中立柱彈出，日本隊逃過一劫。第28分鐘，金珍洙帶球沿左路推進至對方禁區邊緣且果斷倒三角回敲禁區弧，黃仁范接應傳球隨即起左腳勁射，皮球直接鑽入死角，1-0，韓國隊打破僵局。

第33分鐘，上田綺世接應隊友直傳帶球沿左路突入禁區獲得准單刀，金英權追防放鏟將球破壞並將上田綺世帶倒，主裁判判罰上田綺世越位在先。賽至第45分鐘，日本隊前場策動攻勢，佐佐木翔左路得球傳中，上田綺世禁區內躍起爭頂，皮球偏門而出。半場結束，韓國隊1-0領先。

第47分鐘，韓國隊反擊，金珍洙帶球沿左路突入禁區倒三角回敲，羅相鎬前插接應傳球起右腳抽射，皮球稍稍偏出。第50分鐘，韓國隊再度策動反擊，羅相鎬帶球長途奔襲突入禁區起右腳抽射，三浦弦太伸腿封堵射門將球擋出。

第55分鐘，朱世鍾主罰前場自由球傳中，金英權禁區內爭頂轟門，皮球偏門而出。第60分鐘，韓國隊後場斷球策動反擊，金珍洙帶球推進到前場突然直塞禁區，李庭協反越位前插沿中路突入禁區起左腳抽射，皮球稍稍偏出立柱。

第67分鐘，朱世鍾主罰右腳角球傳中，埋伏在禁區中路的金珍洙搶點甩頭轟門，皮球偏門而出。第78分鐘，韓國隊中場斷球策動反擊，黃仁范中路帶球推進直塞禁區，李庭協反越位前插接應扛住防守起右腳射門，皮球擦柱而出。第80分鐘，金珍洙禁區外得球突然起右腳遠射，皮球稍稍偏出。賽至第88分鐘，森島司前場中路接傳球隨即起右腳遠射，皮球稍稍高出橫樑。

最終，韓國隊1-0戰勝日本隊，奪取本屆賽事冠軍。
