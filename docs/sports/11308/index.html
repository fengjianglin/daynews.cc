女排國手見面會怎樣？總是沒正行，袁心玥尬舞自黑，賴導都沒眼看

2019-12-10 580條評論 4,065次閱讀 2989人點贊

中國女排國家隊隊員平時都分散在各個俱樂部，很難有聚集在一起的機會，這次女排世俱杯就提供了這樣一個見面的舞台，讓我們來看看她們見面之後會發生什麼吧。

袁心玥此次臨時加盟天津女排之後，終於又能和朱婷膩歪在一起了，兩人就像連體嬰兒一樣，走到哪都是手牽手。但這張兩人都纏著繃帶的照片還是讓人想起世界盃上的一幕，朱婷和袁心玥在網前手牽手，兩人的手掌同樣都纏著繃帶，她們就這樣一起贏下了世界盃冠軍。這一次她們再次手牽手，但遺憾的是結果並不美好。

天津女排和恆大女排的比賽結束之後，在天津女排屬於「外來務工人員」的袁心玥就跑到恆大女排串門。只見她一邊走一邊尋找目標，看起來很社會啊。

臨時加盟恆大女排的楊涵玉第一個發現袁心玥，並且一把抓住對方，兩人都笑得很開心。恆大女排新自由人王敏看著她們，眼神中滿是羨慕。王敏在恆大是新隊員，袁心玥是女排國家隊主力球員，在王敏心中袁心玥應該屬於傳說中的人物。

楊涵玉從背後抱住袁心玥，這是女排國家隊的傳統，一見面就膩膩歪歪、摟摟抱抱，總是沒個正行。

隨後袁心玥看見了國家隊自由人林莉，後者是里約奧運會中幫助中國女排奪冠的神級自由人，袁心玥與林莉都經歷過里約奧運會的一系列鏖戰，是最親密的戰友。但袁心玥見到林莉的第一反應是：盤她！直接就上手了，而林莉好像也習慣了這樣打招呼的方式。

隨後袁心玥的手又放在了楊涵玉脖子上，她們這種表達親密的方式真是別具一格，同時鄭益昕也走了過來和袁心玥打招呼。

下一秒鐘兩人人就毫無徵兆的抱在一起，並且沒有絲毫違和感。

在決賽結束之後的頒獎晚會上，袁心玥給惠若琪鄭益昕她們跳舞，當場尬舞，非常調皮。只是袁心玥這個拳頭讓人分分鐘齣戲，因為你很難不注意到這麼大的拳頭。

當時還有一名恆大外援在場，袁心玥一點也不扭捏，大大方方亮了一手。惠若琪拉住袁心玥之後，後者還掙脫了惠若琪的控制，又換了一種舞姿，袁心玥的內心一直很活潑。

看到一名外國球員拿著自己的卡通小人，袁心玥還現場模仿了這個動作，十分敢於自黑。

隨後兩人合影，袁心玥感覺意猶未盡，將剛才的模仿秀又學了一遍。

在觀看決賽的比賽時，袁心玥和惠若琪以及賴亞文指導坐在一起，隨後兩個老隊友就開啟了瘋狂自拍模式。旁邊的賴導看著她們就頭大，只能扶額長嘆。

袁心玥和惠若琪還在自拍，賴導已經轉過頭不看她倆。

拍完之後自然要看看好不好看，旁邊的賴導閉目養神，對自己的兩個老隊員是徹底沒眼看了。

這是鄭益昕在聯賽中見到張常寧之後的反應，還是這种放飛自我的畫風。大家怎麼看女排國手見面後的這種反應呢？
