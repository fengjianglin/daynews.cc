來自中國的三大“亞洲第一後衛”！孫悅名副其實，郭艾倫爭議最大

2019-12-12 429條評論 1,863次閱讀 463人點贊

在中國籃球的歷史上出現過很多亞洲頂級的球員，比如姚明、易建聯、王治郅、朱芳雨等人都是同時期亞洲範圍內最好的球員，他們也都曾被冠以「亞洲第一」的稱號。相比較其餘位置，中國後衛出彩的並不多，可即便這樣也曾經有三個人在不同時期被稱作是「亞洲第一後衛」。

最會給姚明傳球的劉煒

巔峰期的劉煒是國家隊雷打不動的主力，也曾在一段時間內被球迷門窗稱呼為「亞洲第一後衛」。但他也有一個摘不掉的標籤，那就是全世界最會給姚明傳球的人。在很多人眼裡，劉煒能進國家隊就是因為他和姚明從小就在一起打球，他們之間的默契的是無法取代的。

總的來說，劉煒絕對是同時期中國最好的控衛之一，而當時中國又是亞洲範圍內的霸主，所以劉煒「亞洲第一後衛」的說法也不說空穴來風。只不過劉煒的性格過於低調，而且他的打法屬於傳統的中國後衛，所以給人們留下的印象並不深刻。當然了，劉煒有缺點，但比起當時同位置的其他球員來說，劉煒攻防兩端是相對平衡的。

孫悅名副其實

2008年北京奧運會上，孫悅迎來了自己職業生涯的最巔峰，他的防守甚至引起了美國隊巨星們打賭，誰能在他頭上完成扣籃誰就可以得到50萬美元。當時的孫悅，身體天賦絕對是亞洲頂級，即便是參加選秀時的體測數據也在中等偏上，只是由於他黃種人後衛的身份讓他的選秀順位一路走低，而且當時湖人也沒有時間去培養新人，所以孫悅的NBA之路並不順利。

孫悅最巔峰的那幾年，放眼整個亞洲都沒有人能在同位置上比得過他，雖然他的進攻並穩定，但在強大的防守能力下這些問題都不是問題。退一萬步說，人家畢竟是給科比當過小弟的人，亞洲還有哪個後衛有過這樣的經歷？當然了，被菲律賓歸化的克拉克森不算。

「亞洲第一後衛」成郭艾倫心魔

2015年亞錦賽上，中國籃球在宮魯鳴指導的帶領下最終獲得總冠軍，而當時郭艾倫也當選了賽事最佳後衛，於是「亞洲第一後衛」的名頭又落到了他的頭上。不可否認的是，在2015年到2018年這段期間，郭艾倫在國內的人氣一路走高，期間他也跟隨遼寧隊獲得了總冠軍，在國家隊的地位也越發牢靠。可以這麼說，那段時間郭艾倫是完全配得上這個稱號的。

但這一切在世界盃失利後卻變得不同了，由於比賽時間被切碎，郭艾倫實在無法在場上有穩定的貢獻，再加上他之前有過綜藝經歷，於是這一切都被網友們拿來諷刺甚至是謾罵。到最後，「亞洲第一後衛」也成為了網友們往他傷口上撒的鹽，很長一段時間這個稱號已經成為了他的一塊心魔。不過幸運的是如今他已經慢慢走了出來，遼寧隊也在經歷了開局的混亂之後漸漸找到了比賽的感覺。
