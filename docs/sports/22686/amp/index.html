2比0中國香港！國足收穫8年來年度收官首勝，但看完比賽，所有人都高興不起來

天天要聞
2019-12-19

全文 782 字，閱讀時間預計 2 分鐘。

北京時間12月18日，東亞杯男足末輪，中國隊在韓國釜山對戰中國香港。前兩輪比賽，中國隊和中國香港同樣兩站皆負，中國隊憑藉凈勝球優勢領先對手。如果中國隊不能取勝，將創造東亞杯隊史最差戰績。

開場之後，中國隊展開圍攻之勢，第8分鐘，中國隊開出左側角球，中國香港後衛解圍造成烏龍助攻，吉翔門線前頭槌破門，中國隊1比0取得夢幻開局。

第17分鐘，中國香港打出陣地配合，在幾次一腳出球之後，前鋒吉奧瓦尼禁區內突施冷箭，皮球被劉殿座撲倒橫樑上，中國隊躲過一劫。

第31分鐘，中國香港隊再次製造威脅，又是吉奧瓦尼一腳遠射，被劉殿座驚險撲出。

35分鐘，于大寶禁區內倒鉤射門，皮球偏出門框範圍。半場結束，中國隊保持住一球領先的優勢。

易邊再戰第51分鐘，香港隊開出前場自由球，在中國隊禁區內搶到第一點後形成混戰，門前射門被于大寶擋出。

第63分鐘，買提江開出右側角球，埋伏在門前的吉翔搶點射門，直接被對手封堵。

第69分鐘，李昂直接自由球射門，又被對方門將直接沒收。

第71分鐘，中國隊終於擴大優勢，董學升禁區內被放倒造點，張稀哲一蹴而就，中國隊2比0領先。

第80分鐘，于大寶中場搶斷策動快攻，韋世豪小禁區線附近接隊友傳中，在無人防守的情況下，將球頂到橫樑上，錯失擴大比分機會。最終2比0的比分被保持到終場，中國隊本屆賽事1勝2負，積3分，提前鎖定第三。同時，這也是國足近8年來，首次在年度收官戰中取得勝利。

中國隊首發：1-劉殿座；4-王燊超、2-李昂、22-于大寶 、12-吉翔、20-李行、13-買提江（81『蔡慧康）、10-張稀哲、16-曹贇定（71』韋世豪）、9-董學升、18-王子銘（45『譚龍）

中國香港隊首發：1-葉鴻輝、2-徐宏傑、3-羅梓駿、5-艾里奧、12-梁諾恆、21-唐建文、6-黃洋、7-黃威、16-陳俊樂、20-鄭兆均、22-吉奧瓦尼
