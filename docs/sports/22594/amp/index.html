深度：C羅再演攀岩式救贖，從連場被換下到5場持續進球，發生了啥

天天要聞
2019-12-19

從連續被換下到最近5場比賽場場進球，而且還在今天凌晨對陣桑普的比賽中上演了一幕滯空騎扣的暴力頭槌，贏得滿堂彩的C羅，不僅重現了他年輕時在老特拉福德飛翔的一幕，還在生涯的尾聲階段又完成了一次「攀岩式」的救贖。

回頭看來，此前C羅在連續被換下時的原因是什麼？而最終用連場進球證明自己的這段時間裡C羅與球隊整體又發生了什麼？值得分析。

【「貝帥」持球效用下滑，薩里尋求邊路作業弱化C羅】

在變陣4312陣型的初期，貝納德斯基依靠自己的持球與突擊速度讓薩里的有尤文在進攻端完成了較好的提速。一旦能將球快速過渡到前場，尤文就能依靠局部的配合或是球星的個人能力去解決問題。

只是，貝納德斯基在持球效用下滑之後，尤文的進攻陷入了較大的問題之中：無法完成快速的持球衝擊，尤文只能依靠緩慢的推進在前場完成陣地攻堅。這時候，意甲較好的防守體系對於中場嚴重缺乏創造力的尤文來說，是一個明顯的遏制。

圖1：貝帥效用下滑，尤文4312體系問題出現

面對都靈與熱那亞，貝納德斯基在前腰位置上的持球效用下滑明顯，對陣都靈他在由守轉攻時只有1次過人嘗試，遭遇到限制失敗；在面對熱那亞時，他在前場的2次持球突破也以失敗告終。

這種局面，是對手洞悉了薩里4312陣型中貝帥的持球效用之後展開的針對性限制。在尤文中場創造力不足的情況下，薩里在進攻端進行了體系的微調，尋求右路的肋部滲透。這時候，C羅更多地來到禁區等待右側的配合傳中，只是尤文右側並未兌現出有效的邊路作業，只能形成局部肋部的滲透打擊。

圖2：尤文右側的肋部作業趨勢

從上圖可以看到，在尤文高位分邊右側的時候，貝納德斯基已經前插到肋部，而且與本坦庫爾形成了前後的站位。這時候，倆人可以形成聯動的縱向配合，只是為了充分利用空間去提速，貝納德斯基在右側肋部接球之後直接斜傳禁區，伊瓜因前插之後完成傳中。

這種戰術體系能在邊路肋部製造進攻縱深，而尋求的配合傳中的實際目的還是希望發揮C羅的搶點能力。只是，尤文的這種肋部作業缺乏更加有效的聯動，很容易被對手針對性的限制。C羅在這種體系下是遊離於核心策略之外的，也是C羅在該時期內球場存在感較低的實質原因。因此，這種想要發揮C羅終結能力的戰術，反而弱化了C羅在球場上的作用。

縱然，C羅雖然能來到右側進行輔助策應，但是年齡增加之後他的爆點突擊能力不及迪巴拉。因此，在C羅身體狀態未達到最佳的情況下不斷被迪巴拉換下，就較為正常，因為薩里想要在邊肋部增加爆點能力。

【強化肋部與中路作業，三叉戟最終確定】

為了扭轉頹勢以及輿論的風評，薩里在此後的比賽中進行了戰術層面的革新。重點，就是球隊逐漸增加了肋部的作業態勢以及中路的串聯。

在肋部強化過程中，薩里將寬度拉開、增兵打擊空位作為作業的重點。而在這個階段，貝納爾德斯基的與誇德拉多的戰術作用逐漸下滑，迪巴拉甚至是皮亞尼奇的戰術地位提升較為明顯。

圖3：主場2-2戰平薩索洛一戰，尤文在肋部瘋狂增兵提升了進攻

這種前插肋部的增兵，讓尤文在禁區外獲得了較多的空位作業機會，皮亞尼奇與博努奇的前插效果十分明顯。

這種通過增加空位兵力投入去製造舒適最後一傳的策略，在後續比賽中又因為增加了鋒線的策應跑動，最終實現了更好地肋部作業效果。這時候的C羅，他在通過自己跑動完成策應的情況下，就能更好的發揮自己在進攻終端的優勢。

圖4：對陣拉齊奧，尤文由迪巴拉、C羅中路策應後完成肋部打擊，C羅包抄得分

上面gif是尤文在進攻端的變化，我們可以清晰地看到，C羅與迪巴拉在中鋒位置上的策應十分關鍵，而這種策應因為邊中場的拉邊傳中效果得到了最大化的展現，均製造了絕佳的射門得分機會。

【進攻方式更多且配合更加有效，C羅再度「攀岩」成功】

當C羅連續4場得分刷新個人征戰意甲的新紀錄時，尤文的進攻體系在賽季中的這次改革中已經有了磨合成型的態勢：迪巴拉佔據十號位，C羅與伊瓜因組成前鋒搭檔的三叉戟，已經展現出了很強的活力。

而在這種局面下，薩里自身的改變也較為關鍵。尤文開始在比賽中尋求皮球運轉速度的提升，並在具備空間的情況下去更加直接的打擊對手的防線身後，這並不是「薩里ball」的特質，但是卻讓現在的尤文運轉的更加流暢。

即便是尤文最近幾場比賽場均創造機會的次數並沒有顯著的提升，但在磨合接近成型的情況下，尤文與C羅迎來了更好地發揮。

圖5：C羅與伊瓜因中路策應分邊，尤文邊路空位傳中製造機會

圖6：伊瓜因與迪巴拉換位，在高位送出直塞，C羅反越位射門得分

今天凌晨，薩里的尤文再度展現出了進攻的變化：三名中場里馬圖伊迪、皮亞尼奇、拉比奧特都很難完成邊路的持球與縱深推進的情況下，尤文通過三叉戟更多地橫向走位，將對方的防線完全壓制在中路。而這時候，尤文的進攻寬度就有了保障，而C羅與迪巴拉倆人在雙側肋部的有效策應，效果明顯。

圖7：C羅與迪巴拉，倆人在雙側肋部的有效策應

當尤文的三叉戟不斷由三角站位取向平行站位的時候，桑普的防線就會被完全吸引，然後無法完成防守寬度的保護。於是，尤文本場比賽在邊路獲得了大量的分邊與傳中機會，而在從中到邊的快速傳遞中，桑普陣型在移動過程中呈現出的空擋，就更加容易讓C羅與迪巴拉去捕捉。

於是，迪巴拉完成了精彩的凌空抽射，而C羅則在後點完成了恐怖的滯空頭槌。

圖8+9：迪巴拉與C羅的兩個精彩進球

兩次簡潔的進攻方式，卻是尤文壓制桑普防線去製造邊路與高位進攻寬度的直接展現。當對手防線處於移動之中時，尤文的傳中作業就能更好地發揮球星的終結能力，這也是C羅最喜歡的進攻方式。

【結語】

在薩里4312陣型中貝爾納德斯基的效用開始下降之後，他對戰術進行了調整，將C羅放在進攻終端並強化右側肋部縱深打擊的體系，磨合併不完善。C羅在身體狀態未達到最佳的情況下，並不是這套體系的核心驅動，而迪巴拉出色的持球能力成為變數，於是，C羅開始被薩里連場換下。

但在後續的整改中，薩里部分摒棄了他的理念，讓尤文逐漸降低球權控制並簡化套路的過程中，以肋部作業與中鋒的策應最終實現了更好地邊中結合。而在這時候，C羅、迪巴拉、伊瓜因就會迎來更加適合自己發揮的空間。如今依然具備出色終結能力的C羅，連續5場比賽取得進球，而今天凌晨的這個美輪美奐的滯空頭槌，則是他這次救贖最完美的寫照。
