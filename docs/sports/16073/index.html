20分鐘砍16分！又一廣東棄將成一流神塔 離進國家隊不遠了？

2019-12-13 1,199條評論 2,886次閱讀 885人點贊

客場不敵山東隊後，山西隊遭遇兩連敗排名也下滑到了第7名，但山西隊這場比賽的表現並非一無是處。除了末節最後時刻瘋狂的追分外，球隊主力前鋒任駿威的發揮也值得表揚。

全場，任駿威先發出場20分鐘，9投7中得到了16分2個籃板，如果不是6犯被罰下，任駿威還有更多的機會去貢獻自己的力量。

這場比賽，山西隊整體表現不佳，一度長時間落後20多分，全隊除了富蘭克林外，就屬任駿威的發揮出色，而他也是富蘭克林外山西隊唯一一個得分超過12分的球員。

任駿威能打出這樣的表現並不容易，因為山東隊的內線實力並不弱。雖然缺乏特別出色的球星，但山東隊的內線儲備和厚度是足夠的，而且山東隊的打法一直非常兇悍，內線球員非常強硬，身高只有2.03米的任駿威在大前鋒的位置上身高處於劣勢，但他還是打出了非常高效的表現。如果不是最後一節因為卡位時和賈誠發生衝突被吹罰犯規，任駿威還能打出更好的表現。

實際上，這個賽季至今，任駿威的表現一直很出色。賽季至今，他場均出場31.1分鐘，場均可以得到職業生涯最高的14.6分，場均還有4.8個籃板2.3次助攻。而且，他的效率不俗，兩分球命中率53%，三分球命中率更是高達52%，場均還可以投中2個三分球。

和弟弟任駿飛一樣，任駿威也有著非常全面的技術和出色的策應能力，而且進攻能力相比任駿飛還要出色，也更加喜歡對抗。就當任駿飛上賽季證明自己、今年夏天入選國家隊後，任駿威也在這個賽季證明了自己。在山西隊表現出色的背後，除了雙外援富蘭克林和莫蘭德的貢獻外，除了神射手原帥的突出表現外，任駿威的貢獻也是不可忽視的。

而從2010-11賽季進入CBA後，任駿威也在第10個賽季真正證明了自己。作為廣東隊的舊將，雖然任駿威在廣東隊表現不好，被諷是廢塔，也未能留隊，但這並不妨礙如今他取得突破。

在和山東隊的比賽後，任駿威已經連續9場比賽得分上雙，而且是連續4場拿到16+，表現非常穩定。無論是內線進攻還是外線遠投，任駿威都有很大的把握，這也讓他變得更加難防。和上賽季相比，任駿威的出場時間是下降的，場均出手次數也幾乎沒有太大變動，但他的數據卻是提升的，可見他的效率有多高。

隨著打出這樣出色的一個賽季，任駿威也再次證明了自己。而2017年夏天就入選過杜鋒男籃藍隊集訓名單的任駿威，也有機會再次獲得杜鋒的青睞，被再次招入國家隊集訓。
