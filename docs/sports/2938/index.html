中國籃球又一次進步！CUBA球隊參戰CBA比賽，姚明真的在努力

2019-12-02 0條評論 25次閱讀 0人點贊

摘要

12月2號，根據著名體育媒體人報導，CBA發展聯盟下一階段賽事將補充進一支特別參賽隊，來自CUBA聯賽的大學球隊山西男籃將參賽。

12月2號，根據著名體育媒體人報導，CBA發展聯盟下一階段賽事將補充進一支特別參賽隊，來自CUBA聯賽的大學球隊山西男籃將參賽。

其中詳細信息寫道：”山西大學男籃將被特邀參加CBDL中國籃球發展聯賽第二階段長治賽區的比賽，作為本階段參賽的唯一一支大學球隊，他們將在小組賽中與上海、深圳、北京、廣西四支球隊交手。”

大學球隊參戰有CBA背景的賽事，這對於中國籃球的發展來說無疑是好事，將會繼續為大學生籃球提供和職業籃球接觸的機會。讓校園籃球更貼近職業，繼續尋求打通壁壘，讓大學籃球能夠有機會有平台去鍛造出更多優秀的籃球人才。

過去兩年，CBA聯賽官方在致力於聯繫CUBA聯賽球隊球員的方面做的極為用心。已是連續兩年都在CBA全明星賽上設定了大學生球員組建的CUBA聯隊和CBA星銳隊對決。讓大學生籃球首次感受和職業籃球的對比。兩年的對決里也為本賽季的CBA聯賽輸送出齊麟、王少傑等優秀的潛力球員。目前CUBA聯賽中多位球員也有著衝擊職業聯賽的希望，甚至是有多位大學生球員的人氣還要高過職業球員。通過媒體宣傳和官方支持，大學生籃球讓外界球迷更加關注。CUBA聯賽能夠沿著正確的方向前進，也是中國籃球的進步！

過去一個夏天，在世界盃輸球的背景下。以姚主席為首的籃協管理層加快多項方案的實施。致力於打造一個好的職業平台提供給各傢俱樂部青訓動力。培養人才。在底層的籃球建設中也繼續加大力度改革，先是提拔以廣西男籃和安徽文一為首的四支NBL職業隊出戰CBA發展聯盟賽事。讓NBL聯賽的強隊能夠有更加好的平台打球。這是一個好的改革方向，充分加重CBA聯賽和次級聯賽的聯繫。

也有多方消息稱，在籃協官方重新收回NBL聯賽的運營權後，未來的NBL聯賽將會合併到CBDL聯賽中，和CBA聯賽共享資源發展。打造出一個多方籃球人，包括大學生球員、草根球員以及職業青年隊被淘汰球員可以追逐職業夢的一個平台。一系列的改革證明官方想要讓中國籃球更好的決心！

所以此次讓山西大學男籃參戰CBDL第二個階段的比賽是非常不錯的。這可以讓大學生球員直面自己和職業邊緣球員的差距，甚至也可以說是找不同。打的好可以增強自身信心，無論比賽輸贏，對於自身進步都有幫助。是可以尋找機會進步的旅程！期待未來有更多的大學生籃球隊進入到和職業隊競技的平台中打球，直面到和職業球員的差距，再去尋求進步，尋求變的更強！
