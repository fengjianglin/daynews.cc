國足贏球他倆卻失意 李鐵親信成回傳帝 昔日主力被同門碾壓

2019-12-19 251條評論 1,288次閱讀 258人點贊

2-0！在稍前結束的一場東亞杯焦點戰中，國足憑藉著吉翔與張稀哲的進球擊敗中國香港，取得了本屆東亞杯的首場勝利，國足2019年度圓滿收官。

本場比賽由於對手實力偏弱，國足在技術、身體上都有很大優勢 ，因此不少球員的發揮都相當出色，前一場被飽受詬病的于大寶本場比賽滿場跑，在攻防兩端都有不錯的發揮；此役首發出戰的吉翔取得了他國家隊生涯的首個進球；此外董學升、譚龍等國家隊新人也交出了一份不錯的答卷。不過在這波球員中依舊有兩人在這場比賽中比較失意，他倆就是李行與姜至鵬。

在武漢卓爾，作為隊長的李行是球隊的中場核心，他在中場的調度指揮對球隊攻防起到至關重要的作用。得益於李行在俱樂部的出色發揮，加上恩師李鐵臨危受命執教國足選拔隊，因此作為一名國家隊新人的他在東亞杯獲得大量的出場機會，可惜李行在這三場比賽的表現辜負了李鐵的信任，尤其是與中國香港一戰，面對實力孱弱的對手，李行的出球還是非常保守，不敢多帶球、多傳向前球，反而多是回傳球，相信經此一戰，李行的國家隊生涯會蒙上一層陰影。

由於前兩場比賽累積兩張黃牌，本場比賽河北華夏幸福邊後衛姜至鵬只能作壁上觀，代替他出戰的是同為根寶基地出來的王燊超。王燊超雖是位後場多面手，但他平時常踢的還是右邊後衛，本次他踢左邊後衛也算是一次客串。然而即便是客串，王燊超在場上也發揮穩健，被對手鏟傷後依舊堅持回到場上，精神鬥志十分高昂。與狀態大幅下滑還做出危險動作的姜至鵬比，那姜至鵬簡直是被這位同門給碾壓，昔日的主力邊後衛是該退位讓賢了。

本屆東亞杯國足1勝2負，這個結果應該在大家的預料之中，由於這支國足二隊集訓時間較短，因此這幾場比賽我們也沒能看出球隊有何技戰術打法，基本就是球到哪，人就衝到哪，有點雜亂無章。2019年度已經接近尾聲，那麼2020年國足是否會有新的改變呢？我們唯有拭目以待。
