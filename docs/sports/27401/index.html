一登場判若兩隊，新疆這種情況，將被廣東輕易掐死

2019-12-20 943條評論 2,068次閱讀 1151人點贊

新疆主場迎戰同曦，本來是一場毫無懸念的比賽，同曦每個賽季都處於聯盟副班長的位置，一個重要的原因就是外援實力一般，國內球員沒有拔尖的存在。而由於菲爾德的受傷，近幾場比賽新疆由於菲爾德的受傷，進攻端火力下降非常明顯。

CBA第十八輪我們可以判若兩隊的新疆，由於剛剛復出，還沒有調整最佳狀態，沒有跟上球隊的節奏，第一節下半段下場之後，阿的江沒有再讓菲爾特登場，他在板凳席座了一節半的時間，而第一節新疆的到了可憐的22分，而第二節新疆僅得19分，如此低效的進攻。第三節菲爾德重新登場，新疆分別第三節37分，第四節29分，進攻明顯提升幾個檔次。

菲爾德的受傷新疆只能選擇讓曾令旭成為球隊的首發組織後衛，在上賽季的季後賽我們已經看到了曾令旭的能力有限，在看看新疆上下半場的進攻水平，雖然曾令旭推進同樣很快，但是出球的水平實在相差太遠，看看周琦就知道了，菲爾德推進，周琦跟進，菲爾德在吸引防守之後，輕鬆分球周琦完成暴扣，但是曾令旭明顯不具備這樣的實力。

倘若後衛線新疆之後菲爾德一點，季後賽再次面對廣東宏遠的時候，只要廣東宏遠能夠掐死菲爾德，消耗菲爾德，那麼，新疆想要戰勝廣東宏遠，將變成不可能完成的任務，更不要說新疆能夠很廣東宏遠爭奪總冠軍？若不改變，本賽季的新疆或許還是與總冠軍無緣。
