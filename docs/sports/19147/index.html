NBA之後，英超球員也曝出不當言論！英超此前在華拿下50億合同

2019-12-17 1,338條評論 3,128次閱讀 1852人點贊

本文為「金十數據」原創文章，未經許可，禁止轉載，違者必究。

據媒體12月16日最新消息，上周五（12月13日），英超球隊——阿森納隊員厄齊爾疑似發表不正當言論。隨後，阿森納俱樂部曾在微博上回應稱，厄齊爾所發表的內容均為其個人觀點。12月15日當天，阿森納俱樂部官網對此事件回應道，這是一個涉及到嚴肅話題的討論，所以我將把它留給俱樂部層面來處理。

資料顯示，厄齊爾是一名來自德國的中場球員，目前效力於英超阿森納俱樂部，2014年德國隊奪取世界盃冠軍的主要成員。此前，這名德國球員曾深受中國球迷喜愛，因為其中國譯名及長相酷似阿凡達，曾被中國球迷取外號為”272″或”阿凡達”。據報道，今年年中，厄齊爾的經紀人Dr. ErkutSögüt在接受採訪時還表示，厄齊爾不僅會考慮未來到中國踢球，也計劃與更多中國品牌進行合作。

值得一提的是，今年10月，美國籃球協會NBA休斯敦火箭隊的總經理莫雷曾因為發表不當言論，導致球隊比賽在中國被停播。據悉，在13日事發之後，央視取消了原定於周日（15日）直播的阿森納同曼城的比賽。據此，不少外國媒體猜測，在此事件帶來影響之下，阿森納和英超比賽或將會為厄齊爾的言論埋單。

據了解，英超是歐洲五大聯賽中商業化程度最高的賽事。德勤發布的《2018年度足球財務報告》顯示，英超以52.97億歐元（約合人民幣409.52億元）的收入位居第一，成為聯賽層面當之無愧的”吸金王”。而這強大的吸金能力背後，還源於英超聯賽擁有龐大數量的中國球迷。今年6月，優酷體育發布的《五大聯賽數據報告》顯示，英超在中國的球迷數高居歐洲五大聯賽之首。

報道指出，2016年蘇寧通過旗下PP體育拿下2019-2021賽季英超中國獨家媒體版權，版權費高達7.21億美元（合50億人民幣）的天價，創下了英超海外版權價格記錄。據此，中國藉此超過美國成為英超最貴的海外轉播權交易市場。而毫無疑問，中國市場也成為了英超聯賽最重要的收入來源。然而，此番厄齊爾做出了這一意外舉動下，英超聯賽在中國市場的收入會不會受到影響呢？我們還不得而知。截至發文，英超聯賽負責人拒絕對厄齊爾這件事發表評論。
