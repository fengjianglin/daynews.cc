NBA向各隊發2021-22賽季四大改革建議

2019-12-21 298條評論 4,077次閱讀 2664人點贊

據The Athletic記者Shams Charania報道，NBA已經向各隊發送了2021-22賽季改革建議：

常規賽賽程縮減至78場

增加季中錦標賽，冠軍隊球員每人100萬，教練組150萬美元獎金。

引入季後賽搶位賽，共8支球隊爭奪東、西部7-8名。

分區決賽4支球隊根據常規賽戰績，重新分配對陣。

季中錦標賽將從11月底開打，12月中旬結束。所有30支球隊都要參加，分區比賽為4主4客，計入常規賽賽程。

6支分區冠軍加上剩下2支戰績最佳球隊進入淘汰賽。1/4決賽在參賽球隊的主場舉辦，半決賽/決賽在第三方主場舉行。

根據此前報道，算上可能的錦標賽，以及季後賽搶位賽，一支球隊有極小的可能最多打83場比賽（78場常規賽，3場杯賽淘汰賽和2場季後賽搶位賽）。

如果一支球隊在錦標賽未能出線，且無緣東、西部前十，只需要打78場比賽。

季後賽搶位賽規則是東、西部7-10名球隊競爭各自分區的2個季後賽名額。第7對第8，勝者獲得第7；第9對第10，勝者將和另一分組的輸球方競爭第8。
