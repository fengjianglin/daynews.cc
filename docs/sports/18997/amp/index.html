瓜迪奧拉確認阿爾特塔已與阿森納有過商談，未來決定尚未可知

天天要聞
2019-12-18

西班牙人承認他的副手已經和槍手聯繫過了，但是他在這個時期仍然需要依靠他的同胞。

曼城主帥瓜迪奧拉證實，阿爾特塔已經和阿森納就接替埃梅里在酋長球場的位置進行了商談。

阿特塔已經成為槍手下一任永久教練的頭號人選，他在周一早上與俱樂部總經理維納伊·文卡特舍姆進行了會談。

夠力足球了解到，雙方的談判將繼續進行，最終決定將於本周末做出。

阿爾特塔之前沒有任何管理經驗，但是他在伊提哈德球場擔任瓜迪奧拉的助理教練已經長達三個賽季。

在此期間，曼城贏得了七項重要榮譽，包括英超聯賽的背靠背冠軍，在打破各種記錄的同時，充分確立了自己在英格蘭足球中的主導力量。

瓜迪奧拉的球員們將在周三對陣牛津聯隊的四分之一決賽中繼續捍衛他們的聯賽杯的獎盃。

儘管人們一直在猜測他的未來，但這位曼城主帥透露，阿特塔將會和他一起出現在格勒諾布爾路的教練席，但是他還不確定他是否能夠繼續依靠這位37歲的副手。

在周二的新聞發布會上，瓜迪奧拉被問及阿爾特塔的立場時，他說：「我知道你所知道的。今天我看見他了。他和我們一起前往客隊，是的。」

「我們很聰明，把他挑選出來，現在其他人都想要他。他是一個不可思議的人，一個很好的經理，這就是為什麼他在我們這裡表現得很好。」

「他正在和阿森納談判，所以我不知道接下來會發生什麼。」

瓜迪奧拉補充說:「是的，我知道了。他是個成年人，知道該怎麼做。他對我非常誠實。」

「俱樂部的組織架構總是充滿活力的，它並不總是一成不變的，今天管用的東西可能明天就不管用了。今天的在的人可能明天就不在了。」

「現在他是我們團隊和員工的一部分，他還在這裡。我們有任何新消息，我都會知道，你也會知道，然後我們再看看會發生什麼。」

「這樣的事情發生過。我們讓李·諾貝斯去了利物浦，現在球隊需要米克爾。他知道我們想要什麼，我們喜歡什麼，但最終，這是他的決定。我們三年前就決定合作，我希望他周三能來這裡。」

曼城在C組以小組第一的成績鎖定了冠軍聯賽淘汰賽階段的席位。

周一在瑞士抽籤後，英超冠軍將在16強遭遇皇家馬德里，瓜迪奧拉將與他在巴塞羅那時代的老對手重新開戰。

「這對我們來說是一次難以置信的考驗。這個賽事的王者，我們會儘可能的做好準備，我們還有兩個月的時間，我們將會在二月份看到我們的情況。」前巴薩主帥補充道。
