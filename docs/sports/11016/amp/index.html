巴塞羅納：五位理想的候選人將取代巴爾韋德

天天要聞
2019-12-12

巴爾韋德並不是巴塞羅那最受歡迎的主帥，他似乎正處於離開俱樂部的邊緣。這些可能是他的替代品。

巴爾韋德自從來到巴塞羅那後就沒能贏得諾坎普球迷的喜愛。他現在已經是第三個賽季了，雖然他已經取得了西甲聯賽的冠軍，但歐冠的長期難以捉摸的勝利仍然讓俱樂部頭疼。

現在市場上有很多有能力的教練接替巴爾韋德的位置，西班牙人很快離開俱樂部也就不足為奇了。

這裡有五位非常有能力的候選人，他們有可能成為下一任巴薩主帥。

1、埃里克·滕哈格

毫無疑問，阿賈克斯教練是目前世界上最好的年輕教練之一。自2017年接管阿賈克斯以來，他在阿賈克斯做了出色的工作，上賽季在歐冠聯賽中大放異彩。

4-1戰勝了皇馬，帶領他的球隊進入了歐冠半決賽，最後輸給了托特納姆。

埃里克·滕哈格可能是巴爾韋德的完美替代者，也是一個充滿靈感和雄心勃勃的教練。隨著阿賈克斯和巴薩多年來的良好關係，這將進一步鞏固兩傢俱樂部之間的聯繫，而埃里克·滕哈格也可以發揮德容法國人的最佳水平，同時也可以從拉馬西亞引進更多巴薩豐富的青年才俊。

2、瓜迪奧拉

現任曼城主帥本賽季在英格蘭隊的場邊並不是一個令人滿意的人物。瓜迪奧拉似乎離聯賽冠軍越來越遠。

瓜迪奧拉似乎在俱樂部受到傷病的困擾，目前曼城落後利物浦14分。即使是在這個賽季的這個階段，這個差距似乎也太大了，曼城需要奇蹟般的恢復才能推翻它。

瓜迪奧拉顯然對曼城不滿意，情緒低落，他似乎已經在俱樂部完成了自己的使命，他需要一個新的挑戰。他回到巴塞羅那可能是瓜迪奧拉重建對比賽激情和活力的絕佳機會。

他肯定想念這個城市和俱樂部里的人。他的回歸可能會重新點燃他的激情，瓜迪奧拉知道俱樂部的內部和外部，並帶領他們取得了重大成功之前，他將不需要時間來解決。

最近瓜迪奧拉的父親維亞馬爾卡宣布，他的兒子可以在未來回到巴塞羅那，這將增加更多的爆點。

3、科曼

排名第三的是另一名前巴塞羅那球員和教練科曼。荷蘭人球員時代在加泰羅尼亞度過了巔峰，在90年代末和21世紀初也是一名助理教練。總的來說，科曼在俱樂部呆了近10年，他對巴塞羅那的了解是非常全面的。

他在費耶諾德和南安普頓表現不錯，但在埃弗頓只取得了部分成功。儘管最近馬爾科席爾瓦已經證明了太妃糖的工作對一個有成就的教練來說是多麼的艱難。

從那時起，科曼接管了荷蘭國家隊，希望能重振雄風，讓球隊回到昔日的輝煌。到目前為止，科曼已經交出成績單。

科曼可能正是巴薩所需要的人，憑藉他重建荷蘭的歷史記錄，他可以在加泰羅尼亞人身上做同樣的工作，對俱樂部已經非常熟悉，這將極大地幫助他。

4、波切蒂諾

把你的注意力放回今年早些時候的五月。波切蒂諾被認為是世界上最好的教練之一。他的托特納姆熱刺隊在歐冠聯賽中一飛衝天，不幸沒有在決賽中戰勝利物浦。

波切蒂諾建立了一個完美的進攻風格，這為熱刺創造了奇蹟。阿根廷人幾乎白手起家，把倫敦隊打造成了英格蘭最好的球隊之一。他以前在南安普頓也做過出色的工作，他的創新思維和爆髮式的打法讓他成為一個令人興奮的管理前景。

托特納姆在賽季開始的時候在他的領導下一直在掙扎，當波切蒂諾被倫敦俱樂部放走的時候，丹尼爾利維的無情再次襲來。儘管很多人會說他的解僱是不公平的，他沒有足夠的時間扭轉局面。無論波切蒂諾在被解職前的成績如何，他都不應被忽視，他仍然是這場比賽中最好的後起之秀之一。

或許在巴薩這樣一支有足夠時間的球隊里，波切蒂諾可以再次展翅高飛，展現出他在熱刺發展起來的同樣吸引人的進攻風格。

歐冠是巴薩最夢寐以求的獎盃之一，再加上他願意帶著年輕球員和作為梅西的同胞，如果給他機會，他可以在諾坎普茁壯成長。

5、哈維

這位前巴塞羅那隊長和俱樂部的傳奇人物可能會在這個名單上落幕，但哈維最近已經進入了管理層，他的下一個目的地可能是加泰羅尼亞人。

哈維目前在卡達執教薩阿德俱樂部，他在那裡做得很好，讓球隊打出了一個精彩的進攻風格體系。

哈維幾乎在巴塞羅那度過了他的一生，他的血管里流淌著加泰羅尼亞人的血液，他知道球迷想要什麼，他知道需求和期望。很明顯，他將教練視為未來的職業。

如果給哈維機會，他可能會和瓜迪奧拉產生同樣的影響。儘管完全取代巴爾韋德對西班牙人來說可能來得有點早。相反，哈維應該像瓜迪奧拉在加泰羅尼亞隊開始他的教練生涯之後才慢慢晉陞的那樣，先成為巴薩二隊教練。

這將是一個理想的情況，但是如果巴薩在巴爾韋德的帶領下的狀態繼續下降，哈維可能會以比預期更快的速度被引進，而他作為主教練的到來可以激勵球迷和在場上的球員將他們的比賽帶到另一個水平。
