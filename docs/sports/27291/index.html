哲科2傳1射扎尼奧洛傳射建功，羅馬客場4-1佛羅倫薩

2019-12-21 282條評論 1,638次閱讀 663人點贊

虎撲12月21日訊 北京時間12月21日凌晨03:45，2019-2020賽季意甲聯賽第17輪比賽繼續進行，羅馬客場挑戰佛羅倫薩。上半場，弗拉霍維奇破門被吹越位，羅馬3分鐘內連入兩球，哲科和科拉羅夫破門，巴代利為主隊扳回一球；下半場，佩萊格里尼破門鎖定勝局，哲科助攻扎尼奧洛破門錦上添花。最終，羅馬客場4-1戰勝佛羅倫薩。此役過後，羅馬10勝5平2負積35分暫居第4位，佛羅倫薩4勝5平8負積17分暫居第14位。

開場第6分鐘，韋勒圖前場左路橫向盤帶右腳低射，皮球被德拉戈夫斯基沒收。第3分鐘，弗拉霍維奇左腳自由球直接射門，皮球高出橫樑。第12分鐘，弗拉霍維奇禁區內倒地鏟射，皮球被洛佩斯擋出，大博阿滕禁區內右側右腳小角度抽射，皮球再次被洛佩斯撲出底線。

開場第13分鐘，卡塞雷斯前場左路直塞，戴爾伯特左腳橫傳中路，弗拉霍維奇跟上左腳抽射破門，主裁判示意，弗拉霍維奇越位在先，進球無效。

第19分鐘，佩萊格里尼大禁區弧頂外右腳挑傳，扎尼奧洛禁區內右側倒地右腳橫掃門前，哲科右腳推射首開紀錄，佛羅倫薩0-1羅馬。


第21分鐘，科拉羅夫左腳自由球直接射門，皮球如炮彈一樣直掛球門死角，佛羅倫薩0-2羅馬。


第34分鐘，佛羅倫薩前場傳中，羅馬禁區內連續頭槌解圍未果，佛羅倫薩外圍遠射被擋，羅馬後防配合失誤，巴代利得球倒地右腳掃射扳回一球，佛羅倫薩1-2羅馬。


上半場，羅馬客場2-1領先佛羅倫薩。

易邊再戰第61分鐘，卡斯特羅維利前場左路隱蔽的斜傳球被洛佩斯抱住、3分鐘後，卡斯特羅維利禁區內倒地，主裁判判卡斯特羅維利假摔並向其出示黃牌。

第73分鐘，羅馬快速反擊，佩萊格里尼前場帶球橫傳，哲科禁區前沿得球橫敲，佩萊格里尼右腳低射破門鎖定勝局，佛羅倫薩1-3羅馬。


第88分鐘，羅馬快速反擊，哲科中場送出直塞球，扎尼奧洛前場右路帶球奔襲突入禁區左腳推射遠角破門，佛羅倫薩1-4羅馬。


第92分鐘，弗拉霍維奇左腳自由球直接射門，皮球越過人牆擊中橫樑彈出。

最終，羅馬客場4-1戰勝佛羅倫薩。

出場陣容：

佛羅倫薩（352）：69-德拉戈夫斯基；22-卡塞雷斯、20-佩澤拉、4-米倫科維奇；29-戴爾伯特、8-卡斯特羅維利（83′，16-埃塞里克）、5-巴代利、78-普爾加、21-科羅拉（83分，11-索蒂爾）；28-弗拉霍維奇、10-大博阿滕（66′，9-佩德羅）；

羅馬（4231）：13-保羅-洛佩斯；11-科拉羅夫、6-斯莫林、23-小曼奇尼、24-弗洛倫齊；21-韋勒圖、42-迪亞瓦拉；8-佩羅蒂（76′，77-姆希塔良）、7-佩萊格里尼（85′，17-雲代爾）、22-扎尼奧洛（91′，37-斯皮納佐拉）；9-哲科；
