恆大或將改名？足協改名嚴令迫在眉睫，8冠王位列其中

2019-12-16 502條評論 1,540次閱讀 360人點贊

足球是大家最喜歡的運動項目，關於足球的一系列賽事也是球迷們最關注的事情，中超聯賽成為了喜歡觀看的比賽，很多球隊的實力也是相當的強勁，上海申花、廣州恆大以及北京人和等等都是大家非常喜歡的隊伍，但是球迷們不僅僅關注的球隊的表現，對於中國足協的一系列規定也是球迷們關心的事情。

中國足協的規定中表示中超所有的球隊都要在3年之內將所有的名稱轉換為中性化，而中超聯賽結束值周意味著這樣的規定開始了倒數，但是很多俱樂部對於這樣的規定還是非常的不喜歡，但是對於球迷來說還是非常的開心，畢竟這是中國足協截至目前最好的規定之一，廣州恆大是大家非常熟悉的隊伍，在比賽中有著「八冠王」的稱號，出現這樣的政策之後意味著廣州恆大也得改名了。

眾多隊伍中。山東魯能、北京國安、大連一方、上海申花、天津泰達、天津天海以及河南建業這些球隊出現這樣的政策之後是不用改名字的，剩下了9支球隊都是需要改名字的，首先就是廣州恆大，因為他的名字有很大的商業性質，所以說廣州恆大不得不改名了，但是這樣的結果對於廣州恆大來說還是非常遺憾的事情，畢竟恆大集團在成為廣州足球的老闆之後，廣州恆大的進步非常的迅猛。

廣州恆大直接完成看球隊最無法想像的三級跳，他們從中甲冠軍隊伍成為了中超冠軍的隊伍，之後更是成為亞冠冠軍，成就也是非常高了，對於球隊來說意義也是相當的大，廣州恆大在中超賽場上還拿到了八次冠軍的獎盃，而廣州恆大的隊伍在球迷們心目中的地位也是非常的高，尤其是在冠軍的獎盃上面刻了廣州恆大的名字，也就意味著在各大賽事中都有廣州恆大的名字，所以說換名字這件事情對於廣州恆大並不是好事，不過從另一個方面來看也是好事。

很多球隊都是想成為百年的老球隊，那麼球隊的名字也是相當的重要，維持球迷們基礎也是相當的重要，不過亞足聯關於廣州恆大的名字還是只認原隊名，在亞冠的賽場上廣州恆大改名的話，那麼他們的名字應該不會在賽場上出現了，畢竟當時廣州恆大的名字改為了廣州恆大淘寶，那個時候就沒有在亞冠賽場上出現。大家認為中國足協這樣的規定對於球隊來說能否有效呢？
