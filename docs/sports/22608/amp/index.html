皇馬好消息！曝穆里尼奧清洗1.22億歐三巨星，埃里克森或離隊

天天要聞
2019-12-19

北京時間12月19日，英國媒體《每日電訊報》透露，在1月份的冬窗開啟後，熱刺主帥穆里尼奧會進行一定程度的清洗，丹尼-羅斯、萬亞馬、埃里克森都有可能離開北倫敦球隊，其中，萬亞馬的去向很可能是中超聯賽，而埃里克森受到了皇家馬德里的關注。英媒指出，穆里尼奧希望對陣容進行瘦身，這也是他一直以來的想法！

當第一次來到切爾西的時候，穆里尼奧就承認，他希望陣容里有21名球員，而來到熱刺的時候，他繼承了波切蒂諾留下的25名球員，很顯然，如果穆里尼奧堅持自己的想法，他就需要對熱刺的陣容進行簡化。

英媒透露，在1月份的冬窗到來之際，穆里尼奧希望萬亞馬可以離開俱樂部，並且計劃聽取外界對於英格蘭後衛丹尼-羅斯的報價，而埃里克森在合同在明年夏天到期，他也可能在冬窗離開。按照《轉會市場》提供的數字，目前萬亞馬的身價是1200萬歐元，羅斯的身價是2000萬歐元，埃里克森的身價是9000萬歐元，三人加起來的身價達到了1.22億歐元。

28歲的萬亞馬是英超賽場上的獸腰，他也已經在熱刺效力了三年，但是自從4月份以來，他就沒有在英超賽場首發過。據悉，中超俱樂部對萬亞馬感興趣，而柏林赫塔也有意簽下萬亞馬。不過從財力來看，柏林赫塔無法和中超俱樂部抗衡。《每日電訊報》倒是沒有指出究竟是哪一支中超球隊有意引進萬亞馬，但是這頭英超巨獸來到中超的話，必然會成為統治力極強的中場。

羅斯的情況比較複雜，目前他堅持希望履行完自己和俱樂部還剩18個月的合同，不過穆里尼奧不怎麼喜歡羅斯，上任以來，穆里尼奧將維爾通亨移到了左路，而不是使用羅斯。目前羅斯還希望代表英格蘭參加歐洲杯，因此，如果冬窗離隊的話，有利於他獲得出場時間，從而獲得索斯蓋特的青睞。

埃里克森和熱刺的合同在2020年到期，目前丹麥球星拒絕續約。埃里克森受到了皇馬的關注，不過列維也承認，不排除同英超球隊做交易的可能性，「老實說，我不排除和英超競爭對手進行交易。」無論如何，熱刺願意送走埃里克森，這是皇馬的好消息！

英媒同時承認，如果穆里尼奧希望在1月份加強陣容的話，那麼他就需要出售球員。補強對象來看，穆里尼奧希望得到一名替補前鋒，來減輕哈里-凱恩的壓力，同時他還希望得到一名後衛球員來加強球隊防守。
