字母哥：我從來不會說超越詹姆斯這種蠢話，因為我知道和他的差距

天天要聞
2019-12-17

北京時間12月17日，今日雄鹿隊當家球星揚尼斯.安特托孔波在接受《密爾沃基哨兵報》記者Matt Velazquez採訪時，被問到了是否有信心超越湖人當家勒布朗詹姆斯。字母哥表示自己不管是公開還是私下，都不會說出自己會超越詹姆斯這種蠢話，因為自己非常明白雙方之間的差距。同時字母哥還認為，在現役球員中，沒有任何一個人有希望超越詹姆斯。現在的球員能做的只是努力追趕詹姆斯的腳步，而不是超越他。

在被問到是否有信心超越勒布朗詹姆斯時，字母哥回答道：「我從來不會說超越勒布朗這種蠢話，不管是在公開還是在私下裡，我都不會這樣說，我甚至沒抱有這種想法。我是一個非常現實的人，所以我清楚我們之間的差距不是時間和努力可以填平的。勒布朗已經取得了令人生畏的榮譽，我幾乎不可能在剩餘的職業生涯中取得比他更多的榮譽，除非我能夠打到50歲再退役。」字母哥說道。

「而且勒布朗之所以偉大並不是因為他在球場上所取得的榮譽，他是一個具有真正運動員精神的傢伙，他教會了我們生活大於籃球，也教會了我們利用籃球為這個社會創造價值。如果想要成為這個聯盟的代表人物，那麼你必須要得到所有球員的承認，而在現在的聯盟中，也只有勒布朗擁有這樣的資格。」字母哥說道。「在現在的球員中，不管你得到了多少個總冠軍，或者是取得了多少重要的榮譽，你都只能去追趕勒布朗的腳步，而不是去超越他。如果你沒有像勒布朗一樣讓年輕人和全聯盟承認你，你就沒資格取代他的地位。」

在過去五年中，詹姆斯只得到了一座總冠軍，甚至去年還沒有進入季後賽，那麼為什麼他還是人們心中公認的聯盟第一人呢？就是因為詹姆斯的代表性和表率作用！想要成為這個聯盟的牌面，代表這個聯盟所有的球員，不光是球打得好就行。詹姆斯的成就一直都不局限在球場上，修建學校，成立基金，資助貧困的兒童，詹姆斯無時不刻的在教導這個聯盟的年輕人，推動著他們成為真正的運動員。而那些打了兩年好球就妄想超越詹姆斯的人，簡直就是痴人說夢，。？
