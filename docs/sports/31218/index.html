安家傑愛將成山東功臣！王一梅缺陣遼寧吞蛋，外援重炮空砍得分王

2019-12-25 858條評論 1,874次閱讀 914人點贊

女排聯賽第5-8名排位賽，李岩龍挂帥的山東女排坐鎮張店區體育館主場，激戰四局以3-1（25-23、23-25、25-23、26-24）險勝遼寧華君。安家傑愛將杜清清作為替補接應登場，為山東狂轟20分成為得分頭號功臣，主攻宋立琪和對角宋美麗、副攻楊涵玉與張倩各自砍18、11、10和12分。遼寧重炮王一梅被主教練趙勇雪藏並未登場，亞塞拜然重炮馬馬多娃空砍26分冠全場，攜手進賬19分的副攻胡銘媛未能救主。

​江蘇中天鋼鐵同樣是在常州體育館主場，面對八一南昌力拚四局以1-3（18-25、22-25、25-9、23-25）失利，就算隊長張常寧主接22次一傳只失誤1次，並且狂轟25分榮膺得分王卻難阻東道主敗下陣來，世界盃冠軍接應龔翔宇和主攻吳晗各拿18和15分，主教練蔡斌想要衝擊六強席位深陷被動局面。即便是重炮劉晏含並未隨隊出征，袁心玥只打一局也無礙八一女排闖關，主攻王雲蕗和副攻高意各砍15和13分。

​女排聯賽第9-14名排位賽，浙江嘉善西塘古鎮直落三局以3-0（25-19、25-23、25-16）零封四川勁浪體育，雲南大學滇池學院同樣是以3-0（25-20、25-23、25-16）送蛋福建安溪鐵觀音，河南銀鴿投資激戰四局以3-1（25-13、25-23、13-25、25-19）擒獲河北海瑞。由此一來，浙江和雲南以及朱婷老東家河南，在競爭第9、11和13名三戰兩勝制中搶佔先機。需要注意的是，浙江、雲南與河南均是主場闖關，下一輪移師客場能否延續勝利值得關注。

​備受關注的兩場半決賽，上海光明優倍激戰四局以3-1（25-23、29-27、24-26、32-30）驚險擒獲廣東恆大，天津渤海銀行直落三局以3-0（25-15、25-19、25-23）零封北京汽車。朱婷時隔19天帶傷復出狂轟23分令人驚喜，但上賽季冠亞軍交鋒一邊倒確實出乎意料之外。按照半決賽次回合的對陣安排，廣東恆大返回主場靜候上海光明優倍，天津渤海銀行移師客場過招北京汽車，上海和天津倘若再勝就將攜手晉級總決賽。
