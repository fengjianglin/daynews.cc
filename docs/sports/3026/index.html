北京時間12月2日0點30分，英超第14輪迎來一場比賽，曼聯在主場迎戰阿斯頓維拉。此前，紅魔積17分位列中游，而維拉積14排名第16。最終本場比分2-2，曼聯在主場戰平阿斯頓維拉，積18分位列聯賽第9，僅次於19分的阿森納。

馬夏爾擔任曼聯中鋒，拉什福德和詹姆斯分居兩側，弗雷德、馬塔和佩雷拉組成3中場。

阿斯頓維拉本場上3前鋒陣容，格拉利什佩戴隊長袖標，與埃爾加齊和威斯利組成三叉戟。

第11分鐘，格拉利什在禁區左側兜射破門！0-1，維拉反客為主！

第42分鐘，拉什福德門前頭槌爭頂，維拉門將希頓不慎將球從手邊滑入網中，官方認定為烏龍球！1-1，曼聯將比分扳平！

半場已過，比分暫為1-1，曼聯在主場與維拉平局，看下半場比賽如何發展。

第63分鐘，萬比薩卡嘗試一腳大力抽射，被門將擋出。

第64分鐘，弗雷德傳中，林德洛夫頭槌破門！2-1，曼聯比分領先！

第66分鐘，明斯門前將球打進！2-2，維拉立即再次將比分扳平！

第86分鐘，馬奎爾力壓對手完成頭槌，被門前的明斯擋出。

全場戰罷，曼聯主場戰平阿斯頓維拉，積18分位列聯賽第9，僅次於19分的阿森納。紅魔多次進攻機會均被對手化解，本場比賽難獲3分。
