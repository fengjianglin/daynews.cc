又一個適合CBA超級外援！在NBA曾單場砍43分 廣東遼寧都可追他？

2019-12-25 777條評論 1,702次閱讀 155人點贊

以底薪合同加盟爵士隊後，傑夫-格林還是沒能避免將被裁掉的命運。作為當年的天賦球星，格林在NBA始終沒能完全證明自己，現在被裁掉後雖然被看好到一些爭冠球隊當配角，但可以預見的是，格林即便加盟湖人等隊能起到的作用和獲得的機會也是有限的。而如果他不介意，他實際上完全可以選擇來CBA展現實力。

作為一個出生於1986年的老將，33歲的格林身高2.06米，是一個可以打大前鋒和小前鋒的全能球員，有不錯的持球進攻能力，防守能力同樣不錯。與此同時，出色的運動能力和全面的技術也成為了他在NBA立足的關鍵。

2007年，格林在首輪第5順位被凱爾特人隊選中，值得一提的是，易建聯就在第6順位被雄鹿隊選中。之後，格林被交易到雷霆隊，和杜蘭特、威斯布魯克組成了最早的「雷霆三少」。隨著哈登隨後進入NBA並證明自己，格林這才逐漸被取代了位置。

而在整個NBA期間，格林場均可以得到13.2分4.5個籃板1.6次助攻，表現相當不錯，而且他曾經單場拿到過43分。無論是在雷霆隊、凱爾特人隊、灰熊隊還是魔術隊、騎士隊、奇才隊等隊，他都有著不錯的表現。在和詹姆斯、韋德成為騎士隊隊友時，格林曾經有過可圈可點的表現。所以當格林今年夏天再次簽下底薪合同時，就連韋德都為他抱不平說：「我不能理解他為什麼連續三年都只簽下底薪，他運動能力不俗，還有三分球能力，可以防守多個位置，而且他也不算太老。」

但隨著這次被裁掉，格林也證明了他在NBA獲得足夠的認可並不容易，而即便是加盟其他爭冠球隊，在其他球隊陣容已經飽和的情況下，格林能夠扮演的角色也是有限的。

不過對CBA來說，格林卻可以成為大殺器。以格林出色的運動能力和全面的技術，如果他願意來CBA，他完全可以打出非常出色的表現。無論是低位強攻還是快攻，他都可以打出非常好的表現。而且，他的打法非常強硬，技術又很全面，還可以通吃大小外援兩個位置，身高也不吃虧，如果來CBA，無論是誰想防住他都不容易。

就目前來說，廣東隊、遼寧隊甚至是新疆隊實際上都可以去追求格林，如果能得到他，整體實力勢必會得到非常大的提升。而格林無論是持球還是無球都有不俗的威脅，比起史蒂芬森要更加適合CBA。而且他的整體實力要超過威姆斯、布魯克斯等人。如果CBA強隊可以開出不俗的合同，相信他們也可以簽下格林，就如同之前簽下馬尚-布魯克斯一樣。
