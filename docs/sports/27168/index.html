曝熱刺核心冬窗轉投曼聯！轉會費僅2500萬 穆帥仍在極力挽留

2019-12-22 929條評論 3,322次閱讀 448人點贊

經過長時間的拉鋸戰之後，阿爾德韋雷爾德終於和熱刺完成續約。目前，熱刺陣中僅剩埃里克森和維爾亨通尚未續約。對於這兩位球員，穆里尼奧都表達了挽留之意。但從目前的情況來看，埃里克森很難留在熱刺。冬窗轉會，丹麥國腳大概率將會加盟曼聯。

據英國《鏡報》報道，由於埃里克森一直拒絕續約，且合同到本賽季結束便要到期，熱刺最終決定在一月份拋售埃里克森止損。此前，熱刺主席列維一直拒絕將埃里克森出售給英超競爭對手。如今他也改變了自己強硬的態度。列維表示： 「老實說，我們並不害怕與對手交易。」

今年夏天，曼聯便曾向熱刺報價埃里克森，只可惜遭到了拒絕。現如今，曼聯依舊有意在冬窗補強球隊中場，埃里克森仍是最合適的人選。相比於夏窗轉會，埃里克森的轉會費用更低了。熱刺為了順利出售埃里克森，只給他標價2500萬英鎊，堪稱白菜價。

埃里克森司職前腰，2013年加盟熱刺。過去幾年裡，埃里克森一直是熱刺主力。上賽季，熱刺在歐冠打入決賽，埃里克森在12場比賽中貢獻了2球4次助攻。本賽季，埃里克森出場16次，攻入2球。穆里尼奧上任後儘力在淡化埃里克森的作用，他並沒有給埃里克森多少機會。

穆里尼奧不用埃里克森，完全是處於俱樂部的利益考慮。從他本人的角度出發，他對於埃里克森欣賞有加。從加盟球隊之初，穆帥便呼籲埃里克森續約留下。直到現在，穆帥的態度依舊沒有改變。在與切爾西的賽前發布會上，穆帥表示：「關於埃里克森，我只能告訴你一件事，那就是我希望他續約留下。除此之外，我什麼也不會告訴你。」
