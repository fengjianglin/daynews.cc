穆里尼奧再出金句！“我給梅西投票難道還要徵求C羅的同意嗎？”

天天要聞
2019-12-17

上輪英超，熱刺在客場2-1拿下big6殺手狼隊，一路攀升到了聯賽第五。但相比於比賽的結果，穆里尼奧在賽前和記者的嘴炮顯然更為引人注目。

眾所周知，蘭帕德是穆里尼奧最得意的弟子之一，在談到恩師穆里尼奧時，蘭帕德總是不吝讚美：「他給了我信心。他是一名戰術大家，無論是訓練還是賽前準備，他都完成得非常出色。當我還在他麾下效力的時候，他總是拉著我給我講解戰術細節，他真的非常偉大。」

本賽季初，由於賦閑在家，因此穆里尼奧經常會作客天空體育點評英超各支球隊的表現，據悉，穆里尼奧曾在蘭帕德0-4輸掉揭幕戰後發簡訊對他進行了批評。穆里尼奧表示，蘭帕德應該在老特拉福德啟用更有經驗的老將，而不是擺出一套「年輕人」陣容，但蘭帕德回應自己手中已經是「無人可用」。

不過儘管觀點衝突，穆里尼奧還是看好愛徒的工作，他表示蘭帕德會成為一名非凡的教練，他需要的是時間。因此，在近期遭遇2連敗並且下輪就要面對恩師穆里尼奧的情況下，記者向穆里尼奧拋出了一個頗為尖銳的問題。

「下場比賽就要您對陣蘭帕德了，我們知道目前切爾西已經遭遇兩連敗了，如果你贏得了本場比賽（對陣狼隊），那你有什麼想對蘭帕德解釋（講解）的嗎？」

拋出該問題的記者話音未落，穆帥就立馬反問記者：「我給梅西投票，難道還需要事先徵求C羅的同意嗎？」穆帥此言初看還真有點費腦筋，但他的意思是，我們都是英超球隊主帥，地位相當，自己沒有任何資格對蘭帕德的工作指手畫腳，當然自己是評論員的時候就不同。

不過穆里尼奧此言一出，大眾的視角都被拉向了梅羅之爭。作為執教過C羅的主教練，穆里尼奧的選擇其實在球迷心中頗具重量。早在2012年執教皇馬時，穆里尼奧為支持弟子就曾爆出過「要是C羅今年不能獲得金球獎，那就是明目張胆的犯罪」這樣的金句。後來「犯罪」一詞就頻繁出現在梅羅爭議的場合。

然而在與皇馬分道揚鑣以後，穆里尼奧卻成為了忠實的「梅吹」，他在後來又爆出了另外一句金句「世界上只有兩種球隊，一種是擁有梅西的球隊，另一種則是沒有梅西的球隊。」值得一提的是，在今年的世界足球先生的評選中，穆里尼奧再次將第一名的5分票投給了梅西。
