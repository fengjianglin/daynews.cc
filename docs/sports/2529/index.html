瘋狂的7-5！亞洲黑馬時隔16年再進決賽，伊拉克出局球員痛哭不已

2019-12-06 0條評論 21次閱讀 0人點贊

摘要

北京時間12月6日凌晨，2019年海灣杯半決賽，世界排名第100位的巴林通過加時賽與點球大戰的苦戰，常規時間2-2後，點球5-3淘汰了大熱門伊拉克，時隔16年再次進入決賽，有望成為海灣之王！小組賽表現完美的伊拉克功虧一簣，射丟點球的球員情緒失控當場痛哭。

北京時間12月6日凌晨，2019年海灣杯半決賽，世界排名第100位的巴林通過加時賽與點球大戰的苦戰，常規時間2-2後，點球5-3淘汰了大熱門伊拉克，時隔16年再次進入決賽，有望成為海灣之王！小組賽表現完美的伊拉克功虧一簣，射丟點球的球員情緒失控當場痛哭。

本次海灣杯依然是8隊參賽，A組的伊拉克表現出色，送給了亞洲冠軍卡達今年面對亞洲球隊的首場敗績，隨即又擊敗了阿聯酋，再戰平葉門後小組第一輕鬆出線。相比較之下，巴林出線之路則堪稱驚險，首輪戰平阿曼，次輪不敵沙特陷入絕境，最後一輪4-2擊敗科威特，以一個凈勝球的優勢奇蹟般力壓阿曼小組第二出線。

此番交手，伊拉克自然是更被看好的一方。值得一提的是，世預賽2隊也分在一組，當時雙方1-1戰平。到目前為止，伊拉克在世預賽拿到11分，巴林拿到9分，都成功壓制住了本小組最強大的對手伊朗隊，可以說是2019年亞洲足壇的亮點。

這場比賽，雙方一開始就大打攻勢足球，穆罕默德-阿里第6分鐘就幫助伊拉克先拔頭籌，第14分鐘巴林依靠哈扎扳平比分，但拜耶什在第18分鐘幫助伊拉克再度反超！半場結束前，穆罕默德-馬洪的進球幫助巴林第二次扳平比分！上半場的對攻戰過後，下半場雙方都顯得有些保守，比分沒有改寫，比賽進入加時賽。

加時賽的比賽中，雙方依然不相上下，沒有哪支球隊能夠壓制住對手，結果也都沒有再度進球，比賽進入到了點球大戰！點球大戰，巴林球員大發神威，5罰全中，而伊拉克球員2人罰丟，這樣一來，巴林5-3拿下點球大戰，總比分7-5淘汰伊拉克進入決賽！

海灣杯是西亞球隊非常重視的比賽，因此無緣決賽後，伊拉克有球員痛哭不已！作為3屆冠軍得主，伊拉克大熱倒灶。世界排名僅僅第100位的巴林，成為了大黑馬，他們時隔16年再次進入到了最終的決賽，而過去4次進入決賽，巴林全部屈居亞軍。這一次，球隊渴望創造新的歷史！而且這場勝利，對於巴林在世預賽的自信心來說，也是一次巨大的提升。
