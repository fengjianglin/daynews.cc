總決賽女單四強出爐，陳夢決戰伊藤美誠，20歲新星對陣黑馬

2019-12-13 807條評論 2,295次閱讀 256人點贊

北京時間12月13日，2019年國際乒聯總決賽繼續進行，女單四強產生。上半區，王藝迪爆冷4比1擊敗孫穎莎，王曼昱4比1擊敗陳幸同，兩人爭奪決賽席位。下半區，陳夢4比3險勝劉詩雯，伊藤美誠4比0橫掃佐藤瞳，陳夢和伊藤美誠對決。

本次總決賽，國乒女單有10人參加，12日的比賽只有丁寧輸了外戰，她不敵佐藤瞳，無緣八強。13日的比賽，國乒女單有6人出戰，三場都是內戰。陳夢和劉詩雯的對決十分精彩，前兩局陳夢兩個11比7拿下，從第三局開始，劉詩雯終於找回了節奏，11比9、11比6和11比9，劉詩雯連續贏下了三局，總比分3比2反超。陳夢在第六局11比5勝出，兩人進入決勝局，陳夢11比1拿下，總比分4比3進入四強。

陳夢半決賽的對手是伊藤美誠，後者4比0橫掃佐藤瞳，四局比分是11-7，11-7，13-11，11-5。陳夢和伊藤美誠的對決會成為半決賽的焦點，陳夢直言要拋開上一次公開賽的勝利，立足去拼對手。

上半區的比賽，奪冠熱門孫穎莎1比4爆冷出局，她輸給了王藝迪。王藝迪在前三局打得十分積極主動，很快取得3比0的領先，孫穎莎追回1局，但第五局還是輸球，1比4無緣四強，王藝迪進入半決賽。王曼昱對陣陳幸同，第一局比賽，王曼昱11比6勝出，第二局陳幸同11比7拿下，兩人打成1比1平。第三局，王曼昱還了一個11比7，並且11比5和11比4再次拿下兩局，總比分4比1進入四強，王曼昱半決賽對陣王藝迪。
