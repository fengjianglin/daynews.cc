最勢均力敵的德比！巴薩勝在“二當家”

2019-12-17 2,584條評論 3,770次閱讀 1062人點贊

揚子晚報紫牛新聞訊（記者 湯敏）19日凌晨3點，火星撞地球的西班牙德比將在諾坎普上演。這可能是近些年來最勢均力敵的國家德比，因為兩隊9年來第一次以同分狀態相遇。怎麼樣，夠不夠燃？

彷彿是為了「公平較量」似得，皇馬巴薩本輪竟然一起收穫平局：巴薩客場與皇家社會2：2戰平，皇馬與瓦倫西亞1:1戰平。本場比賽過後，兩隊都積35分，其中巴薩是16戰11勝2平3負，而皇馬是16戰10勝5平1負。

這也是兩隊自2009-10賽季以來，第一次以平分狀態進入國際德比。在那個賽季之後，兩隊參加國家德比時，總有一隊的積分超過另一隊。

巧合的是，兩隊不但積分相同，隊內的頭號射手進球數也一樣。巴薩的頭號射手梅西和皇馬頭號射手本澤馬都是打入12球，兩人並列榜首。當然，論起火力肯定還是巴薩更勝一籌，因為梅西身邊還有蘇亞雷斯和格列茲曼：烏拉圭人打入9球，在射手榜排第3；法國人打入6球，排西甲射手榜第10。而皇馬的除卻本澤馬外進球最多的竟然是後衛拉莫斯，皇馬隊長打入3球，排在西甲射手榜第50位……
