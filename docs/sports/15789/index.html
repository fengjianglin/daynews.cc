英超第17輪（1）切爾西爆冷輸球，利物浦飛赴多哈

2019-12-15 65條評論 1,089次閱讀 104人點贊

2019年12月14日，英超第17輪首先進行六場比賽。利物浦連場勝利，萊斯特城平局放緩追趕腳步；切爾西隊成為唯一主場輸球球隊。

利物浦 2 – 0 沃特福德

榜首榜尾之戰，沒有懸念。主隊統治了比賽，但直達第38分鐘才有了薩拉赫（Mohamed Salah）的進球；利物浦踢出快速反擊，馬內（Sadio Mané）在後場第一時間傳到前場左路飛速插上的薩拉赫，薩拉赫調整後將球挑過守門員曼聯舊將、前英格蘭國門本福斯特（Ben Foster）的頭頂後進球。馬內下半場開場初的進球被VAR判定越位在先，第89分鐘他在右路底線倒三角傳中，奧里吉（Divock Origi）射門踢呲但球卻陰差陽錯傳到已經快速跑到小禁區的薩拉赫，薩拉赫背著球輕輕一蹭破門。沃特福德客場三連敗。

比賽後，主隊的主力將開赴中東參加在多哈進行的俱樂部世界盃迎戰墨西哥蒙特雷隊；時間上和俱樂部世界盃首場比賽相差不到24小時的聯賽杯比賽，將由留守在英國的其他利物浦隊隊員完成。

伯恩利 1 – 0 紐卡斯爾聯

伯恩利結束聯賽三連敗。客隊卡羅爾（Andy Carroll）有機會率先破門，不過他接到中場海登（Isaac Hayden）傳球頭槌頂偏。兩塊「木頭」（Wood）定局比賽，第58分鐘主隊韋斯特伍德（Ashley Westwood）發出的角球由中路的紐西蘭前鋒克里斯伍德（Chris Wood）頂入；喜鵲軍團結束三場不敗。

切爾西 0 – 1 伯恩茅斯

藍軍在主場佔據優勢，得勢不得分；客隊英格蘭U21國門拉姆斯代爾（Aaron Ramsdale）力保球門不失，也是本場最佳球員。全場結束前6分鐘，伯恩茅斯的一次角球進攻，哥倫比亞中場樂瑪（Jefferson Lerma）將球頂到禁區，戈斯林（Dan Gosling）背身挑過主隊西班牙門將科帕（Kepa Arrizabalaga）破門，VAR判定不存在越位犯規進球有效。櫻桃軍團結束5連敗；而車仔近四場聯賽1平3負，主帥蘭帕德（Frank Lampard）勢必迎來一輪口誅筆伐。

萊斯特城 1 – 1 諾維奇

主隊終止九連勝。第26分鐘，金絲雀軍團隊內最會傳球的阿根廷中場布恩迪亞（Emiliano Buendía）助攻給歐洲杯預選賽後滿血復活的芬蘭前鋒普基（Teemu Pukki）為客隊破門得分，布恩迪亞收穫個人賽季第5個助攻；普基則是第9個聯賽進球，占諾維奇隊18個進球的一半。半場結束前10分鐘，萊斯特城隊的角球進攻瓦爾迪（Jamie Vardy）射門造成荷蘭門將克魯爾（Tim Krul）的烏龍球，「小丑」賽季收穫的第2個烏龍球；而克魯爾也成為本賽季第三位烏龍球門將，包括阿斯頓維拉的西頓（Tom Heaton）和切爾西的科帕。

萊斯特城隊表現其實不錯，瓦爾迪雖然當選本場最佳球員，不過也錯失個人連續第9場聯賽進球的機會；而諾維奇隊兩連敗後迎來勝利。

謝菲爾德聯 2 – 0 阿斯頓維拉

本賽季英超升班馬的對決；謝菲聯兩連勝，維拉三連敗。下半場弗雷克（John Fleck）的兩個進球讓阿斯頓維拉隊連續五場客場不勝；艾根（John Egan）禁區內的手球讓客隊有縮小比賽分差距的機會，不過格拉利什（Jack Grealish）主罰的點球卻擊中門楣彈出。謝菲聯隊的聯賽臨時排名來到第5位。

南安普頓 1 – 0 西漢姆聯

鐵鎚幫反客為主控制了上半場，第37分鐘弗納爾斯（Pablo Fornals）連續兩場比賽奉獻助攻，法國前鋒哈勒（Sébastien Haller）命中本賽季個人第5個聯賽進球。下半場安東尼奧（Michail Antonio）的進球被VAR認定手球在先。西漢姆隊兩連敗後迎來勝利，而聖徒在兩連勝後迎來聯賽兩連敗。
