穆里尼奧遭摸頭殺+飛鏟！賽後吐槽：早告訴球員們曼聯怎麼進攻了

2019-12-05 0條評論 21次閱讀 0人點贊

摘要

1-2！穆里尼奧執教熱刺首敗，面對老東家曼聯的時候，熱刺後防線門戶大開。賽後，穆里尼奧也是進行了吐槽，表示早已布置好對拉什福德的針對性防守，可就是做不好。賽後，索帥還給穆里尼奧來了一記“摸頭殺”。本場比賽，拉什福德梅開二度，讓熱刺吃盡了苦頭。

1-2！穆里尼奧執教熱刺首敗，面對老東家曼聯的時候，熱刺後防線門戶大開。賽後，穆里尼奧也是進行了吐槽，表示早已布置好對拉什福德的針對性防守，可就是做不好。賽後，索帥還給穆里尼奧來了一記「摸頭殺」。

本場比賽，拉什福德梅開二度，讓熱刺吃盡了苦頭。穆里尼奧卻表示：「拉什福德從曼聯左路進攻時，他就很危險，我知道這一點，並向球員們提供了這一切信息。他的第一個進球，是典型的拉什福德方式，我們的防守球員很清楚，但我們今天犯了錯。曼聯和我們一樣，都想進入英超前六。」

曼聯球迷此役還是給穆里尼奧足夠的尊重，穆里尼奧也感受到了老東家球迷的熱情，為此他也表示：「這很溫馨、很有禮貌、有教養，這正是我之前希望看到的。」

毫無疑問，對於穆里尼奧來說，他的難題來了，熱刺這防守能力，著實讓他苦惱不已。連續4場比賽，熱刺每場至少丟2球，此番對陣曼聯，曼聯總共拿到了12腳射門機會，7腳射正幾乎都是絕對機會！也就是說，如果不是熱刺運氣不錯，或許這場比賽熱刺要被打成篩子。

整場比賽，熱刺右路成為了拉什福德肆意來回的走廊。輸球並不可怕，可怕的是穆里尼奧輸得毫無辦法，現有的人員配置，穆里尼奧如何讓球隊改變防守不利的問題呢？

本賽季，曼聯遇到Big6時還未嘗敗績。或許從這個角度來看，穆里尼奧輸得又並不冤。顯然，穆里尼奧也是太信任熱刺的攻擊力，準備和曼聯對轟一番，結果到頭來發現，關鍵時候熱刺的進攻組合也不是那麼靠譜。整體來看，穆里尼奧與熱刺球迷被打清醒了！賽後，索帥給穆里尼奧來了一記「摸頭殺」，穆里尼奧只能「接受」。對了，比賽中，穆里尼奧還一度遭遇前弟子的「飛撞」……
