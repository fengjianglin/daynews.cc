15個月來首次！熱刺球星英超連場破門，穆帥上任後聯賽進球第一

2019-12-16 561條評論 2,749次閱讀 1266人點贊

北京時間12月15日晚10點，英超聯賽第17輪的一場焦點戰在熱刺與狼隊之間展開，穆里尼奧首次帶隊造訪莫利紐球場，挑戰昔日弟子努諾，開場不到8分鐘，客場作戰的熱刺便由小盧卡斯破門取得夢幻開局，27歲的巴西邊鋒在穆帥上任後5場聯賽打入3球，與凱恩並列第一。而本賽季此前10次聯賽出場，小盧卡斯僅打入1球，又一名球員被穆帥徹底激發。

上半場第8分鐘，小盧卡斯右路得球上演了個人表演秀，在突入禁區過掉狼隊中場洪尼後，在禁區右側面對後衛賽斯小角度爆射破門。

在穆帥上任後的5場英超聯賽里，小盧卡斯已打入3球，超越孫興慜，與哈里-凱恩並列第一。而本賽季此前在波切蒂諾治下，小盧卡斯10次聯賽出場，僅僅打入1球。又一名熱刺球員在穆帥上任後爆發，這也是27歲的巴西人自2018年8月以來首次在聯賽中連場破門。

不過在球隊取得領先優勢後，坐在客隊教練席的穆里尼奧相當淡定，似乎勝券在握。
