廣東喜提11連勝，1數據證明易建聯手下留情，不想李春江太難堪

2019-12-11 2,308條評論 4,051次閱讀 675人點贊

文/匯哥

CBA常規賽第15輪全面開打，衛冕冠軍廣東男籃在主場迎來浙江廣廈的挑戰，經過四節鏖戰，廣東以112-97輕鬆地又拿到一場勝利。廣東15分大勝廣廈，2數據讓杜鋒冒火，贏球全靠1數據大佔優勢。

此役，廣廈因為大外援法里德受傷被裁，所以是單外援作戰。不過，由於小外援艾倫-傑克遜表現太差，多次空位都不敢投籃出手，所以主帥李春江乾脆讓傑克遜休息，大部分時間廣廈都是全華班在作戰。而雖然沒有下外援，但卻輪休了老將任駿飛，並且大部分時間都是讓幾位小將在場上打，所以場面還是打得比較緊張的，雙方開局打了將近2分鐘都沒能得分，在防守端都投入了很多的精力。當打開局面後，廣東還是很快就展現出了冠軍強隊的底蘊和實力，開始拉開比分。

不過，廣東本場的罰球和三分球命中率實在是太糟糕了。罰球29罰12中，命中率剛剛41%；三分球26投8中，命中率不到31%，兩個數據都非常辣眼睛，尤其是對比廣廈17罰14中的罰球命中率，廣東全隊估計賽後都會被杜鋒留著加練罰球。

要知道，廣東的罰球一向都是相當準的，CBA官方紀錄片中曾透露過，廣東在平常訓練中都有專門的罰球特訓。尤其是向來以罰球精準著稱的易建聯，本場竟然10罰僅4中，顯然今晚廣東全隊的手感都不太正常。即使這樣，廣東依然贏球了，就在於全隊助攻數對比以29比12佔據著絕對優勢，顯然今晚的廣東打得很團隊，而廣廈則都是在單打獨鬥，所以廣廈輸球也就顯得很正常了。

這場比賽過後，廣東喜提11連勝，1數據證明易建聯手下留情，不想李春江太難堪。易建聯的個人數據並不算差，14分13籃板5助攻3搶斷1蓋帽，但是他在全隊最高的36分鐘出場時間裡，竟然只有7次出手（7投5中），這顯然很不正常。

看過比賽就能發現，他大部分精力都投入在防守端，進攻端的機會都盡量讓給了小將們。這既是成全隊友，也是不想過早地打花比賽，讓昔日恩師李春江臉上太難堪了，果然是真情真性的真漢子啊！現在聯賽最大的懸念來了，本賽季誰會站出來終結廣東的這一波連勝呢？
