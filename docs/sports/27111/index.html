曝巴薩瞄準4500萬殺手 梅西隊友不來搶他接班蘇牙 或成冬窗首簽？

2019-12-22 182條評論 1,329次閱讀 122人點贊

日前，據西班牙《足球先生》報道，巴薩在冬窗準備對里爾20歲的奈及利亞前鋒奧西姆亨出手，他們計劃花費4500萬歐元將他帶到諾坎普，作為蘇亞雷斯的輪換角色。

對巴薩而言，如今最重要的是尋找蘇亞雷斯的替身，雖然烏拉圭人本賽季至今16場西甲打入9球並有3次助攻，仍然是巴薩鋒線除梅西外最強的攻擊點，但他的體能情況和巔峰時期還是有較大的差距，一個能夠輪換他的前鋒必不可少，巴薩過去兩個賽季都是在某場關鍵比賽中馬失前蹄，球隊高層和巴爾韋德都希望在加強陣容厚度，如今巴薩各個位置的人選基本充足，唯獨前鋒位置上，蘇亞雷斯缺少一個合適的輪換者。

此前巴薩曾考慮國米的阿根廷前鋒勞塔羅，但如今的形勢來看，這位梅西的國家隊隊友很難再冬窗到來，巴薩不得不考慮其他的人選，而他們如今正在考慮里爾的奈及利亞人奧西姆亨。奧西姆亨身高1.85米，他出生於1998年，本賽季在法甲他17次出場打入9球並且助攻4次，為里爾殺入法甲前三立下汗馬功勞。在歐冠中里爾雖然並未晉級下一階段歐冠淘汰賽，但奧西姆亨面對切爾西和瓦倫西亞這樣的強敵都取得了進球。

雖然在12月29日奧西姆亨才年滿21歲，但他已經在德國沃爾夫斯堡、比利時沙勒魯瓦等球隊效力過，今年裡爾才花費1200萬歐元從沙勒魯瓦將他簽下，但他根本無需適應期，就融入了里爾的戰術體系。從技術特點來說，奧西姆亨最擅長的是搶點，射門精準，也有一定的前場牽制能力，而且還能夠踢邊鋒位置。蘇亞雷斯這種全面的前鋒的確難找，但巴薩直是要求奧西姆亨能夠在一些不那麼關鍵的比賽中輪換蘇亞雷斯，奧西姆亨還是勝任的。

如今對奧西姆亨有興趣的球隊病不少，英超三大豪門托特納姆、阿森納和利物浦都對他有興趣，而一貫對年輕天才趨之若鶩的多特蒙德也在關注這位奈及利亞前鋒，不過巴薩有信心擊敗這些對手，他們準備拿出4500萬歐元來競爭奧西姆亨，對於里爾而言，這個價格還是很有說服力的。
