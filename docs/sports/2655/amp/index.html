本文文字作者 《中國體育》Zhibo.tv 馬行飛

里皮的突然辭職意味著國足選帥的話題又一次登上熱搜。無論專業人士還是鐵杆粉絲，都希望新來的主教練一定要適合當下的國足。

米盧蒂諾維奇是國足歷史上最成功的主帥

這一刻，會有多少球迷想念中國足球的功勛主帥米盧蒂諾維奇。自2002年一別，米盧一直關注著中國足球的發展，和中國足球人的交流始終沒有間斷。他有一個優點是其他教練不具備的，即唯一帶領六支不同國家隊征戰世界盃正賽的教練。今天的米盧已經75歲了，精力和體力已經不適合再拿起國家隊的教鞭。

奎羅斯在伊朗執教了兩屆世界盃周期

亞洲本來有一名出色的主帥適合接手國足，葡萄牙人奎羅茲。他曾經率領葡萄牙隊參加了2010南非世界盃，接著帶領「波斯鐵騎」伊朗隊連續征戰了2014巴西世界盃和2018俄羅斯世界盃決賽圈。可惜在阿聯酋亞洲杯結束後，奎羅茲就去哥倫比亞國家隊上任了。所以，筆者只能從熟悉中國足球或正在執教中國足球的外籍教練中尋覓。

表1：斯托伊科維奇和李章洙領跑

如果從《暫無國家隊執教經驗》的教練中選，筆者推薦廣州富力隊的斯托伊科維奇和韓國鐵帥李章洙。李章洙在中國執教多年，已經成為了名副其實的中國通。對中國隊現役主力都很了解，執教風格紀律嚴明，擅長體能訓練，短期內能夠提升球隊的戰鬥力，但帶隊打硬仗是短板。

斯托伊科維奇適合長期打造一支球隊

斯托伊科維奇正值年富力強，師從溫格，球員生涯綽號「巴爾幹馬拉多納」，塞爾維亞足協已經多次邀請其回國出任國家隊主帥。他的工作非常穩定。自2008年拿起教鞭，斯帥只經歷過兩傢俱樂部，其中在日本名古屋鯨八隊一教就是6年，俱樂部上下對他的執教能力相當認可。如果想選擇一位性格溫和又充滿魅力的主帥長期紮根中國足球，不過分追求短期效益，斯托伊科維奇是最適合的。

表2：福薩蒂、濟科、斯科拉里

如果從擁有國家隊執教經驗的教練里選，球迷們可能傾向巴西老帥斯科拉里。一方面，他擁有三屆世界盃正賽的執教經驗，還帶領廣州恆大奪得中超冠軍和亞冠冠軍。另一方面，中國隊的歸化球員主要來自巴西，語言的優勢會幫助歸化球員和本土球員更好的磨合。

斯科拉里在德國世界盃周期執教過C羅

但是，71歲的年齡和高額的年薪或許不會讓中國足協拋出橄欖枝。況且，大菲爾過去執教的國家隊都是世界級強隊，沒有執教弱隊的履歷和經驗，貿然拿起中國隊教鞭對他的挑戰遠大於機遇。

濟科在日本隊執教中田英壽

還有一位熱門同樣出自巴西，他就是現任日本鹿島鹿角隊技術總監濟科。他分別於2011年和2016年成為國足主帥的熱門人選，教練里的「亞洲通」。年齡比斯科拉里小5歲，善於交際，微笑更多，年薪物美價廉，濟科這兩個字足以鎮住艾克森、高拉特、費南多等歸劃選手，階段療效快。2004年，「白貝利」帶領日本隊戰勝中國隊問鼎亞洲杯。2006年，他率領日本隊參加了德國世界盃決賽圈比賽。此後輾轉伊拉克，2011年，他率領「沙漠之狐」在巴西世界盃預選賽主客場雙殺中國隊。濟科沒有執教過任何中國球隊，並不意味著他對中國國家隊是陌生的。他就在日本，會晤不需要長途跋涉。為了救火世預賽，保留進軍12強賽的希望，中國足協不妨東渡求賢。
