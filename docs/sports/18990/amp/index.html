單節狂贏31分！杜鋒輸馬布里，拿江蘇隊泄憤 兩大外援瘋狂15中13

天天要聞
2019-12-18

北京時間12月17日，2019-20賽季CBA常規賽繼續進行，廣東客場挑戰江蘇。此役廣東在次節徹底暴走——單節比賽他們轟出了50-19的比分，單節凈勝江蘇31分。在上場被馬布里擊敗，賽後還傳出主帥杜鋒與馬布里衝突的消息之後，本場比賽廣東也算是拿江蘇隊泄憤，半場比賽幾乎就將比賽的懸念終結。

面對江蘇，廣東首節還是有些不在狀態，他們單節被江蘇得到39分，單節33-39落後。此時杜鋒在場邊也是面露不悅，不過廣東很快在次節展現了自己的實力，在客場以風捲殘雲之勢，徹底打瘋了！

次節開始之後，先是易建聯打成2+1，隨後馬尚命中三分，廣東開啟了自己的進攻狂潮。威姆斯直殺內線打成，此時廣東已經將比分反超，並且再沒有給江蘇接近的機會。

布魯克斯搶斷之後的三分，以及威姆斯和布魯克斯之間的連線，讓廣東持續拉開分差，在雙方戰成44平之後，廣東憑藉馬尚和威姆斯的無解表現，直接轟出一波16-0，將雙方的分差拉開到16分。

這還沒算完，隨後布魯克斯繼續命中三分又突破打成，雙方在次節的比分一度來到了讓人瞠目結舌的38-8。在這段狂潮之中，廣東完完全全的展現了自己的實力和特點，不斷逼迫對手的失誤，不斷通過反擊迅速漲分。即便是對手的進攻能夠打成，廣東也能夠依靠快速發球的轉換進攻，一次次的讓對手目送皮球入網。

單節下來，廣東轟下了恐怖的50分，兩大外援的發揮徹底讓江蘇崩潰——馬尚-布魯克斯三分球5投4中，單節8中7轟下了18分，威姆斯單節7中6拿下了13分，雙外援15中13的表現，完全扛起了廣東的進攻，也展現了在上場鬱悶輸球之後，衛冕冠軍在本場的必勝信念，江蘇則有些無辜的成為了廣東的「泄憤對象」。
