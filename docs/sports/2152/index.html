波波維奇：湖人還沒贏過地球之外的球隊，所以他們還算不上強隊

2019-12-09 0條評論 25次閱讀 0人點贊

摘要

北京時間12月8日，馬刺隊主教練格雷格.波波維奇在接受《聖安東尼奧新聞快報》記者TomOrsborn採訪時，談到了之前外界認為湖人是一支偽強隊，排名第一是因為賽程輕鬆，並沒有戰勝過真正強隊的這種觀點。

北京時間12月8日，馬刺隊主教練格雷格.波波維奇在接受《聖安東尼奧新聞快報》記者TomOrsborn採訪時，談到了之前外界認為湖人是一支偽強隊，排名第一是因為賽程輕鬆，並沒有戰勝過真正強隊的這種觀點。波波維奇調侃的表示自己贊同這種觀點，因為湖人還沒有贏過地球之外的球隊，所以湖人還算不上強隊。隨後，波波維奇還大肆的稱讚了湖人主教練弗蘭克.沃格爾的工作，以及勒布朗詹姆斯的領導能力。

在談到前段時間湖人偽強隊的觀點時，波波維奇調侃的說道：「好吧，就我個人來說我贊同這種觀點，你知道的，他們還沒有戰勝過怪物奇兵隊（電影空中大灌籃中的外星球隊），他們現在戰勝的都是地球上的籃球隊，所以，除非他們能夠戰勝怪物奇兵，或者是痛揍一頓地球之外的籃球隊，否則他們還算不上強隊。」而隨後TomOrsborn問波波維奇是在開玩笑還是在認真時，波波維奇反問道：「你覺得呢？難道馬刺是聯盟最強的球隊么？」

隨後，波波維奇大肆稱讚了一番沃格爾和詹姆斯的表現：「就現在來說，弗蘭克的工作做得非常不錯，他把一支剛剛結合在一起的球隊變成了一個整體，為每個人都配置了合適的戰術，讓勒布朗和AD各司其職，我認為他可以獲得本賽季的最佳主帥。而勒布朗則一如既往的出色，他總是能保持自己的競爭力，讓自己的身體處在一個良好的狀態。他目前在球場上所展現出的能量和活力要比大多數年輕人都要強。」而在被問到是否認為湖人能夠奪得今年的總冠軍時，波波維奇回答道：「我不知道，反正不是馬刺。」

不管是東部還是西部，不管是騎士還是熱火還是湖人，只要是詹姆斯在的球隊，波波維奇這個老頭子都是一如既往的支持。在11月份時，湖人的戰績含金量可謂是飽受質疑，許多評論員解說員球迷都在質疑湖人西部第一的成績。而到了12月，先後戰勝掘金，開拓者，爵士等西部強隊後，湖人可謂是狠狠的擊碎了外界的質疑！而這一切波波維奇自然看在眼裡，幽默的老頭子也用自己的方式調侃了一番質疑者。希望湖人能夠保持統治力，撐過12月的魔鬼賽程，也希望馬刺能夠找回狀態，重新向季後賽發起衝擊！
