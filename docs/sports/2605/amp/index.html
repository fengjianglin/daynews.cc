國球19歲少女完美大逆轉！0-2落後絕地反擊3-3，決勝局轟11-1大勝

天天要聞
2019-12-05

摘要

北京時間12月5日，北美乒乓球挑戰賽繼續開戰，在第1個比賽日當中，中國隊的年輕球員就有著相當出色的發揮。中國隊的黃凡真，上演了一場激情四射的大逆轉。

北京時間12月5日，北美乒乓球挑戰賽繼續開戰，在第1個比賽日當中，中國隊的年輕球員就有著相當出色的發揮。中國隊的黃凡真，上演了一場激情四射的大逆轉。

在女單資格賽第1輪的比賽當中，19歲的小將黃凡真出戰的是和她同齡的羅馬尼亞選手安德雷亞，結果這場比賽黃凡真開局打得非常吃力，第1局在關鍵分的爭奪上，她以13比15輸給對手，隨後又以8比11輸掉了第2局，這樣兩局比賽過後，黃凡真已經是0比2早早的落後了。

從第3局開始，黃凡真努力追趕對手，連續打出兩局的11比8，將大比分扳為2比2平，但是第5局的比賽她又以9比11輸給對手，這樣一來大比賽就已經是2比3，第2次落後形勢比較危險了。不過在第6局當中她的表現相當爭氣，以11比5大勝對手，這樣一來六局比賽過後已經和對手打成了3比3平，要通過最後的搶7大戰決出勝負了。

結果在決勝局當中，黃凡真發揮得非常出色，單局轟出11比1，這樣的大比分完勝對手。這樣一來，黃凡真也就以4比3的比分成功的絕殺對手，打進到了資格賽的第2輪當中。出生於2000年的黃凡真是一位相當年輕的天才，在2016年的全國青年乒乓球錦標賽當中，她曾經是拿到了女單第4名，目前也是中國隊重點培養的對象，在2015年的中國青少年乒乓球公開賽太倉站巡迴賽的比賽當中，當時黃凡真曾經和同齡的孫穎莎打得難分難解，最終是在比賽當中輸給了孫穎莎拿到了亞軍。如今孫穎莎已經成長為中國新一代的領軍人物了，黃凡真也在努力的爭取能夠儘早的殺進中國一隊。

在去年黃凡真有不錯的收穫，她在泰國公開賽當中曾經是拿到了青年組的女單冠軍，隨後在2018年底，她又和石洵瑤聯手在世青賽當中拿下了女雙的冠軍，一舉加冕為年輕的世界冠軍。黃凡真目前還在鄭州市商貿管理學校就讀，師從於張震教練，張震教練對她的評價是為人非常刻苦，踏實好學，比賽和學習兩不誤，是非常值得重點培養的好苗子，國家隊未來的希望之星。
