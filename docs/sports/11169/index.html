奧沙利文怒射致勝黑球發泄不滿 趙心童周躍龍雙雙出局

2019-12-11 2,111條評論 4,932次閱讀 2171人點贊

北京時間12月10日晚間，2019斯諾克蘇格蘭公開賽在格拉斯拉結束另外一場首輪比賽爭奪，奪冠大熱門羅尼.奧沙利文在與前上海大師賽冠軍得主多米尼克.戴爾的交鋒中一度如同夢遊，最終連救兩賽點連扳三局逆轉晉級。另外，周躍龍、趙心童出局。

中國軍團在本屆蘇格蘭公開賽上一共有25人報名參賽，在首個比賽日，中國軍團先後有7人亮相，結果梁文博、呂昊天等3人倒下。

截至10日晚間，隨著中國一哥丁俊暉、顏丙濤、肖國棟等人紛紛晉級，使得中國軍團首輪過關人數增加至10人。

奧沙利文在兩個月前的玉山世界公開賽資格賽輪代替棄賽的衛冕冠軍金左手首次參戰，結果屁股沒有坐熱，就被狀態出色的前上海大師賽冠軍「太空人」多米尼克.戴爾打出4桿50+，以總比分5－3強行將其送回家。

此役再度狹路相逢，雙方上演一場快節奏的對攻大戰。

開局，奧沙利文在明顯手冷，在攻防兩端失誤不斷，被戴爾通過零敲碎打以60:0丟掉首局。

第2局，奧沙利文面對相對開放球形，依然無法組織起有效進攻，只得到43分就交出局面，被戴爾反打單桿50+反超局分。

關鍵的爭彩大戰中，奧沙利文在安全球比拼中破綻百出再丟一局0－2。

第3局，奧沙利文迅速調整手感，反打單桿80分將比分改寫成1－2。

第4局，奧沙利文搶先上手，然而只得到40+就交出控球權，太空人在最後兩顆紅球上大做文章。

防守大開大合的奧沙利文因此頻繁出錯，被戴爾通過多次上手以46:44反超局分。

隨後的爭彩大戰中，奧沙利文推進高難度遠台咖啡球獲得上手，之後連打帶K成功解放邊庫黑球，面對難度不大的中袋袋口致勝黑球，火箭竟然做出驚人之舉，發力進攻目標球造成白球跳出桌面被罰分，從而拱手相送對手達到賽點3－1！

好在關鍵的第5局，奧沙利文及時懸崖勒馬，以90：0成功挽救一個賽點2-3。

在隨後的比賽中，奧沙利文逐漸掌握場上主動，又連轟2桿80+以總比分4-3逆轉晉級，下一輪將戰卡希爾與「宋姑娘」之間的勝者。

其它比賽場次，張科長1－4不敵小司機，田鵬飛打出3桿50+以總比4－2淘汰小萊恩斯，梅老闆苦戰魚腩湯姆森於決勝局驚險過關，下一場將和丁俊暉上演德比大戰。

另外，截至發稿，四川名將周躍龍3－4磨王艾伯頓，快槍手趙心童１－4落後阿蘭，徐思2-1二師兄，小天王命懸一線2-3落後瓦菲。(文/邱日東)
