不合財！籃協新規工資封頂，丁彥雨航會損失不少啊！

2019-12-14 1,465條評論 3,123次閱讀 1271人點贊

12月12日，CBA記者張晨曦在個人社交平台發文稱丁彥雨航「很想家」，而這裡的「家」，顯然是指山東男籃。

張晨曦的發文說「昨天在山東訓練中心看到小丁的照片，就忍不住發給他了。他起床後看到就給我發來信息，聽到他聲音，感覺心情狀態還不錯」，按照張晨曦的說法，丁彥雨航渴望回家，「昨天小丁說，很想恢複利索了，趕緊回來，很想家。搞得我心裡也酸酸的。他在我的腦海里總是那麼嘻嘻哈哈的，以至於讓人忘了他也有痛」。

今年以來，小丁簽約睢冉的經紀公司後，一直不與山東男籃聯繫，睢冉也時不時的在網路媒體爆料一些猛料，小丁與山東男籃遲遲不能進一步溝通簽約。後來籃協出了新規，球員新簽約封頂900萬人民幣，也就是說小丁今年回來打球，只能在山東男籃打球，且合同年薪不會超過900萬。以小丁的實力，不能夠立足NBA，回國發展應該是正確的選擇。

今天這個新消息讓我們這些關心丁彥雨航的球迷有了希望，但小丁應該損失不少工資啊！事實證明，他們不合財啊！
