怎樣擊敗廣東？為什麼只有吉林一而再做到

2019-12-03 0條評論 20次閱讀 0人點贊

摘要

78：93，北京主場輸球，儘管他們把廣東得分壓制在100分以內，但是他們的得分更少。整場比賽，廣東似乎是在按部就班的擊敗了對手。他們在第三節被對手反超，落後到7分。這沒有影響到他們的信心，第四節，隨著北京林書豪體力下降，北京沒有其他穩定得分點，他們一舉重新奪回主動，贏下了比賽。

聯賽打了12輪，廣東11勝1負，聯賽第一。衛冕冠軍正在朝著正確的道路大步向前，在面對他們的主要競爭對手時，也是未嘗一敗。揭幕戰克遼寧，後面險勝新疆，擊潰北京。除了廣廈還沒打，他們面對強敵，總能找到辦法贏球，這樣的廣東，可怕嗎？可怕，對於要爭冠的隊伍來說，只有尋找到擊敗廣東的辦法，才有可能站上最高領獎台。

易建聯

那麼，如何擊敗廣東？

相信新疆在想這個問題，遼寧在想這個問題，廣廈、北京也在想這個問題。然而，事情總有意外，這個讓CBA諸強一籌莫展的難題，有一個球隊卻一而再的做到了。吉林繼上個賽季擊敗廣東後，本賽季又一次完成了這個壯舉，堪稱是巨人殺手。那麼，他們為什麼能一而再的連克廣東，他們有沒有戰勝廣東的鑰匙呢？

我們回去復盤本賽季第4輪吉林戰勝廣東的這場比賽，或許從這裡面能得到一些啟示。

當時，兩隊都帶著3連勝的戰績狹路相逢，最終吉林在主場120：115贏下了這場精彩的遭遇戰，讓人大跌眼鏡。這場比賽，廣東的發揮也不是說不好，他們也曾經一度在比賽中領先10分左右，最後卻被翻盤。他們到底輸在了哪裡？

馬尚布魯克斯

有3個人的發揮直接影響了比賽進程，首先是吉林的瓊斯的超常發揮。本場比賽，瓊斯砍下49分10籃板11助攻，可以說是一己之力擊潰了廣東隊的防守。而且瓊斯的進攻是連綿不絕的，4節比賽，都有得分進賬，在他源源不斷的提供得分的前提下，吉林才可以一直與廣東抗衡，最後贏下比賽。第二個影響比賽進程的人是廣東隊的威姆斯，那場比賽堪稱是威姆斯的噩夢，整場比賽發揮如同夢遊，12投僅僅3中，整場出場30.9分鐘，才得了11分，很多次進攻都是因為他，讓全隊的努力無功而返。在隊友馬尚砍下36分，易建聯27分，任駿飛也有20分進賬的情況下，廣東卻無法贏球的原因，就是因為威姆斯的失常。第三個影響比賽進程的人是吉林的代懷博。第四節比賽，兩隊迎來決戰時刻，前三節戰成81平的前提下，誰在第四節發揮更出色就能贏得比賽。而吉林隊在球隊最需要球員分擔瓊斯得分擔子的時候，代懷博站了出來，他一節投進了4個三分球，砍下12分，為球隊奠定了勝局。

瓊斯

比賽過後，當我們再去總結這場比賽，我們發現吉林贏球，看似偶然 ，卻也有其必然的因素在裡面。他們之所以能夠贏下廣東。有幾個因素是必不可少的。

第一，他們擁有擊破廣東防守的尖刀，那就是他們的外援瓊斯，瓊斯超強的得分能力，面對廣東的防守，無論是突破後的中投，快攻得分，三分，助攻、籃板，他總有辦法幫助到球隊。而且，瓊斯良好的身體素質也非常關鍵，在激烈的比賽中，在第四節的時候，他依然能夠為球隊提供火力。這一點對比林書豪就會感受非常強烈，北京和廣東的比賽，第三節北京一度領先了7分，這和書豪的出色發揮分不開。然而，到了第四節，受困於體能影響，林書豪沒有得分，他這個強點的啞火，直接導致了北京的輸球。

第二、球隊能夠跟得上廣東的快節奏。廣東隊之所以強大，是因為他們經常能夠打出快速的防守反擊，擊潰對手。然而，我們看吉林和廣東的比賽，發現他們防返的速度一點不比廣東慢，作為同樣沒有強力內線的他們，可以和廣東打對攻。除了擁有瓊斯，隊內的國內球員，特別是年輕人也非常的給力，代懷博、姜宇星、崔晉銘都是敢打敢拼，一點都不畏懼。年輕有活力，讓他們整場都可以跟得上廣東的節奏，沒有斷檔。

第三、關鍵時刻有球員站出來得分。幾乎沒有球員能早早打花廣東，那麼就要做好第四節比賽決戰時刻。這個時候，兩個球隊都會加大防守。如果單純的只依靠外援得分，球隊就會非常的別動，這個時候就要有國內球員站出來得分。而吉林剛好有了代懷博的發揮，狂砍12分。而北京的四節，全隊只得了14分。

代懷博

這樣，我們有了如下總結：要能贏廣東，每個隊只能根據自身球隊的特點去發揮出自己的長處，並且抑制對手的發揮。從吉林贏下廣東的這場比賽，我們有幾個啟發。

第一、要有擊破廣東防守的利器。無論是利用超級外援的得分能力。（吉林、山西等隊）、或是比廣東更強的內線鋒線得分能力（遼寧、新疆），你必須要有得分，才能一直與廣東對抗。

第二、要有更多的得分點。僅僅靠外援擊敗廣東的難度很大，要有國內球員分擔責任，吉林的代懷博、新疆的可蘭白克都曾經很好的做到過這一點。

第三、利用廣東的失常。很顯然，如果和吉林的這場比賽，威姆斯的狀態沒那麼差的話，廣東可能就不會輸球了。而隨著廣東目前的漸入佳境，要指望廣東犯錯的概率越來越小。

廣東隊憑藉著出色的發揮，在衛冕的道路上奮力向前。而諸強變著法子去擊敗這個強大的對手，這才讓我們的聯賽越來越精彩。這才是我們渴望看到的聯賽。
