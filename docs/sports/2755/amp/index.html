北京時間12月4日訊，國際乒聯挑戰賽系列最後一站北美公開賽將於今日正式打響，國乒仍以練兵為主，並沒有派出主力隊員征戰，然而對奧運單打資格競爭激烈的日乒卻是重兵出擊，目前平野美宇和石川佳純積分相差僅有65分，丹羽孝希由於剛剛參加完男乒世界盃，因此超出水谷隼495分佔據一定優勢，然而遺憾的是，丹羽孝希卻於近日宣布，因傷無奈退出本次北美公開賽。

丹羽孝希今年25歲，曾參加過兩屆奧運會，是一位不可多得的天才型的球員，他每每在比賽當中都能奉獻出神來之筆，引得台上掌聲四起，不過更多時候這位日乒大將卻顯得十分「佛系」，看上去極為淡泊名利，即便本賽季戰績與奧運單打資格直接掛鉤，他還是不以為意，並先後6次公開賽一輪游，直到看上去有些迫在眉睫了，才開始認真對待，自進入11月後之後，丹羽孝希戰績突增，從一輪游上升至二輪游，再到奧地利公開賽的止步8強，男乒世界盃也同樣晉級8強，如此下來，他在12月份的世界排名當中，已經超越競爭對手水谷隼足有495分，可以說是佔據了相當不錯的優勢。

再來談談水谷隼，這位日乒老一哥在這一年經歷甚多，由於眼部問題比賽成績所有下滑，即便丹羽孝希6次一輪游，水谷隼也未能迎頭超越，此外場外花邊新聞也同樣不間斷，對此，不少球迷認為水谷隼這是等同於放棄了奧運單打資格，然而他卻公開表示，「如果缺席這次奧運單打，也就是自己該退役的時候到了」，不過話說得雖恨，但戰績卻是每況愈下，由於沒有獲得世界盃單打資格，與丹羽孝希也是越拉越大，然而慶幸的是，水谷隼獲得了國際乒聯總決賽的參賽資格，為了積極備戰，水谷隼率先退出了本次北美公開賽，如果總決賽表現有所突破，他便可以一舉超越丹羽孝希。

水谷隼退賽事出有因，丹羽孝希也緊隨其後宣布退出北美公開賽，理由則是因為連續作戰導致頸部、手肘出現了不同程度的傷病，不過丹羽孝希依舊是我們熟悉的那個佛系少年，他表示：「雖然不能參加本次比賽，但也以令自己非常滿足的形式結束了本賽季，之後會靜靜等待水谷隼前輩征戰國際乒聯總決賽的結果，不論能否成為單打球員，都會欣然接受這個結果。」

不過，雖然沒能獲得國際乒聯總決賽出戰資格，但丹羽孝希也同樣尚存一絲參賽的希望，目前16仍大名單當中，由於王楚欽被國乒禁賽，因此替補的黃鎮廷獲得了出賽資格，不過此前t2聯賽首站與許昕的比賽當中意外受傷，黃鎮廷是否能夠順利參加比賽還是一件待定的事情，當然，國際乒聯總決賽就像是一次年度盛典，沒有人會輕易放棄，最後，都祝願丹羽孝希能夠收穫一個自己滿意的結局。
