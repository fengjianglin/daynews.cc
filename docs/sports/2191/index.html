2020年CBA全明星周末將於2020年1月11日-12日在廣州體育館舉行，投票在12月6日已經開啟了，12月27日18:00公布結果。CBA本土這九名球員，鐵定入選全明星，孫悅呼聲僅次阿聯。

阿聯本賽季場均得到22.4分10.1籃板，投籃命中率高達53.7%，昨晚帶領廣東隊擊敗青島隊，豪取10連勝。阿聯現在的狀態比上個賽季還要好，他是國內最自律的球員，能夠將客場變成主場。這一次，阿聯應該是票王，不過外援林書豪也有可能是票王。

現在孫悅入選全明星的呼聲僅次於阿聯，孫悅上次打球還是在16-17賽季，休賽期加盟了北控，目前場均得到11.7分2.9籃板3.4助攻1.4搶斷1蓋帽，非常全面，孫悅依舊是國內最出色的後衛之一，相信他的票數會很高。

周琦目前在新疆隊場均得到23分12籃板2.助攻2.5蓋帽，是國內最出色的球員之一，這一點是要承認的，上一場打CBA是在16-17賽季，當時場均只有16分，周琦是有進步的，我相信他經過籃球世界盃的失敗，能夠正確認識自己。

王哲林最近處在風口浪尖，因為他現在沒有福建隊的股份了，有傳言說王哲林要離隊了，我覺得可能性不大。雖然目前福建隊打得不好，但大王場均可以拿下25.2分10.9籃板，得分排在國內球員第一位，鐵定入選全明星。

郭艾倫本賽季不穩定，場均得到19分3.5籃板5助攻，得分還是本土控衛最高的，郭艾倫長得很帥，女球迷很多，他的票數會很高。趙繼偉應該也可以入選，賽季初打得不好，現在狀態恢復過來了。

翟曉川現在是國內第一小前鋒，也是北京隊隊長，本賽季場均得到16分6籃板2.6助攻，曉川的人氣也很高。阿不都沙拉木新賽季場均得到17.5分5.5籃板2.3助攻，效率很高，投籃命中率高達54%，長得也帥。趙睿雖然質疑聲也很多，但能打硬仗，在廣東隊打得很好，也可以入選全明星。

阿聯、孫悅、周琦、王哲林、郭艾倫、趙繼偉、翟曉川、阿不都沙拉木、趙睿，CBA這九位本土球星，鐵定入選全明星。
