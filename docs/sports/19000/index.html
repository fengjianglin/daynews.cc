亞洲球員身價前20榜單：日本8人，韓國4人，中國2人，武磊較尷尬

2019-12-18 83條評論 1,553次閱讀 340人點贊

智道12月17日訊 來自德國《轉會市場》最新亞洲球員的身價數據，近期在歐冠賽場發揮出色的薩爾茨堡紅牛的日韓雙鋒南野拓實和黃喜燦的身價都從原來的500萬歐元升至1250萬歐元，兩人的身價也躍居亞洲層面的第五、六位。在德轉市場的這份數據中，亞洲身價前20的球員中，日本球員佔據了8席，韓國球員有4人，中國球員也有2人，但武磊的身價比較尷尬。

亞洲球員身價TOP20（單位：歐元）

1.孫興慜（韓國，27歲），8000萬，熱刺

2.中島翔哉（日本，25歲），2500萬，波爾圖

3.李剛仁（韓國，18歲），2000萬，瓦倫西亞

4.阿茲蒙（伊朗，24歲），1700萬，澤尼特

5.南野拓實（日本，24歲），1250萬，薩爾茨堡紅牛，明年1月即將加盟利物浦。

6.黃喜燦（韓國，23歲），1250萬，薩爾茨堡紅牛，歐冠小組賽中過掉范迪克進球，狼隊對他虎視眈眈。

7.阿隆-穆伊（澳大利亞，29歲），1200萬，布萊頓

8.馬修-瑞恩（澳大利亞，27歲），1000萬，布萊頓

9.武磊（中國，28歲），1000萬，西班牙人（中資球隊）

10.富安健洋（日本，21歲），1000萬，博洛尼亞

11.堂安律（日本，21歲），1000萬，埃因霍溫

12.久保建英（日本，18歲），1000萬，馬洛卡，是皇馬球員，目前租借效力於馬洛卡。

13.鐮田大地（日本，23歲），900萬，法蘭克福，之前歐聯杯中對阿森納梅開二度，一戰成名。

14.賈漢巴什（伊朗，26歲），800萬，布萊頓

15.酒井宏樹（日本，29歲），800萬，馬賽

16.索馬（敘利亞，30歲），800萬，阿赫利體育（沙特聯賽的勁旅），已經在40強賽中打入7球。

17.武藤嘉紀（日本，27歲），700萬，紐卡斯爾

18.艾克森（中國，30歲），600萬，廣州恆大

19.埃瑟里奇（菲律賓，29歲），600萬，卡迪夫城（英冠球隊的主力門將），在之前40強賽中零封國足。

20.權昶勛（韓國，25歲），550萬，弗賴堡

雖然在亞洲球員身價前20名中含有兩位中國國腳，但武磊的身價比較尷尬。如果把孫興慜8000萬歐元放到轉會市場，肯定很快被搶走；同樣的道理若把1250萬歐元的南野拓實和黃喜燦放到轉會市場，一樣有球隊來求購；但把武磊以1000萬歐元的身價放到轉會市場，除了中資球隊很難會有歐洲球隊來購買。這些話雖然糙但理不糙，武磊的真實身價水分較大，但他依舊是目前中國足球最好的球員，也是最勇敢的中國國腳，敢於放棄金元中超，敢於去追夢去留洋，他是所有中國年輕球員的榜樣。中國足球只有走出去，放進來，才有希望！
