歐冠16強出爐！馬競2-0順利出線，亞特蘭大3-0晉級，熱刺1-3慘敗

2019-12-12 2,915條評論 4,810次閱讀 1781人點贊

北京時間12月12日，歐冠小組賽進行了最後一輪比賽，其中馬競在主場2-0擊敗莫斯科火車頭，最終以小組第二出線，而亞特蘭大則在客場3-0大勝礦工，由於曼城4-1擊敗薩格勒布迪納摩，這支意甲黑馬最終神奇晉級。其他場次的比分為：勒沃庫森0-2尤文、大巴黎5-0加拉塔薩雷、布魯日1-3皇馬、拜仁3-1熱刺、奧林匹亞科斯1-0貝爾格萊德紅星.截至目前，歐冠16強已經全部出爐，五大聯賽神奇包攬了全部席位。

在本輪比賽中，馬競只要擊敗對手，就能夠把握自己的命運，結果這家西甲豪門非常爭氣，在主場與莫斯科火車頭的比賽中，剛開始幾分鐘特里皮爾就浪費一次點球機會，好在第17分鐘對手手球犯規再送點球，這一次菲利克斯主罰命中，為馬競取得領先。下半場馬競再進一球，徹底殺死比賽的懸念，最終他們以2-0擊敗對手，順利以小組第二名出線。在本小組另一場比賽中，已經提前出線的尤文，最終客場2-0擊敗勒沃庫森，此役C羅首發出場並打進一球，他也再度刷新自己在歐冠的進球紀錄。

在另一場關鍵比賽中，亞特蘭大做客對陣礦工，結果他們眾志成城，在上半場頂住了對手的狂攻之後，下半場第66分鐘，卡斯塔涅率先推射破門，為球隊打開了勝利之門。隨後帕薩里奇等人再進兩球，幫助球隊在客場取得一場大勝。由於曼城在客場擊敗薩格勒布迪納摩，亞特蘭大也在前3輪拿到0分的情況下，最終神奇出線。這支進攻非常強悍的球隊，有望在本賽季成為黑馬，如果其他豪門把他們當大禮包，恐怕要付出代價了。

本輪比賽，熱刺做客對陣拜仁，結果他們又遭遇一場慘敗。此役剛開始，拜仁就對熱刺的大門狂轟濫炸，第13分鐘格納布里助攻科曼破門，為拜仁取得領先。儘管塞塞尼翁一度為熱刺扳平了比分，但是隨後穆勒和庫蒂尼奧又連續破門，最終幫助拜仁在主場3-1雙殺熱刺。自從穆里尼奧接替波切蒂諾之後，一直都沒能為球隊解決防守問題，如果接下來情況繼續，以熱刺這樣的防守表現，無論遇到哪支豪門，他們都很難有機會殺進八強！接下來，大巴黎、拜仁、尤文、曼城、巴薩、利物浦、萊比錫、瓦倫西亞、切爾西、里昂、亞特蘭大、馬競、多特蒙德、皇馬、馬競和熱刺等16支球隊，將進行抽籤，如果皇馬不幸抽到利物浦，絕對是提前進行的決賽，大家說呢
