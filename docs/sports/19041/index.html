江蘇隊遭到連敗，比賽表現不盡人意，張常寧發揮很不錯

2019-12-18 109條評論 1,048次閱讀 95人點贊

摘要：江蘇隊最近的表現不是很好，連續的輸球已經讓人難以忍耐了，以逸待勞對決天津隊，江蘇隊輸了，但是這還不是最慘的，隨後江蘇隊再次以2比3輸給了廣東隊，慘遭三連敗。

在和廣東隊比賽時，張常寧拿到了28分，但是她們還是輸了，為什麼會輸球呢？球隊的副攻有著很大的問題，雖然有張常寧這位高手，但是這也不能挽回這個敗局，作為球隊的教練，蔡斌也沒有給出讓球迷們滿意的答案，從對陣天津隊，再到對陣恆大隊，球隊的攔網和罰球問題還是很大的，她們沒有給球隊帶來有效地幫助。

蔡斌在這幾年沒有給球隊太大的變化，現在她們被連敗，球隊連進入4強都是一個問題，這個時候教練是要出來承擔責任的，張常寧在和恆大的比賽中發揮得很不錯，但還是以失敗告終，張常寧也是很無奈。

作為全華班的江蘇隊也是很不容易的，其他的球隊都有外援，但是她們沒有，本來有外援的球隊和沒有外援的球隊打比賽就有些不公平，而江蘇隊是全華班，這也讓她們在對陣其他球員的時候沒有太大的優勢，但是說實話，蔡斌教練對現在球隊連續輸球的問題應該要負一定責任的，現在已經三連敗了，這更是不能讓人忍受的，希望蔡斌能夠去反思一下，兩個賽季過去了，江蘇女排還在原地踏步，甚至還有一些後退。

江蘇女排現在需要一個有魄力的教練，這樣才會有更多的優秀球員被發掘出來，並且球隊如果想要追求成績，那就必須要引進一個或者兩個出色的外援，這樣才能夠獲得讓大家都滿意的表現。

要不總是打著全華班的旗號，但是卻沒有什麼成績，球隊是很難進步的。
