廣東恆大女排0-3不敵科內利亞諾 遭遇世俱杯首敗

2019-12-04 0條評論 21次閱讀 0人點贊

摘要

北京時間12月4日，2019年女排世俱杯進行了小組賽第二輪比賽，中國廣東恆大隊以0-3不敵義大利科內利亞諾隊，遭遇首敗，三局的比分為：16-25、22-25、21-25。

北京時間12月4日，2019年女排世俱杯進行了小組賽第二輪比賽，中國廣東恆大隊以0-3不敵義大利科內利亞諾隊，遭遇首敗，三局的比分為：16-25、22-25、21-25。

廣東恆大隊首發陣容為：主攻科舍列娃、拉巴捷耶娃，副攻曹婷婷、楊涵玉，二傳梅笑寒，接應杜清清，雙自由人林莉和王敏。科內利亞諾首發陣容為:主攻希爾、塞拉，副攻弗里、克魯伊夫，二傳沃洛茲，接應埃格努，自由人德-吉納羅。

首局科內利亞諾隊打的順手，以8-3領先。廣東隊進攻受阻，連續被攔，對方攔網和反擊不斷得分，以12-5拉開比分。梅笑寒二次球得手，楊涵玉發球得分，廣東追成9-14。此後曹婷婷背飛打手出界，隨後科舍攔住塞拉的二號位進攻，11-15。

杜清清衝進短球打中，埃格努和希爾分別進攻得手，科內利亞諾以19-12領先。杜清清再次衝進短球得手，楊涵玉攔網得分，拉巴捷耶娃處理球得分，對方連續進攻得手，22-15。最後塞拉反擊打中，科內利亞諾以25-16先勝一局。

第二局雙方比分交替上升，戰成7平。此後對方進攻連續得分，以12-9領先。栗垚換下科舍列娃，郁佳睿換下梅笑寒，曹婷婷攔網得分，埃格努扣球出界，11-13。曹婷攔住克魯伊夫的背飛，隨後她又處理球得分，廣東隊追成14-15。對方失誤不斷，栗垚反擊打中，雙方戰成18平。希爾反擊打中，栗垚進攻被攔，對方以21-18領先。

杜清清後攻打中，埃格努強攻得手，栗垚二號位進攻打手出界，20-22。埃格努強攻打中，曹婷婷前快打手出界，21-23。克魯伊夫背飛打中，杜清清進攻打手出界，22-24。杜清清發球出界，對方以25-22再勝一局。

第三局廣東隊用梅笑寒出任二傳，陳佩妍出任接應，杜清清改打主攻，外援拉巴捷耶娃和科舍列娃均未在場上。開局以6-8落後，許曉婷換下梅笑寒，杜清清和栗垚輪番進攻得手，對方一攻頻頻打中，14-15。此後埃格努跳發得分，杜清清和陳佩妍進攻接連被攔，對方以19-14領先。張鳳媚換下曹婷婷，廣東隊連續發球失誤，栗垚進攻打手出界，18-22。楊涵玉背飛打手出界，21-24。陳佩妍後二被攔，科利內利亞隊以25-21取勝。

對方埃格努得到全場最高的19分，希爾15分。廣東隊的栗垚得到10分，曹婷婷8分，杜清清8分。
