韋世豪無緣7號？國足號碼部分曝光：張稀哲身披10號，王燊超穿4號

天天要聞
2019-12-10

明天下午六點半，李鐵帶領的國足選拔隊將迎來東亞杯首場比賽，對陣日本隊。據《北京青年報》的消息，國足將身穿紅色球衣出戰，而球員的部分號碼也是提前曝光，其中張稀哲拿到10號球衣，7號球衣並非韋世豪，而是曹贇定，韋世豪身披19號球衣。

這次征戰東亞杯的國家隊，中國足協官方認定是國足選拔隊，其中參加40強賽的大多數球員都缺席，所以國腳身穿球衣號碼也不可避免要發生變化。

今晚據前方記者趙宇的消息，國足選拔隊參賽名單總共23人，組委會規定球衣號碼必須從1~23號排列。其中備受關注的幾個球衣號碼當中，北京國安的核心張稀哲拿到最核心的球衣號碼——10號，跟在俱樂部一樣。

值得一提的是，此前參加40強賽的時候，張稀哲一度被曝出有望身穿國足10號球衣，但最終還是由老隊長鄭智繼續身披10號，而張稀哲則穿18號球衣。

除此之外，7號球衣並非最大熱門韋世豪，而是申花國腳曹贇定，韋世豪拿到了球衣號碼是19號，而在國足對陣敘利亞的比賽里，韋世豪身穿4號球衣。韋世豪在廣州恆大都是身穿7號，同時在此前的國青隊和效力北京國安也都是如此，這次在國足選拔隊沒有穿7號，也是讓球迷有些意外。

而其他球員的球員號碼，根據記者趙宇的透露，包括了蔡慧康6號、于大寶22號、18號王子銘、4號王燊超、13號買提江、14號鄒德海、16號馮勁、梅方3號等等。
