哈德森帶頭反思山東男籃找回自信，暫不更換外援方為上策

2 週 ago

摘要

很多天以來，山東男籃一直被壓抑的情緒所困擾，如何找回自信，一直是他們需要解決的問題。在與上海隊比賽之前，他們終於找到了解決問題的鑰匙，哈德森的帶頭反思成為關鍵。“按說，我們不應該打成這樣啊，雖然有外援的因素，但國內球員久久找不到狀態，才是最關鍵的問題。”俱樂部一名高層說。

很多天以來，山東男籃一直被壓抑的情緒所困擾，如何找回自信，一直是他們需要解決的問題。在與上海隊比賽之前，他們終於找到了解決問題的鑰匙，哈德森的帶頭反思成為關鍵。

「按說，我們不應該打成這樣啊，雖然有外援的因素，但國內球員久久找不到狀態，才是最關鍵的問題。」俱樂部一名高層說。

山東男籃本賽季開局正常，該贏的比賽都贏了，輸了的比賽過程也不難看，這讓人們看到了希望，也掩蓋了球隊存在的問題。

事情在輸掉與青島隊的齊魯德比之後發生了變化，哈德森那場比賽發揮欠佳，而回到主場對陣四川，本來被看作是一場必勝之戰，但在最後時刻被翻盤，兩名外援集體低迷，梅斯的表現成為眾矢之的，換梅斯的呼聲日益高漲，鞏曉彬賽後也意味深長地說了一句：「我們這是在還債！」

的確，過去的這幾年，山東男籃幾度動蕩，球隊建設形同虛設，球員訓練長期荒廢，戰鬥力和凝聚力有所下降，主帥鞏曉彬難免有些著急上火。

壓抑的情緒是會傳染的，連久經沙場的哈德森都受到了影響。在客場對陣稠州銀行隊的比賽中，球隊第四節大比分落後，哈德森狀態就有點不對，而在哈德森被換下後，他的情緒全面爆發，甚至對鞏曉彬出言不遜。

比賽結束後，俱樂部高層與哈德森進行了談話，哈德森表達了自己的不滿，俱樂部高層也對哈德森進行了安撫，但沒想到的是，哈德森還是難以控制自己的情緒，居然連夜不打招呼回到了濟南，這讓俱樂部大為光火。

與此同時，俱樂部更換外援的步伐開始加快，威廉姆斯迅速抵達濟南，此時球隊還在浙江。

其實，連夜返回濟南的哈德森第二天就後悔了，他甚至想著第二天返回諸暨參加與浙江廣廈隊的比賽，但由於機票售罄，俱樂部也想給他一個教訓，山東隊以單外援作戰，不出意外地大敗於對手。

11月29日一早，士氣低迷的山東隊返回濟南，下午訓練之前，哈德森當著全隊的面做了道歉，獲得了隊友們的諒解，訓練得以繼續進行。

11月30日上午，球隊賽前訓練之前，球隊集體觀看了上海隊的比賽錄像，之後，哈德森再次進行了反思，並與隊友們進行了交流：「作為球隊的一員，發生這樣的事是不應該的。我已經到了職業的末期，希望大家共同努力，把比賽打好。」

教練組和其他球員這幾天也在進行反思，統一思想之後，山東隊的精神狀態得到提升，自信心也重新找回，在與上海隊的比賽中，全隊首節就投中6個三分球，防守也十分積極，讓對手三分球六投全失，首節比賽結束，山東隊36：15領先。

最終，全面爆發的山東男籃119：85大勝上海，止住連敗，也為自己留下了喘息的機會，全隊六人得分上雙。接下來，山東隊將主場迎戰江蘇同曦，這是一個相對較弱的對手。

更可喜的是，對上海一戰，此前廣受批評的兩名外援表現不錯，哈德森得到34分12次助攻，梅斯得到14分14個籃板，效率都很高。哈德森投中6個三分球，一向被質疑進攻能力的梅斯敢於進攻，6投5中。

球隊狀態回升，那麼現在問題來了，外援還換不換？

據了解，由於球隊連續作戰，前來試訓的外援威廉姆斯，還沒有得到全面試訓的機會，能力如何尚待觀察，而且球隊更需要一個能在內線立得住的大個子，威廉姆斯顯然還不是最佳人選。

由於每支球隊更換外援的次數有限，所以山東男籃更換外援一事必須謹慎對待，在威廉姆斯能力沒有得到驗證之前，暫時不換外援，保持球隊穩定，或許才是上策。
