中國舉重國家隊被剝奪三枚奧運金牌後，全簽約反興奮劑責任承諾書

2019-12-22 160條評論 3,327次閱讀 215人點贊

2018雅加達亞運會上，中國運動員在獎牌榜上一騎絕塵，但令人奇怪的是，觀眾們並沒有看到中國舉重隊參賽。後來根據網上透露，方知因為興奮劑事件，中國舉重隊被集體禁賽一年。經歷了慘痛的教訓後，中國舉重國家隊為備戰東京奧運會，集體簽約反興奮劑責任承諾書。

2017年1月13日，國際奧委會在其官方網站上發布信息，確認2008年北京奧運會樣本複檢中查出的中國女子舉重隊曹磊、陳燮霞、劉春紅3名運動員興奮劑違規成立，並作出取消她們北京奧運會成績，收回金牌等處罰措施。

根據國際舉重聯合會關於涉及興奮劑的處罰規定：一個協會在一年內出現至少三例興奮劑事件，該協會內所有運動員將被禁賽四年，如果是首犯，則有可能只被禁賽一年。之後，國際舉聯宣布對中國舉重隊禁賽一年，禁賽周期從2017年10月1日至2018年9月30日，因此他們無緣當年的亞運會。

解禁後，中國舉重國家隊在2019年世錦賽上派出20名運動員參加14個奧運級別中的13項比賽，最終獲得29金14銀10銅，其中總成績10金5銀3銅，此外還有11人20次刷新15項世界紀錄，創造了中國舉重隊參加世錦賽的史上最佳成績。中國隊之所以能夠取得如此好的成績，根據國家體育總局舉摔柔中心主任、中國舉重協會主席周進強的總結，隊員們具備強大的精神力量，賽前很多人都寫下請戰書要為國爭光。

同時，中國舉重隊成績的提升也離不開科學的訓練理念和訓練方式的轉變，隊伍更加註重肌肉、平衡、核心力量以及康復環節等等。除此之外，隊員們的衣食住行等後勤保障也得以盡量滿足，讓隊員們深受鼓舞。

有過禁賽一年的深刻教訓，國家舉重隊為了以實際行動表達向興奮劑堅決說不的強烈決心，全體成員均簽署了反興奮劑責任承諾書，從制度上保障興奮劑問題「零出現」！
