重要官宣！18歲國腳確認旅歐入豪門，恆大小將+武磊師弟先行一步

天天要聞
2019-12-20

自從2019年2月開始， 中國留洋球員便開始看見了新的曙光，因為武磊成功在西甲亮相，並且持續獲得首發出場的機會，本賽季還拿到了球隊7號戰袍，意味著他已經成為了一名合格的西班牙人球員和合格的西甲球星，現在武磊的身價已經漲到了1000萬歐元，是中國足壇歷史上身價最高的球員，這種成功讓不少國內球員和俱樂部都眼饞。

如今，2020年冬窗將至，已經有不少國內俱樂部醞釀送出優秀年輕球員旅歐留洋，追隨武磊的腳步，廣州恆大在奪取中超8冠王之後已經蠢蠢欲動希望一口氣送出多名留洋新星，其中韋世豪是恆大最看好的一人，球隊願意保障韋世豪的高薪待遇鼓勵他留洋，由於韋世豪過去一個賽季各項賽事打進15球助攻7次，能力上有希望在歐洲五大聯賽獲得出場時間。

按照近期的相關消息，韋世豪的留洋最快要到冬窗。現在，冬窗未至，旅歐進入法國的狂潮便要開啟，來自法國百年傳統豪門歐塞爾的重磅官宣消息，18歲的國腳田玉達確認旅歐入該豪門球隊，不過目前還只是試訓階段，歐塞爾官方稱田玉達已經確認赴法國參加球隊的試訓，通過試訓將獲得正式簽約加盟的機會。

田玉達是國字型大小隊伍的常客，是U18國足的主力中鋒，出自山東魯能青訓體系，2014年就被魯能視為重點培養對象送往巴西學習一年，2016年在全國U15聯賽中打進22球，在國少、國青踢球期間也表現神勇，10場比賽打進8球，去年年僅17歲的田玉達完成了三級跳成為U23國奧選拔隊的一員，被視為2024年奧運之星。

在去年的濰坊杯，田玉達曾率領魯能戰勝西班牙人殺入決賽，2019賽季外租效力於中甲球隊北京北體大，受到了高洪波的青睞，不過沒有獲得比賽機會。

現在，田玉達確認旅歐以試訓的方式進入法國老牌豪門歐塞爾，在他之前有恆大小將布格拉汗+武磊師弟季驍宣先行一步進入法國豪門球隊，作為恆大19歲新星的布格拉汗，他已經早早確認奔赴法甲球隊南特、里爾試訓，作為武磊師弟去年斬獲中甲MVP的季驍宣，他在武磊加盟西班牙人之後沒幾天便加盟了法乙球隊歐塞爾，本賽季身披10號戰袍，可惜依然還未得到重用，雖然歐塞爾作為法國豪門已經相對沒落，但是隊內競爭依然很激烈，擺在季驍宣和田玉達面前的挑戰都不小。

能走出終究是好的，但是踢不上球也確實雞肋，最令人感到遺憾的可能還是中資背景，歐塞爾和西班牙人都是中資球隊，所以武磊和季驍宣都拿到了象徵核心的球衣號碼7號和10號，但是都不一定能得到真正的重用，田玉達前往歐塞爾背後的直接動力也是中資，接下來的韋世豪若留洋也是入中資球隊格拉納達。如此可見，留洋和中資都是一把雙刃劍，武磊和季驍宣便是典型的2個案例，一個天一個地，處境相差太大。
