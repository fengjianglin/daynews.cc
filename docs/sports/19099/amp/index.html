敢罵李春江？國手中鋒離開廣廈也不愁下家 他是CBA里的香餑餑

天天要聞
2019-12-17

​

在CBA賽場上，李春江一直是特立獨行的存在，經常在比賽中對隊員們口吐芬芳。但不是所有人都百依百順，在德比大戰不敵浙江隊之後，主力中鋒蘇若禹就在社交媒體上吐槽發泄自己的不滿，矛頭疑似對準了主帥李春江。

​

看得出來，蘇若禹也是性情中人，在照片里還有「傻X自有天收」這樣極端的字樣，並且自己配文：「好的時候沒看見你放個屁，一場比賽你就開始咬人？八年了又怎麼樣！你不爽我一開始就讓我走啊…好像他媽的我願意在這待著一樣！」

​

一般而言，隊員們很少會頂撞主教練，不然真的很容易被冷板凳。但蘇若禹真的難以壓抑自己的情緒，因為在比賽的第二節還剩4分鐘的時候，因為一次防守犯規就被替換下場，再也沒有獲得出場機會，全場比賽他2投0中得分吞蛋。

​

蘇若禹敢這麼理直氣壯的最大原因，或許就是自己在CBA里擁有極高的前景。身高2米06，體重106公斤的他在內線擁有得天獨厚的優勢，本賽季他場均可以拿到6.8分4.5個籃板，這還是在出場時間只有20.3分鐘交出的成績單。而且他還要為胡金秋干臟活累活，屬於吃邊角料的存在。2018年夏天，他曾入選杜鋒男籃藍隊的集訓名單。

​

他打球一直勤勤懇懇，任勞任怨，進攻和防守都不惜力，卻經常被李春江痛罵，蘇若禹當然有委屈。在今年夏天，蘇若禹成為自由球員之後，引發了眾多球隊的哄搶，在一番周轉之後，他還是回到了廣廈，但一支球隊只能擁有三個頂薪資格，趙岩昊、胡金秋、劉錚、孫銘徽都虎視眈眈。再加上廣廈屬於私企，最近集團效益還不好，如果再次合同到期，蘇若禹真的可能離開了。

做一個對比，本賽季場均4.3分3.6個籃板的孫桐林，年薪高達600萬，事實上，在簽下這份大合同之前，孫桐林的數據不過是5.8分4.4個籃板，遠遠不如蘇若禹那麼好用。現在的北京隊內線也無人可用，土豪上海戰績一落千丈，他們瘋狂買人彷彿都買錯了。蘇若禹屬於稀缺資源，如果離開了廣廈，那麼他真的不缺下家，甚至還能掙得更多。
