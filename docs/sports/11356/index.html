德甲今日頭版：門興對拜仁會心一擊 多特拒絕出售桑喬

2019-12-09 1,181條評論 2,995次閱讀 1443人點贊

上半程的聯賽進入最後一個月，讓我們一起來看看德媒今日的焦點有哪些。

1.聯賽第14輪最後兩場，科隆客場0-2不敵柏林聯成為新任副班長；客場0-1遭帕德博恩絕殺，不萊梅在危機中越陷越深。

2.沒有努貝爾！德媒稱諾伊爾接近與拜仁續約到2023年，他想要絕對的主力位置。

3.補強後防，德媒稱萊比錫有意在冬窗引進摩納哥邊後衛亨利希斯，並已經送上報價。

4.門興在焦點對決中絕殺拜仁，一隊狂喜一隊驚慌，門興守住榜首，拜仁滑落到第七。會心一擊：拜仁的防線在門興的攻勢下暴露出實質問題，不再進球的萊萬讓拜仁失去依靠。

5.納格爾斯曼率領萊比錫高歌猛進，福斯貝里：「我最好的時代現在才到來。」

6.最後的背水一戰：勒沃庫森在歐冠小組賽末輪對陣尤文圖斯，全力爭勝以求出線。

7.進球、處罰、激情：連續四場進球，桑喬找回了狀態。佐爾克表示多特拒絕出售桑喬：「我們對此沒有任何計劃」。

8.備受追求的哈蘭德：在追逐偉大球員的路上。德媒稱萊比錫仍是哈蘭德最可能加盟的球隊，納格爾斯曼非常欣賞他。

9.穆里尼奧在熱刺：一切只是鬧劇？本周客戰拜仁，是熱刺和穆里尼奧正名的關鍵一戰。

（息木）
