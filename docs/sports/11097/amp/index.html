中國足球還需要高洪波！只有他在探究中國足球該怎麼踢，理念正確

天天要聞
2019-12-11

昨晚的一場東亞杯，國足或許又讓中國球迷傷心了。因為我們的國足選拔隊面對日本U22為主的國家隊最終1:2輸球，而且整個過程慘不忍睹，場面幾乎完全被對手壓制。我們連進入禁區的機會都少之又少，我們的防守總是能夠被日本這些年輕球員三傳兩倒就撕裂。所以輸球就一點不奇怪了。很多人都說這就是我們的整體實力與對手有差距造成的，但真的是這樣嗎？

縱觀正常比賽，我們的國足球員在個人對抗能力、個人技術和比賽經驗上都不比對方差，甚至佔據一定的優勢，但是放到一起就是踢得不好，防守搶不到球，進攻找不到傳球路線。一個非常主要的原因，就是日本隊踢得是歐洲式的快節奏，而我們踢得是中超的節奏。節奏慢半拍所以你對抗能力再強，技術再好，比賽經驗再豐富，跟不上對手的速度，你的優勢就蕩然無存。

我們的球員都習慣了前場有超強外援，那就是拿球之後有外援來接應，或者大腳傳給外援，讓他用個人能力解決問題。而其他球員只要原地等著，後者小步慢走就行了。然而在國家隊，我們的前場沒有超強能力的球員，沒有外援的滿場飛奔接應，於是我們的站著踢和走著踢就讓對手防起來非常輕鬆。因為人家無球跑動積極，而且節奏比我們快，於是乎我們就會看到，我們的進攻完全沒有出球點，一人拿球總是陷入多人的圍搶，基本上就會丟球，進攻完全打不出來。其實這一點在亞冠已經非常明顯了。

所以說，國足為什麼輸球，不是日本隊這些小孩兒有多強，而是我們整體理念的錯誤。中超這麼多年我們幾乎幾代人都是這樣踢球的，他們習慣了，所以在國家隊一時半會兒是改不了的。這麼多年來只有一個人意識到了這一點，而且去做了，也幾乎做到了，他就是高洪波。

追溯東亞杯的歷史，我們最近一次奪冠就是高洪波率領國足在2010年奪冠。當時高洪波沒有徵召任何一個大牌球員，鄭智、李瑋峰甚至是邵佳一都沒有在隊中，當時的中國隊是東亞杯最年輕的球隊。當時有很多媒體質疑高洪波，甚至連央視都不再轉播東亞杯的比賽。

但是高洪波神奇般地做到了，他帶領球隊先是0:0戰平了日本隊，而楊昊在這場比賽中罰丟了點球，讓中國隊遺憾地錯過了戰勝日本隊。而第二場他們3:0大勝韓國隊，打破了持續32年的恐韓症。第三場中國隊又輕鬆戰勝中國香港隊，最終奪冠。而那次的大勝，中國隊不是依靠個人的發揮，而是整體的對抗。

當時高洪波就認為我們的國足的水平應該在亞洲二三流的水平，而且中國隊要想贏球必須找到適合中國足球的踢法。於是他大膽地採用了和當時的大多數中超球隊不同的踢法，那就是走地面配合，打整體，當時的陣型是4-2-3-1。當然了中國隊的打法並不是完全模仿西班牙，我們主要打的還是快速反擊，只不過當時高洪波不讓球員像中超那樣後場長傳找前鋒，而是通過中場球員的一腳傳球，快速通過中場，這樣提高了反擊的傳球的準確率，再加上前場有能夠拿球分球的鄧卓翔，通過他的梳理和直塞找得分的良機。

而且高洪波也是在根據自己的戰術打法選擇球員，因為依靠的是整體，淡化的是個人的能力，強調的是戰術執行力，所以高洪波在每個位置上都找到了2-3個替補，而且經過1年多的反覆強化，國足已經可以打出漂亮的比賽了，比如後面我們曾客場1:0絕殺法國隊，也能在主場3:0大勝勒沃庫森。但是我們看國足的比賽，就能感覺中國隊能夠拿下任何一個對手，每個球員都非常自信。因為他們知道自己該怎麼踢，該怎麼跑位。

但是在高洪波下課被卡馬喬替代之後，這一切都毀掉了，中國球員又回到了中超的節奏和打法之中，而且隨著人員的更迭，一批一批的球員就只能踢中超節奏，中國隊就一蹶不振了。所以我們就看到了里皮帶領的國足也是一樣弱隊贏不了，強隊拿不下，因為里皮也不會費心的研究中國足球應該怎麼踢，適合怎麼踢。

如今的中國足球已經沒有突出的個人能力了，我們如果再不去依靠整體，再不去尋找適合我們的踢法，中國足球還會在渾渾噩噩中極速下墜，墜入萬丈深淵。你怎麼認為呢？
