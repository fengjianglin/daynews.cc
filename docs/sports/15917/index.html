女排聯賽四強留懸念！恆大反客為主雙殺江蘇，張常寧提前晉級夢碎

2019-12-14 919條評論 2,341次閱讀 774人點贊

2019-2020賽季全國女排超級聯賽戰至八強組第3輪，江蘇中天鋼鐵坐鎮常州體育館主場鏖戰五局以2-3不敵廣東恆大，這是繼複賽第1輪客場力拚三局以0-3遭零封后慘遭雙殺，由此改寫有效成績為2勝3負7分落後恆大3勝2負7分，無緣提前出線的同時導致晉級四強前景堪憂，反觀廣東恆大衝擊半決賽門票留懸念但手握主動權。

蔡斌（上海）挂帥江蘇中天鋼鐵坐鎮主場，安排主攻2號張常寧（隊長）和12號吳晗、副攻3號周馨億與11號王辰玥、二傳6號刁琳宇和接應16號龔翔宇、自由人18號倪非凡聯袂首發，第一局17-23和第三局16-21時由主攻10號許璐瑤短暫輪換吳晗。

廣東恆大是由方岩（法國）執掌教鞭，派遣接應2號栗垚（隊長）、主攻5號拉巴吉耶娃（保加利亞）和15號科舍列娃（俄羅斯）、自由人4號林莉（福建）與二傳6號郁佳睿、副攻16號曹婷婷（山東）與14號鄭益昕（福建）搭檔出戰。

​第一局由於刁琳宇跳飄出界和被栗垚反輪強攻，江蘇落後0-2時由張常寧打走一攻但跳飄失誤，恆大利用張常寧後三開炮至4-1可惜栗垚後二出界。吳晗強攻藉手出界但遭科舍列娃暴扣，江蘇因為吳晗打標誌桿到3-6好在輕吊落點，張常寧後三巧打追回2分但遭鄭益昕自接自扣到5-7。恆大通過栗垚後三和拉巴吉耶娃攔網，張常寧開炮與拉巴吉耶娃防反擴大11-6。

蔡斌休戰後拉巴吉耶娃開炮令江蘇止住連卡4分，恆大依靠科舍列娃變扣為吊但被張常寧強攻，拉巴吉耶娃後三出界由13-7連丟2分。科舍列娃強攻但攔不住龔翔宇前交叉，恆大接飛一傳好在科舍列娃重扣和栗垚後二至16-11。鄭益昕相繼背飛與栗垚後三到19-13迫使蔡斌休戰，恆大憑藉栗垚後三和反輪強攻再封堵至23-16連丟5分，好在科舍列娃和鄭益昕下球先勝25-21。

​第二局龔翔宇下球但遭拉巴吉耶娃反輪強攻，恆大通過科舍列娃重扣與吳晗開炮由0-1連得3分反超，江蘇吳晗和龔翔宇封堵栗垚與科舍列娃強攻逆轉4-3。栗垚背平止住連丟3分並且刁琳宇單手調整球失配，恆大依靠科舍列娃暴扣和吳晗鷹眼挑戰顯示強攻開炮，科舍列娃再下球到8-5時被龔翔宇後二，科舍列娃強攻藉手出界至10-7遭張常寧強攻和王辰玥探頭。

江蘇龔翔宇背平與栗垚反輪強攻出界追平11-11，張常寧跳飄下網並且攔防不住栗垚，龔翔宇防反至13-13可惜後二開炮。恆大憑藉鄭益昕背飛連得2分迫使蔡斌休戰，江蘇張常寧一攻和防反逆轉16-15逼迫方岩暫停。科舍列娃強攻再遭刁琳宇攔截15-17但鄭益昕封堵張常寧，恆大利用拉巴吉耶娃輕吊連得3分反超，栗垚與科舍列娃強攻但遭吳晗探頭落敗22-25。

​第三局由刁琳宇跳飄追發林莉、張常寧巧打至3-0，恆大依靠科舍列娃強攻止住卡輪但遭王辰玥偷襲，吳晗開炮與栗垚跳飄、鄭益昕跑動背飛和巧打由1-4連追4分逆轉。江蘇吳晗攔截鄭益昕背飛和龔翔宇防反至6-5，龔翔宇巧打遭科舍列娃攔截和拉巴吉耶娃下球。恆大利用鄭益昕背飛和拉巴吉耶娃封堵連得4分反超9-6，蔡斌休戰後接飛科舍列娃跳飄好在龔翔宇前交叉。

江蘇張常寧後三和吳晗封堵迫近9-10，可惜周馨億背飛下網丟掉2分好在張常寧後三輕吊。恆大由於栗垚被吳晗追發讓球至12-11但反輪下球和背平擴大15-12，江蘇龔翔宇暴扣和刁琳宇跳飄滾網導致曹婷婷墊飛連追2分。恆大栗垚防反後二到18-15迫使蔡斌休戰，科舍列娃與拉巴吉耶娃重扣、鄭益昕背飛和郁佳睿跳飄至24-19，拉巴吉耶娃下球到25-19改寫局分2-1令江蘇提前晉級四強夢碎。

​第四局由郁佳睿跳飄失誤和龔翔宇強攻藉手出界，恆大通過栗垚反輪強攻與科舍列娃巧打至2-3，科舍列娃重扣和栗垚跳飄滾網落界內反超5-4。吳晗平拉開與追發林莉墊飛、張常寧攔截郁佳睿二次球連得3分，江蘇張常寧暴扣至8-6但遭科舍列娃後三和拉巴吉耶娃防反追平。

恆大依靠拉巴吉耶娃封堵龔翔宇和防反到12-10，拉巴吉耶娃跳飄得分至14-12但遭許璐瑤防反。江蘇落後13-16時迫使蔡斌休戰，龔翔宇前交叉遭鄭益昕封堵至14-19，張常寧相繼防反和王辰玥攔網反超23-20上演瘋狂追分，24-21時憑藉王辰玥封堵栗垚後三再勝至25-21。

決勝局因為張常寧開炮和龔翔宇強攻被攔，江蘇張常寧未防起拉巴吉耶娃巧打到0-3，蔡斌鷹眼挑戰張常寧開炮再丟分。恆大由於攔不住張常寧和龔翔宇從4-0連卡3分，拉巴吉耶娃和科舍列娃強攻、栗垚探頭至7-4，但鄭益昕封堵到11-7連丟3分。張常寧後三由鷹眼挑戰顯示腳未過中線，江蘇由於王辰玥跳飄出界至12-14，最終吳晗強攻慘遭栗垚攔截惜敗12-15。
