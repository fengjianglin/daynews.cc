盤點體壇幾大富二代？有錢任性玩體育的富二代？家裡有礦的富二代

2019-12-15 3,261條評論 4,335次閱讀 83人點贊

在觀眾眼中，運動員都是依靠自己的努力通過領取工資、獎金和代言賺錢的，但事實上能夠取得成功獲得獎牌而領取獎金的運動員只是少數，而能夠擁有代言資格的運動員更是少之又少。對於一位運動員來說，努力訓練，在賽場上好好表現，拿下一份大合同是非常重要的一件事，但殊不知，有許多運動員甚至不用簽下大合同就能活得很滋潤，下面我們我們就來看一下中國體壇上的5大低調富二代，不好好打球就回家繼承億萬家產！

第5位、吳前

吳前，1994年7月7日出生於浙江溫州蒼南，中國職業籃球運動員，司職後衛，效力於CBA浙江稠州銀行金牛俱樂部。是一位後衛球員，也是國家隊年輕球員中除了郭艾倫之外最好的實力控衛之一。吳前目前的年薪大約在300萬人民幣，其實吳前就算沒有這樣的工作也能生活得非常舒服。吳前的父母是溫州當地出名的富豪，他的家族經營了很多酒店、飯店等。但是吳前並沒有選擇子承父業，而是選擇了籃球，因為他喜歡籃球。在生活中吳前也非常低調，有一次和周琦、王哲林約飯，周琦和大王都是開車去的，而吳前則是打計程車。吳前不僅名字低調人也低調，進步很快，在國家隊、俱樂部都有耀眼的表現。不少網友調侃吳前不就是無錢的意思嘛，不過無錢竟然是有錢人。

第4位、吳冠希

吳冠希，漢族，1994年9月29日出生。中國男子籃球運動員，司職中鋒，2015年1月，被國家體育總局競體司授予運動健將稱號。大家都知道陳冠希，但是吳冠希這個名字並不是特別熟悉。無關係是江蘇肯亞俱樂部的一名中鋒，但是認識吳冠希卻是因為張常寧，吳冠希並不像吳前那樣出名，但是他的身體爆發力很強，甚至可以追趕易建聯的腳步。

吳冠希是一位官宦子弟，他的父親是體育系統高管，他的家族裡面還有房地產生意。前不久吳冠希曝光了自己的豪宅引起網友的熱議：原來他打球只是愛好而已啊！

第3位、惠若琪

惠若琪，1991年3月4日出生於遼寧大連，女子排球運動員。惠若琪是我國女排的一位實力悍將，曾擔任女排隊長，在2014年世錦賽、2016年奧運會的比賽上惠若琪都立下了汗馬功勞。惠若琪的父親叫惠飛，據悉是某上市公司的高層、同時還是南京馬術協會會長、南京高爾夫球協會會長、南京排球協會會長等，可見惠若琪的家世也是非常顯赫。2018年2月3日，惠若琪正式宣布退役。4月30日，惠若琪和丈夫楊臻博在北京華彬莊園舉行了盛大婚禮。

第2位、魏秋月

魏秋月，1988年9月26日出生於天津，中國女排運動員，司職二傳。女排姑娘魏秋月不僅僅人長得漂亮，家境也是非常的好。魏秋月的家長是天津出名的商業大亨，身價自然就不用多說了。有很多球迷調侃：明明可以靠臉吃飯為何還要打球？如果不好好努力就要回家繼承億萬資產了。

第1位、李曉旭

遼寧男籃李曉旭可以說是中國男籃隊員們家境最好的一個了，李曉旭生活低調，經常在地攤上買東西，對於李曉旭的性格也有很多人猜測他的家庭條件應該不怎麼好，但是事實遠遠超乎大家想像，李曉旭的家裡面是經營鋼鐵和金屬、礦行業的，資產過億應該是輕輕鬆鬆的。之前也有網友猜測，如果統計一下李曉旭家裡的全部資產，買下一支遼寧隊應該不是話下，這是真正的「家裡有礦啊！」以上談到的五位球員都是非常低調的，雖然家裡都很有錢，都是富二代，但是他們在生活中非常低調，在賽場上非常頑強，明明可以繼承家裡的億萬資產，但是他們仍然選擇了運動員這條路，為了夢想而奮鬥。

還有一些鮮為人知的體育界富二代

當然，我們都知道在體育運動中有很多的體育項目可以說是非常賺錢，對於體育明星來說，他們的一場比賽就可以換來很多的錢，這也是很多人選擇打球的一個重要的原因，的卻在這個行業之中，如果你表現得出色，你就會成為這個行業的佼佼者，自然源源不斷票子會來到你的口袋之中。但是，對於一些人選擇體育真的是為了金錢，但是也有很多人選擇體育這條道路非常的純粹，就是對體育的那一份熱愛。

寧澤濤

寧澤濤是中國最成功的游泳運動員之一，拿過許多個世界冠軍。獲得了名利雙收，但這並非寧澤濤最初學游泳的目的，因為父母當初只是為了鍛煉他。實際上，父母也沒有靠寧澤濤改變命運的打算，因為他們也是成功人士，其父是某省級電視台的高管。

張玉寧

張玉寧是知名足球運動員，為了培養他，其父賣掉上海多套房產的故事眾所周知。實際上，賣掉幾套房產並不算什麼，因為張玉寧的父親在多個地方都有生意。

羅漢琛

羅漢琛的名氣沒有李曉旭吳前大，在CBA聯賽的存在感也不太強，未拿到太高的薪水。實際上，羅漢琛比絕大多數CBA球員都富裕，因為他來自一個富裕的家庭，其父是一名極其成功的金融人士。

林丹

就拿我們所熟悉的林丹來說吧，可以說他也是出生在一個非常富裕的家庭中，不能說家有千金吧，但是這輩子不愁吃不愁喝。今天之所以會選擇羽毛球之路，就是因為自己對羽毛球的那份熱愛，他只想通過自己不斷的努力和打拚，最終一天能夠成為超級丹。

有錢任性玩體育的富二代

意甲賽場一名中國型男引起多方關注，他就是國米9位董事之一，蘇寧的「太子爺」張康陽。父親資產高達780億，1991年出生的他在24歲時就成為國米董事，網友稱他年輕多金又低調，外型氣質謙遜有禮，張康陽或許可以超越王思聰成為新一代「國民老公」。1991年出生於江蘇南京，畢業於美國賓夕法尼亞大學沃頓商學院，作為蘇寧控股集團董事長張近東之子，張康陽現在負責蘇寧國際業務拓展工作，24歲時就成為義大利國際米蘭足球俱樂部九位董事之一。其實在體育圈內，不少富二代們已經能玩轉得風生水起，他們依靠家族財富以及繼承權，儼然將成為體壇下一代有力的掌權人。
