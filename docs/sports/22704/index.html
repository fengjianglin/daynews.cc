東亞杯-國足選拔隊2-0中國香港獲季軍 吉翔張稀哲聯袂建功

2019-12-19 706條評論 2,938次閱讀 1157人點贊

北京時間12月18日，東亞杯男足末輪，國足選拔隊2-0輕取中國香港。第8分鐘吉翔破門，第71分鐘張稀哲擴大比分。國足選拔隊取得1勝2負的戰績獲得季軍，中國香港3戰全敗墊底

前2輪的比賽里，國足選拔隊和中國香港皆不敵日韓，同樣遭遇到2連敗。這場比賽里，兩隊也將爭奪本屆比賽的季軍。國足選拔隊若保持不敗便能夠獲得季軍，中國香港則需要獲勝才能獲得季軍。

第8分鐘國足獲得角球，張稀哲將皮球主罰到禁區內，這也在禁區內形成一片混戰局面，吉翔在門前奮力爭搶落點，將皮球狠狠頂入對手球門，國足閃電戰破門取得1-0領先。

第17分鐘中國香港隊在前場連續傳遞製造殺機，基奧雲尼在禁區內迎球大力抽射，劉殿座撲了一下，隨後皮球狠狠的打在球門橫樑上彈出，國足逃過一劫。

第32分鐘黃威在前場左路得到皮球，隨後他輸送威脅直塞球，基奧雲尼轉身大力抽射，但劉殿座奮力將球撲出底線，化解了這次險情。

第34分鐘國足獲得前場自由球，張稀哲將皮球主罰到禁區內，吉翔後點頭槌將球做回，這時埋伏在點球點附近的于大寶非常有想像力的倒掛金鉤，將皮球稍稍打偏。

第39分鐘徐宏傑在背後將帶球突破的王子銘鏟倒，他也因此被主裁出示黃牌警告。國足獲得前場自由球，張稀哲將皮球主罰到禁區內，卻被防守球員奮力頭槌解圍。

第49分鐘曹贇定左路輸送傳中球，門前包抄的董學升甩頭攻門，但卻因為頂得太正，被門將直接將皮球得到。

第54分鐘國足獲得前場自由球，李昂操刀主罰選擇直接射門，打在人牆上後出現折射，皮球出了底線。第57分鐘羅梓駿右路傳中，門前包抄的黃威凌空抽射，將皮球稍稍打高。

第70分鐘董學升在禁區內拿球突破的過程里，被艾里奧踢到支撐腿倒地，國足獲得點球。張稀哲操刀主罰一蹴而就，國足以2-0擴大場上優勢。

第80分鐘于大寶中場斷球，帶球突破後直傳，譚龍左路將皮球輸送到門前，韋世豪搶點包抄甩頭攻門，將皮球頂在橫樑上，國足錯失繼續擴大場上優勢的良機。

第86分鐘王燊超在搶球的過程里，被陳俊樂給別了一下，王燊超倒地表情很痛苦，不得不到場下接受治療。由於國足用滿換人名額，輕傷不下火線的王燊超，又回到了場上。
