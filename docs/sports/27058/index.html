曼城贏球他最慘！表現出色卻淪為替補 本賽季難再獲瓜帥重用

2019-12-22 432條評論 2,198次閱讀 913人點贊

面對萊斯特城，主場作戰的曼城如願贏下比賽。憑藉這場勝利，暫時迫近了和利物浦之間的差距。有道是贏球不可怕、缺誰誰尷尬。對此，無緣首發的羅德里深有體會。明明表現一直挺好，突然就被瓜迪奧拉撤下了首發。之所以如此，是因為他拖垮了曼城的攻防體系。當然，這和他的個人表現沒關係，而是技術特點鬧的。

看到對陣萊斯特城的首發名單，相信不少曼城球迷都感覺要涼吧？飽受詬病的京多安居然又首發，而且還要跟伯納多席爾瓦打雙後腰。萊斯特城的反擊這麼犀利，這不是送人頭嘛！

然而，從實際比賽看，這組後腰搭檔的發揮非常穩健。尤其是京多安彷彿煥發第二春，攻防兩端的表現都很亮眼。值得一提的是，德布勞內長途奔襲後的那次神助攻，就是源自於他的分球。之所以會跟以往的表現大相徑庭，是因為瓜迪奧拉改變了比賽的哲學。

考慮到萊斯特城的犀利反擊，瓜迪奧拉決定把戰火隔絕在對方半場。為此，他讓控球能力更強的伯納多席爾瓦和京多安搭檔雙後腰。這樣做的最大好處，就是確保了曼城中前場的傳控優勢。說白了，每個人都能傳能帶。雖然防守硬度有所下降，但是找回高效傳控打法的曼城容錯率也提升了。面對萊斯特城生疏蹩腳的高位逼搶，把皮球運轉起來的曼城踢得遊刃有餘。雖然被瓦爾迪率先破門，仍然有驚無險的逆轉贏下比賽。正如不少老球迷賽後所說，這才是曼城的正確打法，有內味兒！

問題隨之而來，為啥本賽季的曼城鮮少踢出這樣的比賽呢？原因出在羅德里的身上。平心而論，他在曼城踢後腰時的防守表現很不錯，經常是曼城全隊攔截和搶斷最多的。放在其它球隊，羅德里這表現妥妥的主力。然而，曼城的打法註定了中後場球員都必須有進攻的屬性。偏重防守的羅德里，這一塊恰恰屬於弱項。所以，當他首發時，曼城的傳控下降了一個檔次。

與其同時，曼城的中場容錯率一併下降。趕上傳球失誤，曼城後防將直面對手的快速反擊，回追慢的隱患也一起暴露了。這其中也包括羅德里，他的覆蓋範圍和回追時機選擇，明顯不如費爾南迪尼奧。回顧曼城這賽季的幾場輸球，基本都是源於這一點。事實上，這成了死循環：由於進攻不如以往給力、曼城加強進攻大舉壓上、結果因為容錯率下降被打反擊，如此循環往複。至於羅德里，則屬於典型的贏了個人數據，卻沒能幫助曼城贏球。

所幸的是，瓜迪奧拉及時發現了這個問題，藝高人膽大地拿下羅德里。不得不說，能做出這個決定挺需要勇氣的。畢竟羅德里此前的表現很不錯，而且還是夏窗加盟的標王。也正是這次調整，讓找回自己的曼城贏下關鍵一戰。戰勝萊斯特城後，曼城暫時和利物浦相差十一分，冠軍歸屬仍然還有懸念。

當然，羅德里也不用擔心徹底淪為替補。以曼城超強的整體實力，打中下游球隊時還是可以給他首發機會的。除此之外，羅德里也還年輕。在瓜迪奧拉的指導下，完全有希望補上進攻的弱項。隊友斯特林就是最鮮活的案例，當年的快樂男孩不就被調教成大英帝星了嘛！
