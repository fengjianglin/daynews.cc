魯能球員大婚邀請半數現役球員，山東魯能將帥近全員到場

2019-12-22 80條評論 1,069次閱讀 143人點贊

山東魯能球員宋龍12月21日在青島完成自己人生中的婚姻大事，而在婚禮現場更是邀請半數現役足球球員，山東魯能將帥更是悉數到場。而更有意思的是婚禮座次表安排中，更具有足球特色，在眾多座次安排中，山東魯能成為唯一一支屹立歐洲豪門之間的球隊。

首先在座次安排上面，採用極具足球特色的安排，採用歐洲豪門球隊的名稱對座次進行安排，其中有皇家馬德里桌、曼聯桌、曼城桌、巴薩羅那桌等等，而山東魯能是唯一一支屹立在歐洲豪門的中超隊伍而在山東魯能桌上面坐的大部分是山東魯能俱樂部的領導及球隊教練團隊成員。

而本次宋龍婚禮上面，邀請了中超現役球員眾多。在伴郎方面，有北京國安球員王子鳴，廣州恆大球員鍾義浩，山東魯能球員陳科睿及羅安東。其次還有李建濱、鄭龍、鄒正、朱建榮、牟鵬飛。此外山東魯能球員將帥近乎全員到場，其中有俱樂部總經理孫華、主教練李霄鵬。在曼城桌上面有蒿俊閔、張弛夫婦、鄭錚、王彤夫婦、金敬道、吳興涵、劉彬彬、韓鎔澤；在多特蒙德桌上面有宋文傑、李松益、劉軍帥夫婦、李海龍、姚均晟夫婦、成源夫婦以及曹盛。此外在媒體和球迷方面，邀請了山東魯能知名球迷董軍董大姐、PP體育山東魯能跟隊記者孫睿以及山東體育記者孟祥瑋等人。

這次宋龍大婚之日，讓不少球迷了解到目前山東魯能球員的婚姻情況，比如姚均晟、王彤、劉軍帥都已完成自己的婚姻大事。這讓不少魯能女球迷感到有些沮喪。

最後祝願宋龍先生新婚快樂，能夠在足球場上更好的發揮自己的球技！
