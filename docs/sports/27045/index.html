又見暴力足球！拜仁大將面部遭狠踢險被廢，對手遭70000球迷狂噓

2019-12-22 387條評論 4,804次閱讀 1325人點贊

德甲聯賽第17輪，拜仁主場迎戰沃爾夫斯堡，在比賽過程中，出現了「暴力足球」的一幕，拜仁邊衛大將帕瓦爾的面部被對方球員用腳踢到，痛苦倒地，對方球員奧塔維奧引來拜仁7萬多現場球迷的狂噓。

讓我們看一下這「殘忍」的一幕。那是比賽進行到第66分鐘，拜仁球員帕瓦爾在飛身頂球的那一刻，遭到了對方奧塔維奧狠狠的一腳暴踢，正好踢中了帕瓦爾的臉部，法國人迅速用手捂住臉部，隨即倒地，表情十分痛苦。

從慢動作中可以顯示，奧塔維奧這一腳踢得力度十足，見到帕瓦爾受到欺負，庫蒂尼奧向裁判申訴對方的暴力犯規，因為當值主裁只對這次犯規做出了黃牌警告。而當場的球迷也十分不滿，認為這個動作足以構成紅牌，集體用噓聲聲討對手的莽撞行為。

還好的是，帕瓦爾並沒有被對手踢到關鍵部位，而是堅強的繼續比賽。隊友也被這種精神所鼓舞，本場狼堡的防守動作十分兇狠，之前阿諾德一次也是擊打了萊萬多夫斯基的臉部。

還好的是，拜仁在最後時刻過關，拿下了這個強硬的對手。齊爾克化身神奇替補打開僵局，格納布里再下一城，4分鐘的時間裡打進兩球，幫助拜仁取勝過關。憑藉這場勝利，拜仁進一步追趕積分。
