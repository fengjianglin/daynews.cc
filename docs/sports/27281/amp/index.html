2019足籃成績一團糟，中國女排成唯一亮點

天天要聞
2019-12-21

再過十天，2019年即將過去。過去的一年，中國「三大球」成績參差不齊。中國男足持續慘淡，從亞洲杯到世預賽再到東亞杯，一共輸掉七場關鍵比賽，晉級12強賽蒙上陰影；中國女足無論在世界盃還是在東亞杯，成績低迷，陷入寒冬。

中國男籃在世界盃失敗後，即將首次缺席奧運會，創下新記錄。中國男排在接連失意世聯賽、亞錦賽後，目前還在為一個奧運名額苦苦掙扎。

三大球中，只有中國女排不僅成功奪得東京奧運會入場券，還用十一連勝的戰績奪得了世界盃，成為三大球唯一亮點。

針對三大球成績普遍糟糕的現狀，近日，國際體育總局專門召開會議，研究討論如何發展三大球的難題。然而，面對如何發展，怎麼發展的問題，三大球的前景並不樂觀。

2019年，足球和籃球都經歷了不同程度的慘敗。11月，在世預賽1：2負於敘利亞後，國足爭奪小組第一的希望基本破滅，里皮宣布辭職，引發地震。

12月，在東亞杯延續對陣日韓多年不勝之後，中國男足的世界排名再度下滑一位，從上一期的75位跌至第76位，創下年度新低。

值得一提的是，中國男足在東亞杯上將「逢日不勝」的記錄又延續了一場。從1998年到2019年，21年間，中國男足從未在正式比賽中戰勝過日本。東亞杯後，國足將逢日不勝的帽子戴入2020年。

男足一地雞毛，女足成績持續低迷。

在今夏法國女足世界盃上，中國女足僅取得對陣南非女足這一場勝利，最終苦澀收場。東亞杯上，中國女足在270分鐘僅打入一球，面對世界排名更低的韓國，中國隊打得非常被動，而面對亞洲冠軍日本女足，也被對手遠遠甩開，0：3落敗更是讓球迷絕望。

足球不給力，籃球也差強人意。

今年，在本土舉辦的男籃世界盃上，中國男籃不但沒有小組出線，而且創下了歷史上首次無緣奧運會的紀錄，跌到了谷底。而男籃世界盃，也讓外界看到了中國男籃與世界高水平球隊間的差距越來越大。

與中國女排的輝煌戰績相比，中國男排在阿根廷籍主教練洛薩諾的帶領下，相繼在世聯賽、奧運資格賽和亞錦賽遭受挫折，老帥沈富麟臨危受命，走馬上任，目前正在為爭奪一個奧運名額苦苦掙扎。

常年不勝、日趨下滑，這樣的字眼困擾著中國三大球，也凸顯了這一現狀已經遠遠超越了一個主教練、一群球員、一支球隊的範疇。

足籃排的未來到底在哪兒，確實值得總局好好深思下了。
