馬加特：看好拜仁奪冠，穆勒能為德國踢歐洲杯提供幫助

2019-12-06 0條評論 21次閱讀 0人點贊

摘要

虎撲12月6日訊 前拜仁主帥馬加特日前接受了《AZ（慕尼黑晚報）》的專訪。AZ：馬加特先生，拜仁周六

虎撲12月6日訊 前拜仁主帥馬加特日前接受了《AZ（慕尼黑晚報）》的專訪。

AZ：馬加特先生，拜仁周六會對陣領頭羊門興，上世紀70年代，你也和當時非常成功的小馬駒對決過，有哪些回憶？

馬加特：當我到德甲的時候，德甲的大門興時代剛剛結束了，我印象深刻的是馬特烏斯當年在門興的起步，和他在中場的對抗非常激烈。

AZ：2019年12月的格拉德巴赫氣氛非常熱烈，一些球迷甚至在憧憬冠軍，你認為他們能做到嗎？

馬加特：他們這賽季開局很好，表現很穩，現在就要看他們能不能保持，羅斯能不能讓球隊更進一步，不過通常來說，只有一支球隊可以拿到德甲冠軍。

AZ：你是說？

馬加特：如果拜仁不犯大錯，正常發揮，冠軍會回到慕尼黑，雖然門興現在領跑，萊比錫表現出色，但我不認為他們有能力拿走拜仁的冠軍。

AZ：在科瓦奇下課後，弗里克給拜仁帶來了哪些提升？

馬加特：倒不是因為新教練怎麼樣，給球隊帶來更好的表現，而是因為有些球員在以前的教練手下已經不想或者不能拿出好的表現，這也是換帥的正面影響，但弗里克確實給拜仁注入了一些新的動力。

AZ：比如穆勒，又拿出了很好的表現。

馬加特：弗里克的優勢在於做助教的時候和球隊關係密切，他可以很好地評判球員，穆勒最近幾場對拜仁作用很大，相比於科瓦奇，弗里克把球隊團結的更加緊密。

AZ：穆勒上周授勛，巴伐利亞州長還祝福他明年參加歐洲杯，他回歸還有意義嗎？

馬加特：這你得問國家隊教練（笑），我能說的是，現在距離歐洲杯還有很長時間，穆勒和胡梅爾斯一樣，多年來一直是國家隊重要球員，我確信以他的能力和經驗，當然可以給德國隊在歐洲杯上提供幫助。

(編輯：Mask)
