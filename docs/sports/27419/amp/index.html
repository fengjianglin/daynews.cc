39歲高雷雷又一壯舉！國王杯亮相60分鐘+晉級，中國足球的驕傲

天天要聞
2019-12-20

當武磊代表西班牙人在國王杯進球，並且創造一系列紀錄的同時，中國足球活化石級別的人物高雷雷也在國王杯賽場完成了首秀，以39歲零5個月的年齡，高雷雷代表龐費拉達出場60分鐘，並且幫助球隊取勝。20年前，高雷雷代表國安隊攻破遼足大門，幫助山東魯能奪冠，當時他19歲，如今39歲的高雷雷卻依然活躍在足球舞台上，這樣的堅持，值得尊重！

高雷雷的足球生涯非常傳奇，1980年，他出生於北京，曾經在武漢紅桃K隊踢球，然後在1999年到2006年，高雷雷曾經在北京國安隊打拚。在1999年的甲A聯賽里，遼寧隊一度領跑，而且在最後一輪也有爭冠機會，不過在和國安隊的比賽中，遼寧隊丟了1球，當時為國安隊破門的正是高雷雷，當時替補出場的情況下，他打入了一記精彩的遠射，幫助國安隊1-1戰平遼寧，那一輪的比賽，魯能5-0橫掃武漢隊，最終，高雷雷間接幫助山東魯能奪冠。

在離開國安隊後，高雷雷曾經在澳超紐西蘭騎士隊效力，2007年，他加盟芬超邁帕隊，還跟隨球隊亮相當時的歐洲聯盟杯。隨後，高雷雷還效力過惠靈頓鳳凰隊、芬甲古迪比、明尼蘇達群星隊，在回歸國內賽場後，高雷雷曾經代表北京八喜隊踢球。早在2015年，高雷雷在退役後一度復出，當時他加盟了西乙科爾內利亞隊，當時代表球隊有過2次出場。在今年的8月份，高雷雷又加盟了西乙球隊龐費拉達，雖然他的加盟有贊助商運作的因素，但是在39歲的年齡，高雷雷依然堅持活躍在足球場上，這本身就是奇蹟。

在等待4個月後，高雷雷終於在國王杯賽場迎來了首秀，在龐費拉達對陣佩尼亞競技的比賽里，高雷雷在比賽中替補出場，完成了自己代表新東家的首秀。據悉，在這場比賽里，高雷雷出場了60分鐘，球隊苦戰120分鐘，在加時賽里1-0取勝。

賽後，高雷雷激動的在微博上寫道，「為了今天付出太多太多……39歲零5個月，出場60分鐘，1-0戰勝對手，西班牙國王杯順利晉級！故事待續！」

雖然高雷雷無法和梅開二度的武磊相提並論，但是在國王杯賽場，他也跟隨球隊晉級，而且他也實現了自己的夢想，那就是在39歲的年齡亮相國王杯賽場，這已經是一份奇蹟，也是屬於中國足球的奇蹟。
