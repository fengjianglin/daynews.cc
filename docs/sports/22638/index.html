巴薩恥辱夜揪出頭號罪人！主場射門9-17被完爆，再這麼踢魂沒了

2019-12-19 631條評論 2,859次閱讀 1201人點贊

北京時間12月19日凌晨，2019-2020賽季西甲聯賽第10輪一場補賽打響，巴塞羅那主場迎來皇家馬德里的挑戰。此役比賽乏善可陳，雙方最終0-0握手言和。主場作戰的巴薩沒有拿下皇馬，表現可以說相當糟糕。賽後數據統計顯示，巴薩此役佔據劣勢。

在很長一段時間，應該說自從瓜迪奧拉上任後，巴塞羅那對於控球的執著，幾乎和對於勝率的渴望是一樣的。巔峰時期的巴薩，控球率超過70%如同家常便飯。控球多，雖然不意味著一定能贏，但皮球在自己的腳下，讓巴薩更安心，巴薩慘敗也有，但被圍攻的比賽不多，原因就在於，巴薩能夠拿住皮球。

然而從恩里克時期開始，巴薩慢慢放棄控球，直到巴爾韋德上任，巴薩幾乎徹底放棄對於控球的執著。在國家德比之前一項統計顯示，巴爾韋德到目前執教巴薩踢了141場比賽，有九場比賽，控球率不到50%。此役面對皇馬， 巴薩依然沒有控球優勢，射門數也比對手少。

賽後數據顯示，這場0-0，巴薩射門9次，而皇馬射門17次。巴爾韋德實在太「弱」了，巴薩坐擁梅西、蘇亞雷斯和格列茲曼的3大世界級球星組合，進攻就像是便秘一般，根本就踢不出任何風采，讓人失望不已。不少球迷認為，巴爾韋德已經把巴薩帶到坑裡了！他是巴薩失去靈魂的頭號罪人。

讓人可怕的是，梅西目前的巔峰期肯定是處於尾聲階段了，然而在這個最後對於歷史地位衝擊的時刻，他碰到了巴爾韋德。考慮到梅西很難在阿根廷國家隊有太多的機會，原本渴望再拿歐冠冠軍，如今來看無法讓人期待，這樣一支巴薩真能夠在如今的歐冠中突出重圍，笑到最後嗎？

從里傑卡爾德時期開始，巴薩的主教練位置基本都獲得認可，每一任教練雖然不是同等成功，但巴薩一直都是那支巴薩，這是拉瑪西亞給這支豪門球隊扎的根。然而如今來看，巴爾韋德的巴薩讓人看不懂。擅長控球？在巴爾韋德麾下，這樣的優勢不復存在，改打反擊？巴爾韋德也沒有把速度和防反灌輸到巴薩的血液中。巴薩更像是一支隨意發揮的隊伍，能否進球，甚至要取決於梅西的狀態。這不，梅西被限制，巴爾韋德也不知道如何進球了……
