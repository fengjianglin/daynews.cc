博格達諾維奇25+5+5國王大勝送勇士4連敗

2019-12-16 1,130條評論 3,504次閱讀 1642人點贊

NBA常規賽12月16日繼續進行，最終國王客場以100-79大勝勇士。

首節開始，雙方上來手感都不佳，前4分鐘6-6平手。希爾德、別理察接連上籃幫助國王先行領先，伯克斯三分、罰球反超。不過隨後博格達諾維奇上場帶領國王打出一波10-0！首節戰罷國王以26-18領先8分。

次節上來，博格達諾維奇三分、拋投，巴格利也有2+1，國王一波8-0拉開分差！勇士全民皆兵迅速回敬一波12-1迫近比分！阿里扎三分、補籃連拿5分穩住局勢，互有得分後半場戰罷，國王以51-42領先9分。

易邊再戰，雙方上來一開始還你來我往互有得分。隨後勇士開始進攻受阻，國王趁勢打出一波16-2拉開分差！接著輪到國王進攻受阻，但是勇士也只能斷斷續續拿下6分。國王再打出5-2，三節戰罷國王以79-59領先20分。

末節開打，埃文斯先中三分，但是博格達諾維奇2+1回敬。普爾再中三分，但是巴格利暴扣、博格達諾維奇連中兩記三分！！比賽就此失去懸念。

國王這邊，希爾德19分，巴恩斯15分，博格達諾維奇25分5板5助，巴格利17分6板

勇士方面，伯克斯11分，拉塞爾僅8分
