吳清源杯歡迎儀式 仲邑信也：希望她將來能下中國圍甲

2019-12-02 0條評論 21次閱讀 0人點贊

摘要

歡迎晚宴現場　　弈城圍棋訊 12月1日第二屆“吳清源杯”世界女子圍棋賽決賽前夜祭儀式在福州香格里拉酒

歡迎晚宴現場

　　弈城圍棋訊 12月1日第二屆「吳清源杯」世界女子圍棋賽決賽前夜祭儀式在福州香格里拉酒店舉行，中國圍棋協會副主席、棋聖聶衛平，著名超一流棋手、吳清源弟子林海峰九段，福建省圍棋協會名譽主席戴濱輝先生，福州市體育局局長張涵，福州市體育局副調研員彭樂，國家圍棋隊教練王磊八段，日本圍棋代表團團長孔令文，韓國棋院理事、GG拍賣公司社長、韓國代表團團長姜明周，韓國代表團領隊朴升哲，韓國代表團教練朴正祥，吳清源助理牛力力，日本東北新社ISC株式會社部長總監梅澤秀行，日本圍棋專欄作家大島正雄，以及決賽棋手王晨星五段，崔精九段，日本仲邑堇初段，中國吳依銘二段出席了歡迎晚宴，儀式由王香如主持。

中國圍棋協會副主席、棋聖聶衛平致辭

　　儀式上中國圍棋協會副主席、棋聖聶衛平與韓日代表團團長分别致辭，其中棋聖聶衛平在致辭中講到：這是第二屆吳清源杯決賽，我首先代表中方向遠道而來的朋友表示歡迎，我聽說昨天的比賽王晨星是靠著極好的運氣進入決賽，而崔精是靠著極好的實力進入的決賽，現在就看究竟是極好的實力厲害，還是極好的運氣厲害，我希望他們都發揮出最高水平，讓我們欣賞女子巔峰之戰，我作為中方我當然希望王晨星贏，希望運氣戰勝實力，但是作為職業棋手我是給雙方加油的，說到中日兩國的超新星比賽也是非常有意思，來自日本的小仲同學將於中國的小吳同學進行三番棋比賽，我同樣希望小吳能贏，但是小仲更年輕更有人氣，也希望他們兩個能發揮出最高水平。

日本代表團團長孔令文致辭

　　隨後日本代表團團長孔令文致辭表示：很高興又一次參加吳清源杯，這次比較可惜日本棋手沒有撐到最後階段，但是主辦方仍給了我們一個很好的舞台讓我們小棋手和中國的棋手對局，代表日本棋院表示感謝，今天感觸非常大，參加了吳清源老師的紀念儀式，因為吳清源老師是我在日本見過面的，他真的是一個偉大的前輩，我父親和林海峰老師也是大前輩，我雖然現在還年輕但是看到坐在下面的仲邑堇和吳依銘又是一代人，感受到了圍棋傳承的力量。

韓國代表團團長姜明周致辭

　　韓國代表團團長姜明周致辭表示：我這次來感受到了吳清源圍棋會館，今天又參加吳清源大師的紀念儀式，感受到了中國是一個很愛圍棋的國家，明天開始的決賽希望韓國崔精和中國的王晨星都能發揮出最好的實力。

主持人王香如

致辭過後主持人邀請吳依銘與仲邑堇登台接受現場採訪。
主持人：首先採訪吳依銘，聽說你非常喜歡三國演義，你知道三國演義裡面那些是與圍棋有關的故事嗎？
吳依銘：我印象最深刻的是關羽刮骨療毒哪個故事，當時關羽手臂上被毒箭射傷，但是仍然能夠下棋，並沒有痛苦之色，我想他沉浸在圍棋的思考當中。

主持人：你自己是什麼時候開始喜歡下圍棋的？
吳依銘：我覺得下圍棋可以給我帶來快樂，我也堅信我能走好這條路。
主持人：好非常有志氣，那麼你明天的對手你覺得怎麼樣？有信心嗎？
吳依銘：信心肯定是有的，但是友誼第一比賽第二嘛（此時吳依銘手摸了一下仲邑堇的頭，隨即現場發出笑聲）感覺她（仲邑堇）非常可愛，我相信可以跟他成為好朋友。

仲邑堇與吳依銘

主持人：請問仲邑堇你今年入段以後受到了非常大的關注，並且對日本男棋手也保持不敗，那麼你印象中最深的對手是哪一位？
仲邑堇：在中國參加夢百合杯的時候輸給了李軒豪。
主持人：那麼今天上午參加吳清源大師的活動，你對福州有什麼印象嗎？
仲邑堇：這裡是一個非常好的地方。

仲邑堇一家上台接受採訪

主持人：那麼我們來採訪一下仲邑堇媽媽，聽說您是讓小仲邑堇三歲開始接觸圍棋的，請問那時候就有想培養他走職業棋手的道路嗎？
仲邑幸：三歲的時候並沒有想到當職業，當時家裡正在辦圍棋教室，所以就讓她（仲邑堇）下，結果越下越喜歡，所以興趣也就上來了，自然而然就走到今天了。

主持人：接下來問一下仲邑信也先生，您對於小堇現在的表現有什麼看法，覺得她未來的發展會怎麼樣？
仲邑信也：我在這裡說一個願望，我希望我的女兒將來能夠成達到可以參加中國男子圍甲的水平！（現場響起掌聲）

王晨星與崔精接受採訪

採訪過兩位小朋友後，主持人邀請殺入決賽的王晨星與崔精上台接受採訪。
主持人：今天看到崔精是盛裝出席，非常漂亮，你是第二次殺入吳清源杯決賽，你今年的成績這麼好戰勝了很多男棋手，請分享一下你的感受
崔 精：以前也戰勝過男棋手，但是今年的狀態比較好發揮的比較好吧。
主持人：那麼明天的決賽是在吳清源大師的會館進行，你對吳清源大師有什麼印象嗎？
崔 精：吳清源老師像傳說中的老師，他是一個在中國韓國日本名氣都非常大的棋手，我像韓國其他職業棋手一樣很尊敬他，他的思想是「調和」也是我未來要努力的方向，吳老師已經提前做到了，我非常尊敬他。
主持人：接下來請問王晨星，昨天的比賽可以說是驚險的逆轉了，你現在的心情如何？
王晨星：首先感謝主辦方給我們提供這麼好的一個機會，非常渴望與崔精的對決，非常期望明天的比賽。
主持人：你的丈夫也是非常著名的棋手劉星，不知道你現在當了媽媽又了寶寶之後對比賽會不會有影響？
王晨星：各有利弊吧，可能用在棋上的時間比以前少一些，但是覺得每天心情很好。
主持人：接下來我的問題是從吳清源杯公眾號徵集的網友的問題，請問王晨星平時和劉星在一起會下棋嗎？
王晨星：下的很少，因為我總是贏不了。（笑）
主持人：看來這個誰贏的多的問題就不用再提了，那麼請問崔精你狀態保持的非常好，你每天訓練時間有多長？
崔 精：韓國國家隊訓練時間是早上10點到下午5點，回家後除了鍛煉身體外都在擺棋。
主持人：真的非常用功啊，那麼你覺得女棋手有沒有可能男棋手獲得世界冠軍呢？
崔 精：那就是我的目標，我在努力像吳依銘與仲邑堇，像她們這樣年輕優秀的棋手也層出不窮，以前覺得不可能的東西現在也實現了，女棋手戰勝男棋手奪世界冠軍也可以實現。

王晨星與崔精接受獻花

　　採訪環節過後第二屆「吳清源杯」世界女子圍棋賽歡迎儀式正式結束進入棋手嘉賓用餐時間，12月2日中午12點，第二屆「吳清源杯」世界女子圍棋賽決賽將繼續在福州吳清源圍棋會館進行，12點10分中日女子圍棋超新星邀請賽也將進行，弈城網將直播全部對局，歡迎棋迷朋友屆時登錄弈城觀戰。（馬冬）
