林書豪36分創賽季新高尤度首秀13+14 北京客場23分大勝廣州

2019-12-03 0條評論 21次閱讀 0人點贊

摘要

北京時間12月3日，CBA常規賽，廣州主場以86-109不敵北京，林書豪36分創造CBA生涯新高，尤度迎來加盟北京隊的首秀，砍下13分14籃板。

北京時間12月3日，CBA常規賽，廣州主場以86-109不敵北京，林書豪36分創造CBA生涯新高，尤度迎來加盟北京隊的首秀，砍下13分14籃板。

廣州隊尼克爾森25分12籃板，賽克斯20分10籃板4助攻，劉豐搏12分；北京隊林書豪36分8籃板6助攻4搶斷，翟曉川15分5籃板，尤度13分14籃板3搶斷，方碩12分4助攻，朱彥西12分。

廣州戰績為5勝7負排在第15位，球隊的表現平平，目標亦是衝擊本賽季的季後賽；北京戰績為8勝4負排在第6位，亦是希望客場能夠告捷。值得一提的是，北京隊的新外援尤度被激活，將在本賽季首次代表北京隊出戰。

首節比賽，方碩急停跳投，林書豪命中三分球，翟曉川溜底線反手上籃打進；張永鵬和尼克爾森聯袂搶分助廣州追擊，北京以8-6領先。加尼尤外線命中三分球，林書豪連續跳投打進，翟曉川遠投三分命中，北京以20-10拉開場上比分。賽克斯強吃王驍輝打成2+1，隨後他又強行上籃打進，助廣州隊追擊。王驍輝右側遠投命中，段江鵬拋投命中，該節戰罷，北京以27-21領先。

次節比賽，朱彥西外線命中三分球，尤度貢獻暴扣，北京以34-24拉開10分差距。朱彥西投進追身三分球，林書豪後撤步跳投打進，方碩和朱彥西皆命中三分球，林書豪輕鬆上籃打進，尤度籃下成功吃餅，段江鵬干拔跳投將球打進，助北京隊將場上的優勢繼續拉大。半場戰罷，北京以59-30遙遙領先。

第三節比賽，林書豪空切上籃打進，常林籃下打成2+1，翟曉川命中三分球，北京以69-35拉開場上比分。隨後北京隊連續強攻利用罰球搶分，朱彥西外線遠投命中，助北京以82-50仍然牢牢保持著場上的優勢。賽克斯滑翔上籃打進，劉豐搏貢獻暴扣，孫鳴陽成功補籃命中，助廣州隊追擊。該節戰罷，北京以84-58領先。

末節比賽，賽克斯強吃林書豪打成2+1，劉豐搏也進攻打成，助廣州將場上分差縮小到20分以內。加尼尤轉身跳投命中，尤度籃下強打製造2+1，段江鵬遠投命中，尤度又強行進攻得手，北京以100-74遙遙領先。接下來的比賽再無懸念，北京以109-86勝出。

首發陣容

廣州：陳盈駿、尼克爾森、賽克斯、劉豐搏、張永鵬

北京：林書豪、方碩、加尼尤、翟曉川、常林
