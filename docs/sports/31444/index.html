別人努力打球，他們使勁挖坑！CBA第20輪最水外援非他們莫屬

2019-12-23 927條評論 4,264次閱讀 1139人點贊

CBA常規賽第20輪戰罷。每輪比賽，都是幾家歡喜幾家愁。有的球隊雖然獲勝，卻是喜憂參半。究其原因，還是外援惹的禍。本輪最水外援都是誰？且看下文一一道來。

廣廈主場險勝吉林。威爾斯表現異常亮眼，但普拉姆利卻是完全划水，和勝利毫無關係。該如何清理這顆定時炸彈，是廣廈目前最該思考的問題。

深圳結束了自己的三連敗。但是馬丁的發揮，還是一如既往地醜陋。占著全隊第二高的出手數，卻完全提供不了火力。長此以往，深圳的路怕是走得會越發艱難。

福建慘敗遼寧。康寧漢姆無疑成了場上最坑的那個。全場僅得到6分的他，進攻防守樣樣不行。好不容易有些好轉的福建，是換援，還是硬著頭皮走下去？

第三位

常規賽第20輪，廣廈主場迎戰吉林。最終，這場比賽廣廈艱難拿下，終結了自己的三連敗。

其中，首次出戰的小外援威爾斯表現堪稱是先驚後喜。但是，歡喜之餘，廣廈陣里的一個人，卻依然讓人憂心不已。

那人就是，普拉姆利。全場比賽，普拉姆利出戰31分鐘，僅拿到了14分和11個籃板。數據看似中規中矩，但命中率卻是低得嚇人。全場15投5中的數據，當真是有些難看。

除了在進攻端的乏力，普拉姆利在防守端的表現也並不是非常出色。場均1.5次蓋帽，對於一個2米11的大外援來說，顯然有些不夠。而場均9個籃板，也處於聯賽中鋒的下游水平。

綜合來看，普拉姆利的水平著實有些尷尬。既不能像約翰遜那樣大量得分，也不能像梅斯、莫蘭德那樣成為籃板領袖，更無法提供尤度那種頂級防守。

樣樣都不太精通，普拉姆利彷彿已經成為了一個球場混子。每場交出一些不高不低的數據，就打卡下班。這樣的外援，顯然是達不到廣廈的預期的。

這場比賽，廣廈全場都遭到了壓制。只不過依靠威爾斯在末節的出色發揮，廣廈才驚險獲勝。然而，勝利跟普拉姆利，幾乎就沒啥關係。

對廣廈來說，終結了三連敗是一件喜事。但普拉姆利，卻是他們絕對難以忽視的隱患。迅速拔除這顆定時炸彈，是廣廈目前最該做的事兒。

第二位

常規賽第20輪，深圳主場迎戰山東。最終，深圳終於結束了自己的三連敗，但球隊外援馬丁的發揮，卻還是十分不盡人意。

全場比賽，馬丁出戰27分鐘，17投6中，只拿到了12分6板3助攻的可憐數據。

佔據著全隊第二高的出手數，卻只拿到了全隊第四高的得分。這樣的效率，可是讓人難以忍受。

倘若不是隊友的手感好，命中率高，這場比賽怕是並沒有那麼容易拿下。身為一個外援，出手比隊友多，得分卻比隊友少，這絕對是不合理的。

試想一下，如果深圳其他人的手感並不好，而馬丁仍舊佔據著大量的出手權，仍舊這般低效。那深圳隊又該怎麼贏球？又怎麼可能贏球！

此戰之前，深圳遭遇三連敗，與馬丁的低迷發揮撇不開關係。從現實來看，他確實難以成為球隊值得依靠的第二得分點。

到目前為止，深圳9勝11負，排在聯賽第13。衝進季後賽，已經是一個擺在球隊面前最大的難題。而拜克斯和馬丁這兩個外援的狀態，卻是時好時壞，而且壞的時候居多，實在是難堪大用。

好不容易贏了球，但這賽季的深圳的比賽，卻是讓人怎麼看怎麼不舒服。無論如何，外援的問題，的確是現在擺在深圳面前最嚴峻的問題。

第一位

常規賽第20輪，福建主場迎戰遼寧。

最終，福建大比分不敵遼寧，四連勝慘遭終結。而康寧漢姆，無疑就是導致球隊輸球的罪魁禍首之一。

這場比賽，康寧漢姆全場出戰29分鐘，6次出手僅命中2次，三分線外更是3投0中，只拿到了可憐的6分6籃板的數據。另外，還有1次失誤和3 次犯規。

上半場比賽，福建一直緊咬比分，甚至一度反超了對手。但下半場比賽，康寧漢姆卻僅僅貢獻了2分，導致球隊啞火，被對手迅速擴大了比分。

在遼寧防守重心明顯在王哲林身上的時候，康寧漢姆其實擁有不少的空間。但是，他一是把握不住機會，二是自己也創造不了機會。這就導致球隊異常被動。

縱觀全場，康寧漢姆顯得異常迷茫。在進攻端，他無法提供有效的幫助，在防守端也是沒有太大的建樹。對面的巴斯全場16投12中，如入無人之境，成為導致福建輸球的主要因素之一。

到目前為止，康寧漢姆出戰5次。球隊雖然獲得了四連勝，但他卻只能拿到14.2分的場均數據。這是目前福建上場過的外援中，數據最低的一個。

這賽季的比賽尚未過半，福建如今半程發力，闖入季後賽似乎又成了一件可以期待的事兒。既然有野心，那麼對於外援的選擇，就更是一件嚴肅的事情。

是繼續更換外援，還是鐵了心讓康寧漢姆在場上划水當藍領。福建，你可得想好了。

實力備選：

浙江客場大勝四川。但是，這場的劇本卻和上場極其地相似。蘭茲博格大殺四方，而鄧蒙則繼續划水。

全場比賽，鄧蒙14中4，僅得到10分。雖然球隊贏球了，但鄧蒙最近的表現，可真是讓人憂心。

遼寧客場大勝福建。但師弟好像又鬧起了情緒。全場比賽，他僅僅出手了9次，得到了14分。這和前幾場的大量出手，形成了鮮明的對比。

這已經不是第一次師弟如此打球了。但巧的是，似乎師弟每一次減少出手數量，遼寧都能輕鬆贏球。也許，在主教練換不動的情況下，徹底讓師弟走人，才是遼寧贏球的正確方式。

天津主場慘敗給青島。這場比賽，青島隊的亞當斯和約翰遜超神發揮，無人可擋。但是，托多羅維奇的表現，相形之下，就菜了很多。

全場，他僅得到了7分和5個籃板，實在是有些難堪。要知道，他的隊友蘭德爾可是拼盡了全力，砍下了39分。他這樣的表現，真是有些對不起隊友。

上海客場惜敗給了新疆。這場比賽，莫泰發揮亮眼，砍下了30分加28個籃板的大號兩雙。縱觀全場，上海一度處於領先優勢。但如此好局，最後卻慘遭逆轉，麥卡勒姆自然難辭其咎。雖然拿下了17分，但他全場14投5中，命中率奇低。

最關鍵的下半場比賽，更是8投2中，只拿到了5分。球隊被逆轉，和他的低迷密不可分。上場8次，僅獲兩勝，麥卡勒姆的表現，真是太糟糕了。
