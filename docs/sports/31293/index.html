先後效力過倫敦三強的“萬金油”，世界足壇少有的10號後防球員

2019-12-24 138條評論 1,161次閱讀 55人點贊

又一場倫敦德比結束了，穆里尼奧的球隊輸了，輸球的卻不是切爾西。面對老東家，「魔力鳥」和他的熱刺十分被動，最終被昔日弟子蘭帕德兩球「教訓」。

其實最早的倫敦德比還指的是阿森納與切爾西，只不過隨著熱刺的崛起和槍手的沒落，這個詞有了新的代指。值得一提的是，有這麼一位球員，在這三支球隊都長期效力，他是穆里尼奧的弟子，也是蘭帕德的隊友與對手，他就是威廉·加拉。

球風強悍，大器晚成

加拉是法國運動員，在場上的位置是後衛，也是足壇少有穿10號的後防球員。從技術來講，他是一位非常全面的防守球員，曾是切爾西和法國隊的主力，位置靈活多變，風格強硬。作為世界頂級的後衛，加拉不管是優點還是缺點，都非常明顯。

他速度極快，補位能力超強。但是身高是他最大的短板，因此對於制空權往往沒有什麼優勢可言。即便如此，加拉有非常敏銳的嗅覺，時常可以通過意識來完成禁區內的搶點得分。有一點需要提到的，他是一位大器晚成的球員。

加拉的實力還是不容置疑的，是金子總會發光。在1999-2000賽季的歐冠小組賽，在馬賽隊效力的加拉憑藉自己的進球幫助球隊擊敗曼聯。這一球也讓英國俱樂部注意到他，之後加拉便轉會切爾西征戰英超。

憑藉在英超出色的表現，加拉終於得到了國家隊的注意，順利進入了法國隊。進入國家隊效力是每一個足球運動員的夢想，更何況法國隊是世界老牌強隊，有深厚的文化底蘊，能夠效力於法國隊，對於加拉來說是一種莫大的榮譽。

切爾西鐵衛，轉投阿森納

多年以來，加拉對於切爾西的後防線有著不可磨滅的貢獻，他在「藍軍」新老交替的階段起到了銜接的作用。加拉對於切爾西的戰術體系有多麼重要？我們通過一些場外的事情就可以揣測一二。

加拉可以在公開表達對米蘭的愛慕之意，他可以對於穆里尼奧的安排表達不滿，但是「魔力鳥」卻無可奈何。由此我們能夠看出加拉對於切爾西的重要性。

當然，這並不意味著加拉就是非賣品。對於俱樂部來說，所有的球員都是交易的籌碼，因此交易來的還是非常突然。隨著新援卡瓦略的加入，加拉的空間被壓縮，在場上機會變少的他開始尋求新的出路。巴拉克的到來搶走了他的13號球衣，更是最後一根稻草。其實加拉還真的是不缺下家，多個俱樂部都向他拋出了橄欖枝，一名球員總是要追求表現機會的。

此後，在多方錯綜複雜的關係影響下，加拉在轉會截止前一天，跟阿什利·科爾互換東家，加盟了阿森納。有意思的是在代表阿森納對戰切爾西的時候，加拉攻入了一顆非常漂亮的頭槌，我想這也是他對於老東家的「謝禮」吧。

備受爭議的阿森納隊長

加拉還有讓人熟知的一個身份就是阿森納隊長。隊長對於一個球隊來說，作用是非常大的。首先要有足夠的影響力，讓球員們信服，其次就是要足夠的自律，給所有的球員做好榜樣，最後情商也是不可缺少的，因為隊長要處理好球隊里錯綜複雜的關係。

加拉之前的一任隊長是與他同年同月同日生的「亨利大帝」，2007年，亨利離開了阿森納，溫格直接把隊長的位子給了加拉。「教授」的這個舉動其實也是非常大膽，照理說，應該是副隊長吉爾伯托升為隊長，這個決定一出，不免讓人為整個俱樂部的內部團結捏了一把汗。事實證明加拉還算是一位稱職的隊長，起碼截止到他公開隊內矛盾之前是這樣的。

個性十足，險釀慘劇

加拉十足的個性不僅僅表現在場上，在場下亦是如此。因為他的個性，還差點釀成一出慘劇，怎麼回事呢？

因為訓練和比賽任務繁重，法國隊主教練多梅內克一方面是想釋放一下壓力，另一方面也是想提升一下團隊凝聚力，備戰世界盃，便舉辦了一次活動。他帶著隊員們玩卡丁車去了，沒想到這些球員直接玩嗨了。尤其是加拉，在這次遊戲中，他展現出了自己「高超」的駕駛技術，直接把車給開翻了。

運動員最重要的便是自己的身體了，如果出現任何損傷，那麼將無法參加之後比賽。好在加拉翻車的瞬間頭腦還是清醒的，及時做出了正確的處理，因此並沒有對他的身體造成過大的傷害。雖說沒受傷，但是驚嚇還是免不了的。這出「卡丁車驚魂」估計會被加拉銘記一輩子。

通過這次事件，我們更能夠看的出來加拉是怎樣的一名球員了。他極度情緒化，一旦興奮起來可能什麼都顧不上了，在場上加拉激情十足，往往能夠帶動隊友們的情緒，這也是他能夠成為一個優秀後衛的原因之一。倫敦是加拉的福地，「倫敦三強」切爾西、阿森納、熱刺都有加拉的身影。

再優秀的球員總會迎來退役的一天，加拉也不例外，在2014年10月16日，加拉通過媒體宣布了退役的消息。

在談到這個決定時，加拉動情地說到：「這件事情是很不容易宣布的，我曾經告訴自己，我希望能夠一直延續球員生涯，但是到今天，我發現自己不能繼續了。」
