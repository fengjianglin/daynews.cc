CBA遼寧吉林八大看點，誰能贏下東北德比？師弟瓊斯孰高孰低？

2019-12-13 2,917條評論 3,932次閱讀 2796人點贊

北京時間12月14日19:35，CBA第17輪將迎來一場焦點大戰，遼寧男籃將坐鎮主場迎戰東北兄弟吉林男籃！上個賽季常規賽由於雙方同組大戰4場，每一場比賽幾乎都打得難解難分，其中不乏三加時的經典戰役，新賽季雙方再次交手同樣看點多多，接下來為您樹立本場比賽的八大看點：

NO.1 哪支球隊能夠贏下東北德比？

​

遼籃近期取得了一波三連勝，士氣正旺，同時這也是他們連續三個主場作戰的最後一個主場，球隊上下自然渴望用一場勝利開啟客場之旅；吉林隊無愧於巨人殺手的稱號，他們也是本賽季迄今唯一一支同時戰勝廣東和新疆的球隊，此戰做客挑戰遼寧，他們還會在演巨人殺手的角色嗎？

NO.2 遼籃能否衝擊賽季最佳排名？

​

目前積分榜上除了廣東和新疆稍事領先，從第3名的吉林到第6名的北京都是11勝5負的相同戰績，而遼寧隊本場比賽一旦獲勝，他們將在戰績上反超吉林隊，甚至有可能在本賽季第1次暫時衝到常規賽第3名的位置。此前遼寧隊最高排名是第4名，那麼本輪過後，他們能創造本賽季迄今的最佳排名嗎？

NO.3 師弟直面瓊斯

​

本場比賽兩隊的超級外援多米尼克瓊斯和蘭斯史蒂芬森將迎來直接對話。兩名球員有諸多相似之處：勁爆的身體，出色的單打能力，甚至在個人打法上都十分接近。

​

瓊斯最近狀態十分出色，上一場又砍下了三雙；師弟在融進球隊方面做得越來越好，但在防守方面還需要加強。此役兩人的直接對話必定吸引眼球！

NO.4 鍾誠首戰老東家

​

本賽季租借期滿重回吉林的鐘誠，在個人數據方面迎來了回升。同時這也是近三個賽季以來，他第一次以對手的身份迎來同遼寧隊的比賽。再一次來到遼寧體育館，不知鍾誠會有怎樣的表現。

NO.5 巴斯纏鬥霍爾曼

​

上一場比賽，遼寧隊的大外援巴斯表現出色，得到了25分和11個籃板的兩雙，面對號稱聯盟第一大外援的莫泰尤納斯毫無懼色，並且不落下風。霍爾曼上一場比賽，儘管只有18分和6個籃板進賬，但是二分球卻有10投9中的超高命中率。本場比賽兩名身材和打法都相近的外援將在內線過招。

NO.6 大韓能否統治內線？

​

上一場比賽，遼寧中鋒韓德君比較輕鬆地貢獻了17分和12個籃板的兩雙數據，同時送出了4次封蓋，在油漆區震懾力十足。本輪面對內線實力並不突出的吉林隊，大韓還能統治三秒區嗎？

NO.7 郭艾倫PK姜宇星

​

上一場比賽，遼寧隊的郭艾倫發揮出色，首節就得到了13分，全場貢獻了26分7籃板6助攻和4搶斷的全面數據，是遼寧戰勝上海的首功之臣。姜宇星上一場也得到了20分，並且成為了吉林隊一個重要的得分點。此番兩名後衛的對話，或對比賽走勢產生深遠影響。

NO.8 趙繼偉三分能否再立新功？

​

本賽季趙繼偉悄悄地扮演起了一名穩定的射手的角色，賽季至今三分球命中率達到了40.1%，幾乎是是遼寧隊最穩定的外線輸出。上一場面對上海也有7中3的三分球命中率，這是習慣了繼偉扮演球隊大腦的球迷本賽季看到他的新技能。本場面對吉林，趙繼偉的三分還會再立新功么？
