1-1，94分鐘讀秒絕平！與巴薩主帥同名的小將揮拳怒吼，瘋狂砸地

2019-12-16 39條評論 1,905次閱讀 974人點贊

西甲第17輪，皇馬對陣瓦倫西亞的比賽引起了外界極大的關注，因為隨著巴薩被皇家社會逼平，皇馬迎來了登頂積分榜的機會，此次做客作戰，皇馬如果拿下瓦倫西亞，那麼將在3天後以最飽滿的士氣迎接國家德比對陣巴薩。

然而很意外，皇馬在客場場面佔優的情況下，被瓦倫西亞率先取得進球領先，進球之後，瓦倫西亞防守很堅韌，皇馬怎麼進攻都無功而返，比賽時間也拖入了傷停補時階段，眼看傷停補時也快結束，就在最後時刻，皇馬頭號射手本澤馬最後射門被擋出獲得角球，角球開出後，皇馬門將庫爾圖瓦頭槌攻門，不過被瓦倫西亞門將撲出。

而瓦倫西亞門將撲出後，皮球並，沒有解圍多遠，恰巧落到了本澤馬腳下，本澤馬拿球後近距離勁射，1-1，讀秒絕殺，皇馬全隊瘋狂慶祝，本澤馬打進這球後，也在射手榜追平了梅西。

從巴薩被逼平看到登頂反超希望，到本場比賽1球領先，最後時刻又上演絕平拿到1分，緊追巴薩，皇馬這場絕平也最大限度地提升了球隊士氣，有利於3天後的德比大戰，而本澤馬進球後，1人最搶鏡，他就是與巴薩主帥巴爾韋德同名的小將F巴爾韋德，他跪地揮拳怒吼，對草地連砸4次，儼然彷彿好像是他絕殺了比賽一樣。
