恭喜對手總冠軍，致謝大郅，這才是我們想看到的球迷

天天要聞
2019-12-03

摘要

在任何一個聯賽的籃球比賽中，球迷一直都是比賽的一部分，而且是非常重要的一部分。很多時候，球迷的表現是可以影響到比賽的，比如說不久之前的世界盃，在最後一場比賽中，是主場球迷的呼喊聲給了阿聯無限的力量。

在任何一個聯賽的籃球比賽中，球迷一直都是比賽的一部分，而且是非常重要的一部分。很多時候，球迷的表現是可以影響到比賽的，比如說不久之前的世界盃，在最後一場比賽中，是主場球迷的呼喊聲給了阿聯無限的力量。當然了，也有一些不和諧的瞬間，比如說當年的毆打球員， 還有最近的嘲諷趙睿跟阿聯等，這些也都是我國球迷做出的事情。當這些不好的事情終歸是少數，我們想看到的是這樣的瞬間。

「贏就一起狂，輸就一起扛！」說到這句話，作為一個球迷，你一定不會覺得陌生，在很多地方都能見到這句話，今年世界盃的關鍵戰中也能夠看到類似的標語，其實這句話最早是出現在12-13賽季的季後賽第二輪，但是北京被對手0-3橫掃，結束了當賽季的征戰。但是在他們回去的時候，無數球迷到了火車站接站，球迷們手中高居標語，口中大喊，「贏就一起狂，輸就一起扛！」看到這一幕，無數球員當場落淚，這大概是CBA歷史上最暖心的一幕了。

還有上個賽季，廣東面對所有的質疑，也打破了所有的質疑，成功奪冠，拿到了球隊歷史上的第九個總冠軍。總決賽的最後一場比賽，是在新疆的紅山主場，這裡素有魔鬼主場之稱，但廣東用一場大勝帶走了這個賽季，帶走了總冠軍，早在第四節的時候，比賽就失去了懸念，但是，絕大部分新疆的球迷並沒有提前離開球場，即便是在比賽結束之後，確定廣東是最終的勝者之後，他們還是留在主場，跟所有廣東的球迷一起，送上了最熱烈的掌聲，在我看來，這才是球迷最好的樣子。

在幾乎所有感動的NBA瞬間中，都一定會見到這幾件事，當年太陽城的球迷打出了「nash，its OK，you can go」，還有當年步行者的球迷們，全場高舉「謝謝你雷吉」，同時全場高呼，「one more year」，這些都成為了最感動的畫面。而在13-14賽季，在得知大郅將要退役的時候，每一個八一的客場，都成為了大郅的主場，在這些客場比賽中，經常會見到對大郅的祝福語。而回到八一主場的時候，看台上布滿了「14」，球迷口中高呼，「大郅！留下！」

還有上海的球迷們。對於這個球隊而言，球隊歷史上除了有大姚之外，還有大劉，在大劉離開上海的四年內，每一次來到上海打比賽的時候，球迷都會早早在通道里等待，等他路過的時候跟他擊掌，每一次劉煒得分或者是上場，球迷都會高喊「劉煒回家」，他們就這麼喊了四年，大劉最後還是回去了。

上述說的只是眾多感人畫面中的極小一部分，畢竟篇幅有限，還有太多的球迷感人畫面無法寫出。無論是什麼，好的一面總是會被壞的一面搶風頭，就像在說到CBA球迷的時候，人們總是會想到當年的圍堵球員事件，還有各種辱罵，其實球迷們也都有可愛的一面，類似上述的事情還有很多，有了更好的球迷，才會有更好的聯賽。
