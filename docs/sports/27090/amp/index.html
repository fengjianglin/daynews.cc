義大利超級盃決賽——C羅戰略轟炸機，尤文對攻拉齊奧

天天要聞
2019-12-22

北京時間12月23日00:45，沙特首都利雅得，義大利超級盃決賽——尤文圖斯VS拉齊奧。

拉齊奧本賽季戰力激增，聯賽11勝3平2負進38球失16球火力鼎沸，剛剛意甲15輪主場3:1力擒尤文彰顯強悍戰力。進攻端莫因比萊狀態極其驍勇，聯賽17射5助獨造22球戰力噴火，科雷亞聯手凱塞多合力轟10射戰力不容小覷。拉齊奧攻擊型中場傳控和輸送能力強悍，雙核阿爾貝托、米林科維奇極度活躍，頻頻連線進攻端製造殺機，阿爾貝托視野開闊腳法精湛貢獻11助領跑助攻榜，米林科維手握4助錦上添花。防守端拉齊奧相當穩健，三後衛體系運轉流暢，拉齊奧最近聯賽豪取8連勝，狀態極其火熱，此役再度對決尤文，拉齊奧摩拳擦掌醞釀雙殺。

尤文本賽季遭遇挑戰，聯賽13勝3平1負進31球失17球領跑意甲，最近拿下2連勝狀態回歸神勇。進攻端主帥薩里祭出4-3-3攻擊陣型，三叉戟聯袂登場激活尤文攻擊屬性，伊瓜因作為支點無限騷擾對手防線，C羅和迪巴拉連線中前場頻頻製造殺機，C羅連續進球，最近5戰6射極其驍勇。上輪對決桑普一役，C羅驚世駭俗制空頭槌暴扣，折射其狀態重返巔峰。進攻端C羅動力強勁，防守端尤文穩定性不斷攀升，此役坐擁主場，尤文磨刀霍霍復仇心切。

本場比賽走勢：義大利超級盃是聯賽冠軍和杯賽冠軍之間的對決，擁有較高的含金量，歷史上尤文13次奪冠，拉齊奧7次捧杯。目前，雙方狀態和勢頭均相當不錯，拉齊奧進攻端動力強勁，防守端穩健如鐵，尤文此役難言輕鬆。不過，尤文最近攻防兩端運轉流暢，進攻端C羅戰力噴火，攜手迪巴拉殺傷力巨大，鑒於此，不妨看好尤文主場不敗或3:1勝出，期待C羅。
