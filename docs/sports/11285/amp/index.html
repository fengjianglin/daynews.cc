聲音

巴爾韋德：這場比賽是挑戰嗎？很難，我們要和他們競爭。國際米蘭將全力以赴晉級。

巴爾韋德：沒有梅西？看看我們的賽程表，我只是在為我的團隊做最好的事情。

巴爾韋德：我總是要解釋為什麼有些球員在陣容里而有些不在。阿爾巴和塞梅多沒有醫療綠燈，梅西也休息了，沒有更多原因。

巴爾韋德：我希望看到最好的國際米蘭。他們是意甲聯賽的領頭羊。在第一回合，他們讓我們非常困難。我們是來競爭的，我們想贏。

巴爾韋德：內托明天會上場。他是個偉大的守門員，他目前為止還沒能上場。

巴爾韋德：普吉？我不能確切地告訴你誰來踢球。我們還帶來了阿勞霍和其他球員。普吉昨天上場了，你明天就能看到陣容了。

巴爾韋德：比達爾一月離開？我沒有和俱樂部里任何人說過這樣的話，我也沒有考慮過。我們現在不在一月份。

巴爾韋德：我希望球隊明天有競爭力，因為國米會要求我們這樣做

巴爾韋德：我們知道梅西是誰，他對我們意味著什麼，但也有一些比賽我們沒有他，我們也能打得很好。

巴爾韋德：勞塔羅·馬丁內斯？他又快又強壯。他在諾坎普跑出空擋打進了一個漂亮的進球。他是蘇亞雷斯的接替者嗎？我不想談論這些，談論來自不同俱樂部的球員是不禮貌的。

巴爾韋德：蘇亞雷斯在歐冠客場遭遇進球荒？我沒想過這個問題，我只記得有人在新聞發布會上問我這個問題。去年他有過機會，但沒有改變，但我們對他創造的東西很感興趣，這才是更重要的。

巴爾韋德：拉基蒂奇的未來？我們還沒有到達冬季轉會窗口。這個季節還很長。

巴爾韋德：阿圖爾？自從上次為巴西踢球以來，他一直有一些問題，我們希望他能儘快康復，這樣他就能幫助我們。

巴爾韋德：阿圖爾的恥骨感覺到越來越不適了。前幾天我們和他談過了，他寧願休息一下看看情況是否好轉。上個賽季他整個賽季都在拖這個問題。這個賽季他控制得好多了。

巴爾韋德：法蒂？我們得等著看他明天是否準備好了，他周末有些不舒服。

菲爾波：代表巴薩出征歐冠不可思議！

菲爾波：我們來這裡是為了贏得這場特別的比賽。

菲爾波：在過去的幾場比賽中，我努力像阿爾巴那樣找到梅西，我仍然在適應。

菲爾波：阿爾巴很快就接管了我的位置？我只是想盡我所能，最後教練會決定的。

菲爾波：在巴薩這樣的俱樂部踢球不容易，我在學習和嘗試。

菲爾波：梅西在38歲時會怎麼樣？梅西是世界上最好的球員，他只會繼續進球。

菲爾波：我仍然需要獲得在皇家貝蒂斯的自信，我在那裡踢得更積極。我的隊友現在幫了我很多忙。

莫拉蒂：在歷史，聲望和品質上，巴薩和國米的對決總是非常特殊的。有一種決賽的感覺。

迭戈·瓦萊霍斯（梅西兒時的朋友）：梅西和這裡的其他人一樣，都是普通人。我們摔倒了，騎自行車時劃傷了自己。我們拿著水彈走到街上，朝公共汽車扔去。（圖為阿根廷羅薩里奧 拉巴賈達的梅西壁畫）

巴托梅烏：預計一月不會有任何動靜。我們有一支平衡的隊伍。我們什麼都不缺，但我們也不能失去一個人。有過幾次球員透露過離開意願，但從沒有在一月份。

巴托梅烏：對勞塔羅·馬丁內斯感興趣？這是個陷阱式的問題。我能說的是國米踢得很好，他並不是唯一給我留下深刻印象的球員。國米是意甲冠軍絕非巧合。

戈丁：巴塞羅那對勞塔羅·馬丁內斯的興趣？如果他被別人看中，那是因為他在國際米蘭，國際米蘭是世界上最好的球隊之一，他正在成長。我們很高興他和我們在一起。

戈丁：巴薩非常強大，事實上是世界上最好的球隊之一。我們不考慮誰上場或者誰不上場，我們期待著最好的巴薩來到這裡。我們必須專註於自己，儘力做好準備，爭取勝利。
