摘要

中國男籃最近幾年的戰績並不理想，尤其是新人球星中鮮有獨擋一面的球星出現，日本男籃如今的實力雖然還不及中國隊，但是當易建聯退役之後，中國男籃的整體實力恐怕要下降一個檔次。

中國男籃最近幾年的戰績並不理想，尤其是新人球星中鮮有獨擋一面的球星出現，日本男籃如今的實力雖然還不及中國隊，但是當易建聯退役之後，中國男籃的整體實力恐怕要下降一個檔次。

然而日本方面近年來卻湧現多名籃球天才，其中八村壘就在本賽季獲得了第九順位加盟了奇才，此外他的弟弟八村阿蓮也展現了籃球方面的天賦，尤其是在日本的高中聯賽上，他還斬獲44+14的數據，可以說他也將成為中國男籃未來的勁敵。

八村的親弟 99年生人，與八村壘同是貝南和日本混血，當前已經大二，傳聞這一年在東海大學進步明顯。作為一名裸足只有198CM的PF，他的近框終結和中投以及掩護都具備一定水準，雖說與他哥哥相比仍有一定差距，但終歸是日本未來國家隊重點培養的人物。

如今的日本籃球里，先後有了渡邊雄大和八村壘的成功例子，而八村阿蓮也展現了一定的天賦，恐怕在未來的亞洲賽事上，一旦這三名球員同時參與比賽，中國未來十年之內恐怕要處於下風。

要知道日本在培養體育人才方面向來擅長突破一個成功的個例後大規模複製，在足球方面中田英壽闖蕩歐洲成功後，後續有大量的日本球員在歐洲球隊站穩腳跟；乒乓球自從伊藤美誠出現之後，青年女乒又湧現出長崎美柚和木原美悠等潛力新人，而這兩位日本女乒運動員更是剛剛結束的泰國世青賽上斬獲冠軍，而中國女子乒乓球在本屆比賽里卻顆粒無收。

如今日本在籃球方面也找到了培養的案例，恐怕在八村阿蓮之後，還會湧現出類似的人物，反觀中國即使偶爾有姚明、孫繼海、易建聯這樣的成功案例，但是往往成為後繼無人的孤例，或許這樣的差距在未來將更為明顯，曾一度統治亞洲的男籃地位或許在未來十年拱手讓人。

而這不僅僅是籃球一方面的問題，連同乒乓球以及羽毛球這種傳統優勢項目同樣處於這樣的憂慮之中，可見如今的中國體育陷入培養人才的瓶頸，或許這才是中國體育目前的最大短板。
