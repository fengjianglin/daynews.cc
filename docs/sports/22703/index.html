李鐵：球員的表現令人滿意 自己的未來尚無定數

2019-12-19 523條評論 2,358次閱讀 976人點贊

新華社韓國釜山12月18日電（記者陸睿 耿學鵬）率領中國國家男足選拔隊收穫2019年東亞杯季軍的李鐵18日在韓國釜山說，自己對球員在本屆賽事中的表現很滿意；至於自己接下來是否會帶領中國隊踢世界盃預選賽，他表示還「無法預料」。

12月18日，中國國家男足選拔隊主教練李鐵在場邊指導。當日，在韓國釜山舉行的2019東亞杯男足比賽中，中國國家男足選拔隊以2比0戰勝中國香港隊。新華社記者王婧嬙攝

「未來的事情沒有人能預料，我只能把每一天的工作做好。」李鐵說，「我和我的團隊每天都非常辛苦地準備東亞杯比賽，這些辛苦得到了回報，球員在球場上好的表現就是最大的回報。」

談及首次作為國家選拔隊主教練率隊參賽，李鐵說，這圓了他兒時非常重要的一個夢想，自己感到格外榮幸和驕傲。「我會去跟我的孩子講，爸爸實現了自己一個長久以來的夢想。」

對於球員們的表現，李鐵說，隊員們在剛剛經歷了漫長的中超賽季、甚至沒有一起踢過一場公開賽的情況下就來到韓國參加東亞杯，並盡了最大努力去拼搏，這十分不容易。同時，球隊通過這三場比賽學習到很多，積累了更多寶貴經驗。
