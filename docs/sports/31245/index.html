2020年亞俱杯落戶中國！北汽女排代表參賽，可向郎平要人

2019-12-25 107條評論 1,011次閱讀 55人點贊

2020年女排亞俱杯將於明年5月2日至9日在中國天津舉行，這是繼今年女排亞俱杯後，該項賽事又落戶中國。亞洲女排排球俱樂部錦標賽（簡稱亞俱杯）是亞排聯於1999年創辦，除2003年外每年舉辦一屆，獲得冠軍的隊伍將有資格參加同年的女排世俱杯比賽。 2019年天津女排獲得女排亞俱杯冠軍，自動獲得今年女排世俱杯的參賽資格。北汽女排是2018-2019賽季中國排球超級聯賽女排冠軍，根據賽制將代表中國俱樂部參加2020年女排亞俱杯。若想出現在明年世俱杯的賽場，就必須獲得明年亞俱杯冠軍。

北汽女排作為排超聯賽衛冕冠軍，本賽季已經順利晉級聯賽四強。球隊保留了部分冠軍成員，二傳位置引進了山東女排主二傳、2017年女排世青賽冠軍二傳蔡雅倩。由於本賽季聯賽四強賽不允許二次轉會，所以北汽女排在聯賽開賽前就引進三名實力外援，分別是美國女排主攻巴奇、美國女排副攻迪克森和澳大利亞重炮接應魯爾克。其中迪克森和魯爾克本賽季選擇續約，巴奇是球隊從上賽季歐冠冠軍、意甲聯賽亞軍諾瓦拉俱樂部挖來的主攻，和奧運冠軍劉曉彤搭檔成為北汽女排新的主攻組合，老將曾春蕾本賽季是球隊主攻和接應位置的雙料隊員。

受制於只允許兩名外援同時上場的規定，北汽女排三名外援無法同時上場，這對球隊的整體實力帶來一定影響。二傳蔡雅倩是球隊唯一的主二傳，也是具備潛力的年輕二傳，三名外援也都是有實力的隊員，即便是半決賽面對擁有朱婷的天津女排，北汽女排也具備與對手抗衡的實力。

不過四人與北汽女排的合約只到本賽季聯賽結束，若沒有與球隊續約，明年的亞俱杯是無法同球隊一起出戰。以北汽女排目前的本土實力看，戰勝泰國的春武里和日本的久光製藥這樣的老牌俱樂部存在困難。若想解決球隊人員緊缺的困境，北汽女排或可學天津女排向國家隊主教練郎平要人。

今年女排亞俱杯，天津女排臨時轉會了接應龔翔宇和主攻劉曉彤，同時召回了在國家隊集訓的姚迪、李盈瑩和王媛媛三人，最終獲得冠軍。北汽女排目前在二傳、主攻、副攻和自由人位置存在明顯的缺角，要想通過亞俱杯奪冠直通世俱杯，向郎平要人或是最好的選擇。
