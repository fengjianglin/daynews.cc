爆冷！世錦賽3冠王4-5遭大逆轉出局，丁俊暉手下敗將爆冷險勝進32強

2019-12-24 485條評論 2,208次閱讀 822人點贊

北京時間12月23日凌晨，2020年斯諾克德國大師賽資格賽結束了第三個比賽日的角逐，在這次角逐中，作為世錦賽的3冠王，塞爾比在面對丁俊暉手下敗將克拉克時爆出大冷門，他是以4-5慘遭大逆轉出局，而克拉克則是爆冷險勝挺進32強。

眾所周知，塞爾比是英國斯諾克的王牌球星，他很早就在世界賽場成名，2008年，塞爾比首次獲得斯諾克大師賽冠軍，摘得職業生涯第一座世界冠軍獎盃。而在2014-2017年間，他又在四年內三次奪得世界斯諾克錦標賽冠軍，成為世錦賽的3冠王。

近幾年雖然整體實力相比巔峰時期有所下滑，但也是頂級球手不可忽視的對手，剛剛結束的歐洲大師賽資格賽，塞爾比就先後淘汰達赫迪和吉米懷特挺進正賽，此番來到德國大師賽資格賽，他同樣發揮不錯，首輪比賽，他是以5-0橫掃范爭一晉級。

而克拉克則是斯諾克的小將，他在世界賽場並沒有取得什麼好成績，目前排在世界第95位，與中國斯諾克球員交手也毫無競爭力，曾多次輸給丁俊暉。今天的比賽，兩位球員直接交鋒，本來按照雙方的整體實力和狀態來看，外界十分看好塞爾比贏球晉級，然而他卻意外爆出大冷門。

本場比賽，塞爾比剛開始的狀態還是不錯的，第一局，他雖然遇到了一些困難，但還是以53-44率先贏下第一局，第二局，他迎來爆發單桿轟出100分，最終以100-0再贏一局，取得2-0的領先優勢。第三局和第四局，隨著克拉克積極調整狀態，塞爾比開始陷入被動，最終他連輸兩局，比分被扳至2-2平。

隨後進行了三局比賽，雖然塞爾比通過調整拿下兩局，率先取得局點，但由於在第八局和第九局的手感不太好，最終他是以49-66,34-66連輸兩局，以4-5遭大逆轉出局，而克拉克則是爆冷險勝挺進32強。
