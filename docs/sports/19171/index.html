歐聯杯淘汰賽對陣：西班牙人遭遇英超隊 曼聯阿森納國米皆遇好籤

2019-12-16 424條評論 2,527次閱讀 1591人點贊

北京時間12月16日，2019-20賽季歐聯杯32強淘汰賽對陣抽籤儀式在瑞士尼翁的歐足聯總部進行。最終，武磊所在的西班牙人遭遇英超的中資球隊狼隊，阿森納抽中奧林匹亞科斯，曼聯對陣布魯日，國米的對手是盧多戈雷茨，三支豪門抽到的都是非五大聯賽的對手。

與歐冠抽籤相同，歐聯杯採取種子隊（小組第一球隊）對陣非種子隊（小組第二球場）的規則。同小組的球隊迴避，同足協的球隊迴避，此外烏克蘭的球隊與俄羅斯的球隊迴避。

本賽季歐冠32強中種子隊包括阿賈克斯（荷蘭）、阿森納（英格蘭）、巴塞爾（瑞士）、本菲卡（葡萄牙）、凱爾特人（蘇格蘭）、西班牙人（西班牙）、國際米蘭（義大利）、伊斯坦布爾（土耳其）、根特（比利時）、林茨（奧地利）、曼聯（英格蘭）、馬爾默（瑞典）、波爾圖（葡萄牙）、薩爾茨堡紅牛（奧地利）、塞維利亞（西班牙）、布拉加（葡萄牙）。

非種子隊有希臘人競技（塞普勒斯）、阿爾克馬爾（荷蘭）、布魯日（比利時）、克魯日（羅馬尼亞）、哥本哈根（丹麥）、赫塔費（西班牙）、勒沃庫森（德國）、奧林匹亞科斯（希臘）、頓涅茨克礦工（烏克蘭）、葡萄牙體育（葡萄牙）、法蘭克福（德國）、沃爾夫斯堡（德國）、格拉斯哥流浪者（蘇格蘭）、羅馬（義大利）、狼隊（英格蘭）、盧多戈雷茨（保加利亞）。

歐聯杯的抽籤嘉賓也有一位女足球員，她是四次獲得女足歐冠的海寧。另外一個抽籤嘉賓是曾獲得過歐聯杯冠軍，並在中超的北京國安效力過的前鋒卡努特。
