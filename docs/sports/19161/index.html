易建聯一番無奈話, 道出曾繁日被杜鋒棄用原因, 年輕人長點心吧

2019-12-17 880條評論 2,137次閱讀 1138人點贊

賽季至今，廣東已經取得15勝1負的戰績，16場比賽，僅僅只是輸掉1場，相信很多球迷都沒想到吧，沒看過廣東比賽的球迷可能會覺得，廣東依舊是冠軍最有利爭奪者，每一場球都輕輕鬆鬆贏下對手，事實上，廣東並沒有大家說的那麼可怕，雖然贏的比賽很多，但是廣東真的是每一場都拼盡全力，特別是易建聯，我真的很心疼阿聯，在國家隊的時候，他要大殺四方，回來CBA，阿聯還是打得非常累，理由是什麼，大家應該都知道，廣東今年常規賽沒了周隊，這是一個很致命的殺害。

周隊是廣東衝擊總冠軍必備的球員，沒有他，廣東季後賽很難跟新疆拼，周隊是一個攻防兩端都可以做貢獻的球員，他不在，阿聯需要在防守端做更多的貢獻，這也是杜指導一直用阿聯的原因，很多人吐槽杜鋒死用阿聯，好吧，如果廣東其他人可以站出來，阿聯也不至於那麼辛苦，上一場比賽，有一個細節不知道大家有沒有注意，廣東在一個回合丟掉籃板後，阿聯轉身對著面前的杜潤旺說：籃板不用你搶嗎？簡簡單單的一句話，卻把阿聯的無奈表達得淋漓盡致，說真的，換作其他人，早就爆粗口了。

杜潤旺也好，曾繁日也好，這些年輕人真的很幸運，他們遇到一個好領袖，阿聯是一個有擔當的人，他不會輕易去罵自己的隊友，更多的是鼓勵，當然，像杜潤旺這種常常犯低級錯誤的年輕球員，阿聯是不會跟你客氣，智慧之前就說過，阿聯和杜潤旺組成內線搭檔，都是在考驗阿聯，因為杜潤旺像無頭蒼蠅到處跑最後板不搶，然而我被很多球迷吐槽了，現在看來，我說的話沒錯，年輕人真的要長點心，不要什麼東西都要依賴老隊員，另外，最近我看到很多人為曾繁日打抱不平，一個個追著杜鋒一頓吐槽。

說實話，曾繁日上場作用，杜鋒比任何人都要清楚，不上他，自然有他的理由，我個人看球也可以知道，曾繁日問題很多，他喜歡無謂犯規，防守也走神，而且漏人漏籃板，甚至不搶，有時候，杜鋒不是不給機會，但是你每次上場都犯同樣的錯誤，其它方面也沒什麼長進，杜鋒罵多了自己也煩，所以呢，大家還是別怪杜鋒不給阿日時間，他說過一句話特別好，我覺得杜鋒說過一句話特別在理，很多年輕球員得到上場機會很大的原因是因為他們是年輕，而不是已經具備在場上那麼久的能力。

每一個人都應該清楚知道這一點，不單單是廣東隊的年輕人，還有其他隊的年輕人也是一樣，作為職業球員，千萬不要飄。

美哪深夜發朋友圈，爆出來了一個震撼的內幕！希望與大家在裡面邂逅！
