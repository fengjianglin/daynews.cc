避免波叔式錯誤，是穆帥征服熱刺的隱藏原因，溫格早就隱晦質疑過

2019-12-15 631條評論 3,025次閱讀 1791人點贊

雖然已經離開白鹿巷有些時日了，可關於波切蒂諾下課原因的分析，卻始終不絕於耳。毋庸諱言，波切蒂諾的遭遇從最純粹的足球領域是無法解讀的。他的球隊前一個賽季還能一路連克強敵殺入歐冠決賽，卻在沒有核心球員離隊且個別位置還有所補強的大前提之下，在新賽季沉沙折戟，接二連三的慘敗就彷彿是突然換了一支球隊一般。

波切蒂諾鮮為人知的軟肋

在穆里尼奧入主白鹿巷之下，還有許多聲音把問題歸咎在了這支球隊頗為讓人詬病的薪金體系之上，認為正是列維的太過摳門，讓球員們對於自己的薪酬極其不滿意，因此而生出了異樣的心思。於是，大家的心思不在球場之上，球隊的表現也就隨之一落千丈。

但是，這種觀點顯然是經不起推敲的。這其中的邏輯就在於，在穆里尼奧入主白鹿巷之後，這支球隊的薪酬體系並沒有發生明顯的調整，可球員們的精氣神卻比起波切蒂諾時代要好出了不少。背景沒有變，可對比卻是如此之鮮明，這顯然不是此前的薪金體系不合理就可以解釋的。

一言以蔽之，熱刺球員在賽季初期的糟糕表現，其實並不是沖著球隊去的。否則，這種問題絕對不是換帥就可以解決的了。

如果順著這個思路思考下去，最大的嫌疑，反而出在了波切蒂諾身上，一如昔日萊斯特城球員對待拉涅利的態度。雖然這種風氣絕對不值得鼓勵，可在當代足壇，這差不多已經成為了一種常態。

作為證據的則是，除了那讓人難以理解的場上表現還有糟糕戰績之外，在波切蒂諾下課之初，英國足球名宿克勞奇與老雷德克納普就言辭激烈地指責了熱刺球員，認為他們要對主教練的下課負責任。這其實就已經是在暗示，乃至是明示，波切蒂諾之所以會下課，其實是被麾下的球員們給做掉了。

那麼問題也就隨之而來了，熱刺球員為何會如此對待波切蒂諾？要知道，凡事總是會有著它的緣由的，哪怕就是惡事也同樣如此。

而這，就涉及到波切蒂諾鮮為人知的軟肋了。貌似性情平和的他，其實是一名反傳統的教練，與那些把更衣室的核心秘密死死藏在肚子里的老派教頭不同，波切蒂諾在這方面頗為激進，他選擇了公開、透明的方式。

這直接體現他在2017年時出版的自傳《勇敢新世界》一書之中，披露了許多熱刺內部不為人知的秘聞，還有他與球員之間談話的具體細節，在當時就引發過極大的爭議。

溫格早就隱晦質疑過

比如在波切蒂諾的書中，就有這樣一個極具爭議的片段，波叔直言不諱地表示沃克早在加盟曼城之前，他的心思就已經不在熱刺了，並且在聯賽尾聲的關鍵階段對於球隊缺乏足夠的尊重還要求離開球隊。與此同時，波叔還披露了他與沃克之間的對話細節。

哪怕都已經投入了瓜迪奧拉的麾下，可這仍讓身為當事人的沃克怒不可遏，除了竭力反駁波切蒂諾在自傳之中披露的消息之外，沃克還坦言波切蒂諾把他傷得很深。

這裡必須說明的是，作為英格蘭本土球員，沃克在熱刺更衣室中頗具分量，與不少球員都私交甚篤，遠不是那些「臨時工」或者是「僱傭兵」可以相提並論的。波切蒂諾公開披露他與沃克之間的對話，不僅犯下了更衣室之中的忌諱，也很容易讓其他熱刺球員生出兔死狐悲之感。

而波切蒂諾這本披露性質的自傳，帶來的更直觀的後果則是：在這本自傳出版發行之後，熱刺隊在接下來的5場聯賽之中僅僅只拿到了1場勝利而已。

雖然任誰也沒辦法給出最直接的證據證明，波切蒂諾這本披露性質自傳的出版，究竟給熱刺更衣室造成了怎麼樣的負面影響。但是，人人心中都自有一桿秤，球隊戰績的直線下滑也就是擺在眼前的事實。

另外，在波切蒂諾的管理下，熱刺更衣室也確實出現了大問題。特里皮爾在今年夏天離開了這支隊伍，遠赴西班牙的馬德里競技。為此，這位英格蘭本土球員解釋道：「當然我絕對不是非要離開熱刺不可，但是熱刺內部在上賽季是一片混亂的狀態，對此我不能說得太過具體了，但是我在那種環境之中待不下去了，所以我只是想換個環境。」

特里皮爾的話，無疑是讓人玩味的。一方面，他揭露了熱刺更衣室確實有問題的事實。另一方面，他則拒絕詳細說出更衣室內究竟有什麼問題。這與波切蒂諾把話說透的方式，形成了鮮明的對比。

不妨這樣想想吧，就是這樣一位因為隊內環境不好而離隊的球員，都拒絕把更衣室之中的事情具體披露出去。那任期之內的波切蒂諾，他把更衣室之中的事情公開、透明，是不是值得商榷了？

事實上，除了熱刺球員對波切蒂諾的這種方式不滿意之外，就連老帥溫格都隱晦地指出過波切蒂諾的問題。

就在波切蒂諾推出自傳沒過多久，溫格就直言不諱地道出了自己對於此事的看法：我不喜歡說話，也不想說出事實真相。尤其是當你還在工作崗位上的時候，你就不能如實說出究竟發生了一些什麼事情，難道僅僅只是為了賣書？

穆里尼奧竭力避免

雖然說，世人都一直在提倡所謂的「公開、透明」。但是，任何一個行業都有自己的規矩，行內人就必須要去遵守。可能，在多年之後，足球更衣室也會變得公開與透明，大家暢所欲言。但是，起碼在這個大環境之下，無論是誰想挑戰這個傳統，都註定會碰到頭破血流。

除了傳統之外，其實不妨站在熱刺球員的角度上想想，又有誰會願意麵對這樣一位會把自己私密談話給公開出去的教練？可能一個不留神，就會登上各大媒體的頭條，就像是昔日的沃克一樣成為眾矢之的。

於是，波切蒂諾的悲劇也就在所難免了。以他這種駕馭更衣室的手段，在自傳出版之後還能熬上這麼久才徹底爆發，其實就已經頗為出人意料了。

至於波切蒂諾的繼任者穆里尼奧，自從出道起就一直在竭力避免著類似的錯誤。無論是皇馬、切爾西還是曼聯，穆里尼奧都有足夠的談資，無論是出書也好，還是接受訪談也罷，只要穆帥願意披露其中的細節，相信他絕對可以賺到盆滿缽滿。但是，又有誰見過穆里尼奧會把更衣室之中的秘聞公開化？尤其是涉及到球員不利信息的？

也正是因為波叔以往的做派，穆里尼奧才能如此之快的征服白鹿巷。避免波叔式錯誤，正是穆帥征服熱刺的隱藏原因。

不妨看看穆里尼奧的做法吧，哪怕就是在談到差不多已經確定離隊的埃里克森之時，穆里尼奧都在竭力維護著他與隊友之間的秘密：

「我只是知道他的合同將在本賽季結束的時候到期而已，而我跟埃里克森之間說了什麼，是我不願意公開的私人談話。他對我很誠實，我對他同樣也很誠實，我們的關係真的很棒。我們從第一天就有著很棒的關係，我是不會背叛這種信任的。球隊跟埃里克森知道我是俱樂部的重要一部分，我的工作會在內部分享這段親密的對話，但是我絕對不會公開的。埃里克森本人將決定他是否願意公開這次談話。」

穆里尼奧的話里話外，除了指出了他不會公開與埃里克森的對話之外，還把這種不經過球員同意，就公開私密消息的做派定義為了「背叛信任」。可想而知，穆里尼奧與波切蒂諾之間，究竟有多大的不同。

而這，可能也正是穆里尼奧與波切蒂諾之間的最大不同之處。
