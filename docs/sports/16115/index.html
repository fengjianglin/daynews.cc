中國市場很重要？聯盟開拓新市場，引入墨西哥球隊，蕭華又賺發了

2019-12-13 582條評論 2,091次閱讀 154人點贊

12月13日，NBA官方宣布，他們將繼續執行國際擴張計劃，將於2020-21年在墨西哥城組建一支NBA發展聯盟球隊。蕭華在新聞發布會中宣布，原本成立於2016年的墨西哥聯賽球隊Capitanes在下賽季將會加盟G聯盟。

目前，G聯盟一共擁有28支球隊，其中只有開拓者和掘金沒有自己的附屬球隊。而Capitanes將會成為發展聯盟第一支沒有任何NBA背景的球隊，這是個歷史性的時刻。他們將與NBA簽署5年的初始協議，探索除去美國和加拿大之外的籃球市場。

在新聞發布會上，蕭華春光滿面地說道，「將一支G聯盟球隊帶到墨西哥城是NBA的歷史上里程碑，它證明了我們對墨西哥和整個拉丁美洲籃球迷的承諾。作為一支總部位於美國和加拿大之外的G聯盟球隊，我們期待著Capitanes加入NBA大家庭。」

Capitanes曾經是兩屆西甲巴隆塞斯托職業聯賽的亞軍，進入過美洲聯賽四強挑戰賽。在2019年9月份，他們還參加了NBA G聯盟的國際挑戰賽。NBA G聯盟主席謝里夫-阿卜杜勒-拉希姆補充說，「NBA G聯盟一直致力於在拉丁美洲和國際上發展籃球運動。如今，我們在墨西哥城擁有一支球隊，踏出了實現這一願景的重要一步，我很榮幸地歡迎Capitanes來到G聯賽，並期待著下賽季看到球隊在巴雷拉的比賽。」

對於普通球迷來說，G聯盟增加一支球隊並沒有太大的變化，但作為替NBA引入潛力天賦，不斷輸送新鮮血液的聯盟，這肯定只是他們的第一步。如果將來NBA可以成功開闢墨西哥市場，甚至帶動拉丁美洲的籃球氛圍，這對NBA未來的發展將會有重大意義。

​由於種種原因，NBA在中國的市場遭遇了一些打擊，但中國球迷也不用盲目自信。至少，在聯盟管理層和蕭華的心中，中國市場的損失還沒有達到傷筋動骨的程度。我們在意的所謂尊重和驕傲，也許在別人心中，根本就一文不值。如今，聯盟和蕭華又要賺發了，讓人感到有些懊惱。
