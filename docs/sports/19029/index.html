88分鐘絕殺！東亞杯冠軍產生：日本女足轟13球0失球登頂，中國第3

2019-12-18 2,087條評論 3,386次閱讀 2308人點贊

北京時間12月17日，2019年東亞杯女子比賽全部結束，最終日本隊1-0絕殺韓國隊，以3連勝積9分的成績獲得了本次比賽的冠軍，第3次拿下東亞杯，而韓國隊依靠1勝1平1負積4分獲得了亞軍。中國隊與中國台北分居第三和第四名。

本場比賽前的形勢非常明顯，日本隊2連勝，打入12球，沒有丟1球，只要一場平局就能夠確保獲得本次比賽的冠軍。韓國隊1勝1平積4分，由於沒有拿下中國隊，因此想要獲得冠軍，需要在直接交鋒中擊敗日本隊。這樣的難度可想而知！韓國隊上一次也是唯一一次奪冠要追溯到2005年了，日本隊則在2008和2010年2次奪冠，如果再次奪冠，有望成為東亞杯最成功的隊伍。

從與中國以及中國台北的比賽來看，日韓女足的差距還是很明顯的，日本隊能夠吊打中國台北一個9-0，韓國隊只能打中國台北3-0，日本隊能夠3-0輕鬆擊敗中國隊，而韓國隊與中國隊水平差不多，打成0-0平。因此此役日本隊應該說佔據絕對優勢。當然，中國隊想要獲得第二名，需要一個前提，那就是韓國隊輸給日本隊5球以上。

不過從過程來看，主場作戰的韓國隊踢出了風采，給日本隊製造了不小的麻煩，日本隊最終面對韓國隊直到最後時刻才獲得了一粒點球，籾木結花罰入自己創造的點球，這樣一來，日本女足1-0絕殺韓國，拿下3連勝！2019年東亞杯女子的比賽已經全部結束了。日本隊以3連勝積9分的成績獲得了本次比賽的冠軍，作為亞洲的頂級強隊，日本女足本次比賽發揮比較完美，沒有丟掉1球，打入13球。韓國隊基本可以交差，因為力壓了同檔次的中國隊獲得第二。中國女足的表現非常不能讓人滿意，3場比賽雖然只輸掉1場，但0-3不敵日本隊的比賽堪稱完敗，至於戰平韓國隊，場面沒有任何主動，即便擊敗中國台北的比賽，中國女足的發揮也很糟糕。

本次東亞杯雖然不是什麼大賽，但對於中國女足來說，應該感受到壓力了。追趕日本女足是沒有任何希望了，即便是韓國女足目前的發展似乎也比咱們好，到了奧預賽，想要出線難度可想而知！
