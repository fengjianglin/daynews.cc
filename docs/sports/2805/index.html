埃弗頓+麥克迪恩+阿特金森，利物浦球迷心慌慌

2019-12-04 0條評論 22次閱讀 0人點贊

摘要

北京時間12月5日凌晨4點15分，19/20賽季英超聯賽第15輪將迎來利物浦對埃弗頓的默西塞德德比，默西塞德德比歷來火爆異常，不知道本場比賽雙方又會碰撞出什麼火花。目前利物浦聯賽13勝1平保持不敗，以8分優勢領跑積分榜。

北京時間12月5日凌晨4點15分，19/20賽季英超聯賽第15輪將迎來利物浦對埃弗頓的默西塞德德比，默西塞德德比歷來火爆異常，不知道本場比賽雙方又會碰撞出什麼火花。

目前利物浦聯賽13勝1平保持不敗，以8分優勢領跑積分榜。但是受困於密集賽程的影響，紅軍在最近的比賽中統治力下降，總是讓球迷們速效救心丸不離手。在與那不勒斯的歐冠比賽中球隊主力後腰法比尼奧受傷，將缺席19年剩餘的比賽；而在與布萊頓的比賽中，阿利松禁區外手球被紅牌罰下，本場比賽也無法出場。兩大核心球員的缺陣，勢必影響利物浦的戰鬥力。

埃弗頓這邊本賽季狀態糟糕，目前以4勝2平8負積14分的戰績排名17，僅比降級區高出2分，主教練馬爾科-席爾瓦帥位搖搖欲墜。在上一輪聯賽中，埃弗頓險些掀翻萊斯特城，但是在補時階段被伊希納喬絕殺，而這也是埃弗頓本賽季第4次在補時階段丟球，而他們的對手利物浦本賽季則是多次在最後時刻絕殺對手，所以本場比賽埃弗頓凶多吉少。本場比賽若埃弗頓無法在利物浦身上拿分，而聖徒南安普頓又能在主場取勝諾維奇的話，埃弗頓將滑落至降級區。

從目前雙方的處境來看，本場比賽紅軍主場拿下埃弗頓貌似沒有太大問題，可是根據以往的經驗來看，不管埃弗頓狀態如何，總能給利物浦製造麻煩。

首先說一下奧里吉與埃弗頓的故事，故事發生在2016年4月21日的默西塞德郡德比中，在那場比賽前，奧里吉作為紅軍的當紅小生，在渣叔的調教下正冉冉升起，紅軍球迷對奧里吉充滿了期待。可不幸的事情發生了，奧里吉被莫里惡意鏟傷，結束了其良好的上升勢頭。傷愈後的奧里吉再也打不出以前的狀態，險些被渣叔賣到狼堡，紅軍球迷對莫里恨之入骨。

在上賽季英超聯賽第14輪，紅軍利物浦與埃弗頓在聯賽中再次狹路相逢，而比賽場面一如既往的焦灼，一直到傷停補時階段場上比分仍然是0比0。在96分鐘時范戴克一腳射門打了飛機，但是萬萬沒想到球卻鬼使神差般的被皮克福德托到了橫樑上而反彈到奧里吉面前，奧里吉抓住機會頭槌破門，幫助利物浦拿到三分。奧里吉這個進球正應了小馬哥那句經典的台詞：「我等了三年，我不是要證明我比別人強，我只是想說，失去的東西我一定要拿回來！」經過965天的等待，奧里吉在這個曾經跌倒的對手身上，站了起來。而後在歐冠比賽中，奧里吉再次大發神威幫助紅軍逆轉巴薩、戰勝熱刺拿到歐冠獎盃。平心而論雖然奧里吉有錦鯉屬性，但是他並沒有達到紅軍球迷們在其受傷前所期待的高度，而莫里的那次惡意鏟球是罪魁禍首。

而上賽季第29輪的比賽，埃弗頓更是逼平了利物浦，讓利物浦失去了榜首的位置，並最終錯失冠軍。在那場比賽前，紅軍利物浦領先曼城1分，帶著這1分的領先優勢利物浦客場挑戰埃弗頓。經過90分鐘的鏖戰，埃弗頓沒有讓利物浦佔到便宜，在首回合出現低級失誤的門神皮克福德表現神勇，力保球門不失，最終雙方戰成0比0。那場比賽過後，紅軍積70分，落後曼城1分，雖然在聯賽剩餘的比賽中，紅軍拿滿了27分，但曼城也保持全勝，最終紅軍以1分之差遺憾地錯失聯賽冠軍。

而令紅軍球迷更加恐慌的是本場比賽的裁判安排，本場比賽主裁判和視頻助理裁判是麥克-迪恩和阿特金森。這對裁判一出場紅軍將士們還是要小心一點了，尤其是阿特金斯除了對曼聯那場比賽的爭議判罰，最近又接連在利物浦身上發明了腋窩越位和自由球快吹兩個足球界專有名詞，本場比賽不知道裁判又會出什麼幺蛾子了。遇上這倆裁判，紅軍上下表示心慌慌。
