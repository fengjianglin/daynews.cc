皇馬突遭重創！曝拉莫斯帶傷死磕巴薩 恐無緣下一場生死戰

2019-12-19 2,768條評論 4,736次閱讀 1083人點贊

北京時間12月19日凌晨3點，2019-2020賽季西甲首回合國家德比落下帷幕，在93426名球迷的注視下，皇馬客場0-0戰平巴薩拿下1分，各項賽事連續12場保持不敗。

本場比賽，33歲的拉莫斯完成第43次出戰國家德比的壯舉，超越了哈維，梅西，小桑奇斯和亨托4大傳奇。然而，拉莫斯卻高興不起來，科貝電台透露他帶著腳踝傷勢堅持死磕巴薩，4天之後與畢爾巴鄂競技的硬仗，很可能上不了。

與巴薩的較量，是拉莫斯第43次出戰國家德比，超過了梅西，哈維和小桑奇斯，排名現役第一！本場比賽，皇馬隊長繼續搭檔瓦拉內鎮守防線，全場比賽他與貝爾，卡塞米羅和F-巴爾韋德一起限制梅西。

在拉莫斯領銜的防線面前，梅西罕見地遭遇了低谷，全場比賽僅僅完成2次射門，並只有1次射正，甚至連進入禁區都困難，足以看出皇馬的防守是多麼成功。

不過，為了零封巴薩，皇馬付出了巨大代價，瓦拉內被對方蹬踏，大腿被划出幾道血痕，而拉莫斯更是遭到腳踝扭傷的打擊。

西媒科貝電台透露，拉莫斯在上半場就扭傷了自己的腳踝，但是他深知國家德比的重要性，因此帶傷死磕巴薩，堅持踢完全場比賽。數據顯示，拉莫斯全場完成2次解圍，1次關鍵門線救險以及2次斷球，而犯規次數只有1次，發揮出了極高水準。

但是，這也讓拉莫斯的傷勢加重，這意味著他未必能趕得上4天之後與畢爾巴鄂競技的關鍵戰。熟悉西甲的球迷都知道，畢爾巴鄂競技是西甲出了名的鐵軍，球風極為硬朗，本賽季他們就曾1-0爆冷巴薩，是強隊都不願意遇到的硬骨頭。

一旦拉莫斯無法出戰，皇馬只能冒險派上年輕的米利唐，這等於給齊達內開難題。所以，皇馬雖然從諾坎普全身而退，但他們沒有任何放鬆的資本，缺少拉莫斯的情況下碰上畢爾巴鄂競技，無異於又是一場生死戰。
