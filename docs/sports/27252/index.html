前鋒改踢後衛，21歲多特小將創德甲最快速度！皇馬未來兩翼齊飛

2019-12-21 136條評論 1,554次閱讀 601人點贊

如果要評選最近德甲賽場上最火的球員的話，這位摩洛哥小將一定榜上有名——被皇馬外租到多特的阿什拉夫-哈基米(Achraf Hakimi)在德甲賽場發揮十分出色。在效力多特的近50場比賽中，身為邊後衛的阿什拉夫打進了9粒進球。本賽季，他在歐冠已經攻入4球，攻擊力十足，已然達到世界級水準。

據德甲官網消息，多特蒙德小將阿什拉夫在對陣萊比錫的比賽中跑出了36.2km/h的速度，創下了2011/12賽季有數據統計以來的德甲最高時速。

皇馬主力右邊後衛一直是卡瓦哈爾，而在皇馬官宣簽下新的邊後衛奧德里奧索拉之後不到一星期，阿什拉夫就被租借給了多特蒙德以尋找出場機會。本賽季是阿什拉夫租借多特的第二個賽季，就如同卡瓦哈爾當年租借到勒沃庫森一般，阿什拉夫也在這兩年中也獲得了非常大的提升。

憑藉最近出色的表現，摩洛哥人入圍了「年度非洲青年球員」最終五人名單，以及2019年非洲足球先生的候選名單。1月7日，非洲足球先生結果將揭曉。除阿什拉夫外，前德甲球星奧巴梅楊和納比-凱塔都入圍了名單，奧西姆亨，以及舒波莫廷都入圍了名單，他們都是前德甲球星。儘管得獎希望非常渺茫，但德甲官網特別報道了這件事，為阿什拉夫鼓氣。

「大黃蜂」多特一度是擅長以打快速反擊而聞名的球隊，所以隊內自然也少不了頂級的速度好手。前段時間德國足球媒體《踢球者》雜誌統計了多特蒙德隊內球員的最高時速，排名第一的既不是英格蘭快馬桑喬和德國國腳邊後衛的舒爾茨，而是阿什拉夫！

摩洛哥小將的最高時速竟然達到了驚人的35.03km/h，排名多特蒙德隊內最高，排在第二和第三的才分別是舒爾茨和桑喬。這一速度也堪比皇馬隊內的最快的貝爾（貝爾為36.9km/h）。沒過多久，阿什拉夫在對陣萊比錫的比賽中跑出了36.2km/h的速度，又一次刷新了自己的紀錄。創下了2011/12賽季有數據統計以來的德甲最高時速。

在面對媒體的採訪時，談到了有關自己未來的問題。同時，阿什拉夫表示，他小時候是踢前鋒的，但之後教練把他改造成了邊後衛。

阿什拉夫提到：「在我小時候，我一開始其實是踢的前鋒，當我到了青年隊的時候，我的教練又把我調整到了邊鋒的位置上。」「在那之後，教練又把我改造成了邊後衛，但是我想說，我現在仍有一顆前鋒的心，雖然我的位置是邊後衛，但是我依舊能夠在進攻端幫助球隊，就像我之前在對陣國米的比賽中所做的那樣。」

在之前歐冠中梅開二度導演多特蒙德逆轉國際米蘭更是阿什拉夫的代表作。

《馬卡報》消息稱，在以租借形式加盟多特兩年之後，阿什拉夫將會在明年的6月回到皇馬。按照合同，本賽季結束後，阿什拉夫便將結束兩年的租借期。由於在效力多特蒙德期間出色的表現，伯納烏人自然對摩洛哥人充滿期待。屆時包括卡瓦哈爾和奧德里奧索拉等人都將面臨與他的競爭。

阿什拉夫還談到了當他自己離開皇馬時，齊達內對他說的話。阿什拉夫表示：「齊達內當時認為我需要更多的出場時間，我需要通過比賽去積累經驗。」「我也知道，自己還年輕，而皇馬也不是我的職業盡頭，所以我還可以在其他俱樂部享受足球。我在多特學到了很多東西，成長了許多。」

摩洛哥人不僅能踢邊後衛，也能踢邊鋒。阿什拉夫回歸皇馬就像當年的厄德高一樣，主要問題是球員以什麼條件回歸，是否能夠確保自己在隊內的位置。如果阿什拉夫回歸，這意味著奧德里奧索拉甚至是貝爾和巴斯克斯等人的未來都將陷入疑問。

「未來是否會回到皇馬？為什麼不呢？只要我保持現在的水平，這種可能性是存在的。」
