狂輸35分！CBA洋帥說為保存實力，曾有先例，打廣東讓主帥+2外援

2019-12-18 1,786條評論 3,960次閱讀 2442人點贊

108-143，江蘇男籃主場狂輸廣東宏遠35分，然而無論是哪支球隊的球迷，似乎在第一節比賽時候都沒有想到這個結果，因為首節比賽江蘇男籃可39-33領先6分的。江蘇男籃小外援吉布森首節就拿到了15分，趙睿、胡明軒、徐傑等後衛被吉布森一人打爆，然而不曾想第二節比賽開始江蘇男籃兵敗如山倒，單節19-50狂輸31分後讓比賽懸念終結，比賽早早陷入了尷尬局面。

賽後採訪杜鋒並不是很滿意，尤其是首節丟掉39分是廣東宏遠這賽季第一次出現開頭被對方打得如此狼狽。不過整體實力畢竟廣東宏遠優勢較大，相比上一場意外不敵北京北控，杜鋒說防守端大家表現好了一些，能更加專註投入到比賽中，幾個主力在場時候球權的分享做得不錯。

相比之下江蘇男籃主帥貝西洛維奇的採訪則顯得更加輕鬆，談到首節和第二節的巨大差異性表現，貝西洛維奇說廣東宏遠第二節加突然快了比賽的節奏，而我們的退防做得不是很好，給了對手很多反擊的機會。

貝西洛維奇解釋說確實廣東隊實力非常強，我們在嘗試了一下後還是在後面做出了一些調整，讓更多的年輕球員得到鍛煉機會，保存主力隊員的實力，為下一場比賽做好準備。

江蘇男籃主場貝西洛維奇的採訪無意中透露了一個觀點，廣東太強大了，打得過就打，打不過就跑，就沒有想全力以赴。值得一提的是，上賽季兩隊季後賽相遇，最後一場比賽江蘇男籃全面放棄，甚至貝西洛維奇+雙外援都沒有現身賽場，於是一場43分的慘敗就此誕生。如果再往前說，貝西洛維奇當時帶著有馬尚布魯克斯的那支江蘇隊可是雙殺過尤納斯執教的廣東宏遠的。

下一輪比賽江蘇男籃將會與實力要弱很多的廣州男籃交手，沒有付出全力的江蘇男籃確實更有機會贏球。
