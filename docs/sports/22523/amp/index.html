美媒評十大奧運佔主導地位體育巨星：博爾特第二，中國無人上榜

天天要聞
2019-12-20

近日，美國全國廣播公司（NBC）評選出2010年代（2010-2019）在夏季奧運會佔主導地位的十位超級巨星，包括5名美國運動員和5名他國運動員，中國無一選手入選。

第一名 美國 拜爾斯

體操運動員西蒙妮·比爾斯，2016年獲得四枚里約奧運金牌，世界錦標賽獎牌總數達到25枚之多，拜爾斯保持六年不敗紀錄，以她名字命名的體操動作包括跳馬「拜爾斯跳」：踺子後手翻轉體180°直體前空翻轉體720°、自由操「拜爾斯空翻」：直體後空翻轉體兩周加轉180°。

第二名 牙買加 博爾特

作為過去十年最偉大的田徑巨星，博爾特在2010年代獲得六枚奧運金牌，成為第一位在多屆奧運會上奪得100米和200米雙冠的短跑明星，博爾特在2008年和2009年多次打破世界紀錄，直到2017年職業生涯結束，他的運動生涯令人印象深刻，博爾特兩大競爭對手為同胞布雷克和美國巨星加特林。

第三名 紐西蘭 卡林頓

女子皮划艇巨星卡林頓，在2012倫敦和2016年里約奧運會斬獲200米金牌，並在過去十年七屆世界錦標賽上至少獲得一枚金牌，成為該項運動名副其實的女王。

第四名 美國 伊頓

男子十項全能巨星伊頓，贏得兩屆十項全能奧運會冠軍，兩次打破十項全能世界紀錄。伊頓來自美國俄勒岡州，2011年大邱世錦賽首獲世界冠軍，年僅22歲開啟了自己的統治時代，2012年到2016年，贏得包括室內七項全能比賽在內的所有世界冠軍。

第五名 美國 萊德基

女子游泳巨星萊德基擅長400米、800米和1500米自由泳，在過去兩屆奧運會奪得五枚金牌。2013-2018賽季，萊德基打破了14項世界紀錄，在2015年喀山世界錦標賽上獲得了200米，400米，800米和1500米四項冠軍。

第六名 美國 菲爾普斯

男子游泳世界第一人菲爾普斯，在過去10年奪得12枚奧運獎牌，包括9枚金牌，他職業生涯總共擁有28枚奧運獎牌（金牌23），成為史上獲得奧運獎牌最多的運動員，2016年里約奧運會，他成為美國隊旗手，並以五枚金牌結束奧運生涯。

第七名 法國 特迪·里內

世界男子柔道巨星，贏得2012倫敦和2016里約奧運柔道100公斤以上級兩枚金牌。

第八名 俄羅斯 羅瑪西娜

世界女子花樣游泳巨星，在2012年倫敦和2016年里約奧運總共贏得4枚金牌。

第九名 美國 希爾茲

著名女子拳擊明星，奪得2012年倫敦奧運會拳擊比賽女子中量級金牌，成為第一位贏得奧運拳擊獎牌的美國女運動員。

第十名 波蘭 沃達爾奇克

女子鏈球世界紀錄保持者，過去10年贏得三屆世錦賽金牌、兩屆奧運會金牌，在2016年里約奧運上以奪金的82米29打破世界紀錄，比歷史第二好成績高出3米多，她也是史上唯一一位成績超過80米的巨星。
