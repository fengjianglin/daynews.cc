布斯克茨：不知道裁判腦子裡想些什麼 也不知道VAR什麼時候該介入

2019-12-15 1,414條評論 3,863次閱讀 1154人點贊

消息源告訴ESPN，巴塞羅那將向西班牙皇家足球協會（RFEF）進行投訴，他們認為在周六2:2戰平皇家社會的比賽末段中應該得到一個點球。第93分鐘，皮克似乎在禁區內被拉倒，但當值主裁Antonio Alberola Rojas拒絕了巴薩球員的申訴，他甚至沒有去回看VAR錄像。

在之前的一個類似情況下，巴薩被判罰過一個點球（奧雅薩瓦爾主罰命中使得社會取得領先），當時布斯克茨和迭戈-略倫特糾纏在一起。巴薩主席Bartomeu認為這兩個點球都應該進行VAR回看，他對在聖塞巴斯蒂安進行的比賽非常不滿，並在賽後第一時間聯繫了皇家足球協會的Rubiales。

Bartomeu將在當地時間周日向皇家足球協會提交一份關於VAR使用的正式投訴，他還要求協會解釋這項技術在西班牙的應用過程，支出巴薩是最早支持將這項技術引入西甲的支持者。

奧亞薩瓦爾的點球讓皇家社會取得領先，但巴薩隨後通過埃托萬-格列茲曼和路易斯-蘇亞雷斯的進球進行回擊。主隊隨後憑藉亞歷山大-伊薩克的進球扳平比分，雙方此後都有機會贏得比賽，但巴薩因為在最後時刻沒有取得點球而十分不滿。

「球場上總會有爭議，」巴薩中場布斯克茨告訴記者。「他們那個點球，我和他之間確實有身體對抗。也許我拉了略倫特，但如果我（這個動作）算犯規，你很難在其他場合保持公正。」

「證明這一點的是皮克，（對方的犯規動作）比我明顯多了。我不知道為什麼沒有VAR回放，這種時候VAR必須介入。」

「但是我們不知道裁判腦子裡在想些什麼，也不知道VAR到底什麼時候該介入。我們不知道，也不會知道，因為他們從來不說（標準），也從來不（對決定過程）做出評價。」

如果皇馬在周日能夠客場擊敗巴倫西亞的話，這場平局將使得巴薩將頭名讓給死敵。

下周三，巴薩和皇馬將在諾坎普展開本賽季首次對決，這場比賽本該在10月進行，但由於當時加泰羅尼亞地區的政治動蕩，這場比賽被延期。
