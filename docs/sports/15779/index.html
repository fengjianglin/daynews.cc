李春江丟大面子了，蘇若禹在線發飆，不惜用侮辱性語言

2019-12-16 1,806條評論 4,292次閱讀 3202人點贊

前言：李春江丟大面子了，蘇若禹在線發飆，不惜用侮辱性語言

在我們中國，大家都知道一句老話那就是人要臉樹要皮，意思就是說這大樹沒了皮活不成，那麼人沒有了「臉」同樣是難以抬起頭。不過一個人丟了臉還分很多種，一種是自己給自己臉面丟了，比方說幹了一些丟人的事情，另一種是別人把自己的臉給丟了，比方說孩子給老爸的，員工給老闆的，球員給教練的等等。

那麼作為一個體育作者，今天我們只聊體育方面的，也就是說球員給教練面子丟了的事情。而這個教練不是別人，正是前幾天我一直在聊的廣廈隊主教練李春江指導。李指導可以說在當今中國籃壇中也算是有頭有臉的人了，不過在近期他的弟子還真就辦了一件非常大膽的事情，也是讓李春江丟了大面子。

讓李春江丟面子的人不是別的，就是現在廣廈隊的大前鋒蘇若禹，對於這名球員相信了解的人並不多，這主要是因為他在數據上並沒有高光的時刻，而且在CBA打球階段也沒有做過爭議性很大的事情，給人的感覺就是一副非常憨厚老實的樣子。

而就是這個人們心目中的好好球員，在近期卻在線發飆了，並且不惜使用侮辱性語言來說出心中的不快。大概意思就是說自己表現好的時候沒見表揚，自己不好的是就開始咬人，對我不爽就別不放我走，好像誰他媽願意在這裡呆著一樣。這短短的幾句話李多次出現了侮辱性的詞語，這在當今CBA中還真的挺少見。

我們看到今天很多人說這是在暗指李春江指導，不過在我看來這完全就是錯誤的，因為我們知道李指導是在2013年才開始帶廣廈隊的，而蘇若禹的話里有8年這個信息詞，所以這根本就和李春江沒有一點關係，時間是完全對不上的。在我看來這更多是說給廣廈的管理層。

不過不管說給誰，其實蘇若禹這樣做都是在給李春江丟面子，作為現在李指導手下的球員，即使和管理層有矛盾，那麼首先不能這樣公開在線去發飆，並且還用了侮辱性的言語，這明顯就是讓李指導難堪，會給人一種李指導指導教導無方的感覺。

這一次蘇若禹在線發飆的事件已經影響非常大了，很多人都已經知道，這更加讓李指導難堪。在這裡我還是想說，不管球員受了多大的委屈，作為弱勢一方，這樣的言語顯然對自己是最不利的，衝動造成的後果相信在之後也是慢慢會顯現出來。

最後我想說的是，不管是球員還是我們大家，在面對一些自己理解中不公的時候，最好的解決辦法並不是抱怨，逞口舌之爭雖然一時爽了，但是結果真的會對自己有利嗎？與其自己生氣我覺得蘇若禹應該通過加練來提升自己的球技，畢竟球員想要受到更公平的待遇，靠的就是自身的實力。蘇若禹目前場均能夠得到6.8分，如果他能夠將自己的得分提升到10分以上，並且在廣廈這段沒有外援的時間裡表現搶眼一些，相信被別人嫌棄的情況將一去不復返的。
