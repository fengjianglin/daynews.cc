有驚無險！伊藤美誠4強已無障，世界冠軍負國乒小將遭遇一輪游

天天要聞
2019-12-14

文/梅長蘇

北京時間12月13日，國際乒聯巡迴賽總決賽在鄭州繼續進行，在女單1/8決賽中，中國選手陳幸同和新加坡老將馮天薇展開爭奪，結果陳幸同直落四局擊敗了近來狀態回升明顯的馮天薇晉級八強，世界冠軍馮天薇遭遇本次比賽「一輪游」，而中國台北「一姐」鄭怡靜在同日本「天才少女」伊藤美誠的比賽中遺憾3：4告負止步女單首輪，雖然輸球，不過能把伊藤美誠逼到懸崖邊上已經是非常難得了。

陳幸同和馮天薇的較量中，首局比賽陳幸同開局就取得了7：1的領先優勢，隨後她再得兩分以9：1領先，各得一分之後陳幸同10：2拿到局點，馮天薇追回一分之後陳幸同得分，這樣陳幸同以11：3拿下第一局；第二局比賽陳幸同開局落後，不過她穩住陣腳後逐漸把比分反超至6：5，再得四分後陳幸同以10：5領先，馮天薇挽救兩個局點之後，陳幸同得分以11：7再下一城大比分2：0領先；第三局比賽雙方打的非常激烈，兩個人的比分一直處於焦灼狀態，雙方從8平戰至9平，隨後陳幸同連得兩分以11：9拿下第三局，大比分3：0領先馮天薇；第四局比賽陳幸同開局5：3領先，隨後她連得五分以10：3領先，馮天薇挽救兩個賽點之後陳幸同以11：5拿下第四局，大比分4：0擊敗新加坡老將馮天薇晉級女單八強，下一輪她將迎戰王曼昱。

中國台北「一姐」鄭怡靜迎戰日本名將伊藤美誠，首局比賽伊藤美誠基本發揮出自己水平，鄭怡靜失誤偏多比分一直處於落後，伊藤美誠以10：7拿到三個局點，鄭怡靜不可思議連得五分以12：10反敗為勝拿下第一局；第二局比賽伊藤美誠沒有受到第一局失利的影響，她以11：6扳回一城；第三局比賽鄭怡靜在相持階段佔據優勢，隨後她逐漸把比分拉開並以10：5拿到局點，伊藤美誠連追兩分之後，鄭怡靜得分以11：7拿下第三局大比分2：1領先；第四局比賽伊藤美誠開局5：2領先，隨後她再連得五分10：2領先，鄭怡靜追回兩分之後，伊藤美誠以11：4拿下第四局，雙方大比分戰成2：2平。

第五局比賽伊藤美誠開局3：0領先，隨後她不可思議連丟七分以3：7落後，越戰越勇的鄭怡靜最終以11：4拿下第五局大比分3：2領先；第六局比賽伊藤美誠開局5：0領先，隨後她迅速以11：3拿下第六局比賽，雙方前六局戰成3：3平；決勝局比賽鄭怡靜開局落後，不過她很快把比分扳平，隨後雙方比分交替上升，伊藤美誠10：8拿到賽點，鄭怡靜挽救兩個賽點之後以10：12輸掉決勝局，大比分3：4不敵伊藤美誠，值得一提的是，去年兩個人也在總決賽上相遇，同樣激戰七局，不過當時取勝的是鄭怡靜，今年伊藤美誠成功「復仇」，晉級八強之後，伊藤美誠下個對手是日本名將佐藤瞳，伊藤美誠晉級四強幾乎沒有懸念。
