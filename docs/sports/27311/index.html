利物浦近38輪英超奪101分 克洛普要率隊打破隊史2大魔咒

2019-12-21 571條評論 2,336次閱讀 777人點贊

雖然今年夏天利物浦只花了190萬買人，但是本賽季他們依然延續著上賽季的良好勢頭，或者是勢頭更猛，17輪過去他們只有在第9輪客場1：1戰平曼聯那場沒有贏下。要是從上個賽季第18輪客場2：0完勝狼隊開始，跨賽季直到本賽季第17輪，近38輪英超比賽，利物浦取得了傲人的32勝5平1負的戰績，總共拿到101分。

利物浦唯一輸掉的比賽就是上個賽季第21輪客場1：2不敵曼城，這也是這兩個賽季到目前為止利物浦唯一輸的一場比賽。只是因為這場敗仗，讓上個賽季利物浦延續了一項尷尬紀錄，那就是英超年代，利物浦隊史4次獲得聖誕冠軍，但最終他們都沒能拿到英超首冠！過去11年里，英超只有3次聖誕冠軍沒能最終奪冠，這3次恰恰全部是利物浦，分別是08-09賽季，13-14賽季還有就是上個賽季。

上個賽季利物浦成了英超最冤的亞軍，不過好在他們連續2年殺入歐冠決賽後終於隊史第6次捧起大耳朵杯。獲得歐冠冠軍也讓利物浦第四次獲得參加世俱杯資格（包括世俱杯前身洲際杯），說起世俱杯，利物浦同樣有著和聖誕冠軍無法最終奪英超冠軍的魔咒，那就是此前利物浦3次殺入決賽都未能最終奪冠。因此雖然利物浦非常渴望首個英超冠軍，但相信世俱杯首冠對利物浦同樣很有吸引力。

這個周中英格蘭聯賽杯，克洛普直接留一大幫00後去打，而不是接受英足總的調換時間來看，在克洛普眼裡英超冠軍，歐冠冠軍，世俱杯冠軍才是想要的，其他的都要靠邊。如今利物浦已經早早鎖定本賽季英超聖誕冠軍，這是利物浦隊史第5次拿到英超聖誕冠軍，昨天2：1絕殺蒙特雷後利物浦則是第四次殺入世俱杯決賽。

這樣的話克洛普本賽季就將率領利物浦去打破隊史這2打魔咒，後天凌晨利物浦將和弗拉門戈爭奪世俱杯冠軍，今晚利物浦官方消息稱，隊醫將對范迪克和維納爾杜姆的身體情況進行檢查，以確定他們是否能參加周六與弗拉門戈的世俱杯決賽。范迪克的缺陣對利物浦影響不小，不知2天後克洛普能否率隊順利的先打破第1個魔咒……
