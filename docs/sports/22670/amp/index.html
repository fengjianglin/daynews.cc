ATP杯E組出線形勢 阿根廷克羅埃西亞爭出線 蒂姆或難救奧地利

天天要聞
2019-12-19

北京時間12月19日訊：新賽季的網壇還有不到半個月就將開戰，大多數球員們也都已經開始全力備戰。而新賽季的首站比賽將是一項全新賽事，由ATP巡迴賽和澳網協會聯合舉辦的ATP杯。雖然是項團體賽事，但其單打是有排名積分的，球員既可以為國家出戰，又能當作為澳網前的熱身賽，這也吸引了絕大多數的頂尖選手前來參加，世界前十中只有費德勒因為家庭原因最終選擇了退出。

首屆ATP杯將24支參賽國家分為6個小組，進行三場組內循環賽。每個小組的頭名和最好成績的兩個小組第二，晉級八強。賽制和改制後的戴維斯杯相差不多，都是兩場單打和一場雙打比賽，不過與戴維斯杯的單打隨意安排出場順序不同，ATP杯的單打順序必須嚴格按照世界排名來決定，排名更高的球員只能作為第一單打出戰，不能擔任隊伍的第二單打。今天我們來看看E組的出線形勢如何。

E組的四支參賽隊伍分別是：奧地利、克羅埃西亞、阿根廷、波蘭，將在悉尼進行小組賽。奧地利由世界排名第四的蒂姆坐鎮；克羅埃西亞由丘里奇和前大滿貫得主西里奇領銜；阿根廷由矮腳虎施瓦茨曼和佩拉領銜；波蘭由赫卡茨領銜。

本組波蘭整體實力較弱，頭號單打赫茨卡世界排名37，二號單打馬奇扎克世界排名也只有101，相對其他三個隊伍，波蘭想晉級淘汰賽相當困難了。而在其他三組隊伍的頭號男單較量中，奧利達的蒂姆明顯是實力更強的一方，世界排名高居第四的他在今年硬地場還迎來了蛻變。與克羅埃西亞頭號男單丘里奇的交手記錄中，蒂姆3勝1負領先；與阿根廷的施瓦茨曼交手記錄也有5勝2負的領先。

丘里奇與施瓦茨曼的交手記錄為2勝1負，在今年克羅埃西亞人因為傷病原因，狀態不佳，一度遭遇巡迴賽五連敗，但而正如他自己所說「打的好時像德約」，等到明年初比賽時會以什麼樣的狀態出戰還不好說。在頭號男單的比拼中，蒂姆佔據著一定優勢，但奧地利除了蒂姆外，整體實力較克羅埃西亞和阿根廷要相差不少。

二號男單上，奧地利的諾瓦克世界排名百名開外；而阿根廷的佩拉世界排名第25，且在今年打進了溫網八強，實力不錯；克羅埃西亞的西里奇更是前大滿貫得主，雖然今年狀態不佳，世界排名也掉出前20，但可謂瘦死的駱駝比馬大。所以奧地利在第二場男單中，獲勝希望不大。而到了決勝的雙打比賽，阿根廷和克羅埃西亞的好手就更多了，奧利達恐怕也很難取勝。

綜合來看，本組波蘭實力最弱，晉級基本無望。而奧地利雖然有蒂姆這個強點，但其他人實力有限，或許很難從克羅埃西亞和阿根廷身上取得最終勝利。本組最有希望出線的是克羅埃西亞和阿根廷，兩隊之間的交手結果，或許將直接決定最終誰將晉級。
