重慶國腳接近加盟大連，卡拉斯科走後，他將成一方邊路的關鍵球員

2019-12-17 147條評論 1,146次閱讀 111人點贊

近日，據某國內體育媒體報道，目前正跟隨中國男足參加在韓國舉辦的東亞四強賽的重慶斯威俱樂部的馮勁非常接近加盟大連一方。

現年26歲的馮勁，是斯威隊中為數不多的重慶本地球員。1993年，馮勁出生於四川重慶，14歲加入重慶力帆梯隊，2011年，馮勁隨重慶力帆梯隊在全國中學生足球聯賽中奪得全國季軍。

2013年，重慶力帆主帥王寶山將19歲的馮勁提拔進了重慶一隊。2013年，馮勁雖然進入一線隊，歲重慶力帆征戰中甲聯賽，但卻鮮有上場機會。直到2015賽季，馮勁才逐漸成為了球隊主力。這個賽季他共代表斯威出場19次，其中7次首發。在客場與天津泰達的比賽中，馮勁打進了個人在中超聯賽的處子球。

本賽季的中超聯賽第22輪，重慶斯威在客場以1-1逼平了中超霸主廣州恆大，馮勁在上半場第31分鐘，先後擺脫了恆大幾名後衛的圍追堵截，打入了一記非常精彩的進球。自此，馮勁得到了「重慶梅西」的稱號。

本賽季初，馮勁入選了國家隊大名單，並且在比賽中正式出場。此次東亞杯，馮勁再次入選李鐵執教的國家隊。這也證明了他優異的表現已經打動了國家隊新任主帥李鐵的心。

本賽季結束後，重慶俱樂部陷入財政困境，功勛主帥小克魯伊夫近日已經離開球隊。據相關消息人士透露，青黃不接的大連一方隊非常希望能夠得到這位速度極快的邊路球員，下個賽季，馮勁很有可能將加盟一方。如果馮勁能夠順利加盟，他將為一方隊的邊路進攻帶來很大的幫助。尤其是在外援卡拉斯科鐵心回歸歐洲賽場的情況下，他將成為一方主帥貝尼特斯在進攻中的一枚重要棋子，為這位西班牙戰術大師的臨場戰術調整提供更多變化的可能性。
