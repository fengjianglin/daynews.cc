東京奧運會中國男乒參賽名單逐漸明朗，“三駕馬車”眾心所向

2019-12-11 1,335條評論 3,501次閱讀 59人點贊

距離東京奧運會開幕還有8個多月，國乒參賽名單一直受到球迷們的關注。其實，相對於女乒，男乒競爭程度就沒那麼激烈。從里約奧運會結束，進入東京奧運會周期，倫敦奧運會男單冠軍張繼科漸漸隱退，樊振東與馬龍、許昕組成了新「三駕馬車」，三人在國際賽場上所向披靡，基本就是東京奧運會團體三人蔘賽名單，至於P卡，由於王楚欽摔拍事件，也基本退出了爭奪隊伍，會有林高遠或梁靖崑獲得。

至於哪兩人出戰單打，我認為樊振東肯定先佔一個席位。雖然2019年他大半時間都處於低迷階段，但到了德國公開賽後，他真正王者歸來，連續奪得德國、奧地利公開賽男單冠軍、東京團體世界盃冠軍、男子世界盃冠軍和世軍會加冕「四冠王」，這一系列成績都在說明他已經具備東京奧運會男單資格，他也是巴黎奧運會周期的領軍人物；剩下一個就由馬龍和許昕爭奪了。

男單：樊振東、馬龍，團體：許昕

這種情況是2019年前球迷們所傾向的。馬龍出戰男單，前提是狀態至少恢復到原來的8成水平。大家都知道，馬龍從去年保加利亞公開賽後長達8個多月退出國際賽場，直到今年卡達公開賽才開始復出，以卡達公開賽男單冠軍、布達佩斯世乒賽男單和男雙冠軍和中國公開賽男單冠軍向大家證明他的實力依舊是處於領先地位的。不過馬龍在T2鑽石聯賽馬來西亞站後再次退出國際賽場，男單首秀也是在男子世界盃，時隔4個月沒有打高水平比賽，導致競技水平有所下滑，從而接連輸給張本智和和林昀儒兩位小將。不過距離東京奧運會還有8個多月，時間足夠讓馬龍膝傷恢復，相信之後又可以見到健康的馬龍。

男單：樊振東、許昕，團體：馬龍

讓我們欣慰的是，即使馬龍無法參加東京奧運會男單項目，依舊有人選頂替———許昕。看過男子世界盃馬龍與張本智和、林昀儒的比賽的人應該知道，馬龍的膝傷還沒有恢復好，導致步伐無法快速移動，一些球無法回擊。此時，最糟糕情況就出來了，如果馬龍膝傷太嚴重了，無法在高運動量的情況下參加單打比賽，否則會造成不可磨滅的傷害。那麼馬龍就只能參加團體第三人，最後為國乒保駕護航。

此時許昕的時機來了，雖然布達佩斯世乒賽男單意外失守，但他並沒有「碌碌無為」，放棄單打專註雙打，他依舊有一顆嚮往東京奧運會男單項目的「初心」。俗話說，一切得靠實力證明。許昕做到了，從日本公開賽開始他爆發了。日本、韓國、澳大利亞連續三站公開賽獲得男單冠軍，日本公開賽加冕「三冠王」，亞錦賽一人身兼四項，斬獲三金一銀。之後在T2新加坡站連續橫掃張本智和、林昀儒獲得男單冠軍，世界排名時隔一個月再次登頂世界第一。許昕不僅單打增強，男雙和混雙也沒落下，並駕齊驅，共收穫19個冠軍。只是到時候許昕就有點累了，到時候就要參加男單、團體和混雙了。相信「勞模」許昕能夠做到。

馬龍和許昕，你更傾向誰打單打呢？
