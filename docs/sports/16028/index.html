男排單局吊打25-13！沈富麟挂帥豪取三連勝，江川拼防守險被撞傷

2019-12-14 1,709條評論 2,687次閱讀 1397人點贊

主教練沈富麟（上海）在男排國際賽第3輪，沿用二傳19號詹國俊（上海）、自由人8號童嘉驊（上海）和接應2號江川（北京、隊長）、主攻1號戴卿堯（上海）與10號季道帥（山東）、副攻12號張哲嘉（上海）和13號陳龍海（上海）聯袂出戰。副攻11號耿鑫在第一局23-18時輪換詹國俊替補攔網，但重炮戴卿堯由於攔網意外崴傷腳，被沈富麟關切攙扶離場。

第一局由季道帥防守失效和戴卿堯一傳墊飛到0-2，中國男排利用埃及開炮和攻傳失配、陳龍海封堵連得3分反超。季道帥變扣為吊與陳龍海跳飄壓線、江川強攻藉手出界至6-3，中國男排逼迫埃及休戰後江川未能防起處理球，但還是領先技術暫停到8-5後由季道帥跳發連得2分。張哲嘉偷襲短平快和跳發破壞一傳，戴卿堯強攻至12-6可惜江川攔網被藉手出界。

陳龍海防反近體快只可惜季道帥攔網失效，中國男排由於季道帥讓一傳漏球被追發到13-11。沈富麟休戰後江川強攻和張哲嘉封堵連得2分，中國男排因為季道帥墊飛一傳和江川被攔至15-14。季道帥平拉開領先暫停後埃及後三踩線，中國男排通過江川跳發連得2分但後二開炮。戴卿堯在19-17時攔網崴傷腳被崔建軍換下，張哲嘉短平快和三度跳發球先勝至25-19。

中國男排第二局繼續是崔建軍與季道帥搭檔主攻，詹國俊跳發但陳龍海攔網被藉手出界，埃及傳球連擊失誤到1-1平被陳龍海偷襲。中國男排依靠季道帥巧打藉手出界連得2分，江川背平下球、張哲嘉防反短平快和偷襲二次球，再由詹國俊暴扣領先技術暫停至8-3。張哲嘉攔網與埃及強攻由鷹眼顯示開炮，中國男排在江川跳發輪由4-3製造6-0到10-3但攔防失效。

童嘉驊一傳連續不到位好在江川防反藉手出界，中國男排被偷襲二次球但接發球依然波動至12-9。沈富麟休戰後崔建軍變扣為吊和陳龍海偷襲，中國男排在15-11時由於季道帥自接自扣開炮連丟2分。季道帥鷹眼挑戰巧打觸碰對方小拇指成功到17-14，張哲嘉機智推後場迫使埃及休戰可惜一傳送探頭再丟2分，中國男排通過詹國俊跳發和季道帥攔防再勝至25-22。

第三局由崔建軍相繼跳發得分並且利用對方開炮，中國男排憑藉陳龍海變扣為吊到5-0迫使埃及休戰。江川背平二直線迫使對方自由人防守墊飛，中國男排通過季道帥和張哲嘉封堵建功領先技術暫停至8-2。江川防守意外險被童嘉驊撞傷好在無礙，中國男排利用崔建軍自接自扣和張哲嘉跳發到11-4，在逼迫埃及輪換陣容後陳龍海攔網失效和崔建軍被攔連卡3分。

12-7後詹國俊跳發下網和季道帥後三被攔，中國男排依靠陳龍海短平快與對方攻傳失配，崔建軍跳發破壞一傳由江川背錯至15-9。埃及休戰後由詹國俊倒地妙傳陳龍海偷襲至16-10，中國男排憑藉江川跳發四度直接得分擊掌沈富麟，詹國俊封堵埃及重扣和季道帥調整強攻到23-11，季道帥變扣為吊至24-12但跳發下網浪費賽點，崔建軍平拉開以25-13鎖定勝局3-0。
