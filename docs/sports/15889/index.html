遼寧對吉林，最大的看點卻是師弟最大的心酸！

2019-12-15 243條評論 1,355次閱讀 152人點贊

北京時間12月14日，遼寧主場將迎來吉林的挑戰，由於兩隊一直以來關係都不錯，因此這場比賽也少了點硝煙味。

雖然吉林排名要高於遼寧，不過從陣容來看遼寧更均衡些，而這場比賽最大的看點也就是師弟和瓊斯的較量。

師弟本賽季場均26.6分7.3籃板4.3助攻，發揮正常，瓊斯則場均37.4分7.8籃板9.1助攻，是聯盟的得分王，最近三場比賽更是分別得到了51分、59分、41分，從比賽效果來看，師弟和瓊斯、富蘭克林、馬尚相比，還有差距。

對遼寧而言，限制瓊斯是取勝之道，而遼寧隊內師弟應該是防守瓊斯的最佳人選。雖然師弟的防守態度有很大問題，不過對於身體爆發力十分出色的瓊斯來說，或許只有師弟能跟上他的節奏。

說師弟和瓊斯的對位是最大看點除了賽場上的因素之外，兩人的境遇也是一方面。

雖然遼寧一直表示師弟和球隊的磨合在慢慢變好，不過看過比賽的都知道，這只是遼寧的官方回應而已，一向以自我為核心的師弟在遼寧打的並不痛快，尤其是在和郭艾倫之間。

我們經常能看到，郭艾倫一拿球，師弟就在場上遛彎，要不到球，就消極怠工，經常形成對方五打四的局面。

師弟和瓊斯一個是聯賽最大牌外援，一個是得分王，不過他倆在各自球隊的待遇卻千差萬別，一個只能當陪襯，一個大開大合，是球隊絕對核心。

因此這場比賽也是師弟的證明之戰，不過無論師弟的表現再好，在遼寧恐怕他也不會有瓊斯那樣的位置。
