巴薩機會來了！歐冠暗藏神巧合：淘汰此隊=奪冠

2019-12-16 1,681條評論 2,885次閱讀 185人點贊

北京時間12月16日晚19點，2019-2020賽季歐冠16強抽籤儀式在瑞士尼翁的歐足聯總部進行。5大聯賽16強捉對廝殺，皇馬碰上曼城，馬競PK利物浦，切爾西大戰拜仁，那不勒斯遭遇巴薩。對於擁有梅西、蘇亞雷斯、格列茲曼的巴薩來說，本賽季歐冠的目標，自然是衝擊冠軍。恰好，有一個神奇的巧合，似乎意味著巴薩將奪冠……

這是什麼巧合呢？著名數據大師MisterChip給出答案：1/8決賽淘汰那不勒斯的球隊，最終都奪得歐冠冠軍！

算上這次晉級歐冠16強，那不勒斯歷史上，只有3次殺進歐冠1/8決賽。第1次是2011-12賽季，當時那不勒斯以小組第2的身份晉級16強。1/8決賽，那不勒斯碰上切爾西。首回合，那不勒斯主場3-1取勝，拉維奇梅開二度，卡瓦尼打進1球。不過次回合，切爾西主場4-1翻盤，最終5-4淘汰那不勒斯。

2011-12賽季的歐冠，對切爾西來說如同史詩般壯闊。迪馬特奧救火，切爾西翻盤那不勒斯，淘汰本菲卡，半決賽更是將宇宙隊巴薩淘汰出局，決賽大戰拜仁，德羅巴頭槌救主，點球大戰艱難取勝，隊史首次拿到歐冠冠軍。我命由我不由天，切爾西的奇蹟奪冠，堪稱歐冠經典。

那不勒斯第2次打進歐冠淘汰賽，是2016-17賽季。當年的歐冠小組賽，那不勒斯頭名晉級，不過1/8決賽碰到了皇馬，主客場兩個1-3，那不勒斯輸得毫無脾氣。淘汰那不勒斯之後，齊達內的皇馬又將拜仁、馬競踢出局，決賽4-1大勝尤文，笑傲歐冠。

那不勒斯兩次打進歐冠16強，兩次成為冠軍墊腳石。2019-2020賽季，那不勒斯第3次打進歐冠16強，這次，那不勒斯碰上奪冠熱門巴薩。每一年，巴薩都是歐冠熱門，但過去4年，巴薩在歐冠表現慘淡，3次8強，1次4強，更是被羅馬、利物浦的超級大逆轉釘在歷史恥辱柱上。這個賽季，巴薩能否洗刷恥辱，梅西能否帶領球隊登上歐洲之巔？「歐冠神巧合」不能說明任何問題，只是概率上的巧合，巴薩想要稱霸歐冠，還是要靠自己一場場拼下來。
