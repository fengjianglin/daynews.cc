CBA爆出超級大冷門！昔日總冠軍轟然倒塌，林書豪這下危險了？

天天要聞
2019-12-21

北京時間12月20日，CBA常規賽繼續火熱開打。近來取得兩連勝的北京男籃坐鎮主場，迎戰浙江男籃。話說最近林書豪遭遇惡意防守一事鬧得沸沸揚揚。書豪在NBA被人歧視，「你是黃皮膚的憑什麼這麼牛逼」，在CBA，居然還是被這樣對待：「你也是黃皮膚的憑什麼這麼牛逼」。小編不是說林書豪寶貝，但在球場上受了惡意犯規、惡意動作的他從來沒有抱怨過，只是呼籲大家，停止這樣的「針對性行為」，小編認為，是正常的訴求。最終北京男籃以93-112不敵浙江，兩連勝慘遭終結。

今天北京首鋼的手感相當一般，前11次出手僅僅是命中1球，三分更是3中0。反而是這賽季表現一直比較低迷的方碩小小地爆發了一番，第二節還沒打完已經拿到了7分，出手3次全部命中。首鋼進攻狀態低迷，更顯浙江那邊手感的火熱。吳前、蘭茲伯格、朱旭航三位球員上半場聯手砍下了38分，三人的得分幾乎趕上了北京首鋼全隊。北京首鋼引以為傲的防守，在浙江面前似乎並沒有什麼效果。原因很簡單，浙江知道北京的命門就在內線，雖然首鋼的大外援尤度表現出色，護框效率奇高，但畢竟他只有一個人，首鋼其他內線基本都是上來送分的。蘭茲伯格和朱旭航在內線為所欲為，基本沒有遇到稍微激烈一點的反抗。

從整場比賽來看，北京男籃的進攻火力，完全被浙江隊所壓制。以往首鋼非常依賴書豪的進攻狀態。如果書豪能夠打開，首鋼大概率能夠拿下比賽的勝利，即使書豪狀態不行，他還能傳球，還能引導其他國內球員。但今天首鋼的球員，除了大外尤度和方碩和劉曉宇以外，幾乎沒有一個是正常的。翟曉川全三節得2分，朱彥西0分。即使書豪想傳球，想帶動，他們也沒有辦法被帶動起來——狀態實在太差！當然，也有球迷打趣道，每當書豪表現不好的時候，就是方碩和劉曉宇出頭之日。其實這個邏輯還是很好理解的，書豪打不出來，方碩和劉曉宇作為剩下的持球點，當然要分擔更多，打得更好，也是正常的。

打完這場，小編相信書豪的境地更加危險了。三分之一的比賽下來，林書豪大傷過沒法真正撐起北京的進攻大旗，已經是不爭的事實。書豪的攻堅能力，在CBA所有的小外中，應該算是比較差的。當然，我們也能夠理解，畢竟書豪年輕時就是依靠出色的身體素質在NBA留下輝煌歷史的球員，大傷後他失去了速度，這就意味著他突破的能力大打折扣，加上書豪本身就不擅長三分遠投，這讓他在如今的CBA，顯得比較艱難。如果北京首鋼真的想更進一步，書豪的位置恐怕就有點危險了。
