中國足球再當冤大頭！被4大外教“坑慘”，近10億人民幣打水漂

2019-12-25 189條評論 1,405次閱讀 292人點贊

據《大連晚報》消息，希丁克仍未和足協解約，這意味著，希丁克雖然已經在給埃因霍溫工作，但還在足協拿著薪水。從卡馬喬開始，到後來的佩蘭、里皮，再到現在的希丁克，中國足球都被「坑慘」。 當然，這完全中國足球自己犯下的錯，別人只是按照合同拿錢。

2011年8月，卡馬喬正式入主國足，他的教練團隊年薪是400萬歐元。但是，卡馬喬率領國足成績糟糕，尤其是在合肥輸泰國二隊1-5，成為了國足史上最恥辱一幕！2013年6月，卡馬喬下課。但因為是解約，足協全額支付卡馬喬剩下1年半薪水，也就是600萬歐元。加上卡馬喬一共到手的800萬歐元，在卡馬喬身上，足協花了1400萬歐元。

卡馬喬下課後，國足下一位外籍主帥是佩蘭，年薪140萬歐元。佩蘭也是執教了2年時間，最終解約，賠償金是3個月薪水。也就是說，佩蘭在中國掙了310萬歐元。

里皮是國足史上最貴主帥，第一次任期，里皮執教828天，賺到了3.5億薪水！二進宮的里皮一共在中國男足主教練的位置上待了175天，也能掙到8544萬人民幣，加起來就是接近4.4億人民幣！

如今的希丁克，年薪是300萬歐元，由於他和中國足協簽訂的合同到明年1月，雖然希丁克早已經不在中國，但由於沒有解約，因此，希丁克仍在中國真錢。明年1月合同到期，希丁克在中國也算工作了1年半，將賺走450萬歐元薪水。

客觀來說，這4任外籍名帥的到來，基本沒有幫助到中國足球。卡馬喬、希丁克毫無幫助，佩蘭至少還率隊在亞洲杯上3連勝，而里皮走後，留下一地雞毛。

但是，4個人一共在中國賺走了9.3億人民幣！
