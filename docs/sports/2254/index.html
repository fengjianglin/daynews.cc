我們先從一局棋開始，這是1985年5月22日第1屆中日圍棋擂台賽的第九局，錢宇平對陣小林光一，214手時，長期處於讀秒苦戰的錢宇平認輸，小林光一中盤取勝，但由於對殺的棋形黑意外氣很長，所以如果黑棋繼續下去，認輸的將是白棋，錢宇平錯失成為第一位戰勝日本超一流棋手小林光一的殊榮，

終局黑棋認輸局面，

但經過邵震中和聶衛平等人確認，1-7做好準備，然後黑9白10開始互相緊氣，或形成雙活，或黑棋截斷白棋25子的尾巴。

如果這個圖才是終局局面，那麼輸棋的一方將是白棋。
六年過去了，小林光一早已貴為日本第一人，而錢宇平也非昔日吳下阿蒙。
重要的標誌是，

1990年3月16日，在日本東京，第5屆中日圍棋擂台賽第十局，錢宇平執白勝坂田榮男4目半。

1990年7月1日，在中國南京，第5屆中日圍棋擂台賽第十一局，錢宇平執黑勝武宮正樹5目半。

親手終結擂台賽。錢宇平已經成為中國圍棋繼聶衛平之後又一位冉冉升起的新星。
1991年7月6日，在日本大阪，錢宇平又用一次貨真價實的勝利將這一論斷再下註腳。
這就是第4屆富士通杯半決賽他執黑183手中盤勝小林光一。 錢宇平九段成為繼聶衛平九段之後第二個擊敗小林光一九段的棋手。

黑41是「鈍刀」一流厚實的好手。

43以下緊峭有力，到63黑棋先著效率充分。

白76補是本手，由於受到下邊一塊弱子的拖累，白棋難以有效地擴張上邊模樣。
黑91破壞上邊白空，黑棋實地明顯領先。

白92挖是小林光一突襲的好手。

黑113後，白棋形勢已不容樂觀

白1位跨黑棋將很棘手

白108沖雖然有收穫，但卻是小林十分後悔的一著，此時應立即在上圖1位跨斷，黑棋中央十分麻煩。

被黑135虎，白棋敗勢已定

本局獲勝後，錢宇平九段殺入富士通杯決賽。這是1988年，1990年聶衛平之後又一位中國棋手進入世冠決賽。隨後錢九段繼續訓練，為1991年8月3日與趙治勳九段的決賽作準備，一天他突然覺得「頭部劇痛」，到醫院檢查後發現「神經衰弱症複發，必須休養治療」，

官方棋譜記錄顯示，
1991年7月27日錢宇平在中國 棋王賽執白勝楊暉，

1991年8月1日中國圍棋協會正式通知日方：由於中國棋手錢宇平突然患病，將放棄決賽資格。趙治勳不戰而勝。

1991年8月3日趙治勳和王立誠均不戰而勝，因為錢宇平棄權決賽，小林光一同時放棄了季軍爭奪。

第四屆富士通杯僅僅由趙治勳和女棋手中澤彩子下了一盤指導棋草草收場。

1991年7月底到8月初這一周發生了什麼，成為圍棋歷史謎團。

然後錢宇平九段就淡出了人們的視線。

這以後1993年到1996年錢宇平九段有過短暫復出，
1993年1月5日錢宇平在中國名人賽執黑勝王海軍，

他曾代表中國參加亞洲電視快棋賽，勝劉昌赫，依田紀基，決賽負大竹英雄。
1996年11月24日錢宇平在中國 霸王賽中執黑7目半勝宋雪林是他的最後一次職業演出。

1997年，錢宇平九段離開了職業圍棋圈，鈍刀之名頓成絕響。
