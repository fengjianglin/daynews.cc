曼聯的劇本被巴爾維德拿到巴薩-前場火力全開，後場重大火災

天天要聞
3 天 ago

巴塞羅那在不到半個賽季的時間裡，把相似的劇情重演了幾回，通常都是一波華麗操作，大勝不止進球不斷，然後就像汽車突然拋錨一樣，然後再來一波華麗操作，不知道什麼時候再拋錨一回。

面對實力狀態更好的塞維利亞和馬德里競技都能拿下，卻在格拉納達、萊萬特這樣的球隊身上栽跟頭。

這一次巴塞羅那終於有所改觀，拋錨也只是平局，對手已經不是中下游球隊，而是距離巴薩的位次並不那麼遙遠的皇家社會。

同樣巴薩的劇本像是從曼聯那裡拿來的，前場攻擊線氣勢如虹，後場防線如爆發山洪，如果不是特爾施特根的神勇表現，連勝幾輪必然掉鏈子的情景又要重現了。

巴爾維德突然間又得背上這一口鍋。前場攻擊力，經常受制於後防線的逐步下滑，這是從恩里克時期就已經出現的頑疾。

巴薩一條後防線的球員不斷證明這一點，又不斷讓全體球員承擔這一後果。就像曼聯的德赫亞經常要去補救林德洛夫和馬奎爾的漏人一樣。

眼下的巴薩還能夠以強大攻擊力穩定位置，但同樣不年輕的攻擊線有所閃失有狀態低迷的時候，與後防線的不穩同時出現，巴爾維德又要拿一份相似的劇本了。
