索帥執教曼聯首年勝率超克洛普！學弗爵激勵士氣，為球員表演節目

2019-12-21 2,983條評論 4,896次閱讀 1269人點贊

曼聯本周與沃特福德的英超比賽前，主帥索爾斯克亞組織了一次聖誕午餐，用美食和節目傳遞節日的歡樂，激勵球員的士氣。

索爾斯克亞要求所有曼聯球員都必須參加，這跟上賽季此時的穆里尼奧完全不同。穆帥在聖誕節前已經接近下課，他沒有強求球員一定要出席，而鳥叔本人也躲在自己的辦公室里，俱樂部內部士氣低落。

有參加了去年聖誕午餐會的曼聯工作人員形容，那是二十年來最糟糕的聖誕聚會。因此，索爾斯克亞希望今年能有新的氣象。

過去，曼聯青年隊會在聖誕聚會上表演啞劇，有一次他們甚至重演了弗爵與阿森納大戰中被法布雷加斯用披薩打中臉部的英超經典一幕。

為了提振曼聯的隊內氣氛，完成他的文化重置，索爾斯克亞效仿了恩師弗格森，他打算在宴會上粉墨登場，親自為球員表演節目，並發表演講激勵球員。還打算幫助廚師做菜，讓弟子們嘗嘗自己的手藝。

消息人士稱，曼聯多數球員都很期待今年的活動，他們開著各自的豪車興奮地來到卡林頓。一名內部人士告訴英國《太陽報》：「球員們都很期待周五的午餐，去年很糟糕，在若澤（穆里尼奧）下課前，大家的情緒差得驚人。曼聯的傳統是樂觀向上，但最近的主帥都不怎麼重視聖誕節活動，奧萊（索爾斯克亞）打算改變這種做法。」

在穆帥前，莫耶斯執教的賽季，廢止了弗爵創立的青年隊啞劇慶祝聖誕的傳統，這個活動備受球員好評，除了弗爵的披薩門，魯尼嫖妓也曾被小將們開涮。范加爾曾經一度恢復這一活動，但2015年聖誕他也取消了曼聯的聖誕慶典，因為紅魔在聖誕節之前遭遇3連敗以及6場不勝，范廚師一怒之下不但下令取消啞劇，還要求球員聖誕節當天加練。

本周，索爾斯克亞剛剛度過執教曼聯一周年紀念日，2018年12月19日，他被任命為紅魔臨時主帥，因為戰績出色意外得以轉正。沒想到，索帥成為正式主帥後，曼聯的成績大幅度滑坡。

不過，英國《曼徹斯特晚報》公布的數據稱，索爾斯克亞執教曼聯第一年的戰績，比克洛普在利物浦、波切蒂諾在熱刺的首年勝率更高。索帥29勝12平14負，勝率52.7%；克洛普在利物浦的第一年只有49.2%的勝率，與波切蒂諾帶領熱刺一樣。

曼聯周日的對手沃特福德陣中，也有一名21歲的前紅魔青訓球員卡勒姆·惠蘭，賽前他盛讚曼聯的青訓實力強得荒謬，博格巴讓他無法阻擋。「曼聯青訓的標準強如笑話，在我看來是頂級水平。」惠蘭說，「當博格巴來了之後，他是全隊最突出的，我回憶過去認為他不只高人一籌，我不知道自己最終能否達到那種水平。當年我在曼聯只有14歲，在那裡長大，那是我生命中重要一部分。我在曼聯還有很多朋友，這很奇怪，也許這場比賽是我第一次希望曼聯輸球，我只希望沃特福德取勝。」

不過，英甲球隊彼得堡聯隊老闆本周在推特控訴曼聯，稱今夏球隊想要租借曼聯青訓小將，結果紅魔索要20萬鎊租借費，讓他們知難而退。
