穆里尼奧：維爾通亨經紀人在拖延時間 希望埃里克森能夠續約

天天要聞
2019-12-21

每日郵報報道，托特納姆熱刺主帥穆里尼奧談到球員續約的問題，暗示比利時國腳維爾通亨的經紀人在故意拖慢續約節奏，而且這名後衛的合同將在明年夏天到期。穆里尼奧說道：「如果其中一部分不想實現，那就很難去實現。」此外他堅稱自己希望丹麥中場埃里克森能夠簽下一份新合同。如果冬季不能夠引入新球員那麼留住球隊中堅力量也是一件好事！

每日郵報截圖

穆里尼奧希望中場埃里克森和中衛維爾通亨能夠和阿爾德韋雷爾德一樣簽下新的合同，但是維爾通亨的經紀人導致談判陷入僵局，在周五托比也是成功結束了兩年談判的僵局，最終簽下一份4年合同直到2023年，周薪上翻了3倍達到15萬鎊周薪，現年30歲的他等履行完合同基本就等於終老熱刺了！

之前看起來這名比利時國腳是很大可能離開俱樂部，因為在前任主帥波切蒂諾的帶領下陷入談判僵局，不過自從穆帥的到來讓他改變了想法。阿爾德韋雷爾德曾在採訪中表示很開心與穆里尼奧共事，球隊會在他麾下越來越好，果真熱刺在聯賽已經迫近前四，周末贏下切爾西將回重回前四。

埃里克森和維爾通亨在效力熱刺之前，在荷甲阿賈克斯也是隊友，目前兩位球員的想法似乎都非常一致在拖延時間。他們在12天之後就可以與其他歐洲俱樂部商討自由轉會事宜。說服埃里克森留下的可能性看起來不大，在其他歐洲俱樂部當中包括曼聯、皇馬對其都非常感興趣，據報道稱他自己已經告訴穆里尼奧無意簽下合同。而葡萄牙人則暗示維爾通亨的經紀人讓交易變得困難。

穆里尼奧在採訪時說道：「關於埃里克森，我唯一要告訴你的就是我希望他簽下一份新合同，除此之外不會告訴你們任何事。」另外一名球員維爾通亨，穆帥認為球員想留在俱樂部的時候就會簽下合同，如果其中一個部分不想要，那麼都很難做到，除非球員選擇更換經紀人，否則很難做到這一點。

在談及到阿爾德韋雷爾德談判時，再次暗示他的經紀人給他帶來麻煩。他補充道：「球員開心，俱樂部開心，現在在足球領域中，在這個時刻，有些人看起來比球員還更重要，而這些都是在決策中和解決方案中，所以在這種情況下，家庭和經理人都很樂意讓球員開心，所以這很容易。」

托比的續約讓穆里尼奧得到廣泛的稱讚，因為他到來時就曾表示托比是他想要的球員。「俱樂部的目標和我的完全一樣。」在這個時刻我們是俱樂部，我是俱樂部，我們想要托比，托比想要我們，和他一起工作的人都很樂意實現球員想要了，所以這並不困難。」

「對我來說買這句話很優美很簡單，在別的地方，我不再快樂過了，所以如果他在另一個地方不能更快樂，那就開心吧。」穆帥的效應讓全隊士氣就得到提升，就看下位續約的球員或者簽約的人是誰吧！
