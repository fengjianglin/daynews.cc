主場再次失利，拉科魯尼亞18輪聯賽未嘗勝績，球隊恐"越陷越深"

2019-12-09 0條評論 24次閱讀 0人點贊

摘要

北京時間12月8日，拉科魯尼亞主場1比3完敗昔日西甲對手薩拉戈薩，本賽季連續不勝場次增加至18場。目前他們1勝9平9負積12分排名倒數第一，離聯賽倒數第5的馬拉加有8分差距,保級形勢不容樂觀。

北京時間12月8日，拉科魯尼亞主場1比3完敗昔日西甲對手薩拉戈薩，本賽季連續不勝場次增加至18場。目前他們1勝9平9負積12分排名倒數第一，離聯賽倒數第5的馬拉加有8分差距,保級形勢不容樂觀。

拉科

縱觀整場比賽，拉科在進攻端沒有多少辦法，球隊更多的是邊路起球中路爭點創造機會，他們目前這樣的打法看上去更像傳統英超球隊的進攻方式。在防守端球隊的問題也不少，球隊所丟的第三球薩拉戈薩簡單的配合就將球隊防守打穿。另外本場比賽要不是門將埃爾南德斯出色的發揮，球隊可能將迎來一場更大的慘敗。

全場結束時拉科魯尼亞球迷在看台上注視球場發生的一切，他們的內心肯定無法相信曾經的西甲勁旅到了如此落魄的境界。而場上球員個個表情凝重，有的球員甚至留下了眼淚。

下一輪聯賽拉科魯尼亞將在客場挑戰邦弗拉甸拿。邦弗拉甸拿主場戰績不錯，拉科魯尼亞又將迎來一場艱苦的比賽。

最後，還是希望拉科魯尼亞在之後的比賽有好的發揮，畢竟誰都不願意看到他們再一次降級。
