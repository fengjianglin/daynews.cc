差距！日本真核或加盟歐冠冠軍，國足聽了還敢領薪水？

天天要聞
2019-12-14

​

韓國一個孫興慜已經讓中國的球迷羨慕不已，如今傳出了消息，日本球員南野拓實因為在歐冠小組賽表現出色，他在薩爾茨堡紅牛客場3比4惜敗給利物浦的比賽中貢獻了1球1助，給利物浦主帥克洛普留下了深刻印象。加上利物浦正計劃開拓日本乃至亞洲市場，南野拓實有很大的概率加盟歐冠冠軍，成為孫興慜又一位登陸歐洲豪門的亞洲球員。

​

中國足球和日本足球的差距，就是職業聯賽以後開始拉大的。

看看我們國內的球員，大部分都是安逸於中超，捨不得高薪，實在讓人唏噓不已。如今的中國足協已經表態將向日本學習，當初我們學了德國，學了日本，學了巴西，學了荷蘭，學了各種，其實就是感覺看不起日本韓國，到了了，發現還是要回歸原點。

​

但是國足如果學日本該怎麼做？看看日本的職業聯賽球員的收入水平比我們低很多，大量年輕球員、優秀球員都是到海外高水平聯賽效力，從而得到成長。

而中超在金元政策下，大部分年輕球員去國外並非真的為了學習，而是出口轉內銷，為了炒出好身價。國內過於安逸的足球環境阻礙了很多本該有跟高成就的球員成長。要改變就必須按照國足目前的水平，重新定製薪水，那多少合適？

​

說實在的國足踢成現在這樣，亞洲魚腩隊都贏不了都應該罰款還想領薪水呢？

各行各業的年薪收入沒有不與業績掛鉤的，唯獨國足，踢的越臭年薪越高。中國足球已經職業化多年，成績不進反退，原因雖是多方面的，但不出成績反而拿高薪，本身就違背了職業化規律。當務之急就是把過高的年薪降下來，實行基礎工資+提成的辦法，把各檔次比賽的獎金定下來，這樣就體現了收入與比賽成績掛鉤，能從根本上改變國足的精神面貌。
