巴薩1位置恐再遭危機，3大核心皆遇難題，拒絕32歲老將離隊顯無奈

2019-12-18 1,182條評論 4,879次閱讀 653人點贊

當年瓜迪奧拉之所以能夠締造出紅極一時的夢三王朝，除了與擁有梅西、亨利和比利亞等神鋒息息相關外，中場位置的強大配置顯然足以羨煞任何一家頂級豪門。由哈維、伊涅斯塔和布斯克茨組成的中場夢幻三人組，不僅鮮少受到傷病困擾就連場上狀態也是十分穩健，這使得強如法布雷加斯、蒂亞戈也依舊難獲出場時間。然而紅藍軍團這1優勢位置在哈維、小白相繼離隊後，便因難尋接班人而不斷下滑甚至一度遭遇實力危機。

​好在通過夢三、夢四王朝稱霸了歐陸足壇數年的巴薩，有足夠的財力和聲望在轉會市場上籤下心儀球員，因此拉基蒂奇、德容等中場天才先後披上了紅藍戰袍，這也使得近兩個賽季巴薩中場位置一度重新備受看好。拉基蒂奇、布斯克茨和比達爾這些老將經驗豐富，德容、阿圖爾以及阿萊尼亞等新星潛力無限表現驚艷，人員配置充足、年齡結構合理再加上皆有不俗實力，雖依舊難及「哈白布」巔峰組合但卻也足以度過危機。

​然而從本賽季的場上表現和轉會市場傳聞等層面來看，紅藍軍團的中場位置恐怕要再度遭遇到危機了，這一點從拒絕32歲老將比達爾冬窗轉會離隊便不難看出。據義大利轉會專家迪馬濟奧透露，儘管國際米蘭對比達爾有意且球員也想重返亞平寧，但巴薩方面已向比達爾經紀人重申拒絕其1月離隊。雖然32歲的智利悍將在巴爾韋德帳下只是輪換替補角色，但拒絕其離隊無疑顯露出了中場狀況不佳的無奈現狀。

​本賽季以來巴薩此前的3大中場核心皆遇到了難題，克羅埃西亞天王拉基蒂奇賽季初已淪為了絕對替補，此外其近幾場比賽的表現也出現了較大下滑受人詬病，並且31歲的年齡顯然也難以再成為中場主力核心。而夢三功臣布斯克茨的問題與拉基蒂奇有著幾分相似，年紀不小、狀態波動大已使球隊後腰位置成為了薄弱環。最後正值黃金年齡身價估值達7000萬歐元的阿圖爾，則由於體能欠佳和傷病反覆等因素無法保證出戰時間。

​因此巴薩拒絕32歲老將比達爾冬窗離隊顯露出了無奈，本賽季看似人員臃腫的中場配置其實仍欠缺即戰力，若德容、阿圖爾和布斯克茨等核心能夠扛起中場重擔，紅藍軍團無疑無需強留難成核心主力的比達爾。雖然這位智利悍將能夠為巴薩中場帶來硬度與鬥志，但要知道控制力足夠強大的紅藍「哈白布」時代，沒有比達爾這類型球員也同樣足以稱霸歐陸締造王朝。因此當巴薩還需要比達爾就證明中場換代尚未完成，球隊距離曾經的傳控足球巔峰也依舊距離遙遠！
