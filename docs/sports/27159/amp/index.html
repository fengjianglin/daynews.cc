人們看到的弗格森好像總是很暴躁，但吉格斯從小就了解他溫情一面

天天要聞
2019-12-22

弗格森在挑選小球員的時候，喜歡對孩子們的父母進行「考察」，而不僅僅是小球員本身。英國時間12月21日，吉格斯就在天空體育的節目中，透露了這一點：「他（弗格森）成功地使得孩子們贏得父母的支持而不是反對，讓小球員子的父母知道，來到曼聯的生活只會變得更好而不是更差。」吉格斯表示，爵爺的這些舉動，這讓有天賦的小球員們願意為他踢球——而他們的父母鞭策得更加起勁。人們看到的弗格森好像總是很暴躁，但吉格斯從小就了解他溫情的一面——爵爺是值得曼聯信任和依賴的人。對於自己一直訓練卻得不到合同的曼城，吉格斯是這樣解釋的：「當時，我們那支少年隊是曼城的衛星隊，我們當中的最好球員兩三個人被曼城挑去訓練，所以我小時候認為最好的球員，應該去曼城！」

小時候的吉格斯通常是在周四晚上到普拉特巷（Platt Lane）去訓練。他記得自己很享受那段經歷，不過吉格斯是個曼聯球迷！曾經有很多小球員穿著紅色的曼聯球衣去訓練，但最後被強迫脫掉它。吉格斯很難忘記那個時刻的經歷，他屬於少年老成的那一類人：「我在很小就開始希望為曼聯踢球，我和其他孩子們有些不一樣，我在曼城時他們並沒有真正打算給我一份合同，可能我已經在那裡訓練了兩三年。所以等到曼聯的球探特別是弗格森和我商量，邀請我加入紅魔，我毫不猶豫就答應了！是不是曼城的損失？這個我不敢說，但我相信絕對不是我個人的損失！」

吉格斯回憶道，當時爵爺和助手諾克斯風餐露宿，驅車出沒在英格蘭各個少年賽事的比賽現場，目的就是抓緊時間多簽下一些有潛力的苗子。有一次，在拜訪一個足球學校回來之後弗格森和諾克斯依然沉浸在工作里，他們邊討論剛才的考察工作邊做飯。爵爺雖然當時已經功成名就了，但是他依舊保持著勞動人民的本色，堅持自己與助教一起做飯。當時弗格森將一盒火柴扔到電爐上，引發了一場小規模的爆炸，電爐罷工，險些釀成火災，也成為了曼聯當時的新聞。諾克斯表示：「弗格森會出現在英格蘭大大小小的學校，他關注那些在球場上的天才少年們。」一旦知道有某個小孩嶄露頭角，爵爺就會出現在孩子的家門口，說服家長將孩子送到曼聯試訓。

爵爺喜歡和天賦高的孩子聊天，給他們建議，「引誘」他們加盟曼聯。吉格斯是這樣進入曼聯的，其實貝克漢姆也是這樣，吉萊斯皮也是這樣——他們都成為了弗格森青訓系統的第一批產品。不僅如此，弗格森還喜歡選擇小球員們的生日那一天和他們簽約，這讓他們更加高興和銘記，明白簽約曼聯是自己人生的重要時刻。在貝克漢姆生日的時候，爵爺送來的除了蛋糕還有6年的合同，在吉萊斯皮生日的時候，爵爺邀請了他們全家到曼聯基地觀看球隊比賽。比賽結束之後，吉萊斯皮異常激動地簽下了自己的名字，將自己的未來命運託付給了這個沒有血緣蘇格蘭人——儘管他以後的路分外坎坷多難，也會經歷很多失敗與挫折。
