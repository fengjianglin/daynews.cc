2-5，跌入降級區！今夏投入1.19億歐，卻創造球隊20年最差戰績

2019-12-05 0條評論 24次閱讀 0人點贊

摘要

北京時間12月5日凌晨4:15，英超第15輪迎來默西塞德郡德比，利物浦在主場5-2大勝埃弗頓。本場比賽，馬內貢獻1射1傳，奧里吉雙響，沙奇里和維納爾杜姆各入一球，埃弗頓由基恩和里沙利松打入2球。此役過後，利物浦連續32場英超保持不敗，創下了隊史127年來的紀錄。

北京時間12月5日凌晨4:15，英超第15輪迎來默西塞德郡德比，利物浦在主場5-2大勝埃弗頓。

本場比賽，馬內貢獻1射1傳，奧里吉雙響，沙奇里和維納爾杜姆各入一球，埃弗頓由基恩和里沙利松打入2球。此役過後，利物浦連續32場英超保持不敗，創下了隊史127年來的紀錄。與此同時，埃弗頓聯賽遭遇3連敗，在積分榜上也跌至降級區。

今年夏天，埃弗頓在轉會市場上繼續加大投入，一共花費了1.19億歐元引入了包括伊沃比、基恩、德爾夫、西迪貝在內的一眾實力幹將，在加上原有的里沙利松、沃爾科特、西於爾茲松、迪涅等球員，埃弗頓的球員實力比起英超諸強並不落下風。

本賽季至今，埃弗頓在英超中取得了4勝2平9負的糟糕戰績，積14分排在倒數第3。這也是太妃糖自1999年以來，在聯賽至少進行15輪後，首次進入降級區。據天空體育消息，埃弗頓主帥馬爾科-席爾瓦已經處在了下課邊緣。之後的4輪比賽，埃弗頓將迎來魔鬼賽程，他們將依次對陣切爾西、曼聯、萊斯特城和阿森納。
