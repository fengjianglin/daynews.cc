喬書亞推廣人不愧老狐狸！給喬書亞安排好4月衛冕戰，放棄WBO頭銜

2019-12-10 629條評論 2,371次閱讀 1118人點贊

【文/搏擊江湖】今天，現WBA/IBF/WBO/IBO的重量級拳王英國悍將安東尼-喬書亞（23-1，21KO）的推廣人埃迪-赫恩向媒體表示，打算安排喬書亞放棄WBO冠軍頭銜，而是在明年4月迎戰保加利亞「眼鏡蛇」庫伯拉特-普列夫（28-1，14KO），這也是一場IBF法定衛冕戰。然後那條被放棄的WBO金腰帶，會讓強制挑戰者烏西克和前世界拳王約瑟夫-帕克去爭奪。喬書亞在打完IBF衛冕戰之後，回過頭來再和烏西克VS帕克的勝者進行一場拳王統一戰，最終依然可以拿到4條拳王金腰帶。不得不說，赫恩不愧為老狐狸，這個算盤太精妙了。

埃迪-赫恩說：「目前我們正在面臨IBF和WBO的雙重壓力，IBF要求喬書亞打普列夫，WBO要求喬書亞打烏西克，但是喬書亞通常每年只打2場比賽，明年4月是第一場，那麼由於時間上的衝突，我們只能考慮一下哪個是第一位的，權衡利弊，再做決定。我們不會破壞這個規矩，我會給IBF和WBO打電話，商討此事，普列夫和烏西克，只能選擇一個。喬書亞當然也希望保留WBO金腰帶，但是由於和IBF的強制衛冕戰衝突，所以我們考慮可能會放棄WBO金腰帶，留給烏西克和帕克爭奪。」

我們先說一下這個普列夫，今年38周歲，身高1米94，臂展2米02，保持著28勝1負14場KO的不俗戰績，目前在重量級綜合排名第9位。普列夫唯一的一場失敗是被小克里琴科5回合KO出局，嚴格來說，普列夫已經不是一流行列的拳手，頂多是二流高手。他相對烏西克來說，綜合實力更弱，無法給喬書亞帶來威脅，普列夫的體能、組合拳、重拳都是相對欠缺的，唯獨刺拳還算犀利，但是僅僅憑藉刺拳怎麼可能贏下喬書亞？所以赫恩的戰略已經很明顯了，就是要暫時放棄WBO金腰帶，留給烏西克和帕克爭奪，等烏西克擊敗了帕克，有了WBO金腰帶的光環，再和喬書亞打統一戰，這樣的比賽才能賣座。

毫無疑問，喬書亞如果明年4月和普列夫對決，獲勝的可能性幾乎是百分百，這是一場幾乎沒有懸念的賽事，喬書亞可以很安全拿下比賽，對喬書亞的日後發展絕對是有好處。然後下半年再和烏西克VS帕克的勝者比賽（不出意外就是烏西克），喬書亞依然很可能收回WBO金腰帶。至於維爾德和富里，喬書亞嘴上說有信心擊敗他們，但是喬書亞在2020年是不可能和這兩位頂級高手過招的，最早也是2021年。

以上就是赫恩這隻老狐狸的戰略，不得不說非常狠辣。烏西克和帕克都是赫恩旗下的拳手，撮合比賽非常容易，喬書亞臨時避其鋒芒，先選擇軟柿子普列夫，也是上策。目前喬書亞保持著24戰23勝1負21場KO的輝煌戰績，唯獨輸給了小胖魯伊茲，但是也成功復仇，喬書亞為了保住金腰帶，肯定會每一步走得都很小心，這是完全可以理解的事情。搏擊江湖原創，歡迎分享。
