什麼叫大換血？主帥換人外援全換，福建男籃13輪2勝後迎來6戰5勝

天天要聞
2019-12-21

CBA常規賽第19輪，福建男籃賽前更換了外援，勞森因傷被拿下，36歲的杰特取而代之，然後在一片爭議聲中，這支前13輪僅2勝八一男籃的球隊，居然擊敗了吉林男籃，收穫了4連勝，同時也是近6輪比賽的第5場勝利。

這支球隊發生了什麼？第13輪比賽結束後，凱撒取代朱世龍成為球隊主帥；第16輪後，凱撒改造球隊的第一刀砍向了外援，拿下斯塔德邁爾，換上丹特·康寧漢姆；第18輪後，外援勞森被換下，取而代之的是36歲的杰特。也就是說，福建男籃用了5輪比賽，就實現了換帥及外援全換。什麼叫大換血？這應該就是了。

而就在這樣大刀闊斧改造的過程中，福建男籃居然有條不紊地拿到了4連勝，而換帥後的6場比賽，他們只輸掉了一場。福建男籃在積分榜上的排名，由換帥前的近乎擺爛，到如今與季後賽的門檻只差了2個勝場。而他們第19輪擊敗的吉林男籃堪稱本賽季最大黑馬，他們連廣東男籃、新疆男籃和北京男籃都能斬落馬下，小外援瓊斯更是以場均38扥領跑全聯盟。但這一次，笑到最後的，居然是福建男籃。

不過剛換了外援的福建男籃，這場比賽贏得並不輕鬆。首節比賽，兩隊打得非常膠著，分差始終沒有拉開，吉林單節領先了1分。第2節王哲林開始發威，幫助球隊在半場過後取得6分的領先優勢，與此同時，王哲林半場拿下16分9籃板。易邊再戰，福建男籃繼續擴大比分優勢，但在第4節吉林男籃瘋狂反撲，在最後時刻將分差追到只差2分，瓊斯、陳林堅都命中關鍵3分。但福建男籃把握住了機會，最終主場121-114戰勝吉林男籃。

拿到這場勝利之後，福建取得4連勝，換帥之後6戰5勝，昔日在山東執教時備受爭議的主教練凱撒變身神奇教練，雖然仍舊排名第15位，不過距離第12的江蘇只有2場球差距，季後賽希望大增。與此同時，這場比賽中，王哲林拿下28分10籃板5助攻4蓋帽，在CBA歷史得分榜和蓋帽榜分別超越劉鐵和哈德森。
