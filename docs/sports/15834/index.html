伊藤美誠輸不起？霸氣宣戰陳夢孫穎莎：東京見！我會成為不敗球星

2019-12-15 1,232條評論 3,449次閱讀 1664人點贊

北京時間12月15日，在國際乒聯年度總決賽女單的半決賽大戰當中，最終19歲的日本天才少女伊藤美誠，以1比4的比分輸給了中國的世界第1陳夢，無緣能夠打進到決賽當中，這場比賽其實對於伊藤美誠來說有著不小的打擊，而且當天伊藤美誠是連丟兩冠，女單和混雙全輸了。

比賽的第1局，當時陳夢開局直接轟出了9比1的攻擊波，然而卻沒有能夠把握住機會，反而被伊藤美誠連追8分追平了比分。雙方打成9比9平之後伊藤美誠把握住機會，最後以12比10絕殺了陳夢。然而這一局比賽的失利卻直接激怒了陳夢，陳夢在隨後的4局比賽裡面完全佔據主動，第2局比賽是11比8拿下，第3局11比5，第4局和第5局全部都是以13比11這樣的比分完成了驚險的絕殺，最終上演了大翻盤，以4比1將伊藤美誠淘汰出局。

雖然說在比賽裡面輸給了世界第1陳夢，並且在和陳夢交手的這三次對話當中全部落敗，到目前為止從來沒有擊敗過這個世界第一，但是伊藤美誠並不是很服氣，在賽後接受中國媒體的採訪之時，伊藤美誠也是表示，她覺得今天第1局比賽能夠拿下來給自己增加了很大的自信，但是第2局被對方拿下來之後，就掉入到了陳夢的比賽節奏當中。對於自己來說，這次比賽她的單打確實是發揮的不是太好，前面的比賽狀態波動比較大，但是能夠贏下這些高手還是不錯的。以及和陳夢的比賽結果是1比4，總體來說每局的局分都還是比較接近的，最後沒有能夠贏下來，這也是自己今後需要努力解決的課題，那就是如何在關鍵的時刻拿下關鍵分贏下比賽。

談到當時第1局比賽1比9落後還能翻盤，伊藤美誠表示自己當時沒有想太多，就那麼順理成章的去打就拿下來了，後面的幾局雖然比分接近，但沒有拿下來，還是因為自己在關鍵分的把握上出現了失誤，而且當時對手發揮的也很好，雖然這次沒有打好，但還是集中在了努力去拿每一分上。隨後伊藤美誠還明確的表示，自己確實早就給自己定下了目標，就是要成為一個不會輸給任何人的運動員，儘管這次1比4輸了，但是伊藤美誠覺得距離這個目標已經越來越近了，為了成為一個誰都不會輸的運動員，今後還是要多加訓練，努力的提升自己的實力去接近完成這個目標，向著這個目標不斷的奮進。

而隨後在接受日本媒體的採訪之時，伊藤美誠更進一步的表示，她今後要努力做到的一點就是，即便是自己狀態不好的時候也要努力去爭勝，也要有獲勝的實力，伊藤美誠更是對明年的東京奧運會做出了展望，她要努力做到讓自己在東京成為不會輸球的運動員，她想要帶著十足的信心去東京參加比賽。伊藤美誠等於是用這樣的方式向陳夢和孫穎莎等中國隊隊員宣戰，要在明年的奧運會決賽上和中國隊掰手腕。顯然儘管這次比賽輸掉了，但是伊藤美誠並不是很服氣。
