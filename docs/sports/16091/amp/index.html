CBA的一股清流，山東男籃找到小丁接班人，單場25分打臉王菲

天天要聞
2019-12-13

誰能想到呢？在本賽季遭遇「外援風波」的山東男籃竟然迎來了一波5連勝，而且這5場勝利都是在新外援為球隊試訓期間拿到的。且不談新外援曼尼-哈里斯到底是替換哈德森還是梅斯，就山東隊目前的競技狀態來看，哈里斯恐怕最近幾場比賽依舊無法登場。

在山東隊主場迎戰山西隊比賽之前，很多媒體認為這場比賽是沒有輸贏懸念的，山東男籃排名聯賽第9，而山西則是高居第4位，而且富蘭克林的個人能力完全壓過哈德森。但是，山東男籃在主場取得一場大勝，比賽在第四節近乎失去懸念。

山東男籃能夠擊敗山西取得5連勝，他們靠的不是兩位外援，哈德森在與山西的比賽中只得到22分，而梅斯只有14分，球隊取勝完全靠著本土球員的持續爆發，李敬宇、陶漢林、王汝恆，這三位球員形成了山東隊「本土三叉戟」。

李敬宇在對陣山西的比賽中個人獨得25分，不僅外線三分有足夠的威脅，持球在三分線突破一步之後的急停跳投讓山西隊防不勝防，他兩分球11投9中成為了山東隊最穩定的得分點。

值得一提的是，李敬宇的表現絕非曇花一現，在山東隊5連勝期間，李敬宇在5場比賽中分別砍下15分、18分、14分、21分和25分。可以說，李敬宇很好的彌補了哈德森在進攻端得分不足的缺陷，甚至李敬宇已經開始有新版丁彥雨航的感覺了。

要知道，李敬宇上賽季效力的正是山西男籃，而主教練王菲認為李敬宇實力不足直接把他掃地出門，山東男籃重新接納了李敬宇。而本賽季，李敬宇在山東男籃5連勝期間場均得分達到18.6分，並且用單場25分的出色表現打臉王菲。

其實，李敬宇很早就已經出道，在2008-09賽季就和丁彥雨航等人一起進入山東男籃一隊，一直到2016-17賽季一直是山東男籃的主力鋒線球員，在2012-13賽季場均得分達到12分。不過，後來李敬宇一直受到手腕傷病的影響導致狀態下滑。

現在，李敬宇又重新找回了出色的狀態，而且進攻手段更加豐富了，在丁彥雨航回歸山東男籃之前，他就是山東隊的新版丁彥雨航。

此外，除了李敬宇之外，另外兩位本土球員王汝恆和陶漢林同樣在5連勝期間功不可沒，王汝恆場均得到13.8分，陶漢林場均得到17分。李敬宇、王汝恆和陶漢林的「本土三叉戟」正在慢慢打出水準，希望鞏曉彬能夠用好這三位本土球員。
