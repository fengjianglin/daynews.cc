今天劉國梁主席出了幾身冷汗，意外把重感冒治好了

2019-12-15 995條評論 3,254次閱讀 356人點贊

【前言】特意虛編了一個故事供大家莞爾一笑，不可當真。不過，後面附圖可以讓大家看到運動員的輸贏讓很多人牽心，無論是領導層還是教練層還是運動員們都不容易呀！

聽說劉國梁主席感冒了還來現場觀看比賽以示支持，樊振東、許昕/劉詩雯紛紛想辦法要給領導治病。於是小胖跟巴西天才雨果比賽時開局就2-7、3-8落後，驚得劉國梁主席背上微微冒汗，但小胖很快就12-10拿下第一局。有人告訴小胖，劉國梁主席才風剛好點就又重新感冒了，你的藥劑量太輕治不了病。小胖咬咬牙輸掉第四局，結果有人傳話給小胖，國梁主席說了「小胖是個厚道人，要治好我的病，需要另找高人，小胖還是安心打好比賽就好了。」於是小胖只好一心一意打比賽，很快大比分4:1贏下比賽。

然後許昕劉詩雯上場了，一上來就下猛葯，比賽五局三勝，一上場就連輸兩局。果然驚得劉國梁主席一身冷汗全部出來了，頓時感覺全身舒泰。但為了鞏固療效，許昕劉詩雯通過控制節奏讓整個比賽過程都能讓人發汗！最絕的是，等劉主席的感冒徹底好了，比賽也全部打完了，並且還拿到冠軍。劉國梁深表感謝，特意親自給他們頒獎，有句許想說又沒有說：「許昕劉詩雯同學，幸虧我沒有心臟病呀！」
