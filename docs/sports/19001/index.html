最後2.4秒獻致命失誤！全場14中2！亞洲第一控衛就是個笑話

2019-12-18 250條評論 1,726次閱讀 523人點贊

文/籃郭先生

CBA的常規賽繼續進行，青島男籃與遼寧男籃進行了一場對決。這樣的比賽，從實力來看，肯定是遼寧男籃的實力更勝一籌。但很遺憾的是，遼寧男籃被亞當斯絕殺了，以101-103輸掉了比賽。數據方面，青島男籃的亞當斯貢獻49分12個籃板6次助攻4次搶斷的數據，劉傳興5分7個籃板，約翰遜21分10個籃板，焦海龍9分，趙大鵬7分。遼寧男籃的韓德君23分14個籃板，李曉旭7分，趙繼偉6分6個籃板，史蒂芬森5分，巴斯35分18個籃板5次助攻，高詩岩15分4次助攻。

這些球員，有些表現很好，有些表現很差。可以說，雖然數據是擺在那的，但效率是看不見的。而這一次籃郭先生要說的，則是一名效率很低的運動員，此人就是郭艾倫。這樣的比賽，郭艾倫打了42分鐘，14中2，三分6中1，砍下8分8個籃板7次助攻7次失誤的數據。看起來，是很全面，但看到效率，還是很扎心的。客觀地說，作為一名運動員，對郭艾倫來說，其投籃不準，沒有手感，都是很正常的事情。但是，7次失誤，多少是有點說不過去的。或者說，給人一種實力扮演卧底的感覺。

更加致命的，或許還是最後一攻一防的郭艾倫，完全被亞當斯完爆了。在進攻回合，最後時刻，郭艾倫掌控球權，然後，眼見他壓時間，眼見他啟動，眼見他丟球了。而防守者，則是亞當斯。裁判的第一反應，也是給了青島隊球權。之後，郭艾倫強烈要求看回放，雖然遼寧隊已經沒有了挑戰，但裁判從嚴謹的角度來看，還是看了回放。但是，因為實在是難以判斷，所以，裁判還是維持原判。這樣一來，對郭艾倫來說，也是一次失誤了。而這樣的失誤，自然是超級致命的。也因為這樣的致命失誤，之後的亞當斯，獲得了2.4秒的機會，對他來說，同樣完成了絕殺。

如此一來，一攻一防之間，郭艾倫被小科比亞當斯完爆了。這樣的事情，對我們來說，對中國籃球來說，的確是很尷尬的。因為，郭艾倫已經是我們最好的後衛了，而小科比亞當斯呢？在美國的後衛裡面，或許是什麼都不算的。這背後的差距，真的是巨大的。同樣的，看到這樣的輸出與效率，不少人也表示，亞洲第一控衛的說法，如今看來，應該是一個笑話。因為，其個人的表現與輸出，是完全不夠的。這一點，沒有懸念和疑問。而這裡面，最致命的還最後2.4秒的失誤。對郭艾倫來說，本來有很多種不失誤的可能性的，但是，其過於相信自己，然後的結局就有點悲劇了。這樣的事情，也讓人對郭艾倫的能力產生了懷疑。畢竟，全場7次失誤的表現，也是不合格的。

最後2.4秒獻致命失誤！全場14中2！亞洲第一控衛就是個笑話。對遼寧隊來說，他們很多球員拼搏了一整場，如巴斯，如高詩岩，遺憾的是，都被郭艾倫的致命失誤葬送了。這樣的局面，多少還是有些尷尬的。同樣的，這或許也是中國籃球的尷尬。
