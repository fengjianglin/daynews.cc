關於魯能的三大懸念，涉及一支球隊和三名球員，李霄鵬還是關鍵

2019-12-18 773條評論 4,879次閱讀 2635人點贊

中超聯賽進入冬歇期，對於魯能泰山來說，這個冬天會有諸多調整，但很多調整，還要等待足協新政的確定。消息顯示，足協會在12月下旬才公布2020賽季的一系列新政，這對魯能及中超各隊的調整都會帶來影響。要知道，2020賽季中超很可能在2月份就開賽，冬訓的整體時間會縮短。

對於魯能來說，這個冬天要確認的一件事情就是女足隊伍的組建。如今，山東女足已經開始冬訓，但未來山東女足是否會和魯能牽手，現在還沒明確的答案。中國足協要求，中超隊從2020賽季開始必須配備女足隊伍，隊伍不分級別，可以是女超、女甲和女乙。如今中超隊里，蘇寧的女足實力最強，在2019賽季拿到了四個冠軍。恆大、申花等隊選擇和高校聯手，打造女足隊伍。而對於魯能來說，如果牽手山東女足，球隊將直接征戰女超，如果有備選方案，就可能從女乙打起，具體還需要等待。

魯能泰山的人員調整同樣關鍵，本土球員方面，外租在外的球員是否回歸是個看點。姚均晟和郭田雨在2019賽季中超都有著不錯的發揮，姚均晟還進入國足，郭田雨雖然被足協禁賽，但他在武漢的發揮很不錯。兩人的租借都已經到期，之前也已經發文告別天海和武漢。他們還是魯能泰山的一員，是否會進入2020賽季的陣容，值得關注。此外，因傷休戰一個賽季的李松益有望在2020賽季復出，他可以成為中衛位置的重要選擇，提升魯能後防線的人員儲備。

外援方面，魯能現有的四名外援都在合同期內，佩萊還有一年合約，費萊尼有兩年合同，莫伊塞斯和格德斯的合同都到2022年7月。正常情況下，魯能泰山的外援陣容不會有調整，但不排除針對球隊的現實陣容配置，做出微調的可能，這還要看球隊的總結和計劃。

當然，還有一個很關鍵的因素是李霄鵬，之前各方消息顯示，李霄鵬是國足主帥的熱門人選，他和李鐵競爭。但是，足協如今依然沒有相對明確的答案，這對魯能泰山的備戰和調整都會有影響。但願，一切都儘快確定下來，包括足協的新政、新賽季的時間還有一系列的計劃，只有這樣，才能有機會在2020年衝擊更好的成績。
