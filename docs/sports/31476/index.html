遼籃絕境中拼下寶貴勝利，郭艾倫被批後爆發，奇兵單節16分救主

2019-12-23 598條評論 3,223次閱讀 128人點贊

北京時間12月22日，CBA第20輪的一場焦點比賽在福建男籃與遼寧男籃之間展開。遼寧此前遭遇兩連敗，急需一場勝利提振士氣。賀天舉此役手感火熱，首節就投中4個三分砍下16分，半場福建緊咬住了比分，兩隊打成平手。下半場，遼寧逐漸拉開比分，最終客場101-85大勝福建，終結了2連敗。

遼寧男籃近期狀態糟糕，上輪20分慘敗給山東，遭遇2連敗，排名也跌至第6，大牌外援史蒂芬森狀態糟糕，散漫比賽態度被詬病，主教練郭士強和郭艾倫也飽受質疑。福建雖然僅僅排名第15，不過他們在換帥之後強勢反彈，近6戰5勝，剛剛取得4連勝，士氣正佳。

因為前幾場發揮不佳，郭士強調整了首發陣容，史蒂芬森被放在板凳席，巴斯首發出場，上輪受傷的韓德君也沒有首發。賀天舉開場手感火熱，一上來就投中一個三分和踩線三分。福建在杰特的組織下打得非常耐心，兩人緊咬比分。郭艾倫的手感依舊不好，前5次出手全部偏出，賀天舉成為主要火力點，首節投中4個三分，得到16分。

首節遼寧憑藉賀天舉的開掛表現，首節領先了7分，賀天舉得到了全隊22分中的16分，這樣的表現許久未見。要知道，本賽季賀天舉的單場最高得分也只有15分。韓德君有傷在身，巴斯的身高也很難限制王哲林，後者屢屢殺到內線造殺傷，半場他就得到14分。

郭艾倫在第二節後半段開始發力，連得7分。第二節兩隊的進攻效率都比較高，分差很難拉開，半場兩隊打成了44平。半場結束前，郭艾倫的一次盲目1打3沒有傳球，引起了郭士強的不滿，兩人一度發生爭吵。

第三節一上來，遼寧逐漸拉開比分，帶著9分優勢進入第四節。最後一節，遼寧牢牢控制了比賽，分差越來越大，最終遼寧客場101-85大勝福建，終結了兩連敗。郭艾倫此役回暖，得到25分6助攻，其中下半場得到18分，巴斯拿到25分7籃板5助攻2搶斷、賀天舉18分4籃板，史蒂芬森9投4中，得到14分11籃板5助攻，狀態也有明顯回升。

福建方面，王哲林在下半場啞火，全場得到19分11籃板，杰特19分3籃板6助攻，卡米然19分4籃板，成為一大奇兵。全場比賽，福建只得到85分，是最近幾場比賽新低，凱撒遭遇上任後的第二場失利。

遼寧的戰績變為13勝7負，排名反超山東，升至第5，在兩連敗內憂外患的情況下，遼寧拼下了這場勝利，暫時穩住了局面。賽後點評比賽時，主教練郭士強表示，遼寧贏在了防守：「前兩場的防守沒做好導致輸球，這場在防守對方的擋拆和外線，都做得比較好，最終取得了勝利。」此外，郭士強還表現了最後時刻的出場的年輕隊員郭旭，他三分球2投全中得到6分。
