1.27億豪華陣容輸防守最差球隊，哈登遭遇史詩級包夾火箭仍然贏球

2019-12-06 0條評論 25次閱讀 0人點贊

12月6日，NBA常規賽繼續進行，讓我們來回顧一下戰況和排名。我們先來看一場東部球隊之間的對決。華盛頓奇才在主場迎戰費城76人，兩支球隊的戰績排名差距較大，很多球迷以為76人在客場能輕鬆取勝，但事實卻恰恰相反。

12月6日，NBA常規賽繼續進行，讓我們來回顧一下戰況和排名。

我們先來看一場東部球隊之間的對決。華盛頓奇才在主場迎戰費城76人，兩支球隊的戰績排名差距較大，很多球迷以為76人在客場能輕鬆取勝，但事實卻恰恰相反。首節比賽開打，局勢按照者人們的設想，76人一上來就打了一個8:1，在哈里斯命中三分後，分差已經來到了兩位數。不過奇才今天狀態不錯，他們在第1節後半段由比爾引領進攻，不斷追分，在貝爾坦斯投中三分後，分差縮小到了個位數。

第2節比賽，場上形勢風雲突變。西蒙斯完成上籃後，分差依舊在八分，但此後貝爾坦斯爆發，他連續投中4記三分，一記記三分猶如利劍一般刺穿76人的心臟，在貝爾坦斯的帶領下，奇才立刻將比分追平。此後，八村塁開始發威，他在第2節後半段連得11分，奇才將分差拉開到了兩位數，半場比賽結束時，奇才以65:55反超76人。

第3節比賽，兩支球隊展開拉鋸。比分交替上升，76人無力縮小分差，奇才也無法進一步拉開分差。第3節比賽結束時，奇才91:81領先76人10分。末節決戰，76人展開了最後的反撲在。最後一節還有4分鐘時，依靠著哈里斯的三分，分差縮小到5分。此後76人出現失誤，奇才抓住機會得分穩定局勢，最終奇才在主場119:113爆冷擊敗76人。目前，76人15勝7負排名東部第5，奇才7勝13負排名東部第12.

尼克斯在主場迎戰掘金，首節比賽開打，尼克斯狀態不錯，依靠著巴雷特的三分，他們將分差拉開到6分。不過，掘金的整體實力更為出色，他們在第1節最後兩分半鐘連續投中三記三分將比分反超。首節比賽打完時，掘金以33:27領先尼克斯6分。比賽第2節，掘金完全掌握場上主動，尼克斯方面陷入了得分荒，掘金抓住機會打出一波進攻高潮，一波17:0的攻擊波掘金立刻將比分拉開到20分以上。半場比賽結束時，掘金以67:45領先尼克斯22分。下半場比賽，掘金沒有給尼克斯機會，比賽也演變成了垃圾時間，最終掘金在客場以129:92大勝尼克斯，重新回到了西部第二的位置。

鵜鶘在主場迎來太陽的挑戰，兩支球隊都是年輕球隊，進攻能力非常強。第一節兩支球隊大打對攻，首節比賽打完時鵜鶘以40:38領先太陽兩分。第2節比賽，兩隊的攻勢依舊不減，比分交替上升，場面相當膠著，半場比賽結束時，太陽以74:70領先鵜鶘。下半場比賽，鵜鶘的進攻陷入了停滯。布克在本節狀態奇佳，他連續得分率領球隊打出一波進攻高潮，在他罰球命中後，太陽已經將分差拉開到19分。3節比賽結束時，太陽107:91領先鵜鶘16分，不過第4節比賽，榜眼鮑爾站了出來。他單節得到十分，率領球隊打出一波進攻高潮將比分追平，英格拉姆最後一攻不中，兩球隊進入加時賽的爭奪。

加時賽里，盧比奧成為了太陽獲勝的關鍵，他先是連中兩球，隨後一記精妙的傳球助攻隊友得分將分差拉開到6分。鵜鶘在最後連續三分球不進，英格拉姆也被罰下，鵜鶘最終只能在主場以132：139接受敗局。本場比賽結束後，鵜鶘6勝16負排名西部倒數第二，太陽10勝11負排名西部第8.

火箭隊客場挑戰猛龍的比賽，猛龍全場比賽對詹姆斯-哈登進行了喪心病狂的史詩級別包夾，一過半場如果有擋拆必定有兩人上前包夾哈登，但即便如此，火箭隊發揮了更好的整體性，他們全隊發揮最終力克猛龍，仍然在客場拿走勝利。

而76人今年夏天的5大陣容原本令人期待，但今天他們的1.27億豪華陣容，卻在聯盟防守最差的奇才隊身上栽了跟頭，76人外線失准，內線失誤不斷，最終爆冷不敵奇才，這樣的表現自然是令人失望的。

西部方面，火箭再次贏球之後，領先爵士的勝場差來到2.5場，而第7的森林狼距離爵士也只有1個勝場差，爵士已經被拉開掉入了第三集團，第二集團的掘金、快船、獨行俠以及火箭差距都不算太大，而第一集團，則是湖人一家獨大。
