打不上還要背鍋！CBA又一可憐哥誕生 他看錯姚明恩師了？

2019-12-02 0條評論 20次閱讀 0人點贊

摘要

客場慘敗山東隊後，上海隊主教練李秋平表示他要負主要責任。然而，在提及球隊的表現時，李秋平卻沒有主動擔責的意思，對於球隊新援鞠明欣，他就認為“未達預期”。本來，鞠明欣在加入上海隊後就機會不多，現在還得來背鍋，這對這位廣東隊舊將來說無疑是雪上加霜。

客場慘敗山東隊後，上海隊主教練李秋平表示他要負主要責任。然而，在提及球隊的表現時，李秋平卻沒有主動擔責的意思，對於球隊新援鞠明欣，他就認為「未達預期」。本來，鞠明欣在加入上海隊後就機會不多，現在還得來背鍋，這對這位廣東隊舊將來說無疑是雪上加霜。

作為昔日廣東青年隊的天賦少年，鞠明欣很早就入選了國青隊，和郭艾倫、王哲林、丁彥雨航、翟曉川等人成為同屆球員，那個時候的他跑跳能力和進攻手段都非常出色，他也因此被早早地看做是朱芳雨的接班人。

但由於廣東隊昔日的競爭過於激烈，導致鞠明欣在進入CBA前就被交易到了廣州隊，就此改變了自己的人生軌跡。不過在廣州隊，鞠明欣被大力培養，獲得了非常多的機會。首個賽季，他的場均出場時間就達到了21.6分鐘，場均得到7.8分2.9個籃板。

2015-16賽季，鞠明欣更是打出了職業生涯最好的一個賽季，當時，他場均得到15.9分4.4個籃板，三分命中率達到39%，場均投中1.1個三分球。這個賽季後，他入選了國奧隊，和胡金秋等球員一起代表球隊出戰，那個時候的他被看好能打出來。

然而，之後的三個賽季，鞠明欣在廣州隊獲得的機會明顯下降，他也因此在這個夏天選擇離開廣州隊加盟上海隊，希望能幫助上海隊獲得更好的成績，同時也獲得主教練李秋平的點撥打出更好的表現。

然而，鞠明欣這個賽季在李秋平的體系里卻近乎迷失。12場比賽，他場均只獲得13.5分鐘的出場時間，場均只能得到3.8分，三分命中率只有13%，場均只能投中0.2個三分球，但場均犯規卻達到2.6次，基本上，鞠明欣的各項數據都是職業生涯最差。

但在場上，鞠明欣是非常賣力的，只是他的進攻優勢並沒有被發揮出來，在李秋平的體系里，他這個賽季的最高得分只有8分，但在為球隊做出巨大犧牲後，鞠明欣非但沒有獲得李秋平的信任，反而被質疑表現不達預期。

李秋平的甩鍋行為也讓人對他很是失望，作為主教練，他需要負責把鞠明欣的天賦兌現出來，但到現在，除了上海隊戰績糟糕外，鞠明欣也徹底迷失，作為主教練，李秋平顯然更應該自省而不是點名批評球員，畢竟李秋平做得很不好。

就鞠明欣而言，他應該對李秋平是失望的，這個賽季，他之所以加盟上海隊，除了合同外，他顯然還希望能在冠軍教頭李秋平的手下獲得進步，然而，現在的一切和他的預想都是相反的，而他也看錯了姚明的這位恩師。
