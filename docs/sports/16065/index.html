知恥而後勇！國足青年隊2：1取首勝，終結近期各級別國足三連敗

2019-12-13 118條評論 4,998次閱讀 1197人點贊

北京時間12月13日，「一帶一路·七彩雲南」國際足球邀請賽第二輪的比賽在昆明如期進行。最終，中國男足U20梯隊2：1擊敗塞爾維亞超級聯賽霸主貝爾格萊德紅星的二隊，取得賽事首勝。本場比賽取勝，不僅國足U20梯隊的備戰開了個好頭，同時也洗刷了近期國足各級別球隊接連慘敗的恥辱！貝爾格萊德紅星在塞爾維亞超級聯賽可謂是霸主級別的存在，聯賽冠軍、歐冠名額基本都被其壟斷。本次參加邀請賽，貝爾格萊德紅星的二隊也是全主力出征迎戰。因此，U20梯隊的小將們能收穫一場勝利屬實不易！

本場比賽，發揮最亮眼的球員莫過於申花小將周俊辰。上半場比賽，周俊辰先是連過數人巧射死角破門，幫助中國隊1：0領先，打破比賽僵局。隨後，周俊辰又妙傳助攻隊友幺旭辰打進一球，把領先優勢擴大到了2：0！下半場比賽，對方憑藉速度優勢和身體優勢打出快速反擊，扳回一球，比分來到2：1，但是面對狀態火熱的中國隊，貝爾格萊德紅星最終沒能扳平比分，比賽以2：1結束，中國隊取得了本屆賽事首勝，同時也終結了國足各級別梯隊的3連敗。此前，國足選拔隊在東亞杯1：2輸給日本，國足奧運隊0：1輸給敘利亞國奧隊，國足U20梯隊也0：3輸給了莫斯科斯巴達的二隊。如今國足、國奧的比賽還沒來到第二輪，國青隊的小夥子們率先用一場勝利吹響了國足各級別梯隊反擊的號角！

值得一提的是，本場比賽中閃耀全場的小將周俊辰可謂是王者歸來。去年遠赴泰國參加曼谷杯比賽的周俊辰，因為紀律原因被足協槍打出頭鳥，重罰禁賽1年。遭到重罰後，周俊辰開始了漫長的救贖之路。在足球方面，周俊辰每天保持著高強度的訓練，不斷加練。此外，周俊辰還邊訓練邊繼續學業，此前他已經拿到了同濟大學的錄取通知書。由於良好的表現和認錯態度，周俊辰被足協提前解禁復出，回到了申花俱樂部。本賽季，周俊辰代表申花在聯賽中出場6次。雖然出場時間比較短，但還是能看出他不凡的實力和極高的天賦。假以時日，他定能成為申花、國足的棟樑球員。

​

知恥而後勇，周俊辰遭禁賽1年後王者歸來，國青隊也在開門黑之後取得首勝。希望國足成年隊、奧運隊都能學習小弟們的這種精神，打好接下來各自的比賽。
