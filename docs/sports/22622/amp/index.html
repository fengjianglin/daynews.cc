“也許利物浦的球衣更重”——蒙特雷主帥對戈麥斯的犯規做出反應

天天要聞
2019-12-19

戈麥斯因扳倒對方射手洛赫里奧·富內斯·莫里而吃到黃牌，但在下半場手球時又逃過了第二張黃牌

蒙特雷主教練安東尼奧·穆罕默德堅持認為喬·戈麥斯應該被紅牌罰下，並暗示這名後衛只是因為他是利物浦球員才免於紅牌。

周三晚上，在多哈舉行的國際足聯俱樂部世界盃半決賽中，戈麥斯在上半場扳倒了進球球員羅格里奧·富內斯·莫里，因此被出示黃牌。

這名紅軍後衛後來在接球時躲過了智利裁判羅伯托·托巴爾的第二張黃牌，這激起了默罕默德在邊線上的憤怒。

這也在蒙特雷老闆和尤爾根·克洛普之間引發了一場醜陋的衝突，克洛普離開了他的技術領域，去規勸他的對手，並模仿他揮舞想像中的牌。

兩名教練都被出示了黃牌，穆罕默德說:「對我來說，這是一張紅牌，尤其是第一次犯規後的第二次犯規。」

「所以我和裁判談了談。也許利物浦的球衣更重，所以利物浦球員沒有被罰下。」

菲爾米諾在傷停補時階段的制勝球使蒙特雷在加時賽中錯失了不少機會，儘管利物浦控制了比賽。

穆罕默德說:「這是一場艱難的比賽。我們盡了最大的努力來保持90分鐘的專註，但是比賽是在細節上決定的。

「在最後幾分鐘，我們的注意力沒有集中好。我帶來了一個速度很快的球員，因為我認為比賽將進入加時賽，但後來菲爾米諾得分了。」

「我們在平等的基礎上和利物浦踢了一場比賽。這可能是這項賽事中最好的一場比賽，但我們必須進球才能晉級，我們的守門員在挽救許多機會方面表現出色。」

「我現在很失望，但我為我的球員和球隊感到驕傲，我們真的讓我們的城市和我們的國家感到榮耀。」

「我們輸掉了這場比賽，但我們擁有榮譽和尊嚴。」

默罕默德也被問到他對弗拉門戈有什麼建議，利物浦將在周六的決賽中對陣弗拉門戈。

「弗拉門戈必須效仿我們在這場比賽中的表現，」他說。

「他們必須集中精力，不允許利物浦球員傳球和反擊。」

「這是我們為這場比賽所做的計劃，我認為我們的防守很出色。」
