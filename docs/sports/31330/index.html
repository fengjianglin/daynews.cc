日本主帥奧運第3人已有安排！理解水谷隼1行為，國乒難選迎考驗

2019-12-24 1,581條評論 2,567次閱讀 377人點贊

北京時間12月23日，日本媒體消息表示對日本主帥倉島洋介的採訪中，明言對於奧運會第三人也就是團體名額心理某種程度上有了安排！並對水谷隼近期的行為和表現表示理解！另外相比日本的逐漸明朗化不同，國乒還是非常難有斷絕，並且將再一次迎來考驗！

據悉，隨著時間的推進，日本的TOP12人比賽已經結束，最終老將水谷隼提前退賽，而森園政崇拿到了世乒賽團體的名額，女隊方面則是平野美宇獲得了這一名額，此外對於日本而言，世乒賽的選拔已經暫時告一段落，但未來奧運會的最後一個名額再度成為除了已經獲得名額的張本智和、丹羽孝希、伊藤美誠、石川佳純之外所有日本主力的關注點！

而就此，日本媒體也對日本的男隊主帥倉島洋介進行了採訪，首先倉島洋介表示水谷隼的腰痛惡化導致了提前推送，目前需要靜養2周的時間，另外，對於水谷隼，倉島洋介直言：長時間一直忍著腰疼一邊要為隊伍效力。奧運會競爭中在沒有自己最佳狀態的情況下輸掉了比賽，我想他也一定很多壓力！不難看出作為日本主帥對水谷隼這位日本老將還是十分的寬容並且理解！

隨後，倉島洋介還透露出水谷隼目前從競爭奧運會的沉重壓力解脫出來後，腰傷恢復的很快，並且希望水谷隼首先要做的就是好好治理！另外談到很多人都關係的奧運會陣容問題時倉島洋介透露：心裡某種程度上有了安排，但直到1月6日之前還會好好考慮！可見這位男隊主帥已經有了取捨，但隨後倉島洋介又說到：本次的top比賽中森園政崇拿到了世乒賽資格，神巧也等人打的也很激烈，儘管世乒賽和奧運會沒有直接的關係，但每個人都認為在教練的內心有一種「有我」的感覺！直言自己可以感受到這種心情！

另外一方面，和日本奧運會陣容日趨明朗化不同，國乒的奧運會陣容反而是依然十分模糊，首先男隊方面，誰能抗下單打的兩個名額如今看來很難有所斷絕，無論是馬龍、許昕還是樊振東，都十分有機會，而女隊方面更為複雜，劉詩雯、丁寧、陳夢、孫穎莎等人也很有機會，最終如何選擇，恐怕也只能通過後續比賽的考驗來證明自己！

而即將在2020年1月1日正式開打的地表最強12人賽事可以說是國乒新年的第一場比賽，對於所有參賽選手而言都無比重要，而且新的地表最強12人賽事還有新的賽事規則以及混雙項目的增加，對於國乒而言這也是奧運會陣容決定前非常重要的一戰！
