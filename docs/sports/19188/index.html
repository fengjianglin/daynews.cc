6個月6冠，狂升98名！日本出了1對世界第一，1人打破中國16年壟斷

2019-12-16 376條評論 1,491次閱讀 388人點贊

北京時間2019年12月16日消息，國際乒聯年終總決賽已經順利落幕，經歷了4天的膠著大戰，5個項目的冠軍最後被中國隊和日本隊瓜分，國乒照樣取得優異成績，奪取4冠，然而丟失的一冠卻值得讓人深思和探討，國乒似乎又增添了一位勁敵，2次爆冷擊敗國乒世界冠軍、前世界第一朱雨玲，還不是伊藤美誠。

可能有球迷會覺得這樣的說法比較苛刻，感覺是要國乒每次出征都得包攬才滿意。這裡並無此意，一方面丟冠得找原因，二來我們也需要通過比賽正視對手的成長和衝擊。今天要說的這位叫長崎美柚，2019賽季，幾乎是她崛起之年。

今年17歲的長崎美柚，到去年年底還沒太大動靜，但今年一年，我們國乒球迷似乎聽到了她不少消息。眼下，長崎美柚已經成為日本斷代計劃重點培養的球員之一。本賽季，長崎美柚前前後後拿下了4個巡迴賽冠軍+世青賽雙冠以及年終總決賽冠軍。（7冠）

首先是女雙方面，長崎美柚和木原美悠這對組合，實力越來越強，成績也越來越好，本賽季，2人分別在斯洛維尼亞公開賽冠軍、克羅埃西亞公開賽冠軍、亞青賽冠軍、奧地利公開賽冠軍、世青賽冠軍，總決賽，6個冠軍。

德國公開賽女雙決賽，她們1-3不敵韓國奧運組合田志希和梁夏銀，轉眼到了年終總決賽，這對小姑娘實現了強勢復仇，送上3-0吊打奪冠！並且半決賽遇到國乒世界冠軍組合孫穎莎和王曼昱，在大比分0-2落後的情況下，我們看到了這對日本小姑娘依然很從容，打出好球也在笑，以這種心態，連扳3局上演大逆轉，爆冷擊敗孫穎莎和王曼昱。

女雙世界排名，賽季初長崎美柚和木原美悠排名第46名，5月份創造了最低排名99名，也就從5月開始，經歷了半年左右的比賽，這對日本組合已經成為了世界第一，狂漲98名，力壓孫穎莎和王曼昱，日本似乎很開心，終於出了這麼一對世界第一。這隻能說2位小將確實有點厲害了。

女單方面，由於木原美悠才15歲，成人組的比賽還有待提高，但是長崎美柚就不得不說了。咱國乒世界冠軍、昔日同樣被稱為天才少女的朱雨玲，2次遇見她2次輸球，關鍵整場比賽更可怕的是，長崎美柚在場上依然保持笑容，就跟玩似的。一次算偶然，2次絕非偶然，而是有了實力！

世青賽成就女單和女雙冠軍後，長崎美柚受到了日本媒體的追捧。要知道世青賽從開辦至今，16年來都是中國隊奪取女單冠軍，而長崎美柚這次硬生生打破了中國壟斷！她接受採訪時提到了陳夢、孫穎莎等國乒猛將，沒想過自己的名字也會被記在世青賽獎盃上，眼下，東京奧運她是沒戲了，但她準備為2024年的巴黎奧運會著手準備。眼光夠長遠的。

對於有能力贏中國隊的選手，日本隊必然會重點培養，國乒在享受勝利喜悅的同時，仍然不能掉以輕心！
