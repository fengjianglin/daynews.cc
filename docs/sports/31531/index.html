阿聯再創一賽季新高成廣東男籃第一“外”援，全明星票王沒跑了

2019-12-23 1,170條評論 4,522次閱讀 755人點贊

12月22日，CBA冬至大戰如期開戰，廣東男籃客場116-90大勝廣州男籃，拿下同城德比。此役，廣東男籃全隊6人得分上雙，但最穩的，還要屬拿下全隊最高21分和全場最高20籃板的易建聯！

中圈跳球，阿聯就略勝前隊友尼克爾森一籌博得首個球權，第一次進攻，又是他低位錯位造成了亨特的犯規，給對方雙外一個下馬威。就知道雙外不肯善罷甘休啊，首節還剩3分鐘的時候，阿聯單擋掩護掛住亨特，順下之時又超車尼克爾森，接到了威姆斯的傳球之後直接摁著劉豐搏空接暴扣，以一打三險些打出2+1；兩罰全中後，阿聯連續三次點下進攻籃板，再次強打劉豐搏造殺傷，單節狂攬9個籃板。

首次空接沒進，不要緊。第二節還剩3分多鐘的時候，阿聯防守反擊中接馬尚助飛，終於完成空接，這一扣，上癮了。第三節一上來，阿聯罰球線策應後突然下順，再次面對亨特完成雙手暴扣。末節開打，胡明軒強突不中，易建聯旱地拔蔥摘下進攻籃板，摁著身下的李炎哲直接蛟龍出海打出暴扣2+1！

而這一球，不僅讓廣州男籃主場球迷齊聲高呼MVP，也讓擔任解說嘉賓的蘇群老師感嘆：「能力確實是強啊」，柯凡也忍不住讚歎道：「這就屬於臉對臉教做人啊」。四節戰罷，阿聯出戰33分鐘輕取21分20籃板外加1助攻，皆為全隊最高且無一失誤，7個進攻籃板和總籃板，同樣是他賽季新高，全場最高4次暴扣更是讓球迷倒戈，展現本土第一人的統治力。別忘了，此役馬尚和威姆斯也才分別得到13分和16分。

​本賽季，阿聯場均轟下20.8分11.0籃板1.4助攻1.5搶斷1.6蓋帽，並且新晉歷史總得分王，數據和實力擺在這兒，在目前的全明星賽票選中一馬當先也就沒什麼意外了。正如杜鋒指導曾說的「他不僅是我們廣東隊的驕傲，可以說也是整個中國籃球的寶貴財富，值得所有年輕球員去學習。」

（一星）
