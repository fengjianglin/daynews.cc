全國70多位體育司局長齊聚恆大足校 點贊恆大青訓辦學成果

天天要聞
2019-12-19

12月16日上午，國家體育總局青少司司長王立偉、全國體校聯合會理事長韋迪與北京、上海、廣東、山東、遼寧、重慶、新疆、西藏等28個省區市體育局的七十多位負責人齊聚恆大足球學校，聽取恆大青訓彙報，觀看小球員們足球訓練和文化課學習，深入了解恆大足校育才成才之路。

70多位體育司局長齊聚恆大足校

調研中，王立偉對恆大足校「全精英、全免費」的培養機制給予肯定和好評。他強調，恆大對足球青訓持續大投入體現了大企業的社會責任感，希望恆大足校在深耕青訓的同時，進一步強化與全國校園足球青訓的交流合作，共同推動各地青少年足球水平的提高。

「全精英全免費」一條龍青訓備受關注

恆大足球學校2012年開辦，2014年又設立恆大西班牙分校，以「振興中國足球 培養足球明星」為辦學目的，2018年開始實施「全精英、全免費」培養機制，入讀精英生訓、賽、學、吃、穿、住等費用全免，人均每年投入約50萬元。恆大足校實施5+5「一條龍」青訓體系，9-13歲學生在國內接受為期5年的皇馬外教團隊特訓，13歲以上的優秀學生每年有25名被選拔到恆大西班牙分校，展開為期5年的留學深造。建校辦學七年來，恆大已累計投入超26億。

聽取恆大青訓彙報

16日上午，體育總局青少司司長王立偉、副司長卿尚霖，全國體校聯合會理事長韋迪、常務副理事長劉亞群，中國足協副秘書長戚軍以及北京市體育局副局長張霞、上海市體育局副局長宋慧、廣東省體育局副局長麥良、遼寧省體育局副局長周鐵民等人，與國內28個省區市體育局有關負責人一道調研恆大足校，而且大多數嘉賓都是第一次到訪。看到環境優美、設施一流的校園，大家第一反應都是「十分震撼」，不少負責人表示，在這裡接受恆大引進的世界頂級西班牙皇馬俱樂部青訓教練團隊特訓，相信孩子們一定能打下堅實基礎，實現自己的足球夢想。大家對恆大足校「全精英全免費」培養機制尤其關注，稱「恆大在青訓領域大手筆的投入，令人十分敬佩」，恆大的做法兼具責任心和公益心，期待能為國家培養出優秀的足球人才。

無獨有偶，原中國女足主教練馬元安今年第一次走進恆大足校的感受也是「非常震撼」，他表示「我們當隊員的時候是在泥地上踢球，進業餘體校踢球也沒有教練教，裝備和訓練條件等方方面面都不能比」，期盼恆大能培養出好苗子，代表國家隊出戰並取得好成績。

恆大足校的人才產出已現端倪。2019賽季，足校畢業生布格拉汗、羅涵博文和蔡明民3人隨廣州恆大獲得中超八冠王；以2001年齡段學生為班底的恆大二隊奪得青超U19A組冠軍，恆大三隊則擊敗越南和泰國勁旅奪得東盟杯冠軍；恆大U13隊奪得足協杯和冠軍杯「雙冠王」。2012年建校至今，足校已獲75項重大賽事冠軍，398人次入選各級青少年國家隊，入選恆大俱樂部梯隊球員近百人。

奪得2019青超聯賽U19A組冠軍（資料圖）

引進來更走出去創辦高水平國際比賽獲讚譽

當天，王立偉和各省市體育系統的負責人還集體觀看了恆大小球員們足球訓練，聽取了恆大在西班牙馬德里創辦U14-U17系列高水平國際比賽的情況介紹。

各省區市體育局負責人現場觀摩

恆大足校負責人表示，從2019年開始，恆大每年投入5000萬元人民幣，為留學世界足球強國西班牙的恆大足校精英球員創辦了「恆大杯」馬德里足球冠軍賽，從U14到U17四個年齡段的學生每年和當地同年齡段強隊交手達60場，周周都有比賽，以賽帶練、以賽促練，通過聯賽制的高水平實戰，加速培養年輕人才。

現場的各省市體育系統負責人一致認為，青訓「引進來、走出去」符合足球發展的客觀規律，也是發展振興中國足球的必然路徑。恆大足校以生均年投入50萬元的高標準，選拔優秀學生留學海外，同時盡最大限度多踢高水平國際比賽，堅持下去，一定能加快提高小球員技戰術水平和實戰能力。有負責人稱，希望馬德里冠軍賽成為「造就中國足球未來之星的搖籃」。

2019「恆大杯」馬德里足球冠軍賽（資料圖）

馬德里足協主席帕科·迭斯在2019年「恆大杯」馬德里足球冠軍賽開賽時也曾公開點評：「恆大創辦這項比賽是一個十分有魄力的舉動，能幫助中國的小球員提高競技水平，體驗西班牙足球精髓，還能增進西班牙和中國兩國青少年之間的交流。」而中國足球名宿李應發也認為，多打比賽很重要，而且要多打高質量的比賽。國外職業運動員不打就不打，要打比賽就像正式比賽一樣，做到比賽訓練不脫節，真刀真槍鍛造人才。

高考成才率達96%引體育局長豎大拇指

在恆大足校，王立偉、韋迪和各省區市體育局負責人還詳細了解了恆大足校文化課教學情況，尤其是足球訓練和文化學習如何做到統籌安排。

參觀恆大足校學生「第二課堂」

恆大足校負責人介紹，學校開齊開足文化課，學生參加當地教育局聯考，小學對標廣州教育強區越秀區，通過打造高效課堂和高效課程的「雙高」文化培養體系，確保孩子們的文化課學習質量。來訪嘉賓表示，恆大足校文化課抓得緊，值得其他體育學校借鑒。

了解恆大足校學生學習情況

還有不少人關心，足球淘汰率高，除了為職業俱樂部輸送人才，恆大足校的學生還有什麼出路？恆大足校負責人介紹，足校已為社會培養輸出了3000多名各層次足球人才。前四屆高三畢業生中，有260人考上了大專以上高等院校，成材率達96％。僅2019年，55名應屆畢業生有53人考入大專以上高校，其中24人還入讀同濟大學、北京體育大學、華南理工大學等重點本科院校。聽到這一數據，現場一些體育局負責人直接豎起了大拇指。
