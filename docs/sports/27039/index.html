他昨晚在英超成就三冠輝煌，卻不如穆帥一場復出發布會熱度高

2019-12-22 3,490條評論 4,825次閱讀 1430人點贊

利物浦奪得世俱杯冠軍

他是當今足壇的性格主帥，被利物浦球迷稱為「搖滾教練」，他是尤爾根·克洛普。

北京時間12月22日凌晨01:30，他帶領利物浦在多哈1比0擊敗弗拉門戈，奪得世俱杯冠軍，成就了英格蘭第一個歐冠、歐洲超級盃、世俱杯三冠王。

尤爾根·克洛普出生在賓士、保時捷汽車的發祥地，德國第六大城市斯圖加特，他在德乙的美因茨踢了12年職業聯賽，從未踏足過德甲，成為教練之前，他是一名電視足球評論員，他當年獲得法蘭克福大學體育科學專業的大學學位畢業論文是競走（怪不得渣叔執教的球隊這麼能跑），除了生於斯長於斯，克洛普於2000年接過克勞琛（對，就是我們前中國國青隊主教練）的教鞭執教自己的母隊美因茨，當時還沒有考到甲級教練證書的克洛普卧薪嘗膽4年終於帶領美因茨升上德甲，用4年去打造出一支高水平球隊也成為克洛普執教水平的名片（利物浦2015年聘請克洛普，4年剛好又到了開花結果的時候），而更被球迷津津樂道的是他在多特蒙德期間把德甲霸主拜仁拉下神壇，自己也成長為一名世界頂級教頭。

克洛普昨晚成就英格蘭輝煌三冠

克洛普的足球激情四溢，曾4球把穆里尼奧的皇馬斬於馬下，如果穆帥代表的是無視場面，但求一勝的功利足球，那麼似乎克洛普崇尚進攻和高位逼搶的搖滾足球更符合國人的口味 ，但現實是這樣一位性格主帥，無論在國外抑或國內，熱度都遠不如穆里尼奧，作為一名看球多年的球迷，實在猜不透是什麼原因，除了克洛普的冠軍來得有點遲之外，大概最合理解釋就是：克洛普形象邋遢，沒有穆里尼奧帥氣！為什麼這麼說，因為克洛普當年就因為自己的形象過於邋遢而錯過執教漢堡和拜仁的機會！

10分領跑的英超聖誕冠軍，利物浦可不是95-96賽季天亮前尿床的紐卡斯爾，可以預見的是，當這個賽季結束，專業陪跑27年的利物浦，在這個男人的帶領下，會完成球隊歷史的另一個使命：重奪英格蘭超級聯賽冠軍頭銜！
