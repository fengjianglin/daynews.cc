金球獎科帕獎均榜上有名，韓國球員已登堂入室

2019-12-03 0條評論 23次閱讀 0人點贊

摘要

文/羊城派記者 徐揚揚今晨在巴黎沙特雷劇院舉行的金球獎頒獎典禮，韓國足球大出風頭。效力於英超勁旅熱刺

文/羊城派記者 徐揚揚

今晨在巴黎沙特雷劇院舉行的金球獎頒獎典禮，韓國足球大出風頭。效力於英超勁旅熱刺的前鋒孫興慜在金球獎排名第22位，成為史上排名最高的亞洲球員；在西甲瓦倫西亞踢球的18歲小將李康仁也入選科帕獎（U21最佳球員）10人候選名單。兩位韓國球員登上金球獎和科帕獎榜單，標誌著韓國球員已在世界足球的殿堂登堂入室。

孫興慜為韓國足球贏得榮譽

球迷仍為孫興慜鳴冤叫屈

這是時隔12年後再次有亞洲球員進入金球獎30人候選名單。孫興慜是繼中田英壽、稻本潤一、薛琦鉉、朴智星、中村俊輔和尤尼斯後，第七位入選該候選名單的亞洲球員。這次的排名也讓孫興慜力壓本澤馬、特爾施特根等大牌球星。在此前入圍金球獎候選的六名亞洲球員中，只有尤尼斯獲得選票，最終排名第29位，孫興慜則刷新了紀錄。

儘管如此，仍有很多球迷在《法國足球》官方網站留言為孫興慜鳴冤叫屈，「他的排名應該更高點」，「他應排到前15名」，「孫的排名低了，他帶領熱刺進入到了歐冠決賽，為什麼像奧巴梅楊、塔迪奇等人的排名比他高」之類的聲音不絕於耳，這也彰顯孫興慜在當今足壇的地位。

孫興慜成為亞洲足球的傑出代表

李康仁違約金8000萬歐元

李康仁是參加一個《飛吧，小射手》足球選秀節目成名的，6歲就被譽為」足球神童「。李康仁在8歲時被家鄉球隊仁川聯邀請進入U12梯隊，2011年通過試訓後成為瓦倫西亞U10梯隊中的一員。

西班牙的足球沃土無疑加速了李康仁的成長。今年9月，他為瓦倫西亞一線隊射入首個西甲進球。在國家隊，李康仁作為絕對主力幫助韓國國青隊殺入今年U20世界盃決賽，並榮獲最佳球員的金球獎。

李康仁是韓國足球的後起之秀

如今《轉會市場》球員身價榜中，李康仁身價高達2000萬歐元，已是中國球員武磊的兩倍。西班牙足協想歸化李康仁，巴塞羅那、曼聯、曼城、阿賈克斯也曾想購買他，瓦倫西亞為了留住這顆「明日之星」，為其違約金標價高達8000萬歐元。如今李康仁入選科帕獎10人候選名單，無疑將繼續推高其身價。
