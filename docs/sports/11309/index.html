20分鐘僅得兩分，孫悅狀態低迷！18分鐘12+5+2，王哲林MVP本色！

2019-12-10 1,297條評論 4,477次閱讀 2001人點贊

北京時間12月9日晚上19:35，CBA常規賽迎來福建與北控的對決。福建男籃上一輪在客場以124—90的大比分擊敗了上海隊，迎來了第三場勝利。三分神射手陳林堅打出了本賽季開始以來個人最為出色的一場比賽，出戰28分鐘，三分球12投8中，拿下了36分6籃板4助攻的數據。北控上一輪比賽以90—95不敵山東，目前以7勝7負的戰績排在積分排行榜第11名。

首節上來，北控這邊，王子瑞，王少傑，弗格，連續中距離跳投，福建這邊，王哲林連續中距離跳投，陳林堅，卡米然，勞森三分，比分來到14：8，進入官方暫停。暫停回來，弗格罰球線取分，中距離跳投，孫桐林，許夢君三分入網，湯普森連續中距離跳投。福建這邊，卡米然三分球，中距離，勞森，謝亞財，殺傷內線，何重達三分，首節戰罷，27：20，福建暫時領先7分。

次節開戰，福建這邊，小斯連續跳投命中，勞森連續突破上籃，陳林堅三分，王哲林跳投。北控這邊，湯普森中距離，弗格不斷跳投，殺傷內線，比分來到38：33，北控暫時落後分，進入暫停。暫停回來，孫悅，薩卡基尼，弗格，湯普森，跳投命中，福建這邊，王哲林連續中距離，扣籃，勞森突破，跳投，小斯扣籃，比分來到51：45，北控暫時落後6分，進入暫停。暫停回來，弗格罰球線取分，王哲林灌籃得手，比分來到53：46，北控暫時落後7分，進入下半場。

北控:弗格18分2籃板2助攻1搶斷，湯普森12分7籃板1搶斷，桑尼4分6籃板，王子瑞2分2籃板1助攻,孫悅2分3籃板3助攻，王少傑2分1籃板，孫桐林3分。

福建:勞森13分1籃板6助攻，王哲林12分5籃板2助攻，卡米然8分1籃板，陳林堅6分3籃板1助攻，小斯8分5籃板，何重達4分1籃板1助攻，謝亞財2分。
