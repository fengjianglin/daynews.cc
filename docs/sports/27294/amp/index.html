格策破門克拉馬里奇制勝球，多特客場遭逆轉1-2霍芬海姆

天天要聞
2019-12-21

虎撲12月21日訊 北京時間12月21日凌晨03:30，2019-2020賽季德甲聯賽第17輪比賽正式打響，多特蒙德客場挑戰霍芬海姆。上半場，阿什拉夫助攻格策破門，斯科夫自由球擊中橫樑，小阿扎爾遠射擊中橫樑；下半場，多特蒙德後防失誤，阿達米揚替補破門扳平比分，克拉馬里奇頭槌上演逆轉。最終，多特蒙德客場1-2不敵霍芬海姆。

開場第3分鐘，貝布前場左路帶球突入禁區右腳推射，皮球被比爾基側身撲出，主裁判示意，貝布越位在先。

第10分鐘，阿什拉夫前場右路帶球橫傳中路，格策門前搶點射門並未碰到皮球。

第15分鐘，小阿扎爾開出前場左側角球至禁區內，胡梅爾斯禁區內搶前點頭槌攻門，皮球高出橫樑。

第17分鐘，小阿扎爾前場右路帶球送出直塞球，阿什拉夫邊路套邊插入禁區右腳橫傳，格策門前搶點右腳推射破門，霍芬海姆0-1多特蒙德。


第20分鐘，多特蒙德快速反擊，桑喬前場左路內切突入禁區右腳推射，皮球偏出右側立柱。

第26分鐘，斯科烏左腳自由球爆射，皮球擊中橫樑彈回，貝布頭槌補射，皮球被比爾基沒收。

第34分鐘，多特蒙德快速反擊，布蘭特前場右路帶球直塞，阿什拉夫插入禁區面對鮑曼右腳推射，皮球被鮑曼用腿擋出。

第39分鐘，小阿扎爾前場右路帶球大禁區前沿右腳抽射，皮球被鮑曼撲了一下後擊中橫樑彈出。

上半場，多特蒙德客場1-0領先霍芬海姆。

易邊再戰第50分鐘，格策前場直傳，桑喬禁區中路擺脫防守右腳橫敲，布蘭特禁區內得球右腳推射，皮球被鮑曼撲出。

第56分鐘，多特蒙德快速反擊，阿什拉夫前場右路帶球突入禁區右腳小角度低射，皮球滑門而過。

第60分鐘，桑喬前場左路內切突入禁區被許布納鏟翻在地，主裁判未予判罰點球。

1分鐘後，霍芬海姆後場長傳過頂，鮑姆加特納插入禁區右腳墊射，皮球偏出左側立柱。

第68分鐘，布蘭特大禁區弧頂外右腳斜傳，阿什拉夫禁區內右側得球右腳橫傳，阿坎吉跟上並未碰到皮球。

第79分鐘，霍芬海姆前場右路右腳傳中，皮什切克倒地解圍失誤，貝布禁區內左側得球扣過防守右腳射門被比爾基封堵，替補出場的阿達米揚門前補射扳平比分，霍芬海姆1-1多特蒙德。


第85分鐘，多特蒙德前場右路傳中，替補出場的帕科門前搶點並未碰到皮球。

第87分鐘，克拉馬里奇前場帶球分球，阿達米揚前場左路左腳傳中，克拉馬里奇甩頭破門。主裁判在經過與VAR視頻裁判溝通後，判定克拉馬里奇進球有效，霍芬海姆2-1多特蒙德。



最終，多特蒙德客場1-2不敵霍芬海姆。

出場陣容：

霍芬海姆（433）：1-鮑曼；29-斯科烏、21-許布納、6-諾德維特、16-魯迪；8-蓋格爾（65′，37-洛卡迪亞）、11-格里利奇、18-薩馬塞庫（46′，23-阿達米揚）；14-鮑姆加特納（78′，3-卡德扎貝克）、9-貝布、27-克拉馬里奇；

多特蒙德（343）1-比爾基；2-扎加多、15-胡梅爾斯（46′，26-皮什切克）、16-阿坎吉；14-舒爾茨、19-布蘭特、33-魏格爾、5-阿什拉夫；7-桑喬、10-格策（85′，9-帕科）、23-小阿扎爾（46′，34-拉爾森）；
