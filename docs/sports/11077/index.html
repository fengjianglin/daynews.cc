心疼！29歲恆大悍將國足首秀被打懵：怒吼隊友實屬罕見

2019-12-11 3,006條評論 4,205次閱讀 1459人點贊

在中超聯賽結束後，李鐵國足的國腳們迎來了東亞杯的挑戰。昨晚的比賽中，國足在東亞杯首秀中與老對手日本隊交鋒，面對日本派出的「三隊」陣容，國足依然難以抵擋，在比賽中1比2告負。多名球員在這場比賽中迎來國足首秀，其中就包括29歲的恆大國腳劉殿座，他在防守中多次被隊友坑慘，首秀就被狂轟2球，而他在2球落後時的1個罕見鏡頭更是令人頗感心疼！

這就是本賽季跟隨廣州恆大奪得聯賽冠軍的劉殿座，這名中超賽場表現穩健的國門，在這場比賽中卻被防線上的隊友們坑慘。上半場比賽第15分鐘的時候，同樣上演國足首秀的武漢卓爾後衛明天在右路防守時出現失位，日本隊輕鬆完成突破並傳中，而梅方在門將之前完成解圍，但是解圍動作還是驚出了劉殿座一身冷汗！

很快，國足就在上半場丟掉了第1個球，國足後衛的防守被對手輕鬆打穿，劉殿座也對進球無能為力。而下半場比賽中，國足為了加強進攻，防線上更是漏洞百出。在比賽第58分鐘的時候，對方送出一腳長傳球，原本一次毫無威脅的進攻，卻因為梅方在防線上突然的冒頂失誤，將球權交給了日本隊，讓國足防線再次處於高度緊張之中！

比賽來到第70分鐘的時候，國足的防線在對手的連續打擊之下，還是出現了漏洞。這日本隊一次角球進攻中，國足的後防線球員盯人出現重大失誤，對手在小禁區內無人盯防，輕鬆完成頭槌破門。劉殿座甚至沒能做出撲救反應，這個丟球讓他感到頗為無奈，國足在70分鐘內就2球落後，而劉殿座的首秀被隊友坑慘了！

國足首秀就被「日本三隊」輕鬆攻入2球後，劉殿座似乎被打懵，鏡頭給到劉殿座的時候，他先是原地獃滯了，隨後他突然高舉雙手怒吼自己的隊友，讓他們不要再如此輕易被對手打穿。這一幕令人頗感心疼，劉殿座在比賽中很少會做出如此激憤的表情和動作。不得不說，29歲的恆大門將在國足的旅程沒能有1個好的開始，希望下一場比賽球隊防線能更加給力，不要再讓他如此聲嘶力竭地怒吼了！（老邱 中超球評）
