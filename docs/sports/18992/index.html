小小年紀拉仇恨！巴薩中場第一次國家德比之前如此嘲弄皇馬

2019-12-18 1,062條評論 2,268次閱讀 320人點贊

當地時間12月18日晚間，本賽季第一場西班牙國家德比將在巴薩和皇馬之間展開。作為舉世矚目的德比之一，皇馬和巴薩之間的國家德比由於西班牙特殊的政治因素變得更加火爆。而本場西班牙德比之前，皇馬和巴薩還遭受著共同的困擾，在歐冠里，他們驚險晉級，在聯賽里勉強領先，主教練備受爭議，球員發揮都不穩定，這樣的一次德比或許將引起更多的糟點。

而對於巴薩主帥巴爾韋德來說，他新賽季以來，很少在缺少中場核心德容的情況下開始比賽，周三的國家德比也不會有什麼不同。而荷蘭人也很興奮終於在這場德比中首次亮相，德容本賽季已經在諾坎普確立了自己的核心地位，他將希望幫助巴薩在西甲聯賽中重新奪冠。

對於這場德比，德容也是毫不客氣地揭露了皇馬的傷疤和痛處，德容對記者說：「我對國家德比非常興奮，我最喜歡的一場比賽時巴薩在諾坎普5:0大勝皇馬，」德容解釋說，「這是我的第一場國家德比，是一場非常特別的比賽。去年，我在阿賈克斯對陣過皇家馬德里，這很重要，但這是另一回事，我那時在另一支球隊里。」

那個時候，巴薩球迷已經知道德容會在一月份簽約諾坎普，所以他們對德容在對陣皇馬的比賽中表現出的大師般的出色表現感到更加高興，而他最終淘汰歐冠冠軍。對此，德容暗示，在皇馬的那場比賽是所有比賽中最好的一場，他說：「對我們來說這是一個非常特別的夜晚，事實上，我想說我們在伯納烏踢得最好，在他們的體育場4:1淘汰了他們，踢出了這麼好的足球，我對那場比賽有著非常美好的回憶。「

作為荷甲阿賈克斯的核心和雙子星之一，德容在荷甲時也隨阿賈克斯參加過多次荷蘭國家德比——阿賈克斯和費耶諾德的比賽，那裡的氣氛也讓國家德比處於危險的水平。對此，德容說：「在荷蘭我們也有一個國家德比，是阿賈克斯和費耶諾德，它對世界的影響比這裡小，但荷蘭的每個人都在談論它。氣氛不同於其他比賽，你可以感受到體育場內的緊張氣氛，這是球員們喜歡參與的比賽。」
