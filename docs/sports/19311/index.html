西班牙人降級武磊將面對兩個選擇，回國征戰，或者在西乙打出名堂

2019-12-16 1,223條評論 4,562次閱讀 2795人點贊

中國球員武磊再次在12月15日，也就是今天的23:00將會上場對戰皇家貝蒂斯。已經確定了要上場的武磊被球員給予了太多的希望。而相比於這場比賽的勝負與否，人們其實更看重的消息是武磊如果在西班牙人最後無奈降級之後該何去何從。

不少中國媒體對武磊現在所屬的球隊——西班牙人降級之後，武磊究竟會作何抉擇議論紛紛。現在的西班牙人在聯賽中的狀態非常尷尬。不管是單純的排名，還是球隊比賽中帶來的感官和心理上的期望來說，西班牙人現在想要保級都非常的危險。武磊也遇到球隊的瓶頸期。整個西班牙人現在的戰績尷尬，十六場比賽僅僅只拿下10球。這對於西甲的隊伍來說是一個非常差的成績。

武磊在本賽季也成功的打進了一球，使自己成為了這十分之一。不過現在西班牙人依然是西甲進球最少的一支球隊。雖然武磊能夠達到非常有利的跑位，但是沒有球員能夠有效的為其喂球。擅長跑位的武磊發揮不出來其該有的優勢。但是武磊除了沒人給他喂球之外其面對球門時的表現也是非常尷尬的。武磊在本賽季已經丟失了足足雙位數以上的單刀球，可以說是非常的可惜。武磊丟失單刀球就是展現出在射門上，一些東西還是掌握得不太熟練。

如果今天晚上對戰皇家貝蒂斯的比賽不能夠打出一場勝利，對於西班牙人來說保級的局勢基本上就成為了很困難的一件事情，而球隊如果不能夠保級，武磊個人肯定是會選擇調動。不少人都認為當西班牙人降級之日，武磊將會去到國內繼續踢球遠離西甲的舞台。

作為整個中國存放在世界舞台上現有的一根獨苗子。武磊如果選擇了無奈的回國，想必是對中國足球業也是一件無奈的事情。因為武磊的回歸必將宣告著這一次的留洋計劃失敗。這個留洋計劃本為嗷嗷待補的中國球員和球迷帶來希望，現在卻變成了滿滿的失望。

不過不少球迷表示建議武磊留在西乙，因為相比於國內的聯賽，西乙也能夠提升武磊的個人實力。並且也能夠有效地鍛煉他的能力，使他成為該隊的王牌明星。所以說也許武磊會選擇直接留在西乙也不一定，因為回來之後對於武磊本身球員的身心也是一種煎熬，球迷們也需要一點期待。
