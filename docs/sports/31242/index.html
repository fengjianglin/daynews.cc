索帥：曼聯積分位置比想像中的靠後，希望球隊儘快回正軌

2019-12-25 984條評論 2,797次閱讀 213人點贊

直播吧12月24日訊 曼聯主帥索爾斯克亞近日接受採訪，談到目前處於英超領頭羊位置的利物浦。

索帥表示利物浦如今的成績也是自己想帶曼聯獲得的，對於自己接掌曼聯教鞭12個月以來的成績，他也坦言球隊表現未達預期。

索帥告訴記者：「本賽季賽程已經過半，我們的表現起伏不定，因此我們想在接下來比賽中讓球隊重回正軌，同時確定下賽季的陣容。」

「我希望我們能夠在接下來的比賽中獲得更多積分，讓球隊更快地回到正軌，因此我希望球隊上下能夠更加團結一致，從挫折中獲得更多經驗。

「目前球隊在積分榜的位置比我想像的要更靠後一些。我們希望成為一支戰無不利，無堅不摧的球隊。

「利物浦花了幾年時間才取得現在的成績，所以我們需要繼續前進，他們取得的成績也是我們想要的。

「是的，我們擅長反擊，我們也有好的球員，我們應該繼續保持這樣，這是變革的一部分。我們還要儘可能地在排位靠後的球隊身上獲得勝利。我們在對沃特福德的比賽中創造了許多機會，但是一旦沒抓住機會，就容易讓對手打反擊，因此我們需要儘可能地抓住進攻機會，更好地阻止對方的反擊，不斷學習，保持成長。我相信我們能夠在本賽季，乃至下賽季取得長足的進步。」

曼聯在節禮日將坐鎮老特拉福德球場迎戰紐卡斯爾聯隊，面對這支非常擅長防守的球隊，索帥呼籲球隊要踢得更有耐心一些。

索帥告訴記者：「這場比賽將會很困難。我覺得我們在對陣紐卡的比賽中控球率會比較高。但讓人失望的是，我們能在一場比賽中擁有高達70%的球權，但比賽依然以失利告終。

「我們能在戰績不佳的情況下快速做出回應，這是球隊今年的一大亮點。雖然球隊表現起伏不定，但並未像上賽季末那樣一落千丈，所以肯定會有回應的。」

「說句實話，我在隊內會議告知球員們要拿出氣勢，做出回應，來報答球迷對我們的支持。球迷總是在我們最需要他們的時候不離不棄，我們必須拿出更好地表現來回應他們的支持。我確信你會看到我們的回應。」
