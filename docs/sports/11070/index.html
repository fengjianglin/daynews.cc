國羽2連勝！諶龍第14次擊敗安塞龍取得開門紅 對手送給龍隊一個贊

2019-12-11 2,057條評論 4,287次閱讀 1640人點贊

北京時間12月11日，2019年世界羽聯年終總決賽進入首個比賽日，率先是衛冕冠軍王懿律/黃東萍以2-0擊敗陳炳順/吳柳瑩，取得開門紅。隨後是一場男單焦點戰，諶龍迎戰丹麥一哥安塞龍，經過3局苦戰，諶龍以2-1險勝安塞龍，國羽2連勝。值得球迷關注的是決勝局戰至8-9，諶龍反手回大斜線，打出超級好球，安塞龍直接給個贊。

諶龍在賽季初狀態低迷，直到法國公開賽，才拿下本賽季首冠。國羽男單處於低谷時刻，諶龍必須扛起男單大旗，僅一人參加男單總決賽，必須捍衛國羽男單榮耀。他目前世界排名第4位，首輪對手則是丹麥選手安塞龍，後者世界排名第5位。兩人交手第17次，諶龍13勝4負，完全佔據上風。最近一次交手是2019年法國公開賽，當時諶龍2-1險勝。

伊始比賽，安塞龍率先連得2分，之後諶龍頂後場球。安塞龍在風向判斷上出現幾次失誤，諶龍以4-3反超。雙方都進入多回合戰鬥中，諶龍打出一波小高潮連得5分。安塞龍奮力追趕，諶龍以11-5領先進入技術暫停。局間休息，安塞龍展開反撲，但諶龍依舊保持4分左右領先。在追到僅差2分時，諶龍連下2分以21-17先拔頭籌。

易邊再戰，諶龍開局領先1分，之後安塞龍展開反撲，不斷製造麻煩。諶龍頂著壓力前行，安塞龍保持2分領先。在6-9落後情況下，諶龍連得2分。在多拍回合僵持中，諶龍回球頻繁失誤，安塞龍以11-8領先進入第二局技術暫停。回來後，安塞龍殺球犀利，諶龍推後場球丟分。由於分差過大，諶龍放棄抵抗，安塞龍以21-10扳回一局。

決勝局，雙方爭奪非常激烈。諶龍失誤頗多，安塞龍將比分擴大至8-6。諶龍加強進攻，給安塞龍製造很多不利局面。諶龍追平，安塞龍又領先，經過艱苦戰鬥，諶龍以11-10領先交換場地。回來後，諶龍將比分擴大至14-11安塞龍。諶龍穩住局面，保持2分左右領先。安塞龍追分無果，諶龍以21-17贏球，取得小組賽開門紅。
