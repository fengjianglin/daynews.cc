3小時連丟2冠！日本第一女單賽後豪言真囂張！男女單打冠軍無懸念

天天要聞
3 天 ago

近日，正在鄭州那邊舉辦的2019國際乒聯巡迴賽的總決賽已經接近尾聲了，當天從下午1點開始，到晚上的八點半左右，將會進行4個項目的決賽以及頒獎儀式。

不過這一天的幾場決賽當中卻並沒有混雙在內，因為在前一天混雙的決賽就已經打完了，真不知道他們是怎麼想的，決賽都放在同一天進行不是很好嗎？

一天5場決賽世界完全是夠的，而且這樣還能讓許昕他們多休息下。

不過現在混雙結果都出來了，說再多也沒用，混雙的決賽結束之後，拿到冠軍的確實沒有什麼懸念，我們國乒的世界第一加世界第三組合實在是無解啊！

許昕和劉詩雯在當天在決賽當中成功地戰勝了日本選手伊藤美誠和水谷隼之後，將混雙的金牌收入囊中，不過領獎的時候伊藤美誠又是黑著臉。

這已經不是她第一次這樣了，每次輸了比賽之後她都會表現得很不爽，實力確實有，可老是黑著個臉去領獎，這是真不服輸啊！

當天除了進行了混雙的決賽之外，男女單打的半決賽也都比完了，而且也產生了最終進行決賽的兩名運動員，男單這邊的話因為4強都是國乒的選手。

雖然說都是自己的隊友，可是誰都想拿到冠軍，所以基本上是不會讓著對方的，而2場比賽比完之後，前世界第一樊振東成功戰勝了林高遠率先晉級決賽當中。

而男單這邊半決賽最後一場比賽是許昕對陣馬龍，這同時也是當天的壓軸之戰，最終馬龍以4-2的比分贏下了這一場。

也就是說男單的冠軍會在馬龍和樊振東當中產生，馬龍也將衝擊自己在巡迴賽總決賽的第6冠！許昕這一場比賽輸了之後，也是無緣創造神跡紀錄。

本來許昕有機會拿到男單，混雙以及男雙的冠軍，成為歷史第一個同時拿到這3個項目的運動員，可是輸給馬龍之後，一切都成空了。

女單那邊半決賽的結果也是出來了，最終女單那邊打進決賽的2名選手是陳夢以及王曼昱，這也是陳夢第三次衝擊該賽事女單的冠軍。

而伊藤美誠這位日本第一女單可就有點悲催了，3小時連丟2冠，先是在混雙那邊輸給了許昕他們無緣混雙冠軍，在就是女單這邊，伊藤美誠輸給了陳夢之後止步4強。

在面對伊藤美誠和張本智和的比賽當中，陳夢以及許昕沒有輸過一場，給我們國家掙足了面子。

兩個半決賽結束之後，男女單打的冠軍已無懸念了，除了女雙之外，其他幾個項目就只有男雙我們國乒沒有預定冠軍的位置。

伊藤美誠在賽後採訪當中表示，雖然這一次淘汰了，可是自己距離成為一名不會輸給任何人的運動員更近一步了！輸球後還能放出如此豪言，當真是囂張！

不會輸給任何人？有那個運動員敢說自己永遠不會輸？就連博爾特在自己生涯末期都輸過。

不會輸給任何人相信這也只是一個遙不可及的夢，睡著了就能做到，當然了，話說回來，運動員確實是需要這樣的心態，但更多的應該是放在心底！
