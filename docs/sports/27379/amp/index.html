王一梅老東家招人！女排隊員加盟印尼聯賽，2人都是球隊主力

天天要聞
2019-12-20

繼張曉雅和孫傑兩人官宣加盟泰國聯賽後，又有女排國手出國打球。據印尼排協公布的最新轉會消息，雲南女排隊長劉夢雅和天津女排副攻王佳敏將加盟印尼聯賽JBT女排俱樂部。

這不是女排國手第一次加盟印尼聯賽，就連王一梅也曾在印尼聯賽打過球。2016年，王一梅和張磊加盟印尼聯賽雅加達國家電力俱樂部，幫助球隊獲得聯賽冠軍。作為王一梅的老東家，印尼聯賽整體實力雖然不高，但排球這一項目在印尼這個國家備受喜愛，印尼女排國手阿普利亞兩姐妹也曾因為性別問題吸引不少球迷關注。

印尼女排和泰國女排一樣都要參加2020年東京奧運會亞洲區資格賽，自然印尼聯賽也將在資格賽結束後進行。這次加盟印尼聯賽的國手劉夢雅和王佳敏所在的俱樂部均未能晉級聯賽八強，打完排超聯賽再去印尼打球不存在時間的衝突，江蘇女排隊員許若亞就曾效力於該俱樂部。

劉夢雅是雲南女排隊長，也是雲南女排的進攻核心人。2017年女排亞錦賽與全運會女排決賽的時間撞車，劉夢雅沒有全運會比賽任務，被包壯教練招入隊中，以隊長身份率領球隊晉級四強。劉夢雅剛過完自己31歲生日，這個年紀的女排隊員很難再入選國家隊，能夠在印尼聯賽繼續追逐自己的排球夢想，對劉夢雅來說是對自己能力的肯定。

王佳敏比劉夢雅小7歲。作為天津女排培養出來的隊員，王佳敏曾入選過國奧集訓隊，也幫助天津女排獲得過聯賽冠軍。時過境遷，天津女排已經沒有王佳敏的位置，本賽季轉會河北女排成為球隊的主力副攻。河北女排整體還是太年輕，王佳敏作為球隊年紀最大的隊員，也難以幫助球隊將名次往前進步。

加盟印尼聯賽，是王佳敏第一次出國打球，也是繼離開天津女排之後，又一次代表其他俱樂部打球，期待王佳敏在異國他鄉能有所作為。在身高條件普遍不高的印尼聯賽，183的劉夢雅和186的王佳敏算得上是高個隊員，兩人同效力於一個俱樂部。第二次去印尼聯賽打球的劉夢雅，無論是語言還是球技都能給予王佳敏一定的幫助，王佳敏也可以和劉夢雅攜手為俱樂部爭取獲得聯賽好成績。
