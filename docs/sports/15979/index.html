80後玩完的阿森納，90後的陣容作潛力股-英超獨一份的年輕化

2019-12-14 454條評論 1,960次閱讀 452人點贊

阿森納輸了一場又一場，換了一人又一人，始終在激烈競爭中無法脫穎而出，往日的爭四老梗已經不復存在，如今只能淪為爭六的級別。

不論教練如何更換，阿森納的新風格始終維持：在所有俱樂部幾乎都是老將擔當時，阿森納幾乎是清一色的90後球員。豪門俱樂部33-35歲的老主力硬撐時，阿森納的80後球員已經幾乎損失殆盡。

貝萊林這樣年輕的球員，在阿森納已經是效力多年的老將，即使是從皇馬引進的塞巴略斯，也還有多個賽季的成長空間。

貢多齊這樣的小夥計，更像是平淡無奇的新一批球員中培養未來之星。

做到頂尖未必就一定要水平出眾，只要比身邊的競爭者好那麼一些就足夠了。

當桑切斯、吉魯這樣的夥計已經逐步消失時，大衛魯伊茲這樣的老後衛也不會持續太長時間。

拉卡澤特這樣的法國邊緣國腳，已經多日沒進過法國國家隊，但當吉魯一批球員退出時，拉卡澤特顯然還有很長的發揮時間。

阿森納不幸中的萬幸，就是還有一批1990-2000年齡段的可用之材，水平顯然是無法和豪門的80後硬拼，作為未來儲備還是足夠使用。

就好像阿賈克斯囤小妖持續多年，真正爆發也就是連續擊潰皇馬與尤文之後。
