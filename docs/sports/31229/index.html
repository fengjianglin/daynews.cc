CUBA全明星票王出爐！楊皓喆朱松瑋當選，他們的未來在CBA

2019-12-25 1,148條評論 4,946次閱讀 2110人點贊

近日，屬於CBA全明星票選的階段已到來，關注國內籃球的球迷可以為自己喜歡的球星投票。同時，大學生聯賽CUBA全明星也在開始。北京時間12月24號，CUBA全明星投票截止，票王人選出爐。根據報道2020CUBA全明星陣容網路票選時間截止時，來自北體的楊皓喆和來自汕大的朱松瑋分別獲選前後場組票王。按照規則，他們將成為全明星正賽的兩隊隊長，各自挑選隊員組建球隊。

具體排名里，在後場組的投票中楊皓喆位列第一，來自北大的張寧位列次席。清華後衛王嵐欽排名第八，廈大黎伊楊排名第九。前場組方面，朱松瑋第一，華僑大學核心孟翔屈居第二，北體大劉毅排名第三，北大混血前鋒祝銘震僅列第六。

近兩年來，CUBA聯賽的大力發展使得大學生籃球受到外界球迷廣泛關注，優秀學生球員的積極湧現也充實了CBA選秀大會的選拔人才庫。過去幾年，有多位CUBA球員通過CBA選秀大會進入CBA聯賽。其中作為代表的有今夏剛成為CBA狀元的王少傑，江蘇主力輪換王洪，剛在廣州男籃站穩腳跟的郭凱以及賈明儒。多位學生球員的表現讓外界堅信，校園籃球也有產生職業球員的可能，甚至對於球員個人的全面性塑造更佳。相信未來校園籃球會愈發受到關注。

本次當選全明星票王的楊皓喆和朱松瑋，在未來大概率也將嘗試CBA選秀大會的舞台，爭取進入CBA聯賽。來自北體大的楊皓喆是籃球世界裡逆襲上位的角色，從默默無名的胖子到成為北體大主力，再到球隊核心，最終延畢留校打大五賽季。在大學賽場”大器晚成”的楊皓喆已是CUBA聯賽一位出色的控衛角色。在本賽季的基層賽里，楊皓喆9場比賽，27記三分球，47%的三分球命中率。場均18分3.1次助攻3.1個籃板2.3次搶斷。北體”砍分直升機”不僅會砍分，當提升傳球組織能力後他的威脅更大。明夏，他有機會被CBA球隊選中。

而另一個票王，來自汕頭大學的朱松瑋則是向來背負期待。身高達到2.05米的他被譽為汕大杜蘭特，大前鋒的身高下有著小前鋒的持球技術，爆發力出色。是新時代籃球體系下一位天賦出眾的年輕球員。和今年成為CBA狀元的王少傑一度有”北王少傑，南朱松瑋”的稱號。從理論上講，在王少傑站穩CBA賽場的前提下，明夏朱松瑋也將會是CBA狀元的有力爭奪者。

除此之外，來自北大的張寧、祝銘震，清華的郭健，北化工的李瑋灝以及廈大的黎伊楊。這些球員都是未來有可能衝擊CBA聯賽的球員。或許在未來某個階段，CUBA聯賽將成為CBA聯賽人才的供給庫！我們期待有更多的優秀球員湧現！
