中國撞球一哥歸來！揭秘丁俊暉幕後功臣，推掉商業活動閉關半月

2019-12-09 75條評論 2,431次閱讀 232人點贊

時隔10年再次捧起斯諾克英錦賽冠軍獎盃，丁俊暉掩飾不住自己內心的激動。奪冠後，他握拳高舉雙手，捶胸怒吼慶祝。而在領獎現場，一名同樣興奮的華人引來了很多人的好奇。此人正是丁俊暉復活幕後功臣，著名斯諾克經紀人馮敬文。

在丁俊暉奪冠的那一刻，馮敬文就走下了觀眾席，到了球場上，跟丁俊暉來了個大大的擁抱。兩人緊緊擁抱，慶祝這個久違的冠軍。

這是丁俊暉兩年多來第一個冠軍，終於走出了低谷。這當中的心酸，馮敬文最清楚不過了。在發表奪冠感言的時候，丁俊暉也感謝了馮敬文對他的巨大幫助，在自己處於生涯低谷的時候，馮敬文給了他很多開導。

馮敬文，著名斯諾克經紀人，如今80後另一頂尖球員特魯姆普就在他的旗下。他也是火箭奧沙利文的前經紀人。目前，馮敬文旗下有7名職業球員，除了小特，還有阿里-卡特、布雷切爾、梁文博、尼爾-羅伯遜、傑克-利索夫斯基、奧利弗-萊恩斯。

在丁俊暉奪冠之後，馮敬文也在自己的公司推特上第一時間恭喜了丁俊暉，值得一提的是，他寫道，「偉大的學生，偉大的冠軍。6周浴火重生。」

根據前中國斯諾克名將劉崧的透露，為了在這次英錦賽上有出色的表現，丁俊暉推掉了商業活動，閉關半個月練球。這樣的苦練顯然起到了成效，冠軍就是最好的回報。

丁俊暉上一次英錦賽奪冠是在2009年，在上一次奪冠之前，他也經歷了兩年多的低谷。而2009年奪冠之後，丁俊暉開始爆發，連續5個賽季都有冠軍入賬，排名一度來到世界第一。10年輪迴，如今丁俊暉再度捧起英錦賽冠軍，這會是他生涯又一個巔峰的開始嗎？無論如何，相信馮敬文會是他背後最大的支持者。
