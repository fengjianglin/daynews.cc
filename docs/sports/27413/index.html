高層又插手，國足選帥三選一，足協要當好“背鍋俠”的準備？

2019-12-20 940條評論 2,391次閱讀 82人點贊

昨日（12月19日），記者趙震透露：國足主帥基本放棄外教，而是從3位本土教練中選擇1位執教。據說這是總局的指示，那麼，此舉又一次將足協選帥權架空，正是國足處於低潮之時，最後背鍋的或許又將是足協。

東亞杯後國足選帥再次成為全國球迷關注的焦點，臨近世預賽還有3個多月時間，選擇外教有些緊迫，而本土教練由於熟悉我國足壇自然佔盡優勢，但這項工作已是迫在眉睫。

趙震強調這是總局的指示，國足主帥要從中超成績最好的三名主教練中選擇一位，這正好切合了之前的傳言：國足主帥將從李霄鵬、李鐵、王寶山中選出。

從目前來看，剛剛產生的新一屆足協在國足選帥問題上，繼續延續著「總局指示，足協建議；國足慘敗，足協背鍋」的選帥模式進行著，或者說，強調N次的「專業人辦專業事」這句話再一次落空了。世預賽40強賽乃至12強賽將是國足2020年征戰的目標，總局在選帥的問題上總是定出基調或者指明方向，至於今後國足主帥帶隊成績如何，基本與總局無關，到時慘敗責任再次又落到足協頭上，背鍋俠的名號看來足協無疑已經預定。
