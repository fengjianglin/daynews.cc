因睡覺錯過比賽的國乒16歲新星，不僅不罰還要獎勵，劉國梁亂來？

天天要聞
2019-12-10

文/匯哥

國際乒聯2019北美挑戰賽落幕，國乒在成年組賽事中拿到男單、女單和男雙3塊金牌，依然成了最大的贏家。不過，球迷們卻都在關注16歲小將向鵬的處境。因睡覺錯過比賽的國乒16歲新星，不僅不罰還要獎勵，劉國梁亂來？

在此前的U21組男單賽事中，向鵬打進了半決賽，卻因為倒時差等緣故睡過了頭而超時自動退賽，無緣衝擊U21男單金牌。但他知恥而後勇，在隨後的成年組比賽中以4-2淘汰法國悍將阿庫組，成功斬獲了男單金牌，算是彌補了一下自己此前的過失。不過，向鵬此前的失誤確實有點大，所以，球迷們也都在猜測劉國梁會給出什麼樣的懲罰，畢竟有王楚欽摔拍被禁賽3個月的先例在。

但據最新的消息透露，劉國梁好像並不准備懲罰向鵬，相反還要對他給予獎勵。原因倒也說得過去，一來U21男單比賽含金量不足，國乒本就沒太在意，報名參賽不過是多打點比賽鍛煉新人，反倒是向鵬能夠奪得成年組的男單金牌頗為讓人驚喜，所以功過相抵還要獎勵他。二來，向鵬能在錯過半決賽這樣的重大失誤後，迅速調整好心態，並在成年組男單比賽中奪冠，說明向鵬天賦和心態都是超棒的，對於這樣的好苗子，劉國梁當然不會過分苛責。事實再次證明，競技體育界只有菜才是原罪啊！

但怎麼獎也成了一個難題。16歲新星因睡過頭錯失比賽，劉國梁卻糾結如何獎勵他，與1點有關。雖然向鵬拿到一個含金量十足的冠軍值得恭賀，但他此前畢竟有犯過錯，所以，使用什麼程度的獎勵就非常關鍵。獎勵太重，容易讓向鵬這樣的新人以及背後的教練組心態膨脹，覺得成績可以代表一切，對於向鵬的未來成長絕對不是好事；若獎勵太輕，又可能會讓新人們覺得受了打壓，不能起到最大的激勵作用，因此劉國梁現在也在為這個獎勵的度而頭痛！大家覺得什麼程度的獎勵最合適呢？
