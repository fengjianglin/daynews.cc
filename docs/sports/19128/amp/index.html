誰才是湖人隊中的“扛把子”？如今麥基給出了答案：我們一榮俱榮

天天要聞
2019-12-17

作者：體育叨叨君

截至到目前，湖人隊在本次常規賽中的前26場戰況是3負23勝，其中勒布朗的個人數據為11助攻、7籃板、26分，而戴維斯則是3封蓋、3助攻、2搶斷、9籃板、27分。

目前所有人公認，戴維斯在籃球這一領域中已經成為了球技最高超的人員之一，不管是他的「拋投」還是「走位」都是非常出色。並且在防守方面，他更是屬於那種「覆蓋式的防守」，讓對手無懈可擊。

也因為戴維斯的出色，所以對於喜歡湖人的朋友們來說就產生了一個這樣的疑問，對於湖人來說，他和勒布朗誰更重要呢？畢竟在目前的湖人隊里戴維斯不管是在搶斷方面還是封蓋方面，再或者是籃板和防守都是「第一」，更是有著進攻效率29.8、進攻貢獻5.0的數據，這些全部在勒布朗之上。

我們可以根據此數據來斷定戴維斯的分量要比勒布朗更重嗎？答案是否定的，其實這個問題也沒有任何意義。對於此類問題，我想那些「詹姆斯球迷」和「湖人球迷」一定可以理解，畢竟之前就有很多類似情況引發爭論。

先說目前在本季的PIPM數據，勒布朗為5.04，戴維斯是5.02，可以說是不相上下。再說BPM和VORP的數據，勒布朗分別是9.3和2.5，而戴維斯卻是7.6和2.0，勒布朗明顯要高於戴維斯。同為湖人的兩大主力，勒布朗貢獻值最高，而戴維斯的個人數據最高，所以說二人只能算是平分秋色。

至於誰更重要這個問題，已經是球迷們乃至媒體們談論已久的一個問題了，至於真正的答案，請允許我借用麥基的一句話來回答。

「在湖人隊里，我們每個人都是不可分割的一部分，同時也深知自己的個人定位，我們全部要按照主教練的安排進行比賽，在本賽季能成為西部主宰，我們靠的就是團結一致，而不是完美的個人！」

確實，麥基的話更好地詮釋出了這個答案，沒有完美的個人，只有最強的團隊！畢竟籃球是一門團隊競技的體育活動。不論是誰，只要到了賽場必須要聽從教練的安排，戴維斯也好，勒布朗也罷，通通要以教練為尊。當然，這其中還包括詹姆斯和所有隊員。

湖人正是因為擁有著其他隊伍沒有的「團結」，所以才能讓他們一枝獨秀！

湖人在如今可以稱得上是一個比較「奇葩」的隊伍了，畢竟現在主流的陣容就是「1大4小」或者是「5小」，反觀湖人的「雙塔防守陣容」，這簡直就是籃球領域中「最靚的那個仔」了。不過我們對此也應報以理解的態度，畢竟湖人一直都是以「防守」見長的。

新秀年以來，戴維斯的防守可以算是個人生涯中最牛的一次了，同時這也是防守籃板最少的，而原因就是要為「季後賽」蓄力。畢竟戴維斯噸位不行，所以「高機動性」才是他的最佳選擇。眾所周知，目前在隊伍里專職防守籃板的有勒布朗和麥基就足矣了，所以在這方面戴維斯可以不用分散精力。

從本賽季開始到現在，進攻戴維斯「禁區」的對手命中率僅有38%，這是什麼概念？簡直就可以被對手稱之為噩夢了！

如今的湖人隊的每一場比賽里，戴維斯和勒布朗帶隊做好最嚴密的防守，而詹姆斯和隆多則是負責進攻，可以說他們每個人都非常明白自己的職責，並且做到100%的完美。

如果說目前湖人里詹姆斯是最強的「矛」，那麼最堅硬的「盾」這個稱號戴維斯當之無愧！在這裡也希望戴維斯在本年度可以當選為「最佳防守球員」。如今的湖人想要真正地打出自己的威風，那麼他們二人缺一不可，當然還有我們的濃眉哥和其他成員哈。

截至到目前，本賽季MVP的前五位分別為安特托昆博、東契奇、詹皇、哈登和戴維斯，僅湖人隊就佔據了兩個名額！

歲月催人老，詹姆斯現在的身體隨著年齡的增長已經逐漸開始走下坡路了，所以在未來，希望湖人隊里年輕的球員能夠打出威風！畢竟已經連續6年被拒季後賽的門外，喜歡他們的球迷已經開始等不及了……

從他們自身來講，勝者為王、敗者為寇，唯有成功，才能被人們所牢記！所以說，對於此時的湖人，個人之間的孰強孰弱已經不再重要，畢竟大家是「一根繩子上的螞蚱」。一榮俱榮，一損俱損。

現如今，只有全隊團結一致，拼盡全力拿下布萊恩杯才是最值得關注的一個問題，否則如今的一切，終究還是霧裡看花。

不忘初心，方得始終。為湖人隊加油！祝福他們拿下總冠軍！
