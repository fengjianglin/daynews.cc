意甲今日頭版：加圖索首秀爆冷 尤文繼續衝擊榜首

2019-12-15 1,021條評論 3,813次閱讀 1875人點贊

直播吧12月15日訊 米蘭體育報、羅馬體育報、都靈體育報是義大利的三大體育報，今天我們來看看它們各自的頭版頭條都帶來了哪些內容。

《米蘭體育報》

1.「聯賽冠軍快車。」國米希望在佛羅倫薩坐上快速前進的列車。盧卡庫和勞塔羅要忘記歐冠。孔蒂沉默，與媒體關係緊張：「需要尊重。」法國方面的確切消息：「吉魯接近去藍黑軍團。」

2.尤文在伏擊：C羅嘗試連續4場比賽進球。斑馬軍團今晚主場迎戰烏迪內斯。

3.「那不勒斯上演翻車電影。」聖熱內羅？（宗教傳說中那不勒斯的守護神）帕爾馬製造了奇蹟，2-1絕殺那不勒斯。

4.加比亞迪尼在燈塔德比中打入制勝球，莫塔輸球，桑普1-0獲勝。布雷西亞3-0萊切。

5.米蘭要為生日獻上一場勝利，貝總呼喚伊布：「來蒙扎吧。」

《羅馬體育報》

1.「反那不勒斯。」加圖索首秀爆冷，1-2不敵帕爾馬。帕爾馬讓聖保羅球場安靜。庫盧塞夫斯基首開紀錄，米利克扳平，熱爾維尼奧補時絕殺。那不勒斯遭遇噓聲。

2.「國米，小基耶薩的考試。」藍黑軍團做客佛羅倫薩，這場比賽也會決定蒙特拉的未來。

3.尤文帶著迪巴拉繼續衝擊榜首：今晚對陣烏迪內斯。

4.萊切0-3慘敗布雷西亞，利維拉尼攻擊巴神。

5.拉涅利贏下德比，桑普1-0熱那亞。

《都靈體育報》

1.「尤文飢餓，國米緊張。」今晚斑馬軍團在主場迎戰烏迪內斯。薩里「如果我們要追求歐冠，那就必須在聯賽中全力以赴。」孔蒂，處在有爭議的沉默中，客戰佛羅倫薩，他們想忘掉與巴薩的失利。

2.「致命的帕爾馬，加圖索嗷嗷大叫。」那不勒斯輸球：庫盧塞夫斯基和熱爾維尼奧讓加圖索遭遇開門黑。

3.加比亞迪尼進球，莫塔陷入困境。桑普在第85分鐘贏下德比，莫塔帥位危險。

4.貝洛蒂上場，隊長回來了。都靈追求三兩勝。

（我叫達芬奇）
