酸了!孟非曬皇薩球衣:有大神認得簽名嗎 原來是梅西皇馬全隊簽名

2019-12-18 154條評論 1,312次閱讀 103人點贊

12月17日消息，北京時間12月19日凌晨3點，西班牙國家德比即將打響。著名相親節目主持人孟非通過微博分享了收藏多年的皇馬巴薩簽名球衣，拉了一波仇恨。

​眾所周知，江蘇衛視主持人孟非是一個十足的球迷。2015年恆大打進世俱杯半決賽之後，他表示要前往現場為恆大加油。

前不久，孟非還在個人微博上轉發了一條關於國足歸化的內容，調侃：「想像一下他們九個同時上場為中國隊踢世界盃的畫面…我告訴自己：要冷靜一下。」孟非隨後就刪掉了這條微博，不過還是引起了網友熱議。

​這兩天皇馬巴薩的世紀大戰即將打響，孟非今天上午在微博上曬出了自己的兩件簽名球衣，並在線求助球迷：「剛剛整理柜子時發現前幾年做義大利和西班牙專場時主辦方送的紀念品，有大神認識隊服和上面的簽名嗎？求告知。」

只見巴薩球衣是梅西簽名球衣。

​而皇馬球衣上則似乎是全隊簽名，都給簽滿了。

​還真有皇馬資深球迷給「翻譯」出來了，原來是C羅、本澤馬、貝爾、拉莫斯、J羅等13名隊員的簽名。

孟非回復點贊，「你就傳說中的大神！」

​這一「拉仇恨」行為讓球迷瞬間變「酸」，不少網友還呼籲來一波抽獎。

還有網友在評論里告訴孟非：「圖二從左致右從上到下依次是潘長江、蔡明、姜昆、趙本山、范偉、姜振宇、郭德綱、于謙、佟大為、黃菡、黃瀾。」
