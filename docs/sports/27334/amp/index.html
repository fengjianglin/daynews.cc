體育總局近日連續開會研究男足、男籃問題，目前暫未得到具體結果

天天要聞
2019-12-21

女排又一次扛起了中國三大球的旗幟，然而這麼多年，卻也一直是她們在扛著……

近日，據足球報消息，體育總局近期召開了多次會議，討論中國足球和中國籃球現在面對的困難，以及未來該如何改進。由於今年中國在三大球項目上，僅有中國女排奪得世界盃冠軍，男足和男籃成績太過慘淡，體育總局因此想聯合足協及籃協，還有各省市體育界人員，共同研究和探討足球和籃球的下一步發展。

今年三大球項目，女排在女排世界盃中以全勝姿態輕鬆奪冠。男籃世界盃在中國舉辦，但男籃失誤頻出、最終連奧運參賽資格都沒拿下，主教練李楠辭職也難辭其咎；足球方面，因為客場不敵敘利亞，主教練里皮當場宣布辭職，中國男足再次跌落谷底。這樣的成績，確實抹黑了這兩支國家隊在大眾心目中的形象。

為了解決男籃和男足面對的困難，體育總局目前已經進行過的研討會中，已經談到了一些問題。比如在足球中，大家普遍認為球員技術能力差、現在選帥也是難題，所以未來在構建訓練體系時，一定要加強短板。通過和不同對手的交手，我們也感覺到了和其他隊伍的差距，未來要在差距上努力提升。

在青少年辦賽上，各地代表也給出了意見，現在教育部門、體育部門可能都會辦體育賽事，往往在同一個年齡段會出現多個賽事，並且因為主辦單位不同還會相互抵制。未來要盡量避免這種情況，教育部門辦了小學比賽，那體育部門就去辦中學比賽，避免相互的重複和抵制。

關於具體未來還有哪些舉措和改革，目前暫未得到具體結果，體育總局及社會各界正在努力研究中，據悉，最快在下周，他們會再次召開會議，研討相關問題。或許具體的改革舉措會在之後進行公布。
