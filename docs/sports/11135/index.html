阿森納新帥敲定！皇馬故人曾問鼎歐冠 執教英超豪門經歷為他加分

2019-12-11 865條評論 2,365次閱讀 788人點贊

北京時間12月11日，英國媒體《太陽報》報道，那不勒斯在4-0戰勝亨克之後進入了歐冠16強的淘汰賽，但俱樂部在他們的社交媒體頻道上宣布了這一消息。聲明中寫道：「那不勒斯決定解僱安切洛蒂，他不再擔任俱樂部一線隊主教練。但俱樂部、俱樂部主席德·勞倫蒂斯和安切洛蒂之間的友誼、欽佩和相互尊重的關係依然存在，沒有任何影響。加圖索預計將在周三接管俱樂部。」

據報道，此前剛剛解僱主教練的阿森納渴望與這位經驗豐富的義大利人進行談判，看看是否有可能邀請他來酋長球場執教。槍手的四人尋帥團隊已經對超過12名主帥候選人的名單進行了削減。據了解，他們已經與瓦倫西亞主帥馬塞利諾，前波爾圖主帥維托佩雷拉和波爾多的保羅蘇薩都進行了非正式會談，但談判結果並不理想。而安切洛蒂曾在2010年執教切爾西，四年後帶領皇馬贏得了歐冠獎盃，由此被認為是一名重要的競爭者。這位剛剛被解僱的義大利人有著一口流利的英語，這也將有利於安切洛蒂執教槍手，對阿森納來說也是一個不錯的選擇。
另外，現任尼斯主教練維埃拉，曼城助教阿爾特塔，前尤文主帥阿萊格里以及前熱刺主帥波切蒂諾也都是槍手有意邀請的主教練。而且目前的臨時主帥永貝里也是被考慮的對象，但並不被看好能夠獲得這份工作。

目前，阿森納技術總監Edu和足球主管Raul Sanllehi一起主管著阿森納的尋帥工作。在俱樂部與他們所有的主帥候選人進行交談後，他們將向老闆克倫克和他的兒子喬希提供一份三人候選名單，由他們做出最後的決定。
在上周末逆轉西漢姆聯之後，阿森納的危機得到了一定的緩解，但目前槍手僅排在聯賽第9位，距離降級區只有7分。而球隊的目標至少是進入聯賽前4名，新任主帥的壓力顯然將是無比巨大的。
