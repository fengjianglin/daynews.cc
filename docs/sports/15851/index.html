今晚！馬布里遭遇最嚴峻考驗 新星證明為何領先CBA全明星票選？

2019-12-15 246條評論 1,936次閱讀 303人點贊

北京時間12月15日晚，CBA常規賽展開多場對決，其中最引人關注的是北控主場對陣廣東的比賽。因為比賽時間上與國足對陣韓國衝突，所以本場比賽將由CCTV5+頻道轉播，而不是CCTV5，但相信還是會有很多球迷通過各種渠道觀看本場比賽。

此前，廣東男籃已經取得12連勝，本賽季15勝1負的開局讓他們笑傲積分榜，對於馬布里的球隊來說，主場對陣CBA領頭羊是最嚴峻的考驗。北控的戰鬥力究竟如何，碰到總冠軍球隊，才更能看出差距。北控完全沒有機會嗎？一方面看自身的發揮，另一方面也要看廣東的態度。上一場對陣活力四射的浙江稠州銀行，廣東險些遭到爆冷，這應該給杜鋒提個醒，球隊在連勝的過程中容易出現麻痹思想，但願球隊別背上想贏怕輸的包袱。

馬布里對廣東男籃非常熟悉，過去作為對手，雙方你來我往互不相讓，如今作為主教練，面對杜鋒，這位菜鳥主帥的小紙條上面會記錄哪些內容呢？馬布里最忌憚的應該還是易建聯，因為北控的內線是軟肋，阿聯這點的進攻如果無法限制的話，很可能被廣東男籃擊穿。所以，今晚比賽的最大看點在於馬布里防守易建聯的策略。作為本賽季的選秀狀元，王少傑的成長有目共睹。他視易建聯為偶像與奮鬥目標，當比賽場上遭遇阿聯，他能在防守中阻擊自己的偶像嗎？王少傑當選狀元時，阿聯曾送上真摯的祝福，希望年輕的後輩能夠用實際行動追趕偶像，但想要第一個賽季便防住阿聯，這並不現實。作為國家隊主帥，杜鋒倒是可以好好觀察下這位狀元，也許不久的將來，他有機會立足國家隊。

北控依靠三外援，前三節比賽往往打得還算可以，但球隊除了內線的問題以外，第四節得分難更是老毛病。弗格是球隊的進攻核心，當他遇到趙睿與胡明軒的防守時，能否有上佳表演關係到比賽的走勢。CBA全明星首周票選中，胡明軒入選南區首發引起了一定的爭議。有人說他的數據很平淡，可作為一名新星，在眾星雲集的廣東男籃，數據一般也正常，而且更主要的，他的作用主要體現在防守端。面對弗格，他能證明為何領先CBA全明星票選嗎？就看他的防守以及對比賽的串聯了。當然，也有人提到本賽季狀態迅速回升的孫悅也該有機會入圍全明星賽，馬尚布魯克斯與威姆斯的組合很強勢，這對孫悅的防守同樣是考驗。

能否豪取13連勝，廣東男籃現在最大的敵人就是自己，客場比賽心態很重要，除此之外，就是替補們的發揮，這也是老生常談的問題。與上賽季相比，易建聯的場均出場時間從28.1分鐘漲到了32.1分鐘，杜鋒得想辦法讓年輕內線頂一頂了。
