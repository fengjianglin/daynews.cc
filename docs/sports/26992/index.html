深度：你有高效瓦爾迪？瓜帥用前腰化的丁丁與內切狂魔完成暴擊

2019-12-23 495條評論 1,775次閱讀 296人點贊

主場3-1逆轉萊斯特城，曼城收穫了隊史第1000場主場比賽的勝利，在這場價值與紀念意義同樣突出的比賽中，曼城以讓人信服的方式拿到了3分。德布勞內在前腰位置上的出色發揮，以及在邊路具備持球空間的馬赫雷斯，成為他們逆轉取勝的關鍵。

【出場陣容】

曼城出場陣容：31-埃德森，2-凱爾-沃克，25-費爾南迪尼奧，30-奧塔門迪，22-門迪，8-京多安（78分鐘，16-羅德里），17-德布勞內（90分鐘，10-阿圭羅），20-貝爾納多-席爾瓦，26-馬赫雷斯，7-斯特林（88分鐘，47-福登），9-熱蘇斯

替補未出場：27-坎塞洛，50-埃里克-加西亞，11-津琴科，1-布拉沃

萊斯特城出場陣容：1-小舒梅切爾，3-奇爾維爾，6-埃文斯，4-瑟雲聚，21-里卡多-佩雷拉，25-恩迪迪，10-麥迪遜，8-蒂萊曼斯（77分鐘，26-普拉特），15-哈維-巴恩斯（64分鐘，11-奧爾布萊頓），17-阿約澤-佩雷斯（67分鐘，7-格雷），9-瓦爾迪

替補未出場：12-丹尼-沃德，2-賈斯汀，14-伊希納喬，5-摩根

【始終在尋找空位的德布勞內，是恐怖的】

上輪對陣曼城，德布勞內拿出了驚艷的發揮，2射1傳的出色表現，來自他位置的微調。不再固定於右側肋部進行組織調度的他，從位置上來看更像是一名前腰。而不斷瞄準對方腰位空擋的他，頻繁送出威脅傳遞與轟出精彩的射門。

與上輪聯賽類似，本場比賽德布勞內的位置內收也較多，在兼顧右側肋部作業的同時，實質上扮演著前腰的角色。

圖1：更多地扮演前腰角色，德布勞內不斷找尋空間

上面的組合是德布勞內本場對陣萊斯特城較多的在前腰位置上尋找空位的展現（上圖上端），與上輪3-0取勝阿森納相比（上圖右下角），本場對陣萊斯特城（上圖左下角）德布勞內與鋒線隊友的位置要有寬度拉開上的變化。因為，本場比賽蒂萊曼斯的回撤協防較多，且恩迪迪具備很好的防守覆蓋，這種寬度上的拉開能更多地製造空間。

因此，當德布勞內在前腰位置上覓得空位的時候，他能發揮出自己持球推進以及分球上的優勢，然後讓曼城能打出更加快速、立體的進攻。

圖2：德布勞內在前腰位置上的空位接球與調度

當B-希爾瓦來到左中場位置時，他的持球能力就能保障這一側的有球作業會具備較高的控制力。因此，德布勞內可以安心在中路高位等待分球，然後找到空擋完成調度。

【恩迪迪短板展現，萊斯特城只能由邊打擊身後】

恩迪迪是一名出色的後腰，他在防守保護、機動性拼搶與覆蓋上有很高的水準，只是他也存在短板：在受迫情況下，他不算精細的技術會無法承擔出球的職責。上賽季英超聯賽杯曼聯攻入2球，就是來自他在受迫情況下出球不利被曼聯迅速完成了二次進攻。

此番曼城的高位逼搶很有侵略性，而且，恩迪迪在受迫情況下全場比賽出現了5次受迫性失誤，而且，還有6次向前傳遞的失誤。

圖3：曼城快速合圍，去限制恩迪迪的基礎梳理

蒂萊曼斯在本場比賽中的回撤幅度很大，一方面是幫助協防曼城左側的B-希爾瓦與斯特林，另一方面就是希望能增加球隊向前組織推進的能力，彌補恩迪迪的短板。

但是，在曼城高強度的壓迫下，萊斯特城在中路的推進並不順暢。因此，他們將重點放在了邊路，再通過較多的身後球傳遞去發揮瓦爾迪的優勢，打擊奧塔門迪與費爾南迪尼奧。

圖4：面對曼城的高位逼搶，萊斯特城通過邊路送身後球兌現得分

除了這次得分之外，萊斯特城在上下半場還各有1次有效的身後直塞，最終由瓦爾迪製造了傳中機會，威脅性十足。

【德布勞內前腰化的延伸效用？曼城右路大爆發】

左側以「小球」配合為主去吸引萊斯特城的防線傾斜，然後給中路的德布勞內以及右側的馬赫雷斯製造足夠的作業空間，是曼城本場比賽的一個重要策略。

圖5：馬赫雷斯（左側）與德布勞內（右側）本場比賽製造機會展示

於是，德布勞內與馬赫雷斯覓得了足夠的空間，開始在進攻端瘋狂輸出。馬赫雷斯本場比賽獲得了足夠的開火權，全場比賽他射門次數為10次，是萊斯特城全隊的2倍。而在製造機會層面，倆人也製造了球隊一半的射門機會。

於是，曼城在0-1落後的局面下，連續攻入了3個進球。

圖6+7+8：曼城三個進球的全景展現

上圖6中，門迪左側轉移找到馬赫雷斯，在德布勞內站定高位的情況下萊斯特城忌憚其在肋部的45度斜傳沒有完成對馬赫雷斯的包夾，讓後者成功獲得內切空間製造折射得分；

上圖7中，曼城利用快速進攻的立體化傳遞中，德布勞內雖然沒有在中路完成傳遞，但是他的跑動讓馬赫雷斯的傳球找到了斯特林，後者過人製造點球；

上圖8中，馬赫雷斯在邊路吸引了對方邊後衛的注意，德布勞內找到萊斯特城中場與後衛線的空間，然後完成了推進與精彩過人，給後點的熱蘇斯送出了傳中助攻。

【結語】

在控球率超過63%的情況下，曼城通過德布勞內的內收以及他與馬赫雷斯在右側空間的有效利用，成功攻入3球。在這場強強對話中，關鍵數據佔據絕對優勢的曼城取得了讓人信服的勝利。與萊斯特城積分僅差1分的情況下，曼城接下來能在聯賽中收復多少失地，開始讓人暢想。
