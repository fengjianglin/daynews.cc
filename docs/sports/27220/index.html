等了好久！國足選帥程序下周正式啟動，三候選中李鐵暫時領跑

2019-12-21 98條評論 2,312次閱讀 337人點贊

李霄鵬、李霄鵬、王寶山（從左至右）

2019年還有最後的10天，而對於中國足球來說，這行將過去的一年並不算是十分順利，尤其是國足的表現，始終存在一定的爭議，而在里皮1個月前辭職之後，有關他繼任者的猜測也一直未曾停止，最新的消息顯示，下周五（27日），中國足協教練員委員會和專家團，將對李鐵、李霄鵬、王寶山進行面試，如果不出意外，國足新帥將從這3人當中產生。而三位「土帥」中，李鐵的位置更為靠前。

其實從上個月國足兵敗阿聯酋、里皮辭職之後，誰將接替他成為新任國足掌門人就成為了國內足球圈最吸引人的話題，洋帥還是土帥？什麼樣的人適合現在的國足等等問題都是外界關注的焦點。而無論是誰在這樣的情況下接手國足，都需要做好全方面的準備，成績是一方面，當然還有來自輿論的壓力，畢竟國足主帥的位置不好做，而如果是土帥的話，這樣的壓力沒準會更大一些。

不過相比外教團隊，本土教練員雖然在執教能力上存在一些差距，但要論對於國內足球環境以及隊員的了解程度，沒人比他們更熟悉，而在這些本土教練之中，尤以李霄鵬、李鐵和王寶山這三位最為突出。其實熟悉中國聯賽的人都清楚，這三人也是剛剛過去的這個賽季，16支中超球隊中僅有的3名率隊征戰完整賽季的本土教練員。他們最終能夠進入到足協選帥團隊的「法眼」，成為候選人，這本身也是對這三人能力的一種認可。

三人中年齡最大的王寶山是國內足壇有名的「救火隊長」，北京人和、深圳佳兆業都曾留下過他的身影，而本賽季在河南建業，他更是把自己善於調教實力偏弱球隊、擅長防守反擊的特點發揮的淋漓盡致，最終在俱樂部成立25年這一歷史背景下，王寶山和他的河南建業隊最終獲得了聯賽第8名，41個積分中絕對有教練員的功勞，而成為國足主帥的候選人，他靠的不僅僅是技戰術方面的能力，同時其發掘和挑選年輕人的眼光也是他受到關注的重要因素。

相比之下，山東魯能的李霄鵬顯然順位會更靠前一些，作為連續兩年的中超最佳教練員，李霄鵬打造的那支山東魯能隊特點非常鮮明，雖然2次和足協杯冠軍擦肩而過，但是這位青島漢子的能力毋庸置疑，這也就是為什麼大家對於李霄鵬出任國足主帥的的前景非常看好。還有一點不得不說，那就是他超強的人格魅力，每次山東隊到客場比賽，李霄鵬都會是那個「忙碌」的人，因為場邊總會有對手的球員、教練員過來主動和他打招呼嘮家常，這樣高情商的教練員不也正是國足需要的人選嗎？

作為三人中最年輕的教練員，同時也是做教練時間最短的人，李鐵似乎並沒有太多的優勢可言，但是憑藉連續兩年執教武漢卓爾隊的出色表現，只有42歲的李鐵在上月初成為了國家選拔隊的主帥並率隊參加了本次東亞杯的爭奪，在人員並不齊整的情況下，他帶隊取得了1勝2負的戰績，而這樣的成績並非不可接受。據了解，足協高層也對這次東亞杯的「考核」結果基本滿意，由此看來，李鐵或將領跑這次的國足選帥。

根據安排，下周五（27日），足協教練員委員會、專家團對李鐵、李霄鵬、王寶山進行面試，如果不出意外，國足主帥將從這三人中選出。至於正式人選到底何時公布的問題，其實並不會太久，畢竟新賽季的中超聯賽大概率會在2月底開始，而在那之前亞冠小組賽和超級盃也都會陸續開賽，對於教練團隊來說，他們需要從那時就開始對於新一期國足隊員進行甄選工作。由於3月份國足就將迎來和馬爾地夫的世預賽次回合爭奪，留給國足新帥的時間其實並不富裕。

文/北青體育 張昆龍
