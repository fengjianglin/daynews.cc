國乒傳喜訊！U15世界第一正式入選國家隊，未來對抗張本智和？

2019-12-12 234條評論 2,247次閱讀 879人點贊

北京時間12月11日，中國乒協在官網宣布，陳垣宇、徐奕、梁儼薴正式入選國家二隊，這一重要決定讓國乒隊伍進一步壯大，無疑是一大喜訊。其中，年僅14歲的陳垣宇最受期待，他目前在國際乒聯U15年齡段排名世界第一，之前曾參加過青少年公開賽克羅埃西亞站比賽，一舉加冕雙冠王，是一位非常有潛力的新星，未來或許有機會代表國乒來對抗比他大兩歲的張本智和。

挖掘和培養優秀青年少球員一直是國乒的重要工作，只有人才不斷湧現才能讓國乒持續強大。這兩年，日本00後球員迅速崛起，個別球員在某些比賽中能給國乒製造威脅了，比如張本智和、伊藤美誠。之前結束的世青賽，日本年輕球員也帶給我們巨大衝擊，年輕球員的培養工作再次成為討論焦點。劉國梁也在之前表示要組織青少年集訓隊，進一步拓展人才選拔渠道。

如今，人才選拔又有收穫，三位年輕球員經過考驗後進入國家二隊。因為國乒優秀人才實在太多，能夠入選國家隊當然是優中選優的人才，三人都在選拔賽中取得了非常出色的成績，像陳垣宇、徐奕在PK賽中分別獲得男子組和女子組單打第一名，這才獲得了入選國家隊的機會。而2001年出生的梁儼薴是在二青會體校甲組比賽中脫穎而出，得以入選國家隊。

三位球員中，陳垣宇非常引人關注，他之前進入過大家的視線中，因為在省隊表現非常出色，陳垣宇獲得了代表國家隊參加國際乒聯青少年公開賽克羅埃西亞站比賽的機會。他在那次比賽大放異彩，先是率隊獲得U15組男子團體冠軍，接著又在U15組男單決賽中笑到最後，成功加冕雙冠王，可以說是充分施展了自己的潛力和才華。他也憑藉這些成績排名U15組別世界第一，在U18組別排名第32，相當不錯了。

另外，陳垣宇還在江蘇隊師從目前許昕的教練陳玘，他在那段時間取得了非常大的進步，國內外賽場的出色表現讓陳垣宇更有自信了，最終在PK賽中脫穎而出。進入國家二隊只是第一步，他要努力進步，為自己爭取更多參賽機會，就像向鵬那樣，從青少年組打到成年組，最終要為進入國家一隊而努力，這樣才有機會脫穎而出。

從年齡看，如果發展順利，陳垣宇未來有機會對抗張本智和，這當然是很遙遠的事情，需要陳垣宇不斷努力。國乒人才選拔體系健康又嚴格，每一位入選國家隊的球員都有很大的潛力，但真正能獲得奧運冠軍的還是少數，需要球員們拼盡全力，要不然就會把機會白白浪費！
