國奧大勝非洲勁旅獲得亞軍，還有一大利好是練成了我們的必殺技

2019-12-15 120條評論 1,080次閱讀 65人點贊

⚽️還有不到一個月的時間，U23亞洲杯也就是東京奧運會預選賽將至，中國國奧隊目前在珠海進行最後的練兵熱身。由於此次預選賽沒有絲毫退路，被分進死亡之組的中國國奧需要儘快找到自己的節奏，尤其是在進攻環節有辦法，才有一絲可能實現突圍目標。

⚽️上一場，國奧0-1惜敗敘利亞隊的比賽，前鋒張玉寧屢屢錯失得分機會，確實令人遺憾。不過賽後也並非全是悲觀的聲音，畢竟國奧隊在與敘利亞隊的比賽中還是踢出了不少流暢配合，由於運氣和臨門一腳等因素未能破門。但是熱身賽就是要通過實戰來給隊員找問題和尋找套路配合的，沒有必要過度否定自己。

⚽️14日晚，國奧在四國賽最後一場比賽與馬里隊踢出了流暢的進攻，第25分鐘國奧隊從半場帶球只有簡單的兩腳傳中，一腳是邊路直塞球，另一腳是禁區邊緣高速帶球中的橫傳，最終張玉寧用他擅長的魚躍頭槌完成破門。這是四國賽張玉寧個人首個進球，同時本場比賽國奧隊用類似打法找到了屬於自己的必殺技。

⚽️張玉寧完成進球後，國奧隊繼續沿用這個套路隊馬里發起猛攻。長傳也好，沖吊也罷，此前國足站樁式的打法無論是在什麼樣的對手面前都難以奏效。本場比賽，國奧隊充分利用高速突破和快速直塞，再加上邊路45°或邊路傳至門前的快速傳中完成進攻套路。這樣的打法有效解決了兩大國足短板，首先是停球問題，如果頻繁長傳尤其是高空球，國足球員基本功難以保證人人都完美停球。而地面快速直塞球，在傳控到位的情況下倒是可以發揮出球員們的特點；其次，還可以解決短傳配合難以持久的問題。

⚽️陳彬彬、段劉愚本場比賽都有不錯的發揮，在張玉寧破門後不久，又一次通過直塞球形成單刀險些再次破門。上半場陳彬彬的反擊進球也是快速直塞，接球後快速完成最後一腳射門。地面球如果在短傳配合中存在進攻套路難以實現的情況下，那麼直塞對方身後，利用快馬型球員的特點，加上張玉寧這樣具備衝擊力的中鋒去搶點，那麼還是可以找到屬於自己的殺手鐧。比賽進入到下半場，張玉寧還給段劉愚送出過一記直塞球，段劉愚快速插上門前倒地射門也是險些破門。此外，胡靖航也通過前場的地面簡單傳球得到過一次射門得分的機會，可惜被對方門將撲出。最終國奧3-0大勝馬里，2勝1負獲得了此次四國賽亞軍，整體看一場比一場好。

⚽️本場比賽郝偉的進攻戰術就是跑、傳、射，簡單的來講就是敢於突破，然後盡量在地面完成直塞球，球路向空檔傳，人也往關鍵位置跑，最後形成射門。當然，這裡的跑動和以往最大不同，那就是不能等進攻球員到位再傳，要踢出提前量。至於國奧的必殺技是否會在U23亞洲杯奏效，也需要球員們不斷加強類似套路的磨合。沒有技術、沒有配合，那麼就用速度和直塞空檔的辦法衝擊對手，這種打法還考驗我們的體能和衝擊力。希望四國賽能讓國奧在預選賽創造奇蹟。
