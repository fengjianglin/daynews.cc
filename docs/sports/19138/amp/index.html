鄧亞萍話裡有話？奧運人選不是看誰打得最好，而是誰最穩

天天要聞
2019-12-17

2019年國際乒聯巡迴賽總決賽昨晚落幕，在壓軸的男單決賽中，樊振東4-1力壓馬龍，奪得冠軍。

當央視賽事主播黃子忠詢問嘉賓鄧亞萍關於國乒東京奧運會人選時，這位昔日的「大魔王」莞爾一笑：中國隊的選人標準不是看誰打得最好，而是看誰最穩。

自楊影淡出後，素以解說排球賽事聞名的黃子忠開始兼項乒乓球。他對賞心悅目的「龍胖大戰」讚不絕口，當馬龍比分落後時他幾次用了「懸崖勒馬」這個成語；當樊振東打出暴力殺板時他又稱呼小胖是「小魔王」。其實懸崖勒馬與破釜沉舟、背水一戰的意思多少還是有些區別的；而「小魔王」的稱號是今年球迷特地獎勵孫穎莎的，是女乒專用詞，每當黃子忠說這兩個詞，鄧亞萍也不好意思當面糾正，只能在一旁訕笑。

「龍胖大戰」結束後，點評時，黃子忠趁機問了鄧亞萍一個球迷最為關心的話題，明年東京奧運會男乒人選沒啥懸念，但女乒會派誰去？你會覺得頭疼嗎？鄧亞萍回答：我不頭疼，教練組（劉國梁）應該最頭疼，因為國乒奧運用人從來不是看誰打得最好，而是最穩，最強調一個「穩」字，所以這裡面的變數很大，按照慣例不到明年5月份，國乒不會輕易公布最終奧運名單。

鄧亞萍這麼說，看似與劉國梁的說法不謀而合。此前劉主席曾多次表示，國乒大賽用人不是看誰的狀態最好，而是看誰狀態最差的時能不能贏球，總而言之，就是看誰最穩。尤其是代表最高榮譽的奧運會，更看重隊員外戰的穩定性。

問題是「誰的狀態最好」有清晰具體的數據（外戰成績）作支撐，譬如2019賽季的孫穎莎，毋庸置疑排女乒第一；而「穩定性」卻沒有一個十分明確具體的標準，譬如本賽季的陳夢，世乒賽獲得亞軍，最近兩次擊敗頭號勁敵伊藤美誠，並完成了總決賽三連冠，卻陰溝裡翻船輸給了二流對手石川佳純、馮天薇、田志希（狀態差的時候並沒有贏球），其「穩定性」又該如何判斷？

說到底，外戰「打得最好」與外戰「最穩」之間沒有一道不可逾越的鴻溝，二者必然相輔相成。打得最好的肯定是發揮比較穩定的；而打得最穩的成績肯定也是挺好的。

不過話得說回來，未來Ta是否繼續「打得最好」，是否繼續「最穩」，卻是個未知數，誰也沒本事能未卜先知，我們只能根據Ta近一個時段的表現作為參考依據。

由於國乒的奧運人選至今沒有一個公開透明的選拔機制，就不排除有時候領導會以所謂「成績最好」為由否定「最穩」的，有時候又以所謂「最穩」為由否定「成績最好」的，反正橫豎都有道理（誰嘴大誰說了算），以此來增大自己操作的空間，這樣的例子從1988年漢城奧運會（堅決不用世乒賽冠軍、外戰第一、內戰第一、排名世界第一的何智麗）起，就屢見不鮮。

作為過來人，鄧亞萍也許由衷的讚賞國乒持之以恆的奧運傳統用人方法。作為旁觀者清，也許她認為什麼事情都必須與時俱進，以人為本，到時候了，國乒應該有一個相對公開透明的奧運選拔機制了。

各位怎麼看？這裡是《精英評球》，歡迎回復，發表不同意見。
