比利·喬·桑德斯的野心，下一戰希望對決阿瓦雷茲或史密斯

天天要聞
2019-12-21

北京時間2019年12月20日，現任WBO超中量級世界拳王比利·喬·桑德斯（Billy Joe Saunders）在接受英國天空電視台體育頻道記者採訪時表示，自己希望在2020年能夠挑戰拳擊巨星蘇爾·阿瓦雷茲和卡勒姆·史密斯兩人中的一位。

比利·喬·桑德斯告訴天空體育記者，他希望能在2020年能戰勝這兩位拳手的中的一個。目前，卡勒姆·史密斯是現任WBA超中量級世界超級拳王，同時也被認為是職業拳擊超中量級的第一人。而墨西哥超級巨星蘇爾·阿瓦雷茲也曾擁有WBA超中量級世界常規拳王頭銜，具備在168比賽的實力。

就在上個月，蘇爾·阿瓦雷茲連越兩級上升到175磅，擊敗了綽號「碎骨機」的謝爾蓋·科瓦列夫（Sergey Kovalev），獲得了WBO的輕重量級世界拳王金腰帶。但是幾天前，蘇爾·阿瓦雷茲宣布放棄了自己的WBO的輕重量級世界拳王頭銜，現在的他很有希望回來參加超中量級的比賽。

有消息稱，蘇爾·阿瓦雷茲已經計劃在明年（2020年）五月回歸拳台，參加墨西哥傳統節日五月五日節（Cinco De Mayo）的周末比賽，而比利·喬·桑德斯也被認為是最有可能對陣這位墨西哥拳擊巨星的候選人之一。

比利·喬·桑德斯對天空體育記者說：「我現在想要打大型的比賽已經不是什麼秘密了。拳迷們都在談論我，提到我的名字。而我想說的是我希望能和蘇爾·阿瓦雷茲比賽。希望埃迪·赫恩能促成這場比賽，我已經把我的想法告訴了我的團隊MTK，他們正在和埃迪·赫恩談判，所有人都非常有信心讓這場比賽成功舉行。為了我和我的拳迷們，我真的希望這場比賽能在2020年的新年舉行。

當被記者問及對本場比賽是否有信心時，比利·喬·桑德斯回答說：「我相信我能擊敗蘇爾·阿瓦雷茲，我想自己缺的只是那個黃金機會。」

當然，在比利·喬·桑德斯的候選對手名單上，不會僅僅只有蘇爾·阿瓦雷茲一個人。超中量級的第一人，英國拳手卡勒姆·史密斯也是比利·喬·桑德斯的目標，而且如果比賽在英國本土舉行的話，卡勒姆·史密斯絕對會是比利·喬·桑德斯的首選對手。

比利·喬·桑德斯對天空體育記者說：「我認為我和卡勒姆·史密斯會為拳迷們奉獻一場精彩的比賽。我和他都是英國人，比賽運作起來很容易。如果那些所謂的世界拳擊巨星沒有一個想和我們兩個比賽，那麼就讓我們兩個來一場世界拳王統一戰吧。」

在整個採訪的最後，比利·喬·桑德斯再次對記者強調：「在2020年，我想要和蘇爾·阿瓦雷茲或者卡勒姆·史密斯比賽，沒有其他人。」
