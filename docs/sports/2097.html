太意外！曝穆帥親自召喚中超巨星，魯能冬窗離隊第一人或已定頭牌

2019-12-09 0條評論 4次閱讀 0人點贊

這些年，中超的大牌外援變得越來越多了，導致歐洲球隊也不再排斥從中超引進球員，其中典型是巴薩花了4000萬歐元從廣州恆大引進了保利尼奧，讓恆大外援成為了中超的一大招牌。如今2019年即將結束，2020年冬窗即將開啟，中超聯賽又將出現巨星遭搶的現象。

來自比利時媒體「Dhnet」的最新重磅消息，剛剛擔任英超球隊熱刺主帥的穆里尼奧親自召喚中超巨星費萊尼，該媒體表示山東魯能在足協杯失利之後，費萊尼接到了一個電話，談了數小時，而打電話的正是穆帥，談話內容也遭到了曝光，穆帥已經在召喚愛將追隨他加熱刺，並且視他為熱刺冬窗的頭號引援。

眾所周知，穆里尼奧在執教曼聯時與費萊尼有過夢幻的合作過程，費萊尼成為了穆帥麾下的一名萬能戰士，幾乎可勝任場上任何位置，並且總能成為比賽中的X因素，屢屢率隊涉險過關，穆帥對他是信賴有加，在穆帥離開曼聯之後，費萊尼隨即遭棄，最終選擇加盟山東魯能，在魯能本賽季的表現非常驚艷，堪稱魯能版的「暴力鳥」，攻守全能，以中場球員的身份打進8球助攻3次，並且一躍成為球隊頭牌球星，高達950萬歐元的身價已經是魯能隊內最高的了。

目前費萊尼只有32歲，正值職業巔峰，穆里尼奧引進他顯然一點也不意外，他的支點作用一直都是穆帥的重要棋子。這樣一來，魯能冬窗離隊第一人或已鎖定頭牌球星費萊尼，他將二度追隨穆里尼奧，魯能方面為此可能回收一定的轉會費，費萊尼與魯能的合同原本是在2021年到期，根據權威轉會網站「德轉」的信息顯示，費萊尼當初從曼聯加盟魯能時的轉會只有720萬歐元，此次熱刺搶人難度其實不大，加上穆里尼奧親自開口，魯能回收1000萬歐元左右的轉會費也將成為可能，要知道當初穆帥在曼聯時為了引進費萊尼擲出了超過3000萬歐元的轉會費。

對於魯能來說，冬窗勢必又將面臨一輪重組，好在魯能在外援的引援方面一向追求性價比，費萊尼離去也不會給魯能造成太大困擾，相比之下佩萊的逐漸老邁才是最棘手的問題，新賽季佩萊雖然能繼續留隊為球隊攻城拔寨，但是體能和各項指標定然會下滑，畢竟他新賽季將達到35歲，魯能冬窗可能要補進能一名夠輪換佩萊的頂級外援，確保魯能鋒線始終保持競爭力，期待魯能冬窗的調整。
