北京時間12月4日晚19:35，新一輪CBA比賽繼續上演，6場精彩的對決不容錯過。今晚新疆將要對陣江蘇，積分榜上新疆緊追榜首的廣東隊，目前取得了10勝2負的優異戰績，對手江蘇男籃6勝6負，排名積分榜中游。新疆隊內的中鋒周琦是絕對核心，儘管在NBA和今年的籃球世界盃上，他的發揮為人詬病，但是回到了CBA聯賽，大魔王依舊打出了不俗的表現，場均22.6分+12.2板成為球隊贏球的基石。

遼寧VS浙江也是今晚比賽的重頭戲之一，12輪戰罷，遼寧男籃8勝4負排名第六，浙江以6勝5負排名第八。遼寧隊的大外援史蒂芬森擔當球隊贏球保障，目前以26.4分的場均得分成為隊內得分王，另外中國球員郭艾倫也是球隊的重要一環，他在國內球員的得分榜上排名第四。浙江這邊外援馬庫森-鄧蒙搭檔中國球員吳前，組合成後場雙槍，活力上而言不比別的球員差不少，今晚的對決應該是一場精彩的較量！
