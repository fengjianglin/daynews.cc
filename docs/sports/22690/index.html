2-0！中國足球終於贏了，結束6場不勝尷尬，送對手東亞杯9連敗

2019-12-19 731條評論 4,072次閱讀 826人點贊

北京時間12月18日下午3點15分，2019年東亞杯最後一輪的一場焦點戰在韓國釜山的亞運主體育場打響，兩支前兩戰全敗的球隊迎來正面交鋒，中國男足選拔隊對陣中國香港代表隊，最終憑藉吉翔的開場破門和張稀哲下半場的點球破門，國足2-0擊敗對手，時隔近10年再度戰勝中國香港隊，以一場勝利為2019年收官，結束東亞杯連續6場不勝的尷尬，送給對手東亞杯9連敗。

東亞杯前兩場比賽，國足以1-2和0-1的比分先後不敵日本和韓國，而中國香港則是0-2、0-5不敵韓國和日本。相較於兩天前不敵韓國的首發陣容，李鐵此役作出5處調整，劉殿座繼續擔任首發門將，李昂傷愈回歸，搭檔于大寶組成雙中衛，王燊超和吉翔分居左右，木熱買提江和李行搭檔後腰，董學升單前鋒突前，張稀哲在其身後，曹贇定和王子銘分居左右。

開場不到8分鐘，國足便通過角球機會取得領先，張稀哲開出左側角球至禁區，中國香港5號後衛艾里奧前點冒頂，基奧雲尼門前反應不及，吉翔後點將球頂進球門。

在國足取得領先後，場邊的國足選拔隊主帥李鐵也揮拳怒吼慶祝。

第17分鐘，中國香港隊險些將比分扳平，中國香港隊前場右路打出配合，基奧雲尼禁區右側一腳抽射，劉殿座撲了一下後，皮球擊中門框彈出。

第35分鐘，國足前場左路自由球機會，張稀哲開出自由球至禁區，吉翔禁區右側頭槌擺渡至中路，于大寶在點球點附近完成倒鉤射門，可惜偏出右側立柱。

第39分鐘，場上出現惡劣犯規，王子銘右邊路帶球高速推進，效力於中甲梅州客家的中國香港隊中衛徐宏傑將王子銘鏟倒，亮鞋底雙腳離地，好在沒有實打實踹在王子銘小腿上，但這次犯規相當惡劣，是一次夠得上紅牌的犯規動作，不過來自來過的主裁判布烏敦手下留情，向其出示黃牌警告。

上半場戰罷，國足1-0領先中國香港，技術統計方面，國足獲得七成控球率，不過兩隊射門數均是兩次，國足1次射正轉化為進球。

下半場雙方易邊再戰，開場前國足作出換人調整，譚龍替補登場換下王子銘。董學升和李昂先後射門製造威脅，第71分鐘，國足利用點球機會擴大領先優勢。董學升禁區內背身接李行傳球，轉身時被對手5號後衛艾里奧絆倒，主裁判果斷判罰點球。

張稀哲主罰點球騙過門將輕鬆罰入。

1分鐘後，國足再次換人，韋世豪替補登場，迎來本屆東亞杯首秀，換下曹贇定。

第80分鐘，國足錯失破門良機，于大寶中場附近帶球推進至前場後送出直塞，譚龍右路插上在禁區右側底線附近挑傳門前，韋世豪門前高高插上頭槌攻門，皮球擊中橫樑彈出。

1分鐘後，國足用完最後一個換人名額，蔡慧康替補登場，換下買提江。最終全場比賽結束，國足2-0擊敗中國香港，取得本屆東亞杯首勝，也結束了國足在東亞杯的6場不勝。以一場勝利為2019年收官。

這也是國足時隔將近10年再次擊敗中國香港隊，國足上一次戰勝對手還得追溯至2010年初的東亞杯，國足憑藉曲波的梅開二度擊敗對手，此後國足與中國香港在2018年俄羅斯世界盃預選賽上交手兩次，比分均是0-0。而第三次參加東亞杯決賽階段的中國香港則又一次以三戰全敗收場，遭遇9連敗。
