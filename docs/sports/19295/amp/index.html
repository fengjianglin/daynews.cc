導語：15分大勝廣東後，馬布里接受採訪，真不給杜鋒一點面子！

CBA第17輪，廣東客場挑戰北控隊，目前廣東隊穩居積分榜首位而北控排在第10位，從整體實力上看廣東隊比北控隊強上一籌。比賽開始馬尚助攻胡明軒在籃底打進，隨後北控弗格回敬三分，任駿飛在籃底打進反超比分，首節北控隊36:30領先廣東隊，次節兩隊繼續進行對攻大戰，威姆斯命中三分北控弗格再還一個雙方你來我往互不相讓，比分一直維持在7分左右，次節北控隊以68:61領先7分。

易邊再戰廣東隊加強進攻一度把比分縮小到3分，關鍵時刻北控外援再度建功內突外投連得5分，廣東隊進攻突然停滯，北控隊抓住機會把比分拉開到16分，末節廣東隊未能縮小比分，杜鋒教練再換上杜潤旺利用3分球追分，但是時間已經不多，孫悅命中3分幫助北控隊鎖定勝局，最後北控隊以126:111戰勝廣東隊，送給廣東隊本賽季第2場敗局，同時也終結了廣東隊的12連勝。

數據方面：廣東隊，威姆斯28分8籃板8助攻、胡明軒18分4助攻、馬尚16分7籃板4助攻、任駿飛13分5助攻2籃板、易建聯9分8籃板3助攻2蓋帽、王薪凱9分5籃板、趙睿8分5籃板3搶斷2助攻、杜潤旺8分、曾繁日2分。北控隊，弗格38分8助攻6籃板3搶斷、湯普森19分13籃板3助攻、桑尼18分11籃板、王子瑞16分3助攻2籃板2搶斷、孫悅9分5籃板4助攻2蓋帽、王少傑8分5籃板、孫桐林7分、許夢君6分、張帆5分。

值得一提的是在比賽結束後，記者採訪問杜鋒教練對這場比賽的看法，杜鋒教練直言不諱地說，不是北控隊有多強，是我們自己隊伍有很多失誤，取勝慾望不強烈，空位投籃不進，這些才是廣東隊輸球的關鍵。簡單說了幾句杜鋒教練就結束採訪。隨後馬布里在採訪中說道，今天能贏球是隊員們把我的戰術執行到位了，在攻防兩端都打出了應有的水平，所以我們贏得這場比賽。

這段採訪也是挺有意思的，有球迷評論杜鋒教練有點小家子氣了，第一句話就說對方不強是自己的問題。怎麼都要客套兩句贊一下對方打得好，這樣是明擺著不把北控跟馬布里放在眼裡啊。馬布里在後面的採訪也反擊了杜鋒，只要隊員把馬布里的戰術執行到位，是能戰勝廣東的，所以廣東隊並不可怕。馬布里也是一點面子都不給杜鋒啊。不過從今天兩隊的發揮來看，北控的在攻防兩端確實做的比廣東好，但廣東隊整體實力依然是CBA聯盟獨一檔，希望杜鋒教練好好總結本場比賽失利的原因，在以後打出更精彩的比賽。
