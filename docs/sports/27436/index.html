廣東主場迎戰聯賽弱旅，易建聯還是不能輪休？欲戴王冠，必承其重

2019-12-20 738條評論 2,098次閱讀 952人點贊

北京時間12月20日晚上8時許，廣東東莞銀行結束兩連客之旅，回到主場迎戰聯盟僅僅取得四場勝利的天津先行者，不出意外，廣東應該能輕取對手。

那麼，杜鋒有可能讓阿聯輪休嗎？答案是否定的，不可能。

一、易建聯上一場比賽出場時間低於30分鐘還是在11月29日

阿聯已經連續七場比賽上場時間超過30分鐘，並且其中只有兩場比賽是在主場對陣「浙江雙雄」，但是上場時間確實這幾場比賽中最高的，對陣稠州銀行上場38分鐘，對陣浙江廣廈上場36分鐘。

本賽季阿聯上場時間比上一賽季還有所增加。畢竟阿聯已不再年輕，已年過三旬。歲月不饒人吶。因此阿聯有個短暫的低迷期，對陣浙江稠州銀行，阿聯14投5中，僅得14分；對陣北京更是11投4中，僅得9分。這是阿聯本賽季第一次得分不上雙。

由此可見，上場時間的增多，再加上旅途勞累，對阿聯的身體狀態和體能均有影響。好在阿聯及時調整狀態，在上一場對陣江蘇的比賽找回了手感，17投9中，罰球9罰9中，得到27分15個籃板。

阿聯是聯賽中20隻球隊中唯一超過30歲的本土核心，也說明球隊對阿聯的倚重，尤其是阿聯在內線的威懾力，阿聯的防守效率是整個CBA最高的。

二、時間增加的源頭是球隊的傷病

上賽季阿聯的上場時間是職業生涯新低的28.1分鐘，不僅和球隊控制其上場時間，和周鵬也有很大的關係。

在某種程度上來說，周鵬比現在廣東外援更重要，周鵬攻擊力弱於外援，但是防守更強。能防5個位置，可以無限換防。最重要是，周鵬能打滿四節，外援是四節6人次。周鵬就在一定程度上分擔了阿聯上場時間。

而本賽季，周鵬缺陣至今，還有可能賽季報銷，這對廣東隊的衛冕征程蒙上了一層陰影。

另外蘇偉的受傷，也在一定程度上增加了阿聯的上場時間。雖然每場比賽蘇偉上場時間不是很多，但總會給阿聯分擔十分鐘左右的時間。尤其是遇到對手中有高壯內線的時候，蘇偉作為實打實的藍領中鋒，有噸位，有體重，可以為阿聯在內線分擔不少壓力。

三、阿聯為什麼不能輪休？

1.替補內線不能頂上，內線無人可用

球隊替補內線還沒有成長為主力球員的潛質。比如16年就進進球隊一線隊的曾繁日經過數年的比賽，還是沒成長起來。萬聖偉作為CUBA明星球員，球商好，但是作為內線球員，201CM的身高是硬傷。

而天津隊內線有年近27歲的身高210CM的黑山中鋒托多羅維奇（場均能貢獻20.9分13.0籃板），還有本土年輕內線時德帥（場均能貢獻16.9分7.2籃板）。時德帥本賽季進步明顯，場均得分只排在國內三大明星內線大王、周琦和阿聯之後。另外天津隊還有場均得分9.6分的田雨和8.4分的孟子凱等內線球員。

由此可見，天津內線還是相對於廣東隊阿聯的獨木難支，還是有優勢的。如果阿聯不上，豈不是任由天津內線球員在廣東隊禁區予取予求。

2.球隊戰術需要

阿聯的缺陣，廣東拿手的雙小外戰術根本打不起來，到時候別說輕鬆贏球，一不小心就可能被對手爆冷。

阿聯和球隊的兩個小外援配合嫻熟，當球隊下快攻時，阿聯能快下跟進扣籃得分，也能拉邊，為持球者拉開空間。

尤其廣東經常使用的布魯克斯、威姆斯和阿聯的擋拆戰術，這個戰術沒有阿聯，就沒法執行。

3.聯賽招牌，票房保證

阿聯是目前CBA聯賽的招牌巨星，在CBA，阿聯就是相當于勒布朗-詹姆斯對於NBA的存在。

再加上本場是主場作戰，阿聯打的時間肯定不能太少，更別說輪休了。否則對喜歡他的球迷恐怕會顯得不太友好，而且也會影響到球館的上座率。

結語

阿聯聯的替補乏力實在是廣東的一大隱憂。如果讓87年的易建聯均均戰鬥30多分鐘，估計到聯賽後半段尤其到了季後賽，易建聯的狀態可能會受影響，廣東隊就會得不償失了。

1.短期調整

本場比賽最大的可能就是杜鋒會繼續讓易建聯首發，但是會把他的出場時間切割得更碎，盡量保證他用最節省體力的方式打球，變相的讓他得到休息。以後遇到弱隊，杜鋒盡量如此為之。

2.長期規劃，未雨綢繆

在目前常規輪換內線曾繁日難堪大用的情況和又難以從外部挖到合適內線的時候，只有內部挖潛了。

其實球隊還有一位內線的未來之星，那就是今年才從二隊升上來的李英博。李英博99年出生，205cm的身高雖然有欠缺，但彈跳力好，球風硬朗，有一定的護框能力。

如若加以鍛煉，開發出更多的得分手段，未來還是有很大的發展空間的。比如北控新秀王少傑就是很好的例子。

欲戴王冠，必承其重

阿聯的成長速度堪比坐火箭，從剛進CBA時的6.5分，再到連續兩個賽季得分分別得到10.1分、17.4分。

再到阿聯的第四個賽季，阿聯已經坐穩球隊主力位置，場均能得到20.7分，且在總決賽幫助球隊奪冠，自己也榮膺總決賽MVP，這也是有史以來最年輕的總決賽MVP（當時阿聯年僅19歲）。

CBA的賽場已經不適合阿聯了，阿聯再打了一個賽季後，就逐夢NBA，以第六順位被雄鹿選中。

再到後來阿聯成為國家隊主力。隨著姚明退役，阿聯已經成為了國家隊的不二核心。雖有在家門口輸掉亞錦賽，今年本土世界盃，中國男籃甚至無緣直通明年東京奧運會，但阿聯已經一步一步成長起來了，懂得了如何做球隊領袖。

無論是國家隊還是CBA聯賽，阿聯都是票房保證，所以他必須得上場。這大概就是「欲戴王冠，必承其重」吧。
