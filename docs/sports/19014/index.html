著名拳擊推廣人阿魯姆：“翻版梅威瑟”下一戰可能大戰波特

2019-12-18 719條評論 2,018次閱讀 258人點贊

美國拳擊新聞網12月17日訊 今天，現WBO次中量級拳王/三個級別的世界冠軍特雷恩斯-克勞福德（36-0，27KO）的推廣人鮑勃-阿魯姆在接受媒體採訪時表示，他認為前WBC次中量級拳王肖恩-波特（30-3-1，17KO）是克勞福德下一戰最有可能的對手。

阿魯姆說：「縱觀整個次中量級，我認為肖恩-波特與克勞福德的對決是最勢均力敵的。在肖恩-波特與小斯彭斯比賽前，我並不這麼認為。但是你知道的，肖恩-波特以十分微弱的比分輸掉了比賽。在那場大戰中，肖恩-波特控制了小斯彭斯。正因如此，現在每個人都在說克勞福德VS波特會是一場精彩的大戰。沒有人知道克勞福德與肖恩-波特的比賽能否打起來，但肖恩-波特絕對是克勞福德的絕佳對手。」

對此，克勞福德說：「我與肖恩-波特的比賽由我們自己說了算，而不是由鮑勃-阿魯姆（克勞福德推廣人）與艾爾-海蒙（波特推廣人）決定。如果我和肖恩-波特決定了要大戰一場，我們就會開戰，其他任何人都沒有決定權。你知道的，現在我們還沒有進行洽談，所以談論比賽還為時尚早，我甚至都沒有在思考與波特的戰鬥。」

上周日，克勞福德在紐約麥迪遜花園廣場9回合KO了卡瓦利奧斯卡斯（21-1-1，17KO），成功衛冕WBO拳王金腰帶。賽後，頂級推廣公司開始對波特喊話，希望促成這場次中量級大戰。不過波特昨天表示，他的首選目標是帕奎奧，並非克勞福德。
