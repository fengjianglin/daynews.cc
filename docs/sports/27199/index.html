挑戰福建避免三連敗 韓德君缺陣的遼籃還能以快制高嗎

2019-12-21 997條評論 3,225次閱讀 2062人點贊

在接連負於青島隊和山東隊之後，遼寧男籃22日晚將迎來連續四個客場中的第三個客場比賽，對手是已經獲得四連勝的福建隊。面對狀態出色的對手，主力中鋒韓德君將缺陣的遼籃將困難重重。要想避免三連敗，如今的遼籃必須要像上賽季季後賽大韓不在的情況下，打出速度「以快制高」，但問題是以遼籃目前的狀態還能夠打出流暢的快攻戰術克敵嗎？

前19輪常規賽，遼寧男籃的成績為12勝7負，排名已經滑落至第6位。福建隊本賽季開局糟糕，但該隊啟用原山東隊主教練凱撒出任新帥後，狀態馬上回升，目前以7勝12負的成績排名第15位。從實力上來看，遼籃應該佔據一定優勢，但對比兩隊目前的狀態，則福建隊佔優。本場比賽困擾遼寧男籃的最大問題就是韓德君的缺陣，大韓上場左膝受傷，雖然不太嚴重，但為了保險起見，此役會休戰養傷，這將導致遼籃內線實力下降。

福建隊的內線實力出眾，尤其是國家隊主力中鋒王哲林是本土一流中鋒，沒有了韓德君的遼籃要想限制他將會十分困難。上場比賽，福建隊戰勝了「巨人殺手」吉林隊，王哲林全場得到28分10個籃板5次助攻4次封蓋的全面數據。此役面對內線實力不足的遼籃，王哲林仍然將是福建隊的主攻點。韓德君無法出戰，遼籃內線只有李曉旭、巴斯、賀天舉、王化東四人，但四人在身高方面都無法與王哲林抗衡，能夠出任主力的也只有李曉旭與巴斯。在內線人員不足的情況下，遼籃必須通過全隊的整體攻防來與福建隊抗衡。

上賽季的季後賽首輪，遼寧男籃同樣是在韓德君因傷缺陣的情況下，以3比0橫掃福建隊，當時依靠的是「以快制高」戰術。三場比賽，遼寧男籃充分發揮了自己外線速度優勢，儘可能地提高攻防速度，保證場上人員輪轉，消耗板凳深度不夠的福建隊隊員體能。這套戰術很奏效，比賽的上半場，甚至前三節，福建隊總能和遼籃比分緊咬。不過到了下半場，或者第四節，包括王哲林在內的福建隊隊員體能下降，遼籃總能拉開比分，最終鎖定勝局。相信今晚再次面對福建隊，遼籃依然會使用這套戰術。

不過與上賽季季後賽不同的是，遼籃本賽季的陣容發生很大變化，哈德森換成了史蒂芬森，狀態不佳的「師弟」能否在快速攻防戰術中起到核心作用讓人心裡沒底。此外，遼籃的鐘誠和衛猛兩位經驗豐富的內線老將離隊，削弱了球隊的板凳深度，這些因素都導致遼籃目前的陣容深度不如上賽季季後賽，想要再次憑藉速度優勢力克福建隊絕非易事。

福建隊上場力克吉林，球隊新引進的小外援杰特居功至偉，這位在CBA多支球隊效力過的老熟人得到了22分，並送出多達15次助攻，特別是15次助攻，徹底盤活了福建隊。此役，遼籃一定要想辦法限制住杰特和隊友的聯繫，然後儘可能地加快比賽速度衝擊對手，方能在逆境中獲得勝利。
