國乒大滿貫才是名副其實內戰王！第21次擊敗許昕，決賽衝擊神記錄

天天要聞
2019-12-16

文/梅長蘇

北京時間12月15日，國際乒聯巡迴賽總決賽在鄭州繼續進行，在第三個比賽日爭奪中，許昕/劉詩雯逆轉擊敗日本組合伊藤美誠/水谷凖，為中國隊拿到本次總決賽首枚金牌，隨後在男女單打半決賽爭奪中，陳夢在被伊藤美誠驚天逆轉第一局之後連贏四局以4：1戰勝對手，為國乒獲得女單冠軍掃除最後障礙，她將在決賽中迎戰小將王曼昱，王曼昱在半決賽中以4：0橫掃隊友王藝迪，而在男單半決賽中，近期狀態神勇的樊振東4：2戰勝隊友林高遠結束對對手的兩連敗，而在另外一場比賽中，兩位國乒老將給球迷奉獻一場精彩絕倫的對決，最終馬龍4：2戰勝許昕第七次晉級該項賽事男單決賽。

馬龍和許昕的比賽是兩個人在國際賽場上第30次交手，本場比賽之前許昕只獲得了9次勝利，即便如此，許昕是國乒選手中戰勝過馬龍次數最多的球員，可見馬龍的統治力有多強，這也側面證明了馬龍其實才是名副其實的「內戰之王」，當然馬龍外戰也是不手軟。本場比賽首局馬龍開局4：2領先，穩紮穩打之後馬龍逐漸把比分拉開，最終他以11：7先下一城先聲奪人；第二局馬龍又是7：4領先，許昕在7：10落後時連得三分扳平比分，兩人戰至14：14平時馬龍連得兩分以16：14再贏一局；第三局比賽馬龍開局6：3領先，此後雙方比分交替上升，許昕雖然打的很頑強，但並沒有把比分迫近，馬龍以11：7拿下第三局大比分3：0領先。

大比分落後的許昕放手一搏，他甚至把護腕都給摘了，可見許昕對本場比賽的勝利同樣充滿了渴望，第四局比賽許昕開局強勢領先，最後他以11：4拿下第四局追回一局；第五局比賽許昕又是開局大比分領先，隨後他逐漸把比分拉開至8：3，越打越順的許昕最終以11：4相同比分再贏一局大比分追至2：3，被連追兩局的馬龍在第六局比賽恢復「正常」，開局馬龍3：0領先，雖然許昕隨後連續追分，不過馬龍最終還是以11：6拿下第六局總比分4：2戰勝許昕晉級男單決賽，這也是馬龍在國際賽場上第21次戰勝許昕。

馬龍在決賽中的對手是樊振東，兩個人在國際賽場上的交手次數不少，馬龍取得了壓倒性的優勢，本場比賽是馬龍和樊振東第三次在巡迴賽總決賽冠亞軍爭奪戰中交手，2015年、2016年總決賽上，馬龍均在最後決賽中擊敗了樊振東拿到了冠軍，此番再次交手，樊振東是否會報三、四年前的「一箭之仇」值得關注，樊振東上半年的表現並不好，不過近兩個月樊振東表現非常搶眼，連續拿到兩個白金級公開賽男單冠軍以及世界盃冠軍，相反馬龍因為受到傷病影響，目前還未恢復到巔峰狀態，這對樊振東來說是一個機會，樊振東在2017年拿到總決賽冠軍，而馬龍此前已經5次獲得總決賽冠軍，今年總決賽是馬龍第六次獲得冠軍還是樊振東第二次折桂，讓我們一起期待吧！
