為什麼皇馬最早開啟了金元足球？當年“老佛爺”帶來了菲戈是標誌

2019-12-19 1,142條評論 3,285次閱讀 1660人點贊

北京時間19年12月16日，本賽季歐冠抽籤儀式舉行，皇馬vs曼城成為了歐冠八分之一決賽中的強強對話。由於皇馬是足壇最早的金元足球代表球隊，而曼城是新的「富貴隊」，所以這被譽為是「老財主」與「新財主」的較量。可能很多人不是很明白，為什麼皇馬最早開啟了金元足球？就讓筆者為大家來分析一下原因。金元足球看似是近幾年的事情，其實皇馬高層的眼光很高，早在20多年前，就已經產生了金元足球的雛形。在98年歐冠決賽中，皇馬憑藉米亞托維奇在第66分鐘的進球1-0擊敗尤文圖斯，一個球已經足矣，這是一場非常重要的勝利。而斑馬軍團的主教練里皮在後來又成為了國際米蘭的主教練，也是羅納爾多的教練。

當然以上的這些並不是筆者要說的重點，只是給大家介紹一下皇馬開始有了金元足球的意識背景。皇馬贏得歐冠冠軍固然高興，但他們在聯賽中最終只獲得了第四名，因為聯賽成績糟糕，當時的皇馬主帥海因克斯黯然下課。這在如今看來是不可思議的，因為現在足球的氛圍已經發生了變化，俱樂部更加看重歐冠，因為它比聯賽更有利可圖——很多人沒有發現，這樣的變化就是金元足球的產物。洛倫佐-桑斯於1995年接替門多薩出任皇馬主席，他是一名商業投資人，同時還是皇馬名宿薩爾加多的岳父。桑斯在門多薩時期主要擔任其財務主管，不過他桑斯在任期內並沒有被馬德里民眾所接受，主要原因就是公眾形象不佳，他非常奢侈。

在桑斯執掌皇馬的整個過程中積累了大筆的債務，在菲爾-鮑爾的著作《白色風暴—皇馬百年》一書中曾披露，桑斯欠下了一億美元的債務——這在當時絕對屬於巨大的數字了。99年國際足聯規定，進入歐冠的球隊必須要有合理的財務政策，而皇馬那時候卻陷入經濟危機，所以當時的財務危機讓伯納烏球迷們對歐冠名額憂心忡忡。儘管如此，但他們卻在千禧年捧得歐冠獎盃，在決賽中他們3-0擊敗了也是來自西甲球隊瓦倫西亞，第8次問鼎歐冠冠軍n。不過在奪冠之後，桑斯的主席之路也走到了盡頭，新主席候選人弗洛倫蒂諾是著名的建築大鱷，財力雄厚。雖然「老佛爺」是初涉足球，但他非常精明，非常願意花錢，更重要的是他非常有影響力！

弗洛倫蒂諾當年競選主席成功主要是圍繞兩點，一是抹平俱樂部的債務，二是他承諾會把路易斯-菲戈帶來，當時很多人都認為他說了大話，認為這只是競選的時候為自己拉票的方式。菲戈被認為是世紀之交時期，世界上最好的球員之一，他也是巴薩1998年和1999年兩屆聯賽冠軍的重要成員。不過時任巴薩主教練的范加爾卻對他並不感冒，而且當時有消息稱荷蘭人有時還與主席加斯帕特對著干。正是抓住了巴薩的這一點內幕，弗洛倫蒂諾看到了機會，他以當時創紀錄的5610萬美元將菲戈帶到了伯納烏。所以說「老佛爺」帶來了菲戈，是皇馬走向金元足球的標誌。
