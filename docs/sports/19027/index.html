第四節都用不著“大佬”上場，山東這波七連勝什麼水平？

2019-12-18 580條評論 2,094次閱讀 645人點贊

12月17日，在CBA第18輪比賽中，山東西王男籃在客場以119比103戰勝四川金強男籃，取得七連勝，在積分榜上鞏固了前八的排名。

對於山東來說，這場勝利有特殊的意義。在11月21日進行的第8輪比賽中，山東在主場以97比101不敵四川。鑒於四川是全聯盟戰績最差的球隊之一，對本賽季志在打進季後賽的山東來說，那是一場絕對不能輸的比賽。

那場比賽的失利，把山東推向了風口浪尖，不少人極盡嘲諷之能事。球隊上下也背負了巨大的心理壓力。那段時間，山東邀請了兩名外援試訓，動了裁掉梅斯的心思。

所幸，山東將帥一心，在最困難的時刻沒有倒下，挺過了那段最艱難的日子。此番跟四川對壘，山東已換了一番心境。

攜六連勝之威，山東沒有給四川任何機會，第二節轟下了42分，第三節又進賬36分，帶著21分的優勢進入第四節。可以說，山東在防守端的強悍表現，直接帶動了球隊的進攻，以至整個第四節都成為了垃圾時間。

哈德森在第四節沒有出場，因為分差太大，已不需要他出場。他全場打了30分鐘，三分球9投5中，兩分球8投7中，得到33分8助攻4搶斷。老哈上一次沒在第四節出場，還是第10輪山東客場對陣浙江，那場比賽之後，他賭氣私自離隊。

如今，老哈的心情跟那會兒已完全不同。第四節剛打了3分鐘，鞏曉彬通過翻譯跟老哈交流，極有可能是解釋不讓他繼續打的原因，老哈什麼都沒說，沖著鞏曉彬伸出了一個大拇指。

老哈能夠坐在場邊休息，是因為梅斯的表現讓人放心。這位一度行走在被裁邊緣的老將，本場拿到了29分10籃板，這是他本賽季的單場最高分。上一輪，山東在客場戰勝新疆，梅斯入賬27分12籃。低調、沉穩的梅斯，用一個又一個積極地拼搶，蠶食著備用外援「轉正」的希望。

在賈誠遭到停賽的情況下，陶漢林與梅斯聯手支撐起了球隊的內線。上一場比賽得到17分9籃板的陶漢林，此戰砍下20分12籃板。更重要的是，他送出了4個蓋帽！跟以往賽季相比，陶漢林本賽季在防守端的進步有目共睹。

比賽結束後，鞏曉彬在新聞發布會上表示：「我們本場比賽從防守做起，用防守帶動進攻，針對對方的特點進行了有針對性的布置。全隊的輪轉做的不錯，大家都打得比較耐心。」

儘管四川輸了球，可主教練周金利依然肯定了隊員的拼勁，「大家拼得很不錯，這是七天中打的第四場比賽，體能消耗非常大。我們第二節在防守上出了問題，讓對方的三分球投開了，給自己挖了個坑，其他的時間段，大家都做得不錯。」
