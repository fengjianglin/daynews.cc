JR附體！雄鹿大將5.2秒昏頭葬送希望，字母哥體會到詹皇曾經之痛

2019-12-17 321條評論 2,886次閱讀 329人點贊

北京時間12月17日，NBA常規賽，雄鹿隊主場迎戰獨行俠，最終他們以116-120惜敗對手，18連勝被終結。本戰過後，雄鹿隊的戰績來到24勝4負，一直和湖人齊頭並進的他們，終於落後半個勝場了。而下一場，雄鹿隊將和湖人直接交鋒，無形中氣勢方面就弱了一籌。

但是，也不能怪雄鹿隊，他們本場打得並不差。而且，那種認真、不放棄的精神，一直貫徹到最後一秒，主場觀眾已經很滿意了。尤其是最後還剩5.2秒的時候，雄鹿隊在字母哥的帶領下，已經將分差追近到了只剩3分，一個球就能抹平的坑，卻成了橫亘在雄鹿隊面前的一道天塹，再也無法逾越。

當時，比賽還剩下5.2秒的時候，字母哥再次打進一個高難度的上籃，同時造成了克萊伯的犯規，走上罰球線。如果加罰命中的話，只差兩分，有足夠的時間進行犯規戰術，逆轉希望大增。

結果是，字母哥加罰沒能命中。不過，老天又給了雄鹿隊一次機會，讓斯特林-布朗搶到了進攻籃板。只可惜，斯特林-布朗接下來的操作，有些昏頭，又葬送了這次希望。布朗是怎麼做的呢？他搶到進攻籃板之後，立刻想上籃拿兩分，卻遭到了2米21的波爾津吉斯的封蓋，最終球出了邊線，獨行俠的球權。

雄鹿隊主帥布登霍爾澤臉色難看，對布朗這次操作不是很滿意。首先，時間真的已經不多了，而雄鹿隊還握有1次暫停機會，此時不用什麼時候用？其次，就算不暫停，雄鹿隊只落後三分，也可以尋求外線遠投機會了，而打兩分再犯規博罰球，回頭大概率還是需要扔三分，那時候會更緊迫。斯特林-布朗的處理，值得商榷。

不過，也無可厚非，畢竟在那種緊張刺激的時刻，球員的腦袋是很難完全保持清醒的，很多行為都只是出於本能。布朗的這次操作，讓很多人想到了2017-18賽季總決賽第一場，騎士在客場挑戰勇士的最後4.7秒，JR搶到籃板卻不出手，反而往外邊帶，浪費了取勝良機，把詹姆斯氣得夠嗆。而如今，字母哥也體會到了曾經詹皇之痛。
