悲劇！中國女排勁敵拼盡全力拿到的奧運入場券，朝不保夕

2019-12-10 570條評論 3,931次閱讀 2764人點贊

文/獨具一格

12月9日在瑞士洛桑舉行的世界反興奮劑機構執委會會議，經過投票通過了對俄羅斯全球禁賽4年的決議，俄羅斯從2020年開始4年內被禁止參加夏季奧運會、冬季奧運會和世界大賽，也不得舉辦世界性大賽，俄羅斯奧委會官員不能出席世界大賽。俄羅斯運動員可以在經過嚴格檢測條件後，以中立身份參加國際比賽。

在8月初俄羅斯本土進行的奧運會女排資格賽F組的比賽中，東道主俄羅斯女排曾經以三戰三勝奪得了一張寶貴的東京奧運會入場券，特別是最後一場與韓國女排的較量中，俄羅斯兩局落後，上演大逆轉，連扳三局艱難拿到勝利。而在今天的世界反興奮劑機構一個會議，讓俄羅斯女排的奧運資格化為泡影的可能性極大。

俄羅斯女排如果能夠參加明年東京奧運會，也是獎牌的有力爭奪者，中國、美國、塞爾維亞和義大利處在第一集團，俄羅斯、巴西、日本有爆冷衝擊四強的可能。今年的女排世界盃，在塞爾維亞未派主力陣容的情況下，俄羅斯女排抓住機會亂中取勝，最終站上了領獎台，全隊在科舍列娃回歸國家隊之後，有望整體實力更上一層樓。

國際排聯還未正式公布俄羅斯女排的奧運資格是否無效，由於之前的奧運會歷史上從來沒有出現過這樣的情況，估計國際排聯也要召開會議商討此事。如果國家排聯一旦支持世界反興奮劑機構的決定，俄羅斯女排獲得的資格該如何分配也是國際排聯所要商討的一個問題。

韓國女排是最有可能受益的隊伍，畢竟韓國隊是跟俄羅斯女排直接競爭出線資格失利的球隊，所以俄羅斯女排如果被剝奪了奧運資格，韓國女排理應順延獲得這一資格。韓國女排如果拿到了這一資格，亞洲落選賽依然泰國女排也就一家獨大，黃金一代經過三屆奧運會落選賽，總算也被眷顧一回，躺進奧運會了。
