姜至鵬回應紅牌動作惹爭議，網友表示：你還有點臉嗎？

2019-12-11 971條評論 4,995次閱讀 2365人點贊

東亞杯首場比賽中，中國1-2不敵日本，這個結果看起來感覺差距不是很多，其實上，中國在各方面都處於下風，這場比賽，日本隊在上半場就取得1-0領先，下半場，國足再失1球，中國隊在補時階段由董學升扳回一城。

這次東亞杯不是中國最強隊伍，但是這批球員基本都是各俱樂部主力，甚至一些球員也是國腳級球員，只是在里皮執教下沒能入選，因此在經驗和技術都還是不錯的，反觀日本隊，既不是一隊也不是二隊，本次日本隊只能說是三隊，14名是奧運會適齡球員，同時也有多大10名球員是首次入選日本國家隊。但是這樣一支日本隊面對一支新老結合的中國隊卻是2-1戰勝。不得不令人感到日本足球水平遠遠高於中國。

當然這場比賽最受關注還是中國球員姜至鵬紅牌動作，上半場第30分鐘，姜至鵬在爭搶一粒高空球時，直接飛踹到日本球員頭部，這明顯是一個紅牌動作，但是裁判手下留情，只給一張黃牌，不過此時姜至鵬非但沒有認識到錯誤，反而做出一臉無辜表情，是在讓人無法理解。

對於這張紅牌動作，主帥李鐵在賽後做出回應，自己在賽後找到姜至鵬，姜至鵬解釋是沒有看到對方，只是想把球停下來。李鐵也表示，要與球員溝通，告訴他們如何踢球而不是使用暴力。

李鐵這樣解釋其實讓人很舒服，沒有迴避這個暴力動作，也表明自己立場。不過作為當事人的姜至鵬在回應這個紅牌動作時卻引發爭議。他表示：「那個球其實作為我來說沒有故意去傷害任何人，而且我看到他，在他的位置上不可能夠到這個球，我先夠到那個球，然後他的頭碰到我的腳，並不是我直接踹到他，當然在國家隊的比賽中出現這種情況，以後還是要注意。」

這樣解釋立馬讓網友炸鍋了，「這麼不要臉嗎？我都覺得無言以對了。。球技不好，可以練，這麼這麼不要臉怎麼治？」，「你特么這一腳，別讓人灌了5個球還恥辱！爛泥扶不上牆！」，「怪日本隊自己，哪有人踢球不戴頭盔」
