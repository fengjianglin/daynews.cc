總決賽3連冠後，陳夢透露為何擊敗伊藤，東京奧運衝冠要做好一點

2019-12-15 1,873條評論 4,518次閱讀 110人點贊

一場酣暢淋漓的4比1，陳夢擊敗王曼昱，實現總決賽的三連冠。東京奧運會臨近，陳夢的狀態十分出色，這對她競爭奧運單打參賽名額很關鍵。再次總決賽奪冠後，陳夢在接受採訪時坦言很開心，對於兩次擊敗伊藤美誠，陳夢給出自己的看法。當被問到大賽無冠的這層窗戶紙如何捅破時，陳夢幽默說出自己的答案。

本次總決賽，陳夢的奪冠歷程並不順利，首場比賽她就是在1比3落後的情況下大逆轉，4比3逆轉擊敗隊友錢天一，闖過首輪關。第二輪，陳夢在2比3落後的情況下又逆轉擊敗劉詩雯，同樣是4比3的贏球。半決賽，陳夢面對伊藤美誠，首局9比1落後被逆轉，她沒有被打亂節奏，連贏4局，4比1擊敗伊藤美誠，殺進決賽。決賽對陣王曼昱，陳夢打得更為果斷，又是一個4比1，實現了總決賽女單的三連冠。

談到這次奪冠，陳夢坦言：「我覺得其實首先總決賽三連冠，很開心，尤其是距離奧運會差不多還有半年的時間，臨近的時候，每項比賽都非常重要，這也是為自己在2019畫上一個圓滿的句號，為2020年開了一個好頭。」

陳夢之前表示，2019年不滿意的地方是輸了外戰，談到這個壓力，她表示：「我覺得自己在德國的比賽的時候，還是犯了一些錯誤，尤其是在瑞典的時候，逆轉比賽拿下冠軍，應該是很好的勢頭，但到了德國，比賽中用老眼光看待對手了吧。」

在2019年的比賽中，陳夢兩次與伊藤美誠交手都取得勝利，這次總決賽，陳夢4比1勝出很關鍵。談到這兩場比賽的感受，陳夢坦言：「在對伊藤的每場比賽中，每次都積累很多經驗，技戰術和思想、心態上，每次過程都有不同的感受。從瑞典開始，那會兒大家很久沒交手，互相併補不是很了解，更多是相互試探。這次不再是試探，過掉了試探環節，進入主題。我覺得這次在比賽中，捕捉到對手的思想動態，這一點是非常值得去總結的地方。」

記者提到，陳夢在大賽當中還沒有捅破窗戶紙，比如世錦賽女單決賽，陳夢就輸給了劉詩雯。展望2020年，自己如何去改變，陳夢給出自己的看法：「大家也都說，我有一層窗戶紙沒捅破，在大賽中，我自己也是這麼認為。經歷了世錦賽女單決賽後，我覺得在大賽的冠亞軍決賽，自己有一定感受，接下來不管什麼比賽，我覺得是一拼到底，放手一搏，再輸也就是輸了，如果進入決賽，再有以前的心態，肯定贏不了了。只要未來進入決賽，完全以拼的姿態，無非再是第二嘛，換一種心態，可能突破自己，取得自己想要的那個成績。」

對於陳夢來說，現在她還要競爭奧運會的單打資格，有了總決賽的冠軍，再加上她自己的努力，相信她可以在大賽里，捅破這層窗戶紙。
