狂飆6三分！中國雷阿倫越發穩定，力助山西隊穩居前3

2019-12-08 0條評論 25次閱讀 0人點贊

摘要

北京時間12月8日，昨晚CBA常規賽第14輪繼續進行，山西隊在主場對陣四川隊。比賽開始後山西隊狀態出色，他們半場取得15分領先優勢，雖然後來對方瘋狂追分，但還是以107-94獲勝。

北京時間12月8日，昨晚CBA常規賽第14輪繼續進行，山西隊在主場對陣四川隊。比賽開始後山西隊狀態出色，他們半場取得15分領先優勢，雖然後來對方瘋狂追分，但還是以107-94獲勝。此役山西隊球員原帥發揮出色，他全場比賽出場39分鐘，投籃19中9，其中三分球14中6，得到24分，成為球隊獲勝功臣。

山西男籃新賽季被稱為「最大黑馬」，目前他們在少賽一場的情況下，仍然以9勝3負的戰績位居積分榜第三位。要知道上賽季他們連季後賽門檻都沒摸到，現在竟然能力壓廣廈、遼寧、北京等多支強隊，非常不容易。

比賽開始後原帥依舊首發出場，在首節比賽還剩7分31秒時，他利用隊友葛昭寶的掩護命中本場比賽首記三分球，這球命中後幫助球隊拉開了6分的分差。此後原帥手感出現斷電，先後兩次出手三分不中。

次節原帥手感依舊不佳，接連兩次出手三分球還是未能命中。正當大家覺得他這場比賽會沉淪的時候，原帥就在一次反擊中命中追身三分。這球我們可以看出他的自信心，當一位投手接連投不進的時候是選擇不投還是接著投下去，顯然原帥選擇的是後者。此後他又命中一記中投，幫助球隊拉開14分的分差。半場比賽結束，原帥已經得到8分。

下半場第三節原帥進攻機會不多，但只要有出手三分球的機會還是非常果斷，這節他命中了左側底角的一記三分球。

而到了末節決戰時刻卻成為了原帥的個人「得分秀」，當時山西隊情況危急，已經被四川隊將分差追至5分。此刻原帥站了出來，他首先用一記三分球幫助球隊穩住局勢。隨後中投、空切上籃、三分球，各種得分方式紛紛上演，四川隊本來又一波反撲攻勢徹底被原帥瓦解。

而當福特森在最後階段命中三分球時，又是原帥投中了回應的三分球，此後四川隊再沒反攻勢頭。就在末節原帥單節爆砍13分，最終幫助球隊獲勝。

全場比賽原帥得到了24分，是隊內得分最多的球員，特別是三分球命中率越發穩定。現在的他每場比賽平均能投進3.7個三分球，因此有球迷稱他為「中國雷阿倫」。另外比賽中他展現了一名投手的調整方式，在接連投不進的時候，原帥選擇了依然自信的出手，結果得到了回報。

原帥本賽季場均出場36分鐘，得到17分，其中三分球命中率高達51%。相比上賽季的場均5.6分，三分球命中37%，有了大幅度的提升。他的存在讓這支球隊的進攻更犀利，也讓球隊球員之間更加樂於分享球。

山西隊本賽季迎來了「三雙王」富蘭克林的回歸，另外也簽下了籃板狂人莫蘭德，確實這兩名外援實力很強，他們似乎是這支球隊的主角。但從比賽看來卻不是這樣，因為山西隊的國內球員同樣發揮出色，比如這場比賽原帥、任俊威都是得分在20分以上，這些就是他們能穩居積分榜第三的資本。
