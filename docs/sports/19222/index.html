你大爺還是你大爺！又完成兩大壯舉，C羅仍在用行動打很多人的臉

2019-12-16 515條評論 2,266次閱讀 501人點贊

本月初，克里斯蒂亞諾·羅納爾多錯失了足壇最重要獎項「金球獎」，本賽季受傷病影響的C羅也遲遲找不到最佳狀態，總而言之，在很多人看來C羅已經跌下巔峰走下坡路了。但是，C羅的職業生涯最擅長「打臉」，近期他的狀態開始復甦，連續四場比賽取得進球，在昨天晚上意甲聯賽尤文圖斯對陣烏迪內斯的比賽中，C羅梅開二度幫助球隊取得了勝利。

C羅仍在創造新紀錄

如此一來，C羅本賽季的進球數又超過了10粒。2005-06賽季，當時效力英超曼聯的C羅開始了打進了12粒進球，隨後3個賽季他又分別打進23、42和26球。2009年夏天，C羅加盟皇家馬德里，在西甲聯賽的9年時間一直保持著讓人驚嘆的好狀態。加入皇家馬德里的第一個賽季，C羅就打進了33球。

C羅15個賽季以來的進球數據

隨後，C羅每一個賽季都打入了42+進球，每個賽季都取得了42粒以上的進球，其中進球最多是的2014-15賽季，他為皇家馬德里打進了61粒進球。上賽季，C羅加盟意甲球隊尤文圖斯，整個賽季他為斑馬軍團打入了28球。也就是說，從2005-06賽季開始一直到現在，C羅在長達15個賽季中，每個賽季都能打進10粒以上進球。目前賽季還未過半，C羅極有可能連續14個賽季完成20+進球的壯舉。連續15個賽季打進10粒以上進球，使得C羅成為歐洲足球五大聯賽中唯一一位在同一時期內完成這一壯舉的球員。

C羅慶祝進球

如果按照年度進球數計算，C羅在2019年的進球數也已經達到了35球，如此一來C羅已經達成了連續12年每年進球都超過35球的壯舉，這也是足壇歷史第一次有人做到。C羅一直在用行動證明，你大爺還是你大爺，從沒走下巔峰。
