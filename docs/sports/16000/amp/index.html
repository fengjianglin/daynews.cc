萊切斯特這數據冠絕英超！狐狸城望再續奪冠神跡

天天要聞
2019-12-14

　　北京時間12月14日晚23點，萊切斯特城將在主場迎戰弱旅諾維奇。目前狐狸城在聯賽中豪取8連勝，刷新了隊史135年間的最長連勝紀錄，一路高歌猛進的他們，還能再一次創造奇蹟嗎？

　　目前萊切斯特城積38分排名積分榜第二位，落後領頭羊利物浦8分，領先身後的曼城6分。相比於藍月亮，狐狸城更像是紅軍爭冠道上的攔路虎，他們也具備著這樣的DNA，15-16賽季的萊切斯特奇蹟可並沒有過去多久。

　　球隊本賽季攻防表現俱佳。在進攻端，狐狸城的39個進球僅次於曼城和利物浦，而防守端，僅失10球更是榮膺聯賽防守最強球隊，29個凈勝球同樣冠絕英超，如此的攻守平衡實在令人讚歎。

　　作為狐狸城的進攻核心，瓦爾迪的狀態頗為火熱，一人貢獻了整隻球隊接近一半的進球，目前以16球獨居射手榜榜首，近8場比賽更是場場破門。要知道上一位連續8輪英超破門的球員也是他，而那個賽季，就是他們創造奇蹟的賽季。

　　作為前任的後防核心，馬奎爾的轉會離隊似乎並未影響到球隊，瑟雲聚的出色表現已經讓人忘記了他。本賽季在場均攔截搶斷等關鍵數據上，瑟雲聚均不輸給馬奎爾，他的異軍突起也間接促成了球隊目前失球數最少的局面。

　　而在近期，球隊又宣布續約主帥羅傑斯至2025年。作為前利物浦主帥，這位戰術大師終於再次於英超賽場證明自己，特別是對於年輕球員的使用，麥迪遜、奇爾維爾、蒂勒曼斯、瑟雲聚等新生代在他的手下得到重用，一套全新的核心班底基本打造完成，羅傑斯在其中功不可沒。

　　在10月5日聯賽客場敗給利物浦後，狐狸城在接下來的這兩個月中連戰連捷，各項賽事豪取9連勝的成績，他們的贏盤能力也堪稱恐怖，10場比賽8贏2走，贏盤率高達80%。而本場面對諾維奇，他們近期僅在15-16年的奪冠賽季與其交手兩場，球隊全部取勝且沒有輸盤，對於本場而言，狀態火熱的他們拿下穿盤大勝並不困難。

　　萊切斯特目前專註於聯賽，而利物浦卻面臨多線作戰，8分的差距並不是不可挽回，如若紅軍犯錯，身後的狐狸城相比不會放過這樣的機會。15-16賽季的夢，在利物浦如此統治力的情況下，還會實現嗎？

　　（卡沙）
