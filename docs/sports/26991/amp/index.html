紅軍贏得世俱杯不足喜，隊內傷病才是頭等大事，12月踢10場比賽

天天要聞
2019-12-23

此次征戰世俱杯紅軍多名球員傷缺，包括范戴克缺席世俱杯第一場比賽。再加上本場本場受傷被替換下場的張伯倫，薩拉赫和羅伯遜是因為腳踝的傷病，而洛夫倫和馬蒂普以及法比尼奧也都缺席了比賽，還有維納爾杜姆在周六的英超比賽中肌肉出現了問題，粗略的計算紅軍陣中起碼有8名球員有點小傷病。

球員的傷病很大原因是英超賽事過於密集，特別是在聖誕節附近，再加上世俱杯的兩場比賽。利物浦在12月有10場比賽要踢，因為世俱杯與西漢姆聯的比賽被推遲，算下來利物浦12月3天多一點就要踢一場高強度的比賽，球員身體受傷在所難免。英超賽事的安排對參加歐戰的球隊特別的不公平，利物浦12月踢四種類型的比賽。

利物浦現在傷缺最重要的球員就是兩名中後衛，再加上范戴克也是帶傷作戰。利物浦健康的中衛只有喬-戈麥斯，昨晚世俱杯決賽可謂是這名英格蘭本土中衛的代表作，100次傳球還有90次精準傳球再加上8次的成功爭頂。雖然戈麥斯昨晚發揮神勇但是他本賽季各項賽事只有8場首發，不知道克洛普會不會在英超賽場給他機會。

利物浦本賽季各項賽事只有5場零封，要知道利物浦現在已經踢過30場比賽。雖然後防有范戴克坐鎮，但是他這個賽季不再是上個賽季那麼神奇。再加上身邊的搭檔從馬蒂普換到洛夫倫，再到現在的喬-戈麥斯，甚至隊長亨德森都和他搭檔過中衛。中衛的不穩定導致球隊本賽季只有5場零封，喬-戈麥斯將是克洛普手中最後一張牌。

只要挺過聖誕季賽程，紅軍的傷病員將逐漸歸隊。幸運的是利物浦本賽季所能爭取的冠軍都已拿到手，還有聯賽領先第二名萊斯特城10分。雖然下一輪英超將迎戰前主帥羅傑斯率領的萊斯特城，萊斯特城最近不在狀態，相信利物浦會拿下比賽。如果贏下這場比賽利物浦的領先優勢達到13分，利物浦這個第一座英超冠軍獎盃還能跑嗎？
