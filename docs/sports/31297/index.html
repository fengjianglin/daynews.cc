足壇第一惡人產生！欲毆打生病主帥，還曾衝撞梅西致其被罰下

2019-12-24 559條評論 1,971次閱讀 1008人點贊

加里·梅德爾不會和人交朋友，哪怕是自己球隊的成員。上周末，梅德爾在球場上主演了令人瞠目結舌的一幕：他居然推搡起了自己的主帥米哈伊洛維奇，而後者正與白血病作鬥爭。米哈伊洛維奇在今年早些時候被診斷出患有白血病，上個月他接受了骨髓移植，半個月前第一次回到賽場。

比賽中，梅德爾與萊切隊的希臘球員塔齊特西迪斯發生了爭執，米哈伊洛維奇上去勸解，沒想到梅德爾竟然將怒氣發泄到了主教練頭上，他與後者發生了爭吵，兩人還差點打起來。

最終博洛尼亞3比2擊敗萊切，不過賽後的新聞不是比賽結果本身，而是梅德爾與米哈伊洛維奇的推搡與怒視。更重要的是，米哈伊洛維奇正與白血病做鬥爭，梅德爾卻絲毫不顧及這些，顯得缺乏人性。

值得一提的是，今年美洲杯上，梅德爾就與梅西發生衝突，還導致梅西被紅牌罰下。那場比賽第38分鐘時迪巴拉送出直塞，梅西沒能追上，隨後和梅德爾爆發衝突，主裁判隨後直接出現紅牌將兩人同時罰下，這也是梅西職業生涯的第二張。

從鏡頭中看到，球出底線後，梅德爾往回走的時候，梅西肩膀確實有一個往前頂的動作，但梅德爾情緒更加激動，連續撞擊梅西，將梅西往後頂。

《馬卡報》賽後表示，面對梅德爾的挑釁，梅西什麼都沒有做，他只是攤開雙手，讓梅德爾撞向自己的胸膛。梅西該吃黃牌嗎？可能是的，但是裁判卻給了紅牌，可能就因為他是梅西吧！

梅德爾雖然身高不到1米7，卻是南美最兇猛的後衛之一。場上鬥志十足，十分勇猛。在效力塞維利亞時，梅德爾就曾和梅西發生摩擦，而在2015年的美洲杯上，梅德爾也曾用野蠻手段對付梅西。可以說，梅德爾長期的野蠻糾纏，也是梅西發脾氣的一大原因。

梅德爾可以說已經成為了足壇的一大「惡人」。
