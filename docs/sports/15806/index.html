奧運三金王斯奈爾去世 曾單屆奧運奪800和1500雙金

2019-12-15 980條評論 3,932次閱讀 1336人點贊

北京時間12月14日，據美聯社最新報道，紐西蘭著名的奧運會田徑名將、曾獲得三枚奧運金牌的彼得-斯奈爾，當地時間本周四在德克薩斯州達拉斯的家中去世享年80歲。斯奈爾21歲的時候拿到1960年羅馬奧運會男子800米金牌，1964年東京奧運會他一人獨攬男子800米和1500米兩塊金牌，他是從1920年以來唯一一個在單屆奧運會上一人獨攬這兩個項目金牌的運動員。

斯內爾1938年12月17日出生紐西蘭塔拉納基區的奧普納基。這位身高1米79、體重80公斤的前紐西蘭著名中距離跑運動員，少年時期卻是學打網球的，他19歲時才被當年的著名田徑教練阿瑟-利迪亞德發現並隨即改練田徑。1960年羅馬奧運會，斯內爾以1分46秒48的成績獲得男子800米金牌，當時他才21歲。四年後的東京奧運會，斯內爾以1分45秒1實現男子800米的衛冕，隨後他在男子1500米決賽里以3分38秒1再次奪冠。

在1962年的珀斯英聯邦運動會上，斯內爾還獲得880碼、1英里兩枚金牌。此外他還先後七次打破世界紀錄。1962年1月他曾跑出以3分54秒4創造新的1英里世界紀錄。1964年11月17日，斯內爾又在以奧克蘭3分54秒1的佳績刷新了自己的世界紀錄。1962年他曾以1分44秒3打破男子800米世界紀錄，1964年11月他曾跑出2分16秒6刷新男子1500米世界紀錄。

斯內爾的去世得到了紐西蘭體育名人堂負責人、家庭朋友、紐西蘭體育歷史學家羅恩-帕倫斯基的證實。

帕倫斯基說：「這是一個非常不幸的消息，對紐西蘭來說是一個悲痛的損失。就田徑領域而言，他可能是紐西蘭最偉大的運動員。」斯內爾的朋友兼訓練搭檔、奧運會馬拉松銅牌得主巴里-馬吉說，「再也不會有像他這樣的紐西蘭運動員了。」

1965年，斯內爾結束了自己的職業田徑生涯並移居美國。隨後斯內爾畢業於加州大學戴維斯分校，獲得人體機能理學學士學位，之後他又獲得華盛頓州立大學運動生理學博士學位。

1981年他成為德克薩斯大學西南醫學中心的研究員，後來他成為該大學人類表現中心的主任。2000年斯內爾被評選為20世紀紐西蘭體育冠軍，2001年他被封為爵士並榮膺紐西蘭勳章，2012年他進入國際田徑聯合會的名人堂大廳。

斯內爾的妻子米基在接受紐西蘭記者採訪時透露，儘管上個月他在開車時昏倒，撞上了幾輛停著的車，但她丈夫在周四那天過得很開心。米基說：

「他感覺很好，他去做了一些活動，並計劃做更多，他還想打乒乓球。他躺下來看電視上的新聞的時候我發現他睡著了，我想叫醒他，但他不肯動。醫護人員大約五分鐘後就來了，但沒能叫醒他。」（Red）
