3-2爆冷！1球落後2-1反超，2-2後世界波絕殺！法甲保級隊瘋狂慶祝

天天要聞
2019-12-21

法聯杯8分之1決賽開始打響，其中這輪一場並不起眼的比賽引起了外界關注，那就是亞眠隊對陣雷恩隊，兩隊本賽季的戰績很懸殊，亞眠隊17輪比賽僅僅收穫了4場勝利5場平局，積17分排名第17，也僅僅比降級區的梅斯多1分；而雷恩隊本賽季展現了黑馬本質，17場比賽收穫了9場勝利和3場平局，排名第4，殺進了歐戰區。

然而這場比賽原本外界以為雷恩隊拿下亞眠隊應該沒有多大問題，但萬萬沒有想到的是，雷恩卻2-3爆冷輸給了亞眠隊。除此之外，這場比賽的過程也跌宕起伏，雷恩還是1球領先的情況下，被亞眠隊來連續打進兩球反超，之後雷恩也扳平了比分2-2，隨後比賽第90分鐘左右，亞眠隊再打進一球，上演了逆轉絕殺。

比賽一開始，雙方踢得很膠著，不過實力佔優的雷恩隊率先進入狀態，在發動猛烈的進攻後收到成效，雷恩獲得禁區點球機會，吉茲.納亞馬思操刀主罰命中；然而8分鐘後，亞眠隊丟球後也被敲醒，主場作戰的情況下，踢得很有士氣，科納特助攻祖加打進一球扳平了比分1-1。

亞眠隊扳平比分後，很快又迅速反超了比分，比賽第41分鐘，德雷耶爾助攻約翰瓦勒尼卡打進第2球，從而使比分改寫為2-1，亞眠隊主場反超。隨後兩隊對攻，雷恩在比賽第73分鐘，胡若烏打進一球，又將比分扳為3-3，雙方回到了同一起跑線。

隨著時間的消耗，雙方常規時間即將結束，原本以為兩隊就此2-2結束比賽，但意外的是，在比賽第90分鐘左右，亞眠隊上演了奇蹟， T.莫科恩特前場搶斷後，一記30米左右開外的距離直接怒射破門，世界波，亞眠隊3-2逆轉絕殺。1球落後反超，2-2扳平後又絕殺，最終晉級8強，亞眠隊面對實力以及狀態強於自己不少的對手最終逆轉，全隊瘋狂慶祝，對手全隊則非常失落。
