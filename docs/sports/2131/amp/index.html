保利尼奧談恆大奪冠：一個賽季下來，真正起決定作用的是中國球員

天天要聞
2019-12-09

摘要

2019賽季中超聯賽MVP——廣州恆大的保利尼奧沒有任何爭議，以核心身份幫助廣州恆大拿到隊史第8個中超聯賽冠軍之後，巴西人何塞·保羅·貝塞拉·馬希爾·儒尼奧爾收穫他效力中超期間的第一個賽季MVP。

2019賽季中超聯賽MVP——廣州恆大的保利尼奧

沒有任何爭議，以核心身份幫助廣州恆大拿到隊史第8個中超聯賽冠軍之後，巴西人何塞·保羅·貝塞拉·馬希爾·儒尼奧爾收穫他效力中超期間的第一個賽季MVP。

在這個可能是恆大10年來最困難的賽季里，保利尼奧以單核身份帶領球隊從低谷中爬起，打出一波13連勝，從而完成逆襲，把爭冠主動權牢牢握在手裡。在31歲這年，保利尼奧的個人數據創下職業生涯新高，聯賽出場29次，打進19球還有6個助攻，完完全全的進攻核心數據，以至於大家忘了他原本只是一名攻守平衡的中場。

保利尼奧在職業生涯起步階段非常坎坷，而貫穿他整個職業生涯的赤誠和專註，為他贏得了所有邁上另一個台階的機會。從布拉甘蒂諾到科林蒂安，從熱刺到恆大，再從巴薩回恆大，他的球風已經為世人所熟知：不知疲倦地跑動，強硬地攔截、對抗、護球，轉瞬之間完成攻防轉換，專註於每個時刻和每寸空間，抓住時機完成致命一擊。不華麗。

保利尼奧的職業生涯還遠沒有塵埃落定。他在三個大洲之間不斷折返的軌跡在世界足壇都是罕見的，他的每次折返都在往上走一個台階。保利尼奧是那種離開歐洲主流舞台之後依然沒有放棄提升自我的典型代表。他不是來中超養老的，相反，他依然以最高的標準來工作，從而帶動了隊友，扛起了球隊，提升了中超。近日，保利尼奧接受了南方都市報專訪，談及自己職業生涯的軌跡、選擇和原則。球員踢出來看起來是用腳，其實是用心。

在以下的採訪中，保利尼奧談到了曾在巴塞羅那效力的經歷與感受，以及回歸恆大的前前後後，談到了今年恆大面對困難時的種種際遇——

在中超聯賽，保利尼奧顯示出高人一等的實力。 新華社發

在歐洲踢球，巴薩那年最快樂

南都：去熱刺的機會是你夢寐以求的嗎？英超是世界上最好的聯賽。

保利尼奧：作為一名球員，不論在什麼地方踢球，我都是以自己的最高標準來要求自己。但是我所謂的最高標準，並不是說我一定要在一個國家最好的球隊效力，或者說我的夢想是要去熱刺或者巴塞羅那。我在職業生涯中唯一一次給自己定下明確的目標是代表巴西國家隊出戰，這是我定過的唯一目標。其他事情我都是順其自然讓它發生，就是我們所說的let it be的態度。在科林蒂安的3年我踢得非常開心，當有個去英超踢球機會的時候，我毫不猶豫地抓住了。我在英國度過了非常好的兩年。但是我不強求任何事情。

南都：在巴塞羅那的一年，是你三段歐洲效力經歷里最享受的一次嗎？

保利尼奧：確實在我在歐洲效力的這麼多俱樂部當中，在巴塞羅那這一年是我最快樂的一年。無論是生活上還是競技方面，我過得非常充實。我的家人在巴塞羅那生活得很開心，我的一對雙胞胎兒女在那裡出生。所以一切都在最好的時候發生了。

保利尼奧榮膺中超MVP，實至名歸。 南都記者 黎湛均 攝

在巴薩練就面對困難的信念，帶到廣州

南都：在世界上最好的球隊踢球，進球率很不錯，有沒有覺得自己是球星了？

保利尼奧：我覺得與其說用球星這個詞，我更喜歡的說法是職業生涯的贏家，這是更加確切的說法。我在巴塞羅那接觸到了世界上最好的球員——梅西、蘇亞雷斯、拉基蒂奇、伊涅斯塔、布茨克茨、皮克……這些都是自己位置上世界上最好的球員。

在巴塞羅那這段經歷，我感覺我像去了一所學校上學。一個賽季下來或者在比賽中，非常困難的時候，在別人認為你們無法取得突破的時候，巴塞羅那給我的經歷就是我們可以，我們可以在這個時候想到一個辦法來突破。

然後我把這個信念帶到了廣州，帶到了我身邊的隊友當中。有些人會質疑，在這麼困難的環境下怎麼可能取得好成績，這一點我是不認同的，我覺得巴塞羅那的經歷更多的是給了我一種信念上的加強，我從巴塞羅那見識到了一些我從來沒有見識到的東西，而且這個東西會一直留在我身體里。

南都：從巴薩踢了一年又回到廣州，別人很難理解。除了金錢，還有其他什麼因素？

保利尼奧：做出這個決定一點都不困難，一點都沒有猶豫。我跟我的家人都是百分之一百的堅定想回到廣州。確實，在巴塞羅那的生活，我們全家都過得很開心，但是就像我剛剛跟你所說的，廣州隊對我來說有特殊的感情。這是一個在我最艱難的時刻，張開雙臂迎接我、幫助我走過困難時光的地方。

在職業足壇，沒有人能預知到明天會發生什麼，但是在那一刻，我和我的家人都非常堅定，當我們收到了這份來自中國的合同，當我們收到來自中國的這份嚴謹規劃的時候，我們都非常喜歡。所以我們義無反顧做出了重返中國的決定。當然了，廣州這個城市的環境非常地吸引我，這個城市的球迷對我非常歡迎。

會跟鄭智在更衣室討論很多東西

南都：第二次到恆大，外界感覺你的責任感更強了，雖然鄭智還是隊長，但在今年，很多人認為你是球隊的精神領袖。是什麼讓你在球隊的角色看起來有所轉變？

保利尼奧：我覺得這些責任感，或者所謂的領袖說法，首先是由我在廣州這麼多年來的一舉一動——在場上也好，在場下也好的所作所為累積出來的。鄭智是我們不可撼動的隊長，也是不可撼動的領袖，但是我也希望通過我的經驗，可以在球隊當中承擔更多的責任，幫助年輕球員成長。所以私底下我跟鄭智在更衣室也會一起討論很多東西，也許我們會共同做出一些決定。但是我想說其實這一切都源於我在場上的付出，也源於隊友對我的幫助。

南都：恆大隊友對你評價頗高，你在工作中你遵循什麼原則？

保利尼奧：我一直跟我的朋友講，我永遠不想成為一名巨星，我想成為一名優秀的職業足球運動員，也想成為一名優秀的男人。生活當中或者足球場上，最重要的是要尊重你身邊的人，尊重你的對手，尊重你的家人。在尊重的基礎上去超越任何困難。一個好的男人最重要的是真誠，你要對你的朋友真誠，你要對你的家人真誠。我們會花費很多時間在工作上跟隊友相處，我永遠提醒自己，永遠對你的隊友真誠的同時，要有自己主觀的判斷，你要有自己的個性，敢於表達自己的立場，我覺得這是最重要的。

真正起決定作用的是中國球員

南都：今年上半年會不會尤其感到疲憊？很多時候都幾乎是一個人帶著十個中國球員在踢，而且成績一直不錯。

保利尼奧：我不會用疲憊這個形容詞來形容上半年。很多人理解的，外援在一個球隊當中，尤其是在中國的球隊當中，會起到非常重要的作用，這一點我不否認。在很多的球隊，或者說在很多的時間段，外援可以起到非常重要的作用，但是一個賽季下來真正起到決定性作用的是中國球員，因為他們才是大多數。大多數人才能把事情做好，而不是靠一兩個個體的力量。足球最終是一個整體項目，在這個整體之中起到決定性作用的永遠不是外援，而是中國球員，因為他們才是最大的那部分。

南都：你效力過很多不同的俱樂部，有高有低，有主流有邊緣，有頂級俱樂部也有普通俱樂部，在不同俱樂部的定位和感受分別是什麼樣的？

保利尼奧：作為球員來說，無論是在世界上最大的俱樂部、亞洲最好的俱樂部之一，還是在其他比較小型的俱樂部，感受最深的部分其實跟它的級別沒有多大的關係。這些俱樂部最大的區別是在於硬體設施方面。大俱樂部的條件會很好，硬體會非常的完善；小一點的俱樂部在這方面可能會遇到更大的阻礙，但是其實作為球員上了場之後，這些都不存在。場上就是11人對11人，在場上定勝負，跟你穿什麼隊的球衣，跟隊徽背後的價值沒有任何關係。在場上就是拿出自己的所有，不論是在大俱樂部也好，在小俱樂部也好，胸口隊徽的意義是一樣的。可能更大的落差是存在於我們後勤保障方面、硬體設施方面的一些東西，但球員的努力是沒有區別的。我總以最高的標準要求自己，跟在巴薩還是恆大或者是布拉甘地諾沒有關係。

保利尼奧慶祝進球。 新華社發

不會去考慮巴西國家隊徵召的事

南都：今年的美洲杯巴西隊主帥蒂特沒有招你入隊，但他來天河體育場看恆大比賽了，你們是怎麼溝通的？

保利尼奧：蒂特當時確實是來到了廣州，在天河體育場看了我們的比賽，但是他沒有跟我做任何的交流，我跟蒂特在廣州期間沒有任何的交流。他純粹是來看比賽考察球員，那麼我姑且認為他也有在考察我吧，但當時場上還有塔利斯卡還有奧古斯托，很多不同的巴西球員都在他的考慮範圍之內。蒂特作為巴西國家隊的主教練，他會有他的看法，或許在這個時間段，塔利斯卡和奧古斯托處於他們最好的時機，他會考慮招他們入選。這都是一個主教練範圍內很正常的事情。我作為球員在巴西國家隊度過了非常美妙的時光，我也相信無論蒂特做出什麼樣的決定，他都有對巴西國家隊高度負責的態度。所以還是順其自然。

南都：實際上是不是已經退出巴西國家隊了？

保利尼奧：我不會去考慮國家隊徵召的這個問題，因為我首先是廣州恆大的球員，我的目標就是如何幫助廣州隊取得好成績，其他的事情不在我的考慮範圍之內。
