西甲積分榜，巴薩皇馬並駕齊驅，最近兩場比賽，將分出高低

2019-12-14 1,910條評論 4,651次閱讀 3191人點贊

西班牙甲級聯賽16輪之後（巴薩和皇馬少賽一場，補賽將在下周四進行），目前，巴薩和皇馬在少賽一場的情況下，同積34分，領跑積分榜。

北京時間，今天晚上23：00點巴塞羅那將在客場挑戰皇家社會隊。16日凌晨四點，皇家馬德里將客場挑戰瓦倫西亞。皇家社會和瓦倫西亞分別積27分和26分，排名聯賽第4和第8。都有爭奪參加歐冠資格。這兩隊有一定實力，也很難阻止兩隊取勝。皇家馬德里前段時間有點低迷，通過齊達內的調教，漸漸進入佳境；巴塞羅那隨著梅西的回歸狀態正佳。這場比賽，也是兩隊德比戰前的最後一場比賽。兩隊真正的決戰，是19日凌晨三點皇家馬德里客場挑戰巴塞羅那的比賽。

從射手榜和助攻榜看，巴塞羅那的得分點更多。梅西和蘇亞雷斯分居第一和第四。皇家馬德里更多依靠本澤馬得分。從助攻榜上看，皇家馬德里有3人進入前十，巴塞羅那只有梅西進入前十。兩隊進攻核心梅西和本澤馬都進了射手榜和助攻榜前十，數據相當。除了兩隊成績的競爭，還是兩大核心球員的直接較量。其精彩程度值得期待。個人比較看好巴塞羅那主場取勝。
