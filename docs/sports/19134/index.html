抽中那不勒斯，梅西再迎刷新紀錄良機！一數據卻對巴薩並不有利

2019-12-17 372條評論 3,780次閱讀 368人點贊

北京時間12月16日晚上，2019-20賽季歐冠聯賽八分之一決賽分組抽籤揭曉答案。在八分之一決賽最引人關注的無疑是皇馬和曼城之間的較量，齊達內和瓜迪奧拉兩位中生代優秀教練之間將會展開一場決鬥。另一支西甲豪門巴塞羅那隊抽到了一個並不算強的對手，他們將會先客後主面對那不勒斯隊。

對於巴塞羅那隊來說，他們的這個簽表算得上是一個上籤。那不勒斯隊剛剛經歷了換帥風波，他們面對的是一個沒有歐冠經驗的加圖索，這是巴塞羅那隊的一個利好消息。目前在意甲聯賽那不勒斯僅僅拿到了21分，他們排名聯賽第8的位置。本賽季的那不勒斯實力相比前幾個賽季有很大下滑，在淘汰賽開始之前球隊必須調整到最佳狀態。

在此前巴塞羅那隊從未同那不勒斯隊有過正式比賽的交手，這對於梅西來說將會是再一次刷新紀錄的紀錄。在此前梅西曾在對陣34支不同歐冠球隊身上取得進球，如果能夠在面對那不勒斯的比賽再次取得進球的話，梅西將會進一步拉大自己在這項紀錄中同C羅和勞爾之間的差距，C羅和勞爾在歐冠都在對陣33支不同球隊身上取得過進球。

雖然本賽季那不勒斯隊內部有些混亂，但是這支球隊還是不容小覷的。在球隊鋒線上有另一個梅球王——梅爾滕斯，本賽季梅爾滕斯在歐冠小組賽打入了5粒進球。球隊另一員大將是米利克的表現同樣讓人印象深刻，在後防線上那不勒斯擁有馬諾拉斯，在2017-18賽季歐冠羅馬逆轉巴薩的比賽中，正是馬諾拉斯打入了絕殺進球。

巴塞羅那隊在此前曾經在歐冠淘汰賽遇到過7次義大利球隊，在這7次遭遇義大利的球隊當中，巴塞羅那隊被淘汰了4次。從這項數據情況來看，巴塞羅那隊面對意甲球隊很難佔到優勢。在最近兩次面對意甲球隊的比賽當中，巴塞羅那隊都輸掉了比賽，這兩次分別是面對尤文圖斯和羅馬隊。

雙方首回合較量將會在2020年2月25日來進行，比賽將會在那不勒斯主場聖保羅球場來進行。次回合比賽將會在3月18日來進行，屆時巴塞羅那將會移師主場面對那不勒斯的挑戰。對於巴塞羅那隊來說，先客後主的賽程是對他們有利的。
