朱婷向郎平抱屈:我這麼努力球隊還是輸了 恩師:耐心等待隊友成長

2019-12-11 2,885條評論 5,001次閱讀 3308人點贊

12月11日消息，1年前女排世錦賽，中國女排獲得銅牌，回到國內接受央視採訪時受到了主持人張斌的提問「質疑」，甚至被不少媒體調侃為「批鬥大會」。而在今年女排世界盃奪冠之後，張斌再次採訪了主教練郎平，這次還忍不住自黑了一把。

​當時，朱婷和袁心玥、顏妮、張常寧一起接受採訪，然而張斌的問題咄咄逼人，讓女排姑娘們難以回答，現場的氣氛頗為尷尬。

比如問隊長朱婷，「為何本屆世錦賽上，我看到你不高興不興奮，沒有里約奧運會時顯得高興。」朱婷也是表示，「里約奧運會的時候自己只是一個小隊員，在場上把自己的事情做好就可以了，本屆世錦賽自己要做的事情比較多。而且自己是一個比較悶的人，不太擅長與人溝通，還有很多不足的地方。」

而張斌的這次採訪也被很多媒體，網友認為是對女排姑娘們的「批鬥大會」。

​今年再次談到朱婷，張斌一上來就表示，「其實當時我採訪朱婷的時候，所有喜歡朱婷的球迷都對我扔過來一塊板磚，就說我們的隊長你太苛責她了，你希望隊長進步，但隊長需要時間。當世錦賽打完，您需要跟朱婷坐下來深入談嗎？還是把交流放在每一天當中？」

​郎導回答，「都有。我們中國女排有個傳統，就是每一屆大賽下來都會做一個全隊總結。每個人按照位置和這次比賽表現來總結，完後我們隊友、教練都會給她一些提示。我們就更明確今後的工作方向。」

​張斌又表示，朱婷內心應該很焦灼，肯定找郎平訴苦，這次女排世界盃朱婷成了另一個隊長，郎導給了什麼建議。

郎導說，「我們其實經常給她建議，但還得她自己去消化。我們都做過隊長，我們也覺得有時候和新隊員在一起，有時候在某些關鍵時候，關鍵時刻對比賽局勢的把控、判斷，心理準備都有差距。你要以自己的標準同等要求隊友，是不太可能的。

你就要耐心等待，或者去鼓勵隊友，傳授你的經驗，傳幫帶，我們在這方面給了她一些建議，希望她耐心等待隊友們的成長。也希望她不要有這麼大的壓力，要更耐心一些。朱婷是一個很聰明悟性很高的球員，她也會不斷消化這些事情。要給她時間，我們也會總結，若干場比賽，或者比賽完了她的一些表現。要求她還是比較高的。我覺得每一任中國女排隊長，不論你技術狀態如何，你都要承擔在場上的核心責任，來把大家團結在一起。並不是說你扣幾個球的問題。包括她有時候也會很委屈。」

​張斌追問朱婷委屈什麼，郎導說，「可能覺得，你看我都這麼努力了，今天我們還是輸了，教練還在說我們核心隊員不夠堅強，還不夠穩固，她可能會有些委屈。我們那會也是這樣的，就覺得我已經盡了這麼大努力了，教練還老說我們帶的不夠，那我們怎麼帶啊。」

​張斌表示隊友已經開始接受朱婷作為隊長，何時開始擁抱隊長，郎導表示：「一個時間吧，一個就是通過大賽的磨合，包括我們平時有很多交流，比如技術會議交流，二傳組和主攻組，副攻組，都會很多交流，我們也希望大家就一個目標，怎樣把問題解決好，把比賽打好，我們每個人需要做什麼。

需要我們的隊員去理解隊長，隊長要耐心等待你的隊友，大家一定要往一起湊。不要說你看這場球打不好都算我的，我說我們輸球了都是主教練負責，沒有把比賽準備工作做好，技術練好，或者臨場指揮問題。你們不要有壓力和內疚，每個人都往集體靠，我能為集體做什麼，這是最重要的。」
