利物浦坐收漁利領跑榜首，曼城3比1力克狐狸城，瓜帥前途未卜

天天要聞
2019-12-22

北京時間12月22日凌晨，在本輪英超的一場焦點大戰中，曼城3比1擊敗了萊斯特城，在積分榜上依然落後萊斯特城1分，位居第三。雖然曼城球迷很高興，但是利物浦球迷應該會更高興，這意味著他們和第二名的差距可以越拉越大了，提前奪冠的概率將大大增加。

在上個賽季拿到英超冠軍後，曼城在夏季依舊沒有放棄補強，可能歐冠才是他們的最終目標。本賽季曼城在聯賽方面踢得比上個賽季差了很多，也可以說是利物浦本賽季踢得實在太好了，總之就是兩個隊目前的積分差距很大，可能曼城球員還寄希望於利物浦會失誤，所以也沒有全力去踢每一場比賽。這是瓜迪奧拉執教的第四個賽季了，他對球隊氣質的提升已經初見成效，球隊給人的感覺已經從之前暴發戶變成了現在的奪冠熱門。

本場與萊斯特城的比賽更像是一個給瓜迪奧拉考察對手球員的機會，畢竟馬上冬窗就要來了，擁有大量轉會資金的曼城肯定會有所行動的，而萊斯特城又充滿許多天賦滿滿的球員，應該會很符合瓜迪奧拉的口味。曼城的中後衛經過這麼多年的改造依舊不能完全讓人放心，於是瓜迪奧拉把目光又放到了萊斯特城年輕中衛瑟因居上。但是本場比賽瑟因居的表現並不亮眼，還被德布勞內輕鬆突破並送上了一個精彩的助攻。德布勞內真的是已經不知道該怎麼吹他了，現役的第一進攻中場，他的傳球給人的感覺就是穿透力特彆強，總能輕鬆地打破對手的後防線。

曼城之前從萊斯特城買的馬赫雷斯相比上賽季來說也表現得更好了，這應該是他已經完美融入了球隊的表現，每當有重量級比賽的時候，總能看到他有好的發揮。

曼城最近的歐冠狀態都是碰到弱隊就輕鬆虐菜，一碰到強敵就被淘汰，可能是虐菜實在是太敗人品了。本賽季曼城的簽運依舊不怎麼好，在十六強就碰到了皇馬。因為之前執教過巴薩，所以瓜迪奧拉應該對皇馬還是有所了解的。

在本賽季聯賽奪冠希望不大的情況下，歐冠成為了瓜迪奧拉最後一塊遮羞布。這也是曼城一直想要突破的瓶頸，看著隔壁利物浦已經在渣叔的帶領下拿到歐冠，不知道瓜迪奧拉會不會著急。而關於他自己的去留可能也會在賽季結束畫上一個大大的問號。你覺得看好瓜迪奧拉繼續留在曼城嗎？

（羅掌柜）
