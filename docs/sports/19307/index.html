哈德森39+8梅斯27分米卡空砍35+12 山東客場擊敗新疆迎六連勝

2019-12-16 1,081條評論 2,335次閱讀 1055人點贊

北京時間12月15日20點，2019/2020賽季CBA第17輪繼續展開角逐。新疆隊主場107-116負于山東隊，山東隊迎來六連勝。

技術統計

新疆隊：米卡35分12板2助2斷、阿不都沙拉木20分4板2助2斷、周琦18分10板2帽、曾令旭15分6板12助、西熱力江7分5助、可蘭白克7分

山東隊：哈德森39分8板7助4斷、陶漢林17分9板2帽、李敬宇11分4板 、梅斯27分12板3助3斷2帽

比賽回顧

比賽開始前，周琦領取了月度國內球員MVP獎盃。第一節，阿不都沙拉木追身三分命中，曾令旭突破上籃得分，新疆隊開局7-2領先。賈誠兩罰命中，陶漢林上籃得分，米卡轉身扣籃得分，新疆隊繼續保持領先。周琦追身三分命中，隨後自己補籃命中，新疆隊將比分拉大到18-12。哈德森連續命中兩記三分，山東隊逐漸迫近比分。可蘭白克左側底角控衛是三分命中，曾令旭上籃得分，新疆隊27-20領先。隨著米卡2罰1中，新疆隊28-21領先結束第一節。

第二節，周琦左側出生三分命中，阿不都沙拉木接球上籃得分，新疆隊35-28領先山東隊。哈德森命中三分後，隨後利用罰籃三罰全中，梅斯上籃得手，山東隊追至36-37。米卡上籃打成2+1，李敬宇左側外線空位三分入網，賈誠中投落袋，山東隊46-45反超新疆隊。哈德森突破上籃得手，王汝恆正面出生三分命中，山東隊53-49拉大比分。米卡飛身上籃得分後，哈德森再次投進追身三分，山東隊56-52領先結束上半場。

第三節，阿不都沙拉木左側底角三分命中，西熱力江再次轟入三分，米卡轉身上籃打進新疆隊64-65僅落後1分。哈德森壓哨三分命中，梅斯突破上籃得手，山東隊70-66領先新疆隊。梅斯拋投打成2+1，山東隊75-68將比分拉開。哈德森拋投得分，梅斯暴扣得手，山東隊79-70迫使新疆隊喊停。暫停結束，唐才育2罰1中後，梅斯放空籃得分，哈德森轟入三分，山東隊打出10-3攻擊波將比分變成89-73，米卡2罰1中，阿不都沙拉木突破上籃得分，新疆隊77-89落後山東隊結束第三節。

第四節，陶漢林空位跳投命中，梅斯轉身挑籃命中，山東隊95-81領先。曾令旭投入半截籃，阿不都沙拉木罰籃兩罰全中，新疆隊85-98緊追比分。張慶鵬左側投入三分後，新疆隊內外開花，可蘭白克追身三分命中，米卡拋投打2+1，新疆隊打出一個9-3的攻擊波將比分追成94-104。米卡兩罰全中後，哈德森空位三分命中，山東隊107-96繼續領先新疆隊。周琦2罰1中後，王汝恆反擊一條龍命中，李敬宇右側外線投入空位三分，山東隊114-99領先。周琦上籃打成2+1，哈德森罰籃兩罰全中，米卡轉身打板上籃命中，山東隊116-104領先。終場前12.4秒，阿不都沙拉木兩次罰籃命中，可蘭白克2罰1中，最終，新疆隊107-116負于山東隊。

首發陣容

新疆隊：曾令旭、西熱力江、米卡、阿不都沙拉木、周琦

山東隊：王汝恆、哈德森、李敬宇、賈誠、陶漢林
