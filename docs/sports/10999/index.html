17歲華裔中鋒身高2米21！胳膊巨長世界名校內線，球迷：抓緊歸化

2019-12-12 282條評論 2,616次閱讀 676人點贊

2019-2020賽季CBA聯賽正在進行中，廣東14勝1負高居聯賽榜首的位置，新疆和北京隊分列二、三位。從第4到第14，競爭比較激烈，只有3個勝場的差距，尤其是處於季後賽邊緣的球隊。

現在大部分球隊還是依靠外援在打，得分榜前十位全都是外援，現在場均得分排在第一的是吉林隊的多米尼克-瓊斯，場均上場40分鐘可以得到37.2分。但是從下賽季開始CBA外援政策將調整，外援的使用將減少到4節4人次，這能有效提高國內球員的上場時間，增加整體國內球員的實力。

在今年的男籃世界盃中，只有易建聯發揮穩定，其他球員表現還是不算理想。阿聯一年一年還是會老去，中國男籃的未來還得是年輕人的。

在亞洲籃壇來說，中國男籃的內線是絕對的優勢，像姚明、王治郅、巴特爾、易建聯等等，現在又一位身高超過2米20的球員展現在球迷面前。他是一位華裔中鋒，根據歐洲籃球及NBA選秀專家王健消息，他的中文名叫周志豪，母親是第一代加拿大華裔，外公是廣州人。

周志豪身高2米21，胳膊非常長，下肢力量不錯，現在才17歲，未來還有提升的空間。周志豪在普渡大學打球，普渡大學是世界百強大學。普渡大學曾經還出了兩位NBA狀元秀，1980NBA狀元喬-巴里-卡羅爾，1994NBA狀元格倫-羅賓遜。

周志豪表示願意以後為中國男籃打球，有很多球迷呼籲：趁著年輕，抓緊歸化吧。
