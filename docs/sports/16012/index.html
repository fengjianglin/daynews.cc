中國女籃今年表現如何？東京征途好事多磨，這一點最值得球迷欣慰

2019-12-14 764條評論 4,143次閱讀 614人點贊

11月17日，在紐西蘭奧克蘭進行的東京奧運會女籃資格賽亞洲區預選賽A組第三輪比賽中，中國隊以127比49大勝菲律賓隊，將戰績鎖定為2勝1負。由於同組另一場比賽韓國隊負於紐西蘭隊，在三隊戰績同為2勝1負的情況下，中國隊憑藉三隊之間比賽凈勝分之優而排名第一，成功晉級明年2月6日至9日的東京奧運會女籃資格賽。在經歷了千頭萬緒的2019年後，中國女籃仍行進在通往東京的大路上。

11月17日，中國隊球員慶祝勝利。新華社記者 郭磊 攝

如果要用關鍵詞形容2018年的中國女籃，那麼「驚喜」和「霸氣」是絕少不了的。平均年齡只有23.4歲的中國女籃，去年在西班牙女籃世界盃上打到了第六名，這是過去24年里，中國女籃在這項世界大賽上的最好成績。此前不久，她們還在雅加達亞運會上奪回在2013年亞錦賽上失去的亞洲冠軍。眼下這支中國女籃中的骨幹力量，在2018至2019年WCBA聯賽的總決賽中，又將聯賽史上從未奪得過總冠軍的廣東女隊推上王座。這一切讓人感到鼓舞，讓人對中國女籃的2019年充滿期待和信心。

但如此耀眼的2018年，也為中國女籃的2019年平添了不確定性。當中國女籃2019年年初重新集結時，由於在世界盃舞台上的出色表現，韓旭、邵婷、楊力維和李夢4人在爭取加盟WNBA的機會，廣東隊主力黃思靜因為在聯賽中的傷病需要康復治療，這使中國女籃失去了一套主力陣容。加之要備戰今年10月的世界軍人運動會女籃比賽，中國女籃隊中的孫夢然等人也無法參加9月舉行的亞洲杯，最多只趕上了11月舉行的東京奧運會女籃資格賽亞洲區預選賽，因此中國女籃主教練許利民早就預感：「今年會有摔跟頭的準備！」

「跟頭」果然真來了。在9月亞洲杯上，補充了多名首次代表中國隊參加國際大賽的隊員後，中國女籃雖然擊敗了世界盃亞軍澳大利亞隊，卻在決賽中輸給了老冤家日本隊，遺憾地沒能將亞洲杯奪回來。在11月14日東京奧運會女籃資格賽亞洲區預選賽A組首輪比賽中，剛剛聚齊不久的中國女籃，又以1分之差憾負給老對手韓國隊，直到第二輪大比分擊敗紐西蘭隊後，才重新將出線主動權掌握在自己手中。

當回顧中國女籃的2019年，人們很容易會想起韓旭加盟了WNBA球隊；李月汝、黃思靜等人率領廣東隊首奪WCBA總冠軍；中國女籃登上軍運會女籃比賽冠軍領獎台；中國女籃3×3籃球隊歷史性地奪得世界盃冠軍，甚至還會想到在亞洲杯上，她們擊敗了勁敵澳大利亞隊；當然人們也會記得輸日本隊、負韓國隊。但在這一個個記憶片段的背後，在它們之間存在的「邏輯」卻很容易被人忽略：對於一支球隊而言，最重要的是人。因為成績、輝煌是人創造的，形勢、對策是人分析的，拋開一切龐雜的東西，中國女籃2019年真正的目標是什麼？最終是否達到了？這才是衡量中國女籃今年表現最重要的標尺。

毫無疑問，是否取得參加明年東京奧運會就是這個標尺。正是因為認清了形勢，並順勢而為，中國女籃的2019年才能在好事多磨的情況下終得圓滿。明年2月6日至9日，她們贏得了在家門口佛山進行東京奧運會女籃資格賽的寶貴機會。天時、地利、人和，在小組前三名均可參加奧運會的情況下，中國女籃與東京奧運會已近在咫尺，這是值得喜愛中國女籃的球迷們最欣慰的一點！
