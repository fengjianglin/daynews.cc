梅老六輪休與國米歐冠之戰！多特或被悶殺在小組賽？

2019-12-10 331條評論 1,743次閱讀 211人點贊

梅西在拿到金球獎隨後的一場比賽中上演了帽子戲法，兩粒世界波進球加上一粒撞牆配合讓世人看到這位「老妖精」的可怕！西甲聯賽剛剛結束，巴薩又將在北京時間12月11日凌晨對戰意甲目前的老大國際米蘭！而據悉巴薩主帥巴爾韋德新聞發布會已經確定了梅西將會缺席本次比賽的大名單。

國米在歐冠F組的出線狀況可以說是生死一線，多特與國米同積7分，但是國米以凈勝球的優勢排在多特之上！如果說國米能夠順利的戰勝巴薩那麼主動權就在自己的手上了。如果說國米不能夠戰勝巴薩，就要看多特的發揮了。而巴爾韋德此舉安排倒也是合理，巴薩已經是鎖定了歐冠淘汰賽的名額，對於國米來說卻會拚死一搏。一、巴薩完全可以以逸待勞。二、巴薩可以藉此機會給球隊內的年輕球員更多鍛煉。三、巴薩如果因為這場不是特別關鍵的比賽導致球員受傷，將會大大影響球隊的穩定性。

巴薩本次比賽將會比較放鬆的進行，而國米與多特一定要力爭勝利並且要多進球，在這樣的時刻每一粒進球都對球隊的晉級有著巨大的作用。那麼你覺的晚上多特和國米哪個會出局呢？歡迎各位球迷朋友們下方留言討論！
