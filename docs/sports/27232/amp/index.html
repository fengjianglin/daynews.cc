3次豐田杯+世俱杯決賽全敗，利物浦本次勝算有多少？

天天要聞
2019-12-21

38年前，12吋電視機，上海無線電十八廠製造。

電視上看到的是第二屆豐田杯，弗拉門戈在上半場就取得3球的領先。

他們的對手是一支叫利物浦的英格蘭球隊。

那時「白貝利」濟科的名字，比達格利什、約翰斯頓更早被中國球迷所知。

白貝利濟科

達格利什

這是弗拉門戈隊史上僅奪得一次豐田杯，而利物浦，至今為零。

38年後，這是利物浦隊史上第二次參加世俱杯決賽，第一次是在2005年，但當時對陣巴西球隊聖保羅時，三個進球被吹，最終0比1告負。

如果將賽事擴大至世俱杯前身豐田杯，那麼這將是利物浦第4次參加決賽，其中2次豐田杯全部告負，1981年輸給弗拉門戈之後，他們還在1984年豐田杯上0比1負於阿根廷球隊獨立隊。換言之，利物浦3次參加世俱杯（豐田杯）全部告負，周中與弗拉門戈的比賽，利物浦仍在力爭隊史上第一個世俱杯冠軍。

利物浦球迷該慶幸的是，38年來，歐洲與南美足壇實力對比發生了巨變，歐洲俱樂部整體實力已超越南美俱樂部。

自2000年以來，南美冠軍僅拿過4次冠軍，其餘均為歐洲冠軍所得，2013年以來歐洲冠軍已連續6屆捧杯。

利物浦破零的時候該到了！
