巴特爾18歲兒子長相帥氣，身高2米想打職業賽，父親已開始培養他

2019-12-21 448條評論 4,160次閱讀 3125人點贊

籃球作為一項需要天分和後天努力的體育項目，「子承父業」非常常見，畢竟擁有出色的運動天賦會是打籃球的一大優勢，所以縱觀世界籃壇，詹姆斯的兒子布朗尼已經開始嶄露頭角，而科比的女兒也具有驚人的籃球天賦。作為中國籃球界的一大巨星，巴特爾不僅擁有頂尖的籃球天賦，在身高上也很突出，被稱為中國的「移動長城」，作為他的兒子，巴恩澤自從出生以後，就得到了很多外界媒體的關注。如今他已經18歲了，身高超過兩米，未來想打職業籃球比賽，巴特爾支持兒子的夢想，也會用自己的經驗來培養他！

職業生涯

出生於內蒙古的巴特爾身體上強壯，擁有大草原男人的體魄，體型高大的他很早就開始打籃球，擁有2米11的身高和超強的體魄，巴特爾在籃球隊中就像一堵牆一樣，很少受傷。無論進攻還是放防守，巴特爾都極具天賦，被認為是中國男籃的希望，由於球技迅速增長巴特爾很快就被國家隊看中，打了幾場比較重要的比賽。

眾所周知，美國的NBA聯盟才是世界籃球的最高殿堂，作為亞洲球員的巴特爾就受到了NBA聯盟的關注，他的打球風格頗有大將風範，完全符合NBA聯盟的要求，所以2001-2002賽季，巴特爾就來到了NBA馬刺隊。在NBA打球的巴特爾得到了應有的關注，也跟隨馬刺隊獲得了人生中的第一個NBA總冠軍，也是亞洲球員第一位獲得NBA冠軍的人。

子承父業，未來可期

不得不說，巴特爾的NBA之路比較順利，回到祖國之後，巴特爾在CBA打比賽遊刃有餘，也成為了最佳球員，直至退役之前，巴特爾一直都是球隊中的主力球員。2015年，巴特爾退役，如今他的兒子已經18歲了，身高超過兩米，擁有帥氣的長相和出色的籃球天賦，畢竟他的父親可是中國籃球界的領軍人物。

巴恩澤從小開始就同樣熱愛籃球，如今在球隊進行了專業的訓練，作為巨星之子，巴恩澤想要在籃球界打出一片天下要更容易一些。首先他有天賦，也有父親指導，還有不錯的關注度，相信在他長期的訓練之下，肯定可以進入職業球隊打比賽，期待他成為下一個「巴特爾」。
