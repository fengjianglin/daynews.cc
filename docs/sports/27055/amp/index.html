央視直播英超倫敦德比，熱刺大戰切爾西，穆里尼奧對陣老東家

天天要聞
2019-12-22

穆里尼奧上個月接掌熱刺的教鞭，幫助球隊重新建立起了信心，本輪倫敦德比大戰，穆帥要領軍在主場迎戰他的老東家切爾西，將首次與蘭帕德在英超上演師徒過招。論起臨場指揮的經驗，以及調兵遣將的能力，穆里尼奧都明顯要高於蘭帕德。以熱刺在穆帥執教後的狀態，相信可以憑藉主場之利力壓切爾西闖進前四。【開賽時間：北京時間12月23日凌晨0點30分，CCTV-5直播，劉嘉遠解說。】

去年被曼聯辭退的穆里尼奧，上個月再戰英超，換帥後的熱刺隨即也見到起色，在穆里尼奧領軍後打了7場。除了聯賽不敵曼聯，在歐冠輪換出擊輸給拜仁之外，其餘各項賽事5場都是保持全勝。在聯賽積榜上積26分攀升到了第五位，僅以3分落後於第四位的切爾西，這一仗如果贏球，就能憑藉較佳的得失球反超進入前四，球員們的鬥志肯定是非常高昂。

執教熱刺後穆里尼奧改變了他一貫保守的戰術風格，幫助中前場的球員重新找到進攻良方，在這7場比賽中已打進了19球之多。球隊成功反彈，有賴於被前任棄用的埃里克森，解放了中場阿里及重用實力不俗的摩拉，配合狀態極佳的凱恩和孫興慜，前鋒線重新打出了活力與鬥志。後防位置的阿爾德韋雷爾德剛剛與俱樂部續約，勢必用良好的表現來回報穆里尼奧的信任，為球隊提供最強的後盾。

切爾西本賽季在蘭帕德的領軍之下，一度取得各項賽事的7連勝，只是最近5場聯賽1勝4負的表現有些滑落，被熱刺追到只有3分的距離，如果再有閃失隨時會跌出前四位，面對的壓力非常之大。這個賽季藍軍受重用的亞伯拉罕、芒特和普利西奇等小將，令前場的活力與速度得以提升。只是年輕球員有些缺乏穩定，狀態的起伏難免會影響取勝。加上最近7場都有失球的防線漏洞百出，祖馬和托莫里的表現難以讓人放心。這次面對火力全開的熱刺壓力極大，在客場出擊能保住不敗離場，已經算是不錯的結果。

值得一提的是，上個賽季蘭帕德在英冠的德比郡初掌帥印，曾經在聯賽杯互射點球將穆里尼奧帶領的曼聯淘汰出局。不過，傳統豪門打本土杯賽大多不是以最強陣容應戰，因此也很難看到真正水平。現在的藍軍問題確實不少，球隊出現疲乏狀態下開始陷入進球不如失球多的困境，要等到下個月解禁轉會後買人，才可以緩解板凳深度不足的燃眉之急。展望今夜這場師徒對決，蘭帕德認為，從以往穆里尼奧反水戰的歷史來看，相信此戰絕對不會對自己手下留情。
