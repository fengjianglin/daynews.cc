獨家專訪斯托伊科維奇：如果換掉我，將是一個很大的錯誤

天天要聞
2019-12-19

記者王偉報道 2019賽季中超聯賽，廣州富力最終排名第12，成績未如理想，在廣州富力已經執教四年多的主教練斯托伊科維奇也受到了一定的質疑。斯托伊科維奇的「漂亮足球」在整個中超獨樹一幟，但防守確實是制約富力前行的關鍵，本賽季他們的失球總數達到了驚人的72個。

賽季結束前，斯托伊科維奇接受了本報的獨家專訪，梳理了球隊今年的表現，談到了球隊的防守、自己的執教思想以及射手王扎哈維。而對於自己的帥位問題，斯托則直言：「如果換掉我，將是一個很大的錯誤！因為我相信新來的主帥無論是誰，都不會比我更清楚、更了解富力這支球隊！」

在接受本報獨家專訪時，與廣州富力還有兩年合同的斯托伊科維奇明確表態，希望繼續為廣州富力效力。然而，到目前為止他的工作能否繼續還沒有明確定論。

◆《足球》：斯托教練您好，2019賽季富力隊的聯賽起伏跌宕，精彩比賽不少，但也有一些大比分輸球的紀錄，截至目前對本賽季，您有什麼樣的總結？

斯托伊科維奇：我記得賽季初的時候有幾場球，可能因為最後時間段的丟球導致了我們丟失了一些分數，當時我們的期望是拿到更多的分數，開個好頭，有利於接下來的比賽，但是那幾場丟球可能讓我們丟了10分左右，也是相當的不走運。

我覺得球隊在信心方面受到了一些影響，當球員們踢下一場比賽的時候，他們也會承擔著一些壓力，這樣的壓力也會導致在球場上面不能把他們良好的一面發揮出來，當然，沒關係，未來還有很多比賽可以踢，我覺得可以利用每一場比賽逐漸踢出球隊的技戰術和願景。

當然我也一直在不斷總結這個賽季球隊的得失，正如開始我說的那樣，如果能有更多一點運氣的話，有更多發揮空間，我相信我們整體和排名會靠前一些。這是其中的一部分。

◆哪些場次的比賽是您最難忘的？

很難去單獨找出這樣的一場比賽，因為每一場比賽都是挑戰，每一場比賽都有不同的情緒、不同的情感，都會帶來不一樣的能量，所以很難去單獨說哪一場比賽印象最深刻。賽季初，我們場面上踢得非常好，但沒有拿到我們想要的分數。第一場平重慶，在天津客場輸了比賽，那段時間讓我非常有壓力，也非常失望。不是因為對手踢得比我們好，而是我們自己的一些失誤，導致該拿的分數沒有拿到。

◆球隊成績出現困境的時期，有不少換帥的傳聞，那時您跟富力高層有過交流嗎，交流的內容是什麼？

當時我也聽到過這些傳聞，這些有可能真也有可能是假，我無法做出什麼干預。我能做的就是專註自己的工作，自己所能做的事情，去帶好球隊，因為在足球的世界裡沒有一項工作是輕鬆的。

我知道當球隊成績不好的時候，外界會有換帥的聲音，但是我覺得，如果換掉我，將是一個很大的錯誤！因為我相信新來的主帥無論是誰，都不會比我更清楚、更了解富力這支球隊。在過去的四年裡面，我給這支球隊創造出了很和諧的氣氛，當我剛來到中國的時候，整個俱樂部、整個球隊都不在一個很好的氛圍當中，我覺得通過我的努力讓球隊更加團結，讓球隊的氛圍更加良好，給外界展示出了風貌和形象。

正如我剛來中國執教的時候說的，我來中國不是為了錢，我是真正希望為中國足球帶來一些貢獻。如果我想要賺錢的話，可以去不同的地方去賺錢。

◆很多人感覺這幾年您沒有發揮出來執教水平。

正如剛才我所說的，我個人性格是，越多這方面負面的東西，會讓我自己變得更加強大。看看我的肌肉，很強大！我在骨子裡是一個永不放棄的人，可能有些教練遇上相似的情況，會承受不住這樣的壓力，會離開自己的崗位。

最重要的一點是，希望大家不要慣性認為自己就是一個失敗者，即使在失敗的時候自己接受這樣的失敗，但也得昂起頭來，走出來，不要因為隊友、隊員的失誤去責怪他們。不管有什麼事情都得先想想自己，承擔起自己的責任，先把自己做好，才能起一個帶頭的作用。每個人都去更多地反思和思考如何去做得更好，如何去為了球隊做更多的奉獻，不能只是去責怪或者抱怨。

◆《足球》：很多人對於您一直堅持進攻的打法相當認可，您是怎麼看的？

斯托伊科維奇：外界很喜歡我們的風格，我們踢的是很進取、很開放的足球。我是富力的簽約教練，我和富力還有合同，所以我會一直忠於富力，為了廣州富力去拼搏、去努力。我知道其他球隊的球迷喜歡富力的進攻風格，對我來講，富力是我的家，我在這裡執教的四年的時間，我會為富力這個家努力拚搏，全力奮鬥。

◆剛才您提到了球隊的防守，從上賽季到本賽季，整個中超都在聊富力的防守話題，作為球隊的主教練，你覺得富力的防守問題怎樣才能根本性解決？

我覺得重點首先要實戰，要刻苦努力地訓練，要多交流、多總結。通過錄像視頻的分析，全方位地去給他們做一個深度的剖析，同樣的還要讓球員們找回他們的自信。因為球隊的上限、能力是擺在那裡的，作為教練，我最大的職能或者目標就是要激發球員們最大的能量，來展示出他們最真實的實力出來。

我跟大家分享一下比較有趣的事情，在足球場裡面最困難的就是如何去取得進球，但是我相信大家也知道我們在進攻端沒有太大的問題，在足球領域最簡單、最基礎的事情就是去做好防守，但在這一方面我們確實面臨著一些難題。這是一個很奇特的現象！我們創造了很多進球和很多助攻，像扎哈維在幾個賽季都保持了高產射手的形象。我覺得我們應該接受外界的一些批評，因為事實擺在那裡，我們的防守確實做得不足，比如說受到老闆的一些批評，我們也接受。

◆在您看來，這個賽季球隊出現的問題是不是只是防守端的問題？球隊為什麼成為中超丟球最多的球隊？

我們要為球隊的防守致歉，確實外界有很多人批評我們的防守，可能導致球員們在信心方面有一些受挫，當然我們每一場比賽也是通過努力希望把信心贏回來，可以球隊用這些贏球保持勝利者的姿態。現在在中超踢比賽並不輕鬆，競爭也非常大，球員們必須要在心理上和身體上作出充分的準備，才能夠在90分鐘內做好戰鬥的準備。

◆在防守教練到來後，您們做了哪些調整？

我們想要讓球員知道如何真正幫助提高他們的防守，因為新來的教練，他也不是有魔術筆，他不可能來了馬上改變這些事情，但是這是一個很強烈的信息，想要告訴球員們，我們是想要幫助他的，在防守這一方面幫助他們提高。我想展示我在工作上的方法論，是中超裡面最好的、最先進的。我覺得進攻訓練和防守訓練不是隔離開的，總體來說是我的整套訓練的科學性，質量都是很好的。

我不會批評這名或者那名隊員，因為每一次探討的結論是希望他們做得更好，我作為教練要負最大的責任，也是希望激發他們最大的潛能，能夠讓他們拿出100%的能力幫助到球隊。

我想大家都看得到，有些失球是個人的失誤，並不是戰術上的失誤，所以我的工作最重要的就是通過訓練來提高他們，第二方面是通過不斷練習，給隊員們提高信心。在中國執教注意力和信心是非常重要的。

◆《足球》：從2015年下半年到2019賽季，您執教富力有四年多的時間，富力的凌厲進攻成為中超一面旗幟，您是現任中超外教里執掌同一球隊時間最長的，在您執教的過程中，您對富力足球的定位是什麼？或者說富力足球的精神在您眼中怎樣定義？

斯托伊科維奇：首先我要打造一支穩定的球隊、可持續發展的球隊。我不希望今年我們取得一個好成績，然後我們明年就要降級，我不希望是這個樣子。我知道我們球隊的一個定位，在目前的這個階段想要爭奪一些錦標、一些冠軍有一定的難度，但是我希望打造這支球隊踢出現代足球，可以讓球迷開心，讓觀看我們足球的人享受到快樂，這是我的目標。

我來中國執教這麼久，我對中超聯賽非常熟，我覺得在中超聯賽踢球是很大的挑戰，也是很困難的。很容易就會陷入很困難的情景或者一個階段當中，所以我希望能夠帶領一支球隊保持一個比較穩定的態勢，不要出現大的波瀾起伏。

◆如果至此給您在富力的執教做個總結，100分滿分的話，您給自己多少分？過去兩三個賽季球隊的成績有所起伏，但是您到富力的最初階段曾獲得最佳教練提名，執教富力這幾年您覺得發揮了幾成功力？

要明確一點，其實我不太關注我個人的一些名譽，我來中國執教就是希望打造一個好的團隊、一支球隊。教練的利益和聲譽不可能在球隊之上的，他永遠是服務於球隊的。到目前為止，作為教練最開心的是，球隊整個執行力上可以貫徹我的指導理念和思想，執行我的戰術理念，這是我最大的一個滿足。

其實這幾年我在富力的工作思想和理念是沒有變化的，工作基本都是一樣的。我覺得無論在哪裡執教，特別是在中超執教，如果保證球隊人員不變的話，球隊想要保持在很高的水準，而且是持續幾年，其實是很難的。所以我們需要帶來一些新的刺激，一些更好的球員，令球隊有一個良性的競爭，然後再有一個良性的化學反應。

作為教練，每個人都有一顆爭奪冠軍的心，作為廣州富力隊的主教練，我也想帶領球隊贏更多的球，但有時候我們要有一個好的定位，並且要面對現實！我想打造一支穩定的、起伏不要太大的球隊，當然作為教練，我希望給球員良好的教育，可以給他們提供優質的訓練資源和內容，讓球員們不論場上還是場下都能作為一個榜樣，做更好的職業球員、做更好的人。這才是我的理念和初衷。

◆年初您提到過本賽季向亞冠衝刺的目標，如果向這個方向再進一步的話，還需要做哪些具體的工作？

很多球隊都會提出這樣的目標，但是沒有人百分百確保說一定能達到這樣的目標，因為有時候我們還是要回歸到現實，看清楚球隊的實力和定位所在。我想要做的就是要有一個長期長遠的目標，不是急功近利在一些近期的成績方面有什麼突破，我是想為富力建立穩固的根基。

◆《足球》：本賽季富力隊引進了多名內外援，外援有登貝萊、薩巴，內援方面引進了鄒正、晉鵬翔、金波、桂宏、范雲龍等人，對於本賽季的表現，您對引進的這些隊員的發揮是如何評價的？薩巴和登貝萊是否符合您的預期？

斯托伊科維奇：我們將雷納爾迪尼奧租借給天津天海之後，希望對位找到一位功能方面和他類似的隊員，當然對於薩巴來說也相當的不容易，我相信在大家看到他的進球和助攻數據之後，對他的表現應該也是滿意的，畢竟這是薩巴來到中超踢球的第一個賽季，也需要有一個適應的時間。中場方面我們在烏索離開以後引進了登貝萊，也是在對位的調整。

引進的幾名內援，像范雲龍是非常有經驗的隊員，也達到了我給他的一些在中場位置指導上的要求。鄒正之前受到一些傷病的影響，需要更長的一段時間找到中超比賽的強度和節奏，去做一些適應。

在進攻端和防守端的一些失衡，使球隊的整體成績有一個起伏。我希望我們下賽季能夠再引進兩名實力更強勁的隊員來幫助球隊，這樣整支球隊的競爭力會更加提升，然後再去為亞冠的目標努力。

我們希望在人員引進方面做到主動，我們的管理層在四個月以前就著手準備，為下賽季打造一套新的陣容，新的球員引進已經在努力了，希望這樣的努力可以得到回報，不想在聯賽結束之後再去討論這些問題，如此局限性就會很大。

為了保持球隊的競爭力，通過人員上的交換和流動，我們球隊的人員結構和更新換代也是非常有必要的。這樣才能讓我們在關於亞冠的目標上能夠更好地設定。

◆您覺得廣州富力在球員結構上怎樣？需要什麼調整？

我們的隊員也不是那些年輕的隊員，他們經驗豐富，也是踢過很多場比賽的隊員，所以要想在他們個人能力上做一個飛躍式的提高其實是非常困難的，我覺得作為一個整體，作為球隊，我們還是有一個上升的空間。不僅僅是咱們富力的隊員，有些其他隊的中路隊員可能在球場上面一瞬間就丟失了這樣的信心，會影響到他們的發揮，如何在信心方面保持一個持續的、持久的信心是非常重要的。

我的印象是，可能有時候輸掉比賽之後，球員們看到了一些負面的評論或者批評，會影響到他們幾個人的一些心理，造成他們信心缺失。

對我個人而言，我的性格是，我覺得這些批評、這些指責會讓我更加強大，因為通過這些批評，一方面我可以找到自己的一些不足，另一方面我會想用我的實際行動在球場上證明，個人的實力和價值。

當然，我們的訓練隊員們還是非常刻苦的，隊員們也很投入，因為這是富力取得好成績的根本的保證。

2019賽季我一直在思考爭取拿更多分數，讓球隊的排名可以更加靠前。也是希望有一定的成績和排名，讓球隊這幾年來可以保持一個比較持續穩定的趨勢，然後在下賽季我們在帶來一些新的元素，讓球隊的競爭力更加強大，去競爭亞冠的名額。

◆《足球》：扎哈維在富力創造了新的單賽季進球紀錄，他之所以能創造這麼多進球，甚至還有插花腳破門的精彩表現，您給他提供了哪些細節上的指導？

斯托伊科維奇：我做廣州富力的教練，其中一個很大的成功就是引進扎哈維，這是到現在都讓我一直很開心的事。因為當時扎哈維在國際足壇也不算很出名，說起他很多人不知道是誰。當時能將他帶到球隊，我想這也是為富力引進了一筆很大的財富，因為現在大家都知道他的能力所在。

日常訓練方面，我和他分享最多的就是耐心，希望他更加耐心，不要心急。有時候我會給他指導：一些更好的機會，你可以把球分出去，而不是持球進攻。通過隊友做些配合，會讓你的進球機會和效率大大提升。當然一些跑位的問題平時我們也有很多探討，作為教練，這也是我應該做的。

我希望通過自己的努力，讓扎哈維提升自己，他也通過自己的努力去提升水平。有時候他上半場表現糟糕時，我也有過換掉他的想法，但總的來說，他的表現還是讓我非常滿意。作為一個導師，我可以帶給他一些不一樣的視角。他很尊重我，無論我們的意見是否統一，他會以我的建議作為討論的結束，而不會去挑戰教練的權威，在足球上，他非常職業，我們看得到他平時的訓練都非常刻苦。

◆創造單賽季進球紀錄對扎哈維和球隊有何特殊意義？如果比較球技，您認為自己比他更出色嗎？

這是他在中超獲得的巨大成就，我認為他配得上這樣的榮譽。當然，我還有一個希望，就是通過他的進球幫助球隊獲勝。如果對比我踢球的時候，我應該踢得比扎哈維更好一些。（笑）
