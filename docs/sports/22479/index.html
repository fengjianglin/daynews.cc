遼寧兩連敗後還有壞消息！韓德君不幸再度受傷，郭士強要背鍋？

2019-12-20 388條評論 2,659次閱讀 1740人點贊

北京時間12月19日，在剛剛結束的一場CBA焦點之戰中，坐鎮主場的山東男籃以114-94大勝遼寧，在延續了自己8連勝的同時，送給對方兩連敗！值得一提的是，本場比賽是遼寧功勛外援哈德森離開球隊後首次面對老東家，而今天的哈德森狀態神勇，全場出手22次命中13球，三分12中5，狂砍41分10籃板7助攻！遼寧男籃三大外線今晚繼續他們的打鐵大戰，師弟三分10中1，趙繼偉全場10投2中，郭艾倫15中6……更加糟糕的是，在輸掉了這場比賽遭遇兩連敗後，遼寧男籃再度收到壞消息，他們的首席內線大將韓德君在比賽末節的一次回防中突然和山東男籃的賈誠絆在了一起，痛苦倒地，隨後一瘸一拐地走了下去。

從回放中我們能夠看到，韓德君這球是自己「絆倒」了自己，雖說和賈誠也有一定的身體接觸，但很明顯是因為他過度疲勞，肌肉長期緊繃的情況下的意外受傷！有的球迷對此感到非常擔心，就怕是韓德君的跟腱出現了問題，在跟腱繃緊的時候被另一隻腳提到，這種情況最容易跟腱斷裂！

說到這裡，小編不得不談一下郭士強。今天郭士強在調兵遣將方面，明顯是比不上對面的鞏曉彬的。鞏曉彬知道要用外援去帶動國內球員，知道國內球員打開了，比賽才能拿下。而郭士強卻一昧只會死操三大外線，然而今天郭艾倫、師弟和趙繼偉的投籃並不准，幾乎把山東主場的籃筐都給打歪了。而且比賽到了最後兩分多鐘，明知道這場會輸了，郭士強卻堅持要繼續使用主力，不讓年輕人上場去鍛煉、感受比賽。韓德君傷愈並不算太久，經不住這樣的長時間的使用，出現傷病，似乎也並不稀奇！

目前韓德君傷病的具體情況，還無法得知。但從韓德君倒下的痛苦表情，以及下去時候的一瘸一拐我們便能知道這次他受的傷其實並不輕！如果韓德君遭遇什麼嚴重的傷病，那遼寧這賽季估計就十分危險了，畢竟失去內線首席大將，即使能夠殺入季後賽，恐怕也會像上個賽季那樣，走不遠！

況且本賽季的遼寧狀態本來就不佳，最近剛剛取得連勝，又接著來了個兩連敗，士氣隨即又跌。過去兩個賽季，遼寧的戰績不升反降，主教練郭士強要負很大的責任，很多遼寧球迷對他已經非常不滿，本場比賽他過度使用主力，導致韓德君在垃圾時間裡受傷下場，「郭士強下課」的聲音或許會更大了！
