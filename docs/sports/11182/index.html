CBA最貴外援被裁！7場球給多少錢？人傻錢多？5200萬替身來了

2019-12-11 2,337條評論 3,447次閱讀 475人點贊

新賽季的CBA目前已經進行了差不多三分之一的賽程了，各支球隊為了自己的賽季目標也繼續努力著，因為CBA外援政策的改變和往年不同的是新賽季CBA興起了換外援的熱潮，截止目前超過多支球隊更換了外援。而新賽季廣廈隊花費了重金將NBA球星法里埃德招入隊中，但是這位在NBA球星似乎並不適合CBA的賽場，作為能在NBA立足的球員，他在CBA的場均數據僅有17.3分10籃板，作為薪金如此高的外援顯然這樣的數據不合格，要知道本賽季法里埃德的薪金為440萬美元，是本賽季CBA薪水最高的球員！

而根據報道，目前法里埃德因為手腕受傷已經返回了美國養傷，而媒體爆料他已經被球隊裁員，並且用前NBA灰熊隊球員普拉姆利取代了他的位置！普拉姆利今年31歲，是2012年進入NBA聯盟的首輪秀，截止目前他擁有346場NBA常規賽的經歷，職業生涯場均可以得到4.9分4.5籃板數據，是NBA賽場上有一定的籃板球能力和蓋帽能力，而且是一位防守出色的藍領球員！值得一提的是普拉姆利在2016年曾經和雄鹿隊簽下了一份4年5200萬美元的大合同！而且他曾在2014年入選過美國男籃的陪練隊！他職業生涯最好的一個賽季是在2013-14賽季的太陽隊度過的，那一年他場均可以貢獻8.1分7.8籃板的數據，是球隊的主要輪換球員！

法里埃德被裁掉確實有點可以，作為天賦出眾的彈跳男，他的天賦屢屢為我們奉獻精彩的空中接力和封蓋，但是無奈在和球隊磨合越來越好的情況下手腕受傷，這也是他被裁的主要原因！本賽季他一共為廣廈隊效力了7場球，那他能拿走多少薪水呢？

大家都知道，CBA在美國球星眼中是「人傻錢多」的代表，此前NBA球星的中國行活動是他們瘋狂吸金的常規操作，而且不僅如此，CBA也是他們眼中的「養老院」，很多在NBA即將退役的球星在職業生涯的暮年也會來CBA來撈一把。甚至此前也有很多外援拿著高薪卻出工不出力！而新賽季為了保障各球隊的利益，約束外援行為，姚明出台了新規定，CBA球隊與外援簽合同的方式有兩種，一是月簽，換句話說就是按月支付薪金，而第二種是按場次簽，前10場為試用期，前20場為非保障合同，20場後轉為全額保障。這樣的做法一方面可以讓外援盡心儘力，另一方面，球隊也有了主動權，不合理的外援可以被替換！

而根據媒體透露，法里埃德每一場比賽的費用大約為9.5萬美元，由於是試用期階段，真實數額會略低於這個數目。7場比賽法里埃德保守估計可以得到了55萬美元左右。這樣的薪金對於NBA中的籃球巨星來說雖然不值一提，但是對於普通人來說這樣的金額是一份巨款，大部分球迷一輩子的工資也估計沒有這麼多吧！
