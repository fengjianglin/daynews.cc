天天要聞

皇家馬德里隊的特別球員維尼休斯在西班牙幾乎讓他的批評者們閉嘴

天天要聞
7 天 ago

在周日皇家貝蒂斯隊戰勝畢爾巴鄂隊的比賽中，華金上演了精彩的帽子戲法，贏得一片歡呼雀躍的讚歎聲。即便是稍微有點靈魂的人，也不會吝惜一個逗號、一個母音或一個感嘆號。在他的整個職業生涯中，他的頭三粒進球都是在主場打進的，比賽還沒過20分鐘，就被塞進了運動員的球門，從史上最偉大的阿爾弗雷多·迪·斯特凡諾手中奪走了西甲年齡最大的帽子戲法進球記錄，最重要的是貝蒂斯的勝利。

選擇你最喜歡的。他的年齡嗎?(38)。事實上，他是一個標誌性的球員，他熱愛的俱樂部的隊長?還是說這個非常有趣、興高采烈的傢伙真的在一個月又一個月地進步?(這個事實與這個帽子戲法無關。這是一個客觀的事實，儘管用譴責和輕蔑的語氣來說，離我們過去所說的「中年」還有一年半的時間。是完成的質量贏得了你的芳心嗎?

看,你選擇——它是關於個人品味,但在西甲足球的一個最高級的周末到處都你看,發生了一件不尋常的和令人興奮的,華金的表演掩蓋一個年輕人飾演華金的立場和14個月大時,貝蒂斯的出色的隊長打入一球,早在2001年9月,安靜地表明他取得了顯著的進步。

他叫維尼修斯·何塞·派索·德·奧利維拉。小維尼休斯，我要重申一下我上個賽季說過的話:這位19歲的巴西球員絕對是皇馬的瑰寶。

這讓我憤怒了好幾個月，就像馬塞洛和卡里姆·本澤馬指出的那樣，很多球迷和媒體很快就忘記了這個孩子是上個賽季馬德里最傑出、最有成效和最重要的足球運動員之一。僅僅因為他的表現還不能令人信服，更不用說成為精英了，他周圍就籠罩著一層悲觀情緒。此外，因為一個稍微年輕一點的巴西人(羅得里戈)已經突破並顯示出無論他有什麼其他的天賦，他在對方禁區有更銳利的咬合力，許多人對這兩者進行了不必要的和無益的比較。

維尼修斯二世和羅德里戈有著相同的國籍、大致的年齡和共同的挑戰，但是他們在聖地亞哥伯納烏和馬德里的瓦爾德貝巴斯訓練場上一起成長，形成各自的足球皮膚，互相補充。他們是馬德里的雙料資源——而不是兩個孩子被扔進競技場的大坑，目的是只有一個人能夠生存和進步。什麼該死的垃圾。

維尼是特別的。強大和擁有可愛的優雅，運動和技術，一個感覺群眾更多來。很快。他的最後一擊受到了不必要的批評。他在進球前並不冷酷無情，他偶爾會抓住機會，很明顯他經常會被機會的咆哮震聾，而且越來越多地聽到馬德里球迷的抱怨。這些都是真的。但是，他的這一部分工作仍在進行中，讓我向你們指出一些事情。

維尼休斯在19歲零2個月的時候打進了他的第15個高級進球。相比之下:華金直到22歲生日的一個月後才實現了那麼多的高級進球。克里斯蒂亞諾·羅納爾多比這位巴西人多花了7個月的時間才在體育和曼聯踢進15個球，而世界上最偉大的球員里奧·梅西直到19歲零4個月時才在巴塞羅那一隊踢進15個球。維尼休斯擊敗了他們三個人。更重要的是，有相當明確的證據表明，馬德里的形勢正在導致他們年輕的天才資產停滯不前，或者至少是原地踏步。

維尼休斯的大部分進球都來自弗拉門戈，無論是在巴西聯賽，南美杯還是解放者杯。如果你花時間和精力去觀察它們，你會發現驚人的事情。首先，維尼休斯有能力出色地完成比賽。就拿他本賽季對陣奧薩蘇納的揭幕戰來說，他的右腳從禁區左側切入，然後射進魯本球門遠角。那種力量，那種自信，那種「令人驚嘆的因素」——這些都是維尼休斯在弗拉門戈的標誌。

首先，在巴西，他被信任扮演一個角色。他的角色有點像本澤馬或內馬爾，不是9號前鋒，而是一名中鋒，速度快，能快速下墜，能旋轉，然後能抓住機會，能讓隊友立即反覆尋找帶球的人。甚至在17歲的時候。他在厄瓜多對陣埃姆勒克的比賽中的兩粒進球，是由於他的左腳沒有充分利用，幫助球隊在1-0落後的情況下以2-1取勝。在巴西對陣塞阿拉、恰科恩塞和瓦斯科的比賽中，他在禁區的「禁區狐狸」區完成了最後一擊。當這個男孩相信的時候，當他被相信的時候，他可以做很多事情。

維尼休斯的不幸是，本澤馬在皇馬的一周又一周的比賽中都佔據了核心位置，而現在他的狀態是最好的。齊達內喜歡儘可能的踢4-3-3，這是一個更大的悲哀。這意味著當本澤馬首發時，「邊鋒」是必須的。也許不是老式的觸地戰術，而是邊路球員，他們必須開始用粉筆畫線，這樣球場才會水平延伸，而皇馬的中場球員可能會有空隙。

「球員總是想進球，但我不會要求他進球，」齊達內在周六2:0戰勝西班牙人後說。「今天維尼休斯的工作非常出色。他在防守和進攻上都做得很好。」

隨著氣候、財富、注意力、語言、文化、壓力、責任和批判焦點的變化，地位、戰略和學習也發生了變化。如果你想知道為什麼維尼休斯在對奧薩蘇納的比賽中取得「世界冠軍」後跪下來哭泣，這些就是你的線索。

馬塞洛談到維尼休斯的情況:「儘管他只有19歲，但他的頭腦很清醒。在皇馬，你可以踢三場精彩的比賽，然後踢一場糟糕的，他們會批評你，要求更多。馬德里的壓力比巴西大得多。」

這名邊鋒在周六對陣西班牙人的比賽中表現出了顯著的進步，這幫助他和他的球隊在接下來的兩場比賽中都有漂亮的表現。

維尼休斯的觀點是:「當我離開弗拉門戈的時候，我有點難過，但是當我來到馬德里的時候，我在頭兩個月里的進步比我在巴西一年的進步還要大。這裡的訓練不同。皇馬的大部分訓練都比主場的比賽好。在訓練中，沒有人願意失敗:每個球員都在努力提高自己。」

如果你願意，不妨回顧一下他在對陣西班牙人時的表現。在三次重要的比賽中，維尼休斯擺脫了戈麥斯、洛佩茲和吳磊的防守，並在客隊門將洛佩茲的防守下成功撲救。我們沒有對球瘋狂的猛擊，也沒有瘋狂的失誤——只是一種冷靜，「把球對準目標，讓門將努力，如果他想把球擋在外面的話」的態度，讓他在去年2月對陣阿爾維斯的比賽中以2-0拿下了點球。

在戰勝西班牙人的比賽中，他不斷地抬起頭來，尋找托尼·克羅斯(第57分鐘的進球機會)，卡塞米羅，費德里科·瓦爾韋德或者本澤馬(後者在第70分鐘有一個非常明確的機會)。他可能在不到一個月的時間裡還沒有開始在西甲踢球，他的知名度可能已經被強大的華金這個周末掩蓋了，但是有人已經在訓練場上和維尼西烏斯一起努力了，這個可愛的，有天賦的，令人興奮的孩子一直在觀察，學習，很明顯在成長。

在這個過程中，他也獲得了力量，因為像他這樣的人讓足球變得偉大。
