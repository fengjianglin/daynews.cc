利物浦2-1蒙特雷晉級決賽，菲爾米諾補時絕殺

2019-12-19 976條評論 4,774次閱讀 1268人點贊

北京時間12月19日凌晨1:30，2019年世俱杯半決賽第2場，中北美冠軍聯賽冠軍蒙特雷對陣歐洲冠軍聯賽冠軍利物浦。

上半場，薩利赫助攻凱塔破門打破僵局，蒙特雷火速扳平，易邊再戰，菲爾米諾絕殺。最終全場比賽結束，利物浦2-1蒙特雷。

【精彩回放】

場邊的索斯蓋特。

第4分鐘，戈麥斯分中路，沙利奇接球內切兩步弧頂處的遠射打偏了。

第11分鐘，薩利赫前場右路直塞，凱塔跟進禁區內推射破門，利物浦1-0領先。


第12分鐘，帕蓬禁區右側的抽射打高了。

第13分鐘，蒙特雷前場左側定位球機會，後點小角度抽射被門將撲出，富恩斯中路補射破門扳平比分，利物浦1-1蒙特雷。

第18分鐘，張伯倫中圈附近連續晃動擺脫。

第22分鐘，薩拉赫前場右側腳後跟傳球到禁區，米爾納禁區右側小角度捅射被封堵。

第25分鐘，帕蓬禁區右側前沿直接抽射，被阿利森飛身撲出。

第43分鐘，利物浦前場左側直塞，凱塔插上單刀機會錯失良機。

第50分鐘，帕蓬前場中路自由球直接射門，被阿利森飛身撲出。

第61分鐘，利物浦右側角球機會，中路頭槌擺渡，奧里吉破門，但越位在先進球無效。

補時階段，阿諾德前場右側低平球傳中，禁區內菲爾米諾插上搶點破門，利物浦2-1蒙特雷。

【出場陣容】

利物浦：1-阿利森；7-米爾納（75』66-阿諾德），12-戈麥斯，14-亨德森，26-羅伯遜；20-拉拉納，8-凱塔，15-張伯倫；23-沙奇里（67『10-馬內），11-薩拉赫，27-奧里吉（85『9-菲爾米諾）

蒙特雷：1-巴洛韋羅；33-梅迪納，3-蒙特斯，4-桑切斯，11-萬吉奧尼；29-羅德里格斯，16-奧爾蒂斯；8-帕蓬，20-皮薩羅，17-加拉爾多；7-富恩斯

替補：6-古鐵雷斯，10-烏雷塔維卡亞，14-薩爾迪瓦，19-拉雲，21-岡薩雷斯，22-卡爾德納斯，23-巴斯克斯，24-雷森德斯，25-喬納森，32-梅薩，94-梅賈
