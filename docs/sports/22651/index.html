中國女網的2019年：眾金花亮點頻出，一人成績突飛猛進

2019-12-19 433條評論 1,627次閱讀 609人點贊

阿卡普爾科，2019年3月2日，中國姑娘王雅繁在墨西哥網球公開賽女單決賽中，以2比1擊敗美國選手肯寧，奪得了職業生涯第一個WTA女單冠軍。這個冠軍讓她的WTA單打世界排名上升了16位，至第49位，這也是她首次躋身世界前50名。以此為契機，王雅繁最終在今年WTA女單排名中位列第48位。由於她的崛起，今年中國網協女球員在WTA年終單打世界排名前50位中佔據了4席。

墨西哥公開賽王雅繁獲得女單冠軍。新華社記者辛悅衛攝

2014年9月18日，中國第一個、也是目前唯一一個單打大滿貫球員李娜宣布退役。自那時起，中國女子網球進入了「後李娜時代」，這個時代有什麼特點？中國網協球員在2014年至2019年WTA年終單打世界排名榜上的比重變化，也許能說明問題：在世界前100名球員中，中國球員人數從2014年至2019年對應的數字分別是3、1、4、5、4和6；而在前50名的人數上，這6年的對應數字分別是1、0、1、3、3和4。人們可以從這兩組數字對比中，清楚地讀出一個事實：中國女網在經歷了一個低潮後，正在逐年壯大，而且上升勢頭相當猛。

一個有趣的時間巧合是，2014年中國新增了天津、南昌和武漢三站WTA賽事，其中武漢網球公開賽是僅次於北京中網的WTA「超五級」賽事。2015年，珠海又舉辦了WTA超級精英賽。今年，WTA年終總決賽落戶深圳。而WTA頂級賽級別的鄭州網球公開賽的舉辦，使中國國內集齊了從125K級到皇冠明珠級的WTA賽事。這些賽事的舉辦到底對中國網協女子球員能力的提升有多大助力？這恐怕很難說得清，但可以看到的是，2013年張帥在廣州拿下職業生涯的單打首冠，2016年彭帥在天津奪得單打首冠，2018年王薔在南昌拿到單打首冠後，又在廣州拿下第二冠，並在武漢和北京連續打入4強。

今年中國女子網壇同樣不乏亮點：除了突飛猛進的王雅繁外，張帥年初搭檔斯托瑟奪得了澳網女雙冠軍；鄭賽賽在聖何塞站單打比賽中奪冠，圓了自己的WTA首冠夢想，成為繼2011年李娜後，又一位奪得WTA頂級賽單打冠軍的中國網協女球員。此外，她還搭檔段瑩瑩，獲得法網女雙亞軍。事實上，之前幾年，中國女子網壇一直不乏亮點湧現，以老將彭帥為例，2016年她傷愈復出後很快拿下職業生涯的單打首冠，2017年又奪得第二冠，年終世界排名鎖定為第27位。張帥在2016年的澳網女單比賽中實現了大滿貫單打成績的突破。

亮點多但是分散，呈現出整體性的躍升與進步，這就是中國女子網壇在「後李娜時代」給人們留下的印象。即將過去的2019年如是，那麼明年中國女子網壇的風流人物又會是誰呢？

來源：北京晚報

本報記者 李遠飛
