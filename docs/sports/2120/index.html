都想走？高薪難留優質球員，中超聯賽太落後，三大外援渴望離開

2019-12-09 0條評論 21次閱讀 0人點贊

摘要

我們的中超聯賽現在水平到底有多低？這個答案可能中超聯賽自己是感覺不出來，但是按照一些已經出國踢過低水平歐洲聯賽的球員表示，我們的聯賽水平真的太差了。

我們的中超聯賽現在水平到底有多低？這個答案可能中超聯賽自己是感覺不出來，但是按照一些已經出國踢過低水平歐洲聯賽的球員表示，我們的聯賽水平真的太差了。此外，還有一點能體現出這個問題，就是一些大牌的外援來到這裡之後，往往都是踢了很短時間就打算離開，即使是這裡的薪資水平對比他們在歐洲的時候高很多，他們也是堅決打算離開。其實這樣的情況已經很能說明問題，那就是在這些人的心中，中超這個聯賽，其實對於他們的足球事業發展是有阻礙作用的。

​

最近這段時間，有多個中超的外援都有了離隊的想法，首先是卡拉斯科。這個球員堪稱是大連一方的絕對核心球員，不管是盤帶，還是得分，過人，都是絕對的強大。現在他其實和一方還是有4年合約在身的，但是這個球員現在的態度卻相當堅決，那就是一定要離開中超聯賽。

這個事情當然是從今年夏天的時候就開始，只是當時一方採取了比較嚴厲和果斷的態度將其留住。但是現在既然球員走人的心思這麼強烈，其實一方再想強行留人就沒什麼意思了。

除了他之外，上海的兩位球隊的外援奧斯卡和申花隊的伊哈洛，現在也打算離開。奧斯卡現在和球隊的合約是到2020年到期，但是現在雙方的續約意願不是很強烈。在今年夏天的時候，奧斯卡已經和多個歐洲球隊有過聯絡，商談他重新回到歐洲踢球的可能性。而這些歐洲球隊對他的回應是，只要是肯接受薪資降低就可以回歸。同樣也是夏天，他曾經和國際米蘭傳出過緋聞，當時他可能也是捨不得在中超的薪資，但現在面對中超這個爛攤子，他可能也不想等下去了。

另外一個是伊哈洛，他本身就爭議很大，在經歷了這個賽季的低迷和被指責之後，現在離開的想法也相當強烈。據悉，他未來很大可能會回到英超聯賽，這個自己非常熟悉的戰場。

面對這些薪資不菲，但卻依然打算堅決逃離中超聯賽的外援，我們的很多球迷也是不滿意，可是沒辦法，誰讓我們的聯賽水平太低呢？即使是高薪資也留不住人，所以說與其繼續邀請這些大牌來，還不如好好發展我們自己的青訓呢。事實證明將錢這樣用，對於我們自己的足球確實沒本質促進作用。
