阿不都沙拉木21+5+5+4周琦11+14 新疆5人上雙主場逆轉上海

天天要聞
2019-12-23

北京時間12月22日20點整，CBA常規賽第20輪全面展開，新疆男籃坐鎮主場紅山體育中心體育館迎戰上海男籃。前三節比賽，客場作戰的上海始終保持領先。最後一節，新疆絕地反擊，最終新疆主場106-97逆轉戰勝上海。四節比分分別為：25-30；25-23；26-24；30-20（主隊在先）。

CBA常規賽前19輪戰罷，少賽一場的新疆男籃以14勝4負排名第二位，落後領頭羊廣東三個勝場。上海則是7勝12負排名第16位。上輪聯賽，新疆主場力克三外援的同曦，其中可蘭白克三分球9投6中狂轟全隊最高的24分，成為致勝關鍵。上海在上輪主場擊敗了墊底的八一，終於終結了5連敗。本場比賽，上海主帥李秋平將面對舊主，不知道能否在客場完成反戈一擊。

技術統計

新疆方面：全隊5人得分上雙，阿不都沙拉木21分5籃板5助攻4搶斷，費爾德21分5籃板8助攻3搶斷，可蘭白克15分3籃板，米卡15分10籃板，周琦11分14籃板。

上海方面：全隊5人得分上雙，莫泰尤納斯30分28籃板，麥卡勒姆17分3籃板，王潼14分3籃板，施宇晨11分4籃板5助攻，羅旭東11分2籃板。

首節比賽，施宇晨快攻得分，周琦補籃得分，可蘭白克三分命中，莫泰勾手取分，隨著范子銘的補籃得分，6分54秒，新疆15-14領先進入官方暫停時間。阿不都沙拉木兩罰全中，麥卡勒姆連得4分，唐才育快攻得手，新疆始終處於領先。最後2分鐘，麥卡勒姆兩罰全中，莫泰勾手命中。首節結束，新疆25-30落後上海。

進入第二節，雙方率先在內線纏鬥，米卡暴扣得分，莫泰勾手回敬。王潼上籃得分，麥卡勒姆兩罰全中，隨著莫泰籃下暴扣得分，38-29，上海領先9分，8分06秒，新疆叫暫停。此後，新疆展開反撲，周琦補籃得分，阿不都沙拉木快攻打進，新疆將分差縮小至3分。6分33秒，上海叫暫停。暫停之後，張春軍三分命中止血，費爾德飆中三分回敬，阿不都沙拉木和周琦連續衝擊籃下得分，新疆反超1分，上海再叫暫停。最後時刻，蔡亮拋投得手，麥卡勒姆跳投命中。第二節結束，新疆50-53落後上海。

來到第三節，莫泰籃下強攻得手，張春軍外線三分命中，上海領先新疆8分。8分36秒，新疆叫暫停調整。此後新疆加強了內線進攻，米卡和范子銘先後衝擊籃下得分，新疆將比分追至64平，4分57秒，上海叫暫停。莫泰打中2+1，周琦外線三分回敬，雙方再次戰至74平。最後時刻，麥卡勒姆三分命中，費爾德兩罰全中，新疆76-77落後上海。

決勝的第四節，阿不都沙拉木強攻得分，西熱力江兩罰全中，蔡亮因假摔領到第6次犯規被罰出場，新疆領先4分。莫泰籃下連續得分，羅旭東突破跳投命中，上海僅落後1分，8分09秒，新疆叫暫停。可蘭白克和費爾德接連在外線命中三分，隨著費爾德的拋投命中，92-83，新疆領先9分迫使上海叫暫停。莫泰和王潼在籃下連續追分，可蘭白克三分命中，隨著俞長棟籃下補籃得分，新疆領先10分。1分52秒，上海再叫暫停。最後時刻，莫泰和施宇晨進攻先後不中，上海再無追分機會。最終，新疆主場106-97逆轉戰勝上海。

新疆首發：費爾德 可蘭白克 阿不都沙拉木 周琦 范子銘

上海首發：羅旭東 施宇晨 張春軍 王潼 莫泰尤納斯

分類: 體育
