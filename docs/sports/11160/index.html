連犯5大錯誤！CBA頭號黑馬轟然倒下：關鍵3罰不中，丟致命籃板

2019-12-11 435條評論 2,895次閱讀 1797人點贊

CBA常規賽，新疆客場114-110險勝山西，終結對手5連勝的同時，保住了自己聯賽第2的位置。此役對於新疆來說，堪稱驚險萬分。最後時刻，山西隊憑藉原帥連續3分命中，以及劉冠岑突破得分，一度領先新疆2分。但關鍵時刻阿不都沙拉木挺身而出，命中至關重要的反超3分。而山西隨後進攻犯規，並且丟掉關鍵籃板，最終主場惜敗。

賽後，山西主帥王非接受了採訪。王非表示：球隊這場打得不錯，但是在最後時刻出現了不該犯的錯誤。在關鍵的比賽中，球隊還需要進步和成長。王非還點出了球隊最後時刻罰球不中，防守阿不都三分的問題。王非指導也是心有不甘，表示球隊最後完全是有機會贏下比賽的。

正如王非所說，山西此役原本應該是勝利一方。來看看山西最後時刻犯下的錯誤。第一，終場前2分14秒，閆鵬飛2罰不中，對面的米卡2罰全中，新疆103-102反超。第二，原帥連續飆中兩個3分，劉冠岑籃下強打命中並造成犯規，但他隨後加罰不中。第三，阿不都外線接球後，防守球員出現換位失誤，阿不都張手3分命中111-110反超。第四，富蘭克林出現罕見的進攻犯規。第五，西熱力江第二罰不中，山西丟掉後場籃板，徹底宣告比賽無望。

3次罰球不中，一次防守失誤，一個關鍵的防守籃板被對手搶到，山西就這樣讓勝利從指尖溜走。也難怪王非指導賽後有些垂頭喪氣，因為原本他們有機會再次製造一個大冷門。而在富蘭克林全場狂轟42分12板12助10斷的四雙之夜，山西卻輸掉比賽，也讓這個紀錄之夜顯得美中不足。

對於新疆來說，這場比賽的勝利意義重大。單外援作戰，他們頂住了壓力，守住了自己第二的排名，因為一旦本場輸球，他們就將被山西反超。當然，對於山西來說能夠和新疆打到這樣的程度，已經足以見得他們CBA頭號黑馬的成色。
