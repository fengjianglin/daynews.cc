奧運冠軍王楠近況，41歲住豪華別墅，丈夫輕鬆賺10億

天天要聞
2019-12-18

在我國引以為傲的乒乓球項目當中，出現了許多像鄧亞萍、張怡寧、王楠等實力出眾的女性運動員，她們在職業賽場上取得了巨大的成就，因此，她們的個人魅力以及能力在國內都是非常有知名度的，得到了很多球迷的喜愛，而如今她們三位都已經嫁人了，其中王楠還嫁給了自己忠實的粉絲，過著十分幸福的生活。

郭斌是王楠運動員時期的「忠實男粉」，王楠的每一場比賽郭斌只要有空都會到現場加油，恰巧他和國乒教練劉國梁是好友，在他的牽線搭橋之下，郭斌就和王楠產生了交集，隨後便順利走入了婚姻殿堂。作為一個房地產富商，郭斌自然坐擁著上億的資產，而奧運冠軍王楠本身就累積了不少的財富，如今嫁給郭斌更是豐衣足食，41歲變成了體壇最富有的「女富婆」之一。

在國內競爭激烈的乒乓球領域當中，王楠憑著出色的天賦成為了我國乒乓球歷史以來取得冠軍次數最多的運動員，可見她的實力非同一般。王楠在乒乓球項目上的成就，直到如今還沒有人能夠打破，極大地推動了我國乒乓球事業的發展，北京奧運會之後，一代名將王楠選擇了退役，接受了組織的安排，成為了一名在中央工作的人員，不久後就和房地產富商郭斌結婚了，婚後的王楠辭掉了工作，專心在家當上了「富太太」。

值得一提的是，退役後的王楠很少出現在公眾的視野裡面，只不過她前段時間參演了一檔節目，節目組突然造訪王楠家中，王楠所住的豪宅也曝光了。在節目中，王楠帶著主持人來到了別墅的庭院內，院子非常寬敞，還種滿了許多品種的果樹，伸手就能夠嘗到新鮮的水果，由此看來，王楠的別墅是一座莊園式別墅，從規模上看它的價值起碼過億。王楠在節目中曝光的莊園式別墅引起了很多網友的熱議，很多人都表示了對王楠的羨慕之情。

王楠的丈夫郭斌是做房地產生意的，不僅在北京有很多套房子，就連在國內其他地方郭斌也擁有著一定的房產資源，王楠住著價值上億的豪宅也不是什麼奇怪的事情，除了豪宅之外，郭斌還有許多豪車，經濟實力相當雄厚，如今他的房地產事業也發展得非常好，今年8月時，郭斌投資的房地產順利開盤，短短的時間裡面他就輕鬆賺到了10億，身為郭斌妻子的王楠想必也很開心。

王楠在運動員時期就以出眾的外貌得到了很多人的關注，而嫁人後的王楠盡顯貴氣，不僅皮膚狀態保養得非常好，身上還散發著一股大方優雅的氣質，變得更加自信和美麗了。兩人結婚後，郭斌對王楠始終如一，他們也生下了一兒一女，一家四口其樂融融，看來王楠真的是嫁對人了，不知道你對王楠和郭斌這對夫妻有什麼樣的看法，說出來分享一下吧。
