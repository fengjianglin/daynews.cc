國足輸球輸人！李鐵場邊喊破嗓子，賽後表揚部分國腳：表現很好

天天要聞
2019-12-11

北京時間12月10日18點30分，2019年東亞杯男足比賽打響，國足0-2不敵日本隊。上半場，鈴木武藏破門，姜至鵬逃紅牌。下半場，明天爆射擊中橫樑，三浦弦太頭槌破門。國足延續21年不勝日本的尷尬紀錄，遭遇東亞杯開門黑。賽後，李鐵表示——球員很努力，我們場上有些生疏，控球不自信。這場比賽，很多人是第一次代表中國隊出場，他們表現很好，但兩個丟球很遺憾。一場一場打吧！

東亞杯歷史上，國足兩次奪冠，分別是2005年和2010年。上一屆比賽，國足一勝難求，最終排名第3！本次比賽，國足基本是二隊參賽。不過，日本隊派出的是三隊乃至四隊。平均年齡，國足28歲，而日本則是23歲出頭。

交手戰績來看，國足被碾壓，自1975年以來，中國隊與日本隊在國際A級賽中交鋒22次，中國隊6勝7平9負處於劣勢。自1998年2-0擊敗日本後，國足21年未能戰勝過對手。東亞杯上，國足7戰日本4平3負無一勝績！

第8分鐘，國足打出流暢配合，最終董學升小角度射門被日本門將撲出。第14分鐘，姜至鵬角球開出，董學升頭槌高出！第16分鐘，國足差點丟球，日本隊開出前場定位球，畠中槙之輔搶點射門，劉殿座毫無反應，但最終立柱彈回場內！

第18分鐘，畠中槙之輔左路傳中，劉殿座抱住皮球！隨著比賽深入，日本隊掌握控球權，國足在中前場很難碰到球。第23分鐘，三浦弦太動作很大，壓倒董學升，前者染黃。第25分鐘，馮勁傳中被對手解圍。第27分鐘，梅方抬肘打到了上田綺世，主裁判掏出黃牌。第29分鐘，日本隊打出流暢配合，鈴木武藏搶點破門，國足0-1落後！

第31分鐘，姜至鵬心態失衡，他抬腳很高，直接踢到了日本球員橋岡大樹的腦袋。吃到黃牌後，姜至鵬還一臉無辜，但事實上，這是明顯的紅牌動作！如果有VAR，肯定就是直接紅牌了。

場邊，李鐵已經急了，他來到場邊指揮！上半場最後時刻，國足只能在後場倒球，傳不出去皮球了。此外，日本隊打得很細心，國足開始摸不到球。當國足有反擊機會時，對手一逼搶，我們直接傳出邊線。第45分鐘，董學升勉強頭槌偏出。半場結束，國足0-1落後！

易邊再戰，日本隊控制節奏，第52分鐘，張稀哲送出之上，但譚龍啟動慢了一下，對手門將得到皮球。第53分鐘，國足差點扳平，明天禁區里爆射擊中橫樑，錯失良機後，明天抱頭遺憾！

第70分鐘，國足又丟球了，日本隊右側角球開出，三浦弦太頭槌破門，0-2！

第77分鐘，明天鏟倒遠藤溪太，主裁判掏出黃牌！覺得沒有扳平希望後，國腳們也有些泄氣。

最終，董學升扳回一城，國足1-2輸球！
