同時輸球！穆帥索帥說什麼？穆帥為孫興慜辯解！索帥怒斥球員

2019-12-24 372條評論 2,925次閱讀 2005人點贊

剛剛結束的2019-20賽季英超第18輪的比賽中，曼聯輸球了，熱刺也輸球了，而他們輸球之後，和切爾西的差距又被拉大了。當然，比賽已經結束，球迷更加關心的或許是賽後兩隊的主帥怎麼看這場比賽。而賽後熱刺主帥穆里尼奧接受採訪時認為球隊下半場的表現變好了，只是紅牌改變了比賽。而索爾斯克亞這怒斥曼聯球員表現不佳，就像在踢告別時一樣。

賽後穆里尼奧接受採訪時表示：【我認為切爾西上半場表現更好，但是半場0比1落後和0比2落後是兩回事，至於兩個丟球，只能責怪我們自己。我們下半場踢得更好，然後我們少一人作戰，所以比賽變得很困難，儘管如此，我們的球員也做出回應，我們踢得很自信，而且努力去獲得可能改變比賽的進球。如果他什麼都沒做，裁判立即給呂迪格黃牌，孫興慜的反應或許不同，但是裁判沒那麼做，那麼他可能有情緒反應。在我看來，這就是足球，我認為這是一張奇怪的紅牌。】

不得不說，這場比賽熱刺上半場比賽表現真的不怎麼樣，不過下半場比賽埃里克森出場之後，球隊的進攻的確有了很多的改觀，可惜的是得勢不得分，而且在最關鍵的時候，孫興慜被罰下了，不然的話，熱刺真的可能會改變比賽最終的走勢。不過，孫興慜拿到紅牌似乎也是好事，因為這種情況下下輪比賽埃里克森有可能會出場，而目前熱刺缺少的就是中場出球的球員，如果埃里克森可以出場的話，球隊的出球就沒有太大的問題，只不過這不是長久之計，畢竟埃里克森目前還未和球隊續約。

而曼聯輸球之後，主帥索爾斯克亞非常憤怒，賽後索爾斯克亞表示：【上半場比賽踢得就像我的退役賽，但是我的告別賽氣氛更好，這場比賽我們非常壓抑，節奏緩慢，一點都不激烈。第一個丟球來自我們的失誤，第二次丟球也是嚴重的失誤，三分鐘丟兩球對我們來說很困難。今天我非常失望，因為對陣這樣的球隊，你需要拿到更多的積分。】

看來索爾斯克亞對本場比賽曼聯的表現非常不滿，的確曼聯球員本場比賽缺乏鬥志，這還不說，球隊幾乎不是一個整體，前場的馬夏爾、拉什福德和詹姆斯幾乎各自為戰，而林加德一如既往的穩定，這樣的表現怎麼能夠令人感到滿意。而和熱刺相同的是，博格巴上場之後，球隊的表現的確更好了，可惜一樣得勢不得分，最終輸掉比賽也非常正常。

不得不說，剛剛結束的這兩場比賽中，熱刺和曼聯的表現都非常掙扎，而他們輸球或許也輸得心服口服，只不過兩隊雖然輸球，但是也不是沒有收穫，曼聯的中場核心博格巴復出了，他或許可以改變球隊的現狀。而熱刺方面，或許穆帥也意識到了埃里克森的重要性，只是不知道後的比賽穆帥是否給他首發的機會呢？
