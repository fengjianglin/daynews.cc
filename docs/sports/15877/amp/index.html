2-2！巴薩今夜爆出大冷門，29小時後可能收到1大壞消息

天天要聞
3 天 ago

北京時間12月14日23:00，西甲第17輪，格列茲曼和蘇亞雷斯破門，巴薩客場2-2皇家社會，終止各項賽事6連勝(4場西甲+2場歐冠）！ ​​​​

在賽前，英國機構預測巴薩客場取勝概率高達58%，平局可能性只有22%。

這一場2-2爆冷平局，讓巴薩終止對皇家社會的5連勝，上一次在皇家社會身上丟分是2016年11月28日客場1-1。

在16輪過後，巴薩35分還能坐在榜首的位置，但只比少賽1輪的皇馬高出1分。巴薩可能在29小時後收到壞消息：如果皇馬客場能戰勝瓦倫西亞的話，他們將要把榜首拱手相讓。

第10分鐘，布斯克茨在防守角球時禁區內拉拽迭戈-略倫特的球衣，被主裁判判罰點球；

第11分鐘，奧亞薩瓦爾主罰點球命中，巴薩客場0-1落後；

在第38分鐘，蘇亞雷斯助攻格列茲曼單刀破門，扳平1-1；

下半場剛開場4分鐘，梅西反越位成功，單刀機會橫傳，蘇亞雷斯打空門得手，巴薩2-1反超；蘇亞雷斯最近各項賽事的6粒進球，有5球來自梅西的助攻。

在第62分鐘，蒙雷亞爾左路傳中，特爾施特根擋了一下，球剛好落在伊薩克的腳下，後者輕鬆推射破門，最終皇家社會主場2-2爆冷逼平巴薩。

在92分鐘，出現巨大爭議，蘇亞雷斯右路的傳中球，皮克在禁區內被對方球員拉拽球衣倒在地上，主裁判卻沒有判點球。
