國安冬訓計劃已定，3個訓練地點引爭議，球迷：更像度假

2019-12-03 0條評論 23次閱讀 0人點贊

摘要

中超聯賽已經結束了，各個球隊對自己本賽季的表現反應各不相同，恆大應該是最滿意的了，拿到了最後的冠軍，開開心心慶祝。

中超聯賽已經結束了，各個球隊對自己本賽季的表現反應各不相同，恆大應該是最滿意的了，拿到了最後的冠軍，開開心心慶祝。而最遺憾的球隊莫過於北京國安了，本賽季前半程領跑，並且一度成為前半程的冠軍球隊，結果到了後半程一手好牌卻沒打好，被恆大後來者居上，尤其是最後一輪比賽，國安已經儘力拿到自己想要的結果，然而卻以2分之差錯失了冠軍！

如今國安痛定思痛，已經著手為新賽季做準備了，而且據《北青報》的報道，北京國安已經選好了冬訓的地點，做好了拉練計劃。國安選的第一個冬訓地點是西班牙穆爾西亞，這是位於西班牙東南部的海港城市。小編查了一下這裡的天氣情況，穆爾西亞是一個地中海式半乾燥的氣候，溫度在夏天非常高，但是冬天還是比較溫和的，全年溫度起伏只在16-19攝氏度。選擇這裡作為拉練地點確實是一個比較舒服的地方。國安球隊大約會在明年1月3日的時候出發，在哪裡進行一些基礎的體能儲備訓練和熱身賽。

大約20天後國安球隊將會回國迎接春節，過完春節就會去往第二個地點中國海南進行拉練。不得不說年後海南的天氣還是不錯的，國安也挺會挑地方。在海南呆個10天左右，他們就會去到第三個地點蘇州，在蘇州進行集訓，大約10天左右的時間，然後就結束了自己的冬訓計劃。

其實看到國安選擇的這3個訓練地點，首先給人的感覺就是舒服，而且都是一些旅遊勝地。所以也引起了網友的爭議。

有人覺得這安排更像是度假，還有人覺得拉練沒必要全世界各處跑，在北京進行正常訓練就可以了。不過怎麼安排自己的冬訓是球隊的自由，能夠出國訓練並且全世界跑，只能說明俱樂部老闆還是有錢有實力的。也許國安是為了下賽季不惜一切代價給球員最好的訓練，如果明年能拿個冠軍，這些都是值得的。
