東亞杯女足0:3完敗日本，兩戰0球，賈秀急得把隊員名字喊個遍

2019-12-14 638條評論 2,254次閱讀 1232人點贊

在剛剛結束的一場東亞杯比賽中，中國女足以0:3輸給日本隊。連續兩場收穫失利，在第一場0:0戰平韓國隊之後，連續兩場沒有進球。而對於這場比賽，賽前人們還對女足抱有希望，考慮到剛剛在永川四國賽中戰勝紐西蘭，戰平巴西，球迷們還希望女足能夠創造奇蹟。但是，殘酷的現實讓我們再次看到了真實的差距。

本場比賽僅僅3分鐘，中國隊後防就出現失誤，林宇萍解圍失誤，被岩渕真奈抓住機會遠射，還好被彭詩夢撲出。第9分鐘，中國隊再次傳球失誤，被日本隊斷下後連續配合傳球，最終由岩渕真奈鏟射破門。第44分鐘，又是中國隊後場帶球失誤，張睿被斷，日本隊長谷川唯直塞助攻岩渕真奈再下一城。第55分鐘，中國隊再次出現中後衛盯人失誤，被岩渕真奈補射打進帽子戲法。

不停的失誤、失誤和失誤成為中國隊本場比賽的關鍵詞。而主教練賈秀全在場邊急得幾乎把場上隊員的名字喊了個遍，但是也沒有改變這個殘酷的結局。賽後賈秀全在接受採訪時也說：「我覺得隊員們非常努力，但是丟的這些球都是很低級的失誤。我一直要求球員不要在場上犯低級錯誤，今天出現這些錯誤只能說我們技不如人，好好練吧。」

而除了失誤，這場比賽還透露出女足在方方面面的問題，戰術上缺乏明確的思路，前場沒有支點，中場缺乏攔截，造成後防線壓力過大，多次在高壓逼搶下犯錯和出現失誤。加上日本近些年來堅持的技術流傳控打法，讓女足姑娘們吃盡了苦頭。而這些問題不光是女足存在的，在中國男足里一樣突出。而男足在東亞杯里同樣沒有好的結局，不僅1:2輸給日本，還爆出了個姜至鵬對日本球員的爆頭飛踹，讓人對國足的前景再次感到失望。
