第91分鐘絕殺！國奧取開門紅，21歲魯能小將替補上演絕殺

2019-12-09 0條評論 25次閱讀 0人點贊

摘要

北京時間12月8日晚7點35分，為備戰即將到來的2020東京奧運會男足預選賽，郝偉率領的中國國奧隊在珠海迎來大賽前的最後一屆四國賽，坐鎮珠海市體育場迎戰塔吉克U22，最終憑藉替補登場的張凌峰首開紀錄和田鑫第91分鐘的絕殺進球，2-1擊敗對手，取得開門紅。

北京時間12月8日晚7點35分，為備戰即將到來的2020東京奧運會男足預選賽，郝偉率領的中國國奧隊在珠海迎來大賽前的最後一屆四國賽，坐鎮珠海市體育場迎戰塔吉克U22，最終憑藉替補登場的張凌峰首開紀錄和田鑫第91分鐘的絕殺進球，2-1擊敗對手，取得開門紅。

人員方面，主力門將陳威繼續首發登場，，魏震和楊帥搭檔中衛，馮博軒和童磊分居左右，黃政宇和黃聰組成雙後腰，劉彬彬、胡靖航和黃紫昌列在前衛線，此前司職中衛的李揚客串中鋒突前，段劉愚、張玉寧和曹永競等主力攻擊手替補待命。

第17分鐘，陳彬彬左路搶斷後下底在禁區左側橫傳，可惜此役客串中鋒的李揚未能及時趕到位置形成包抄。

第17分鐘，國奧隊製造威脅，楊帥中場線附近送出長傳，李揚伸腳將球蹭入禁區，黃紫昌插上面對出擊的門將吊射，皮球高出。

第25分鐘，國奧隊錯失絕佳機會，黃政宇中場附近得球後送出過頂挑傳，李揚反越位成功插上獲得單刀機會，可惜被出擊的門將率先將球解圍，未能形成射門。

上半場戰罷，雙方0-0悶平，下半場雙方易邊再戰，第50分鐘，國奧隊作出換人調整，郝偉一次性連換四人，迪力穆拉提、田鑫、陳蒲和張凌峰替補登場，換下李揚、黃紫昌、陳彬彬和童磊。

第55分鐘，國奧隊率先打破場上僵局。黃政宇右路帶球突入禁區將球交給胡靖航，後者腳後跟回敲，剛剛替補登場的張凌峰跟進推射破門。

第61分鐘，塔吉克隊通過角球機會由10號球員謝爾里丁門前頭槌破門。

第65分鐘，國奧隊再次作出調整，段劉愚和趙劍非替補登場，換下楊帥和黃聰。

第75分鐘，國奧隊再次換人，張玉寧替補登場，換下下半場被換上場不久的陳蒲。

第82分鐘，國奧隊利用自由球機會打入一球，段劉愚開出右路自由球至禁區，其俱樂部隊友趙劍非頭槌頂遠角入網，進球後的趙劍非瘋狂慶祝。

而此時邊裁已舉旗示意越位。

後點跟進鏟射但沒碰到皮球的隊友越位位置干擾對方門將，進球無效。

進入傷停補時階段第1分鐘，國奧隊上演絕殺，下半場替補登場的田鑫禁區左側得球扣過防守球員後搓射遠角得手。

最終全場比賽結束，中國國奧2-1擊敗塔吉克，取得本屆四國賽開門紅。
