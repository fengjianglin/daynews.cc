恆大首簽即將誕生？球隊欲引進爭議國腳，天價轉會費成其最大阻礙

2019-12-24 514條評論 1,531次閱讀 322人點贊

中超聯賽已經結束有很長的一段時間，各支球隊都已經進入到長達一個月的假期，在此期間俱樂部也是對於球隊各個位置進行補強，只為爭取下賽季能夠取得更好的成績。現如今冬季轉會窗口也是即將開啟，很多大牌外援紛紛和中超球隊發生緋聞，除了外援之外國內球員的轉會更是吸人眼球，這不衛冕冠軍就和一位當紅國腳傳出將要加盟的消息，讓我們一起來看看是怎麼回事吧！

本賽季恆大隊的奪冠道路異常的艱難，直到最後一輪戰勝上海申花才勉強奪冠，這也是主教練卡納瓦羅第一次率領球隊取得聯賽冠軍，其中的意義是非常大。下賽季將會在2月底開打，球隊對於目前左邊後衛的人選還是有些不滿，於是想要通過引援來改善。

眾所周知本賽季恆大的左邊後衛是非常薄弱，李學鵬本賽季因為傷病長期缺陣，榮昊、鄒正也是早已離隊，所以這個位置的補強是非常必要的。於是恆大就把目光放在河北華夏幸福隊的姜志鵬身上，作為國內左邊後衛數一數二的存在，如果他加盟會讓球隊產生翻天覆地的變幻。

姜志鵬如果加盟恆大肯定會引起渲染大波，畢竟恆大現在聚集著很多當紅的國腳，姜志鵬加盟會使得他們的實力更上一層樓。作為最受爭議的一名國腳，能力方面來說他是沒有任何的問題，但是因為在國家隊一系列負面新聞讓他加盟的前景變得撲所迷離。

作為姜志鵬現在所效力的河北華夏幸福來說，這樣一位正值當打之年的國腳離隊是他們不願看到的，即便是真的選擇交易也會要價不菲。這時就有傳聞稱華夏幸福想要索取轉會費高達2.4億人民幣，這樣的天價明顯就是不想將其賣出去，不知道恆大隊對此會有何感想。對此，各位網友是什麼樣的看法，歡迎大家在下面的評論區留下你們的看法，感想大家的支持！
