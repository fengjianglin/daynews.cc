一年時間，賈秀全成功把中國女足“男足化”，沖奧前景渺茫？

2019-12-18 2,056條評論 4,001次閱讀 2567人點贊

中國女足以一勝一平一負結束東亞杯征程，末輪雖然1-0戰勝中國台北，但賽後賈秀全坦言這是帶隊一年多踢得最差的一次大賽，這一場比賽掌控力還不如對手。

他指出這是球員心態失衡，需要全社會共同教育才能提高，這一番發言，立刻引起球迷熱議，中國足球隊官方微博也在第一時間「淪陷」，多是要求主教練下課呼聲。

球迷憤怒的原因在於賈秀全賽後發言，並沒有首先從自身反思帶隊原因，而是把問題推給外界，頗有「甩鍋」意味。

此前，世預賽國足0-1不敵敘利亞，里皮賽後發布會說，隊員沒有踢出訓練中的內容，是主教練責任，我選擇辭職。兩者發言，高下立判。

第二輪0-3完敗日本後，賈秀全面對球迷說道：「」我想贏3個，你們想贏10個，但技不如人啊！」這一次，他把球隊實力歸咎於球員能力，或許這一番回答後，球迷內心情緒已經開始積攢。

快人快語的賈秀齊教練生涯從不缺著名「語句」，最著名莫過於1998年「3號隋波」事件，導致後者提前結束球員生涯。

今年法國女足世界盃期間，談及王霜為什麼不首發時，他表示隊伍需要團隊，不是球星，此言一出同樣把王霜推上風口浪尖。

當然球迷要求賈秀全下課，最大原因還是執教一年多，女足戰績全面下滑，曾經的世界勁旅如今只能小勝中國台北，隊員場上配合生疏，戰術單一更讓大眾對球隊進軍東京奧運會前景感到迷茫。

2018年5月份，賈秀全接替冰島教練西格-埃約爾松出任中國女足教練，同年8月帶領球隊在雅加達亞運會奪得亞軍。

1年前的亞運會，中國女足狀態十分出色，小組賽7-0戰勝中國香港，16-0塔吉克，2-0擊敗朝鮮，1/4決賽5-0過關泰國，半決賽1-0立刻中國台北，打進31球零失球，足以見得進攻火力之猛。

與日本的決賽，雙方鏖戰90分，最終遺憾地被對手替補隊員絕殺。

但考慮賈秀全接手僅3個月，因此這一次中國女足戰績功勞更多歸功於前任冰島教練。

一年後，東亞杯中國女足0-3完敗日本，雙方已不在同一水平。

這場比賽，中國女足方面吳海燕、王霜、張睿、李影古雅沙等上次亞運會決賽面對日本的主力球員均在陣中，在球員框架不變情況下，一年時間，實力反差如此大，主教練或許需要承擔更多責任吧？

明年2月份，東京奧運會女足預選賽第三階段將開打，中國與澳大利亞、泰國和中國台北同組，每個小組前2名出線，進入最後2個參賽名額爭奪。

一年前大眾看到這樣分組，不會為中國女足出現費心，但東亞杯最後一輪全主力出戰的女足，拿中國台北隊毫無辦法，僅憑藉角球制勝，讓球迷開始擔憂賈秀全執教能力。

全場大多數戰術都是長傳沖吊，這種對體能要求極高戰術，讓身體本就處於下風的女足，執行起來難上加難。更讓球迷感到不滿的是以控球技術佔優的王霜，在該戰術體系下，威力大大削弱，以往靈氣消失不見。主教練是否是按照球員技術特點，在制定戰術？

中國女足面臨和中國男足同樣的人次儲備不足現狀，法國女足世界盃期間，曾有數據統計，美國註冊女足人數超過180萬，即便是擊敗中國女足的義大利也有2萬註冊數量，而中國僅有3000人左右。

相比較男足強大的市場號召力，女足吸引力十分有限，如果戰績持續惡化，缺乏社會資金進入，面臨的處境只會比男足更糟糕。

2010年前後，中國女足曾跌落谷底，連續無緣奧運會和世界盃。2013年郝偉接任後開始進入恢復期，2015年女足世界盃打進8強，隨後外教布魯諾-比尼到來，2016年中國女足時隔8年重返奧運會，可以看到在女足人才儲備現狀不變前提下，可以通過教練不斷排兵布陣磨合，讓現有人員戰鬥力最大化激發。

如今，中國女足在賈秀全帶領下，狀態持續低迷，同時外界議論的聲音，讓隊員心理壓力倍增，距離東京奧運會預選賽還有3個月左右，換帥或者能讓這支隊伍重新輕裝啟程，激發出更大戰鬥慾望。
