重鎚碰妖刀 中國雙雄爭奪第五屆富士通杯季軍 劉小光逆轉馬曉春

天天要聞
5 天 ago

　　第五屆富士通杯半決賽上，中國雙雄馬曉春、劉小光雙雙摺戟，令中國棋迷扼腕不已。痛失良機的兩位大將並沒有結束當屆富士通杯的征程，因為他們還要進行一場三四名爭奪戰。

　　一般說來，世界大賽的季軍之爭有點像雞肋，但在早期的富士通杯上，這個季軍還是頗有分量，因為富士通杯規定前三名直接進入下屆本賽，不佔本國名額，對於當時的中國圍棋，多出一個名額是很有意義的事情。

　　截止這盤棋之前，中國大陸棋手還沒有在世界大賽中有過「內戰」，而韓國的曹薰鉉、徐奉洙曾在第二屆富士通杯季軍爭奪戰上交鋒。這一次，馬曉春和劉小光同樣為爭奪第三名而戰，也實現了世界大賽上首次中國德比。

　　當時馬曉春和劉小光分別是中國棋界二號、三號人物，兩人在國內大賽中交鋒甚多，棋風輕靈的馬曉春經常讓剛猛的小光一籌莫展，劉小光曾感嘆：「人怎麼能下過妖呢？」不過在兩人交鋒史上，劉小光也有1988年第二屆中國天元挑戰賽二連敗三連勝大逆轉馬曉春的壯舉，而次年的第二屆中國名人挑戰賽，馬曉春3比0完勝劉小光，開始了自己名人十三連霸的輝煌歷程。

　　首次將兩人的對決舞台延伸到世界大賽上，妖刀和重鎚都想證明自己。

　　第五屆富士通杯三四名決賽


　　黑：馬曉春 九段

　　白：劉小光 九段

　　黑貼5目半

　　共198手

　　白中盤勝

　　對局日期：1992年8月1日

　　對局地點：日本東京

　　早在1980年的全國個人賽上，當時的兩位新銳就有交手，馬曉春快勝，但並未能阻止劉小光首次加冕全國冠軍。

　　　　第一譜 1-50

　　季軍爭奪戰，馬曉春猜到黑棋先行。

　　白10打入積極，但時機　並不適合，被聶衛平指責為「無理手」。

　　黑11飛壓，到黑35的定型是常見下法，白10一子近乎浪費了，馬小早早佔據上風。

　　這過程中，黑33不夠簡明，往另一邊長更容易把控。

　　白38長出作戰，黑39於44位雙住就很好。

　　白44、46用強，棋局有點混亂了。

　　　　第二譜 1-50（即51-100）

　　小光先在右上角定型，然後白6長，意圖強攻黑棋。

　　黑7以下次序井然。

　　黑19斷嚴厲，馬小仍把控全局。

　　白20以下，小光竭力騰挪，但仍步履維艱。

　　黑43貼出，白棋兩邊都很困苦。

　　但黑47大壞棋，此時簡明吃住白棋兩子棋筋，黑棋優勢明顯。

　　白48好次序，白50挖是小光一流的好手。

　　　　第三譜 1-50（即101-150）

　　白6逃出兩子，黑棋中間也變得危險。

　　黑11是手筋，但並不能起到多大效果。

　　馬小選擇轉身戰術，到白26，黑棋中央一團被吞吃，小光扭轉了局面。

　　黑27防守右邊很大。

　　白30二路托，小光再出強手。

　　黑31應該先於33位打吃，實戰次序有誤，小光白34以下機敏先手定型，白40再補回頭，黑棋大損。

　　黑45斷，最後一搏。

　　白46以下計算清晰。

　　　　第四譜 1-48（即151-198）

　　白6提掉，大局已定。

　　白16補角，黑棋實空明顯不足了。

　　馬小收了一會官子，見無力回天，於白48時投子。

　　這場「榮譽之戰」，劉小光在開局不利的形勢下，中盤階段舞動大鐵鎚，打亂了馬曉春的節奏，最終贏得勝利，以第三名的成績為本屆富士通杯征程划上句號，這也是小光在世界大賽中的最好成績。這對亦敵亦友的歡喜冤家首次世界大賽交鋒是小光後來居上，四年之後，兩人在第九屆富士通杯半決賽上還將有一次對決。

　　冠亞軍決賽在大竹英雄和王立誠之間進行，誰會成為新的世界冠軍呢？
