巴薩皇馬平庸0-0！巴薩缺一人梅西成孤雁！皇馬賣C羅17射門都不進

天天要聞
2019-12-19

12月19日，2019/20賽季西甲第10輪一場補賽，西班牙國家德比巴塞羅那主場0-0與皇馬戰平，雙方均無進球，平局收場。但是這場強強對話，實際才真正放映了雙方的實際問題。

上半場從首發來看，皇馬要比巴薩成功，巴薩塞梅多輪換布斯克茨出場不成功，去了右路幾乎被皇馬打小將打爆。而皇馬輪換3人，卡塞米羅、貝爾和門迪進入首發，都發揮了重要作用。上半場皇馬實際控制了中場，而且有多次射門機會都沒有抓住，巴薩這邊的中場幾乎失控，格列茲曼還是不在一個節奏。只有梅西和蘇牙在前場進攻，射門有9次，只有區區2次射正，非常少見。這個水準，歐冠難以走遠。

下半場第55分鐘，巴爾韋德終於將比達爾替換塞梅多出場，這讓巴薩中場終於有了些硬度，場面也逐漸好轉，但是比達爾勇猛有餘，實際沒有布斯克茨，巴薩的中場還是缺乏調度，場面上看，就是出不來球，或者就是沒有高質量的調控，梅西被皇馬多人進行看防後，本場絕對機會並不多，所以蘇牙在前場的機會也不多。而皇馬的中前場整體已經被齊達內修理的非常順暢，為前方輸送多次炮彈，但本場射門高達17次，領先巴薩8次之多，也是不進球，

所以運動幫縱觀本場比賽，皇馬實際就是缺少C羅這樣一錘定音的人物，伊斯科不錯，但是他與貝爾和本澤馬的威力，遠遠小於C羅當初在的組合。皇馬大開大合的進攻已經非常流暢，實際C羅仍就可以完成致命一擊，他也就是需要皇馬這樣的強力支持，看看凌晨意甲他在尤文的進球，跳的太高了！真的很漂亮，34歲了，皇馬現在仍然能有很多這樣的斜傳或直傳球，但沒有這樣的強力點，事實證明本澤馬和貝爾單打都沒有完成。

而巴薩的問題恰恰不同，巴薩本身的特點，他們太需要內馬爾這樣的球員了，格列茲曼很努力，但是巴薩現在的中場不行，即使有了硬度也缺乏創造性，需要前鋒回來支援和支撐，梅西現在根本無法完成又當爹又當媽的角色，就需要內馬爾這樣能傳能控，還能射門的球員，他的存在能夠完成中前場的連接，也就是說，他和梅西可以進行瞬間的分工，蘇牙甚至會有更多空間。

本場比賽，如果皇馬還有C羅，巴薩擁有內馬爾，完全將會把比賽提升一個檔次，而且肯定會有進球。皇馬買來阿扎爾也就是想代替C羅，而梅西們不斷想內馬爾回歸，就是因為現在踢得不舒服，只要梅西被嚴防，巴薩現在就無計可施，本場最為典型。事實證明，93426人見證德比0球！皇馬賣C羅就是錯誤，而巴薩如果無梅西再無內馬爾，將迅速淪落！
