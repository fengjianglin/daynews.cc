歐洲大師賽資格賽┃田鵬飛絕殺喬·佩里，中國選手7人勝出

天天要聞
2019-12-19

17日斯諾克歐洲大師賽緊鑼密鼓地開始了資格賽首輪比賽。中國12位選手率先出戰，經過激烈的角逐後，中國選手7人晉級，5人慘遭淘汰。

■中國選手情況

晉級選手：田鵬飛、梅希文、陳子凡、羅弘昊、李行、呂昊天、周躍龍

出局選手：常冰玉、張健康、陳飛龍、斯佳輝、白朗寧

■完結比賽比分

■完結比賽回顧：

•田鵬飛絕殺喬·佩里

開局田鵬飛狀態一般，喬佩里接連以125分、65分拿下前兩局。

第三局田鵬飛找到手感，打出單桿100分，以109分結束本局比賽。

接下來喬·佩里以71-19贏下第四局。

第五局，田鵬飛開始發力，以73-31拿下本局比賽。

第六局，田鵬飛率先得到53分進入防守，喬·佩里設置斯諾克田鵬飛兩擊不中，最後不得以大力擊球，運氣站在田鵬飛一邊並沒給對手留下太好機會。喬·佩里擊球不進，田鵬飛順勢打出51分，以104分結束本局比賽。雙方戰至3-3平

第七局，田鵬飛以66:40拿下比賽，以4-3的比分率先拿到賽點。

第八局，田鵬飛以21分領先，隨後喬·佩里打出單桿102分高分贏得本局勝利並再次將比分扳平。

第九局決勝局，喬·佩里以48分領先，轉而進入防守；田鵬飛上手打出51分，由於走位不當，痛失連續進攻殺死比賽的機會。喬·佩里果斷上手擊紅球入底袋得到53分，關鍵時刻，喬·佩里出現致命失誤，擊黃球底袋不進，同樣錯失大好局面。

在接下來二人防守大戰中喬·佩里率先犯錯，留藍球於底袋入口，田鵬飛一擊命中，最終以5:4的總比分絕殺喬·佩里晉級。

•張健康惜敗阿里·卡特

本場比賽二人狀態都不是很好，卡特長台命中率較低，張健康則把握機會能力較差。

前兩局卡特以51、67分連下兩局，接下來張健康把握住機會，以69、67分扳回兩局，比分戰至2-2平。

第五局與第六局，阿里卡特以59、99分再勝2局，拿到賽點。

第七局張健康率先得到63分，場上沒有太多機會，卡特無奈放棄本局比賽。

第八局最後時刻張健康完全有機會拿下比賽，但最終還是技不如人以31-93遺憾告負，以3:5的總比分敗於卡特慘遭淘汰。

•馬克·威廉姆斯逆轉克雷澤·斯特德曼

威廉姆斯與世界排名第74位斯特德曼的比賽在北京時間22:30分準時進行。

雙方在比賽中均表現出極佳的競技狀態，斯特德曼更是打出近期最好的一場比賽。

馬克威廉姆斯以102分，70分贏下第一、二局。斯特德曼以單桿106分在第三局勝出，接下來二人你來我往，比分交替上升，在第6局結束時戰成3-3平。

第七局，威廉姆斯出現低級失誤，送給斯特德曼袋口大禮，斯特德曼笑納後，也同樣出現意外失誤。最後在二人的防守大戰中，斯特德曼79:49勝出，並以4:3的比分率先拿到賽點。

第八局，威廉姆斯並沒有讓對手輕易拿下比賽，憑著良好的大賽經驗，以68:39的比分艱難勝出。

第九局決勝局，馬克·威廉姆斯乘勝追擊，打出單桿73分，以113分取得本局比賽的勝利，以5:4的總比分驚險過關，進入下一輪。
