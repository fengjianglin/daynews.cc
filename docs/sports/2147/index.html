寶貴首秀達成 利物浦年輕球員瓊斯期待更多比賽時間

2019-12-09 0條評論 21次閱讀 0人點贊

摘要

這名18歲的球員在周六利物浦戰勝伯恩茅斯的比賽中首次在頂級聯賽中亮相。利物浦中場柯蒂斯·瓊斯表示，他期待在俱樂部完成他的英超處子秀後，得到更多的上場時間。

這名18歲的球員在周六利物浦戰勝伯恩茅斯的比賽中首次在頂級聯賽中亮相。

利物浦中場柯蒂斯·瓊斯表示，他期待在俱樂部完成他的英超處子秀後，得到更多的上場時間。

這位18歲的球員在周六利物浦3-0戰勝伯恩茅斯的比賽的最後15分鐘上場，這是他第四次代表紅軍出場，此前已經參加了足總杯和聯賽杯的比賽。

這位從9歲起就效力於利物浦的英格蘭19歲以下國家隊球員很榮幸拿到聯賽首秀的機會，他說教練尤爾根·克洛普在上場前給了他所需要的信心。

他告訴利物浦官方網站：「作為一名當地的少年，我想每個人都夢想著能在自己的少年時代效力的俱樂部踢球。」

「能在這麼年輕的年紀就做到這一點是莫大的榮譽，我很感激，我只想說非常感謝主教練和他的團隊對我的信任。」

「每個人看到的比分是完美的，所以我可以上場，並且很輕鬆的去比賽，展示我的能力。」

「我在熱身的時候跟教練說了，他告訴我要上場比賽，保持冷靜，提出自己的風格，這給了我足夠的信心。我去到場上，我只是想踢我自己的比賽。」

「當你進入不同的比賽，和一些沒有經驗的年輕人一起比賽時，總是會有點困難。」

「但是今天我覺得我周圍的其他球員讓這變得更容易了，我期待著更多的比賽。」

這名年輕球員賽後在推特上說，他一直夢想著自己在安菲爾德的第一次頂級聯賽的比賽。

「我從小就夢想著這一天，」他寫道。「難以置信的感覺，今天是我在利物浦的英超處子秀。又是一場大勝，全隊又得3分。感謝所有的簡訊和支持！」

在主場輸給曼聯後，利物浦在積分榜上的領先優勢擴大到11分，現在他們比衛冕冠軍曼城多了14分。
