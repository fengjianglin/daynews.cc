2019年武磊留洋報告出爐：43場8球2助攻2造點，失單刀超兩位數

2019-12-23 629條評論 2,223次閱讀 1008人點贊

⚽️隨著2019年西班牙人隊年度最後一場比賽的結束，武磊闖蕩歐洲整個一年的經歷也暫時告一段落。以中超金靴身份登陸西甲，經過一年來大量比賽的磨練與考驗，武磊2019年留洋報告（成績單）也正式出爐：

⚽️2019年，跨越兩個西甲賽季，武磊從2018-19賽季下半程開始，一直到2019-2020賽季的上半程結束，一年來共為西班牙人隊在正式比賽出場43場，其中28場比賽是首發出場，共打進8個進球，貢獻2次助攻，此外還有2次製造點球的經歷。只看西甲聯賽，武磊在今年共出場31次打進4球助攻1次。數據上看，雖然武磊成為西甲、國王杯和歐聯杯正賽第一位進球的中國球員，但確實與2018年的各項數據明顯縮水。唯一可圈可點的就說西班牙國王杯首秀獨中兩元的表現，但是攻破西乙B球隊萊里達體育，也說明不了太多問題，只有歐聯杯資格賽和正賽的各1個進球還能拿來當點兒小成就。

⚽️2018賽季，武磊代表上海上港出場41場進29球10次助攻，創紀錄的奪得了個人首個中超聯賽冠軍，並且還以27個進球勇奪中超金靴。個人數據方面，武磊在西甲明顯感受到了與中超聯賽的巨大差距，數據對比懸殊。其中他在2018年29場中超聯賽打進27球助攻8次，場均貢獻超過1球；而在2019年31場西甲聯賽只有4個進球和1次助攻，數據對比極其辣眼。從這個角度甚至也反映出中超聯賽恐怕最多相當於西乙水平。

⚽️其中，這43場比賽里，武磊也浪費了諸多進球良機。就比如他的單刀球，代表西班牙人隊各項賽事，一年之內浪費單刀球機會超過兩位數。這其中就包括與皇家馬德里隊比賽時獲得的進球良機，結果在拉莫斯的拚命回防之下，他竟然被嚇得主動打飛單刀。而他在今年西甲的表現也分為驚艷的上半年和糟糕的下半年這兩部分。

⚽️加盟初期，身披24號球衣的武磊確實有些初生牛犢不怕虎的架勢。當時的武磊可以說是帶著肩部鎖骨傷勢帶傷作戰。上半年，他在16場西甲聯賽打進3球，幫助西班牙人隊獲得了歐聯杯資格賽的機會。雖然數據並不太耀眼，場均進球只有0.19球，平均355分鐘才能完成1個進球，但是從發揮來說算是較為出色。下半年，武磊經歷了球隊的多線作戰模式，西甲聯賽15場比賽只進1球，累計出場838分鐘才完成了1個進球。

⚽️今年的西甲聯賽里，武磊一共完成了40腳射門，但是只有4次破門。武磊在西甲場均射門也只有1.3次，表現機會可以說非常之少。相比2018年全年中超95腳射門，武磊各項進攻數據都嚴重縮水。中超場均3.3次射門，西甲場均整整少了2腳射門，破門數量更是縮水7成以上。兩支球隊戰績也是落差懸殊，在上海上港勇奪中超火神杯，而在西甲從年初踢到年末，西班牙人竟然穩居倒數第一。從聯賽冠軍到聯賽副班長，這個差距過於明顯。

⚽️2018年的武磊星光閃爍，打破長期以來中超本土無金靴的怪圈，為上港奪下隊史第一座聯賽冠軍獎盃。而2019年在西甲聯賽，武磊則體驗了一回一隻腳已經掉入西乙聯賽的糟糕經歷。更為糟糕的是，他在西甲的表現至多算是西乙水平。北京國安租借回西乙聯賽的悍將比埃拉，在9月份以來的13場西乙聯賽打進10球助攻2次，這位在中超一度呼風喚雨的強力外援，也就是能在西乙逞強的水平。武磊與之相比，恐怕也只能匹配西乙水準。因此，西班牙人或許只有降級武磊才能迎來出頭之日。

⚽️最後詳細羅列2019年武磊的留洋數據：

-西甲出場31次，22次首發，累計出場時間1910分鐘，打進4個聯賽進球，還有1次助攻，平均477分鐘才能完成1個進球，平均超過半場時間（47分鐘）才能有1次射門機會；

-歐聯杯（含資格賽）一共出場11場，5次首發，累計出場561分鐘，一共7腳射門打進2球助攻1次，平均280分鐘完成一個進球；

-西班牙國王杯出場1次，踢滿90分鐘，6次射門打進2球，也包括1次浪費單刀球的好機會，不然他真有可能在國王杯處子秀上演帽子戲法。

全年，武磊共為西班牙人隊踢過2561分鐘的比賽，43場8球2助攻2造點，失單刀超兩位數，這就是他最終的成績單。
