九冠王是怎麼煉成的，廣東一數據領先聯賽，遼寧還差得遠

2019-12-02 0條評論 23次閱讀 0人點贊

摘要

在經歷了上個賽季的奪冠之後，廣東成為了聯賽歷史上奪得冠軍最多的球隊，也就是人們常說的“九冠王”。雖然說，我國的整體籃球水平還不是很高，跟美國有不小的差距的，但這不是廣東一家獨大的原因。

在經歷了上個賽季的奪冠之後，廣東成為了聯賽歷史上奪得冠軍最多的球隊，也就是人們常說的「九冠王」。雖然說，我國的整體籃球水平還不是很高，跟美國有不小的差距的，但這不是廣東一家獨大的原因。送走了八一，迎來了新疆跟北京，強隊更迭，始終站在山頂上的，只有一個廣東，他們是怎麼做到二十年如一的強大？其實核心還是在於球員，比賽是由球員來打的，球員的實力也很大程度上決定了比賽的走向。其實不難發現，廣東一直都有強力國手，這是為什麼呢？

我國有一句話叫做「未雨綢繆」，在事情還沒有到最壞的地步，就要做好應對的打算。廣東就是這麼做的，他們向來都很注重對於新人的培養，這也是為什麼，在王7跟朱8等人退役了之後，廣東立馬就湧現出了趙睿、二飛等國手的原因，下面就用粵疆遼三個強隊對於新人的培養來分析一下，為什麼廣東會這麼強大，為什麼他們會是九冠王。

目前來看，廣東是聯賽中打法最先進的球隊，他們開始嘗試打小球，而且注重防守反擊，本賽季也是高居積分榜的榜首。他們也很注重對於新人的培養，上個賽季就嘗到了新人的甜頭，這個賽季更要重視，本賽季，廣東一共有14人出場，其中7人都是一二年級的新人，其中徐傑、杜潤旺、王薪凱三人的出場時間都達到了100分鐘以上，曾樊日本賽季沒得到重用，但也有80分鐘的出場時間，還有萬聖偉也得到 22分鐘的表現機會，那次絕殺大家應該都記得。

新疆本賽季的戰績也很不錯，周琦跟西熱的回歸讓他們實力大漲，大有叫板廣東的意思。從陣容來看，新疆擁有更多的國手和實力球員，除了外援之外，球隊中多人都有國籃經歷，比如說今年剛剛打了世界盃的周琦跟阿不都，還有最佳觀眾席看比賽的可蘭，以及實力不減的西熱。在球隊已經擁有如此強悍即戰力的情況下，新疆還是堅持給新人更多鍛煉的機會，唐才育是目前球隊最重視的年輕球員，他得到了147分鐘的上場時間，此外還有4名年輕球員得到了上場機會。

遼寧呢？遼寧為止還是處於磨合期，球隊的戰績也不是很理想。可能就是因為目前球隊的戰績不穩定，球員之間的磨合也沒有做好，所以球隊中的新人幾乎，沒有什麼上場的機會。截止到目前，遼寧隊中一共有三個年輕球員得到了上場的機會，其中除了郭旭上場27分鐘外，其他二人都是2分鐘，可以說，球隊中的新人幾乎沒有用武之地。

從年輕球員的出場時間中可以看出，在三支強隊中，廣東是最重視培養新人的，他們對於新人的信任也換來了更多優秀球員的出現，這樣是他們為什麼能夠在歷史的浪潮中一直站到現在。而遼寧恰恰相反，他們過於依賴球隊的主力，新人得不到鍛煉機會，而且球隊的戰績也不是那麼理想，有些方面，遼寧確實是應該跟廣東好好學學。
