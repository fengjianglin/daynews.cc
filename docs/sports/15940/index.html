馬龍林高遠相繼挑戰鷹眼失敗，看馬龍賽後如何評價鷹眼系統加入

2019-12-14 2,677條評論 3,688次閱讀 1623人點贊

2019年國際乒聯總決賽在鄭州開賽，總決賽是聚集了全年最優秀的運動員。國際乒聯決定在總決賽上使用「鷹眼」系統，這也是為了在明天奧運會及其他大賽上正式使用做好準備。

鷹眼技術可以對擦邊球、擦網球、發球等做出高清回放和動畫演示，對主裁的判定不滿的選手也可以提出回看錄像，通過回放進行判定。一場比賽，選手有2次挑戰的權利，如果挑戰成功則次數不減少。

通過兩天的比賽，國乒隊員馬龍林高遠都各挑戰了一次鷹眼，林高遠是在和梁靖昆雙打比賽中被判發球回拋違例，通過鷹眼數據回放，林高遠的發球斜拋過大，裁判判罰沒有任何問題，林高遠挑戰失敗。（下圖）

在今天下午進行的最後一場男單八分之一決賽中，馬龍對戰鄭榮植，在比賽過程中馬龍和鄭榮植的發球先後被裁判口頭警告一次，馬龍在第二次高拋發球時被直接判罰失分，隨後馬龍也向裁判提出挑戰鷹眼，通過鷹眼回放，馬龍發球在球下落過程中頭部有遮擋，馬龍鷹眼挑戰失敗。但馬龍沒有受到干擾，最終4:2拿下比賽，晉級8強。（下圖，附發球規則）

鷹眼除了對發球階段進行高清捕捉外，擦邊球擦網球也可以實現精準判斷，雖然現在還沒有運動員提出挑戰這方面，昨天王藝迪對戰平野美宇的比賽中，在回放中對平野美宇的擦邊球也是採用鷹眼模擬動畫回放，清晰的給出了擦邊的情況。（下圖）

對於鷹眼的使用馬龍在賽後採訪也表示：「第一局裁判就給了個發球警告，認為我身體或者頭對發球有遮擋，第二局也警告了鄭榮植，其實我估計鄭榮植也不明白為什麼，第三局裁判判罰我的時候，我就想用鷹眼看一下到底是因為什麼。馬龍認為鷹眼系統對比賽的公平公正方面的幫助非常大，也非常有必要。」
確實鷹眼的使用提高了判罰的準確性，但也對運動員的動作規範提出了新的要求，相信在鷹眼的介入下，乒乓球運動會向更好的方向發展，一些誤判會大大的減少，比賽的公開公正性會更強。
乒乓網-白開水
