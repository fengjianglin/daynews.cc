克洛澤接班人還在等待拜仁的召喚，其經紀人已經與利物浦開始接觸

天天要聞
2019-12-21

多哈時間12月20日下午，利物浦主帥克洛普出席了世俱杯決賽賽前發布會。克洛普表示：「我們想要贏得世俱杯的冠軍，這個大家都知道這很困難，因為對手都很優秀。而且對手是我們不怎麼熟悉的其他洲強隊。」接著渣叔談到了歐洲足壇和世界其他地方對這項賽事的看法有所不同，但他希望自己能讓利物浦在這項賽事中的表現，稍微改變一下這些人的看法。克洛普對此表示：「從我們來到多哈之後，得到的感受是不同的。我們看到了蒙特雷的戰鬥精神以及他們對比賽的投入程度。明天對陣弗拉門戈的比賽也是如此，這會是一場艱苦的比賽，他們也許更具實力。」 對於德國同胞維爾納與利物浦的「緋聞」，克洛普則表示：「利物浦現在不思考這些東西。」

bild的記者透露，克洛普對克洛澤接班人維爾納十分賞識，他曾在今年夏天就希望將維爾納帶入安菲爾德。 經過漫長的追逐，這樁轉會似乎已經陷入僵局。有消息稱，早在去年夏天的時候，記者就爆料了維爾納與萊比錫紅牛的合同中有一條3000萬的解約條款，但實際此解約條款只針對拜仁慕尼黑有效——維爾納也希望自己去拜仁。雖然說目前利物浦和萊比錫的溝通相對順暢，但維爾納的經紀人與拜仁關係緊密——拜仁仍舊沒有放棄這位德國本土射手。bild記者透露，克洛澤接班人還在等待拜仁的召喚，其經紀人已經與利物浦開始接觸。對於球員本人的態度，德國記者表示多加英超俱樂部已經退出競爭轉而去爭搶多特蒙德邊鋒桑喬。

可以說維爾納到底去哪兒，完全取決於本人想法，筆者認為他可能其為了加盟拜仁再等待一個賽季的時間。筆者還看到德語當地的一些討論，有人提到維爾納的經紀人的問題——和拜仁關係緊密。目前實質上和維爾納傳過的就只有利物浦，球迷們也知道這小小希。維爾納親口傳說的Es gibt nur einen Verein，其實就明示了自己轉會的話就只有一傢俱樂部，不言自明。所以現在爆這個消息，可能是經紀人跟拜仁這邊以及利物浦都有一頓瞎操作。 但具體的事情進展，今年利物浦在德國的消息頗多（不同於以往）。包括克洛普這位德國主帥率領球隊打世俱杯的消息，德國記者都很少報道，不知道是俱樂部明修暗道，暗度陳倉還是真的準備在德國發力？

拜仁對德國球員的吸引力非常大，這是重要原因。一是德國經濟極其強勢，德國國內的足球市場就很大，去了拜仁能獲得極高的曝光度和贊助合同——這些大家都知道。 所以德國年輕球員不需要去別的國家闖蕩，甚至去了別的國家闖蕩，都不一定能有在拜仁獲得的認同和隱性價值高（典型是克羅斯）。二是拜仁的名宿及高層幾乎控制德國足壇，上次拜仁和德國足協因為誰是一號門將吵起來了，拜仁直接給足協放話：如果諾伊爾不首發就所有拜仁球員不去國家隊——這種影響力幾乎歐洲獨一無二。三是德國人的民族自豪感認同感很少有人出去闖蕩，德國球迷都知道，拜仁這個球隊跟拜仁州聯繫緊密，拜仁州德國經濟最強，你看幾乎所有德國超大型公司都搶著去贊助，這是事實。
