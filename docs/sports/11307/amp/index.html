丹麥童話傳奇：彼得大帝舒梅切爾

天天要聞
2019-12-10

說到九十年代的最佳門將人們很難忽略丹麥人彼得•舒梅切爾！

在整個九十年代優秀門將有很多，像范德薩、帕柳卡、羅西、卡恩等等，帕柳卡、范德薩成熟而穩重，卡恩霸氣十足極富感染力，但是若論綜合實力舒梅切爾才是最好的那個。

身高臂長的丹麥人反應神速、擅撲單刀球，看他撲單刀簡直就是一種享受，曾有過連續幾次反覆撲救單刀的壯舉！這在其他出色門將那是很難做到的。

舒梅切爾作為豪門曼聯的主力門將榮譽無數，聯賽冠軍、杯賽冠軍拿了無數，這對於一個在頂級豪門效力的球員來說稱不上有多大難度。帶領並不算歐洲足壇頂級力量的丹麥國家隊奪取歐洲杯冠軍卻稱得上是一個不大不小的奇蹟！

近些年來歐洲杯非傳統勢力奪冠只有兩次，一次是丹麥，一次是神話發源地希臘。希臘奪冠那年除了雷哈格爾的出色調教外與球隊眾志成城是密不可分的，可以說希臘奪冠是集體通力合作的原因。而丹麥能奪取冠軍與隊內幾個大牌球員的發揮密不可分，舒梅切爾作為防線最後一環猶如定海神針般的發揮起了非常大的作用！

舒梅切爾的職業生涯起步于丹麥，在加盟丹麥豪門布隆德比後他幫助球隊連續三次奪取聯賽冠軍，是俱樂部史上最佳門將之一。

1991年慧眼識才的曼聯主帥弗格森爵士僅僅用五十萬英鎊就簽下了舒梅切爾。從此舒梅切爾開啟了一段傳奇之路！

92-93賽季英格蘭頂級聯賽改制，曼聯奪得了第一屆英超聯賽的冠軍，舒梅切爾的高接抵擋功不可沒。

93-94賽季舒梅切爾與球隊再接再厲獲得了英格蘭史上第一個雙冠王！

94-95賽季雖然曼聯沒有獲得冠軍，但舒梅切爾卻個人創造了主場不失球時間記錄。

98-99那個跌宕起伏的賽季、那個令人窒息的冠軍杯決賽早已深深印進球迷腦海！足總杯史上最佳進球、歐冠史上最經典的大翻盤都產生在那個賽季，創造這一值得銘記時刻的就是舒梅切爾和他的曼聯隊友！在隊友進球後舒梅切爾連續的空翻慶祝動作也被鏡頭記錄了下來成為經典。

作為一個門將舒梅切爾不僅防守出色，進攻能力也不遑多讓！在一場歐洲聯盟杯對陣羅斯托夫的比賽中，舒梅切爾就曾沖入禁區用一記強有力的頭槌幫助曼聯扳平比分！

彷彿是為了證明自己的進球並非偶然，在代表國家隊與比利時的比賽中舒梅切爾又打入一球。職業生涯兩粒進球，一粒為曼聯打入，一粒為丹麥打入，都是舒梅切爾生平最重要的兩支球隊。

提到舒梅切爾的職業生涯除了曼聯之外就不得不提那偉大的歐洲杯奪冠路程。並不是歐洲足球主流的丹麥隊在92歐洲杯的路程是非常吃力的。當他們碰到英格蘭時，人們就想丹麥完了，肯定會輸給英格蘭！哪知道那場比賽舒梅切爾雖然忙得不可開交但幸運的是他沒有在球門裡面撿球。

歐洲杯豪強林立，在丹麥與法國隊相遇之前他們的處境是極其糟糕的！小組賽前兩戰一平一負，想要出線勢必登天！當時的法國隊可是擁有坎通納、帕番等超級球星。丹麥隊的教練卻信心十足，因為他知道在球隊的大門前有一尊大神存在，他能將所有來球都拒之於門外，他就是舒梅切爾。那場比賽正如教練所期盼的那樣，舒梅切爾高接抵擋力保城門不失，隊友打開了法國隊的大門，丹麥強勢晉級淘汰賽！

九十年代初在足球歷史上大名鼎鼎的「荷蘭三劍客」和「德國三駕馬車」正處於巔峰期，那支球隊也不願與他們相遇！然而上天卻與舒梅切爾和他的丹麥隊開了個玩笑，在歐洲杯半決賽和決賽中丹麥先後碰到了荷蘭和德國！

荷蘭一直是全攻全守的代表，他們進攻如摧古拉朽般強勢，退守也是相當迅速。那時的荷蘭除了三劍客外還有一個極具天賦的攻擊手——博格坎普！儘管比賽中丹麥丟失了兩球但在荷蘭如潮攻勢下舒梅切爾撲出去的球遠比兩粒失球多！沒有被荷蘭拉開比分丹麥隊終於把握住機會扳平了比分。此後的比賽又是靠著舒梅切爾的神勇發揮將比賽拖入了點球決戰！

與荷蘭隊的點球決戰成就了舒梅切爾，他用經典一撲淘汰了荷蘭隊！有得意者必定有失意者，那個被舒梅切爾撲出點球的球員就是大名鼎鼎的范•巴斯滕！

雖然殺入決賽，丹麥依舊不被看好，因為他們面對的對手是世界足壇最成功的球隊之一——以鐵血著稱的德國隊！

在「荷蘭三劍客」風行的年代，「德國三駕馬車」是唯一能與他們抗衡的組合。已經領教過「荷蘭三劍客」威力的丹麥深知這種頂級組合的威力！老實說丹麥能挺過荷蘭這一關，運氣也佔據了相當大的成分。接連與「荷蘭三劍客」、「德國三駕馬車」相遇，丹麥的運氣真是背到家了！

屋漏偏逢連夜雨！丹麥的傷病問題太嚴重了！幾乎排不出一套主力陣容的他們還是要硬著頭皮去踢與德國隊的比賽。丹麥幾乎所有人都沒有信心戰勝德國，只有舒梅切爾表示他有信心阻止當時最好的射手之一克林斯曼進球！

與「荷蘭三劍客」有博格坎普輔佐一樣，「德國三駕馬車」身邊也有哈斯勒這樣球技高超的好幫手！但是這屆杯賽上天註定要讓丹麥神奇到底！任憑德國怎麼折騰就攻不破丹麥的大門，最終被丹麥攻破城池的德國隊將冠軍拱手相讓。

舒梅切爾的隊友回憶這場比賽時說道：德國隊的攻勢如潮，他們一直在進攻，舒梅切爾一直在撲救。

正如舒梅切爾的隊友所言，92歐洲杯丹麥之所以能奪取最後的桂冠大半功勞是屬於舒梅切爾的！這個與弗格森爵士一樣有著紅鼻頭的男人依靠的他高大的身軀和敏捷的反應幾乎以一己之力扛著球隊前進！在足球史上以一己之力帶領球隊前進的有很多，但以門將位置帶領球隊前進的卻只有兩個，一個是前蘇聯的雅欣，另一個就是被稱為「彼得大帝」的舒梅切爾！

提起優秀門將近年來大家都會想到卡恩、布馮、卡西利亞斯等人，但是與舒梅切爾相比他們無論是領袖氣質還是挽大廈於既倒的能力都還有一定的差距！舒梅切爾無愧於近三十年來的最佳門將。

更令舒梅切爾欣慰的是他的兒子小舒梅切爾也是一個出色的門將。小舒梅切爾甚至帶領萊切斯特城在曼聯、利物浦、阿森納等一眾豪強手中搶走聯賽桂冠！頗有乃父之風。
