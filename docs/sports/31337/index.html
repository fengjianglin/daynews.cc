第18輪 | 年度收官戰各有悲喜，巴薩主場全年不敗獨佔榜首

2019-12-24 455條評論 2,038次閱讀 1042人點贊

國家德比戰平皇馬兩天後，繼續坐鎮主場面對阿拉維斯的巴薩「重拳出擊」，梅西打入一記世界波解鎖本年度50球成就，巴薩達成諾坎普一年正式比賽保持不敗成績，以2分優勢壓過皇馬領跑積分榜。壓軸出戰的皇馬在伯納烏運勢欠佳，0-0遺憾戰平畢爾巴鄂競技，以三連平結束了2019全部比賽。

在一場保級關鍵戰中，萊加內斯主場2球力擒西班牙人，將鸚鵡軍團釘在倒數第一的位置上。

2019歲末的積分榜上，巴薩暫時壓過皇馬兩分位居榜首，塞維利亞和馬競位居三四位，身後還有緊追不捨的皇家社會，赫塔菲等，至此2019年的西甲聯賽全部結束，2020年精彩將繼續上演！

埃瓦爾3-0格拉納達

坐鎮主場迎戰格拉納達的埃瓦爾此前遭遇5輪不勝，但在2019年的收官戰役中，他們3-0大勝本賽季表現不錯的格拉納達，日本球星乾貴士下半場打入一球鎖定勝局，他也斬獲了自己本賽季首個進球。

馬略卡0-2塞維利亞

做客馬略卡主場的塞維利亞憑藉巴內加的一傳一射，以2-0戰勝對手，穩居積分榜第三的位置，巴內加助攻迭戈-卡洛斯破門，下半場利用點球再下一城，成為塞維利亞取勝的關鍵。

巴塞羅那4-1阿拉維斯

巴薩迎來2019的收官戰，最終領頭羊以4-1的比分大勝阿拉維斯為這一年收官，並在2019年保持主場不敗，MSG全部取得進球，蘇亞雷斯上演助攻帽子戲法，梅西打入一記精彩遠射讓他的年度進球數達到50個。

比利亞雷亞爾1-0赫塔菲

黃色潛水艇坐鎮主場迎來近來勢頭極佳的赫塔菲挑戰，這之前赫塔菲7輪西甲收穫5勝2平的絕佳戰績，但本場莫伊-戈麥斯的絕殺球幫助潛水艇送給赫塔菲一場失利。

巴拉多利德1-1瓦倫西亞

頭號射手羅德里戈傷退，第83分鐘被巴拉多利德率先破門，但陷入絕境的瓦倫西亞這一次上演起死回生奇蹟，小將馬努-巴列霍的壓哨絕平幫助蝙蝠軍從巴拉多利德主場帶走一分。

萊加內斯2-0西班牙人

積分榜最後兩位的球隊萊加內斯與西班牙人在黃瓜軍團主場狹路相逢，最終憑著鋒線雙槍布萊斯維特和恩內斯里的各一粒進球，萊加內斯主場2-0擊敗西班牙人拿下保級關鍵戰，武磊下半場比賽替補出場，一次前插一度造成點球，但因越位在先被取消。

奧薩蘇納3-4皇家社會

奧薩蘇納主場迎戰皇家社會的比賽一波三折，厄德高的出色發揮讓皇社客場一度3球領先，但頑強的奧薩蘇納連扳兩球，隨後兩隊再各入一球，全場比分最終定格在3-4，皇家社會客場驚險擊敗對手。

皇家貝蒂斯1-2馬德里競技

貝蒂斯主場迎戰馬競，上半場兩隊各命中一次對方門柱，下半場替補出場的科雷亞成為馬競奇兵，奉獻一傳一射，最終幫助床單軍團客場拿下貝蒂斯，進軍積分榜前四。

萊萬特3-1塞爾塔

萊萬特主場迎戰降級區內的塞爾塔，塞爾塔頭號射手阿斯帕斯為球隊打破僵局，但下半場萊萬特迎來爆發，連入三球逆轉取勝，以積分榜第九位的好成績結束了這一年的比賽。

皇家馬德里0-0畢爾巴鄂競技

伯納烏球場迎來本賽季西甲的最後一場戰役，皇馬主場迎戰畢爾巴鄂競技，最終經過90分鐘的鏖戰，運氣不佳的皇馬3度擊中球門門框，只得以0-0與畢爾巴鄂競技握手言和，結束了2019年最後一場西甲比賽。
