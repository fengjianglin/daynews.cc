泰拳場上最暴烈的鐵人，卻有一顆最謙卑的心

2019-12-24 40條評論 2,058次閱讀 1110人點贊

每一個看過羅唐比賽的人，都會被他暴烈的風格所吸引。黝黑精壯的體格，永不知疲倦的進攻，榴彈般兇狠的拳頭，鐵斧一樣的低掃腿，羅唐將自己的身體化為人體兵器，用瘋狂的進攻摧毀對手的肉體，也碾壓他們的鬥志。羅唐的綽號是”鐵人”，而這個名字在泰文中意思是”坦克”，無堅不摧，無人可擋。

ONE 冠軍賽泰拳蠅量級世界冠軍喬納森·哈格蒂曾在比賽中以勇猛著稱，可遇到羅唐時卻在他的”炮火”轟炸下節節敗退，靠著意志力苦苦支撐，才沒有被擊倒，用一場點數敗北勉強維持了前任冠軍的最後一絲體面。但對22歲的羅唐來說，拿到世界冠軍並不是他的終極目標。”衛冕冠軍金腰帶比奪得冠軍要更加困難。”羅唐從未被勝利和成就沖昏頭腦。他知道保持謙卑才是在賽場立足的方式。只有這樣，自己才會不斷進步。

“自負沒有任何好處，如果你自負，就會慢慢遠離成功。對於泰拳手來說，每一場比賽都是殘酷的。如果你出名了，就更不能忘記你是誰，你來自哪裡，永遠不要忘記你走進拳館的第一天，不要忘記對泰拳的熱情。”

羅唐已經有了兩屆Omnoi拳場的世界冠軍、ONE 冠軍賽泰拳蠅量級世界冠軍，職業戰績259勝41負1平，但他知道自己距離真正的成功還有很長的路要走。有很多30多歲的傳奇泰拳手依舊在奮鬥，是他們創下了泰拳享譽世界的名聲和輝煌。”我是年輕一代的拳手，必須尊重泰拳前輩。教練不會告訴你這些，但作為拳手，這是我自我覺醒的一部分，泰拳手的精神代代相傳。今天的榮譽來之不易，我會用每一場比賽的機會來捍衛它。”

羅唐並沒有因為自己的巨星地位和成功而改變，他依舊對泰拳充滿熱情，並願意為之付出一切。”我不會幻想自己不可戰勝，我會努力訓練，爭取打好每一場比賽。拳賽有勝利也有失敗，並不是每場比賽我都能獲勝，但我希望盡自己最大的努力。泰拳是一項運動，它令我們學習和成長。它教會我們如何爭取勝利，也教會我們如何對待失敗，並從失敗中學習教訓。”

羅唐成長在一個貧困的家庭，為了養活9個兄弟姐妹，他甚至去撿垃圾賣錢。儘管他現在已不用在垃圾堆里工作，可他永遠不會忘記生活教會自己的東西。貧窮給了羅唐更多的動力，他在拳館每一天的訓練都是為了保證家人再也不要回到以前的日子。他在生活中磨練出鬥志，並將這種鬥志運用在賽場上。”我們必須時刻提醒自己，當獲得成績時，不要迷失自我，當在擂台上一敗塗地時，不要氣餒。謙卑不會直接讓你的拳頭更有力量，但它是我們作為拳手的一部分。這是我們必須擁有的東西，只有謙卑的人才會不斷進步。自滿的人都會付出代價！”

10月13日，”ONE 冠軍賽：世紀”將在東京兩國國技館擂響戰鼓，羅唐將在這場歷史性賽事中衛冕自己的泰拳蠅量級世界冠軍金腰帶，對手是巴西拳王沃爾特·貢薩爾維斯 （Walter Goncalves），屆時他將會展現謙卑的力量，證明自己才是真正的冠軍。
