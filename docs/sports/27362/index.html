蘭茲博格砍40+13吳前23+7 浙江客場19分大勝北京

2019-12-21 1,530條評論 3,169次閱讀 308人點贊

北京時間12月20日，CBA常規賽，北京主場以93-112不敵浙江，這也爆出冷門。

北京隊尤度24分10籃板6蓋帽，林書豪21分5籃板4助攻，方碩18分4助攻；浙江隊蘭茲博格40分13籃板，吳前23分7助攻，朱旭航20分5籃板4助攻。

北京以13勝5負的戰績排在第3位，此番主場面對浙江亦是要全力爭勝，鞏固自身的排名；浙江以9勝8負排在第11位，此番客場挑戰北京困難重重。雙方過往交鋒紀錄，北京以31勝21負佔據優勢，其中主場以19勝7負明顯佔據上風。

首節比賽，朱旭航接連上籃打進，吳前和蘭茲博格皆命中三分，程帥澎籃下強攻打進，浙江以16-5佔得先機。蘭茲博格外線又進，朱旭航拉杆上籃打進，鄧蒙遠投命中，張大宇半截籃打進，浙江將比分進一步拉開到32-13。尤度籃下貢獻暴扣，想要助北京隊扭轉頹勢。該節戰罷，浙江以32-17領先。

次節比賽，蘭茲博格和張大宇上籃打進，助浙江仍然保持著優勢。尤度的籃下進攻打得有聲有色，林書豪成功上反籃得手，北京將比分追成30-37。吳前籃下強攻打成2+1，朱旭航上籃打進，隨後他外線又遠投命中，張大宇籃下強打製造2+1，吳前跳投打進，助浙江又將場上比分拉開。半場戰罷，浙江以58-39領先。

第三節比賽，張大宇中投打進，蘭茲博格快攻貢獻暴扣，吳前遠投命中，蘭茲博格強吃翟曉川又打成2+1，浙江隊以72-41一舉取得超過30分的巨大優勢！朱旭航和蘭茲博格又連中三分球，浙江隊在場上的優勢也變得愈發穩固。方碩底角命中三分球，尤度快攻打成，助北京追擊。該節戰罷，浙江以87-61領先。

末節比賽，吳前遠投命中，賴俊豪貢獻暴扣，仍助浙江隊搶分。尤度和翟曉川相繼衝擊籃下打成，林書豪拉杆上籃打進，但仍然難以助北京有效的迫近分差。吳前遠投命中，朱旭航拋投將球打進，蘭茲博格快攻上籃打進，這也令比賽再無懸念，浙江以112-93勝出。

首發陣容

北京：林書豪，方碩，張卓，翟曉川，加尼尤

浙江：吳前，程帥澎，蘭茲博格，朱旭航，張大宇
