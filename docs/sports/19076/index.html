中國女排三大新星橫空出世！天賦實力超強，一人被稱為“新朱婷”

2019-12-17 1,798條評論 3,931次閱讀 721人點贊

眾所周知，現如今的中國女排實力非常的強。對於其他國家來說，中國女排的存在是一個巨大的威脅。其中運動員朱婷、張常寧、袁心玥等幾位主力成員在賽場上的表現非常的出色，為國家贏得了多項的榮譽。現如今排球賽場上，出現了越來越多的新面容，尤其河南女排的三個年輕運動員天賦實力超強，被球迷稱為中國女排三大新星。

「新朱婷」的誕生

在最近的排球比賽上，作為新人的毛曉靜、翟嗣棋和韓雯靜在賽場的表現非常出色，受到球迷們的關注。這三位球員對於大家來說非常的陌生，首先是自由人的毛雅靜，身高一米七八。比賽上的毛雅靜動作靈敏，並且與隊員的配合非常的好，已經位於一傳榜單上的第二位，而第一位則是中國女排王夢潔。

其次是毛雅靜的隊友韓雯靜，身高一米9，並且身體素質非常的好，是打主攻的位置。韓雯靜的實力不容小覷，在賽場上的她身上有一股狠勁，扣球迅猛並且殺傷力十足，讓對方招架不住。韓雯靜在比賽上的表現讓不少球迷直呼：看到朱婷的身影，因此韓雯靜被球迷們稱為「新朱婷」。不過現在毛雅靜年齡才21歲，身上的潛力還沒有完全被發掘，前途一片光明。

注入新能量

韓雯靜的隊友翟嗣棋也是打主攻位置，實力稍微比韓雯靜弱一點，但總體實力是相當的不錯，在一傳位置上表現優秀。翟嗣棋可以說是一個全面性的運動員，善於在場上冷靜思考，與隊友相互配合。這三個球員的出現讓不少球迷感到又驚又喜，驚的是著三個運動員的實力超出了球迷們的想像，喜的是中國女排又注入了新的能量。

這三個球員都來自河南女排，可以說河南女排非常的重視她們，作為河南女排的教練焦帥向大家說道：年輕缺乏經驗是我們的短板，但年輕運動員可塑性非常的強。只要多參加比賽，經驗和實力也會大大的提高的。很多球迷對河南女排非常的看好，雖然其他省女排請來了不少大將，比如天津女排請來的朱婷、廣東恆大請來的林莉等等。這些球隊的實力非常的強，對於河南女排來說是一個很大的挑戰，希望河南女排能夠在比賽上取得不錯成績！
