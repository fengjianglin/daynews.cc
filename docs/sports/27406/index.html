單憑一神技就帶隊勇奪歐戰冠軍，這人比馬拉多納還要橫

2019-12-20 1,473條評論 4,111次閱讀 2038人點贊

作為11人上場作戰的團體運動，梅羅二人證明了1人帶隊贏球還能偶而發生，但能夠在杯賽賽場上以一己之力帶隊奪冠的，除了1984年歐洲杯的普萊蒂尼以及1986年世界盃的馬拉多納，或許真沒幾個傳奇能被寫入歷史。而事實上，就在本世紀初，一個不那麼出名的荷蘭前鋒也曾完成過這樣的壯舉，而這個人，名叫皮埃爾-范霍伊東克。

夢回2001年夏天，人們會興嘆弗格森與他的曼聯完成三連冠霸業，也會驚訝那幾筆改變足壇格局的世紀轉會，但在那個時期，足壇的風向似乎已逐漸與荷蘭足球脫離了關係，在范加爾和阿賈克斯的歐冠奇蹟以及荷蘭在法國世界盃的敗走後，沒人會認為荷蘭足球能成為當時足壇的焦點。

然而就在接下來的01/02賽季，范霍伊東克以及他的東家費耶諾德低調地搶去了歐洲足壇的聚光燈。那個賽季，費耶諾德成為了那幾年間首支闖入歐冠正賽的荷甲球隊，但雖如此，他們仍未能在歐冠舞台帶來驚喜，帶著1勝2平3負的戰績小組出局，轉而投身歐洲聯盟杯的賽場。而在這座相對低調的歐戰舞台，范霍伊東克卻成為了球隊的英雄。

那時候的費耶諾德，至今能讓大部分球迷念得出名字的或許只有范佩西、卡盧以及日本國籍小野伸二，而在那時候，前二者也只是初出茅廬的小伙，因此在很多時候，32歲的范霍伊東克依然是球隊場上的關鍵人物。

在歐盟杯32強賽，費耶諾德首先遇到了來自德國的對手弗賴堡，儘管首回合他們主場取得1-0的領先，然而次回合50分鐘過後他們卻處於0-2落後的危機當中，必須爭取在剩餘時間取得進球。而就在關鍵時刻，范霍伊東克帶著他的自由球絕技站了出來，在大禁區角邊上一記石破天驚的弧線球射門直掛死角，幫助球隊扭轉局面，最終也以3-2的大比分逆轉晉級。

而范霍伊東克的英雄戲份，也僅僅就此拉開了帷幕。在接下來的16強賽，面對蘇超勁旅格拉斯哥流浪者，又是次回合1-2落後的情況之下，范霍伊東克在上半場比賽的尾段連續祭出自由球必殺技，兩粒複製粘貼般的世界波，瞬間助費耶諾德逆轉了比分，最終也以4-3的總比分繼續驚險晉級。

而相似的劇本也延續到了1/4決賽，面對同國死敵的埃因霍溫，次回合又是1-2的總比分落後，在全場比賽的補時階段，他接應隊友一記高吊球傳中，力壓身前後衛乖巧攻門，皮球划出一記美妙弧線精準飛入網窩，為球隊驚險續命。而在最終的點球大戰他更是罰進了最後一粒制勝點球，將費耶諾德送到了四強。

半決賽上，費耶諾德遇上了羅納爾多、西多夫等巨星領銜的意甲豪門國際米蘭，而此時范霍伊東克和他的球隊似乎早已無所畏懼。首回合1-0贏球後，次回合在聖西羅球場，范霍伊東克開場不久就接范佩西的傳中用一粒頭槌攻破對手大門，隨後隊友托馬森再進一球，3-0的優勢也成功抵住了國米最後時刻的反撲，最終他們以3-2的比分神奇挺進決賽。

而到了決賽的舞台，對陣同樣眾星雲集的多特蒙德，此時費耶諾德距離冠軍只差一步，而在這場比賽當中，范霍伊東克繼續著他大場面先生的神奇戲碼。當上半場比賽進行至最後階段，托馬森製造點球，此時范霍伊東克頂住壓力一蹴而就，為荷甲勁旅率先取得領先。

而僅僅在幾分鐘過後，范霍伊東克在那個熟悉的位置，用他最熟悉的腳法，讓皮球划出一道熟悉的弧線，又一次攻破萊曼把守的射門，為費耶諾德半場確立了2-0的領先。

儘管下半場多特展開了瘋狂的反擊，但勝利註定屬於范霍伊東克和他的球隊，最終費耶諾德3-2擊敗多特，在幾乎沒人看好的情況下爆冷捧起了那個賽季的聯盟杯冠軍。

就這樣，32歲的范霍伊東克在生涯臨近遲暮之時登上了巔峰，肩負著祖國球迷的夢想，幾乎以一己之力率隊贏得了個人生涯最高的榮譽。儘管在長達18年的職業生涯當中，范霍伊東克只在費耶諾德留下了短短的幾年光陰，但在生涯最後一刻選擇重返故地結束綠茵之旅，可見他對這家球會的熱愛。

在鬱金香鋒線天團中，他並非最頂尖的一個，但當年的這場神奇歐戰冒險，足以讓他成為在荷蘭足球的歷史當中留下光輝一筆。
