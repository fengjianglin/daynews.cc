專訪申花女足“隊花”熊熙：希望大家支持女足，目前還沒有男朋友

2019-12-22 78條評論 1,093次閱讀 89人點贊

近日，來自申花上體女足的球員熊熙在接受媒體採訪時表示，「火」了之後有球迷關注的感覺非常好，希望大家多支持女足的發展。

你本人現在在微博上很火，你覺得是什麼原因呢？

沒有，沒有很火。就是之前因為全運會那個絕殺球嘛，突然一下子就火了。然後最近因為加入了申花這個大家庭，又火了一波(羞澀的笑了起來)。

火了以後感覺怎麼樣？

就是有更多球迷的支持，這種感覺特別好。因為以往我的比賽都沒有球迷支持，現在加入申花以後，有很多球迷可能因為申花的原因都來支持我們，感覺很棒。

平常玩遊戲嘛？

平常不怎麼玩。

遊戲圈裡的「農藥」是什麼意思？

這個我不知道誒，我不玩遊戲(又一次害羞的笑)。

平常自拍會修圖嘛？

就直接用那種自拍的軟體，直接拍完也不用修那種。

平時比賽風吹日晒，會怎麼保養自己的皮膚？

就是抹防晒霜嘛，很多人一般會抹一層，我們出去訓練可能會抹兩層三層這樣子。

球場外有沒有時間打扮自己，穿一些自己喜歡的衣服？

可能就周末吧，周一到周五如果沒有訓練的話，我們還是要上課的。

聽說你喜歡易烊千璽？

是的是的，最近看了《少年的你》，感覺演得特別好，非常非常不錯，男朋友什麼都好(嬌羞的笑容)。

現在有男朋友嘛？

還沒有，先把球踢好吧。

想和球迷說點什麼？

希望大家能支持我們，支持女足的發展，謝謝大家啦。
