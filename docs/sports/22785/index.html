中超老熟人回歸！重慶官宣張外龍二進宮 曾率隊追平中超最佳排名

2019-12-18 20條評論 3,879次閱讀 2279人點贊

北京時間12月18日，在重慶斯威宣布不與小克魯伊夫續約後，官宣與韓國教練張外龍再度牽手。2016和2017賽季，張外龍就在重慶執教，並且2016賽季率隊追平了隊史最佳的第8排名。作為中超老熟人，除了重慶之外，張外龍還執教過青島、大連、建業等多支球隊。

張外龍於中超以及重慶斯威而言並不陌生，其曾在2011年年初出任青島中能主教練，該年年底出任大連阿爾濱主教練，次年4月辭職，之後又重回青島執教，並率領球隊順利保級。2013賽季繼續執掌球隊，意外遭遇解約，該年青島中能隊不幸降級。張外龍在青島時期的七秒防反快攻理論十分實用，也一度成為他在外界的代名詞。

此後不久，張外龍與中超球隊再續前緣，2015年年底，2015年底，張外龍接替王寶山成為重慶力帆主教練，簽約年薪約為50萬美元一年，並帶隊征戰了2016、2017兩個賽季，其中2016賽季帶隊位列中超第8，這也是重慶隊史的並列最佳排名。2017賽季結束後，張外龍合約到期，重慶當代力帆沒有選擇與他續約，這名韓國老帥在沉寂了四五個月之後，選擇了接過河南建業的教鞭。

從建業卸任後，張外龍在家並沒有閑著，這一年多的時間，他整理了中國從U9-U20各個年齡段青訓情況的資料並附上自己的見解，希望能夠幫到中國足球。老帥認真的態度以及對中超、對重慶這支球隊的熟悉是斯威重新選擇他的部分原因。

另外，張外龍的身價是近幾年重慶選擇的教練中最低的，目前球隊遭遇資金困境仍未解決。讓張外龍回歸，於球隊而言是最為合理的選擇。值得一提的是，張外龍與其團隊目前已抵達重慶與高層進行了會面，開始著手新賽季的備戰共工作。這兩年，重慶完成了球隊的換血，同時小克魯伊夫和張外龍的執教風格並不相同，如何重新捏合調整球隊，將成為張外龍的一大考驗。

附：重慶官宣

經過一段時間的考量與甄選，結合俱樂部現階段發展規劃，今日，重慶當代力帆足球俱樂部正式與韓國籍主帥張外龍簽訂工作合同。從2020年1月1日起，張外龍將正式帶隊備戰2020賽季的中超聯賽。

張外龍教練曾在2016、2017賽季執教重慶當代力帆足球隊，幫助球隊完成賽季目標的同時，還在2016賽季率隊創造了隊史中超最高積分記錄。執教期間，張外龍教練的努力和付出深得重慶球迷的認可和喜愛。

與張外龍教練的再度攜手，是符合俱樂部現狀的適當之選。期待2020賽季，重慶斯威將士們能夠在張外龍教練的領導下，腳踏實地，拼搏不息，以全新面貌迎接新的挑戰！

個人資料
姓名：張外龍
出生日期：1959年4月5日

1959年出生於韓國，就讀於延世大學，從延世大學足球隊入選韓國國家隊。

執教經歷：

1989年-1996年在日本TOSU Futures任助教和教練。

1997年-1999年出任大宇隊首席助教。

2000年出任J聯賽川崎綠茵足球俱樂部教練。

2001年到2003年出任J聯賽札幌岡薩多足球俱樂部教練。

2004年出任仁川聯合足球俱樂部教練。

2005年率領剛成立兩年的保級球隊仁川聯合足球俱樂部勇奪K聯賽亞軍，因此成為當年的韓國最佳教練。

2008年，擔任日本大宮松鼠足球俱樂部的主教練。

2011年1月出任青島中能足球俱樂部主教練，取得了青島中能職業聯賽以來的最好成績（第六名）。

2011年年底至2012年4月出任大連阿爾濱足球俱樂部主教練。

2012年-2013年 重新執掌青島中能足球俱樂部主教練，率領球隊順利保級。

2013-年2015年，出任韓國足協技術部副主席，韓國國家隊三人顧問團成員之一。

2016年-2017年，出任重慶當代力帆足球隊主教練。

2018年4月-2018年9月，出任河南建業足球隊主教練。
