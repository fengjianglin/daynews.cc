中國體育界開疆闢土的四大天王，誰最強？劉翔爭議大，姚明人氣高

2019-12-10 564條評論 2,488次閱讀 292人點贊

衡量一名職業運動員是否偉大的重要因素有哪些呢？小編認為一個職業運動員是否偉大與他在賽場上取得的成就有著很大的關係，而他帶給人們的影響以及在平時生活中樹立的榜樣力量也是很重要的。只有憑藉著自己的發揮對別人起到積極的促進作用的人才是優秀的職業運動員。接下來就讓小編帶著大家了解一下中國體育界開疆闢土的四大天王，誰最強？。

李娜

作為中國網球賽場上發展的最出色的一名職業運動員，李娜是我們中國第一位拿到網球大滿貫的職業運動員，而且她有兩次這樣偉大的成就。雖然在世界賽場上不是排名第一，但是她個人身上蘊含的偉大力量以及堅持不懈的精神卻給別人帶去了非常積極的影響。大家都知道運動員是為國家效力的，有的時候也會享受到國家的各種政策，不過有的人卻選擇了絲毫不作為，用著納稅人的錢卻不為國家辦實事，但是李娜不一樣！職業生涯她取得的成就、獲得的工資都是憑藉著自己努力得來的，而她也是人們心中最偉大的偶像，李娜體壇一姐不是吹的。

劉翔

想必田徑賽場上最引人注目的還是黑人運動員，畢竟他們的身體素質強悍，和很多人比起來在田徑項目上的優勢更加的明顯，但是劉翔的出現卻改變了人們一貫的看法。作為一名黃種人，在田徑賽場上劉翔做出了非常偉大的成就。2006年的瑞士洛桑姚明成功的打破世界紀錄，而他也被稱為世界上跑得最快的男人。在110米跨欄這個項目上劉翔做出的偉大成就值得所有人敬佩。職業生涯飽受爭議，兩度奧運退賽，劉翔雖然實力強勁，爭議大。可他對中國體育的貢獻絕對是不可磨滅的。

姚明

姚明人氣高：中國男籃有很多職業運動員都進入了NBA，但姚明的發展絕對是最出色的那一個。2002年姚明憑藉著自己出色的天賦以狀元秀的身份加盟NBA，而他在火箭隊也成為了球隊的核心成員，扮演著非常重要的角色。職業生涯姚明取得了許多偉大的成就，而姚明在退役以後同樣入選了籃球名人堂。如果沒有他，中美兩國的籃球文化交流絕對會滯後很長時間，我們中國的籃球也不會像現在這樣火熱。所以說姚明對中國的影響力是很大的，也可以稱得上是中國人最敬佩的職業運動員，也是中國體壇人氣最高的人。

劉國梁

我們中國的乒乓球一直發展得非常成功，在世界賽場上很多國家的乒乓球隊伍都不願意麵對中國，因為中國人的乒乓球實在是太強大了。作為我們國家乒乓球賽場上最偉大的職業運動員，劉國梁在職業生涯期間取得了非常多的榮譽，是許多乒乓球迷心中的偶像。在退役後他還擔任了中國乒乓球國家隊的主教練，執教這麼多年劉國梁帶領國家隊的年輕小夥子們取得了非常偉大的成就。面對著中國乒乓球現在的發展局勢，劉國梁不僅培育老球員，而且也在竭盡所能地挖掘一些極具天賦的年輕運動員。有了他的存在，中國的乒乓球事業確實得到了很大的保證，用偉大這個詞語來形容他也絲毫不為過。

正是有了這些出色的職業運動員我們中國體育才會發展得如此成功。那麼大家又有什麼想要說的話呢？歡迎在評論區下方分享哦。
