摘要

北京時間今天晚上19點35分，CBA常規賽的比賽將會繼續打響，今晚最受關注的比賽毫無疑問是北京首鋼對陣廣東宏遠的比賽。這兩支球隊可以說是死對頭，從當年馬布里時期一直到現在，每一個賽季的京粵大戰都是焦點。

北京時間今天晚上19點35分，CBA常規賽的比賽將會繼續打響，今晚最受關注的比賽毫無疑問是北京首鋼對陣廣東宏遠的比賽。這兩支球隊可以說是死對頭，從當年馬布里時期一直到現在，每一個賽季的京粵大戰都是焦點。記得上個賽季的首輪常規賽北京首鋼和廣東宏遠就碰面了，當時廣東宏遠憑藉著威姆斯、周鵬以及趙睿等人的發揮順利拿下了比賽。今天這場比賽是雙方本賽季的第一次交手，這對於兩支球隊來說都是個巨大的考驗。

相比起上個賽季廣東宏遠這次少了鋒線核心周鵬，所以誰來防守翟小川其實是一個比較大的問題。同時在面對強隊的時候廣東宏遠失去周鵬後的問題也突顯出來。而北京首鋼在新賽季打的小外援上則有了變化，他們的小外援由阿隆-傑克遜變成了林書豪。目前來看林書豪和北京首鋼還是完美契合的。只不過林書豪的失誤也確實多，在經歷了幾次傷病之後，林書豪的能力似乎也沒有顯得那麼突出了。不過對於主打團隊的北京首鋼來說，這賽季他們確實有了不小的進步，不過今晚對於他們來說仍然是個巨大的考驗，廣東宏遠的實力遠遠在北京首鋼之上。

廣東宏遠主教練杜鋒指導在下午接受採訪的時候其實就談到了周鵬的傷情。杜鋒指導表示現在手術比較成功，他現在還處於康復階段，具體他要什麼時候才能復出，杜鋒指導並沒有給一個明確的說法，看起來周鵬需要康復的時間還是有點久的。

而除了周鵬之外，杜鋒指導還提到了萬聖偉。萬聖偉作為杜鋒指導的愛徒，他這兩個賽季一直讓我們看到了進步。不過在最近兩場比賽的大名單中我們都沒有看到萬聖偉的名字，包括今天晚上的比賽。對此杜鋒指導解釋道：「我現在有18個運動員，青年隊上來5個，我都會盡量給他們上場時間。」從這裡我們也看出在之前幾輪比賽之後，杜鋒指導終於要給這些年輕人更多的機會。像上一場比賽杜鋒指導就讓李英博去硬扛了小斯7分鐘左右，雖然說李英博被虐得很慘，但是我相信他肯定是有收穫的。我們也希望這些宏遠的年輕人們可以好好加油，在接下來的比賽中把握住機會。
