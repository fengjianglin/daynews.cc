歐洲大師賽正賽名額出爐！丁俊暉傅家俊梁文博等10位中國球員晉級

2019-12-20 166條評論 2,974次閱讀 890人點贊
（此處已添加小程序，請到今日頭條客戶端查看）

北京時間12月20號早間時分，經歷兩天戰火的2020歐洲大師賽資格賽96場資格賽全部結束，中國軍團25人蔘賽，最終丁俊暉、傅家俊、梁文博、肖國棟、田鵬飛、顏丙濤、趙心童、周躍龍、魯寧、呂昊天等10人晉級32強，獲得明年1月的22舉行的正賽資格。

丁俊暉

而臨時排名前十六選手方面，除奧沙利文不參賽，特魯姆普首輪爆冷，馬克-艾倫次輪被魯寧零封，墨菲次輪2-5不敵阿爾菲-伯頓，大衛·吉爾伯特首輪2-5不敵J.布朗，利索斯基次輪4-5憾負傑克遜·佩奇，斯圖亞特·賓漢姆4-5被梁文博淘汰，馬奎爾首輪不敵馬丁·古爾德，喬·佩里首輪4-5被田鵬飛絕殺外，晉級正賽僅剩下馬克·威廉姆斯、希金斯、塞爾比、羅伯遜、丁俊暉、凱倫·威爾遜、巴里-霍金斯等7位。

傅家俊

雖然只剩下7位前16選手，但每一位都不是省油的燈，要想輕鬆奪冠也絕非易事。畢竟歐洲大師賽也是正兒八經的排名賽，是職業球員必爭之地，很多球員窮盡職業生涯也未能染指一桂冠，競爭依然會很劇烈。

顏丙濤

歐洲大師賽的歷史

說到歐洲大師賽，不得說一下它的歷史，在1988/89賽季之前，英倫三島以外是沒有斯諾克排名賽的。世界台聯為推廣斯諾克決定通過一些海外賽事來擴大它的影響力，於是就產生了第一個歐洲斯諾克賽事，也即是歐洲大師賽的前身-1988年多維爾舉辦的歐洲公開賽。歐洲公開賽舉辦9年之後一度改名為愛爾蘭公開賽，之後1999年、2000年停辦，直至2001年再次以歐洲公開賽舉行，但是2002年再次停辦。之後在2003、2004年艱難舉辦兩屆後再次更名為馬爾他杯，馬爾他杯也僅僅舉辦了四屆。直至2016年歐洲大師賽得以復活，正式命名歐洲大師賽。特魯姆普隨即在2016、2016連續兩年獲得歐洲大師賽冠軍，而今年的衛冕冠軍是吉米·羅伯遜，當時他9-6擊敗喬·佩里奪冠。

吉米·羅伯遜VS喬·佩里

歐洲大師賽資格賽結束後，進入32強的選手獲得歐洲大師賽正賽參賽資格，將在2020年1月22日至26日在奧地利的多恩比恩市舉行，今年的半決賽之前的賽制也與往年不同，往年均是採取7局4勝，而今年半決賽之前全部採取9局5勝，半決賽採取11局6勝，決賽採用17局9勝，冠軍將獲得80,000英鎊獎金。
