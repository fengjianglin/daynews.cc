梅西C羅再爭年度大獎！馬內金球後再遭無視，C羅盼能4連莊

2019-12-18 1,448條評論 2,655次閱讀 1660人點贊

北京時間12月18日，2019年度環球足球獎第二次公布候選名單！在最佳男足球員的候選中，7名候選人被縮減至4人，利物浦的馬內、阿利松以及曼城的B席爾瓦被淘汰出局！現在梅西、C羅、范迪克以及薩拉赫4人將爭奪最後的榮譽！

環球足球獎由迪拜體育局在2010年創立，原始初衷是為了表彰足球經紀人和俱樂部經理人！2011年，該獎項增設了最佳球員、最佳教練和最佳球隊等獎項！葡萄牙巨星備受這個獎項的青睞，他已經連續3年奪得最佳男足球員殊榮（16、17、18）。18年金球獎，莫德里奇力壓C羅奪魁，但環球足球獎的最終勝利者卻是C羅！從表明看來，這個獎項的評選還是極其公正的！

但19年的候選名單卻引發了巨大的爭議，因為馬內再次無緣最終的競逐！金球獎的評選中，馬內排名第4無緣最後前 3，這一度在社交媒體上引發了激烈的討論。現在的情況更為尷尬，馬內的排名甚至還在薩拉赫之後，這對馬內來說實在不公平！

客觀來講，馬內的表現已經得到了外界的認可，而且在金球獎之後得到了不少球迷以及專業人士的支持，為何他仍舊不被看好？

客觀來講，馬內的表現絕對可以去挑戰一下C羅和薩拉赫！雖然馬內難以撼動梅西和范迪克的位置，但爭取一下前3是絕對沒有問題的！薩拉赫之於利物浦十分重要，但馬內也當仁不讓。C羅在關鍵比賽中起著決定性的作用，但馬內同樣是利物浦的關鍵先生！

數據方面來說，3人沒有任何的優勢可言，但馬內就是不受外界青睞！無數球迷為塞內加爾人感到氣憤，他應該收穫符合表現的榮譽！

事已至此，再多的支持也無濟於事！好在馬內對此沒有任何的怨言，這也是他吸引球迷的地方！兢兢業業，不遺餘力的為利物浦付出，他的努力外界都看在眼中！存在著天然的劣勢，馬內可能很難觸摸到金球獎。但足球的歷史上一定會留下馬內的名字，這位非洲球員讓球迷學到了足球之外的很多東西！沉默不等於放棄，馬內一定會為金球而更加努力！

12月29日，環球足球獎將在迪拜揭開最後的懸念！范迪克能否洗刷「金球獎」的7分之辱，梅西能否再次問鼎年度大獎，C羅能否完成4連莊，薩拉赫會不會成為一匹黑馬？這些問題都會被一一解答！當然，支持馬內的聲音必然會存在，他配得上任何人的讚譽！
