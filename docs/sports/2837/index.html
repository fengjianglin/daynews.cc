法國隊被分入死亡之組後找不到熱身對手，兩進決賽“強到沒朋友”

2019-12-03 0條評論 21次閱讀 0人點贊

摘要

兩周前，歐洲杯預選賽全面收官，法國隊以H組頭名身份順利晉級。歐洲杯小組賽抽籤儀式過後，法國與德國、葡萄牙被分在了同一小組，堪稱本屆歐洲杯最強悍的死亡之組。歐洲杯前的備戰工作至關重要，但是法國隊卻因實力強大而陷入了找不到對手的尷尬境地。

兩周前，歐洲杯預選賽全面收官，法國隊以H組頭名身份順利晉級。歐洲杯小組賽抽籤儀式過後，法國與德國、葡萄牙被分在了同一小組，堪稱本屆歐洲杯最強悍的死亡之組。歐洲杯前的備戰工作至關重要，但是法國隊卻因實力強大而陷入了找不到對手的尷尬境地。

在不久前進行的國家隊比賽日期間，各支歐洲球隊參加了2020年歐洲杯預選賽的最後兩輪比賽。此前，法國隊在倒數第三輪對陣中主場1-1戰平土耳其，無緣提前出線的同時也因勝負關係不敵土耳其滑落到了小組第二位，出線形勢稍稍出現了波瀾。

法國隊後兩輪預選賽的對手分別為摩爾多瓦與阿爾巴尼亞，前者是歐陸出了名的魚腩，後者也不足以對法國隊造成威脅。兩場比賽法國隊分別以2-1與2-0拿下勝利，由於土耳其與冰島在倒數第二輪中1-1戰平，因此法國隊提前一輪出線，並在收官戰中鎖定了小組第一。

土耳其沒能在主場拿下冰島使得他們將小組第一的位置拱手相讓，這將對歐洲杯小組賽的抽籤產生影響。根據歐洲杯抽籤的分檔規則，法國隊作為成績第7好的小組第一被列為了第二檔球隊，土耳其則作為成績第4好的小組第二被列為了第三檔球隊。

第二檔球隊相較於第三檔球隊更容易避開一些實力強勁的對手，因此小組第一出線對於法國隊來說能夠起到一些作用。由於2020年歐洲杯由12個城市聯合主辦，再加上新舉辦的歐洲國家聯賽，這導致明年的歐洲杯從參賽名額的分配到小組賽的抽籤與分組都格外複雜。

歐洲杯賽制複雜奇葩現象頻出：舉辦國無緣參賽，抽籤前知道對手

周日凌晨，歐洲杯小組賽抽籤儀式在羅馬尼亞的布加勒斯特舉行，法國被抽在了德國作為東道主的F小組。法國隊第二檔的身份使得他們一定會與一支歐洲頂級強隊同處一組，但令人意外的是，第三檔球隊中實力最為強勁的葡萄牙也被抽入了F小組，死亡之組就此誕生。

法國主帥德尚看到抽籤結果後

法國隊是2018年世界盃冠軍，葡萄牙隊是2016年歐洲杯冠軍，德國隊則是2014年世界盃冠軍，前三屆大賽冠軍同處一組這是此前從未出現過的「盛景」。明年歐洲杯想必一定會是看點十足，同時這樣的抽籤結果似乎也使得法國隊「勇奪」預選賽小組第一失去了意義。

雖然同組對手實力強勁，但去年剛剛奪得世界盃冠軍的法國隊沒有任何理由懼怕對手。根據過往的戰績來看，自2014年世界盃後法國對陣德國取得了3勝2平，而近10場與葡萄牙的的較量中法國隊贏下了9場，僅有的一次輸球恰恰發生在2016年歐洲杯決賽上。

歐洲杯揭幕戰將於2020年6月12日舉行，在此之前各支已經拿到門票的球隊將沒有任何正式比賽的任務。各隊可以利用下半賽季的幾個國家隊比賽日好好磨合一下陣容，組織幾場有針對性的友誼賽為歐洲杯熱身，而被分入死亡之組的法國隊又在友誼賽問題上犯了難。

根據法國媒體《隊報》的消息，很多已經獲得歐洲杯參賽資格的球隊都不願意與法國進行熱身。由於法國隊實力強勁，這些國家隊擔心在熱身賽中輸給法國會對球員的心態造成影響，這並不利於備戰歐洲杯，因此婉拒了法國足協發出的熱身賽邀請。

FIFA世界排名

此前，比利時足協原本打算在明年3月約戰法國隊，但這一計劃被比利時主帥馬丁內斯拒絕。除了輸給法國隊不利於備戰歐洲杯外，比利時目前FIFA排名第一的位置恐怕也將不保，這對於球員來說是一個不好的暗示。同時，英格蘭足總也拒絕了法國足協的邀請。

找不到相同量級的對手進行熱身賽，這自然會對法國隊的歐洲杯備戰造成不利影響，尤其是在被分入了死亡之組的背景下。在過去的兩屆大賽中，法國隊分別取得了亞軍與冠軍，他們強大的實力讓潛在的對手「望風而逃」，這似乎就是人們常說的「強到沒朋友」。
