李尋歡：沒有人能保證國足必勝中國香港

2019-12-17 1,306條評論 2,359次閱讀 512人點贊

本屆東亞杯前兩輪戰罷，中國隊與中國香港隊同樣是連輸兩場，同樣是一球未進。

12月15日，中國國家男足選拔隊球員譚龍（左）和韓國隊球員李英才在比賽中拼搶。當日，在韓國釜山舉行的2019東亞杯男足比賽中，中國國家男足選拔隊以0比1不敵韓國隊。新華社發

最後一輪比賽，中國隊與中國香港隊碰面，此戰將於12月18日（周三）15點15分開戰，很多球迷朋友都在等著看國足如何狂勝中國香港，認為贏少了都不算贏，還有個別人賭咒發誓般表示「如果中國隊贏不了中國香港，便會如何如何」。在此想提醒大家的是，足球之所以是「世界第一運動」，在很大程度上便在於其結果的「不可預知性」，沒有人能保證中國隊在某一場具體的比賽中必勝中國香港，更別說「贏少了不算贏」了。

咱們不說幾十年前的「5•19」，中國隊1:2不敵中國香港那場球了，就是近些年，國足也沒在中國香港隊身上占太多的便宜。

資料顯示，從2003年至今，中國隊六次遭遇中國香港隊，成績是4勝2平保持不敗，看上去還不錯？但還要進一步分析，最近兩次中國隊遇到中國香港，都踢成了平局，比分都一樣，0:0，一場是2015年9月，另一場是2015年11月。這兩場平局，曾經在中國足壇引起巨大波動，客觀上造成了此後教練人選、選拔理念、足協高層等多米諾骨牌似的連鎖反應。

怎麼，這些都忘了？現在憑什麼就覺得，中國隊可以「穩贏」中國香港了？

可能還會有人說，這不都過去四年了嘛，那兩場0:0，沒有太多的參考價值了。「過去四年了」，我很認可；「那兩場0:0，沒有太多的參考價值」，也說得過去；但結論呢？恐怕也不能完全是樂觀的、積極的吧。過去的幾年間，中國足壇朝令夕改、顛三倒四的事多了去了，何曾有一段持久的時間踏踏實實按照規律發展足球。

現在中國國足的整體水平，在紙面上，與四年前那一支對比，哪一支更高？如果您非要找一個參照物的話，我來提醒一下，2015年，廣州恆大還拿到了亞冠的冠軍，基本上是打遍亞洲無敵手，現在呢？不提也罷。至於有人想說「國家隊」和「俱樂部球隊」是兩碼事，恭喜您，說的非常對，這是常識，可是並不一定是絕對的，對吧。

最後鄭重聲明一下，本人絕無看衰中國國家隊、看低中國足球的意思，我只是想提醒大家「沒有人能保證，12月18日，國足必勝中國香港隊」。

（齊魯晚報•齊魯壹點記者 李志剛）
