皇馬“舊疾”已成優勢，齊達內補強1短板，2億雙星或湊齊爭冠版圖

2019-12-24 2,506條評論 4,771次閱讀 1939人點贊

在葡萄牙天王C羅離開了效力9年之久的伯納烏後，剛剛用5年4次奪冠締造了歐冠霸業的皇家馬德里，立即陷入到了鋒線無力、四大皆空的尷尬境地中，一個賽季請來3位主帥更是突顯了弗洛倫蒂諾的狼狽。不過請回齊達內再加上弗爺不俗的轉會操作能力，已漸漸讓銀河戰艦看到了走出深淵重返巔峰的希望，儘管賽季至今球隊在歐冠和聯賽的成績都不算太好，但場上表現、賽後數據和陣容儲備均足以令人感到滿意。

​不論是本輪悶平畢巴還是上輪客場從諾坎普帶走1分，皇馬在射門、威脅進攻等數據方面都佔據了絕對優勢，雖然由於鋒線臨門一腳能力欠缺導致未能全取3分，但鋒線無力並不能掩蓋球隊其他方面所取得的進步。其中賽季至今球隊在防守方面的改變便是有目共睹，這一皇馬和齊達內過往最明顯的「舊疾」竟已成為了優勢，要知道哪怕歐戰3連冠時期銀河戰艦防守端也是屢受批評。

​齊達內完整執教皇馬的16-17、17-18兩個賽季里，在西甲聯賽的場均丟球數分別為1.07球與1.15球，而同一時期死敵馬競的場均丟球分別為0.71球與0.58球，由此不難看出調教球隊防守端是齊達內的一大短板。但重新拾起皇馬教鞭後齊達內已漸漸補強了這1短板，本賽季18輪聯賽過後銀河戰艦僅產生了12個丟球，這一防守數據已與死敵馬競的11個丟球相差無幾，與此同時比榜首巴薩的21個丟球數據好了將近1倍。

​而更可喜的是功勛主帥齊達內不僅補強了防守短板，還做到了讓皇馬未來幾個賽季都無需再為防線擔憂，因為除了核心拉莫斯以及半主力馬塞洛年齡偏大外，庫爾圖瓦、瓦拉內和卡塞米羅等中後場核心均正值盛年。不過解決了皇馬防守「舊疾」的齊達內還需做好老本行，這位球員時代便展現出了滿滿進攻才華的法國巨星，首度執教銀河戰艦時同樣以犀利進攻而聞名歐陸，然而本賽季皇馬卻因種種原因陷入到了鋒無力窘境。

​皇馬在16-18兩個賽季的聯賽場均進球數分別為2.79與2.47，但本賽季18輪聯賽取得33粒進球的他們場均僅有1.83球入賬，因此要想爭奪各路冠軍恐怕齊達內還需多在進攻端下功夫，而2億雙星阿扎爾和阿森西奧或許就能夠湊齊爭冠版圖。目前正在養傷的兩人均稱得上是齊達內愛將，受傷前阿扎爾用組織盤帶、突破過人盤活了進攻端，而擁有大心臟的阿森西奧若作為替補同樣威力十足。因此當2億雙星阿扎爾與阿森西奧健康歸來後，這支已補強防守短板的皇馬無疑將充滿了看點！
