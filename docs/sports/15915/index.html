四國賽-中國國奧3-0擒馬里國奧獲亞軍，張玉寧破門陳彬彬建功

2019-12-14 1,460條評論 4,199次閱讀 1836人點贊

北京時間 12月14日19時35分，2019年珠海四國賽，中國國奧對戰馬里國奧。歷經一番角逐，中國國奧憑藉張玉寧、陳彬彬的進球以及對手的烏龍球3-0戰勝馬里國奧。如此一來，中國國奧本屆賽事斬獲2勝1負，屈居亞軍。敘利亞國奧則斬獲2勝1平，獲得本屆四國賽冠軍。

中國國奧此前先是2-1絕殺塔吉克國奧，隨即0-1慘遭敘利亞國奧絕殺。此番一役，是中國國奧在本屆四國賽的最後一場比賽。該場對決，郝偉繼續考察球員，童磊、楊帥及迪力穆拉提等人紛紛入圍首發。

第4分鐘，中國國奧前場展開高位逼搶，杜姆比亞左邊路接傳球選擇回傳已棄門跑到禁區邊緣的門將，皮球擊中段劉愚反彈飛入禁區，張玉寧將球卸下稍作調整起左腳推射遠角，門將馬薩馬退防到門前倒地側撲把球撲出底線。第10分鐘，段劉愚前場右路得球擺脫防守起左腳傳後點，陳彬彬迎球起腳凌空抽射但將球打飛。

第14分鐘，陳彬彬前場左路帶球內切突入禁區晃開防守起右腳抽射，皮球擊中對方隊員腿部發生折射擦柱而出。第19分鐘，馬里國奧反擊，卡雷姆貝前場右路接隊友斜塞隨即起右腳遠射，皮球偏門而出。第20分鐘，伊薩-凱塔反擊過程中帶球推進到禁區外起右腳遠射將球打偏。第25分鐘，中國國奧反擊，迪力穆拉提接陳彬彬直塞帶球沿左路突破傳中，張玉寧沿中路突入禁區迎球沖頂命中，1-0，中國國奧打破僵局。

第27分鐘，庫馬雷禁區外接到隊友斜傳球，庫馬雷隨即轉身起右腳抽射，皮球稍稍偏出球門立柱。第30分鐘，陳彬彬前場左路得球傳中，張玉寧前插到門前準備搶點攻門，杜姆比亞搶點解圍時先捅到球，皮球彈地後擦柱而出。第38分鐘，達博前場中路接傳球隨即突然起右腳遠射將球打偏。第41分鐘，張玉寧反越位前插接隊友直塞突入禁區面對棄門出擊的門將起腳捅射，門將倒地封堵將球抱住。

第42分鐘，中國國奧後場斷球後反擊，段劉愚斜塞左路，陳彬彬前插接應，門將馬薩馬衝出禁區撲防陳彬彬卻慢了一拍，陳彬彬隨即推射空門命中，2-0，中國國奧擴大比分。第45分鐘，伊薩-凱塔禁區弧接應庫馬雷傳球起右腳抽射，陳威穩穩將球撲住。半場結束，中國國奧2-0領先。

第50分鐘，中國國奧反擊，張玉寧禁區外接傳球直塞禁區，段劉愚前插突入禁區面對門將起左腳射門被門將撲出底線。第53分鐘，陳彬彬前場左路傳中，黃政宇前插到門前迎球腳後跟磕射，門將將球抱住。第60分鐘，杜比亞前場中路得球突然起右腳遠射將球打偏。第62分鐘，段劉愚主罰右路角球傳中，趙劍非禁區內躍起爭頂將球頂向門前，拉西納-凱塔伸腿一擋，皮球直接鑽入自家網窩，中國國奧憑藉對手送出的烏龍球取得3-0。

第71分鐘，中國國奧反擊，胡靖航接隊友斜塞帶球沿右路突入禁區起腳勁射，門將封堵近角將球撲出底線。第72分鐘，段劉愚主罰角球傳中，趙劍非爭頂攻門，門將飛身側撲將球撲住。第79分鐘，伊薩-凱塔前場中路得球起腳遠射將球打飛。最終，中國國奧3-0取勝。
