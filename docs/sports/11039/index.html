球員時期並不風光的秦志戩，為何劉國樑上任之後，成為男隊主教練

2019-12-11 805條評論 1,933次閱讀 790人點贊

2018年，時任國乒主教練的劉國正，在瑞典世乒賽帶領國乒獲得男團冠軍，並且在亞運會上大膽啟用新人，在馬龍和許昕缺戰的情況下，樊振東林高遠梁靖崑都有出色表現，尤其是王楚欽在半決賽失利的情況下，頂住巨大壓力，在決賽中完成逆襲。而劉國正的這一做法也得到了國乒球迷的普遍好評。

不過劉國樑上任之後，對國乒大洗牌，隨著秦志戩，馬琳，陳玘等人紛紛回歸球隊，而劉國正隨後成為了一名主管教練，現在的職責是主管王楚欽。這就引來了國乒球迷的不滿。球員時期，劉國正曾經挽救7個賽點，拯救國乒，而教練時期，劉國正帶領國乒獲得大賽冠軍，執教能力可圈可點，而如今劉國樑上任之後，球員時期並不出色的秦志戩，為何會成為男隊主教練？

其實劉國梁有自己的想法，可以這麼說，好的球員未必是好的教練，好的教練也未必都是好球員。秦志戩球員時期是國乒一隊的球員，最好的成績是1994年的世界盃男團冠軍成員，2001年世乒賽的混雙冠軍。但是在冠軍球員扎堆的國乒，確實有些平談無奇。但是在成為教練之後，秦志戩通過自己的執教方式，培養出來的兩位世界冠軍，確實無出其右。

2005年秦志戩退役之後，就成為了國乒的一位實習教練，隨後收了兩位弟子，就是如今拿到無數冠軍的馬龍和許昕，秦志戩作為教練之後，就一直是馬龍和許昕的主管教練，在教練領域，秦志戩有一定的發言權，頭腦清晰，懂得運用心理戰術，幫助球員提高成績和心態，馬龍和許昕的成長，和秦志戩有密不可分的關係。

如今秦志戩是樊振東的主管教練。樊振東今年上半年的低迷狀態，讓球迷非常擔心，自從秦志戩接管樊振東之後，對樊振東進行了全面的心理和戰術指導，樊振東通過半年的努力，終於恢復到最佳狀態，這更加能夠證明，秦志戩在教練指導方面的能力。

劉國梁眼光獨到，並不是任人唯親，曾有球迷表示，劉國梁辭職事件發生之後，秦志戩力挺劉國梁，所以劉國樑上任之後，秦志戩才成為國乒男隊主教練，但是有一個疑問，如果秦志戩只是力挺劉國梁，而沒有真材實料，各位想一下，劉國梁會不會冒著國乒丟冠的風險，去啟用秦志戩呢？顯然不會，因為劉國樑上任之後，一切以國乒的榮譽為主。在用人方面也是求賢若渴，所以秦志戩能夠成為男隊主教練，也是因為他的執教能力。
