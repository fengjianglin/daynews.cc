遼寧隊115：105擊退吉林隊，豪取四連勝

CBA聯賽第十七輪，遼寧隊坐鎮主場VS吉林隊，吉林隊被稱為「CBA」的巨人殺手，他們本賽季贏過廣東隊，大勝新疆隊，因此，對於遼寧隊來講，同樣是一個非常大的挑戰。最終，遼寧隊頂住了壓力，以115：105擊退了吉林隊。由此一來，遼寧隊收穫了4連勝，同時，也以15勝5負，70.6%的勝率，本賽季首次闖進CBA積分榜四強。而，吉林隊則以11勝6負的成績，掉出了積分榜四強。

在個人數據上，吉林隊這邊有3人得分上雙，瓊斯30分8籃板5助攻3搶斷，霍爾曼20分11籃板5助攻，姜偉澤17分2籃板，崔晉銘8分3籃板3助攻，劉天意8分2籃板，姜宇星7分6籃板2搶斷，柴長易6分，代懷博5分3籃板。遼寧隊這邊有5人得分上雙，史蒂芬森23分6籃板3助攻、趙繼偉23分3籃板10助攻2搶斷，郭艾倫20分3籃板6助攻，韓德君17分13籃板3助攻，巴斯26分7籃板2助攻3搶斷，李曉旭3分10籃板。

從雙方比賽來看，遼寧隊進入狀態十分快，特別是趙繼偉連續3分球命中，同時，連續送出精彩的助功，幫助球隊很快就取得了2位數的領先勢，首節比賽打完，遼寧隊以35：19領先了對手16分。第二節比賽，吉林隊霍爾曼和瓊斯聯手，一度將落後的分差縮小到了只有7分，不過，隨後趙繼偉與巴斯發力，幫助遼寧隊在半場結束時，以59：49領先了吉林隊10分。

下半場易邊再戰，遼寧隊繼續發力，一度將領先的分差擴大到了18分，不過，吉林隊打得十分頑強，通過內外結合，一點點蠶食遼寧隊領先的分差，隨後遼寧隊巴斯和趙繼偉聯手穩住了局面，三節比賽結束，遼寧隊以88：76領先了12分。第四節比賽，雙方打得比較膠著，但是，主動權一直在遼寧隊手中，最終，遼寧隊以10的優勢戰勝了吉林隊，豪取聯賽四連勝。

賽後，郭士強談到球隊有三點贏球原因

在賽後，吉林隊主教練王晗對本場比賽進行了點評，他表示球隊在第一節和第三節未段沒有打好，球隊在進攻端出了問題。同時，球隊也存在一個重要問題，那就是瓊斯個人單打與隊友之間進攻的問題，球隊是需要團隊配合進攻與個人能力展現相結合。本場比賽，瓊斯在第四節比賽一分鐘沒有上，可能也是王晗特意安排，通過比賽來給球隊做一些戰術演練。

遼寧隊主教練郭士強在總結點評中，談到了球隊贏球的三點原因，他表示第一點，球隊防守方面做得非常不錯，防對方小外援做得非常好，他近段時間場均50分，本場比賽限制了他的發揮。第二點，球隊整體防守做得非常好，協防做得很不錯。第三點，球隊在進攻端做得非常好，球隊在進攻端的梳理做得很出色。

從郭士強的點評來看，遼寧隊贏下這場強強對話，他心裏面還是十分的開心。對於球隊目前的狀態，郭士強表示，現在球隊磨合方面對賽季初要好了很多，各方面都是有所好轉，當然了，還要時間去繼續磨合。

知名解說王猛和男籃名宿王仕鵬大讚趙繼偉：看他打球真舒服

本場比賽，遼寧隊之所以能夠比較輕鬆的拿下比賽勝利，與一個人大爆發是分不開的，沒有錯，他就是被外界稱為鞍山保羅的趙繼偉，在比賽中，趙繼偉就像是遼寧隊的守護神一樣，第一節比賽，通過精準的三分球和精彩的助攻，幫助球隊取得了2位數的領先優勢，到了第三節球隊出現困境時候，又是趙繼偉站了出來，幫助球隊再次走出泥潭。

全場比賽，趙繼偉吹下了23分10助攻3籃板2搶斷1蓋帽的全面數據，其中，得分和助攻都創造了本賽季個人新高。對於趙繼偉的表現，知名解說王猛和男籃名宿王仕鵬都不約而同合對他進行了點贊，王猛表示趙繼偉打球太聰明了，把球隊梳理得非常好，能快、能慢，節奏特別好，看他打球是一種享受，真舒服。王仕鵬也表示要特別點贊一下趙繼偉，思路太清晰了，經過了這個夏天，他打球更成熟了，真棒。同時，也希望遼寧隊能夠儘快找到前兩個賽季的那種自信。
