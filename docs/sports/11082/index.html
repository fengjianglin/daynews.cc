塞塞尼翁：穆帥拿我和阿什利-科爾比較，真是受寵若驚

2019-12-11 1,360條評論 3,288次閱讀 1686人點贊

虎撲12月11日訊 周中的比賽，塞塞尼翁有機會迎來自己加盟熱刺之後的首次首發。賽前，他也表示穆帥給過自己很多建議，對提升他的信心很有幫助。

「他坐下來和我交流，告訴我勇敢做自己，也讓我多和隊友溝通，因為我相對比較內向。」

「他希望我展示自己的個性，展示出我想出場的意願，展示我為獲得出場機會願意拼搏的決心，說了很多鼓舞人心的話。」

「我確實需要和他人進行更多交流，這是肯定的。我還年輕，不論場上還是場下，我都得變得更活躍，並且用自己的能力，去影響比賽走勢。」

「我需要努力工作，殺入到一線隊，展示自己的實力。坦白說，穆帥的話很有道理。」

談波切蒂諾

「他是我加盟熱刺的原因之一，他的離任當然讓人感到有些失望，因為我夢想著為他效力。」

「不過球隊聘請穆帥也讓我感到興奮，他是一位名帥，贏得了很多榮譽。作為一名年輕球員，你希望在他手下收穫成長。他經常給我一些建議，告訴我能在哪些方面進行提升。」

談穆帥曾拿塞塞尼翁和阿什利-科爾進行比較

「能和這麼偉大的球員相提並論，真是受寵若驚。」

「這也讓我變得更加自信，想要變得更努力。小時候，我經常看科爾踢球，他是我的偶像之一。」

「我是一名攻擊性比較強的邊後衛，位置前提也能踢，我可以進球，也能給隊友助攻。」

(編輯：姚凡)
