危機？女單：被打1-11，劉詩雯出局！孫穎莎“再翻船”！央視賽程

2019-12-14 1,187條評論 2,768次閱讀 1507人點贊

2019國際乒聯年終總決賽，女單1/4決賽，孫穎莎「爆冷」出局！

女單半決賽時間：2019年12月14日19：00 女子單打半決賽；19：50 女子單打半決賽。CCTV5直播！

（一）

女單1/4決賽：劉詩雯3-4陳夢！

本場比賽跌宕起伏！0-2，3-2，3-3，3-4，雙方貢獻了一場精彩紛呈的比賽！

七局比分：7-11，7-11，11-9，11-6，11-9，5-11，1-11。

這場女單1/4決賽，如同過山車！

陳夢2-0領先之後，劉詩雯頑強的連追3局，大比分以3-2反超。

第六局比賽，劉詩雯突然斷電，被陳夢11-5贏下。

決勝局，劉詩雯狀態依舊不佳，陳夢11-1贏下決勝局！

總大比分：陳夢4-3戰勝劉詩雯！

半決賽：陳夢VS伊藤美誠；CCTV5直播。

比賽時間：2019年12月14日19：00 女子單打半決賽；19：50 女子單打半決賽。

賽後，劉詩雯表示：整體局面有點下風，3-2領先後第六局開局輸的太快，連續專註度不夠，是一個大問題。現在的狀態距自己期待的水平還有距離，其實是期待能在明年奧運會前和伊藤美誠交手，年輕一代的隊員都很有實力，努力爭取和她打！

有球迷說：劉詩雯專註度問題確實是存在，好幾次均是如此，發現問題未必是壞事，有時間解決問題。

（二）

另一場女單1/4決賽：孫穎莎4-1王藝迪！

雖然說國乒內戰，誰輸誰贏均正常！只是孫穎莎意外的連續「翻船」（女雙半決賽，孫穎莎王曼昱2-3日乒組合）。

五局比分：11-13，5-11，8-11，11-9，8-11。

祝賀王藝迪！希望孫穎莎好好總結，再接再厲！

有球迷說：以暴制暴，國乒兩姐妹都是狠人！
