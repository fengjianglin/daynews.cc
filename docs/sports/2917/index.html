遼寧男籃迎來了兩連勝，球隊向著三連勝的目標衝擊。上一場比賽郭士強進行了賽季以來陣容最大的變化，球隊輪休了韓德君。這也是遼寧隊必須要進行輪休，本賽季遼寧男籃只有韓德君一名正牌中鋒。有了上個賽季球隊韓德君受傷的隱患郭士強也是特別的謹慎使用韓德君，對於遼寧男籃來說已經傷了一個叢明晨，內線不能夠再傷了，郭士強還是未雨綢繆的。

同樣的遼寧男籃也暴露出來了隱患內線大韓不在籃板和內線防守都是一個問題，本場比賽郭士強派上了李曉旭擔任球隊的首發中鋒。本賽季的李曉旭表現的很一般，在進攻端中投也不穩定，籃球也漏了不少。防守腳步也跟不上了，不過本場比賽遼寧男籃悍將李曉旭終於爆發了拿到了13分13個籃板的數據，他也解決了遼寧男籃的一個問題就是籃板。

如果李曉旭能夠每一場比賽都能夠穩定發揮那麼在韓德君沒有替補中鋒的情況下李曉旭可以充當替補，這樣一來能夠給韓德君更多的時間休息。遼寧男籃外線球員停賽還有足夠的球員，但是韓德君不上就出現了很大的問題。還是希望郭士強能夠從遼寧男籃內部挖掘出一名內線替補球員，常規賽還能夠堅持一下季後賽則是很大的問題，想要奪冠必須解決這個問題。

遼寧男籃上一場比賽大量的使用年輕人，包括郭旭、王化東都得到了很多時間。馬壯也得到了一些機會，郭旭本場比賽完成了大爆發，郭旭出生於1995年今年已經25歲了。這個年紀並不能算是新星了，但是因為之前他並沒有得到什麼機會所以還是在賣潛力。本場比賽郭旭成為了替補神兵拿到了18分，另一名新星王化東出場15分鐘拿到了0分2籃板。

但是郭士強似乎並不是很滿意遼寧男籃的3名新星們，在談到新星們的表現郭士強表示年輕球員有機會上場要珍惜，因為不管是進攻還是防守，都要對球隊起到作用。否則對於他們來說給了那麼多時間做不好是不應該的。最應該在防守方面投入更多一點，郭士強表示對三名新星的防守不是很滿意。郭士強對於年輕球員的使用還是很苛刻的。

本場比賽三名年輕球員更多的還是想要靠進攻證明自己，在之前的比賽當中郭士強對於年輕球員使用的很少。下一場比賽韓德君歸來之後王化東就很難得到機會了，在遼寧男籃的年輕人需要耐得住寂寞。遼寧男籃是一支爭冠球隊，所以球隊不太喜歡重用一些年輕人。郭旭、王化東和馬壯這些球員未來是遼寧男籃重要球員，但目前並不是。
