魯能舊將吉爾家中巨幅海報搶眼，或紀念生涯最佳時刻

天天要聞
2019-12-20

山東魯能舊將吉爾已經結束了回歸巴甲聯賽的第一個賽季，如今巴甲聯賽和中超聯賽一樣，都已經進入到了冬歇期，所有球員都已經開始了假期生活，而我們熟悉的巴西球員吉爾，他在回到巴西後的假期里，依然堅持了和中超時期同樣的一個習慣，那就是繼續保持訓練。

​吉爾的自律性此前在中超聯賽效力時，都是非常出名的，在山東魯能時，除了冬歇期他會回到巴西一段時間，一年內其他時間段的中超休賽期里，吉爾基本都會留在中國，他也不會前往中國各地去遊玩，而是老老實實地呆在山東，每天都在健身房裡堅持訓練，讓他自己的身體狀態始終保持在一個最佳的狀態，也令他在中超聯賽中一直都發揮十分穩定，深受球迷的信任和喜愛。

​如今已經32歲的吉爾，依然保持了這個在中超的習慣，而且回到巴西後，他的日常訓練也更方便了，他在自己巴西的家中就有健身房，他可以直接在家裡完成每天的訓練安排。而近日在吉爾曬出的他在家中訓練的照片里，我們還發現了吉爾家中健身房裡的一幅巨型海報，十分搶眼。

​這幅海報就在吉爾健身所在的身後牆壁上掛著，而這幅海報之所以掛在健身房的牆壁上，或許這幅海報也是為了紀念他職業生涯中最巔峰的時刻，他也想通過這種方式，來激勵自己，要不斷變得更強，這才能配得上曾經巔峰時刻的自己。而這幅巨型海報中的內容就是，吉爾身穿巴西國家隊球衣，和世界上最好的足球運動員梅西一起同場競技的畫面。

​吉爾的職業生涯雖然經常可以入選巴西國家隊，但是真正代表巴西國家隊出戰正式比賽的場次只有11場，而在這11場的國家隊生涯中，其中就包括了巴西國家隊與阿根廷國家隊的巔峰對決，吉爾把這場比賽中的畫面留了下來，這也將成為他的人生中最難忘的回憶之一。
