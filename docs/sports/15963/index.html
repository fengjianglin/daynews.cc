羽毛球總決賽：三戰三敗需要反思，兩個看點最值得期待

2019-12-14 267條評論 4,111次閱讀 1766人點贊

羽毛球總決賽第一階段比賽全部結束，何冰嬌在女單賽場三戰三敗，止步第二輪，表現差強人意，令人難以期待，需要反思，是中國隊表現最差的選手。中國女單一姐和混雙排名前兩位的種子選手，順利晉級，這兩個看點最值得期待，也是總決賽的奪冠熱點，值得關注。

經過三個比賽日的爭奪，40人/組合經過60場緊張激烈的比賽，一半選手遭到淘汰，無緣晉級第二階段比賽，止步總決賽。包括中國選手何冰嬌、杜玥/李茵暉和李俊慧/劉雨辰。其中，何冰嬌三戰三敗，首戰0：2完敗陳雨菲，第二戰1：2敗給山口茜，第三戰0：2遭到辛杜碾壓，表現一般，面對世界頂級選手，沒有展現出自己的實力。

杜玥/李茵暉和李俊慧/劉雨辰各戰三場比賽，分別贏了一場；杜玥/李茵暉前兩場比賽1：2敗給了陳清晨/賈一凡、1：2福島由紀/廣田彩花，最後一戰面對波莉/拉哈尤，以2：1取勝，拼盡了全力；在實力和比賽經驗上還存在一定的差距，特別是第二場比賽，第二局一度以20：15領先，結果功虧一簣，被對手實現了超級逆轉，這場比賽值得總結。李俊慧/劉雨辰，三場比賽的結果分別是：1：2蘇卡穆約/吉迪恩、2：1嘉村健士/園田啟悟、0：2渡邊勇大/遠藤大由。

中國羽毛球隊8人/組合攜手出戰總決賽，何冰嬌、杜玥/李茵暉和李俊慧/劉雨辰在第一階段比賽結束之後，被淘汰出局，諶龍、陳雨菲、陳清晨/賈一凡、鄭思維/黃雅瓊和王懿律/黃東萍挺進了第二階段淘汰賽，除了石宇奇因傷無緣比賽之外，代表了目前中國羽毛球隊最強的實力，也是目前國家隊的最核心選手。

陳雨菲和混雙組合是具備奪冠實力的選手，是總決賽最值得期待的兩大看點。目前陳雨菲世界排名第二，總決賽有望衝擊世界第一，奧原希望、山口茜和戴資穎成功闖進了半決賽，經過抽籤，陳雨菲再遇山口茜，13日陳雨菲以2：0橫掃對手，時隔一日再次交手，結果值得期待。

鄭思維/黃雅瓊跟渡邊勇大/東野有紗的比賽正在進行當中，雅思組合在小組賽首場比賽中0：2被對手橫掃，這是一場復仇之戰，雅思組合開局非常順利，首局快速領先，以21：7拿下首局比賽，表現完美。
