深度：復盤德甲狼堡0-2拜仁，探河南建業王寶山變陣及引援可能性

2019-12-24 2,734條評論 4,852次閱讀 3889人點贊

北京時間12月21日晚，德甲第17輪上演了一場「矛盾之爭」，德甲攻擊力最強球隊之一拜仁慕尼黑主場迎戰德甲本賽季防守最好球隊狼堡。經過90分激戰，穆勒、萊萬比賽尾聲階段先後送出助攻，齊爾克澤和格納裡布進球，拜仁上演3分鐘2球好戲2-0擊敗狼堡。

本場比賽狼堡雖然敗北，但其極具針對性和充滿韌性防守給球迷留下極其深刻印象，若不是拜仁第83分鐘用18歲小將齊爾克澤換庫蒂尼奧收穫奇效，比賽結果難料。狼堡德甲最佳防守球隊實至名歸！

本場比賽，狼堡陣型為4231，4後衛+雙後腰配置讓對手拜仁慕尼黑長時間無計可施。作為河南建業鐵杆球迷，採薇反覆研究了比賽錄像，認為復盤此一戰對河南建業王寶山新賽季變陣和引援有借鑒和啟發意義。

一、採薇觀點：狼堡隊德甲防守最強，河南建業防守問題突出，狼堡防守值得建業借鑒

本賽季至今，德甲聯賽進行了17輪，狼堡隊6勝6平5負積24分名列德甲積分榜第9名（18支球隊）。其中，狼堡隊前9輪僅僅被打進5球，前17輪丟球總計18球，丟球數低於排名聯賽榜首的萊比錫20失球和聯賽排名第3位的拜仁22失球，與聯賽排名次席的門興同為德甲失球最少。說明狼堡防守優秀非一場兩場，狼堡德甲最佳防守球隊實至名歸。

狼堡德甲失球最少

得益於前場奧漢德扎、巴索戈、卡蘭加的強勢發揮和中場伊沃、馮卓毅、周定洋「中場鐵三角」穩定支持，河南建業2019賽季進攻端攻進41個進球，場均1.37近5年最好。但同時，河南建業在2019賽季失球數達到46個，場均失球1.53個也是近5年失球最多。採薇觀點：2020賽季，王寶山的河南建業想在進攻端進一步提升攻擊力難度較大，但防守端通過引援和打法調整或許可以收到立竿見影之效果。

河南建業近5賽季場均進失球

除了防守數據出色值得建業學習外，狼堡現球隊氣質與河南建業也較類似。一是球隊已屬平民球隊，隊中已無大牌球星。區別於拜仁現陣容諾伊爾、穆勒、萊萬、庫蒂尼奧等這些如雷貫耳的球星名字，狼堡隊現陣容已沒有知名球星，哲科、德布勞內等已是狼堡回憶。狼堡隊中現役最貴球員施拉格爾身價1500萬歐，在中超也只能排到10名左右。二是狼堡近兩年也是聯賽保級球隊。2016-2017和2017-2018兩個賽季，狼堡都只排在第16名，依靠升降級附加賽勉強留在德甲。經過調整，2018-2019賽季狼堡戰績才回到聯賽中游。三是狼堡面對強隊時也打防守反擊戰術。

二、復盤：狼堡防守出色反擊犀利，拜仁靠替補奇兵尾盤取勝

布局：狼堡雙後腰阿諾德+吉拉沃吉成亮點

拜仁vs狼堡首發

本場比賽，狼堡主打4231（433）陣型。後衛線魯西隆、布魯克斯、蒂塞蘭和姆巴布從左至右排開，後腰放上27號阿諾德和23號吉拉沃吉，24號施拉格爾擔綱前腰，40號維克托和8號斯特芬分列左右兩翼，11號克勞斯擔綱中鋒。

狼堡主帥格拉斯納顯然有備而來：拜仁場均2.7個進球火力兇猛，而狼堡場均只有1個進球，打對攻戰無疑自尋死路，堅決打防反才是狼堡務實之選。但狼堡並不是一味龜縮半場，二是使用積極的防守策略，一定程度上提防線，盡量使皮球停留在拜仁半場。進攻中快速通過中場，後衛可直接發動長傳攻勢找前鋒，門將有機會也要開大腳直接找前鋒線。

拜仁前場支點萊萬場均1.1個進球，狼堡想防住萊萬非常困難，選擇增加一個後腰實行雙後腰切斷巴西中場庫蒂尼奧與前場萊萬的聯繫。

萊萬火力兇猛

進程：狼堡防守切斷拜仁中場與前場聯繫，進攻依靠快速過中場和高位逼搶

比賽開始後，拜仁選擇壓迫性進攻。但狼堡穩住陣腳密集防守，2分50秒，狼堡門將卡斯蒂爾斯得到皮球，之後迅速發大腳球找前鋒11號克勞斯。這樣的場景在比賽中多次復現，說明門將大腳找前鋒是狼堡既定戰術之一。

門將直接大腳找前鋒

門將直接開大腳找前鋒

當然拜仁也對此有所防範，狼堡的門將開大腳戰術不可能一蹴而就。

狼堡的雙後腰戰術十分有效，阿諾德+吉拉沃吉不但能限制庫蒂尼奧的調度，還兼顧了對拜仁邊鋒和邊後衛的壓上防守。第10分鐘，拜仁14號佩里西奇左邊路拿球，阿諾德直接用犯規阻斷拜仁進攻推進。隨後，阿諾德又與隊友克勞斯協力關門斷下拜仁5號帕瓦爾的球權。

本賽季高列德甲助攻榜首位的穆勒表現越來越好，其不停游弋到右邊路向對手禁區輸送炮彈。第12分鐘，拜仁阿拉巴後場直接長傳找到穆勒，穆勒在右邊路獲得充裕空間，二娃起腳至禁區後點，14號佩里西奇近距離頭槌被狼堡門將撲出！本場比賽，穆勒在右邊路的傳中成為比賽關鍵，此是後話。

穆勒覓得機會

佩里西奇頭槌被撲

但狼堡的有球部位高位逼搶很有成效！第15分鐘，拜仁佩里西奇持球，狼堡姆巴布+吉拉沃吉圍堵逼搶，施拉格爾斷球推進，直塞給前場11號克勞斯，克勞斯形成單刀，但諾伊爾及時出擊封堵住狼堡單刀球。拜仁隊逃過一劫。

狼堡高位逼搶成功

施拉格爾直塞克勞斯形成單刀

諾伊爾神撲救險

狼堡高位逼搶收起效，拜仁逃過一劫

第36分鐘，狼堡隊獲得一次很遠位置的前場定位球機會，兩名隊友站在皮球前作掩護，阿諾德仍然形成了有效射門並射正。

超遠定位球狼堡依然形成有效射門

定位球有掩護

第37分鐘，狼堡中後衛32號蒂塞蘭直接大腳長傳找到左邊鋒維克托，維克托停球非常漂亮。雖然此球之後並未威脅到拜仁球門，但狼堡不過多通過中場傳遞，後場直接找前鋒的戰術意圖非常清晰。

第39分鐘，拜仁中場庫蒂尼奧得球，但狼堡迅速兩人包夾，23號後腰吉拉沃吉斷球。本場比賽，拜仁巴西中場徹底迷失在狼堡雙後腰戰術中，其不但不能給萊萬形成有效輸送，自己帶球也常常遭遇包夾丟球。

巴西人徹底迷失在狼堡泥潭

狼堡雙後腰非常有效

第44分鐘，狼堡右邊後衛19號姆巴布助攻到前場，直接給斯特芬送出直塞球。本場姆巴布表現非常穩定高效，其鎮守的右邊路幾乎沒有給對手留下進攻機會。同時在進攻端，作為右邊後衛的姆巴布常常能壓上去，既有漂亮的傳中又能送出直塞球，表現非常搶眼。本賽季河南建業楊闊傷停後右邊路陷入癱瘓，這種球員河南建業要是能拿下，建業攻防兩端將發生翻天覆地的變化。

狼堡進攻套路之反擊：反擊犀利，但門前把握機會欠佳。

狼堡本場主打防守反擊戰術，在穩固防守的基礎上一旦獲得機會即快速推進到前場，除了門將大腳找前鋒和後衛、後腰直接發起攻勢外，狼堡中路打出過多次高質量反擊。比如下半場一上來，狼堡隊一次防守中在中場獲得球權，之後快速推進到前場並轉移到右邊路，右邊鋒8號斯特芬傳中球來到後點，魯西隆再傳門前，40號維克托拍馬趕到，但伸腳碰到皮球就會空門打進的必進球卻被狼堡維克托在毫釐之間浪費！

狼堡犀利的反擊

狼堡錯失必進球

狼堡錯失進球

狼堡問題：進攻端把握機會能力不強，防守中左邊後衛位置缺乏保護，左中衛與左邊後衛身後空擋有致命空擋

本場比賽，狼堡在面對強敵拜仁慕尼黑時，攻防兩端均體現出了較高的戰術素養。進攻中第15分鐘通過高位逼搶獲得單刀被諾伊爾封堵，下半場第46分鐘通過快速反擊獲得門前機會，這是兩次絕佳的得分機會，但狼堡均沒有把握住。

在防守端，狼堡通過雙後腰戰術較好地限制了拜仁中場與前場的聯繫，使巴西天才中場庫蒂尼奧全場碌碌無為。狼堡對有球部位的高位逼搶讓人印象深刻。但是，狼堡高效的防守卻沒有堅持到最後一刻，其在比賽尾聲階段被18歲小將齊爾克澤意外破門後後防線突然崩潰，吞下0-2的苦果。這兩個失球全部出現在狼堡左邊路，一方面說明對手穆勒等在右邊路的發揮確實驚艷，另一方面也暴露出狼堡防守端的失位和相互保護不到位問題。

第85分鐘，拜仁穆勒在右邊路與隊友做出撞牆式配合，其突入進去後倒三角傳球給點球點附近包抄的18歲小將齊爾克澤，後者冷靜推射打破僵局。我們通過靜態圖結合動態圖可以看到，狼堡左邊後衛魯西隆壓到邊路後，其身後沒有隊友為其補位，這個空擋被靠腦子踢球的穆勒及時發現並被利用。

穆勒切入禁區

穆勒倒三角傳球

齊爾克澤打破僵局

丟掉第一球後，狼堡堅持了幾乎一整場的防守熱情突然泄氣，3分鐘後左中衛與左邊後衛身後再次出現巨大空擋，這個空擋被萊萬發現並利用。萊萬接隊友傳球後，頭槌為格納布里做球，格納布里笑納大禮鎖定勝局。

狼堡防守突然崩潰

萊萬做球

格納布里鎖定勝局

萊萬助攻格納布里

三、狼堡vs拜仁一戰對河南建業防守啟發及建業新賽季變陣、引援思考

2019賽季，河南建業的陣型從未有過任何變化：5後衛+3中場（單後腰周定洋，雙前腰伊沃、馮卓毅）+2前鋒（一高卡蘭加、一快巴索戈或奧漢德扎）。陣型穩定的優點是有利於球員磨合和配合，缺點是缺乏變化容易被對手研究透。

依採薇以往的觀點，採薇認為河南建業不宜變陣，建業不具備打先進足球的能力，3中衛5後衛的戰術要長期堅持。但依據建業主教練王寶山多次訪談釋放出的信息，王寶山並不會滿足於目前的球隊陣容打法和成績。王指導新賽季要追求更流暢的打法更具控制力的場面，必然會進一步豐富河南建業的進攻套路，同時後防線有些場次變成4後衛將成為一種可能。

如果4後衛成為河南建業一種戰術選擇，河南建業目前的陣容缺乏哪種球員呢？狼堡一戰的啟示作用又是什麼呢？

4後衛意味著最後一道後防線少了一人。這一人放到哪裡？

採薇認為首先應進一步充實到後腰上。也許有人會反問：「河南建業後腰不是有定海神針周定洋了嗎？還要後腰幹嘛？」。我們知道，周定洋2019賽季來到河南建業後，其第3輪開始替補登場，之後迅速融進球隊，第5輪開始取得進球，整個賽季出場26場貢獻2個進球和3次助攻。周定洋解決了河南建業多年後腰不穩的頑疾，在建業禁區前沿構造了一道有力的屏障。但是，這遠遠不夠。分析建業本賽季失球就會發現，建業後腰位置在周定洋攻上去的時候仍然會出現較大空擋，以前，這樣的空擋可以由經驗豐富的中衛顧操上搶給予彌補，但本賽季顧操這個作用別人還沒有能完全體現出來。

狼堡與拜仁一戰也可以給我們一些啟發。本場比賽，雙後腰阿諾德+古拉沃古成為狼堡排兵布陣最大亮點。在雙後腰阿諾德+古拉沃古的掃蕩下，本場拜仁中場完全失靈，庫蒂尼奧形同虛設。河南建業是否考慮可以給周定洋再配備一個後腰幫手？這樣，在面對強敵如恆大、上港時，建業的中場厚度會進一步加強，後腰攔截能力將進一步升級，正兒八經地與強隊拼中場成為一種可能。

這個後腰位置由誰擔任呢？採薇認為萬金油王上源可能是合適的候選人之一。鐵杆球迷們知道，在2019賽季中，王上源曾經在第2輪、第3輪、第13輪打過3場後腰位置，且在第2輪有過後場長傳直接助攻巴索戈的優異表現。在剛剛結束的男足東亞杯比賽中，王上源在李鐵的國家隊中也是打後腰位置，且是國家隊中後腰位置表現最為出色的球員。協助周定洋共同保護後腰位置，遇到機會還可以插上助攻，王上源能夠勝任該位置。

如果打4後衛，河南建業缺少什麼樣的球員呢？採薇認為就是缺少狼堡隊中右邊路19號姆巴布這樣能上能下的球員，速度出色，身體條件出色。該防守的時候從來不會失位，該進攻的時候大膽壓上去。最重要的是，壓上去後姆巴布既能傳中又能送出直塞！放眼望去，河南建業目前陣容中沒有這樣的球員。如果能有這樣的球員送出靠譜的傳中球，可以想像，前場卡蘭加的作用會進一步放大！

飛奔的姆巴布拉都拉不住

姆巴布

因此，不管是5後衛還是4後衛，有速度又有腳下的邊路球員是河南建業後防的最大短板，也是建業新賽季引援的首先應該考慮的問題！

當然，如果政策允許，河南建業還可考慮引進一名高大正牌中衛。4後衛要求中衛覆蓋面積更大，移動速度更快，經驗更豐富，目前建業陣中中衛位置陳灝、韓軒都具有一定的功力，王上源也能客串，但都不具備絕對的定海神針能力。顧操上兩個賽季有此能力，但本賽季已荒廢，新賽季狀態如何能否重回巔峰還有待觀察。

這樣，河南建業後場兩個邊進可攻退可守，前場卡蘭加獲得更多支持，後腰有王上源進一步加強，穩固後防的同時也加強了中場硬度，河南建業王寶山期盼已久的攻守平衡或許能實現。
