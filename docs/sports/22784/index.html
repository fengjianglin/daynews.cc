巴薩最強11人誓勝皇馬？梅西領銜三叉戟 格老闆世紀大戰首秀

2019-12-18 3,142條評論 4,791次閱讀 2026人點贊

本賽季首回合西班牙國家德比戰即將在北京時間周四凌晨打響，原本10月底的較量，因為種種原因推遲至今。對於巴薩來說，好消息是他們可以排出幾乎最強的陣容迎戰皇馬，而這一戰的勝負也將直接影響未來榜首排名的走勢。賽前西班牙媒體透露了巴薩的首發11人名單，梅西將領銜最強三叉戟出戰，而格列茲曼也將迎來自己的世紀大戰首秀。

首先來看看防線，如果比賽是在10月底進行，巴薩的防線將面臨傷病困擾。但如今，這些隱憂已經基本消除。阿爾巴和塞梅多都已經恢復健康。和皇馬的這場大戰，阿爾巴幾乎肯定將首發，而塞梅多恐怕只能替補待命，羅伯托將是右後衛首選。中衛位置上，皮克和烏姆蒂蒂將成為搭檔，門將也沒有懸念，還是特爾施特根。

雖然拉基蒂奇此前一直有轉會傳聞，但在這種超級大場面下，這位克羅埃西亞中場大師的經驗仍然無可匹敵，他肯定將佔據中場首發的一個位置。布斯克茨在後腰位置上也無可替代，而另一個主力席位則屬於弗朗基-德容。比達爾、阿勒尼亞雖然也有不俗的狀態，但巴爾韋德顯然更相信荷蘭少俠。至於阿圖爾，他恐怕在聖誕節前都不會回歸。

巴薩此役如何必勝皇馬？鋒線最值得期待。剛剛捧走第6座金球獎的梅西處於巔峰狀態，若能在對陣皇馬的比賽中再度上演帽子戲法，梅西又將再度引爆流量。蘇亞雷斯同樣狀態不俗，他也是皇馬最害怕的中鋒之一。格列茲曼將迎來個人首次世紀大戰，加盟巴薩後，法國帥哥漸漸適應了紅藍軍團的節奏，加上登貝萊有傷在身，卡萊斯-佩雷斯和法蒂都無法撼動他的位置。

巴薩戰皇馬預計首發（4-3-3）：

門將：特爾施特根

後衛：塞爾吉-羅伯托、皮克、烏姆蒂蒂、阿爾巴

中場：拉基蒂奇、布斯克茨、弗朗基-德容

前鋒：梅西、蘇亞雷斯、格列茲曼
