沒有詹姆斯，歐文什麼也不是；擁有詹姆斯，濃眉哥快封神了

2019-12-10 1,400條評論 3,675次閱讀 289人點贊

12月9日，聯盟中有兩場非常平常的比賽。缺少歐文的籃網隊險勝掘金隊；萬眾矚目的湖人隊大勝森林狼隊。提到湖人和籃網，有3個球員不得不提。騎士時期的歐文號稱單打小王子，總決賽中打敗勇士隊的畫面歷歷在目。但是歐文離開詹姆斯之後每況愈下，現在什麼也不是了；濃眉哥則恰恰相反。他在鵜鶘隊時期時常淪為空砍群主。戴維斯自從和詹姆斯一隊之後，他的表現基本上快封神了。

籃網隊又獲勝了，而且擊敗了掘金。這可是一個含金量非常高的對手。自從歐文高掛免戰牌之後，籃網隊的戰績還可以。他們最近12場比賽贏下9場比賽，其中不乏步行者、綠軍、熱火和掘金這樣的強隊。丁威迪最近發揮非常搶眼。而此時歐文就有些尷尬了。籃網隊有沒有他並沒有區別。

根據教練透漏，歐文有可能在聖誕節前復出。歐文一旦付出，他和丁威迪如何共存就成為教練組需要思考的問題。如果籃網隊戰績還不如以前，那麼歐文就成為了背鍋俠。這已經不是第一次了。歐文在凱爾特人的時候也是這樣。綠軍沒有歐文的情況下戰績反而更好。

這就是歐文。曾經，他想向世人證明自己。但是現在看來，他離開詹姆斯之後基本上什麼也不是了。歐文沒有過硬的帶隊成績，也沒有在季後賽中證明自己。凱爾特人可是傳統強隊啊！主教練史蒂文斯也是一個睿智的教練。但是偏偏球隊戰績一般。歐文在籃網有可能再次受到質疑。

有人失意就有人得意。湖人對陣森林狼的比賽中，戴維斯全場得到50分7籃板6助攻，投籃達到了29中20。最關鍵的是湖人贏球了，戴維斯再也不是空砍群群主了。這就是詹姆斯帶來的變化。老詹本場比賽也得到32分13助攻的大兩雙數據。兩個人合力擊垮了森林狼隊。

截止到目前為止，湖人一共取得21勝3負的聯盟第一戰績。這就是詹姆斯的帶隊力量。在詹姆斯身邊，濃眉哥基本上快要封神了。這就是戴維斯為什麼要來湖人的原因吧、鵜鶘隊時期，他贏得了數據卻輸掉了比賽。也許總冠軍才是他最終的追求。

這就是詹姆斯的統治力！他能讓身邊人變得更好。現在想想，歐文當初離開詹姆斯真的對嗎？
