“國家隊版”天津女排出征世俱杯，朱婷想讓父母現場助陣

2019-12-02 0條評論 23次閱讀 0人點贊

摘要

2019年女排世俱杯即將於12月3日-8日在浙江紹興進行，包括廣東恆大隊，以及朱婷領銜的天津隊在內的8支世界排壇勁旅，將力爭佳績。而此前馳援天津的國家隊名將中，除袁心玥外，張常寧、龔翔宇、王夢潔，還沒有現身天津隊訓練場。朱婷想讓父母現場助陣對於女排世俱杯，朱婷已不陌生。

2019年女排世俱杯即將於12月3日-8日在浙江紹興進行，包括廣東恆大隊，以及朱婷領銜的天津隊在內的8支世界排壇勁旅，將力爭佳績。而此前馳援天津的國家隊名將中，除袁心玥外，張常寧、龔翔宇、王夢潔，還沒有現身天津隊訓練場。

朱婷想讓父母現場助陣

對於女排世俱杯，朱婷已不陌生。今年回到國內打球的朱婷，最希望有家人在觀眾席為自己助威。

此前兩屆世俱杯，朱婷都是代表土耳其瓦基弗銀行隊出戰，也已經連奪兩屆MVP。今年回歸中國排球聯賽，朱婷任務不輕，除了要幫助天津隊登頂，也要力爭三奪MVP。

今年回歸中國女排聯賽，朱婷最希望有家人在觀眾席為自己助威。朱婷在前幾天和父母的視頻聊天中，就提出想請他們前來紹興觀賽。雖然朱婷爸爸媽媽確實希望來現場為女兒加油，卻也害怕影響女兒備戰和比賽。「不想影響孩子打比賽，我們在家看電視也中，還沒最後定，看情況吧。」朱婷爸爸朱安亮接受採訪時表示。

據悉朱婷今年回歸國內打球，和家人在比賽期的見面次數其實不多。今年女排聯賽，天津女排來到許昌客場迎戰河南女排，朱婷的爸媽才趕到許昌去匆匆見了女兒一眼。

天津隊「小國家隊」陣容縮水

雖然在上個月世俱杯官網，張常寧、龔翔宇、袁心玥和王夢潔還作為臨時轉會球員，出現在天津女排的世俱杯出場陣容名單。但隨著比賽臨近，只有袁心玥一人與天津女排會合。

12月3日，天津隊就將與諾瓦拉隊迎來世俱杯首場小組賽。張常寧和龔翔宇至今仍未現身，與天津女排此前公布的參賽名單形成對比。之前張常寧、龔翔宇、袁心玥和王夢潔臨時轉會天津，加上目前天津隊擁有的朱婷、李盈瑩、姚迪等國家隊名將，外界戲稱天津女排組成了一支小國家隊。

顯然，天津女排引進強援，志在本次女排世俱杯上創造歷史。但世俱杯即將開打，只有袁心玥出現在天津女排的訓練場上。此前的排超聯賽，江蘇女排0比3不敵廣東恆大女排。比賽中張常寧和龔翔宇均狀態低迷，兩人得分都沒有上雙。賽後有消息稱，張常寧和龔翔宇都是感冒發燒中上場比賽。

張常寧和龔翔宇如果缺席世俱杯，天津女排的豪華陣容也就隨之縮水。王夢潔剛剛憑藉世界盃上的表現，被評為世界最佳自由人；張常寧與朱婷組成了中國女排網前最佳對角；而龔翔宇自從里約奧運會之後，已在國家隊坐穩了主力接應的位置。

王夢潔

儘管如此，天津隊依然擁有朱婷、李盈瑩、袁心玥、姚迪、王媛媛等國手。主教練陳友泉依然可以派出清一色由國手組成的首發陣容。除了包打主攻線的朱婷和李盈瑩，副攻線將交給國家隊第一副攻袁心玥，和天津第一副攻王媛媛聯手； 國家隊替補二傳、天津主力二傳姚迪也與朱婷配合日益默契。王夢潔如果火線馳援，將補全天津隊的防線。

朱婷第三座MVP不好拿

對朱婷來說，女排世俱杯的MVP已不陌生。她幫助瓦基弗銀行在2017年和2018年連續問鼎，朱婷也成就MVP兩連冠。但今年朱婷想要三度捧起MVP，就要幫助天津隊首次奪冠。

今年女排世俱杯參賽的八支隊伍，中國、土耳其、義大利和巴西各有兩隊出戰。除了天津隊和恆大隊主場作戰，土耳其聯賽兩豪門瓦基弗銀行與伊薩奇巴希，分別三次和兩次拿到過世俱杯冠軍；科內利亞諾作為義大利一流強隊，本賽季引入強力接應埃格努，也加入冠軍的爭奪之列。

小組賽，天津與瓦基弗銀行、諾瓦拉和海灘隊同組。朱婷本賽季回國征戰排超，就要從小組賽與老東家過招。本賽季瓦基弗銀行在失去朱婷後，外援大換血，加比、哈克和奧傑年諾維奇成為隊中三大新援。小組賽與天津交手，古德蒂勢必會對朱婷重點盯防，這場對決很可能決定小組第一的歸屬。

相比之下，廣東恆大整體實力稍遜，2013年時該隊出戰世俱杯時曾引入多名國手拿到季軍，斬獲中國在世俱杯唯一的獎牌。本屆比賽，恆大從山東招募楊涵玉、杜清清和梅笑寒，陣容進行補強。

顯然，MVP的爭奪將十分激烈。朱婷、伊薩奇巴希的博斯科維奇、諾瓦拉的布拉科塞維奇，以及米納斯的謝拉，都是MVP的有力競爭者。而想要在世俱杯上拿下MVP，首先要帶領隊伍贏得冠軍。第五次參加世俱杯的朱婷想完成MVP三連莊，就必須帶領天津隊拿下最後的冠軍。
