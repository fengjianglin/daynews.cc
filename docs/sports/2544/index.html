國家運動員們雖然表面風光無限，但其實她們經歷了很多常人沒有吃過的苦，比如中國女女排成員顏妮。顏妮出生於1987年，如今已經32歲了，她是中國女排的主力副攻。身高193cm的顏妮，因為身高的優勢讓她早早就開始接觸排球，經過了多年的訓練後顏妮在排球項目，已經有了自己的一席之位。

在比賽中顏妮的移動和攔網都非常出色，在2009年第11屆全運會中，顏妮獲得了比賽的亞軍，從此她便與排球結下了不解之緣；

在2014年的亞運會上，顏妮作為主力副攻，她幫助中國女排獲得了亞軍，這對於顏妮來說，已經是個不錯的成績了；

2015顏妮憑藉著自己的努力進入女排世界盃比賽，並且跟隨中國女排一起奪得了冠軍的稱號；在同年的女排亞錦賽中，顏妮作為一名主力副攻幫助中國女排獲得了冠軍；

2016年的里約奧運會上，顏妮與中國女排一起再次摘下了冠軍的榮譽稱號！不僅是在團隊上取得優異的成績，在個人的榮譽上顏妮也是碩果累累：

2016年5月，顏妮獲得「國際級運動健將」的稱號；除此之外，顏妮還獲得了第四屆女排亞洲杯「最具有價值球員」獎、以及第四屆女排亞洲杯「最佳副攻」獎，各種獎項證明了顏妮是一位非常努力的運動員，這也是對她實力的肯定。

如今中國女排的隊伍不斷強大，也不斷有新的女排新星加入，顏妮作為中國女排的老成員，卻有了身體上的疼痛。無論是在哪個體育項目中，運動員的身體情況是非常重要的，她們的身體狀況決定了她們是否能夠以充沛的精力站在賽場上。各種榮譽的背後是無數的跌倒、無數次的受傷；正是因為她們內心的執著才讓中國女排站在了這麼高的位置。32歲的顏妮身體受傷，腰部纏滿了繃帶，但是她依然堅定的站在比賽場上，她用自己的行動再一次證明了自己對排球的熱愛，同時也讓我們看到了她的努力。雖然中國女排不斷有新的成員加入，但是顏妮已經算得上是一名女排老將了，她與中國女排配合的默契度自然是非常高的，帶傷作戰的顏妮，只有在賽場上她才能夠獲得快樂，也許這就是女排精神的一種表現吧！
