不滿隊員鬆懈，雅尼斯暫停爆粗口，嘗試死亡五小他成首鋼追夢格林

天天要聞
2019-12-11

​北京首鋼主場和廣州龍獅的比賽，賽前被認為是一場懸念不大的比賽。一周之前，首鋼剛在客場輕取對手。

首節廣州龍獅主教練胡安派出了雙外援，抱著有棗沒棗先摟一杆子的心理，試探一下一場激戰過後可能會放鬆的對手。但首鋼進入狀態很快，內外結合，首節就建立了11分的領先優勢。

第二節雙方都使用雙外援後，首鋼一度將分差拉大到21分。半場結束時，首鋼59-42領先17分。看到場面如預期中演進，腰部不適的雅尼斯坐上了板凳席，由助理教練哈里斯指揮比賽。

​第三節比賽，首鋼雙外援對廣州單外援。本以為可以輕鬆收下比賽，提前打卡下班，卻突然出現波折。廣州內線外援尼克爾森連續攻擊得分，在內線如若無人之境。中鋒孫鳴陽也在外線連續命中三分，加上陳盈駿里突外投，廣州一度將分差縮小至個位數。

看到自己的球員鬆懈的表現，雅尼斯請求了暫停。暫停時，雅尼斯多次使用了F打頭的髒字，加強自己憤怒的情緒和語氣。但在暫停結束後，首鋼球員似乎還沒能立刻緊起來，防守還是連續出現漏人。急得雅尼斯在場邊狠狠的跺腳。

比賽末節，首鋼還是未能一鼓作氣讓對手提前繳械。雖然進攻一直有穩定的輸出，但缺乏強度的防守也給了廣州連續得分的機會。

​比賽還剩3分鐘，110-96領先14分。雅尼斯認為廣州沒有反撲逆轉比賽的能力，用球隊第四控衛王旭替下朱彥西，打起了死亡五小。三名控衛林書豪、劉曉宇和王旭，加上分衛周儀翔，和內線常林，這套陣容的平均身高只有191cm。

比起前任閔鹿蕾，雅尼斯在過去兩個賽季中使用新人不可謂不大膽。在閔鹿蕾手下兩個賽季只有1次出場紀錄的王旭，在雅尼斯執教的兩個賽季中出場50次，上賽季甚至進入了常規輪換陣容。另一名身體天賦平平的鋒線張卓，憑藉著勤奮，在雅尼斯手下曾多次擔任首發。

雅尼斯對年輕球員的容錯度很高，同時也會積極調動板凳末端球員的積極性，讓防守薄弱的段江鵬和張才仁積極擁抱全新的角色。本場比賽，由於對手實力不強，張才仁沒有報名，雅尼斯把機會留給了王旭。由於林書豪的到來，加上方碩和劉曉宇保持健康，王旭本賽季的出場機會非常少，此前只打了兩場比賽。雅尼斯為了讓所有球員都保持對比賽的渴望，在此前與八一和廣州比賽時都對陣容進行了輪換。

首鋼在比賽最後幾分鐘排出的小個陣容中，常林扮演類似勇士隊中追夢格林的角色。但這套陣容在最後的時間裡打得並不好。最終比分定格在112-104，首鋼以8分優勢取勝，收穫三連勝。下一輪比賽，兩天後首鋼將遠赴客場挑戰凱撒接手後狀態提升的福建隊。
