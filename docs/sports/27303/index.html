熱刺終於不再摳了？列維作出讓步的信號：托比續約周薪翻3倍

2019-12-21 1,434條評論 3,766次閱讀 1730人點贊

據熱刺官方消息，俱樂部剛剛與球隊中衛托比-阿爾德韋雷爾德完成續約，雙方將合作到2023年。

此前，托比與熱刺的合約將於明年夏天到期，多家豪門對於這名比利時中衛都頗感興趣，而托比本人也有意離開熱刺尋求新的挑戰。不過這一切在前不久發生了轉變，原因是穆里尼奧來到了白鹿巷。

據悉，在穆里尼奧上任以後，葡萄牙人就開始了自己的遊說之旅，最終在穆里尼奧的促成下，雙方達成了續約。在完成續約之後，托比第一時間表達了對穆帥的欣賞：

「很榮幸能與贏得如此多榮譽的主帥共事，穆里尼奧教會了我很多新的東西，我會持續學習並提高自己。」

在托比完成續約之後，倫敦標準晚報披露了他的工資細節，據悉比利時中衛的周資從此前的5萬英鎊漲到了15萬英鎊。很顯然，雙方扯皮了半年之久的薪資討論最終在穆里尼奧的介入下達成了一致，一向摳唆的列維這次終於作出了讓步。

事實上由於薪資問題想要離開熱刺的球員並不在少數，埃里克森就是其中一個。很難想像這名英超頂級中場目前的周薪僅有7.5萬英鎊，也許單純的數字比較難有概念。我們來做一個簡單的對比，在曼聯，中場核心博格巴的周薪是27萬英鎊，馬塔和馬蒂奇的周薪都有16萬和12萬英鎊，即便是遊離於球隊主力之外的羅霍都能拿到8萬周薪。7.5萬英鎊的周薪對於熱刺的中場核心來說實在是有點不夠看。

不過這一次即便列維願意讓步了，埃里克森也不願意繼續待在球隊。從穆里尼奧執教球隊之後，埃里克森的上場時間就直線下降，根據筆者不負責的猜測，埃里克森應該已經和穆里尼奧有過表態，這個隊是不待了。

事實上，熱刺球員鬧加薪已經不是第一次了，早在17年，沃克轉會曼城之後，熱刺更衣室就有點「大廈將傾」的味道。當時轉投曼城的沃克周薪一下暴漲到了15萬周薪，比當時熱刺隊內頭牌凱恩的12萬周薪還要高。對此丹尼-羅斯就在面對媒體時表達了加薪的強烈意願，然而這件事最終以羅斯本人受罰結束。

時隔2年多，目前熱刺隊內仍然維持著低薪的架構，除了頭牌凱恩的周薪達到20萬英鎊以外，沒有哪名球員的周薪能夠超過10萬英鎊。因此，在上賽季球隊闖進歐冠半決賽以後，球員的鬧加薪情緒是一日高過一日，最終這樣的情緒成為了球隊在球場上的表現，執教多年的波切蒂諾被迫離開。

因此本次托比的續約或許可以被視為列維向熱刺隊內發表的一次有力宣言，表明了自己的讓步和野心，相信熱刺眾將在這樣的激勵下能夠重新找回狀態，而隊內的加薪潮恐怕將會在不久就將襲來。
