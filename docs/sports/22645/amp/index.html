巴薩0-0皇馬仍領跑積分榜，貝爾進球被吹

天天要聞
2019-12-19

北京時間12月19日凌晨03:00，2019-20賽季西甲聯賽第10輪的一場補賽，巴塞羅那坐鎮諾坎普球場迎戰皇家馬德里。這是歷史上第243場國家德比對話，同時也是一場榜首大戰。

上半場，皮克、拉莫斯先後門線關鍵解圍，阿爾巴失良機，雙方互交白卷，易邊再戰，貝爾進球被吹。

最終全場比賽結束，巴薩0-0皇馬。

【比賽亮點】

●新紀錄！拉莫斯迎來自己職業生涯第43次西班牙國家德比，超越桑奇斯、亨托和哈維(均為42次)，成為國家德比歷史第一人！

●17歲48天，法蒂成為國家德比歷史上第二年輕的出場球員。最年輕的球員是16歲278天的文森特-馬丁內斯，他於1941年10月19日在一場國家德比出場。

【精彩回放】

第11分鐘，伊斯科腳後跟做球，本澤馬禁區內一腳低射被特爾施特根沒收。

第18分鐘，卡瓦哈爾右路45度起腳傳中，卡塞米羅頭槌攻門被皮克門線處關鍵解圍。

第26分鐘，皇馬右側角球開出，瓦拉內頭槌攻門頂高了。

第27分鐘，卡塞米羅前場右側一腳遠射，被特爾施特根飛身撲出。

第31分鐘，巴薩前場進攻機會，阿爾巴左路傳中被庫爾圖瓦擊出，梅西跟進抽射被拉莫斯門線解圍。


第32分鐘，巴爾韋德外圍一腳遠射偏出。

第41分鐘，梅西45度起腳將球吊入禁區，阿爾巴插上禁區左側單刀機會，打偏了。

第44分鐘，皇馬左路傳中，朗格萊頭槌解圍不遠，巴爾韋德一腳凌空抽射被特爾施特根沒收。

諾坎普球場。

第58分鐘，格里茲曼禁區左側橫傳，梅西門前沒停好球，錯失良機。

第65分鐘，卡瓦哈爾右路45度傳中，後點本澤馬頭槌攻門擊中邊網。

第67分鐘，皇馬反擊機會，貝爾韋德中路分球，禁區左側貝爾爆射打在邊網上。

第72分鐘，皇馬中路挑傳，門迪禁區左側橫傳，貝爾門前推射破門，但門迪接球時處在越位位置，進球無效。



第81分鐘，門迪左路傳中，朗格萊和貝爾頭碰頭撞在一起。

【出場陣容】

巴薩首發：1-特爾施特根；2-塞梅多（55』22-比達爾），3-皮克，15-朗格萊，18-阿爾巴；21-德容，5-布斯克茨，20-羅貝托；10-梅西，9-蘇亞雷斯，17-格里茲曼（84『31-法蒂）

皇馬首發：13-庫爾圖瓦；2-卡瓦哈爾，4-拉莫斯，5-瓦拉內，23-門迪；8-克羅斯，14-卡塞米羅，15-巴爾韋德（80』10-莫德里奇），22-伊斯科（80『27-羅德里戈）；9-本澤馬，11-貝爾
