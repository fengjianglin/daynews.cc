史上最"獨"外援華麗轉身，那個晚上他跟鞏曉彬聊了什麼？

2019-12-16 652條評論 1,936次閱讀 927人點贊

116:107，誰都沒有想到，山東西王能在客場戰勝新疆！畢竟，在此前的17年中，山東大漢在天山腳下都沒有笑著離開過啊！就連氣勢最盛的2012-13賽季，還不是照舊沒有脾氣？

誰也不會想到，新疆會在自己的主場輸球。他們本賽季的主場保持全勝，即便缺少了發動機費爾德，可他們依然是CBA最頂尖的球隊，面對剛剛走出了低谷的山東，有什麼理由輸球呢？

但是，在這麼一個不經意的時間，困擾山東這麼多年的魔咒，竟然被掃除了！這個結果，著實出人意料，也著實讓山東球迷振奮。賽後，陶漢林在微博上一吐為快：17年的等待，我們這代人完成了！這句話，道出了全體球員的心聲。

上賽季率隊打進了總決賽的阿的江，將失敗的原因歸結為費爾德的缺陣，以及全隊的輕敵思想。其實，他忽略了一點：現在的山東，已經理順了各種關係，找到了正確的軌道，在心態和氣勢上不輸任何對手！

平心而論，跟廣東、新疆、遼寧等具備奪冠實力的球隊相比，山東在整體實力上有所不濟，但是這支隊伍在防守端全民皆兵，梅斯玩命的姿態無時無刻不在影響球隊；在進攻端以哈德森為核心，賈誠、陶漢林、李敬宇、王汝恆等本土球員找回了自信。態度和心態可以彌補實力的不足，不然的話，山東怎會取得六連勝？

在山東鏗鏘殺伐的道路上，有兩個標誌性的事件不得不提。

第一是鞏曉彬的五十歲生日。那個晚上，山東在客場絕殺了江蘇。在此之前，球隊在進攻端過於依賴哈德森，本土球員有勁使不出來，該贏的球拿不下來，鞏曉彬背負著巨大的壓力，難免心浮氣躁。過了生日之後，鞏曉彬彷彿換了個人，在比賽中「穩」了下來，更多地給球員以鼓勵，在戰術安排上也常有點睛之筆。

通過對比數據不難發現，在六連勝期間，鞏曉彬的用人思路比較穩定，輪轉陣容固定在8人左右，每名球員在球隊中扮演什麼角色，什麼時候需要做什麼事情，自己都有一個清晰的定位，到了場上更容易發揮出真實水平。而在此前的比賽中，鞏曉彬頻繁調試陣容，在比賽中大面積用人，這場比賽打主力，下一場可能打不了幾分鐘，每名隊員的心裡都沒有底，上了場容易發「毛」。

第二是哈德森的離隊事件。山東打完江蘇之後，接著客場對陣浙江，哈德森的狀態不好，單打獨鬥較多，第四節被摁在了板凳上。此前不管在哪支球隊，老哈都是絕對主力，從未享受過這種冷遇，竟然一氣之下脫離球隊，提前返回了濟南！對於這種違反隊規隊紀的行為，鞏曉彬處理得很到位，沒有跟哈德森硬碰硬，給他留了冷靜窗口期之後，兩人進行了深入溝通。這個小插曲之後，哈德森跟全隊的融合漸入佳境。

在山東六連勝期間，哈德森場均出手三分球10.5次，命中率高達54%，他的場均助攻是8.8次。而此前的10場比賽，哈德森場均出手三分球13.3次，命中率只有31.6%，場均助攻為6.6次。通過對比即可發現，經歷了「離隊門」之後，哈德森更加註重跟隊友的配合，在出手投籃方面變得謹慎了很多，不像之前那麼隨意。

可以說，鞏曉彬生日之夜的「轉身」，以及哈德森成功「被改造」，是山東本賽季至今最大的亮點，也可能是整個賽季的轉折點。
