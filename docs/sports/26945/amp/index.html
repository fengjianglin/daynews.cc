伊藤美誠，你聽到了嗎？這才是中國乒乓王者的聲音：你太天真了

天天要聞
2019-12-23

2019年即將過去，中國各體育項目也基本上結束了一年的征程，中國的國球乒乓球也不例外。而2020年對於中國體育人來說，又是一個特殊的年份——奧運年。奧運會對於從事體育運動的運動員來說，其意義不言而喻。

對於中國的國球乒乓球而言，明年的東京奧運會又有了新的挑戰。據了解，明年的東京奧運會上，乒乓球運動將新設一項混雙比賽的項目。這樣，乒乓球項目的金牌總數也由過去的四枚增加到五枚。並且根據賽程安排，混雙還是乒乓球比賽中最先決出金牌的項目。誰能在明年的東京奧運會上獲得混雙金牌，不僅將為本隊的征程取得開門紅，對乒乓球項目餘下的比賽將產生重大影響，更是將載入乒乓球歷史史冊——奧運史上的第一塊混雙金牌。

而提出增加混雙項目的不是乒乓球王國中國，而是明年東京奧運會的主辦國日本。從這一提議來看，球迷們都能感受到日本乒協對此項目金牌的志在必得，不然他們也不會費儘力氣來增加這一項目。

面對於中國來說，在東京奧運會上最大目標肯定是包攬乒乓球的全部五塊金牌。對於其他項目而言，中國隊都有很大的把握，但對混雙這一塊金牌，卻有著太多不確定的因素。首先在於乒乓球運動員的重視程度，乒乓球運動員絕大多數的最終目標是爭奪大滿貫，團體和單打，但對於混雙卻考慮得並不多。其次在於混雙的賽制不同於其他比賽，混雙比賽只允許一個國家或地區的協會只能報一對選手參賽，也就是說，不管中國乒乓球運動員的實力有多強大，但在混雙比賽中，也只能有一對選手應戰。這樣，就增加了比賽的不確定性，畢竟比賽中狀態也很重要。

最後，既然日本主動提出增設這一項目，他們肯定不是為了給中國隊增加一塊金牌，而是想著給自己增加一塊金牌。他們作為東道主，自然會想盡一切辦法來得到這塊金牌。

為了獲得這一塊金牌，日本這些日子以來，一直在精心訓練他們的混雙。日本隊在嘗試張本智和/平野美宇，伊藤美誠/森園政崇及吉村真晴/石川佳純等各種組合搭檔後，終於排列出了水谷隼/伊藤美誠這對組合，他們有很大可能成為日本奧運會上混雙的選手。

而日前，伊藤美誠在獲得2019年日本體壇十大最活躍運動員的稱號後，竟狂妄地表示要在東京奧運會上成為不敗選手，以不敗戰績率領日本隊奪冠，打破中國隊的壟斷。這個堪稱給中國乒乓球的戰書，當真是狂妄得不知自己的斤兩。

不過，日本隊再狂妄，在昔日的世界乒乓球一姐，乒乓皇后鄧亞萍看來，都不值得一提。

日前，鄧亞萍在參加央視座談節目時，談了談自己對中國乒乓球明年參加奧運會的看法。在談到日本隊想奪得奧運史上第一塊混雙金牌時，她霸氣表示：「凡是被國乒盯上的，無論是金牌還是人，都沒跑了！」央視體育主持人邵聖懿聽了鄧亞萍這話，也開心地表示：本來球迷們心裡還沒準兒，但有鄧亞萍這番話，大家的心裡也就踏實了！

不過，鄧亞萍在說出這個霸氣的話後，國乒多年的隨隊記者李武軍卻笑著給鄧亞萍戳了一下「傷疤」： 你職業生涯里沒有拿過一次混雙金牌！

聽了這話，鄧亞萍有些無奈地表示：「被你戳穿了！我的最大痛點，武軍你狠！」不過，鄧亞萍也透露，當年大家都不重視混雙，一般都是臨近比賽了，才會抽出一個小時左右的時間臨時配對訓練一下。

但如今，混雙比賽已經正式成為奧運比賽項目，且是第一個決出勝負的比賽項目。它的地位早已不是當年，國乒對這項目非常重視，也基本上確定了許昕/劉詩雯這對混雙組合。並且正如鄧亞萍所說：凡是被國乒盯上的，無論是金牌還是人，都沒跑了！不知日本隊在聽到鄧亞萍這個霸氣的話語時，心裡會怎麼想，是不是感覺心裡原來有底的混雙，瞬間就少了七分底氣。
