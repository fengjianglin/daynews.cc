逗!郎平訓練結束訓話:去談戀愛吧,曾春蕾徐雲麗懵逼,隨後一鬨而散

2019-12-24 1,490條評論 4,137次閱讀 1367人點贊

12月24日消息，今天，電影中國女排放出預告片，其中，中國女排主教練郎平對於愛徒朱婷的一番心靈拷問，以及打巴西隊1/4決賽前給朱婷發的激勵簡訊，師徒之情令人感動。

​一上來就是里約奧運會小組賽，中國女排連輸荷蘭、塞爾維亞、美國等強敵，5戰2勝3負以小組第4的成績面前晉級八強。

郎平當時受到了國內媒體的眾多「拷問」，鞏俐扮演的郎導緩緩走下台階，面對黑壓壓的記者輪番炮轟。

​有記者問，「主教練你有什麼要說的嗎？」

還有人直接在新聞發布會上開炮，「咱們這場打得這麼難堪，您有考慮換個戰術嗎？」

​郎導露出了自信的微笑。

​1/4決賽決戰東道主巴西隊之前，郎導給隊員們訓話，問大家，「我今天還得問一句，你們愛排球嗎？」

​隨後，閃回到了郎導剛回國家隊開始選拔隊員的場景。

當時張常寧還在打沙灘排球。

​郎導和2004雅典奧運冠軍李珊扮演的助理教練賴亞文一同觀看了張常寧的比賽，郎導問，「那個女孩叫什麼名字？」賴導回答，「張常寧。」「我們要了。」

​「這二傳什麼名字？」「丁霞。」「我們要了。」

​賴導介紹，「這個是惠若琪，陳忠和指導發掘的核心球員。」

​郎導談到採取大國家隊集訓時表示，「以前國家隊是12個人，現在我可能選30個人，30個如果不夠，那就100個。」

​隨後，又回到打巴西隊之前的靈魂對話，郎導問朱婷，「為什麼打球？」

​朱婷大聲回答，「為我爸媽！」

​郎導說，「那你永遠的打不出來！再想。」

​朱婷想了想又說，「成為你！」

​郎導將一個排球砸向地下，「那你就搞錯了。」

​郎導將弟子們帶回了郴州訓練基地，這裡是當年老女排魔鬼訓練的地方。

​留下過女排姑娘們摸爬滾打，傷痕纍纍的過往，以及金燦燦的5連冠。

​中國女排姑娘們重回老女排故地，充滿了新奇。

​隊長惠若琪帶著張常寧、劉曉彤等隊員聆聽郎導訓話。

​郎導說，「其實，我就是放不下。我們這代人都是苦過來的，你們應該打出你們自己的排球。放開了打，豁出去打。我和你們在一起。」

​最後，郎導在打巴西之前給朱婷發簡訊，「你是我見過最棒的球員，你不用成為我，你只用成為你自己。」

​朱婷扣球得分，中國女排淘汰巴西，最終2016年里約奧運奪冠，時隔12年重奪奧運冠軍。

​有趣的是，一次訓練結束後，郎導對大家說，「今天就練到這，去談戀愛吧。」

​徐雲麗、曾春蕾等隊員愣住了，簡直不敢相信自己的耳朵。

​隨後如獲赦令，大家一鬨而散。十分逗比。
