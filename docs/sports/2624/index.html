紅軍沒冠軍相？鬼才信！輪換半隊還能狂勝 渣叔一隻手已摸到獎盃

2019-12-05 0條評論 21次閱讀 0人點贊

摘要

​英超第15輪比賽，利物浦主場對壘埃弗頓，雖然是一場默西塞德郡德比，但主帥克洛普卻大膽進行了人員輪換，但即便如此，紅軍依舊5-2橫掃太妃糖，達成了聯賽32場不敗的超長紀錄。上個賽季利物浦在歐冠賽場掄元，同時在英超有著非常出色的發揮，最終在與曼城的爭冠賽跑中屈居次席。

​英超第15輪比賽，利物浦主場對壘埃弗頓，雖然是一場默西塞德郡德比，但主帥克洛普卻大膽進行了人員輪換，但即便如此，紅軍依舊5-2橫掃太妃糖，達成了聯賽32場不敗的超長紀錄。

上個賽季利物浦在歐冠賽場掄元，同時在英超有著非常出色的發揮，最終在與曼城的爭冠賽跑中屈居次席。進入新賽季以後，球隊在聯賽依舊高效，前14輪取得13勝1平的不敗成績，以40個積分笑傲榜首，領先優勢巨大。

雖是默西塞德郡勢不兩立的短兵相接，紅軍卻進行了大幅度的輪換，除阿利松因停賽無法出場外，拉拉納、米爾納、沙奇里、奧里吉這些並非常規首發的球員，全都得到了寶貴的出場機會。

雖然狀態相差懸殊，但考慮到德比因素的加成，不少球迷仍為紅軍捏了一把汗——擔任本場解說嘉賓的克勞奇也直言「如果我是埃弗頓球員，看到利物浦的首發陣容絕對會不爽，明擺著就是看不起人嘛。」

比賽進程卻遠比我們預想的順利，渣叔的球隊用水銀瀉地的進攻打消了所有擔憂，最終以5-2擊敗對手，拿到了第234次默郡德比的勝利。大幅輪換仍獲一場狂勝，雖然沒能零封對手，但球隊雄渾的實力和板凳厚度已經難掩冠軍相。

聯想到上個賽季大幅領先的局面下痛失英超冠軍，被外界詬病「沒有冠軍相」，經過長時間卧薪嘗膽後，渣叔的球隊已經相當有說服力。

賽程近半，領先第二名8分，如果說沒想過捧起捧杯，估計連利物浦自己都不信。

需要指出的是算上本場比賽，紅軍在12月將經歷26天踢9場的超恐怖賽程，如果能保證這段時間內穩定度過（不失太多分、且沒大面積傷病），那麼可以說紅軍一隻手已經觸到了覬覦已久的英超冠軍獎盃。

當然，經歷過伊斯坦布爾奇蹟的他們也深刻明白「先胖不是胖，後胖壓倒炕」的道理，道阻且長，紅軍永不獨行。
