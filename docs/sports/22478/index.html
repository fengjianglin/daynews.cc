太調皮！國乒前世界第一與劉詩雯有趣互動，一旁的許昕一臉懵圈

2019-12-20 2,414條評論 3,845次閱讀 2034人點贊

眾所周知，乒乓球是我國的國球，作為體育強項自然在諸多獎項上也是近乎包攬，充分展現了一個乒乓球強國的形象。據近日消息，在日前剛剛舉行了本年度的國際乒聯比賽的頒獎盛典。一方面來說，這項乒壇盛事本就讓不少球迷們表示激動，紛紛表示這下不僅可以看到我國的乒壇大咖，就連其他世界乒壇的大咖們也都一個不落了。而另一方面，當真的看了直播盛況以及我國的乒壇大咖們有趣互的某些路透細節動圖之後動球迷就更加激動了，儘管口中說著：「真幼稚」，但對他們的喜愛是增添了不少！

值得注意的是，在當天的頒獎典禮上我國乒乓球隊的馬龍與劉詩雯這兩人的逗趣互動堪稱球迷們口中的經典！在本屆國際乒聯頒獎上，這兩人分別是當選了年度最佳男女運動員獎項,並且在今年的含金量最高的世錦賽上，這兩人同樣奪得了單打冠軍的好成績。只是沒想到，賽場上威風凜凜，頒獎台上嚴肅滿滿，但是在台下卻是童心未泯。那球迷們所說，這兩人甚至是還有點搞笑！

當天發生了這樣逗趣的一幕：坐在台下的時候，許昕、劉詩雯、馬龍和丁寧挨著依次坐一起。幾乎所有人都在認真關注活動的時候，只見馬龍拿起了水瓶應該是想喝水。只是沒想到沒一會馬龍就開始調皮起來了，只見他把水瓶上的標籤撕掉了，然後趁著別人不注意把標籤貼在了劉詩雯的衣服上。而發現異動的劉詩雯轉過頭馬上就發動了反擊，把衣服上的標籤撕掉，往馬龍的羽絨服上貼上去，還不忘按兩下粘牢一點。這兩人的大動作讓兩側的丁寧和許昕發現了，當看到這兩人的舉動，丁寧和許昕是一臉懵，那無奈的樣子似乎在感嘆：這倆怎麼這麼幼稚呢。

值得一提的是，在本年此前的世錦賽中劉詩雯和馬龍就已經取得了單打冠軍的好成績，而此次又在國際乒聯中取得了最佳運動員的殊榮。在本次比賽中，總決賽上馬龍取得了男單亞軍，而許昕和劉詩雯的組合取得了混雙冠軍，以及進擊東京奧運會的混雙資格。不得不說，這是我國乒乓球領域混雙的又一次實力證明，這或是未來我國首次獲得奧運會混雙冠軍的最大希望。如此看來，或許也正是因為這兩人在本年度所取得的斐然成績吧，所以才能在台下如此自如吧。

不過反觀一直以來作為老對手的日本乒乓球隊員此次收穫就非常小了！名手石川佳純和平野美宇在總決賽剛開始的時候就慘遭劉詩雯王藝迪橫掃，最後只得一輪游就匆匆出局。從近日傳出的相關消息來看，儘管失去了世界乒聯的比賽奪冠機會，但是石川佳純還算比較幸運地獲得了奧運會單打的資格機會，現目前已經回到日本開啟短暫的休息調整。從近日曝光拍攝的廣告來看，石川佳純美人胚子盡顯，簡直是宛如鄰家少女、溫婉可愛，這也不由得吸引了更多支持她的中國粉絲。不管怎麼說，距離2020奧運會時間說長也不長說短也不短，各位男神女神加油啦！
