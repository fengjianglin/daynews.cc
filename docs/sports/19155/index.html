中國男足的5位老大哥，鎖喉日本球員的排第五，范志毅排第三

2019-12-17 488條評論 4,330次閱讀 2897人點贊

雖然說我們中國足球這些年來發展不好，國足運動員們也飽受詬，但是在這麼多年的發展歷程中國還是出現了很多優秀的職業運動員的。大家都知道在球隊里必須要有一個老大哥，這個人要在球隊里扮演很重要的角色，他也是整個球隊的精神領袖。如果沒有了他，那麼球隊就很難在賽場上打出好的表現。接下來就讓小編帶著大家了解一下中國男足的5位老大哥。

第五：李瑋鋒

鎖喉日本球員的國腳排第五：作為一名足球運動員，李瑋鋒從來都是一個帶著情緒打球的人，這也讓他在球場上十分的受歡迎。當年他在賽場上鎖喉日本運動員的場景更是讓很多足球迷們歷歷在目，這也是中國足球賽場上出現的最硬氣的一個職業球員之一。

第四：鄭智

鄭智只能排第四因為前三太厲害了：作為現在中國足球運動員的一個代表人物，鄭智已經成為了人們心目中國足的最佳代表，他在賽場上也一直都是人們的榜樣，不過有的時候他犯的錯誤也會被人們忽視。雖然說有的時候鄭智得到的讚賞與他在賽場上作出的表現不是正相關的，但是職業生涯他還是中國足球的老大哥，對於中國足球來說他有著非常大的重要作用。

范志毅排第三

作為中國足球運動員的代表人物，當年的范志毅在賽場上非常的硬氣。那個時候他是中國所有足球迷心目中的偶像，而他的名言也為人們數次引用。如果在賽場上沒有的這種精神，中國足球很難在當年的發展過程中變得那麼的優秀，而我們也要感謝他的出現。

第二：魏群

能夠成為球隊的老大哥，魏群自然有著過人之處，他的兄弟義氣是出了名的好。當年在中國的足球賽場上沒有人不知道魏群的名字，而他在球場上救出球隊小弟的事情也被人們津津樂道。無論是從個人魅力方面還是從賽場實力方面，魏群都是中國足球運動員的代表人物，當時還時任國家隊隊長的馬明宇也是魏群的小弟，這也足以見得為他到底有多麼的優秀，他也絕對是中國足壇上的一位大哥級別的人物。

第一：徐弘

雖然說魏群是中國足球賽場上的老大哥，但是當年的徐弘就連魏群見了也得對他客客氣氣，畢竟人家的地位非常的高。雖然說在平時的生活中徐弘是一個低調內斂的人，但是在賽場上以及球隊的更衣室里每個人都對他非常的服氣。如果職業生涯後期沒有被坑慘的話，他在中國足球的賽場上也絕對會做出更多偉大的成就，雖然當年曾經有過一段風波，但他也絕對是球隊的老大哥。

正是有了這麼多出色的足球運動員，我們中國足球也才能在這麼多年的發展歷程中變得越來越優秀。在賽場上如果沒有這些球隊老大哥，球員們的心理將面臨著很大的壓力，而他們也很有可能在賽場上做出一些不理智的事情，所以說對於每支球隊來說擁有這些老大哥都一個非常重要的存在。
