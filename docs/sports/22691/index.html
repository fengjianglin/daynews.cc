國足45分鐘1次射正險些1-2，李鐵急了！里皮才是笑到最後大贏家？

2019-12-19 1,716條評論 4,090次閱讀 3087人點贊

北京時間12月18日，中國男足在東亞杯上的最後一場比賽開打結果半場比賽過後，中國隊僅僅一球領先於對手，就引發了球迷們的不滿。

中國隊開場之後9分鐘，由吉翔在禁區內接到了隊友傳球之後頭槌搶點破門得分，然而如果沒有這個進球的話，恐怕中國隊上半場連真正射正的幾乎都沒有，半場比賽結束之後呢，幾項數據統計很能直接說明問題。在射門上雙方居然是2:2，中國隊一共只有兩次射門，而射正的機會只有這麼一次，也就說明中國隊當時只有這一次射門是真正威脅到了對手的球門，而控球率方面雙方也達成了50%，中國隊並沒有佔據明顯的優勢。

相比之下，反而是落後的中國香港隊製造了非常明顯的威脅，當時他們有兩次射門機會都是威脅到了中國隊的球門，差點洞穿了恆大門神劉殿座的十指關，當時這兩次配合當中的中國隊的後衛都出現了盯人不緊，而且站著看對手射門的荒唐一幕，也是把劉殿座以及場邊觀戰的李鐵急得夠嗆。其中特別是對手在左路這一次的爆射，差點打出了一記非常精妙的世界波，當時中國隊的右後衛和中後衛都在原地站著看對手發力射門。看到國足這樣的比賽態度也真的是讓人頭疼，要知道這場比賽，可是中國隊的底線是真正輸不起的比賽，別說輸不起了，就是打平或者只用一個都顯得非常丟人。

知名的足球評論員董路，則是在社交媒體上向國足積極喊話，董路表示上半場兩次射門都是通過定位球獲得的，其實這也無可厚非，因為香港隊採取了陣地密集防守的策略，當時韓國隊打中國香港也是採取了定位球的方式，通過定位球得分，場面上也是不太好看，所以中國隊想要改變這一點，就必須在下半場積極的提升攻防轉換的速度，要把速度提起來，再就是踢球要踢得簡單點直接點，董路也是提出了這兩大建議，要求中國儘快的做出改變。

然而問題在於，中國隊在進攻和防守當中，確實是缺少節奏，缺少節拍，李鐵非常信任的中場悍將李行在中場，他的防守看起來不錯，但是指揮調度根本不夠格，經常連球都拿不住，而隊友之間的跑動穿插更是非常的有限。顯然看起來中國足球不僅是能力不足，態度也不足。這樣下去，中國足球可真的沒有希望了，而這3場比賽，國足也用糟糕的表現證明，世界盃冠軍大帥主動放棄了高額年薪炒了中國足球的魷魚，是有多麼的英明，他成為了幕後真正的大贏家。
