13勝12場KO的重量級“炸藥”欲在明年暴打維爾德喬書亞，強勢稱王

天天要聞
2019-12-19

【文/搏擊江湖】今天，英國不敗重量級新秀「炸藥」丹尼爾-杜布瓦（13-0，12KO）向媒體展示了自己的雄偉計劃，打算在2020年年底的時候向當今重量級兩大拳王維爾德和喬書亞發起強烈挑戰，而且絲毫不懼喬書亞和維爾德，尤其是維爾德，儘管維爾德有著天生神力，有著超高的KO率，但是杜布瓦要嘗試一下維爾德的拳頭，甚至暴打維爾德，將其趕下王位。不得不說，年僅22歲的杜布瓦確實有膽量，不愧為炸藥的威名。

杜布瓦說：「維爾德知道自己在做什麼，他的拳擊解讀能力很強，他知道如何感覺到一個拳手的存在，知道他們什麼時候遇到麻煩，什麼時候去KO對手完成比賽。他已經成為冠軍很長一段時間了，並且證明了自己，而且他正處於職業生涯的巔峰時刻。如果我們能讓這場戰鬥發生，那將是很棒的一天。 我想要那場戰鬥，我可以和任何人對抗。不管是維爾德還是喬書亞，我都有信心讓他們領教我的拳頭，我計劃明年晚些時候向重量級金腰帶發起衝擊。」

杜布瓦身高1米96，臂展1米98，保持著13戰全勝12場KO對手的完美戰績，他似乎每一場比賽都在提升自己。畢竟22歲的年紀還有很大的提升空間。杜布瓦還得到過傳奇巨星倫諾克斯-劉易斯的指導，被劉易斯看作是英國重量級的希望。目前杜布瓦還跟隨前重量級拳王金剛布魯諾一起訓練，布魯諾對杜布瓦非常支持，相信這個強壯無比的小夥子會在未來2年內超越喬書亞、維爾德和富里，成為重量級的霸主。

筆者看過杜布瓦的比賽，他有著強悍的體魄和摧枯拉朽的重拳，兩隻手同樣可怕，難能可貴的是，他有著很強悍的抗打能力，挨到重擊的同時也可以打出致命的進攻，他可能會給抗打脆弱的喬書亞帶來極大的威脅。甚至也可以給維爾德帶來強勢挑戰，不排除依靠組合拳KO維爾德和喬書亞的可能。杜布瓦目前唯獨欠缺的就是技術精細化和大戰的經驗。如果再打磨2年，等維爾德老去再打，或者把握更大，但是杜布瓦想在明年年底就挑戰維爾德喬書亞，可見他的信心超乎常人。重量級往往一拳就可以改變結果，維爾德和喬書亞面對杜布瓦，沒有必勝的把握，真打起來，有些懸了。

據了解，杜布瓦將在北京時間12月22日迎戰日本拳手藤本京太郎，這是一場12回合拳賽，雙方爭奪空缺的WBC重量級銀腰帶，還有WBO國際重量級拳王頭銜。相信杜布瓦擊敗33歲的藤本京太郎不成問題，這也是杜布瓦在2019年的收官之戰，可以為聖誕節獻禮。目前杜布瓦在重量級綜合排名第26位，已經是很靠前的一個名次了。如果他明年再打2-3場比賽，還真有資格叫板維爾德和喬書亞。

杜布瓦的身高和劉易斯一樣，都是1米96，體格強壯如山，重拳了得，完全可以向維爾德、喬書亞叫板。據說曾經在2016年擔任喬書亞陪練的時候重拳擊倒過喬書亞，他的重拳能力驚人。關鍵是22歲的年紀，未來15年，都是值得關注的重點人物。搏擊江湖原創，歡迎分享。
