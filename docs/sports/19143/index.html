詹眉打瘋了，霍華德建功又刷新紀錄！湖人6600萬神射迷失

2019-12-17 2,553條評論 4,248次閱讀 1263人點贊

12月16日，湖人隊以101比96擊敗老鷹隊。在取得了7連勝的同時，湖人隊迎來了三連客的3連勝，戰績來到了24勝3負，追平雄鹿隊，與之並列聯盟第一！

湖人隊險勝老鷹隊

本場比賽里，湖人隊外線發揮不佳，依靠著詹姆斯、安東尼戴維斯和霍華德等人的出色發揮，得以險勝對手。詹眉打瘋了，霍華德建功又刷新紀錄！湖人6600萬神射迷失！

安東尼戴維斯面對老鷹隊薄弱的內線，開始大發神威。濃眉哥轟下了27分13板3助1斷2帽的大號兩雙數據，他在進攻端雖然外線命中率不佳，但是內線的單打卻效率很高。擁有48%投籃命中率的濃眉哥，2分球是17投11中，命中率高達65%。

安東尼戴維斯

可以說，濃眉哥在內線的進攻徹底打瘋了。而在防守端，濃眉哥強勢的壓迫防守，讓對手的老鷹隊四號位賈巴里帕克只有10投4中，得到12分。場均場均得到16分，擁有51%投籃命中率的帕克，被濃眉哥限制住了。

詹姆斯則在進攻端火力全開，再次證明了他依舊是聯盟第一人。該場比賽里，詹姆斯轟下了32分13板7助1斷3帽的全能數據。詹姆斯在進攻端21投12中，命中率高達57%，三分球是10投4中，命中率高達40%。詹姆斯是全場比賽里，三分球命中次數最多的球員，也是三分球命中率最高的。

詹姆斯徹底打瘋了

可以說，詹姆斯在外線和內線同時迎來了爆發。在內線的突破與上籃，對手簡直無法防範。防守端，詹姆斯讓對位的亨特只有10投2中。詹姆斯是本場比賽的得分王、籃板王（與濃眉哥持平）、助攻王（與特雷楊持平）和蓋帽王（與霍華德持平）。可以說，詹姆斯用自己的全能表現，又一次征服了對手，拯救了球隊。並且，詹姆斯的正負值為全場最高的+10（與卡特持平）。在濃眉哥5次犯規，上場時間受限的情況下，可以說詹姆斯徹底打瘋了，以一己之力帶隊前行。

霍華德，則扮演了禁區守護神的角色。該場比賽里，霍華德繼續替補上場，他得到了2分8板1助1斷3帽的全面數據。在多次封蓋對手的同時，面對舊主霍華德在籃板球的保護方面，也做得很不錯。立足於內線防守的霍華德，讓老鷹隊的內線得分只有34分，而在他的協助之下，湖人隊的內線得分高達54分。霍華德，則內線的防守建功了。

霍華德在內線防守建功

並且，霍華德還在今天刷新了自己的蓋帽紀錄。根據數據顯示，霍華德在送出了2記蓋帽之時，他的生涯蓋帽總數已經來到了2087個，超越馬努特-波爾的2086個，來到了歷史蓋帽榜第15位。

霍華德刷新了自己的蓋帽紀錄

詹姆斯、安東尼戴維斯以及霍華德，合力砍下了59分34板11助3次搶斷7個蓋帽，他們的聯手爆發，幫助湖人隊在今天力克對手。

詹姆斯與安東尼戴維斯

需要關注的是，今天比賽里，湖人隊的三分球效率很低。全場3分球是31投5中，命中率為16.1%，為本賽季新低。如果不是詹姆斯在三分線外成為救世主的話，湖人隊的三分球命中率還會更低，而本場比賽的勝負恐怕也難以言說。

湖人隊多名神射手集體打鐵，550萬合同在身的卡魯索3分球是2投0中，970萬合同在身的布拉德利3分球是2投0中，518合同在身的隆多分球是4投1中，3000萬合同在身的丹尼格林3分球是5投0中，1600萬合同在身的波普3分球是2投0中。

湖人隊三分球低迷，詹姆斯是遮羞布

薪金超過6600萬合同的湖人隊5大射手，三分球共計15投1中，命中率僅為6.7%。而板凳席上的兩大神射手丹尼爾斯以及庫克，已經至少2場比賽沒有獲得上場時間了。
