德比之戰大輸30分！冠軍教練不敢相信，著急向球員大聲連發5問

2019-12-15 2,213條評論 3,175次閱讀 1915人點贊

CBA德比之戰，廣廈隊大輸30分，令人意外

昨天，CBA迎來一場重頭戲，浙江隊與廣廈隊上演CBA德比之戰，結果令人意外，廣廈隊在第二節比賽就完全崩盤，然而，浙江隊也沒有收手的意思，最終，廣廈隊以91：121狂輸了30分，由此一來，廣廈隊以11勝6負的成績掉出四強，而，浙江隊則以9勝7負的成績，上升到積分榜第七位。

本場比賽，對於廣廈隊而言相對比較困難，此前大外援法里德因為一些原因被球隊辭退，小外援傑克遜老婆生娃請假回了家，新外援普拉姆利剛到球隊，這是他的第一場CBA比賽，反觀，浙江隊整容齊整，狀態正佳。雙方第一節比賽，打得相當膠著，比分也是28：28戰平。第二節比賽，廣廈隊還一度領先對手7分，但，浙江隊雙外援明顯佔優，接下來，蘭茲博格大爆發，突破、二分、三分張手即來，進入了無解狀態，帶領浙江隊迅速追平，並且反超了比分。

反觀，廣廈隊此時卻出現了短路，球隊主力控衛孫銘徽連續出現無謂犯規，並多次出現失誤，分差被浙江隊拉開，半場比賽打完，浙江隊以64：44領先了廣廈隊20分。下半場比賽，廣廈隊依然沒有找到好辦法，而浙江隊也沒有要收手的意思，分差被繼續拉開，一度來到了35分。最終，廣廈隊以30分之差慘敗。

冠軍教練李春江不敢相信球隊如此崩盤，暫停時向球隊大聲連發5問

廣廈隊在近幾個賽季可以說是一支異軍突起的球隊，他們在CBA冠軍教練李春江的帶領下，成為了CBA一支強隊，在CBA2017-2018賽季，他們拿到了隊史第一個常規賽冠軍，並且，在季後賽闖進了CBA總決賽，在總決賽面對強大的遼寧隊，由於球隊外援不給力，加上內線悍將蘇若禹受傷，使他們痛失CBA總冠軍。

經過CBA總決賽的洗禮，廣廈隊年輕球員也是快速成長起來，像胡金秋、李京龍、 孫銘徽、 趙岩昊、蘇若禹、劉錚等人，逐漸成為了CBA的一股新勢力，開始嶄露頭角。本場與浙江隊的德比之戰，雖然，廣廈隊是以單外援參賽，但，以廣廈隊的整體實力，也不應該如潰敗，輸到一塌糊塗。

對此，主教練李春江也是十分不理解，甚至，很憤怒球員在場上無所欲求的表現，在第四節比賽還有4分鐘時間，眼看分差被接拉到了35分，並且，還有繼續被擴大比分的趨勢，坐在場邊的李春江也是沒有再忍，叫了一個暫停，此次叫暫停，李春江就是要給球隊訓話，對球員的防守非常不滿意，強烈要求球員不要放鬆，要球員們要加強防守，李春江對著球員怒吼，「你們怎麼防守的，讓別人想怎麼打就怎麼打，別人怎麼防你，你們能不能做到，一樣如此防別人呢？」 「你們，能不能做到！」

「能不能做到？」李春江大聲說了足足五遍，球員們也是陸續回答「能！」，甚至，已經很憤怒的李春江還爆出了粗口。從這1幕也可以看出李春江是十分著急，看到球員在場一盤散沙的表現，很是痛心，甚至，想自己上場去干。值得慶幸地是，廣廈隊球員在被李春江批了一頓後，在場上明顯有改變，造成浙江隊多次失誤，打出了一波得分小高潮，但，最後，廣廈隊還是以30分敗北。

接下來，廣廈隊最應該解決的就是外援問題

在賽季初，廣廈隊可以說是信心滿滿，球隊也是喊出了保四爭二，甚至，奪取本賽季總冠軍的口號，在外援上也是做了調整。但是，球隊在外援上依然是眼光比較差，引進的外援不管是傑克遜、杰倫-雷諾茲，還是後來名氣大的法里德，沒有一個是稱心如意的，甚至，這些外援沒有一個有攻堅能力。

相對比廣東隊馬尚、威姆斯，遼寧隊的史蒂芬森、巴斯，吉林隊的瓊斯，山西隊的富蘭克林等一流外援，廣廈隊外援的水平，最少相差一個等級。因此，接下來，廣廈隊最為關鍵的是在找到合適的外援，要找到能在關鍵時候力挽狂瀾的外援，同時，球隊的心態也是快速調整過來，只有這樣，球隊才能夠實現賽季初提出的目標。說實話，廣廈隊這批年輕球員，已經不輸與CBA其他各隊，來到了CBA一檔，假如真能找到兩個好的外援，想必廣廈隊在李春江的帶領下，完全能夠取得不錯的成績。
