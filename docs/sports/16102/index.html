大逆轉！世界第一1-3變4-3，險被00後新星掀翻，一舉動仍顯狀態差

2019-12-13 2,088條評論 3,176次閱讀 1637人點贊

北京時間2019年12月13日消息，國際乒聯年終總決賽還在繼續進行，剛剛結束的女單1/8決賽中，國乒勁敵伊藤美誠4-3險勝中國台北隊一姐鄭怡靜晉級8強。隔壁桌進行的是一場國乒內戰，世界第一陳夢交鋒國乒世青賽女單冠軍錢天一。

這位00後新星近來的表現可不俗，上次曾爆冷擊敗了師姐劉詩雯，這次同樣給仇恨夢帶來了很大的麻煩，讓陳夢在場上不停思考，所以比賽的節奏也不是很快。

前兩局比賽，2人以11-7，8-11打成1-1平，錢天一的正手爆沖還是非常猛，上次在對陣劉詩雯的時候已經露了一手了。第三局比賽，雙方繼續形成相持，從開局咬到7平。陳夢想要正手快拉遠角，立刻被錢天一快速回擺直線，比分到9平，總算出現了2人對拉的場面，陳夢先拿到局點，錢天一仍然緊咬比分，追到10平。錢天一連得2分，12-10，2-1領先陳夢。

第四局比賽，陳夢的反手失誤比較多，整個人在場上似乎沒有什麼精氣神，反而是不停地搖頭，雙方1-1開局，錢天一3-1取得領先，陳夢再次追上比分，又開始相持了，陳夢剛剛取得8-5的領先優勢，就被錢天一追到7-8，陳夢被迫叫暫停。

回到比賽，陳夢10-7拿到局點，轉眼又丟了，這樣的狀態是真的不好，坐在觀眾席的馬琳指導十分著急。司馬剛看了陳夢很多場比賽，但凡陳夢出現了這一動作-按球拍，說明狀態不好，鄧亞萍在解說的時候也提出過這樣的事情。14-12，錢天一再下一城，3-1領先陳夢。小將大爆發，拼世界第一拼得很兇。

第五局比賽，陳夢4-2領先，錢天一繼續追分，陳夢7-3領先，看她的技戰術是想要調動錢天一跑動救球，但是反手軟綿綿的，似乎失誤的有些隨意。錢天一逐漸將比分追到7-8，陳夢再這樣下去可不行啊。13-11，陳夢艱難扳回一局。

第六局比賽，錢天一3-1領先，陳夢6-5反超，錢天一叫暫停。回到比賽，陳夢又吃了發球。好在還能繼續得分，9-7領先，緊接著就發球失誤送分……11-8，比賽到決勝局。

第七局比賽，雙方1-1開局，雙方繼續相持到3平，多回合球，錢天一一直想調動陳夢。11-6，陳夢4-3險勝晉級。

1-3到4-3！世青賽冠軍7局惜敗，世界第一拒崩盤，8強再戰劉詩雯，當然錢天一的表現同樣很優秀了，力拚陳夢，險些將世界第一掀翻。
