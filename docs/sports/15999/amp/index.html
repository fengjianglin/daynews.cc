國青爆大冷！2-1擊敗東歐勁旅二隊，對手輸不起圍攻裁判

天天要聞
2019-12-14

這些天，中國足球的日子並不好過。國足在東亞杯首輪1-2不敵日本隊遭遇開門黑，而國奧則在珠海四國賽0-1不敵敘利亞，而U20國足國青隊則在「七彩雲南」國際足球邀請賽首輪0-3慘敗給莫斯科斯巴達克二隊，中國男足在短短的幾天內遭遇3連敗，壓力非常大。

就像人生一樣，觸底總會反彈，不會永遠低沉，中國男足終於迎來了一場大勝。

北京時間12月13日15時30分，2019年雲南國際足球邀請賽第二輪展開爭奪，國青隊2-1爆冷擊敗東歐豪門貝爾格萊德紅星，迎來本屆賽事的首勝。首輪0-3慘敗給莫斯科斯巴達克二隊後，主帥揚科維奇大發雷霆，認為沒有一個球員正常發揮。

正所謂「知恥而後勇」，本場比賽國青的小夥子們表現得非常積極。比賽第13分鐘時，足球教父徐根寶培養的19歲小將周俊辰在禁區內打進了一粒世界級進球！當時，他在禁區內連續晃動後，貝爾格萊德紅星的防守隊員集體懵圈，隨後他冷靜起右腳射門直入死角，門將反應不及皮球應聲入網，國青取得夢幻開局。

比賽第36分鐘，周俊辰再度在左路形成突破，他在禁區內晃過兩名防守隊員後果斷橫傳，跟進的北體大小將幺旭辰推射破門，國青隊取得兩球領先。

下半場比賽，貝爾格萊德紅星二隊一度扳回一球，但無力回天。爆冷輸給國青隊後，貝爾格萊德紅星的球員明顯心態失衡。比賽結束後，這支東歐勁旅的球員們集體圍攻裁判發泄不滿，轉播鏡頭也記錄了這一幕。從畫面中可以看出，身著藍色隊服的貝爾格萊德紅星二隊球員情緒極其激動，圍住了中國裁判組，最終在中國球員和教練組的掩護下，才得以脫身。

雖然比賽結束後出現了不和諧的一幕，但並不能掩蓋國青本場比賽的出色發揮以及19歲妖星周俊辰的出色發揮。本場比賽他打進1球助攻1次獨造2球，堪稱國青獲勝的最大功臣。

同時，必須感謝徐根寶「十年磨一劍」，為中國足球培養了武磊、張琳芃、顏駿凌、曹贇定、呂文君等國腳，很難想像，如果沒有徐指導的十年耕耘，中國足球今天會是怎樣的景象。（熊邇軒）
