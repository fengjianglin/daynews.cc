東亞杯頒獎中超成大贏家！孫雯+國足隊長登台，搶走日本國腳C位

2019-12-19 411條評論 3,804次閱讀 1685人點贊

北京時間12月18日晚，2019年東亞杯比賽全部結束，韓國1-0擊敗日本，獲得東亞杯冠軍。賽後頒獎典禮，梅方代表中國隊登台領獎，他搶走了本該屬於日本球員的C位。值得一提的是，代表韓國領獎的是前恆大外援金英權，他是韓國隊隊長。

在頒獎儀式現場，除了梅方代表國足獲獎，還有孫雯也在現場，兩人合影。按照冠亞季軍的慣例，韓國球員和日本球員應該在中間，但是，梅方直接站在了中間，日本球員則站在了旁邊！

前恆大外援金英權也登台了，他代表韓國隊拿到了冠軍獎項。值得一提的是，金英權和梅方還做過隊友。在恆大時期，金英權曾有出色表現，幫助恆大拿到了亞冠冠軍，上賽季結束後，金英權離開了恆大。

此外，在中超北京國安效力的韓國球員金玟哉獲得最佳防守球員獎，日本球員小川航基以3個進球獲得最佳射手獎，幫助韓國隊攻入奪冠制勝球的黃仁范榮膺本屆東亞杯MVP。最終，金英權捧起了冠軍獎盃！

也就是說，本次東亞杯，中超搶鏡，金英權是前恆大球員，而金玟哉是現役中超球員，在中超的磨礪下，兩人都有出色表現。去年俄羅斯世界盃，金英權還攻破了德國隊大門，至於金玟哉，已經收到了英超的邀請。

至此，本屆東亞杯落幕，韓國隊成為最大贏家，不僅韓國男足實現3連冠，韓國女足也表現出色，拿到亞軍，力壓了中國女足。此外，韓國女足表現頑強，只是在最後時刻被日本女足絕殺！因此，韓國兩支球隊在東亞杯的表現，是可以讓韓國觀眾滿意的。韓國男足1-0擊敗日本隊的比賽，現場來了近3萬球迷！
