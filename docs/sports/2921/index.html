天津女排兩隊長下崗！朱婷被扶正袁心玥空降，世俱賽正式名單揭曉

2019-12-03 0條評論 21次閱讀 0人點贊

摘要

天津女排原本坐擁超強三叉戟朱婷、李盈瑩和胡克爾，不過主教練陳友泉希望在世俱賽創造佳績，大手筆將袁心玥、張常寧、龔翔宇和王夢潔申報在大名單內，相當於圈定郎平衛冕世界盃七大主力中的五位，想要搭建女排准國家隊陣容與歐美諸強抗衡。可惜，張常寧和龔翔宇在排超期間身患重感冒，前者在3日揭幕戰還有確認出席的商業活動。分身乏術的張常寧、龔翔宇和王夢潔只能拒絕空降馳援世俱賽。

​正如主教練陳友泉最終敲定的世俱賽陣容，天津女排同時損失三位世界盃冠軍，好在朱婷好閨蜜袁心玥帶傷空降助陣，後者由於肩傷反覆不得不缺席排超大部分小組賽，直至最近才咬牙復出就火速加盟精神可嘉。雖然袁心玥暫時還沒有融入天津女排的戰術體系，但是由於姚迪一直在國家隊參與訓練，有如此好的攻傳配合基礎令袁心玥上手很快，唯一需要注意的是控制好傷病，畢竟後續還有排超和國家隊任務。

​根據天津女排正式公布的世俱賽名單，新老隊長姚迪和楊藝下崗而朱婷被扶正，後者大賽經驗豐富並且英語交流能力強，作為中國女排隊長的她接班可謂駕輕就熟。鑒於天津女排在世俱賽引援受挫的客觀事實，主教練陳友泉對於首發陣容只能局部微調，改由袁心玥頂替李雅楠與王媛媛搭檔副攻，三叉戟沿用朱婷、李盈瑩和胡克爾懸念不大，不過首發和替補陣容實力差距明顯，因此連續作戰對於體能儲備要求會更高。
