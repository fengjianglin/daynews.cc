西甲2019數據榜：本澤馬、卡索拉在列，梅西包攬了射手王，過人王

2019-12-25 787條評論 3,695次閱讀 162人點贊

智道12月24日訊 時光如梭，2019年轉眼飛逝。《Squawka football》更新了2019年西甲球員的數據榜，其中前阿森納球員「三弟」卡索拉戰勝了傷病，成為創造機會最多的球員，而梅西一人包攬了最佳射手王、過人王和進攻三區最多傳球三項進攻數據。

2019年西甲球員數據榜單如下：

射手王：梅西（巴薩），一共打入了34個進球！

過人王：梅西（巴薩），一共成功過人140次！

進攻三區最多傳球：梅西（巴薩），一共是740次！

助攻王：坎帕納（萊萬特），一共奉獻了11次助攻！

創造最多機會：卡索拉（比利亞雷亞爾），一共是84次為球隊創造得分機會。或許很多阿森納的球迷還會想起他，如果球員沒有傷病，或許三弟還是兵工廠的中流砥柱，或許溫格就不會離開。

鏟斷王：卡塞米羅（皇馬），一共成功鏟斷119次！

遭犯規最多：帕雷霍（瓦倫西亞），一共讓對手犯規87次！

撲救王：艾托-費爾南德斯（萊萬特），一共成功撲救138次！

對方禁區內最多觸球：本澤馬（皇馬），一共257次在對手禁區里拿球，C羅離去，本澤馬棄鍋背槍，成為銀河戰艦的主火力。
