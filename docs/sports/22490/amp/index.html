8連勝！山東男籃單節一波流擊垮遼寧隊 師弟郭艾倫表現仍慢熱

天天要聞
2019-12-20

北京時間12月19日，山東男籃在主場迎戰遼寧男籃。近日，遼寧男籃在場上的狀態有所起伏，特別是球隊內部史蒂芬森以及郭艾倫的狀態不穩讓球隊最近戰績不佳。此役，山東在第三節打出一波流，最終他們以114-94戰勝遼寧。如今，山東喜迎8連勝！

在這場比賽前，山東男籃的戰績十分不錯，他們已經取得了7連勝，而關於遼寧男籃內部的問題漸起，史蒂芬森在球場上的狀態不夠穩定，時好時壞的狀態讓球隊很受傷。特別是今天的比賽中，史蒂芬森的表現就很難讓球迷們感到滿意。

早在上一場比賽，面對青島男籃，史蒂芬森的狀態就已經十分低迷。因為腿傷僅僅出戰21分鐘的史蒂芬森，除了在防守中表現不夠積極，屢次成為亞當斯的「背景」之外，他在進攻中的表現其實更加糟糕。短短21分鐘的出場時間內，史蒂芬森一共出手16次，在全隊僅次於出手19次的巴斯、與韓德君持平。然而，他兩分球7投2中，三分球9投0中，甚至連罰球也只有可憐的3罰1中，在上一場比賽中，師弟僅僅得到5分。

然而，到了本場比賽，師弟在前三節的表現讓人覺得匪夷所思，前三節竟然只得到6分，不過，到了下半場，師弟的狀態終於有所恢復，他全場20投8砍下18分，9籃板。然而，師弟的三分球竟然10投僅1中。

只是，在遼寧隊內，表現慢熱的又何止是師弟呢？郭艾倫在前三節竟然9投1中，不過還好到了第四節，他的表現開始所有恢復。全場比賽，郭艾倫15投6中砍下16分。而在上一場比賽中，郭艾倫在42分鐘的出場時間裡，僅僅交出14投2中貢獻8分。

不過，由於這兩名大將的慢熱，山東隊在第三節砍下37分，而他們只讓遼寧在這一節中得到19分。而遼寧隊在第四節有所起勢的時候，山東隊卻在哈德森的帶領下打出了氣勢，他們內外線開花，在自己的主場讓遼寧隊十分尷尬。哈德森也在全場比賽砍下41分10籃板，他也投中了5記三分球。在山東的這一波8連勝中，哈德森功不可沒，如今山東繼續連勝，他們也送給了遼寧兩連敗。
