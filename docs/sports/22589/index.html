CBA太兇狠如拳擊，林書豪呼籲：公平打球吧

2019-12-19 1,623條評論 2,791次閱讀 1312人點贊

近日，在北京首鋼以88比87險勝上海大鯊魚一役，豪哥全場11投4中僅得到13分，主要是他全場受到「重點照顧」，被對手「手腳並用」嚴防，導致他在賽後在社媒曬出了自己受傷的照片，吐槽CBA的比賽就像打拳擊，並配文：「救命，我不想有鼻青眼腫。今晚上是拳擊籃球，哈哈。」北京電視台在賽後播放了很多比賽畫面，畫面中的上海隊隊員對林書豪展開拉拽、推搡、上腳的各種小動作，對於這些動作，賽後林書豪的態度也只是：我就是乖乖去打球。但這節目播放後，引發了球迷兩極觀點的熱議。而打完上海後，豪哥昨晚率領球隊迎戰浙江廣廈，最終他得到20分9助5板，但本場比賽全場9次被侵犯，依舊受到「重點照顧」。

​

書豪今日更新了自己的個人社交媒體，曬出了昨晚與廣廈一戰的高光時刻，並以「提問-回答」的方式，回答了球迷10個問題。其中，有一個問題的回答比較引人關注。有球迷問他，CBA的身體接觸是不是比NBA兇狠更多？他回答：有的時候圖片比文字更有說服力，並配上了下圖。圖片中，林書豪被對手打到臉部導致面部扭曲，被多人包夾犯規，以及被對手飛身壓人等，都是比較「狠」的犯規動作。

廣廈在防守林時依然非常兇悍，似乎沒受到上一戰北京對上海後的熱議所影響而收歛，賽後，豪哥無奈再次發聲表示：「我並不需要受到甚麼特殊照顧，只希望得到公平對待，因為我已經31歲，我受過傷，我的身體是最重要的，我真的希望可以有公平的打法打下去，不要因為小動作受傷。」

不止一個從NBA過來的外援說過，CBA的強度高於NBA。他們這麼說，一是因為在NBA，他們往往不是重點防守對象，但在CBA，他們往往是隊里的頭號得分手，是對方的頭號防守對象，壓力之大可想而知；二，則是因為CBA對老外的防守，很多都處於犯規狀態，但裁判往往不吹。所以，很多外國球員覺得CBA強度高於NBA，就是這個原因。

那這樣好不好？對不對？如果你不是某隊的球迷，那這樣的做法就是百害而無一利。首先是對國內隊員壞防守習慣的縱容，世錦賽上的郭艾倫就是很直接的例子（郭艾倫本來防守很好，並且沒有惡意的小毛病，就是愛掏球，尤其人過了他，還要伸手摸一把），還有羅旭東，杜峰帶著打國際比賽，七分鐘五犯被罰下，要把那晚放書豪的幾個動作放國際賽場，怕是五分鐘就得下去。

這些壞習慣，在國內用慣了，形成下意識的反應，到國際賽事上就是被吹。當然，對大多數國內運動員來說，根本沒有打國際賽事的能力與資格，可以無視這些，但他們是CBA的一部分，是整個聯賽風氣形成的重要組成部分。他們用各種犯規、惡意動作防守外籍球員，那些有國家隊隊員 的教練看了，便會說，你看誰誰誰防外籍球員防的多好，你就不能像人家那樣防嗎？於是，國家隊隊員也開始上各種犯規惡意動作，久而久之成條件反射，到了國家賽場，被抓著狠吹。

我們說回林，毫無疑問，僅就現在的常規賽而言，林是遭遇防守最野蠻的小外。「對黑人唯唯諾諾，對華人我重拳出擊」，這話曾流行一時，有人不屑鄙視，有人奉為圭臬。這話對，也不對。我相信大部分球員沒有這種想法，但少部分球員，確實會這麼想。林書豪在NBA時，有個國內基層教練說過，林書豪這種球員，中國能找一千個。言語里儘是不屑。有少部分球員，確實會有這樣的想法。木秀於林風必摧之，這不是心理揣測，這是人的本能反應。同等條件，面對比自己優秀很多人，是人都會嫉妒，只是有的人化嫉妒為動力，有的人化嫉妒為報復。

事實上，不僅在cba，林書豪在NBA受到的待遇可謂更是艱難，應該判罰犯規的哨子，在林書豪這裡直接聽不到哨聲，應該吹罰技術犯規的犯規，在林書豪這裡只是普通犯規，NBA有明文規定，對頭部以上犯規都應該判罰技術犯規，但是每當對手對林書豪頭部以上犯規的時候，裁判只是會簡單的吹罰個普通犯規草草了事。而之前他在社交媒體所說出的「他們還吹罰我犯規，哈哈。」更像是一種無奈的反抗。

最後我們來說說為什麼林書豪會遭到犯規這麼多，我給出的答案：因為突破太慢！ 單純看每個球，確實是犯規。但書豪突破，現在給人的感覺，真是完全靠身體，速度不行了，第一步還行，後續加速能力堪憂。書豪身體對手國內後衛都是有優勢的，但是速度不算有優勢，一般都能跟上，加上本土後衛防守習慣也不太好，很容易下手和侵犯圓柱體。反而是有些小後衛，比如菲爾德、亞當斯、福格這些靠速度的，很多都是擋拆一步過，加速也快，反而沒有那麼多犯規。在沒有防守三秒的cba，靠節奏突破這招沒那麼好使，但也很好使，前提是—— 穩定的中距離，國際比賽牛逼的後衛大神，沒有一個中距離不牛的，納瓦羅、斯潘、阿羅約、卡帕佐，甚至升級後的盧比奧。
