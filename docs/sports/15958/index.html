僅剩兩個健康中衛，克洛普承認利物浦防線告急

2019-12-14 1,316條評論 3,424次閱讀 1877人點贊

利物浦主帥克洛普承認，在周六與沃特福德的比賽臨近時，安菲爾德的後防出現了很大的問題。

由於傷病的原因，這支聯賽領頭羊已經失去了中後衛喬爾·馬蒂普，但是現在德國人已經證實了德揚·洛夫倫的情況比他希望的更令人擔憂。

克羅埃西亞國腳上周末在伯恩茅斯的比賽中一瘸一拐的下場，在周中對陣薩爾茨堡的比賽中下半場早段被替換下場，這場比賽幫助紅軍在E組取得頭名。

克洛普承認在那場勝利後，他希望洛夫倫只是因為抽筋而受傷，然而事實證明，隨著賽季繁忙期的臨近，這對紅軍來說更加重要。

「德揚已經離開沃特福德了。」在周五與紅軍續約到2024年的克洛普表示：「他會在我的合同到期之前回來！這是肌肉受傷，比上次更嚴重。我們有問題。」

克洛普承認現在的傷病情況並不理想。

「只有兩名健康的中後衛令人擔憂，這不是我們第一次有這種情況。」他補充道：「法比尼奧也踢不了了。這並不太酷。我們只能把他們放到棉花里，問他們想要什麼樣的訓練。」

他一直在思考如何解決這個問題，並承認年輕的塞普·范登伯格和基-亞納·霍弗都將出現在陣容中。

「我們必須有創造力。」他說：「我們有基-亞納和塞普。我們也可以踢不同的體系。但我們有問題。」

從周六到1月5日，利物浦將在四場比賽中打七場比賽。

在與沃特福德的比賽後，紅軍將前往阿斯頓維拉參加聯賽杯四分之一決賽。然而，考慮到他們的主力陣容將會參加國際足聯的世俱杯，他們將會更換陣容。

從卡達回來後，克洛普的球隊將與萊斯特城、狼隊和謝菲爾德聯進行英超聯賽的比賽，然後與埃弗頓進行足總杯德比。
