武磊後還有他！22歲新星驚艷自由球絕平歐冠常客，激動得當場滑跪

2019-12-20 2,985條評論 4,383次閱讀 849人點贊

2019年的最後幾天，武磊的2粒精彩進球給這個冬天增添了几絲暖意。不過，在目前的歐洲足壇，除了武磊之外還有其他在追求足球夢想的中國球員，而且與武磊同一天斬獲進球，他就是22歲的陳奕。葡萄牙U23聯賽最新一輪，陳奕在補時第2分鐘打進一粒驚艷自由球，幫助球隊絕殺扳平本菲卡隊，進球後他甚至激動得當場滑跪慶祝。

12月20日葡萄牙U23聯賽，陳奕效力的科瓦彼達迪隊客場挑戰歐冠常客本菲卡隊。在此前的90分鐘常規時間內，科瓦彼達迪隊0比1落後。時間進入傷停補時，眼看比賽即將結束，科瓦彼達迪隊獲得禁區外一個位置非常好的自由球機會。中國球員陳奕站在了罰球點上。

我們看到，小將陳奕深呼一口氣，經過助跑後他用右腳踢出一記高質量的自由球，球在擊中對方右側立柱內側後彈進了球網。這粒進球不管旋轉度、力度還是角度都掌握得非常好，可以說是一粒非常精彩的自由球。重要的是，客場面對葡萄牙豪門本菲卡，陳奕和他的隊友們直到比賽最後時刻仍然沒有放棄，所以才有了最後的進球。而在打進這球絕平對手後，陳奕甚至激動得當場滑跪慶祝。

這不是陳奕第一次在葡萄牙進球。早在今年9月份第9輪葡萄牙U23聯賽中，他就曾在第29分鐘打進一粒世界波，幫助球隊2比1取勝對手，拿到一個關鍵的三分。可見，陳奕在與本菲卡這場比賽中打進的自由球絕非偶然，因為他在禁區弧頂附近確實有一腳遠射功夫。

出生於1997年的陳奕，雖然年紀輕輕，但是卻比武磊還要早走出國門。出道於亞泰青訓的他，在經過2012年亞少賽的閃光後，於2014年開始正式留洋。在這5年間，他先後效力過馬夫拉隊隊、圖裡茲人隊和科瓦彼達迪隊。而在效力圖裡茲人隊時，他甚至成為了這支球隊的場上隊長。作為一名外籍球員，陳奕能夠在比賽中擔任隊長一職，這足以看出球隊對他的信任。我們期待陳奕未來能順利成長，爭取早日進入國足。
