國乒小魔王賽前高強度對練！教練組將採用新策略，林高遠更換髮型

天天要聞
1 週 ago

2019年國際乒聯總決賽將於本周四在中國鄭州拉開序幕，目前各協會的球員都已經來到鄭州進行賽前的備戰，明日晚間他們還將出席國際乒聯舉辦的頒獎盛典，年度最佳運動員等諸多獎項的獲得者將在明晚正式公布，對於最終的結果外界球迷也是比較期待，馬龍和劉詩雯最有可能當選年度最佳。

在本賽季劉詩雯還沒有拿下過巡迴賽的女單冠軍，因為各種因素她在巡迴賽場的表現也是並不理想，到來到大賽劉詩雯成績出色拿下了世乒賽女單和女子世界盃的冠軍，這次總決賽她也將全力以赴希望能夠取得好成績，在總決賽歷史上她是唯一實現過三連冠的球員。

之前劉詩雯就一直是和男選手訓練來提高強度，這次她也是和徐晨皓一起進行了多板的對拉，在賽前進行充足的備戰，孫穎莎和王曼昱兩位小魔王也開始了高強度的對練，這次總決賽她們兩人都是被外界看好的年輕球員，有望在這裡取得出色的表現，達到預期目標。

男隊方面梁靖崑和樊振東都找了直板打法的隊友進行訓練，梁靖崑找到許昕，樊振東則是找到了趙子豪，幾人在賽前互相交流希望能夠在比賽開始之前達到一個比較穩定的狀態，在訓練過程中梁靖崑和許昕也是產生了爭議球，都提出進行鷹眼挑戰，當然兩人也是在開玩笑，訓練還不能享受這一待遇。

林高遠和馬龍兩人也是在進行對抗練習，林高遠還換了頭型希望能夠在總決賽有一個好的開始，去年他打進決賽輸給了張本智和獲得亞軍，今年他也將全力以赴爭取有所突破，而馬龍的狀態也是比起之前恢復了不少，在訓練中和林高遠打的也是比較激烈，強度一點不比其他人低。

毫無疑問對於這站總決賽大家都非常重視，馬龍和劉詩雯分別保持著賽事的一項紀錄，劉詩雯在2011至2013年取得三聯冠，馬龍則是五次奪冠成績十分出色。

在去年的頒獎盛典上國乒女隊的穿著也是遭到了不少球迷吐槽，許多人都沒有穿上外界期待的裙子，去年有不少人都是穿的褲子，今年教練組採用了一項新的策略，規定在訓練中拉不上球的將進行必須穿上裙子的處罰，可以看出為了讓大家穿裙子教練組也是絞盡腦汁，明晚進行的頒獎盛典，來自國乒的美女們會以什麼樣的裝扮亮相就讓我們拭目以待。
