中超客場積分榜：恆大創新紀錄，武漢卓爾第4，深圳佳兆業創新低

2019-12-03 0條評論 21次閱讀 0人點贊

摘要

相對於作為成績基礎的主場積分，客場就是衡量一支球隊打硬仗的實力。去年的中超聯賽，上海上港最終拿到冠軍，其強大的客場取分能力是關鍵，今年的聯賽同樣如此。整個賽季下來，恆大在客場所向披靡，拿到39分的超高分，超越了2013賽季恆大的34分，成為了中超歷史上客場最佳成績。

相對於作為成績基礎的主場積分，客場就是衡量一支球隊打硬仗的實力。去年的中超聯賽，上海上港最終拿到冠軍，其強大的客場取分能力是關鍵，今年的聯賽同樣如此。

整個賽季下來，恆大在客場所向披靡，拿到39分的超高分，超越了2013賽季恆大的34分，成為了中超歷史上客場最佳成績。升班馬武漢卓爾名列第四，球隊主場客場得分幾乎相同。而最終降級的北京人和與深圳佳兆業都只拿到3個積分，但凈勝球或進球數少於2013賽季的武漢卓爾，創造新的客場最差成績。

恆大不強嗎？不，今年的恆大比哪一年的都要厲害，因為他們在客場拿到了恐怖的13勝2負的成績，唯一的兩場失利分別是輸給江蘇蘇寧和北京人和，其中後一場比賽中充滿著意外。

正當人們還在感嘆國安歷史最高亞軍得分的同時，恆大不知不覺之間又創造了一個新的記錄，其39個客場得分超越了2013賽季。當年恆大總共拿到77個積分，但球隊主場實力更強，客場10勝4平1負的34個積分雖然不錯，但比今年還是有些差距。

39個客場積分，我相信，未來很多年裡面都將是中超聯賽中的紀錄。

國安今年客場收穫了31個積分，表現不錯。上港有些可惜，最後兩個客場1平1負，這導致球隊客場只拿到29分，名列第三。

升班馬武漢卓爾在客場拿下了6場勝利，拿到21分，這樣的分數也遠超一年級時候的石家莊永昌（2015賽季客場14分），延邊富德（2016賽季客場7分），貴州恆豐（2017賽季客場17分），北京人和（2018賽季客場15分）。

山東魯能和蘇寧都是18分，大連一方和重慶斯威都是16分，申花拿到13分，建業和華夏幸福拿到12分，天津泰達拿到11分。這些客場拿到10分以上的隊伍，成績都超過了30分，也很順利的提前保級成功。

相對於主場，富力的客場就有些難看了，總共只拿到7分。不但0比4不敵深圳佳兆業，還1比2輸給華夏幸福。唯一的兩場勝利來自客場擊敗天津天海與北京人和。

天津天海同樣客場表現一般，他們只戰勝了申花、戰平了泰達、上港、大連一方、華夏幸福和重慶斯威。

本賽季，客場表現最為糟糕的要數深圳佳兆業與北京人和，兩支隊伍一個賽季下來，客場一場未贏，只拿到了3場平局，這其中還包括他們相互在戰平的兩場比賽。

3個積分，也讓他們追平了2013賽季武漢卓爾3分的中超客場最低得分。

今年的北京人和與2013賽季武漢卓爾一個賽季的客場凈勝球都是-24個，但卓爾那年客場打入12球，人和只有9球。而可憐的深圳佳兆業最後一輪客場慘敗，這也讓他們成為了中超聯賽到如今為止，客場成績最差的隊伍。
