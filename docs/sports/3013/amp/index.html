摘要

先來今天的戰況預報吧，昨日剛擼完，今日比賽就淋離盡致，掘金大戰馬刺，掘金內線約老師手活細膩，把阿德摸的全身酥麻，老爺子一看。

先來今天的戰況預報吧，昨日剛擼完，今日比賽就淋離盡致，掘金大戰馬刺，掘金內線約老師手活細膩，把阿德摸的全身酥麻，老爺子一看。

不對勁！高舉龜派氣功波第一招—縮陽入瓮，比賽的輸贏就變成了掘金三分準確率的賭博，很明顯，今天的掘金三分一點不含糊，有那麼一股子愣頭青的闖勁，手起刀落，刀刀入心，外線德羅贊的精力都放在了撲防三分上了，一個巨星你讓他一直搞折返跑，心裡非常爽，季後賽的選手不僅要看場上局勢，還要觀察球員情緒變化，掘金一看這德羅贊猴急了，於是乾柴只需星火，.在對抗中給你來點料，德羅贊領上二級犯規，一臉不爽更衣室里撿肥皂，沒了中距離的強烈威脅，三分手感投開了，約老師內線也就如魚得水，更上一層樓，順勢收下比賽，比賽來到2:2平，離我預測4:3的比分基本接近，德羅贊想成為領袖，需要成長的路還很長！相信波波維奇會幫他的，但如果是阿斗，雖有諸葛亮在蜀國該亡害的亡。進入今天的正題，說完了西部，今天開始後四章開始說著東部群雄，費城76人VS籃網，這比賽也是很有意思，刺刀拼到最後，籃網沒有絕對高強度對抗核心，恩比德在線獨霸天下，三分線外火力驚人，76人心驚動魄的收下比賽，但籃網做的真的很不錯了，內線一打一不行，但我們2打2卻玩的嗨翻。吃了大碗寬面的中鋒艾倫是正的強，你問我有多強，光頭強的強，各種傳切配合暴扣大帝，籃板球也不差，今天的你配的上著季後賽的地板，，既然說到這，我們就先來了解下這兩個球隊的構成的一些文化，話說這費城76人名片那毋庸置疑是我們街頭三歲孩子都知道的答案——艾弗森，一個天才球員，一個江湖浪子，一個狂妄的讓你舒服的hip hop，單挑神，重點是還一擊命中，一個人帶球隊殺入中決賽，有仇必報，從敵人身上跨過去，以一己之力對抗巔峰的OK組合，還贏了一場，現役騎士主教練就曾是獨孤九劍下的亡魂人，所以在我眼裡他就是武俠里的令狐沖，任盈盈眼裡的沖兒。桀驁不馴，天賦異稟。好友聚場，第一句話總是拿酒來，他讓人們對變相這個動作有了新的理解，讓小球時代提前到來。但隨著他的離開，76人進入了 漫長的重建，有多長呢？2001-2019，雖然2002-2003賽季有過瞬間的燦爛，但也擋不住這長達18年的黑暗天空，對於費城人民來說，他們的時代里，18年里沒有偶像，長時間的擺爛讓他們擁有了大量的高順位選秀機會，終於在2016年奪的狀元簽，才有了今天大帝恩比德，和狀元西蒙斯，管理層也勵精圖治，補強了射手雷迪克，前鋒吉米巴特勒，超高中鋒博班，實用型中鋒門羅，以及角色射手，一個大陣容組建完畢，他們天賦爆棚，但季後賽經驗匱乏，唯一的經驗就是吉米巴特勒，但關鍵還是在大帝身上，打個籃網都快哭了，那猛龍、綠軍、雄鹿等，你不是得跪下，所以，兩位超級巨星的學習成長能力決定了球隊能走多遠。順便說說籃網吧，至少給上該有的尊重，對於一個在湖人過的水生活熱的拉塞爾來說，籃網的節奏更適合他成長，但問題的關鍵是這孩子的上限太明顯了，一個球隊的老大上限可以看得見那麼這個球隊的上限也就那樣吧，如今比分來到3:1 76人可能會放一場來搞搞戰術，但也可以直接送他們回家，季後賽首輪游是板上定釘的事了，但今年的拉塞爾算是打出了該有的東西吧，但要說他能帶一隻球隊進入東決、西決的底板。

那一斤檸檬我就預定了，最後我們聊一下這西蒙斯和大帝的上限吧，先來看這西蒙斯，小伙身體倍棒，同位置沒人身體能搞過他，如果他在具備了中距離和遠投的情況下，難么他就是一個很恐怖的存在了，就好像天使在一級變擁有了16級被動的存在，中投是可以練出來的，這裡要說一個他的前輩，曾經的狀元給力芬，以前也是投籃弟弟，在他的眼裡只有轟炸籃筐，但現在的大叔芬的中距離有模有樣，三分還絕殺過比賽，所以未來的西蒙斯前途無限，詹姆斯估計只是他的正常發育，要是努力，我覺得喬丹的位置可以嘗試，最主要的是這娃聰明啊，和威二錘比那就是一個老師，一個三年級哦，在來看大帝，基本技術成熟，但經驗太low ,假以時日，這個組合很厲害，並且整容的配備極其容易滿足，條件容易滿足了：1、防守不不成為樓棟2、具備基本投射能力3、會策應球 其他的就是把球給西蒙斯，大帝中線站位，三個角色球員在熟悉位置跑動策應，準備回防和快攻接球上籃，贏球就是那麼簡單。

當然，為了輪換，還得搞個第六人。這兩隻對的等級不在一個層次，加上籃網太平淡，76人局勢不明朗，所以更多侃談，盡在二輪揭曉。
