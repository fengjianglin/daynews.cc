超級逆轉！中國台北一哥決勝局25-23翻盤 泰國世界冠軍21-4虐同胞

2019-12-11 869條評論 2,381次閱讀 708人點贊

北京時間12月11日，2019年世界羽聯總決賽繼續首日爭奪，繼國羽奧運冠軍諶龍苦戰2-1險勝丹麥世錦賽冠軍安賽龍後，世界第2、中國台北一哥周天成則是鏖戰76分鐘，在先丟一局的不利情況下，最終以總比分2-1超級大逆轉擊敗印尼一哥金廷，尤其是決勝局拼到25-23險勝翻盤。至於在女單的一場德比大戰，泰國世界冠軍因達農2-0輕鬆橫掃同胞布桑蘭，首局更是21-4殘暴吊打。

總決賽首日率先進行B組的兩場男單小組賽，首場比賽諶龍三局險勝安賽龍，三局比分依次是21-17、10-21、21-17，取得對安賽龍交鋒18次的第14場勝利。

至於第二場男單對決，同樣是一場勢均力敵的比拼，由世界第2的周天成對決印尼一哥金廷，後者世界排名第8。只是在過去10次交鋒當中，周天成以4勝6負處於下風，而金廷素來是一位善於對強手的進攻型高手。

在如此對決的首局比賽，周天成無疑狀態不佳，他在首局6-6平後，就被金廷轟出15-5攻勢壓制，以11-21先丟一局總比分0-1落後。進入到第二局比賽當中，周天成纏鬥到10-10平後，他逐步發力拉開比分，以21-15扳回一局，雙方前兩局維持總比分1-1平。

及至決勝局大戰，雙方開啟瘋狂拉鋸戰模式，周天成在局末繼續與金廷打成19-19平。雙方也是繼續纏鬥至23-23平，周天成關鍵時刻連得2分，從而以25-23驚險拿下決勝局，以總比分2-1大逆轉擊敗金廷取得總決賽開門紅。

至於在女單的一場泰國德比對決當中，泰國一姐、前世錦賽冠軍因達農表現出色，輕鬆以總比分2-0速勝橫掃同胞布桑蘭，尤其是首局以21-4殘暴吊打，次局則是以21-16拿下，用時39分鐘完成橫掃。
