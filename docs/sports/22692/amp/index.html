2019-12-19

眾所周知，在中國有兩樣球類運動，放眼世界，是誰都打不過。其一是屢戰屢敗，在段子界有較高熱議度的中國足球，幾乎贏不了人。其二，便是我國國球乒乓，即便當下中國實行「養狼計劃」，鼓勵國內教練外出培養國外運動員，然而中國依舊屹立世界巔峰巋然不動，乒乓王座無人能染指。國外為了能有與中國相比肩的乒乓實力，所以這幾年，總會拍一些隊伍，來中國交流，取經學習。在這過程中，也讓我們認識了，像福原愛這樣，甜美可人的外國妹子。

在當今各國中，日本乒乓隊算是一個在乒乓項目上較為出色的隊伍了，雖然與中國還是有一定的差距。但近幾年也是奮起直追，十分重視乒乓，能看出是在這上面下了苦功夫的。今天我們要說的，是這位來自日本，曾與中國國家乒乓隊有過多次交流，被賦予「日本女乒一姐」之稱的石川佳純！

石川佳純是日本女乒的3大主力之一，和伊藤美誠，平野美宇一樣，在新一代日本乒乓隊伍中擁有舉足輕重的地位。她曾與中國乒乓國家隊隊長馬龍多次同框合影，漸漸被人們熟知，也曾有傳聞，說石川與馬隊之間上演跨國戀情，雖然不同國家，雖然石川佳純在乒乓職業生涯上的成就與馬龍的「大滿貫」之間還有著一定的差距。但似乎球迷們對這樣球壇明星間的八卦緋聞，還是喜聞樂見的。當時一大堆人高喊，看好這對傳聞的CP。

在現實中石川佳純，由於教練是個中國人，所以對中文的掌握，也是不亞於，能隨意講出一口10級純東北話的福原愛。因此，各種場合下，她與馬龍間的交流也是十分融洽。石川也多次於公開場合，示愛馬龍，然而，石川對外界回應表示，對馬龍的愛慕之意，只是小迷妹對自己偶像的敬佩和崇拜，外界也沒必要過分渲染。

日本乒壇美女，中文10級，不輸福原愛，示愛馬龍後，這樣對外回應。2016年10月14日，馬龍在自己的微博上公開自己與女友夏露的戀情，曬出兩人合影並附文「八年不易，我們很好」。而在一年後的2017年12月10日，中國國乒隊長馬龍在微博曬出新生兒子照片，2017年1月已低調領證完婚的他正式晉級「奶爸」行列，對此石川佳純也私下表示祝福。如今石川佳純的，也成長為一名，能在日本乒乓隊伍中，獨當一面的強勁選手。她聲稱，自己還會以馬龍為榜樣，砥礪前行，一直讓自己，在乒乓的職業生涯中做到更好！
