郝偉國奧斬獲四國賽亞軍，一人表現出色 球迷：他該進國足了

2019-12-15 1,358條評論 2,693次閱讀 1618人點贊

中國國奧隊在珠海四國邀請賽，前兩場比賽取得了一勝一負的戰績。其中國奧第二場比賽0比1惜敗給敘利亞，讓球迷看到了希望。因為雖然國奧隊沒有贏下敘利亞，但球隊在比賽中創造了很多進球機會。如果國奧鋒線球員門前把握機會的能力可以再強一些，那麼國奧至少可以保證這場比賽不輸給敘利亞。珠海四國邀請賽第三場比賽，郝偉國奧迎來了馬里隊的挑戰。

馬里隊是一支非洲球隊，實力應該不算太強，這一點從馬里隊在這場比賽攻防兩端頻頻失誤就能看出來。其中馬里門將一次出擊失誤和後衛烏龍球，更是直接送給了國奧隊兩個球。最終國奧隊在主場3比0大勝馬里，拿到了珠海四國邀請賽亞軍。郝偉帶領國奧隊在珠海邀請賽拿到亞軍，讓球迷看到了球隊在東京奧運會亞洲區預選賽決賽階段突出重圍的希望。

當國奧在東京奧運會亞洲區預選賽決賽階段的分組抽籤出來時，大家一致認為中國國奧隊進入了死亡之組。因為國奧跟烏茲別克、韓國、伊朗隊分到了一組，這三支球隊在亞洲都是公認的一流強隊。中國國字型大小球隊在近幾屆大賽中，跟烏茲別克、韓國、伊朗交手時都沒有佔到什麼便宜。所以球迷和媒體將國奧分組稱為死亡之組，這一點也不為過。

郝偉國奧面對死亡之組，球隊只能拿出亮劍精神。郝偉和國奧球員認為，雖然球隊深陷死亡之組，未來出線前景確實不容樂觀，但是他們寧願在小組賽戰死，也不願意被對手嚇死。所以國奧這段時間一直在積極備戰，希望能在東京奧運會亞洲區預選賽決賽階段取得好成績。國奧隊在這次珠海邀請賽上，基本上達到了練兵熱身的目標。除了馬里隊實力太差不堪一擊之外，像敘利亞這樣的對手熱身價值還是非常高的。

國奧在珠海四國邀請賽上，球隊頭號中鋒張玉寧表現得可圈可點，多次在比賽中力挽狂瀾，所以在很多球迷看來，張玉寧應該入選國足挑大樑。張玉寧本身在中國足壇屬於年少成名的球員，他不到20歲就跳級入選國足。張玉寧國足首秀就上演了梅開二度的好戲，所以張玉寧一直被認為是97國奧頭號球星。國奧未來能否在東京奧運會亞洲區預選賽決賽圈的比賽突出重圍創造奇蹟，就要看這位本土鋒霸的表現了。相信屆時張玉寧在國奧的發揮，是不會讓球迷失望的。
