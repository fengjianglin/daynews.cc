從球員到教練對廣東隊窮追猛打，馬布里克的不是杜鋒，是易建聯

天天要聞
2019-12-16

交手10次只輸給過對手一場，弗格再狂也不曾贏過廣東隊，攜著12連勝步伐的廣東隊，竟然會輸給北控隊，確實是一場冷門般的比賽。可是，當看到對面站著的馬布里時，很多人就明白了廣東隊為什麼會輸掉比賽了。從球員時期對廣東隊窮追猛打外，到了做教練時還是不依不饒，賽後兩位主帥的衝突，表面上馬布里噴的是杜鋒。其實，總結過去馬布里跟廣東隊之間的交手，說白了，馬布里克的就是易建聯。因為幾乎每一次馬布里贏下廣東隊的比賽，易建聯在該場比賽中必將陷入低迷，毫無表現。

比賽從一開場就讓北控隊主導著比賽，他們在弗格的帶領下，不但進攻端打出了極其危險的進攻得分，防守上也表現非常積極，讓廣東隊很難有輕鬆出手的機會。特別是在攻防轉換上，北控隊退防非常快，廣東隊最擅長的快反根本打不出來。

從首節落後開始，廣東隊幾乎就是處處被北控隊抑制。雖然杜鋒通過換人調整加強了聯防力度，但效果並不好。一方面北控隊的技戰術布置很有針對性，另一方面廣東隊本場比賽的手感也一般。這其中易建聯和馬尚-布魯克斯的手感就非常冰涼。雖然下半場廣東隊一度看到了逆轉反超的希望，但關鍵時刻易建聯沒能站出來，導致廣東隊的幾番努力也功虧一簣，始終就是無法反超比分，反而被再度拉開分差。最後，比賽早早失去懸念，廣東隊以一場慘敗結束12連勝。

賽後，隨著杜鋒和馬布里之間的衝突被炒作，馬布里對廣東隊的敵意，就再一場挖掘出來。沒錯，從球員時期效力北京隊開始，馬布里幾乎每一次打廣東隊就像打了雞血一樣特別來勁。很多人都說，馬布里晚期的職業生涯，就是為打廣東隊而努力的。所以，過去那麼多年來，除了前兩個賽季被北京隊拋棄外，馬布里幾乎每一次打廣東隊，都是贏家。

為什麼馬布里每一次針對廣東隊時，總能得心應手？其實，他並不是熟悉了解杜鋒的技戰術打法和用人，也不是對廣東隊的情況了解到極致。而是，馬布里對廣東隊的技戰術打法，有一個很露骨的針對性戰略布置，那就是針對易建聯去的。

所有宏迷都記得，2013-14賽季，廣東隊做客五棵松球場，當時廣東隊在首節取得領先，而阿聯更是成為廣東隊攻防的核心樞紐。然而從第二節開始，馬布里就對阿聯這個點不斷製造殺傷，早早就把阿聯逼入犯規陷阱里。打掉阿聯這個點後，廣東隊的攻防頓時就像一隻無頭的蒼蠅，被北京隊摁著打，最終北京隊逆轉了比賽。

自從阿聯回歸CBA，自從馬布里跟廣東隊結下冤讎，可以說，每一次馬布里跟廣東隊交手，馬布里都成了贏家。而馬布里克制廣東隊的關鍵，不是杜鋒的戰術和用人，而是針對阿聯這個點他特別有心得，他知道打掉阿聯這個點，廣東隊就必輸，而且他更懂得如何限制阿聯這個點。翻看一下阿聯跟馬布里的交手，幾乎可以說，每一次打馬布里的球隊，阿聯的數據非常糟糕，手感非常冰涼。輸給北控隊的這一戰，阿聯打了35分鐘，只拿到9分8個籃板的數據就是最好的證明。

總結廣東隊為何每一次碰馬布里都那麼艱難？看看阿聯的表現就知道。每一次打馬布里的球隊，阿聯毫無表現。
