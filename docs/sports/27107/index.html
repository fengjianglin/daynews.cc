英超最新積分榜：曼城3球逆轉仍居第3，紅軍少賽1場領先10分

2019-12-22 51條評論 1,109次閱讀 140人點贊

昨晚今晨，英超聯賽第18輪比賽開打，曼城主場3-1逆轉萊斯特城，瓦爾迪先拔頭籌，馬赫雷斯、京多安和熱蘇斯分別破門完成逆轉；埃弗頓主場0-0戰平阿森納，安切洛蒂和阿爾特塔現場觀戰；狼隊客場2-1擊敗諾維奇，賽斯和吉梅內斯分別進球逆轉。

曼城主場3-1逆轉萊斯特城，距離對方只差1分。第21分鐘，萊斯特城打出反擊，瓦爾迪左路突進挑射破門；第29分鐘，馬赫雷斯內切後射門擊中瑟雲聚破門，扳平了比分；第40分鐘，斯特林禁區內突破被佩雷拉碰倒，主裁判吹罰點球，京多安主罰命中；第68分鐘，德布勞內右路突破後橫傳，熱蘇斯後點搶射破門。

南安普頓客場3-1擊敗阿斯頓維拉，第21分鐘，客隊朗反越位射門被撲出，因斯補射破門；第31分鐘，客隊普勞斯開出角球，斯蒂芬斯頭槌攻門擴大比分，第51分鐘，因斯禁區內轉身抽射梅開二度；第75分鐘，維拉隊長格拉利什禁區外凌空抽射直掛死角。

其他比賽中，埃弗頓主場0-0戰平阿森納，連續3輪對陣BIG6保持不敗；伯恩利客場1-0擊敗伯恩茅斯，羅德里格斯打進絕殺；謝菲聯客場1-0擊敗布萊頓，麥克伯尼打進唯一進球；紐卡斯爾主場1-0擊敗水晶宮，阿爾米隆破門；狼隊客場2-1逆轉諾維奇，賽斯和吉梅內斯下半場連入2球。
