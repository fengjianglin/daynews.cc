一場國家德比後才發現，那美好已過10年，感嘆現在只因忘不掉過去

2019-12-21 1,155條評論 5,001次閱讀 792人點贊

巴薩對陣皇馬的國家德比以平局收場，巴薩坐鎮主場的表現不能夠讓球迷感到滿意。現在的巴薩早已經不是當初的巴薩，雖然梅西、布斯克茨、皮克還依然在，但早已經不是「夢三時代」的樣子。有個很巧的事兒，那就是這場國家德比的比賽日這天，正好是巴薩獲得六冠王十周年的紀念日。十年前瓜迪奧拉帶領巴薩宇宙隊完成了足壇歷史上唯一一次六冠王壯舉。

沒有對比就沒有傷害，看看那個時候巴薩的橫掃歐洲，再看看現在巴薩踢個西甲保級隊都很難在場上有絕對的主動。10年過去了，這落差的確太大了。尤其十年後的這一天又趕上國家德比，在場上巴薩又是被皇馬「按著」踢，怎能讓人不敢傷。那個時候即使賽程密集，巴薩依然在伯納烏6比2大勝皇馬，而現在在主場都得靠「運氣」來保平，巴薩真的失去了太多的東西：

沒了懂巴薩的主教練

瓜迪奧拉之所以能夠在巴薩獲得那麼高的成就，去了曼城和拜仁就大打折扣。那是因為瓜迪奧拉的足球理念就是百分百的巴薩足球，而無論拜仁還是曼城都無法複製出一個新的巴薩。其實除了這兩隊，其他球隊也複製不出新巴薩，因為巴薩的傳控踢法很「特別」，外來的人想融入很難，自家培養的出去想成巨星也不容易。巴薩有太獨特的自我特點，所以需要一個特別能夠悟透巴薩足球哲學的主教練，瓜迪奧拉自然是最成功的一個。

在瓜迪奧拉離開以後，巴薩想繼續用有巴薩基因的主教練來延續輝煌。後來者基本都在吃瓜迪奧拉留下的老本，再也沒有人做到能把傳控達到極致的效果。雖然恩里克帶領巴薩也拿過五冠王，其實那個時候的巴薩已經開始在體系中融入一些防守反擊因素在裡面。想再找一個瓜迪奧拉這樣的接替者哪有那麼容易，我很贊同一句話「中場出身的球員更有潛質成為出色的教練」。因為中場球員在場上的位置和作用，會讓他們的大局觀相對優於其他位置上的球員，所以當他們成為主教練的時候就更加能洞悉場上的一舉一動，即使發現本方問題又能抓住對手漏洞。

想成為瓜迪奧拉在巴薩的接班人，還得滿足一點那必須要是巴薩青訓培養出來的，對巴薩足球能夠有足夠深刻了解的人，現在的巴爾韋德明顯做不到這一點，雖然他在巴薩踢過球，但也只能算是一個過客而已。所以他的足球理念是很難和巴薩百分百契合的，現在真正應該等的人就是哈維了。只是不知道還要等多久，如果科曼真的在本賽季末接替巴爾韋德，他的帶隊成績未必能好到哪去。懂巴薩的人，我們只能依然期待中。

沒有了精氣神兒

這點是現在巴薩比較要命的地方，時常能發現巴薩球員在場上很焦急，但又不看出精氣神兒。什麼是精氣神兒，精氣神兒就是讓人看到球員就覺得勝利在望。而現在的巴薩展現給我們看到的只是死氣沉沉，三條線上都沒有太多的活力，跑動方面也沒有那麼積極。本賽季巴薩最能跑的球員是新來的德容，他也是巴薩本賽季出場時間最多的球員，但是他一個人喚不醒整支巴薩。如果普約爾還在的話那該有多好，他在場上就不會允許隊友垂頭喪氣，他會用自己的鐵血來讓隊友振奮，也會用振臂高呼讓隊友保持注意力。普約爾最牛的地方就在於，雖然很硬氣，但是在場上又能保持十分冷靜的頭腦，不會因為對手的挑逗就失去理智和對手大打出手。

沒有了拉瑪西亞的輝煌

曾經巴薩在場上可以派出11位來自拉瑪西亞的球員，而這次對陣皇馬的國家德比就只有4位了，這四位中只有羅貝托一人算是「新人」，其他三人都過了30歲。真的很想看到巴薩的一線隊有更多的拉瑪西亞青訓球員，即使他們能力不那麼出眾，會影響到一些比賽的最後結果，我也願意看這樣的比賽，因為他們就是巴薩的根基和未來。巴爾韋德沒有給他們太多的機會，其實也能夠理解。除了說巴爾韋德自己沒有魄力以外，也因為現在的一部分偏激球迷早已經失去了對巴薩的「理智」，球隊不拿三冠王，不踢得漂亮加贏球，都會被罵是失敗的賽季，巴爾韋德和高層哪敢以輸掉比賽為代價來給青訓小將更多機會。

再者，巴薩的這些「老球員」們早已經都混成了球隊大佬，他們只要不願意讓位置，哪個主教練也不敢動他們的位置。綜合上述原因，現在的拉瑪西亞很難短期再輝煌，至少要等到夢三這一波人都退役後才有希望，至少要等到巴薩球迷中的冠軍粉都不在了，大家都能夠客觀看待巴薩的「低谷期」問題的時候，就是拉瑪西亞重新崛起的開始。

沒有了永動機，巴薩寸步難行

巴薩的永動機就是中場，就是「哈白布」組合，現在就剩下布斯克茨一人根本沒辦法支撐起中場，雖然後來者無數，但無人再能把傳控踢成哈維和伊涅斯塔那樣。就如本文前面所說，巴薩的傳控足球很特別，並不是一名巨星球員安插到巴薩中場就能夠踢出完美的傳控足球，對中場球員的要求更為苛刻，所以哈維和伊涅斯塔的接班人太難找。要不就維持現在這樣中場四不像；布斯克茨踢傳控、拉基蒂奇踢反擊、德容直上直下踢快速足球等等；要不就是徹底放棄傳控，打造新中場。以「現實主義」來看，第二種選擇明顯會更加容易一些，也更加容易出效果。比達爾總能在巴薩當奇兵就有一定這樣的道理在裡面。

沒有時光機，剩下的竟是歲月痕迹

歲月太殘酷，是因為他從不會為誰而停留。巴薩的輝煌過去，留在梅西、皮克和布斯克茨臉上的只有歲月的痕迹。他們依然在拼搏著，但一個王朝開始走向低谷的時候，問題會多得讓你力不從心。有一次詞我們用在朝代更替叫「無力回天」，其實現在的巴薩也是如此。

如果能夠有時光機該多好，讓現在的梅西、皮克和布斯克茨都能夠年輕5歲或者更多，巴薩現在很多的問題或許都已經不是問題。

如果能夠有時光機該多好，讓夢三的這些人都穿越到現在成為「小將」，然後開啟新的「夢三王朝」。

如果能夠有時光機該多好，讓巴薩宇宙隊無限次循環著，至到生命的終點。

10年時間在我們個體的生命說長也長，說短也短。但是在競技體育領域，10年足夠換一代人，也足夠讓一個巔峰王朝球隊昔日榮光不再。看看昨日的輝煌，10年後再看巴薩，作為巴薩球迷自然會有很多感慨。我們忘不掉過去，是因為過去太美好。值得慶幸的是，我們經歷過那獨一無二的美好，現在需要做的就是等待下一次春暖花開，芳香自來。

你知道嗎？

冬日裡，

沒有風，

便是這個世界上最大的善意。

現在的巴薩就是在冬日裡，如果我們無休止的指責，那就是寒冬里從未停止過的寒風。
