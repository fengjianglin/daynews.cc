游泳冠軍劉子歌為何嫁給53歲恩師教練？劉子歌低調產女後生活幸福

2019-12-24 252條評論 2,292次閱讀 601人點贊

中國體育事業迅速發展的同時，越來越多的人開始成為職業的體育運動員，同時也促使了體壇情侶的產生，有了同樣的目標，再加上朝夕相處，男女運動員之間很容易產生感情，但曾是我國蝶泳大將劉子歌愛慕的人並不是隊友，而是恩師。30歲的劉子歌做出了一個重大的決定，那就是嫁給已經53歲的恩師教練，雖然有人不理解，但是結婚以後，她告訴外界，這才是她想要的生活，兩人在一起相處很舒服，生活很幸福。

打破世界紀錄

從2004年開始，劉子歌就在世界盃短池游泳錦標賽開始嶄露頭角，在女子混用200米比賽中獲得冠軍，2007年，劉子歌進入國家游泳隊，並在當年的全國游泳冠軍賽女子200米蝶泳中獲得第一名。最讓劉子歌開始紅遍全中國的還是2008年北京奧運會上，女子蝶泳200米決賽中，她一直以第1名的優勢甩開其他對手，最終用冠軍證明了自己的實力，也成功打破原有的世界紀錄。

生涯成就

作為中國游泳運動員的劉子歌在職業生涯中獲得了很多成就，2007年，18歲的劉子歌進入國家游泳隊，開始征戰國際賽場。主攻蝶泳的劉子歌在比賽中優勢明顯，獲得了很多獎牌，2007年的全國游泳冠軍賽上，劉子歌被教練委以重任，參加了蝶泳女子200米的比賽，憑藉實力，她闖進總決賽，最終獲得了勝利，拿到金牌。

讓劉子歌一鳴驚人的還是2008年北京奧運會上，她不僅拿到了女子200米總決賽的金牌，還打破了之前的世界紀錄，為國爭光。劉子歌的能力很強，而她的婚姻生活同樣備受關注，因為她的丈夫不是同訓練隊的隊友，不是明星，不是模特，也不是企業家，而是曾經擔任過她的教練的金煒。

嫁給教練

年輕時期也是中國游泳領域人才的金煒曾進入國家隊，退役後的執教經驗相當豐富，也指導過劉子歌，而他也比劉子歌大23歲。這段跨度有點大的戀情在當時引起了不小的轟動，年輕有為的劉子歌完全可以找一個年輕一些的男友，她的戀情一時間讓人難以接受。2016年，劉子歌和金煒步入婚姻，結婚以後的劉子歌也公布了她當時為什麼會選擇金煒的原因，在她最低落的那段時間裡，是金煒在身邊一直鼓勵她，給予她信心和關愛。

結婚以後，金煒還一直照顧她，成熟穩重的金煒給她創造出了一種穩定的家庭環境，讓她沒有壓力的生活。從劉子歌如今的狀態中也可以看得出來，她很幸福，已經30歲的劉子歌顯得很年輕，和二十多歲的年輕人相差不大，擁有幸福穩定的婚姻，才讓她越來越年輕！

劉子歌低調產女

奧運會蝶泳冠軍劉子歌與其教練金煒的婚姻想必很多朋友都知道，去年27歲的劉子歌選擇交給了大她23歲的教練金煒，引起了不小的轟動，大家都覺得這樣懸殊的年齡差距婚姻有很多不妥，但這畢竟是劉子歌自己的選擇。如今，這段婚姻已經走過了1年多的時間，就在前兩天劉子歌順利的產下一女，51歲的丈夫金煒第三次當上了父親。

劉子歌是2008年北京奧運會蝶泳冠軍，打破世界紀錄奪冠，風頭無二，但是這位奧運冠軍十分的低調，不上社交平台，沒有商業廣告（在游泳中心的安排下拍過2個廣告，代言收入僅有1.66萬元），極少出現在公眾的視野當中，可以說是最低調的奧運會冠軍。在去年無緣里約奧運會之後，劉子歌也正式退役，她做出的婚姻決定十分的轟動，劉子歌決定嫁給年長她23歲的恩師金煒，對於這樣的選擇她稱道，從9歲起，自己就跟著金煒，已經習慣金煒幫忙安排一切，自己什麼都不用操心，父母也十分的感激金煒，可以說沒有金煒，就沒有劉子歌的今天。

但是這段年紀相差23歲的婚姻，還是引起了不小的討論。在感情上，劉子歌是全心投入的，她把自己獎金全部交給了丈夫金煒，給他錢買價值百萬的路虎豪車。兩人婚後的生活也是十分低調，就連劉子歌懷孕後去產檢也是去的最普通的婦產醫院，沒有任何的調子和隱藏。51歲的丈夫金煒之前就與前妻育有一兒一女，劉子歌也與兩個孩子相處得十分的融洽，如今劉子歌再生一女，金煒也是第三次當上父親。

婚後生活幸福

游泳事業已經如此成功的劉子歌也擁有一段備受關注的戀情，因為與她相戀的男友身份很特殊，既不是體育界的運動員，也不是娛樂圈的男明星，而是和她年齡相差20多歲的游泳隊教練。曾經擔任中國游泳隊教練的金煒主動發掘出劉子歌的游泳天賦，並且對她悉心培養。

在劉子歌事業最低谷的時候，是金煒在她身邊一直鼓勵著她突破困難，因為長久的陪伴，漸漸的讓劉子歌開始對他有好感，甚至逐漸動情。2016年，27歲的劉子歌和金煒步入婚姻，相差二十歲，卻在結婚以後非常幸福，過後，劉子歌表示自己喜歡成熟穩重的男人，儘管丈夫比她年齡大很多，但這讓她更踏實，如今，生下女兒的劉子歌非常幸福！

劉子歌於1989年出生於遼寧本溪，劉子歌擅長蝶泳，由於劉子歌的身體素質非常適合游泳，而且她的水感很好，所以她很早就開始接觸游泳並對游泳產生了很大興趣。劉子歌有很好的游泳天賦，因此她受到了教練金煒的青睞，2004年的時候劉子歌在世界盃短池賽俄羅斯站200米混合泳中榮獲金牌！由於表現突出，2005年劉子歌進入上海游泳隊，在經過了兩年的辛苦訓練，劉子歌的游泳技術日益提高，終於2007年被選入國家隊。

進入國家隊後的劉子歌迎來了職業生涯的巔峰時期！在2008年的北京奧運會女子200米蝶泳比賽中，劉子歌具有超強的衝刺能力，最終以2分04秒18的優秀成績打破世界紀錄，獲得金牌！

2009年也是劉子歌豐收的一年，7月劉子歌在羅馬世錦賽中獲得女子蝶泳200米亞軍，10月份在十一運會上，劉子歌打破亞洲記錄，以超高的成績56秒07奪得100米蝶泳冠軍！11月份劉子歌在世界盃短池游泳系列賽斯德哥爾摩站再次以優異的成績打破世界紀錄，獲得金牌！因為劉子歌打破了很多世界紀錄，斬獲了很多冠軍，受到了很多人的喜愛和追捧因此被人們稱為「蝶後」。

劉子歌的游泳生涯獲得了大滿貫，她的感情生活也備受人們關注。在2016年的時候劉子歌與教練金煒在北京舉行婚禮，很多人知道這個消息的時候大吃一驚，因為金煒不但有過一段婚史，年齡還比她大足足23歲！
