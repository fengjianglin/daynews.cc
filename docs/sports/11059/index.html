2-1拿下里爾，切爾西明確了1月轉會目標，頂替阿扎爾最好的還是他

2019-12-11 2,439條評論 4,921次閱讀 3403人點贊

近日，蘭帕德表示，切爾西仍在尋求取代艾登•阿扎爾的人，現在他們的轉會禁令已經解除，將在1月有所動作。

憑藉塔米•亞伯拉罕和塞薩爾•阿茲皮里庫塔進球，藍軍周二晚上在斯坦福德橋以2-1拿下了里爾，闖入歐冠16強。

蘭帕德從法國球隊身上獲得了急需的東西，但這並沒有阻止他們下個月改善球隊的計劃。

當切爾西主帥被問及一月份的目標時表示，「我們仍然在尋找阿扎爾的替代者。但現在情況有些變化。從現在開始，我們需要考慮重點加強哪些方面！」

「隨著艾登的離開，我們在那個特殊的位置失去了很多動力和創造力。我們已經錯過了一些機會，但我們仍然會努力地去競爭。」

在352場比賽中打進110個球後，阿扎爾今年夏天轉會至皇家馬德里。

塔米•亞伯拉罕在對里爾的比賽中打進了本賽季的第13個進球，輕鬆成為本賽季藍軍的頭號得分手，普利西奇在所有比賽中也有6個進球，排在球隊得分榜的第二位。

克里斯蒂安•普利西奇已經頂替了阿扎爾的位置，但是蘭帕德承認他需要在一線隊找到更多的進球來源。

近段時間，切爾西一直與多特蒙德的英格蘭球星桑喬聯繫在一起，有報道稱藍軍正在考慮以1億英鎊的價格收購這名想要離開多特蒙德的年輕球員。
