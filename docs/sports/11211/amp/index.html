沒投籃真不行！廣廈正式裁掉CBA最貴外援，場均17+10比周琦差遠了

天天要聞
2019-12-10

在大概20天之前，上賽季在NBA場均還能拿下12分8板的法里德來到中國加盟浙江廣廈隊，初來之時，很多球迷都表示這就是CBA的第一外援，畢竟在幾個月之前，他還能給在NBA大殺四方，而對於這種級別的外援，廣廈隊也是第一時間獻上了最貴的合同，他們與法里德簽下了一年440萬美元的合同，超越了此前北控外援德爾加多年薪430萬美元合同，成為CBA年薪最高外援。但是就在今天，在僅僅打了7場比賽之後，有媒體報道，目前廣廈已經裁掉了法里德。

但是頂著第一外援的名頭來到CBA之後，法里德卻表示的極其不適應，在為廣廈隊出戰的7場比賽里，法里德場均只能貢獻17.3分10籃板，從數據來看比他在NBA時期都沒有提升太多，這樣的數據甚至連當初被他在NBA擠走位置的周琦也不如。

在CBA這不到一個月的時間裡，法里德的表現其實也算不上太差，只能說是中規中矩，但是他有一個很大的問題，就是和廣廈的重點培養對象胡金秋同質化，搶了胡金秋的很多機會，導致胡金秋個人表現下降，還有就是他投籃能力不行的缺點被放大， 不能夠為隊友拉開空間，如今小球興起，像這種內線技術很差，投籃能力，純靠彈跳來立足的內線已經越來越難打了，估計福建的斯塔德邁爾也差不多了，所以名字大不如適合球隊，比如不要整天盯著nba，歐洲可以多看看，他們更容易熟悉cba規則而且還不貴。

對於法里德來說，他這種類型的球員，適合的是攻擊能力強悍，需要你來個干臟活累活衝擊前場籃板的，那麼法里德會適合這個球隊，然而目前的CBA不存在這樣的隊伍。沒有自主攻擊能力就是原罪，所以說只能面臨被淘汰的結局。
