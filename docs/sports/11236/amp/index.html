從東亞杯看中國足球：可能比賽還沒開始，我們就已經輸了

天天要聞
2019-12-10

12月10日（今晚18點30分），本屆東亞杯即將開賽，中國隊首戰將迎戰老對手日本隊，對於這場比賽，恐怕不管結果如何我們都無法高興的起來。

不是有意唱衰中國隊，只是今年國家隊的表現我們都看到了，不是我們不努力，只是確實實力就是這樣，想在亞洲突圍真的很難。再加上近些年辦的紅紅火火的中超今年也是爭議不斷，連年終的頒獎典禮都引來了大量質疑聲，下賽季的政策也是遲遲不宣布。請原諒！這樣的環境下真的無法力挺國足。

再回到東亞杯，為什麼說高興不起來？還是看看中日韓三支球隊的陣容吧！

首先來看日本隊，本屆東亞杯日本隊所有球員都來自J聯賽，而且清一色都是年輕球員，有16位球員年齡不到25歲，更有10位是首次代表日本隊參賽，全隊平均年齡只有23歲，人家這才是真正意義上的選拔隊啊！

至於韓國隊，陣容就相對成熟得多，不過也全是K聯賽的球員，並沒有在歐洲效力的球員，全隊平均年齡不到27歲，同時身價也是最高的。雖然陣容比較成熟，但是畢竟主場作戰，人家韓國隊肯定是力爭奪冠的，也想打破東道主無冠的魔咒，這樣的出場陣容可以理解。

最後是我們的國家隊，相比日本全部由新人組成的陣容，中國隊的陣容和韓國隊類似，基本是由邊緣國腳組成，其中還不乏像張稀哲、于大寶、韋世豪這樣的國腳級球員，可以說這是一支介於國家隊和國家二隊之間的球隊。

這也就看出了足協的意圖，這真是一支「選拔隊」，不過是為世界盃預選賽而做的選拔，這可能也是近期足協唯一的目標了。相比之下，日本這支選拔隊則是為了明年的本土奧運會，所以全部由新人組成。至於韓國隊人家就無所謂什麼選不選拔了，畢竟人家打進世界盃沒什麼問題，只要陣容差不多能奪冠就好。

雖然東亞杯的比賽央視會有直播，但我們依然無法抱有太多期待，我們也無法高興的起來。相比日韓，本屆東亞杯對於我們的國足來說有什麼意義呢？

和韓國隊比我們無疑是弱勢的一方，輸球估計在所難免，不過好在輸給這樣一支韓國隊也沒有什麼遺憾的。關鍵是和日本的首場比賽，面對這樣一支全部由新人組成的日本隊，我們能戰勝他們嗎？

假設能，這樣就值得我們高興嗎？人家是國家三隊甚至四隊，我們可是准主力出戰，而且以大打小，勝了只是再正常不過的結果了。但是萬一面對這樣一支日本隊我們依然無法戰而勝之呢？等待我們國足的又會是什麼？口誅筆伐估計是避免不了的了。

算來算去，也只有和中國香港的比賽值得期待了，雖然也不一定有十足的把握能戰勝中國香港，但最起碼算是最有希望的一場。

所以，酸辣君覺得別在乎本次東亞杯的排名，不用比就知道，冠軍沒希望，別墊底就好。

跳出比賽成績和排名這樣沒有意義的層面，我們看到的是日本、韓國巨大的後備力量。這麼多年來，可以說在亞洲範圍內，日韓基本不用派出主力陣容就可以佔有一席之地，而我們的國足卻依然要靠著大牌教練和全主力陣容甚至外加歸化球員才能勉強立足。這才是我們的真正差距！

國家隊層面其實我們早就習慣了，我們也不指望能趕超日韓，但是聯賽層面我們也沒有了以往的強勢和底氣。曾經還可以靠著恆大這樣的模式在亞冠賽場碾壓日韓球隊，但是近幾年隨著恆大實力的下降，聯賽泡沫的破滅，我們才突然發現，在俱樂部層面國內球隊和日韓球隊的差距其實一點也不必國家隊小。當浦和紅鑽這樣的J聯賽保級球隊都可以打進亞冠決賽時，我們還拿什麼去和他們爭？

條條大路通羅馬！只是日本、韓國已經找到了適合自己的路，中國足球卻依然在探索的過程中，何時能回到正軌上，怕是要打上一個大大的問號了！

作為本次國足選拔隊的主教練，李鐵雖然遺憾落選了中超最佳教練，但是本賽季武漢卓爾的表現已經證明了他的執教能力。這次帶隊征戰東亞杯，可謂是意氣風發。只不過對於中國足球來說，東亞杯的成績好壞並沒有什麼實際作用，但是之於李鐵來說確實是一次難得的展示自己的機會，更是一次對本土年輕教練的考核。

酸辣君覺得與其關注這支國足選拔隊的表現，倒不足把注意力放在主教練李鐵身上，這可能也是足協想要關注的點。

我想就算這支由一眾准國腳組成的選拔隊不幸再次讓我們失望，但是若能看到主教練的閃光也是一個收穫。希望聯賽的好運能帶到東亞杯，李鐵也能給我們帶來中國足球的希望！
