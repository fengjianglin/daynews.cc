不是大滿貫，卻比大滿貫選手還厲害，她與大滿貫之間只差一場比賽

2019-12-19 2,600條評論 4,533次閱讀 1969人點贊

國乒一向是一個非常強悍的體育項目，甚至有的球迷這樣調侃國乒的強大，那就是，在國平隊伍中，連掃地的大媽都是世界冠軍，更不養提其他人了，劉國梁此前在訓誡眾人的時候，就非常嚴肅的告訴隊員:不要以為拿了幾個世界冠軍就覺得自己厲害了，在國乒的隊伍中，最不缺的就是世界冠軍，這話聽起來是不是有點不太對勁啊，什麼叫做幾個世界冠軍，難道拿了幾個世界冠軍不厲害嗎？真是無知限制了小編的想像。

其實這話在劉國梁口中說出來一點兒也不奇怪，畢竟劉國梁是乒乓球大滿貫選手，他是完全有資本說出這樣的話的，相信比較了解國乒歷史的人都知道，劉國梁是第一位拿到大滿貫這個榮譽的人，非常的厲害，就是因為有了劉國梁開的這個好頭，接下來國乒才會發展的如此迅猛，現在誰也沒辦法比國乒相匹敵，國乒就是世界乒壇無人可以撼動的霸主。

在女隊中，也產生了很多優秀的大滿貫選手，而今天小編劇要給大家介紹一位乒乓球運動員，雖說她目前的職業生涯中，並沒有拿到過大滿貫，但她的實力卻並不比大滿貫選手的差，她就是劉詩雯，劉詩雯現在堪稱是國乒的老將了，在今年的比賽中，劉詩雯的表現很有大將風範，已經28歲的劉詩雯仍然保持在自己的巔峰狀態，當時劉詩雯出道的時候，就被大家所看好，甚至還有的人說劉詩雯是「小鄧亞萍」，可見劉詩雯的實力是多麼的「恐怖」。

劉詩雯進入國家隊的時候，還不滿13歲，一般這個年紀的孩子還在打少年組的比賽，但是劉詩雯已經可以獨當一面了，07年的時候，劉詩雯就在全錦賽中，取得了女單冠軍，當年劉詩雯也不過才16歲的年紀，少年得志的劉詩雯並沒有因此而自滿，而是繼續虛心的學習更多的乒乓球技巧，讓自己的乒乓球實力更上一層樓。

在2009年的時候，劉詩雯就獨挑大樑，參加了世界盃，在這場比賽中，劉詩雯擊敗了李曉霞，要知道李曉霞可是乒乓球大滿貫選手，劉詩雯也因此奪得了自己第一個世界盃冠軍，現在劉詩雯距離乒乓球大滿貫只差一個奧運會的男單冠軍，在明年的東京奧運會上，如果劉詩雯能夠一舉奪冠的話，那麼她就會成為國乒女隊的第6位大滿貫，不知道大家認為劉詩雯行不行呢，歡迎留言哦。
