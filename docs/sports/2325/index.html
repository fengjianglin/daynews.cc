朱婷繼續缺陣 天津背靠背0比3脆敗！將與廣東恆大爭世俱杯第七

摘要

北京時間12月7日，2019年女排世俱杯進行了爭奪5-8名的交叉賽，中國天津隊以0-3不敵巴西米納斯隊，將與廣東恆大隊爭奪第七名，三局的比分為：23-25、26-28、19-25。

北京時間12月7日，2019年女排世俱杯進行了爭奪5-8名的交叉賽，中國天津隊以0-3不敵巴西米納斯隊，將與廣東恆大隊爭奪第七名，三局的比分為：23-25、26-28、19-25。

天津隊的首發陣容為：主攻於鋆偉、李盈瑩，副攻袁心玥、王媛媛，二傳姚迪，接應胡克爾，雙自由人孟子璇和劉立雯。米納斯隊首發派上二傳馬克里斯，接應布魯納，主攻麥克倫丁、阿科斯塔，副攻塔伊薩、維維安，自由人雷亞。

首局雙方爭奪激烈，比分交替上升，戰成12平。此後李盈瑩強攻打手出界，對方一攻得手，袁心玥快攻打手出界，雙方戰成16平。胡克爾後攻出界，對方進攻得手，袁心玥扣球得分，17-19。李盈瑩強攻打中，胡克爾反擊打手出界，天津21-20反超。對方連續進攻得手，李盈瑩吊球得分，22平。對方連續進攻得分，李盈瑩進攻打手出界，23-24。塔伊薩前快打中，米納斯以25-23先勝一局。

第二局米納斯隊打的順手，進攻和攔網不斷得分，以14-9領先。袁心玥快攻打中，對方連續進攻得手，米納斯16-11領先。於鋆偉強攻打手出界，隨後她反擊得分，13-16。胡克爾強攻得手，對方扣球出界，天津追成15-17。王媛媛扣球出界，胡克爾一攻下球，對方後三出界，17-19。隨後袁心玥快攻打中、袁心玥和胡克爾雙人攔網得分，天津追成19平。此後雙方比分交替上升，戰成22平。

布魯納扣球出界，胡克爾後攻打中，天津24-23領先。24平後於鋆偉一攻得手，李盈瑩反擊被攔，25平。布魯納推後排空當得分，李盈瑩一攻打中，26平。塔伊薩前快打中，於鋆偉一傳失誤，米納斯28-26再勝一局。

第三局雙方比分交替上升，戰成8平。此後天津隊進攻受阻，對方連續得分，以12-9領先。此後對方不斷進攻和發球得分，以18-13領先。袁心玥扣球下網，胡克爾發球失誤，米納斯以23-18領先，並以25-19取勝。

於鋆偉拿到全隊最高的13分，李盈瑩和胡克爾均得到12分。對方塔伊薩得到本場最高的15分。

在另外一場爭奪5-8名的交叉賽中，中國廣東恆大隊以0-3負於巴西海灘隊，三局的比分為：22-25、20-25、14-25。巴西海灘隊將與米納斯隊爭奪第五名，天津隊將與廣東恆大隊爭奪第七名。
