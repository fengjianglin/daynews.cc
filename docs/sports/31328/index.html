沙爾克天才門將離隊接班諾伊爾，這一切都是拜仁“挖掘機”的錯嗎

2019-12-24 1,754條評論 4,815次閱讀 904人點贊

據德國《圖片報》消息，本賽季在沙爾克04異軍突起的年輕門將努貝爾將於明夏自由身加盟拜仁，等待接班諾伊爾。2011年，諾伊爾從沙爾克加盟拜仁，而今，同樣在「礦工」擔任隊長的22歲天才門將來投，沙爾克為拜仁的防線著實做了不小的貢獻。近年來，雖然沙爾克多次躋身德甲前列，屢屢出現在歐冠賽場，但仍止不住人員流失，幾乎每個賽季都會失去重要球員，「挖礦」成了他們擺脫不掉的命運。

努貝爾下賽季去拜仁接班諾伊爾

被主席「逼走」的核心

「今天的年輕人金錢的喜愛遠遠超過了對於俱樂部的熱情，在這個時代要找到陪著球隊同甘共苦的球員可是太難了。」2017-2018賽季臨近尾聲時，沙爾克主席特尼斯不顧球隊爭亞軍的大好形勢，怒噴隊中兩位核心邁爾和格雷茨卡，「科隆降入德乙後，他們的國腳（赫克托和霍恩）選擇留下來，我很敬佩。而我們的則不一樣，他們都和經紀人穿一條褲子。」多次尋求續約無果，眼見悉心培養多年的球員就要為他人做嫁衣，也難怪特尼斯沉不住氣了。就在特尼斯公開怒懟本隊國腳後，邁爾將社交平台簡介中「沙爾克球員」一項刪掉了，而賽季結束後，他便前往英格蘭加盟了水晶宮，球隊為他開出4年合同，總薪金超過3000萬英鎊，這樣的價錢在德甲只有奧巴梅楊之類的頂級球星才能拿到。

邁爾因為不續約遭遇冷遇

儘管在水晶宮大部分時間只能枯坐板凳，但邁爾卻並不為自己的選擇而後悔，畢竟當初沙爾克一心挽留格雷茨卡而忽視了相同位置的他。為了留下「磁卡」，沙爾克不但將赫韋德斯打發去了意甲，把隊長袖標交給他，還答應將300萬年薪提升到1000萬的檔次，這個數字就算拿到拜仁也能算頂級行列，接近托馬斯-穆勒，遠超諾伊爾、胡梅爾斯和羅本等一眾大牌，沙爾克可謂出手闊綽，誠意盡顯。只不過格雷茨卡「志存高遠」，不為所動，一心想要追求冠軍。結果是，關係莫逆的二人又「攜手」離礦而去，沙爾克又一次落得人財兩空，或許他們早已經習慣了這樣的結局。

格雷茨卡在拜仁得到了一切

沙爾克為何留不住人

足球的世界裡同樣以雙向選擇為主，球員可以挑俱樂部，球隊也能選球員。多數情況下，合同到期前2至3年，俱樂部就會展開行動。想留的要提早著手續約談判，不想留或是留不住的也要儘快推向轉會市場，賣個好價錢。為了彼此的權益，極少會有拖到最後時刻一拍兩散的情況出現。畢竟像畢爾巴鄂競技不願放略倫特，最終導致球隊失去轉會費，球員不受重視坐冷板凳導致狀態下滑的兩敗俱傷，是所有人都不願意見到的，和氣才能生財。此前，沙爾克已經多次做過類似的「止損」操作，比如2013年1月，他們將時任德國國青隊隊長霍爾特比賣去托特納姆熱刺，就只得到了200萬歐元的回報，原因無他，球員合同只剩半年到期，與其留著幾個月付工資還可能影響球隊整體，倒不如多少賣點錢。

霍爾特比以白菜價轉會熱刺

當然，沙爾克也並非每次都這麼順利，儘管他們從諾伊爾、薩內、克雷爾和德拉克斯勒身上賺到了不菲的轉會費，卻也在同期目送馬蒂普、科拉希納茨和舒波莫廷等眾多主力球員以自由身離隊，按照德轉的估價，沙爾克的損失至少有1.3億歐元之巨，對於這支德甲球隊而言，簡直就像失去了整個世界一樣。要知道，沙爾克在2018-2019賽季的轉會市場中投入創下隊史紀錄，凈花銷也不過區區6000萬。

科拉希納茨和馬蒂普如今已成英超頂尖後衛

還剩多少可以浪費？

既然球隊吸引力略遜，要麼更早的開始續約工作，要麼至少在合同還剩一年之時及時出售，即便被賣家「壓價」，也好過兩手空空。只是，沙爾克似乎太過寬容，他們習慣了被大牌球員用「拖字訣」將合同耗盡，最後只得目送青訓瑰寶離開。相比之下，早幾年多特蒙德也有過同樣的命運，陣中大牌幾乎被挖殆盡，球隊也從挑戰拜仁一落到險些降級。但俱樂部從管理下手，一方面及早確認主力球員前途，另一方面則將「黑手」伸向更多中小球隊，及早挖掘代替者。這就使得目前的多特蒙德，通過出售成名球星賺的盆滿缽滿，另一方面也不斷地獲得新鮮血液的補充，進而形成良性循環，逐漸凝聚起穩定的實力。

克雷爾是沙爾克作出為數不多的好生意

有不少聲音認為，拜仁的「挖掘機」損害了德甲眾多「小強」球隊的成長，但看到多特蒙德從賣人中走出獨立發展之路，其他俱樂部或許應該有所觸動。似乎只有沙爾克既無冠軍也無建設，高管換了一茬又一茬，絲毫不見起色，球隊文化與傳統漸減消失，有的只是短視與急功近利。皇家藍經營層面的混亂，而這恰是沙爾克育人卻不留人的根源所在。因為新主帥和體育經理要樹立權威，為球隊奉獻16年，忠心耿耿拒絕豪門邀請，帶傷出戰落下病根的老隊長赫韋德斯被掃地出門。而奧地利左後衛福克斯則被球隊欺騙，「直到賽季結束前1個月，他們還說對我很滿意，讓我不用擔心合同的事。」直到賽季結束，福克斯受到了球隊的逐客令，為沙爾克兢兢業業付出5年時間的他不得不前往英格蘭，隨後成為「萊斯特奇蹟」的參與者。類似的例子還有許多，花著大把大把冤枉錢，揮霍著本可以形成不俗戰力的「自有資源」，惡性循環下不僅是實力的損失，更是認同的缺乏。而這些，並不是簡單的把責任推到拜仁「挖掘機」身上就可以解決的。（zc）
