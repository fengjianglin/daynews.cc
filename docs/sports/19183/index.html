2019年度英超最佳球員，滿眼都是利物浦球員

2019-12-16 1,126條評論 4,042次閱讀 800人點贊

2019年已經接近尾聲了，歐洲主流聯賽2019~20賽季也已經接近中期，目前為止，在今年我們已經看到了許多不同球員的精彩表現。福克斯體育為大家盤點了一下2019年年度英超最佳陣容。看一下福克斯體育盤點的最佳陣容里有沒有你喜歡的球員。

門將-阿利森-利物浦

阿利森是上賽季利物浦贏得歐冠冠軍和英超亞軍的關鍵一員。

本賽季利物浦的開局十分強勢，在英超聯賽的積分榜上已經領先第2名，萊斯特成隊10個積分到目前為止，他們在本賽季只丟了14個球，這是在目前英超聯賽所有球隊中排名第二低的。

當然阿利森是利物浦可以保持現在狀態的重要因素之一，他在過去的幾個月里做出了一些精彩的撲救。從2018~19賽季的數據來看，他的撲救成功率高達79.2%。到目前為止，在本賽季他也取得了兩場不失球。

右後衛-阿諾德-利物浦

在右後衛位置，阿諾德是不錯的選擇，他在利物浦也有一個輝煌的賽季。在很短的時間內，這位英格蘭的年輕球員已經成為了世界上最好的右後衛之一，更不用說英超聯賽中了。

亞歷山大-阿諾德的速度是他和其他像他一樣的球員的不同之處——年輕球員的快速跑動在側翼很好地配合了利物浦的體系，這個體系由同樣快的跑者組成，像薩迪奧·馬內和穆罕默德·薩拉赫在前面。今年到目前為止，他已經進了一個球，送出了7次助攻。

中衛-范迪克-利物浦

歐足聯2019年最佳男球員，2019年金球獎的亞軍——還需要多說嗎？

范迪克不僅是英超最好的球員之一，也是世界上最好的球員之一。這位荷蘭中後衛在整個2018-19賽季中只有一次被帶球過人，這證明了他是一個多麼優秀的後衛。本賽季到目前為止，利物浦只丟了14個球，而這位28歲的球員是利物浦防守如此強大的主要原因之一。

中衛-詹姆斯·塔爾科夫斯基-伯恩利

除了范迪克，很少有中後衛能在2019年有好的表現。如果有一個人在某種程度上至少可以和范迪克相提並論，那一定是詹姆斯·塔爾科夫斯基。

這名27歲的後衛是伯恩利最好的球員之一，他本賽季在伯恩利沒有錯過一分鐘的比賽。

事實上，他在去年3月缺席的兩場比賽是他在伯恩利唯一缺席的時光。

左後衛-羅伯遜-利物浦

這裡還有另一位表現得更像邊鋒的後衛——安德魯·羅伯遜是上賽季利物浦的助攻領袖之一，在2018-19賽季中他貢獻了11次助攻。本賽季，到目前為止，他的助攻數又增加了5次——但不要搞錯了——這位蘇格蘭國腳也是一位同樣出色的後衛。

中場-詹姆斯麥迪森-萊斯特城

詹姆斯·麥迪森是我們今年最喜歡的中場球員——這位英格蘭和萊斯特城的球星是今年英超聯賽中最好的球員之一。今年到目前為止，他可能只進了10個球，貢獻了8次助攻——但每一次進球和助攻都是在萊切斯特的關鍵時刻，因此證明了他對球隊的重要性。

本賽季，狐狸隊在英超積分榜上排名第二，得分比他們令人難忘的2015-16賽季奪冠還要多——麥迪森的參與是他們成功的主要原因之一。

中場-法比尼奧-利物浦

法比尼奧上個賽季加盟利物浦，從那以後，這位巴西人就再也沒有後悔過，他在2019年至今已經打進2球，貢獻3次助攻。

在過去的幾個月里，這名中場球員經常是利物浦進攻的起點，他的帶球能力使他成為我們英超十一人中的多面手。

​中場-德布勞內-曼城

幾年來，德布勞內一直是曼城最好的球員之一。這名比利時中場的進攻意圖和傳球能力使他成為他職業生涯中最好的球員之一。

今年到目前為止，他只進了5個球，貢獻了10次助攻——但是不要讓這些「普通的」數據欺騙你——德布勞內的表現常常幫助曼城走出困境。最後，大多數為曼城的成功而喝彩的人是塞爾吉奧·阿圭羅和拉希姆·斯特林——但是在我們看來，比利時人似乎比其他任何人都更重要。

鋒線-馬內-利物浦

薩迪奧·馬內的入選是另一個顯而易見的選擇。這位利物浦邊鋒以英超最佳射手的身份結束了2018-19賽季，這個賽季，他已經進了9個球，貢獻了7次助攻。

馬內還在2019年金球獎的角逐中名列第四，排在梅西、范迪克和克里斯蒂亞諾·羅納爾多之後——這應該可以證明他在過去一年裡對利物浦的影響有多大。梅西還表示，馬內是他最喜歡的球員。

鋒線-瓦爾迪-萊斯特城

萊斯特城的瓦爾迪是典型的中鋒，他全年表現出色。這名32歲的球員最近失去了英格蘭主帥索斯蓋特的喜愛，但我們期待他儘快回歸球隊，因為——老實說——現在英超聯賽沒有比他更好的射手了。

在正在進行的2019- 2020賽季，他為狐狸隊出場17次，攻入16球，貢獻3次助攻——僅僅比他在2018-19賽季的進球數少了2個。

鋒線-奧巴梅楊-阿森納

奧巴梅楊可以說是目前阿森納最好的球員。他以22粒聯賽進球結束了2018-19賽季，這也幫助他在英超聯賽中首次獲得金靴獎。

在2019- 2020賽季中，奧巴梅揚已經在16場比賽中打入11球並提供1次助攻。加彭國腳最近的狀態對槍手來說至關重要，我們希望在接下來的下半賽季也能保持這樣的狀態。

這份陣容，是你所希望看到的嗎？2019年，英超最強是誰？
