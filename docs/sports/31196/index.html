勇士大勝森林狼！格林恢復巨頭本色，拉塞爾關鍵罰球鎖定勝局

2019-12-25 583條評論 1,961次閱讀 880人點贊

北京時間12月24日，勇士主場迎戰森林狼的挑戰，上一場勇士戰勝了鵜鶘，這次如果還能戰勝森林狼，就會取得新賽季第一次連勝，對勇士來說，賽季首次連勝的誘惑還是很大的，如果一直沒有一次連勝，勇士的戰績也太尷尬了。森林狼近期戰績不佳，一波十連敗到手，對森林狼來說，當然也想要戰勝勇士來終結十連敗，畢竟現在聯盟比勇士還弱的球隊不好找了，但是森林狼唐斯因傷繼續缺陣，枸杞哥維金斯單核帶隊有點不穩啊！

勇士畢竟還有雙巨頭在場，拉塞爾和格林一攻一防，球隊還是有一定戰鬥力，拉塞爾近期狀態不錯，前兩場比賽對陣鵜鶘拿下25分3籃板7助攻，對陣開拓者砍下26分7助攻，格林當然也想要率隊取勝，上一場戰勝鵜鶘，格林最後時刻的防守至關重要，格林本賽季遭到太多質疑，對格林來說，帶隊取勝可以擊破這些質疑，這次對陣森林狼，格林進攻端不會有太大貢獻，但想要贏球，格林的防守還是勇士取勝的一大法寶。

比賽剛開始兩隊進攻手感都不太好，首節戰成22-22平，第二節拉塞爾帶隊打出進攻高潮，單節打了森林狼一個35-21，幫助勇士一舉建立了14分的領先優勢，這還沒完，第三節勇士繼續擴大分差，一度把領先優勢擴大到20分以上，眼看比賽就要打花了，森林狼板凳席還是很猛的，第二陣容全力反撲，第三節結束時候把分差縮小到20分以內。第四節剛開始森林狼繼續縮小分差，一波攻擊高潮使得分差來到了個位數。

勇士雖然第二陣容被對手完爆，但是首發實力還是佔據優勢的，拉塞爾等首發上場之後迅速幫助球隊穩住局面，兩隊開始你來我往，分差一直保持在十分左右。比賽進入到最後三分鐘之內，森林狼球員馬丁三分命中，幫助球隊92-99把分差縮小到7分，關鍵時刻，達米安-李三分回應，緊接著森林狼維金斯出現失誤，伯克斯再命中一記三分，勇士105-92，瞬間把分差擴大到13分，森林狼這波反擊未能奏效，反而被勇士抓住機會擴大了分差，森林狼想要翻盤的希望不大了。

最後幾十秒森林狼沒有放棄，一邊快速得分縮小分差，一邊不停犯規送勇士球員上罰球線，最後22秒，麥克勞林上籃命中打成2+1，森林狼104-109暫時落後勇士5分，只有五分分差了， 科爾無奈喊了暫停，森林狼突然之間起勢，迅速縮小了分差，打了勇士一個措手不及，科爾當然不想到手的鴨子飛了，這場勝利本來可以穩穩拿下，沒想到最後時刻反而出現了一些威脅。

暫停回來維金斯對拉塞爾犯規，拉塞爾罰球非常穩定，兩罰全中後森林狼基本沒有逆轉的希望了，最終勇士主場113-104戰勝森林狼，收穫了賽季首次連勝，也送給了森林狼十一連敗。

全場比賽，勇士一方拉塞爾30分4籃板4助攻，伯克斯25分2籃板8助攻1帽，格林9分14籃板4助攻2斷1帽，考利-斯坦12分7籃板2斷3帽，達米安-李14分7籃板3助攻，帕斯卡爾8分2籃板2助攻2斷，克里斯6分11籃板。森林狼一方維金斯22分7籃板2助攻1帽，麥克勞林19分2籃板4助攻3斷，吉昂11分8籃板3斷1助攻1帽，蒂格12分4助攻，格拉漢姆8分3籃板

對於勇士來說，今夏成功得到拉塞爾還是非常不錯的，至少在庫里和克萊都因傷無法出場的時候，勇士進攻端還是拉塞爾能夠提供很大的火力。格林是勇士巨頭之一，雖然本賽季一度遭到質疑，但近期格林表現還是不錯的，這場比賽防守端格林絕對是頂級，一個人就搶下了14個籃板，並且罰球5罰5中，最後時刻的罰球也是頂住了壓力，可以說格林是球隊的靈魂，就算在現在這支勇士裡面，格林也是毫不爭議的巨頭，當然，如果這場比賽沒有7次失誤就更加完美了，不過也能理解，人無完人，格林重心全在防守上，進攻端出現一些失誤也在所難免，大家說是不是呢？
