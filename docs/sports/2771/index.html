英錦賽丁俊暉贏球將戰奧沙利文 塞爾比羅伯遜晉級 中國4人尚存

2019-12-04 0條評論 22次閱讀 0人點贊

摘要

北京時間12月4日消息，2019斯諾克英錦賽繼續進行。在第三輪的一場焦點戰中，丁俊暉6比4擊敗卡特，他將在下一輪迎戰火箭奧沙利文。此外，塞爾比、艾倫、顏丙濤等人也順利晉級第四輪。2019世界斯諾克英國錦標賽在英格蘭約克的巴比肯中心進行。

北京時間12月4日消息，2019斯諾克英錦賽繼續進行。在第三輪的一場焦點戰中，丁俊暉6比4擊敗卡特，他將在下一輪迎戰火箭奧沙利文。此外，塞爾比、艾倫、顏丙濤等人也順利晉級第四輪。

2019世界斯諾克英國錦標賽在英格蘭約克的巴比肯中心進行。今年是第43屆英錦賽，同時也是本賽季「三大皇冠賽事」的第一站。中國軍團25人全員出動參加了本次比賽，而國外選手方面，此前已經取得兩連冠的奧沙利文能否衝擊三連冠值得期待。值得一提的是，本次英錦賽總獎金也是第一次突破100萬英鎊，創造英錦賽最高記錄，其中冠軍獎金更是達到了20萬英鎊。

目前大師賽名額激戰正酣，丁俊暉和卡特都有機會入圍，而兩人一個排名15、一個16，是直接的競爭對手。比賽開始後，丁俊暉迅速進入狀態，雖然晉級表現一般，但本場一上來他連轟單桿64分、104分、52分和59分，瞬間取得了4比0的領先。要知道，在過去的一年多里，這樣酣暢淋漓的領先對於丁俊暉來說實在是久違了。

不過，卡特也絕非泛泛之輩，第五局，他先是單桿100分穩住局面，隨後又拿下85分連追兩局。第七局，丁俊暉單桿71分拿下拿到賽點，但隨後一局他僅僅得到19分便送出機會，卡特一桿連續上手後完成逆轉。關鍵的第九局，丁俊暉單桿52分沒能完成超分，卡特最後時刻逆轉以77比64拿下，再追一局後讓氣氛頓覺緊張。第十局，丁俊暉重新調整，他利用卡特開局階段留出的機會單桿116分拿下，從而6比4戰勝對手。

奧沙利文在本輪迎戰泰國名將桑坎姆。火箭本場手感一般，只拿下3桿50+，但桑坎姆這樣的對手完全不能阻擋火箭。最終，奧沙利文6比2輕鬆勝出，下一輪將對陣丁俊暉。顏丙濤在本輪的對手則是利索夫斯基。整場比賽，以進攻見長的兩人都沒能打出自己的手感，全場兩個僅僅各有2桿50+進賬。最終，顏丙濤6比4險勝對手，同樣晉級第四輪。　

其他場次中，梁文博6比4擊敗沙拉夫，馬奎爾6比2勝多特，塞爾比6比3勝奧唐奈爾，威爾遜6比1勝佩里。第四輪中，梁文博將和李行之間展開德比，希金斯對陣賓漢姆，顏丙濤對陣羅伯遜，再加上丁俊暉迎戰奧沙利文……場場都是焦點戰。
