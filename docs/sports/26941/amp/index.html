虐心！34歲的C羅凝視對手瘋狂慶祝奪冠，淚水在眼眶打轉

天天要聞
2019-12-23

北京時間12月23日凌晨0點45分，義大利超級盃決賽在沙特利雅得體育場開打，意甲冠軍尤文圖斯對陣義大利杯冠軍拉齊奧。1-3，尤文輸球，無緣冠軍。34歲的C羅非常失望，看著瘋狂慶祝的拉齊奧，C羅幾乎要落淚……

這場一戰定勝負的決賽，尤文圖斯再次排出迪巴拉、伊瓜因、C羅的3前鋒組合。拉齊奧方面，因莫比萊領銜出戰。34歲的C羅近來狀態非常不錯，連續5場比賽為尤文進球，尤文2-1擊敗桑普多利亞一役，C羅展現霸氣彈跳，打進衝上雲霄的頭槌破門，震撼世界足壇。

不過，拉齊奧卻是不折不扣的尤文剋星。尤文雖然坐擁迪巴拉、C羅、伊瓜因的三叉戟鋒線，但防守並不理想。第16分鐘，拉齊奧打破僵局，藍鷹軍團打出精彩配合，阿爾貝托10米處右腳推射破門。

尤文金童迪巴拉一度扳平比分，不過，小因扎吉執教的拉齊奧踢得更有章法，第74分鐘，盧利奇進球，拉齊奧2-1領先！尤文反撲，無奈老問題無法解決——中場串聯、輸送能力太弱，皮球無法交到C羅腳下。

第95分鐘，拉齊奧殺死懸念，卡達迪操刀主罰自由球，皮球從人牆博努奇的頭頂飛過，貼著橫樑和立柱，竄入右上死角！3-1！拉齊奧鎖定勝局，小因扎吉與全隊瘋狂擁抱慶祝。

看著狂歡的拉齊奧球員、教練、球迷，C羅有些沮喪。這場比賽，C羅有些孤單。奪冠、歡慶，本是C羅熟悉的場面，他的職業生涯，31次奪得冠軍獎盃，堪稱人生贏家。不過這個夜晚，C羅感受到輸球的滋味。對「永爭第一」的C羅來說，這種滋味很苦澀，很不好受。

鏡頭再次對準C羅，34歲的葡萄牙巨星眼裡含著淚，幾乎快要哭出來了。加盟尤文之後，C羅拿到過意甲冠軍、義大利超級盃冠軍，但這次義大利超級盃，尤文輸得徹徹底底。

C羅的決賽神跡，也宣告失守——過去12場決賽，C羅全部贏了，包括4次歐冠決賽，3次世俱杯決賽，1次歐國聯決賽。球迷期待著C羅上演連續13次決賽取勝奪冠的壯舉，可惜的是，C羅碰上了更出色的拉齊奧。本賽季意甲，尤文就1-3輸給拉齊奧，這次，斑馬軍團又倒在拉齊奧這道坎上。
