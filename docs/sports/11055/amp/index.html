利物浦1月轉會窗口目標鎖定巴西前鋒埃瓦尼爾森：我真的想留下來

天天要聞
2019-12-11

隨著離1月轉會窗口開啟的時間越來越近，利物浦的球探們也將目光放在了搜羅合適的球員上。年輕的巴西前鋒埃瓦尼爾森就是其中一位，在1月轉會窗口開啟前他就被利物浦球探給盯上了。

據報道，利物浦在與他的經紀人接觸後，已經與這位20歲的球員聯繫在了一起，並且密切關注著他在弗盧米嫩塞俱樂部的表現，據信目前還沒有正式的報價。

埃瓦尼爾森在巴西聯賽最後一輪擊敗科林蒂安的比賽中打進了他職業生涯的頭兩個進球，而他的合同將在2月到期，所以他下賽季將不會回到俱樂部。

在戰勝康蒂安之後，埃瓦尼爾森談到了他和利物浦之間的聯繫。

「讓我想想，聯賽現在已經結束了，我將和我的經紀人一起看看，我們已經談判了很長時間，我們在看什麼是最好的。」

「這些目標很有幫助，我在等著看會發生什麼，我不知道後面會發生什麼，這些都還不能說。這是一家向我敞開大門的俱樂部，我很高興，我真的想留下來。」

上個賽季，埃瓦尼爾森是巴西u20聯賽的最佳射手，並曾在斯洛伐克的薩摩林俱樂部短暫租借過一段時間，在那裡他六次出場進了三個球。

據巴西媒體Netflu報道，他的進球狀態已經引起了利物浦的注意，他們一直在關注著埃瓦尼爾森，並與他的經紀人取得了聯繫。

隨著埃瓦尼森的合同在2月份到期，弗盧米嫩塞可能準備以400萬英鎊的價格出售，而不是在明年白白失去他。

巴西足球記者布萊克利在接受《足球耳語》採訪時表示:「弗盧米嫩塞俱樂部資金緊張，所以即使出價400萬英鎊，這對一家英超俱樂部來說也只是一個相對微薄的數字，但這筆錢可以轉化為2100萬雷亞爾，這是俱樂部目前困境中的一筆財富。」

「巴西的球隊喜歡向英超聯賽和中國出售球員，但這並不僅僅是因為出價。從長遠來看，他們受到的輻射是有益的。」
