巴薩主席說漏新帥人選！功勛回歸已提上日程 加盟條件一併曝光

2019-12-10 1,434條評論 3,357次閱讀 1880人點贊

據西班牙《每日體育報》報道，巴薩主席巴托梅烏近日接受了媒體採訪，他表示巴薩隨時都歡迎瓜迪奧拉回歸，暗示巴薩有換帥的可能，一直飽受詬病的巴薩主帥巴爾韋德將要下課，而瓜迪奧拉則是巴薩新任主帥。另外，他還回應了巴薩關於勞塔羅-馬丁內斯和內馬爾的轉會傳聞。

巴托梅烏說道：「瓜迪奧拉能否回歸巴薩並不取決於我，當初離開這裡是他自己做出的決定，我能保證的就是巴薩隨時都歡迎他回歸。」

一直以來，巴爾韋德下課的呼聲從未停止過。此前有消息稱，巴薩下一任主帥將是名宿科曼，然而巴托梅烏的這一番言論似乎直接宣布了巴薩選帥的結果，曾在巴薩取得過輝煌成就的瓜迪奧拉才是接替巴爾韋德的第一人選。

隨後，巴托梅烏談到了關於梅西續約的話題。

「我希望在我離任前，可以每年都與梅西簽下新合同，或者簽一份終身合同，我們會像當年對待伊涅斯塔一樣對待他。他取得的成就使得他擁有了選擇的權利，梅西就是世界上最好的足球運動員，不僅是在現在，未來的幾年裡他還是會保持這樣的水準。巴薩的歷史被他改寫，同樣，世界足壇也因有他而變得不同。」

國米前鋒勞塔羅-馬丁內斯與內馬爾近期都與巴薩傳出了緋聞，巴托梅烏對此做出了回應。

「球隊在冬窗並沒有引援計劃，現在球隊的陣容非常均衡，各種類型的球員都有。同時，我們也不願讓任何一名球員離隊。比達爾不會在一月離隊，賽季結束後他可能會走。」

「關於勞塔羅的問題充滿了陷阱，我所能說的就是他在國米表現很好，他並非唯一一位給我留下深刻印象的球員，他的球隊在意甲領跑絕非偶然。」

「對於內馬爾離隊，我們也無能為力，因為巴黎直接支付了違約金。我和內馬爾談過，他有很多疑問。去年夏天巴黎總監表示願意出售內馬爾，我們已經竭盡全力想讓他回歸，但是巴黎的要價是無法接受的。」
