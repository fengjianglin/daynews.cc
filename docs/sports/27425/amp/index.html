魯能的何小柯17歲能在中超打上主力嗎？

天天要聞
2019-12-20

何小柯，U15國家隊主力前鋒，只要有他參加的比賽，幾乎每場都會進球，無論是國內賽場還是亞洲賽場。山東魯能足校為其制定了詳實的職業規劃，目前已經被足校提到U19年齡段進行歷練。

對於從15一下子跳到19，很多球迷覺得這是在拔苗助長。其實跳級不是球員決定，而是教練組決定的。對於足校來說，自己學校不同學段有什麼水平是很容易掌握的，再者只是在足校裡面又不是在真正的賽場上。其實魯能當年的射手韓鵬就曾經在16歲時被提拔到一線跟隊訓練了一年，然後回到當時剛剛成立的足校又學習了兩年，然後在一線隊慢慢成為主力球員。現在的何小柯和當年的韓鵬一樣，在同齡球員中有身體、速度，而且具有明顯的優勢，將其提拔到U19，在身體差距過大的情況下，參考武磊，要想上場比賽，就得需要用腦子踢球，畢竟這個年齡閱讀比賽的能力和踢球過程中決策的合理性對其是個考驗。山東魯能現在已經將其送往巴西基地，而且巴西基地正聯繫西班牙聯賽讓其鍛煉。 目前何小柯身高是1米76，何小柯目前的帶球突破、擺脫過人以及身體優勢很明顯的，有了這些技術基礎，再加上其觸球的感覺也不錯，加以歷練。

其實在中超，17歲打上比賽的不在少數，比如武磊，張琳芃，黃博文，崔鵬、周海濱。只是近年來，因為聯賽進入了依靠外援的時代，步入了畸形發展的時期，使得年輕球員得需要U23政策保護才能上場，可是真正的佼佼者並不會因此而埋沒，比如黃紫昌、蔣聖龍、朱辰傑、段劉愚等。

那麼，友友們對於何小柯17歲進入中超有什麼看法呢？
