決勝局1-4落後，19歲國球黑馬12-10，絕殺張本智和搭檔+日本冠軍

2019-12-08 0條評論 21次閱讀 0人點贊

摘要

北京時間12月8日，國際乒聯北美挑戰賽當中，中國19歲的小將徐海東爆出了大冷門，他在16進8的比賽當中，將日本隊的主力球員、張本智和的搭檔、日本冠軍吉村真晴淘汰出局。

北京時間12月8日，國際乒聯北美挑戰賽當中，中國19歲的小將徐海東爆出了大冷門，他在16進8的比賽當中，將日本隊的主力球員、張本智和的搭檔、日本冠軍吉村真晴淘汰出局。

這場比賽徐海東和對手打得難分難解，在前三局徐海東以11比7和11比8外加11比9連下三城，早早的3比0領先，已經爆出了不小的冷門。因為26歲的吉村真晴是日本隊的主力，在一個多月前是剛剛和張本智和以及丹羽孝希率領日本隊打進了世界盃賽的四強，當時他們淘汰了德國隊，而且吉村真晴在單打當中將39歲的世界冠軍波爾斬落馬下，表現相當的神勇。

而目前世界排名第54的吉村真晴，在0比3落後的情況下，面對著世界排名160名開外的徐海東，他展現出來了強大的後勁兒，他以11比5和11比8外加11比6連勝三局，這樣一來雙方在前6局打成了3比3平，必須要通過最後的決勝局來決出勝負了。

決勝局開打之後，徐海東一上來連丟2分0比2落後，隨後他利用出色的回球扳回一分，但是吉村真晴立即下殺板，連得2分以4比1領先徐海東三分，形勢對於徐海東很是不利。關鍵時刻，吉村真晴回球不過網，徐海東追成2比4，隨後比分又來到了3比4和3比5，吉村真晴沒有處理好徐海東的正手大斜線，終於追成了4比5，並且徐海東終於迫使對手出現失誤將球打丟，雙方打成5比5平。

落後三球頑強的扳平，讓徐海東看到了贏球的希望，此後雙方爭奪的更為激烈了。徐海東回球對手沒有接到，6比5。隨後吉村真晴回球失誤7比5，徐海東接發球出現失誤7比6，徐海東趕緊調整了策略，讓對手反手不能夠輕易的回球，吉村真晴回球下網，8比7！隨後吉村真晴釘在了徐海東的反手上連下兩分，8比9！吉村真晴完成反超。但是關鍵的時候，吉村真晴沒有把握住一次直線的機會，把球打得稍微大了一些，雙方打成9比9。緊接著他又主動送分，徐海東10比9率先迎來了賽點，但對手頑強的連續下殺完終於迫使徐海東出現了失誤，雙方打成10比10平。最後兩人的爭奪進入到了白熱化的程度，吉村真晴回球失誤11比10，最後一個球，吉村真晴又出現了失誤，這樣徐海東就以12比10這樣一個比分，驚險的在決勝局裡面絕殺了日本隊的主力，將對手淘汰出局。
