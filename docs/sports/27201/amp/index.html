世俱杯決賽前紅軍收雙重喜訊：後防核心復出無礙，全隊喜提14天假期

天天要聞
2019-12-21

目前紅軍全隊正在卡達多哈備戰明天凌晨進行的世俱杯決賽。

因為生病缺席了世俱杯半決賽的范迪克昨天終於加入到球隊合練的陣容中，雖然紅軍官網表示荷蘭中衛能否在決賽中出場還要通過隊醫最後的賽前評估，但是紅軍的隨隊記者James-Pearce稱紅軍4號料將出戰決賽。球隊在今天也只安排了一趟訓練課。

Ian-Doyle也對范迪克及時復出參加決賽感到樂觀。

另外一名傷員維納爾杜姆的情況跟范迪克差不多，也是需要通過了賽前最後的傷病評估才能決定球員是否能夠出現在世俱杯決賽場上。

這場與弗拉門戈的比賽將是紅軍38年之後再度面對這支巴西傳統強隊，對上一次雙方的交手是在1981年的豐田杯上，當時紅軍遺憾落敗。這次不知道同樣是在亞洲場地進行的交鋒，紅軍球員能為前輩們「復仇」嗎？

在踢完與弗拉門戈的世俱杯決賽之後，紅軍將立刻啟程返回利物浦，備戰聖誕新年快車期的比賽。

進入新年之後紅軍還將有足總杯與埃弗頓默西賽德郡德比的賽事需要分心，然後在聯賽中球隊還要接連面對熱刺、曼聯這兩個強硬的對手，狼隊也是潛在能夠給紅軍帶來麻煩的球隊。

好在熬過了這幾場硬仗之後，紅軍會迎來一個調整放鬆的時期：本賽季英超引進了冬歇期，從2月份開始每支球隊都能得到2周的休整時間。

而在昨天紅軍也在官方網站上宣布2月份球隊賽程時間調整的消息，紅軍客場對陣諾維奇、主場迎戰西漢姆已經作客去到沃特福德的比賽時間都作出了相應的改動。

官網最新公布的賽程時間調整如下：

諾維奇vs利物浦：改到當地時間2月15號下午17點30分；

利物浦vs西漢姆：改到當地時間2月24號晚上20點；

沃特福德vs利物浦：改到當地時間2月29號下午17點30分。

也就是說在明年2月1號踢完主場對陣南安普頓的比賽之後，紅軍全隊將迎來一個14天的假期。對於連續高強度作戰的球員們來說，在這個時間得到一個充足休息的機會用於身體狀態的調整和作戰疲勞的放鬆絕對是個好消息。

今年年初在足總杯的賽事中出局之後，紅軍眾將士也曾經得到10天的休假時間，球隊也決定利用這段沒有比賽的空窗期去到迪拜進行海外拉練，結果…在這裡就不想再重複了…

隨後在2月中旬的時候紅軍又組織了一次馬貝拉集訓營，但是訓練營結束之後回到聯賽當中的同樣效果不好。

所以希望這次冬歇期，球隊就不要搞什麼海外集訓營了。畢竟踢完與金絲雀諾維奇的聯賽第二回合較量之後，球隊就要面對客場對陣馬德里競技的歐冠淘汰賽客場比賽，以防守強悍著稱的馬競對上打法狂放激進的紅軍，可以預見這會是對身體消耗極大的對決，紅軍方面應該要做好充足的準備。

明年的2月1號也正好是本賽季冬季轉會窗口關閉的時間，如果有新援加盟的話，在這14天的賽事空窗期期間可以隨隊訓練磨合陣容培養默契無疑是最好的時機。紅軍球迷們希望看到誰的加盟呢？
