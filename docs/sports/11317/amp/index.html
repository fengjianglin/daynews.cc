有球迷說：踢日本，少輸當贏。踢韓國，少輸當贏。踢香港，平了就贏。

作為國足的球迷，沒有多大期望，就是最大的驚喜。

12月9日新聞發布會，國足選拔隊主帥李鐵表示：東亞杯是非常傳統的賽事，我和球員們都很重視這個比賽，都很榮幸能有機會代表國家踢球和工作，所以我們會全力以赴打好每一場比賽。

李鐵還表示：無論是面對韓國日本還是中國香港，都要全力以赴。這都是遭遇戰，對雙方都有困難的地方，對我們來說也是個挑戰，相信我們所有的球員已經做好這樣的準備。

李鐵在談話中，全程面帶微笑，並且霸氣十足！
