國羽第1冠！2大王牌13次德比戰 雅思狂勝11次，黃鴨衛冕失敗

2019-12-15 69條評論 2,648次閱讀 1706人點贊

北京時間12月15日，2019賽季世界羽聯年終總決賽進入最後的冠軍爭奪戰，國羽在女單、女雙和混雙3個項目中打進決賽。混雙項目提前包攬冠亞軍：國羽2大王牌組合鄭思維/黃雅瓊、王懿律/黃東萍成功會師，僅35分鐘的交手，雅思組合2-0橫掃黃鴨組合，奪得冠軍，國羽收穫第1金！這也是雅思組合第11次擊敗黃鴨組合，雙方交手13次，黃鴨組合僅勝2次。

在2018賽季年終總決賽中，同樣是雅思組合、黃鴨組合會師決賽，最終黃鴨組合在7連敗雅思之後，首次取勝奪得冠軍；在2019賽季中，國羽兩對王牌組合依舊制霸羽壇，攬括大部分的大賽冠軍。不過他們的狀態並不是一直穩定在高水準，雅思組合此前的幾站比賽成績略有下滑，而且一直延續到本周的總決賽。不過實力、經驗以及運氣讓兩對組合最終又一次會師決賽。

2對組合的晉級之路：

  • 王懿律/黃東萍：小組賽3戰3勝，分別擊敗德差波/沙西麗、徐承宰/蔡侑玎、陳炳順/吳柳瑩；
  • 鄭思維/黃雅瓊：小組賽2勝1負，擊敗喬丹/梅拉蒂、費澤爾/維德佳佳；輸給渡邊勇大/東野有紗；
  • 半決賽中，王懿律/黃東萍2-0德差波/沙西麗，鄭思維/黃雅瓊2-1渡邊勇大/東野有紗

黃鴨組合與雅思組合此前已經交手12次，黃鴨組合僅勝2次，雅思組合勝10次佔優。不過雙方上一次交手實在福州賽決賽，黃鴨組合2-1取勝奪冠；在本周的比賽中，黃鴨組合的狀態一直很好，而雅思組合卻起起伏伏並不理想，不少球迷都認為黃鴨組合能成功衛冕冠軍，然而在今天的決賽中，雅思組合的狀態更好，2-0橫掃取勝，斬獲冠軍！

首局，黃鴨組合開局4-0領先，雅思組合迅速追至6平，黃鴨組合再得2分之後，出現連續失誤，雅思組合打出一波8-0的反擊潮，將比分反超至14-8；佔據上風的雅思組合沒有輕易失誤，黃鴨組合的衝擊沒能帶來效果，雅思組合保持優勢，以21-17拿下首局。

次局，開局短暫纏鬥後，雅思組合就取得領先，黃鴨組合失誤不斷，雅思組合以11-4領先7分進入局歇；下半局黃鴨組合有所反擊，但雅思組合進入狀態後完全將其壓制，並不斷的擴大分差至19-8；黃鴨組合反擊追分但未能改變結局；最終，雅思組合以21-14拿下第二局，奪得冠軍！
