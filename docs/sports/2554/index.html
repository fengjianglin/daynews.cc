北京時間12月7日晚20點，2019-20賽季西班牙足球甲級聯賽的一場焦點戰將在伯納烏球場進行，齊達內執教的皇家馬德里對陣馬欽執教的西班牙人，賽前，西媒《馬卡報》對兩隊的首發進行了預測，按照西媒的說法，武磊將和卡萊里搭檔鋒線，不過兩隊的身價差距過大，西班牙人全隊的身價也就是皇馬的一個零頭，還不如阿扎爾一人高。

雖然西班牙人和皇馬的比賽還沒有開打，但是兩支球隊的實力差距明顯，賽前，輿論一致看好皇馬贏球，畢竟在積分榜上，兩隊一個是第2，一個是倒數第2，也算得上是首尾大戰，馬欽的球隊凶多吉少。

按照德媒《轉會市場》提供的數據，皇馬全隊的身價高達11.9億歐元，其中，僅僅阿扎爾一人的身價就有1.5億歐元，相比之下，西班牙人的身價只有1.49億歐元，從數字來看，西班牙人完全無法和皇馬相匹敵，兩支球隊的實力差距過大。要知道，武磊的身價是1000萬歐元，一個阿扎爾的身價等於15個武磊。

《馬卡報》在賽前給出了兩支球隊的首發陣容預測，皇馬方面會排出一個4312的陣型，門將位置是庫爾圖瓦，四名後衛分別是馬塞洛、拉莫斯、瓦拉內和卡瓦哈爾，中場的三人是莫德里奇、卡塞米羅和巴爾韋德，前鋒線上是本澤馬和貝爾，出現在前腰位置的是伊斯科。

再來看一下西班牙人隊的首發，按照西媒的預測，西班牙人會排出一個352的陣型，迭戈-洛佩斯是首發門將，這位綽號「章魚哥」的門神將反戈舊主銀河戰艦。在後防線上，西班牙人的三名球員分別是卡萊羅、貝爾納多和大衛-洛佩茲，拖後中場是馬克-羅卡，四名中場是佩德羅薩、格拉內羅和皮帕，前鋒線上是武磊和卡萊里兩人。

這場比賽對於國足球員武磊來說有著特別的意義，因為這是他來到西甲聯賽後，第一次在西甲賽場面對皇家馬德里。上個賽季，武磊曾經代表球隊對陣巴薩，當時他和梅西的對決也被熱炒，在賽場上，武磊曾經遭到皮克的嚴密看防。這一次，武磊將和皇馬隊長拉莫斯以及瓦拉內過招，相信水爺和瓦拉內的防守會給武磊帶來巨大麻煩。值得一提的是，拉莫斯和瓦拉內可都是拿過世界盃冠軍的中衛。

雖然西班牙人的贏面不大，但是這場比賽，馬欽的球隊還會力爭拿分，畢竟再輸球的話，球隊可能成為西甲副班長，這是球隊上下不希望面對的一幕！
