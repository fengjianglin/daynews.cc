倫納德：在洛杉磯人氣不如詹姆斯，但PG我們是洛杉磯最強二人組合

天天要聞
2019-12-14

北京時間12月14日上午,NBA常規賽繼續進行,兩場焦點戰值得我們關注,湖人對熱火,詹姆斯也重回老東家邁阿密,最終兩隊為大家奉獻了一場精彩的比賽,詹姆斯28分9板12助攻,濃眉哥33分10板1助攻,最終湖人隊113:110熱火,終結了熱火本賽季主場不敗的金身,湖人也拿到了本賽季的第23勝,目前23勝3敗的戰績排名聯盟第一!

詹姆濃眉

另一場焦點戰當然是洛杉磯快船隊對森林狼隊,而這場比賽比湖人隊開賽時間晚一點,最終憑藉著倫納德保羅喬治兩位當家球星的優異發揮,最終快船隊124:117戰勝了森林狼,全場比賽小卡喬治合力得了88分,喬治31投17中手感火熱,倫納德24投10中罰球19投全中砍據了解，這還是快船隊史第一次雙40+，也是NBA歷史第21對這樣的數據，值得一提的是上一對是保羅-喬治與威斯布魯克下42分,真是憑藉兩人之力就快要戰勝了對手!

小卡 喬治數據

賽後,倫納德面對記者的採訪,也顯得很興奮,小卡表示,今晚我們手感都非常出色,我的隊友(PG)發揮得特別好，他是天生的得分手!也是贏球的關鍵,面對最後一節森林狼的有力衝擊,他(喬治)做出了回應,守住了勝利!而在被一女記者問道覺得兩人組在洛杉磯的地位，小卡回復到我們在洛杉磯人氣目前還不如勒布朗和戴維斯,但是我們(保羅喬治)將會是洛杉磯最強二人組的,說完就扭頭走回更衣室!

喬治 倫納德

如今洛杉磯雙雄狀態火熱，今天洛杉磯湖人,洛杉磯快船都贏得了比賽,當家球星都表現很好,關於倫納德喬治還是詹姆斯濃眉誰更強也許在今後會有更多人議論,會有更多球迷有不同的看法，你們覺得哪個更厲害呢？
