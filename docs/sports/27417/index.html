轉會費=4個武磊！亞洲巨星有望成第二個孫興慜，賽季效率超C羅

2019-12-20 1,550條評論 3,215次閱讀 2007人點贊

當孫興慜在英超聯賽和歐冠聯賽踢得風生水起的時候，亞洲又一位巨星來到了英超聯賽，那就是日本球星南野拓實，他以725萬英鎊（約合850萬歐元）的身價加盟了利物浦，而本賽季這位日本球星的效率非常高，在出場22次的情況下，有著9球11助攻，效率比C羅都要高，這樣的球員能加盟利物浦，靠的就是硬實力！

南野拓實曾經在大阪櫻花踢球，在日本J聯賽得到磨練後，他加盟了薩爾茨堡紅牛，來到奧地利聯賽後，南野拓實一步步的前進，慢慢成為了球隊的絕對核心。在奧地利超級聯賽，南野拓實一共打入了42球，助攻了20次，製造了62球，這個數據是同期聯賽最高，這說明，他已經是奧地利聯賽的MVP。

本賽季，南野拓實的表現也是可圈可點，這名亞洲球星在各項賽事出場22場，打入9球，同時奉獻11次助攻，合計製造了20球，按照《露天看台》提供的數據，平均每80分鐘，他就可以製造1個進球。

要知道，本賽季的C羅在各項賽事出場了20次，打入了12球，助攻了1次，就效率來看，C羅不如南野拓實。梅西在本賽季各項賽事出場了17場，打入9球，助攻14次，效率要強於南野拓實。不過這也不算意外，畢竟在歐洲五大聯賽里，效率可以和梅西相提並論的，可以說是鳳毛麟角。

南野拓實的轉會費是850萬歐元，而武磊加盟西班牙人的轉會費是200萬歐元，也就是說，南野拓實的轉會費是武磊的4倍還多。事實上，利物浦觸發了違約金條款得到了南野拓實，而如果不是這樣的方式完成交易，球員的轉會費可能更高。

在加盟利物浦後，南野拓實會穿上18號戰袍，他將和孫興慜上演亞洲德比，而從發展路徑來看，南野拓實和孫興慜非常相似，都是加盟歐洲的中游球隊，然後再加盟英超的頂級豪門。或許在未來，他會成為亞洲的第二個孫興慜。

值得一提的是，2014年的亞青賽，韋世豪梅開二度，幫助球隊2-1擊敗擁有南野拓實的日本隊，當時後者也有1球入賬，但是卻是韋世豪的手下敗將。5年後，韋世豪在廣州恆大踢球，南野拓實加盟了利物浦，或許，韋世豪佔優的只有2000萬的年薪！
