直言不諱！穆里尼奧坦言5000萬帝星無緣首發，與拜仁一戰踢得太差

2019-12-14 2,687條評論 4,631次閱讀 692人點贊

12月15日晚，英超第17輪將迎來一場比賽，熱刺做客挑戰狼隊。狼隊目前在聯賽的排名比熱刺還要高，而且，狼隊在英超已經11輪不敗。這樣一個對手對於熱刺來說並不好對付。上賽季雙方的兩回合交鋒中，狼隊在主場2-3落敗，但到了客場卻3-1擊敗熱刺。因此，這會是一場五五開的比賽。

在賽前的新聞發布會上，穆里尼奧回應了關於洛賽爾索的相關問題。他說：「我們的陣容並不龐大，因此，洛賽爾索遲早會獲得機會。但是，我們有些球員出場順位會更靠前，只是因為他們表現更好。如果你問我周日的比賽洛賽爾索是否會首發，答案就是不能！」穆里尼奧真是直言不諱。的確，孫興慜和阿里的表現證明了他們配得上踢首發，而洛賽爾索跟他們比起來，差太多了。

在周四熱刺與拜仁的歐冠小組賽中，穆里尼奧就讓洛賽爾索首發出場。但是，洛賽爾索的表現太糟糕。他在進攻端毫無作為，沒有送出關鍵傳球和成功直塞球。在防守層面也弱爆了，硬生生被過了7次。第65分鐘的時候，穆里尼奧忍無可忍，把他給換下場了。

顯然，如果洛賽爾索不能夠在無數不多的出場機會中迅速拿出亮眼的表現，他只能淪為邊緣角色，甚至被徹底棄用。當記者詢問洛賽爾索什麼時候才能激發出自己的潛力的時候。穆里尼奧表示他也不知道，因為有些球員能夠快速適應一支新球隊，而一些球員會花很長時間適應。

德轉身價高達5000萬歐元的洛賽爾索今年夏天從皇家貝蒂斯租借加盟熱刺的時候，被寄予厚望。他上賽季在皇家貝蒂斯出場32次，打進9球，送出4個助攻，表現還是很搶眼的。但是，在熱刺他依然處於比較迷茫的狀態。洛賽爾索在聯賽出場8次，只有1次首發，沒有進球和助攻數據。在歐冠賽場，他出場3次，2次首發，打進1球。只有23歲的洛賽爾索能夠走出困境嗎？拭目以待。
