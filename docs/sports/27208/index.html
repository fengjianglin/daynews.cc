淡化恩怨，蘭帕德：切爾西對熱刺不是我和穆帥之間的比賽

2019-12-21 2,892條評論 4,888次閱讀 2706人點贊

切爾西主教練弗蘭克·蘭帕德淡化了與他的前主教練、現托特納姆熱刺主教練若澤·穆里尼奧之間的個人競爭。

穆里尼奧在一個月前接受了熱刺的工作，他以五戰四勝的成績縮小了與老東家的差距。這名托特納姆熱刺的主帥現在在藍軍球迷中是一個不受歡迎的人物，但是蘭帕德不希望周日的比賽成為焦點。

「問題是這不是我和若澤的比賽。這是一場對熱刺的比賽。我想打敗他。我尊重他，也知道他是一個多麼優秀的教練。他想要擊敗我，因為他曾經執教過切爾西，他曾經執教過我，現在他執教著托特納姆並且想要擊敗我們。」蘭帕德在科巴姆訓練中心告訴記者。

「我將永遠非常尊重我的所有教練。如果我給主帥發簡訊或打電話，我總是叫他們「老闆」或「老大」。即使是一去不復返。當你面對這樣一個有影響力的人，我知道那是不一樣的。」

「與他對抗是一種榮譽，但我想擊敗他。現在叫他老闆會很奇怪。現在我是前球員了，在這邊，我不會叫他老闆。但這並沒有減少我對他的尊重。」

「如果你去找最兇猛的對手，球迷會給出答案。這並不意味著他們忘記了歷史或者不尊重歷史。穆里尼奧是一個想要工作的教練。他抓住了這個機會。穆里尼奧也不會太在意這些。」

「看看他帶著曼聯回來的次數，確實看起來（他不在乎）。我們知道這是一個巨大的競爭。也許，在我這個時代，它（切爾西對熱刺）是最重大的倫敦德比。無論是在球場上還是在球場上，我都有這樣的感覺。所以可能是這樣。」

「任何競爭，任何同城德比，都會略微增加賭注。我記得那種渴望勝利、永不言敗的感覺。我們都是這麼想的。」

蘭帕德也認為托特納姆前鋒哈里·凱恩的職業道德和他在作為球員期間的相似，他希望塔米·亞伯拉罕能以他的英格蘭隊友為榜樣。

「在足球界，沒有比哈里·凱恩更值得他（亞伯拉罕）尊敬的人了。」蘭帕德說：「哈里·凱恩是全能9號。如果不是最好的，那麼接近世界上最好的。」

「他應該尊重這一點。我知道他是這樣做的，而且他正在努力學習哈里，必須努力擁有世界上最好的職業道德。他會有所不同，但他可以遵循這種模式走得非常、非常遠。」

凱恩的職業道德和渴望被認為是他成為世界上最好的前鋒之一的關鍵因素。蘭帕德承認他是英格蘭隊長的超級粉絲，他在熱刺的球星身上看到了自己的影子。

「從外面看，他確實讓我想起了自己。他已經最大化了他所有的天賦，或者說正在最大化他的天賦。」蘭帕德補充道：「他是個有進取心的男孩，有家庭觀念，想要工作，想要在事業上有所成就。我非常尊重他。周日我們將是對手，但我們非常尊重他。」
