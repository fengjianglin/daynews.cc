周一，在索爾斯克亞到來之前，曼聯的工作人員不得不使用電動工具強行打開卡靈頓凍結的大門。

今天，受斯堪的納維亞半島南下的冷空氣影響，曼徹斯特的氣溫一夜之間驟降到零下5℃。索爾斯克亞開著路虎來到曼聯的卡林頓訓練基地，結果發現門被凍死了打不開，真可謂「屋漏偏逢連夜雨 」。

工人們不得不使用電動工具來為這位深陷輿論焦點的主帥開門，最終，索帥還是順利的進入了訓練基地。索爾斯克亞最近的日子可謂真的不好過，周日晚上，憑藉著中後衛明斯的進球，維拉在老特拉福德以2-2的比分頑強地逼平了紅魔，目前僅僅積18分，排在聯賽的第9位。

據英國《太陽報》報道稱，現在曼聯的隊內氣氛就像最近的空氣一樣，非常僵硬，甚至可以說是陰冷，球員之間的交流非常少。周日晚上，他們在卡林頓基地完成了一趟訓練課，不過索帥還沒等到訓練結束，就獨自加著路虎離開了訓練基地。

周一，當工人為索爾斯克亞打開訓練基地的大門時，他一臉疲憊的表情進入了訓練基地，這正和曼聯最近的狀態一模一樣。然而，曼聯似乎沒有太多時間來準備與前主帥穆里尼奧的大戰。
