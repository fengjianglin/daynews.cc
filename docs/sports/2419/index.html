距離梅西捧得第六座金球獎已經過去好幾天了，然而關於金球獎的討論卻還在如火如荼的進行。今天下午著名的足球數據統計網站Squawka Football發表推特，再次引起大家對金球獎的討論熱潮。

下午5時，Squawka Football發文稱「告訴孩子，2019年金球獎的第三名是薩迪奧·馬內」。眾所周知，在12月3日進行的金球獎的頒獎典禮中，梅西686分奪得魁首，范戴克679緊隨其後，第三名是476分的克羅斯蒂亞諾·羅納爾多，馬內以347分的得票排名第四。

一個多小時後，該網站再次發出薩迪奧·馬內和C羅去年的數據對比，對C羅的嘲諷毫不掩飾。事實上本賽季的馬內發揮極其出色，本年度為利物浦出站45場比賽，打入28球並助攻4次，效率還要高於薩拉赫。從Squawka貼出的數據對比來看，2019年相比於C羅，馬內表現的更加出色。

此前在FIFA年度最佳評選時，梅西將首票投給了馬內，當時還一度有球迷認為梅西是怕投給直接競爭對手范戴克會威脅到自己的得票。事實上，梅西對馬內的欣賞由來已久，在金球獎頒獎典禮後接受記者採訪時，梅西表示「很享受看馬內踢球，他在利物浦渡過了非常棒的一年。」
