2019中超聯賽已經收官，金元足球堆積起來的8冠王隊中國足球的影響好與壞暫時不與評說，大牌外援的引進確實擠壓了國產球員的生存空間，尤其是年輕球員，儘管有著U23政策的保護，但是國青、國少在國際賽場上表現越來約不盡人意，儘管有歸化球員的加入，國家隊在世預賽的表現也沒有達到預期，雖然40強賽我們仍然有突圍的可能，但是以國足在40強賽的表現，即便是進入12強賽，誰又敢2022世界盃抱有幻想？這屆不行，那下一屆呢？2026年我們還有希望嗎？目前的U23到2026應該是當打之年，我們通過本賽季聯賽中比較優秀的U23隊員的表現來看一下，誰能撐起國足的未來？

廣州恆大
