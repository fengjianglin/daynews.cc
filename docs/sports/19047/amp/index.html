槍手新帥已確定，阿森納高層放手一搏，安帥無緣，前隊長即將上任

天天要聞
2019-12-17

英超傳統豪門阿森納最近的煩心事不少，特別在上一周0-3不敵曼城後，球隊17輪取得5勝7平5負的成績，積22分排名英超第10的位置，目前距離英超第4的切爾西有7分的差距。

槍手想要衝進前4重返歐冠，在剩下的20輪聯賽中必須奮起直追，否者將有可能第4年無緣歐冠。這對於球迷還是俱樂部高層來說，都是令人痛心的事情。

埃梅里因為戰績太差下課，但臨時主帥永貝里上任後也沒能改變兵工廠的頹勢。因此，確定新主帥變成阿森納當前最需要完成的工作。近日終於出現了突破性的發展，槍手即將聘請阿爾特塔作為主教練。

根據消息指出，阿爾特塔接受了小克倫克的最終面試，阿森納將會為他提供一份年薪500萬英鎊的合同。行業人士分析，本周很有可能就會宣布阿爾特塔的正式上任。

消息引起不少兵工廠球迷的討論，不少球迷認為安切洛迪還是最適合阿森納的人選。如果俱樂部執意選擇阿爾特塔，那將是關乎阿森納命運的一場賭博。

如果用阿爾特塔和安切洛迪的履歷做比較，他們明顯不是同一水平的主帥。阿爾特塔是一個從來沒有正式帶過隊的助教，而安切洛迪則是贏得無數聯賽冠軍的超級名帥。

那麼，阿森納為什麼會選擇阿爾特塔，很大可能是因為這名前槍手隊長的足球理念以及價格相對便宜。作為瓜迪奧拉的助教，圈內人對他的評價都相當不錯，甚至連曼城高層都覺得他能成為瓜帥的接班人。

之所以說安切洛迪更適合如今的阿森納，是因為他可以有效地平息目前阿森納的亂局。不過，自從安帥離開AC米蘭後，接下來的5傢俱樂部的執教時間均未超過2年。如果安帥上任後，不到2年又再次推倒重建，這對阿森納的長遠發展非常不利。

實際上，無論阿爾特塔和安切洛迪都不是目前阿森納的最優解。以往，安帥之前的球隊都是絕對豪門，均有無限多的資源供給，而這正是阿森納所無法提供的。此外，新帥身份的阿爾特塔更沒有實際的成績作為參考，有無效果都是未知數。
