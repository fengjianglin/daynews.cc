77%的勝率！數字很驚心，中國乒乓球很危險

77%的勝率，日本女乒一姐伊藤美誠正在對中國乒乓球的霸主地位發起最猛烈衝擊

伊藤美誠是中國女乒最大的勁敵！在今年的多個大賽上，連續闖入決賽，也先後擊敗過多位中國女乒頂尖選手，並且伊藤美誠的成長也是有目共睹的，在剛剛結束的奧地利公開賽，乾淨利落的戰勝世界冠軍朱雨玲捧杯更是可以看出她的強大實力！根據統計，伊藤美誠今年面對中國女乒的勝率已經達到了77%，這是什麼概念呢？基本上，中國女乒的頂尖選手都曾經敗在她的手裡！

伊藤美誠的成長，必然會化作2020年中國女乒東京奧運會奪冠最大的阻力！屆時，伊藤美誠將會有主場優勢，進一步成長下，伊藤美誠東京奪冠可能極大！

這還只是外籍運動員對中國乒乓球的縮影！

中國乒乓球難再現霸主地位，對手的衝擊需要重新認識自己

伊藤美誠的衝擊只是一個開始，同為日本球員的男乒選手張本智和對中國男乒的勝率達到了44%，雖然低於50%，但是要知道中國男乒之前在與對手交戰的時候，都是毫無懸念的獲勝！44%的突破已經讓中國男乒非常有壓迫感了！

外籍對手尤其是日本乒乓球選手對於中國乒乓球的衝擊已經顯現出來！之前中國乒乓球奪冠都是連續不斷的，但是近年來，中國乒乓球卻顯現出乏力的情況！即使奪冠也是陷入苦戰！這與之前捨我其誰的中國乒乓球相差巨大！

當然，有對手是好事，這能夠增加這項運動的吸引力，但是中國乒乓球也要重新認識自己，不要一邊養狼，一邊被狼吃掉！正視問題，解決問題，才能保證明年的東京奧運會冠軍不會旁落！
