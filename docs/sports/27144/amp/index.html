江蘇排球60年！80歲袁偉民同框24歲徒孫張常寧 頒獎64歲愛徒孫晉芳

天天要聞
2019-12-22

12月21日消息，今天上午，為慶祝江蘇排球隊成立60周年，80歲的前國家體育總局局長、中國女排五連冠功勛教練袁偉民、64歲老女排隊長孫晉芳、24歲的中國女排世界冠軍張常寧和龔翔宇等老中青三代排球名將參加了南京舉行的慶祝活動。

袁偉民同框徒孫張常寧

​江蘇排球隊於1958年正式成立，距今正好60周年。建隊之初，條件極其艱苦。大家就在室外苦練基本功。

​1965年第二屆全運會分區賽在成都舉行，江蘇女排對陣山東隊，江蘇隊的姑娘們奮勇救球。

​年過八旬的江蘇排球隊元老隊員張國蓉回憶起了60年前江蘇排球隊建立初期的艱難歲月，「1958年，我們在備戰1959年的第一屆全運會，當時江蘇隊是『媽媽隊』，6個上場隊員有5個是媽媽。

​我當時生完女兒還不到3個月，就去打球。當時訓練場還是土地，泥巴地，晚上就睡在體育場的看台裡面，冬天冷死，夏天熱死，冬冷夏熱特別難熬，但是困難再大，我們也要打球，為了江蘇隊榮譽而戰。」

​而已經80歲的中國女排功勛教練、郎平恩師袁偉民是蘇州人，他來到現場，因為和張國蓉是同一代人，聊著天十分開心。

​袁老還和4位50年代江蘇女排隊員合影留念。

​袁偉民的老女排隊長孫晉芳、二傳張潔雲也都來了。

​新一代的江蘇女排現役隊長張常寧、接應龔翔宇也來了。

而孫玥、趙蕊蕊、惠若琪等江蘇名將則遺憾缺席了這次慶祝活動。

​50年代男排排代表。

​60年代代表。

​70年代代表。

​80年代代表。

​袁偉民親自為兩大弟子孫晉芳和張潔雲頒發榮譽證書。

​張常寧和龔翔宇也拿到了屬於自己的榮譽證書。

​老女排隊長孫晉芳談到女排精神時表示，「中國女排精神不是屬於某一個人，而是屬於這個集體，中國女排的魂就是集體主義，作為女排精神形成的親歷者，我們不但要堅持維護女排的這種精神，我們也希望未來的排球人一代代傳承下去。」

​在電影《中國女排》中，江蘇女排自由人、如今已經是南京師範大學一名教師的陳展，扮演了球員時代的孫晉芳。為了還原偶像，陳展把當年中國女排比賽的錄像看了一遍又一遍。

陳展說，「不管是誰代表中國隊站在那個賽場上，都會拼盡全力，這種精神直到現在也沒有改變。」

​正是由於女排精神代代傳承，江蘇走出了孫玥、趙蕊蕊、惠若琪、張常寧、龔翔宇等一大批國手。無論是2016年里約奧運會中國女排奪冠，還是今年中國女排再次站上世界盃最高領獎台，都能看到江蘇隊員。

如今，張常寧、龔翔宇正在積極備戰明年的東京奧運會。張常寧表示，「女排精神就是祖國至上、團結協作、頑強拼搏、永不言敗。我們會珍惜每一次訓練課，在備戰期間努力完善自己，希望能在賽場上發揮出自己的能量，努力拚搏每一分。」
