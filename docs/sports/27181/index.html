穆里尼奧又一神跡！反戈舊主主場勝率92%，英超演18億大戰

2019-12-21 2,345條評論 3,998次閱讀 224人點贊

英超聯賽第18輪將迎來一場焦點戰，穆里尼奧執教的熱刺在主場對壘切爾西，這場比賽是英超賽場上的爭四大戰，主角當然是穆里尼奧，狂人將帶隊面對老東家。一個偉大的數據是，過去13次帶領不同球隊在主場面對老東家，穆里尼奧豪取12勝1平的不敗戰績，這將是對切爾西的震懾，央視CCTV5體育頻道將直播這場英超焦點大戰。

穆里尼奧打的就是「老東家」

執教生涯，穆里尼奧多次帶隊面對老東家，雖然也有輸球場次，但是在主場面對舊主，穆里尼奧從未輸球，總體戰績是13場比賽12勝1平，超高的勝率（92%）預示著，熱刺在主場面對切爾西可能收穫一個好的結果。前不久，穆里尼奧帶隊做客老特拉福德球場面對舊主曼聯，雖然輸球，不過那是客場比賽，在主場面對舊主，穆里尼奧從未有過輸球經歷。

師徒大戰

在執教切爾西期間，穆里尼奧和蘭帕德結下了深厚的友誼，不過這一次在英超賽場上，穆里尼奧和神燈將成為對手，這也是兩人第一次以教練身份在球場上鬥法，相信穆里尼奧和蘭帕德都不想成為輸球的一方。

英超爭四大戰

這場比賽之前，切爾西和熱刺的積分差距僅僅是3分。藍軍在17輪拿到了29分，排名積分榜的第4位，而熱刺在17輪拿到了26分，排名聯賽第5位。從兩支球隊的積分情況來看，如果穆里尼奧帶隊贏球的話，那麼兩支球隊的積分將持平。對於熱刺來說，他們一度排名第14位，但是在穆里尼奧的帶領下，這支球隊有望在短期內逆襲到第4名。事實上，自從執教熱刺以來，穆里尼奧一共帶隊踢了5場聯賽，他取得了4勝1負的戰績，唯一的輸球場次是輸給了曼聯。

身價對比

兩支球隊的身價對比情況來看，熱刺全隊的身價是9.59億歐元，切爾西全隊的身價是8.42億歐元，熱刺全隊最高的是凱恩，身價1.5億歐元，切爾西全隊身價最高的是坎特，身價達到1億歐元，兩支球隊加起來的身價是18億歐元，約合139.7億人民幣。

這場英超比賽，既是倫敦德比，又是師徒大戰，還是英超爭四大戰，毫無疑問，這將是本輪英超的焦點大戰，央視CCTV5體育頻道會對這場比賽進行現場直播，球迷們可以選擇觀看這場重磅對決。
