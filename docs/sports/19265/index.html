揮拳怒吼+飛撲慶祝！穆帥神了，89分鐘換上熱刺棄將，90分鐘絕殺

2019-12-16 792條評論 2,654次閱讀 809人點贊

穆里尼奧還是最特別的一個。熱刺客場2比1有驚無險取勝狼隊後，穆里尼奧徹底瘋狂，他揮拳怒吼，飛撲向自己的戰友進行瘋狂慶祝。客場險勝對手，維爾通亨立下頭功，但不得不提的是，穆里尼奧終場前的換人才是決定性的。他在第89分鐘才換上「棄將」埃里克森，沒想到後者上場後1分鐘就幫助隊友完成了絕殺。真是神奇的換人。

英超第17輪，穆里尼奧率熱刺客場挑戰狼隊。主場作戰的狼隊可謂名符其實的強隊殺手，本賽季他們多次讓傳統豪強陰溝翻船，主場1比1平曼聯、客場1比1平阿森納、客場2比0挑翻曼城，都是他們的得意之作。

所以，客場面對強勢的對手，穆帥給球隊的指示就是防守反擊。儘管小盧卡斯在第7分鐘就為球隊取得領先，但熱刺卻被對手在下半場扳平，此後的局面更是完全被主隊控制。一時間，熱刺門前風聲鶴唳。眼看比賽將以一場平局收場，此時穆里尼奧卻做出一次意外的換人。他在第89分鐘換上埃里克森，正是這次換人改變了最後的結果。

第90分鐘，熱刺獲得右路角球機會，埃里克森將球送到對方禁區，維爾通亨搶在對方解圍前甩頭攻門，球在經過反彈後直接鑽進了對方球門。正是憑藉這粒進球，熱刺在最後時刻完成了對對手的絕殺。

對於熱刺來說，這樣一場勝利來得太不容易。從本場的賽後數據來看，主場作戰的狼隊幾乎各項數據都領先於客隊，但是他們卻栽在了最後1分鐘。而對於穆里尼奧來說，他在終場前換上心思早已不在球隊並遭到自己棄用的埃里克森，本身就是一次賭博。讓他意外的是，他這次換人賭對了。

所以當比賽結束時，穆里尼奧徹底瘋狂了，他立即揮舞雙拳，一邊怒吼一邊飛撲向身邊的戰友，一起瘋狂慶祝。隨後，穆里尼奧更是冒著寒冬里的大雨走向場內，開始一個個地擁抱弟子，和弟子們一起狂歡，享受勝利的喜悅。在這個艱難的夜晚，穆里尼奧又一次笑到了最後。
