獨造2球！吳曦閃耀全明星，進球後第一時間與葡萄牙巨星擁抱慶祝

2019-12-23 1,460條評論 2,996次閱讀 1815人點贊

對於吳曦來說，這絕對是一場讓他永生難忘的比賽。在馬德里進行的一場全明星慈善賽，作為中國球員唯一代表參賽的吳曦不但打滿90分鐘，而且表現搶眼，個人貢獻1傳1射、獨造2球，幫助世界明星聯杯6比6戰平西班牙傳奇隊。而他的隊友中，不乏曾經叱吒風雲的名將，如葡萄牙傳奇巨星費戈。

12月22日，由西班牙組織的一場全明星慈善賽在馬德里展開，對陣雙方分別是由費戈領銜的世界明星聯隊和由耶羅領銜的西班牙傳奇明星隊。上半場比賽，阿爾維斯為世界明星聯杯率先打破僵局，不過路易斯-加西亞很快就將場上比分扳平。隨後，皮耶羅、莫倫特斯、努諾-戈麥斯先後進球，上半場比賽結束，世界明星聯隊3比2領先。

下半場比賽，列拉率先進球，為西班牙隊再次將場上比分扳平，3比3。隨後的一幕相信會讓中國球迷熱身沸騰。世界明星聯杯獲得角球機會，身披22號球衣的吳曦禁區內頭槌搶點，將皮球頂入對方大門。作為吳曦這場比賽的隊友，身披10號球衣的費戈第一時間跑過來和他一起擁抱慶祝，雙方隨後也相互交談了一番。之後的比賽，費戈、莫倫特斯、德拉雷德和盧克先後進球，兩隊戰成6比6，最終以一片和諧收場。

對於中國球迷來說，能夠看到吳曦站在這樣全明星賽場上實在太過驚喜，特別是與他同場競技的不乏曾經的世界足壇巨星，如小羅、費戈、耶羅、埃托奧、卡洛斯、皮爾洛、莫倫特斯等都是我們耳熟能詳的傳奇球星。可以說，能夠和他們站在一塊，吳曦就已經證明了自己。當然，他也要感謝自己在蘇寧的主帥奧拉羅尤，因為正是後者的提攜和大力推薦，才讓自己獲得一次與世界足壇巨星交手的機會，這無疑將他讓受益匪淺。
