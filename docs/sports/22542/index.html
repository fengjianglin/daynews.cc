戰勝鄧亞萍大呼呦西的何智麗，被日本老公拋棄後，願意再嫁中國男

2019-12-19 2,365條評論 4,991次閱讀 3319人點贊

中國體育有六大強項：跳水、體操、射擊、乒乓球、羽毛球和舉重，而這六項中，最強的還是跳水和乒乓球，中國跳水隊號稱夢之隊，2008年北京奧運會，中國跳水隊奪得了八枚金牌中的七枚；而乒乓球也成為了中國的國球，無論是乒乓球世錦賽，還是奧運會，中國乒乓球隊也包攬了大部分金牌。

人有失足，馬有失蹄，即使再強也難免有失足的時候，但從1988年漢城奧運會正式有乒乓球項目後，中國女乒單打在八屆奧運會上全部奪得冠軍，無一次失手，這份榮耀也算是前無古人後無來者了。1988年漢城奧運會女乒單打冠軍是陳靜，但實際上當時還有個更熱門的奪冠人選，她就是何智麗。

1987年在印度新德里舉行的第39屆世界乒乓球錦標賽上，中國女將管建華、戴麗麗、何智麗三人打入了半決賽，韓國女將梁英子也打入了半決賽，半決賽管建華對陣何智麗，戴麗麗對陣梁英子，由於管建華對陣梁英子歷史戰績更好，組織上便安排何智麗輸球，讓管建華進決賽，增加奪金幾率。而何智麗上場後卻完全忘記了組織的安排，啪啪啪的以3比0戰勝管建華並最終奪得冠軍。雖然奪得冠軍，但組織對此十分震怒並決定處罰，1988年漢城奧運會參賽名單出爐後，國外媒體一片嘩然，他們很震驚前一年的世錦賽冠軍何智麗居然沒資格參加奧運會，很明顯的是沒有入圍大名單，並不是因為競技因素，而是受到了處罰，受到冷落的何智麗在1989年黯然退役，而那一年她才25歲。

隨後，何智麗遠赴東洋日本，隨後嫁給了日本男人小山英之，並冠夫姓改名為小山智麗，1992年小山智麗神秘復出，併入選了日本女乒國家隊，1994年10月在日本廣島舉辦的第12屆亞運會上，小山智麗狀態神勇，接連擊敗前奧運會冠軍中華台北的陳靜、中國的喬紅和世界冠軍中國的鄧亞萍，奪得乒乓球女子單打冠軍。

在與鄧亞萍的鏖戰中，每得一分，小山智麗便大呼呦西，這讓電視機前的億萬觀眾感覺十分刺耳，畢竟中日兩國歷史的交惡，對於日本很多中國人還是很排斥的，現在看著自己曾經的同胞幫助日本隊打中國隊，還不停的用日語來叫喊，實在是接受不了。賽後採訪中，小山智麗說這是她最重要的一枚金牌，她很高興能幫助日本隊爭奪榮譽，也就是說在她心中，這枚金牌價值遠超1987年的世錦賽金牌。

此事過後，國內輿論一片嘩然，不少媒體痛罵小山智麗為汪精衛、吳三桂，也因為如此，其後多年，小山智麗也不敢回上海老家一步。而在家庭方面，小山智麗也遭遇到了挫折，其丈夫小山英之在結婚後，慢慢顯示出日本男人的本性，酗酒、好色，甚至帶過女人回家，2000年悉尼奧運會前夕，小山智麗與小山英之正式離婚，不過離婚後她依舊保持了小山智麗這個名字，她給出的理由是改名太麻煩，隨後她又在大阪買房定居。

2005年5月，第48屆世錦賽單項賽在上海舉行，小山智麗時隔多年後首次回到中國，也因為時隔多年，中國民眾也沒多少記得之前的事情了，大家都很歡迎她，這十幾年來小山智麗已經多次回國探親，已經五十多歲的她還處於單身狀態，當媒體問她的婚姻情況時，小山英之也坦言現在的她很享受單身生活，也開始願意接受中國男人了。
