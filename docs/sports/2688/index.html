前利物浦球星：馬內過去一年表現很好，但相對梅西，他是平民階級

2019-12-04 0條評論 21次閱讀 0人點贊

摘要

在利物浦球星馬內獲得金球獎第四名後，非洲的許多球迷對此表示了憤慨和不滿，而曾經在利物浦效力的塞內加爾前鋒迪烏夫則持略為平衡的觀點。幫助利物浦贏得歐冠冠軍的馬內在金球獎的評選中落後於梅西、范迪克和C羅。塞內加爾的許多球迷很快就指責金球獎的評選有嚴重的種族偏見。

在利物浦球星馬內獲得金球獎第四名後，非洲的許多球迷對此表示了憤慨和不滿，而曾經在利物浦效力的塞內加爾前鋒迪烏夫則持略為平衡的觀點。

幫助利物浦贏得歐冠冠軍的馬內在金球獎的評選中落後於梅西、范迪克和C羅。塞內加爾的許多球迷很快就指責金球獎的評選有嚴重的種族偏見。

在利物浦三個賽季踢了55場比賽的迪烏夫告訴法新社：「我們不能談論種族主義，而只能談論偏袒，」

但迪烏夫也認為來自歐洲大陸的球員處於劣勢：「作為一名非洲足球運動員當然是一個障礙，因為主要的決策者和有影響力的媒體都不是來自非洲大陸。」

「但到目前為止，梅西的表現值得獲得金球獎。」

「馬內有一個特殊的一年，但相對於梅西，他是平民階層。」

投票小組由來自國際足聯里的每個國家的一名記者組成，每名記者可以評選前5名球員。梅西最終以686分險勝，僅比范迪克領先7分。C羅總共476分，馬內347分。

在46名非洲評選者中，只有12人把馬內放在首位。梅西和范迪克都獲得了歐洲大陸更多的第一名。

梅西以187分的成績獲得了非洲最高分，馬內170分名列第二。在歐洲記者投票中佔主導地位的范迪克在非洲收集了154分。

曾經為Canal+Africa在非洲大陸報道足球的阿布巴克里·巴指責投票小組中的非洲記者缺乏團結。

他對法新社說：「沒有種族歧視，但如果非洲的選票投給了非洲大陸的球員，情況就會不同。」

「金球獎不再考慮球員在球場上的表現了。你必須把它交給一個商業價值更好、技術更好的球員，而不僅僅是在球場上得表現。非洲選民不明白，有時有必要現實地投票。」

茅利塔尼亞記者拉薩娜·卡馬拉在塞內加爾《Sud quotiden》報撰文稱，他對主席台上沒有非洲人感到失望。

「我們拒絕相信種族主義，」他說，並指出尤西比奧、維阿、羅納爾多、羅納爾迪尼奧和里瓦爾多，這些前金球獎得主都是「有色人種的球員」。

「但當非洲人投票反對非洲人時，我們不會看到明天隧道的盡頭，」他寫道。
