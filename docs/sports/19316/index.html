90度崴腳！東契奇第1神跡終結，巴神血戰44分鐘拒絕大逆轉

2019-12-16 1,218條評論 2,462次閱讀 273人點贊

如今的獨行俠和熱火的比賽，也已經不是當年韋德諾維茨基的時代了，如今是東契奇和巴特勒的對決。北京時間2019年12月15日，熱火客場以122-118戰勝獨行俠，東契奇崴腳全場就打了2分鐘，拿下2分1籃板，巴特勒打了44分鐘拿下27分7助攻。

當年的小牛和熱火，從2006年結下樑子到2011年這幾年，一直都被拿來做為宿敵之一對比，如今另個隊史人物退役，而獨行俠現在是東契奇的時代。東契奇剛剛成為21歲之前拿到得分40+三雙最多的球員（兩次），而詹姆斯21歲前拿到過一次40分三雙，而巴特勒則剛剛輸掉了在主場對湖人的比賽，背靠背對牛魔王的體能來說不算是大事兒。

而這場對大紅的東契奇，巴特勒身先士卒上來就是連續拿分，投中三分，並且連續製造殺傷，熱火在開局打出了19-7的攻擊波。東契奇在開局才兩分鐘的時候就崴腳下場，這讓整個獨行俠蒙上了一層陰影，而熱火在這個期間竟然打出了瘋狂的攻擊波，24-7的開局讓獨行俠挖了個大坑。慢鏡頭回放，是契奇不慎踩在肯德里克-納恩的腳上，隨即東契奇一個人單腿蹦著回到了更衣室。

沒有了東契奇的獨行俠有多慌亂？這個賽季獨行俠的強悍就是一切由東契奇開啟，這個球隊除了波神之外甚至沒有什麼拿得出手的球星，結果東契奇下場10分鐘獨行俠讓熱火瘋狂的打出各種反擊，一下落後了20分。值得一提的是東契奇已經連續20場至少20+5+5的表現，刷新著歷史紀錄，但是這場比賽似乎要就此終結了。

果然在傷之後的東契奇回去之後做了檢查，X光檢查了陰性沒有結構性損傷也算是不幸中的萬幸，不過這場比賽東契奇就此結束不會回歸，紀錄就此終結。而熱火在這場比賽的三分投的太好了，第二節剛開始就已經命中了9個三分，大比分領先獨行俠。這場比賽也最能看出來沒有東契奇的獨行俠，到底能打成什麼樣子。本賽季東契奇的場均助攻聯盟第三，而獨行俠全隊助攻排名第13，全隊35%左右的助攻都是源於東契奇的助攻。東契奇在本賽季MVP的真正價值，在詹姆斯和濃眉，倫納德和喬治這種雙星縱橫的時代，東契奇的獨自帶隊的能力確實太過耀眼。

或許是東契奇的受傷給了獨行俠很大的刺激，下半場開始他們就打出了15-4的瘋狂的攻擊波，一舉將分差縮小，這期間波神的表現出色，隨著小哈達威的三分命中獨行俠20-6的攻擊追到了個位數。要說巴里亞真是打熱火太有心得，末節就靠他的瘋狂發揮一點點追分，追到了觸手可及的位置。憑藉巴里亞的瘋狂追分，全隊也受到感染開始不斷追分，最終反超了熱火。進入加時賽，依然是巴特勒倒地的投射幫助熱火先拿下2分，而比賽最後10秒巴特勒還保護了一個關鍵的籃板，製造犯規得手。雖然兩罰不中，但隨後兩罰全中帶走比賽，這場巴特勒算是居功至偉，最終熱火險勝獨行俠。
