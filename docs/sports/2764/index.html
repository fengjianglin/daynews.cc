北京時間12月4日，CBA常規賽繼續進行，大黑馬山西隊主場對陣廣廈，如果取得4連勝，將殺進前三名。遼寧客場挑戰浙江，這場比賽球隊的目標是延續連勝的步伐，郭艾倫和趙繼偉的發揮依然是看點。新疆客場對陣江蘇，周琦帶隊衝擊4連勝，贏球的概率更大。

在3日的比賽中，北京大勝廣州，山東擊敗同曦。4日的對決，央視體育頻道將直播第三名和第四名的比賽，廣廈客場挑戰黑馬山西隊。山西本賽季的進步很快，王非打造的全新山西隊有著很強的戰鬥力，不只是外援富蘭克林發揮出色，本土球員原帥、任駿威等人同樣有出色的表現。如今，山西3連勝，在少賽一場的情況下排名第4位。

廣廈已經完成了外援的更換，只是法里埃德並沒有達到預期的效果，上一場他們輸給了深圳隊，第3的位置受到威脅。本場交鋒，山西在主場還是有一定的優勢，月最佳富蘭克林的發揮依然是關鍵，本土球員對比廣廈或許有一些差距，但他們在主場還是可以展現出自己的實力，有望在主場衝擊4連勝，衝擊前三的位置。本場比賽，王非和李春江的鬥法同樣是比賽的一個看點。

遼寧隊客場挑戰浙江，兩隊勢頭都不錯，遼寧是兩連勝，浙江是3連勝。對於遼寧來說，近期球隊整體的進攻明顯提升，尤其是趙繼偉的外線找到了手感，這可以給遼寧隊帶來更大的幫助。浙江的贏球關鍵還是鄧蒙、蘭茲博格和吳前，這三人是球隊的勝負手。鄧蒙的得分能力出色，郭士強如何限制他，這是一個比賽的看點。吳前是國手，他的得分能力出色，郭艾倫和趙繼偉也要更多地對他形成一些限制和干擾。

新疆客場挑戰江蘇，周琦帶隊衝擊4連勝。從整體實力來看，新疆還是佔有一定的優勢，江蘇更換外援後，吉布森的得分能力不俗，但他與全隊的融合還是需要一些時間。新疆無論是外援還是本土球員，得分點眾多，這也是他們成績出色的原因所在。周琦在內線會有更為明顯的優勢，他的效率同樣很高。

其餘比賽，上海對陣青島，這是上海更換外援後的第一場比賽，新外援麥卡勒姆取代了納納利，他和莫泰的組合值得關注。青島的外援亞當斯是得分高手，但近期表現同樣出現了起伏。北控客戰天津，這是馬布裡帶隊必須要爭取拿下的勝利。吉林對陣福建，面對弱隊，吉林是否可以捍衛主場成為看點。王哲林輪休一場後應該會復出，他是球隊的攻防關鍵，這場客場比賽，福建要想贏球，王哲林還是要帶動全隊的進攻。
