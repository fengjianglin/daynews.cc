生死戰廣東恆大女排1-3不敵遼寧 進四強命運交給天津女排

2019-12-17 2,265條評論 4,097次閱讀 2892人點贊

北京時間12月17日，2019-2020賽季中國女排聯賽進行了複賽第四輪比賽，上海隊主場以0-3不敵北京隊，仍奪得小組第一，北京隊獲得小組第二，兩隊攜手晉級四強。山東隊以3-1擊敗八一隊，獲得小組第三名，八一隊獲得小組第四名。在另外一場關鍵到進入四強的關鍵戰中，遼寧隊主場以3-1戰勝廣東恆大隊，這樣廣東隊能否進入四強，要看另一場天津對江蘇的結果而定，目前天津與江蘇的比賽還在進行中。

遼寧3-1勝廣東

遼寧隊的首發陣容為：主攻紐康比、段放，副攻胡銘媛、由佳琪，二傳丁霞，接應馬馬多娃，自由人宮美子。廣東隊的首發陣容為：二傳郁佳睿，接應栗垚，主攻科舍列娃、拉巴捷耶娃，副攻鄭益昕、曹婷婷，自由人林莉。

首局遼寧隊打的順手，以18-12領先。許曉婷換下郁佳睿，但遼寧隊連續反擊得分，以25-17先勝一局。第二局廣東隊開局以10-2領先，遼寧隊利用馬馬多娃的進攻連續得分，追成14-17。廣東隊及時穩住陣腳，拉巴捷耶娃連續進攻得手，鄭益昕連續發球得分，以22-14領先，並以25-17扳回一局。

第三局雙方比分交替上升，戰成7平、10平。此後遼寧隊進攻連續得手，以15-12領先。遼寧隊快攻和強攻連續得手，20-14。栗垚和科舍分別進攻得手，廣東追成18-23。段放進攻得分，遼寧25-19再勝一局。

第四局遼寧隊以10-6領先，馬馬多娃進攻不斷得分，紐康比發球得分，遼寧隊14-7領先。廣東隊失誤不斷，遼寧隊以19-11領先，並以25-14取勝。

遼寧隊段放得到全場最高的20分，馬馬多娃19分。

上海0-3負北京

上海隊首發派上主攻拉爾森、楊婕，接應利普曼，副攻張鈺倩、王馨瑤，二傳卞雨倩，自由人王唯漪。北京隊首發派上主攻巴奇、金燁，副攻張宇、迪克森，二傳蔡雅倩，接應曾春蕾，自由人王琳。

首局雙方爭奪激烈，戰成12平。此後北京隊的巴奇、金燁連續進攻得手，以17-13領先。上海隊雖奮力反擊，但北京隊攻防穩定，以23-21領先。巴奇連續進攻得手，24-22。迪克森進攻打中，北京隊以25-22先勝一局。

第二局北京隊延續強勁勢頭，以12-9領先。上海隊失誤不斷，北京隊連續得分，以17-11領先，並以25-19再勝一局。

第三局北京隊完全佔據了場上優勢，以13-6領先。此後北京隊比分一路領先，以25-10輕鬆取勝。北京隊外援巴奇得到全場最高的18分。

在另外一場比賽中，主場作戰的山東隊以3-1(25-22、25-22、11-25、25-20)戰勝八一隊。
