外媒評2019賽季中超最佳陣：恆大2將入選！董蘅毅成最大亮點

經過1年時間30輪的較量，2019賽季中超聯賽也終於落下了帷幕，廣州恆大最終捧得冠軍獎盃，不過在國外專業足球媒體《WhoScored》的中超賽季評分中，冠軍球隊廣州恆大卻只有2人入選了賽季最佳陣容，不過依然是入選人數與北京國安並列最多的球隊，2019賽季中超亞軍北京國安，同樣有2名球員入選了賽季最佳陣。

​外媒評選的中超最佳陣是按照442的陣型排列的，其中在前鋒位置上，本賽季打進29球刷新中超進球記錄的廣州富力前鋒扎哈維，毫無疑問入選了最佳陣，而與扎哈維搭檔鋒線的球員，就是來自於山東魯能的高中鋒佩萊，他在本賽季中超聯賽中，為山東魯能攻入了17球，並且有4個助攻，扎哈維和佩萊的組合，也足以成為中超最具威脅的鋒線搭檔了。

​在中場位置上，入選的4名球員均是外援，並且來自四支不同的中超俱樂部，他們分別是上海申花球員莫雷諾，上海上港球員奧斯卡，北京國安球員奧古斯托，以及大連一方球員卡拉斯科。這四名球員均是各自球隊本賽季的中場核心，其中奧古斯托更是以8.1分成為了外媒評選的中超賽季最佳陣中的最高分球員，他的能力和作用，在中超賽場是也是有目共睹的。

​而在後防線上，廣州恆大韓國籍中後衛朴志洙，成為了唯一一名入選最佳陣的後衛球員，而與朴志洙共同入選中超賽季最佳陣的後衛中，還有一名廣州恆大的隊友，他就是右邊後衛張琳芃。而入選中超賽季最佳陣的另外兩名後衛，分別是來自於北京國安的中後衛于大寶，以及河北華夏幸福左邊後衛姜至鵬。

​最後在外媒評選的2019賽季中超最佳陣門將位置上，出現了一個稍顯陌生的名字，他就是董蘅毅。能夠力壓國足主力門將顏駿凌，以及本賽季表現出色的王大雷、劉殿座等人入選賽季最佳陣，董蘅毅的實力顯然得到了外媒的肯定，他也成為了最佳陣最大的亮點。而董蘅毅的原名叫董春雨，本賽季是武漢卓爾的主力門將，已經28歲的他也可謂是大器晚成，此前入選了國足東亞杯名單，本賽季3次當選全場最佳，9次零封對手，入選最佳陣也是實至名歸。
