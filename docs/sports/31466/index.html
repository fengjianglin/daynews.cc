歐洲金靴榜：萊萬領銜，梅西場均1球的效率僅列第四

2019-12-23 2,953條評論 4,665次閱讀 880人點贊

除了英超外，其它聯賽都進入了冬歇期，在歐洲金靴榜單中，來自德甲的萊萬多夫斯基和維爾納分別以19球和18球佔據前兩名。曾6奪歐洲金靴的梅西暫列第5 ，而C羅排在了第18名。

歐洲金靴最新排名：

1、萊萬多夫斯基，拜仁慕尼黑，17場19球，38分

2、蒂莫-維爾納，萊比錫紅牛，17場18球，36分

3、因莫比萊，拉齊奧，16場17球，34分

4、瓦爾迪，萊斯特城，18場17球，34分

5、梅西，巴塞羅那，13場13球，26分

6、本耶德爾，摩納哥，16場13球，26分

7、魏斯曼，沃爾夫斯貝格，17場17球，25.5分

8、本澤馬，皇家馬德里，12球，24分

9、盧卡庫，國際米蘭，12球，24分

10、哈蘭德，薩爾茨堡紅牛，16球，24分

12、姆巴佩，巴黎聖日爾曼，11球，22分

13、奧巴梅揚，阿森納，11球，22分

14、亞伯拉罕，切爾西，11球，22分

18、C羅，尤文圖斯，14場10球，20分

雖然梅西在賽季初缺席了很多比賽，但從現在的場均進球率來看，梅西場均一球的效率可以說只是常規操作，但這仍遜色於更為神勇的前三名。
