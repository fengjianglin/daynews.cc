CBA第19輪過後，20支球隊積分排名全解析

2019-12-24 202條評論 2,927次閱讀 499人點贊

山東和遼寧「漁翁得利」、福建繼續翻身

12月20日，CBA聯賽第19輪比賽的4場比賽全部結束。主場作戰的福建隊以121比114擊敗吉林隊，北京以93-112輸給浙江、上海以106-93擊敗八一、主場作戰的廣東以121-101擊敗天津。第19輪過後CBA的積分榜情況如何呢？

廣東、新疆依然突出，北京面臨考驗

廣東取得兩連勝，以17勝2負排名榜首；新疆隊也兩連勝，以14勝4負排名第二。輸給浙江之後，雖然北京隊可以以13勝6負位居第三，但他們的位置已經很危險。如果他們繼續輸，北京的排名將下降，山西目前排在第四位，戰績是12勝6負。

山東、遼寧「漁翁得利」

第19輪比賽前，吉林隊的比分是12比6。如果他們能贏得福建，吉林將鞏固自己的地位。不幸的是，他們最終輸掉了比賽。因此，吉林和山東、遼寧的戰績是12勝7負，山東和遼寧都是「漁翁得利」，山東排名第五，遼寧排名第六，吉林排名第七。

浙江排名第十

從第八到第十一，分別是北控的11-8、Guangxia的11-8、浙江的10-8、青島的10-8。最近浙江和青島的勢頭不錯。廣夏最近的成績不好，在連續三次失敗後，他們跌到了第九位。如果團隊仍然找不到合適的小外援，廣廈的排名可能會繼續下降。

福建繼續逆襲

從第十二到第二十，分別是江蘇的9-10、Tongxi的天津、深圳的廣州、福建的7-12、上海的7-12、四川的馬拉、中國的馬拉、中國的馬拉、八一的馬拉。最引人注目的是福建隊，福建隊已經連續贏了四場比賽。儘管泰-勞森的突然受傷讓福建蒙上陰影，可是普爾-杰特的首秀拿到22分15次助攻還是讓人眼前一亮，福建還有希望繼續逆襲。

目前，四川隊、廣州隊、天津隊和八一隊很難創造奇蹟了。隨著福建隊的出色表現，他們已經脫離了倒數的局面，希望福建可以繼續保持下去。積分榜第三和第十一名的隊伍競爭相當激烈。都是比較關鍵的卡位戰，相信還會持續一段時間。從12到16位，這些隊伍處於臨界點。他們需要抓住每一個機會去贏得比賽。
