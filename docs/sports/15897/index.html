西甲：皇家社會大戰巴塞羅那，傷病外加紅牌，這場恐怕難勝

2019-12-15 703條評論 4,508次閱讀 1046人點贊

巴塞羅那在北京時間12月14日22:00將迎來與皇家社會的對決，這是一場西甲第四名大戰西甲第一的「龍頭好戲」。對於皇家社會來說，拿下這場比賽顯得至關重要。積分27分的皇家社會需要這場勝利拉開與後面緊追不捨的赫塔費的差距。而對於巴塞羅那來說這場比賽更不能有失，因為現在皇家馬德里以同樣的積分在身後虎視眈眈。

從歷史戰績來看，皇家社會本就是凶多吉少，在最近雙方交戰的六場對決中，皇家社會沒能夠擊敗巴塞羅那一場，雖然每次差距都不大，最近三場對決就差一個球，但是巴塞羅那就像一個魔咒一樣牢牢的纏住了皇家社會的脖子，讓他們無法動彈，不論多麼有希望最後依然得到的是失敗，從2016年4月10日雙方西甲對決皇家社會贏了巴塞羅那之後，皇家社會面對巴塞羅那再也沒能夠獲得勝利。

除了歷史戰績堪憂之外，皇家社會的傷病也是堪憂，現在官方確定傷病的球員包括中場球員伊利亞拉門迪和蘇魯圖薩，後衛球員埃盧斯通多。而巴塞羅那現在確定受傷的球員只有奧斯曼登貝萊。除此之外皇家社會的後衛勒諾爾芒也因為累積兩張黃牌（兩黃變一紅）停賽。現在的皇家社會完全面臨著尷尬的中衛荒，為這場本就不好打的比賽再掀起一陣波瀾。

在陣容和首發方面，通過多年的看球經驗，我覺得雙方大概率都使用4-3-3的陣型來進行對決。皇家社會這邊鋒位預測波圖，威廉若澤以及奧亞薩瓦爾首發；巴塞羅那這邊將以格列茲曼，蘇亞雷斯，梅西首發。總的來說皇家社會本就尷尬的中場位置恐將派出厄德高，格瓦拉以及梅里諾來應付。

總的來說不管是從哪一方面來看，巴塞羅那拿下這場比賽的概率真的是非常的大。在經歷過一段時間的傷病潮之後，巴塞羅那也算是完美地堅持了過來。對於現在的巴塞羅那來說穩住便能夠獲得自己想要的冠軍之位。

不管如何，這場比賽的結果將直接決定西甲的全新排名，這場比賽過後前面的排名也許會直接大變樣，所以這場比賽對於強隊的意義不言而喻，究竟是皇家社會大發神威斬衛冕冠軍於馬下。還是巴塞羅那穩定發揮送皇家社會出前四，這一切的一切都將在今晚的比賽之後揭曉答案，結果究竟如何讓我們拭目以待。
