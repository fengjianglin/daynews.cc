梅西不僅會進球！距離曼聯名宿的助攻紀錄，或在戰皇馬時打破

2019-12-15 1,787條評論 4,291次閱讀 956人點贊

西甲聯賽第17輪一場焦點大戰，巴薩客場挑戰皇家社會。奧亞薩瓦爾點球破門，幫助皇家社會首開紀錄。格列茲曼和蘇亞雷斯先後進球，幫助巴薩逆轉比分。伊薩克推射得分，幫助皇家社會扳平比分。最終，巴薩以2比2的比分與皇家社會握手言和。

在多賽一場比賽的情況下，巴薩目前仍然力壓皇馬1個積分，暫居西甲聯賽積分榜首位。可是很多人認為，本輪比賽過後，皇馬將會力壓巴薩，登頂西甲聯賽積分榜。

在本場比賽中，梅西並沒有取得進球。事實上，是梅西非常無私地讓出了一粒進球。第49分鐘，巴薩中圈附近送出過頂長傳，梅西帶球插入禁區後形成單刀。此時梅西選擇將皮球傳給右側的蘇亞雷斯，無人盯防的蘇亞雷斯推射空門得分。

以梅西的能力，如果他這球選擇射門的話，很大幾率會打進，就算沒有打進，這也是一次正確的射門抉擇。可是，梅西還是將球分給了蘇亞雷斯，自己收穫了一次助攻。

據統計，本場比賽收穫1次助攻後，梅西在西甲聯賽中一共送出了168助攻。這一項數據，早就已經冠絕西甲聯賽。就算放到5大聯賽，也只有一人能夠壓上梅西一頭，他就是曼聯名宿吉格斯。在這職業生涯中，曼聯名宿吉格斯在聯賽中一共送出了169次助攻。

目前，梅西距離打破曼聯名宿吉格斯的聯賽助攻紀錄，只差2次助攻。也就是說，下一場巴薩對陣皇馬的西甲聯賽，梅西很有可能將會創造新的歷史紀錄。

不得不說，梅西是一名非常全面的球員，在進攻方面，不僅僅會進球，還會助攻。這也難怪儘管梅西已經32歲，可是他依舊是巴薩目前無可爭議的領袖，沒有任何人能夠撼動他的地位。

以梅西目前的狀態來看，想要在巴薩對陣皇馬的西班牙國家德比中送出助攻，並非難事。可是，相比於送上助攻，打破曼聯名宿吉格斯的紀錄，球迷們肯定更希望梅西在巴薩對戰皇馬的比賽中多進幾球。
