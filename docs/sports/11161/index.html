勁爆！CBA最被低估國手關鍵三分+決勝籃板，他比周琦更值得信賴？

2019-12-11 1,479條評論 3,573次閱讀 1412人點贊

北京時間12月10日，CBA第15輪繼續開打，單外援出戰的新疆男籃客場114-110險勝山西隊男籃，由於費爾德的缺陣，新加盟的外援埃里克-米卡儘管能打出29分16籃板的數據，但從比賽走勢和默契程度來看，他依舊需要一段較為場的時間去適應阿的江指導的體系，面對殘陣的新疆男籃，王非指導迎來最佳證明自己的機會，多名主力球員發揮出色，任俊威18分8籃板，原帥15分3籃板，其中富蘭克林得到42分12助攻10籃板10搶斷，這是CBA聯賽近10年來的第一次四雙數據，但卻依舊不能贏得一場比賽勝利，這或許是山西男籃今夜最大的遺憾吧。

比賽真正意義上的決戰時刻是從最後一分鐘開始，其中阿不都扮演著重要角色，先是最後的15.5秒，他命中大心臟的關鍵3分幫助新疆隊反超比分，在最後10秒的關鍵回合中，他又搶下一個決定比賽勝負的籃板球，如果是今晚新疆男籃的最大優勢是內線球員，那麼阿不都強行撐起外線火力，全場出戰37分鐘，13投8中，得到20分4籃板2助攻的數據，成為今晚客場險勝的英雄，他沒有辜負阿的江指導的信賴。

事實上，隨著阿的江指導有意讓周琦和埃里克-米卡承擔去進攻重任，整個新疆男籃前面三節的戰術重心都放在內線體系中，兩人也確實憑藉個人能力撐起新疆男籃大部分的球權消化，周琦僅僅5次投籃，卻可以憑藉自身的牽制力拿到13次罰籃機會，得到20分12籃板7個蓋帽的豪華數據，米卡同樣第四節的關鍵時刻打出強硬2+1，兩人幾乎把山西男籃的內線打爆了，但在下半場，周琦的體能明顯感到下滑，進攻慾望不強，場上的存在感與上半場相比，威脅力下降不止一個檔次，此時的阿不都突破和造殺傷來幫助新疆取得分數，第三節5投4中，拿下8分，第四節拿到7分中，全是關鍵時刻得分，堪稱球球都是金子般。

阿不都在新賽季中，他依舊不止一次證明自己具備關鍵時刻接管比賽的能力，在阿的江指導的排兵布陣和人員輪換上，他的戰術地位比周琦更值得信賴，特別是第四節的決戰時刻，阿不都多次上演力挽狂瀾的好戲，如果說賀希寧是最具存在感的大心臟先生，那麼阿不都應該是最被低估的全能國手。
