CCTV5直播曼聯兩場關鍵戰，索帥將在3天時間接連遭遇穆帥和瓜帥

摘要

上一輪聯賽，曼聯隊又被一支中下游的球隊阿斯頓維拉給逼平了，這已經是曼聯隊連續兩場英超聯賽被中下游球隊給逼平了。最近4輪聯賽曼聯僅收穫了1勝2平1負的戰績排名第9名，已經逐漸淡出了歐戰區了，如果戰績未能有所好轉的話，本賽季想要重回歐戰區就變得困難了！

上一輪聯賽，曼聯隊又被一支中下游的球隊阿斯頓維拉給逼平了，這已經是曼聯隊連續兩場英超聯賽被中下游球隊給逼平了。最近4輪聯賽曼聯僅收穫了1勝2平1負的戰績排名第9名，已經逐漸淡出了歐戰區了，如果戰績未能有所好轉的話，本賽季想要重回歐戰區就變得困難了！而在接下來的兩輪聯賽，曼聯將會連續遭遇強敵熱刺和曼城，CCTV5體育頻道將會曼聯隊的這兩場關鍵比賽進行現場直播，索爾斯克亞的帥位能不能保住，這兩場比賽的勝負很關鍵！

12月5日凌晨，英超第15輪聯賽，曼聯VS熱刺

北京時間12月5日凌晨，英超聯賽第15輪，曼聯隊將在主場迎戰剛剛換帥的熱刺隊，這是穆里尼奧上任熱刺主帥之後首次回到老特拉福德球場。與曼聯隊近期遭遇2連平不同，熱刺隊在穆里尼奧上任之後，已經在英超和歐冠賽場上收穫了3連勝的佳績了，在英超的積分榜上連升9個位置，目前已經衝到了前5名。而穆里尼奧之前正是因為與曼聯隊中的多名球員鬧不和才從曼聯隊下課的，如今時隔一年之外率領另外一支球隊重返老特拉福德球場，面對自己在曼聯隊的繼任者索爾斯克亞，穆里尼奧肯定要用一場勝利來證明自己！

穆里尼奧重返老特拉福德球場，讓這場比賽在還未開始就吸引了全球很多球迷的目光：穆里尼奧重返老特拉福德球場，迎接他的會是掌聲呢？還是噓聲呢？北京時間12月5日凌晨英超比賽現場見分曉，CCTV5央視體育頻道將會隊本場比賽進行直播！熱刺隊最近兩輪聯賽都獲得了勝利，打進6球失4球，進攻火力猛，但是防守並不穩定；而曼聯最近兩輪聯賽都是平局，打進5球丟5球，進攻不錯，防守也同樣漏洞百出，本輪對決誰能夠先穩固防守，誰獲勝的機會就大一些！

12月8日凌晨，英超第16輪聯賽，曼城VS曼聯

在剛剛經歷了一場與熱刺隊的強強對話不到3天時間，曼聯隊將在客場迎來一場曼徹斯特德比大戰，這次的對手是更加強大的曼城隊！曼城隊在第14輪聯賽再次被對手在比賽最後時刻扳平了比分，在英超積分榜上已經落後榜首的利物浦11分了，甚至連聯賽第二名的位置都已經保不住了，這對於瓜迪奧拉來說是不可容忍的。如果曼城隊還想要保持對冠軍的爭奪權，那麼在接下來的任何一場比賽都不能在出現差池了，否則就將徹底失去對英超冠軍獎盃的爭奪權了，而曼徹斯特德比大戰更是瓜迪奧拉不能出現差池的比賽！

同城德比大戰一直都是最激烈的比賽爭奪，在以前曼聯隊一直都在同城德比大戰中力壓曼城隊一頭，僅幾年隨著曼城隊的投入加大和弗格森老爺子的退休，曼城隊才逐漸在曼徹斯特德比大戰佔據了優勢。本賽季，曼聯隊的比賽狀態更是不能與曼城隊相提並論了，曼城隊雖然沒有了上個賽季那樣所向披靡的殺氣，依舊在積分榜上領先曼聯隊11分之多。第16輪聯賽，剛剛遭遇了一場與熱刺隊大戰的曼聯隊在主場恐怕很難抵擋曼城隊強大的攻勢，索爾斯克亞恐怕難逃一敗，搞不好會因此下課！本場曼徹斯特德比大戰，CCTV5體育頻道同樣會有現場直播！

​曼聯隊在狀態低迷的時候連續遭遇熱刺隊和曼城隊兩支強敵，這對於索爾斯克亞率領的曼聯隊來說是一大考驗，如果這兩場比賽曼聯隊都輸掉了，那麼索爾斯克亞的帥位恐怕就難保了，索爾斯克亞恐怕要步波切蒂諾和埃梅里的後塵成為英超第3名被下課的豪門教練了！
