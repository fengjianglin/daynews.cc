阿魯姆，我不認為小斯賓斯會在近期重返拳台

2019-12-14 1,916條評論 3,633次閱讀 1887人點贊

2019年12月14日，在美國紐約的拳擊聖地麥迪遜廣場花園，現任WBO次中量級世界拳王特倫斯·克勞福德將在自己的世界拳王衛冕戰中迎戰立陶宛拳手埃吉迪尤斯·卡瓦力奧斯卡斯（Egidijus Kavaliauskas）。

在賽前的新聞發布會上，不少拳迷十分關注特倫斯·克勞福德是否有計劃和現任IBF、WBC次中量級世界拳王小埃羅爾·斯賓斯進行世界拳王統一戰。

對此，作為特倫斯·克勞福德賽事推廣人的Top Rank（頂級推廣公司）CEO鮑勃·阿魯姆並不認為特倫斯·克勞福德與小埃羅爾·斯賓斯的比賽會在2020年甚至2021年舉行比賽。

鮑勃·阿魯姆表示，根據他最近被告知的關於小埃羅爾·斯賓斯在10月份車禍中受傷的恢復情況，他懷疑小埃羅爾·斯賓斯在身體上根本沒有做好重返拳擊台的準備。

據悉，自從10月10日凌晨在達拉斯市中心駕駛法拉利撞車事故中蘇醒之後，小埃羅爾·斯賓斯極其團隊沒有提供任何有關自己身體狀況的最新消息。甚至連專門負責小埃羅爾·斯賓斯對外宣傳工作的經紀人也沒有對小埃羅爾·斯賓斯過去兩個月的健康狀況發表任何評論。

鮑勃·阿魯姆在接受IFL電視台採訪時表示，有可靠消息來源告訴他，由於鮑勃·阿魯姆的傷勢，他在「可預見的未來」將無法參加比賽。據報道，小埃羅爾·斯賓斯在出院後被控醉酒駕車，不過萬幸的是在那次車禍中小埃羅爾·斯賓斯僅僅面部撕裂，牙齒受損。

鮑勃·阿魯姆在接受IFL電視台採訪時說：「我不認為明年會舉行這場比賽。並不是說賽事推廣人不想舉行這場比賽，而是因為我非常懷疑小埃羅爾·斯賓斯是否會在可預見的將來重返拳壇。儘管我發看到他重登拳台是很不幸的結局。但我得到了一些內部消息，所以我才會這麼說。我現在意思是，讓我們不要關注小埃羅爾·斯賓斯了。小埃羅爾·斯賓斯是個可愛的年輕人，但是他經歷的一次可怕的事故，現在應該讓他靜靜的恢復。」

不過，現年88歲的鮑勃·阿魯姆拒絕向記者透露是誰告訴了他小埃羅爾·斯賓斯現在的健康狀況。

鮑勃·阿魯姆對IFL電視台記者說：「告訴你們消息來源？我不認為這是合適的。」

不能與小埃羅爾·斯賓斯雖然遺憾，但是並不意味著沒有其他備選方案。事實上，在本周的早些時候，鮑勃·阿魯姆在接受採訪時就曾表示，他希望能把特倫斯·克勞福德和肖恩·波特撮合在一起進行一場比賽。今年的9月28日，在美國洛杉磯斯台普斯中心舉行的世界拳王統一戰中，肖恩·波特輸給了小埃羅爾·斯賓斯，丟掉了自己的WBC世界拳王金腰帶。

不過，現在的特倫斯·克勞福德還是要先擊敗卡瓦力奧斯卡斯。來自立陶宛的卡瓦力奧斯卡斯目前是WBO次中量級的強制挑戰者，兩人的比賽將於當地時間本周六晚上在紐約麥迪遜廣場花園舉行，比賽將通過ESPN電視台進行現場直播。
