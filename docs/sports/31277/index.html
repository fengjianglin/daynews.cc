曼聯和當初利物浦有何區別？利物浦只是守弱，曼聯只會反擊

2019-12-24 916條評論 4,567次閱讀 2656人點贊

本賽季，曼聯對陣聯賽中下游球隊的時候，表現非常掙扎，曼聯對陣聯賽第八以後的球隊，11場比賽只拿到了10分，場均不到1分，但是對陣前八的球隊，7場就拿到了15分，正是因為如此，不少球迷都戲稱曼聯是「劫富濟貧」。說起劫富濟貧，不得不說一下目前排名榜首的利物浦，前幾個賽季，利物浦也是劫富濟貧，那麼曼聯和利物浦的區別在哪裡呢？

現在，許多球迷都在期望曼聯可以像當初利物浦一樣，在劫富濟貧之後，逐漸開始崛起。但是，曼聯和利物浦雖然同樣難以戰勝中下游球隊，但是雙方是有著本質的區別的，利物浦當時經常輸或者平中下游球隊，純粹就是因為防守不利。當時的利物浦攻強守弱，球隊整個賽季無論遇到誰都是進攻，進攻再進攻。而遇到強隊的時候，對方攻出來了，利物浦不怕和對方打對攻，所以往往可以贏球，而對陣弱隊的時候，利物浦一樣攻出來了，但是對方反擊的時候，往往也可以獲得很多的機會，這才導致利物浦出現「劫富濟貧」的情況。

利物浦雖然劫富濟貧，但是他們對陣中下游球隊的時候，幾乎每場比賽都可以進球，當然他們丟球也不少。克洛普入主利物浦的第一個完整的賽季是16-17賽季，那個賽季利物浦利物浦雖然只排名第四，但是他們的進球很多，整個賽季打進78球，但是也丟掉了42球，他們場均進球2.01球，幾乎每場比賽都有進球，不管對方實力如何。而17-18賽季利物浦依然排名第四，但是他們的進攻更強了，整個賽季打進84球，但是也丟掉了38球，而這個丟掉38球，主要是因為冬窗范迪克加盟了，不然利物浦的丟球依然多。

從數據來看，利物浦是一直都在進攻的，他們的口號就是「輸也要輸在進攻的路上」。不過，曼聯則有所不同，本賽季的曼聯18場比賽打進26球丟掉22球，他們的場均進球只有1.4球，丟球也有1.2球。可以說，曼聯本賽季是攻不出去，同時也守不住。而利物浦呢？當時純粹是守不住而已，雙方的區別就是這麼簡單。

目前的曼聯，主要問題在於戰術太過單一，球隊只會防守反擊，面對弱隊的密集防守，他們就無計可施，而當初的利物浦呢？面對密集防守他們有辦法進球，但是他們無法保證自己不丟球。這就是為什麼，利物浦接連買來范迪克和阿利松之後，球隊馬上就完成蛻變的原因。他們的前場進攻端沒問題，打反擊，打陣地戰都玩的轉，就是後防線不給力。而目前曼聯的前鋒似乎只會反擊，狹小空間範圍內打不出來配合。這也是曼聯為啥總是無法打破密集防守最大的原因。

​

目前的曼聯和當初的利物浦其實還是有本質的區別的，而最大的區別或許就在主帥身上，老實說當時利物浦的陣容，並不比現在的曼聯好，當時利物浦前場依靠的是馬內、菲爾米諾和拉拉納，中場依靠的是庫蒂尼奧、亨德森和米爾納，這樣的組合其實比起現在曼聯的組合還是有所不如的，但是這種情況下，克洛普把利物浦的進攻完全調教好了，而進攻好了以後，加強一下後防線，球隊馬上就成型了，現在的曼聯呢？中前後場或許都有問題，如果想要回到巔峰，或許真的需要找來一個克洛普這樣的名帥好好調教一下了！
