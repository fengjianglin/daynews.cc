朱婷舊傷複發誰之過？天津女排並非全部責任，排協本可效仿土超

天天要聞
2019-12-09

如今，世俱杯已落下帷幕，令人失望的是，國內兩大豪強恆大和天津分別位居第7、第8墊底的名次，而更令人失望的就是，作為中國女排核心的朱婷，在本屆世俱杯當中不慎舊傷複發，這引起了不少球迷的不滿。朱婷舊傷複發誰之過？天津女排並非全部責任，排協本可效仿土超

要知道，早在朱婷回國之前，就早已明確了回國的任務，那就是更好的備戰未來東京奧運會的賽程，至於朱婷為何要回國參加排超聯賽，最根本的原因就是賽事較少，能夠擁有更多和國家隊隊友合練的時間，可是讓人沒想到的是，才剛剛回國參賽不久的朱婷，就遭遇到了如此的傷病，這可謂是和郎平召集朱婷回國參賽的初衷明顯不符，也正是因為朱婷的舊傷複發，這也讓天津女排遭到了外界不少的指責，認為朱婷的受傷，這和天津女排的過度使用有著極大的關係。

當然，這一點也不能全怪天津女排，畢竟花了大價錢引進的朱婷，在有著極大希望的情況下，自然想要通過朱婷的全能身手來保障天津女排的實力，為天津女排贏得世俱杯的最佳戰績，只不過朱婷作為世界第一主攻，無論是古德蒂還是郎平，對朱婷的使用都是慎之又慎，可是到了天津女排以後，天津女排對朱婷的使用也是有目共睹，從世俱杯首戰開始，朱婷就將進攻、一傳、防守和攔網一肩挑，幹了所有能夠參與的工作，這明顯和瓦基弗以及國家隊並不同，而這最終的結果就是，繁重的工作做成了朱婷的舊傷複發，讓不少球迷都是心疼不已。

不止如此，導致朱婷舊傷複發的原因遠不止是天津女排對朱婷的過度使用方面，還有就是中國排協對賽程的安排讓人難以理解，本賽季的排超聯賽，排協為了留給國家隊更多的集訓時間，處於好意將聯賽賽場縮短，可是這樣一來的後果就是讓球員們的疲勞程度增加，增添了傷病隱患，而且在世俱杯的間歇期對賽事的安排，排協的做法也是有待商榷，要知道，自從11月30日排超8強賽首輪開始，雙線作戰的天津女排就要迎來11天7戰的賽事，這堪稱是魔鬼賽程，本就是給球員帶來了傷病隱患。

反觀土超聯賽，瓦基弗銀行和伊薩奇巴希在進行完第10輪比賽過後，土超聯賽就給予兩隊充足的休整時間，而在世俱杯過後，土超聯賽第11輪比賽也將於12月中旬才將再次開打，顯然，這和排超聯賽之間有著不小的差距，排協如此密集的賽程安排，或許將會釀成好心辦壞事的後果，一旦球員們因為如此密集的賽程而再次遭遇傷病的話，這將對中國女排來講得不償失，如今的排協或許也該反思一下，是否該效仿土超聯賽來保護好國內球員呢？

對於朱婷的使用問題，小編也是希望此次朱婷的舊傷複發能夠引起天津女排和排協的重視，朱婷回國最重要的任務是在於東京奧運會，不要去計較一城一池的得失，而影響到了未來的東京奧運會。朱婷舊傷複發誰之過？天津女排並非全部責任，排協本可效仿土超
