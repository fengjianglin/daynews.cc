恆大奪冠，蘇寧壓哨超魯能，中超落幕盤點，附官方最終積分榜

2019-12-01 0條評論 20次閱讀 0人點贊

摘要

12月1日下午，隨著8場比賽的全部結束，2019賽季中超聯賽也曲終人散。隨後，中超官方發布了賽季最終積分榜。其中，廣州恆大淘寶憑藉下半賽季的出色表現，尤其最後3輪的連勝，最終力壓國安和上港，問鼎冠軍，並實現了八冠王的榮耀。北京中赫國安絕殺魯能後，最終獲得亞軍。

12月1日下午，隨著8場比賽的全部結束，2019賽季中超聯賽也曲終人散。隨後，中超官方發布了賽季最終積分榜。

其中，廣州恆大淘寶憑藉下半賽季的出色表現，尤其最後3輪的連勝，最終力壓國安和上港，問鼎冠軍，並實現了八冠王的榮耀。

北京中赫國安絕殺魯能後，最終獲得亞軍。本賽季，國安大致呈高開低走態勢，並經歷了換帥動蕩。本來，開局10連勝，獲得半程冠軍的他們，還是非常有希望爭奪冠軍的，可惜下半階段沒能頂住壓力，只能眼睜睜看著冠軍再度溜走，只能來年再戰了。

衛冕冠軍上海上港本賽季在武磊離隊之後，總是磕磕絆絆，並在幾場關鍵戰役中掉了鏈子，最終只能飲恨，接受衛冕失利，只能獲得季軍的事實。

江蘇蘇寧易購在連續多輪與魯能形成1分分差的僵持局面後，最終在末輪2-0戰勝天津天海，壓哨完成了反超。這個成績非常不容易，至少說明他們始終沒有放棄追趕的念頭，前進的動力。第4名的成績也是對他們的最佳肯定。

山東魯能泰山在2-3被國安絕殺之後，有些遺憾地讓出了第4名位置，最終被擠到了第5名。雖然第4名和第5名沒有實質區別，但最後一輪被超越，還是有些讓人遺憾。好在接下來，魯能將與申花進行足協杯決賽次回合爭奪，並有望再度奪得足協杯冠軍。如果能足協杯奪冠，那三線作戰的魯能，本賽季也可以比較完美結束了。

作為升班馬的武漢卓爾，雖然收官戰主場1-2不敵河北華夏幸福，但最終能積44分排名第6，成為一匹黑馬，說明他們還是非常有實力和韌勁的。主帥李鐵也是很有能力的。與此形成鮮明對比的是，另一支升班馬深圳佳兆業則黯然降級。

天津泰達主場2-0完勝重慶斯威，也讓他們本賽季圓滿結束。相比較上賽季最後一輪還在掙扎著保級，本賽季的泰達在主帥施蒂利克帶領下，憑藉「阿森納」組合的強大威力，最終換了個活法，讓人刮目相看。

河南建業主場1-0戰勝了廣州富力，排名最後定格在第8。相比較上賽季，本賽季建業也取得了較大進步。更值得稱讚的是，最後幾輪關鍵場次比賽中，他們充分發揚公平競賽精神，努力打好了每一場比賽的每一分鐘，捍衛了中原勁旅的榮譽。

大連一方在最後一輪2-0擊敗北京人和，用一場勝利為本賽季畫上句號。不過，在球迷眼裡，這個句號並不完美。賽季初大手筆完成「九宮格」引援的他們，並未踢出預期成績，第9名的戰績也與昔日大連王朝相去甚遠。下賽季，球迷的期待會更高，就看一方如何提升了。

重慶斯威末輪不敵泰達，最終排名第10。與上賽季相比，他們也有了明顯進步。只是在下半賽季「小摩托」費爾南多離開，隊長卡爾德克受傷後，他們的整體實力受到了影響，所以沒有上半賽季那麼搶眼。

河北華夏幸福客場2-1戰勝卓爾後，也用勝利收官。他們最後積33分，超過富力排到了第11名。經歷了換帥等諸多波折的華夏幸福，最後如願完成了保級，但在阿根廷雙星馬斯切拉諾、拉維奇離開之後，下賽季他們將何去何從呢？

廣州富力0-1不敵建業，以3連敗結束了本賽季。雖然鋒霸扎哈維獲得金靴並打破了中超單賽季進球紀錄，但富力也僅差3個球就打破遼寧隊當年創造的單賽季丟球最多紀錄。如紙糊般的羸弱防線，一直是富力為人詬病的地方。下賽季，他們可不能繼續這麼活，否則就真有降級危險了。

上海綠地申花末輪也遭遇了失利，他們是0-3客場輸給了冠軍恆大。但考慮到他們即將和魯能展開足協杯決戰，這場失利也能理解。不過，最終第13名的尷尬成績，確實讓這支上海灘豪門失色不少。下賽季，申花也必須做出改變了。

天津天海是本賽季最為動蕩的一支球隊，投資人被司法控制，俱樂部變更，主帥3度調整……但不管怎樣，他們最終頑強完成了保級，留在了中超。可是，下賽季怎麼辦，這已經是擺在他們面前的一道現實難題。

深圳佳兆業從中甲升級之後，一度讓人充滿期待，尤其是前幾輪，大有扮演黑馬角色之勢，可惜，隨著聯賽的深入推進，他們最終掉了鏈子。等發現遇到危機，奮起直追時，為時已晚。這一個賽季里，深圳隊的投入也非常大，現在看來，只能當交了沉重學費了。

北京人和雖然也很努力，但還是被傳說中的「升班馬」二年級定律給羈絆住了，並悲慘降級。從最近幾個賽季來看，降級球隊很難短時間內再打回來，人和能做到嗎？讓我們拭目以待。

至此，本賽季中超大幕正式落下。讓我們下賽季再見！也感謝大家對「凌波薇笑」的關注。接下來，波仔在關注中超、中甲、中乙冬季轉會等動態信息的同時，還會聚焦CBA賽事，歡迎大家繼續支持！
