又一位中國人衝擊世界拳王！新疆最硬男人慾阻擊全勝拳王

2019-12-23 1,700條評論 3,340次閱讀 1283人點贊

近日，據世界權威拳擊統計網站BoxRec公布的認證預告，定於2020年2月2日在海口觀瀾湖舉辦的拳擊超級晚系列賽將有一場重要的比賽已經敲定，屆時來自新疆的中量級拳手伊力夏提將與巴西不敗拳王法爾考展開世紀大戰，如果伊力夏提能夠獲勝，他將有望向中量級世界拳王發起挑戰，而這也是中國拳手有史以來爭奪世界金腰帶級別最大的一次。

據了解，拳擊超級晚是由盛力世家、海口觀瀾湖和美國TOP RANK斥資打造的中國搏擊史上最高規格的拳壇盛世，作為當晚中國軍團向世界金腰帶發起衝鋒的主力軍，26歲的伊力夏提被拳迷寄予厚望，自從2015年轉戰職業拳壇至今，伊力夏提取得了17勝1負12場KO對手的不俗戰績，是盛力世家重點培養的拳手之一，與祖力皮卡爾被譽為「新疆雙子星」。事實上，在今年11月初海口舉行的拳擊超級晚發布會上，TOP RANK推廣人阿魯姆就已經透露了伊力夏提接下來的對手人選，阿魯姆當時表示巴西人法爾考或擊敗過村田諒太的前WBA中量級掌門布蘭特將成為伊力夏提通往世界拳王前的攔路虎。

如今隨著法爾考浮出水面，也就意味著伊力夏提必須越過這道坎。現年30歲的法爾考目前取得了25戰全勝17場KO對手的驕人戰績，曾在2012年倫敦奧運會拿到了中量級銀牌，在決賽中以13-14的1分惜敗給了兩度WBA中量級冠軍村田諒太，實力不容小覷。法爾考目前在WBC160磅排名第5、IBF暫列第6，具備向阿瓦雷茲、戈洛夫金和查洛挑戰的資格，而他也想通過海口一戰鞏固自己的排名，向法定挑戰者的位置發起衝擊。

綜合兩方的實力對比，伊力夏提應該稍處劣勢，這場較量他必須使出全力去拼對方才有更大的勝算，基於雙方不俗的KO率和勇往直前的拳擊風格，這場硬漢之戰註定會十分慘烈。除伊力夏提以外，盛力世家另兩位中國好手楊永強和祖力皮卡爾也將攜手出戰，向各自職業生涯的新高發起衝擊。至於中國勇士將為世界十億拳迷帶來哪些驚喜，海口觀瀾湖將全面揭曉答案！
