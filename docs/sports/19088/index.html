曼聯球迷怒斥索帥：他根本不是一個主帥，他在曼聯永遠無法奪冠

2019-12-17 219條評論 1,350次閱讀 144人點贊

剛剛結束的英超第17輪的比賽中，曼聯主場1比1戰平埃弗頓，這場比賽的結果是曼聯球迷不能接受的，要知道這場比賽埃弗頓傷病嚴重，防守型中場戈麥斯、施奈德林等人全部因傷缺陣，球隊的中場核心西古德森也因傷缺陣，球隊的中場都是後衛客串的，但是這種情況下曼聯依然沒有獲得勝利，賽後不少曼聯球迷感到不滿。

比賽結束之後，球迷接受採訪時表示：【我只能說主帥太平庸了，球員也都在關鍵的時候掉鏈子，歸根結底還是打法有問題。我們今天的戰術計劃是什麼？對手都沒有中場了，我們還是被戲耍了，整場比賽他們就一個中場，但是我們在幹什麼？我們想要幹什麼？索爾斯克亞要幹什麼？陷入困境的時候我們要有作為，要有打法，要去分析對手，好的主帥會找到贏球的方法，但是索爾斯克亞什麼都不會。】

隨後球迷激動的表示：【索爾斯克亞根本就不是一個主帥，主帥需要交給球員戰術和戰法，看看我們的球員，他們的反擊很棒，但是索爾斯克亞只會反擊，但是你不能一場比賽90分鐘全部都去反擊，你需要有其他的戰術，但是索爾斯克亞始終都是老一套，有索爾斯克亞在我們永遠無法拿到冠軍。我覺得下賽季我們需要換帥了，波切蒂諾現在閑著，我們需要把他帶來，他才是一個真正的教練。】

索爾斯克亞執教曼聯也有一年的時間了，他來到曼聯之後的表現有好也有壞，總得來說索爾斯克亞在管理方面做得不錯，在引援方面也做得不錯。不過他的缺點同樣明顯，他的戰術安排太過單一，就像球迷說的那樣，就是防反，球隊的戰術太單一，戰術單一也就算了，他的換人和陣型變化也單一，看看他最近一段的換人，球隊打不開局面，格林伍德換林加德，然後球隊變陣424，拉什福德和馬夏爾雙中鋒，詹姆斯去左路，格林伍德在右路，這個套路球迷都知道了，何況對方的主帥呢？

事實上，索爾斯克亞在很多方面做得都不錯，但是唯獨就是他的戰術布置，臨場指揮是一個很大的問題，如果曼聯想要重回巔峰，或許真的如球迷所說的那樣，球隊需要更換一個主帥了。當然， 如果不更換主帥的話，或許應該讓索爾斯克亞和弗格森爵爺好好學學戰術指揮，不然的話，曼聯在他治下，真的難以奪冠，畢竟想要奪冠光能贏強隊是沒用的。
