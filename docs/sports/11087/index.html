拜仁的巴薩中場情節-折騰的庫蒂尼奧與穩固的蒂亞戈反差

2019-12-11 93條評論 2,512次閱讀 1043人點贊

同樣是巴塞羅那出品的中場球員，同樣在中場人才逐漸凋零的拜仁慕尼黑，命運截然相反。

蒂亞戈在早早離開巴塞羅那之後，在拜仁已經效力多年，即使沒趕上巔峰時期，也成了拜仁不可或缺中場一員。

庫蒂尼奧在利物浦立足多年，轉戰同樣中（前）場逐漸凋零的巴塞羅那，卻重金而來，突然離去。

在拜仁的庫蒂尼奧，遠沒有在巴薩的比達爾那般重要，加之教練的更換，更像是進入了一地雞毛的窘境。

倒是蒂亞戈依然還是重量級人物，無法忽視。

神奇的人物在拔高庫蒂尼奧，更神奇的人物在貶低蒂亞戈，即使在拜仁新敗的場次中，相比於庫蒂尼奧，蒂亞戈的作用也異常明顯。

蒂亞戈與萊萬多夫斯基這樣的挑傳與射門配合，依稀有當年哈維與梅西、比利亞協作的味道。

攻防兩端都有建樹的蒂亞戈，能夠比庫蒂尼奧更勝一籌，無數場景都在證明。

蒂亞戈這樣為數不多還能夠在中場及時補漏的夥計，避免了拜仁出更多的事故。

拜仁經常性中場複雜的操作，倒是同時影響庫蒂尼奧和蒂亞戈的發揮。

庫蒂尼奧沒能融入，蒂亞戈沒能夠更進一步，同時出現所謂的轉會段子，倒也像是對拜仁的提醒：中場已經連續損失克羅斯、比達爾之後，已經無法再承擔中場人員損失與消耗。
