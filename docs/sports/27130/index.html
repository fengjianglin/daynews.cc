奧運冠軍、浙江團省委副書記周蘇紅的新職務

2019-12-22 1,985條評論 4,442次閱讀 3191人點贊

今年7月出任共青團浙江省委副書記後，前中國女排主力、奧運冠軍周蘇紅有了新職務。

昨日，浙江公布省委省政府新聞發言人名單及電話，長安街知事（微信ID：Capitalnews）在其中發現，周蘇紅已擔任團省委新聞發言人一職。

周蘇紅是中國女排04黃金一代的代表性人物。她15歲進入浙江省青年隊，17歲入選國青隊，20歲時進入了國家隊。

作為運動員，周蘇紅獲得了許多榮譽，包括2003年世界盃冠軍，2004年雅典奧運會冠軍，2008年北京奧運會季軍，2002、2006、2010年三屆亞運會冠軍。

2011年退役後，周蘇紅任職於浙江體育職業技術學院。2016年，她出任浙江團省委班子首設的掛職副書記職務。這一職務不對應行政級別，也不享受相應的工資待遇。

今年5月5日，浙江省委組織部發布公示：周蘇紅擬任浙江省直單位副廳級領導幹部；7月中旬，周蘇紅正式擔任團省委副書記職務。

此後，浙江共青團網站「領導班子」欄目更新，副書記周蘇紅職務後的「掛職」二字被去掉。

在浙江團省委，周蘇紅主要分管高校，而運動員出身的她，也格外關注高校排球運動的開展。只要有機會，她就會和學生們交流排球。

今年「國慶」前，中國女排用一波10連勝，提前一輪蟬聯2019年第十三屆女排世界盃冠軍。與此同時，她們以「十冠王」的佳績刷新歷史，被譽為「中華民族不懈奮鬥的心靈史詩」。

長安街知事此前介紹過，多位中國女排的功勛隊員，退役後都有了新身份，包括周蘇紅的後輩、2016年里約奧運會奪冠的惠若琪、徐雲麗、魏秋月等，其中——

今年4月13日，惠若琪當選江蘇省青聯副主席。

2018年11月30日，徐雲麗當選為福建省青聯副主席。

2018年1月，魏秋月當選為共青團天津市委副書記（兼職）；8月，魏秋月被聘為天津體育職業學院副院長。
