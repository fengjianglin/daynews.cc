喜憂參半！歐師賽中國三人進正賽三人止步！丁俊暉迎戰磨王艾伯

2019-12-20 116條評論 4,568次閱讀 923人點贊

北京時間12月19日，2020歐洲大師賽資格賽進入了第三天第二輪比賽的爭奪。64位選手將通過本輪角逐32個正賽名額，獲勝者將參加明年的歐洲大師賽正賽。在今天剛結束的幾場比賽中，中國選手喜憂參半。三人闖進正賽三人遺憾出局。

呂昊天5-4李行

呂昊天與李行的這場中國內戰可謂打的相當精彩，兩人目前世界排名呂昊天排名第30位，李行排名第44位。排名相差不大。首局比賽，李行單桿66分以87-47取得開門紅。但是接下來三局比賽，幾乎成了呂昊天的個人表現賽，第三局，他單桿轟下89分一桿致勝，第二局他單桿轟下128分長成功破百，第三局他又打出了單桿62分以103-23拿下。在先丟一局的情況下，呂昊天連贏三局將比分超出。然而，接下來比賽局勢又瞬間逆轉，瞬息萬變。在第5-7三局比賽中，呂昊天失誤增多。李行抓住機會以85-40，73-0以及61-22也連贏三局並且4-3成功拿下賽點。這還沒完，在絕境之下，呂昊天再一次大爆發，關鍵第八局，他單桿打出66分以97-0拿下成功將比賽拖去決勝局，關鍵決勝局，呂昊天再次打出單桿破百以108-10拿下決勝局的勝利。並最終5-4絕殺李行，成功闖進明年的歐洲大師賽正賽。

周躍龍5-2格雷斯

本場比賽，兩人發揮的都不錯。但是相比之下，我國選手周躍龍更勝一籌，經過7局的較量，周躍龍最終5-2戰勝對手強勢晉級正賽。值得一提的是，在贏下的5局比賽當中，周躍龍有三局都是一桿打死，其中轟出兩桿破百和一桿單桿80＋，分別是第三局打出單桿130分，第六局打出單桿116分！完全打出了手感。進攻火力全開。這樣的跳躍龍，值得期待！

肖國棟5-2克萊斯勒

與此同時，肖國棟這邊也傳來喜訊！肖國棟最新世界排名第24位。在中國當今選手當中，實力和人氣都可在排在前五位。而他今天的對手則是百名開在，這場比賽，肖國棟贏得還算輕鬆，首局比賽，他就單桿轟下120分過百，第四局，他打出單桿90分，第七局打出單桿67分。相比之下，克萊斯勒只在第六局到處單桿108分算上本場唯一高光時刻。整場比賽，不論是從防守還是進攻還是穩定性來說，肖國棟都高出對手一籌，晉級也在情理之中。恭喜肖國棟！

有喜就有憂，在為以上四名中國球員的出色發揮高興的同時，也有兩名中國球員遺憾出局，中國00後小將羅弘昊苦戰9局最終4-5遺憾輸給了泰國一哥塔猜亞，其實整場比賽，羅弘昊的表現絲毫不遜色對手，在贏下的四局比賽中，羅弘昊機會都是抓住機會一桿拿下，整場比賽，他打出兩桿單桿60＋一桿70＋以及轟出一桿單桿131分的高分。羅弘昊雖敗猶榮！泰國一哥也值得晉級。另外一場比賽，陳一凡面對頂尖高手凱仁威爾遜1-5慘敗，兩人實力有明顯差距，陳一凡還需努力提升！

總得來說，以上中國選手今天的比賽表現可圈可點，讓人看到了中國斯諾克未來的希望。接下來不過多久，中國斯諾克領軍人物丁俊暉就將再次登場，丁俊暉晉級賽對手是赫赫有名的大磨王艾伯頓，兩人此前有過17次交手，丁俊暉13勝4負佔據絕對優勢。此役再次碰面，丁俊暉面對昔日世錦賽冠軍的磨功，又有怎樣的表現呢？各位球迷怎麼看？丁俊暉能順利晉級進入明年的正賽嗎？歡迎留言！
