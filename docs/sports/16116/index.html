易建聯遭小將“羞辱”，2分鐘後做出強硬回應，真的惹不起

2019-12-13 3,202條評論 4,204次閱讀 1267人點贊

北京時間2019年12月12日晚，CBA常規賽迎來第16輪比賽，浙江稠州銀行客場挑戰廣東東莞銀行，浙江目前排名聯賽第八，而廣東排在聯盟榜首位置，從整體實力來看，兩隊相差頗大，不過此役浙江打得十分具有侵略性，在攻防兩端給到廣東的壓力很大，半場過後分差依然焦灼，其中浙江球員賴俊豪和阿聯的對位備受矚目，賴俊豪面對阿聯打得也是具有攻擊性，在首節更是送給阿聯一個大血帽。而阿聯在隨後更是有點脾氣，不滿判罰大力扔球。

比賽首節，浙江隊打得很具攻擊性，而賴俊豪的表現可圈可點，其中比賽進行到首節中段時，這位95後小將大帽易建聯畫面讓球迷看著熱血沸騰，相信看直播的球迷都會對這一球印象深刻，這一球生生將阿聯的單臂扣籃攔下，攻防轉換後，他又在進攻端頂著阿聯打成一記2+1，阿聯的心態似乎發生了一點變化，2分鐘後阿聯在一次進攻中，強硬上籃直接造成賴俊豪受傷離場。

當時易建聯在籃下要球單打，面對剛剛蓋帽的賴俊豪，阿聯使出一個左迴環轉身，賴俊豪失去防守位置後下手犯規，電光火石之間兩人發生激烈碰撞，阿聯遭受侵犯後高高躍起肘子順勢一揮，賴俊豪痛苦應聲倒地，隨後隊醫扶著賴俊豪回到替補席接受一些簡單理療，好在是硬傷，只是在強烈對抗中產生瞬間暈厥反應，從慢鏡頭來看，阿聯並沒有多餘的動作，屬於連貫的動作，而賴俊豪在休息調整一下後重新回到賽場上。

回到場上的賴俊豪給到阿聯的進攻壓力十分大，阿聯3節過後4/12​僅僅拿到11分，在第三節剩餘2：24時，阿聯因不滿裁判的一次判罰直接將球重重摔在地上，此舉也是被旁邊的裁判警告了一次，裁判直接找到阿聯說了幾句，杜鋒發現不對勁也是馬上將阿聯換下場休息調整了。
