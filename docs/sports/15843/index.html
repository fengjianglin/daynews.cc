147先生出局！3屆世錦賽冠軍轟142分，90後天才轟2桿破百4桿50+

柳號撞球觀察

北京時間12月15日凌晨，2019斯諾克蘇格蘭賽公開賽結束2場半決賽的較量。3屆世界冠軍馬克-塞爾比，與90後天才傑克-利索夫斯基晉級決賽。塞爾比以6-1擊敗147先生大衛-吉爾伯特，利索夫斯基6-5戰勝「小鋼炮」馬克-艾倫。

2場半決賽，對陣分別為塞爾比VS吉爾伯特、利索夫斯基VS艾倫。1/4決賽，塞爾比5-4逆轉「火箭」羅尼-奧沙利文、吉爾伯特5-2戰勝世界第一賈德-特魯姆普，塞爾比與吉爾伯特贏得是兩位奪冠熱門。利索夫斯基5-3戰勝塔猜亞-烏諾、艾倫5-1大勝斯科特-唐納森。

其中，塞爾比與吉爾伯特在淘汰兩位奪冠熱門奧沙利文、特魯姆普之後，已經成為奪冠熱門，他們之間的半決賽也被期待。不過，這場半決賽，吉爾伯特沒有展示出淘汰世界第一時的狀態，塞爾比則在整場比賽中攻防俱佳，6-1大勝晉級。

這場半決賽，塞爾比在進攻方面打出了1桿破百（132分）3桿50+（76分、84分、69分），在防守端壓制了吉爾伯特的進攻，僅僅在第五局打出兩桿50+，要知道，吉爾伯特在淘汰特魯姆普比賽中，打出一桿142分和3桿50+（71分、98分、59分）。

與第一場半決賽沒有任何懸念分出勝負不同，利索夫斯基與「小鋼炮」艾倫的比賽，則是比分交替上升，直到最後時刻決出勝負。利索夫斯基手感火爆，轟出2桿破百（135分、118分）4桿50+（82分、59分、74分、95分）。

利索夫斯基第一局就轟出135分，前五局則是艾倫3-2領先。利索夫斯基1桿破百2桿50+連扳三局，5-3拿到賽點，艾倫連扳兩局再次5-5，決勝局，轟出95分，總比分6-5險勝晉級。即將開始的決賽，兩位狀態都很火爆的球員，誰能贏得最終勝利呢？
