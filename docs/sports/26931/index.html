CBA新格局：粵疆京遼霸佔4強，7隊之間只差1場，季後賽競爭慘烈

2019-12-23 1,666條評論 4,400次閱讀 3454人點贊

北京時間12月22日，CBA第20輪的9場比賽結束，廣東客場大勝廣州，繼續領跑；新疆主場戰勝上海，緊隨其後；山西主場15分之差輸給北京，北京仍舊排名第3，山西跌至第5；山東客場輸給深圳，無緣9連勝，直接跌到第8，遼寧客場大勝福建，升至第4，廣廈主場逆轉吉林，重返前8，第4到第9名球隊的排名發生重大變化。

廣東客場挑戰廣州。廣東此前17勝2負高居榜首，而廣州已經7連敗，排名倒數第三，兩隊實力差距懸殊。廣東首節就取得11分優勢，整場比賽，他們牢牢控制著節奏，最終客場116-90大勝對手，取得3連勝，易建聯拿到21分20籃板的大兩雙數據。

新疆主場迎戰上海。新疆排名第2，上海排名倒數第5，主教練李秋平備受質疑。客場作戰的上海打得非常頑強，上半場大部分時間處於領先，半場領先了3分。新疆在最後一節爆發，最終主場106-97戰勝上海，取得3連勝，周琦只得到11分14籃板，而上海大外援莫泰瘋狂砍下30分28籃板。

山西主場迎戰北京。山西是本賽季的大黑馬，在少賽一場的情況下排名第4，北京排名第3，不過林書豪最近表現低迷。困境之下，林書豪迎來爆發，本場比賽他18投12中，其中三分球5投4中，得到32分，最終北京在客場以15分優勢大勝對手，送給山西本賽季的最慘一敗。

山東客場挑戰深圳。山東此前豪取8連勝，狀態正佳。深圳本賽季狀態不好，8勝11負的戰績僅僅排名第14。山東隊進入狀態較慢，第二節時一度落後了16分，山東隨後發力，一度反超比分，不過最後時刻深圳打出一筆12-0，鎖定勝局，最終104-95戰勝山東，終結了山東的9連勝。

遼寧客場挑戰福建。遼寧近期狀態糟糕，剛剛遭遇2連敗，史蒂芬森、郭士強和郭艾倫都飽受質疑。福建近期狀態爆棚，換帥之後6戰5勝，重新看到季後賽希望。賀天舉首節爆發，單節轟下16分。福建在上半場緊咬比分，半場打成平手。遼寧在下半場逐漸拉開分差，最終客場101-85大勝福建，終結了兩連敗。

其他比賽中，廣廈主場105-98逆轉吉林，四川主場88-119慘敗浙江，八一主場107-102險勝了三外援的同曦，終結了13連敗，天津主場103-127輸給青島。

20輪過後，前三位排名不變，廣東18勝2負繼續領跑，新疆少賽一場，15勝4負排名第2，北京14勝6負仍居第3，不過第4到第9位的球隊排名發生重大變化，遼寧升至第4，山西跌至第5，廣廈升至第6，吉林第7山東跌到第8，北控也跌出前8，位居第9。從第11的青島到第5的山西之間，只差了1場比賽，競爭慘烈。八一拿到本賽季第二勝，不過2勝18負的戰績，仍舊排名倒數第一。
