武磊賽後脫衣謝場，六塊腹肌明顯！留洋漲球又增肌，確實刻苦

2019-12-21 411條評論 2,328次閱讀 343人點贊

北京時間12月20日凌晨，國王杯首輪，西班牙人客場對陣萊里達。上半場，雙方均未能打破僵局；不過，易邊之後，中國球員武磊站了出來，他上演梅開二度，並且終場前差一點戴帽，全場比賽結束，西班牙人憑藉武磊的神級發揮，客場2-0擊敗萊里達。

首先，我們來回顧一下武磊的兩粒進球，雖然上半場雙方互交白卷，但下半場，武磊大爆發，他先是在比賽第53分鐘，弧頂右側接到皮亞蒂直傳，隨即單刀殺入禁區，推射入網，1-0。這粒進球，展現了武磊極佳的無球跑動意識，以及面對單刀球時的冷靜，這與其之前在國家隊屢屢被吐槽浪費單刀大相徑庭，很明顯，武磊的腳下球技術提升了！

而武磊的第二粒進球，更加精彩：比賽第90分鐘，佩德羅薩前場左側傳中，禁區前點的武磊，非常機敏地選擇側身搶點，一記蠍子擺尾攻門，世界級得分，2-0。

就這樣，武磊完成了登陸西甲以來，各項賽事的首次梅開二度好戲，而他也成為第一位在國王杯進球的中國球員。事實上，武磊還有機會在本場比賽中完成帽子戲法，只是運氣稍欠：

當時，比賽進入讀秒階段，第94分鐘，隊友直塞，武磊繞過後衛前插，拿球後直殺禁區，隨即小角度射門，稍微偏出，而跟進的隊友也沒能鏟射空門，武磊遺憾錯失完成帽子戲法的機會，而隊友一旦打進此球，武磊還能增加一次助攻數據，但最終，武磊梅開二度，也已足夠，一個單刀，步點調整很出色，一個外腳背彈射蠍子擺尾破門，靈性十足憑藉一己之力，幫助西班牙人晉級。

賽後，武磊毫無疑問是全場最佳，他脫下球衣，與對手握手致意，並且向遠赴觀戰的西班牙人球迷鼓掌，卸下戰袍之後，武磊露出了非常不錯的肌肉線條，六塊腹肌也是頗為明顯，可以看出，武磊留洋是真的進步了，不僅是球技上，身材保持上也是越發精壯。
