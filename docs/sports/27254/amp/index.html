大羅預測利物浦在明年拿兩冠，預計穆里尼奧的球隊比上賽季有進步

天天要聞
2019-12-21

西班牙時間12月20日，「外星人」羅納爾多參加了一檔世俱杯的活動。在活動的採訪中，羅納爾多預測了利物浦能夠獲得世俱杯的冠軍，並且還對利物浦明年的戰績做出了預測：「我預測明年夏天，利物浦將拿到兩個重要冠軍。」根據分析，羅納爾多口中的重要冠軍，無非就是英超、歐冠以及足總杯了。從利物浦目前的情況來看，他們拿到英超冠軍幾乎是「板上釘釘」的事情，因為他們在英超積分榜上的領先優勢實在是太大了。對於克洛普以及他的球員們來說，與其思考如何奪得本賽季的英超冠軍，不如思考一下他們如何以「創紀錄的姿態」贏得利物浦的首個英超冠軍，這是利物浦明年最穩妥的一個冠軍了。

筆者需要特別說明一下，羅納爾多的預測大部分還是挺準的，並沒有延續他某位巴西前輩的「烏鴉嘴」。在今年歐冠決賽之前，大羅就預測利物浦能夠取勝，雖然他沒有猜准比分，不過從大趨勢上來看，羅納爾多預測勝負的準確率還是很高的。那麼大羅預測利物浦明年的第二個冠軍是什麼呢？相信聽了「外星人」的這些話，利物浦的球迷們是很期待的，因為明年的第二個重要冠軍，不是歐冠就是足總杯冠軍。難道利物浦要蟬聯歐冠冠軍嗎？從利物浦淘汰賽第一輪的抽籤形勢來看，克洛普球隊的簽位還是不錯的，而且淘汰賽第一輪的兩會合，應該不會耗費利物浦過多的體能，當然這些是筆者的個人看法。

從個人的角度來說，筆者當然還是希望利物浦能夠蟬聯歐冠冠軍，正式確立渣叔的一個王朝。當然，利物浦能夠拿到足總杯的冠軍，那也會是一個不錯的結局。接著，羅納爾多就對穆里尼奧開始了點評。12月以來，熱刺是英超除了利物浦以外勝率最高的球隊，其中新上任的穆里尼奧功不可沒，可以說穆帥為熱刺帶來了全新的戰術理念，是他們現在能夠扭轉乾坤、接近歐冠區的關鍵。那麼熱刺在明年夏天會以一個什麼樣的名次結束本賽季呢？羅納爾多沒有給出具體的名次預測，但是他預計熱刺在本賽季結束的時候，其名次會比上個賽季更好：「我預計穆里尼奧的球隊比上賽季有進步。」

上個賽季熱刺是排在聯賽第四，「驚險」晉級了本賽季的歐冠。羅納爾多預計穆帥的球隊會有進步的話，鑒於利物浦在英超的領先優勢實在太大了，那麼熱刺的理想目標是第二或者第三。縱觀現在英超的積分榜，除了利物浦的榜首位置穩定，其實第二名萊斯特城的位置也挺穩定的，領先第三名的曼城挺多分。所以筆者分析，大羅既然預測熱刺能夠比上賽季進步，那麼最有可能的結果就是——穆里尼奧的球隊擠掉瓜迪奧拉的球隊，排在英超積分榜第四位，無疑這是穆帥vs瓜帥的競爭。

有些人可能會說，英超競爭激烈，到了明年夏天一切都不好說。而且英超現在排名前幾的球隊打球隊，除了第二的狐狸城沒有歐洲賽場的事情，其他幾支球隊都要應付多線作戰，到時候他們的體能也會出現問題。而且英超中下游球隊不弱，本賽季英超強隊「栽」在弱旅身上的比賽是有發生。筆者也是同意這個觀點的，不過有一個事實不得不提——英超是本賽季歐洲五大聯賽中，前幾名球隊最早拉開積分差距的聯賽。按照英超的形勢來看，前幾名的球隊估計會差距越拉越大，希望克洛普和穆里尼奧向著羅納爾多預計的方向努力吧！
