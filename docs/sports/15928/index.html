利物浦俱樂部最強之人！白菜價簽下心儀球員，為球隊省下1億英鎊

2019-12-14 195條評論 4,280次閱讀 278人點贊

歐冠小組賽最後一輪強勢晉級，英超豪門利物浦便為球迷再送上一個好消息，球隊以725萬英鎊的白菜價簽下薩爾茨堡紅牛前鋒南野拓實！這則消息剛被報道出來的時候所有的利物浦球迷都十分興奮，畢竟24歲的日本人本賽季在歐冠賽場上的表現有目共睹，在安菲爾德球場出色的表現也給利物浦主帥克洛普留下了深刻的印象，俱樂部能夠如此迅速地簽下這名實力派球員讓所有人拍手叫好！在這一波操作的背後離不開利物浦體育總監愛德華茲的巨大貢獻！

今年40歲的愛德華茲在2016年11月正式成為利物浦俱樂部體育總監，他出色的個人能力讓利物浦在近幾個賽季受益匪淺，眼光毒辣、辦事雷厲風行，為利物浦在轉會市場上一次又一次的買回物超所值的球員！愛德華茲可以說是利物浦俱樂部隱藏的最強之人。從出售庫蒂尼奧再到購買薩拉赫、范迪克、阿利松等人的操作，愛德華茲讓利物浦擁有了一套真正可以成為歐洲頂級球隊的陣容，近幾個賽季利物浦一步步從低谷出走了出來，主帥克洛普居功至偉，但愛德華茲同樣功不可沒！

在歐冠小組賽中表現不俗的南野拓實受到曼聯、AC米蘭等球隊的關注，利物浦主帥克洛普對南野拓實也有著濃厚的興趣，但愛德華茲超強的運營能力讓利物浦搶先一步得到這名物美價廉的球員，白菜價拿下南野拓實為克洛普送上一份禮物。近日根據《利物浦回聲報》消息，球隊簽下南野拓實後，很有可能將英格蘭未來巨星桑喬忘記！眾所周知桑喬本賽季在多特蒙德過得並不開心，利物浦、曼聯等球隊都是桑喬的追逐者！

不得不說愛德華茲又為利物浦俱樂部省下一筆高昂的費用，桑喬如今的身價已經超過1.2億英鎊，而現在利物浦簽下了能夠勝任前場多個位置的南野拓實，毫無疑問日本人的加入會抑制利物浦對桑喬的追逐，從財政方面考慮，南野拓實顯然更有利於俱樂部長遠發展！本賽季南野拓實達目前為止打進8球送出8次助攻，桑喬打進10球並有10次助攻，雖然在數據上遜色於19歲的桑喬，但考慮到南野拓實本賽季在薩爾茨堡隊內更多擔任的是組織核心的角色並非終結者。

愛德華茲無形中為俱樂部省下1億多英鎊，自從愛德華茲勝任利物浦體育總監以來，紅軍在引援方面總是讓球迷眼前一亮，眼光毒辣讓薩拉赫、范迪克等人超常發揮達到個人巔峰狀態，同時迅速的操作讓其他俱樂部望而興嘆！本賽季利物浦在聯賽中表現不俗，克洛普與球隊續約，相信在未來幾年裡，德國人與愛德華茲會讓這支球隊變得更加完美！歡迎在評論區留言討論。
