綜述：德甲半程塵埃落定，萊比錫領跑第一集團

天天要聞
2019-12-22

新華社柏林12月21日電 新華社記者劉暘 距離德甲半程結束還有兩場比賽，但對第一集團座次已構不成影響。領頭羊既不是拜仁慕尼黑，也不是多特蒙德。近十年來，這種情況還是頭一遭。

12月17日，萊比錫隊球員維爾納（右）進球後與隊友慶祝。新華社/美聯

冬歇期前最後一輪德甲聯賽中，萊比錫3:1戰勝奧格斯堡，坐穩了頭把交椅。萊比錫延續了上賽季的生猛態勢，本賽季保持每場比賽都有進球，近來火熱的狀態使他們成為德甲歷史上第一支連續8場比賽每場至少進3球的球隊。主帥納格爾斯曼迎來他執教生涯中最長的連續12場德甲比賽不敗的紀錄。

萊比錫進攻犀利，半程打進聯賽最高的48粒進球。維爾納貢獻了18粒進球，在德甲射手榜上位居第二，僅比榜首萊萬多夫斯基少1個進球。

12月17日，多特蒙德隊球員阿坎吉（中）與萊比錫隊球員維爾納拼搶。新華社/歐新

在過去10個賽季中，有7次半程冠軍最終奪得了賽季冠軍，而萊比錫主帥納格爾斯曼對半程冠軍沒有表現出特別興奮：「這不能證明什麼，聯盟不可能給我們頒發個半程冠軍獎盃。在積分榜上，我們球隊標誌在最前面，可能會給我們帶來一點點心理優勢，但最終我們要有實在的東西抓在手裡。」

萊比錫和拜仁掰手腕，從他們升入德甲的第一個賽季就開始了。本賽季自夏季轉會期起，奧地利紅牛飲料大亨迪特里希·馬特希茨就和拜仁前任俱樂部主席赫內斯暗自較上勁。

萊比錫半程領跑，讓聯賽看上去有了新鮮感。同時有媒體評論指出，目前半程冠軍沒有明顯的實力優勢。自2011-2012賽季後，德甲半程冠軍都是40分以上，而現在萊比錫只有37分。

12月21日，拜仁慕尼黑隊球員穆勒（左）與沃爾夫斯堡隊球員維克托在比賽中拼搶。新華社發（菲利普·魯伊茲攝）

對此，納格爾斯曼反駁道：「我們不要試圖給所有事情下結論。如果拜仁現在領先我們15分，大家肯定覺得這個聯賽很無聊。如果拜仁不領先，又有人說聯賽水平很糟糕。聯賽質量不應該由領頭羊得多少分來衡量，而是用德國球隊在歐戰賽場的表現來衡量。」

本輪賽前與萊比錫積分相同的門興格拉德巴赫，在客場被柏林赫塔0:0逼平，領跑了多輪後，以第二名（35分）結束半程。本賽季在主教練馬爾科·羅澤帶領下，球隊保持高昂鬥志，收穫聯賽第四多的進球。門興在他們贏得德甲冠軍的5個賽季中，有4次都奪得半程冠軍。目前戰績讓很多「小馬駒」的球迷看到這支老牌勁旅重塑輝煌的曙光。

衛冕冠軍拜仁落後門興2分，位居第三。儘管本賽季拜仁經歷很多波折，包括科瓦奇下課，赫內斯退休，聚勒、科曼、埃爾南德斯、格雷茨卡、托利索等一批大將受傷，但強大的板凳深度還是讓他們在歐冠賽場取得小組全勝、滿分晉級的成績。目前聯賽排名也極有可能在下半程得到提升，沒有人低估拜仁的反撲能力，他們依然是本賽季最有實力的冠軍爭奪者。

12月14日，拜仁慕尼黑隊球員慶祝球隊進球。新華社/歐新

拜仁的國家德比宿敵多特本賽季狀態和戰績都不樂觀。雖然多特斥巨資打造陣容，但目前看效果並不理想。球隊以8勝6平3負的戰績，距離榜首有7分分差，這個成績單遠不能令人滿意。作為2012年德甲冠軍，多特本賽季似乎距離冠軍越來越遠。

第16輪主場迎戰萊比錫的焦點戰中，下半時1分鐘內讓對手連偷兩球，到手的勝利變成平局；第17輪客場挑戰霍芬海姆比賽中，多特最後時刻丟球失利。同期拜仁都取得勝利，多特將領先座次拱手讓出，被拜仁反超。現年62歲的瑞士主帥法夫爾面臨下課危機。

12月17日，多特蒙德隊主帥法夫爾在場邊指揮。新華社/歐新

德國前國家隊門將卡恩在電視節目中評論道：「多特對失利的恐懼遠遠多於對勝利的信心，這樣下去他們不會奪冠。他們有豪華陣容，有經驗豐富的球員，可以在令人窒息的節奏中打出令人陶醉的進攻，但就是不能在90分鐘內完全掌控比賽。想要奪冠，他們必須表現得更好，而且要有持續性。」

多特俱樂部競技主管佐爾克承認：「從近兩輪糟糕的比賽來看，我們還遠不能討論奪冠的事兒。這兩輪5個積分從手邊溜走，我們得找找原因。」

多特的魯爾德比勁敵沙爾克則擺脫了上賽季的糟糕狀態。66歲的主教練胡布·史蒂文斯上賽季臨危受命，將球隊從沒落邊緣拯救出來，艱難地挺過過渡期，本賽季主帥大衛·瓦格納將球隊重新帶回第一集團競爭中。和多特相比，沙爾克投資要少得多，但性價比要高得多，目前與多特同積30分，因凈勝球劣勢排在第五名。

12月21日，沙爾克04隊球員在比賽中慶祝進球。新華社發（烏爾里希·胡夫納格爾攝）

本賽季給人帶來最多驚喜的當屬柏林聯合，上賽季靠升降級附加賽首次升入德甲聯賽的柏林聯合在三支升班馬球隊中目前戰績最出色，以20分位居第11名。他們曾在客場險些逼平拜仁，在主場擊敗多特以及當時的聯賽領頭羊門興，令人刮目相看。

在降級區的三支球隊分別是雲達不來梅、杜塞爾多夫和帕德博恩。不來梅本賽季遭遇人員變動和傷病影響，成績較上賽季大幅下滑。升班馬帕德博恩目前只贏了2場比賽，積9分穩坐副班長位置，一副「將牢底坐穿」的態勢。最後兩場比賽即使杜塞和帕德博恩都贏球，仍然不能逃離降級區。
