僅用一天就完成觸底反彈，這樣的林書豪該不該被首鋼換掉？

2019-12-23 1,531條評論 3,588次閱讀 60人點贊

在山西男籃的家門口，林書豪一掃之前的低迷和陰霾，用一場爆發幫助北京隊贏得了寶貴的勝利。

上一場主場對陣浙江的比賽，是林書豪加盟北京隊之後，打出了最為糟糕的比賽之一。那一戰，主場作戰的北京隊從一開始就被對手牢牢壓制，全場比賽幾乎沒有看到任何逆轉的機會。作為球隊進攻核心的林書豪，雖然得分最終超過了20，但卻付出了16次打鐵的代價。

單場比賽投丟16個球，也創造了本賽季北京首鋼球員個人單場的打鐵紀錄。那場比賽里的書豪，就像被魔咒封印了一樣，毫無進攻感覺。無論是外線的三分球還是平日里拿手的中距離跳投，完全失去了應有的水準。

也就是在那一場比賽之後，外界對於林書豪的質疑達到了一個新的頂峰。

最近一段時間以來，關於林書豪的爭論始終不絕於耳。書豪曾經在社交媒體上表示，不希望在比賽當中被一些粗野、危險的防守動作傷害。在那之後，關於林書豪是否應該被「重點保護」、林書豪是否應該享有特權等等話題，網路上爭執不休。可偏偏在輿論風暴的焦點中央，林書豪打出了一場糟糕的比賽。

當比賽進入到下半場，北京隊已經翻盤無望的時候，五棵松的觀眾席上甚至響起了「換林書豪！」、「上大漢！」這樣的呼聲。可對於林書豪來說，他只能努力屏除這些外界的干擾。

林書豪說：「我在賽季的時候，從來不上網看那些評論，不管是球迷還是記者。我為我的球隊而戰，我會專註做好最好的自己，除此之外不會理會外面的話。我知道自己的角色是什麼，知道球隊需要我做什麼。在紐約的時候也是一樣，我經歷過有很多人支持我，但很快又開始罵我。我不會受這些影響，用全力去進步。我們要把進攻打好，我是先發控球，需要時間想想如何去調整。聯賽到現在，我們能打到第三，代表我們並不是全不好，只是這場打得不好。」

競技體育的世界裡就是這樣，沒有人會聽你解釋，勝利者贏得讚美，失敗者面對罵聲，從來都是如此。

我記得結束了新聞發布會，回到更衣室之後的林書豪，又被問到了關於自己改如何調整的問題。可是書豪的回答令人有些意外，他說：「其實我不會去刻意調整什麼，還是要保持一個平和的心態。今天我沒有打好，如果問我會不會後悔一些出手，我會說不後悔，有好機會卻沒投進，我回去要去總結，不怪別人，只說自己。」

其實來到CBA這幾個月的時間裡，林書豪已經太多次被問到過關於如何調整的問題，而他嘴裡出現最多的回答永遠是：「我就是打自己的籃球就好……」

面對山西隊，林書豪在一開場就堅決地執行個人進攻，他的手感遠比上一場出色得多。在書豪這個點進攻完全打開之後，山西隊防守北京隊的擋拆時始終顧此失彼，最終也沒有扭轉頹勢。32分外加6次助攻和6個籃板，是遭遇低谷後的林書豪，觸底反彈的成績單。

現在，讓我們再回到最開始的討論，現在的北京隊，應不應該考慮換掉林書豪？

林書豪已經不再年輕，而他的運動天賦、對抗能力都難以和很多美國黑人外援相比。所以我們經常會看到，當面對對手強硬的防守時，林書豪可能會遇到比較大的麻煩。而現在，當常規賽已經接近半程的時候，各支球隊也相繼進入了一個比較疲勞的時期，林書豪也並不例外。

很多人認為，對於內線相對薄弱的北京隊來說，如果啟用「漢密爾頓+尤度」的雙塔外援組合，這兩人一個擅長進攻，一個精於防守，長期困擾北京隊的內線問題將立刻迎刃而解。同時，北京隊外線的本土球員配置，放在整個CBA也屬於上乘。這樣一來，雖然外線會受到一定程度的削弱，但內線的補強效果則非常明顯。

說實話，這樣的想法的確有一定道理。可是對於如今戰績穩定且排名靠前的北京隊來說，還並沒有到非換外援不可的地步。

首先如果換掉林書豪，北京隊的票房將會受到明顯的影響。現如今書豪是北京隊乃至整個CBA的招牌，是聯賽里影響力最大的球員之一。很多球迷買票來五棵松，就是為了現場看林書豪打球。對於一家職業俱樂部來說，成績當然非常重要，但市場價值和票房，也是不容忽視的關鍵因素。

另外，經過了將近半個賽季之後，大家對於林書豪的優勢和劣勢，都已經有了較為清晰的認識。林書豪打出過很多精彩的比賽，也確實存在一些劣勢和不足，但總體看來，北京隊本賽季的表現仍然處在一個相對平穩的階段。貿然更換外援，也許會對球員的情緒產生一定的影響。

當然，這並不是說北京隊不能做任何調整。只是在我看來，現階段並不是一個合適的時機。倘若隨著時間的推移，書豪的身體和狀態出現波動，北京隊的成績出現起伏，屆時再考慮進行外援的調整和未嘗不可。

這樣一來可以考察，兩名大外援的搭檔是否真的適合北京隊。漢密爾頓和尤度都是特點鮮明的優秀外援，但二人一起只打過一場季前賽，還沒有在正式比賽里接受檢驗。另外也可以讓林書豪得到一定程度的休息和緩解。因為即使是調整外援，也只是暫時的過程，漫長的賽季里，北京隊無疑還需要林書豪做出更多的貢獻。

競技體育的賽場上，永遠靠表現說話。但也不能總是「一場論」，一場糟糕的比賽，並不意味著林書豪不行；打出一場好球，也並不意味著林書豪就有多完美。

客觀全面地看待一個運動員的特點和不足，也能夠讓決策者們在做出決定的時候，多一分謹慎和耐心。
