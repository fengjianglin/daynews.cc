本賽季躍居聯盟第四的黑馬為何上賽季卻是一支默默無聞的球隊呢？

2019-12-03 0條評論 23次閱讀 0人點贊

摘要

聯賽進行了12輪，上賽季止步季後賽的山西男籃一躍成為了目前的四強選手，甚至排在了遼寧男籃的前面，在死亡之組的C組能夠打出這樣的戰績實屬不易，估計很多球迷都在疑問，這樣一直黑馬成色十足的球隊為什麼上賽季默默無聞呢？

聯賽進行了12輪，上賽季止步季後賽的山西男籃一躍成為了目前的四強選手，甚至排在了遼寧男籃的前面，在死亡之組的C組能夠打出這樣的戰績實屬不易，估計很多球迷都在疑問，這樣一直黑馬成色十足的球隊為什麼上賽季默默無聞呢？

論一名優質外援的作用

一個優質而且符合球隊打法的外援對於一支球隊來說是至關重要的！強如廣東男籃，也是選到了馬尚布魯克斯和威姆斯這兩個外援才能拿下第九冠！而之前幾個賽季因為外援的問題也是一直倒在爭冠的道路上，浪費了易建聯多年時光！回到山西男籃，我們可以用數據看見一個優秀外援所帶來的影響！

首先以原帥為例，原帥上賽季場均不到6分，儘管三分球命中率在聯盟平均水準以上，可是無論是亞當斯，還是沙巴茲還是後來的布朗，都沒能很好的激活原帥，原因在於亞當斯和沙巴茲屬於單打獨鬥型的外援，個人單打能力不錯，也能夠撕裂對面的防守，可是組織串聯能力太差，能夠自己得分而不能為隊友輸送炮彈，而後的布朗組織能力不錯，但是年事已高，無法突破內線分球，對於定點投射頂級的原帥來說，這樣的傳球是無法轉換成得分的！而本賽季富蘭克林的到來恰好滿足了以上幾點，他能夠在隊友被限制的時候通過自己的個人能力單打取分，也能夠在對手夾擊自己的時候及時將球輸送到空位隊友手中，這樣一來，以原帥為代表的山西國內射手群得到了大量的空位三分機會，當然隊友通過一個夏天的辛苦努力三分能力也得到了加強，能夠很順利的投進三分，因此山西也成為目前聯賽中三分命中率第一的球隊，這是相輔相成的。富蘭克林可以通過傳球破解對方的夾擊，也節省了自己的體力，隊友接到他的助攻能夠得分，球隊也能獲得勝利。因此原帥目前場均可以扔進四個三分，能夠得到17分左右。在看山西男籃國內的其他球員，閆鵬飛，任駿威，郭曉鵬，劉冠岑等主力的三分命中率都在四成以上，這也使得山西男籃目前在聯賽中排在第四的位置。

所以說一個優質的外援，不僅可以帶動國內球員的得分，而且可以使球隊的成績更上一層樓。

論一位冠軍教頭的能力

上賽季匆忙接手山西男籃後，並沒有給這位成名已久的冠軍教頭太多的時間去改變這支球隊，隊內的頭號得分手與國內助攻王集一身的曾令旭遠走新疆，而隊內僅剩兩位大學生球員王洪和班鐸，這兩位球員儘管是CUBA的天之驕子，但是來到CBA，受限於身高，並沒有打出讓人期望的成績。因此，王非指導在自由市場上引入任鵬鵬和孫偉博，從結果來看，這兩筆引援失敗的很徹底。

外援的選擇和國內引援一樣難產，簽下的布克開戰前夕溜了，匆忙簽下的麥卡洛居然不如國內球員，看著廣東男籃雙小外玩的風生水起，於是乎王非指導乾脆也打起了雙小外，簽下了曾經的美國第一高中生穆罕默德，這位兄台打法實在勁爆，可是不會傳球啊！小白人外援亞當斯也一樣是個會進攻不會傳球的主，於是兩人經常能貢獻出賽季十佳的扣籃，可球隊戰績始終不見起色，無奈找了名在CBA摸爬滾打多年的外援布朗，儘管有些組織能力卻攻擊力不足！

匆匆忙忙，48輪的聯賽根本不會給你再多的時間去調整外援與國內球員的磨合，慘淡的一個賽季就這樣過去了！面對來自球隊內部、網路上紛至沓來的質疑，王非指導並沒有作出過的解釋，而是在休賽期早早地開始了集訓，打磨這支球隊，儘管沒引進強力內援，但為期兩個月的國外訓練讓這批國內球員的基本功、體能有了進一步的提升！

直接的結果就是，開賽至今，國內五名球員三分命中率上40，原帥、任駿威、閆鵬飛等人每場都有穩定的發揮，籃板聯盟第六，助攻、蓋帽聯盟第二，失誤聯盟倒數第三，三分命中率聯盟第一，過去被球迷戲稱不會防守的跑轟大隊，如今防守效率已經是聯盟第七…….經過一個賽季的調教，這個山西男籃已經在王非指導的掌舵下平穩前行！

論一群球員努力的結果

山西男籃這批球員其實成名很早，大寶、閆鵬飛、田桂森等人出道不久就成為了國奧男籃的主力，原帥是與周琦、趙繼偉同期的神射手，在那屆世青會被譽為中國的雷阿倫，郭曉鵬2011年隨隊獲得亞洲青年男籃錦標賽冠軍，等等！尤其是山西男籃本土培養的這批球員，王非指導初到山西男籃時對他們的評價是他們很有天賦，只是被一些情況給耽誤了。什麼情況呢？是山西男籃管理層的急功近利，數年間竟然換了有近二十名教練，一個教練有一個教練的執教理念，有句土話說的好：貪多嚼不爛。這些球員接受了這麼多的籃球理念，他們真的能接受嗎？答案肯定是不能的，所以說在他們的成長過程中，並沒有一名教練從一而終的指導，因此王非指導來的第一件事就是挖掘他們的天賦！

休賽期，王非指導狠抓基本功，從球員和教練的社交媒體上可以看到，小到一個籃下動作都要仔仔細細的完成，並且全隊前往海外特訓兩個月，這在整個CBA都是罕見的。好在這批球員的努力沒有白費，目前的戰績就是他們努力最好的結果！

賽季不過過去四分之一，C組廝殺的慘烈前所未見，正如王非指導所說：我們不是個強隊，所以每場都要拼下去，山西男籃要保持本心，努力的創造一個奇蹟！

我們是未名之人來自無名之處！
