西甲-武磊替補造點被VAR抹殺 西班牙人0-2輸掉保級大戰 繼續墊底

天天要聞
2019-12-22

北京時間12月22日，2019-2020賽季西甲聯賽第18輪繼續進行，萊加內斯主場2-0戰勝西班牙人，連續三輪不敗。武磊替補出場製造點球，但被吹罰越位在先。此戰之後西班牙人繼續在西甲積分榜墊底。

第6分鐘，萊加內斯發動進攻，佩雷斯嘗試一腳遠射高出。第12分鐘，西班牙人後衛馬克羅解圍失誤，奧斯卡得球送出直塞，布萊斯維特突入禁區射門得手，萊加內斯首開紀錄。

第14分鐘，萊加內斯反擊，布萊斯維特右路突破後射門被門將沒收。第22分鐘，格拉內羅禁區外傳中，迪達克跟進被後衛解圍。第26分鐘，奧斯卡主罰自由球直接射門被人牆擋出。第32分鐘，達德爾中場附近遠射偏出。

第36分鐘卡萊羅禁區內放倒恩內斯里，裁判看過Var後沒有表示。第40分鐘，佩雷斯中路直接射門被門將沒收。第43分鐘，布萊斯維特禁區線附近直塞，恩內斯里射門偏出。

第49分鐘，卡萊里長傳，坎普薩諾禁區內凌空射門偏出。第51分鐘，布拉斯維特禁區弧頂附近直接遠射被門將沒收。第54分鐘，布萊斯維斯分球，羅德里格斯左路倒三角傳中，恩內斯里包抄破門，萊加內斯2-0擴大比分。

第66分鐘，坎普薩諾右路內切稍作調整後遠射高出橫樑。第70分鐘，達德爾主罰直接自由球傳向禁區，羅拉頭槌被門將沒收。第78分鐘，武磊替補登場。

第88分鐘，卡萊里射門被撲，武磊補射被放倒，但因為越位在先沒有被判點球。

比賽結束，萊加內斯2-0西班牙人。西班牙人輸球後以10分繼續排在西甲積分榜最後一位。

萊加內斯首發：1-奎利亞爾、4-奧梅羅、5-霍納坦-席爾瓦、6-梅薩、7-布萊斯維特、16-羅薩萊斯、12-阿瓦澤姆、21-魯文-佩雷斯、24-K-羅德里格斯、26-恩-內斯里、27-奧斯卡-羅德里格斯

西班牙人首發：13-迭戈-洛佩斯、15-戴維-洛佩斯、20-貝爾納多-埃斯皮諾薩、24-卡萊羅、34-維克托-戈麥斯、10-賽爾希-達德爾、23-格拉內羅、21-羅卡、17-迪達克、31-坎普薩諾、12-卡萊里
