國羽捷報頻傳！17歲天才少女橫空出世，獨攬兩金成最大贏家

2019-12-22 427條評論 4,436次閱讀 3018人點贊

北京時間12月21日，2019全國U17-18羽毛球總決賽落幕，來自廣州的小將李怡婧獨攬兩冠成為U17年齡段的最大贏家。

本次總決賽，李怡婧雙線作戰，與羅徐敏搭檔出戰女雙，與陳文傑搭檔出戰混雙。面對密集的賽程壓力，李怡婧展現了驚人的連續作戰能力以及強出其他同齡運動員一籌的實力，最終拿下了女雙、混雙兩個冠軍，成為U17年齡段女線最閃耀的新星。

2002年出生的李怡婧，如今也只有17歲，是國羽在女子雙打項目重點培養的又一位天才少女。李怡婧的天賦有多出眾呢？來自廣州的她在還沒有參加過全國青年錦標賽的情況下就憑藉著在省內比賽的優異表現得到國青羽毛球隊教練組的青睞，直接被提拔進隊。不僅天賦了得，李怡婧的訓練還非常刻苦，這也讓她的上升速度非常驚人。在今年，李怡婧代表中國隊出戰了世界青年羽毛球錦標賽，在女雙、混雙兩個項目均闖進了四強。雖然折戟半決賽，但17歲的李怡婧仍然被專家和球迷一致看好，有望在未來成為像高崚、趙芸蕾那樣的國羽領軍人物。

這一次的總決賽，對於獨攬兩金的李怡婧而言不過是小試牛刀，她的舞台在更廣闊的國際賽場上。
