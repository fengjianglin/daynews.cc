曼聯倆球員已被告知下個月離開老特拉福德！

2019-12-24 1,737條評論 4,195次閱讀 2325人點贊

據外媒報道曼聯兩名球員已經被被告知下個月可以離開俱樂部，由於狀態下滑表現不佳，這兩名球員已經所在位置已經出現人員過剩，曼聯CEO伍德沃德準備在1月份的轉會窗口將他們擺上貨架，曼聯主帥索爾斯克亞已經告訴這倆球員，他們在俱樂部都沒有長遠的未來。

挪威人在12個月前取代了穆里尼奧之後，馬蒂奇首發位置就被年輕球員麥克托米內取代了，他對索爾斯克亞執教也不怎麼滿意。據了解至少有兩家英超俱樂部對馬蒂奇感興趣。

2017年夏天，穆里尼奧以4000萬英鎊的價格將馬蒂奇從切爾西帶到老特拉福德球場，並且和葡萄牙人關係密切。他一炮走紅和博格巴一起鞏固了他在曼聯中場的位置，但隨著去年12月穆里尼奧的下課，這位31歲的球員在索爾斯克執教下影響力出場機會越來越少。儘管曼聯在去年夏天賣掉了埃雷拉和費萊尼，但本賽季馬蒂奇只參加了兩場聯賽，並且在所有比賽中只出場7次，雖然他與索爾斯克亞進行攤牌談判，但塞爾維亞人處境並沒有改變。

自從穆里尼奧離開後，另一位前本菲卡球星感到越來越孤立，他知道他需要一個新的挑戰，因為他的職業生涯接近尾聲。這位將滿30歲的後衛是2014年范加爾執教的時候從里斯本引入，當時花費了1600萬英鎊，但羅霍從來沒有老特拉福德中站穩腳跟。

在索帥陣容中這名左後衛的排名已經落後於盧克·肖和布蘭登·威廉姆斯，雖然上賽季轉會埃弗頓的計劃落空後，但他還是非常渴望離開曼聯，他知道想延長自己在阿根廷國家隊的職業生涯，就必須換個環境經常參加一線隊的比賽。
