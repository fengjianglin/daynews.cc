瘋狂！9145萬升班馬殺進英超前5：創造19年神跡 曼聯熱刺也要仰望

2019-12-24 1,759條評論 4,404次閱讀 944人點贊

北京時間12月23日，權威機構OPTA貼出一則有趣的數據：身價排名英倒數第1的謝菲爾德聯在聖誕賽程到來之前，高居積分榜第5位，把熱刺，曼聯和阿森納都甩在身後，追平了伊普斯威奇00/01賽季創造的紀錄，締造升班馬奇蹟。

謝菲爾德聯是本賽季的英超升班馬，一線隊總身價只有9145萬歐元，莫說跟曼城，利物浦，熱刺以及曼聯這些頂級豪門相比，甚至都比不過同為升班馬的諾維奇（1.2億）和阿斯頓維拉（1.3億），在整個英超當中排名倒數第1！

然而，足球比賽並不是單純的身價比拼，52歲的主帥懷爾德為謝菲爾德聯打造出了極具戰鬥力的體系。依靠嚴密的防守，高效的反擊和定位球戰術，謝菲爾德聯變成了一支鐵軍。

謝菲爾德聯不僅能戰勝弱隊，對陣強隊也毫不手軟。他們1-1逼平熱刺，3-3絕平曼聯，1-0戰勝阿森納，面對利物浦和萊斯特城也不過是1球小負。英超強隊紛紛在他們面前爆出冷門，即便要贏球也得脫層皮。

北京時間12月21日，謝菲爾德聯客場挑戰布萊頓，憑藉麥克布涅第23分鐘的進球1-0笑到最後，收穫英超3連勝，他們的排名也來到了第5，穆里尼奧的熱刺，索爾斯克亞的曼聯，動蕩中的阿森納均排在他們身後。

權威機構OPTA數據顯示，謝菲爾德聯是近19年來，戰績最出色的英超升班馬，上一次有球隊如此瘋狂，還要追溯到2000/2001賽季的伊普斯威奇。作為本賽季英超身價最低的球隊，謝菲爾德聯能夠挺進前5，將3大傳統豪門甩在身後，確實足夠瘋狂。

英超似乎總是擅長製造黑馬神話：15/16賽季，萊斯特城異軍突起，奇蹟般奪得英超冠軍，18/19賽季的狼隊，作為升班馬拿到第7，排在曼城，利物浦，切爾西，熱刺，阿森納和曼聯之後。

本賽季，身價倒數第1的謝菲爾德聯低調崛起，目前排名英超第5，他們會不會創造新的神話？我們拭目以待。
