單賽季2000分難度有多大？倫納德0次，威少1次，一神人做到10次

2019-12-16 423條評論 1,782次閱讀 413人點贊

在進入小球時代之後，聯盟中比賽整體的得分數都得到了一定的提升，而這也讓許多得分數的數據也都有了一定的增長。而對於這些在聯盟中有著上佳表現的得分手來說，他們狀態火熱時，在單場比賽中砍下30+，甚至40+的得分也不是什麼難事，但如果想要將這種狀態保持一整個賽季，就是一件十分困難的事情了。更不要說單賽季砍下2000分了，這要求該球員不僅要持續保持在一個火熱的得分狀態中，還要保證身體的健康狀況，不能因為傷病而缺席太多比賽。

單賽季2000分就意味著在82場常規賽中，場均得分要至少為24. 4分，這絕對不是一件容易的事情，那麼我們就來看看現役巨星們都分別有幾次拿到過單賽季2000+的得分。

倫納德作為近幾個賽季聯盟中最炙手可熱的巨星，他上賽季在猛龍成功拿到了總冠軍，而自己也榮膺了FMVP，而他在職業生涯前期的得分能力還不夠突出，因此倫納德至今為止得分最高的一個賽季也只有1888分，那是在16/17賽季，他在常規賽中出場了74次，場均得分為25. 5分。

目前聯盟中的三雙之王威少已經兩次完成了單賽季2000分的成就，他一直都是一名得分能力十分出色的雙能控衛球員，他在2016年就以場均31.6分的數據拿到了賽季得分王，而且他還在職業生涯中連續三個賽季拿到了三雙。

再來看看今夏加盟湖人，與詹姆斯組成雙巨頭的安東尼.戴維斯，他是在2012年進入聯盟的，至今已經兩次拿到完成單賽季2000+的成就了，分別是在2018賽季和2019賽季，最高場均得分為28. 1分。

甜瓜安東尼目前也已經兩次在單賽季斬獲2000分了，而且還曾在12/13賽季以場均28. 7分的數據拿下了得分王。另外，杜蘭特和哈登這兩位得分狂魔都已經六次完成這一成就了，而現役球員中，最多完成單賽季2000分的依舊還是寶刀不老的詹姆斯，他已經十次完成這一成就，是現役球員中最多的，而且已經連續七個賽季總得分突破2000分了。
