郝海東：01年預選賽時我就打包票，國足能進世界盃

2019-12-22 440條評論 1,975次閱讀 462人點贊

虎撲12月21日訊 前國腳郝海東近日在卡達出席活動時，回顧了當年代表國足出戰世預賽的歷程，並談到中國和卡達之間足球發展的差距。

如果任何一個球隊，他主場球隊還不如客場，那怎麼可能出線？

「97年十強賽的一切都歷歷在目，確實當時是整個中國足球最有特點的一批。如果任何一個球隊，他主場戰績還不如客場，那怎麼可能出線？」

「但是在97十強賽的時候，哪怕都成這種情況，我們的客場成績，如果拿到參加世界盃的強隊身上，包括亞洲最強隊上面，比如日本、伊朗這些，如果他的主場成績像我們，客場成績也像向我們，那早就被淘汰了。他們客場的成績可能不見得能拿到中國隊，哪怕是97年那時候的分數。」

「(97年)我們講實話在客場很好，贏科威特、平卡達、平沙特，只是客場輸了伊朗，但問題是我們在主場掉了很多分，客場分數沒問題，兩場都有可能贏沙特。這就是當時發生的這一切告訴我的東西。」

01年預選賽的時候，我就和足協說，這次不出線都不行，他們都不信！

「01年預選賽的時候，我就和足協說，這次不出線都不行，他們都不信！因為我們經歷過、我們打過。職業化發展以來，經歷了無數個外教，積累到今天，跟亞洲所有的強隊都打過，分組情況明了的時候，(就知道)我們一定會贏！」

「我們的主場成績沒問題，客場成績很好，只輸了一場，但是沒關係，提前兩輪出線。在世界盃預選賽十強賽這種比賽上，如果能提前兩輪出線，這得什麼情況才能達到！這不是運氣，不是抽籤，不可能。我們所有的認知、邏輯、成長環境，如果不認知到這些，永遠贏不了。」

卡達今天的足球水平，是通過20年努力得到的

「我一直在講，他們都是拿一個錯誤糾正另一個錯誤，烙烙餅。三五年換一撥人上來，依舊因為他們沒打過、沒經歷過、不懂。只是他們以為的以為，他們不知道，卡達的今天(足球水平)，早在二十年前就開始謀劃布局了，而我們在做什麼？這就是差距。正真的改變，不要說頭槌練少了、體能不行(這種借口)，要從根本上改變。」

(編輯：姚凡)
