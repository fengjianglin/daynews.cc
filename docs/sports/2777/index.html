留在巴黎還是轉會意甲？庫爾扎瓦將面臨選擇

2019-12-04 0條評論 25次閱讀 0人點贊

摘要

對於27歲的庫爾扎瓦來說，他正處在職業生涯的十字路口：這位前法國國腳與巴黎聖日耳曼的合同將於明年到期，本賽季他代表法甲冠軍在聯賽中出場8次，是巴黎在左後衛位置上的第二順位人選。

對於27歲的庫爾扎瓦來說，他正處在職業生涯的十字路口：這位前法國國腳與巴黎聖日耳曼的合同將於明年到期，本賽季他代表法甲冠軍在聯賽中出場8次，是巴黎在左後衛位置上的第二順位人選。諸如尤文圖斯，國際米蘭和AC米蘭這樣的意甲豪門已經向他拋出橄欖枝，而庫爾扎瓦也將在冬季轉會窗後做出自己的決定。

巴黎官網上庫爾扎瓦的圖片

巴黎生涯：起起伏伏的「帶刀後衛」

2015年夏天，庫爾扎瓦以2500萬歐元的價格從摩納哥轉投巴黎聖日耳曼。不知不覺，法國國腳已經在巴黎度過了4年時光，在代表巴黎出戰的108場比賽中，庫爾扎瓦打進12球助攻19次，他幫助球隊獲得了3次法甲冠軍。

Whoscored：來到巴黎之後庫爾扎瓦各個賽季的數據

由於傷病，競爭和狀態起伏等原因，庫爾扎瓦一直沒有坐穩巴黎的主力左後衛位置，不過他也一直是球隊的重要成員。每個賽季，庫爾扎瓦都能穩定的貢獻一定數量的進球和助攻，2017年10月31日，庫爾扎瓦更是讓自己的名字登上了頭版頭條，在對陣凱爾特人的歐冠比賽中，他頭頂腳踢完成帽子戲法，創造了歐冠的歷史。庫爾扎瓦的進攻能力一直是他的最大優勢，法國後衛不僅能利用個人能力完成突破，配合能力也相當出眾。在比賽里，他的傳中，小範圍配合甚至遠射能力都能給球隊帶來幫助。

傷病過後的競技狀態：有所下滑但基本功仍在

連續的小傷病讓庫爾扎瓦的身體素質受到了不小影響，現在他的爆發力、速度都不如以前。庫爾扎瓦是一位攻強守弱的邊後衛，雖然速度和爆發力下滑，但庫爾扎瓦依然有著不錯的進攻端配合能力，擺脫小技術（連續的小範圍變向和挑球）和終結能力（每個賽季都能貢獻穩定的進球和助攻數據），善於大力抽射，助攻方式也較為多樣，善於肋部小範圍配合，能低平球傳中也能弧線傳球掃到中路的他依然是球隊進攻端的重要一員。

庫爾扎瓦頭槌得分，他的爭頂能力很不錯（Ligue 1官方YouTube）

庫爾扎瓦助攻姆巴佩得分（Ligue 1官方YouTube）

雖然傷病有所影響，但庫爾扎瓦邊路二過一，主導小配合的能力依然在，在壓制局面下，他的進攻底子依然能起到不錯的作用，傳中方式較為多樣，有著不錯拿球基本功與小技術的庫爾扎瓦不少時候在場上（特別是球隊處在壓制對手的局面下）依然能發揮不錯的進攻端作用。

庫爾扎瓦搶斷後助攻姆巴佩得分（Ligue 1官方YouTube）

相對來說，庫爾扎瓦的防守只能算平平，身體素質處於巔峰狀態的時候，庫爾扎瓦比較依靠自己的速度完成回追，而在身體素質下滑之後，他的到位率和防守能力也下滑了，作為左後衛他的防守意識和位置感都較為一般，容易出現顧此失彼的情況，庫爾扎瓦在防守端的特色是善於利用腿長攔截對手，在比賽中他的正面搶斷和攔截球路能力都不錯，也善於爭奪五五開球，總體來說他的防守能力一般。

Whoscored對庫爾扎瓦技術特點的評價

總體來說，傷病的連續影響讓庫爾扎瓦的身體素質下滑了不少，現階段他依然是一位攻強於守的左邊衛，他的進攻能力依然不錯，防守能力則要遜色不少。在球隊呈壓制對手的局面下，庫爾扎瓦的進攻基本功依然能帶來很大幫助，但在球隊處於下風的時候，庫爾扎瓦在防守端的弊病也容易被對方抓住。

轉會傳聞：下一站意甲？

庫爾扎瓦和巴黎的合同將於2020年到期，按照目前的情況，庫爾扎瓦和巴黎續約的可能性較小。目前巴黎的主力左後衛位置屬於貝爾納特，庫爾扎瓦也極有可能在不久的將來離開王子公園球場，今年1月或者夏天，他將有可能轉投其他球隊。

對於庫爾扎瓦而言，接下來他也將做出選擇，是留在巴黎擔任替補（輪換）還是離開法甲聯賽？據多家媒體報道，庫爾扎瓦的下一站很可能是意甲聯賽，急需左邊翼衛的孔蒂將庫爾扎瓦視作了國際米蘭冬季轉會窗的引援目標之一，而需要補充邊後衛位置厚度的尤文圖斯也可能和巴黎進行轉會互換的交易，並將法國國腳加入交易當中。此外，AC米蘭也對27歲的法國人很感興趣。對於庫爾扎瓦而言，他也需要在不久的將來做出續約或者離隊選擇。
