梅西很忙：西媒聲稱巴薩要與梅西續約，必須承諾簽下四名主力球員

2019-12-25 3,891條評論 4,872次閱讀 2326人點贊

據西班牙媒體Diario GOL報道，梅西有望與巴塞羅那俱樂部達成續約，但要求俱樂部在續約前承諾簽下四名球員。

梅西這個賽季對巴薩來說也是至關重要的，他的數據也反映了這一點。他是西甲的頭號射手，有13個進球，甚至為隊友貢獻了不少助攻。巴薩俱樂部希望儘快和梅西續約。

據報道，梅西的父親豪爾赫已經和巴薩董事會，特別是俱樂部主席巴托梅進行了會談。看起來薪資方面的所有條款和條件都已經達成一致，但Diario GOL聲稱，梅西方面提出一個要求，只有在巴薩承諾簽約四名球員時，梅西才會正式簽署新合同。

中後衛烏姆蒂蒂幾乎肯定會在接下來的兩個轉會窗口中離開諾坎普，而皮克也表示俱樂部需要開始尋找一名年輕的中後衛。梅西希望俱樂部能簽下國際米蘭的什克里尼亞爾和那不勒斯的庫利巴利中的一位，這兩位球員是轉會熱門，身價也很昂貴。

梅西對巴薩現在的右後衛不滿意。從各個角度來看，塞梅多在巴薩都不算成功，而羅伯托的貢獻也不大。梅西希望右邊衛的人能和左邊衛的阿爾巴一樣好，看起來拜仁慕尼黑的基米希是他的首選。

巴塞羅那也需要簽下一名布斯克茨的替代者。目前，在巴爾韋德的陣容中，沒有球員是純粹的防守型中場，而比達爾和德容在這個位置上並不成功。梅西的選擇目前還不得而知。

最後，梅西還要求俱樂部簽下姆巴佩，但這一轉會似乎是巴薩力所不能及的，因為姆巴佩的夢想是為皇馬效力。然而，巴薩可以選擇薩拉赫或桑喬。

你認為巴薩能在2020年簽下四名球員並與梅西續約嗎？
