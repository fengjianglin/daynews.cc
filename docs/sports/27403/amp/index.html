如果許昕沒有劉詩雯搭配，混雙項目上他是否依然可以發揮優勢？

天天要聞
2019-12-20

導讀：如果許昕沒有劉詩雯搭配，混雙項目上他是否依然可以發揮優勢？

對於乒乓球混雙項目，這個賽季成了一個重中之重的問題和焦點，作為首次亮相奧運會的項目，這個項目存在也給各大乒協在明年東京奧運會的競爭奪冠上帶來了可能，增加了比賽的不確定性。所以這個賽季關於奧運混雙項目的參賽人選，除了一些勁敵外戰球隊外，國乒也是尤為重視，想通過比賽的考量以及變換球員搭配來選擇出最合適、最有利於大局的混雙組合。

一個賽季下來，光光看戰績，在國乒中許昕/劉詩雯這對黃金組合無疑最為亮眼，尤其是這一次年終總決賽再次奪冠為國乒鎖定了奧運參賽資格，更是讓這對組合成了參賽熱門，但是同樣也有另一種聲音的存在，認為許昕搭配混雙組合併不是非劉詩雯不可，許昕沒有劉詩雯依然可以發揮出自己的雙打優勢，那真的是這樣嗎？

人民藝術家許昕的個人能力確實非常的出色，單打能力和雙打能力似乎一點也不偏科，甚至在雙打上的造詣要比單打更為出色，因為無論他搭配哪一位國乒主力，他都能夠打出足夠耀眼的戰績，男雙樊振東，林高遠，混雙孫穎莎、陳夢他都有過搭配，也打出了出色的成績，以至於讓很多人覺得許昕的雙打不用考慮和誰搭配，反正和誰搭配都不會差，沒必要一定是劉詩雯。

從個人能力上看確實很有道理，許昕作為黃金左手，雙打上所佔據的優勢是非常明顯的，雙打比賽講究的也是左右手搭檔，但是混雙項目和雙打還是有點不同的，它更考慮男女球員之間的技術配合以及默契，許昕的打法是左手直板兩面反膠弧圈結合快攻，無論是落點還是速度都有很多強的造詣，這一點和老將劉詩雯有太多的相似之處，可以形成相得益彰的效果。

因為劉詩雯的打法屬於右手橫板弧圈結合快攻，不論是速度還是落點都可以跟上許昕的節奏，所以兩人搭配起來才能夠給人一種行雲流水的效果，至於許昕沒有劉詩雯的配合還能不能發揮出自己的雙打優勢，答案是肯定的，同樣劉詩雯也可以和馬龍搭配，甚至王楚欽，而許昕也可以選擇孫穎莎、陳夢這樣的球員，但是選來選去，許昕配合劉詩雯這對組合是最為合適的。

為什麼呢？因為戰績說明了一切，參賽經驗更是能夠有足夠的說服力，這個賽季下來，經過國乒教練組的重重考驗，劉詩雯/許昕這對黃金混雙組合接連登頂卡達公開賽、布達佩斯世乒賽、年終總決賽已經向我們證明了他們兩人的化學作用。雖然這個過程中也經歷了不小的挫折與挑戰，但是相信在接下來的時間裡兩人能夠配合得更好，這本來就需要一個過程，如今要換其一，更加不利於明年東京奧運會的參賽大局，你覺得呢？歡迎你的留言評論喲！
