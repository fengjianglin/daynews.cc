家中意外受傷，維特塞爾讓多特蒙德很心塞！誰能頂替中場核心？

天天要聞
2019-12-10

多特蒙德連勝柏林赫塔與杜塞爾多夫，暫時走出了前段時間的困境。可是球隊剛剛找回狀態，就不得不面臨一個問題：中場核心維特塞爾受傷，將缺席上半程接下來的比賽。

令主帥法夫爾感到鬱悶的是，維特塞爾並不是在比賽或者訓練中受傷的，而是在家中意外摔倒。他的面部受傷，送到醫院後不得不接受緊急手術，俱樂部在周一的官方公告中表示：「維特塞爾成功的進行了手術，已經可以離開醫院了。」

雖然治療過程簡單，但他要休息一段時間。多特蒙德正在進入上半程關鍵的收官階段，周二是歐冠小組賽對布拉格斯拉維亞，聯賽中有美因茨、萊比錫和霍芬海姆三個對手，其中萊比錫是聯賽冠軍的競爭對手。

主帥法夫爾表示：「他不能參加上半程接下來的比賽。好消息是，他已經回家了。」可是法夫爾沒有透露會用誰來頂替維特塞爾。比利時人是多特蒙德中場的絕對核心，本賽季只有缺席了對柏林聯隊的一場聯賽。而且現在多特蒙德踢的是單後腰，對這個位置的要求很高。

德萊尼

現在看來，這對失去主力位置的德萊尼、達胡德來說是表現自己的好機會。最近被法夫爾改造成為中衛的魏格爾也有可能回到自己熟悉的後腰位置上。
