中國女排4位女富豪，都是各自球隊主攻，最小一人僅19歲

天天要聞
2019-12-21

目前，大家最關注的就是各大體育賽事，在這麼多的比賽項目中，女排是大家非常喜歡觀看的賽事，尤其是中國女排在各大比賽中的表現，其中有很多球員在賽場上的表現也是相當的優秀，留下了很多精彩的表現。朱婷、張常寧、顏妮以及龔翔宇等等都是大家非常熟悉的球員，為中國奪得了很多榮譽，更是中國女排的「功臣」，很多球迷不僅僅關注他們在賽場上的表現，還非常關心球員們的私生活，尤其是球員們的身價更是網友們關心的對象。

一些球員被球迷們稱為了「小富婆」，相信很多喜歡中國女排的球迷們都知道，朱婷是目前實力最強勁的主攻手，在賽場上的表現也是相當的優秀。不過她的父母僅僅是河南小縣城的農民，當時為了幫助朱婷實現夢想，不惜借錢也要供她學習排球，朱婷也並沒有讓父母失望，成功登上了「世界舞台」，也為家庭減輕了很多負擔，目前也成為了「小富婆」。

惠若琪也是大家非常熟悉的女排球員，在賽場上的表現也是相當精彩，更是中國女排在里約奧運會時的隊長，她不僅實力非常強勁，顏值也是非常的高，所以說吸引了很多粉絲的關注。中國女排在拿到冠軍之後，惠若琪成為了大家關注的焦點，所以說不僅拿到了非常高的獎金，還收到了一些商業活動的通告，惠若琪的身價也是越來越高了。

張常寧是大家非常喜歡的球員，也是被稱為「小富婆」，她是中國女排不可或缺的主力之一，畢竟朱婷實力多麼強勁，沒有張常寧等等球員的幫助，她也不可能拿到勝利。在拍超聯賽的新賽季中，張常寧成為了江蘇女排的主力球員，知名度更是上漲了很多，也就代表著張常寧的身價越來越高了。最後一名「小富婆」，年紀僅僅19歲。

這名「小富婆」就是李盈瑩，是在朱婷之後實力非常強勁的主攻球員，並且成為了國家看重的球員，在天津隊也是主力球員，身價也開始提高了很多。一些球迷表示關注球員們的身價一方面是對他們的關心，但是另一方面對於球員來說並不是好事，畢竟中國女排的姑娘們還是希望大家能多多關注她們在賽場上的表現。在大家印象中，中國女排的姑娘們哪位球員的身價也是非常高呢？
