昔日中超悍將已一年無球可踢 河南建業生涯成他最高光時刻

2019-12-20 125條評論 1,087次閱讀 106人點贊

如果時光可以倒流回四年前，我想李智超一定會選擇留在航海體育場，而不是去往南京加盟江蘇蘇寧。這名實德青訓出身的中場悍將每每總在轉會的重要節點上出現波折，這也讓他在30歲正是職業球員的黃金年齡便無球可踢。

李智超1989年出生於大連，場上司職後腰，出自於大連實德青訓系統。2008年李智超被時任大連實德主帥科薩諾維奇提拔到了一線隊，由於恰逢當年的大連隊正在進行新老交替，因此李智超很快便迎來了首發機會。在科薩手下李智超更多出任前腰位置，他也成為了實德89一批中較早踢上首發的球員。憑藉著在聯賽中的發揮，08年底李智超入選了中國國青隊並作為主力參加了亞青賽正賽，不過中國隊卻在四分之一決賽中點球大戰倒在了烏茲別克腳下，無緣世青賽。而在這支中國國青隊中還包括張呈棟、吳曦、鄭錚、張琳芃、曹贇定、劉殿座等多位日後的中超名將。

實際上在大連實德效力期間李智超更多作為替補前鋒出場，在2011賽季主場同天津泰達的比賽中，李智超收穫了他個人職業聯賽處子球。2012年底大連實德俱足球俱樂部解散，李智超加盟了同城球隊阿爾濱，由於引援名額原因他只能委身預備隊，之後更是因為轉會問題同阿爾濱產生糾紛，不得不自掏腰包買斷合同成為自由身。此後他先後輾轉效力麗江嘉雲昊和北京北控，擅長從低級別聯賽引進球員的賈秀全發掘了他的大連老鄉，並將李智超帶到了航海體育場。

2015賽季加盟河南建業後，賈秀全將李智超固定在後腰的位置上並委以主力重任，這充分的發揮了他拼搶兇狠、前插果斷和遠射出眾的特點。在整個2015賽季中李智超共出場21次打入2球並有1次助攻，累計出場時間達到了1696分鐘，這也讓他成為了中超聯賽防守型後腰中最為亮眼的明星之一。不僅僅是李智超，在賈秀全手下畢津浩、買提江、吳龑等一批平民球員都超水平發揮，幫助河南建業取得了聯賽第5的好成績。

2016年正是中超軍備競賽最為激烈的時期，憑藉在前一年出色的發揮李智超自然也吸引了多家勁旅的關注，最終中超新貴江蘇蘇寧以超千萬的轉會費簽下李智超，並與他簽下了一份為期四年的大合同。來到江蘇之後李智超的發展並不如預期般理想，佩特雷斯庫對張曉彬與拉米雷斯的後腰組合及其信任，因此李智超只能充當替補。最為經典的是該賽季中超第5輪，李智超在傷停補時階段替換拉米雷斯出場，登場僅僅8秒，第一次觸球便助攻任航得分，幫助江蘇蘇寧2-0戰勝上海綠地申花。

短短的一年的蘇寧生涯中李智超出場13次，出場時間只有可憐的331分鐘。2017賽季在時任主帥費拉拉的力主引進下，李智超加盟了志在沖超的武漢卓爾，雙方簽下了一份3年的合同。在這個賽季中武漢隊的命運卻十分波折，先後經歷了費拉拉、唐堯東、陳洋三位主帥，雖然李智超在聯賽里出場了25次，但武漢卓爾卻意料之中的沖超失敗。此後武漢隊選擇李鐵出任球隊新任主帥，也許是一朝天子一朝臣，費拉拉時期的愛將李智超和黃希揚被打入冷宮，雖然武漢卓爾在2018賽季沖超成功，但李智超僅僅出場4次。雖然在一線隊中出場次數不多，但李智超卻在預備隊聯賽中進球如麻，就連外援佩德羅-儒尼奧爾都自嘆不如。

今年沖入中超之後，李鐵不僅未將李智超列入一線隊大名單，甚至就連預備隊都未給其報名。也就是說在兩年的時間裡，李智超只踢了69分鐘聯賽，本賽季中期李智超同武漢卓爾俱樂部解約，重新成為自由身。短短四年時間從中超炙手可熱的後腰，到無球可踢的境地，不得不感嘆命運的無情。

「祝賀河南建業25周歲生日快樂，很榮幸曾為之奮鬥過，航海那裡有美好的回憶，加油建業。」這是李智超個人社交媒體中僅有的一條內容，從話語中不難看出他對建業生涯的懷念。30歲的年紀並不算大，未來李智超是否還會延續自己的職業生涯呢？
