阿聯頭號強敵還沒完！郭艾倫16+6，師弟36+6+4+3斷，書豪壓力來了

2019-12-25 1,214條評論 2,205次閱讀 249人點贊

今天CBA遼寧和廣廈的常規賽，遼寧最終客場114比111，大比分逆轉取勝！

最近的這段時間，遼寧的戰績一直起伏不定、波動很大——此前，他們101比103，惜敗青島之後，緊接著又94比114，大比分不敵山東，2連敗！

雖然101比85，大比分力克福建，讓遼寧暫時喘口氣，但是他們的確急需要一場連勝來止住頹勢，改變一下球隊壓抑的氣氛！應該說，對陣廣廈，遼寧對這場比賽勢在必得！

但開局，遼寧打得並不順利，頻頻受挫！別看郭艾倫2中2拿到4分，趙繼偉2中2拿到7分，史蒂芬森2中1，拿到2分，效率都不俗！但是遼寧打得的確缺乏整體性！

隨著第2節和第3節，廣廈的外援威爾斯連續2節得分上雙，高效拿到22分，遼寧一度落後多達21分！

3節打完，遼寧落後19分，眼看著比賽已經失去懸念！可沒想到的是，遼寧最終在第4節找回了自己的進攻節奏！

4節開始，遼寧背水一戰，打自己最擅長的防守反擊，除此之外，還加入了全場緊逼！一時間，天賦不如遼寧的廣廈立馬陷入被動！被打了一波19比4！

雖然最終廣廈在進攻端打得韓德君6犯離場，但是史蒂芬森和郭艾倫早已經不可阻擋！

全場比賽，郭艾倫12投7中，其中三分球4投2中，拿下16分6籃板2搶斷；末節，郭艾倫7中4獨得9分！而史蒂芬森全場比賽20投12中，其中三分球2中1，罰球11中9，狂轟36分6籃板4助攻3搶斷的恐怖數據！對飆威爾斯！尤其是末節，師弟更是頂著5犯的壓力狂轟16分2籃板2助攻，8投全中！率隊完成反超！

顯然，這是一場蘇醒之戰，遼寧在關鍵時刻找回了自己的套路和贏球法則，這說明，他們仍舊是一支能夠打逆風球的強隊，作為阿聯和廣東今年奪冠衛冕的頭號強敵，他們還沒完！

不過目前，遼寧的戰績在本場之後依然還排在CBA聯盟第4，需要繼續加油！

當然，目前遼寧距離林書豪率領的北京勝負差距只差1場球，應該說，遼寧和師弟不能喘口氣，但書豪的壓力也來了！

最近2場球，北京也大比分輸浙江1場，因為賽季的深入，書豪同樣疲乏，表現波動很大！他能帶領北京穩住嗎？
