馬尚36+7易建聯15分 廣東主場20分大勝天津迎兩連勝

2019-12-21 188條評論 1,524次閱讀 141人點贊

北京時間12月20日20點，2019/2020賽季CBA第19輪繼續進行，廣東隊主場121-101戰勝天津隊迎來兩連勝。四節比分為：18-23、26-26、37-22、40-30。

技術統計

廣東隊：馬尚36分7板7助2斷、任駿飛17分6板2助2斷、易建聯15分9板、趙睿13分5板8助4斷、胡明軒11分3板、威姆斯16分6板3助4斷

天津隊：時德帥23分7板、托多羅維奇22分12板6助4斷、金鑫10分6板4助、劉帥9分3板5助2斷、蘭德爾13分5板2助、潘寧10分2板

比賽回顧

第一節，易建聯跳步扣籃得手，趙睿滑翔上籃命中，馬尚低手上籃打進，廣東隊開局6-4領先。胡明軒挑籃得分打2+1，金鑫扣籃得分，兩隊戰成9-9。劉帥跳投命中，馬尚反籃命中，兩隊打成11平進入官方暫停。暫停結束，杜潤旺三分命中，托多羅維奇勾手上籃得分，孟子凱三秒區飛身挑籃得分，天津隊19-16領先。任駿飛中投命中，蘭德爾兩罰全中，天津隊23-18領先結束第一節。

第二節，托多羅維奇三分命中，易建聯2罰全中隨後暴扣得手，廣東隊24-26落後天津隊。何思雨三分球命中，時德帥補籃得手，天津隊35-28領先廣東隊。威姆斯急停跳投得分，馬尚上籃命中，廣東隊緊追比分。蘭德爾反擊上籃打進，威姆斯上籃打2+1,馬尚連投帶罰，廣東隊將比分追到44-46。趙睿犯規，劉帥三次罰籃全部命中，天津隊49-44領先結束上半場。

第三節，易建聯左側底角命中兩分，馬尚命中追身三分，廣東隊將比分追成49平。金鑫連續上空籃命中，天津隊53-49領先。威姆斯飛身上籃命中，任駿飛勾手上籃得分，趙睿三分命中，廣東隊越打越順一個18-0的超級得分高潮將比分反超為67-53。時德帥三分止血，蘭德爾三罰命中，天津隊59-67結束得分荒。任駿飛上籃得分，威姆斯後仰跳投命中，廣東隊71-61領先優勢達到10分。馬尚空位三分命中，威姆斯跳投命中，蘭德爾和托多羅維奇兩罰全中，廣東隊81-71領先結束第三節。

第四節，易建聯右側底角命中兩分，隨後打成2+1，胡明軒左側底角命中三分，廣東隊一個14-6的小高潮將比分變成95-77，天津隊請求暫停。暫停結束，天津隊內突外投連追5分以82-97落後。暫停結束，廣東隊繼續擴大領先優勢，趙睿三分命中，胡明軒三分錦上添花，廣東隊118-98領先20分。最終，廣東隊121-101戰勝天津隊。

首發陣容

廣東隊：胡明軒、馬尚、易建聯、趙睿、任駿飛

天津隊：金鑫、劉帥、托多羅維奇、史儀、時德帥
