是林書豪他太軟？還是CBA球員太臟？鼻青臉腫事件帶來的反思

天天要聞
2019-12-22

近期CBA最熱門的話題，莫過於效力北京首鋼的林書豪，最近兩場交手上海大鯊魚和浙江廣廈的比賽，頻頻遭到對手針對，大量使用較為粗暴的方式來去防守他，弄得他渾身是傷。賽後林書豪也在個人社交軟體，曬出臉被抓傷及眼睛被打傷的照片，並開玩笑說「這場球賽打得像是拳擊賽一般，我不想變得鼻青臉腫」。此話一出，立即引來正反兩方論戰。有人認為中國CBA球員缺乏運動家精神，只會使用旁門左道攻擊對手；卻也有人認為這就是CBA的防守強度，是林書豪自己太軟弱無法適應。

這樣的爭議事件，有幾個問及值得好好思考：這樣的防守是否已經超出合理範圍？中國球員打球真的比較臟嗎？面對惡性犯規攻擊，林書豪和北京首鋼又該如何因應呢？

真正紮實的防守，是以腳步緊跟對手，保持自己的重心，並適度運用身體的力量，給對手施壓，即便出現犯規，也都是針對球犯規，而非和對手發生不必要的肢體接觸。CBA的球隊是如何防守林書豪的，下方的影片說明了一切。可以發現大多數的防守者，幾乎可說是「手腳並用」，「推、拉、抓、夾、拐、絆」樣樣來，就算林書豪都已經跌倒仍不放過，繼續把他壓在地上，不讓他爬起來。當他投籃出手，就直接往他身上撞過來。最誇張的是，林書豪交手浙江的時候，還有多次裁判已經吹判廣廈球員犯規，但他們卻刻意繼續和林書豪糾纏在一起，似乎有故意挑起事端的意圖。

以上所有小動作，在電視轉播的時候，因為距離和角度的關係，觀眾幾乎無法察覺，但是當鏡頭從另一個角度拍攝，就會發現這每一個動作，都在侵犯林書豪的圓柱體，而且都不是針對球在防守，而是直接往林書豪身上招呼。這些動作都已經逾越合理防守界線，都是有可能導致球員生涯毀於一旦的危險動作，顯然是嚴重的惡意犯規。另外，他們都是一次派出好幾個人來夾擊林書豪，讓他無處可逃，不免讓人懷疑是在刻意針對他攻擊。

近兩戰林書豪數據下滑 是因為無法適應CBA強度嗎？

林書豪從小在美國長大，一直以來都和身體素質比自己出色的對手競爭，因此不論是心智還是身體對抗性，都已經磨練到一定的水準，這也是為什麼他能夠在世界籃球最高殿堂NBA征戰那麼長的時間。

而且當他剛加入CBA時，中國的本土後衛幾乎對他毫無招架之力，一對一單防很容易就被他吃掉，就連身高210公分上下中鋒，也拿林書豪沒轍，在空中對抗只有被林書豪犯規進算的份，那時候不論是觀眾還是媒體，都極力稱讚林書豪的對抗性更勝中國本土球員。

北京首鋼本季的前16戰，林書豪不僅能貢獻場均25.5分6.3籃板5.7助攻2.2搶斷，表現相當穩定，還帶領北京首鋼保持聯盟比較靠前的成績，而且他的三分命中率還從季初的20%，攀升到現在四成左右的高水準，說明他對CBA的強度適應無礙。

即便從球季一開始，各隊就給予林書豪相當大的防守壓力，但還不至於侵犯到他的身體。不過最近兩場比賽，對手卻開始變本加厲。雖然他的投籃命中率依然維持在50%，但場均外線出手次數減少兩次左右，而且在體力大量消耗情況下，他的罰球命中率跌到只剩58.8%，導致他平均只能攻下16.5分。

也就是說，在正常情況下，其他球隊無法限制他，只能用粗暴的肢體碰撞或是小動作干擾他，才能夠稍稍降低他的火力。雖然說只要是球星就會被嚴加看管，但就像前面提到的，這些都已經不是合理的防守，而是帶有惡意的危險動作，所以林書豪的數據下滑，並非無法適應CBA強度，而是來自於不合理的防守對待。

林書豪不是唯一個案 這是聯盟普遍存在的問題，其實林書豪並不是受到這種「特殊待遇」的唯一案例。每當CBA球隊擋不住對方球星時，似乎就會使出類似的手法發動攻擊，最常遇到這種招待的，自然是各隊的外援。

效力遼寧本鋼的前NBA球員史迪芬森，上個月交手八一的比賽，就在爭球過程中，遭到對手肘擊，不只當場見血，牙齒還被打斷。賽後他還幽了自己一默，說這應該是自己這麽多年來作惡多端的報應。以另一種方式表達對於這次惡意犯規的不滿。

而且不只是外援，中國本土球員也有可能遭受同樣對待。像是中國第一控衛郭艾倫、CBA聯盟歷史得分王易建聯，都曾因為對手整場比賽小動作不斷而被惹惱，在場上大發雷霆。這些例子都顯示出，小動作或是惡性犯規，已經成為比賽日常的一部分，聯盟應該好好正視這個問題。

中國球員比較沒球品？所以打球比較臟？因為CBA惡意犯規的事件層出不窮，導致cba球員被外界貼上「球品差」、「打球很臟」的標籤。但造成這種現象的原因，純粹是來自球員個人的道德面嗎？我覺得不是。

80、90年代的NBA也是以兇悍粗暴的肢體碰撞聞名，犯規的情節一個比一個還要嚴重。最著名的例子，莫過於活塞隊針對公牛球星邁克爾喬丹制定許多防守方針，其中大部分都是屬於比較的手段，很多危險的動作都有可能導致球員生涯終結。後來許多球隊紛紛效法，久而久之便形成一股風氣。大約從2000年初期開始，聯盟為了保護球員，並增加比賽可看性，希望減少這種肉搏戰的戲碼，決定遏止這樣的歪風，首先在2004-05球季取消hand check（防守犯規的一種，防守隊員用手阻擋進攻隊員的走位；不能張開雙臂阻擋防守隊員的移動，合法的hand-checking技術只允許用手接觸對方的身體來跟蹤對手的位置，但手部不允許加力，也不允許阻礙對手的視線。）之後的每一季又把裁判尺度逐漸調整，將更多危險動作列為惡意犯規，才漸漸演改善球賽的品質。

NBA現行這種以保護球員為吹判的核心原則，不只是一種硬性規範，同時也用行動教育球員，什麼樣的動作是被允許的，什麼樣的動作是被禁止的。就連前幾年NCAA最著名的天殘腳格雷森艾倫，進入NBA之後，同樣受制於這樣的吹判標準，小動作和惡性犯規都減少許多。

反觀現在的CBA，裁判的尺度猶如80、90年代的NBA，許多明顯的蓄意犯規，他們都認為是合理動作，即便響哨，也只是吹判為一般犯規而已。林書豪先前的比賽平均能獲得9.8次哨音，但近兩場比賽，即便對方手段比以往更粗暴，他場均獲得的哨音反而降為9.5次，而且對手使出這麽多危險手段，裁判卻只祭出一次「違反運動道德犯規」，顯然CBA裁判的尺度寬鬆許多，也造成球員會認為自己的危險動作是被允許的。所以說並非中國球員比較骯髒或球品比較差，而是這個制度容許這些行為的存在。球隊最重要的資產就是球員，沒有球員，球隊哪還能進行比賽？身為世界上規模最大的職業籃球聯盟之一，CBA明顯在保護球員上做得還不夠。想要使聯盟更加進步，修正吹判尺度或許是首要之務。

面對惡性犯規 林書豪和北京首鋼該如何應對？要從根本解決惡意犯規充斥的問題，得修正吹判尺度，但這是一條漫長的歷程，所以在這之前，球員和球隊還是得想辦法保護自己。

可以發現林書豪和北京教練團，其實都已經做了不少努力。像是林書豪會常常利用空檔，跟裁判做溝通，提醒他們需要更注意對手的小動作。總教練Yannis也會在戰況比較穩定時，將林書豪提前換下場。交手浙江的第三節後段，由於球隊領先將近30分，Yannis教練就提前讓他下場休息，否則以往他都會打滿第三節。

​

另外，北京首鋼還能做的，或許是其他隊友要多為林書豪挺身而出，用行動告訴對手，我們不會讓人予取予求，就像是當年奧克利保護MJ那樣。對林書豪而言，沒有什麼比健康更重要，尤其是前幾年受到傷病纏身，好不容易重回球場，更需要格外注意。

林書豪「鼻青臉腫」事件 揭開CBA長久以來的弊病惡性犯規氾濫的問題，是CBA長久以來的弊病，透過林書豪的遭遇，這個問題終於獲得各界關注，包括媒體、球迷，甚至是球員，當然，聯盟更應該重視這個問題，並立即採取行動。CBA聯盟如果想要走得更遠，勢必得更重視球員權益，因為球員才是建立這個聯盟的根基。
