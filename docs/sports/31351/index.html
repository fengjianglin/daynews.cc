阿森納又要買人！冬窗緊盯德甲殺手 用他替代奧巴梅楊？

2019-12-24 2,397條評論 4,056次閱讀 342人點贊

奧巴梅楊與阿森納的合同還剩18個月到期，他一直沒有與球隊完成續約。加彭國腳希望去一家能夠贏得獎盃的俱樂部，這已經是公開的秘密。目前來看，阿森納已經很難說服奧巴梅楊留隊。未來的兩個轉會窗口，槍手要被迫出售奧巴梅楊，收回部分轉會費，否則後年夏天就可能免費失去他。

本賽季至今，奧巴梅楊已經打入11粒英超進球，佔到了阿森納總進球數的46%！在奧巴梅楊留隊前景不明朗的情況下，阿森納也必須做到未雨綢繆。據《天空體育》報道，阿森納已經相中了德甲勒沃庫森陣中的前鋒凱文-弗蘭德。這位德國邊緣國腳，也被視作是奧巴梅楊的替代者。

據《天空體育》透露，阿森納球探最近幾場比賽一直在考察這位27歲的德國球員。稍顯可惜的是，弗蘭德最近狀態不佳，已經連續9場比賽沒有進球。不過，弗蘭德的實力已經贏得了阿森納的認可，他曾在歐冠小組賽中攻破過馬競的球門，槍手有意在冬窗引進他。

天空體育稱，弗蘭德吸引阿森納的地方，是他在前場的多面性，作為左腳球員，他可以打9號位中鋒，也可以打右邊鋒。更重要的是，弗蘭德目前的合同只剩下一年半就要到期，槍手有望以一個較低價格談成這筆交易。

最近幾年，阿森納一直有從德甲球隊買人的習慣。目前球隊陣中的奧巴梅楊、扎卡、帕帕斯塔索普洛斯、科拉希納茨、萊諾5名球員，在加盟阿森納之前都曾在德甲效力。隨著弗蘭德的到來，阿森納陣中的德甲陣營變得更加充實。弗蘭德也將成為阿爾特塔執教阿森納之後的首筆引援。
