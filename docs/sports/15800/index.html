西甲今日頭版：令人羞愧的VAR 有獎賞的重頭戲

2019-12-15 454條評論 1,369次閱讀 161人點贊

熱愛西甲，關注伊比利亞半島頭條，一起來瀏覽今日各大西班牙報紙的頭版消息。

《馬卡報》：

1、「有獎賞的重頭戲」，當地時間今晚21:00，皇馬將在本輪西甲聯賽中客場挑戰瓦倫西亞，皇馬如果取勝就能登頂積分榜首，塞拉德斯的球隊則為了重返歐冠區而戰，本場比賽也是兩隊射手本澤馬和羅德里戈的對決。

2、巴薩在本輪西甲聯賽中客場2-2戰平皇家社會，他們對VAR表示不滿；馬競主場2-0擊敗奧薩蘇納，莫拉塔破門。

3、在其它兩場比賽中，格拉納達主場1-2不敵萊萬特，埃瓦爾客場0-0悶平畢爾巴鄂競技。

《阿斯報》：

1、皇馬將在當地時間今晚21:00客場對陣瓦倫西亞，只要取勝就能夠成為西甲領頭羊，齊達內需要決定是否冒險使用身背4張黃牌的卡塞米羅，他需要考慮之後的國家德比。

2、巴薩客場2-2與皇家社會踢平，比賽中布斯克茨禁區內犯規被判點球，但皮克在第91分鐘遭遇類型情況卻沒有給巴薩點球。

3、馬競主場2-0戰勝奧薩蘇納，重返歐冠區。

《世界體育報》：

1、「經典盜竊」，在本輪西甲聯賽的一場比賽中，巴薩客場2-2打平皇家社會，比賽末段皮克被對手放倒但裁判和VAR都無動於衷，格列茲曼和蘇亞雷斯先後為巴薩進球，但這不足戰勝一支出色的皇家社會。

2、莫拉塔和薩烏爾破門，馬競主場2-0力克奧薩蘇納。

3、當地時間今晚21:00，皇馬將在客場與瓦倫西亞交手；16:00，西班牙人主場迎戰貝蒂斯。

《每日體育報》：

1、「令人羞愧的VAR」，巴薩在本輪西甲聯賽中客場2-2與皇家社會戰平，皮克禁區內一個明顯的被犯規沒有被判罰點球，阻止了巴薩在阿諾埃塔球場取勝，VAR對此無動於衷，巴托梅烏將給西班牙足協主席盧比亞萊斯寫信抱怨，而皇家社會得到的點球VAR沒有異議。

布斯克茨賽後表示，「VAR應該通知裁判，我們永遠也不會知道為什麼它決定視而不見。」巴爾韋德坦言，「不看回放我也能說這是點球，而布斯克茨的那個不是。」德容也認為，「如果布斯克茨那個球吹點球，皮克這個也應該吹。」

2、皇馬將在當地時間今晚21:00客場對陣瓦倫西亞，取勝他們就能登頂西甲積分榜。

3、當地時間今天16:00，西班牙人將坐鎮主場與貝蒂斯交手，他們將迎回舊人魯比和伊格萊西亞斯。

4、庫蒂尼奧在本輪德甲聯賽中大放異彩，他上演帽子戲法並送出兩次助攻。

（Paul）
