北京時間12月5日晚深圳男籃主場迎戰四川男籃一戰中，憑藉著賀希寧的超遠三分絕殺，深圳男籃最終98-95險勝對手。值得一提的是，這次絕殺是比賽還剩0.1秒完成，而且是CBA史上最遠距離絕殺。某門戶網站直接用「比庫里還牛！ 賀希寧上演CBA歷史最遠距離絕殺」為標題，來定義賀希寧的這次絕殺。

那麼，上演了這次絕殺的賀希寧到底何許人也？這名於1997年1月出生於山東男籃的22歲小伙，其實擁有一顆大心臟，似乎專為大場面而生，而且面對越是強大的對手，他發揮越是神勇。

簡單梳理下賀希寧的履歷不難發現，去年12月21日，深圳男籃113-103擊敗廣廈男籃的比賽，首發出戰的賀希寧打了34分鐘，拿下22分8籃板4助攻，其中兩分球12投9中，22分8個籃板均是其賽季新高；今年1月29日，深圳男籃客場120-112擊敗遼籃，終結了後者的25連勝，賀希寧全場砍下27分5籃板3助攻1搶斷，得分創下了當時職業生涯的新高；今年4月8日，季後賽半決賽首戰面對廣東男籃，賀希寧出戰35分鐘，18投12中，3分球5中3，轟下30分5籃板……

那麼既然是濟南人，賀希寧為什麼沒有成為山東男籃的一員，而是到了深圳男籃了？這就不得不感謝林書豪了。2012年夏天，15歲的賀希寧參加了東莞籃球學校組織的林書豪訓練營濟南站海選，順利的成為了10名幸運兒的其中一員。進入林書豪訓練營後，賀希寧被時任東莞男籃青年隊教練的原園招入東莞青年隊，才開始專業訓練。也就是在這個時期，賀希寧有機會接觸到林書豪，並於2013年與林書豪有過合影。可以說，賀希寧職業生涯的起點，就源自林書豪訓練營。

2015年，賀希寧成為青年聯賽的現象級球員，他場均能拿下22分。當年10月的第一屆青運會男籃比賽，代表深圳隊出戰的賀希寧幫助深圳隊以絕對的領先優勢奪得冠軍，並以核心球員身份榮膺了MVP。隨後升入深圳男籃一線隊的賀希寧被租借到了北控男籃，與馬布里等球星在一起，成長迅速。上賽季，他回到了深圳馬可波羅隊，開始他的高光表現。

至於賀希寧被稱為「濟南勒布朗」，則源於賀希寧在國青隊時的經歷。當時入選國青隊的賀希寧因為身體好加膚色黑被隊友被調侃為「濟南勒布朗」。不得不說，面對這樣神勇的賀希寧，山東球迷心情很複雜，這個地地道道的山東球員，沒有與山東男籃有任何交集，這不能不說是一大遺憾了。
