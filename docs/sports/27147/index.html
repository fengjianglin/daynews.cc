天海託管到期面臨解散，租借球員都要離隊？李瑋鋒做回應透露實情

2019-12-22 925條評論 4,929次閱讀 2703人點贊

距離中超聯賽落幕已經過去了一段時間，相信過去的一個賽季對不同的球隊來說感覺都是不同的。對於天津天海來說，他們剛剛經歷了一個動蕩的賽季，年初球隊老闆束昱輝鋃鐺入獄，球隊被天津體育局託管，隊內人心渙散，中途經歷三次換帥之後最終驚險保級成功。雖然保級成功，但他們下個賽季並不一定有機會能留在中超聯賽中，對他們來說這個冬歇期的日子一點也不好過，球隊必須要解決好自己的生存問題。

年初束昱輝被捕使得權健陷入混亂

天海隊與天津體育局達成的託管協議即將於2020年1月1日到期，如今距離這個期限也不過還有不到10天的時間，球隊的資金也基本用盡，到目前為止還沒有企業願意出手接管球隊。此外，權健旗下的另一支球隊大連權健女足已經解散，俱樂部已經通知球員自尋下家，按照中國足協的規定，2020賽季開始各個中超球隊旗下必須要有一支女足球隊，甚至連男足隊伍資金都無法解決的天海隊，又談何再去運營一支女足球隊呢？這是天海生存面臨的第一道關卡。

保級成功之後天海仍然需要解決生存問題

除此之外，天海隊賽季初期為了節省運營資金，從魯能、蘇寧、恆大等球隊租借了不少球員，包括姚均晟、張曉彬、阿蘭、雷納迪尼奧等，如今球隊面臨這樣的局面，外界紛紛猜測租借球員留下的可能性不是很大。但是據主教練李瑋鋒透露，目前還沒有聽說任何一個球員跟他講過要離開，包括租借球員在內留隊的可能性更大。目前看來天海「球員超市開張」這樣的說法並沒有根據，更像是外界的一種炒作。

天海隊內的租借球員動向受到人們關注

在談到俱樂部未來的時候，李瑋鋒也直言自己無法掌控俱樂部的未來，一切只能靜觀其變服從俱樂部的安排。既然球隊還在，那就要按計划進行冬訓備戰工作，目前俱樂部也已經做好了冬訓的安排，不出意外球隊還會前往國外進行集訓。由於新賽季的各項政策還沒有落地，俱樂部和球員目前也都處在觀望狀態，等到具體政策出台之後俱樂部和球員才會有下一步的動作，因此從一定意義上講足協新賽季的政策對這支球隊的命運有關鍵性作用。

李瑋鋒：俱樂部的未來我無法掌控

現在的李瑋鋒與初來時的風光形成了鮮明對比

從風光無限的天津權健到如今苦苦支撐的天津天海，一支俱樂部有時候就是這麼脆弱，這客觀上也反映了目前中國足球的窘境，像天津天海這樣一陣風就會倒的俱樂部多的數不過來。我們希望天海俱樂部能夠活下來，也希望越來越多的俱樂部能夠健康運營和發展。您對於天海的命運有什麼看法？您認為天海還能出現在下賽季的中超賽場上嗎？歡迎您在評論區留言發表您的寶貴看法，最後，如果覺得好，點個關注再走啦~
