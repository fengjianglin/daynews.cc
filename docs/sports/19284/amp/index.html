曾經去西部是為了躲開詹姆斯，沒人願意跟他呆在一個賽區？

天天要聞
2019-12-16

熱火隊球星吉米.巴特勒在接受《邁阿密先驅報》記者Anthony Chiang採訪時，透漏自己曾經離開公牛選擇森林狼的最大原因是為了避開勒布朗詹姆斯。巴特勒表示就算當年的西部實力整體非常強勁，但好歹還有一些進入總決賽的希望。

但是留在東部，根本不可能跨過詹姆斯的球隊。同時巴特勒還認為如果是合同到期成為自由人的球星，基本沒人願意跟詹姆斯呆在同一個賽區。

對此，巴特勒說道：「當年我離開芝加哥絕對不是一樁簡單的交易，其中有一些複雜的事情，我不能告訴你們全部的事實，但我可以告訴你最大的原因是因為我想躲開勒布朗。在當時的東部，勒布朗就像是一座大山，你在賽季沒開始之前就知道誰會是今年的東部冠軍。

每一支實力強勁的球隊嘴巴上都會喊著：讓我們今年衝擊總冠軍吧！但實際上每個人心裡都明白，就算再怎麼努力也只能是勒布朗的墊腳石。」

「所以，我才會來到寒冷的明尼蘇達，我不想每個賽季都知道自己會出局。儘管當時西部的整體實力非常強勁，但只要努力，總能從中看到一些進入總決賽的希望。而留在東部，每個人都知道自己根本跨不過勒布朗的球隊。」

巴特勒說道。「這是一個公認的事實，即便勒布朗上賽季因為傷病跌入到了低谷，但還是沒有人想和他呆在一個賽區。只要是合同到期成為自由人的球星，他們在選擇球隊時一定會考慮勒布朗這個重要因素。」巴特勒說道。

確實，一直以來大家都說西強東弱，詹姆斯只敢呆在孱弱的西部，卻不知道一手造成西強東弱的人就是詹姆斯自己！他在東部的八年以來打跑了多少球星？

霍華德，喬治，德羅贊，巴特勒，米爾薩普，安東尼。當這些球星因為詹姆斯而逃離東部後，東部的整體實力肯定比不上西部。而如今巴特勒又從西部回到了東部，很難說是因為詹姆斯的原因還是因為討厭了的明尼蘇達寒冷的氣候而來到邁阿密。

但巴特勒的選擇絕對是明智的，如今的詹姆斯和湖人，就像是當年的騎士熱火一樣壓的其他球隊喘不過氣。
