廣東男籃年輕球員上場多，是怎樣成長起來的？

天天要聞
2019-12-20

廣東男籃有很多年輕球員，球隊成績還是這麼好，重要的是傳承文化，傳幫帶為榜樣，教練肯給機會鍛練新人。

說起廣東宏遠，讓人津津樂道的是風雲五虎:杜鋒、陳江華、王仕鵬、朱芳雨、易建聯，這五人組合是當時國家隊的中流砥柱。除了易建聯還在繼續征戰，其他四人均退役，在不同崗位為中國籃球貢獻力量。

杜鋒是現任廣東的主教練，兼任國家隊教練。陳江華由於傷病問題不得不提前退役，曾經一條龍過掉如日中天的科比，現如今遭遇讓人嘆息。王仕鵬常妖刀出銷上演絕殺本色，現從事籃球解說工作。朱芳雨萬分先生，手起刀落三分如雨下，現任球隊總經理。

在廣東五虎時代，球隊氛圍好，很注重年輕人的培養，阿聯、周鵬、任駿飛等人還年輕，在幾位老大哥的帶領下，他們逐漸成長為球隊的重要人物。當時的周鵬還是年輕後生仔，現在成為球隊的精神領袖，能從一號位防到五號位，防守悍將，常常單防對方的外援，這不是一般人能幹的事。任駿飛在球隊，默默無聞，能里能外，成了球隊不可缺少的中間力量。

球隊歷經了傳幫帶，阿聯周鵬二飛繼續為球隊貢獻力量，身邊又多一些後起之秀，以趙睿為代表，敢打敢拼已然成了他的座右銘，防守態度好，常常讓外援進攻受阻，這賽季出手不多，但穩定性較好，在外授與阿聯打不開的局面，第一個衝鋒陷陣就是他。

杜鋒敢鍛練新人，邊罵邊給機會，這是杜教練的執教風格，因為球隊的戰術打法需要這樣做，要做到快速傳導球，快速發球以及防守反擊，必須有更多人次的輪轉，保持上場人員都有充沛的體力，同時能給阿聯創造更多休息時間，家有一老如有一寶，在關鍵時刻要用得上，特別是季後賽。

這樣也給了新人們帶來了更多的上場時間，以賽代練，也只有廣東有這樣的實力。小將們如胡銘軒、曾繁日、杜潤旺、徐傑、萬聖偉等人都有可圈可點的地方。

有了阿聯、周隊、二飛，以及強大的外援馬尚與威姆斯的保駕護航，新人們都很多試錯的機會，有很多實打實的上場磨練，比同齡人就略高一籌了。胡銘軒經過了上賽季的磨練，這賽季進步明顯，底線三分提高了穩定性。曾繁日也是苦練內功，常被教練罵得狗血淋頭，但得到的也是最多的，又愛又恨，不是誰都有這福氣。杜潤旺相比上賽季，更多投入了防守，三分投射就沒了之前的驚艷，畢竟在場上多了防守大任，要做到攻守兼備，這賽季是一個很好的提升。徐傑限於身高，靈活性很強，常常與小外援進行貼身纏繞，精力與體力大減，影響三分正常發揮，處理球能力強，至今保持零失誤。

這樣的一支攻守兼備的團隊，能不出好成績，那也是不正常的。廣東會繼續在後面比賽進行快速大面積的輪換上場，就要做到專註度強，侵略性強，又能體力充沛，以快打快，又能以賽代練，一舉多得。

在這樣一支有底蘊的球隊，年輕球員不能在教練給機會的情況成長起來，那很快也會被淘汰，優勝劣汰是競爭規律。自律與勤勞才是前進的方向，像阿聯每場比賽都提前到場進行熱身訓練，如此刻苦勤勞，也給新人們樹立起一個正確的榜樣。
