西班牙國家德比火爆開打，巴塞羅那主場迎戰皇馬

2019-12-18 1,497條評論 4,386次閱讀 418人點贊

北京時間12月19號，西甲第10輪的比賽正式開打，萬眾矚目的西班牙國家德比也正式開打，巴塞羅那坐陣主場迎戰皇家馬德里，本場比賽也是全世界球迷近期最為期待的一場比賽。雙方球隊都是當今足壇的頂級豪門，隊內的球員也是頂級的，尤其是巴塞羅那近期梅西獲得了人生的第六次金球獎，讓全世界的梅西粉再一次陷入狂歡。阿扎爾、本澤馬、羅德里戈皇馬新三叉戟已經日益成熟，而格列茲曼、蘇亞雷斯、梅西巴塞羅那三叉戟也已是嚴陣以待，本場比賽不知誰能贏下比賽。

巴薩近期成績火爆，近7場正賽不敗，對手中不乏強隊，狀態越打越好，已漸入佳境。目前，球隊更是積35分排名聯賽第一。巴薩本賽季火力強大，聯賽15場只有2場交白卷，均進球達到2.73，依舊冠絕西甲，而且遠超其他球隊。截止到目前，梅老六在聯賽當中已經打入了12個進球，處在射手榜第一。蘇亞雷斯也是以9球，排名第三，聯賽當中無論誰遇到巴塞羅那，對於球隊的後防線絕對是一個很大的考驗。反觀，皇馬近期成績也是越打越好，近10場正賽連續不敗，一共收穫8勝2平，而且有6場實現零封，狀態目前正盛。上場比賽球隊客場3-1帶走了瓦倫西亞。球隊本賽季防守端表現相當不錯，聯賽至今15場有7場實現零封，場均失球只有0.73，目前西甲第2好。值得注意的是，本場比賽阿扎爾無緣本場比賽，後場大將馬塞洛受傷，也是缺席了本場比賽，這讓球隊面對進攻火力強大的巴塞羅那估計有點吃力。

回顧雙方的交手戰績，近6場交戰當中，巴塞羅那5勝1平保持不敗，佔據了絕對的優勢。自從C羅走後，皇馬在面對巴塞羅那就明顯感到有些吃力。本場比賽各大機構也是給出了主讓半一球中水位的盤口，說實話本場比賽皇馬拿分艱難，阿扎爾、馬塞洛、阿森西奧等眾多主力球員受傷，讓球隊元氣大傷。個人看好，巴塞羅那主場打出。
