交手18次，獲勝4次！樊振東贏得不容易，也基本鎖定東京單打名額

天天要聞
2019-12-16

文/梅長蘇

北京時間12月15日，國際乒聯巡迴賽總決賽繼續在鄭州進行，在最後一個比賽日爭奪中，樊振東/許昕3：1擊敗中國台北選手林昀儒/廖振珽獲得了總決賽男雙冠軍，而女單決賽中，陳夢4：1擊敗王曼昱成功衛冕，這也是陳夢連續第三年拿到總決賽冠軍追平了劉詩雯曾經創造的「三連冠」偉業，而在剛剛結束的男單決賽中，世界盃冠軍樊振東4：1戰勝了國乒隊長馬龍，第二次獲得總決賽男單冠軍，四次打進總決賽冠亞軍決賽的樊振東頭兩次均輸給馬龍無緣冠軍，本場比賽也算是遲到的「復仇」。

馬龍和樊振東的比賽格外引人注目，兩個人本場較量代表了當今世界男子乒壇最高水平的比賽，首局比賽樊振東開局3：0領先，穩住陣腳後馬龍連續追分把比分迫近至5：6，樊振東連得兩分再次把比分拉開至8：5，馬龍追回一分之後樊振東連得三分以11：7拿下第一局大比分1：0領先；第二局比賽雙方打的非常激烈，開局兩人從1平、2平一直戰至5平，馬龍連得三分把比分超出，樊振東隨吼還以顏色連得三分把比分扳至8：8平，馬龍連得兩分拿到局點，樊振東挽救兩個局點之後比分來到10：10平，樊振東再得兩分以12：10贏得第二局比賽大比分來到2：0。

第三局比賽雙方開局戰成3：3平，樊振東連得四分以7：3領先，馬龍追回一分之後樊振東再得兩分以9：4領先，馬龍連追兩分之後樊振東得分，樊振東10：6拿到第三局局點，樊振東再得一分之後以11：6拿下第三局大比分3：0領先；第四局比賽雙方開局依舊爭奪激烈，雙方從1平戰至6平，6平之後馬龍突然發力連得五分以11：6拿下第四局扳回一城；第五局比賽馬龍開局兩分落後，不過他隨即把比分扳平，隨後雙方從3平戰至5平，樊振東連得三分以8；5領先，各得一分之後比分來到9：6，馬龍再追一分之後樊振東進攻得分以10：7拿到賽點，馬龍挽救一個賽點之後樊振東得分，最終樊振東以4：1擊敗馬龍第二次獲得總決賽男單冠軍，落敗的馬龍無緣自己第六座總決賽男單冠軍，此前他已經獲得了5和總決賽男單冠軍。

本場比賽之前，馬龍和樊振東交手過17次，樊振東僅僅獲得了其中3場比賽的勝利，本次總決賽冠亞軍爭奪戰中擊敗馬龍，這是樊振東第4次在國際大賽中擊敗馬龍，僅從雙方交戰記錄來看，馬龍依舊佔有壓倒性優勢，不過目前來看，樊振東已經慢慢在接近於自己的巔峰狀態，而馬龍受到傷病影響，他目前的實力與巔峰時期相比已不可同日而語了，相信今後的比賽中樊振東對陣馬龍會越來越有優勢，我們也期待樊振東的表現會越來越好，經歷了上半年的低迷之後，樊振東從9月份開始參加的各項賽事就沒有讓男單冠軍旁落他人，他先後獲得了德國的、奧地利公開賽、世界盃比賽以及巡迴賽總決賽冠軍，目前來看，樊振東獲得一個東京奧運會男子單打參賽名額基本沒有懸念，馬龍的「低迷」增加了樊振東東京奧運會男單奪冠的籌碼。
