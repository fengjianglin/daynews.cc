英超第十六輪盤點：千里單騎飲紅酒，藍月藍軍齊掉隊

2019-12-09 0條評論 25次閱讀 0人點贊

摘要

表現最好的球隊：曼聯隊誰說曼徹斯特的天是藍色的天，曼聯的“紅小鬼”們用一場德比的勝利，讓球迷看到了紅魔復興的希望。拉什福德、馬夏爾和詹姆斯等人輪番衝擊曼城老邁不堪的後防。繼主場戰勝熱刺以後，曼聯又拿下了衛冕冠軍曼城，成為了本賽季英超“劫貧濟富”的代言人。

表現最好的球隊：曼聯隊

誰說曼徹斯特的天是藍色的天，曼聯的「紅小鬼」們用一場德比的勝利，讓球迷看到了紅魔復興的希望。拉什福德、馬夏爾和詹姆斯等人輪番衝擊曼城老邁不堪的後防。繼主場戰勝熱刺以後，曼聯又拿下了衛冕冠軍曼城，成為了本賽季英超「劫貧濟富」的代言人。

表現最差的球隊：伯恩利隊

巴恩斯本輪沒有出場，本賽季狀態出色的克里斯·伍德在鋒線上略顯寂寞。在客場挑戰熱刺的比賽中，紅酒軍團被熱刺的反擊，一次次的打垮。孫興慜更是上演了讓人嘆為觀止的千里走單騎。伯恩利可以輸球，但是不該這麼輸！

表現最讓人驚喜的球隊：埃弗頓隊

多倫不勝，主帥下課，降級危險，讓太妃糖在動蕩不安中搖搖欲墜。本輪迴到主場，迎戰客場成績不俗的切爾西，俱樂部啟用了前埃弗頓球員鄧肯·弗格森。這位前俱樂部功勛球員顯然給球員們帶來了十足的信心。球員們在古迪遜球踢出了信心，一場大勝一掃此前的陰霾。憤怒的小鳥，又回來了！

表現最讓人失望的球隊：切爾西隊

藍軍上輪主場戰勝維拉，球迷都以為他們重新找回了狀態。本輪面對慘敗默西塞德德比，主教練下課，積分滑落到降級區的太妃糖，顯得非常緊張。門將凱帕和球隊的後衛失誤不斷，讓對手打出了久違的氣勢。本場遠征古迪遜公園球場，藍軍就連1分都沒撈到，只能在客場收穫一場鬱悶的失利。

表現最頑強的球隊：狼隊

在1比2落後布萊頓的情況下，狼隊在客場再次發揮了強大的團隊作戰能力，若塔梅開二度，扳平了比分。努諾的球隊繼續上演「平局之王」的本色，面對近期狀態火熱的海鷗軍團，狼隊硬生生從客場帶走了1分。

本輪mvp球員:拉什福德（曼聯）

曼聯近5場聯賽，拉什福德4場比賽都取得了進球。特別是在被主教練索爾斯克亞安排到左路之後，拉什福德狀態大幅度回升。目前，他已經以10粒進球，在射手榜上追平了排名第三的奧巴梅揚。曼聯更是連克熱刺和曼城，排名同樣迅速回升。
