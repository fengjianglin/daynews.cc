兩年沒球打仍是本土頂級！孫悅：不恨任何人，只想證明給自己看

2019-12-24 192條評論 1,715次閱讀 550人點贊

最近兩場比賽，34歲的北控男籃後衛孫悅，狀態好的令人驚訝。

主場戰勝江蘇肯帝亞的比賽，孫悅輕鬆得到了22分；之前一場對陣青島，孫悅更是砍下了27分，追平了他CBA生涯單場得分的紀錄。迄今為止，孫悅場均可以得到13.1分、4.1次助攻外加1.3次搶斷和1.2個蓋帽，你沒有看錯，這就是一個兩年沒打球的老將，交出的成績單。在CBA的本土後衛里，孫悅的得分不是最高的，他並不是北控的進攻核心，這支球隊的球權基本掌握在外援的手裡；但若論綜合評定和整體效率，孫悅依然是頂尖級別的存在。

不知多少人還記得，孫悅上一次單場砍下27分的場景。2015年3月10日，2014-15賽季CBA總決賽第一場，北京客戰遼寧。那場比賽，孫悅出戰42分鐘，單場命中7記三分，拿到27分8籃板5助攻2搶斷3封蓋的全面數據，北京以19分的優勢大勝遼寧，為那一年最終奪冠定下了堅實的基調。

那是孫悅迄今為止最後一次嘗到總冠軍的滋味，在那之後他經歷了肩部手術、腳踝韌帶撕裂、腰椎受傷並且不斷反覆等一次次傷病的折磨。從2016-17賽季之後，孫悅之後的兩年，都沒有出現在CBA的賽場上。很多人以為孫悅就這麼完了，他距離退役只是時間問題。可當孫悅上演了「大聖歸來」之後，他的表現讓所有人折服。

「我作為球員，不想讓自己的職業生涯，以這樣的方式結束。我一直都相信，我是有能力在球場上給球隊做出貢獻，也是想給自己個交代，證明給自己看。」孫悅說。

對於一名職業球員來說，長期無法打球是一件極其痛苦的事情，而在漫長的低谷期保持狀態，更是難上加難。我們問孫悅，你是如何做到的？

孫悅說：「首先，是一個對職業的態度。同時作為一名老將要想保持狀態，更需要長期點滴的積累，包括休賽期自己加強薄弱的環節。訓練都要完成好，強度也要達到，還是飲食等等方面都要保持規律。一個運動員在役期間，都要是一個職業的過程，而不只是在場上的那些時間。你不能只在賽前兩個月，或者賽季幾個月要怎麼去做。你要始終在意，如何保持身體的狀態。」

中國的職業籃球隊，一般都實行集體管理，每天去哪、做什麼、吃什麼都由球隊統一進行安排。可是對於長期遠在美國的孫悅來說，能約束他的只有他自己，讓自己懈怠下來是一件太容易的事情，而要想始終保持自律，全靠強大的自制力。孫悅坦言，其實不僅僅是訓練，單單是吃飯這一件事，其實也是一種考驗。

「飲食上就是少油少鹽少糖，少吃脂肪。賽季期間我們會攝入更多的一些熱量，保證自己有足夠的體能去應對比賽。但是在休賽期的時候，我們會吃得更清淡一些，讓自己保持健康一些。不管在哪裡，我也是這樣規劃自己的飲食，」孫悅說，「其實對於中國運動員來說，能吃的東西太多，中國的美食也太多，所以適應起來不那麼容易。但是這也是職業球員工作的一部分，是你必須去適應的東西，你有義務和責任把這些做好。」

球迷們喜歡用「大聖歸來」來形容孫悅的回歸，而媒體的報道上，也常常寫道孫悅的「復出」。可是在孫悅眼裡，他從來都沒有過復出，即便是那段遠離賽場的日子，他依然認為自己還是一名運動員，一名時刻做好準備上場的運動員。

「其實我一直沒有做過『復出』的決定，因為一些原因，我過去兩年沒有打職業比賽，但我從來沒覺得我要退役，我會因此而退役。我不想這樣結束我的職業生涯，重新站在場上，對我自身是一個證明，證明給我自己看我還行。在以前年輕的時候，我總是想證明給這個看，證明給那個看，但後來時間長了，我發現自己已經過了那個年紀，過了那種狀態。不太在乎別人的看法，只想證明給自己看，證明自己還可以，還能在場上發揮自己的作用，」孫悅說，「我沒有覺得自己退役了，我也在準備著隨時回到球場，我要做的就是時刻做好準備。中間有一些曲折的過程出現，我告訴自己，你要做好準備，不能等哪一天機會出現的時候，你因為沒準備好而沒有把握住。」

過去的兩年，是孫悅職業生涯的一個低谷期，他遠離球隊，遠離日子的隊友和教練，一直陪伴在他身邊的只有家人。孫悅尤其感激妻子陳露對自己的支持和幫助，他說：「我們在一起很長時間了，互相之間非常了解也有很好的默契，她很能設身處地地為我去著想。有的時候我的事情，她可能會更著急，所以有時候我也會去安慰她，沒關係這段時間就當做是韜光養晦的一個過程，把我們該做的事情做好。因為有些事情我們控制不了，我們能控制的就是自己的情緒和行為，所以還是自己做好迎接一切的準備。」

談到孫悅，前東家北京首鋼自然是一個繞不開的話題。因為首鋼俱樂部不給孫悅報名的決定，直接導致孫悅兩年無球可打。現如今孫悅加盟了同在北京的北控隊，曾經有人問孫悅，是不是想對首鋼完成復仇？孫悅當時笑著說，現在是和諧社會，哪有那麼多仇啊恨的……在他自己的眼中，感恩比仇恨更有意義。

孫悅說：「這麼一路走過來，我一直有一個讓自己不斷前行的概念，就是我永遠是感恩的。不管別人眼裡是好事還是壞事，在我眼裡我總愛往好的方面想。大家不都說，原諒別人其實是對自己最好的原諒，我覺得真的是這樣，不要去給自己背上太大的壓力和包袱，也不要讓自己去仇恨誰，去記恨誰，這些東西不太適應出現在，一個人的人生中，只會讓你越來越累，還是要把事情想得光明一些。」

在中國籃壇，孫悅一直是一個充滿爭議的球員，他的天賦是所有人公認的，但與此同時他也承受了很多誤解和非議。有人說他懶惰，有人說他愛耍大牌，可孫悅從來沒有為自己辯解過。「我這麼多年的想法都是，懂的人不用解釋，不懂的人也無需解釋。你肯定希望身邊留下的，都是那些更懂你的人，當然誤解肯定會有，所以還是做好自己。因為真的假不了，黑的也白不了。時間會給所有的事情一個真正的答案，當真相大白的時候，所有人都會知道怎麼回事，不需要去為自己辯解太多。」孫悅說。

孫悅是很多人眼中的「人生贏家」，他拿過NBA和CBA的總冠軍，在國家隊有過出色的發揮，贏得過大合同，也擁有幸福美滿的家庭。很多人眼中的人生目標，孫悅都已經實現了。34歲，是很多中國球員已經退役的年齡，又是什麼支撐著孫悅不知疲倦地在場上拼殺？

「我沒有把這些經歷當成很大的成就或者里程碑，我只是把這些榮譽當成我人生中的經歷和過程，起起伏伏就是人生，就看你怎樣去面對，正確地做出選擇面對它的方式。在我的人生中不存在所謂這樣的終點，走過了之後，回頭看看如果是好的，OK沒問題，再往前走可能會遇到很多更好的其他的事情。」孫悅說，「人生中有很多重要的事情，現在我不光有了家庭，也有了孩子，也會面臨很多其他領域的東西，不要把自己局限住，要更多往前看，尋找更多的可能性。」
