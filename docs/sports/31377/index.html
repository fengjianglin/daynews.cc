馬欽告別信：我所做的都是為球隊好 希望西班牙人一切順利

2019-12-24 802條評論 3,060次閱讀 831人點贊

直播吧12月23日訊 今天，西班牙人俱樂部官方宣布主帥馬欽下課。隨後，馬欽在自己的個人推特上發表了一份告別信。

馬欽在告別信中寫道：「離開一傢俱樂部從來都不是一件容易的事情，特別是當你非常堅定地相信一個理念，並且為之付出全部努力和工作的時候。我帶著最大限度的投入和熱情來到這裡，希望能夠幫助到皇家西班牙人俱樂部，我也非常感謝球員們和其他的工作人員們從一開始就為之付出。

「昨天是我作為西班牙人的主教練執教的最後一場比賽，為此，我想對在這幾個月里一直信任我的那些人表示感謝。首先，我要感謝『鸚鵡軍團』的球迷們對我的喜愛和支持，我在球場內外都能感受到這一點。我也要感謝工作人員們以及那些組成了這傢俱樂部的人們對我的鼓勵和幫助。

「我向你們保證，在任何時刻，我都是在做著我認為對球隊有好處的事情。然而，足球世界裡並不總是『良好的工作=好結果』的，不過我們必須一直前進。你們需要繼續前進。（西班牙人）還有時間實現自己的目標，新援們將會幫助你們。

「鸚鵡軍團，我祝你們一切順利。」

馬欽在今年10月7日接替加耶戈成為西班牙人的主帥，在其任內，西班牙人在聯賽取得1勝7負2平的戰績，在歐聯杯小組賽當中出線，並且晉級國王杯第二輪。
