3大歸化領銜！國足最強6人浮出水面，12強賽打日韓仍存在一大短板

2019-12-21 420條評論 1,335次閱讀 184人點贊

北京時間12月20日，在最新接受巴西媒體的採訪之時，效力於中超廣州恆大俱樂部的射手高拉特，公開的表示自己已經完成了入籍，成為了一名中國球員，接下來他的目標，就是要率領中國隊殺進世界盃賽當中。

高拉特在這樣一個時間點完成歸化，對於中國足球來說，有著非常重要的意義，這意味著在擁有了艾克森和李可之後，中國隊在前場將具備更為強勁的組織能力和殺傷力，要知道李可畢竟是一名工兵型的後腰，而艾克森則是一位相對來說不太具備組織和突破能力的前鋒，更多的在於去完成搶點得分把握機會，而高拉特就不一樣了，高拉特既具備組織的能力又具備突擊的能力，恰好是中國隊現在最為急缺的節拍器。

而在高拉特加盟之後，中國隊接下來出戰世界盃預選賽的最強陣容也基本上浮出水面了。東亞杯比賽中李鐵整體上帶隊表現還是不錯的，從目前的情況來看，李鐵最終被轉正的可能性非常大，而李鐵和里皮一樣都喜歡打433，如果繼續執教中國隊的話，李鐵很可能也還是會高舉433的套路。那麼這樣一來，中國隊在前場，鋒線上就可以派出由高拉特，武磊以及艾克森領銜的最強三叉戟，這也是中國隊現階段可以派出的最強三叉戟組合。而在中場方面，目前中國隊使用的會是李可，蒿俊閔和吳曦的組合，池忠國將是和李可輪換的重要人選。

其實就目前的情況來看，中國隊在中前場應該說是人才濟濟，進攻和防守的人才都有，基本上可以說是派出一個最強的6人組合了，然而比較令人糟心的就是在於後防線，中國隊的後防線實在是太差了。如果李磊沒有受到傷病困擾的話，他和李學鵬都是左後衛位置上比較值得信賴的人選，右後衛王剛雖然說有點冒失，但是助攻的能力還是比較強的，最大的問題就是出現在中後衛的位置上，現在中國隊自打馮瀟霆競技狀態雪崩式下滑以來，就徹底沒有人可用了，19歲的朱辰傑尚顯稚嫩，而張琳芃下滑的速度實際上比馮瀟霆還要快，已經不太適合在中國隊繼續打主力了。可以說中後衛位置是中國隊目前唯一的漏洞。不管是李鐵還是誰來出任中國隊的新任主教練，都必須找到一個合適的人選來儘快的填補掉這一漏洞，不然的話，打到12強賽階段，中國隊仍然沒有辦法和日韓抗衡。
