遼寧重返爭冠行列捷徑浮現：應簽回2.18米舊將 他能彌補致命短板

2019-12-11 48條評論 1,364次閱讀 223人點贊

末節發力在主場擊敗同曦隊後，遼寧隊順利取得了兩連勝。這場比賽，遼寧隊大外援巴斯表現一般，在36分鐘的時間裡，他13投8中得到18分9籃板3助攻，但還有3次失誤和3次犯規。

同樣是這場比賽，同曦隊外援哈達迪的發揮要更加出色。在35分鐘的時間裡，哈德的10投7中得到19分14個籃板5次助攻2次蓋帽，失誤則是2次。在和巴斯的對話中，同樣是34歲的哈達迪表現更加出色。

這個賽季，已經34歲的巴斯顯得狀態明顯老化。在場均30.4分鐘的時間裡，他場均只能得到19.9分7.8個籃板，雖然命中率達到61%，但在場上，巴斯的威懾力已經明顯不如過去。

在首個賽季加盟遼寧隊時，巴斯是飛天遁地的打法，內線強攻或是吃餅，他都能在籃下有足夠的破壞力。但這個賽季，隨著年齡的進一步增大，巴斯的威懾力明顯不如之前。在場上，他已經很少能夠在籃下完成強悍或者不講理的進攻，大部分都是吃餅或者中投，單打得分的時候已經很少。在遼寧隊急需要有人站出來的時候，巴斯是很少能夠站出來的。

換言之，現在的巴斯更多是錦上添花，而無法做到雪中送炭，這也讓他在遼寧隊的重要性下降。

這個賽季，遼寧隊因為沒有引進可靠的替補內線，導致主力內線的壓力更大。而在替補內線李曉旭發揮起伏，主力中鋒韓德君受到傷病困擾表現也是時好時壞的情況下，大外援巴斯被寄予了厚望。然而，他目前的表現並不出色，這也導致遼寧隊過去多少年的內線優勢，在這個賽季突然變得平庸。

巴斯的表現倒是足夠穩定，但遼寧隊對他的要求顯然不只是穩定，而是需要他能爆發，展現出CBA超級外援的風采，畢竟小外援史蒂芬森和球隊的磨合一直不太順利，作為已經在遼寧隊打第三個賽季的巴斯，自然需要承擔更多的責任。然而，巴斯並沒有打出特別理想的表現。

如此情況下，遼寧隊真的得考慮簽回前NBA中鋒梅傑里了。在賽季開始前，梅傑里曾代表遼寧隊參加非凡12的比賽，而且有出色表現。相比身高不足的巴斯，梅傑里的身高達到了2.18米，臂展和彈跳都非常出色，而且移動非常靈活，可以完成一條龍。另一方面，他還非常喜歡投三分，三分球也有不俗的威脅。如果讓他和韓德君搭檔，勢必能在保護籃筐、進攻選擇上有更多的辦法。而在巴斯表現平庸的情況下，遼寧隊也該考慮改變外援陣容了，因為一旦梅傑里回歸，立即能重返爭冠行列。
