三分命中率成廣東颳起快打旋風之基石，頑疾被杜鋒的嚴厲所治癒

2019-12-23 2,345條評論 4,460次閱讀 1285人點贊

三分命中率成廣東颳起快打旋風之基石，年輕全華班更穩健頑疾被杜鋒的嚴厲所治癒

（本文所有實拍圖由大佬鳴現場拍攝）

每年廣東隊作客挑戰廣州隊的比賽，天河體育中心體育館都會座無虛席！今年恰逢冬至佳節，或許家家戶戶都要團聚的緣故，昨夜廣州隊主場相對而言沒有前幾個賽季那麼火爆，但不可否認，能夠親臨現場的球迷朋友是幸福的，兩隊共同上演了一場精彩激烈的對決：最終憑藉人氣球員易建聯雙二十及廣東隊6人得分上雙的驚艷表現，衛冕冠軍客場以26分的巨大優勢如願拿下廣州隊，繼續著自己衝擊第十座冠軍的征程。

儘管最終的結果有些懸殊，但比賽的過程可以說還是跌宕起伏的，廣州隊坐鎮主場並非沒有競爭力，更不是一開始便繳械投降的主——雙方分差曾縮小到1分，35-36；直到上半場結束時，廣東隊才把分差優勢變成了兩位數，46-56。廣州隊的兩名外援亨特和尼科爾森一內一外具有很強的單兵作戰能力，上半場宏遠對他們的限制只能說一般，然而在自身進攻層面，三分球始終沒有投開，多次上演空位機會已出現但沒投進的一幕，命中率沒有達到往常的水準。在此情形下，廣東隊擅長的快速攻防並未能如願展開，加上廣州隊的戰術打法同樣以快著稱，上半場同樣頻繁利用廣東隊的失誤打反擊，因此即便實力有差距，但雙方的分差始終未能拉開。

此役轉折點同樣來自第三節，廣東一上來就打出10-5得分小高潮，將優勢擴大到15分。隨後廣州隊連得6分，將分差縮小到個位數。趙睿外線3分為廣東穩住局勢。隨後廣東再次打出10-3進攻衝擊波，把領先優勢擴大到了19分……至此之後，廣東隊穩步前進，沒有再犯鬆懈的毛病，保持住了優勢，自然保住了勝局。然而本場比賽所展現的一切，在筆者看來，三分球是否投開、三分命中率是否穩定，成為了關鍵，亦是廣東隊能否順利颳起快打旋風、迅速用「一波流」奠定生理基礎的基石！

回憶一下比賽有太多近似的畫面：阿聯搶到後場籃板後迅速交給威姆斯，後者骨子裡就充滿快的意識，第一時間向前推進或傳球，其他球員在此時刻也已落入自己擅長的位置，幾秒鐘的時間、幾次倒手，在對方防守立足未穩之際，就已覓得空位得分良機，進球是唯一的選擇。而在此基礎上，倘若創造的三分球空位投射機會都能把握住，「一波流」自然能如願形成，比分就會被一下子拉開。不可否認，廣東隊有很多場比賽都是以這種形式拿下對手的。

不過此役對於筆者而言，印象更深的則是比賽最後五六分鐘的時候，當易建聯、威姆斯下場，主帥杜鋒換上一眾年輕生力軍而組成全華班之後，他們依然拿出了與宏遠隊實力相匹配的表現，甚至更好！這相信也是賽後杜指導露出欣慰笑容的原因。想必球迷都不陌生，過去廣東隊不止一次出現過大比分領先後就鬆懈的情況，年輕球員總不能好好珍惜出場機會，輕易葬送主力球員辛辛苦苦掙來的優勢，大家俗稱為「習慣性鬆懈」……

但這個賽季，特別到了這個時刻，這種現象正在逐步減少，甚至大有消亡、絕跡的趨勢。年輕生力軍們愈發靠譜，也越來越穩健，早已沒有了怯場，而更多是能夠大膽展示自身的特點，並從防守做起，每球必爭，繼續保持或擴大領先優勢。像杜潤旺、胡明軒、徐傑、曾繁日、萬聖偉等小將，也在杜鋒的調教下逐步成熟起來，鬆懈的現象逐步能夠杜絕，與他們的發揮密不可分。

然而年輕全華班更穩健，頑疾能夠逐步被治癒，筆者認為主帥杜鋒的嚴厲要求、激情指揮，在其中發揮了不可替代的至關重要的作用。這是一個潛移默化的過程，並非一朝一夕、一次兩次就能發揮作用，長期的嚴厲督促，已逐步讓這些年輕球員在意識里形成了無論何時何地都不可以鬆懈的習慣，習慣成自然的道理大家都懂，有了這樣的習慣積累，逐步形成一種思維定式，自然不會再犯過去的錯！

很多人說杜指導太凶了，好像總是在罵人，首先大家不要過於主觀、過度解讀，每名教練都有屬於自己的執教風格，這種無比投入其中的激情指揮，這種比場上隊員還要著急的易怒情緒，本身就是杜鋒的執教風格，他也正是在用自己獨特的人格魅力感染著麾下的球員，讓他們逐步得到成長、進步。如果非要說罵，那也是將大家「罵醒了」，生力軍們從此逐步改掉了頑疾，球隊磨合漸入佳境，也始終在一條正確的道路上前行。

漫長的賽季仍在繼續，擺在廣東隊前的難關還有很多，在杜指導的率領下，相信球隊可以逐步克服難關，向目標前進。
