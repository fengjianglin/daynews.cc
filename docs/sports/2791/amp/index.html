金球獎評委：梅西從不竊取獎盃！但今年他和C羅都該在3名之後

梅西以7票優勢摘得今年金球獎、第6次得獎創紀錄，配不配、該不該的主觀討論都在其次，客觀結果是記者朋友們更欣賞他。義大利唯一金球獎評委、《米蘭體育報》記者保羅·孔多撰文闡述自己的投票理由，標題亮明立場：「我有異議，這是屬於利物浦和范迪克的一年。」文中表明對梅西能力和足壇地位的肯定，甚至用了當今「最佳球員」、不是之一的字眼將其和C羅分出高下，但限定於今年，絕代雙驕不如利物浦球員有力度。保羅·孔多原文如下，給球迷提供歐洲記者的視角供參考。

「從結果和榮譽、呈現出的足球質量、歐冠全過程淘汰對手和英超角逐到最後一輪等方面而言，利物浦無疑是2019年最成功球隊。欽佩於利物浦的奇蹟，我從沒想過把非紅軍球員放在金球獎的第一位。

然而，利物浦的出色反而產生了一些對他們不利的問題：《法國足球》的評委們把7名利物浦球員納入該獎30人初選名單，這導致了嚴重的分票。好在，他們設立了最佳門將獎，把我從要不要投給門將的糾結中解脫出來，包攬歐冠和美洲杯、終極榮譽上很漂亮的阿利森自然是最佳門將，第二、三位我分別投給特爾施特根和奧布拉克。過去我也曾有過類似糾結，2006年的布馮和2014年的諾伊爾。

保羅孔多的原文解釋

阿利森有了『歸屬』，對於我而言，金球獎只是利物浦其餘6個候選之爭。我把第一票投給范迪克，因為在進攻球員越來越受重視的當今足壇，已經很久沒有出現過這樣一位出色、並且能主導比賽的防守型球員了。荷蘭人的領袖力、從未被過掉的一對一能力、在對攻敞開局面下的正面破解力、甚至在防守人數劣勢下的單兵水平，給予後場安全保障具備統治氣質，使得利物浦敢於進攻放手一搏。更不用說他還能參與進攻和進球，比如上輪英超對布萊頓的梅開二度。當選年度最佳當之無愧。

我的第二位投給弗蘭基·德容。承認熱刺的強大和進決賽歷史性（孫興慜和阿扎爾不被列入前5我也非常痛心，遺憾），可我心目中理想的決賽是阿賈克斯對利物浦，以阿賈克斯淘汰皇馬和尤文的方式，他們和利物浦最能代表上賽季歐冠。因此我傾向第二位是德容，年僅22歲的中場舵手。

至於梅西和C羅，老實說，我覺得兩個魔鬼的金球獎排名不能高於第3名。絕非質疑他們個人表現（除了C羅今年11月的表現有些蒼白），但他們的俱樂部都局限於國內獎盃，冠軍杯上都讓人失望了。尤文未進四強，普遍低於人們對C羅加盟後的預期；巴薩進了四強，但又在一個噩夢夜晚被不可思議地逆轉，連續3年都有類似難以解釋的單場潰敗。綜合進國家隊因素，C羅獲得了歐國聯冠軍，梅西無緣美洲杯再度失敗，C羅更好一些，因此我把金球獎第三位投給C羅，第四位是梅西。

當然了，我藉此重申，我始終認為梅西是當今足壇最好球員。等他的職業生涯結束時，我們再討論他能否超越馬拉多納和貝利的歷史地位。今年的金球獎給了梅西不能說是錯誤，梅西不是偷來的，6個金球獎從沒有哪一個是偷來的，只是以金球評選標準——團隊榮譽和個人表現的綜合而言，在本年度，我覺得有人比梅西更合適。

我的第五位肯定還要給利物浦球員，我選了馬內，個人認為他今年做的比薩拉赫更好一些，塞內加爾得了非洲杯亞軍。如果奪冠了，說不定很多人會把第一票就給他。
