數說英超這10年：曼聯最壕氣 藍月亮成新霸主

2019-12-22 2,681條評論 4,379次閱讀 1505人點贊

21世紀第二個10年即將過去，英國媒體BBC盤點了過去10年英超的盛況，從球隊到教練再到球員，誰拿了最多獎盃，誰又進球最多而誰又最捨得花錢呢？讓我們一起來看看。

這個10年剛開始的時候曼聯是英超冠軍，朴茨茅斯還在頂級聯賽，蘭帕德還在為切爾西踢球，當時英國俱樂部的轉會紀錄還只是3250萬英鎊，10年彈指一揮間，很多事情都發生了翻天覆地的變化。

英超冠軍獎盃依舊在曼徹斯特，只不過是在曼聯死敵曼城手中，這個10年藍月亮4度登頂。2010年朴茨茅斯還在英超，不過此後他們連續降級，一度掉入英乙，目前則是在英甲。現如今的神燈已經脫下戰袍換上西裝成為切爾西少帥。轉會費紀錄？博格巴的轉會比當年的羅比尼奧足足貴了5000多萬英鎊。

10年間英超一共踢了3774場比賽，產生了10323個進球，所有俱樂部在轉會市場上的花費加起來超過了100億英鎊。接下來就看一下具體盤點——

曼城的崛起

過去10年曼城4奪聯賽冠軍，還有2個足總杯和4個聯賽杯，是最大贏家，堪稱這個10年的王者。

2011-12賽季曼城奪冠之前俱樂部已經多年未曾染指英格蘭頂級聯賽冠軍，上一次還要追溯到1968年，不過一切從2008年開始發生了變化，那一年阿布扎比財團入主藍月亮。

2010-11賽季，也就是這個10年第一個完整的賽季曼城拿下足總杯冠軍，從此一發不可收拾，2012年拿下歷史上第一個英超獎盃，阿圭羅的讀秒絕殺永載史冊。曼奇尼之後智利工程師佩萊格里尼在2014年為球隊拿下第二個英超冠軍，此後就是瓜迪奧拉的兩連冠。

過去2年英格蘭6大主要獎盃當中曼城獨攬5個，除了2018年足總杯旁落切爾西，剩下的全部奪冠。不過儘管他們拿到了這個10年最後一個冠軍，但卻不是以領頭羊的身份進入下一個10年，目前利物浦領先衛冕冠軍多達14分。

過去的這個10年曼城最大的對手是切爾西，2009-10賽季切爾西勇奪聯賽和足總杯雙冠王，而在此後的9年內還拿到了2個聯賽冠軍以及2個足總杯和1個聯賽杯。

其它的球隊？首推弗格森的曼聯，爵爺帶領紅魔在2011和2013加冕聯賽桂冠，而萊斯特城在2016年更像是奇蹟。

這個十年除了英超傳統豪門曼聯、曼城、利物浦、阿森納、切爾西之外只有4支球隊染指過冠軍，狐狸城是其中之一，而且是最重要的聯賽奪冠。另外伯明翰（2011年聯賽杯）、維岡競技（2013年足總杯）、和斯旺西（2013年聯賽杯）也都有過高光時刻，但不幸的是，後三者全部降入英冠。

當然，這個十年英超俱樂部在歐戰當中也有斬獲，切爾西（2012）和利物浦（2019）分獲歐冠冠軍，此外藍軍還在2013年和2019年拿下歐聯杯，而曼聯則是2017年歐聯杯冠軍。

曼城的霸權還體現在10年總積分榜上，他們以812分高居第一，同時也是唯一一家場均超過2分的俱樂部，贏球率高達65.78%。而3-0拿下阿森納的比賽也正好讓他們的凈勝球達到了500整，838粒進球足足比排第二的切爾西多了127球。

從4強爭霸到6隊爭雄

這個10年開始的時候英超還是Big4的天下：阿森納、切爾西、利物浦和曼聯，不過到現在這個陣容已經擴大到6支球隊，再加上曼城和熱刺。

這種狀態在總積分榜中體現的非常明顯，不管是積分還是場均積分，排名第七的埃弗頓和阿森納之間都有著巨大的差距。

有趣的是，在英超凈勝球榜上，除了6強，只有埃弗頓、萊斯特城和謝菲爾德的數據為正數，儘管謝菲只踢了17場英超。

英超強隊諸多所以競爭異常激烈，每年只有1個聯賽冠軍和4個歐冠參賽名額，過去10年英超強隊為此你爭我奪好不熱鬧。

當然，這個十年還有一個趨勢就是紅衰藍升，自從弗格森退役之後曼聯再也沒能染指過聯賽冠軍，而瓜迪奧拉的到來則是讓曼城王朝名揚四海。

上圖是2009-10賽季到2019-20賽季曼城和曼聯每個賽季場均得分變化曲線圖，從16年之後藍月亮逐漸趕超同城死敵。

當然，北倫敦的激烈爭奪同樣值得關注，前7年阿森納聯賽排名都要高於同城死敵，但過去3個賽季熱刺強勢崛起逐漸趕超。

另外利物浦的爆發也引人注目，2011-12賽季他們場均1.489個積分並不高，之後羅傑斯讓紅軍的表現上了一個台階，而克洛普的到來則是讓這個數據直接提升到了1.867分，他們且本賽季的表現無人可擋。

不得不說的瓜迪奧拉

如果說曼城是過去10年英超最成功的俱樂部，那麼瓜迪奧拉就是最成功的主教練，加泰羅尼亞人出道以來已經拿下了20個冠軍獎盃，其中5個是在英格蘭所得。曼城在他的帶領之下勝率高達74.62%，場均得分更是恐怖的2.374。排在第二的弗格森勝率為69.70%，均分則是2.273。

在這份榜單上克洛普位列第四，不過以目前利物浦的勢頭超越第三的前切爾西主帥孔蒂只是時間問題，畢竟他在場均得分上只落後義大利人0.002分。

過去10年溫格是執教場次最多的教練，他帶槍手一共打了323場比賽，608分的積分也是教練當中最高的，但場均得分這一塊教授並不突出，只能排在第八（只計算帶隊超過一個完整賽季的主帥），在他前面的還有曼奇尼、安切洛蒂和穆里尼奧。

出場最多的球員

過去10年紅軍隊長亨德森成為出場次數最多的球員，從桑德蘭到利物浦他一共出戰306次，排在第二的是他隊友米爾納，剛剛續約的英格蘭全能型中場只比亨德森少1次。

出場次數這種當然少不了門將，前10里有4位就是，排名第三的福斯特出場303次，德赫亞（292）、切赫（274）、喬-哈特（271）同樣榜上有名。

另外福斯特983次撲救在門將中位列第一，切赫是零封場次最多的門神，274場里有107次沒讓對手攻破他把守的大門。

另外大衛-席爾瓦297次出場在所有球員里排第四，同時他還以89次助攻位列助攻王，比第二名的埃里克森多27次，另外西班牙人創造的768次得分機會同樣位列第一，第二名的阿扎爾只有595次。

這個10年最佳射手當然很容易猜出來，他就是曼城前鋒阿圭羅（173球），在他之後是熱刺的凱恩（134球），魯尼（114球）和盧卡庫（113球）是另外兩名10年內進球超過3位數的球員。

在射手榜前列的球員有一個人不得不提，那就是出身於草莽的瓦爾迪（96球）。2010年他加盟英格蘭北部超級聯賽俱樂部夏利法斯鎮，北部超級聯賽是個什麼性質的比賽呢？它是整個英格蘭足球體系中的第七級聯賽，10年過去，他不僅拿到英超冠軍而且目前以16粒進球高居射手榜第一，另外他還代表國家隊出戰了26次。

瓦爾迪還是這個十年英格蘭最具效率的前鋒（只考慮打進50球以上的球員），射門成功率高達23.3%。和他最接近的是小豌豆埃爾南德斯（22.5%），另外一個超過20%的則是迭戈-科斯塔（20.4%），無論是阿圭羅還是凱恩，射門轉化率都不高。在這個榜單上，現任切爾西主帥蘭帕德是唯一入選的中場球員（17.1%）。

花了多少錢呢？

隨著轉播合同金額的大幅度提高，英超各大俱樂部的荷包也越來越鼓，為了繼續留在頂級聯賽他們自然是要花錢了。

2010年英超國內加海外轉播合同的價值是32億英鎊，而最近的一次，也就是今年8月的新合同，從2019一直到2022年，有望突破90億大關。

所以英超俱樂部普遍財大氣粗，當然有范迪克這樣高價高能的成功典範也有安迪-卡羅爾這樣失敗的例子，此乃足球界常態，甚至失敗比成功的還要多。

讓我們來對比一下，根據德勤事務所的報告，2009-10賽季英超在兩個轉會窗口一共花費4.8億英鎊，10個賽季過去，僅僅夏季轉會窗口，這個額度就比10年前加起來還要高，高多少呢？14億英鎊！完全不在一個量級。

之前已經提到過，這個10年剛開始的時候轉會紀錄是羅比尼奧的3250萬英鎊，以現在的購買水平來看提高的同樣非常多。

英超第一筆超過5000萬英鎊的轉會費是托雷斯2011年1月份離開利物浦加盟切爾西，而聖嬰的身價在2014年8月被打破，當時曼聯斥資5970萬英鎊從皇馬買來迪馬利亞。

當然，這遠不是曼聯的極限，2016年夏天他們破世界紀錄以8900萬英鎊從尤文圖斯買進博格巴。曼聯的整體收入也是逐年遞增，2019年更是高達6.27億英鎊，所以他們財大氣粗也在意料之中，英國足球最貴的3筆買人都由紅魔貢獻，除了博格巴還有今年夏天8000萬英鎊的馬奎爾以及2017年7500萬的盧卡庫，和盧卡庫並列的是從南安普敦轉會利物浦的范戴克。

另外尼古拉斯-佩佩（7200萬英鎊），凱帕（7100萬英鎊）、羅德里（6300萬英鎊）、莫拉塔（6000萬英鎊）、馬赫雷斯（6000萬英鎊）、迪馬利亞（5970萬英鎊）則是前10的另外幾名球員。

2020年新年第一天，轉會窗口的大門又將打開，揮舞著支票簿的英超俱樂部們又可以開始採購了，為了更多的進球、更多積分以及更多獎盃，他們必須這麼做。
