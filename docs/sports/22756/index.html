國際乒聯下最後通牒，國乒首個奧運參賽人員卻“懸而未定”

2019-12-18 584條評論 2,113次閱讀 256人點贊

在國際乒聯職業巡迴賽總決賽上，國乒收穫了4冠的成績單，特別是老將許昕和劉詩雯3:2艱難戰勝日本組合水谷隼伊藤美誠，獲得了混雙冠軍，拿到了東京奧運會的混雙資格。

總決賽結束之後，乒協主席劉國梁隨後也對總決賽進行了總結，在總結中也表達了對混雙組合的表現並不是十分滿意，還對這對組合提出了新的要求。雖說拿到了參加奧運會的資格，但是最終能否參加奧運會卻懸而未定。根據國際乒聯的相關規定，近期之內，國際乒聯將向獲得資格的各協會發出確認函，各協會一經確認，名單就不能更改，如果各協會名單有變，必須參加奧運落選賽，去競爭奧運資格！ ​​​也就是說如果乒協確認了混雙的參賽資格，許昕和劉詩雯也將成為國乒首個獲得奧運門票的選手，這也讓迷霧重重的奧運門票競爭逐漸變得清晰。

從國乒的態度來看，許昕劉詩雯也是國乒重點打造的奧運組合，特別是即將開始的直通釜山選拔賽上，破天荒的加入了混雙比賽，這無疑是加強昕雯組合的實戰能力，但是對兩人的體力也是極大的挑戰，畢竟兩人在這一年身體都經歷過傷病。一旦確認之後，兩人身體如果出現傷病無法參加奧運會之後，不僅影響混雙項目，還會影響國乒的整個奧運參賽陣容。

從國際乒聯的確認函來看，如果乒協不確認的話，新組合將參加落選賽，至於落選賽的時間和地點安排目前還沒有定論，而參加落選賽也就意味著重新洗牌，對昕雯組合的自我否定。況且新組合也未必在短時間內磨合到位，即時落選賽出現，能否在奧運賽場上完成奪金任務，依然是未知數。因此這種不確認的概率太低。

乒協到底如何定奪，球迷也可以在評論區發表自己的見解。
