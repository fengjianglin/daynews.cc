吃得好掙得多！體能訓練更貼心，新疆總是讓人來了捨不得走

2019-12-24 643條評論 2,852次閱讀 1706人點贊

CBA常規賽第二十輪全部結束，坐鎮主場的新疆男籃在主場迎戰上海大鯊魚，昔日冠軍主教練李秋平不遺餘力想從新疆身上帶走一場勝利，這場比賽莫泰尤奈斯砍下30+28的超豪華大兩雙數據，但最終板凳深度更強的新疆在第四節憑藉一波流逆轉，讓李秋平的願望落空，兩支新老土豪的較量最終以新疆的勝利告一段落。

一直以來，新疆都被冠之以土豪的稱號，原因是球隊在招攬國內球員以及引入外援方面捨得投入，隨著近幾年更多的俱樂部加入本土球員的人才爭奪、在外援的投入上也是與日俱增，新疆這個土豪早就名不副實，俱樂部在球隊建設上的投入仍居聯盟前列，但更加傾向於球隊梯隊建設和內部軟硬體的完善上。至於球員的收入新疆一如既往維持高水平，新疆球員收入高是不爭的事實。

新疆球員不僅僅是收入高，近期媒體又曝光了新疆某一日的一日三餐菜譜，葷素菜品和主副食花樣多達16、7種，說新疆球員吃的最好未必，但在聯盟新疆在球員身上的膳食用心可謂首屈一指，當年新疆籍球員入選國字型大小球隊，球隊還曾專程前往國家隊免費服務，可見新疆球員不僅掙得多，吃得也很好。

吃得好、掙得多還不是新疆俱樂部吸引優秀球員的趨之若鶩的全部，球隊一切從球員的身體狀態處罰，完備的醫療康復、訓練設施也是保障之一，在王非入主山西的第一年，王非指導做的第一件事就是提升球員的健身房建設，購入大量劍痕設備，科學和系統的訓練也是今年山西成為黑馬的主要因素之一。同樣，新疆的訓練房一直也是聯賽中數一數二的。

與上海的比賽賽前，新疆球員在美女體能師的指導下進行拉伸熱身瞬時引來羨慕嫉妒的眼球。在以往的概念里，球隊體能訓練師都是男性，包括國家隊女子項目，女性做體能師鳳毛麟角，新疆男籃引入美女體能師幫助訓練在聯盟中確定是獨一檔，而且還是白美的「洋妞」，這樣的待遇是動心一定不僅僅是球員吧。

沒有把引入美女體能師參與俱樂部管理想的「邪惡」的意思，僅僅是針對新疆的一些列推陳出新的做法，我們不得不佩服，遙遠邊陲的新疆，經濟的確還欠發達，但他們的理念並不落後，包括最早的人才流動方式，新疆總有令對手措手不及的辦法，不歧視更不會虧待，新疆對待人才的態度令人折服，在這裡個人價值、團隊榮譽都可以得到很好的結合。吃得好、掙得多、能圓夢，加盟新疆誰還捨得走？

經過二十年的沉澱，新疆早已走出急功近利的誤區，現在的新疆本土球員正逐漸成為球隊的中流砥柱，也有更多自己培養的球員被輸送到各個俱樂部，現在的新疆不僅僅在為球隊榮譽去拼搏，也開始為這個聯賽和中國籃球做出自己的貢獻，也希望更多的俱樂部能夠上進，承擔更多的社會責任。
