CBA最新積分榜：大黑馬重返前4，山東飆升第5，8隊之間只差2場

2019-12-21 1,260條評論 4,744次閱讀 470人點贊

北京時間12月20日，CBA第19輪的最後5場比賽結束，廣東主場121-101大勝天津，17勝2負的戰績繼續領跑，少賽一場的新疆14勝4負緊隨其後，北京主場19分慘敗給了浙江，暫時保住了第3的位置，山西主場戰勝深圳，超越輸球的吉林，升至第4，8連勝的山東連勝4位來到第5，遼寧、吉林和北控分居第6到第8位。八一1勝18負繼續墊底。

本輪率先結束的比賽中，新疆主場險勝同曦，繼續排名第二。山東主場大勝遼寧，從第9暫時升到第6，遼寧則跌到第7，北控輸給青島，跌到第8，廣廈則在客場爆冷輸給了四川，遭遇3連敗，跌出前八。

廣東主場迎戰天津。廣東高居第一，天津倒數第二，兩隊實力相差懸殊。上半場，廣東打得不夠興奮，半場落後了5分。第三節他們轟出一波37-22瞬間反超了比分，最終主場121-101大勝天津，取得2連勝，17勝2負的戰績仍舊強勢領跑，布魯克斯轟下36分7籃板7助攻，易建聯15分9籃板。

北京主場迎戰浙江。北京剛剛取得兩連勝，近6場比賽贏了5場，浙江此前9勝負排名第11。比賽的進程出人意料，北京首節就被打懵，一度落後了21分。隨後三節，浙江都牢牢控制住了優勢，最終出場112-93大勝北京，送給了對手賽季最慘一敗。

福建主場對陣吉林。福建在換帥後強勢反彈，近5戰4勝狀態火爆。吉林是本賽季的大黑馬，此前12勝6負高居第4。上半場兩隊打得非常膠著，福建半場領先了6分。第三節福建一度拉開比分，頑強的吉林在第四節最後時刻追到只差2分，不過福建還是把握住了機會，主場121-114戰勝吉林，取得4連勝，凱撒上任之後6戰5勝，重新燃起季後賽希望。

山西主場迎戰深圳。山西是本賽季的黑馬，11勝6負的戰績高居第5，深圳是一支「神經刀」球隊，既能上演神級絕殺，也能莫名其妙地輸給一些弱隊。山西在第二節拉開比分，半場領先了13分。下半場，山西始終控制了局勢，最終主場99-87戰勝深圳，取得2連勝，戰績變為12勝6負，升至積分榜第4。

另一場比賽中，上海主場戰勝八一，終結了5連敗。19輪過後，CBA的積分榜發生變化，廣東、新疆和北京仍舊位居前三，山西升至第4，山東則從第9飆升到第5位，遼寧跌至第6，吉林跌到第7，北控排名第8，3連敗的廣廈則直接跌出了前8，不過從第11位的青島到第4的山西之間，只差了2場比賽，競爭非常慘烈。八一遭遇13連敗，1勝18負的戰績繼續墊底。
