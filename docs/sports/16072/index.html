2-1！國青迎四國賽首勝，19歲新星傳射，終結男足國字型大小3連敗尷尬

2019-12-13 123條評論 2,280次閱讀 460人點贊

北京時間12月13日15點30分，2019「一帶一路 七彩雲南」國際足球邀請賽第二輪打響，國青2-1擊敗貝爾格萊德紅星，迎來首勝。此役，國青新星周俊辰傳射建功！

第一輪，U20國足0-3不敵莫斯科斯巴達克二隊，此外，貝爾格萊德紅星二隊5-3擊敗了沙爾克04二隊。賽後，國青主帥揚科維奇痛批全隊表現，「這場比賽對比賽、對隊員都很失望，沒有一個隊員在正常發揮。」

值得一提的是，中國男足最近戰績糟糕，國足、國奧和國青是全線崩盤！國足1-2不敵日本，場面和過程讓人看不到希望；國奧0-1不敵敘利亞，進軍東京奧運會根本沒希望；國青也表現糟糕，遭到批評。

第二輪比賽，莫斯科斯巴達克二隊3-2戰勝沙爾克04二隊。隨後國青出戰，這一次，國青沒有讓人失望。第13分鐘，周俊辰在禁區里很冷靜，面對對手防守球員的包夾，周俊辰射門擊中門柱內側彈進球門內，1-0！

第36分鐘，周俊辰再次爆發，他大膽突破隨後傳中，後點幺旭辰得球推射破門，2-0！

下半場，貝爾格萊德紅星二隊一度扳回一城，但國青還是守住2-1勝利！

周俊辰出生於2000年3月，只有19歲。去年9月，因為在隨中國U19國家男子足球隊出訪拉練期間違規違紀，周俊辰一度被禁賽一年，同時被取消代表國字型大小的資格。

不過，周俊辰認錯態度良好，足協做出決定，將原處罰期縮減為2018年9月9日起至2019年4月11日止。如今，周俊辰是國青的當家球星。

此前，周俊辰還和西甲球隊格拉納達傳出緋聞，小將去到了西甲隊試訓。對於周俊辰來說，他是中國足球的希望，未來有望留洋。

國青首發如下：

23-韓佳奇，3-葉力江，4-溫姓小將，5-黃嘉輝，6-蔣聖龍，8-徐磊，9-劉超陽，16-徐越，17-周俊辰，19-何宇鵬，22-阿卜杜肉蘇力
