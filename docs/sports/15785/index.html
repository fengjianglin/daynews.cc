勞森砍35+12康寧漢姆20+20 福建客場37分優勢狂勝深圳

2019-12-15 400條評論 2,410次閱讀 512人點贊

北京時間12月15日，CBA常規賽，深圳主場以77-114不敵福建，品嘗到慘痛失利的滋味。

深圳隊賀希寧16分5籃板，賈雷爾14分7籃板2蓋帽，楊林禕10分7助攻3搶斷；福建隊勞森35分12助攻，康寧漢姆20分20籃板，黃毅超19分，陳林堅19分8籃板。

深圳隊戰績為8勝8負排在第11位，經歷賽季初低迷後，球隊逐步走上正軌，此番也寄希望主場能夠戰勝對手；福建戰績為4勝12負，球隊迄今的表現著實糟糕，亦是希望早日止頹。雙方過往交鋒紀錄，深圳以19勝13負佔據優勢，其中主場以13勝3負明顯佔據上風。

首節比賽，盧藝文站出連搶4分，陳林堅快攻打成2+1還以顏色，深圳以4-3領先。勞森快攻上籃打進，黃毅超外線相繼命中兩記三分球，緊接著他又籃下輕鬆吃餅。黃毅超的表現神勇，他又接到勞森的妙傳快攻打進，福建以20-9拉開比分。康寧漢姆和黃毅超又連續衝擊籃下，勞森飛身上籃打進，助福建以26-13領先結束該節。

次節比賽，康寧漢姆劈扣，劉洺宇飛身上籃打進，勞森正面弧頂處遠投命中，福建以33-13取得20分優勢。黃毅超又中三分球，勞森的突破製造2+1，緊接著勞森又連中兩記三分球，福建將分差拉大到49-23。勞森的籃下突破依然讓深圳隊感到難以阻擋，康寧漢姆跳投打進，劉洺宇再進空位三分球。半場戰罷，福建以60-26領先。

第三節比賽，勞森再進三分球，陳林堅籃下突破打成2+1，王哲林翻身跳投打進，康寧漢姆再完成戰斧劈扣，福建以73-38又拉開場上比分。王哲林和康寧漢姆相繼強攻打進，黃毅超飛身上籃命中，福建竟然將優勢擴大到40分以上！陳林堅遠投得手，勞森輕鬆上空籃命中，在該節戰罷，福建以96-50遙遙領先。

末節比賽，勞森突破打進，陳林堅外線強投三分命中，緊接著他又籃下拋投打進，福建以105-69遙遙領先。接下來的比賽再無懸念，福建牢牢保持著場上的優勢，以114-77勝出。

首發陣容

深圳：李慕豪、何忠勉、賀希寧、盧藝文、拜克斯

福建：黃毅超、何重達、王哲林、陳林堅、勞森
