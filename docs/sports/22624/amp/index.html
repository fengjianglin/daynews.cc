十年一遇接受採訪，熱刺主席列維：不怕把埃里克森賣給英超對手

天天要聞
2019-12-19

十年一遇！托特納姆熱刺主席丹尼爾列維居然接受採訪！據倫敦標準旗幟晚報報道列維在接受採訪時表達過自己對於熱刺新球場的問題以及球隊及引援問題作出回答！其中一個採訪時則表示熱刺並不害怕將丹麥中場埃里克森賣給英超的競爭對手！

倫敦標準旗幟晚報截圖

熱刺主席列維聲稱俱樂部並不害怕出售埃里克森給任何一家英超競爭對手，新賽季開始這名丹麥中場因為鬧轉會的問題出場時間已到到縮短基本上是已替補形式出場，在各項賽事出場19次收穫2個進球和3個助攻其中一個就包括上周末擊敗狼隊的致命角球！這位27歲球員明年夏天合同到期，他拒絕了熱刺給他提供20萬鎊周薪合同，並可以在1月1號直接可以跟其他俱樂部商談自由轉會事宜！

然而據標準體育了解，列維下個月將願意聽取其他球會對於他的報價以避免免費放走他，曼聯是他國內追求者之一，而皇馬則是他海外主要的追求者！

主席列維

在接受獨家採訪時，當被問到埃里克森是否會被批准出售給另外一家英超俱樂部時，列維說：「我們真的不害怕與我們的競爭對手交易。」說實話在英超賽場上不乏向競爭對手相投的例子：比如范佩西當年從阿森納轉投曼聯，特維斯從曼聯去曼城等等！

後衛線兩名比利時球員阿爾德韋雷爾德和維爾通亨還有六個月的合同到期，有傳言稱穆里尼奧的到來後，他們正在考慮向俱樂部承諾自己的未來。列維補充道：「我的看法很簡單。對於一名球員來說簽下一份新合同，不僅條件是正確的，而且球員必須要做到這一點。這取決於那些球員願意留在熱刺，我們拭目以待。」

「我不想過多地評論個別球員，我認為這是不公平的，每一種情況都不同。也許有一個球員想留下，也許有一個我們不想留下的球員。」

儘管穆里尼奧一再表示他對他目前的陣容很滿意，但熱刺已經與下個月要想簽下的兩名球員聯繫在一起，一名前鋒將作為凱恩替補，一名後衛將給予他後場做選擇。然而列維讓俱樂部在一月轉會窗口遠離任何交易，並補充道：「何塞在會上說，我們現在需要的是讓球員踢得更好，這就是他們正在做的事情。」

「他已經明確表示，他不會在一月份尋找新球員，他對自己所擁有的的感到滿意，這就是他為什麼這麼說的原因。」列維上個月請來穆里尼奧接替波切蒂諾的帥位，當被問到他是否為前波爾圖、切爾西、國際米蘭、皇家馬德里和曼聯主帥設定了結束俱樂部11年無冠的目標時。他說：「首先，我們無法保證在前四名中獲得關將軍。沒有人在哪裡有任何神聖的權利。我們都回來看足球，因為足球真的太有競爭力。」
