斯諾克大冷！中國手下敗將5-1橫掃進64強，世界第11爆冷連遭大逆轉

2019-12-22 83條評論 1,032次閱讀 79人點贊

北京時間12月21日凌晨，2019年斯諾克德國大師賽資格賽結束了第一個比賽日的角逐，在這次角逐中，作為中國斯諾克的手下敗將，邁克爾懷特在面對世界排名第11的利索夫斯基時爆出大冷門，他是以5-1橫掃挺進64強，而利索夫斯基則是連遭大逆轉慘敗出局。

眾所周知，邁克爾懷特是英國的名將，他在世界賽場曾有過不錯的成績，不過近幾年整體實力有所下滑，2019賽季，他與中國斯諾克選手交戰表現的非常糟糕，與丁俊暉兩次交手都沒有取得勝利，可以說，他算是中國斯諾克的手下敗將。

而利索夫斯基則是英國斯諾克的球手，他雖然沒有奪得過世界大賽的冠軍，但利索夫斯基的世界排名一直都高居前列，前不久的蘇格蘭公開賽，由於先後淘汰希金斯、塔猜亞，馬克艾倫等三大頂級球星奪得亞軍，他的排名也升至世界第11位。

今天的比賽，兩位球員直接交鋒，本來按照雙方的整體實力和近期的狀態來看，外界並不看好邁克爾懷特能夠戰勝利索夫斯基，然而，就在今天凌晨結束的比賽中發現，邁克爾懷特不僅戰勝了利索夫斯基，而且還大爆冷門。

本場比賽，邁克爾懷特展現出非常不錯的調整能力，第一局，他在開局打得並不是很好，利索夫斯基一上來就取得領先優勢，直到比賽進入後半段，邁克爾懷特通過及時調整狀態，慢慢扭轉局面，最終利索夫斯基是以60-63遭逆轉輸掉第一局。

在贏下第一局之後，邁克爾懷特打得更加自信，他沒有給利索夫斯基機會，最終是以95-0,82-0連贏兩局。第四局，邁克爾懷特雖然被利索夫斯基以119-0扳回一局，但在隨後的比賽中，他再度通過調整佔據場上的主動，儘管第六局邁克爾懷特遇到了一些困難，但他最終還是以65-58逆轉贏下第六局，以5-1橫掃挺進64強，而利索夫斯基則是連遭大逆轉出局。
