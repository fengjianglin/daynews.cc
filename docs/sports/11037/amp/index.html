26天9場比賽，利物浦遭遇魔鬼賽程，狐狸城蓄勢待發，欲再造奇蹟

天天要聞
7 天 ago

可能讓利物浦自己也沒有想到的事情是，本賽季的英超爭冠，最大的對手並非是賽季前預估的曼城，反倒是萊斯特城。雖然萊斯特城的實力對比利物浦肯定是弱了不少，但是這個球隊最近的8連勝，仍然將積分咬得很緊，這讓人非常的驚訝，難道萊斯特真的有望第二次拿到冠軍嗎？

而且接下來利物浦將會迎來一段非常艱難的賽程，12月份的時候，其實對於任何英超球隊都是相當艱難的，尤其是作為上賽季的歐冠冠軍，紅軍在這段時間的賽程更是相當可怕。球隊要面對英超，聯賽杯以及歐冠，此後他們還要前往卡達，去踢2場世俱杯。

現在是12月11日，但是從12月4日開始算起的時候這個球隊從4日到29日這26天時間裡，要總計踢9場比賽，不到3天就有一場比賽要打。尤其是在聯賽杯之後，他們就要尋思前往卡達，對陣維拉的比賽22小時之後他們就要乘飛機前往對他們而言相當遙遠的卡達。

那麼是不是會對世俱杯進行一定的取捨呢？在這裡我們需要看到這樣一個情況，那就是渣叔自己也明確表示，現實的足球畢竟不是打遊戲，他肯定會對球隊進行輪換。世俱杯的大名單中，利物浦基本上都是派上所有的主力前往，這意味著聯賽杯和英超，他們可能會針對性的做一些放棄。這聽起來可能有點不可思議，難道捨棄英超？這倒不是說放棄英超的奪冠，而是特定的場次選取一些替補去征戰。

但對於萊斯特而言，這就是他們最好的追分機會。雖然利物浦現在已經是33場在聯賽中33場不敗了，但他們最近的賽程實在是太可怕，這的確是給了萊斯特一個大好的機會。

萊斯特上一次奪冠還是在2015-2016賽季，迄今為止已經是3年時間過去了，並且隨後這3年，萊斯特甚至還有一個賽季瀕臨降級，基本上他們的名次就是中游而已。

多年之後，在羅傑斯教練的帶領下，這個昔日的最大黑馬冠軍，再次以這樣的姿態示人，不知道這次他們能否完成奪冠呢？這個賽季的情況對比他們奪冠的那一年，不同在於今年利物浦確實有一家獨大的傾向，而當年他們奪冠則是沾了混亂局面的便宜，如果今年還能拿冠軍，那麼恐怕再也沒有人敢質疑萊斯特了。
