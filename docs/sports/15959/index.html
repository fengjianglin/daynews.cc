國足新希望：20歲名宿之子馬賽迴旋戲耍歐洲豪門球員

2019-12-14 2,194條評論 4,954次閱讀 1960人點贊

U20國青在昨天結束了一場友誼賽，揚科維奇率領的這支國字型大小球隊在雲南迎來了歐洲豪門貝爾格萊德紅星的挑戰，雖然對手沒有派出主力出戰，但是實力依然不容小覷。讓人感到驚喜的是，中國U20展現出了超強的能力，憑藉周俊辰的獨造2球，2比1爆冷擊敗對手。在比賽過程中，1名20歲的魯能小將貢獻了1次世界級的戲耍過人，讓人看到國足未來鋒線的希望！

這就是山東魯能隊青訓出品、天才新星劉超陽，他作為中國足壇名宿劉斌的兒子，可謂備受關注。而在這場國青隊的比賽中，劉超陽獲得了出場機會，在比賽下半場一次進攻中，劉超陽上演世界級過人，他在對方禁區前沿獲得隊友的界外球傳球後，做出了馬賽迴旋的精彩動作，成功突破對方4人包夾完成射門，戲耍歐洲豪門的防守球員！

如此精彩的過人表演幾乎很難出現在中國男足成年國字型大小的正式比賽中，國足最缺乏的就是這樣敢做動作的球員，劉超陽讓人看到了希望。而在隨後的比賽中，劉超陽還有過1次精彩的前插攻門。這是球隊在中前場的一次控球，隊友四十五度長傳禁區，劉超陽利用出色的跑位獲得頭槌機會，只可惜最終的頭槌被對方門將撲住！

國青隊這場比賽的發揮令人印象深刻，除了20歲的名宿之子劉超陽的馬賽迴旋過人，在比賽上半場，球隊的中場悍將徐磊同樣有精彩表現。這次他在中線附近接到隊友的高空傳球後，用一個漂亮的停球將球卸下來，隨後又用一個極其逼真的假動作成功戲耍了貝爾格萊德紅星隊的球員，動作十分瀟洒自然，對手直接被甩開數米遠！

除了在進攻端的突破，徐磊在防守端也非常積極努力，在這次禁區的防守過程中，徐磊成功搶斷對方的腳下球，隨後再次用急轉變向成功戲耍對手，並造成了對方球員的兇狠犯規。不得不說，這場比賽讓中國足球看到了未來的希望，特別是包括進球隊員周俊辰在內的幾名年輕小將在比賽中的突破，給人留下了深刻印象，希望他們能在中超有好的發展，早日成長為國足做貢獻！（老邱 中超球評）
