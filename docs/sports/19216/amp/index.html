國奧對手麻煩不斷，伊朗和國家隊一樣陷低谷，韓國恐召不回李剛仁

天天要聞
2019-12-16

韓國國奧主帥金鶴范。

在珠海四國賽最後一場比賽中，國奧3比0擊敗馬里，2勝1負獲得亞軍。截至目前，郝偉接手後，國奧已打了11場熱身賽，7勝1平3負進22球失10球，差強人意。接下來，球隊將在海口繼續集訓，按照郝偉的意思，還要再打兩場熱身賽，「我想多打比賽，但因為不是國際比賽日等問題，安排不了了。」郝偉說。

國奧在衝刺，而郝偉團隊，也在分析對手，從目前來看，三個對手中，除了烏茲，韓國和伊朗的狀態都很一般，而且，他們有共同的「軟肋」，就是防守不行。

烏茲國奧已開始集訓。

烏茲沒有留洋球員，都在國內效力，2018年亞運會上，以國奧隊為班底的烏茲，曾6比0大勝卡達，1/4決賽90分鐘與韓國戰成3比3平，加時賽結束前2分鐘，才因為一個點球輸掉了比賽。

2019年，烏茲的成績更好。6月時，他們曾兩勝埃及國奧，9月更是雙殺伊朗國奧，其中一場是4比1大勝，10月時，他們踢了3場熱身賽，兩場是對韓國國奧，一場1比3，一場2比1，其中輸掉的那場，他們上半場就少打1人，還有一場是對烏茲國家隊，以1比3失利；11月的迪拜杯，烏茲連平伊拉克、沙特，但1比2輸給東道主阿聯酋。

烏茲聯賽是11月30日結束的，12月7日，他們也開始了集中。16日，烏茲國奧將前往阿聯酋迪拜拉練集訓，同時在當地打熱身賽。

應該說，烏茲防守一般，13場比賽僅4場比賽沒丟球，但攻擊力強悍，打進了24個球，由阿布迪霍利科夫、赫什博耶夫和米拉赫馬多夫組成的三叉戟，值得國奧防線關注。

伊朗國奧境況不佳。

至於伊朗國奧，和伊朗國家隊一樣，都陷入了低谷。

9月以來，在前國腳埃斯蒂利的帶領下，他們打了6場熱身賽，只贏了1場，戰績糟糕。其中9月客場對烏茲遭遇兩連敗，10月在多哈2比1澳大利亞，11月客場對印尼1平1負，然後在多哈又1比2輸給了卡達。

這其中，客場對印尼不勝，給伊朗國奧的打擊比較大，當時，考慮到泰國的氣候，他們特意前往印尼熱身，但卻1平1負，而埃斯蒂利只是強調備戰時間不夠。

伊朗國奧目前有球員在歐洲效力，如前腰舍卡利曾在俄超喀山紅寶石效力2年（已回歸國內），隊長諾拉夫坎所有權屬於比甲沙勒魯瓦，目前租借回塞帕罕，右邊鋒德爾菲今年初同樣加盟沙勒魯瓦。

目前，伊朗的問題是防守差，最近的6場比賽都有失球。

韓國國奧。

至於國奧首戰對手韓國國奧，今年踢了6場熱身賽，10月在本土與烏茲打了2場，首戰對手少打1人的情況下3比1取勝，次戰1比2輸掉了比賽；11月的迪拜杯，韓國國奧先後戰勝沙特和巴林，但隨後被伊拉克和阿聯酋逼平。

12月9日，韓國國奧開始了新一輪集訓，但在名單內，旅歐球員只有被拜仁租借到弗賴堡的鄭優營，而前巴薩三傑中的兩個，目前效力德乙達姆施塔特的白升浩，效力比甲聖圖爾登的李升祐，都沒有入選，今年世青賽MVP，效力西甲瓦倫西亞的李剛仁，同樣不在。

此前，金鶴范曾表示，希望李剛仁回歸助陣，但從目前看難度很大，畢竟，這種比賽，俱樂部可以不放人，同樣的問題在白升浩和李升佑身上也存在。

韓國國奧。

韓國國奧目前防守和進攻都有問題，防守上，今年9場比賽只有3次零封對手，連柬埔寨都可以攻破他們球門，對此，主帥金鶴范也比較頭疼，不斷調整防線，希望找到最佳組合；進攻上，主力中鋒吳世勛本賽季效力K2聯賽，30場比賽只打入7球，在國奧也只打進3球；另一名中鋒曹圭成同樣效力K2聯賽，雖然打進14球，但在國奧，表現同樣一般。

如果李剛仁、李升祐等不能歸隊，韓國的進攻，無疑大打折扣。
