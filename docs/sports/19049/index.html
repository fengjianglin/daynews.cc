小盧卡斯：不後悔拒絕曼聯去了巴黎 穆里尼奧鼓勵我做更好的自己

2019-12-17 77條評論 3,883次閱讀 1337人點贊

GOAL網站報道，熱刺巴西邊鋒小盧卡斯坦言，自己曾經一度非常接近加盟曼聯，但是最終他當時選擇去了巴黎。小盧卡斯說，自己不後悔這個選擇。對於熱刺新主帥穆里尼奧，小盧卡斯說和他的交流總是很輕鬆，總是鼓勵他要做更好的自己。

GOAL網站報道截屏

在18年年初來到熱刺的小盧卡斯，是一名極具盤帶能力的邊鋒，在剛剛結束的英超聯賽中，小盧卡斯就上演了一次個人突破，幫助熱刺在狼隊客場首開紀錄。在來到熱刺之後，小盧卡斯已經為熱刺出場78次，打入了20球，其中上灑基歐冠半決賽客場3-2阿賈克斯的比賽，半場上演帽子戲法，幫助球隊大逆轉，是小盧卡斯的代表作。在波切蒂諾手下始終是替補的他，在穆里尼奧上任之後，坐穩了球隊首發的位置。

在接受ESPN巴西頻道採訪時，小盧卡斯回憶起了13年登陸歐洲足壇的往事。「是的，當時曼聯還是弗格森執教，我和曼聯有過會談，我非常接近加盟曼聯。我最終選擇加盟了巴黎聖日耳曼，我在那裡度過了5年非常美妙的時光。我贏得了很多，也學到了很多，我不後悔我這個決定。」

小盧卡斯強調，儘管很難有遇到過很多的困難，但是自己在這個過程中成長了許多，因此他沒有後悔自己這個決定。「現在我來到了英超，這是一個新的開始，新的希望。」小盧卡斯說。

談到現在熱刺的新主帥穆里尼奧，小盧卡斯滿是讚許：「他來了之後，全隊更加輕鬆。他還和我說了在聖保羅時候的事情，那時候他還在執教皇馬。這真的很放鬆，他經常提醒我鼓勵我，要做一個更好的自己。或許我們倆都說葡萄牙語的原因，所以交流變得更加容易。」
