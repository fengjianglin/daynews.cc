張辛昕：為國安拼到韌帶撕裂的“昕姐”，退役後遭申花小將質疑

2019-12-22 154條評論 3,048次閱讀 2061人點贊

2018年3月31日，北京國安俱樂部在自己的主場為張辛昕舉辦了退役儀式，並為他送上了定製的紀念球衣。相信在那一刻有不少國安球迷留下了眼淚，眼淚中有回憶，有遺憾，更有祝福，這一切都是對「昕姐」為球隊的付出所做出的肯定。

張辛昕退役儀式

縱使傷病多次侵襲，他永不言棄足球夢

張辛昕的足球生涯與很多球員一樣有輝煌也有低谷。他一共出戰過16個賽季的職業聯賽，獲得過2次中超冠軍。但是在他16年的職業生涯中遭遇了多次重傷，經歷了4次大手術，職業生涯可謂是一波三折。

就像張辛昕在自己的退役長文中寫到的那樣：「除了傷病，運氣一直都對我不錯，從小就不是隊里最突出的隊員，刻苦和努力追趕隊友一直是我的目標。」

訓練備戰

作為一名後衛，張辛昕自然不會是球場上最亮眼的那一伙人，他給球迷最大的印象就是低調而不失拼勁，每一次出場都是在盡全力做好自己的本職工作。

從進入體校到參加全運會，再到成為職業球員，最後宣布退役，張辛昕這一步步走來可謂是風塵僕僕。經歷過球隊退賽，經歷過重傷，張辛昕的足球世界五味雜陳，他的職業生涯真的可以用「悲壯」來形容。

毫無疑問，張辛昕職業生涯中最輝煌的時期是披上綠色戰袍的這9年。但是在國安2009年奪得聯賽冠軍的時候張辛昕卻遭遇了重傷，這使得他不得不接受手術的治療，正是這次手術影響了他的足球生涯。

效力國安時的張辛昕

他自己在退役長文中回憶這次傷病時說：「一次手術會影響到重心的平衡，導致另外一條腿後面的兩次手術。」職業球員的傷病只有他們自己最了解痛苦程度，這次傷病也使得他在病床上躺了一年，但是受過重傷的他依然沒有放棄自己的足球夢。

在經歷了傷病的侵襲以後，張辛昕的足球生涯開始逐漸走向光明，其中也不乏高光時刻。2013年6月29日，當時北京國安對陣青島中能，張辛昕在球隊落後的時候打進了一記遠射幫助球隊扳平比分。這是張辛昕留給球迷為數不多的進球記憶。

張辛昕的踢球最大的特點就是頭腦清晰，大局觀強，速度快。這也是為什麼多任主帥都喜歡使用張辛昕的原因。退役後的他也專門感謝了與自己共事過的教練，感謝了他們對自己的培養。

從傷病恢復

退役轉型搞青訓，謙虛好學他渴望做好工作

退役後的張辛昕更多的工作是培養年輕球員。從球員變成教練，從台前走向幕後，這是大部分球員的歸宿，張辛昕也一樣，他目前的職務是北京國安預備隊領隊，與此同時還管理球隊青訓的一些工作。

其實張辛昕在退役後的教練生涯是從U19國青開始的，他在U19國青的教練組中擔任訓練員的角色，他自己對於這份工作評價是：「如果能早點有這段經歷的話，對我的幫助會更大。」

張辛昕擔任國青教練

在他任職U19國青助教時也遭遇過「滑鐵盧」。當時上海申花小將周俊辰在訓練中直接炮轟張辛昕，說他不懂球。這件事對於張辛昕的影響是比較大的，外界不免開始質疑他的能力，甚至一些申花的球迷也開始指責他。好在球隊及時處罰了周俊辰，控制住了這次風波的影響度。

回到國安後的張辛昕還曾擔任了青訓網點管理經理的職務，他對於這份工作的理解是鍛煉自己工作能力的好機會。據了解這份工作的主要任務是負責管理球隊與各中、小學在青訓方面的聯繫，在合作發展中發掘人才，為俱樂部囤積後備資源。

現在我們可以在網路上瀏覽到許多張辛昕與孩子們的合影，可以看出他對於這份工作確實是十分重視，另一方面也算是回報了俱樂部對於他的信任。低調謙虛內斂的性格或許正是俱樂部對其信任的依據。

張辛昕與孩子們合影

在經歷了8個月的青訓網點管理經理的工作後，他成為了國安隊預備隊領隊，在教練這個身份上也可謂是成長迅速。張辛昕一向是一個愛學習的人，從他在年輕時給老大哥們打替補的時候就非常愛學習，並且對待自己的工作刻苦至極，成為了教練員之後他依舊保持著自己的好習慣。

足球是一輩子的伴侶

在前不久剛剛結束的老甲A比賽中我們又一次看到了張辛昕在綠茵場上奮戰的身影。雖然風采不及當年，但是不變的是他對於足球的熱愛。

張辛昕曾經在一個訪談節目中也談到過自己對於退役後生涯的打算，他說自己不會離開足球，會一直從事與足球相關的工作。

張辛昕任職國安俱樂部

職業球員對於足球的愛無法與普通球迷相比擬，他們對於足球的感情外人或許也無法感受，而作為球迷，只能從他的一言一行中去體會，或許也正是張辛昕為足球為國安所做的一切匯成了球迷對他的愛。

在張辛昕退役的時候，他特別感謝了國安的球迷。他說：「真的很想感謝大家，在職業生涯最好的時間和大家一起度過，也讓我感受到了全亞洲最好的主場氛圍！能在工體戰鬥是我的榮幸，感恩所有的一切。」

從「昕姐」變成張指

每一次國安主場散場後都會有大批的球迷堵在俱樂部門口向球員們索要簽名合影，而張辛昕一直都是人氣頗高的那一位，而他也不會輕視任何一名球迷，永遠都是耐心的為球迷簽名。

北京國安球迷最大的特點就是具有包容性，他們會支持所有為國安拼搏的球員，當然有時候也會對自己的球員有質疑，但是到了真正比賽的時候他們都會不遺餘力的為球員們加油助威。或許正是這樣的包容讓張辛昕感受到了一份真摯的愛。

張辛昕的足球生涯還很長，雖然已經從球員轉型成了教練，但是他始終沒有離開足球，足球對於他來說如同伴侶，他對於足球的感情是純粹的。足球場上的他是一名思路清晰的球員，場下的他是一名謙遜好學的教練，相信在未來我們仍能看到他的風采，因為他熱愛自己所從事的事業，有志者事竟成。
