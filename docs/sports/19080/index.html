戰意高昂，穆勒諾伊爾自信拜仁能復仇切爾西

2019-12-17 2,438條評論 4,203次閱讀 554人點贊

曼努埃爾·諾伊爾和托馬斯·穆勒認為拜仁慕尼黑有信心在歐冠16強中擊敗切爾西。

德甲冠軍會在2011-12賽季歐冠決賽後再度對陣切爾西，那場決賽切爾西1-1戰平拜仁慕尼黑後點球獲勝。

諾伊爾和穆勒在那場令人心碎的主場失利中為拜仁效力，拜仁在2012-13賽季贏得了三冠王，他們在溫布利的歐冠決賽戰勝多特蒙德。

在漢西·弗利克從尼科·科瓦奇手中接過臨時教練一職後，拜仁重新找回了一些更好的狀態，特別是在歐洲賽場上的出色表現，諾伊爾認為他的球隊能夠在這場比賽中保持「健康的自信」。

「我們期待在倫敦的比賽，我們在那裡有過一些美好的經歷。」這名門將通過拜仁官網表示：「他們是一支危險的球隊，我們必須非常認真地對待他們，並且集中注意力。很明顯，我們想進入下一輪。我認為我們可以帶著健康的信心參加比賽。我們對抽籤結果很滿意：對手是一支有吸引力的好球隊。」

穆勒表示，2012年的決賽與本賽季16強的比賽「毫無關係」，他補充道：「很明顯，在歐洲冠軍聯賽的16強中，你找不到任何一支小球隊。我們相當有信心。我對抽籤很滿意。儘管切爾西的球員會傷害我們。他們的攻擊能力是眾所周知的。」

「但關鍵是團隊的表現，而不是個人。我的良好感覺是基於此刻我們作為一個團隊感到非常高興。我們想成為奪冠大熱門，並證明自己的實力。」

邊後衛基米希說：「我們將帶著信心參加比賽。還有很長一段路要走，所以我們還不能確定球隊的陣容，但我相信我們會成為奪冠大熱門，我相信我們會取得進步。他們很難評估；他們目前的表現並不穩定。」
