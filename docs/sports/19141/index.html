被外國歸化後在奧運會連奪三金，遭網友大罵：我們不歡迎叛徒

2019-12-17 487條評論 1,487次閱讀 265人點贊

體育運動毫無疑問是當今世界上最偉大的，說起當今世界上水平和榮譽最高的體育聯賽是什麼呢？當然就是奧運會了。運動員如果在奧運會上能夠打出精彩的表現，我相信很多的人都會記得。說起運動員能夠代表國家出戰確實是非常的有意義的事情。但是也有不少的「歸化球員」，這些歸化球員就好像僱傭兵。今天我們要說的這一位是來自韓國的運動員，他的名字叫做安賢洙。他曾被國家隊除名，後加入俄羅斯國籍連奪三金，卻遭被網友大罵是叛徒。

安賢洙是一位短道速滑男子運動員，安賢洙的實力非常的出眾，他出生於1985年。他在很小的時候就展現出了非常高的速滑天賦，在他當時出道之初也就只有17歲，但是他那時候卻進入了國家隊，那時候的他眼神中還沒有那種尖銳的目光，一副未經世事的樣子。在進入了國家隊適應比賽走後，安賢洙成為了韓國隊內的三冠王，他拿下了1500米和1000米還有5000米接力的三個冠軍，他是韓國隊內的第一位三冠王。他在場上的實力讓我們認識這一位來自於韓國的「歐巴」，

安賢洙的生涯最輝煌的時刻就是在2006年，那時候安賢洙在短道速滑基本上無人能敵。但是非常的遺憾的是，安賢洙在生涯的這個年齡段本來應該是盡情地展現自己的時候，卻遭遇了傷病的困擾，在傷病的影響下，安賢洙的實力大大下滑，儘管他的狀態恢復得不錯，但是安賢洙最終還是達不到之前的巔峰水準。

大家都知道韓國隊的短道速滑的成績非常的出色，有著非常的多的出色的人才，最終安賢洙還是輸給了自己的晚輩們。在奧運會的大名單之中，安賢洙也是被韓國隊除名了。

但是安賢洙為了繼續追逐自己的奧運會夢想，安賢洙最終決定加入俄羅斯的短道速滑國家隊，在2011年的時候安賢洙接受了俄羅斯隊的邀請，更改了自己的國籍，加入了俄羅斯的國家隊，還把自己的名字給改了，名字叫做維克多-安。

在2014年的索契冬奧會上，安賢洙夢回巔峰，拿下了短道速滑的三個金牌。安賢洙為俄羅斯隊贏下了屬於他們的第一枚短道速滑的奧運會金牌，安賢洙也因為這一成就獲得了俄羅斯「英雄」的稱號，這一次「王者歸來」證明了安賢洙仍舊還是世界上最強的短道速滑選手。

因為他的成績，安賢洙在俄羅斯的生活十分愜意，但是在韓國國內，對於被外國歸化後在奧運會連奪三金的安賢洙卻遭到網友的謾罵。韓國網上說得最多的一句話就是：安賢洙你不要回來了，我們不歡迎叛徒。韓國網民們罵人的實力確實不輸給我們中國的網民。安賢洙如今也已經退役，如今的安賢洙回到了自己的祖國韓國隊。
