今晚！昔日CBA總決賽宿敵碰撞 新援立足未穩 遼籃能反客為主？

2019-12-24 1,616條評論 4,226次閱讀 413人點贊

北京時間12月24日晚，CBA常規賽上演兩場對決，其中浙江廣廈主場迎來遼籃，這是兩支球隊本賽季的第二次碰撞。作為兩個賽季前的總決賽對手，兩支球隊非常熟悉，從主教練到球員再到戰術打法，誰想贏下對手都不容易，特別是本賽季以來，兩支老牌強隊遇到了很多的困難。

本賽季第一次交手時，廣廈反客為主從一開場就打了遼寧一個立足未穩，那時候聯賽剛剛開始，史蒂芬森還在磨合，遼籃狀態並不在線。反觀廣廈，在傑克遜的帶領下，整體性的優勢非常明顯，不過僅僅十幾輪過後，廣廈已經發生了巨大的變化。開賽初期的雙外援已經全部更換，小外援威爾斯上場比賽剛剛登場，大外援的表現也並不出眾。遼籃能把握住對手外援立足未穩，磨合併不到位的狀況，從而反客為主嗎？這註定是一場拉鋸戰。

遼籃的外線發揮很重要，廣廈最近受到傷病影響，外線可用的人並不多，特別是孫銘徽與趙岩昊的缺陣，使外線不僅缺少炮台，也少了突破的高手。李春江對遼籃的戰術很了解，郭士強需要有所調整，他可以適當地增加史蒂芬森的持球進攻，這個強點可以給廣廈更多的威脅。相比於性格主帥李春江頻繁更換外援，郭士強既然選擇留下師弟，就給他布置更多的戰術吧！

遼籃對陣廣廈，還有一個看點在郭士強叔侄與李春江父子間的對話。郭士強與李春江最近的日子都不好過，執教一支球隊時間久了，難免會讓球迷有不同的看法，尤其當自己的侄子與兒子在隊中，更可能產生一些話題。一些球迷對郭士強使用郭艾倫的方法有不同的觀點，而關於李京龍的出場時間，部分廣廈球迷也有自己的想法。郭艾倫與李京龍，特殊的關係讓他們吸引太多的目光，想要打消大眾的質疑，唯有用良好的表現回擊才行。郭士強叔侄能贏下這場較量嗎？郭艾倫需要打得更穩健，不要過分在意自己的數據，球隊好才是真的好。廣廈是一支整體作戰的球隊，千萬別把自己陷入包圍圈中，串聯起球隊才是核心該做的。

廣廈的主場一向不好打，沒有哈德森，遼寧第一次客場打這支老對手，能攻克這裡嗎？希望遼籃頂住壓力，爭取帶著勝利結束三連客之旅。
