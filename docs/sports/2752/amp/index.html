前曼聯主帥穆里尼奧執教熱刺後，仍未能說服丹麥核心埃里克森續約，因此，「The Athletic」報道披露，熱刺有意在冬窗將他出售，避免明夏一無所獲。曼聯只需要用2500萬鎊加馬蒂奇的代價，就有機會換到埃里克森，但主帥索爾斯克亞的態度，可能讓轉會無法完成。

埃里克森過去數年都是熱刺核心，但今夏的轉會鬧劇讓他狀態下滑，表現不佳。穆帥不介意出讓丹麥中場給老東家，而且馬蒂奇是他的愛將之一，在切爾西、曼聯都合作過，索爾斯克亞卻不喜歡用他。

因此，進行交換是理想的方案，熱刺缺乏防守型中場，而曼聯則沒有真正的10號——雖然曼聯同樣欠缺後腰，但馬蒂奇與索帥的關係已經很難修補。

不過，索爾斯克亞對埃里克森的質疑，也可能讓轉會無法進行。埃里克森27歲，剛好在曼聯可以引進的球員年紀邊緣，但The Athletic稱，索帥擔心埃里克森只想加盟皇馬，他如果加盟，未來可能像博格巴一樣傳聞不斷。

索爾斯克亞已經明確表示，他只希望簽下渴望為曼聯踢球的球員，因此埃里克森的立場讓他不滿意。萊斯特城的英格蘭中場詹姆斯·麥迪遜是索爾斯克亞更為青睞的年輕10號，但「新小貝」可能會跟藍狐續約，他的球隊目前高居英超第二。

曼聯冬窗被認為需要引進球員，才能提升爭奪英超前四的機會，但目前轉會操作進展不順利。尤文圖斯中鋒曼朱基齊，也被索帥嫌棄，認為他太老，而紅魔主帥青睞的19歲挪威射手埃爾林·哈蘭，俱樂部則沒有能力引進。

如果索爾斯克亞繼續執教，曼聯在明年1月的冬季轉會窗沒買一人，將是不足為奇的結果。此前索帥公開表示，如果買不到合適的球員，他可能爭取租借一兩名新援。挪威人堅稱，他的決策是出於長遠計劃以及曼聯的最大利益考慮，不會為了挽救自己而隨意買人。
