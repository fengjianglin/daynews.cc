馬布里為其他球隊上了一課，要逆轉，首先要防住對手

2019-12-05 0條評論 20次閱讀 0人點贊

摘要

馬布里率領球隊挑戰天津隊，在首節比賽中，北控隊處於被動狀態。球隊堅持防守策略，無奈在本場比賽開局之後，時德帥異常勇猛，三分與中投，頻繁突破北控隊的後防線，使得北控隊處於被動狀態，由於球隊著重強調防守，自身進攻效率比較低，眼看著北控隊就要崩盤了，球迷為馬布里極其球隊捏了一把汗。

馬布里率領球隊挑戰天津隊，在首節比賽中，北控隊處於被動狀態。球隊堅持防守策略，無奈在本場比賽開局之後，時德帥異常勇猛，三分與中投，頻繁突破北控隊的後防線，使得北控隊處於被動狀態，由於球隊著重強調防守，自身進攻效率比較低，眼看著北控隊就要崩盤了，球迷為馬布里極其球隊捏了一把汗。

原本認為，馬布里叫暫停時候，想要布置進攻戰術，但馬布里重點強調防守，此時球迷摸不到頭腦，明明球隊處於落後的狀態，應該強化進攻才對，為何老馬還可以強調防守呢？北控隊防守強度越來越高。在天津隊搶斷之後，孫悅還奉獻出追防大帽。在首節老馬的球隊落後對手6分。

但是，第二節比賽中，球隊也沒有選擇與對手對攻。馬布里知道，要想逆襲比賽，就必須提升球隊的防守強度。因為強化防守，對手就很難輕易上籃命中，對方球員在高強度的對抗下，體能下降定然非常嚴重。在本場比賽中，北控隊也遭遇了很多困難，桑尼休戰，弗格再次受傷，因此球隊外援優勢也削弱。

在球隊落後的時候，馬布里沒有慌亂，誠然具有大將的風采，第二節中，湯普森幾次吃餅成功，張帆等人也有所貢獻，在高難度的對抗中，天津隊球員體能下降比較嚴重，而北控隊球員由於經歷了一個夏天的高強度訓練。體能儲備要比對手更加的出色，在天津隊球員體能透支的情況下，而北控隊球員的體能非常充沛，為最終完成逆襲，埋下了伏筆。在上半場結束後，頑強的北控隊扳平了比分。

在下半場比賽中，北控隊一如既往的堅持強硬防守的策略，使得對手非常難受，最終導致天津隊命中率大幅度下降，而北控隊在強硬防守下，愈戰愈勇，單節打出了32-15的懸殊比分，一舉扭轉的比賽的走勢。在比賽過程中，馬布里的球隊曾經大幅度落後，有些球迷甚至想要馬布里下課。但是馬布里比這些球迷更加懂球。

基於強硬的防守，使得對手進球難度越來越大，而自身的信心越來越強。在第三節是全場比賽的分水嶺，北控隊在本節比賽中，命中率超高，而對手由於體能消耗過大，命中率持續走低。北控隊之所以能夠完成大逆襲，就是用防守消耗對方，再進行反擊。

在第四節比賽中，北控隊收割比賽，最終，北控隊以105-90戰勝了天津隊。全場比賽，北控隊搶斷取得了16-8的領先，失誤也12-19少於對手，尤其是在第三節比賽中，北控隊防守時的對手失誤連連。

因此，對於一個實力並不是非常理想的球隊來講，要想完成逆襲，必須在防守上西足功夫。馬布里雖然在球隊落後狀態下還布置防守戰術，遭到一些球迷的質疑，但馬布里最終取得成功。
