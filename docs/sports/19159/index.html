穆里尼奧偷笑！英超四隊僅熱刺抽中大禮包，利物浦曼城簽運不佳

2019-12-17 221條評論 1,797次閱讀 345人點贊

2019-20賽季歐冠16強淘汰賽的抽籤結果出爐，英超四隊里，利物浦、曼城和切爾西都遭遇強隊，晉級形勢不容樂觀，不過穆里尼奧執教的熱刺卻迎來大禮包——萊比錫紅牛，這樣的對陣還是非常理想，這讓熱刺進入歐冠8強的概率極大，有趣的是，萊比錫紅牛的主帥納格爾斯曼年僅32歲，當穆里尼奧首次奪得歐冠冠軍的時候，納格爾斯曼才16歲。

英超的四支球隊都晉級歐冠16強，不過在這次抽籤中，四支球隊的三支都遭遇豪門對話，其中，歐冠衛冕冠軍利物浦碰上了馬德里競技，雖然克洛普的球隊實力佔優，而且是衛冕冠軍，但是西蒙尼的球隊作風頑強，利物浦想要晉級的話，難度極大。曼城的運氣也很糟糕，瓜迪奧拉的球隊碰上了齊達內執教的皇馬。要知道，齊達內執教皇馬的三個賽季，銀河戰艦都奪得了歐冠聯賽冠軍。而且在歐冠淘汰賽階段，齊達內的球隊從未被淘汰，可以說，齊祖就是皇馬晉級的光環所在。

切爾西的運氣也極為一般，英超豪門碰上了德甲的拜仁慕尼黑，目前蘭帕德的球隊在英超賽場陷入了掙扎中，而拜仁的狀態非常理想，庫蒂尼奧也被激活，因此，藍軍的簽運也是極為一般的。

相比之下，熱刺的簽運就非常理想，穆里尼奧的球隊遭遇了德甲球隊萊比錫紅牛，在熱刺可能遭遇的對手裡，萊比錫紅牛甚至是實力最差的一支。

雖然萊比錫紅牛有著32歲的少帥維納爾斯曼，但是他的經驗還是缺乏，尤其是相比兩次奪得歐冠冠軍的穆里尼奧，更是差了很多。應該說，穆里尼奧重戰歐冠淘汰賽，這一次的簽運是極為理想的。當曼城、切爾西和利物浦要同皇馬、拜仁、馬競死磕的時候，熱刺則只需面對歐冠經驗一般的萊比錫紅牛。

當然，穆里尼奧也不能太過大意，因為萊比錫並非絕對魚腩，在德甲賽場，這支球隊可是領頭羊，在聯賽15輪過後，萊比錫紅牛拿到了33個積分，凈勝球也有26個，因此，萊比錫紅牛也有機會淘汰熱刺。
