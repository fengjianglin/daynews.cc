國足23人名單弄錯15個！中國足壇再現奇葩事，美女記者這次火了

天天要聞
2019-12-09

國足東亞杯首戰將在明天開打，面對強敵日本全隊都在緊張備戰，但在賽前一份國足東亞杯名單卻鬧出了大笑話。

今天中午國內某著名媒體發布了一篇國足的東亞杯前瞻文章，其中國腳名單上23位球員的名字竟然有15人都寫錯了，實在令人尷尬。最無語的是，恆大本土最佳射手韋世豪竟然連姓氏都改成「魏」。

先看看這份讓人笑哭的國足名單：

-門將：劉殿柱、董春雨、鄒德海；

-後衛：王松超、于大寶、梅芳、蔣志鵬、李昂、紀翔、楊帆、王上源、明天；

-中場：蔡惠康、張賢哲、康健、曹雪丁、金景道、穆亨特江-莫扎帕；

-前鋒：武士超、魏士豪、董雪生、馮金、譚龍。

首先說回咱們的韋世豪，在這份名單中他變身「魏士豪」，另外兩名恆大隊友也被改成劉殿柱和梅芳。至於其他12名國足「新援」王松超、蔣志鵬、紀翔、武士超、張賢哲、康健、曹雪丁、馮金……等都一一入選。相信看到這份名單主帥李鐵根本不知道自己竟然召入了這些球員。

更令人無語的是，鬧這麼大烏龍不是那些不靠譜的小道自媒體，是身為傳統媒體的「正規軍」一家北方老牌報社。至於寫該報道的記者，也自然被「人肉」了出來。從文章署名看到，該記者名叫邢成博，確實是一名專業體育記者，曾現場採訪過世界盃以及亞運會等大型賽事。

在世界盃期間，她採訪時出鏡的打扮更成為外界的焦點，甚至有網友表示『有了她誰還看比賽』。事後也有球迷猜測這名單失誤的原因，可能是用語音錄入，然後發給報社，不過後勤人員並沒有檢查就直接打出來了。目前相關報社以及人員都沒有對此事給出說法。

這種小花絮就當作國足賽前的一個樂子吧，明天咱們就在東亞杯首戰日本，看看李鐵帶領下的新國足能否給球迷們驚喜。
