原本能更早登陸法甲 桑切斯：2018年大巴黎就想要我

2019-12-16 715條評論 3,057次閱讀 184人點贊

與里爾簽了4年合同的桑切斯上周五在法甲2-1戰勝蒙彼利埃的比賽中打進了他在俱樂部的第一個進球。

里爾中場組織者雷納托·桑切斯聲稱，他在2018年幾乎轉會到了法甲勁敵巴黎聖日耳曼。

這位葡萄牙國腳在8月份以2500萬歐元的俱樂部紀錄從拜仁慕尼黑轉會加盟里爾。

然而，這名22歲的球員透露，如果拜仁在前一年願意把他賣給巴黎聖日耳曼，他會更早地在法國踢球。

「我確實收到了巴黎聖日耳曼的正式邀請，」桑切斯說。

「但是我的俱樂部和我的前教練尼科·科瓦奇不想讓我離開。」

與里爾簽了4年的合同桑切斯，上周五在法甲2-1戰勝蒙彼利埃的比賽中打進了他在俱樂部的第一個進球。

在2016年7月從本菲卡加盟拜仁後，22歲的他很難進入一線隊。

德國冠軍打消了來自曼聯的興趣，得到了這名出生在阿瑪多拉的球星，但是在他來到拜仁一年多之後，他就被租借到了威爾士的斯旺西城。

拜仁主席魯梅尼格在7月份確認了這名葡萄牙人轉會到法甲的消息時表示:「桑切斯向我們提出了轉會申請，因為他想加入一家他能經常踢球的俱樂部。」

「不幸的是，在拜仁我們不能保證他能做到這一點。比賽對他來說很重要，因為他想在2020年歐洲杯上留在葡萄牙國家隊。」

「我要感謝雷納托取得的成就，並代表拜仁祝願他未來一切順利。」

儘管桑切斯成為首位在21歲以下獲得歐洲最佳球員金童獎的葡萄牙球員，但他在德國的表現並不理想，在他的第一個賽季里，他一球未進，也沒有助攻。

他在斯旺西租借了一個賽季，並進入了英超聯賽，也並沒有激發人們對他的信心。
