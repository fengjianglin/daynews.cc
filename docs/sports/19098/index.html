巴薩依舊榜首，歐冠又抽到好籤，梅西還遇上一件大喜事

2019-12-17 286條評論 1,771次閱讀 644人點贊

北京時間12月16日（昨天），對巴薩和梅西來說，都是個好日子。首先是凌晨皇馬客場1-1瓦倫西亞，這樣巴薩與皇馬都同積35分，巴薩以凈勝球的優勢力壓皇馬，依舊排在西甲榜首。要知道之前巴薩2-2客平皇家社會，巴薩球迷都擔心皇馬會反超登頂，結果銀河戰艦沒能抓住這個好機會。這樣，即將到來的國家德比，巴薩不僅佔有主場優勢，還有「打平依舊西甲第1」的心理優勢。

昨天晚上，歐冠16強抽籤，巴薩算是抽到了上籤，16進8淘汰賽對手是那不勒斯。這是一支曾被馬拉多納揚名的意甲勁旅，目前剛換帥，球隊狀態不是太好。不過，巴薩高層阿莫爾表示：「加圖索和安切洛蒂一樣是有實力的教練，我們需要帶著取得積極結果的態度去踢客場，然後在巴塞羅那結束這輪淘汰賽。」上上個賽季歐冠被羅馬逆轉的慘痛教訓，也讓巴爾韋德出言謹慎：「毫無疑問，我們必須得好好準備。」

和巴薩的上籤相比，皇馬昨晚就抽到一個下籤了，銀河戰艦在16進8淘汰賽，將遭遇曼城。當然，這事不能怪曼城，皇馬在歐冠小組賽只拿到第2名，這才是他們在淘汰遭遇強敵的根源所在。雖然瓜迪奧拉的球隊本賽季狀態不佳，幾乎失去衛冕希望，曼城歷史上也從沒奪過歐冠冠軍，但正是這兩個原因，導致本賽季瓜帥可能會放棄聯賽衛冕，一心專註歐冠。而一直想與巴薩爭西甲冠軍的皇馬，歐冠前景可能不是太美妙。

北京時間昨晚，梅西個人還迎來一件大喜事，《馬卡報》舉行了上賽季西甲頒獎典禮，梅西領取了皮奇奇獎（西甲金靴）和迪斯蒂法諾獎（西甲最佳球員），上賽季梅西以36球奪得了西甲金靴+歐洲金靴，幫助巴薩奪得西甲冠軍，此外，梅西還被評為西甲最佳球員。上賽季的歐冠金靴也屬於梅西。

雖然梅西本賽季初因傷缺席了多場西甲比賽，但現在仍以12球與本澤馬一起領跑西甲射手榜；梅西還以6次助攻，與瓦倫西亞的羅德里戈共同領跑西甲助攻榜。不出預料的話，本賽季西甲金靴和西甲最佳球員仍屬於梅西！
