拜克斯率隊三連敗，薩卡基尼夢遊北控連勝終結！CBA第19輪最水外援

2019-12-21 3,598條評論 4,997次閱讀 3285人點贊

CBA常規賽第19輪戰罷。每輪賽後，都是幾家歡喜幾家愁。有的球隊慘遭連敗，內憂外患，外援當真是太鬧心。本輪最水外援都是誰？且看下文一一道來。

深圳客場慘敗山西，吞下三連敗。這場比賽，拜克斯又水了。連續3場的低迷表現，他成功帶領球隊一次次走向了失敗。再不換人，深圳想闖入季後賽，難！

廣廈客場慘敗四川，吞下三連敗。而作為單外援出戰的普拉姆利的表現，當真是一言難盡。換來換去，廣廈最後換的這個大外援，卻好像成了最坑的那個。

北控慘敗青島，四連勝遭到終結。而這場比賽，薩卡基尼，又一次成了挖坑的那個人。連續三場，薩卡基尼的狀態又走上了下坡路。想必被取代的日子，已經不遠了吧。

第三位

常規賽第19輪，深圳客場挑戰山西。這一戰，深圳輸得毫無懸念，慘遭三連敗。

而拜克斯，則又一次成了帶領球隊走向失敗的罪魁禍首。

全場比賽，拜克斯出戰28分鐘，16投6中，僅拿到了14分5板2助攻的可憐數據。三分線外更是3投0中，毫無準星可言。

在最近的三場比賽里，拜克斯分別拿到了9分、16分，和這場的14分。他的低迷狀態，無疑就是球隊陷入連敗困境的最大因素。

由於深圳自身優質的內線，所以王建軍有意讓拜克斯作為一個串聯手，在場上打球。但這樣的身份，顯然不太適合拜克斯。他場均僅有6.6次助攻，僅排在聯賽的第11位，完全沒有達到預期的串聯效果。

有可能是模糊的定位，讓拜克斯陷入了迷茫。但更大的原因，卻是拜克斯本身實力的欠缺。畢竟，得分助攻雙雙排在前列的外援也並不少。

場場低迷，每場比賽都讓人如此失望。拜克斯的表現，已經到了深圳球迷忍耐的底線。前後兩個賽季的表現差距如此之大，到底是為何？

如果深圳再不趕緊換援，進入季後賽，怕是就成了一個奢望。

第二位

常規賽第19輪，廣廈客場挑戰四川。最終，廣廈在全場遭到壓制的情況下，逆轉失敗，慘遭三連敗。

而作為單外援出戰的普拉姆利，這場比賽的表現，卻是讓人一言難盡。

全場比賽，普拉姆利出戰33分鐘，僅得到了10分12板和1次助攻，還有1次失誤和整整5次犯規。

一直說傑克遜水，但現實卻是殘酷的。在傑克遜不在的四場比賽里，廣廈只贏下了八一，其餘三戰全敗。

而這落敗的三場，正好是普拉姆利出戰的三場比賽。這三場比賽，他場均只能交出15分8.3板的數據，還有2.3次失誤和4.3次犯規。這樣的表現，無疑是不及格的。

鬧來鬧去，廣廈換了多次外援，結果最後換來的這個普拉姆利，卻好像是最水的那個。

9勝2負的完美開局，到現在卻被玩成了11勝8負。內憂外患，尤其是外援如此鬧心，廣廈可真是有苦難言啊！

第一位

常規賽第19輪，北控主場迎戰青島。最終，北控被對手輕鬆拿下，四連勝遭到了終結。

而薩卡基尼，無疑就是北控輸球的關鍵因素。

全場比賽，薩卡基尼出戰34分鐘，僅得到10分11板5個助攻。命中率更是非常之低，10投2中，慘不忍睹。除此之外，他還貢獻了2次失誤和4次犯規。

全場比賽，薩卡基尼和湯普森，被青島的大外援約翰遜打得七零八落。防也防不住，籃板也搶不過。可以說，北控這場比賽的失利，最大的原因就是內線被欺負得體無完膚。

同樣是外援，但這倆人在約翰遜面前，卻成了待宰的羔羊一般，只能眼睜睜看著約翰遜一次次搶走籃板，造成犯規，得分不停。

連續三場比賽，薩卡基尼的得分分別為18、13和10分。可以看到，在第16輪打廣州那一場短暫的回暖之後，薩卡基尼的狀態又走上了下坡路。

這場比賽，孫悅和弗格已經竭盡全力，卻仍不能帶領隊伍取勝。這和薩卡基尼與湯普森的低迷發揮是密不可分的。

隨著比賽一場場打完，薩卡基尼的問題暴露得越發明顯。兼之北控正在試訓新外援。估計，薩卡基尼被取代的日子，已經不遠了。

實力備選：

同曦客場惜敗給了新疆，痛失好局。在楊又一次砍下了47分，帶領隊伍艱難前進的時候。

亞布塞萊又拖了後腿。

連續兩場比賽，他的低迷表現，都與楊的超神表現形成了鮮明的對比。手握三外援的優勢，卻還要把楊累死，實在讓人看不下去。

浙江客場大勝北京。但鄧蒙的表現，卻有些出人意料。全場他出戰39分鐘，竟然只得到了7分。雖然蘭茲博格和其他隊友表現太好，一直保持著巨大的領先。

但這得分如此之低，還是有些說不過去吧。

天津客場挑戰廣東隊。面對聯賽王者，蘭德爾好像完全亂了陣腳。全場15中2，三分7中0，只得到了13分，竟然還有6次失誤！

雖然實力本身就有差距，但是這樣的發揮，也實在太菜了些吧。
