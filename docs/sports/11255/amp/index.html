數據控丨小茲維二發得分率倒數第一，普娃姐妹這2個數據完全相同

天天要聞
2019-12-10

給大家介紹一位來自日本的網球迷，同時也是一名數據愛好者，喜歡通過各種數據的方式對網球運動員進行分析對比。他的數據範圍不是以比賽為單位，而是以分為單位。一張張精彩絕倫的圖表背後，是這位球友對於網球極限的熱愛和對數據的痴迷。

今天，先給大家帶來這位學者的兩張佳作。第一張是2019賽季頂級球員一二發得分率分布圖。

首先，我們先來看男子球員的一發得分率。2019賽季，只有6名男子球員的一發得分率來到80%以上，沒有一人高於85%。拉奧尼奇與卡洛維奇佔據前兩名的位置，伊斯內爾排在第三，奧帕卡與奎雷伊幾乎並列第四，而特松加排在第六名。前五名是典型的大炮選手，唯一的例外是法國人特松加。

克耶高斯、斯特魯夫和凱文·安德森一發得分率都是79%，費德勒排在第10名，一發得分率在78.5%附近。排在費德勒身後的是貝雷蒂尼，義大利帥哥的一發得分率高達ATP第11名。德約科維奇、納達爾和總理的一發得分率都是不到71%，小德＞納達爾＞梅德韋德夫。令人感到非常意外的是，蒂姆與卡恰諾夫的一發得分率都不到74%，他們甚至比不過小威的75%。

弗格尼尼的一發得分率為69%，和薩巴倫卡相當，怪不得今年中網搶七6-1領先庫庫什金結果打成6-6。施瓦茲曼的一發得分率只有不到68%，與本西奇持平。男子世界前100中一發得分率最差的是西崗良仁，一發得分率只有63%，甚至比不過絕大多數世界前100的女子選手，就連奧斯塔彭科都在西崗良仁之前。

我們接下來再看女子球員的情況。塞蕾娜·威廉姆斯的一發得分率高達75%，鶴立雞群，比第二名高出2個百分點。我們剛剛提到，迪米特洛夫、卡恰諾夫、蒂姆、德米納爾和戈芬等人的一發得分率都不如小威。

普利斯科娃姐妹與巴蒂三人幾乎並列第二，一發得分率都接近於73%；格爾格斯與貝爾滕斯分列第五第六，稍稍高於72%；大坂直美一發得分率為72%，排在身後的科維托娃稍稍高於71%，凱斯和維基奇兩人高於70%；在過去一年大放異彩的安德萊斯庫一發得分率稍稍高於66%。

值得一提的是，奧斯塔彭科一發得分率稍稍高於65%，領先哈勒普的64%，真是意想不到。而科貝爾一發得分率為63%，這也可以看出她的糟糕狀態。在這份表單肉眼可見，卡薩金娜一發得分率排在世界前一百的倒數第四位，而二發得分率排在倒數第一，這就是如今俄羅斯人世界排名從世界前10降到69名的原因。

接下來我們在來看二發得分率。納達爾以接近60%的得分率排在榜首，費德勒次之。伊斯內爾二發得分率在57-58%之間，小德以57%排在第四，拉奧尼奇排在第五，盧布列夫排在第六。以上6人二發得分率都在56%以上。剛剛我們提到蒂姆一發得分率不如巴蒂，在二發得分率方面，蒂姆寶寶以55%排在ATP第10，高於總理梅德維德夫。

我們先前提到的一發得分率最低的西崗良仁，二發得分率為54-55%，領先西西帕斯、沙波瓦洛夫和錦織圭一眾好手；而一發得分率高於80%的特松加，二發得分率只有不到52%，相差太大。

嗯呢，小茲維列夫二發得分率剛剛高於44%，是世界前100名二發得分率最差的男子選手。今年他在法網對陣德約時，發出了罕見的7發失誤，成為雙誤小王子。

女子方面，巴蒂以不到52%的二發得分率排在第一，孔塔次之。科維托娃排在她們身後，二發得分率不到50%；一發得分率最高的小威廉姆斯，二發得分率稍稍高於49%，而安德萊斯庫不到49%，大坂直美不到47%，薩巴倫卡接近於46%，斯維托麗娜無限接近45%，莎拉波娃高於43%。

普利斯科娃姐妹不僅一發得分率相近，就連二發得分率也排在一起（卡普甚至比姐姐還低），那麼探尋這兩人成績差距的任務，只能放到下期回球得分率當中。奧斯塔彭科發球局鬆動的原因揭曉，二發得分率只有不到41%，排在世界前一百倒數第五，當然，這得益於她的雙誤能力。卡薩金娜以 39%排在倒數第一。

這位數據控專家引進了一條曲線，那就是一二發得分率相加為120%。絕大多數男子網球運動員都在這條曲線右側，但是也有少數人在這一曲線左下方，比如小茲維列夫、弗格尼尼、施瓦茲曼和西崗良仁，其中弗格尼尼位於最中央的位置。經過計算之後，拉奧尼奇一二發得分率相加最高。

女子網球運動員只有4人位於這一曲線右上方，分別是小威廉姆斯、科維托娃、巴蒂和格爾格斯。所以，這一日本數據控又針對女子球員製造出一個全新的圖表。

在第二張圖表中，這位數據控引進了6條曲線，縱軸依然是二發得分率，橫軸是一發得分率，但是為了更加適合女子球員的現狀，他將範圍分別縮小35%-55%和50%-80%。

我們可以看出，紅點標記的世界前十球員一二發得分率相加都在110%以上，哈勒普最接近於110%的曲線。高於120%的4名球員中只有格格一人不在世界前十的行列，當然這得益於她的爆炸性發球。所以，科娃、巴蒂、小威、格格這4人，我們都可以稱之為大炮型選手。

排在110%-120%之間的大多數由黃色和綠色標記，所以我們可以得出結論，只有當一發二發得分率相加大於110%時，才有希望躋身於世界前50名。我國選手王薔，正好位於110%曲線的中點處。

另外三條曲線更為有趣，分別是一發得分率-二發得分率為10%、20%和30%，這可以反映出球員一二發之間的差距。在這時，普利斯科娃姐妹再度現身，二人相依相偎坐落在於30%的曲線旁邊，要怪就只能怪她們一發太好了。

排在20%和30%之間的大多數都是發球高手，比如大坂直美、普娃姐妹、凱斯、薩巴倫卡、科娃、巴蒂等人，但是唯二的例外就是奧斯塔彭科和莎拉波娃，因為她倆的二發能力實在是太差了。

如果我們把男子球員放到這個圖標里，可以看到小茲維列夫、帕爾雷和奎雷伊是3個僅有高於30%的球員，不管他們的一發有多強，二發能力都必須要提高。（來源：網球之家 作者：楊過）
