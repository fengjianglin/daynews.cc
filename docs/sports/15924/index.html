英國記者透露，穆里尼奧離開曼聯後，從未考慮過齊達內剩下的球隊

2019-12-14 2,698條評論 3,935次閱讀 537人點贊

北京時間12月14日，熱刺的一位球迷在社交平台上透露，穆里尼奧給一位名叫約翰-凱斯勒的熱刺球迷寄去了親筆信，信件的內容非常感人。這位球迷在前不久確證了重病，而且他的病情很不樂觀，穆帥在這份親筆信中，鼓勵了該球迷戰勝病魔。那麼穆里尼奧是如何得知球迷重病的呢？原來穆帥在訓練比賽的間隙，他最愛做的一件事情就是閱讀球迷的來信，他在閱讀熱刺球迷給他的信件中，得知了這位名叫約翰-凱斯勒的球迷得了重病，於是他就特地給這位球迷回信鼓勵了他。這一舉動說明了穆里尼奧無論去哪一傢俱樂部執教，他都把球迷們擺在很重要的位置，英國天空體育記者Fabrizio Romano也就此事讚揚了穆帥的品格。

這位天空體育記者還透露了一件比較重要的事情，關於「皇馬試圖在穆里尼奧簽約熱刺之前，胡截穆帥」的傳聞並不靠譜，因為穆帥一直嚮往著英超球隊：「穆里尼奧在離開曼聯後，從未考慮過齊達內剩下的球隊。」Fabrizio Romano是穆帥在天空體育擔任解說期間結識的好友，他對於穆里尼奧的言論還是有可信度的。穆里尼奧與熱刺簽約之前12小時發生的故事，一直是西班牙體育頻道熱衷報道的話題。而西班牙記者那邊的一切言論，指向了皇馬高層早就不支持齊達內在帥位上了，儘管齊祖帶領球隊在賽季中期進入了穩定期，但是「老佛爺」仍然牽掛穆里尼奧。所以在穆帥正式重回執教崗位之前，在西班牙那一塊兒，他接班齊達內的傳聞沒有斷過。

這回天空體育對記者正式闢謠了這件事情，Fabrizio Romano的意思很明確了，他在幾個月的時間內與穆里尼奧一起工作，他們這些英國記者比西班牙記者更清楚穆里尼奧的職業需求。穆帥在本賽季開始之前，選擇與英國天空體育簽解說約（後來因為解說太精彩，他又參加了不少周中足球節目的錄製），當時該體育頻道的工作人員就說了，為什麼穆里尼奧選擇在重新回到英超之前，就職於天空體育？這一舉動就是說明了穆帥把執教生涯的目標還是放在英超，他在離開曼聯後，從未考慮齊達內剩下的球隊，當然也包括其他的西甲球隊。

筆者在此想要提一件事實：在穆帥重新開始執教生涯之前，關於他最熱烈的傳聞並不是西甲，也不是英超，而是德甲，為什麼呢？因為穆里尼奧多次在公開場合承認他正在學習德語，並且他在天空體育周中的足球節目中，多次談到德甲的比賽。恰好德甲的老大老二本賽季都在聯賽中遭遇了各種不順，這被人認為——穆帥很有可能去執教拜仁或者多特。然而現在看來，穆里尼奧一直「情定於英超」，他屢次暗示自己於德甲球隊有關係，實際上也是「煙霧彈」。

有些人可能對「穆里尼奧從未考慮過齊達內剩下的球隊」這一點有所懷疑，因為當穆帥執教皇馬的時候，齊祖還是皇馬青年隊主帥，兩人的合作還算愉快，為什麼之後他們的關係會急轉直下呢？他們之間的分歧發生在齊達內希望皇馬一線隊簽下自己的兒子，而穆帥在位期間就是不同意，因為穆里尼奧認為齊祖大公子不具備混跡於豪門一線隊的實力。這件事情不算大，但是這讓穆里尼奧一直「不待見」齊祖的為人處世方式，所以他不願意去接手齊達內剩下的球隊。
