巴薩尋槍太難！33歲齙牙蘇無人能替，他是梅西最佳搭檔

近幾個轉會窗口我們都會聽到有關巴薩尋槍的新聞，烏拉圭人蘇亞雷斯年事已高，巴薩高層計劃著為烏拉圭人離開巴薩後提前做準備，但幾個轉會窗口已經過去，作為蘇亞雷斯的接班人的確不好找！烏拉圭人自加盟巴薩以來堪稱是梅西的最佳搭檔，無論是內馬爾還是格里茲曼在與梅西產生的化學反應中都沒有烏拉圭人穩定，蘇亞雷斯與梅西的默契程度以及比賽的穩定性是無人能比的。儘管已經33歲了，但齙牙蘇仍無人能替！

自從2014年夏天登陸諾坎普，蘇亞雷斯便成為巴薩隊內表現最穩定的球員之一，迄今為止烏拉圭人代表巴薩打進190球，本賽季代表巴薩出戰21場打進13球並奉獻8次助攻，蘇亞雷斯隊內的地位不言而喻！此前與巴薩傳出緋聞的有奧巴梅楊、勞塔羅-馬丁內斯以及本賽季紅得發紫的小將哈蘭德等人，巴薩方面希望這些人能夠在諾坎普球場完成像蘇亞雷斯一樣的壯舉，但綜合各方面因素這些傳聞的轉會對象並沒有降臨諾坎普！

巴薩尋槍太難！隨著時間的推移，曾經球隊的功勛球員不是已經退役就是離隊亦或是年齡增長狀態出現起伏，但在巴薩隊內梅西與蘇亞雷斯卻始終保持著穩定的輸出。在當今足壇中鋒球員中，萊萬多夫斯基、蘇亞雷斯、凱恩以及伊卡爾迪等人實力超凡脫俗，但巴薩想要真正尋找一名適合自己球隊體系的中鋒球員卻是難上加難。蘇亞雷斯在巴薩的攻擊線上扮演的並不只是攻城拔寨的角色，烏拉圭人更是起到為球隊穿針引線的作用。

在蘇亞雷斯與梅西的連線中，二人經常上演投桃報李的好戲！上輪西甲聯賽對陣阿拉維斯的比賽中33歲的蘇亞雷斯憑藉一己之力幫助球隊獲勝，1球3助攻的數據是他交出的成績單。眼看著冬季轉會窗口即將開啟，巴薩高層勢必再次在轉會市場上開始尋槍，一直有著濃厚興趣的勞塔羅-馬丁內斯本人表示留在國米讓巴薩夢碎！此前有消息稱巴薩希望能夠將阿森納隊長奧巴梅楊帶至諾坎普，雖然阿森納表現不佳，但加彭人的狀態似乎並沒有因為槍手的整體狀態而受影響。

如果能夠引進奧巴梅楊以目前的狀況來看，似乎也是不錯的選擇，但想要真正找到一名能夠發揮出和蘇亞雷斯一樣的球員幾乎不可能！在現今足壇誰能夠像蘇亞雷斯一樣，能在和梅西的配合中發揮出最大威力，你看好誰？歡迎在評論區留言討論。
