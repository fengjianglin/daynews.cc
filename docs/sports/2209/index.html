瘋狂3-1！不敗球隊僅剩1支，國米隔空收大禮，意甲或徹底變天

12月8日，意甲聯賽第15輪結束了一場焦點大戰，尤文圖斯客場1-3不敵拉齊奧，遭遇了本賽季首場失利。尤文輸掉這場比賽之後，歐洲五大聯賽不敗球隊僅剩1支。本輪戰平的國際米蘭則是隔空收穫大禮，原本可能被反超的他們，反倒是拉大了分差，意甲徹底變天的可能性越來越大。

與拉齊奧的這場比賽，C羅與迪巴拉依舊搭檔鋒線。比賽進行到第25分鐘，C羅的進球幫助尤文圖斯首開記錄，但是拉齊奧在傷停補時階段扳平了比分。下半場比賽，尤文圖斯沒有太大作為，拉齊奧則是攻勢如潮，在米林科維奇反超比分幾分鐘之後，因莫比萊還射失了一個點球。憑藉凱塞多最後階段的進球，拉齊奧主場3-1擊退強敵。

尤文圖斯輸掉這場比賽之後，整個意甲已經沒有不敗球隊，五大聯賽的不敗球隊也僅剩1支，那就是英超的利物浦。客場挑戰伯恩茅斯的利物浦，延續了自己強大的攻勢，張伯倫、凱塔和薩拉赫的進球幫助他們3-0大勝。贏下這場比賽之後，利物浦的戰績變成了15勝1平，積分也上升到了46分，比曼城多了足足14分，本賽季的懸念或就此終結。

尤文圖斯的失利，最大的獲益者自然是國際米蘭，他們與羅馬的比賽率先進行。在這場比賽中，主場作戰的國際米蘭始終沒能找到破門機會，最終0-0與對手握手言和。戰平之後，國米的積分為38分，球迷非常擔憂他們在榜首剛剛待了不久就要讓出寶座。不過，尤文圖斯卻輸了一個1-3，國米不僅繼續領跑，優勢還變成了2分。

近幾個賽季，國際米蘭一直試圖挑戰尤文圖斯的地位，但總是功敗垂成，老婦人在意甲的統治地位越來越穩固。本賽季，國際米蘭再度進行調整，孔蒂成為球隊主帥，而且引進了盧卡庫等人，各方面都有所提升。相比國際米蘭，尤文圖斯則不如以前流暢，遇到的困難越來越多。國際米蘭本賽季終結尤文的統治，可能性變得越來越大。
