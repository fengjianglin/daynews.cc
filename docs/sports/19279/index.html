91分鐘神奇絕殺！英超爭四形勢：穆里尼奧下輪率隊贏球就進歐冠區

2019-12-16 1,059條評論 2,011次閱讀 894人點贊

英超第17輪結束了多場比賽，結果熱刺客場2-1絕殺狼隊，曼聯主場1-1戰平埃弗頓，目前排名第四的切爾西本輪輸球，熱刺距離前四又近了一點。下輪比賽，熱刺主場對陣切爾西！熱刺機會真的來了！

熱刺此役客場挑戰狼隊。開場不久，小盧卡斯右路上演個人秀，爆桿射門得手。在打進這球後，小盧卡斯已經在穆帥執教後的5場英超比賽中打進了3球，自2018年8月以來首次在連續兩場英超比賽中都能收穫進球。第36分鐘，戴爾禁區內接隊友傳球後射門擊中立柱，熱刺錯失機會。然而狼隊就是狼隊，第67分鐘，希門尼斯送出助攻，特拉奧雷得球後推進隨後一腳世界波破門，狼隊1-1熱刺。

就在球迷認為要平局收場的時候，絕殺又來了！傷停補時階段，熱刺角球機會，埃里克森罰出來到了禁區中路，維爾通亨搶點頭槌入網，熱刺2-1絕殺對手！穆里尼奧再次拿到了極為關鍵的3分！

同時進行的一場比賽。曼聯主場對陣埃弗頓。第36分鐘，埃弗頓獲得角球機會，德赫亞在勒溫的干擾下未能擊到皮球，林德洛夫在毫無準備的情況下將球撞入自家大門。曼聯已經在英超中連續12場比賽丟球，這也是他們自1971年以來的最長紀錄。曼聯上次在英超中零封對手還要追溯到三個月前，當時他們在主場1-0擊敗了萊斯特城。下半場第76分鐘，格林伍德低射破門追平，但曼聯依然無緣勝利！

目前英超利物浦老大，萊斯特城老二，曼城老三的位置難以動搖的情況下，「爭四」值得關注。然而切爾西本輪0-1爆冷輸給伯恩茅斯，看上去有後勁的曼聯也都沒有拿下比賽，爭四集體「剎車」，給了熱刺機會！最新的積分榜顯示，切爾西29分，熱刺26分，曼聯25分，切爾西的壓力來了，熱刺的機會也來了！

下輪比賽，熱刺主場對陣切爾西！這正是熱刺沖四的關鍵戰，如果能夠擊敗對手，熱刺有望直接來到積分榜第四的位置！那麼對於穆里尼奧來說，就太神奇了！
