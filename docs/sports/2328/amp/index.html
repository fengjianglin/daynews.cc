不過13連勝後，隨著塔利斯卡的回歸，兩位外援前鋒埃爾克森與塔利斯卡的化學反應不太成功。最後3輪塔利斯卡回到首發，而埃爾克森作為替補最終讓恆大3連勝，拿到這一個寶貴的中超冠軍。

下一個賽季恆大的目前相信不會只是中超冠軍，隨著埃爾克森的歸化，以及後防布朗寧的歸化，恆大的陣容可是空前強大。不過本賽季的中超過後，視乎卡帥還是沒有找到埃爾克森與塔利斯卡最佳化學反應。

不過球迷們不需要擔心，恆大的另外一位大腿已經可以參加比賽，他就是高神高拉特。下一個賽季高拉特將會成為恆大的歸化球員，因此他也不佔用外援名額，而高拉特與埃爾克森、塔利斯卡、保利尼奧都有過合作。作為一名正印的前腰球員，高拉特不僅個人進攻能力強，對於組織能力更有一手。

雖然保利尼奧是一位頂級球星，但是他並不善於持球推進，因此埃爾克森與塔利斯卡沒法產生化學反應也很正常。當高拉特回歸以後，埃爾克森與塔利斯卡同時出現在鋒線，而保利尼奧可以回到他最適合的後腰位置。恆大這樣的進攻組合，相信在亞洲也很難找到能與之一決高下的球隊！
