「英超」不止是球場上，在轉會市場上利物浦也遙遙領先

天天要聞
2019-12-21

北京時間12月19日，利物浦官方宣布引進薩爾茨堡紅牛前鋒南野拓實的交易已經完成，這位日本前鋒已經通過了體檢，2020年1月1日他將正式成為利物浦球員。尤爾根·克洛普說南野拓實是一名「盡自己最大努力幫助別人」的球員。

這筆交易的達成，會讓利物浦的競爭對手們比紅軍在積分榜上領先10分更加擔心，因為他們意識到，英超領頭羊進一步得到加強，他們的優勢更大了。

克洛普對利物浦足球總監邁克爾·愛德伍茲大加讚揚，因為他及時發現了解約金只有微不足道725萬英鎊的南野拓實，並且他與薩爾茨堡紅牛主席克里斯托夫·弗倫德的良好關係也起到了至關重要的作用，使得紅軍完成了這筆低價格高質量的轉會交易。

利物浦足球總監邁克爾·愛德伍茲

如果說愛德伍茲在這筆交易之前有什麼業績值得一說的話，首先要說的就是從南安普敦轉會而來的薩迪奧·馬內，馬內現在已經成為世界上最好的球員之一；另一份不錯的交易就是納比·凱塔，凱塔的在利物浦的表現還有很大上升空間。

在克洛普的任期內，利物浦還沒有出現一筆稱得上是失敗的轉會。多米尼克·索蘭克和洛里斯·卡里烏斯在利物浦雖然稱不上是成功，但是他們的轉會價格都在400萬英鎊左右，在這個年代基本上可以忽略不計。當初以7500萬英鎊的價格引進維吉爾·范戴克讓一部分人產生疑問，但是現在再看看，范戴克的身價起碼可以翻一倍。穆罕默德·薩拉赫也是如此，當時利物浦為他支付了3800萬英鎊，現在看來簡直少得離譜。還有他們以800萬英鎊的價格簽下安德魯·羅伯遜，這筆簽約的價值起碼漲了10倍。

在利物浦進行這些神操作的同時，曼聯也一直活躍在轉會市場中，他們揮舞著大把的鈔票簽下了阿萊克西斯·桑切斯、內馬尼亞·馬蒂奇以及羅梅盧·盧卡庫等人，用高昂的代價引進了萊斯特城第三好的中後衛；最後的結果就是艾德·伍德沃德掃清了那些一直想讓他在俱樂部設立足球總監的障礙。然後在看到埃爾林·哈蘭德在歐冠聯賽中的精彩表現後，又「發現」了一位身高1.91米的進球機器，開始考慮應該在他身上花多少錢。說實話，這根本就稱不上是轉會政策，尤其是你居然忽略了同一支球隊里潛在的只有725萬英鎊的交易。

曼聯CEO艾德·伍德沃德

英足總的一位消息人士近日向《The Athletic》網站透露：「麥克·戈登（芬威集團主席，利物浦大股東）對邁克爾·愛德伍茲的尊重和重視是無法量化的。」把這位體育總監在利物浦的地位和克洛普相提並論。和英超其他體育主管比起來，這確實無法反駁。也許看看愛德伍茲為利物浦簽約球員的名單會更直觀一些。

  • 薩迪奧·馬內——3700萬英鎊，南安普敦
  • 洛里斯·卡里烏斯——400萬英鎊，美因茨
  • 喬爾·馬蒂普——自由轉會，沙爾克04
  • 拉格納·克拉萬——400萬英鎊，奧格斯堡
  • 喬爾吉尼奧·維納爾杜姆——2500萬英鎊，紐卡斯爾聯
  • 多米尼克·索蘭克——400萬英鎊，切爾西
  • 穆罕默德·薩拉赫——3800萬英鎊，羅馬
  • 安德魯·羅伯遜——800萬英鎊，赫爾城
  • 亞歷克斯·奧克斯拉德-張伯倫——3500萬英鎊，阿森納
  • 維吉爾·范戴克——7500萬英鎊，南安普敦
  • 納比·凱塔——5300萬英鎊，萊比錫紅牛
  • 法比尼奧——4000萬英鎊，摩納哥
  • 謝爾丹·沙奇里——1300萬英鎊，斯托克城
  • 阿利松·貝克爾——6700萬英鎊，羅馬
  • 阿德里安——自由轉會，西漢姆聯
  • 南野拓實——720萬英鎊，薩爾茨堡紅牛

而阿森納的勞爾·桑內利和埃杜完全無法達到愛德伍茲的高度。今年夏天他們以7200萬英鎊的高價從里爾簽下了尼古拉斯·佩佩，但這並沒有像愛德伍茲的轉會一樣成為經典，在八卦媒體的評論中也很快就銷聲匿跡了。

勞爾·桑內利和埃杜

切爾西的霸道女總監瑪麗娜·格拉諾夫斯卡婭以及技術顧問彼得·切赫將在即將開啟的冬季轉會其中接受考驗，他們聲稱希望在切爾西能和弗蘭克·蘭帕德有良好的合作關係，因此蘭帕德在引進球員方面擁有「很大的發言權」。不過考慮到過去幾年裡，藍軍主教練在轉會問題上都沒什麼話語權，所以對他們這種說法目前還持懷疑態度。

瑪麗娜·格拉諾夫斯卡婭和彼得·切赫

再說托特納姆熱刺，極具個性的何塞·穆里尼奧，公認英超最摳的老闆丹尼爾·列維，總覺得他們因為轉會選擇的問題或者缺少轉會的問題而引發爭吵只是遲早的事。

在今夏的轉會市場上，曼城的表現似乎不錯，但現在看來，他們在中後衛的位置上遲遲沒有出手實在是一個愚蠢的做法，並且在大衛·席爾瓦明年夏天離開後，球隊面臨著重大的調整，到時候還得兼顧防守的問題，接下來衛冕冠軍有的忙了。

何塞普·瓜迪奧拉

萊斯特城主教練布倫丹·羅傑斯和體育總監喬恩·魯德金的合作關係可能是最接近克洛普和愛德伍茲的了。他們在2018年簽下了卡格拉爾·瑟因居、里卡多·佩雷拉以及詹姆斯·麥迪遜等幾位不錯的球員，每個人的花費都在2000萬英鎊左右。今年夏天他們又以4000萬英鎊的價格買斷了摩納哥中場尤里·蒂勒曼斯，可以說是今年夏天最好的簽約之一。但是，萊斯特城的問題是，和其他英超「小」球隊一樣，他們無法像英超傳統六強那樣長期保留隊中優秀的球員，因此他們也無法在長期的穩定性上緊跟利物浦的腳步。

布倫丹·羅傑斯

雖然利物浦這一次簽下南野拓實和克洛普以往的轉會習慣有所不同，但是這讓他們再一次走在了其他球隊的前面。據報道這位日本國腳在安菲爾德簽了四年半的合同，這可能會讓他成為利物浦無比輝煌時期的一份子。而比起那些為了能立竿見影地解決問題而揮金如土的俱樂部，利物浦卻腳踏實地，一步一步建立起完整而又強大的體系，為了更長期穩定的成功打下基礎。對於競爭者們來說，遙遙領先的利物浦已經越走越遠。
