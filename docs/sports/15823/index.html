國羽第1冠！雅思2-0黃鴨奪賽季第7冠 中國包攬總決賽混雙冠亞軍

2019-12-15 287條評論 1,654次閱讀 467人點贊

北京時間12月15日，2019年世界羽聯總決賽進入到收官日爭奪，在首場混雙決賽的對決當中，國羽兩對王牌混雙王懿律/黃東萍與鄭思維/黃雅瓊展開德比大戰。世界第2的黃鴨組合是衛冕冠軍，他們去年在決賽擊敗雅思組合奪冠，如今連續第2年在決賽碰面。雅思組合打出一場神勇表現，直落兩局總比分2-0橫掃黃鴨組合奪冠，也是斬獲他們的賽季第7座混雙冠軍，黃鴨組合未能衛冕收穫亞軍，國羽也是包攬混雙冠亞軍，以及斬獲今年總決賽國羽的第1冠。

作為國羽在5個單項當中，如今最具統治力的單項，混雙是國羽在世界羽聯各站賽事當中，今年合計奪得冠軍數最多的一個單項。儘管世界第1的雅思組合在下半程表現有所疲軟，並未能夠重現去年的超級統治力，但他們賽季此前也已經奪得6冠，尤其是在世錦賽完成衛冕2連冠，斬獲今年世錦賽國羽的唯一一座冠軍。

至於世界第2的黃鴨組合也是迎來爆發一季，他們賽季此前同樣是收穫6冠，在雅思組合下半程出現起伏情況下，黃鴨組合繼續扛起國羽的混雙重任，因而兩對國羽王牌混雙賽季已經合計奪得12冠。

在今年的總決賽當中，黃鴨組合以4連勝強勢打進決賽，而雅思組合則是小組賽首場爆冷輸球後，隨後3連勝打進決賽。國羽的如此兩對王牌混雙順利會師決賽，並且去年同樣是會師決賽，而去年的結果是黃鴨組合擊敗雅思組合奪冠。

如今國羽兩對世界前二的王牌組合，連續兩年在總決賽的混雙決賽對決，雙方過去12次交鋒對決，雅思組合以10勝2負處於絕對上風。本場首局比賽雅思組合開局慢熱0-4落後，但很快反撲追到6-6平。黃鴨組合再得2分重新奪得領先優勢，但雅思組合瘋狂打出一波12-2攻勢，已經將領先優勢擴大到18-10。儘管黃鴨組合有所反撲追擊，但雅思組合還是輕鬆以21-14拿下首局比賽，取得總比分1-0的領先優勢。

進入到第二局比賽當中，雅思組合在開局短暫1-2落後，很快就連續進攻打出一波7-1攻勢，強勢以8-3反超比分。黃鴨組合勉強追回1分後，雅思組合繼續用犀利的進攻主導比賽，也是逐步將領先優勢擴大到11-4。進入到後半局比賽，雅思組合繼續壓制黃鴨組合，比分也是繼續拉開到14-6，雅思組合已經基本掌控局勢取得19-8領先奠定勝局，哪怕之後遭遇黃鴨組合頑強追擊，依然以21-14再贏一局，最終強勢連贏兩局以總比分2-0橫掃隊友奪冠，斬獲他們的賽季第7座混雙冠軍。
