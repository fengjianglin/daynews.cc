談起足球明星，現役我們最熟知的恐怕就是C羅和梅西了吧，他們兩個幾乎壟斷了近幾年來足球界的所有的個人獎。今天主要講講C羅，喜歡看足球比賽的朋友對C羅都比較了解吧，C羅是葡萄牙著名的國家運動員，在去年C羅從西甲的皇家馬德里轉會到意甲的尤文圖斯隊，目前就在尤文圖斯效力。

C羅1985年2月5日出生於葡萄牙馬德拉島豐沙爾，他的原名是叫克里斯蒂亞諾·羅納爾多，C羅在職業生涯期間，戰績驚人，憑藉其出眾的球技和速度，為其球隊創造過無數的輝煌，也曾先後打破過多次世界紀錄，並且創造了新的記錄。橫掃足壇各大獎項，連續多年獲得世界足球先生、金球獎、歐冠最佳射手、歐洲金靴獎等國際大獎。並且曾經連續十一次入選國際足聯年度最佳陣容。在世界足壇上，C羅的風光事迹，可謂是婦孺皆知。

c羅不僅球技高超，而且顏值還高，剛毅、清爽的外形顯得不僅帥氣，而且魅力十足。自然深受粉絲們的喜愛。C羅作為一個國際巨星，經常會往返於各國之間，他也會來中國參加各種活動，他在中國的粉絲也是特別的多。曾經C羅在中國活動中的一個節目上，主持人準備了各行女星的照片讓C羅去評價，像賈玲，迪麗熱巴，劉雯等。

先是主持人拿出賈玲的照片，C羅表示搖搖頭，可能是覺得賈玲胖了點的緣故吧，接著拿出迪麗熱巴的，C羅表示還是不喜歡。要知道熱巴可是多少人夢寐以求都得不到的女神吶，C羅也沒多看幾眼，這可能是中國人和外國人的審美觀不同。不過當拿出第三張劉雯的照片的時候，C羅瞬間瞪大了眼，口裡念叨到：這女孩很有氣質。果然，巨星的眼光就是毒辣。

劉雯是一位國際超模，無論是從身材顏值，還是從穿衣打扮上來看，我個人覺得那種氣質簡直是完勝C羅現在的老婆喬治娜，怪不得C羅看到她時兩眼一直是直勾勾的。果然劉雯的高級氣質贏得C羅的中意，劉雯作為國際一線超模憑藉著自己的魔鬼身材和獨有的氣場收穫眾多的獎項和粉絲，作為超模的她，在國際排名上也是非常靠前的一位，難怪C羅這麼中意她。

其實不難看出C羅喜歡劉雯，因為在C羅的歷屆女友中，基本上都是非常有氣質的女人，而且劉雯不但有氣質，她身上散發出來的那種女神味道恐怕誰都抵擋不住吧。
