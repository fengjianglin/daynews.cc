韓德君受傷有驚無險！郭士強你長點心吧！

天天要聞
2019-12-21

最新消息稱，大韓在醫院經過一系列檢查，暫無大礙，球迷們也可以鬆一口氣了，但是我覺得應該給郭士強敲響警鐘。

比賽最後兩分多鐘，遼寧男籃落後20分，郭士強早就應該換下主力，給主力球員留有尊嚴。而且韓德君本場比賽出戰31分鐘，已經屬於超負荷運轉。韓德君的受傷就是因為體能下降，左腿絆右腿摔倒的，這在體能好時根本不可能發生。所以我覺得郭士強應該為此負有全責，一旦韓德君再次倒下，遼寧男籃季後賽都困難

郭士強是一個非常看重輸贏的教練，在我對郭士強的印象里，不管自己的隊伍領先多少分，不到最後兩分鐘，他是絕對不可能換掉替補。

無論輸成什麼樣，在他的世界裡就沒有放棄比賽這一說，遼籃不想輸可以理解，任何一個參加比賽的人都是這樣，更何況各隊的粉絲。但作為一個教練，就應該拿得起，放得下。

郭士強是應該找個理由下課了

雖然他是遼寧男籃的肱骨教練不錯，但是無論哪個教練都需要與時俱進，郭士強雖然沒有太大歲數，但他的思想卻是是CBA所有主教練中最保守的。我覺得郭士強應該調整下思路，對於遼籃的未來未必是壞事。
