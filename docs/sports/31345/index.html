中國足球趣史：一條後衛線上出了三任國家隊主教練

2019-12-24 3,910條評論 4,994次閱讀 1740人點贊

1972年的中國國家隊名單——

守門員：張業福、王俊生；後衛：王杭勤、相恆慶、戚務生、高豐文（隊長）、曾昭能、徐根寶、谷明昌；中場：李國寧、藺新江、遲尚斌、何成、楊禮敏；前鋒：張中威、李宙哲、王后軍、王積連、沈福儒、常萬誠

1972年的國家隊

領隊：馮德寶；副領隊：楊秀武、柳正炎；主教練：年維泗；教練：張俊秀

1972年國家隊後防線上出了三任國足主帥

戚務生、高豐文、徐根寶，後衛線上前後出了三任國家隊主教練，谷明昌也擔任過國家隊助理教練。中場藺新江、遲尚斌、楊禮敏都執教過甲A……

高豐文國家隊教練組

高豐文，1973年退役。退役後，他曾以足球專家身份赴南葉門和蒲隆地任教，1984年擔任中國少年隊主教練，1986年任國家隊主教練。1987年，他帶領奧林匹克隊闖入奧運會決賽圈，這是中國隊第一次在世界大賽中獲得出線權。1989年遭遇兩個「黑色三分鐘」，中國隊險些打進世界盃。1990年亞運會，中國隊敗給泰國隊未能晉級四強，因而辭職。

後曾開辦「高豐文足球學校」，培養出了杜震宇、王棟、陳濤、代欽華等優秀球員。

徐根寶

徐根寶，1976年退役後任教練。1979年任山西隊主教練，把這支隊伍送進了第四屆全運會的八強。他1982年任雲南隊主教練，1983年加入國家隊教練組，1987年任國家二隊主教練1989年任國奧隊主教練，1991年成為國家隊主教練，1992年離任。

甲A聯賽開始後，執教過上海申花、上海中遠、廣州松日、大連萬達，以及上海上港隊的前身上海東亞。執教經歷中培養出了高洪波、范志毅、武磊、張琳芃等得意門生。

戚務生的教練組

1976年，戚務生退役後開始從事教練員工作，1978年在索馬利亞國家隊執教一年，1983年任曾雪麟的助理教練，共同執教當時的國家隊。1987年擔任廣州白雲隊主教練，把這支降級隊伍晉陞到甲級隊(A組)。1989年，他執教大連隊時成績斐然，把甲級B組的大連隊一下晉陞為甲級A組的勁旅，並獲得1990年甲A聯賽第三名、1991年足協杯亞軍、1992年足協杯冠軍，他也被中國足協評為當年的”全國最佳教練員”。1994年-1997年，任中國國家隊主教練。

中國足球職業化開始後，執教過武漢雅琪、雲南紅塔、青島中能、天津泰達、廣州廣葯、梅州五華等球隊。
