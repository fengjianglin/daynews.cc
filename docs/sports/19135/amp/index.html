5天3個好消息！西甲華裔老闆心情大好，戰平皇馬後坐賓士笑著離開

天天要聞
2019-12-17

7天前，中國老闆陳雁升遠赴巴塞羅那，親自出馬只為儘快解決自己名下西班牙人俱樂部的困境。然而在這7天的時間裡，陳老闆現場督戰兩場比賽，西班牙人只取得1負1平的糟糕戰績，依舊看不到任何起色。與此同時，西甲另外一位來自於亞洲的富豪老闆林榮福，幾乎與陳雁升同一天抵達西班牙又同一天離開，但相比較於陳雁升來說，林榮福的這段「考察之旅」可謂是萬分愉悅。

眾所周知，新加坡華裔富豪林榮福是西甲百年豪門瓦倫西亞的大老闆，只不過在他執掌球隊的這幾年時間裡充滿了爭議。對於俱樂部，這位股票大亨向來是「散養」的態度，甚至屢次長時間不聞不問。根據西班牙媒體消息，林榮福上次在瓦倫西亞俱樂部內現身還是去年5月份，距離今天已經過去了整整19個月。

在這19個月的時間裡，林榮福大部分都是消失的狀態，只有本賽季剛開始時他毫無徵兆的做出了一項驚人的決定：解僱了與他意見不合的球隊功勛主帥馬塞利諾，扶植菜鳥教練塞拉德斯上任。客觀來講，林榮福的這一決定無疑是冒險且業餘的，但這是他的球隊，他想怎麼樣都可以。

神奇的是，換帥之後的瓦倫西亞並沒有受到太大影響，戰績甚至可以說更好了。5天前在荷蘭客場挑戰阿賈克斯的歐冠小組關鍵之戰，林榮福現場觀戰，最終瓦倫西亞1-0力克對手，時隔12年再次以小組第一的身份直接晉級十六強。

​

昨天凌晨西甲第17輪，瓦倫西亞主場迎戰皇馬，林榮福同樣在私人包廂中觀看了本場比賽。最終瓦倫西亞在全場劣勢的情況下先進1球，險些擊敗狀態正佳的皇馬。目前17輪戰罷，瓦倫西亞距離歐冠區只有3分之差，成績十分出色。根據《馬卡報》拍攝到的照片，林榮福在俱樂部主席摩西的陪同下走出球場，笑容滿面的坐上一輛賓士豪華車，直奔機場駛去，接下來他將飛回新加坡，繼續「隱身」。

值得一提的是，繼歐冠頭名出線以及戰平皇馬之後，林榮福在飛回新加坡的途中又收穫一個好消息。那就是在歐冠16強抽籤中，瓦倫西亞抽到了實力不強的歐冠新軍亞特蘭大，晉級八強的希望大大增加。要知道，距離瓦倫西亞上次殺入八強已經是2007年的事情了，本賽季他們有望重演12年前的壯舉。
