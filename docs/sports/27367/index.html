山東男籃“老大”第三次效力福建隊，替代勞森，臨時工難以轉正

2019-12-21 89條評論 1,609次閱讀 243人點贊

12月20日，福建男籃官方發布公告，在CBA常規賽第18輪客場對陣廣州男籃的比賽中，泰-勞森韌帶輕微拉傷，根據傷病評估，勞森將缺席近期的部分常規賽。福建男籃教練組及俱樂部經過慎重考慮，決定用普爾-杰特替換勞森。CBA官方已經同意福建隊外援更換的申請，這也是杰特第三次效力福建隊。

泰·勞森，1987年11月3日出生，美國職業籃球運動員，身高1.80米，司職控球後衛。本賽季勞森代表福建隊出戰18場比賽，場均上場36分鐘，可以貢獻27.6分5.1籃板7.4助攻，助攻數據排名聯盟第7；三分球命中率43％，排名聯盟第15。2017年勞森登陸CBA聯賽加盟山東隊，那個賽季賽季為山東隊效力46場， 場均可以貢獻25.5分4.6籃板6.5助攻2.2搶斷。上賽季中期再度回歸山東男籃，場均依舊可以得到26.5分8.8助攻3.9助攻。

杰特，1983年12月2日出生，司職後衛。2012年9月16日，山東男籃與杰特簽約，加盟的首個賽季便幫助球隊殺入總決賽，奪得了亞軍，場均拿到24.9分4.1籃板5.1助攻2.3搶斷的數據。杰特在山東共打了四個賽季，除了保持穩定的得分外，還用個人出色發揮帶動全隊整體發揮，是當時山東男籃當之無愧的「老大」。此後，杰特還效力天津隊兩個賽季。

2018年11月2日，福建隊官方宣布簽約杰特，替換了手臂受傷受傷的小外援史密斯。常規賽期間，福建男籃曾短暫用芒福德替換杰特，但芒福德加盟福建男籃的三場比賽里場均只得到19分5.3籃板4.3助攻1.7搶斷，最終福建隊又不得不換回杰特，杰特也幫助福建男籃時隔多年再次打進了季後賽。上賽季，杰特共為福建隊出戰41場，拿到21.8分5.4籃板7.2助攻的數據。

本賽季，福建隊請來了同樣曾經效力于山東男籃的後衛勞森，杰特無緣續約。但是，因為勞森的傷病，使得福建隊得以與更加實用的杰特再續前緣。但這一次，杰特的身份確定是「臨時工」，只要勞森傷病痊癒，他基本上可以提前預判會被放棄。畢竟勞森在凱撒的麾下效力過一個賽季，更加適應凱撒的戰術思想。

但也或許會有意外發生，那就是杰特與球隊的磨合更加順暢，率領球隊打出連勝，就像現在的山東男籃一樣，雖然已經簽約新外援哈里斯，但因為梅斯在最近幾場發揮出色，山東男籃取得了8連勝，因此哈里斯也只能在場邊觀戰，獲得不了報名的機會。
