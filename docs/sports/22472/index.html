有誰注意到，郭士強暫停時賀天舉看芬森的眼神，山東記者犀利發問

2019-12-20 2,103條評論 4,297次閱讀 1998人點贊

客戰山東男籃以20分輸球告終，遼寧男籃在青島被絕殺又在濟南被打爆，更可氣的是韓德君還是受傷離場。這個夜晚只有一個主角，那就是哈德森。

首節3次犯規的哈德森在下半場迎來了爆發，第三節16分，第四節18分，全場41分。作為替代者再看看史蒂芬森，全場20投8中拿到了18分，看似還不錯的數據，其實有12分是在山東男籃末節大比分領先時候放了史蒂芬森，讓他獲得了幾次反擊上籃的機會。三分球極其不準卻依舊盲目自信出手，最終投出了10中1的數據，哈德森則是12投5中。

賽後發布會郭士強帶著史蒂芬森出席山東記者很犀利，給郭士強提了一個問題評價一下哈德森。郭士強嘴上不情願也還是要誇一誇哈德森，然而畢竟哈德森是山東男籃的人了。

史蒂芬森與哈德森的表現並不是這一場比賽有差距，整個賽季都有巨大的差距，首當其衝的就是態度。

賽後山東記者直接問了史蒂芬森一個問題，你的表現達到預期了么？你接下來要怎麼做。

史蒂芬森接受採訪的能力顯然遠遠超出了他打球的能力，打起中國武術太極拳也是一絕。史蒂芬森躲開問題解釋說：我對自己還是非常有信心的。我現在肌肉有些拉傷，處於比較艱難的時刻，目前投籃也不是很好。但我是一個非常堅強的人，我接下來要做的就是回到訓練場上，回到比賽場上，讓隊友更多參與到進攻和防守中來，打我們的團隊籃球。

不知道大家有沒有注意到，第三節比賽中史蒂芬森連續不合理投籃打鐵，山東男籃趁機連續反擊上籃將比分拉開到了10分以上，郭士強暫停時候賀天舉一個嫌棄的眼神送給了史蒂芬森，如果這就是團隊籃球，那可就危險了。
