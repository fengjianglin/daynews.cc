前女排隊員，性格最樸實無華，退役時連儀式都只有幾句話

2019-12-17 1,411條評論 3,083次閱讀 1019人點贊

作為一個運動員，大家覺得他們最在意的事情是什麼呢？小編認為一個運動員最開心的時刻就是站在領獎台的那一刻，那一刻他們會收穫到很多的鮮花與掌聲，並且自己辛辛苦苦的進入國家隊，每天任勞任怨的進行訓練，都是為了這一刻當獎牌掛在脖子上的那一瞬間，所有的辛苦都可以釋懷了。
　

　
頒獎典禮總會結束的，但是留給球迷們的回憶卻是難以磨滅的，雖然頒獎典禮作為一個冠軍運動員來說非常的榮耀，但並不是所有的運動員都能夠親身經歷這一榮耀時刻，除了頒獎典禮對於一個運動員最重要的應該就是退役儀式，運動員與普通職業不同，普通職業一般要等到五六十歲才可以正式退休，但是運動員由於他們的職業特殊性，一般在三十左右就會進行退役，有些人就會舉辦一個風風光光的退役儀式，給自己給球迷們留下永生難忘的回憶。
　　

令人印象深刻的應該就是前女排隊長惠若琪的退役儀式了吧，場面相當壯觀，很多球迷球員都去到了現場為她送別，但其實不是所有人都像惠若琪這樣能夠擁有一個盛大的退役儀式，曾經中國女排的徐麗雲，她的退役儀式就非常的簡單，就用簡短的幾句話結束了自己的退役儀式，可以說是相當樸實無華了。
　　

徐麗雲被粉絲們親切地稱為長城，就是因為有她的防守，中國女排防守變得堅固無比，但是即使有了這樣一個高調的外號，徐麗雲依然在女排中屬於一個很低調的人。在徐麗雲退役之後，他並沒有離開球迷們的視野，而是經常去給女排隊員們送溫暖，在比賽的時候經常可以看到徐麗雲在台下忙碌著，要不然就是忙著給球員遞水，要不然就是忙著負責安慰發揮不好的球員。
　　

其實，中國女排中有著很許許多多重情重義的隊員們，即使在退役之後也願意默默無聞的為中國女排付出，誰說女人之間的友誼就不如男人堅固呢？各位小夥伴們在日常生活中有沒有遇到一個默默付出不求回報的好隊友呢，歡迎在評論區說出自己的想法喲。
