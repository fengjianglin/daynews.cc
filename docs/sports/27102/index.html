英超爭2大戰出現巨大爭議！1.6億巨星跳水贏點球 VAR也沒用

2019-12-22 791條評論 2,109次閱讀 426人點贊

英超聯賽第18輪，一場焦點之戰在曼城和萊斯特城之間展開爭奪。由於目前兩支球隊距離榜首利物浦的差距比較大，所以本場比賽對於對陣雙方來說，更像是一場爭奪聯賽第二名的比賽。

本賽季萊斯特城的整體表現相當出色，本輪開打之前球隊的排名更是在曼城之上。不過整體實力佔據優勢的曼城在比賽開始之後還是佔據了場上的絕對主動。但是率先打破僵局的卻是萊斯特城。

瓦爾迪的一次超車衝刺幫助萊斯特城首開紀錄。不過落後之後的曼城很快就利用馬赫雷斯的個人能力扳平了比分。此後在上半場比賽最後時刻，場上出現了一次頗具爭議的判罰，也正是利用這次爭議，曼城反超比分。

比賽第41分鐘，斯特林禁區內突破倒地，裁判第一時間判罰點球。從慢鏡頭回放來看，斯特林和萊斯特城後衛的腳確實有輕微接觸，但是慢鏡頭給出的回放卻很像是斯特林踩到了對方的腳發生的倒地，而且倒地動作略顯誇張，完全有跳水的嫌疑。建立了一個球迷交流群。大家感興趣的可以加一下：acd33ds 添加的時候備註【足球迷】。群里不定時會給大家發一些福利。

在裁判判罰點球之後，萊斯特城球員也是向裁判示意斯特林是在跳水。但是經過VAR視頻裁判提醒，主裁判堅持了自己的原判。隨後京多安主罰命中，場上比分變成了2:1，曼城反超了比分。

需要指出的是，這已經不是本賽季英超VAR第一次出現爭議了。這個賽季在多場焦點之戰當中，VAR的使用和判罰都曾引起過巨大爭議。因為這件事，不少球迷還曾表態這個賽季英足總要保送利物浦奪冠。但是如今曼城也靠VAR獲利，不知道這次球迷們又該說英足總要保送誰了。
