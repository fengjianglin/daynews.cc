迎戰山西男籃賽前，李敬宇賽前加練，內心感慨萬千

2019-12-11 377條評論 1,478次閱讀 307人點贊

周四晚，山東男籃主場迎戰山西，對手是本賽季CBA最大黑馬，如今聯賽排名第四，而且汾酒陣中有山東男籃租借加盟後本賽季「大放異彩」的隊員劉冠岑，加之外線射手李敬宇從山西的租借回歸，讓這場比賽的關注度特別高。

本賽季從山西輾轉回歸山東之後，「小李飛刀」李敬宇場均貢獻9.1分，是球隊本土的重要得分點，尤其是外線表現的穩定性，早已是得到了齊魯球迷的認可，「這些年改變最大的就是心態吧，打了這麼多年聯賽，尤其是什麼都經歷過以後，才明白了需要用平常心去對待比賽，今後我也會珍惜每一場比賽。」

賽前一天訓練時候，李敬宇還專門到力量房進行了加練，對於迎戰自己的老東家山西隊，李敬宇感慨，其實自己就是平常心，「就是很平常的一場常規賽而已，山西隊今年打得的確不錯，而且兩個外援一個會組織，一個有得分，都和球隊融入的很好，作為我來講，明天平常心去比賽，發揮出自己應有的水平就行了。」

作為賽季前交易互換的對象，山西隊中的劉冠岑本賽季的表現同樣耀眼，場均7.3分，剛剛在主場惜敗新疆，五連勝戛然而止，「我原本可以做得更好！」劉冠岑賽後有些自責。之前，對陣江蘇的比賽中劉冠岑拿下18分，對陣浙江貢獻17分。
