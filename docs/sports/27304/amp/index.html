冠軍教練的影響力！穆里尼奧在曼聯最欣賞的後衛續約成功

天天要聞
2019-12-21

在穆里尼奧還在執教曼聯時期，他就對熱刺中後衛托比-阿爾德韋雷爾德垂涎三尺（下簡稱托比），這名比利時國腳對抗能力強，技術出色，兼具後場出球的能力，要知道穆里尼奧選擇熱刺的重要原因，就是球隊的陣容吸引到了穆里尼奧。

不過，在穆里尼奧來之前，托比與熱刺的合同就將在明年夏天到期，顯然這位年滿30歲的中後衛也動了獲得榮譽的心思，曾在此前的採訪中表示，希望去一支更有競爭力的球隊，不過冠軍教練穆里尼奧的到來最終打消了托比離隊的心思。

在對陣伯恩茅斯的比賽中，托比連續兩次精準長傳，助攻阿里進球，同時也創造了另一次進球機會，讓更多球迷體驗到了一名優秀的「出球中衛」，對球隊的影響和重要性，也了解了在曼聯時期，穆里尼奧對托比的渴望是正確的。

在穆里尼奧接手球隊後，熱刺取得了5勝2負的成績，不僅拿到了2個客場勝利，3個主場比賽均取得了勝利，球隊也找回了重回聯賽前四的信心，托比也受到穆里尼奧的信任，7場比賽場場首發，全部打滿全場，是球隊中最重要的角色之一。

托比與熱刺續約到了2023年，也算是將職業生涯最黃金的時光都奉獻給了熱刺，希望他能與穆里尼奧一起取得一些冠軍，不枉兩人的互相欣賞，除了托比外，熱刺球員埃里克森的合同也將在明夏到期，此前埃里克森已經表明了態度希望自由轉會離隊，不過近期他也在接受丹麥球員工會採訪時表示，對自己無法登場比賽的情況感到無聊，同時也認可了穆里尼奧對球隊的改變，也許這位丹麥核心國腳也動了留隊的想法。

值得一提的是，根據每日電訊報的消息：穆里尼奧並不准備在冬窗對熱刺進行大變動，不過可能會在精簡陣容的前提下，補進一名替補中鋒和一名後衛，增強球隊的競爭力，而熱刺的下一場比賽是12月23日，在主場對陣切爾西的倫敦德比，而且這也是穆里尼奧首次對陣自己的弟子蘭帕德，屆時大家不要忘記關注哦。
