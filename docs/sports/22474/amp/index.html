開始裁員！穆里尼奧兜售熱刺兩大將 中場鐵腰受到中超追捧

天天要聞
2019-12-20

在穆里尼奧接手以後，熱刺的表現與成績與波切蒂諾時期可以說是判若兩隊。隨著冬窗的來臨，穆帥也將在上任後對球隊進行第一次改造。根據英國《電訊報》的報道，除了引援之外，穆帥希望在明年1月出售丹尼-羅斯以及萬亞馬兩名球員，以此達到精簡球隊陣容的目的。

在穆里尼奧初次執教切爾西時，他就承認喜歡自己的球隊只有22人：21名球員外加門將，他希望一線隊所有的主力球員都有一名替補，也希望所有的球員能夠在一線隊發揮作用。在接手熱刺後，球隊的陣容達到了25人之巨，這對於穆里尼奧來說顯然有些飽和，因此他認為可以對球隊陣容進行精簡。

《電訊報》指出，穆帥希望萬亞馬能夠在冬窗離開球隊，同時他也很樂意傾聽其他球隊對於英格蘭飛翼丹尼-羅斯的報價。目前埃里克森依然沒有與熱刺續約，他也很可能會在冬窗被球隊出售，以避免到了明年夏天熱刺面臨人財兩空的窘境。

目前28歲的肯亞後腰萬亞馬已經為熱刺效力了三年的時間，但自今年4月份以來，他便再也沒有為熱刺首發出戰過。據報道，德甲球隊柏林赫塔以及一些中超球隊對他有著很大的興趣。

羅斯的情況則有一些複雜，目前他與熱刺的合同還有18個月才會到期，此前他已經堅稱自己會履行完與熱刺的合同後再離開球隊。但在穆帥手下他顯然已經沒有位置，此前一個月穆里尼奧一直在用維爾通亨改打左邊後衛，並且取得了不錯的效果。與此同時，羅斯還需要加強自己的競爭力以確保可以參加明夏的歐洲杯，他希望在剩下的6個月的時間裡可以成為球隊的常規主力。
