最新一期新秀榜單：勇士新秀力壓莫蘭特排名第一，八村塁殺入前五

2019-12-06 0條評論 21次閱讀 0人點贊

摘要

北京時間12月6日，NBA官方公布了最新一期的新秀榜單，勇士隊的帕斯卡爾升至第1名，熱火隊的納恩升至第2名，上周排名第一的莫蘭特由於傷病情況降至第3，本周發揮出色的八村塁殺入第四，尼克斯隊的巴雷特繼續排名第五。

北京時間12月6日，NBA官方公布了最新一期的新秀榜單，勇士隊的帕斯卡爾升至第1名，熱火隊的納恩升至第2名，上周排名第一的莫蘭特由於傷病情況降至第3，本周發揮出色的八村塁殺入第四，尼克斯隊的巴雷特繼續排名第五。

第1名帕斯卡爾

​

上周帕斯卡爾排在新秀榜單的第4名，這一周帕斯卡爾表現繼續強勢，雖然勇士隊遭遇了4連敗，但帕斯卡爾表現極為的出色，對陣熱火他得到17分，對陣魔術他得到 17分6籃板2助攻，對陣老鷹他得到24分9籃板6助攻，對陣黃蜂他得到16分6籃板，效率非常的高，這4場比賽他場均得到18.5分5.5籃板，投籃命中率高達52.1%。

第2名納恩

​

上周納恩排名第三，最近的4場比賽，納恩場均得到10.3分，投籃命中率僅有29.3%，在與帕斯卡爾直接對決中，

納恩得到了15分4籃板2助攻的數據。

第3名莫蘭特

​

上周莫蘭特排名第1名，上周由於莫蘭特背部有傷，所以都沒有參加比賽，球隊也遭遇了兩連敗，至今為止莫蘭特場均18.6分和6.4助攻，都是本屆新秀中表現最為出色的。

第4名八村塁

​

上一周八村塁並沒有能夠排進前5，奇才對陣快船隊的比賽中，八村塁得到了30分9籃板的數據，創造了職業生涯的新高，最近的三場比賽他場均得到20.3分，命中率高達52%，更為可喜的是他對於聯盟節奏的逐漸適應，攻守兼備，未來可期。

第5名巴雷特

​

賽季至今巴雷特場均得到14.7分5.3籃板3.2助攻，巴雷特最為致命的就是他的投籃命中率，三分球命中率僅為31.7%，罰球命中率僅為53.3%，在上周對陣雄鹿的比賽中，巴雷特9中0得到二分，而尼克斯隊繼續輸球。

其他被提名的球員分別是：猛龍隊的特倫斯戴維斯、黃蜂隊的華盛頓、森林狼隊的卡爾弗、老鷹隊的雷迪什、鵜鶘隊的海耶斯。
