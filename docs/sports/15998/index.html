白岩松悲觀主義，董路對國足心存希望，到底誰更接近事實？

2019-12-14 161條評論 1,544次閱讀 208人點贊

最近中國足球成為各路媒體的熱議話題，國字型大小男足連輸比賽，再次引起了大家的廣泛議論。其中最為出名的還是白岩松最近在接受媒體採訪時，用一個40年老球迷的角度剖析了中國足球發展的弊端。而白岩松言論出來之後，董路立馬發表了不同意見，不僅炮轟了白岩松那些悲觀主義，還把自己對國足的希望談了一下。雖然兩人見解相反，但是他倆說的都沒錯，只是站在了不同角度，但是到底誰更接近事實呢？讓我們來分析一下。

白岩松對於中國足球整體是抱著一個失望的態度，在他的言論中大意就是中國足球現有的問題就是免費足球場地太少，而且踢球的孩子也少，這樣就沒法為青訓補充足夠的新鮮血液。還有一個問題就是管理人員不作為，足協並沒有制定長期的發展計劃並認真實行。對於白岩松的這套理論其實和廣大球迷心中所想基本相差不多，確實在現實生活中存在這樣的問題，很多孩子家長不選擇讓孩子去踢球，除了要學習以外，其實還有一大部分人是因為承擔不起高昂的費用。畢竟培養一個足球運動員在中國要花費天價的培訓費，一般家庭根本無力支付，就算培養出來，能不能踢出名堂還是一個未知數，到了後期還需要大量金錢來支撐。所以這才導致了中國踢足球的孩子不多，能踢出來的更少，最後導致為國出戰的國腳只能在有限的球員裡面選擇，水平一般就可以理解了。

董路覺得白岩松的這些言論是陳芝麻爛穀子的車軲轆話，並沒有什麼新意。在炮轟白岩松言論之後他也說出了自己對中國足球的見解與想法。在直播中他指出，中國足球現有問題並不是免費球場和踢球孩子少的問題，其實很多其他國家也做不到這點，但是人家足球照樣不錯。他認為孩子上補習班也花錢，甚至比踢足球還貴，但是照樣爆滿。董路認為不熱愛足球的孩子沒必要非讓他們踢球，只要那些喜歡足球的孩子，想踢球的孩子培養起來，還是足夠的。關鍵在於，有沒有能夠發現孩子天賦的平台，和給有足球天賦孩子施展才華的機會。然後董路又提到自己在做的足球小將，只要孩子有天賦踢球，他們不收錢，不簽約，隨時歡迎，孩子們也可以隨時離開。而且董路最後表示自己對20年後的中國足球充滿希望，白岩松那些悲觀的說法讓現在踢球的孩子們會感到泄氣。

其實白岩松和董路的言論都是有一定的道理，白岩松在生活中發現問題，在40年看球生涯中總結問題。董路通過自己實踐來解決問題，並且對自己所做的事情心懷希望。我們不能輕易評論任何一個人的觀點是錯誤的，因為他們說的都是中國足球發展路上的一些避不開的障礙。兩個人只是站在了不同的層面和角度來看待中國足球，所以才會出現這樣的分歧。網友既有支持白岩松的，也有贊成董路的，甚至還有人把董路的做法與徐根寶做對比。

至於說兩個人誰的觀點更接近事實，這一點其實很好來判斷，白岩松的出發點和角度都是站在了全國大環境下來看待中國足球，而董路雖說自己親力親為，用實踐行動來改變環境，但是如果大環境做不到改變，中國又會出現幾個董路呢？如果真正做到了遍地都是董路這個想法的人，或者出現很多與董路做著相同事情的人，那麼中國足球絕對可以發展起來，但是到目前為止也只有一個徐根寶和一個董路而已！
