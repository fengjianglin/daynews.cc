射門8對8！國足2比0贏球卻並不輕鬆，李鐵場邊搶鏡，他太想贏了

2019-12-19 1,853條評論 3,433次閱讀 2309人點贊

必須要贏的一場比賽，國足選拔隊的開局很完美，但過程卻並不順利。在第8分鐘就進球的情況下，國足選拔隊門前連續遇到險情，幸好劉殿座和門柱阻擋了對方的射門，力保球門不失。第70分鐘，董學升製造點球，張稀哲主罰命中，國足選拔隊才鎖定勝局。最終，國足2比0擊敗中國香港，拿到本次東亞杯的首勝，鎖定第3名。

本次東亞杯，國足以選拔隊的名義出戰，前兩場1比2輸給日本，0比1不敵韓國，這是真實實力的差距。第三場面對同樣兩連敗的中國香港，主帥李鐵賽前就亮出了態度，那就是必須贏球。從首發陣容的選擇看，梅方、金敬道傷停，姜至鵬停賽，李鐵大範圍調整首發。後衛線，李昂復出，他和于大寶搭檔中衛，吉翔和王燊超出任邊後衛。中前場李行和買提江後腰，張稀哲前腰，曹贇定、董學升和王子銘組成全新的進攻線。

從比賽的場面看，開場後中國香港很快回收防守，鏡頭顯示，他們擺出了631的陣型。面對鐵桶陣，國足必須要用好定位球，第8分鐘，張稀哲的角球傳中落點不錯，中國香港球員解圍出現了失誤，首次首發的右後衛吉翔機警的抓住機會，頭槌破門，為國足取得領先。早早的進球對國足來說更有利，中國香港只能是放棄密集防守，主動壓出來進攻，而他們的進攻，的確威脅到了劉殿座把守的球門。

第17分鐘，中國香港在右路打出連串的配合，國足的防守明顯有些顧此失彼，禁區內基奧雲尼的勁射被劉殿座撲了一下，球擊中門柱彈出，國足險些丟球。第35分鐘，又是基奧雲尼，在和隊友做了一個簡單的配合後，基奧雲尼直接選擇射門，注意力集中的劉殿座將球撲出。從半場的數據看，兩隊都是2次射門，可以看出差別並不大。

進入下半場的爭奪，李鐵用譚龍換下王子銘，國足的進攻還是邊中結合，而中國香港利用定位球製造威脅，陳俊樂亂戰的射門被于大寶擋出，十分危險。兩隊陷入比較膠著的局面，第65分鐘，買提江主罰角球傳中，禁區內又是吉翔搶點，只可惜他沒能形成射門，球被後衛破壞。

直到第70分鐘，國足擴大領先優勢，這次進球的是張稀哲。董學升禁區內接球後擺脫被放倒，裁判判罰點球，張稀哲主罰點球命中，國足2比0領先。就在這個進球後，李鐵用韋世豪換下了曹贇定。第80分鐘，于大寶策動攻勢，譚龍傳中，韋世豪的頭槌擊中門梁，十分可惜。比賽最後時刻，王燊超在一次防守中受傷，這時候國足的換人名額已經用完，王燊超一瘸一拐繼續堅持比賽。最終國足守住了2比0的勝果，拿到了東亞杯的勝利，這對李鐵來說很關鍵。從全場的數據看，國足是8次，中國香港也是8次，雖然贏了，但未來還是有很多需要提升的地方。
