切爾西將在周三英超快車的主場迎戰升班馬阿斯頓維拉，斯坦福橋球場將安排特別儀式歡迎這位前隊長特里回歸，盡顯有情有義。

據悉切爾西已安排特別的歡迎儀式，包括開賽前在大熒幕播放特里（John Terry）過往的精彩片段，細數特裡帶隊切爾西奪到的15項錦標。同時切爾西俱樂部方面安排了歡迎儀式，讓主場球迷賽前歡迎他進場，但當中首先需要得到阿斯頓維拉俱樂部方面的批准，現在由於身份尷尬，作為阿斯頓維拉助教的特里也未必會接受。除此之外，切爾西也會自發製造致敬橫幅或海報，向前隊長特里作出崇高的謝意。比賽方面，特里和藍軍主教練蘭帕德上賽季曾在英冠交鋒，阿斯頓維拉最終在升級附加賽中淘汰德比郡，這樣或許是某種程度上造就藍軍切爾西有機會乘虛而入，聘用蘭帕德回巢斯坦福橋執教。

效力切爾西19年的鐵血隊長特里，帶隊奪得過5次英超冠軍，2017/18年賽季效力過阿斯頓維拉，隨後掛靴轉任助理教練，周三將是首次以對手助理教練身份重返斯坦福橋。
