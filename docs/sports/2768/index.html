女排喜憂參半！郎平老東家掀翻南美豪門，朱婷率隊惜敗歐冠冠軍

2019-12-04 0條評論 20次閱讀 0人點贊

摘要

北京時間12月3日，女排世俱杯首個比賽日結束，中國兩支球隊喜憂參半，郎平的老東家恆大女排3-1掀翻南美豪門米納斯，而壓軸登場的天津女排在2-0領先的情況下慘遭逆轉，2-3惜敗歐冠冠軍諾瓦拉，拼盡全力的朱婷也無力回天。本屆世俱杯的競技水準非常高，因為參賽的八支球隊都大有來頭。

北京時間12月3日，女排世俱杯首個比賽日結束，中國兩支球隊喜憂參半，郎平的老東家恆大女排3-1掀翻南美豪門米納斯，而壓軸登場的天津女排在2-0領先的情況下慘遭逆轉，2-3惜敗歐冠冠軍諾瓦拉，拼盡全力的朱婷也無力回天。

本屆世俱杯的競技水準非常高，因為參賽的八支球隊都大有來頭。土耳其的瓦基弗銀行是衛冕冠軍，另一支土超豪門伊薩奇巴希是世俱杯常客。來自義大利的諾瓦拉是歐冠冠軍，埃格努領銜的科內利亞諾被看作是加強版的義大利國家隊。來自南美的海灘和米納斯也是老牌勁旅，米納斯更是上屆比賽的亞軍。咱們的天津和恆大就不用說了，本賽季都引進了超級明星，而且還有臨時引援，具備叫板頂級強隊的實力。

從分組情況看，恆大遭遇科內利亞諾和伊薩奇巴希，想要衝擊前四就要努力戰勝實力相對較弱的米納斯，經過四局苦戰，恆大做到了。四局比賽都咬得很緊，但恆大關鍵分把握出色，兩大外援科舍列娃和拉巴德捷耶娃勇挑重擔，本土球員也發揮出了超高的水準，臨時加盟的幾位年輕球員確實值得信任。收穫勝利的同時，咱們的本土年輕球員還得到了充分鍛煉，實在太有價值。

拿到這場勝利之後，恆大就要努力衝擊伊薩奇巴希和科內利亞諾，如果能再勝一場，進四強就非常有希望了。目前看來，恆大有希望扮演黑馬角色，去衝擊郎平帶隊時創造的世俱杯最佳戰績。當然，球員們不能被成績壓力束縛，在這個高水平舞台就是要努力發揮出自己的真實水平，通過比賽來積累經驗並得到全方位提升，這才是參賽的意義。

首個比賽日壓軸登場的是天津女排，她們的對手是諾瓦拉。雖然諾瓦拉是歐冠冠軍，但埃格努出走讓她們的整體實力明顯下滑，考慮到這個小組幾支球隊實力接近，每一場比賽都要全力爭勝，如果能拿下諾瓦拉，天津女排晉級四強的希望無疑更大了。在朱婷的帶領下，天津女排攻防兩端發揮出了較高的水準，臨時加盟的袁心玥也發揮了重要作用，她們以25-17、25-15拿下前兩局，本以為是一場輕鬆取勝，但之後卻出現了大逆轉，天津女排一攻成功率低，一傳及保障方面失誤較多，18-25、15-25、11-15，連丟三局的天津隊慘遭逆轉，無緣開門紅。

雖然惜敗諾瓦拉，但因為衛冕冠軍瓦基弗銀行的實力也有所下滑，海灘實力較弱，天津女排仍然有希望衝擊四強。接下來的第二比賽日，廣東恆大將迎戰奪冠大熱門科內利亞諾，而B組第二輪將在5日進行，天津女排迎來了寶貴的休整時間，該好好總結。
