2019年，哪個球星讓中國球迷吃最多檸檬？我選擇韓國18歲的李剛仁

2019年，哪個球星讓中國球迷吃最多檸檬？我選擇韓國18歲的李剛仁

回顧2019年，哪個球員讓中國球迷吃檸檬最多？我會把範圍縮小到東亞，因為他們的膚色跟我們一樣，身體素質跟我們完全一樣，他們能做到的，我們理論上也能做到，不同的在於足球環境，足球體系。而我會選擇韓國的球星李剛仁，因為他才18歲，代表一個國家青訓系統培養出來的頂級人才。18歲李剛仁以2000萬歐元的身價高居現役亞洲球員第三名，18歲的李剛仁成為世青賽中第二年輕的金球獎得主，第一名是當年18歲的球王梅西。

2019年，李剛仁爆發，高光，成名的時刻在波蘭世青賽，韓國男足獲得了世青賽亞軍的好成績，李剛仁以2球4助的高光表現成為本屆金球獎得主，年齡定格在18歲3個月零27天，這是世青賽歷史上第二年輕人金球獎得主。第一年輕是18歲零8天的球王梅西。

李剛仁的足球之路開始於6歲的時候上了一檔足球真人秀節目，10歲的李剛仁就開始了留洋生涯。他是瓦倫西亞青訓營的一員，2018年10月份，17歲253天的李剛仁完成了在瓦倫西亞一線隊的首秀，這是一場國王杯比賽。2019年1月12日西甲聯賽對陣巴拉多利德，李剛仁完成西甲首秀，成為韓國球員在歐洲聯賽登場的最年輕球員。2019年9月26日對陣赫塔菲的比賽，李剛仁完成西甲首個進球。2019年李剛仁成為金童獎40人候選，科帕獎第9名。

李剛仁的主要能力是精準射門，輸送高質量的傳球和致命傳球，還擁有出色的護球能力。到這裡我們發現一個最強的球員是中場球員，比如梅西，由於具備極其優秀的助攻能力，他其實更是一個中場球員，當然他也是最優秀的鋒線球員。
