瘋狂6-2！25分鐘3球，世俱杯東道主慘敗，哈維執教巴薩還需更努力

2019-12-19 1,737條評論 2,640次閱讀 1160人點贊

在世俱杯第5-6名的爭奪戰中，東道主阿爾薩德隊對陣非洲冠軍突尼西亞希望，最終東道主阿爾薩德2-6不敵突尼西亞希望，這場比賽也創造了本屆世俱杯最大比分差，值得一提的是，阿爾薩德的主帥是前巴薩巨星哈維，最終哈維帶隊獲得了第6名，在過去世俱杯3場比賽中，僅僅收穫了1場勝利，卻被狂灌10球。

作為東道主，阿爾薩德其實備受球迷期待，因為在過去亞冠賽場，阿爾薩德實力還是很明顯，一路殺進了4強，不過最終還是難以殺出沙特球隊的重圍，在半決賽中不敵利雅得新月，而勝者也獲得了亞冠冠軍。本屆世界盃，利雅得新月以亞冠新科冠軍的身份參加世俱杯，最終殺入了半決賽，對陣南美冠軍弗拉門戈，不過也很遺憾，在1球領先的情況下，被對手3-1逆轉，無緣晉級決賽；而上輪面對非洲冠軍突尼西亞希望時，利雅得新月1-0小勝挺進4強。而阿爾薩德同樣作為亞洲勁旅，面對非洲冠軍球隊卻2-6慘敗。

在開局30分鐘時間裡，阿爾薩德就被突尼西亞希望打爆，艾爾諾尼取得夢幻開局，巴德利梅開二度，阿爾薩德丟失的第3球後還贈送了紅牌套餐，。少打一人的阿爾薩德如夢初醒，在扳回1球後，依舊無濟於事，下半場被對手再進3球，最終2-6失利。

事實上，阿爾薩德在哈維的執教下，一開始還算順利，在亞冠階段展現了一定的奪冠實力，世界盃首戰也是獲得了勝利，不過隨著比賽的深入，越接關鍵比賽，阿爾薩德就越不穩，往往最終失利，這其中作為主帥的哈維也有一定的責任，畢竟突出的問題就是大賽經驗欠缺，在很長一段時間裡，當哈維從球員退役轉為教練時，外界都期待哈維未來或回到巴薩，成為巴薩的主帥接班人，如今看來，哈維還需更加努力，目前的水準，比巴爾韋德都少幾個檔次。
