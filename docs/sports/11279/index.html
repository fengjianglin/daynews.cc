上港主帥留不住！英超勁旅已開三年合同 佩雷拉這話已暗示離隊

2019-12-10 173條評論 2,437次閱讀 658人點贊

北京時間12月10日，英國媒體《每日電訊報》消息，埃弗頓方面願意為佩雷拉提供三年合同，該媒體還透露，雙方已經進行了積極的談判。

上周四，埃弗頓主帥馬爾科-席爾瓦下課，隨後俱樂部立即在世界範圍內甄選新任主教練，此前英國媒體透露，佩雷拉是埃弗頓新帥的熱門候選人。

英國媒體爆料，近期埃弗頓和佩雷拉已經進行了談判，埃弗頓方面願意為葡萄牙教頭提供3年的合約。《每日電訊報》稱，上海上港並不想放走佩雷拉，並準備了一份有吸引力的合約，以便留住他們的主教練。而佩雷拉本人則更傾向於回到歐洲主流聯賽執教。

英國媒體稱，佩雷拉目前在上海上港的收入高達2300萬歐元，如果他選擇續約，很可能拿到更高的年薪。

埃弗頓方面雖然已經將新帥候選人鎖定為佩雷拉，但是俱樂部還是做了兩手準備，如果最後佩雷拉拒絕埃弗頓，他們很可能轉而追逐前切爾西主帥安切洛蒂，前提是義大利人被那不勒斯解僱。

此外，埃弗頓還和前阿森納主帥埃梅里取得了聯繫。據悉，如果埃梅里在短期內重回英超，槍手將省下一大筆賠償金，但是目前還不清楚埃弗頓對埃梅里有多大興趣。

值得一提的是，這不是埃弗頓第一次聯繫佩雷拉，早在2013年，雙方就差點達成協議，俱樂部老闆法哈德-莫西里也很看好佩雷拉。但是俱樂部主席比爾-肯賴特則力主讓大衛-莫耶斯回歸，儘管現在看來莫耶斯不可能回歸。

目前，鄧肯-弗格森擔任埃弗頓臨時主帥，並在上任後取得了開門紅，但埃弗頓一直致力於盡快任命一名正式主帥。

儘快確立新任主教練，可以讓埃弗頓更加有效的規劃一月的引援，新任主教練也會有更多的機會了解自己的球隊。新年前埃弗頓將對陣曼聯、阿森納、伯恩利和紐卡斯爾。

此外，前拜仁主帥科瓦奇現場觀戰了埃弗頓對陣切爾西的比賽，該媒體稱，科瓦奇對英超很感興趣。
