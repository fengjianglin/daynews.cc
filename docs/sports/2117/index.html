中國最優秀籃球基層教練，培養出4名CBA球員，王汝恆賀希寧劉冠岑

2019-12-09 0條評論 22次閱讀 0人點贊

摘要

王淑玲，今年61歲了，原山東女籃的隊員，師從山東名帥葉鵬。退休前任濟南經七路第一小學的體育老師。她最為自豪的就是為CBA輸送了4名員，深圳的賀希寧、山東的王汝恆、山西的劉冠岑和山東的文雁行四名後衛都是師從王淑玲。

王淑玲，今年61歲了，原山東女籃的隊員，師從山東名帥葉鵬。退休前任濟南經七路第一小學的體育老師。她最為自豪的就是為CBA輸送了4名員，深圳的賀希寧、山東的王汝恆、山西的劉冠岑和山東的文雁行四名後衛都是師從王淑玲。

王淑玲年輕時曾經是山東女籃的隊員，當年他跟隨葉鵬訓練比賽，並且還有機會進入國家女籃，後來因為傷病，被迫提前退役，後來進入經七路一小當成為一名普通的體育教師。

大約20年前，濟南經七路第一小學有很多孩子跟著她學習，這裡面年齡大的是王汝恆，後面有劉冠岑、文雁行和賀希寧等人陸續進入學校，他們的籃球啟蒙都是王老師，這些人正是因為王淑玲老師才走上職業球員的道路，可以說她是這些球員的恩師。

王淑玲是威海榮城人，退休以後她現在在濟南西郊某小學籃球教學館仍然在帶孩子練籃球，王淑玲的特長就是教那種零基礎的孩子練籃球，她擁有多年的籃球經驗，在全國非體校的老師能帶出4名CBA球員在全國也是不多見的。

王淑玲為人低調不喜歡接受採訪，她為人正值，從來不收家長的禮，如果孩子有潛質她就用心教，很多孩子沒有走上職業球員的道路，但是跟隨她學習改變了人生，考上了名牌大學。她是中國籃球基層最默默無聞的教練，很多家長都願讓意讓自己的孩子跟著王老師練習籃球，希望王淑玲老師將來能為更多的CBA球隊培養出優秀的籃球人才。
