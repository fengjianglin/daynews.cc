高洪波只是熱身賽之王？誰又真正關心過那年的國家隊

天天要聞
2019-12-22

2009年的前半年，國家隊主帥是殷鐵生，後半年是高洪波，後來高指導一直帶隊到2011年8月10日，也就是在合肥與牙買加的友誼賽，那是「高一期」的最後一場比賽。那場比賽最終是中國隊1-0小勝，進球的人是後來范志毅點名道姓批評的人，後衛趙鵬。

很多事情以訛傳訛也就變了味兒，趙鵬當真一文不值？當年在國家隊進的球可比後來相同位置的馮瀟霆多，當然這些都是後話，後衛進球多也不代表防守水平同樣高。

當時的代理主教練殷鐵生在不算長的執教時間裡，除了友誼賽之外，還參加了阿曼四國邀請賽，國足兩戰皆負，排名第四。還參加了2009年的第十五屆亞洲杯預選賽，1月14日第一場2-3負於敘利亞，這裡可以看到了，打敘利亞我們沒有碾壓的能力，這不是最近的情況，十年前就這樣。1月21日，第二場6-1戰勝越南。

殷鐵生的國足陣容，門將位置上首選宋振瑜，後衛線杜威和曹陽是首發常客，中場周海濱、黃博文、杜震宇、蒿俊閔、曲波、姜寧等人比較常見，鋒線就是郜林、韓鵬、朱挺，偶爾幾個中場也會拉到前場當前鋒使。

2009年5月，高洪波通過公開競聘，成為中國國家隊第25任主教練，當年他43歲，是「改革開放」30年以來最年輕的國家隊主帥。郜林說高指導是一個用心血去改變中國足球形象的男人，當年的西河體育中心，職業生涯末期的高洪波帶領廣州松日統治著甲B聯賽，有人說，等到高洪波當上國足的教練了，中國隊就有希望了。

2009年5月29日，高洪波開始了他在國足主教練的位子上的上半場，在上海的友誼賽，對手是德國隊。

2008年的世界足壇，所有人都記住了巴薩的六冠王，TIKI-TAKA無疑是當時最領先潮流的戰術打法。尋求地面配合，短傳滲透，丟球後積極反搶，不輕易丟失球權。這些高洪波都看在眼裡，他是喜歡這種足球風格的，對於國足來說，他也希望把它打造成這樣一支隊伍。

這場比賽開始5分鐘，蒿俊閔破門，隨後德國隊迅速扳平比分，最終雙方1-1戰平。也正是這場比賽蒿俊閔的表現，讓德國的俱樂部對這個22歲的中國中場球員產生了興趣，2010年2月1日，德甲沙爾克04俱樂部宣布，蒿俊閔正式加盟球隊，雙方簽約1年半。

那場比賽國足控球率53.5%，德國46.5%，當然德國隊沒有盡遣主力，一方面是客場，另一方面友誼賽德國也沒必要使出全力，所以1-1也算是不錯的比分。至少控球率方面來看，高指導是要求球隊在場上要有控制力的。

正式成為主教練不到一個月的時間，高洪波已經開始他在國足的戰略規劃，他將要面對的第一場比賽就是當年11月份的還沒有踢完的第十五屆亞洲杯預選賽。

5月的友誼賽，馮瀟霆獲得出場。7月，鄧卓翔開始出現在球隊出場陣容中。從5月到11月，這半年的時間裡，除了門將（楊智）和前鋒（郜林或韓鵬）變動不大之外，中場的配置一直在不停變動。倒是陣型變化不多，對陣德國的4-2-3-1一直延續了很久，當然這是最好的選擇，雙後腰是為了加強防守，國足防守本來就不太行，當時的思維就是先擋住對方的進攻，在伺機尋求反擊的機會。

到了11月份的亞洲杯預選賽，整體上的首發陣容就差不多出來了。門將楊智，後衛線上是趙鵬-杜威-馮瀟霆-榮昊的組合，後腰趙旭日再搭配一位楊昊或者黃博文，前衛線上是蒿俊閔、于海，其餘的有曲波、鄭智、鄧卓翔候選，鋒線郜林，韓鵬、鄭龍候選。

兩場面對黎巴嫩的比賽分別以2-0和1-0取勝，當然，2009年對高洪波來說還有一件重要的事，在12月25日的比賽中，張琳芃身穿45號球衣在練習賽首發出場，此後張琳芃就成為了高洪波國家隊陣容中不會缺少的一人。那個時候的張琳芃還在中甲的上海東亞，後來2010還是沒有沖超成功，但那個時候的升班馬恆大已經看中他，在2010年11月以1200萬完成轉會。

現在的國足主教練還有從中甲選拔球員的么，別說中甲了，中超的比賽看不看都另說。

2010年，1月份以0-0戰平敘利亞，2-1戰勝越南，順利取得了亞洲杯決賽圈的門票。1月27日的練習賽中，武磊第一次出現在首發陣容中。2010年的武磊19歲，還在中甲。這年2月份來到了在日本東京舉辦的第四屆東亞足球錦標賽，中國隊將和日本隊、韓國隊以及中國香港隊進行比賽，這一屆東亞杯鄧卓翔取代了蒿俊閔在首發中的位置。

首場0-0戰平日本，第二場3-0戰勝韓國，最後一場2-0戰勝中國香港，三戰兩勝一平拿到東亞杯冠軍，這一次可不是友誼賽了，上一次拿東亞杯冠軍是朱廣滬執教的2005年東亞杯。

鄧卓翔對陣韓國隊的進球

對陣中國隊時出場的14名球員中，後來有5人在2010年的南非世界盃出場，東亞杯韓國隊的23人大名單中有13人入選了韓國隊征戰2010年南非世界盃的最終名單。所以別說什麼韓國沒上主力，現在換哪個級別的國家隊踢韓國也拿不到3-0。2019年東亞杯不剛剛0-1輸了么，你說你是選拔隊，別鬧了，人家韓國派主力來了？

這個冠軍或許是「高一期」最高光的時候，所有人的目光都來了。拿一個東亞杯冠軍就以為國足躋身亞洲頂級的人不在少數，事實上是遠沒有達到那種水平，東亞杯後的3月，第一場熱身賽國足0-2負於葡萄牙，我看了一下當時的新聞下面的評論，罵聲一片。6268人參與評論，633人跟帖，能客觀看待這場比賽的評論寥寥無幾，最熱的一條評論是「我希望中國男子足球隊集體發生空難」。

輸葡萄牙正常嗎？再正常不過了，上半場葡萄牙幾乎是全主力出戰，葡萄牙隊什麼時候在世界足壇落伍了？也就是在菲戈等人黃金一代退役，C羅還沒成長起來的時候有過一段低谷。這場比賽國足控球率不到27%，這個不用看比賽也知道局面上是被全面壓制。葡萄牙25次射門，10次射正，國足只有5次射門，1次射正。

不0-2，你想著2-0呢？18年中國杯被威爾士0-6場面崩得都不能看，還有1-4捷克。葡萄牙下半場換下了多名主力，奈何國足還是沒能取得進球，這是差距，是需要承認的差距。之前踢的熱身賽的級別是看不出來問題的，所以國足還需要更多的熱身賽，去踢勒沃庫森，去踢法國。

對於熱身賽，贏了的時候有人會說一場熱身賽而已，別人收錢辦事，有什麼好高興的。輸了會有人說真的垃圾，趁早解散。中國足球想搞好真難，真的。

5月份的熱身賽對手，找到了德甲「藥廠」勒沃庫森，比賽結果是3-0，勒沃庫森沒有派主力，這是客觀事實，此前勒沃庫森剛剛6-1中國國奧，用的也是幾乎一樣的陣容。

這次的新聞貼22413人參與討論，1878人跟帖，果然勝利比失敗更能刺激到人。最熱評論，「認為是假球的點支持」，5582人點贊，比跟帖看熱鬧的還要多。

6月14日，與法國隊的熱身賽最終以1-0戰勝對手，距離高洪波剛剛執教國足的2009年的5月29日，過去了一年多。

2010年之後沒有大型的國際賽事，只有熱身賽和友誼賽，畢竟中國沒有去成10年的南非世界盃。

2011年1月，第十五屆亞洲杯在多哈舉辦，中國與科威特、卡達以及烏茲別克分在一組。小組賽第一輪以2-0戰勝科威特，第二輪以0-2不敵卡達，最後一輪2-2戰平烏茲別克，最終小組賽階段出局。這屆亞洲杯，中國隊進4個，丟4個，顯然防守方面存在有問題。有些人喜歡拿這個小組沒出線說事兒，其實一個亞洲杯算得上什麼，2014年巴西世界盃才是重中之重，亞洲杯就算奪冠又怎麼樣，吸引更多的人來摘桃子而已。

誰家亞洲杯不拿來練兵用？作為教練，可能最高的榮譽就算國家隊主教練，作為球員，最高的榮譽也莫過於代表國家隊出征世界盃。這些失利之後，3月在香河與延邊長白虎的練習賽中，高洪波排出了一個之前從未用過的配置來考察還有能力進入國家隊的成員。

MF，呂鵬、祝一帆/曲波、王選宏、于海

下半場又換上譚天澄、馮卓毅、張稀哲等人，一直到7月份的世預賽，高洪波的後防球員差不多見到雛形，劉建業、杜威、李學鵬、榮昊、馮瀟霆、趙鵬，甚至還有吳曦。此時的吳曦出現在後衛的位置上很正常，2012年效力於上海申花的吳曦場上的位置就是後衛，他前腰位置是高洪波在2014年執教江蘇舜天后才確定的。

7月份國家隊也不得不確定大體上的配置了，因為2014年巴西世界盃的世預賽已經開始。在7月份的7-1寮國，6-1越南兩場世預賽之後，高洪波在8月10日迎來了「高一期」的最後一場比賽，最終在合肥的大雨中以1-0戰勝牙買加。足協隨後換帥，卡馬喬任新一屆國家隊主教練，衝擊2014年世界盃。

有趣的是同樣是8月10日這一天，新浪體育在國足1-0牙買加之後發起了為本場比賽的球員打分的活動，時至今日投票早已結束，我也很幸運地得以看到結果。蒿俊閔作為助攻球員，趙鵬作為進球球員都獲得了球員中最高的6.9分的評價。而評分最高的人是高洪波，7.3分。

賽後同樣是新浪體育的統計，6455人認為此時不應該換帥，佔50.5%，4671人則認為應當聘請洋帥（卡馬喬）提高成績，佔36.5%，其實也確實沒那麼多人選擇支持和相信高指導。

2011年8月10日，趙鵬打進了熱身賽的唯一進球。高洪波對於趙鵬的無疑是知遇之恩，那天的趙鵬沒有像年初亞洲杯那樣失誤頻繁，而是用頭和腳以及自己所能做的一切去不斷地化解對方的攻勢。

這是合肥歷史上第一次舉辦國家隊國際A級比賽，合肥體育中心體育場來了14087名觀眾。有兩位球迷在觀眾席上舉著橫幅，上面印著鮮紅的五個大字「洪波，謝謝你」。當時，體育場記分牌下面的看台前還有兩條橫幅，一個寫著「給他一個機會，還你一個名帥」，一個寫著「換高帥，親者痛，仇者快」。

合肥的瓢潑大雨中，教練席上穿著淡藍色襯衣的高洪波寂寞地站著，背挺得筆直。

那個時候的國足有實力嗎？有，但絕對還沒有到亞洲頂級的水準。但是到2014年世界盃還有一段日子，它本可以變得更好。賀煒說的沒錯，卡馬喬確實有名氣，但是他有時間嗎？他沒時間了，上來就是世預賽，這些球員他都熟悉？這個我沒法知曉。合肥的比賽他來看了嗎？沒有。

高洪波只是熱身賽之王？誰又真正關心過那年的國家隊。球隊需要磨合，需要不斷參加比賽，甚至需要去不斷經歷失敗。東亞杯、亞洲杯、邀請賽其實輸贏都沒什麼，贏了又怎麼樣，輸了又如何，足球世界裡誰贏誰，誰輸誰都正常，平時的訓練只不過是在努力的讓球員們在正式比賽中有犯更少錯誤的可能。面對強隊，你的一個錯誤就會送掉全局，對我來說足球比賽水平越高就越不是再比誰比誰更會玩活兒，誰腳下技術、技戰術能力越好，而是比誰更少犯錯。

大多數人可能都沒法理解足球的真正意義所在，足球需要幾代人的積澱，需要時間，越急越一敗塗地，理念越落後越經歷失利。

不必再提「高二期」，六連勝之後，世上再無「高家軍」。
