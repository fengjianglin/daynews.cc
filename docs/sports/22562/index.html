克洛普：今晚第一次讓讓亨德森客串中衛，慶幸沒踢加時

2019-12-19 1,201條評論 2,944次閱讀 1947人點贊

虎撲12月19日訊 北京時間今天凌晨結束的世俱杯半決賽，利物浦客場2-1戰勝蒙特雷，賽後，克洛普接受了採訪。

比賽

克洛普：「這場比賽發生了很奇怪的事情。這個中場組合是我派上場過的，攻擊力最強的一套中場組合。而中衛位置上我第一次讓亨德森客串中衛，我們在中後衛位置上沒有足夠的高度。」

「面對這樣的情況，如果我們不在場上加把勁，那我們不可能拿下比賽。我們今天看到，球員們盡一切努力拿下比賽勝利，他們表現非常努力，我們賽前知道球隊遇到了傷病困擾，但是我們也知道我們會在比賽中克服這一困難。」

「我們來這裡並不是來向其他球隊展示歐洲球隊高人一等，我們就是來贏比賽而已。」

「為什麼有人會認為歐洲足球就高人一等呢？你不能僅從一場比賽來判定一個球隊。蒙特雷的球員非常優秀，我們這場比賽派出了一套全新的陣容，但我們在比賽大多數時間都佔據主動。」

「我對球員們本場比賽表現感到滿意，因為我賽前就知道這會是一場艱難的比賽。」

不希望踢加時

「說實話，我挺怕踢加時的。所以很高興菲爾米諾絕殺了比賽。我們在比賽中沒有太多選擇，但也不得不做一些改變。」

傷病

「我不知道哪些今天因傷缺席的球員下一場比賽可以確保上場。我看到蒙特雷有10-15名替補坐在板凳席。我們只能希望傷員們儘早康復，做好準備。決賽我們將對陣弗拉門戈，這將是一場開放的比賽。我們會看看屆時誰表現得更加出色。」

「弗拉門戈主帥若熱-熱蘇斯是一名非常成功的、很有經驗的教練。我們對他非常尊重，但我們還是會儘力取得比賽勝利。」

「范戴克現在還沒能隨隊訓練，在媒體離開後，范戴克來到球場進行了一些嘗試，但還是感覺不好。今天早上他沒能和球隊一起訓練。周六的比賽能否上場，我們還得看情況，現在我也不知道。我們會看看他需要多久才能康復。也許范戴克周六決賽可以上場吧，我們拭目以待。」

南野拓實

「我不能說他會不會來的問題。但如果他今天在我們球隊，那會給我們帶來很大幫助。我也希望他能夠加盟，但我們還需要等待。」

球隊隊長（如果薩拉赫當隊長）

「米爾納是副隊長，亨德森是隊長。」
