穆帥：熱刺近1年聯賽沒能取得客場勝利，我上任後贏了2場

2019-12-16 26條評論 1,339次閱讀 301人點贊

虎撲12月16日訊 北京時間今天凌晨結束的英超第17輪，熱刺客場2-1戰勝狼隊，賽後，熱刺主帥穆里尼奧接受了採訪。穆里尼奧在採訪中的大部分時間都在讚揚對手狼隊的主帥和球員。

穆里尼奧：「我對球隊的表現，對這場比賽的結果感到自豪。狼隊是一個非常難纏的對手。我在周五的賽前新聞發布會上就說過了，他們的技戰術層面以及心理上都非常強大。」

「我們知道這會是一場艱苦的比賽，他們的表現沒有令我們感到驚訝。我們開局不錯，很快打入1粒進球，之後我們幾乎可以打進第二球，但對陣狼隊，這不足以殺死比賽。他們非常強大，他們擅長利用邊路，然後到中路。特拉奧雷和若塔倆人速度極快，很難阻擋他們。」

「當比分被扳平為1-1的時候，狼隊不僅僅滿足1分。我們知道我們在最後階段有能力得分然後在領先後的最後5分鐘，我們嚴防死守，最終全取3分。」

「我覺得你也可以問問狼隊主帥努諾，我認為他們的實力比人們認為的要更加出色。考慮到他們現在的實力、在積分榜的位置以及他們的雄心。他們的實力不會只滿足於前六，他們可以做得更好。他們是一支非常優秀的球隊，教練、球員，都非常棒。所以能夠戰勝他們對我們來說是一場偉大的勝利。」

「我們比賽當中有一段時間控制了比賽，面對他們這樣的球隊，想要控制比賽是非常困難的。特拉奧雷、若塔可以突然間殺死你。」

「裁判也很棒，現場的氛圍、完美的球場，雙方激烈的爭奪，兩支力求全取3分的球隊，上演了一場精彩的比賽。」

「我們在英超大約1年了，沒有取得客場的比賽勝利，但是在我上任後，短時間內我們就取得了兩場客場聯賽的勝利。現在，我們關注的不會是積分榜底下的位置，我們要向上看。這是一場非常特別的比賽，因為對手是一個非常特別的球隊。」

(編輯：姚凡)
