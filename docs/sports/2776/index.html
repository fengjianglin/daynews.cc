日乒世青賽連奪三冠背後的大功臣，為何他會離開國乒？

2019-12-04 0條評論 21次閱讀 0人點贊

摘要

導讀：日乒世青賽連奪三冠背後的大功臣，為何他會離開國乒？北京時間12月3日，隨著2019國際乒聯年終總決賽的逼近，各項賽程安排也基本完結。

導讀：日乒世青賽連奪三冠背後的大功臣，為何他會離開國乒？

北京時間12月3日，隨著2019國際乒聯年終總決賽的逼近，各項賽程安排也基本完結。在近期結束的2019國際乒聯世界青少年錦標賽的比賽中，國乒遭遇了日乒前所未有的衝擊，國乒女隊繼奧地利公開賽之後再次遭遇全軍覆沒，連失三冠，女雙、混雙皆不敵日乒組合，女單更是連四強都未入，雖然男隊不負眾望，斬獲了三冠（男團、男雙、男單）挽住了國乒的主體的地位，但是這樣的戰績顯然不符合國乒一貫的表現，日乒新生代球員與國乒之間的差距正在不斷縮小，其背後的教練是誰？

日乒年輕球員背後的華裔教練中澤銳被認為是這場比賽最大功臣，正是因為這位華裔教練的指導，日乒新生代球員的培養才能夠在這樣一段時間裡取得這麼大的進步。他原名叫做王銳，還沒有任教日乒之間是效力於河北乒乓球隊的，後來因為退役不得不尋求新的發展空間才扭轉海外，後來因為種種原因，他加入了日籍，除了這一次出場的很多年輕球員被他執教外，日乒女隊三大主力平野美宇也是其得意門生。

平野美宇現在看來威脅和挑戰沒有那麼大，但是她剛剛出道的時候，可是給國乒造成了不小的挑戰和壓力，當然也正因為平野美宇出色的表現，才讓這位華裔球員被人所熟知，這一次在他的帶領下，日乒年輕小將長崎美柚、木原美悠、小鹽遙菜給國乒帶來了不小的衝擊，無疑再次向外界詮釋了他的執教能力。

除了這次世青賽外，作為本賽季最後一場白金級別的公開賽，長崎美柚/木原美悠這對新秀女雙組合在他的帶領下一舉斬獲了成人賽事的女雙冠軍，這一次長崎美柚斬獲雙冠的亮眼表現讓我們再一次重新審視了這位華裔教練，除了她的天賦之外，這位教練的作用是不可忽略的，以至於不少球迷感概，這樣出色的教練為什麼當初一定要離開國乒，難道國乒留不住這樣一位教練嗎？

很顯然，有些出色的球員和教練不斷在海外尋求發展，其一是為了能夠有更大的發展空間，另一個很重要的原因就是國乒的競爭太大了，即便他們沒有離開，也會不斷邊緣化，教練人才流失是國乒一個無法避免的問題，這位華裔教練就是最大的典型。

當然我們也不能因為這位華裔教練執教海外球隊有了出色的表現就有過多的偏見，國乒除了發展自身之外，更是推動乒乓球運動在全球範圍內發展的一股強有力的力量，球員和教練去海外尋求發展往往是為了自己能夠在乒乓球上有更好的發展，雖然某種程度上看的確有功利性，但是往往也是不願意放棄乒乓事業的無奈之舉，你覺得呢？歡迎你的留言評論喲！
