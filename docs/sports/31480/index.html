國米4球大勝熱那亞展霸氣，孔蒂激活盧卡庫或奪冠

2019-12-23 1,545條評論 3,527次閱讀 1463人點贊

近期已經連續各項賽事三場不勝、狀態十分低迷的國際米蘭迎來了一場大勝，球隊4比0拿下熱那亞，這場勝仗給2019年畫上了一個完美的句號，這也是國米自2015年後首次以聯賽榜首的身份結束一個自然年。如果進入2020年還能延續目前的強勢的話，賽季奪得聯賽冠軍是有可能的。

在本場傷員眾多再加上勞塔羅-馬丁內斯和布羅佐維奇停賽的極其艱難的情況下，盧卡庫依舊用自己的完美表現帶領整隊獲得了一場無可挑剔的大勝，這也說明孔蒂慧眼識珠，可以在本賽季將其納入陣中。本輪比賽中盧卡庫共射門5次其中3次射正，有1次傳威脅球，光是個人就有2個進球和1個助攻，賽後評分里，盧卡庫也當之無愧的獲得了全場最高的9.2分。

如果說這些數據只是表現出了盧卡庫的高超球技，那麼在下半場中國米獲得點球時，本是應該由國米第一點球手盧卡庫來主罰，但他在點球點卻把球交給了埃斯波西托來罰的確夠無私。當這個點球被踢進球門後，盧卡庫和埃斯波西托激情慶祝時很多球迷都被感動到了，因為這不光只是一個進球這麼簡單，更是代表了國米與其他球隊不同的地方就是團結。賽後孔蒂也表示人們沒有必要等到今天才看到球隊在每個方面的進步，正如他昨天所說的，困難確保每個人都感到參與其中，他們都挺身而出承擔責任。

孔蒂的這次辭彙的確是很好的，因為這些話詮釋了一支球隊的現狀，在傷病和戰績不穩的雙重打擊下，球隊還是祭出了極佳的表現，隊內足夠團結也能一起度過傷病的大關口。本賽季國米之所以能夠一直取勝，也是因為在孔蒂執教後帶給這支球隊的團結力和凝聚力，如今這隻國米隊內良好的氛圍相信只要看過國米比賽的人就會感覺到，這也是極其可貴的一點。

本場比賽中的盧卡庫是半場最佳，這樣一位核心球員願意給隊友創造機會，增加自信心的確也是極其誠懇的，因為他將勝局看得格外重要。隊內的另外一位年輕小將塞巴斯蒂亞諾-埃斯波西托在本場也創造了歷史，因為當他在打進這顆點球後，他本人就成為了國際米蘭歷史上在意甲聯賽第二年輕的進球球員，很明顯這也是一種傳承，國米的未來儲備同樣引人動容。

這場比賽的賽後數據是射門17比9，射正9比3，角球6比3，的確也是優勢明顯，防守方面搶斷16比8，越位3比0，這也說明年輕球員對於進攻的渴望，所以才會落入對手的越位陷阱，不過目前的比分剛剛好，接下來可以蓄力戰其他對手。
