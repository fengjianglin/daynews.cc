NBA常規賽12月5日繼續進行，最終雄鹿客場以127-103輕取活塞。

首節開始，格里芬率先上籃得手。雄鹿這邊字母哥連續籃下進球，米德爾頓也上籃命中領先。斯內爾快速回敬三分，但是米德爾頓三分、罰球拉開分差，德拉蒙德連續強攻籃下命中。羅斯連續得分帶領活塞打出7-0反超！伊利亞索瓦接連進球縮小分差，首節戰罷活塞以24-22領先2分。

次節上來，字母哥和羅斯各進一球後雄鹿打出一波12-2拉開分差。斯內爾、加洛韋快速接連命中三分幫助活塞打出一波8-3縮小分差。但是馬修斯里突外投連拿6分、洛佩茲也三分罰球幫助雄鹿一度領先到18分！活塞迅速回敬一波7-0，半場戰罷雄鹿以59-48領先11分。

易邊再戰，雄鹿上來先拿4分。活塞這邊德拉蒙德、格里芬合力砍下7分縮小分差。布萊德索、字母哥接連上籃得手，德拉蒙德也連續內線打進帶領活塞打出7-2！字母哥連投帶罰拿下5分，加洛韋三分回敬。但是隨後洛佩茲連續得分，雄鹿趁勢打出一波12-0拉開比分！三節戰罷，雄鹿以92-72領先整整20分。

末節開打，莫里斯和米德爾頓各中一記三分。加洛韋打成2+1、莫里斯罰球，但是布萊德索連續上籃得分，字母哥再罰中2球拉開分差。比賽就此失去懸念。

雄鹿這邊，字母哥35分9板，米德爾頓17分8板5助，布萊德索13分6板6助

活塞方面，德拉蒙德23分14板，格里芬16分10板，羅斯14分
