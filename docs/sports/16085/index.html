恆大舊將金英權：中國球員在國家隊踢得很粗野，我不會讓自己受傷

2019-12-13 827條評論 2,362次閱讀 348人點贊

東亞杯第二輪，韓國隊將在釜山亞運會主體育場迎戰中國男足，首輪輪休的金英權，將重新披掛上陣，並與金玟哉組成中衛搭檔。因為曾在恆大效力多年，金英權比其他隊友更了解中國球員。在他看來，中國球員在國家隊踢得很粗野：「在中超踢球時，我沒有見過中國球員會出現那樣的粗野動作，可當他們為國家隊效力時，我覺得他們踢得非常粗野。」

金英權

現年29歲的金英權，已經是韓國國家隊的「元老級球員」。早在2010年，他就被時任主帥趙廣來招進韓國隊，並在對陣奈及利亞的友誼賽中上演處子秀。在恩師洪明甫上任後，金英權一躍成為主力，並受到多家歐洲俱樂部的關注。

2012年7月，金英權以250萬美元的轉會費，從日本的大宮松鼠轉投廣州恆大。憑藉出眾的防守技巧和腳下技術，金英權在恆大迅速確立主力位置，並幫助球隊接連拿下中超、亞冠、足協杯和超級盃的冠軍。與此同時，他在國家隊的地位，也進一步提升。在申台龍執教期間，金英權甚至一度被任命為隊長。

金英權和馮瀟霆

受中超新政和傷病的影響，金英權曾被恆大註銷報名資格。為了尋求更多的比賽機會，他在今年轉投大阪鋼巴，在各項賽事中，他累計出場36次，打進1粒進球。

金英權

本屆東亞杯，保羅·本托把最好的4名中後衛，都帶到釜山參賽，包括金英權、金玟哉、朴志洙和權敬原。為了讓金英權得到充分的休息，本托在首輪比賽中，安排金玟哉和權敬原搭檔雙中衛，次戰面對中國男足，金英權將進入首發陣容，並佩戴上隊長袖標。

金英權

最近幾天，韓國隊都在釜山亞運會副場進行恢復性訓練。訓練間隙，金英權接受韓國媒體的採訪，並對中國球員的粗野動作，表達自己的看法：「我們現在的狀態很好，接下來對陣中國隊和日本隊，我們在各個方面都已經做好充分的準備。在中超踢球時，我沒有見過中國球員會出現那樣的粗野動作，但當他們為國家隊效力時，我感覺他們踢得非常粗野。我們不能因為對手踢得粗野，就選擇主動避讓，這隻會讓比賽變得更加艱難。我告訴我的隊友，面對這樣的局面，你應該敢於硬碰硬，而不是退讓。我肯定不會讓自己受傷。」

金英權

身高達到1米86的金英權，曾為韓國隊打進3粒國際A級比賽進球，在他看來，韓國隊應該充分發揮定位球上的優勢：「對陣中國香港時，我們在定位球上做得並不好。如果你能通過定位球得分，那比賽會變得更加簡單，我們必須要做出改進。」
