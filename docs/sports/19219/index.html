易建聯啞火，他13投7中橫空出世！輸球卻獲表揚，繼承母親高顏值

2019-12-16 713條評論 4,499次閱讀 3496人點贊

北京時間12月15日晚，CBA戰火重燃。衛冕冠軍廣東宏遠客場挑戰今年的黑馬北京北控。要是放在上賽季，可能這場比賽就是一場完全沒有懸念的比賽，一個是衛冕冠軍，一個是聯盟的副班長，孰強孰弱一目了然。可是本賽季北京北控在馬布里的率領下已經有了翻天覆地的改變，本場比賽開始之前北京北控以9勝7負的戰績排行積分榜第十位，與第八名的山東積分相同。

本場比賽在上半場的角逐中，北京北控給衛冕冠軍造成了很大的麻煩，進攻端依靠弗格一個點打的廣東男籃落花流水，弗格僅僅用了半場球就砍下了24分4籃板5助攻，半場戰罷北控主場68-61暫時領先。易變再戰，北控到第三節更加勢不可擋！三節戰罷砍下100分，給衛冕冠軍好好上了一課。最終廣東男籃以111-126不敵狀態火熱的北京北控，迎來了本賽季的第二場失利。

本場比賽易建聯的發揮依然不佳，僅僅得到9分。我們可以看出阿聯在體力上面對著很大的瓶頸，尤其是在第二節比賽即將結束的時候，廣東隊進攻不中，球彈出來非常遠，易建聯站在三分線附近竟然讓彈出來的球從自己的襠下鑽過去，而他完全失去了對球的控制，最後北控隊抓住機會打成了一次漂亮的快攻，回防的阿聯氣喘吁吁，滿頭大汗。現在廣東男籃在內線真的太依靠阿聯了。不過本場比賽廣東男籃有一位奇兵橫空出世，讓我們眼前一亮，他就是小將胡明軒。

本場比賽，胡明軒全場出戰25分鐘，13投7中砍下18分4助攻，其中上半場胡明軒就已經6投5中得到了13分，除了外援威姆斯，胡明軒就是全隊得分最高的人。身高191cm的胡明軒本賽季代表廣東男籃出戰17場比賽，其中16場比賽首發出戰，在21.1分鐘的場均出場時間內，可以得到7.8分2個籃板和1.1次助攻，命中率也超過百分之50，這對於一位21歲的年輕球員來說已經非常難得了。雖然今年胡明軒僅有21歲，不過他卻早早地成為了國家隊的「常客」，早在2年前，胡明軒就憑藉出色的個人能力入選了中國男籃紅隊的大名單。

而胡明軒之所以年少成名，圈粉無數，不單單依靠的是他飄逸的球風。他俊俏的外形，帥氣陽光的長相也讓他一直是聚光燈下的寵兒。胡明軒在生活中也是一位非常孝順的大男孩，出生在新疆烏魯木齊的他經常通過自己的社交平台曬出自己與媽媽的合影，而胡明軒的媽媽也非常支持兒子的事業，經常來到現場觀看兒子的比賽。從圖片中我們可以看出，胡明軒的媽媽顯得非常年輕漂亮，氣質極佳，曾經有很多球迷都誤以為她是胡明軒的姐姐。大概是由於有其母必有其子吧，胡明軒的顏值顯然是媽媽遺傳的！

​不得不說，在籃球場上年輕就是資本。有很多球員打了一輩子球也沒有學會閱讀比賽，不能在球場上找到自己的定位。而胡明軒僅僅21歲，在一支衛冕之師就已經可以打上首發，每場有20多分鐘的出場時間，他真的比一些30多歲的球員還要優秀，而且他未來的路還很長，相信這位杜鋒指導的愛徒以後肯定能在國家隊獲得更多的出場時間，我們也期待著未來這位年輕人能給我們帶來更多的驚喜，早日成為國家隊後衛線上的主力核心。
