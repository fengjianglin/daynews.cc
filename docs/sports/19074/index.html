他曾三勝桃田賢斗，橫掃孫完虎奪冠，如今將迎娶世界冠軍令人羨慕

2019-12-17 214條評論 2,413次閱讀 525人點贊

北京時間12月15日，前國羽男單隊員喬斌在個人社交平台官宣，已經和倫敦奧運會女單金牌得主李雪芮在重慶市大渡口區民政局領證，原文如圖片所示：

今年的國羽喜事不斷，王適嫻當母親了，諶龍當父親了，王儀涵也找到了另一半，喬斌和李雪芮又公布了領證的消息，這也宣示著兩人結婚的好日子也越來越近，李雪芮滿臉幸福洋溢，在喬斌發布動態後，李雪芮隔空和喬斌互動，灑下「狗糧」：

李雪芮想必很多球迷都認識，1991年出生，重慶人，八一隊出身，國羽女單輝煌的「兩王一李」時期的大姐大，倫敦奧運會女單金牌得主，後在里約奧運會半決賽受傷，經過兩年時間恢復又堅強的選擇復出，是一個要強不服輸的90後重慶妹子！

介紹完李雪芮，就該介紹一下准老公喬斌了，喬斌成績沒有李雪芮輝煌，和李雪芮成績一比，喬斌的成績可以說平淡無奇，很多剛關注羽毛球的球迷怕是還不認識國羽隊有一號這樣的球員，不奇怪，就是經常看比賽的球迷，也很長時間沒有看到喬斌的比賽了！

因為我是羽毛球媒體號，基本上每場比賽都會看，據我所回憶，上一次看喬斌的比賽，還是在2018年的年初，哪一場賽事忘記了，喬斌獲勝後把球拍高高扔上天空，還差一點砸到對手，後來又一次勝利，喬斌高興的又一次扔球拍，然後坐到地上慶祝，但喬斌參賽的比賽都沒有走到最後，甚至沒有挺進四強，再往後的幾場比賽，選手名單表上都有喬斌的名字，但連續幾場比賽喬斌都是退賽，沒有再出現過比賽現場！

再之後，便沒有了喬斌的消息，畢竟喬斌成績一般，引起不了媒體的注意，哪像李宗偉退役，搞的全世界人人皆知，似乎都沒有聽說喬斌官宣退役，再次聽到喬斌的消息，就是和李雪芮談戀愛了，並且已經從國羽隊員轉型為教練員，第三次聽到喬斌的消息，就是喬斌求婚李雪芮，第四次聽到喬斌的消息，就是這次和李雪芮領證結婚了，估計第五次聽到喬斌消息，就是和李雪芮結婚了，不得不佩服喬斌的速度，這麼快就將迎娶世界冠軍李雪芮，令人羨慕！

正兒八經的說一下喬斌的成績，喬斌職業生涯最好成績是奪得2016年的韓國公開賽男單冠軍，並且是戰勝了東道主韓國一哥孫完虎，並且是打出21-7比分差橫掃孫完虎奪冠，或許一個超級500級別賽事對於林丹、諶龍來說拿不出手，但比起現在的年輕小將趙俊鵬、陸光祖、周澤奇等人，喬斌已經算是佼佼者了！

喬斌還有一項記錄，曾連續3勝日本一哥桃田賢斗，是桃田賢斗的「噩夢」，當然，那時候的桃田賢斗還不是日本一哥，也沒有到巔峰期，可惜呀，2018年桃田賢斗崛起，而喬斌2018賽季逐漸消失在國際賽場，兩人就此再也沒有在賽場遇見過，不知如今站在男單世界之巔的桃田賢斗，是否會想起曾經那個可怕的對手-喬斌！

最後祝喬斌和李雪芮能早日步入婚姻的殿堂，從此過上幸福美滿的好日子，也希望成為教練員的喬斌，能為國羽隊多培養幾位優秀的好苗子，感謝感謝！祝福祝福！
