考驗許昕劉詩雯的時候到了！乒聯發布消息，他倆離奧運就差一場球

2019-12-03 0條評論 24次閱讀 0人點贊

摘要

在男乒世界盃上，樊振東連續擊敗林昀儒、張本智和兩位未來之星，拿到了個人第3個世界盃單打冠軍。而老將馬龍受限於自己的身體，連續輸給張本智和、林昀儒拿到第4。隨著男乒世界盃的落幕，國乒接下來的重要比賽就是在鄭州舉行的國際乒聯年終總決賽了。

在男乒世界盃上，樊振東連續擊敗林昀儒、張本智和兩位未來之星，拿到了個人第3個世界盃單打冠軍。而老將馬龍受限於自己的身體，連續輸給張本智和、林昀儒拿到第4。隨著男乒世界盃的落幕，國乒接下來的重要比賽就是在鄭州舉行的國際乒聯年終總決賽了。12月3日，乒聯發布了消息，考驗許昕和劉詩雯的時候到了，他倆現在距離奧運會只差最後一場球了。

東京奧運會比起過去幾屆奧運會有一個最大的區別，那就是增加了混雙作為正式項目，如此一來，乒乓球項目第一次在奧運會有了5枚金牌設置。不僅如此，混雙項目每隊只能有1對選手參加，而且並不能增加奧運參賽選手，每隊依然只有男女最多各3個名額。這也就意味著，奧運會的混雙比賽偶然性將加大，而且參加混雙的選手也將面臨著兼項，這對他們的體力和精力都是考驗。

在劉國樑上任乒協主席之後，國乒也試驗過不同的混雙組合，許昕/劉詩雯、樊振東/丁寧、王楚欽/孫穎莎、林高遠/朱雨玲等組合都曾經在國際比賽中亮相。而其中，還是以許昕和劉詩雯的組合成績最為突出。2人拿到了世乒賽、亞錦賽的混雙冠軍，保持著非常高的勝率。儘管劉國梁強調國乒至少要培養出2對具備奧運奪金實力的選手，但如今看來，只有許昕和劉詩雯比較保險。

12月3日，國際乒聯在官方社交平台發布信息，對12月份的重要賽事進行了詳細介紹。其中，尤其對12月12日到15日在中國鄭州舉行的乒聯總決賽進行了很詳細的解讀，而在談到混雙時，國際乒聯表示混雙的賽制與其他項目略有不同，共有8對選手參賽，其中前4名將直接獲得東京奧運會的參賽資格。正因為如此，本次乒聯總決賽的混雙項目勢必會迎來非常激烈的較量。

而這也就意味著，許昕和劉詩雯在8對選手參賽的情況下，只要贏下1場球就可以晉級四強，從而拿到東京奧運會的參賽資格。如今國乒隊內關於奧運名額的競爭也非常激烈，許昕和劉詩雯有望通過混雙項目，率先從激烈的競爭中脫穎而出。當然，即便如此，許昕和劉詩雯也只是拿到混雙的奧運參賽資格，會不會參加單打比賽，還要等劉國梁、國乒教練組他們進一步去考察。

相比之下，劉詩雯參加奧運單打的可能性很大，她在今年包攬了世乒賽、世界盃的單打冠軍，狀態十分火熱。而許昕則要面臨與馬龍、樊振東的激烈競爭，樊振東如今狀態復甦，拿到一個單打席位的懸念不大。倒是馬龍儘管實力依然出眾，但他的身體卻讓人擔憂。馬龍還需要在奧運會前進一步調整自己的身體狀態，許昕也依然存在著參加單打的機會。具體怎麼安排，就要看劉國梁和教練組的想法了。
