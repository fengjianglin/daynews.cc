英超十年！BBC評選十大時刻：阿圭羅絕殺奪冠、利物浦逆轉巴薩…

天天要聞
2019-12-20

近日，英國著名媒體BBC對英超近十年進行大盤點，評選了十年來最令人震驚的十大時刻，包括利物浦逆襲巴薩、阿圭羅絕殺奪冠、蘇牙驚天一咬，老佛爺退休等，排在第一的當屬不讓就是狐狸城上演的「藍狐奇蹟」。

英超十年的十大時刻（按時間順序排列）

1、魯尼的倒鉤破門（2011年）

在2011年的曼市德比中，曼聯2-1力克曼城，魯尼的倒鉤絕殺贏得滿堂喝彩。比賽第78分鐘，納尼右路傳中，無人盯防的魯尼在禁區內飛身而起倒掛金鉤，皮球直奔死角。這成了魯尼職業生涯中最漂亮的一個進球。

2、巴洛特利慶祝名場面（2012年）

在2012年曼市德比中，巴洛特利踢進球後沒有狂歡，也沒有笑容，而是霸氣十足地掀起球衣露出內衫，上面寫著「why always me？」，他的這個舉動成為多年來人們的談資。

3、阿圭羅絕殺奪冠（2012年）

2012年英超最後一輪，第92分鐘曼城1-2落後QPR，隨後阿圭羅踢進絕殺球逆轉取勝，曼城取得了44年來的第一座英超冠軍。

4、蘇亞雷斯驚天一咬（2013年）

在利物浦對陣切爾西的比賽中，蘇亞雷斯與切爾西後為伊萬諾維奇爭奪位置過程中發生肢體接觸，蘇亞雷斯一時情急低頭咬了伊萬手臂，這也是蘇牙的第二次咬人事件。

5、弗格森退休（2013年）

2013年，曼聯官網宣布，弗格森將在賽季結束後退休，從低迷到興盛，老佛爺見證了曼聯的成長，甚至為曼聯贏得了38座獎盃之多。

6、傑拉德滑倒（2014年）

在利物浦對陣切爾西的比賽中，傑拉德滑倒，葬送了利物浦英超奪冠的機會，同時也葬送了他個人的首個英超冠軍，成為永久的遺憾。

7、英格蘭女足奪得世界盃季軍（2015年）

2015年女足世界盃，英格蘭女足首次打進四強，擊敗德國隊獲得第三名，這也是英格蘭女足在世界盃上的最好成績。

8、萊斯特城上演「藍狐奇蹟」，奪下英超冠軍（2016年）

近年來最不可思議的足球事件之一，一個從教練到球員都平平無奇的球隊，最終擊敗曼聯、曼城、阿森納等豪門奪得英超冠軍。

9、英格蘭打破點球魔咒，擊敗哥倫比亞重返8強（2018年）

2018年俄羅斯世界盃1/8決賽，英格蘭世界盃隊史上首次贏得點球大戰，5-4擊敗哥倫比亞，時隔12年重返世界盃8強。

10、利物浦逆襲巴薩殺進決賽（2019年）

在今年的歐冠半決賽中，利物浦對陣巴薩，首回合0-3，次回合4-0，利物浦上演最強逆襲，絕殺巴薩晉級決賽，最終贏得歐冠冠軍。

（小喬）
