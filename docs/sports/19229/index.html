他不是國內最好的一批，但卻仍是山東球迷心中的“寵兒”

2019-12-16 3,589條評論 4,652次閱讀 200人點贊

12月15日晚上結束的CBA第17輪，山東男籃116：107新疆男籃，山東男籃首次紅山體育館戰勝對手。首勝背後，既有新疆單外援出戰和自身表現不佳的原因，也有山東雙外援的穩定輸出和國內球員的出色表現。而在其中一名球員8投3中，拿到11分。28歲的他，在CBA已經打到了第8個賽季，場均得分數據最高沒有超過12分，場均籃板最高3.5，場均助攻最高1.8。就是這樣一位從數據上看普普通通的球員，卻在山東球迷心中擁有這眾多的粉絲，球迷親切的送他外號「小李飛刀」、「鯨魚」。他就是李敬宇。

作為一名球員，李敬宇場均得分僅排在國內第48位，在他前邊除了我們耳熟能詳的王哲林、周琦、易建聯、郭艾倫 國家隊常客外，還有最近崛起的賀希寧、姜宇星等一批球員，即使在山東隊內也僅排在國內球員第4位，在他最擅長的三分球上本賽季命中數也僅排在整個國內球員第15位。從打球方式上，鯨魚作為一名鋒線球員，鯨魚在進攻中主要的手段還是在三分線外發炮，一旦對手對其進行緊逼，其在球場上的作用就會大大下降。再加上鯨魚前幾年受傷大修後，身體素質有所下降，在防守上更多依靠經驗，特別是對上對方小外之後，壓力比較大。

但這些絲毫都不影響球迷對他的喜愛，李敬宇身上有著令我們每一個普通人心動的「閃光點」：

雖然數據不亮眼，但卻對山東至關重要。山東作為一名內線實力厚重的球隊，最近幾個賽季以來，特別是李敬宇出走山西的兩個賽季，山東的外線三分球非常不穩定，缺乏穩定的輸出，導致對手防守收縮嚴重，而李敬宇的回歸恰恰補足了山東最大的短板，讓對手不能全力夾擊內線。

雖然打球方式不全面，但卻總讓人能看到進步和希望。出道之年，就有讀秒絕殺對手的表現，讓我們看到了希望；大傷之後重回賽場，讓我們看到了一種韌勁；最近幾場，他在中距離的拋投，讓我們看到他不斷努力、不斷突破自己。如果說丁彥雨航的表現更多讓我們感受到的是天賦、是希望，李敬宇則讓我們看到了平凡的努力，讓我們每個平凡人更加不放棄。再加上李敬宇俊朗、飄逸的外表、文雅的舉止，更是為他吸引了不少球迷。

總之，他不是國內一流球員，但卻是山東球迷的「寵兒」、是山東球迷的「心頭好」，個人一家之言，歡迎交流討論。也希望希望鯨魚在今後的CBA中不但提高自己，為球隊提供更多戰術選擇，做出更多貢獻！祝願山東男籃越打越好，為球迷貢獻精彩比賽。
