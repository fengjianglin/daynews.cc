德轉更新西甲身價，皇馬巴薩兩隊合計下跌超過2億歐

2019-12-21 111條評論 1,191次閱讀 175人點贊

直播吧12月20日訊 德國轉會市場更新了各俱樂部球員的身價，皇馬和巴薩兩隊的球員身價都出現了下跌的現象，兩隊球員身價的下跌總和已經超過了2億歐元。

據統計，皇馬全隊的球員身價下跌了1.195億歐元，巴薩全隊的球員身價下跌了8150萬歐元，巴薩和皇馬在歐洲俱樂部球員身價排行榜上已經分別下滑到了第3和第4位，排名前兩名的是曼城和利物浦。

對此，西甲轉會市場價值評估負責人布拉塞奧表示，「國家德比的互交白卷證明著巴薩和皇馬的趨勢，大手筆的光輝引援時代已經結束，皇馬在一個艱難的階段後曾經歷了輕微上漲，但他們依然還有上升空間，皇馬的陣容看起來領先於巴薩一步。」

不過，兩隊整體都是球員身價下跌的趨勢，皇馬的阿扎爾身價從1.5億歐元下跌到了1.2億歐元，巴薩的格列茲曼身價也從1.2億歐元下跌到了1.1億歐元。

至於梅西也避免不了身價下跌，他的身價從1.5億歐元降到了1.4億歐元，布拉塞奧對此認為，「梅西依然處於頂級水平，但他在明夏就要年滿33歲了，因此他的身價也會慢慢下降，哪怕他在場上表現還是最高水平。」

（Paul）
