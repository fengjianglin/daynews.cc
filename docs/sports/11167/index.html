新疆男籃9名球員“玩命”，富蘭克林只能空砍“四雙”

2019-12-11 170條評論 2,061次閱讀 654人點贊

富蘭克林有「四雙」但是沒勝利。

沒有費爾德的新疆男籃，依然有勝利！

12月10日晚，儘管費爾德因為腿傷缺陣、儘管對手陣中的富蘭克林砍下「四雙」，但新疆男籃最終還是114比110驚喜戰勝山西男籃，不僅取得一場足以為自己正名的勝利，而且也暫時拉開了與對手之間的差距。

在CBA官網最新更新的球員傷病名單上，費爾德因為腿傷而被擺上傷病名單，因此，本場與山西男籃的比賽開始前，客場作戰的新疆男籃並不被各方所看好。但是，這些「唱衰」新疆男籃的人顯然忘記了一點，那就是新疆男籃從建隊以來就一直所倡導的「飛虎精神」——簡而言之，「飛虎精神」就是逆境中不放棄的比賽態度。

在富蘭克林將比分扳成5比5之後，新疆男籃由米卡利用罰球再次反超比分，而且此後很長一段時間之內，新疆男籃都佔據著比分上的領先。富蘭克林所領軍的山西男籃，只是在首節的後段短暫取得過一次領先。

因為費爾德的缺陣，新疆男籃在1號位上的確存在不足，但不管是曾令旭還是西熱力江，他們都很好地填補了費爾德缺陣可能帶給球隊的不利影響。比賽進入第二節之後，雙外援出戰的山西男籃曾經兩度將領先優勢擴大，但曾令旭、唐才育以及俞長棟等球員的突然爆發，幫助新疆男籃在第二節比賽也僅僅凈負對手4分而已。

曾令旭很好地填補了費爾德缺陣的空白。

從首節范子銘和郭曉鵬拼搶地板球，到曾令旭、俞長棟以及唐才育等球員的爆發，新疆男籃的球員本場比賽最犀利的「武器」，其實就是他們的拼搏精神、或者說「玩命」的比賽態度。而除了以上提到的四名球員之外，加盟球隊不久的米卡，也終於在本場比賽中迎來了全面爆發。

第三節開場階段，米卡連得6分幫助新疆男籃繼續緊咬比分，而比賽進入最為關鍵的第四節，米卡又一次連得7分幫助新疆男籃緊咬比分。本場比賽，米卡砍下29分、16個籃板的大號兩雙，相比於斯托克斯，米卡的攻擊範圍的確不夠大，但他的積極性以及籃下對抗的強硬，還是足以讓人暫時「忘記」有傷在身的斯托克斯。

當然，新疆男籃在本場充分展現拼搏精神的，依然不僅僅是這幾位球員。周琦的得分、周琦的防守同樣讓人印象深刻。比賽進入關鍵的第四節之後，葛昭寶連續兩次投籃均遭到周琦的封蓋，本場砍下20分、12個籃板、3次助攻的周琦，一共完成個人賽季最高的7次封蓋，可以說，周琦同樣是新疆男籃在本場比賽極其「玩命」的一位球員。

新疆男籃本場出戰的9名球員都是英雄。

對了，怎麼能夠忘記阿不都沙拉木、西熱力江以及可蘭白克這三位本土球員。阿不都沙拉木不僅在下半場砍下15分，而且在僅剩15秒時投中足以改變比賽的三分球。為阿不都沙拉木送上這次助攻的，就是本場一共送出5次助攻的西熱力江。而在比賽的最後時刻，西熱力江和可蘭白克頂著壓力在罰球線上的表現，同樣是新疆男籃最終能夠險勝山西男籃的重要原因——西熱力江雖然罰丟了一個，但新疆男籃還是搶下了這個關鍵前場籃板球。

出場的9名球員球員集體「玩命」，新疆男籃的確沒有不贏球的理由，而面對這場失敗，山西男籃陣中最為遺憾的球員，毫無疑問就是砍下42分、12個籃板、12次助攻以及10次搶斷的富蘭克林。畢竟，這不僅是富蘭克林CBA生涯的第一個「四雙」，而且也是CBA歷史上的第三個「四雙」，在一場可謂自己乃至CBA的里程碑之夜，卻只能與球隊以失敗收場，富蘭克林心中的憋屈可想而知。

但話又說回來了，富蘭克林其實也沒有太多可以憋屈的，因為，他恰恰也就是山西男籃最終輸球的原因之一。第四節還剩1分多鐘，富蘭克林完成對曾令旭的搶斷達成「四雙」，但他的上籃隨即遭到周琦的封蓋，米卡完成扣籃之後，新疆男籃取得105比105的平局。

在阿不都沙拉木僅剩15秒投中那個反超比分的三分球之後，米卡又造成直插籃下的富蘭克林進攻犯規，富蘭克林關鍵時刻的兩次失誤，一方面可以證明新疆男籃的球員究竟有多「玩命」，另一方面也足以證明富蘭克林很像是球隊最終落敗的「罪人」——至少是「罪人」之一。

新疆男籃客場與山西男籃的比賽就此結束，缺少費爾德但9名出場球員集體「玩命」的新疆男籃，也因此贏得了一場艱苦而且讓人足夠敬佩的比賽，這場勝利，對於之前不敵吉林男籃的新疆男籃而言，的確是非常提振士氣。

當然，富蘭克林雖然沒能帶走勝利，但他的「四雙」還是足以讓整個CBA所銘記。只不過，「空砍」，的確是缺了那麼一點意思，但這，真的不能怪新疆男籃。
