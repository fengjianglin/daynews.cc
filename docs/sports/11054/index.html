郎平罕見發聲，敲打朱婷和中國女排全隊：擺正位置從零開始好嗎？

2019-12-11 114條評論 3,638次閱讀 2184人點贊

北京時間12月11日，在剛剛結束的女排世界俱樂部杯賽當中,中國隊的一眾世界冠軍們領銜的兩支中國球隊，在比賽當中表現的慘不忍睹，天津女排排名倒數第一，廣東女排排名7支球隊當中的倒數第二，而在兩個月前的女排世界盃賽當中，她們當中的很多名精英球員組成的中國女排國家隊，連續戰勝了巴西，俄羅斯和美國等一眾強隊最終拿到了冠軍，這樣的表現形成了巨大的反差。

這次女排世界俱樂部杯賽，天津女排和廣東女排陣中雲集了眾多的世界冠軍，天津女排中包括了朱婷，姚迪，李盈盈，袁心玥。和王媛媛，而廣東女排這邊也有鄭益昕和林莉以及楊涵玉。這8名球員都是兩個月前，幫助中國隊拿下世界盃11連勝的重要成員，其中朱婷和袁心玥以及林莉還是現役的奧運冠軍加世界冠軍雙料大賽冠軍。

結果，女排8大世界冠軍帶領隊友們出戰，而且隊中還包括了像科舍列娃以及胡克爾在內的優秀的外援，最終卻取得這樣的成績，充分的讓人看到中國女排，如果作為個體各自散開進入到俱樂部隊和對手周旋的話，實力差距還是蠻大的。而我們的對手，包括了科內利亞諾和瓦基弗銀行等強隊在內，她們不僅整體實力非常強，還擁有像埃格努，博斯科維奇以及哈克這樣的優秀巨星。這些人的存在給中國隊帶來了巨大的威脅，也充分的讓人意識到，這次的世界盃賽當中，義大利沒有參加，而塞爾維亞則是派出2隊出戰，博斯科維奇並不在陣中，確實讓中國隊11連勝奪冠的含金量顯得有那麼些美中不足。而明年的東京奧運會上，這兩大勁敵必將成為中國隊奪冠路上最大的攔路虎。

對此，剛剛過完59歲生日的中國女排主教練郎平，在最新接受中央電視台的採訪之時，也是向弟子們給出了正式的警告和最新的要求。郎平明確的拿出了她標誌性的16個字來勉勵姑娘們，她表示：還是那句話，走下領獎台，一切從零開始。隨後對比世界盃和世界俱樂部杯中國隊的表現來看，郎平具體的表示說，很明顯大家要把心靜下來，要真正的擺正自己的位置，做好最困難的準備，努力的做好每一天，這樣的話，中國隊才有可能在明年的東京奧運會上取得理想的成績。

不得不說，郎平真的是高瞻遠矚，令人讚歎，因為這次對比世界俱樂部杯，人們都發現了中國女排的不足之處。而恰恰只有郎平能夠將中國女排捏和成團，讓她們發揮出強大的團體戰鬥力，這是這些俱樂部教練並不具備的。
