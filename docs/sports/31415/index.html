拳王泰森指導網壇傳奇小威廉姆斯打沙袋，粉絲點擊量暴增

2019-12-24 2,923條評論 3,997次閱讀 2863人點贊

近日，前世界重量級拳王邁克-泰森，指導網壇名將塞雷娜-威廉姆斯打沙袋的視頻，爆紅網路，成為體育迷爭相熱議的焦點。現年53歲的泰森，是重量級歷史上最年輕的世界冠軍，同時在1987年統一了所屬級別，儘管退役已經十多年了，然而他仍然是許多狂熱粉絲的最愛。

與泰森相比，塞雷娜-威廉姆斯則是另一個領域的傳奇，通常被大家稱之為小威廉姆斯的她，自1995年初涉網壇之後，已經贏得了23個大滿貫單打冠軍，與姐姐維納斯-威廉姆斯（大威廉姆斯），堪稱世界網壇史上的「雙擘」！

由於泰森、小威廉姆斯都是職業體壇的佼佼者，所以兩人能夠在一起，自然能夠碰撞出難以置信的火花。從公布的視頻中看，泰森正在指導38歲的小威廉姆斯打沙袋，不知小威之前練了多久？總之她的擊打力量看上去十分突出，泰森不得不扶著沙袋以抗擊巨大進攻，雖然小威廉姆斯動作略顯生猛，但動作還是一板一眼，只見她一口氣出擊了近70拳。訓練結束後，泰森稱讚小威廉姆斯：「你具有巨大的力量。」

隨後，泰森與小威廉姆斯合影，兩人面對鏡頭擺著姿勢，同時泰森在發布這張照片時，在上面備註：我不想與史上最偉大者交鋒，小威廉姆斯，我很愛你、很尊重你！該照片從昨晚發布之後，粉絲點贊量竟然超過了難以置信的9.5萬次，視頻點擊量更是暴增，這足見泰森、威廉姆斯在公眾心目中的地位。

有粉絲留言：這是兩個傳奇！還有粉絲留言：泰森、小威廉姆斯都是史上最偉大者！又有拳迷留言：如果小威廉姆斯涉足拳壇，她肯定會是世界冠軍！

據了解，泰森指導小威廉姆斯訓練這段視頻，是不久前在佛羅里達州完成，說起來兩人頗有淵源，泰森的女兒幾年前曾拜師小威廉姆斯，所以泰森與對方十分熟悉。同時，不得不說的是，依照泰森、小威廉姆斯所取得的成就，兩人在一起絕對可以引起巨大轟動，這就勿怪粉絲們如此痴狂了。
