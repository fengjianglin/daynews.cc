英超官宣曼聯失球有效原因！德赫亞被打臉，英超卻認定是正常接觸

天天要聞
2019-12-16

曼聯在主場1-1被埃弗頓逼平，雖然縮小了與第四名切爾西的差距，但卻不是想像中的結果，目前依然落後歐冠區4分，而且排名也被實現絕殺的熱刺反超。比賽中，曼聯因為林德勒夫的烏龍球先落後，下半場靠梅森·格林伍德替補上陣救主，扳平比分，但最終無法取勝。不過，曼聯的丟球存在巨大爭議，德赫亞在角球中出擊，被對方前鋒多米尼克·卡爾維特-勒溫一手打在臉上，皮球因此落到後點，猝不及防的瑞典冰人將球踢入了自家網窩。

主裁邁克爾·奧利弗斷定此球有效，VAR視頻助理裁判克雷格·鮑森介入後，依然維持原判。曼聯球迷對此非常憤怒，因為慢鏡頭顯示，埃弗頓9號的手明顯擊打了德赫亞，因此影響了曼聯門將的擊球，皮球才會落到後點。中場休息時，德赫亞也找到裁判組申訴，但判決已成事實，已經無可挽回。

英超官方則在英國天空體育電視台上做出官宣，解釋埃弗頓的進球有效，是因為勒溫與德赫亞的空中對抗屬於「正常接觸」（normal contact）。

這一解釋當然無法令曼聯球迷信服，紅魔名宿加里·內維爾也認為應該是犯規。「我很驚訝，通常如果一名門將遭到這樣的動作侵犯，手臂都打到臉上了，應該是犯規。」內維爾分析道，「在慢鏡回放前，我最初認為德赫亞應該更強勢，但他確實遭到了肘擊。德赫亞也沒有過分抱怨，勒溫的左臂碰到了他。我對這個判罰很驚訝，但門將確實需要更加強勢。」

德赫亞處理高空球確實一直都是他的弱點之一，剛來英超時，斯托克城等對手紛紛把握他的空中弱勢猛攻。雖然在弗爵及其門將教練埃里克·斯蒂爾指導下，西班牙國門的空中技術有所提高，但仍非標準的英超守門員。如果德赫亞抬起膝蓋，而且更猛烈出拳，哪怕有對手糾纏，這球仍應可以解圍。

此外，曼聯全隊在這次角球防守中的布置也存在明顯漏洞，居然讓勒溫和米納兩名埃弗頓球員直接沖向自己的門將，沒有任何隊友去保護德赫亞。

內維爾就認為，馬奎爾和林德勒夫的中衛組合能在有球狀態下給曼聯帶來控制力，但他們的防守仍然有待加強。數據顯示，曼聯已經連續12輪英超丟球，創造1971年以來48年最差紀錄。

「我喜歡這對中衛，他們都能很好地閱讀比賽。但這是不是能為曼聯重奪聯賽冠軍的中衛組合，能夠在歐冠爭霸？我不那麼有信心。」內維爾說，「他們能提供更好的控球。如果你看看最近曼聯的中衛，像這賽季開始時的7名中衛，圖安澤比、菲爾·瓊斯、斯莫林、羅霍、林德勒夫、埃里克·巴伊和馬奎爾。我說過，不清楚哪兩個人配對是完美的。現在這兩個是其中最好的搭檔，他們能控球，控制防守位置，對比賽的理解勝過其他隊友。但他們還需要繼續提高，這取決於奧萊（索爾斯克亞）和俱樂部是否認為他們是能奪取英超冠軍的中衛。曼聯現在還差得遠，但他們必須朝著這個目標進行重建。」

本賽季曼聯1-1戰平利物浦的比賽，也曾受益於VAR沒有吹罰拉什福德進球前林德勒夫對奧里吉的疑似犯規。因此，對於裁判的判罰，曼聯無法更改也沒理由抱怨太多，索爾斯克亞需要真正根治曼聯防守定位球的弱點，這是確定無疑的。
