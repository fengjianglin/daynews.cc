國足對陣韓國陣容或大調整，四人有望進首發，他的表現最值得期待

天天要聞
2019-12-13

國足東亞杯第二場比賽將對陣韓國隊，本屆東亞杯國足選拔隊主要目的是練兵。所以國足對陣韓國隊的比賽，李鐵可能給一些國腳新人上場機會。尤其是國足東亞杯首戰對陣日本的比賽，沒有獲得出場機會的國腳，將有望在國足挑戰韓國的比賽中首發登場。首先在國足門將位置，李鐵將給鄒德海或者董春雨上場首發機會。

國足東亞杯首戰踢主力門將的劉殿座，在比賽中整體表現中規中矩。考慮到門將位置的特殊性，為了讓鄒德海和董春雨這兩位尚未完成為國首秀的門將，有機會在東亞杯上亮相圓夢。國足對陣韓國的比賽中，李鐵會在門將位置給新人首秀機會。中衛位置上李昂在上一場國足對陣日本隊的比賽，出現了因傷上場的情況。如果李昂在國足對陣韓國的比賽，不能及時傷愈復出，這個位置或由老將王燊超踢主力。

國足中場位置金敬道雖然在東亞杯首戰對陣日本的比賽，表現不俗貢獻出了一腳精彩助攻，並且幫助董學升頭槌破門。但是金敬道不幸受到了傷病困擾，根據媒體報道已經因傷提前告別了東亞杯。金敬道不能踢比賽，就需要李鐵在中場位置進行調整。李鐵一直特別器重的愛將李行，很有可能在國足對陣韓國的比賽，頂替金敬道的位置出任進攻核心。

李行有一定的組織傳球能力，而且具備後排插上遠射破門絕技。希望李行要是在國足對陣韓國的比賽，頂替金敬道首發出戰之後，可以給球迷帶來眼前一亮的表現。最後在國足對陣日本隊的比賽中，一分鐘都沒有上的韋世豪，在東亞杯第二輪首發希望很大。根據媒體報道李鐵在國足對陣日本隊的比賽中，之所以沒有給韋世豪出場機會，主要是韋世豪高燒不退沒有達到上場要求。

從這幾天國足訓練中，韋世豪已經擺脫了高燒的困擾。所以韋世豪有望在國足對陣韓國的比賽，首發挑起球隊進攻大梁。值得一提的是韋世豪在上一屆東亞杯，就代表國足出戰過於韓國的比賽，並且有不錯表現打入了非常漂亮的進球。所以國足球迷希望韋世豪再次代表國家隊出戰韓國，可以延續自己抗韓英雄的本色，給球隊帶來力挽狂瀾的表現和進球，幫助國足戰勝韓國。
