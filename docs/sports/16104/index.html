英國拳手不看好泰森-富里，認為他將在二番戰中遭維爾德KO

2019-12-13 3,247條評論 4,647次閱讀 459人點贊

近日，次重量級拳壇新秀、英國人維達爾-萊利，談及了明年2月將要上演的泰森-富里VS德昂泰-維爾德二番戰。現年22歲的萊利，雖然只取得了3戰全勝，2次KO對手的戰績，不過卻在上個月，以教練身份指導了「網紅拳手」KSI與洛根-保羅之戰，最終贏得了勝利。

儘管與富里都來自英國，然而萊利並不看好富里，他認為相比一番戰，富里二番戰就沒有那麼幸運了。去年12月，富里VS維爾德一番戰在洛杉磯斯台普斯中心上演，儘管整場比賽富里表現不錯，可惜卻在最後階段被兩次擊倒，其中第二次擊倒時，在不少人看來是受到了裁判垂青。最終，一番戰雙方以平局握手言和，然而無論是富里還是維爾德，都認為自己才是勝利者。

在萊利看來，一番戰富里更加出色，只是面對重拳驚人的維爾德，再出色的拳手也有可能被逆轉。萊利說：「維爾德可以在二番戰中擊敗富里，雖然我認為富里贏得了一番戰。當時，富里有足夠的專註力，所以取得了那樣的表現，只是死而復生的方式不會在二番戰中出現了，如果接下來他被擊倒，那就會趴在那兒起不來，你不能和維爾德打賭，他可以每一個回合都輸，當打中你時，一切都結束了。」

萊利甚至認為，不久前結束的奧蒂茲VS維爾德二番戰，奧蒂茲表現要比富里更好，那場比賽結束之前，奧蒂茲同樣在點數上處於領先，然而一個閃失，卻遭到了維爾德重拳逆轉，滿盤皆輸。

萊利說：「維爾德是瘋狂的！奧蒂茲打了一場比富里更激烈的對抗，他取得了領先，然後轉身被擊中了，一切都結束了，所以你不能和維爾德打賭。」

筆者認為，明年2月上演的富里VS維爾德二番戰，肯定比一番戰更具懸念，其中最大的看點，就是維爾德重拳能否抓住機會？或者何時抓住機會？並且富里會不會讓對方攻擊得逞？一番戰中，富里除了兩次遭擊倒外，其餘時間表現均遊刃有餘，但隨著對抗深入，尤其是當體能、反應下降後，很容易被維爾德抓住機會，並且萊利有一點說得很對，一番戰富里可以死而復生，二番戰就沒有那麼幸運了！小編此前說過，富里VS維爾德複賽只有兩種結果：一是富里點勝，二是維爾德點數落後的情況下，逆轉KO對手。
