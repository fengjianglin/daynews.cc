舍普琴科給梅西投票，出於純粹的競技因素，若他是大羅會回歸米蘭

2019-12-14 2,367條評論 4,022次閱讀 144人點贊

北京時間12月14日，前著名球星、現任烏克蘭國家隊主帥舍普琴科接受了《法國足球》的專訪。在十五年前，舍普琴科獲得了金球獎，這成為了舍瓦職業生涯最光輝的時刻，談到這次榮譽，舍普琴科依舊記憶猶新：「這是我球員生涯最難忘的時刻了，這個獎項的意義也遠不止於此。」舍瓦說的「不止於此」，就是當時的烏克蘭還不是足球強國，舉國上下也就出了舍普琴科一個超級球星，所以舍瓦的榮譽不僅僅代表著他達到個人榮譽，他的身上承擔著烏克蘭足球崛起的希望。如今烏克蘭國家隊的整體戰績比舍普琴科球員時期提升了不少，雖然隊中沒有像他當時這麼出名的球星，但是烏克蘭足球的進步大家都看得到。

烏克蘭足球在新世紀以來的飛速進步，可以說無論從球員的角度還是教練的角度上來說，舍普琴科都居功至偉。現在烏克蘭國家隊以歐洲杯預選賽小組第一的身份（力壓上一屆歐洲杯冠軍葡萄牙）進軍歐洲杯正賽，在如今球迷們看來是一件很正常的事情，但是這種「正常現象」在舍普琴科還是球員時期，是無法在烏克蘭國家隊身上得到體現的，因為當時的烏克蘭真的是「歐洲三流」。所以說對於自己祖國足球的進步，舍瓦也是感慨萬千：「我知道有不少球迷希望在國內給我樹立雕像，但是我不希望如此，因為烏克蘭足球還遠遠沒有到達成功的地步。」對於烏克蘭國家隊在歐洲杯的目標，舍瓦表示他從來不會把壓力強加於球員們，他會把目標先定在小組出線。

也正是因為舍普琴科對於烏克蘭足球的特殊意義，他在這次採訪中特別表明了，他暫時不會選擇回歸AC米蘭：「執教AC米蘭的確是我一直以來的夢想，然而我只有一個，不可能分成兩半。目前烏克蘭足球更加需要我，如果我是羅納爾多的話，我就會回歸米蘭。」舍瓦口中的羅納爾多，指的是曾經效力過AC米蘭的大羅。因為羅納爾多是來自於足球強國的球星，桑巴足壇的巨星很多，「不差他一個」。在他退役之後，巴西足壇「並不是很需要他」，所以大羅有充足的時間會考慮歐洲足壇繼續自己的足球事業，現在羅納爾多就在經營西甲球隊。

若舍普琴科是大羅的話，他會回米蘭執教，然而烏克蘭足球說白了就舍瓦一個巨星，他不幫助祖國的足球發展誰幫助呢？儘管舍瓦想回米蘭，但是現實很難讓他這麼做。既然這是紀念金球獎15周年的採訪，自然舍普琴科也要談到今年的金球獎歸屬。舍瓦表示，他很難評判今年金球獎的對錯，因為他不是投票者。不過舍普琴科推斷，梅西能夠獲得今年的兩個獎項，是出於「純粹的足球」。在之前的世界足球先生評選中，舍瓦就作為國家隊主帥，把他的五分票投給了梅西，他是這樣解釋的：「我給梅西投票，出於純粹的競技因素。」

舍瓦並沒有花多少篇幅去解釋「純粹的競技」，但是個人的理解是：梅西在足球方面的造詣，的確是過去的一年裡最出色的。包括舍普琴科在內的投票者，都是站在純粹的足球角度去投票的。舍瓦撇開了名氣、公關團隊的炒作等一系列「周邊事情」，從純粹的競技因素判斷，的確是梅西在場上表現最出色，所以他就把自己的五分票投給了梅西。所以大家對於金球獎這類獎項的投票，也不必過度「深究其原因」，因為大部分投票者也許就像舍普琴科那樣簡單、直接，他們從純粹的競技因素考慮，把選票投給了今年場上足球踢得最好的球員，這沒問題。
