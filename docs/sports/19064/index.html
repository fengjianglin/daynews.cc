「評論」歐冠解簽之拜仁VS切爾西 拜仁簽運如何 純看冬季操作

2019-12-17 1,915條評論 3,159次閱讀 1910人點贊

歐冠十六強的抽籤塵埃落定了。拜仁慕尼黑將對陣切爾西。今兒就來聊聊這支簽

看到這支簽，很多人第一時間，想到的是2012年的歐冠決賽雙方的那次交鋒。拜仁曾經離冠軍很近很近，在常規90分鐘內，拜仁射門次數27-7，角球次數17-1，場面上完全碾壓。並且在久攻不下之後，終於在第83分鐘由穆勒進球取得了比分的領先。但是在常規時間的最後時刻，德羅巴的頭槌將比賽拖入了加時。而在雙方鏖戰120分鐘之後，比賽進入點球大戰，雖然馬塔的點球被諾伊爾化解。但是拜仁這邊，奧利奇和施魏因施泰格都沒有將點球罰進。伴隨著德羅巴罰進點球。切爾西捧得歐冠獎盃。拜仁遺憾的屈居亞軍。

切爾西在12年戰勝拜仁贏得歐冠

2012年的那場對決的確很經典，但其實，對於如今的雙方來說，也僅剩情懷上的內容。翻看花名冊。那場2012年決賽的切爾西大名單，如今只有1個人還在球隊里，那就是當時是球隊中場核心，如今已經成為球隊主帥的蘭帕德。而拜仁這邊，2012那場決賽18人名單和如今拜仁大名單的交集，也只剩下諾伊爾，穆勒，博阿滕三人而已。

看到不少朋友在抽籤結果出來後說夢回2012， 完全可以理解這種情懷。但也要知道物是人非，現在的兩隊和2012那會兒完全不是一回事了，那場比賽，對如今不具備任何參考價值。甚至兩隊如今的實力。和當初也無法同日而語。單看競技層面，如今兩隊的交鋒，或許無法像2012那樣，代表著當時歐洲的最高水平。全新的兩隊，即將迎來全新的交鋒。而就以目前狀態來看，兩支球隊其實有一定的類似之處。同時也有各自的優勢所在。

相似之處1：教練！

本賽季，蘭帕德成為了切爾西的主帥。而在本賽季之前，蘭帕德的獨立帶隊經驗，僅僅是帶領德比郡征戰英冠，而且最終德比郡也並未升級成功。所以，從執教一支大家眼中的強隊征戰頂級賽事的角度而言，在資歷方面，蘭帕德無疑是一個「嫩帥」

蘭帕德

拜仁方面同樣如此，在科瓦奇下課之後，弗里克接手了教鞭。弗里克在擔任拜仁代理主帥之前的獨立執教經驗，還要追溯到19年之前，帶領當時還在低級別聯賽的霍芬海姆。之後的經歷則更多的以助理教練為主，既擔任過特拉帕托尼時期的薩爾茨堡紅牛的助教，也作為勒夫教練團隊中主要負責戰術設計的智囊，跟隨德國隊一起捧得了2014年的世界盃。儘管戰術方面頗有造詣和自己的想法。但帶領頂級球會，對弗里克來說。同樣是第一次。

弗里克

除了在履歷和資歷上，兩位教頭都缺乏執教強隊征戰頂級賽事的經驗這個相似之處外，兩位教練都在短短的時間內，給球隊打下了自己的烙印，這同樣很相似。而且在戰術風格上，兩位教練給球隊所打造的戰術風格，也有相似之處。

相似之處2：踢法

在弗里克上任之後，拜仁的踢法非常的清晰，基於後防線的人員不整，弗里克從防守策略上來說，非常的講究高位逼搶。而且高位逼搶的幅度很大，整體陣型的前移非常明顯。而在進攻端，也比較注重球員的無球跑動和積極的前插，力圖將球隊的節奏加快，最終形成一種快節奏的，犀利的足球風格。而且往往是從比賽的第一分鐘就開始將自己的戰術完全執行出來，力求儘快掌握比賽節奏。

這種踢法的好處在於可以盡量將戰火燒在對方半場，減輕本方本身臨時拼湊的防線的壓力。缺點在於這種踢法本身比較消耗體能，而且身後的空當容易被利用。比如最近3輪德甲聯賽，輸給藥廠的比賽，就是身後的空當被利用，大勝不萊梅的比賽中，首開紀錄的是不萊梅，利用的依然是快速反擊。輸給門興的比賽，就是在上半場瘋狂的前壓逼搶之後，下半場體能出了問題，重新被門興掌握了節奏，並且用快速反擊打拜仁身後，最終造成了最後時刻的點球。

拜仁高位逼搶截圖 直接壓迫對方門將

切爾西這邊，蘭帕德給到球隊的要求，同樣是高位逼搶，同樣是要求前場隊員需要積極的前插。同樣是容易在身後留下空當，同樣是容易在比賽的後半階段出現體能問題。這一點在本賽季英超的前5輪聯賽中體現的尤為明顯。在前5輪比賽中，切爾西的11個進球中有8個是在上半場打進的，而之後的比賽很容易後勁不足。而且身後的空當也經常被利用。

第一輪對陣曼聯，在開場15分鐘的時候，切爾西是大兵壓境圍攻曼聯的，控球率高達75%。但是隨著比賽時間的深入，漸漸的後勁不足，壓上之後沒力氣回追了，曼聯的4個進球，1個是反擊中創造的點球，另外3個直接就是反擊進球。第二輪面對萊斯特，切爾西開場7分鐘就率先取得進球，而且控球率高達70%，但是踢著踢著，因為體能不支，下半場被萊斯特掌握了主動權，甚至最終全場控球率萊斯特還以51%-49%佔優。第三輪面對諾維奇，又是在領先的情況下被追平，一直到下半場，才依靠亞布拉罕可遇而不可求的世界波取得勝利。第四輪打升班馬謝聯，上半場2球領先，最終比分卻是2-2。同樣是後勁不足，第五輪打狼隊雖然是5-2的大勝，但是在4-0以後也曾被對手打進2球。

當然蘭帕德也在逐漸的進步中，當他發現切爾西開場階段發力過猛的問題後，也開始逐漸的注重在比賽中體能方面的平衡。但總體來說，依然存在著後勁不足的問題。切爾西本賽季在聯賽中總共31個進球，其中有24個，是在比賽前60分鐘打進的。比賽最後30分鐘，總共只有7個進球。下滑還是比較明顯的。

切爾西球員拼搶

相似之處3：風雨飄搖的後衛線

兩隊同樣採取高位逼搶策略，其實也都和雙方後衛線不是特別給力有一定的關係。落入陣地防守之後，基於人員本身不夠給力，容易頂不住。所以必須要將防守線前移，儘早的去化解對方的進攻，來減緩後衛線的壓力。

拜仁這邊的情況眾所周知。算算人頭，後防線4個位置，人員齊整也只有6個人，分別是阿拉巴，基米希，盧卡斯，聚勒，帕瓦爾，博阿滕。這個人數本身就非常單薄。結果盧卡斯和聚勒重傷，基米希又基於他的全能屬性經常要去打後腰之後，後防線人員更是捉襟見肘。於是我們能看到邊鋒出身的小將戴維斯開始打邊後衛（儘管近來進步明顯，踢的還不錯），我們能看到對抗有所欠缺的阿拉巴去客串中衛，我們能看到防守習慣還是後腰習慣的哈馬去客串中衛等等。雖然根據比賽時間，盧卡斯打切爾西的時候應該已經可以復出。但狀態如何是個問號，能否理解弗里克的戰術要求也是問號，甚至，本就人員單薄的拜仁中後場，其他球員是不是會遇到傷病困擾，同樣也還是問號。

拜仁球迷都在等待盧卡斯的復出緩解後防人荒

切爾西這邊同樣好不到哪裡去，後防線上祖馬已經多次被證明難堪大用，克里斯滕森，托莫里等人也都尚且年輕，就目前即戰力而言，他們還稱不上是歐洲的一線中衛。儘管呂迪格近期再度復出。但本賽季已經兩次遭受傷病的他，玻璃屬性相當明顯。首先要確保在接下來的2個月時間內不能再遭受傷病。其次，本賽季之前因傷出場寥寥的他，也面臨著和隊友磨合的問題。

呂迪格對切爾西後防同樣重要

所以從以上三點來看，兩支球隊是有相似之處的。不過就目前的基本面來看，拜仁似乎還是略佔上風一些。這個上風，主要還是依託於中前場球員的即戰力。

拜仁的優勢：攻擊線

在中鋒位置上，拜仁儘管只有萊萬這一名球員，但萊萬如今絕對算得上是「世一鋒」！人擋殺人佛擋殺佛。15場聯賽18個進球高居德甲射手榜第一，歐冠出場5次打進10球，高居歐冠射手榜第一。這兩項數據已經證明了一切了。

世一鋒

切爾西這邊，中鋒位置有3人，打主力的亞伯拉罕儘管算是青年才俊，但是在能力上和萊萬相比無疑差距非常大，尤其是打硬仗的能力有所欠缺。雖然這個賽季亞伯拉罕在英超層面儘管已經打進了11個進球，但是對手都偏弱，分別是諾維奇，謝聯，狼隊，聖徒，沃特福德，水晶宮，維拉。面對曼城，曼聯和利物浦時顆粒無收，甚至是完全被鎖死，基本沒啥機會的那種。在歐冠舞台，也只有在面對小組墊底一場不勝的法甲球隊里爾的時候有進球，打瓦倫西亞和阿賈克斯也沒啥發揮。至於吉魯，本賽季英超5次出場，0進球，巴舒亞伊，本賽季英超11次出場，只進1球。

切爾西前鋒亞伯拉罕

而在進攻型中場和邊路方面。拜仁擁有的庫蒂尼奧，科曼，格納布里，穆勒，佩里西奇等人，即使不能被稱作巨星，至少也都可以被稱作是歐洲乃至世界範圍內的一線攻擊手。切爾西這邊的芒特，普利西奇，科瓦契奇，巴克利，奧多伊等人，雖然都還處在上升期，尤其是芒特更是現在英格蘭著力培養的新一代攻擊中場，奧多伊曾經也是拜仁的目標。但從即戰力角度。的確還比不上拜仁。而切爾西的兩位老將佩德羅和威廉，則都已經走上了下坡路。佩德羅本賽季英超5次出場沒有進球，威廉英超16次出場，也只有2個進球而已。

博弈點：中場硬度

所以，基於雙方同是高位逼搶的戰術風格，基於拜仁中前場球員的實力比切爾西更強。這一組對決的博弈點就呼之欲出了。就是在於中場的位置的硬度了。

拜仁在弗里克治下，輸給了藥廠和門興。你光看中前場球員的紙面實力，拜仁同樣強於這兩支球隊，而且這兩支球隊的踢法，也是注重逼搶，注重快速推進，和切爾西的踢法有類似之處。所以如果拿這兩支球隊戰勝拜仁的比賽來做參照的話，切爾西無疑也是有機會的。這裡面的關鍵問題就在於，拜仁中場的防守缺乏足夠的硬度。拜仁只有哈馬這一名防守型後腰而已。一旦後防線缺人，哈馬要客串中衛。那無論基米希，蒂亞戈，格雷茨卡，托利索這4名中前衛排出怎樣的組合去進行後腰位置的搭檔。防守硬度都會顯得薄弱。所以，如果面對切爾西的時候，盧卡斯復出後狀態OK，拜仁中後場也沒有遭遇其他傷病，那哈馬回歸後腰位置，應該是一種最合適的選擇。不然的話，隱患非常明顯。

哈維·馬丁內斯

而切爾西這邊，當時最強的防守型後腰坎特在陣中，這將極大的提升切爾西在中場的防守硬度，這是他們和拜仁在陣容方面相比中的優勢！而且縱觀拜仁在弗里克上任之後所有取得勝利的比賽，除了穆里尼奧的熱刺擁有類似於戴爾這樣的英格蘭國腳級的純防守型中場外，其他對手其實在中場哪怕擁有防守後腰，能力上在國際足壇也未必排的上號。而打熱刺這一場，由於小組出線形勢早已塵埃落定，所以熱刺壓根沒有全力出擊去拼，因此也不具備參考價值。所以雖然拜仁中前場人才濟濟。但是在面對坎特的時候，他們還能討到多少便宜，這同樣是一個巨大的問號

坎特

所以，可以預見的是，如果拜仁中場到那時依然缺乏防守硬度，那切爾西的青春風暴一旦颳起，一旦將節奏拉快打的犀利一些，拜仁的後場防守隱患是很明顯的。而拜仁哪怕中前場牌面看似更加豪華，但坎特的存在，對於無疑會是一道屏障，對拜仁的進攻起到足夠的限制。

所以，對於這個抽籤的結果來說，站在切爾西的立場的話，其實未必是個壞事，門興和藥廠可以戰勝拜仁。那球風和這兩支球隊相近的切爾西，同樣也有了依樣畫葫蘆的機會！但是對於切爾西來說，雖然轉會禁令已經解除，但是，鍛煉年輕人的路子已經確定，蘭帕德又是非常喜歡用年輕人的教練，所以，切爾西會不會去買人本身就是個問號。所以這一組對抗最終會朝著怎樣的走勢發展，其實還是取決於拜仁自身。能否將抽中切爾西變成上上籤，就看拜仁冬窗能不能操作一波了。畢竟拜仁不像切爾西這樣打定主意要用年輕人，而且人手本就不足。

拜仁高層冬窗願意投入嗎？

前文已經說的很清楚了，拜仁的後防不給力，中場硬度是隱患，但切爾西攻擊線打硬仗的能力本身並不算強。所以，拜仁除了指望盧卡斯趕緊復出並找到狀態之外，本身中後場人數不足的問題還是需要得以解決。不然就只能祈禱盧卡斯復出的同時，拜仁中後場除了聚勒外，不再增加任何一名新的傷員。但以拜仁過去多年的傷病傳統來說，難！包括最近1個月里，博阿滕，阿拉巴等人都曾相繼受傷。後腰位置的格雷茨卡和托利索也出現了傷病。

因此拜仁如果能夠在冬歇期的時候，補充後防線的人手，來消除傷病的影響，同時讓每個人位置上的球員，符合他本身的特點，盡量杜絕和避免客串的問題發生。那哪怕依然使用高位逼搶的戰術，在尺度和幅度上，也可以在平衡性上有更多選擇，不需要將尺度和幅度設置的像現在一樣那麼極端，身後的空當自然也就小不少，那留給切爾西能去利用的空間自然也就大大縮小。只要拜仁能做好防守，唯一的懸念，就要看在面對坎特的時候，拜仁的攻擊手能展現出怎樣的狀態了，只要能想辦法突破坎特這一關，那切爾西本身孱弱的後衛線，不會是拜仁中前場攻擊群的對手。所以總體來說，只要做好中後場防守，拜仁的贏面無疑是大大增加的。

但如果冬歇期時候拜仁最終沒有對後防線進行補強，那基於拜仁輸給過門興和藥廠，所以面對風格類似的切爾西，可能真的也未必是一支好籤。至少絕不是一帆風順的上上籤。甚至有可能形成拜仁基於中前場球員實力更強，佔據場面優勢，但是坎特的存在讓拜仁的進攻效率打了折扣，反而因為拜仁身後的空當，被切爾西打反擊。
