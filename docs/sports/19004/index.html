聯賽杯｜利物浦故意放棄，曼城志在衛冕，曼聯和萊斯特城作何感想

2019-12-18 876條評論 3,679次閱讀 2500人點贊

聯賽杯作為英格蘭的一項傳統賽事，素有「爆冷溫床」之稱。和足總杯一樣，它也是一場定勝負，但是，如果打平便要擇日再戰，這對賽程吃緊的大多數英超球隊而言苦不堪言，所以往往就會過早的選擇放棄，只是作為替補和青訓球員的練兵場。

可是，一旦進入1/4賽事之後，又往往會引起重視，畢竟也是一項錦標！3場球就有可能奪得一個獎盃，這可比英超冠軍難度小多了。

從本賽季的8強來看，無論出於什麼情況，英超球隊多達6支，這也實屬難得。另外一個有意思的現象是，曼徹斯特（曼聯/曼城）和默西塞德（利物浦/埃弗頓）各有兩支球隊，半決賽、甚至決賽的對陣，也讓人值得期待。

為了參加世俱杯，在協商無果之後，克洛普的利物浦派出了「純二隊」參賽，赤裸裸的要放棄的節奏。這對曼城、曼聯和萊斯特城來說，絕對是好事，畢竟又少了一個強勁的競爭對手。

尤其曼城和曼聯本輪的對手都是絕對意義上的「弱旅」，無論首發陣容如何，只要球隊端正態度，晉級半決賽幾乎就是板上釘釘的事。並且，對於兩隊的主教練和球員而言，本賽季的英超可以說已經提前結束了，至於歐洲賽場也是困難重重。面對這個賽季唯一可能的桂冠，應該也是志在必得！

面對利物浦送的這個「瓜」，維拉該作何打算？

維拉對陣利物浦，雙方最近5次交手，利物浦5連勝佔據絕對優勢。維拉最近狀態糟糕，排名基本就是降級區;而利物浦則是一路高歌猛進、穩居榜首。

上輪比賽，維拉淘汰了「放棄比賽」的狼隊，而利物浦則是「出於無奈」的贏下了阿森納。本場比賽，面對「完全不一樣」的利物浦，不知維拉到底作何打算？是憑藉主場之利，全力拿下比賽，去努力一把冠軍？還是做個順水人情，再送紅軍一程，而自己還是回到聯賽去老老實實的保級？看好維拉晉級。

曼城志在衛冕，面對英甲球隊應該會是一場「砍瓜切菜」的勝利吧？

牛津聯對陣曼城，雙方在聯賽杯中曾有過一次交手，曼城客場3-0戰勝對手。英甲球隊牛津聯最近同時參加3大杯賽，也影響到了聯賽成績;曼城最近狀態還可以，歐冠晉級，本輪聯賽客場更是3-0大勝阿森納。

上輪比賽，牛津聯淘汰了聯賽對手桑德蘭，而曼城則戰勝了南安普頓。本場比賽，對於牛津聯而言他們可能更希望能做客曼城比賽，輸贏其實已無所謂;作為聯賽杯的衛冕冠軍，曼城由於本賽季聯賽基本無望，歐冠不確定因素又太多，面對這個極有可能的冠軍獎盃，應該是志在必得。看好曼城晉級。

聯賽狀況雖然大相徑庭，萊斯特城估計會面臨和埃弗頓同樣的困擾！

埃弗頓對陣萊斯特城，雙方最近5次交手，萊斯特城3勝2負稍佔上風。兩隊本賽季的成績和狀態可謂大相徑庭，前者狀態糟糕、排名臨近降級區，而後者狀態大好、排名高居第二。

上輪比賽，埃弗頓贏下了沃特福德，而萊斯特城淘汰了英甲球隊伯頓。埃弗頓雖是主場作戰，但不知球隊的心思到底在不在這？對於客場作戰的萊斯特城而言，同樣面臨這樣的問題，力拚對手晉級有可能會面臨「曼市雙雄」的阻擊，放棄比賽會不會有點過早？看好萊斯特城晉級。

曼聯手握上上籤，不會和聯賽一樣「送溫暖」吧？

曼聯對科切斯特，雙方從未交手。曼聯最近狀態有所回升，但是，本輪聯賽戰平埃弗頓，再次上演「劫富濟貧」的戲碼，錯失追分的同時，排名也被熱刺超越;作為英乙的強隊，對於科切斯特的球員而言，能夠做客老特拉福德比賽應該比結果更重要。

上輪比賽，曼聯贏下了切爾西，而科切斯特淘汰了聯賽對手克勞利。和曼城一樣，聯賽杯也可能是曼聯本賽季唯一的冠軍獎盃，如果索聖時間還夠的話，也算是他對俱樂部的一個交代。且不說下一輪會碰到誰，如果這場比賽輸球，估計索聖就直接下課了！看好曼聯晉級。
