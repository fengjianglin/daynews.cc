她是“滿分”奧運會冠軍，退役後去美國，被網友罵是“叛國”行為

2019-12-22 229條評論 1,468次閱讀 201人點贊

​能夠拿到奧運冠軍多少運動員的畢生夢想，說起來我們中國的奧運健兒們也為國家奪得了無數的金牌，他們每一位都是非常優秀的運動員。我們中國一直是世界上的體育大國，中國一直都是世界上體育成績最優秀的體育大國之一。我們在北京奧運會上的優異表現拿到了獎牌數的第一名。在中國體操運動或許是拿到最多奧運獎牌的體育運動之一了，。中國的體操運動實力一直都非常強勁。通過體操這一項運動，讓世界上的不少國家認識到了我們中國，認識到了中國的體育實力。

體操運動中出現過了太多優秀的體操運動員，有很多的運動員我們想必都非常熟悉，就比如李寧、程菲這樣的優秀運動員。今天我們要說的這位體操運動員並不是最出名的運動員，她的名字叫陸莉，她是「滿分」奧運會冠軍。

這個名字或許很多的體育迷們都沒有聽說過，但是她曾經在1992年的巴塞羅那奧運會上獲得了體操高低杠冠軍。她那時候奪冠的成績是滿分，這是非常驚艷的成績。這一件事情在當時不僅僅是聞名於中國，還震驚了世界。

陸莉出生於1976年，她出生在一個富裕的家庭，因為挑食導致她的身材非常的瘦小，父母一度以為她有厭食症，但是最終用了很多的方法還是沒有太大的作用。最終父母把陸莉送到了體操學校，希望能夠讓他做運動來恢復和提高自己的食慾，確實非常有效，而且陸莉因為身材矮小瘦弱，加上對於體操的天賦，她的很多動作都完成得非常優秀，得到了體操老師的欣賞。

在這之後，陸莉開始學習專業體操，在學習專業體操之後，自然是一件非常痛苦的事情，陸莉那時候也曾經想過要放棄，但是陸莉還是最終堅持了下來，陸莉在努力的訓練之後，確實有了回報，她的成績進步的飛速，在當時基本上是隊內最優秀的運動員。陸莉很快就進入了體操隊，在體操隊的時候得到了熊景斌教練的賞識，這一位教練可以是說是陸莉的恩師了，為陸莉設計了一套專業的體操運動，在當時陸莉也是努力的訓練這一套動作，最終取得了不錯的成績。在後來的巴塞羅那奧運會上，陸莉用這一套動作完成了滿分的成績。陸莉的名字也是從那時候起開始聞名世界。

長期的體操訓練對於陸莉的身體影響非常大，在18歲的時候陸莉就選擇了退役，在退役之後陸莉選擇走到了大學去讀書，在畢業之後去美國引爭議。因為當時大家的思想還比較保守，陸莉受到了很多的國內的網友的議論，很多人都認為陸莉會加入美國國籍，認為陸莉的行為就是「叛國」行為。當時陸莉自己也表示絕對不會更改自己中國國籍，在之後嫁給了美國的丈夫，如今他的兒子是美國國籍，但她自己仍舊是一名中國人。
