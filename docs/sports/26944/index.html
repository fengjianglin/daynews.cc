山西要崛起，必須要解決的關鍵點！

2019-12-23 2,412條評論 4,636次閱讀 1407人點贊

冬至大戰，面對首鋼，山西猛龍被鬧蒙了，整場士氣不佳，只得到70多分，但還是找到了一些關鍵問題！

首先，球隊需要一個第二持球點，富蘭克林，被針對！球隊還是一板一眼，找他！時間被延誤，大部分出手都是逼著來，這一點不是自己的節奏，穩定性不高！而落陣地，大部分戰術是原帥需球來來開，當對方再預判原帥跑動路線的同時，戰術就出不來了！所以我們很多次看到國內持球人會進攻停滯的畫面，這就是猶豫不知道怎麼打的表現！然後富蘭克林又得過來接應，球到手了，也該出手了，所以打的很不舒服！我們看到了，他在單幹和助攻的節奏上，也出現了矛盾，影響了球隊士氣！所以，山西，需要找到第二套進攻持球點，兩套靈活變化，才能主動出擊！

再者，山西內線，需要明確每個人的打發！葛召寶和莫蘭德，不能一樣的戰術，兩人都有缺點！利用好優勢就可以！大寶，傷病大有影響，已經不能勝任首發這個位置了！首發只會讓對手抓著打！衛猛不知道，目前什麼情況，王指導可以考慮一下！畢竟賽季才過1/3，還有時間調整！

最後，說說，山西有時候會上五小！考慮每個球員的條件，有的環節上，可以考慮多打打來彌補內線的問題！像面對北京這種防守球隊，以後我們可以把速度帶起來，或許還有一些可能！

總之，山西，需要靈活！山西需要精氣神！很多時候，看比賽只要自己落後5分，就糾心，就看到球員有不安的狀態！這完全不行，有幾場落後20分還能追回來的那股勁，已經很久沒看到了！是因為，大家發現，以前的打發，不起作用了！心裡沒底！而那個時候，之前只要拿出一個戰術就馬上有效果！而且各球隊都在磨合中，現在大家都成形了！所以，很多戰術都已經被針對了，球員就會出現迷茫不自信！考驗山西的時刻，已經到了！再努力吧！首鋼的調整真的有底蘊，很快就過來了！山西國內球員，也需要一個精神領袖，把這股勁帶起來，就像可蘭這種，書豪這種！
