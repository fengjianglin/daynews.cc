歐冠奪冠賠率變化：利物浦仍第2，曼城升0.5，西甲豪門驟增

2019-12-19 354條評論 4,961次閱讀 2475人點贊

上周中，本賽季歐冠十六強全部出爐，歐洲五大聯賽歷史性的首次包攬了1/8決賽的所有名額。瑞士尼翁當地時間周一，令人期待的歐冠淘汰賽對陣通過抽籤產生，8組對決分別為：多特vs 巴黎；皇馬vs 曼城；亞特蘭大vs瓦倫西亞；馬競vs 利物浦；切爾西vs拜仁；里昂vs尤文；熱刺vs萊比錫；那不勒斯vs 巴薩。1/8決賽對陣出爐後，最新的奪冠賠率也隨之公布，各大豪門在抽籤前後都有少許變化，上屆冠軍利物浦仍居第二、總身價13億歐元的曼城升0.5，西甲豪門馬競則驟增。

首先，在整體的排名上，只有兩支球隊在排位上有所變化，抽到奪冠大熱門巴薩的那不勒斯從第十二降至第十三，而將與大巴黎爭奪八強名額的多特從第十三升至了第十二。前四的排名則依次為：曼城、利物浦、巴薩和大巴黎。值得一提的是，歐冠史上最成功的球隊、2018年冠軍得主皇馬僅與熱刺並列第七。在奪冠賠率的變化上，部分豪門在淘汰賽對手出爐後，各自有所增減。

要說1/8決賽最重磅的對決，非皇馬與曼城的較量莫屬。從球員總身價上看，曼城以13億歐元排名世界第一、皇馬11.9億歐元，緊隨英超冠軍身後，這場對決將是當今世界上的最貴對決，總價值高達24.9億歐元。另外，兩隊主帥齊達內（3次）和瓜迪奧拉（2次）總共拿到過5次歐冠冠軍，這是他們首次在歐冠進行鬥法。當抽籤結果出來後，兩支熱門球隊的奪冠賠率便有所提升（提升表示更不被看好晉級，反之則更被看好），曼城從抽籤前的4.0增至4.5，皇馬則從17.0增至21.0。

上屆冠軍利物浦與西甲豪門馬競的對決同樣備受關注，紅軍紅箭三俠是歐洲足壇攻擊力最強的組合之一，而馬競則是公認防守最好的球隊。從賠率上看，利物浦在這場「矛盾對決」中，成為更被看好能夠晉級的一方，克洛普的球隊（5.5）僅有0.5增漲，而馬競則從原先的26.0增至34.0，增幅達到了31%。

另外一場強強對話為切爾西VS拜仁，這是2012年歐冠決賽的重演。拜仁是本屆歐冠小組賽中唯一保持全勝的球隊，德國冠軍的奪冠賠率從9.0降至了8.0，切爾西（41.0）則保持不變。另外，巴薩（7.0→6.0）和熱刺（26.0→21.0）這兩支避開強強對話的豪門均有所下降。抽到多特的大巴黎從7.0增到8.0，C羅領軍的尤文（13.0）則繼續保持不變。從賠率上看，曼城依然是本賽季的最熱門球隊，而你覺得最有希望奪冠的球隊是哪支呢？
