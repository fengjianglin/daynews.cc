山東男籃戰平遼寧，三分球創新低：陶漢林單節11分哈德森單節三犯

2019-12-20 1,919條評論 3,301次閱讀 1509人點贊

12月19日晚19點35分，CBA常規賽第十九輪，山東男籃在主場迎戰強大的遼寧男籃。山東隊近期取得7連勝士氣正盛，核心外援哈德森保持了極佳的競爭狀態，面對舊主他再度火力全開。 目前山東隊以11勝7負排名第9位。目前遼寧隊以12勝6負排名第4位。兩隊歷史上共交手58場，遼寧隊以40勝18負佔據絕對優勢。

山東隊首發陣容：王汝恆、賈誠、陶漢林、李敬宇、哈德森；遼寧隊首發陣容：趙繼偉、史蒂芬森、郭艾倫、李曉旭、韓德君。

第一節開始，李曉旭上籃得手，率先得分；哈德森持球推進，繞過掩護，弧頂超遠3分，不中；隨後哈德森防守韓德君形成爭球，韓德君貌似隱形眼鏡掉了，被賀天舉換下，直接回更衣室了。隨後賈誠、哈德森、李敬宇先後得分，山東隊7-4領先；陶漢林接到韓德君傳球，打成2+1；然後陶漢林再次打進2分，12-4，打停遼寧隊。

暫停之後，郭艾倫持球進攻，衝擊籃下造成犯規，2罰1中；遼寧隊5次遠投不中，哈德森連續2次犯規，需要穩一下了；郭艾倫繼續遠投不中，哈德森妙傳陶漢林，上籃造成犯規，2罰2中，14-5；李敬宇三分不中後，賈誠投進2分；趙繼偉打進3分，遼寧隊第7次遠投終於投中；王汝恆突破投籃不中，賈誠補籃筐；此後哈德森防守史蒂芬森被吹罰第3次犯規，2罰1中；巴斯替換史蒂芬森上場；梅斯也上場換下哈德森。

陶漢林拿球，背身打李曉旭打成拿到2分，22-13；哈德森下場後遼寧隊連追4分，22-17；巴斯再得2分，22-19；關鍵時刻，張慶鵬命中2分，24-19；最後時刻，李敬宇投3分不中；而遼寧隊高詩岩投中3分，漂移的壓哨三分！第一節比賽結束，山東隊25-22領先3分。陶漢林單節拿到11分，但山東隊三分球7投1中，遼寧隊11投2中

第二節開始，巴斯率先投進三分，扳平比分；此後李敬宇三分不中，李敬宇本場還沒有找到手感；郭艾倫造成犯規，2罰中1，26-25反超山東；關鍵時刻，李敬宇命中2分；巴斯打中2分後，梅斯造成犯規，2罰1中，雙方再次打平。此後，陶漢林籃下暴扣得分，來自梅斯的妙傳！王汝恆、張慶鵬連續出手三分不中，山東隊三分命中率太差了，14投1中。王汝恆與哈德森的配合也形成失誤，鞏曉彬叫了暫停。

暫停回來，哈德森遠投不中後，梅斯斷球暴扣得分；但史蒂芬森投進三分，將比分再次反超為33-32；哈德森關鍵時刻突破上籃，造成犯規，2罰2中；此後雙方你來我往，屢投屢不中。郭艾倫給左側弧頂李曉旭，三分命中，巴斯再進兩分，34-38；關鍵時刻，李敬宇打進2分，雖然李敬宇三分球沒有投進，中投還是比較準的；此後哈德森高拋命中，38-39；巴斯造成梅斯打手犯規，2罰2中，38-41；王汝恆讀秒打進3分，41平；隨後，雙方各有一次進攻不中，上半場結束，雙方達成41-41。

上半場雙方力拚防守，命中率都很低。山東隊三分球21投2中，超低的數據；遼寧隊三分球19投5中，命中率也不高；山東隊兩分球40投16中，遼寧隊44投15中，山東隊略高一些，但都有些偏低。

半場數據，山東隊方面：陶漢林11分5籃板、哈德森7分7籃板3助攻、王汝恆7分2籃板、李敬宇6分3籃板2搶斷、賈誠4分2籃板、梅斯3分6籃板2搶斷、張慶鵬3分。

遼寧隊方面：李曉旭7分3籃板、史蒂芬森6分4籃板、韓德君6分4籃板、郭艾倫5分2籃板2搶斷、趙繼偉3分4助攻、巴斯9分6籃板、高詩岩5分2搶斷
