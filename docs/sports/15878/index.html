改國籍的跳水世界冠軍，歷盡艱辛終圓奧運夢想，為圓夢帶全家移民

2019-12-15 1,459條評論 4,577次閱讀 1226人點贊

北京時間12月14日11:34，前中國跳水世界冠軍，現入籍澳大利亞的李世鑫在個人社交平台上發布了新動態，動態中和網友分享了自己參加大洋洲跳水錦標賽暨東京奧運會資格賽的獲獎照片。並配文寫道：「大洋洲跳水錦標賽暨東京奧運會資格賽。贏下一個珍貴的奧運會席位。是的，前所未有的與那舞台如此接近！」

從動態中可以看出來，這位前中國跳水世界冠軍在奧克蘭大洋洲跳水錦標賽暨東京奧運會資格賽中一舉奪金，同時也贏下了一個對他來說無比珍貴的奧運會席位。這位改國籍的跳水世界冠軍，歷盡艱辛終圓自己的奧運夢想，可喜可賀！為圓夢奧運，李世鑫放棄了國內的一切，帶著老婆孩子全家移民澳大利亞，甚至放棄了自己的中國國籍，入籍澳大利亞！

1988年12月出生的李世鑫，今年已經31歲了，他曾是中國海軍隊的選手，也是中國國家跳水隊的主力隊員。2014年底他從海軍隊退役，2017年，他還復出代表上海參加了全運會。他職業生涯中取得的最好成績是在2011和2013年連續斬獲世錦賽1米板金牌，成為1991年游泳世錦賽設立男子1米板項目以來，唯一蟬聯過冠軍的運動員。

除了比賽成績，李世鑫身上更出名的是他身上那個「清唱國歌男」的愛國標籤，2011年2月25日，在國際泳聯跳水大獎賽俄羅斯奔薩站的比賽中，中國選手張新華、李世鑫包攬了男子3米跳板冠亞軍。但在隨後的頒獎儀式升國旗、奏國歌時，因為主辦方的失誤，竟然播放錯了中國國歌，情急之下，站在領獎台上的亞軍得主李世鑫突然發聲清唱起了國歌，他沙啞的聲音感動了全場，回國後李世鑫也一唱成名。

不過雖然名氣有了，但在高手如雲的中國跳水「夢之隊」，只奪得過兩枚非奧運一米板項目世錦賽金牌的李世鑫，顯然競爭力不足，他的奧運夢想一直遙不可及，在2012年倫敦奧運會選拔賽上，奮力一搏的李世鑫在激烈的競爭中遺憾的再度失利，再度與奧運夢想擦肩而過！

2014年10月25日，李世鑫在獲得國際泳聯跳水大獎賽馬來西亞站男子三米板金牌後，開始淡出國家隊。此後參加奧運無望、心灰意冷的他選擇了退役，之後他拒絕了組織上安排的工作，自己當了一名潛水教練，在工作中他收穫了愛情，結識了現在的妻子，相識相戀，結婚後生了第一個女兒。

不過倔強的李世鑫並不想放棄自己的奧運夢想，在國內無法實現，他決定另闢蹊徑。2017年他賣掉國內產業，帶著妻女舉家移民澳大利亞。在那裡，他改國籍重新復出，今年5月份，李世鑫正式拿到了澳大利亞的公民身份，此後他代表澳大利亞參加了今年的光州世錦賽，面對外界對他「歸化」他國的質疑，李世鑫表示：「我不奢求能夠得到大家的理解，但每個人都有權利選擇自己的路。」

今年的光州世錦賽上，李世鑫雖然是代表澳大利亞出戰，但是比賽前他都會和之前的中國隊友互相打氣，比賽跳完動作後他們也會緊緊擁抱，一起自拍，好像自己仍在夢之隊的名單裡面。李世鑫在比完自己的項目後，在看台上，他也和中國隊的前隊友坐在一起，與以前一樣，為中國隊的比賽加油，好像自己一直都是中國跳水隊的隊員！

去年李世鑫又有了一對雙胞胎女兒，這樣他就有了三個孩子，而參加澳洲當地比賽又沒有獎金，李世鑫的生活壓力也徒增。李世鑫在澳洲沒有正式的穩定工作，接受採訪時他說，自己每天6點起床，上午照看孩子，下午3點到6點去訓練。而他能夠賺錢的時間只有晚上6點到8點到俱樂部去當兼職游泳教練，時薪25澳元，相當於100多人民幣。此外，他還兼職做一份健身教練的工作，每天晚上9點才能回家休息。

今年10月4日，李世鑫在微博上發文表示自己因為沒錢交註冊費，被禁止訓練了，他寫道：「練了一半，被跳水協會的工作人員告知禁止訓練… 說我新季度的註冊費沒交。」11月1日，李世鑫又發微博表示：「想訓練都是挺不容易的… 得躲著場地工作人員，被發現了還得被清出場地… 不知道是否被列入黑名單了！」看來這個時候他還沒交上註冊費，只能偷偷摸摸訓練。

打工養活三個孩子，交不起註冊費這一件件的不如意事，不得不讓人感嘆這位31歲的中國跳水前世界冠軍歸化國外，獨自闖蕩的艱辛! 不過為了奧運夢想，這樣的日子，李世鑫沒有任何抱怨。歷盡艱辛，他終於可以出現在2020的東京奧運了，希望他取得好成績，加油，李世鑫！
