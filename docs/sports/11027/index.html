貝克漢姆指定的引援！江蘇蘇寧舊將R馬獲認可，或前往美國踢球

2019-12-12 1,653條評論 3,717次閱讀 738人點贊

哥倫比亞前鋒羅傑-馬丁內斯曾經在中超賽場上給中國球迷留下了深刻的印象，他強壯的身體，和充滿爆發力的奔跑，讓他一度成為江蘇蘇寧鋒線上最值得信賴的進攻球員，然而年輕的R馬心高氣傲，一直希望前往比中超更高的平台去實現自己的價值，在2018年被江蘇蘇寧送到了西甲比利亞雷亞爾去追求夢想。

​然而現實往往是殘酷的，來到比利亞雷亞爾的R馬也感受到了與世界最高水平的足球差距，在比利亞雷亞爾的半年時間裡，獲得了9次出場機會打進了1粒進球，在半年之後，就被比利亞雷亞爾賣到了墨西哥超級聯賽豪門墨西哥美洲，一直效力至今。

​在墨西哥美洲R馬也找回了自己最熟悉的踢球方式，上賽季為球隊出場44次，打進15球還有3個助攻，成為球隊的主力前鋒，本賽季墨超聯賽也有19次出場，貢獻了2球7助攻。在墨西哥的良好表現，也吸引了美國職業足球大聯盟新軍，邁阿密國際隊老闆貝克漢姆的注意。

​據外媒《theathletic》報道，邁阿密國際正在努力和R馬進行談判，他也是俱樂部老闆指定要引進的前鋒球員。目前R馬與墨西哥美洲的合同還很長，直到2023年6月30日才到期，而R馬目前的身價為500萬歐元，而貝克漢姆背後有多個強大財團支持，對於資金實力雄厚的邁阿密國際來說，引援費用並不會是太大問題，只有墨西哥美洲肯放人，R馬也有前往美國踢球的意願，這筆交易預計很快就會完成。

​R馬的職業生涯已經先後效力過6傢俱樂部了，在當初加盟中超江蘇蘇寧時，僅僅只有22歲，如今才25歲的R馬依然還很年輕，還有潛力可以激發。正處於自己競技狀態上升期的R馬，若加盟美國職業足球大聯盟，他將會創造什麼樣的數據表現，也十分令人期待。
