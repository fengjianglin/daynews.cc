足協秘書長：網傳新政不夠成熟，足協將進一步放權

2019-12-05 0條評論 21次閱讀 0人點贊

摘要

北京時間12月5日，昨日在中乙頒獎典禮上，足協秘書長劉奕特別澄清了有關近日相傳的足協新政，同時表示中國足協將進一步放權。他說到：“國足在迪拜兵敗敘利亞之後，為我們衝擊2022年卡達世界盃又憑添許多荊棘，所以這次要求我們拿出實實在在的舉措和改革的方針和政策。

北京時間12月5日，昨日在中乙頒獎典禮上，足協秘書長劉奕特別澄清了有關近日相傳的足協新政，同時表示中國足協將進一步放權。

他說到：「國足在迪拜兵敗敘利亞之後，為我們衝擊2022年卡達世界盃又憑添許多荊棘，所以這次要求我們拿出實實在在的舉措和改革的方針和政策。那麼我們在想，剛剛今天下午也討論到了，職業聯盟的成立以及對中乙、中甲、中超三級聯賽下一輪的改革，之前大家也都看到了（網上）對這些政策的一些廣泛的討論。

那麼我今天澄清一下，這些政策現在都還不成熟，到時候總的一個目標是，中乙、中甲、中超三級聯賽能夠健康的發展，那麼中國足協的未來，在新一屆班子的帶領下，我們將更加的放權，讓市場發揮更多的活力，這是我們（可以）唯一擔保的一點，所以我們要充分意識到中乙聯賽對於中國職業聯賽發展具有重要意義。」

最後，根據央視新聞的報道，劉奕再次強調了中國足協將要放權。中國足協將不再持有中超聯盟的股份，退出聯賽的日常管理，下賽季將只行使重大事項的監督監管權。在未來的足球聯盟董事會，中國足協在重大事件上擁有一票否決權，但不參與日常運營。
