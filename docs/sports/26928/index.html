山西主場慘敗王非不滿隊員打的太軟 第三節居然沒有一次犯規

2019-12-23 1,762條評論 3,362次閱讀 1358人點贊

北京時間12月22日，山西男籃主場迎戰北京首鋼，山西全場只得到了78分，創下賽季新低。賽後主教練王非表示，北京隊給我們上了一課，山西隊打的沒有侵略性，拼勁和鬥志都不如北京。

在採訪中，記者問如何看待只拿到了78分時，王非指導談到，」北京是全聯盟防守最好的球隊，不論是個人防守還是整體性上都是做的非常不錯的。所以整場比賽我們只得了78分，上半時31分，即使我們防守對方得到93分，但是進攻沒有太好的空位機會，球的周轉很困難，變成單打獨鬥是北京所希望的，同時對方內線有封蓋最好的球員，這樣會給進攻帶來麻煩。

在談到山西隊目前還有那些不足的時候，王非指導很誠懇的坦露，北京隊在很多方面都比山西做的好，值得山西去學習。

」一旦我們在之前比賽進攻當中做得好的時候，可以有三個到四個得分上雙的球員，今天就比較困難，這些是我們需要向北京學習的，我們球員需要打得更強硬，雖然北京有很多國手在一起時間長也很有經驗，我覺得我們球隊需要打得更有侵略性，整個下半時做的是不夠的。第三節沒有犯規，第四節只有3次，拼勁、鬥志包括身體接觸需要更好的提高，和北京隊比賽學習很多東西，希望球隊在過程當中知道該如何做得更好，現在其他球隊對山西隊的重視和對外線的壓迫比較大，需要更好的改善和提高。「
