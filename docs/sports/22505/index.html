35歲奧運冠軍陳一冰，如今創業成功當上總裁，妻子是翻譯界美女

2019-12-20 77條評論 4,161次閱讀 2918人點贊

在我國，很多體育界的運動員在退役之後會走上創業的道路，比如中國體操界的「體操王子」李寧就讓李寧體育品牌享譽世界，且市場前景越來越好，比如林丹老婆謝杏芳，退役前也是我國羽壇一員大將，如今創立的「杜芬」母嬰品牌公司非常成功，轉身成為美女總裁。除了他們之外，中國體操界還有其他一些運動員通過創業來展現個人能力，實現抱負，今年35歲的奧運會冠軍陳一冰也在退役後開始創業。，如今的陳一冰成為了公司總裁，並且在生活中也抱得美人歸，他的妻子是翻譯界的知名美女！

職業生涯

1984年出生的陳一冰的體操啟蒙比較早，因為出生時是一名早產兒，他的身體健康情況明顯不如其他孩子，父親為了鍛煉他的身體素質，就把五歲的陳一冰送進了體操隊，為的是強健體魄。沒想到性格頑強的陳一冰在體操方面做得越來越好，因為實力突出，他很快也得到了體操教練的重視。

1994年，陳一冰進入天津體操隊，才十歲的年紀，就已經展現了他非凡的天賦。經歷過長達六年的迷茫期，陳一冰逐漸開始確定人生目標，2001年獲得了進入國家隊的機會，但是他卻多年未能回家和家人團聚。2008年北京奧運會上，陳一冰終於迎來實力大爆發，個人獲得兩枚奧運會金牌，一個是男子團體冠軍，一個是男子吊環冠軍，陳一冰也被稱為「冰王子」和「吊環王」。

感情生活

長相帥氣的陳一冰曾和體操女神何雯娜擁有一段戀情，不過最後分手結束戀情，退役之後的陳一冰開始創業過程，當時的他面臨著兩個選擇，留在天津體育隊擔任教練，或者去社會創業。陳一冰選擇後者，開辦體育公司，並且創業成功，小有成就，成為了公司的總裁。因為個人有人脈，陳一冰的創業之路要比一般人簡單一些，但其中的一些心酸也只有自己知道。

陳一冰也擁有了一段幸福的婚姻，妻子曾經是一名美女翻譯，名叫單競緹，為了追求單競緹，陳一冰費勁了心思。終於在兩人相戀後，相互扶持，妻子支持陳一冰的創業事業，最終和他一起走向人生巔峰。在此我們祝願陳一冰和妻子能夠在之後的生活中更加幸福。
