國足惡戰前傳來兩個好消息，此人是艾克森最大幫手？李鐵也要變通

2019-12-18 1,004條評論 3,089次閱讀 242人點贊

國足的東亞杯之旅雖然沒有那麼受到重視，但是畢竟對於國足而言，想和韓國和日本有正式的交鋒目前來看是比較難的，即便對方不是一線陣容，但是對於我們而言也絕對有鍛煉的價值，畢竟如今的國足在整體實力上來看，已經快跌破二流水準了。而在接連輸掉兩場比賽之後，面對中國香港，對於國足而言，其實也是一場惡戰，畢竟國足在最近兩次交鋒當中，均被中國香港0:0逼平，而且是在世界盃預選賽這樣的大賽上，而且在國足並非全部主力的情況下，這次交鋒絕對是一場惡戰。

不過對於國足而言，這場惡戰前傳來兩個好消息，一個就是韋世豪出戰的幾率非常高，在里皮時期，韋世豪就是被重點照顧的對象，不過讓球迷們不理解的是，明明韋世豪邊路突破更加系列，而且可以非常好的搭配艾克森，但是里皮卻總喜歡在中路放楊旭，而讓艾克森頂替韋世豪的位置，這就很尷尬了，希望國足的下一期主帥能夠有所變通。

另外一個好消息那就是國足面對中國香港，很有可能決定了未來主帥的歸屬，也許國足完成了一場大勝，那麼李鐵很有可能會被扶正，這樣國足有了主帥，最起碼將在位，才能更加穩定，國足的備戰才能更加充分，雖然對於球迷們和媒體而言，仍然覺得洋帥才是最佳的選擇，但是從卡馬喬開始，似乎即便是大牌的教練，也無法幫助國足完成質的改變，而此前的高洪波險些創造奇蹟，而李鐵雖然可能不是最好的選擇，但是也未嘗不是一個好的開始，但是即便是李鐵，也是需要變通的。

很多球迷認為此人是艾克森最大幫手。此前的國足比賽，很多球迷認為艾克森被孤立，似乎除了幾名球員之外，其餘的球員都不給艾克森喂球，而相信很多球迷都對這一幕印象深刻，艾克森跑出了機會，里皮也示意王剛傳球，但是王剛看了一眼，將球回傳，很多球迷認為這是對艾克森的鼓勵，其實我更願意相信這是對於自己的不自信，害怕將球傳丟的表現。但是不得不說，韋世豪絕對是艾克森的絕配，而且在俱樂部當中，兩人就是隊友，再加上武磊，這條鋒線其實還是非常值得期待的。

但是如果真的是李鐵的話，也許要變通，在對陣日本的比賽中，在李昂受傷的情況下，李鐵很顯然沒有做出最正確的選擇，依然是按部就班的換人，而直到確認李昂無法堅持，才無奈被迫連續換第二個人，這也導致了日本隊直接攻破了國足的球門，將比賽的優勢擴大，不過李鐵畢竟在國足主帥的位置上也很少，還有很多需要提升，而且這次丟球或許也就是一次巧合，而李鐵也正是想要做出這樣的判斷。但是在球員的選擇上來說，確實還是要變通一些，不能一成不變，還是要有針對性。

對陣中國香港的惡戰一觸即發，那麼這場比賽會是怎樣的結果呢？會是一片謾罵聲，還是鼓勵和讚揚的聲音呢？李鐵又能否成為國足的主帥呢？韋世豪能否在這場比賽立功呢？讓我們一起拭目以待吧。
