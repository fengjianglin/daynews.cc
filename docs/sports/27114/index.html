藍狐1-2慘遭逆轉！32歲老將單刀破門，本賽季18輪英超17球居於第1

2019-12-22 3,838條評論 4,762次閱讀 1171人點贊

北京時間12月23日凌晨，英超聯賽第18輪，曼城主場坐鎮阿提哈德球場迎接萊斯特城的客場挑戰！在第21分鐘，32歲老將瓦爾迪首開紀錄率先攻破曼城的大門，幫助萊斯特城客場1-0取得領先！在後場左側的巴恩斯往前給直塞，到了前場左側的32歲瓦爾迪的腳下，隨後瓦爾迪甩開曼城的防守球員，直接殺入到了曼城禁區內左側肋部，直接右腳搓射將球射入了球門右側，曼城門將埃德森已經封死了角度，但是瓦爾迪任然一擊致命。瓦爾迪18球依舊處於英超射手榜第1位。

第29分鐘後，馬赫雷斯取得進球幫助曼城將比分扳平了！在前場右側的馬赫雷斯得球後，帶球往前突破，內切到了萊斯特城禁區內右側肋部，隨後皮球折射到萊斯特城防守球員的腿上彈進球門右側底線，萊斯特城門將舒梅切爾措手不及。

第40分鐘，在萊斯特城禁區內左側底線，斯特林突破之後，佩雷拉要斷斯特林腳下皮球，但是未能成功，隨後斯特林倒地在萊斯特城禁區內，裁判聽了VAR之後示意犯規給了點球，由京多安操刀主罰點球，京多安助跑迅速大力射門，直接把球射進球門右側，萊斯特城門將舒梅切爾碰了一下，但是沒有用，曼城主場2-1逆轉萊斯特城！
