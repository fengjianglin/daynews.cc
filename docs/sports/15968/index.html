出場時間倍增，離開山西男籃獲重用，CUBA第一控在CBA站穩腳跟

2019-12-14 1,226條評論 3,442次閱讀 1151人點贊

一提到CBA選秀球員，很多人第一時間想到的會是狀元王少傑、陳盈駿、姜宇星以及郭凱，其實除了這四位狀元秀之外，還有很多知名度並不高的選秀球員仍然在CBA聯賽中延續著自己的籃球夢想，目前效力於江蘇男籃的王洪就是其中之一。

2016年夏天，他曾被山西男籃在首輪第11順位所選中，之後他在山西男籃效力長達三個賽季，在這三個賽季里，他一共代表山西男籃打了79場比賽，場均出場時間不到10分鐘，個人的數據也是在3+1左右。

在2018年夏天，山西男籃放棄了續約王洪。

在幾乎走投無路的情況下，他選擇了前往江蘇肯帝亞男籃試訓，憑藉熱身賽以及夏季聯賽的發揮，他成功獲得了一份合同。

而在本賽季至今，他保持全勤，場均出場時間達到了18.5分鐘，個人數據提升到了場均5.2分、1.6次助攻、1.5個籃板以及1.4次搶斷，兩分球命中率高達63%。

和上賽季場均7.9分鐘相比，他的出場時間直接翻倍。

從這也足以看出，王洪目前已經躋身了江蘇男籃的輪換陣容，他也可以稱得上在CBA站穩了腳跟。

在部分比賽中，他更是展現出了極為出色的競技狀態，比如說對陣山東男籃的比賽中，他砍下15分、7個籃板、1次搶斷、1次封蓋，對陣天津男籃的比賽中，他貢獻了8分、5次搶斷。

這些場次中，王洪的飄逸和靈動也是依稀讓人看到了他在大學時期打球的風采。

作為前CUBA第一控衛，王洪的整體實力絕對是毋庸置疑，在太原理工大學效力期間，他曾經多次帶隊殺入過全國四強，在2016年夏天他還曾帶隊殺入過決賽，只不過在和清華大學的對決中遺憾失利，那場決賽中，王洪砍下了21分、10個籃板、3次助攻、2次搶斷的出色數據。

在2015年夏天，他還曾入圍過世界大學生運動會中國男籃的12人名單，當時他是球隊的隊長。

作為一位身高1.82米的後衛，王洪的優勢是組織能力不錯，突破中的速度比較快，但是他的缺點也相對明顯，他的外線投籃並不是很穩定，這也限制了他的發揮。

對於27歲的王洪而言，能夠在本賽季在CBA站穩腳跟也算是不錯的結果。
