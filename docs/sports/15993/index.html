最難理解的CBA總冠軍競爭者：郭艾倫與蘭斯之痛何時休？

2019-12-14 532條評論 1,592次閱讀 512人點贊

在體育比賽中，勝利往往可以沖洗掉一切不和諧的聲音和事情！

然而當遼寧男籃力克近況不佳的上海男籃後，關於郭艾倫與蘭斯史蒂芬森不和諧的事情依然重複發生，在已度過應有的磨合期後，一向內部團結得像一塊鐵板的遼籃也已變味，而這種情緒不僅讓主帥郭士強無可奈何，而且還直接影響到球隊的戰績。

擁有為亞洲第一控衛頭銜的郭艾倫，擁有頂級內線韓德君，外加趙繼偉本賽季的成長，遼籃國內球員配置確實十分豪華，也讓人相信這是支總冠軍最強爭奪者！

雖然上賽季因大韓受傷而痛失衛冕，但新賽季更換掉形同雞肋的哈德森，迎來更年輕，能力更強蘭斯史蒂芬森後，單論帳面實力，遼籃完全不遜於廣東宏遠、新疆飛虎，然而殘酷的現實卻告訴球迷，如果再不解決郭艾倫與蘭斯之間的傷痛，爭冠的希望會如流水一樣慢慢消失。

公正的講，雖然是名新外援，但蘭斯史蒂芬森擁有統治CBA，左右比賽的能力，在獲得足夠的信任和機會的情況下，史蒂芬森視砍分如兒戲一般，這個在NBA以刺頭著稱的球星，顯然是一個寧當雞頭也不願意當鳳尾的人物，他需要樹立自己在球隊的地位，那種球隊絕對核心，擁有對球絕對支配權的地位。

然而蘭斯史蒂芬森卻忘記自己球隊的老大早有人選，郭艾倫才是遼籃最核心的靈魂人物。

不管多大牌的外援，哪怕是哈德森也無法擠掉郭艾倫的戰術地位，所以當刺頭史蒂芬森碰到郭艾倫時，他要做的一件事情就是順從，丟掉持球引領球隊的想法，他唯一需要做的是做好防守，跑好位置，等待接球，並將該死的球扔進籃框里，除此之外無他。

然而蘭斯史蒂芬森並非哈德森，他們打法安全不同！

在脾氣上，蘭斯史蒂芬森更不會像哈德森學會隱，所以一向以小孩子脾氣著稱的蘭斯史蒂芬森出現消極也在意料之中，只不過他不像哈德森那樣適當而止，而是很直接表現出來。

郭艾倫與外援不合，在與哈德森合作時也有過，在球只有一個的情況下，極度需要球權的郭艾倫註定要與能力強的小外援不和，這已是不爭的現實，關鍵是在這種情況下，身為主帥的郭士強也束手無策，這才是球迷最不理解的地方。

作為旁觀者，咆哮君認為解決這個難題有上中下三策可供選擇：

1、上策：公開強調郭艾倫為球隊核心，裁掉蘭斯史蒂芬森

若蘭斯史蒂芬森不服則換掉，新援以哈德森為模板，要求新外援性格收斂，低調含蓄，一定要選那種願意接受郭艾倫在球隊的核心地位的外援。

2、中策：主教練豐富打法，尋找能共融的方法

所謂方法總比困難多，作為一名成名已久的主帥，郭士強應該發揮名帥的價值，豐富現有打法，儘可能讓郭艾倫與蘭斯史蒂芬森共存。

3、換掉郭士強，另選能駕馭球隊的鐵腕主帥

刺頭就得有鐵帥來治理，如果郭士強無法平衡遼籃兩位大將的爭執，那麼可以通過換帥，找一位新的外教來解決這類問題。

綜上所述，咆哮君認為史蒂芬森離下崗已不遠，畢竟遼籃不可能為一個外援而去動搖根基，因為拿掉郭士強會影響郭艾倫積極性，而不給郭艾倫核心地位，會讓球迷不滿意。
