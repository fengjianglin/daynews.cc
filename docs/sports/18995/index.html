外媒預測尤伯杯陣容：陳雨菲領銜國羽兩世界第1坐鎮 日韓成勁敵

2019-12-18 364條評論 2,342次閱讀 552人點贊

北京時間12月17日，儘管2019年的羽毛球賽季剛剛落下帷幕，但由於2020年是東京奧運年，因而羽毛球備戰也是在繼續緊鑼密鼓進行。如今外媒針對明年重頭戲之一的尤伯杯陣容進行預測，國羽由兩項世界第1女單陳雨菲與女雙陳清晨/賈一凡領銜出征，但日本隊與韓國隊會成為最大勁敵，國羽能否時隔4年重奪尤伯杯令人期待。

尤伯杯作為世界女子羽球團體錦標賽，是世界上最高水平的女子羽毛球團體賽。國羽在過往歷史當中曾經14次奪冠，成為絕對的尤伯杯霸主，而日本隊則是以6次奪冠位列第2位外加印尼隊與美國隊各奪冠3次，還有韓國隊則是1次奪冠。

只是在上一屆尤伯杯，國羽並未能夠實現4連冠，她們在半決賽爆冷以總比分2-3輸給泰國隊，遺憾止步四強出局，而日本隊則是總比分3-0輕鬆橫掃泰國隊奪冠。

至於2020年的尤伯杯將在5月17日-24日，在丹麥奧胡斯舉行，也將是東京奧運會之前的最重要的一次團體練兵。如今外媒針對中日韓3國的陣容給出預測，無疑日韓兩國會成為國羽的最重要對手。

根據外媒預測的陣容，國羽的陣容為：一單陳雨菲，二單何冰嬌，三單韓悅，一雙陳清晨/賈一凡，二雙李茵暉/杜玥。

至於日本隊的陣容為：一單奧原希望，二單山口茜，三單高橋沙也加，一雙福島由紀/廣田彩花，二雙松本麻佑/永原和可那。

韓國隊的陣容則是：一單安洗瑩，二單成池鉉，三單金佳恩，一雙金昭映/孔熙容，二雙李紹熙/申昇瓚。

目前國羽在女單與女雙均是霸佔世界第1的寶座，其中陳雨菲藉助首次奪得總決賽女單冠軍，她超越戴資穎生涯首次登頂世界第1，也是幫助國羽女單時隔4年6個月重新登頂NO.1。陳雨菲賽季7進決賽全奪冠，因而陳雨菲也是將成為國羽女單絕對領軍人物。

至於日韓一單奧原希望與安洗瑩也是實力強勁，這其中，奧原希望曾經登頂過世界第1，也獲得過世錦賽冠軍。安洗瑩則是驚艷世界的17歲天才，今年成為首位奪得超級750賽冠軍的00後球員。

國羽2單何冰嬌與3單韓悅發揮不穩定，其中何冰嬌哪怕位列世界第7名，但她起伏太大，而韓悅很可能會被00後新星王祉怡取代。韓國的二單與三單也實力一般，而日本二單山口茜實力強勁，因而國羽單打想要完勝日本並不容易。

除此之外，女雙兩對組合的爭奪當中，世界第1的凡晨組合在總決賽奪冠，她們賽季豪取6冠證明自己的統治力，並且成為現役首對積分破10萬的女雙組合，只是國羽二雙李杜組合需要繼續進步。反觀日本的油菜花組合與高妹組合實力強勁，韓國的金融組合與李申組合也是實力不容小覷，國羽想要時隔4年重奪尤伯杯並不會那麼容易。

（文/醉卧浮生）
