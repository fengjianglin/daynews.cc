泰達關鍵一環有多人競爭但卻無法令人安心，新賽季球隊應考慮補強

天天要聞
2019-12-23

距離冬季轉會窗口開啟還有不到兩周時間，相信中超各隊在等待足協新政的同時也開始物色一些適合自己球隊的新援，只待1月1日轉會市場開放後，按照主教練對下賽季球隊的建設規劃對隊伍進行人員架構上的調整，這其中泰達隊也不例外，雖然球隊本賽季相較於前幾年成績上有所提高，但很多關鍵位置還需要進行適當的補充。

如果按照網傳的足協新政版本，中超聯賽恢復亞外外援變成3+1，那麼泰達隊在外援遴選上也將面臨抉擇，抉擇的關鍵就在於巴斯蒂安斯是否還有留隊的必要，如果留隊，那麼德國中衛也只能作為球隊的第五外援，這樣一來泰達隊就要在亞洲範圍內選擇一名中場球員。而且一旦買提江離隊，那麼球隊在中場位置就一定要補充新援，外援最好，來個即插即用的內援也尚可，最理想的人員結構就是前場阿森納組合+外援中場+亞外中衛的五外援配置。

另外在U23球員需求以及邊後衛的位置上，泰達隊也需要進行人員引進。U23球員方面，由於楊帆下賽季不再是U23適齡球員，那麼一線隊中只剩下謝維軍、蘭菁軒、毛昊禹和王政豪符合U23新政，當然還有早已跟隊的原恆大小將陳國抗，這些球員能否承擔起U23球員首發和出場重任還有待考量。邊後衛位置上，泰達隊白岳峰和趙宏略是當仁不讓的主力，多面手榮昊可臨時客串，替補席上有老將譚望嵩，這些球員都已經年過30，為他們尋找合適的替補也是自然而然的事情。除了上述位置需要引進新援外，泰達隊還有一個位置看似穩妥，但是卻始終無法令人安心，那就是門將。

本賽季初，泰達隊門將位置報了杜佳、楊啟鵬和騰尚坤三名球員，賽季後半段補報了丁博威。綜合一個賽季的表現來看，出場24次的杜佳是施蒂利克心中的絕對主力，騰尚坤出場6次，楊啟鵬4次。原本杜佳主力，楊啟鵬替補，騰尚坤候補的次序毋庸置疑，但楊啟鵬卻在今年的聯賽中發揮欠佳，當然他也確實不夠走運。杜佳傷停後，楊啟鵬剛打了一場比賽就在與河南隊的比賽中受傷，缺戰了近一個月的時間，半個賽季後，杜佳在與一方的比賽賽前意外受傷，楊啟鵬又一次臨危受命，但表現極其糟糕，與津門球迷心中的那個楊啟鵬相差甚遠。所以原先門將位置杜佳與楊啟鵬的雙保險現在來看也只有杜佳能夠令人放心，因此泰達隊也應該考慮是否應該在門將位置上引進一些像池文一這樣有實力的球員作為補充，畢竟門將是球隊相當關鍵的位置，而無論是杜佳、楊啟鵬還是騰尚坤他們都缺乏足夠的霸氣和穩定性。
