“昕雯”聯播，持續熱播，再度合體創佳績

2019-12-10 121條評論 1,780次閱讀 783人點贊

如果說許昕是左膀，那劉詩雯就是右臂，左膀右臂強強合體。自劉國梁回歸國乒之後，為了確保奧運會中首枚混雙金牌，打造了一對「老將」組合，於是許昕/劉詩雯組合開始登陸舞台，展現屬於他們的風采。作為兩名將近30歲的老將，經驗豐富，技術全面，閱賽無數，兩個人的組合也讓我們看到了什麼叫默契十足，此呼彼應。兩個人真是你成就了我，我成就了你，可稱得上是劉國梁主席的「金牌組合」。

「昕雯組合」自去年11月奧地利公開賽配對以來就少有敗績，所以他們也成為了東京奧運會混雙重點培養對象，這個組合的誕生也被給予了很大的希望，他們的責任就是衝擊奧運混雙金牌，拿到屬於他們的榮譽，綻放屬於他們的光芒。「昕雯組合」在2019年也有不俗的成績，在匈牙利公開賽奪得冠軍，卡達公開賽冠軍，瑞典公開賽冠軍，世乒賽混雙冠軍，亞錦賽混雙冠軍，可以說是收穫滿滿，「昕雯組合」也在國際賽事的比賽中站住了腳，穩固了地位，在年終總決賽許昕/劉詩雯也獲得了參賽資格，年綜總決賽對於他們來說還有一個重要任務，也是對他們的一個考驗，就是打進四強，拿到奧運資格，再度合體的「昕雯組合」將會給我們播報什麼樣的精彩呢，我們拭目以待！

對於兩名老將來說壓力還是非常大的，首先是體力上，在韓國公開賽，兩人身兼多項，在決賽中就以1-3不敵香港組合黃鎮廷/杜凱琹，首度嘗到失利的滋味，多次的勝利能給他們帶來自信，給予他們力量，但也會讓他們大意，雖然同為老將，但畢竟是一對新的組合，還是要做好睏難準備。對於明年的奧運會兩個人很可能還會身兼多項，所以如何去調整，如何去規劃，也是需要考慮的問題。其次還有精力上，對於劉詩雯來講在奪得世乒賽女單冠軍後已經鎖定了奧運單打名額，那該如何去分配精力，是側重於單打還是混雙，還是兩者兼顧，但作為老將有時候會力不從心，不像年輕人那樣精力充沛，所以對於他們，這也是非常嚴峻的考驗，也希望兩位能咬牙堅持。還有對手，隨著混雙加入奧運項目，各個國家協會都非常重視，在瑞典公開賽時日本組合水谷隼/伊藤美誠就一路打進決賽，在決賽中與許昕/劉詩雯上演強強對話，雖然許昕/劉詩雯以3-2拿下了比賽，但贏得並不輕鬆，雖然我們足夠強大，但也不能忽視對手的實力，一定要做到知己知彼，方能百戰不殆。

年綜總決賽將於12月12-15日在鄭州舉行，許昕/劉詩雯是唯一一對中國混雙參賽選手，並且關乎到奧運資格，所以至關重要，許昕這次繼續身兼三項，希望他們能經得住組織的考驗，在奉獻精彩的同時拿到屬於他們的勝利，期待「昕雯聯播」，繼續熱播，我們共同關注！

本次參賽的八對組合：黃鎮廷/杜凱琹、林昀儒/鄭怡靜、許昕/劉詩雯、水谷隼/伊藤美誠、趙大成/申裕斌、皮斯特耶/巴拉佐娃、弗洛雷/加斯尼爾、斯祖迪/佩格爾。關注比賽，一起為中國選手加油，關注比賽，我們一起聊賽事！
