女排不老女神曬復古照，因誤診重傷也不埋怨教練，今38歲依然單身

2019-12-20 2,759條評論 4,979次閱讀 477人點贊

北京時間12月18日晚上19:45，前中國女排著名國手、排球不老女神趙蕊蕊在個人社交媒體上發布了新動態，動態中趙蕊蕊曬出了自己最新的自拍復古美照，同時她還給照片配文寫道：「有一種態度叫堅韌，心若寬，每天都可以有小確幸！」

從照片中看來，人美腿長的趙蕊蕊素顏出鏡，臉上帶著健康、自信的迷人微笑。她身著一件老式的黑色長款風衣，白T恤，黑褲子，黑球鞋，留著齊劉海的少女髮型，腳蹬一輛老款的二八式自行車。歲月似乎沒在趙蕊蕊臉上停留過，已經年近四旬的不老女神趙蕊蕊渾身上下洋溢著一股青春氣息，她還是當年那個一笑傾城的玻璃美人。趙蕊蕊的自行車前後還搭載著幾盤美麗的鮮花，看起來她應該是在拍攝一組復古的寫真照片。

看了趙蕊蕊的復古寫真，她的前隊友，雅典奧運會冠軍劉亞男也誇讚道：「美翻了！」趙蕊蕊的姐姐趙蓉蓉也點贊道：「復古風格，28寸的車子挺帶感的！」其他網友也是紛紛留言點贊，有網友表示：「很文青的感覺，蕊蕊你以前真的是排球副攻嗎」；「有一種不老女神叫蕊蕊姐」；「好復古又有年代感的車子啊」；「一直沒有變化，真的是太美了」；「蕊蕊是天天健身么？凍齡了」；「少女蕊和28大杠最配哦」！

趙蕊蕊，1981年10月出生於江蘇南京，身高1米97的她從小就展現出來過人的排球天賦，被稱為”中國女排第一高”，她是當年世界上的頂級副攻、中國女排「黃金一代」的主力球員、雅典奧運會冠軍得主。趙蕊蕊18歲就進入國家隊，國家隊幾任主教練，無論是胡進還是陳忠和都看好她，認為她是可造之材，給了她很多上場比賽的機會。

在她10餘年的國家隊生涯中，幫助中國女排先後奪得亞錦賽、亞運會金牌，女排大獎賽、世界盃的冠軍以及奧運會金牌等多項榮譽。趙蕊蕊堪稱人美球也狠，每每在比賽糾纏的關鍵時候，她都能利用犀利的進攻為球隊打開局面。但她亦有另一個綽號—玻璃美人，這說明她非常容易受傷，也正是傷病大大影響了她的職業生涯。

趙蕊蕊訓練時很能吃苦，根本不像她外表那樣嬌弱，屬於那種自覺給自己瘋狂加練的人。但在備戰雅典奧運會期間，有一次她竟然在訓練場上哭著跟陳導說：「我的腿真的疼得不行了」。但是當時的隊醫姜偉在不了解情況下，就武斷地說她嬌氣，認為她的腿沒問題，可以堅持。

此後趙蕊蕊去醫院拍片檢查出來的結果是疑似骨裂，可是當趙蕊蕊拿著診斷結果和片子交給隊醫姜偉看後，姜偉堅持認定趙蕊蕊只是得了骨膜炎而已。而由於當時正處在奧運備戰的緊張氣氛中，主教練陳忠和選擇了聽從隊醫的意見，認為趙蕊蕊應該沒有大礙，沒有同意她的休息請求，這一錯誤的決定致使她在後面的訓練中造成了骨折重傷的慘劇。

此後雅典奧運會上首場與美國女排的比賽，經歷了140天的傷病恢復，終於再度登場的趙蕊蕊，比賽第一局的一次進攻，卻再一次讓她的右腿骨折，「玻璃美人」被痛苦地抬下了場，這讓她不得不提前結束了自己的奧運征程，也嚴重影響了她的職業生涯！2009年，傷病纏身的趙蕊蕊再也堅持不下去了，年僅28歲就不得不宣布退役，離開了她心愛的排球場！

正是因為誤診才造成趙蕊蕊的重傷，陳忠和為此也是非常自責，不過趙蕊蕊卻沒有因此而埋怨教練。北京時間12月10日，趙蕊蕊在社交媒體上曬出了和前主教練陳忠和最近一次的合影，並發文回憶了當年陳忠和對她的關愛，同時她也表示自己的傷病跟陳導有一定的關係，卻存在隊醫誤導的主因，但陳導從未推卸過責任，這也使自己從未怨恨過他，希望所有人都善待這樣一位善良的好人。

不得不說趙蕊蕊真的是一位善良的好女孩，因誤診重傷也不埋怨教練，退役後，她成為一名美女作家，已經出版了好幾本著作。趙蕊蕊在賽場上和退役後的事業上都取得了成功，不過她的感情路走得卻並不順利，如今38歲的她依然單身，不過好在性格開朗的趙蕊蕊對此並不在意，正如她自己所說「心若寬，每天都可以有小確幸！」
