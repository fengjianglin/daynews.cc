兩巨星來襲！一人引發兩筆交易，熱刺下血本了

2019-12-10 1,547條評論 2,751次閱讀 854人點贊

如何讓一隻曾經進入歐冠決賽的球隊復甦，在熱刺老闆列維的眼裡顯然是讓穆里尼奧來處理。給他足夠的資金讓他引援，構建新的球隊體系。

在英格蘭媒體一致認為，列維不會給穆里尼奧引援金額的時候。熱刺已經開始準備這個冬窗的引援人選名單了。在熱刺眼中除了意甲的庫利巴利之外，還有一位球員是熱刺特別想要的。

他就是現在的皇馬前腰羅德里格斯，這就是穆里尼奧想要的球員。在齊達內的治下j羅並沒有多少可以活動的空間，他的位置也是在經常在陣型的切換中失去。

據西班牙媒體報道j羅將會在冬窗期離隊，而且最有可能有錢去的，除了尤文等球隊之外，另外一個就是穆里尼奧教的熱刺。在鳥叔心中，j羅也是可以提升目前目前熱刺中場的關鍵人選。

畢竟在球隊的中場核心埃里克森即將離隊的情況下，拿下j羅是非常有必要的。在穆里尼奧反擊流的打法中，j羅的出球以及策應也是能夠起到促進球隊攻守轉換以及保證球隊進攻質量的作用。

皇馬和熱刺對於這筆交易，也已經有了眉目。這個冬窗期熱刺會對j羅進行追逐。列維也是為穆里尼奧提供了足夠的資金支持。

在這個冬天熱刺恐怕會成為英超引援力度最大的球隊，要知道庫利巴利和j羅並不是說一般數目就可以拿下的，熱刺想一次性搞定也需要大量的資金。
