蘭帕德率切爾西出線創多項紀錄 沒有阿扎爾，藍軍進攻火力仍強大

2019-12-11 471條評論 3,144次閱讀 2226人點贊

北京時間12月11日，切爾西在主場2-1拿下里爾，成功晉級歐冠淘汰賽。在自己職業生涯的第二個賽季，蘭帕德成功帶領球隊小組出線，並且他還創造了多項紀錄。

蘭帕德成為2010-11的老雷德克納普後，第一位打完六場小組賽並且帶隊殺進歐冠淘汰賽的英格蘭教練 ，同時，他也是第一位以球員和教練身份都打入歐冠淘汰賽的英國人。要知道，在今年夏天，蘭帕德和他的切爾西經歷了不少困難。

今夏的轉會禁令和阿扎爾的離隊，這讓切爾西的實力大損，蘭帕德回到斯坦福橋擔任起了主帥，半個賽季過去了，蘭帕德帶隊的戰績超出了很多人的預期，目前，切爾西在競爭激烈的英超比賽中排在聯賽第四的位置，如今球隊再次拿到了歐冠出線權，蘭帕德功不可沒。

實際上，在歐冠小組賽中，切爾西和阿賈克斯、瓦倫西亞、里爾在同一小組，雖然不能比巴薩、多特以及國米這個小組更加恐怖，但是實際上，切爾西的這個小組卻更加兇險，這四支球隊本身差距不大，誰出線都不能算是冷門。在小組賽首輪，切爾西甚至在主場被陷入內訌的球隊瓦倫西亞擊敗，這讓他們的出線前景蒙上了陰影。不過，隨後，球隊連續擊敗了里爾和阿賈克斯，並且在主場連綴3球上演了4-4逼平阿賈克斯的好戲，球隊似乎能夠看到出線的希望。

更重要的是，球迷們在這場比賽中看到了切爾西在蘭帕德的帶領所展現出來的硬氣，藍軍此前客勝里爾，此番必須主場取勝才能確保晉級。最終，球隊憑藉著亞布拉罕和阿斯皮利奎塔先後進球，切爾西奪得小組第2出線。
