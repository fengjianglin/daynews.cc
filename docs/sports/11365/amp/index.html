比足協杯奪冠更可貴的是什麼？申花未來三年再奪冠的信心從哪來？

天天要聞
2019-12-09

「我會繼續努力，和中國足球繼續努力！」當臉色漲得有些發紅的朱辰傑用略顯青澀的聲音說出這句話的時候，很難把剛剛獲得「中超最佳新人」稱號的他，跟在中超和國家隊賽場上橫刀立馬的中後衛聯繫在一起。作為申花年輕一代球員的代表，已經在中超賽場上站穩了腳跟的朱辰傑，跟同樣進入申花一線隊並且獲得了出場機會的蔣聖龍、劉若釩和周俊辰這些「99後」「00後」隊友一道，成為這支上海球隊絕對值得期待的未來。

在2019賽季中超頒獎典禮上，上海綠地申花俱樂部榮膺青訓優秀俱樂部獎，這也是對他們在剛剛過去的這個賽季青訓工作最大的肯定。2019賽季，申花兩支青訓梯隊分別奪得青超聯賽U19B組和足協杯男子U19組兩項冠軍。

此外，在中超預備隊聯賽中，基本以年輕隊員為主出戰的申花預備隊不畏強敵，穩紮穩打，最終獲得了亞軍，成為這個賽季預備隊聯賽中最大的一匹「黑馬」，在預備隊比賽中得到了鍛煉的周俊辰、徐皓陽和汪海健等年輕隊員，下賽季也將在一線隊獲得更多的出場機會，成為球隊的「新鮮血液」。

值得注意的是，儘管申花本賽季聯賽中的成績並不算好，而且一度陷入保級泥潭，但是對於年輕隊員的使用和鍛煉力度，依然是整個中超中最大的。整個2019賽季，申花累計使用U23球員的上場時間達到了6463分鐘，是中超球隊當中最多的，即便在劉若釩參加國奧隊集訓時受傷、可以不使用U23球員的情況下，朱辰傑、蔣聖龍、陳釗和叢震這些年輕球員，依然獲得了穩定的出場時間，由此也不難看出，申花致力於年輕化的思路，絕不僅僅停留在「說一說」的層面上。

從2018賽季中期登上中超這個舞台，到2019年底捧起足協杯冠軍獎盃，朱辰傑坦言，這樣的經歷，對於他這個年齡的球員來講，是一筆極為寶貴的財富。「尤其是這個賽季，這是我參加的第一個完整的中超賽季，其間經歷過低谷，包括球隊的，還有我個人的，也得到了教練和球隊當中那些老大哥的幫助，在國家隊也同樣如此，他們的經驗和一些切身體會，幫助我少走了很多彎路，也幫助我找到了努力的方向，所有這些，都對我的成長有很大的幫助。」

事實上，與朱辰傑一同受益於申花俱樂部未來發展計劃的，還有各個年齡段梯隊的所有年輕球員，而來自西班牙和克羅埃西亞等國家的外籍教練的加盟，也讓這些未來的「申城之花」有了「百花齊放」的可能，或許，這樣能夠看得見並且值得期待的未來，正是綠地申花俱樂部董事長敢於說出「未來三年，申花隊還要拿冠軍」的信心和底氣所在。
