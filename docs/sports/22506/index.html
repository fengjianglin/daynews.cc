國足又刷新底線，網友吐槽：論收入，韓國日本與國足確實差距不大

2019-12-20 596條評論 1,961次閱讀 525人點贊

又鬧出大笑話的國足最近再度刷新底線，真是沒有最低，只有更低啊。。。

這次是于大寶。國足在東亞杯鎩羽而歸後，這位隊長居然面對記者大言不慚地表示：與日本韓國這樣的球隊，是有一定的差距，但差距不是非常大。

此話一出，立刻成了全網路的笑柄。球迷紛紛吐槽，多大算大？還想有多大？每場輸5個，被人家踢得滿地找牙嗎？

還有球迷調侃道，差距確實不大，都有腳，就是沒腦子。論收入，韓國與國足差距確實不大~

于大寶大概是覺得，跟日本和韓國都只輸了1個球，所以自我感覺”差強人意”吧。可他似乎忘了，兩場比賽場面之難堪，數據之尷尬，可遠遠不止一個球的差距。

于大寶潛台詞很明顯，中國是二隊，輸成這樣不丟人。可他不知道是睜眼瞎還是有意無視一個事實：日本是平均年齡比中國小5歲、連3隊都談不上的青年軍。被一群娃娃踢得狼狽不堪，差距還不大？

跟韓國就更不說了，要不是韓國臨門一腳太臭，恐怕輸3個都算少的。況且，如果日本和韓國把全部旅歐球員組成一支國家隊，只有武磊一個旅歐球員、還快在歐洲混不下去的中國得輸幾個，于大寶想過嗎？

有時候看看國足教練和隊員的自評，經常會恍惚：他們說的跟球迷們看的是同一場比賽么？

國乒，哪怕贏再多，外界再狂贊，教練隊員也要找各種不足，就算真沒有也要硬找；

國足，哪怕輸再慘，外界再吐槽，教練隊員也要找各種優點，就算真沒有也要硬找。

中國乒乓誰也打不過，中國足球誰也打不過 ​​​，這難道是偶然的嗎？

還是網友說得好啊，于大寶能說出這種話，要麼是傻到家，要麼是不要臉，要麼是自我要求實在太低，3者必居其一。

國足，啥時候能有點自知之明，啥時候能知道點羞恥，啥時候能不再牛逼哄哄，本本分分做人、踏踏實實練球呢？
