15分不敵北控，廣東隊新賽季第二場失利，易建聯11中4拿到9分

2019-12-16 3,649條評論 4,718次閱讀 1631人點贊

北京時間12月15日訊，CBA常規賽繼續進行，廣東隊來到客場挑戰北控隊，這場比賽之前，廣東隊戰績不錯，球隊取得15勝1負，排在聯盟第1位，而北控男籃戰績為9勝7負，排在聯盟第10位，掉出前8位，外援德隆將不會出現在比賽當中。

這場比賽首節，馬尚助攻胡明軒上籃命中，福格命中三分，趙睿上籃也有，福格造成犯規，兩罰全中，馬尚出現失誤，王少傑內線上籃命中，趙睿三分沒有，福格跳投命中，北控男籃取得9-4的開局，胡明軒命中外線三分，易建聯搶斷完成上籃，廣東隊追平比分，隨後，北控隊打出一波11-4的進攻，再次拉大比分，節末，杜潤旺命中三分，首節結束，廣東隊落後6分。

​

比賽第二節，兩支球隊在進攻端都有不錯的發揮，北控隊憑藉著整體的表現，把比分差距拉大至12分，隨後，威姆斯和馬尚開始發力，兩位球員拿到16分，幫助球隊縮小分差，節末，孫悅命中外線三分，半場結束，北控隊領先廣東隊7分。

比賽第三節，廣東隊投籃命中率出現了下滑，連續錯失投籃機會，另外還出現了多個失誤，單節球隊只拿到21分，而北控隊憑藉著三位外援的表現，單節凈勝廣東隊11分，比分差距來到18分。

比賽最後一節，北控隊始終佔據場上主動，廣東隊沒有做出有效的調整，比分差距維持在15分左右，在還剩下3分鐘的時候，比賽也失去懸念，最終廣東隊在客場以111-126不敵北控隊，輸掉了這場比賽。

數據方面：福格38分6籃板8助攻，薩卡基尼18分11籃板，湯普森19分13籃板，王子瑞16分，而廣東隊這邊，易建聯11中4拿到9分，威姆斯28分8籃板8助攻，馬尚·布魯克斯16分，胡明軒18分。

這場比賽廣東隊在攻防兩端的表現都不是很好，讓北控隊拿到120多分，在限制福格方面沒有很好的解決辦法，在進攻端，除了威姆斯之外，馬尚·布魯克斯和易建聯狀態不佳，兩位球員加起來26投10中，只拿到25分，這也是他們輸掉比賽的主要原因。而北控隊這邊，福格再次打出了MVP級別的表現，全場24投13中拿到38分6籃板8助攻3搶斷，在攻防兩端都幫助到了球隊，全隊4人得分上雙，9位球員拿到得分，靠著整體發揮，送給了廣東隊本賽季第二場失利。
