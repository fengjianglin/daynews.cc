籃筐都歪了！雙槍啞火43中15，底薪合同干頂薪活，瓜哥帶不動啊

2019-12-25 1,747條評論 4,104次閱讀 345人點贊

此前已經取得3連勝的開拓者在主場迎來了鵜鶘的挑戰，比賽開始，雙方在前三節打的有來有回，但是到了下半場卻是風雲突變，開拓者進攻突然斷電，手感最好的安東尼在第四節沒得到很多出手機會，開拓者在整個下半場都被鵜鶘壓制著打。

開拓者最終以94-102不敵鵜鶘，終結了自己的連勝，不過唯一值得慶幸的是，開拓者仍然處在西部第8的位置，不過開拓者可不能再輸了，因為馬刺贏球了，如果再輸個一場兩場，自己在西部的局勢又變得糟糕了。

在這場比賽中，開拓者的後場雙槍利拉德和麥科勒姆雙雙啞火，兩人合計出手43次，只命中15球，其中三分球更是20投只進了2個！自己主場籃筐都被砸歪了，而雙槍的失准決定了這場比賽的走向，雖然鵜鶘是魚腩隊，但是雙槍對於開拓者來說有多重要不言而喻。

有幾場是雙槍沒發揮而球隊贏球了的，幾乎沒有，而近來狀態逐漸回暖的麥科勒姆也是再次陷入掙扎，賽季初他表現一度非常的糟糕，但是隨著賽季的深入，他還是找回了自己的手感，不過這場比賽又是回到了賽季初。

開拓者後場雙槍的一個很大的問題就是他們的狀態並不穩定，利拉德經常是一場超神一場超鬼，麥科勒姆相對來說還是穩一點，不過這賽季麥科勒姆手感也是非常的掙扎，這對於開拓者來說是一個非常不好的消息。

如今球隊糟糕的戰績更是雪上加霜，開拓者幾乎每場都要贏下來，如果後場雙槍的發揮還時不時不穩定的話，這對於開拓者來說是一個致命的打擊，唯一的好消息就是開拓者擁有了安東尼。

安東尼雖然狀態大不如前，但是還是能夠帶個雙槍足夠的幫助的，就像在與鵜鶘的比賽中，雙槍啞火後，安東尼砍下了全場最高的23分，外加9個籃板，雖然最後沒能贏球，安東尼還是儘力做到他能做的事情了，這賽季的安東尼簽的是底薪合同，而且還是無保障的合同。

但是安東尼已經不止一場得分上20了，而且也經常救隊伍於危急時刻，可以說安東尼是拿著底薪合同卻干著頂薪的活兒，這場比賽看了雙槍的表現，安東尼太難了，帶不動啊！

這賽季的開拓者西部的局勢非常的嚴峻，前有強敵後有追兵，再加上陣中雙槍的狀態起起伏伏，面對的問題非常的多，但這何嘗不是考驗一支球隊的時候呢？

如果開拓者能夠成功的挺過這些難關，那麼我們相信開拓者一定會變得更強，不止是球隊的戰績，還有球員們的心氣和鬥志，這些對於一個漫長的賽季和艱難的季後賽來說，是不可或缺的。

西部強隊如林，排在開拓者前面的雷霆目前也是沒有決定擺爛還是進季後賽，而後面的馬刺則是全力追趕中，開拓者接下來要十分小心了！
