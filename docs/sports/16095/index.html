國安冬窗將喜迎“強援”：回歸時間提前曝光，助主帥衝擊亞冠冠軍

2019-12-13 837條評論 4,365次閱讀 186人點贊

本賽季作為爭冠球隊之一，北京國安最終輸給了廣州恆大，獲得中超第二，無緣冠軍。而賽季結束之後，北京國安有兩件重大事件，就是續約熱內西奧續約1年和確定下賽季亞冠分組情況，現在距離下賽季還有一段時間，而球隊提前迎來了利好消息。

據外媒《tiempo de canarias》報道，拉斯帕爾馬斯主席拉米雷斯在出席新聞發布會的時候，回答了關於比埃拉去留的相關問題。其中拉米雷斯透露球隊繼續續租比埃拉的可能性是0%，解釋繼續續租的話，俱樂部有經濟上的困難，球隊必須保持冷靜，同時還強調12月21日的比賽，是比埃拉在球隊的最後一場比賽，他將在一月初回到北京國安。

賽季中期，​比埃拉一度受到重傷，北京國安決定放棄​比埃拉，引進巴西前國腳費爾南多頂替外援名額，事實證明這是錯誤的決定。因為費爾南多加盟之後，作用不大，而租借到拉斯帕爾馬斯的​比埃拉，在傷愈復出後成為了球隊戰術核心，已經打進10粒球，是球隊的最佳射手，狀態火熱。如果北京國安留下​比埃拉，那麼爭奪中超冠軍希望無疑更大些。

如今在拉斯帕爾馬斯主席拉米雷斯透露比埃拉返回北京國安已成定局，北京國安也將在下賽季開始之前迎回自己的核心外援。下賽季初，北京國安將重新集結，第一個冬訓地點放在西班牙的城市穆爾西亞，屆時候外援們直接在那裡亞與球隊匯合，而比埃拉不出意外也是在那時候歸隊參加冬訓。

前段時間，北京國安官宣續約熱內西奧一年，是希望熱內西奧下賽季帶領球隊衝擊更多的冠軍，其中就包括了亞冠冠軍。據悉，熱內西奧已經定下了一個目標，就是要在2020年贏得亞冠冠軍，從而獲得2021年在中國舉辦的世俱杯的參賽資格。

現在北京國安除了擁有幾名實力派外援外，還有多名本土國腳，當年廣州恆大奪得亞冠冠軍也是如此。而比埃拉無疑是熱內西奧帶隊衝擊亞冠冠軍的重要拼圖，跟奧古斯托的搭檔，可謂是中超最強的中場組合之一，但本土球員方面，北京國安還需要完成補強。
