內馬爾重大退步·！1.6億巨星本年度參與28球，巴西球員里排第7

2019-12-24 1,003條評論 1,985次閱讀 876人點贊

2019年度即將畫上句號，在過去的這一年裡，內馬爾的表現只能說一般，在金球獎的評選中，這位巨星無緣前30，而一個數據是，在2019年度的巴西球員製造進球榜上，內馬爾僅僅排名第7位，他參與制造的進球數只有另外一名巴西巨星巴博薩的一半，而對方是內馬爾的「妹夫」。

在2019年度，巴爾博薩在桑托斯有著夢幻般的發揮，他出場了59次，打入了43個進球，另外他還有著13次助攻，一個人就參與制造了56個進球，是本年度，製造進球最多的巴西球星。有趣的是，巴爾博薩在2019年和內馬爾的妹妹拉斐拉複合，兩人又成為了戀人關係，這意味著，巴爾博薩是內馬爾的妹夫。

巴勒莫的布魯諾-恩里克在2019年度參與制造了50個進球，排名第2位，曼城的熱蘇斯參與制造了44個進球，排名第3位。格雷米奧的埃韋頓參與制造了34個進球，可以排名榜單的第4位。

菲爾米諾在2019年度成為人生贏家，巴西球星贏得了美洲杯、歐冠、世俱杯和歐洲超級盃冠軍，即將過去的這一年裡，菲爾米諾一共製造了33球，可以排名第5位，不過他的冠軍數可是巴西球員里最多的。帕爾梅拉斯的杜杜製造了30球，可以排名榜單的第6位。

內馬爾的身價高達1.6億歐元，是所有巴西球員里最高的，但是在2019年度，巴西球星僅僅製造了28個進球，排名榜單的第7位。作為巴西球員里名氣最大的一員，內馬爾在製造進球榜上卻只能排名第7位，不得不說，這位巴西巨星出現了巨大的滑坡。

2019年，內馬爾經歷了傷病，還錯過了美洲杯，而沒有他的情況下，巴西隊捧起了最後的冠軍，這讓內馬爾也極為遺憾。在巴黎聖日耳曼，內馬爾依然無法帶隊在歐冠奪冠，他依然只能用法甲冠軍來慰藉自己。同時，內馬爾還捲入了轉會傳聞里，巴西人渴望重回諾坎普球場，但是最終沒有如願。內馬爾在感情生活上也非常不順利，在同布魯娜分手後，巴西球星沒有找到新女友，還一度被巴西老鄉起訴強姦。

總之，2019年度對於內馬爾來說是噩夢般的一年，這位巴西巨星在場內外都迷失了自己，不知道他是否還能成為世界前三！
