身處藍軍不得志，兩任教練將他用錯，坎特或以1億身價追隨齊達內

2019-12-17 177條評論 2,031次閱讀 480人點贊

坎特，這名球員很早之前就被傳聞是皇馬的獵物，只是切爾西不想放人，以及球員自己從來沒有表態過，讓這筆潛在的轉會一直被擱置。

但是現在最新的消息是，坎特如今已經決定，在明年夏天的時候轉會離開切爾西，加盟到皇馬，這對於切爾西而言簡直是一個沉重的打擊。

自從當年從萊斯特加盟到切爾西之後，坎特的表現就完全符合他的世界第一防守後腰的名頭。2018年的時候他還跟隨法國隊一起拿到了世界盃，我們現在無法確切知道坎特為何離開切爾西，但是最近2任切爾西教練薩里和蘭帕德似乎都格外倚重若日尼奧，他們要麼將坎特位置前提，要麼直接將其發配在邊路，當然坎特自己是不會說出這其中的原因，他只是強調自己想要尋求新的挑戰。

轉會費方面，坎特的身價有望達到1億，但是尷尬的是皇馬的主席一直對於坎特這樣的超級低調實用派不是很感冒，暫且不說他願意不願意拿出1億來買坎特，單純就其想法而言，他甚至覺得皇馬現在有一個卡塞米羅完全就足夠了，根本不需要引進坎特。

但是齊達內顯然不是這樣認為，這個教練現在看來對於法國球員真的是有著超天然的熱愛，之前他一直吵著要求球隊簽下博格巴，現在則是坎特。在齊達內的心裡，坎特的能力可要比卡塞米羅強不少，而且卡塞米羅其實是現在的皇馬陣中唯一一個正牌防守後腰，一旦他出現受傷，那麼球隊會遭到沉重打擊，與其冒險，還不如簽下一個更好的保證來。

不過還有一點可以肯定的事情是，除了皇馬，也必然有其他球隊希望引進坎特，這個球員實在是太低調太好用了，這樣的球員引起其他豪門關注也是非常正常的事情。據說尤文和拜仁，現在都是坎特的追逐者，但現在看來，坎特的想法更像是想去皇馬。

其實說到這裡也給切爾西提了個醒，這2個賽季他們將坎特的位置胡亂調整，導致球隊的成績也沒有保證，而且從薩里和蘭帕德的執教經驗來看，他們其實並非是很成功，在坎特的使用上真的是讓人感到無語。手裡有著世界最頂尖的防守後腰，這2個教練居然在挖空心思想著怎麼將其打造成為組織甚至是邊路核心，這隻能讓人嘆為觀止了。
