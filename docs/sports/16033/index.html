小魔王也輸了！被超級黑馬爆冷淘汰 林高遠爭了口氣復仇林昀儒

2019-12-14 2,125條評論 4,721次閱讀 2698人點贊

2019國際乒聯年終總決賽，女單1/4決賽中，劉詩雯3-4慘遭陳夢逆轉還屬正常水平；孫穎莎也輸了，她狀態更差1-4慘敗超級黑馬王藝迪，爆出本次年終總決賽的一大冷門！而在男單1/4決賽中，林高遠則爭了口氣4-0霸氣復仇林昀儒！

孫穎莎本賽季表現十分突出，布達佩斯世乒賽上女單和女雙全部淘汰了伊藤美誠，又斬獲3個白金站冠軍、全錦賽單打冠軍等，目前排名飆升至世界第二，是本次總決賽女單頭號種子。如何戰勝孫穎莎，已經成為當今世界乒壇的一大難題，但讓人沒想的是，她搭檔王曼昱2-3不敵長崎美柚/木原美悠，無緣女雙決賽。士氣受挫後竟然在單打中也提前出局！

王藝迪儘管是一員國乒猛將，但是在隊內次序上，她還靠不近前六，和陳幸同、何卓佳等人苦苦爭奪國乒第七大主力位置。不過，這次總決賽上，她先是擊敗平野美宇，直接幫石川佳純獲得日本奧運女單資格，被日本人徹底記住。在擁有主角光環後，大迪乘勝追擊，爆冷擊敗小魔王孫穎莎，成為了此次總決賽女單最大黑馬！

林高遠本賽季狀態高開低走，儘管他在上半年有過高光表現，但輸給林昀儒後狀態大跌。今年世界盃團體賽上，雖然馬龍因傷缺席多場比賽，但是本該順位替補的林高遠卻被梁靖崑搶先，無緣出戰對陣日韓的兩場最激烈的比賽。林高遠狀態出現問題，他需要自我救贖，才能在隊內與梁靖崑的競爭中保持住領先優勢！

這次面對宿敵林昀儒，林高遠打出了男兒的豪邁氣質，全場霸氣怒吼提振士氣，比張本智和還要兇狠！而小小林（林昀儒）恐怕也沒有見識過小林子（林高遠）這種殺紅眼的表現，最終在林高遠步步緊逼情況下直接被打懵，以0-4的成績結束了這場「雙林之爭」。

獲勝後的林高遠坦言道：「賽前其實沒有太多把握，畢竟剛輸過，陳指導和秦指導給了我一些建議，和一些左手的套路和習慣，這些非常好的建議幫助自己在場上更好地執行技戰術。賽前挺緊張，也很謹慎，想得也比較多，把自己技戰術上的缺點、他打我彆扭的地方都想得比較多。比賽中自己做得特別好，沒有太多雜念。」
