巴薩4-1大勝後，引發球迷熱議：梅西就是球王，比達爾是全場MVP

天天要聞
2019-12-23

依靠格列茨曼、梅西、蘇亞雷斯、比達爾的聯袂進球，巴薩在主場4-1大勝阿拉維斯，繼續穩居榜首，並且多賽一輪領先皇馬3分。至此過後，巴薩在2019年自然年完成了主場不敗的壯舉，取得24勝4平，而梅西也完成了連續6年轟下50+進球的神跡。

賽後巴薩官博這樣寫道：「全場比賽結束，蘇亞雷斯3傳1射，巴薩4-1戰勝阿拉維斯！上半場第14分鐘，蘇亞雷斯助攻格子首開記錄；比賽第45分鐘，蘇神再次貢獻一記助攻，比達爾爆射破門。易邊再戰，阿拉維斯扳回一球，比賽懸念再起；下半場第69分鐘，蘇亞雷斯完成助攻帽子戲法，梅西憑藉一記神奇遠射打入他本年度的第50粒進球；第75分鐘，蘇亞雷斯點球破門，鎖定勝局！一場酣暢淋漓的大勝，感謝球員們帶來的進球盛宴，大家來點評一下本場比賽吧！薩迷們喜歡這份聖誕禮物嗎？」

賽後球迷對於巴薩年度收官戰的大勝和梅西創下的神跡，也是直接「圍攻」了俱樂部官博。

—對於有強迫症的人來說，這第50個球進得太特么爽了！ 不敢想像當年91球是怎麼玩出來的……點球判罰的時候基本就知道梅西又要讓點了，沒辦法他就是一個只想著隊友的人，能進球，能助攻，能組織，有隊友，我不知道這樣的特質算不算球王，如果他不是，那這世界就不存在球王！

—一轉眼，2019年巴薩的最後一場也結束了，這一年裡，有梅西奪得六個金球獎的喜悅等，有歐冠失利的失落，不過這些馬上就過去了，我們向前看，在2020年希望巴薩越來越好，也希望我們每個人在新一年裡越來越順利。加油！巴薩!

—別的不說好吧，雖然開局踢得有些隨意，但是這場MVP我給比達爾。前場有他，中場有他，後防還有他。如此拼的比爹讓人怎能不愛……

本場比賽，巴薩在下半場一度被阿拉維斯扳回一球，阿拉維斯士氣大漲，就在這時候，梅西站出來為球隊分憂，直接用一個無解世界波的進球，破滅了阿拉維斯反撲的鬥志和慾望。值得一提的是，梅西破門之後，也是笑得非常開心，因為這一夜又是屬於他的夜晚。

梅西打進了本年度第50球，實現了10年內第9次打入50+進球，在這一紀錄中，梅西還領先C羅和貝利。同時也是連續6年進球數達到50+的神跡，而下賽季梅西將32歲，未來還有更多的紀錄等待他創造！
