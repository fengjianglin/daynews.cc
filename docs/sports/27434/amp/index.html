韓德君最新傷情報告！遼籃官宣，郭士強會使用新人應急嗎？

天天要聞
2019-12-20

遼寧男籃客場對陣山東男籃，比賽的最後一分半鐘節點，遼寧男籃隊中主力內線韓德君在一次進攻回合中倒地受傷，當場被隊友攙扶下場。

北京時間12月20號，遼寧男籃官方官宣了大韓的傷病情況，寫道：”大韓傷情應無大礙。在昨晚遼寧本鋼隊客場對陣山東西王隊的比賽中，球隊隊長韓德君在比賽即將結束之前受傷離場，經與球隊方面聯繫證實，昨晚去當地醫院進行檢查之後得出的結論是：沒有特別大的問題，但還需要進一步的觀察。”

從官宣的內容來看，大韓只能說目前沒有太大情況，未知確切傷情。基於上賽季其曾有賽季報銷的經歷。恐怕這次再受傷，即便是小傷，遼寧男籃也會謹慎行事，大概率會讓大韓將傷徹底養好了再復出。在近來一個階段，遼籃內線將處於真空！

本賽季，大韓場均15.7分10.6籃板，是球隊唯一場均拿到兩雙的球員。更是內線支柱，球隊要找尋人員補足這個缺漏並不簡單。此前，韓德君剛榮獲最新一期的周最佳球員。遼籃名宿楊鳴指導評價道，大韓是目前遼籃最重要的球員，缺誰都不能缺他，倘若傷缺，這支遼籃可謂是實力被廢掉一半。

賽季中期，遼寧男籃也沒有辦法從本土球員群去引援補強，最終只能夠通過內部挖掘。這便會考驗到郭士強的執教能力。上賽季，遼籃在韓德君賽季報銷的情況下，球隊實力大減，季後賽被新疆男籃淘汰，彰顯出韓德君的重要性，同時也可以看出球隊內線位置的薄弱。顯然郭士強指導沒有吸取到教訓，沒有清楚認識球隊內線位置需要備好替補。當衛猛被清洗出隊，鍾誠租借到期回歸吉林，遼籃僅僅是上提王化東這位年輕內線，且後者作為新人本賽季幾乎一點上場機會都沒有，是實打實的全菜鳥。放眼遼籃替補席，無一個合格內線替補！

但無人可用，郭士強指導只能死馬當活馬醫。大膽使用新人內線是他唯一的出路。目前在CBDL聯賽打球的盧梓傑和劉雁宇大概率會召回一隊，和王化東攜手一起充實內線缺漏，盡最大可能補足大韓傷缺後的位置。對於三人來說也會是一個上位的機會，看誰能夠把握住！

這次韓德君再受傷，也可以看出遼籃管理層和教練組對於球隊的應急方案考慮是有多差。球隊在上賽季現實的案例下，休賽期竟沒有一點主動補強內線替補的舉動。在沒有引援合適內線替補的前提下，大喊鍛煉年輕內線本賽季至今也成為泡影。依舊是保守的觀念。到了現如今這個情況，遼籃可謂是自作自受，上賽季就因大韓受傷陷入窘況，本賽季再重蹈覆轍，恐怕這一切，需要遼籃高層和教練組去好好反思了！
