世俱杯後再談中國女排奧運名單問題，一傳和全面性是選擇關鍵

2019-12-10 371條評論 2,389次閱讀 845人點贊

女排世俱杯結束了，高調出征的天津隊失望而回，折損了朱婷的她們在比賽中墊底。通過兩支參賽的中國球隊與國外球隊的交手情況看，一旦缺少了朱婷，中國球隊與國外的球隊相比差距是很巨大的，基本上沒有與之抗衡的能力。在東京奧運會「大考」越來越近的時侯，由於我國女排聯賽與國外強國的差距，指望在聯賽中培養出像艾格努、哈克等超級巨星的可能性不大，只能是依靠我國的舉國體制提前準備，通過在郎平的國家隊里集中訓練，力爭挑選出最適合參加奧運會的人選，以便在東京奧運會時可以和國外強隊抗衡。國外強隊一般攻擊力都很強，發球破壞性也普遍很大，與她們交手，對一傳的要求是很高的，也需要有比較全面的能力，因此在挑選奧運人選時預計這兩方面能力較高或比較有潛質的隊員入選機會是比較大的。

張常寧

朱婷作為中國女排目前隊員裡面唯一的「帥才」，一傳能力和全面性非常地突出，攻擊力更是無人可比，是中國女排的「核武器」，雖然在世俱杯上老傷複發缺席了剩餘的比賽，聯賽目前也處於休戰的態勢，但就算在奧運會開賽時朱婷傷勢還未恢復完全，相信郎平也會把朱婷帶上。李盈瑩在世俱杯上面對強手時的表現說明，她確實還是只能在中國女排里擔任替補，張常寧雖然還存在著或多或少的不足之處，但是她的一傳能力和全面性足以讓她坐穩朱婷對角的位置。

朱婷

在面對對手強大的進攻能力時，副攻位置上出色的攔網能力是很必須的，副攻攔網上的疲軟也是天津隊在世俱杯被對手在進攻上予取予求的重要原因。顏妮本賽季雖然有狀態下滑的跡象，也受到小傷病的困擾，但是她出色的攔網能力還在，因此在奧運會賽場上也還將會是郎平最倚重的副攻。袁心玥在世俱杯上中規中矩的表現說明，另一個主力副攻的位置非她莫屬。其餘的副攻，都與兩大主力有一定的差距，世俱杯上表現不錯的鄭益昕和楊涵玉是最值得培養的隊員，身材更高更年輕的楊涵玉由於更符合高大化的選材理念，成長的潛質更高預計將會在競爭中勝出。從世俱杯上姚迪和朱婷、李盈瑩等攻擊手的配合來看，目前最適合和中國女排邊攻手配合的二傳還是丁霞。替補的選擇從原來的姚迪和刁琳宇，又加入了在世俱杯中表現出彩的梅笑寒。不過梅笑寒要想打動郎平超過兩位前輩，恐怕還得要在聯賽上有更好的表現才行。

倪非凡

世俱杯上胡克爾和楊藝的表現與強隊的暴力接應們相比，可以說差距是非常地明顯，這也從側面反映出了龔翔宇是目前最適合中國女排的接應，哪怕她的攻擊力沒有那麼強，她的一傳保障和全面能力也是中國女排最急需的。在接應的替補上，曾春蕾雖然比較全面，但是由於年齡的原因提高的上限很小了，杜清清、毛鈞怡、陳佩妍都是條件很好而且上升空間比較大的年輕隊員，如果能在奧運會開賽前的備戰中有很大的提升，那麼就很可能得到去奧運會的機會，在聯賽和世俱杯上均有出色表現的杜清清是機會最大的。世俱杯上天津隊的慘敗表明，擁有一個出色的自由人對球隊是多麼的重要。具有很強的一傳和防守能力的王夢潔毫無疑問依然是中國女排自由人位置上的首選。為了保險起見，避免出現突發傷病的情況，最好還是不要冒險只帶一個自由人去奧運會，老將林莉雖然防守功力尚在，但同樣也存在提高上限不高的問題，江蘇的小將倪非凡，近兩年已經嶄露頭角，成長的天賦極高，非常值得帶到奧運賽場上培養。

李盈瑩

通過中國球隊在世俱杯上的經驗教訓，從一傳和全面性方面考慮，預測中國女排適合參加東京奧運會的人選將是主攻:朱婷、張常寧、李盈瑩，副攻:顏妮、袁心玥、楊涵玉，接應:龔翔宇、杜清清，二傳:丁霞、刁琳宇，自由人:王夢潔、倪非凡。

龔翔宇

丁霞
