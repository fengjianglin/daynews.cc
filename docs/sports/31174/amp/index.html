足壇罕見一幕！球員場上集體卧倒匍匐前進，邊裁在場邊笑開了花

天天要聞
2019-12-25

北京時間12月24日消息，近期非洲足壇發生了一件比較罕見的事情，在坦尚尼亞的一場當地聯賽中，大批蜂群突然飛進球場導致比賽被迫中斷，場上的球員們也不得不迅速採取緊急措施避險自救。幸運的是沒有人在此次突發事件當中受傷，而場邊的攝像機則記錄下了整個過程。

根據媒體報道，這場比賽交戰雙方為Young Africans與Iringa United，下半場第53分鐘場上比分為主場作戰的Young Africans3-0領先。就在此時，一大群蜜蜂突然從天而降飛進球場，最先發現異常的主裁判立即響哨示意比賽暫停，緊跟著球員們也紛紛倒地趴在球場上以躲避蜂群可能的攻擊。

從攝像機拍下的畫面可以看到，兩隊球員都趴在地上紋絲不動，並用球衣捂著腦袋，當上空的蜜蜂逐漸變少時，才開始一點點的向安全地區匍匐前進。至於場邊的球迷更是第一時間逃離了現場，也算是逃過一劫。

該事件被報道之後，不少人都為現場的球員與球迷捏了一把冷汗，畢竟如果被蜂群攻擊的話，後果不堪設想。不過好在這些親身經歷者們還都比較樂觀，尤其是其中一位邊裁，趴在地上顯得十分開心，起身後還與身後的攝影師開起了玩笑。

最終比賽在中斷數分鐘後恢復進行，剩餘的時間內主隊Young Africans再入1球，以4-0的比分大獲全勝。
