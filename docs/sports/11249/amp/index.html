參考圖，先看兩例不常見的圍棋五路尖沖，此手一出，江湖都嚷嚷動了。

武宮正樹的五路尖沖

1988年9月3日，日本東京，武宮正樹執黑林海峰執白，第1屆富士通杯世界職業圍棋錦標賽決勝戰（日本讀買新聞主辦），黑中盤勝。

黑69手走在參考圖的扳也很大，但是如果武宮正樹走這裡扳一手，而不是五路尖沖，這局棋還會這樣引起廣泛關注嗎？

阿法狗的五路尖沖

2016年3月10日韓國 首爾，李世乭執白谷歌圍棋AlphaGo執黑五番棋第2局， 黑中盤勝。

所有數據顯示，AlphaGo認為人類棋手是絕對不會下37這步棋的，不過AlphaGo很任性的認為，37更好！

一時間評論如潮：

黑37下五路超出了人類的理解範圍！從大局看，也沒什麼不好的，畢竟方向是對的。

「AlphaGo特別注重子與子之間的配合，圍棋怎麼下都可以了！這步棋要火！

人是不敢下這裡的，因為難度太大！但是AlphaGo可以，這也許就是AlphaGo的自信吧。

高者在腹，下者在邊，中者占角。這說明AlphaGo對中腹的掌控能力確實是獨一無二的。

第一章節

注意武宮正樹以後的走法，73，77很緊湊。

74，75一個回合大有說頭，首先74不如四路厚實的拐。

75還是可以回到上一圖扳一手的，這是研究室的主流意見。

第二章節

2016年，阿爾法GO橫空出世，對阿爾法GO的尖沖的一手，李世乭的應接有誤，阿爾法GO認為，白40不好，他的意見是貼長後點三三。

參考圖阿爾法GO力主的變化

因為上邊黑棋並沒有擴張，所以角上的利益更可觀。

實戰進行

經過12分鐘的長考，李世石最終下在了38位上貼，或許是受壓力的影響，李世石這步棋的方向明顯有問題。畢竟往這個方向貼黑棋，有嚴重的幫助黑棋下邊成空的嫌疑。

第三章節

再看看絕藝老師的尖沖小目，同樣令人感慨。

絕藝老師實戰對局中直接尖沖白棋小目，

此圖還原了一間高夾後飛壓的變化，阿爾法GO教學工具中出現過，黑棋完全可戰；

1-19，黑棋兩邊都走到，勝率升至6 7%。

黑棋吃掉了白角，大獲成功；勝率升至57%。看來外勢在絕藝老師眼裡很重要呢。

取材於2018年1月份絕藝老師的實戰例。

這一手尖沖小目，或許也是可以嘗試的下法。

關於尖沖

尖沖一般是對方在三路，然後四路尖沖，這個很常見。

依照傳統圍棋理論的認知，壓三路還是便宜的。所以，只要是能連壓三路，基本不虧。

絕藝老師的四路尖沖，阿法狗的五路尖沖，雖有令人費解之處，但都有強大的後續手段支持，希望大家仔細體會。
