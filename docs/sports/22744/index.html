榮譽之戰！創造11年輸球紀錄的國足，將怎樣結束2019？

2019-12-18 498條評論 2,839次閱讀 618人點贊

從亞洲杯時輸給韓國、伊朗，到中國杯中輸給泰國、烏茲別克，再到世預賽輸給敘利亞，東亞杯輸給韓國、日本，2019年，中國男足已經輸掉7場比賽，追平2008年，創造近11年的自然年輸球場次紀錄。

2008年，中國男足先後在東亞四強賽輸給韓國、日本，在友誼賽輸給墨西哥、阿曼、伊朗，在世預賽輸給卡達、伊拉克，無緣南非世界盃。這一年，福拉多下課，殷鐵生暫代國足主帥。2018年，中國男足單年度19場比賽5勝7平7負。

今年至今，中國男足經歷15場比賽，目前戰績為7勝1平7負。

2019年是中國男足歷史性的一年，以李可和艾克森為代表隊歸化球員登上國足的歷史舞台。

但是，歸化球員無法徹底改變千瘡百孔的國足，從一場2-1逆轉吉爾吉斯斯坦的亞洲杯小組賽開始，以一場三中衛送禮、里皮辭職的0-3結束亞洲杯之旅。

中國杯上，「泰國梅西」頌克拉辛全場戲耍國足，韋世豪一腳廢人腳踹向烏茲別克球員舒庫羅夫，都讓中國男足丟盡顏面。

6月，李可在與菲律賓的友誼賽中歷史性出場，9月，艾克森又在與馬爾地夫的世預賽中創造歷史。但是10月，中國男足在客場0-0戰平菲律賓，歷史上首次未能攻破對手球門。

11月，國足又在迪拜的中立客場1-2負於敘利亞，爭奪小組第一的希望基本破滅，里皮賽後再次辭職。李鐵帶領國足選拔隊征戰12月的東亞杯，前兩戰都以一球差距輸給了韓國和日本，姜至鵬的少林功夫再次成為中國男足的競技污點。

這一年，從里皮，到卡納瓦羅，再到里皮，再到李鐵，中國男足帥位連續更變。從現狀，到希望，一切與2008年沒有太大差別。

2019年12月18日15點15分，國足將迎來中國香港，這也是中國男足年度最後一戰，也是我們熟悉的榮譽之戰。
