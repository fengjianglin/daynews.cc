利物浦提前選定新帥！真相揭曉令球迷沸騰 內幕曝光上任時間確定

2019-12-19 1,306條評論 2,505次閱讀 467人點贊

北京時間12月16日，據多家英國媒體報道，利物浦高層已經商討過傑拉德回歸球隊執教的相關事宜，傑拉德本人也希望能成為利物浦主帥。

據悉，傑拉德一直以來都與利物浦高層以及球隊現任主帥克洛普保持著密切的聯繫，作為球隊的功勛名宿，目前正在蘇超格拉斯哥流浪者執教的傑拉德在利物浦球迷心中也佔據著十分重要的位置，因此，讓他回歸的聲音一直沒有停息過，目前這件事已經被利物浦高層正式提上日程。

值得一提的是，近日利物浦主帥克洛普與球隊完成續約，新合同將在2024年到期，而傑拉德也與格拉斯哥流浪者隊完成續約，合同截至日期同樣是2024年。有些人認為這只是一種巧合，但另外一種可能也無法排除，那就是到2024年兩人合同到期後，克洛普極有可能離開利物浦尋求新的挑戰，而名宿傑拉德有望來到安菲爾德執教。

此事引起了不小的關注，利物浦另一位名宿達格利什在接受採訪時談論了這件事。

「傑拉德與格拉斯哥流浪者續約到了2024年，但是這不意味著五年後傑拉德一定會重返利物浦擔任主教練，我不敢確定。聖誕節已經臨近了，大家對美好的童話故事心馳神往。但是，無論是我還是傑拉德或者克洛普，也可以說是所有人都無法預測五天後會發生什麼，五年後的事就更無法預知了。很多人都願意將二人的合同聯繫到一起看，但是我想說的是，大家還是應該現實一點。」

目前，克洛普帶領利物浦在英超一路高歌猛進，帶著巨大的優勢領跑積分榜，在本賽季已經進行的17輪英超比賽中，利物浦取得了16場勝利，僅有1場平局，還未嘗一敗。排名第二的是萊斯特城，已經落後利物浦10分之多，而賽季開始前就被視為利物浦最大對手的曼城已經落後了14分，就連曼城主帥瓜迪奧拉都曾明確表態，本賽季想要追上利物浦已經是一個幾乎不可能完成的任務了。雖然如今克洛普在利物浦如魚得水，但是五年後，他也難免需要一些休整的時間，或者有想要尋求其他挑戰的想法，到那時，傑拉德似乎就是最好的新帥人選。
