小魚兒吐槽體壇2：楊影，一生只愛丁寧一個人的偏見型解說

天天要聞
2019-12-25

小魚兒推出吐槽體壇系列，此為第二篇，往日文章點擊關注查看哦！

著名詩人木心有一首叫《從前慢》的詩大家都耳熟能詳，其中一句「從前的日色變得慢/車，馬，郵件都慢/一生只夠愛一個人」，感動了不知多少人。央視著名解說楊影女士語速慢，她的一生似乎也只愛一個人，這個人就是丁寧。只要是丁寧的比賽，楊影就無條件一面倒支持她，哪怕對手也是位中國球員。若那個人還威脅著丁寧在國家隊的地位（譬如劉詩雯），楊影是一定要在解說中進行批判的。

楊影慣用的招數：第一招，批評丁寧的對手有技術漏洞，技術沒有丁寧全面；如果對手技術上沒有出現什麼問題，楊影會採用第二招，批評丁寧的對手心理素質不好，沒有丁寧那麼大的心臟；若是對手全場表現平穩，楊影還有第三招，「精準」地指出，對手今天的運氣太好了。而她怎麼誇讚丁寧呢？丁寧如果輸球了，那就是她狀態不佳、身體有傷，卻仍然頑強拼搏，這種精神值得所有人學習；如果丁寧贏球了，那就是丁寧打的是實力球，在實力上碾壓對手。

因為如此帶有傾向性地解說，楊影和她的解說嘉賓劉偉，多次被送上熱搜，讓央視體育頻道名譽受損。大家都說，中國乒乓球這麼強，為什麼解說這麼弱呢？以前的蔡猛只會跑火車，以為貴為世界冠軍的楊影是來拯救中國羸弱的乒乓球解說的，沒想到她又過來砸了一個更大的坑——楊影也被派去解說過其它項目，結果在解說網球時，以一句「米爾扎削了一板」技驚四座、艷壓群芳。

——原創文章，歡迎大家批評指正，小魚兒期待你們的關注和點贊！
