太難了，正負值-26加3失誤，羅斯仰天長嘆，沃克6中0仍是贏球功臣

2019-12-21 212條評論 1,920次閱讀 514人點贊

凱爾特人今日坐鎮主場迎來活塞，第一節兩隊一直膠著，綠軍只領先了2分，第二節主隊展現出了強大的實力，他們在主場球迷的加油聲中打出了一波14比2的攻擊波後迅速拉開了比分。儘管活塞進行了反撲，但凱爾特人仍然帶著12分的領先優勢進入下半場。

第三節前半段活塞還能憑藉穩定的進攻咬住比分，但之後綠軍徹底攤牌了，他們希望早點拿下比賽然後可以讓主力進行休息，這一節他們又贏了10分。第四節主隊開局再次打出一波小高潮，確立起了27分的巨大優勢，最終凱爾特人以114比93輕鬆拿下了活塞。

活塞方面：德拉蒙德12分11個籃板4次助攻6次搶斷，大莫里斯10分6個籃板2次封蓋，羅斯14分4助攻。

凱爾特人方面：布朗26分5個籃板5次助攻3次搶斷，塔圖姆26分5個籃板3次助攻，威廉姆斯18分3個籃板。

今天羅斯真的太難了，他出場24分鐘，12中6，正負值-26外加3次失誤。如果按照球迷們的一貫標準，他肯定是要背鍋了。幸好現在大家早就沒有再用MVP的標準來要求羅斯，很多人只希望他能健康地打完這個賽季就好了。也正因為如此，羅斯表現好的時候都是拚命吹，表現差的時候卻沒多少人黑，這也是連聯盟第一人詹姆斯都享受不到的待遇。

輸球後的羅斯只能仰天長嘆，不過之後他馬上恢復了平靜。羅斯表示自己會向前看，這個賽季還有很多場比賽，他希望能幫助活塞取得讓球迷滿意的成績。

現在活塞面臨著很尷尬的處境，雖然他們名義上有三巨頭，但卻打不出讓人滿意的效果，羅斯和格里芬都有玻屬性，必須省得用。德拉蒙德雖然沒有大的傷病，但畢竟他只是一個藍領中鋒。這支球隊未來真的有可能要重建了。

相比羅斯，沃克無疑要幸運得多，因為他有可靠的隊友。今天沃克6中0，只拿到2分，但他仍然是贏球的功臣。沃克今天手感不好，於是他把精力放在了組織和防守上，送出了11次助攻1次搶斷1次封蓋，再加上隊友們足夠給力，所以他的正負值才達到了全隊最高的+18。

不得不承認，相比歐文，沃克確實更合適凱爾特人，他用極快的速度融入了球隊的戰術體系，綠軍確實是做了一個明智的決定。
