史上最狂新秀！菜鳥賽季就被全聯盟針對，結果卻打服全聯盟

天天要聞
3 天 ago

聯盟作為全球籃球職業賽場上的最高殿堂，這裡一直都匯聚著全世界最出色的一批職業球員，而在這群英會聚的戰場上，自然也從來都不缺乏一些實力出眾，天賦異稟的天才球員。這些球員往往可以在新秀時期就展現出極強的個人實力，拿出令整個聯盟都為之震驚的表現。

魔術師約翰遜在他的新秀賽季就已經拿到了總決賽MVP的獎項，這絕對是令人感到不可思議的成就，而羅斯則是在22歲就力壓詹姆斯，斬獲常規賽MVP，還有如今的東契奇，字母哥，這都是年紀輕輕便已經可以斬獲不少榮譽，拿出精彩表現的天才新星。但他們比起下面這個人來，都還是要遜色不少的，而這個人，也幾乎是無法被超越的傳奇了。

這個人就是「籃球之神」喬丹，他是在1984年進入到聯盟的，當時的公牛在選秀大會中做出了球隊歷史上做正確的決定，那就是選擇用第三順位的選秀權簽下了喬丹。儘管喬丹當時並沒有拿到狀元秀，但他的天賦卻早已顯露無遺。

在新秀賽季中，喬丹場均砍下28.2分6.5籃板5.9助攻的數據，投籃命中率高達51.5%。這讓當時的聯盟都為之驚嘆，這完全不像是一名新秀球員可以做到的事情。他當時的場均得分數只比拉里. 伯德低0. 5分，位列聯盟第三，而且因為喬丹出色的表現和無與倫比的籃球天賦，讓當賽季公牛的主場上座率得到了極大的提升，而他也在一年級時就入選了全明星陣容。

但當時的喬丹卻在全明星大賽中惹惱了許多知名巨星，他們認為喬丹太過於傲慢了。他先是在扣籃大賽中穿著訓練背心就上了場，這讓許多人認為他不尊重這個比賽，也不尊重在場參賽的球員。隨後，據當時的球員回憶稱，年輕的喬丹與以色列.托馬斯在同乘電梯時表現的十分傲慢，無視托馬斯，這讓微笑刺客感到十分惱火。於是在那屆全明星大賽中，托馬斯聯合魔術師等人孤立了喬丹，不給他傳球，喬丹全場只拿到了7分。這讓喬丹十分受打擊，並且決心用賽場上的表現來做出回應。

在那以後，喬丹每當在賽場上遇到微笑刺客，都會狀態爆棚，他職業生涯面對托馬斯場均可以砍下31.2分6.3籃板5.9助攻的數據，並用兩次三連冠，十個得分王以及六個FMVP來征服了整個聯盟。
