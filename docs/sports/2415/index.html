魯能再失足協杯 李霄鵬主動攬責 直言最差魯能碰最好申花

2019-12-07 0條評論 23次閱讀 0人點贊

3-0，申花笑，魯能哭。

山東魯能再次無緣足協杯冠軍，連續兩年亞軍，也未能獲得明年亞冠參賽資格。賽後新聞發布會上，李霄鵬主動攬責，直言這是魯能今年踢得最差的一場比賽。

李霄鵬首先祝賀申花奪得冠軍，稱他們今天的表現非常出色，相比之下魯能較差，給支持魯能的球迷說聲抱歉。之所以失利，從準備到發揮，以及心態的調整上都很失敗。「我還是感謝隊員，做了最大的努力。」

對於魯能連續兩年打進足協杯決賽，但都未能奪冠的窘境，李霄鵬表示：「兩次決賽有所不同，去年是差一點，今年是差了很多。這場球是全年最差，而我們碰上了全年最好的申花。隊員過分緊張了，這是我的問題。對冠軍的渴望，好像馬上就要到手冠軍了，隊員的壓力我知道，怎麼減壓我還沒不會，這是一種學問。」

記者繼續提問，連續兩年倒在了決賽，這支球隊未來提升需要什麼，明年單線作戰什麼目標？李霄鵬說：「不是冠軍的球隊都找借口，一個是硬實力，一個是心理，冠軍球隊必須有心裡過硬的球員。明年不管怎麼樣還是由董事會決定，個人建議，這個隊要挑戰聯賽60分，60分以上才是強隊。」
