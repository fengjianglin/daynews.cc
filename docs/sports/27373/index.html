為何球迷都看好北京會淘汰天津女排，看完分析，豁然開朗

2019-12-20 1,107條評論 3,364次閱讀 2314人點贊

破案了！為何球迷都看好北京會淘汰天津女排，看完分析，豁然開朗。網友熱議一針見血地指出：北京女排優勢在打球的人多，可以有不同的組合。而反觀天津女排能打球的人少，一套陣容很容易被熟悉，容易破解。我們應立足自己，在一套陣容的情況下，換輪次換人。朱婷不能打的話，有很大的困難。天津女排隊伍要做好：天津女排教練應變要快，隊員要堅決執行部署。發揚天津女排精神，力拚對手。本來天津女排實力差，更要靠精神支撐，永不放棄。每球必爭。我們實力比北京弱，為什麼這麼說：有朱婷也不佔優勢。北京車輪戰，隊員不累，組合多，適宜起來難。隊員打的就累。一旦體力不行了，技術就無從談起。

朱婷不能打的話更弱了，我們天津女排要清醒地認識這一點，主力配合還要加強。北京隊的優勢就是天津隊的弱勢，天津隊困難時幾乎無人可換。半決賽天津vs北京，如果朱婷不上場，天津必敗無疑。天津有個球迷，他當初反對朱婷去天津。理由是：我們有一個李盈瑩，就已經讓天津打不死防守下降了很多，如果再來一個朱婷，沒法玩了。所以天津的軟肋就是防守。世俱杯被打的潰不成軍就突出了這一點，現在的天津除了進攻能看，防守是真的沒有以前頑強了。所以北汽用發球和進攻一頓猛砸，有戲。朱婷胡可而李盈瑩都一米九以上，防守很吃力的，所以北汽是有機會的。我估計半決賽朱婷會上場，因為朱婷很想要聯賽冠軍。

北汽3:0客場橫掃上海，北汽年輕球員主打，體現出衝勁兒和活力，配合外援表現搶眼。蔡雅倩狀態提升，終於多次嘗試後二次球得了一分。替補二傳也上場鍛煉，張建章此戰顯示出膽魄。本賽季，北京女排的保障環節做得好，防反成功率高。金燁有亮眼表現，關瑞格和小二傳雷海林也登場亮相了。金葉確實後排防守穩健，任凱伊二號位進攻也輕靈，這兩個主攻成長起來了，真好！看了幾場北汽的比賽，那小二傳敢打敢拼，完全不見生澀，全隊整體作戰能力依舊強悍，板凳深度可觀。可見張教練帶隊有方，俱樂部請去的老鄭頭比天津雇來的日本教練更對症更合契啊。朱婷能上還有的打，朱婷不上則很難，當然排球就是抑制與反抑制，我們雖然有弱點，但對方也不是無懈可擊，盡量揚長避短吧，也不一定完全沒機會。

破案了！為何球迷都看好北京會淘汰天津女排，看完分析，豁然開朗。北汽女排取得的進步還像總有人視而不見。一提起上賽季確實北汽女排奪冠了，其實更應該注意的成績是她們取得了常規賽亞軍！要知道以前她們連續三年都是常規賽第七名。常規賽是在二次引援之前，還沒有丁霞和王夢潔，北京和天津都在自己的主場3:0戰勝了對手，實力基本相當。但是今年北汽的一傳水平不穩，而且李盈盈比去年進步，去年王媛媛有傷病，而且胡克爾加盟，這是在朱婷不上場的情況下，天津實力要比北汽高出一截。北汽女排以前年年第七，球隊一直努力在證明我不行！現在北汽奪冠了，證明我行，我可以做到。難道現在又開始回到老路上去，證明自己我不行了？

對此，大家怎麼看？
