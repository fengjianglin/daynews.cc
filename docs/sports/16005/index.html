超級大冷門！世界第一爆冷出局！土豆哥大爆發半決賽對陣塞爾比！

2019-12-14 318條評論 1,585次閱讀 577人點贊

北京時間12月12日凌晨三點，2019斯諾克蘇格蘭公開賽另外兩場半決賽展開較量。現世界第一特魯姆普對陣吉爾伯特，泰國一哥塔猜亞烏諾對陣利索夫斯基。在較早之前結束的兩場比賽中，馬克艾倫5-1大勝輕鬆晉級，塞爾比5-4絕殺奧沙利文逆轉晉級。火箭贏得4局有三場都單桿過百可惜還是輸了。

特魯姆普-吉爾伯特

作為目前的世界第一和新一代斯諾克領軍人物，特魯姆普在斯諾克領域擁有很高的人氣，特別是在中國除了丁俊暉和奧沙利文之外，小特的人氣當屬第一。前四輪比賽，小特贏得比較輕鬆，前兩輪都是4局直接橫掃，上一輪面對淘汰丁俊暉的世界冠軍多特則是4-2順利晉級。而吉爾伯特現在世界排名第十，前四輪當中，他遇到了兩名中國選手，其中在第三輪，他4-3險勝肖國棟艱難晉級。兩人最近一次交手是在今年9月份，特魯姆普6-4獲勝。

第一局，特魯姆普先發制人，單桿轟下66分後以110-6拿下首局取得開門紅。第二局比賽，特魯姆普率先上手，面對打好局勢，小特明顯著急了在拿到34分時進攻失誤留下機會，吉爾伯特單桿轟下71分直接超分，兩人打成1-1平。

第三局，吉爾伯特展現出了火熱的手感，上手之後打出了行雲流水的進攻，最終單桿轟下98分乾淨利落拿下第三局比賽。吉爾伯特連贏兩局實現反超。雙方前三局比賽結束，一共只用時40分鐘，打的非常快。

第四局比賽，雙方各自互送大禮，吉爾伯特遠台拼進紅球再次率先上手，遺憾的是僅僅拿到11分打綵球不進並且給小特留下了進攻機會。特魯姆普上手後拿到20分也出現進攻失誤將機會又送給了對手。吉爾伯特打到29分再一次失誤，兩個人就這麼禮尚往來了幾個回合，第四局也打的零零碎碎。最終防守糾纏中，吉爾伯特再一次找到機會以52-43艱難拿下第四局。吉爾伯特連下三城3-1領先小特進入半場休息！

第五局比賽回來，特魯姆普終於找回了一些手感，展現出自己強大的進攻實力，拼進紅球上手後單桿轟下101分過百成功追回一局2-3。關鍵的第六局，小特防守留下下球機會，吉爾伯特打進紅球成功上手，這一上手又是單桿轟出142分高分。面對對手如此火熱的手感，小特也只能幹坐在板凳上摸了摸自己的下巴。4-2！吉爾伯特率先拿下賽點！

關鍵的第七局，又是吉爾伯特率先打進紅球上手，隨後圍繞高分綵球連續得分，單桿轟下53分並最終拿下第七局！5-2！吉爾伯特淘汰世界第一特魯姆普與塞爾比會師半決賽。奧沙利文與特魯姆普全都止步八強！實在是太讓人意外了！
