曝前巴西U17國家隊前鋒將加盟中超！曾在西亞19場造14球

2019-12-06 0條評論 24次閱讀 0人點贊

摘要

近日據巴西媒體《joga10 news》報道，巴甲球隊巴拉納競技前鋒馬塞洛，已經同意前往中超俱樂部重

近日據巴西媒體《joga10 news》報道，巴甲球隊巴拉納競技前鋒馬塞洛，已經同意前往中超俱樂部重慶力帆效力，他的合同將為期三年。目前重慶力帆官方還沒有宣布這筆轉會，預計在2020年1月份中超冬季轉會窗口開啟後，馬塞洛正式加盟球隊。

​馬塞洛曾經作為前鋒在巴西U17國家隊出場過，出生於1992年的馬塞洛，今年27歲，身高182公分，主力腳為右腳，可以踢中鋒、影鋒、右邊鋒、右邊前衛四個位置，職業生涯大部分時間都是右邊鋒。同樣這並不是馬塞洛第一次來到亞洲踢球，他曾經在2017年夏天就被阿聯酋豪門阿爾納斯爾相中，租借加盟一年，在這一年時間裡，馬塞洛為阿爾納斯爾在19場比賽中造14球，貢獻了10球3助攻，效率不俗。

​雖然效率不低，不過據巴西媒體介紹，馬塞洛自2009年加入巴拉納競技以來，他曾經是一名邊路非常快的球員，但是現在經驗雖然更豐富了，但是速度沒有以往那麼快了。由此可見，如果重慶力帆是把他當做費爾南多的替代者，或許馬塞洛的表現會和預期有一定的出入，不過馬塞洛身體非常強壯，本賽季巴甲聯賽，馬塞洛為巴拉納競技貢獻了9球2助攻，在中超賽場相信也能很快的適應。

​不過馬塞洛的身價並不高，根據《德國轉會市場》上的估價，他目前身價只有180萬歐元，而且與巴拉納競技的合同也將在2019年12月31日到期，作為一名效率不低的前鋒，重慶力帆引進他並不需要轉會費，也是一筆性價比比較高的轉會。本賽季重慶力帆自從費爾南多前往河北華夏幸福之後，重慶力帆的戰術打法也發生了很大的改變，下賽季在球隊主力前鋒卡爾德克不變動的情況下，下賽季馬塞洛將會在邊路為重慶力帆提供什麼樣的幫助，也讓我們拭目以待。
