皇馬諾坎普圍攻巴薩！把萬人噓聲都征服 齊祖妙手打造16年最強

天天要聞
2019-12-19

皇馬憾平巴薩，表現更出色卻未能破門。齊達內連創紀錄，激活最佳狀態的白衣軍，上半時表現堪稱傑出，在諾坎普一度圍攻巴薩禁區，還遭遇2分鐘內2次點球爭議。奇兵貝爾險複製C羅昔日絕殺巴薩一幕，如此皇馬又何懼曼城？

皇馬戰平巴薩

半場12射=16年最佳，皇馬在諾坎普圍攻巴薩

兩隊近50場交鋒首次0-0戰平，也是2002年11月後首次均未破門，打不過比賽內容依然值得品味。相比巴薩首發變化較大，齊達內陣型與人員安排基本均在西媒預計內，僅貝爾先發算個小意外。上輪被雪藏的卡塞米羅擔任後腰，巴爾韋德克羅斯分居兩側，進攻時主打菱形中場4312，防守時改為4411，伊斯科貝爾輪流負責協防邊路，也會輪流前插與本澤馬配對，既可利用伊斯科本澤馬在前場憑控球能力負責由守轉攻推進，也能中長距離直塞直接尋找空切的貝爾。

皇馬半場射門12次，創近16年聯賽客戰巴薩新高

皇馬開場後就不遺餘力猛攻，上半時表現極佳，尤其保持三線距離方面很出色，中場攔截高效，陣型鋪開與收縮時機把控相當不錯，導致德容朗格萊等屢次中後場向前出球被斷，巴薩禁區內解圍球後落點也基本被客隊控制，便於連續發動攻勢。

創世紀大戰出場紀錄的拉莫斯，率領皇馬一度圍攻巴薩

優勢階段無法進球，是皇馬本賽季一大弊病，卡塞米羅頭槌沖頂被皮克門線救險，巴爾韋德遠射稍偏，機會把握能力欠佳是後C羅時代的通病。巴薩上半時雖只3次射門機會，可梅西掃射空門，阿爾瓦單刀墊射偏出，均是絕佳破門良機，儘管格列茲曼幾乎消失，德容出球被遏制，可成型攻勢較少的巴薩也不缺機會，不過西媒一致認可，皇馬上半時是統治比賽的一方。前45分鐘客隊完成12次射門，4倍於衛冕冠軍，控球率也達49%，場上一度壓制主隊禁區圍攻。半場12射，是近10年皇馬聯賽世紀大戰新紀錄，也是2003-04賽季後客戰巴薩時半場新高。

不靠玄學，齊達內激活最佳皇馬

若評皇馬賽季最佳，齊達內足以與本澤馬並駕齊驅，他將熱身賽人見人欺、兩月前還被升班馬馬洛卡擊敗的球隊，逐漸磨合調教成型。

齊達內無疑是本場世紀大戰的隱形贏家

皇馬上半時便有卡瓦哈爾、巴爾韋德與門迪3人保持傳球成功率100%，巴爾韋德還有2次攔截與2次射門，攔截數隊內第1，後兩人均是今季被齊祖提拔為半主力的新秀，門迪還險完成助攻。莫德里奇貴為歐冠三連冠大功臣、去年金球獎得主，依然要在賽季最重要賽事當替補，也代表齊祖革新陣容的決心，正是更年輕又具活力的陣容，讓皇馬在局部戰役與個體對抗中佔優勢。

齊祖愛徒巴爾韋德，上半時表現非凡

雖然此役齊祖並無太多新招奇招，可本賽季他逐步調節提升球隊狀態，輪換陣容合理用兵等方面，還是充分展現出歐冠三連冠功勛教頭的功底。客戰巴薩壓上進攻毫不保守，主動爭取場面主動權，陣型安排上，齊祖不拘泥某一套打法，近期輪流採用433與4312，甚至切換得比執教皇馬一期時更頻繁，下半時球員體能下降，一度防線吃緊漏洞增多，齊祖又加派羅德里戈與莫德里奇，改打4231，險由貝爾前插偷襲破門。

卡塞米羅成為皇馬中場的中流砥柱

防守中皇馬破綻依然存在，不過三後腰布陣對兩中衛掩護作用突出，自2007年12月（巴普蒂斯塔絕殺，1-0）後首次聯賽交鋒零封巴薩。全場皇馬前場30米傳球171次，也遠勝巴薩108次，創造機會同樣12比8佔優。拉莫斯感嘆：「在諾坎普從巴薩腳下搶球有多難，可我們做到了。」

奇兵貝爾險立奇功，越位咫尺之遙

丹青妙手齊祖要肆意揮毫潑墨，也得有稱手文房四寶，貝爾本澤馬伊斯科卡塞米羅缺一不可。貝爾首發是此役齊祖最大變招，之前大聖受傷缺席半月，復出後只替補打21分鐘比賽，體能狀態均是未知數，齊祖卻在世紀大戰押寶這位與其關係不睦的弟子，便是寄望其速度能在邊路打開通道，巴薩左閘阿爾瓦上周才復出，客戰皇家社會因發揮糟糕被換下，打蛇打七寸，齊祖希望貝爾趁其狀態不穩時主攻右側尋找突破口。上半時貝爾個人雖機會有限，不過皇馬右側的確打活，卡瓦哈爾為卡塞米羅送出精準傳中險破門，巴爾韋德同樣頻繁前插。

門迪略微越位，貝爾進球被判無效

下半時雙方體能下降，皇馬球員無法在前場再高效逼搶斷球，中場被迫向後撤與防線靠近，齊祖隨即指示伊斯科本澤馬回撤協助中場，貝爾頂前擔任中鋒，利用其衝刺速度在皮克朗格萊身側或背後尋覓空隙。效果立竿見影，貝爾5分鐘內獲兩次絕佳機會，前一次禁區左肋抽射擊中邊網，後一回門前捅射破門，只是為其傳球的門迪前插瞬間，左肩略探出皮克右肩平行線。貝爾距定勝負進球就差這准「體毛越位」，否則皇馬將極可能自2016年後聯賽做客諾坎普首勝，大聖也將複製3年前C羅絕殺巴薩的輝煌一刻。

兩分鐘2點球懸疑，拉莫斯怒噴不公

上輪聯賽戰平後，巴薩上下抨擊VAR不介入審查疑似點球，之後還招致皇社方面用畫面截屏反擊，如今輪到皇馬方面叫屈，上半時瓦拉內兩分鐘內遭遇對手兩次禁區內疑似犯規，西甲金哨埃爾南德斯均未理會。

拉基蒂奇拉拽瓦拉內球衣造成後者摔倒

皇馬高管布特拉格諾不滿，「看起來那麼清晰的問題，為何VAR不回看？我們該有兩個清晰的點球，VAR該介入」，拉莫斯也怒噴：「兩個清晰的點球，半場時我們就看到畫面了，可我們沒法改變什麼，或許其他賽事中我們會得到哨子……」

朗格萊踩踏瓦拉內腿部

瓦拉內腿部傷痕明顯，布特拉格諾抱怨VAR不介入

18分鐘拉基蒂奇放倒門前搶點的瓦拉內，是較明顯一次，前西甲金哨安杜哈爾表示：「拉基蒂奇對瓦拉內有拉拽動作，犯規十分明顯，若VAR審查，該是個點球，非常清晰的點球，VAR在哪？」另一前西甲名哨伊圖拉爾德觀點一致：「拉拽動作明顯，持續時間也很長，是個點球」。在此之前，朗格萊還曾抬腿踹中瓦拉內右大腿，伊圖拉爾德認為這也是主裁與視頻裁判貝諾切亞的誤判，「很清晰的點球，因為是個危險動作，鞋釘直接蹬中腿，瓦拉內該在2分鐘內獲得兩個點球。」

齊祖造1神跡+創客戰巴薩不敗新紀錄，如此皇馬何懼曼城？

拋開爭議不提，平局對皇馬也算可接受的結果，畢竟若按原定賽程雙方應在10月末對壘，對當時剛遭遇聯賽首敗、歐冠前兩輪不勝的白衣軍而言，幾乎不可能撼動當時狀態正佳的巴薩。可時隔2月再戰，皇馬反而是在諾坎普發揮更出色、更有資格抱怨平局不公的一方，本身就是齊祖的成功。法國人執掌皇馬後客戰巴薩還從未兵敗，2勝3平，成史上首位前5場世紀大戰客場均未敗北的皇馬主帥，更難得的是，這也是皇馬隊史首次在分差與宿敵相差5分以內，且紅藍軍是聯賽領頭羊的局面下，客戰巴薩拿分，之前是12連敗。

齊達內創紀錄，連續5場率隊客戰巴薩保持不敗

巴薩36分領跑，是2001-02賽季（皇馬31分）後西甲前17輪積分最少領頭羊，雙雄爭霸依然存不小變數。西甲仍有21輪，如今同分領跑格局，對今夏才投入3億巨資換血的皇馬並不壞，下半程世紀大戰又是在伯納烏打響，皇馬有機會後發制人佔據優勢。

如此皇馬完全可與歐洲最佳球隊一較高下

傷病對齊家軍影響依然較大，阿扎爾馬塞洛兩位左路主力均缺席，超級替補阿森西奧也重傷休養至今，若這些生力軍能在聖誕假期後或下半程復出，又將是增強球隊基本實力與陣容厚度的有益補充。約維奇維尼修斯兩位新秀的高潛力，也有待齊祖開發，風調雨順之時往往一切都會邁入正軌良性運轉，瓜帥的藍月軍團雖是歐冠奪冠賠率第1的熱門，可若皇馬能保持此役氣勢、狀態與活力到明年交鋒時，又何懼曼城？
