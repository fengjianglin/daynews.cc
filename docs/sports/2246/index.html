泰拳王播求遭“歧視”！70KG級世界第一回應挑戰播求：他太老了

老驥伏櫪，志在千里，烈士暮年，壯心不已！作為搏擊圈70KG級年齡最大的世界前十拳手，泰拳王播求一直保持著超強的自律，期待著自己能在擂台上多打幾年，以至於37歲的播求依然刻苦訓練，前不久登上MASFIGHT擂台，照樣能1回合KO荷蘭悍將克里斯-吉比。播求在中國搏擊圈威望很高，甚至有拳迷認為中國搏擊圈70KG級選手至今依然沒人能戰勝播求。但在70KG級頂級高手眼裡，泰拳王播求的時代，已經過去了。比如近日，70KG級世界第一在採訪中「歧視」了播求一把。

這位70KG級世界第一便是馬拉特-格里戈里安！說起馬拉特-格里戈里安，此人目前在Combatpress踢拳輕量級排名世界第一，是當今70KG級最炙手可熱的踢拳巨星，沒有之一。儘管他曾4度敗給西提猜，但最後一次，馬拉特-格里戈里安贏了西提猜，而且比賽期間還擊倒了西提猜。不誇張地講，馬拉特-格里戈里安，是目前70KG級選手中，把踢拳技術同力量完美融合的第一人，甚至喬治-佩托西奧和西提猜（力量欠缺）都比不上他。

日前，馬拉特-格里戈里安進行了Glory輕量級冠軍的衛冕戰，在此期間，有格鬥媒體對其進行了採訪。在採訪中，格鬥媒體向馬拉特-格里戈里安提出了挑戰播求的問題。記者問道：想不想和播求打一場？馬拉特-格里戈里安回答稱：「播求在我心中就是傳奇一樣的人物，我一直很想和他打一場，但是現在他已經老了，我現在如果發出挑戰的話，是一件不好的事情！而且他的確太老了，打不了了！」

很明顯，在馬拉特-格里戈里安的話中，透露著他的真實意思：播求成就很高，是一個傳奇人物；但如今他太老了，我挑戰他傳出去不好聽！的確，馬拉特-格里戈里安如今28歲，正值壯年，而且技術和力量都處在巔峰時期，此時挑戰37歲競技狀態下滑的播求，著實是有點「欺負」人。但那句「他的確太老了，大不了了」，也確實有點「歧視」播求的意思，不過這也可以理解為當今70KG級頂級王者的霸氣所在。

其實不止播求，以馬拉特-格里戈里安目前的實力，恐怕像「電腦鬥士」雅桑克萊和「醫師」喬治-佩托西奧這一輩的頂級老將，如今都很難將馬拉特斬落馬下。畢竟馬拉特-格里戈里安曾創造過一個恐怖KO紀錄–28秒KO蘇波邦，而後者也是70KG級的頂級高手，曾戰勝過西提猜，KO過江通猜。對於馬拉特評價播求太老的事情，你怎麼看呢？他給你的感覺是狂妄還是霸氣呢？歡迎在評論區留下你的觀點！
