阿森納的救世主來了？18歲天才橫空出世，感謝曼聯的走眼之恩

2019-12-11 344條評論 3,996次閱讀 171人點贊

加布里埃爾·馬丁內利，或者，你可以叫他「首秀之王」。

在2019年12月10日到來前，命途多舛的阿森納，已經整整8年沒有在英超客場上演過半場輸球、然後逆轉獲勝的戲碼了。2011年10月29日，憑藉范佩西的帽子戲法，上半場1比2落後於切爾西的阿森納，以5比3拿下了倫敦德比的勝利。如此遙遠的客場逆轉取勝，實在是阿森納球迷不可多得的記憶。

如今，在2019-20賽季又一場倫敦德比到來時，已經目睹了7輪不勝的阿森納擁躉們，終於可以暢快淋漓了。上半場0比1，下半場3比0，由永貝里代理執教的阿森納，終於以3比1戰勝西漢姆聯的賽果，迎來了兩個月以來的首場聯賽勝利。

對於近13場客戰倫敦德比僅有3勝的阿森納而言，當選最佳球員的奧巴梅楊和貢獻傳射建功的尼古拉斯·佩佩固然重要，但如果要找出改變氣場、吹響衝鋒號的存在，那終究屬於在第60分鐘扳平比分的巴西小將——加布里埃爾·馬丁內利。

首發90分鐘，2次射門，1個進球，創造1次機會，8次反搶(並列全場第一)，4次搶斷(本隊第一)，20次衝刺(全場最多)。以18歲174天的年齡，馬丁內利成為了阿森納隊史在英超首發處子秀最年輕的進球者，沒有之一。也正是在他推射得分後，阿森納才掀起了反攻浪潮，並一舉在十分鐘之內完成逆轉，走出了持續了兩個月的頹勢。至少在這一夜，年輕的巴西小將扮演了兵工廠的救世主。

毫無疑問，馬丁內利是本賽季阿森納的最大驚喜，即便是初來乍到、機會有限，他的數據也足夠讓人驚艷：英超8場186分鐘，進1球；歐聯杯4場330分鐘，3球2助攻；聯賽杯2場180分鐘，4個進球。憑藉聯賽杯和歐聯杯處子戰的四個進球打底，巴西前鋒來到英格蘭的首個賽季，已經以14場8球創造了個人生涯紀錄。

此前，在出道後效力於伊圖諾的兩個賽季，馬丁內利的總數據也只有34場10球而已。

加布里埃爾·馬丁內利究竟是誰？至少在今年夏天，將目光完全投向了塞瓦略斯、蒂爾尼、佩佩和戴維·路易斯的阿森納球迷，應該對這個名字並不熟悉。2001年6月18日出生，巴西、義大利雙國籍(出生於巴西，父親來自義大利)，轉會費僅有670萬歐元。由於職業比賽經驗屈指可數，之於歐洲足壇也是一片空白，馬丁內利並沒有在加盟阿森納時吸引太多的目光。

有趣的是，就在馬丁內利來到北倫敦之前，他還曾與曼聯產生過一段孽緣糾葛。從2015到2017年，效力過科林蒂安和伊圖諾梯隊的馬丁內利，曾經四次來到過曼徹斯特，參加了曼聯的試訓。那時候，他與埃夫拉和費萊尼都留下過合影，而博格巴的噓寒問暖更讓其印象深刻。

在一些青年隊賽事中，馬丁內利還與目前升上紅魔一線隊的格林伍德並肩作戰。但最終，巴西小將並沒有得到曼聯開出的合同，不知是「走眼」抑或另有打算，紅魔與這位巴西前鋒擦肩而過。幾個月之前，在引發多家歐洲豪門一哄而上後，馬丁內利也將目的地再度定為了歐洲：不是巴塞羅那，不是皇家馬德里，更不是埃因霍溫，而是後來居上的阿森納。

伴隨著處子球的接踵而至。這個崇拜C羅的大男孩兒，已經要在阿森納創造屬於自己的天地。
