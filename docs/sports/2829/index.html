CBA第13輪，廣東男籃客場挑戰八一男籃，這場比賽將於北京時間12月5日進行。一個是9冠王，一個是8冠王。可是央視沒有直播計劃，在當天CCTV5是直播女排世俱杯，CCTV5+是直播深圳男籃對陣四川男籃。

廣東能贏八一多少分？今時不同往日，現在的八一隊已經淪為魚腩球隊，廣東隊的新人又一次難得的機會來了。相信杜鋒會以鍛煉新人的一場教學比賽。檢驗隊中新人的真正目的。而且也不會大比分勝出，更多的因素是杜鋒與王治郅的關係，不過比賽過程會很激烈。

廣東主教練杜鋒的執教向來就被球迷引發許多爭議，他用人大膽，上次對陣新疆的那場比賽，最後一次關鍵的進攻居然上了小將來自北大的萬聖偉，最後萬聖偉不負眾望，空切籃下接到傳球打進2分。最後也有驚無險的以3分優勢贏下了那場三分爆發的新疆。

由於兩隊實力相差懸殊，易建聯是否能夠輪休？12場比賽阿聯出場時間都不少，這場比賽如果輪休也很正常，廣東可以藉此鍛煉年輕球員，同時給與阿聯一定的積攢體能機會。八一曾經的八冠王已經成為歷史，目前的八一隊由於沒有外援所以看他們的比賽很揪心，由於上賽季排名最後兩名的同曦，還有北控擁有三外援政策加上休賽期補強，實力明顯提，反倒是八一還是原地踏步，目前和福建成為了難兄難弟，所以過去的輝煌並不能代表現在的八一，即使九冠王的廣東和其相碰也沒有火星撞地球的那種激烈感，反而略顯平淡無奇。

王治郅與杜鋒以前都是國家隊隊員，他們都熟悉對方，儘管八一整體實力較弱，我想王治郅也不會放棄這場與榜首球隊的較量，儘可能去拼這場比賽。王治郅說：「現在我們隊這邊會盡量盡自己最大能力去打，現在我們跟強隊還有一些差距，但是我們落後的時候大家沒放棄，這是我們一直在強調的，需要通過防守來爭取更好的一些機會。

看點就是八一到底可以和廣東隊抗衡多久，郭昊文面對強敵可以得到多少比賽時間，能得多少分！
