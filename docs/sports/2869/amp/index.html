Benemax-Virgo 2019北美洲挑戰賽+

12月4日-8日，Benemax-Virgo 2019北美洲挑戰賽+將在加拿大萬錦市拉開戰幕。作為本年度最後一站挑戰系列賽，諸多乒壇好手趕赴加拿大，抓緊最後機會爭奪積分。

值得關注的是，隨著2020東京奧運越發臨近，由世界排名決定奧運參賽資格的日本隊隊內競爭近乎白熱化，多位主力選手也將參加本站賽事，儘可能賺取更多積分。男子方面，丹羽孝希、吉村真晴2位主力選手報名，丹羽孝希也是本次賽事的男單頭號種子。女單方面，日本隊也出動了不少主力選手，石川佳純、平野美宇、佐藤瞳、加藤美優、早田希娜包攬了女單前五號種子。

中國隊此番派出26人軍團，男隊由徐瑛彬、薛飛、徐海東、向鵬、劉夜泊、周愷、曹巍、賽林威、曾蓓勛、趙釗彥、任浩、袁勵岑、梁國棟、曹彥濤14人組成，其中僅有徐瑛彬、薛飛2位種子選手，其餘12位選手均需從資格賽打起。車曉曦、李佳燚、劉曦、張薔、王曉彤、齊菲、楊藝韻、楊蕙菁、張繽月、郭雨涵、黃凡真、冷雨桐12位選手組成了本次中國女隊的陣容，其中僅有張薔1位種子選手。

除了男單、女單，本次北美洲挑戰賽+還將進行男女雙打、混合雙打、U21男女單打的比拼，資格賽將於12月4日-6日進行，12月5日-8日將展開正賽的較量。

2019國際乒聯世界巡迴賽總決賽

12月12日-15日，2019國際乒聯世界巡迴賽總決賽即將在中國鄭州隆重舉行。經過全年12站分站賽的激烈角逐，年終總決賽匯聚了本年度在公開賽賽場表現最優異的選手，比賽必將相當精彩。目前，全部參賽名單已經產生。

根據規則，本年度需參加至少5站巡迴賽（其中至少1站白金賽事），且積分位列前16位選手，方可獲得總決賽單打資格。男單方面，此番的東道主中國隊共有7位選手入圍16強，其中王楚欽於此前確認退賽，由排在積分榜第17位的黃鎮廷補位出戰。男單完整名單如下：

許昕、樊振東、林高遠、馬龍、張本智和、林昀儒、梁靖崑、卡爾德拉諾、波爾、奧恰洛夫、弗朗西斯卡、趙子豪、M·法爾克、鄭榮植、水谷隼、黃鎮廷

女單方面，中國隊相當強勢，總計有10位選手躋身積分榜前16位，獲得總決賽參賽資格。孫穎莎領銜榜首，完整名單如下:

孫穎莎、伊藤美誠、陳夢、王曼昱、陳幸同、劉詩雯、丁寧、王藝迪、朱雨玲。平野美宇、石川佳純、馮天薇、何卓佳、鄭怡靜、佐藤瞳、錢天一。

雙打方面，各組合本賽季需參加至少4站分站賽，積分排名前8位的組合將晉級總決賽。

男雙參賽名單：

鄭榮植/李尚洙、樊振東/許昕、梁靖崑/林高遠、波爾/弗朗西斯卡、何鈞傑/黃鎮廷、杜達/邱黨、廖振珽/林昀儒、陳建安/庄智淵

女雙參賽名單：

木原美悠/長崎美柚、田志希/梁夏銀、孫穎莎/王曼昱、陳思羽/鄭先知、平野美宇/芝田沙季、巴拉佐娃/馬特洛娃、杜凱琹/李皓晴、吳穎嵐/蘇慧音

混雙賽事的入圍規則稍有不同，由排在積分榜前8位的搭檔獲得參賽資格，且每個協會只有一個參賽名額。由此，獲得總決賽混雙參賽資格的組合如下：

黃鎮廷/杜凱琹、林昀儒/鄭怡靜、許昕/劉詩雯、水谷隼/伊藤美誠、趙大成/申裕斌、皮斯特耶/巴拉佐娃、弗洛雷/加斯尼爾、斯祖迪/佩格爾

此外，值得關注的是，在本次總決賽中奪得混雙前4名的組合將直接獲得2020東京奧運參賽資格，爭奪必將空前激烈。

12月12日，2019國際乒聯總決賽就將正式於中國鄭州打響。目前票務工作已經啟動，觀眾可在線上登錄大麥網了解購票詳情，或通過大麥APP、大麥網天貓旗艦店、大麥網手機端M站、大麥網埠購買，也可以在賽事當天，於河南省鄭州市奧體中心現場購票。
