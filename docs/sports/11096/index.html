22+10+6！恩比德錘爆掘金，約基奇沮喪離場，這就是狂野的東部

2019-12-11 133條評論 1,358次閱讀 277人點贊

在今天結束的一場比賽中，76人主場以97-92戰勝掘金，送給對手三連敗。本場比賽76人五星恩比德+西蒙斯+哈里斯+理查德森+霍福德全部出戰。但今天扛起球隊的是恩比德和哈里斯。全場比賽恩比德18中8砍下22分10籃板6助攻的數據，哈里斯17中7拿到20分7籃板4助攻。兩人合力砍下了42分16籃板10助攻的數據。

其他人西蒙斯拿到了7分9板7助，理查德森和霍福德合砍14分。

另外今天掘金很不幸，在首節當家控衛賈馬爾-默里臀部受傷，直接返回更衣室。之後無法出戰，掘金還是和76人打到了最後也是非常不容易。今天約基奇表現中規中矩，13中7得到15分11助攻7籃板的數據。

約基奇整體來說，串聯是沒有問題的，就是在進攻端不夠有侵略性，就在末節最後時刻，約基奇發狠了，他持球一個衝擊拿下了2分，但恩比德之後2罰全中殺死懸念。

雙方賽後還是比較友好的，恩比德和約基奇擁抱致敬，兩人還笑成一團。雖然恩比德經常和人吵架，比如之前就和唐斯打架了，但和約基奇的關係還挺好。

今天比賽過程雙方實力在伯仲之間，上半場76人以55-50領先5分，互有攻防。下半場掘金開始發力，開局就是12-2反超76人。但哈里斯和霍福德連續發力，恩比德也開火，三節76人仍然以83-76繼續領先7分。

末節巴頓如有神助，他不斷得分帶隊打出8-0迫近分差。但關鍵時刻，霍福德跳投命中，76人取得了5分領先優勢。約基奇持球突破搖搖晃晃的打進2分。將分差追到3分，隨後立刻對恩比德犯規，大帝2罰全中，巴頓最後一攻打鐵，掘金無奈吞下苦果。

全場比賽掘金方面除了約基奇外，巴頓26分7板7助，哈里斯14分。

在過去幾年NBA是西強東弱，但本賽季真的有一種狂野東部的感覺。而且目前西部和東部的交手，東部是佔盡優勢的。特別是之前西部的奪冠大熱門快船被雄鹿屠殺，倫納德和喬治慘敗字母哥。目前西部唯一可以和東部抗衡的就是湖人。湖人戰績21勝3負和雄鹿一致。但其他球隊西部不如東部，

東部勝率70%以上的球隊有四支，分別是雄鹿+凱爾特人+熱火+76人，而西部只有快船和湖人。東部前八的魔術勝率為11勝12負，第七的籃網為13勝10負。而西部第八的太陽為11勝12負，第七的雷霆也是11勝12負。

換句話來說，在東部想進季後賽，也沒有那麼容易。反而是西部進季後賽更加容易了，西部今年強隊是真不多。而東部是遍地開花，狂野的西部已經過去了，現在是狂野的東部，誰出來都要脫一層皮。
