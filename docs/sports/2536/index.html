又一個帕森斯？年薪近3千萬場均11分，昔日神鋒坐實高薪低能

說到高薪低能，大家第一時間會想到誰？一定是有「高富帥」之稱的錢德勒·帕森斯了。2016年夏天，帕森斯以4年9000萬美元加盟灰熊，打那開始，關於錢德勒的新聞只有兩個主題，一是受傷，二是約會。在灰熊度過醉生夢死的三年後，帕森斯於今年7月被交易至老鷹，本賽季截止目前，他出場3次共得11分，年薪高達2510萬美元。如果說帕森斯是高薪低能的第一代表，第二當屬效力於公牛的奧托·波特。

27歲的波特，身高2.03米，臂展達到2.15米，場上司職小前鋒。2013年選秀大會上，奇才用首輪第三順位選中波特，球隊當時希望波特能成長為沃爾、比爾身邊的第三巨頭，菜鳥賽季，波特的場均上場時間只有8.6分鐘，僅得2.1分1.5籃板0.3助攻。隨後兩年，隨著出場時間上升，波特逐漸嶄露頭角，場均得分逐年遞增，分別為6.0分、11.6分。到16-17賽季，他在場均32.5分鐘里，可以得到13.4分6.4籃板2.0助攻，三分球命中率達到43.4%。

2017年夏天，成為受限制性自由球員的波特，得到了多支球隊的青睞，籃網給他開出4年1.06億美元的資質報價，奇才最終選擇匹配。接下來一個賽季，波特在上場時間沒有減少的情況下，個人數據全面下滑，場均得到12.6分5.6籃板2.0籃板。奇才當初續約波特，是看中了他的潛力，可拿到大合同後，他卻止步不前。上賽季中期，奇才和公牛達成交易，送出波特，得到波蒂斯和賈巴里·帕克。

來到公牛後，波特一度打出高水準，15場比賽場均得到17.5分5.5籃板2.7助攻，投進1.9記三命中率達到48.8%。可到了本賽季，波特的表現又一次縮水，首發出戰9場，僅得11.2分3.4籃板1.6助攻。

和帕森斯不同，波特的高薪低能，不是因為傷病影響，本人也沒有自甘墮落，他只是沒有兌現天賦，或者說奇才當時看走了眼。按波特的能力，以及現在的表現，拿一份千萬年薪挺合適。而現在，他處於頂薪合同第三年，薪水達到2725萬美元，下賽季增加到2849萬美元。

對一名職業球員來，上場比賽、努力打球是指責，在此基礎上，追求更多個人收益，是可以理解的。從這個角度講，帕森斯不能算合格的職業球員，他在養傷期間，並沒有積極恢復訓練，而是沉迷享樂。至於波特，有球隊給他那麼多錢，不掙才是傻子，至於場上表現，努力做到最好就行。
