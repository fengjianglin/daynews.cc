曼聯青訓再結碩果！熟悉的曼聯要回來了嗎？

2019-12-14 303條評論 2,231次閱讀 480人點贊

本賽季，曼聯的表現起起伏伏，第一輪4：0大勝切爾西後，球隊的表現就不斷下滑，而在前不久接連被英超保級隊逼平，更是讓索爾斯克亞險些陷入了絕境，而經過一段時間調整，曼聯接連戰勝了熱刺和曼城兩個強敵，這樣精彩的表現也讓索爾斯克亞挽回了「民心」和高層的信任，曼聯不僅重回聯賽第5的位置，還打進了英聯杯8強，並在歐聯杯的賽事中提前一輪出線，多線奏捷，也讓曼聯上下恢復了信心。

在接連贏下兩場硬仗後，索爾斯克亞也對球隊首發進行了些許輪換，4231的陣型中，小將圖安澤貝搭檔馬奎爾出任中衛，小將威廉姆斯和老將阿什利揚分居左右，中場方面詹姆斯-迦納搭檔傷愈歸隊的馬蒂奇組成雙後腰，前場格林伍德、馬塔、佩雷拉一字排開，馬夏爾首發出戰，從首發陣容來看，索爾斯克亞派出了老帶新的組合，有4名曼聯青訓獲得了首發機會，而佩雷拉意外地出現在了前場邊路。

比賽開始後，與曼聯力爭小組頭名出線的阿爾克馬爾並不保守，憑藉積極的跑動和利用球場的寬度轉移，讓曼聯開場的高位逼搶失去了效果，雖然失去了場上的主動權，但曼聯保持了充足的耐心與馬爾克馬爾進行周旋，雙方在上半場都沒有創造太多機會。

隨著下半場的到來，上半場跑動更多的阿爾克馬爾體能有所下滑，而曼聯在索爾斯克亞的安排下，加強了前場的推進和肋部的攻擊，同時在防守中，通過持續的高位逼搶，壓縮阿爾克馬爾隊的邊路進攻，讓對手無法從容的通過邊路轉移進行快攻。

這一戰術很快奏效，第53分鐘佩雷拉成功過人，傳給處於肋部空當的馬蒂奇，馬蒂奇在禁區前沿直塞傳給馬塔，曼聯成功在肋部打出精彩配合，而馬塔的橫傳也由接到球的阿什利揚一腳爆射打破場上的僵局。

破僵後，曼聯的攻勢如潮，而阿爾克馬爾更加被動，第57分鐘，曼聯的高位逼搶奏效，曼聯前場搶斷，格林伍德禁區外爆射破門，僅僅3分鐘後，表現出色的格林伍德再次突如禁區製造點球，馬塔順利的罰進了鎖定勝局的點球，又是3分鐘後，曼聯再一次擊穿阿爾克馬爾的肋部空間，格林伍德在右路勁射破門，曼聯11分鐘打進4球，比賽塵埃落定，而製造了2球1點的紅魔小將格林伍德，無疑是本場比賽的最佳球員，格林伍德有著精湛的射術，而且左右腳十分均衡，這也是他連續3個歐聯主場代表曼聯取得進球。

此外球隊中場佩雷拉和馬塔也在比賽中表現不俗，一改在英超聯賽中的頹勢，這很大程度上與兩人的對抗能力相關，在強調對抗的英超聯賽中，兩名球員的身體劣勢將被無限放大，但在歐聯賽場中，身體的對抗不再是主旋律，這也給了佩雷拉和馬塔更多的發揮空間。曼聯隊在近期表現出的進球能力並不弱，但本場比賽更多是通過快速進攻創造的進球，如何在聯賽中破解鐵桶陣，顯然還會困擾著曼聯，畢竟「劫富濟貧」的行為，很難讓球隊進入聯賽前四。

取得三連勝後，索爾斯克亞的帥位無疑再一次穩固了下來，賽後索爾斯克亞也表示球隊需要補充一些位置，提升球隊整體的競爭力，不過伴隨著越來越多曼聯青訓小將的出頭，曼聯真的取得了很大的進步，這場比賽後，似乎也讓人看到了曾經那支無所不能的紅魔。
