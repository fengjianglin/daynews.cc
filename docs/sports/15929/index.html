梅西高居歐洲球員最新排行榜第一，C羅僅排名30開外

2019-12-14 1,257條評論 2,690次閱讀 1207人點贊

在最新歐洲球員排名榜上，梅西大幅領先C羅排名第一，利物浦和萊斯特後衛在防守席位球員中處於領先位置。

梅西本月初獲得了第六座金球獎，根據CIES足球天文台的最新數據，阿根廷人再次被認為世界最佳足球運動員。

InStat績效指數是一種獨家工具，可以客觀地評估球員的表現，對歐洲各地的球星彼此進行排名。

毫無意外，梅西以401的分數排名高居首位，而得分332的尤文圖斯前鋒C羅僅僅在歐洲排名第31名。

荷蘭阿賈克斯的哈基姆·齊耶齊以377分的得分位居第二，力壓第三名的巴黎聖日耳曼明星姆巴佩。

保羅·波格巴是曼聯隊中表現最出色的球星，但僅以329的得分排在第39位。

獨特的CIES工具還可以根據聯賽表現、場上位置和年齡來給出足球運動員的得分。

范·迪克和安德魯·羅伯森分別排在中後衛和邊後衛類別之首，葡萄牙後衛里卡多·佩雷拉也在榜單中，他的萊斯特隊友本·奇爾威爾的得分與C羅的相同。

以下是排名前十的球員及其得分：

10.安德魯·羅伯遜（利物浦）353

9.德布魯因（曼徹斯特城）354

8. 伊莫比萊（拉齊奧）355

7.蒂莫·沃納（RB萊比錫）357

6.內馬爾（巴黎聖日耳曼）358

5.迪瑪利亞（巴黎聖日耳曼）360

4. 馬赫雷斯（曼徹斯特城） 362

3. 姆巴佩（巴黎聖日耳曼） 366

2. 哈基姆·齊耶奇（阿賈克斯） 377

1.梅西（巴塞羅那） 401
