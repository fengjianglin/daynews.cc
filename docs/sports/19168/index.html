梅西C羅抽中保姆簽！閉著眼踢都晉級 絕代雙驕預定決賽見？

2019-12-16 1,438條評論 3,208次閱讀 954人點贊

歐冠淘汰賽的抽籤已在北京時間12月16日晚全部結束。自從C羅離開皇馬前往尤文後，絕代雙驕的對決便再也沒有出現過，2018年世界盃阿根廷和葡萄牙同時止步16強，而在歐冠16強的比賽中，都身為小組第一尤文和巴薩雖然也沒有能夠相遇，但我們看看兩家豪門在淘汰賽中的對手，絕代雙驕是否都能挺進8強，在日後的比賽中相遇呢？

尤文圖斯VS里昂

對於斑馬軍團來說，抽中里昂可謂是一個上上籤，畢竟他們成功避開了皇馬等豪門。在絕對實力上，坐擁C羅、迪巴拉、德里赫特等巨星的尤文圖斯顯然要更勝一籌。而本就實力較弱的里昂又迎來了一不利消息，法甲球隊的隊內頭號射手德佩在與雷恩的比賽中受傷，賽後被診斷為左膝十字韌帶撕裂，而中場大將阿德萊德也詭異般地在同一場比賽中遭遇十字韌帶的傷病。

如此一來，缺少了兩大核心球員的里昂要想晉級可謂是難上加難。而從過往戰績來看，尤文對陣里昂也是保持著3勝1平的不敗戰績。C羅的球隊只要正常發揮，晉級沒有太大懸念。

巴塞羅那VS那不勒斯

從巴薩的角度來看，抽中那不勒斯也算是一個好籤。但那不勒斯在本賽季歐冠中一直都扮演著「豪門終結者」的角色，他們在小組賽中在主場2-0戰勝利物浦，客場也是1-1逼平紅軍。要知道，這是利物浦本賽季唯一一次輸球，那不勒斯絕不是一個好欺負的對手。

而對於一支擁有梅西的俱樂部來說，巴薩的目標自然是要努力爭取最後的冠軍。但值得一提的是，今夏加盟那不勒斯的馬諾拉斯曾經用一粒頭槌進球幫助前東家羅馬完成大逆轉擊敗巴薩。面對心魔，這是巴薩證明自己的一個好機會，上賽季他們在客場被利物浦連扳4球止步半決賽，與一支能夠擊敗利物浦的球隊對決，即使在實力上明顯佔優，紅藍軍團依然不可輕敵。

從過往戰績上看，這次對決時巴薩和那不勒斯在歐戰賽事的首次交手，但在今夏的熱身賽中，巴薩分別以2-1和4-0的比分擊敗了那不勒斯，還是具有一定心理上的優勢。而巴薩最大的心理優勢還要屬陣中有梅西這樣的球員，相信梅西會率領球隊力爭取勝。只有尤文和巴薩都晉級，C羅和梅西才有在日後相遇的可能性。
