單局28-26絕殺！郎平接班人率全替補，橫掃女排豪門4大世界冠軍

2019-12-08 0條評論 24次閱讀 0人點贊

摘要

北京時間12月8日，女排世界俱樂部杯賽7~8名的決賽展開爭奪，這也是兩支中國球隊天津女排和廣東恆大女排本次比賽當中的最後一場比賽了，最終廣東恆大女排以3比0橫掃天津女排，拿到了第7名，而豪言爭冠的天津女排最終慘遭墊底，排名倒數第一。

北京時間12月8日，女排世界俱樂部杯賽7~8名的決賽展開爭奪，這也是兩支中國球隊天津女排和廣東恆大女排本次比賽當中的最後一場比賽了，最終廣東恆大女排以3比0橫掃天津女排，拿到了第7名，而豪言爭冠的天津女排最終慘遭墊底，排名倒數第一。

更讓人無語的是，在比賽當中，廣東恆大女排的歐洲三冠王主教練方岩，意識到這場國內球隊之間的較量，對於已經成名的兩位世界冠軍林莉和鄭益昕來說沒有太大的考驗價值了，所以在排兵布陣方面，方言直接是把這兩位世界冠軍雪藏了，而用了更為年輕的替補陣容出戰，甚至是隊長栗垚都只是在比賽裡面擔任替補，兩大外援科舍列娃和拉巴德捷耶娃只上了一個人，所以這就已經充分的說明了，這場比賽的恆大隊幾乎就是替補陣容，最多只能算是半主力。

結果正是這樣一個替補陣容，卻戰勝了兵強馬壯的天津隊，天津隊今天雖然說朱婷沒有出戰，但是在首發陣容方面，外援胡克爾登場，奧運冠軍袁心玥和19歲的世界冠軍李盈瑩，世界冠軍二傳手姚迪全部披掛上陣，再加上外援也首發，所以天津女排陣中，依然擁有了王媛媛，李盈瑩，姚迪和袁心玥四大世界冠軍，然而這樣一個陣容出戰廣東恆大女排，最終卻在第1局的比賽當中，沒有能夠把握住關鍵分，反而被對手以28比26這樣一個比分絕殺，真的是讓人相當無語。

隨後後兩局的比賽當中，天津女排的主教練陳友泉也開始走馬換將，在第2局的比賽當中，李盈瑩被換下場再沒有出過場，袁心玥打了兩局的比賽，第3局的比賽沒有出場，王媛媛和姚迪則是打滿了全場，然而即便如此，她們還是慘敗給了對手，最後兩局的比分為20比25和21比25，這樣的結果已經充分說明，這賽季的排超聯賽當中，曾經兩勝廣東恆大女排的天津隊是有水分的，不是因為有朱婷，可能那兩場比賽她們都未必能贏。而方岩則用這場勝利，證明自己確實是國家隊女排帥位上，郎平最合適的接班人，陳友泉則被打回原形。

這樣的結果也足以讓天津女排，看清楚自身和對手巨大的實力差距，天津女排主教練陳友泉在賽後也是認栽了，他公開道歉說這次世界盃賽，確實能夠看到天津隊和其她球隊之間存在的巨大差距，等到回到天津之後，球隊會刻苦的訓練，通過訓練將這種差距彌補上。
