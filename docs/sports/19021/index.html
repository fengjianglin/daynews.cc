奇葩：把巴西球員歸化，再送回巴西，這波操作太逗了

2019-12-18 126條評論 3,834次閱讀 210人點贊

2019賽季的中超已經結束了半個月，目前各傢俱樂部都已經開始進行下賽季的球員更替工作。雖然，目前很多球隊的人員流動情況還處於操作之中，並沒有太多新消息流出，但最近恆大俱樂部的一則轉會消息還是引發了媒體和球迷的關注。

近日，據巴西體育媒體人在網上的爆料，巴甲聯賽的聖保羅隊目前有意從中超球隊廣州恆大引進他們正在外租的前鋒阿蘭。據悉，聖保羅俱樂部已經與恆大方面對這一轉會事宜進行了溝通。雖然，阿蘭將是下賽季廣州恆大全華班計劃中的一個重要球員，但由於足協下賽季針對歸化球員上場人數的限制政策，使得已經年過30的阿蘭在明年的恆大很難獲得上場機會，為了不浪費資源，恆大需要繼續將其租借出去。

雖然已經離開恆大近一年，但上個賽季的阿蘭在代表天津天海出戰的中超聯賽中，表現的還是相當不錯的。2019賽季，阿蘭共在聯賽中出場27次，打入9球並有3次助攻。雖然曾與天海俱樂部鬧出過一些不愉快，但他依然可以說是本賽季球隊的保級功臣之一。

據此前國內媒體的報道，阿蘭已經完成了歸化手續，加入中國籍成為了一名中國球員。如果阿蘭能夠在下賽季成功的回歸巴甲聯賽，那麼他將創造一個中國足壇的新紀錄——以中國歸化球員的身份加盟海外俱樂部的第一人。這是否也算是促進中國球員留洋深造的一種曲線救國的方式呢？

據了解，阿蘭與恆大的合同將在明年初到期。眾所周知，恆大為了說服阿蘭、艾克森、洛國富等球員進行歸化，已經在他們身上投入了大筆的資金，不可能輕易將他們放走。因此，恆大需要儘快和阿蘭完成續約，再討論對他進行交易的可能性。也許對於不差錢的阿蘭來說，回歸巴西聯賽踢球是他現在最好的選擇。
