巴媒曝高拉特親承已完成歸化！國足沖2022最強助力，誰給他讓位？

2019-12-20 1,332條評論 2,486次閱讀 1409人點贊

北京時間12月19日，巴媒爆料道：「高拉特親承已正式完成歸化入籍，現在已經是中國人，期盼代表中國隊衝擊2022年卡達世界盃。」 若高拉特正式完成歸化入籍，對於國足絕對是巨大利好消息。因為就高拉特的技術特點來講，其是國足進攻核心的不二人選。

事實上，高拉特、費爾南多等人將歸化入籍代表國足參加世預賽，早已不是什麼秘密。時至今時，巴西媒體爆料高拉特親承完成歸化入籍，無疑給眼下正處於低迷期的中國足球注入了一支強心針。

畢竟，從此前的世預賽亞洲區40強賽前半程比賽，到剛剛結束的東亞杯比賽，國足在賽場上的發揮均差強人意。尤其是國足在1-2輸給許敘利亞隊，里皮引咎辭職，球隊力爭小組頭名出線的前景也極其渺茫。而國足之所以發揮低迷，跟缺少進攻核心有直接關係。

其中里皮執掌國足期間，中場啟用池忠國、蒿俊閔與吳曦，但三人均不是傳射兼備的組織型攻擊手。而這類球員，正是國足陣容最缺的。如若高拉特可以代表國足參加世預賽，以高拉特的技術能力特點來講，其有能力將國足的進攻梳理得井井有條，讓國足的進攻質量得到本質性提升。

當然，除了高拉特外，費爾南多、阿蘭等人的歸化入籍應該也不遠了。同時，河南建業中場核心伊沃近期不斷表示，願意歸化入籍，代表國足征戰世預賽。所以，一旦有更多攻擊型歸化國腳進入國足，那麼國足的進攻殺傷力必然會再度得到本質性提升。所以綜合上述種種，雖然國足今年運勢不佳，且目前處於無主帥狀態。但高拉特等歸化國腳相繼到來，將成為國足衝擊12強賽與衝擊卡達世界盃決賽圈的最重要籌碼。
