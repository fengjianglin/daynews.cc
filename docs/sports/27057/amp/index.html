不要因現在的排名膨脹，3比1的失利或只是開始，誰進歐冠還不一定

天天要聞
2019-12-22

著名媒體The Athletic透露，曼聯心儀已久的狐狸城中場麥迪遜即將在冬窗與球隊續約。儘管曼聯一直想要引進他來提升自己前腰位置的進攻組織能力，但麥迪遜本人興趣不大，原因是他覺得曼聯沒有歐冠資格。

麥迪遜所在的萊斯特城隊目前雖排在聯賽第二，但繼上一場1比1被諾維奇終結了聯賽連勝之後，本輪又被曼城3比1擊敗，同時曼城也將兩隊的積分差追至1分。縱觀整場比賽，狐狸城被曼城全場壓制，除了偶爾幾次反擊外，基本只能龜縮後場防守。從技術統計上來看，關鍵數據都落後曼城。

分析一下目前的積分，之所以現在能排在第二，得益於對陣中下游球隊時自己穩定的戰績，以及目前BIG6中阿森納和熱刺之前隊內的混亂。隨著熱刺阿森納換帥完成，以及聯賽逐漸步入下半程，經過冬窗調整後，狐狸城還能否穩定在歐冠行列內，將被打上個大問號。本場輸曼城或許還只是開始，下輪對手就是目前的榜首利物浦，而更可怕的是如果再輸給利物浦，一是有可能將連勝建立起的自信心徹底打垮，二是讓其他球隊學會了如何擊敗自己的訣竅，比如說掐斷自己的後場出球線路。一旦這樣的局面產生，成績下滑就是不可避免的了。

再來說回麥迪遜。之前對戰中下游球隊或許不夠明顯，本場對陣曼城，自己球隊後場出球能力弱的缺點暴露無餘。曼城直接在前場進行壓迫，單後腰配置的狐狸城很難將球運轉到中前場。麥迪遜只能回縮到自己的半場接球，幫助出球。他全場只有42腳傳球，1次威脅傳球，可以說根本沒有表現機會。打反擊只有瓦爾迪一個靠譜的點，可選擇性太少，沒有可操作的空間。

被他瞧不上的曼聯，前場三人組拉什福德、馬夏爾、詹姆斯是個頂個的快，都是反擊的一把好手，進攻點多，好選擇，就缺一個能傳好球的指揮者。而麥克托米奈和弗雷德組成的雙後腰，攻守平衡，不用分心回撤幫助處理球。從戰術層面講，正是麥迪遜充分發揮自己組織進攻的合適球隊。而從球隊實力來說，曼聯目前是英超潛力最佳的球隊，球員年齡結構輕，可以一起打拚多年。雄厚的財力支持，可根據需要對相應的位置進行補充。據悉挪威中鋒哈蘭德已經在來的路上。來到這樣的球隊，才是對各項冠軍發起衝擊的最佳選擇。

現在萊斯特城排名是比曼聯高，按現在的排名曼聯只能去歐聯，但到賽季末呢？誰進誰不能進歐冠還不一定呢。真香定理怕是要繞不開了。
