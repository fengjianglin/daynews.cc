CBA14輪綜述：吉林再斬新疆，北京反絕殺深圳，山東進前九

2019-12-09 0條評論 21次閱讀 0人點贊

摘要

12月8日，CBA常規賽第14輪結束所有爭奪，有著“巨人殺手”之稱的吉林隊主場再次擊敗周琦領銜的新疆隊，北京首鋼隊則依靠裁判最後時刻判罰的三分犯規罰球反絕殺深圳隊，山西隊五連勝之後勝場只落後新疆隊1場；江蘇隊擊敗八一隊之後，勝場達到50%，躋身前12名季後賽資格線內。

12月8日，CBA常規賽第14輪結束所有爭奪，有著「巨人殺手」之稱的吉林隊主場再次擊敗周琦領銜的新疆隊，北京首鋼隊則依靠裁判最後時刻判罰的三分犯規罰球反絕殺深圳隊，山西隊五連勝之後勝場只落後新疆隊1場；江蘇隊擊敗八一隊之後，勝場達到50%，躋身前12名季後賽資格線內。

廣東隊擊敗青島隊之後，獲得10連勝領跑積分榜，領先場次擴大到2場。新疆隊客場不敵吉林隊，本賽季第二次不敵對手，新疆隊11勝3負排在第二位。不過，他們的領先優勢並不大，山西隊本輪擊敗四川隊之後，取得五連勝，僅落後新疆隊1個勝場且擁有少賽一場優勢。浙江廣廈取勝之後，守住第四名的位置。北京首鋼本輪為去世的吉喆而戰，依靠林書豪最後一秒的三罰全中，球隊實現反絕殺擊敗深圳隊，北京隊達到10勝4負，暫列第五位。遼寧隊本輪取勝之後，戰績變為9勝5負，暫時排在第六位。巨人殺手吉林隊則排在第七位。少賽一場的浙江稠州銀行排在第八位。山東西王男籃兩連勝之後，悄然間進入第九名。輸給山東的北控隊則排在第十名。蘇州肯帝亞本輪贏球之後，排在第11名；南京同曦則排在第12名。

不過，前12名位置並不穩固。本輪遺憾輸球的深圳隊同樣為7勝7負，因相互戰績處於劣勢，排在了第13位。在他們身後，還有6勝8負的上海久事、青島雙星、時代中國廣州等。
