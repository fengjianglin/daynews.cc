同樣是一套陣容打天下，郎導獨孤求敗，這個教練卻一勝難求

2019-12-11 729條評論 2,155次閱讀 1242人點贊

現在中國女排的最強陣容是什麼？那當然是七仙女陣容了，這是女排主教練郎平指導親手打造出的強力陣容。就是依靠這樣一套陣容，郎導率領中國女排四處征戰，越戰越強，越戰越勇，直至在世界盃上所向披靡。可以說，目前這套陣容還不曾有過敗績。就是依靠這套幾乎不變的陣容，中國女排能夠零封美國，挫敗巴西，擊垮俄羅斯，橫掃日本……，讓自己傲視群雄，贏得寂寞，獨孤求敗。

然而在中國還有一個教練也是以一套陣容征戰四方，只不過征戰的格局不同。郎導征戰的格局是國際賽事，這名教練征戰的目標是國內聯賽。在國內聯賽中，這名教練可以說是絕無僅有的不知變通的一員，以一套陣容四處硬抗，硬是在精疲力竭中，越戰越衰，幾乎輸掉了所有重要的比賽，讓球隊在一場場憋屈的比賽中，離目標越來越遠。這名教練就是江蘇女排的主教練蔡斌。

郎導雖然主要使用一套陣容，但這個陣容卻非常平衡，無論是進攻還是防守都非常犀利和穩固，在各個位置上都有出類拔萃的球員鎮守，使得整個球隊的運轉銜接非常流暢，各個進攻點相互支援，呈現出四方出擊八面開花的效果。之所以能取得這樣的效果，源於球員在國家隊集訓時，郎導悉心教導。球員中，無論是主攻的朱婷張常寧還是副攻顏妮袁心玥，無論主力還是替補，都能取得有目共睹的進步。所以郎導能夠率領這支球隊取得一次又一次的勝利。

其實，郎導並非只會一套陣容打天下，而是有很多備選方案，比如世聯賽時頻繁換人連環出擊，擊垮埃格努，擊敗了不可一世的義大利就是變通的傑作之一。之所以用一套陣容打天下，是因為這套陣容已經修鍊到極致了，就如一把萬能鑰匙，遇到什麼樣的鎖都可以打開一樣，在沒有出現特別困惑的問題的情況下，這套陣容完全夠用，而且完全實用。所以郎導選擇這套已經打磨好的陣容，不會做出輕易的改變，完全是智慧的表現。

反觀江蘇女排主教練蔡斌，打理江蘇女排也有些時日，可是江蘇女排的整體結構就像一個瘸子一樣長短不齊。江蘇女排擁有聯賽中名列前茅的邊攻，卻也擁有幾乎可以忽略不計的副攻。然而就是這幾名副攻一直佔據著江蘇女排的主要位置，蔡斌教練似乎對此非常欣賞，沒有一點要做出改變的意思，或者說，在自己的能力範圍內實在缺乏改變的手段。蔡斌教練就是守著這樣一套陣容，在排超諸強中穿梭，處處被打臉卻不知道臉為什麼腫了。

江蘇女排並非沒有後備力量，只是一個抱殘守缺的人缺乏一種挖據新人的勇氣。當天津女排在已經擁有朱婷的情況下，依然將自己年輕的攻手王藝竹提拔到一線隊伍進行強化鍛煉。而江蘇女排在攻手並不充裕的情況下，硬是將自己已經脫穎而出的攻手吳夢潔死死地卡在青年隊。老的副攻形同虛設，新人被埋沒，蔡斌教練真的是既荒廢了訓練，又缺乏伯樂的眼光。

與天津一戰，江蘇女排以逸待勞，佔據主場優勢，而天津女排舟車勞頓，帶著世俱杯上身體和身心的疲憊，再加上朱婷受傷，遠赴客場。這本是千載難逢的機會，可是江蘇女排在良好的開局下，依然輸掉比賽。畸形的陣容，不能給對手持續的壓力，面對變化缺乏應對手段，這是江蘇女排輸球的原因。誰之過與？主教練蔡斌難辭其咎。

郎導的一套陣容是萬能鑰匙，打開一扇扇通向勝利的大門。蔡教練的一套陣容就像一個沙袋，迎接著來自各方的拳頭。聯賽關鍵比賽中，江蘇女排已經連輸兩場，四強目標堪憂。本來江蘇女排是很有實力的，只是陣容整合得實在太滑稽。守著一堆黃金白銀，最後卻只打造出一條青銅項鏈，還被人誇讚為能工巧匠，真讓人無語了。
