廣東宏遠國手痴迷遊戲影響狀態！杜鋒曾為此找家長 CBA歷史少見

2019-12-22 149條評論 2,287次閱讀 1019人點贊

就在這兩天，有關八一男籃隊長雷蒙頂撞王治郅成為了CBA焦點，其實對於目前CBA聯賽來說，已經進入了90後年輕人主打時代，而這些年輕人都有著自己鮮明的個性，他們不同於以往的老一輩球員，稍微有一些不舒服就會直接的表達出來，而目前大部分主教練又都是老派的作風，互相之間肯定會有一些衝突，例如王治郅之前就公開表達過禁止球員在訓練場使用手機！而根據廣東媒體人關辛爆料，廣東宏遠的國手胡明軒之前就因為痴迷遊戲而被找過家長，而找家長這個舉動在CBA聯賽中還是非常少見的！

相比於過去來說，目前年輕球員的私生活空間要自由了很多，畢竟以前球隊晚上甚至會要求上交手機或者電腦，如今這樣的要求已經從CBA聯賽中消失，然而由於電游的風行，一些球員對此也是非常痴迷，如果有印象的話，在CBA官方紀錄片中杜鋒就曾經對此批評過胡明軒，說他在弄裝備的時候有精神，一到比賽的時候卻失去了狀態！現在回想起來胡明軒上賽季的確有過一段時間狀態非常低迷，顯然杜鋒所說的就是那段時間，所以杜鋒才有了找家長的一幕出現，要知道在CBA聯賽中，幾乎沒有出現過找家長這樣尷尬的情況出現！

另外痴迷遊戲這樣情況也是CBA聯賽中比較普遍的，由於年輕球員平時的娛樂活動越來越多，他們對於籃球的專註度也比老前輩差了很多，例如有著中國籃壇最嚴厲主帥之稱的范斌就曾經表示，現在的年輕人都想著如何去玩，如何去放鬆，怎麼去花錢，買東西，可卻失去了對於籃球的熱愛，他們應該先去把球打好，然後再去學會花錢消費！

或許很多球迷現在都無法接受杜鋒、王治郅等主教練對於球員的嚴厲，可他們真的都是為了球員，如果不對於這些球員嚴格管理的話，恐怕他們將會把更多的精力放在球場外，同時也希望我們的年輕球員能夠先把球打好，這才是他們應該做的！
