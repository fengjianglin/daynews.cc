亞洲一哥也難逃！英超恥辱一面，再精彩的比賽和進球也只是遮羞布

2019-12-10 545條評論 2,421次閱讀 494人點贊

在上周六進行的英超第16輪，熱刺主場5-0大勝伯恩利，穆帥上任之後的熱刺一路高歌猛進。孫興慜更是在上半場1v8狂奔80米打入一條龍進球。

另一邊，曼城與曼聯的曼市德比也奉獻了一場精彩的對決，面臨帥位危機的索爾斯克亞帶領著年輕的曼聯連克穆里尼奧的熱刺和瓜迪奧拉的曼城，索帥也給曼聯球迷們吃下了一顆定心丸。

縱使周末的英超依舊精彩，也不乏孫興慜這樣令人瞠目結舌的精彩進球，但在英超賽場卻出現了不和諧的因素-種族歧視，這個永遠逃不掉的話題。

日前，一名13歲的伯恩利小球迷在熱刺對陣伯恩利的比賽中因為涉嫌種族歧視而被驅逐出場，而他攻擊的對象正是「亞洲之光」孫興慜。

隨後，伯恩利俱樂部發表聲明稱：「在12月7日進行的英超聯賽第16輪托特納姆熱刺主場對陣伯恩利的比賽中發生了一起事件，一名13歲的伯恩利球迷被驅逐出球場，原因是他對熱刺球員孫興慜做出了帶有種族主義歧視的行為。」

「與此同時，伯恩利足球俱樂部將聯繫這名少年和他的家人，以實施一項強制教育計劃，並與伯恩利社區足球俱樂部進行合作。」

「伯恩利足球俱樂部重申其對任何形式歧視的零容忍政策。我們將繼續與有關部門合作，幫助消除足球比賽中日益嚴重的種族歧視問題，我們敦促所有的球迷在最嚴格保密的情況下，通過我們所提供的比賽日簡訊服務或應用程序報告任何此類問題，以杜絕相同事件的再度發生。」

伯恩利俱樂部的處理乾脆利落、雷厲風行，也並沒有包庇這個小球迷。

13歲的小男孩能夠做出這樣的行為，同他受到的教育和家庭內的影響是分不開的，13歲的孩子況且都這樣，更不用提其他人了。只能說部分人仍然活在18世紀日不落帝國的驕傲下，或許只是嘴上不說罷了，改變不了的是骨子裡的傲慢。

縱使孫興慜表現的再出色，也征服不了這個小球迷，他還是個孩子，不要放過他。

無獨有偶，周日進行的曼市德比同樣出現了歧視行為。主場的曼城1-2不敵曼聯，輸球之餘更輸了人品。

主場球迷在曼城比分落後時，肆無忌憚的發泄著內心的憤怒，在曼聯獲得角球，曼聯中場弗雷德準備主罰時，場邊的曼城球迷對其大喊種族歧視的口號，更有人投擲打火機等雜物，被投擲物擊中的弗雷德按住自己的頸部，比賽也一度中止。

但在那群人身後，我看到了有一個母親讓自己的孩子背過身去，用自己的方式迴避，這該死的歧視，為何它們要存在這美麗的足球場上。

曼城俱樂部官方宣布，一經核實，球迷將會終身禁止入場。這一點要為曼城的態度點贊。

離事發地點最近的斯特林不知內心有何感想，作為英格蘭國家隊和曼城的一員，斯特林一直致力於反種族歧視，此前更是呼籲用扣9分等重罰來嚴懲。

看到自家球迷的歧視行為，此情此景，斯特林的內心只有他自己能懂。
