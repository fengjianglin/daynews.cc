卡魯索：詹姆斯賽後恢復身體的方法，第一次看到以後，不敢相信

2019-12-10 808條評論 4,632次閱讀 778人點贊

北京時間12月9日，NBA常規賽繼續進行。湖人主場迎戰森林狼。兩支球隊實力差距比較大，一支是季後賽級別的球隊，另一支是總冠軍級別的球隊，所以理論上來講，如果大家都是正常水平發揮的話，本場比賽的懸念應該不大。

比賽開始以後，湖人就打出了非常流暢的進攻，戴維斯在內線翻江倒海，予取予求。濃眉哥無可阻擋，籃下暴扣，中距離跳投，輕鬆突破上籃。全場貢獻50分7籃板6助攻，湖人主場142比125擊敗森林狼。

全場比賽，詹姆斯出場28分鐘，20投12中，得到32分4籃板13助攻；戴維斯出場39分鐘，29投20中，得到50分7籃板6助攻；卡魯索出場30分鐘，11投6中，得到16分4籃板4助攻；霍華德出場22分鐘，5投3中，得到8分6籃板。

卡魯索本場比賽發揮極為出色，是湖人取得勝利的功臣之一，比賽中膝蓋出血依然在場上拼搏。賽後，卡魯索被問到為什麼這麼拼？卡魯索說：「沒有和詹姆斯做隊友之前，我根本不知道他這麼訓練，他是聯盟第一人，這是有道理的。每一場比賽結束以後，詹姆斯都會做五組冷熱浴，大概50分鐘，非常痛苦，這是恢復身體非常有效的方法。當你看到如此成功的球員，還這麼嚴於律己，沒有道理不努力打球！」卡魯索說完以後，本人去查了一下，冷熱浴一組十分鐘，5分鐘熱水5分鐘冰水浸泡身體，哎，詹姆斯太偉大了！這樣的球員，不成功都不可能！
