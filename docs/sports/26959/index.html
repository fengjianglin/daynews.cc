對於梅西與羅納爾多的距離，穆里尼奧直言：如今沒有個人足球文化

2019-12-23 1,533條評論 3,299次閱讀 375人點贊

穆里尼奧說過「羅納爾多的巔峰期比梅西、C羅驚艷」，很多沒有經歷過羅納爾多時代的球迷們可能不理解，因為無論從個人榮譽還是集體榮譽的角度來說，（撇開世界盃）現在的「雙驕」都是遠遠領先於大羅的，為什麼穆帥會認為羅納爾多的巔峰期，比現在的兩位頂尖巨星都驚艷呢？難道穆里尼奧也是「厚古薄今」的人？現在這一切有了答案，英國時間12月22日，提前公布出來的「穆帥掌舵熱刺」的紀錄片片段中，穆里尼奧就講述了梅西與羅納爾多的距離：「他們都說每個時代的頂級球星，但是在我看來，他們兩個不僅僅是不同的球星，還是有距離的球星，梅西沒有羅納爾多的個人足球文化。」

在穆帥看來，在羅納爾多時代雖然巨星雲集，而且大羅巔峰期的長度不如梅西，但是羅納爾多就是在他不算長的巔峰期，創立了屬於自己的個人足球文化。筆者理解的穆里尼奧觀點，其實就是——當我們提到羅納爾多的時候，我們就會想起具有觀賞性質的足球文化，一系列經典的破門以及過人動作，以及一場場經典比賽，這些都是個人足球文化。而在梅西身上，我們提到最多的就是他強大的數據，雖然說梅西代表的「純粹足球」是當今足壇少見的「美麗足球」，但是大家記住了梅西哪個經典破門動作？或者是他的經典過人動作？似乎沒有，即使是專職研究梅西的人，脫口而出的只能是「梅西自由球破門數已經世界第一了」。

這就是梅西在個人足球文化上的缺失了，其實從這點上來說，小羅的個人足球文化也是在於梅西之上的。不過穆里尼奧認為這不是梅西的錯，而是整個時代的錯，梅西與羅納爾多的距離，實際上體現在時代的距離。穆帥表直言：「梅西缺乏的羅納爾多元素，其實在於當今足壇，已經沒有個人足球文化了。」那麼如今的足壇存在著哪種文化？那就是體系文化。穆里尼奧暫時無法確認，個人足球文化與體系文化之間有沒有衝突，但是他認為兩者並存的可能性很小：「既然是以教練的體系為主了，那麼球員的個人特色（不是個人能力），就很難發揮了，他們需要為體系犧牲掉很多個人的足球元素。」

看到穆里尼奧暢談「梅西與羅納爾多的距離」，筆者倒是想起了一種說法，早在傳控足球時代初期的時候，就有人說過：這是一個戰術體系主導的時代，球員們漸漸都會變成體系球員，像羅納爾多、馬拉多納那樣在任何體系下都能夠踢出跑兩足球的球星，今後可能沒有了。十年之後再來看看這個觀點，還真的就是這麼一回事！「體系球員」這個辭彙，就是在傳控足球時代開始之後，才盛行的。越來越多的球員被打上了體系球員的標籤，他們在比賽中發揮好與不好，更多取決於他們身處哪個戰術體系，而不是個人能力強不強。

其實當今足壇發展到體系時代，也是足球專業化發展的必然結果，將來的世界足壇，戰術體系還會佔據更大的主導作用。羅納爾多時代，擁有個人足球文化的球星不少，哪怕有些並不是很偉大，再往前數，馬拉多納時代擁有個人足球文化的球星就更多了。穆里尼奧說如今沒有個人足球文化，是一個事實，也是足壇發展的一個必然過程，所以這個時代的巨星更多在於教練，而不是球員。如何將體系文化與個人足球文化結合好，或許才是未來足壇需要考慮的東西，畢竟足球想要維持世界第一運動，其個人文化中的觀賞性，是不可或缺的。
