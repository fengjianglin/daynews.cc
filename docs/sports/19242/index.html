17年來首勝！山東西王客場116：107勝新疆豪取六連勝

2019-12-16 133條評論 1,452次閱讀 302人點贊

北京時間20：00，山東西王男籃客場挑戰新疆男籃。在經過四節比賽之後，山東西王男籃以116：107客場贏下新疆男籃。這是17年來，山東首次在新疆主場取得勝利，對於西王男籃來說，這是個值得銘記的夜晚。

這也是新疆本賽季第一次主場輸球。

賽後，山東隊主帥鞏曉彬誇讚了隊員，「新疆隊是個強隊，可能由於外援問題他們發揮不好，我們球隊準備充分，本場比賽隊員也比較努力。」

數據方面，哈德森39分8籃板，三分14投8中，7罰7中，出色的發揮讓他成為全隊最高得分者。陶漢林17分9籃板，小鯊魚在本場比賽在內線也是火力全開，但第4節6犯離場讓人遺憾。此外，梅斯27分12籃板，李敬宇11分4籃板，王汝恆9分3籃板，賈誠8分4籃板，張慶鵬5分0籃板。
