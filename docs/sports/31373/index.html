44號會助廣廈戰勝遼寧？李春江太了解遼籃，郭導的奇兵或只有1人

2019-12-24 748條評論 1,927次閱讀 355人點贊

明晚遼寧男籃將迎來這波4連客場當中的最後一戰，對手是和遼籃淵源頗深的廣廈隊。在遼籃這次南下征戰4連客場的時候，有人就做過這樣的分析，遼寧男籃肯定會遇到麻煩，而且4個對手也是一個比一個難纏。最後一個對手浙江廣廈，還在今年聯賽的第二輪戰勝過遼籃。

那麼明晚廣廈隊是主場迎戰遼寧，而且3輪過後廣廈還要在主場再戰遼寧，今年的賽程就是這麼編排的。只有到第44輪，遼籃戰廣廈才回到瀋陽主場。

為什麼提到了主客場的問題呢？估計有些球迷一看就會明白，廣廈的主場那是肯定不好打的，遼籃應該是沒少領教過。況且，明晚這場比賽對雙方來說都十分關鍵，誰都不會馬虎一點，甚至是雙方必爭的焦點場次。贏球，可能就代表著這支球隊要走出困境。

也許有人會問，不就是一場普通的常規賽嗎，輸贏能至於有那麼大的利害關係嗎？其實，大家還真不能小看這場比賽，它的重要性並不是簡單的誰輸誰贏的問題，而且球隊提升士氣走出困境的一個催化劑，兩支球隊都太需要這場勝利了。

可能大家也知道，廣廈的處境和遼寧男籃幾乎是一模一樣，球隊的不穩定，外援不夠給力，磨合的效果不佳，基本是兩支球隊的通病。況且，又都是連敗的難兄難弟，又都是上一場才贏球，緩過一點氣來。

廣廈是經過3連敗之後，上一場才戰勝吉林，給李春江多少緩解一點壓力。否則，李春江以及大公子都有相當的危機感了，他們已經聽到了罵聲。

而遼寧男籃也是如此，此番南下的4個客場，一連就輸了兩個，球隊的狀況幾乎很糟，輿論壓力四面八方而來。幸好，遼籃上一場戰勝福建，也是一場非常降火的勝利，讓郭士強也緩過一口氣來。但是，雙方無論是誰，僅僅贏了一場是肯定不夠用的，至少誰都輸不起了。你想想，這場比賽的意義重要不重要？

儘管說雙方誰都會力爭想要贏下這場比賽，但是贏下比賽關鍵還是要看實力的。當然，球員的狀態，臨場發揮，主教練的應變能力等等也都非常重要。

客觀來講，上一輪廣廈之所以能戰勝吉林，真正的功臣是新來的小外援威爾斯，也就是場上的那位非常活躍的44號。就連李春江也不得不承認，是威爾斯的發揮給廣廈帶來的這場勝利。這場比賽，威爾斯只是在第一節稍有緊張，隨後就找到比賽的節奏，很快適應了場上環境。他能突能投，外線也有籃子，而且防守也夠積極。出場41分鐘，砍下34分11個籃板4次助攻。是廣廈今年所有的外援當中，唯一一個砍下30+得分的外籍球員。

說實話，李春江對威爾斯的表現是相當滿意的，很可能把他當做明晚擊敗遼寧男籃的一大利器。

那麼遼籃應該有什麼對策呢？郭士強會以怎樣的部署來限制防守威爾斯的發揮呢？恐怕這也是一個問題，遼籃有個特點，對於能突能投的實力型小外援，防守都不太理想。明晚的比賽，對於威爾斯的防守恐怕是依然比較吃力。那麼，郭士強的手裡還有沒有什麼王牌或者是奇兵可用呢？

我覺得，這事也有點棘手，因為李春江對遼寧男籃幾乎是太了解了。他知道怎樣限制韓德君，怎樣最大限度地防守郭艾倫或趙繼偉，以及如何抑制巴斯的特點。當然，這只是相對而言，能不能限制得住，還要看臨場發揮。只不過郭士強必須要有所準備，李春江肯定會對遼籃的重點人給與重點照顧。

然而，郭士強也應該針對李春江的特點，或許還真有一個奇兵可用。是誰？就是李春江還不太熟悉的史蒂芬森。正好史蒂芬森前兩場低迷的狀態，容易給人造成假象。如果明晚史蒂芬森能夠恢復當初的狀態，那就極有可能成為遼籃的一大奇兵。
