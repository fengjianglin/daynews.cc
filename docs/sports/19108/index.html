CBA最繁忙的一天！姚明給球迷出難題 18隊同時開戰支持誰？

2019-12-17 367條評論 1,422次閱讀 459人點贊

北京時間12月17日，CBA聯賽第18輪將展開全面角逐，這一天也是CBA最繁忙的一天。同一時間，18支球隊進行9場大PK，除了山西隊與浙江隊之外，其餘18支球隊全部出戰。如此多比賽全部集中在同一時間開賽，這可把球迷們難為壞了，究竟該選擇哪一場比賽看呢？姚明和CBA聯賽可真給媒體和球迷們出了一道難題。

​

CBA聯賽第18輪的比賽全部集中在北京時間12月17日19點35分開賽。這個比賽開賽時間確實是黃金時段，不過，對於絕大部分球迷而言，他們卻不得不面對選擇困難。在這一時間段，廣東、新疆、遼寧、北京等幾大熱門球隊全部有比賽，而且，全部同時間跳球，這也就意味著，選擇了廣東就可能要放棄新疆，選擇了遼寧或許將放棄北京。

而且，除了以上四支熱門球隊之外，像北控大戰上海，深圳決戰吉林，這兩場PK同樣充滿戲劇性和懸念。對於球迷而言，他們當下希望能夠全部一飽眼福，可當全部比賽同一時間開賽，魚與熊掌不可兼得，放棄一部分比賽肯定是少不了了。

​

不過，相比之下，明年的比賽安排卻少的可憐。同樣是黃金時段的19點35分開球，可比賽卻只有一場，山西隊迎戰浙江隊。雖然山西隊和浙江隊是本賽季兩大黑馬球隊，打法也極具觀賞性，不過，一天卻只安排一場比賽，確實有點寒酸了。

​

相隔前後一天的賽程，CBA聯賽方面的安排確實有些讓人看不懂。這一邊是一天安排18支球隊出戰，而另外一天卻只排了一場PK。可以說，無論是從球迷的角度，還是CBA聯賽的推廣、曝光率上來看，這都不是很合理的。看樣子，在推動CBA發展的道路上，姚明和CBA聯賽還任重道遠。
