田徑歷史第一人，北京奧運冠軍金牌被收回，博爾特：我並沒太失望

天天要聞
7 天 ago

世界賽場上曾經出現了許多偉大的職業運動員，而在田徑賽場上最偉大的職業運動員就是博爾特。職業生涯他取得的偉大成就幾乎沒有人可以比得過，接下來就讓小編帶著大家深入了解一下博爾特田徑歷史第一人，北京奧運冠軍金牌被收回的事。

博爾特是在牙買加的加特里洛尼區出生的，他的家庭非常的普通，父親是一名經營小商店的老闆。博爾特出生的時候身上最大的天賦就是他的嗓音，那時候他的父親覺得自己的孩子長大以後一定能成為歌唱家，但是事情卻事與願違。

因為經營不善，父親的小商店再也開不下去。那個時候全家欠了許多的錢，沒有辦法的父親只能讓博爾特去體校學習，畢竟在當時的牙買加，窮人唯一的選擇就是跑步。家裡有這麼一個孩子，母親和父親也只能在外面辛辛苦苦的打拚，為的就是博爾特能夠在學校學習更長的時間，而博爾特也一直在努力，沒有辜負自己父母辛辛苦苦付出。

在田徑賽場上博爾特展現出了自己超級強大的天賦。2002年的世界青年錦標賽上他拿到了200米的金牌，從那以後他的職業生涯就像開了掛一樣，至今還是賽場上的偉大神話。但是他的職業生涯獲得了那麼多金牌，有一個卻被回收了，這其中又有什麼樣的原因呢？

2008年北京奧運會舉辦期間博爾特他們拿到了4×100米接力賽的冠軍。那個時候牙買加的陣容十分的強大，這四個人中有博爾特以及鮑威爾這樣實力強勁的選手，而在奧運賽場上他們也創造了世界紀錄，跑出了37秒10的優秀成績。所有賽場上的觀眾以及全世界的粉絲們都在為博爾特他們創造的成績而歡呼。

雖然在那場比賽上博爾特他們獲得了許多偉大的成績，但是那幾年裡國際田聯對興奮劑開始嚴格的監察，所有的運動員都要有一份b型的尿液進行複檢，這也包括4×100米賽場上的牙買加隊。當時在牙買加隊的卡特的尿液檢測結果顯示出甲基乙胺，但是卡特自己並不承認這個事情。因為在2010年的國際賽場上甲基乙胺並沒有被認為是一種興奮劑的材料，所以牙買加隊也表示非常的不滿意，但是國際田聯可不買賬。

雖然牙買加隊最終向法庭上訴，但是國際仲裁法庭卻沒有理會他的這一請求。結果在4×100米這個項目上法不容情，博爾特奧運金牌被北京收回，這也是他職業生涯最遺憾的一件事情，沒有輸給自己的對手，而是輸給了這種藥品。　　

雖然說博爾特遭遇了這樣的事情，但是他本人的心情並沒有受到什麼太大的影響。博爾特：這沒什麼好傷心的，我並沒太失望。博爾特表示自己還有非常強大的實力，而且發生這種事情並不是自己想要的結果，但我會保持著一個很好的心態。而之後他在賽場上又創造了許多偉大的神話，關於這件事情大家也早已經淡忘。

作為一個偉大的職業運動員，博爾特這些年來取得的成就確實值得人們去尊敬，而當年發生這樣的事情恐怕也不是他自己想要看到的結果。那麼大家有什麼樣想要說的話呢？歡迎在評論區下方留言分享哦。
