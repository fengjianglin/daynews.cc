小球時代，“打瘋了”的廣東隊沒有球隊能擋得住

天天要聞
7 小時 ago

經常看NBA的球迷就會知道，勇士之所以能開創王朝，靠的就是小球打法，而在CBA，廣東這兩年採取的小球打法也席捲全聯盟。

或許是上個賽季廣東隊成功奪冠，其它對紛紛效仿廣東隊的打法，採取雙小外，結果成績卻是差強人意。

當然，廣東隊擁有易建聯，這是他們能夠成功的重要原因。但是，他們又不僅僅是擁有易建聯。

客場面對江蘇隊，賽前很多人都猜測廣東將會取勝。結果不難猜測，但過程卻讓人大開眼界。

首節，吉布森就開始了表演，他在告訴我們什麼叫NBA個人英雄主義。三分、跳投、突破、罰球，一頓操作猛如虎，連投帶搶拿下全隊的前13分，在出場的5分40秒內拿下15分2籃板帶領江蘇19-13領先廣東6分。

或許是被吉布森感染，江蘇隊多點開花，首節就以39-33領先廣東6分。這樣的比分，加上上一場廣東的失利，讓人們彷彿又看到了擊敗廣東的希望。

然而到了第二節卻是風雲突變。從11分44秒趙睿兩罰全中到11分08秒任駿飛三分命中，36秒內廣東隊連得8分，迅速抹平分差。這才是剛剛開始，好戲還在後頭。

三分命中、防守反擊跳投命中、底線發球快速推進打進、一系列的操作讓人眼花繚亂，當節奏落入廣東手裡時，江蘇隊只能忙於拆東牆補西牆：單節就讓廣東逼出10失誤。這也是為什麼江蘇隊罰球22-9領先卻半場落後25分的重要原因。

廣東能在第二節打下50-19的分差，一方面是三分精進（56%命中率），二是以進攻帶動防守，將速度提起來，有空位就投（半場68%命中率），而這恰恰是小球的精髓，也是小球的特點。廣東將小球打法演繹得熟能生巧。

上一次面對福建隊，投瘋了的廣東隊半場就轟下創紀錄86分，今天投瘋了的廣東半場一樣轟下讓人驚嘆的83分。而單節拿下50分，也是本賽季廣東單節第二次轟下50+分。

打瘋了的廣東隊甚至在比賽還剩下7分37秒時就把外援撤下來，用全華班對抗江蘇。

上一場不敵馬布裡帶領的北控後，廣東全隊卵足勁，也用一場酣暢淋漓的大勝一掃陰霾。同時也在向全聯盟宣告：廣東還是廣東，不容小覷。

上一輪比賽輸後，杜峰不服氣，認為是球隊投籃不準才導致輸球的。這一輪，當廣東打出他們熟悉的節奏，沒有球隊能阻擋。
