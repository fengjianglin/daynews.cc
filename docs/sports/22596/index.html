皇馬巴薩德比現震撼1幕！克羅斯自己撂倒後，對梅西做出了這1舉動

2019-12-19 431條評論 1,638次閱讀 421人點贊

0-0，這場平局看似是悶平，但是場面依舊很激烈，雙方的黃牌數量比雙方的射正數還多；這場比賽的勝負也將改變多項數據，那就是誰將佔領榜首，誰將成為了射手榜頭名，因為梅西和本澤馬的進球數相當，兩隊此役的對決，如果兩人有進球的話，在射手榜方面的競爭更有意思。

這場比賽儘管巴薩巨星沒有進球，但他依舊是本場最備受外界關注的焦點，當然，梅西的表現依舊值得點贊，全場得分7.5，隊內最高，這足以說明了他的表現，而在這場比賽中，32歲的梅西也展現了超強實力，拉不倒，踢不倒，防守要多人協防。

齊達內的戰術部署確實到位，梅西不得不站位靠後，踢得變成了哈維，在比賽第60分鐘左右，格里茲曼的橫敲，梅西罕見停大了，這很明顯是站位沒有調整好，按照以往的經驗，他一般會直接射門，這次卻停到了，不過梅西依舊奮力奪回球權，這1點可以看出，32歲的梅西也夠拼。

然而這僅僅是其中一個突出點表現，更震撼的1幕發生在比賽最後階段，也就是第92分鐘左右，皇馬和巴薩當時還0-0，梅西大爆發，欲絕殺皇馬，皇馬自然也不會隨便讓梅西得逞計劃，當時梅西在中場拿球快速突破時，把守中場的皇馬核心克羅斯奮力阻擋，在防守過程中自己被撂倒了，然而對梅西做出了一罕見舉動，快倒下時拉扯梅西，梅西的褲子險些被拉破。

而就在這樣的情況下，梅西依舊沒有倒下，這太強悍了，要不是實在被對手拉著無法上前，梅西極有可能拖著克羅斯上前，這也是比賽最後時刻，還有這樣的爆發力，真的難以想像，而梅西不止一次這樣讓皇馬中場尷尬了，上輪對陣皇馬，莫德里奇同樣做出了類似舉動，都被梅西如同拖著行走一樣。
