皇馬聖誕聚餐！一人神似梅西“嚇壞”球迷，拉莫斯扎小辮搶鏡

2019-12-17 57條評論 1,115次閱讀 173人點贊

北京時間12月17日，皇家馬德里舉辦了一年一度的聖誕聚餐，皇馬籃球、足球運動員悉數到場！儘管歐冠淘汰賽抽中了曼城，但皇馬球員並沒有因此而感到緊張，他們個個神態飛揚，顯得相當的輕鬆！皇馬大佬佛羅倫蒂諾席前為每一位球員送上祝福，團隊氛圍十分和睦！

首先，佛羅倫蒂諾來到了拉莫斯這一桌！三位皇馬巨佬同框，這是何等的氣派！拉莫斯這次可謂是精心打扮，頭髮抹得發亮，後邊還扎了一個小辮子！看來水爺這是要留長頭髮了，想要找回年輕時的感覺？事後，拉莫斯將這張照片分享到了自己的社交賬號上！

隨後是莫德里奇和貝爾！兩位近來的情況可謂天壤之別，莫德里奇狀態慢慢回暖，但貝爾依舊穩坐替補席！不過在這開心的日子裡，貝爾並沒有吝嗇他的笑容！阿扎爾的傷退，貝爾或許能在國家德比迎來機會！

克羅斯和虎哥！全體皇馬隊員中，只有克羅斯一人的打扮符合聖誕氣息。寬哥身著一件酒紅色的毛衣，有點鶴立雞群的感覺！卡塞米羅相當的熱情，一個勁的和老佛爺聊著家常！

巴西雙子星！羅德里戈和維尼修斯貼身而坐，兩人的關係早已超出了其他隊友！維尼修斯最近狀態回升，他與羅德里戈的組合讓人期待！羅德里戈這天仙顏值備受粉絲喜愛，這次又吸引了不少的粉絲喲！

本澤馬和阿扎爾！阿扎爾進入會場的時候還一瘸一拐，坐在凳子上則喜笑顏開！最近和巴斯克斯一同在療養，為什麼兩人沒坐在一起？老佛爺對本澤馬讚譽不斷，弄得法國人都有一點不好意思了！

皇馬救星，一人比肩阿森納全隊的傳奇門將庫爾圖瓦！上輪聯賽，庫爾圖瓦用一記頭槌幫助本澤馬破門，他唯一一次攻門就射正，這與阿森納全隊持平！2米高的褲娃低頭看著自己的老闆，他不害怕嗎？

這一桌比較特別，能不能稱之為「病號桌」？這一桌的球員大部分都有傷病在身，看來極為仁兄還是有先見之明的，早已協商坐在一起了！眼睛放大一點，仔細瞧一瞧，有沒有被嚇到？右邊最後一個是誰？對，你沒有看錯，他並不是梅西！他只是像梅西而已！

這人的出現真是把不少球迷給「嚇壞」了！這側臉與梅西有啥區別呢？不少球迷調侃道：「老佛爺真氣派，為了給皇馬球員一個聖誕驚喜，他把梅西都買過來了！」。不過這真是一個冒牌貨，真正的梅西估計在巴薩的桌子上切著牛排呢！

皇馬這麼知名的機構，怎麼也得在聖誕節期間做點好事！席後，齊達內就帶著拉莫斯、馬塞洛以及幾名籃球隊員前往了當地一家醫院！幾人帶著禮物慰問了醫院中的幾名兒童，為他們送上了聖誕祝福！玩得開心，做得有意義，這次的活動很是成功！
