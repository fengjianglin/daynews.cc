一方出了兩個國字型大小隊長，三新星均獲首發機會，貝帥卻只看中一將

大連一方回到中超兩個賽季了僅僅出了李帥一個國腳，但是踢了一場比賽之后里皮就沒有使用他了。結果里皮也辭職了，李鐵執教的國足選拔隊也沒有招入李帥。大連足球的落寞從國字型大小球員的人數當中就可以看出來了。在國奧隊當中原本單歡歡進入到了國奧隊的大名單當中，但是很遺憾單歡歡傷病的情況比較嚴重，國家隊和國字型大小都沒有大連一方的球員。

但是U20國家隊大連一方可以說佔據了非常大的比重，黃嘉輝、何宇鵬、李學博和外租的張健生都是入選了U20國家隊。U20也是在之前進行了三場比賽，大連一方的三名球員全部都獲得了首發的機會。第一場對陣莫斯科斯巴達二隊的比賽當中黃嘉輝進行了首發，並且他還是球隊的隊長。可以說他是這支U20國家隊的領袖級別人物。

第二場比賽U20國家隊對陣貝爾格萊德紅星隊，黃嘉輝和何宇鵬進行了首發。黃嘉輝依然是球隊的隊長，第三場比賽U20國家隊面對沙爾克04二隊，在這場比賽中黃嘉輝並沒有出場。大連一方的另兩名小將何宇鵬和李學博聯袂首發，這場比賽何宇鵬成為了球隊的隊長，可以說大連一方出了兩個國字型大小的隊長，承包了國字型大小的隊長。三名球員還是很被重用。

不過比較尷尬的就是這三名球員不是國奧適齡球員，想要進入國奧就需要特彆強的實力。大連一方這個年輕段球員當中還有不少，目前來看這三名U20國字型大小球員當中本賽季只有何宇鵬獲得了不少的機會。何宇鵬是非常幸運的，如果王獻鈞不走何宇鵬也不會得到什麼機會。雖然在中超何宇鵬的身體和經驗還處於劣勢，但是他在上場之後非常的拼，機會是留給有準備的人，何宇鵬也打進了中超處子球。

在貝尼特斯來到球隊之後也只有何宇鵬得到了一次替補的機會，貝帥只看中何宇鵬一將了。黃嘉輝雖然在國字型大小球隊是隊長但是在大連一方只獲得了一次機會，他並不重用。在他身前二次轉會來的周瀟獲得的機會都要比黃嘉輝要多，同樣作為萬達西班牙留洋計劃的黃嘉輝已經落後了。下賽季黃嘉輝在一方在前景也不是很好。

李學博下半賽季是大連一方的二號門將，他也是萬達西班牙留洋計劃的球員。等了半個賽季李學博也沒有獲得出場的機會，大連足球在U20國字型大小有球員站穩腳跟就已經很不容易了。大連足球需要一段很長時間的沉澱才能夠完成復甦，等到大連一方能夠在國奧隊有5個球員，在國家隊有3個球員就是大連足球逐漸復甦分代表。對於大連足球來說現在還只能有羨慕的份。
