劉國梁這一項決定，引得網友紛紛替許昕劉詩雯鳴不平，老將太辛苦

2019-12-18 676條評論 3,761次閱讀 2525人點贊

北京時間12月17日，國際乒聯年終總決賽落下帷幕，國乒斬獲4塊金牌，日乒僅有1塊金牌入賬，而本次賽事焦點之戰中混雙項目令人記憶猶新，「昕雯聯播」組合在混雙決賽上遭遇日乒黑馬組合伊藤美誠/水谷隼，前期0-2大比分落後的情況下，兩位實力悍將扛住巨壓力挽狂瀾，讓對手連「吃」三局，最終實現大逆轉，總比分以3-2贏下了這場令人激動不已的勝利。

而接下來國乒要開始準備2020年的第一項賽事，團體世乒直通賽，為了讓更多新生代的選手有機會在大賽上歷練，國乒還是採取直通賽的方式，並非沿用去年的世乒賽陣容，然而劉國梁主席在此次的世乒賽直通賽上做出新的決定，會在直通賽事上新增混雙項目，包括許昕/劉詩雯組合以外還會增加5對組合角逐PK。

然而這樣的決定讓網友們沒有搞懂，因為團體世乒賽上並沒有設立混雙項目，劉國梁的這項決定導致許昕/劉詩雯將要繼續身兼多個項目，然而許昕已經30，劉詩雯也即將步入29，許昕已經長期「三線作戰」，此次的年終總決賽也是如此，導致在首日手腕就出現了傷情，如此高強度，老將能夠堅持下來實在令人欽佩。

然而劉國梁主席在總決賽後接受採訪表示雖然完成了目標，但是結果並不滿意，很明顯在本次總決賽上的角逐，國乒對陣日乒組合贏得比較艱難，並且奧運會混雙項目都只有一對組合參賽，容錯率太低，所以為了提高勝率希望更多磨練這對「昕雯聯播」組合，而大部分網友表示練兵沒有問題，可是也要根據實際情況，許昕與劉詩雯這樣的老將，除了年齡大身上還都是傷，除了練兵也要養兵，不能太折騰。而有的網友認為支持這樣的決定，因為在奧運會上會更加保險，大家對此有何看法？歡迎評論區留言~
