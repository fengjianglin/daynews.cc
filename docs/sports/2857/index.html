劉國梁正式官宣重大決定！聯賽停辦，為衝刺東京奧運會讓路

2019-12-03 0條評論 24次閱讀 0人點贊

摘要

在過去的幾十年里，乒乓球始終是中國體壇最受關注的項目之一。尤其是每逢夏季奧運會節點時，國乒的一舉一動更是會被外界放大。前天結束的男子單打世界盃，馬龍連負張本智和、林昀儒兩位年輕選手，樊振東臨危不亂最終為中國隊拿下冠軍，都成為了廣大球迷的焦點。而這波新聞的熱度尚未退去，又從國乒傳來了可能會再次引發外界熱議最新動態！

原來，就在本次男子單打世界盃進行的時候，中國乒協第九屆理事會還同期召開了一次會議。中國乒協主席劉國梁向外界公布了一些重要決定，其中就包括官宣乒乓球超級聯賽停辦一年。據相關消息，劉國梁表示，此舉主要還是考慮到全力衝刺2020年東京奧運會和與之相關的國際積分賽的因素，故此做出停辦乒超的決定。

不過，此次停辦乒超也並非完全是突如其來的意外，因為今年上半年就已經傳出過乒超聯賽可能會停辦的風聲。劉國梁在上任乒協主席之初曾對內設立了一套非常嚴格的考核標準，其核心指標主要是國乒選手在世界比賽、洲際比賽等國際比賽上的成績，由此也可以從側面看出，國乒在奧運會備戰的關鍵時期，對國內的一些賽事安排做出調整都是有可能的。

實際上，自從有乒超停辦的傳言以來，外界對於這件事就一直有著不同的聲音。有輿論認為：備戰奧運是重中之重，現在國際乒聯的巡迴賽本身就不少，讓國乒運動員平常有更完整的時間進行訓練、恢復，集中精力打好這些比賽，是合理的安排；但與此同時也有人認為：從世界範圍內大部分體育項目來看，職業聯賽和奧運會、世界盃、世錦賽大型國際賽事之間基本上是可以長期並存共進的，就乒超聯賽而言，或許可以積極去尋找一個多方兼顧的辦法，而不是直接取消聯賽，畢竟從長遠來看，體育需要職業化和市場化，打奧運會的運動員只是金字塔尖的幾名選手，還有很多運動員是非常需要像聯賽這樣級別的舞台的——而作為明年奧運會東道主，日本方面一邊不忘時刻衝擊國乒，一邊也沒把他們的T聯賽聯賽丟下。

無論如何，現在乒協方面的決定已下，在此我們也是希望國乒能夠在奧運會上用成績證明自己。至於乒超聯賽未來會以怎樣的方式歸來，乒協能否利用停辦一年的空檔期在商業化和市場化策略方面給今後的聯賽帶來全新方案，為中國乒乓球尋求職業化創新，以世界乒聯領軍者的身份讓這項運動開拓更宏大的格局，相信大家一定會拭目以待！
