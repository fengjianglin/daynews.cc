28勝12平14負 索帥執教曼聯將滿一周年 他確實最適合這群年輕人

天天要聞
2019-12-16

當地時間周三就將是曼聯解僱穆里尼奧而選擇了索爾斯克亞擔任球隊主帥的一周年紀念日。英超紅魔如今只能排名聯賽第六，而波切蒂諾的名字依舊盤旋在老特拉福德上空，他依舊存在著成為曼聯主帥的可能性。

最初，似乎一切並沒有什麼兩樣，穆里尼奧在去年12月主場1-3不敵利物浦的比賽中下課，但是在青訓小將格林伍德打入扳平比分的進球逼平埃弗頓後，球隊達成了一項特殊的紀錄，這是他們連續第4000場在球隊中擁有至少一名青訓球員的比賽。

而對於埃弗頓而言，臨時主帥鄧肯-弗格森在夢劇場帶走了他應得的1分，曼聯在賽後累積25分，比同期穆里尼奧被解僱時還要少1分。然而，好消息是，曼聯如今僅僅落後前四名4分，這比起索爾斯克亞剛剛接手時落後13分的差距還是有了較大的改觀的。

索爾斯克亞如今執教曼聯總共54場比賽，其中28勝，12平，失利14場。實話說，這樣的數據看上去十分的平庸，甚至可以說是平平無奇。但是挪威人依舊在執教的原因或許並不僅僅是因為數據，而是他在場上場下給球隊帶來的變化。

球星們來了又走，發揮不佳的大牌球星們離開，而年輕的，經驗有所欠缺的年輕球員們被提拔上來，相比起穆里尼奧時代，球隊上下的氛圍更加積極而樂觀了。

這趟旅程起起伏伏磕磕絆絆，曾經有著不可思議地逆轉大巴黎的高峰，也有著幾周以後被埃弗頓4-0痛擊的低谷。對陣熱刺和曼城的背靠背勝利標誌著轉折點，然而本周悶平埃弗頓的表現又證明了，他們離殺入前四還差的有點兒遠。

正如索爾斯克亞在賽後承認的那樣，「這並非比賽水平的倒退，而更加像是我們的真實水平。」

有著速度奇快的拉什福德、丹尼爾-詹姆斯和傑西-林加德的曼聯隊如今似乎成了反擊的專家，他們能夠迅速地撕破防守造成殺傷的能力，正是他們能夠在排名前五的球隊身上拿到13分的原因。但是他們在對陣排名他們以下的球隊時，12場比賽僅僅獲得12分，這或許是因為索帥的球隊並沒有什麼B計劃。

當球隊表現低迷而缺乏反擊的空間時，曼聯隊很快沒了進攻的辦法。他們急需創造力，或許像博格巴這樣的球員，又或者他們需要一位能夠控制節奏的，經驗豐富的中場。年輕和缺乏經驗，讓這支曼聯隊在每次進攻中熱衷於傳出最後一傳，而非試圖打的簡單一些，保持控球權控制節奏。

「當你置身事外當然會簡單一些，當你坐在看台上的時候，」索爾斯克亞說。「我們隊缺乏冷靜的頭腦，隨著經驗的豐富事情或許會好轉。孩子們都想這麼做，都希望把事情做好，但是我們確實缺乏那麼點鎮靜。」

擊敗曼城和熱刺的比賽多少打消了關於索肖帥位的質疑，此前曾經有消息稱倘若索帥在爭四戰中和前四的差距被拉大，那麼他將面臨下課噩運。面對埃弗頓的比賽中無奈戰平則是可以接受的結果，球隊經驗的缺乏阻礙了他們。但是這樣的結果不應該時時發生，要知道波切蒂諾依舊賦閑在家，曼聯隨時有機會請他執教。

自從三月份擊敗南安普頓隊的比賽以來，球隊再也沒有後來居上逆轉贏球，本場比賽，林德洛夫不慎自擺烏龍，但是在格林伍德77分鐘扳平比分之後，球隊本可以繼續進攻爭取反超的。

然而球隊上下依舊沒有表現出爭勝的慾望，這是索帥另外一個需要注意的地方，儘管在執教曼聯一年之後，球隊確實比索帥執教前排名更高。

「我確實十分享受執教的過程，我想我們走在正確的道路上，」索帥在談到執教曼聯一年以後說道。「我期待著明天的工作，孩子們都想要提高。我們如今是一個大團隊，一個整體，一起向前。」
