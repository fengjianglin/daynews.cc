足壇夏窗引援回顧：西甲三雄皆失算，唯獨尤文收穫喜人

2019-12-14 319條評論 2,912次閱讀 1390人點贊

隨著聖誕節即將來臨，五大聯賽的賽程也即將過半，這也意味著冬窗馬上就要開始了，各支強隊都已經開始了躍躍欲試。當然引援也是看運氣，我們來看一下在這個夏窗被引進而且身價比較高的幾位球員發揮如何。

菲利克斯作為這個夏天的標王，以1.26億歐元的身價從本菲卡加盟了馬競，讓馬競在失去了格里茲曼後得到了最大的補充。賽季初菲利克斯表現得讓人眼前一亮，經常有驚艷的發揮，但是因為身體比較瘦弱，所以經常容易受傷，這也讓他的狀態總是很難保持。不過菲利克斯獲得了今年的金童獎，這也說明越來越多的人認可了他。雖然馬競目前的戰績並不算很好，但西蒙尼還是很器重他的，相信他會被培養得越來越好。

反觀離隊的格里茲曼則是巴薩在夏天求購內馬爾失敗而選擇的代替品，但就是這樣依舊讓巴薩掏出了1.2億歐元的轉會費。不得不說巴薩現在出手是真的闊氣，基本都是1億起價。格里茲曼在巴薩的表現只能算是勉強及格，沒有實質上地提高巴薩的戰力，現在他又被重新擺上了貨架，這筆交易目前看起來還是很失敗的。

作為巴薩最大的對手，皇馬這個夏天也花了不少錢，阿扎爾是其中最貴的一筆。但是阿扎爾在轉會皇馬後，彷彿大家就很少再聽到他的新聞了。隨著體重的增加，他的競技狀態也一直下滑，本來老佛爺想的是買阿扎爾來接班c羅，現在看來連個貝爾都代替不了。來來去去這麼多人，皇馬現在還要靠本澤馬撐著，看來本澤馬沒白背鍋，畢竟他才是皇馬的真核心。

再看一下後場球員，盧卡斯和德利赫特是夏季市場身價最高的兩個後衛，一個加盟了拜仁，一個加盟了尤文。盧卡斯初期表現還可以，但是遇到了傷病困擾，這也是拜仁這賽季表現不盡如人意的原因，一個花了7000萬買回來的球員在養傷，戰績能好就怪了。

反觀德利赫特，賽季初被人戲稱為手球哥，自帶0比1屬性，每場比賽基本都會送點，但是隨著比賽增多，突然開竅了慢慢恢復了他在阿賈克斯時的競技水平，這樣的德利赫特是真讓人放心，他也成了上述這些球員中目前看起來最值的一個。再往下就是他的前隊友德容了，阿賈克斯出品，必屬精品。巴薩成功找到了未來的中場核心，他才22歲，讓人們對他的未來充滿了期望。你覺得誰會在下半賽季崛起呢？

（羅掌柜）
