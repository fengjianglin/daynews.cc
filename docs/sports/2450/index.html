北京時間12月6日，足協杯決賽次回合在上海虹口足球場打響，上海申花迎戰山東魯能，首回合比賽，魯能1-0領先，賽前，雙方首發陣容出爐。今日頭條圖文直播，CCTV-5視頻直播，比賽19:35開球。

魯能和申花是自1995年足協杯決賽相遇後，第2次在決賽階段相遇。不過與24年前相比，決賽變為兩回合的賽事，魯能首回合主場1-0取勝，而且次回合較量，魯能兵強馬壯，面對申花已經連續8次不敗，取得4勝4平的成績。

申花次回合比賽1-0取勝，將把比賽帶入加時，以2球優勢擊敗魯能，才能實現逆襲。而且進入中超時代，申花只有在2009賽季主場以4-1擊敗過魯能，取得2球（或以上）的優勢拿下魯能。

更為值得一提的是，先主後客的球隊，已經連續6次奪得足協杯冠軍，這一次是魯能將這個定律延續到7年，還是申花打破魔咒？

上海申花首發陣容：27-李帥；3-畢津浩、5-朱辰傑（U23）、10-莫雷諾、16-李運秋、20-金信煜、22-沙拉維、23-柏佳駿、25-彭欣力、26-錢傑給、28-曹贇定

替補：4-蔣聖龍（U23）、7-王永珀、8-張璐、9-伊哈洛、12-陳釗（U23）、18-高迪、32-艾迪

山東魯能首發陣容：14-王大雷；5-鄭錚、9-佩萊、11-劉洋、13-張弛、17-吳興涵、22-蒿俊閔、25-費萊尼、30-莫伊塞斯、33-金敬道、35-戴琳

替補：3-劉軍帥、6-王彤、18-周海濱、20-韓鎔澤、21-劉彬彬、23-格德斯、36-段劉愚（U23）
