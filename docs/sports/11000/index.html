大有可為！郝偉愛徒連續神發揮太亮眼，魯能青訓已成國奧逆襲希望

2019-12-12 77條評論 1,018次閱讀 95人點贊

北京時間2019年12月11日晚間，珠海四國賽迎來了第二輪的較量，國奧隊主場迎戰敘利亞吸引了球迷的關注和較量，要知道敘利亞是一支戰火不斷的國家，能夠擁有U22年齡段的球員已經非常不易，最關鍵的是這一場比賽國奧隊是全主力出戰，朱辰傑、楊立瑜也回到了國奧隊，首發陣容的實力足以趕超一支中超俱樂部，但是最終的比分卻令球迷不寒而慄，最終國奧隊0-1爆冷不敵敘利亞。

這一場比賽國奧隊的場上局勢非常好，幾乎佔據了大部分的主動權和進攻優勢，奈何小將張玉寧多次浪費良機，最終也沒有保證國奧隊主場戰勝對手，這一戰最大的收穫便是魯能青訓，畢竟趙劍非、段劉愚、黃聰的表現非常出色，其中最搶眼的便是段劉愚，作為魯能中場的未來班底，段劉愚的球技、球商已經是國內目前最出色的年輕中場，他的傳球調度、組織能力非常出色，這得益於李霄鵬、郝偉對他格外的栽培，放眼中超聯賽、中甲聯賽段劉愚的天賦無人能及，當然這也只是在中場的範圍之內。

本賽季中超聯賽段劉愚攻進了2粒進球和5腳助攻，並且還是在19場出場的次數下，這樣的數據擺在任何一名年輕小將面前都是頂級水平，在希丁克執教期間段劉愚等魯能小將均沒有進入國奧隊大名單，但是郝偉上任國奧隊主教練後，魯能天賦滿滿的小將得到了發揮，其中最大的收穫便是段劉愚，雖然本場比賽0-1不敵敘利亞，但是國奧全場佔據著優勢，小範圍配合、精彩的進攻讓球迷看到了逆襲的希望，當然這得益於魯能青訓的結果，顯然國奧隊目前的台柱子則是魯能青訓，像徐根寶青訓已經淡出國奧隊的行列。

畢竟魯能青訓是國內目前最頂尖的體制，魯能中眾多小將在國奧隊有著精彩發揮就是最好的答案，如今郝偉重新幫助國奧隊重回正軌，不僅僅需要的是其他青訓的頂尖小將，像魯能天賦滿滿的段劉愚依舊要重用，誰有能力誰首發才是最合適的辦法。
