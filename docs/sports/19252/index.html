無壓力4-0！大巴黎法甲完爆10人球隊，姆巴佩再現風騷腳後跟技術

2019-12-16 3,352條評論 4,762次閱讀 2510人點贊

北京時間12月16日4:00，法甲第18輪，巴黎聖日耳曼客場挑戰聖埃蒂安。內馬爾姆巴佩領銜首發陣容。

本場比賽有超過1000名巴黎人現場為球隊助威！

圖赫爾賽前採訪表示：聖埃蒂安很強，此戰強度不會低於上周打蒙彼利埃。本賽季球隊傷病不斷，但大家仍讓勝利不曾間斷。姆巴佩對我們來說真的太重要了！

聖埃蒂安 0 ：4 巴黎聖日耳曼

上半場，帕雷德斯的遠射幫助巴黎客場領先，阿霍盧身後飛鏟帕雷德斯被裁判直接紅牌罰下，半場臨近尾聲，姆巴佩接內馬爾傳球單刀破門，巴黎暫時2-0領先。

第9分鐘，默尼耶右路傳中，後點的貝爾納特推射被擋出，禁區外的帕雷德斯迎球怒射，皮球發生折射後應聲入網！1-0！巴黎客場領先！

第19分鐘，特朗科背後踢倒了迪馬利亞被裁判黃牌警告。第23分鐘，阿霍盧身後飛鏟帕雷德斯，裁判直接出示紅牌將其罰下。

第39分鐘，迪亞洛正面封堵放倒了德尼-布安加，裁判向其出示黃牌。第43分鐘，內馬爾直塞球來到姆巴佩腳下，後者禁區內面對門將低射破門！2-0！

半場結束，巴黎暫時2-0領先。

下半場，迪馬利亞接內馬爾傳中抽射中柱，內馬爾造點親自主罰被立柱彈出，庫爾扎瓦助攻伊卡爾迪破門，比賽最後階段，姆巴佩單刀再次擴大比分。

第52分鐘，內馬爾禁區內與隊友完成撞牆配合後一腳勁射被呂菲耶化解。第54分鐘，內馬爾左路下底傳中，迪馬利亞後點抽射打在立柱彈出。迪馬利亞直塞伊卡爾迪，後者橫傳中路，姆巴佩面對空門沒有夠到皮球。第60分鐘，德比希禁區內鏟倒了內馬爾，裁判判罰點球，隨後內馬爾點球擊中立柱，巴黎錯失擴大比分的機會。

第72分鐘，庫爾扎瓦左路下底傳中，伊卡爾迪後點抽射破門！3-0！巴黎鎖定勝局！

第88分鐘，姆巴佩接隊友傳球，面對門將單刀破門。4-0！

現場煙火四溢！

全場比賽結束，最終聖埃蒂安 0 ：4 巴黎聖日耳曼。
