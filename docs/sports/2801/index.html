羽聯最新排名：國羽兩項榜首壓日本 林丹第16陳雨菲總決賽沖第1

2019-12-04 0條評論 25次閱讀 0人點贊

摘要

伴隨2019年印度賽義德莫迪賽在上周末落幕，如今世界羽聯公布了最新一期的世界排名，日本如今僅有桃田賢斗佔據男單世界第1，而國羽則是佔據2個單項的世界第1，分別是女雙凡晨與混雙雅思高居榜首。

伴隨2019年印度賽義德莫迪賽在上周末落幕，如今世界羽聯公布了最新一期的世界排名，日本如今僅有桃田賢斗佔據男單世界第1，而國羽則是佔據2個單項的世界第1，分別是女雙凡晨與混雙雅思高居榜首。國羽男單3主力中林丹位列世界第16，明年可以無限制參加低級別賽，陳雨菲女單世界第2有望沖榜首。

男單：桃田賢斗第1，林丹第16明年可無限制參加低級別賽

由於世界排名前列的頂級高手，基本缺席了上一站印度賽義德莫迪賽，因而世界排名前十位均無變化，日本一哥桃田賢斗繼續穩居第1，他還在全日本錦標賽完成衛冕加冕3冠王。至於國羽排名最高的是第4名的諶龍，石宇奇則是第7名，林丹繼續維持第16名，並且2020年將擺脫前15名參賽限制，可以無限制參加低級別賽事搶分。

世界排名前十位依次是：桃田賢斗、周天成、安東森、諶龍、安賽龍、喬納坦 、石宇奇、金廷、伍家朗、常山干太。

至於中國台北選手王子維藉助在印度賽義德莫迪賽，奪得生涯巡迴賽首冠，他反超陸光祖，上升3名升至世界第19名，而陸光祖跌至第20名。

女單：陳雨菲第2追戴資穎，馬林復出第2冠升7名

女單世界前十的排名也沒有變化，中國台北一姐戴資穎繼續位列榜首，而國羽一姐陳雨菲排名第2名，且陳雨菲在總決賽有望逆襲戴資穎，去衝擊生涯首次世界第1。國羽其餘選手當中，韓悅第20名，王祉怡第21名，蔡炎炎第23名，陳曉欣第26名。

世界排名前十位依次是：戴資穎、陳雨菲、奧原希望、山口茜、因達農、辛杜 、何冰嬌、李文珊、安洗瑩、內維爾。

西班牙奧運冠軍馬林藉助上一站印度賽奪冠，斬獲她復出後的第2座冠軍，世界排名也是上升7名，已經升至世界第11名，距離殺回世界前十僅有一步之遙。

男雙：國羽雙塔組合第3名，何譚組合奪首冠升第16

印尼組合吉迪恩/蘇卡穆約和阿山/塞蒂亞萬繼續壟斷世界前兩名，國羽最高排名是頭號男雙組合李俊慧/劉雨辰排在第3名。日本組合嘉村健士/園田啟悟第4名，再往後第5名法加爾/阿迪安托，第6名渡邊勇大/遠藤大由，第7名李洋/王齊麟，第8名韓呈愷/周昊東，第9名阿斯特魯普/拉斯穆森，第10名催率圭/徐承載。國羽其他組合當中，何濟霆/譚強奪得生涯首冠，升至第16名，歐烜屹/張楠第23名，黃凱翔/劉成第25名，邸子健/王昶第33名。

女雙：國羽凡晨繼續高居第1，力壓3對日本組合

國羽頭號女雙組合陳清晨/賈一凡繼續排在世界第1名，而日本的3對組合位列2-4名，依次是福島由紀/廣田彩花、松本麻佑/永原和可那與松友美佐紀/高橋禮華。再往後，第5名李紹希/申昇瓚，第6名金紹映/孔熙容，第7名杜玥/李茵暉，第8名波莉/拉哈尤，第9名基地塔拉庫爾/拉溫達，第10名李汶妹/鄭雨，劉玄炫/夏玉婷第16名，董文靜/馮雪穎第21名。

混雙：雅思與黃鴨壟斷世界前二，魯愷已退出國家隊

混雙依然是國羽的最強單項，而兩大王牌組合雅思組合與黃鴨組合，也是繼續壟斷世界前兩名，且他們都是積分達到10萬分以上。至於國羽3號混雙組合何濟霆/杜玥第13名，魯愷/陳露第18名，但魯愷已經宣布退出國家隊。

（文/醉卧浮生）
