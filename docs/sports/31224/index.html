2019國羽失去了最偉大的對手，看到了衝擊東京的希望

2019-12-25 565條評論 1,835次閱讀 268人點贊

自從里約奧運會之後陷入低谷以來，中國羽毛球隊就在砥礪前行。2019年是東京奧運會的衝刺之年，國羽失去了最偉大的對手李宗偉，隨著陳雨菲在年終總決賽上奪冠並登頂世界第一，也讓球迷看到了他們在東京閃光的希望。當年初張軍上任中國羽協主席時說出國羽要在五個項目上衝擊東京奧運會金牌時，也許很多人把他的話僅僅是作為對國羽的鼓勵，然而現在的國羽已經悄然聚集著能量，等待著最後的爆發。

李宗偉退役宣告一個時代的結束

2019年最讓人的揪心的羽毛球新聞莫過於37歲的馬來西亞名將李宗偉宣布退役，這位中國羽毛球隊歷史上最偉大的對手，帶著與癌症抗爭的信念和對世界冠軍的渴望離開賽場，留下一生的對手林丹在賽場上孤獨的努力著，一次次「一輪游」讓這位曾經的大滿貫輸球已經不是新聞。

在過去十幾年時間裡，「林李對決」一直都是世界羽壇最讓人關注的比賽，從2004年2月22日兩人首次在湯姆斯杯預選賽上相遇，40次的巔峰對決，成為世界羽壇一場場經典。兩次奧運會決賽、兩次世界賽決賽，林丹都取得了勝利，李宗偉的職業生涯拿到了69個國際賽事的冠軍，但始終缺少一個世界冠軍的頭銜。37歲的李宗偉曾經連續349周佔據男單世界排名第一，他的退役也宣告了一個屬於林丹的時代就此結束。

偉大對手退役的同時，國羽的男線也陷入了低谷。整個2019賽季國羽在各項賽事中僅僅收穫了6個男單，這些冠軍中還要一半來自於低級別的賽事。35歲的林丹一直堅持在賽場，除了年初在馬來西亞公開賽上收穫了冠軍之外，大多數時間都在「一輪游」中度過，距離東京奧運會也是漸行漸遠。30歲的諶龍一直到賽季末才開始有所起色，而被寄予厚望的石宇奇則在傷病中等待著康復。如今世界羽壇的男單是屬於桃田賢斗的，誰又能成為他一生的對手呢？

47個冠軍是國羽繼續前行的動力

雖然男線上疲軟，但整個2019賽季國羽的進步還是非常明顯。在5月份的蘇迪曼杯上，國羽在決賽中擊敗日本隊，時隔四年重奪蘇杯。而在8月份的世錦賽上，國羽只拿到了混雙一項冠軍，這也是1983年國羽參加世錦賽以來的最差戰績。兩項世界大賽國羽喜憂參半，這說明國羽目前整體水平仍在一流行列，但並沒有任何優勢可言。

持續了一整個賽季的世界羽聯巡迴賽也許更能體現國羽的進步，2019賽季國羽在巡迴賽中一共拿到了47個冠軍，排名第二的日本隊僅僅拿到了29冠，印尼則收穫了25冠。值得一提的是，在年終總決賽中，國羽難道了女單、女雙和混雙三項冠軍，應該說女線在經歷了前兩年的低谷之後，已經逐步有所起色。

中國女單本賽季得到了13個冠軍，陳雨菲在參加的16站賽事中15站進入四強，七次進入決賽全部奪冠，並且依靠年終總決賽的冠軍幫助中國女單時隔48個月後重返世界第一。作為世界羽壇2019年表現最出色的女單選手，陳雨菲在與頂尖高手的對決中逐漸找到了自信，已經成為東京奧運會國羽除了混雙之外第二個重要的沖金點。除此之外，國羽女雙本賽季收穫了八項冠軍，其中陳清晨/賈一凡、杜鑰/李茵暉多次日本女雙的集團優勢中殺出重圍，進步同樣讓人滿意。

混雙是國羽東京奧運的最大希望

在整個東京奧運周期，國羽最拿得出手的當屬混雙，2019賽季繼續著強勢表現，鄭思維/黃雅瓊、王懿律/黃東萍構成的混雙雙保險，只要參賽就幾乎沒有讓冠軍旁落。特別是鄭思維/黃雅瓊在2019賽季收穫了世錦賽冠軍、蘇迪曼杯冠軍和總決賽冠軍以及三站超級賽的冠軍，唯一缺少的就是奧運冠軍，他們也是國羽最信賴的混雙組合。然而在世界羽聯年終總決賽上，兩人小組賽第一場比賽就輸給了日本組合渡邊勇大/東野有紗，好在他們及時調整，在關鍵比賽中擊敗印尼組合費澤爾/維德佳佳，最終一路打進決賽，收穫了冠軍。一場失利傳遞出一個信號，國羽混雙雖然具有優勢，但也並非絕對優勢，任何閃失都有可能在東京折戟。

國羽男雙並沒有因為蔡贇、傅海峰的退役就徹底失去競爭力，重點打造的李俊慧/劉雨辰這一對雙塔組合，在與世界頂尖組合的競爭中並沒有處於絕對下風。目前世界羽壇的男雙呈現百家爭鳴的局面，李俊慧/劉雨辰本賽季只拿到了兩站比賽的冠軍，在年終總決賽上小組未能出線，要想在東京有所作為，還需要加倍努力了。

還有14項比賽等待著國羽衝刺

羽毛球的奧運資格並非來自於各項選拔賽，而是從今年4月30日紐西蘭公開賽開始，持續到2020年5月1日截止，這期間所有賽事都將作為奧運積分賽進行積分，以此來作為東京奧運會參賽資格的選拔標準。在今年7個的爭奪中，有20項賽事被納入奧運積分賽，在奧運積分榜上，目前還沒有一個協會拿滿了五個項目的全部參賽名額，國羽目前在女單、女雙和混雙上已經鎖定了兩個參賽席位，在男單和男雙上目前還只拿到1席。在剩下的奧運積分周期內，還有14項比賽等待著中國羽毛球隊進行衝刺。

按照世界羽聯的規定，每個協會只有在奧運積分榜上單打前16，雙打前八中有兩人和兩對組合才能拿到每個項目兩個參賽名額。國羽男單目前只有諶龍排名第五，積分接近7萬分，鎖定了一張東京奧運會參賽門票，另一位老將林丹目前只排名22位，距離前16還有一定距離。而因傷缺席多時的石宇奇目前只排名第27，要想前往東京參賽，就必須在抓緊時間康復，在剩下的比賽中打出高水平。國羽男雙目前只有李俊慧/劉雨辰排在前8名內，另一對組合韓呈愷/周昊東排在第11名，從兩人實力來看，打入前8的難度也比較大。

整體來說，2019賽季國羽的成績雖然與巔峰時期還有很大差距，但進步已經非常明顯。2020賽季對於國羽來說並不輕鬆，首要任務是爭取奧運會的滿額參賽席位，而後才是在東京向冠軍發起衝擊。
