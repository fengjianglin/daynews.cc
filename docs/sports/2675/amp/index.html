北京時間2019年12月4日，2019年女排世俱杯結束了第二個比賽日的爭奪，在備受球迷關注的焦點對決中，以東道主身份出戰的廣東恆大女排是以0:3不敵意甲冠軍科內利亞諾，無緣兩連勝。

在本屆世俱杯的八支參賽隊伍中，科內利亞諾被球迷譽為加強版的義大利女排國家隊，更是本屆世俱杯的最大奪冠熱門。

在這支科內利亞諾的首發七人中，除了艾格努，塞拉，德吉納羅以及弗里四大義大利主力國手以外，還有美國女排平拉開女王希爾，荷蘭女排背飛女王德克魯伊夫以及波蘭女排光速二傳沃沃什，這樣的首發陣容真是豪華到令人咋舌，正因如此，科內利亞諾是力壓伊薩奇巴希，瓦基弗銀行以及天津女排，成為本屆世俱杯的最大奪冠熱門。

面對這樣一支各方面都毫無漏洞的球隊，廣東恆大女排獲勝的可能性基本上是不存在的，而比賽的過程以及結果也正如球迷賽前預期的那般，廣東恆大女排最終是毫無抵抗之力地完敗給了科內利亞諾。

在本場比賽上，科內利亞諾的接應艾格努依舊是神一般的存在，全場比賽，艾格努在進攻端可謂是如入無人之境，超手暴扣是把廣東恆大女排的地板砸得叮咚作響，真是太強了。

艾格努在賽場上的超強表現不僅征服了全世界的球迷，更令廣東恆大女排的隊員拜倒在石榴裙下，賽後，廣東恆大女排的隊員是變身成為艾格努的迷妹了，在隊長栗垚的帶領下，廣東恆大女排的本土隊員是和艾格努來了一張大合照，在合照的過程中，艾格努不僅連連道謝還擺出了比心的手勢，艾格努的這一舉動是被球迷贊為「真接地氣」。

雖然艾格努是中國女排衛冕東京奧運會冠軍的最大對手，但這卻無法阻止球迷對強者的敬佩，艾格努的中國粉絲團是越來越壯大了。
