8連勝後3連客，山東男籃迎來2大強敵，小姚明或有機會完成救贖

天天要聞
2019-12-21

CBA常規賽19輪過後，豪取8連勝的山東男籃，以12勝7負的戰績在積分榜上的排名上升到了第6的位置。對於鞏曉彬的球隊來說，這是含金量十足的8連勝，因為對手既有新疆男籃、遼籃、上海男籃、四川男籃這樣的總冠軍球隊，也有山西男籃這樣的賽季黑馬，更是打破了17年22場不勝的「天山魔咒」，所以這足以表明鞏曉彬已經對球隊進行了成功改造。不過，接下來，山東男籃將迎來連續3個客場比賽，其中包括2大強敵。能否將連勝保持下去，對於鞏曉彬來說是嚴峻的考驗。

第19輪與遼籃的比賽，哈德森幾乎是超水平發揮，41分10個籃板7次助攻的數據，是向老東家證明自己依然是MVP級別的外援。但你要知道，對於35歲的哈德森來說，出場時間超過40分鐘，意味著體能的透支。而且為了打好這場比賽，鞏曉彬在第18輪與四川男籃交手時，讓哈德森整個第4節在場下休息。所以，下輪客場挑戰深圳男籃，哈德森狀態會不會有所下降，目前還真不好說。

還記得山東男籃擊敗遼籃後，鞏曉彬進更衣室跟球員說過的話嗎？「這場拼得不錯，當然，遼寧隊自身狀態也不是很好，我們的防守和進攻做得都不錯，大家打得很有耐心，趕緊回去吃點飯休息」。一句「趕緊回去吃點飯休息」，足以表明鞏曉彬對球員體能的擔心。畢竟，山東男籃一直都是在靠老兵帶隊：張慶鵬38歲，哈德森35歲，梅斯33歲。

對於鞏曉彬來說，目前山東男籃最大的優勢在於思想統一，「（球隊）非常團結，球的運轉和串聯做得很好，大家現在思想很統一」。而思想統一是因為鞏曉彬做了大量的工作，尤其是與哈德森的溝通。正是在哈德森向全隊道歉後，山東男籃開啟了這一波8連勝，而擊敗遼籃後，哈德森贏得了全隊由衷的掌聲。

接下來3個客場，具體來說是下輪南下客場挑戰深圳男籃，然後北上挑戰天津男籃，接著繼續北上挑戰吉林男籃。目前深圳男籃以8勝10負列積分榜第13，這支上賽季殺入季後賽4強的球隊，對於山東男籃來說是不容小覷的強敵。不過這支球隊近況不佳，近5輪僅獲1勝，且對手是弱旅天津男籃。最近的一個主場。他們居然輸給了福建男籃，所以對於山東男籃來說，這場比賽依然有機會衝擊9連勝。

至於天津男籃，只要山東男籃自己不犯錯，取勝應該不在話下。這波3連客最大的強敵，是目前以12勝6負排名第4的吉林男籃。該隊主場本賽季還沒有敗過，排名第1的廣東男籃和排名第1的新疆男籃在他們的主場落敗而歸。不過需要指出的是，這個賽季前的10月10日，山東男籃曾客場112-101擊敗了吉林男籃，那時候哈德森還沒有加盟球隊，結果那場比賽朱榮振砍下15分讓人眼前一亮。此番再度客場挑戰對手，被稱為「小姚明」的朱榮振也許有機會完成救贖——本賽季至今，他僅出場10次，場均3分1.5個籃板……
