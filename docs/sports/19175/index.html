前拳壇P4P之王洛馬琴科：2020年4月，我和IBF不敗拳王不見不散

2019-12-16 3,092條評論 4,019次閱讀 629人點贊

美國拳擊新聞網12月16日訊 今天，現WBA/WBC/WBO輕量級拳王、兩度奧運會冠軍烏克蘭天才瓦塞爾-洛馬琴科（14-1，10KO）在接受ESPN記者專訪時表示，他歡迎現IBF輕量級拳王美國重炮手特奧菲莫-洛佩茲（15-0，12KO）加入到輕量級拳王「俱樂部」，2020年4月不見不散。

洛馬琴科說：「我無法對洛佩茲與康梅（29-3，26KO）的比賽進行過多分析，因為那隻進行了兩個回合。這就是拳擊，結果已經發生了。洛佩茲有力量，他還是一個聰明的拳手，但洛佩茲只是我的又一個一個不同的對手而已。我希望統一整個輕量級，歡迎洛佩茲來到輕量級拳王俱樂部，2020年4月不見不散。」

頂級推廣公司已經正式公布，洛馬琴科與洛佩茲的輕量級終極統一戰將在2020年4月舉行，勝者將成為該級別真正的霸主。兩人都簽約於頂級推廣公司，所以比賽很好撮合。洛佩茲22歲，年輕氣盛，只是大戰經驗不足，但力量和體格上有些優勢。洛馬琴科大賽經驗豐富，技術精湛，但在135磅體格沒有什麼優勢，在輕量級的4場比賽3場是點數贏的。這場135磅的終極統一戰還是充滿懸念的。
