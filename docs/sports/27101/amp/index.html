埃里克森已經攤牌，桑喬不會出走大黃蜂

天天要聞
2019-12-22

●阿爾特塔一月份轉會目標敲定

阿森納新主帥阿爾特塔已經透露他執教槍手的第一筆簽約是安東尼奧·扎扎納，這名來自塞維利亞的17歲中場被認為是歐洲幾傢俱樂部的目標。西班牙媒體稱扎扎納已經拒絕了來自西甲豪門皇家馬德里的邀請。

●曼聯準備購買巴薩後衛

據報道曼聯準備與英超對手埃弗頓和南安普頓爭奪巴塞羅那後衛托蒂博。據加泰羅尼亞媒體報道，這名年輕球員是在夏天才從圖盧茲加盟巴薩的，很多歐洲俱樂部都在有意向簽入他。AC米蘭也準備追逐此後衛，但是英格蘭豪門仍然處於領先地位。預計托蒂波將花費1700萬英鎊，而巴塞羅那希望確保回購條款，這樣他們就可以控制球員的未來。

●熱刺中場已經跟熱刺攤牌！

根據《旗幟晚報》的報道克里斯蒂安·埃里克森已經決定不與熱刺續約。熱刺後衛揚·維爾通亨已經延長了他在熱刺的合同，熱刺高層想一氣呵成續約埃里克森，但是丹麥中場埃里克森並沒有對他在北倫敦的未來做出180度的轉變，他可能在明年1月或者明年夏天離開俱樂部。

●多特邊鋒一月轉會窗口不會離開

多特蒙德邊鋒桑喬被告知他要到明年夏天才能離開德國，因為他將重返英超聯賽。雖然利物浦、切爾西和曼聯都對這位英格蘭國腳感興趣。但有挪威記者稱多特蒙德已經排除了在一月份出售這名19歲球員的可能性。

●曼城有意向大巴黎後衛

曼城主帥瓜迪奧拉有意向引進巴黎聖日耳曼後衛馬爾基尼奧斯，當前曼城的聯賽蟬聯冠軍機會被防守問題危機所拖後腿，據《萊10體育》報道英超冠軍正在與這名後衛就轉會進行談判，馬爾基尼奧斯作為一名中衛同時也可以作為一名右後衛或者防守型中場。由於巴西國腳在工資問題上與巴黎聖日耳曼高層未能達成一致，如果曼城報價合適將有機會加盟。

●馬競有意向曼聯失寵的中場球員

塞爾維亞人自從9月以來只打了一場比賽，賽季開始時他的狀態非常不穩定。據義大利天空體育報道，馬蒂奇已經被排除在索爾斯克亞的計劃之外，馬競準備在他31歲的時候再給他一次機會。西蒙尼目前在中場需要增援，但是馬蒂奇可能要等到明年夏天他的合同到期後才能自由轉會。
