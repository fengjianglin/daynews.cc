拜仁客場1-2輸給門興，跌出積分榜前4位。上半場拜仁攻勢佔優但沒有破門，下半場剛開場不久，佩里西奇接穆勒傳球凌空抽射破門；第60分鐘，主隊霍夫曼開出角球，本瑟拜尼頭槌攻門扳平比分；補時階段，馬丁內斯鏟倒小圖拉姆被判點球，並吃到紅牌，本瑟拜尼主罰命中上演絕殺。

多特蒙德主場5-0大勝杜塞爾多夫，上半場第42分鐘，皮什切克直塞，羅伊斯推射破門；第58分鐘，桑喬助攻，小阿扎爾推射擴大比分；第63分鐘，羅伊斯橫傳，桑喬將球打進死角；第70分鐘，小阿扎爾傳球，羅伊斯推射梅開二度；第74分鐘，阿什拉夫橫敲，桑喬將球打進空門。
