北京時間12月6日訊，國際乒聯唯一一站北美賽女單正賽抽籤結果已經全部揭曉，可喜可賀的是，中國女乒包括種子選手張薔在內的12人全部晉級64強正賽，沒有一人資格賽遭淘，其中我們比較熟悉的有劉曦、車曉曦、李佳燚，不過在上半區1/4小組，儘管國乒坐擁4人，但重頭戲卻落在了國乒兩大勁敵身上，二人狹路相逢的背後，本次對決也將直接影響到明年奧運的參賽資格，兩位不是別人，正是來自日乒的老一姐石川佳純和新生代球員早田希娜。

國際乒聯加拿大挑戰賽，是本賽季最後一站公開賽賽事，按理來說與亞洲相距甚遠，外加天寒地凍和一片白雪皚皚，參賽人員應該不多才對，然而僅國乒就派出26人，日乒更是重兵出征，雖然水谷隼和丹羽孝希相繼退賽，但女乒方面石川佳純、平野美宇、早田希娜、佐藤瞳、橋本帆乃香、加藤美優、芝田沙季一干主力全部參賽，尤其是前兩位還存在著對奧運單打資格激烈的競爭關係，雖說最新一期世界排名石川佳純超越平野美宇，但按照明年1月的有效積分計算，平野美宇其實還比石川佳純多出65積分，除了國際乒聯總決賽之外，作為最後一站公開賽的北美賽場則成了二人的紛爭之地。

本次大賽，石川佳純和平野美宇分別為一二號種子，簽位佔盡優勢，因此二人決賽前不會相遇，然而除了國乒這一最大變數之外，平野美宇可以說是抽得了上上籤，而石川佳純晉級決賽就顯得不那麼明朗了，因為同在上半區1/4小組的還有一名同樣對奧運有著極度渴求的日乒大將，她就是早田希娜，此前奧地利公開賽早田希娜首輪就將平野美宇淘汰出局，在單打方面展現出了一定的提升，如今北美賽女單1/4賽區抽得第16號簽位，與石川佳純一首一尾，如果過程還算順利，那麼二人就將在8強如期相遇，屆時勢必會形成一場激烈的碰撞，因為一旦早田希娜將石川佳純擊落下馬，那麼就極有可能在明年的東京奧運佔有一席之地。

與平野美宇和石川佳純不同，早田希娜並沒有獲得世界盃和亞洲杯這樣的單打資格，目前世界排名第26，本協會第6，雖然這一成績無法獲得奧運單打資格，但日媒一直看好早田希娜的雙打能力，不論是女雙還是混雙，高大的早田希娜在護球能力上已經展現出了不俗的天賦，本賽季與伊藤美誠奪得世乒賽女雙亞軍，與張本智和勇奪奧地利混雙冠軍，所以日媒對早田的期待也並非是空穴來風，甚至還更傾向於早田取代石川。

目前來看，還真有轉變為現實的可能，那就是石川佳純最終落選單打資格，因此對於早田希娜來說，如果北美賽女單8強與石川佳純相遇，絕對會全力阻擊不留任何遺憾，當然，在相遇之前兩位也還先要通過國乒這座五指山，屆時日乒對奧運爭奪究竟是何結果，讓我們一同拭目以待。
