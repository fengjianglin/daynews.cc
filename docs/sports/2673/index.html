看深夜的C羅，還是早晨的梅西，還是兩場比賽都看？

2019-12-05 0條評論 22次閱讀 0人點贊

摘要

北京時間12月4日訊：2020年美洲杯抽籤儀式在哥倫比亞舉行，參加的外卡球隊卡達被分到北區，與巴西、哥倫比亞同組。而澳大利亞被分到南區，與阿根廷、烏拉圭同組。

北京時間12月4日訊：2020年美洲杯抽籤儀式在哥倫比亞舉行，參加的外卡球隊卡達被分到北區，與巴西、哥倫比亞同組。而澳大利亞被分到南區，與阿根廷、烏拉圭同組。

北區：哥倫比亞（東道主）、巴西、委內瑞拉、厄瓜多、秘魯、卡達；南區：阿根廷（東道主）、智利、烏拉圭、巴拉圭、玻利維亞、澳大利亞。

美洲杯分為兩個賽區，共12支球隊，按照地理位置分為南北賽區，每個賽區將各有6支球隊，將進行小組賽（前四名晉級），四分之一決賽、半決賽、之後南北大區會各自產生一名參賽隊伍，會師最後的決賽。2020年6月12日，揭幕戰阿根廷VS智利將在阿根廷紀念碑舉行，最後的決賽將在哥倫比亞舉行。

然而巧合的是，4年一次的歐洲杯也將在同一時間6月12日舉行。球迷們陷入幸福的煩惱，是看比賽規模更大、轉播質量更強、位於全球足壇中心的歐洲杯，還是觀看強隊如雲、南美激情的美洲杯。首先歐洲杯奪冠更加激烈，球隊、球星都多，每個隊都擁有5-6名球星。法國、德國、西班牙、荷蘭、英格蘭、葡萄牙、比利時、義大利，像死亡小組法國、德國、葡萄牙在小組賽即將展開火星撞地球，場場都是生死戰，比賽場面堪比決賽，能看到姆巴佩VS而時的球星C羅，與其同場競技，是球迷也是球星的幸福。美洲杯今年巴西奪冠，在南美由於國家少，所以比賽日程、規模、經濟效應遠遠小於歐洲杯，但美洲杯球星也是大牌雲集，五星巴西不多解釋，內馬爾領銜阿利松、菲爾米諾、法比尼奧、熱蘇斯、羅德里戈、馬爾基尼奧斯等歐洲豪門主力幹將，可謂是兵強馬壯、星光閃耀，南美雙雄之一的阿根廷擁有剛剛奪得金球獎的梅西、尤文雙子星伊瓜因、迪巴拉、炙手可熱的中鋒勞塔羅-馬丁內斯，曼城當家射手阿圭羅，還有大巴黎射手榜第一的伊卡爾迪，前鋒身價不要太豪華，中後場球員年齡結構偏大，陣容過於老化，後防線能否頂住南美球隊強力的衝擊，這是對阿根廷的考驗。

球迷們既要熬夜看歐洲杯，早晨還要看美洲杯，球迷們需要好好休息，注意身體健康。其實大部分球迷長期夜裡觀看五大聯賽、歐冠，看球更有感覺，而早晨看球更失去了直播的樂趣，有種早起看重播的感覺。球迷們更要做好選擇，是看夜裡的C羅，還是看早晨的梅西？
