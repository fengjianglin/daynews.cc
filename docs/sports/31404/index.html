只恨不爭氣！這些年曾被寄予厚望，但漸漸沒了下文的6位中超小將

2019-12-24 996條評論 2,055次閱讀 470人點贊

在中國足壇有一個很奇怪的現象，很多青年隊時期表現優異的球員往往很快就會墜落。

比如有一位叫王楚的球員過去曾在法國梅斯青訓營與皮亞尼奇、馬內齊名，如今兩位當年的好友已經在歐洲的頂尖豪門踢出一片天地，王楚卻淪落到了中乙聯賽。

而類似王楚這樣的故事，如今還正在中國足壇不斷的重演。

上賽季中超第11輪，年僅21歲的守門員郭全博第一次代表北京國安隊站上了中超的賽場。而在這場比賽中，郭全博高接抵擋一戰成名，此後他一舉坐穩了球隊首發門將的位置，更是憑藉在聯賽中的出色表現一度入選了國足的集訓名單。

然而當時間來到2019年，郭全博突然遭遇了全線崩盤。他不僅失去了在國安隊的首發位置，就連在國奧隊的位置也被上海上港隊的替補門將陳威佔了去，在郝偉接任國奧主帥後，近來郭全博甚至連國奧隊的集訓大名單都沒能進入。

15-16賽季，年僅20歲的唐詩在葡萄牙貢多馬爾隊交出了讓人眼前一亮的表現，全季30次出場打入了12粒進球。如此表現，也讓唐詩成為了當時國奧隊陣中的核心球員。

中超於2017賽季開始施行U-23政策，唐詩在這年年初被緊急租借到了北京國安。一年後，唐詩又轉會加盟了廣州恆大，而這可能是大家印象中最後有關於他的消息了。這位當年的國奧隊核心如今仍在恆大隊效力，但本賽季他僅在聯賽中獲得了共計66分鐘的出場時間。

與國內其他年輕球員大多只能到葡萄牙低級別聯賽留洋不同，徐新留洋時所效力的可是大名鼎鼎的馬德里競技隊。

2015年底，徐新帶著馬競青訓出身的光環回到國內加盟了廣州恆大。然而，留洋歸來的經歷，並未讓徐新能夠在這支中超豪門隊內佔據一席之地，首個賽季他僅獲得了5次替補登場的機會，而後得益於U-23政策他才逐漸有了更多的出場時間。但在失去U-23球員身份後，徐新的出場時間再次下滑，本賽季已經25歲的他在聯賽中僅獲得了7次首發的機會。

2016年底，當時年僅20歲的胡睿寶獲得了前往英超曼城隊試訓的機會，而後其聲稱恆大隊拒絕放他加盟曼城的消息，更是讓這位小將名震一時，也讓許多球迷相信能被曼城看中的他未來一定值得期待。

胡睿寶最終沒有加盟曼城，而是去了其經紀人旗下的丹麥瓦埃勒BK隊，之後則租借加盟了德乙的達姆施塔特。2018賽季，胡睿寶在歐洲轉了一圈又回到了廣州恆大，並得到了一份年薪至少高達800萬的合同（當時的傳聞是2700萬），能拿到這樣的高薪，又一次引起了球迷們對其實力的期待。然而，回到恆大後胡睿寶一共只為球隊打了1場比賽，直到今夏租借加盟河南建業，他才逐漸開始能打得上比賽。

2017年初，國足主帥里皮在中國杯上派出了當時不見經傳的王靖斌，也是創下了球員職業生涯首秀便是代表國家隊比賽的紀錄。在代表國足登場的第二場比賽中，王靖斌面對克羅埃西亞隊頭槌打入了絕平進球，這讓當時的球迷們紛紛感嘆里皮為中國足球挖到了一塊寶。

不過，王靖斌的職業生涯著實有些”首秀即巔峰”的意思。此後，王靖斌又回到了默默無聞的狀態，他曾為廣州恆大隊打過幾場比賽，而後被租到了上海申鑫隊，目前他則是效力於一支名為聖伊格納西奧的西丙球隊（西班牙第四級別聯賽），據悉這是西甲阿拉維斯隊的B隊。

這些曾被寄予厚望的球員，如今的處境著實令人唏噓。那麼，大家覺得為何我國足壇總是會出現這樣的球員呢？
