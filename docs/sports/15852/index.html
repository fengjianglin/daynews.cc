多角度回放！巴薩皮克倒下為何不給點球？看完裁判的位置你就知道

2019-12-15 707條評論 1,865次閱讀 488人點贊

12月15日凌晨，巴薩同皇家社會的西甲客場比賽，最終以2-2的平局收尾。不過巴薩最後皮克被拉到，裁判並沒有給與點球判罰，引起了極大不滿。

這個點球的對標，就是上半場剛開始的時候，皇家社會獲得的這個點球，裁判判罰布斯克茨角球防守時，他拉扯了略倫特的球衣，這阻止了後者的跳起爭球，而這個球無論他看不看VAR，實際他都會判罰點球，因為他的位置太好了，他必須認可自己。他就在弧頂處，看得很清晰。但也決定了本場點球尺度。

所以當比賽快要結束時第92分鐘，皮克被拉倒的瞬間，實際蘇亞雷斯左側挑傳，皮克在後點先是遭到了梅里諾從背後極為隱蔽的推搡，同樣皮克無法起跳，而且看上去皮克前傾左肩與迭戈-略倫特一起倒地，略倫特左手拉著皮克的衣服。《運動幫》認為，確切的說，這裡面有兩個犯規動作，但裁判都沒看到。

如果根據本場點球的判罰尺度，這個球的確是個點球。實際梅里諾從背後隱蔽的推搡動作就犯規了，但是如果換成裁判的角度，你從這個角度看看裁判的位置，實際他根本看不到梅里諾的推搡，就更看不到略倫特的拉拽了，他看到的是皮克的衝撞，所以VAR他也不看。根本不想給巴薩點球。

所以本場主裁羅哈斯，他在第一時間就認定皇家社會沒犯規，壓根也沒有去看VAR的打算，就是因為他的站位欺騙了他，二個可判罰的點，他都沒有看到，特別是皮克被推，他沒有看到，所以他認為略倫特背身，不可能將皮克拉倒，他也不看VAR。

可以說，這個球巴薩確實被冤枉了，很多西班牙媒體都認為皮克被拉倒應該判。但是實際皇家社會梅里諾這隱秘的一推，才是真正的罪魁禍首。
