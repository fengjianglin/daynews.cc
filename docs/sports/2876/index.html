6分鐘砍4分3助1搶斷！新疆後衛線現超新星 阿的江應多給他機會？

2019-12-03 0條評論 23次閱讀 0人點贊

摘要

上賽季，新疆男籃殺進了總決賽，但被廣東男籃橫掃。出局的方式極其慘淡，很大程度上是因為本土內線差距過大，新疆男籃在這個位置上被打得滿地找牙。曾令旭與王子瑞雖然都是前國手，但都弱點都過去明顯。曾令旭年齡大，速度不夠快，身體也不夠強壯，王子瑞則身體與技術都存在問題。

上賽季，新疆男籃殺進了總決賽，但被廣東男籃橫掃。出局的方式極其慘淡，很大程度上是因為本土內線差距過大，新疆男籃在這個位置上被打得滿地找牙。曾令旭與王子瑞雖然都是前國手，但都弱點都過去明顯。曾令旭年齡大，速度不夠快，身體也不夠強壯，王子瑞則身體與技術都存在問題。

恰恰，廣東男籃又儲備了一批本土頂級後衛。趙睿是國家隊主力，能依靠強壯的身體強攻，還有出色的三分球能力，防守端也是一把好手。胡明軒與徐傑也都是好手，速度快，防守兇狠，也有不俗的進攻能力。

差距在比賽中也著實顯現出來，總決賽中，王子瑞打不上球，曾令旭也是人見人欺。進攻中，曾令旭非但無法在趙睿面前得分，他連胡明軒與徐傑都突破不了。防守端，曾令旭又稱為突破口，屢遭欺負。

也正因為打得糟糕，曾令旭也被罵聲包圍，被批配不上高薪。對症下藥方能藥到病除，新疆男籃也對問題看得很清楚，休賽期重新簽回舊將西熱力江。西熱力江是新疆男籃原本的外線核心，既是防守大閘，也能充當進攻箭頭。

身強體壯，西熱力江速度快，爆發力也強，突破威力無窮。還有一手穩定的三分球能力，西熱力江的火力能匹敵絕大度數本土後衛。防守上，西熱力江更是一把好手，曾能很好限制馬布里等頂級外援，對付郭艾倫趙睿等本土巨頭也是不落下風。

本土後衛線大幅度變強，但對於新疆男籃來說還不夠，與廣東遼寧後衛群還存在差距。幸運的是，如今新疆男籃後衛線又浮現能挑大樑的人物，他就是小將劉力鵬。劉力鵬是新疆男籃從選秀大會上籤下的球員，是個沒有太多光鮮履歷的球員，但卻在比賽中展現出了極強的戰鬥力。

對陣福建男籃的比賽，劉力鵬僅打了6分鐘，但卻讓人眼前一亮。身高1.96米的劉力鵬在比賽中展現出了一流後衛水準，運球嫻熟，傳球視野極佳，能不斷傳出好球。僅打了6分鐘，劉力鵬就有3次進攻入賬。能傳還能攻，劉力鵬頻頻殺進禁區得分，還能出手三分球，拿到了4分。

防守端，劉力鵬也是一把好手，因為他既能跟上對手的腳步，也能利用身體頂住對手，防守技巧也是一流。在短暫的出場時間裡，劉力鵬很好的扮演了外線核心的角色，打得比老大哥曾令旭還靠譜。

能打出如此高水準著實難得，因為本場比賽之前，劉力鵬僅在對陣廣州男籃時出戰了4分鐘。給機會就能爆發，劉力鵬證明了自己有成為頂級後衛的潛質，也值得主教練阿的江大力培養。對於阿的江來說，他應該多給劉力鵬機會，爭取讓他在常規賽打出來，進而在季後賽舞台上扮演大角色。

面對廣東遼寧虎狼般的後衛群，新疆男籃不能坐以待斃，必須給予劉力鵬機會，從而避免在這個位置吃虧太多。
