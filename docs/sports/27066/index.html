溫格有些後悔在四年前，沒有親自點名讓克洛普接班自己執教槍手

2019-12-22 371條評論 1,906次閱讀 467人點贊

北京時間12月21日，溫格造訪利物浦下榻的酒店，與克洛普一起接受beIN體育採訪的部分內容，被公布了出來。在採訪中，溫格先祝利物浦在世俱杯中好運，克洛普也表示了自己以及他的球員們，會向著今年最後一個重要冠軍衝刺的。接著兩位知名主帥就相互調侃起來，由於克洛普剛與利物浦續約到2024年，所以這次訪談的重點內容，就是渣叔的續約。溫格扳著指頭算了算，他說如果克洛普在安菲爾德執教到2024年的話，那麼他就執教利物浦超過九年時間了。雖然這樣的執教年份無法超過溫格以及弗格森在各自球隊的執教時長，但是執教一支球隊、尤其執教是頂尖球隊九年時間，在歐洲主流聯賽中已經屬於罕見。

克洛普對此表示自己的壓力並不大，也不用擔心自己會中途下課，並且渣叔還反問：「你執教槍手也有戰績低迷的時候，你中途下課了嗎？」渣叔指的就是溫格帶領球隊「爭四」失敗的幾個賽季，好在槍手高層仍舊給予溫格足夠的信心，造就了兵工廠的溫格時代。所以克洛普認為自己也不用擔心「萬一哪一天在利物浦成績不好」的問題，因為他與球隊管理層之間已經建立了如同家人般的信任——這種主帥與高層的關係，在如今金元足球時代很難得。既然說到了槍手主帥的事情，溫格有些後悔地說道：「我現在有些後悔自己沒有在四年前，親自點名尤爾根（克洛普）接班我來執教（槍手）。是的，我真的有些遺憾。」

溫格的意思也很明確：如果教授當初點名克洛普作為自己的接班人，那麼現在槍手的主帥很有可能就是渣叔了。看看克洛普現在帶領利物浦的執教成績，如果他帶領兵工廠的話，至少現在槍手也不會落到「後溫格時代排在聯賽中游」的地步。現在的利物浦與槍手是「一個天、一個地」的狀況，不過有些新球迷們可能不知道，在四年前利物浦的狀況遠遠不如槍手，當時槍手還是穩定的歐冠球隊，而利物浦在羅傑斯執教的後期，只能爭取歐聯資格。而且當時兵工廠的陣容也是充滿著年輕且具有潛力的球員，畢竟溫格挖掘年輕球員的能力一向很強。而四年前的利物浦單從陣容上來看，並不是什麼強隊。

事實證明，克洛普完全有能力把「快要散架」的球隊，帶到歐冠冠軍，還有可能幫助利物浦實現英超冠軍「零的突破」。所以溫格有些後悔在四年前，沒有親自點名，讓克洛普接班自己執教槍手，不然的話兵工廠也不會是現在的這種狀況。這讓筆者想起了弗格森在退休的之前，也明示了高層：爵爺要瓜迪奧拉來曼聯接班自己。然而紅魔的高層就是不聽老爵爺的話，因為他們被弗格森「壓制」了太久，好不容易等到了爵爺退休，當然要按照自己的思路建設球隊。事實證明，當初曼聯高層沒有按照弗格森的要求選人，導致了紅魔在「後弗格森時代」陷入了不穩定之中。

槍手的「後溫格時代」實際上和當初曼聯經歷的「後溫格時代」是一樣的，如果溫格能夠在退休之前為高層指定自己的接班人，並且高層足夠聽話的話，現在的槍手會好很多。畢竟溫格與弗格森與「穆三年」、「瓜四年」等流動性的主帥不同，那兩位教父級別的教練已經成為了球隊文化的一部分，他們對球隊的了解遠遠超過了「沒幹幾年」的俱樂部管理層。現在溫格有些後悔在四年前，沒有親自點名讓克洛普接班自己執教槍手，說明了他了解兵工廠的狀況，當然槍手高層能不能「聽話」，就是另外一回事了。
