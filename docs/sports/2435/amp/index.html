相信大家都知道，乒乓球是我國的國球，在各大賽事中都能夠看到國乒的身影，尤其是國乒中有很多實力強勁的球員，張怡寧、王楠、鄧亞萍以及劉國梁等等都是大家非常熟悉的球員，在比賽中留下了很多精彩的瞬間，更是成為了球迷們最難忘的球員，也是一代人的回憶，他們為祖國帶來的榮譽非常的多，是大家的榜樣，也是國乒的大滿貫得主，很多球迷表示國乒的精神是非常值得學習的。

鄧亞萍、張怡寧以及王楠等等都是大家非常喜歡的球員，雖然退役了，但還是焦點運動員，尤其是當她們在役期間，很多球迷都將她們開始了比較，即使她們退役之後，還將他們子女們也開始了比較。其實運動員們天賦非常的高，她們的子女天賦也是相當的高，有一些已經是開始了專業的乒乓球訓練，尤其是在很多比賽中開始了嶄露頭角了。鄧亞萍是大家最熟悉的球員，在那個年代更是對手恐懼的運動員，被球迷們稱為了「初代大魔王」，他的兒子叫做林翰銘，在很小的時候就開始了乒乓球的訓練，在13歲的時候就成為了校隊的主力球員，之後更是拿到了北京隊的亞軍。

雖然林翰銘已經是拿到了北京隊的亞軍，不過鄧亞萍還是認為自己的兒子在乒乓球方面並沒有非常突出的天賦，與專業隊球員來說還是差得多，在之後應該也不會成為國乒隊的一員了。王楠的兒子叫做郭麒，在他9歲的時候就開始了乒乓球的一系列的賽事，而王楠對兒子的培養是非常的看重，並且也是自己上手，在訓練的時候對郭麒的態度更是非常的苛刻，這樣看來王楠是非常希望郭麒能夠成為下一個自己，所以說對兒子的培養是非常的用心，不會因為郭麒是自己的兒子而心軟。

張怡寧是大家非常熟悉的球員，在比賽中的表現更是非常的出眾，被球迷們稱為了「大魔王」，退役原因也是沒有可以媲美自己的對手，可想而知，張怡寧的實力是非常的強大，她在自己38歲的時候生下了女兒，叫做俏俏，她對自己的女兒也是非常的疼愛，在每次有比賽的時候，都會帶著俏俏前往現場觀看，也是非常希望自己的女兒能夠代替自己的位置。俏俏在4歲半的時候就開始了專業的訓練，張怡寧也表示俏俏在乒乓球方面的天賦是非常的高。

運動員們的子女都是焦點，他們不僅僅生下來就是網友們關注的對象，壓力也是與神俱來的多，大家認為張怡寧、王楠以及鄧亞萍的子女們誰是最有希望的球員？
