中國德比或上演！武磊與劉洋齊聚西班牙賽場，2大中資可創新歷史

2019-12-19 759條評論 1,836次閱讀 70人點贊

今日凌晨，西甲聯賽結束了一場焦點大戰，巴塞羅那對陣皇家馬德里，最終雙方0-0握手言和，12月20日凌晨2點，西班牙還將迎來一場中國球迷的焦點大戰，由於西班牙國王杯首輪比賽開賽，西班牙人做客挑戰西乙B的球隊萊里達體育，該場便將出現兩名中國球員，分別是代表西班牙人出戰的武磊、代表萊里達體育出戰的劉洋。

此劉洋非魯能劉洋，這位22歲的劉洋在2015年便開始了留洋職業生涯，起先加盟了薩瓦德爾俱樂部，後來才加盟了如今的萊里達體育，在出國前，劉洋曾是大連阿爾濱的一員，由於他是遼寧球員，在2017年還代表遼寧隊踢過全運會。

劉洋本賽季在西乙B賽場還沒有出場紀錄，從實力層面來講，他是很難獲得一線隊比賽機會的，但是萊里達體育也有中資背景，加上國王杯的練兵策略明顯，萊里達體育無論如何也是很難踢贏西班牙人的，而且劉洋已經在梯隊站穩腳跟，在這樣的情況下2中資可創造歷史讓武磊和劉洋提前上演中國德比，不需要等待韋世豪留洋西甲之後，劉洋在接受採訪時直言渴望與武磊同場對決，並且希望與武磊交換球衣。

根據相關歷史資料顯示，歐洲賽場的中國德比具有相當高的商業價值，在2003年，埃弗頓與曼城的一場比賽中，孫繼海和李鐵上演了中國德比，該場比賽的現場直播觀眾達到了3.5億，創造了足壇奇觀，足以證明中國德比的魅力之大，2019年年初武磊留洋踢比賽也引發了上億人的關注，反映了國內球迷對於留洋球員的追捧熱度一直未曾下降，此次中國德比拋開含金量不說，引發的關注度就能讓兩支俱樂部的商業價值得到快速提升。

武磊方面，他在最近一輪西甲得到了輪休的機會，此役國王杯出場基本沒有懸念，武磊與費雷拉、坎普薩諾、皮亞蒂、格拉內羅、納爾多等主力都進入了比賽大名單，馬欽渴望激活西班牙的鋒線火力，國王杯是很好的一次練兵機會。

按照西班牙人的比賽大名單分析，馬欽派出的352首發陣容基本浮現，武磊、坎普薩諾組成雙前鋒，格拉內羅、洛薩諾、伊圖拉斯佩組成三中場，皮帕和哈維-洛佩斯分居兩翼，納爾多、卡萊里、尤易斯-洛佩斯搭檔三中衛，門將迭戈-洛佩斯，由於西班牙人鋒線本賽季西甲17輪僅僅打進3球，武磊與坎普薩諾將面臨巨大挑戰，急需完成正名打進更多進球，助球隊走出保級泥潭。
