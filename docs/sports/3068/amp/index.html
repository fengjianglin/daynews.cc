世青賽女單四強打法百花齊放預示未來 國乒17歲直板閃耀削球復興

2019-12-01

吳洋晨

一天之內，石洵瑤、陳熠和蒯曼全部在1/4決賽輸球，世青賽女單四強僅剩下了吳洋晨一人。

吳洋晨是本屆世青賽女單頭號種子，是中國女隊不多見的直板選手，而且是兩面反膠，17歲的她目前是U18年齡段的世界第一，成年組排名第554位。

晉級路上，她4-0戰勝了朝鮮的邊宋京，4-2淘汰了日本的出澤杏佳，4-0擊敗了美國的王艾米。在世青賽的歷史上這是第一次有直板選手進入女單四強。半決賽，吳洋晨的對手是日本選手17歲的長崎美柚–日本世乒賽選拔賽第一階段的女單冠軍。長崎美柚和木原美悠這對女雙組合在巡迴賽上已經多次有所斬獲。

小鹽遙菜

四強中還有兩名削球選手，朝鮮的金恩宋和日本的小鹽遙菜將爭奪另外一個決賽名額。這也就是說，決賽中肯定會有一名削球選手。中國選手武楊曾是2009年的世青賽女單冠軍，如今時隔十年之後再次有削球選手將有機會衝擊女單的冠軍。

在奧運會的比賽中，還從沒有削球選手或者直板選手獲得過女單的冠軍，但世乒賽卻有。中國選手童玲是削球選手，她在1981年獲得了世乒賽的女單冠軍，而韓國直板選手玄靜和則是1993年哥德堡世乒賽的女單冠軍。

金恩宋是第一個在世青賽上進入女單四強的朝鮮選手，而日本上一次有選手進入女單決賽還是2010年，當時石川佳純輸給了朱雨玲。

在世青賽的歷史上，國乒從未丟過女單的冠軍，這也是國乒最王牌的項目。如今四強中僅有吳洋晨一人，壓力肯定是有，但作為同年齡段的世界第一，她還是具備衝擊金牌的實力的。

向鵬

男單國乒也僅有一人進入四強，16歲的向鵬承擔起了捍衛國乒榮譽的重任，向鵬半決賽的額對手是日本的戶上隼輔，另外一場半決賽，中國台北的馮翊新對陣瑞典的莫雷加德。
