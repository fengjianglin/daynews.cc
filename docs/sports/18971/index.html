4連勝！國乒最強黑馬選拔賽僅丟1局，爆冷張本智和並非曇花一現

2019-12-18 1,665條評論 2,581次閱讀 191人點贊

北京時間12月17日，國際乒聯總決賽順利落幕。目前，國乒隊內選拔賽正在進行。本次選拔賽有23名男單選手參賽，選拔賽排名前五將獲得明年一月份釜山世乒賽選拔賽的參賽資格，也就是地表最強十二人選拔賽。隊內選拔賽採用三局兩勝的賽制，比賽節奏很快。

國乒主力馬龍，樊振東，林高遠，許昕，梁靖崑，趙子豪等六人沒有參加本次選拔賽。國乒小將王楚欽目前已經被提前解禁，王楚欽選拔賽前六輪保持全勝，排名第一。直板小將薛飛同樣是六戰全勝，並列排名第一。此外，國乒名將孫聞目前比賽四輪，孫聞2:0戰勝隊友劉丁碩，2:0戰勝隊友徐晨皓，2:1戰勝張煜東，2:0戰勝夏易正，四戰全勝。

孫聞在今年日本公開賽爆冷戰勝日本名將張本智和，送其在主場一輪游。隨後，孫聞一度備受球迷關注，教練組也給了孫聞很多參賽機會。但是孫聞隨後並沒有太多出色的發揮，公開賽多次止步資格賽，有點曇花一現的感覺。但是孫聞實力絕不僅僅如此，乒超聯賽孫聞曾爆冷戰勝國乒絕對主力樊振東，孫聞遇強則強。但是本次選拔賽，孫聞不再曇花一現，可以成功打出來。

其實地表最強十二人選拔賽同樣備受球迷爭議，因為選拔賽要求國乒主力悉數參賽。但是像馬龍，許昕，劉詩雯，丁寧這樣的老將，均是有傷在身。連續高強度的作戰，對於今後備戰奧運會可以說是有害無益。此外，今年選拔賽還增設混雙項目，劉詩雯許昕要兼項作戰，強強對抗中，如果意外受傷後果無法想像。近些年，隨後日韓等選手的崛起，日本伊藤美誠，張本智和，韓國選手鄭容植，張宇鎮，中國台北名將林昀儒，德國名將奧恰，弗朗西斯卡等人頗有國乒主力的實力。所謂地表最強十二人，其實早已名不副實。
