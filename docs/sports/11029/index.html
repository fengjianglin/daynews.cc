叫板恆大！柔佛DT主帥目標小組出線，已開始對恆大詳細分析

2019-12-12 119條評論 1,838次閱讀 417人點贊

隨著2020年亞冠聯賽分組抽籤結束，各支已經取得亞冠正賽資格的球隊也確定了自己的分組，而中超冠軍廣州恆大，則將在明年亞冠小組賽中，遭遇韓國K聯賽勁旅水原三星，和馬來西亞球隊柔佛DT，和另外一支日本球隊。

​這對於廣州恆大來說，這個抽籤結果還是可以接受的，畢竟小組中還有一支並不被好看的馬來西亞球隊，主客場對陣柔佛DT的6分，顯然是勢在必得的，不過柔佛DT主帥並不認為他們將會成為亞冠小組中的「魚腩」，他對於球隊明年亞冠聯賽的前景依然很有信心。

​據馬來西亞媒體《hmetro》報道，在得知小組分組對手有廣州恆大和水原三星之後，柔佛DT主帥盧西亞諾-菲格羅亞也表達了自己的想法，他說當和亞洲強隊分在一起後，他們將會面臨艱難的比賽，但是他們會努力來爭取達成小組出線的目標。他們的俱樂部高層和教練團隊，已經開始針對小組對手恆大水原等隊進行詳細分析，評估之後柔佛DT全隊會更加有信心，亞冠聯賽一直以來都競爭非常激烈，但是相信他們的表現一定會比上一屆要更好，上一屆亞冠的經歷給了球員們提供了很多經驗，使他們提高應對亞冠比賽的能力，他們將能夠和強隊展開競爭。

​從柔佛DT主帥的發言來看，對於明年亞冠聯賽遭遇廣州恆大並不懼怕，並且已經做好了準備。而在2019年亞冠聯賽中，柔佛DT和中超球隊山東魯能分在了一個小組，雖然僅僅取得了1勝1平4負的成績小組墊底，但是與山東魯能的主客場比賽中，魯能贏得都非常艱難，如果不是佩萊在主客場的兩粒關鍵制勝球，山東魯能也很難在這支馬來西亞球隊身上佔到太多便宜。這支神秘之師在明年亞冠聯賽究竟會給廣州恆大帶來什麼樣的麻煩，也讓我們拭目以待。
