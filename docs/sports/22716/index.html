「西甲」巴薩VS皇家馬德里，三個原因保證巴薩不敗

2019-12-18 1,438條評論 2,406次閱讀 1207人點贊

北京時間明天凌晨，第243次西班牙國家德比將在諾坎普上演。在此前的242次交手中，巴薩以96勝51平95負稍佔優勢，尤其是上賽季連續在伯納烏取勝後巴薩87年來首次在國家德比的總戰績方面超越皇馬更是讓銀河戰艦十分不甘。另外雖然兩支球隊在積分榜上同積35分，不過綜合兩支球隊目前的實力和境遇來看，這次德比大戰坐鎮主場的巴薩或保持不敗延續戰績優勢。

一、阿扎爾、馬塞洛缺陣，皇馬左路缺乏活力

阿扎爾的受傷對於皇馬的影響顯而易見，比利時人受傷後的4場比賽皇馬總共打進了8球，而他受傷前的4場比賽銀河戰艦則是在4場比賽里有13個進球入賬。根據最新的消息顯示阿扎爾已經扔掉拐杖可以獨立行走並且能夠簡單進行恢復性訓練，不過顯然他將錯過自己的第一次西班牙國家德比。除了阿扎爾，馬塞洛的因傷缺陣對於皇馬影響也很大。在齊達內的戰術理念中，球隊在球場的一個邊路控球，然後通過迅速的傳導轉移在另一邊解決問題是中心的套路。由於馬塞洛出色的持球能力，皇馬的左路往往是進攻的發起點。不過由於巴西人和比利時人雙雙缺席比賽，齊達內將不得不選擇更加保守的策略來應對客場的比賽。從數據方面來看，阿扎爾本賽季只有1個進球和2次助攻，但是他的突破和串聯給球隊帶來的幫助不能僅僅因為數據就被抹殺。馬塞洛在受傷之前已經奉獻了4次助攻，巴西人在進攻端的作用顯而易見。

阿扎爾和馬塞洛缺陣，齊達內極有可能會選擇伊斯科和門迪來取代他們的位置。在對陣巴黎的比賽打出高光一戰之後，伊斯科的信心和狀態都迅速恢復。上輪比賽齊達內將他安排在左路進行戰術試驗，安達盧西亞人表現並不算亮眼。不過要想在諾坎普穩紮穩打，伊斯科的纏鬥能力仍是皇馬目前最需要的要素之一，因此齊達內選擇伊斯科首發的可能性應該要大於維尼修斯。左後衛位置上門迪近期比賽進步明顯，法國人出色的跑動能力使得他在攻防兩端都相對均衡，但是進攻能力和處理球的合理度相對馬塞洛還有明顯不足。首次國家德比經歷對於任何球員來說都不容易，年輕的門迪還需要做好睏難準備。

二、三叉戟狀態火熱，MSG皆為皇馬苦主

巴薩羅那上輪比賽在皇家社會身上收穫1場平局，如果不是皇馬也沒能戰勝瓦倫西亞那加泰羅尼亞球隊將在國家德比到來前丟掉積分榜領頭羊的位置。儘管球隊在客場盡顯疲態，不過梅西、蘇亞雷斯和格列茲曼的三叉戟組合還是展現了火熱狀態。三名頂級球員在短短12分鐘內的提速就讓巴薩迅速扭轉局勢，如果不是領先後球隊收縮陣型，梅西、蘇亞雷斯和格列茲曼很有可能在比賽中還有斬獲。C羅遠走亞平寧之後，銀河戰艦在進攻端過度依賴本澤馬的個人能力，貝爾對球場缺乏專註度，顯然威爾士人不是這次國家德比的主角。而巴薩這次，M-S-G的攻擊組合本賽季截至目前已經合力打進了33個進球還有18次助攻。

另外從三叉戟的個人數據來看，皇馬也是他們非常喜歡的對手。梅西對陣皇馬時的高光已讓人習以為常，尤其是在C羅離隊後梅西成了國家德比中唯一的超級巨星，他也將是皇馬最大的威脅。另外皇馬也需要提防蘇亞雷斯和格列茲曼。儘管烏拉圭人近兩個賽季狀態有所下滑，但是皇馬仍然是他最喜歡的對手之一，過去5次在諾坎普面對皇馬，烏拉圭前鋒打進了6球。尤其是上個賽季在狀態一般的情況下，蘇亞雷斯在主場的國家德比中上演帽子戲法扭轉自己的頹勢，皇馬的後防需要謹慎對待蘇牙。另外格列茲曼也將迎來自己的首次國家德比，在經歷了一段時間的磨合後近期法國人與梅西、蘇亞雷斯的配合默契度在上升，過去兩輪聯賽他也均有進球入賬。在過去25次對陣皇馬的比賽中，格列茲曼打進了8個進球還有5次助攻。對於皇馬來說，M-S-G都是苦主，球隊的後防線需要打起十二分精神來應對。

三、6場不勝，巴薩成皇馬最大剋星

自里傑卡爾德時期開始，巴薩漸漸成為讓皇馬最頭疼的對手，過去十幾年的交鋒中，皇馬球迷很難忘記羅納爾迪尼奧在伯納烏千里走單騎、6比2刺眼的大比分以及皮克搖動的5個手指這些場面。而據數據統計顯示，自梅西首次參加國家德比以來，雙方的交手記錄為巴薩20勝11平13負。儘管在穆里尼奧後期和齊達內一期工程時皇馬一度扭轉局勢並連續扳回幾局，但是上個賽季皇馬的低迷還是幫助巴薩延續了國家德比的信心。雙方過去6次交手，巴薩取得4勝2平的不敗戰績，打進15球僅僅丟了4球，優勢明顯。另外C羅離隊後的4場較量，皇馬只有2球入賬。葡萄牙人在9年的皇馬經歷中對陣巴薩總共打進了18個進球，其中更有連續6場給對手傷口撒鹽的精彩發揮，後C羅時代皇馬還需要好好解決進球的問題。

另外從雙方近期戰績來看，巴塞羅那過去20個聯賽主場取得了18勝1平1負的出色戰績，過去10個聯賽主場更是取得10連勝。而皇馬過去20個聯賽客場只有50%的勝率，他們取得了10勝6平4負。而近10個客場，齊達內的球隊只有4勝3平3負，這10個客場比賽中，皇馬只有14個進球入賬，場均不到2球。客場攻擊乏力，這是目前這支銀河戰艦面臨的最大難題。

從多個角度來看，西班牙國家德比巴塞羅那都更有優勢。不過齊達內從來不會做任人宰割的羔羊，過去4次作客諾坎普，法國人帶走2場勝利2場平局保持不敗，或許這可以給皇馬些許信心。另外，布斯克茨、德容和拉基蒂奇組織的中場目前還存在不小的問題，這也是皇馬可以利用的要素。如果齊達內的球隊提高反擊效率在攻擊端早早有所斬獲，或許比賽還有更大看點。

競彩推薦：勝

亞盤推薦：巴薩-0.5/1

比分推薦：3:1
