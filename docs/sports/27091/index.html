拉文33+5+5公牛險勝送活塞四連敗，德拉蒙德19+14

2019-12-22 325條評論 3,306次閱讀 1795人點贊

NBA常規賽繼續進行，公牛客場119-107勝活塞，活塞遭遇四連敗。

薩托蘭斯基開局就中三分球再有上籃之後再中三分，公牛打出10-2的開局。肯納德和莫里斯各中三分，布朗連續上籃追上比分後雙方交替得分，首節活塞28-25領先。第二節賽迪斯-楊命中三分後公牛反超了比分，弗雷澤、阿奇迪亞克諾和加洛維各中三分後楊和鄧恩也中三分，雙方仍然咬住比分。節末莫里斯連中三分，半場活塞59-53領先。

第三節斯內爾和肯納德各中三分球，卡特連續上籃得分後薩托蘭斯基完成扣籃，公牛反超了比分。卡特隨後命中三分，德拉蒙德回應2+1後拉文和懷特也中三分。加洛維打成3+1但懷特又連中三分球，節末又是莫里斯連中三分縮小分差，三節過後公牛93-89領先。

第四節懷特和莫里斯各彪三分後德拉蒙德上籃追平比分，拉文連續三分球回應，馬爾卡寧連續上籃。弗雷澤補籃止血，但最後兩分鐘拉文再中三分將分差拉開到12分，比賽就此失去懸念。

全場比賽拉文得到33分5籃板5助攻，薩托蘭斯基16分8籃板5助攻2搶斷，鄧恩9分4助攻4搶斷，馬爾卡寧15分7籃板，溫德爾-卡特12分13籃板，科比-懷特19分。活塞馬基夫-莫里斯得到23分，德拉蒙德19分14籃板5搶斷2蓋帽，肯納德16分，斯內爾11分，弗雷澤13分9助攻，加洛維10分。
