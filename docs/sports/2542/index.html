32-5，湖人又來了，比賽還未結束，博揚和戈貝爾便提前走回更衣室

2019-12-06 0條評論 24次閱讀 0人點贊

摘要

121-96，湖人大勝爵士，送對手3連敗。比賽一上來，博揚-博格達諾維奇，英格爾斯連中三分球，率先幫助球隊打開局面。不過，湖人也不示弱，濃眉哥和麥基連續內線砍分，格林續上火力，飈進兩記三分球，湖人內外火力全開，分差很快來到兩位數。

121-96，湖人大勝爵士，送對手3連敗。比賽一上來，博揚-博格達諾維奇，英格爾斯連中三分球，率先幫助球隊打開局面。不過，湖人也不示弱，濃眉哥和麥基連續內線砍分，格林續上火力，飈進兩記三分球，湖人內外火力全開，分差很快來到兩位數。隨後，詹姆斯在添一把火，秒傳隆多投進三分，自己又打成2+1，半場結束時分差已經來到18分之多。雖然下半場米切爾連中三球，將比分縮小。但是湖人的進攻實在是太猛了。詹姆斯在弧頂同樣的戰術，給波普連著喂球，後者連得8分。緊接著，濃眉哥大開殺戒，分差來到20分開外。即便格林兩記三分及時止血，但隆多又開始了，又是秒傳，又是連中投籃，一頓操作直接宣布比賽結束。

數據方面，詹姆斯20分12助攻，濃眉哥26分6板，波普14分，霍華德7分9板3帽，隆多14分9板12助攻，庫茲馬13分4斷。爵士這邊，米切爾29分4板5助攻，戈貝爾13分10板，博格達諾維奇23分。有一說一，湖人現在不光防守好，這進攻屬實厲害。之前麥基投進三分，現在霍華德投進三分，隆多默默放下戰術板，來，帶我一個。卡魯索和格林這邊，你們都投三分了，那好吧，「禿」進去就是扣。漂亮活都被分完了，到庫茲馬這裡，說我防守不好，蓋給你看，說我進攻沒狀態，那就秀一手壓哨三分，都給我吹爆。看到這樣的情景，三個狀元在替補席隨即拿起薩克雷的毛巾，那就是一個蹦迪現場啊。而就在此時，波普帶著墨鏡走來了，樂什麼樂，誰是取勝關鍵，你們心裡都沒點數嗎？

反觀爵士就慘咯，今夏引援相當難受，他們換來了康利，本以為他會比盧比奧做的更好，會徹底釋放米切爾的能量，與西部頂級後場組合，板板手腕。結果，康利到了爵士水土不服，投籃命中率至今只有37.0%，生涯最低，要知道這可拿著3251萬的年薪啊。當然了，這一場比賽，康利休戰，爵士暴露出的問題，不單單只是康利投籃投不進。他們的攻防體系相比上個賽季，進攻還是那樣，沒有太大起色，都說米切爾有些頭鐵，但看看爵士現在的狀態，米切爾太難了。而防守就更是天差地別了，上個賽季爵士的防守效率一度聯盟第一，但這個賽季，別的不說，四號位幾乎就是交學費狀態。值得注意的是，32-5，湖人又來了，比賽還未結束，博揚和戈貝爾便提前走回更衣室

在這場比賽中，湖人快攻拿到32分，而爵士只有5分。其中，很多時候並不是那種失誤後，湖人一條龍直接得分。而是，爵士四號位實在是防不了。最明顯的例子就在第三節，米切爾造成犯規，兩罰全中，湖人底線發球，詹姆斯也不是很快一條龍，去刻意搶轉換，而是控球到中場，直接掉給濃眉哥，後者反籃上進。還有就是濃眉哥在45度角附近給波普擋拆，詹姆斯在弧頂拿著球，波普直接空切到籃下，同樣的戰術連打爵士兩次。除了四號位防不了，連著被字母哥，西亞卡姆，以及這場的濃眉哥打爆之外，戈貝爾沒有自主進攻能力的漸漸凸顯出來。此前，他連著拿到最佳防守球員，自己委屈沒進全明星。其實，想想也是情理之中。

在小球時代，他沒有自主進攻能力，說他防守好，但到了季後賽，保羅小打大，把他針對的面無全非。如今，盧比奧走了之後，沒人給他毫無保留的喂球，內線又缺少幫手，攻防兩端都打的掙扎。當然了，還有英格爾斯。想當初季後賽，他打雷霆的時候，那個三分叫一個準啊。但這個賽季，三分球命中率只有31.0%，可以說掉出投手行列了。在距離比賽還剩9秒鐘的時候，隆多控球耗時間，博揚和戈貝爾先後從替補席站起來，提前走回更衣室，沒辦法，近6戰5負，連著輸給雄鹿、步行者、猛龍、76人、湖人，難受。實話實說，就目前來看，爵士需要一次改變。克勞德的離開，讓他們少一個聯盟頂級3D球員。而英格爾斯攻防不在線，博揚頂四號位，進攻可以，防守太難受了。即便康利狀態回歸，等到了季後賽，這鋒線也只能是交學費了。
