姆巴佩21歲時已打入103球，然而一巨星相同年齡進球比他還多12個

2019-12-22 312條評論 1,490次閱讀 522人點贊

12月20日，法國球星姆巴佩迎來了自己21歲的生日。據統計，到目前為止，姆巴佩已經在正式比賽中打入了103個進球，那麼在相同的年齡，他是不是眾多球星中進球最多的呢？

西班牙一家專業的足球數據公司統計了各大球星到21歲時的正式比賽進球，結果發現姆巴佩的進球並不是最多的。21歲時進球最多的是羅納爾多，巴西人當時已經打入了115個進球。

羅納爾多在21歲時已先後為巴西克魯塞羅、荷蘭埃因霍溫、西班牙巴塞羅那效力，他在埃因霍溫就打入了54球，在巴薩也打入了47球。

當然姆巴佩的進球也不少，他到目前為止已經打入了103球，其中在摩納哥打入27球，在大巴黎打入76球。

排在第三的是內馬爾，這位巴西球員在21歲時打入了80球。第四是阿圭羅，21歲時打入78球。盧卡庫和勞爾並列第五，兩人21歲時都打入了73球。

接下去是21歲時打入57球的本澤馬、44球的萊萬多夫斯基。值得一提的是，梅西排名第九，21歲時打入了42球，而伊布是31球。C羅在21歲時進球不多，只打入了27球。葡萄牙巨星在加盟皇馬後進球數才大幅提升。
