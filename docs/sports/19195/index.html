"宏建杯"中外拳王爭霸賽落幕！中國巨人董建軍復出完勝對手

2019-12-16 163條評論 4,363次閱讀 1826人點贊

12月15日，”宏建杯”中外拳王國際爭霸賽在甘肅蘭州落下帷幕，這是蘭州50多年來水平和規格最高的一場格鬥比賽，參賽的外方拳手來自八個不同的國家，都是金腰帶級別的拳手，這場厚重的西北大戰讓”一帶一路”上的絲路重鎮蘭州煥發了生機。

本次比賽由蘭州宏建建業集團有限公司主辦，甘肅省秦人體育文化傳媒有限公司承辦，甘肅省社會體育管理中心、蘭州市體育局作為指導單位，同時也得到了甘肅省搏擊協會、蘭州國資利民資產管理集團有限公司、蘭州益民出行汽車服務有限公司等多家單位的大力支持。

本土作戰的董建軍的比賽可謂當晚的重頭戲，董建軍一出場便受到了家鄉父老潮水般的掌聲和歡呼聲，他也不負”中國格鬥第一巨人”的名號，重拳如鐵鎚般砸向對手，德國悍將里諾的搖閃雖然也很精妙，但是泰山的壓迫感和力量十分驚人，一度將對手壓制在繩邊並打得他搖搖欲墜，在第四回合給了對手一個讀秒，最終拿下了回國後的首場勝利，接下來董建軍會有哪些動作呢？我們十分期待。

聯合主賽在焦富開和卡洛斯之間上演，面對身高和體重都佔據優勢，且擊敗過多位中國拳手的卡洛斯，焦富開打得很艱難，他的進攻無法撕破對手的防線，反而多次受到卡洛斯刺拳和後手重拳的打擊，三局戰罷卡洛斯點數擊敗焦富開，這是焦富開首次升到77公斤級作戰，還需要多多適應。

當晚有一場火光四濺的對決，來自日本拳手平塚大士和蒙文佩之間，日本拳手平塚大士開場不久脛骨便因為多次掃踢而皮開肉綻，鮮血直涌，看得觀眾一陣揪心，但是平塚大士絲毫不在意，依然用受傷部位猛擊對手，配合不俗的拳法，最終拿下這場比賽，頑強的品質給大家留下深刻印象，也贏得了全場的掌聲。獲勝後他拿出了母親的照片，原來他每次出戰都會帶著去世母親的遺照來鼓勵自己，現場觀眾再一次被感動和圈粉。

“拳擊小湯圓”和哥哥”黑金剛”的比賽也是昨晚的焦點之戰，兩位小拳手一出場就引起了大家的興趣，而小湯圓在台上靈活的搖閃和精準的打擊更是換來了現場觀眾的掌聲，這場拳擊表演賽以平局收場，不過大家卻在他們身上看到了”少年強則中國強”的希望。

賽前我們採訪宏建集團董事長田炫宏，從小就參加學校武術隊的他在事業有成之後，希望通過格鬥賽事的舉辦，讓政府能夠重視體育，重視全民健身，同時將甘肅蘭州推向世界，昨晚的比賽拳手們表現很出彩，我們希望中國的拳手可以在世界上綻放光彩，也希望通過格鬥將我們的心連接在一起。
