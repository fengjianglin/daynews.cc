十年超長待機！梅西1特質已超越老馬比肩貝利 直追喬丹+網壇三巨

摘要

在2019年金球獎頒獎典禮上，梅西收穫了職業生涯第6座金球獎。西班牙媒體《馬卡報》撰文稱讚道，梅西的六座金球獎是前人無可比擬的成就，他的超長巔峰期已超越足壇一眾傳奇，可以比肩貝利，在全世界體壇範疇內直追喬丹和費德勒。

在2019年金球獎頒獎典禮上，梅西收穫了職業生涯第6座金球獎。西班牙媒體《馬卡報》撰文稱讚道，梅西的六座金球獎是前人無可比擬的成就，他的超長巔峰期已超越足壇一眾傳奇，可以比肩貝利，在全世界體壇範疇內直追喬丹和費德勒。

此次獲得金球獎距梅西贏得他的第一個金球獎已經十年了，周一晚上他在巴黎慶祝了他的第六個金球獎。在經歷了C羅和莫德里奇先後獲得這個獎項之後，32歲的梅西再度舉起金球獎，他仍然是足球運動的王者，皇冠只屬於梅西。

在梅西的天賦面前，所有對手和隊友無不認可他出眾的實力。梅西的比賽是獨一無二的，他每天都能使你感到驚訝。梅西的六座金球獎更是前人無可比擬的成就。這不是別人贈予的獎盃，這是梅西靠自己的天賦和努力而取得的，他當之無愧。

在一項競爭激烈的職業運動中，連續保持十多年的巔峰狀態是非常困難的。在足球世界中，這尤其困難。1953年皇馬傳奇球星斯蒂法諾在27歲的時候才加盟皇馬。他的第5次歐冠冠軍獎盃是在1960年獲得的，這也是他最後一次獲得該項殊榮，當時斯蒂法諾已經34歲，處在職業生涯的末期。

馬拉多納離開巴薩以後才取得了偉大的成就，他在那不勒斯踢的呼風喚雨，並在1986年帶領阿根廷獲得了墨西哥世界盃的冠軍。四年後的義大利世界盃結束時，馬拉多納已經30歲了，雖然他已經證明了自己在當時的足壇像上帝一般存在，但他的巔峰已經過去了。

出道於阿賈克斯的荷蘭傳奇人物克魯伊夫在1974年收穫了個人最後一個金球獎，當時他才27歲，但是之後此項榮譽便和他沒有關係了。

球王貝利在1958年贏得了個人的第一座世界盃冠軍，而十二年後，在墨西哥的阿茲台克球場，貝利贏得了個人的第三座世界盃冠軍，那也是他的最後一座世界盃冠軍獎盃，當時貝利30歲。雖然在37歲復出時仍有不俗成就，但紐約宇宙隊所提供的舞台遠不如世界盃。

不局限在足壇，梅西職業生涯所獲得的成就即使放在整個體壇也是無可比擬的。榮譽無數的藍壇傳奇人物喬丹在28歲時才獲得了第一個NBA總冠軍，隨後開啟第一波三連冠，在首度退役一年半之後，喬丹復出並又獲得了另外三個冠軍。

還有網壇三巨頭，傳奇人物費德勒在2003年獲得溫布爾登網球公開賽冠軍後，在14年後仍然還能在此項賽事奪冠。納達爾在法網的紅土賽場上更是獲得了12次冠軍，時間跨度從2005年一直到2019年。德約科維奇則是在2005-2019年間七度問鼎澳網。

無論梅西的巔峰期還能持續多久，他在足壇已經做到了獨一無二，是時候代表足球去和體壇眾多巨頭一較高下了。
