海淘對比找不同，安切洛蒂適合阿森納嗎？

2019-12-12 204條評論 1,314次閱讀 114人點贊

對於選擇在此時入主酋長球場的教練來說，後溫格-埃梅里時代的阿森納既暗藏著致命荊棘，也充滿了無限機遇。一方面，歷經連續兩個夏窗的大規模清洗與引援，阿森納現階段的陣容成熟穩定，但被寄予厚望的球星們並沒有拿出力能扛鼎的表現，年輕球員實力不俗但天賦上限很難達到爭四/爭冠要求，這班人馬需要高水平的戰術設計和管理手段才能達成團隊效應。即便高層有意改變經營策略投入軍備競賽，阿森納在近年來落下的功課也不可能在一兩個轉會窗內就補錄完畢，大量人員更迭帶來的動蕩曾讓無數豪門陷入陌路迂迴，埃梅里的前車之鑒足以引人深思。

另一方面，阿森納在本賽季的成績已經刷新了多年來的下限，那麼在如此之低的起點上開始工作無疑會大大減輕主帥的壓力。輿論普遍對新時代的拓荒者抱有一種寬容的態度，他只需安靜地領導球隊完成重建，甚至可以將首個賽季作為邊打邊學的「實習期」來適應環境、謀劃藍圖，槍手高層和球迷對主帥態度比較寬容，球隊擁有大筆流動資金可以在未來的夏窗補強陣容，這是一線主帥理想的執教平台。

拔擢沒有頂級聯賽執教經驗的新人還是有很大風險的，瓜迪奧拉和齊達內的故事很難複製，盲目信任名宿而陰溝翻船的案例倒是不勝枚舉，永貝里極有可能只是擔任過渡者的角色。高層還是希望邀請一線名帥前來執教，在相繼考察了勒夫、阿萊格里、雅爾迪姆、努諾-桑托和阿爾特塔等人後，剛剛在那不勒斯下課的安切洛蒂似乎已經取得了「桿位」。

【勒夫：冠軍教頭，可遇而不可求】

新時代的英超聯賽雲集了眾多大陸名帥和球星，戰術知識的更新周期逐漸縮短，具有開創精神的打法和用兵套路層出不窮，強強對話中見招拆招的博弈令人眼花繚亂，這對豪門主帥的理論功底、應變能力和資源整合能力提出了很高的要求。作為風格老派的英式Manager，溫格在其執教後期的表現距離曾經的戰術大師越來越遠，其相對粗放式的人才培養模式能夠有效地提升球員的下限，但卻無法幫助他們在成名之後更進一步；埃梅里擅長變陣，但過多的試錯沒能帶來戰術紅利，反而造成了混亂，西班牙人似乎還適合執教所謂的中產球會。

勒夫一直在等待拜仁和皇馬的邀約。

在一眾熱門候選人中，堪稱戰術大師的無疑就是勒夫，他在履歷、業務能力、影響力還是實戰經驗最方面都堪稱一流。球員生涯黯淡無光，從教後躬耕基層逐步積累，穩紮穩打進而攀上事業巔峰，勒夫的成功從一個層面反映了社會分工日益細化的潮流趨勢對足球產業的重塑和再造。

新時代的足球教練工作更加依賴於戰術建設和數據分析，那些「擁有三流身體條件和一流大腦」（克洛普原話）的學院派主帥更容易得到出人頭地的機會，勒夫就是這樣的能夠穩坐中軍、決勝千里的新型主帥。當初正是憑藉「在1分鐘內闡述四後衛防守體系要義」的表現，勒夫贏得了時任德國足協主席馬菲爾德和國家隊主帥克林斯曼的青睞。

勒夫曾引領了足壇的少帥風潮。

長期執教一支球隊並取得過輝煌成就的主帥很容易陷入固步自封的境地，溫格在2005/06賽季之後的表現便是明證。執教德國隊已經超過10年的勒夫也是當今足壇服務單一球隊時間最長的主帥之一，但他卻沒有受到過這種晚期風格的折磨。勒夫最值得稱道的一點就是沒有冠軍主帥的架子，能夠虛心聽取外界的建議，敢於推翻過往不斷進行再造和重建，勇於改變且能夠虛心聽取外界的建議，在陣型和戰術選擇以及人員搭配方面不拘一格、與時俱進，頻繁啟用新人試錯而不計較短期利益，工作思路既不保守、也不僵化。如果勒夫能夠在2020歐洲杯後選擇前往俱樂部接受挑戰，阿森納無疑是一個不錯的選擇。不過，北倫敦球隊似乎已經沒有繼續等下去的耐心。

【阿萊格里：金牌教頭，資源整合高手】

勒夫在大賽決戰期的表現難稱完美，阿萊格里卻是兩回合淘汰賽制的專家。從執教履歷、戰術思路和用兵習慣上來看，阿萊格里和溫格的老對手弗格森極為相似。在球隊主力陣容連年失血、中軸線兩次全面翻修的情況下，阿萊格里依靠出色的資源整合能力維持著球隊的競爭力，他的陣型魔術和臨場用兵令人目不暇接。在這個球員忠誠度很低、流動性較大的時代，阿萊格里就是各大豪門趨之若鶩的球隊核心和靈魂。

即便不會像弗格森那樣頻繁開動吹風機，但阿萊格里對更衣室的管控和不穩定因素的排查也是細緻入微、一絲不苟的。阿萊格里在接受記者採訪時總是妙語連珠，但他不是溫格那樣的「慈父」，也是埃梅里那樣的「忍者」，他會因為球員不能很好地執行戰術而在場邊暴跳如雷，也會毫不客氣地彈壓那些不服從管教的功勛老臣，當年「敲打」博努奇就是明證。對於已經習慣了在溫室里成長的阿森納球員來說，阿萊格裡帶來的改變將是徹頭徹尾的，唯一影響義大利人與槍手合作的因素或許就是兩年前面試的那段不愉快的經歷。

阿萊格里整飭更衣室的力度很大。

【雅爾迪姆＆努諾-桑托：中產名帥，上限有限】

成為像溫格一樣長期掌控英超豪門的Manager是很多教練的畢生夢想，這是比誘人的高薪和火爆的競技氛圍更能吸引教練精英落戶北倫敦的籌碼。除了制定戰術及臨場指揮，溫格還在球員工資和青訓建設方面等方面擁有很大的權力，在加齊迪斯＆米林斯塔特軸心形成之前，教授也是全權負責轉會運作和球探體系。在溫格卸任後，阿森納將會像其他英超俱樂部一樣開始依靠體育總監/經理、專業技術小組或是轉會委員會來專門負責引援操作，具備看人下菜能力、能夠適應外向型經營策略的主帥將有希望成為槍手高層青睞的目標，雅爾迪姆和羅傑斯就是這種類型的教練。

以穆里尼奧為榜樣，葡萄牙湧現出了一大批風格務實、成名很早的新銳教頭，本賽季活躍在英格蘭賽場的馬爾科-席爾瓦和努諾-桑托便是個中翹楚，在摩納哥有過輝煌戰績的雅爾迪姆也是如此。雅爾迪姆被部分法國媒體比作穆里尼奧二世，但如果仔細考量其在宏觀戰略構思以及用人等方面的表現可知，摩納哥主帥與溫格倒是有很多相似的地方。他們都在摩納哥取得過成功，也都培養出了大量的青年才俊，他們都不僅僅只是戰術專家，也都是技術、體能或心理專家，他們都擅長足球將作為一個整體來理解並對所有各方面的變數都加以綜合利用。

雅爾迪姆執教生涯五次更換門庭，迄今為止還從未令東家失望過。從布拉加和里斯本競技，到奧林匹亞科斯和摩納哥，他每到一處總能幫助球隊轉危為安，阿森納需要的正是這種適應能力超強、可以快速上手的教練。

努諾-桑托的風格與穆里尼奧比較相似。

努諾-桑托和雅爾迪姆一樣擅長打造整體足球，他們都能夠及時有效地發現球隊的問題並拿出富有創意的解決方案，在和球員交流和控制更衣室方面也都做得不錯。努諾-桑托擁有豐富的英格蘭聯賽執教經驗，不存在水土不服的問題，他不是帶領豪門重建的理想人選，但卻能在過渡時期幫助球隊完成軟著陸。

潛力不錯的少帥也是阿森納重點考察的目標。

【納格爾斯曼＆泰代斯科：潮牌少帥，長線戰略】

早在兩年之前，溫格曾有意將執教理念相似的埃迪-霍維選定為繼任者，教授為此還專門將威爾希爾作為「考察作業」租借給伯恩茅斯。近年來，歐洲足壇出現了一大批這種名聲在外但尚未取得實質榮譽的少帥，儘管他們總能出現在各路豪門的選帥名單上，但豪門在選擇他們時卻會慎之又慎。貿然押寶少帥很可能落得滿盤皆輸的下場，胡安德-拉莫斯、博阿斯和克林斯曼的故事恍如昨日，圖赫爾、羅傑-施密特和尼克-科瓦奇等人皆屬於此類，最近加入這個名單的是納格爾斯曼和泰代斯科。

就在其他聯賽的球迷還在稱呼年齡40+的主教練為「少帥」時，德甲賽場已經湧現出了一大批剛過而立、與場上球員年齡相仿的主帥，納格爾斯曼和泰代斯科就是這樣的天才。在執教沙爾克04期間，見多識廣的老將納爾多就認為泰代斯科具備令人驚嘆的知識儲備和探索精神，像極了當年執教巴薩B隊時期的瓜迪奧拉，他必然會在未來獲得執教豪門的機會。

可以用「30%的時間關於戰術制定，70%用於社交」來概括納格爾斯曼的工作狀態，泰代斯科雖然更加註重戰術研究，但同樣也會花大量時間用來進行球員的心理建設，他擅長通過提出許多問題來幫球員們去找到答案，這種以人為本的執教理念非常適合大家庭式的阿森納俱樂部。如果兵工廠最終選擇納格爾斯曼或泰代斯科，那便意味著球隊將專註於長線戰略，兩名年輕教練將和球隊共同成長。

【永貝里＆阿爾特塔：教壇新星，壓力山大】

以穆里尼奧帶領波爾圖問鼎歐冠為起點，新時代的足壇湧現出了大量擅長鑽營數據和戰術的學院派主帥，甚至還出現了薩里這種足球愛好者出身的高級發燒友執教豪門的案例。同學者型主帥的扎堆湧現形成鮮明對比的是，退役後拿起教鞭的明星球員執教失敗的案例比比皆是。由於聯賽競爭激烈且容錯空間較小，「踢而優則教」的例子在英超並不多見，索爾斯克亞的帥位已經岌岌可危，蘭帕德的成色如何還需要時間檢驗。

當然，永貝里和阿爾特塔失去了這次機會未必就是壞事。在這個普遍流行通過換帥這種「休克療法」來解決危機的時代，成績的壓力讓俱樂部難以保持對年輕人的耐心，能夠在失敗後愈挫愈奮並最終功成名就者屈指可數，只有耐心地進行沉澱和積累才能在一線隊激烈的競爭中站穩腳跟。

埃梅里擅長資源整合，安切洛蒂在這方面的能力更勝一籌。

【安切洛蒂：英超故人適合槍手】

自帶領AC米蘭完成「中興」開始，安切洛蒂開始了遊歷四方的執教之旅。儘管在幾大豪門的結局都令人唏噓不已，但不可否認的是，安切洛蒂所到一處總能迅速完成資源整合，幫助球員們兌現天賦，取得超出輿論預期的成績，這種素質正是埃梅里所不具備的。經過近兩三個窗口的集中採購，阿森納隊內雲集了一大批頗具實力的球員，球隊欠缺的是穩定的陣型架構和富有針對性的戰術部署，為球隊構建藍圖正是安切洛蒂的強項。

兩個賽季之前，安切洛蒂就曾出現在阿森納的選帥名單上。

安切洛蒂能夠與球員打成一片，為此他不惜得罪了弗洛倫蒂諾和德勞倫蒂斯等大佬，與貝盧斯科尼和阿布拉莫維奇也曾就戰術和引援問題發生過爭執，但在高層很少插手球隊競技事務的阿森納，義大利人或將獲得較大的轉圜空間，避免執教拜仁和那不勒斯時期的悲劇。安切洛蒂曾在英超取得過巨大成功，雖然藍衫之旅已經過去了十年，但其靈活多變的戰術風格依然足以應付如今「名帥錦標賽」的要求，義大利人與槍手的合作值得看好。

【總結】

即便不具備歐陸頂尖豪門的那種向心力，但阿森納依然具備吸引一線名帥落戶的資本。良好城市地緣優勢帶來的影響力和曝光率，健康的財務狀況和出眾的購買能力，夯實的青訓基礎和不錯的一線隊陣容，相對溫和的管理層以及較為寬鬆的工作環境，以上優勢項目的加成意味著阿森納就是當年最適合名帥大展宏圖的執教平台之一，俱樂部在選帥方面握有很大的主動權。

不同類型的教練適合不同的路線、計劃和戰略，相似的藍本在不同的執行者手上會產生截然不同的效果。安切洛蒂的履歷無需質疑，看人下菜的能力依然是當世頂尖水平，義大利人是豪門需要的那種「牌面」。當然，阿森納在現階段的當務之急並不是依靠金牌教頭去爭取榮譽，而是應該儘快確立未來的體育方向和建隊計劃，制定好復興路線圖再按圖索驥，盡量重蹈曼聯在後弗格森時代的覆轍。
