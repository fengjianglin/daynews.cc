收官之戰！塞爾比9:6奪冠，兩站本土冠軍赫恩能給50萬磅？真敢想

2019-12-16 1,200條評論 3,909次閱讀 1090人點贊

蘇格蘭公開賽作為2019年最後一站排名賽，以塞爾比9:6擊敗利索夫斯基奪冠落下帷幕。這是塞爾比繼英格蘭公開賽奪冠後，本賽季第二個排名賽冠軍，同時也是本賽季「本土系列賽」第二個冠軍。

眾所周知，英格蘭公開賽、北愛爾蘭公開賽、蘇格蘭公開賽、威爾士公開賽被稱為4大「本土系列賽」。2016年斯諾克掌門人巴里赫恩老爺子推出一項重大獎勵，任何一位斯諾克選手單賽季能連續奪得4站「本土系列賽」冠軍，將額外得到100萬英鎊大獎，可惜4年來這項獎勵從未被觸發。

英格蘭賽作為本賽季「本土系列賽」首戰，塞爾比一路過關斬將決賽9:1大勝吉爾伯特，開啟衝擊百萬大獎的征程。奈何到了北愛爾蘭賽1/4賽，塞爾比4:5不敵巫師希金斯，止步8強出局，也意味著衝擊百萬大獎夢碎。

蘇格蘭賽塞爾比又異軍突起，4:0橫掃基米羅伯頓挺進8強與奧沙利文會師。兩人苦戰9局，奧沙利文三桿破百率先拿到賽點，塞爾比韌性爆發連勝兩局晉級半決賽。隨後又6:1大勝吉爾伯特與90後利索夫斯基會師決賽。決賽上利索夫斯基發揮淋漓盡致，拼接長台準度，與塞爾比苦戰15局，最終塞爾比以9:6鎖定勝局。

蘇格蘭賽作為2019年收官之戰，塞爾比一舉奪冠，成就單賽季斬獲「本土系列賽」兩個冠軍的第一人。可惜的是，巴里赫恩老爺子並沒有推出兩冠的獎勵。賽後塞爾比坦言，希望巴里能看到這場勝利，百萬英鎊沒了，能不能給50萬英鎊？球迷們，能給嗎？
