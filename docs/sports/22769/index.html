巴薩vs皇馬：梅西對決本澤馬，兩新星誰能閃耀？

2019-12-18 1,097條評論 2,892次閱讀 339人點贊

北京時間12月19日周四凌晨03:00，2019-20賽季西甲第10輪補賽迎來首回合國家德比，巴塞羅那主場迎戰皇家馬德里。目前兩隊同積35分，並列西甲第一，梅西、本澤馬12球並列射手榜第一。

比賽信息

比賽名稱：西甲聯賽第十輪（補賽）

比賽對陣：巴塞羅那vs皇家馬德里

比賽時間：北京時間12月19日03:00

比賽地點：諾坎普球場，西班牙

交鋒記錄

西班牙國家德比有著悠長的歷史，從1902年開始計算，雙方共交手275次，其中巴塞羅那以115次獲勝次數高於皇馬的99次。

雙方近5次交鋒，3次巴塞羅那勝利，2次以平局收場：

2019-03-02 西甲第26輪 皇家馬德里0-1巴塞羅那

2019-02-28 西班牙國王杯半決賽 皇家馬德里0-3巴塞羅那

2019-02-07 西班牙國王杯半決賽 巴塞羅那1-1皇家馬德里

2018-10-28 西甲第10輪 巴塞羅那5-1皇家馬德里

2018-05-06 西甲第36輪 巴塞羅那2-2皇家馬德里

傷停信息

巴薩：奧斯曼-登貝萊，阿圖爾。

皇馬：阿森西奧，巴斯克斯，阿扎爾，馬塞洛，哈梅斯-羅德里格斯。

比賽看點

【皇馬客戰巴薩，諾坎普上演西甲頭名之爭】

西甲第10輪補賽，皇馬做客諾坎普挑戰巴薩，齊達內的目標是率隊終結近期對巴薩的不勝，同時超過對手坐上西甲榜首。

巴薩本賽季聯賽後來居上，16場戰罷比皇馬多贏1場、少平3場，雙方總積35分並列榜首，巴爾韋德要在主場保平爭勝。

【梅西大戰本澤馬，兩新星盼一戰成名】

西甲射手榜，梅西、本澤馬12球並駕齊驅；助攻榜，梅西6次領跑，本澤馬5次緊追。首回合西班牙國家德比，也是巴薩皇馬頭號球星的對決。

本賽季，德容成為巴薩製造進攻威脅的利器，皇馬新援羅德里戈也迅速在歐冠和西甲閃耀，兩位新星將在德比戰場迎來大考，發揮出色的球員必將一戰成名。

【遲到的德比，皇薩大戰安保問題值得關注】

本場比賽是西甲第10輪，本該於10月進行，但因為西班牙動蕩的局勢，比賽最終取消。

在多方斡旋下，比賽終於要在明晨打響，賽前安保問題備受關注，我們也希望當諾坎普哨響時刻，足球能夠遠離政治，回歸它本來的模樣。

近期戰績

巴薩：各項賽事近10場7勝2平1負，進22球丟11球。

皇馬：各項賽事近10場7勝3平0負，進28球丟6球。

賽前聲音

巴爾韋德：我們還處於聯賽的上半程，後面還剩下很多場比賽，不過我們兩支球隊都希望在這場比賽之後獨自領跑積分榜。很顯然，不管雙方在此前的積分是多少，在這場比賽中我們勢均力敵。

最近幾年，我們在伯納烏戰績不錯，而在我們的主場皇馬一直都表現很好，比賽一向都強度很高，希望我們能踢好這場比賽。

無論對方的狀況如何，我們必須找到自己的踢球方式。對方有很多其它的選擇，我不知道齊達內明天會排出怎樣的陣容。他可能會派上伊斯科、貝爾、維尼修斯或是羅德里戈……但無論如何，我們必須專註於自己。

對我們來說，抑制本澤馬的發揮是非常重要的，毋庸置疑。他本賽季打進了12個進球，是皇馬的最佳射手。

齊達內：我們在各個方面的最好時刻迎來了國家德比，在身體和精神上都是如此。

我並不擔心控球率，還有很多事情，我們需要做的是展現出最佳版本的自己，這也是我們要努力去做的。

這是一場11人對11人的比賽，我們很清楚自己的對手，巴薩非常強大。巴薩擁有出色的球員，比如說梅西，當我們也有自己的武器。我們要做的，就是踢出精彩的足球。
