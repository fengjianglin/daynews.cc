穆里尼奧有多強？他的最佳陣，能夠統治歐冠

英國BBC最近發起了一次評選，讓球迷們參與選出穆里尼奧執教生涯的最佳11人陣容。最終，葡萄牙巨星克里斯蒂亞諾-羅納爾多以88％的支持率高居第一，捷克門神切赫則力壓卡西當選最佳門將。下面，我們來看看這套穆里尼奧的最佳11人陣容。從整體實力來說，這11個人可以統治整個歐洲！

穆里尼奧是一個崇尚防守反擊的主教練，而他比較喜歡的隊員需要攻守平衡，體能充沛。在門將位置上，彼得-切赫成為了穆里尼奧最佳11人的首發門將。切赫曾經在切爾西和穆里尼奧相遇，這個捷克門神的門線技術牢固、反應迅速並且發揮穩定。若非對陣雷丁隊的重傷，切赫完全有能力與布馮競爭21世紀的世界最佳門將。而後防線上，薩內蒂、拉莫斯、特里和阿什利-科爾組成了穆里尼奧的最佳4後衛人選。值得一提的是，這4人全部拿過歐冠冠軍。並且，這個4個後衛都具有恐怖的帶刀屬性。

中場方面，蘭帕德以68％的支持率位居中場人選第一名。而馬克萊萊和哈維-阿隆索，也進入了穆里尼奧11人當中的最佳3中場名單。這3名中場攻守兼備，並且硬度極強。穆里尼奧對於中場的要求就是跑動多、傳球准。馬克萊萊能夠很好分擔中場的攻防轉換，而哈維-阿隆索則是隊內絕對的節拍器。蘭帕德能攻善守，後插得分能力十分出色。不過韋斯利-斯內德沒有進入這份名單的首發陣容，也讓不少球迷絕對遺憾。畢竟穆里尼奧率領國米完成三冠王的那個時代，斯內德是當之無愧的核心。

穆里尼奧的職業生涯執教過許多頂級前鋒，其中就包括了C羅、德羅巴和阿扎爾。沒錯，這3人被球迷們評選為穆里尼奧執教時代的最佳前鋒。C羅在皇馬的時候，是穆里尼奧最信賴的進攻輸出點。穆氏皇馬當時反擊兇猛，進攻犀利。而C羅的存在，更是讓皇馬的進攻火力最大化。不過遺憾的是，卡里姆-本澤馬沒有入選首發名單，實際上本澤馬在穆里尼奧的手下也極為關鍵。而德羅巴作為正印中鋒，這個象牙海岸黑又硬是穆里尼奧當初指定的不二人選。一方面，穆里尼奧成就了德羅巴。另一方面，德羅巴也成就了穆里尼奧。

阿扎爾在穆里尼奧二進宮切爾西時期表現優異，因此他入選這份榜單也毫不讓人意外。在切爾西，阿扎爾成為了一名真正的世界級球星。而離開切爾西前往皇馬，後者目前也發現了阿扎爾不可估量的戰術價值。設想一下，假如巔峰期的阿扎爾和C羅在同一個球隊的兩邊，並且中間還站著一個德羅巴，對手的心理陰影面積估計是不小。而中場的3個人又是體能充沛的跑不死，並且球商還極高。後防線上拉莫斯、特里、阿什利-科爾和薩內蒂隨時可以開倒車，這套陣容能夠匹敵世界足壇歷史上任何一支超級強隊。

在落選名單里，埃辛、卡瓦略、卡西利亞斯、厄齊爾也一一在列。進入21世紀以來，穆里尼奧和瓜迪奧拉二人的確稱得上是世界足壇最具統治力的兩名主教練。如今，穆里尼奧執教熱刺，後者雖然剛剛輸給曼聯，但是勢頭明顯反彈。相信孫興慜、凱恩、阿里等人也會在穆里尼奧手下越來越好，穆里尼奧的執教生涯從來不乏優秀的球員。而現在在英超聯賽，穆里尼奧還在繼續創造屬於自己的輝煌。
