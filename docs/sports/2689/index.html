國際乒壇的獨特風景，前30名只有3人，球迷呼籲他退役後留隊執教

國乒雙子星時代，直板選手佔據國際乒壇半壁江山，而隨著小球改大球，各種規則的改進，直板似乎「瀕臨滅絕」，如今成為了一個特殊的存在，直板的劣勢在規則修改之後逐漸明顯，橫板選手大行其道，成為國際乒壇的霸主，而直板選手越來越少，如今以許昕為首的直板選手，在2019年12月份，排名前30名的球員中，只有3人是直板選手。

許昕，12月排名第一

黃鎮廷，12月排名第19

趙子豪，12月排名第29

如果說現在的規則更有利於橫板打法，這話一點也不假，但是直板打法並沒有落後到被淘汰的跡象，只是想要在直板打法上做出成就，確實比橫板更為困難。許昕作為如今直板選手里的佼佼者，2019年獲得多站比賽冠軍，12月躍居國際乒聯積分榜第一位。

我們喜歡看許昕的比賽，完全是因為他在直板打法上有獨到的見解，世界第一正手似乎只有在直板選手當中，才能體現的淋漓盡致，而如今只有許昕一人才能讓直板打法更為光芒四射，在直板和橫板的較量當中，許昕的存在讓橫板不再一家獨大。但許昕之後呢？

黃鎮廷也是一名出色的直板選手，12月排名第19位，但是在直板打法的領悟上，和許昕相差甚遠，僅有的排名不能完美詮釋黃鎮廷在直板領域的高光，但是黃鎮廷的存在，讓球迷認為，直板不再是國乒的「專利」，還有人肯為這種即將被橫板取代的打法，去奉獻自己的職業生涯。

雖然直板打法依然可以和橫板打法抗衡，但是越來越有利於橫板打法的乒乓球時代，很多小球員不願意付出更多的努力去學習直板，體育競技的殘酷，往往讓球員更願意選擇一種捷徑，在直板和橫板之間的捷徑，所以橫板球員越來越多，也就不足為奇。

國乒小將當中，直板選手也越來越少，只是國乒依靠直板發家，不願意將直板打法徹底消失，趙子豪在今年的奧地利公開賽，作為黑馬式的球員，一舉闖入決賽，並且在賽前，直板教練王皓親自擔當趙子豪陪練，可見國乒依然對直板情有獨鍾。而趙子豪通過奧地利公開賽一站，已經躍居世界排名第29位。

國乒在雙子星時代，直板佔據半壁江山，而如今直板只能作為一個特殊的存在，成為賽場上的一道獨特風景。許昕退役之後，誰來扛起直板大旗？是趙子豪嗎？誰都無法確定，但是直板仍要繼續下去，只為一種特殊的情懷，球迷呼籲，許昕退役之後，應當留隊執教，發掘小將當中，可以勝任直板的選手。將直板發揚光大。
