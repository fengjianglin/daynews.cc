2020年東京奧運會羽球積分賽沒有一個國家/地區獲得滿額門票

天天要聞
2019-12-21

國羽男單目前只有諶龍擠進奧運晉級線內

　　2020年東京奧運會羽球積分賽即將進入倒數最後4個月比賽。目前絕大部分選手都參加至少10站，從目前形勢分析， 還沒有一個國家/地區獲得5個比賽項目滿額門票。

　　為期一年的東京奧運羽球積分賽從今年4月30日紐西蘭公開賽開賽至今，已進行近20個主要賽事。

　　作為東道主日本目前成績最好，不出問題主隊就可以鎖定9張門票，桃田賢斗，更是首位在奧運會積分賽突破10萬分大關的選手，以第一名取得奧運會門票基本已沒懸念。

　　

根據世界羽聯官網的資料，在截止日期（4月28日）之前共有14個大小賽事讓選手們進行東奧資格最後衝刺。

　　日本選手常山干太排在第11名，如果明年4月28日，他的排名只要沒有跌出前16名，主隊單打獲得滿額。日本選手西本拳太排在第18名，他也還有機會與常山競爭。

　　國羽目前男單排在第5的諶龍積分接近7萬分，已經鎖定一張門票，排在第22的林丹與第23的陸光祖積分連4萬分都沒有。石宇奇則在積分賽只排名第27名，若要超越2位隊友排在前16名，他必須在明年1月份打出受傷前最佳狀態。

　　印尼男單在前16名包辦4人，分是喬納坦 （第4）、金廷（第6）、蘇吉亞托（第14）與拉胡斯塔維托（第16）。印尼現在面對以前國羽的苦惱，人才太多門票太少。

　　丹麥男單也有3人排在前16名，排在第3的安東森基本確定門票到手，反而是2017年世錦賽冠軍安賽龍需要與傑姆克競爭，2人積分接近。

　　

女單國羽與日本已幾乎拿下2張滿額門票。中日各有3人排在前16名內。陳雨菲排在第1，但排在第9的何冰嬌地位不穩，她有可能被去年青奧會亞軍王祉怡超越。日本有奧原希望與山口茜雙保險。

　　

印尼男雙、日本女雙與男雙、國羽的混雙與女雙都拿基本拿到2張滿額門票。

　　雙打只有排在積分賽前8名才拿到2張滿額門票。印尼男雙前8佔3席，吉迪恩/蘇卡穆約、阿山/塞蒂亞萬已提前拿下門票。

　　日本的嘉村健士/園田啟悟、渡邊勇大/遠藤大由穩居前5名，同樣出線在望。國羽只有李俊慧/劉雨辰排在前8名內，韓呈愷/周昊東排在第11名。

　　

混雙排在前2名的鄭思維/黃雅瓊、王懿律/黃東萍已提前拿下門票，其他國家/地區都很難拿到滿額門票。

女雙中日韓各有至少2組在前8名內，滿額沒有懸念。
