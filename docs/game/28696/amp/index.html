快樂玩王者丨50星哪吒攻略：四一分推，兵線運營專家

天天要聞
2019-12-22

“為你我用了半年的積蓄，漂洋過海的來看你”，聽到這歌詞，是不是感覺很浪漫呢？如果有一個人，為了你不遠千里飛到你身邊，會不會很感動呢？今天就為大家介紹峽谷里的浪漫好男人——哪吒。

一分鐘了解哪吒

1. 回復系英雄剋星

哪吒被動里其中一個的特點，就是減少對手的生命回復，所以他較有效克制一些回復系的英雄，減少他們的回血效果。

2. 全圖支援+敵方視野

哪吒大招開啟時可以看到敵方全部英雄的視野，同時可以選擇飛到一個敵人身邊進行攻擊，能很方便地支援隊友和壓制敵人。

3.留意局勢、忌盲目取人頭

由於哪吒可以全圖飛奔，因此不少人會為了收人頭而盲目飛敵方殘血英雄，導致浪費了一次大招，甚至有時也會讓自己陷入險境，用自己滿血之軀去和敵方殘血一換一（更惡劣的就是傳說中的”千里送人頭”）。因此每次要起飛的時候，都要先衡量一下到底有沒有需要。

快樂學技能

被動：火火火火——哪吒在普攻或者技能造成傷害之後，都會在敵人身上”點亮生命的火~火火火火火~”，這把火會對敵人造成真實傷害，持續3秒；同時，減少對方25%的回復效果。

這個被動是哪吒其中一個特色，持續3秒的真實傷害有時會給你驚喜：敵人絲血逃跑，但是他身上有你這團火，3秒內你就能收到驚喜（人頭），由於是真實傷害，所以即使對方身上有盾，大多數情況也是無法救回的。除了瑤的真實護盾或者能及時回復血量之類等等的手段。

一技能：火花四射——轉起一團火花，對身邊敵人造成傷害，同時疊加一層6%的移速，持續5秒，最多有5層，即最高可達30%移速。

這個技能是哪吒打傷害的主要來源，同時也由於移速的加成，很多時候對方想跑也會被哪吒追上。另外要注意技能是：每打中一個敵人就可以增加一層，所以在人堆中施放的話，可以一下子就能疊起移速，很多時候就是靠這個被動穿深入於人群之中然後再逃去無蹤。還有一點，打中小兵和野怪也能疊加這個效果，因此在趕路的時候，可以一技能刷一下身邊的小兵和野怪來加速。

二技能：常在你左右——用繩子暫時束縛對手（包括小兵和野怪），對方會短暫暈眩，然後跳到他的另一側，在1秒之後、4秒之內可以再次施放一次（又跳一次）。施放二技能也可以疊加移速的被動，而且同時還能在3秒內，增加最多20%免傷。

哪吒就是靠二技能用來留住敵人，而由於二技能也能疊加移速，還有免傷（記住這個免傷效果沒有疊加），所以一般人被哪吒纏住後很難逃掉，甚至送命：一邊敵人被捆綁、另一邊哪吒能加移速，要打架哪吒還有免傷，就算打不贏哪吒還有移速加成逃走，為簡直是無賴！二技能也能用於日常逃命，比如說翻越小兵，或者在野怪位後方跳進野怪坑（要注意是跳進，基本不能跳出）。還有一種功效就是用敵方前排來做踏板，跳到後方打對方輸出（不過這個方法一般比較少用，切後排還是用大招來得直接）。

三技能：漂洋過海來砍你——哪吒的三技能解釋起來算是比較複雜，簡單概括就是：可以立即看到全部敵人在哪，並且阻止他們回城，然後挑一個飛過去揍他，衝到身邊的時候能撞開他；由於飛得太high，落地之後慾火焚身，能燒到身邊的所有敵人（真實傷害），同時還有增加30%韌性和免傷效果（這麼興奮估計是打了雞血）！在飛過去的時候如果感覺不對勁，還可以提前下飛機，但仍然會慾火焚身和打雞血（火焰包圍、打雞血等原來應有的效果）。

三技能是哪吒的核心技能，也是最好玩的地方。但是操作相對比較複雜，首先要先點一下，這時哪吒會垂直飛起來，地圖上能看到所有敵人（包括隱身的），同時打斷他們回城（在他們身上會觸發被動技能中的少量真實傷害，運氣好的話立即就能收掉絲血的敵人的人頭……）；這時候三技能旁邊會多出一堆敵人的頭像（小細節：敵人頭像有顯示當前血量，血量少的還會呈暗黑狀態），再次按住三技能不放，滑動到各個敵人頭像上，大地圖會馬上移到敵人的位置，放手之後，就開始飛過去了。衝到對方身邊落地之後，首先會撞開他，這裡也有一個小細節：可以用方向鍵控制向哪邊撞，選擇往最有利的方向撞（比如撞出塔外或者撞進我方人堆又或者撞離敵方的保護範圍，總之記住一定要選方向）。

哪吒的大招操作雖然複雜，但只要打打匹配，多飛幾次練習練習就好了。核心問題是：什麼時候飛？飛誰？怎麼飛？很多情況下都會飛敵方殘血的英雄，但是即使是無腦飛殘血也一定要看情況，比如敵方殘血根本逃不掉的時候（在人堆中或者被夾擊等等），你再飛過去也沒有什麼作用，搞不好飛到的時候對方已經掛了，那你原來的兵線還要不要？跑回去路上會不會被伏擊？開局初期哪吒大招一分多鐘冷卻，這段時間內對方都會非常安心不會再被飛。你虧了那麼多、要冒這些風險，就是僅僅為了得到一個助攻嗎？所以說，大招的運用才是哪吒這個英雄最核心的內容。

快樂上戰場

其實哪吒這名英雄玩起來還是非常快樂的，大招可以飛往任何一個敵人，給對方造成的心理壓力是非常大的，尤其在中後期等級裝備都起來的時候，冷卻平均40多秒，血量不高的敵人絕對要想著回家，不然自己會很容易成為了哪吒的槍下亡魂。

哪吒在前期對線的時候還是先注重清兵線。由於哪吒的坦度和傷害都不算很高，所以要注重自己的血量，避免和對方過多消耗（同職業對線，其實哪吒1V1很難打贏哪個英雄，除非對面特別菜）。對線時主要用一技能燒一下兵線，躲躲對方技能，CD好了再回頭一技能清兵，有被動的持續真傷下，清兵線並不會太久。到四級的時候，就迎來哪吒的強勢期了。清好兵線，並開始密切留意形勢，早期可以多點飛過去收殘血人頭，給對方建立一種壓制的心理恐懼。但不建議過早就去和對方一換一，畢竟早期的兵線非常重要。

到了中期開始就可以實行哪吒比較主要的”四一分推”打法了，盡量單帶。如果對方不來抓，就直接拆塔。如果對方過來抓你，就儘早離開，不要被對方抓到，多注意小地圖，提前避免危險情況。萬一被抓時不要慌，冷靜利用一、二技能逃跑（一技能疊移速、二技能翻過敵人，或者在野怪坑後面利用野怪翻過去逃命），或者也可以用大招反向飛遠處的敵人（中途下飛機也可以），快速逃離現場。如果多人來抓你，真的逃不掉，就盡量拖住吧，為隊友推進爭取更多時間。

哪吒的四一分推，並不是意味著要把兵線一推到底，只需要推過中線就可以了。更多還是留意局勢，發現即將進行團戰或者團戰剛剛開始時，就可以開啟大招，飛過來參團了。入團後的哪吒主要目的是切掉C位，切忌把自己當成坦克扛傷害，如果不是很逆風的時候，不要盲目獻祭自己（玩哪吒，心態是雙向的，你既可以把對方切到崩潰，但也可能隊友認為你千里送人頭而崩潰）。哪吒的疊移速、免傷、韌性、真傷、防回復、二技能翻越，都是可以讓你從進團打傷害到從容逃離的法寶。脫離團戰後進可帶線退可回城，實乃戰場中牽（惡）制（心）對方的好吒（渣）兒。總之重點是要以兵線為主，團戰可以輸，但兵線一定要帶！

另外再說幾點小心得：

1. 當隊友們在集體推進、或者打龍（總之你認為是重要行動），如果丟失了對方全體的視野一段時間，可以開啟大招查探對方人員動向，避免隊友被蹲或者被搶龍。

2. 如果在對局過程中發現我方落後，而對方C位經濟發育都超前了，就要積極觀察，隨時找機會飛過去強行切C，務必保證能切，能一換一就是賺的，目的是要打壓對方C位發育和經濟。

3. 除了觀察小地圖，還要看大地圖查看戰場情況，如果能用三指拖屏當然很好，但如果手指不太靈活，記得在設置里拉高“鏡頭靈敏度”，方便用右指在屏幕右側拖動觀察大地圖。

4. 飛敵方的時候也可以用”聲東擊西”的方法迷惑對方，比如要飛某個敵人，可以開大飛他後方的人，然後在他旁邊停下，給他一個”Surprise”！

5. 逆風局時如果不是非常需要，盡量不要抱團守家，哪吒的最大特點是他的機動性，想盡辦法要把兵線帶出去，才能為團隊爭取更好的機會。

6. 點開大招後，如果覺得沒有好機會，也千萬不要不選人而停下來，因為就算不飛任何人，技能CD也沒有任何返還。這時候應該隨便選其中一個人飛，然後馬上再按大招停下，起碼能獲得全身火焰、韌性和免傷等效果。

7. 召喚師技能推薦【終結】，可以在戰場中作為收割補充傷害。而且哪吒對最後斬殺的血量並不需要要計算到最精確，因為哪吒被動中有後續3秒的真實傷害，因此即使終結技沒有將對方擊殺，他也可能會被後續火焰的真實傷害收掉。

8. 設置一個局內快捷消息”大招已經好了”來提示隊友，你已經準備好可以隨時飛了；另外在飛之前也可以發送”發起進攻”來提示隊友，你即將開啟大招。如果是路人局的話，最好提前幾秒發送，因為隊友可能無法第一時間了解你的意圖。

9. 當隊友正在拆水晶時，哪吒可以開啟大招，打斷對方敵人回城。同時，可以飛向離敵方水晶最近的敵人和他纏鬥，阻止他跑過去保護水晶。

10. 比較怕的英雄（包括但不限於）：東皇太一（吸住基本可以唱一首涼涼）；張良（同東皇太一）；瑤（你有真實傷，我有真實盾）；大喬（退可傳送回城氣死你，進可召喚隊友圍毆死你）；劉邦（作為隊友玩雙飛流爽爆，但作為敵人你飛誰他保誰，真的是被噁心到了！）。

遇到以上敵人，要做好心理準備，心態千萬別崩！

出裝&銘文

出裝方面，由於哪吒肩負切敵方C位脆皮的任務，因此不需要全輸出，而且哪吒需要頻繁進出戰場，經常會吃到傷害，因此推薦半肉裝，有生存和輸出。

（抵抗之靴+紅蓮斗篷+暗影戰斧+魔女斗篷+極寒風暴+破軍）

抵抗之靴：提供法術防禦和減少被控時間，讓哪吒在前期不易被抓死，當4級以後，配合大招落地的韌性加成更讓哪吒的被控時間減至最低。

紅蓮斗篷：提供防禦和生命值，同時能燃燒身邊的敵人，是哪吒的核心裝備。由於哪吒經常要清理兵線和切敵方脆皮，紅蓮斗篷可以提供不錯的傷害。如果經濟落後的情況，可以先出小件的”熔煉之心”。

暗影戰斧：脆皮剋星，而且還有15%的冷卻縮減，為大招減短很多的冷卻時間，哪吒的核心裝備之二。

魔女斗篷：法系防禦裝，提供生命值、法術防禦和法術護盾，除非對方沒有法師或者法師不強，否則都建議早點出，畢竟哪吒要常要進出戰場，會吃到很多魔法傷害。經濟落後建議可以先出小件的”神隱斗篷”。

極寒風暴：哪吒的核心裝備之三，高額防禦，高額冷卻縮減，又能減速身邊敵人，和哪吒大招配合得天衣無縫，極寒風暴加上暗影戰斧，冷卻縮減幾乎滿了，讓哪吒的大招減至40多秒發動一次，對敵方脆皮非常有威脅。

破軍：提供高額物理傷害，對半血以下敵人還有收割效果，脆皮的噩夢。缺點就一個字：貴！

如果對方射手比較猛，哪吒也可以根據情況出一個不詳徵兆；如果喜歡經常遊走的也可以出個霸者重裝，免去回家的煩惱；遊戲後期為打團戰也可以把鞋子換成復活甲。

銘文方面推薦使用【調和+鷹眼+異變】：調和可以增加哪吒前期的坦度和少量回復，有助前期對線時的生存；鷹眼和異變就是常規的百穿效果了，用來切脆皮效果不錯。

（紅色：異變x10，綠色：鷹眼x10，藍色：調和x10）

總結

來到了S17賽季，哪吒可以說是比較少見上場的英雄，但憑他靈活的機動性、獨特的全圖支援，可以非常好地牽制著敵方。只要多留意觀察局勢，控制好兵線，勝利就能手到拿來！

快樂玩王者丨一個專為王者榮耀快樂玩家而設的欄目，希望能人人快樂、個個上分！
