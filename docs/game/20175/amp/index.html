冬幕節將至，聊聊風暴英雄中的冰雪元素

天天要聞
2019-12-18

馬上風暴英雄將迎來一年一度的冬幕節活動—玩具大戰。猜測玩家對玩具大戰這個活動並不陌生，去年我們已經體驗服過了這個趣味十足的遊戲。今年暴雪返場了這個受人喜愛的活動。提及冰雪元素，風暴英雄中也有著這麼一對曾經的戀人，吉安娜和阿爾薩斯，從他們的身上可以看出許多冰雪元素，阿爾薩斯鎮守北極嚴寒之地諾德森，而吉安娜有掌控著很強的冰系法術。今天叫讓我們分別聊聊這些內容吧。

今年的玩具大戰活動將於12月18號登陸風暴，就在死亡之翼活動之後，整個活動一直持續到明年的3月3日，整整比去年多了2個月時間。玩家可以充分參與冬幕節的狂歡了。在冬幕節活動期間，所有的戰利品箱將升級為冬幕節戰利品箱，可以開出特別的獎勵，去年冬幕節可以開出的，這一次都可以已開出，同時，風暴還為大家加入了琪拉的新皮膚和可愛的小火車坐騎等等一系列全新的獎勵。去年沒玩夠或者沒獲得心儀獎勵的玩家，今年的活動可千萬不要錯過哦！

談到阿爾薩斯和吉安娜這對舊日情侶，我們都為之感到可惜。曾經的王子和天才女法師並沒能終成眷屬，令人惋惜。不過，在風暴中，他們卻可以再續前緣。風暴英雄中，巫妖王阿爾薩斯是一名強力的坦克英雄，擁有著極強的坦克能力和控制能力，可以作為團隊的主t和副t。阿爾薩斯的w技能和e技能分別具有定身和減速的效果，在團戰殘局中可以十分輕鬆的留住敵人，配合隊友完成擊殺。終極技能召喚辛達苟薩有著令人震驚的減速60%的效果，並且可以持續4秒。在團戰開始之時，釋放該技能可以完美地限制敵方團隊的走位。

吉安娜作為現今魔獸世界最強女法師，在風暴英雄中也是最強法師之一。他的w技能暴風雪擁有著極強的傷害能力，硬吃三下的話怕是要尋找泉水了，該技能最好配合隊友的減速技能以打出最高傷害，諸如上述提及的阿爾薩斯的大招。吉安娜的終極技能可以選擇召喚水元素，雖說是召喚物，卻有著強力的傷害，可以體驗一下六打五的快感。

要提示風暴玩家的是當前版本死亡之翼的限時活動馬上就要結束了，忘記領取任務的同志也不要著急，現在還可以領取獎勵（之前沒完成就不行了）。6個稀有戰利品箱子是十分豐厚的獎勵。購買了暴雪嘉年華通行證的玩家，上線就可以領取死亡之翼這名英雄，趕快化身死亡之翼，讓整個樞紐都籠罩在你的怒火之中吧。
