近戰騎養成手冊，三招帶你練出最強鐵騎

天天要聞
2019-12-21

在《亂世王者》的世界裡，近戰騎一直是深受大家喜愛的兵種，我們今天就來對這個強力兵種養成的要素來進行盤點。

1、 兵種簡介。

作為前排輸出單位，近戰騎的基礎攻擊、防禦和生命屬性都比較高，技能【閃避】提升了近戰騎的生存能力；而優先攻擊遠程的【衝鋒】，讓近戰騎可以在戰鬥中針對敵軍生存能力較差的遠程部隊進行打擊，讓近戰騎可以在面對遠程部隊時打出較高的殺敵數字。

在近戰騎的養成體系中，除了白起的”挫志”效果，後續的大部分加強也都是在提升近戰騎的閃避和衝鋒傷害。高生存和針對性，應該是近戰騎最為顯著的兩大特點。

2、 武將、技能、兵法和軍師計策。

1、 武將搭配。

近戰騎陣營的武將中，白起可以提供20%的暴擊率，2次衝鋒且第二次衝鋒強化傷害，還有削弱敵軍遠程攻擊力的”挫志”，主動技能也是當前版本唯一可以增加近戰騎衝鋒幾率的技能，可以說是當之無愧的近戰騎核心武將。

武將搭配方面，從輸出角度看”白起+曹丕”的組合無疑是最暴力的。所以通常大家會選擇這個組合來做近戰騎的第一陣容。

近戰騎二隊，一般會由趙雲、太史慈和司馬懿3個武將中的2個來進行搭配。

2、 技能搭配。

在技能的選擇上，最主力的搭配方式無疑是【突襲】、【強襲】和【近戰騎攻擊】。

當然，由於近戰騎強悍的單挑能力，頻繁進行單人作戰的朋友會臨時把【近戰騎攻擊】換成【殲滅】來達到擊殺對方有生力量的目的。

3、 兵法。

考慮到監軍府帶來的收益，增加近戰騎攻擊的【騎砍】、增加近戰騎生命的【鐵蹄】和增加近戰騎防禦的【鐵馬】3個兵法是必選的。

第四條兵法，如果主要玩法是集結、活動，攻城戰鬥較少，全兵種攻擊的【雄兵】是一個不錯的選擇。

當然，由於高屬性近戰騎強大的單挑能力，很多人會給”白起+曹丕”的主力組合領悟增加攻城攻擊的【催城】，其餘監軍府武將學習【雄兵】。

4、 軍師計策。

近戰騎的軍師計策，首選當然是【合圍突擊】，可以給近戰騎提供全方位的加成。

體驗服新出的單挑兵法【神速姿態】在攻城和活動撞秘寶、縫隙的時候也有不錯的效果。

3、 龍脈、監軍、神獸和科技。

近戰騎在戰鬥中，會被槍兵的龍脈”飛槍”、項羽”投槍”優先攻擊；由於站位關係，敵軍大部分的輸出都可以打到近戰騎，承傷壓力比較大。同時由於技能【強襲】的強勢輸出地位，無法學習生存類的技能，對龍脈、監軍、神獸注靈和科技提升的屬性依賴比較大。

這幾部分都是很重要的”內功”，越高越好不存在方向問題，努力養成即可，所以我們下面主要主要說一下龍脈的附加詞條。

近戰騎的龍脈附加詞條，首選近戰騎攻擊、近戰騎生命和騎兵攻擊，第四條在近戰騎防禦和騎兵生命兩個當中選品質、屬性更高的那個，當然，如果拿到橙色屬性的全兵種攻擊也是不錯的選擇。

近戰騎有今天的強勢地位，很大一部分原因就是高手玩家抓住一切機會提升近戰騎的三圍屬性，讓這個兵種面對遠程時優勢更為突出；在被槍兵針對性攻擊時，甚至可以依靠強大的血防加成和【閃避】的特點，吸收較多的輸出而戰損並不是太高，通過強勢養成打出逆天的戰績。

以上就是當前版本，養成一隊強力近戰騎的幾大要素，就是這幾大要素，讓強勢的近戰騎可以逆天改命，成為很多高手最喜歡的兵種之一。
