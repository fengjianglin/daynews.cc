競速大講堂——海底、門派、迷魂塔全都有

你會因成就點數不夠而感到憂慮嗎？你還在為一本獸訣或高級符紙而罵人狗托嗎？快來競速大講堂學習吧！看了這份攻略競速大師將離你不再遙遠！做個人人追捧的職業隊長拿盡讓成就黨羨慕的56點成就吧！除了心靈上的滿足每次門派、海底、迷魂塔還有前十禮包入賬可以開出讓人羨慕的高額獎勵！你還在等什麼？快來學習競速技巧吧！

一、門派闖關

1、陣容選擇

五魔方寸+五魔輔助+大唐+龍宮+魔王。

五魔方寸：方寸在這個組合中很關鍵，因為有時候剩下一個獅駝首席時，因為鷹擊動畫特效較長，所以很消耗時間，這時候方寸出現，一個定身只需要一秒動畫特效，瞬間就節約了很多時間；

五魔輔助：魔普陀最佳，因為秒死還能給法傷最高單位加靈動，不要小看這一點點的傷害，競速考驗的就是細節,每多一點傷害，就可以多節省一秒；

大唐：四掃大唐，血量較多的主怪分分鐘就會被大唐掃殺。

這裡輸出千萬要不同門派，畢竟門派這東西，門派種類越多越容易佔優勢。

2、法寶選擇

無量經+附靈玉/飛劍+風袋+匯靈盞。

3、裝備選擇

因為門派闖關對特技沒有需求，速度需求也很低，因此可以攜帶一些法爆、忽視類的環裝，下掉常用的特技裝備，至於寶石，腰帶和鞋子的黑寶石是必帶的，五魔輔助儘可能也要換上舍利子。

4、寵物選擇

法寵，越暴力越好，當然如果你有秒四黃胖子，50%概率秒四珍露，那就更好了。

千萬要注意，因為進場生效的紋飾都有紋飾生效時間（不屈、共患），因此不要帶這種紋飾，如果你想要增加傷害，請選擇凝魂、靈慧、挫骨、硬化、眷戀這樣不需要入場生效的紋飾。

5、陣法選擇

輸出最大化的場合下，五級天陣當然是最好的選擇。

6、變身卡

法系酒肉和尚

物理大當家

暴擊、連擊都不一定會觸發，但是傷害加成永遠都會生效，因此請選擇傷害加成。

7、臨時屬性

臨時符請打滿法傷、物傷，三絲魚翅提高物理傷害、干燒岩鯉提高法術傷害，廚藝食品也必不可少。

8、細節小技巧

①開好流量

爭分奪秒中，掉線卡頓絕對是大忌，如果你是桌面版，請保證網路，如果你是手機，請開啟4G-5G。

②保持活躍

自動的人都是三秒，但是如果手速夠快，不到一秒就可以完成操作。

③恰到好處的防禦

在碰到非本門派首席時，該門派首席會在3-5回合之間隨機逃跑，因此在只剩下一個首席的時候，防禦可以節省大量時間。

二、海底尋寶

同為競速，因此在門派、陣法、裝備、寵物等等選擇上，海底尋寶與門派闖關基本一致。因此重複部分，阿霄就不再多講，此處阿霄重點從海底尋寶的機制出發講述一下海底尋寶的小技巧。

1、三星寶箱和二星寶箱的選擇

因為我們一般都是法係為主的隊伍，因此藍色箱子（法抗怪）我們一般選擇二星，而紫色寶箱血量較高，還有封系巫師，因此我們也一般選擇二星，而其他選擇三星。

2、橙色箱子

因為三星橙色泡泡一旦爆炸傷害非常高，因此我們一般在手動操作時會秒自己對位的泡泡（人和寵物一人點一個）。

3、紫色箱子

請五魔封系保證自己的速度，確保比雨巫快。

4、不要打粉色箱子

5、注意在最後結束前一定要再打滿一輪

比如你打了幾個寶箱，然後看最後時間打三星不夠一輪，那麼請直接選擇打一星打滿一輪，打滿一輪會給非常多的積分獎勵喔~

三、勇闖迷魂塔

迷魂塔和門派、海底就更不一樣了，為了達成三分鐘出塔，我們一般需要跳跳龍神發揮，無限迷魂親信才可以完成，因此需要非常耐心地多次刷塔才能刷到理想的成績。

當然，迷魂塔和海底以及門派闖關不同，可以一封四輸出，一般如果兩個物理輸出就可以擊殺後排目標怪，其他法寵可以選擇防禦。

最重要的還是儘可能要五個人！

以上就是關於門派、海底、迷魂塔競速的攻略。
