又一新英雄將上線，神秘商店強勢來襲，這些皮膚五折你選誰？

天天要聞
2019-12-18

現在不只有新皮膚還有一些新英雄的誕生，比如之前已經上線的繁星吟遊和曹操KPL天狼征服者皮膚，新英雄有魯班大師還有蒙犽，後續還有很多的更新，S18賽季皮膚，五嶽新皮膚還有韓信的傳說皮膚等等。

不過大家最關注的還是新英雄曹丕的上線，曹丕是曹操的二兒子，甄姬的丈夫，據可靠消息爆料曹丕有可能會在下月初正式上線，新英雄的定位是名戰士，雖說如今的戰士英雄已經不少了，但曹丕確實與其他的戰士不同，他的控制能力極強，並且自身擁有四項技能。

曹丕的一技能是個位移技能，二技能是個護盾，大招是範圍團控技能，具體的效果如何還要等官方的正式上架，下面我們在聊一下網友爆料的韓信皮膚，韓信是不少玩家都很喜歡的打野英雄，但是他的皮膚並不是很多，最近就有網友就爆料說韓信又將更新新皮膚上架。

從這麼網友爆料上來此款皮膚十分帥氣，有著一頭紫色的頭髮和一雙紫色的翅膀，相信如果韓信的傳說皮膚會是這樣的，小夥伴們你們會買單嗎？皮膚商店最近也將迎來更新，為了聖誕節的到來，之前的皮膚都將會被更換，如甄姬的冰雪圓舞曲還有虞姬的加勒比小姐等等都是會下架的。

所以小夥伴想要入手的可不要錯過了時間，除了這些王者中一個比較值得期待的活動，那就是半年一開啟的神秘商店又要重新歸來了，要知道神秘商店可是有超低折扣的，這次的神秘商店又將給我們帶來哪些不錯的皮膚呢？讓我們敬請期待一下吧。
