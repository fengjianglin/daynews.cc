DNF：100級彩蛋透露，至少出現7個使徒！三次覺醒也不夠打啊

2019-12-12 906條評論 2,128次閱讀 498人點贊

國服普雷還沒正式減負，100級就已經蓄勢待發，距離14號的發布會也不到48個小時了，不少玩家就靠這個在支持，如果還是沒有好玩的全新內容，那麼就碎裝備走人，送天空套都攔不住！

而韓服12.12更新，100級前置內容已經出爐，過場動畫已更新！那就是所有副本所在區域都「Boom」的一下炸了，實錘了DNF即將再次大轉移！

100級彩蛋透露，至少出現6個使徒！三次覺醒也不夠打啊

但這必須要有個原因啊，結合CG宣傳片和100級彩蛋，這應該是使徒引起的，早在幾個月前，DNF放出的CG動畫，原來早就透露了颶風的存在，索德羅斯VS卡西利亞斯的時候，背景就很接近藍色風暴，而這一點恰好就是12.12更新的內容。

而後來出現赫爾德，背景是紅色，與魔界大戰副本最後劇情一致（國服暫未更新這個副本），看來又是「二姐」的鍋！那麼就來數一數目前劇情透露出的使徒數量。

卡西利亞斯和赫爾德毋庸多說，最近劇情加入的有希洛克、巴卡爾，而根據100級彩蛋以及雷米大聖堂的更新來推測，米歇爾、奧茲瑪應該也會出現，當然還有最強的BOSS卡恩咯！CG劇情片最後就是他收尾，玩家們全倒下了，地上躺著一個拿著巨劍的鬼劍士……

這已經是7個使徒齊聚了！而普雷也沒涼喲，雖然玩家們變強了！從幾E不到的傷害到了數百E甚至千E，但使徒BOSS也從1個增加到多個。但想在100級硬剛多個使徒做不到的，三次覺醒也不夠打！其實一個卡恩就能單挑一個跨區的玩家。因為卡恩的設定就是「無敵」！單手碾壓巴卡爾，沒有劇情削弱的話，怎麼能爆裝備。

所以12.14的發布會，宣布的新BOSS應該只是其中的1~2個使徒，其它使徒要麼還未亮血條，或者是幫助玩家的，畢竟就算三次覺醒傷害+100%，普通玩家也不過1000E不到的水平，別說卡恩了，單刷個普雷Raid試試？安徒恩、盧克、普雷這三個團本，玩家們根本沒有正面應對過使徒！全都被大幅度削弱過。

按照劇情來看，100級會很精彩，使徒齊聚，三次覺醒的職業NPC也來了！但最大問題依舊是，如何擺脫打樁機制和建立平民副本。
