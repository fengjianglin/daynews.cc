LPL轉會匯總，FPX宣布三上單，IG組成三輔助，RNG小虎迎來替補

2019-12-19 1,873條評論 3,594次閱讀 1317人點贊

LPL轉會期終於完美落幕了，經過轉會期之後，LPL各支戰隊也呈現出了不同的狀態，而參加S9世界賽的三支戰隊也是進行了補強，FPX戰隊宣布了三上單陣容，IG戰隊組成了三輔助陣容，而RNG戰隊小虎終於迎來了替補，下面讓我們來匯總一下轉會期各支戰隊的陣容情況吧。

SN戰隊陣容：

教練：Fluidwind，上單：Bin、BiuBiu，打野：Sofm，中單：Angel，ADC：Weiwei、Assum，輔助：SwordArt，從紙面來看，SN戰隊新賽季的陣容實力不容小覷。

WE戰隊陣容

教練：Domo，上單：Morgan、Poss，打野：Beishang，中單：Plex、Yimeng、Teacherma，ADC：Jiumeng，輔助：Missing，經過重組之後，WE戰隊也是以新選手為主。

TES戰隊陣容

教練：白色月牙，上單：369、Moyu，打野：Karsa，中單：knight9，ADC：Photic，輔助：QiuQiu、Yuyanjia，從陣容來看，TES戰隊也算得上是LPL賽區的「銀河戰艦」了。

EDG戰隊陣容

教練：Clearlove，上單：Jinoo、Aodi、Xiaoxiang，打野：JieJie、Xinyi、Junjia，中單：Scout，ADC：Hope，輔助：Meiko，EDG戰隊S10陣容也給了廠長不小的挑戰。

FPX戰隊陣容

教練：戰馬，上單：Khan、GimGoon、Xiao7，打野：Tian、MingJing，中單：Doinb，ADC：Lwx、April，輔助：Crisp，經過補強後，FPX戰隊組成了「三上單」陣容。

IG戰隊陣容

教練：Fly，上單：Theshy，打野：Ning、Leyan，中單：Rookie，ADC：Puff，輔助：Southwind、Baolan、Fate，至此，IG戰隊也是形成了「三輔助」陣容。

RNG戰隊陣容

教練：Mata，上單：Langx，打野：xiaolongbao、Xixu，中單：Xiaohu、YueKai，ADC：UZI、Betty，輔助：Ming，最終，Xiaohu也是迎來了自己的替補Yuekai。
