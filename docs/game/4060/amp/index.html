隨著暴雪嘉年華上9.0資料片的公布，女妖之王希爾瓦娜斯在8.0中的一切行為似乎都有了答案。在8.0資料片的開端挑起戰火，焚燒世界樹泰達希爾，讓數以萬計的暗夜精靈死於非命；唆使部落與聯盟進行大規模對抗，讓戰士們為了無意義的鬥爭而身首異處；與娜迦女王艾薩拉合謀，釋放古神恩佐斯，並期盼古神的回歸能夠造成艾澤拉斯更多的死難。

這一切如今被證實都是出於典獄長的授意。在他的謀划下，女妖之王破壞了生與死之間的平衡，讓歷次大戰的死難者靈魂流入噬淵，增長了典獄長的力量。而本應該接收靈魂的仲裁者和暗影界的其他領地，卻因為沒有靈魂的補充而實力大減。

如今看來，典獄長的陰謀將顛覆艾澤拉斯的死亡位面，如果我們不能在9.0中團結儘可能多的暗影界盟友，那麼女妖之王口中萬物無存的未來就極有可能實現。

而就在近日，這個神秘剪影的正式形象被外網網友泄露了出來。在外網瘋傳的魔獸9.0藝術設定圖中，出現了一個渾身青綠的巨人形象。巨人有著濃密的白色鬍鬚，頭戴尖頂的金色王冠，肩上是同樣金色且造型誇張的肩甲。眾多金色的鎖鏈纏繞在巨人的周圍，這個藝術設定圖中的形象和9.0宣傳片結尾的黑影巨人十分吻合。

隨著藝術設定圖的披露，玩家對這位神秘巨人的身份眾說紛紜。根據暴雪官方已經公布的說法來看，這位神秘巨人被稱作”典獄長”，他生活在暗影之地最無可救藥的一塊區域——噬淵。傳說只有最窮凶極惡的靈魂才會在死後被丟入噬淵。這裡和暗影之地的其他地區不同，不在高貴仲裁者的管轄之內，也沒有人關心被放逐到噬淵的靈魂究竟面臨著怎樣的結局。

然而和身處暗影界中心，審判靈魂的仲裁者一樣，我們對典獄長的來歷一樣一無所知。有玩家從典獄長巨人型的姿態中猜測，這是一個不為人所知的泰坦巨人，但是這一猜測卻不怎麼經得起推敲。

在官方對於9.0背景的介紹中，稱仲裁者為泰坦降臨艾澤拉斯世界之前就已經存在的暗影界領袖。那麼與仲裁者分庭抗禮的這位典獄長應當也和泰坦無關。一時間大家對這位神秘人物的猜測又走入了死胡同。

雖然我們暫時還不清楚典獄長的身份，但對於他的邪惡計劃我們卻已經有了大概程度的把握。

綜合以上的泄露消息及推論，9.0新資料片的故事應當就是以阻止典獄長及希爾瓦娜斯的陰謀作為主線。而在故事的推進過程中，我們還將了解典獄長與仲裁者這兩位神秘大佬的真實身份。甚至根據官方訪談的消息，連月神艾露恩的真身可能也牽涉其中。新的冒險究竟會有怎樣的走向，新的故事又會帶來怎樣的結局，9.0資料片求你快點上線吧！
