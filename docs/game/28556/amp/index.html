IG新下路真的很強！德杯數據出爐：puff好猛，9項數據排第一

天天要聞
2019-12-23

德瑪西亞杯小組賽的比賽在12月22日晚上正式結束了，在每支戰隊都打了四場BO1之後，成功決出了八支晉級淘汰賽的戰隊。不過也有些沒有打進淘汰賽的戰隊或者選手備受大家的關注，其中就有IG的新下路雙人組。因為IG這套陣容是一隊和二隊的結合，所以配合默契方面問題都挺大，但是從VG轉會加入IG的puff和南風，卻在這次德杯後收穫了大量的認可。

主要是兩人在比賽中的確發揮出色，在之前因為puff可能將代替JKL成為IG首發，最開始一度引發了不小的質疑，但是德杯上puff的表現算是打服了不少人，就從德杯的數據上來看，他也表現很出色。

​分均傷害

這個算是考核ADC選手最基本的數據了，之前IG的JKL在這一塊也表現很出色，因為他非常敢打敢拼。而puff在這方面也很出色，德杯這麼多AD選手中，puff的分均傷害排在第一位，高出第二位April將近200個點，可見他這次德杯有多C了。這還是IG在拿到了兩勝兩負的戰績後的表現。

​傷害佔比

這個數據是顯示在這支戰隊中，ADC的輸出貢獻到底有多少。之前JKL這個數據一直佔比不高，主要是因為theshy和Rookie都太會打傷害了，而在這次IG德杯的陣容中，puff可以說是第一核心選手，他的傷害佔比也是所有AD中最高的，第二高的是LGD的Kramer，Kramer在LGD有多院長相信大家都知道，而puff比他的佔比還高，也說明了puff在這支IG中有多C。

​傷害轉化率

這個數據顯示ADC選手是不是吃資源太多了。Puff又是第一名，而且傷害轉化率高達175%，這個數據如果超過140%就算是表現很出色的了，沒想到puff都快接近200%了，是真的離譜。也就是說他打出的傷害絕對比他吃的資源多很多。

​除了上述三項數據之外，puff還有六項數據排在AD位置的第一名，分別是KDA、場均擊殺、場均助攻、總擊殺、總助攻、經濟佔比，加起來一共有9項數據排在德杯的第一名，可以說他是德杯小組賽上表現最好的AD也不為過。第二應該是Kramer。總體來說，IG這個新下路的表現已經很超出預期了，puff不一定有JKL強，但是南風puff的組合肯定是比JKL寶藍組合強的，明年春季賽的IG就算JKL不續約，感覺也可以期待一手。

那麼，對於這波puff各項數據的表現，你們有什麼想說的嗎？
