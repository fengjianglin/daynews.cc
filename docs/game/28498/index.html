陰陽師：京都決戰勝利大岳丸犧牲，暗示SSR八尺瓊勾玉SP大岳丸？

2019-12-23 1,582條評論 4,812次閱讀 1958人點贊

經歷了好幾天的戰鬥，陰陽師正式服的京都決戰終於勝利了，原來昨晚的失敗只是雲外鏡營造的「假象」，真要看還是得看最後這天團結一心、眾志成城的陰陽師，不僅周圍海國大將血量掉得快，就連有兩條血的海鳴都沒能支撐多久，現在時間還剩好幾個小時，各位陰陽師還能抓緊時間肝點寶箱、拯救阿氛。

說實話，對於京都決戰勝利後的劇情，迷惑重重，大岳丸和海鳴的對話中，得知之所以要入侵、佔領京都，也是為了被污染的鈴鹿山，而且這種污染是不可逆的，只能全體「搬家」，那麼……究竟是什麼能污染鈴鹿山，這個有神器坐陣的地方？

其次就是大岳丸，沒有多說廢話，發現鈴鹿山不能恢復，自己率領的海國又搞得京都生靈塗炭，直接獻祭自己，恢復京都的一切。這又帶來一個疑問，就這麼簡單的犧牲自己，沒有多說，對於荒川那邊的劇情伏筆也沒有解釋，所以……最後就留下八尺瓊勾玉和海國作在深海重逢？

而且PV的最後，還有一句「再見，少主」，不同於目前已知式神的音色，根據音色透露的這人年紀應該不大，又能叫少主，那麼有沒有可能是八尺瓊勾玉擬人，類似於神器雲外鏡化為式神，八尺瓊勾玉也有成為式神的一天？

當然，大岳丸應該沒那麼簡單就犧牲，作為SP製造機，或許他最後的「任務」就是把自己SP化？由此可以猜測，後續應該會轉戰地點，要麼是鬼域（大岳丸的靈魂、大江山之戰提到過），要麼是鈴鹿山（調查污染的源頭），或者是劇情中提到的——月宮。

這樣發展，後續SP的式神備選就多了，鬼域有閻魔，鈴鹿山大岳丸（復活？），月宮輝夜姬，看來明年大家的勾玉依舊不能放假啊……（SP金魚姬會有嗎，如果有該怎麼安排？）

綜上可見，陰陽師劇情、地圖涉及越來越廣了，讓你永遠猜不到接下來會有什麼「驚喜」等著你（就像荒川犧牲，大岳丸獻祭），真的希望，別再出「妖命」了！
