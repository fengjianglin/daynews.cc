CF：新英雄級角色重磅來襲，大幅度延長連殺判定時間

天天要聞
2 週 ago

摘要

穿越火線中已經擁有了非常多的角色，從之前的普通角色到後來出了強化型角色，已經有翻天覆地的改變。玩家們在選擇角色的時候也不再只是觀看其外表，而是更加重視角色的功能，看看在實戰中究竟有沒有一戰之力，現在又將出一款全新的英雄級角色，功能也是十分強大。

穿越火線中已經擁有了非常多的角色，從之前的普通角色到後來出了強化型角色，已經有翻天覆地的改變。玩家們在選擇角色的時候也不再只是觀看其外表，而是更加重視角色的功能，看看在實戰中究竟有沒有一戰之力，現在又將出一款全新的英雄級角色，功能也是十分強大。

三防屬性，要說什麼樣的角色比較實用，玩家們一定會首先想到「三防」屬性。所謂的三防也就是減少受閃光彈影響的時間，減少角色高處掉落受到的傷害和防止友方手雷的傷害。而三防屬性的確隨著遊戲的發展，角色更新頻率加快，而不再那麼稀有了。

連殺的代價，新角色的一個非常亮眼的功能，可以大幅增加連殺的判定時間，從而讓玩家們取得更多的連殺機會。如果再配合上飾品的連殺延時功能，相信拿個五連殺、六連殺都不是什麼困難的事情了，而且此功能在全模式通用，可以說是非常實用的功能了。

積少成多，其實也就相當於一個增加經驗值的道具，每當玩家對敵人輸出100傷害時，可以獲得1擊殺相應的經驗值。也就是說，玩家在局內對戰時，就算沒有成功擊殺敵人，打出的傷害都會記錄並在賽後增加角色獲得的經驗值。這個功能只適用於一些競技模式，因此想要快速升級的玩家一定非常想擁有這樣的功能。

角色覺醒，這個功能各位玩家應該並不陌生，之前也有別的角色有覺醒功能。在挑戰模式中按E鍵後可以讓角色發動覺醒。覺醒效果還是非常給力的，發動覺醒的玩家可以獲得生命值自動恢復的效果，並在施放覺醒技能時候對周圍產生強大的範圍傷害，而隊友可以獲得無需裝彈和移動速度提升的效果。相信各位喜歡玩挑戰模式的玩家一定非常喜歡。

當然了，作為一個英雄級角色必不可少的就是經驗值加成，這些基礎的功能就也不多說了，新角色的整體打扮還算可以，不過蘿蔔青菜各有所愛，每個玩家喜歡的外觀也不一樣。
