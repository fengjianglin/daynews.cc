魔獸懷舊服畢業了？當工具人千萬注意，不然拿不到錢還被噴

2019-12-25 989條評論 3,532次閱讀 2023人點贊

魔獸懷舊服經過幾個月的發展，如今玩家群體已經慢慢穩定了下來。遊戲新鮮感一過，能夠留下的，幾乎都是真心想要玩遊戲的玩家，但這些玩家並不都是當初的老玩家。許多玩家當初只是聽說了魔獸世界的火爆，但並沒有玩過。如今趁著懷舊服開啟才第一次進入遊戲，感受魔獸世界的魅力。

魔獸世界之所以被稱為最經典的遊戲，不但因為其優秀的遊戲品質，而是因為諸多的遊戲玩法、俗語、習慣都是在魔獸世界中形成的。

其中許多遊戲文化一直延續至今，稱為遊戲玩家相互遵循的規則。有些規則被帶到了其他遊戲，有些規則獨屬於魔獸世界。許多規則對第一次接觸魔獸懷舊服的玩家來說，可能並不是那麼清楚。

魔獸懷舊服中有著諸多的潛規則，作為新人的你一定要弄清楚，不然可能就會和玩家之間產生糾紛。今天胖大海便和大家一起來看看，魔獸懷舊服中那些約定俗成的潛規則。

由於魔獸世界是一個將團隊協作的遊戲，和國產無腦輸出的遊戲有著很大區別。無論是在打副本還是拍裝備都有一些大家相互遵循的潛規則。例如在打副本的時候，最害怕的是不懂裝懂，如果不會一定要說出來。

許多新人往往為了打副本，而謊稱自己打過。結果卻總在關鍵地方失誤，讓整個團隊跟著浪費許多時間。除此之外輸出低過治療的不能roll裝備，也是許多玩家相互遵循的潛規則。主要為了防止許多玩家划水。但如果是新人，一般還是會有優待的。

該死的時候一定要勇敢。許多新人往往引到怪之後不斷躲避，往往造成整個團隊團滅。遇到這種情況，勇敢的自我犧牲是最好選擇。打團一定要聽指揮，別自作聰明。有需求才roll點，不要胡亂點。

上面這些規則都是大家相互遵循的，新人只要跟著做便好，也並不難理解。但是在魔獸世界中還有一些有爭議的規則，玩家之間並沒有統一的理解。往往每一個團隊都有自己的小規則。這些規則往往便是玩家之間產生矛盾的誘因。

最近一位玩家在遊戲畢業後，被僱傭當工具人打副本。結果在當工具人的時候，卻爆出了一件適合自己職業的裝備。於是直接roll點，拿到後卻引起了其他玩家的不滿。認為既然是被僱傭來當工具人，便不能參與副本roll裝備。

到底工具人能不能在副本中roll裝備？其實這一直是一個非常有爭議的話題。從表面來看，既然碰到了大家都不要的裝備，又剛好適合自己為何不能roll？但作為工具人，被人出錢僱傭過來，首先證明你在副本中已經沒有需求，不能roll裝備是很正常的。不然既然你有需求，那麼為何要掏錢僱傭你？

這兩種觀點各有各的理由，到底誰對誰錯其實並沒有統一的定論。往往在不同的立場有不同的答案。其實想要解決這種問題也很簡單。只需要在打副本之前約定好便行了，或者在roll點的時候徵求大家同意，便不會發生這種事情了。

無論是在打副本還是其他遊戲活動中，將一些可能遇到的爭議提前說出來，許多錯誤都可以避免。這應該是應對所有潛規則的萬能方法了。你覺得作為一個工具人，可以在副本roll裝備嗎？歡迎評論，說出你的觀點。
