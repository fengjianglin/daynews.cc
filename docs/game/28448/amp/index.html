暗帝現在什麼定位？東二夢想打樁測試，紅13暗帝千億起步

天天要聞
2019-12-23

Dnf職業平衡一直都是玩家們非常關心的事情，而策劃也是一直都在為平衡而做著努力，但是畢竟作為一個遊戲來說，絕對的公平是絕對不可能存在的，要不然這個遊戲也是失去了意義，所在能夠在dnf這個遊戲裡面能看到的就是每一個版本都會有版本之子，他們也被稱為幻神職業，當然與其對應的就是一些下水道職業了。

而在這兩種職業之間，還有一類職業他們的狀態不高不低，一般稱之為是34c的職業，他們沒有幻神職業的強大輸出能力，但是卻同樣也有著不俗的傷害，最關鍵的是他們還有一些控制或者說輔助的能力，在團本裡面因為這些綜合起來就變成了非常受歡迎的職業了，今天要來談的這位就是34c之王暗帝了。

說起暗帝這個職業第一個想到的一定就是她的控制能力了，沒錯，這個職業剛開始出現就是因為超強的控制能力而受到大家的喜歡，尤其是在安圖恩版本裡面，加上異界的加持，控制基本上是無縫連接的。但是這麼多版本走下來，暗帝好像也只有她的控制能夠說上一說的，輸出能力真的難堪。而在上次的版本改動裡面，暗帝也是被策劃照顧，進行了一次特色的加強，最主要的是她的被動技能大加強讓這個職業終於擺脫了只能控制的困局，一躍成為34c職業。

那麼這個職業現在的狀況到底是如何的呢？最近東二夢想就打造了一個超強的暗帝，而且在修鍊場也是進行了一波傷害測試。這一次他打造的暗帝全身紅13起步，一些細節打造方面也不用多說，堪稱完美，那麼這麼強大的暗帝在修鍊場能打出多少傷害呢？在東二夢想的測試裡面，他打出了1000億出頭的傷害。

說實話這個數據雖然在一些幻神職業手裡面一般，但是要知道的是現在國服裡面高端暗帝本來就少，再加上這個職業和現在的幻神劍帝以及召喚差距甚大，所以說能夠打出這個傷害已經很不錯了。對於暗帝來說，其現在的特色還是控制，不管版本如何變化，只要她的控制不削弱，那麼她在團本裡面永遠會有一席之地的。就算現在的傷害不行，但是誰又能確信，以後的版本裡面暗帝不會被加強呢？所以說玩暗帝的玩家一定要忍住啊！
