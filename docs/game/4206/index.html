“吃雞”更新一套新衣服，成為“光子自己人”可獲得，有錢買不到

2019-12-09 0條評論 14次閱讀 0人點贊

歡迎諸位小夥伴們來到本期天哥開講的《和平精英》「精英那點事兒」~接下來呢，咱們聊聊聖誕模式里的「特效槍」、在「廁所」里使用無人機觀察細節以及成為「光子自己人」才可以獲得的新衣服等有趣的玩家遊戲經歷與發現~

得，廢話不多說了，還是各位小夥伴們熟悉的「老配方」：有圖有證據，邊看邊聊天兒吧~

玩家在「聖誕模式」里拿著DBS，手持「特效冰棍槍」！

相信諸位小夥伴們早已經知曉，目前《和平精英》的12月版本已經在體驗服開啟了測試~此前有玩家稱「聖誕模式」在內測的時候就出現過了，事實上在12月版本里測試的該模式與內測期區別非常大~不僅空投飛機變成了「空投聖誕老人」，而且就連新武器、新道具也出現了~上圖呢，我們可以開到玩家此時拿到的武器就是空投新武器DSB霰彈槍，其威力自然毋庸置疑，不過出現概率比AWM都要低哦！

再來說說《和平精英》遊戲里出現的首個「特效槍」吧，這把槍名為「冰霜手槍」，攻擊敵人或者隊友都可以將其冰封起來，原地變成了一個「冰棍靶子」~不過呢，該武器也同樣需要在空投中撿取，但是掉落概率極高。不過考慮到其射程較近，所以目前為止僅能當作一個「玩具槍」來使用！

玩家在「廁所」里用無人機，長相很奇怪，來自其他遊戲？

同樣與聖誕模式一起「回歸」的極寒模式也在《和平精英》12月新版本里出現了，光子也在原有的基礎玩法上進行了一系列的升級改變，比如增加了滑雪板、無人機等等~其中無人機可以用來查看周邊的環境~上圖呢，則是玩家在決賽圈裡的一個「廁所」里使用無人機，從隊友視角來觀察無人機從窗戶上飛出去時，忽然覺得長相有些奇怪，似乎很眼熟似的~

這位玩家告訴天哥，他覺得無人機像極了「明日之後」裡面的裝備，甚至還揣在這些道具裝備其實也是為了光子的新游「代號」而做準備，就如同「特種作戰」模式里的5大職業那樣，通過多種技能來提前為「代號」做測試~不過呢，這種說法純屬玩家們的猜測，截止目前為止光子並沒有任何相關的信息公布，所以也僅僅是猜測而已哦~

更新一套新衣服，成為「光子自己人」可獲得，有錢買不到

提及《和平精英》遊戲里的衣服呢，相信諸位都非常熟悉了，目前遊戲中的衣服獲得方式由任務、開箱、輪盤抽獎、服飾幣兌換這4種組成，老玩家們對於每一件衣服都「如數家珍」，可以說出其獲取方式以及出現的時間~而最近呢，一位小夥伴找到天哥詢問上圖中這件衣服的獲得方式，這套衣服全身白色，搭配紅色短款雪地靴，看起來相當酷炫~通過多方打探才發現，這套衣服是光子悄然更新的一套新衣服，僅有成為「光子自己人」之後才可以獲得~

所謂的「光子自己人」則是成為視頻巡查員，通過遊戲中檢測外掛數據的任務來獲得這套名為「專家巡查員」的新衣服，而且要達到「高級巡查」並且賽季巡查達到25級這兩種條件才能得到！所以這套衣服哪怕是有錢也買不到，僅能成為「萬里挑一」的視頻巡查員之後，才可以通過任務的方式來獲得，所以普通玩家與這套衣服幾乎是「無緣」了，因為此前有巡查員玩家稱目前全服僅有數百個巡查員而已，能夠被選上的幾率相當低~

得，本期的《和平精英》「精英那點事兒」就和大家聊到這裡啦，感謝各位的閱讀，接下來我們評論區里見，一起聊聊吧~

PS：別忘了點個關注和贊哦，原創手打不易，謝謝！
