LPL三大AD缺席德杯，阿水被RNG合同坑了？Mlxg闢謠PDD透露內幕

2019-12-19 57條評論 1,521次閱讀 476人點贊

英雄聯盟德瑪西亞杯將在後天21號正式開戰，各隊也提前宣布了參賽名單，不過這回德杯的看點會少一點，因為LPL三大AD都缺席了。Jackeylove就不用說了，現在大家連他的下落都未能知曉，都不知道他接下來會去哪個隊伍。而FPX戰隊在奪得S9世界冠軍以後，隊員都有些疲憊，一直想好好休個假，因此他們決定採用全新人陣容，Lwx是不上的了。RNG戰隊也宣布Uzi因傷病而暫離賽場，希望他能好好休息。

Uzi在賽場上征戰了這麼多年，老觀眾都知道他的傷病很嚴重，打比賽時後背經常貼滿了運動貼布以緩解疼痛，平常還要接受針灸等治療，以前也經常需要休賽養傷。但是今年Uzi幾乎沒有休息過，只在年初讓Wink頂替了兩場比賽，之後就一直征戰不停，只想在世界賽重新拼搏一回。

然而Uzi的拼搏卻換來了一個更為糟糕的結局，RNG止步十六強，這讓他大受打擊，網路上傳出了各種Uzi退役的流言。好在Uzi並沒有放棄，RNG戰隊與他成功續約，他只是暫時休戰養傷，為明年S10世界賽養精蓄銳！

但是在RNG戰隊官宣德杯名單的同時，貼吧卻傳出了一個詭異的消息，Jackeylove缺席德杯居然與RNG戰隊有關？此前就有流言稱阿水加入RNG，小虎轉上Uzi轉中，不過這個劇本跟小說一樣離奇，大部分網友只是一笑了之。

而根據這名貼吧網友所述，EDG老岳直播爆料稱圈內人原本以為Uzi打算退役了，阿水表哥就讓他跟RNG簽了一份意向合同，如果阿水轉到其他隊伍得付一千萬違約金。SN戰隊跟RNG有合作關係，不必付違約金就能買阿水，但因為打野WeiWei臨時轉AD打出了奇效，他們不想買了。EDG戰隊看到千萬違約金以後望而生畏，直接放棄了。現在只剩TES戰隊有資本掏出千萬違約金解救阿水，否則他只能給Uzi當替補了。

阿水爭奪戰的劇情越來越離譜了，即使這種爆料也有人深信不疑，在貼吧論壇瘋狂傳播，稱他陷入合同糾紛，春季賽要涼了。那麼JKL真的被RNG合同坑了嗎？其實這個所謂的爆料漏洞百出，根本沒有可信度。

首先JKL想知道Uzi有沒有退役，他發條簡訊就知道了，他與小明Uzi的關係還是很不錯的，怎麼可能沒確認情況就簽合同了？而SN戰隊跟RNG戰隊的關係有那麼好嗎？連一千萬都不用給？打野轉AD的選手訓練賽表現再好，不可能頂替S8冠軍吧？意向合同一般就是有這種簽合同的想法，但是沒有合同的效力，RNG也沒理由索要千萬違約金……

總而言之，這個所謂的爆料是一個爛瓜，估計是有人故意帶節奏黑阿水或者RNG戰隊的。實際上昨晚Mlxg已經在直播間里闢謠了，他是這樣說的：「你們去微博看一下，我馬上官宣要去SKT了……真聽不出來是開玩笑還是認真的？沒這麼容易信吧？我傻了，有點辨別是非的能力好吧？就跟阿水來RNG一樣，這到底有誰信啊？」

阿水去RNG就等於Mlxg去SKT，這都是不可能發生的事情，他根本沒理由跟Uzi競爭首發。網友還是別胡思亂想了，那些故意放料的人非蠢即壞，只是借著阿水這個工具人來黑RNG罷了。

那麼問題來了，轉會期都結束了，阿水為什麼還沒與新隊伍簽約？難道他想放棄職業回去當主播了？昨晚PDD也在直播中透露了內幕：「阿水可能會短暫休息三個月左右，下個賽季初再登場！」

其實JKL目前是自由人的身份，而自由人是不受轉會限制的，他隨時可以簽約上場打比賽，根本不需要急於一時。既然阿水不打算參戰德杯，那他可以等春季賽再正式選隊，現在不如和另外兩名ADC一樣休息放鬆，畢竟他也是全年無休的。這樣一看，PDD的爆料還是蠻合理的！

看來大家都不需要擔心JKL的情況，他只是給自己放個假罷了，等到春季賽開始才能得知阿水爭奪戰的結局，現在還是先關注德杯比賽吧！那麼對於Mlxg闢謠RNG阿水的這件事，大家有什麼看法？歡迎留言，分析你的看法！
