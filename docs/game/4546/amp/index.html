魔獸世界：PVP轉PVE或將到來，戰場很難挽回等級落後的玩家

天天要聞
2019-12-06

摘要

《魔獸世界》第二階段其實有點失敗，榮譽系統先解鎖，而戰場則是放在第三階段，這就導致了野外PVP的變質，為了榮譽經驗而組團在野外實行PPPPVP，加劇了陣營的失衡，讓那些進入50級地圖的未滿級玩家無法繼續練級，劣勢陣營即使滿級玩家想要進副本都得跑屍大半個小時。

《魔獸世界》第二階段其實有點失敗，榮譽系統先解鎖，而戰場則是放在第三階段，這就導致了野外PVP的變質，為了榮譽經驗而組團在野外實行PPPPVP，加劇了陣營的失衡，讓那些進入50級地圖的未滿級玩家無法繼續練級，劣勢陣營即使滿級玩家想要進副本都得跑屍大半個小時。其結果就是把那些相對休閑的懷舊玩家勸退，不得不AFK或者轉頭去PVE服。

榮譽系統解鎖伊始，就開始出現了不少希望提早解鎖戰場的聲音，榮譽上線的第一周結束，希望戰場提前的內容就佔據榜首。暴雪也看到了玩家的反饋，並以實際行動支持了玩家的意願，把戰場從第三階段分離出來，提前到12.10號。從這就可以看出，暴雪現在已經不敢得罪玩家，當玩家的反饋聲音出現，暴雪就會迎合玩家的意願而做出變動，這算是一件好事吧，畢竟榮譽系統不包含戰場，確實不是個明智的做法。

關於戰場玩家的反饋得到支持，雖然讓休閑且落後於第一梯隊的玩家看到了希望，但並不能讓這些玩家真正的找回曾經的信心去繼續遊戲。持續一個月的糟糕的野外練級環境，已經磨滅了不少情懷玩家的信念，戰場解鎖後或許能改善野外的練級環境，可就真的能拉回那些已經因此而AFK的玩家？答案是很難！以目前的情形看，大部分因為野外被PPPVP而留下陰影的玩家，都不大期望戰場能讓他們繼續遊戲。喜歡玩手游的可百度三刀俠客網即三大不留點xxyoy點康姆 ，神兵神裝上線就送，而且遊戲幣和資源無限用，不氪金就能成為大佬哦。

在藍貼放出了戰場提前的消息之後，截至到現在，已經越來越多的玩家表示需要儘快開啟轉服，不是單純的限制模式，而是放寬限制能從PVP轉到PVE。因為不能轉服，如果放棄PVP，那隻能在PVE裡面重新練一個號，這對於不喜歡AA隊的玩家來說，那是相當難受。全服互轉的聲音已經很大，暴雪肯定也能看到，為了讓玩家繼續遊戲，沒理由不迎合這些玩家。

所以，現在美服就有傳暴雪將會在聖誕前後正式開啟PVP和PVE的轉服功能，跟正式服一樣需要在商城裡用戰網點來獲取轉服機會，而同模式下的一些大服和鬼服之間的互轉將繼續維持當前的機制（無需戰網點），後續才會進一步放寬更多的戰網點轉服條件，這其實對部分玩家和暴雪來說都獲益的方式。

總之，以現在懷舊服的局面，暴雪方面提到因為懷舊服而增加了魔獸玩家數量，卻沒得到實際到手的成果，玩家既然想要用戰網點來獲取轉服機會，何樂而不為呢。而從戰場的提前看，美服玩家的反饋很快就會得到暴雪的支持，此次的轉服必然會如戰場一樣滿足玩家的意願，不管是暴雪或者玩家都不想因此而讓玩家流失。
