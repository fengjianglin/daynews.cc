DNF：韓服體驗服劍帝三覺詳解，新增95級被動和主動技能成幻神

2019-12-19 155條評論 3,873次閱讀 2745人點贊

韓服體驗服在12月18日更新了100級版本先行體驗，我們進行了測試體驗，原來國服的特色加強已經完全照搬到了韓服，所以以後沒有特色加強來說，甚至韓服的劍帝要比國服更強了，新增的國服的特色加強包括了返本歸元和三花聚頂的調整，還有技能攻擊力的增加，此外還有一個特別的改版，那就是二覺刪除了掌風的攻擊，演出速度更快！

現在的二覺進行斬擊後，直接跳斬，掌風部分刪除，並因為刪除掌風的攻擊而增加了4次多段攻擊的斬擊傷害，總傷害並提升12.3%。

三覺分為全職業三覺上篇，和女鬼劍三覺下篇，覺醒任務超級簡單，不需要擔心難度。

上篇覺醒技能新增

上篇覺醒不需要任何的刷圖，對話和進圖對話循環兩次就OK了。完成三覺上篇的全職業新增3個被動技能，提升屬性一致，並且覺醒後二覺技能自動學習，不消耗SP。以劍帝舉例子。

[剎那的領悟]全職業95級被動技能，學習後增加20%的基礎攻擊與轉職後技能攻擊力，每增加1級提升2%，100級僅能學習到2級，增加22%的技能攻擊力。

[一步之遙]全職業95級被動技能，學習一級精通，增加5%的技能攻擊力，上限11級。

[覺醒之機]全職業100被動技能，增加15%的一覺二覺技能攻擊力。

三覺下篇技能

目前僅有女鬼劍擁有下篇覺醒，學習後原被動技能刪除，改為新被動技能和主動技能。

被動【劍仙之境】，學習後1級增加20%的技能攻擊力，每增加1級提升2%的技能攻擊力。

95級主動技能【業海苦獄】，使用時沖向敵人施放內徑攻擊，之後產生大範圍爆炸攻擊。這一個技能是一個二段攻擊技能，第一段出手極快，第二段類似於湮烈掌的爆炸攻擊，但是技能的外觀和攻擊範圍非常大。

三覺主動技能【月夜圓舞】，這一個主動技能就不用多加說明了，這一個技能僅有一次斬擊傷害，是劍帝當前最高的百分比技能。

但是當前有一個非常有趣的設定，三覺僅能需要選擇一覺或者二覺共享CD，這一個設定應該是為了平衡玩家的傷害，因為沒有三覺的職業100級被動是增加15%的一覺或者二覺傷害，所以劍帝選擇施放三次覺醒就會失去一個覺醒的傷害，從而保證職業傷害的平衡。

此外95級的不被動[一步之遙]完成下篇後刪除，改為一個主動技能，相當於5%的技能傷害換了一個主動技能輸出，從而達到職業傷害平衡，所以策劃這一次的玩法設定確實經過了細節考慮。視頻版本晚點發。
