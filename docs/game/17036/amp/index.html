更新三位新英雄 百花齊放時代來臨！雲頂9.24更新公告詳解

天天要聞
4 天 ago

哈嘍~這裡是小司！美服在今日凌晨發布了更新公告，國服也將在明天同步更新至9.24版本，9.24版本加入三個新英雄，並且平衡改動使得下個版本將會更加的百花齊放，我們一起來看看都更新了什麼吧！

UI更新

·滑鼠懸停在「購買經驗」按鈕上時會顯示需要多少金幣到達下一等級

·現在會在面板顯示當前的等級和進度

·現在會顯示當前等級的抽卡概率

·互換了「刷新商店」和「購買經驗」的位置，請注意不要點錯啦

·增加了一個按鈕，當你瀏覽其他玩家的棋盤時，點擊此按鈕能快速回到自己的棋盤畫面

·在選秀時瀏覽棋盤是點擊此按鈕能快速回到選秀畫面

解析：人性化更新，鼓掌就完事了！

新英雄

賽娜

價格：2金

種族：影

職業：系魂

技能：黑暗洞滅，賽娜以最遠的友軍單位為目標，射出一道可以穿透友方和敵人的黑暗之光，被命中的敵人受到50/100/150的魔法傷害，被命中的友軍英雄會獲得額外的25/45/65普攻特效魔法傷害。

盧錫安

價格：4金

種族：光

職業：系魂

技能：聖槍洗禮，盧錫安朝著敵人發射10+(2*每秒普攻次數)顆子彈，每顆子彈造成25%/30%/50%物理傷害，附加攻擊特效，附帶30/40/50額外魔法傷害。

阿木木

價格：5金

種族：地獄火

職業：守護神

技能：木乃伊之咒。阿木木用繃帶將周圍的敵人統統纏住，對2/3/4格內的敵人造成150/250/1337的魔法傷害，並且可以把他們眩暈，持續2秒。

新職業羈絆

系魂

當其中一個系魂英雄死亡的時候將進入幽靈狀態，變得無法選定，保留所有裝備和戰鬥屬性加成，並戰鬥直至友方系魂英雄死亡。

解析：當賽娜與盧錫安組成2系魂後，死亡的一方會繼續戰鬥，只有當他們兩個都死亡才算真的死亡。這兩夫妻的持續作戰能力簡直了，可以考慮給他們兩個其中一個帶肉裝，另一個帶全輸出裝，可以大大的延長輸出時間。

元素羈絆改動

雲霄 2/3/4

全體友軍提升15%/20%/30%的閃避幾率→20%/25%/35%

解析：雲霄劍刺的雲霄羈絆更加有用了。

雷霆 2/3/4

雷霆英雄在暴擊或受到暴擊時對相鄰敵方單位造成70/250/500→100/300/500點傷害。

解析：版本更新後用狗熊和奧恩組成2雷霆在前期更加好用了，並且後期找到劫後組成3雷霆提供的傷害更高了。

地獄火 3/6/9

觸發機制:由技能觸發改為技能和暴擊觸發

地獄火英雄的技能和暴擊在接下來的5秒內灼燒地面，對目標額外造成80%/175%/275%的魔法傷害→70%/120%/160%

燃燒地面範圍:1/1/1格→1/3/5格

解析：新版本的地獄火效果更容易觸發了，但削弱了總傷害的同時減少了輸出時間，所以每秒造成的傷害提高了。並且6地獄火能同時點燃3格，9地獄火同時點燃5格，也就是說英雄會更容易踩到燃燒的地板，主要是有些英雄走動比較多，會規避掉大量的地板燃燒傷害。之前的地獄火羈絆很難利用地板燃燒的優勢，那麼現在6地獄火將會比之前更加好用了。

光 3/6/9

光英雄死亡時，基於自身最大生命值的25%對其他光英雄進行治療，並提升他們10%/20%35%的攻速→15%/25%/35%

解析：光在上版本修改了回血機制後直接淪為了下水道羈絆，這次的強度提升或許能讓光更有用一些。

海洋 2/4/6

每4秒為隊友回復15/35/60法力值→15/30/60

解析：9.23提升了4海洋的法力值回復，使得海洋法成為版本一大霸主，所以就有了這次的回調，是有一定的削弱的。

影 2/4→3/6

影英雄在開局會獲得4秒的傷害加成。

3影：在影英雄自身參與擊殺時刷新羈絆效果，在4秒內提升自身70%的傷害。

6影：在任意影英雄造成擊殺時刷新羈絆效果，在4秒內提升全體影英雄140%的傷害

解析：由於賽娜的加入，影英雄現在有6個單位了，所以進行了羈絆節點的調整，不然太容易就湊到終極羈絆了。羈絆更難湊的同時收益也更高了，新版本中影依然是非常強勢的羈絆。

職業羈絆改動

狂戰士 3/6

狂戰士的攻擊有40%/100%的幾率對目標背後錐形區域造成傷害

現在6狂戰士羈絆還會給狂戰士額外提供25攻擊力

解析：6狂戰士在9.23基本沒有人會玩，而現在6狂戰士額外提供的25攻擊力是一個挺大的提升，應該能讓他回到大家的視線當中

遊俠 2/4/6

2/4遊俠每3秒有30%/60%/的幾率在接下來的3秒內攻速翻倍→35%/80%

6遊俠每3秒有100%的幾率在接下來的3秒內攻速提升250%

解析：這回遊俠的提升夠大了，特別是4遊俠和6遊俠，6遊俠的2.5倍攻速非常的香，但同時陣容的組成就會缺少前排，除非非常胡能夠瞬秒對面，不然還是很難用。新版本想玩遊俠會比較推薦4遊俠，有輸出也有前排容錯率更高。

召喚使 3/6

增加召喚生物40%/100%的生命值和持續時間→30%/120%

解析：對3召喚使的削弱很明顯就是針對版本強勢的煉獄法海了，3召喚使帶來的提升太大了。在削弱3召喚使的同時大幅增加了6召喚使的收益，鼓勵玩家去嘗試6召喚使的陣容玩法。
