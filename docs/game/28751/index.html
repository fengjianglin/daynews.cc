射手一哥一姐即將跌落神壇，四件裝備調整，金幣溢出解決方案推出

2019-12-22 1,031條評論 3,118次閱讀 1056人點贊

本文原創首發於公眾號：電競怪客

hello，大家好，我是怪客君~關注並私信我「戰令」即可領取戰令進階卡！

體驗服前天進行了一波更新，目前已經開啟新賽季測試。其中大家最為關注的新娛樂模式「覺醒之戰」已經正式開啟測試，預計將在下賽季版本更新的時候同步上線正式服。本次測試只開放31名英雄，具體介紹請看今天的第二篇文章，對新模式感興趣的小夥伴們不要錯過哦~

之前體驗服總是對冷門英雄調整，試圖打撈下水道英雄的熱度，而12.19體驗服對四位熱門英雄進行了調整，這四位英雄分別是孫尚香、馬可波羅、諸葛亮和不知火舞。另外還有三件裝備調整，這次調整對於我們的日常排位造成怎樣的影響呢？一起來看看吧！

英雄調整

▶孫尚香：削弱

怪客解析：二技能命中後的穿透效果百分比降低，這對於孫尚香來說是一次不小的削弱。因為孫尚香的傷害主要來源於21連招，基本上沒人能吃得消大小姐的一套。而之前對於大小姐的削弱都沒有動刀其傷害的根本，所以一直以來大小姐一直保持著全期強勢。這次的調整，讓疊加穿透效果後傷害不會過於爆炸，是其他玩家的福音。對於孫尚香來說，主要是削弱了前期的傷害，發育周期將會變長。

▶馬可波羅：削弱

怪客解析：削弱了馬可波羅13技能的基礎傷害，作為補償加強了Ad加成和減少了能量消耗。雖然Ad加成增加了0.02Ad，但是總體傷害量還是降低的。而所有技能能量消耗從75減低至45，可以更加頻繁的使用技能，提高馬可波羅的容錯率。

▶諸葛亮：加強

怪客解析：2技能的冷卻時間指的是2技能使用的間隔時間，不是存儲時間，所以這次冷卻時間減少0.5秒，意味著諸葛亮可以更快的使用2技能，連招使用起來會更加靈活。不過數值上來看0.5秒好像影響不大。

▶不知火舞：削弱

怪客解析：不知火舞的二技能在命中敵人後會削弱敵人的抗性，這次調整讓不知火舞二技能削弱敵方抗性效果加成降低，等於削弱了不知火舞一套技能打出來的傷害。不過對於沒有出魔抗的脆皮英雄來說沒啥影響，該秒的還是會秒。

裝備調整

1、時之預言：平調

怪客解析：法術攻擊降低20點，作為補償增加被動雙抗效果，加強前中期防禦能力。以前幾乎要六神裝才能觸發全部雙抗，現在前中期就可以得到很好的防禦效果。

2、風之輕語/救贖之翼：加強

怪客解析：自從上次裝備調整，減少了護盾值，去掉了移速加成，救贖這件輔助裝的使用率下降了不少。不過護盾效果在調整之後變成了集體加盾，更適合在團戰中使用，所以這次救贖的護盾值加強，會提升這件裝備在輔助裝中的地位，有可能會替代現在更受歡迎的奔狼。

3、衝擊鎧甲：加強

怪客解析：降低延時眩暈的延遲時間等於降低容錯率。不過這件裝備對於使用時機需要有一定的判斷能力，喜歡玩亞瑟、哪吒這類戰士英雄的玩家不妨嘗試一下。

紅包系統優化

最後是關於戰隊紅包系統的優化，新賽季戰隊紅包除了可以發鑽石外，還可以發點券、榮耀積分、英雄、皮膚、英雄碎片和皮膚碎片紅包。這也是金幣、碎片等資源溢出的一種解決方案。（心細的小夥伴也可以看到紅包背景與五嶽皮膚相關）

以後不僅可以在戰隊發紅包，還可以在全服發紅包，而且當單個紅包金額超過1000鑽石時，還會觸發紅包雨特效。

好啦，以上就是12.19體驗服更新的主要內容。對於英雄和裝備調整，以及紅包系統，你有什麼想說的？歡迎留言評論。
