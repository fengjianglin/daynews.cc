傳奇：遊戲里三個職業當中，道士到了後期真的最厲害嗎？

2019-12-09 0條評論 15次閱讀 0人點贊

摘要

傳奇是一款非常經典的MMO類遊戲，其中最具有代表性的無非就是遊戲當中，戰士、道士、法師三個職業相互對立，卻都帶有不同的特色了。有些人說法師在後期當中是最為強力的職業，也有人說道士到了後期比法師更厲害。其實不同的玩法，不同的裝備，打造出來的角色到最後的強度也是存在一定的區別的。

傳奇是一款非常經典的MMO類遊戲，其中最具有代表性的無非就是遊戲當中，戰士、道士、法師三個職業相互對立，卻都帶有不同的特色了。有些人說法師在後期當中是最為強力的職業，也有人說道士到了後期比法師更厲害。其實不同的玩法，不同的裝備，打造出來的角色到最後的強度也是存在一定的區別的。今天筆者就來談談這三個職業，都分別有什麼優缺點，是否道士到了後期就真的是三個職業當中最強力的呢？

法師：

​玩過這款遊戲，特別是常常參加攻沙的老玩家，肯定會對這個職業印象特別深，畢竟這個職業是所有職業當中輸出最高的職業，特別是後期學會了3級盾的法師，基本上你是砍不動他的。並且這個職業前期群攻的技能也特別多，升級速度也是十分迅速，唯一讓然感到可惜的地方就是血量太少。

戰士：

戰士應該算是一個比較費錢的職業了，為什麼這麼說呢？首先是因為前期戰士的命中率不高，傷害也總是差點意思，所以前期想要升級基本上都要花錢讓老玩家帶你打怪。到了後期慢慢裝備起來之後，這個職業就開始強勢了，基本上PK都能憑藉一個衝撞而先發制人，整體來說後期基本上戰士都是戰無不勝的。超強的輸出能力，再加上比別人高几千滴的血量，真的強力。

道士：

​道士後期學了狗之後，可以說體驗也會提升一個等級，但是無奈這個職業的續航能力好，但是整體的爆發能力卻遠遠不如法師，雖然別人打你，你能加血和隱身逃走，但是要持續輸出往往道士是很難做到的。特別是到了後期階段，法師的盾已經有了，戰士的血量也養起來了，你基本上很難殺死對方，總有種無力感。

今日推薦：《魔神單職業》今晚新區

全新定義神途三職業，三職業不再是傳統的戰法道，大家都知道戰法道三職業各有特色，但是在單對單PK上，總是戰士強，法師道士偏弱。心晨遊戲《魔神》這個三端互通版本雖然說是單職業，卻可以轉職成三個不同的職業，全新定義了三大職業，由三個近身戰士職業組成。

​

單職業這些年發展很快，很適合現在的玩家玩，各個層出不窮的構思令人讚歎。這個版本的構思真是令人稱讚。歡飲各位老鐵進服品鑒！
