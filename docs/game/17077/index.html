12月23日體驗服大改，先知主宰2分鐘刷新，龍王護盾可恢復

2019-12-14 1,007條評論 2,873次閱讀 1788人點贊

大家，我是小正，今天體驗服迎來一次非常大的改動，我們熟悉的王者峽谷即將迎來翻天的變化，千萬要提前知道。下面小正帶大家看看吧。

峽谷調整

1. 對抗路

以前遠程兵的生命回復改為增加經驗，經驗兵額外提供65點經驗，對抗路的英雄大多數再帶回復，所以生命兵的意義不大，這次直接改為了經驗兵，對抗路就有了等級壓制，算是給了對抗路一點發揮的空間。

2. 中路

中路河道之靈生命調整至與邊路河道之靈一致，但是經濟和經驗獎勵和外網一致（遠高於邊路河道之靈）

現在中路河道之靈其實挺雞肋的，血量很厚，中路法師自己一個人很難收掉，所以大多數情況下都是選擇支援，更多是讓給路過的打野，這次調整可以讓中路法師自己solo掉了，成了一個資源點。

3. 先知主宰/主宰

先知主宰首次刷新時間點：第5分鐘 → 第2分鐘

先知主宰基礎生命：22000 → 12500

先知主宰基礎攻擊：360 → 200

先知主宰再次刷新間隔調整為4分鐘

先知主宰擊殺全隊獲得50點經驗和40點金幣 → 擊殺全隊獲得50點經驗和25點金幣

主宰擊殺全隊獲得200點經驗和100金幣 → 擊殺全隊獲得200點經驗和50金幣

調低了主宰和先知主宰的生命，修改了先知主宰的刷新時間，降低了擊殺boss後的金幣，也意味著第一條暴君和主宰同時出現，第一條暴君不一定必強了，總體來說就是讓前期更容易打架。

4. 先知先鋒/主宰先鋒

主宰先鋒基礎生命：5000 → 3500 (在10分鐘成長至與之前一致)

主宰先鋒基礎攻擊：200 → 150 (在10分鐘成長至與之前一致)

主宰先鋒基礎經驗：145 → 30

主宰先鋒第10分鐘成長到滿經驗：290 → 150 (約為一波兵線經驗的50%)

主宰先鋒基礎金幣：60 → 120

主宰先鋒第10分鐘成長到滿金幣：90 → 150

先知主宰降低生命，更容易防守，並且提供了更多的金幣，意味雙方需要考慮拿先知主宰是否真的划算了，如果不能組織一波有效的進攻，就是給敵方送經濟。

5. 暴君

暴君擊殺全隊獲得280點經驗和60金幣 → 擊殺全隊獲得50點經驗和100(每分鐘額外增長5)金幣

暴君再次刷新間隔調整為4分鐘

調整了暴擊刷新時間，以前3分鐘，同時降低了第一條暴君的經驗，意味著第一條暴君影響不是很大了，因為50點經驗不會讓全員到達4級，重點還是暴君和先知主宰的取捨。

6. 黑暗暴君

暴君擊殺全隊獲得200點經驗和60金幣 → 擊殺全隊獲得75點經驗和150金幣

同樣降低了黑暗暴君的經驗，不過好在buff沒有改。

野區調整

1. 小型野怪

刷新頻率：小怪刷新間隔 70秒 → 80秒

2. 中型野怪（腥紅石像、蔚藍石像）

Buff持續時間 70秒 → 80秒

刷新頻率：紅藍BUFF怪刷新間隔 90秒 → 100秒

3. 野怪成長

金幣收益：10分鐘成長為基礎的150% → 10分鐘成長為基礎的200%

基礎血量：10分鐘成長為基礎的200% → 10分鐘成長為基礎的250%

基礎攻擊：10分鐘成長為基礎的125% → 10分鐘成長為基礎的150%

4. 腥紅石像Buff

每次普攻擊對敵人造成30%減速持續2秒，並附帶2段灼燒效果，每段造成20~48點傷害 →

每次普攻擊對敵人造成30%減速持續2秒，並附帶2段灼燒效果，每段造成30~128點真實傷害，傷害效果對野怪會強化為60~256點

降低了野怪刷新時間，但是成長金幣增加，其實對於打野是好的，畢竟野怪是刷不完的，成長金幣帶來的收益應該會更大點。

5. 風暴龍王

獲得感電：對身邊範圍內的非友方單位自動進行雷擊，每擊對英雄造成4%最大生命真實傷害。

獲得雷電護佑：直接獲得基於自身最大生命50%的護盾

→

獲得感電：對身邊範圍內的非友方單位自動進行雷擊，每擊對英雄造成5%最大生命真實傷害，這個傷害效果對英雄以外的單位會增強為15%

獲得雷電護佑：每次脫戰額外5秒後會都可獲得基於自身最大生命30%的護盾，護盾在風暴覺醒buff持續時間結束後不會立即拔除

增加了雷電的傷害，增加了對野怪和小兵額外傷害，護盾不在是一次性的，改成了脫戰後自動獲得，不過只有30%，相對來說這個龍王buff更加靈活了。

總之，這次改動是非常大的，就是不知道能不能活出體驗服！
