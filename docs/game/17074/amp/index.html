DNF：12.13體驗服更名！春節版本即將更新，新副本帶來職業改版

天天要聞
2019-12-14

一轉眼2019就要結束了！作為最近幾年內容最為缺乏的一年，玩家們已經期待DNF趕緊更新，最好是穿越幾個版本直接奔向100級，而12月三大版本已出其一，剩下的兩個就是「14號的發布會」和「春節版本」，無論哪個更新對於玩家來說都很重要，前者是玩下去的動力！後者是變強的程度，而發布會未來，體驗服卻已先行！名字又改了……

12.13體驗服更名！春節版本即將更新，新副本帶來職業改版

12.13小更新，體驗服的名字從阿拉德變為了魔界，這是第三版了，天空之城、西海岸改成阿拉德、天界，如今又更名為魔界，很明顯，這是大版本更新的前奏！而春節版本是1.14實裝到國服，如果按照往年測試的規律，提前1個月到體驗服，那麼也就這幾天的事咯。國慶套才下架，新的禮包已經備好！

其實改為魔界並非沒有來源，那是因為國服僅剩的改版「魔界大戰」副本必須要上線了，本來是12.10的內容，但1.14的春節版本SPK並沒有打算同步100級，那麼大概率這個「魔界大戰」就是春節版本的內容了，玩家都覺得沒勁，因為「魔界大戰」實質就是全職業技能帶來改版的職業平衡！

一些職業得到了技能新效果後，傷害大幅度提升！加強多的可以增加10~20%的輸出，而一些增加技能歡樂程度的，比如魔道和天帝，只能增加5~10%的輸出，這樣的結果就是DNF下個大版本全職業再次洗牌，不少下水道會翻身，當然也可能更加下水道……

另外第三批得到無我加強的職業也該來了！劍魂是試驗品優先加強，暗帝和戰法等職業的改版則是確定無我技能會普及，而在國服較弱的職業很多，有名的一些職業分別是：劍宗、驅魔、刺客、審判、風法……

至於玩家期待的2020春節套屬性，就算體驗服這幾天更新了「魔界大戰」，也會拖到12月底甚至是1月初才會確定屬性的，但送95B普雷史詩升級券和大量黑瞳石卻成了玩家公認的獎勵。那麼小夥伴們，12套禮包的代幣券準備好了嗎？
