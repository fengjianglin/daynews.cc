2019年畫面最好看的10款遊戲 高“顏值”征服你了嗎？

2019-12-25 452條評論 2,129次閱讀 979人點贊

如今的3A作品大多在追求更高的畫質和更逼真的細節，許多玩家也越來越在意遊戲的畫質效果。那麼2019年的新遊戲中，畫面表現最好的遊戲都有哪些呢？外媒Gameranx就用一段視頻進行了總結，從材質、光照、特效等方面進行對比選出了10款畫質最佳的2019遊戲，一起來看看吧~

10，《往日不再》

簡評：環境效果、光照效果、落日、天氣效果等不錯，夜裡的雨景尤其好看；大量的變異者瘋狂奔跑的景象也很震撼。



9，《皇牌空戰7：未知空域》

簡評：天空的效果非常棒，地面看起來也很真實，天氣效果尤其出色，氣勢浩大的暴風雨令人嘆服；而飛機本身的細節也很到位。



8，《戰爭機器5》

簡評：將很多前作中的元素帶到了更加開放的地圖中，在日光下可以更清楚地看到它的畫質細節，沙漠、雪地、天氣、角色的護甲和面部表情等都很好，PC版尤其超級好看。



7，《生化危機2：重製版》

簡評：RE引擎很給力，角色面部十分細緻驚人，且遊戲很好地展現了陰暗、黏滑、噁心的室內細節，而在外面街上的效果也很好。



6，《鬼泣5》

簡評：角色的建模很好，面部細節極其逼真，堪稱今年最頂尖的水準。戰鬥中的各種特效也非常到位。



5，《塵埃拉力賽2.0》

簡評：今年最好看的競速遊戲（之一），將泥土、灰塵、路面凹陷、草地等各種效果都很好地展現了出來。



4，《地鐵離去》

簡評：開篇保留了經典地鐵味兒，擁有昏暗的走道和感染力豐富的氛圍及光照；之後來到戶外，將雪景、沙漠、沼澤等各種不同環境表現得淋漓盡致。



3，《使命召喚16：現代戰爭》

簡評：劇情模式非常好看，採用了攝影測繪技術來構建更加真實的環境；而子彈穿過乾燥牆面、震碎門面等戰鬥場面也很棒。



2，《死亡擱淺》

簡評：巨大的環境看起來都特別真實，各件貨物、裝備都有獨立建模，各種天氣效果也是細節爆表。



1，《控制》

簡評：細節豐富而逼真，地毯、桌椅、異世界怪物、各種破壞效果、各個角色的外貌都很好，如果開啟光追效果更加驚人。




除此之外，Gameranx也提到了一些其它畫面很不錯的2019遊戲，如《大鏢客2》PC版、《孤島驚魂：新曙光》、《全境封鎖2》、《聖歌》等。大家覺得今年哪些遊戲的畫面效果最好呢？
