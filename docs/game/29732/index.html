DNF：100級深淵畢業後，傷害提升5倍！旭旭寶寶將破1W億

2019-12-24 561條評論 1,880次閱讀 242人點贊

隨著100級版本的曝光，不少玩家開始放棄普雷了，主要還是95級結束得太快了，任何遊戲的100級都一個坎，DNF也不例外，所以地下城與勇士才同時上線三次覺醒、神話裝備、數百件新史詩等，據發布會介紹，DNF滿級就是100了，以後不會在進行擴張，其實這是因為屬性太過於膨脹了，短短几年，打樁從千萬到千億！

100級深淵畢業後，傷害提升5倍！旭旭寶寶將破1W億

而韓服測試服100級也開放接近一周了，由於PL基本無限，不少玩家已經刷齊了100級史詩，雖然不是完美畢業，但湊齊了12件，包括11件深淵史詩+1件神話裝備，而有一個玩家測試了100級與95級的傷害差距，直接提升了5倍！不少玩家或許不相信，一陣雨直播的時候也測試過，僅刷出2件了首飾+1件神話，傷害都接近翻倍，而2件首飾還沒刻印打造，屬性太強了。

其實玩家從95-100級，有一次職業加強，劍宗和劍豪們三次覺醒了，其它職業被動提升了大概20~30%的傷害，而100級史詩又從兩方面來提升，首先是面板直接加了1000多力量或智力（下一篇仔細介紹），其次就是白字爆傷等傷害屬性，就拿100級太刀「太極陰陽劍」來說，相比夜語黑瞳太刀，提升了接近50%的傷害，再加上冒險團等蚊子腿提升，難怪加起來提升了5倍。

難怪9個9的傷害上限才改了不到1年，100級直接改為了10個9，明顯是知道100級深淵史詩的巨大提升率，不然普通玩家小技能都是9個9，所以玩家提升5倍，傷害上限提升10倍，看來100級的希洛克Raid，還有很大空間嘛。

而作為國服傷害最高的兩位玩家，狂人和旭旭寶寶，95B畢業後，已經是2000E左右的水平，旭旭寶寶略低一點，但100級深淵畢業後，都將突破1W億的水平，對於普通玩家來說，那就是人均打樁1000E！在國服稱號和技能寶珠的加成上，國服隨便一個普雷C，傷害都不低於200E的。

可這帶來一個問題，玩家人均200E傷害的時候，普雷BOSS都已經1W多E血，那麼100級希洛克，起碼得5~10W億血量的水平，這時候也明白「紅玉的詛咒」副本，BOSS等級為什麼高達150~160了，不然搬到國服根本不夠打。

總之，DNF的屬性膨脹太快了，9個9的時代才開始，10個9的更新已到來！100級之後，玩家們可以平A秒殺安徒恩，跳X帶走光盧克。
