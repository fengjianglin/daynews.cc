陰陽師：體驗服平安夜活動獎勵提前看 有勾玉和藍票的快樂聖誕節

2019-12-23 1,170條評論 3,644次閱讀 1938人點贊

陰陽師正式服即將開啟的平安夜系列活動，估計不少玩家容易遺忘或者很好奇這次活動涉及到的資源和獎勵，這裡就帶來具體的情報以及提前透露一下，一起來看看吧。

先說一下活動的限制，15級以上玩家才能參與，這個本來沒必要啰嗦的，但是介於決戰京都活動中，15級以下的玩家並沒有看到平安京毀滅，可能也是為了防止太小的小號刷禮物，所以有這樣的限制、然後說說具體的時間，將於12月23日正式開啟，截止聖誕節第二天，26號就沒法給禮物了。

先給式神禮物，之後等一陣子會收到式神的禮物，不同的式神給的獎勵雖然不同，但實際上都是統一的，這裡包括有50勾玉+好感度金蛋+2W金幣，還有一個是藍票+金幣+經驗加成BUFF，同時還有一段涉及到的式神的信。

其中涉及到的給禮物的式神包括玉藻前、青行燈、白狼、雪童子、小鹿男，這些式神給的祝福和信件，祝福內容不同也非常有趣，其中玉藻前很有親爹的感覺，燈姐則是傲嬌，白狼更像戀人，雪童子是貼心小棉襖，小鹿男就是聖誕老人+馴鹿這種屬性，總而言之這活動是個有勾玉和藍票的快樂聖誕節，福利度還不錯。起碼給了不錯的雲外鏡出貨的機會。
