龍捲風對飛機來說有多危險？海盜遊戲《ATLAS》飛行器最怕龍捲風

2019-12-08 0條評論 19次閱讀 0人點贊

摘要

在看完電影《中國機長》之後，我們不得不感嘆國家對於飛機的飛行條件要求之嚴格，在沿途路上甚至連氣流、天氣等因素都算上了，那麼像龍捲風等極端自然氣候對於飛機來說影響究竟有多大呢？

在看完電影《中國機長》之後，我們不得不感嘆國家對於飛機的飛行條件要求之嚴格，在沿途路上甚至連氣流、天氣等因素都算上了，那麼像龍捲風等極端自然氣候對於飛機來說影響究竟有多大呢？這點我們可以從海盜冒險生存遊戲《ATLAS》的飛行器上面找到我們要的答案，在《ATLAS》中，飛行器只要一看見龍捲風就要飛速逃離，否則就會被捲入，落得了機毀人亡的下場。

和現實裡面的飛機不同，《ATLAS》裡面的飛行器主要是藉助氣流的力量，這點和現代的滑翔翼有點類似，只不過在《ATLAS》中只要玩家能夠掌握技巧，跨越島嶼級別的飛行都能夠實現，是一種相當強力的飛行道具。

但是這樣的飛行工具也會存在一個弊端，那就是很受氣流的限制，假如說天氣不好，玩家很容易在《ATLAS》中出現飛行事故。由於沒有大陸架阻擋，《ATLAS》裡面的氣候也往往是非常惡劣的，就比如說龍捲風，現實裡面的龍捲風都是成個出現且消失的很快，但在《ATLAS》中的龍捲風都是成群出現，且規模、殺傷力都遠大於現實中的龍捲風，很難想像在這種天氣下飛行會出現什麼樣的後果。

隨著科技的發展，人類開始對自然氣候等影響飛機飛行條件的因素進行分析，並且能夠提前對飛行過程中所發生的一切都進行預測，在天氣不好的情況下是不可能讓飛機起飛的，這種嚴謹的工作態度也讓飛機成為了現在這個世界上最為安全的交通工具之一。大家對此有什麼看法？要是你也想在《ATLAS》中御風而飛的話，可以在Steam上對這款遊戲進行下載體驗哦~
