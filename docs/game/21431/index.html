CFML丨季後賽即將開啟在常規賽榮獲第五的SV季後賽又會走到多遠？

2019-12-20 166條評論 3,761次閱讀 2071人點贊

CFML2019職業聯賽秋季賽季後賽將於12月21日正式開賽，屆時SV將與AG率先打響第一戰，SV在本賽季的常規賽拿到了積分榜第五的位置，那麼對於SV而言，當季後賽來臨後，他們又能走到多遠？

【細數常規賽表現 後期發力成關鍵】

在常規賽前三輪的比賽中，SV先後遭遇WE/BS以及R.LGD，在這三輪比賽中，SV並未迎來他們的首勝。其中在與WE以及BS的交手中，SV並非沒有一戰之力，無論是逼平WE還是與BS有來有回，這都讓我們看到這支隊伍的潛力。

當常規賽進入第四至第六輪的比賽時，雖然SV遭遇AG/SN以及EDGM等強隊，但是SV在與AG的交手中，隨著自身狀態的提升以及隊伍整體的戰術的變動，從而令他們迎來首勝。之後與SN以及EDGM交手的過程中，頻頻獲勝也不斷提升SV隊伍的士氣與對比賽節奏的把控。

常規賽最後三輪的對決中，SV迎來eStar/Q9以及KZ，最後三輪迎接的強隊對SV而言無疑時魔鬼賽程，但是第一場與eStar的對決中，SV卻憑藉節奏的變化以及青尤寡婦等人的發力爆冷獲勝，而在與Q9的交手中，雖然SV未能獲勝，但是頻頻發難Q9，讓Q9陷入被動，這更體現出SV後期的強勢。

【季後賽開賽 若戰勝AG則迎戰eStar】

正是常規賽的出色發力，令SV在常規賽積分榜中拿下第五名的排名，從而在季後賽中迎戰排名第八的AG。對於SV而言，在常規賽中SV便曾將AG擊敗，那麼再次迎戰AG，顯然SV做了充分的準備。

當然由於AG在CFM亞洲杯的出色表現也令SV更需要進行一定的調整，比賽中寡婦是否能夠繼續為隊伍提供穩定的火力輸出，狙擊手與突破手之間的配合能否到位，這都將成為左右季後賽SV是否勝利。在季後賽首輪比賽中，SV需要對AG的青菜嗜血加以針對，避免AG通過這兩名選手的發力掌握比賽節奏。

如若SV在季後賽首輪擊敗AG，那麼SV將在季後賽第二輪迎戰eStar。由於SV在常規賽中曾擊敗eStar，因此相信SV對eStar有更深的了解。由於eStar剛剛獲得CFM亞洲杯的冠軍榮譽，因此SV若進入季後賽第二輪之後欲擊敗eStar，那麼則需要針對eStar兩套不同的陣容進行準備。

即將到來的季後賽對於SV而言，若能戰勝AG晉級，那麼這將意味著SV將重返職業聯賽四強席位，相信在本屆季後賽中，SV將全力以赴彰顯屬於SV的風采。
