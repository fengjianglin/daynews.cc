這個遊戲好玩嗎：一探《刺客信條：大革命》背後的真實歷史背景

2019-12-12 459條評論 2,462次閱讀 269人點贊

歷史上的某一天，法國國王路易十六接到了一份圖紙，上面畫著的是一個奇怪的機械裝置，這是一個高度約為1米的方形木台，兩側立著高高的柱子，柱子之上則橫架著銳利的刀，路易十六研究了一下，並親自對設計圖進行修改，而後這款被後人稱作「斷頭台」的裝置正式被法國政府投入使用，讓人沒想到的是，在僅僅9個月後，法國大革命爆發，國王路易十六自己成為了這機器下的亡魂。

《刺客信條》系列遊戲一直以來都是育碧旗下的代表作品之一，這個系列遊戲最大的特點就是將真實的歷史融入到遊戲的世界當中，讓玩家所扮演的角色與眾多歷史人物進行互動，目睹歷史車輪滾滾向前，甚至作為其中不可或缺的一部分參與到歷史事件當中，同時歷史作為遊戲的背景也並非只是《刺客信條》當中一個無關痛癢的裝飾品，它也影響著玩家對於遊戲的思考，是遊戲中重要的組成部分，甚至於讓遊戲擁反哺現實的能力，為古建築的修復工作提供數據支持

《刺客信條：大革命》就是這樣的一款遊戲，作為一款以歷史上最為重大的事件之一的「法國大革命」作為故事背景的遊戲，育碧在4年多的開發時間中結合多位歷史學家的指導，將18世紀的法國巴黎在遊戲世界中進行還原，讓人們親身體現那段珍貴的歷史。

在這樣的努力下，遊戲已經不單只是遊戲，也成了人們用來學習歷史的一種方式，讓玩家可以更好地了解巴黎的過去，而這也正是我們今天所要談的內容。

有人曾將法國大革命比作是十八世紀的世界裡最為盛大的一場焰火，那麼今天，就讓我們走進火花之中，結合著遊戲來一探《刺客信條：大革命》背後所承載的真實歷史吧。

埋下火種，路易十四建立凡爾賽宮

羅馬不是一天建成的，同樣的法國秩序的崩潰也不是在一天中發生的。

法國大革命發生於1789至1799年，這個時間段恰好是英國作家查爾斯·狄更斯在《雙城記》的開頭中寫下的「最好的時代，最壞的時代」，這是一個矛盾交織的時代，泥沙俱下，新舊並存，法蘭西政治與社會都處於極大的變化與動蕩之中，最終導致了法國大革命的發生……

而假如說說到法國大革命，憂患的種子要追溯到路易十四國王當政時期，法國國王路易十四自號「太陽王」，在位時間長達72年，是歷史上在位時間最長的君主之一，同時也有人將其稱為是法國歷史上最偉大的君王，不過儘管如此，這位帝王卻和法國大革命的爆發有著非常密切的關係。

公元1643年，隨著第十三世路易的去世，當時年僅5歲的路易十四登上王位，而由於他此時年紀過小的原因，法國實際的權力一直都掌握在自己的生母以及紅衣主教馬薩林的手中，此時的路易十四則更像是一個「表面皇帝」，這種情況一直到1661年馬薩林去世才得以改變。

1661年，23歲的路易十四正式成為了法國的主人，在親政之後他做的第一件事就是開始了凡爾賽宮的擴建工作。

或許是由於童年時期雖作為君主卻並無實際權力的不愉快經歷，路易十四對於巴黎的皇宮以及盧浮宮、杜樂麗宮等宮殿都沒有留下什麼好印象，再加上為了能夠讓自己能夠真正地不再被他人所左右成為法國政治權利的中心，他逐漸地萌生出了將政治中心搬離巴黎的想法，並最終將凡爾賽宮作為理想地點。

《刺客信條：大革命》中曾經一語道破路易十四為何將凡爾賽宮作為法國新政治中心的原因。巴黎作為曾經的政治中心，在那裡貴族以及大臣都擁有著緊密的關係網和權力，這對於年輕的路易十四是非常不利的，而假如說通過建立新政治中心的方式將貴族和大臣們都帶來凡爾賽宮生活的話，那麼這些貴族及重臣將無法與原本的舊勢力保持聯繫，並且路易十四也能夠通過這樣的方式將掌握這些人的一舉一動，從而達到真正的權力集中。

遊戲中的凡爾賽宮：外部

其實原本凡爾賽宮在路易十三時期便已經存在了，但一直都只是作為「狩獵行宮」的存在，直到路易十四親政之後才通過一連串的修葺改造才使得凡爾賽宮逐漸成為了遊戲中所呈現出的樣子，而為了做到這些路易十四前後總共花費了將近十年的時間來對凡爾賽宮進行擴建，投入資本更是超過了一百萬磅。

宮殿完工後，路易十四成為了凡爾賽宮永遠的主人，儘管說後來的法國君主也有多位在此居住，但仍舊無法與之相提並論。

凡爾賽宮存在的意義並不僅僅是皇帝的一處行宮那麼簡單，作為路易十四集中政治權利的策略之一。通過搬離舊宮來到新宮，他將以往作為地方權力的化身的貴族們變成了宮廷內的成員，解除了他們的地方權力，同時這些舉家來到皇宮生活的貴族們為了應付從早到晚的宮廷聚會而不得不在製作宴會華服上付出巨款，且再也無暇管理地方問題，就這樣，路易十四建立了法國歷史上首個中央集權、專制的民族國家，這個政治體制直到法國大革命時期才被推翻，同時也是引起了法國大革命的政治誘因之一。

《刺客信條》中在凡爾賽宮舉辦的宴會

遊戲中的凡爾賽宮：內部

除了興建以凡爾賽宮為代表的宮殿外，在對外問題上路易十四也幹了一件大事，那就是積極地對外征戰。在路易十四親政的54年時間中，法國參與的大大小小的對外戰爭數不勝數，其中最重要的共有五次，分別是1667年至1668年的遺產戰爭，1672年至1678年的法荷戰爭，1683年至1684年的重盟戰爭，1688年至1697年的大同盟戰爭，以及1702年至1713年的西班牙王位繼承戰爭。雖然說在幾次大戰之後法國獲益不菲，但相比較於貴族階級的利益，戰爭為法國普通民眾帶來的則是更多的苦難。

隨後時間來到了1715年，路易十四駕崩，他的曾孫路易十五正式即位，而隨著路易十四的生命一起結束的還有法國最強盛時代的完結，當統治法國長達72年的路易十四逝世後，他所留下的是一個幾乎虧空殆盡的國庫，這個被伏爾泰稱為是「偉大的路易十四時代」的年代過後，法國社會急劇變化，最終導致了1789年的法國大革命。

一點題外話：凡爾賽宮在《刺客信條：大革命》中具有著非常重要的地位，這裡既是遊戲中1776年路易十六登基後接見刺客組織成員夏爾·多里安，以及聖殿騎士團成員暗殺夏爾之處，也是遊戲當中法國大革命的序幕解開之後，聖殿騎士與刺客組織之間鬥爭的中心點，另外遭到刺客組織流放的亞諾也是在這裡再次遇到了愛麗絲並在其勸說之下而重返巴黎。

路易十六展開三級議會，法國大革命爆發

在經歷了法國歷史上最偉大的王朝之後，隨著路易十六的繼位，在百年的時間中所積累的弊病開始逐漸的展露無遺，同時這位歷史上有名的「鎖匠國王」又並沒有像路易十四那樣的手段與策略，最終導致了波旁王朝無以阻擋的衰落。

路易十六繼任王位後先後幾次被捲入到了戰爭當中，而因為接連的戰亂，在經歷了路易十四與十五揮霍後的國庫再也無法支撐法國的經濟，最終導致了法國陷入了財政赤字的問題之中，並且即便說法國經濟已經到了如此苦難的地步，凡爾賽宮中卻仍舊是一派窮奢極糜的形象，相傳路易十六僅僅是為皇后購買一條項鏈就花費了高達600萬法郎，而在法國民眾已經無法正常生活的問題上，路易十六與他的皇后則更是講出了一句足夠媲美我國古代「何不食肉糜」的言論——「既然百姓吃不起麵包，那為什麼他們不吃蛋糕呢？」

此時法國貴族階級與普通民眾之間的矛盾已經達到了無法調節的地步，人們迫切尋求改變，星星之火即刻便可發展為燎原之勢。

《刺客信條》中所展示的貧民窟

雖然人民生活困苦，但遠遠生活在凡爾賽宮的貴族們顯然並沒有察覺到這一點，為了挽救法國經濟，路易十六破天荒地決定召開已經有近百年的時間都沒有召開過的三級議會，而讓他所沒有想到的是，他的這一舉動卻相當於間接地引燃了大革命的暗藏多年的火種。

《刺客信條：大革命》中的三級會議

1789年5月5日，法國國王路易十六為了尋求納稅制度改革而在凡爾賽宮召開三級議會，從此之後近兩個月的時間中數不清的議會被召開，但三個級別的代表們卻從來沒有就國家經濟問題達成一致過，多次協商無果後，甚至於第一二階級為了滿足自身利益而開始拒絕第三階級的代表進入議會會場，階級矛盾進一步加劇。最終在1789年的6月，作為三級議會中最龐大的一級，第三階級的民眾們發布《網球場宣言》並成立了具有革命性的國民議會，並於幾天之後的7月14日攻佔了當時法國關押政治犯的巴士底獄，法國大革命一觸即發。

歷史上的網球廳宣言，第三階級成立國民議會

《刺客信條：大革命》遊戲當中的「監禁」一章所刻畫的便是現實中法國民眾在7月14日攻佔巴士底獄的故事，在巴士底獄被攻佔之後的數日之內，法國國內政治進一步惡化，由法國民眾建立的議會與支持君主制的貴族們展開了曠日持久的鬥爭。

同年8月，為了安撫第三階級民眾的情緒，法國頒布了《人權宣言》，遺憾的是這種表面改革實際上只是哄騙人民的手段，《人權宣言》中並沒有照顧到沒有財產的第三階級民眾的權力，最終這也導致了10月的凡爾賽婦女運動（又稱十月事件）的爆發。

在《刺客信條：大革命》遊戲的合作人物「娘子軍」當中細緻的講述了有關這段故事的歷史。不重視法國普通民眾權力的貴族階級讓人們非常不滿，最終在10月5日這一天，多達700餘名女性在巴黎大堂集結並朝著凡爾賽宮進發，雖然說聖殿騎士團對於這一行動進行了干預，但是在刺客組織的幫助之下，民眾們最終將凡爾賽宮佔領並迫使路易十六回到了巴黎將其軟禁在了杜伊勒里宮中。

此時的法國已經走上了一條無法回頭的路。

遊戲中的凡爾賽婦女大遊行

隨著路易十六被軟禁在位於巴黎的杜伊勒里宮中，保皇與普通民眾之間進行了數年的鬥爭，期間雖然說皇室成員曾多次想要逃離或通過君主立憲制的方式來重新建立政權，遊戲當中的聖殿騎士更是從中數次對革命進行干預，但最終當時間來到1792年時，由最廣大的民眾所組成的國民議會最終通過投票的方式宣布決定廢除舊政體並建立法蘭西第一共和國，而就在法蘭西第一共和國成立後的第二年，國王路易十六被國民議會判決為死刑，路易十六也就這樣成為了法國歷史上唯一一位被判處死刑的皇帝，而諷刺的是在執行死刑時所使用的斷頭台還是他自己親自參與設計的。

法國大革命後路易十六被判處死刑

關於歷史中的法國大革命與《刺客信條：大革命》

法國大革命是歐洲歷史上最偉大也最激烈的一次社會革命，其意義在於將舊的政治與權力機構廢除而建立全新的制度，這一時期也被一些人稱之為「曙光之年」，而《刺客信條：大革命》遊戲的厲害之處則在於就和以往的《刺客信條》系列遊戲一樣，他完美地將這段歷史與聖殿騎士以及刺客組織之間的糾葛鬥爭聯繫到了一起，將現實中的鬥爭刻畫為是讓新生的中產階級代替舊貴族階級領導聖殿騎士團的一次革命。

或許說《刺客信條：大革命》並不算是《刺客信條》系列當中最出彩的一部作品，但在某種意義上，能夠將如此複雜的政治、歷史故事與遊戲相結合，讓真實的法國大革命在遊戲當中再度發生，同時也讓玩家作為親身經歷者成為了革命當中重要的一環，能夠如此細緻的將真實地歷史融入到遊戲的故事中實在是讓人不禁拍手稱奇，這款遊戲好玩嗎？我想答案已經不言自明了。
