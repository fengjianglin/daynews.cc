Meiko採訪首談廠長轉型教練，一語道出老將的重要性

2019-12-22 64條評論 1,090次閱讀 106人點贊

今年的英雄聯盟轉會期已經結束了，咱們LPL賽區的大部分隊伍也都公布了新賽季的陣容。由於明年的全球總決賽將會在中國舉辦，所以不少LPL隊伍也是紛紛摩拳擦掌，想要在明年取得好成績幫LPL在家門口拿下第三個全球總決賽冠軍。而在每年S賽之後的德瑪西亞杯比賽都是各大隊伍練兵磨合的好機會，所以德杯比賽一定會非常精彩的。

之前EDG就官宣了一個爆炸性的消息：「LPL001號選手斷開連接，Clearlove(廠長)宣布退役。這也是不少粉絲猝不及防，當然廠長也並沒有離開電競這個圈子，而是轉型成為了EDG的新教練。」今天也是廠長以教練的身份在賽場上的首秀，BP做的也不錯，最終EDG成功拿到了這次德瑪西亞杯比賽的首勝。而最終EDG的新人國產青訓上單小祥的船長發揮亮眼，最終拿下這局比賽的MVP。

而在賽後Meiko也接受了採訪，他也是首次談到了廠長轉型教練這個話題，Meiko直言：「就他今年轉的教練么，挺適合做教練的，大局觀方面也蠻好的，尊重他的意見，希望他在教練位置越來越好」

廠長作為咱們LPL001號選手，已經在職業賽場上奮戰了8年了，曾經也為LPL賽區建功立業拿下了無數的榮耀。之前PDD就在直播中表示：廠長是操作型的選手，他從來都是靠腦子玩遊戲的。在比賽語音中我們可以得知廠長經常能精確洞察到敵方打野的動向，進而做出正確的反應進行反蹲。而且廠長對於視野的掌控力是非常的強的，經常可以利用老將經驗繞過敵方視野打一個措手不及。

而在2016的LPL常規賽EDG對戰IG的一句比賽中，明明IG上單在草叢中放下了真眼，但是廠長就是利用一點視野漏洞讓對面沒用發現自己，最終成功Gank。其實廠長一直以來都是隊內的指揮，也正如Meiko在採訪中所說，他的大局觀一直都非常不錯，所以說廠長也是非常適合教練的

最後恭喜廠長自己的教練首秀成功，也希望廠長在以後以教練的新身份繼續為LPL發光發熱，EDG加油!
