火柴人遊戲的代表作之一，沒有等級與戰力，只有操作與樂趣

2019-12-12 568條評論 3,762次閱讀 653人點贊

hello！大家好，又到了一日一度的楊某講遊戲環節了，趕緊系好安全帶，準備上車吧！

今天要給大家介紹一款非常好玩的火柴人遊戲：《火柴人街頭打手》

其實還是火柴人打火柴人。

《火柴人街頭打手》是火柴人遊戲最具代表性的作品之一，遊戲以闖關為目的，通過各種操作擊敗所有敵人即闖關成功。這款遊戲值得一提的是它沒有各種各樣的屬性，沒有等級，沒有戰力，遊戲其實更偏向於操作方面，這與現在的很多冒險遊戲有很大區別。同時，在老牌火柴人遊戲的基礎之上，這款遊戲還對角色技能的操作性進行了改進，遊戲技能相對於老版本火柴人遊戲來說操作性要更強一點。

這個是火柴人的一技能。

一技能相對來說沒有什麼特別之處，主要就是上圖這個招式連打3次，不過在第三次的時候拳頭會發出火焰，傷害要比前兩次高得多。這個技能需要注意的地方就是在連招時使用效果很好，在前4關時連招到最後一招時釋放此技能，可以直接將小怪擊殺。

這個是火柴人的二技能。

二技能比起一技能來說就要有實用得多，特別是在撿東西的時候……

二技能其實就是類似於旋風腿之類的技能，只不過二技能在身體旋轉的時候角色還可以任意移動，持續時間大概3秒左右，敵人一碰到火柴人外圍的黃光就會被踢飛，配合上三技能可以發揮出更大的作用。

為什麼說二技能在需要撿東西的時候特別有用呢？原因很簡單，在二技能持續時間內直接往要撿的東西跑去，除非對方有槍，否則誰能攔得下你？並且二技能的使用會相對靈活一點，角色使用二技能時如果處於移動狀態，那麼可以輕易地將敵人踢開；而如果角色沒有移動的時候，那麼敵人就不會被你輕易踢開，這樣一來你就可以對敵人造成持續傷害。

這就是楊某認為最帥的三技能了。

三技能應該是楊某使用最頻繁的技能了，原型有點像龜波氣功，其實就是往前方推出一陣光波，威力很強，一次技能可以打掉敵人至少四分之三的血量。

然而，在前3關的時候，三技能幾乎是完全無敵的存在，完全無法破解，除非你離對手特別近。因為三技能需要大概1.5秒的蓄力時間，所以如果敵人離你很近的話，那麼就很有可能將你的技能破解掉。只不過這款遊戲一次性出現的小怪最多只有5個，所以只要你稍微會玩一點的話，前三關應該是不會有敵人可以靠近你的。

為什麼要強調是前三關呢？因為從第四關開始，敵人就開始拿槍的了……

有些地圖會有武器出現。

當然，沒有了等級和戰力，如果從頭到尾就只有一樣的操作的話，那麼遊戲未免顯得有些無趣。因此，這款遊戲還給了玩家們各種各樣的武器，在一些地圖中會出現，其實另一方面也是因為敵人會越來越強大，所以咱們的主角火柴人也得充分武裝一下是不是。

上圖的刀其實在第二關的時候就已經有出現了，這是最早的武器。這款遊戲有多種類型的刀，威力其實差不多，選擇什麼樣的刀主要是看玩家個人的操作習慣，因為有些刀是砍的，有些刀是揮的，還有些刀是像斧頭一樣劈的，甚至有把小刀是像劍一樣刺的，威力都一樣，看你喜歡哪種操作手感。當然，前提是你得撿得到……

不管在哪款遊戲，散彈槍的魅力總是無可替代的。

前面提到這款遊戲從第三關開始就會出現拿槍的敵人，因此為了不讓火柴人落伍，遊戲方毅然決然地把散彈槍給安排上了。作為這款遊戲最有趣的武器之一，散彈槍可以輕鬆一槍穿透前方的敵人，並且威力十分強大，一發子彈至少可以打掉對手二分之一的血量，相當於低配版的三技能了。當然，與很多遊戲一樣，散彈槍的威力主要還是得看散出去的子彈有多少打在敵人身上。

黃色的火柴人開始拿起火箭筒了……

從第6關開始，遊戲的難度基本就已經很大了，就比如上圖的黃色火柴人，手裡拿的是什麼東西？這不是炮筒嗎……好吧，不誇張的說，這個黃色火柴人一炮可以轟掉你五分之一的血量，甚至更多，因為傷害多少主要是取決於是炮彈直接轟到了你還是炮彈的餘波轟到了你。

那麼，從這裡開始，我們就得拿出真正的操作實力啦。首先，普通遠程攻擊肯定是很難打得過的了，雖然三技能很強，但是在蓄力的時候就已經被敵人的子彈給破掉了。因此，我們要利用前面的二技能與三技能的配合，首先用二技能將敵人擊倒在地，然後利用敵人的倒地時間進行三技能的蓄力，等到敵人站起來的時候，你的三技能也就剛好可以打到他們了，接下來就不停地使用三技能，只要把握好蓄力的時機，黃色火柴人的大炮應該是打不到你的。

這把武器貌似是救世主拿的武器，下面可以發射榴彈的。

當然，在這款遊戲的某些場景你還有機會撿到上圖這把武器，原型應該是穿越火線中救世主的武器，不過榴彈發射裝置只是擺設而已，其實用不了。不過，即使沒有榴彈發射裝置，這把槍還是很厲害的，可以當作自動步槍來使用，只要子彈能不停地射到黃色火柴人的身上，制止它的活動，那麼它的炮筒還是照樣轟不到你的。

這個技能是會消耗生命值的。

此外，這款遊戲中威力最強，最實力的技能可能就是上圖這個「L」技能了吧。「L」技能的傷害範圍很廣，遇到敵人拿著衝鋒槍不停掃射時，你就可以用這個技能將敵人擊倒，否則你很難找到機會靠近敵人。不過，除了這種情況，其它時候最好就不要用這個技能，因為這個技能需要消耗不少生命值！這也是這款遊戲中唯一一個需要消耗生命值的技能，所以最好謹慎使用。

以上內容就是楊某認為的這款遊戲比較突出的地方，在這裡強調一下，楊某並不是一個推銷遊戲的，不然也不可能經常介紹一下很冷門的遊戲。楊某隻是熱衷於玩各種各樣的遊戲，並把這些遊戲的樂趣分享給大家，僅此而已。

好了，今天楊某講遊戲就先講到這裡啦。
