DNF12.24元旦版本爆料：可交易天空套，spk特色隨機，坑慘玩家

天天要聞
2019-12-20

「地下城與勇士之小狐狸君愛談遊戲電競」第七百六十九期《DNF12.24元旦版本爆料：可交易天空套，spk特色隨機，坑慘玩家》，隨著2020年元旦的臨近，國服的元旦版本也是終於更新完畢，當然這一次的特色道具——瑞鼠迎新年，五福來獻禮，再次秀出新花樣。這一次特殊的盒子是可交易一次的，等於說是天空套變相可交易了。就像之前的絕版追憶國慶套、二覺裝扮盒子，都是時裝交易的先例！不過史派克的特色玩法——隨機獲取，就是一個坑！表面上幾率很高，其實能開到的玩家少之又少！老規矩搞快點，直接上本次元旦版本活動和獎勵的截圖展示：

DNF12.24元旦版本爆料：

當然此次是國服的元旦+春節預熱版本，活動的主題也是非常的明確：鼠年小道具！所以活動道具僅在1.01~1.02這兩天才能獲取！和之前的情人節、植樹節……等小道具盒子一樣，獲取的道具看似不錯，但是是隨機的，最好的天空套、+12強化、+11增幅券，低級的道具史詩之路的門票。所以，還是老樣子90%的玩家是保底道具。想要天空套的玩家，也是可以試一試的，因為這一次的道具盒子是能上架遊戲拍賣行的。

可交易天空套，

其實這個天空套此次還算比較仁慈的。畢竟這一次不是一鎚子的事情。玩家可以多次開啟，按照玩家的強化墊子理論，連著不出天空的幾率也是很低的。所以和之前只能開一個盒子相比，本次活動還算不錯吧。當然，由於史詩和天空套都是目前無法交易的道具，但是韓服史詩和時裝都開放了，不知道國服史派克還要撐得什麼時候。

spk特色隨機，坑慘玩家

不過最終的結局，小狐狸君現在都預料到了。不相信的玩家，可以等到2020年1.01開服看公屏的數據，能出天空套的絕對是屈指可數的幾個人。目前官方的幾率也是沒有放出來。估計公告上0.1%的幾率就不錯了。
