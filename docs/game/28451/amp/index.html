DNF：韓服100級版本可能存在的誤區，神話屬性是固定的？

天天要聞
2019-12-23

Hello，大家好，我是月月。最近因為韓服DNF發布會公布100級版本即將到來的消息，勇士們都非常關注100級版本究竟是怎麼樣一個副本。而這個時候，我們也從不同渠道陸陸續續了解到了100級版本的內容。不過，消息多了，難免會有一些消息魚目混珠、良莠不齊，有些消息也不一定是真的。所以，我們今天來談談100級版本的哪些消息，可能存在誤會呢？

1.神話裝備屬性

神話裝備，是100級版本新出的裝備品級。新的裝備品級，代表著更強的裝備屬性，這一點毋庸置疑。通過韓服爆率得的裝備屬性內容，也證實了這一點。不過，神話屬性有一個內容卻是發生了爭議。這個內容，就是神話專屬屬性的內容。比如神話手鐲的專屬屬性2是技能攻擊力增加1%~6%，一開始我們都以為是這個屬性增加是隨機的。不過最近卻傳來一個消息，這個加成是根據裝備的品級來區分的。比如我們的神話手鐲品級是最上級，那麼它的加成也是最高——有6%的技能攻擊力加成。之前我們之所以認為屬性加成是隨機的，那是因為陷入了一個固定思維——裝備辭典的不應該都是最上級、最好的嗎？

2.深淵模式

目前，韓服100級的深淵模式不同以往，刪除了蹼泳深淵，只剩下特殊深淵。至於新深淵的門票，則是18個”引導石”。是不是沒聽過這個材料的名字？沒聽過就正常了，因為這個100級版本的新材料。不過就算這樣我們也不用太擔心，我們可以用史詩靈魂、邀請函、星空靈石根據不同比例兌換這個材料。而100及版本，也有不少副本可以獲得這個材料。值得一提的是，100級深淵模式的難度非常大，如果沒有一身較好的裝備可能打不動怪，這一點沒想到吧？另外在刷深淵的過程中如果遇到金色的深淵柱子，就將必會掉落SS裝備；如果是粉絲的深淵柱子，則會有較小几率掉落SS裝備。這一點，跟現在的國服的史詩之路有點相似。

3.裝備保值問題

這一點，估計是大部分玩家最關心的一件事。不過正如DNF策劃們之前承諾的一樣，100級版本的裝備價值是可以保值的。我們可以通過收集材料，對現有的裝備再次升級。不過，升級的裝備卻是固定的，不能有我們自己選擇，超界防具統一升級為板甲3：大自然之息，普雷首飾統一升級為首飾4：精靈使的首飾套裝，普雷特殊裝備統一升級為特殊4：標準化同步器套裝。當然，100級的裝備可以相互刻印、繼承，如果我們想要換甲，也可以通過後續收集裝備完成。不過，目前100級的裝備好像沒有”轉甲”功能，或許以後會加上去吧?

總而言之，現在100級版本還是混亂的。即使是在韓服，100級版本也只是存在於測試服，正式服並沒有更新出現。所以，很多消息都有不確定性。即使我現在說的內容，可能也會馬上被打臉（應該不會吧？）所以，我們在傳播消息的時候要謹慎，避免被居心不良的人利用。比如關於裝備繼承問題，無論是韓服策劃，還是國服策劃，都保證過可以保值的，我就想不明白為什麼這麼多玩家還會說不能繼續升級。最後，歡迎勇士們分享自己所知的100級內容，我對新版本還是非常期待的！
