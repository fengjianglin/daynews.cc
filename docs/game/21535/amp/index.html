把勞拉小姐姐打包帶走，《古墓麗影》全系Steam低價促銷

天天要聞
2019-12-19

在眾多的知名遊戲之中，誕生過很多經典角色，讓玩家們印象深刻。而在探險遊戲之中，勞拉小姐姐的名字很多玩家都耳熟能詳了，她和她的故事甚至被翻拍成了經典的影視作品，《古墓麗影》系列也是一個老牌的經典IP遊戲系列，有著漫長的歷史和許多讓人記憶深刻的故事。

從1996年開始，勞拉.克勞馥這個虛構的遊戲角色以及其靚麗身影一直陪伴著玩家們開啟一次次精彩冒險，23年過去了，小姐姐的陪伴一直如初，《古墓麗影》也成為了冒險題材遊戲中的口碑經典。

追溯勞拉小姐姐的故事，《古墓麗影》系列遊戲基本可以分為兩個時代，一個是前《古墓麗影》時代，一個是後《古墓麗影》時代。而這個系列分界的作品就是《古墓麗影9》，可以說是系列的重啟之作，無論是遊戲的畫面、玩法還是勞拉的故事都煥新重生，之後的《古墓麗影：崛起》、《古墓麗影：暗影》都是《古墓麗影9》的故事延續，被合稱為《古墓麗影》重啟作品三步曲。

而《古墓麗影9》之前的作品，雖然也有著極高的人氣，但無論是遊戲畫面，還是遊戲內容玩法以及北京故事都大相徑庭。不過勞拉小姐姐的冒險故事，無論是哪一部作品都充滿的驚奇讓人熱血沸騰，如果你還沒有玩過整個系列的遊戲，那麼機會來了。

9月20日-27日，Steam平台開啟了《古墓麗影》系列遊戲巨幅促銷，遊戲最低只要3元即可到手。

《古墓麗影：暗影》原價379元，現在打折促銷僅售125元。

《古墓麗影：崛起》原價160元，現在打折僅需24元即可入手。

2013年的首部重啟之作《古墓麗影9》原價68元，現在僅需10元馬上入庫。

而重啟之前的《古墓麗影》諸多遊戲，最低的僅需3元，最高不超過10元，如果你想打包帶走，那麼《古墓麗影》合集是個不錯的選擇，原價為1195元！現在只要158元，不管是收藏還是遊戲，都非常划算，該死的G胖，錢包拿走！

當然，想要與勞拉小姐姐展開精彩的冒險之旅，探索神秘的古墓，領略大自然的風光，那麼流暢和清晰的遊戲畫面是少不了的，所以一片性能出色的顯卡當然也是少不了的？

所幸《古墓麗影》系列遊戲不是什麼顯卡殺手遊戲，因此不需要頂繼續顯卡即可暢玩，對於配置需求最高的《古墓麗影：暗影》支持最新的「實時光線追蹤」技術，是系列遊戲中最吃顯卡性能的遊戲，滿足《古墓麗影：暗影》暢玩需求的顯卡，即可暢玩整一個系列遊戲。

暢玩光追的話，還是推薦使用一片RTX 2070 super顯卡，性能摸到RTX 2080的屁股，價格便宜很多，算是RTX 20系列中性價比最高的高端遊戲顯卡了。
