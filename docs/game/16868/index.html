雲頂之弈：新羈絆"月光"，1費3費各一張，刺客陣容沒法玩了

2019-12-16 1,789條評論 3,553次閱讀 2188人點贊

在半年前的雲頂之弈模式第一賽季中，設計師當時增加了一些新英雄，而新英雄基本上都是強行加強一個套路，比如海克斯英雄一下子塞了四個。但在第二賽季中，設計師決定調整新英雄的發布方式，他們不再單純地增加新英雄，而是添加新職業讓這些英雄的存在感變得更高一些。

本周國服更新的9.24版本中，我們迎來了全新職業”系魂”，系魂是專屬於盧錫安和賽娜的職業，只有這兩個英雄才有。系魂英雄讓盧錫安變成戰場上不死的carry角色，賽娜則提供強化輸出能力。而在明年一月的10.1版本中，又有了全新的職業羈絆效果，那就是”月光”職業。月光職業和系魂職業一樣，只有兩個英雄，分別是蕾歐娜和卡爾瑪。

她們同時在場時會獲得效果，每隔7秒會提高隊伍的20%暴擊流、暴擊傷害和法術強度，最多疊加4次。設計師希望通過這個新職業，來挽救目前的坦克陣容，因為最近兩個版本都是刺客橫行，而坦克英雄又很難有輸出空間。蕾歐娜是屬於守護神英雄，守護神英雄會提高自身的護甲，而蕾歐娜的技能不再是第一賽季的群體控制，而是自身獲得超高的傷害減免。

而卡爾瑪則是秘術英雄，秘術英雄會提高全隊的魔抗，也是一個防禦類的英雄，她的技能是為最近的隊友提供護盾和攻速加成，保護隊友進行輸出。我們可以看出，蕾歐娜和卡爾瑪都不是輸出型的英雄，而是保護型的英雄，他們適合搭配守護神秘術的坦克陣容。而坦克陣容最大的問題就是缺乏輸出能力，而月光羈絆可以提高全隊的暴擊率和暴擊傷害，還有他們的技能效果。

設計師也希望通過月光英雄，去克制目前流行的刺客陣容。畢竟守護神英雄可以剋制戰士刺客，秘術英雄可以剋制召喚使，新版本月光英雄很容易湊到2個，蕾歐娜是1費，卡爾瑪是3費。玩家不用擔心自己中期就被刺客或者召喚使陣容搶死了，而到了遊戲後期的話，月光羈絆為隊伍帶來的收益太高，只要己方英雄站得越久，他們獲得的傷害能力就越高。
