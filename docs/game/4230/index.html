雲頂之弈出現一個“四金王”，玩家：拳頭要我吃雞，我沒得選

2019-12-09 0條評論 15次閱讀 0人點贊

摘要

出現一個“四金王”，玩家：拳頭要我吃雞，我沒得選！在雲頂之弈中，打野怪會爆裝備、英雄或者金幣，如果是白色寶箱一般是英雄或者金幣，藍色寶箱比較常見，一般是裝備和金幣，金色寶箱就有點稀少了，有時候一局遊戲一個金色寶箱都沒有，運氣好也就一兩個。

出現一個「四金王」，玩家：拳頭要我吃雞，我沒得選！

在雲頂之弈中，打野怪會爆裝備、英雄或者金幣，如果是白色寶箱一般是英雄或者金幣，藍色寶箱比較常見，一般是裝備和金幣，金色寶箱就有點稀少了，有時候一局遊戲一個金色寶箱都沒有，運氣好也就一兩個。

但是最近雲頂之弈中出現一個「四金王」，何為「四金王」？這位玩家在打野怪時一下爆出四個金色寶箱，這爆率也太離譜了吧，難道是系統出了故障？正常來講，一下子爆出四個金色寶箱是不可能的，但是這位玩家偏偏遇到了，真是「四金王」啊，而且這把陣容質量也很高，再加上四個金色寶箱，玩家直呼：拳頭要我吃雞，我沒得選！

9.24版本的肉裝阿木木火了，比3星豬妹還厲害！

在雲頂之弈第一期中，豬妹算得上是第一團控英雄了，她的大招範圍大，控制時間長，在有回藍裝和肉裝的情況下，豬妹可以頂在前排一直放大，對面後排主C全程被動，一下輸出打不出來，可以說相當鬱悶。那麼新版本有什麼厲害的團控英雄了？額，到目前為止，除了娜美好像還沒有，石頭人雖然也是團控，但是這傢伙容易假賽，範圍也不夠大，而娜美是團控，但是很脆，容易被秒。

不過好消息就是在9.24版本，新加入了阿木木這個英雄，阿木木作為法坦，控制還是挺足的，從上圖可以看出，二星阿木木的控制範圍幾乎佔據了全部棋盤，其控制範圍比娜美還要大，這控場效果非常厲害，而且9.24版本的肉裝阿木木真的很火，因為有地獄火和守護神羈絆，阿木木在搭配陣容時也很容易，唯一缺點就是這張卡是五費卡，太貴，想要湊二星還是蠻難的，估計到時候會引發搶阿木木大戰，但是肉裝阿木木真的比3星豬妹還厲害，大家試試看就知道了。

森林拉克絲沒用？99%玩家不知道的森林羈絆可以這樣玩

在雲頂之弈中，拉克絲有十種元素，在遊戲後期，這張卡牌成為了湊羈絆的關鍵卡牌，不如海洋拉克絲對於法海陣容來說就非常重要，有了海洋拉克絲，玩家可以湊齊六海洋六法師的傳奇陣容，使陣容強度提升一大截；在比如鋼之拉克絲，有了這樣卡牌，可以湊四鋼鐵，對於以夢魘為主C的陣容來說，有了四鋼鐵，夢魘的生存能力又大大提高，所以說，拉克絲在後期還是非常有用的。

不過有一張拉克絲一直不受待見，這張就是森林拉克絲，因為森林羈絆很容易湊，而且在後期也不是很強，所以森林拉克絲作用不大。但是就是這張被大家忽視的拉克絲其實有一個妙用99%玩家都沒有發現，那就是湊有拉克絲的三森林羈絆。三森林羈絆的複製英雄功能是一把雙刃劍，複製的好會有妙用，複製的差則有點雞肋，所以如何保證森林羈絆的複製人選是關鍵。但是有了森林拉克絲則沒有了這種煩惱，後期只需要上拉克絲和另一個主C的森林英雄，則可以保證複製的都是厲害的英雄這樣一來，戰力就可以穩定提升。

好了，本期內容結束了，大家怎麼看遊戲中出現一個「四金王」呢？歡迎分享自己的看法，我是盟盟，喜歡的請點個關注，咱們下期再見啦！
