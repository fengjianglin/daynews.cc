沒錯，主c就是上面的三位了。

陣容介紹:希維爾、魔騰、劍聖、雷克賽、亞索、卡茲克、迦娜、琪亞娜。

陣容羈絆

3雲霄（琪亞娜看元素格）4劍士2沙漠 2鋼鐵3刺客 2秘術

陣容前期玩法

前期可以掠食者開局，有陣容裡面的棋子就先拿著，然後可以轉雙劍雙鋼鐵

慢慢的在前打完狼之後開始D牌，這時候就5人口了，看D牌的品質上卡，琪亞娜，卡茲克都可以上，然後在攢錢，有牌就拿牌，

中期陣容

鱷魚是很好的打工仔，我們後期要雙沙漠就可以，因為現在的4沙漠不是很厲害，沒有必要追求4沙漠。

之後就可以賣鱷魚上卡茲克，八人口上劍聖。

最終陣容

這套陣容的主c為帶劍士轉職書的夢魘，用輪子媽來打控制，因為陣容相對其他陣容非常脆弱，可以依靠鋼鐵的特性將挖掘機放在第一排- 一個人抗傷害，將鋼鐵的作用最大化，相當於有一個復活甲，所以鋼鐵在這個陣容中很重要必須要，裝備了繳械跟沉默的希維爾放在中間打控制。

這套陣容成型之後基本上無敵，主C就是夢魘，裝備也靈活多變。但是最重要的就是拿鏟子。沒有鏟子就成型不了。

由於劍聖和風女8人口後才好拿到，大家也轉換陣容也就是上4沙漠，大家可以借鑒一下，基本上玩法一樣，畢竟這是一個看臉的遊戲。

我在這裡只提供共簡單的思路，大家還是去要自己親身實踐。

沒有無敵的陣容，只有走永遠的歐皇。祝大家上分愉快。
