王者模擬戰總是走為上計？學會三十六計中的厚黑計策，歡樂慶余年

2019-12-22 184條評論 4,019次閱讀 1389人點贊

導語

王者模擬戰是一個回合策略競技性遊戲，8人展開沙盤推演，決戰至最後一人為勝出。沙盤推演源於實戰，屬於模擬實戰的一種，是戰鬥就離不開兵法。

華夏上下五千年的歷史，也可以說是一部戰爭史。這些豐富的鬥爭經驗也被總結為諸多兵法，我們熟悉的「兵法三十六計」就是其中之一。

模擬戰對局中除了吃雞，我們總是被迫使用走為上計，那還有哪些計策可以主動使用呢？

一、瞞天過海

計策解析：用偽裝的手段迷惑、欺騙對方，使對方放鬆戒備，然後突然行動，從而達到取勝的目的。

具體運用：對局中，我們可以示敵以弱，將關鍵英雄拿下放到待戰區。一般情況下，敵方都是按住我方頭像觀察陣容，以為我們場上英雄就那麼幾個，放鬆警惕。在開戰前3秒左右，我們快速將英雄放到戰場上，一舉拿下戰鬥。

例如：我方陣容是蜀刺，戰前可以將關鍵英雄玄策和李白放到待戰區，用以迷惑敵方。或者敵方在猶豫是否升級人口前，一般都會看我方陣容進行判斷，如果此時示敵以弱，敵方就不會升級人口，我方勝利的機會就大了一些。

二、聲東擊西

計策解析如上圖所示。軍事上使敵人產生錯覺的一種戰術。

具體運用：對局中突然改變陣容或者站位，讓敵方不能按照設計好的劇本去走。常用的是控制位或者主C位置突然更換，手速快的也可以快速更換場上英雄，都可以達到聲東擊西的目的。

舉例：如圖，我方3刺客本來計劃切敵方後排元芳，敵方採用聲東擊西之策，將程咬金的位置突然調整，由前排調整到了右后角，結果我方刺客就去打程咬金了，導致對局失敗。

三、擒賊擒王

計策解析：對戰中要先擒拿主要敵手。

具體運用：對局中，集中我方優勢兵力，以優先擊殺敵方主C為第一方案。敵方主C如果被擊殺，就少了最主要的輸出，剩下的就在我們掌控之中，慢慢的一個一個擊殺敵方其他英雄，勝利必然屬於我們。

舉例：如圖所示，我方蜀刺陣容，玄策、李白為主C；敵方七蜀陣容，黃忠是主C。通過觀察敵方站位，利用刺客對角跳的機制，集中我方5名刺客（呂布有刺客令旗），全力進攻黃忠。黃忠一去，敵方再也無力回天。

四、關門捉賊

計策解析：預先設置陷阱，讓敵方跳進來，關門，放狗。

具體運用：對局中，預先判斷敵方刺客的戰鬥位置，設置好我方的控制位置，讓敵方刺客進入到我方預設的地點，就可以限制刺客的輸出，贏得戰鬥勝利。

舉例:如圖所示，我方控制位是弈星和廉頗，總共設置了5個「狗位」，敵方5個刺客必然會跳向我方後排，跳進「狗位」以後，基本按照我們的劇本進行。我方輕鬆取勝！

總結

凡戰者,以正合,以奇勝。兵法運用可以理解為「奇」，陣容和英雄質量等為「正」。雙方「正」相差不大的情況下，運用「奇」會有奇效。如果有一方直接人口、星級碾壓，即使採取「奇」，獲勝幾率也不會太大。

實際上，對兵法的運用已經深入到我們的骨髓里，很多時候都是不自覺的就運用了，無意識的運用只是偶爾為之，不能形成經驗。這裡所列的幾個可以主動運用的計策，就是讓我們變「無意識運用」為「有意識運用」，在對局中主動去運用，大大增加己方對戰的獲勝幾率。再就是，很多計策並不是孤立的，不是說就用這一個，而不能用另外一個，我們也可以聯合運用。

我是名字帶彩蛋的林子大之，歡迎小夥伴們一起留言討論。

快樂遊戲，大吉大利！
