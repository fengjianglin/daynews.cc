《獵人》手游：秘境副本的重要性

在往期的攻略教學中，我們一起認識了秘境副本！那麼對於探索獵人世界的獵人來說，秘境副本究竟有多重要！接下來，就讓我們全方位的分析一下，秘境副本的重要性！

要點一：主線任務

秘境關卡會隨著主線任務與人物的的等級解鎖，隨著等級的提高，會逐步開啟更高難度的副本，最高可挑戰王者級別。因此秘境副本是隨著主線任務一起的。玩家跟隨主線任務的同時，通過秘境副本可以獲得大量的經驗、道具獎勵。

而且後期主線任務完成後，秘境副本也同樣可以帶來大量的經驗獎勵哦！

要點二：過渡裝備

對於秘境副本來說，難度越高產出的獎勵也越高，王者級地圖最高可產出粉色神器裝備！但是，相對於挑戰副本來說，秘境副本相對還是更弱一些。而且部分秘境副本可以產出大量的稀有裝備、神器裝備，可以支撐玩家輕鬆的度過過渡期。另外高等級地圖還會產出虛空地圖與遺迹地圖的門票，滿足玩家後期挑戰副本的門票要求。

要點三：裝備兌換

秘境關卡中產出裝備是隨機的，但是秘境副本還會產出晶石。單人通關秘境關卡時會隨機掉落對應等級的晶石，比如綠冰晶石、黃冰晶石等等。這些晶石可以在晶石商店兌換各種裝備，其中包括本職業紫色稀有武器以及首飾。

更重要的是，晶石商店有20、30、40、50四個等級級別，兌換的裝備比級別商店高5級。比如30級裝備，兌換出的是35級的紫色裝備。因此對於玩家來說，秘境副本帶來的中期裝備是玩家不可或缺的。

要點四：TP、SP點

技能等級的差距是一些裝備無法彌補的，而TP、SP技能點除了可以升級獲得。其他便是來自於副本，當玩家在特定關卡獲得SSS級評分後，可在章節評價中領取獎勵。其中就有大量的SP、TP技能點，可以提升技能等級。

所以，大家一定要重視秘境副本，中期的成長中要將秘境、挑戰副本相結合，才能更快的提升戰力。而且秘境關卡的高難度副本是很有挑戰性的，因此想要SSS級別通關需要對副本非常熟練，當然也需要一些裝備基礎。後期我們會詳細介紹一些超難關卡的詳細打法攻略，幫助玩家們早日解鎖評價獎勵。
