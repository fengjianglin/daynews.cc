SMG目前所面臨的指揮困境，可以學一學4AM當初的處理方式

2019-12-23 984條評論 2,373次閱讀 650人點贊

絕地求生SMG戰隊自從春季賽結束後，隊員表現就特別低迷，成績也一落千丈，從三大杯賽到夏季賽到大師賽以及年底的杯賽，SMG都沒能打出春季賽以及之前的成績，其實他們所遇到的問題也很簡單：指揮問題不明確。

眾所周知，在更名為SMG戰隊前，17戰隊的指揮一直是大京，可自從跟SMG戰隊合併後，大京的上場次數變得非常之少，一個戰隊的指揮就是一個戰隊的核心，核心變動帶來的是全方位的改動，戰術轉移、近戰接團站位等等，指揮又好像粘合劑一樣，可以把戰隊四個人粘成一整塊，而沒有粘合劑的戰隊就是一盤散沙！

期間大京也有上過場，但效果不好，實際原因觀眾們也知道：大京的指揮多次受到隊內核心選手17shou質疑，此前在訓練賽中，還發生過爭吵，17shou或許並不是要質疑大京的指揮，但遊戲中有自己的想法讓大京的戰術並不能完全按照想法執行。

其實這樣的問題之前4AM也遇到過，那是王總剛來到4AM時，想著王總個人意識到位，獨狼經驗豐富，在大局上應該也有很高的造詣，於是在王總剛來到4AM戰隊的那段時間，4AM是準備讓王總指揮的。

那段時間也確實是王總指揮，但總是打不好，原因是韋神老有自己的想法，打亂了王總的戰術布置，戰隊就此出現了分歧。而王總又是一個心思幾位細膩的選手，當初被懟了之後還不咋愛說話，因為剛來4AM戰隊，人生地不熟的。

劉大大記得，當初還有不少4AM粉絲跟王哥說：韋神沒有什麼壞心眼，就是單純地說遊戲裡面的戰術，不要往心裡去；又有粉絲甚至跟王哥說到：不要給韋神面子，就學當初的拳王阿魯卡：噶得偉，你不要和我犟，沒有用！當初阿魯卡估計也是4AM裡面唯一一個敢當面罵韋神的選手了。

後來，4AM覺得這樣下去也不是一回事兒，於是還是做出了一個決定：韋神收回指揮權，王總專門輸出！

後來的效果大家也看到了，王總的個人戰績多次在大賽中獲得第二名，比如FGS和PCL夏季賽，PGC全球總決賽的總擊殺數也是第二，所以人稱”王老二”，不過在PCM大師賽中，王總在隊伍狀態不是很好的情況下，拿到了殺人王一榮譽，在傷害輸出得分方面，王總的作用已經發揮得淋漓盡致。

而且現在4AM常常啟用22分戰戰術，王總和永遠一個小隊，也算是一個小隊指揮，而韋神負責和小鹿在一塊兒時的指揮以及最終雙方信息綜合時，由韋神來做出最終的決定，這樣的人員分布就是雙核體系。

所以SMG戰隊其實完全可以參考這樣的設定：每個選手在比賽的時候確實都有自己的想法，但是一個職業戰隊只能有一個最終發號施令的人，想指揮也不是沒有辦法，可以像4AM這樣22分站，每一個小隊各確立一個小指揮，再在小指揮中確立一個最後的總指揮，這樣局部打得更加靈活，整體也更加具有邏輯和目標性。

有一說一，從黃金大獎賽的小組賽來看，大京對於SMG的重要性不言而喻，他兩次埋點重創進攻的4AM和UC，這也讓SMG的正面接團勝率提高不少，但是作為SMG的核心，17shou也不可能一直不上場，如何處理好”獸寵若京”這個組合，發揮其最大的作用，是SMG戰隊當下要考慮的最大問題，4AM真的是一個不錯的榜樣。

以上只是劉大大的個人看法，覺得有道理的小夥伴可以點贊收藏，有其他想法的小夥伴也可以在評論區留言討論哦！
