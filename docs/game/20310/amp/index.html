CFer競技賽丨專訪神勇鑫：賽場上的失利可以教會我更快成長

天天要聞
2019-12-17

在CFHD-CFer競技賽的首個比賽日的爭奪者，作為八強賽的種子隊伍的AG.Red戰隊在成功從小組賽突圍之後，本以為會順利晉級到CFHD-CFer競技賽的八強賽中，但是卻被名不見經的KYEG逆轉淘汰，那麼這支由TGA金槍王神勇鑫領銜的戰隊究竟在比賽中發生了什麼致使輸掉了比賽，我們在賽後邀請賽到了神勇鑫進行採訪，一起來看看吧。

問：這次CFHD賽事是抱著什麼目標來參加比賽的？

答：無論是什麼比賽肯定要往更高的目標去爭取的，我們當時的目標肯定是進入總決賽，最起碼也是進入最終的四強，沒想到淘汰賽第一輪就被淘汰了。

問：這次賽事沒能完成賽前目標，你覺得最大的原因在哪些方面？

答：首先我們幾個是剛組的隊，其次在配合上可能有時候會出現一些失誤，然後就是對這個遊戲的理解可能還不到位，在細節上理解的不太透徹。

問：小組賽輪空沒有練槍，然後淘汰賽直接開戰，這有沒有影響到你們的發揮呢？

答：其實小組賽輪空影響不是太大吧，主要還是臨場的問題，還有就是自身的問題，比賽輸了就要去總結經驗，下次再戰！

問：當時第三圖比分是5-1，比分比較大，當時是心態出現問題了嗎？

答：當時第一圖我們贏了，第二圖是4-3，最後兩分讓人拿了，是因為我們當時道具投擲和戰術出現了失誤，心態上出現了點問題，最後導致了比賽被翻盤的情況。

問：本次賽事失利後，接下來你們有什麼規劃嗎？

答：秋去春來，賽事明年還會有的，我們接下來好好去訓練，來年再戰，年輕不服輸！

問：你對於今天自己的表現有一個什麼評價呢？

答：自己發揮一般吧，中規中矩，失誤的地方也有，沒把握好機會，狀態也不會很好吧，比賽輸了也沒有辦法，接下來還是要去調整一下自己。

感謝神勇鑫接受我們的採訪，作為一支種子戰隊，在淘汰賽首輪就被淘汰，的確令人可惜，而TGA金槍王神勇鑫作為這支戰隊的中流砥柱，相信隊伍輸掉了比賽他心裡會更難過。當然，失敗是成功之母，比賽失利可以教會我們更快成長，加油神勇鑫！我們相信你們會再次打回來的！
