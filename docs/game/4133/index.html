夢幻模擬戰：超時空試煉歐米伽再立功！一人殺穿4S，白毛全程遛狗

2019-12-09 0條評論 15次閱讀 0人點贊

摘要

本周夢幻模擬戰超時空4S5的難度有些高，敵方角色不但擁有超高的屬性，還有一個可以開覺醒技能的露娜。因為露娜的關係，敵人可以非常快速的貼臉跟我們戰鬥。這直接讓通關難度大幅度增加。傳說君這邊並沒有什麼好的解決方案，所以還是用歐米伽降低了通關壓力。

本周夢幻模擬戰超時空4S5的難度有些高，敵方角色不但擁有超高的屬性，還有一個可以開覺醒技能的露娜。因為露娜的關係，敵人可以非常快速的貼臉跟我們戰鬥。這直接讓通關難度大幅度增加。傳說君這邊並沒有什麼好的解決方案，所以還是用歐米伽降低了通關壓力。

隊伍配置比較簡單，就是全員保歐米伽，並未他提供高額的增傷確保能夠一擊必殺。配置如圖，白毛、愛麗絲、奶騎、利亞納。

第一回合需要給歐米伽強化到極致，我給了進擊、愛麗絲大爆發和白毛超絕。即使這樣歐米伽在不開冷血的情況下也並不能直接擊殺狗子。會差幾百血，所以無奈的我只能開了冷血後進行攻擊。之後的站位如圖，莉亞娜最好召喚一個天使擋槍，否則可能直接被殺2人GG。

二回合開始歐米伽直接點殺二哈然後向左邊移動，奶騎跟愛麗絲跟好歐米伽，附近剛好有3格安全位置。白毛向右下方移動，勾引皮耶魯。這裡白毛有捉迷藏在身，並不擔心減速效果，如果是其他角色就會比較難辦。

最終站位如圖，這樣露娜會向右邊移動，而皮耶魯則會用鉤子鉤白毛，如果白毛往上跑皮耶魯則會去鉤愛麗絲。所以白毛的位置只能在下方。上邊莉法妮和炸彈也會想白毛方向移動。

打到這裡就比較簡單了，白毛繼續向右邊安全與區移動，風箏莉法妮和皮耶魯。歐米伽可以直接收掉露娜並找個相對安全又能攻擊到莉法妮的位置待機即可。

剩下的回合數就是等技能CD和皮耶魯的位置了，只要位置合適就可以一擊必殺通關拿獎勵了。所以這基本上是依靠歐米伽才同掉的關卡。這裡我也同步一下練度。

傳說君的歐米伽目前6627戰力，攻擊綠字452，羈絆攻擊還是9級並沒有升級到10級，裝備白嫖弓、蜥蜴外衣、蛇頭、霸者。刺客科技的攻擊點滿了。這樣的歐米伽在buff完進擊、愛麗絲大寶貝，白毛超絕作用下還必須要開冷血才能點掉那個狗子，如果讓狗子開出來超絕難度應該更大才對。

所以如果你沒有高練歐米伽，還是建議考慮下毒奶的套路。這張圖正常攻略的話應該沒有太多迂迴空間，利用毒奶削弱對方整體的實力可能效果更好。感興趣的朋友可以研究下。本周超時空只能祝大家好運了。
