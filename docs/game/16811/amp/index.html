三國殺狂妄霸道到沒邊的台詞，嘲諷力MAX！

天天要聞
2019-12-16

眾所周知，在三國殺這款卡牌遊戲中，一句優質的台詞往往就能撐起一個武將的靈魂，或是霸道或是悲傷，一句好的台詞及配音能夠讓玩家對這個武將更有立體感，今天小編就來帶大家看看三國殺中那些狂妄到沒邊的台詞吧！

魏延：我自橫掃天下，蔑視群雄又如何

魏延的這句台詞可以說是最能代表狂妄這兩個字了，但是還是會讓人熱血沸騰，魏延也確實配得上這句話，在蜀國後期魏延可以說是蜀軍中唯一一員大獎了，可惜諸葛亮識人不明重用了楊儀這個小人，陣亡語音：”奸賊，害我！”更是道出了這個武力值爆棚的武將最後的心聲。

魏延在遊戲中屬於爆發性武將，奇謀的距離以及多殺搭配上狂骨的過牌使得魏延在菜刀流武將當中也是屬於絕對的T1級別，瞬間爆發能力極強，喜歡一回合秒人的小夥伴可不要錯過。

公孫淵：一生縱橫，怎可對他人稱臣

公孫淵戎馬一生，封王拜爵，最後翻盤出魏國自立為王，這一句台詞更是顯現出了公孫淵心中的宏圖大志，並且他也去實施了他的宏圖大志，雖然最後的結局是失敗沒能避免英雄遲暮，但是心有大志又怎能久居人下！

在遊戲中公孫淵定位適合忠內，懷異這個技能的優點在於能夠有效的破壞敵方的手牌質量或者壓制對方的裝備數量，可攻可守。新手玩家要注意發動懷異的時機以及考慮場上局勢再棄牌。

鍾會：我欲問鼎天下，試問誰與爭鋒？

這局霸道的台詞出資鍾會的第一個動態皮膚，台詞本身是非常霸道，但是配音方面略顯不足，總能給人一種陰柔的感覺，和這句霸道的台詞很有違和感，鍾會和公孫淵一樣也曾經想過自立為王，只不過不同的是他反叛還沒多久就失敗了，真是可歌可泣。

在遊戲中鍾會的定位一般默認都是內奸，因為鍾會本身的負嘲諷加上不動白的特性使得不管他是去打反賊還是忠臣都是坑隊友，既不能給主公保護也扛不起輸出的大旗，如果你是忠反，切記不要選鍾會。

賈詡：我自冷眼看世界，不問天下是與非

這句話出自賈詡的謀定天下這款皮膚，不得不說這局台詞和賈詡的生平簡直契合度搞得可怕，賈詡是一個絕對的利己主義者，也有人說他是三國時期最聰明的謀士，他總能跳出棋盤外去看待一些事情，這也是他能夠在亂世壽終正寢的重要原因。

賈詡在遊戲中幾乎所有位置都可以用，不管是主忠內還是反賊似乎都有著他的一席之地，一個好的賈詡要注意開大招的時機，玩的好就是賈詡玩不好就只能是甲魚了。
