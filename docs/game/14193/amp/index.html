CFS丨首支淘汰隊伍誕生！北美KZ兩連敗無緣八強

天天要聞
2019-12-10

12月9日，2019CFS世界總決賽小組賽敗者組的比賽開打，第一場比賽是B組的KZ對陣GV，這場比賽輸了的一方會直接被淘汰出局。結果GV氣勢如虹，打出6-0的開局，兵不血刃的以10-4的比分戰勝KZ。KZ也成了本次比賽第一支被淘汰出局的隊伍！

KZ戰隊是北美賽區CFS預選賽的冠軍，這支隊伍比較年輕，之前並沒有在國際大賽中出現過，但是能夠拿下北美賽區的冠軍，可見他們還是有一定實力的，只是在CFS總決賽的兩場比賽中沒有發揮出來。

北美冠軍KZ成為CFS世界總決賽首支被淘汰的隊伍，實力不足是一大原因，但是他們的運氣也實在是太差了。小組賽抽籤，他們被分到死亡之組，同組的對手是中國、巴西、歐洲賽區的冠軍，三個對手任何一個的比賽經驗都是他們的好幾倍。

KZ的運氣差不僅體現在分組上，他們遇到的地圖同樣很難。一場港口，一場衛星基地，都是比較新的地圖，尤其對於比賽經驗不太豐富的年輕隊伍KZ來說，對這兩張地圖的理解有限。

在比賽中我們可以看到KZ的隊員具備基本的槍法，但在細節上出現了許多低級失誤，至少在這兩張地圖上，KZ的細節處理還沒有達到與對手同樣的高度。

KZ的隊員首次CFS之旅表現平平，選手中KD值最高的是15ol的0.76。對於他們來說，這次比賽更重要的任務是學習，通過與世界頂尖隊伍的交手，他們積累經驗、提升自己的速度也會快很多。

最後，恭喜GV成功脫險，與SV爭奪一張八強門票。也期待KZ的小夥子們繼續努力，北美賽區的強隊不多，未來他們一定還有很多參加這樣世界大賽的機會，希望他們能夠努力提升自己的實力，爭取在2020年的世界賽舞台取得更好的成績！
