陰陽師亞服超級福利活動：自選SSR/SP概率UP活動 看完果斷酸了

2019-12-13 2,314條評論 3,793次閱讀 1936人點贊

話陰陽師近日亞服的更新中意外追加了一個新活動，本次的活動內容看點非常多，是國服沒有甚至可以說羨慕不來的福利，那就是自選SSR/SP概率UP活動，看完後果斷酸了。

這次亞服推出的新活動是自選SSR/SP概率UP活動，玩家可以7位高稀有度式神作為主要UP對象，主選式神1位，獲得5倍召喚概率，享受1次UP機會，剩餘六位副選式神獲得3倍召喚概率，享受2次UP的機會。

這卡池乍一看不錯，但實際算下來是個坑爹的池子，因為沒有所謂的2.5倍概率UP活動，基礎SSR/SP出貨率沒有提高，也就是說主式神抽中概率只有4%，概率還是低了點，實際上還是坑。但是對比國服的SR召喚福利活動簡直就是piapia的打臉，這活動福利差距太大太明顯了，根本沒得比。畢竟概率再低也比我們隨機SR要強，萬一抽到了呢？

之所以出現這種看起來坑實際也坑，但乍一看還覺得不錯的池子，主要還是國服式神推出速度太快，藍票和勾玉消耗跟不上，一旦錯過了檔期SSR/SP式神的UP，後續再想獲得就非常非常的困難了。所以這麼一對比，這活動的自選UP意外的確實良心和不錯。

像日亞服這樣的福利活動不少，比如說之前的男女SSR分別UP的卡池，甚至還有SSR概率DOWN的額外活動，給玩家足夠的機會去解鎖月見黑成就從而獲得大量黑蛋這種，當然這些活動是早在一年多以前就有的，但國服至今一丁點要實裝的趨向和機會都沒有，所以很明顯這次的自選概率UP國服鐵釘不會有。
