轉會期剛開始就結束，uzi直播因提及阿水問題，還被噴蹭熱度

2019-12-19 161條評論 1,437次閱讀 350人點贊

2019年LPL轉會期正式迎來收官，憋了這麼久轉會期剛開始，就直接迎來收官結束了，也太快了！回顧這次的LPL轉會風暴，LPL的多數隊伍都變弱了，只有兩支戰隊徹底變強了。如今LPL的第一強隊應該就是FPX戰隊，就在剛剛剛正式官宣了，前T1上單選手Khan被簽下，而金貢也再次續約成功，瞬間讓FPX成為S10賽季最有機會奪冠的隊伍代表之一！

而LPL第二支強隊就是TES戰隊，karsa這次轉會期成功加入，並且跟左手兩人在韓服瘋狂上分，並且得到很好的成績。加上下路問題也把二隊選手調輪換，而且原打野選手球球也成功續約！可以說有這無限的可能性，現在被成LPL第二強隊也算實至名歸！

而關於御三家的消息，RNG的兩個上單選手AJ和lovezrr正式離隊，而打野選手是PDD推薦的小龍堡，（這裡存在玄學問題，不好說），以及還加入了一個AD選手Betty，不出意外也是坐冷板凳，畢竟uzi已經成功續約。還有就是二隊打野S1xu（王康燦）、中單yuekai（張月開）選手順利通過試訓考核，也調入RNG戰隊！

而RNG最讓人意外的就是mata的回歸了，本想著這次他是準備退役了，沒有想到直接回到了RNG戰隊當教練了！也算是意外滿滿！

WE戰隊的話自從xiye和大舅子離開之後，大家都知道WE算是準備拆新家蓋新房了。好的選手是一個都沒有買到，倒是教練方面買了一個。對於WE的現狀，估計沒有多少人看好！不過最令人好奇的是，EDG居然一個人都沒有官宣，唯一離開隊伍的就是前段時間的RAY皇！

而說到這次轉起期最讓人期待的就是IG戰隊，尤其是關於JKL的消息。本來不少人都期待今天會有一個的結果，然後就在剛剛轉會期要結束的時候，IG官博發出評論：jkl選手目前還是自由人，有消息IG還是會通知大家！這也就是說明，JKL至今還是一個未解之謎！

不過從賽高的可靠爆料來看，阿水肯定不會再續約了！而他如今的選擇就只有SN和EDG還有TES之間，畢竟這些隊伍有他之後，都會發生質的變化！具體阿水會做出什麼樣的選擇，讓我們拭目以待吧！

不過對於阿水的問題，剛不久前uzi在直播跟letme一起談論了一些。uzi稱：雖然我不知道他會如何做選擇，但我覺得他應該會續約！而後letme稱：阿水不是自有人嗎？自有不是轉會之後也能加入！結果被不少人狂帶節奏，稱他直播蹭熱度！而uzi直接表示轉會期以我無關，然後就直接去玩吃雞了！

對此大家怎麼看這次的轉會風暴，歡迎留言討論
