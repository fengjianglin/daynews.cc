RNG新上單好猛！和OMG訓練賽戰績曝光：連續三把暴打curse

天天要聞
2019-12-20

12月21日德瑪西亞杯終於要開賽了，相信大家對於各大戰隊轉會期後的新陣容都非常期待，尤其是一些轉會期間動靜比較大的戰隊。在12月20日，德杯前一天，有細心的粉絲扒出了RNG和OMG的三場訓練賽，其中RNG拿到了兩勝一負的戰隊，新人上單月開更是暴打了OMG的知名上單curse。

​訓練賽戰績的第一場如下，這十個賬號都是雙方訓練賽專用賬號，並不是平常直播打國服使用的賬號，飄飄花那邊是RNG，那個叫做「MLXG」的其實是SMLZ使用的。第一把雖然戰績上是OMG這邊獲勝，但其實是RNG贏了，應該是獲勝方提前投降結束了訓練賽，其中OMG上單curse在S9賽季表現非常出色，沒想到被RNG上單直接打炸了。

​第二把訓練賽的戰績如下，依舊是RNG這邊提前投降獲勝，其中還是上路RNG的上路又把curse打炸了，這也太強了吧！而這把RNG的戰術也有一些透露，其中小虎玩了盧錫安中單，betty玩的是蛇女下路。

​第三局訓練賽OMG終於贏了，贏得方法也和大家猜測的一樣，SMLZ和icon終於carry贏下了比賽。不過贏歸贏，RNG的上路又贏了OMG的上路！

​因為現在RNG的上單有兩位選手，一個是狼行一個是新上單月開。狼行如果連續三把暴打了curse的確會讓人非常激動，但是很可惜這個上單應該不是狼行，因為狼行是F閃，月開是D閃，這次訓練賽的上單都是F閃，所以應該就是新上單月開了。沒想到他的表現這麼出色，因為curse也算是新興國產上單的代表選手，月開連續三把都壓制，可見實力不可小覷。

​而打野這邊RNG應該是小龍堡，因為玩了兩把豹女，一把蜘蛛，都是小龍堡的招牌。雖然說訓練賽也只是一個參考，馬上德杯開賽了正賽打得好才是最重要的，不過至少通過這次訓練賽可以讓粉絲們對RNG多一點期待了吧！

那麼，對於這三場RNG和OMG訓練賽戰績，你們有什麼想說的嗎？
