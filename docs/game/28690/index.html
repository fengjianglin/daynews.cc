英雄聯盟：龍女重做圖首次流出，紫色皮膚手拿鐵龍爪，新版

2019-12-22 2,104條評論 3,127次閱讀 280人點贊

在今年，拳頭舉行了一次投票，目的是選出重做英雄，而參與投票的英雄有龍女，狗熊，稻草人，夢魘和蒙多，在一番激烈的角逐之後，龍女以全網第一的票數入選重做名單，而和龍女一起進入這份名單的，還有稻草人和狗熊。在前段時間，設計師曝光了狗熊和稻草人的重做設計圖，我們可以看到，在重做之後，這兩個英雄的風格更加粗野狂放了，而且稻草人更加符合恐怖製造者的身份，而狗熊也是有了半神的樣子，玩家對此十分滿意。而最後剩下的，就只有龍女了。

在不久之前，拳頭的設計師也是曝光了龍女重做後的圖片。龍女重做共分為三部分，第一部分是大致形象，我們可以看到，新版本的龍女相比現在而言，體型方面小了許多，但是更加健壯，而不是像現在這樣如同一個營養不良的女性，眼神更加兇惡，而且武器也變的粗大。現版本的武器如同兩個鐵疙瘩，在人形態的時候，攻擊方式也讓很多人吐槽，沒有一絲龍族後裔的感覺。

而新版本的龍女，則是改變了這一形象，武器變大了，而且腳部也更加細節，之前還是人腳，現在則是龍足，從身體狀態表明，龍女是半人半龍。當然了，最大的特徵還是她頭上的雙角，比之前更加顯眼了。在最後一版設計圖中，我們可以看到，新版龍女的皮膚是紫色的，手上有紅色的光芒，這可能意味著龍女的一個技能依舊是可以強化普攻的，而且從服飾來看，新版龍女的定位應該是一個重裝戰士。

現在的龍女定位是比較尷尬的，因為她可以出攻速裝，也可以出半肉裝，甚至是AP裝，但是很奇怪，雖然這三個出裝方式都很強，但是卻很少有人玩，因為她比較吃資源，如果在前期發育不好的話，她是沒有太大作用的，出攻速裝沒傷害，半肉裝扛不住傷害，AP裝吐痰可能連別人的盾都打不破，這就很尷尬了。因此設計師這次重做龍女，想必也是要改變這這種現狀，讓她的定位更加明朗。
