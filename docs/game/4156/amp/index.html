魔獸世界：簡單的好團長遇上內心戲十足的玩家，一句話的大戲

天天要聞
2019-12-09

摘要

《魔獸世界》懷舊服的黑裝備和DKP制缺陷的話題估計不少玩家很容易的就互相關聯起來，現在有個玩家“自曝”的DKP團就有一個超級有趣也是大家都有的“偏見”，結果是充滿歡樂的氣氛，但整個過程其實反映出來的是一個簡單卻超級好的團長，一場以個人為主的內心大戲。

《魔獸世界》懷舊服的黑裝備和DKP制缺陷的話題估計不少玩家很容易的就互相關聯起來，現在有個玩家「自曝」的DKP團就有一個超級有趣也是大家都有的「偏見」，結果是充滿歡樂的氣氛，但整個過程其實反映出來的是一個簡單卻超級好的團長，一場以個人為主的內心大戲。

事情是這樣的，因為團長找到小德問了下他下個CD的MC還來不來。就這樣的一句話，讓小德浮想聯翩，瞬間聯想到了之前其他成員的問題和自身的問題，因為自身並未缺勤過任何一次的團隊活動。團長問到還來不來，在沒有新人的加入下，為何團長還出此疑問？大戲就此開場。

以該玩家自省的問題，就是上個CD出到灼燒指環的時候，以「出分就後悔，但沒人叫讓」的情況下拿到了。只是，前提是該玩家已經有一個灼燒指環，而牧師（團長媳婦）則是沒有。小德以一分的優勢拍到指環，但團長不為私情果斷要自己媳婦下次再拿。拿到指環之後，該玩家因故三天沒上線。

以上是自身問題，而該DKP團還出現過數次的「畢業」退會的情況。比如第一個CD的法師拿到碧空之歌后，三天不上線，上線退會；獵人憋分拿到葉子以存在感薄弱果斷退會；狂暴戰拿下削骨沉迷PVP；法師SH統御之後直接團本中途退會。這些都是DKP團普遍存在的問題，特別是懷舊服，只是被小德聯繫起來一想，事情就不那麼簡單了。

所以，該玩家內心感嘆「時代變了」，這是DKP制的問題還是人心的變化？用他自己的描述，就是內心難以平靜，唏噓不已！但為了做好自己，絕不跟其他人一樣只為「私利」，於是安撫團長表示自己肯定準時出勤，絕不缺席！只是，轉折點就此產生，也是最歡樂的地方，團長在聽到小德不會缺席之後，只是簡單地回了一句話：喜歡玩手游的觀住弓中浩貝樂手游，

「哦，我還說你都畢業了，我打算讓我弟開小號進本拿點裝備呢，你要來就算了。」

就是那麼簡單的一件事，團長簡單的一句話：「下個CD來不來」。就讓小德瞬間腦補出來一場大戲，從自身上升到團隊再到人心。但也說明了懷舊服的現狀，信任已然很難維持團隊的完整，團長帶團不易，成員對親友的擔憂，即使誰都沒問題，也會出現各種的互相猜忌。

但不得不說，這個團長還是很堅持原則的，不唯親友是主力，即使是自己的媳婦，哪怕之前出現過各種拿到裝備就退會的情況，在懷舊服裡面能做到如此的堅持原則，已經實屬不多了。
