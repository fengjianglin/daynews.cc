讓你嗨上天！這款顛覆性音游收穫TGA兩項提名和97%好評如潮

天天要聞
2019-12-20

往上看，關注我。（￣︶￣）↗

說到音游，通常人們會想起這樣的玩法模式，每一首歌都成為一個獨立的關卡，合著音樂不斷掉落的節奏點，玩家通過集中節奏的契合度（是否形成Perfect）來獲得更高分數，比拼的是玩家的手速和節奏感。但最近，一款音游新作卻徹底顛覆了這個傳統模式，並獲得了巨大成功，這款遊戲就是我們今天要介紹的《再見，狂野之心》

遊戲首先於18年TGA亮相，然後今年初首先登陸Switch，憑藉出色的品質在19年獲得了TGA兩項提名。上周，遊戲正式登陸Steam，迅速獲得了97%好評如潮。

非典型性音游，有音樂有飆車還有動作

事實上，很難用一個準確的類別來對這款遊戲進行分類。遊戲的操作極其簡單，只需左右兩個方向鍵與一個確定鍵；遊戲的流程也很短，大約兩小時就可完成。但遊戲的製作組卻做出了小而美的極致，其中包含的元素豐富得讓人震驚。

首先，確實可以說這是一款音樂遊戲。在音樂聲中，玩家需要通過擊中更多、形狀更大的心，來獲取更高的分數。

但也可以說這是一款飆車或者跑酷遊戲，玩家伴隨音樂，在軌道上奔跑、踩滑板、騎摩托車，不斷變幻軌道。

甚至，還融入了不少動作、射擊元素，軌道上會有怪物對你發起攻擊，而你也需要通過發射激光、揮舞利劍等方式進行反擊。

優秀的關卡設計，精妙且絕不重複

基於玩法的徹底顛覆，讓遊戲的關卡設計也和傳統音游有了很大的不同。傳統音游每一關除了音樂不同，關卡都是單一的踩點打節奏。而在本作中，軌道式地圖給了關卡更多的變化空間，不同載具、障礙、攻擊方式的切換讓關卡變化多端。

而其中最有趣的設定，應該是雙子的響指關卡，雙胞胎通過打響指，可以在兩張障礙分布不同的地圖中，快速切換。玩家必須時刻關注現在的地圖和切換後的地圖，才能有效躲避障礙，難度頗高。

但無論多種類元素還是多樣化關卡，都在遊戲的高速節奏下一氣呵成完美融合，摩托與劍，激光與惡魔，狼與鹿，飛龍與帆船，在你的眼前翻轉，往複，彷彿置身於異界。律動！翻折！跳躍！飛翔！連續一兩個小時玩下來，你會體驗到彷彿磕嗨的奇幻感覺。

迷幻又酷炫的視聽體驗外，還有著出色的劇情細節

這種磕嗨的感覺，還來自於遊戲出色的音樂的畫面。畫面採用大面積色塊切割，背景以深色冷色調為主，人物和軌道物品則顏色豐富五彩斑斕，形成強烈的對比。強烈的色彩和光效，搭配動感十足的電子音樂，營造出一種迷幻酷炫的氛圍。

但遊戲卻不僅僅只有花哨的外表，實際上，遊戲講述了一個很有藝術性的成人童話般的故事。一位年輕的姑娘心碎了，在她的夢裡出現了一隻蝴蝶，帶她穿越軌道，擊敗死神，追逐失落的聲音，尋找另一個自我。整個劇情充滿了隱喻和細節，到了最後，你會發現，音樂越好聽, 視效越精彩, 故事也越殘酷。

遊戲名稱：《再見，狂野之心（Sayonara Wild Hearts）》

推薦度：★★★★★（除了稍微小眾流程略短，遊戲本身的品質挑不出任何缺點）

平台：Steam

語言：中文

價格：34元（19日前折扣-21%，原價45元）
