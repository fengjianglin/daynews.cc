一款無雙題材的Fate系列動作角色扮演遊戲

2019-12-25 2,087條評論 3,023次閱讀 1030人點贊

HELLO～大家好，這裡是小白的每日一游推薦時間。世上的遊戲千千萬，有許多好玩的遊戲由於缺乏宣傳，所以不被廣大玩家所熟知。在這裡小白每天會為大家推薦一款評價很高但是不太出名的遊戲～

《Fate/EXTELLA LINK》是一款由Marvelous Inc.製作的動作角色扮演類遊戲，本作屬於EXTRA系列，內容相當於前作的番外篇，講述月海原聖杯戰爭之後的故事。除了前作登場的16名從者外，本作增加了包括查理曼在內的數名新從者。​

本作是一款無雙題材的遊戲作品，是Fate系列的又一力作，作品粉絲向有點嚴重，月球人高興壞了。遊戲在表現方面整體優於前作，尤其是人物建模，畫面質量，還有細膩程度方面有了不少提高。戰鬥方面也做了不小的改動，打鬥更加流暢，也更加華麗了，打擊感也加強了不少，成百上千的連擊帶來比無雙更無雙的爽快感。同時遊戲還加入了PVP模式，增加了遊戲的可玩性。相反的在改進了前作不足的同時，本作也有一定的缺點，那就是讓人看不懂的劇情，小白對Fate了解不多，感覺根本無法把劇情連接起來，估計也只有真正的月廚才能夠玩明白吧。推薦給月球人、Fate系列粉絲、還有想入坑或者體驗下無雙割草的小夥伴。遊戲支持簡體中文，Steam售價278RMB。

好啦，今天的遊戲推薦就到這裡了，小白這裡還有好多好玩的遊戲，咱們下期繼續。
