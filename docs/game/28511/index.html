螞蟻電競2019嘉年華，兩款顛覆行業的電競顯示器震撼首發

2019-12-23 1,685條評論 3,672次閱讀 1325人點贊

2019年12月21日上午，惠科集團旗下的高端電競品牌ANTGAMER螞蟻電競在武漢理工大學召開了2019 ACUL年度總決賽暨螞蟻電競新品發布會。

ANTGAMER螞蟻電競事業部總經理馬興海先生作為主講人為大家介紹了螞蟻電競的品牌口號從過去的「去突破」上升到了「勝在人為」，即【用確定性的裝備為玩家贏得勝利的確定性】。

ANTGAMER螞蟻電競共發布了兩款產品，主打電競賽事的螞蟻電競ANT27VQ，以及主打影音娛樂的螞蟻電競ANT32VQC。

螞蟻電競ANT27VQ主打賽事用機，平面顯示能夠確保每一位選手所見畫面一致。螞蟻電競欲把它打造成一款賽事/訓練專用的電競「公屏」。

螞蟻電競ANT27VQ擁有2K解析度，165Hz刷新率、1ms響應速率、垂直同步、HDR400這些頂級的電競基因，都能夠為用戶帶來極佳的電競體驗。

螞蟻電競這次不僅在「快」上誠意十足，也首次在電競顯示器上開始重點關注色彩。「實景屏」的概念首次被提出，100%AdobeRGB、100%DCI-P3、100%NTSC、100%sRGB，4個100%的色域覆蓋，色差△E＜2，醫療級灰階校準。這些恐怖的參數即便是很多專業顯示器也望其項背。

2999的定價更是讓全場所有人都驚到了，不少後面不少觀眾發出了「買爆」的呼聲。

第二款顯示器則是主打「私屏」的螞蟻電競ANT32VQC。這款顯示器擁有和螞蟻電競ANT27VQ同樣的電競/色彩參數，更是把屏幕尺寸提升到了31.5英寸。

螞蟻電競ANT32VQC最吸引人的部分，沒過其首發了三星1000R曲率的VA面板了。「月牙屏」的概念首次被引入，你可以體驗到前所未有的沉浸感。作為曲面顯示器的死忠粉，我一定會入手嘗嘗鮮~歡迎持續關注相關評測~

發布會結束後，惠科股份有限公司 MNT事業部的總經理-馬興海先生、營銷總監-彭新春先生、產品總監-王滌非先生，同到場的媒體老師和資深3C玩家們進行了深入的互動交流，向我們詳述了ANTGAMER螞蟻電競的品牌、產品以及未來的規劃等。並且非常耐心的回答了在場所有人的提問。

我在現可以第一時間體驗了場螞蟻電競ANT27VQ，顏值色彩真的讓我流連忘返。

除了新品發布，ACUL電競賽事也是本次活動的一大看點。現場比賽緊張精彩的令人窒息。

TT的MOD大賽也吸引了不少玩家的目光，每一個主題MOD作品都收穫了不少玩家們的肯定和投票。

現場也能體驗到很多TT家的產品，新發布的G821 USB/藍牙/無線三模鍵盤讓我期許不已。

作為一名3C的原創者，我能夠受邀參加也倍感榮幸，希望ANTGAMER螞蟻電競這個2018年出世的國產電競品牌能夠茁壯成長，在未來為市場和消費者提供更多優質的電競產品，期待下次能夠繼續參與其中。感謝您的閱讀，我們下篇再見~
