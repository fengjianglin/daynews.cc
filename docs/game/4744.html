騰訊手游打遍全球無敵手？CODM包攬谷歌年度遊戲三項大獎

往上看，關注我。（￣︶￣）↗

又到了年末，各大平台都風風火火開始了這一年的總結。在手游領域，當然最權威的就是蘋果IOS年度遊戲榜單，和谷歌Google Play年度遊戲榜單了。不過，從獲獎遊戲就看出，兩家平台的口味完全不同。

IOS年度遊戲榜單：熱不熱門不關心，重要的是有B格

今年蘋果給出的獎項相當精鍊，只區分iPhone、ipad、Mac三個設備，分別給出了一款最佳遊戲，以及最終的年度遊戲大獎。而遵循蘋果歷年評選的一貫喜好，這次依舊是主打藝術風格的精品優勝。

1、總年度&年度iPhone遊戲：《光遇》

這是知名手游製作人陳星漢，繼12年《風之旅人》後，再度拿下IOS的年度遊戲大獎。遊戲延續了陳星漢的禪式美學，柔和優美清新的壁紙級畫面，大量量使用風、雲、花草等自然元素，讓遊戲充滿著藝術氣息。

2、年度ipad遊戲：《Hyper Light Drifter》

可能不少玩家對這款遊戲都不是特別熟悉，這是一款復古像素風格的動作冒險遊戲，遊戲中沒有一個文字，完全依靠畫面推進劇情和流程，帶來一場”荒蕪世界裡的孤獨戰鬥”，一看就B格滿滿。

3、其他：年度Mac遊戲《Gris》&遊戲年度趨勢”經典新生”

年度Mac遊戲則被平台跳躍解謎遊戲Gris拿走，這依舊是一款畫面如夢如幻，格調感十足的遊戲。用清新透明感滿溢的色彩，通過流動、暈染、變化，構築出星辰、大海、植被，一個彷彿夢境般的世界。

遊戲年度趨勢”經典新生”，則是因過去數十年間的經典老遊戲如《跑跑卡丁車》《軒轅劍》《刺客信條》等，紛紛在今年推出了手游版本並收穫成功。

GP年度遊戲榜單：賺錢才是硬道理，熱度=品質

和IOS的少而精不同，谷歌榜單則開啟了大批發模式，除了最重要的”年度最佳”，其他多個獎項如最佳獨立、最佳休閑等，獲獎者都多達五個，其中有相當多我們耳熟能詳的作品。

1、 最佳遊戲&最受歡迎遊戲&最佳對戰遊戲：《使命召喚：手機版》

騰訊天美工作室研發的CODM無疑是今年GP榜單的最大贏家，無論官方評選還是玩家投票，都獲得了最大獎項。而作為創下手游首周下載量新記錄，成為今年最熱門遊戲的CODM，包攬三大獎項無疑也代表著谷歌官方對於遊戲發展的觀點：熱門、賺錢，才是最重要的！

當然有點諷刺的是，這款大熱遊戲雖然是國內廠商研發，卻依然是一款”全球上線、國服除外”的產品，國服具體的上線時間，仍遙遙無期。

2、 最佳創新類遊戲：《阿賽羅》、《刺客信條：燎原》、《刀塔自走棋》、《上古捲軸：刀鋒》、《Minit》

在這個類別里值得注意的是，多多工作室的《刀塔自走棋》榜上有名。雖然有著被抄襲風波，代理權轉換等各種爭議，但無論是遊戲的創新度，還是熱門度，獲得這個獎項都是實至名歸。

3、 其他獲獎情況如下

最佳獨立遊戲：《Ailment》、《毛線先生》、《格里姆瓦勒》、《星露穀物語》、《小房間故事》

最佳競技遊戲：《荒野亂斗》、《使命召喚：手機版》、《轟炸足球明星》、《Tacticool—5v5 Shooter》、《Tennis Clash》

最佳休閒遊戲：《憤怒的小鳥：夢想爆破》、《Diner DASH Adventures》、《高爾夫之巔》、《馬里奧卡丁車》、《碎碎神秘莊園》
