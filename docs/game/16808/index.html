4AM成功奪冠後虎牙給予最大冠軍牌面 韋神錯過見證奪冠時刻

2019-12-16 946條評論 3,166次閱讀 940人點贊

在今晚剛結束的和平精英首屆職業聯賽，4AM戰隊經過六天的競爭後脫穎而出成功奪冠，隨後被虎牙安排上給足了冠軍的最大牌面，而4AM老闆韋神後知後覺打完比賽後才知道自己的隊伍又奪冠，還送上了奪冠祝福。

　　在這次和平精英首屆職業聯賽，吸引了目前國內最精英的和平精英戰隊參與，而4AM在經過30局的比賽對決之後，成功在倒數第二局成功吃到蛋糕並提前鎖定了冠軍榮譽，這不僅是和平精英職業聯賽的第一個總冠軍，同時也是4AM和平精英分部拿到的第一個重量級冠軍獎盃。

　　在奪冠之後，作為4AM的簽約平台，虎牙給與了最高的牌面，眾多和平精英主播的直播封面全部都換上了4AM奪冠的封面，這也的牌面在這之前都是從未出現過的。而作為4AM的老闆，韋神在今晚因為自己有比賽，錯過了見證奪冠的瞬間，但仍然在第一時間送上了祝福，並且還祝福要一起為更多的冠軍而努力。

　　如今4AM成為唯一一支拿到了絕地求生聯賽冠軍和和平精英冠軍的隊伍，鞏固了國服第一吃雞戰隊的地位。
