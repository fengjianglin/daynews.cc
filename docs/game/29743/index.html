《漢家江湖》經歷屬於你的快意江湖

2019-12-24 1,998條評論 3,702次閱讀 1263人點贊

萬物可養成，《漢家江湖》歸根結底好像也是一款養成遊戲，各種角色養成。不過其中的自由度可不小。戰鬥隊伍由主角和三位夥伴組成，每一位角色都有武學、內功和天賦的基礎能力，不過發展三維的不同，卻沒有像其他遊戲的「上位」，基本上每一位夥伴都有自己獨特的能力。不過因為玩家太少，套路開發不足，還是有些夥伴淪為倉管。

冒險背景設定在南北朝時期，你跟隨師父一起找到了「十大名劍」中的無名劍，卻被覬覦名劍的賊人追殺，其間師父為了保護主角被賊人殺害，悲憤交加下你拔出石中的無名劍（EX咖喱棒），突然如有神助，暫且擊退賊人，而後師父的友人趕來，擊殺賊人，救走了你。這位前輩埋葬了師父後，送予你一份禮物。帶著前輩的贈禮，師父的遺志，還有「無名劍主」的新身份，你開始了你的冒險。背景劇情設定實在是太嚴肅甚至有點苦大仇深，但實際上這卻是一款有點無厘頭的遊戲，在遊戲中，你的夥伴有原創人物，如桃花塢傳人桃花仙子，天山派弟子凌雪，月白劍主張月，也有李白，蘇軾甚至黃飛鴻這樣的角色。你遇到的對手不僅是其他俠客，有時候還會遇上蚩尤。這麼混亂的角色登場之下，劇情卻有條不紊。由於超高的自由度，有時候甚至會分不清楚哪些是主線。（早期甚至因為自由度太高新人根本不知道該幹什麼）

很多夥伴都有負面狀態（早年這個飽受玩家詬病啊！！！！），比如托雷容易獲得「脫臼」debuff，屈心容易獲得「逃跑」debuff，可他們又偏偏擁有強力的能力。為了解決這個情況，大部分夥伴隨著劇情的進行，會新增一些「潛質」或者有特殊詞條的裝備，對這個自帶的負面效果進行修飾（不一定是強化哦）。
