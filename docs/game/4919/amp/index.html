1.鼠年限定皮膚公布 刀妹鎖了

在之前公布了新一批黑夜使者和黎明使者的皮膚時，拳頭也公布了2020年的新年限定皮膚，這次依舊是3位英雄獲得年限皮膚。

分別是老鼠、狗熊和刀妹，因為明年是鼠年所以老鼠是非常合適的英雄，密斯哥並且賭Miss美照一張，老鼠年限一定出至臻！

狗熊勉強還能理解，但是密斯哥就想不明白刀妹怎麼就和鼠扯上關係了？

密斯哥之前還猜可能會有貓咪，弄成對立皮膚之後又是新系列的皮膚。

來年腥紅之月也會添加維克托、千珏、掘墓，那麼問題來了，這些皮膚你最期待哪個呢？

2.TES經理髮文暗示野輔離隊

最初TES戰隊的組建讓許多觀眾期待，上野亮眼的發揮，中單的黃金左手，下路原JDG的Loken和原WE的Ben，但看陣容配置就比較豪華了。

但隨著版本的迭代變遷，之後的電競路TES走的並不順暢。

前些日子TES經理髮了一條他和Xx和Ben三人夜宵的照片，其中的文字引人深思，貌似Xx和Ben兩人都要離開TES戰隊。

Xx這邊也發布了一條微博，細心的發現他的微博也沒有了戰隊的前綴，加上qiuqiu離隊的消息，看來今年TES戰隊是有大變動了。

3.拉克絲皮膚特效更新

拳頭的親女兒拉克絲很久沒有更新皮膚了，確實很久沒出皮膚了，都2個月了。

而且即將上線的雙生皮膚還破天荒的給到了輔助索拉卡，但是沒事，既然不更新皮膚，更新皮膚特效也是一樣的。

這次拉克絲皮膚特效的優化受益最大的就是原皮膚了，自帶彩虹特效。
