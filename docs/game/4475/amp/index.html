吃雞：策劃二次空降虎牙，透露“英雄級”武器消息，有特殊功能

天天要聞
2019-12-07

摘要

大家好，歡迎大家來到孟婆小叔開講的《和平精英·吃雞小課堂》。不知從何時開始，光子搞起了策劃與主播聯動的環節。之前策劃小白裙98K就曾與主播開黑，本來以為這次會是四級包小姐姐，沒想到又是小白裙出現了。本來還想知道四級包對“皮大衣”的看法，現在看來是不可能了。

大家好，歡迎大家來到孟婆小叔開講的《和平精英·吃雞小課堂》。不知從何時開始，光子搞起了策劃與主播聯動的環節。之前策劃小白裙98K就曾與主播開黑，本來以為這次會是四級包小姐姐，沒想到又是小白裙出現了。本來還想知道四級包對「皮大衣」的看法，現在看來是不可能了。

不過小白裙二次空降虎牙，也帶來了不少關於新版本的消息，其中就有關於「英雄級」槍械皮膚的消息，一起來看看吧！

投擲系統即將同步，遊戲迎來「丟鍋」時代！

想必玩家們也都知道，端游更新了近戰武器投擲的功能，不僅僅是平底鍋，就連大砍刀之類的，也可以隨手就扔，而且還能造成一定的傷害。在小白裙與主播互動的環節中，小摳腳就曾問會不會同步端游的功能，而小白裙直接給出回答，說目前官方正在考慮。看來這個功能很有可能會出現在手游中，玩家們就期待著丟鍋時代來臨吧！

「英雄級」槍械皮膚即將上線，有特效無功能！

之前國際服出現了多款「英雄級」槍械皮膚，升級之後會出現很多特效，還會讓敵人的盒子發生變化。許多玩家都希望《和平精英》，也能夠更新這樣的槍械皮膚。雖然沒有特殊功能，但是這個特效玩法還是很不錯的。面對這一問題時，小白裙給出的答案是：不會出破壞遊戲平衡的「英雄級」槍械皮膚。

看上去像是拒絕，實際上的意思是會出，只不過沒有特殊技能而已。要知道國際服的槍械皮膚，也只是增加了特效，並沒有任何功能。也就是說《和平精英》中也會出現「可進化」槍械皮膚。不過據國際服玩家所說，一把槍械皮膚滿級大概要4000左右，具體上線到《和平精英》要多少，就不得而知了。小叔預計特效沒有那麼華麗，難度也會降低一些。

新版本內容搶先知，體驗服招募還未結束

每次更新遊戲中都會出現很多問題，各種BUG層出不窮。為了避免這一問題出現，光子這次在體驗服的招募上，可是下足了功夫。直接放出了25W個名額，已經過去很多天了，可是名額依舊有剩餘。想必光子是想通過增加玩家數量，來發現更多遊戲中的BUG，畢竟當前版本的問題太多了。

另外關於新版本的玩法，就是聖誕節了，預計會在12月20號前後開啟。到時候也會有相應的時裝更新，玩家們可以期待一下！

本期《和平精英·吃雞小課堂》到這裡就結束了，感謝你們的閱讀。
