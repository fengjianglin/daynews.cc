一經爆料就「虎虎生威」的機甲風，五虎上將的新皮膚。關羽、張飛、趙雲、馬超、黃忠，此五虎上將齊聚峽谷，天美必然會出一套皮膚，來加持五人屬性。眼看已經離幾位齊聚峽谷的日子越來越遠了，玩家們也逐漸失去了興趣，不料爆出了關羽的機甲皮膚……

誰也沒有想到，就是這麼一張爆料皮膚，會引得後續天美「食言」！

隨著這一張關羽的機甲皮膚爆出，也揭開了關於五虎將機甲皮膚的討論。還是怪玩家中有太多優秀的人才了，只要給了一個啟示，隨後後邊的「路子」自己走，於是很快，眼看著該系列皮膚就要爆個齊全，無奈，天美直接「腰斬」！

天美將這件事情定性為涉及商業秘密，直接暫停該5款皮膚的安排。以往，每一款新皮膚和新英雄上線之前都會有提前爆料，但是這次可能是涉及的內容太多，天美才下此決心的吧！

目前，可以知道的是原本，五虎上將系列皮膚是準備出的，此次算是延遲吧。不過對於很多玩家不太喜歡這類型的皮膚，只能說，遊戲中使用五虎上將英雄，再穿上統一的服裝，還要啥排面？可不要再說什麼果寶特攻之類的了，一個隊伍就是要整整齊齊，創峽谷新潮！

沒有了五虎，我們迎來了五嶽

試想一下啊，在天美公布亞瑟新皮膚之後又換成了雲中君，在五虎上將皮膚即將集齊的時候，又緊急暫停，這一波操作單憑天美赤裸裸的解釋不沒有任何說服力的。雲中君代替了亞瑟，第二款玩家自設計的皮膚還是在安排，那麼五虎上將的皮膚則是給到了五嶽系列來代替，只是，五嶽為毛是這5個英雄呢？

五嶽系列：韓信、甄姬、莊周、伽羅、典韋。衡山是韓信、恆山是甄姬、嵩山是莊周、華山是伽羅、泰山是典韋？真心想不明白，天美有什麼樣的說辭才會這樣安排！目前在我們看來，這完全就是為了達到某種目的不擇手段啊，因為五嶽和這幾位英雄之間並沒有任何的關聯啊。關聯度這麼低，不像是天美的風格。

唯一能說服自己的是典韋代表的是泰山，熟知如今的典韋可謂是站在「泰山之巔」，憑藉超高的勝率排行榜，可謂是常年霸榜！如此說來典韋還挨點邊，其他英雄呢？

「韓信有白龍吟、莊周有星空，典韋有戰令、伽羅有五五開黑節、甄姬有遊園驚夢，我都這麼支持你了，你還要我怎麼樣天美？」

用韓信、莊周、甄姬、典韋、伽羅這五位英雄來代替關羽、張飛、馬超、黃忠、趙雲，這依據在哪呢？寶哥提醒大家，可以對比一下這幾位英雄的熱度——上場率！（這麼說是不是有點道理？天美在做什麼，你要懂的！畢竟要過年了！）
