每年都會有LPL聖經誕生，之前的三大聖經分別是EDG廠長的「下路要我去，我怎麼去啊，對面一直進我也去」，RNG上單letme的「你們一直不給我反饋，緊張得手抖」，還有去年IG拿下了S8世界冠軍後，紀錄片放出了ning的聖經「我喊下路來，下路也不來，都是死人一樣」。終於在今年S9世界賽結束之後，FPX發布的世界賽紀錄片中，也公布了新的聖經，關於打野tian的。

​在FPX最新發布的世界賽紀錄片中，一上來就是聖經片段。在小組賽輸掉比賽後，tian非常生氣的一個人坐在了一邊，之後開始吐槽：剛剛輸的那盤，後選鱷魚和中路來counter，那他們只有AD霞有控制，沒有一條線有用，沒有一條線能打出效果。

​之後開始復盤環節，tian表示：如果這波我閃現沒有被刀妹打死，那麼然後呢？戰馬回應：沒有然後啊，我覺得很正常啊。結果tian開始聖經播報：那你是說我打得很爛嗎？結果戰馬直接回應：那你覺得金貢打得很爛嗎？tian再次回擊：那你要我說什麼？之後鏡頭給到FPX其他隊員，大家都是一臉尷尬坐在那裡。

​

​

​隨後金貢本人主動的道歉：我打得不好，不好意思，但是每個人都有打得好的時候，也有打得不好的時候，你會生氣因為我打得不好，是我的問題。tian繼續表示：主要是我不想抓上，他要我過去的時候，我就覺得打不贏。

​最後紀錄片中金貢也表示：自己並不喜歡玩抗壓的英雄，但是自己非常想要贏下比賽，所以水杯關注了自己並不在乎，只要比賽能贏就行了。

​

​從這段FPX的聖經來看，在S9世界賽初期，隊內的確出了一點小問題，不過好在大家都講自己的想法說了出來，最後在賽場上達成了一致。奪冠真的沒表面看上去那麼容易，都是一路很心酸地走過來的，大家有爭吵有協作這才是最正確的一支隊伍，遊戲裡面上頭了我感覺正常，相信大家在遊戲里也有過和朋友隊友吵架的，但是知恥而後勇自己調整，再團隊合作取勝，所以最後FPX值得這個冠軍！

​那麼，對於這段最新的聖經，你們有什麼想說的嗎？
