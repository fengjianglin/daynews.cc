夢幻西遊：每次周末好活動伺服器都卡，這下終於找到原因了

2019-12-05 0條評論 14次閱讀 0人點贊

摘要

大家好，我是小帝君（關注微信公眾號“遊戲小帝君”，每天給你帶來不一樣的夢幻西遊精彩資訊！）年輕人該多讀書、多旅行，放下手機，出去走走，接觸新鮮的事物，到最後你會發現…還是遊戲好玩。.

大家好，我是小帝君（關注微信公眾號「遊戲小帝君」，每天給你帶來不一樣的夢幻西遊精彩資訊！）年輕人該多讀書、多旅行，放下手機，出去走走，接觸新鮮的事物，到最後你會發現…還是遊戲好玩。.

不知不覺夢幻西遊已經陪伴我們走過了十幾個年頭，一直以來夢幻西遊最精彩的玩法之一就是周末活動，可是每次周末活動上號都感覺特別的卡，甚至可以說走一步退兩步。今天有位玩家發圖表示終於找到原因了。

No.1每次周末活動都卡，這次終於找到原因了

玩家竟然在皇宮發現了一位名稱叫作中國移動代維的玩家，難道說就因為這位玩家想要趁周末玩遊戲導致全服的網路沒有維護好嗎？其實並不是的這只是個玩笑吧啦，不過夢幻西遊中的遊戲名真的不怎麼好起特殊化的，比如看山哥的收藏，遊戲名為夢幻西遊的角色，還有其他許多名稱好像現在都不允許起了。好在夢幻不會隨便刪除玩家的檔案，不然連遠古產物都看不到了。

No.2玩家打寶圖，省錢省出新高度。

玩了夢幻十幾年，小編感覺現在的玩家越來越節省，小資玩家裝備不僅要求屬性，還要求永不磨損。而普通玩家為了追求屬性甚至選擇失敗2次，3次的裝備。更是在一些簡單玩法中，光著身子連裝備都不帶，就好像這樣能賺到很大的便宜一樣，就像圖裡這位玩家，打個寶圖身上一件裝備都不帶，最起碼你帶個武器不是。哎，真是會過日子，小編甘拜下風。

No.3策劃簡直過分，沉默模式變相收費，只為讓回歸老玩家買單

近期有玩家發給小編這樣一張圖片，原來玩家終於找到遺失許久早期的角色，所謂遺失不過是早期玩家離開後不玩的佔據多數，面對這種情況策劃竟然過分的退出沉默模式，不是點卡欠費，也不是異地登錄，而是以保護角色虛擬物品為由，想玩就要充值一定數量的點卡才能解鎖。簡直就是變相懲罰玩家遺失或者遺忘角色的懲罰。明明已經有了體驗和異地登錄的保護措施還要多此一舉。

No.4幫派迷宮暗改，多開玩家和工作室或遭打擊

幫派迷宮是幫派休閑玩法之一，最早期的獎勵比較簡單，由於需要幫主提前開設，所以玩的人並不多，在幾次更新以後，現在的幫派迷宮簡直成為多開黨和工作室的樂土，花費一定的金錢開了迷宮以後，就可以獲得諸多的獎勵，比如附魔寶珠等等，可是最近更新以後，有玩家發現開啟迷宮的花費好像提高了許多，變相的打擊了多開玩家和工作室的積極性。
