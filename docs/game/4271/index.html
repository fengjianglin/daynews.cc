第三款源夢皮膚正式曝光，伽羅喜提鼠年限定，S18戰令皮膚是他！

2019-12-08 0條評論 19次閱讀 0人點贊

摘要

王者榮耀這款遊戲最近舉行了一個王者創意互動營活動，邀請了廣大召喚師進行設計，目前已經有兩款皮膚成功入選，其中蔡文姬的繁星吟遊已經在遊戲中上線，而第三款源夢皮膚也已經曝光了。鼠年限定確定會給伽羅這位英雄，而S18賽季的戰令皮膚也來了！

王者榮耀這款遊戲最近舉行了一個王者創意互動營活動，邀請了廣大召喚師進行設計，目前已經有兩款皮膚成功入選，其中蔡文姬的繁星吟遊已經在遊戲中上線，而第三款源夢皮膚也已經曝光了。鼠年限定確定會給伽羅這位英雄，而S18賽季的戰令皮膚也來了！

源夢皮膚自從開始設計過後，就有許多召喚師加入，擁有千萬玩家的王者榮耀，有才藝的人並不少，尤其是在第一款源夢皮膚上線過後，玩家們的動力就更足了，畢竟誰都想自己的作品能夠得到認可。第二款源夢皮膚確定為亞瑟，而第三款源夢皮膚則是雲中君，天美也在邀請廣大玩家對此進行設計！

鼠年限定最開始許多玩家都以為是五虎將，但是自從官方公布答案過後，才發現根本就不是他們，而是另外五位英雄，其中伽羅將是最大贏家，限定傳說皮膚已經確定會給伽羅。一般限定皮膚都給那種比較火爆的英雄，就像上官婉兒那樣。

然後就是S18賽季，因為當前賽季已經要完了，29號的時候就會結束，玩家們可以在戰令系統中看到，其實在每個賽季戰令系統上線的時候就已經有了結果，但是問題就來了，那麼誰會獲得S18賽季的戰令皮膚呢？因為在新賽季會有兩位英雄能夠獲得戰令限定皮膚！

而S18賽季中，后羿就將獲得一級戰令，有網友就曝光了他一級戰令的皮膚。一級戰令和80級戰令皮膚還是有很大的差距，畢竟只要進階了戰令系統就能拿到一級戰令皮膚的獎勵，但是八十級戰令還需要升級才行。這次的戰令皮膚也還是很不錯的，至少后羿這位英雄比較熱門！
