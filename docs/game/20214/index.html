爐石巨龍降臨：標準模式各職業卡組勝率排行

2019-12-17 2,745條評論 3,733次閱讀 267人點贊

拯救牧師與削弱薩滿行動刻不容緩

●前言

截止目前為止天梯環境已經逐漸趨於穩定，但不包括可能會在明天到來的一波熱補丁。現在我們將過去一周時間裡，各職業的高勝率卡組進行匯總，以供大家參考。

數據來源：hsreplay

◆龍德

◆宇宙奧秘獵

勝率：58.1%

解析：勝率穩坐前三，也是當前的主流卡組之一。解場、過牌、爆發樣樣都有，唯一的缺點可能是因為貴吧。

◆任務獵

AAECAR8IrwSrBqCAA+OLA9ePA6SlA66xA5+3AwvTAdsJgQqghQPHnQPkpAO7pQOYqQOOrQP4rwOCsQMA

對局數：8600

解析：有朋友問舒瑪如何進構築，現在，它真的來了。早在預構築時期就有人提出配合獵人任務站場，如今看來效果出眾，勝率也相當可觀。如果在新版本有開出這張橙卡的朋友，不要著急分解；試試這套卡組，一樣可以上分。

◆宇宙法

勝率：55.6%

對局數：31000

解析：貴，除了貴就是強；平穩渡過前期即可。中期開始每合超模隨從往上拍，猶豫不決就解場；論控場能力只怕沒人敢和法爺比較。這套卡組攜帶了許多低費法術，與高費隨從間更有配合，手牌不至於過於臃腫。

◆奇蹟法

勝率：51.9%

對局數：3400

解析：即使不像狂野一樣用火妖來刷傷害，但把齊恩瓦拉當龍母來用效果也不差。況且這套卡牌還加入了召喚體系，88巨人一回合生幾個，場面壓力十足。

◆元氣騎

對局數：25000

解析：自閉的元氣騎，打薩滿高勝，但還是偏向於娛樂性一點。想要抓薩滿的玩家可以試試這套卡組，上分娛樂二者得兼，豈不美哉？

◆光鑄騎

AAECAZ8FAvoGw6QDDtwDrwePCdb+Atn+At3+AsWhA8ikA5amA4euA4iuA5uuA5yuA66wAwA=

勝率：55.4%

對局數：10000

解析：預構築的又一匹黑馬，原本以為捨棄了中立卡牌會產生種種副作用，但事實告訴我們，職業騎一樣可以更強。站場、貼BUFF、超模隨從78費連拍，還能補充手牌；中速騎領銜高勝。

◆心火牧

AAECAa0GBqUJ+wzl9wLQ/gKnhwOppQMM+ALlBPYH1QjRCtIK8gz3DK+lA9KlA/2nA4SoAwA=

勝率：53.8%

對局數：2100

解析：心火牧也是上古卡組了，但每個版本都可以玩得風生水起，畢竟經典卡永不退環境。並未加入太多新版本的卡牌，只因為這版本對牧師的確不太友好。除了本套卡組以外，其餘牧師卡組在hsreplay上勝率均未超過45%（甚至大多數都沒有超過40%）。

◆迦拉克隆亡語賊

◆迦拉克隆賊

◆迦拉克隆薩

◆迦拉克隆任務薩

◆迦拉克隆動物園

◆手牌龍術
