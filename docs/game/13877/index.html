《獵人》手游：時裝屬性與時裝合成

2019-12-12 119條評論 1,397次閱讀 239人點贊

時裝玩法是《獵人》手游中最受歡迎的玩法之一，玩家可以通過時裝改變自己的外形，讓自己變得與眾不同。同時時裝還可以為玩家提供不錯的屬性增益。相同部位屬性的合成，還有機會獲得更加珍貴的粉色時裝。所以今天，就讓我們一起了解一下，時裝究竟可以帶來哪些屬性加成。

時裝屬性：

時裝總共分為五個部位，分別是頭飾、長袍、腰帶、下裝、耳環。每一件裝備都有不同的屬性加成，其中包括附加屬性、可選屬性、套裝屬性等等。玩家購買時裝時，每一件裝備都有一次自主選擇屬性的機會。玩家可以根據自己職業的特點，選擇最適合自己的時裝屬性。

在選擇時裝可選屬性後，如果想要再次更換屬性。則需要消耗一個屬性重置魔方，該道具可在商城中購買。粉絲時裝和紫色時裝屬性加成的類型是相同的，但是屬性加成的數值會更高一些。因此選擇時裝除了漂亮之外，也要注意屬性數值哦！

穿戴同品質時裝到達固定件數，可以觸發套裝屬性。時裝套裝會增加玩家的各類基礎屬性，粉色套裝的屬性值比紫色更為全面，數值也更高。粉色五件套還會提升玩家的攻擊屬性強化值，以及50%的城鎮移動速度，在城鎮中效果非常明顯。

時裝合成：

玩家的時裝主要通過商城/活動獲得，活動可以獎勵一些時裝。商城中也可以直接購買時裝套裝/單件，購買時裝時有時間選擇，即可以選擇短期時裝，也可以選擇永久時裝。一套永久紫色時裝的價格為2080粉鑽，單件購買的價格與之相同。

合成時裝需要消耗一個幻彩魔方，將兩件同部位時裝進行合成，不同部位的合成概率不同。頭部合成幾率為15%；上下衣合成幾率為12%；腰帶與耳環合成幾率為20%；合成後必定獲得一個紫色時裝，並且有以上概率獲得一個同部位的粉色時裝。點擊添加時裝即可快捷購買或者添加時裝，點擊左下角必定合成選項可選擇種類。

玩家在合成時裝時還有概率會獲得更加稀有的粉色炫彩時裝，炫彩時裝與流雲風華時裝屬性相同，但外觀更加別緻。不同顏色的時裝也可以觸發套裝效果，每一件合成的時裝同樣有一次選擇屬性的機會。玩家可以根據喜好搭配自己喜歡的顏色，使得自己的角色更加的獨特，搭配武器強化的特效，盡顯王者之氣。
