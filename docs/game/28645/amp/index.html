元氣騎士：2.4版本大變更，近戰反彈實力坑隊友，機錘已成過去式

天天要聞
2019-12-22

隨著冬日漸深天氣也越發寒冷，距離聖誕節也僅有寥寥數天的光景，作為熱衷於《元氣騎士》這款手游的資深玩家，等待無疑是一種讓人為之煎熬的事情。記得上次大版本更新仍在10月份，拋開毫無新鮮事物的萬聖節版本，總共已有近3個月的時間未有新的遊戲內容加入了。正當所有玩家都在翹首以盼全新的遊戲版本到來的時候，《元氣騎士》2.4終於在大家的千呼萬喚中始出來。

作為真正意義上的更新，和之前萬聖節的敷衍截然不同，聖誕節版本毫無疑問為小夥伴們帶來了許多全新的遊戲體驗。據老宅不完全統計，在2.4版本中不論褒貶，全新的設定就有十數種之多。說到這裡或許有的小夥伴會表示好奇，為何新版本會出現降低口碑的貶義評論呢？

這是因為在《元氣騎士》新版本中問題頗多，對玩家的遊戲體驗帶來了諸多困擾，其主題大體可分為兩大方面：

  • 第1是對部分武器玩法、職業模式的強度調整，致使部分玩家難以接受。
  • 其2是聖誕節版本中好比「多如牛毛」的BUG。

要知道在歷屆版本中，如果出現對遊戲影響較為輕微的BUG等問題，玩家大多會將其稱歸類為「涼屋特性」而一笑了之。可是在2.4版中出現的問題，就連老宅也難以忍受，本期就藉此機會舉其中的個例和大家一同探討一下，看：2.4版本大變更，近戰反彈實力坑隊友，機錘已成過去式。

1、經典套路：機錘

關於機錘玩法，對於萌新來講或許有些陌生，但是對於那些久經地牢洗禮的老玩家而言一定印象深刻。作為老牌的經典套路，不論是輸出還是自保都堪稱完美，尤其是在近戰範圍提升天賦上線後，它的強度一度達到巔峰。

機錘玩法的核心並不複雜，是由白武鎚子與機器人英雄（激光增粗）兩相組合，再輔以攻速加倍挑戰因子強化，就可組成機錘套路的雛形。如果想要領略到這種玩法的完整版，那麼就需再度收集：近戰反彈、近戰範圍提升、攻速提升、狂戰士雕像、調酒師、磨刀石等強化方式，方可將威力發揮到極限。

滿配後的機錘傷害屬性並不高，但是與那些神器相比卻同樣不遑多讓，在數秒內擊敗強大的BOSS也毫無壓力。能獲得這種強大的能力，個人認為要歸功於激光增益的震蕩波與近戰反彈的反傷效果。

要知道高攻速與近戰反彈相搭配不僅能獲得堅不可摧的防禦效果，還能憑藉類似於「反甲」的能力以彼之道還施彼身，所以敵人的彈雨越密集則我們的傷害則越強大，因此機錘玩法才會得到業內的廣泛好評。只不過這種經典玩法，卻在新版本中成為過去式了……

2、天賦的削弱

在《元氣騎士》中首先獲得削弱的是：近戰反彈天賦，關於這項天賦技能對武器的強化效果，老宅已經在上面的機錘玩法中詳細解析了，就不再過多描述。可是它對其他技能天賦的連鎖能力，不知道小夥伴們是否了解？

當近戰反彈與聖騎士的能量護盾相互搭配的時候，能量護盾受到傷害時不僅能恢復藍量還能將彈雨全部反彈，對目標造成2次攻擊。這種能力與盾衝天賦、精靈雕像以及精緻法杖組合，都能得到完美的連鎖效果。

可是在最新的《元氣騎士》中，近戰反彈技能的二次傷害居然被取消了，而且更讓老宅難以忍受的是，反彈後的流彈仍然可以對玩家造成損傷！老宅直接懵逼了，如果這種情況出現在聯機狀態下，我方玩家一個「扣殺」將球反射給隊友，隊友由於血量不夠倒地GG，這種情況豈不成了100%實力坑隊友了。

那麼換一種思路：如果隊友用武器格擋來球再反射回去，我方英雄接住，再反射回去……如此看來豈不成了第2.4屆「元氣騎士杯」乒乓球的友誼賽了。

削弱遠不止如此，涼屋還移除了激光增粗天賦對錘類武器的攻擊範圍加成。所以，以後版本中再也不會有機錘這種玩法的存在了，玩家中只會留下一代經典套路玩法的傳說。
