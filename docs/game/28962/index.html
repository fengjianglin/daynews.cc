12.19體驗服更新：蒜香火舞削弱，諸葛亮加強，4件裝備調整

2019-12-20 211條評論 1,751次閱讀 311人點贊

大家好，我是老劉備。昨天體驗服大更新，更新了新活動玩法「覺醒之戰」，以及也對「無限亂斗」進行了升級。更重要的是，本次體驗服更新對孫尚香、火舞進行削弱，對馬可波羅進行平調，對諸葛亮進行加強，同時還對時光法杖、風之輕語、救贖之翼、衝擊之鎧進行加強！

孫尚香

削弱孫尚香的2技能穿透效果，大家都知道前期只要孫尚香21連招，任何脆皮都要剩下血皮，而且前期直接裸黑切的孫尚香，甚至可以直接秒殺脆皮。大寶備認為本次更多的是削弱孫尚香前期作戰能力，後期孫尚香暴擊成型以後，哪怕沒有2技能命中傷害也很高！

不知火舞

火舞2技能1級時候減少抗性一樣，2技能每升1級減少20點抗性，滿級減少100點抗性。

①這項調整會降低火舞前期裝備未成形時的爆發能力。

②降低火舞后期裝備成型絞肉能力，但對脆皮無影響！

不知火舞已經是頂端局出場率最高的中路法師！

不知火舞的削弱更多的還是因為高端局的影響！

諸葛亮

存儲時間沒變，變的是3次2技能的CD，單次減少0.5秒，如果連續3次使用2技能就是減少1.5秒，這其實是增加諸葛亮的容錯率，但是大寶備認為這點調整對於下限玩家影響不會太大！

馬可波羅

①1技能基礎固定傷害減少20點，但是AD加成增加0.02AD，也就是說馬可波羅只有AD加成達到1000點時候，才能彌補基礎固定傷害的削弱，而AD達到1000點是不可能的事情。

（3技能同理）

②能量消耗這項調整未實裝體驗服！如果實裝以後，馬可波羅能量消耗都會全期低於正式服最低50點，能量消耗降低其實也是提升馬可波羅下限的調整！

時之預言

法術攻擊：160→140

唯一被動-守護：

調整前：獲得額外的物理和法術防禦，數值等於你的法術攻擊*0.2，且最多不超過200點。

調整後：獲得額外的物理和法術防禦，數值等於你的法術攻擊*0.4，且最多不超過250點。

調整前，需要1000點法強才能達到200點雙抗，調整後只要625點法強就能達到250點雙抗。這件裝備調整之後，會更適合法戰/法坦/法輔使用，比如嫦娥、莊周、夢奇、露娜、孫臏、瑤、明世隱、大喬等等英雄！

衝擊之鎧

唯一主動-衝擊：

調整前：使用後2秒對500範圍內外圈敵人造成50%減速1秒，內圈敵人造成眩暈0.5秒

調整後：使用後1.5秒對500範圍內敵人造成50%減速1秒，內圈敵人還會額外造成眩暈0.5秒

衝擊之鎧對於預判要求很高，導致大家不願意去使用這件裝備，而這件裝備調整觸發時間以後，會更適合很多英雄，如亞瑟、哪吒、宮本武藏等戰士！

風之輕語/救贖之翼

風之輕語：

護盾量：250+25*英雄等級 → 380+38*英雄等級

救贖之翼：

護盾量：400+40*英雄等級 → 600+60*英雄等級

風之輕語和救贖之翼因為上次調整影響，並且其他輔助裝崛起（說的就是你奔狼），也導致他們的使用率下降。

大寶備做個對比吧：

風之輕語：

10.17之前：500+50*英雄等級（最多1200）

當前正式服：250+25*英雄等級 （最多600）

12.19體驗服：380+38*英雄等級（最多950）

救贖之翼：

10.17之前：800+80*英雄等級（最多2000）

護盾量：400+40*英雄等級（最多1000）

12.19體驗服：600+60*英雄等級（最多1500）

而這兩件裝備調整之後是團隊集體加盾，更適合團戰中使用，如果新賽季奔狼的BUG修復的話，那麼這兩件裝備就會頂替奔狼的地位！

你對本次調整有什麼想法，歡迎評論區留言！
