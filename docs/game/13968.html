2020遊戲大作扎堆？神仙打架的一年

2019-12-12 240條評論 2,912次閱讀 489人點贊

★遊戲馬蹄鐵原創

2020大作之年

在今天《生化危機3重製版》發布了預告片之後，相信很多人已經對於未來的2020年充滿了期待。因為明年將會是遊戲大作集體大爆發的一年，這一年裡將會出現非常多的超級作品。隨便數數，頑皮狗的《最後生還者2》，CD Projekt Red的《賽博朋克2077》，育碧家的看門狗以及可能會有的刺客信條新作，最終幻想7重製版，仁王2，三國志14，V社半條命VR新作Alyx等等。2020年不怕沒有好遊戲玩，就怕你的錢包撐不住。

大作爆發標誌著迎來下一個世代？

眾所周知主機即將迎來下一個世代，也就是PS5和Xbox two，今年穀歌也是推出了雲遊戲服務STADIA。世代變化會迎來大作潮，這並不稀奇。想想上一個世代交替的時候，也是出現了像《GTA5》等大作的誕生，因此遊戲大作在2020年迎來一次爆發也就不足為奇了。明年將會有非常多高質量的遊戲作品，值得玩家們去購買體驗。例如憑藉《巫師》三部曲而聞名全球的工作室CD Projekt Red帶來的新作《賽博朋克2077》這個全平台大作，從它宣布製作的那一年開始就備受期待，2020年也是終於要和玩家們見面了。

TGA估計會很難的一年

今年The Game Awaeds即將在幾天之後舉行，作為已經在圈子內形成了一定規模和人氣的年度頒獎盛典，TGA的評價和獎項是每年對遊戲影響比較高的。而明年的各大獎項毫無疑問將會是一場神仙打架，還沒有進入2020年的秋季就已經可以看到如此多的大作預告，年度遊戲花落誰家也是值得期待！
