部落衝突：13本和攻城訓練營被吐槽，新英雄會讓人失望嗎？

2019-12-04 0條評論 17次閱讀 0人點贊

摘要

各位首領們大家好，《部落衝突》這些天一直都在爆料年底大更新的內容，對於這些東西，玩家的態度當然是越早揭開真相越好。但COC擠牙膏似的爆料還是讓不少玩家不爽，不過也沒辦法，SC的一貫作風就是這，大家也只能慢慢等了。

各位首領們大家好，《部落衝突》這些天一直都在爆料年底大更新的內容，對於這些東西，玩家的態度當然是越早揭開真相越好。但COC擠牙膏似的爆料還是讓不少玩家不爽，不過也沒辦法，SC的一貫作風就是這，大家也只能慢慢等了。

目前已知的內容，就是13級大本營和新的攻城器械了，概念圖看著都挺不錯的，但從玩家的分析中可以看出，大家並不看好他們的實戰模型和作用，覺得有點敷衍玩家。剩下的新英雄，網上也有著各種傳言，估計很快就會有官方消息。

先來看13級大本營，官方的概念圖中，這座大本營看著十分的雄偉霸氣，鐵黑色的基座也給人一種堅不可摧的感覺，整體顏色和周邊的冰天雪地也可以融為一體，看得出，美工確實是用心了。

但在實戰圖中，玩家給出的評價就是一個字「丑」，並且還調侃為是箭塔和地獄塔的組合。期初對於這樣的結論，曉東只是當個玩笑話的，但當我看到這幅圖後，才發現玩家們的腦洞實在是大啊，這個13本的進化過程確實沒毛病。

而對於新的攻城機器，開始曉東還以為會像降落傘一樣隨意投放，但事實上，它只能當作是個移動堡壘，因為它的降落需要空地。這也就意味著，攻城訓練營基本只能放在對手村莊的外圍，那麼這個訓練營的作用要麼就是先手清邊，要麼就後手收尾。並且裡面只會放出1個大皮卡，如果先手的話很容易會被集火秒掉，後面的法師也就等於白送了，和大部隊配合也是挺不錯的。只能說，用這個還是得看對手陣型的。

最後再來聊下新英雄，早前視頻中，官方對於他只放出了一張殘破的照片，於是玩家全都變身福爾摩斯，從照片中的蛛絲馬跡開始猜測新英雄。有說是建築工人王，有說是哥布林王，還有說是野豬騎士王，到底是什麼，誰也沒個定數。

而在新版本來臨之前，又有玩家爆出了新英雄的消息，就是圖中這個皮膚黝黑的「鐵憨憨」，據說攻擊手段是以魔法為主的，咱們暫時就叫他「法師王」吧。只是這樣子也太丑了吧，如果真是這樣的話，玩家們不知道又該會怎麼想呢？

好了，本期的內容就和各位小夥伴聊到這裡，你覺得新英雄會是什麼呢？歡迎評論區留言，說出你的看法吧。
