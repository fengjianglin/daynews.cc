LPL轉會期情況，The shy轉會費6000W，JKL去向不明

天天要聞
20 小時 ago

在經過整個S9賽季的洗禮，有很多選手在憑藉著場上出色的表現提高了身價，並在S10賽季轉會中有了更多的選擇。其中The shy更是被某LPL隊伍開出了高達6000W的轉會費，但是IG仍然沒有選擇出手，相信關注S9世界賽的玩家都知道The shy有多強，可說是世界最強上單選手都不為過，開出6000W的高價轉會費也是合乎情理，當然IG沒有出手也正是看中了The shy強得離譜的實力。

除了LPL隊伍想買The shy之外，還有T1也報了100W美金，雖然相比LPL隊伍的6000W這相差了近10倍的價格，但是T1老闆並不知道LPL的轉會價格，在他看來100W也是很有誠意了。

雖然The shy留在IG，但是目前JackeyLove的去向不明，另外從IG買入Puff和Southwind這個下路雙人組來看，很有可能不會續約JackeyLove。目前為止找過JackeyLove的戰隊有PDD的戰隊、RNG、SN、TES等，但是JackeyLove一直沒有公布轉會消息。然而非常有意思的是在LOL季前賽這個對AD非常不友好的版本，UZI表示打不動AD了，所以很多玩家都猜測RNG要買下JackeyLove用來打AD，而UZI轉去中路。其實UZI的確在皇族打過一段時間的中單，因此這個想法也不是沒有可能。

RNG這次也是轉出打野karsa到TES，對於karsa在RNG的表現，相信各位玩家有目共睹，可以說是RNG隊內發揮最穩定的一個位置，其實karsa離開RNG也讓很多RNG粉絲不舍。RNG新買入的xiaolongbao和S1xu將接替karsa擔任打野位，xiaolongbao出自的電競黃埔軍校YM，其打法兇悍，意識超群，操作犀利，也是非常擅長盲僧，結合了karsa和MLXG的優點，相信他能在RNG打出優異的成績。另外YM的小天和寧王都拿到了S系列賽世界冠軍，不知道xiaolongbao能否延續電競黃埔軍校的名號，攜手RNG一舉奪下S10世界冠軍呢？

轉會期EDG也爆出不少有趣的消息，EDG廠長轉型擔任教練，相信聽到這個消息很多玩家心裡都默默流淚：廠長不上場了，我的青春結束了。另外國產ADiBoy離開EDG，加入了VG，作為天才AD出道的iBoy在加入EDG之後就有不俗的表現，大家都十分佩服他的光速反應，說他是EDG的大腿之一都不為過，但是這次EDG沒有入圍世界賽也可能給到iBoy一些打擊，所以想去VG另謀發展。

另外大家都比較關心的FPX轉會情況，金貢成功續約，除此之外Doinb還成為了LPL本土選手，他直接跑到韓國把Khan拉攏過來，金貢能抗壓，Khan善進攻，FPX攻守兼備完美了。
