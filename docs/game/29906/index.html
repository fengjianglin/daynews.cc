DNF：12.23更新，春節上線強化保底功能，+12成功率可達到100%

2019-12-23 271條評論 2,169次閱讀 764人點贊

DNF：12.23更新，春節將上線強化保底功能，武器上12要變簡單了！

以下為具體的內容詳情：

對於那些玩了百分比職業的玩家來說，其角色最低也要擁有一把強化到白12及以上數值的武器，這樣才能算是達成「畢業」。不過這話雖然說得簡單，但實際操作起來卻很難，運氣好的玩家就不說了，+12的武器直接都可以白嫖，相反對那些運氣較差的玩家，幾千萬上億金幣下去都不一定能強化的出來。

而我們的策劃或許也良心發現意識到了這一點，早在幾個月前就透露的武器的強化保底功能，根據體驗服論壇上最新的爆料公布，預計終於要在1.14的春節版本中更新上線，玩家的武器上12要變簡單了，難度要進行減少了。

說起這個後續即將在1.14春節版本更新的強化保底功能，相信玩家們對其都有一定的了解，必定這早在幾個月前策劃就曾提前透漏過一次，且在體驗服的拆包內容中還提前發現了相關的文件，只不過一直被策劃拖到準備在春節版本才會實裝上線而已。

針對於這個強化保底功能，如上圖所示簡單來說就是可以增加武器（僅限武器）強化11/12的成功率，比如+11上+12，每次強化失敗都會增加下一次強化成功幾率的數值，也就是說你累積失敗的次數越多，下次武器成功+12的幾率數值就越高，臉黑的玩家直至最後或許都可以達到100%的強化成功率。

當然，有關強化失敗可增加下一次成功率數值的功能，策劃還加入了一定的限制，如果你使用那些強化卷，比如+12的黑鐵強化捲去強化武器，那麼無論失敗與否都不會增加下一次的成功數值，只有到凱麗那裡進行強化，或者使用強化器去強化，這兩者在上11/12失敗的情況下，才可以增加下一次上11/12的成功率，一點空子都不會讓你去鑽，難受了。
