穿越火線先遣服更新結束，臨近年關年度重磅版本就要來襲，首先是年度套裝，現在年度套裝還沒有全部公布出來，商城當中只有一個英雄級角色，現在推出的英雄級角色按照策劃的尿性必定是加入到年度套裝當中，並且這位新角色不僅有三防屬性還有幾個特殊的新屬性—連殺的代價：在所有可以連殺的模式中，大幅度增加連殺最大的間隔時間。積少成多：在爆破、團隊、特殊戰中，每累計傷害到達100就可以獲得擊殺一個人的經驗，但是這個擊殺不計入戰績。

並且新角色還擁有挑戰強化技能，就如同櫻的覺醒一般，可以說這個角色的屬性是當下最強的屬性沒有之一，現在想要延長連殺時間需要升級英雄級還有隨機戒指才可以勉強達到，但是這個英雄級角色就可以直接大幅度的提升連殺的持續時間，基本上是可以和一個滿配的戒指相比，甚至屬性還要更加優秀，看著這麼強的英雄級角色，看來錢包又是要瘦了。

本次更新最大的重點還不是年度套裝，而是額外背包，現在玩家的英雄級武器已經滿倉庫都是了，只有七個背包讓許多的英雄級武器都在倉庫當中暗無天日，原本我們用的都是B背包，現在新增A背包，需要先購買3-7號背包永久才可以滿足開啟條件，不過價格確實是令人不可以接受，想要完全開通雙背包就要63700點券才可以差不多是一款英雄級的價格了，並且額外背包還不是永久的要一直續費才可以使用。

並且還推出了A背包套裝手雷包。A背包每個自帶一個手雷位置，需要擴充手雷攜帶量就要購買相應的新增道具，價格3000點券90天，要記住AB背包只能在倉庫當中切換，同一款武器不能放在兩個背包，兩套背包的英雄級武器屬性也不能共享，使用的是哪套就是哪套的屬性，基本上就是說這個背包屬性還是很雞助，差不多1000塊僅僅就是可以讓你在倉庫快速換武器，這個真的沒什麼必要。

先遣服還推出了新的挑戰地圖空間裂縫，BOSS看起來還是可進化的BOSS，近期推出的挑戰地圖難度都特別高這次推出的挑戰地圖不知道會不會適當地降低一下難度，不然很影響玩家的遊戲體驗，慢慢的就跟不上版本最終被淘汰掉，獎勵看起來應該是副武器，所以沒有什麼太大的亮點，畢竟現在挑戰模式玩家本身就不多。
