《風雲島行動》公布新英雄少女弓手，箭居然全靠撿？

天天要聞
2019-12-13

要說《風雲島行動》這款手游，也算是眾多生存競技手游中的一朵「奇葩」了，各種奇絕設定層出不窮，異類英雄比比皆是，今天要和大家分享的這個英雄就屬於「奇葩」中的「奇葩」，別的英雄落地就能「硬剛」，而這個英雄竟然有可能打著打著就「束手就擒」，這又是何道理？

要解釋這個得從《風雲島行動》的遊戲機制說起，在這個遊戲中玩家英雄的遠程攻擊基本上靠射箭來進行，每一發箭都需要拾取後才能射擊，射出去就沒了。

別的英雄都是靠技能戰鬥，射箭輔助，唯獨射手英雄慘了，技能都要靠射箭來釋放，一旦沒有了箭矢瞬間就「手無縛雞之力」，風鈴兒！對，說的就是你！

溜了溜了

當然並不是說這個英雄不夠強力，作為少數幾個遠程英雄之一，風鈴射程遠、技能範圍廣、威力強，最重要的是長得可愛，除了身板脆了點以及日常缺箭就沒啥缺點了，所以還是得到了不少玩家的喜愛，畢竟強只是一個版本，可愛才是一輩子的啊！

於是乎廣大風鈴兒的玩家進入遊戲後的第一件事，就是先跳一個偏遠山區，發揮「三光精神」搜刮視線中的每一座房子。

並且不放過視線範圍內的任何一棵樹，至於為什麼是樹，那是因為《風雲島行動》除了地面上隨機出的箭外，還能靠砍樹來造「箭」，等到存上百八十根箭，然後再出來箭指江山。

所以如果遊戲前期看不到風鈴兒玩家的人影，後期就能看鋪天蓋地的「箭影」了。

那麼如果在遊戲前期，自己箭矢不多的情況下碰到敵人怎麼辦？那就得祭出風鈴兒的「大招」——錦囊計「快速生長」了，瞬間催生七棵樹，媽媽再也不用擔心我的箭不夠用了！但這個錦囊技能一局遊戲只能用一回，何時釋放就得玩家們自己把控了。

怎麼樣，這個英雄是不是有夠「奇葩」呢，不得不說在《風雲島行動》中比她奇葩的英雄還有很多，具體能有多「奇葩」？等下周測試去遊戲里你就知道了！
