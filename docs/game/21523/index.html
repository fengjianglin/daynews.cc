CMCC湖南總決賽冠軍專訪：劍指全國冠軍

2019-12-19 672條評論 3,832次閱讀 913人點贊

12月15日，2019首屆中國移動電子競技大賽湖南賽區總決賽在長沙開福萬達廣場金廳盛大開幕，從湖南全省選拔出的28位選手在決賽中上演了一場場精彩紛呈的巔峰對決，讓我們感受到了電競湘軍的強大實力。最終，《英雄聯盟》SOLO賽冠軍被來自常德的選手任波奪得，團隊賽冠軍被岳陽戰隊「正月初五影業隊」收入囊中，《爐石傳說》項目賽冠軍的桂冠則被長沙本地選手徐鑫濤摘得。

在英雄聯盟團隊賽決賽中，正月初五影業隊直落兩盤，乾淨利落的戰勝對手，行雲流水般的團戰配合給觀眾們留下了深刻印象，拿到湖南省區冠軍後，他們也獲得了晉級全國總決賽的資格。賽後，小編第一時間聯繫到戰隊選手李文傑、彭佳偉、廖康健、彭一博、付才良、越雲貝，請他們聊聊奪冠背後的故事。

小編：正月初五影業隊是如何組建的，成軍過程中有什麼趣事嗎?

李文傑：我們幾個都是英雄聯盟老玩家了，經常在線上一起開黑。這次看到移動舉辦電競賽事的宣傳，大家就有了組隊參賽的想法。也算是追夢吧。因為隊伍里大多數是湖南人，所以戰隊報名參賽地在岳陽。

彭一博：這支戰隊是因為CMCC而組建，能夠聚在一起奮鬥，我們還是很感謝主辦方提供了這樣一個電競舞台。說到趣事，可能很多玩家都不知道我們隊名「正月初五影業」是什麼意思，可能大家會想到LOL里的演員梗。但其實是因為隊伍發起人的工作單位是一家名叫正月初五的影視公司，這家公司為我們參賽提供了贊助，所以就直接冠名了。

小編：在剛剛結束的湖南總決賽中你們發揮出色，參加這次比賽有什麼特別感受嗎?

彭佳偉：雖然我們也有在線上進行訓練，但真正來到線下參加電競比賽還是第一次，看到現場大批的觀眾、專業的電競舞台;聽到S9官方解說西朴解說我們的比賽;甚至連iG戰隊的職業選手都來到現場，大家還是蠻緊張的!比賽雖然贏了，但上單和打野的配合還是出現了不少失誤，接下來我們還會繼續磨合，把隊伍調整到最佳狀態。

小編：想特別問下廖康健選手，拿到團隊賽MVP你感覺如何?

廖康健：很激動!最開心的是獲得了去iG訓練營參觀學習的機會。LOL是一個節奏很快的遊戲，團戰時需要玩家在瞬息之間釋放大量技能以及頻繁走位，所以網路非常重要。能贏得比賽拿下MVP，我最大的感受就是決賽現場的移動雙網雙千兆網路特別出色，延遲比我們在線上玩遊戲時低了不少，高速流暢的網路給了我們很好的競技體驗。

小編：除了移動雙網雙千兆為大賽提供網路支持外，煥新升級的中國移動「動感地帶」品牌也在湖南總決賽現場被點亮，大家有接觸過動感地帶嗎?

越雲貝：動感地帶是一個針對年輕人的品牌，而玩電競的大多也是年輕人，兩者之間還是挺契合的。我在決賽現場也看到了動感地帶的品牌展示，等5G慢慢普及後，希望升級後的動感地帶能給我們帶來更潮更有趣的流量及內容服務，如果有電競相關當然就更好了!

小編：最後再次恭喜正月初五影業隊獲得湖南總決賽LOL團隊冠軍，接下來你們有什麼目標?

付才良：我們代表的是電競湘軍，所以一定不能給湖南玩家丟人。接下來我們會加緊訓練，劍指全國第一!
