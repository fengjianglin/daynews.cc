靈魂戰紀：最新更新內容爆料，遊戲內容迎來大幅度調整

天天要聞
2019-12-21

一個遊戲的趣味性和新鮮感，自然是離不開對遊戲不間斷的更新和維護，網路遊戲更是如此，只有不斷維護遊戲，查缺補漏，更新遊戲，才能始終保持遊戲的熱度。自然的，遊戲更新的重要性可以說是不言而喻。

我們的靈魂戰紀也是如此哦，經常會有遊戲的更新，有對遊戲的維護和調整，也有的是新內容或者活動的上線。但不論怎麼說，這對於玩家來說無疑是有好處的，畢竟誰不希望自己喜歡的遊戲更好玩呢？這次的遊戲也是迎來了一波不小的更新，小夥伴們來康康吧！

55級的頂級黃金裝備，這無疑是遊戲中的超頂級的裝備了，而強化起來的消耗也是非常大，畢竟強化消耗和裝備質量是成正比的。55級黃金裝備的強化所需金幣調整，包含傳承套、暗中出生套、金色審判套：

+1至+7，每次強化所需金幣-30%

+13至+18，每次強化所需金幣+5%

感覺這調整，應該是能省下些金幣？

這個調整的角色50級禮包中的【能量轉換器】*10調整為【防止強化破壞裝置】*10，就是專門給裝備強化時用的。

遊戲中除了那些主線劇情的副本以外，還有這種區域突襲的高難度副本，比之劇情副本的難度更高，但是也是更能掉落寶貴的材料和裝備。【最後狂歡】、【殘渣蜂巢】、【土匪公路】、【鋼鐵城堡】、【金色城堡】五個副本調整

怪物攻擊力下降20%左右

組隊時怪物血量加成降低

這個對副本的調整無疑是讓這些副本的難度降低了些許，對於玩家來說這可真是個好消息哦。
