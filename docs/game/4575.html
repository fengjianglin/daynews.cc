12月6日，LOL全明星正式開始，在感受到了團體賽和雲頂之弈的魅力同時，最讓網友們期待的就是這職業選手之間的Solo大賽了。對於UZI來說，這個Solo賽可是自己最愛的，畢竟Solo王這個稱號可是得到了公認的，Solo冠軍頭銜已經證明了一切，對於這又一次的Solo賽，UZI也是全力以赴的，在團隊沒有拿到好成績的時候，拿下這個全明星的Solo冠軍也算是一絲慰藉了，也證明了，多年老將也是可以依舊操作在巔峰的。

​

有意思的是，這場Solo賽對手可是鬧笑話了的哦，可以看到，這位對手在擊殺了UZI拿到一血的時候可是極度開心的，都是準備摘掉耳機去握手慶祝了，內心是很開心的，畢竟是擊敗了UZI啊。但是隨後工作人員上來了，表示比賽繼續，原來這位小兄弟可是規則都沒有弄清楚啊，並不是拿到一血就勝利了，這空歡喜一場啊。話說，要是知道了拿到一血不算贏，這個小夥子會不會就不越塔強殺了呢？

感受到了壓力，UZI也是火力全開，其實前期是有點逆風的，但是後面靠著一波極限操作，UZI險勝晉級，可惜了對面的小夥子了，有點意思。

​

贏下這場勝利後，UZI也是尷尬的笑了笑，畢竟險勝還是蠻玩心跳的，壓力有點打，這要是輸 了，輿論怕是又要起來了，主要是被越塔了，有點難受。還好結果是好的！

​

Solo賽大戰依舊在繼續，這接下來的戰鬥只會越來越激烈，也是期待UZI的表現，再次拿到Solo王就滿分了，誰會是最大攔路虎呢？Theshy還是Bang呢？
