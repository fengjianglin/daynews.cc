日星隱曜，山嶽潛形，玩家在《方舟生存進化》造出超真實岳陽樓

天天要聞
5 天 ago

要說我國有哪些古典明樓，大家肯定會想到范仲淹先生曾經用《岳陽樓記》所描述的那座岳陽樓，”若夫淫雨霏霏，連月不開……”，范仲淹先生用《岳陽樓記》將那岳陽樓的風光給描述的淋漓盡致，彷彿自己的眼前就是岳陽樓一樣。可詩詞再美，也比不過自己親眼看一看，不過在現代這個節奏很快的時代，相信很少有網友能一覽岳陽樓之美了吧！這個時候在Steam上線的沙盒神作《方舟生存進化》就能幫忙，這是一款自由度如此之高，風景如此之美的遊戲，以至於貼吧大神玩家能在《方舟生存進化》中再現岳陽樓的美景，小編看到後也是大寫的佩服這位玩家。

最近就有一位玩家做到了這樣的事情，它利用《方舟生存進化》的沙盒機制，將岳陽樓成功的再現到了《方舟生存進化》裡面，儘管在細節上有所差異，但整體上還是能讓我們感受到岳陽樓那極其美麗的環境的。

從圖片上我們可以看見，岳陽樓有山、有水、有樹、有花，不管是什麼樣的閣樓，只要將這四點全部聚集起來就一定會是非常美妙的地方，如此之美的場景，也難怪岳陽樓能讓無數文人騷客戀戀不捨了。並且我們也能看出《方舟生存進化》對於沙盒性質的用心，要是你不自己體驗一下的話，你很難想像這會是一款沙盒遊戲做出來的效果。

既然這個玩家能在《方舟生存進化》中做出岳陽樓，那麼其他的玩家也一定可以造出相同的景色，只不過這件事對於絕大部分玩家來說是很困難的一件事，首先你需要耗費大量的時間去採集各種資源，然後還需要你的電腦有一個很好的配置，因此像小編這樣的庸人還是只看看大神作品就好。對於這位玩家所營造的岳陽樓美景，大家有什麼看法？可以在下方評論區留言討論。
