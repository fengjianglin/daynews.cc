iFTY與OMG沙漠房區之爭，iFTY打出0換4，賽後老菊採訪很真實

天天要聞
2019-12-05

摘要

不知道玩家們是否還記得此前在PCL夏季賽季後賽階段，OMG與iFTY的衝突嗎？因為小獅子跟龍四哥聊工資的事情，導致iFTY老闆A+親自下場怒撕，這場鬧劇最終也是以A+道歉結尾，不過再結合此前OMG教練老虎挖iFTY選手的事情，雙方的粉絲可是互相看對方不順眼了。

不知道玩家們是否還記得此前在PCL夏季賽季後賽階段，OMG與iFTY的衝突嗎？因為小獅子跟龍四哥聊工資的事情，導致iFTY老闆A+親自下場怒撕，這場鬧劇最終也是以A+道歉結尾，不過再結合此前OMG教練老虎挖iFTY選手的事情，雙方的粉絲可是互相看對方不順眼了。

雖然說目前OMG在國內的成績並不好，而iFTY則是老隊新生如日中天，但俗話說瘦死的駱駝比馬大，作為曾經拿到PGI全球邀請賽FPP模式冠軍的OMG，在經過隊員變動後，依舊有國際擊殺王小獅子和功勛元老小海坐鎮，紙面實力並不弱，玩家們也十分期待雙方能夠在賽場上互相碰一碰。

在絕地求生微博杯BD組小組賽中，OMG與iFTY相遇，雖然說兩支戰隊在海島地圖有跳點重合，但始終沒有見過這2支戰隊打一場正面戰。不過在最後一場沙漠地圖的比賽中，這2支戰隊終於滿足了玩家們的願望，OMG跟iFTY終於來了一場4打4的房區遭遇戰。

OMG在攻iFTY所在的房區以前，已經順利拿下了EDG的房區，不僅保持了滿編還拿到了10個淘汰，可以說狀態大好手感火熱，而iFTY依靠著房區所帶來的優勢不斷進行打靶同樣保持滿編。

由於OMG拿到的房區並不在安全區內，而iFTY的房區正好卡住了圈邊，OMG想進圈必定要跟iFTY打一仗。於是OMG選擇2人沖房，2人房區架槍線。BT率先擊倒了薄涼後，遠處FTG的槍線直接擊倒小獅子，人頭被龍四哥拿到，隨後小秋也被iFTY淘汰。此時BA順勢擊倒BT，OMG僅剩下小海一人，最終iFTY打出0換4並且成功以14殺吃雞。

在賽後採訪階段，老菊代表iFTY接受了主持人宅宅的採訪，宅宅也是特別指出了與OMG的那波團戰中，面對薄涼率先倒地隊內的溝通交流是如何的，而老菊也是直言「不要慌，打冷靜點，他們都被打殘了，他們都要打葯，他們很好接的。」

或許OMG的粉絲聽到後會有那麼一點不舒服，但王老菊說的也很真實，畢竟OMG在進入iFTY房區時是面對了FTG跟BA的槍線。

雖然說粉絲們之間頗有種水火不容的味道，但龍四哥在直播時表示其實雙方的關係都還不錯，也希望雙方的粉絲能夠早日化解恩怨吧。
