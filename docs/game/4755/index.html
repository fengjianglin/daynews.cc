提起英雄，就讓我想起了天下無敵的美猴王，而說到孫悟空，就讓我想起了西遊，今年年底，由中法團隊聯合打造的西遊單機手游《非常英雄》即將於大家見面，這部被提名動畫界奧斯卡Annie Awards最佳角色動畫的佳作，六學創始人並沒有在其中飾演孫悟空，而是純手繪風格哦！文體中外兩開花，希望大家前往TapTap多多關注！

【中法團隊 文體開花】

《非常英雄》是由中國製作人帶領法國製作團隊打造的西遊遊戲，所以在畫風上有著濃厚的西方特色，製作團隊希望在保留東方神話故事的神韻同時，加入了西方卡通手繪風格，使角色及場景更加親切與國際化。遊戲關卡根據BOSS和故事的變換，也會由明顯的視覺變化，從林地到荒漠，從地表到冥府，即便是沒有玩過《非常英雄》PC端的玩家，也會被其頗具特色的藝術風格所吸引。

【多重玩法 N卡合一】

眾所周知，《西遊記》中九九八十一難地理迥異，面對的苦難也大相徑庭，師徒四人所展現的本領也各不相同，為了充分還原這一特點，遊戲只能同時一個角色登場，通過切換不同的角色，利用角色的專屬技能來解決眼前的謎題。在這一核心玩法上，每個關卡的操作方式也發生變化，如在冥府中，玩家變成小時候的樣子，即為平台跳躍，找到靈魂後恢復原狀則變為彈幕射擊，從根本上改變玩家的遊玩模式，獲得多重遊戲體驗。

【白骨精開美容院 牛魔王開澡堂】

師徒四人雖然是原著中的絕對主角，但每個反派也是刻畫的重點，製作團隊在深讀《西遊記》後，為每個登場BOSS都做了詳細的背景設定，和技能機制，以牛魔王為例，它雖然是法力無邊的龐然大物，但內心卻很脆弱，由於被人說臟，變成了潔癖，把火焰山改成了洗浴中心，每天岩漿泡澡…荒誕不羈的同時，仔細想想又覺得合情合理。

“西遊”作為中國傳統文化的一部分，能夠通過《非常英雄》手游這種遊戲題材加以傳播，是完美世界遊戲的願景之一，未來，將會有更多優秀作品與玩家見面，敬請期待吧！
