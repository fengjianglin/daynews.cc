DOTA2：獅城小Major前瞻 茶隊又要一輪遊了？LGD VG同組內戰

天天要聞
2019-12-16

最近並沒有什麼DPC官方賽事，但是依然有精彩的比賽，12月17日就要進行的新加坡世界賽因為參賽隊伍質量很高被稱作是獅城小Major，中國戰隊LGD，VG和茶隊應邀參賽。最近賽事舉辦方也公布了比賽的具體分組。茶隊被分到了A組，而LGD和VG則被分到了B組，本文素材來源競技寶，感謝jjbone的數據支持。

先說說A組的吧，和茶隊同組的還有A隊，J.Storm，秘密，TNC和VP。感覺這幾支隊伍除了J.Storm其他都是難啃的硬骨頭，A隊在成都Major敗者組淘汰了茶隊，TNC作為上次Major的冠軍實力也不差，唯一可能出現點問題的是VP和秘密。

從之前萊比錫Major俄區的預選賽來看，雖然俄區整體水平下降，但是VP還是那個CIS一哥，秘密呢則是隊內四號位Yapzor家裡有事，Midone臨時替補，又可以看到快樂男孩了，不知道快樂男孩的四號位打的怎麼樣。還想說一句關於茶隊的，之前茶隊剛打完Major預選賽，戰神七就在微博暴躁回懟噴子，之後WESG就被IG2：0無情碾壓了，這次也重要有個中外對抗的機會了，粉絲也不用找借口說什麼國內比賽練陣容了，終於可以看看戰神七的BP到底怎麼樣了。

之後說說B組，除了LGD和VG以外，B組還有EG，紅星老師，Navi以及液體。

這幾支隊伍感覺並沒有實力特別出眾的隊伍，要說現在狀態的話可能液體和EG的狀態還算不錯，可能會給LGD和VG造成一定的威脅，新Navi也可能在這次的比賽中讓人眼前一亮，9Pasha的加盟以及隊內的一號位水晶哥的穩定發揮還是讓Navi強了一個檔次。但是B組的整體水平和A組相比還是差了不少。

這次比賽的小組賽全部都是BO2，給大家預告一下12月17日的比賽吧，上午十點茶隊就要對上之前成都Major的仇人A隊，之後茶隊也要分別對上VP和TNC，感覺茶隊盡量在VP身上拿個2：0吧，之後TNC和A隊1：1握手言和就皆大歡喜了。

之後下午3點B組的VG率先對陣EG，同一時間LGD對上Navi。晚上8點LGD再打EG，而VG則要碰到液體。感覺EG小組賽第一天蠻慘的，要連續碰到兩個實力比較強的對手，這次可以看666扛著隊伍走，Abed和RTZ一起劃的劇情了。
