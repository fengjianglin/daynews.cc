王者新賽季上分推薦，打野露娜連招玩法教學

天天要聞
7 天 ago

嗨小夥伴們，又見面了，不知道大家認為在王者裡面操作難度以及操作上限最高的英雄是誰呢？是有三段位移的韓信？還是開大跳舞的貂蟬？又或者是台詞與操作一樣秀的曜？每個人都有不一樣的看法，提起秀的英雄，蘿伊首先想起的還是露娜，無論在哪個賽季哪個段位，只要召喚師水平足夠，露娜就是月下無限連的存在~

露娜出裝與銘文推薦

裝備：游擊彎刀，抵抗之靴，噬神之書，虛無法杖，極寒風暴，輝月。銘文:10紅月10心眼10狩獵

出門裝選擇狩獵寬刃，之後預購進化水晶，這個裝備對於露娜這種類型的打野的收益是十分大的，在前期增加了自身的藍量和血量，在血族之書沒做出來之前可以保證她的續航能力。但這件裝備後期較為乏力用處不大，所以當等級到達13或14級時可以捨棄，空出裝備格來做輝月提升自己的保命能力

而對於是出痛苦面具還是虛無法杖而言，我比較傾向於出虛無法杖，因為虛無法杖的法強收益和法穿收益是遠高於痛苦面具的，雖然痛苦面具會提供5％的冷卻縮減，但是當露娜出了極寒風暴再拿藍buff後就可以達到40％的最大冷卻了，所以個人認為出虛無法杖的收益是比出痛苦面具的收益高的。

在做出守護者之鎧後我們可以先把野刀升三級，而這個時候恰好是快要15級的時候，三級的野刀不僅會給我們增加一個強擊的被動，還會增加我們的藍量和法強，可以說是很契合露娜這個英雄的屬性了。

露娜技能加點推薦

主一副二有大點大。

露娜實戰思路與連招露娜在四級前沒有大招比較弱勢，所以在四級之前盡量別打團。由於露娜對藍buff的需求很大，所以打野路線是比較推薦從藍野區刷到紅野區，當一片野區刷完後，我們應觀察小地圖，如果對面下路壓的很深的話，我們可以嘗試著去抓一下，這個時候我們可以先a空氣兩下，然後使用一技能打出印記，接著使用3a2a3aa3的連招，如果對面還差一絲血就可以擊殺時我們可以選擇斷大擊殺，當然要在保證自己斷大後仍然可以安全的離開戰場，不然是不推薦斷大擊殺的，這套連招在前期便可以打出不俗的傷害，可以很容易的幫助線上建立起優勢。

一般情況，當我們抓完第一波人後大約是開局兩分鐘的時間，這個時候剛好是小龍刷出的時間，碰巧也是藍buff要失效的時間，小龍對前期節奏十分重要，所以如果隊友也有開龍的想法，我們可以選擇先開龍而放棄先打藍buff。

露娜在沒出血族之書前在小規模的團戰續航不足容易被擊殺，所以在沒出到血族之書前不推薦進場打持續輸出，應該是打一套就撤退，然後等待cd打下一套，所以這裡就要說到兩個露娜最基礎的連招。

連招一:13a2a3a3，這便是大家常說的月下無限連，這套技能連招的好處就是可以在短時間內打出足夠的爆發，而且還可以讓自己在打完一套技能後還有逃生的能力；

連招二:1323，在露娜釋放二技能的時候會有一個護盾，所以這套連招可以讓露娜使用最少的血量消耗來壓低對方血線，給隊友或者自己製造出擊殺的機會。
