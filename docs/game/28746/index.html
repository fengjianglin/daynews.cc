萌新入坑《風暴英雄》竟被“大富翁”吸引，原來活動獎勵是這麼多

2019-12-22 2,861條評論 4,460次閱讀 1622人點贊

作為一個玩了十年MOBA類遊戲的老玩家，沒有接觸過風暴英雄之前我還是很質疑風暴英雄的遊戲模式的。什麼？打不了錢？買不了裝備？那這遊戲怎麼玩？但是架不住坑爹損友軟摸硬泡，也是趁著暴雪冬幕節活動的機會，來體驗一把風暴英雄。

相比其他MOBA類遊戲，風暴英雄的主界面還是比較的簡潔明的。各種比賽模式可以說是一目了然，即便剛剛入坑的我也是毫不費勁的就可以打上一盤人機了解一下遊戲內容。只不過風暴英雄的英雄價格著實是不太便宜，聽說最近的更新還出了一個新英雄死亡之翼，進去一看我的天一萬五千金幣，我得攢到猴年馬月才能攢夠錢買這英雄。

不過好在可以提前進行英雄的試用，免得攢夠錢買了英雄以後覺得不喜歡玩，退群還沒地方退錢，白白浪費好不容易攢的金幣。在試用模式里我也是簡單的了解了這個英雄的一些技能，感覺這個英雄有點變態，沒法被控制不說傷害還高的不行，想必在實際作戰中會特別厲害。就是技能的釋放時間太長了，不知道能不能打到人。

這個特別活動可以說是我非常喜歡的環節，像極了小時候玩的大富翁，完成任務獲得硬幣，有了硬幣就可以搖色子讓小火車開動。每次都會獲得一個任務，完成了任務可以獲得硬幣讓小火車繼續行動。不過這個火車道上的牌子就比較討厭，我的色子搖到了六，本想著能一下開出好遠，可是一撞到牌子就停了下來。不過好在完成了撞到牌子以後的任務還能獲得獎勵，不得不說這個琪拉的頭像還是比較可愛的。

在拉我入坑的損友們的瘋狂Carry下我也是完成了不少任務，比較可惜的是完成任務以後獎勵的補給包並沒有開到想要的小火車頭。對於小夥伴們開著托馬斯小火車滿地圖的跑我也只能報以羨慕的眼神——直到他偷偷告訴我”兄弟，你點我一下就能上車。”哈？WTF？這遊戲還能這麼玩的？我抱著懷疑的態度點了一下他的小火車，沒想到還真的可以上車，心中頓時獲得了巨大滿足，感覺對面打我也不疼了，我還能再抗十分鐘。

而最值得慶幸的是，在這個冬幕節活動的第二天，可能是暴雪爸爸給我的入坑禮物，終於是人品爆發在玩具箱中獲得了心儀的小火車，這下可以自由自在的在風暴英雄里開車咯。
