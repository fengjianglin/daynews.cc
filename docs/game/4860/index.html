《木筏求生》又更新了·又打折了·新的內容能玩多久呢

2019-12-04 0條評論 19次閱讀 0人點贊

摘要

STEAM本次的瘋狂周三打折活動中，《木筏求生》隨著新的更新內容進行一波打折，原價68元現價48元。

STEAM本次的瘋狂周三打折活動中，《木筏求生》隨著新的更新內容進行一波打折，原價68元現價48元。

幾個月前的《木筏求生》遊戲內容感覺有些少，不太符合68元的價格，甚至有評論說遊戲內容只值38元，我與小夥伴在那個時候玩了一段時間的木筏，玩法還不錯，同樣是覺得內容太少，後期遇到的島嶼、動物有一種千篇一律的感覺，玩久了容易疲乏，歸根結底還是內容不夠豐富。

與鯊共舞

隨著近期《木筏求生》的幾次更新，雖然還沒有再一次開始玩，看到更新內容感覺還是不錯的，今天的這次更新應該是《木筏求生》第一章內容全部更新完成，才是第一章噢，製作組的大餅還是畫得挺大的，先看看今天更新的內容吧

1、增加2個新目的地，改造現有的無線電塔和小島。

2、增加新的歷史內容，個人理解為新的背景劇情故事。

3、增加新動物，熊和潛伏者。潛伏者有點像變異鼠。

4、新的道具物品。包括尋找島嶼的新接收器，還有方向盤、發動機、燃油箱、管道系統。對應發動機和燃油箱而出的生物煉油系統。

5、新的武器MACHETE彎刀，看上去就一把大砍刀。

6、一些新的植物，松果和樺樹。

7、新的照明系統。

8、取消大霧天氣，更改鯊魚活動模式，研究藍圖也有相應修改。

9、修復一大堆BUG。

休閑釣魚

對木筏求生這個遊戲來說，我是比較喜歡的，這次新的內容感覺至少可以玩三個周末，但這是一個多人遊戲，單人就太空虛和無聊了，建議能有一群小夥伴一起玩就樂趣多多。現在的遊戲內容越來越豐富，在大海中遊盪遇到的島嶼不在是簡單的幾種，隨機性越來越多，製作組還會持續更新，還能小小期待一下。

最後，號召一下小夥伴們，有想一起傲遊大海，闖蕩陌生島嶼、恐怖洞穴的可以找我約時間一起探險啦。
