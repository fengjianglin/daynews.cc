電競職業選手——對戰房的背後是一段行走於邊緣的人生

2019-12-23 3,032條評論 4,237次閱讀 1204人點贊

憑君莫話封侯事，一將功成萬骨枯。——《己亥歲》唐·曹松

2003年11月18日，國家體育總局正式批准將電子競技列為第99個正式體育競賽項；2008年，電子競技又被國家體育總局再次改批為第78號正式體育競賽項。

自電子競技逐漸走向公眾視野以來，一個又一個爆炸性新聞不斷刷新著電競圈內圈外對這個行業的認知——Wings戰隊在Dota2第六屆國際邀請賽上成功奪冠，斬獲9139002美元（約合61958000元人民幣）的總獎金創造了新的吉尼斯世界紀錄；LGD俱樂部DOTA2分部隊員Ame轉會費開出5000萬人民幣的天價；英雄聯盟RNG俱樂部隊員Uzi接受NIKE代言簽約，一年代言費用高達500萬美元……

這些被資本和規模撐起的數據，使「電競職業選手」成為一個看起來輕鬆簡單而又「來錢快」的職業，讓很多年輕有遊戲天賦的孩子對這一職業趨之若鶩。但實際上，這是一個比大多數人想像中殘酷得多的行業。

IG奪得英雄聯盟S8總冠軍

家庭社會環境阻力

「如果我成績不好了，我就去廚師培訓學校學習，然後當一個廚子」——《DOTA2》TI8、TI9冠軍OG隊員Ana對好友袒露心扉

電競選手從起步時就是阻力重重的，最重要的一點就是「電子競技選手」不是社會主流認知中的一種職業，而只是「打遊戲的」。就連《DOTA2》TI兩連冠，獲得近千萬美元獎金的OG隊員Ana也在和好友Forev講，「如果我成績不好了，我就去廚師培訓學校學習，然後當一個廚子」。

即使到如今，電競選手作為一種職業已經能逐漸被大眾所理解，它也絕不是父母願意孩子首選的職業道路，甚至年輕人自己也不會將電競職業選手作為自己未來工作的第一發展方向。縱觀國內成名選手，能達到大專及以上學歷的屈指可數——《英雄聯盟》RNG.Letme畢業於海口經濟學院；GT.Savoki畢業於上海師範大學；《DOTA2》IG.super畢業於西安交通大學；VG.Ori畢業於西南政法大學等等，更多的職業選手經歷則與Uzi、jackeylove類似，初高中畢業就進入了電競職業圈。這種電競選手職業現狀，不免讓社會公眾一提到「電競選手」就自動等同於「低學歷低素質人群」，即使一線電競選手的經濟收入已經高於大學教授很多，但相較於後者極難獲得社會認可。

RNG.MLXG接受採訪

RNG俱樂部英雄聯盟分部前打野選手mlxg曾在一次採訪當中明確表示「小時候被父母送進過戒網癮中心」。他在2016年英雄聯盟LPL春季賽獲得冠軍，2018年獲得MSI季中賽冠軍並且入選2018雅加達-巨港亞運會英雄聯盟電子體育表演項目中國代表隊，並隨隊奪得金牌。這樣熱愛電競並取得一定成功的職業選手在小時候尚且被父母強制送去「戒網癮」遑論其他只是喜歡玩遊戲的普通孩子。

從一個學習成績一般的遊戲愛好者成為一名職業電競選手，中間的路道阻且長。

職業圈結構畸形

「千萬別打假賽」——《刀塔之夜》newbee俱樂部教練banana

《DOTA2》第九屆國際邀請賽在上海舉辦，作為對緊張賽程的補充節目，《刀塔之夜》網路短綜藝同時開播。在節目最後一期的嘉賓寄語時，到場的Newbee俱樂部DOTA2分部教練banana猶豫之後說出他的寄語「希望年輕選手們千萬別打假賽」。

Newbee.banana對DOTA2職業新人寄語

電競職業選手是一個結構並不健康的行業。僅以《英雄聯盟》為例，活躍在LPL（英雄聯盟職業聯賽）的隊伍是16支，以每支隊伍平均6名職業選手的數量來計算，16隻隊伍共計約96名選手，能打進世界賽總決賽階段的只有其中三支隊伍，能奪得冠軍的只有其中一支，還需要分別擊敗來自歐洲賽區、東南亞賽區、北美賽區等等來自世界各地的強敵。這是一個非常困難的過程。而《英雄聯盟》在中國的活躍玩家總數超過4600萬人，作為職業選手能賺到錢有未來的人數為4600萬選96。

大量個人實力上等的遊戲玩家，要想在遊戲圈子掙一碗飯只能走向另外兩條路——遊戲主播和其他三線四線不知名小隊伍。遊戲主播流量呈現頭部聚集特點，只有有限幾個主播能獲得較大流量；混跡於三四線隊伍的選手則拿著每個月四五千元的工資，在大城市入不敷出，甚至參與打假賽賺錢，因此才出現上文中職業教練「別打假賽」的希冀。

從事職業電競，是一場鮮血淋漓的鬥爭，能站上金字塔尖吸金的只是少數人，更多人只能在邊緣蠅營狗苟。

職業發展閉環

「舞台下，也許更能看得清前方道路」——《英雄聯盟》EDG隊員clearlove微博

12月16日，英雄聯盟EDG俱樂部官方微博發布了陣容變動最新消息，原俱樂部內打野選手clearlove轉為隊伍教練，其本人也在微博發布短消息「舞台下，也許更能看得清前方道路」。又一名職業選手轉型成為了戰隊教練。

clearlove的退役轉型微博視頻截圖

電競選手是一個發展閉環職業，行業內絕大多數選手在退役後會轉型成為戰隊經理、領隊或教練，帶出來的選手再次循環，從而形成一個職業閉環，這也是限制電競職業化的重要原因之一。

回顧2003年電子競技成為正式體育競賽項目以來，能成功從這個怪圈突圍出去的職業選手屈指可數——人皇sky李曉峰算一個，退役之後出版了自傳，開了自己的電腦外設公司還參演了電影；伍聲2009算一個，依託淘寶店帶來的第一桶金，先後成立了多家公司，成為名副其實的「大酒神」。此外絕大多數職業選手退役後其中一部分只能去成為一名遊戲主播，比如PDD、茄子、YYF等，再或者成為一名遊戲比賽解說，更多的選手則從此默默無聞淡出玩家們的視野。

如今的伍聲2009已經是近20家公司的實際控制人

職業壽命短暫，入行早一般在16歲左右，到退役也不過26歲，其中更不乏大器晚成的職業選手，比如2019年英雄聯盟S9全球總決賽冠軍上單gimgoon，奪冠時已經24歲，職業生涯已經步入後期了，他本人也說過，如果沒拿到這個S9冠軍，可能就要選擇退役了；職業前景窄，遊戲主播、遊戲解說、戰隊教練，退役後屈指可數的職業發展選擇也令這份看似風光的職業產生了不少尷尬。

結語：看得到的風光背後是看不到的艱辛

被玩家們稱為DOTA創世神的2009曾在一次採訪中說過一句話被電競職業圈奉為圭臬，「沒成績前，世界不會在乎你的自尊」，這大概是職業選手最殘酷而真實的寫照。第一代電競選手已經先後退役，他們走過的是最坎坷的路，DOTA2選手sansheng當時家人不支持他打電競，因此他背著一捲鋪蓋離家開始在電競職業路上闖蕩，最困難的時候窘迫到身上只剩幾十塊，為了能繼續把職業打下去，他只能在吃住在網吧，撿別人的煙頭抽。後來他在Newbee奪得TI4冠軍獲得幾百萬人民幣獎金之後，還會常常和圈內好友談到那些往事，並感嘆現在環境好了，至少看打遊戲不是「洪水猛獸」了。但事實上電競職業選手要擺脫現在的職業邊緣化地位仍然任重而道遠。
