LR請另謀高就，魔獸懷舊服更新後STSM存在脫戰BUG，官方制裁刷G？

天天要聞
2019-12-12

【遊戲沒有圈兒】《魔獸世界》懷舊服迎來階段性更新，對於那些有背包壓力的玩家而言，「鑰匙鏈」的推出的時間恰到好處。提到本次更新，其實主要為第三階段PVP內容：戰歌峽谷與奧特蘭克山谷。眾所周知，魔獸懷舊服分為6階段，而在短短時間將階段推進至「3」，一方面是為魔獸正式服9.0版本讓行，另一方面則是緩解野外PVP的壓力。

目前魔獸懷舊服的玩家均可在主城（戰爭大廳、勇者大廳）進行戰場排隊，好戰者不僅可以沉浸於此，還能在不同聲望等級時收穫珍貴道具與裝備。值得一提的是，本次開放的戰場確實是「跨服戰場」，陣營一邊倒的區服玩家暗自欣喜了吧？也不要高興得太早，被老區「國家隊」碾壓也會時有發生。目前，戰場房間爆滿，野人玩家（單人匹配）極大可能會匹配到小隊戰場中。而匹配時間需要6-8分鐘左右，所以這裡更推薦兩個戰場一起匹配，擇優進入。

當然，不要忘記在主城領取相關任務：

「遊戲更新後出BUG」，這在遊戲行業中或許早已是司空見慣。但今日的魔獸懷舊服卻讓部分玩家頭疼不已，確切一點是LR（獵人）這個職業。是怎麼回事呢？在STSM（斯坦索姆副本）刷G玩法中，「右邊的巷子」（前後門正常）會觸發脫戰BUG。那麼以LR寶寶拉怪為主的隊伍中，其職業地位似乎受到了不同程度的影響，如果證實此次確實是更新導致的瑕疵，那麼等待修復就好。但反之，官方是出於整治刷G為目的，那麼這個所謂「BUG」就要常駐了。LR另謀高就？

各大社交論壇，包括官方論壇都在抱怨：

還有一個小細節，據玩家反映，目前更新後的魔獸懷舊服在幀數方面有所下降，對於視覺享受要求較高的玩家而言，它也是BUG般的存在（「畫質黨」可以測試一下）。最後，隨著戰場的逐步開放，PVE玩家又將迎來曙光，練小號、野外觀景的安全係數一定有所提升吧？
