CFS2019世界總決賽，主旋律中巴大戰，全程“未爆冷”的一屆？

天天要聞
2019-12-11

經過四天的激烈爭奪，四支拿到靜安體育中心場館賽的隊伍出爐，他們是中國的AG和SV，巴西的VG和BD，半決賽的對陣，兩場中巴大戰，AG打VG，SV打BD。

回看這幾天的比賽，雖然有很多的插曲，但是從結局來看，其實也沒啥爆冷之說，可能有的玩家會說，白鯊不是被淘汰了么？這還不算爆冷？

我們來回顧一下本次世界總決賽的隊伍構成，CFPL賽區三支，巴西CFEL賽區三支，歐洲兩支，北美兩支，埃及一支，土耳其一支，為什麼給巴西和中國三支，無疑是因為這兩個賽區的隊伍數量多，隊伍質量高，CFS世界賽想要好的聲譽，自然要保證比賽質量，小組賽打完之後，中國三支隊伍晉級了，巴西三支隊伍也晉級了，歐洲隊伍進了一支，另外埃及的隊伍也晉級了（他們同組的土耳其隊伍沒打，所以小分積10分也算個優勢），從八強結果來看，賽前被認為實力強的隊伍都晉級了。

我們再來看一下四強，兩支中國隊伍，兩支巴西隊伍，巴西的IMP被淘汰了，中國的白鯊被淘汰了，我們來看一下他們參加賽區預選時候的成績，IMP是巴西三號種子，白鯊是中國的三號種子，留下的隊伍，都是各自賽區的一二號種子，這不正說明沒有爆冷么？

從八強到四強，大的方向上的確沒有爆冷，留下最終的半決賽均為中巴大戰，說比賽沒有爆冷，也是在肯定VG比白鯊出色的地方，輸給VG之後，解說白鯊如此評價那場八進四：確實打不過，都研究透徹了，知道別人咋打咋守，還是干不過！白鯊不止一名隊員手感不在狀態的確是輸掉比賽的原因，但是，我們不得不承認，人家VG確實是打地好；況且，在小組賽的時候，VG還10:5贏過SV呢。

在今年的世界賽開賽前，官方公布了賽制更改的一些細節，比如八進四從BO1改成了BO3，比如比賽地圖移除沙丘增加衛星基地，這樣的改變，我們在比賽中也是注意到了隊伍做出的應對或者是說賽制對於比賽的影響，八進四如果不是BO1，那麼巴西IMP就帶走AG了，如果不是打衛星，那麼國外戰隊就不會必Ban一張衛星基地的圖。

大的範圍內，CFS2019世界賽沒有了「爆冷」，說明比賽越來越健康有序了；而小範圍的「爆冷」無疑也為比賽增添了很多精彩，總而言之，CFS世界賽在整個CF賽事體系中的分量越來越重了。
