Dota2新加坡邀請賽VG奪冠

在本次Dota2新加坡邀請賽上，VG完成「一串五」成功奪冠。

VG在小組賽僅列第三位，以敗者組身份進入淘汰賽階段，從敗者組首輪開始分別以2-1 TNC，2-1 Secret，2-0 Alliance，2-0 Gambit，和最終的3-0 EG連續擊敗五支世界強隊奪得了本次ONE Esports新加坡邀請賽的冠軍！

決賽中連下對手兩城後，手握三個賽點的VG在第三局拿出了船長和小魚雙核，同時給皮隊掏出了全能騎士；而EG這邊在生死邊緣選出了謎團和虛空雙團控，同時還拿出了Cr1t的絕活小小，準備進行殊死一搏。

最後一波開5戰雖然EG集虛空和謎團雙大秒掉了小魚人，但是買活的小魚人瞬間讓EG感到絕望，買活趕過來瞬間拿下三殺，直接澆滅了EG想要奪冠的心！

賽後rOtK在接受採訪時直言：」CN Dota Never Die！「

鹹魚點評：原來這才是今天大家吃得最開心的一頓餃子。

LOL德瑪西亞杯第二日EDG晉級

今天是《英雄聯盟》2019德瑪西亞杯第二日。

EDG先是在第一場比賽中擊敗GD，隨後在與LGD的比賽中，EDG這邊鱷魚無人可擋，雙C瘋狂輸出，高地擊殺LGD多人後贏下比賽，取得了兩連勝。

在今晚的比賽中，EDG擊敗TES成功晉級淘汰賽階段。

LGD今日在分別輸給EDG和TES之後與GD碰面，LGD中路強開，全員果斷突進團滅GD，取得了德杯首勝。

隨後LGD又擊敗了V5斬獲連勝。在其他比賽中，今日此前兩勝一負的BLG碰上一勝兩負的IG，IG最終擊敗BLG。

鹹魚點評：EDG花的這50萬愣是打出了500萬的效果。

《絕地求生》每周封禁10萬賬戶

儘管《絕地求生》早已沒有當初那樣風光，但依然還有擁有著很大一部分玩家群體。據《絕地求生》社區經理Hawkinz在Reddit上的回復，他們每周都會封禁10萬左右的賬戶。

在12月8日到14日這一周，《絕地求生》共封禁了116531個賬戶。雖然硬體封禁已經被採用，但是在很多情況下還並不能這麼做。

在2018年，《絕地求生》每周封禁9萬個賬戶，現在看來，外掛情況反而更嚴重了一些。

鹹魚點評：光封禁的用戶每周都有這麼多，可想而知《絕地求生》的銷量到底有多麼恐怖了。

遊戲推薦

今天推薦的是FPS遊戲：《自由人：游擊戰爭》。

《自由人：游擊戰爭》是一款集戰略戰術，養成於一體的射擊遊戲。開局一把小手槍和幾個民兵，你將擔起領導的重擔，做生意賺錢，供養軍隊，剿滅四處遊盪的土匪，挑起戰爭並利用優秀的戰術和精湛的槍法殲滅敵對勢力並統治世界。

遊戲的操作有兩個層面:世界地圖模式以及第一人稱射擊戰鬥部分。在地圖模式中，玩家可以訪問任何地點、進行交易、升級裝備、攻擊敵軍、圍攻城鎮、接受任務等等。而在FPS戰鬥部分，玩家則會以第一人稱視角進入戰場，作為步兵戰鬥並利用戰術地圖實時地向軍隊下達命令。

目前聖誕特惠八折優惠，售價64元。
