還記得《魔獸世界》被我們救出的這位守護者嗎？已經進本了

2019-12-11 324條評論 1,477次閱讀 512人點贊

在魔獸8.3中，又一個我們曾經的老朋友萊登陷入了古神的腐化。副本中的萊現在被稱作虛無者，他的命運已經無法扭轉，這位曾經的至高守護者，如今正致力於實現古神恩佐斯的黑暗計劃，除此之外再無其他。

而萊經由泰坦鑄造的鋼鐵身軀也被腐蝕，畸形的觸手代替了他的右手，腐化尖刺與虛空能量在身上泛濫，究竟是什麼樣的事故導致至高守護者落入這樣的結局呢？

萊又被稱作萊登，在泰坦造物的古語中是”萊主上”的意思。這是泰坦造物們對這位曾經至高守護者的尊稱。

作為萬神殿泰坦創造的守護者，萊由泰坦之父阿曼蘇爾親自塑造，因此其面容也仿照至高泰坦本人進行雕琢。正因如此，萊在艾澤拉斯的眾多守護者中顯得格外出眾，並被賦予了”至高守護者”之名。

如果你在魔獸世界之餘，還是一位爐石傳說的卡牌愛好者，那麼或許在近期的卡牌中見過這位守護者萊。他被設計成一張無法通過正常途徑納入收藏的卡牌，只有當你在牌桌上達成非常苛刻的條件，以七名魔古教徒作為代價，才能將萊的本尊召喚出來。

而作為至高守護者，爐石傳說也給了萊非常誇張的特效，他可以對所有敵方單位造成20點的傷害。要知道在爐石傳說中，對戰玩家間的生命值通常才不過30點。

與自己奧杜爾的同僚們不同，萊從遠古時期開始就一直生活在世界的南方，負責看護這裡的起源熔爐和其他泰坦設施。然而當黑暗泰坦薩格拉斯覆滅萬神殿時，泰坦諸神的殘存靈魂跨越宇宙奔向了自己造物的體內。

只有萊從這些殘破靈魂中看清了泰坦覆滅的幻象，造物主的死亡摧垮了萊的意志，他將自己封閉在泰坦遺迹中，對周遭的一切都視而不見。

世事變遷，泰坦造物的魔古一族因為失去守護者的領導而陷入內亂。一名叫做雷神的魔古前往群山中質問萊。在看到萊落魄的樣子後，雷神奪取了萊體內的泰坦之力，成了魔古歷史上名噪一時的始皇帝。

雖然雷神已經被我們在熊貓人版本中擊敗，萊也被我們從密室中解救出來。但這位失神的守護者並沒有重新振作。在我們冒險的數個版本中，萊的內心一直充斥著空虛與無助。

當8.3版本反抗恩佐斯的戰役正式打響時，玩家曾一度尋求萊的幫助，但最終這位守護者沒能抵抗住古神的腐化，並在故事線的結尾墮入黑暗，成為8.3副本的第九號BOSS。曾經的守護者最終站在了玩家的對立面上，這個結局雖然不算太過意外，但也足以令人唏噓。

除了萊登以外，我們還將在8.3面對虛空龍母、蟲人先知等等各式各樣的敵人，而最終玩家將進入尼奧羅薩的中心，一勞永逸地消滅恩佐斯，並未魔獸的古神線畫上一個圓滿的句點。
