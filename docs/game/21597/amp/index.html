世界唯一的輔助FMVP，MATA的傳奇還在繼續

天天要聞
2019-12-19

謹以此文，獻給世界唯一的輔助FMVP

RNG官博於12月16日晚間官宣：LOL歷史上唯一一名用輔助位置獲得FMVP的選手Mata，會回到RNG。不過這一次是以教練的身份。歷史上也不是沒有輔助選手擔當教練的先例，比如說以前在SKT和豬讓哥哥（piglet）一起堪稱LCK賽區第一下路的福滿多（Poohmandu）。成績也是相當喜人，側面也映證了輔助的遊戲理解對整個團隊的幫助是巨大的。

可福滿多畢竟只是福滿多，而說到輔助害得看我們M姓來整活。

如果說Madlife讓廣大玩家知道了M姓輔助可以秀的眼花繚亂，那麼直到我看到了Mata我那才意識到——M姓輔助原來還可以是FMVP。

我叫Mata

在轉會期的最後一天爆出這樣的消息，說真的出乎意料之外但在情理之中。從2012年加入Ozone到如今Mata已經在職業賽場鏖戰了8個年頭。而Ozone（SSO）也就是後來我們所熟知的SSW。

僅在出道的第三年，爭奪第二章世界賽門票的時候就遇上了去年S3總決賽的冠軍SKT，OGN總積分同分加賽以3:0乾淨利落的帶走對手拿到去往S4的第二張門票。

我還記得那一年沒有SKT的S賽，在皇族進入決賽的那一刻，所有人都覺得那是最有希望的一年，可對手的強大好像讓夢破碎的那麼的輕而易舉。

從那時起民間有了「星不散，永奪冠」的傳言，也是從那時起所有人都知道了有一位輔助他叫Mata。

S4總決賽冠軍2014年英雄聯盟全球總決賽FMVP

轉會VG

那些年電子競技還屬於新興產業，即使是在電競產業鏈完整的韓國，在天朝天價的挖人合同面前，單靠夢想也是無法留住那些有夢想的年輕人的。

畢竟，人總是要吃飯的，何況電競還是碗青春飯。

於是Mata帶著Dandy來到了LPL的VG，星不散的美夢終究還是散了。

在VG的時光好像並不是那麼美好，那時VG的比賽幾乎把把前期都是優勢大的沒邊，可到最後總是不能順利結束比賽，這似乎和後期決定遊戲走向的ADC無法撇開關係。

隊伍成績的不佳，平日里積攢的不滿，這位曾經正面擊潰無數世界頂尖選手的心態終究還是發生了一點小小的變化。

年少輕狂

先有Mata後有天，單手傑斯弄瓦西里

誰又沒有年少輕狂（心態爆炸）過呢，哪怕他是一座輔助玩家心目中的神像。

「只有經歷地獄般的磨練才能練出創造天堂的力量，只有流過血的手指才能彈出世間的絕唱——泰戈爾」

轉會RNG

在VG積分不足無緣S之後，Mata這曾經的FMVP終是不甘寂寞，在大家都以為他會回LCK的時候，宣布加入RNG，勢必要重回S賽奪回屬於自己的榮耀。

Mata與UZI攜手橫掃了當時所有的LPL下路組合，一舉奪得最高的積分鎖下世界賽的門票。

最終不幸倒在了SKT成就三冠王的道路上，成為了背景板。那一年小狗哭了，可作為狗妃Mata又何嘗不難受呢。

轉會KT

Mata離開RNG去到KT之後，幾乎把把MVP，帶領著隊伍也是再一次一路挺進S賽8強。

在8強遇上IG的他，可以說是提前上演了決賽戲碼。

那場比賽到目前為止也是在我個人心目中僅次於ROX vs SKT的那場BO5。阿水那閃現上去拿命換來的輸出，終究是讓Mata的傳奇落下了帷幕。在此之前幾乎沒有人看好小IG，而在那場BO5之後再也沒有人質疑IG，這也從側面反映出了那年KT的強大。那年Mata雖然倒在了8強，可大家都看得出來他依然還是當年那位意氣風發的FMVP。

轉會SKT

Mata的心性在這8年的漫長旅途中已經磨鍊得堅不可摧，如果說一開始的Mata只能被稱作最強大腦的話，那麼現在的Mata已經可以被稱得上獨當一面的領軍人物。

SKT有了Mata的加入之後好像底氣十足，即使在MSI之前成績一直是飄忽不定。但在MSI回國之後，由於有著兩名定海神針（Mata、Faker）的存在，隊伍內心態調整相當快速，直接開始以連勝的姿態拿到一張世界賽門票。

Mata最後一次以選手的身份再次來到了世界賽的舞台。

SKT在半決賽對上G2 1 ：2 落後的情況下，臨時決定換上Mata。Mata臨危受命，對方高地泰坦大招沒收Wunder奧恩二段R，可惜faker好像真的被人們過度的神化了。Mata也無力挽救過於重大的失誤。

好像就連鏡頭也忽視了Mata，可那又怎樣呢？

戰士就應該在戰場上倒下，正面被對手擊潰是一種榮耀。

職業生涯的最後一場比賽，由自己親手來畫上句號，還有什麼遺憾呢。

直播太累

眾所周知，Mata是擁有中日韓三國直播間的男人。

三個直播間播著確實太累了，還是去當教練吧。讓我們笑著說再見！

我是教練

Mata自己也是在採訪中表示自己很懷念RNG。只是這次回來不再身披戰袍而是穿上一身帥氣西裝，頭髮並未梳成大人模樣，也許這就是那位往昔的王者少年所不變的堅持吧。

「你以為挑起生活的擔子是勇氣，其實去過自己真正想要的生活才更需要勇氣。——薩姆門德斯」

既然Mata也許已經找到了自己想要的生活並且以教練的姿態重生，那麼我們也就只需要祝福。

結語

Mata雖有FMVP光環加身，可其本人卻可以說是一個羞澀靦腆的代名詞。衷心祝願一代傳奇人物能夠繼續在賽場上，以教練的身份再續傳奇。想想在不遠的將來我們又可以看到如此溫馨的畫面，淚拉了出來。
