冬冠揭幕戰，XQ10-0零封EMC，四核體系很亮眼，橘子未能首發

天天要聞
2019-12-22

KPL秋季賽已經圓滿落幕，冬冠杯的比賽也正式拉開帷幕，預選賽的比賽是緊張刺激的BO1，每一場都不容有失。因為本次冬冠杯是在中國澳門舉行的，所以揭幕戰也是由澳門的本土隊伍EMC率先登場，對手是青春風暴的XQ。EMC這邊有我們比較熟悉的選手orange，但是這場比賽並沒有登場，XQ的陣容則是和秋季賽相比沒有任何變化。

XQ在紅色方一上來就放出了公孫離，EMC也是直接笑納。而XQ則是選擇了野核娜可露露，雙邊搭配上馬超和狂鐵，九尾也拿到了招牌英雄娜可露露。前期雙方在暴君處開啟團戰，EMC先手開暴君，但是張飛沒到四級先手被秒，孫策和隊友脫節也同樣被秒。XQ也是一路順風順水，先秒掉公孫離之後直接拿下先知主宰。十一分鐘的時候aolin的鎧在中路試圖蹲娜可露露，但是經濟相差太大了，砍了三刀依然沒有殺掉娜可露露，等到XQ隊友支援到位之後直接秒掉了鎧。而EMC剩下的人沒有選擇賣鎧，一擁而上卻潰不成軍，只剩下一個周瑜已經完全沒有還手之力。XQ順勢一波拿下比賽，獲得本次預選賽的第一個小分。

這局比賽真的是完全的實力碾壓，XQ這邊的野核陣容在第一波團戰打出優勢之後，EMC基本上就一直處於逆風狀態了。XQ打得也是一點都不急躁，就是慢慢滾雪球，推塔拿龍入侵野區，如果是BO7的比賽的話或許XQ會浪一波，但是在BO1的比賽中是一點都不容有失，打得非常穩。

當然如果說最後一波EMC不出去打的話可能還能再拖一拖，但是也只是早輸和晚輸的問題了。XQ這一局打得很像是一個四核體系，可以看到除了輔助位之外其餘的四個人吃的經濟都差不多，綁兔稍微高一點也是最後一波團戰打完才補起來的。季後賽被RW零封之後，XQ一直都沒有放鬆訓練，也是鉚足了勁想要在冬冠上有比較好的發揮，目前看來確實是很有效果。而EMC這邊和KPL隊伍的實力差距還有點大，打野位的TIME似乎和隊友的磨合還不太到位，而橘子也沒能登場，希望在之後的比賽中能夠有機會吧。

大家對於這局比賽有何看法呢？歡迎在評論區留言交流。我是天黑君，期待下次再見！
