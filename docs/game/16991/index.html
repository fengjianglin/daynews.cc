Dota2：LGD奪得WESG冠軍，選手BP真的做的比教練好嗎

2019-12-14 928條評論 4,111次閱讀 2310人點贊

昨日，WESG大中華區Dota2項目已經圓滿結束，LGD先後擊敗EHOME.i和IG奪得了冠軍和20萬的獎金，最近LGD的比賽BP權交還給了選手，目前是Ame在負責BP。在Ti8、Ti9的時候，LGD的BP是被噴的非常慘的，那麼今天就讓我們一起來看一下AME的BP真的會比教練來BP做的更好嗎？

來看一下LGD奪冠這四局的BP

首先是LGD對陣EHOME.I

然後是LGD對陣IG的BP

有水友總結了雙方的BP時的內心想法：

對方視角：ban了劇毒蝙蝠？這是要玩SV嗎？ban了全能？一定是SV！等等，不對！有可能是猛獁。這手先出一個輔助看看情況。選這麼快？體系？新套路？怎麼ban小黑？醬油是甩技能的脆皮？還是要出火貓這類的打帶球…這AME，BP這麼強嗎？居然看不透。

Ame視角：先把我怕的3都關了，再把對面厲害的關幾個，查豬來個團隊裝工具人。fy玩個能幫我留人的。偉哥來個保得住我的。最後一手出maybe，不被針對Maybe隨便打。好了！還剩1min40s備用時間，搓搓手。

在賽後接受採訪時，LGD眾人也一致表達了對AME的BP的誇讚之情

從這幾局的BP我們可以看出來，AME的BP思路是非常的明確的：保證自己和Maybe的優勢對線，給查理斯選一個開團的工具人，給fy選擇Gank能力強、秀的空間大的英雄，同時給xNova選擇一個保人能力強的五號位。選手BP大概率不會像教練BP那些想太多，只需要按照自己的思路來就可以了，當然這也是建立在LGD五個人強大的個人實力基礎上的。

但是這樣的BP同時也會帶來一定的問題，那就是如果對手刻意留手大招勾引你上套是比較容易中陷阱的。AME的BP如果想真正的能跟世界強隊掰手腕，還是需要修鍊修鍊的，這時候就需要「數據分析師」這樣的角色了。

自從LGD集合以來打的幾個比賽成績和效果都不太理想，甚至在Major海選上輸給了FTD的青訓隊FTD.A。這次WESG中國區比賽，能擊敗上屆Major的季軍IG，而且是兩把碾壓，我們熟悉的LGD終於要回來了！大巴黎，咚咚咚！

以上就是本期的全部內容了，對於Ame的BP你有什麼想法呢？歡迎留言討論。
