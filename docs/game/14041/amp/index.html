藍洞公布PUBG賽事計劃 引入全新的系列賽模式！4AM將直通柏林賽

天天要聞
7 天 ago

12月10日晚，絕地求生賽事官博發布了一則名為「2020年PUBG電競賽事規劃」公告，公告中描述了藍洞官方在明年的賽事計劃。同時官方還稱，明年將引入全新的「絕地求生全球系列賽」以豐富明年的賽事體系。

和今年不同的是，明年官方決定在PGC之前再設立三個世界賽規模的比賽，這個比賽被分稱為PGS，這個PGS全年共舉辦三場——4月份場、7月份場以及10月場。4月份的PGS是在柏林舉辦，今年PGC前四的戰隊將直接被邀請，也就是說PCL賽區的4AM戰隊將直接有資格去參加四月份的PGS。除此之外從官方公布的規則來看，三次晉級PGS的名額將從PCL聯賽中產生，而PGS也在一定程度上決定了PGC的名額。賽程方面，這三場PGS的賽程將會和本次PGC看齊。

可以看出來，自從官方開始規範各大賽區的聯賽後，賽事體系也在一步一步的完善中。如果明年PGS以及PGC的賽事質量能夠像今年的PGC一樣，那麼這幾大賽事還是很令人期待的。
