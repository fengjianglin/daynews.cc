脆皮剋星鎧皇，12個射手8個怕他，這3位卻能秀哭他

天天要聞
2019-12-14

王者榮耀裡面有不少英雄都深受玩家們的喜愛，尤其是那些長得帥還實力強勁的角色，更是屢屢登頂各種榜單。要說王者榮耀里單打獨鬥實力最強的英雄，自然是繞不開鎧皇的，畢竟這個英雄被大家叫做「站擼王」。作為王者榮耀脆皮剋星的他，12個射手8個不敢與之硬剛，然而卻有3位ADC能秀哭他，這是怎麼一回事呢？

首先我們說說鎧皇，這個英雄有控制有輸出還能扛傷害，在遊戲里是個非常好用的戰士英雄。尤其是鎧皇開大的時候，戰鬥力簡直是爆表，所以有的玩家配合閃現打出「一刀斬」，往往能夠秒了對面的脆皮。在峽谷的戰鬥里，很多射手英雄雖然也很厲害，但是面對鎧皇還真不一定打的動，有可能還會被其反制，所以鎧皇也有「射手剋星」的外號。那麼到底是哪些ADC，能夠秀翻這位「站擼王」呢？

第一位是虞姬，這個英雄在峽谷里不僅僅是鎧皇打不動的射手，就連很多刺客也拿她沒辦法。玩過這個英雄的都知道，虞姬開了二技能，是可以免疫物理傷害的，這就讓鎧皇的攻擊沒了效果。更不用說虞姬還有加速技能，可以一路「放風箏」打鎧皇。所以說鎧皇遇到虞姬，其實是不願意和她浪費時間的。

第二位是公孫離，說到峽谷里最飄逸的射手，那阿離自然榜上有名。公孫離的位移技能效果很不錯，只要預判到位手速過人，就能拖著敵人打。面對這樣的敵人，鎧皇即便想減對方的速都不容易，更不用說開大變身了，那樣鎧皇速度會更慢，這麼一來更難以打中公孫離，所以阿離也是不怕鎧皇的。

第三位是李元芳，這個射手不僅走發育路是一把好手，打野也能讓對手望其項背。李元芳自身也是帶位移的，同時飛鏢還能打出延遲傷害，這在戰鬥中都是相當實用的。尤其是李元芳的大招，可以減速敵人，在這個過程中他用連招連續攻擊鎧皇，很可能就將其一波帶走了。就算一時間打不贏，李元芳也能趁機逃脫，所以鎧皇見了李元芳輕易不願意招惹他。
