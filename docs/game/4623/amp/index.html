「地下城與勇士之小狐狸君愛談遊戲電競」第六百九十四期《DNF國服最「強」寵物+稱號：加61個技能，屬性介紹2頁，卻沒人要》，近日國服更新了一個活動，裡邊有時裝、寵物、稱號……等道具。其中這個寵物和稱號，非常的犀利，直接增加61個技能！可以說非常的強勢，不過玩家看了介紹之後表示，這寵物不過是史派克的「面子工程」罷了！對玩家的提升幾乎可以忽略不計！因為僅僅是1級主動BUFF。屬強、傷害、四維、三攻……完全沒有提升。這個寵物稱號，看屬性描述佔了3頁，可是全部是沒用的！史派克特色真是要逼退玩家。老規矩搞快點，直接上本次活動寵物和稱號的截圖展示：

DNF國服最「強」寵物+稱號：

首先看一下這個最「強」稱號。61個職業主動BUFF技能+1。所以有61個技能，足足佔據了3頁的介紹頁面！但是點亮的BUFF只有一個！為什麼會這樣呢？很明顯策劃懶得做稱號，每個職業一個要做61個，索性做成一個，把所有的技能都塞進去，雖然對每個職業有效的只有1個，但是這樣就不用真對每個職業做了。以不變應萬變！難怪國服的BUG那麼多，一個稱號的數據都這麼臃腫，更不用說副本和職業了。

提升61個技能的等級：

窺一斑而見全豹，通過這個稱號的設定，我們看到了史派克策劃對於遊戲的優化，是多麼的粗糙！一個稱號或者寵物，屬性描述中有61個技能，但是對於每一個角色而言，其中60個全部是沒用的，等於說99%的是無用數據，只有1%才對角色有效。DNF客戶端10G，但是僅僅是一個二維的刷圖遊戲。其中的無用數據可以說佔據了80%。DNF初期僅僅是500M的遊戲，發展到現在遊戲越來越卡了！

屬性介紹佔據3頁，卻沒人要：

之前國服出國類似的稱號，但是設定是每個職業一個稱號。就是鎮國大將軍稱號。老玩家都知道，哪個就是當時最長的稱號了。但是長江後浪推前浪，這一次除了一個介紹更長的稱號，61個技能、3個頁面，但是卻沒人真的想要！
