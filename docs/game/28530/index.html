陰陽師百聞牌：狗吞桃陣容實屬厲害，可通關秘聞之間第三章

2019-12-23 661條評論 2,618次閱讀 1001人點贊

大家好，歡迎小夥伴們的到來！分享遊戲里有趣好玩新鮮事，實時把遊戲心得分享給大家，之前說通關秘聞之間第一章可以獲得犬神SSR覺醒牌，其實是小編說錯了，原來只是運氣好，通關後隨機得到的，並不是通關百分百SSR卡牌。秘聞之間就是主線劇情，可以了解《陰陽師：百聞牌》遊戲的構造和故事，不過很多玩家都在吐槽第三章的大天狗很難打，本期就來說第三章如何通關，話不多說，讓我們來看一下吧~

​狗吞桃陣容實屬厲害，可通關秘聞之間第三章！

同樣也是零氪玩家沒有強勢的卡牌，我選用了狗吞桃陣容對付大天狗，但大天狗作為牌手有50點生命，需要速戰速決，山童和鳳凰火帶有真傷，被動貫穿傷害波及牌手，拖到後面己方牌手都被耗死了，我也是打了3局才險勝通關的，拿到一張座敷童子的家裡安全卡牌，忽然有點小失落，因為有玩家通關拿到SSR卡牌，心想著自己也有機會解鎖一張，看到獎勵終於知道卡牌是隨機給的。

快攻陣容的思路打不過，前期力量沒疊起來式神傷害不高，50點生命戰鬥牌打完大天狗只是殘血，別看敵方只有2個式神，有牌手被動效果越戰越勇，復活一次力量和生命都增加，且大天狗是很強的機器人操作。

​需要用好己方牌手的被動，3張被動牌分別針對戰鬥、法術、形態，當限制戰鬥山童就消停3回合，因為山童使用戰鬥牌無效還扣2生命，牌手大天狗很聰明，會用鳳凰火攻擊或發動法術牌，己方牌手被動針對讓對方只能用1個式神，如果擊敗山童氣絕，限制法術鳳凰火也就老實了，她只會拖到戰鬥區進行普攻。相反山童在場限制戰鬥牌，山童的攻擊沒有疊加效果，就製造不出高傷害。

除了靈活運用己方牌手被動，還需要正確的式神，可以挑高爆發和肉厚的式神，抗揍的才能站場，氣絕了可碰不到大天狗。小編選用了狗吞桃，犬神前期肉厚有鬼火就增益疊力量，後期拿到心技一體形態牌，瞬間力量翻倍使出惡即斬傷害可觀。酒吞童子主要是炮灰，有不屈鎖血保命，三勾覺醒開始爆發輸出。陣容選了妖刀姬可以無視，覺得用判官也可以，有核心牌都可以上，關鍵是零氪，只好用狗吞桃陣容，終於在第3次挑戰成功。

狗吞桃實屬厲害，看到不少玩家吐槽說要削弱該組合，匹配天梯總是遇見，怎麼說呢，狗吞桃是新手玩家上手最快最容易的，卡組要求不高但很強，天梯五段五階打來打去，還是覺得狗吞桃好用，如果削弱的話，玩家要重新搭配陣容，用別的思路去打，其實卡牌那麼多有不少克制的。

玩家都喜歡說一句話：打不過就加入吧！本期內容到這裡就結束了，有什麼看法可以留言回復，到評論區暢所欲言吧，感謝閱讀，我們下期再見！
