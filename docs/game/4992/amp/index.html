文丨浪的像鬼@bigfun社區

《大王不高興》的主角閻小羅在跟隨孫悟空學會七十二變後，這一神技幫助她度過了不少難關。而在遊戲中，七十二變則會讓閻小羅變身成其他勢力中的重要角色，以應對遊戲中的不同挑戰。閻小羅可變身的角色分別是西方地獄的路西法，東方天庭的楊戩，西方天堂的加百列。

路西法在主角達到10級後自動解鎖，楊戩和加百列則分別要通過第六章「結緣古神」與第八章「逃出西方地獄」的主線劇情，這兩章還分別需要主角達到20級和30級。

這4名主角只有等級共享（另外三個都是閻小羅變的，等級相同也是無可厚非的），各自的神器等級則需要分別去培養，而ta們的基礎屬性與各自技能的專精方向也是完全不同的。

主角不能和其他英雄那樣穿戴裝備，於是神器就承擔了裝備的職責，神器滿級為30級，升級神器也會提升生命、攻擊與防禦。每次升級神器都會獲得一定的八卦石，用來升級和解鎖神器的天賦樹，每位主角都擁有8個技能，除去自帶的普攻、初始主動與被動技能外，剩下4個主動技能與1個被動技能均是由神器天賦樹來解鎖的。

而神器的天賦樹只需點亮。

本文會為大家介紹本人已經解鎖的2名主角的屬性、技能、神器以及各自所擅長的領域。

閻小羅

神器等級1，未開光

真·主角，閻小羅本身的攻擊和防禦在4名主角里中規中矩，技能以我方buff與AOE技能為主。

技能：

普攻技能：對敵人進行5次射擊，每次造成19%/20%/21%/23%/25%/28%傷害。

快快護駕：使我方一名隊員行動條增加20%/22%/24%/26%/28%/30%，消耗2個魂火。

閻王駕到（被動技能）：自身攻擊暴擊時，會提高自身5%攻擊力，可疊加10層。

東方快車（神器解鎖）：對全體敵人進行6次攻擊，每次造成34%/35/37%/41%/44%/49%傷害，消耗4個魂火，2回合CD。

開光（神器解鎖）：使一名隊友無視敵人20%/25%/30%/40%/50%防禦，持續2回合，消耗2個魂火。

洪荒之力（神器解鎖）：增加我方全體10%/12%/15%/20%/25%的攻擊力，消耗2個魂火。

生死簿（神器解鎖）：對1名敵人造成180%/189%/198%/216%/234%傷害，並驅散目標2個BUFF。

老君改造（神器解鎖&被動技能）：普攻變為群體攻擊，但傷害變為原來的80%。

路西法

路西法是閻小羅可變身的第一個角色，傷害能力略強於閻小羅，但也比閻小羅脆一些，技能以BUFF與DEBUFF技能為主。

技能：

普攻技能：對一名敵人造成6次鞭打，每次造成17%/18%/19%/21%/23%/25%傷害。

原罪爆發：對敵方全體進行6次攻擊，每次造成25%/26%/28%/30%/33%/36%傷害，根據場上所有的DEBUFF效果數量造成額外傷害，每有一個效果造成5%額外傷害，最多20個。消耗4個魂火，CD3回合。

魔神護盾（被動技能）：路西法受到敵人攻擊時，產生持續一回合的護盾，多可吸收最大生命值10%的傷害。

執念傀儡（神器解鎖）：使一名隊友復活2回合，並回復20%/30%/50%的生命。消耗3個魂火，CD6回合。

卑劣契約（神器解鎖）：對一名敵人造成100%的傷害，並且偷取敵人10%/15%/20%攻擊力加給自己，持續兩回合。消耗2個魂火。

地獄領域（神器解鎖）：當路西法生命值低於50%的時候，會觸發一個保護友方的結界，該結界可以增加友方全體10%防禦力。

靈魂惡咒（神器解鎖）：單體造成200%傷害，同時驅散敵人身上所有的增益效果。

絕望封印（神器解鎖）：單體造成120%傷害，並擊退目標25%行動條。

以上就是兩大主角角色的全部技能介紹了，可見兩個角色都有其擅長的領域，這就需要搭配不同陣容來進行使用才能發揮主角最大的作用。而且主角出戰時只能攜帶3個技能，第一個技能也就是普攻是不能被更換的，也就是說不管是被動技能還是主動技能就只能帶兩個技能出戰，所以這就牽扯到一個神器天賦樹的點亮取捨問題，畢竟點亮天賦樹是需要消耗大量八卦石的。

下面老鬼我就逐一對主角技能的取捨和主角與陣容搭配這兩個方面的個人看法分享給大家。（PVE向）

閻小羅：

技能方面推薦攜帶在神器天賦樹中點出的被動技能老君改造和主動技能開光。

陣容搭配方面推薦搭配擁有單體強力輸出人物的陣容。

舉例分析：閻小羅搭配UR德古拉+SSR瑪門+SSR范無救+SSR利維坦。閻小羅通過被動技能老君改造為隊伍提供無需消耗法力點的AOE傷害，主動技能開光無視防禦效果50%這個BUFF搭配瑪門的強力單體輸出在BOSS戰中能夠打出相當可觀的傷害。

路西法：

技能方面推薦原罪爆發和神器天賦樹中點出的執念傀儡。

陣容搭配方面推薦搭配擁有能夠提供多個減益效果的隊伍。

舉例分析：SSR阿斯蒙蒂斯+SSR瑪門+SSR利維坦+UR撒旦。路西法的主動技能原罪爆發能夠根據場上所有的減益效果數量增加額外傷害，從而為隊伍提供高額的AOE傷害；執念傀儡能夠讓1名隊友暫時性復活，復活的隊友在復活後第2回合結束時死亡。（遊戲中一場完整的戰鬥一個也沒有幾個回合，所以這個技能完全可以視為復活隊友。）

結語：

以上就是兩大主角角色的全部介紹了，兩大主角角色擁有各自擅長的領域，望諸位玩家根據自己的遊戲路線及英雄陣容搭配使用主角角色。
