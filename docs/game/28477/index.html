這款武俠網遊號稱，越南版“傳奇”？佔領越南80%遊戲市場

2019-12-23 2,171條評論 4,593次閱讀 1683人點贊

編者語：如今國內遊戲市場已經非常繁榮，以騰訊為首的遊戲巨頭，讓國內遊戲市場成為世界規模最大的遊戲市場。國內早期的互聯網企業，如騰訊、網易、新浪都是靠遊戲才渡過了公司發展最艱難的時期。從這方面說，國內遊戲市場幾乎拯救了中國的互聯網企業。

網易靠著夢幻西遊、大話遊戲成功轉型，騰訊靠著QQ遊戲系列才真正在國內真正站穩腳跟。金山軟體便是因為劍俠情緣系列才存活至今。而金山系列的劍網情緣網遊，不但在國內贏得了許多玩家的喜愛，在國外也得到了玩家的歡迎。其中越南便是劍俠情緣最成功的的國家，越南的遊戲市場幾乎便是在劍俠情緣的火爆下，才慢慢形成的。

劍俠情緣1在巔峰時期，佔領了越南80%的遊戲市場，而且成功讓其代理的遊戲公司VinaGame成為越南最大遊戲公司。而劍俠情緣也是越南市場最早的以時間收費進行收費的網路遊戲。用一句話總結，劍俠情緣1在越南的地位，如同熱血傳奇在國內玩家心中的地位，被稱為越南版「傳奇」。

劍俠情緣網路版作為金山試水網路遊戲的第一款網遊，在遊戲上線後便迅速躥紅。成為當時市面上最為火爆的武俠網遊。武林盟主和將軍裝扮的求伯君和雷軍，廣發英雄帖為劍俠情緣宣傳，不知道雷軍現在看到當時的照片有何感想。

因為有著單機遊戲成績的積累，劍俠情緣網路版成功也是必然。遊戲在網路化的同時，保留了許多單機版的經典對話場景和玩法。相比現在的逆水寒、天涯明月刀等武俠網遊，感覺最大的不同，便是如今的武俠網遊似乎缺少了靈性。

當初的劍俠情緣網路版的遊戲世界，給玩家的感覺是有血有肉的。以中國南宋為遊戲背景，讓玩家在遊戲中似乎能感受到當時的國讎家恨。雖然遊戲畫面早已經不能和如今的武俠網遊相比，但卻讓玩家更有代入感。

劍俠情緣最經典的宋金戰場便是依託遊戲歷史改編而來。但是最火爆時期，玩家為了體驗宋金戰場，甚至需要用按鍵精靈掛機，才能擠進去。

劍俠情緣網路版現在回頭來看，在當時水平也不是頂尖。但卻成功地讓西山居汲取到了做網遊的經驗。之後的劍俠情緣2成為西山居在武俠網遊的巔峰之作，在越南也至今也擁有大量老玩家。

整個劍俠情緣系列集大成的毫無疑問是劍俠世界，但最值得紀念的卻是劍俠情緣。更多的玩家喜歡劍俠情緣2更出色的畫面，真人捕捉的動作表現、豐富的裝備搭配。但胖大海卻認為，這一切都是在劍俠情緣1之上升華而來。

劍俠情緣1給玩家是一個有血有肉的遊戲世界，而劍俠情緣2已經慢慢將遊戲背景弱化，一片歌舞昇平，缺少了真實感。

總結：最好的武俠網遊到底是哪款，每個玩家都有不同的答案。但胖大海相信，其中定有劍俠情緣1的位置。畢竟當初為了賺點遊戲幣，冒死衝勁高級地圖撿別人不要的裝備，換成遊戲比的日子還歷歷在目。是的，只要自己喜歡，那便是最好的遊戲！

歡迎關注遊戲資訊大玩家，和胖大海一起玩最好玩的遊戲，聽最好聽的八卦！
