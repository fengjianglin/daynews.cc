絕地求生新機制提前顯示縮圈位置 內圈蹲點lyb會受傷

2019-12-09 0條評論 16次閱讀 0人點贊

摘要

《絕地求生大逃殺》中有“縮毒”機制，迫使玩家不斷移動到一個變小的安全圈內。不過有的玩家會藏在安全圈內某個角落，專門暗殺路過的其他人，甚至整局遊戲都不換位置，俗稱“lyb”。而現在《絕地求生》在測試新的機制，試圖整治這種行為。

《絕地求生大逃殺》中有「縮毒」機制，迫使玩家不斷移動到一個變小的安全圈內。不過有的玩家會藏在安全圈內某個角落，專門暗殺路過的其他人，甚至整局遊戲都不換位置，俗稱「lyb」。而現在《絕地求生》在測試新的機制，試圖整治這種行為。

這種新機制名為「藍洞模式」，它會在地圖上提前顯示下一次縮圈大小和位置，而且這個藍圈內的玩家蹲點守人會受到傷害。官方稱，之所以添加這個模式是因為：1，以前玩家不知道下個圈在哪兒，碰巧在圈內的玩家獲勝幾率略高影響平衡；2，由於縮圈位置未知，有的玩家可能全程主要是在跑圈，最大威脅反而不是來自其他玩家。

因此，新的機制會從每局開始就顯示下一個圈的位置和大小，直到倒數第二階段；而且如果有玩家提前待在內圈蹲點守人，他們會持續受到傷害。（不知道是圈內所有人都受傷，還是原地不動才受傷）

看來這一新的機制會取消遊戲縮圈的隨機性，並能迫使部分「lyb」玩家更多地移動。不過目前這一機制僅在吃雞實驗室中啟用，如果效果足夠好才會被加入公測伺服器中。大家對這一新的內圈機制有何看法？
