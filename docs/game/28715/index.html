笑笑直播談及阿水轉會，親手示範轉位置的後果，把水友都逗樂了

2019-12-22 439條評論 2,199次閱讀 172人點贊

雖然LPL轉會期已經結束，但阿水的去向還沒有著落，很多人都對這位冠軍AD的去向表現出很高的關注。最近有流言說阿水會去RNG打AD，也讓很多玩家覺得不可思議。最近笑笑直播的時候就談到了這件事情，結果後來直播的亮點卻被笑笑的中單英雄搶走了，這究竟是怎麼回事呢？

網上有很多關於阿水要轉會至rng的消息，笑笑在直播的時候就表示這是絕對不可能的，自己在網上居然還聽到了小虎轉上路，Uzi轉中路，阿水打下路的人員變動配置情況，認為這實在太離譜了，即使是Faker這樣的高人氣選手，一支戰隊都很難為這樣一位選手同時改變兩位選手的位置。笑笑在這之後也沒有說太多，讓網友們耐心等待消息。

笑笑表示：一個打了多年職業的選手，想換位置基本上是很難的，有趣的是，笑笑當場示範了一下，點了一把中單猴子，說要告訴水友們換位置的後果。眾所周知，笑笑平時主要打輔助位，雖然輔助並不是那麼厲害，但起碼下路還能混得下去。而猴子中單這種騷套路，笑笑之前應該是從來沒打過，有很多剛剛進入直播間的路人進來後都蒙了，還以為笑笑在看其他主播的直播。

這局遊戲笑笑用猴子中單卻打出了意想不到的效果，開局幾分鐘就拿到了對線的人頭，之後螳螂又把賈克斯的閃現逼了出來，笑笑瞬間用猴子技能跟上，戰績變成了1-0-1，西卡在旁邊都直言這把笑笑站起來了。在下路的一波團戰中，對面有三個殘血，差一點笑笑就把這幾個人頭全部拿下，之後無奈的吐槽自己的下路雙人組，認為他們實在太慫了，如果射手能夠跟上一點輸出的話，對線基本上笑笑就無敵了。

雖然說下路的那波團戰死亡斷了笑笑一點節奏，但好在沒有太大影響，後期孫哥的猴子依舊C的飛起，這波操作也是逗樂了不少水友，有很多粉絲甚至覺得如果讓笑笑打中單的話，傷害可能都要比小虎高出許多，打輔助真是埋沒了孫哥的天賦。當然，這些都是玩笑話了，笑笑這把的段位比較低，所以轉了位置也沒有很大影響，而職業賽場上高強度的競爭，英雄的熟練度就成了很重要的標準了，大家覺得呢？
