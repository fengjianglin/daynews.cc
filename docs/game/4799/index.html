外媒VG247評選 10年來最佳遊戲TOP50（上）篇

2019-12-04 0條評論 16次閱讀 0人點贊

回想一下2010年以來大家自己的狀況，你的生活改變了多少？ 也許你有了另一份工作，住在不同的房子里，或者有了不同的朋友圈。遊戲行業亦是如此——在過去的十年里，遊戲市場已經發生了巨大的變化。近日，外媒VG247就公布了10年來最佳的50款遊戲，本期給大家介紹第50-26名。

50、《群星》

《群星》集合了很受歡迎的兩種遊戲類型：科幻和大型策略。 但是卻比其他P社遊戲更好搞懂，雖話說如此，但《群星》仍然是一款複雜的遊戲，玩家需要經營一個帝國，探索銀河系，無論你是在打造政府，開採資源，還是建造一個戴森球體，《群星》的微觀管理有時會讓人感到無法抗拒。

49、《城市天際線》

也許有人並不認為《城市天際線》是一款戰略遊戲——它更像是一款冷戰模擬遊戲。 建立你的道路網路，鋪設基礎設施，建造你的建築，然後看著夜幕降臨，以及高速公路上的車燈閃爍，整個世界從你身邊路過。

48、《彈丸論破：希望的學園和絕望的高中生》

一部分是視覺小說，一部分是偵探模擬小遊戲，《彈丸論破：希望的學園和絕望的高中生》也許是最好的一種沒道理的遊戲。 陷入瘋狂的黑白熊的魔掌之中，希望之峰的「終極」學生可以隨時離開——只要他們能夠殺死自己的同學並逃脫懲罰。

47、《精靈寶可夢Let’s Go》

《精靈寶可夢Let’s Go》的故事和《精靈寶可夢：黃》一樣，除了這次你可以選擇皮卡丘和伊布之外，還可以在電視上進行遊玩！ 這是一個把《精靈寶可夢Let’s Go》絕妙的樂趣介紹家中男女老少的和偉大的方式。

46、《超級馬里奧：奧德賽》

《超級馬里奧：奧德賽》這款核心系列的馬里奧遊戲不錯，這也許不應該讓人覺得出乎意料。當然，《超級馬里奧：奧德賽》的表現也說明了任天堂在整個系列中保持的質量，但是《超級馬里奧：奧德賽》最令人印象深刻的是它是如何成為《超級馬里奧銀河》和《超級馬里奧64》的優秀的繼任者。

45、《堡壘之夜》

不可否認，《堡壘之夜》對遊戲行業產生了巨大的影響。《堡壘之夜》起初是一款合作類塔防遊戲，但是慢慢就變成了一款，大家都知道的超級龐大的大逃殺遊戲。 《堡壘之夜》最初是對「吃雞」的一個模仿，現在變成了一個全球性的現象，為Epic遊戲商店的推出也提供了動力。

44、《熱血無賴》

從一個文化現象到一款遊戲，《熱血無賴》的影響力就像你驅之不去的夢魘。《熱血無賴》給人的感覺就像是一個跨越時間的遊戲。 作為一個款B級的開放世界作品，它的遊戲表現並不十分順利，預算也遠遠不及《GTA》之流，但它借鑒了《蝙蝠俠：阿卡姆》的戰鬥系統，並專註於打鬥，從而塑造了自己的形象。

43、《戰地：叛逆連隊2》

《戰地：叛逆連隊2》仍然是有史以來最好的多人遊戲射擊遊戲之一。 它沒有一個卧倒操作，在主機上它只能以30幀運行，但是遊戲一些出色的地圖和戰鬥機制都出現在了之後任何一款射擊遊戲中。《戰地：叛逆連隊2》的多人模式有一種自由感，這種自由感讓它感覺與其他遊戲完全不同，而且這種自由感也讓本作有了更多再次遊玩的動力。

42、《文明5》

《文明5》是小編玩的第一款「文明」系列遊戲，一千多個小時過去了，小編還是樂此不疲。 每種文明都有自己的特點，獨特的武裝單位，以及能戲劇性地改變你的勝算的建築，而同時也取決於玩家所處的時代。

41、《FTL》

在《FTL》里，你將感受駕駛太空船拯救銀河系的氛圍。這項任務危機四伏，每次遭遇將向你呈現解決方案多種多樣的獨特挑戰。如果重型導彈彈幕破壞了你的護盾怎麼辦？是將所有能量輸送至引擎並嘗試逃離，還時給更多武器提供能量並消滅敵人，還是登上他們的飛船，直搗黃龍？在這款「roguelike太空船模擬」遊戲中，你將駕駛飛船，與船員一起在一個隨機生成的銀河系裡展開冒險，體驗光榮的勝利與痛苦的失敗。

40、《毀滅戰士》

當提到街機遊戲時，人們經常會想到——像《Thumper》、《俄羅斯方塊》和《超級六邊形》這樣的遊戲，讓你完全沉浸在遊戲中，幾乎不用思考就能移動，就像你是機器的一部分。 而《毀滅戰士》就是這樣一款將這種精神狀態完美呈現的第一人稱射擊遊戲。

39、《刺客信條：奧德賽》

有很長一段時間，「刺客信條」系列就像是流動的溪水，沁入到了玩家的心脾，但是有一段時間這個系列卻在玩法的革新上進退兩難，隨著《刺客信條：起源》找到了一個新的RPG遊戲風格發展方向。 新作《刺客信條：奧德賽》也接受了這鐘想法，登上了奧林匹斯山頂，在幾乎每一個方面都進行了擴展和改進，創造了一個層次更深的遊戲。

38、《逃生》

《失憶症：黑暗後裔》以令人毛骨悚然的恐怖氣氛重新點燃了遊戲中的恐怖元素。而《逃生》把精緻的手術台捆得緊緊的，只穿著一件在微風中拍打的醫院病號服，在血腥中把它砸成了一款恐怖大作。

37、《寶可夢GO》

十多年前，每個孩子都渴望得到一張閃亮的寶可夢卡片，現在，寶可夢再次回到了每個人的口袋裡——在手游《寶可夢GO》中，這個系列找到了一種誘人的新玩法。 老實說，我永遠不會忘記2016年的夏天，你去公園的時候，不可能不看到一群群老老少少，緊緊抓著手機，試圖去抓住遊戲中可愛的寶可夢。

36、《尼爾：機械紀元》

《尼爾：機械紀元》是橫尾太郎的白金工作室專業領域發生碰撞的產生的作品，結果非常出色。《尼爾：機械紀元》是一款ARPG超級大作，除非你至少玩了三遍，否則整個故事都不會很清晰，每一次後續的遊戲都提供了新的視角、新的故事時刻和新的遊戲機制。

35、《動物之森》

《動物之森》是只有任天堂才能製作出的系列遊戲。 雖然說也有類似的系列，但是《動物之森》的能量、風格和基調能讓玩家感到放鬆且沉迷其中。 理論上，和一隻會唱歌的海龜一起航行到一個小島上去摘水果，然後每天賣水果，這是一件單調繁忙的工作，而其確實如此——但《動物之森》的美妙之處在於，儘管如此單調你仍然想要這麼做。

34、《旺達與巨像》重製版

看看這一望無際的風吹草動的平原，凝視這些巨大的，雄偉的野獸和平地吃草，望望這夢幻般的，繪畫般的美妙風景。 《旺達與巨像》是一款簡單的遊戲，有著程式化的藝術，但它設法通過視覺和主題引發更多的情感。

33、《神界：原罪2》

通常這些經典的RPG遊戲都不是那麼容易上手，拉遠的鏡頭讓人很難去關注角色和遊戲世界，而你用滑鼠移動只會讓你覺得更加疏離。當大家在PC上玩《神界：原罪2》的時候，這些都不重要了（如果你喜歡的話，現在你也可以在主機上玩），因為這是一個非常有趣的遊戲，所有的煩惱都消失了。

32、《龍之信條》

《龍之信條》看起來就像是一部DVD電影，遊戲裡面的每個人看起來都像是黏土模型，但是如果你不去關注這種醜陋的外觀，你會發現它的創造力在ARPG遊戲中是無與倫比的。 首先，有典當系統，它可以讓你在不同的級別與NPC同伴並肩作戰，你也可以出租你的裝備，借用其他玩家的打造的東西，但是其中大部分裝備看起來相當嚇人。

31、《生化危機2：重製版》

《生化危機2：重製版》將在未來幾年裡被視為重製經典遊戲的權威範例。 它不是一個完全複製品，它包含著一些自由發揮，但它設法完美地捕捉到原始遊戲的精髓，同時仍然有自己的想法和驚喜。 最重要的是，這使得一個戴小帽子的大個子男人（暴君）看起來很可怕。

30、《極樂迪斯科》

《極樂迪斯科》是一款開放世界角色扮演遊戲。 你是一個擁有獨特技能系統的偵探，你可以用整個街區來開闢你的道路。 審問令人難忘的角色，破解謀殺案或者收受賄賂。 成為一個英雄，或者成為人類的絕對災難。這是Steam對迪斯科樂園的描述，而且這款獨立遊戲甚至評價比大多數遊戲都要好。

29、《最終幻想15》

不可否認，《最終幻想15》是小編玩過的唯一的《最終幻想》遊戲，但它仍然是我過去十年中最喜歡的遊戲之一。《最終幻想15》講述了一個完整的故事，四個好男兒開著自己的「蝙蝠車」四處亂跑，幫助當地人，晚上避開怪物。

28、《合金裝備5》

《合金裝備5》可以說是是有史以來最深入，最豐富的潛行遊戲之一。這是每個工具在遊戲中都有許多種用途。工具新用途的出現不是在指導玩家執行任務的固定套路，而是向玩家提出創新解決方案的挑戰。

27、《火炬之光2》

《火炬之光2》被認為是「暗黑」色彩更加豐富和輕鬆的「表親」，是一款非常好的ARPG遊戲。 本作與《暗黑破壞神》的比較是在意料之中的，尤其是考慮到遊戲的兩位聯合創始人Max Schaefer和Erich Schaefer曾經創造了最初的《暗黑破壞神》，並與他人共同創立了暴雪北。但是與《暗黑破壞神》的比較已經被比爛了，所以這裡不會再次比較。但《火炬之光2》獨樹一幟，值得一提。

26、《血源》

《血源》是這個問題的答案: 「什麼是最好的『黑魂』遊戲？ 」 （雖然說《黑魂3》有更高排名）《血源》有一個更快的戰鬥組合，專註於精確的打擊感，以及洛維克拉夫特 / 維多利亞時代的世界，滲出的氣氛和恐懼，在那裡不可知的怪物潛伏在我們的邊緣。
