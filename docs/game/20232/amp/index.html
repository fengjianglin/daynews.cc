Steam上這新游太坑人，幾張壁紙就賣錢，還嘲諷玩家沒人陪？

天天要聞
2019-12-17

隨著12月份的到來，跨年夜距離我們越來越近，遊戲日報檸檬怪發現在Steam平台，一款名為《跨年模擬器2020》的遊戲悄然上線。

根據遊戲介紹顯示，這是一款可以根據玩家所在地區的真實時間執行跨年倒計時的遊戲（程序），在倒計時結束前，你將沉浸在逼真地晝夜循環、天氣環境中。待到2020年1月1日零點整，你將看到一場新年煙火秀，與購買了這款遊戲的玩家一起度過這個難忘的一刻，雖然這款所謂的遊戲並不能聯機。

其實可以看出這款所謂的遊戲僅僅是通過幾張壁紙和音樂配合一個跨年倒計時而已。

《跨年模擬器2020》

遊戲將於12月21日解鎖，但值得注意的是，這款《跨年模擬器2020》只能夠在2020年1月1日之前購買，因為一旦錯過2020年1月1日零點，這款遊戲的倒計時將會消失。

《跨年模擬器2020》

《跨年模擬器2020》是一款為廣大孤獨症患者定製的遊戲（程序），屋裡（耳機/音響里）優美的鋼琴曲，窗外（屏幕里）緩緩落下的雪花，幫你進入完美的冥想狀態（發獃），使你感到極度舒適（孤獨）。

《跨年模擬器2020》

其實，這樣一款看似僅僅是一個根據電腦時間變換場景的遊戲（程序），卻能在如今這個年代觸動到不少人，因為在這個連空氣中都充滿了孤獨的年代，有無數人相聚在遊戲中跨年，搞不好這樣一款遊戲還能成為爆款。

當聖誕節來臨之際，親人、朋友都沒在身邊，至少還有遊戲陪你，真好（慘）。
