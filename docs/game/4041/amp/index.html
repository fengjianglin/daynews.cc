於是乎，各大MMORPG遊戲湧現出了一大批神仙捏臉玩家，有捏出獨一無二的自己的，也有捏超級偶像明星…….比比皆是，只要玩家們開發腦洞，就能解鎖各種自己想要的面部造型。

儘管大家都是往美里整，最好給自己整個天仙，又能看又能玩多爽的路線走，但是也有一群玩家不走尋常路，捏出了不少令人捧腹的大作，比如朱碧石了解一下？

還有《TERA》里的，試問：帥哥，你是怎麼混進高等精靈的?

瞧瞧下面這位，直接給球星拉塞爾·威斯布魯克換了女裝，這女裝給滿分！

不得不說這些惡搞大佬心靈手巧，塑造的面部造型惟妙惟肖，但是也有玩家手抖，一不小心變成了戰場上的殺馬特的，比如下面這位↓↓↓

沒錯，就是殺馬特，還是出現在具有波瀾壯闊的古戰場的《戰意》。樓主本來表示自己本來想整個鬍子頭髮全白的人物角色，儼然給人一種具有作戰實力雄厚的沙場老將的感覺，然鵝，現在頭髮是白了，鬍子倒還是一團黑糟糟。

更搞笑的是其他玩家們的一致評論，認為殺馬特在戰場實屬罕見，給瀰漫硝煙的戰場增添了一絲絲喜慶，戰場上肅穆的氛圍瞬間炸裂！不過也有玩家很是中意這款白色染髮劑，也想搞來試試，若是大家喜歡這款白色的，可以關注遊戲微信公眾號進行簽到，就能領取。

慶幸的是，一位玩家透露表示染胡劑、染眉毛劑後續會上線供玩家們自由調整面部造型，相信樓主看到這一消息，對於自己心心念念的沙場老將，指日可待！

另外，除了戰場上殺馬特的既視感外，還有玩家貢獻了自己使用了白色染髮劑後的低配版孫堅！我們都知道歷史人物的孫堅，作戰悍猛，常置生死於度外，如今《戰意》戰場上以孫堅造型出征，豈不是就能以臉擊退敵人就完事？

你以為《戰意》這款遊戲只有白色染髮劑，綠色的染髮劑值得你擁有！

不得不說《戰意》這款遊戲捏臉系統的自由度甚高，玩家們可以隨心調整發色外，還能對鬍子、眉毛、嘴唇、面部刺青等等面部小細節進行設計調整，可以滿足玩家們大膽的想像力。要知道網易遊戲的捏臉技術一流， 相信《戰意》也不會差到哪去，像沙場老將，又或是幹練帥氣的少將，信手沾來！

話說，在座的各位有沒有其他更有趣的捏臉，不妨分享一起樂一樂？
