作為一名遊戲運營人員告訴你遊戲內部的真相，比如抽卡概率造假？

天天要聞
2 週 ago

摘要

作為曾經的遊戲官方運營，經常會遇到玩家對自家遊戲的各種“反饋”和“建議。”而其中一些玩家的“反饋”，經常讓我感到尷尬——因為有些問題實在太難回答啦！或者說，有些問題實在太“可愛”啦！只是作為官方人員，我還是得非常禮貌、非常可愛地忽悠著回答（畢竟不能說真話嘛…）。

作為曾經的遊戲官方運營，經常會遇到玩家對自家遊戲的各種「反饋」和「建議。」
而其中一些玩家的「反饋」，經常讓我感到尷尬——因為有些問題實在太難回答啦！或者說，有些問題實在太「可愛」啦！只是作為官方人員，我還是得非常禮貌、非常可愛地忽悠著回答（畢竟不能說真話嘛…）。
現在，就以一名非官方人員的身份，來聊聊那些手游運營們不會告訴你的真相~！
….沒錯，我要開始噴人啦！因為最近暴躁老哥真的很多~！也歡迎大家討論哇~

1.」抽了100發都出不了UR，你們3%概率是不是暗改了？」
概率問題，是我在二次元產品中，接觸得最多最多的問題。而概率問題千千萬萬，最終都只會指向一個——我抽了XX次了都出不了金卡，你們概率造假！
首先！我要在此跟這類玩家說：
1）概率是不會跟著你抽多少次變的！就是你抽一萬一千次！概率都不會有任何改變！等於你擲骰子，你第一次擲到6的概率是1/6，第二次擲到6的概率還是1/6，第一萬次擲到6的概率還是1/6！所以如果UR概率是3%，那你抽多少次的概率都是3%！
2）因此根據抽取次數可以算的，應該是「連續X次都抽不到UR」的概率，但算這個是毫無意義的，這隻能檢驗你有多非。
3）3%的概率其實是非常非常低的，有多低？你想想玩飛行棋連續兩回合擲到6的難度，大概就是那麼難。所以抽不到UR才是大概率事件。
4）好了，所以我們承認我們暗改了。因為UR實在太不好抽，而眾所周知大家都特別喜歡UR，如果氪金了許久還抽不到UR豈不是馬上勸退？於是手游一般會加入各種保底機制，例如在X抽後必出UR，例如新手池概率等等，這也是為啥很多玩家會產生「」抽了N次概率會上升」的錯覺，那都是因為我們改的啊親！
綜上所述！別再說遊戲改概率了！是概率本來就很低好嗎！
（當然不排除有些噁心廠商會給氪佬暗改，但一般對新玩家是絕對不會改的，因為要留住）

2.「你們遊戲氪金禮包一大堆，真是***會不會運營遊戲」
對於這類玩家，我必須科普四個概念：
A. 手游是一件商品
B.你不是唯一的用戶
C.手游工作人員大部分是打工的
D.手游工作人員大部分是打工的，是莫得感情的務工機器
好了接下來我說的話可能會讓大家很傷心，很氣憤，覺得咱們遊戲運營都是唯利是圖，不愛遊戲的***。
不過，我們就是！大家必須知道，跟你們對線的各位手游運營/策劃們，都是有KPI壓著的阿！公司不賺錢就要被炒了！ 說句不好聽的！雖然大部分運營策劃們還是很喜歡遊戲，很喜歡玩家！可是還是難免想著要壓榨大家來完成業績的呀！（過於真實！
好了，現在一個個來說。
1）手游是一件商品——首先大家必須認清，手游是一件賺錢的商品，其首要屬性，就是賺錢。無論我們禮包多不多，都是要賺錢的。如果你覺得禮包不多，那是因為它用別的方式來騙氪。所以氪金禮包多不多，主要取決於遊戲機制設計，而不是運營。例如FGO，它沒有禮包吧？但你能說它不氪金嗎？例如王者榮耀，它沒有禮包吧？但它不氪金么？當然，你可能會說，那是因為這兩個遊戲本身優秀，而不是氪金禮包問題。完全沒錯，所以這就涉及到第二個話題了。
2）你不是唯一的玩家——對阿，所以那些本身遊戲機制設計，在付費模型上沒那麼優秀的遊戲，就只能用禮包來刺激氪金了，不然咋賺錢。而且有一個事實是，可能你會覺得這些禮包很撒幣，但其實人家營收可是非常好的，甚至還會有很多大R玩家反饋，禮包出得不夠爽，不夠多。所以，無論是藍月傳奇還是FGO，都是從設計開始就決定了氪金機制、以及目標用戶，如果你覺得人家禮包多、氪金，那隻能說你可能根本就不是這款產品的目標用戶。（你甚至只是為了提供給氪佬娛樂的工具人）
3）手游工作人員大部分是打工的——當然，不能不承認，雖然主要是遊戲設計問題，但也會出現很多過度消費玩家的情況。為啥捏？因為咱們都是打工的鴨！對於大部分工作人員來說，用心創造快樂是其次的，能夠保住飯碗才是最重要的！So…有時候KPI壓下來也是莫得辦法的…你以為我們不知道多撒幣嗎!
4.）當然，剛剛只是吐槽，其實大部分加入遊戲行業的人，還是對遊戲有很大的熱愛的。不過阿，這裡必須重申一下，商業化手游真的就是為了騙氪的商品，而不能不承認，非商業化的手游在手游里佔比很小，至少目前國內情況是如此的。對比之下，以氪金情況來算，主機的體驗會好很多，一次買斷，終身無憂，鹹魚換帶，10元暢玩。（不引戰，不接受對線，純個人感覺）

3.「你們氪佬強這麼多，想錢想瘋了？」
如上所述——是啊。
我們靠氪佬給我們發工資阿，如果氪佬不強，遊戲早倒閉了，你連玩都沒得玩呢大哥。
可能有人說，你看方舟、FGO這些氪佬就不強…那是因為人家沒有出PVP阿…
大部分PVP的網遊，本質就是氪佬碾壓，除非是像MOBA、CS：GO這種競技PVP網遊。因為大部分PVP遊戲就是靠給氪佬賣碾壓感為賣點的。所以這就回到上面的問題，這是遊戲機制決定的鴨！
你不爽，就去玩PVE類的…或者就單機類的…要不就氪大金…

4.「一開服就炸服，就不知道做預防嗎？」
首先必須聲明，【炸服】的命題不是為了辯解，【炸服】絕壁是遊戲商的鍋，而且這個鍋對於遊戲商的打擊是非常大的，是遊戲商非常不想見到的，嚴重的甚至會導致整個項目不能回本，特別是目前炸服一星遍地走的情況下。
所以，沒人想【炸服】。那為啥還會炸服呢？
在此挑幾種典型言論來聊聊，為啥遊戲公司防不住【炸服】。
1）.「就不懂買多幾個伺服器嗎？」——伺服器很貴的，要控制成本呀，誰知道來多少人，特別是在刷初始號非常猛的年代。
2）「不懂通過預約數來預估嗎？」——造成炸服的原因很多種，不過大家理解的多人擠爆主要是算在線人數哦…所以這個數很難預估。
3）.「就不懂先測試下嗎？」——測試跟實際運行是有很多距離的，廠商只能給伺服器做壓力測試，但實際情況是很難預料的，會產生很多想不到的事情。另外，在之前的遊戲性測試中，由於樣本有限，很多問題是暴露不出來的。
4）.」又搞炸服營銷，以為我們真就你一個遊戲？「——炸服對遊戲商造成的損失是很巨大的，除了評分等表面原因，主要是很多預約玩家都會在上線首日湧入，炸服所造成的流失，是很沉重的打擊，所以不會有廠商蠢到搞什麼炸服營銷，除非公司有二五仔。
說這麼多，只是真誠地說一句，希望大家對炸服多點寬容，炸服了，休息一下，等補償失憶。對於評分還是根據遊戲內容客觀評論更好，如果是辣雞，再1星差評防止其他玩家踩坑，這才是1星的意義呀！~
畢竟大家也希望好遊戲，不要因為遊戲商的愚蠢炸服意外，就因此關閉了吧~？

5「我玩了遊戲XX年了，這遊戲就應該這樣做balabala」
警告！這一Part我要開啟鍵盤噴人模式了！玻璃心人員請自行退散！
經常在平台收到這種留言，我是最怕這種留言的，因為很多人就真的是不懂遊戲阿！誰說玩遊戲多久知道怎麼做遊戲的…我還坐了10多年公交車呢，我也不會製造公交車呢不是？在此呼籲有此言論的玩家們，請尊重遊戲，尊重遊戲研發，如果您真的覺得自己會做遊戲，請先去學習最基礎的遊戲研發知識。

當然，所有遊戲研發都是歡迎玩家提意見的，但應該是以「玩家」身份、或者「朋友」身份來交流——你可以是作為「遊戲產品的消費者」提出異議，可以是作為」喜歡這個遊戲的朋友」來提出意見，但請千萬，不要趾高氣揚地以「遊戲製作人」的身份來批評別人不懂做遊戲，這讓人感到非常的不尊重，特別是發言還特淺顯的情況下。
其中，我特別記得某玩家的一句言論——
「我如果去遊戲行業，一定是大佬，不過我太懶了，不想做。」
這位玩家，請問哪個行業的大佬是不用勤奮的？發夢大佬嗎？
有夢想，就去做！想製作遊戲，就請先學習遊戲！虛心+付出，才能改變世界！不是嗎？

完。
