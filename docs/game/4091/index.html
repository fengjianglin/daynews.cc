《暗黑破壞神4》發售日之謎：暴雪高層輪番爆料，莫非是好事臨近

2019-12-10 0條評論 17次閱讀 0人點贊

摘要

在今年的嘉年華現場，暴雪正式公布了壓箱大作——《暗黑破壞神4》，儘管玩家看完預告片之後都非常高興，但有經驗的人都知道：按照往常的慣例，從遊戲亮相到正式上市，三年起步、五年不晚，否則豈能配得上『暴雪出品，必屬精品』的名頭。

在今年的嘉年華現場，暴雪正式公布了壓箱大作——《暗黑破壞神4》，儘管玩家看完預告片之後都非常高興，但有經驗的人都知道：按照往常的慣例，從遊戲亮相到正式上市，三年起步、五年不晚，否則豈能配得上『暴雪出品，必屬精品』的名頭。

最近一段時間，暴雪門下的多位高管人員，從遊戲總監到藝術總監，紛紛出來為《暗黑破壞神4》站台，向玩家闡述本作的設計理念。根據游大師的再三研究，不但從中發掘出遊戲的開發進展，同時還隱約揭開不少謎題的答案，下面就和大家分享一番。

在預告片登場的惡魔莉莉絲，給玩家留下深刻的印象，為了營造片中驚悚、駭人的黑暗氛圍，設計師們打造了一場神秘的召喚儀式，結果證明非常有效，讓《暗黑破壞神4》又回到老玩家熟悉的邪典風格。引起玩家注意的，還有一位主持獻祭儀式的神秘男子，其真實身份一直飽受爭議。

根據暴雪官方媒體站點的資料，玩家可以從中挖掘出一個極為關鍵的消息：引導盜墓賊們送死的人，是最後一位初代『奈非天』——拉斯瑪。拉斯瑪的本名是利納瑞安，其父是創造天使伊納瑞斯，其母則是莉莉絲。

在官方小說《原罪之戰》中，拉斯瑪在巨龍塔格奧的教導下，掌握了生與死的平衡力量，從而成為史上第一位死靈法師。為了保衛庇護之地，拉斯瑪不惜與天使和惡魔兩派勢力為敵，並在大戰結束後不知所蹤。

耐人尋味的是，拉斯瑪與莉莉絲的關係並不好，偏偏暴雪安排他來召喚，恐怕背後另有所圖，絕不僅是為《暗黑破壞神4》安排一個BOSS那麼簡單。

除此之外，暴雪還針對裝備詞綴、開放世界等核心玩法進行了解釋，其中最讓人在意，還是PVP玩法的設計。如果沒猜錯的話，暴雪打算在《暗黑破壞神4》將PVP作為重點玩法，讓競技之風席捲庇護之地，一掃三代PK無人問津的局面。

不過，目前大多數內容還停留在設計階段，距離可玩的DEMO版本出現，預計還有很長一段時間。如果有人告訴你，明年嘉年華之前能玩上《暗黑破壞神4》，那一定是吹牛。
