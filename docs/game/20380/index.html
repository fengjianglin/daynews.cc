「雲頂之弈」拼多多不止是劫！你沒想過的拼多多打法教學

大家好，我是幻滅的泡沫，這是我在雲頂區的第二份投稿，今天帶來的是一篇拼多多套牌，速八陣容的速成攻略。這篇文章需要一定的基礎，建議大家先讀一下我的上一篇文章：【雲頂之弈】選擇蛋卷的三大目的

那麼書接上文，蛋卷這套牌其實打的就是一個設計曲線的不合理：3星森林3星鱷魚配合合適的裝備與羈絆強於大多數3星2費卡。而最近的版本更新，設計師也是注意到了這一點，將蛋卷的核心卡3星強度大幅降低，導致成型後強度降低，強勢期大幅縮短，那麼蛋卷這套牌是徹底與大家說再見了。而其他的賭怪，也就只剩下賭掠食一家，沒有海洋法的優勢對局和土格子Bug後，個人認為這套陣容也是從S級變為A級陣容，也就是前期很胡能玩，不胡彆強求的牌。總體下來就是，目前環境將沒有賭怪，或者說賭怪強度不夠高，是降速的版本。對於降速的版本就要用後期陣容來吃雞，在大家都很後期的時候，誰貪誰就贏。

當時也是有很多人質疑：這套牌這麼貴，根本湊不出來。而實際這套牌甚至經常要上9再上個千珏，組成羈絆王陣容。實際上這麼貴的陣容要成立，需要至少兩個先決條件：

1. 環境足夠慢——當時前期幾乎不掉血，虛空刺搶著賣血賣不動

2. 後期有足夠多的輸出點大哥以及多個控制點——看著這一個個的不是控制就是輸出，沒有一個打醬油的，潘森一張卡帶飛全隊

那麼反觀現在，上面已經總結了環境降速，只剩第二點有沒有類似的單卡有強力控制或強力輸出？

答案就是：有，而且更強力！之前的潘森主C位置只有潘森一張，現在有兩張，劫和煉金！而控制端泰坦就是小日女，娜美比大蟲子範圍更廣，還有石頭人和阿木木，也是一身控制！除了沒有龍王死歌三法這種AOE輸出，我們並不比之前差，依舊可以用質量碾壓對手！那麼接下來先上成型陣容：

那麼本文主題所謂速八陣容，就是要在4階段大家都是7級的時候，在合理運營後，同一時間點大家在7級d牌，二你能夠在上8後擁有30+經濟，然後d到一定量的橙卡搭配合理的裝備，質量碾壓對手。讀到這，有的同學就要開噴了：你不就是貪比把d鍵扣了么，我上次這麼打，狼哥打完就剩十點血，上8直接暴斃。為了不像這些同學一樣暴斃，請大家耐心繼續往下讀，接下來我們就講解一下怎麼能夠迅速上八不暴斃，以及上八之後怎麼轉陣，裝備要選擇什麼。

那麼從開局說起！首先要明確一點，這套陣容需要轉陣，轉陣期很容易失誤造成大量掉血，所以需要盡量贏下每一局，我們不d牌所以連勝可遇不可求。就算你d了很多，也會被比你更胡的人斷掉連勝。因此開局裝備是非常重要的，你的打工仔需要裝備保證你不掉血，並且你的大哥需要這些裝備保證坦度不暴斃，他們活著就是輸出！那麼上版本我會搶布甲和反曲，因為布甲又可以做復活甲又可以做綠叉，掠食者多也可以做紅buff；反曲次之，因為反曲只有綠叉可做，前期颶風沒有紅Buff香。這版本我暫時也沒想到搶什麼，我的選擇是布甲，總之布甲，大劍，腰帶三選一，復活甲鬼書冰拳是你優先要合成的裝備，其次可以合成龍牙和救贖。搶一個沒人要的就行，三選一都選不到，你就是太貪。

那麼打完1-2小兵，一般賣掉你選秀選到的英雄，不是土元素局買下所有除了岩雀的1費怪，土局可留岩雀配合琪亞娜。之後的小兵同理，裝備不適合掠食者，或者掠食者不天胡，如果賣掉能夠吃利息，就賣。2-1之前除非三森林雙山脈或者水元素局青龍刀螞蚱，2費3費卡全不要。2-2的時候如果還沒有2星卡，或者你覺得2星卡不夠多不夠強，或者對子很多，d牌1~2次，不要多d會崩盤。一般來說這時候你就知道你這盤是否適合打掠食者了，不適合就賣。

優先抓取：三森林>雙山脈>鬼書婕拉>雙雷霆>鬼書狗頭≈鱷魚>皎月≈吸血鬼

剩下的不玩六光、六遊俠、掠食就賣了吧，弟中弟

個人感覺d牌之後平均會有2~3個2星怪，4人口的時候雖然不是最強的，也能保證不掉太多血。選秀依舊優先做前排的肉裝，或者做鬼書給婕拉（或妮蔻狗頭，沒有婕拉的話）。復活甲>冰拳=鬼書>救贖>龍牙。之後和蛋卷一樣，吃整數利息，區別在於有多的2星怪不虧太多利息可以升人口上。

這裡再補充一個小技巧，萬一有不知道的朋友可以了解一下，選秀拿完怪之後按R返回自己棋盤，在最後1秒可以把剛選到的牌賣了多吃1塊利息。

到3階段你一般會開始抓到4費卡，掘墓安妮奧巴馬可留，石頭人不虧太多就拿。有鬼書婕拉鬼書妮蔻可上雙海洋。有螳螂可以上蛋卷的全二星版本。有大天使二星千珏或二星塞恩轉影諸如此類的廢話我就不說了，總之盡量吃更多的利息。如果你輸了一盤，可以考慮少吃10塊錢的利息，d一點點提升質量。選秀之後你會開始被劍刺入，沒關係，你打不過他大家都打不過他，你的強度在第三第四名就足夠了。逐漸白嫖質量卡，儘可能往三召喚湊，沒有的話三刺，雙海洋，雙系魂都行，核心就是不要被入太多，不要掉到50血以下。

4階段開始你會被花式血入，不要緊，他們花錢了。依靠前期優勢，賣血賣到你能上8並且能有30+經濟，開始我們的強勢期。d牌，d到煉金直接上，鬼書必備，復活甲冰拳龍牙冰心四選二。d到劫上3召喚，三雷霆後面再湊也來得及，復活甲冰拳龍牙救贖光轉職五選二。d到娜美盡量上雙海洋，要不然娜美被切容易放不出大。d到阿木木也可以上，一星雖然不是特彆強也能用，湊三火也簡單。剩下倆弟弟我就不點名了，爬就完事了。拉克絲看情況，能上就上，有時候甚至不如再上一個一星掘墓或安妮。後期選秀可以考慮做個光轉職給劫，因為後期很少有人要鏟子，法海減少龍牙也不是剛需。總之什麼強上什麼，如果你不知道什麼強，有啥羈絆湊啥羈絆，三召喚三刺三光雙秘術雙沙漠雙海洋。。。（鬼書煉金是金羈絆當然最優先）
