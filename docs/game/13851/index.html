爐石主播們新版本都在玩什麼？濤妹直接被薩滿打自閉，遁入戰棋

2019-12-12 118條評論 1,025次閱讀 99人點贊

「巨龍降臨」終於正式上線啦，各大爐石主播也終於離開了「酒館戰棋」，開始享受新卡的樂趣。

不過新版本似乎還是「薩滿英雄傳」，濤妹直接就被打自閉了。

於是他再一次遁入「酒館戰棋」，然後繼續掉分。

講道理，新版本的薩滿確實凶，加上迦拉克隆體系的戰吼薩壓制力實在太強了，不怪濤妹。

給大家一套目前比較毒瘤的戰吼薩，不僅強力，還只有一張新橙卡。

狗賊叔叔更有意思，剛開完卡包就直播分解破甲騎士。

而且他也不疊甲了，開始玩起了龍戰。看來嘴上說著不怕，心裡還是虛得不行。

狗賊叔叔玩的是狂野，卡組是一套宇宙龍戰。價格很貴，但強度嘛……非狗賊死忠粉請慎重。

王師傅在第一天居然沒玩牧師，你對得起「臟鼎」這個稱號嗎？

不過有一說一，王師傅這套手牌龍術還是不錯，很有操作感，有點當年巨人術的感覺。

需要注意的是，這個版本下龍對面大概率會拍488，直接把我們打崩。建議把辟法龍人給換掉。（真是低賤的龍種族）

會長又玩起來了他的老本行盜賊。

這套盜賊有多個贏點，可以靠蠟燭龍鋪死對面，也可以刷托格瓦爾來不斷找寶藏。

不過現在488橫行，上龍感覺就跟自殺一樣。而要耍雜技的話，賊的治療量又明顯不足。所以會長還是那個菜B，一直被暴打。

總之，這套卡組大家看看就好，卡全的話可以玩一下，不全別衝動合。
