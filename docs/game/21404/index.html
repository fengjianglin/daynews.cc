“我在這個版本里比以前更厲害了”

2019-12-20 1,784條評論 4,631次閱讀 1907人點贊

新加坡世界邀請賽昨日小組賽階段已全部結束，在首日就分別貢獻1-5和0-4的茶隊、LGD雖然第二天奮起直追，但無奈挖坑挖的太大，最終兩隊沒能抹平首日落下的距離，雙雙被淘汰回家。

A組裡茶隊0-3-2積3分，雖然和TNC分數一樣，但無奈TNC唯一贏的一輪比賽就是對陣茶隊，根據勝負關係沒有加賽，茶隊直接被淘汰；

B組這邊LGD第二天收穫三個平局，分別戰平了液體、VG和紅星，但無奈第一天的戰績實在太差，贏才能晉級的情況下他們卻只收穫了平局，離開也就理所當然了。

VG雖然擺脫掉一起上飛機的命運，成功進入到了淘汰賽，但隊伍的狀態也不是特別理想。小組賽5戰1勝3平1負，唯一贏的那輪還是對陣小組墊底液體。雖然進入到了淘汰賽，但VG的前景也不容樂觀，淘汰賽敗者組沒有犯錯的機會，輸一輪就直接淘汰回家。

今天只有了兩場勝者組之間的對決，VG也剛好趁此機會休息一下，調整下狀態，希望能在淘汰走的更遠一些吧。

R神的微博如約而至：

只希望下次看到R神微博時能晚個幾天，時間越晚，說明VG走的越遠，是不是很有道理！

今天兩輪淘汰賽結束的相當之快，紅星2:0錘翻了Alliance，秘密大魔王則是倒在了EG前進的鐵蹄之下。

這樣一看還是B組是實力強勁一點啊，兩組正面對抗中都是B組隊伍獲勝晉級，看來LGD回家也「情有可原」啊。現在B組前兩名EG和紅星已經會師勝者組決賽，不知道VG能否在淘汰賽碰到他們呢？大家一起期待一下吧。

明天共有三輪比賽要打，前兩輪先進行敗者組首輪的對決，最後是勝者組決賽。VG敗者組首輪的對手是TNC，這兩支隊伍可以說是冤家路窄，剛剛在成都Major的決賽里交過一次手，這次又在生死戰中相遇了。

兩隊比賽時間是明天下午14:00，核桃tv和龍神tv有版權，想看比賽的同學鎖定這兩兄弟的直播間即可。

楓哥今天的直播已開，下午和烏總他們血灑國服，晚上激情對黑，喜歡的同學不要錯過。

今天楓哥玩了把超級無解肥聖堂刺客，11分鐘就已經黯滅加身，整場比賽也成為楓哥聖堂的「宰豬」時刻。

爽了一把之後楓哥就開始划起來了，一把野生絕活大聖各種從樹上掉下來被暈，一把弟弟魚上來就0-4，開局就走在要翻盤的路上。不過楓哥對此也有很好的解釋，下午開黑隨便打打，精力全都放在晚上的對黑當中。

半步獸王境

這兩天楓哥又開啟了「贏最後一局=贏一天」的模式，不管前面戰績理不理想，反正收官局是贏的就OK。其實真要統計下來，楓哥對黑的勝率還真不低，也難怪楓哥會說「這個版本我比以前厲害的」。

今晚對黑楓哥究竟是真的贏一天，還是只「贏一天」呢，小殭屍們鎖定鬥魚直播間9999，我們一起見證喲！
