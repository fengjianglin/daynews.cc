今天給大家推薦一個9.23版本的強勢陣容：雲霄光劍士。

陣容構成：光元素薇恩、武器、劍魔、狗頭、牧魂人、快樂風男（金鏟鏟+斗篷合成的金色護符) 雲霄亞索、風女、光輝、劍士亞索、劍魔、大元素使光輝。

陣容是以光元素為主的，光元素的特性是在光元素單位陣亡後，剩餘的光元素單位，會獲得生命值回復和一定的攻速。核心英雄裝備推薦和站位參考：

薇恩巨人殺手、羊刀和繳械或者颶風（新版本颶風改了合成不需要金鏟鏟了，改為魔抗加反曲弓）。

劍魔復活甲、科技槍和鳥盾或者龍牙，亞索飲血劍、龍牙和金色護符，裝備是要看手裡有什麼散件自由搭配的，不一定要用我推薦的這些，攻速裝備多考慮一下薇恩，畢竟是個C，如果肉裝比較多的話，可以考慮給狗頭或者武器。

本套陣容較為簡單，陣容以薇恩為主C，前期過度可以使用三森林（大樹、翠神和妮寇），也可以使用3地獄火，最優選是森林。隨著等級慢慢起來，可以加入一些光元素的英雄。

中期可以考慮買一些優質的，便於激活特質的英雄打工，由於風女的出現，後排英雄的生存得到了一些保證，盡量把脆皮英雄放到風女旁邊。

牧魂人召喚的小鬼也算是光屬性英雄，小鬼死後也會給剩餘的光屬性英雄回復一定的血量和一些攻速，薇恩羊刀疊起來後，再加上光屬性英雄陣亡得到的攻速加成，薇恩將開啟爆炸輸出。

亞索搭配風女激活雲霄效果，會讓全隊增加閃避幾率，後期拿到拉克絲後，會激活3雲霄效果，閃避幾率會進一步加強。

最後D出一張光拉克絲，組成9光必定吃雞。
