CFPL丨季後賽馬上打開，這些人將成為戰隊關鍵先生

天天要聞
2019-12-22

CFS2019世界總決賽已經結束了，雖然中國軍團無緣冠亞軍，但有些選手的發揮依然亮眼，如綠豆半場16-2的戰績，子豪單局五殺，念舊爆頭連殺等等。CFPLS15季後賽馬上開打，不知道這些選手能否在國服比賽舞台中，繼續成為關鍵先生。一起來回顧下他們在CFS中的精彩表現。

敵人報團進攻A大，包點狙擊手綠豆靈活走位拿到雙殺，他一直卡在包點持續偵查，在隊友搜查陣亡後，綠豆又接連殺2人，最後換手槍擊殺，完成單局五殺。

雙方在B倉庫交火，白鯊所有人被寵兒牽制住火力，李思楠在後方一頓潑射三殺，再切換手槍又殺一人，可惜被Even終結，李思楠單局四殺。

敵人進攻受阻，但沒有放棄二次進攻，念舊在爆頭擊殺一人後，再跳出來掃射，拿下雙殺，這一波三殺直接幫助隊伍拿到回合勝利。

殘局人數相差不大，4me先殺一人有了手感，持續找人，拿到雙殺，已經鎖定最後一人，完成單局三殺拿到賽點，最後一回合4me再次三殺滅隊，幫助隊伍取得勝利。

殘局處理，己方佔有人數優勢，綠豆一桿狙擊先瞄準酒吧出口擊殺掉一名敵人，隨後換子彈再瞄準下水道進入的敵人，完成雙殺滅隊。

敵人報團進攻，歐歐和隊友配合卡在包點，敵人搜點過程中，歐歐完成三殺刷屏，直接將敵人打成僅1人存活，隊友順利拿下這一回合。

白鯊作為潛伏者進攻，雖然遭到保衛者的刷屏，但子豪完成單局五殺。在進攻途中子豪雙殺入賬，卡在窗口下再拿到雙殺，最後小道1v1滅隊。

對手快攻，念舊在捲簾底攔截，使用ak三個爆頭擊殺，最後換手槍又殺一人，已經四殺的他在面對最後一個敵人沒有子彈，換子彈的時候被隊友搶了五殺。

儘管世界賽裸裝和國服全裝有一些差距，但職業選手的基本功都在，只要在賽前勤加練習，比賽時保持火熱槍感，他們依然會有極限的精彩操作，甚至是力挽狂瀾的發揮，期待季後賽中他們的表現。
