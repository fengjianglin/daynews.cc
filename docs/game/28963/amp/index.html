雲頂來襲，法海控制流：只要控制連的好，除了劇毒都得倒

天天要聞
2019-12-20

各位召喚師們大家好，小皮昨日巧遇了一個陣容，我的神裝四劍士沙漠刺二星鋼鐵夢魘居然在一番控制下被直接抬走，開局一個普攻都沒有釋放出來，雖然很巧合，但是小皮細看了下該陣容發現其控制鏈的確很足，如果開局就被控到了，很可能到死都打不出任何傷害。小皮當時的夢魘開局在空中跳躍的時候被岩雀的岩凸抬到了，然後致死都沒有動過了，就算是鋼鐵羈絆下也不能倖免，下面就讓我們來看下這套守護法海控制流吧。

英雄配置

火男，石頭人，泰坦，阿木木，錘石，岩雀，海洋，娜美

替補成員：娜美——奇亞娜（海洋）

整套陣容組成控制十足，岩雀控制是單點控制兩秒，石頭人群控2/2.5/5秒，泰坦單點控制3/4/6秒，阿木木群控2秒，娜美群控1.5/2/2.5秒，從數據上不難看出，如果控制無縫連接上甚至可以讓你十秒鐘內都無作為。

羈絆特效

羈絆上法海是基本框架（四海洋三魔法師），魔法師能夠在控制期間打出不俗的輸出，四海洋可以提供額外的法力值回復讓技能釋放頻率加快，四守護的額外護甲提升讓前排的坦度得到保證，山脈羈絆隨機1500護盾讓我們中期非常強勢。

裝備配置

火男：鬼書+青龍刀+大帽子

石頭人：鬼書+龍牙（冰拳）+復活甲

火男依舊是法海流輸出的不二人員，鬼書的重傷效果配合火男大招效果十足，大帽子的技能傷害提升效果讓火男大招傷害再次提升，青龍刀可以讓火男釋放技能後可以快速釋放第二個甚至第三個技能，其實這樣裝備的火男兩個技能基本就是清場效果了。石頭人可以選擇出肉裝，也可以出法強裝，鬼書與石頭人的技能搭配也十分好，群體重傷瞬間爆炸。

大選時優先選擇大棒，藍水晶，其次根據其他玩家陣容選擇拿護甲還是魔抗，優先滿足火男的裝備，其餘的裝備建議給到石頭人、阿木木這樣的群體控制單位，這裡提一下，我們這套陣容比較適合拿暴擊拳套合成竊賊手套，往往偷來裝備的可變性能為陣容提供奇效。

陣容站位

石頭人，泰坦，阿木木頂前排，吸收傷害的同時快速獲取法力值，火男放後排中間，其餘的人分開站兩邊吸引刺客火力。

陣容過度

前期根據刷的牌決定打工仔，雷霆守護是目前前期作戰能力較強的打工陣容，儘早找到岩雀和石頭人的話湊齊山脈羈絆能讓我們前期很好的站穩腳跟，裝備直接給到打工仔保證隊伍強度，注意不要給到我們需要留的牌了。打完石甲蟲後小D一波，保證低費牌都上二星了，這樣五人口後就可以上海洋守護魔法師打工了，只要全二星的五張牌（吸血鬼、泰坦、石頭人、岩雀、辛德拉），石甲蟲後我們是能很好的保持連勝的。後續保持隊伍強度的同時吃連勝利息，七人口時把石頭人升到二級，八人口後找阿木木以及娜美，在找這兩張牌的時候可以把別人不要的五費牌捏手裡再刷新，增加自己拿到五費牌的幾率，刷新到了再賣掉，這樣也不會損失金幣。

陣容分析

這套陣容主要是靠打出控制，然後法海瘋狂輸出，很懼怕成型後的劇毒陣容，劇毒的增加藍耗讓我們的控制技能很難放出，就更別提後續的傷害技能，但是面對不是很胡的劍刺還是沒有問題的，連續控制讓對面根本切不起來，而且劍刺的身板較脆，只要控住，吃不了幾波傷害的。

關於本期法海控制流的推薦就到這裡了，大家還在等什麼呢?速速上手來一波吧，最後記得給小皮點個關注喔。
