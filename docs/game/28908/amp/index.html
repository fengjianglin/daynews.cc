魔獸世界：以後的發展方向會取消野外地圖？只設置副本和戰場嗎

天天要聞
2019-12-21

有玩有表示現在的魔獸世界早就改了，和以前的魔獸世界不一樣了。暴雪現在刪除了碾壓機制，取消了命中屬性（副手武器命中懲罰還在），加上早之前取消的抵抗機制。所以說現在的技能是100%命中，另外根據動態平衡小號一個技能打到大號身上，傷害是會有加成的。剛開120的時候，練級快的人發現滿級了想去虐小號結果被人家守屍，或者一個115級小號追著幾個滿級號砍的，甚至本來沒滿級前可以輕鬆打的小怪也突然像吃了葯一樣能把人錘的滿頭都是包。

其實就是動態平衡惹的禍，現在沒事不要輕易去惹小號，尤其是115還能用橙裝的小號。我是部落的，剛滿120級的時候，去做世界任務遇到一個聯盟的116級血dk在和兩個滿級部落的人打架。我以為我們這邊級別高加上人多贏定了，於是也加入了戰鬥打了半天，最後咱仨人被一個部落小號追著跑，最後掛了還被守屍了。

法爺頂個冰盾再加上特質裝的自動護盾三層，配個閃現回血可以陪小號玩好多天，都不想還手。要打起來的話，pvp天賦下溜溜球可以定點放，水寶寶冰環後排溜溜球一放全倒，天賦支持下溜溜球可以使暴風雪瞬發，以自己為目標中心放暴風雪，近戰全死要是不死也沒事，寒冰護體一頂一個閃現先掛機等一會，泡杯咖啡刷一會朋友圈，等技能CD也好讓那些撲街早的跑屍體回來再收割一波。

魔獸世界當年80年代巫妖王有一個地圖叫冬擁湖，會按照雙方參戰人數分配力量，當時曾經出現過一方只參加1人的情況。在一個平均生命不到2萬的版本，出現了一頭60萬生命的熊德，真的是一巴掌一個小朋友。正面完全沒法打，比團本boss都硬，不過戰場不是比殺人，做任務拿積分一個人是怎麼也比不過40人的。

8.35末期直接艾星被毀，腳男集合所有的艾澤里特精華使愛醬從新煥發了生機，但同時腳男也失去了都有的力量，原本的怪受到新力量的影響產生了進化，都有玩家等級全部回到10級，要想從新獲得力量必須從來等級上限為45。懷舊為了練級要跑遍整個艾澤拉斯大陸，正式服的地圖這麼大，不出城門就能到滿級。以後魔獸的發展方向會取消野外地圖，只設置副本、戰場和要塞。你們覺得呢？
