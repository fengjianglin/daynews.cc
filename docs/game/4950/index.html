冷少被姿態問對全明星的看法，直言我不配，聖槍哥扎心了

2019-12-03 0條評論 16次閱讀 0人點贊

摘要

相信很多老玩家都還記得OMG這支全華班隊伍，他們從建隊開始就是全華班的陣容，不過從S4被RNG的前身皇族戰隊擊敗之後，OMG就一步一步的走向了末路。

相信很多老玩家都還記得OMG這支全華班隊伍，他們從建隊開始就是全華班的陣容，不過從S4被RNG的前身皇族戰隊擊敗之後，OMG就一步一步的走向了末路。而現在的OMG，只剩下孤兒院院長冷少一個人苦苦支撐，冷少心裡也是苦啊，隊友都是些什麼「臭魚爛蝦」，但是今年OMG是終於捨得把那些人全踢掉了。

畢竟冷少雖然強，但一樣也拯救不了OMG，今年的OMG自然也就沒成績了。而沒成績的冷少，即使他的人氣並不差，但是也比不上今年參加世界賽的兩位中單選手，因此冷少今年就無緣了LPL第一屆的全明星比賽。不過冷少作為OMG的牌面，還是要去到現場代表OMG戰隊的。

而作為這次世界賽的特邀嘉賓，退役選手姿態，在全明星賽前就兼當了一份記者的工作，而姿態就在全明星後台抓到了冷少和LNG的聖槍哥。被抓到了自然肯定要被姿態問問題了。而冷少被問的是他自己對全明星的看法。冷少則是很直白的說出，這個問題他不配發言。雖然很直白，但是冷少的心裡還是難免會有一點難過的吧。

不過姿態也是告訴了冷少，不用顧忌聖槍哥在隔壁，說就完事了。而這次冷少就直接說出他自己支持Shy哥，讓隔壁聖槍哥也扎心了，看聖槍哥的表情實在是非常搞笑。不過聖槍哥本人也做出了回應，覺得冷少說的很對，畢竟對面都是世界冠軍。仔細一看，IG三人組和FPX兩兄弟都是世界冠軍，而他們這邊只有Doinb一個人。這次聖槍哥還嫌不夠扎心，自己再往自己身上多插一刀。

因為現在是轉會期，相關的爆料非常多，據說離開蘇寧的SMLZ要回去OMG了，而LNG在sofm離開以後將會迎來現TES的打野選手XX。他們兩個作為隊伍的絕對核心肯定是非賣品，不過下一年他們的陣容就將會比今年好了，或許他們還能衝擊一下S10，讓自己也成為世界冠軍，那樣就能名正言順的參加全明星而不用被扎心了。
