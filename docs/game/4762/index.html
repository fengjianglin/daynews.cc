夢幻西遊：李永生一鍵換裝變土豪，掏出的寶貝都價值連城

2019-12-05 0條評論 19次閱讀 0人點贊

摘要

夢幻西遊李永生，一直在CC以直播普通夢幻玩家的日常。他因為是夢幻西遊的底層玩家而出名，所以他保持著自己的初心，用底層玩家的形象出現在玩家面前。不爭排名，做好自己的每一件事。如果說成為主播後，他最大的不同，就是以前他是自己管自己玩遊戲就可以。每天就是上線刷怪，做師門任務，跑商。

夢幻西遊李永生，一直在CC以直播普通夢幻玩家的日常。他因為是夢幻西遊的底層玩家而出名，所以他保持著自己的初心，用底層玩家的形象出現在玩家面前。不爭排名，做好自己的每一件事。

如果說成為主播後，他最大的不同，就是以前他是自己管自己玩遊戲就可以。每天就是上線刷怪，做師門任務，跑商。活動、排行榜，這些RMB玩家熱衷的事物，似乎都與他無關。現在的他，已經是50萬人訂閱的大主播，自己也除了玩遊戲之外，經營著自己的工會點點滴滴。

今天的直播與平常略有不同，因為冬天來了，家裡沒有空調的李永生在嘗試用最原始的方式取暖。要不是因為不能在房間里生火的規定，估計他就要擺放一堆篝火在身邊，一邊給烤火一邊取暖了。李永生今天在做龍宮任務的時候，突然想到天氣變冷了，於是給直播間的觀眾表演了一鍵換裝。

換裝之前是這樣的：

播了一會後，感覺實在冷不住。就找自己的椅子代播了一會。自己則去後台換了一身時裝。

瞬間化身RMB玩家，這種風格的帽子，現在市面上買到的也只能是仿製品的A貨。李永生的這頂帽子是供銷社時期的真毛皮帽，抗寒冷效果+50%。

換上了時裝之後，李永生的操作也彷彿得到了屬性增強。李永生今天就一件任務，就是替幫派賺15萬兩銀子。

做跑商的過程可以說是非常無聊了，李永生這時又給大家拿出了自己的傳家之寶，一個上古時代的鬧鐘。說自己設定好時間，再過兩個小時應該就能賺夠銀兩，接下來去和別的主播玩連麥的遊戲。彈幕卻認為，如果把這個鬧鐘拿出去典當的話，這些銀兩應該一下子就能賺足了。

歷時1小時20分鐘，白虎任務終於完成了。在這裡，小編不得不佩服永生的毅力，現在很多人都是用腳本打跑商任務，或者直接就不做。李永生居然每天都花固定的時間做任務，還能把任務的每一環都背下來，也實在不負毅力帝這個名號。

但是相對他的直播，粉絲們似乎更感興趣他的家當，永生的家給人一種古老樸素的感覺，很多農村的老玩意，他就能像玩具一樣隨便掏出來。如此接地氣的永生，下一次直播又會給大家展示什麼值得懷念的玩物呢？
