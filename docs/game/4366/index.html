利用VR頭顯透視功能，NTW推MR拳擊遊戲《HitMotion》

通常，VR頭顯的透視系統主要用來幫你看清邊界，避免撞到周圍的障礙物。而近期，New Technology Walkers利用VR頭顯的透視功能，開發了一款MR健身遊戲：《HitMotion：Reloaded》，目前已經登陸HTC Viveport。

據了解，《HitMotion》目前僅支持HTC Vive Focus Plus一體機，其特點是專為VR一體機頭顯開發，利用頭顯的透視功能，讓玩家可以在VR頭顯中體驗MR健身，為的是劇烈運動的同時能夠看到周圍，提高安全性，同時遊戲會識別玩家周圍環境，在多個方向生成遊戲內容。

也就是說，在《HitMotion》中，你可能會從四面八方飛過來的方塊，而你的任務就是躲避它們的攻擊以及用拳打回去。有趣的是，遊戲中的方塊像《Beat Saber》一樣也分為多種顏色，藍色的方塊需要用左手去打，而黃色的需要用右手。

除了左右手外，不同的方塊還需要你去用不同的拳擊姿勢去擊打，比如刺拳、勾拳、上勾拳，你的反應越快得到的分數越高，打錯了方式也會丟失分數。

《HitMotion》的畫面來看整體比較簡單，不過它的特點是巧妙利用頭顯的MR透視功能，讓你可以安心在家裡玩動作劇烈的VR遊戲，不用擔心撞牆。當然，市面上大多數VR頭顯的透視攝像頭不具備較高解析度，因此遊戲可能看起來比較粗糙，如果用Varjo的高解析度透視功能玩，效果肯定會好很多。
