爐石傳說：雙12開啟冬幕節活動，除了聖誕卡背，還有泰蘭德

天天要聞
2019-12-04

摘要

《爐石傳說》最新卡包“巨龍降臨”將於12月11日上線，本次卡包的關鍵詞“超模”已經讓玩家感到新版本對於遊戲的顛覆。緊接著後一天又會上線冬幕節（就是現實世界的聖誕節）活動，除了新的聖誕卡背，更有無數玩家翹首以盼的泰蘭德復刻。不禁讓人感嘆，暴雪這次也太大方了吧！

《爐石傳說》最新卡包「巨龍降臨」將於12月11日上線，本次卡包的關鍵詞「超模」已經讓玩家感到新版本對於遊戲的顛覆。緊接著後一天又會上線冬幕節（就是現實世界的聖誕節）活動，除了新的聖誕卡背，更有無數玩家翹首以盼的泰蘭德復刻。不禁讓人感嘆，暴雪這次也太大方了吧！

首先是新版本巨龍降臨，暴雪於11月底開始了新卡牌的爆料工作，近半個月的爆料讓我們發現了新版本的關鍵詞竟然是「超模」。各種存在於玩家腦洞里的卡牌紛紛出現，不僅身材不虧，特效還超強無負面效果。

新卡牌普遍都超模

這波新卡牌一旦放出，勢必會讓現在如死水一般的天梯發生天翻地覆的變化，新版本的天梯將會掀起怎樣的腥風血雨，讓我們拭目以待（就算出新卡包也不能阻止我玩酒館戰棋）！

緊接著後一天將會開啟冬幕節活動，本次活動持續時間從雙12的凌晨2點，一直持續到2020元旦節的凌晨2點。按照爐石的慣例，將會開啟極具特色的冬幕節大亂斗，雖然官方未透露亂斗細節，不過看這次獎勵的新卡背來說，還是挺不錯的，再加上亂斗會獎勵的卡包，這波獎勵還是挺香的。

新卡背還是挺好看的

當然除了新卡與新活動，又趕在雙12上線，自然也會出剁手活動。本次爐石的剁手活動主要有2個，一個是巨龍再臨新卡包的合集，100個新版本卡包與戰士英雄死亡之翼「耐薩里奧」及其卡背。另一個則是冬幕節促銷合集，含前5個版本的卡包各6包（共30包），與新的德魯伊樹人英雄「黑澤爾巴克」及其卡背，是否有新的彩蛋還需等新版本上線。不過2個合集同時出售，也讓玩家大喊錢包遭不住！

黑澤爾巴克是爐石的原創英雄

當然，除了以上一大堆新內容，還有一排小字讓人格外矚目，那便是官方承諾了一年的泰蘭德將會復刻了。泰蘭德作為牧師英雄，自16年短暫活動可獲得後，便再未推出其它獲得途徑，這也讓之後進入遊戲的玩家後悔不已。不過官方在年底終於要實現19年年初的承諾「復刻泰蘭德」,具體如何獲得還要看官方12月6日的介紹。

泰蘭德

總的來說，本次新版本活動還是很給力的，不僅有卡包拿，還有免費英雄獲得，小夥伴們期待泰蘭德的復刻活動嗎？
