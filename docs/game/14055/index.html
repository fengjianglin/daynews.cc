石器時代M燒錢的陣容組合流派之五虎流詳解

2019-12-11 2,045條評論 3,823次閱讀 1168人點贊


一、五虎流介紹

本篇給大家帶來主流的五人組隊陣容-五虎流。

五虎流顧名思義，即五隻貝魯卡系（老虎）搭配在一起的陣容，該陣容不僅帥而且非常強力，而且還可應對各種活動，尤其組隊鬥技場中勝率甚高。

五虎流的強大之處主要依靠貝魯卡系的1怒技能【迴旋切割】可以直接攻擊後排，並利用覺醒技能【王者之風】三回合增益，再配合一些增益減益精靈與其他寵物增益，形成高度的前3回合爆發。

而被動技能【嗜血】可以提供五虎的續航能力和其他增益效果，尤其是火系或者機械系貝魯卡，可以產生很強的增攻效果。

優點：速度快，爆發力高，容錯高，可以直接對輸出後排，打法與陣容簡單沒有複雜套路。

缺點：硬體要求高，不然容易成五貓而不是五虎。

簡單總結五虎流，就是簡單，快速，暴力；與激勵流對比，有更高的容錯率和更簡單的速度段位分配。

○研究建議

生命>生命%>攻擊>攻擊%>暴擊概率>暴擊效果>敏捷>敏捷%>命中>抗性

○性格訓練建議：

性格：暴躁 訓練：攻擊、敏捷

(關於為何選擇「暴躁」，是因為訓練是攻擊與敏捷對點，單獨選擇勇猛其實收益不太高。)

○圖騰建議：暴擊攻擊圖騰

○符文建議：

暴擊•攻擊符文。技能：快攻 進階技能：致死>強化鋒銳 改造：攻擊>暴擊概率

抗性•敏捷符文。技能：快速 進階技能：強化突破 改造：攻擊>暴擊概率

抗暴•生命符文。技能：強命=清醒 進階技能：溫和 改造：敏捷>生命

三、陣容推薦

陣容的推薦關鍵就是如何最大化提高五虎的傷害。

其中遊戲中有不少能提升貝魯卡系傷害的關鍵精靈：

【專註精靈】與【高級專註精靈】

【破甲精靈】與【高級破甲精靈】

【屬性精靈】

職業與精靈搭配並不是唯一。建議棒職業騎葫蘆系與狒狒系，主要是棒職業天生有額外的抗性，當然這不是唯一搭配。

其中有幾點需要注意：

1. 五隻老虎的速度盡量差距不要太大，這樣可以讓爆發力集中一個時間點上。

2. 五虎流關鍵的優勢是可以集中打後排，所以選擇好後排的目標統一擊殺。

3. 施放屬性精靈的屬性手，速度調整僅需要比五隻老虎快一點點。

四、剋制方案

五虎流其實是很難有克制方案的，該玩法容錯率很高，如果硬要說克制方案也是有的。

1.2或者3個高速人龍or高速狂人龍，使用1怒技能【一擊必殺改】在相同硬體下，是可以秒殺貝魯卡系的，當五虎只剩下3虎或者2虎就沒有威力了。

2.使用1或者2迴旋鏢的五九尾隊，該隊伍速度極快，並且隊伍很硬，打法極其猥瑣。迴旋鏢專門打人物，降低雙抗，再配合九尾大招控制後排，並且沒必殺可以普攻控制前排，同等硬體下，貝魯卡系是很難與九尾系拼速度的，就算有3個凈化手，也很難讓五虎都能做穩定輸出。
