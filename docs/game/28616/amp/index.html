12.22輔助T度排行｜太乙張飛頂端局非ban必選，鍾馗崛起

天天要聞
2019-12-22

大家好，我是老劉備。今天個大家打來輔助的T度排行，相較於賽季初的版本，現在的輔助T度更加穩定。這賽季是一個射手的版本，而一些輔助因為野核變弱，導致輔助也受到部分影響！

看不清，請放大

T0

這個版本更新以後，太乙和張飛受到版本影響逐步成為高分段的T0英雄。

①太乙得益於被動可以快速刷錢滾節奏，以及大招復活的容錯率。

②張飛操作簡單，主要還是對意識的把控，而高分段大家的意識都不會弱。張飛這個版本在高端段強於牛魔原因就是，他不僅僅是一個人的輔助，而是一個團隊的輔助。

這個版本的核心是團隊強則強，而不是個人強則強！所以這也是為什麼低分段牛魔強，高分段張飛強！

③盾山是職業比賽最強輔助，為什麼？因為巔峰對決的指定輔助就是他！

我們這裡來客串說下張飛和太乙的召喚師技能：

頂端局的張飛暈眩和治療各半，勝率也非常接近，但是治療的死亡或降低，間接影響了MVP率和評分，所以治療的容錯率會更高！

頂端局的太乙主要還是閃現，其次是暈眩，而使用暈眩的容錯率會低一些！

T0.5

明世隱的問題還是因為低分段大家不懂得剋制他，而到了高分段以後，隨便拿出一些團控型英雄就能把他活活按死，而且如果明世隱體系如果沒有經濟壓制的話，就等於少一個人看視野！

Q：T0.5輔助的守著她鍾馗是什麼鬼？

最近這段時間，不知道大家有沒有挺聽說，虎牙的某個鐘馗主播把幾個頂級主播打自閉，只要碰到守著她的鐘馗，下一局必定禁鍾馗。而這個鐘馗主播，也成功用鍾馗打到安卓QQ的巔峰賽第一（12.15）。

這是頂端巔峰賽鍾馗的趨勢圖，除了勝率有所波動，你會發現出場率、禁用率、熱度都在不斷的上升。

而且鍾馗強在於輔助，而不是在中單。中單會被兵線所拖累，而作為輔助的話，你是敵人你永遠都要擔心鍾馗從哪裡出勾。雖然這個英雄是非1則0的英雄，但是當你的勾無限趨向於1時候，那麼對方連走位都要膽戰心驚！

鬼谷子&大喬

這兩個英雄本次的T度排行我都給予降低了一個級別，大寶備說下個人想法吧！

鬼谷子這個版本弱的原因在於野核變相被削，一旦對方的射手發育成型以後，只要鬼谷子不能側面吸到後排（後期正面吸很難很難）。而作為開團型輔助，你想要保護自家的C位是很難的。同樣是強開團的輔助，盾山既可以強開也能保護，盾山這個版本作用會更高於鬼谷子。

這個版本重點不是運營，而是前中期快速發育，中後期比拼團戰能力了，也這是這個非運營的，你辛苦運營一波還不如別人一波團戰結束你們。

魯班大師

當前魯班大師我暫定排名T2，但是我認為魯班大師遠不止T2水平。

你認為一個T2輔助能把戰力打到13000分嗎？

魯班大師調整之後，頂端巔峰賽的熱度也在不斷的上升！
