暴走大俠：分解赤靈果換碎角銀值嗎？大神給兩類玩家提供3種建議

2019-12-12 568條評論 1,536次閱讀 87人點贊

氪金不暴走，大俠全靠肝，大家好，我是「爆肝大俠」小魚乾，歡迎來到由小魚乾開講的《暴走大俠新鮮事》。本期小魚乾給大家聊一聊新版本的「碎角銀」吧！

在上次大版本更新後，官方加入了「碎角銀」這個新貨幣，可以在兌換商店兌換獎勵，還有多少玩家是不知道的呢？入口就在「分解」裡面，可以看到有兩個標籤頁，一個是「裝備」，右邊的就是「其他」，分解材料，可以獲得「碎角銀」。

小魚乾就隨意說幾種材料的價值吧，1破鏡丹換8碎角銀，類似的，紫金鑰匙價值是20，英雄酒是12，萬獸丹是8，絕學筆錄是2，天蘊草是2，淬鍊卷是2，赤靈果是2，一隻坐騎是一百多。

那麼我們再來看看兌換商店，一個破鏡丹就要40碎角銀，一個天蘊草要10個碎角銀，等等這些，價值是分解的5倍。

商店裡有英雄酒、絕學筆錄、赤靈果、天蘊草、破鏡丹、萬獸丹這幾種材料，還有一些絕學殘卷，還是有些有用的東西的。目前想獲得碎角銀，除了前段時間更新出BUG白送的一些碎角銀，就只能通過分解材料了。

那麼是否值得分解材料去獲取碎角銀來換東西呢？又該分解哪些呢？有「大神」玩家針對兩類玩家，給出了建議。

首先是不氪金或者微氪的玩家，如果你沒有「貓小狸」、「鸞小青」等高級寵物，則非常建議把赤靈果全部分解掉，換成碎角銀。因為普通寵物對實戰的影響並不大，即便你有變異寵物，升級也不會增加「變異特性」的數值的，所以說與其用赤靈果來升級寵物，還不如分解掉兌換為好。

赤靈果可以在關卡中免費獲得，存一存分解個幾百上千還是沒問題的，雖然不是很值，但總比沒有好，累積起來也不少了。

第二類玩家，如果已經開啟了7名以上紅顏的「雙修」，則可以把所有的花束分解掉，畢竟400以上好感度，再增加下去，也只是略微增加雙修獎勵，而且7名已經達到上限，可以不用再用花來堆好感度。

但無論是哪一類玩家，都不建議分解破鏡丹、萬獸丹、天蘊草和紫金鑰匙這些，畢竟這些是可以實打實提升戰力的，坐騎增加的也是主角屬性，而紫金鑰匙等下一個階段也可以開下一階的寶箱，必須存著。​

那麼獲得了碎角銀，該兌換什麼東西呢？「大神」也給3類玩家提供了4種方案。

第一類是不氪金或者微氪玩家，這類玩家還是老老實實拿著碎角銀去兌換破鏡丹，破鏡丹對戰力的提升是很大的，到了中期，突破一次就要50個破鏡丹，相信很多不氪金的玩家都遇到過修為早就滿了但破鏡丹不夠的情況。

小魚乾建議不氪金的玩家不要每天花400幫貢去兌換兩個破鏡丹，因為現在出了典藏技能排山倒海，平民玩家節省點幫貢儘早拿到大獅子吼才是正事，而破鏡丹在世界BOSS和這個碎角銀商店兌換就行了。

第二類玩家是中等層次的玩家，這類玩家有兩種選擇，一種可以兌換英雄酒和破鏡丹，300碎角銀能來一次高級5連，這種是比較建議的。第二種是兌換排山倒海的4種殘卷，用於提升排山倒海的等級，兌換的時候要注意先兌換前兩種便宜的，因為是按平均等級計算排山倒海的等級。

而重氪玩家就隨意了，什麼缺換什麼，不受限制。

那麼以上就是本期《暴走大俠新鮮事》的全部內容啦！最後，感謝您的閱讀，各位路過的大俠給小魚乾點個關注吧~
