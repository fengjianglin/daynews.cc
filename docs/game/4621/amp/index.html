摘要

近一年時間，我的現實人生和遊戲人生都經歷了巨大的變化，之前我是五開黨，如今變成了單開，之前我是單身，現在……這其中發生的事情，非常值得和各位老朋友念叨念叨，因為整個過程就是一個“爽”字！下面就聽我一一道來，那些我身處“紅利期”中所發生的事。

曾經的高端五開 龍城飛將承載美好回憶

玩大話算來有十六年的時間，從當年懵懵懂懂，每天捉飛魚給漁村村長「賺」錢，到手動在傲來國酒店門口燒法，再到如今，我也算是個合格的老玩家了。

前段時間一直在龍城飛將這個伺服器，從單開到五開，可以說各有好處吧，單開可以有更多的隊友進行交流，過程更加輕鬆有趣，而作為一個五開黨，我可以很自由的支配和安排時間，很自由的做任何自己想做的事。

不知道玩五開的是不是都有這樣的心路歷程，就是最初五開的時候，心裡總會把某個號作為主號，好東西好資源都優先給過來，而隨著時間越來越長，這種「差別待遇」也就越來越小，因為手心手背都是肉嘛，也就無差別進行硬體升級和投資了。

我的號在五開黨里也算是中高端的了，當時的男人還上過葉子豬的展示，雖然現在這個男人號已經不在手中了，但當初的展示還是為我保存下了一些美好的回憶：

相信內行人都看得出來，這個睡人的打造的確是花了很多心思的，在區內這隊五開也經常受到玩家朋友們的激賞，會來諮詢我一些心得，我自然是無私的分享給大家，幫助大家在配號配裝時少走一些彎路也是好的。

龍城飛將，承載了我那一段時期最美好的回憶，攻克窮奇、攻克煉獄、直至攻克15星，PVE玩法中挑戰的快樂我都有所體會和享受，如果一定要我說一件遺憾的事，我想了一圈，沒有，再想一圈，還是沒有。

後來因為現實原因，一個不可抗拒的因素，其實就是脫單啦，我必須離開大話一段時間，同時需要賣掉自己的號以及號上的物資。

這個時候我其實很不舍，但誰料到，在忍痛割捨的同時，我發現了一件超爽的事！

賣號過程中 遇到極度舒爽的事

買號配裝備的過程是很幸苦的，要算屬性，要逛市場，要和商販們砍價，要蹲在格子前等著搶心儀的裝備，等到賣號的時候，同樣需要仔細的了解一下現今的市場，沒想到這一了解，我整個人陷入了極度的欣喜中。

當時買入的裝備都算比較便宜，到我要賣出的時候，看了一下市場，我震驚了，很多裝備的價格比我買入的時候暴漲了好多好多！

例如這張星符，是我買來給大仙配的，雙水系狂暴程度加青龍星陣（看上了忽視睡的宮位），很不錯的一張星符：

當時買入的價格是3200元，並沒有跟買家過多討價還價的情況下，讓對方直接指定的：

在出售的時候，本來想著能保本就行，結果沒想到…你們猜猜我賣了多少？

5800！本來準備將五開的五個號全部出售，但因為現實事情的緊迫，最終我只賣掉了四個號，但收入已經比當時的投入翻了幾倍，大大超出了我的預期，留下的這個號，有種感覺是白送的…

這裡必須要贊一波大話的保值和升值，老玩家應該都很清楚，大話中的裝備、寶寶等各種道具，這些年貶值的和升值的比起來就是九牛一毛，絕大多數都能保證當初入手的價格，甚至一路上漲，前幾年入手的很多東西在現在看來，真是一次非常划算的投資！

雖然我沒有趕上二十萬買絕境年，一百萬出售這種巨大的好事，但以現在的收穫看來，已經非常非常令人滿意了。

（買來衝到14級，還沒來得及用，就遇到要清號，最後賣了這麼多著實吃了一驚）

我最終留下的是一個仙號，萬萬沒想到，本來打算離開至少半年的我，僅僅兩個月的時間就因為這個號而決定回歸了……

朋友的瘋狂召喚 目前的日常以及打算

前段時間清號之後，每當空閑時就會和群里一幫老朋友們語音，這是一群認識多年的老友，他們目前在鶴舞雲天伺服器，得知我因為現實要清號的消息後，這幫人每天群里不斷的勾引誘惑我，問我何時回歸，讓我前往鶴舞一起征戰，搞得我每天都處在天人交戰的煎熬中…

直到我再也忍受不了這種誘惑，果斷回歸之後，這幫傢伙又忽悠我移民到四上和他們組隊！雖說是忽悠，但我其實非常的心甘情願，因為這款遊戲對我來說，最重要的還是這些老兄弟的友情，有他們在，就有一種說不清道不明的歸屬感。

上個月，我毅然決然的跟隨碧海潮生伺服器來到四上區，然後移民鶴舞雲天，我終於回家了！

從四下移民到四上，實話實說，成本並不小，有些時候還會因為號的底子比四上的差等等原因，會導致整體貶值，但這些都阻擋不了一顆想要找到歸屬的心！

目前每天就和老朋友們一起組隊，一起享受遊戲人生，雖然依舊沒太多時間玩，不過有一點可以肯定，就是無論怎樣，我都會從中獲得無限的樂趣和享受！

後記

大話這款遊戲的魔力真的很大，對於我們這些老玩家來說，只要愛上過，就很難再愛上別人，即使分別也會心心念念，迫不及待要回到它身邊。

最後想對那些因為現實離開，如今想要回歸的玩家們說一句：大膽的回來吧，據我仔細觀察，紅利期會持續的時間很長，從物質方面來說，如今回歸必然只賺不虧，從精神層面來說，如今老玩家們扎堆回來，遊戲內相當熱鬧，更是一個大好的時代！
