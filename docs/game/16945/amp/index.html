DNF徹底告別打樁！一百級的怪能控能破招 能放大狗，最理想的版本

天天要聞
2019-12-15

韓服冬季發布會結束了。大家關心的三覺來了，關心的希洛克副本還沒做好，史詩可疑交易了，這些都是重頭戲，但是有玩家關心過一百級怪物是什麼樣子的嗎？怪物機制一直都是個大問題，從超時空開始，大家就開始吐槽破防打樁再破防再打樁的機制太無聊了，而一百級的怪物則徹底告別了打樁時代，這是一批最為活躍的怪物。

首先這些怪物是可以控制，或者被多hit技能壓制的。說到控制和多hit技能壓制，大家也該明白了，瞎子暗帝又變強了！召喚又無敵了！不過雖然如此，扎熱六加三的時代畢竟過去了，能控很不錯，混團就別想了。而且這些怪物也是能控制你的，如上圖所示，玩家被怪物的三路大冰給冰住了！不是說怪物機制不好嗎？這下子給你冰起來，還好不好？

怪物不僅能放大冰，還會放大狗！由於一百級的新地圖全部坐在天界，所以這些地圖裡的怪物實際上就是當地的原住民，人家雖然不去冒險，但是轉職覺醒這種事情還是會做的。想想紅眼暴走嗜血一開，結果被怪物反手一發大狗秒了，那場景也太美了吧。好了，不是嫌棄怪物笨拙嗎？這下怪物厲害了，大狗都會放，到時候被怪物錘翻了，可千萬別出來哭。

然後說破招問題，一百級所有的怪物都可以抓破招了。在超時空漩渦後期，破招機制被刪除了，改成了虛弱增傷，而一百級取消了破防打樁這套玩意，虛弱增傷自然也不能再用了，於是破招機制又來了。還是那句話，破招有百分之二十五的增傷，能打破招盡量打破招，一百級的神話裝備有一條屬性就是提高百分之四十二的破招白字。

可以說，考驗手法考驗走位的時代又回來了。為了提高傷害，你要嘗試打破招，為了躲避機制，你要把怪物控制住，滿地圖亂跑放大狗的怪物也非常猛，一個走位失誤，你就要魂歸西天了。現在很多玩家都在噴沒有操作性，不需要看攻略就能打普雷。

現在一百級的怪物完全避免了這些問題，講道理，算是一個比較理想的時代了吧？不過好不好還是要看更新，希望地下城能有一個好的結果。
