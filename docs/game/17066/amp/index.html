Cp符文系統上線，皮甲套徹底崛起，玩家：策劃太偏心

天天要聞
2019-12-14

國服dnf如今已經是來到了95版本的末期了，對於當前的副本來說，很多玩家都是已經完美畢業，再加上普雷團本的材料除了升級之外好像別的用處也不大，也不像盧克團本那樣材料還可以兌換一些卡片什麼的，所以現在很多玩家都是放棄了普雷團本，都是靜等新版本的到來，準備挑戰能高難度的副本。

而在前段時間，國服的策劃也是爆料了年底版本的一些更新內容，其中最主要的就是魔界副本了，作為100級過渡的副本，魔界在韓服早就上線，因為獨特的技能機制已經強大的怪物能力讓這個團本非常的困難。而對於這麼困難得副本，策劃肯定會給他配上相對應的獎勵才行，於是CP符文系統應運而生。CP符文系統是獨立於裝備的又一個能夠提升角色屬性的系統。

雖然說獨立於裝備，但是符文系統也是以裝備為基礎的，因為符文系統能夠提升裝備的傷害，在玩家的測試裡面，符文系統對於小技能的提升非常的高，而與之對應的，符文變相的提升了皮甲的傷害，所以說等到魔界副本降臨的時候，那些升級皮甲的玩家就非常的舒服了，畢竟皮甲在之前還是非常不吃香的，在這裡我們以劍魂為例吧。

劍魂在沒有得到特色加強之前，一些大神玩家比如說木葉是秉承著重甲套的，他們相信重甲的提升是最高的，所以說對於重甲也是最高打造了。但是劍魂特色加強之後，重甲套對於劍魂來說漸漸沒落，輕甲套開始崛起，因為輕甲可以提升劍魂的一二覺，讓他能夠打出足額的傷害來，很多劍魂玩家都是轉為輕甲。而cp符文系統的到來，對角色小技能傷害提升，讓皮甲套對劍魂更加重要。

在現在的版本裡面，其實擺在大家面前的最大的問題就是裝備獲取太容易了，但是想要好好打造卻非常的困難的，對於全新的魔界版本，紅12應該是門檻，因為之前很多玩家都是選擇本命套，想要換甲升級很顯然很多人都是不捨得的，所以說新版本皮甲應該還是比較的少的。從cp符文系統的上線，我們也能夠看出來策劃對於皮甲的偏愛了，畢竟之前的版本裡面皮甲實在是太沒落了，這一次的加強也是理所應當的事情，就是不知道大家換甲的時候會不會心疼呢？
