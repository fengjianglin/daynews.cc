德杯首日驚現兩個五殺 RNG晉級卻被調侃“太子隊”

天天要聞
2019-12-22

12月21日，一年一度的德瑪西亞杯終於拉開了帷幕。作為冬季轉會期之後的第一個大型比賽，德杯一直以來都是各支隊伍用來練兵的「開學考試」，而且今年官方還對比賽的賽制進行了大幅的修改，原來一打德杯全員替補的情況一去不復返了，包括SN，OMG，TES，EDG，VG等隊伍都祭出了全主力陣容。

另外因為參賽的隊伍較多，為了壓縮比賽的時間，本次德杯的小組賽被切割成了兩個部分，杭州和蘇州兩個線下場館將同時進行比賽，ABCD四個小組的20支隊伍將會在兩天內打完所有的小組賽，所以對於參賽的隊伍來說，他們沒有任何的調整時間，也許某一小局的失利就會提前回家。

而從結果來看，德杯也一如既往的保持了「高爆冷率」的傳統，今天A組和C組的小組賽用一個詞形容的話就是「爆炸」，比賽的過程很爆炸，粉絲的情緒也很爆炸，最終的小組排名同樣很爆炸。

在職業賽事里，五殺一般都是極難出現的，不過沒想到的是，德杯第一天就直接來了倆，而且從某種程度上來講，這兩個五殺其實都和OMG有著密切的關係。

首先是剛回歸OMG的司馬老賊，他在對陣YM時使用輪子媽斬獲本次德杯的第一個五殺，隨後剛剛從OMG轉會到LDL戰隊LGE的「車皇」Chelly又用女槍拿下了第二個五殺…

在一整天的比賽結束之後，A組和C組的排名只能用「有趣」兩個字來形容，賽前被看好的隊伍基本都翻車了，比如蘇寧和LNG。

蘇寧轉會期引入了Sofm，又將二隊的「小TheShy」Bin提拔到了一隊，整體實力大幅提升。但幾場比賽打下來，Sofm狀態起起伏伏，Bin雖然操作不俗，但也犯了新人選手的通病-敢抓敢死。

至於LNG，在官宣Maple入隊時就有很多粉絲不滿隊伍的決定，這一情況在德杯上也得到了體現，Maple和其他隊伍的中單相比的確有著一定的差距，雖然德杯只是一次簡單的測試，來年的LPL春季賽才是真正的戰場，但LNG在小組賽失利之後，回去之後估計要好好思考一下新賽季的陣容搭配了。

當然，單論關注度的話，德杯小組賽首日的人氣王依然是RNG，他們在轉會期同樣進行了引援，YM的打野Xiaolongbao入隊，捎帶手還買了閃電狼的ADC選手Betty。

但從今天的比賽表現來看，RNG雖然艱難晉級了淘汰賽，但這幾名選手的磨合程度顯然還不夠，開局兩連敗直接將自己逼到了懸崖邊上，本就已經備受質疑的小虎依然沒能承擔隊伍的Carry重擔，頂替Uzi出戰的Betty居然意外成為了隊伍最穩定的第二Carry點。

一說起RNG小組賽晉級，估計很多小夥伴都非常的不服，因為RNG在C組三隊同分的情況下直接晉級了，並沒有如觀眾所想的那樣進行加賽。關於RNG「太子隊」的說法甚至鬧上了微博熱搜，但其實從德杯賽前就公布的賽制來看，RNG晉級其實是合理的。

LPL官方在12月16日公布了德瑪西亞杯的比賽規則，在規則的第6部分第2條里，明確提到了同分的勝負判定方式，簡單來說就是：

「三隊同分算比賽獲勝平均時長，LNG最短，RNG第二，OMG最長，所以OMG直接淘汰。本來LNG和RNG應該加賽拼名額，但因為RNG和LNG的比賽獲勝平均時長在150秒以內，所以直接按勝負關係判定，RNG在今天的比賽里贏了LNG，所以RNG晉級。」

說實在的，這個規則確實有些不太合理，但考慮到比賽開始前一周就已經發布了，所以權當LNG和OMG倒霉吧，OMG再多堅持20秒，也許就能晉級了。

在明天，B組和D組的小組賽將繼續進行，IG，EDG，LGD和TES都將陸續上場，希望今天的比賽也能更加精彩吧！
