爐石酒館戰棋：我的眼裡只有魚人，速八與吃雞並存

天天要聞
2019-12-13

●前言

還記得之前寫的文章，當時靠魚人上六千（現在回看裡面有很多紕漏，所以這次的文章也算是補充以及在修改吧。），時隔半個月之後，我又靠魚人打上了六千，這不過這次反向上分。

12月初原本打到7000+的我還自信滿滿的妄圖上個8000。但魚人國王的出現後，讓我突然開始貪，有了夢想……當然結局就是在速八與吃雞兩個選項間徘徊。分數直線下跌，最低掉到過6200

當然，狂掉1000多分的主要原因是因為徹底拒絕其他套路……眼裡只有魚人，無數次死在找魚的路上。也正因如此，積累了一些玩魚人的心得。

●英雄的選擇
如果你不知道該選誰，參照下圖的數據：
◆HSreplay統計的英雄爛分與吃雞率

吃雞能力排名：耐法，拉法姆，銅須，喬治，范克里夫，賣血女巫妖，館長，逐星，尤格薩隆。
爛分能力排名：尤格薩隆，銅須，范克里夫，拉法姆，耐法，館長，逐星，賣血女巫妖，喬治。（賣血女巫妖和喬治爛分率不足50%，需要注意）
不知道選誰，就跟著圖選吧。
●魚人的思路
我們都知道玩魚人的關鍵卡，前期有加血魚，領軍，後期各種BUFF，6星找巨殼龍聖盾劇毒。這些就算是雲玩家也知道，就不重複了……
玩魚人有2個種思路，前期純魚人與中期突然轉魚人。我們重點說從頭到尾純魚人的思路。
◆升星的問題
我理解玩魚人是需要儘可能速上5星（賣血），其他等級不要做太久的停留。
4星也儘可能不去碰……速度上5碰巨殼龍，在5星酒館靠刷找魚人國王和銅須。玩魚人，你要盡量保證你的科技領先於其他人，位於第一批到達5星的人中。下面說原因。
大概升級順序：最近第二回合升二本；第五六回合升三本，7，8回合升4本。
◆為什麼4星不要碰5星？
雖然5星卡中，有魚人國王+銅須2個核心，但廢物同樣多，而且他們對魚人幾乎一點用沒有……然後還浪費你1個對子。
4星碰5，基本只有2個結果，有魚人國王和沒魚人國王。有你胡了，沒有你炸了。

什麼？你說銅須？銅須雖好，但在你碰出銅須的回合基本是對場面沒提升的。然後被血入一回合，然後你下回合你回陷入一個糾結的抉擇，是升本還是找BUFF。

◆選擇升本的結果

因為上回合拿了銅須被血入一波，下回合強升5後就沒費了……大概率被打死。（太賭了）

那麼，4星爆刷找BUFF找碰呢？這個抉擇相當於慢性死亡，因為你會為了維繫場面而不斷找BUFF，血量的壓力有讓你很難花費很多錢升本，就算之後你穩住場面有空升5，但此時你手上的對子也都碰了個七七八八……想碰6星巨殼龍就需要重新湊對子，而強行上6又可能會被血入。
所以，我的建議是血量允許的情況下，賣一波血強上5，下回合在5星刷銅須，刷碰，刷魚人國王，想辦法鎖血吃雞。如果血量不健康……早早轉型吧。
◆4星劇毒魚的問題

前中期，有領軍後，魚人基本都是攻高血少，所以劇毒魚的意義不大，他們已經是物理劇毒了，浪費3塊錢虧節奏。
◆動畫效果
有銅須在場，並且拿到發現魚之後，一定要快快快！！！被動畫效果卡浪費金幣是小事，沒調整位置GG才是關鍵。
◆巨殼龍BUFF選擇
之前的文章中我認為風怒是一個比較坑的技能，但之後幾次實驗，發現風怒還是有價值的……比如對陣機械的時候，風怒魚有更多的機會解掉鈷衛士防止對面無限刷盾，在對面劇毒要幹掉你聖盾嘲諷融合怪時提前幹掉對方。

◆魚人內戰
魚人內戰時，聖盾劇毒之後的BUFF就是亡語，靠亡語的1/1破對面聖盾。這個時候銅須就可以買掉了，狂戰優先，找不到就隨便那個聖盾嘲諷怪放第一個，作用就是2個頻率破聖盾。
魚人體系，大後期高血量的聖盾劇毒魚是無敵的，完美團的套娃也打不過（40+血領軍，40+血融合怪，套娃輸出嚴重不夠）。但魚人成型難，而且會突然暴斃，很多次我35血，然後4星搶上5，被暴捶了20+血。
●後記
所以，強行玩魚人是有風險的，要麼速8要麼吃雞……如果你想上分，最穩妥的方法還是見事不妙，機械爛分……
