為抵抗騰訊圍剿，網易大改火了16年的夢幻西遊，12月18日正式上線

天天要聞
2019-12-18

大家好，我是X博士。

夢幻西遊對於網易來說，是個奇蹟般、且地位不可撼動的存在。

十六年前它駐紮進了許多玩家的青春，成為不少人的啟蒙網遊；十六年後當一票同齡網遊逐漸老去，而它依舊保持著非凡的生命力。

就拿前段時間公布的「10月全球最賺錢遊戲排行榜」來說，電腦版夢幻西遊穩居前三，而手游版則在MMO領域內無可匹敵。

雖然這套「吸金大法」讓無數玩家前仆後繼為其買單，但網易這幾年一直在琢磨著如何才能突破2.5D回合制的局限，讓夢幻西遊這個老IP再度開花。

於是，就有了這款等了一年多、測了N次，終於在12月18日全平台公測的《夢幻西遊三維版》。

劃重點一：回合制到3D即時戰鬥，到底有什麼不同？

要說這個「三維版」跟我們熟悉的夢幻西遊最大的不同，大概就是在視角和戰鬥方式上。

以往2.5D固定視角下看不到的細節、攀爬不了的屋頂房檐，現在在3D自由視角下通通可以實現。

在實時演算的全局光影和動態天氣的作用下，三維版的視覺表現有了質的飛躍，還是那個熟悉的Q版人物，但一顰一犟都更顯靈動。

比如在選角過程中六大角色就在暗暗較勁，一沒被玩家選中就氣到爆炸，跺腳、翻白眼、抱胸甩臉色……什麼小動作都有。

場景也不僅是照搬前作那麼簡單，像是長安城，從方正有序整齊劃一的房屋布局，到喧囂熱鬧的市井街道，復刻的正是歷史上盛唐文化中心的那座城，包容的文化讓玩家甚至能在留香閣看到胡姬月奴的胡旋舞。

而在戰鬥方面，捨棄掉夢幻西遊成名發家的「回合制」、搖身一變成為即時戰鬥模式的《夢幻西遊三維版》，也並非戰法牧那麼簡單。

每個職業都有一個需要進行資源控制的技能點數，比如龍宮的「龍魂」、骨精靈的「律令」，想要打出一套高傷害就得把控好「資源積累和釋放」的節奏。

玩家還能學習「七十二變」來獲得特殊的外形和技能，像是化身鳳凰翱翔於天、變身游魚下水參戰，360度切入戰場；有時刮來的大風和水體阻力還會影響玩家控制方向。

除此之外，幫戰、抓鬼、封妖、掃塔……這些熟悉的玩法雖然被三維版繼承了下來，但在3D的世界裡卻擁有截然不同的戰鬥規則。

比如加入了Roguelite元素的「雁塔地宮」，每次進入新的層級都將面對隨機詞綴BUFF的BOSS和挑戰環境，需要玩家針對時局及時調整對策。

再如「封妖」玩法中多了一份儀式感，玩家必須結成法陣並將妖怪不斷引到降妖陷阱中，才能最終捉拿妖怪。

劃重點二：西遊人物集體大整容？孟婆美得不像話，還能捏出奧利給

在將夢幻三界立體精緻化之餘，《夢幻西遊三維版》還力求為玩家打造一個更自由更開放的世界。

比如開篇創角就是一手黑科技，用動態表情捕捉技術直接抓取你的五官神態特徵，不僅能捏出一個丁老闆，還原時下最具正能量的詛咒巨魔奧利給都不在話下。

可是還沒等到正式公測，這個全新的夢幻三界就被玩家探索了個「底朝天」。

遊戲里一草一木均可互動，像是偶遇飛鳥游魚，這些受驚的小動物會立刻四散逃開；

花果山上的石頭可以撿起來砸桃子，用掉下來的桃子喂猴就能得到「猴子的饋贈」；

邊境村落里還有一隻拉磨的驢，鞭打它就會工作，不過如果打得太急了它還會罷工。

就連曾經德高望重的一代老前輩都趁機整了個容，你看看這還是你認識的孟婆、龜丞相不？

玩家一路遊山玩水，還能結交各種仙魔精怪，這些NPC還會根據你的人氣和好感度對提供實質性的幫助。

比如跟大聖打好關係後，他就會駕著筋斗雲帶你遨遊世界：

可是如果一不小心麻煩太多次了，大聖的火爆脾氣可不是你惹得起的……

而這些所謂人氣值的高低，就取決於玩家平時做出的選擇。

像是面對被惡鬼侵擾的青樓妓子，你是好聲好語安慰呢，還是一副「鋼鐵直男樣」直接動起手來讓她清醒一下？

如果玩家總是選擇過於直接（注孤生）的選項，可能人氣值就會hold不住蹭蹭蹭地掉，這樣走在路上都會遭到路人唾罵，一旦「惡貫滿盈」還會被通緝關牢房。

所以在《夢幻西遊三維版》里，遊戲還是比較鼓勵玩家積小善成大德，比如燒香拜佛、幫老大娘去井裡打水都能漲人氣；但是並不妨礙你想要體驗一下成為惡人的滋味，畢竟故事結局的走向，掌握在你自己的選擇當中。

課後總結：

X博士曾經採訪過一位夢幻西遊的老玩家，「三維版給了我一種重新開始的感覺」，這是他的評價。

雖然這遊戲目前在師門、捉鬼、科舉等等玩法上依舊是熟悉的配方熟悉的味道，但敢於放棄自己堅守已久的回合制轉戰「三維」，也代表著夢幻西遊這個經典IP正再度啟程。
