更新內容

1.第六章主線關卡開放，相關紀念活動開啟

2.博士等級上限提升到120級，理智上限提升到135

3.取消碎石頭的每日上限

4.相關卡池開放，高級憑證去更新資源預載

修復+追加更新

1.白金天賦蓄力攻擊，在攻速被提高的情況下傷害不穩定問題修復，同時略微提升攻擊力（白金加強）

2.所有幹員技能和天賦的凍結變成束縛，只改文字，效果不變

3.更新抵抗描述，效果加強變成：暈眩、寒冷和凍結的持續時間減少50%

4.修復爆鴿索敵邏輯，優先攻擊仇恨值高的幹員（和布置順序以及嘲諷等級有關 ）

5.空一技能對免疫暈眩敵人表現錯誤修復，同時霜星+浮士德免疫水面

6.葦草新增語音，不當啞巴了

7.優化基建內UI式樣和二級排序

8.添加基建幹員進駐後，後勤技能排序

9.訪問好友基建會顯示好友名字，如果沒參與過線索交流，按鈕會高亮。

其他優化

1.公招系統調整（保底四星之類詞條一定要用掉）

2.組合包調整優化

3.遊戲內部UI貼圖布局優化

4.幹員立繪清晰度調整

5.理智葯可多選，默認選擇邏輯優化
