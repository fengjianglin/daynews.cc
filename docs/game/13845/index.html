還能這麼玩？國產廠商開發文字版俄羅斯方塊，但又被噴抄襲

2019-12-12 3,450條評論 4,649次閱讀 3665人點贊

國產遊戲在如今的玩家眼裡，大都是「低質量遊戲」和「撈錢氪金」的代名詞，市場決定了遊戲大多數開發者的目的。但當國產遊戲被噴成為日常的今天，最近在Steam商店上出現了一款創新的益智類遊戲《一字不落》，很多玩家都表示看起來非常有意思！

遊戲是由國內廠商「倉鼠動力」打造，遊戲過程中玩家需要操縱落下的漢字或者筆畫，將其與周圍的漢字組成新的字體，目標則是結合出遊戲規定的漢字。遊戲整體是基於中國漢字筆畫的思路，再由筆畫到漢字的上下結構、左右結構，以此來組成一個個漢字。

比如在這個包圍結構中，「人」和「口」便可以組成「囚」，如圖所示，筆畫會從遊戲上方緩緩落下，玩家需要操控和思考的便是筆畫的落點和落下後與周圍筆畫結合的契合程度。

遊戲從易到難一共有十餘個關卡，越往後進行遊戲，目標字體便會愈加複雜，相對應的結合和玩家的思考也需要增加！

整體來看遊戲的UI和畫風是比較細膩的，不知道是不是因為製作急促的緣故，現在遊戲的內容暫時僅有這一種玩法，稍顯單薄。而且根據介紹里的截圖，遊戲主界面是豎屏，兩邊都存在空白區域，這樣的界面是否更適合手游呢？不過在這種類似於「俄羅斯方塊」的機制下進行創新，遊戲總體看還是很有想法的！

但大家對國產遊戲似乎仍然存在不滿和糾錯的態度，有熟悉遊戲的玩家提出這種文字組合的遊戲在幾年前的日本市場就已經出現過了！這就是抄襲！不過也有玩家表示同一種玩法無可厚非，只要能得到市場的認可就行！目前這款遊戲還沒有正式進行發售，感興趣的玩家可以先行到Steam商店中關注這款遊戲！
