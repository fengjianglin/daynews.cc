國服狼王和北美狼王聯手有多強？狼人殺WPL2019告訴你

2019-12-19 2,864條評論 4,781次閱讀 1294人點贊

由《狼人殺官方》舉辦的狼人殺世界頂級賽事：WPL2019主播邀請賽已正式結束。12月14日、15日連續2天的終極一戰，看得小編在內的不少粉絲大呼過癮。

這次比賽中，不但有像書亞信這樣的明星選手加入，還有有”狼人殺男神”之稱的大衛也強勢入局。

北美狼王、曾在美國伊利諾伊大學學習體育管理專業…一系列的關鍵詞說的正是本次WPL2019主播邀請賽中的大衛。

與此同時，大衛更是全美NCAA橄欖球第一個中文解說，作為首位從體育解說跨界狼人殺的電競選手，大衛還在採訪中表示：對比體育解說，我更喜歡狼人殺。這大概可以說史上是最硬核最有底氣的”不務正業”了啊。

在15號的比賽中，大衛與同門派有”國服狼王”之稱的33聯手出擊，包攬下了當天比賽的雙MVP，分別在帶領好人陣營和狼人陣營中大獲勝利。

其中大衛在拿到預言家的身份後，緊緊抓住警徽，在查殺了5號和12號兩張狼人牌後，再次將警徽給到正確的人，引導好人成功取勝；33就顯得非常狠了，在悍跳預言家的同時，還獲得了眾多好人的好感，狼刀更是”刀刀斃命”，接連屠神（女巫和獵人），輕鬆將比賽拿下。

也正是因為這兩輪新月派選手完美的表現，新月派從總積分上完成了逆轉，從”萬年老二”的排名成功登頂了一波！也意味著這次狼人殺WPL主播邀請賽新月派才是最大贏家，雖然前面都是武林門領先，但是最終的結局，還是讓新月派奪得魁首，讓不少玩家見識到了狼人殺真正的魅力！

因此在這次狼人殺WPL主播邀請賽的總積分榜上，我們可以看出各個門派的積分其實相差有點多的，特別是第一名的新月派和最後一名的錫王殿門派，積分可以用”天差地別”來形容。

當然，其他門派亦有亮眼表現，就在14日第二場的比賽中，JY門派的玩具槍在拿到獵魔人這張神牌的情況下，上演救世主的主角，多次為好人陣營立下功勞，特別是最後一輪發言，以絕妙的邏輯歸死最後一狼，成功拿下了那一場比賽的MVP！

值得一提的是，狼人殺WPL主播邀請賽的比賽雖然已經結束了，但是接下來還會有線下淘汰賽和線下總決賽的比賽，大家可以持續關注《狼人殺官方》WPL2019虎牙直播間，觀看最新比賽直播或回放哦！

雖然這次狼人殺WPL主播邀請賽沒有出現狼王JY的身影，但也說明在本周12月20日~12月22日的線下淘汰賽中，狼王JY極有可能會正式亮相哦！要跟上狼王的邏輯可不是件容易的差事，小編也得抓緊時間上《狼人殺官方》多玩幾把，提高自己的水平了呢！
