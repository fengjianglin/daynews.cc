中國玩家“羽毛”打破《只狼》世界紀錄

2019-12-25 1,295條評論 3,856次閱讀 2444人點贊

中國玩家「羽毛」打破《只狼:影逝二度》速通紀錄，以一秒優勢的艱難奪冠。

《只狼》一直是魂Like遊戲速通玩家們的熱烈追捧的對象，豐富的劇情，酣暢的打擊。之前更是在各大段視頻平台出現身影，影響力度可見之大。這在今年TGA的年度遊戲《只狼:影逝二度》更是如此。

今日，一名來自國內的知名速通玩家「羽毛(LostFeather)以1小時13分44秒的成績，在全boss及全佛珠的條件下通關遊戲，創造了新的世界紀錄！和第二名「Nemz38」的成績相比，僅僅只拉開了一秒。而Nemz38的成績排在之前的榜單第一位的統治時間長達6月之久。

由於《只狼》發售已經有一段時間，遊戲中各種速通的玩法也已經開發的差不多了。因此這看似非常短暫的一秒，其實背後所付出的心血是大量的，無數次技巧的銜接著實不易，「羽毛」的付出值得大家的肯定。

《只狼》確實挺好玩的，尤其是對一些喜歡「自虐」的玩家來說，在1年前面對「義父」的折磨，成為魂系遊戲玩家內心深處無法磨滅的噩夢。

但從傳統意義來說，《只狼》只能說有魂味。但絕不是能代表一款標準的魂系遊戲。從遊戲設計的角度上看，魂系遊戲是由碎片化劇情，炫彩但不奪目的技能特效，精妙的連貫性地圖，豐富的戰術構思，多樣的武器系統，以及魂系遊戲最有代表性的「回合制」戰鬥系統，方方面面完美的組合在一起的，在這個基礎上在發揮構思，無論開發者套上什麼世界觀，他始終都有魂系遊戲的那個味。

但是《只狼》缺少了很多魂系遊戲的標準設計，雖然脫離重心，但也算是一款突破。個人覺得《只狼》最多也只能算是帶有宮崎英高強烈個人特色的動作角色扮演遊戲，但並不阻礙他是一個好遊戲。
