王者榮耀在今日正式上線了蔡文姬福利皮膚，想必小夥伴們已經第一時間入手體驗，同時官方承諾的諸葛亮黑板解謎提示也正式公布，那麼未來都有誰的皮膚上線呢？一起來看看吧！

首先這次填字遊戲的答案採用英文單詞解題，其中豎排提示問題和小編猜測答案如下所示：

  1. 功夫廚神在桌子上擺放的紅色蔬菜，有人覺得也是水果——西紅柿（Tomato）
  2. 莊周最愛做的事情是——睡覺（Sleep）
  3. 里奧梅西的英文名字，非姓氏——Lionel
  4. 常常用來形容刺客遊走抓人的單詞——Gank

橫排提示問題和答案如下所示：

一、國際求救信號——SOS

二、魔法小廚娘最擅長做的美食——Pizza

三、答案還不夠明顯嗎？——SNK

四、大喬和孫策是一對——夫婦（Couple）

五、就是else單詞。

通過以上題目可以得知，所有提示中第三個才是此次填字遊戲的最終答案，而小編把所有單詞加入表格後得出了SNK三個英文字母，對此官方也第一時間予以了確認，即天美終於拿到SNK的正版授權，未來SNK三位英雄將迎來首款皮膚！

其中娜可露露的皮膚已經在此前有所曝光，是一款擁有全屏背景的史詩級皮膚，局內特效以藍黃色為主色調，加入了花燈元素，而橘右京和不知火舞的還未公布，據悉橘右京的新皮膚技能特效已經加入體驗服，也是定位史詩級，因此小編猜測官方不會厚此薄彼，不知火舞的同樣定位史詩，當然最終效果還以正式發布為準！
