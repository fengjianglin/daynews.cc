優秀的時光機器——《領地人生：MMO》

2019-12-16 473條評論 1,408次閱讀 81人點贊

超級時間黑洞，慎入!

【遊戲介紹】

　　《領地人生：MMO(Life is Feudal: MMO)》是一款由Bitbox Ltd.製作並發行的動作冒險網路遊戲。本作可以讓你和成千上萬的其他玩家一起進入一個活生生的、呼吸的、不斷進化的世界。你可以在Abella的土地上創造和影響歷史, 讓這些事件波及永恆。

　　開始你的旅程，踏入一個巨大的 21x21km, 完全現實的世界, 完全與自然的天氣，晝夜循環，和變化的季節。不同地區產生不同數量的區域特定資源，而氣候在整個土地上交替出現，並能在作物的成功生長和自然資源的利用方面發揮重要作用。每個地區都能為勇敢的冒險者提供獨特的優勢和有利的條件。

　　探索一個廣闊的世界：選擇一個適合你簡陋的家，有城牆的城市，或者有堡壘的地方。土地改造成完善的地區後才開始工作。出去結識新朋友, 與鄰居結成聯盟——想像一下與一群人一起生活的可能性。遊戲中不包括開始的城市, 所有的一切都將由玩家們合作建立雄偉的王國和繁榮的社區。

　　王國將會崛起, 王國將會淪陷。如果正確地運用, 筆可以比劍更鋒利。會館可以變城鎮，城鎮變成城市，城市變成王國，橫跨大地。你需要什麼來組成聯盟來捍衛自己的土地和統治Abella?

【評測前言】

　　《領地人生》早在2014年面世時，就被冠上《騎馬與砍殺》精神續作的標籤。這樣一款同樣存在於中世紀背景下，同樣強調真實性與自由度的遊戲，玩家需要做和可以做的事情，似乎也與後者十分相似。它在最初並未引起太大轟動，而其後也因為過於硬核與消耗時間，一直都顯露出小眾的氣質。

　　不過，當一款遊戲從聯機正式走向Online時，許多內容都會因此產生巨大改變。在一個小屋裡和朋友遊玩、建設的感覺，和占土為王，與數以萬計的玩家合縱連橫，在巨大沙盒中馳騁之間的差異，可不是簡單的「遊戲體驗」四個字能夠概括的。於18年初正式MMO化之後，《領地人生：MMO》就以極快的速度在各大直播平台展露端倪，憑藉主播帶動的風潮，這款畫面落伍、UI和玩法似乎都有點過時的遊戲，以全新的姿態顯露在大眾的視野。

【陳舊畫面蓋不住的遊戲性】

　　坦白說，與陷入軍備競賽之中的其它MMORPG不同，《領地人生：MMO》是一款賣相相當不怎麼樣的遊戲。它有著不符合東方人群審美的建模和粗糙僵硬的人物動作，寫實的中世紀風格讓遊戲的畫面在大部分時候都呈現出不那麼討人喜歡的泛灰色調，作為新手的你行走在這片荒野之間，肯定會疑惑一件事：這真的是2018年登場的一款網遊嗎?

　　它也和許多在你上手之後就很快能夠融會貫通的遊戲不同：《領地人生：MMO》是那種就算你跟著教程走完新手教學，也依然會對遊戲玩法和遊戲機制感到一頭霧水的類型。能夠做的事情、互動的物體太多了，世界每一個角落都可以供你盡情探索。在初始的新手島嶼上可以讓你學習到有關遊戲的一切，然而當你自信滿滿的踏上Abella時……

　　是的，你一無所有，如果在初始的荒野上遭遇到野狼，甚至都沒有逃跑之外的其它選項。遊戲從此之後甚至不會給予你任何提示，而是放任你就此嘗試著生存下去。在Abella上遭遇到的任何一位玩家都是潛在的威脅：你當然可以試著向他低姿態的索取一些前期必要的物資，更多的時候，可能來襲的是一次毫無預兆的屠殺。當然，遊戲中的罪惡點數可能會幫助你逃過一劫。

　　這份體驗很少能夠在一款線上遊戲中獲取到。與《H1Z1》的生存模式相同，它擁有相當多的擬真要素：你在之後的歲月中建設起來的營地，最初的雛形一定是你第一個壘起來的小木屋。而在這其中你扮演的角色也不盡相同。鐵匠、木匠、草藥匠、泥瓦匠、農夫，每一個職業傾向還各自具備五個截然不同的分支，將會對其後的遊戲旅途產生巨大影響。無論你是想要和其他玩家組成聯盟，還是享受自己的孤狼領主生涯，都必須在初期展開自己的規劃部署。《領地人生：MMO》更多的展示在玩家眼前的，是極似EVE的那種寬廣的可能性賦予的快感：在這裡，你幾乎能做自己想像的到的一切事情，比如從零開始成為一位領主。

【你能做的事情太多了…一天12小時夠嗎?】

　　平心而論，儘管看起來十分粗糙，但《領地人生：MMO》存在著詳盡的架構和意想不到的豐富系統，甚至可以說，它比任何一款生存遊戲都要複雜得多。

　　在資本的原始積累階段，你需要藉助建立角色時的種族優勢和先天技能點數來獲取到效率的提升。木材、燧石、植物纖維，最原始的武器和工具帶來的是刀耕火種一般的原始體驗。而在這期間，你可能也會面臨探索的困境：例如野外的狼群等危險動物。

　　而我敢保證，這個時期相比之後的階段只會讓你覺得太過短暫。你甚至需要自己將領地的每一片土壤都親手規劃完整並且開始施工，而不是像其他遊戲一般提交一個訂單，隔天就刷新出一座巨大都市來。而在遊戲中期，不僅需要召集同伴來壯大你的野心，還需要提防「審判日」的到來(玩家之間可以隨意攻擊他人領土的日期)，劍與外交都比單機遊戲中的擬真感來得真實得多。真實遊玩《領地人生：MMO》的體驗會讓人覺得，這款遊戲並不需要玩家氪金，它需要氪的是白花花的時間。一旦你沉浸其中，再多的時間也是不夠用的，這大概也是入坑前需要謹慎參考的一點。

　　最後，如果是遊玩過方舟一類遊戲的玩家，在本作的上手應該會相當快，這一類遊戲多少還是有一些共通點的。想要快速成型，還是不能埋頭苦幹，找一個YY公會抱大腿才是正確玩法。

　　當然，在某些方面，本作也是做足了小眾遊戲的調調。目前雖然Steam商店頁面標註了「中文版」，但實際遊戲之中大部分文本還是英文，包括幾乎所有和NPC的對話文本。遊戲目前也只有歐洲、北美兩個伺服器，你必須藉助加速器才能進行遊戲的遊玩。

　　另一點則是遊戲當前的更新過於頻繁，即使藉助加速器，遊戲的更新速度在很多時候也不能夠令人滿意。一款玩不到的遊戲是沒有辦法讓人獲得好的遊戲體驗的。並且遊戲本身的伺服器質量嘛……emmmm，詳見Steam評價頁面。當然，筆者在遊玩的途中倒是沒有被害過幾回。

【評測總結】

你選擇的是一條時空穿梭的道路……在這個遊戲里，你真的能領略到抬頭第二天的恐怖感受。

休息會，掃碼玩玩遊戲吧~
