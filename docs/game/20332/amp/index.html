看看人家CEO的兒子 富二代一路打進決賽還拿了冠軍！你們呢？廢柴

天天要聞
2 天 ago

CFS2019年的世界總決賽於12月15日落下帷幕，冠軍是一支來自巴西CFEL賽區的戰隊，不是去年奪冠BD，而是一支平均年齡在20歲左右的年輕隊伍——VG

這支新星戰隊戰績優異、發揮穩定，決賽賽程中沒有一張地圖失利，讓人無比佩服。從各個其他戰隊的評價來看這是一支個人技能優秀、團隊意識到位、比賽心態穩定的全能戰隊。其中讓大家印象深刻的肯定少不了狙擊手dgz（外號：大公子），這位選手年僅18歲，去年未能來參賽還是由於未成年無法獲得簽證，沒想到2019年剛踏上中國這片土地就和VG隊員們拿到了首個CFS世界冠軍，真是後生可畏啊！

在賽後採訪期間我們了解到隊員們平時並不像我們國內職業戰隊一樣會在一起生活訓練，直到比賽前的兩周，他們才會回到俱樂部里進行準備。雖然在俱樂部訓練時間不長，但VG的隊員們每天的訓練強度都很高，一直在鑽研CF中的各類技巧，不在俱樂部平時也只玩CF儘管平時不強制訓練，但本身的遊戲時間一樣不少。在巴西電子競技的整體收入並不高，主要來自俱樂部發的工資。巴西的職業選手目前也沒有中國職業選手那麼高的社會地位，在巴西，電競選手就是常人眼裡的一個普通職業。雖然這次VG奪冠了，但是在巴西本土他們的粉絲比同樣來自巴西的隊伍BD少的多，小夥子們期待這次回國能有所改善，也是著實可愛。

小插曲，圖中奮力揮拳的小胖子mNds的父親是VG戰隊的CEO，妥妥的富二代好爭氣用實力為戰隊為老爸贏得了榮譽！

CFS2019的賽程結束了，作為落敗的兩支中國隊伍也無需氣餒，積累經驗、穩定心態2020年我們中國賽區再上征程！
