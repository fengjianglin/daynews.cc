王者榮耀：無人倖免，玩家在這3個英雄開大招時，智商至少減一半

天天要聞
4 天 ago

大家好，我是阿涼。

王者榮耀中，能稱得上「上頭」行為的主要有這幾種：

不顧後果往對方人堆裡面沖；

而另一種是看到部分英雄，忍不下心中的怒火強行一換一；

自己發育太好了，一時情不自禁去1打5。

無論哪一種上頭行為，其實都對遊戲勝利沒有太大的幫助。比如有時候不太看對面經濟、等級，看到隊友在附近，就馬上開團，一個人沖對面幾個人，這種情況大概率是含冤而死。因為隊友未必能跟上你的技能，而如果次數多的話，可能結束後還會被舉報送人頭。雖然，你心裡想的跟他們正好相反，那麼好的機會能開到對面幾個英雄，結果隊友不上，他們全是SB。

而有的玩家則對部分英雄有著莫名的仇恨，比如很多玩家在看到魯班的時候，都會安耐不住，眼裡只有魯班，一心只想殺魯班。然後因為團戰發動的時機不對，被對方打了個1換3，局勢瞬間扭轉，變成逆風。

​自己發育太好了，忍不住找對面5個人，這種「上頭」，有另一種解釋，就是浪。而浪跟秀之間，就差一個勝利。贏了就是天秀，輸了就是上頭。

大多數玩家都知道，上頭行為很容易輸掉表賽。一般情況下，還是能用理智戰勝上頭行為。

但是，一旦玩家操作的是這3個英雄，只要開了大招，就很大概率會上頭。理智什麼的都丟掉，智商也不要了，反正開大了，我就一定要衝進去！！

陷陣之志——張飛

說起開了大招，智商就減半的英雄中，一定會有張飛。張飛在沒有大招的情況下，大多數玩家都是很理智的，該給盾給盾，該拉扯拉扯，很多時候一個盾給的及時，帶著親兒子射手2打3，2打4。

但在張飛有大的時候，只要開出來，保護後排？我沒輸出？不可能，我就是要打他們全部。開大後張飛的技能會變成進攻型技能，技能釋放後還會帶來一定的減速效果，一隻沒有出輸出裝的張飛，追著一堆人打是很常見的。

不過，自己眼裡——我控住對面5個；對手眼裡——300塊我收下了。

走位是什麼？——高漸離

高漸離雖然不算特別熱門的中單，但由於他的技能的獨特性，還是被大多數玩家熟知的。他的技能除了1技能是遠程以外，剩下的二技能跟大招都需要近身才能打出傷害。

大招沒0.5秒一次AOE，開啟大招的時候，會減少自己所收到的傷害，假設對方5個人在圈內，可以減少15%+5%*5，也就是40%減傷，同時大招會帶來不錯的加速效果。所以一旦開啟大招的高漸離，就像一個自爆車一樣，沖人堆最好，越多敵人越好。

技能的特殊效果也就讓玩家在開啟大招時，很難控制住自己體內邪惡的力量。

躲什麼技能？對面要躲我好吧！

全都得死——凱

說起凱，很多人都遇到過，自己和隊友在開視野的時候，被草叢一隻凱衝出來天神下凡，刀刀暴擊，然後幾個人沒打死一個鎧。也有很多玩家遇到過，追一隻殘血的凱，突然被反殺幾個。

能抗，能砍，是人們對凱的第一印象。所以在很多時候，玩家玩凱的時候，也通常會想像自己是那個天神下凡的凱爹，無論多少敵人在前面，擋住的都得死！

但事實上，凱的被動是讓他單挑更無敵，大招是格擋傷害而不是無敵。所以悲劇也就由此而來。

每個人都以為自己是砍王，能1打5，但是1打多的前提條件都忘了。每個1打多的凱，都是經過嚴密的計算，對方几個人，有沒有大招，控制技能交了沒？然後決定是不是要衝對面幾個人。

除了這3個，開啟大招就很容易上頭的英雄以外，你覺得還有哪些英雄，玩起來特別容易上頭？歡迎在評論區留言~
