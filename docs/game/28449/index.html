游我推薦：開機甲談戀愛，一次體驗帶給你雙倍的快樂

2019-12-23 1,667條評論 2,898次閱讀 654人點贊

「游我推薦」是遊戲頻道推出的一檔全新欄目，每天，這裡都會向你推薦一款值得下載的好遊戲。關注這個欄目，在不知道玩什麼時，可以從這裡尋找靈感。

從遊戲頻道首頁的「游我推薦」，或本文末尾，可查閱欄目過往內容。

「開機甲還是跟美少女談戀愛，你會如何選擇？」這是網路上流傳的一道經典選擇題，對於廣大的男性同胞而言，這道難題無異於「老婆和老媽同時掉入水中，先救哪一個？」一樣難以作答。開機甲我所欲也，談戀愛亦我所欲也，二者可得兼乎？《櫻花大戰》是也。

時隔十三年的重啟之作

1994年科樂美推出的《心跳回憶》以極高的質量重新定義了戀愛養成遊戲，使得全世界的玩家開始重新審視這一遊戲類型。《心跳回憶》在短短兩年時間內就賣出了一百萬份，讓不少遊戲製作商意識到戀愛養成類型還是蘊藏著巨大潛力的。1996年世嘉將愛戀養成和戰棋玩法相結合，將美少女、機甲、日本寶冢歌劇巧妙地融合在一起，創造出了經典的《櫻花大戰》。

《櫻花大戰》初代

在3D遊戲開始之初，那些低邊行的3D畫面顯然很難與美聯繫在一起。而過去的SFC主機由於機能和解析度的限制，因此還無法完成高畫質的2D遊戲製作。在這個時候，憑藉著SS強大的2D機能，《櫻花大戰》以極高的畫面質量獲得了玩家們的一致好評。該系列也成為了世嘉旗下的一個重要IP。

精緻的2D畫面

隨後《櫻花大戰》系列還登錄了SS、DC、PS2、Wii等家用機平台，GB、PSP、NDS、3DS等掌機平台也不乏《櫻花大戰》系列的身影，連PC平台都推出了不少作品，可謂是全平台開花。《櫻花大戰》系列在2006年迎來了《櫻花大戰5》，這一作的口碑非常差，且沒有中文版，中國地區的人氣也並不高，之後世嘉便逐漸放棄了這個具有20年歷史的經典IP。

《櫻花大戰5》

近幾年電子遊戲領域掀起了一股「重啟」潮流，在《戰神》《生化危機2》《使命召喚：現代戰爭》等作品紛紛走上重啟之路後，《新櫻花大戰》也在粉絲的呼喚中以全新的姿態重新回歸。

《新櫻花大戰》

闊別十三年後復出固然是一件非常值得慶幸的事，但在早期宣傳的過程中，《新櫻花大戰》卻遭到了一些玩家的指責：系列原畫不再是由藤島康介負責，而是邀請了《死神》的創作者久保帶人擔任主要畫師，從《新櫻花大戰》的宣傳圖中就可以很明顯地感受到「死神」的味道，這一改變對於系列老粉絲而言一時間是難以接受的。

明顯的「死神」味道

另一個讓玩家不滿的地方在於，戰鬥部分從偏向策略的戰棋模式改革為偏向動作的無雙模式，這種玩法類型上的改變對於一個系列而言風險是非常高的。世嘉明知如此，卻毅然決然地選擇改變，這使我對《新櫻花大戰》的質量既擔心又好奇。

全新的戰鬥方式

臉紅心跳的戀愛體驗

  • 頻繁互動增強代入感

《新櫻花大戰》的戀愛養成佔比是非常大的，在進入遊戲後的一個多小時的時間裡，玩家都不會碰到任何與戰鬥相關的玩法，給人一種徹徹底底的美少女戀愛遊戲（Galgame）的感覺。但不同於大多數Galgame類型的遊戲，《新櫻花大戰》中的選項是有時間限制的（LIPS系統），玩家必須在極短的時間內做出抉擇。做一個紳士？或是做一個偽君子？亦或是放飛自我，做一個變態？極短的時間更能反映出玩家內心的真正想法。面對內心的拷問，如果玩家真的無法做出抉擇，限定時間一到系統便會默認玩家沉默不語，這也是一種「真實」了。

LIPS限時選擇系統

得益於《新櫻花大戰》從2D轉型為3D，角色的動作與面部表情使得角色的性格刻畫更加飽滿。大老師（比企谷八幡）曾經說過，在人類的交流中語言只佔三成，剩下的七成由眼神、動作和微小舉止來傳遞信息。這句話雖然並不一定準確，但這個道理無疑是正確的。萌妹子的臉紅嬌羞、甩頭傲嬌、怒髮衝冠都在舉手投足間表現地淋漓盡致。配合女友濾鏡更是讓人無法拒絕，坐在電視機前面忍不住想要高呼「媽媽，我戀愛了」。

媽媽，我戀愛了

如同其他Galgame遊戲一樣，不同的選擇會對角色的好感度產生影響，只有好感度達到一定程度，才能夠進入更高級的階段，才有機會進行更多的互動，加深彼此的了解，所以強行尬聊（撩）是非常危險的。萬一真的選擇了比較出格的回答，接下來迎接你的可能就是歡樂搞笑的局面了。

小嘴抹了蜜

當有重要抉擇的時候，就不要放飛自我了，因為玩家選擇模擬量LIPS強弱的不同，會影響故事後續的發展。慷鏘有力的說出「我們一定會拿到冠軍」，女孩子會認為你很有男子氣概；軟弱無力地說出同樣的話，那女孩子還能對你有好感嗎？

氣勢很重要

無論你是把妹達人，還是鋼鐵直男，在《新櫻花大戰》的戀愛模擬部分都能感受到極致的快樂。紙片人老婆固然好，但是3D萌妹子她不香嗎？

是什麼蒙蔽了我的雙眼

  • 大牌聲優為演出助力

光看起來惹人喜歡還是不夠的，萌妹子的聲音同樣非常重要，聲音可以讓一個角色更加深入人心。原本在遊玩之前我是沒有過多關注過《新櫻花大戰》的配音聲優的（Character Voice，簡稱CV），但當進入遊戲後耳朵就不自主地產生了反應，因為這些聲線實在是太有辨識度了。女主角天宮櫻的CV正是動漫《我的青春戀愛物語果然有問題》中一色彩羽的CV佐倉綾音，克拉麗絲的CV同樣是該劇中雪之下雪乃的CV早見沙織。

一色彩羽（黃髮）和雪之下雪乃（黑髮）

動漫《中二病也要談戀愛》中的小鳥游六花的CV內田真禮與凸守早苗的CV上坂堇也加入其中，甚至連釘宮理惠和日本聲優御三家之一的水樹奈奈也都請來了，這個龐大的CV陣容對廣大阿宅們造成了巨大的衝擊，根本控制不住打開錢包的雙手。

豪華聲優陣容（部分）

勉強及格的戰鬥體驗

《新櫻花大戰》除了戀愛之外的另一個組成部分就是戰鬥了，故事的時間設定在《櫻花大戰5》的12年後，「降魔大戰」於太正19年在帝都爆發，帝國華擊團在主人公神山的帶領下展開戰鬥。

戰前會議

每一個角色駕駛一台光武機器人，不同的光武招式各不相同。戰鬥方式與操作方式基本上就是典型「無雙割草」，也就是由輕攻擊與重攻擊組合形成的招式（俗稱C技）。雖然大家早就熟悉了無雙系列核心的C技，但是《新櫻花大戰》卻不知出於何種原因，將輕攻擊設定為圓圈鍵，蓄力攻擊設定為三角鍵，無雙大招設定為方塊鍵。這相當於把無雙系統中的輕攻擊與無雙大招按鍵做了交換，長時間玩無雙系列的玩家形成的肌肉記憶很容易輕易的按方塊鍵，一不小心就把最強的招式給浪費了。

出招表的核心是C技

每次放無雙大招時，還會給角色播一段動畫，這種方式雖然可以增強表現力，但是固定播片看幾次之後就會乏味了，動作遊戲的戰鬥過程中被打斷還是比較影響節奏的，好在玩家可以選擇快速跳過節約時間。

華麗的特寫

光武的動作偏僵硬，打擊感也不是特別好，不知道是不是為了刻意製造出機器人的手感。快速移動和攀爬牆壁這兩個設計做的還不錯，很有科技感。

攀爬牆壁

精確閃避這個系統也不是什麼特別的新設計了，作為發行了《獵天使魔女》的世嘉來說，將魔女時間這一設定引入《新櫻花大戰》也是很容易理解的。

精確閃避

Boss戰還是略有難度的，Boss的攻擊頻率實在是太高了，正面攻擊非常容易被打到。好在場景中有紫色的箱子可以打碎用來回復體力和無雙槽，反覆開無雙打Boss成為了比較安全的過關方式。

組隊Boss戰

很多關卡中都有著重複的素材，處處體現著壓縮成本的理念。有些場景過於空曠，有些平台跳躍沒什麼太多意義，不過《新櫻花大戰》的重點主要還是在劇情方面，也就沒必要對戰鬥方面過於苛責了。從戰棋類轉型為動作類，做成這個樣子也算是勉強合格了吧。

章節評價

精緻的人設與場景

在遊玩《新櫻花大戰》的過程中，絕大部分時間都在跑圖和對話。為了方便玩家實時查看自己的位置和目標，玩家在一開始便獲得了一台智慧通訊器，可以看做是一部功能強大的手機。當玩家不知道接下來該去哪，或者不知道位置時，就可以打開智慧通訊器中的地圖頁簽，圖中綠色箭頭的代表主線任務，藍色的箭頭代表支線任務。玩家可以通過智慧通訊器上的小圖標輕鬆的找到自己需要的信息。

智慧通訊器

有時智慧通訊器會突然響起鈴聲，如果是重要的消息則需要馬上查看，如果是不怎麼重要的消息，稍後在看也是可以的。這讓我想起了《命運石之門》和《凱瑟琳》中同樣具有重要作用的手機。這種接取信息的方式比去固定地點接任務更加靈活，效率也要更高一些。

來電話啦~

跟隨任務的指示，玩家會逐漸掌握遊戲中的所有區域。大帝國劇場可移動範圍最大，細節最為豐富，其他幾個地點如銀座小巷、銀座大街、帝都中央火車站等雖然移動範圍不大，但是每次去都會觸發不同的劇情事件和支線事件，玩一些小遊戲也可以適當地轉化一下心情。不過讓人很不爽的一點是每次進門都會進行黑屏載入，雖然這段時間並不是很長，但是單單往返於幾個地點就要進行數次黑屏，良好的沉浸感都被打破了。

往返於各個場景

仔細觀察可以發現場景中的細節做得還是非常到位的，細心觀察澡堂前的鏡子，可以發現鏡像對稱的細節非常真實，連鏡子中的排風扇也是真的在旋轉，這可比《海賊王：尋秘世界》中的鏡子做得用心多了。

鏡像對稱很到位

鏡子裡面怎麼跟外面都是後背？

遊戲中的主要角色都是將3D模型進行2D渲染處理的，次要角色則採用了手繪的方式，在一些過場動畫的時候會使用更加二次元的方式來進行表現，有一種看動漫的感覺。值得一提的是日本動畫公司SANZIGEN在今年9月份也宣布對《新櫻花大戰》進行動畫化，將於2020年4月開播，希望動畫也能有不錯的表現。

過場動畫

情懷的傳承

《新櫻花大戰》作為時隔多年的重啟之作，與一般系列遊戲的續作相比有著不同的意義。這個老牌IP曾風光無限，也曾黯然離去。這次的回歸不僅是給《櫻花大戰》粉絲們的一個交代，也是世嘉給自己一個機會。如果這一作失敗了，這個IP可能會被繼續雪藏，但若是成功了，一定會同《戰神》重啟一樣重新開闢新的系列。

《新櫻花大戰》是一款慢熱型的粉絲向作品，它並不完美，但它足夠讓老玩家淚目，讓新玩家感受到太正浪漫的美好。希望《新櫻花大戰》能為接下來的作品開個好頭，讓這個系列重新煥發生機。
