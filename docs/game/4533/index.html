微博杯：4am狀態回歸殺入決賽，XDD以45殺獲小組賽淘汰榜第一

微博杯S3的小組賽已經結束，32支隊伍經過為期六天的循環賽，決出了最終積分前16的隊伍，晉級決賽。4am在前兩輪倒數的情況下，憑藉今日68分的高積分成功闖入決賽！

4am在本次微博杯的前期表現並不是很好，兩輪比賽過後僅獲得43分，排在倒數第二。而今日最後一日的比賽中，4am狀態回歸，第三局以16殺吃雞就追了一大波分，來到了晉級的門檻位置，後幾局穩定得分後，以111分第12，成功殺入決賽！

​

其實大家都以為本次微博杯4am沒戲了，沒想到4am追分能力這麼強，網友也吐槽4am之前是在演，今天算是攤牌了，畢竟世界第三。但前幾日隊員確實狀態不佳，好在最後一天都回來了，而且韋神本次微博杯的表現可以說是出乎意料的亮眼。

本次微博杯也是XDD成年以來的首次比賽，RNG以168分第二的位置晉級，可以說XDD的首秀十分完美。XDD也以單人45殺獲得了淘汰榜第一，在職業賽場證明了自己的實力，真是少年可畏！

本次SMG戰隊也是在最後幾局獲得不錯的分數後，才確定晉級決賽，本次SMG的陣容可以說大變了，Semllcc和Sixgod的加入獲得了極好的效果，可以說是穩定大腿了。而OMG卻再次無緣決賽，最後兩輪重新換上BT後仍是效果一般，最後排在了第29名，倒數第4。

決賽將在8號進行，晉級的隊伍實力都十分強，特別是Ark和RNG，小組賽拉了後面隊伍不少分，很有機會爭冠，讓我們期待決賽16支隊伍的發揮吧！
