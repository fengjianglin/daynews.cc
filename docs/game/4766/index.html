裝備堆砌流武將的一生之敵，裝備只能在自己手上不超過一回合

2019-12-05 0條評論 19次閱讀 0人點贊

摘要

大家好，我是靈運。今天我要介紹的遊戲是中國桌游三國殺。原創不易，喜歡請點個雙擊么么噠！

大家好，我是靈運。今天我要介紹的遊戲是中國桌游三國殺。原創不易，喜歡請點個雙擊么么噠！

三國殺的武將技能有一個特點，發動越難的收益往往越大，可以升級和通過積累一些特定東西的技能強度都很不錯，凌操和賀齊就是裝備技能堆砌型的代表，通過裝備來提升技能強度，滿裝備的凌操和賀齊可以躋身一線武將強度，但是遇到他們這局遊戲就算廢了。

凌操和賀齊裝備可以逆天？這四個武將教你做人

襲人於不意，潰敵於無形。如果經常打排位的朋友相當熟悉這句話，被卡距離的武將會被他煩到不行。這句台詞是劉封發動”陷嗣”的時候說的，準備階段，你可以將至多兩名角色的各一張牌置於武將牌上，稱為”逆”。這對一般武將來說跟張遼相似，只不過是少一張牌而已，但是對於凌操來說是致命的剋星。張遼的”突襲”只是針對手牌，凌操本來就一過牌量大，摸牌多出名，根本不怕他。但是劉封可沒有隻能拿手牌的限制，凌操的強度全依賴裝備牌給他帶來更多的手牌，而劉封傷他根本，每回合都抽掉一張裝備，即使有”逆”的副作用，那也穩穩是賺的。凌操要是遇見了劉封，就只能當一個四血的”英姿”玩了。

用兵之道，變化萬千。張郃是三國武將中特別招仇恨的一個，一個技能四種效果，三種都是坑人的。你可以棄置一張手牌並跳過一個階段，就會不同的效果。使用最多的跳過摸牌階段和出牌階段，跳過摸牌階段，棄置一張手牌，技能效果和使用方式和張遼的”突襲”完全相同，不同的是需要先棄置一張手牌，所以相對於張遼來說張郃更加缺手牌。讓賀齊恨得牙痒痒得是他”巧變”的第三種，跳過出牌階段，可以將場上任意一張牌移到合理的位置，被移動的牌可以是已經使用等待判定的錦囊也可以是其他角色裝備區的裝備牌。這對賀齊來說是天大的打擊，一種裝備花色意味著多一種技能，照張郃既抽手牌又拿裝備的手段來說，一局下來都積累不下來一件裝備，四血五技能更是無從說起。

吾意不在此地，已遣別部出發。作為我們的排位靠前的頭幾把交椅的王基，憑藉著出色的過牌能力和百搭性，一直都是勝率高升，但是克制凌操和賀齊的都是拆遷流武將，跟王基有什麼關係？王基的”奇制”不單單能提高自己的過牌能力還可以破壞對手的手牌結構和裝備。當你於回合內使用基本牌或錦囊牌指定目標後，你可以棄置不是此牌目標的一名角色一張牌，然後該角色摸一張牌。如果這”奇制”棄置的是凌操的裝備牌，雖然一棄一摸，表面上沒有什麼損失，但是凌操想死的心都有了，一張未知的手牌怎麼比的過能讓自己額外摸牌的裝備呢？

化守為攻，出奇制勝。能移動場上的裝備牌算是張郃的獨門絕技，但是不小心被界曹仁給偷學去了，而界曹仁在排位賽場上的風頭可以張郃厲害的多，”解圍”你從背面翻回正面時，你可以棄置一張手牌，移動上場的一張牌，如果場上有賀齊，那麼有什麼比偷走他的裝備牌更有趣的事情呢？不過”解圍”得配合”據守”使用，兩個回合才能移動一次裝備，偷人家東西的效率是沒有張郃這個做師傅的高，但也足夠讓賀齊恨得牙痒痒了。

本期凌操和賀齊的一生之敵內容就到這裡了，還有誰能剋制他們兩個呢？在留言區說說吧，掰掰！
