熱血傳奇：難纏的小怪物，被纏上就是浪費時間，據說還能爆麻痹戒指

熱血傳奇憑藉著新穎的玩法以及在當時來說還算炫酷的遊戲體驗，在本世紀初已經問世便迅速火遍全國，無數熱血青年沉迷在光怪陸離的遊戲世界中。

熱血傳奇著款遊戲，本質上就是玩家刷怪練級，獲取各種各樣的稀缺裝備。所以在遊戲中，無數的怪物便是玩家的首要敵人，在遊戲中怪物的有著不同的攻擊方式甚至有著不用的隱藏技能，擊殺怪物便能隨機獲得不同品質的裝備。而在這些怪物中，有著一些非常噁心而且難纏的存在，基本上遇到這些怪物，為了不浪費時間，只能甩開大腿跑了。今天就給大家盤點一下遊戲中這幾個然玩家頭疼且難纏的怪物。

楔蛾這個怪物相信大家並不陌生，絕大多數玩家都被它噁心過。楔蛾在遊戲中算是比較平常的怪物，在各大地圖中都有著它的身影，儘管它沒有超高的攻擊力，但是他的攻擊會給玩家造成眩暈效果，在遊戲初期，玩家刷圖練級時，遇到楔蛾的攻擊可謂真的是頭疼，畢竟楔蛾是一種非常普通的怪物，即便擊殺它丫不會獲得什麼好的裝備。但它動不動就眩暈你，這真的是太浪費時間了，畢竟，玩遊戲還是要花點卡的。不過，坊間有傳聞說楔蛾有一定幾率可以爆出麻痹戒指，不管是真是假，這種幾率怕是低之又低吧。當時還特地花了點時間專門刷楔蛾，但是啥都沒有獲得，得不償失。

在祖瑪寺廟中分布最為廣泛的要數角鷹了，角鷹同樣是沒有太大的攻擊力，但是它卻可以召喚另一種怪物蝙蝠，如果任由它召喚，不管不顧的話，那蝙蝠便會充斥在整個地圖中，跑圖的話那真的是太噁心人了。所以遇上角鷹，還是應該第一時間擊殺它。

另外一個噁心人的怪物便是黑色惡蛆，之所以把它拿出來說一說，不是因為它有多高的攻擊，只是因為這怪物基本上黏上你你就跑不掉了，移動速度太高了，黏上只能擊殺它，可惜的是擊殺它可能連藥品都賺不回來，真的是即難纏又沒用的一種存在。

赤月中的月魔蜘蛛，超高的血量以及超高的移速，再加上變態的被動——眩暈，使得它的難纏程度更高。相較於上面說的幾種難纏的怪物而言，在遇到月魔蜘蛛時，可得更加小心了，稍有不慎，可能會被它眩暈到直接回城，所以，遇到月魔蜘蛛的話，千萬不要手賤去招惹它。
