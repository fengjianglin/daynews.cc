魔獸世界：如果希女王帶領亡靈進本，那聯盟和部落會和解嗎？

2019-12-17 1,956條評論 4,076次閱讀 600人點贊

在魔獸世界的劇情故事中，亡靈從來就不是追隨部落只是跟隨希女王加入了部落大家庭。亡靈唯一的效忠對象就是希女王，所以劇情動畫里親兵用懷疑的眼神看了女王一眼，這種事是不可能存在的。女王去哪它們就去哪，它們本身就是被遺忘者團體。很多人當初玩被遺忘者口口聲聲是對不死族的喜愛，結果被遺忘者的意志一個削弱幽暗城從蝙蝠都刷不出來很快降到正常人口，就能看出玩家到底什麼態度了。而關於希爾瓦娜斯之後的走向，就只有看暴雪怎麼安排了。

首先劇情需要，暴雪讓洛丹倫丟那就必須丟，就像達納蘇斯樹都燒了。再說血精靈經過凱爾薩斯王子幾波腦殘操作，元氣大傷，天災軍團北撤剩下的那點兒勢力真的不夠看。何況對聯盟來說，曾經洛丹倫王國是人類七國之首，收復失地而已，何況部落聯盟這兩版本齷齪不斷，矛盾各種激化打一仗很正常。

先找一個對方陣營里的異見者，然後帶著大軍去和對方領袖「單挑」。然後把對方內部能幹的，有領導力的全趕走的趕走，整死的整死。扶持一個不知道哪冒出來的傢伙當新領導人，美其名曰獨立中立。然後滲透親自己的勢力，逐步控制各個方面。暴雪早就說後悔中立種族了，劇情不好展開7.0乾脆軍團把熊貓人所有門派都擊敗，所有殘部新建了個殘寺派，就這麼休養生息可以一直休養到魔獸關服。

我覺得以後會在部落和聯盟里再分別分出鷹派和鴿派，部落和聯盟的鴿派可以正常交流組隊下副本，但是所有pvp收益減半。雙方鷹派分別由希爾瓦娜斯和泰蘭德領導，鷹派不能和對方陣營交流組隊，但是pvp收益加倍。鴿派和鷹派一旦選定，需要刷特定聲望一個星期以上才能轉換。所以以後不會再有陣營戰爭了，聯盟已經入駐了部落，還成為了部落的種族首領。不過同樣的，聯盟似乎也不大可能會有「光鑄亡靈」這個種族了，反倒部落亡靈可能能選擇聖光亡靈的外觀。

就憑米奈希爾這個姓公主才是洛丹倫正統繼位者（她兄弟已經涼透了順位也該輪到她），而且公主給被遺忘者帶來了聖光，這也符合舊洛丹倫人類王國的白銀之手騎士團，烏瑟爾泉下有知甚感欣慰，從此以後光鑄亡靈聖騎士，光鑄惡魔聖騎士，光鑄德萊尼聖騎士加入圖拉揚聖光軍團聖光繼續忽悠著眾生。你們覺得呢？
