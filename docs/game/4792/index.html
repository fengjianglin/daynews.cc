過氣的“紅白機”，又來顛覆行業了

2019-12-05 0條評論 17次閱讀 0人點贊

摘要

不知道70、80、90年代的小夥伴們還記不記得這個小東西呢？

沒錯，它就是由任天堂發行的第一代家用遊戲機——FC，我們一般喜歡叫它紅白機（貌似我國都是盜版的）。

那個沒有智能手機、節目匱乏的年少時光，因花花綠綠的像素而增色不少。而作為家用遊戲機鼻祖，紅白機遊戲成為了許多人童年不可缺少的組成部分。

那時我們經常叫上三五好友，藏在家中，偷偷打開電視機連接好遊戲，然後就是愉快的一點。

隨著我們一天天長大，紅白機也漸漸淡出了我們的生活，取代它的是經常的電視節目、更好玩的電腦遊戲、多功能的手機等等。

雖然紅白機消失了，但是任天堂還在，我們的信仰還在。有人說任天堂就像nai子，說是為了孩子在玩的都是大人。最近幾年的遊戲產品可謂是賺足了男同胞們的錢。

在17年的時候任天堂又推出了遊戲機 Switch，遺憾的是一直未提供中文系統，所以國內玩家只能「盲玩」，吃「生肉」。但這依然阻擋不住大家都任天堂的熱情。

在前兩天的騰訊 Nintendo Switch 的媒體見面會上，國行版 Switch 終於來了，任式好遊戲也來了！

今天騰訊主題為「為你的笑容而來」的國行Nintendo Switch產品上市發布會，在上海正式舉辦。也預示著國行版 Switch 正式開售，遊戲陣容方面，未來數周內，Nintendo Switch上的精品馬力歐相關遊戲會陸續與中國用戶見面。

不知道大家喜歡紅白機時代的什麼遊戲呢？超級瑪麗？坦克大戰？雪人兄弟？還是魂斗羅、雙截龍、炸彈人呢？

在評論區分享下你玩過遊戲吧，另外別忘了給我點一個關注哦。
