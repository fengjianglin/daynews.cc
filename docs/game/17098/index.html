LOL賈克斯新皮膚，每次擊殺都能升級皮膚特效，特效趕超未來戰士

2019-12-13 178條評論 2,110次閱讀 718人點贊

LOL在S10季前賽推出了首款成長型皮膚，這款皮膚的推出意味著英雄聯盟的新皮膚將更加具有趣味性。賈克斯作為LOL中的熱門英雄一直備受推崇，先前的武神皮膚更是贏得了不少小夥伴的讚許，爽快地打擊感讓武神皮膚成了賈克斯的入門必備皮膚，但這次推出的王國機神或許將超越武神！

賈克斯（武神皮膚）

賈克斯的武神皮膚是賈克斯的T1級皮膚，也就是賈克斯最好的皮膚，不管是皮膚打擊感還是皮膚的特效，這款皮膚都做的非常完美，只不過玩家對於皮膚的追求向來是永無止境，雖然這款皮膚足夠完美，但因為賈克斯持續熱門，導致拳頭不得不做了一款全新的賈克斯皮膚，畢竟道理大家都懂的！

賈克斯王國機神

而這次賈克斯的新皮膚「王國機神」跟其他皮膚完全不是一個風格，你甚至可以理解他為另類電玩皮膚，這款新皮膚的特效實在太酷炫了，幾乎每一個動作都進行了特效升級，甚至開大和E技能的特效完全達到了未來戰士的水平，但這款皮膚也有一個缺點，那就是打擊感遠不如武神皮膚，只能說特效做足了，但打擊感還是不夠。

可成長型皮膚設定

當然，作為一款新皮膚，王國機神也用上了全新的皮膚設定，當賈克斯每次擊殺敵人時，他頭上的數字都會實時顯示擊殺敵方英雄數量，這個數量最高可以顯示到25，而當擊殺25個敵人之後，賈克斯頭上的小屏幕會顯示GG，這個是不是非常嘲諷呢？當然，一旦讓賈克斯拿到了25個人頭，基本上也不需要繼續掙扎了！

賈克斯

這款皮膚的成長進化不僅僅體現在賈克斯頭頂的面板之上，還體現在賈克斯的後背。當賈克斯升到六級時，賈克斯會背生雙翼，配合賈克斯的Q技能，甚至可以感覺到賈克斯真的飛起來了！不得不說，拳頭在製作皮膚上是越來越用心了，相信成長型皮膚會逐漸成為皮膚製作的主流，之後的大多數皮膚都會應用成長型設計，畢竟這樣的皮膚在使用中更具觀賞性！

網友評論

當然，有網友對於皮膚並不買賬，甚至認為拳頭在S賽之後皮膚更新速度實在太快了。對於這一點，飯飯君只想說，拳頭並不是一家慈善公司，雖然拳頭偶爾也會做慈善皮膚，但絕大部分時間拳頭還是要吃飯的，對於皮膚這種非必需品，有玩家有這樣的需求，他們也有能力買皮膚愉悅自己，而對於普通玩家，出多少皮膚對於他們來說並沒有區別，再者說，皮膚並不影響遊戲的平衡性，所以，拳頭出皮膚完全是他的自由。你認為這款賈克斯的新皮膚會受到追捧嗎？期待你在下方評論區留言討論喲！
