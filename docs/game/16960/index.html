打野並不是只有大神才能玩，只要學會了技巧，你也可以成為大神

2019-12-15 2,652條評論 3,956次閱讀 1454人點贊

打野這個位置雖然在S17賽季里並不是最重要的一個位置，但卻是前期能做最多事情的一個位置，我們做的事情多，那麼對勝負的影響就大，而一旦我們發揮得好，那麼也就更容易贏，不過很多玩家都以為打野只有大神才能玩，但事實上打野這個位置只要掌握了一些技巧，每個人都可以成為別人眼中的大神。

要想玩好打野這個位置，我們就需要對版本的趨勢有一定的了解，打野是一個前期可以帶動節奏滾雪球的位置，在S17賽季里，打野主要是圍繞著發育路來進行遊戲，發育路的經濟要比別的地方高很多，一旦崩了，可能中後期就沒得玩，而一旦打出優勢的話，那麼勝利也就唾手可得了。

在當下的版本里，遊戲節奏是非常快的，但是與之格格不如的就是對抗路，兩個坦克英雄可能打半天都沒有任何的全屏提示，十多分鐘的時候雙方的防禦塔還是滿血狀態，所以我們甚至可以忽視對抗路，讓他們自己慢慢發育，只要把射手養起來了，中期的輸出根本沒人頂得住，在對版本有一定的了解之後，我們再來看看打野的一些小技巧。

Ban位的選擇，相信大部分的玩家都已經是鑽石段位以上的了，在真正進入王者峽谷之前都需要先Ban英雄，作為打野，我們最怕的並不是射手或者法師，而是輔助，輔助在遊戲里的主要任務就是保護後排，而打野一般都是以切後排的為主，如果我們已經想好要玩哪一個打野英雄的話，那麼就要儘可能地將我們自己害怕的輔助給Ban掉，這樣在真正進入到王者峽谷之後，切後排的難度會降低很多，而不是隨便Ban掉一些英雄。

其次就是英雄的選擇，相信大部分玩家如果喜歡某一個位置的話，那麼肯定對該位置的大部分英雄都

很熟悉，而英雄的選擇由兩個很關鍵的因素決定。

1：對方以及己方的陣容，這款遊戲並非一個人的遊戲，這一點大家段位越高就越能感覺到，根據陣容來挑選英雄才是最合理的，如果對方陣容都是一些比較靈活的英雄，那麼選擇典韋打野的話可能都碰不到對手，更別說切後排，打出高額的輸出了。

2：版本強勢的英雄，在每個版本里都有一些英雄會比較厲害，並不需要有直接的加強，當同一個位置的英雄都被削弱的時候，那麼沒有削弱的英雄其實就是加強，每個版本都有一些英雄是版本之子，就算我們玩膩了這些英雄，但是為了上分的話，還是要盡量選擇他們，當選出來的時候，其實就已經有了天然的優勢。

打野路線，一般來說很多玩家都喜歡藍開，刷到四級然後開始抓人，但是這種路線其實有時候並不是最合理的，在這個賽季里，發育路和對抗路的區別實在是太大了， 抓對抗路的難度在前期雖然不大，但是很難滾起雪球，實際的作用並不大，而且紅藍方也有很大的關係，發育路並不一定會是下路，所以我們要根據進入到遊戲之後來判斷打野路線，一般選擇紅藍開，要以對抗路為目標，發育路在哪我們就對應什麼buff開局，這樣到達四級之後就可以抓射手所在的發育路。

不過還有一種另外的打野路線，不過要滿足一個條件，那就是發育路的射手是能抗壓的射手，而且是事先說好的，打野會比較少去，這時候我們就可以選擇和小龍相反方向的buff開局，這樣的路線可以在到達四級的時候尋找機會開龍，給整個團隊帶來優勢，不過如果是單排的話，那麼還是盡量以發育路為主比較好，這樣的話勝率會比較高，哪怕射手我們覺得他們玩得很一般，但發育路和射手就是版本最重要的一個位置，就算是菜鳥也要扶著它起飛。

最後就是要如何打出優勢了，這可能是大家最關心的地方，平時可能大家都有玩過打野這個位置，但是打得很一般，感覺就算抓人也抓不出什麼效果，單刷野的話，自己的發育可能比輔助還要差，這時候別說切後排了，不被秒就已經不錯了，但這卻是很多打野玩家的現狀，我們的除了要學會上面的這些小技巧以外，還要學會如何打出優勢，如何滾雪球。

而對於打野來說，擴大優勢最好的方法之一就是反野的了。

反野一般可以分為兩種，一種是抓完人之後和隊友一起反野，一種則是獨自反野。

第一種是比較常見的反野方式，在抓完中路後，中路附近的位置對方的戰鬥力會有很大的下降，這時候我們就可以和法師以及輔助進行反野，最少可以反掉三隻小野豬，而邊路的話就是反buff或者蜥蜴和飛鳥。

而獨自反野則比較危險，在沒有抓人，以及只有自己的情況下進行反野很容易被對手留住，想要獨自反野就需要學會觀察好小地圖，如果看到對手的打野露頭的話，那麼就可以在相反的野區進行反野，但是要記住一點，只要被發現就要撤退，切記不能過於貪心，不然很容易偷雞不成蝕把米。

除了反野以外，在當下的版本里，打野還需要學會如何合理地蹭兵線，可能蹭兵線在之前的版本里是一種不道德的行為，但是在S17賽季如果不蹭兵線的話，那麼打野就算數據再好，經濟也很少能領先對手，特別是對於韓信，李白，孫悟空這些打野英雄來說，如果不蹭兵線的話，是很難carry的，蹭兵線已經成為打野發育的一種合理的方式之一，但是蹭兵線卻要有一定的條件才行。

在蹭兵線的時候，必須要有線上英雄在場，兩個人一起分享兵線所帶來的金幣和經驗，而這樣也會讓兵線的價值最大化，比一個人收益更高，但是超過兩個人的話，那麼收益會效果會越來越低，畢竟不是的每一個位置都可以蹭兵線的。

當我們掌握了以上的這些技巧之後，那麼相信大家的打野也會更上一層樓，成為別人眼中的大神玩家。
