PS部門老大讚揚PSV 然而索尼已放棄掌機業務

摘要

很多玩家希望索尼再次進入掌機市場，推出PSV的後繼掌上遊戲機，以隨時隨地遊玩像是《漫威蜘蛛俠》、《戰神》、《神秘海域》這些索尼第一方遊戲。然而，壞消息是沒有可能了。

很多玩家希望索尼再次進入掌機市場，推出PSV的後繼掌上遊戲機，以隨時隨地遊玩像是《漫威蜘蛛俠》、《戰神》、《神秘海域》這些索尼第一方遊戲。然而，壞消息是沒有可能了。

索尼的首款掌上遊戲機PSP在當時那個手機欠發達的年代成為了時尚電子產品的象徵，也得到了大量用戶的喜愛，雖然最終銷量沒有超過對手任天堂的NDS，不過仍然取得了十分優秀的成績。然而，在後續的PSV卻遭到了極大的失敗，在同3DS的競爭中早早敗下陣來。憑藉手裡的多款重磅遊戲IP，加上強勁的主機性能，索尼能否重整旗鼓，再次與任天堂在掌機遊戲市場正面對決呢？

在接受Game Informer採訪時，索尼旗下PS部門總裁Jim Ryan回應稱，索尼對推出全新的掌機並不感興趣，而且指出掌機已經不再是索尼的業務。不過他本人也表示自己是數百名認為PSV很棒的人群之一。

Jim Ryan說：「PlayStation Vita在很多方面都很出色，遊戲體驗也很好，但顯然我們現在已經不再涉足這一領域了。」

話雖然這麼說，也並不意味著今後索尼就絕對不會推出新的掌上遊戲機設備。目前這家公司正在全力準備明年推出的新一代家用遊戲機PS5，自然無暇對待掌機市場佔據優勢的任天堂。考慮到2011年推出的PSV最終僅賣出1500萬台，並且缺少足夠的遊戲支撐，如果索尼真的要重新回歸掌機市場，肯定有大量的工作要準備。如果現在你想要玩掌機遊戲的話，任天堂的Switch只能是唯一的選擇。
