2019年11月Steam熱銷20大遊戲出爐 EA和微軟均上榜

2019-12-18 663條評論 2,369次閱讀 598人點贊

每月中旬，V社都會發布Steam上個月銷量最高的作品，這是基於遊戲產生的收入而不是發售頭兩周的銷量。這也是為什麼這份榜單直到月中才出爐，因為月底發售的遊戲需要兩周的時間才能統計銷售額。有鑒於此，V社通常不公開實際的銷售額，而以下榜單是採用發售日順序排列。

這份榜單中，11月15日發售的《星球大戰絕地：隕落的武士團》和《帝國時代2：終極版》都有上榜。看起來微軟和EA都在將他們的遊戲帶到Steam平台上取得了成功。

1.《Ratopolis》

2.《動物園之星》

3.《Pistol Whip》

4.《浪漫沙加3》

5.《Kritika:Reboot》

6.《統一指揮2（Unity of Command 2）》

7.《損友傳奇（The Legend of Bum-Bo）》

8.《帝國時代2：終極版》

9.《指揮：現代作戰（Command: Modern Operations）》

10.《終結者：反抗軍》

12.《熾焰帝國2》

13.《足球經理2020》

14.《自動化帝國》
