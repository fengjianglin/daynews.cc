fgo新學妹午茶禮裝要不要換 比肩寶石翁的人權卡 避免被大佬拔毛

天天要聞
2019-12-06

摘要

對於fgo近期實裝的學妹午茶禮裝，因為其本身高額的成本，不少玩家在猶豫到底要不要換，這裡帶來比較詳細的分析，一起來看看吧。

對於fgo近期實裝的學妹午茶禮裝，因為其本身高額的成本，不少玩家在猶豫到底要不要換，這裡帶來比較詳細的分析，一起來看看吧。

本次fgo國服藉助1500DL系列活動，實裝了迦勒底午茶時光這個新禮裝，因為其高昂的兌換成本，足足需要5000綠方塊，還有其對於自己幫助不大，對於好友幫助極高的效果，讓部分玩家在琢磨到底要不要更換的問題。

禮裝效果本身是自己使用的時候可以獲得5%的羈絆，但是好友用的時候就能獲得15%，也就是說這個禮裝是典型的福利好友的，對於自己的幫助似乎並不是很大，但實際上並非如此，畢竟助戰系統本身是就是互助用的，凡是非活動時期，對於禮裝的選擇，大部分玩家都樂意去攜帶帶有羈絆加成的禮裝，畢竟從者的羈絆非常難刷，但收益很高，除了聖晶石還有作為離婚證的專屬羈絆禮裝可以獲得。

所以這個禮裝看似對自己幫助不大但對好友收益極高的價值就體現出來了，這是很好地一個防止大佬好友拔毛的禮裝。作為BOX比較充裕的玩家來說，他們在乎的很多時候不是你的從者能否幫上忙，而是你的禮裝是否能讓他得到不錯的收益，他們拔毛好友的標準除了看從者寶具等級之外，就是看禮裝了，所以這個禮裝最大的價值就是緊緊抱住大佬的大腿防止被其拔毛，順帶還能獲得不錯的友情點，且也能和之前的午餐禮裝一起使用。

基於這樣的地位和作用，這玩意可真的是不輸寶石翁的人權卡，畢竟一個大佬好友提供的幫助，可遠遠高於幾個滿破五星禮裝。

部分玩家會面臨一個很現實的情況，那就是綠方塊不夠怎麼辦，這種情況下優先換自己需要的資源，這個可以延後，如果綠方塊不夠可以理解，該禮裝兌換開放時間一直持續到了聖誕，到時候稍微肝一下就能獲得大量的綠方塊。當然如果你實在是著急，可以考慮友情點消耗一波，連搓丸子帶抽金卡狗糧一氣呵成。
