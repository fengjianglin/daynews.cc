2020年歐美取消聯賽 PCL10號開啟轉會 4月柏林PGS賽事4AM獲直邀

2019-12-11 146條評論 1,865次閱讀 939人點贊

昨日PCL賽區12月份的第一個杯賽「第三屆WEGL微博杯」正式落下帷幕，RNG戰隊以164分奪下總冠軍，XDD左梓軒_Xleft 斬獲73個人頭，奪得了微博杯的殺人王。

12月11號開始每晚17點我們將迎來第二個杯賽「鬥魚黃金大獎賽S8」，今晚開啟的是主播線上揭幕戰

昨日晚間PUBG賽事官方公布了2020年的賽事計劃，我們來看看有哪一方面的內容：

  1. 2020年 將會舉辦4場高規格的PGS國際賽（4月、7月、10月和11月PGC全球總決賽）第一場PGS國際賽將於2020年2月開啟選拔，4月份柏林開戰（2019PGC前四自動獲得直邀名額）

Gen.G、Faze、4AM、ENF這4支戰隊，獲得直邀名額。

2. PCL賽區的轉會窗口開啟（12月10日–1月10日）

3. 2020年開始鑒於歐洲、北美戰隊減少，觀眾玩家減少，PEL、NPL賽區將會取消此前冗長的聯賽賽制，由原先的杯賽取代。藍洞此舉，受到歐美廣泛好評。
