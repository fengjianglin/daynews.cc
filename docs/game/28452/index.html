DOTA2 - 茶隊中單ChYuan採訪：希望能成為Sumail一樣的選手

2019-12-23 2,087條評論 3,109次閱讀 140人點贊

ChYuan是來自馬來西亞的Aster選手，20歲的他已經成為了最年輕的職業選手之一。gameishard.gg在新加坡One Esports國際職業邀請賽期間採訪了他，聊了聊東南亞選手在中國的情況。

你好ChYuan，你已經在DOTA2職業圈打了兩年了，你覺得遇到最難的事情是什麼？

當我在東南亞的時候我總是沒辦法戰勝TNC，Fnatic和Misk。我總是拿到第四名這讓我很沮喪，我也沒辦法打入Major賽事。

你覺得你和xNova很像嗎？

他已經在Ti中拿到了亞軍，這是我沒辦法相比的。我希望我能和他一樣成功。

誰是你在DOTA2圈的偶像？

Sumail，因為我希望能像他一樣拿那麼多獎項，尤其是他非常年輕。

你的家人和朋友怎麼看你步入DOTA2職業圈？

他們很直接的說我應該去找一份正經工作，我對他們說需要給我一點時間證明我自己。我的家人慢慢理解到了電競和玩遊戲不一樣，這也是一份職業。他們在新聞上了解比賽然後慢慢支持我。

他們來看過你比賽嗎？

當我在馬來西亞的時候，他們會來看我訓練。他們沒有看過我在海外進行比賽。

你對年輕的職業選手有什麼職業建議嗎？

我的建議是不要放棄，我相信每個人都會遇到自己的機會。

你在外國的Aster戰隊打比賽，現在看來馬來西亞的新人都被中國隊發掘了，你覺得這對馬來西亞有幫助嗎？

當然，我不認為東南亞有很多機會，尤其是不會說英文。我覺得中國能提供像我這樣的人更多機會。

中國隊和東南亞隊之間有什麼差異？

並沒有太多差異，但中國隊的日程表更緊湊。我們在早上10點訓練到下午6點，以前的話我記得是從中午12點開始訓練。
