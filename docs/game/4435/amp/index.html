本周的熱門遊戲事件有點多，一句兩句無法說清楚，還是跟X博士一起走進本周的熱游情報看看究竟吧。

劃重點：B站8億拿下S賽三年獨家轉播權 巴西隊美女輔助正式「出道」

本周我們迎來了LOL全明星賽的歡樂賽程，在SOLO賽上來自LPL的四名選手都拿下了首輪勝利，晉級16強。

SOLO賽上最有看點的就是最後一場Tian和小花生新老「神僧」的對決，小花生剛剛來到LPL可能也是想要證明一下自己，結果把自己玩進去了。

以結果來看，LGD好像上當了。

還有一個令人吃驚的消息，B站在最近的競標中以8億的價格拿下了S世界賽三年轉播權，真的是有錢任性，另外虎牙和企鵝電競則分別拿下了LPL比賽的轉播。

這麼大個事鬥魚竟然都沒參與，看來是真的缺錢了。

除此之外，本周巴西賽區還收穫了一枚日裔00後女輔助Mayumi，這位小姐姐不僅五官、身材非常Nice，在賽場上的成績也十分不錯，不僅一夜之間冒出大批「十年老粉」，甚至還引來了不少明星選手的關注。

不少粉絲還調侃IG，如果把她簽下來阿水可能就考慮續約了。

不過這位選手在社交媒體上特別活躍，申請了微博之後三天已經發了14條微博了，顯然是奔著成為明星選手去的。像巴西這種外卡賽區如果能出一個明星選手的話，確實會對整個賽區都帶來很大的幫助，倒也沒什麼問題。

不如多整點。

劃重點：不要不信這一套！遊戲開服不看黃曆後果很嚴重

說了熱門端游，本周手遊方面也有一些大事件，相信不用X博士講大家也都知道我說的是誰。

說實話在12月3日三蹦子公布新玩法內容「後崩壞書」的時候，X博士還有點納悶，新劇情、新角色、新關卡還有新的浮空戰鬥模式，這幾乎是直接把「崩壞4」做進了崩壞3里啊？！

這也等於提前亮了底牌呀，不得趕緊開個小會防止隔壁《戰雙帕彌什》直接把玩家帶走？不過現在看來會也不用開了……

戰雙這次開服操作是真的厲害，開局發錯點券就不說了，拔線回檔回錯時間又被部分人領了一萬，再次回檔還沒回全，只回了一部分。

X博士翻翻日曆，發現有時候老祖宗的話不得不信。

不過好在最後送了個自選S和限量版皮膚，這事兒才終於算過去了，要X博士說，還不如1W黑卡就當白給了，省得這麼多麻煩。

戰雙這算是圓上了，但同日新發布的《灌籃高手》手游就沒這麼容易了。

這款遊戲也和戰雙一樣，開局送的鑽石發錯了，IOS一區直接給了100倍獎勵，合RMB五千多塊錢。

這種超大額度的錯發，即便回檔玩家也不會有太大的意見，給點補償就行了，但運營偏偏選擇了最秀的選項——封號！

他們竟然把所有收到獎勵的賬號封了416天，幾乎團滅了一個大區，戰雙真是被你給比下去了。

雖然後續有一些修復和補償措施，但面對這樣沉重的打擊，很多玩家也已經沒辦法再愉快的玩耍了。

這一連串的運營事故幾乎能載入史冊，也給後來者一個警醒，發布遊戲時一定記得要看黃曆。

劃重點：國行switch正式公布發售日期 遊戲定價更便宜

主機方面本周倒是也有大新聞，那就是騰訊代理的任天堂Switch國行終於開啟預售了，目前售價為2099元，正式發售時間為12月10日。

目前已經公布即將發售的遊戲有馬車8、馬里奧奧德賽以及超級馬里奧兄弟U，售價全部都是299元，要比某寶上便宜不少。

另外同步發售的還有Switch的一些配件，包括手柄、充電握把、底座等，售價都非常不錯，而且擁有全國聯保，另外幾款amiibo互動模型的價格確實香瘋了。

不過購買國行主機的朋友還需要注意一些有關區服和賬號的問題，購買卡帶和機器之前一定要謹慎考慮。

劃重點：《荒野大鏢客：救贖2》steam版又炸了 好評率低至66%

這裡先跟steam玩家和Epic玩家道歉，上周X博士剛調侃完其他平台來給Steam玩家當試驗品，報應立馬就來了，人家R星壓根就沒給你修復。

Steam版本剛一上線，玩家該進不去遊戲還是進不去，和當初首發PC時一樣，誰也別想玩。

（X博士沒想到這圖還能重複利用）

由於批無法登陸的問題，也導致Steam版本的評價直接降到了66%，講道理要是Epic能評價，估計也是這樣。

最難受的就是上周Epic打折，大表哥2隻需要196元就可以了，而且還是修復過的。

結果Steam版本一上線，不僅沒修復，價格還是249元，虧得我都哭了。
