fgo：金方塊最值得兌換的五種資源 穩賺不虧 夢火優先順序並非最高

2019-12-15 120條評論 1,210次閱讀 120人點贊

fgo因為聖誕復刻活動帶來的系統改版，導致金方塊原本這種比較偏向土豪限定的資源，也變成了平民玩家可以觸及得到的了，不過因為數量依舊有限，所以金方塊兌換什麼資源成為了不少玩家所苦惱的一件事情，本次就來詳細說說金方塊最值得兌換的五種穩賺不虧的資源吧。

NO.5金芙芙

看著不錯實際上性價比不高，因為金芙芙這玩意雖然是不斷提升從者上限的，但是提升的幅度還有數值不攢夠一定的數量真的體現的不算明顯，這點數值說實話你用金方塊實際上是有點虧的，但對比其他的道具比如友情點什麼的，就賺很多了。

NO.4結晶

很多玩家考慮結晶的問題，如果入坑不晚的話一般不會考慮這個玩意，全助戰310的從者培養目前給的結晶足夠用，而且從者的養成是個大黑洞，你結晶屯那麼多就QP和素材跟不上也沒用，而且結晶每次活動都會給，除了明年的死亡長草期長了點會消耗不少外，常規養成不碎石頭的情況下活動給的完全足夠用。

NO.3夢火

理論上來說夢火的需求度非常的高，因為不少老玩家的大英雄和四大天拐以及核心打手差不多都滿羈絆了，所以常駐囤一個夢火是很有必要的，能解放其使用度，而且還能收穫額外的聖晶石，相當不錯。不過這個道具的兌換優先度不如後面兩個，如果後面兩個不用兌換，那麼夢火優先度會提高很多。

NO.2禮裝

提供禮裝和QP加成的那些特色禮裝，這個性價比非常非常的高，尤其是蒙娜麗莎這樣的禮裝，雖然後續會不斷的推出新的類似這種加成的禮裝，但因為效果可以疊加，所以性價比穩賺不虧，這種類型的加成禮裝越早點拿到越吃香。

NO.1衣服

魔術禮裝的優先度是非常高的，就算你平常真的不怎麼用得到一些衣服，不過獲取的價值還是很高，尤其是在衣服方面的價值非常高，對於並非開服老玩家來說，很多衣服沒有拿到不得不說是一個遺憾，通過金方塊可以很好地彌補這個遺憾，目前對於入坑不是很早的玩家來說有四套衣服是需要消耗金方塊獲取的，其入手性價比和必要性分別如下：金色慶典>王室品牌>月之海記憶=月之背面的記憶。
