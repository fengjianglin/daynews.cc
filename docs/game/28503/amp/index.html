鬥魚黃金大獎賽：VC決賽圈2V3先手放倒兩人，卻遺憾未能吃雞

天天要聞
2019-12-23

鬥魚黃金大獎賽經歷過全明星賽事之後，正賽繼續上演。TSG、BA、TITAN、SU4、VC等頂級強隊給大家帶來了六場極為精彩的比賽。目前排名在前三名的分別是第一名TSG55分，第二名BA53分，以及第三名Titan50分。從比分就可以看出，前三名隊伍的實力極為的接近，第一名的TSG同第三名的Titan僅僅只有五分之差，由此可見，鬥魚黃金大獎賽的賽況是何等的激烈。

除了TSG、BA和Titan在比賽中表現的異常出彩之外，排名第六的VC戰隊也是表現的可圈可點，在第四局的時候成功吃雞。稍顯遺憾的是在第六局比賽之中，VC以少打多2V3，本來有著吃雞機會的，但是確實可惜的錯過了。

本局決賽圈還剩兩支隊伍分別是Titan和VC。賽場的形勢對於VC極為的不利，Titan三人佔據山坡的高處，而VC戰隊僅剩兩人處于山腳下的位置。這樣一來，VC佔據了地形和人員上的優勢。

Titan和VC都互相利用手雷試探，期待著用手雷來打開局面，但雙方都收效甚微，此時VC戰隊的Monkeyking找到了機會，先後剛槍擊倒了Titan戰隊的HongHong和Reve。本來按照想像的劇本，VC將2V1吃雞了。但是，由於Monkeyking剛了兩人血量太低，被Tiatan的獨狼Krystal擊倒，隨後Krystal調轉槍頭淘汰掉VC最後一人，成功吃雞。

雖然，VC最後關頭未能吃雞，但是本局也是取得了第二名的成績，尤其是Monkeyking這波精彩的1穿2操作還是相當的秀的。鬥魚黃金大獎賽仍在繼續，期待一下VC戰隊在接下來的賽程中能夠繼續給大家帶來精彩的比賽吧！
