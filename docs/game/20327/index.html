DLC與續作的矛盾產物——從《P4G》和《P5R》看加強版遊戲

2019-12-17 990條評論 4,050次閱讀 141人點贊

本文是撰稿人投稿，僅代表作者觀點

——「《三國志14》要發售了耶！」

——「不急，等『威力加強版』！」

所謂「加強版」遊戲

如今的遊戲圈，一款遊戲僅以一部完整的本篇作品「站穩腳跟」的情況已經很少見了，越來越多的遊戲選擇向服務性遊戲方向轉型.

傳統單機遊戲也漸漸跟上時代的腳步，DLC與加強版遊戲一步步地走進我們的視線。

想要賺更多錢的遊戲公司，自然是比較樂於為自己已發售的遊戲添加DLC的購買選項。

所謂的DLC，就是指伴隨著遊戲本體發售的追加內容，小到遊戲內的貨幣皮膚，大到一個等同於遊戲本篇體量的遊戲內容板塊。

DLC在如今幾乎存在於各種大小型遊戲之中，當然，也有如《戰神》這樣的少數遊戲還在堅持著難得的初心與良心。

這並不是說DLC的風靡是一件悲哀的事，相反，高質量的DLC在不僅擴大了遊戲本篇的內容、彌補了遊戲的局限性的同時，還大大滿足了粉絲們的需求。

比如《巫師3 狂獵》的兩個大型故事DLC《石之心》《血與酒》就是遊戲圈中經典的良心DLC。

我們熟知的「DLC狂魔」光榮特庫摩則是因為在各種遊戲中大力度販售小型DLC而導致口碑不佳。

許多遊戲在發售DLC之後還會推出整合DLC和原版內容在一起的「完整版」或「年度版」等。

然而，有一些遊戲在發售後則是得到了一次「進化」，這個進化之小無法讓遊戲扣上「續作」的標籤.

同時，並不是通過大量修修補補的DLC就能夠實現這樣的進化，所以就誕生出了遊戲「加強版」這個說法。

整合兩大良心DLC的《巫師3 狂獵 年度版》。

DLC、加強版和續作三者顯而易見，是遞進的關係——關於一個遊戲的豐富、升級和發展的對應關係。

以KT的《無雙大蛇2》為例，各種服裝、武器、關卡DLC就是其對遊戲內容擴展的層次.

《無雙大蛇2 終極版》則是整合了DLC和全新內容的加強版本層次，《無雙大蛇3》則是系列發展到下一步的續作層次。

現在的許多遊戲都遵從著這樣的發展規律穩步向前，而且其中大部分是日廠遊戲。

《無雙大蛇3 終極版》——系列最短時間內推出加強版的作品。

DLC的發展之迅速，使其誕生出了「季票」這一概念；玩家只要擁有了季票，就能獲得之後推出的DLC內容或者以低價購買DLC內容，季票通常可以通過購買典藏版遊戲本體或單獨購買獲得。

一般遊戲的季票數量都不會多於2到3個，少數吃相難看的遊戲才會推出多項季票圈錢，當然服務性遊戲就是另外一說了。

季票的存在使得零碎的DLC被稍微地整合化、簡潔化，遊戲的追加內容能偏向一個整體板塊被玩家接受，而DLC理念到達極致的時候就能誕生出加強版遊戲的概念。

在未及續作的高度上，DLC的重疊堆砌加強遊戲的體驗，二者矛盾之中就產生出了加強版遊戲。很大部分遊戲的加強版本都是基於這樣的環境下應運而生，不是絕對的，但也絕非是意外的。

這是現在遊戲發展的一個小規律，然而以前卻不是這樣的。

在很多年以前網路還沒有像現在這麼普及、這麼方便的情況下，DLC是無法在線上推出的，絕大部分廠家是直接跳過DLC的步驟，從加強版遊戲開始製作起，最經典的便是「《三國志》系列」和「《信長之野望》系列」的「威力加強版遊戲」了。

但是這些加強版遊戲就是簡潔明了地優化系統、增加內容的「疊加式加強」，和下文即將提到的「《女神異聞錄》系列」的「整體式加強」不是同一類型。

現今條件發達，除了DLC之外，「疊加式加強」的遊戲可以很方便地通過升級包的方式增加遊戲內容，但也導致部分玩家的一種消極心理出現，就是「不玩本篇，只等玩加強版本遊戲」的想法。

但是，照目前來看，DLC、加強版、續作的層層遞進關係暫時不會被打破，作為其中最能夠掙錢、也最能夠方便地完善整體遊戲的做法——推出加強版遊戲，會被越來越多的廠商所採用，也會被越來越多的玩家接受和理解。

但前提是加強版遊戲必須對得起它的身價，再多內容的加強版，如果沒有誠意的話，同樣收穫不了粉絲的肯定，就更不用說路人的賞臉了。

在遊戲領域中，擁有推出加強版本遊戲傳統的，亦存在著許多聲名赫赫的IP的，就比如「《真三國無雙》系列」與「《戰國無雙》系列」的「猛將傳」、「《怪物獵人》系列」的「G版本」、「《戰國BASARA》系列」的「宴、皇」等等。除此之外，也有一些添加各種全新副標題的加強版遊戲，它們在本質上都是相同的。

「《怪物獵人》系列」每一作的經典「G版本」加強。

不過即便素質優秀，也不排除一部分不願意都買賬的玩家。

因為推出一個加強版的遊戲，意味著玩家還需要再掏一次錢買下這個「1.5倍升級版」的遊戲。

除了一些遊戲的粉絲和擁躉之外，大多數玩家可以分為兩派：一種是玩過原版遊戲，不想花大錢購買對自己來說擁有重複體驗的遊戲外加一個冷冰冰的升級包的玩家；另一種無視正統本篇作品，只等加強版遊戲推出後再出手體驗「完全形態」的玩家。

此兩種玩家有一點想法或許是一致的：對廠家這種做法的無奈和悲哀。

而隨著技術日漸的增長，許多遊戲的加強版本已經可以通過單獨購買升級包來升級成加強版本，在上文中所提到的眾多光榮遊戲和「《怪物獵人》系列」等都已經做到了這個轉變。

但是，以「《女神異聞錄》系列」為主的一部分遊戲是沒有辦法通過單獨購買升級包來進化的，其甚至都無法做到存檔繼承，因而得到了大量對此感到莫名其妙的玩家們的差評。

接下來，筆者將以「《女神異聞錄》系列」的經典作品為例，詳細闡明為什麼這部分遊戲是沒辦法做到藉助升級補丁升級的，以及它們是否有存在的價值等問題。

《怪物獵人 世界 冰原世紀》實質上也是加強版遊戲。

加強，並非代表簡單的相加

近兩年，《女神異聞錄5》紅遍遊戲圈，其加強版遊戲《女神異聞錄5 皇家版》自公布到發售都備受矚目，但是期間還是不乏有一些萌新問道二者有何區別，甚至還有一些激進的玩家直噴以全價購買加強版遊戲的做法。

但是，許多玩家並不明白其中緣由，如果《女神異聞錄5》能簡單地通過一個DLC升級包完成《女神異聞錄5 皇家版》的升級又何嘗不為呢？

關於這個，我們來從系列另外兩款經典作品——《女神異聞錄4》和《女神異聞錄4 黃金版》來探討一下。

大部分的加強版遊戲，添加的內容主要有如下幾個方面：

1.對原版遊戲漏洞和局限之處進行修復，優化和完善體驗；

2.增加全新資料片，豐富劇情和玩法，增加針對不同玩家的難度選項，提高老玩家再度體驗的新鮮感；

3.提升遊戲的包容度，迎合更多玩家的需求，極力體現出新遊戲之於老遊戲的「完整」。

這是許多遊戲推出加強版共有的特點，而《女神異聞錄4 黃金版》相較於《女神異聞錄4》升級進化的元素卻與以上這幾點不太對得上號。

從暗黃到金黃——奇蹟般地重現完美。

首先，《女神異聞錄4 黃金版》對原版遊戲的操作進行大幅度改良，UI設計也全部煥然一新。

在戰鬥中，除了增加了許多原版沒有的要素，比如雙人同時攻擊等，還提高了戰鬥的整體流暢度，這就需要將戰鬥系統重新打磨和製作。

然後就是添加大量的全新要素，比如遊戲中非常重要的人格面具圖鑑的翻新、故事的重新構思和創作、社群等級的搭配重組、新音樂的添加與運用等等。

尤其是對於角色扮演遊戲最重要的養成、收集系統，是無法簡單地疊加、堆砌新內容進去的，而是需要從遊戲整體出發、隨其他遊戲系統同時進行改動的。

《女神異聞錄4》原版的故事劇情在《黃金版》中得到新的演繹，尤其是加入了新角色和各種新劇情，整個故事的發展雖然大體上是一致的，但細節方面的完善和豐富程度使整個故事彷彿得到了升華，以至於原版的出色故事都顯得黯然失色。

而一個細節的改動或添加，往往就會影響到該方面整體甚至是所有與之相關的其他系統的變化。

《P4G》將許多新劇情完美融入了原來的故事之中。

原本，《女神異聞錄4》是一款很完美的作品，然而在《黃金版》面前，它又如同一部半成品般的存在。

這並不是指《女神異聞錄4》沒有存在的價值，而是指《黃金版》這個加強版本對於《女神異聞錄4》來說是由內到外的完美升級，不能和簡單疊加新內容這樣的「加強版」同日而語。

總體來說，《女神異聞錄4 黃金版》是在《女神異聞錄4》的成功基礎上翻新了一遍，用許多材料將漏洞填滿，又用了很大功夫將粗糙的地方修理得精緻：它不是在一塊石頭上刷一層漿，而是將這塊石頭雕刻成更精美的藝術品。

而遊戲加強版本所謂的「進化」，指的就是這層含義。

以「《女神異聞錄》系列」為主的這一類遊戲的加強版並不是簡單的「1+0.5」，而是「1×1.5」，二者是不能畫上等號的。

這樣理解的話，關於《女神異聞錄4 黃金版》，廠商將遊戲的加強版全價發售的情況也是對得起這其中的進化的，至於願不願意花錢購買的玩家，也就屬於「願者上鉤」了。

所以，並不是所有的加強版遊戲都是適用於通過升級包強化遊戲的，這類良心的加強，雖然也不多見，但是也不應該被誤解。

加強，也並非代表全面進化

而與《女神異聞錄4 黃金版》形成反方向鮮明對比的，竟然是出自同一個系列的、且最近熱度也非常高的《女神異聞錄5 皇家版》。

《皇家版》成功向期待著的粉絲們展現了一個道理：加強，也並非就是全方位的升級與進化，即便這款作品的系列歷史上還有著加強版本的成功先例。

「《女神異聞錄》系列」推出補足前作的「完整版遊戲」已是傳統操作，就比如有著像《女神異聞錄4 黃金版》那樣的經典升級。

但是在發售前造勢頗大的《女神異聞錄5 皇家版》在發售後卻落得一個「沒有誠意」的評價，在粉絲中的口碑跌落更甚。

遊戲發售前吊人胃口的溜粉絲行為本身就不得人心。

簡單介紹一下《女神異聞錄5 皇家版》相較於無印版的《女神異聞錄5》的「皇家」體現在何處。

首先，提升遊戲的迷宮探索體驗，加入鉤繩、收集石頭、收集花瓣等要素全面提高迷宮關卡的新鮮感。

強化戰鬥系統，增強「換手」機制和加入全新的合擊必殺「Show Time」，所有BOSS戰也增加了更多內容的補充。

除此之外，其他要素都是小修小補，比如UI、立繪、語音、音樂，添加了人格面具的特性和戰鬥後自動補充彈藥的細節等等，這些內容和《女神異聞錄4 黃金版》增加的小要素差別不大。

但是，《女神異聞錄5 皇家版》在故事劇情方面沒能做到讓大部分粉絲滿意。

在前期流程的遊戲體驗和原版重複度過高的同時，遊戲大肆宣傳的「第三學期」不僅解鎖條件苛刻，其實際內容也只有一個月的自由活動時間和一個迷宮的長度，這對於相當期待這個新內容的玩家們無疑是潑了冷水。

後期劇情和原版出入較大，但歸根結底是為改變而改變、為新鮮而新鮮，完全是出於吸引粉絲掏錢的目的做出的敷衍了事的修改。

和《女神異聞錄4 黃金版》不同，《女神異聞錄5 皇家版》並沒有加深遊戲的劇情深度，許多日常支線劇情都是硬塞進來的，文本量大幅增加卻少有實際有內涵的內容，整個新加入的故事劇情顯得很拉跨和敷衍，貌似只是為了能夠在宣傳的時候吸引眼球而製作的。

遊離主線、定位尷尬的新角色——芳澤霞。

造成《皇家版》評價不如原版的整體原因很簡單：忽視了被改變的內容的重要性。

眾所周知，《女神異聞錄5》本身就是一個在各方面都很完整的遊戲了，尤其是富有深度的劇情方面。

而《皇家版》為了能夠吸引粉絲買賬，對後期的主線劇情進行大刀闊斧的修改，而且還為了滿足部分玩家群體的需要強制增加某些不討喜的角色的戲份等等，都不知不覺地傷到了一些粉絲的心。

客觀來講，如果從外人角度來看，《皇家版》依然是入坑「《女神異聞錄》系列」的好作品，但從部分粉絲的角度來看，《皇家版》對不起他們的等待與期待。

其實這也很容易看出，遊戲開發商Atlus也是想要多掙點錢才極力推出加強版本的遊戲，而且還把《女神異聞錄5》的後日談內容放在2020年年初推出的新作《P5S》上，說好聽點叫為《女神異聞錄6》賺奶粉錢，說難聽點就是吃相稍顯難看。

但這其實也很容易讓人理解，不止Atlus，其他廠商很少能夠避免得了有「增強版」遊戲可以推出、錢可以賺卻不賺的情況。

除非一款遊戲的完整度已經上升到自身都無法對其進行完善和升級進化，不然像《女神異聞錄5 皇家版》這樣的「完整版遊戲」還是會越來越多的。

只不過，《女神異聞錄5 皇家版》的評價下滑也告知了玩家們一個道理：並不是所有加強版遊戲都能夠在原作的基礎上進行進化和拔高一個層次，這取決於廠商的態度和原版遊戲的高度。

《女神異聞錄5》本身就是一款非常完整的佳作。

結語：加強版遊戲始終服務於粉絲

加強版遊戲和外傳遊戲不是一個概念。比如「《真·三國無雙》」系列的《猛將傳》和《帝國》兩個版本，前者就是實實在在的加強版本，後者則是資料片性質的外傳遊戲。

加強版本的遊戲不像外傳遊戲那樣能拓展出別的玩法、開闢新的市場，說白了其主要作用就是讓老玩家擁有同一個遊戲的全新體驗，也就是通過豐富遊戲的內容來延伸遊戲的生命力。

但是過多地推出加強版本遊戲又會讓玩家產生審美疲勞，更嚴重的還會感到反感。可能只有粉絲才會堅貞不渝地追隨自己所愛的遊戲，願意第一時間花錢享受原版遊戲和加強版遊戲的雙重體驗，即便這其中可能還會存在部分重複的內容。

《最終幻想15 皇家版》——通過修修補補挽救粉絲的評價。

縱觀遊戲發展歷史，很少有遊戲的加強版本的銷量是蓋過原版遊戲銷量的，當然這其中不包括那些轉換遊戲平台等的遊戲。

總的來說，玩家必定能夠接受加強版本遊戲存在的現狀，但也不得不承認，加強版遊戲始終是服務於粉絲的需要和出於賺粉絲的錢的。

為了擁有一個更好的遊戲體驗，推出加強版遊戲可以看作是廠商對自己遊戲更高的追求，至於出於什麼目的，就交給玩家自己判斷了。

我們能做到的，也唯有認同加強版遊戲存在的意義與價值罷了。

羊洛 本文作者
