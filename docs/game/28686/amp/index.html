Ray專訪：從Scout身上學到很多，明凱是一位了不起的選手

天天要聞
2019-12-22

昨日韓媒FOMOS（포모스）發表了題為《[김용우가 만난 사람] 6년 만에 찾은 LCK 무대…kt ‘레이’ 전지원 이야기》（【採訪】時隔6年來到LCK舞台—KT 「Ray」全志願的故事）一文，原文（翻譯）大意如下：

最近見到的KT教練康棟勛也這樣說到，在「Ray」17歲的時候就已經關注他了，當時就想著像Khan金東河（現FPX選手）那樣帶他過來發展的，但是沒有實現。康教練這裡所提到的「Ray」全志願在2015年EDG二隊AD GAMING(現EDG.Y)出道。

2016年離開中國加入北美APEX GAMING，之後在DIG和C9兜兜轉轉，生活可謂跌宕起伏。2017年重新回到EDG成為主力隊員。2018年在地區選拔賽中淘汰RW從而進入了世界賽。

Ray在2019年LPL夏季聯賽選擇了休息，沒有給出相應的理由。2019夏季賽結束之後2020賽季選擇加盟KT和康棟勛教練坐上了同一條船。雖然明年就是作為職業選手的第6年了，但是第一次登陸LCK的Ray說到「4年前就想來LCK了」。

康教練對於Ray的評價「雖然是有長處的選手但長期在海外聯賽活動的同時優點也都消失了」。但是康教練還是認為會像成長為LCK最強上單Khan一樣，Ray也會為了進入屬於他的第二春而揮汗如雨。

Q：明年雖然是職業生涯第6年，但是第一次來LCK。回到韓國的理由是？

理由很多，雖然作為韓國選手但在韓國舞台上一次也沒有登場過。在海外聯賽也呆了很長時間，也沒什麼動力了。想在新的環境下挑戰一下，也想在LCK提升自己的能力。

Q：2019年LPL夏季聯賽選擇了休息，當時也沒怎麼說理由，現在可以說明一下嗎？

當時是完全無法進行比賽的狀況了，之前雖然也說過，當時是沒有動力的狀況。最後對我來說已經完全筋疲力盡，無法集中了。

Q：對於前面所說的「動力」，能具體說說嗎？

作為職業選手已經是第5年了，但是目前為止沒有收穫什麼。一起開始的選手們都看到了職業發展方向，但我卻沒有。我覺得自己已經很努力了但是沒有收穫，所以心情也變急躁了吧。同時隊伍成績也不好，也很煩躁，對我來說雖然是一種鞭策，但壓力也很大，所以需要休息。

Q：休息的那段時間裡做了什麼？

我很喜歡玩。在海外生活了5年但一次也沒有旅遊過。不是在家直播就是打遊戲所以想嘗試一下新的生活。玩了很長時間也體驗了很多東西，當然也經歷了第一次出國旅行。我雖然做了想做的事但當時給我的感覺就是和遊戲的距離變遠了。應該要維持一個界限但當感覺自己和遊戲的距離變遠的時候，也曾考慮過如果不當職業選手要去做什麼，甚至想過去做兼職。因為是通過打遊戲來掙錢，當時也曾考慮過如果不打的話該怎麼生活下去。總之當時真的想了很多，想到的結果就是我最厲害的還是打遊戲。自己擅長打遊戲也想按自己的想法生活下去，所以又以職業選手的身份重新回來了。

Q：回到韓國選擇的隊伍是KT

雖然是通過採訪來表達我的心意，但還是想謝謝各個關注過我的隊伍。事實上我沒有什麼履歷可以展示，雖然是個人的想法，但我認為自己還沒有展現出真正的實力，看到這種可能性並聯繫過我的隊伍很多。因為和EDG的合同自動延長一年，KT當時都已經放棄了。我當時也考慮過要不要在打一年，但是康教練找到我，聽了我的想法之後問我想不想一起合作，說是幫我解決合同問題。所以是託了康教練的福才來到了KT。

Q：明年就是第6年了，但對於LCK來說是新人。

想到了『finally』這個單詞。很早以前就想來了但因為合同的原因一直沒來成，事實上17歲出道的時候康教練就想帶著我了，這次終於獲得非常好的機會來到韓國的隊伍。事實上並沒有期待什麼，來到韓國隊伍感覺就像做夢一樣，因為在LPL夏季賽休息的3個月里還想著該怎麼生活下去，結果就轉會到KT了，到現在還感覺不是真的。

Q：通過之前的採訪就感覺到你是「開放的心態」。

高一呆了3個月退學之後就來到中國，當時17歲。社會經歷從EDG開始也從那裡學到不少。以一個充滿霸氣的新人心態去了之後，發現比想像中的要累。雖然現在覺得中國食物很好吃但當時都不能入口，將近一年的時間裡只吃辣椒醬和白米飯。當時的教練Reapered和PawN（退役）還有Deft（DRX）來了之後才開始變得話多。當時PawN和Deft是我的榜樣，去北美的時候遊戲ID「DeftPawN」的理由就是這個，不僅是榜樣也從他們那裡學了很多。

Q：然後是2016年離開EDG去了APEX（現Dignitas）

去美國的時候還不懂人情世故，當時還不怎麼成熟。18歲去美國之後學會了人應該怎麼活。美國是一個很nice的國家，當時才開始了解到當地的人們都十分開放。在開放的國家生活之後性格也發生了變化，形象也變得開朗起來，這就是我經歷了很多的美國生活。

Q：在經歷APEX,DIG之後與2017年轉會到C9

2017年北美LCS夏季聯賽的時候得了抑鬱症。原因很多，壓迫感很強烈，並且當時和Impact在同一隊伍以一種競爭關係相處著。現在回想起來並不是因為競爭的關係，而是當時的心理負擔比較大，心裡意志經常容易垮掉，惡評中印象較深的「為什麼讓Ray出場，impact不上場的話就沒有看的價值了」。因為這句話自尊心受到傷害，想平淡地度過去但一直留在心裡。一直積累著所以心裡負擔加重了，也和隊伍走得越來越遠。不知道是不是在海外待久了才這樣。

Q：2017年離開美國又回到EDG，可以聽一下理由嗎？

LCS是一個非常好的聯賽，但我自己覺得和他們走遠了，當時也覺得對不起隊友。自己下賽季也可以繼續，但在治療的過程中認清了自己的現狀，覺得自己要出場比賽，不是，是想打比賽。賽季結束之後C9引進了「LICORICE」，覺得不會讓自己出場，所以產生了心理壓力。當時也找了韓國隊伍也和其他很多隊伍商討過，但EDG提供了最重要的條件（職業規劃，可以作為首發出戰）。問我為什麼回EDG的理由還有一個就是有關心我的人，這讓人很詫異，理由之一就是中國的粉絲們。我認為有了粉絲才能被稱為職業選手，當時也留下了很多記憶，想再打好一次，也覺得當時有實力可以做好。

Q：EDG職業生涯中留下印象的隊友。

Scout是我第一次遇到的同歲朋友，他真的很出色，我也從他身上學到很多，離開EDG之前也和他聊過，多虧有他我才能堅持至今。雖然是好朋友但是一起沒打出成績真的很遺憾。

Q：2020賽季Clearlove明凱成為了EDG主教練

我作為業餘選手進入隊伍的時候明凱就十分亮眼，像光一樣的存在，真的很厲害，該稱他為六變形無死角選手嗎？真的各個方面都很厲害，沒有不足的地方。也和我說過「你打得好的時候我也厲害的話應該能拿不少冠軍了」，真得很了不起的一位選手。一開始不知道要當教練，只料想到會退役，我離開之前也和Clearlove說了，他會發展不錯的，他也知道我為什麼離開EDG。他對於EDG是一個不可缺少的一部分。

Q：在KT的話也會和SoHwan金俊迎競爭上崗

我不覺得這是競爭，也不想去競爭。之前和Impact競爭過，當時壓力也十分大，對生活也造成了影響。我認為大家都是一個隊的隊友，一起成長，取長補短，應該從這個角度出發。

Q：周圍的人有什麼反應？

收到了很多祝賀。了解我緣由的人都說「要好好打」，有能力就要在LCK讓別人看到。4年前就想來LCK了，現在終於回來了。（笑）

Q：很好奇和隊友們相處得怎麼樣？

沒有覺得什麼不便，雖然目前還不怎麼熟但我認為這是一個新的經歷。隊友們性格很好所以很舒服，有興趣想去了解他們。

Q：你認為能作為榜樣的有誰？

之前採訪說過的Impact和Reapered教練（現C9）。從他們那學到了怎麼打遊戲，是很好的榜樣。最近提到這個就會想到Kiin金基仁，Summit朴湡台選手，我也有想成為他們那樣的選手，能進攻能抗壓，在線上很強勢。現在對於我是抗壓選手的評價比較多，但我之前是攻擊性的上單，線上也很厲害，和現在的Nuguri張夏權選手很像，但現在的風格和之前完全相反。現在是以那兩位選手為榜樣進行訓練。

Q：2020年的目標和想說的話

現在終於要登上LCK的舞台了，希望能夠順利適應新環境，也想將一直以來積蓄的能量爆發出來，到目前為止我的遊戲生涯一直都很艱難，也經歷了許多曲折，所以十分希望能夠衝擊更高的榮譽，在EDG戰隊的這段時間裡我確定了自己的遊戲風格，之前我是那種敢打敢秀的選手，現在想重新找回原來的自己。
