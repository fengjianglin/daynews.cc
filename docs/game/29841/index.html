DNF懷舊六十版本有五大不可能，情懷不能複製，玩家耐心不如從前

2019-12-24 230條評論 1,386次閱讀 456人點贊

近來很多玩家在討論不遠將來的一百級版本的事情，大多數人對一百級版本都沒有抱著很大的期望，甚至還有人等著裝備不能繼承好脫坑。鑒於大家都不看好一百級版本，就有玩家說，既然繼續升級不行，那麼懷舊一下六十版本可不可以？似乎很多人對六十版本還有不小的情懷在那裡。其實懷舊六十版本是非常不可行的，你希望的東西，都不見的多好。

一、交易系統不能懷舊。

懷舊到那個沒有拍賣行的版本，那就很好了，商人么馬上就能囤積全大區的重要材料，重要裝備，到時候怎麼辦，你買還是不買？爆一件粉裝誠然值錢，可是你也不能一直穿著藍裝刷圖啊，傳承要買，粉武器要買。到時候你選擇噴商人，還是噴版本？

二、升級系統。

升級那是真的慢啊！以前能一管疲勞刷一下午，現在能嗎？如今有時裝寵物稱號的加持，速度都很快，回到六十版本那個跑圖如烏龜爬的版本，你的情懷未必受得了啊。復活幣也不是天天送，到時候沒復活幣你就多多享受站街等虛弱把。

三、金幣價格。

六十版本不太可能有多開刷金這種事情，你選擇飛機飛就得等著封號，那升級速度，工作室早該跑路了。好了，再回到十五比例的版本，一件傳承兩三千萬，貴的七千萬，慢慢買去吧。血葯，藍葯，等等，慢慢買。

四、掉落系統。

呵呵，六十版本掉落是共享的。哥布林的手骨被搶走也就罷了，紫裝被搶走，粉裝搖篩子又沒有搖到，三個復活幣刷了次深淵，結果你的無影劍被一個聖職者撿走了，舒服不？人家喜歡狗眼，你管得著嗎？

五、福利待遇。

現在光混活動，復活幣血葯都是用不完的，而六十版本，這些東西是要賣錢的。創建角色送復活幣，全部用光以後，每天可以領一個不能疊加的復活幣。什麼送這個送那個，做夢去吧，轉職送一把領悟者的大棒子，你還得用到二十五級以後呢。

總而言之，版本是可以回去，但是人心回不去。版本回去了，現在的職業歧視，暴躁罵人也會回去，到時候狂戰就是送頭狗沒人要，男槍炮召喚倒成了刷圖超一線職業。你說六十版本沒有這些東西，可是現在有啊！我們能把版本重做，那麼也能把玩家統統重做，讓三十歲的大叔，變回去當年的初中生嗎？遊戲不過是一堆數據，做成什麼樣子都可以，而人類暫時沒有發明時光倒流的技術，夢想回到六十版本，不過是庸人自擾。
