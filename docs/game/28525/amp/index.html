國產上單的希望？LGD新上單太猛了，鱷魚一打三拿到三殺

天天要聞
2019-12-23

12月22日晚上結束的德瑪西亞杯V5和LGD的比賽，這場比賽的勝負對於兩隊的晉級都非常關鍵。這一局中peanut的節奏依然很好，前期多次抓上幫助自己上單鱷魚肥了起來，而LGD上單cult的鱷魚也給出了回報，一波一打三拿下三殺，非常精彩！最終LGD比較輕鬆的擊敗了V5，拿到了比賽的勝利。

雙方這一局的BP如下，V5這邊ban了奇亞娜、刀妹、奧拉夫、盧錫安和卡莎，選擇了上單妖姬，打野盲僧，中單劍魔，下路女槍加泰坦；LGD這邊ban了鐵男、莫甘娜、挖掘機、女警和阿卡麗，選擇了上單鱷魚，打野蜘蛛，中單冰女，下路霞加女坦。

4分鐘的時候，V5上中野三人野區包夾小花生，將其擊殺，V5拿到了一血。5分鐘的時候，LGD三人抓上，越塔擊殺了mole劍魔。8分鐘，peanut蜘蛛又去抓上，又一次擊殺了mole劍魔，同時中路Yuuki冰女被V5三人包中，Yuuki藉助防禦塔以及輔助支援二打三打出了一波1換1。

12分鐘，V5五人包上，LGD四人在塔下，上單鱷魚沒有支援，雖然V5第一時間擊殺了Yuuki的冰女，但是Kramer的霞完美倒鉤，LGD打出了一波1換3。14分鐘，V5三人抓下想要擊殺LGD上單鱷魚，但是沒想到征服者鱷魚傷害太高，吸血無敵，成功一打三拿到了三殺。

20分鐘，V5在中路盲僧先手開團，配合隊友擊殺了Kramer的霞。21分鐘，V5妖姬想要消耗沒想到直接被LGD控住，自己被擊殺，隨後Yuuki冰女在上路被V5三人包夾擊殺。25分鐘，V5輔助做視野被LGD眾人抓死，隨後LGD選擇打大龍，V5其餘人趕來阻止，雙方一波混戰打出了一波1換1，阻止了LGD拿大龍。29分鐘，V5找到機會秒掉了Yuuki冰女，之後V5五打四，但是鱷魚太猛，LGD擊殺了V5四人，拿下了大龍。

31分鐘，LGD直接越塔開團，隨後一波結束了比賽。
