魔獸世界懷舊服中的無形殺手，如何成為一代刺客大師

天天要聞
2019-12-21

魔獸世界懷舊服已經火爆了3個多月，至今熱度不減！隨之前幾天的又一劑猛葯的灌下，兩大經典戰場戰歌峽谷與奧特蘭克山谷的落地，每天上線排隊就成了魔獸界的一大亮點，今天你排多少？

經過了許多天的戰場生涯，不管是大規模的40人奧山戰役，還是小規模的10V10戰歌峽谷，盜賊的數量還是在各個職業中比例很大的，因為盜賊這個職業充滿了神秘感和操作性！同樣，盜賊操作的水平的高低、決定了你最終是小毛賊和盜聖。

在遊戲中經常聽到一些玩家在說背刺這個技能沒有什麼用途，因為找後背太難了（一般PVP中只有被定身，才有機會給你背刺的機會），而且消耗的能量還那麼多，其實無論在平時練級SOLO還是在與玩家廝殺，背刺對於盜賊來說是一項非常給力的技能（傷害足以彌補它消耗能量的代價）。 那麼如何運用好這項技能呢？

通常盜賊比較常用的方法：戰鬥中先鑿擊，然後繞到對手背後進行背刺，但必須注意自己能量值多少（積累能量一定要累積到能夠連續釋放一次鑿擊和一次背刺） ，不然鑿擊得手後，對方陷入暈眩狀態，結果都醒了還不夠能量使用背刺就尷尬了。利用這個方法不僅能夠對敵人造成兩次較大傷害，而且，在你釋放出背刺後，敵人還需要一點轉身時間（可以利用側後身45%角繼續背刺），抓住這點時間，你還能對敵人造成兩次普通攻擊的傷害。

就此機會我給大家推薦一個我自己琢磨出來的另類天賦，戰匕！可能大家比較常見的戰鬥天賦都是配合著單手劍或單手錘，衝動開了就是一頓的砸臉。我這個天賦的另類之處，就是一旦被抓住背身，那麼這一次刻骨銘心的背刺會讓你記住一輩子的。

想到這裡肯定有人會說，PVP的話，誰會給你那麼長時間的後背讓你來搞？這是當然，不過在戰場中完全有可能，別忘了例如奧山平原上的混戰，幾十人對轟，人群中還有幾個人顧得上自己的菊花不保？奢侈點的話，在喝上一碗盜賊專供的菊花茶，估計板甲職業沒人關注治療的話，都得被你這一波刺爆。

另外對於還在處於練級的盜友們，背刺這個技能打怪來說，每個怪都要繞後背的話，就會大大的降低了我們SOLO升級的效率，還是抓緊時間到60，我們一起去戰場搞事情吧！
