同盟種族是魔獸8.0版的主要內容之一。玩家可以通過完成一個任務線與那些曾經有過友誼的人結成同盟。當然，這裡的盟友不只是說說而已。同盟種族和舊的核心種族除了解鎖方法不同外是相同的。玩家也可以在遊戲中玩這些同盟種族。

在遊戲的早期，暴雪為部落增加了三個同盟種族，分別是暗夜精靈之子，高嶺牛頭人和瑪格漢獸人。在每個種族加入部落的背後，都有一個相應的故事情節。今天，我將向你展示瑪格漢獸人是如何加入部落的。

隨著部落和聯盟之間的戰爭越來越激烈，雙方都迫切需要新的盟友加入。部落的獸人提到，許多年前(也就是在6.0資料片期間)，部落前往另一個世界的德拉諾，幫助那裡的棕色皮膚獸人。現在是回到他們身邊的時候了。我們將再次前往平行世界的德拉諾，讓這些瑪格漢獸人派遣軍隊加入部落的事業。

然而，再次回到平行的德拉諾世界並不容易。幸運的是，青銅龍看到了僕人的希望，願意幫助我們回到德拉諾的過去。

然而，在德拉諾的世界裡，我們發現它不同於我們離開的時候。世界變得越來越貧瘠。德萊尼人到處屠殺獸人，要求他們皈依聖光信仰。

你問了當地的瑪格漢獸人。事實證明，和平確實在德拉諾解放後出現了一段時間，但並沒有持續太久。不久，一群名叫納魯的生物來到了德拉諾，在那之後，德萊尼人變得非常憤怒，聽著納魯的引誘，把世界的荒蕪歸咎於獸人。德萊尼人甚至逮捕了一群瑪格漢獸人，強迫他們皈依聖光，並與他們的前同胞成為敵人。半獸人稱叛徒為光明同盟。

在與光明同盟和德萊尼的戰鬥中，許多瑪格漢獸人在戰鬥中死亡，包括霜狼部落的首領杜隆坦。甚至連戰歌氏族首領，地獄咆哮的唯一兒子，也被德萊尼抓了起來，並被洗腦，變成一個光縛者。

知道真相的步兵幫助瑪格漢抵抗聖光的暴政。然而，即使他像腳男，也很難面對整個德萊尼人的力量。隨著瑪格漢的衰落，德萊尼人出來消滅德拉諾的所有異教徒。

在最後一刻，地獄咆哮要求你把瑪格漢剩下的部分帶到艾澤拉斯，他自己也因此而死。

以上是小編帶來的瑪格漢獸人招募任務線。我不知道那些瘋狂的德萊尼會有什麼下場呢?
