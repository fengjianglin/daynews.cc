完美為了這款遊戲改變自己的氪金風氣！堪稱今年最重視的手游

2019-12-18 1,217條評論 2,248次閱讀 1014人點贊

★《新笑傲江湖》★

這是一款由完美世界出品的武俠類手游，沒錯，就是完美世界，以氪金量出名的廠商，不得不說完美世界的手游真的是氪金嚴重，而這款新游能否改善玩家們眼中的這家廠商呢？

遊戲點評

精緻的畫面體驗

作為一款武俠手游，首當其衝的就是它的畫面體驗了，毫無疑問廠商在這方面下的功夫不可謂不多，遊戲的整體畫風以我國傳統的國畫畫風為主，往大了看，場景視角的構圖十分講究技法，給人一種很大氣的感覺。往小了看，在人物的刻畫上又極其細膩，著重突出角色自身的氣質。

在場景部分，遠處的景色儘是山水，洛陽、福州、華山等地的獨特風光都被用水墨畫的風格所構築出來。玩家遠遠望去這風景就是一幅幅山水畫，偌大一個江湖各處景色都有著其特別之處，想必這能夠滿足廣大風景黨在遊戲中遊山玩水的需求了。

角色設計

由於《笑傲江湖》一書並沒有特指其故事發生的年代，所以不少作品中角色的樣貌衣著基本都是靠著金庸先生在書中的描述來還原。在《新笑傲江湖》中也是如此，在人物形象方面，整體畫風采用融入了一些水粉、國畫元素的國風風格，給人以一種清新淡雅的感覺。也是與整體武俠風格比較搭配

原著中的角色們在遊戲中也被以一種更符合現代年輕人審美的形象還原出來，不過依舊沒有脫離原著中的描寫，甚至可以說在細節的刻畫上下了大功夫。比如說東方不敗身上就有著不少細節，如衣服上的太陽紋、江崖海水紋以及腰間的日月飾品等。精心刻畫的角色形象加上原著帶來的劇情加成，能很好的引起新老玩家對於本作中角色的好奇心，會讓人想要去了解在這個新作中有了新形象的老角色們是否也有了其他的變化，畢竟本作的劇情並不局限於原作。

角色配音方面也十分有趣，在遊戲中，有些角色說的並不是什麼標準普通話，而是各地的方言，比如青城派的那些人都操著一口四川話，在我觀看那段劇情的時候可被那些人的配音給逗樂了。可以說在塑造人物和場景方面，遊戲做的非常不錯！

算不上成功的氪金改革

在今年的CJ期間，完美世界高級副總裁佟慶先生透露完美世界解決用戶痛點的方式，一個是取消VIP，其次是增加免費資源產出，實現免費和付費玩家平衡。筆者體驗《新笑傲江湖》後，這兩點確實在遊戲中得以實施。

本來《新笑傲江湖》沒有VIP系統，也沒有時不時彈出的首充或各種誘導充值的界面，但增加了競技遊戲常見的「成長計劃」(通行證系統)。與同類產品相比，算是比較好地引導玩家理性消費，值得鼓勵。但是卻仍然讓一批玩家感覺不適~

準備混日常混保底就想玩的玩家請謹慎入坑。副本爆率下調就這6 7天打江湖話本剿匪和迷宮就出過一次橙裝基本都是垃圾分解。雖然風雲錄出橙裝幾率高但是一天只能一次，除非12塊錢禮包給一次次數。（最高的一個禮包128我記得。6 塊12塊38 68 128）商城裡面得綁元每日限購每周限購都得買。強化石天工石銘文石想玩這都是必清的東西。（最噁心的是不讓你綁元換銀兩xxx），因此玩家如果不想氪金的話，一定要佛系遊戲才行！

小高總結

讓小高覺得不錯的是其「補償」系統。哪怕玩家沒打，只要達到一定等級即可獲得相應百分比的資源。這樣的設定大大降低了玩家的壓力和疲勞感，且不至於和其他玩家差距太大。同時，鼓勵玩家進行特定遊戲行為，如活躍、拜師和幫會等，可獲得額外的獎勵比例。

可以看到完美的確在變好，遊戲整體也很不錯，只是氪金和資源產出方面還需斟酌一下。遊戲也將於12月19日開啟公測，感興趣的小夥伴可以通過下面的方式進入遊戲預下載哦~
