Doublelift與TSM老闆前女友Leena同游上海，心疼大猩猩隊長

2019-12-16 850條評論 4,910次閱讀 3247人點贊

熱血電競，品味人生。An妮遊戲說為您播報，期待您的關注！

說起大師兄Doublelift，他算是電競圈中的活化石了。而且今年他成功率領TL擊敗C9拿下LCS夏季賽冠軍，以一號種子晉級到S9全球總決賽。拿下這次冠軍後，也刷新了Doublelift的人生記錄。當前大師兄已經奪得了7個聯賽冠軍，是目前世界上排名第二個人聯賽冠軍最多的選手。（第一是李哥）

只可惜今年大師兄雖然打破了世界賽無法晉級決賽的魔咒，但是依然沒能打破S賽的十六強的魔咒。在今年S9全球總決賽D組的生死戰中，大師兄所率領的TL不敵我們LPL的iG戰隊，遺憾止步十六強。雖然TL輸了，但是我們還是要把掌聲送給他們，也要把掌聲送給大師兄。堅持這麼多年，依舊能保持這麼好的競技狀態，實在是難得！

在S9結束後，各大賽區進入了休賽期。而我們的大師兄Doublelift趁著這段休息期，來到了魔都上海旅遊。不過，這次大師兄不是一個人，好像還帶著一位女伴。

北美第一AD選手大師兄來到上海的消息，很快就被一些媒體知道了。他們對大師兄進行了採訪，而大師兄的女伴也被隨之曝光。

而這次大師兄帶的女伴就是他前段時間的「緋聞女友」Leena。說起Leena大家可能不是很理解，但是她之前的男朋友相比大家都有一定的了解。Leena的前男友就是前任TSM隊長、現任TSM老闆大猩猩隊長。

Leena的前男友

要知道大師兄之前是TSM的AD選手，當時Leena還是大猩猩隊長的女友，也就是大師兄的老闆娘。而在大師兄離開TSM來到TL後，Leena就被曝出與大猩猩隊長分手，與大師兄在一起了。不過這些在大師兄帶Leena來上海旅行之前都是謠言，而Leena之前也公開表示：大師兄只是朋友，並不是大家想的那種關係。

而近日大師兄的發布的旅遊照片，印證了他和Leena的關係並非想像中那麼簡單。圖中的Leena正親吻著大師兄的臉頰，雖然說外國貼臉親吻是一種禮儀，但是這樣把親吻方式只能是男女朋友之間才能做的。看得出來，這次大師兄不光是來旅遊的，還要向全世界宣告Leena就是自己的現任女友。

很多網友看到這一幕後，紛紛心疼大猩猩隊長。沒想到大師兄不但離開了TSM，還把老闆娘給帶走了，真是太不厚道了！不過，An妮覺得感情這事不能強求，雙方的意思才是關鍵。既然Leena選擇了大師兄，想必大師兄一定有更吸引她的地方。
