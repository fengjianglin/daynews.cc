遊戲中我們是朋友，聊天侃地，在這裡我們可以無拘無束的發言，不會有任何人阻撓，還有大家最喜歡吐槽的小編，請把口水收集好，隨時準備和小編一起吐槽！

魔獸世界懷舊服騎士因一個技能秒卡扎克，可惜再過幾個版本就沒了。這個技能就是騎士的清算。

記得以前玩部落對騎士技能不清楚，在燃燒平原被一個騎士秒了，看著戰鬥記錄研究了十幾分鐘不知道怎麼回事！那時候清算無限疊加，和盜賊決鬥攢清算，然後秒殺卡扎克。

最早的清算是沒上限的，可以無限疊加。一騎士找一個賊決鬥，捅爛了好幾把白色匕首，扭頭一到清算秒了卡扎克。這個是真事兒，就是因為這個bug般的事件，清算被改成最多疊加5次了。

這個技能1.3前的版本能無限積累清算還可以玩玩，後面的清算被改過幾次，不能觸發命令聖印，而且命中率被改低了。你們是沒體會過好不容易集滿一次清算打出5個白數字，然後發現裡面有3個未擊中的心情。

我有一次戰場，被DZ暈，旁邊一個zs一個fs，三人一起殺我，脫控後一刀秒瞞血DZ，平砍前制裁即可可以做宏制裁接平砍昏迷中無法閃躲招架格擋！

這個職業就是看臉，普攻+命令出暴擊，一般職業半管就沒了，普攻未命中，命令不觸發，對面屁事沒有，自己半管血沒了。

我最新發現死亡之後清算不會消失，昨天黑龍門口被群毆之後跑屍找了個在人群中喝水的牧師秒了，然後開了無敵瀟洒離去，幾十個部落在一邊乾瞪眼。

清算蠻好玩的，和我朋友T2畢業匕首盜賊，決鬥，穿綠龍皮甲，奧山豬頭錘，盜賊先手，醒了，制裁，直接秒盜賊，要不就是，昏睡了，可惜這個皮甲現在削弱了。以前沒有級別限制，昏睡30秒好像，自己坐下吃喝。
