和平精英：落地撿到什麼槍最強勢？AKM穩定人心，噴子無所畏懼

天天要聞
2019-12-18

在和平精英中，一般跳傘落地後身邊都會有幾名敵人，這個時候誰最先拿到武器就能獲得極大的優勢。所以這個時候大家都會衝進房區，瘋狂的尋找槍械來準備迎戰。那麼在我們跳傘落地的時候，拿到什麼槍械才是最猛的呢？

首先第一個要說的就是衝鋒槍了，我們都知道衝鋒槍在近距離的對戰中表現得十分出色。不僅有著快速的射速，傷害也不是很低。而且瞄準也比較簡單，最關鍵的是，衝鋒槍所使用的9mm子彈非常容易補充，這樣就能夠在前期的對戰中發揮的很優秀，殺掉幾名玩家自然不是什麼難事。

第二把要說的就是我們熟悉的AKM了，這把槍在沒有安裝瞄準鏡的情況下壓槍並不是很難。而且AKM的子彈傷害非常出色。在前期剛剛開始遊戲大家都沒有什麼防具的時候，一把AKM足以將敵人瞬間擊殺。所以見到AKM撿起來不要慫，上去收割就完事了。

第三把就是大家最喜歡的槍王M416了，這把槍穩定性極佳，雖然在前期剛剛撿到可能沒有什麼配件，但是比AKM穩是肯定的。除了穩定性以外，M416的傷害能力也是不錯的。最重要的是大家對這把槍都很熟悉，很容易就能發揮出它的威力，所以如果在落地撿到它的話，還是很容易擊殺自己附近的敵人的。

第四把就是大家熟悉的噴子s686了，這把槍雖然在後期非常雞肋。但是在前期的對戰之中，還是很少有槍能夠比他更加優秀的。雖然只有兩發子彈，但是瞬發傷害爆表，而且攻擊範圍很大。所恨能夠很輕鬆的將身邊的敵人擊殺。在遊戲的前期拿到一把s686，可以說就相當於無敵了，大家覺得是不是這樣呢？
