這裡有最新最全的英雄聯盟資訊，菌菌帶你看不一樣的召喚師峽谷~

在LPL全明星周末結束之後，迎來了拳頭舉辦的LOL全明星賽事，這個賽事的玩法也是多種多樣，分別有：無限火力模式、1V1單挑賽、雲頂之弈慈善賽，5V5表演賽、雙人共玩五個環節，而Solo賽、無限火力和5V5表演賽是最受粉絲們歡迎的，而參加這次LOL全明星的都是各自賽區的明星選手，例如：TheShy、Uzi、Doinb、Tian、Faker、Caps等都是世界級明星選手，而這次Solo賽這些人都將會參與進去。

菌菌是一位喜歡打野的玩家，比較喜歡玩瞎子，所以就十分期待最後一場1V1單挑賽，由LCK的野王小花生對陣LPL的新野王小天，兩位都是世界上最好的瞎子玩家之一，所以菌菌在賽前就預測了一波，希望他們兩個都拿出瞎子來SOLO，想必大家也是這樣的，兩人不負眾望共同選擇了瞎子這個英雄，下面我們來看下賽況！

兩人的出門裝和符文都大致相同，兩人都選擇了不滅之握符文，因為玩瞎子的人都知道帶這個Solo會十分得勁，小天出門裝選擇了兩把多蘭劍+布甲+回血寶珠+一瓶血，而小花生的出門裝比較經濟，也是兩把多蘭劍+布甲+一瓶復用型藥水，菌菌認為小天這樣的出門會更具優勢，因為瞎子前期的碰撞最主要的還是E技能的消耗，而回血寶珠會更實用。

兩人前期還是比較乏味的，但是小天的細節做得更好，E技能多次命中，一直都在壓制著小花生的血量並且推線，補刀也是小天比較多，多次把兵線壓在了小花生塔下，這樣提前把兵線推過去是有利的，因為刷圈的時候自己會比較有優勢。

在兩人反反覆復地消耗之後，小花生忍不了了，上來就是揍小天，但是小天完全不虛，交點燃然後和小花生對A，最後一腳把殘血的小花生踢回塔下，但是被小天的精準Q收掉，小天拿到了一血，並且把兵線推了過去。

回到家的小天出了兩件關鍵裝備，提亞馬特+荊棘背心，菌菌認為出提亞馬特的優勢就是推線了，因為之後會有刷圈機制，哪邊兵線有優勢哪邊的勝率就會提高，因為圈是往地圖中心刷的，除了提亞馬特推線妥妥的，而出荊棘背心的目的就是削弱對方的吸血能力，畢竟瞎子的W是主要的恢復手段，小天選擇出了荊棘背心不止削弱了對方的恢復能力還提高了自己的坦度，小花生這邊選擇回去出了小穿甲和布甲鞋，在之後還補齊了一個死刑宣告，菌菌認為還是小天的荊棘背心比較划算。

兩人回到了線上繼續對線，瞎子的Solo還是挺無趣的，小天憑藉著等級和裝備的優勢，持續把線推過去，一直到了縮圈，小天這邊肯定是佔優的，於是兩人選擇了皇城PK，最終小天R+點燃把小花生帶走，拿下了最後一把Solo賽，恭喜小天！

小天晉級下一輪將會面對來自歐洲G2戰隊的野王Jankos，感覺這幾個小組十分得勁啊，看完了LCK和LPL野王較量，接下來就可以看LES和LPL野王之間的對抗了，不過目前小花生已經加入了LGD戰隊，確認下賽季參加LPL2020賽季了，這兩位野王提前對決了，希望大家給這兩位LPL野王一些鼓勵！
