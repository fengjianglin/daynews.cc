eStar冬冠名單惹LOL玩家指責 貓神ID遭質疑 曝光時間成主因

天天要聞
2019-12-13

王者榮耀是目前最受歡迎的手機遊戲，在我國有數億玩家，可以說是國產遊戲的巔峰之作。作為MOBA類遊戲，王者榮耀的誕生時間比較短，從2015年發行後，到現在也不過只有四年，而與其同為同類型的端游LOL則有近十年的歷史。自王者榮耀發行以來，與英雄聯盟的比較就屢禁不絕。LOL玩家多次指出英雄涉及抄襲，比如花木蘭（銳雯）、后羿（艾希）、亞瑟（蓋倫）等多位英雄都曾被被提及。甚至連很多KPL職業選手的ID都無法倖免。

比較出名的就是當初兩個諾言的矛盾。當時的王者榮耀還沒現在這麼受歡迎，KPL的比賽也沒有現在正規。職業選手的數量也不多，但就是在為數不多的選手中，有一位的ID卻引起了大家的注意，他就是eStar諾言。相信對LOL比賽有一定了解的小夥伴都知道，諾言在LPL中是明凱（EDG戰隊打野選手）比賽ID翻譯成中文的意思，並且用時很久，遠比eStar諾言要早。而且廠長成名時王者榮耀還沒發行，自然導致很多LOL粉絲的厭惡。

最終，雙諾言之爭以EDG諾言勝利為終止，但截至今日，依舊有粉絲糾結於此。近日，王者榮譽eStarPro戰隊公布了冬冠的首發名單，卻意外引發了很多LOL玩家的憤怒，甚至有很多網友前去eStar炮轟。根據評論的內容來看，還是曾經出現過的ID相似問題，甚至有人稱「eStar就是專門蹭熱度的戰隊！」並且這次指責的對象不僅局限於諾言，甚至連貓神（Cat）、WUXIN都不能倖免。而其中，爭吵最激烈的無疑是貓神的昵稱。

貓神是目前最受歡迎的王者榮耀職業選手之一，粉絲無數。這次關於貓神的質疑，是因為他與LOL解說Cat有相同ID。LOL玩家認為，解說Cat從15年就開始打比賽，而貓神是在16年出道，從登場實踐來看貓神無疑是抄襲了解說Cat的ID。但王者榮耀玩家認為，貓神從登場那天就眾所周知，而解說Cat一直默默無聞，直到近兩年轉型成解說才逐漸被大眾認知，而且即便現在也沒有貓神的影響力大，說貓神偷ID純屬無稽之談。

說實話，這種爭論完全是沒有必要的，畢竟LOL和王者榮耀根本就是兩個不同的遊戲，互相之間也並不會在同一個賽場比賽，因此兩個諾言永遠不會成為對手，解說Cat也永遠不會為貓神當解說。至於英雄問題，即便時王者榮耀忠實粉絲，也不得不承認最開始很多英雄的相似程度的確很高，也正是因為這樣，很多英雄才會大改，比如花木蘭。但如今，王者榮耀英雄設計的精妙程度，比起LOL毫不遜色，借鑒都已不復存在！

很多吃瓜群眾可能也會感到疑惑，貓神這個ID都用了好幾年了，為什麼會在這個時候爆發出衝突，真的是後知後覺?其實，這次引發矛盾主要是eStar冬冠名單公布時間的原因。十二月12日，英雄聯盟第十七支戰隊正式進軍LPL，該戰隊就是eStar。而且eStar是以與PDD合作的形式加入的LPL。雖然絕大多數網友對PDD和eStar都是表示恭喜和支持，但總有記仇的玩家，依然不放過任何吵架的機會，其實完全沒有必要，難道這對LPL來說不是好事嗎？

大家認為LOL選手和王者榮耀選手最大的區別在哪裡？
