人人都是寶箱專業戶，魔獸世界懷舊服野外開寶箱賺錢方法大揭秘

2019-12-10 260條評論 1,729次閱讀 303人點贊

Hello，各位《魔獸世界》的朋友們大家好！

魔獸懷舊服目前已開放三個多月時間，相信大部分第一時間進軍懷舊服的玩家都有一個相同感受：「G幣不夠用，技能不敢亂學，點天賦慎之又慎，害怕40級騎不上心愛的小馬」，當然也包括小編在內。直至現在，很多小號玩家依然在飢餓線上徘徊，過著包里長期G不過百的日子。

那麼今天小編就帶大家來一起盤點一下60年代野外開寶箱賺錢的好方法，同時可以練採集類副職業，希望對懷舊服玩家能起到一定的作用。

寶箱刷新機制

首先，我們要重點說一下寶箱刷新機制，基本每張地圖都有最多3個左右的寶箱，3個寶箱開完後，刷新時間大概在15分鐘左右，也就是說我們可以在相鄰的兩塊地圖之前來回跑動。由於考慮到懷舊版本沒有飛行騎術，所以沒有千G馬的玩家可考慮集中在一塊地圖，等待的同時可以練副職業、打怪賺錢均可。

寶箱出貨機制

根據地圖相適應等級，每個地圖寶箱開出來的物品等級會比地圖等級低5級別左右，例如40級的地圖，寶箱開出來的物品就是35-40級之間，50級的地圖，即可開出45-50級裝備，如下圖所示：

地圖選擇

記得以前有採集插件可以顯示野外寶箱刷新位置，但放眼目前懷舊服，必然是只能人工尋找了。與此同時，選對一張合適的地圖，便能事半功倍，增加收益。

首先，推薦的地圖是冬泉谷，做冬泉霜刃豹的地方，大家都不叫熟悉，畢竟處於LM區域，BL玩家的話就得小心了，不然寶箱開到一半被LM盜賊偷襲，可謂損失慘重。推薦冬泉谷的原因有兩點：其一，箱子密集；其二，容易開出「欺詐寶珠」，這件飾品在60版本價格不菲，將你的角色變成敵對陣營，相對於免費獲得了永久轉陣營，價格上千G也不算貴，具體看各服金幣比例。

其次，推薦環形山和希利蘇斯。這兩塊地圖相連，並且地圖不大，尋找寶箱相對簡單，兩邊來回跑動，箱子開出的物品等級基本在47-58左右。對比效率，和冬泉谷相差無幾，畢竟等級擺在那裡。比較容易開出的藍色裝備有「亞歷山大的戰斧」、」黑暗幻影斗篷」、「世界之樹的的嫩枝」，都能賣出不錯的價錢。

另外提示一下，希利蘇斯也是採藥玩家的好去處，黑蓮花刷新點，能夠做到開箱採集兩不誤，收益加倍。而且希利蘇斯共計有9隻稀有怪，刷新時間大概在4小時左右，如果打完開出裝綁紫裝，那就恭喜你發家致富。

以上為60版本部分地圖開寶箱建議，當然跑地圖開寶箱也是一件需要毅力、需要RP的事情，所以希望大家開箱路上放平心態，早日開出自己心儀的物品！

本期分享到這裡就結束了，關於魔獸世界懷舊服開寶箱賺錢小竅門，您怎麼看呢？歡迎參與討論，提供更多優質思路哦！
