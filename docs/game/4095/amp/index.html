王者榮耀：暈眩流馬可波羅攻略，和誰對拼都不怕

天天要聞
2019-12-09

摘要

馬可波羅算是職業比賽中的熱門射手，據統計在2019KPL秋季賽，上場率位居全英雄第四，高達37.7%，排在其前的只有盤古、曜、豬八戒，由此足以可見馬可波羅的強度。

馬可波羅算是職業比賽中的熱門射手，據統計在2019KPL秋季賽，上場率位居全英雄第四，高達37.7%，排在其前的只有盤古、曜、豬八戒，由此足以可見馬可波羅的強度。

但是在常規排位賽中，勝率並不是很高，只有47%，不是因為這英雄不強，而是因為他的上手難度高，技能放不準、反應相對較慢的玩家很難直接掌握他，高難度的英雄一般也對應著高收益，馬可波羅就是這樣的英雄，在當前強調對線的版本中，馬可波羅幾乎不怕任何人，暈眩流的馬可更是可以在對線、團戰中都打出不俗的表現，這次芃叔就為大家講解下當前版本下馬可波羅的打法。

技能核心理解

被動——核心

馬可波羅的技能和普攻都會破壞目標1層防禦（對小兵為5層），達到10層後回復30點能量，並且進入防禦破壞效果持續5秒，期間馬可波羅每次傷害都會額外造成70+30%AD真實傷害。

這決定了，馬可波羅的輸出需要一個前置條件「破壞10次防禦」，也就是技能或普攻命中10次，而在接下來的5秒內造成的傷害就會有額外的真實傷害加成，才能打出爆發輸出。

一技能——遠程消耗手段

馬可波羅朝指定方向連續射擊並獲得10%移速加成，每一槍都會觸發普攻法球效果，額外攻擊速度會影響技能射出子彈數目，在攻速達到0/75%/150%時射出子彈數5/7/9。

該技能的升級不能影響傷害，隻影響釋放技能消耗的能量，但還是推薦主升該技能。因為不管對線還是打架都需要頻繁釋放一技能消耗敵人，如果命中那還好說，未命中的話一技能將消耗75能量，對於戰鬥節奏會造成一定負擔。

二技能

位移技能，還附加一個被動，當500範圍內存在敵方英雄時，馬可波羅增加20%傷害和一定移速加成。

被動觸發時，馬可波羅身邊會出現如圖所示的波紋圓環，該效果同樣可以用來探草叢。

三技能

馬可波羅朝指定位置閃爍，並向周圍發射彈幕，可觸發普攻法球效果，額外攻擊速度會影響技能射出子彈數目，在攻速達到0/75%/150%時射出子彈數5/7/9。

既然是一段位移，當然也就可以用來突進和逃跑，和二技能搭配可以達到兩段位移的效果，逃跑時選擇先二再三的手法，再用移動或普攻取消大招，因為取消大招不是瞬發的，如果先三再二，會造成一定的卡頓。

小結

馬可波羅的被動決定了他的打法，通過一技能消耗累計破壞防禦次數，快要達到10層後可通過大招入場，觸發防禦破壞效果，再由大招的快速打擊頻繁觸發被動的真實傷害，打出爆發傷害。但並不是一味的遠程消耗，懼怕近身，馬可波羅二技能附帶的被動表明，周圍有敵人會使他的輸出加成20%，所以當被近身我們要會放風箏，也要瞅準時機開大入場。

出裝銘文思路

馬可波羅的出裝銘文需要經過嚴謹的計算，因為作為一個射手他需要卡攻速閾值，又擁有與攻速相關的技能，所以合理的攻速對他很重要。

1.馬可波羅核心是攻速類裝備，而馬可波羅的技能都可以觸發裝備的法球效果，所以末世、純凈蒼穹最為合適，首選的3件裝備就是攻速鞋、末世、純凈蒼穹。

2.銘文推薦10紅月，10鷹眼，7狩獵3奪萃。這樣在出完3件套並且自身到達10級時，即可堆到150%攻速，馬可是一個偏向中前期強勢的英雄，所以儘快達到150%攻速，我們儘早進入強勢期，發揮最大效用。

3.之後選做一件肉裝，因為中前期以物理傷害居多，法師又沒完全發育成型，所以通常做護甲類裝備。目前常出的有不祥徵兆和冰痕之握，通常情況下我會選擇不祥徵兆，這件裝備在和射手對拼、防止刺客突臉上有很好的效果，大大增加我們的容錯率，而冰痕之握則更偏向進攻，在順風或己方控制較少情況下比較適用。

4.有了輸出有了防禦，在中後期還需要一件可以破壞敵人護甲的裝備，而當下暗影戰斧和破曉都是適合馬可波羅的裝備。敵方輔助為軟輔，且邊路英雄非坦克類，可以選擇暗影戰斧；敵方陣容中有坦克類英雄，或出肉人數大於等於2人，且有不祥徵兆（不祥徵兆會減30%攻速），則選擇破曉。

5.最後選擇出魔女斗篷，入場時可以降低法師傷害。也可以選擇賢者的庇護+名刀的組合，不過對經濟要求較高，合成不那麼順滑了。

綜合以上，我推薦的出裝如上，但還是要根據具體的局勢適時調整出裝，方能應變各種局勢。

召喚師技能比較

召喚師技能中，比較適合馬可波羅的有3個，分別是閃現、凈化和暈眩。

狂暴流馬可，我記得在S6的時候還是非常流行的，那時候馬可出無盡帶奪萃也很厲害，後來馬可重做了，狂暴流就消失了。狂暴的60%攻速10%攻擊，讓馬可波羅前期固然強勢，可是一旦到了中後期，攻速溢出，效用就很低了。

閃現是一個萬金油的召喚師技能，有閃現後，英雄更加靈活，馬可波羅開啟大招的時候同樣可以用閃現調整位置。但是馬可波羅本身就有兩段位移，已經夠靈活了，而且大招入場之前，本來就可以使用一次二技能調整一下距離和位置，大招也有固定方向的位移，兩段位移都沒辦法幫你調整到合理的輸出位置嗎？所以閃現並不適用。

凈化不但可以解除負面效果，還可以在接下來1.5秒免控，是一個不錯的技能。該技能進可攻，退可守，被控制可以解控，大招入場可以提前開凈化躲避控制，讓大招可以放得更順暢。

暈眩CD是90秒，比閃現、凈化都少了30秒，節奏上也能適應更頻繁更激烈的團戰。暈眩會眩暈敵人0.75秒，之後減速1秒，和凈化相比，暈眩更偏向進攻，作為控制是限制敵人行動的，在殘血後，暈眩可以殘血反殺的，而凈化就沒這種效果。

另外暈眩在對線期更強勢，打大部分射手時，比凈化更好用，例如馬可打公孫離，有暈眩控住公孫離還有一戰之力，帶凈化則顯得劣勢的多，尤其在這個重視對線的版本，對線強則影響全局，而且暈眩在團戰中的效用也很強，像這波入場配合暈眩，完全有一種戰士突後排的感覺。

暈眩流打法應用

暈眩要求玩家打的有進攻性，主觀上偏找人打架，不能慫，越慫越被欺負，因為暈眩的範圍比較小，又只能暈身邊的人，當被人追著打時候，可控性較差。而在找對手硬拼時，則足以顯現它的強大。

前期我們盡量找機會用一技能消耗或者對拼換血，玩馬可千萬不要想著一下就把對面打死，我們不怕換血，能疊加10層破壞防禦固然好，疊加不出，只要能做到等量換血也不虧。現在馬可的技能傷害在前期是加強的，傷害很高，而且我們帶了眩暈後，在殘血對拼時，常常可以反敗為勝，0.75秒的眩暈，時間雖然不長，但是此長彼消，就可以造成擊殺。

帶暈眩的馬可波羅很強勢，但並不是莽，有凈化入場容錯高，帶眩暈則更考驗入場時機，我們需要仔細觀察敵方的控制技能是否交過，利用敵方人員密集時，開大入場，暈眩控場，打出真實傷害。

如GIF，我提前觀察，敵人把技能都給隊友後，我再開大入場，配合暈眩，將敵人全部收割，完成四殺。

總結

暈眩，是生而逢時的召喚師技能。當前版本比較強調對線能力，打架也非常頻繁，暈眩技能本身就進攻性比較強，0.75秒的控制和1秒的減速，讓他在單對單對拼時，不懼怕任何人。馬可波羅又是一個中期崛起的英雄，強勢的召喚師技能搭配強勢英雄，在強勢期可以無限擴大優勢，進而終結比賽，這也是帶暈眩的馬可波羅最大的意義。
