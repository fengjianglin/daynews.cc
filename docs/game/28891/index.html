《塞爾之光》英雄本歸來，全新試煉亟待挑戰

2019-12-21 820條評論 4,886次閱讀 3411人點贊

對於一個MMORPG玩家來說，最激動人心的時刻是什麼時候？

或許是魔獸世界開荒了十幾個小時後最終首殺的一刻、或許是DNF強化武器成功增幅15的瞬間。但對於真正的老玩家來說，最振奮的，毫無疑問是當年那些高難副本的復刻！曾經那些從辛苦開荒到熟練速刷的副本，以新姿態重新出現在玩家面前以待挑戰時，瞬間腦海中便會湧出絲絲縷縷的細節。

比如《劍靈》紅蓮副本的炎煌疊滿火種一吼團滅，又比如《魔獸世界》卡拉贊副本的名梗：”卡拉贊沒畢業打什麼卡拉贊？……這一切的一切，都像是昨日剛發生的事情——和大學室友在宿舍沒日沒夜的肝等級肝副本，被剛入坑不久的小夥伴硬拉著”自強血色”，雖然過程很痛苦，但內心的愉悅和成就感還是很足的。

而如今正值《塞爾之光》大版本更新，看著宛如”新三神”般的挑戰副本，咱不禁也回想起征戰在”老三神”的日子，那些BOSS，那些機制，彷彿歷歷在目……

熔岩洞穴

作為新人遇到的第一個副本，熔岩洞穴整體上的難度並不算太高。僅以關底的老山王而言，BOSS技能機制還算簡單，兩個AOE高傷害附帶長時間控制的技能，一個指定方向的位移擊退，除此之外便是一些小範圍AOE技能，躲起來沒什麼壓力。

值得注意的有兩點：

第一是老山王的兩個高傷害AOE，儘管技能抬手時間比較長，但以前開荒也經常會有朋友失手中招。不過好在AOE之後附加的debuff，隊友間可以互助解除，所以威脅並不算大。

第二個要點在於老山王的點名技能，BOSS會隨機指定一名玩家進行位移擊退，當時開荒的時候，看到”老山王已選中誰誰誰請注意躲避”之後都慌得不行，滿地圖亂跑，結果導致隊友一起吃到傷害，再加上一些小範圍AOE技能，經常因此而團滅……

蟲族巢穴

蟲巢整體上的難度按照當年開荒的體感來看，與熔岩洞穴持平。唯一卡過的點在於關底BOSS育母蜘蛛那裡的兩個機制，其一為禁錮BOSS的兩根石柱，其二為後續刷新的小蜘蛛。

前者直接與BOSS的血量掛鉤，後者則是能加速柱子的攻略進度。開荒的時候由於不知道這些機制，每回都是沒打掉柱子，導致BOSS血量回滿，後續輸出也不足以在狂暴之前打掉一管血……

海獸巢穴

相比於前兩個副本而言，海獸巢穴的難度提高了不少，但根據當年開荒的經驗來看，關底BOSS的難度並不是很大。有魚槍的存在，想要打過BOSS暴鯊獸只要輪流上魚槍即可。

主要的難點在BOSS之前的一波小怪。如果輸出不夠，無法在第二波小怪刷新前清完小怪的話，很容易造成後排腹背受敵被秒殺，最終團滅。

總的來說，記憶中的這三個副本屬於有驚無險的類型，難度上並不算大，但這並不意味著其強化版的挑戰副本可以輕易躺過。儘管副本機制、BOSS技能都沒有多少變化，但屬性閾值的倍增毫無疑問降低了副本的容錯率，掛機等大佬帶，多半會吃到各種AOE死掉，而大佬想要輕易1托N還是挺有難度的，所以，利用好各種機制還是有必要的。

更高的難度意味著更好的獎勵，挑戰副本中的獎勵分為首通和總星數，其中不乏神之鐵砧這種用來給高階套裝進階的材料，其他各類打造裝備、換取魂石的材料等應有盡有，只可惜，沒了夥伴一同開荒。

如今四星挑戰副本海獸巢穴已然開啟，而戰友你，會選擇前來參加試煉嗎？
