《英雄聯盟》Knight9飲血劍伊澤瑞爾 征服者吸血流續航無敵

2019-12-16 628條評論 1,685次閱讀 665人點贊

伊澤瑞爾可以說是英雄聯盟人氣最高的角色之一了，基本上常年活躍在職業賽場和路人局。雖然在這次的S10季前賽中，伊澤瑞爾失去了行竊預兆符文，但征服者符文和後續的補償加強，同樣讓他在目前版本擁有超高的出場率，排名下路位置第三。在最近兩周，很多職業選手都在積極嘗試伊澤瑞爾的全新符文和出裝打法。

​

比如Rookie的召喚艾黎打法、Doublelift的強攻打法，甚至還有一些選手出暮刃和奪萃之鐮。而在周末的《進擊！季前賽》節目中，TES中單選手Knight9則是帶來了伊澤瑞爾的新教學。Knight9在這場對戰中發揮非常出色，最終以24/7/13的超神戰績，幫助隊伍完成了翻盤。得益於伊澤瑞爾的機動性和持續輸出能力，Knight9多次有極限反殺對手的鏡頭。

​

符文搭配

雖然在行竊預兆移除後，不少玩家嘗試了其他的基石符文，但目前看起來，征服者是最好的選擇。Knight9也是選擇了征服者，能夠加強對線的對拼能力和團戰的持續輸出能力。

征服者-伊澤瑞爾是一個持續輸出的英雄，疊滿征服者後可以提供額外的攻擊力和回復能力。

氣定神閑-伊澤瑞爾的技能輸出頻繁，氣定神閑重做後可以增加最大法力值，間接提高了傷害。

傳說歡欣-Knight9是多件吸血裝的打法，配合傳說血統和征服者可以達到超高的吸血能力。

致命一擊-提高伊澤瑞爾的斬殺和收割能力。

​

法力流系帶-伊澤瑞爾非常依賴技能，進一步提高伊澤瑞爾的法力值和法力回復效果。

超然-中後期給予冷卻縮減，讓伊澤瑞爾可以堆滿40%的CDR，充分發揮他的技能組。

屬性符文選擇攻速、攻擊力和抗性，不會穿插普攻的伊澤瑞爾不是好EZ，由於Knight9這局是打中單位置帶了魔抗，如果打下路位置就帶護甲。

技能加點

中單位置打得激進一些選擇閃現+引燃，配合打野更容易擊殺，下路位置選擇閃現+治療術，加強對拼時的自保能力。

​

技能加點都是主Q副E，Q技能是伊澤瑞爾的主要輸出和poke手段，主加Q技能沒有懸念，最近的熱修復補丁也是加強了Q技能傷害。E技能則是位移技能調整輸出位置，而且冷卻時間會隨著技能等級大幅下降。W技能是固定冷卻時間和固定額外AD加成，前期加一級就足夠了。

裝備選擇

出門裝選擇多蘭之刃和血瓶，新版本的伊澤瑞爾沒有行竊預兆帶來經濟差距後，線上還是以穩住補刀發育為主。除非是搭配泰坦這種硬控英雄，勾到就直接打一套，疊滿征服者的時候傷害還是很高。之前可能還有爭議回家是先出十字鎬還是耀光，這個版本的伊澤瑞爾只能先出女神之淚，快速疊滿層數升級魔宗，讓自己中期擁有參戰能力。

​

在第二件裝備的選擇上，Knight9和職業選手的主流選擇一樣，放棄冰拳選擇三相之力。三相之力是一件中期非常強力的裝備，能夠搭配Q技能打出很高的傷害，而且提供的生命值屬性比起冰拳的護甲更加泛用一些。除非對手是純物理輸出陣容，否則這個版本優先選擇三相之力，冰拳無法帶來額外的傷害，這個版本的伊澤瑞爾本身沒有裝備優勢了。

Knight9的鞋子選擇明朗之靴，對於伊澤瑞爾而言，中期的CDR比單純的攻擊速度更重要一些，可以幫助他堆滿CDR放出更多技能。由於有了征服者後，伊澤瑞爾回到了S7賽季那種持續輸出的打法，相比最近一年流行的AP混傷打法，Knight9後續都是選擇AD裝備。破敗王者之刃是第三件裝備的最佳選擇，提供了攻速和攻擊力效果，風箏和百分比傷害很重要。

​

除非對手的上中有坦克英雄，否則伊澤瑞爾這個版本不出輕語，水銀彎刀和飲魔刀可以提高自己的生存能力，提供的吸血效果搭配征服者，能夠讓伊澤瑞爾在對拼中保持健康的血量。Knight9在遊戲大後期時，甚至會出飲血劍，三把吸血裝備最大化自己的持續作戰能力。如果你在團戰中受到刺客的針對，那麼選擇守護天使來代替一件吸血裝也是不錯的選擇。

對線和團戰

伊澤瑞爾是一個上下限很高的英雄，而新版本沒有行竊預兆後，玩家還需要一些時間適應打法。伊澤瑞爾的前期能力稍弱，比較吃技能命中率和走位，前期還是以補刀和找機會Q對手疊法力流系帶為主，WA或者WEQ是常見的換血手段，如果Q空了就不要繼續作戰。在出了魔宗和三相之力後，是伊澤瑞爾最強勢的時期，尤其在這個中期對拼頻繁的版本，伊澤瑞爾有機會完成團戰收割。

​

Knight9在高地上以一敵三，技能精準命中敵人，配合征服者傷害打出擊殺。

在對線期間時，伊澤瑞爾非常吃輔助的發揮，搭配不同的輔助英雄有不同的對線打法。如果對抗德萊文或者厄運小姐這種線霸英雄，伊澤瑞爾更多需要依靠兵線或者草叢來消耗對手，不能直接打一波。新版本下路迴廊則是給了伊澤瑞爾風箏對手的資本，尤其對抗那些沒有位移能力的英雄。如果是對抗盧錫安或者卡莎這種單挑英雄，盡量不要給對手沖臉的機會，要和輔助走在一起打正面對抗。

​

Knight9使用E率先近身對拼，飲血劍和土龍魂的盾讓他殘血完成三殺收割。

伊澤瑞爾非常依賴技能命中，尤其在沒有冰拳的減速效果時，Q技能的命中率更加依靠經驗和預判。在面對刺客陣容的沖臉時，伊澤瑞爾需要儘可能避開對手，而不是嘗試反殺對手，躲掉第一波爆發後，你有多件的吸血裝備可以保持健康血量進行後續作戰。由於這個版本的征服者非常強力，伊澤瑞爾在技能之間還是要積極普攻，儘快疊滿征服者。如果是脆皮單挑時，可以E-破敗-W-Q-A打殘敵人，你有飲血劍護盾時有血量優勢。

​

結語

這個版本的伊澤瑞爾還是更適合滿CD和多件吸血打法，配合征服者符文先一套打殘對手，然後迅速追擊依靠自身回復能力強完成收割。穿甲裝的打法缺乏續航能力，暴擊裝的打法不夠出裝平滑，混傷流的打法成型速度太慢。而這次《進擊！季前賽》節目的Jinjiao也使用了下路伊澤瑞爾，同樣也是三相破敗飲血的思路，看起來這種打法是S10賽季伊澤瑞爾的主流思路了。
