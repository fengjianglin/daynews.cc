fgo54選1：最難入手的幾位四星從者 追求全圖鑑玩家的無腦首選

2019-12-04 0條評論 14次閱讀 0人點贊

摘要

fgo對於國服大熱門的54選1活動，這一定是fgo玩家全年最糾結的時刻了，因為情況非常複雜和特殊，所以分成多種不同的情況，去分析，本次現說明一下該活動中涉及到的最難入手的幾位四星從者，她們可以說是追求全圖鑑玩家的無腦首選。

fgo對於國服大熱門的54選1活動，這一定是fgo玩家全年最糾結的時刻了，因為情況非常複雜和特殊，所以分成多種不同的情況，去分析，本次現說明一下該活動中涉及到的最難入手的幾位四星從者，她們可以說是追求全圖鑑玩家的無腦首選。

劇情限定從者

54選1中涉及到的劇情池限定的從者包括示巴女王、段藏、柳生、黑A、高文、崔斯坦、愛迪生、李書文、尼祿，還有下面要說的四位。其中這些都是屬於日常歪不到的四星，但是考慮到近期和未來的卡池，這些並不是最優選，比如說段藏在之前的黑狗卡池就有UP，高文和崔斯坦在後續CCC復刻中也有UP，其他的更不用多說了。

黑無毛

對老玩家來說黑無毛不算稀有，早期在劇情池毒性不高，而且黑無毛UP較多的時候，誰都不缺一隻黑無毛，不過後續她就沒有出過UP卡池，近期唯一一次可能就是HF劇場版的相關卡池了，不過這個卡池抽的性價比很低，所以在不想要呆毛王的話，不如直接換一個實惠（關鍵是呆毛王可以日常等歪）。對於入坑時間不早的玩家來說，換個黑無毛不錯，主要是其寶具倍率非常高，常規用著也還不錯

美狄亞lily

C子的UP極其的稀有，八百年等不到一次，雖然強度確實不怎樣，但如果追求開圖鑑的話，C子也算是首選之一了，具體的功能性還是以奶媽為主，但現在的大環境下，基本上不會給玩家太多的回血打消耗的副本，主要是現在的高難本官方就默認讓你速通的，所以打消耗會異常的辛苦，而且之前FZ活動的太太作為奶媽的作用也不錯，後續還有三星的葯神，為了強度選C子沒必要，為了圖鑑就很有必要了。順便一提C子後續在伊莉雅的卡池中沒有單UP，不論是為了伊莉雅還是美游，她都不是特別好拿。

戈爾貢

動畫第七章的活躍讓很所人對其印象深刻，比C子還稀有的UP，雖然未來會有一次單UP的機會，但可惜因為是在閃閃祭中，所以抽的意義和性價比並不是很高，在考慮圖鑑的情況下戈爾貢也是首選之一，強度就算了，當敵人的時候是充電寶，自己用的時候，就算有強化本也不怎麼好用，復仇者還真就是必須要達到功能或者硬實力達到標準才算能用，否則下場和仇信長差不多。

黑槍呆

強度和稀有度都是一流，目前開千里眼沒見過這從者UP，之前錯過沒拿到的話，無腦選黑槍呆是最好的選擇。立繪卡面絕對的頂級，南半球天下第一系列，槍階金卡都是貴族，而黑槍呆更是貴族中的貴族，當時UP就很難抽到更何況現在這種大環境下，保證高稀有度和強度的情況下，沒有比黑槍呆更加合適的選擇了。
