OMG組建夢幻全華班，打野卻買了H4cker，意在何為？

天天要聞
19 小時 ago

昨天12月16日，LPL賽區轉會期的最後一天，各大戰隊都憋到了最後才遲遲官宣。晚間的幾個小時里，微博熱搜成了電競圈官宣的專場，一條條轉會消息和官宣陣容看懵了不少觀眾。不過，在眾多官宣隊伍中，有一支全華班引起了阿澤的注意，這支隊伍的紙面實力甚至比S9全華班的RNG還要強一些，五個位置不是明星選手就是新人王。

早在轉會期開始的時候，OMG的官博就隱約暗示了SMLZ的回歸，當年SLMZ的大嘴一噴五的傳奇畫面，就是在OMG時期的比賽中完成的。「黑暗勢力」自S3興起，S5逐漸走向衰弱，S7如果不是SMLZ的力挽狂瀾，恐怕已經掉入了次級聯賽的行列，同年SMLZ帶領隊友們再次振興黑暗勢力，殺入了洲際賽，可惜最終沒能闖入世界賽。

S8老賊離開了OMG，來到了RW，這支隊伍在常規賽一路連勝，可惜在世界賽前冒泡賽的最後關頭，被EDG一腳踢回了現實，再次止步世界賽。而這一年，OMG的命運是曲折的，缺乏了SMLZ後的隊伍，實在大打折扣，僅靠著icon一人帶著隊伍打比賽，如果不是聯盟取消了升降機賽制，恐怕OMG再次回掉入保級賽邊緣。

S9的OMG原本找到了國豪，眼看著春季賽就要起飛，沒想到在簽合同的時候，被BLG截了胡，再加上BLG當時的配置確實比OMG好，價格也更高，國豪最後選擇了BLG，而OMG再次跌入谷底。這一年SMLZ加入了SN，但是隊伍的成績一直處於中游，雖然打進了季後賽，但還是倒在了FPX、IG、RNG組成的大山面前。

如果說S9是SMLZ和OMG不得志的一年，那麼S10賽季，就是他們觸底反彈的時刻。轉會期初始，OMG就第一時間聯繫了老賊，並且一口氣趕走了原來隊伍中的六名選手，答應徹底圍繞著SMLZ再次組建一支衝擊世界賽的隊伍。SMLZ的要求也很簡單，在中單位置確認不會變的情況下，重新要來小五輔助（小五唯一能和SMLZ一年交流的輔助選手），上單的新人王Curse，以及原SN的打野H4cker。

這個陣容的大致情況都能猜到，唯獨打野位置沒有猜對，爆料的消息是OMG的新上野都是新人王實力的選手，在RANK和比賽中非常猛，但是H4cker在S9常規賽的表現非常一般，夏季賽後半段還被新人Weiwei按在了替補席上，OMG和SMLZ為什麼要買這樣一名選手呢？

其實仔細看一下H4cker這名選手的履歷，就能清楚他的實力了。H4cker最開始是不太出名的，後來蘇寧買來了Knight之後，兩人的中野組合漸入佳境，並且以上野為核心，在左手和狼行穩定壓線的同時，H4cker極具進攻性的打野節奏才顯現出來，讓SN在比賽前期就能快速的積累優勢，因此還打入了S8賽季的季後賽，拿下了年度最佳陣容提名和二陣打野，也是一名不折不扣的S8新人王。

當時，H4cker的替補就是現在FPX的小天，可以說S8賽季是H4cker最亮眼的一個賽季。但是到了S9賽季，SN重組之後，中單連續輪換，上單找不到合適的選手，甚至出現了Angel和Maple中上互換搖擺的情況。上中兩條線不管怎麼換，都是出於劣勢被壓線狀態，別說H4cker和Weiwei了，Karsa來了也不好使啊！H4cker最大的問題是頭鐵，不穩定，容易上頭，但是他和香鍋一樣，屬於那種前期敢於找節奏，敢進攻敢打敢拼的選手，這是他的優勢，也是風險所在，就看能不能在OMG中鍛鍊出來了。除此之外，應該還有一名新人打野，之前一直在和冷少雙排，也是很猛，但是沒有在大名單里。

而OMG的中上同樣很強，在S9夏季賽的單殺榜單上，OMG的新上單Curse拿下了單殺榜第二的位置，綜合得分僅次於Theshy，是一個不折不扣的新人王上單。雖然很多觀眾一直吹SN的上單Bin很強，TES的上單369很強，但是OMG寧肯放走同樣強力的阿樂，也要留下Curse，可以見得新上單的實力。

中單的Icon就更不用說了，LPL賽區新一代「孤兒院院長」，帶著四個雜毛隊友也能贏比賽，如今給他配置了這麼強的隊友，還怕贏不了比賽？當OMG的三條線都配置了實力超群的隊友，H4cker想拉胯都難，原本侵略進攻型打野選手最怕的就是線上隊友劣勢，可是Curse、Icon和SMLZ都是以個人實力著稱的選手，在LPL里能壓制他們的人還真不多。恐怕這裡最開心的就是Icon了，有隊友的感覺真好！

而最開心的，還是我們這些OMG戰隊和SMLZ的粉絲們！OMG轉會期一口氣弄走了6名隊員，又一口氣買下（續約）了四名大腿級選手，還真是下了血本了。連Icon都在微博中表示，OMG下個賽季的目標只有一個，那就是衝擊S10的世界賽！不管怎麼樣，看到OMG這麼下定決心改變隊伍環境，先不說能不能打出好成績，起碼OMG的態度讓我們看到了，黑暗勢力正是崛起！反觀御三家的態度，還真是讓人著急啊，轉會期買了一堆「臭魚爛蝦」，最重要的是，三家居然都沒有買到上單！你覺得S10賽季，加上IG、LGD、VG這些老牌隊伍，誰能起飛呢？
