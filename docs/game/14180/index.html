原育碧旗下手游大廠Gameloft接連關閉工作室+裁員

2019-12-10 2,388條評論 3,902次閱讀 2654人點贊

作為前育碧旗下的手游廠商的Gameloft在時代里一直在走下坡路，對國內市場不夠重視的情況下，國外的辦事處也在相繼關閉，據Gameloft內部人士透露Gameloft英國辦事處已經面臨關閉，今日（12月10日）又有消息稱Gameloft馬尼拉分公司也倒閉了，同時澳洲布里斯班工作室進行了15次裁員。

之前，Gameloft英國，愛爾蘭和北歐的前董事總經理John-Paul Burke在領英上公布的消息，Gameloft英國辦事處出現了7次裁員，近日，他又發帖稱Gameloft英國辦事處已面臨關閉。而據外媒澳大利亞Kotaku最近發布的一份報告中指出，Gameloft澳洲布里斯班工作室出現了15次裁員。

而據Gameloft向外媒Gamesindustry確認，Gameloft馬尼拉分公司的倒閉對7名員工產生了影響，還發表了一些聲明稱：

「我們將繼續為其他歐洲辦事處在英國市場提供廣告和發行業務，不幸的是，在過去2-3年中，這兩項活動所產生的業務水平不再證明在該市場保持本地存在是合理的。我們正在建立業務中心，並依靠最有活力的地區來支持我們的合作夥伴，從而確保我們的商業網路（最大的移動遊戲發行商）不斷為遊戲開發商、移動運營商和品牌帶來獨特的支持。」
