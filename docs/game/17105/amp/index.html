遊戲直播和遊戲之間究竟誰在食物鏈頂端？

天天要聞
2019-12-13

現在遊戲直播越來越火，也有越來越多的人選擇成為一名遊戲主播，遊戲直播這個行業在這個時代以一種蓬勃的方式發展，很多數據都表明目前遊戲直播的確是現在電競行業里的一個新興產業，就現在觀看直播的人數來說，遊戲直播似乎已經佔據了整個直播行業的大半江山，而在遊戲直播的大半個江山裡，電競又是整個行業的領頭人。

2019年英雄聯盟全球總決賽在FPX戰勝G2之後也隨之拉上了帷幕，而整場比賽中國以3：0的優勢贏下來的，這個消息點燃了許多遊戲玩家的激情，中國戰隊奪冠的消息在微博以及朋友圈等地方隨處可見。這在以前幾乎是見不到的局面。

現在遊戲與遊戲直播似乎是同時出現在大家眼裡的，現在對於很多不玩遊戲以及不了解遊戲的人來說，現在的遊戲基本就是和直播掛鉤的。遊戲直播行業雖然不是近幾年才有，但是近幾年的確是遊戲直播發展的大好環境。

1、 遊戲直播驚人的發展速度

如果說電競點亮了很多人的夢想，為很多願意玩遊戲的人提供了實現夢想的空間，那麼遊戲直播則是為很多有天賦但是夠不到職業的人提供了機會，他們可能因為各種各樣的原因不能成為一個職業選手，但是在遊戲直播開始出現之後大家也很願意把自己變成一個遊戲主播。

直播的出現似乎是一個巧合，但也就是這個巧合，在今天改變了人們對事務的多元化認知方式，直播現在似乎是一個互聯網最熱話題，不論是國外的twitch還是國內致命的直播平台鬥魚以及虎牙等等，他們的發展似乎都只用了短短几年的時間就佔據了直播市場很大的份額。

時間如此短的情況下，他們將整個市場佔據了大半，這其中一定少不了電競產業的帶動，許多現有的數據都能像我們展示現在中國已經是全球最大的直播市場，而這個最大的直播市場目前大頭多數都是在幾個比較知名的平台。

遊戲直播的起步階段應該是在首屆英雄聯盟全球總決賽上。2011年首屆全球總決賽開始時，全球有超過20萬人同時在線觀看。這一龐大的數據引起了大家的關注，這對電競產業來說既是巨變也是催生，這一數據不僅撼動了電競賽事產業，也加劇了整個直播平台的出現。

從2014年開始，鬥魚以及虎牙等平台就逐漸活躍在人們的視野中，這是一個新時代–直播是時代來臨的預告。由此中國的直播平台開始了大發展，而遊戲和遊戲直播開始了自己的相伴之路。他們之間摩擦出來許多新的火花。

2、 遊戲直播與遊戲之間的相生相伴，共同發展

當遊戲遇上了直播平台之後，他們之間最直觀的事情就是會捧紅很多遊戲主播，例如鬥魚很火的張大仙、阿冷、大司馬以及周淑怡等主播，他們分別代表了每一個領域在直播平台的大熱直播。直播平台不僅僅是捧紅了這些主播還幫助許多遊戲留下了熱度。

例如絕地求生以及DOTA2等遊戲憑藉直播一直保持著自身的熱度，這些主播和遊戲的熱度都說明了在現在的這個階段遊戲和平台之間是一個共同發展的關係，兩者互相為對放的發展提供了路徑，實現了共贏。

大家似乎都看到了直播帶來的效應，其傳播能力非常廣泛，現在許多遊戲為了迎合直播需要的趣味度，在策劃時就優先考慮其在直播中會帶來的效果，因為現在每一款熱門遊戲只要是開始發行我們就能在網上找到相關的直播，並且還能附贈通關秘籍等等。

除了能夠由直播帶來遊戲的熱度之外，目前各大平台對電競賽事的轉播也非常看重，對於賽事主辦方來說，直播平台能夠很可觀地帶來觀看人數，而對平台來說，這些賽事也會輕微平台帶來非常高的熱度。

現在許多平台都在爭電競賽事的轉播權，B站在今年花重金買下了S賽三年的直播權，而鬥魚TV則是目前擁有電競賽事轉播權中最多的平台了，不僅如此，現在許多平台還會對職業戰隊進行贊助，將遊戲與平台的關係越連越緊。

3、 電子競技的成長

電子競技從一個沒有專業的直播到首播創下記錄，從開始發展到現在的千萬人同時觀看，這其中遊戲平台似乎提供了一個專業的媒介，也可以說如果直播平台沒有發展得這麼成熟，那現在的電子競技可能不會有這麼大的場面。

大家都知道第一屆英雄聯盟全球賽有20萬人同時在線觀看，如果後來沒有直播平台，那麼今天的英雄連盟可能不會有這樣的觀看人數。對於平台來說，他們面對電競產業這一塊，僅僅只是轉播一下遊戲他們不會有這麼多的點擊量。

因此他們通過與各大戰隊簽約，將戰隊選手吸納到自己旗下，把自己的平台變成與遊戲息息相關的主體，遊戲產商很樂意看到這樣的局面，雙方都願意利用自己的優勢相互合作達到自己的目的。利用這樣的方式的確是更容易使電競賽事成為爆款，也更利於平台打開自己的知名度。

4、 玩家變成觀眾

遊戲很多的熱度來源於直播平台的加持，在直播澎湃的今天，大家很輕易就能發現，很多人不玩遊戲並不會影響他們對遊戲的理解，這是來源於直播帶來的熱度以及效應，我們很輕易能發現許多觀看體育賽事的人本身是並不喜歡運動的，同樣的道理在電競中更適用。

許多年輕人源於自己本身沒有時間熟悉遊戲或者自己遊戲城 玩得不好，因此觀看直播就成了一種非常好的方式，這會讓很大一部分人有玩家慢慢轉變成了觀眾，並且忠誠度非常高。

遊戲在其經歷了最初的發展之後，慢慢歸於平靜，而直播平台這個時候就開始興起了，不知不覺遊戲直播就變成了生活里必備的東西，而遊戲直播也在不斷地影響著整個遊戲行業的改變，他們現在一定是一個相生相伴、共同發展的共同體，他們之後會不會發生改變就會變得很有思考性了。
