魔獸世界懷舊服的各種裝備g團價格，參考一下，別當「土財主」！

審判伺服器聯盟，我目前t1頭450，碧空2800(到目前為止，自己碰到的，朋友的，公會的團，最低)，t1手套600，燃魔1500。裝備價格變化很大，就拿統御來說，前面幾個cd出過兩次，一次1000.一次1200。這周的統御居然要2600。波動，前面幾個cd，700左右能拿下，這個cd1400。

火圈，碧空，短暫都屬於極品。多少錢看老闆心情。我們dkp團，我第二周老十滿分拿了個火圈，當時就後悔了，過後才知道火圈多牛。第四周又出了一個，我們團有個法師也是全分拿的。我省了一半。

我這周碧空3600以為穩了，結果來個4500，腿1500拿的，大概的價格就是靈風頭3600，碧空4000，短暫500，凌風褲子2000，法術能量1500，t1不記得了。

還有人說自己是dkp團，團里這周聖騎可以拿碧空了，100分收穫，我們團5把碧空3個統御，團長安排聖騎也可以碧空競爭了。

還有玩家說自己目前買了三件1100薩費隆斗篷 1200短暫 1300法術指環 。火圈t2褲未見過 t2頭2k以上 t1套500以上 統御2000以上 碧空見過一次拍了1萬1 。燃魔自己當時不懂貨不說了。其實我覺得靈風頭超過600一般不要，因為黑龍太簡單，15個人都能過。我們工會內銷，火圈1000，碧空1000，護腕400！就等出t2頭腿了！
