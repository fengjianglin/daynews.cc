LOL：EDG德瑪西亞杯陣容公布，主教練廠長領銜全新陣容亮相德杯

2019-12-17 162條評論 1,092次閱讀 94人點贊

北京時間12月17日中午，EDG官宣了即將到來的德瑪西亞杯出征陣容及轉會公告，原VG戰隊上單Aodi回歸EDG，同時原FPX打野Xinyi加入，此外本次德杯主力ADC選手為Hope，中輔Scout和Meiko不變，轉型為主教練的Clearlove將帶領全新的EDG陣容在德杯進行首秀。

北京時間12月17日中午，VG戰隊官宣上單Aodi和打野Youdang離隊：

【VG英雄聯盟人員公告】

經過VG俱樂部與選手友好溝通，VG英雄聯盟分部上單選手趙奧迪（ID：Aodi）、打野選手曾憲鑫@VG丶遊盪 （ID：Youdang）正式與VG解除合同，現已離隊。

奧迪和遊盪都給大家帶來過精彩的比賽和深刻的印象，VG衷心感謝他們的努力和付出。

未來，他們將在新的俱樂部繼續自己的職業生涯，讓我們一起期待奧迪和遊盪在比賽中的身影，打出風采！

加油！祝好！

隨後EDG官宣了德瑪西亞杯出征陣容和轉會公告：

教練組：

明凱（Clearlove）正式轉型擔任主教練一職，相信通過他豐富的賽場經驗和嚴謹的執教態度，能繼續帶領EDG踏上為最高榮譽而戰的道路！

程品倫（Corgi）正式加盟EDG電子競技俱樂部英雄聯盟分部教務組。Corgi曾執教AHQ Fighter等多支職業戰隊，擁有豐富的執教經驗。相信他的加盟將會給EDG帶來更多的戰術儲備和優秀經驗。

選手陣容：

經過三方多次溝通，原VG英雄聯盟分部上單選手趙奧迪（Aodi）以轉會形式回歸EDG電子競技俱樂部英雄聯盟分部。

作為從EDG培養出的職業選手，00後的Aodi通過自己勤奮的訓練態度和過人的遊戲天賦，得到了大家的認可。希望重新回歸EDG的Aodi能儘快融入團隊，為EDG拿下更多的榮譽！

同時，從2019英雄聯盟職業青訓營中脫穎而出的黃祥（xiaoxiang），將以上單選手的身份加入德瑪西亞杯出征陣容之中，希望xiaoxiang能快速成長，為自己今後的職業道路打下堅實的基礎！

在經過友好協商，常平（Xinyi）以自由人身份正式加盟EDG電子競技俱樂部英雄聯盟分部，與Jiejie、JunJia一起擔任打野位置，共同為EDG的榮譽而奮戰！

Xinyi曾效力於JDG、FPX等俱樂部，比賽風格穩健，擁有豐富的職業經驗，希望Xinyi能迅速融入團隊，在今後的征戰中打出滿意的成績，為EDG拿下更多的榮譽！

在冬季轉會窗口期間，EDG針對選手引援投注了大量精力，但結果不盡如人意。在這樣的前提下，我們決定在完成現有陣容調整的基礎上，著重在教務體系以及青訓體系上進一步加大投入，今後將會推出更多的舉措以保證隊伍成績。同時，也將在2020賽季的賽訓成績上加大獎勵力度，希望藉此激勵年輕選手的快速成長，也激勵老隊員幫助隊伍成績提升。

夢依舊，志前行。EDG已在盡全力備戰即將到來的2020賽季，相信在主教練Clearlove的帶領下，這支新生的EDG將堅定地走在充滿挑戰的登頂之路上！EDG矢志不渝！加油！

本次德瑪西亞杯EDG的出征陣容並沒有顯著的補強，在引入Aodi和Xinyi的同時，把多位年輕小將升至LPL一隊進行鍛煉和成長，形成了上中輔「3老帶2新」的全新體系，期待小將們和教練Clearlove的德杯首秀。小夥伴們怎麼評價現在EDG的陣容呢？
