《地平線·零之曙光》：末世下的希望！世界觀設定與劇情非常出色

2019-12-20 2,347條評論 4,427次閱讀 892人點贊

地平線·零之曙光

儘管遊騎兵已經算是索尼旗下比較成功的製作團隊，但在《地平線·零之曙光》之前，他們的幾個遊戲並不能算成功，包括《殺戮地帶》，儘管一直受到期待，但遺憾沒能成為爆款。不過隨著《地平線·零之曙光》超過千萬級別的銷量，卻徹底讓遊騎兵成為了熱門的代名詞。

一開始，筆者看到了《地平線·零之曙光》的介紹視頻，末世，沙盒，戰鬥系統，基本都是3A遊戲的標配，尤其是畫面，的確達到了美輪美奐的境界。抱著試試看的心態開始遊戲的筆者，發現比起這些標準的3A規則，更讓人意外也讓人驚喜的，則是本作的劇情，確實是非常出色，甚至可以用突出來形容。

說起來，相比起沙盒，筆者更喜歡比較有約束的線性遊戲，之前也因為類似的原因，中途放棄了幾款沙盒。所以儘管《地平線·零之曙光》的畫面如此美妙，但理性的分析一下，筆者覺得或許這個遊戲並不適合自己。當然，劇情的出色，讓筆者很容易就陷入其中，不得不說，沙盒的元素，難得的沒有影響筆者的激情，直到通關。

地平線·零之曙光

《地平線·零之曙光》是在2017年2月就發售的作品，也算時間較早了，筆者就順便講講劇情，如果想要親自體驗，可以跳過劇透的段落。說實話，即便是現在，玩這款作品，筆者認為，她的劇情設定仍然具有震撼力。

既然要講劇情，那就從法羅·泰德開始吧。

他所創立的法羅公司，所掌握的科技已經非常尖端，主要的領域則是生產機器人。

機器人，可以解決任何問題，獲得初步成功的法羅公司開始涉及環保等行業。在法羅公司，技術上展現出無與倫比天賦的則是伊麗莎白·索貝克博士，這也讓法羅·泰德的公司一直順風順水，據說在2049年，法羅公司在一次民意調查中，獲得了九成以上的支持率。

本來已經是富可敵國的法羅·泰德卻動起了歪腦筋，他決定將機器人推向戰爭，利潤將會更大。

伊麗莎白·索貝克博士洞悉了法羅·泰德的想法，毅然決然的選擇了辭職，但法羅公司的腳步卻並沒有因此停止。

地平線·零之曙光

法羅公司所生產出的戰爭機器擁有非常強的自我複製啊能力，還可以將任何物質都變為能源供自己使用。

挑起戰爭，然後出售戰爭機器從中獲利，成了法羅·泰德的法寶，但有一天，這些戰爭機器卻失控了，他們不停的複製自己，不停的吞噬著身邊的一切資源。這些資源，也包括生物，甚至人類本身，經過計算，16個月以後，地球的全部生物將完全被吞噬。

事態完全出乎了法羅·泰德的預料，無奈的他只好找到了伊麗莎白·索貝克博士，經過研究，想要破解這些機器，需要幾十年的時間，現在根本沒有任何辦法。

地平線·零之曙光

伊麗莎白·索貝克博士當機立斷，帶領著眾人制定了零之曙光計劃。至於這個計劃的最終目的，就是對地球的生態進行再造，只不過人類已經無法抵擋這些機器的摧殘，那一切的希望，也只能交給AI了。

人類最頂尖的人才創造了蓋婭，用來負責整個環境系統的運作。另外，蓋婭還會管理其他模塊，比如有的模塊負責破譯那些戰爭機器，有的模塊負責植被系統，有的模塊則擁有科技知識，不過這個模塊被法羅·泰德給刪除了，他甚至覺得，知識才是世界毀滅的原罪。

為了以防萬一，那些研究人員還做出了一個哈迪斯模塊，可以在蓋婭的改善失敗之後，重新複位。

問題是這個哈迪斯模塊竟然被入侵了，並且還妄圖奪取蓋婭的最高許可權，蓋婭選擇自爆，在犧牲自己之前，用伊麗莎白·索貝克博士的基因創造出了一名女嬰，就是亞蘿伊。

蓋婭只能寄希望於亞蘿伊可以修復自己，並且消滅哈迪斯。

當然，這個世界也已經不是原來的樣子，一個原因是蓋婭的受損，導致模塊之間的紊亂，結果世界上有很多機械生物。另一方面，則是因為法羅·泰德毀掉了曾經的科技與文明，讓這個世界只能以原始的形態慢慢發展。

至於我們的亞蘿伊，儘管有著伊麗莎白·索貝克的基因，但依然物是人非。

這條路，註定漫長且艱難，這也是我們玩家的旅途。

故事大概就是如此。

和故事比起來，其實《地平線·零之曙光》的武器系統，戰鬥系統，都屬於中規中矩，裝備製作方式，感覺也遠遠不夠豐富。

所以有的玩家，認為遊戲在戰鬥層面，相對弱一些，也是情有可原。

不過遊戲也在坐騎，挑戰都多方面做了拓展，至少在你的旅程中，不會太枯燥就是了。

地平線·零之曙光

如果你對劇情有所期待，那麼《地平線·零之曙光》的確不會讓你失望，另外末世題材與沙盒的組合，也堪稱絕配，你不但可以行走於世界，也可以對那些大型機械獸下手，更可以欣賞絕美的風景。

雖然是個有點老的遊戲，如果你還沒玩過，筆者認為還是值得推薦的。
