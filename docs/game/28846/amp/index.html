一把史詩之路瞬間畢業，玩家卻波瀾不驚：沒以前的感覺了

天天要聞
2019-12-21

要說最近在dnf當中最讓大家開心的一個事情，那麼一定就是史詩之路的開放了，作為每一次版本末期都會放出的福利，史詩之路是非常受歡迎的。今年的史詩之路和以往的不一樣，是能夠產出當今最強裝備超界的，這也讓無數玩家心之所往。隨著DNF普雷版本全新的史詩之路活動開放，也是讓很多老玩家又重新體驗到了當年刷深淵的感覺，也讓很多玩家的小號飛速的畢業。比如下面這位玩家，一把星河裂縫直接給了自己兩套防具。

如果要是放到以前90版本，一把深淵能夠齊兩套防具，那恐怕會成為一個不小的新聞，被大家痛罵臉狗，畢竟在之前的版本裡面，史詩的獲取途徑非常的單一，只有深淵裡面才會出現，所以說能夠得到一套史詩需要的不僅僅是運氣了，而在現在的版本裡面，史詩獲取途徑非常的多，深淵只是其一，以至於現在史詩已經是滿大街了，現在時代真的已經變了，不光玩家自己內心毫無波瀾，其他的吃瓜玩家也並不覺得羨慕。

當然主要原因還是現在版本大家的裝備都越來越好了。基本上老玩家都已經早早防具畢業，朝著普雷畢業努力；甚至一些玩家早早的已經普雷畢業，翹首等待100級版本。再加上這次星河裂縫鬼畜般的爆率，就連大家的小號現在都也超界加身了。

所以現在版本即使是深淵畢業也完全沒有之前版本的快樂了，畢竟在現在這個版本只要肯刷圖，人人都能夠有畢業SS，現在玩家比拼的更多的是裝備的打造。也正是因為這樣，現在很多的老玩家都是選擇脫坑，很多人也是開始懷念起之前的dnf，但是畢竟遊戲是需要往前看的，版本更迭是遊戲必須的過程，就算是玩家再怎麼懷念也是沒有辦法的。

從這次的活動可以看出，策劃已經在籌備100級版本的事情了，所以開始加速普雷版本玩家們的畢業速度。所以對於一些佛系的玩家來說，完全可以繼續佛系下去，說不定再等一個月，普雷裝備都開始免費送啦！而對於100級，一定會給大家帶來不一樣的玩法感受，相信一定不會讓各位玩家失望的，所以大家就耐下心來，準備迎接全新的100級的到來吧，大家認為呢？
