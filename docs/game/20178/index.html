騰訊2019年度收官手游，《雪鷹領主》配得上原作小說和動漫么

2019-12-18 439條評論 2,861次閱讀 427人點贊

臨近2020年，騰訊也推出了2019年度收官大作《雪鷹領主》，愛看小說的朋友肯定不會陌生這四個字，由白金作家「我吃西紅柿」的同名小說《雪鷹領主》改編而來，不僅做成了手游，還推出了動漫、漫畫等衍生作品。

《雪鷹領主》小說、動漫和漫畫的口碑極高，那麼由騰訊開發的手游，會不會延續原作IP高品質口碑作？今天紅姐介紹遊戲三大特點！

新手剛進入遊戲，可供有5種職業選擇，分別是物理輸出神槍、輔助琴師、刺客影刀、法師、遠程御箭：

神槍能夠免控並給予自身生命值增幅BUFF

法師有護盾技能和高爆發傷害

琴師能給自身和隊友回復大量生命值

刺客和御箭都有高爆發技能傷害

玩家可以根據本身的玩游性格自由選擇，選擇好職業後，就能DIY捏臉啦！這裡的捏臉參數非常豐富，大到整個臉型、五官，小到眼妝、美瞳等等，如果不想浪費時間捏臉，還能用前攝掃描自己臉或者相冊里的照片，自動生成臉型參數。

遊戲特點二：玩家能夠自創技能。

在遊戲中玩家完成首日主線任務，即可獲取道具「絕學精要」，總共有12個絕學技能，每個技能創建時都會消耗一個絕學精要。前期玩家做主線任務送的「絕學精要」夠用。

絕學有3個大分類，分別為炎龍、靈鯤和雷虎，每個分類下都有2大方向，每個方向又有兩個技能可供選擇。技能效果有群體-控制、單體-削弱、單體-控制等等，怎麼搭配就有怎麼樣的效果。

玩家點擊「技能裝配」，拖動絕學到裝配的位置，可以裝備兩個絕學，但是兩個絕學技能使用中間有一個10秒鐘的CD，CD期間不能釋放第二個絕學技能。

遊戲特點三：快速提升戰力，裝備、寵物、坐騎缺一不可。

遊戲里打造裝備需要裝備圖鑑、指定裝備的材料和綁銀，需要注意裝備圖鑑需要花費綁銀在【圖鑑坊】購買。玩家能夠穿戴頭盔、護肩、衣服、褲子，武器、手套、鞋子、披風。根據打造的裝備等級不同，消耗不同數量的材料。

除了裝備，培養一隻好寵物，也是提高戰力的重要方法，通過培養可以提升寵物的屬性加成，提升寵物的天賦技能，但是不同等級的寵物每次培養需要消耗的寵物升星石不同，升星所需的培養值也不同。

提升戰力所需的材料可以通過日常任務獲取，目前遊戲流通的貨幣共有三大類，分別是鑽石、銀幣、天心紗，非綁定的鑽石能夠1:1000兌換成非綁定銀幣，其他的綁定的只能換綁定的，更多內容在紅手指雲手機內下載《雪鷹領主》體驗。
