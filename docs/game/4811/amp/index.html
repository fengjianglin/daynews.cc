摘要

獵人在懷舊服中的地位是幾乎完美的存在，不論是升級階段，還是後面的刷G階段又或者是打團本都是不可缺少得存在！萬金油說的就是這個職業！

獵人在懷舊服中的地位是幾乎完美的存在，不論是升級階段，還是後面的刷G階段又或者是打團本都是不可缺少得存在！萬金油說的就是這個職業！不過不同的玩家玩遊戲的看法不同，所注重的遊戲內容也是不一樣，有的人喜歡打團PVE有的也是在PVP的道路上不可自拔，不同的玩法就要配上不同的天賦，接下來我就為大家帶來幾套獵人的天賦推薦。

首先就是PVE極限DPS天賦，這個天賦得最需要的因素就是一個寵物，來自詛咒之地的稀有寵物斷牙，他每秒1.0的攻速是你在MC中提升自己DPS的最大支撐，主天賦一定是獸王天賦，其次是射擊天賦，具體天賦點如下圖。這個天賦非常注重的一點就是，在打團的時候一定要記著讓團隊中的奶媽好好照顧你的寶寶，該給的BUFF一個都不能少並且記得奶好。如果隊伍種有出血盜賊的話你的傷害也會有進一步的提升。

寵物的技能學習滿級爪擊、耐力、火抗，其次點出疾跑，暗抗奧抗。寶寶控制的手法上，只要來移動爪擊就行了，其他都不要開，可能一開始看上去輸出會比較低，但是隨著後面大家缺藍的缺藍缺能量的缺能量，你會發現你的寶寶依然堅挺。

其次就是寶寶的轉火控制，因為懷舊服中，寶寶打死一個怪是會自己跑回來的，一定要記得轉火才能保證輸出不斷鏈。對內肯定也有射擊獵的存在，讓他把射擊光環加好，保證你自身得輸出，大藍發紅根據你的輸出階段CD去嗑，各種藥劑和捲軸前期一定要配好。怎麼樣才能算輸出合格呢？就是在自己自身得輸出沒有任何得問題下，寶寶的傷害應該佔有你總輸出得百分之20以上，這樣寶寶的輸出才算合格，也就是打出了極限的DPS輸出。

只要你在轉火方面的細節做好，該有的BUFF都有，並且自身得輸出沒有任何問題，你的DPS很有可能排在團隊前五的位置。但是在這之前你一定要和團長說好，因為一般打團本是不讓帶寶寶的，因為寶寶的走動行徑有的時候會非常的莫名，這也會導致ADD的發生，如果說因為寶寶的走位問題導致了團滅，那麼下次你要帶寶寶進團那就有點困難了！
