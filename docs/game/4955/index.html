魔獸爭霸3重製版模型放送話不多說系列第1期：為了洛丹倫

2019-12-03 0條評論 17次閱讀 0人點贊

摘要

各位玩家晚上好，本肥喵今日繼續和大家一起分享遊戲熱潮！前幾期文章分享了人族新舊模型的對比，反響尚可，

各位玩家晚上好，本肥喵今日繼續和大家一起分享遊戲熱潮！

前幾期文章分享了人族新舊模型的對比，反響尚可，不過也有許多吐槽水印太多，模型不夠高清等問題，其實是因為之前都是在WE（魔獸爭霸地圖編輯器，下同）里截圖，沒有遊戲里的渲染，於是乎肥喵我昨天「痛定思痛」，進入遊戲近距離截圖，重新製作UI。不過只有重製版，原版玩過的相信都知道什麼樣，我也就不放上對比了。

今天起就給大家奉上新版模型，順序和種族下滑選項一樣，人族、獸族、不死族、暗夜精靈，再加上中立英雄、娜迦，由於四個可選種族多數是由幾部分組成，因此每期都將是一個獨立的種族，人數太多的將分多期。除此之外的還有想知道的或者肥喵我遺漏的四族單位都可以私信我幫你放上文章。

首先是人族，人族（對戰+戰役）由人類、矮人、精靈等組成，人類也由洛丹倫（對戰、戰役）、庫爾提拉斯（戰役）、達拉然（戰役）等組成，本期文章從洛丹倫起。

洛丹倫旗幟

一、阿爾薩斯-米奈希爾

洛丹倫大孝子，作為War3的主角，是擁有全WE模型最多的兩個其中之一（另一個你們猜猜是哪一個，可以評論告訴我），一個是集萬千寵愛於一身的洛丹倫王子，一個是手持霜之哀傷尚未失去全部意識並與巫妖王意志搏鬥的天災軍團總司令（由於其中一個是死亡騎士因此將放在不死族的文章中）。

洛丹倫王子阿爾薩斯新版模型

霜之哀傷阿爾薩斯新版模型

二、烏瑟爾-光明使者

初代聖騎士之一，樓上王子的導師，白銀之手騎士團領袖，最偉大的人類聖騎士。

烏瑟爾新版模型

三、奧斯瑪爾-加里瑟斯

洛丹倫覆滅後殘餘部隊的大元帥，當上大元帥以他的能力想必不用懷疑，不過自視甚高。

加里瑟斯新版模型

四、尼古拉斯布贊勛爵

曾經的翻譯是「尼古拉斯大人」，戰役英雄。

尼古拉斯新版模型

格雷戈里新版模型

六、生命使者哈拉克

曾經的翻譯是「哈拉生命使者」，戰役英雄。

生命使者新版模型

七、獸人剋星達格倫

曾經的翻譯是「獸人屠殺者達貢」，戰役英雄。友情提示，這絕對不是灰燼使者亞歷山德羅斯，模型上的武器圖案是拳頭，而灰燼使者是手掌。

達格倫新版模型

瑪古洛斯新版模型

最後的這兩個肥喵我有話說，之前我說這兩個是激流堡英雄的時候，有朋友在下面評論說那個拳頭就是洛丹倫的白銀之手騎士團，我這邊有懷舊服的白銀之手戰袍，拳頭標記明顯不一樣，附上激流堡早期的旗幟以及現在旗幟和戰袍。

白銀之手戰袍

激流堡旗幟（右下角為洛丹倫聯盟標誌）

激流堡標誌

但是，按戰役劇情來說應該就是洛丹倫的聖騎士，不知道外包的製作組是什麼想法，順帶還「盜」了灰燼使者的模型。

今天對洛丹倫單位的模型放送到這裡就結束了，感謝您的閱覽，明天本喵將繼續分享魔獸爭霸3重製版中的其他模型！
