自帶主角光環？S賽紀錄片成IG蘇小洛專訪，三C一言不發

2019-12-17 2,877條評論 4,485次閱讀 234人點贊

S9世界總決賽已經過去了一個多月，在這一個月里，各個賽區為了S10也都進行了翻天地覆式的轉會。關於S9的總決賽，其中的細節依然有不少玩家關心，往年俱樂部和官方都會早早地發布紀錄片，今年僅有FPX在這期間不斷更新。雖遲但到，今天官方終於發布了長達50分鐘的S9世界總決賽紀錄片，記錄了FPX、IG、RNG三支戰隊從小組賽到總決賽的征程，場下的決策和矛盾，一覽無遺。

因為止步16強，RNG在長達50分鐘的紀錄片中只佔了短短兩分鐘，乏善可陳，但小狗心碎的背影讓人不禁惋惜比賽的殘酷。紀錄片的大部分篇幅都在敘述FPX和IG的故事，FPX的故事在俱樂部的紀錄片里已經有大概描寫：貢子哥的緊張、Doinb的擔當與安慰、Tian的調整已經下路組合的穩定，在官方紀錄片里他們一如既往的符合大家的印象：熱血而又充滿緊張。

倒是IG部分的紀錄片給了大家當頭一棒，在FPX部分，所有隊員都擁有鏡頭和採訪。而在IG部分，幾乎成了戰隊經理蘇小洛的專場：長達十多分鐘的IG部分蘇小洛就是主角！

除此之外，Ning在紀錄片中甩鍋BP的「新聖經」也非常有意思，從紀錄片中可以感受到，FPX隊員在輸掉比賽後的處理方式都是將鍋攬在自己身上，承認自己的失誤。但到了IG，場下的場景便是三C沉默不語，Ning重拳出擊。

其實在去年IG官方的紀錄片里，蘇小洛就已經成為了主角，今年對待三C的態度更是讓粉絲氣不打一處來。

非常有意思的是，在紀錄片中JKL的身體問題一直存在，IG卻沒有醫療組為此做準備。在回國後的直播里，阿水也透露自己在世界賽期間因為飲食問題吃了很多泡麵，IG的管理層不愧是你！
