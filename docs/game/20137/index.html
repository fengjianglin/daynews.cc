酒館戰棋：既要吃雞也要刺激，瓦娘集火養大狂戰斧，一刀秒3怪

2019-12-18 2,200條評論 3,186次閱讀 1286人點贊

大家好！我是爐石閑說，為您分享爐石魔獸趣聞趣事！

在酒館戰棋模式中，經過1個多月的玩家們的實際探索，其實也形成了某些套路的應對方法。如最早的熊媽媽、男爵等被「獵媽人」針對，在機械流和魚人流中盛行的「聖盾」技能，被奈法利安的「奧爆術」針對。同時，如果沒選到奈法利安英雄，用大螺絲或者帕奇斯（暫時推出酒館）的英雄技能，也能掃射2個對方隨從，也是破盾的方法。然而，契合所有英雄使用的破盾方法，還是在4星拿到的多頭蛇或者6星拿到的死神4000型這兩個「狂戰斧」隨從，頂在最左邊的第一攻擊位，來破掉對手的聖盾。同時，因為多頭蛇的野獸種族，星級也比較低，一般比較多人選用多頭蛇來重點培養。可以說，多頭蛇或者死神4000型已經成為了大多數吃雞卡組的「標配」。

除了能夠破掉對手聖盾之外，多頭蛇如果能夠buff養到一個高攻血，同時避開劇毒隨從的話，那麼它將成為最鋒利的「殺人吃雞」武器。在一定程度上，如果成型的多頭蛇的狂戰斧攻擊時，也會「濺射」傷害到一些無聖盾的劇毒隨從，等於也是「無傷」清毒的一個非常使用的手段。今天就給大家分享下瓦娘常規的「養狂戰斧」方法，一刀3怪，爽感吃雞！

瓦娘在開局的時候，選到的是導演這個比較強勢和好用的英雄，在前期的場面控制得不錯，血量也只被入了4點。在升級4星酒館的時候，瓦娘看到了在角落中的多頭蛇。因為場面的棋子還沒確定成型，所以瓦娘打算養多頭蛇狂戰斧來形成一個強勢點，以期吃雞。

那麼如何養大野獸種族的多頭蛇呢？機械管理員、展覽館法師和兔妖教頭都是不錯的手段。在狂戰斧初步成型之後，就初顯崢嶸，秒了不少對方血量低的小怪。

在5費回合，瓦娘拿到了銅須，加倍buff效果直接讓培養狂戰斧計划進一步提速，最終，50/50多的多頭蛇，基本上能夠打出至少2輪的輸出，殺死了對手不少的棋子。

在最終的決賽階段，瓦娘在僅僅剩下3血的情況下，憑藉著狂戰斧的出色發揮，斬落了對手多個棋子，最終成為了最後一個吃雞的選手。可以說，在套路無法成型的情況下，專註一個強勢棋子來培養，也是一個非常不錯的選擇。但前提是你要對對手要有足夠的了解以及正確擺放棋子位置，防止狂戰斧「出師未捷身先死」。

以上就是對瓦娘的「狂戰斧」套路的分享，有其他觀點的，歡迎在下方評論點贊！
