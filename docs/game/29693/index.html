魔獸世界：冬幕節限定雪球真好用！堪稱打斷讀條、地形殺神器

2019-12-24 731條評論 4,666次閱讀 1636人點贊

隨著《魔獸世界》冬幕節的到來，玩家在遊戲中又多了許多樂趣。玩家不僅可以在遊戲中做任務獲得限定寵物、玩具和武器幻象，還能用雪球打雪仗。

這個雪仗就和現實中的雪仗一樣，玩家可以對他人使用雪球，雪球擊中的單位受到擊飛效果。於是在正式服中有玩家研究出了雪球在大秘境中的各種妙用途徑。首先雪球可以遠程打斷小怪讀條，其次還能擊退反傷使者。

玩家使用雪球不會佔用GCD，技能輸出循環不會受到干擾。甚至有玩家把扔雪球綁定在了打斷技能上，使自己變相多了一個打斷技能，增加大秘境容錯率。

最重要的是雪球的獲取難度非常低，玩家只需到主城的冬幕節NPC那裡購買就行，2銀一組。就算一口氣買上十組也才花20銀，簡直和白送一樣。

值得一提的是，玩家購買的雪球不會隨著冬幕節的謝幕消失，玩家現在可以去NPC那多買幾組雪球以備不時之需。

懷舊服雖然沒有那麼多的冬幕節活動，但玩家也能用雪球打雪仗。不過懷舊服中的雪球只能在奧特蘭克山谷戰場中獲得和使用。玩家可以在奧山看到地上隨機刷新的雪堆，玩家點擊雪堆就可獲得雪球一組。

個別玩家利用雪球的擊退效果開發出了奧山地形殺玩法。這個地形殺玩法在現在流行的「大橋守衛戰」中尤為好用，只要與敵方玩家保持45度左右使用雪球，大概率會將其擊飛到橋下。不是滿血的玩家幾乎都會因為摔下大橋陣亡，少數倖存者也會被下方的盜賊偷襲。

不過也有玩家抱怨有個別玩家對著隊友使用雪球，故意打斷自己隊技能讀條，甚至把自己擊飛摔下大橋。這就是傳說中的保護我的敵方，痛擊我的方隊友吧。

悄悄告訴各位讀者，如果在奧山遇到在家裡掛機的隊友，也可以用雪球把他們慢慢擊飛出戰場的傳送門喲。

目前距離冬幕節結束還有一周時間，正式服的玩家不要忘了多買幾組雪球保存，懷舊服的玩家也要抓緊時間享受把敵人地形殺的快感。
