“吃雞”新模式工程兵測評，會修車補胎，架設掩體可擋50發子彈

2019-12-07 0條評論 17次閱讀 0人點贊

大家好，我是小魚乾，說實話小魚乾現在是有些頭暈眼花了，從今天和平精英體驗服更新完之後，小魚乾就一直在遊戲中測試，為大家帶來最新「特種對決」模式的測評。前兩篇文章，小魚乾已經測評過5大兵種中的「後勤兵」和「機動兵」，這篇文章小魚乾就給大家帶來「工程兵」的測評吧！

首先再簡單介紹一下「特種對決」模式的規則，簡單來說就是有5大兵種，在遊戲內可以通過拾取對應的兵種箱子，變身為對應的職業。每個兵種有4個等級，拾取到箱子就會擁有基礎技能，每拾取到一種晶元，就會提升一級，並獲得一個新的技能，最多升到3級。

工程兵的基礎技能就是架設掩體，這個基礎掩體就像一個「凹」字型的盾牌，可以像帳篷一樣選擇安放的位置，但一定要選擇一個平坦空曠的地方，才能放下這個盾牌，所以也有一定局限性，不是每個地方都適用的。

這個基礎盾牌的血量不厚，大約二十發M416即可打破，小魚乾感覺用處不是很大，因為假設的時候還要選擇合適的位置，也需要暴露在敵人視線中，危險性實在太大，而且就像帳篷一樣脆，實在是有些雞肋。

拾取到一個晶元之後，工程兵即可升到1級，同時獲得一個新技能：維修載具。

擁有這個技能後，只需要乘坐載具，就會發現右下角多了一個修復的圖標，點擊即可修復受損的載具。但這個1級的修復技能，只能回復部分血量，如果載具血量太殘，也是不能回滿的，並且已經爆掉的載具無法維修，因為根本無法乘坐。

同時還能修復破損的輪胎，即便4個輪胎都爆掉，也能一次性全部修復好！這個技能還是有一定用處的，因為老是有些玩家無聊，就愛打輪胎，搞得一條路的輪胎全爆了，利用這個技能就能修復輪胎了，但缺點就是冷卻時間有點久，長達180秒，也就是3分鐘。

並且更換兵種之後，冷卻時間就會從新開始計算，所以還是修完載具再換兵種吧。

當拾取到第二種晶元後，工程兵就會升到2級，同時獲得「強化維修載具」技能。

這個技能就是強化了一下維修載具技能，修復一次可以立即使載具恢復到滿血、滿輪胎，並且還能為載具加一次油！這簡直就是全能兵種了，不僅能修車，補胎，還能加油，隊伍中有這個兵種，只需撿到一輛車，基本就不用擔心載具的問題了，只要不被打爆，可以從開局用到結束。

當集齊3種晶元後，工程兵就能升到3級，同時獲得「強化架設掩體」技能。

這個技能就是把基礎技能的掩體強化了一下，掩體的面積更大了，同時也更耐打，小魚乾親自測試了一下，這個大掩體需要整整50發M416才能打爆，可以說是非常耐打了。並且這個強化掩體還能充能儲存，每60秒儲存一個，最多儲存3個，這樣就可以短時間內同時放置3個掩體，在沒有掩體的決賽圈，簡直就是神器了。

並且這個掩體的設計，是一個「凹」字形的結構，也就是說可以只露一個頭來架槍攻擊，而不僅僅是一個掩體，還是非常好用的，血量也足夠厚。

這個工程兵給小魚乾的感覺，就是兩個字：實用。相比起比較花哨、炫酷的機動兵，這個兵種就是老老實實輔助隊友了，只能說是有自己的能力吧，用於協助團隊還是不錯的。

後面小魚乾也會給大家帶來剩下兵種的詳細測試，歡迎關注喲！
