DNF：劍魂加強原來是一場夢，100級來了，夢醒了

天天要聞
2019-12-21

地下城最悲催的職業是哪個？我想很多人都會想到劍魂，這一個躺在地下城十一年的職業，因為技能帥氣被大眾喜愛，只要是玩家，8成的人都有劍魂這個職業，不過很多已經當做小號玩了，能夠堅持當做大號的不多，原因很簡單，因為版本不斷的改進，劍魂職業已經沒有了特色，武器大師僅僅是稱號罷了。

壓抑時間長了就會爆發，普雷版本的不給力讓玩家徹底失去了希望，大眾紛紛脫坑，玩家的流失引起了官方的重視，不得已情況下對劍魂等一些帥氣的職業進行了特色加強，在85版本，劍魂一躍成為了王者，強化效果非常好，成功留住了大部分玩家，一時間人們紛紛打磨劍魂裝備，就是為了去團本中表現一次，體會下揚眉吐氣的感覺，所以我能理解大部分玩家團本中喜歡開斬鋼。

前不久，韓服見面會曝光了100級所有的內容，地下城也迎來了巨大改變，從下副本的模式到武器裝備的改革做的都算不錯，但是看到三覺的技能之後劍魂玩家陷入了沉思，原來所謂的加強就是提前把三覺被動上線而已，新版本之後劍魂以及原地踏步，不是說劍魂不夠強大，而是其他職業都變強了，可憐的劍魂玩家剛享受了兩個多月的幻神體驗，新版本一來夢就碎了，劍魂依舊成為了不起眼的二線職業。

最新的報道已經確定了劍魂的地位，預計四鬼劍還是倒數第一，因為阿修羅加強幅度非常大，已經成為了打樁達人，而紅眼在新的等級中整體實力也會提升22%到26%之間，預示著同級的裝備下，打樁傷害會超越劍魂，鬼泣就不用多說了，沒加強的情況下就能挑戰劍魂，三覺出來後定能遠超劍魂。

縱觀地下城的發展史，鬼劍職業是玩家人數最多的職業，所以人們稱遊戲名字為「鬼劍士和地下城」，但是官方對於每個職業的態度並不一樣，紅眼永遠是親兒子，因為玩家多，這個職業的收入要比其他職業強很多，官方不敢怠慢，每次都是提前的強化，而劍魂成為了官方的棄兒，名義上是為了分散玩家，畢竟大家都玩劍魂，其他職業玩家就少了，十一年期間基本上沒有對劍魂加強過，在脫坑潮影響下也不過是提前更新了三覺被動罷了，現在全職業都有了「無我技能」，劍魂還剩下什麼？
