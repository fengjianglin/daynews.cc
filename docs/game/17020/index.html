能在國際象棋打敗人類的AI卻沒能在《我的世界》挖到鑽石

2019-12-14 267條評論 2,975次閱讀 1166人點贊

據外媒報道，看起來教授人工智慧(AI)玩《我的世界》是一項不小的挑戰。於今年早些時候宣布的MineRL競賽要求研究團隊開發出能在《我的世界》中成功開採出鑽石的AI機器人。這並不是一個不可能完成的任務，但它確實需要掌握遊戲的基礎知識。

AI玩家需要知道如何砍樹、合成十字鎬、探索地下洞穴同時還要躲避怪物和熔岩。這些技能大多數成年人都能在進入遊戲幾個小時後學會，或通過觀看YouTube上教程則能更快。

但根據將在AI大會NeurIPS上公布的結果顯示，在參加MineRL競賽的660名參賽選手中，沒有人能夠完成挑戰。儘管機器人能夠學習中間的步驟，比如建造一個熔爐來合成十字鎬，但沒有一個AI機器人能成功找到鑽石。

「我們提出的任務非常艱巨，」參與組織這次挑戰的微軟研究院首席研究員Katja Hofmann告訴媒體，「雖然沒有一個提交的方案完全解決了這個任務，但他們已經取得了很大進展並學會了在這個過程中製作所需的許多工具。」

這可能會讓人感到吃驚特別是當大家認為AI已經在國際象棋、圍棋和《Dota 2》等遊戲中超越人類。但它反映了這項技術的重要局限性以及MineRL裁判為真正挑戰參賽隊伍設置的限制。

按照規定，MineRL的機器人必須使用模仿學習和強化學習相結合的方法來進行學習。雖然通常情況下AI通過結合這兩種方法來接受大型挑戰，像AlphaGo系統最初就是通過輸入舊比賽的數據來學會下圍棋，然後，它通過一遍又一遍地玩來磨練自己的技能並最終超越人類，但採取了類似方法的MineRL機器人可用的資源相對有限。

這就好比聯盟大球隊跟小球隊之間的差別，前者能夠得到各種各樣的豐富資源–教練、營養學家、金錢能買到的最好的裝備，而後者卻只能擁有極為有限的資源。MineRL的這種方式看似不公平，但實際上反映了將AI融入現實世界的挑戰，因為很少有公司和研究實驗室能跟谷歌旗下的DeepMind抗衡。
