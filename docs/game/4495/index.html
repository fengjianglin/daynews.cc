Outlast是一款極為經典的恐怖遊戲，曾入選IGN公布的本世代最恐怖遊戲榜單，並佔據第二的位置，它在國內有一個很簡單的名字-逃生。

這個恐怖遊戲可以說是玩家和主播們的夢魘，不知道多少人被它嚇到自閉，最著名的莫過於CSGO主播茄子的那句「我柜子動了！」，他因過於恐懼而誤以為家中柜子動了，B站百萬UP純黑的逃生視頻也堪稱經典。

極為搞笑的一幕

日前遊戲開發商Red Barrels公布，他們正在打造一款名為The Outlast Trials的多人聯機恐怖遊戲，在這款遊戲發布時估計又會有一大批玩家和主播集體陷入自閉吧（滑稽）。

Red Barrels稱這款新作並不是Outlast正統續作，遊戲故事發生在冷戰時期且使用同一世界觀。

新作的封面圖

IGN將Outlast評為本世代第二恐怖的遊戲

Red Barrels還表示本作最多支持4人聯機，到時候也許會出現「還有多餘的柜子嗎」這樣奇妙的話語。

再加上攝像機電池是遊戲中極為關鍵的道具，要是新作中電池數量有限，那The Outlast Trials可能會成為新一代友盡遊戲（滑稽）。

不過The Outlast Trials被玩家們玩成搞笑遊戲的可能性更大，看看Steam上的另一款多人恐怖遊戲Dead by Daylight，玩家們硬是將其玩成了派對遊戲。

在與朋友聯機時，無論是與怪物周旋還是互相傷害，不管最後是否勝利，玩家們都能玩得很開心。

其實包括Dead by Daylight在內的多人聯機恐怖遊戲都有這種情況，玩家初入遊戲時因怪物和壓抑場景而恐懼。

可在熟悉怪物能力和環境之後，驚慌失措就會變成宛如在自家後花園閑逛般悠閑自得，Dead by Daylight中玩得好的人類玩家往往會主動去找怪物的麻煩。

不知道大家更想看到充滿歡聲笑語的Outlast，還是保有原汁原味恐怖氣氛的它？
