海外網友熱議全明星solo：UZI需要更好的隊友，他是最好的ADC

2019-12-08 0條評論 14次閱讀 0人點贊

摘要

12月8日下午一年一度的lol全明星賽事正式落下了帷幕，最後一場比賽是solo賽的決賽，在決賽中Bwipo成功以2-1的戰績擊敗了UZI，拿下了今年全明星solo賽的冠軍。

12月8日下午一年一度的lol全明星賽事正式落下了帷幕，最後一場比賽是solo賽的決賽，在決賽中Bwipo成功以2-1的戰績擊敗了UZI，拿下了今年全明星solo賽的冠軍。比賽結束之後，關注這次全明星solo賽的很多海外網友也紛紛發聲，這邊貝塔幫大家整理了一下，一起來看看他們說了什麼吧！

​-恭喜bwipo拿下了全明星solo賽的冠軍，不過為什麼每年UZI的1V1都能打得這麼好？太瘋狂了。

-UZI真的像個機器一樣，他的滑板鞋真的是個藝術品。

-請給UZI一個好一點的戰隊和更好的隊友吧，他真的是有史以來最好的ADC。

-實際上我仍然認為theshy才是solo賽的冠軍，因為他在最後一局中明明該贏下的，只不過他犯了S9世界賽上和FPX第四局一樣的失誤。

-我認為在這種1V1的模式下，有於雪球和火圈的存在，ADC的效果並不理想，無法放風箏。但是UZI已經打出了我們想要的效果了。

-看了決賽，更為Theshy可惜了。

​-如果bwipo第一把也和後兩局一樣留著R等到火圈縮小，那麼第一局也會是他的勝利。不過這些都沒關係了，他還是拿到了冠軍。

-ADC現在真的很爛。

-希望這次全明星後，大家可以注意到bwipo這位選手，儘管身材差人又不愛乾淨，但是他絕對是lol名列前茅的選手，他真的非常出色。

-bwipo全明星大滿貫了。

​-已經夠了，1v1比賽是LPL全明星最重視的，我們LEC第二次拿到了1V1冠軍，真的太好了。

-UZI又拿到了一個亞軍。

-bwipo除了正賽之外，是其他模式的神。

-全明星賽拿到了大滿貫，bwipo真的是明星中的明星。

-為什麼UZI一直要玩韋魯斯？

-剛剛Reddit上還有一篇UZI教你在現在版本下玩AD，在UZI被淘汰之後，這篇帖子也被刪除了。

​那麼，對於這些海外網友的評論，你們有什麼想說的嗎？
