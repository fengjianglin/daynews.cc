“一面之詞”，魔獸世界還有多少單陣營才了解的劇情？

天天要聞
2019-12-08

摘要

從魔獸世界開始到上個版本的軍團再臨，聯盟和部落之間一直都是小打小鬧，儘管相互之間摩擦不斷，但是主線都是大家聯合起來，一起打亡靈天災，一起打上古之神，一起打燃燒軍團，等等，就算有戰爭，也會在共同的敵人面前停火下來。

從魔獸世界開始到上個版本的軍團再臨，聯盟和部落之間一直都是小打小鬧，儘管相互之間摩擦不斷，但是主線都是大家聯合起來，一起打亡靈天災，一起打上古之神，一起打燃燒軍團，等等，就算有戰爭，也會在共同的敵人面前停火下來。

但是在這個版本的《決戰艾澤拉斯》，暴雪為了拱火聯盟與部落玩家的情緒，刻意製作了很多挑起矛盾的劇情。除了我們熟悉的希爾瓦娜斯燒樹、決戰洛丹倫、達薩羅之戰等等，還有好多劇情藏在陣營戰役裡邊，如果只玩一方的劇情，壓根沒法知道整個故事的全貌。

因此暴雪也特地推出了《一面之詞》這個成就，要你的賬號下面的聯盟和部落角色都完成所有的陣營戰役劇情才能解鎖。

如果你只玩聯盟，那麼當牛頭人酋長貝恩·血蹄出於良心，私底下阻止了希爾瓦娜斯的邪惡計劃，帶著吉安娜復生的哥哥，將其送還給吉安娜時，可能你還有點一臉懵逼，吉安娜的哥哥不是早在數十年前就死於舊部落控制的紅龍嗎？怎麼突然出現的？部落啥時候把他復活起來的？這些都只在部落的陣營戰役裡面才能看到。

同樣，部落玩家也很「委屈」，聯盟玩家指責部落和邪惡的薩萊茵（當年阿爾薩斯手下的吸血鬼）勾結在一起，可是在部落的劇情里完全沒有薩萊茵的痕迹。在聯盟的陣營戰役裡面，才能看到希爾瓦娜斯勾結薩萊茵來對付聯盟，但最終載著薩萊茵整艘船被聯盟炸掉，薩萊茵也葬身海底。

自軍團降臨的不同職業大廳有不同的獨享劇情以後，現在的不同陣營也有了不同的劇情。看來，作為劇情愛好者來說，光玩一個陣營的號，還真沒法徹底了解魔獸整個宏大的世界呢，小編不禁摸了摸自己的肝，疼啊！

還有哪些重大的遊戲劇情是單陣營才有的呢？歡迎大家在評論區補充~
