9.24B版本劍刺削，雷霆劫削，蛋卷削。網友：雲頂要變天了

2019-12-18 566條評論 2,551次閱讀 886人點贊

剛剛得知，雲頂之弈9.24B版本的更新內容已經出來了，說實話，改動量還是蠻大的。更新時間，美服定在本周四，國服應該會同步更新，也可能會遲個一兩天。

不一一打字了，一張圖片了解所有改動。

一些拙見

1、賭3星將成為過去式！如果低費賭3星到後期仍然強力，那隻會越來越多人放棄運營，所以，這次改動過後，3星卡的強度將大幅下降。

2、希維爾或許能擺脫「工具人」稱號。沉默刀和繳械刀的觸發機率降到為了25%，對劍刺來說，影響很大。

3、遊俠或許能有一席之地。遊俠里最貴的兩張卡，老鼠、艾希再次加強了，有可能會在9.24B中戰得地位。

4、版本三巨頭前途未卜。可以看出，雷霆劫、劍刺、賭掠食，都被大砍了一刀，在下個版本會有什麼起色，只能拭目以待了。

5、火男回調。之前削得太狠了，導致火男強度下降太多。所以將布蘭德的最大法力值改回80點，怎麼說也是張4費卡，強度不能太次。

最後，各位棋友覺得此次改動怎麼樣？哪個陣容將會在9.24B中崛起呢？
