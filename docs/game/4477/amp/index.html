摘要

有這樣一類遊戲，它或許不需要非常大的容量，但擁有著全語音和各種精緻的CG。

在不少人接觸遊戲之初就給玩家留下了深刻的印象。

它們或許沒有3A大作那樣可供玩家自由操控探索酷炫的世界。

但它以文字為載體在玩家的腦內構建出一個又一個神奇的故事。

文字冒險遊戲可涉獵的範圍相當廣泛，從正常的gal到恐怖獵奇的故事。

從讓人臉紅心跳的戀愛到充滿懸疑地尋找真兇。

死印

隨著時代的不斷進步，文字冒險遊戲也開始玩出了各種花樣。

在玩家童年中接觸到的這類遊戲大部分就已經支持全程語音功能。

知道該遊戲的都是猛士

玩家根據自己的喜好選擇選項，能夠觸發相對應的劇情或者是解鎖CG。

其中GAL和女性向戀愛遊戲有著相當高的人氣。

《Fate Stay Night》這款文字遊戲的分級你懂的。

除此之外女性向經典戀愛遊戲《薄櫻鬼》也是一個版本接著一個版本的出，活躍至今。

《薄櫻鬼 真改》系列不管是人物的動態動作，還是在戰鬥方面的表現都非常有感染力。

當然，除了戀愛向這個類別，不少人應該還非常喜歡懸疑推理類的文字冒險遊戲。

經典的《逆轉裁判》系列，通過畫面和圖片相結合，給玩家們呈現了非常多有趣的逆轉案件。

打越參與的《極限脫出》、《AI：夢境檔案》等作品用著3D的建模，也讓AVG遊戲不再局限於2D平面。

一些遊戲也開始啟用live 2D技術，讓人物的動作表情都變得相當豐富。

某知名FPS遊戲

但不管形式怎麼變化，AVG遊戲本質依然是信息量巨大的文字。

在懸疑類遊戲中，和真兇有關的線索可能就隱藏在日常篇的劇情里。

在戀愛遊戲中，角色的喜好故事也能通過文字信息傳達。

文字交代遊戲背景遊戲設定，方便玩家更快地入戲。

同樣，文字冒險遊戲的玩法也開始逐漸遍地開花。

非常常見的就是以選項決定遊戲劇情的發展。

國產遊戲《Will：美好世界》用排序的方式構建新劇情。

《命運石之門》將選項巧妙地隱藏在日常回復的簡訊中。

文字冒險遊戲涉獵的範圍其實比不少玩家想像中的更廣。

許多獵奇恐怖向的作品用文字為載體同樣不遜色。

如愛的戰士虛淵玄的恐怖純愛向作品《沙耶之歌》給過玩家成噸的暴擊傷害。

同樣隨著時代的發展，手機在一定程度上也讓文字冒險類遊戲有了更多的生存方式。

鍾愛冷飯的卡普空移植了《逆轉裁判》。

國產遊戲《流言偵探》雖然褒貶不一 也勇敢地邁出探索的第一步。

文字冒險遊戲的愛好者相信看到近年來這些遊戲的不斷進步，有著非常多的感慨。

也許沒有出眾的遊戲性，但它們擁有著用文字創造天馬行空世界的特殊能力。
