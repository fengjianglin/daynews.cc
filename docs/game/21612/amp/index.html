選手專職做教練之後，一般都會對首發選手非常嚴格

在EDG官宣廠長轉型成為EDG的主教練之後，網路「蹭熱度之王」Joker也第一時間發表了自己的看法，那就是「廠長當教練之後，EDG的哪些選手可要慘了」。坦率的說這一次囧王者認為，Joker說得對。因為只有選手才是最了解選手的，他們在做選手的時候，有哪些在選手間流行的，而教練組不知道的潛規則，暗號和一些見不得光的小動作。他們都門兒清，當他們上位成為主教練之後，這些小九九就真的瞞不住曾經的「隊友」了。在職場中，這種情況屢見不鮮，例如當你的同事被提升為主管之後，你們以前搞過的小動作，現在肯定不敢搞了。因為你的「新主管」比你更清楚其中的門道，還是老實點好。所以S10賽季，EDG的選手們真的有點慘了。
