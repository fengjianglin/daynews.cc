同名漫畫改亂斗遊戲——《大王不高興》

2019-12-24 179條評論 3,409次閱讀 2329人點贊

《大王不高興》作為一款漫畫IP改編的手游作品，遊戲對於IP的形象及故事情節還原上做得十分細緻到位。同時作為二次元回合卡牌玩法，遊戲同樣也出現了不少有趣的設計。不過遊戲依然受限於IP本身的體量，在人物、故事方面恐怕很難在短期內作出大量的擴展，不過從遊戲運營的角度來說，或許遊戲後期以聯動形式與同類漫畫進行互動或可以產生更好的效果。


出色的IP還原

手游《大王不高興》是由中國人氣漫畫家使徒子與靈游坊聯合打造的漫畫改編手游，遊戲採用策略回合制的玩法，以高端還原的漫畫人設情節展開，同時又兼具回合制玩法的戰鬥策略性，為漫畫粉絲帶來一場代入感十足的東西方神界大亂斗。

對於廣大的漫迷來說手游《大王不高興》第一點就是遊戲對於漫畫的還原度，遊戲通過動畫、漫畫分鏡、文字對話等多種方式將漫畫中的情節一一還原。

而遊戲在人設上採用L2D的動態呈現讓遊戲中每一個角色都栩栩如生，同時遊戲還為這些漫畫中的角色設計了大量的皮膚，以確保每個角色的獨特個性。不過遊戲作為一款IP產品，出場角色始終受限於原作IP，在後期運營上或許更多地會採用聯動形式來擴展遊戲的角色池。

略具深度的戰鬥策略

在《大王不高興》中採用了經典的二次元卡牌回合制玩法，不過遊戲在基礎的回合玩法上加入了一些全新的元素，從而大大地增強了遊戲在戰鬥中的策略性。遊戲在四名隊員加助戰的回合玩法基礎上還加入了主角及護法的設定，除了故事中主角的閻小羅之外，玩家還能夠選擇加百列、路西法、楊戩等角色作為主角出戰。

遊戲中的主角與隊員在攻擊時除了進行普通攻擊技能之外，還有需要耗費能量的特殊技能。不過每回合回復的能量由4點、6點、8點遞增，玩家在抉擇技能前還需要多考慮後續角色的技能是否能夠順利釋放。

此外隨著戰鬥的進行，護法的能量條會不斷累積，當達到100%時即可釋放強力的護法大招，足可以改變遊戲的戰鬥局面。不過與主角相同，遊戲中目前只有4名護法，在可選擇性上還是有所欠缺。

可肝可咸但十分豐富的內容玩法

《大王不高興》在內容框架上採用了最直觀的線性化劇本敘事，通過還原漫畫中的每個情節讓玩家由淺入深地了解遊戲的整個世界觀及玩法設定。不過這也決定了遊戲在前期有著漫長的推圖期，需要花費玩家不少的時間，好在遊戲加入了二倍速和自動戰鬥，只要在戰力足夠的情況下，完全還是能夠省去大部分的操作時間。

此外遊戲在內容玩法的設計上以每日輪轉的活動本來填充玩家的日常活動，同時也有固定的日常本供玩家刷經驗、金幣等。而多人副本的加入更是讓遊戲在高難副本的挑戰性上具備了不少的可玩性。但是總體來說，遊戲依然是以體力值來限制玩家的日常活動，一旦度過遊戲前期的開荒期後，日常的體力消耗可以有效地控制遊戲時長，玩家也能夠自由的選擇將體力分配至哪項玩法上。

卡牌式養成極具深度

說到遊戲的養成玩法，其實可以劃分為主角與英雄夥伴兩個部分。遊戲中的主角（就是閻小羅、加百列等）他們的技能體系不同於一般英雄，隨著劇情的推進解鎖的各種技能，能夠根據玩家的需求裝配至技能欄中。同時還有一套神器養成系統，通過淬練裝備提升神器等級來增強主角的各項能力值。

而在英雄養成方面，等級提升、升星、技能強化等一系列養成均以卡牌吞噬的方式展開，對於氪金玩家來說能夠快速地進行養成提升強度。而大部分玩家依然能夠通過日常本等一系列玩法對玩家角色進行養成。

此外遊戲中的裝備系統對裝備進行了細緻的分類，如猛攻、體魄、精準、神行等一系列帶有套裝效果的裝備，讓玩家能夠更加直觀地了解各個裝備的效果，並且在配裝時的選擇也相當便利。同時豐富的裝備體系決定了遊戲在後期圍繞著裝備配裝的選擇策略所展開的對戰玩法。

總結：

《大王不高興》作為一款漫畫IP改編的手游作品，遊戲對於IP的形象及故事情節還原上做得十分細緻到位。同時作為二次元回合卡牌玩法，遊戲同樣也出現了不少有趣的設計。不過遊戲依然受限於IP本身的體量，在人物、故事方面恐怕很難在短期內作出大量的擴展，不過從遊戲運營的角度來說，或許遊戲後期以聯動形式與同類漫畫進行互動或可以產生更好的效果。

若遊戲能夠將漫畫直接植入遊戲中，或能夠帶動更多的玩家深入了解遊戲的世界觀情節。對於喜歡《大王不高興》系列作品的粉絲來說，本作相當值得一試，哪怕你沒看過漫畫，也能夠在遊戲過程中喜歡上這部有趣的漫畫。

休息會，掃碼玩玩遊戲吧~
