你也搞全華班？轉會期他們送走最後一位外援

天天要聞
2019-12-18

★遊戲馬蹄鐵原創

蛇隊變全華班

蛇隊也就是Snake，在今年被李寧收購贊助後改名LNG。原本好不容易在今年盼來了兩位雖然有點神經刀，但的確上限很高的雙C之後，很多人以為蛇變麒麟要化龍了，結果打野Sofm這個賽季的表現並不好，加上原本就有發揮不穩定的神經刀C位，結果LNG夏季賽排名常規賽第7，季後賽第6。中游，一個普普通通平淡無奇的位置。

轉會期LNG也是沒有續約Sofm，而是從TOP戰隊簽下了Xx，另外又從SNG簽下了Maple。而且在今天又送走了最後一位韓援選手Plex，然後LNG就突然變成全華班了。

李寧全華班沒毛病

LNG背後的李寧這個中國人都懂，因此他們轉變成全華班也沒什麼太大問題。送走的兩位外援中，其實這個中單Plex有點可惜，因為他的確是很有潛力的選手，個人能力很強。結果Plex去了WE，而LNG的Maple和前往SNG的Sofm有點互換的意思。總之現在LNG的陣容就變成了：

上單：Flandre

打野：Xx

中單：Maple

下路：Asura，Light

輔助：Duan，lwandy

兩位新人Light和lwandy分別屬於LNG和EDG的二隊00後選手，LNG在補強了首發陣容的同時，也是儲備了一對更年輕的雙人組。雖然目前LNG在這次轉會期上沒有什麼驚天動地的超級明星選手交易，例如阿水什麼的，但他們組建的新陣容樸實無華，屬於經濟實用型。

從蛇隊到麒麟

Snake在今年的變化其實很大，從春季賽到轉會期，目前也就只有上單和ADC是沒變的。如果LNG不再進行相關轉會操作的話，那麼這套陣容的LNG在明年LPL就會是一支全華班。很預測這樣一支戰隊的實力強弱，畢竟選手之間的確也是存在微妙化學反應的。有沒有明星選手不重要，團隊氛圍和默契程度是決定一支戰隊上限的關鍵。

而對於不少LPL觀眾粉絲來說自然不會介意多幾支全華班，到底最終是否真的能夠如願看到蛇隊變成全華班呢？關注等待LNG的後續操作吧！
