雙十二「課程乾貨篇」電競專業該如何認知遊戲項目

2019-12-13 3,318條評論 4,396次閱讀 1267人點贊

在電競專業的教學中，理論與實踐結合是較為重要的一部分。雖說電競專業的教學重點不是電競項目，但是電競項目的了解是基礎的，在有了一定基礎的情況下，我們去聽課才能更加容易迅速地掌握更多深層次的知識。

鑰娘將從最近火熱的三大頭部賽事，英雄聯盟、爐石傳說、絕地求生三大項目進行簡單而清晰的介紹。巧合的是，這三大項目分別對應了不同的遊戲類型，包括MOBA類、卡牌類、FPS類。選取的講解內容同樣來自於鑰浪電競教育首期電競體驗培訓班，本篇推文也將作為課程乾貨系列的第一篇教學呈現給大家，有不足的地方或是想提的問題都可以留言給鑰娘，鑰娘第一時間會進行回復喲。

MOBA類遊戲—【英雄聯盟】

MOBA（Multiplayer Online Battle Arena）中文譯為多人在線戰術競技遊戲，一般指5V5，三條路對戰。

簡介：《英雄聯盟》(簡稱LOL)是由美國拳頭遊戲(Riot Games，已被騰訊收購)開發、中國大陸地區騰訊遊戲代理運營的英雄對戰MOBA競技網遊。

卡牌類遊戲：【爐石傳說】

卡牌遊戲，簡稱TCG。此類遊戲是以收集卡牌為基礎的，在遊戲時遵循自己的規則，依靠玩家的策略與智慧來贏得勝利。一般的說，這些卡牌都有一定的價值，玩家之間可以交易交換自己的卡牌。
