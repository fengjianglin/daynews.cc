Perkz直播給AD選手排名：Uzi再次名列榜首 Lwx僅第五

天天要聞
2019-12-15

S9總決賽已經過去了一個月，各大賽區都進入了休賽期和轉會期，職業選手們紛紛開啟了直播進行休閑。

G2戰隊的當家AD選手Perkz就在昨日開啟了直播，回答了觀眾們的一些問題。其中有一個問題是：誰是西方最好的AD選手？Perkz就這個問題進行了回答。

基於他自己的親身體驗，Perkz給出了以下的排名：

1.Rekkles

2.Doublelift

3.Kobbe

4.Hans Sama

5.Upset

6.Patrik

7.Sneaky

Rekkles在Perkz心中佔據了第一位，因為他能夠使用所有版本的英雄，比如霞、卡莎、希維爾，他都能很熟練地運用。而且，Perkz覺得Rekkles是所有西方的下路選手中霞玩的最好的，並且他還有一手獨一無二的蓋倫&悠米下路。

而Doublelift排在第二位，和Rekkles很接近。因為他有著無可挑剔的操作水平，而且在玩一些不是很主流的下路組合的時候有著非常高的強度——比如韋魯斯&塔姆，卡莎&加里奧。但是Perkz認為，大師兄在玩霞對抗Rekkles的時候還是有些缺陷。

Kobbe則是排在大師兄之下，因為他在夏季賽的時候表現了非常棒的進步。Perkz表揚了Kobbe在團戰和Carry隊伍的能力上的突出表現。

隨後，Perkz還排名了所有他交手過的AD選手，其中Uzi位列第一。

1.Uzi

2.Jackeylove

3.Rekkles

4.Teddy

有人質疑，為什麼沒有S9的冠軍AD選手Lwx呢？事實上他把Lwx排在了Teddy之後。

這就是Perkz心目中的AD排名了。
