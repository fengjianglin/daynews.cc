LOL：強攻還是征服者？新版本鱷魚玩法，讓你輕鬆馳騁召喚師峽谷

2019-12-05 0條評論 14次閱讀 0人點贊

摘要

藍天大地還有清風，正指引我前進。今日風塵對我私語，明日星辰為我閃耀。隨湯普森看英雄聯盟， 你會得到你最想要的東西。

作為目前最火爆的5V5對戰類型塔防遊戲，英雄聯盟目前取得的成就相信是「前無古人，後無來者」的，而隨後英雄聯盟又更新了雲頂之弈的模式，吸引了更多的玩家去玩這個遊戲，以目前的形勢來看，英雄聯盟這款遊戲，還要火爆很久的時間。

隨著遊戲的更新，全新的S10季前賽版本也已經與我們見面了有一段時間了，作為季前賽版本，今年的改動對於我們來說也是蠻大的。首先就是偷錢符文的取消，其次就是征服者的加強，還有就是一些大大小小的符文改動。之後就是召喚師峽谷的改動，小龍變得更為重要，峽谷地形也會隨著龍的改變而改變，因此新版本需要注意的東西變得更多了。那麼在季前賽，你是否找到了可以充分的英雄呢？如果還沒有，看看這篇關於鱷魚的攻略吧！

作為一個老牌上單英雄，鱷魚很早就進行了上線，在英雄聯盟早期的時候，這個英雄就備受玩家們的喜愛，超強的爆發能力，極強的續航能力，以及穩健的線上壓制力，都是這個英雄所具備的特色。鱷魚在剛剛過去的世界賽中，也是多次登場，可見在上個版本中他的強勢。那麼在全新的S10賽季來臨之時，在全民征服者的符文選擇中，鱷魚是否還應該繼續選擇強攻呢？

首先說一下鱷魚的玩法，鱷魚作為一個老牌上單英雄，目前也經常出現在中單的位置來壓制一下AD刺客，因為鱷魚在具有高靈活性的前提下有著不錯的爆發能力。被動是可以獲得怒氣，Q技能造成傷害，怒氣情況下傷害提升回血增多。W穩定控制技能，有怒氣可以造成1.5秒眩暈，沒有則是0.75秒。E技能有怒氣會穿甲，並且傷害到敵人可以再次釋放。大招就是增加生命值增加傷害，並積累怒氣。總而言之，玩好鱷魚。線上利用E起手或是W起手，然後利用Q來消耗並回血，再次釋放E來規避地方傷害，這是最簡單的連招，也很實用。

那麼鱷魚究竟應該選擇征服者還是強攻呢？上個版本中，強攻作為鱷魚的核心天賦，一直受到玩家的青睞，而如今征服者的強度大大提升，是否就意味著可以取代征服者了呢？實則不然，全新版本的征服者對於鱷魚來說，收益並沒有強攻來的實在，鱷魚帶強攻，線上是很有機會打出壓制效果的，並且配合技能的傷害，可以對地方造成高額的傷害，並且連招期間不會讓敵方打掉血，這也就意味著，只要有連招你就可以消耗對方，並且自己還不掉血。當然，這前提是你對鱷魚的熟練掌握。

那麼帶徵服者的前提是什麼呢？是在你覺得一套技能打完之後，對面並沒有和你對拼的資本，你還能繼續和他對打，這種情況下可以帶徵服者，多數情況下對面都以肉為主，如果是靈活的英雄，征服者真的沒有太大作用。總而言之，一個人有一個人的喜好，但是收益卻是在這，多數時間裡，強攻對於鱷魚的提升相比於徵服者要強得多得多。最後，希望這篇文章可以對您有所幫助，我是湯普森，記得關注我，每天都會為大家更新更優質的遊戲內容。
