新英雄Boss賽特

新英雄賽特，拳擊手，塊頭大，打擊感強，他的出現讓很多人以為自己穿越在玩格鬥遊戲。因為很久沒有出現這種近身肉搏的戰士英雄了，賽特身上到處都是「哲學」的味道。尤其是他的大招抱摔，雖然本質上就是男版蔚，是一個擊飛控制效果，但爆炸的AOE傷害也是讓他在未來有又肉又控的全能前排潛質。

全面的新英雄

賽特身上有很多優秀的特點，有護盾，有回血，有大小多種控制，還有極高的爆發傷害。技能大開大闊，雖然沒有劍魔那麼靈活，但是他的W技能範圍也非常大，而且是中心為真實傷害，其他範圍內為物理傷害。唯一的缺點可能就是沒有位移，雖然有加速技能，但是直接翻牆類的長距離位移是沒有的。

抱摔變格鬥遊戲

賽特的大招抱摔是無法控制其方向的，他命中目標只會朝著技能方向抱摔。因此如果想要利用這個技能抱回對手打控制，那麼需要從反向抱摔。

打擊感強烈的抱摔

這個技能也是讓很多人吐槽感覺自己不像是在玩LOL，而像是在玩DNF！因為在DNF中的確有非常多的職業有抱摔這個技能，例如男女柔道…

類似這種….

新英雄是個肉坦？

由於被動的關係，賽特非常適合走肉裝，血越多他的收益越高。而且賽特自身是個打肉很強的英雄，因為他的技能基礎傷害高，還附帶對手最大生命值百分比的傷害。賽特的出現，無論是走哪一條路，都將會成為未來遊戲中非常強大的前排控制英雄。而且以目前各項數值來看，這個英雄必削。能夠在年末看到不一樣的ADC，再看到這樣一位充滿「哲學氣息」的肉搏英雄，LOL還是好玩啊！也為設計師這突如其來的勤快更新點贊。
