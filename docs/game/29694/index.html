聖誕節有哪些好玩的遊戲，這2款硬核動作遊戲你會喜歡嗎？

2019-12-24 207條評論 2,755次閱讀 826人點贊

聖誕節到了，各大商場自然是非常熱鬧的，但是對於喜歡遊戲的小夥伴來說，選擇一款自己喜歡的遊戲，在虛擬世界中和網友度過也是不錯的選擇。還可以截圖「發圈」，該有的儀式感都有，甚至比真實世界更加有趣。那麼，下面就給遊戲愛好者推薦2款年度硬核動作遊戲，一款是獲得TGA2019年度遊戲獎的《只狼：影逝二度》，另一款是動作冒險遊戲《鬼泣5》，有沒有玩過的小夥伴，接下來一起看看吧。

只狼：影逝二度

2019年3月22日該遊戲在全球的PS4、Xbox One和PC平台上線，是一款動作類的遊戲，你將扮演獨臂忍者，守護古老血統的年輕御子，劇情中御子被抓走，你要挽回榮譽，踏上危機四伏的征程。另外，《只狼：影逝二度》還在2019年12月13日獲得TGA 2019年度遊戲獎。

《只狼》沒有詳細的角色培養系統，主要的兩個屬性就是血量和傷害，和其它同類型遊戲相比更偏向於動作反饋，同時保留敘事方式，正面講述遊戲劇情，讓玩家更加容易理解，真正融入到遊戲之中。通關後至少也能知道它大概講了個什麼故事，對這個世界觀也會有一個最基本的理解。

遊戲展現的激烈的戰鬥體驗，對操控技巧有較高的要求，可能有許多手殘玩家都無法通關，但對應硬核動作遊戲愛好者來說，這款遊戲能讓他感到挑戰，有種「明知山有虎，偏向虎山行」的感覺。還會被每一個動作、招式、風格、節奏給吸引，越有挑戰越喜歡玩，你覺得你會喜歡這樣的遊戲嗎？

鬼泣5

《鬼泣5》是由日本CAPCOM開發的一款動作冒險遊戲，也是2019年3月發售並登陸PC、PS4和XBOX One平台，但遊戲據悉設定卻不同，講述的是城市出現大批惡魔，玩家需要和惡魔進行對抗。2019年12月，獲得TGA 2019頒獎典禮最佳動作遊戲獎，所以可玩度還是很高的。

遊戲中有多個角色可以控制，不同角色的屬性和戰鬥方式有所不同，遊戲的操作方式在遊戲中有詳細的介紹，同時還會不斷解鎖新的招式，所以喜歡探索動作遊戲的玩家一定會喜歡的。

另外，遊戲不只是系列的集大成者，而且在戰鬥、敘事、關卡節奏、重複遊玩等方面均有長進，被評選為2019年度動作遊戲將是有原因的，在《獵天使魔女3》問世之前，很難想像會有第二款動作遊戲能想比較，但是《鬼泣5》可以。

希望以上推薦的兩款遊戲能為你的聖誕節添加更多樂趣，如果你還有更多好玩的遊戲，歡迎在評論區與大家一起分享哦~
