DNF：首個未上線就涼的副本！1月活動獎勵，送“符文”新裝備

天天要聞
2019-12-16

地下城與勇士又要經歷一次大轉移了！這一次是因為魔界大戰引發的次元風暴，蔓延了阿拉德大陸的各種地方，100級重建新的地圖，比如建立了「西海岸聯盟」，但國服距離100級還差一個大版本，那就是造成大轉移2.0的「魔界大戰」副本，目前已更新到體驗服，應該會在1.14上線！但這副本卻被玩家吐槽，還不如當初的泰波爾斯有吸引力。

首個未上線就涼的副本！1月活動獎勵，送「符文」新裝備

因為魔界大戰副本的口碑本來就很差，BOSS是打樁，門檻高到離譜，而獎勵是職業技能改版，明顯是來拖進度的，如果變成職業平衡才對，小夥伴們肯定支持，而不少玩家都打算放棄這個副本的開荒，等著送材料！史派克Get到了，魔界大戰更新的同時直接送「符文」這個新裝備了。

由於副本和獎勵全照搬，史派克擔心玩家達不到門檻！「引導」模式提前穿越，這在DNF中是首個同步更新「新手」模式的副本，從泰波爾斯到普雷，都是玩家催了很久才更新引導模式，而魔界大戰，1月更新的同時直接開放，不僅如此，還加入了回歸獎勵！一上線就史詩級削弱，削弱的同時還順便送材料，比Raid減負穿越更新還給力。

「符文」和「護石」是新裝備，類似於勳章和守護珠有額外的裝備欄，這兩件新裝備可以讓職業的技能變得更強大或者更歡樂，但1月活動直接送了……就跟普雷更新的時候送黑瞳石或者是夜語黑瞳武器一樣滑稽，所以史派克也沒打算將「魔界大戰」這個副本作為春節版本的主打內容。

綜上，無論是玩家和史派克都沒有看好「魔界大戰」，未上線就涼的大型副本，這在DNF中也是首次出現，結合12.30的國服發布會，100級各種內容大概率會在3月份前後更新，送「符文」這個裝備的目的也是讓玩家們儘快畢業，之後才能更快速的投入到100級中。

所以，在1月更新魔界大戰的時候，如果覺得副本無聊的話，可以放棄開荒！因為這副本除了一張150點四維的耳環傳說卡之外，也沒有其它新屬性的卡片了。
