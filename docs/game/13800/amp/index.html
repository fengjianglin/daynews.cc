雲頂之弈：第二賽季出現“雷電法王”，傷害和坦度的完美結合

天天要聞
2019-12-13

第二賽季出現「雷電法王」，傷害和坦度的完美結合！

在雲頂之弈第一期中，特別是在槍手熱門的那幾個版本，電刀是一件經常出的裝備，因為羈絆的特殊性，電刀很容易觸發，曾經有玩家就出到了很多把電刀，於是盧錫安成了人造雷電法王。在第二季中，出電刀的基本絕跡了，因為沒有什麼主流英雄適合電刀。除了版本熱門陣容，其實深挖當前版本的冷門，盟盟發現有一個英雄非常適合電刀這件裝備：寒冰。

寒冰的攻速是經過加強的，而且開出技能之後，攻速更會暴漲，平A速度非常快，所以很容易觸發電刀效果。有玩家根據這個特性給玩家合了兩把電刀和一個巨殺，然後還將寒冰升到了三星，最後寒冰成了雲頂第二季的雷電法王。寒冰因為有遊俠和水晶羈絆，一個可以增加攻速，另一個可以增加自保能力，第二賽季出現「雷電法王」，傷害和坦度的完美結合，她就是寒冰！

另外，根據最新的更新消息來看，9.24版本的遊俠又進一步增強了，遊俠由30%/60%/100%的幾率在接下來的3秒內攻速翻倍變成了35%/80%/100%，所以雙遊俠和四遊俠發揮空間更大，感興趣的玩家不妨試試這個套路。

玩家不能D牌了，拳頭設計師：新版本叫「F牌」！

雲頂之弈已經出來快半年了，隨著玩的時間越長，在快捷鍵的使用方面也都差不多知道了。D代表刷新卡牌，F代表升級經驗，數字鍵12可以看上一家、下一家的情況，選秀是按數字3可以回到自己的陣營。在選秀時，大家可能會一邊看自己的裝備一邊觀察場上還剩哪些裝備，所以說3鍵還是非常有用的。當然用得最多的還是DF鍵，但是最近的更新消息設計師公布了一個事情，那就是DF鍵互換了，所以說以後再也不是D牌了，而是F牌了。

相信這一改動會讓很多玩家不適應，因為大家D牌都D習慣了，現在突然換到了F鍵來，肯定會適應不過來，所以說在新版本可能會出現很多失誤的情況，本來想要升級卻D牌，哦不好意思，自己都說成D牌了，拳頭設計師表示：以後是「F牌」。對於這個改動，盟盟不知道設計師是何意，本來DF好好的，為啥突然要改。除了DF鍵互換，不過大家可以改建喲。另外也進行了一些界面優化。比如在滑鼠懸停在購買經驗一欄，會出現到達下一級所需要的金幣，對於數學不好的玩家來說，這將是一個很好的消息；還有還將再增加一個主頁按鈕和輪播按鈕，主頁按鈕可以回到自己主場，輪播按鈕可以觀察其他對手情況。

雖然多了這些按鈕會讓玩家方便很多，但是盟盟建議還是要掌握快捷鍵，因為用快捷鍵速度還是比用滑鼠快很多，所以要進行操作是快捷鍵有很大優勢。
