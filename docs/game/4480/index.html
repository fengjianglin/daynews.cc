明日之後：特殊道具盤點及獲取攻略，來領新物資

摘要

隨著萊文市的開啟，為了商隊和倖存者們在末世更好地探索萊文市，這次熊大給大家帶來特殊道具的盤點，來看看這些道具你見過嗎？未知的幼蟲卵？帶著唇印的親筆信？如何獲取特殊道具？和熊大一起往下看看吧~

【生鏽的鐵質鑰匙】

道具獲取方式可以通過釣魚，獲取到生鏽的鐵質鑰匙。鑰匙表面的紋路顯示著，似乎需要去開啟某樣東西，或許瓦爾美正需要這樣東西。

瓦爾美出現在行政區附近，刷新地圖瓦爾美可能出現位置，會在這幾個紅圈標明的地方。

【未知的幼蟲卵】

未知的幼蟲卵可以通過打感染者獲得，沒見過的蟲卵，表面摸起來濕濕軟軟黏糊糊的，裡面的蟲子也沒有見過，看起來可能會是危險的新生物，需要將幼蟲卵交給海倫娜微恩，並護送至科技會檢驗。

海倫娜微恩會出現在萊文市，工業避難所附近，地圖上標明了海倫娜微恩刷新位置，注意的是她只會在白天出現。

【舊時代的黑膠唱片】

舊時代的黑膠唱片可以通過從梵郎格蘭手中獲取，對於舊時代的黑膠唱片來說，表面的些許劃痕可能會比較明顯，但在時代的角度上，已經是保養的很好了。黑膠唱片也同樣需要交給海倫娜微恩。

【海倫娜微恩親筆信】

道具獲取方式，找到海倫娜微恩，從海倫娜微恩親手拿到的信件，信封表面上會看到淡淡的唇印，需要做的是將這封親筆信交給梵郎格蘭，嘿嘿，好像知道了什麼不得了的事情…

梵郎格蘭會出現在萊文市居民區附近，地圖紅圈位置是，刷新梵郎格蘭會出現的地方，同樣要注意，他只會在白天出現。

【膠捲（城市美景分享）】

道具獲取方式需要先找到朵洛希哈維，從他手上拿到相機膠捲，在萊文市拍下城市最美一刻吧，萊文市美景分享。

【膠捲（危險神秘的大叔）】

同樣是從朵洛希哈維手中獲得相機膠捲，去需找梵郎格蘭拍照，看來梵郎格蘭就是那個危險神秘的大叔…又知道了什麼不得了的消息了…

【古老的野獸化石】

想要找到古老的野獸化石，可以通過打感染者獲得，從沒見過的野獸化石，從化石表面上看應該是已經沒有什麼生命跡象了（瑟瑟發抖）。將化石交給朵洛希哈維，她似乎在手機這些東西。

朵洛希哈維出現在萊文市商業區附近，刷新位置會出現在地圖標紅圈的地方，同樣朵洛希哈維只會在白天出現。

以上盤點的7種特殊道具，每天最多能獲取一個，觸發臨時委託任務的次數為每日3次，完成任務可以獲得物資獎勵（新幣、地圖專屬資源等）以及熟練度的採集，獲取物品將計入每日臨時委託任務次數，上交物品則不計入次數。快來收集特殊道具，完成任務獲取物資等獎勵吧~
