狼狗加強，曜削弱，排位賽大變天，掛機不禁賽，勝利不加星

2019-12-14 182條評論 1,471次閱讀 530人點贊

最近天美也不知道是怎麼了，要麼密不透風一點料子沒有，要麼一下子爆幾百個大料出來！昨天體驗服突然上線了N個新東西，其中包括英雄、野怪、自走棋、信譽系統等等調整，還有多個新皮膚資訊！

（網圖，非真實皮膚）

因為涉及的新內容實在太多了，老王一次性全把它們寫出來估計能搞個1萬字的論文，為了兄弟們看得不那麼累，所以老王將這些新內容分成了幾篇，這幾天陸續發出來。（想看新皮膚內容的記得要每天盯著老王更新，魯班和大小喬都有新皮膚）

這篇文我們先來看看比較重要的英雄及排位賽調整，待會文章則是關於自走棋的調整。

一 英雄調整

1、曜

星削的比例由50%降到了40%，星削對於曜而言就是相當於免傷，兄弟們可以理解為曜的移除傷害能力由50%變成了40%，這絕對是一波不小的削弱。

2、安琪拉

安琪拉大招增加的旋轉速度被回收了，對於這點老王覺得是合理的，因為提升後的轉速實在是太快了！

有網友惡搞了一波安琪拉，這次轉速加一點點，過幾天再加一點點，最後就變成直升機了！這誰頂得住啊！

3、成吉思汗

天美上次對哈士奇1技能的被動進行了無情調整，觸發條件由對同一目標造成3次攻擊觸發額外傷害變成了4次攻擊才觸發。為了彌補上次的削弱，這次把期觸發的印記傷害上調了，以及提升了一些移速。

4、盤古

不知道到底是天美真盯上了老王，還是老王的嘴被開過光！昨天剛吹了一篇盤古攻略，天美昨晚就把盤古給削弱了！點擊藍字可以回顧昨天盤古攻略，在削弱上線正式服前抓緊時間玩！

暴錘猴子，戲虐典韋，橫行野區，射手看見他就跑！只有這個英雄

5、嬴政

兄弟們要注意，嬴政的體驗服調整均是建立在重做後的數據之上，新嬴政大招由55把飛劍變成了90把飛劍。

6、魯班大師

魯班大師雖然機制真的強無敵，但在中低端局表現實在太差了，天美不得不撈一把，把基礎生命值和大招加強了一波。調整後的大招很強，被吸到的敵人不立刻交閃幾乎跑不掉。

二 信譽系統調整

1、掛機不能獲得星星

新增掛機不得星系統，只要在對局內掛機時間超過總時長的50%，即使排位贏了也不會加星星。但老王覺得這個機制還需完善，之前排位掛機是禁賽，現在是勝利不加星，而且50%的時間放得太寬了，老王認為掛機超過20%的時間就應該獲得處罰！

2、積分補償提升

一旦隊伍中出現隊友掛機的情況，無論輸贏系統都會補償一定的勇者積分，但對於高段位的玩家而言，那20積分簡直猶如雞肋。於是天美這次也對補償的積分進行了梯度式調整，段位越高補償的積分數越多。
