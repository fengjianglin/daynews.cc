三國殺：如果引入全部將池，國戰會有哪些神組合？

2019-12-07 0條評論 17次閱讀 0人點贊

摘要

如果國戰用的是OL全部的將池，那會有哪些強力的搭配呢？相信有一些殺友開過類似的腦洞，甚至在面殺時實踐過。那我們就來看一下，假設是全部武將都加入國戰，會有哪些神組合吧！馬良/夏侯氏，殺當無中用馬神如果火線加入國戰，有望取代黃月英，成為新一代的蜀國開團核心。

馬良/夏侯氏，殺當無中用

馬神如果火線加入國戰，有望取代黃月英，成為新一代的蜀國開團核心。那麼，誰跟馬良搭配，能發揮出他最大的威力呢？夏侯氏就是一個很合適的選擇。

夏侯氏回合內可以重鑄殺，而馬良重鑄的時候就相當於在開無中生有，過牌量爆炸，一套操作後還可以把牌都給隊友。

以這個組合的能力，每回合穩定重鑄兩張殺基本沒啥難度，因此末了又能【燕語】給隊友補兩張牌。大過牌、高爆發、強包養，【馬良/夏侯氏】堪稱頂級發動機組合！唯一一個小小的缺憾是，【樵拾】過的牌得棄掉。

關索/張飛，人肉連弩安排上了

關索是軍爭場全身份的T0級神將，【征南】給了他極其強力的後期發育。雖然沒有拿到連弩的關索也很強，但如果拿到了這件夢寐以求的裝備，強度會直線提升三個檔次。

那若是給關索配上一個張飛，讓他從頭到尾出殺都沒有次數限制呢？可以肯定的是，一旦這個組合靠死人牌發育起來了，那麼他們一定勢不可擋！

李典/荀攸，輪輪AOE大忘隙

李典之所以是魏國的一大核心，就在於可以【恂恂】找AOE，然後召喚魏國開團，一波大忘隙原地起爆。如果讓他拿到了連弩，更是直接起飛！

那麼，一個可以每輪穩定開A大過牌的李典究竟有多強，無需贅述了吧？而且，這一對組合還能通過忘隙的摸牌，緩和荀攸【奇策】後容易空城被集火的問題，可謂是黃金拍檔。

辛憲英/曹丕，超長續航

國戰場素來有「魏國開團，少了太子降一檔」的說法。雙將機制，使得大多數武將回合內都能有較高的收益。因此，曹丕的【放逐】相比軍爭場更顯霸道，控死對面核心的同時，還能保證己方多人的聯動，在大魏中的地位與李典並列雙核。

而辛憲英的【才識】則能以很小的代價每回合回1血，有極強的回狀態能力。一個能每輪穩定回血，可以超長續航的曹丕，強度可想而知。我想沒有多少人願意碰到這樣一個組合吧！

孫尚香/徐氏，大吳雙核

香香是國戰的吳國大核，徐氏是軍爭勝率爆表的反賊頭子。沒想到，這兩個女人組成的雙核，竟然還能有1+1>2的效果！

別的不多說，就想像一下，一個回合外可以問卦過牌的香香，一個回合內可以制掉自己裝備的香香，一個進可發大招退可控判定的香香，到底會強到什麼程度？我反正是服了，你們呢？

孫尚香/步騭，雜耍團王炸組合

香香和步騭在軍爭場有不錯的配合，那如果兩人組成一個團隊，在國戰出場，會有怎麼樣的化學反應呢？

答案是，如同火星撞地球一般！步騭回合內發動【定叛】，棄裝備可以摸4牌（定叛1+梟姬2+弘德隊友1）；而若是全部裝備收回，以2裝備為例，可以掉1血摸8牌（定叛1+梟姬4+弘德隊友3）。這爆炸的收益已經無法用語言來形容了，更讓對手絕望的是竟然可以多次發動！

步騭的技能可以很好地配合香香，而更妙的是，香香無論是發動【梟姬】還是【結姻】，都可以觸發步騭的【弘德】。這相性，簡直完美。大家都戲稱吳國開團是「雜耍團」，那麼【孫尚香/步騭】必定是當仁不讓的王炸！

李儒/賈詡，雙大招極限輸出

兩個毒士湊一塊的「黑白無常」組合，先別說強度，玩起來就十分帶勁。

這兩人的大招都能製造驚人的傷害，可以瞬間改變局勢，再配上【絕策】和【完殺】，一回合把場子砸下來都不是什麼難事！不過美中不足的是，這兩人的大招都是無差別攻擊，隊友也無法倖免，所以有時候可能會事與願違。

袁術/孔融，不棄牌的庸肆

「術爸」是較早出現的頂級強將，直到現在仍然在軍爭場屹立不倒。要說他的缺點的話，就是每回合結束後都得棄好多牌，一方面是有些可惜，另一方面也降低了自己的防禦。

不過，有孔融在的話就不一樣了。【袁術/孔融】可以把棄的牌都給隊友，形成團隊內的無代價【庸肆】，而孔融的【名士】又能稍稍提升防禦力，相性不錯。這個組合若是能加入國戰，必然是一個不可小覷的強力搭配。

陳宮/沮授，宇宙級防禦

沮授的【矢北】已經是相當強力的防禦技能，只有在面對多重傷害時才會暴露出負面性。而有了陳宮的【智遲】，沮授遭遇二次傷害的幾率就要小很多了，除非技能製造的傷害，不然都奈何不了這個鐵桶組合。

除了宇宙級的防禦力，這個組合在回合內也能有所建樹。陳宮的輔助、沮授的過牌，都能為群雄開團帶來貢獻。

上述這些組合的強度你覺得怎樣？如果國戰放開全部將池，你心目中還有頂級組合？在評論區分享你的看法吧！～
