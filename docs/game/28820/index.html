體驗服全新系統搶先看 新版本玩法前瞻

2019-12-21 260條評論 1,472次閱讀 481人點贊

國服發布會臨近，體驗服也在近期進行了第一波大更新，全新決戰魔界風雲版本也到來了。新的護石頭符文又被大家簡稱為CP系統。在本次新版本中到處充滿著亮點，接下來就隨筆者一起來看看吧。

一、CP系統(國服名稱為護石符文系統，以下簡稱為CP系統)

新版本中最大的改動，莫過於增加了CP系統了，相信關注韓服的勇士們一定早就知道這一CP系統。其主要目的就是改變技能形態，提高勇士們傷害的同時增加遊戲的可玩性與每一位勇士的獨立性。

每個角色一共可以裝備2個護石，每個護石上可以鑲嵌3枚符文，這一系統可以改變技能形態。從傷害、冷卻、脫手、增加hit數、減少hit數等改變中進行選擇，打造屬於自己的獨特的技能和職業。

二、新增怪物卡片

魔界大戰版本中新增了10張附魔卡，其屬性除滿級黑眼之夏勒弗茲卡片能夠為耳環提供150四維，高於滿級光盧克的125四維之外，其他卡片屬性皆與現版本中附魔卡屬性相同，唯一區別在於武器附魔可以進行雙屬強12附魔，但對於當前版本來說用處不大。

這10張新附魔卡的更重要意義在於可以對現在拍賣行進行衝擊，提高相同屬性卡產出量，屆時頂級附魔卡價格可能會有所降低。

對於奶系職業而言，可以將耳環更換為滿級黑眼之夏勒弗茲，可再額外提升25點四維，但是頂級附魔卡在版本初期價格相對會非常之高，是否需要追求極限就看個人想法了。

三、附魔卡片及寶珠限制交易次數

新版本中不僅增添新的附魔卡，更對附魔卡和寶珠進行了交易次數的限制，從原有的隨意交易改為可交易1次。

這一改動極大限制了黃牛黨對附魔卡和寶珠進行囤貨，蓄意抬高附魔卡片價格的問題，可以讓所有卡片恢復到正常價位，可以說這一改動對玩家們非常友好。

四、新增功能NPC

新的NPC在亞瑟旁邊，可以直接進行時空石的交易和普雷團本頻道的選擇，同時分擔了亞瑟的工作，雖然這一改動作用不大，但為增加遊戲便利性，裝備的提升由特殊裝置完成。

各位勇士們一定要注意，不用再去找亞瑟升級裝備了。

五、職業平衡

這次平衡的職業有聖騎士(男)、死靈術士、驅魔師、決戰者、魔道學者、馭劍士、阿修羅，數據以最新版本及正式服為準。

以上職業獲得加強，有喜歡的勇士們可以抓緊玩起來。

六、系統菜單新增人物詞典

新增的人物詞典主要可以清晰的提供各個NPC所在的位置，為新玩家和回歸玩家快速上手，能夠迅速找到NPC位置。

同時人物詞典頁面也可直接查看NPC好感度，可以讓收集類玩家更清晰的看到NPC好感度提升之後話語的改變。同時也能實時注意跟凱麗和克倫特的好感度，畢竟很多強化玩家喜歡把這兩個NPC好感度提升到滿。

七、任務欄優化

任務欄優化更加清晰，能夠非常直觀看到主要任務和其他任務，最重要主線劇情的查看也更加方便，滿足了追求故事劇情的玩家，普通玩家也可以洗洗觀看一遍，然後猜測之後劇情走向，也是一個非常不錯的樂趣來源。

八、冒險團改動

現在冒險團可以設置在線、忙碌、離開三種狀態了，更加適合掛機玩家，可以更加直觀的看到朋友們的狀態，省去了苦苦等待。

九、回歸冒險家改版

回歸冒險家改版之後可以讓回歸的勇士們更快的追上遊戲版本，甚至計算好回歸時間，要比一直在玩的勇士們速度更快。

只要達到等級並且完成簡單的任務就可以拿到豐厚獎勵，獎勵中更是有下個版本的新系統道具——符文原石禮盒。加快CP系統的畢業速度，可以說是現在頂尖玩家都非常想要的獎勵。

而且回歸玩家在組隊的時候還能夠增加整個隊伍該副本對應獎勵的數量，只要達到副本門檻，都是非常搶手的，極其容易跟上遊戲進度。

目前體驗服只更新了第一波，筆者預計後續應該還會更新新版本活動，各位勇士們可以多多關注DNF助手與體驗服的相關信息喲。
