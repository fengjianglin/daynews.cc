光子賽高：《PUBGM》全球日活5000萬，你是其中一份子嗎？

哈嘍小夥伴們好呀，這裡是誠然。日前光子對外公布了《PUBGM》海外全球的傲人成績：自《和平精英》在國內日活躍用戶突破5000萬之後，其下《PUBG Mobile》產品也在海外獲得了日活5000萬，月活躍高達1億，下載量突破6億的好成績。就此看來，好比每26人今天就有一人登陸了遊戲，13人中至少有1位本月登陸了遊戲，由此可見活躍度之高。

不僅如此，作為強調合作的射擊遊戲，玩家們還都呈現出「活躍群」的現狀，如果你身邊的朋友在玩《和平精英》或者《PUBGM》，那麼你肯定也多多少少會被拉過來偶爾開黑一把。這兩款作為移動端領航地位的遊戲，在社交上也同樣是佼佼者：帶妹吃雞或者被妹帶吃雞都是家常便飯。可能也正因如此高的女性玩家比例，讓《和平精英》魅力四射。

但是如果要深入探討一下《和平精英》或者《PUBGM》獲得如此傲人成績的原因，恐怕不止「妹子多」這一個簡單的原因，而是要問：為什麼玩家們這麼喜歡？

首先不得不承認的是，無論是《和平精英》還是《PUBGM》他們的誕生都是站在了巨人的肩膀上的。端游《絕地求生》的火爆，讓他們在移動端也同樣擁有著大量的基礎用戶，而《絕地求生》這個IP，在全球依然擁有著「高品質」的標籤，作為移動端的產品，緊緊跟隨端游的遊戲進度和內容，比如槍械的更迭，載具的上線等等，讓手游內容更加豐富化。

但是這些也僅僅是基礎，作為移動端的大作，除了跟隨著端游IP的更新，勢必不可少的是自身的創新和發展。比如在遊戲模式中，不僅僅是端游中經典的團隊玩法，還有著具有創新性的暗夜模式、火力對決等等，讓很多玩家體驗到了傳統FPS的樂趣。

在排位競技的同時也能愉快的娛樂競技，讓遊戲能夠保持新鮮的同時滿足不同的需求，這就是《和平精英》和《PUBGM》的成功之處。同樣的開啟付費模式之後，不管是《和平精英》還是《PUBGM》一直不斷地更新主題賽季和各種好看有趣的外觀，也讓遊戲增添了不少的樂趣。

所以任何光彩奪目的背後不可能僅僅只是靠運氣，強大的IP，堅持自身的創新和優化都是必不可少的。由此也讓《PUBGM》獲得不少傲人的頭銜：比如谷歌2019年度最受歡迎、最佳對戰遊戲，第36屆金搖桿獎年度最佳移動遊戲等等，其全球賽事PUBG MOBILE CLUB Open雖然僅僅舉辦了兩屆，但是在海外直播和視頻頻道已經獲得了全球直播觀看量第5，手游第一的好成績。

如今的《PUBG Mobile》日活達到了5000萬，那麼正在看這篇文章的你，是否是其中一員呢？
