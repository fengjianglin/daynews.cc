《魔獸世界》：瓦王被玩家“挾持”進鐵爐堡，跪見銅須國王

天天要聞
2019-12-09

摘要

再過兩天到12月11日，戰歌峽谷和奧特蘭特山谷兩個《魔獸世界》中最經典的戰場就將正式上線。屆時野外PVP大亂斗的現象將減少，不過在這之前野外PVP仍將是獲取榮譽的主要途徑。而近日在碧空之歌伺服器野外就發生了有趣的一幕，我們尊敬的“瓦王”居然被眾人挾持到了鐵爐堡，跪見麥格尼銅須。

再過兩天到12月11日，戰歌峽谷和奧特蘭特山谷兩個《魔獸世界》中最經典的戰場就將正式上線。屆時野外PVP大亂斗的現象將減少，不過在這之前野外PVP仍將是獲取榮譽的主要途徑。而近日在碧空之歌伺服器野外就發生了有趣的一幕，我們尊敬的「瓦王」居然被眾人挾持到了鐵爐堡，跪見麥格尼銅須。

相信讀者肯定會產生疑問，因為劇情原因，目前懷舊服中瓦王並沒有出場。此時瓦王應該在塵泥沼澤和瓦莉拉、熊皮組隊痴迷於競技場3V3，完全無心攝政。其實開頭提到的「瓦王」並不是瓦里安烏瑞恩國王，而是一名獸人戰士。

這位獸人戰士或許是因為喜歡瓦里安國王一生的傳奇經歷和他身上堅韌不屈的品質，才用部落角色取了這麼一個聯盟NPC名字，說他「生在曹營心在漢」都完全不過分。而正是因為他的這個名字，導致了接下來發生了有趣一幕。

眾所周知，榮譽系統上線後，為了提升軍銜等級獲得裝備，大量玩家在野外尋找敵對陣營玩家進行攻擊。這位「瓦王」也是如此，常年在灼熱峽谷偷襲黑石塔路上的落單聯盟。

可俗話說得好「常在河邊走哪有不濕鞋」，這位「瓦王」沒多久就被聯盟組隊給逮住了。面對眼前的一片血條，反抗是肯定不可能有結果的，「瓦王」選擇了站在原地等一個安詳的去世。

不過這裡聯盟並沒有像往常一樣一頓集火直接帶走獸人戰士，而是決定「挾持」這位特別的瓦王到鐵爐堡做客。

於是聯盟方便讓幾位牧師輪流通過心控操縱獸人戰士前行，期間如果控制遞減太高，便讓盜賊用悶棍或者法師使用變形術控住獸人戰士等待控制遞減重置。

聯盟以此往複，真的慢慢將「瓦王」帶到了鐵爐堡。最後這位瓦王真的被押送到了鐵爐堡，併除了被不知情的聯盟失手錯殺了一次，期間沒有收到任何衛兵的攻擊。

不知是該說這位「瓦王」心態好，還是說其實他本人也想去鐵爐堡視察一番。因為他隨時都可以直接下線，以此讓聯盟的計劃落空。可他不僅沒有下線，在鐵爐堡中被聯盟誤殺之後，還非常配合地接受了靈魂石在原地復活。

估計「瓦王」當時正在唱薛之謙的《演員》，心裡想著那句「該配合你演出的我儘力在表演」。

這要是小編被被敵對陣營心控押送到主城，肯定立馬下線，可以說「瓦王覲見矮人國王」這一出好戲既有聯盟玩家奇思妙想的功勞，也有瓦王儘力配合的功勞。

懷舊服的玩家可以說真的把《魔獸世界》玩出了新花樣，除此之外懷舊服的玩家還在遊戲中創辦期刊雜誌，售賣大閘蟹、代辦ETC。對於暴雪來說，這遊戲讓他們75元玩一個月真是虧大了。
