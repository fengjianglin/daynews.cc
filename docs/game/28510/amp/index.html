BA居居：操作最強的選手是一諾，花海，九尾，初晨是次級時期更強

天天要聞
2019-12-23

KPL秋季賽已經結束，冬冠杯的比賽激戰正酣，而在冬冠開始之前，作為解說的BA居居也對各支隊伍進行了點評。其中也對一些選手的實力進行了評價，他表示他心目中個人操作最強的幾位選手分別是AG超玩會一諾，estar花海，XQ九尾，次級聯賽時期的初晨以及QG的139。

一諾自然不用多說，他和居居是BA時期的老隊友了，當時也是一流的戰邊，他本賽季轉型C位之後在AG超玩會的表現也是無可挑剔。一諾的天賦讓很多KPL選手都非常羨慕，玩一個英雄上手非常快。並且一諾也是唯一一個戰邊，野核，射手都非常出色的選手，本賽季AG超玩會奪冠和一諾的表現是絕對分不開的，雖然沒有拿到FMVP，但是個人操作已經得到了所有人的認可。

Estar花海是上半年最火熱的新星，花海早在沒登錄KPL之前就名氣很響了，進入KPL之後更是被認為補足了estar的最後一塊短板。春季賽和世冠的兩個冠軍對於花海來說也是最好的肯定，他的個人操作和反應速度都是頂尖的，KPL首秀就拿到五殺，這也是唯一一個。

XQ的九尾是打Hero的時候一戰成名的。一開始大家對XQ的幾位新人充滿了未知，覺得他們沒有什麼資格頂替老人，尤其是不少阿泰的粉絲都對九尾有意見。但是九尾用賽場上的表現證明了自己確實值得。不只是成名英雄不知火舞，包括貂蟬，元歌這些刺客類英雄都爐火純青。本賽季還開發出了不少奇奇怪怪的中單，冬冠和Hero的比賽中更是拿出了關羽中單，也堪稱是新一代中路萬花筒了。

第四個是次級聯賽的初晨，為什麼要加上這個限定呢？其實初晨在次級的時候比現在要亮眼的多，但是他玩的都是一些李白，阿軻，露娜這樣的秀兒英雄。到了KPL之後反而變得平庸，這也和版本有關係，現在的初晨已經很少像當初那樣頻頻上演高光操作了，EDGM的成績也始終沒辦法進一步突破。

最後則是QG的139，也是居居的老隊友。居居給139的評價非常高，但是139從來都沒有在賽場上打出來過。可能是139在BA訓練賽中表現得特別猛，但是比賽和訓練賽終究不一樣，139目前也是枯坐替補席，到底水平如何我們也不得而知啊。
