就在今天，SN正式宣布“新上單”加盟，JKL的官宣時間也已經確定

2019-12-09 0條評論 16次閱讀 0人點贊

摘要

LPL轉會市場再過10天就要落下帷幕，除了“兮夜”的去向還不明朗外，其他選手大都有了歸宿。而作為轉會期最大熱門的JKL，據說最終去向也要被確定了，之前大概率說是去SN，因為SN來了一個“超級上單”，被粉絲吹捧成“吊打”theshy的存在，準備和JKL組成“銀河戰艦”。

LPL轉會市場再過10天就要落下帷幕，除了「兮夜」的去向還不明朗外，其他選手大都有了歸宿。而作為轉會期最大熱門的JKL，據說最終去向也要被確定了，之前大概率說是去SN，因為SN來了一個「超級上單」，被粉絲吹捧成「吊打」theshy的存在，準備和JKL組成「銀河戰艦」。而在今天，SN終於官宣了這個消息。

而這名新上單就是來自二隊的上單，ID名叫做Bin，從今天起就正式擔任一隊的首發上單了。可能是為了證明自己的實力，Bin在韓服開始了自己的沖分之旅，目前已經來到了韓服的第10位，不過他的戰績，小米不知道怎麼評價。要麼就很C，要麼就很崩。

或許這跟他的英雄池和打法有關，小米看一下他的英雄池。應該是一名carry型上單，而且打法很兇悍，要麼在線上打崩你，要麼線上被打崩，一定要和你拼個你死我活。在這一點上，他的模板應該就是theshy，至於粉絲傳得很厲害的「吊打」theshy，小米是不信的。

而SN除了新上單加盟外，還有一個好消息。那就是有SN內部工作人員爆料，阿水加盟的消息將會在11號進行官宣。這樣一來的話，關於LPL轉會期最大懸案「阿水去哪兒」，就要迎來大結局，大家也就不用猜來猜去了。

再加上之前SN直播時採用集體閉麥，以及在中單angel的直播間，出現了疑似阿水的聲音。JKL加盟SN也就是板上釘釘的事了。不過小米還是很好奇兮夜最後會去哪。因為到現在還沒有他的靠譜點的消息傳出。
