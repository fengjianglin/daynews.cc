夢三國神隊友系列：這姜維的大招，真是太絕了

2019-12-24 365條評論 1,602次閱讀 171人點贊

姜維作為近些年來才崛起的一個英雄，各個地方都能看到他的影子，早些年間，這個英雄剛出來的時候，還比較火，後來就不行了。中間真是不知道過了多久，大概是INBA官渡出來以後，這個英雄才算又火了起來，INBA官渡里，這個英雄簡直就是清線狂魔啊，配合增強的分裂刀，GANK隊友也是一個很有力的存在啊！不過現在的姜維，在大招改版以後，更開始向一個團控兼輸出的方向發展了！

張郃！

張郃這邊在成功拿下了炎魔以後，張郃也是成功吃到了復活BUFF，這有了狀態就是浪啊，張郃瞬間開大，單打對方的文丑，文丑哪能是張郃的對手啊，很快就被張郃給擊殺了！

雙殺！

這單殺雖然帥氣，但是有時候也是要付出代價的，對方四人立刻傳送，瞬間形成了一個多打一的局面！不過隊友的支援速度真是絕了，之前還在BOSS區域那裡，這會兒已經在對方的下路二塔了，這速度真是可怕，隊友姜維迅速開大，而且禁錮給的也非常及時，幫助張郃躲過了致命傷害，而張郃則成功拿到了雙殺！

四殺！

這波刷新大招真是太帥了！姜維在關鍵時刻一個刷新大，瞬間打蒙對方！估計對方這時候都不知道自己在哪了，因為姜維大招的關係，場面實在是太亂了，只能想辦法拉開距離，但是姜維大招大家是知道的，對方是出不去的，而己方就可以盡情輸出了，張郃趁勢四殺！這波四殺，真是姜維的功勞，要沒有姜維的話，估計張郃就得交待在這裡了！
