《獵人》：騰訊新款動作格鬥手游上線，高度還原《全職獵人》劇情

2019-12-06 0條評論 13次閱讀 0人點贊

摘要

最近收到很多小夥伴的私信提問：最近世面上的新手游好少呀，《獵人》手游值不值得入坑？

最近收到很多小夥伴的私信提問：最近世面上的新手游好少呀，《獵人》手游值不值得入坑？《獵人》在測試前我已經在體驗服玩了很長的一段時間，對於熱愛格鬥的我來說，這一款遊戲體驗之後根本停不下來，不僅是因為動作格鬥遊戲本身的魅力，還有喜歡《全職獵人》動漫的我來說，這款遊戲帶給我完全不同的體驗。

高還原度的玩法體驗

市面上不缺由各種動漫IP衍生的手游，但大多數是掛著IP而重新定製的遊戲，甚至還有將熱血格鬥的動漫改為卡牌玩法，畢竟對於動畫劇情的還原需要花費非常多的精力打磨才能達到理想的效果。所以《獵人》手游宣傳時強調以《全職獵人》為藍本，讓玩家扮演獵人中的無名遺願來見證原著角色的成長和相關故事，強調角色養成和社交的動作遊戲，而體驗這款遊戲之後，從遊戲的創建到後期的玩法讓你從切身實際的再次”觀看”《全職獵人》。

遊戲的主線劇情任務就是以獵人考核作為路線，所以在做主線任務的過程中，就能體驗到獵人們是如何成長，代入感非常強，並由劇情而衍生出核心玩法【秘境】，秘境分為初見考試、競爭與邂逅、成長的挑戰、內心的陰影、快速成長、拍賣會的意外六個大章節，每一個章節都會有不同的劇情分支但又相互關聯，在樂此不彼的體驗劇情發展時，還能獲得各種材料和裝備獎勵，並且還不會覺得枯燥乏味。

高度自由的職業技能系統

一款動作類遊戲，如果缺少職業特色就會失去樂趣，大多數傳統動作手游以近戰劍士、遠程弓手和魔法師三類職業作為選擇，所以玩起來就感覺沒什麼亮點，所以對《獵人》的多樣化職業充滿的新奇感。

《獵人》將職業劃分為強化系、變化系、放出系、具現化系、特製質系和操作系，11個職業轉職分支讓職業之間各有特點，增加了職業的可選擇性，每一個職業都有不同的操作要求和在隊伍中的不同作用，並且和傳統的動作手游不同，很多手游選擇一個轉職之後就無法變更，如果想要體驗同系列中的其他轉職需要重新升級，而這一個升級過程非常的枯燥無味，正因為這樣的想法，《獵人》的職業系統讓玩家早升級前期就能體驗到雙職業的核心技能，並且在玩家45級覺醒前可以隨時切換同系轉職，高自由度的切換讓自己選擇出最合適的職業。

我創建的角色是強化系，轉職選擇了狂戰士，但狂戰士是一個近戰坦克輸出職業，所以想要體驗技能脫手輸出劍術家，《獵人》的技能系統中有自由轉職的功能，可以自由切換同系列中的職業，第一次更換轉職時免費，後續需要少量的金幣，不需要從頭升級減少時間和枯燥感。

除了職業類型的豐富化，每一個職業都有不同等級的技能，可以根據自己的操作習慣選擇合適的技能進行加點，這樣的技能分支玩法和傳統的4技能動作手游有了很大的不同，自由性和操作性更高，自帶2個技能加點方案讓玩家可以針對PVE和PVP隨時調整無縫切換。

而這裡最有趣的是覺醒技能機制，45級後可以進行覺醒，但是覺醒後不能再隨心所欲的切換角色，但可以獲得更強的覺醒主動和被動技能，進一步的強化職業特點，比如玩的劍術家覺醒後擁有傷害極高的主動技能[一瞬雪月花]，還有增加劍術家基礎生命和異常抗性，並有幾率減少技能CD的被動[殘心]，所以覺醒後的職業才是最完美的職業。

獨特的夥伴養成系統

《獵人》強調的是養成，角色養成的內容非常多，獵人再對角色養成上進行了整合UI，即使是新手玩家也能非常直觀的了解自己需要做什麼。

獵人將養成獨立整合為【養成】ui，包括成就和時裝合成，而鍛造則是角色戰力養成的關鍵玩法。

在鍛造系統中主要是對裝備的養成，包括大家熟悉的強化系統和寶石鑲嵌系統，並有裝備洗鍊和祝福附魔系統，整合性的UI讓玩家可以隨時調整屬性，便利性大幅度的增加。不過裝備養成和傳統的動作手游區別不大，都是後期和核心內容，而特有的夥伴養成才是《獵人》的特色養成玩法。

夥伴系統是一個充滿趣味的核心收集養成玩法，夥伴除了念力寶箱獲取外，免費的獲取途徑可以在主線劇情任務中獲得部分的《全職獵人》故事中的主角，此外每天限時開放的陷阱塔挑戰也能獲得夥伴寶箱碎片和夥伴信物，從而合成獲得免費的夥伴。

通過夥伴收集可以讓玩家對故事中的主角有進一步的了解，收集到的夥伴還能助戰，所以可以選擇自己最喜歡的一個著名角色和自己一起挑戰各大副本。夥伴除了助戰外還需要對齊進行突破升級以及訓練養成，增加夥伴的屬性。收集不同的夥伴還會形成共有趣的羈絆效果，帶來的屬性加成。

豐富的裝備玩法

大多數的傳統的動作手游對於裝備系統並沒有太多的強調，隨著劇情獲得不同的裝備，主要的核心部分是強化，職業裝備千篇一律。《獵人》針對裝備分為普通、稀有、神器和史詩以及幻界等多種品級，並由此產生多種裝備的獲取途徑玩法-秘境。

秘境是活動中的一個玩法分值，而它也是《獵人》最核心的玩法之一，秘境分為每日玩法根據等級開放獲得不同的材料，裝備玩法則是虛空、遺迹遺迹幻界。虛空分為不同等級開放的初級、中級、高級和原始虛空，可以獲得不同級別的史詩裝備；遺迹則是獲得神器級別的裝備；當玩家達到58級後開啟的幻界咒縛之島可以獲得最強的幻界史詩裝備，讓玩家在裝備獲取上每個等級階段都能帶來不同的玩法。

自由公平競技玩法

動作格鬥遊戲的魅力就在於競技，《獵人》在競技玩法上帶來了多樣化的體驗，決鬥系統中可以進行自由練習，讓玩家熟悉當前的職業技能和連招輸出，還能自由創建房間開啟單人或者組隊以及車輪戰的競技體驗，也可以隨時匹配對手開啟對戰，決鬥系統不僅可以讓玩家獲得排名成就，也能獲得裝備等獎勵。

此外特有的限時競技玩法還增加了玩家的社交，每周二、四、六開啟的天空競技場可以和好友組隊或者和陌生人組成隊伍競技獲得寶石和銀幣獎勵；比武大會更是展現玩家技術的競技模式。

而這些競技玩法不會因為氪金就能穩贏，數值天平系統讓競技更公平，這也就是《獵人》的特色玩法之一。

個人體驗

從劇情來說，《獵人》強調還原原著，所以可以和原著里的角色並肩作戰，對於熟悉和不熟悉《全職獵人》的玩家來說都有非常好的劇情體驗；從養成方面來說，多樣化的養成方式和裝備獲取渠道，大幅度的增加了傳統動作遊戲的單一化成長趣味性；從格鬥競技上來說，因為數值天平讓平民玩家和土豪玩家的差距拉近，以技術作為核心玩法。而作為動作格鬥遊戲本身，多樣化的技能組合可以爽快並華麗連招，整體表現不錯。

如果你和我一樣喜歡動作格鬥，也喜歡《全職獵人》這部作品，那麼這一款遊戲確實值得體驗。
