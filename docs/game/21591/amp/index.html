EDG和WE開始與V5互比誰“慘”！新賽季EDG和WE陣容讓人堪憂

天天要聞
2019-12-19

一年一度的冬季轉會期如今已經結束，LPL的戰隊可謂是幾家歡喜幾家愁，聯賽墊底的VG組成「銀河戰艦」，老牌強隊EDG和WE開始跟V5比「慘」，意圖爭出誰是LPL「最慘」戰隊。

兮夜、大舅子和957三名隊伍核心的離開，讓WE元氣大傷，為了能夠及時止損，「沒錢」的WE發揮自己最擅長的「押寶」，花較少的價格簽約新人，真就開啟五年計劃。

WE簽下LNG替補中單Plex和JDG替補上單Morgan，招攬EDG二隊中單Yimeng，提拔自家二隊AD Jiumeng進入一隊，這幾名選手均是初次在LPL打比賽，別看Plex和Morgan當過戰隊替補，他倆在前戰隊根本就沒上場打比賽的機會。

新賽季WE就指望這幾名小將打出成績，有趣的是，被迫成為老人的悲傷、Poss、Missing三人，還要肩負起帶動隊伍新人的使命，WE真的是太難了！

EDG的情況比WE的要稍微好點，廠長退役轉到教練位置，iBoy離隊加入VG，讓EDG給人一種「凄慘」的感覺，實際上EDG新賽季陣容保持90%的原班人馬，頂替iBoy的AD是Hope，這名選手也是初次擔任LPL首發。

另外，EDG轉會期買到FPX替補冠軍打野Xinyi，Xinyi在FPX時，因為小天狀態太好，他一整年都沒有上場的機會，現在加入EDG，Xinyi或許能夠坐鎮首發，前提是JieJie和Junjia狀態沒有他好的時候。

早前EDG曾說過要以Meiko為核心建隊，現在人員都沒有變動，難道把iBoy轉給VG，讓VG組成「銀河戰艦」，自己啟用新人AD，就是以Meiko建隊的最大核心？

V5是LPL唯一一支沒有參與到冬季轉會期的隊伍，好像冬季轉會期跟V5沒有太大關係一樣，管澤元前兩天還調侃V5，稱V5新賽季或許無法湊齊五人，不過，V5官博這兩天頻繁營業，放出隊伍選手集體做夜宵的照片，照這個情況看來，V5新賽季是維持原班人馬不變。

新賽季LPL或許會發生很多有趣的事情，你們覺得V5、EDG、WE這三支隊伍，那個在新賽季會更「慘」呢？
