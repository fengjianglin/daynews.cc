FGO2.5新從者暫定評價，最高A+評分

2019-12-22 967條評論 2,541次閱讀 780人點贊

FGO日服開啟主線就去2.5已經兩天，第一批新從者大家也都熟悉起來，日服wiki appmedia也對新從者們做出了初步的評價，一起來了解一下吧。

超人俄里翁目前暫定的評分為A~A+級，由於寶具是輔助寶具，所以應該是算在輔助寶具的從者之列，與始皇帝、貞德和五星BB（輔助性）同級，而最有參考價值的應該就是始皇帝了，二者都是自BUFF型的輸出，政哥哥比較偏向防禦型的長線作戰，超人比較偏向爆髮型的輸出，這個評分在輔助型寶具排行榜裡面屬於最高評價了，因為往上的S級和SS級就是五大拐的領域了。

這裡給出的超人評價是①寶具的弱體無效和三回合無敵貫通非常優秀，②使用三技能的暴擊傷害可以比肩寶具傷害，③就算沒有三技能，以寶具的BUFF也能擔保高輸出。

而周回則是沒有評價，在周回作戰上沒有什麼優勢，這也是自BUFF輸出手的通病，當然術業有專攻，也無需太過糾結這方面的情況。

歐羅巴的暫定評分為A級，在光炮榜屬於平均水平，能與強化過後的大帝和雷帝平分秋色也證明了歐羅巴的未來還是可以期待的。

這裡給出的歐羅巴高難評價是①無回合限制的三次無敵非常優秀，弱體耐性提升對部分高難也很有效果，③暴擊輸出也很優秀，④作為傳統型的光炮輸出，似乎沒有什麼針對上場的場合。

周回的評價是，有充能效果並且還有提升輸出的buff非常給力，防禦力下降的效果也能提供一定的輸出輔助能力。

整體來說就是中規中矩的強啦，也沒有是花里胡哨的東西，有防禦有充能有傷害提升，高難周回都能用得上，也基本沒有什麼使用難度，還是非常優秀的啦。

三星騎階從者曼迪卡爾多評分暫定為C~B級，算是平均水平偏下的階段，不過單體的評分還有C、D、E級，所以如果能達到B級也算是比較有優點的三星從者。

appmedia給出的評價是：通過自己的即死來達到交換隊員的能力非常不錯，寶具的輸出不錯，不過因為技能使用有HP限制，需要半血以下才能使用二技能的普攻即死，所以作為輔助使用起來有點不方便，雖然有高幅度的輸出buff，但是一旦使用的話自己就會死亡。

周回沒有太大優點，畢竟沒有自充，二技能又不能隨便用。

整體來說就是二技能的兩面性導致曼迪卡爾多的評分較一般，作為輸出的話，二技能能夠帶來極高的輸出，而一旦使用的話就會死亡，作為輔助的話，二技能可以讓自身直接退場，但是又有HP的限制，這兩個限制要是能解決一個話，相信性能就會高出很多了。
