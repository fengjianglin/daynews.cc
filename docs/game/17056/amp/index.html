DNF：12.14體驗服更新，卡片限制交易1次實裝，但並不全是好事！

以下為具體的內容詳情：

等了這麼久DNF體驗服終於是更新開服了，而此次更新的主要是有關1.14春節版本的相關內容，但不包含像氪金禮包，新活動這些，主要是一些比如職業改版（修羅、劍宗加強了），新副本上線，舊副本改版等等，而這次小寶主要來介紹的是其中一個很多玩家都非常期待的體驗服更新，那就是有關卡片/寶珠限制交易1次的功能，終於是提前實裝到了體驗服當中。

沒錯，在今天12.14體驗服更新之後，玩家經過實測早在韓服上線不知道多長時間的，卡片寶珠限制交易1次的功能，終於是提前先實裝到了體驗服當中，後續預計將在1.14春節版本正式實裝到正式服的遊戲當中。

而有關卡片寶珠限制交易1次的功能，相信就不用了小寶多介紹了吧，顧名思義就是玩家無論是從拍賣行上，還是以郵件的方式等，任何可交易的卡片/寶珠，只要進行過一次交易之後立馬就會變成賬號綁定，也就是說任何玩家（指黑商）都不要再想著去惡意囤貨了。

因此，後續只要等到1.14春節版本更新這個限制交易1次的功能一上線，由於沒有黑商們的參與，玩家們或可以用更低的價格到拍賣行買到你想要的畢業寶珠，畢業卡片。

玩家們到時候可以用更低的價格買到畢業寶珠/卡片，這固然是好事，但要是換一種思路的話也不全是好事。

至於原因很簡單，那就是你們後續如果有幸從各種地圖中再掉落到，或者各種副本的翻牌中再翻到了可交易的卡片的話，因為僅限交易1次的原因，你們能換來的收益相比可能會更低。所以小寶在上面才會說後續這個功能的實裝並不全是好事，但儘管如此，對比起來小寶覺得利絕對是大於弊的，而你們覺得呢？
