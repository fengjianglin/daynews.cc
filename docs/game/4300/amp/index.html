TOP5 不知火舞

不知火舞是上線已久的老英雄了，在經歷了幾次加強後慢慢崛起，終於在S16登上了排行榜。火舞憑藉其足夠多的控制和突進技能，在前期的gank抓人方面比起其他法師具有得天獨厚的優勢，在這個推塔收益上調、前期節奏緊湊的版本，這樣的中單再合適不過了。後期火舞的傷害更是爆炸，對前排後排均有巨大的威脅，推薦大家拿來上分。

銘文推薦：

藍色10個輪迴，綠色10個心眼，紅色10個夢魘。

出裝推薦：

噬神之書，抵抗之靴，迴響之杖，痛苦面具，博學者之怒，虛無法杖。

玩法技巧：

不知火舞前期清線慢，線權弱，需要叫上輔助來幫忙清線，大喬鬼谷子這種清線快的輔助是最好的。清完線馬上去邊路，找機會配合己方線上英雄擊殺對面邊路並拔塔。中期裝備基本成型後多嘗試進對面野區蹲草叢，蹲到一波就是大賺，就是一個節奏點。後期敵方傷害起來了，火舞身板太脆不宜貿然進場，通過2技能邊緣輸出並用被動調整位置，找到機會1閃秒掉對面c位結束比賽。

TOP4 諸葛亮

諸葛亮相信是不少中單法王們喜愛的英雄了，又帥氣又能carry，帶妹上分首選。諸葛亮是一個出道即巔峰的英雄，炮台版本時諸葛亮陷入低迷，在經歷了幾次加強，迎來了脆皮版本之後，諸葛亮也逐漸回到了人們的視野之中。現在排位中也經常能看到諸葛亮的身影。諸葛亮擁有法師中最獨特的收割機制，使得他一旦進入順風，遍擁有無人能擋的壓制力，是新賽季上分的好手。

銘文推薦：

藍色10個狩獵，綠色10個心眼，紅色10個夢魘。

出裝推薦：

抵抗之靴，噬神之書，痛苦面具，迴響之杖，輝月，博學者之怒。

玩法技巧：

一級點二技能三個2出被動快速清線，然後偷豬或者抓邊路。然後每波兵線清玩跟隨打野抓人。諸葛亮缺少控制，前期傷害沒有那麼高，所以切記前期一定要跟團，通過頻繁抓人、蹭助攻、拿人頭建立起自己的經濟，經濟高了就能為所欲為。後期打團注意自身站位不能靠前，多用一技能消耗，二技能躲開敵方技能，或者找小兵野怪刷出被動後上去談彈一下。有機會了就大招收割，可以二技能配合閃現走位防止大招被擋。

TOP3 張良

張良自從改版後就一直是一個強度很高的法師，俗話說得好「你強任你強，東皇加張良」。張良作為一個節奏型法師，大招強大的單體控制，二技能高額的單體傷害使得他在前期起節奏方面的能力十分強大，而且單殺能力也足。對面任何一個人只要稍微不小心，就會被張良大招捏住按到死，從而被滾雪球，陷入不利的境地。

銘文推薦：

藍色10個狩獵，綠色10個憐憫，紅色10個夢魘

出裝推薦：

疾步之靴，痛苦面具，迴響之杖，輝月，博學者之怒，魔女斗篷

玩法技巧：

張良前期雖強，但後期疲軟，所以一定要在前期建立起優勢把對面壓死。出疾步之靴是為了加強前期的遊走支援能力，張良的二技能清線很快，並且能很好地干擾敵方中單清線。兵線一到只需放兩個2技能，然後掉頭就走，不浪費一點時間。讓己方邊路引誘一下敵方，對面上鉤後閃現上去出其不意大招控住擊殺。不斷重複這樣的過程，破掉三路外塔，節奏就起來了

TOP2 上官婉兒

上官婉兒也是一個出道即巔峰的英雄，雖經歷幾次降溫，仍然熱度不減。因為上官婉兒有極其突出的抓人、切後、遊走、突進能力，並且還有不可選中機制，所以這個版本她依然是十分強大的。大手子手裡的上官婉兒各種閃現輝月、剎車普攻，秀得人頭皮發麻，不失為一個值得好好練習的英雄

銘文推薦：

藍色10個狩獵，綠色10個心眼，紅色10個夢魘

出裝推薦：

冷靜之靴，迴響之杖，痛苦面具，博學者之怒，虛無法杖，輝月

玩法技巧：

婉兒清兵很快，21技能連發就能收掉一波兵線。四級前比較弱勢，猥瑣消耗。四級後就進入了我們的強勢期，跟著打野去邊路，直接2331333連招沖向敵方，對面不死也殘，防禦塔順勢拔下。後期不宜貿然刷大進場，而是21邊緣消耗，婉兒的21連招傷害極高，可以直接秒殺滿血脆皮。大招用來收割，等對面控制交完後再肆無忌憚地進場

TOP1 嫦娥

嫦娥這賽季在巔峰賽上十分強勢，是ban位常客。作為一名法坦，嫦娥在具有極高肉度的同時又有著不俗的輸出，而且前期反野能力強，容易配合隊友把對面打野打自閉。拿了藍的嫦娥是爸爸級別的英雄，刺客切不死，戰士也切不死，還能反殺。除了伽羅之外沒有什麼天敵了，如果放出來建議一搶

銘文推薦：

藍色5個狩獵5個貪婪，綠色心眼，紅色夢魘

出裝推薦：

秘法之靴，時之預言，熾熱支配者，極寒風暴，博學者之怒，虛無法杖

玩法技巧：

嫦娥前期反野能力很強，清線也快。點二技能迅速清完第一波兵線後馬上去偷豬，如果對面紅開可以嘗試偷藍，最不濟也能反掉蜥蜴。嫦娥的重心要放在野區，利用她打野快的特點，留意對面打野的小失誤，一步一步蠶食野區。如果隊友前期強勢則可以光明正大反野。嫦娥最好配合阿軻、裴擒虎等不要藍的打野，這樣可以把藍給自己。有了藍的嫦娥戰鬥力大大加強，幾個人在一起都不一定殺得死。四級的嫦娥擁有一個大秒殺任何人的實力，非常恐怖。要利用好前期的強勢，以對面打野為切入點，擴大自己的優勢，從而成為勝勢
