俠客風雲傳心心念的女俠客覺醒搭配攻略

以下內容來自俠客風雲傳公眾號

最近呢~

遊戲又迎來一次大更新，相信很多大俠，對此次更新的內容都是非常期待的，那麼現在就由我帶大家來分析一下此次的版本更新，供大俠們作為參考！

此次更新有深受玩家喜愛的2名女性俠客，領悟了覺醒~相信這2名女神級別的俠客絕學也不會遠啦！

東方教主

1

東方教主，在橙色三練武后，可以選一個被動技能進行使用，分別是勾魂攝魄和磨牙吮血，目前來看如果隊伍中已經存在掃地（奶媽）類型的俠客的話，建議大家可以使用勾魂攝魄類型的奧義。

磨牙吮血：攻擊流血目標有30%吸血效果。

勾魂攝魄：攻擊異性時有25%幾率造成異常狀態（沉默、冰凍、混亂、眩暈中的一種狀態）。

目前遊戲內性質來說，東方作為女性俠客，而遊戲內男性俠客上場率又是比較高的，此被動雖然幾率不高，但猜測應該直線類型招式也可以同樣觸發多次，並且分開計算，如果是這樣的話東方自身就會擁有一種控制效果，對於目前版本來說控制住敵方關鍵俠客是很容易扭轉戰局的呢~

如果想要使用磨牙允血，增加自身的存活效果也是可以的呢，攻擊流血狀態的俠客增加30%的吸血效果，也是可以增加自身續航的哈~

建議佩戴招式：

1：纏絲八爪或百足蜈鉤，纏絲八爪本身就擁有較低的幾率可以進行控制，而百足蜈鉤則是配合東方教主的被動效果攻擊敵方一排俠客來起到控制的效果呢~

2：如來九印或催魂腐心掌，前者也是配合東方教主打出一排傷害起到控制作用，後者則是一個被大家忽視的一個爆炸系傷害招式，若有使用的玩家都會知道這類招式的傷害哈，是非常恐怖的呢！

建議佩戴心法：

心法方面目前來看，沒有東方教主量身定做的，可以攜帶至陰打斬殺，也可以攜帶素女心法增加BUFF等，可以根據自身陣容配置來進行調整哈。

黃衣少女

1

黃衣少女，在橙色三練武后，可以選一個被動技能進行使用，分別是花飛蝶舞和戮力同心，此2個被動，效果都比較類似，都是免傷類型效果，大俠們完全可以根據自身隊伍的實際情況來進行選擇呢~

花飛蝶舞：女性俠客獲得免傷7%的效果，持續整場戰鬥。

勠力同心：自身當前行所有俠客（包括自身）獲得免傷10%的效果。

建議佩戴招式：

1：情義七劍，黃衣少女本來就是靠奧義增加場上內勁俠客內力來進行使用的，情義七劍擁有回復怒氣的效果，還擁有為前後方俠客增傷的效果，較為適合黃衣少女進行使用哈~

2：素女劍法，或者石破天驚拳！既然是一個BUFF俠客，那麼就讓BUFF疊的更高一點吧，黃衣少女不是傷害類型俠客，給其攜帶增加BUFF的招式是首選哈~

建議佩戴心法：

無相心法，素女心法都是比較適合黃衣進行使用的呢，因為心法現在較多的關係，如果覺醒後我上場黃衣少女的話我是會選擇素女心法進行使用~

素女心法4階後擁有開場幾率給全隊增加免控光環的效果，在戰局中如果觸發到關鍵位置還是可以扭轉戰局的~

二位女神俠客的覺醒就介紹到這了~感謝大家的收看哈~
