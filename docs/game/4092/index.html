魔獸世界的沒落，鬼蟹為何要背鍋？看看他設計的這些機制就知道了

2019-12-10 0條評論 18次閱讀 0人點贊

摘要

作為一個魔獸世界的老玩家，淺談一下wow給我的幾次衝擊，第一次是開放小戰場，野外混戰急劇下降，負面影響刷分隊形成。第二次是開放25人副本，副本簡單化，團隊活動下降。

作為一個魔獸世界的老玩家，淺談一下wow給我的幾次衝擊，第一次是開放小戰場，野外混戰急劇下降，負面影響刷分隊形成。第二次是開放25人副本，副本簡單化，團隊活動下降。第三次是開放5人副本難度選擇，這個是最失敗的，失去了最吸引玩家的地方，但是wow在玩家心裡還是最成功的優先，沒有之一。

鬼蟹在魔獸如日中天的時候接手了魔獸，隨後魔獸一天不如一天。他又去了如日中天的lol，然後LOL隨後也是江河日下。這本事也是厲害，做壞一款遊戲不算什麼，難得的持之以恆的去毀掉一款又一款經典游，美其名曰平衡，平衡到後面就只要有一個職業就夠了。玩過帝國時代的就知道鬼蟹所謂平衡的便秘感受，法蘭克號稱騎兵民族血比較多，但是沒有血統升級，結果血量比別的騎兵民族也就多了12滴血。

就是那種看起來種族有優勢實際上沒什麼用的所謂平衡，放到wow里就是各種數值趨同，職業大一統，那優先還分十幾個職業幹嘛，只要三個坦奶輸出不就完了嗎，論可玩性和平衡性還不如仙境傳說。魔獸世界沒落，絕對是鬼蟹背鍋，看他改的些什麼玩意。本來獵人戰場每把前三，他把法力值改成集中值後，以前那麼豐富的技能全沒了，只能用一招集中射擊，大招都沒用了，因為性價比太差。

打戰場被隨便一個爛法師開個罩子用不費藍的寒冰指吊打，後面獵人連續加強二十多次，技能傷害不斷成倍，團本還是dps墊底。我就想問到底測試過沒有，真要測試過會看不出來這麼大的問題？本來豐富多彩的技能全部沒用，傷害可以忽略不計，這是人做的事嗎？很多玩家就是從那時候afk的，一個遊戲從所有技能都有用武之地，變成了所有人物都只有兩三個技能能用，還看不到任何改回來的希望，怎麼玩得下去。

LOL也是被鬼蟹毀了，也是打著平衡遊戲的口號，把以前英雄改的面目全非，讓很多老玩家，都不想玩。魔獸世界最好的記憶是歷盡千辛萬苦救出了將軍，陪他一起走進暴風城，所有的NPC都向將軍跪下，只至皇宮勇敢地揭露了黑龍的陰謀。那個時代魔獸的樂趣在於任務中尋找故事，WLK之後就變味了。
