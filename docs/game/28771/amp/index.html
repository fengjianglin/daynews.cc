魔獸世界：暗黑團隊接手後副本就失去了新意？懷念60不僅僅是情懷

天天要聞
2019-12-21

以前的魔獸世界感覺就是在一個大陸上探險、遊歷、感受它的文化。人與人之間的互動也很有愛，練到45沒錢買坐騎，工會的朋友湊錢買了記個小本本，一筆一筆都還得清清楚楚。走過廣場看見不認識的魔法師，打聲招呼要點麵包和水，人家真的會停下來給你做。去到一個新的地方看見第一次看到的風景真的會驚嘆，會好好欣賞。但是後面的無盡下本，每天排得滿滿當當的事情好像每天都在魔獸里上班一樣真的累。

60年代的時候用UC70年代用YY，那個時候的團還是非常和諧的，活動的時候有說有笑除了開荒的時候以外。TBC初期開荒格魯爾過了團長一人給了一張點卡，我們團還有一對就是因為在團里認識的，現在結婚了。魔獸世界公測第一天等到晚上12點開服，當年為了釣個魚可以約個隊友跑遍整個地圖。現在的小朋友已經體會不到魔獸真正的快樂了，社會太浮躁只會刷刷刷，老玩家現在也沒有足夠的時間來玩遊戲了。

老版本魔獸並非沒有劇情，相反正是因為劇情完全鋪展開了，所以不那麼顯眼，並不像後面的版本那樣，總是有一個大主線（實則是製作方面越來越應付）。舊版的劇情體現在每一個主線支線的任務里體現在不同副本中，甚至體現在創建角色界面的種族說明裡，當然自從開放了90級魔獸就已經在走向死亡了。

懷念60年代不僅僅是情懷還有成就感和滿足感，越困難越有挑戰的價值！第一裝備獲得困難，60年代可以為了一個T0裝備瘋狂刷3大。第二角色練級困難，特別是聖騎、牧師、德魯伊薩滿。60以後魔獸世界就被暗黑團隊接手了，之後就是刷刷刷不停地刷刷刷，毫無新意和誠意。以前大小副本講究人與人之間配合，特別是5人本每個人各盡其職，樂趣無窮現在副本胡亂瞎打也行，根本沒有當年的味道。

現在的魔獸根本不想玩，就跟暗黑3一樣實在提不起興趣，還是喜歡舊版魔獸喜歡節奏慢點享受玩遊戲的過程。下班有時間就做做任務，升多少是多少沒時間就不上了，魔獸14年了該有的都有了，現在再回到最初就好好體驗一下過程。
