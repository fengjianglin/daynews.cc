大話西遊2玩家換五個神兵 想得個藕絲步雲履也太難了

2019-12-13 347條評論 4,402次閱讀 2108人點贊

​對於阿宅而言，玩遊戲最激動的時刻，莫過於被幾率了。但是幾率這事，真的都是靠臉，什麼時候來，來個大的小的，這真都說不好。有時候眼巴巴盯著屏幕目不轉睛盼著幾率，幾率大神卻不一定眷顧。有時候玩著手機看著電視沒想著被幾率，幾率大神缺偏偏在這個時候光顧。不過如果是用固定積分去換取獎勵，心跳可能會更快了。尤其是換神兵，是藕絲還是墊底，那真是一瞬間的事。

來看看今天這位五開玩家，地宮固定積分換神兵，運氣如何吧。

攢了一年多的積分了，日日夜夜辛辛苦苦五開做五倍得牌子，每周不落的殺地宮，無非就是為了這一刻的到來。終於，激動人心的時刻來了！看這樣子，藕絲是沒戲了，不過這兩個索魂幡和一把枯骨刀里，會不會有雙回呢？要是雙回的話，也是不錯的。

來看看第一個索魂幡，是男鬼用的沒錯，只可惜屬性讓人難堪，絕對是個墊底的垃圾神兵了，哎，不過別灰心，看看下一個什麼情況。

第二個索魂幡，男魔反擊的，比墊底稍微好那麼一點點，不過估計也是不太好出手，只能當做墊底出售了。

第三個神兵，男角色的反震帽子，也是墊底一枚。看看下一個會不會是雙回吧。

第四個神兵，最垃圾的反擊附毒，這下好了，最後的希望也沒有了，不是雙回。

第五個神兵，新出龍族萌妹的神兵。

看來要去哭一會了，基本上是血虧了，期待了一年多的時刻就這樣夢想破滅了。算了，人生本來就不該有夢想，繼續做一條沒有夢想的鹹魚吧。
