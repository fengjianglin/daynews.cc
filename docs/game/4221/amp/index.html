英雄聯盟這款遊戲現在已經很成熟了，各個位置都分工明確，英雄的定位也是很完善，雖然我們在平時的遊戲中可能看到很多出人意料的東西，比如蛇女下路，石頭人打野，奶媽上單等，但是職業比賽一般來說都是很規範的，就算出現意外分路也是因為Counter或者可行性很高的，但是職業賽場上出現過唯一一個五個位置都打過的英雄，你知道是哪個么？接下來就告訴大家。

這個英雄就是傑斯，可能很多人在聽到這個英雄的時候感覺很對，但是又感覺哪裡不對，總感覺哪裡怪怪的，沒錯，你感覺奇怪的地方就是打野位置，因為幾部絕大多數人都不知道傑斯竟然在職業比賽中上過打野位置，在傑斯打野這場比賽中還有一個很有名隊伍，它就是ROX。

傑斯在各個位置最出名的先和大家說一下，上單Theshy傑斯一炮四個PDD，之後就是黃金左手中單傑斯讓整個LPL都驚掉下巴，在S5總決賽上，當時的EDG下路羊駝就用傑斯打過AD，當時打的還是老對手FNC，不過正是這場比賽，也讓羊駝多了一個稱號，關於傳送的，你們慢慢想吧，輔助位置玩傑斯的就是MATA，一個不僅對遊戲理解極其厲害而且又能不斷發掘下路打法的選手。

打野位置傑斯是由當時韓國戰隊EEW的開發的，當時對戰的還是ROX，至於戰果就有點記不清了，不過不得不說LCK選手的開發能力確實強於我們賽區，雖然現在我們是第一賽區，但是論起對遊戲的理解還是不如他們，就連很多我們國服的套路都是從韓服傳過來的，這就是事實，不服氣不行，就比如前一段沒改版的偷錢女槍，就是韓服人才根據版本開發出來的，高中低端局都適用，還有火了很久的魔宗卡莎，雖然前期比較弱，但是後期輸出能力提升一大截，很多我們之前玩的套路都是從韓服傳過來的。

雖然比起開發能力我們可能有所不如，但是我們卻戰勝了他們成為第一賽區，就像現在社會中一種很常見的作用，老闆負責把所有有能力的員工聚在一起，讓他們為自己效力，我們賽區在電競中也扮演著這樣的角色，這可能也是我們厲害之處吧，不知道你們怎麼看？
