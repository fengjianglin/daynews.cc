VG迎來SKT冠軍教頭？蕉太狼、IBOY加盟，水友：崛起了？

天天要聞
2019-12-19

最近關於LPL的大事可能就是轉會期的結束，隨著德杯的即將開始，LPL的轉會期也步入了尾聲各支俱樂部也紛紛進行了交易。除了我們之前關注的RNG、與SN紛紛爆出消息之外，也有阿水這樣至今成謎的大額交易。而在這些交易中有一隻不起眼的隊伍，也進行了不小的操作，而經歷了補強後的它也從去年的墊底戰隊變成了一支勁旅。它就是VG作為曾經的”弱隊”，我們可有些期待它在S10的表現。

比起原本的比賽陣容，VG算是進行了不小的調整。首先是教練方面的人選，可以說教練對於俱樂部就如同將軍對於士兵，頭羊對於羊群。kkOma就是VG這支俱樂部選擇的靈魂，作為曾經供職於SKT這支傳奇戰隊。而扣馬教練的履歷也非常豐富，長達七年的帶隊經歷。在他的執教生涯中曾經兩個季中賽冠軍、三個全球冠軍、七個賽區冠軍。可以說他就是LOL教練界的Faker，而這次離開了SKT的扣馬教練想必也是想尋找新的挑戰，於是這樣的他來到LPL的VG也不令人意外，我們也期待他能夠續寫他傳奇教練的傳奇。

而除了教練之外，對於俱樂部最後的成績影響更多的還有選手。在選手方面VG加盟了天才AD選手Iboy，可能有部分玩家並不太清楚這位選手。但曾經Iboy可是與這次轉會期票王阿水並列為天才ADC選手，而且這次VG的選人更多是偏向新人並未選擇經驗豐富的老將。想必在經驗豐富的扣馬教練調教下，這次的VG也能帶給我們不一樣的表現，而作為LCK的優勢運營方法，想必扣馬教練也會很好的培養這些新人。除了有利於提高VG的成績外，也有助於提高整體LPL的戰術水平。

而除了AD位置的新人，還有一名我們非常熟悉的選手也加入了VG，他就是曾經的IG替補蕉太狼。作為IG的二隊中單，蕉太狼在主力隊員Rookie不能出戰的情況下扛起了IG中單的大旗。作為新人的他可以說臨危不懼迎戰強敵，為LPL的觀眾們留下了深刻的印象，這讓我們記住了這個酷似動漫中蕉太狼形象的中單。對於他離開LPL，我們也是感到惋惜，但在IG擁有肉雞的情況下也無法讓蕉太狼得到上場的機會。

而這次加盟VG對於蕉太狼來說是他在LPL真正的起點，到底能不能在這個名利場里留下屬於他自己的名字還要看他在VG的具體表現。雖然說比起其他的選手蕉太狼算是有了不錯的起點，但電子競技最終一切還是要靠實力講話，只有好的成績才是一個選手說話的底氣，也希望VG能夠越來越好。真正符合粉絲們的期待，成長為LPL的一支”銀河戰艦”，在電子競技的賽場上閃耀起飛。
