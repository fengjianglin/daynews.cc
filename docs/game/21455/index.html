造夢西遊外傳三周年慶版本12月26日更新上線！好禮驚喜不斷

2019-12-20 311條評論 3,136次閱讀 917人點贊

要說最近有什麼好事，除了漸漸來臨的元旦假期，還有造夢西遊外傳三周年慶版本也即將更新上線啦！說到造夢西遊，小夥伴們大概會想到一身近戰武藝超群的悟空，遠程高傷害的三藏法師，防禦超群的八戒！《造夢西遊外傳（三周年慶版）》於19年12月26日更新上線！

造夢西遊外傳講述的六耳獼猴篡取真經，悟空藉助時光之舟穿越時空阻止劫難的故事。遊戲採用了全新的MOBA玩法，支持多人團隊競技！一同開黑展示操作！這次造夢西遊外傳三周年慶更是給大家帶來了不少福利和全新內容！（本次曝光內容以26號實際上線為準）

▋1.AA英雄平等王上線

平等王是一個擁有快速位移，並帶有持續毒傷的英雄，在造夢系列當中，平等王的原型是蜈蚣，具有能在地下穿梭的能力，而進入造夢西遊外傳的世界後，平等王擁有了人形身體，相信在遊戲中一定會有精彩展現！

▋2.全新玩法-世界BOSS挑戰

不好啦！不好啦！世界BOSS揮舞著小拳拳它過來啦，新玩法世界BOSS挑戰也將隨著周年慶的來臨一併開啟，此次挑戰的這隻世界BOSS是鴻蒙之時形成的凶獸「吞天饕餮」每人有三次的挑戰機會，擊殺饕餮後參與戰鬥的小夥伴會獲得大量獎勵，即使沒有打敗B，只要參與世界BOSS也會獲得禮包獎勵！

▋3.周年慶狂歡眾多福利在等你

周年慶的開啟不僅讓大家擁有了免費獲取平等王的方式，而且還有送周年稱號，英雄限免，掉落翻倍，以及大量禮券贈送等福利哦！期待遊戲更新，大家一起開啟幸運轉盤吧！

▋4.雙喜臨門！周年慶+聖誕活動

12月26日不單單是關於周年慶的更新，還蹭上了聖誕節的車，小夥伴們在遊戲更新後可以參加聖誕活動，獲得聖誕麋鹿獎勵！騎著麋鹿去逛街！你就是街上最靚的仔！

總的來說造夢西遊外傳三周年的慶典不單單是給玩家贈送福利，開啟新玩法，更是感謝小夥伴們一直的陪伴，造夢西遊外傳生日快樂！(❁´◡`❁)*✲ﾟ*快快點擊下方鏈接預約吧~
