新版“天魔繚亂”來了！玩家感動落淚，鳳求凰優化也將完成

哈嘍，大家好！我是九零小哥，私信「地獄火」，即可免費領取皮膚。在王者榮耀這款遊戲中呂布是一位出場率非常高的英雄，尤其是近期版本，這位英雄憑藉強大的真實傷害成為了上單霸主。對於這樣一位英雄天美自然會很重視，先後為他上線了多款皮膚，品質最高的皮膚就是「天魔繚亂」了。之前小編給大家講過這款皮膚即將優化，而在最近新天魔繚亂的海報曝光了！那麼這次會是什麼效果呢？下面我們來一起看看吧。

首先這次優化有兩點目標，1.對局外展示效果全面升級。2.將天魔繚亂的技能特效和手感進行改進。現在天美已經向玩家們展示出了新版天魔繚亂的全新造型，從這個海報中我們可以看出這個版本的天魔繚亂更加成熟，魔化的感覺更強了，估計買了這款皮膚的玩家一定開心死了。

皮膚優化完成後很有可能會返場，沒入手的小夥伴一定要把握住機會。看到這個模型後還有玩家表示：如果手感優化能達到聖誕那款皮膚的手感就完美了。其實還有兩款皮膚即將優化完成，那就是鳳求凰和武陵仙君，這兩個皮膚優化幅度不大，期待度也沒有天魔繚亂的高。

不得不說這次優化還是很值得期待的，希望天美可以多上線幾款天魔繚亂這樣的皮膚，畢竟特效太帥了，玩起來也很有感覺。
