魔獸世界8.3版本之前，必做的10件事，讓你贏在起跑線

天天要聞
2019-12-18

玩遊戲找熱點，點擊關注不迷路

最近關於魔獸世界8.3版本的消息真的是越來越多了，暴雪也公布了大致的上線時間會在2020年初，眼看著2019年也沒剩幾天就要過去了，小編總結了一份清單，這10件事都是在魔獸世界8.3版本上線之前必須要做的，而且難度不高，助你贏在起跑線上。

一、成就

有些成就正在魔獸世界8.3版本《恩佐斯的幻象》中將會絕版，所以不要忘了在英雄或史詩的難度中擊敗艾薩拉女王，並在版本更新前打完所有10層或15層大秘境。

隨著希爾瓦娜斯即將拋棄部落，她已經不在奧格瑞瑪了，所以成就《黑暗女王之死：殺死奧格瑞瑪的希爾瓦娜斯·風行者》也即將絕版了，想去挑戰的小夥伴要抓緊時間了。

二、練小號

科拉克的復仇是一個非常有效率的讓你的小號迅速達到60級以上的方式。在15周年慶祝活動中，明年1月8號結束之前，各位小夥伴都是可以獲得死亡之翼這個坐騎的。一定要記住活動的截止日期，過了這個村就沒這個店了。

三、鎖定聲望到19999

艾澤拉斯之心已經可以升到75級了，各位小夥伴可以把所有陣營的聲望鎖定在19999/10000，從而在魔獸世界8.3版本上線之後節省40000神器能量。

四、完成所有新同盟種族的解鎖條件

機械侏儒和狐人是一對全新的同盟種族，將會在魔獸世界8.3版中對玩家們開放，但是要想能第一時間玩上新同盟種族，各位小夥伴必須完成以下這些成就：

  • 沃頓奈聲望崇拜、獲得成就「深埋於沙的秘密」
  • 銹栓抵抗軍的聲望崇拜、獲得成就「麥卡貢的威脅」

五、狂攢海員的達布隆幣

各位小夥伴在魔獸世界8.3版本上線之前，千萬不要隨便亂花的你達布隆幣。暴雪將增加新的達布隆幣商人，會出售海島探險和入侵的寶箱，包括坐騎，戰鬥寵物，幻化物品等。

六、地精和狼人聲望崇拜

所有滿級的地精和狼人玩家，如果跟各自陣營的聲望達到崇拜以後就可以開啟全新的傳承護甲任務線，來獲取各自的傳承護甲。尤其是狼人的傳承護甲確實有點帥呀，地精的就不提了。

七、攢材料

在魔獸世界8.3版本中居然沒有新的配方，這一點確實非常奇怪，所以這也意味著各位小夥伴可以從現在開始就提前大量的伐木草藥、礦石、皮革等，當新版本來臨之後為自己積累一定的金幣，然後在相時而動。

八、收集各種橙色武器

在魔獸世界8.3版本暴雪終於開放了橙色武器的幻化，所以各種橙色武器還不全的小夥伴可以利用這段時間去收集一下，具體可幻化的橙色武器如下：

  • 薩弗拉斯,炎魔拉格納羅斯之手
  • 雷霆之怒，逐風者的祝福之劍
  • 埃提耶什，守護者的傳說之杖
  • 埃辛諾斯戰刃/埃辛諾斯戰刃
  • 索利達爾，群星之怒
  • 瓦蘭奈爾，遠古王者之錘
  • 影之哀傷
  • 巨龍之怒，泰蕾苟薩的寄魂杖
  • 戈拉德，龍王之暮/提里奧什，遠古噩夢

不過要想入手這些橙武也是需要一定運氣成分的，所以不要放棄任何一次嘗試，抓緊時間刷起來吧。

九、攢銹栓抵抗軍徽章

在新的8.3補丁中，麥卡貢的銹栓抵抗軍聲望追趕物品——銹栓抵抗軍徽章將無法作用於抵抗軍崇拜後的角色，這可能是暴雪對於高端公會們為積攢下一級的神器能量而屯積零件箱的應對措施。不過這意味著它仍然可以用於小號在麥卡貢聲望上的追趕上，但不能用於通過聲望補給包將零件箱轉換為神器能量。

十、攢棱彩法力珍珠

如果各位小夥伴的大號在納沙塔爾的聲望達到崇敬，就可以使用這些珍珠在NPC處購買底棲系列裝備代幣，代幣是賬號綁定的，因此可以郵寄給你的小號。

以上就是在各位小夥伴等待魔獸世界8.3版本到來之前，可以做的10件比較有意義的事情，做了之後你一定會跟小編一樣發現是真香啊。
