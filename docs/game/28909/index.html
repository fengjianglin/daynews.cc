航海王燃燒意志：頂上戰爭，主線難點全回顧，關卡強度有所降低

2019-12-21 1,979條評論 3,356次閱讀 253人點贊

諸君安好，霧夏菌報道。

隨著新版本新劇情的開放，之前頂上戰爭劇情的難度也有所削弱，之前沒有通關的玩家，可以再去嘗試一下，那麼這一回合就來看一下頂上戰爭的那些主要難點。

1、青雉

有一關的青雉，能夠元素化中斷，還可擁有一次復活，不久前也提到過，就是那個冰與火。，卡住了諸多玩家。在擁有白鬍子、紅髮等角色後，通關難度會降低不少。同時需要帶上一兩名奶媽，提供續航。

2、赤犬

赤犬在頂上戰爭要打很多次，第三波的赤犬是相對比較難的，護盾厚，傷害不低，同時低血量會有兩次召喚小兵的機制，所以同樣需要奶媽，保證續航。不過因為難度降低，赤犬的護盾被砍了不少，所以這個通關難度也就下來了。

3、黑鬍子

黑鬍子這裡，感覺難度沒有想像的那麼大，主要是需要破盾，這個護盾的厚度非常誇張，正常的一個斬斷無法破掉，但被破掉盾的黑鬍子，坦度並不高。而且黑鬍子的護盾並不是每回合刷新，所以破盾結束後，就可以集中火力輸出了。再一個帶上佳妮法，解除負面效果。

4、鬼蜘蛛

在黑鬍子之後，會有專門的鬼蜘蛛這一關，這裡也有人被卡住，這個主要是因為一個小兵提供的BUFF效果太猛了。而我們短時間又無法對鬼蜘蛛完成擊殺，所以不要浪費太多精力在鬼蜘蛛身上，需要優先擊殺小兵，將鬼蜘蛛留到最後處理即可。

5、三大將

三大將這裡有點難度，一定要嗑藥，黃猿是三人中相對最容易擊殺的。

用對位角色與黃猿對位，磕上一些命中藥丸，黃猿的閃避非常高，不嗑藥很難擊殺。嗑藥之後，對黃猿的擊殺會最容易。

再一個是赤犬和青雉，都有一次免死機會。青雉起初也會有中斷機制，但被擊殺後會喪失這一機制，之後非常容易擊殺。赤犬相對肉一點，其他沒什麼需要注意的。

6、其他

頂上戰爭後續會打好多遍赤犬，特別耗時間，再一個就算打了過去，新的劇情，難度依舊不低。當然，對於一些氪金玩家來說，可能覺得這次沒有之前難度高。但對於普通玩家來說，具有不小的挑戰性。

菌說

白鬍子在頂上戰爭依舊比較重要，中斷機制比傳統的肉坦要好用不少，所以還是要考慮入手的。其次製藥方面要多搞一些，對於輸出的提升非常高，以攻擊、暴擊優先。而且三大將這一關，還需要額外的命中，那麼命中藥的重要性就體現出來了，也要提前準備的。

​好了，這一回合到此結束，感謝閱讀，拜拜了您嘞~
