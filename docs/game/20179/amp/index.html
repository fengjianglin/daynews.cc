三國殺：如果地主是張春華，你拿什麼農民最好打呢(新堯三國殺)

天天要聞
2019-12-18

各位小夥伴們大家好,我是新堯.在三國殺中要想快速獲得招募令,鬥地主一定是你的必經之路.通過在鬥地主中獲勝獲得金票,然後用金票來換取招募令和雁翎甲.然而鬥地主中四害往往是讓玩家們頭疼的問題,常常出現在隊友中,讓玩家們苦不堪言.然而還是有許多玩家會選擇春哥當地主.

儘管小編個人認為春哥並不算太強.但是大家選擇她的目的,無非就是免疫賣血流武將,而且自身可以在殘血的時候瘋狂過牌.但是事實是農民也並沒有那麼傻,翻車的概率其實很大,那究竟是哪些農民武將常常讓有信仰的春哥翻車的呢?一起來看看吧

馬岱

遇到馬岱可謂是三國殺中難受級別最高的,儘管他可能會斷殺,但是有殺的他能把你直接砍到0血.他的殺才是最鋒利的刀,一旦命中,緊跟潛襲判定,少一點體力上限.本來春哥就是因為四點體力才可以讓傷勢的收益變得十分強大,但是遇到了他,可謂是一生宿敵,能直接把你砍自閉,如果還有一個保護型的隊友.獲勝的希望基本破滅了.所以遇到了他,開局一定要打出點優勢,不然後面就難熬了.

孫魯育

被她難為的可不止是春哥一人,只要是可以大量過牌爆發的人,例如劉焉.都被她給克製得死死的,止息讓你能出的牌不超過體力值,而且還不能出錦囊牌,一出就結束出牌階段.所以你會發現這樣的農民,可以很好的牽制住大部分地主,不過要是防禦牌不夠,也是很容易被人給解局的.面對春哥的時候,春哥如果賣血就不能夠出太多牌,不賣血又是個白板,求她的心理陰影面積.

馬良

小編鬥地主最煩遇到的農民,沒有之一,一牌二用.遇到開局必須刷樂來封他嘴.不過遇到這種情況到底應該先殺他隊友還是先殺馬良呢,這裡我們暫且不討論,可以在評論區下方留言討論.為什麼說馬良容易讓春哥翻車呢?難不成要給她牌?沒錯,就是要給她牌,但是牌可以給普通的殺來卡它牌,如果是可以直接算好傷害直接殺的話,給什麼牌都無所謂,也就是讓他的手牌已經成定局,不會有傷勢在來給她信仰過牌,這招真狠啊.

界黃蓋

估計要是再不放上來,就有點不給蓋子面子了.”相比公瑾,我還是更願意挨她的打”畢竟挨一次打就可以摸3張牌,比自身回合內苦肉划算多了.但是自身回合內該苦肉的還得苦肉,不然你總不能碰運氣打中她吧,小編覺得他倆73開,儘管看起來是蓋子天克她,但是春哥也不是沒有贏的希望的,因為苦肉不可能把把詐出桃子,很有可能只是一些廢牌,這樣慢慢的體力就被磨下去了,如果春哥手牌拿得好即使是閃不了他的紅殺,但是攢桃加上傷勢的過牌,還是有機會的.
