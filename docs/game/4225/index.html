9地獄火3召喚使2遊俠

下個版本，地獄火被削弱了，灼燒的時間從5秒下調到了4秒，傷害也下調了，但是阿木木的加入完全不減地獄火的威力！

看似只有9地獄火，這時我們要做的就是給牧魂人一個地獄火熔渣裝備，10地獄火效果更好！不僅如此，這樣還可以有3召喚使羈絆！

牧魂人：地獄火熔渣（鏟子+大棒）

運營方法

具體如何運營就不說了，很簡單，全力升人口，看到地獄火就拿。速度升8，第一時間找到核心隨從阿木木！其實不給牧魂人地獄火熔渣也是可以的，關鍵是要找到地獄火拉克絲。這樣才能組成9地獄火。阿木木的裝備也不一定要防禦裝備，給法強裝也很好，2星的阿木木要是有帽子、珠光護手，可以秒殺基本上所有的後排隨從！這套陣容看似很難，但是其實運氣好點，前期鎖住血，給後期提供更高的容錯空間，後期無敵的！
