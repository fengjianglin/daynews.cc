王者榮耀：年度最佳候選出爐，e星成最大贏家，AG4人入選，QG陪跑

天天要聞
2019-12-17

各位王者玩家大家好，我是小雨。又到了年末，2019年度的KPL年度最佳選手、最佳陣容以及最佳新秀候選人名單已經出爐，eStarPro各個位置都有人入選，AG超玩會也有4人入選，QG雖然也都有人入選，不過大概率和其他俱樂部一樣成為陪跑。

最佳選手貓神優勢巨大

年度最佳選手候選人分別是一諾、貓神、飛牛和渡劫。小雨認為貓神當選的幾率最大，畢竟他率領e星拿到了春季賽和世冠的冠軍，並且在秋季賽殺入四強。一諾雖然幫助AG奪得秋季賽冠軍，但是春季賽在BA並沒有進入季後賽，希望不大。全年無冠的飛牛和渡劫則毫無競爭力，只能陪跑。

最佳輔助和打野e星依然優勢大

最佳輔助方面首先排除770和星宇，沒有冠軍加成基本是不可能當選的。相比較無銘和愛思兩人，小雨更看好愛思，因為從AG和e星的正面交鋒中可以看出，愛思不僅會指揮，開團能力也要略強於無銘，不過無銘有兩個冠軍加成，結果難料。

最佳打野位置花海基本會無懸念當選，打野核的他不論從數據還是表現來看，都要優於另外三人。當然，如果是把藍領野單拿出來，那無疑會是Giao。

最佳中路、發育路還是e星的天下

年度最佳中路還是貓神無疑。老帥重回AG之前已經將近1年沒有打比賽了，而Mojo雖然本屆常規賽表現上佳，但未能率隊奪冠，整年成績更是無法與貓神相提並論，九尾更就不用說了。

最佳發育路是最有爭議的點，花海再次入圍，他跟一諾的競爭會非常激烈，不過一諾受上半年戰績影響，結果難料。刺痛和虔誠分別只獲得了一次亞軍，希望渺茫，基本陪跑。

最佳對抗路、新秀e星基本包攬

年度最佳對抗路基本是諾言的了。前3名選手雖然在實力上絕對不輸諾言，但是戰績上除了六點六有個冠軍，其餘兩個都是亞軍，很難和兩冠的諾言相比。

最佳新秀花海的當選基本也是板上釘釘的了。雖然清清和千世也非常厲害，不過相較於花海，明顯還是要低了一檔，久凡基本沒有任何希望。

從候選人名單可以看出，兩冠在手的e星必然成為最大贏家，唯一有競爭力的就是AG。​大家心目中的最佳分別是誰呢？歡迎留言告訴小雨。
