玩過這麼多年的某訊的網路遊戲之後，我總結出一個道理，那就是某訊的運營理念：竭澤而漁。

不知道大家有沒有發現，大部分某訊的遊戲開發團隊與運營團隊是兩個團隊，都是開發團隊將遊戲做好並吸引無數的玩家入駐之後，負責撈金的運營團隊悄然接手，然後各種氪金的道具慢慢的出現。

地下城與勇士。

以前的DNF有一種很神奇的魔力，一個動不動就讓你掉線，讓你制裁，讓你光頭，換線就黑屏的遊戲，但還是能讓你有玩下去的動力。

但是如今這遊戲除了氪金，還是氪金，很難找回當時的激情。所以很多人寧願脫坑，也不願意繼續征戰阿拉德大陸。

劍靈。

在劍靈還在內側階段的時候，他的飢餓營銷模式非常成功，但這成功並不是偶然，其主打的畫風、免費遊戲策略、角色辨識度度、可玩度非常高的遊戲性 ，都能決定其成為爆款、擁有超長的遊戲壽命，所以劍靈的開服初期所有伺服器天天排隊。

但是遊戲開服幾年之後，玩家們便發現這款遊戲刷的程度要遠遠重要於遊戲操作，每天的日常基本需要五個小時才能做完。

斗戰神。

這個是我玩的時間最長，也是覺得最可惜的一個遊戲。

作為一個借鑒於《悟空傳》的網路遊戲，遊戲IP絕對是非常吸引人的，其2.5D的遊戲畫面，這也使得當是斗戰神在內測之時就非常火熱。

但當量子團隊被更換之後，高階真言、高級幻甲、神級寶寶，各種氪金道具層出不窮，但唯獨遊戲劇情停滯不前。

伺服器也從當時的一個伺服器變成現在的二合一，五合一，四十二合一。

LOL。

為什麼要把LOL算上呢？因為LOL玩家的大量退游是大家都看得到的，本來去年的S8世界賽上，IG戰隊拿到了LPL的首個世界賽冠軍，這個不亞於核彈攻擊的事件，無疑會讓無數老玩家重回遊戲。

但是當時因為某項計劃失敗的運營方惱羞成怒，仍然強力退出KDA活動，直到被王校長懟了之後，才慢悠悠的掛上了奪冠公告，這不亞於一盆冰水迎面潑了那些回歸玩家一臉。
