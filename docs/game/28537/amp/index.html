王者榮耀：戰績最差玩家，對戰2027場次，網友：送他一張清零卡吧

天天要聞
2019-12-23

hello~大家好，我是榮耀王者小仙女。王者榮耀作為一款MOBA的競技遊戲，毫無疑問的說，是目前手游中人氣最高的，尤其是排位賽的段位競技模式，最容易引起玩家的熱血。對於一個有兩年遊戲經驗的玩家來說，只要玩個一千多把，基本上就能掌握英雄的熟練操作了，就算是上不了最強王者段位，能夠上個永恆鑽石段位，那也是輕輕鬆鬆的事情。而最近小仙女就發現一個比較奇葩的玩家，對戰了2027場，英雄勝率卻不忍直視，堪稱史上戰績最差青銅玩家，網友看到之後表示：天美送他一張戰績清零卡吧。

首先我們來看下，這位玩家帳號的個人主頁，資料上明顯的可以看出，目前還只是個秩序白銀段位，歷史最高段位也只是永恆鑽石，擁有54個英雄48款皮膚，對戰了2027場次。看到這裡小仙女就有所疑惑了，遊戲都已經在線了1357天，相當於是一個四年的老玩家了，可當前卻還是個青銅段位，總不能實力差的跟新手差不多吧。按道理上來說，一個資深的老玩家，不應該是這樣的水平，但是事實就是如此，這讓人很難以想像，這位玩家在遊戲中到底都經歷了些什麼，戰績竟然會差到這種程度。

看完個人主頁後，咱們接著再來看下他的常用英雄，而看到英雄勝率的那一刻，更是讓人匪夷所思。玩了2027場，曹操丶呂布丶關羽丶韓信等英雄，就佔了一百多場，其中韓信對戰了整整一百場，英雄勝率居然是百分之零，也就是說這位玩家使用的韓信英雄，在遊戲中是一把都沒有贏過，這樣的戰績真是前無古人後無來者。除了韓信之外，像曹操丶呂布丶關羽這幾個英雄的勝率，也是慘不忍睹。像這樣的戰績，稱他是史上戰績最差的青銅玩家也不為過把，哪怕是叫一個新手來，玩個幾百場，也不至於玩成這樣吧。

第一頁只是英雄出場次數比較多的英雄，往下翻其它的英雄勝率也好不到哪去，雖然在本賽季中，英雄的出場次數比較少，單加起來也有十幾場吧，甚至像百里守約，孫策和貂蟬，這麼強勢的三個英雄，勝率全部都是零。按照天美的計算方式來看，英雄場次計算的是所有匹配模式中，標準場次只計算排位賽，而玩家目前還是個白銀段位，作為一個老玩家，連這麼低端局都打不贏，是不是有點說不過去？看到這裡，說句實在話小仙女要是有戰績清零卡的話，都想送他一張，重新開始遊戲體驗之旅。
