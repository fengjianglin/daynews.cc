兩款輕鬆上頭的休閑手游推薦，一玩就停不下來

2019-12-15 230條評論 1,378次閱讀 110人點贊

【奧賽羅賽車】和【1比特的冒險】是我們推薦的第306和307款手游，感謝一直支持和關注我的小夥伴們。

兩款非常休閑的豎版手游，看起來很簡單，但是一玩起來就停不下，適合玩家們打發碎片時間，下面我們來具體看一下。

（一）【奧賽羅賽車】（onslot car）

輕度玩法的賽車手游，豎版，美術很簡單，上帝視角。

遊戲的操作可以說是很單一了，賽車在賽道上自動前進，玩家只能控制自己的賽車的油門，在直道上需要加速，在彎道上就必須減速，否則就會翻車，賽道上還有NPC賽車，玩家需要注意盡量不要撞車，前期可以不用追求極限速度，盡量穩定的闖關最為重要。

如果中途翻車，也沒關係，玩家還有一次續命的機會，這一點還是比較人性化的。前期主要玩法就是不停的闖關，玩家積攢的比賽場次和闖關進程，可以用來解鎖成就，並且可以獲取更好的賽車，穩定性更強，速度也更快，後面闖關也就更加輕鬆。

總的來說遊戲簡單且上頭，解鎖更好的車子和後面的特殊關卡挑戰也保證了遊戲的深度，遊戲中有中文翻譯，不過是機翻，不影響玩家體驗，很有趣的一款手游。

（二）【1比特的冒險】（OneBit Adventure）

畫風簡單復古的豎版rpg小遊戲，美術簡單但是不單調，操作起來也比較方便，玩家控制角色的方向，角色每次移動，地圖上其他的怪也會跟著移動，遇到怪的之後繼續按方向鍵就可以攻擊。

雖然說遊戲表面上體量是小了一些，但是玩法還是比較全的，首先角色有職業區分，還有裝備、天賦、裝備強化，道具等等系統，讓這款看似小遊戲突然變得耐玩了許多。

遊戲關卡也是隨機的，角色需要不停的前進，如果不小心走到屏幕底部，就算突然死亡而闖關失敗，關卡中還有隨機寶箱，隨機洞穴，強化NPC等等，有點roguelike 的設定。如果闖關的時候不小心掛掉了，只需要再等一段時間就可以重新開始了，可以選擇等級從中途開始。如果不想等待，那麼只能花鑽石復活，或者是再開一個存檔，玩其他的角色。鑽石需要氪金或者是日常任務獲取。

總體來看遊戲非常的有趣，雖然說沒有中文，但是遊戲不用翻譯也能玩的明白，不過剛開始玩會注意不到角色血量，經常不小心的掛掉，有點小鬱悶，感興趣的玩家可以去試試看。

兩款手游在ios商店都有下載，可以搜素英文，周末玩一下，輕鬆又愜意。
