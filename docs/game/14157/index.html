地下城手游，全民預約。勇士們快集結吧

2019-12-10 2,489條評論 4,382次閱讀 725人點贊

「你好呀，我叫賽利亞。需要幫忙嘛！」

相信很多DNF的老玩家都會對這兩句話有所懷念。

地下城與勇士手游將要發布了。

之前總是有許多一次又一次的小道消息，磨滅這老玩家的耐心。

地下城承載了多少玩家的童年與青春。

老一批玩家都已經結婚生子，有了自己的事業，自己的家庭。

不能將更多的時間放在遊戲上了。

現在地下城與勇士手游已經發出全民預約了。

讓給為老玩家們能再次拿起手中的寶劍去洛蘭深處，懲罰那些可惡的哥布林吧。

重燃十一年熱愛，歡迎回家，勇士

現在手游只公布了4個職業和轉職的分支

鬼劍士：狂戰士 鬼泣

魔法師：元素師 魔道學者

神槍手：漫遊槍手 槍炮手

格鬥家：氣功師 散打

相信後續會有更多的職業出現。

遊戲高度還原端游，完美繼承了端游獨特的打擊手感。走位 聚怪 連招！一擊必殺。

還額外的添加了手游專屬團本。由於DNF的招數技能太多，手游按鍵有限，將會採取一鍵連招、滑屏等操作方式。

端游的各大城鎮基本保留，艾爾文防線 格蘭之森 赫頓瑪爾 西海岸 天空之城

滿滿的回憶有沒有？也許手游的操作模式會和端游有很大區別，但是地下城這個遊戲IP就足以帶來很多老玩家的喜愛。唯一希望一點的就是不要有充值送神器，首沖送天空套什麼的，來破壞遊戲平衡。增加過多的影響遊戲平衡的氪金元素。

我相信地下城與勇士手游的出現會火爆各大遊戲平台。可以參照類似於地下城模式的時空獵人。

地下城的新老玩家們，你們希望公測時，出現什麼職業，保留端游的那些特色呢？
