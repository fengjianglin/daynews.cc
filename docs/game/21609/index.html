國內山寨廠商多無恥？女學生減壓做的遊戲都偷走，竟然要賣1000元

2019-12-19 667條評論 2,772次閱讀 1579人點贊

大家好，這裡是正驚遊戲，我是你們的正驚小弟。

最近小弟在Steam閑逛的時候，發現了一款「神奇」的遊戲！這遊戲叫做《瘋狂砸地鼠》，從表面上看來，是一款畫風還算不錯的打地鼠遊戲。小弟心想，沒事做玩點兒這種無腦遊戲也挺好的。

於是，小弟往下拉，被這價格震驚了！這遊戲竟然要1098元！而且這遊戲之前要513元，這特么還漲價了？！

這讓小弟的好奇心瞬間起來了，這3A大作都只有個400、500元的，這隻有18M大的打地鼠遊戲居然能賣1098元？要知道這價格能買11個大聖歸來普通版呢！難道能加智商嗎？能走向人生巔峰嗎？小弟決定當一回「傻逼」試一試！

遊戲一開始，連個彩蛋設置欄都沒有，音樂音效通通沒有，進來就是干！

第一關的獲勝條件很簡單，30秒30個地鼠，小弟輕輕鬆鬆解決了。實際打地鼠遊戲的音效和手感其實還挺好的，就是這一鎚子價值好幾元，但是沒漲智商。

這到底是什麼神仙遊戲！真的是很大程度提高的我智商呢。好吧我是傻逼。

小弟在被此遊戲的騷操作驚呆的時候，意外發現這個遊戲的作者好像在B站有賬號，而且還是個妹子，看視頻兩個遊戲真的非常相似。不過視頻的時間是2018年7月，而這個遊戲是2019年12月上架的。

然而深究後發現，這個遊戲根本不是這妹子上架到Steam上去的，而是被人盜用了。妹子本人只不過是隨手做了個遊戲給自己解壓而已。發布B站之後，有人向她要了這個資源，然後就被盜用了，而妹子本人還在一臉懵逼的狀態當中！

來幫大家總結一下，也就是說這個賣1000元的打地鼠遊戲，不僅遊戲做得渣，還涉嫌盜用別人的遊戲私自上傳！

這件事可把各位網友給氣炸了，紛紛進行了無情的譴責：

——太慘了，隨手做個遊戲，就莫名其妙的遊戲就變成了別人的遊戲

——舉報抄襲，涉案金額1098*17 =1.8w，那麼數額巨大，可以算刑事案件了嗎

——這個抄襲的人可能是沒有腦子，把別人練習的遊戲上架，還1000多

小弟和你說：以後大家給陌生網友傳東西的時候一定要長個心眼，要有版權意識，畢竟你真的沒辦法知道對方是不是惡意的！防人之心不可無啊！

一個正驚問題：你還知道Steam上有哪些坑爹遊戲？
