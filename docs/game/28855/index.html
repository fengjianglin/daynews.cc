「地下城與勇士之小狐狸君愛談遊戲電競」第七百七十四期《DNF100級傳說周期一覽：掉落自動轉甲，屬性媲美普雷，10天換一件》，100級的神器和傳說都是能升級、能繼承的，所以本質上說，和現在的95思路差不多，區別就是可以直接繼承，當然目前看來100級的神奇和傳說都是有專屬的副本的，都是每日刷6次。那麼玩家如何選擇副本和做裝備呢。主要是神器是96級玩家就能穿戴的裝備、傳說是98級的玩家可穿戴的裝備，所以本質就是等級差。當然和95級一樣，存在著問題就是既然可以直接刷100級史詩，那麼做傳說和神器的意義何在？老規矩搞快點，直接上本次100級傳說裝備的製作和掉落截圖展示：

DNF100級傳說周期一覽：掉落自動轉甲，

傳說裝備掉落後自動轉甲，掉落的幾率還真不低。一般而言，和現在的夜襲副本有點類似，玩家每日限定6次。不過100級的副本，不需要滿級就能刷這兩個特殊的副本！這一點比較人性化，所以等於說玩家可以一邊做裝備，一邊升級了！這個思路是非常好的，就像之前的紫裝一樣，玩家升級做任務，拿到裝備然後不斷的打造裝備，最後繼承到100史詩上，這個思路是可以接受的！

屬性媲美普雷Raid史詩：

那麼這個100級傳說的強度如何呢？首先說基礎面板，完全和95史詩夜語黑瞳武器是一致的，然後是屬性，95傳說全職業屬性都一樣。武器是：52%的進呢過攻擊力、30%的最終傷害。普雷武器是55%的技能攻擊和30多最終傷害。差距微乎其微了。可見傳說的設定就是表明100級要徹底淘汰95版本的裝備了。姜策劃的100級史詩大有回歸85版本的趨勢，手搓系列、技能冷卻、幽魂設定……

10天換一件傳說：

其實國服玩家最期待的是製作周期，周期短一點的話，可以製作。周期太長就等著活動個贈送：磁療和成品！掉落的話還是比較看臉的。目前韓服隊員一件是1200個材料，每天能拿到100~120個材料，還能隨機掉落，所以說平均一件按照10天算是可以接受的。
