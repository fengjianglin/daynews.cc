DNF：過氣副本得不到優化，只能靠累積在線活動蹭熱度

2019-12-11 2,082條評論 3,884次閱讀 1841人點贊

小夥伴們好呀，我是貓幼，今天也是充滿希望的一天。

隨著版本的不斷迭代，新的副本不斷上線，這樣一來，遊戲內的福利的確是越來越多了。但是也面臨了一個問題，玩家們的負擔也就越來越重，不得不對一些副本選擇放棄，這樣一來，一些副本就成為了非常雞肋的副本。

就像最近更新的周常副本，以前是非常雞肋的，現在這麼一改，的確得到了大部分玩家們的認可。以前如果想要獲得所有獎勵，每天好幾個角色都要去阿比斯處走一遭。現在一來，一周一個角色只要打三次就可以了，並且一天就可以打完。時間減少了，獎勵也豐富了不少，裡面的寵物裝備雖不比春節寵物裝備，但是用來過渡已經足夠了，裡面的升級券也可以讓小號升級變得更加速度。

巨龍圖相信大家非常的熟悉，做一個每日就可以刷一次深淵，並且還可以獲得的不少的強烈氣息，是固傷玩家的福地。但是現在鍛造爐也經常免費送，這個副本已經變得十分雞肋。普通的深淵已經沒有辦法畢業泰波爾斯史詩了，想要畢業必須要刷超星空，哈林史詩現在一抓一大把，誰還會去刷這個副本呢？

還有安徒恩周常和盧克每日，安徒恩從團本變成安徒恩周常，這樣更比較像單人的烏龜。但是現在還有多少玩家去打？為了附魔？現在畢業附魔也只要幾百萬。為了裝備？哈林史詩就可以超越安徒恩裝備了，並且升級起來也非常簡單。盧克每日就更加雞肋了，現在沒有人再去做海博倫了，打起來還又臭又長。

異界現在還是一個比較熱門的地方，可以摸金，算是一個優化的比較好的副本了。但是別的地方，就不敢恭維了。很多副本基本無人問津，青龍黃龍大會要不是為了二覺也根本沒有玩家去玩了。所有優化副本，現在也是非常重要的事情，副本做好了，就不用煞費心機的去做一些登錄在線活動了，玩家們都有事情做，何愁不在線呢？

好了，今天就說到這裡了，歡迎大家的評論點贊，跪求大佬們的關注。
