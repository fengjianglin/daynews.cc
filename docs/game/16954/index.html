魔獸世界：40個20級小號，能不能打得過一個滿級120的號？

2019-12-15 52條評論 3,088次閱讀 378人點贊

魔獸世界有玩家提問，一個團的20級小號，能不能打過一個滿級120的號。根據自己經歷，抵抗未命中，還有小號傷害不高，大號裝備防禦綜合考量沒什麼勝算，要不就組個40人的團打個80級的怪就知道了。魔獸世界等級相差10級以上基本打不動了，40個20級打120級的號，都能打過團滅40個小號。

現在正式服室內pvp是有補償機制的，就是說基本不存在裝備碾壓，裝等好的打對手自己看起來打了一萬點傷害，其實對手只掉了八千。裝等差距越大換算比例越大，當然，具體怎麼計算沒這麼簡單，我說的只是簡化版方便理解。20級小號去打120級的，估計個個都是超級兵了。40個20級，每個人都跟120差100級。什麼概念？以前超4級就很難打，超7級基本上就抵抗，這直接超100級。站那不動讓你們輸出一管藍，回頭一個冰槍一個小朋友。

魔獸世界有個東西叫等級壓制，當年40級的人打一個60級裝備不太好的還有希望，20打100級的開多少死多少。知道100級的裝備和60級以前的裝備差多少個時代，60級外星的一套綠裝屬性都快超過地球的垃圾紫了。60以後每10級裝等升的不是一點半點，20級小號血過萬嗎？120級多少血，站在那裡讓你打都打不死，一個技能死一片，如果遇到獵人，人家站那裡不用動，光寵物就咬死這群小號了。

不要說一個團的20級小號，即使是一千個20級小號，也不要想著殺120級的任何一個職業，等級壓制懷疑人生。破測試，只是壓縮數據後產生的畸形結果，就跟隨機本萬聖節打無頭一樣，滿級是正常打，如果裡面有一個幾十級小號，一發就秒掉無頭。換成沒壓縮數據前，不管你多少個20級，都是未命中，偶爾打中也是－1。

滿級有裝備的角色和20級的角色不是一個概念，如同人與螞蟻的區別，即便來兩個團也干不過。別說差幾十級，差10級就基本打不掉血了，不是抵抗就是閃避，偶爾打中一下也是扣個兩位數甚至個位數。60級暗夜野德在貧瘠之地與二十多個小號打，毫無壓力把對方團滅了。因為可以進去秒一波，打的血少了直接開旅行跑路，反正小號追不上，脫開距離然後加加血再回來繼續一波。
