LOLpbe服無限火力重新回歸，國服還會遠嗎？

2019-12-19 2,221條評論 4,421次閱讀 2764人點贊

在眾多玩家的期盼之下，無限火力模式終於回歸，不過當然不是在國服，而是在pbe測試服。很多玩家都在期盼著無限火力模式什麼時候才能回歸國服，等不及的小夥伴可以掛個加速器去pbe服搶先體驗一下無限火力的精彩，我用的是迅游加速器。

無限火力模式最吸引人的一點就是無限的藍量加上冷卻時間非常短的技能，解決了玩家在普通的對戰模式下想要釋放技能，奈何技能還在冷卻中的痛苦。整個模式的戰鬥節奏是很快的，進入無限火力模式之後就像打開了開關一樣，你只是一個毫無感情的戰爭機器，就算犧牲了也別擔心，一會兒就可以重獲新生。

此次更新的無限火力模式為隨機英雄，所以在遊戲中也極大可能會遇到自己根本沒有玩過的英雄，不過不用擔心，用哪個英雄不是練手呢，說不定你就能在這個模式中找到自己的天命英雄呢。

雖然這個模式很受歡迎，但是很多玩家都知道這個模式並不是長期開放的，也許若即若離的感覺才能更讓我們念念不忘。

現在無限火力已經在pbe服上線了，玩家們可以趕緊上線體驗了。想要等國服上線再體驗的玩家也可以等一等，國服應該不遠了。
