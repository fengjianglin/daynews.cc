CSGO：十一月信箱問答第三彈——兔子隊Misfits回歸可能性很大

2019-12-06 0條評論 15次閱讀 0人點贊

摘要

在DeKay的十一月信箱問答第三彈中，玩家主要圍繞著Misfit的回歸，Complexity目前的陣容的潛力及OG選擇mantuu擔任主狙的理由等問題展開提問。Q：假如沒有Astralis這支戰隊，你覺得哪支隊伍能統治CSGO的職業舞台？

在DeKay的十一月信箱問答第三彈中，玩家主要圍繞著Misfit的回歸，Complexity目前的陣容的潛力及OG選擇mantuu擔任主狙的理由等問題展開提問。

Q：假如沒有Astralis這支戰隊，你覺得哪支隊伍能統治CSGO的職業舞台？

A：如果沒有A隊的話，我認為Liquid的崛起能來的更早一些，當然持續時間也會更久。Liquid一直有一個毛病，那就是總會輸掉決賽，假如沒有Astralis在前面阻擋他們的話，他們的信心自然也會暴增。在那之前我就不太清楚了，其實如果沒有A隊的話，許多戰隊都具有成為世界最強的潛力的。

Q：有任何關於兔子隊Misfits回歸的消息嗎？

A：他們肯定是有這個打算的，但我相信這很大程度取決於新聯賽的加入。從我得知的消息來看，他們是除了Dignitas外另一個看好老NiP核心陣容的俱樂部。我覺得他們最終還是會回歸的。

Q：在招回GuardiaN之前，NaVi有考慮其他填補Zeus空缺位置的人選嗎？

A：據我所知，他們除了GuardiaN外基本沒有考慮其他選手。GuardiaN是他們心目中的第一人選，而且他本人也非常願意回歸。

Q：你怎麼看待如今的古堡？你覺得它能重新回到比賽圖池嗎？還是會永遠作為一張休閑地圖呢？

A：幾乎所有職業選手都討厭這張圖證明古堡仍有很多需要改善的地方，這代表著巨大的工作量。不過職業選手不喜歡它不代表Valve會放棄，就像Vertigo，剛開始沒有人覺得它能成為比賽圖池一樣。所以我覺得古堡是有可能回歸的，不過這個時間是無法預測的。

Q：你覺得Coldzera明年能成為Top5選手嗎？

A：我不認為他能排到這麼高，但他肯定能留在Top20的行列中。現今FaZe的組織結構（假設他們不會改變的情況下）會帶給coldzera很大的壓力和負擔。如果他想成為Top5，他需要一個像TACO那樣的工具人來為他創造空間。或許在未來broky能擔任起這樣的角色，但我覺得可能性不大。

Q：能合理地預測一下Dignitas陣容的潛力嗎？我覺得他們會排在世界第15到20之間，並且進不了Major。

A：我覺得最好的情況是世界第10-15之間，並且能進入到Major。陣容里的一些選手，比如Friberg和f0rest，在近期的表現證明了他們仍然能適應高水準的賽事。如果他們和Hallzerk這樣具有潛力的新星搭配，我覺得或許會產生一些奇妙的化學反應。但就像我說的，這只是最好的情況下。

Q：你覺得Complexity目前的陣容能取得多大的成就？k0nfig能再次成為隊內的明星選手嗎？

A：我覺得只要Complexity打夠足夠多的線下賽，他們就能成為一支頂尖的強隊。不過有一點讓我擔心的是隊內的選手們或許會因為外界過高的期望值從而感受巨大的壓力。毫無疑問，這支北美俱樂部在打造這一陣容的基礎上花費了大量的資金，他們的每次陣變都會引起巨大的關注。我對於這支戰隊感到很樂觀，他們有潛力成為一支頂尖強隊，k0nfig也將再次成為明星選手。

Q：OG選擇mantuu是因為他是目前俱樂部能簽下最好的狙擊手嗎？就我所知他在此前的隊伍甚至都不是主狙擊手。

A：我不認同這個觀點，要知道oskar現在也沒有合同在身，而且他顯然是一位更加優秀的狙擊手。我認為OG選擇mantuu更多的還是因為這位選手更適合隊伍的體系，而且mantuu在圈內的風評也很好。
