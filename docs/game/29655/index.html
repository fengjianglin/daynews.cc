DNF100級4人團本爆料：新增9張附魔，傳說屬性逆天，奶媽成大贏家

2019-12-25 383條評論 1,742次閱讀 823人點贊

「地下城與勇士之小狐狸君愛談遊戲電競」第七百九十四期《DNF韓服100級4人團本爆料：新增9張附魔，傳說卡屬性逆天，奶媽成為最大贏家》，隨著本次韓服的更新之後，新的4人團本也全部送出，韓服冬季發布會的承諾，只有男格鬥職業的三次覺醒沒有更新了（希洛克Riad屬於畫餅充饑）。當然新副本自然會送出的新的道具，此次加入了的9張新的附魔卡，其中3張傳說級別、6張神器級別！無論是新的副本還是道具，對玩家來說，都有著不錯的體驗和提升。老規矩搞快點，直接上本次新副本和道具的截圖展示：

DNF韓服：100級4人團本爆料，

本次的4人團本，被玩家戲稱為鳥背plus，當然僅僅是副本的地位而已。就像鳥背是普雷Raid先行版本，都是在大鳥的背上戰鬥，而新的4人本就是希洛克Raid先行副本。這個副本玩法、副本的規模，都是比較大的！當然由於目前團本處於審美疲勞時期，姜策劃也是開發出了不一樣的玩法，一個副本包含了B1~B4個區域，刷完不同的BOSS，才能進入下一個區域。怪的機制和之前並沒有多大的改變，還是小遊戲+打樁的玩法。

新增9張附魔，傳說卡屬性逆天：

新的附魔卡，再次刷新的裝備的打造高度。現在首飾附魔提升到了30，雖然是星星點點，不過積少成多呀！而新增的首飾+70的附魔，可謂是非常大的提升。這幾張傳說附魔的屬性，算是得到了提升。剩下的6張神器卡片，屬性倒是一般。現在國服送出的附魔，都是和這幾張的屬性差不多。所以國服玩家到了100版本，附魔基本不用歡樂。除了3張比較強勢的傳說附魔。

奶媽成為最大的贏家：

之前的版本奶爸和奶媽的首飾附魔都是比較低的。只有武器和上下衣才能達到75。現在首飾提升非常的大。本來100級史詩四維加成就非常的大，下奶附魔也是大幅提升，使得奶媽的輔助能力大幅提升。
