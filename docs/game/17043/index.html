你沒聽錯！這支叫VG的巴西CF戰隊，竟然沒一支中國CF戰隊擊敗它

2019-12-14 663條評論 1,830次閱讀 361人點贊

給大家看三張圖，均為CFS賽事戰況結算圖

圖1，說明的是VG戰隊10:5 SV

圖2，說明的是VG戰隊2:0 白鯊

圖3，說明的是VG戰隊2:0 AG

VG與SV的比賽是CFS的小組賽首輪對抗，VG這邊以10:5的比分擊敗了SV，SV只能爭該組的小組第二，這也就決定著，VG在八進四會遭遇其他小組的第二：因為AG是小組第一出線了，所以不會遇到AG，再因為SV和VG是同一個小組的，所以，白鯊是最擔心的那支隊伍，白鯊有50%的幾率打VG。

CFS小組賽結束後，進行了關鍵的八進四抽籤，經歷了重抽的小插曲之後，白鯊遭遇到了VG，同時，由於VG在上半區，也決定著，如果白鯊敗了，AG就要迎戰VG，他們是首輪阻擊VG的最後一支中國隊。

在與VG的對抗中，白鯊敗下陣來，一張地圖拿了4分，一張地圖拿了6分，場均5分，白鯊後來在微博講到：的確是打不過，其實已經研究透了對面怎麼打，研究透了對面的站位。

在12月13日的半決賽中，AG迎戰VG，兩張地圖都是8:10落敗，雖然是大比分0:2輸掉，但是卻覺得分外可惜；不過，我們也虛心接受，VG的確在槍法和臨場調整上更加出色一些。

很多玩家在交流，為什麼巴西的CF變得這麼強了？與我們中國CF類似，巴西也有職業聯賽，我們的叫CFPL，他們的叫CFEL；聯賽的存在使得所有隊伍有了交流對戰的平台，穩住成熟的賽事體系地的確是進步的土壤。與巴西隊伍的對抗，小編很容易響起CFS2015年的世界總決賽八進四，全明星陣容的EP不敵巴西隊伍INTZ，另外還有就是漢宮出征CFS巴西邀請賽的時候，雖然最終拿下冠軍，但遇到巴西隊伍時也是很驚險的，隨後SV在2018年不止一次輸給巴西過BD，再到2019年白鯊在CFS巴西邀請賽憾負VG，巴西CF戰隊給我們的壓力與日俱增。

在巴西，得一次職業聯賽CFEL的冠軍可以收穫25000雷亞爾，摺合成人民幣只有4萬+的數值，而在我們國內，即使選手不拿冠軍，強隊的選手依靠直播依然可以過得非常好，而巴西的選手則不同，如果不能在CFS世界賽取得驕人的成績，打CF的收入就不能支撐他們的開銷；對於CFPL賽區選手來說，打不好只是收穫輿論壓力（而且這些壓力還會隨著時間而消散），而巴西CF選手打不好就不可能一直堅持打CF了，就要去做傳統的工作。

同樣是CF，之於不同賽區選手的意義不同，也許，這就是巴西CF進步迅速的原因吧！
