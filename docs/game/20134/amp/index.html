DNF：普雷裝備畢業下，三套紅11超界對比，輕甲依舊一家獨大

天天要聞
2019-12-18

95版本裝備的強度，一直讓玩家爭論不休，但是，事實證明，輕甲還是最強的防具，在普雷裝備畢業下，20秒綠沙袋打樁最高。這也就導致，國服玩家穿輕甲的比較多，形成了「一家獨大」的局面。為了改變這種現狀，越來越多的職業特色加強，讓皮甲也能踹口氣。針對紅眼，玩家打造了三套紅11超界皮輕重，來實測傷害差距。

作為紅眼，在當前版本，皮輕重這三套超界防具，穿的還是較多的。而其都增幅紅11，且細節方面，完美無瑕的打造。如圖所示，玩家穿上三套11超界後，站街的面板，都4307力量，獨立攻擊力2823。

為了追求極限傷害，紅眼佩戴的武器是鍛造+8，右邊普雷7件完美畢業，稱號是神選之人。在這種情況下，20秒綠沙袋手法一致，施放技能瞬間也一樣，來對比三套防具的差距。

首先是輕甲，一二覺技能很爆炸，整體的輸出，也是非常高，20秒綠沙袋高達823E。紅眼的傷害，在諸多職業中，能排在前列，普雷畢業搭配輕甲，紅11打造超過800E很輕鬆。

其次是皮甲，雖然在一二覺方面，沒有那麼的出色，但是，小技能輸出，卻比輕甲強勢。其20秒綠沙袋，打出了797E的傷害，跟輕甲有些差距。不過，等魔界大戰CP符文系統上線，皮甲還能提升一些，到時候就會媲美輕甲。

最後是重要，比輕甲略差，但比皮甲略高。總之，是介於皮甲和重甲之間，單刷比較舒服的一套超界防具。作為第一紅眼，旭旭寶寶就穿的是重甲，畢竟屬於本命套。

由此可見，在普雷畢業下，輕甲的優勢，還是大一些，畢竟一二覺的傷害，提升比較高。不過，皮甲雖然劣勢，但只是暫時的，魔界大戰是一個崛起的契機。而且，皮甲小技能輸出很舒服，比較著重於續航。在以後沒有打樁的年代，爆發就不會吃香，續航成為贏家，也就是說，皮甲的小技能，刷起圖來，就非常的舒服。
