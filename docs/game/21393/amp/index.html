《火焰之紋章》系列回顧（九）：連續致命的失敗

天天要聞
2019-12-20

本文由多玩新聞中心《印象》欄目出品，轉載請註明出處。

　▌再次挑戰主機市場

《火焰之紋章》系列在GBA三部曲之後，於2005年推出了首部3D化續作《火焰之紋章：蒼炎之軌跡》。

《火焰之紋章：蒼炎之軌跡》

遊戲請來了人氣畫師北千里擔任角色設計師，還請了Digital Frontier動畫工作室製作了大量的過場動畫，整款遊戲的3A級製作使得遊戲品質極高，並獲得了大量好評。

然而因為遊戲發售的時候，已經是NGC平台的末期了，僅僅靠《火焰之紋章》的口碑。是無法帶動整個平台硬體銷量的。而NGC原本就並不高的普及率，也讓《蒼炎之軌跡》的銷量並未達到預期，成了一款叫好不叫座的遊戲。

雖說如此，對於Intelligent Systems來說，《火焰之紋章》系列在海外市場與家用主機遊戲市場之上的機會依然是相當大的，他們最後還是決定將續作計劃放在家用主機平台上，不過這個平台肯定不會是NGC了。

　▌Wii平台續作

NGC的後續機種，任天堂的第七世代家用遊戲主機，其最初的宣傳就是一款擁有全新互動方式的遊戲主機。

2005 E3展會上 岩田聰展示了第七世代新主機

在2005年5月的E3展會上，岩田聰公開了一款代號為「革命（Revolution）」的新主機。雖說現場並未公開新主機的硬體數據規格，但具有突破性的遊戲手柄設計，給了玩家們深刻的印象，這款新的主機就是以體感互動作為主要特徵的Wii。

《火焰之紋章：蒼炎之軌跡》的續作《火焰之紋章：曉之女神》，也正是在Wii主機公布的同時開始開發的。

Intelligent Systems這次吸取了教訓，很早就開啟了開發計劃，其目的就是想趕在Wii主機的發售初期推出本作。

為了讓遊戲的畫質與表現力能達到最佳狀態，本作在開發時就否決了NGC版的開發計劃，專註在Wii平台上進行開發。

Wii主機

在開發初期，開發團隊並不知道Wii主機會推出哪些手柄，因此他們就採用了傳統手柄的操作方式進行遊戲設計。

在逐漸了解到Wii的硬體規格之後，開發團隊又認為體感功能對於一款幾乎沒有動作要素的SRPG遊戲來說是沒有必要的，強行增加體感功能反而會增加工作量拖長開發時間。

最終他們決定堅持最初的想法，《曉之女神》也是Wii主機上少有的不具有體感功能支持的遊戲之一。

遊戲並不支持體感

有趣的是，Wii主機除了體感操作，還有一個特點就是支持WIFI聯網。《曉之女神》的開發團隊就曾考慮過在遊戲中運用這個功能，通過聯網為遊戲增加DLC地圖與角色，甚至是做出多人在線聯機戰鬥來。

然而最終這個計劃也因為遊戲平衡與開發技術方面的各種困難，導致聯網要素並未做到遊戲之中，這是非常殘念的事情。

遊戲截圖

　▌更大的製作規模

《火焰之紋章：曉之女神》的主要開發成員和前作沒有太大區別，不過因為製作規模的擴大，開發團隊的人數由前作的100多人增加到了將近200人。

遊戲中的過場動畫依舊由Digital Frontier動畫工作室負責製作，本作參與過場動畫的開發人員是相當多的，幾乎佔了整個團隊的一半。不僅如此，本作的過場動畫還使用了動作捕捉，這使得過場動畫中的角色看起來更加自然，細節也要更多一些。

此外，本作過場動畫還在突出遊戲角色的同時，著重渲染了遊戲中史詩的氣氛，製作出了各種大場面，《曉之女神》的視覺表現力可以說是完全超過了前作《蒼炎之軌跡》的。

視覺效果更佳的過場動畫

本作劇情設定在了前作的3年之後，遊戲中不僅有新角色的登場，還有部分前作中的友方角色登場。而前作中埋下的眾多伏筆，也在本作中一個一個的展開。

本作的文本量幾乎是前作的1.5倍，因此本作的世界觀是相當廣並且錯綜複雜的，這也是系列首次使用了如此大規模結構的劇情表現方式。

更加龐大的劇情

本作總共分為四個章節，每個章節玩家所控制的隊伍都是不同的派系。雖說這讓本作的可玩角色非常多，但這也使得玩家在遊戲流程中需要不停調換使用角色，失去了一些培養角色的樂趣。

本作首次加入了二次轉職系統，角色在分別在10級與20級時可以使用不同的轉職道具進行轉職。角色在二次轉職（最上級轉職）之後，能力會得到大幅度加強，玩家在戰鬥時也能體驗到開無雙的快感。

遊戲戰鬥畫面

　▌接連失利

《火焰之紋章：曉之女神》於2007年2月22日登陸Wii平台，這距離Wii平台的發售日僅有3個月，但遊戲的評價與銷量都並不算好。

本作MC平均分78分，Fami通評分33分，在系列中僅高於《聖戰之系譜》。憑藉新主機的高人氣，本作的銷量比前作稍高，但在日本地區也僅售出了約17萬份。

遊戲宣傳圖

《火焰之紋章：曉之女神》飽受批評的地方在於，遊戲在遊戲性的改革上太少了，基本只是對於前作的玩法進行了一些改善。

而在視覺表現方面，除了過場動畫，遊戲實際畫質也沒有得到多少改進。並且作為一款Wii平台遊戲，也沒有活用Wii的機能，一些媒體評測甚至還在文中直言「這不像是一款真正的Wii平台遊戲」。

如果說《蒼炎之軌跡》還能將失敗的原因歸咎於主機的不爭氣，那麼《曉之女神》的低評價就只能說是Intelligent Systems自身的原因了。

《火焰之紋章：曉之女神》封面

兩次衝擊家用主機市場的失敗， 讓Intelligent Systems不得不再次退出了主機平台，將續作放在了任天堂新掌機NDS之上。

一方面他們需要積累資金，以彌補主機平台的開發虧損。另一方面，他們也要考慮如何讓《火焰之紋章》實現真正的革新……
