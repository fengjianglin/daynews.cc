DNF：旭旭寶寶沉迷新史詩之路！連遇隱藏還抱怨怎麼不出五黃啊

2019-12-12 1,695條評論 3,073次閱讀 1297人點贊

DNF昨日更新之後推出了「史詩之路·星河裂縫」的玩法，在這次活動中一共分為兩個部分一個是「星河裂縫」另一個則是「超·星河裂縫」。這兩個都是限制DNF95級角色才能進入「星河裂縫」每天都有10次免費進入的機會，而「超·星河裂縫」需要10點疲勞值和一個波塞姆的祝福花瓣作為門票，而次數是沒有限制的。沒想到這個活動出來之後引得DNF主播旭旭寶寶沉迷其中無法自拔。

寶哥作為一個DNF的老牌主播相信只要玩DNF的人都知道他，並且寶哥在DNF里氪金的力度也是相當的大，所以「超·星河裂縫」的門票對寶哥來說那簡直就不算事。在體驗過這個玩法之後寶哥就一發不可收拾沉迷了進去，這也就算了，關鍵是寶哥還連遇隱藏BOSS運氣好的逆天！

隱藏BOSS是「超·星河裂縫」中才有的，當玩家擊殺了「超·星河裂縫」中的領主之後會有幾率碰到隱藏BOSS，相信玩遊戲的玩家應該知道這個「有幾率」是什麼意思吧。這個隱藏BOSS會隨意掉1—5件超界史詩裝備。而寶哥連遇隱藏BOSS就算了，竟然還總是連爆3件超界史詩裝備，讓不少看直播的玩家眼紅不已，但是寶哥竟然抱怨「怎麼沒出五黃啊」。

如果寶哥出了五黃估計都有人想順著網線過去吸歐氣了！想要了解更多的遊戲資訊那就關注小編吧！
