夢幻西遊之海底秘境：打敗龍王和靈感大王不再是夢

天天要聞
2019-12-11

大家好，我是電競嗨森君！最近有小夥伴對海底秘境這個任務比較好奇，想要知道這個副本怎麼通關，海底秘境一共有25關，今天小編就挑出比較有挑戰性的兩關，給大家說說它的通關方法。

首先說一下第23關，這一關的主怪就是龍王，血量為7500，其它的四個小怪分別為普陀、獅駝、龍宮以及大唐，血量基本在3500-6800之間。

建議陣法：鳥翔陣

建議召喚獸：高敏吸血鬼，最好是帶鬼魂的

建議助戰夥伴：觀音、鎮元子、紅孩兒，其餘的兩個可以選擇唐僧、地藏王或者西海龍王和孫悟空

戰鬥技巧解析

在這一關中，龍王毫無疑問對抗起來會比較困難，不僅因為他的法傷很高，還因為他對物理攻擊和封印的抵抗能力都很高。這就對我們的法防修鍊有了很高的要求，否則龍王一個龍騰釋放出來，就有可能會被秒殺。由於龍王的攻擊性實在太強，我們也無法直接將龍王秒殺，最好的策略就是先將小怪們控制住，以防它他們也給己方帶來傷害。

如果你是高級玩家的話，可以選擇以法攻為核心的多攻陣容，會輕鬆許多。

重點講解一下平民玩家的玩法，在陣容方面選擇雙封、一法系以及一治療會更有保障。在這裡，鎮元子由於自己的獨特技能飄渺式還是佔了很大優勢。雖然龍王具有抗封能力，但還是有一定的幾率被成功封印，封印之後，造成的傷害值減少，己方的治療壓力也沒那麼大了，所以還是有必要選擇兩個封系門派來給龍王施加壓力。

重要的就是解決掉小怪，減輕團隊的負擔，可以按照大唐、獅駝、龍宮，最後是普陀的順序來進行清理。小編不太建議用孫悟空，雖然他很厲害，但是在這場戰鬥里，紅孩兒的爆發力會更強，輸出的物理傷害也更高。

龍王這一關還是讓不少玩家感受到了龍宮的過人之處，但其實只要把握好戰鬥節奏，想要取得順利也沒有那麼困難。如果你有能力的話，小編還是建議購買一些超高性價比的夥伴來幫助戰鬥，有了這些左膀右臂的加持，便可能會扭轉局面。

在打敗龍王之後，你就會來到第24關靈感大王，這一關的怪物從右到左分別屬於獅駝、龍宮、獅駝、普陀以及方寸。

建議陣法：龍陣

建議召喚獸：同龍王戰

建議助戰夥伴：觀音、鎮元子、紅孩兒、地藏王、西海龍王

戰鬥要點解析

與秘境的前面幾關大有不同，靈感大王這一關的團隊陣容會相對更加平衡，不僅有三個面傷單位，還有一封和一治療。這樣一來，如果在前幾關造成了沒必要的人員損失，在這一關就會比較吃虧，可能會全員崩潰。這也是為什麼小編一直在強調要帶上兩個封系夥伴的原因，先把敵方的主要力量封印住，尤其是那兩個獅駝的怪。兩個回合過後，在考慮點殺的順序時，可以先從最右側的獅駝或者方寸開始清理，因為主怪還有普陀的防禦值都很高，想要打倒比較困難。

在現在的夢幻西遊版本里，已經不會再出現夥伴被封印後，無法再釋放物理攻擊的情況。所以，如果你帶的夥伴屬於物攻型，例如孫悟空，那麼可以先考慮除掉敵人的獅駝；如果你帶的夥伴屬於法攻型，比如鎮元子，可以優先考慮清除敵方的方寸，減輕團隊壓力。除了鎮元子，還有一個法攻型夥伴值得推薦，也就是鐵扇公主，不僅可以給團隊提速，還能在鳥翔陣的基礎上，為己方帶來更多的優先權。

在清理掉這一關的兩個小怪之後，情況基本也會穩定下來，即使一不小心沒看住，靈感大王釋放了鷹擊，也不會對團隊的血量造成太大的影響。

其實靈感大王與剛才龍武那一關相比，難度並沒有很大的提升，只要把握住時機，再加上一些耐心和定力，還是可以打敗靈感大王的。

以上就是海底秘境23關以及24關的打法攻略，總體上看，難度都不是很高。在打敗了龍王和靈感大王后，接下來就要直面秘境降妖迷宮中的最終BOSS——蜃氣妖分身了，那最後一關該如何打呢，下一期小編告訴你答案！
