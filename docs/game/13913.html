王者模擬戰坦射的玩法，快來學習一下啦

2019-12-12 1,422條評論 2,586次閱讀 536人點贊

目前而言，模擬戰雖然處於一個比較平衡的狀態，也就是大家常說的所有陣容都有拿第一的可能，這句話其實只是相對而言，就目前情況來看，相比上個版本的七長城遊戲體驗是好了很多，可是如果一般模擬戰前幾名大概在27回合左右結束這局遊戲，所以如果想拿第一，肯定要拿後期強勢的陣容，那後期強勢的陣容其實很多，坦射，扶桑法，刺客，以及精英戰士。

今天先說坦射，坦射成了版本上分利器，碰上刺客可以調站位，那坦射應該怎麼玩呢？坦射強在哪裡？我們一點一點來

首先牌的組成

公孫離三星，奕星兩星，蘇烈兩星，盾山兩星，伽羅兩星，廉頗兩星，黃忠良心，孫臏兩星

坦射最大的優勢在於它即使沒三星的也能打得過大部分陣容，應該有那種被沒一個三星的坦射打輸的人吧！當然它最大的弊端就是牌太貴，但是貴有貴的道理，便宜有便宜的貓膩。

至少我們要知道成型養豬是打不過成型坦射的。這個事實正好可以證明我說的這句話。

我們接下來就要說坦射的運營了。

前期的過渡對於所有後期陣容都適用

前期第一回合分兩種情況（這裡不算野怪局）打得過，打不過，我想大家應該有這種意識，前期第一把看得出到底打不打得過。打得過是最好的打不過的話，請往下看

這裡我第一把輸了，這個時候我們就要留十塊錢吃利息。如果打不過了我們就要知道接下來的第二把就盡量別針扎，直接認輸，如果錢不夠十塊把能賣的賣了，吃十塊錢的利息

緊接著的第三把同理，第三把可以吃20塊的利息，打完這三把就到了野怪局，打野怪局之前也不刷，等打完這個時候可以升級，然後拿強勢牌或者長安牌。

然後緊接著繼續留30吃利息，但是也不可以刷牌，如果還是贏不了繼續輸也沒關係，最後可以升級（可以升級的意思是你升完級還能剩30金幣以上，當然不能太死板，還是得大家靈活一點）。

到了中期，你已經成功熬到七級了，七級的你就可以開始刷牌了！

七級的我們大概是這個樣子，現在我們要做的是攢牌，刷阿離，攢其他得坦射牌，抽到一個坦射牌，就買一個場上的牌，知道坦射陣容成型，最後升八級上最後一個牌，坦射就正式成型了。

這個時候大家要做的就是等待勝利的到來，然後保持金幣至少在20金幣以上，刷牌就完事了，記住一點，除了阿離，其他的不要刻意追三星，最後可以升九級上兩黃忠，或者一個呂布

最後我的主頁有坦射的視頻，大家可以去看看，明天見！！
