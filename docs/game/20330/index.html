Clearlove正式退役轉為主教練！在台下，或許更能看清前行的道路

2019-12-17 99條評論 1,844次閱讀 709人點贊

作者：瓦羅蘭文藝復興聯合會——框框框子

這裡是#瓦羅蘭文藝復興聯合會#，

剛入駐今日頭條不久，請各位多多關照~

我們有全網最優秀的攻略、賽評、同人故事、背景挖掘、皮膚賞析、音樂樂評等高質量乾貨。

有最可愛的和對遊戲最富有熱情的寫手們。

如果你對我們的文章感興趣的話，歡迎點擊小頭像查看以往文章哦，我們會定期更新一些高質量的優秀內容，同時也麻煩點一下關注照顧一下頭條的萌新啦~

這個關注一定會值回票價的。

12月16日 EDG官博官宣了一則人事變動：前EDG打野Clearlove正式轉型為隊內主教練。

「一片樹林里分出兩條路，而我選擇了人跡更少的一條。」

「從此決定了我一生的道路。」

明凱退役了。

當這個消息出現在熱搜榜單上的時候，我不知道該說什麼。

在轉會期的最後一天爆出這樣的消息，即在意料之外，又在情理之中，八年前走上電子競技這條道路，七年前為中國拿下第一個英雄聯盟世界冠軍 ，六年前從WE加入EDG，四年前為LPL拿下第一個官方賽事獎盃，三年前被掛上4396的名號，兩年前開始將擔子轉給後輩，一年前在台下看著小將帶領EDG沖向世界舞台。

今年正式划上句號。

廠長 、諾言 、MSI總MVP 、LPL第一打野 、艾歐尼亞深夜屠殺者 。

反向證明世界第一打野 、4396 、馬踏飛箭 、魚死網破 、NMSLWSND 。

前者是明凱，後者也是明凱，好也罷，壞也罷，走過八年職業生涯，很多榮辱和過去都已經被時間徹底衝散。該打過的仗都已經打過了，當擊敗的敵人都已經擊敗了，該守住的榮譽也已經守住了，自此以後，自有王者的冠冕為他存留 。

名為選手的人生，已經結束。

名為教練的征途，才剛剛開始。

1.

明凱的職業生涯，是由一款叫《信長野望》的遊戲開始的，我們無需關注這款遊戲的地位到底達到了什麼地步，我們只需要知道LPL最早一批職業選手有不少都出自於此，包括明凱、無狀態、捲毛等一系列老選手都是出自於此。

而明凱離開信長野望轉LOL的原因也非常簡單——因為信長野望讓他活不下去。

站在今天的角度上，我們或許會覺得他走上職業這條路是為了夢想。但客觀地說，在當時那個電子競技遠沒有發展起來的時代，對於大多數早期的職業選手而言，夢想遠遠不及一張長期飯票重要。

世界冠軍選手穿著皺巴巴的隊服去擠地鐵拉贊助，累的睡在人來人往的車廂里。

一頓李庄白肉成為幾個年輕小伙一輩子揮之不去的回憶，即使後來分道揚鑣也記得清晰無比。

無關夢想，無關榮譽，這才是那年電子競技的殘酷現狀。

而明凱加入的前兩個戰隊，都不是那麼有趣，其中一支戰隊的名字叫”藍BiuBiu”，戰隊圖標充滿了二次元的氣息，這支戰隊只存在了不到半年，後來被Team Phoenix戰隊 帶走了其中大量隊員。

當然，Team Phoenix同樣沒有存在太久，否則今天的FPX戰隊或許還需要換一個名字，在8月加入WE戰隊後，他終於開始有了一份穩定的收入，按照上單草莓在退役後的曝光來看，這份收入並不算高，但作為職業生涯的開始，進入當時國內堪稱豪門的WE俱樂部，無論如何都是一個相當不錯的起點。

起點有了，結果卻未必會太好，S2是一個非常有趣的時代，英雄聯盟在全球尚沒有搭建完整的聯賽體系，大多數戰隊除了奪冠和贊助以外不會有太多收入。而魚龍混雜的賽事又讓大多數比賽到最後都成為了全球選手大亂斗的修羅場。風起雲湧的賽場，日新月異的體系，加上拳頭對於早期遊戲版本和賽事力度都掌控不足，讓WE對整個S2世界賽都無所適從。

八小時的網線大戰耗幹了WE的精力，而道歉在老WE四位選手退役後才姍姍來遲。

當然，這不重要。那年在拉斯維加斯，WE依舊一路過關斬將奪得了IPL末代冠軍，先後擊敗了FNC、CLGEU、M5等多支強隊的他們已經證明了自己的實力，國服客戶端也增加了WE戰隊的永久免費頭像。

在S3那個英雄聯盟肆無忌憚擴張的時候，WE頭像是很多玩家的首選，甚至延續至今。

但對於大多數玩家來說，那個時代的明凱並不出彩，相對於落地金身的若風，韓國隊伍塔下跳舞嘲諷的草莓，乃至於號稱國服第一機器人的捲毛和世界頂級ADC微笑。那個在前期被反野導致大逆風，最終強行頂起400真視藥水強做視野的明凱，確實不夠耀眼。

但良好的開始，也往往是一個枷鎖。

2.

2012的奪冠，給WE帶來榮譽的同時，也帶來了枷鎖，中單實力下滑讓WE無法保持以往的壓制能力，下路的優勢雖然依舊滿滿，但S3是一個刺客橫衝直撞的版本。

除了老對手IG和二隊PE外，WE已經無法擊敗OMG和皇族這兩個對手，春季賽進入季後賽憾負OMG的WE甚至無法擊敗IG這個老對手。被Ban掉寡婦的明凱在野區內橫衝直撞，卻無法保證比賽的勝利，骨子裡渴望勝利的選手們內部很快爆發了衝突。

很多隊內衝突可以掩蓋，但WE不行，因為我們馬上就在賽場上看到了這一刻。

2013年8月16日，LPL夏季賽WE vs LMQTC的比賽中，明凱和若風互換位置，明凱中單，若風打野，但這種臨陣換位置自然不會有什麼好事，作為聯賽末流隊伍的LMQTC輕鬆擊敗了WE並且打出了碾壓性優勢。

這場比賽帶來了兩個災難性的後果。

1.因為被LMQ在常規賽多拿了一分的原因，WE以一分之差未能進入LPL夏季賽季後賽，而未進入夏季賽季後賽則喪失進入全球總決賽預選賽的機會。

雖然進入了也不一定能從當時狀態火熱的OMG和皇族身上搶下名額，但沒進入終究是沒有進入。

2.因為未能進入全球總決賽預選賽，喪失了進入全球總決賽的機會，WE五人內部矛盾進一步擴大，直接導致世界賽期間以及後續WCG相關的事件爆發。

即使是當事人在那時，也很難評價雙方的對錯，在2019年已經稀鬆平常的離隊轉會，在2013年似乎是有些罪大惡極的事情，不管選手私下如何，在微博上掀起的大戰一時成為了年末最”精彩”的風景——即使幾位選手在不久之後就冰釋前嫌。

無論怎麼說，明凱和捲毛以及阿布的離隊，對於當時的LPL來說並不算是壞事，WE的沒落至少在那個時代已經無法阻止，而捲毛、明凱和阿布三人的出走，加上從PE轉會而來的Namei以及曾經作為LM戰隊中單擊敗了他們的U，統治了LPL長達一年半的超級戰隊EDG誕生了。

這支隊伍在2019年確實戰績不佳，但在2014年建隊開始到2018年末，EDG都一直是LPL最具競爭力的隊伍之一，甚至在很長一段時間裡，這個之一也可以去掉。

一手將這個王朝打造而出的正是明凱，而在接下來的時間裡，他依舊會親手掌握這個他打造而出的巨輪繼續前行。

當然，這就是另一個故事了。

3.

2015年的EDG和明凱，都到了歷史上最巔峰的狀態，運營賽季讓EDG的打法如魚得水，而意識和操作乃至於職業經驗都處於巔峰的明凱也開始在賽場上大殺四方。

那年的他強到什麼地步？

LPL聯賽歷史上打野最高擊殺記錄並不是大多數人想像中的瞎子、獅子狗、螳螂、千珏此類英雄，甚至不是皇子、酒桶、奧拉夫這一類戰士英雄，而是豬妹。

我知道這個結論會讓大多數人驚掉眼鏡，但這個記錄確實是明凱2015年4月10日對VG的比賽中創造出來的。春季賽將自己一身實力發揮到頂峰的明凱打出了無數經典操作，豬妹15殺打崩VG、十分鐘連續上中下三路狂抓21分鐘打爆LGD，從有懲戒的努努手下搶到大龍最終讓EDG奪得2015年春季賽冠軍——那也是EDG奪得的最後一個春季賽冠軍。

我們不知道站在佛羅里達州立大學的場館中，這位已經在英雄聯盟里奮戰了四年的職業選手是什麼想法，但最終鎖下寡婦的那一刻，以及對隊員們的堅定眼神和話語，都說明了一件事：

他來到這個地方，目的就是為了屠神。

“We BO5 king，WE ，LGD，NOW SKT”

漫天彩帶落下，EDG最終3：2擊敗SKT拿下世界冠軍，明凱在第一的2015年MSI季中邀請賽冠軍的同時，也獲得了季中邀請賽總MVP，而一直到今天為止，這都是LPL戰隊惟一一次在番戰中擊敗SKT。

2015年季中邀請賽的出征曲叫做《Hero》，是時任IG中單宋義進（Rookie）和時任OMG中單余家俊（Cool）共同演唱。

在MV的最後，音樂結束的時候，明凱對著鏡頭豎起了自己的大拇指。

相對於兩年前他在全明星宣傳的MV當中驕傲的說到”要證明誰才是世界第一打野。”這次他只是和其他隊員一樣，豎起了自己的大拇指。

想要證明自己是世界第一打野，賽場上卻成為了別人的背景板，成就瞎王inSec英名。

平靜的豎起大拇指帶領隊員遠征佛羅里達，寡婦絕殺擊敗SKT，成為真正的第一打野。

但讓很多人沒有想到的是，塔拉哈西的獎盃就已經是很多故事的終點。

EDG和明凱，再也沒達到過同樣的高度。

4.

對於任何一個EDG粉絲而言，2015年MSI後的EDG，無論怎麼談都是一言難盡，在2018年IG奪得S8世界賽冠軍之前，EDG一直是LPL外戰成績最好的隊伍——RNG最多也只做到了平起平坐。

但LPL的頹勢以及EDG在聯賽內部還不錯的成績，讓很多壓力也來到了EDG身上。

“內戰內行”的帽子一旦扣上，就很難再拿下來了，強大的輿論迫使EDG對自己穩健運營的節奏做出改變，在對外作戰採用越來越激進的打法，但賽區的整體劣勢讓絕大多數隊伍在外戰時都堪稱無力回天。

天塌下來自然要高個子頂著，作為LPL”最高”的EDG，被詬病非常正常，而在EDG里”最高的”明凱，各種各樣的節奏自然會接踵而來。4396對於一個劣勢局打野來說並不算奇怪，畢竟劣勢局即使是C位也可能打不出多少輸出，但朗朗上口外加玩家對失敗的不滿，讓節奏迅速發酵，馬踏飛箭雖然堪稱經典一梗，但那場比賽EDG並沒有輸掉比賽，甚至”馬踏飛箭”的明凱，那場比賽還拿下了堪稱完美的1/0/8數據。

但這並不重要，人們總是需要一個發泄口的。

隨之而來的S7全球總決賽小組賽三拿盧錫安被淘汰，讓EDG徹底站在了風口浪尖之上，雖然現在的”御三家”都有小組賽被淘汰的歷史（IG2015年、RNG2019年），但EDG這次的時間和地點都太不巧了一些。連續的成績下滑讓EDG不得不做出改變，年齡已經變大的明凱也逐漸開始淡出賽場。

2018年春季賽尚可算是主力，到夏季賽的後期就基本上由Haro上場，冒泡賽9場比賽的表現讓Haro穩住了自己在世界賽的地位，直到2019年EDG的徹底落寞。在2019年夏季賽，即使是最危機的時候，EDG依舊選擇了新打野Jiejie上場，讓明凱穩穩坐在了替補席上，已經說明了他接下來的去程。

作為一名已經26歲的大齡選手，他已經沒有再去試錯的機會了。

喜歡也好，不喜歡也罷，春季賽季後賽面對SDG那次被3：0，就是他在賽場上留下的最後痕迹，IPL5上逆風頂起400的茂凱，全明星上不知所措的人馬，S4全球總決賽上絕望退場的螳螂，2015年強行搶龍的雷克賽，絕殺大魔王妖姬連勝紀錄的恐怖寡婦，被自己”粉絲”盲僧當成背景板的蜘蛛，讓二追三關鍵局EQ閃挑起對手的皇子，還有那個在最後一場比賽里無力回天的酒桶。

年少成名，打職業第一年就奪得世界冠軍，本來應該是主角劇本。

嘔心瀝血，背負罵名換隊抗韓絕殺大魔王，無論如何都難以忘記。

擁有了該擁有的榮譽，擊敗了能擊敗的敵人，從LPL賽區建立開始走到今天，無論獎盃、榮譽還是知名度，都堪稱LPL唯二存在 。

但我們回顧這一段故事，卻只留下了一段回憶和一聲嘆息。

沒能在職業生涯最後用一場戰鬥為自己退役划上句號，終究不是那麼完美。

5.

18歲的明凱決定走上電子競技這條道路，21歲的明凱用自己的雙手開闢了完全不同的人生。

或許對大多數粉絲來說，”背叛”WE前往EDG是一個不光彩的開始，而獲得榮譽後的描述也更像是成王敗寇的無趣語言。但在那個秋天的晚上他做出選擇那一刻起，渴望勝利的靈魂就已經開始燃燒，成為鑄就冠軍獎盃的燃料。

這條路痛苦、無趣又帶著輝煌，即使結局會有些不盡人意，但正如弗羅斯特那首詩一樣。

「一片樹林里分出兩條路，而我選擇了人跡更少的一條。」

「從此決定了我一生的道路。」

名為選手的人生，已經結束。

名為教練的征途，才剛剛開始。
