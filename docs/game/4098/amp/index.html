12月無盡4000一個！拍賣行翻了40倍，掃拍商人也會按錯

大家都知道，在普雷之後的版本，無盡的永恆這個材料可以說是涼涼了，但依舊有不少商人不死心，在冰點的價格吃入幾百萬！就等著100級或者大轉移2.0重回巔峰，12月份的價格幾乎所有跨區都是100金幣左右，但有一個跨區卻出了一點意外，拍賣行竟然漲到了4000！如果經常搬磚異界的玩家都知道，這價格可是翻了40倍。

​

從背景圖標來看，膠囊寶寶活動在進行，這就是12.10之前的版本，再來看無盡材料的價格，4000一個的天價，並且拍賣行僅剩9頁！經過玩家實測，3000上架被瞬秒，很明顯，這是一些商人在掃拍，但普通玩家很納悶，難道商人們又收到消息，無盡材料會在12.14發布會上大放異彩么，但其它跨區雖然有漲幅，但很小，僅幾十金幣！所以，這是掃拍商人把價格設置錯了。

正常來說，95末期還去囤無盡的商人，只會在40金幣以下掃拍，因為無論冬季發布會出什麼內容，他都不虧，至於4000的無盡明顯是按錯了……跟玩家上架的時候一樣，但大家是少按一個0，而掃怕則是多按0。但這次可便宜了玩家，出來混總是要還的！本來100的無盡，玩家們紛紛3999上架，上架1組就白得700多W金幣，堪比光速搬磚模式。

可惜，掃拍商人沒多久就沒金幣了……因此，拍賣行的無盡材料價格也回歸了正常，這是發生在跨3A的事，其它跨區只能羨慕，只能說，搬磚的小夥伴們還是經常要看一看拍賣行的材料，要是魔剎石、誕生之芽也跟無盡材料一樣出現這種情況，2020的春節套根本就不會發愁去哪搬磚了。
