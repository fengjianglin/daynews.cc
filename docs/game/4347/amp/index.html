大家好，這裡是愛吃雞的小女孩槐序。

香蕉計劃邀請賽可以稱得上是絕地求生職業化以來，國內舉辦的第一屆賽事，不少職業選手都藉由那次比賽正式出道。

雖然那次比賽採用了線上賽的模式，賽事的規則也不如現在完善，但它對於絕地求生的影響力依然不可忽視。

最近有網友翻出了當時香蕉計劃邀請賽的選手大名單，可以看到參賽的選手林林總總一共有236位（不算替補），其中不乏大家十分熟悉的NB211、12月、圖拉夫、星魂等知名選手或是主播，也算得上是盛況空前。

不過兩年多過去了，再回首時卻難免有些唏噓，因為其中一半以上的選手都已經離開了職業賽場。

比如圖拉夫和星魂現在已經是全職主播了，A+大哥也已經退居二線專心當起了戰隊的老闆，Mu神回到了DOTA2。

當然這些只是大家比較熟悉的名字，更多的像天祿的Icegod、RNG（當時的RYL）的Jiyun等選手可能已經被大多數人遺忘了。

選手當然也不想放棄，只是這兩年絕地求生的境況確實不如人意。

新IFTY的四名隊員全是年輕人

就拿剛結束不久的PGC全球總決賽舉例，2018年PGI時觀賽人數一共有75.9萬（不計算中國觀眾），一直到今年的PNC全明星賽都還有47萬的樣子，但到了PGC就只剩下18.6萬了。

僅僅一年過去，觀賽人數就下降了76%之多，絕地求生未來的走向不禁讓人感到擔心。

但也有細心的網友在大名單的一角發現了熟悉的ID——Godv。

當初韋神從英雄聯盟轉型到絕地求生時並不被人看好，畢竟這是兩款截然不同的遊戲，韋神此前似乎也沒有FPS經驗。

相比於韋神，4AM（當時還叫YQHP）隊內更受關注的是戰神NB211，此外當時參賽的17shou、小絕等人氣FPS主播比起韋神也更被看好。

然而當鉛華洗凈，流年逝去，當初的許多天才都一一隕落。

17shou和他的SMG陷入泥潭、NB211差不多到了退役的年紀、大多數戰隊的成員都換了一茬又一茬。

只有那個叫Godv的小胖子還始終一如既往地前進著，甚至差點站上了世界之巔。

這當然與韋神的天賦有關，但大家也不能忽視了這個男人的努力。

這是一個能在訓練場里枯燥地練槍練兩個小時的男人，這是一個能夠日復一日地堅持到CS:GO中完成爆頭數作業的男人，這是一個能帶著「中耳炎」等一身的毛病堅持出國打比賽的男人。

那些比他有天賦的人都沒他努力，那些比他努力的人卻又少了一分靈性，所以Godv成功了，而別人沒有。

不過當初三幻神之一的小龍都熬退役這麼多人了還沒成年，那以後的絕地求生豈不是屬於……
