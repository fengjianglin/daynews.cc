厄運東挖奧水是否被暗改？《魔獸世界》懷舊關於掉率的幾個傳言

天天要聞
2019-12-20

小編獵人常年混跡厄運東，和完美厄運不同，這裡沒有穩定產出，主要收益是看人品的挖礦，搏奧術水晶，就是打完最終BOSS出來小鬼群之後進洞的2-3個富瑟銀礦。但從最近更新開始，明顯感到了奧水掉率的下降。獵人雖然效率不如術士，但之前幾趟下來也能1個奧水保底，而最近幾天挖了幾百鏟，竟然不見其蹤跡。厄運頻道也有人刷屏，說挖不出奧水了，難道是暴雪真的暗改?下面來詳細討論，附帶八下懷舊服感人的掉率問題。

暴雪暗改

暴雪一直有暗改的傳統，之前爐石的競技場，就有多次調整單卡出現概率的先例。但網遊不同，即使小的修復或調整，至少會有論壇回復(如DK劍的改動)，多是明改。奧水是和黑蓮花一樣的硬通貨，很多關鍵道具的必備高級材料，在最近開轉服之後，價格又有大漲。而術士單刷平均15分鐘一次，獵人慢的也有20分鐘一次，許多其他職業如薩滿、牧師也能單刷，每天無數趟的厄運刷礦，資源過於飽和，這是暴雪不願看到的。

除了奧水，費伍德的劇毒水，希利蘇斯空氣精華(期望90比1)也是暗改傳言的熱門對象。

但暗改這種東西即使有，也永遠無法明證。

動態爆率

有人提出動態爆率的概念，就是不同伺服器、不同天數，不同時段的掉率，都是不同。所以單一樣本，單個時間段的統計都沒有意義。按照過往術士們的公認情況，奧水掉率平均2.8%，在3%左右波動，普遍情況是，多的時候大家都多，少的時候大家都少，多少證實動態爆率的存在。不知道有沒有程序員冒個泡，看看如何實現。

另外，大量60版資料庫的掉率參考都不可信，因為諸多數據來源私服，例如N服，而私服普遍都有爆率的調高。

例如下面屠龍概率，水元素和刺蹄都達到1.5，總之，我是不信的。

這次更新之後的群體哀嚎，只是近2天情況，還需進一步觀察。

流言1-人品守恆定律

意思是前面挖得太多了!

人品守恆涵蓋所有遊戲，多數只是聊以自慰。例如一個角色一個星期刷了三本屠龍，下星期不出的概率很高。所以，每當一次性吃得過飽，大家會自嘲後面的還債。

流言2-技能點越低越容易出

傳言，挖礦技能等級越低，出奧水的概率越高，最好是剛滿足挖富礦，再來個連續失敗就更好，純心理作用。

流言3-產出CD

這個說法是：一定範圍單位時間內的資源產出總量是有上限的，例如1個小時內，大家一起挖，最多只會爆出多少塊水晶，但這種情況多見於低端頁游。

人見人刷的厄運東

流言4-墊刀一說

墊刀一說可能是網遊最大的流言，意思是你自己沒挖到，其實給別個挖到的墊了刀，作了犧牲品。

流言5-越刷越爛

很多時候，在野外打怪，常有怪物「越打越窮」的印象，且怪物刷新越快，掉落越差(類似新怪需要養一養才會肥)，有人猜測是遊戲防止惡意刷怪，進行了掉落的保護。不過這也僅僅猜測而已，並且副本情況可能不一。

​最後，真正的臉黑黨，早已看淡一切……例如小編就當屠龍不存在。就是刷個深淵競技場，也近200次，老高碰面無數，野蠻角鬥士散件積攢一堆，唯有不見衣服(自用)，而有人帶老闆平均一天一件。

當然，最好的防臉黑方法，就是切換位面，例如挖厄運東，連續水晶不出，讓別人組你切個位面，可能就出了，都是玄學，可以試試。
