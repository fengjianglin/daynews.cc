2020年春，將是LPL的序幕

2019-12-24 53條評論 1,226次閱讀 174人點贊

在即將邁向2020年之際，相信許多小夥伴回顧2019年一定會有許多的感嘆，這一年的LPL一直都備受質疑。這一年可以說是LPL最有意義的一年，可以說這一年已經可以品味到LPL的五味雜陳，巔峰、低谷、奪冠。這也是之所以被稱之為最強賽區的原因，不經歷風雨怎麼能見彩虹？不畏前險怎麼能喊出那句話：我們是冠軍。這一年有太多的故事，就讓我們來回顧我們的夢，LPL的夢。

巔峰

FPX崛起，新老交替

趣加電子競技俱樂部（FunPlusPhoenix）成立於2017年，並且宣布進軍LPL。但在2018年的時候，表現平平一度處於LPL中下游。（2018年LPL夏季賽，以8勝11負西部第4成功闖入季後賽，9月6日，在第一輪季後賽比賽中1：3不敵JDG，結束夏季賽之旅。）但在2019年，卻以不同的姿態回歸。在2019年的時候FPX做了兩個事情，一、強化自己的品牌建設 二、引進選手並重新構建FPX核心，最為知名的就是Doinb和Tian。可以說，這兩個操作間接也讓FPX不僅在名聲上和實力上得到了很大的提升。並且以13勝2負的高姿態獲得常規賽第一名進入春季賽。雖然無緣決賽，但它的意義是非凡的。不僅僅FPX這樣的新隊伍在春季賽脫穎而出，TES、DMO、JDG等，在春季賽表現也可圈可點。這也讓LPL傳統豪強隊伍無一不動容。

JDG黑八衝出，勢不可擋

JDG在本次的春季常規賽表現並不盡人意，由於2018年的冬季轉會期也讓JDG失去了原本隊內兩大強力，而imp的年齡問題也讓許多粉絲懷疑JDG在春季賽能否站住腳跟。經過春季賽常規賽的跌跌撞撞，終於以常規賽第7的身份進入季後賽。然而根據春季賽季後賽的規則，JDG必須連斬3支隊伍才可以進入決賽。這對原本就在常規賽磕磕碰碰的他們異常艱難。然而就在被所有人不看好的情況下，神擋殺神佛擋殺佛。連斬了WE、RNG、FPX順利進入決賽。這也是LPL有史以來最大的黑八奇蹟。雖然飽受質疑，但JDG沒有放棄。

IG奪首冠，狀態無可媲美

在春季賽的開賽初期，IG仍然保持的超高的水準。可以說IG在這次的春季決賽中，仍然帶給我們的是那支冠軍隊伍應有的實力。本來大家都在期待是否能上演「黑八奇蹟」，然而iG戰隊絲毫沒有留一絲機會給JDG戰隊，以摧枯拉朽一般的態勢3比0碾壓了JDG，並且前兩場的比賽用時均不超過30分鐘，iG果然是永不加班，最後iG也如願捧得春季賽總冠軍獎盃。可以說，IG的狀態在這個時候是無可媲美的。在獲得春決冠軍的同時，也意味著獲得了MSI季中賽的資格，這也是能再次檢驗LPL賽區是否是第一賽區的第一道測試。

低谷

MSI被意外橫掃

在MSI的小組賽階段，IG仍然延續著火熱的手感。在小組賽階段，IG幾乎仍然以最強戰隊的實力打敗了眾多地區的冠軍隊伍。雖然在小組賽最後一場的比賽中輸給了SKT，但這並不影響IG以小組頭名的優勢穩坐出線寶座。然而，就在小組賽之後的淘汰賽中被TL戰隊連下兩局，幾乎可以說IG的狀態這時候已經發生了變化。原因就在於前兩局的遊戲中可以說IG是以巨大的優勢被TL強勢逆轉了，而第三局雖說贏了，但卻是以IG最不熟悉的陣容贏下了比賽。就在人們都以為IG的狀態調整回來的時候，第四局卻直接被一波打懵。

洲際賽再失利

在過去的兩屆亞洲對抗賽中，LPL賽區都在決賽力克LCK賽區，達成了兩連冠。隨著第三屆亞洲對抗賽來到韓國首爾舉辦，今年代表LPL出征的四支隊伍也發生了巨大變化，iG、JDG、TES、FPX，有三支隊伍都是第一次參加國際比賽。洲際賽的參賽規則也是通過兩個賽區中春季賽表現優異的前四隻隊伍組成。但，就在洲際賽的開賽初，LPL賽區就出現了巨大的問題。雖然在面對LMS賽區和CSS賽區表現優異，但在面對LCK賽區的時候。除了IG的狀態達標外，其他三支隊伍幾乎沒有找到自己的位置，可以說是被LCK隊伍壓制了。不僅如此，在接下來的決賽中，iG與TES先後敗北，只有FPX戰隊打出了在LPL的戰術水準，為LPL拿下一分。JDG戰隊最後登場，在前期有較大優勢的情況下遭到DWG逆轉，最終LPL賽區1:3落敗，首次無緣洲際賽冠軍。

IG狀態低迷，隊內出現矛盾

回歸夏季常規賽後，IG狀態出現了一些很大的變化。與春季賽相比，他們的排名一落千丈。直到小組賽最後一日，才真正得到了進入季後賽的門票。在季後賽，在所有人都認為IG可以擊敗LNG的，但卻直接被LNG以3-0的巨大優勢直接淘汰出局。這三場比賽中，LNG幾乎都是以巨大的優勢完成的對局，可想而知IG的隊內氛圍出現了巨大問題。之後，也被爆出IG上野出現了很大的矛盾，一度爭鋒相對。而這樣的劇情卻一直持續到S賽季…

EDG出乎意料，傳統豪強沒落

EDG這個2019可以說是非常艱難的一年，在春季賽上艱難出線，但卻被DMO在首輪直接淘汰。在夏季賽上，雖然狀態有所調整但在常規賽上卻早已不見鋒芒。在夏季賽上，EDG的確調整回了狀態也在常規賽的最終排名以小組第五的成績進入季後賽。然而這對於傳統豪強EDG來說，這對於滿足觀眾的期待感顯然是不夠的。在季後賽的首輪，EDG的4局遊戲仍展現出了不凡的實力，就在所有人都意外的覺得EDG要回歸了，二輪卻直接倒在了BLG上。這也代表著，EDG在誕生後，首次無緣S系列賽。

FPX奪冠，仍不能削減質疑

在經過春季賽季後賽的慘淡收場後，FPX繼續在夏季賽調整自己的狀態。不僅延續了在春季賽常規賽的火熱手感，在夏季賽更是一舉拿下七連勝的戰績。並且這個記錄是英雄聯盟常規賽有史以來最大的連勝紀錄，兩個賽季以27勝3負的戰績證明了自己的實力。並且在夏季賽季後賽，FPX終於打破了「偽強隊」的質疑，3:1戰勝BLG晉級決賽，又在夏決中3:1戰勝RNG，奪得2019LPL夏季總冠軍。然而，這一系列還是不能打破許多LPL粉絲對於今年LPL的質疑，或許LPL不該稱王吧。

重生

小組賽艱難出線

雖然說，LPL的三支隊伍都順利進入了S系列賽，但賽制是殘酷的。抽籤分組上，FPX相較於IG、RNG來說是最好的一組簽。幾乎不用碰上目前仍然很大奪冠熱門的幾支隊伍，然而IG和RNG這裡的情況就不同了。RNG和IG被分到了C和D組，這兩組中分別有奪冠的熱門SKT、DWG、FNC。RNG被分到C組，也就是死亡組。這組裡實力強勁，也是檢驗RNG是否是一隻真正強大的隊伍。然而IG和RNG卻都遇到了生死戰這一情況，但RNG卻在生死戰輸給了FNC，而IG這裡則是擊敗了TL艱難出線。到這裡，許多LPL粉絲都認為今年的LPL是偽強賽區，就連歐洲賽區都打得這麼費力。然而淘汰賽上，IG和FPX證明了實力。

重生，LPL內戰

在淘汰賽的抽籤階段，無疑不是觀眾最愛的環節。目前LCK有三隻隊伍進入了半決賽，這也直接證明了LCK賽區的實力，並且LEC在這次半決賽中也誕生了三支強力的隊伍，這也讓人們更加相信LPL今年的處境異常艱難。在抽籤上，IG抽到了LCK的內戰王者GRF、FPX則是對陣淘汰了RNG的FNC。可以說，這兩局淘汰賽頗有一番戲劇性，一邊是LCK飽受爭議的強隊，一邊是淘汰了RNG的歐洲賽區。兩邊都充滿了看點，然而FPX這裡卻以碾壓的姿態取得了勝利。反觀IG這裡，可以說IG在四局的對局中已經在開始找回自己的狀態，Ning王的回歸可以說是最大的亮點，上野的矛盾也在默契中無形化解，可以說IG帶來了一把真正的視覺盛宴。在雙方都以3-1的方式淘汰隊伍時，對LPL觀眾來說也是最為難過的。內戰一觸即發，誰將問鼎全球總決賽？相信誰也不敢說，因為IG回歸了，那支給人帶來奇蹟的隊伍。

FPX問鼎，新王登基，LPL第一賽區

在擊敗了IG之後，可以說這支隊伍再次獲得了新生。首次參加全球總決賽的FPX戰隊五名選手也都是首次參賽，可以說隊內的所有成員都沒有參加過S系列賽。甚至於Doinb僅僅只限於有一些國際賽事的經驗，更多的還是他們自己的心態以及調整。然而面對的G2，卻是常年駐紮S系列賽。並且在今年G2完成了一系列的征戰，並且就只差一個S世界賽冠軍就可以完成歷史上的大滿貫。即將達成大滿貫、歐洲主場作戰、五位歐洲頂級選手，坐擁天時地利人和的G2戰隊賽前奪冠的呼聲極高。連全球總決賽三冠王SKT都敗在了他們手中。FPX在輿論不看好的情況下，面對這樣一支隊伍，FPX可以說初生牛犢不怕虎了。用三局遊戲乾脆利落的橫掃打懵了G2戰隊，打醒了歐洲觀眾的美夢，也為LPL賽區衛冕了全球總決賽冠軍的榮譽。捧杯吧，FPX！

這一年，巔峰、低谷、重生，就是LPL賽區的一年。這就是最強賽區，這就是我們雖熱愛的地方，它叫LPL。
