《陰陽師：百聞牌》：網易年底最大底牌？

遊戲平台：手機

陰陽師自走棋《陰陽師：百聞牌》上線了！不僅經典式神擁有全新原畫，效果卡也都有著獨立的設計與風格。與大多數CCG遊戲不同，《陰陽師：百聞牌》沒有採用「以單職業為核心」的組卡思路，而採用了牌手1V1對戰，多式神戰鬥區輪換作戰的特色系統。

除PVP對局玩法外，《陰陽師：百聞牌》目前已經推出2種PVE玩法，分別是「秘聞之間」和「怪談研習社」。在「秘聞之間」中，玩家可以在通關中體驗《陰陽師：百聞牌》原創的衍生劇情，感受全新平安世界。「怪談研習社」則在傳統玩法中加入Roguelike元素，每冊怪談皆為獨立故事，在遭遇不同的戰鬥或者事件中，堅持的越久沒獎勵越豐厚。

玩家評論：

——畫風跟遊戲機制都非常的卓越，絕對可以成為存活度極高的tcg手游。問題也很簡單，扣兩星給獎勵機制。給玩家的正反饋太少了——這不是遊戲設計的問題，而是運營的不同策略。

——這畫風！愛了！這機制！愛了！這氪金程度！不愧是網易！除了雙貨幣系統感覺有點氪金以外，其他的都挺好的。

《戰雙帕彌什》：好遊戲毀在了運營手裡

遊戲平台：手機

《戰雙帕彌什》是一款末世科幻題材的3D動作手游。作為一款二次元動作手游，遊戲除了出色的畫面和動作設計，還將三消玩法加入了動作連招里，算是頗具新意和實力的一款動作手游新貴。玩家將化身指揮官，帶領人類最後的希望，共同對抗被帕彌什病毒感染的機械大軍。

遊戲本身是個好遊戲，不過遊戲一開服就發生了送1w黑卡並炸服的運營事故，雖然官方也進行了補救，但對遊戲口碑依舊造成了很大影響。可以說這是2019年Q4最受關注手游犯了2019全年度最經典的手游運營錯誤，值得同行警惕。不過小弟只想和大家說，這件事告訴我們，老祖宗的話你得聽。

玩家評論：

——上線前，同行競品都在做主動防禦，搞活動的，搞營銷的，生怕被搶走用戶。上線後，同行都在搞聯動，這家發1w金幣，那家發1w金幣。

——發生這種運營事故，收不收回都是很尷尬。但是遊戲本身還是個可以和崩三一拼的遊戲，有一說一，這次的打擊感還是很可以的，動作出招流暢，操作鍵控不卡手，真的很有進步。

《灌籃高手》：多發補償還封號？

遊戲平台：手機

《灌籃高手 正版授權手游》是一款真人實時籃球競技手游，東映動畫授權並全程監製。遊戲以井上雄彥原作漫畫、東映動畫製作的同名動畫片《灌籃高手》改編而成，玩家可操控原作出場角色,習得技能,體驗原作動畫的經典劇情和各個高校球隊帶來的強大實力，感受競技賽事帶來的熱血與感動。

遊戲也在開服當天發生了運營事故。雖然沒有因為人數過多導致炸服，但遊戲由於官方的失誤操作，導致IOS一區的玩家補償發了58500鑽石的巨額補償。於是開服第一天官方就立即關服維護，強制收回給玩家發出的58500鑽石，但是拿到獎勵的玩家不知道為何也拿到了416天的封號。後續官方說明這只是臨時封號，也給出了補償方案，但是，果然12月5日忌開市是真的！

玩家評論：

——說實話一開始挺害怕人物會是Q版的，剛進遊戲看到女神晴子精緻的立繪和應景的BGM本來稍微鬆了口氣，進對局看到看到大頭娃娃有些失望的，但是吧玩起來手感竟然還挺好0.0.而且還原度還挺高，嗯，對沒錯我真香了。

——拋開遊戲運營事故來說，遊戲的大招動畫太熱血了，而且代入感很強，真有種在打比賽的緊張感，老是不進球也是會急得要命的。目前感覺還是很好玩的，不是那種無腦遊戲，會防守、搶斷和掌握蓋帽的時機很重要的，貌似技能也是能帽的。

《電競傳奇》：感受一把專業電競比賽的樂趣

遊戲平台：PC

如果你當不了一名電競選手，你還可以選擇當一名電競經理人啊！《電競傳奇》就是一款以電子競技為題材的模擬類遊戲，在遊戲中玩家將作為經理人，管理一家初創的電競俱樂部。遊戲目前從steam移植到了手機，可以在手機上遊玩啦！

在經營途中，你將需要從創建俱樂部起手，並通過訓練選手，參加聯賽等方式將手中這個名不經傳的初創業餘戰隊培養為稱霸世界大賽的頂級強隊！遊戲對LOL的模擬比賽還是很到位的，可以選擇不同賽季比賽，比賽中雙方也會偷塔也會打野，還會有人心態崩了送人頭。

玩家評論：

——玩這遊戲有點尷尬的是沒玩過LOL不知道裡面的英雄，然後有點搞不懂不過遊戲是真的挺好玩的！推薦推薦

——總算出來了，比pc版感覺精緻。玩起來停不下來，培養明星選手後，看到他們在比賽中大殺四方，拿下比賽會，滿滿的成就感！繼續肝去了

《鳳凰點》：貼臉能射準的《XCOM》

遊戲平台：PC/PS4/XboxOne

《鳳凰點》是一部開放世界題材的回合制策略遊戲，是一款在《幽浮》系列經典設計之上，做出了亮眼創新的戰略遊戲。本作中玩家需要完成進攻怪物、救援、搶劫等任務一系列。某些大型任務會要求玩家進攻怪物的大型巢穴，搶奪怪物的資源物品。

遊戲的難度較為硬核，有著偏高的門檻與較長的上手時間，但也因此產生了很高的策略性與可玩性。雖然仍有部分細節處理略顯粗糙，不過戰鬥與經營的框架已經搭建完善，之後的DLC更新也會追加豐富的後續內容，《鳳凰點》的未來表現值得期待。

玩家評論：

——還是熟悉的配方，不過可以戰場上撿箱子換裝備，然後有了個意志點。翻譯文字有顯示問題，也是這個意志點顯示重疊文字了。

——還不錯，有老幽浮那味，因為是眾籌作品，所以體量上也就不要求那麼多了，總的來說還挺好玩的
