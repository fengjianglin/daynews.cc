No.2

今日陣容推薦

今天給大家推薦的這套陣容

在9.24版本剛剛加強過

俗話說的好 一代版本一代神

趁現在肯定要恰一波雷霆的分呀！

推薦陣容：雷霆狂戰劫

推薦英雄：荒漠之咬-雷克頓、山隱之焰-奧恩、武器大師-賈克斯、雷霆咆哮-沃利貝爾、蒙多醫生-蒙多、影流之主-劫、亡靈戰神-塞恩、狂戰士-奧拉夫

推薦裝備：奧拉夫-飲血劍（大劍+斗篷）、羊刀（大棒+反曲弓）、復活甲（大劍+鎖子甲）；劫-復活甲（大劍+鎖子甲）、救贖（腰帶+眼淚）、離子火花（大棒+斗篷）

組成羈絆：6狂戰士3雷霆2極地（如果有雷霆拉克絲可以換下奧恩，組成4雷霆）


運營思路：

開局選秀優先順序大劍>大棒>反曲弓，裝備優先合成復活甲和羊刀。前3波小怪打完後直接升人口，推薦奧恩、狗熊、鱷魚、武器組成3狂戰士2雷霆，沒有的話也可以用常規的森林德魯伊、3掠食者等等。

一般來說有雙雷霆基本可以連勝到第一次選秀，選秀完可以升5人口，上其他的狂戰士即可。打完石頭人可以晚幾個回合再升6，需要保證升6後身上還有30塊錢左右。

7人口主要是找二星奧拉夫，其次就是湊齊6狂戰士2雷霆2極地的羈絆（就是上齊除了劫之外的7個英雄）。如果質量都差不多2星後就可以考慮升8，之後再找劫就OK啦。

如果8人口一直不來劫，也可以把奧恩替換為煉金+老鼠湊3劇毒，或者奧恩替換EZ+布隆，也就是昨天推薦那套陣容。雷霆在下個版本即將削弱，如果你還沒有嘗試過的話，要抓緊時間啦~
