雲頂之弈：絕境中湊出超豪華陣容，殘血逆襲打到對手要求1V1

天天要聞
2019-12-20

在雲頂之弈中，有部分陣容比較難湊，比如9光、9地獄火等等，這些陣容屬於豪華難湊的陣容，戰鬥力未必有多強，但稀有度卻非常的高，所以喜歡湊這種豪華陣容的玩家，還是比較多的。比如小編最近就在湊9光系魂陣容，中間也是感觸良多。

9光陣容的難點

因為盧錫安的加入，讓光之陣容多了一員，於是9光陣容的難度下降了一些，只要湊齊2個光之護符，即使沒有拉克絲，玩家也能湊出9光了，還能搭配系魂陣容，真是美滋滋。但中間的過度就會比較困難，尤其是沒能拿到鏟子，湊出光之護符的情況下，立馬就會面臨絕境。小編有一局就是過度不行，中期就涼了，也是悲劇。

核心英雄的裝備

9光配系魂陣容，核心英雄自然是系魂2人組，他們是陣容的核心輸出。盧錫安是前排用來犧牲的，而賽娜則是必須要保護好，只有賽娜活著，盧錫安才能不斷輸出，所以小編給賽娜帶了一個復活甲，這也是非常關鍵的，後面合成了光之護符，肯定也是要給賽娜帶上一個的。

除了賽娜之外，就是盧錫安了，這個英雄往往是放第一排，作為靈體來戰鬥的，而小編通過測試發現，盧錫安變成靈體後，裝備的加成還是存在的。於是，小編就帶上了一個巨人殺手，之後盧錫安戰鬥力爆炸，1星的輸出高達3000多，一躍成為全隊主輸出，還是最安全的輸出，只要賽娜不死，盧錫安就是最強的仔！

光之護符優先給賽娜

這是重點要說的，那就是賽娜一定要帶上光之護符，這是她存活的一大保障。因為9光陣容需要9級才能湊出來，而小編這局達到8級後，已經湊足了9光需要的英雄和裝備，但等級還差一些，所以只上了8光。

在此之前，小編也是被吊打了好幾次，非常擔心撐不到下個回合。關鍵時刻，小編將光之護符給了賽娜，讓這個英雄存活了更久了，於是陣容的戰鬥力有了飛躍地提高，順利的贏了一局。所以，賽娜帶光之護符，還是非常重要的。

陣容成型

在經過一波的努力後，9光雙系魂的陣容，終於是湊齊了，而小編還配上了一個劍聖，跟劍魔組成了雙劍組合。這個陣容中，還有幾個1星英雄，3星更是一個沒有，但戰鬥力強的可怕，每死一個英雄，剩下的光英雄就可以恢復25%的血量，還能提升35%的攻速，再配合救贖，讓每個英雄都能存活更長的時間，並且輸出都很強。

對手要求1v1

小編在陣容成型後，也是依靠9血一路逆襲，很快打入了決戰局。對手是一個3雷霆加3劇毒的組合，整體的戰鬥力也很強，但面對夢幻的9光系魂陣容，依然是不夠看，直接被吊打了。雖然對方有54血，但依然無法逆襲，後面甚至喊出了1v1的話。對此，小編也是無視了，明明佔據絕對優勢，誰沒事跟你1v1。

超夢幻10光陣容

在最後穩贏的情況下，小編居然抽到了3個光之拉克絲，劍聖也升2級了，於是小編直接下了奶媽，上了光之拉克絲，組成了最夢幻的10光系魂雙劍士陣容，這樣的陣容真是太夢幻了，戰鬥力也是強無敵，輕鬆放倒了對手。

總結

小編最後還是9血逆襲了對手成功吃雞了，而這個陣容的強度是不用多說的，戰鬥力絕對的強悍無比，但缺點也很明顯，那就是裝備的限制。沒有拉克絲的話，就需要湊出2個光之護符，才能讓陣容成型。

這個陣容如果湊不出9光的話，戰鬥力還是差不少，所以在中期的時候，如果不能湊出光之護符，建議放棄這個陣容吧！那麼，小夥伴們對此有何看法呢？
