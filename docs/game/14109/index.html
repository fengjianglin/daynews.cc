LOL：鱷魚最大親媽出現？無視E技能單挑穩贏，玩家直言想掛機

2019-12-11 752條評論 1,817次閱讀 408人點贊

英雄聯盟至今有146位英雄，每一位英雄的技能都不相同，鱷魚就是其中之一，說道上單霸主，想來大家心裡想到的就是鱷魚了，這麼久了，一直都是上單對線T1級別的，很少有英雄能夠跟他SOLO，特別是在前期的時候，鱷魚之所以這麼強，就是因為有位移、有爆發、有控制，機制相對來說是非常完美的一個英雄，而且滾雪球的能力非常的強，只要被單殺一次，那麼鱷魚就能夠無限越塔了。

得到了六級的時候，就算是叫打野也無濟於事的，因為到了六級，鱷魚的大招能夠加傷害加生命值，最重要的是能夠加怒氣，滿怒氣的鱷魚，秒人真的非常的輕鬆，而玩鱷魚最大的細節就是對於怒氣的把控，想要玩好鱷魚就必須要把控好怒氣的量，有怒氣和沒有怒氣的鱷魚是完全不相同的，傷害不在一個檔次，而且在這個版本中，因為青龍刀和征服者的存在，在後期的時候鱷魚一樣非常的強力，那麼要用誰才能夠針對鱷魚呢？

鱷魚最大親媽出現？那麼這位英雄到底是誰呢？她就是奎因，奎因是一位射手英雄，但是一般都不會拿去打ADC的，一般都是走中路或者上路，在上路因為都是短手，奎因可以壓制大部分的上單，而且奎因的技能非常適合打短手英雄，E技能能夠將敵人踢開，並且自己會後退一段距離，防止敵方英雄近身，而E技能也是打鱷魚的核心技能，在鱷魚E的時候奎因也可以利用E技能來打斷鱷魚，沒有E技能的鱷魚就是一個拔了牙的鱷魚，對於奎因來說沒有任何的威懾力。

而且因為手長的緣故，只要鱷魚上來補兵就可以平A或者Q技能來消耗鱷魚，對面的鱷魚將會玩得非常的難受，Q技能還能夠致盲鱷魚，鱷魚的W技能用不出來就非常的難受，並且奎恩的大招能夠給他帶來長時間的高額移速，趁著上路兵線進塔的時候，可以去到處遊走，將自己的優勢擴大到隊友，鱷魚只能眼睜睜地看著，一點辦法都沒有的，等鱷魚吃完塔兵之後，奎因又回到了線上繼續壓制。到了後期的時候，奎因可以選擇單帶來牽制鱷魚。

能夠使得鱷魚心態爆炸，不過奎因最值得注意的就是防止Gank，畢竟是一位脆皮沒有位移的英雄，被Gank的話，容易交出性命，特別是不能讓鱷魚拿到人頭，不讓鱷魚會快速的做出布甲鞋來針對一手，被裝備領先的話，奎因就打不動鱷魚了，需要在線上多做好視野，多利用W技能探視野來監測周圍的動向，看看周圍有沒有敵人。
