人人皆可帶寵，盤點《魔獸世界》懷舊服各類召喚道具

2019-12-24 1,145條評論 4,509次閱讀 3175人點贊

60版本百花齊放的的年代，不少新奇的道具給人留下深刻的印象，特別是一些召喚類，人人COS獵人，享有帶寵樂趣。下面盤點詳細。

召喚須有戰鬥能力，至少有用，純寵物類不計在內。另外，幾率觸發的召喚，如黑劍、龍之召喚，左右狗爪等，也不統計。

一、飾品類：

1.機械幼龍、秘銀機械幼龍、奧金機械幼龍

工程三龍，冷卻時間1小時。窮人只能用前兩個，因為終極的奧金版，材料需要8個奧金錠，僅這一項就差不多600多G了。

普通龍只作玩具，奧金幼龍是高端團隊的秘密武器。一些極限競速的BOSS或副本，如安其拉公主、NAXX，奧金幼龍都有不錯的發揮。其特效buff和燃燒平原的精英法師龍一樣，增加目標受到火焰法術的傷害，多層疊加，全團有效。從安其拉開始，法師全面轉火法，如有多人攜帶奧金幼龍，將對團隊整體DPS提升很大。

奧金幼龍的圖紙來源冬泉谷麥瑟瑞爾的精英法師龍人，常年有人在此狩獵藍龍肉，產出較多，售價一般。

2.便攜火炮

斯坦索姆血色區的炮手威利掉落，個頭較大，但持續時間非常短，只有10秒，會自毀。

3.巴羅夫管家鈴

60版最經典的飾品之一，20分鐘CD，召喚3個NPC打人很疼，被廣泛用於PK，特別是對抗獵人的陷阱。

另一大用處是打安其拉的公主，因為相當於一次性召喚3個目標，在沙包大量陣亡之後，能起到一定的候補作用。

4.木喉防禦者

CD10分鐘，召喚的熊怪持續20秒，PVP實用，但獲取難度相當高，需要木吼聲望的崇拜，而木喉算是60年代最難沖的聲望之一。販售需要後續版本的開放，大概就是15敏附魔的時間。

熊怪戰力不錯，單次加血600點，閃電箭傷害400點，可能觸發暴擊。突然地召喚也能亂掉對方陣腳。缺點是較脆皮，碰到盜賊可能幾刀就切了。

5.克蘇恩的觸鬚

賣萌專用，小克掉落最差道具之一。

CD時間3分鐘，持續30秒。單看觸鬚的傷害不低，攻擊頻率1.5左右，打布衣一下200左右，血量還高達6000點。但這東西最大缺點是不能動!只能呆在原地，還是近戰!(什麼效果都吃，包括恐懼、冰凍等等)。

最後，補充一個薩滿50級的職業任務飾品，「被迷惑的水之魂」，圖騰也算召喚的一種。水之魂回藍效果可觀，缺點是開放版本過後。

二、副手類：

1.上古角石魔典

黑龍最臭名昭著的掉落之一，大眾玩具，強X界的翹楚。

魔典用來插旗還可以，召喚正經的3000血骷髏，1分鐘碾著人打，一拳大概100點，不吃恐懼不吃羊。

而副手類的好處是，戰鬥中也可以更換!

2.亡者之書

縮水版的上古角石魔典，斯坦索姆血色區的關底巴納扎爾產出，掉率非常非常低。和魔典一樣都是15分鐘冷卻，持續1分鐘的作戰，但沒人看出與魔典骷髏的區別，因為刷到的實在太少。

三、消耗類

1.狗哨

血色修道院的馴犬者洛克希掉落，召喚血色狗，等級30，能用3次，持續時間高達10分鐘。

在懷舊服的美服內測版，鬥魚的兩個主播黑歡喜、棒老三曾經在凄涼之地偶遇，上演一段精彩PK，因為封頂40級，所以當時老黑召狗助戰的時候，戰力還相當可觀。

2.尖刺項圈

暮色森林烏鴉嶺墓穴里的稀有精英瑪拉索姆公爵掉落，召喚地獄犬，能用1次，COS術士，召喚的地獄犬等級較低。

艾薩拉有個尋找礦脈的任務，任務物品也能召喚地獄犬，但那個沒有作戰能力。

3.機械格林奇

10分鐘CD，持續10分鐘，獲取方法是在聖誕節期間做完任務，等到24號晚上拿到禮物，開條紋禮物盒獲取。

4.機械雪人

冬泉谷的經典任務，雪怪計劃讓你跑去三個地方去嚇唬人。也是很坑的一個任務，機械雪人打人會進戰鬥，經常有人放出之後，被衛兵群毆，然後一臉懵逼。

另有一種「安靜的機械雪人」，是工程製作的寵物，不能打人。

5.蛛怪之眼

斯坦索姆蜘蛛掉落，效果是召喚和術士一樣的眼睛，用於偵查。拾取綁定，只能用1次。

6.閃光的獵豹雕像

聯盟黑海岸的幽靈豹掉落，能用1次。召喚出來的幽靈豹能力不錯，持續10分鐘。當然，最感興趣的是過來抓寵的獵人。

7.林精之心

又是聯盟專屬任務，召喚一個林精幫忙戰鬥，持續高達20分鐘。

8.斯菲萊克斯的攜帶型伐木機

奧山任務獲取，價格不便宜，但戰力很強，是燒塔利器。

補充一個荊棘幼崽的種子，厄運東開花苗獲得，與假人一個效果，算一個目標，打安其拉公主有一點點用。

四、工程其他：

1.活動假人、高級假人、高級活動假人

只能防禦不能進攻，吸引怪物來打，弱勢職業練級有用，會爆保險絲。

2.侏儒作戰小雞

價格不貴，消耗品，30分鐘冷卻，持續1分鐘，或者小雞陣亡。做過各地小雞護送任務的，都知道這傢伙有多弱。

3.聯合收割機組件

同樣消耗品，用過就沒了。一個只能召喚一次，持續10分鐘。造價的原因，戰力就那樣，與奧山的收割機差一檔次。

​最後，補充一個塔納利斯的烏龜護送任務。和老烏龜對話接任務之後，它會一直緊緊跟在你的屁股後頭，跑得很快，持續大概30分鐘。這期間，你可以不帶它找老婆，而是作為打工的苦力。因為烏龜能打人，傷害還可以。帶著這傢伙去做其他任務，相當於獵人寵物的效果。當然，在你帶著它滿處跑的時間內，同位面的其他玩家是不能做護送的。
