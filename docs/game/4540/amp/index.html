雲頂之弈：“送死流”盧錫安出現機制問題，不能上兩個賽娜

天天要聞
2019-12-06

摘要

“送死流”盧錫安出現機制問題，不能上兩個賽娜！在雲頂之弈9.

「送死流」盧錫安出現機制問題，不能上兩個賽娜！

在雲頂之弈9.24版本中，將會迎來兩位新英雄賽娜和盧錫安，兩人之間的系魂羈絆是的雙方只要有一人沒有陣亡，則另外一人就可以在亡靈狀態繼續輸出，所以說，玩家以此開發了一套「送死流」盧錫安戰術，開局直接讓盧錫安上去送，保賽娜，讓盧錫安在亡靈狀態下安全輸出。有玩家看到這個套路，上了兩個賽娜，企圖兩個賽娜保一個盧錫安，但是「送死流」盧錫安出現機制問題，不能上兩個賽娜，這是怎麼回事呢？

原來這位玩家卻忽略了系魂羈絆的機制問題：如果有多個系魂英雄在場，只有第一個陣亡的英雄可以獲得系魂羈絆，剩下的陣亡後是不能復活的。所以說這位玩家上兩隻賽娜，不但沒起作用，反而幫倒忙，前排死亡之後，獲得了系魂羈絆，之後盧錫安就不能獲得，所以也沒打不出送死流盧錫安的精髓。現在只能在測試服獲得這些消息，至於之後會不會改動，則到了正式服才知道，如果正式服沒有改動，大家不妨試試這套送死流盧錫安玩法。

現版本你還願意玩賭刺嗎？

在現版本中，經過幾天的排位實戰操練，盟盟發現現在的賭狗基本是十賭九輸，吃雞概率非常低。現版本幾個比較流行的賭陣容套路，比如森林刺、掠食者刺，前期雖然因為一直在賭三星，所以陣容質量比較高，基本吊打其他玩家，但是到了中期，其他主流陣容開始慢慢成型，強度慢慢上來之後，這些陣容的壓制力就沒那麼強了，然後到了後期，就完全沒有了優勢，其一是人數劣勢，由於需要賭，所以一般都是六人口七人口，而主流幾個陣容都是八人口甚至九人口，所以人數上不佔優勢。

其二是卡牌質量差距，雖然低費卡三星很強，但是高費卡升二星之後的戰力看到要比地方人看三星三星強，比如現版本比較厲害的火男、煉金、劫，這些兩星之後都是bug級別的存在。所以說，現在的賭刺已經失去了改版之初的那種統治力，現在版本，想要吃雞，拼前中期發力是不可取的，只有發育到後期厲害的陣容才最可能吃雞，所以說想要上分還是老老實實玩幾個主流陣容，只要玩透了玩精了，上分絕對是輕輕鬆鬆。

玩家給拉克絲一個法師帽，結果她成了法術永動機！

六法師羈絆有一個很好用的作用：雙倍施法，很多英雄在搭配上魔法師轉職書後，可以發揮出意想不到的作用，比如劫雙重分身，石頭人大招兩次等等。最近又有一個比較有意思的玩法，給拉克絲一個魔法師轉職書（法師帽），然後全部給輸出裝備，拉克絲開啟大招後擊殺可以回藍，所以最後拉克絲成了法術永動機，由於輸出高，不斷擊殺敵人，然後又可以回藍，所以拉克絲一直處於放技能的狀態，面對如此恐怖的輸出，能有幾個陣容扛得住啊。此陣容娛樂效果滿分，不過想要湊二星拉克絲還是挺難得，所以不建議排位上分使用。
