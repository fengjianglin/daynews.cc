刺客伍六七：暗影刺客染指小雞島，阿七或已恢復記憶？

刺客伍六七：暗影刺客染指小雞島，阿七或已恢復記憶？

雖然刺客伍六七近期才開始繼續更新，但是相信關注刺客伍六七的小夥伴們在此期間早已將之前的PV溫習數遍了，確實，從刺客伍六七第二季的前五集我們並沒有獲得與PV中相符的重要信息，不過，隨著今天第八集的更新，我們總算是將正式劇情與PV中顯示的高能內容結合到了一起，那麼我們到底能從兩者中獲取什麼信息呢？我們一起來看看吧。

首先我們先提煉一下第八集的信息，如大多數人所預測的一樣，十三的師傅青鳳與阿七之前均為暗影刺客的成員，而在PV中出現的綠毛也是暗影刺客中的一員，其名為赤牙，不出意外的話，他應該是小雞島迎來的首位正式作為反派登場的暗影刺客，並且阿七將會與之有一場大戰。

從PV中我們能夠得知，不僅是大保小飛，就連大春也在赤牙的手下隕落，在這種情況下，阿七勢必是要開大號的，不過從PV中透露的信息來看，阿七與過去的自己對話時出現了這麼一句話：「我已經不想做刺客了」，這句話是否是意味著阿七早已恢復了記憶，只不過過去的經歷使他不願重拾曾經的生活，相反他更傾向於享受在小雞島的生活。

不過即使是這樣，想必暗影刺客也不會讓他如願，畢竟他沒死的消息已經傳到玄武國了，而且目前看似作為暗影刺客首領的十三的師傅青鳳也勢必不會輕易放任阿七享受愜意的生活，因為結合前後劇情來看，青鳳很有可能就是導致前任暗影刺客身死以及阿七被追捕的元兇，此外由第八集所推導出的信息，作為暗影刺客中唯一能威脅到青鳳的存在，阿七在劫難逃。

同時我們在PV中也能看出，青鳳很可能也將會在小雞島露面，而且從中我們還能知道他一直封印起來的右眼還蘊藏著其他的力量，不過基於他最終BOSS的屬性，大概也不會在小雞島展現出過多的要素，在第二季大概率還是會由赤牙擔任本季的最大BOSS，而他與開啟大號的阿七之間的戰鬥還是相當值得大家期待的。

而在第二季通過阿七與赤牙的交手，相信他在以後的劇情中勢必要直面自己的記憶以及真實身份，因為即使他想逃避，越來越多的敵人也已經找上門了，所以無論是為了保護身邊的人，還是為了保護小雞島，他都勢必要正視自己從而做出新的打算，而這一切關於今後劇情的推測，我們就在往後的日子裡慢慢期待吧。
