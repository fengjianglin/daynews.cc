《怪物獵人世界》冰咒龍還沒看到獵人就被“忍殺”了

2019-12-09 0條評論 7次閱讀 0人點贊

摘要

《怪物獵人世界》（Monster Hunter World）狩獵怪物的方式很多，身為經驗豐富的獵人，

《怪物獵人世界》（Monster Hunter World）狩獵怪物的方式很多，身為經驗豐富的獵人，你可以地形殺，陷阱殺，暴力硬殺，或是叫恐暴龍幫你殺…不過，也許你可能沒試過「忍殺」。我知道，這不是《只狼》，但這回第五期團的神人，他一樣能辦到在怪物沒發現的情況下完成狩獵。

沒錯，又是tatsuffy。在《怪物獵人世界》一旦交戰，左下角的小地圖就會呈現紅框，象徵怪物已發現你，正式進入戰鬥狀態。不過，tatsuffy這次狩獵冰咒龍的過程中，一次也沒有讓小地圖變紅框。換句話說，冰咒龍從頭到尾都處於未察覺獵人的非戰鬥狀態。

是的，聰明的獵人應該已經發現，tatsuffy靠的就是「煙霧彈」的使用時機。一開始，他先向冰咒龍投射石頭，一時之間冰咒龍沖了過來，卻不會發現tatsuffy，令他順利得以騎龍撞牆，展開第一波輸出。

之後就是關鍵，在完成擴散彈輸出攻擊後，tatsuffy立刻收起武器丟煙霧彈，冰咒龍起身後仍維持這著未發現獵人的非戰鬥狀態。

完成裝填後，tatsuffy 稍微移動位置，並向地面投射石頭，且再次施放煙霧彈，誘導冰咒龍靠近，再次騎龍撞牆，故技重施。

另一種方式，是在接近冰咒龍前就施放煙霧彈，然後在煙霧中擊發撤甲彈，會使冰咒龍陷入找不到獵人而一前一後的狀態。當然，令人佩服的是，tatsuffy連需要幾顆撤甲彈就能讓怪物倒地都計算在內。

再來，tatsuffy也使用了睡眠彈+ 龍擊炮的技倆，在冰咒龍倒地起身前就先發射睡眠彈使其入睡，且在怪物完全安眠之前擊發龍擊炮增加輸出。

就這樣，冰咒龍從頭至尾都沒發現過獵人，就這樣死在tatsuffy的擴散重弩下，全程僅花費5 分30 秒，就算這只是一般的MR 自由任務等級，這個速度也比一般獵人還快了。

tatsuffy所使用的重弩為岩漿嘆息熔山大炮，而裝備為精英迅龍頭盔α、上位麒麟胸甲γ、精英迅龍手套β、精英暴君角龍腰帶β、精英迅龍護靴α與睡擊護石，如果你好奇的話。

如果你想問神人的狩獵為何如此樸實無華且枯燥，其實在5月時，tatsuffy就已經在歷戰王滅盡龍身上實驗過煙霧彈忍殺了，經過長時間的練習，才能有如此瘋狂且精採的狩獵。

當然，對於遊戲上市至今一次煙霧彈都沒用過的觀眾來說，tatsuffy對遊戲理解的展示，才是真正玩透《怪物獵人世界》的蒼藍星。

最令人覺得有趣的是，底下有一位凡人吐槽：這委託內容並不是「暗殺冰咒龍」啊。是的，這難度也太高了，根本是別款遊戲了。

這冰咒龍一定欠tatsuffy很多錢。
