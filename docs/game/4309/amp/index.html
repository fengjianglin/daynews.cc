摘要

然而瑪加薩的刺客們失敗了，一部分忠於血蹄家族的牛頭人保護了貝恩，將他護送至陶拉祖營地，在那裡貝恩開始積蓄力量，準備奪回雷霆崖。

然而瑪加薩的刺客們失敗了，一部分忠於血蹄家族的牛頭人保護了貝恩，將他護送至陶拉祖營地，在那裡貝恩開始積蓄力量，準備奪回雷霆崖。

阿格娜

此時的薩爾並不知道奧格瑞瑪發生的一切，艾澤拉斯的元素之靈已經不再回應薩滿的召喚，薩爾只能攜帶一塊土靈石板來到外域納格蘭的元素王座尋求這裡元素之靈的幫助。大地之怒戈達烏告訴薩爾，他從艾澤拉斯帶來的石塊說明此處的元素因大地的裂變而感到恐懼，一次類似曾在德拉諾發生的大災變即將發生。薩爾為此陷入了深深的焦慮之中。

與薩爾同行的女薩滿阿格娜安慰了他，這位性格直爽的女獸人，毫不避諱的指出，德拉諾雖然已經毀滅，但艾澤拉斯希望尚存。這番話給予了薩爾莫大的鼓勵，在之後的日子裡兩人的感情也逐漸升溫成為了一對戀人。

薩爾與阿格娜

再次拜別祖母蓋亞安之後，薩爾和阿格娜一起重新返回了艾澤拉斯，而此時他也得知了凱恩遇害的噩耗。回到卡利姆多的薩爾出席了自己老朋友的葬禮，並對之前離開凱恩時的爭吵而感到悔恨。他最終留下凱恩長矛上的一塊意為「治癒」的符文石作為永久紀念。但一場隨之而來的大規模地震，讓薩爾頓覺時間緊迫，他已經無暇前去奧格瑞瑪質問加爾魯什，只能匆匆踏上了飛往大漩渦的飛艇。

藍龍隕落

時間回到數年之前，瑪里苟斯在格瑞姆巴托之戰後，回到了自己位於諾森德的冰凍洞穴中。擊敗死亡之翼的喜悅並沒有持續多久，他依舊沉浸在妻子辛達苟薩和孩子們去世的悲傷之中。這無盡的折磨逐漸摧毀了藍龍的意志。他變得愈發瘋狂，最後魔法守護巨龍偏執的認為凡人對奧術能量的過度使用，將導致艾澤拉斯的毀滅，他開始進攻達拉然並殺死所有遇到的法師。

瑪里苟斯

達拉然的領袖首席大法師羅寧向紅龍女王求援，阿萊克茲塔薩在龍眠神殿召集了綠龍與青銅龍的巨龍軍團，組成了龍眠聯軍，對瑪里苟斯發動了攻擊。最終在魔樞之戰中，紅龍女王親自結束了這位老友痛苦的生命。倖存的藍龍軍團則由卡雷苟斯接任了守護巨龍的職責。

（大家應該還記得這位吧，就是太陽井之戰中愛上太陽井化身安薇娜的那隻藍龍）

叛變的大德魯伊

時間回到現在，薩爾在抵達大漩渦之後，馬上加入了大地之環治癒元素的儀式之中，但凱恩的死和對部落未來的擔憂讓他始終無法集中注意力，導致儀式數次失敗。深感困惑和無力的薩爾獨自尋求元素的幫助。

但所有的元素之靈都對他沉默不語，只有炎魔之王拉格納羅斯回應了他的召喚。拉格納羅斯向薩爾宣稱他的火焰將會燃盡世界之樹和整個艾澤拉斯，他讓薩爾看見了未來化為火海的奧格瑞瑪幻像。深感震驚的薩爾跌倒在地，隨後他便與阿格娜一起動身前往海加爾山。

在那裡薩爾帶領的大地之環薩滿們和大德魯伊瑪法里奧·怒風帶領的塞納里奧議會會合，紅龍女王阿萊克茲塔薩，綠龍伊瑟拉，藍龍卡雷苟斯，青銅龍諾茲多姆率領守護巨龍軍團也前來相助，最終他們擊退了炎魔之王拉格納羅斯的進攻。

化身為火焰德魯伊的范達爾·鹿盔

但是在隨後進行的治療世界之樹儀式上，被喪子之痛折磨而墮落反叛的精靈領袖范達爾·鹿盔突然出現並用黑暗魔法擊倒了薩爾。他利用了薩爾與元素之間的關聯，使得薩爾的靈魂被流放到了元素位面之中。

解救愛人

阿格娜發誓一定要救出自己的愛人，她召集來自聯盟與部落的冒險者（玩家），前往天空之牆、無盡深淵、深岩之洲和火焰之地去拯救薩爾的靈魂，協助他戰勝自己的疑慮、慾望、任性與憤怒。

在風之位面中，薩爾對自己指認加爾魯什·地獄咆哮為大酋長感到悔恨不已，深感自己辜負了部落與大地之環以及阿格娜，他將艾澤拉斯的遭遇都歸咎為自己的軟弱。

在水之位面，薩爾展現了自己內心的慾望，他放下自己的武器，與部落的敵人瓦里安·烏瑞恩言和，卸下大酋長的重擔只為與阿格娜過普通人的生活。

在土之位面，薩爾靜止在岩石之中，代表了他的耐心與堅決。

在火之位面中則是薩爾的憤怒和憎恨，他痛恨古爾丹殺死了自己的父母、布萊克摩爾將自己變成了奴隸並殺害了塔蕾莎以及加爾魯什間接殺死了凱恩。

最終薩爾在阿格娜的幫助下成功地戰勝了所有的負面情緒，找回了自己的靈魂，經歷這一番劫難之後，薩爾表達了自己對於阿格娜的愛，並與她結為了終身的伴侶。

薩爾與阿格娜結為夫妻

決戰龍眠神殿

之後，薩爾與其他巨龍守護者來到龍眠神殿，一起商討如何對付死亡之翼。眾人商議良久，最終一致認為只有神器巨龍之魂才能徹底消滅耐薩里奧。於是青銅龍諾茲多姆打開了時間穿越通道，他們決定前往上古之戰的永恆之井奪取那時還純凈巨龍之魂。最終在擊敗了腐化的時光守護者姆諾茲多之後，眾人拿到了神器，回到了龍眠神殿。

（這段劇情怎麼看怎麼像復聯4回到過去拿寶石再回來打滅霸）

由於巨龍使用這件神器只會被反噬撕裂，薩爾成為了使用巨龍之魂的唯一人選，他將自己的大地之力灌注進神器之中，使死亡之翼再也無法對巨龍之魂的攻擊免疫。

就在此時，耐薩里奧率領他製造的暮光龍大軍對龍眠神殿發動了攻擊，薩爾與諸位守護巨龍也已經準備就緒，一場決定艾澤拉斯世界命運的大戰，即將爆發……
