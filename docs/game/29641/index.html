DYG.JC是一支喝涼水都塞牙的隊伍，冬季冠軍杯抽籤，再次沒運氣

2019-12-25 1,346條評論 2,325次閱讀 1394人點贊

冬季冠軍杯的選拔賽已經結束，參與的幾支隊伍，最後衝出重圍的只有TS與eStarPro，這兩支隊伍確實都算是強隊，尤其是在現在的版本中，打法都進行了改進，找到了新版本中最適合的戰鬥方式，但冬季冠軍杯的比賽與KPL不同，這是一個短時間難度越來越高的賽事，需要有非常強的爆發力，才能夠走到最後。

實力當然是最重要的一點，但後面賽程的抽籤，也同樣重要！運氣是實力的一部分嘛，小組賽參與的隊伍一共有八支，分別是：eStarPro、AG超玩會、ROX、SLT、TS、QGhappy、GOG、DYG.JC，其中五支KPL隊伍，三支是KRKPL的隊伍，這一次抽籤註定至少會有一組KPL內部隊伍抽到一起，相較於與海外賽區的隊伍對戰，KPL肯定會更加有難度，參與選拔賽的海外隊伍，實力上的提升並不大，KPL依舊是很大的威脅。

比較有意思的是最後抽籤結果是A組：eStarPro vs SLT；B組：AG超玩會 vs ROX；C組：TS vs DYG.JC；D組：QGhappy vs GOG；其中eStarPro、AG超玩會、QGhappy都抽到了海外賽區的隊伍，按照他們目前的實力，淘汰賽中獲勝的幾率大概會有九成，基本上可以說是保送到四強，這是KRKPL隊伍實力決定的事情。

而DYG.JC與TS顯然不太幸運，只有他們在第一輪相互之間成為了對手，註定會有一支隊伍離開，兩隊都是KPL的隊伍，實力上肯定更強一些，尤其是在選拔賽中，TS實力彷彿有了很大的提升，讓粉絲也重新燃起了希望，如果繼續拼一拼，往後面走一走並不是不可能，DYG.JC在秋季賽中的表現也不錯，但說實話運氣一直都不太好。

DYG.JC可以說他們是一支喝涼水都塞牙的隊伍，他們連續好幾個賽季高開低走，給人留下了很深的印象，每一次崛起都會讓人覺得有希望奪冠，但最後都會變成冷門，命運或許特別會捉弄他們，這一次別人都抽到了海外賽區的隊伍，唯獨他們抽到了TS，別人基本上可以說是保送四強，而如果他們想要成為四強，需要先越過去TS。

雖然TS目前版本確實比較強勢，但如果DYG.JC能夠先擊敗他們，晉級四強的分量要比其他隊伍更重，大概率之後會對陣eStarPro，只要繃緊神經，也可以去攀越眼前的高峰，雖然這次冬季冠軍杯的抽籤，再次證明他們沒有運氣，剩下的只能夠用實力去證明了，加油吧！
