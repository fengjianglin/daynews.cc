LOL德杯小組賽前瞻：Uzi缺席德杯，RNG或將不敵LNG“新主場buff”

2019-12-19 2,833條評論 4,995次閱讀 1668人點贊

北京時間12月21日，2019德瑪西亞杯小組賽火熱開戰！晚20:00由RNG vs LNG。在轉會期RNG的雙替補上單、打野Karsa和教練Steak牛排離隊，LNG則是雙中單、野輔和教練朱開離隊。我們將通過兩支戰隊的陣容分析，結合專業電競競猜機構給出的競猜指數，對這場比賽做出前瞻預測。

從專業電競競猜機構給出的競猜指數來看，即便Uzi因傷病缺席德杯，Tabe暫代Mata登台BP，RNG的勝算仍然高過LNG。LDL新星打野XLB和Xiaohu在前期的聯動至關重要，另外輔助Ming需要扛起指揮大旗，他豐富的經驗將是RNG的制勝利器。不過「聖槍哥」Flandre如果能在上路壓制Langx，同時Asura下路對線Betty打出優勢，LNG就有獲勝的希望。

2019德瑪西亞杯RNG戰隊陣容：

分析師：Tabe 王柏勤 （暫代主教練Mata上台BP）

上單：Langx / Yuekai

打野：XLB / S1xu

中單：Xiaohu

ADC：Betty

輔助：Ming

RNG在轉會期最大的損失就是失去了全能型打野Karsa，不論是LDL頂尖打野XLB還是S1xu，既缺乏與LPL戰隊交手的經驗，又需要與RNG其他隊員進行磨合，RNG將在德杯麵對嚴峻的考驗。而Uzi因傷病缺席德杯更是讓RNG的競爭力有所下降，雖然在近兩年沒落的LMS賽區「幻神」FW閃電狼中，被迫獨當一面的Betty潛能得到了激發，但對手Asura是近年來LPL中排名前列的國產ADC選手，初登德杯的Betty未必能從Asura手中討到便宜。另外S9世界賽中Xiaohu糟糕的狀態飽受各界詬病和質疑，他能否在和Maple的對決中重新找回狀態還是一個未知數。

2019德瑪西亞杯LNG戰隊陣容：

主教練：Cyo（林昕宥）

上單：Flandre / Chenlun17

打野：Xx

中單：Maple

ADC：Asura / Light

輔助：Duan / Iwandy

LNG在轉會期更換了中、野兩個位置的選手，原TES戰隊打野Xx和原SN戰隊中單Maple加入。TES在今年LPL春、夏季的成績十分亮眼，Xx從更名前的TOP到今年TES的成長也是有目共睹，只要Xx在德杯維持好競技狀態，相信他能給RNG的新人打野來場「教學局」。縱觀Maple在SN全年的表現，雖然偶有亮點但總體中規中矩，和Xiaohu的中路對線將是兩隊三線中水平最為接近的一路，兩人都有可能成為對方的突破口。「LPL知名狗吹」Flandre難得和Uzi缺席的RNG對陣，他和Asura在前期打出的優勢越大，LNG反超RNG的勝算也就越大。

北京時間12月21日，2019德瑪西亞杯小組賽火熱開戰！小組賽將在LNG全新主場蘇州陽澄國際電競館和LGD杭州電競館同時進行，LPL十六支戰隊將和LDL四強同台競技，向著德杯冠軍發起衝擊，LNG在新主場能否連戰連勝，讓我們拭目以待吧！
