是RNG耽誤了小虎？Letme直言希望小虎離開，還說了1件S7的往事

天天要聞
2019-12-20

維持了一個月的轉會終於在前幾天畫上了句號，到現在為止，基本上LPL的所有戰隊都已經湊齊了自己的拼圖。不管是好是壞吧，改變都已經做出來了，剩下來的就是得慢慢磨合了。

單論改變來說，在S9墊底了一整年的VG無疑就是最大贏家了。但如果說爭議最大，人氣最高的，御三家之一的RNG怕是怎麼也逃不脫這個光環了。

一整個S9，RNG似乎就一直在輿論中心沒有出來過。先是姿態退役，狼行加入，RNG的上路生態從一向「五五開」變成了偶爾崩盤，再到賽季中後段，中單小虎的表現又頻頻拉胯，而這一拉就拉倒了S9的賽場上。最後一場比賽中單瑞茲全程被蛇女壓制，整場只打出了2200的傷害。RNG在S9提前出局，往日的虎大將軍變成了人人喊打的「ZZOO」。而令人意外的是，這個轉會期RNG不僅沒對小虎做出絲毫改變，而且連替補都沒給找一個。

「在宣布德杯名單的那一刻，RNG的官博也炸了。」

碰巧，在昨晚RNG的前隊員Letme直播中也談了下小虎，從他的視角來看，這件事好像跟我們想的完全相反，從小龍堡3kw的事情談起，Letme表示作為一個新人3kw有點開玩笑的感覺。現在的LPL里值這個價的也就Theshy，小狗，Ming，左手和Rookie這幾個人。當觀眾問道：「這裡面沒有小虎嗎？」Letme答道：

「小虎還好吧，不過我更希望他去其他的隊伍。這是真的，不是假的，這是我發自內心想說的話。也不知道為什麼，感覺他去其他隊伍會打得更好。這句話並沒有其他的含義。」

除此之外，Letme還談了一件S7的事情。

「大家對RNG的印象就是，只要上路不C就是打得不好，這已經形成一種刻板印象了，狼行其實打得挺好的。也是因為Shy哥的到來，讓所有的上單都失去了光芒。我17年那會，可能就是16年留下的光芒吧，大家對我的期望莫名的大，我也不知道為什麼大家對我的期望能那麼高。也不知道哪來那麼多人天天擱那捧我，挺難受的，想想大家沒有必要啊。」

Letme的話，或許沒有那麼好聽。甚至在某些人看來，可能會直接說是「沒有情商」。在Theshy時代，還能發光的上單選手，到了本人嘴裡卻成了一文不值，真的是有些打老粉絲的臉上了。不過換種角度說，這種想法或許也才是一個真正職業選手的心理：

「把自己放得太高，有時候很可能就會忽視對手的實力。站在地平線上，才能看出哪些人才是真正的巨人。」
