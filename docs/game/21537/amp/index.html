JKL將休息3個月？PDD直播爆出3個猛料，並闢謠了1個選手身價傳聞

天天要聞
2019-12-19

​如今英雄聯盟LPL轉會期已經結束了，不少隊伍都已經宣布了自己下個賽季的選手大名單，可是阿水去哪兒了仍然是轉會期間最大的謎團，畢竟JKL作為自由人，可以不在轉會期加入隊伍，因此大家都對JKL的去向非常好奇。而在PDD的直播之中，不斷有「水友」問起轉會的疑團，PDD本身還是比較「寵粉」的主播，於是就挑選了幾個回答，沒想到爆出3個猛料，並闢謠了1個選手身價傳聞。

RNG為何不買中單？

​最近RNG的「德杯」出戰選手已經爆出了，其中Xiaohu仍然坐穩了RNG首發位置，讓不少粉絲非常不滿意，覺得RNG真就不買個中單嗎？哪怕替補也好呀。關於這個問題，PDD爆料出其實在轉會期間，RNG的老闆準備花大價錢買下「黃金左手」Knight，可是TES根本不賣，畢竟Knight是隊內終極核心、建隊基石的存在。畢竟RNG為了組成「全華班」，只能買國產中單，而在國產中單之中，厲害的根本就不賣，現在RNG陷入兩難局面。

小龍堡真賣了3000W？

​小龍堡作為從未打過LPL的選擇，雖然是PDD旗下非常有天賦的選手，但是3000W的身價傳聞讓不少玩家驚嘆，而這個消息最初就出自娃娃直播爆料。對於這個問題，PDD直接闢謠了，說小龍堡3000W完全就是「扯犢子」，怎麼不說3個億呢？確實，小龍堡畢竟只是LPL的新人，3000W身價簡直不要太離譜。

estar訓練賽被暴打？

​根據最近的轉會消息，estar只爆出了Wink這名選手，而根據傳聞，estar訓練賽上直接被各個LPL隊伍暴打，以至於都沒有隊伍想跟estar打了。而PDD作為estar的老闆之一，對於這個問題說了很多，說這個消息確實如此，自家上單甚至都被打自閉了，並表示自己最初本來想該戰隊ID，可是有規定說兩年之內不可以改。這個消息倒不算意外，畢竟現在estar的陣容太弱了，剛來LPL，遭受各個老牌LPL隊伍的「毒打」在所難免。

JKL將休息3個月？

​對於這個消息的爆出並不是「水友」問，而是PDD自己爆出，畢竟大家最關心的莫過於JKL的去向了。PDD表示自己聽說JKL要休息兩到三個月，不過並不能保證這個消息的真實性。而根據目前各家戰隊宣布的「德杯」陣容來看，JKL的去向已經越來越小，休息3個月，差不多是直接從明年春季賽開始打，時間剛剛好。可是對於JKL來說，如果是完全休息不打訓練賽的話，無疑是個壞消息。

對於上述消息，大家有什麼看法呢？
