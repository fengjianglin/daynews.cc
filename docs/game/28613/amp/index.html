「小魏遊戲」喜迎雙旦，遊戲推薦

天天要聞
2019-12-22

哈嘍大家好，我是小魏。2019年的Steam冬季特賣已於12月20正式開啟，超多遊戲打折，各種史低等你來。

迎完”黑五”迎”雙旦”，今年的steam冬季特賣也有不少我喜歡的遊戲，就讓我們一起來看看吧！

《只狼：影逝二度》

作為TGA 2019年度遊戲，《只狼：影逝二度》講述了一個名譽不再、傷痕纍纍的忍者，一個從死亡邊緣撿回一命的戰士，為了年輕的皇子，為了挽回榮譽，不畏死亡踏上危機四伏的征程。

作為《黑暗之魂》製作團隊的新作，《只狼：影逝二度》依然有著硬核的操作技巧，而戰鬥方面，甚至比任何一部魂系列遊戲都更加激烈純粹。可謂”痛並快樂著”。

敢於挑戰的玩家們一定不要錯過這款遊戲。遊戲能讓玩家們感受殘酷的日本戰國時代，在黑暗、扭曲的世界，展現對於人性的思考。

《只狼：影逝二度》目前steam減價35%，售價人民幣174元。促銷截止於1月2日。

《過山車之星》

《過山車之星》旨在讓玩家們打造自己夢想中的主題公園，建造和設計令人難以置信的過山車公園，並在真正鮮活的世界裡管理自己的公園。

沒錯，作為一款模擬建造經營類遊戲，擁有不錯的畫面和非常齊全的過山車種類，並將管理遊樂園的樂趣也融入其中。作為《過山車大亨》的精神續作，《過山車之星》絕對值得體驗。

《過山車之星》本體目前steam減價80%，售價人民幣24元。不過本作DLC眾多，小夥伴酌情選擇哦！促銷截止於1月3日。

《波西亞時光》

國產遊戲《波西亞時光》是一款結合RPG，模擬經營，以及人物養成於一身的農場類遊戲。玩家們可以在遊戲中建設你的工坊，經營你自己的農場，發揮創意，加入社區以及探索和戰鬥。

作為國產獨立遊戲的一個標杆，《波西亞時光》以其清新的畫面，多樣的人物，創新融合的玩法，受到不少玩家們的喜愛。遊戲輕鬆愉快，玩法眾多，喜歡休閒遊戲的小夥伴們，趕快入手吧。

《波西亞時光》目前steam減價50%，售價人民幣49元。促銷與1月3日截止。

《八方旅人》

角色扮演遊戲《八方旅人》於2018年7月13日登陸任天堂NS平台，其後又登錄了PC平台。遊戲採用虛幻4引擎打造，畫面採用結合復古像素點陣與3DCG的風格。

講述了八位旅行者的八次冒險之旅。玩家們化身為八個角色之一，踏上史詩般的旅程，穿越廣闊而神奇的奧魯斯特拉大陸，探索八位旅行者各自迷人的故事。

《八方旅人》是一款傳統的日式RPG，玩家們遊玩其中，很有復古的感覺。像素風，回合制與虛幻4的結合，會帶來怎麼樣的遊戲體驗，小夥伴們親自來試一試吧！

《八方旅人》目前steam減價40%，售價人民幣241元。促銷截止於1月3日。
