LSPL皇帝名不虛傳！兮夜首戰拿雙4殺，13殺0死接管比賽

天天要聞
2019-12-23

這裡有最新最全的英雄聯盟資訊，菌菌帶你看不一樣的召喚師峽谷~

12月22號冬至這一天，德瑪西亞杯第二天的賽程持續進行著，今天是BD兩組的對決，今天的比賽也是備受關注的，DMO戰隊在兮夜加入之後像「活了」過來一樣，大家去看DMO的比賽基本都是沖著兮夜去的吧，想看看離開WE的兮夜能打出什麼成績，德瑪西亞杯也是兮夜新戰隊首戰，在面對SDX的比賽中，兮夜13/0/8的戰績統治戰場，打次級聯賽的隊伍還是有一套的！我們一起來看下戰報。

DOM禁：潘森、鐵男、奇亞娜、加里奧、蜘蛛，選：刀妹、瞎子、阿卡麗、賽娜、塔姆

SDX禁：千珏、豹女、女警、奧拉夫、鱷魚，選：劍魔、豬妹、波比、霞、泰坦

DMO上單選到了刀妹，打野選了瞎子，兮夜選了阿卡麗，三個都是突進的英雄，所以SDX中單選了波比，防止DMO亂秀，DMO這邊陣容偏進攻，而SDX這邊偏防守，後期的話還得看霞，而DMO這邊三核輸出，可能性比較多。

6分鐘，SDX中上野很有想法，三人包DMO上路想越塔，但是刀妹呼叫兮夜阿卡麗，阿卡麗傳送下來，刀妹撐了很久並且升六級，配合刀妹反殺SDX三人，刀妹拿下兩個人頭，這波開始SDX上中野很難了。

13分鐘，豬妹蹲下路，率先開了賽娜，但是被塔姆一口吃掉，團戰瞬間開啟，DMO的刀妹和阿卡麗傳送下來，SDX的劍魔也傳送下來，這波瞬間形成5V5，刀妹和阿卡麗下來就直指霞，泰坦陷入太深，被兮夜擊殺，DMO持續追擊，兮夜如猛虎下山，再收霞的人頭，這波DMO0換2結束團戰。

17分半，塔姆被SDX擊殺，DMO少人的情況下SDX選擇開火龍，刀妹進場直接開R命中三人，刀妹和阿卡麗直接衝進去，刀妹率先被秒，然後泰坦也被擊殺，面對SDX全部殘血的情況下，兮夜的阿卡麗開啟收割模式，火龍被瞎子拿下，阿卡麗最後Solo劍魔單殺，並且科技槍刃收下波比拿下四殺，此時的兮夜7/0/3戰績，不愧是LSPL皇帝！

23分鐘，火龍刷新，雙方即將開戰，泰坦率先開團，塔姆率先被秒，刀妹此時進場R中五人，征服者直接疊滿然後對著SDX五人一頓狂揍，因為兮夜被波比R飛了，兮夜痛失幾個人頭，兮夜很生氣也是用大招跑路趕回戰場，收下劍魔人頭，並且宣告自己超神了，DMO直奔大龍，刀妹獨自Solo火龍，大龍+火龍魂全部拿下，SDX宣告死亡。

25分鐘，帶著雙BUFF的DMO選擇進攻，SDX想奮死一搏，但是無奈DMO眾人裝備等級高太多了，兮夜再次放飛自我，追著四個人打，最後再次拿到四殺，並且推了中路高地水晶，然後轉到下路拆下路高地水晶，SDX無力回天了，DMO中路一波拆掉了SDX基地，恭喜DMO！恭喜兮夜！

賽後點評：其實沒什麼好點評的，兮夜真的太猛了，打次級聯賽真的有一套，一手自己擅長的刺客，直接把SDX打自閉了，兩次四殺也是宣告誰才是LPL頂尖中路，我想說DMO這次真的血賺，恭喜DMO拿到德杯首勝，也恭喜兮夜加入新戰隊拿到首勝！
