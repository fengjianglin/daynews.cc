《茶杯頭》測評：復古風硬核遊戲，手殘玩家的噩夢

2019-12-11 247條評論 4,236次閱讀 200人點贊

各位九零後玩家們想必都是和我一樣，從小看著碟片長大。在那個動畫資源有限的年代，《貓和老鼠》陪伴了我整個童年。甚至至今看來依舊津津有味。同系列的動畫片還有《米老鼠唐老鴨》，這也是九零後心頭永恆的經典。類似的卡通人物也給很多遊戲製作商提供了靈感來源。比如以30年代動畫為靈感來源的遊戲《茶杯頭》。與一般的卡通類遊戲不同的是，這款遊戲除了整體畫風非常復古又可愛，真正吸引我的其實是這款遊戲超級硬核的玩法。

非常可愛的人物形象

遊戲測評：9.0/10.0

遊戲類型：橫幅射擊類

上世紀30年代的靈感，復古畫風滿滿

受19世紀30年代動畫風格影響，遊戲製作商以米老鼠為靈感來源，在米老鼠形象基礎上進行加工改變，成功製作出這款滿滿復古畫風，人物形象非常可愛的《茶杯頭》。這款遊戲也在2017年獲得了英國金搖桿獎最佳視覺設計獎以及當年的年度Xbox遊戲獎。

茶杯頭主要是以與BOSS對戰為主，是一款經典的橫版捲軸類射擊遊戲。看似簡單的設計其實每一個boss在最後都會讓我大開眼界，boss的每一個招式都不太好躲，想要擊敗boss也是非常困難。

另外，在遊戲音效與遊戲畫面上製作商也是極力在向19世紀30年代的經典致敬，遊戲的畫面和音效都在努力的還原當時的場景。比如火爆當時的賽璐璐手繪動畫，經典的水彩風格遊戲背景，以及最大限度還原的爵士樂音效，這一切都讓我在玩遊戲的時候彷彿也回到了小時候坐在遊戲機前玩遊戲的場景。

復古動畫風效果

精緻的畫面細節，復古遊戲最強出擊

這款美術動畫風格的遊戲已經被大家誇的很多了，今天我們來講一講遊戲里的各種精緻入微的細節。我個人認為這款遊戲的細節之所以戳心是因為他完全是從一個玩家，一個受眾的角度出發，以一個玩家的視角製作遊戲畫面，最大程度上還原玩家的真實感受。當玩家受到攻擊時會在屏幕上產生相關的視覺效果。作為一個細節控，我不得不說這些細節讓我對這款遊戲的好感度爆棚。

說實話，個人覺得這款遊戲算是復古風中難度很大的一款了。如果真的手殘並且沒有耐心能夠忍受一次次的失敗重來，建議繞路避行。此款遊戲僅支持本地聯機有點不太人道，畢竟這款遊戲玩起來如果能夠和好兄弟一起並肩作戰的話很多難點也就沒有那麼難解決了。

被章魚攻擊時墨汁會出現在平面上形成視覺障礙

不要被簡單的劇情和看似簡單的玩法欺騙

這款遊戲的劇情看似簡單，就是一對兄弟與惡魔一起打賭，結果打賭輸了，他們就將自己的靈魂出賣給了惡魔，代替惡魔為他們收債的故事。這個遊戲也有兩個結局，一個是他們最後終於打敗了惡魔，找到了自己的靈魂，還有一個是他們最終加入了惡魔，成為了惡魔團隊的一員。我覺得這款遊戲可能就是告訴我們——拒絕賭博，從我做起。（手動狗頭）

在與boss戰鬥時的場景

雙結局畫面

遊戲屬於橫版捲軸類射擊遊戲，遊戲操作也相對來說比較簡單，最基本的射擊跳躍，不同方向的指令，以及配備的六種武器，三種大招以及三個輔助裝備。關卡有5個小關卡和幾十個boss關卡。與同類型的遊戲比如國產硬核獨立遊戲元氣騎士相比，茶杯頭的基本操作比較容易，關卡雖然沒有那麼多但是不用怕不夠玩，等你真的開玩了你就知道其實每一個關卡都需要耗費好久。

整個玩下來難度還是非常大的，新手關卡我就死了好多次，在boss關卡中一點不誇張的說，我每一個boss都得死個四五次才能通過，最後的大boss真的打了兩個晚上。而且與類似的硬核獨立遊戲相比，比如以撒之類，我個人感覺這款遊戲的難度相比之下會更大一點。另外如果玩家沒有耐心在同一個關卡死幾次甚至幾十次，建議還是不要入這款遊戲，玩不來的。

精美的遊戲畫面

操作感爆棚，操作難度大

細節把控很好

【減分項】

操作難度過大，手殘黨繞路

比較磨耐心 boss難打

僅支持本地聯機

我個人覺得在這款遊戲中，你的每一個決策都會對接下來的劇情發展產生重要影響，這是一款決策和操作同等重要的遊戲。另一方面我認為這款遊戲最重要的就是玩家可以在其中收穫到一次次失敗但是從不放棄的精神，自己以及在一次次的失敗之中汲取經驗的過程，你可能需要花費自己的十幾次生命來換取對於某位boss的熟悉感，你也可能需要耗費幾個晚上來獲得遊戲通關，但是你用多次死亡換取來的經驗絕對不是毫無意義，這些經驗都會在你接下來的通關過程中發揮很大的作用。總之，如果你喜歡此類卡通風格的硬核遊戲，這款遊戲絕對適合你。不過手殘黨還是建議繞行。
