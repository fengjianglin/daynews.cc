夢淚把偷家的本領發揮到極致！關羽差點玩成了韓信，出裝很靈性

2019-12-10 277條評論 1,198次閱讀 52人點贊

熟知夢淚的小夥伴想必都知道，當年夢淚曾用一手韓信給觀眾表演了一波教科書般的無兵線偷家，也是因為那一場比賽，王者榮耀水晶因為夢淚的韓信而被迫加強。雖然說現版本的水晶比之前結實了不少，但對於夢淚來說，他卻不會因為水晶的加強而放棄偷家的念頭。

在一局1800分段的巔峰賽中，前三樓的隊友選擇完畢後，夢淚補到了一手關羽。如果你有經常看夢淚直播的話，應該也知道，夢淚的關羽儘管不如他的韓信、李白出名，但基本的操作還是有的。在前段時間的KPL比賽中，夢淚就曾在比賽中拿出過關羽這名英雄。

在這一局遊戲中，夢淚的關羽差點再次上演極限偷家，過程中發生了一些有趣的鏡頭，把觀眾玩家的情緒都調動起來了。

進入遊戲後，夢淚的關羽在提示下選擇走對抗路，對線到的正是楊戩這名英雄。前面我們也說過，這是一局1800分段的大神局，夢淚匹配到的玩家實力都不弱的。在前期的一對一較量中，夢淚的關羽通過精準的預判和操作，不僅反殺了對面楊戩，還趁機清掉第二波兵線，然後於敵方草叢中安全回家，為打野減輕了負擔！

夢淚一方在前中期經濟方面一直壓著對面，但隨著遊戲的持續進行，對面通過幾波團戰逐漸找回些優勢，遊戲進行到15分左右時，雙方經濟不相上下。因為有輔助張飛的存在，夢淚的關羽這一局出到了暗影戰斧加破軍，並在幾波團戰中成功然後將敵方關鍵C位擊殺。並通過不斷帶線，起到了很好的牽製作用。

雙方在互相的博弈中，將比賽時間拖到了20分，風暴龍王隨之來襲。在後期的一波團戰中，正在下路清線的夢淚見敵方五人正在和隊友搶奪風暴龍王，在看到敵方五人被隊友牽制後，夢淚關羽帶領下路兵線直衝敵方水晶，就在敵方水晶即將被拆掉的水晶，對面沈夢溪和楊戩及時趕到，夢淚關羽宣告失敗。如果換做是韓信的話，估計夢淚會成功。

雖然沒有偷家成功，但夢淚的關羽卻也是盡了全力，我們通過出裝來了解下，為了能夠用關羽更快的A塔，在短暫的時間裡將鞋子換成攻速鞋，並在此基礎上出到了極影這件裝備。不過，好在這局遊戲最終以夢淚一方勝利而告終，不然，真是辜負了夢淚關羽這一身靈性的關羽出裝。
