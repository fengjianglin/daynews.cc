堡壘團滅4AM獲原17經理點贊，SMG還是放不下倫敦那顆雷

天天要聞
2019-12-18

12月17日的黃金賽C組小組賽可謂精彩紛呈，兩支高人氣戰隊4AM和SMG頻繁撞車，看點十足，在第三局和第五局比賽中，4AM小鹿兩次一穿三SMG，第六局比賽中，堡壘和smallcc找準時機勸架，4AM2殘2倒，堡壘一顆雷，加上smallcc兩桿槍線，成功團滅了4AM！

這本來是再普通不過的賽場鏡頭，但發生在SMG和4AM這兩支戰隊之間，就一點也不普通，由於4AM倒地兩人，正準備扶人，結果堡壘一顆雷過來，直接補掉了倒地的兩人，加上另外兩人殘血且無掩體，所以smallcc和堡壘一人一個，幾乎轉瞬之間就團滅了4AM，所以在比賽結束後，不少粉絲都在大肆鼓吹”一雷四響”。

說起一雷四響，估計小夥伴們第一時間想到的還是FGS倫敦那顆雷，決賽圈，4AM醒目一顆雷瞬間炸掉了滿編的17，最後獨狼吃雞，堪稱世界賽史上的傳奇TOP1鏡頭；也正是這顆雷引發了後續17shou微博親自下場，最終兩支戰隊的關係急轉直下，跌入冰點，加上後來17以及SMG的成績一落千丈，有人說，醒目這一顆雷直接炸毀了17，不管是場上還是場下。

所以這一次，堡壘和smallcc瞬間團滅了4AM，粉絲們要吹一波，其實粉絲吹很正常，誰都希望自己的主隊能夠打好比賽，尤其是團滅宿敵，那當然是最好的，但是，賽後，原17戰隊經理城小樓就點贊了這樣一條微博：堡壘也是偷襲，醒目也是偷襲，醒目也不是正面1V4啊！其實這個出發點就是錯的，是的，醒目確實不是正面，咱也不說醒目打得是四個生龍活虎滿狀態的人，就說一點，醒目他總算是個實打實的1V4啊，就算把CBT擊倒的兩人算上，堡壘也只淘汰了三人啊，smallcc不配擁有姓名嗎？這樣一條微博，明顯是粉絲不服瀉火，也能理解，但是很難理解經理居然會點贊這樣的微博！

看來，醒目那一根刺還是深深地扎在了心裡。

其實第三局被4AM團滅，同樣有這樣的可能：大京先擊倒4AM一人後，著急補人，結果被淘汰，最後跟4AM正麵糰戰，3V2也是著急補人結果被小鹿一穿三，不知道是SMG急於拿分，還是見到4AM分外眼紅，想第一時間按死對手，總之這樣的操作讓他們自己漏洞百出，因小失大。

在絕地賽場上，信息錯綜複雜，高光和被高光是很常見的，如果17當初能夠虛懷若谷，大方面對一雷四響，或許並沒有什麼，4AM被高光的次數也不少，什麼橋頭一穿四、破綻樓之類的，如果選手甚至管理層去執著於這個點，那4AM也不會成為當下PUBG第一人氣戰隊。

只有放下才能走得更遠，別讓一時的憤怒轉化為仇視，更別讓仇視成為自己的心魔，因為最終桎梏的還是自己！
