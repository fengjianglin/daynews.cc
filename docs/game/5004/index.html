英雄聯盟雲頂之弈挖掘機更新後，這麼玩一擊即中

2019-12-03 0條評論 15次閱讀 0人點贊

摘要

雲頂之弈玩法中有一套黑科技是以挖掘機技能輸出傷害為基礎的“法”掘機，元素崛起版本中推出挖掘機之後並不是很受歡迎，不溫不火的狀態，之前是熟於人氣還比較高的低費鬥士牌，新版本中成了掠食者打工仔之後就真的是沒多少人喜歡了。

雲頂之弈玩法中有一套黑科技是以挖掘機技能輸出傷害為基礎的「法」掘機，元素崛起版本中推出挖掘機之後並不是很受歡迎，不溫不火的狀態，之前是熟於人氣還比較高的低費鬥士牌，新版本中成了掠食者打工仔之後就真的是沒多少人喜歡了。但經過多次實踐，發現挖掘機也能大放異彩的方法，那就是變成魔法師，只要裝備成型，一擊即中。

　　一起來看看詳細陣容吧！

　　魔法師職業：3/6魔法師，在這裡法師有一半的幾率可以連續釋放兩次技能

　　辛德拉、吸血鬼以及帶著法師帽的挖掘機共同構成3魔法師職業羈絆，這就是形成的主要羈絆，可以釋放兩次技能的幾率為50%，保證了輸出量的強大。

　　掠食者職業：3/3掠食者，掠食者的構成輸出傷害的話能直接擊殺生命值在25%以下的敵人

　　蠍子、狼人、挖掘機組成3掠食者職業羈絆，強大的擊殺敵方殘血隊員的能力，如果對方英雄的回復能力很強，那麼該羈絆對於敵方的壓制也會很強，能夠很好保存自己。

　　海洋種族：4/6海洋，為隊友回復法力，25點/4秒

　　4海洋種族羈絆能很好地回復我方的法力值，有利於加快技能釋放的速度。

　　守護神，有錘石和泰坦組成的2守護職業羈絆能夠提升防禦能力，更好地保護隊員。

　　鋼鐵種族，能更好地保護挖掘機的生命，延長輸出時間。

　　核心英雄選擇：雷克塞 虛空遁地獸，技能為狂怒之咬，選擇的裝備為法師之帽+熾天使之擁+滅世者的死亡之帽。很容易達到3星，到了3星之後傷害輸出會很高，加上裝備的加成，即使在面對滿血有肉裝的對手坦克時，也能一擊斃命。

　　辛德拉 暗黑元首的技能是水力法球，裝備選擇燭光護手和莫雷洛秘典，辛德拉作為人氣較高的2費海洋法師牌，長期利用或者前中期的積累中作用都很可觀。

　　兩個和新英雄對裝備要求比較高，其他主要選擇防禦裝備就OK，這個陣容現在還是很有吸引力的，不過到之後的排位賽可能沒其他陣容那麼穩定，所以現在可以盡情地嘗試。
