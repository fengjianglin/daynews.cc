EDG新上單曾是主播，比賽頻犯路人惡習，令廠長頭疼

2019-12-23 844條評論 3,184次閱讀 2079人點贊

德瑪西亞杯之所以外號叫做「EDG杯」是因為EDG包攬了很多年的德杯冠軍，但是今年EDG的情況，卻讓粉絲們捏了一把汗。

首先是ADCiboy轉會VG，比賽在即ADC轉會，新AD與輔助磨合時間較短，配合併不默契。

其次就是廠長轉型教練，選手轉型教練初登賽場，面對其他戰隊的老油條「教練」顯得有些緊張，BP上雖無大失誤，但同樣的也沒有太多亮眼之處，期待後續表現吧。

今年的EDG，在我看來準備還是充分的，打野和上單都有三名選手進行輪換，代表隊伍可以更換多種風格，對於一個新人新教練的隊伍來說，多變的風格能夠在BP上佔一些優勢，但在遊戲中，可能會因為配合不熟練等原因出現問題。所以是一把雙刃劍。

EDG今年除了劍仙jinoo外，還買回來了原VG的上單aodi以及「50W狀元上單」xiaoxiang，aodi這名選手大家都比較熟悉，就不拿出來說了，重點我們來看看今年拍下轉會期青訓最高級的xiaoxiang，看來這50W花的到底值不值。

首先我要先給水友們糾正一個問題，就是xiaoxiang是主播轉行並非純青訓新人，他原來是huya的一位主播，擅長玩劍姬，別的英雄不敢說，但劍姬玩得確實不錯，算是國服分段最高的劍姬玩家之一。

之前有看過他直播的水友應該清楚，xiaoxiang是2002年出生的歲數不大，網友評價比同樣玩劍姬的可樂和14要強，但缺點就是英雄池太淺，也不會打船長和諾手。

所以說EDG買的並非是從零開始的純新人，而是一個國服路人王，這樣去想50W的價格，也就合情合理了。

當然Rank猛不代表比賽猛，這是水友們都清楚的事情，那就來看看比賽，我拿了兩場德杯的比賽數據來做分析，對GD那場就不詳細說了，一個LDL隊伍的數據也沒有太大的參考價值。

也基本上沒有太多可以說的，對位傷害碾壓一半，對線期幾乎沒有太大的問題，最後還拿了個MVP。

來說說打V5的這場，奧恩打鱷魚，鱷魚征服者奧恩餘震。

對線期間和平發育，這個版本鱷魚想要打死奧恩不太容易，到十分鐘小龍團之前，奧恩和鱷魚的補刀都幾乎是持平的

十分鐘小龍團戰

奧恩搶先TP落地，下路兩人跟潘森和岩雀在河道SOLO,奧恩叫羊分割戰場但是撞了V5的下路，逼掉兩人閃現，隨後鱷魚TP落地完成收割。

這波團在我看來，對面兩人脫節，鱷魚TP為落地，賞金有大，三名隊友都在河道的情況下，奧恩應該選擇撞潘森和岩雀，而不是選擇分割對面下路，最後導致一個沒殺死鱷魚落地完成收割。

這波為什麼我要拿出來說？因為這是明顯的路人局思維，想著單殺脆皮，而職業隊伍的打法其實更偏向於團隊。目前的版本，10分鐘的時間段內女警並沒有太高的戰鬥力，其實沒有必要去管他，而賞金這個英雄在當前版本內，可以說是T0級別的ADC，前期有高爆發和快速支援能力，所以奧恩的正確操作應該是幫助隊友擊殺河道的兩人。

當然對於一名主播剛轉職業的選手來說，這種問題存在也是正常的。

十二分鐘上路越塔

EDG四人拿下峽谷先鋒後，選擇上路推進並且想要越帶線的女警，奧恩空大泰坦空鉤後，奧恩又一頭撞了上去，但此時的隊友已經要撤退了，由於W交早了，奧恩被前來支援的莫甘娜Q在塔下最終死亡。

通常職業選手在空掉關鍵技能的情況下，會選擇放棄這一波，而路人局則不同，路人局可能更傾向於通過自己的操作或是其他辦法，找到對手的失誤從而越塔，哪怕一換一都是不虧的。但這是比賽，首先不談能不能一換一，即使是一換一也並不賺，因為這波的決策本該是0換1再加一座外塔。拋開溝通不及時的原因，路人局思維又害了xiaoxiang。

十五分鐘小團戰

奧恩被四人擊殺，硬要說奧恩有鍋也沒問題，但是我感覺這就是單純的沒想通會先手動一個肉。

二十分鐘高地團戰

這波其實算是奧恩的Carry點，大招開到四人，鱷魚開大後也被他第一時間留在牆角拖延了鱷魚進場的時間。抓的時機沒問題，團戰的把控也沒問題，加分項

二十分鐘團戰

狹窄地區團戰，大招分割戰場，基礎操作沒什麼好點評的。

三十分鐘團

這是遊戲結束前的最後一波團戰，奧恩開大想把兵收掉，想法是沒問題的，但是在我看來沒必要。因為即使沒有小兵靠鱷魚和潘森輪流抗塔，早已沒有多少血量的防禦塔還是會被點掉，畢竟後期女警點塔的能力還是很強的。

總結

比賽結束後，從傷害來說奧恩的傷害比鱷魚高，對線期互相打太極拳沒有什麼值得說的。xiaoxiang作為一名新人，在局勢把控上有所欠缺，面對突發性的團戰，很多時候沒有辦法給出最優的處理。

思維還沒有從路人王轉變過來，雖然經歷過一段時間的青訓，但畢竟經驗不足，操作上來說是沒問題的。期待能夠跟EDG碰撞出更多的火花。

目前看來50W花的肯定是不虧的，年齡小有潛力，但是後續會怎麼樣就不得而知，畢竟LOL歷史上的天才選手有很多，最終隕落的也有很多。
