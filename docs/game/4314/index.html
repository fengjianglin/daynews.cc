史克威爾經典遊戲~《半熟英雄》一款天馬行空的佳作

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

《半熟英雄》原名《Hanjuku Hero》是SQUARE（史克威爾）在88年推出的一款創意型作品，最初登錄於FC平台，92年後又發布了SFC版的續作《半熟英雄，世界半熟》。

作為一款創意型遊戲《半熟英雄》之所以能成為經典的原因不僅在於其融合了以往RPG遊戲所特有的角色扮演，寵物養成，即時戰鬥等內容，還再其遊戲本身所獨特的世界觀和誇張的故事情節，以及加入蛋獸召喚這些新穎的元素而形成的一款風格獨特的遊戲作品。

遊戲背景

在一個遙遠的星球上，有一個半熟王國。半熟王國的國王是一個愛吃蛋的小孩子。只要有蛋吃他可以什麼都不管，他手下的大臣們為此也十分擔憂。這個半熟國王有一個十分厲害的本事，他可以從各種各樣的魔法蛋中召喚出各種威力無比的怪獸。有一天，邪惡的全熟大軍大舉來犯。於是半熟國王帶上一群魔法蛋和親信們開始出征了

系統介紹

半熟英雄遊戲類型屬於即時性策略類遊戲。主要分為三個獨立的章節，但難度有所增加。玩家初始擁有一座城堡和三名角色(一名主角和兩名大臣)，其中城堡的稅收是固定的，但可以通過升級來提升相應的防禦，不過這隻限有將軍的城堡，因為空城的防禦力為0。武將也是可以招募的且與其關係為僱傭關係，當金錢不足時會自動脫離。遊戲的畫面隨著時間的變化而分別呈現出戰爭和內政兩部分，每月一開始就進入戰爭部分，而月末進入內政部分（在戰爭部分時如果聽到背景音樂逐漸變小直至消失時即意味著將進入月末內政部分）當進入內政部分時遊戲畫面會變成相應的舞台模式，並發生相應的事件，例如來自本國農民的收成報告（最好是2倍收入，最差是1/2），志願者（將軍）的加入，迷路的女孩（ 勒索錢財），以及地震（自己的城LV降低）等事件。然後就是內政畫面，此時右上角為自己的錢數，中間為可執行的命令。相應指令如下圖所示

戰鬥方式

遊戲中的戰鬥模式可以分成「野戰」、「防衛戰」、「攻城戰」等，其中野戰是指當玩家在移動時遭遇對手軍隊而進入戰鬥的模式，至於「防衛戰」則是指當玩家自己的城堡遭受敵人攻擊時而發生的城堡守衛戰( 不可使用道具王牌)，。而所謂的「攻城戰」，則是由我方進攻對方守城的戰鬥，勝利後玩家將可將該城納為己有。

蛋的祈禱

不知各位玩家在玩的時候有沒有注意到隨著蛋的使用次數的增加召喚出來的蛋獸反而越來越弱，當達到一定召喚次數時，無論召喚多少次召喚出來的蛋獸都是初始蛋獸蛋超人，這可怎麼辦？有沒有升級的方法？其實是有的，答案是——祈禱（蛋的重做或者升級）。

在地圖上有幾座無法攻佔的怪物城堡，當你把相應的怪物打敗後再進入就能進行相應的祈禱，進行祈禱的將軍或者進入的怪物城堡的不同，產生的效果也隨之不同，如果按照某種特定的順序進行相應的祈禱就能召喚出相應系列的最強蛋獸。例如神系的漢迪斯，半魚的美杜莎等

可探索的洞窟

幾乎在每張地圖上都有一個奇怪的洞窟，它是用來幹什麼的？答案是用來——探索的。當你放一支軍隊進去時可能會出現兩種情況中的一種，一種是直接消失，另一種是在月未找到強大的道具，並回到相應的城堡（ 消失意味著死亡，所以要預先存檔）
