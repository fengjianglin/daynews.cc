在王者榮耀中，雖然說英雄的一技能沒有大招重要，但是在平時的時候還是要靠一技能打輸出的。有的英雄一技能是可以打出高額的傷害，也有的英雄可以利用一技能打控制，而有的英雄卻能利用一技能給隊友增加高額的增益。那麼如果英雄一技能增強50倍的話，王者峽谷會發生什麼樣的變化呢？宮本要上天，但守約默默卻退出群聊！

宮本這個英雄由於經常被系統削弱，所以經常成為玩家的調侃隊友，一到系統更新的時候，就會有大量的網友詢問官方，這一次宮本一技能取消格擋效果。那如果把宮本的一技能增強五十倍會發生什麼樣的現象呢？這個時候宮本會上天，因為他的一技能可以格擋其他英雄的技能，而且傷害增加五十倍，可以利用一技能不斷向前推進，沒有任何人能傷害到他！

虞姬在射手中算不上強勢，但自保能力卻是頂尖的。她的二技能可以格擋大量的物理傷害，並且在開啟以後增加自身的移速，避免刺客的突進，而一技能可以不斷遠程消耗敵人，保證自己輸出位置的安全。如果她的一技能能夠提高50倍的話，虞姬是完全不怕任何敵人的，她完全可以在安全敵方不斷向前釋放一技能打傷害，等敵靠近二技能加速撤退就行了！

相信很多玩家十分害怕東皇太一，因為他可以通過一技能不斷召喚黑色法球，被黑色法球觸碰到以後傷害雖然不多，但會給東皇太一提供治療效果，導致很難打掉東皇太一的血量。那麼他的應增加50倍會發生什麼樣的現象呢？本身東皇太一的一技能就能增加3個法球，擴大50倍的話就是150個法球，在王者峽谷中基本上是橫著走！

看到其他英雄的一技能增加五十倍以後，百里守約默默退出了王者峽谷的群聊，獨自浪跡天涯去了。因為在王者峽谷中再也沒有他的容身之地，他從一個射手徹底成為了一個輔助，再也無法面對其他英雄了。他的一技能技能雖然偵查的面積擴大了，但卻沒有什麼實質的效果，僅僅是監測敵人而已，所以一代T0射手徹底沒落！你們認為還有比百里守約更慘的英雄嗎？
