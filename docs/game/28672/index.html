德杯出戰名單一覽：繼FPX後，IG也上二隊？可惜這次沒有蕉太郎

2019-12-22 2,573條評論 4,366次閱讀 298人點贊

冬季轉會期結束，今年的最後一場比賽即將開打，作為眾多戰隊拿來磨合新人嘗試新打法的賽事，官方也漸漸默許了這種行為。就在前不久，FPX剛剛公布了自己的德杯參賽陣容為全員二隊，就連教練都是二隊的，沒過幾天，IG也公布了自己的德杯陣容，居然一個S8的奪冠陣容選手都不在。

​

看完教練以及五個位置的選手大家可能會發現，好像好幾個都還挺熟悉的，不過可惜的是，S8的選手是一個都不在了。唯一熟悉的就是在一隊露過面還打了世界賽的小樂言，以及VG的下路二人組也來了，加上上單、教練以及中單都是二隊的選手，IG此舉顯然也是希望鍛煉一下自己的新選手和新陣容了。

​

之前小樂言在S賽的表現讓觀眾和粉絲們可謂是恨鐵不成鋼，但是不少樂言的粉絲都是為樂言證明表示，樂言只是因為大賽比較緊張打的比較拘束，但是在LDL的時候還是很給力的。不知道這次在德杯上樂言能否好好把握一下機會為自己正名一下呢？

​

不過除了IG在磨合陣容，RNG也是如此，UZI因傷病暫時休賽，急需磨合一下新陣容的RNG也是派上了新人中單月開和打野小龍堡，就是不知道在UZI不在的情況下，還有如此多的新人在此，能否打出理想的成績呢？讓我們拭目以待了。
