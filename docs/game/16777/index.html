獨立團隊格鬥遊戲新作《鬼斗》開放體驗 主打無數值即時競技格鬥

2019-12-16 947條評論 3,549次閱讀 1354人點贊

獨立遊戲開發廠商G5宣布，旗下智慧型手機遊戲新作《鬼斗》（Android）於今日（4）日開放玩家搶先體驗。官方表示，遊戲以「熱血丹心！戰出真本事！」作為口號，主打無數值即時競技格鬥，希望帶給玩家兒時傳統大型機台的格鬥體驗。

【以下內容為廠商提供資料原文】

彩墨斗心，無數值真格鬥！

　　《鬼斗》採用東方傳統彩墨風，遊戲戰鬥場景、人物都用極具特色的水墨風格進行包裝。讓玩家對決都處於精美的山水繪畫之中。此外遊戲致敬童年傳統大台格鬥的經典，摒棄數據養成，只要努力誰都能成為格鬥大師。

簡化操作，成就對決之王！

　　為了讓玩家能不受場景、時間限制，隨時隨地，拿起手機就能享受一把3 分鐘KO 對手的比賽。製作團隊簡化在手機上的操作方式，只需要簡單方向鍵搭配輕、重攻擊鍵就可以變化各種招式。減去搓招煩惱，即時配對、不再擔心網路延遲殺，流暢對戰，感受拳拳到肉的酣暢淋漓。真人競技，控血反殺創造無限可能。

硬派設定不妥協、目押立回全都要！

　　雖然針對移動設備進行操作簡化，但是格鬥的內魂是不會發生改變的。經典格鬥所必備的目押、立回等系統被大多完整保留了下來，為的就是給硬派橫版即時格鬥愛好者們在移動平台上有一個可以秀技的地方。

特色造型顯個性，碾壓對手凹造型！

　　持續更新的超過12 個性格各異的角色，33 款炫酷造型持續更新中，隨著不同心情更換角色與造型，總有一個角色是你喜歡，總有一款造型是你夢寐以求。

　　目前《鬼斗》仍處於測試狀態，並不代表最終遊戲品質。如果您對於遊戲有任何意見或者改進建議的話，歡迎到相對應社群給予意見回饋。希望遊戲能滿足大家對於格鬥遊戲的期待，也希望能與大家一同成長。
