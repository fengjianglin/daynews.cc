CFM中越丨3Y有話說：eStar獲得預選賽冠軍 隊伍的努力沒有白費

天天要聞
22 小時 ago

剛剛結束的CFM亞洲杯中國區總決賽中，eStar以3比0擊敗R.LGD從而獲得最終的冠軍，也獲得了代表中國出征亞洲杯決賽階段的資格，這對於三歪以及eStar而言，無疑是一大突破。

【eStar的第一個冠軍 我們十分激動】

對於三歪而言，中國區預選賽的冠軍也是eStar進入席位制後獲得的第一個冠軍，三歪表示「能夠獲得中國區預選賽冠軍對我來說是很振奮的一件事。因為很久沒拿冠軍了，也很想拿一個冠軍，所以我們隊伍而言這也是很大的突破。」

提到這個冠軍時，三歪表示「這個冠軍是我們來到estar的第一個冠軍。這個冠軍對於我和隊伍是很大的鼓舞，也讓我們接下來能有更好的備戰去應對接下來的比賽。

通過三歪的自述可以看出他們對冠軍的渴望，在之前的NEST2019總決賽中的敗北，令他們與冠軍榮譽擦肩而過，而這次的預選賽冠軍無疑給隊伍打了一劑強心針。

【做好自己 成eStar獲勝法寶】

由於eStar在半決賽以及決賽中分別遭遇Q9以及R.LGD兩支強敵，在面對Q9的比賽中，三歪提到「之所以能夠擊敗Q9，是因為我們心態的調整，從以往的畏手畏腳到如今的敢打敢拼，提升了隊伍的自信心，從而也幫助我們獲得比賽的勝利。」

如果說面對Q9，eStar憑藉著對自己的信心取勝，那麼在迎戰R.LGD，eStar則憑藉豐富的經驗去擊敗對手。「由於R.LGD的大賽經驗較少，因此在比賽中，我們便與對方拼殘局處理能力。」

以此可見，在面對R.LGD時，eStar已經做好了充分的準備。當然對於eStar而言，本次比賽啟用了雷靜這名新人，隨著雷靜的加入，也令eStar有更多變動的戰術，從而在賽場中更主動進攻。

【期待決賽 希望冠軍留在中國】

eStar在小組賽以及半決賽與決賽的亮眼發揮，讓我們看到這支隊伍的快速提升，而在三歪的眼中看來，四名隊友的穩定發揮無疑成為eStar獲勝的關鍵，真是他們的敢打敢拼也或許讓三歪看到自己曾經的影子。

在總決賽中，eStar率先遭遇Shine，對於這支隊伍三歪表示並沒有過多的了解，但是對於eStar而言，只要將訓練賽的戰術與打法全部在賽場中展現，那麼他們更相信自己能夠進入最終的決賽階段。

若在決賽中與AG再次遭遇，eStar也將更全力以赴爭取獲得最終的冠軍榮譽，當然無論最終哪支隊伍獲勝，三歪更希望冠軍榮譽留在中國。

目前三歪與隊友再次投入到緊張的備戰中，對於他們而言，在最終的決賽階段中，他們將展現自己的全部實力，讓越南賽區隊伍看到中國賽區隊伍的強勢，從而將最終的冠軍榮譽再次留在中國！
