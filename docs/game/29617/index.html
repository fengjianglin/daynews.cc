暗黑破壞神2 難倒新人的普通大菠蘿也能輕鬆擊殺

2019-12-25 3,534條評論 4,707次閱讀 1780人點贊

好多人說暗黑破壞神2對新人不太友好，一方面是裝備太多太雜了，另一方面就是剛進入遊戲世界的人物刷怪不太輕鬆，尤其是在遇到BOSS的時候，第一幕BOSS安達利爾我們可以放風箏搞定，第三幕BOSS墨菲斯托我們可以隔著河搞定，就是這個第二幕的督瑞爾和第四幕的暗黑破壞神擊殺難度非常的大

曾經就有人戲言暗黑2地獄還是那個地獄，普通是噩夢，而噩夢也更像普通。督瑞爾是因為房間太小無法跑位，而且他還有聖冰減速光環讓人受不了。暗黑破壞神他家廣場地方是夠大了，就是攻擊方式太多攻擊力太高，搞的玩家一碰到他就發怵，這倆BOSS也就成了新人的噩夢。

暗黑破壞神的攻擊方式

知己知彼百戰百勝，想戰勝他就得先了解他，暗黑破壞神在普通暗度下的等級是40級，他的技能非常多，我們要知道他的技能才可以在他攻擊的時候方面我們躲閃，不至於被他直接秒殺掉。

暗黑破壞神的技能有骨牢、寒冰之觸、火風暴、火焰新星、火牆和地獄閃電，從名字就可以看出來，暗黑破壞神的攻擊基本為元素攻擊，而且偏向火和電元素。這幾個技能當中需要我們大範圍移動躲閃的就是火風暴和地獄閃電，這倆都是那種持續攻擊型技能，被打到後接踵而來的攻擊就能秒人，所以一定要躲閃。

如何應對暗黑破壞神

1.我們大眾玩家在打到第四幕的時候基本都在二十級左右，我的建議就是最好在24級的時候去擊殺暗黑破壞神，因為人物在24級的時候可以學習很多新的技能，而且大部分還都是非常有用的大技能，這樣我們就有更多的手段來對付他。

2.在清理完小怪後，首先在離他較遠的地方開一個傳送門，萬一掛掉後還可以直接進來撿屍體繼續戰鬥。千萬不要在離他近的地方開門，大菠蘿會對著傳送門放骨牢，門被困住你死掉進來直接就站骨牢里無法移動了，接著還是會被殺死。

3.適當的堆一下自己的抗性，在前三幕看到火抗和電抗高的裝備一定要留一下，殺大菠蘿的時候穿起來讓自己的火焰和閃電抗性高點，確保自己不被秒殺。

4.大菠蘿嘴裡噴閃電的時候一定要繞圈閃躲，不管是逆時針還是順時針都可以，直線跑動幾乎躲不了，還有他在使用火風暴的時候閃開火焰覆蓋區域。這倆技能會秒人，其他的幾個技能不需要閃躲，看命就行了。

各人物基本的擊殺大菠蘿的策略

知道了應對的策略就可以去戰鬥了，每個人物不同他們的戰鬥方式也不一樣，簡單說說僅供參考。

女巫最簡單，傭兵吸引注意力後自己在另一邊使用靜電力場，直接可以將BOSS的血量削弱到底，然後再補一次攻擊他就掛了。

死靈法師詛咒衰老，配合土魔降低他的攻擊速度和移動速度，土魔要不斷的召喚到他面前，骷髏死完了去別的地方繼續召，重複個三四次暗黑破壞神也就掛了。

近戰人物近身打就行了，看見火風暴和閃電記得躲閃，不被秒掉吃藥補血就行了。最好有帶有壓碎性打擊屬性的武器，殺大菠蘿非常快。

有遠程攻擊能力的就是放風箏，比如弓馬和陷阱刺客，離他遠點進行攻擊，閃躲大菠蘿技能時也更容易。

說的這些希望可以幫到你，我是一個暗黑老玩家，歡迎大家留言評價和討論。
