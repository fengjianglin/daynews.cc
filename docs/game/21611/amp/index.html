《英雄聯盟》新版大亂鬥勝率前五英雄，Mort明年專註雲頂之弈模式

天天要聞
2019-12-19

喜歡極地大亂斗模式的玩家可能已經留意到，最近幾個月來，設計師Mort一直對這個模式進行一些平衡性調整，包括上周上線的9.24版本中，也有一些英雄得到加強或者削弱 。畢竟，在扭曲叢林模式移除後，極地大亂斗和雲頂之弈模式成為了主要的娛樂模式，如果你不想下棋的話，那極地大亂斗模式就是唯一的打發時間選擇了。

​

在12月18日時，Mort在推特上分享了新版本大亂斗模式的一些數據。他表示在上周的改動後，遊戲目前處於一個不錯的狀態，英雄的勝率區間是42-57%，這次削弱的英雄勝率仍然超過50.5%，加強的英雄仍然低過49.5%，但總體來說還不錯。畢竟，Mort不想大刀闊斧地砍死某些英雄，以下是當前版本中勝率最高的五個英雄。

1-提莫

對於那些遇到提莫卻沒有警戒心就想吃血包的人，提莫的蘑菇可以發揮巨大的作用，會讓對手不得不花費金幣購買藥水。如果出了冰杖面具的減速套，在團戰中也能夠起到很強的控場能力，再加上他的致盲效果能夠限制突進物理英雄對後排的輸出。

​

2-茂凱

茂凱是一個大亂斗模式的完美英雄，兼顧輸出、續航、坦度、控制和AOE，而他的大招在嚎哭深淵模式中，能夠作為一個強力的先手開團敵人。尤其坦克裝備成型後的茂凱，會讓對手無法快速處理掉，己方的後排就能夠盡情輸出。

3-凱特琳

嵐切的改動對於凱特琳而言是一個加強，讓她可以更好地過渡前期，相比之下，先出嵐切的勝率比無盡之刃的高，目前暴擊流勝率比穿甲流打法要高。W技能在大亂斗模式中是神技，能夠讓己方獲得草叢的主導權，然後你可以利用手長的優勢進行壓制。

4-斯維因

強力的控制是大亂斗模式中最實用的工具之一，斯維因的E技能在這個模式中就很強，他能夠將敵人拉向自己，讓隊友可以集火秒殺。斯維因被動收集魂屑可以回復法力值和生命值，讓他可以有很強的續航能力，大招在團戰中能夠分割戰場和限制走位。

5-吉格斯

由於嚎哭深淵的地圖十分狹窄，吉格斯這種技能距離長的英雄一直受到青睞，他的W和E還有小範圍的群體控制能力，擊飛效果可以配合亞索。在有了帽子後，吉格斯的大招很容易打殘敵人和完成收割，Q技能可以一直poke消耗對手，帶了清晰術就不用擔心藍耗。

以上五個英雄是全分段勝率最高的，但如果是高分段中，茂凱、迦娜、辛吉德、沃利貝爾和特朗德爾的勝率則是最高。看起來，在高端局中帶雪球強開的打法更加流行，玩家也願意出肉。至於全分段勝率最低的五個英雄，則是亞托克斯、千珏、雷克塞、巴德和阿茲爾，這些英雄都需要一些操作，以及對打法有深刻的理解。

​

雖然提莫的勝率有些超標，但西方很快進入聖誕假期，9.24B版本不太可能會對大亂斗模式進行後續的改動。事實上，這也是Mort最後一次負責極地大亂斗的平衡性改動了，他在個人推特上透露了自己的去向。從明年開始，他就會將這個工作交給模式團隊的設計師，而自己的工作重心則是專註於雲頂之弈模式的遊戲改動。
