莫問紅塵煩惱事，逍遙走遍天外天，《天之痕》那些不為人知的故事

2019-12-17 2,896條評論 4,553次閱讀 2447人點贊

大家好，我是龍牙之評，今天我們繼續聊中國武俠遊戲，上周曾向大家介紹過擁有龐大公司背景的《軒轅劍叄：雲和山的彼端》，遊戲所展現出中的西方文化碰撞，讓當時不少玩家留下深刻印象，既然提到這部《軒轅劍》有史以來評價最高的作品，那麼自然也是少不了另外一款，被無數人稱之為難以超越經典遊戲，它便是《軒轅劍叄》外傳《天之痕》。

大多數人並不知道，這款在當年感動無數玩家的遊戲，其創作者背後居然還隱藏著一段不為人知的勵志往事，今天龍牙就為您講述當年遊戲與製作人吳欣叡的故事，以及《天之痕》之所以成為歷代《軒轅劍》中「最受歡迎作品」的真正原因。

毛獸的奮鬥史

關於吳欣叡（毛獸）的故事，我們要從90年代中期開始說起，隨著《軒轅劍貳》在日韓地區上市，大宇也逐漸在海外市場上嘗到甜頭，所以除了自研產品以外，也開始嘗試代理一些其他國家的遊戲，正是在這樣的環境下，當時出身於日語專業的毛獸，便以文編一職進入公司，每天的工作就是負責幫某款進口遊戲作文字翻譯，此時的他還並不知道，自己在日後居然能成為一款遊戲的製作人。

後來由於種種原因，毛獸所翻譯的那款遊戲並沒有被引進成功，而最初把他介紹到公司的大學學長，也就是從《軒轅劍壹》開始就擔任配樂的「蘇竑嶂」恰好在此時選擇離開大宇，於是毛獸便抓住這個機會，自告奮勇的向蔡明宏表示自己很喜歡音樂，並希望能在日後承擔起學長的工作，就這樣毛獸成為了DOMO小組的新進成員。

很快毛獸就開始人生中的「第一次」配樂工作，幫一款名叫《阿貓阿狗》的遊戲製作背景音樂，雖然並非音樂專業出身，但憑藉對遊戲行業無比熱忱，不但出色完成所有配樂工作，甚至還抽出時間對遊戲內的文字對白進行潤色與調整，強大的學習能力與拚命的工作態度，讓整個DOMO小組紛紛對這個剛入行的年輕人刮目相看，《軒轅劍叄》開發期間毛獸被蔡魔頭（蔡明宏）委以重任，除去幾乎以一人之力肩負起遊戲BGM的製作，不少故事劇情與人物描寫，也正是在毛獸的參與下而變得栩栩如生，而他也成了當時唯一文案與配樂兼具的製作人員。

時間來到2000年，也就是《軒轅劍叄》正式上市後，DOMO小組又開始馬不停蹄的籌劃著下一款遊戲的開發工作，而隨著大宇另外一款RPG《仙劍奇俠傳》的熱賣，蔡魔頭也希望能藉此機會在新作中加入一些類似「仙劍」式的感情要素，只可惜之前負責《軒轅劍叄》總編劇的鮑弘修，更擅長的是構建出那些恢弘龐大的世界觀格局，而並非去描寫一段兒女情長的愛情故事，無奈之下DOMO小組又再次把目光投向那位身兼數職的年輕人，於是毛獸又再次承擔起雙份的工作，除了原本的音樂製作以外，還要以一位遊戲策劃的身份，去編寫一個與之前風格完全不同的《軒轅劍》故事。

遊戲要設定在哪個朝代？登場人物又該如何設計？對於之前完全沒有相關經驗的毛獸來說，擺在他面前的就如同一道世界級難題，雖然毛獸沒有原創劇本的實力，但如果採用《仙劍》作為參考模板，在結和《軒轅劍》本身進行二次創作的話，憑藉自身的文筆水平，或許也能編織出一個帶有豐富感情色彩的故事。抱著試一試的心態，毛獸將「仙劍」式的愛情觀代入到一段具有上古神話色彩的背景當中，隨後他又把《軒轅劍貳》和《軒轅劍叄》的人物與背景設定，與本次的故事主線做了某些巧妙的連接，在毛獸的不斷努力下，這些看似毫不相關的內容，竟完美融合在一起，並順帶填補前作中那些尚未交代清楚的若干謎團，於是一款同時具備神話、歷史與愛情元素的作品《軒轅劍：天之痕》便就此誕生。

永恆的經典

《軒轅劍：天之痕》或許在格局上並不如前作那麼宏偉，但依然憑藉著獨特的敘事手法，為玩家們講述了一段充滿歡笑與淚水的冒險故事，從小身邊復國使命的前朝遺孤陳靖仇，受師傅之命尋找傳說中流落在人間的上古神器，在誤打誤撞之間，他遇到了乖巧可愛的銀髮少女於小雪，以及個性直爽的鮮卑族姑娘拓跋玉兒，隨著劇情的發展三人也在一次次的打怪、吃飯、睡覺、女裝等事件中，建立起極為深厚的情感羈絆。

雖然研發周期尚不及之前的四分之一，但《軒轅劍：天之痕》在視覺效果與背景音樂等方面，卻擁有著並不遜色於《軒轅劍叄》的製作水平，無論是氣勢磅礴的通天塔，還是繁花似錦的聚落城鎮，又或是如仙境般的天外村與海底皇宮，一邊遊覽著DOMO小組獨特的水墨畫世界，一邊聽著有古箏、琵琶、編鐘等中國古代樂器所演奏的優美旋律，讓玩家彷彿置身在一幅古色古香的畫卷當中。

玩法設計方面遊戲更是在繼承前作優點的基礎上，額外增加了不少豐富有趣的內容與元素，除去前作中的法寶、護駕等功能得到進一步的加強，更是首次增加類似電子寵物般的符鬼系統，玩家可以利用煉妖壺捕捉到的妖魔來對它進行投喂，除了能提升符鬼的各項基本屬性以外，還能從中學習到不同種類的五行術技能，如果搭配得當的話，這些小可愛甚至還能在戰鬥中起到決定勝負的作用，讓遊戲可以玩出「中國寶可夢」的感覺。

另一方面《軒轅劍：天之痕》還在許多意想不到的地方埋藏了大量的彩蛋，比如從某款作品穿越過來的藍格怪衣，東萊附近兩次恰好路過的小青蛙，水井下那個別有洞天的加班工作室，以及某隻劇情boss讓人哭笑不得的吐槽方式，如果玩家想要觸發這些隱藏要素的話，除了要像某些日式RPG一樣來一場地毯式搜索，同時還要在戰鬥過程、地圖場景乃至人物對白中留意其中的蛛絲馬跡，這些十分用心的內容與細節，令不少玩家在通關不止一次以後，也依然能在遊戲中找到許多深入探索的樂趣。

隨著的故事逐漸走向尾聲，遊戲劇情卻突然發生了180度的反轉，不善言辭的小雪選擇了與眾人不辭而別，了解到事情真相欲啊玉兒卻慘遭寧珂郡主的毒手，在伏羲神殿里，當小雪得知自己的犧牲能夠換來玉兒的復活，遊戲也在此刻迎來最艱難的抉擇，在結局的選擇上，由於玉兒線完整交代了其他人物的去向，因此也被玩家們視為官方欽定的標準結局，但實際上DOMO小組最初由於趕工的原因，一度只採用「如憶玉兒」的單線設定，也就是遊戲中的小雪結局，後來發現還有剩餘的開發時間，才繼續製作另一個分支，並在此基礎上增加結尾圖片CG，所以玉兒線才會顯得比較完整。

有不少細心的玩家發現，無論故事最後通往哪個結局，陳靖仇都不願意犧牲小雪為代價復活玉兒，小雪卻一直苦苦哀求著大家回心轉意，再結合玉兒毀容時小雪的割肉療傷，這位拓跋姐姐在小雪心目中的地位，甚至遠遠勝於遊戲里的第一男主陳靖仇，在不少一男兩女的RPG作品中，編劇往往會通過某些肢體接觸橋段，來描寫出男女之間的愛情故事，但《軒轅劍：天之痕》卻更多應用了刻畫人物內心的手法，讓三人在遊戲里產生真正的友誼，無論是為復活玉兒耗盡真氣的小雪，還是以七世代價讓小雪和靖仇知道真相的玉兒，正是因為這些簡單而又純粹的真摯感情，玩家才會被他們的故事所感動，並為最後的結局悲傷不已。

人物塑造方面《軒轅劍：天之痕》圍繞主角三人展開劇情的同時，也用細緻入微的筆墨刻畫出不少個性鮮明的遊戲角色，拯救天下蒼生獨自背負罪名的宇文拓，策劃魔界降臨費盡一切心機的獨孤寧珂，為復國大業而終其一生的陳老師傅，還有一路陪伴的主角成長的張烈大哥，在一段段虛幻與歷史相結合的故事中，他們無論是正義或邪惡的一方，都有著自己的理念、自己的苦衷與自己的堅持。

由於工期原因DOMO小組在製作《軒轅劍：天之痕》的同時，還分出一部分人手製作另一款SRPG《軒轅伏魔錄》，這也因此導致遊戲在不少地方存在瑕疵，儘管它並不是一款完美的《軒轅劍》作品，但遊戲在畫面表現、劇情描寫與氣氛營造等方面，在後續的系列中鮮有超越者，而大宇也憑藉該作在當年捧回無數獎項，之後的數年中《軒轅劍：天之痕》更是衍生出《上古神器》《軒轅劫》等一系列同人作品。我永遠也不會忘記，坐在藍色海面上彈奏的琵琶的玉兒姐姐，與夕陽下梳著銀色白髮小雪妹妹，那是他們生命中最美好的時光，也是我們此生中最寶貴的遊戲回憶。
