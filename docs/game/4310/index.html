我的世界：種子中輸入迷你網址，百里無生機，官方這是故意的嗎？

2019-12-08 0條評論 17次閱讀 0人點贊

摘要

今天，十方帶領大家探索的是萬能的種子。說道種子，種子的玩法非常多，經過長時間的發展。玩家們也是探索出來了各種各樣類型的種子，有著冒險、挑戰、資源種子等等。但是今天十方給大家介紹的是一個另類的種子，輸入的並不是我們經常挑戰的數字。

今天，十方帶領大家探索的是萬能的種子。說道種子，種子的玩法非常多，經過長時間的發展。玩家們也是探索出來了各種各樣類型的種子，有著冒險、挑戰、資源種子等等。但是今天十方給大家介紹的是一個另類的種子，輸入的並不是我們經常挑戰的數字。而是迷你網址，作為mc的老對頭，這個種子又會有著什麼樣的驚喜呢？

我們立刻便開始進入種子，為了更好的觀察種子，我們選擇創造模式進入。

剛進入種子地圖，就發生了頗有趣味的一幕。在剛剛落地的時候時候，周圍竟然有著兩條狼。不過可惜的就是，如果有著馴服工具。剛開局我們就可以擁有可愛的狗狗了。十方也是挑戰過不少地圖，但是開局就有著狼還是第一次見，大多數都是豬、羊等等。

十方並沒有去理會狗狗，因為我們是來探索地圖的，並不是生存的。路過狗狗以後，前面有著一條小河。在小河的後面，有著諾大的一座雪山。不過，環顧雪山一周以後，並沒有發現有任何的資源。

並非雪山沒有資源，從剛開局一直到雪山，除了有兩條狼之外，便沒有任何的資源。如果將水算成資源的話，那也是很少很少。其次，再次環顧40區塊以內的地圖，並沒有發現村莊、動物。這也就意味著，這個種子當中，你是很難生存的。甚至可以說是，這個種子不會讓你生存。

緊接著，我們再來看一看種子的大致地形。種子的地貌是呈昏暗形狀的，樹木也是同樣如此。給人一種非常不自在的感覺，即便是創造模式，你也不會待太久，更不要提去創造驚天動地的建築了。

緊接著，以x軸依舊走，大約在100區塊的時候，才發現了海洋。本以外，海洋面積會很大。結果，也是跟小。如果玩生存，你出生在海洋，你還有可能生存下去。如果是陸地，就比較危險了。在200區塊的時候，地圖主要地形依舊沒有發生變化，這種現象是很少見得。

整個種子是以陸地為主的，地圖體系80％都是雲杉體系，到處都是雲杉樹。雲杉樹的顏色是比較偏灰色的，長時間待在這個地圖放中，給人一種不自在的感覺。有興趣的小夥伴們可以去嘗試一下！

最後，我們來給這個種子評一下等級，暫且按星級劃分。 論資源和地圖環境，這個種子只能有著2星級別。跟「455」、「999」這些資源種子真的差太多太多了。 如果給這個種子說一個特色，那隻能是百里無生機了。喜歡冒險的小夥伴們可以去嘗試一下這個另類的地圖哦！
