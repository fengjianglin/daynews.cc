玩家用梗威脅官方，使命召喚無休模式回滾，更新只為保護一隻雞？

摘要

12月7日凌晨 ，使命召喚16進行了一次小更新，其中最關鍵的就是恢復了全年無休（7/24）訓練場地圖

12月7日凌晨 ，使命召喚16進行了一次小更新，其中最關鍵的就是恢復了全年無休（7/24）訓練場地圖。在12月4日第一賽季剛剛上線，訓練場被老地圖墜機之地替代，玩家等級重置等改動，引起了大批玩家的不滿。玩家表示墜機之地蹲人的太多，遊戲不暢快；升級速度降低，又要當肝帝；Reddit社區上也是吵得沸沸揚揚。

其中有網友甚至開始威脅IW工作室，他在Reddit上發布了一張瞄準一隻雞的圖片，並且表示「把全天候訓練場給我還回來，否則這隻雞就死定了」。開始的時候遊戲日報君以為這只是玩家的在用梗表達對遊戲更新的不滿，後來發現使命召喚多人遊戲的設計總監JoeCecot回應了，他的原話是「好了，好了，別殺這隻鳥（雞），之後更新」。

圖片來自Reddit

在這條消息後，搞笑的網友還回復「下次更新就是移除這隻雞」。玩過老滾的玩家一定知道，殺雞可是非常危險的，假如你在遊戲中不小心殺死了一隻雞，那麼不管什麼原因全村的人都會出來打你，可謂是「一人殺雞，全城追殺」。這個網友發這張圖，估計也是在暗示官方，你不改要被所有人罵。

當然，目前官方已經兌現了他的諾言，訓練場地圖回歸，墜機之地加入隨機地圖。這個COD4中的經典地圖未免也太慘了，上線僅僅3天就被下架，就像它的名字一樣「永遠的墜機之地」，「永遠」只存在一瞬間。使命官方還是很關注玩家動向，雖然現在遊戲仍然存在很多的小毛病，玩家反饋依然很多，但至少沒有特大範圍的差評。在對遊戲內容機制處理方面，使命召喚16還是可圈可點的。

此外，《使命召喚：現代戰爭》PC版開啟了限時促銷，暴雪戰網國際服標準版6.7折僅售281元（39.99美元），戰爭通行證未參與打折，促銷截止至12月9日凌晨。

使命召喚系列打折活動很少，各位想玩還沒有入手的大佬，可以考慮一波，這個價格已經低了。
