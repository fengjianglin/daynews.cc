《戰雙》天崩開局事件：自己做的那些“孽”，含淚也要吞下去

2019-12-06 0條評論 14次閱讀 0人點贊

摘要

諸君安好，霧夏菌報道。各位的《戰雙》刷到哪裡了？不得不說戰雙的確火了一波，不過不是正面的火，而是負面的。甚至還牽連出了什麼“把玩家當人看”、“自首”等諸多奇葩字眼。

諸君安好，霧夏菌報道。

各位的《戰雙》刷到哪裡了？不得不說戰雙的確火了一波，不過不是正面的火，而是負面的。甚至還牽連出了什麼「把玩家當人看」、「自首」等諸多奇葩字眼。

1、1萬黑卡天崩開局

這個就是事情的主要起因，原本遊戲是正常開服的，而且伺服器也頂住了大量玩家登錄的壓力，然而突如其來的一波部分玩家郵件，讓遊戲直接天崩。這個郵件的內容，就是直接送了1萬黑卡，

1萬黑卡代表著什麼，代表著可以直接開局40抽，而10連抽必出A卡，40抽必出S卡。挺好的不是嗎？問題就出在這個1萬黑卡，並不是所有人都有，只是部分玩家領到了，也就是說有一部分玩家，開局領先了其他玩家一大截，當你還在辛辛苦苦給B級角色升級進化，有人卻已經在用S卡大殺四方了。

2、處理方式「幼稚」

出了這種事情，肯定要第一時間處理，所以在開局不久便直接全服維護。正常的操作有兩個，一個是將錯就錯，對全服其他沒有領到1萬黑卡的角色補發。另一個就是回收領到1萬黑卡玩家抽到的角色，對他們進行部分回檔與對應補償，然後全服再發一份補償即可。

然而遊戲官方是怎麼處理的？

因為沒有辦法及時篩查所有領到1萬黑卡的玩家，或者說是怕麻煩不願意去篩查，只是全服發放一份停服維護補償，只有200黑卡、500基準券和一套露西亞的時裝，這和1萬的數額可是差得老遠了，至此玩家直接炸鍋。

3、把玩家當人看

這個其實有點斷章取義。

遊戲出了問題後，官方來了一波直播，這種時候還敢來一發直播，肯定師從梁靜茹，勇氣MAX。而在直播的時候，有很多彈幕在刷遊戲「不把玩家當人看」，人家回復了一句「就是把玩家當人看才直播的」。

這對話沒毛病。但單獨挑出來就完全變味了：把玩家當人看才直播的。是不是突然變得很拽很囂張了？同樣的一句話，上下語境分割開來，意思發生了天翻地覆的變化。所以在吐槽一件事情時，盡量要耐心一點，搞清楚整個過程，而不是摘取片段，很多人都吃過這個虧，包括本菌。

4、最終處理

最後這事搞得越來越大，無數玩家給遊戲刷低分評價，這個對遊戲的衝擊非常大。因此官方不得不下血本，直接全服每人一個自選S角色。所有的S級角色，隨便選一個。此外再補償一套限定皮膚。

這個可以說足夠誠意了。而對於那些一萬黑卡的玩家，官方也表態會全部追查。但結尾又搞了一波騷操作，對部分願意主動提供事故賬號ID的玩家，表示衷心感謝。意思是我擔心抓不到你們，而且抓你們比較麻煩，你們「自首」好不好？

可能嗎？如果想要讓玩家自首肯定也需要額外給這類玩家一些補償，不然別人為什麼要自己送上門？你犯的錯，誰願意替你買單？

5、兩大服獎勵不統一

這個原本無可厚非，有點差別正常，其中星火服包含了官方+IOS+B站。而信標服則包含了九游與其他手機商店玩家。

星火服比信標服多幾百黑卡和一些體力，此外有定製傢具，信標服沒有所謂的傢具。而且這次出事故的1萬黑卡，也大都是星火服的，這就不得不讓人懷疑遊戲差別對待。

菌說

從這次的事件，完全能夠看出這款遊戲應對突發事故有多麼「幼稚」，怕麻煩想省人省力，卻又沒法給玩家一個滿意的答覆，最終啥都不討好，反而該做的還得做，額外還要再貼更多的資源。甚至到了最後，還不忘搞一波無償「自首」。

最初如果咬咬牙，沒有領到的玩家們全部補償1萬黑卡，這事也就過去了，結果是還是不夠果敢。事已至此，人無完人，也沒必要再苛責，喜歡這款遊戲的玩家可以接著刷，而且好在這遊戲不主PVP，所以影響也在可接受的範圍內。

好了，這一回合到此結束，感謝閱讀，拜~
