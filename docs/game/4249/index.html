王者模擬戰：7刺客3星李白遺憾落敗，屈居第二，只因墨子沒上場？

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

哈嘍，大家好！我是九零小哥，私信「地獄火」，即可免費領取皮膚。在王者模擬戰中，刺客這樣的英雄是比較受玩家喜歡的，今天小編在遊戲的時候就測試了一下，全局使用七名刺客，到最後的時候，也只有一個刺客羈絆，但是卻也收穫了第二名的好成績，而與第一名也只是因為一個墨子而已，那麼這局遊戲到底怎麼回事，下面隨著小編了解一下吧！

在這局遊戲剛開始的時候，小編就已經做好了打算，測試一下七刺客的陣容，所以在遊戲開局的時候，就開始瘋狂的收集刺客英雄。

刺客英雄在前期可能是不如稷下搭配其他英雄，所以在前期時，掉的血量還是很多的，小夥伴們在使用刺客英雄的時候，最好搭配一下其他類的英雄。

在中期的時候，小編的戰鬥力就比較強了，再加上小編攢錢能力也還行，所以英雄紛紛升到了二星，甚至元歌這名刺客還升到了三星，所以也是罕有敗績。

小編其實在前期還收集到了一個天賦，那就是趙子龍和墨子同時在場的天賦，但是一直到了遊戲後期，才收集到墨子這個英雄，因為同場中，使用墨子的玩家太多了，甚至一名玩家還集齊了三星墨子，這就導致小編很難收集到墨子，所以這個天賦一直沒有用上。

到了後期的時候，小編的李白成了3星，此時戰鬥力可以說非常強了，也是經常憑藉李白一人翻身取勝，但是小編為了刷出李白浪費了不少的金幣，這就導致決勝局的時候，金幣不夠無法升級，導致墨子無法上場，最終輸掉了遊戲。

其實小編在這一句中是有勝算的，只要在最後一局時將墨子升到二星，上到場上，與廉頗組成稷下羈絆，同時還可與趙子龍相互搭配，利用天賦，就可以爆發出較大的威力，說不定就會贏得最終一局了。
