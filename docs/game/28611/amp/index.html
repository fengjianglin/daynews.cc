教練廠長

EDG在今年轉會後的德杯上表現不錯，當然也有人覺得是目前還沒有遇到強敵的原因。在他們的小組中當屬TOP最強勢，也是在今天最後一局比賽。廠長退役變教練之後，EDG在今天前幾局比賽中的BP沒有太大問題，作為多年職業老將，廠長對於版本的陣容解讀理解還是很到位的。

EDG對戰V5陣容

EDG新上單奧恩表現不佳

EDG的新上單比較適合打Carry型的英雄，偏團的工具人奧恩在他的手裡似乎熟練度不夠，至少比其他之前的鱷魚表現有很大落差。V5的陣容人均帶控，這個控制鏈在團戰中給EDG的陣容造成了很大的壓力。而且這一局EDG只有中單Scout表現出色，而V5的ADC顯然要比EDG的發揮更優秀。

連續的控制鏈讓V5在前中期取得了不錯的優勢

V5最終在30分鐘抓住EDG防守漏洞破上順勢擊殺一波！

這一戰結束雙方都是2-1，但是難度不同的是，V5之後要對戰的是LGD，而EDG則是要打TOP。從今天這個狀態來看TOP的實力是這個小組最強的，因此EDG有被淘汰的危險。不過也存在可能和昨天一樣3支2-2的情況，或許玩家們可以再見識一下那個150秒的神奇規則了！
