夢幻西遊：為什麼我生活技能里突然多了一個融會貫通，你們有沒有

2019-12-12 128條評論 3,186次閱讀 975人點贊

年輕人該多讀書、多旅行，放下手機，出去走走，接觸新鮮的事物，到最後你會發現……還是遊戲好玩。

嗨~大家好，歡迎來到小帝君說夢幻，我是本次的瞎編，小帝君。作為一個擁有十四年夢幻經歷的小白，小編也是真的不夠細膩，十月份大改以後居然沒有注意到夢幻西遊還增加了這樣的一個生活技能。

NO.1為什麼我生活技能里突然多了一個融會貫通，你們有沒有？

它就是融會貫通，學習效果是增加自身最高屬性點3點，雖然看似並不是很多，但是對於已經極限的封系來說，三點屬性也是至關重要的，也許就因為這三點屬性就可以拿到全場一速也說不定。

對於融會貫通這個技能來說，由於無法主動學習，所以必須精通所有的生活技能，也就是點滿18個技能，強壯神速不算，光其他16個按照300R一個也需要4800R多，還沒有算攢幫貢需要的時間，這裡在去掉常規的一些技能後，也就說為了這三點屬性的提升許多玩家至少要花費超過千元。

NO.2項鏈也能加體質，怪不得說是全服第一的項鏈。

大家都知道在夢幻西遊里有這樣的一位神豪，熱衷於收集遊戲里的一些遠古產物甚至可以說是BUG存在的道具，這條的項鏈就是他所收藏的，70級破血狂攻、珍寶、大唐官府專用初靈123，體質15點。那麼有許多朋友會問，體質15點是怎麼來的。小編找了許多以前的資料也沒有發現出處，現在只能有兩種猜測，一是初期專用溢出的屬性，二是為早期熔煉效果的BUG，大家知道最早期的熔煉能夠對未有屬性加成的裝衣服武器備附加上屬性。至於為什麼是項鏈就不清楚了。

NO.3我的夢幻都變成這個樣子，該怎麼玩下去。

今天有玩家發了一張自己的截圖，說死所有的旗子都變成了這種情況，甚至裝備欄也看不到自己攜帶的裝備，對於這種情況，小編提醒大家第一件事就是重啟遊戲，如果不行就點擊登錄界面的其他工具進行修復。

如果都嘗試以後，還是不行，那麼你只能選擇卸載以後重裝遊戲啦。
