遊戲版號12月再下一城，受益概念股有哪些？

天天要聞
2019-12-04

摘要

12月3日晚間，12月份首批國產網路遊戲版號核發名單于原國家新聞出版廣電總局網站發布，共46個遊戲獲批，包括2款端游，1款PS4遊戲，1款網頁遊戲，其餘均為移動遊戲，其中，申報類別為移動-休閑益智類的遊戲共有24個。

12月3日晚間，12月份首批國產網路遊戲版號核發名單于原國家新聞出版廣電總局網站發布，共46個遊戲獲批，包括2款端游，1款PS4遊戲，1款網頁遊戲，其餘均為移動遊戲，其中，申報類別為移動-休閑益智類的遊戲共有24個。

在本次名單中，其中騰訊的《家的故事》、盛趣遊戲的《龍之谷2》、巨人網路的《仙俠世界手游》、電魂網路的《萌喵消消樂》、東方明珠《救贖之路》、米哈游的《未定事件簿》等遊戲獲得版號，但列表中未見網易旗下遊戲身影。截至目前，今年已有1393款遊戲獲得國產遊戲版號。

國內遊戲市場概況

2018年中國遊戲市場中，實際銷售收入達2144.4億元，同比增長5.3%。分類別看，移動遊戲市場實際銷售收入1339.6億元，同比份額繼續增加，佔比為62.5%；客戶端遊戲市場實際銷售收入619.6億元，同比份額減少，佔比為28.9%；網頁遊戲市場實際銷售收入126.5億元，同比份額大幅減少，佔比為5.9%；家庭遊戲機遊戲市場實際銷售收入10.5億元，佔比為0.5%。從增速來看，移動遊戲增速最快，2018年同比增長15.4%，但增速有所下滑。

國金證券認為：中國遊戲開發商、發行商、渠道眾多；換皮遊戲、低質量抄襲遊戲大行其道；玩家出現審美疲勞，商業化面臨瓶頸。同時，集中了優質資源的頭部產品也出現了生命周期越來越短的現象。版號恢復不能改變遊戲產品供大於求、同質化率高、玩家審美疲勞的現狀，反而會讓問題加劇：堆積了大半年的數千款良莠不齊的遊戲，可能在短時間內衝進市場，進一步削弱玩家的耐心，出現「雅達利崩潰」現象。

經歷了2018年版號暫停發放帶來的市場低迷，目前國內遊戲市場正逐漸走出過渡期。在舊遊市場表現保持穩定的同時，新游中重量級產品頻出，細分類型、IP改編也迎來多樣化作品，遊戲市場將有望持續景氣，充滿活力。

相關受益公司

完美世界（002624.SZ）：完美世界實施大文化發展戰略，通過在影視、遊戲、動畫、文學、媒體等多個領域深入布局構建大文化業務矩陣，積累了對IP（知識產權）的開發與運營模式。

以「誅仙」IP為例，完美世界與不同明星進行合作，通過明星代言與遊戲主題曲合作等形式為IP粉絲提供更多的次生文化，對其進行沉澱，同時將IP影響力輻射到更廣、更年輕的群體之中。對IP進行的精細化運營也使得完美世界的多款IP雖然推出多年，但到目前為止仍然具備較高的商業價值。

同時，完美世界目前IP儲備豐富，擁有「射鵰英雄傳」「倚天屠龍記」「神鵰俠侶」等多個經典武俠IP，這些IP均具備了較高的影響力。

此外，完美世界控股集團的影視、文學等業務也將成為其未來遊戲IP的重要補充部分，對於挖掘和培育新的IP具備促進作用。IP儲備再結合其IP開發體系，完美世界有望在大文化布局中獲得更好的成果。

三七互娛（002555.SZ）：公司主要從事互動娛樂業務以及汽車塑料燃油箱製造、銷售業務，其中互動娛樂業務以手機遊戲和網頁遊戲的研發、發行和運營為主，同時布局影視、動漫、音樂、VR及直播等泛娛樂業務，核心團隊由經驗豐富的手機遊戲、網頁遊戲發行團隊，以及技術領先的研發團隊組成。2019年以來，公司持續輸出《一刀傳世》、《斗羅大陸》、《精靈盛典》等手游爆款，未來新品儲備（自研自發+代理髮行）包括《大天使之劍》《超能球球》《拳皇》《暗黑後裔》等，覆蓋MMO/卡牌/模擬經營/ARPG/SLG等多品類。

世紀華通（002602.SZ）：公司於2019年完成收購盛躍網路，成為A股市值及利潤最大的遊戲公司,公司以汽車零部件業務起家，陸續收購了七酷網路、天游軟體、點點互動等遊戲公司，打造了研運一體化的遊戲平台。

2019年，公司以298億元對價完成了對盛趣遊戲（原盛大遊戲）母公司盛躍網路的收購；盛趣遊戲是中國遊戲產業的重要開創者，先後推出和運營了《熱血傳奇》、《傳奇世界》、《泡泡堂》、《龍之谷》、《最終幻想14》等精品遊戲，註冊用戶超過21億。進入移動遊戲時代，盛趣遊戲先後成功發行代理遊戲《擴散性百萬亞瑟王》與推出自研遊戲《熱血傳奇手機版》、《傳奇世界手游》、《龍之谷手游》等現象級作品。

騰訊控股（0700.HK）: 騰訊是目前中國最大的互聯網綜合服務提供商之一，也是中國服務用戶最多的互聯網企業之一，其業務涵蓋社交、支付、娛樂、資訊、工具以及平台。娛樂業務包括遊戲、影視、動漫、電競、閱文集團以及QQ音樂。騰訊遊戲成立於2003年，是全球領先的遊戲開發和運營機構，也是國內最大的網路遊戲社區，通過在MOBA、FPS、RPG、ACT、體育競技、競速、棋牌等多個產品細分領域的耕耘，旗下囊括英雄聯盟、穿越火線、地下城與勇士、王者榮耀等多款精品遊戲，致力為玩家提供”值得信賴的”、”快樂的”和”專業的”互動娛樂體驗。在開放性的發展模式下，騰訊遊戲採取內部自主研發和外部多元化合作兩者結合的方式，已經在多個細分市場領域形成專業化布局，並取得良好的市場業績，為國內遊戲行業的領軍者。
