夢幻西遊：浩文回歸夢幻後再出神器，為玩家點出39神劍，往擺高價

2019-12-09 0條評論 16次閱讀 0人點贊

摘要

浩文回歸夢幻也有一段時間了，這個月的服戰還沒開始。在服戰結束之後，他就會開啟屬於他的39天猴組比武。雖然這個比武並不是所有伺服器聯合的，可是參賽的隊伍卻不少，不少玩家只需要投入幾千塊，就能體驗其中的樂趣。不過獎勵也是很少的，總冠軍的獎金也就是500塊錢，這差不多就是打臨時符和買葯的錢吧。像武神壇的冠軍是有很炫麗的稱謂和坐騎，自然浩文是無法提供給獲獎的玩家，因為這就是一個私人組織的小比賽。今天我們就來看看浩文，也是為玩家點出了一把神劍。

無底洞的好神奇，這把100級的斧頭，它絕對是特別適合無底洞的。武器的初始傷害不算高，因為鑲嵌了10級的太陽，之後的總傷害也就是405點。雙加為21點力量和21點敏捷，無底洞肯定是很需要這個敏捷，不單單是能加速度，還能有固傷上面的提升。符石組合是萬丈霞光，可以讓技能的加血恢復能力更加強。這把武器的亮眼點肯定就是晶清訣了，能解除所有的異常狀態，就算隊友都被封了，也能一下子就解掉。

最近，39級比武是真的有點火，這個帶熱的就是打造師。他們並不需要有高的技能，只需要把四打點到40級就足夠了。現在像生日快樂這種伺服器，在節日禮物使者處湧現了大量的商人，他們高價收購40級的裝備製造書，然後做成成品賣給39級比武玩家。不管是武器還是裝備，只要能出一個簡易，那麼就是很有價值和意義的。

我們熟知一般的主播資源，那都是鑒定160級的裝備，因為這樣才具有看點。就在最近，浩文也是接了一波資源，令人感覺到奇怪的是，這居然是一批40級的裝備。在經過一番鑒定之後，沒想到居然炸出了一把神劍。 其實現在BT版的玩著不錯，畢竟無級別和神獸系統都送，而且還有用不盡的夢幻幣，你可百度三刀俠客網即三大不留點xxyoy點康姆 獲取資源。

這把武器是40級的龍泉劍，命中162點，傷害也有146點。然後就是雙加有6點力量和5點的耐力。特效就很神奇了，一個神佑加上簡易。這把武器適合的門派就很多了，給輔助來用加了耐力和神佑。物理系用雖然不是滿傷害，可是又加了6點的力量。最後，浩文也是建議號主把這把武器擺高價，他覺得最低都能值500塊，運氣好賣700、800塊也沒問題。
