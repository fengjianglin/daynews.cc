不止是騰訊網易掙錢？國產遊戲出海瘋狂吸金，一年狂賺老外490億

2019-12-10 2,337條評論 4,110次閱讀 1614人點贊

不可否認，國產遊戲在近幾年來取得了長足的進步，無論是題材還是質量都能令玩家們眼界大開。不過，這些遊戲雖然能夠得到國內玩家們的支持，但距離「走出國門，面向世界」這一目標，還存在著不小的差距。

當然，凡事無絕對，在大部分國產遊戲開發者早們已將「自產自銷」作為營銷策略的當下，「SLG遊戲」這一門類卻在國外市場顯得十分耀眼。這類遊戲不僅可以躋身海外市場，而且還倍受老外玩家們的的青睞……

劃重點一：有趣耐玩消磨時間，有別於「短平快」遊戲的極佳選擇

SLG遊戲也就是所謂的策略遊戲，一般而言它又將被分為兩大類，一類是《星際爭霸》、《命令與征服》這類即時戰略遊戲；另一類則是《三國志》、《魔法門之英雄無敵》這種回合製作品，X博士今天要說的也正是這一類。

這一切的開始，其實還是源於2014年一款名為《列王的紛爭》的遊戲在海外市場上線。短短一年的時間，智明星通旗下這款名不見經傳的國戰手游就一連佔領了七十多個國家的手游收入榜首，時至今日都還有每月過億的流水。

《列王的紛爭》的全球爆火，讓國內不少開發者都感到震驚。因為一款優秀的SLG遊戲就必然會擁有探索、擴張、開發、消滅等玩法，但伴隨著這些內容層層遞進的，同樣也還有當前快節奏的生活節奏下玩家們那極為寶貴的時間。

「慢熱」，便是形成SLG遊戲獨有魅力的一部分，它的遊戲方式和《王者榮耀》、《刺激戰場》這類「短平快」遊戲大相徑庭。雖然目前的國內玩家更傾向於後者，但《列王的紛爭》的成功卻給國內遊戲開發商們指明了一條新的道路。

2018年七月，《火器文明》擊敗《堡壘之夜》佔領德國暢銷榜頭名；同年九月，友塔網路的《黑道風雲》達到了日本暢銷榜14名；而IGG公司的《 王國紀元》也是宣布了全球收入超過了56億。

就近期來看，莉莉絲遊戲旗下的《萬國覺醒》也是自六月起便挺進了海外手游收入榜的前三，僅次於目前大熱的《刺激戰場》以及《荒野行動》這兩款「吃雞」遊戲。

以上這些，無一例外的都是國產SLG作品，不過令人玩味的是這些遊戲的主要消費者都不是國內玩家。像《黑道風雲》就根本不在國內上架，一來是因為題材涉及暴力犯罪的緣故，二來是SLG作品在國內巨大的遊戲市場中，只能算作是小眾遊戲。

雖然這類遊戲在國內並不吃香，但SLG遊戲那種「養兵千日，用兵一時」的樂趣則深受老外青睞。通過一組數據X博士發現，在2018年中國移動遊戲海外出口收入已經達到了69.2億美元（約490億人民幣）。其中竟然有將近一半的遊戲都是SLG題材！

那麼，SLG遊戲除了運營和戰鬥的快樂外，究竟還有什麼魅力能讓老外對其如此青睞呢？

劃重點二：社交互動，合縱連橫，感受SLG遊戲獨有的參與感

這個問題的答案，其實就在於「社交」上。SLG類型的遊戲對於社交的倚重是遠高於其他遊戲的。舉例來講，在MOBA遊戲里有不少僅靠單排也能上分的高手，但在SLG遊戲選擇作為「孤狼」便等同於成為最好捏的「軟柿子」。

因此，幾乎所有此類遊戲玩家都會選擇一個適合自己的派系或團體加入其中，而這種玩家之間的小圈子往往是由國籍和地區來劃分的。這樣一來，要想戰勝敵人更為倚重的就是人與人之間的交流，其次才是「技術」或「氪金」。

（《率土之濱》中的戰況不可謂不「亂」）

敏銳的廠商，當然也知道這一點。以之前提到的《萬國覺醒》為例，該遊戲便是將遊戲中的陣營劃分為現實中的各國國家，讓世界各地的玩家都能獲得文化認同與參與感。

所以，玩家能不能在SLG遊戲里有立足之地，完全不是一兩個人能夠決定的，這也導致了此類遊戲普遍「戰略大於戰術」的特殊情況。相比起靠操作、裝備壓制對手，那種合縱連橫戰勝敵方軍團的快感才是玩家們喜愛的關鍵。

除此之外，SLG遊戲還擁有經營統籌獲得累計收益的參與感，而這便是一個漫長卻不枯燥的過程。從這點來看，不僅很好的提升了遊戲的用戶粘性，更是其他類型遊戲很難企及的一點。

課後總結：

正是因為SLG遊戲擁有這些得天獨厚的優勢，所以才能在樂於交流的國外玩家市場如此暢銷。但這同樣也說明了一個問題，很多國內出海廠商還沒有研究明白國外的其他類型遊戲是怎麼成功的，所以只能借鑒此前出海成功案例扎堆去做SLG。
