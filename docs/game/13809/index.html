王者榮耀：玄雍版本懸念站發布，蒙犽原畫和玩法細節一覽

2019-12-13 2,587條評論 3,741次閱讀 302人點贊

王者榮耀全新玄雍版本即將在下月上線，為了讓玩家提前掌握版本信息，官方開啟了「嬴政夢魘，危機預兆」的主題懸念站，在嬴政的夢境中玄雍正經歷一場殘酷的戰鬥，通過下圖標記處可以看到，背景海報中除了嬴政外，還有另外一個人的影子，可惜參與者都有誰目前還留有懸念，有待後續揭曉！

值得一提的是這次玄雍版本更新後，嬴政不僅模型翻新升級，加入了凸顯君主氣質的紅黑金三色，其2技能也正式改版為可以清空敵方裝備AD加成的效果，感興趣的玩家可以期待一下新玩法！玄雍版走向暫且不論，我們再來看看新英雄蒙犽的消息。

關於蒙犽技能機制小編已經在上文有過專題介紹，此次對部分細節進行補充說明，首先蒙犽原畫展示了一位身著紅黑色服飾的少年（配色符合玄雍系）手持「渾天」作戰的場景，黑白雙色的頭髮是一大亮點，在背景設定中蒙犽為了向父親蒙恬證明自己，只身前往稷下求學，在成長的同時，這位熱血叛逆的少年還結交了一群好朋友。

玩法方面，大家都知道蒙犽的能量條達到最大值後會進入高熱狀態，這個狀態能讓其技能強化，比如1技能可穿透、2技能分裂流星彈數量加倍以及大招導彈的引導最小間隔時間減半，因此未來蒙犽的定位與當前絕大多數射手的定位不同，即蒙犽主要依賴技能輸出，而不是靠普攻打傷害的射手，另外還有以下三個細節需要注意：

  1. 蒙犽被動效果為高熱狀態下1/2技能CD縮減速度加快，同時攻速與CD成正比關係，即攻速越高，CD刷新越快！（主打攻速裝）
  2. 1技能掃射期間是可以移動和釋放其它技能的。
  3. 大招飛彈支援僅持續8秒，通過搜索交戰的隊友提供火力支援。

蒙犽身為炮台射手優勢明顯，即火力和支援能力強，而短板同樣明顯，沒有位移技能是硬傷，容易被敵方刺客或戰士盯上，那麼小夥伴們你覺得新英雄如何呢？歡迎留言討論！
