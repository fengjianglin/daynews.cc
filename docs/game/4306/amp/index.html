摘要

這幾天老哥們應該都在肝《怪物獵人：世界》冰原DLC的第二波大更新，這次需要面對的是“獄狼龍”以及新古龍“冥赤龍”，同樣新BOSS也帶來了新的衍生裝備，那麼這個獄狼龍的武器以及防具究竟怎麼樣呢？今天我們就來評測一波。

這幾天老哥們應該都在肝《怪物獵人：世界》冰原DLC的第二波大更新，這次需要面對的是「獄狼龍」以及新古龍「冥赤龍」，同樣新BOSS也帶來了新的衍生裝備，那麼這個獄狼龍的武器以及防具究竟怎麼樣呢？今天我們就來評測一波。

獄狼龍武器：可能是新的畢業武器

首先這次獄狼龍的武器算是一條新的派生，你可以直接製作R11的獄狼龍武器，然後如果要升級到R12版本就需要另外歷戰獄狼龍的聚魔素材以及獄狼龍的天玉。這裡要劃重點：歷戰獄狼龍只會在Lv7的聚魔凍土地帶出現，所以製作獄狼龍武器就要自己升或者找朋友升一個Lv7凍土。

獄狼龍武器的特徵是攻擊力比標準要稍微低一些，0會心但有不錯的龍屬性，同時還有長紫斬，更有裝飾品洞，整體性能算是比較給力的，但是強歸強素材卻是十分難搞。另外還有一個不能忽略的優勢就是很有牌面的「呼吸燈」特效，獄狼龍武器自帶閃爍類似於RGB呼吸燈效果，而這個「呼吸燈」也有十分硬霸的效果，就是使得武器的體感威力上升30%。

所以根據以上總結的武器特性，所以主打屬性攻擊的武器如雙劍、片手劍、弓的表現會很不錯，值得一提的是鎚子有標準攻擊力（1404）的同時還有紫斬和不錯的龍屬性，對於弱龍的怪物應該比滅盡龍錘要強一些。

另外就是獄狼龍銃槍第一次出現了通常Lv7的炮擊，搭配上炮術5以及增彈之後，普通炮擊1發51，全彈射擊一發62+50。

輕弩除了攻擊力不低之外，可速射散2和電擊彈，而且電擊彈雖然裝填數只有2，但是不需要裝減少后座力以減輕配件，搭配上迴避裝填也許是另外一套玩法，而在重弩方面彈種和雷狼龍一樣，攻擊力高了15，但變成了狙擊龍彈，看你喜歡機關龍彈還是想要多15面板的攻擊力。

防具：散件實用性高，整體性能出眾

防具方面整體的性能十分優秀，不論是技能總等級還是洞數都很不錯，主要技能是力量解放，然後每個部件都會有一些散技，像是風壓耐性、龍屬性攻擊強化、耐震、減輕膽怯、集中、破壞王等，都算是比較常用的技能。

套裝技能雷狼龍一樣是力量解放上限突破，如果穿上全套再搭配一些適當的裝飾品，要出耳塞、耐震和風壓等防禦型的技能並不困難，基本可以當做生存和抗壓性超強的套裝，輸出方面可以靠力量解放來補。

但是你不能忽略的一個問題就是力量解放真的不好用，因為100%會心基本很容易都完成，如果100%會心再搭力量解放這等於是溢出了，但是遷就力量解放而不要弱特/看破又會覺得好像虧了50%會心一樣，同時力量解放的另一個效果「減輕固定消耗耐力行動的消耗量」也不好用，基本上只有弓和蟲棍會用到，然而主打屬性的弓必然是穿銀火龍，又或者是我自己不會用的關係？

至於外觀方面是以黑白色為主色調，整體設計有點麒麟和慘爪龍混搭而成的感覺，染色範圍還是比較少，但值得一提的整個頭髮是都可以染色的，如果想玩出點什麼「奇怪的顏色」都是可以的。
