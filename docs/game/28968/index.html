如果將武則天改為直售英雄，補償玩家榮耀水晶，你能接受嗎？

2019-12-20 1,530條評論 3,614次閱讀 1353人點贊

哈嘍，大家好，我是燦燦醬，喜歡記得關注哦~

武則天在王者榮耀中一直是個熱議不休的話題性英雄，其主要原因就是因為貴！當之無愧是王者榮耀中最貴的英雄了！

11月26日，王者榮耀對武則天進行了一波調整，使其脫戰後的被動由原本回藍1%變成了2%，而這次更新的原因是因為，武則天作為一個高級人民幣英雄，但是已經大幅度脫離當前版本，從萬金油行列跌成了T3行列。

如果追溯上一次調整時間，則是在遙遠的2018年5月末，當時是對武則天的技能傷害進行了削弱。

但無論如何調整，想讓武則天變成一個人人都可以玩的英雄，就目前情況來說是不可能的，畢竟價格擺在那裡。

不過說起來，武則天出現在水晶商店中應該算是歷史遺留問題，早期的王者榮耀與現在無論是遊戲世界的平衡、玩家體驗、匹配機制等等都是從零開始摸索的。

而武則天成為水晶商店兌換英雄自然也是受到不少玩家們的怨念與吐槽，於是王者榮耀官方便也明確表示了除武則天外，榮耀水晶商店不會再新添英雄。

因為在王者榮耀中，大多數英雄都是可以使用金幣或者鑽石購買的，除武則天外，只有嬴政和早期首充才有的趙雲需要真金白銀購買，但無論是6塊錢首充送的趙雲還是68元點券購買的嬴政，與武則天的價格對比都只是個零頭。

因此，其過高的價格，讓很多玩家都理所當然的認為就應該強過於其他英雄，不然怎麼會物有所值？

所以，武則天1、2技能cd短，大招有全圖視野+控制。但正因如此，武則天的存在就影響到了遊戲平衡的問題。

如果想要武則天配合遊戲內機制隨時調整，並且不讓玩家有所怨言，最直接的辦法就是將武則天從水晶商店中解放出來，但這顯然對曾經花錢購買的玩家們很不公平，所以，如果將武則天變成直售英雄，然後王者榮耀對之前擁有的玩家們補償一顆榮耀水晶，你能接受嗎？
