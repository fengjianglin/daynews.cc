IG新下路帶隊翻盤，體會到阿水感受，S9首發打野卻被青訓隊員吊打

2019-12-22 3,242條評論 4,907次閱讀 2821人點贊

今天12月22號英雄聯盟德杯，由IG戰隊對抗SDX戰隊。IG戰隊在轉會期損失了下路Jackeylove，但也找來了VG雙人組，他倆都是韓服千分王者，現在終於有了證明自己的機會了，而這次德杯中上位置由二隊選手頂替。打野小樂言自從被調上一隊以後，好像就沒正常過幾次，每盤比賽都找不到節奏，讓網友大失所望。不過這一場的對手SDX是小樂言最熟悉的LDL隊伍，SDX打野YourRiver更是連LDL都沒打過的青訓隊員， 小樂言還能找回當初在次級聯賽亂殺的手感嗎？讓我們看看雙方陣容！

IG戰隊選用了奇亞娜、賽娜、塔姆、蘭博、青鋼影；而SDX戰隊選用了霞、泰坦、劍魔、豹女、加里奧。

開局雙方互換野區，而IG塔姆想要干擾豹女刷F6。SDX豹女被打成殘血，閃現過牆，塔姆跟閃，但他沒帶點燃，被豹女殘血逃了。隨後SDX泰坦趕來支援，幫助豹女反殺塔姆。此時SDX加里奧交出TP，四人聯手越塔強殺IG賽娜。

4分鐘SDX豹女再來下路，配合泰坦擊殺塔姆，再找機會拿下風龍，IG下路已經崩盤。8分鐘IG奇亞娜拿下峽谷先鋒，而SDX戰隊趕來留人，擊殺了IG青鋼影。

11分鐘SDX戰隊拿下火龍，再推掉下路一血塔和二塔。而IG雙人組只能換到上路，利用峽谷先鋒換掉上路一塔。

13分鐘SDX戰隊想推上路一塔，IG戰隊想來包夾。此時SDX加里奧TP支援，五人提前動手，秒掉塔姆，但劍魔衝動越塔送了。最終小樂言留下泰坦，但他衝動一打四，又把人頭送了回去，雙方打了一波2換2的團戰。

15分鐘IG五人集齊下路，奇亞娜蘭博交出雙大招，強行秒掉豹女劍魔，拿下土龍，現場女解說來了一句：「小樂言終於開竅了！」

18分鐘IG青鋼影想與劍魔互拼，SDX戰隊交出TP支援，小樂言也想來反打。結果SDX戰隊兩人同時TP，落地後形成五人包夾，反殺IG上野，再釋放第二個峽谷先鋒，推平下路高地！

19分鐘SDX戰隊推掉中路一塔，小樂言入場大招逼出霞的閃現，導致他失去戰鬥力。隨後IG蘭博賽娜收割戰場，打出了一波1換3的團戰，挽回了不少經濟。

22分鐘SDX戰隊試圖逼大龍，但被奇亞娜賽娜雙大招勸退。隨後IG戰隊沒有回頭打土龍，而是在中路埋伏。結果SDX戰隊趕來接團，IG賽娜被泰坦鉤到，直接被秒。隨後IG青鋼影閃現大招想換霞，卻被輕鬆反殺，丟了第三條小龍。

27分鐘雙方在中路動手，IG小樂言被嚇出空大，又被泰坦鉤住。好在IG塔姆閃現吞救下小樂言，反殺泰坦。此時SDX劍魔TP落單，只能閃現逃跑，而IG上野對他鍥而不捨，從自家中路一塔追到對方下路二塔。

結果IG青鋼影剛ER近身，SDG加里奧就大招支援，將青鋼影反殺。同時IG小樂言在野區犯傻，在殘血沒大招的情況下，E技能跳豹女的臉，然後交出死亡閃現。IG上野再一次白給，SDX戰隊直接拿下大龍以及遠古龍魂，取得了土龍的護盾效果！

30分鐘SDX戰隊帶著大龍BUFF分推，IG蘭博先在中路高地前丟出大招， 逼出霞的飛天。同時IG上野想對劍魔動手，但小樂言被加里奧大招反殺了。好在IG賽娜立功了，找到機會擊殺了霞，配合蘭博收割戰場，打出了一波1換4的團戰！

32分鐘IG青鋼影越塔強殺劍魔，推掉下路二塔。35分鐘SDX戰隊在中路出手，卻打了一波3換1的團戰，丟了遠古巨龍。隨後SDX戰隊強行拿大龍，IG戰隊趕來包夾，小樂言獻祭自己丟大招，順利團滅了SDX戰隊，再從下路一波翻盤，贏下比賽！

這場比賽IG新下路Puff賽娜立功了，他在天崩開局的情況下穩住局勢，絕境中謀發育，多波團戰收割戰場，打出了全場最高傷害，再加上Q治療以及大招護盾，他為隊伍提供了非常強悍的戰鬥力！看來Puff的能力還是可以的，明年IG戰隊有望重現三叉戟。這場比賽Puff體會到了阿水的感受，輔助開局白給，前期抗壓沒人保，後期扭轉局勢帶隊翻盤！

但打野小樂言再一次讓粉絲失望了，他曾在LDL次級聯賽中亂殺一通，是S9世界賽的首發打野，許多人都稱他未來可期，他卻一次次地打了粉絲的臉。之前可以說小樂言跟明星選手待一隊，一出場就是冒泡賽世界賽，心理壓力比較大，所以沒有發揮好。但今天小樂言打的是德杯，隊友同是二隊選手，對手是連LDL都沒打過的青訓隊員，小樂言卻全程刷野，帶崩上路，控不住小龍，各種白給，完全就是靠著隊友躺贏！

相比起RNG小龍堡，小樂言的表現真的差太多。昨天小龍堡一上來就當院長，打法思路異常清晰，帶動三路節奏，也敢嘗試搶龍，連LPL一線打野Xx都不如他。這下觀眾不禁產生了疑問：「小樂言真的在次級聯賽吊打了小龍堡嗎？」

接下來IG戰隊還有三場比賽，這些對手可都是LPL隊伍了，他們還能頂得住嗎？讓我們期待一下比賽的發展！
