《鬼泣：巔峰之戰》測評：動作手游的又一次創新，拭目以待

2019-12-23 199條評論 2,817次閱讀 1632人點贊

文丨CrackersAria@bigfun社區

大家好，這裡是ARIA。提起動作遊戲（Action Game即ACT），大家心中能想到的大作名單應該還是傳統的ACT巨頭——《鬼泣》、《戰神》、《獵天使魔女》以及《忍者龍劍傳》。這其中不論哪一款遊戲，都會通過華麗的連招和爽快的打擊來帶給玩家一場視覺上的盛宴。雖然時至今日，《忍龍》和《獵天使魔女》已經沉寂多年，《戰神》轉行搞起了「爸爸去哪兒」，只有《鬼泣》系列還依舊活躍在玩家的視線中，年初發行的《鬼泣5》更是在今年的TGA上榮獲了最佳動作遊戲的稱號，確實是ACT愛好者的一大幸事。而今天，A醬就為大家帶來由CAPCOM正版授權，剛剛開啟首測的動作手游《鬼泣：巔峰之戰》的試玩評價。究竟如何，請大家聽A醬慢慢道來。

*註：本測評一切數據基於《鬼泣：巔峰之戰》首測刪檔內測，後續改動請以遊戲實際為準。

畫面以及音樂

角色

首先來一張但丁近照，不得不說手游的「鬼泣」在人物還原上還是做得比較不錯的。並且手游中的時間線定位在3代鬼泣期間，所以但丁的造型還非常年輕並非現在大家經常看到的大叔形象。初始武器也是3代經典的「叛逆之刃」和「黑檀木與白象牙」。蕾蒂也是完全還原了3代造型，可惜目前還沒有解鎖，只能在劇情中看她和但丁互動了。

界面

《巔峰之戰》的UI整體設計的非常簡潔，平時菜單是收起狀態僅僅能看到方向搖桿鍵，因此玩家可以很直觀的欣賞周圍的場景。菜單打開後所有功能都集中在了一個頁面，內容一目了然，點擊功能按鈕切換的詳細界面則是進行了單獨的精心製作，比如事務所界面的辦公桌風格，劇情回顧功能的筆記本加照片風格。不論我們需要做什麼都能不費力的找到相應的功能，相對於那些界面上全都是按鈕並且按鈕套按鈕的手游，UI簡潔算一個優點。

場景

至於遊戲場景例如酒吧、事務所也是非常的還原，甚至事務所中還有一台能夠使用的點唱機。室外場景例如街道和雪山之類也能看出是用心製作，而且場景中具有可破壞元素。有些位置擺放的物品像是雕像、座椅以及障礙物等是可以被破壞的（傳統藝能），裡面有紅魂或綠石可以收集。遊戲第一章還亂入了「鬼泣4」的場景，是不是很熟悉呢~

背景音樂

遊戲中BGM則是另一個亮點。無論是休息途中的酒吧內的溫柔的爵士樂，還是戰鬥過程中讓人心情激揚的搖滾樂，都非常的「鬼泣」。而劇情探索途中的背景聲音又恰到好處的安靜下來，給人空間去思考謎題或品味劇情。

從以上幾點上看來，《巔峰之戰》無論是視覺還是聽覺上，玩家所能感受到的一切要素都是用心設計過的，完全能看出製作組的誠意。有那味兒了，點個贊。另外事務所內的點唱機是可以使用的哦！這個彩蛋著實夠大，小夥伴沒事的時候可以去點點看。

戰鬥系統以及養成系統內容

戰鬥系統

既然是「鬼泣」系列，ACT自然是核心。在玩家初始進入遊戲時就提供了「新手」和「達人」兩種選項，使得初入動作類的萌新和動作大神都有適合自己的路線選擇。

在操作方面設置的也並不複雜，角色動作除了方向和基礎的攻擊跳躍之外還包括了上挑和前沖，並且設置了搓招系統以讓按鍵互相搭配來實現不同的招式表現。戰鬥動作方面也很華麗，角色招式都和本體基本相同，有多種武器供玩家搭配使用。可是也不要因為這種簡單的基礎設置就覺得《巔峰之戰》是一款簡單的動作手遊了，起碼A醬這個水平在遊戲內基本屬於被暴打的狀態（畢竟動作手殘）。而打出宣傳片和演示視頻中那種無限連的效果就是大神的任務了，A醬這裡先膜拜一下。

遊戲的具體戰鬥內容分為主線模式和回顧模式兩種。主線模式這一塊被處理成了類似「鬼泣」本體一樣的具有解密風格的連貫劇情，可以在這遊玩新的故事內容。而回顧模式在通關劇情模式後會解鎖相應章節，這裡是玩家獲得各種材料的主要途徑，並且每個章節能獲取的材料也不盡相同。關卡還設置有評分系統，不同的星級評價會給與不同的獎勵。

養成系統

《巔峰之戰》的角色養成分為角色和武器兩部分。畢竟「鬼泣」中的角色就那麼幾位，所以本款手游中是沒有抽角色這個設置的，初始就可以使用但丁。因此養成角色就成了遊戲的主要目標之一。首先是角色本身的數值，可以在惡魔獵人界面選擇角色強化，之後用劇情中或者劇情回顧模式獲得的紅魂來強化基礎屬性。注意這個點數是可以利用「遺忘之石」來重置的，並且重置會返還所有消耗的紅魂。所以萬一加錯了點數也不必驚慌，還是可以搶救一下的。

除了基礎屬性，還有一個需要強化的項目——戰鬥技巧。戰鬥技巧相當於角色自身獨有的天賦樹，包含了新的招式動作和一些額外屬性以及特性，需要消耗指定材料才能點出，這裡應該就是《巔峰之戰》給大家的一個肝點了，戰鬥技巧匯流排路並不短且有各種分支，目前但丁的第二套武器方案就是需要解鎖戰鬥技巧的最後一個節點華麗流才能使用。只是戰鬥技巧並不像角色屬性一樣可以重置，雖然最終都是需要點滿的，但是根據玩家自己的操作習慣總有先後順序，所以點之前還是稍微研究一下設計出一條比較適合自己的路線為好。

最後說一下武器養成，首先武器需要靠抽取獲得。目前已經開放了四把傳說武器、七把史詩武器以及十七把稀有武器。在首測資源管夠的前提下A醬嘗試了一下抽選，不得不說非洲血統還是蠻真實的哈哈哈，首次累計三十次抽取會有保底，A醬也是在第三次十連的時候才抽到了傳說。和前面的養成方式類似，武器的養成也是是在武器頁面使用指定材料進行進階從而提升屬性和附加效果。越稀有的武器需要的材料也越難獲取，自然回報也是倍數增長。總之就是，武器養成系統放在大範圍中和其他手游一同比較也算是中規中矩的一個設計，在預料之內。

關於A醬自己的一些看法

以上就是關於手游《鬼泣：巔峰之戰》的內容介紹，接下來A醬說說自己的感受。總體來說，《巔峰之戰》應該還是屬於一款有潛力的作品的。手游的外在部分包括視覺聽覺方面已經做到了一定水準，但是也有稍顯青澀的地方，例如在劇情過場中兩位角色對話時面部表情比較僵硬且對話文本也沒有標註出發言的角色，所以他們在說話的時候A醬只能靠聽的來分辨現在是誰在說話。其次場景中的風景和建築是做到位了，但是有些位置的空氣牆就有些莫名，比如看起來能夠走過而事實上不能靠近的小巷，以及不能直接走下必須跳起來前衝下去的台階（沒錯就是台階），大概是害怕玩家在連招過程中失足掉下去吧。

戰鬥過程中的問題說實話還是有一些的，首先是無法取消招式動作，也就是說沒有動作遊戲玩家都知道的「柔化」系統。出招之後必須等動作做完才能銜接下一個招式，這就意味著在遊戲過程中不能急於瘋狂搓招，而是要按照順序有節奏的逐步打出自己的招式，非常考驗玩家對出招表的記憶力。這裡A醬猜測製作組大概是想做出類似《只狼》那樣的節奏打鐵的樣子？總之A醬是很不適應，也許是A醬自己玩法有誤吧。另外一個稍微有些嚴重的問題就是鎖定，鎖定表現很不靈活，經常出現本來要閃避攻擊結果但丁直接閃到怪臉上的情況，還是比較難受的。這裡也希望在後續測試中可以對戰鬥不靈活的方面進行優化，以便讓更多玩家能夠打出各種招式的華麗效果。

更多的問題嘛，不是沒有，但是A醬覺得《巔峰之戰》尚在首測階段，猶如一個剛剛登場的新秀。儘管這場表演不太盡人意，但是也還有繼續打磨和上升的空間，所以機會還是要給的。A醬也相信這次為期三天的測試能夠給製作組反饋足夠有用的信息，讓製作組清晰的掌握到玩家的需求，從而能夠在下次測試給廣大玩家一個滿意的答覆。

以上，感謝收看，鞠躬~
