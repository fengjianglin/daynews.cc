DNF：100版本搬磚圖！135級的格藍迪，普雷C才到門檻

2019-12-19 1,528條評論 4,882次閱讀 213人點贊

地下城的100級版本內容逐漸曝光，不管是魔戰套的「超進化」，又或者是100級武器的屬性，95版本徹底被淘汰，就連盧克這個搬磚團本都刪了，普通玩家需要轉移副本了，比如100級專屬搬磚圖，或者漩渦摸金，當然價值最高但就是改造5的「智慧史詩」，初期估計要幾個E！但這幾率太低了，還是來看一看100級的普通搬磚副本吧，格藍迪王者歸來喲！

100版本隨機地圖！135級的格藍迪，普雷C才到搬磚門檻

但體驗服還未上線所有難度的副本，100級地圖只有簡單、專家兩個難度，深淵似乎是1個史詩靈魂刷一次，歌蘭蒂斯失業了，變成了一個聖職者NPC來換深淵門票，但深淵的血量快有1W條了，而12.18隻是部分副本更新，12.24才會上線更高難度的副本，不過玩家們期待的遠古BOSS、格藍迪、異界BOSS全都回來了，裝備百花齊放，BOSS也是攢了幾年一起來！

比如135級的格藍迪！中間會遇到魂遺迹的波羅丁等遠古圖的BOSS，或者是遇到精英怪，其實就是隨機圖，中間的每個房間都會遇到一道門，不同的BOSS隨機出現（GZS被史詩級削弱），機制也很複雜，有反傷，又或者物攻才能造成傷害的BOSS，反正這就是100級大家面臨的搬磚圖難度。

還有更難的爬塔（紅玉的詛咒），一陣雨的普雷C都刷得很累，一共46層，刷到40層就非常困難了，BOSS都是150級左右，比135級的格藍迪就是天和地的差別。

而目前國服的格藍迪不過84級，血量200條左右，大家100級就會刷135級的格藍迪拉，無論是血量還是防禦都蹭蹭漲，就算你是普雷C，也不過剛好到達門檻的水平，因為是隨機BOSS，遇到機制複雜一些的，堪比Raid開荒，更何況，12.24還會開放更高難度的，如果血量繼續增加，5000條血的搬磚副本？玩家的95B需要升級成100級裝備才能搬得動咯。

​由於是體驗服，各種金幣、材料的收益還不能評估，就連深淵爆率也不能作為參考，不過搬磚的門檻的確是普雷C才行了，但小夥伴們別激動，普雷C在100級就等於傳說C，所以換了裝備後，5000條血量的搬磚圖也能接受，至於紅玉的詛咒，旭旭寶寶到46層都很難……
