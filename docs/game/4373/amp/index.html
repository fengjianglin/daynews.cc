8l其實並不叫8l,他的名字是由7個l和一個1組成，而且他還有句名言「不要拿你的年薪挑戰我的零花錢」。

想當年我們玩傳奇差1級到40級都要花一個月時間，而8l卻以每天15級的速度在升級，這個速度，算了一下，大概要花500萬，加上裝備的錢最少要花2000萬。這個數字在當時是有多麼的恐怖

而8l最出名的就是一人攻下了沙巴克在攻沙的那天，當時皇宮已經被一個行會佔領，而行會的大部分成員都在野外打野，只留了幾個人在皇宮留守，當他們還在聊天的時候，只看見一個黃名進來，一個照面就秒了他們，會長問他們幾個人，他們說一個人，然後行會所以成員直接趕到了沙巴克，進去後發現這個人就是8l，然後行會所有人直接集火他，但是發現這個人幾十萬的血，兩個復活戒指輪著用，這還怎麼玩啊，後來8l一個人直接打下了沙巴克。

在09年左右，8l在10基本就是無敵的狀態，於是就在伊人風采所在的12區練了幾個號，準備找伊人風采pk，8l最開始練的是法法，但到200多級後又換成戰戰號，但是伊人風采並不感冒跟8l單挑，一直在避戰，二人之間的單挑可以避免，但是行會戰還是對上了好幾次，雖然8l是後來才來的12區，但是砸下來的錢還是超過到了伊人風采，所以行會戰中，伊人風采的道戰組合還是不敵8l的戰戰。兩分鐘左右，伊人風采就只能選擇飛了，後來伊人風采乾脆不上線，直到8l離開21區才繼續上線，到10年的時候，跨區行會戰倆人還是對上了，反正不管怎麼打都是8l略佔上風。

為什麼說8l改變了傳奇的發展軌跡，因為他，盛大數次修改遊戲數據，數次專門為他打補丁，在當時10區，由於8l的氪金開禮包，把整個10區的物價拉的很低，其他老闆直接無法承受，要轉服。當時有許多人說8l其實是拖，但當時的年代不會有拖的存在，也不流行拖。8l大多是隨性所為，打的當時小有名氣的伊人風采直接退服，碾壓了一些小老闆。
