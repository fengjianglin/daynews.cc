EDG官宣新陣容！網友：別的戰隊轉會期花錢，EDG居然還盈利了

2019-12-17 1,212條評論 2,847次閱讀 1648人點贊

昨天是今年LPL冬季轉會期的最後一天，不少戰隊在昨天都官宣了不少轉會相關的消息。然而在昨天之前，相信大家都聽說過，今年冬季轉會御三家（WE、RNG和EDG）的情況不妙，買不到人。然後就在昨天，EDG官宣了廠長轉型成為教練的消息，並且之後將擔任EDG英雄聯盟分部主教練一職。

同時還官宣了IBOY轉會到了VG戰隊。

然後昨天晚上就沒再宣布任何消息了，粉絲們本來還一直等著EDG官宣新隊員，不過就在今天中午，EDG終於官宣了新名單，同時也官宣了之後德瑪西亞杯的出征陣容。

可以看到EDG2019德瑪西亞杯出戰陣容的主教練是由廠長擔任，其中上單有：Jinoo、Aodi和Xiaoxiang，打野有Jiejie、Xinyi和JunJia，中單Scout，ADC為Hope以及輔助Meiko。除了上野有替補以外，中下都沒有替補。

除此之外EDG還順便宣了轉會消息：

1、原VG上單選手Aodi以轉會形式回歸EDG電子競技俱樂部英雄聯盟分部；

2、2019英雄聯盟職業青訓營中脫穎而出的黃祥（xiaoxiang），將以上單選手的身份加入德瑪西亞杯出征陣容之中；

3、前FPX打野選手Xinyi以自由人身份正式加盟EDG；

最後，EDG表示，在這次冬季轉會期期間，他們針對選手引援投注了大量精力，但結果不盡如人意。所以他們決定著重在教務體系以及青訓體系上進一步加大投入，後面會推出更多舉措來保證隊伍成績。在看到這個新名單後，不少網友表示：「就這？」，有很多網友認為EDG的這個新名單不但沒有補強，反而削弱了，將IBOY賣給了VG賣了不少錢，買的選手身價都不如IBOY，所以對此有網友調侃說：「轉會前預算有幾千萬，現在轉完會，賬上有一個億了」，「別人家主隊轉會期瘋狂砸錢，我的主隊轉會期結束後，居然還盈利了！」大家對此怎麼看呢？
