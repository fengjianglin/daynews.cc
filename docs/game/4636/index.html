要問誰是王者榮耀拿「三殺」最多的英雄？相信很多網友都猜不到。據官方統計全服「三殺」排行榜，阿軻僅第三，第一拿「三殺」如同喝水，次數高達1億2694萬！

第四位：韓信

韓信拿「三殺」的次數為2798萬次，排名第四位。韓信雖然非常靈活，號稱是「韓跳跳」，但是他只適合單抓，前期抓誰誰就要崩。不過開團的能力不行，而且還是那種後期開團就「廢」的刺客。

第三位：阿軻

阿軻拿「三殺」的次數為4960萬，排行第三。阿軻只所以會超過韓信，主要的原因是她的大招，如果團戰中能收割到殘血，那麼阿軻瞬間就起飛了。擊殺後會立馬刷新所有技能，別說拿三殺，就算是五殺也是非常的輕鬆。

第二位：諸葛亮

諸葛亮「三殺」次數是7694萬，排行第二。諸葛亮號稱是峽谷「殘血收割機」，因為他的大招一旦鎖定殘血，如果沒有隊友擋，就算你回到全血都要被秒殺。而且擊殺後還會縮短大招CD，最多5秒一個。

第一位：小喬

相信很多網友都沒想到她會是拿「三殺」最多的英雄，「三殺」次數破了億，12694萬。

小喬團戰能力極強，不僅1技能的傷害非常高，大招更是無解。只要2技能將敵人擊飛，開啟大招閃現進場，按下「輝月」就可以坐等「五殺」，最少都能拿「三殺」。
