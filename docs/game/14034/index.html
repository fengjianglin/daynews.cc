LOL：微博之夜年度人物票選活動開啟，Uzi、TheShy等九位選手登榜

2019-12-11 2,439條評論 4,366次閱讀 1524人點贊

北京時間12月11日，2019「微博之夜」年度人物票選活動開啟，在電競明星人物方面，LPL賽區包括Uzi、TheShy、JackeyLove、Clearlove、Rookie、Ming、Doinb、Tian、Xiaohu在內的9位選手登榜，目前Uzi、TheShy、JackeyLove三人分列第14、18、19名。

截至目前排名最高的Uzi收穫62137票，TheShy以42070票排名18，JackeyLove以39556票排名19。Rookie排名51，Doinb排名58，Ming緊隨其後排名59。

缺席了今年LPL夏季賽的EDG精神領袖Clearlove排名36，收穫11521票；S9全球總決賽FMVP選手Tian排名72，RNG中單Xiaohu排名85。目前登榜的9位選手，FPX有中野兩人，iG則是三C位，RNG是中單和下路雙人組，還有EDG的打野老將Clearlove。

2019「微博之夜」票選活動的時間為2019年12月11日-2020年1月11日，每位用戶每天有10票投票機會；連續參與投票5天，每天投票機會增加至15票；連續參與投票10天，每天投票機會增加至20票；每日用戶常規票數使用完，可通過做任務繼續拉票。1月1日0點起，每位用戶每天有50票投票機會。小夥伴們趕快行動起來，為自己喜愛的選手和明星投票助威吧！
