當年最經典的電子遊戲，14年熬出一部續作，為何玩家卻惡評如潮？

天天要聞
2019-12-05

摘要

如果之前有關注過遊戲眾籌的玩家，想必都聽過《莎木3》的大名。這款由日本知名製作人鈴木裕領銜製作的遊戲堪稱許多老玩家的“圓夢之作”，遊戲發布之初便引來了極高的關注量，許多系列粉絲都難掩內心的激動之情，紛紛表態“《莎木3》一出此生無憾”。

如果之前有關注過遊戲眾籌的玩家，想必都聽過《莎木3》的大名。這款由日本知名製作人鈴木裕領銜製作的遊戲堪稱許多老玩家的「圓夢之作」，遊戲發布之初便引來了極高的關注量，許多系列粉絲都難掩內心的激動之情，紛紛表態「《莎木3》一出此生無憾」。

當《莎木3》公布時，全世界都沸騰了

作為開放世界遊戲的鼻祖，《莎木》的前兩代作品確實在遊戲史上留下了不可磨滅的印記，也因此收穫了一大批忠實粉絲。但本次《莎木3》發售後，卻有很多玩家發現現在的《莎木》已經不再是當年那個具有劃時代意義的「先驅者」了，曾經在遊戲史上「開天闢地」的《莎木》，如今無論是畫面還是玩法都遠遠落後於其他主流遊戲。而像《莎木》這樣因為續作質量不佳而口碑一落千丈的作品在遊戲史上也不算少數，今天就讓我們一起細數那些「晚節不保」的遊戲吧。

《莎木3》的畫面放在今天確實難以恭維

不要再毀我的童年了——《雙截龍4》

年齡稍大，經歷過紅白機（學習機）時代的玩家，一定都對《雙截龍》十分熟悉，這款遊戲當年憑藉著優秀的打擊感與較高的挑戰性成為了許多玩家最喜歡的遊戲之一。所以ArcSystem宣布將於2017年發布《雙截龍》的正統續作時，許多老玩家都為止振奮不已。但當《雙截龍4》真正發售時，卻給所有粉絲當頭潑了一盆冷水——彆扭的操作、拙劣的關卡設計、層出不窮的BUG，玩家們不但沒能成功緬懷青春，反而還成為了「情懷消費」的犧牲品。最終，知名媒體IGN也只為這部拙劣的《雙截龍4》打出了3.5的超低分。

在這波「情懷」里，玩家成了被消費的對象

永遠沒有永遠——《永遠的毀滅公爵》

在FPS遊戲發展初期，1996年發售的《毀滅公爵3D》曾經是可以與《雷神之錘》與《DOOM》比肩的射擊遊戲不朽神作，一經推出就引爆了整個歐美遊戲市場。1997年，製作組3D Realms宣布續作《永遠的毀滅公爵》將於1998年正式與玩家見面，可其後的開發過程卻一波三折，從1998年一直跳票到了2011年，《永遠的毀滅公爵》才最終得以發售。但玩家們14年的等待並沒有換來期待中的遊戲，《永遠的毀滅公爵》發售後，過時的內容與玩法引來了如潮水般洶湧的惡評，對於《毀滅公爵》來說，也許續作永不發售才是最聰明的做法。

這也許是迄今為止遊戲史上最大的笑話

既生瑜何生亮——《波斯王子》

育碧公司出品的《刺客信條》大家應該都非常熟悉了，作為育碧罐頭廠的王牌產品之一，每次新作推出都會受到不少玩家的關注，但在《刺客信條》之前，育碧也曾經推出過玩法十分類似的《波斯王子》系列。作為一個歷史長達30年的知名系列，《波斯王子》是所有攀爬類冒險遊戲的鼻祖，育碧也是花了重金才從原作者手上買下了遊戲版權。但隨著自家IP《刺客信條》的快速成長，玩法類似的《波斯王子》也逐漸被公司輕視，在推出了極其敷衍的《遺忘之沙》後，《波斯王子》終於在玩家鋪天蓋地的批評聲中被育碧徹底打入冷宮，恐怕在今後很長的一段時間裡，我們都再難見到「波斯猴子」矯健的身影了。

如今恐怕沒什麼人會再想起這隻波斯猴子了

你的人生值多少錢——《模擬人生4》

對於許多模擬類遊戲愛好者來說，《模擬人生》幾乎都是必玩的項目之一，看著自己的角色一步一步成長起來，最終達成完整人生的成就感也確實遠非一般遊戲可比。但隨著其發行公司EA近幾年的「吃相」越來越難看，系列新作《模擬人生4》的價格也變得越來越離譜，如今想要購買包含全部DLC的完整版《模擬人生4》的玩家至少需要付出2000-3000元，這對於一款單機遊戲來說著實有些不可思議。如此明目張胆地利用遊戲DLC斂財，在一線遊戲作品中，大概也只有《模擬人生4》一家可以做到了。

2000多買個單機遊戲，值嗎？

其實對於玩家來說，看到喜歡的遊戲推出續作本應該是一件開心的事情，但經典作品往往也都是難以複製的，所以這種「續作不如前作」的情況才會屢屢發生。最後，除了小編所提到的這些遊戲，不知道對各位玩家來說，在你們心目中又有哪些令人感到遺憾的遊戲續作呢？歡迎大家在評論區留言，和我們一同分享自己的遊戲經歷！
