體驗服4英雄調整！安琪拉大招護盾加厚，姜子牙死亡光環有效

天天要聞
2019-12-10

摘要

就在最近，王者榮耀體驗服迎來了新一輪的調整。應該也是新賽季臨近，因此英雄改動非常的頻繁，而在最近的一次改動中對4個英雄進行了調整，3個都是上一輪削弱的英雄。

就在最近，王者榮耀體驗服迎來了新一輪的調整。應該也是新賽季臨近，因此英雄改動非常的頻繁，而在最近的一次改動中對4個英雄進行了調整，3個都是上一輪削弱的英雄。

首先來看成吉思汗，上一輪的改動直接將成吉思汗狠狠的砍了一刀，1技能的被動從3次改成了4次，雖然1技能的傷害加強，但依舊補不上這個坑，而這樣改動後下賽季的二哈或許又要重回下水道了。最近的一次改動中將1技能的傷害又提升了一些，整體來說還是削弱幅度比較大。

安琪拉也是連續幾次都進行了調整，安琪拉一直都不缺傷害，不過安琪拉的大招缺陷很大，那就是非常的依賴控制，如果直接放大敵人是很容易躲開的。上次加強了她大招的轉向速度，這次又加強了大招的護盾，喜歡這個英雄的玩家非常多，新賽季應該能成為比較熱門的法師。

姜子牙一直是職業比賽中的香餑餑，關鍵就在於他的光環前期優勢太大，官方也是多次削弱。而在上次改動中直接將他光環所提供的經驗每120秒一次性獎勵，而不是每10秒就獎勵一次，這樣導致了姜子牙前期的真空期完全沒有作用，作為彌補提升了他的傷害，這次又給他增加了效果，死亡時光環依舊有用，整體來說作用不如之前，但也依舊強勢。

嬴政之前一直處於重做狀態，而這次也是首次對重做後進行了調整。1技能和3技能藍耗降低，2技能冷卻降低，這都增強了嬴政的持續賴線能力，重做後嬴政的表現還是非常強勢的。

新賽季即將到來，體驗服的改動還是非常多的，那麼小夥伴們最希望哪些英雄獲得加強或削弱呢？
