陰陽師2019年SSR/SP式神PVE強度排行榜 陰陽師PVE式神推薦

天天要聞
2019-12-11

陰陽師進入2019年後，目前版本中對於SSR/SP式神中PVE的強度該如何評價呢？一起來看看本次的2019年第一季PVE強度排行榜吧，練好T1式神很快樂。

T1：八岐大蛇、茨林、小天狗、茨木、玉藻前、兩面佛、大天狗、陸生

目前遊戲中PVE活動絕對的主力就是他們了，至少有三中PVE活動中他們是最為核心的存在，甚至三種要更多，其中最典型的莫過於是傲視群雄的茨林以及玉藻前。兩面佛推拉流各種舒適，甚至比大天狗還要舒服，陸生傳統的PVE國家隊反擊核心流。八岐大蛇快樂就完事了，想要PVE玩好，這幾位養成極品，這遊戲想怎麼玩怎麼玩。

　T2：犬夜叉、殺生丸、輝夜姬、酒吞童子、荒

平常偶爾能用到的式神，比較偏向特殊環境和場合，有不可替代的作用，犬夜叉因為退治活動的最強多段效果很出彩，殺生丸更多的是依賴其禁復活+復活的功能性，輝夜姬現在用的少了，畢竟PVE以追月為主。酒吞偶爾可以用用，不是一線主力，同理荒收尾+協戰能力優秀但也不是主力。

T3：彼岸花、鬼切、面靈氣、白藏主、花鳥卷

偶爾會在PVE活動中用到的式神，但不是特別常用，有1-2個活動表現不錯。彼岸花作為不得不用的式神可能會用來刷狗糧和業原火。白藏主更多是活躍在秘聞副本中作為椒圖的替代品來保式神，部分活動也有出色的表現。面靈氣目前來看太過慢熱，能用到的地方是退治、超鬼王還有部分難打的活動。鬼切可能有人不服，個人之所以把鬼切排這麼低是因為這貨喜歡砍隊友，這誰頂得住啊。

T3：彼岸花、鬼切、面靈氣、白藏主、花鳥卷

偶爾會在PVE活動中用到的式神，但不是特別常用，有1-2個活動表現不錯。彼岸花作為不得不用的式神可能會用來刷狗糧和業原火。白藏主更多是活躍在秘聞副本中作為椒圖的替代品來保式神，部分活動也有出色的表現。面靈氣目前來看太過慢熱，能用到的地方是退治、超鬼王還有部分難打的活動。鬼切可能有人不服，個人之所以把鬼切排這麼低是因為這貨喜歡砍隊友，這誰頂得住啊。

T4：一目連、青行燈、妖刀姬、山風、雪童子

很少能在PVE中得到重用的式神，一目連過去還能在秘聞中活躍一下，現在算了吧。雪童子同理，偶爾可以用一用的類型，但不要指望太多，畢竟凍人太看臉。山風唯一能用的就是有砍暴擊效果的秘聞副本中，比如說紅葉皮膚塔。妖刀姬活躍在陰界之門或者魂十，不過現在基本上沒人用了，空有能力但沒就業機會。燈姐活躍在低於鬼王的極限刷分套路還有偶爾的超鬼王活動中，主要還是用來看臉供火極限刷分。

T5：小鹿男、閻魔、荒川、鬼燈、賣葯郎、桔梗、御饌津

　　這個T度的式神PVE幾乎沒有任何用處，因為大部分都是在鬥技的表現更加出彩一些，不過確實桔梗和御饌津有一定的PVE能力，但真的要麼太看臉，要麼實際用起來太卡手，桔梗原本有資格在逢魔中混一混的，結果那個被動護盾的動畫直接讓她蹲倉庫了。賣葯郎算是最為例外的，鬥技PVE都沒用的SSR
