Monet分析版本之餘推薦動漫，Yang評價虛無之靈花里胡哨

天天要聞
2019-12-10

​​從職業選手角度看新版本更新，Yang和Monet分別將從專業的三號位與一號位視角幫你解讀版本。在世外之爭顛覆性更新之後，我們分別邀請到了即將征戰萊比錫Major的VG戰隊三號位選手：Yang，以及晉級布克維Minor的RNG戰隊一號位選手：Monet，他們對待世外之爭的更新看法如何？不妨一同進入大神採訪實錄，深入了解職業選手眼中的世外之爭。

本次採訪將針對不同內容進行提問，分別包括：新英雄、綜合性遊戲改動以及選手個人相關問題，感興趣的Dotaer可以自行根據標題查閱。

虛無之靈採訪實錄

Q1：你認為虛無之靈的設計如何，目前應如何定位？

Yang：因為虛無之靈是個新英雄，我目前還沒有玩過，但是從我看過的比賽中我覺得，這個英雄打中單和四號位比較好。感覺虛無之靈的輸出雖然不是很高，但是在團戰中製造混亂的能力很強，如果要是能夠上CW的話還得被削弱個一兩次。

Monet：我認為這個英雄打2號位或者3號位都還不錯。目前來看這個英雄不僅需要等級，還需要裝備，但是在有了一定等級和裝備之後，他的成長性很好，逃生能力很強。

Q2：談一談你對於虛無之靈的出裝建議？

Yang：法系的話，吹風是一件核心裝備。而如果打物理輸出的話，則可以選擇電錘黯滅。不過這英雄還是打法系更好。

Monet：紛爭面紗、飛鞋都是不錯的選擇，中後期再補充輝耀增加傷害和帶線能力也很強。他的打法其實有點類似於法系火貓。比如說戰場上的持續傷害能力、強逃生能力等。

​Q3：如何看待未來虛無之靈在CW上的發揮？

Yang：我認為這個英雄打四號位還可以，走輔助路線，因為他的輸出不高，但是控制能力和拉扯能力還不錯。我不會在比賽中用他打三號位。

Monet：等到Major賽事結束後，虛無之靈應該可以加入CW之中，我比較看好這個英雄，它很有可能在比賽中大放異彩。

Q4：你認為哪些英雄是虛無之靈的絕佳隊友？哪些英雄又是他的天敵？

Yang：配合肉盾型三號位會很不錯，有輸出的英雄都很配虛無之靈，你把他當成正常四號位玩就好。

Monet：我覺得擁有沉默類技能的英雄比較克制虛無之靈。

Q5：虛無之靈是否有1號位潛質？會考慮練習嗎？

Monet：肯定會玩的，多一個英雄對於英雄池提升和比賽中的bp很有幫助。

電炎絕手採訪實錄

Q1：你認為電炎絕手設計如何？對這位新英雄的看法是什麼？

Yang：我覺得這個英雄設計挺棒的，有點像DOTA2的原始英雄。相比來說，紫貓的話就太飄逸了，感覺花里胡哨的。電炎絕手就很紮實，這英雄定位的話打個5號位挺好。

Monet：她的三技能特別厲害。舉個例子，我作為一號位經常能夠在對線時碰到這個英雄，在前期她可以出一個枯萎之石，再配合她的三技能，能夠打我200多血量，更何況我還是護甲很高的英雄。到了中期，電炎絕手配合上瑪爾斯或者薩爾這種英雄，通過框和新英雄大招的配合，傷害爆炸，中了這套連招的人基本必死。目前我感覺她是個前中期爆發非常強的英雄。

​

Q2：你對於電炎絕手的出裝建議是？

Yang：5號位出裝啊，團隊裝拉滿。其實現在輔助和大哥也沒啥區別，輔助裝備比如說大鞋這種，就很適合她。

Monet：我覺得電炎絕手是比較偏向於三四號位的英雄，前期的話相位或者秘法都是可以的，再往後就出吹風，同時，電錘、暈錘我也都看別人出過，感覺蠻強勢的。:

Q3：你一般怎麼稱呼電炎絕手，有沒有什麼有趣的外號？

Yang：說到這個，最近好像都沒怎麼見過這個新英雄，我忘了。

Monet：姨媽吧，感覺都這麼叫。

綜合性遊戲改動

​Q1：如何看待新版本有關信使的諸多改動？

Yang：先說說信使25級帶物品這個改動吧，感覺有點花里胡哨的，25級信使帶裝備不太現實，因為現在版本節奏太快，往往等不到那個時候。不過我覺得人人有雞這個改動挺舒服的，像我們這種老瘤子玩家就喜歡一人一個雞，挺舒服的。

Monet：我從我的感受來談一下，每個英雄都有自己的信使之後，我感覺優勢路運雞變得很頻繁，之前在兩到三分鐘前作為Carry位選手很難用到雞，一般來說都是五號位死了後帶點裝備過來，現在可以通過信使頻繁運裝備、補給。但是優勢路有個問題必須說一下，就是對方的四號位可以繞到後面去殺雞，所以運東西的時候也要非常小心。至於信使打到後期可以用裝備這個改動，我覺得大家可能還比較不習慣，因為節奏太快了，一般也打不到那個時候。

Q2：前哨對於比賽的影響？

Yang：我看來前哨的增加，對於比賽的雙方而言等於是多了一個選擇。像以前打到10分鐘、15分鐘時大家都在刷，都在搶賞金神符。但是現在的話，就可以圍繞哨塔展開進攻或是防守。前哨的出現，令有優勢的隊伍也能夠更好地擴大優勢或是進行壓制。

Monet：我認為這是一個很大的改動。從職業選手的角度來說，前哨在比賽對局中有非常重要的戰略性意義，前哨每5分鐘都會提供團隊經驗加成，如果你能夠控制住兩座前哨，那麼對方就不會有經驗提升，這令優勢一方更容易通過滾雪球獲得更大優勢。以後版本節奏也會越來越快，大家會根據前哨做文章。像是狼人、小魚人這種偏向於打架的偽核在這個版本可能會比較強勢。從戰隊打法上來談的話，我認為戰隊打法風格也要更凶一點。

​Q3：如何看待賞金神符地點改動？

Yang：河道賞金神符一般都沒人控，現在都在搶哨塔。

Monet：開局變得不一樣了，現在經常是三個人去搶一路的賞金神符，這樣就能夠穩定搶到兩個賞金神符，而如果對方硬著來，那我們就肯定會收到人頭。

Q4：你一般怎麼稱呼前哨？

Monet：我們有時候喊順口了，就也叫它「鍋」，哈哈。

Q5：中立道具加入對於比賽的影響在哪裡？

Yang：我覺得中立道具的加入挺好，但是在隨機性上還是太大了一些。比如說正常對線打到7分鐘，2打2的邊路對線，突然對方爆了一個秘法戒指，就等於給對方加了800塊錢左右的團隊經濟。而你要是爆了一個打野爪，那就沒什麼用。所以如果要多考慮比賽競技性問題的話，我覺得可能真的需要對中立道具的隨機掉落進行一定程度的限制，讓這個隨機性在一個相對來說可控的程度內。

Monet：現在可能對線稍微劣一點，但是爆個鐵樹之木、學徒之禮的話，就能夠令英雄得到一個不錯的加強。這等於是節省了很多可能之前會用在過度裝備上的花費，而讓關鍵性道具更容易出爐。

Q6：聊聊你目前最喜歡的三個中立道具？

Yang：之前的第三隻眼。可惜被刪了。鐵樹之木很好，然後就是巨神殘鐵，這個道具很厲害。

Monet：海洋之心、鐵樹之木。學徒之禮。這三件前期裝備都是敏捷英雄特別喜歡的。比如說，學徒之禮能夠提供+13點力量，讓敏捷英雄血量得到一個很高的提升，前期戰鬥力增強不少。

Q7：聊聊你不喜歡的或者認為雞肋的三個中立道具？

Yang：打野爪，這個排第一沒疑問。然後就是我剛才提到的+100射程的林野長弓，還有一個是三級裝備裡面的魔力箭袋。

​

Monet：騎士劍。我都不知道它是幹嘛的，作為一個三級裝備，不配好吧，和吸血鬼獠牙相比來說效果差不多甚至可能略遜一籌。感覺用起來好弱，反正沒有吸血鬼獠牙厲害。其他的裝備感覺都還能用，騎士劍是我每次拿起來都覺得浪費了，而且主要和第三級裝備相比，其他裝備都厲害得多。

​

Q8：面對如此巨大的更新，職業選手是如何迅速調整並適應新版本的？

Yang：其實職業選手和普通玩家沒有區別，也就是大家打得多、看得多，學到的就多。我對於版本的理解就是，現在的版本節奏更快，比賽也不會拖到很後期，輔助位相對來說更加好玩了。

Monet：版本的更新要仔細閱讀一下，然後記在腦子裡。不過還是要多打一下自己體驗新版本，比如說可以一邊玩的時候，出了一個中立道具不知道是做什麼用的，就可以切出去看一下。找到這條改動，就明白了，哦，原來是這樣。

Q9：本次更新後對於黑皇杖有一定程度的削弱，有一部分傷害技能能夠無視技能免疫，這是否會對於一號位出裝造成影響？

Monet：其實不會有什麼影響，出BKB是為了少吃一些減速、控制類技能。該出還是必須要出的，只能說在開啟BKB後也要相比以前來說更加謹慎一些了吧，需要注意一下這些可以無視技能免疫的傷害類技能。

選手個人相關問題


Yang

Q1：猛獁A杖改動很大，是否有體驗過？

Yang：沒有。這個真的很氣，我還沒玩就被削了。好氣。

​

Q2：能否三號位角度談談，你認為新版本更新後具體的影響是什麼？

Yang：可能最大的感受就在於信使改動這裡。現在每個人一隻雞，大家可以一直拉補給，導致線上對拼更加激烈。同樣，你也可以用雞一直拉裝備，這就導致了線上一旦擁有優勢，優勢方可以更好地擴大這個優勢。

Q3：此次萊比錫Major目標是什麼？

Yang：大家打得開心就行，目前目標是能夠拿到前六。

Q4：VG已基本可以確定被直邀TI10，為什麼還會選擇在春節去打萊比錫Major？

Yang：其實我們之前就打算鴿一個Major，那麼既然我們選擇去打萊比錫Major，下個Major比賽就不去了。

Monet

Q1：:目前RNG對於布克維Minor的展望是？

Monet：我們隊對於版本的理解還不夠深，這次Minor還是抱著一個學習的態度去打的。如果能夠拿到比較好的成績，那肯定也會很開心。

Q2：能否請Monet分享一下最近喜歡的動漫？

Monet：《這個勇者明明超強卻過分慎重》。因為這部番真的很搞笑，好久沒有看過這樣的了。
