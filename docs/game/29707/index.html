LGD新上單天秀一打三，Peanut人看懵了，LPL每個人都這麼凶？

2019-12-24 101條評論 1,472次閱讀 164人點贊

正在進行的德瑪西亞杯不知道各位觀眾看了沒有，德瑪西亞杯作為我們LPL自主舉辦的賽事，近年來也是受到了觀眾們越來越多的重視。因為在德瑪西亞杯的比賽中每支俱樂部都會安排新人上場練手，所以德杯也起到了檢驗戰隊新賽季實力的作用。每年德杯上總會有一兩名新人表現突出給觀眾們留下深刻的印象，今年也不出意外，RNG的小龍堡、EDG新上單xiaoxiang以及LGD新上單Cult都發揮得十分出色，尤其是Cult鱷魚一打三反殺三人，打野小花生看到後都懵了，我們一起來看看吧！

在今年冬季轉會期，LPL多傢俱樂部都花重金從LCK賽區購買了一些實力比較厲害的選手和教練，比如FPX戰隊的選手Khan、LGD戰隊的選手Peanut以及VG戰隊的教練Kkoma。其中Peanut的加入，使得LGD的粉絲再次希望，如今LGD戰隊擁有Peanut和Kramer兩位強力外援，因此在新賽季取得一個好成績應該是很容易的。在德瑪西亞杯上LGD也是派出新陣容進行磨合，在LGD對陣V5的比賽中新上單Cult發揮十分突出，打野Peanut直接都看懵了。

在這局比賽中Cult使用鱷魚對線劍魔，在前期打野Peanut幫助Cult連著兩次越塔強殺了劍魔，自此之後劍魔就一直被Cult壓著打。到了遊戲中期對方打野盲僧帶著輔助泰坦一起來抓Cult，此時Cult鱷魚擁有黑切、提亞馬特、死刑宣告、布甲鞋這四件核心裝備。Cule先開大招將劍魔打殘，同時閃現躲到泰坦的Q技能，在疊滿10層征服者後，一個滿怒Q直接將劍魔打死，然後又一個滿怒W暈倒盲僧並將其擊殺，最後再將泰坦收掉拿下三殺。

此時鏡頭下的Peanut一臉震驚，相信小花生心裡一定在想：LPL每個人都這麼凶嗎？不的說今年德杯中，好幾個新手上單都表現十分亮眼，難道我們LPL繼AD位置、打野位置崛起後上單位置也要崛起了嗎？不知道這幾位新手在常規賽面對TheShy、Khan等頂級上單時表現如何，讓我們拭目以待！

觀眾們對LGD新上單Cult鱷魚一打三反殺三人有什麼看法呢？歡迎評論交流一下，如果可以的話，我希望大家給我點一個小小的關注，你的一個小小的關注，對我來說就是最大的鼓勵！
