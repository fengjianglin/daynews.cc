LOL：LMS又一教練轉戰LPL，WE戰隊官宣原AHQ戰隊教練Domo加入

2019-12-16 2,794條評論 3,939次閱讀 593人點贊

北京時間12月16日中午，WE戰隊官宣LMS賽區AHQ戰隊教練Domo（龔育德）正式加入擔任主教練！Domo教練自2015年起先後在TPA、J Team以及AHQ執教，並在今年年率領AHQ以LMS二號種子的成績出征S9全球總決賽。

WE官宣正文內容：

【公告】

在與龔育德（ID：Domo）充分地交流溝通後，我們很高興地宣布龔育德（Domo）現加入西安曲江WE電子競技俱樂部英雄聯盟分部教練組，擔任主教練一職。

Domo是一名前英雄聯盟職業選手，退役後轉型為教練。自2015年起，先後在TPA、J Team以及AHQ執教。並於2019年率領AHQ以LMS二號種子的成績出征S9。

Domo教練有著豐富的執教經驗，對於英雄聯盟也有著獨到的理解，相信他的加入能夠為WE提供更多優秀的戰術，並催化隊伍的化學反應。新的賽季，一起努力，共同進步。

西安曲江WE電子競技俱樂部

2019年12月16日

今年的LPL冬季轉會期，S7豪門戰隊WE的雙C選手Mystic和xiye先後離隊，新賽季預計將以一眾新生代小將班底征戰LPL的WE面臨巨大的考驗。今年LPL春季賽中S2時期WE的當家ADC選手WeiXiao以主教練身份回歸後曾帶領WE成功闖入季後賽，而Domo教練同樣肩負著復興WE的重任，希望這位執教經驗豐富的Domo教練能夠頂住壓力，在明年LPL春季賽中帶領WE打出好成績。
