SMG機場圈堵橋遭遇4AM，因一個失誤反被4AM團滅

2019-12-18 680條評論 4,798次閱讀 3468人點贊

鬥魚的黃金大獎賽S8正在進行當中，在C組的小組賽第二局中，SMG和4AM戰隊發生了激烈的交戰，最終4AM團滅了SMG，這個鏡頭也引起了很多小夥伴的討論，一方面是因為這兩支戰隊都是高人氣戰隊，另一方面，這個鏡頭確實有很多值得說道的地方，一起來看一下吧！

這一局刷了一個機場圈，機場圈對於地圖中部和北部的隊伍來說，轉移都很困難，要等沿海的隊伍轉移完了之後才能有順位轉移，這一局就是P城的SMG戰隊率先轉移到機場，但是他們並沒有選擇去機場內部搶更好的位置，而是選擇堵橋。

在P城北部的4AM戰隊正好扎進來，此時SMG戰隊幾人集火掃4AM戰隊的車隊，4AM則直接往右邊直接拐過去了，在SMG掃車正酣時，永遠直接停車反打，一槍M24撂倒一人。

也正是這個突破口，4AM左拐的前點兩人迅速上車，直接搶到了SMG前面，SMG反而被4AM給堵住了，而永遠和王總則在橋附近，對堵橋的SMG形成了包圍之勢。

雙方在僵持一會兒之後，SMG的大京率先發現了永遠的側身，直接擊倒，但是有一顆小樹擋住了永遠，補不到，大京想換位置補人，結果被王總直接連打帶補，人數優勢直接被扳平。

近點的堡壘準備和王總進行battle，但是更北邊R城的QM此時從橋上過來，堡壘無奈只能和隊友匯合，往圈內撤退。

前面也說了，4AM的兩人趁著永遠打開的突破口直接驅車趕到了SMG的前面，堵住了SMG，王總滯後，SMG和4AM正面3V2，一番纏鬥當中，韋神再次祭天，不過小鹿站出來，直接一波一穿三，團滅了SMG戰隊！

賽後有網友說SMG就不該堵橋，其實SMG被4AM戰隊團滅並不是堵不堵橋的問題，戰術其實是沒錯的，如果他們成功堵橋，不僅是4AM，包括後面來的QM戰隊也會被他們收入囊中，成功堵橋並不是一定要團滅4AM和QM，把對方趕走也算成功，這樣，橋頭就是一個人造弱側，對於後續運營也很不錯的效果！

當然，這一切運營失敗的原因就在於正面沒打過，沒打過有兩個比較大的失誤：著急於補人。

大京率先擊倒永遠後著急於補人，結果被王總打措手不及，六神擊倒韋神後一梭子連打帶補，結果被小鹿直接穿掉了。如果大京不著急，等後續QM過來，王總就會被包夾，如果六神不著急，打殘血的小鹿還是有機會的。

可以看出來，現在整個SMG戰隊現在太過於著急拿分，擊倒一個人之後就想要著急於將這分數收入囊中，但總是因小失大，得不償失啊！

以上只是劉大大的個人看法，覺得說得有道理的小夥伴點贊收藏加轉發哦，如有不對，還請在評論區中留言~
