摘要

眾所周知，魔獸世界懷舊服自開服以來，不管哪個大區哪個伺服器，首當其衝的人口數據就是FS永遠排第一，這也跟FS的這一職業有關，瀟洒的動作，「騷」氣的操作，奇妙的能力，爆炸的傷害，簡直就是暴雪的親兒子！當然光有這個稱號還算不上「親兒子」，設計師設計的套裝裝備一直讓FS存在於第一梯隊，基本上不管幹什麼都缺不了FS，這也是經典舊世「法爺」尊稱的由來！這個「爺」可不只是搓個水面這麼簡單！下面分析一下各個套裝效果及FS選擇項的追去：FS套裝基本都對PVP有用散件就不再提了！

一、博學者套裝

套裝效果：2件加成200護甲值。4件提高所有法術和魔法效果所造成的傷害和治療效果，最多23點。6件被擊中有幾率冰凍敵人持續三秒。8件加所有魔法抗性8點。

點評：這件套裝的話T0屬性還算湊合，就是耐力加成太低，尤其單件的裝備是沒法看的，PVE還湊合打打，PVP就算了。【戰士跪了……T0幹啥都廢物…..】

著裝效果：我是看著很醜……60分？差不多了！

二：巫師套裝

套裝效果：2件所有魔法抗性加8點，4件被擊中有幾率冰凍敵人持續三秒，6件提高所有法術和魔法效果所造成的傷害和治療效果，最多23點，8件加成200護甲。

點評：黑翼之巢後開放的獲取難度高，金幣需求量打，對於小德，MS，SM，聖騎士可以說是寶貝疙瘩，FS的話不建議搞！

著裝效果：30分夠高了，基本沒法師帶著東西，看著還很難受！

三：奧術師套裝

套裝效果：3件提高所有法術和魔法效果所造成的傷害和治療效果，最多18點，5件法術攻擊目標時使其魔法抗性降低10點，8件使法術造成威脅值減少百分之15。

點評：MC掉落的T1套，對於打PVE來說是寶貝，PVP加成相對較小，相較於T0.5的巫師強那麼一點，跟T2直接沒法比。

著裝效果：70分，無特色，也不知道作為親兒子，前階段難道不需要養眼嗎？

四：靈風套裝（勇士跟少校不在贅述，穿的人太少直接T2詳解了）

套裝效果：2件使寒冰箭、奧數飛彈、灼燒、火球術造成的威脅值降低。5件使暴風雪、魔爆、烈焰傷害範圍增加百分之25。8件在釋放寒冰箭等法術時，百分之10概率使下個法術少於10秒的法術瞬發！

點評：「閃現的凈水機」這個肩膀屬實亮眼哦，穿齊了那真是非常猛的存在了，百分之10法術瞬發可以讓變羊大臉盆恐怖到一個層次。這件套裝完全可以用到60年代結束，打NAXX應該也不是很乏力，有句話說得好，T2在手，戰士滿街跑，那簡直就是碾壓。PVP跟PVE完爆，與R14雖無法比較但是瞬發真的是奇蹟。

著裝效果：85分，肩膀上的倆大燈沒幻化之前真的是亮眼的存在，反正小編是很喜歡這套裝，但是湊齊的難度也是很大，但是對於老闆來說應該湊齊的快，畢竟團里FS真的是太多了。
