巔峰對決陣容一模一樣！AGY勝EDGMY上演黑七奇蹟，今嶼9/1/8獲MVP

2019-12-07 0條評論 19次閱讀 0人點贊

摘要

KPL秋季賽已經進行到總決賽階段，而作為KPL發展聯盟的KGL聯賽也進行到了季後賽階段，KGL的季後賽是8支隊伍進行比賽，單敗淘汰制。常規賽排名第二的EDGM.Y對陣到常規賽排名第七的AG.Y。

KPL秋季賽已經進行到總決賽階段，而作為KPL發展聯盟的KGL聯賽也進行到了季後賽階段，KGL的季後賽是8支隊伍進行比賽，單敗淘汰制。常規賽排名第二的EDGM.Y對陣到常規賽排名第七的AG.Y。這兩支隊伍雖然常規賽的排名看上去差距很大，但是實力卻非常接近，前面三局打成了3-3平，比賽進入巔峰對決。

KGL的巔峰對決陣容和KPL上的風格差別不大，幾個常見的英雄姜子牙，公孫離，盤古和盾山都選了出來，而令人驚訝的是兩支隊伍的邊路也是一模一樣的，都選擇了版本強勢的曜。兩邊的陣容一模一樣，可以說是非常考驗選手的個人操作了。開局一分半AGY先手開到了koko並且拿到一血，但是EDGMY也馬上進行了反打，雙方打了個二換二，第一條暴君被EDGMY拿下。

從此EDGMY就一直在佔據主動，但是十七分鐘的時候EDGMY想要先手開團，卻被Twoz的盾山反開，AGY打出一波零換三，順勢拿下主宰。最後一波團戰EDGMY的koko盾山先手二閃背到了兩個人，但是AGY的今嶼一直在輸出，兩個曜都在切後排，最後AGY剩下一個盤古在和軒澤的曜在PK，而今嶼已經在正面推水晶了。雖然說最後曜擊殺了盤古，但是想要回家已經來不及了，今嶼的公孫離化身孤膽英雄單人拆掉水晶拿下勝利。

其實EDGM.Y的隊員們很多都打過KPL，包括大家熟悉的koko，邊路軒澤和藍領打野堪陽。而AGY的幾名隊員想必一些老觀眾也都比較了解。今嶼，阿澤和Twoz以及染安都是在AG打了很久次級比賽的隊員，個人實力也都不錯。中單笑影則是在本賽季KPL常規賽上過場的中單。常規賽的時候EDGMY表現得更好，但是這次的季後賽AGY一點都不慫，好幾局都是在前期佔到了優勢，最終也上演了黑七奇蹟。

這局比賽的MVP毫無疑問給到了9/1/8的今嶼，今嶼在AG打了很長時間了，射手一直都玩得不錯。不過AG超玩會的一隊這個位置是一諾，在整個KPL上都是頂級，並且年齡也非常小，今嶼想要在AG一隊首發還是太難了。這次的季後賽今嶼也有更多的機會被其他俱樂部看中，也許能夠通過選秀大會尋求一些機會吧。

大家對於AGY上演「黑七奇蹟」有何看法呢？歡迎在評論區留言交流。我是天黑君，期待下次再見！
