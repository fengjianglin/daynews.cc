王者：遊戲中的奇葩身份，李白是狄仁傑的罪犯，諸葛亮是“殺手”

王者榮耀4年迭代至今，已經誕生96位英雄了，而這麼多英雄，每一位都飽含設計師的精心打磨——擁有獨立的背景故事。在王者峽谷這個獨立的大世界中，很多英雄的背景非常有意思，比如李白的奇葩身份，你以為他是大唐第一劍仙？實際上是狄仁傑眼中的「罪犯」！而諸葛亮就更令人頭痛了，文縐縐的他是令人聞風喪膽的「殺手」！

李白在大部分人眼中都是虛無縹緲的謫仙，一手青蓮劍歌配合破軍不知道傷害了多少射手玩家的心~然而他在背景故事中，實際上是狄仁傑的「罪犯」。李白2次平白無故闖入長安城，在狄仁傑的眼中就是不折不扣的罪犯，如果不是武則天阻擋狄仁傑的逮捕行動，李白早就被以強闖朱雀門，破壞長安城治等問題逮捕起來了，所以說李白就是狄仁傑的「眼中釘，肉中刺」

而諸葛亮在背景故事中的地位也是很尷尬。諸葛亮和曹操算是不分伯仲的兩大梟雄，一直在故事中互相對抗著，而在蔡文姬的背景故事中卻提及到了諸葛亮，在蔡文姬見到她的義夫曹操後，曹操直言蔡文姬父親的離世就是諸葛亮下的手，因此諸葛亮也算是背了一個大黑鍋。沒辦法誰讓這倆是對抗關係呢，被摸黑也正常。實際上在背景中蔡文姬父親的殺手另有其人，並不是諸葛亮，小編猜測是典韋。

首先來看典韋的一個很尷尬的身份，那就是蔡文姬眼中的「寵物」。典韋變得狂躁易怒實際上是被人強行改變後才變成這樣的，在典韋的背景故事中有所提及，是一個「梟雄」的實驗讓他變成這樣。所以從關係上來看應該就是曹操改造的典韋，讓他暴躁易怒，而且背景中還提及到，蔡文姬父親的死是自己（典韋）害死的，所以他才願意一直守著蔡文姬長大，從這裡看得出來，諸葛亮確實背了一個大黑鍋！

除此之外，遊戲中還有一對宿敵，便是墨子和魯班大師。墨子在借走魯班之後，在路上卻把魯班弄丟了，對魯班七號一向以兒子身份看待的魯班大師頓時不樂意了，直接就杜絕了和墨子的一切關係，哪怕在此之前他們還算良性競爭，互相比比機關術的造詣共同進步也是很美好的事，可是弄丟魯班大師的心血就太過了！所以這一對直接成了解不開恩怨的「宿敵」！

相應的，魯班七號玩家的地位也變得尷尬起來，如果匹配到的路人掏出了自己剛買的魯班大師，說他是魯班的父親，魯班七號連反擊的話都沒有，奇葩身份石錘了~最後，關於狄仁傑為什麼這麼偏激的對待李白：狄仁傑愛長安城，達到了長安城每一塊磚都有他獨特的記號，李白上來二話不說，就給人朱雀門打上一朵劍花，這狄仁傑能不急眼？
