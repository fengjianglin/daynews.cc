FIFA20模擬：熱刺3-1狼隊，凱恩梅開二度，阿里長驅直入

2019-12-16 175條評論 2,330次閱讀 786人點贊

EA旗下的FIFA遊戲一直是全球足球電玩愛好者首選，該遊戲以真實程度高而聞名於世，已經使用了多年寒霜引擎的FIFA在引擎上就已經領先了實況足球太多，所以FIFA的用戶也越來越多了，再加上版權上的優勢，FIFA已經牢牢佔據了足球遊戲龍頭老大的位置。而隨著近些年FIFA的UT模式越來越吸引人，也越來越吸金，FIFA遊戲的商業模式也非常健康，所以他們也能投入更多的錢去更迭作品，做更好的足球遊戲。而早在2019年10月初，FIFA就官方宣布了遊戲玩家已經突破了1000萬人，可以見得FIFA在全球是有多麼的受歡迎啊。

2019年12月15日晚，熱刺對陣狼隊，在本場比賽首發出路之後，筆者用熱刺同樣的首發在FIFA20和狼隊踢了一場比賽，半場時間為8分鐘，遊戲難度為世界級，全場比賽最終熱刺3-1戰勝了狼隊，熱刺方面凱恩邁開二度，阿里也打進一球。下面讓我們看看本場比賽的進球畫面。

上半場凱恩在對方禁區混戰中抽射破門。

下半場狼隊通過一次反擊，最後由穆蒂尼奧將球打進扳平了比分。

隨後凱恩接阿里的助攻，再次將比分反超。

在此之後，阿里一人直接殺入對方禁區，將比分改寫成了3-0。

全場比賽熱刺佔盡了優勢，全場射門次數達到了17次。
