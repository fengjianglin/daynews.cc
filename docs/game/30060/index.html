LPL最強中野誕生，20分鐘打出15-1完美數據，隊友只需躺贏

2019-12-23 23條評論 3,582次閱讀 2675人點贊

德瑪西亞杯小組賽階段，TES戰隊迎戰V5戰隊，作為LPL的兩支隊伍，TES在轉會期進行了補強，卡薩的加入無疑增強了戰隊的整體實力，而Knight和卡薩的配合更是天衣無縫，在Rank中罕有敵手。相比之下，V5戰隊的實力就大不如前了，不僅沒有引入強援，還賣掉了解約了明星選手otto！

TES陣容

卡薩來到TES，這對於TES來說是一件大事，本身的TES主打中上，但先前熊熊在野區的影響力太小，這讓TES實力沒法完全發揮出來，而這次引入卡薩，則讓TES擁有了LPL最頂尖的中野組合，卡薩和Knight的韓服Rank證明了這一點。

TES vs V5

在這次的德瑪西亞杯中，卡薩和Knight拿到了酒桶、奇亞娜這對活寶組合，酒桶的強制位移和奇亞娜的定牆簡直是絕配。作為針對，V5這邊則是拿到了塞拉斯和皇子，這對組合的Gank能力極強，只是在爆發上略遜於酒桶、奇亞娜。

TES中野發力

比賽開始後，卡薩主導設計了一級反野，這讓V5非常被動，皇子甚至在交出閃現的情況下被對手擊殺，而拿到優勢的卡薩並沒有埋頭刷野，而是來到中路幫Knight進行Gank，Mole的塞拉斯雖然很強，但面對著毫無破綻的TES中野雙人組，Mole只能乖乖交出人頭！

中野聯動

比賽進行到8分鐘，TES中野雙人組利用視野優勢抓到了來中路做視野的V5輔助，Knight甚至沒有交大招就將輔助瞬間秒掉了。TES雙人組宣告起飛，而Knight也轉變了思路，放棄了遊走的想法，僅靠對線就將Mole徹底打崩了！

越二塔強殺

Knight甚至在裝備優勢的情況下越過二塔強殺V5打野，這讓V5士氣大跌，甚至整支隊伍都失去了戰鬥力，說到底，這場比賽中的卡薩作用太大，先是幫助Knight建立起優勢，隨後Knight利用經濟差不斷秒殺V5中野輔，最終讓V5徹底失去了翻盤的機會。

22分鐘結束比賽

甚至進行到20分鐘的時候，TES中野雙人組已經拿到了15個人頭，僅Knight死亡一次，TES上單369全程對線補刀，隊友只需要躺好就完事了！卡薩和Knight這對明星組合也成了LPL最強的中野雙人組，S10將會成為這兩人表演的舞台！你認為卡薩和Knight可以主宰接下來LPL的春季賽嗎？期待你在下方評論區留言討論呦！
