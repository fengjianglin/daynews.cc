夢幻西遊手游：估價2500萬的物理神器引網友讚歎，號主卻氣到自閉

2019-12-14 560條評論 1,935次閱讀 924人點贊

裝備打造一直被譽為發家致富的不二法門，畢竟極品裝備沒人嫌多。但有時候，即便打造出極品的裝備，可能也因為各種原因無法售賣，從而血虧。日前，遊戲君就在遊戲中遇到這樣一位玩家，好不容易打造出一件物理神器，本想著能就此一夜暴富，卻沒想到因為專屬屬性而無法售賣，讓該玩家一時間氣到自閉。

說實話，對於專屬屬性，相信不少玩家是又愛又恨。點了專屬固然能讓裝備的屬性更強，但若打造出的專屬裝備不合自己用，那可就虧到不行了。該玩家就是如此，因為身上的石頭只夠打造70級的裝備，所以該玩家便想先打造件70級的武器試試水，沒想到在點了專屬後，居然打造出這件物理神器出來。最關鍵的問題是，該玩家本身的等級是89級，拿著這件70級專屬簡易物理神器根本沒用，賣又賣不掉，血虧到不行。

就該玩家的這件70簡易雙加的武器來說，遊戲君看了瞬間心動。基礎物傷高達628，雖然沒爆，但也非常極品了。要知道專屬打造基礎物傷的上限也不過641。除此之外，這件武器還是體力雙加，10點體質雖然平平無奇，但這22點的力量就厲害了，畢竟70級雙加武器，極限綠字也就25點。不誇張地說，這件武器對於卡69的物理輸出門派來說，簡直就是神器一般的存在。

對於這件武器，不少網友在評論區直言，這武器要是沒專屬，要賣的話，起碼能賣到2500萬。本來能一夜暴富的，結果是因為專屬痛失暴富的機會，也難怪該玩家氣到自閉了。還有網友對該玩家89級打70級裝備還點專屬的舉動疑惑不已，認為都知道是在打70級裝備了，還點專屬，這能怪得了誰。其實，打造裝備本來就是看運氣的，點了專屬也不一定出好裝備。該玩家或許是抱著這樣的心態，哪曾想一下就出了極品，真是成也專屬，敗也專屬啊。

其實若這件裝備只是沒法出售，那倒還好，畢竟裝備不能賣，還可以直接賣賬號嘛。但該玩家本身是卡89的，卻打造出一件70專屬簡易雙加的武器，這就算是賣號也賣不出去。這等於是把該玩家最後一條路都堵死了，實在是令人唏噓。

圖片來自網易大神，作者夢幻遊戲君

當然，雖然這件極品裝備因為專屬變得一文不值，但遊戲君覺得既然裝備賣不出，那不如收藏起來，每次打造裝備時拿出來看看，說不定能還能給自己帶來些許歐氣。

那麼，大家在遊戲中是否有遇到過如該玩家一般的情形呢？不如在評論區留言，與遊戲君分享下吧。
