胖哥鏈（Pong Chain）是如何在遊戲中脫穎而出的

2019-12-07 0條評論 18次閱讀 0人點贊

摘要

如今不管在那裡都少不了互聯網領域，衣食住行統統在網上應有盡有，互聯網時至今日飛速的發展一直處在遙遙領先的狀態，與時代共同進步，把量的積累成為質的改變；互聯網的進步，區塊鏈的實現，來自於互聯網領域肯定也少不了遊戲+區塊鏈的時代。

如今不管在那裡都少不了互聯網領域，衣食住行統統在網上應有盡有，互聯網時至今日飛速的發展一直處在遙遙領先的狀態，與時代共同進步，把量的積累成為質的改變；互聯網的進步，區塊鏈的實現，來自於互聯網領域肯定也少不了遊戲+區塊鏈的時代。而遊戲本身也是區塊鏈的一個生態圈，由於擁有種類繁多,而備受大眾玩家的喜愛；但傳統遊戲發展至今,也存在本身的局限性；今區塊鏈 遊戲”的模式的快速發展也帶動了許多遊戲公司的開拓公鏈。

而我認為來自於韓國的胖哥鏈(pong Chain)就是全球遊戲公鏈的開拓者，自運行以來，已經上了兩款遊戲，其中值得肯定的是ISO共贏模式，這個模式真正做到了去中心化的管理和應用，無論是在胖哥鏈（pongchain）生態社區還是pong流通的二級市場，胖哥鏈（pongchain）都在穩紮穩打的前行，胖哥鏈（Pong Chain）社區生態用戶已突破10萬人次，生態代幣pong在二級市場表現不俗。

而胖哥鏈（Pong Chain）致力於打造全球最大的遊戲網路時代，用創新的技術方案，來解決傳統遊戲行業資源與渠道的壟斷，以及實現現有區塊鏈遊戲體驗感弱兩大重點。由此看來，胖哥鏈可以算得上是區塊鏈遊戲中的一匹黑馬；就拿8月份，胖哥鏈（Pong Chain）首發上線Bit-z來說，開盤便獲得了300%的漲幅，成為bit-z全網漲幅第一名。

那麼胖哥鏈（Pong Chain）憑什麼能成為眾多區塊鏈遊戲的領先者 ？又是為什麼讓它本身的價值在上漲呢？其這與胖哥鏈（Pong Chain）的發展運營方式離不開關係，它以遊戲平台作為切入點,既確保了大方向的正確,又符合了現實的發展規律。但冰凍三尺，非一日之寒；想改變遊戲行業除了技術上的阻力,還可能受到既得利益者的阻礙。

最重要的事從運營項目到生態代幣上線，只有胖哥（Pong Chain）鏈做到了；然而最為主要還有ISO是首個以社區利益為核心的發展機制，ISO中的每個步驟都提現在用戶者身上，圍繞社區利益而設計的，確保在每一個項目產生的價值，希望能回饋到社會中去，他們以這種回饋的方式產生的持續發展的動力，做到了利民利己 ；以社區玩家利益為核心，做到了無障礙支付，可以實現兩個用戶隨時隨地線上線下進行運行；做到了資金全球安全化，在區塊鏈上，玩家可以擁有遊戲內的資金。

而這些資產具有更廣泛意義上的流通性；這就是他脫穎而出的存在之處，做到了公平，公正，真正去中心化的遊戲社區。因此胖哥鏈（Pong Chain）在這成功的背後都是有數十、數百或數千個不成功的遊戲來搭建的基礎，和它一直在潛心做事的態度分不開。

目前，胖哥鏈（pongchain)上線以來運行的多款區塊鏈遊戲均獲得市場的高度認可，且通過獨創的ISO共贏模式，打造了集趣味、公平、贏利、益智、安全於一體的去中心化鏈上遊戲平台獲得了胖哥鏈（Pong Chain）社區的高度認可；截止目前胖哥鏈（Pong Chain）除模擬城市和K3遊戲外，新遊戲「梭哈」亦將會成為（pongchain）胖哥鏈繼模擬城市和K3之後又一優秀的代表作。

胖哥鏈（Pong Chain）遊戲公鏈也應運而生；負責遊戲底層公鏈及生態的搭建，面向全球遊戲開發者提供區塊鏈遊戲開發的底層技術和智能合約：實現了遊戲資產的所有權和流通性和安全性。且採用會員邀請制的模式，根植了”共識即收益”理念的全新遊戲產品，顛覆了傳統遊戲中的玩家收益分紅模式。

而且還採用通過競選的公平方式，讓每位遊戲玩家都能公開地參與到遊戲聯運當中，通告持有PONG競選遊戲聯合運營，競選成功後，即可獲得遊戲的運營權和收益權，讓每個支持和持有胖哥鏈（Pong Chain）的用戶，都有公平的機會，獲得巨大的發展機會。

其實區塊鏈和遊戲的結合時間並不算長,可是其取得的後果卻異常驚人。除了其不只有傳統遊戲的可玩性之外,更是完美的規避了傳統遊戲的漏洞,給了玩家一個”安心玩遊戲”的環境,相信區塊鏈遊戲也必然會教育遊戲市場走向高峰,而作為ISO共贏模式下的胖哥鏈（pongchain),無論是在一級市場還是二級市場。

當前，在區塊鏈遊戲被污名化和伴隨著Fomo3D、貝爾鏈等打著區塊鏈遊戲行資金盤模式的遊戲崩盤，許多投資者對區塊鏈遊戲開始觀望和持反對意見，但胖哥鏈（pongchain)的穩健運營和其低調的態度，讓其無意以過多喧嘩吸引用戶關注。

再回過頭來，你還會發現胖哥鏈（Pong Chain）其實做到了其他開發者沒有做到的，並且做到了交易記錄不可逆轉，資金調配安全隨意，做到了利民利己的服務核心，得到了大眾的認可與支持，實現了全球最大遊戲區塊鏈網路，在我個人看來這是最值得讚揚的。總之，我個人是非常看好胖哥鏈（Pong Chain）的，因為他用創新的技術解決了傳統遊戲行業里不能解決的一些問題；我相信，時間會證明胖哥鏈（Pong Chain）默默為遊戲做出的一切努力都是有回報的。
