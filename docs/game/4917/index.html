《我的世界》史上最困難BOSS誕生 開創造模式都打不過的無敵怪物

2019-12-04 0條評論 16次閱讀 0人點贊

摘要

泰坦，是希臘神話中統治世界的古老神族，他們極其強大戰無不勝，若不是受到其父親天穹之神的詛咒，奧林匹斯神族根本無法將其戰勝並取代。今天，給大家推薦的模組就取自其意，由AndrDneehtfoerusaert團隊製作的《泰坦生物》，在這個世界中，你將見到如同泰坦神族般無敵的存在。

泰坦，是希臘神話中統治世界的古老神族，他們極其強大戰無不勝，若不是受到其父親天穹之神的詛咒，奧林匹斯神族根本無法將其戰勝並取代。今天，給大家推薦的模組就取自其意，由AndrDneehtfoerusaert團隊製作的《泰坦生物》，在這個世界中，你將見到如同泰坦神族般無敵的存在。

模組來源

在模組中心搜索泰坦生物，即可找到本模組，僅支持1.7版本。但你不得不注意的是，本模組對電腦要求極高，請為你自己的顯卡負責。

維度

在這個模組中添加了兩個全新的維度，它們都需要擊殺末影龍後才能進入。虛空之地是一個充滿混亂的世界，世界會反覆變成不同的顏色。凋靈斯拉、強化凋靈和凋靈炮台會瘋狂攻擊你，哪怕你開著創造模式都很難在這裡生存下來。

相對於虛空之地，無名之地就穩定了很多，這裡充滿了黑曜石和敵對生物。雖然也有一定危險，但這裡充滿了可以製作強力裝備的礦物。

物品

面對強力的敵人，當然也要有強力的裝備。使用模組中特殊的礦物，就可以合成虛無裝備和末晶裝備。這兩種裝備都可以一刀砍出上百的傷害，只有用它們你才有挑戰泰坦生物的資格。假如你有幸擊殺了神，就可以獲得最終兵器——創世之刃，這個物品甚至可以打破基岩，其大小也十分可怕，連物品欄都裝不下。

泰坦生物

這個模組最吸引人的地方是，它帶來了一群幾乎無法挑戰的BOSS，它們每個都如同神靈一般俯視著你。和它們相比，末影龍就如同一個剛出生的嬰兒一樣稚嫩。

大多泰坦生物都是放大後的原版遊戲生物，但是它們都擁有上千上萬的血量與非常可怕的攻擊力。除此之外，它們還會瘋狂破壞周圍的方塊並召喚出數量眾多的隨從。

這些泰坦怪物的每次攻擊都會帶來地動山搖的效果，甚至連雷電之力也被他們掌控。哪怕你開啟創造模式，它們也可以輕鬆擊殺你。

神

與BOSS不同，神只有一個，它就是凋靈斯拉。當他出現時，黑暗降臨，電閃雷鳴，但你甚至都無法看清他的樣子。也無法形容它的任何攻擊，因為它通常都會秒殺一切生物。這幾乎是一個無法戰勝的傢伙（因為你的電腦可能不允許），過於強大的能力讓每個初次看到他的玩家都目瞪口呆。它與notch一樣，都是無敵的象徵。

如果你還沒有見過泰坦生物的強大，那一定不能錯過這個模組。那麼你覺得凋靈斯拉和HIM，誰更強大一點呢？
