VG 2:1再滅Secret，今天能完成一穿二嗎？

2019-12-22 3,681條評論 4,658次閱讀 3476人點贊

新加坡邀請賽今天還在繼續，昨天三輪BO3戰罷VP 2:0輕取NaVi，VG讓一追二逆轉TNC，而勝者組決賽里小組賽至今不敗的EG仍然保持了自己的強勢，2:0橫掃紅星率先闖進了決賽之中。

EG這次的新加坡之旅頗有一絲無敵的味道，小組賽五輪比賽他們收穫了四勝一平，一個系列賽都沒輸過。而他們唯一戰平的隊伍正是勝者組決賽里的對手紅星老師，不過這次EG沒有給紅星扳平比分的機會，直接2:0將對手斬於馬下。

小組賽5個BO2里EG拿下9勝1負，淘汰賽兩輪BO3他們又收穫兩個2:0，目前總戰績13勝1負勝率高的驚人。現在EG已經率先殺入到總決賽中，距離最後的總冠軍僅有一步之遙，他們能否將無敵的狀態保持到最後，我們拭目以待！

不過EG究竟有多強我們也不在意，以前看中外對抗國外的強隊就那麼幾個，其他都是魚不足為慮；現在再看所有國外隊感覺都很強，不然也不會兩個隊沒進淘汰賽就一輪遊了。

我們的焦點還是集中在VG身上，看看這個僅存的火種究竟能在敗者組燒多久。

昨天逆轉TNC後，VG敗者組第二輪遭遇到大魔王秘密。秘密這賽季把MidOne飛了換成馬桶哥，組成了「最屌這一團」要在新賽季整活一哈。只是在新加坡邀請賽開始前隊伍四號位突然「家裡有事」，喊來MidOne過來替補，於是實力DOTA又變成了快樂DOTA。

VG和秘密這輪系列賽打滿了三局，三局比賽我們真真切切感受到了MidOne的快樂，他也成功把秘密帶回了「快樂老家」。在前兩場雙方戰至1:1平的情況下，秘密突然選出來一套水人影魔劇毒三核的蛇皮陣容，對VG的PUCK和小魚毫無辦法。

這場至關重要的生死局以VG碾壓而告終，18分鐘人頭24-5，在天輝野區一波團滅後無心戀戰的秘密直接敲出GG，不想再掙扎了。

昨天讓一追二擊敗TNC，今天2:1淘汰Secret，雖然兩輪比賽贏的都不輕鬆但VG都笑到了最後，他們的狀態似乎開始回暖了。

今天VG的征程還沒有結束，淘汰掉秘密後他們將在敗者組第三輪面對強敵Alliance，核桃tv和龍神tv全程解說，關注比賽的同學不要錯過。

VG想走到最後還要跨過3個敵人，他們需要先滅Alliance再斬紅星，最後再去挑戰EG。現在VG似乎已經取代LGD成為新的「全村人希望」了，上個賽季VG斬獲兩個Major冠軍，成都Major上他們也殺進了決賽，這次新加坡邀請賽也比LGD走的更遠。

VG加油啊！
