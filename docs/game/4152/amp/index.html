529法傷的冰法可謂是十分恐怖了，基本上脆皮職業來最差都是一換一吧，冰環冰箭吹風，4秒內傷害期望2700，這個組合基本冰環能用就能用，而且還只是個小招。來看看其他玩家如何評價吧：

裝備太好，隨便秒殺脆皮

羊開符炎爆氣定炎爆下線，3分鐘後又是一個殺神鐵門口未嘗一敗，2700是按照冰環後暴擊幾率提升50%到60%計算所得。如果是暴擊幾率提升到100%那傷害期望是3400。

PK再厲害拿得到補貼嗎？開的了G團嗎？帶的了老闆嗎？別人畢業站鐵橋百人圍觀，你站城門口就那幾***一天打打打。我暗牧436暗傷，加個耐血才3500，穿高三維裝4300血但是只有200暗傷打不死人，現在還有點尷尬，要傷害就沒血量，要血量就沒傷害打不死人。

大佬，太厲害，勤奮，我一直在猶豫純冰傷是不是犧牲太多，暴擊至少少了8%左右，8%冰傷摺合也有小100的法傷，另外智力少法力值也少了，會不會在老10這種boss ，老9這種boss缺藍，畢竟喚醒8秒又是4個寒冰箭了。

你每次打完mc翻一下所有法師的技術統計看看暴擊率就明白了，按道理裝備正常的法師自身都會有11%到15%的暴擊率加上深冬之寒應該是21%到25%的暴擊率，實際上mv全程下來沒有一個法師暴擊率超過20%的，所以數據只是數據，還要看實際效果才行。

法師身板太脆，最多野外一換一

拿冰法打術士就是找虐，這版本元素好打些，後期t3級別可以靠奧火爆發壓制術士，後期賊法爆發太高了，一般職業扛不住，即使是術士，還是得看天賦和裝備的。

感覺野外環境，弱勢陣營，一場戰鬥能舒舒服服讀一條寒冰箭真不容易。這身被先手有點脆，你這個血量我惡魔術士吃個胖子瞬召條狗給你腐蝕，痛苦，獻祭上滿了讀個恐懼看著你死。

我不久前在燃燒挖礦，遇到一個法師上來就羊我，上來就是兩發寒冰箭外加一個冰錐還是啥技能，第三發寒冰箭打出來之後我還欠他血。。我奶騎，野外爐石沒cd,我誰都不怕，有cd,誰都不怕我。
