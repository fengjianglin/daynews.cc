以地域化為核心，KPL正鼓勵俱樂部成長轉型為企業

2019-12-19 152條評論 2,053次閱讀 854人點贊

出品｜人民電競

作者｜王哲瑋

編輯｜王晨

題圖｜KPL

隨著解說的一句「讓我們恭喜AG超玩會」，2019年的KPL秋季賽宣告落下帷幕。

「銀河戰艦」4:1擊敗老對手QGHappy，奪得總冠軍，也報了2017年春季賽總決賽中0:4慘敗給對手的一箭之仇。

除了兩支老牌勁旅再次相遇的話題性，這次總決賽的另一個特殊性在舉辦地點：武漢——繼上海、成都、深圳、北京、西安後，第六個舉行KPL總決賽的城市，同時也是首個俱樂部城市主場，2019年春季賽和夏季冠軍杯冠軍eStarPro將於明年正式入駐。

雖然eStar Pro很遺憾地沒能打進總決賽，但這並不能改變武漢這座城市在KPL歷史中的里程碑地位，而以「地域化」為內核的聯盟版圖擴張，同時也進階到了新領域：俱樂部這一腰部力量，開始在KPL發展中成為真正的重心。

地域化是變革前沿

KPL走到今天，回答了中國電競產業的許多疑問。

例如「移動電競是不是偽命題」，實際上KPL在自己誕生的第二年就回答了這個問題，移動電競不僅不是一個偽命題，並且擁有相當的爆發力：2017年，KPL全年內容觀看及瀏覽量超過103億次，總決賽單日觀看量超過2.4億——即便放在電競全行業，這些數據也稱得上是頂級流量。

「從大家認為這是一個偽命題，到大家相信、認為這是一個趨勢，越來越多的廠商和行業合作夥伴加入進來。」KPL聯盟主席張易加在接受採訪時感慨。

但在一鳴驚人之後，如何不斷打磨、完善一個賽事體系的基礎設施，才是一個賽事品牌是否能夠長遠發展的關鍵。隨後的2018年，KPL也開始真正進入自身變革的深水區——地域化和國際化。

首當其衝的就是地域化。從2018年春季賽開始，KPL開始實行上海、成都的「雙城主客場制」，聯盟中原有12支隊伍中的6支從上海遷往成都，施行類似NBA的東西部對抗。

從某種角度看，「地域化」其實已經成為了KPL變革進化中，實質意義上的最前線，許多變化都是隨之鋪陳開來。例如賽制被劃分常規賽、季後賽，直至最後總決賽，就是為了適應「雙城主客場制」的針對性改革；而國際化進程中，在眾多境外地區設立《王者榮耀》戰隊的做法，也可以理解為一種特殊形式的「地域化」。

在傳統體育中，尤其是足球這樣的全球化運動中，競技與城市相融互促的關係已經無數次被驗證，諸如曼徹斯特、巴塞羅那這樣的名字出現在輿論中，許多人的第一反應可能都是足球隊，然後再是城市。而與之相關的商業合作、政策福利、文化塑造等層面，也自然就水到渠成了。

「地域化是提高電競大眾影響力最快速的方法之一。」張易加不只一次在公開場合表示，當電競產業開始地域化時，便有機會與城市進行深度綁定，促進城市文化的傳播，從而達到雙贏的結果。而這也是KPL全力推進地域化的根本動機。

最初考慮到賽事發展還不夠成熟，KPL決定採用分步驟的策略去實現地域化，第一步就是雙城主客場制模式。而如今eStarPro確認落戶武漢，建立了首個KPL俱樂部主場，顯然是聯盟官方認為當前賽事的成熟度，已經到了更進一步推行改革的層級。「這是我們進入到下一個階段具有里程碑意義的事件。」張易加認為。

截至目前，KPL官方公布過的最新觀賽數據時間是2018年：全年內容觀看量已超過170億（同比增幅65%），KPL秋季賽總決賽單日直播觀看量破3億（同比增幅26%）——2019年雖還未完結，但不出意外的話，今年的觀賽數據或將超過200億，只要保證同比增長超過17%以上。

俱樂部是核心產品

eStarPro落戶武漢，對於KPL而言意義重大，但絕不是讓一支隊伍搬個家那麼簡單。

張易加認為，當前KPL在推進地域化過程中的難點主要有兩個：第一是舉辦賽事成本高，包括場地成本，賽事的直轉播、宣傳等成本；第二，各地電競產業發展水平不均衡，人才、場館等產業配套資源不一定能跟得上。「當然產業是在前進的，未來這種情況會有所改善。我們也在尋求更好的方式，一方面能控制以及降低俱樂部成本，另一方面能提升商業收入的可能性，這樣俱樂部在進行地域化過程中會更有動力。」

今年8月，eStarPro在武漢舉辦的主場落地儀式，得到了武漢市政府的大力支持，武漢旅遊發展投資集團更是在其中起到了舉足輕重的作用，而當時到場祝賀、並發表合作計劃的還有寧美互聯、武漢白鱀豚保護基金會、漢口銀行、東風暢行（武漢）、湖北廣電等本地企業和機構。

從中不難看出，這可能就是KPL在探索地域化過程中，給出的大方向：鼓勵、引導俱樂部強化自身運營實力，整合本地政商資源，從而實現與城市的有機融合。

這也意味著，隨著地域化的不斷推進，KPL聯盟與俱樂部彼此之間的權責平衡，可能也正在發生微妙的變化：聯盟處於不斷變「輕」的過程，釋放更多權力給俱樂部，鼓勵他們真正實現自身的良性運轉；而反之俱樂部則需要更多考慮變「重」的問題，完善各項商業基礎設施，實現向一家企業的轉型，而不再是像初建時期的「小米+步槍」，將100%的重心都傾斜在賽訓上。

「更多隊伍擁有自己的城市主場，長期來看是一個必然的結果。」張易加認為趨勢是一定的，但至於什麼時間發生，還是要取決於市場和俱樂部職業化發展的程度，是否能尋找到更好的合作資源，是否能確定本身落地到某個城市的發展意向。「大的方向，還是希望更多的俱樂部，能像eStar一樣有自己城市主場的計劃。」

俱樂部的成長，某種程度上也是KPL不斷進階的縮影。隊伍數量已由最初的12支擴軍到如今的16支，而早期的升降級制度也更替為固定席位制。值得注意的是，聯盟已經確認將階段性地停止增加席位數量，理由是為了「更好的孵化和培養市場」，能夠讓現有俱樂部有時間沉澱，尋找更好的城市資源和商業方式。

說到底，還是為了實現地域化這個大命題。畢竟一個志在長遠的電競賽事品牌中，真正的中堅不是頂層的聯盟，而是腰部的俱樂部——很顯然，相比三年前初建時，如今的KPL已經逐漸顯現了一批初具規模的俱樂部，例如落地武漢的「七朝元老」eStarPro，例如今年的冠亞軍AG超玩會、QGHappy等——某種意義上，他們才是KPL打造出來的核心產品。

除去已經落地的武漢，張易加在接受採訪時還透露，廣州或將成為下一個俱樂部主場城市，因為即將舉行的冬季冠軍杯總決賽，就在這裡開戰。「按照我們的慣例，去尋找下一個合作城市主場時，會先通過前期賽事的鋪墊，給這個城市做一些預熱。」

走過三年的KPL，下一幕正在開啟。
