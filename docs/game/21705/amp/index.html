LOL9.24b版本：元素龍集體削弱，快節奏環境有所減緩

天天要聞
2019-12-18

在12月17日測試服也上線了9.24b版本，如果不出意外的話這回事今年最後一次版本調整了。下次更新就是直接S10賽季進入10.0版本了，不過在這一天的調整內容方面還是比較大的，因為對小龍以及相關內容進行了調整。

之所以會對小龍進行調整，主要是考慮到如今的環境中算是快節奏的情況。而小龍提供的各方面增益毫無疑問是非常大的，前期如果拿多幾條龍增加的屬性可能就已經可以讓對面壓力頗大。為了適當減緩對局節奏，因此對於小龍進行了削弱。

設計師在推特上公布了9.24b改動，其實除了小龍還有兩件裝備得到了加強。但是比起小龍改動，這兩個裝備確實是顯得毫無存在感。鋸齒短匕和夜之鋒刃的攻擊力提高了5點，沒了。小龍的話基本上是全方面削弱了，哦倒也不是說全方面削弱，風龍倒是加強了。加速效果從持續3秒40%提高為持續6秒50%

遠古巨龍的BUFF持續時間降低了30變，從180變為150秒。同樣灼燒傷害也降低了15~45點變成75~225點。一定程度上削弱了擊殺遠古龍一方的線上消耗能力，畢竟有額外的灼燒傷害。

至於水龍的治療量也是降低了，都是為了削弱線上續航能力。已損生命值回復從3~12%降低到2.5%~10%。並且龍魂效果的治療量從180(+0.4ad)(+0.25ap)(0.1額外生命值)和90(0.4最大法力值)且持續3秒變成了，160(+0.36ad)(0.225ap)(0.09額外生命值)和80(0.35最大法力值)持續3秒。僅看數據和加成上我們都可以看得出來擊殺水龍的線上回復都有著明顯的減少。

而火龍的屬性加成也是顯著降低，特別是後期。從5%~20%降低到了4%~16%。越到後期這個屬性加成降低的也越明顯，一定程度上削弱了多火龍一方的增幅。雖然我覺得如果人家都好幾條火龍了也確實基本上是屬性碾壓了……

龍魂效果傷害由90(+0.25ad)(+0.15ap)(0.03額外生命值)降低為80(+0.225ad)(+0.135ap)(0.0275額外生命值)。還需要說嗎？明顯就是降低了這個傷害，防止增幅太多。

土龍的雙抗加成也是降低，同樣也是後期加成更加明顯的減少。雙抗加成從8%~32%降低為6%~24%。比原來要脆一點點，對於坦克和肉輔我覺得差別還好吧。對於ad而言嘛……說實話大概就是多出一個小件暴斃的傷害吧？

龍魂效果提供的護盾值也是降低咯，225(+0.2ad)(+0.15ap)(0.15額外生命值)降低為200(+0.18ad)(+0.135ap)(0.135額外生命值)

達龍雲認為，這一波對小龍的削弱可以說是全方面拖慢當前的對局節奏。只是削弱攻擊加成傷害加成的小龍以外顯然不夠的，如果防禦續航加成沒有削弱也等於給了一個發育起來的機會後期又打不過。因此也是同步進行了一次削弱，總體上而言還算得上是一次平衡性削弱。雖然我覺得ADC依然是該死都會死，區別是撐多久才死。
