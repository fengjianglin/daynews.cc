迷之操作？RNG開除教練，EDG兩員大將離開，阿水開小號暗示去向

天天要聞
2019-12-17

明天就是德杯比賽的日子了，昨天各大俱樂部也是集中爆了一波料，很多網友都表示，比賽前開除教練？首發ADC換人?這操作給我看懵逼了，明年也太亂了。

下面一起去看看昨天究竟發生了什麼。

阿水轉會消息成迷？但是被石錘

關於IG戰隊的Jackylove轉會消息依然沒有官宣，然而明天就是德杯比賽的時間了，按規定今天必須官宣。但是心急的網友早已按奈不住了，開始從各大戰隊韓服賬號的比賽記錄上尋找。

終於有網友在SN戰隊的輔助蛇蛇的排位戰績上，發現蛇蛇在跟weiwei雙排，並且weiwei全是射手位，猜測可能是阿水在用weiwei的號在和蛇蛇磨合。

大概今天我們就要看到，Jackylove加入SN的圖片了。

RNG開除教練補強

就在IG等一眾戰隊，尋找教練無果的時候，RNG卻宣布將教練牛排和Dandy解約了。這頓操作也是把網友都看迷了，結果網友們還沒來得及猜測RNG的新教練是誰的時候，RNG直接官宣了Mata!

RNG官宣Mata轉職教練加入RNG，前皇族戰隊輔助Tabe轉型分析師加入RNG，這下RNG這是準備把小狗的輔助都湊齊了？明年再官宣個Zero加入就真的齊了。

不過憑心而論，Mata雖然現在狀態不行了，但是轉教練可以說問題不大。一般每個戰隊的輔助，都是隊伍的大腦，在局勢分析、決策和大局觀上都是沒問題的，之前也早就有網友猜測Mata退役後會轉型教練，只是沒想到去了RNG，可以說這波轉教練是如魚得水吧。

廠長退役轉教練，iboy加入VG

EDG昨天也是爆了兩個消息，其中一個就是ADCiboy胡顯昭轉會至VG戰隊。這其中其實有很多的原因，一方面是小昭最近表現確實不盡人意，不如替補ADCHope。另一方面，大概也是選手對明年EDG的陣容不太放心，想要選擇一個資源傾斜度更高的戰隊，而明年有kkoma加入的VG戰隊，顯然就是他的最優選擇。

其實對於老玩家來說，另一個消息才是最關鍵的。昨天最大的消息應該就是廠子退役了吧？誰也沒想到廠長說的「會去S10」是選手轉職教練的形式參與。

今年4月5號是廠長最後一次上場比賽，沒想到這就是永遠了。

回顧今年11月份的時候，廠長跟水友打水友賽，這是他最後一次作為選手的身份進行的水友賽，直播過程中粉絲們給明凱說了很多暖心的話，可廠長面對粉絲的冠軍期望，只能猶豫地將「好」和「會」改成了「謝謝」，不想欺騙粉絲又不能明說退役的消息，大概當時心裡也不是滋味吧？

當煙花升起的時刻，那個曾屬於明凱的EDG國王時代不會隨年華逝去，而只會隨年華的飄零中常常記起。26歲的明凱，站在BP台上，深情的目光望過去，都是自己22歲的樣子。他是隊長，他是領袖，他是傳奇，他是野區之王。當他離去，世界又該聚焦與誰？
