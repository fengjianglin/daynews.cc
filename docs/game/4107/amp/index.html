夢幻西遊手游：玩家花2000買號嫌虧本，網友怒罵：光神獸就回本了

天天要聞
2019-12-09

摘要

在夢幻里，不少玩家都會選擇直接在藏寶閣中買賬號，方便安全還剩時間。但總有玩家在買了賬號後不放心，生怕自己買虧了。

在夢幻里，不少玩家都會選擇直接在藏寶閣中買賬號，方便安全還剩時間。但總有玩家在買了賬號後不放心，生怕自己買虧了。日前，有玩家在網易大神中分享自己的買號經過，該玩家在藏寶閣中花了2000買了個115級的號，怕買虧本了，便發文詢問網友，引來不少網友的圍觀，接下來就帶大家一起去看下，該玩家買的這一賬號吧。

首先來看這號上的裝備，武器是100級的湛盧，敏耐雙加帶光輝之甲特技，並且物理傷害達到675，法術傷害達到154，治療強度達到223，打上了11級崑崙玉。光從裝備屬性角度來看的話，這件武器雖然不算很極品，但也非常不錯了，對於一般輔助門派來說也夠用。

鞋子是100級的平步青雲，法術防禦103，速度112，帶有特技凝神決，打上了11級的黑寶石。這件裝備也不錯，雖然離極限屬性還有段距離，但用一用還是沒問題的。號上的其他裝備，沒有什麼亮眼的特效，並且寶石也才10段，就不一一介紹了。

再來看看修鍊，物防、法防修鍊已經點滿了，攻修也點到了22級。寵物的修鍊也不錯，寵物攻修點到了22級，其他修鍊也點到了20級。可以說，就這賬號的修鍊，該玩家這2000元花的就不虧。

最後，對這號上的寵物，遊戲君真的是羨慕到吃檸檬，兩隻神獸在身上，還擔心什麼虧本？並且兩隻神獸的內丹還打上了生死決跟摧心浪。光這兩隻神獸就已經賺到了，玩一段時間大可把號當金幣號賣，怎麼算都不虧。不過，雖然不虧，但這號也並不算強，先不說這號上像樣的裝備沒幾件，就是寵物也沒幾個能打難度的。所以2000買這號雖然不虧，但體驗感並不好。最好的辦法就是玩了一陣後，直接把號賣了再買個號就成了。

圖片來自網易大神，作者我家親愛噠小跟班

就拿這賬號來說，該玩家花2000元秒掉已經是走大運了。可該玩家卻發文詢問網友買的虧不虧，引得不少網友在評論區留言諷刺，”這號虧不虧，你不知道？不知道你買來幹啥？””光兩個神獸就不虧好吧，居然還問虧不虧””虧了2000，趕緊退款吧”。

也不怪網友們發言諷刺，明明賺大了，卻來問虧不虧本，任誰遇到這事都得生氣。那麼大家在買賣賬號的過程中，是否有遇到賺大了的情況呢？不如在評論區留言，與遊戲君分享下自己買賣賬號的經歷吧。
