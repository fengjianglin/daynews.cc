這四個武將就像馬蜂窩，讓人毫無興趣不敢動手，一碰就炸

天天要聞
2019-12-23

大家好，我是一個吃不起飯，眼看就要餓死的實力遊戲解說靈運。今天忍受著飢餓繼續給大家解說中國經典桌游三國殺，各位在遊戲中一定遇到過經典的進退兩難的局面，而接下來要說的這四個武將，就是一度會讓人犯難的存在。

讓人毫無興趣不敢動手，一碰就炸的四個武將

一米張魯

義舍：結束階段，若你沒有”米”，你可以摸兩張牌，然後將兩張牌置於武將牌上，稱為”米”；當你移去最後一張”米”時，你回復1點體力。

眾所周知，張魯作為五斗米教的掌門人，只會沒完沒了的存米，不論是改判，拿牌還是發牌，張魯每用完兩張米，都會恢復一點血量。這就引發了一個世紀性的難題：一米的張魯殺不殺？殺他的話，張魯免費拿一張牌，而且馬上就又恢復一點血量。不殺的話，感覺又太虧了，實實在在的令人感到左右為難。曾記得和張魯單挑過一次，來來回回的存米，根本拿他沒有一點辦法，如果最後不是因為他掛機了，小編真的要摔手機了。

張魯在2v2或者3v3中都有不俗的表現，受技能影響，張魯需要有人打自己或者頻繁改判定，來不斷的刷掉自己身上的米。當每回合張魯都能把自己的米刷掉時，這個張魯是很難被殺掉的。

二權鍾會

覺醒技，準備階段，若”權”的數量大於等於3，你選擇一項：1.回復1點體力；2.摸兩張牌。然後減1點體力上限，獲得”排異”。

鍾會兄作為鍾繇最出色的兒子，深得廣大玩家器重，作為大魏知名不動白，沒有覺醒之前的他可以說是完完全全的白板（可以依靠兩張權存牌，神技！）。不過，覺醒以後得鍾會簡直太棒了，又能補牌，還能給隊友補牌，甚至排異還能殺人（不過，小編從來沒用過排異給對手補過牌，真的危險性太大了）！當我們在場上遇到二權的鐘會的時候，其實內心是糾結的，鬼知道鍾會覺醒以後會發生什麼？一回合摸四張牌（摸兩張，排異補兩張），而且被動還能存牌，這簡直甩那個男人八條街好么！存兩回合牌直接收全場了啊，真不愧是滅了蜀漢的存在。

鍾會作為一個厚積薄髮型的武將，前期是沒有一點優勢的，可以說做忠臣或者做反賊表現都差不多，很一般。如果拿來做主公或者做內奸，你可能會有不一樣的收穫歐！

二田鄧艾

鑿險——覺醒技，準備階段，若”田”的數量大於等於3，你減1點體力上限，然後獲得”急襲”。

不知道為什麼，總感覺三國殺最近鄧艾判定刷紅桃的幾率提高了，鄧艾前三個判定是紅桃的幾率非常高。開局鄧艾連著三個紅桃，然後直接掛機的我見過不少了，難道鄧艾最近和小喬有什麼不為人知的故事？鄧艾和鍾會共同滅掉了蜀漢，所以他倆的技能機制差不多，一個是屯牌，一個是減距離。覺醒之後都是一陣爆發，尤其是鄧艾，直接把一個人偷的乾乾淨淨都是很平常的事情。所以在一局遊戲中，遇到二田的鄧艾，你該怎麼辦呢？

鄧艾在遊戲中是一個能夠適應多種身份的武將，簡單來說就是打什麼位置也沒有出彩的表現，打什麼位置也都還基本可以，因為被動減距離的原因，起碼不會被卡距離，總的來說一般情況，不知道各位看法如何。

最強防禦，二血孫策

魂姿——覺醒技，準備階段，若你的體力值為1，你減1點體力上限，然後獲得”英姿”和”英魂”。

這已經不是單純的一個武將了，這是一個梗！一個所有玩家津津樂道，為之痴迷的男人，因為他要麼二血要麼沒血，不可能有人眼睜睜看著一個一血孫策活蹦亂跳的蹦躂還沒人收拾的。覺醒後的孫策集摸牌拆牌補牌於一身，橫掃天下，血戰八荒，如同天上降魔主，真是人間太歲神！這個孫策，就問大家怎麼樣？
