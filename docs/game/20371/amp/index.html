你可能並不玩這些英雄，但不得不承認它們真的很火爆

天天要聞
2 天 ago

如果要確定一款遊戲的人氣、知名度什麼的，大多數玩家首先想到的就是遊戲的玩家數量、營收情況等數據。可實際上，除了這些非常物質的東西之外，還有其他一些東西可以很好的體現出遊戲的名氣什麼的。今天，舞文就和大家一起去看看那些角色、裝備早已火遍世界各地，但是知道它們出處的人卻並不一定玩該遊戲的超人氣遊戲角色和道具。

Dva——《守望先鋒》

最近海外神秘網站P站的統計數據顯示，《守望先鋒》是海外紳士搜索最多的遊戲，而Dva也是紳士們搜索最多的角色。其實不僅如此，在國內遊戲市場中，雖然因為種種原因，選擇了長期玩《守望先鋒》的玩家並不算多，甚至不少人根本不玩這款遊戲，可他們依舊知道Dva來自《守望先鋒》。因為在直播行業發展越來越好的今天，長相甜美且被很多人看做國民老婆的Dva一直都是很多美女主播、職業coser平時喜歡cos的存在。

蒂法·洛克哈特——《最終幻想》

或許，大多數玩家甚至都不知道她的全名叫做蒂法·洛克哈特，但是卻都知道她叫蒂法，來自《最終幻想》。畢竟，《最終幻想》中的蒂法，本身也是一位非常漂亮的女性角色不說，很多知名的女主播和coser也非常熱衷cos這個角色。

賽麗亞——《地下城與勇士》

作為曾經國內市場中火爆程度最高的遊戲之一，《地下城與勇士》歷經多年依舊有著一批忠實的玩家粉絲存在。而這款遊戲中的初始NPC賽麗亞更是所有玩家每天都要照面N次的存在，不僅如此由於曾經不少玩家將其當做夢中情人的原因，很多不玩這款遊戲的玩家也知道了這一NPC的存在。

提莫隊長——《英雄聯盟》

作為目前全球範圍內火爆程度最高的競技遊戲之一，《英雄聯盟》的每一次大型比賽尤其是S系列比賽都牽動著世界各地眾多玩家粉絲的心。其中那個萌萌噠的提莫，在受到了無數遊戲玩家喜愛的同時，一些不玩這款遊戲的人也從看到他的形象開始就深深的愛上了他。

魯班七號——《王者榮耀》

作為當下國內火爆程度最高的手機遊戲《王者榮耀》粉絲自然不再少數。儘管截止目前，依舊有不少玩家比較抵觸這種搓觸摸屏的遊戲。但是，我們依舊不得不承認，其中最萌也最挨打的角色”魯班七號”熱度同樣非常之高。就國內遊戲市場的熱度來看，”魯班七號”的知名度絲毫不下於《英雄聯盟》中的”提莫隊長”，一些不玩遊戲的玩家注意到他也在情理之中。

貝優妮塔——《獵天使魔女》

《獵天使魔女》這款遊戲雖說也有著不錯的受眾群體，但遊戲的熱度確實只能算是一般。不過，無論有沒有玩過這款遊戲，相信對其中經典角色貝優妮塔的御姐形象都會非常熟悉吧。不少遊戲玩家，在看到遊戲中的御姐形象的時候，基本都會習慣性的將他們與貝優妮塔進行對比，哪怕自己沒玩過《獵天使魔女》。

屠龍刀——《傳奇》

作為金庸武俠小說《倚天屠龍記》中的第一寶刀，很多遊戲中都有出現這件武器，其名號也非常響亮。不過，在提到遊戲中的屠龍寶刀的時候，大多數玩家首先想到的還是《傳奇》。畢竟，作為最早在國內火爆起來的網路遊戲，《傳奇》本身就擁有極高的人氣不說，遊戲中的第一寶刀屠龍的人氣也不要太低，再加上最近幾年各種《傳奇》類型魔性的頁游、手游廣告的出現，更是讓《傳奇》遊戲中的屠龍人氣提升了非常之多。

山姆——《死亡擱淺》

不管大家對《死亡擱淺》的評價如何，自己又到底有沒有去玩這款遊戲，有一點我們不得不承認——小島秀夫依靠《死亡擱淺》有了新的突破的同時遊戲自身也受到了全球玩家的關注。而遊戲中的”快遞員”山姆的受關注程度自然也不低，只是不知道各位關注著山姆大叔的朋友有真正體驗過《死亡擱淺》了嗎？

不知火舞——《拳皇》

作為SNK經典遊戲《餓狼傳說》和《拳皇》中的經典角色，由於在前者當中只是男二號安迪·伯加德的女友，但在後者中是一位常駐女格鬥家，且自身非常漂亮性感的原因，大多數玩家都還是非常喜歡這個女性角色的。再加上如今不少女主播都經常cos這位角色的原因，不少人其中包括一些沒玩過《拳皇》系列遊戲的朋友，都知道《拳皇》中有一位漂亮性感的女性角色名為不知火舞。

其實，在如今的遊戲市場中，還有很多遊戲角色和裝備的熱門程度非常之高，高到了讓沒玩過這些遊戲的玩家都了解到了這些角色和裝備的存在。那麼，各位還知道哪些角色和裝備讓沒玩過遊戲的玩家也都知道了的呢？歡迎大家在評論區中分享。
