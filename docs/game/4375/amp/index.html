懷舊服奧山即將開啟，來聊一聊雷矛要塞的攻堅戰

天天要聞
2019-12-07

摘要

12月11日魔獸官方將在懷舊服開啟奧山與戰歌峽谷2張戰場地圖。玩過奧山PVP的都知道，聯盟在前線的設計相比較於部落有著一定的弱勢，但在防守上聯盟則有著著名的雷矛大橋，這也側面體現了聯盟在均勢中，應當注重防守反擊，有效消耗部落進攻力量與士氣，最終取勝。

12月11日魔獸官方將在懷舊服開啟奧山與戰歌峽谷2張戰場地圖。玩過奧山PVP的都知道，聯盟在前線的設計相比較於部落有著一定的弱勢，但在防守上聯盟則有著著名的雷矛大橋，這也側面體現了聯盟在均勢中，應當注重防守反擊，有效消耗部落進攻力量與士氣，最終取勝。所以部落想要真正一舉擊潰聯盟，大橋的爭奪將是整場戰爭最為重要的關鍵點。

第一， 橋的特殊地形導致進攻方寬度不夠。Pvp經驗豐富的玩家都知道，但凡開闊地形都是非常不利於防守的，進攻方四面八方能突入防守方的輸出集群，控制技能使用得當可以非常輕易的瓦解防守方的站位，最後被分割消滅。然而那座橋，聚攏了進攻方的突進部隊，使得防守方可以非常輕易的對突擊力量實施群控和高效的aoe傷害。防守方的輸出集群可以非常安全的站在橋後的扇面區域，但凡不是有實力上有懸殊對比，往往是聯盟守方佔據絕對優勢，導致大橋守衛戰演變成曠日持久的拉鋸戰。（不過也有非常多的急性子聯盟玩家非常反感大橋守衛戰，他們知道這有可能會拉鋸相當長的時間，還不如一波送了拉倒）

第二， 雷矛墓地的位置很不好，當部落方拿下雷矛墓地以後，會被有組織的聯盟堵在墓地里分割坑殺。因為復活後大家多少會有一些卡頓，身上又沒刷buff，目標也來不及選擇。地面上很可能已經存在聯盟敵方早就布置的陷阱，早就開始釋放的暴風雪等減速aoe，碰到厲害的坑殺隊，你復活一秒鐘過後，就身上掛著各種負面效果，還剩6成血量了，毫無勝算。因此，有組織的奧山隊，往往在部隊充分聚攏並在橋面取得推進優勢之前是不拿雷矛墓地的。

那麼對有組織有指揮的部落團隊來說，其實進攻雷矛的戰術還是非常豐富的。最樸實無華的，斜坡攻勢，鑒於橋面上述的缺點，乾脆放棄作為突入點。有組織的集結在橋下，然後按常規打法從橋下的斜坡往要塞內部整體推進。 這裡的關鍵技巧是集結紀律，集結的地點要求是敵方輸出集群的射程臨界點。讓他們很難受，往前一步就夠得著你，但是他可能會掉下來這種程度就剛剛好。 本方人員必須嚴格遵守不接戰原則，所有的cd技能全部保留，對面有時候手賤撩你兩下絕對不還手，保持集結度，治療負責刷滿血量就好。只等團長一聲令下，40人從下坡整體突進，會讓防守方的橋面地形優勢蕩然無存。進攻方還保有先手、技能配合以及陣型等優勢，反客為主，場面非常熱血。

整體來說，奧山的精彩之處就在於各種策略與反制，拼的是指揮官的大局分析雙方人員的執行力，而不是裝備碾壓，這也是為什麼在那個年代，魔獸世界的戰場PVP能夠一直玩家認同。如今魔獸世界戰場即將開啟，而新服也早已在前幾天上線，想必就是為了那些想要加入魔獸世界展現一番作為的新人或老友吧。
