王者榮耀：新皮膚和聖誕福利下周上線，玄雍新英雄即將解鎖

天天要聞
2019-12-19

王者峽谷即將迎來聖誕系列福利，其中多款聖誕局內表現開放獲取，比如明日（20號）零點開啟收集聖誕拐杖糖兌換幸福聖誕播報的活動，同時還上架了加入聖誕狂歡相框的600金幣頭像框寶箱，喜歡聖誕主題的小夥伴記得做任務獲得，另外史詩皮膚7日體驗卡禮包同樣在零點上架，本期加入了呂布末日機甲、蔡文姬舞動綠茵、李元芳逐浪之夏、韓信白龍吟、諸葛亮黃金分割率和程咬金的功夫廚神。

附上幸福聖誕播報局內展示（加入水晶球元素）：

另外下周版本更新後，阿軻節奏熱浪、妲己魅力維加斯、韓信街頭霸王、大喬和孫策貓狗日記皮膚的聖誕星元部件將再次上架，其中節奏熱浪和貓狗日記皮膚的武器定價300點券，韓信和妲己的武器需要90紫星幣兌換，其它聖誕部件則需要消耗80點券獲得，除此之外SNK英雄的首款皮膚——娜可露露晚螢也會在下周上架，該皮膚定位史詩，首周710點券，新皮膚加入了鳶形燈籠、紙傘和螢火蟲等元素，技能特效絢麗，可以說是一款非常良心的史詩級皮膚，而另外兩位SNK英雄的新皮膚還要等到明年，預計先上橘右京的，不知火舞的最後登場。

當然娜可露露皮膚只是前菜，春節版本王者峽谷還將上線多款全新皮膚，比如已經確定的賽季皮膚、兩款戰令皮膚、五嶽皮膚和貂蟬FMVP皮膚，不知大家更期待哪一款呢？而新英雄方面，玩家對於玄雍版本還有哪些英雄登場還是頗為關注的，為了滿足大家的好奇心，官方開啟了玄雍英雄解鎖站，並預告了下一位解鎖的英雄是女性角色！

目前已經曝光的女英雄只有曜的姐姐「鏡」，因此若無意外下一位新英雄便是操作難度等級最高的「鏡」，小編還是很期待這位加入玄雍陣營的「鏡」其技能機制如何，而蒙恬和王翦的上線估計還要等一段時間了！
