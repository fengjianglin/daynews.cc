《神武3》電腦版：6發烽火誰擋得住？全明星三龍宮不敵大天策

天天要聞
2019-12-19

今天，胖熊給大家帶來的是全明星排位賽中130-150級分組的一場戰鬥。對陣雙方分別是來自紫氣東來的三弟是狗戰隊和揚名立萬的揚名立萬戰隊。

雖說龍宮炸藍流有一定削弱，但三弟是狗戰隊的陣容是3龍宮1天魔1普陀，走的是全法傷套路，而且也可以順帶炸炸藍，打消耗戰。

而反觀揚名立萬戰隊這邊，則是以天策為主的1天策1龍宮1佛門1天魔1五庄的常規陣容。

開局方面，觀戰方揚名立萬用的是龍陣，而對戰方則是虎陣，並沒什麼非常規舉動的出現。而首發寶寶方面，對戰方帶的是清一色的速攻蚩尤，而觀戰方則是血攻劍氣和單攻寶寶混搭的五龍首發。

對戰方天魔拿到1速，2速3速由觀戰方的天魔和五庄拿到，但是1速和2速在第一輪封印中都失手了，只有觀戰方的五庄封到了對面虎頭龍宮，不然第一回合要吃三個龍宮帶普陀法傷加成的疾風驟雨的話，還真可能吃不消。到第一回合結束時，雙方勢均力敵，寶寶數量對戰方略勝一籌，但是觀戰方的血量還算健康。

第二回合，雙方都忙於出寶寶來提高隊伍的整體狀態，並沒出現什麼大問題。而且在出寶寶方面，雙方不約而同選擇了處隱攻寶寶，來規避群秒傷害。

第三回合，對戰方龍宮開啟龍行八荒，配合隊友疾風驟雨的群秒，給觀戰方血線造成不小的壓力。

第四回合，對戰方判斷觀戰方憤怒將滿，使用了對觀戰方佛門使用了綿里藏刀，但可惜的是，佛門並沒使用特技，而是由五庄使用出了天降甘霖，回復了一波氣血，緩解了血線壓力。

而由於這波氣血回復，導致觀戰方的寶寶站場能力提升，減少了拉寶寶的回合消耗，而變相增加了對戰方三龍宮的群秒壓力。

第五回合，對戰方龍宮在被封和寶寶數量不全的情況下，強行使用三個烽火連城點殺觀戰方滿血五庄，企圖利用點殺來扭轉局勢。

但其實，這樣點殺後對全局的影響並不大。對戰方一直解決不掉觀戰方場上的寶寶，而自己的寶寶數量又被頻頻消耗，單個普陀的回復並不能帶起隊伍整體的血量，而且還總是被五庄和天魔封印，這讓3龍宮被迫陷入被動的局面。9

第八回合，觀戰方五庄開出了天降，還有天策開出了天地護法這兩個大特技，將全隊狀態提升至近乎滿血的健康狀態。

雖然在這之中觀戰方的天策好像並沒做出什麼大貢獻，但是在清寶寶和保持血線方面，天策一直將自己的輸出狀態保持得很好，沒有給對面3龍宮點殺或者順路帶走的機會。

第十五回合，觀戰方天策強勢點殺對面一龍宮並點殺成功，打開了局勢。

第十七回合，對戰方三龍宮一齊發力，用恐怖的群秒給觀戰方帶來了不小的麻煩。但是觀戰方依靠人手一個天降，還有天策的天地護法，硬抗對戰方三龍宮的群秒輸出。

就在這樣不斷的拉扯中，對戰方由於開局寶寶損失過多和後期特技能力不如觀戰方的三輔助，即便頻頻使用隱身狀態的技能尋找突破口，使用了涅槃，但是無奈對戰方這邊全線準備充分，並沒能找到翻盤的機會。

在負隅頑抗了近乎20回合之後，對戰方的三龍宮陣容，於第50回合敗下陣來，輸給了傳統的觀戰方陣容，而這也證明了天策在150級依舊有強大的單體輸出和靈活多變的輸出能力。

總的來說，在一系列的更新改版後，萬獸的點殺能力被大幅削弱，而這也變相給予了天策崛起的機會。在一鼓作氣和運籌帷幄等技能的加持下，天策在全明星篇排位賽或者服戰都有了不少的出場機會。而至於即將到來的12月21-22日的山東青島全明星線下賽，大天策能否出戰，大家又怎麼看呢？
