2018年7月份，新門派花果山強勢登場，作為3大新門派中的最後一個壓軸之作，可謂是萬眾矚目。根據2狗的講述，當時全服第一物理系麥兜，240萬購買第一扇子的原因，就是為了玩新門派花果山！

花果山引入了全新的戰鬥機制——如意神通，簡單來說就是擁有8個如意神通技能，但是每回合隨機刷3個。記得在去年的門派發布會上，官方對外的宣傳基調就是這個門派會非常強，畢竟他的師父是孫悟空。

對於這個門派，幾次門派修改都是各種加強，尤其是在2019年10月份的門派大改中，花果山的門派技能、神器，得到了全方位的增加。

西柵老街的指揮封狼，近期將花果山作為了主力物理，160聯的比賽中，在團隊遭遇變故的情況下，他依靠「花果山+咒師普陀」的陣容，拿下了季軍的好成績。而在2019群雄逐鹿年度賽的4進2比賽中，二狗同樣是用花果山作為主力物理，戰勝了無比強大的珍寶閣，花果山不需要血線和超強的點殺能力，在這場比賽中也表現得淋漓盡致！

而在近期的劍會PK中，西柵老街封狼依靠花果山+神木林的點殺陣容，同樣效果也非常不錯。無論是打人還是打寶寶，輸出過萬都是家常便飯。

而且花果山這個門派除了當頭一棒之外，其實還有很多事情可以做，殺威鐵棒攻擊時候不會觸發目標金甲的效果，潑天亂棒不能被保護，而且會濺射，在清場的時候同樣非常好用！

西柵老街封狼對於花果山的使用，非常注重抗性，防禦調到了2900多，傷害只需要保持在3400以上，在實戰中的輸出數據同樣是非常的恐怖。

對於花果山這個門派，很多人認為是順風超級猛，逆風啥也不是，不過在封狼眼中並不是這樣的。他認為花果山更像1個「加強版的力五庄」，而且是不需要氣血上限的力五庄，起來就能馬上輸出！

國標點殺就配合傳統法系，雙封就配合女魃墓，花果山也是非常百搭的物理系。用花果山最為致命的一點，就是刷不出你想要的技能，這一點經脈中的「貪天」就非常重要了。要有信念，一直當你就一直有！

花果山這個新門派出來已經有1年多，這一次真的要迎來他的強勢崛起了。風水輪流轉，明天到我家，未來的「物理1哥」花果山要定了！

我是如風，我為自己代言，點點關注不迷路，聊夢幻，聊服戰，小編是最專業的，內容全部原創！
