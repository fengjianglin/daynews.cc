未完成的克魯蘇式遊戲——《冥河：舊日支配者之治》評測

2019-12-15 3,726條評論 4,705次閱讀 2963人點贊

作品完成度低，建議暫時觀望。

嘰咪嘰咪，用愛發電。

《冥河：舊日支配者之治》是一款克蘇魯風格的RPG遊戲。遊戲氛圍怪誕恐怖，著實是玩家San值殺手。由於筆者從未接觸過克蘇魯題材的遊戲，故對本作的評價將基於傳統的RPG遊戲。簡單的試玩後，我認為遊戲的優劣均十分明顯。下面將對該遊戲進行詳細的介紹。

背景

背景設定在霍華德·菲利普斯·洛夫克拉夫特的世界中，也就是克蘇魯的世界中。傳說克蘇魯全稱為「偉大之克蘇魯(Great Cthulhu)」，沉睡之神，拉萊耶之主，德雷斯構建的元素論中象徵「水」的存在之一，形象為類似章魚頭、人身，背上有蝙蝠翅膀的巨大人形生物。傳言克蘇魯偶爾與人類進行精神接觸。收精神接觸的人類將會承受或大或小的精神壓力，有很多人就是因為這樣的接觸而發瘋。克蘇魯對人類的威脅無疑是巨大的，據說一旦星位正確，克蘇魯就會蘇醒並與拉萊耶城一起浮出海面，人類就會滅亡。而且據《克蘇魯的召喚》描述，克蘇魯粘性的身體不是由地球上的物質構成的，而且他的身體結構雖然可以破壞，但卻無法毀滅。另外還有說法稱克蘇魯不但是舊日支配者，同時也是史前一個外來文明種族的大首領。

遊戲就是基於這種世界而展開的，在一開始，主角就在夢境里受到了精神的衝擊，而主角所苟活的世界，已經難以分清夢境與現實的分界了。

遊戲性

就如很多開放性遊戲宣稱的那樣，本作從「捏臉」開始就賦予玩家自由。雖然創造角色自由度不算高，但也是有模有樣的了。角色的設定有幾個部分：性別、年齡、職業、信仰體系、名字頭像和屬性點。

對角色屬性點影響大的有年齡，職業兩項。而信仰則直接決定了玩家角色的San值。說到San值，這是克蘇魯題材中必備的機制之一。在本作中，玩家需要時刻注意San值，一旦San值歸零，也就意味著你打出了GG。

遊戲內的通貨是香煙，而使用其它的一些藥品都會上癮，在末世中，唯有這些藥品可以暫緩精神的壓力，這也是遊戲的特色之一。這樣的設定就決定了玩家的每一次行動，使用物品，休息等都會影響角色的精神狀態，精神狀態又可以影響角色的生存能力。簡而言之，慎行。

遊戲的戰鬥環節很有挑戰，戰鬥不是要想著怎麼剛，而是要想著怎麼跑。殺人會累積恐懼值，掉San，掉血，而得到的回報卻不成正比，這樣的話，誰還會去頭鐵無雙呢？光從這一點，我就感覺戰鬥機制設計的很失敗。也有可能是製作者為增加遊戲難度，或者是不鼓勵玩家無雙，所以戰鬥才會設計的這麼的噁心人。

戰鬥模式是回合制模式，玩家的移動，攻擊，開包等行為都需要消耗行動點。反正精髓在於算計，而算計的最優解就是趕快逃命。尤其要強調的一點是，如果玩家只是孤身一人而不招募任何對有的話，只會狗帶的更快。

缺陷

劇情

首先是劇情。劇情算是這個遊戲最大的敗筆了。玩家初看還覺得遊戲劇情打磨的很是精良，懸念此起彼伏，真相就在前方。但是這種感覺是經不起時間的推移的。玩到後期，就會發現劇情並不完整，很多前面埋下的伏筆到了後來根本沒有呼應。甚至主線劇情也是突然的戛然而止。我就想問，這個遊戲真的做完了嗎？

BUG

BUG滿天飛，而且有很多是導致程序崩潰的惡性BUG，加上遊戲莫名其妙的自動存檔機制，一旦玩家遇到BUG而沒有手動存檔，你就準備著重新來吧。

漢化

我懷疑漢化是由谷歌翻譯完成的，很多很明顯的錯誤一看就是機翻，事實雖不是如此但也差不多了。遊戲糟糕的漢化肯定是中文玩家的噩夢，這種遊戲脫離了文字就不可能玩的下去，糟糕的漢化讓人沒有讀下去的耐心。好在製作者已經發現了這個問題並保證將進行糾正，讓玩家看到了點希望。

總結

遊戲的口碑盡數毀在了劇情不完善以及BUG之上，如果拋去這幾點的話，我必須承認遊戲的質量還是可以的。但是此二者已經是一部作品的硬傷。如果開發者可以注重對這幾個問題的彌補上的話，我相信遊戲還是比較成功的。

最後，本作尚不推薦入手，如果有感興趣的玩家可以暫時觀望。
