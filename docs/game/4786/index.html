長居ban位原因，由於瑤的技能機制問題，在前中期可以說是很無解了，有控制有視野有護盾。一技能自動追蹤，二技能無需手動操作，且都是範圍傷害可以不斷觸發冰杖被動，最可氣的是你技能吟唱到一半了不知道在哪彈出個瑤的一技能給你打斷，這可是非常氣人的。其實以上的都不算什麼，有的時候在排位中可以遇到打野自帶瑤這個裝備cp，這可是無形中的虐單身汪呀！

另外給大家附上瑤附身在隱身英雄下的狀態：「附身隱身狀態的蘭陵王、鬼谷子和阿軻會暴露視野；附身位移距離過遠的英雄會被甩下來（會返還50%CD）。」
