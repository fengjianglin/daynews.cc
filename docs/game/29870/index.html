山寨Switch現身國外：完美解決搖桿漂移，內置800款遊戲

2019-12-23 605條評論 1,693次閱讀 648人點贊

Switch最近兩年在全球大火讓任天堂的股價瘋狂飆升，在這樣的背景下任天堂也推出了Switch lite，並且與騰訊一起發售了國行版本。由此種種都足以見得Switch的影響力！但是影響大就難免會有仿品：曾經假iphone橫行華強北，仿冒的假鞋已經成為日常。最近在國外就出現了山寨Switch：Nanica Smitch。

不得不說這名字取得非常有意思，幾個反向字母操作下來，看起來非常正式。令人驚訝的是，這款名為Nanica Smitch的機器在哥倫比亞銷售。在外觀方面，無論是外形還是手柄，都與正版十分相似，對於不熟悉機器的普通人來說，簡直可以做到以假亂真。

不僅外形相似，機器甚至可以做到拆卸手柄！並且完美解決了搖桿漂移問題：去除搖桿。任天堂學會了嗎？「解決不掉搖桿問題，就解決掉搖桿」

開機之後，這個盜版機器內部包含了800多款遊戲，包含《炸彈人》《雙截龍》等等。根據其包裝盒右下角的提示，應該是採用了灌輸盜版遊戲ROM的做法。

價格方面，國行Siwtch價格算是良心，2099。但在山寨Switch面前依然是個弟弟：這款機器售價50美元，約合人民幣350元。對於哥倫比亞這樣經濟水平較為一般的國家來說，還是非常具有吸引力的

據說這款山寨遊戲機在國外的銷量還不錯，因為臨近聖誕的緣故，恰好戳中了消費者為孩子購買禮物的熱潮。對於中國玩家而言，這樣的山寨產品一出現，腦袋裡便會自然而然的出現三個大字「華強北」。早年間，從Gameboy到PSP、3DS等等，都在華強北出現過山寨的身影，如今這國外的山寨Switch難免讓人懷疑是不是華強北製造！

對於這樣一款竊取了外觀、遊戲、硬體的全方位侵權產品，任天堂方面還沒有任何回應，不知道最強法務部何時才會出戰呢？
