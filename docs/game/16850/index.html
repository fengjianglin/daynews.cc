遊戲製作人：坂口博信對於遊戲行業的《最終幻想》

2019-12-16 195條評論 1,117次閱讀 202人點贊

幾乎和所有遊戲從業者一樣，坂口博信也曾決定退出令他傷心的遊戲行業，但在真正結束痛苦之前，他決定全力製作一款屬於自己的遊戲，用以紀念自己的夢想。因為這是他對遊戲最後的幻想，於是他把這個遊戲命名為《Final Fantasy》。

失敗還是失敗

1983年，電友社的社長宮本雅史面試坂口博信，兩人惺惺相惜。於是當宮本雅史在組建遊戲開發團隊的時候，年僅20歲的坂口博信便受邀加入了史克威爾，一躍成為了史克威爾的靈魂人物。

就如眾多年輕的創業人士一樣，坂口博信等人幾乎沒有任何製作商業遊戲的經驗，有的只是一番對於遊戲的熱情。於是他們的第一部試水作品《死亡陷阱》，理所應當的被丟入了歷史的長河之中，消失不見，甚至連水花都沒有濺起。

1986年他們決定蹭一波主機RPG的熱度，於是這部由坂口博信親自撰寫劇本並出任總指導的《國王騎士》，在跨多平台的情況下，總銷量達到了20萬，算是一個挺不錯的成績。 也在同年，史克威爾開始正式的獨立運營。

較為成功的遊戲，高層的信任，一切似乎都走上了正軌。

但讓人沒有想到的是，由於運營管理層的錯誤決定，史克威爾迎來了第一次危機。

史克威爾將所有的資源壓在了FC的周邊設備DS上面，DS系統的物理層面的設計存在多種致命的缺陷，於是在經歷了短暫的火爆後就迅速降溫。

哪怕史克威爾接連推出《深黑地牢》，《水晶之龍》等品質不錯的遊戲，但遊戲銷量卻隨著硬體系統的原因每況愈下。 以至於在1987年初，成立不到四年，獨立不到一年的史克威爾連最基本的運營也難以為繼了。

連續不斷的挫敗，使得坂口博信對自己的能力產生了嚴重的懷疑，他甚至決定金盆洗手，退出遊戲行業。但在結束這所有的痛苦之前，他決定將自己對遊戲的所有幻想，放在他最後的這部作品裡，用以紀念他的青春，以及在史克威爾的這段美好時光。

他將這款遊戲命名為《最終幻想》。

《最終幻想》

《最終幻想》初代以及二代的銷量雖然沒有超過百萬份，但卻足以救史克威爾於水火之中。 真正意義上《最終幻想》，就是《最終幻想》第三代，它不僅相比於初代與二代有了很大的進步，而且叫好又叫座，在1990年發售後，它的總銷量達到了可怕的300萬份。 《最終幻想3》宣告了自己的王者降臨，至此，最終幻想系列登上了歷史的舞台，開啟了屬於自己的時代狂潮。

不過，真正將最終幻想系列推向神壇的還是《最終幻想7》。

1997年，日本遊戲產業進入盛世，《最終幻想7》做為坂口博信最具有代表性的作品，於1月31日正式發售。 深刻的人物設定，撼人心靈的故事情節，以及業界最為優秀的遊戲畫面，使得FF7獲得了空前的成功，總銷量達900多萬份，直指千萬大關，創造了遊戲史上絕對的奇蹟。

而在此之後的FF系列中，史克威爾也從未讓人失望，哪怕是銷量最低的FF9也有高達530萬份的銷量，足見其受歡迎第程度。

在2000年，坂口博信被選入AIAS名人堂，成為繼宮本茂之後第二位進入該名人堂的製作人。

幻想的破滅

進入新世紀之後，功成名就的坂口博信沒有停止自己的幻想。 在他幻想的未來世界裡，《最終幻想》電影能夠代替遊戲，將自己的才華與美好的故事傳達給世界上更多的人；每個人都能在《最終幻想》遊戲中能找到歸宿，共同享受這美好的世界。

這一次，他的幻想破滅了。

不僅耗資1.5億美元的電影《最終幻想：靈魂深處》只收回了3000萬美元的票房，而且《最終幻想11》用了長達8個月的時間，註冊用戶才達到20萬，遠遠低於他們預期。這幾波操作，讓史克威爾一夜回到了15年前，再次陷入了嚴重的財政危機之中。

不過，這一次，坂口博信沒有辦法再力挽狂瀾了。 高層經營者們對其連續的錯誤決斷進行了清算，勞苦功高的坂口博信以退休的體面形式被解除了所有的職務。

值得一提的是，直到今日，《最終幻想14》的全球冒險人數已達1400萬，成為了史克威爾最穩定的收入來源之一。 坂口博信幻想的網路遊戲世界最終成為了現實。

最後的幻想

離開史克威爾之後，坂口博信創立了Mistwalker（霧行者），名字來源於他女兒的一個夢。

2011年，霧行者於Wii發布《最後的故事》，銷量達到了60萬份，對於一個獨立遊戲公司，對於平台獨佔遊戲而言，這不可謂不多了。

作為一個主打RPG，主打故事的遊戲公司，他們一直穿行在霧中，等待著下一個遊戲界的巨浪。

彼時，他們必將率先突破迷霧，再次站在世界之巔。

「能夠讓自己感動的遊戲，才是真正出色的遊戲。」——坂口博信

或許你已經對中國的遊戲行業失望透頂，理想與現實的鴻溝讓你感受到絕望。

我不會勸阻你離開，我只是覺得，在你真正離開前，再嘗試最後一次吧。 拉上你志同道合的小夥伴或者哪怕只有你一個人，做出屬於你自己的「Final Fantasy」，為自己的青春，為自己的夢想，為自己的瘋狂，畫上一個圓滿的句號。 加油！

願你們享受遊戲，熱愛生活，這裡是第九藝術社，我是荷包蛋。
