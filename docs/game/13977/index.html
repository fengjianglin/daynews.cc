黃金大獎賽S8分組搞事情？流量隊一個都跑不掉，全圖都是跳點衝突

2019-12-12 1,559條評論 3,761次閱讀 2613人點贊

絕地求生微博杯S3剛剛結束，由鬥魚主辦的黃金大獎賽強勢接棒，不過這一次的賽制和分組都引起了網友的熱議，本次比賽採用了全新的”瑞士賽制”，分組首次採用人為分組，邀請了PGC成績最好的4AM和VC來分組，在看熱鬧不嫌事大的情況下，這次的分組就非常激烈，比較知名流量隊伍一個都沒能跑掉！

P城大戰：SMG和RNG

其實從微博杯開始，觀眾們就在討論這個事情，在微博杯之前的訓練賽，RNG和SMG倒是天天打，但是在正式比賽時，這兩支戰隊居然做到了完美的”和睦相處”、”共享P城”，這一次，這兩支戰隊再次分在了一組，不知道是”友誼長存”還是”反目成仇”？

橋頭大戰：IFTY和OMG等三隊

眾所周知，IFTY和OMG一直在爭西橋頭，微博杯就一直在落地CSGO了，AMY也是西橋頭的隊伍，AMY已經被收購了，現在叫TITAN，但只不過是變了個名字而已，他們的打法還是不會變的，這三支隊伍也被分在了同一塊兒！

G港大戰：天霸和VG

眾所周知，天霸和VG兩支戰隊都是跳G港的隊伍，不過一個是下城區，一個是集裝箱，但是在跳傘和轉移的時候，兩支隊伍是很有可能會撞到一起的，這也算是節奏上的衝突，而且VG和天霸的正面能力都很強，所以很有看點。

學校之爭：4AM和CBT

本以為這次4AM扮演一個分組角色就會幸免於難，但是在最後關頭，VC戰隊用一個”調換符”更換了一個隊伍，將逍遙劃分到B組的CBT划到了A組，眾所周知，CBT這支戰隊的跳點、轉移路線、打法風格，跟4AM戰隊非常相似，所以在轉移上，4AM會非常棘手，花園、米奇妙妙屋可能都會旁落，而旁邊還有一個萬年老鄰居：QM。4AM這次的日子也不好過！

除了劉大大舉的這幾個例子，還有很多隊伍都有跳點上的衝突，比如SSS和YTG在沙漠電廠跳點衝突，EDG和LGD在電廠跳點衝突，總之，這次比賽的小組賽第一輪，全圖都是跳點衝突，考驗的就是每一支戰隊臨場應變的能力啦！

當然，這樣的衝突也不是持續的，這就要說到本次比賽的瑞士賽制：小組賽一共分為三輪，第一輪就是AB組，第二輪就是AB組的前八名分在C組，後八名分在D組；第三輪就是CD組的前八名分在E組，後八名分在F組。

更秀的是，這次比賽的小組賽積分將計入總決賽，對於能夠參加決賽的隊伍來說，就是36場的超長賽制比賽了！不知道各位小夥伴是怎麼看這次比賽的賽制和分組的呢？
