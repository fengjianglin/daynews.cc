樂高無限正式服今日更新，奇幻森林模組、休雪世界歡樂游等更新！一起開啟樂高奇幻之旅吧！

更新時間

12月12日

=奇幻森林模組=

最神奇的寶藏力量,往往蘊藏在危險深處,無畏艱險的勇士們,出動吧!傳說有一片魔幻大陸,盤踞著強大的BOSS- — -樹精、牙猿、沙蛇、熔岩龍,秘法師、金屬騎士,戰勝他們,就可以獲得強大的能量和物料,製造出珍稀裝備道具,還有叢林摩托、熔岩龍、金屬戰車等炫酷載！

=休雪世界歡樂游=

第二期賽季活動開啟!「冰雪 世界歡樂游」安排上!在這個冰天雪地的12月給大家送溫(bao)暖(bei) ! 獎勵不但有豐厚的金幣、鑽石,還有漂亮的裝扮散件綠條紋衛衣,中分波浪髮型，「冰雪大狂歡」定製專屬稱號,冰雪傢具禮包，還有「野生攝影師」皮膚喔！

=自定義UI升級=

1、可以在遊戲大廳點擊設置直接調整鍵位布局啦！

2、增加固定搖桿模式和方向鍵模式,更精確瞄準你的目標！

=新增圖紙=

傢具店、小洋房、觀景台…琳琅滿目的圖紙商店又上新啦!小鎮系列再添新貨,各式各樣獨具匠心的建築樓房將為你呈現煥然一新的生存模式，快搭建起來吧,找一處依山傍水的地方打造你的溫馨浪漫小鎮,邂逅奇遇時光！

=新增人仔皮膚=

復古太空人來啦!他來自遙遠太空，氣質與眾不同,身著鎧甲和失重披風,可別被他迷倒咯。穿上這套皮膚,當你被攻擊時還能增加移動速度,逃跑技能滿分!

=新增裝扮=

俏皮可愛的雙馬尾髮型上新，還有帥氣瀟洒的機車夾克，來個混搭范兒怎麼樣?誰還不是個可愛帥氣的小甜心公舉呢？

=新增多種形式分享功能=

花式分享模式開啟,去分享你的成就、遊玩數據、榜單、裝扮吧!分享的圖片樣式還不-樣喔~還可以看到專屬點評!靈魂造型師快去搭配你的穿搭,遊玩達人快去打卡你的成就排名!曬出你的個人專屬海報!
