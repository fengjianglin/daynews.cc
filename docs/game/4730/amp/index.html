夢幻西遊：主播抗揍展示氪金玩家賬號，行囊隨便點開都值一輛寶馬

天天要聞
2019-12-05

摘要

夢幻西遊是一款休閑養成遊戲，遊戲只分兩種玩家：氪金玩家和非氪金玩家。兩種玩家雖然同時生存在一個空間，但是遊戲體驗截然不同。CC平台&quot;夢幻一哥&quot;抗揍，原來就只是一名普通的北漂IT男而已，他曾經玩夢幻西遊就屬於非氪金玩家。

夢幻西遊是一款休閑養成遊戲，遊戲只分兩種玩家：氪金玩家和非氪金玩家。兩種玩家雖然同時生存在一個空間，但是遊戲體驗截然不同。CC平台”夢幻一哥”抗揍，原來就只是一名普通的北漂IT男而已，他曾經玩夢幻西遊就屬於非氪金玩家。現在成為人氣主播後，在遊戲中的花費也不是特別大，用他的話說：自己離氪金玩家還遠著呢。他靠夢幻遊戲賺錢，大佬靠花錢玩夢幻西遊。

在直播間有水友起鬨，要抗揍給大家展示一下什麼是真正的大佬。於是抗揍從粉絲群中找到了一個隱藏在粉絲中的RMB玩家。之前大家見過隨便上一個水友號就是第一殺人方寸，這次特別找來的RMB玩家，究竟土豪到何種程度呢？

我們來看看土豪的背包行囊。打開行囊進入眼帘的就是10級的星輝石，由於抗揍都沒見過10級星輝石，只能告訴玩家，9級的星輝石市場價是8888，10級星輝石需要3個9級星輝石合成。這樣的星輝石，在背包里有5個。大約價格為兩萬五。可是抗揍想得太簡單了，你以為背包里就5個10級星輝石這麼簡單？

突然看到一個11級的星輝石，直播氣氛瞬間尷尬了起來，抗揍愣了一會才說，這個8萬5，因為10級到11級是有幾率失敗的。可見土豪為了合這個星輝石，想必沒有少花錢。以至於後面看的15級瑪瑙都顯得不值一錢，區區1萬4。

其他的就是寶寶裝備的零散件了，從70級到100級的都有，看樣子土豪只是單純有收集癖，有好的裝備也不管現在要不要用，先買下來再說。最不值錢的應該是這個94級的縛龍筋，大概只能賣4萬塊錢。

抗揍直言，看低級獸決的時候從來都沒有顫抖過。這有這一本低級獸決，讓他感覺心頭一顫。RMB玩家用來墊書的獸決在他看來都是高級獸決。最後，抗揍給直播間的觀眾做計算，護腕之類的裝備加起來25萬，雜貨加起來15萬。RMB玩家背包打開就是一輛寶馬。

不過對RMB玩家而言，這40萬也只是平時的零花錢。對於平民玩家而言，恐怕這價格，連他們眼中寶馬的首付都付不起！人與人之間的差距，就是這麼巨大，且枯燥。
