DNF：歡樂的100級史詩！布甲出場自帶BGM，元素師變DJ了

2019-12-21 839條評論 3,921次閱讀 2416人點贊

如果稍微了解到100級裝備的話，應該清楚，以後的DNF想計算裝備提升率太難了，屬性不再是單一的增傷，有反傷、減傷、手搓、極限CD等，防具也不再單一的5件套，可以2+3搭配，中間還有史詩改造和神話裝備，就連屬性觸發後都是隨機，目前100級史詩還在翻譯，但已經出現一套非常有趣的防具，元素師表示：真正的專屬防具，燈光師？以後請叫我DJ！隊友在輸出，我在打碟加傷害。

歡樂的100級史詩！布甲出場自帶BGM，唱歌還給隊友屬性

這是100級的布甲防具，屬性非常歡樂，套裝叫「天上的舞會」，雖然屬性看似是6件為一套，但實際上有一件神話裝備，完整的屬性依舊是5件就能觸發，但歡樂的屬性是布甲自帶！名字都叫舞會了，怎麼會缺了音樂呢？LOL有DJ琴女，DNF以後就有DJ元素師了。

因為這套布甲史詩登場會自帶BGM，觸發效果是通過「旋律之力」這個屬性，進入地下城就有100點基礎值，釋放1個技能增加5點，旋律之力如果達到150點，就會觸發第一個團隊BUFF，增加隊友10%的三速，到300點觸發第二個BUFF，減傷、雙暴、回血！如果旋律之力MAX了，那麼這套布甲史詩就會唱歌了……

不過旋律之力如果降低到200以下，那麼歌曲就會停止了，也就是說，元素師就算傷害低，技能釋放也別停！五彩斑斕的技能之下，還有悅耳的背景音樂，真正的DJ職業！這肯定是燈光師這個外號給的靈感。

沒想到吧，史詩套的屬性已經這麼歡樂了，熾天使的唱歌只是技能名字，而100級布甲可是實打實的真唱歌，雖然給隊友加的屬性不是太強，但這麼歡樂的史詩，連戰法玩家都搶著刷了。

可惜這套布甲的套裝屬性有點低，但單件屬性還未翻譯好，如果這套「天上的舞會」單件屬性略好的話就能接受了，而100級史詩防具有35套之多！帶神話裝備的就有20套，屬性種類很多，不再是超界·史詩那麼單一了，耳環也是如此！見過會分身的史詩嗎？100級耳環屬性，被攻擊時會觸發一個20S的分身，後跳後可移動到分身的位置！

總結，目前曝光的100級史詩屬性最歡樂的無疑是這套「天上的舞會」，為元素師量身打造，燈光師升級成DJ師，如果BGM能自選的話……希洛克Raid開啟點歌模式！
