女神異聞錄5：體驗JRPG遊戲暢快戰鬥，感受小人物的勇敢成長之路

2019-12-25 146條評論 1,294次閱讀 341人點贊

喜愛日式RPG的玩家，肯定對《女神轉生》、《永遠的伊蘇》、《勇者斗惡龍》和《最終幻想》並不陌生，作為90年代日本國民四大RPG遊戲，它們在眾多玩家心目中的地位是難以撼動的。《女神轉生》系列自誕生起便擁有了最好的JPRG遊戲之一的稱號，但是商業表現上卻不盡如人意。操作難度大、部分內容來源於宗教，惡魔名字與宗教設定相同，種種原因疊加在一起使得該作未能在歐美市場引起巨大反響。作為《女神轉生》的衍生IP，《女神異聞錄》吸取了之前的教訓，不僅對遊戲難度進行調整，同時也對針對歐美市場對遊戲的宗教元素進行了修改，經過大刀闊斧的改動之後，《女神異聞錄》系列迅速走紅，而《女神異聞錄5》更是被稱為JRPG新的巔峰。

《女神異聞錄5》是由日本著名遊戲製作公司ATLUS所推出的一款角色扮演類遊戲，憑藉極具深意的劇情故事、潮到「爆炸」的整體畫面設計與酣暢淋漓的戰鬥體驗，該作也被譽為近些年作為火熱的JRPG遊戲。ATLUS是一家成立於1986年的遊戲公司，他的很多作品都堪稱為藝術品，《女神轉生》在題材方面大膽創新，它憑藉著獨特的個性與自由的構想成為RPG歷史上一處獨特的風景線。雖然並沒有像《最終幻想》一樣贏得大量玩家的注意，但是卻為ATLUS建立起了自己的風格，後續的《女神異聞錄》、《凱瑟琳》均取得了不錯的成績。

歷時七年《女神異聞錄5》終於在2016年成功問世，1996到2016，轉眼間《女神異聞錄》系列已經陪伴玩家走過了20年的歲月，作為20周年的紀念作，它再次帶來了「驚艷感」。獨特的美術風格、優秀的音樂製作，充滿深意的故事劇情以及完善的遊戲系統，《女神異聞錄5》幾乎在每一個方面都進行了細緻打磨，從而為玩家帶來了極佳的遊戲體驗。優秀的作品不會被時代所埋沒，2017年末，TGA、IGN分別授予《女神異聞錄5》「最佳角色扮演遊戲」，同年12月，GS在評選年度十大遊戲時，《女神異聞錄5》更是位居第二。作為一款經典的JRPG遊戲，該作是如何贏得玩家與市場的認可？接下來就與各位玩家一起進入《女神異聞錄5》的「內心世界」。

華麗的畫面與音樂設計帶給玩家極致的視聽享受

《女神異聞錄5》的故事並不複雜，一個男子高中生意外獲得了特殊能力，他白天在學校正常上課，到了夜晚則利用「Persona」的特殊力量進入到他人的意識世界，為了不斷向社會中扭曲腐壞的大人們發起挑戰，他逐漸成為了「偷竊心靈的怪盜」。為了能讓遊戲故事顯得更加生動有趣，美術團隊在畫面設計上進行了大幅調整。剛進入遊戲後，充滿時髦、炫酷、新潮感的用戶頁面便給人留下了深刻印象。因為加入了大量紅色與黑色的元素，高對比度給玩家帶來了強有力的視覺衝擊，同時強烈的色彩對比更是彰顯出一股「反叛」的意志，這也為遊戲的主題表達奠定了基礎。同時，充滿動感的過渡動畫、轉場效果顯得十分耀眼，配合上戰鬥演出時所表現出的獨具風格的過場動畫，玩家將會享受到不一樣的遊戲風格。

《女神異聞錄5》在美術風格上玩出了花樣，充滿潮流感的色彩搭配、炫酷的UI設計以及獨特藝術氣息的遊戲場景帶給了玩家一場視覺盛宴。同時，該作的音樂設計也極為優秀，經過3年的潛心工作黑將司創出了不少「神曲」，完美與憂傷、輕鬆、熱血等關鍵詞相契合，從而進一步提升了玩家的沉浸感。《女神異聞錄5》的音樂製作相當優秀，只要戰鬥曲目響起，隱藏於內心的「熱血感」就會被瞬間激發。優秀的畫面與音樂設計為玩家帶來了一場極佳的視聽盛宴，同時也為遊戲的戰鬥體驗與思想表達鋪墊了道路。

都市懸疑故事與校園生活相結合，劇情故事充滿吸引力

《女神異聞錄5》聚焦於一位普通的高中生，通過他的變化傳達出「自我成長與何為正義」的精神。主角在機緣巧合之下擁有了「Persona」的能力，在遊戲中該詞被譯為「人格面具」，玩家可以利用不同的人格面具來武裝自己，從而可以在虛擬的意識世界中與敵人進行戰鬥。在《女神異聞錄5》中，玩家可以通過「改心」的方式讓那些沒有證據而逃避法律制裁的人良心發現，從而坦白自己的罪行。雖然怪盜的出發點是好的，但是這一行為卻與法律產生了衝突，在正常的社會制度之下，「義警」是不允許出現的。

起初，怪盜團的成員是為了自己而戰鬥，隨後逐漸成為了守護世界的英雄，在眾人的推崇之下他們感受到了正義的魅力。但是突如其來的「墜落」，又讓他們深陷迷茫之中，到底如何才能維護「正義」？在法律面前，他們的行為似乎不能稱之為「正義」，但是面對那些逃脫法律制裁的壞人，又不能袖手旁觀，面對「兩難選擇」玩家不禁也會陷入到深思之中。另外，遊戲中的反派來源於「七宗罪」，每一個人都象徵著人類的執念與慾望，通過細膩的人物描寫，正邪之間的對抗充滿了思想性。

除了向玩家提出一個個哲學問題，《女神異聞錄5》還加入了一些日常元素，玩家在戰鬥之外還能夠體驗日常的校園生活，這也為緊張刺激的遊戲流程增添了一絲輕鬆的氣氛。

高度自由的設定帶給玩家多樣的玩法

在優秀的劇情之下，《女神異聞錄5》通過豐富多樣的戰鬥系統為玩家帶來了酣暢淋漓的戰鬥體驗。作為一款日式RPG遊戲，「回合制」成為了戰鬥的標準配製，乍眼一看，遊戲的戰鬥系統似乎為玩家設下了重重阻礙，但是在固有的框架之內，製作團隊融入了高度自由的設定，從而讓玩家享受到不一樣的操作體驗。

主角一共有五個維度的評價，分別為：勇氣、知識、魅力、靈巧、體貼，遊戲中會出現各種各樣的任務，玩家只要逐一完成就可以提高相應的等級。每一個屬性之間並不是相互獨立的，它們相互影響，從而構成了一個統一整體。其次，怪盜團所具有的人格面具各不相同，其所對應的抗性屬性與弱點屬性也存在著很大的不同，如果能找到敵人的弱點屬性，那麼就可以使其產生倒地特效。豐富多樣的戰鬥系統為玩家提供了多種多樣的戰鬥方式，從而使得「回合制」變得更加有趣。另外，《女神異聞錄5》中還存在著迷宮探索的設定，每一個迷宮都象徵著人類內心的慾望與強烈扭曲的意志，面對極具意象化的異世界，玩家將會領略到獨特的風格。

充滿個性的人設，感受小人物的成長之路

《女神異聞錄5》重點描述了多位高中生的成長之路，他們在機緣巧合之下成為了「怪盜團」的成員，為了與世俗中的不公進行對抗，他們在黑暗之中默默進行著戰鬥。整個故事框架給人一種，諾蘭的《蝙蝠俠：黑暗騎士》的感覺，對於「正義」的探討讓玩家如痴如醉。為了維護公平正義，為了將逃避法律制裁的壞人繩之以法，主角等人開始了一場場緊張刺激的戰鬥，隨著劇情的不斷深入，他們逐漸從幼稚的高中生成長為「正義的夥伴」，從他們身上，玩家感受到了責任所帶來的力量。

結語

毫無疑問，《女神異聞錄5》絕對算是近些年最為優秀的JRPG遊戲，獨具風格的畫面與音樂設計滿足了玩家的視聽享受。同時，在充滿思想性的故事背景下，玩家對於「正義」也有了深刻的認識。高度自由的戰鬥系統為玩家帶來了不錯的操作體驗，通過多方面的搭配組合，《女神異聞錄5》最終向玩家呈現出一個近乎完美的答卷。
