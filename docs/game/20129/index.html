三國志11：經典MOD青山依舊1.0 SIRE1.29新功能和新玩法盤點

2019-12-18 674條評論 3,502次閱讀 1245人點贊

大家好，我是為大家推薦 經典遊戲的 玩家非攻，前段時間一直在玩血色衣冠5.0和SIRE1.28D. 現在終於安裝完青山依舊的版本,SIRE也順利更新到了1.29版本，下面就來為大家介紹和盤點下新地圖，和新功能。

青山依舊,是以三國武將為主，對多數武將能力和特技進行修正，使得武將屬性，特技，更為貼近歷史人物形象。更為我們提供了非常多製作精良的新劇本，進行遊戲.

諸葛亮 勢力核心 三級光環 統御 智力 政治 魅力 光環全開 弩兵雙SS級 特技 神算 ！寶物特技 治軍神算可以配合 龐統特技 連環 神算每次計謀都可以暴擊，可以一次控制3個部隊.缺少特技 百出 但是弩兵雙SS又浪費了一點。諸葛亮的武力太低，對追求極致輸出的玩家來說，也並太合適作為主力輸出。

劉備 五位屬性得到很大的提升 激勵配合 廖化的 百戰 再搭配個關羽或者張飛 補充暴擊 做主力輸出.

SIRE1.29 新功能介紹

光環可視化 滑鼠移到部隊上可以看到光環

武力光環效果戰法成功率改為增加暴擊傷害說明：伴隨一起調整的是戰神暴擊威力增益改為增加而非重置原暴擊威力，且只對部隊有效，對耐久無效，現在如果配置30再加上原來的暴擊 20，就是暴擊威力 150%。

以及加入了水軍系統:

水軍系統使得原有的防守策略和城市地域特徵全部都改變了，港口已經不能在堵住大量AI進攻了，很多小河，淺灘都會成為敵人的進攻路線，整體防守難度大大增加了哦。 需要打水軍補丁

功績系統

每位武將可以額外配置最多5個功階特技，依據功績解鎖，未解鎖的特技以灰色顯示（需要打補丁）

退休老碼農大神的迷霧系統

想暗黑地下城，星際爭霸,紅色警戒等遊戲一樣，進入遊戲有黑色迷霧，只有己方建築和視野提供視野.
