強隊並非不可戰勝！跑跑卡丁車P1聯賽第五周回顧

天天要聞
2019-12-11

跑跑卡丁車P1聯賽常規賽第五周已經全部落下帷幕，我們先來關注一下積分榜！

　　常規賽第五周的比賽結束後，雖然Weibo再遭敗績，但是依舊以13勝2負的戰績積13分，高居積分榜第一。緊隨其後的是XROCK，-他們目前的戰績是11勝4負，積11分。從第三名開始，各支戰隊的名次與上周相比出現明顯的變化，首先是EL掉落到第四名，AG.M則上升到了第三。Newbee保持了自己第五的名次，狼隊Wolves從第八名上升到第六名，這也意味著表狼隊離季後賽又更進一步了！CL上升到第七名，目前與狼隊僅差1分。RSGM掉落到第八名，但目前積分和CL一樣，只是凈勝分相差了1分。LGD和KSSC分列第九名和第十名，值得一提的是，KSSC在本周第三日的比賽中，以一場酣暢淋漓的勝果終結了自己的連敗。

　　強隊紛紛折戟，強隊並非不可戰勝！

　　跑跑卡丁車手游P1聯賽的賽場上，有一些戰隊一直都表現的非常亮眼，比如Weibo、XROCK、EL、AG.M等戰隊，稱之為強隊可以說是實至名歸。進入常規賽階段後，這些強隊更是火力全開，保持著很高的勝率。積分榜第一的Weibo戰隊更是在很長一段時間內保持著「不可戰勝」的神話。來到常規賽第五周的對決後，這樣的情況終於出現了變化！

　　首先是CL豪取三連勝，分別戰勝了XROCK、AG.M和EL三支強隊。雖然從比賽中我們可以看出，CL戰隊在組隊競速賽上存在著一些問題，但是組隊道具賽和個人競速對決的實力毋庸置疑。這樣一支強勢覺醒的CL戰隊，相信即便是面對Weibo也有一戰之力。

　　其次Weibo在第五周的比賽中，也是再次品嘗到了失敗的滋味。而這次掀翻P1聯賽「銀河戰艦」的隊伍則是Newbee，雙方戰至個人競速對決環節，Newbee的實力選手靠山直接2:0戰勝Weibo明星選手歐神，用一場酣暢淋漓的勝利告訴我們，強隊並非不可逾越！

　　除過EL和Newbee兩支戰隊有著精彩的發揮外，KSSC也是在第五周的比賽中打破自己的連敗魔咒。在第五周的第三個比賽日里，KSSC迎來了和LGD的對決，在這場比賽里，他們似乎找回了預選賽時候的狀態，2:0乾淨利索的戰勝了LGD。雖然KSSC已晉級無望，但依舊可以盡情拼搏享受聯賽！常規賽最後一周，期待他們能夠有更好的表現。

　　個人競速對決的「出場率」明顯增加！

　　除了強隊的賽場統治力有所下降外，常規賽第五周的比賽還有一個很明顯的變化，那就是個人競速對決的「出場率」明顯增加。第五周一共進行了15場對決，其中有9場對決都打滿三盤，最終通過個人競速對決來分出勝負。這樣的改變使得選手們需要更加註重在個人競速對決中的發揮，更穩定、更極限、更低的失誤率是取勝的關鍵。CL戰隊能夠豪取三連勝，很大一部分原因就在於本周菠蘿選手的完美髮揮。在個人競速對決中，菠蘿分別戰勝了848、演繹騷和天王星三名明星選手，從而幫助CL連拿三分，目前也是有望闖入到季後賽。

　　不僅CL也證明了個人競速對決的重要性，從「銀河戰艦」Weibo戰隊身上我們也可以看出，個人競速對決是戰勝強隊的殺手鐧。Weibo戰隊在前三周的比賽中一直保持著不敗金身，他們也一直沒被拖入到個人競速對決環節。而進入到第四周的比賽後，他們三次被拖入個人競速賽，最終遭遇常規賽首敗。來到第五周的比賽，他們再次被Newbee拖入到了個人競速環節，結果1:2輸給了Newbee戰隊，再次感受了失敗的滋味。從勝率上來分析，目前Weibo在常規賽階段有四場比賽被拖入個人競速，只取得了兩場的勝利，即Weibo個人競速只有50%的勝率。這樣的數據也告訴了其他戰隊，想要戰勝Weibo，個人競速非常關鍵。

　　多支戰隊分差接近，多場「死斗」註定上演！

　　常規賽第五周的比賽落幕後，我們也看到了，除過Weibo、XROCK和AG.M外，其他戰隊的出線狀況都不太明朗。EL目前的戰績是8勝7負積8分，而處在季後賽晉級邊緣的CL戰隊，也有著6分。最後一周若EL戰隊發揮不佳，依舊有可能被擠出前6名，當然這個幾率並不大。而排在更後面的Newbee、Wolves等戰隊則基本上都要迎來「生死戰」。排名靠後的RSGM、LGD更是需要背水一戰，第六周的每一場對決，對於他們來說都是「生死戰」。相信第六周的比賽會異常精彩，究竟哪些戰隊能夠闖入季後賽，讓我們一起期待！

　　第六周，跑跑卡丁車P1聯賽也終於迎來了常規賽階段的「終局之戰」，究竟誰去誰留？讓我們拭目以待。

　　「競速之巔，拖漂無限！」這場競速盛宴即將迎接開賽以來的第一個「競速之巔」，常規賽最後一周的對決，不容錯過！
