絕地求生大獎賽：4AM狀態堪憂，SMG猛虎下山淪為背景板

2019-12-23 388條評論 1,533次閱讀 364人點贊

12月22號，由鬥魚獨播的絕地求生黃金大獎賽正在如火如荼的進行當中，相信很多人對這一個賽事並不陌生，鬥魚黃金大獎賽發展到如今已經經歷了8個賽季，若是沒有絕對的硬核實力，是不能夠堅持這麼久的。如今匯聚著4AM，OMG，RNG等國內各大俱樂部的強隊，為我們上演一番龍爭虎鬥。

如今黃金大獎賽已經正式進入決賽階段，這就意味著鬥魚黃金大獎賽迎來了最精彩的一階段，在小組賽或者是單挑賽上面沒有發力的戰隊，一定會在這一個階段爆發出來。但是有一支戰隊卻是例外，它就是4AM戰隊，作為PCL夏季賽冠軍的獲得者，更是衝進世界總決賽PGC前三名的那一支隊伍，在首日決賽當中卻落下倒數第二名的成績。

總決賽首日的6場比賽下來，僅僅只拿到了19分的成績，這樣的成績對比在PCL或者是PGC上面，還不知當時成績的一個零頭！為什麼4AM會奪得如此差勁的成績呢？有網友調侃道：「是不是因為小4AM奪下PEL冠軍，你們就鬆懈了？」 也有人說：「是不是你們膨脹了？」 但這都只是網友們的臆想而已，最有可能的原因還是在於他們戰隊的狀態上面，此外就是韋神所說的進行狀態調整了。

在SMG戰隊方面，此次SMG戰隊首日積分暫列第五，但卻在賽事中再度淪為背景板了。在決賽圈3V1的情況下，三人猛虎下山慘遭RNG獨狼雨帝卡死吃雞，其中堡壘更是勇敢跳下懸崖倒地了。不得不說SMG這下真的太傷了。

在此次鬥魚黃金大獎賽中，TSG戰隊的表現就與4AM形成了強烈的對比，在首日6局比賽當中拿下了兩雞直接登上第一位，與4AM有著36分的差距，這也就意味著4AM想追趕上來，具有著一定的難度。畢竟SG這兩場吃雞的表現可謂非常驚艷，第一局TSG就挑了SSS戰隊與Ark戰隊的包圍，第五局更是與BA進行打靶對戰，兩局都是以TSG勝利收尾，現階段他們手感可熱乎著，並不好對付。如今鬥魚獎金大獎賽的決賽戰鬥仍在繼續之中，鬥魚直播平台將會全場全程藍光直播並且會有各大戰隊的第一視角進行觀看！

不過不管如何，比賽這才剛剛開始，一切皆有可能，畢竟4AM戰隊的確給到我們太多太多的奇蹟了。不知道在後續的賽事中，4AM能否力挽狂瀾創造佳績呢？
