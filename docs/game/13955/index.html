轉會期僅剩一周時間！還有這些大牌沒官宣，S10誰才是真銀河戰艦

2019-12-12 34條評論 1,051次閱讀 79人點贊

2019年LPL冬季轉會期可以說是一場漫長的馬拉松比賽，幾乎每一支戰隊都在「憋大招」，這裡的幾個主要原因包括，其中今年的自由人市場太過火熱是一大原因，包括JKL、xiye、SMLZ等選手都相繼成為自由人，加上LCK那邊GRF的震蕩，也讓很多戰隊有了很多選擇。

不過LPL的轉會期截至時間是2019年12月19日，目前剩下一周左右的時間，卻還有大量的選手沒有確定自己的下家，另外幾乎所有的LPL戰隊都沒有宣布新賽季陣容，對於即將開始的德瑪西亞杯來說，剩下的時間已經不多了。

截至12月11日，自由人市場上還要許多知名選手沒有宣布下家，包括今年最大牌自由人JKL，還有Maple、SMLZ、xiye、SofM等選手均沒有確定下家，或者說戰隊方面還暫時還沒有正式官宣。另外網友們還關心著IG二隊的重擔Forge的去向，認為在今年他在LPL的幾場亮眼表現，有可能贏得一個LPL戰隊的首發中單位置，因此也被認為是轉會期的一個重要選手。

但目前所有LPL幾乎都保持著往年的風格，將「憋氣大賽」進行到底，除了WE已經宣布中單和ADC離隊之外，OMG也進行了大幅度的調整，多人同時不再續約被宣布離隊。看來面對明年在中國本土舉辦的S10總決賽，很可能會有不少的戰隊決定今年進行引援補強，爭取在明年的中國比賽中獲得登場機會。

雖然前有TES宣布Karsa加盟，後有Khan幾乎確定加入FPX，一個月以來LPL的轉會期還是沒有太大消息，看來今年憋氣大賽還要等待最後一天才會正式揭曉，大家不妨還是等到12月19日來看看今年哪家戰隊是最大贏家吧！各位覺得S10哪支戰隊的陣容最豪華呢？
