EDG新上單成德杯最大黑馬？表現不輸369，粉絲：RNG出價3000萬

2019-12-23 71條評論 1,417次閱讀 206人點贊

在本次德瑪西亞杯上，要說最讓大家意外的隊伍，非EDG莫屬。作為曾經LPL的皇帝，EDG王朝在最近開始崩塌。S9上沒能進入世界賽，也是創造了隊史最差戰績。在這樣的情況下，EDG在轉會期內並沒有太大的引援，基本新人都是從青訓提拔上來。

看到EDG在轉會期內的不作為，不少粉絲對此感到寒心。特別是在功勛老將廠長宣布轉型之後，不少人直言青春真的結束了。然而讓人沒想到的是，就在大家不看好EDG的情況下，憑藉這套新陣容EDG以小組頭名的身份出線。

在更換陣容之後EDG能夠小組頭名出線，除了中路scout發揮出色之外，新上單才是本次EDG獲勝的最大功臣。作為EDG轉會期內最大牌的引援，從青訓上花了50萬買入的xiaoxiang在德杯首發登場。

在面對TES這支號稱新賽季冠軍有利競爭者時，xiaoxiang通過自己完美的發揮打出身價。我們都知道上路鱷魚是完全打不過凱南，但xiaoxiang在面對369這位LPL一線上單時，不僅線上沒有被壓，團戰里更是做到了自己應該做的事。

在關鍵的團戰中，更是由他先手控制住了敵方霞，配合隊友一套技能秒殺。正式因為這波擊殺，讓EDG趁機推掉了敵方高地，將局勢搬回。如果這波不是因為xiaoxiang先手擊殺了霞，前期獲得巨大優勢的TES很有可能在大龍團擴大勝勢。

作為一名新人，能夠和369這樣成名的上單對線不被壓制實屬不易。更別說兩人的英雄存在明顯的剋制關係，xiaoxiang在369拿到優勢後依舊有如此精彩的發揮，讓很多EDG粉絲非常開心。

看到EDG新人上單首次亮相德杯就打出了這樣的表現，RNG粉絲對此非常羨慕。不少人直言EDG愛德朱的抽卡能力太頂，50萬直接買到了SSR級別的選手。反觀RNG戰隊，在新打野和新AD表現不錯的情況下，上中”虎狼”二人組持續拉跨，讓很多粉絲非常失望。對此不少RNG粉絲調侃，如果EDG明年打算將這位新上單賣掉，那麼RNG願意花3000萬買入。小夥伴們EDG這名新上單的表現如何評價呢？一起來說說吧~
