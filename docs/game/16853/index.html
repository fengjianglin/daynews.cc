Switch、Xbox自適應手柄入選《時代》2010年代十佳電子產品

2019-12-16 676條評論 3,187次閱讀 553人點贊

《時代》雜誌近日評出「2010年代十佳電子產品」，Switch、Xbox 自適應手柄當選其中。

對於 Switch，《時代》雜誌表示：

說到遊戲，從 1989 年低調的 Game Boy 開始，就沒有什麼能超越任天堂的記錄。但近年發售的 Switch 無疑是任天堂迄今為止對遊戲界做出的最大貢獻，相比其他競爭對手，Switch 輕而易舉的成為近十年來最重要的主機，玩家們會請求開發者將它們的下一部作品帶入 Switch。受智能手機所啟發，伴隨任天堂不斷成長的各類經典與招牌，Switch 成為了任天堂遊戲理念的巔峰之作。

對於 Xbox 自適應手柄，《時代》雜誌表示：

微軟的 Xbox 自適應手柄為行動力受限的玩家而造，可根據玩家的需求進行深度自定義調整，這可能是我這十年來最喜歡的電子產品，也是我希望其他大型遊戲公司能夠學習效仿的產品。不僅因為它看起來很漂亮，更是因為它重視了那些長期被大型遊戲忽視的玩家。微軟與 Cerebral Palsy Foundation、AbleGamers 等機構合作，親自設計了從巨大按鈕到外部包裝等所有內容，其中有十幾個介面可以連接輔助控制器、按鈕和操縱桿，兼容 Xbox One 和 PC。它將笑容帶給了每個人，它是遊戲行業最重要的發展之一。

除 Switch、Xbox 自適應手柄外，其他進入榜單的電子產品有：

  • Apple iPad (2010)
  • Tesla Model S (2012)
  • Raspberry Pi (2012)
  • Google Chromecast (2013)
  • DJI Phantom (2013)
  • Amazon Echo (2014)
  • Apple Watch (2015)
  • Apple AirPods (2016)
