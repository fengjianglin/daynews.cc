天天要聞
2019-12-16

除了建隊第一年的廠子捲毛娜美，之後EDG是啥操作，上單koro淘寶淘的，S4坑了一年總算在S5練出來了，S4打完雙C實力不夠，買了三星十人最便宜的雙C，捲毛退役從城市爭霸賽提了meiko上來（沒錯meiko當初的軌跡跟藍公主一樣）。

，加上廠子推薦的mouse來打輔助，mouse打了倆BO2不太行被蛇隊打了個2：0後換上meiko之後再也沒輸過。S5夏季賽koro各方面原因不上場，從保級隊淘來AJ首發，S5打完AJ下放二隊koro抬回首發，然後S6春決就是koro這個點被looper暴打丟了冠軍，pawn有腰傷，於是乎把在SKT打不上比賽的scout買了過來一下子就是白菜價一簽五年，然後把蹲替補席一年多的mouse轉上單

，就這麼把老化的中上重建了，S6夏季賽靠著廠子和deft爆表發揮愣是把scout帶開竅了mouse也打的像人了，就這麼全勝奪冠……S7deft走了想買loken被京東截胡，突發奇想把在ROX替補的中單zet帶過來打ADC，廠子休戰把愛蘿莉簽回來打首發，明明S6上單大問題接著就用mouse和koro，提了個青訓上單MINN，上了一場BO3把當屆冠軍WE吊打一頓後就沒上過了

。夏季賽廠子回歸首發koro賣掉提了個青訓上單奧迪，賽季初打的一塌糊塗zet謎之發揮，洲際賽回來突然全院狀態爆表，末尾階段iboy上場之後一路奪冠。S8把在C9板凳坐穿的老熟人Ray帶回來，打野提青訓haro，S8第一次全年無冠，進了S8保留了最後一絲尊嚴。S9春季賽還是信任原班人馬，BP問題嚴重還請了喜歡謎之BP的heart，haro越打越撈把廠子抬出來進季後賽結果季後賽謎之抬下去，夏季賽還是沒引援又提了個青訓jiejie，在春季賽狀態爆表的Ray下場後從V5淘來jinoo……

這就是EDG更新換代的作風，有什麼大牌引援嗎？這還沒算beame和雅典娜等人，要不是這幾個打的實在一般可能連scout都沒有了就用他們頂pawn
好像當年pawn和deft就是三星十個人里最便宜的…

然後是最近隊內的測評是替補hope優於iboy而把iboy賣給了VG 之前也是一手對內測評jiejie優於haro賣給了RW 。同時業內選手的待遇和工資也是相對最低的。而且現在也是不斷的消費廠長的情懷。作為粉絲當然不能理解EDG管理層的作為。但是從他自身的利益考慮，用最低的付出換最高的收益。就好理解了！不知道你對於EDG的管理層作為怎麼看？
