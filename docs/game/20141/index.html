UZI排位遇企業級輔助，秀出寶藍級遊戲理解！澡子哥：你倒是A他啊

2019-12-18 699條評論 2,339次閱讀 470人點贊

在S10世界賽結束後，寶藍在直播的一段對話被網友們譽為LPL第五聖經，成為了企業級富輔助玩家排位必備的寶典。具體內容為「你們老是罵我賣AD，可輔助就算站在AD前面，對面又不是青銅白銀誰會管你輔助啊？輔助a一下就20血，除非是團戰需要ADC收割，不然我才不會比ADC先死。」而就在UZI近日的一場排位局中，他真就偶遇了一位嚴守寶藍聖經的奇葩輔助玩家。

不知道是為了節目效果，還是真就嚴守了寶藍對於輔助英雄的頂級遊戲理解，澡子哥偶遇的莫甘娜玩家在前期幾波換血中，近乎都沒有A過敵方ADC，嚴格地遵守輔助放完技能就躲在ADC身後的聖經，搞得UZI心態有些急躁了。

在三級一波對拼中，在UZI交出雙招逃出敵方攻擊範圍的同時，莫甘娜玩家除了給薇恩套上了護盾外，技能全空並且忙於跑路放任對面殘血的AD追擊澡子哥，導致UZI的心態終極炸裂。在直播時直接開罵：「你倒是A他啊，不要怕啊，還跑幹什麼！」

在UZI極限一換一之後整個人都氣炸了，通過拍打鍵盤來發泄自己的情緒「不給我一點操作空間，我整個人都不好了！」在15分鐘前，在這位輔助玩家的努力下，有UZI坐鎮的下路直接被對面給打穿掉了。澡子哥看著自己1杠5的戰績，也能發出無奈的自嘲。

也許是遊戲的理解不一樣，輔助在保自家AD的時候，究竟要不要給對面壓力，還是學這位佛系輔助，小夥伴們對此怎麼看呢？歡迎在下方留言。
