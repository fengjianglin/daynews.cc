“沒有Uzi的RNG”登上熱搜前十！論新賽季雙人路的重要性

天天要聞
2019-12-21

北京時間2019年12月21號，英雄聯盟德瑪西亞杯正在激烈地進行。不過，雖說我們一直都在討論「AD現狀」的一個話題，但是從目前的比賽情況來看，雙人路取得優勢之後的隊伍大多數都是贏得了比賽的勝利。這自然也是跟新賽季的變化所分不開的。特別是那一場RNG與OMG之間的對決，沒有Uzi登場卻讓「沒有Uzi的RNG」登上了熱搜，而且一衝就衝到了前十的位置。這自然也是體現了Uzi對於RNG戰隊的重要程度。

那麼，在「AD現狀」的情況之下，雙人路依舊是最為關鍵的一環嗎？答案當然是肯定的。這就要說到新版本中元素龍的重要性。首先就是「龍魂」的存在，在首個擊殺了4條元素亞龍之後的隊伍會獲得一個龍魂。而不與之前的版本相同，當前版本中每一種元素的龍魂都是有著非常關鍵的作用。這自然大大增加了元素龍在比賽中的戰略意義。

其次，就是遠古巨龍Buff的改版。新增的「斬殺」效果更是成為了團戰中非常強大的存在。比如在OMG與RNG比賽的最後一場團戰中，Smlz的女警配合著「龍魂」跟遠古巨龍的「斬殺」效果幾乎是一下平A就對小虎的辛德拉完成了擊殺。可想而知，在如今的比賽中，獲得了「龍魂」以及遠古巨龍Buff的一方會有多麼的恐怖了。

另外，再從OMG與RNG這一場比賽的情況來看，在25分鐘左右的比賽時間中，雖然OMG與RNG經濟依舊是持平的狀態，但因為OMG這邊擁有著四條小龍以及「龍魂」，RNG其實在團戰中已經是處於相當劣勢的一方了。也就是說，在新賽季不僅僅只是看團隊經濟，元素龍也已經成為了影響團戰走向的關鍵部分。

當然了，也正因為元素龍的變化，再加上遠古巨龍Buff的「斬殺」效果，不少人也紛紛表示「AD現狀」。但事實上，在新賽季的賽事中，一組出色的雙人路的的確確會對比賽產生巨大的影響。因為雙人路是離元素龍最近的一條線，所以在比賽的對線期，雙人路如果能夠把控著推線權，對於前期的元素龍控制無疑是有著非常大的幫助的。這自然也是證明了新賽季Uzi對於RNG是有多麼的重要了。

因為不談別的，至少在對線上面LPL能夠贏下Uzi跟小明組合的雙人路並不多。而一旦Uzi跟小明在對線期取得主動的線權，那麼對於前期元素龍把控的概率也就大大增加了。所以，在當前版本之前，沒有了Uzi的RNG或許可以在德杯中奪得冠軍，但在如今的新賽季版本中，沒有了Uzi的RNG還真的挺艱難的。
