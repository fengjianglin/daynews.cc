魔獸世界新劇情或有兩個方向！希女王進本幾率小，洗白可能性大？

2019-12-11 1,869條評論 3,333次閱讀 2230人點贊

魔獸世界真正的魅力在於玩家的認真用心，玩家們會把自己代入到魔獸世界當中進而去挖掘魔獸的歷史，然後呈現在自己的眼前。其實我們都明白，這些東西只是暴雪設計師的一個想法、一個代碼就可以顛覆的東西，但正因為有了這樣的玩家才讓這冰冷生硬的代碼活了起來，讓我們有機會去了解魔獸這個世界。

​我只是茫茫多魔獸玩家中的一個，70年代開始進入這個世界和朋友們做做日常，然後通過日常任務提升聲望來見證這個世界的變化，這也是我喜歡魔獸的原因。其實即便到了現在魔獸依然還是那個魔獸，通過玩家自己的努力去提升聲望，然後場景會發生變化，這個過程其實於我個人而言一直都比較震撼，就像在極短的時間內見證了一個世界的變化一樣。

魔獸世界下一段劇情有兩個可能，一是暴雪自打耳光把女王送進本；二是雙方聯軍進攻恩佐斯發覺女王正獨自對抗徹底洗白。不過按暴雪現在的尿性自打耳光的可能性也有，畢竟打得多了也就不痛了。希女王對打大王時說過，你們都得死，而且選擇忠於女王有後續任務，說什麼和艾薩拉合作用死亡對抗她的主人。那麼很明顯就是要洗白希女王了，而且現在的死亡應該並不是真的死亡，應該就是女王用來對抗古神得一個手段。最後劇情可能會反轉，只有死一次才能獲得一種力量，一種死亡的力量，一種可以抗衡古神的力量，而且最後大家都死了應該還能活過來，可能會包擴沃金都能復活。

至於很多人說現在魔獸變了，那只是因為我們這一批玩家已經進入了上有老下有小的階段，沒那麼多時間與精力去體會這一切了。個人希望在有精力有時間的情況下堅持下去，因為這不單單是一個遊戲了，這是我們這個年齡段的人的一個回憶。魔獸從未改變，若有機會我定會堅持下去。

說到底暴雪已經不再是以前的暴雪，他們已經做不出經典之作，暴雪原來宗旨是用心做自己喜歡的遊戲，如今連自己都過不了關怎麼拿出來？一部部經典之作都是經過不段的更過再創新最後才問世，暴雪粉絲也會因為長久期盼等到驚喜。可在10年前這全都變了公司創始人首度設計師4位靈魂人物離職，暴雪換了位銅臭味老闆全變了。
