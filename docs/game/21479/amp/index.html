宣告和KID連麥：現在很窮，只能接單還債，可以接受去巴西打比賽

天天要聞
2019-12-19

很多人對電競職業選手都充滿了幻想，彷彿只要能夠成為職業選手，那麼後半生基本就會衣食無憂，每天吃飽飽。其實不然，只有成為了職業選手中的佼佼者，那麼才有可能實現經濟獨立。說起LPL職業選手中最「慘」的一位，肯定會有小夥伴們想到「死亡宣告」。宣告可謂是自己親手葬送了自己的前程。如今不僅沒有比賽可打，而且還欠了很多的外債。

最近德雲色的KID在直播時，和死亡宣告來了一波現場連麥。相信大家印象中的「死亡宣告」都是一身肌肉，聽起來就很「兇殘」，搞不好桌子就被砸壞了。而事實上，正常情況下的宣告還是比較隨和的，Kid表示當時最早一批的職業選手中，PDD就和他講過：宣告是一名很隨和的選手。

由於之前宣告一直都在做「陪玩」，打的都是黃金白金局，連峽谷之巔的許可權都已經被收回了。因此宣告和Kid開黑甚至都需要借號。不過宣告表示自己還是希望能夠打高端局、質量局的。不過先得把欠的錢還完，只有不斷的接單賺錢還完了債，自己才能安心沖分，隨後兩人說著說著就聊起了電競圈的八卦新聞。

之前有人爆料，LPL的兩名職業選手去往了巴西賽區打職業。眾所周知，在英雄聯盟各大賽區中，LPL應該算得上是強度最高的賽區了，巴西賽區和LPL相比，強度要低得多。隨後宣告透露，如果有機會的話，自己可以去巴西賽區打職業的。Kid雖然退役了，但畢竟對於職業圈的各種消息還是非常靈通的，Kid說巴西賽區的工資還可以，差不多三萬。宣告回復到：我打的好我就回來了，我現在如果打我就證明自己這一年不會發脾氣。現在都怕我砸基地。

看得出來，死亡宣告還是非常熱愛這個舞台，熱愛打比賽的感覺的。不過因為自己的種種原因，導致現在沒有比賽可打。雖然可惜，但是也怨不得別人。想要成為一名好的職業選手，在電競圈有一定的影響力，收穫很多的粉絲，除了有必要的技術以外，情商也是很重要的一點。

就拿PDD來舉例，做職業選手的時候，PDD就有「世界第一上單皇子」之稱，首先在技術上就非常過關，退役以後，PDD的直播也搞得風生水起。憑藉著自己的搞笑幽默收穫了大批的「小豬仔」。在當初直播圈實錘開掛期間，PDD更是以各種各樣的理由消失在了大家的視野之中，緊接著復出的時候就給大家帶來了一個好消息，直接修建了一所皮皮小學。現在更是搞青訓，做戰隊。可謂是電競圈真正的屌絲逆襲，真正的成功人士。

所以宣告想要再次復出，打比賽的話，除了要保持自己的狀態，控制自己的情緒和提高自己的情商也至關重要，小夥伴們，你們覺得在電競圈的現役、退役選手中除了PDD以外，哪些還算是情商特別高的呢，小皮先說一個：姿態，該你們說了~
