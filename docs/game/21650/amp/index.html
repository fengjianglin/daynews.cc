《Minecraft》（官方中文名：《我的世界》）是一款風靡全球的第一人稱視角的3D高自由度沙盒遊戲。於2009年5月13日發行。由瑞典遊戲設計師馬庫斯·阿列克謝·泊松（該遊戲內的用戶名Notch）開創，現由Mojang AB維護，是Xbox工作室的一部分。

《我的世界》的這種像素風格和樂高積木的磚塊顆粒形狀簡直就是天作之合，自2012年的第一款IDEAR系列的21102 我的世界 森林之後，便一發不可收拾
