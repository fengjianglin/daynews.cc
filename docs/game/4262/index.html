LOL：一句話總結現狀，RNG最真實，DEG，IG最心酸，你覺得呢？

2019-12-08 0條評論 20次閱讀 0人點贊

摘要

英雄不問出處，菜鳥沒有歲數。大家好我是久音！目前處於LPL的轉會期，加上S10將會在中國舉辦，LPL

英雄不問出處，菜鳥沒有歲數。大家好我是久音！

目前處於LPL的轉會期，加上S10將會在中國舉辦，LPL的很多俱樂部都在招兵買馬，增強自己戰隊的實力，畢竟如果能夠在本土拿到一座S賽的冠軍獎盃，自然是一件非常開心的事情，大家也都知道，目前LPL已經有16支戰隊了，並且根據之前透漏的信息，下賽季還會增加一到兩個名額，就在近日，網友們對LPL16支隊伍「一句話現狀總結圖」也是火了：RNG最真實，EDG、IG最心酸！

VG:穩坐榜尾，笑看紛爭

作為LPL最為穩定的一支戰隊，VG已經連續兩個賽季排名墊底，雖然隊員的rank分很高，但是組合到一起總是打不出成績來，並且如果看了VG的比賽，大家都會發現，VG基本上前15分鐘是一種無敵的存在，甚至可以與IG等強隊打的不相上下，但一旦到了中後期，就會出現運營能力差的問題導致優勢喪失。OMG:不降級有恃無恐，不增強混吃等死

大家也知道，目前LPL的名額是買斷機制的，相比於之前，不存在降級的風險，即使成績不甚理想，OMG依舊可以憑藉著全華班得到不少粉絲的支持，尤其是基本上除了冷少，再難有第二個人可以站出來了。

LGD:不斬無名之輩，不招有用之人

作為LPL最為樂觀的一支戰隊，LGD一直是季後賽的最大黑馬，劫富濟貧，雖然進不了季後賽，但每次到了季後賽的前期，LGD就會成為各大戰隊進入季後賽的最大障礙，即使是S8的冠軍IG也曾被LGD斬於馬下。

RW:張無忌獨木難支，俠之隊無緣季後

RW也是為數不多的一支全華班，除了下路的張無忌之外，其餘的選手基本上大家都很少認識吧，carry能力更是不必多說。

V5:我猜，otto比我更想說些什麼

雖然賽季中期V5的雙中單體系有所成效，但是之後被瓦解之後，V5也是難求一勝，本來新賽季V5是準備引入chovy的，而前幾張主教練雖然在韓國駐地，還是沒有將chovy爭取到LPL，之後也是表示會繼續堅持全華班。DMO:要變強中單得換，有實力仍需磨合

DMO最大的問題就是中單一直都是工具人，其實和RNG有點像，上路以及下路的carry能力都很強，打野xiaopeng也是LPL的新晉打野，未來非常值得期待，如果可以有一位強力的中單加入，DMO肯定可以走得更遠，畢竟第一次來到LPL的時候，DMO就進入季後賽，並且零封當時廠長所在的EDG。

JDG: Flawless黯然離隊，Kanavi奮力爭取

S9賽季的結束，JDG打野只剩kanavi一人，雖然在doinb的直播中經常出現，但是夏季賽僅有的幾場比賽表現都不算亮眼。

WE:新人不明所以，老將合約到期

WE的鼎盛時期應該就是S7代表LPL參加S賽了，S9之後下路大舅子回到韓國，而兮夜也是合約到期，目前也沒有任何的人員流動信息，也是讓粉絲極為擔心。

SN:中，上二隊首發，老賊何去何從

Smlz離開SN，可以說，SN最穩定的carry點消失，雖然maple和蛇蛇還在，但是夏季賽的表現也是一般，並沒有特別的出彩。LNG:大家好，我們給大家表演一段雜技

由於有著聖槍哥的存在，LNG總是會有一些神奇的上單英雄出現，風格也和huni極為相似。IG:大腿低調，小鬼難纏。認清核心，何須妄言EDG:廠長劍指S10，何種方式未可知

廠長夏季賽沒有上過一場，尤其是冒泡賽上對於S賽名額的爭奪，即使新人發揮失常，廠長依舊穩坐替補席，就可以看得出來，雖然之前廠長在直播中表示EDG新賽季會加油，劍指S10的很大方式是以教練的身份出現。BLG: Kuro離隊普天同慶，尋找中單迫在眉睫

RNG:求生欲告訴我，不能評論!

由於uzi的存在，並且是LPL目前成績最好的一支全華班，網友也是直接認慫，沒有對現狀進行太多的評價。TES:常規賽驚為天人，季後賽缺少領袖

TES也算是夏季賽的一匹黑馬，但是之後由於面對RNG的時候下路表現太差，直接無緣S賽，雖然中單黃金左手會有亮眼的發揮，但是一旦中單沒有聲音，TES就會陷入被動，上單369還是太年輕，發揮不夠穩定。FPX:不要再誇小天了，我們去勉勵一下LWX吧

拿到了S9的FMVP之後，大家對於小天也是盛譽過滿，導致小天都忍不住了，希望大家可以讓lwx認清自己在FPX中的行為。
