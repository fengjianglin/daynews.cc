DOTA2：新版本推出之後，NEC人徹底沒了，這英雄還有救嗎？

2019-12-16 346條評論 4,349次閱讀 1739人點贊

自從DOTA2更新新版本後，可謂是幾家歡喜幾家愁啊，兩位新英雄的強勢令天梯生態環境變得極為惡劣，而野區的改動，讓不少老牌英雄找到新出路。

不過，相比起來 某些英雄就沒這麼好過了，他們只能眼睜睜地看著其他英雄變強，而自己卻始終坐在冷板凳上無人問津，恰好NEC就是這麼一位英雄。

提起NEC相信不少玩家一定印象深刻，作為DOTA2遊戲中唯一能給對手增加復活時間的英雄，他的恐怖大家都曾感受過。

但新版本中這位英雄如今的地位卻岌岌可危，經過多次削弱後，NEC早就跟之前NEC不是同一個英雄了。在A電競君看來NEC目前的問題主要有以下三點。

對線能力削弱太多

大家都知道曾經的NEC堪稱一個恐怖的線霸人物，但他的被動技能經過多次修改後，傷害實在太低了。

而隨著遊戲版本節奏的變化，NEC目前的分路也存在一定爭議，其實這個英雄本身確實很靈活，3個核心位置他都能擔任，可以他目前的實力，想在線上打出優勢有點難。

特別是在上下兩路，如今的雙人路大多數英雄實在太猛了，例如天怒和大樹這種五號位，本身擁有不俗的輸出外加不怕死，NEC面對他們很難佔到便宜。

另外在中路對線NEC想要佔據優勢將十分家的基本功，況且對手也不一定比你菜啊。

中期有些乏力

過了對線期之後，NEC將會迎來一個新的難題那就是出裝的選擇。新版本紛爭面紗改動後，已經不適合NEC了，所以這個裝備基本上被淘汰了。

而大多數的人選擇還是以肉裝為主，可這也會導致一個問題，那就是輸出有些乏力。肉裝NEC雖然能頂，可他的光環根本就沒有多少傷害。

而1技能和大招又需要消耗很多藍量，滿級D需要190的藍，而大招比他只多不少，況且這還是沒有計算其他技能的情況下，一場團戰NEC不可能只放1個D就行了。

雖然NEC有著被動回藍，但以他目前的大招傷害，想要收成功擊殺對手真沒那麼容易，當個控制技能用倒還不錯。

另外NEC這個英雄本身沒有控制，大招真空期還特別長，自己老胳膊老腿根本就追不上人，說白了就是看隊友臉色。

克制道具太多

最關鍵的一點就是，剋制NEC的道具實在是太多了，光是一個大骨灰就能讓NEC頭疼好一陣，另外還微光也是，光是這兩個小道具就能讓NEC的戰鬥力下降一大半了。

而NEC最怕的還有BKB，可以說NEC面對BKB第一波是沒有任何脾氣的，他沒有有效應對BKB的手段，這讓他發揮起來相當難受。

希望在接下來的遊戲更新中，冰蛙可以注意到這個英雄吧。
