EDG1比0戰勝TES，以小組第一戰績出線，網友：這上單撿到寶了

2019-12-23 3,002條評論 4,962次閱讀 1642人點贊

在剛剛結束的英雄聯盟德瑪西亞杯小組賽第二日EDG對陣TES的比賽中，EDG以1比0的比分戰勝TES贏下比賽，此戰過後EDG本次小組賽的戰績也來到了3-1，成功晉級到下一輪淘汰賽，對於這樣的結果不少EDG的粉絲也是感到非常的滿意。

回顧本次打比賽EDG的表現可以說是相當的不錯，比賽開局3分鐘Xinyi繞後抓中，閃現頂起Knight，Scout跟上傷害拿下一血，10分鐘雙方峽谷先鋒爆發團戰，Hope女槍放出完美大招，Scout拿下雙殺助EDG完成1換3，16分鐘Karsa野區果斷開啟團戰，369天雷進場打出成噸AOE，TES完成0換3，前20分鐘TES建立起3K經濟優勢，26分鐘TES野輔強開蛇女，但Scout交出雙招成功逃生，EDG打出一波0換4隨後一波結束比賽。

賽後網友們對於本次的比賽也是議論紛紛，有網友表示「沒有人能戰勝七點的edg，過年咯過年咯，澱粉哭了，EDG愛德朱50W抽了SSR？這個50w花出了1000w的效果，愛德朱真有你的，如果休賽期拿下卡薩阿水，加上這50w上單，那不得起飛咯！」

還有網友表示「銀河戰艦就這？就這？連菜電都打不過！TES這下路也沒比去年好到哪去啊，我建議最後的mvp給霞吧，太下飯了，這AD簡直辣眼睛···卡薩終於也體會到下路拉胯的感覺了。」

不過也有網友有不同的看法，有網友表示「EDG真有你的，沒想過輸的比賽輸了，沒想過贏的比賽贏了，不過相比選手的發揮今天的解說更讓人下飯，這解說有點像西班牙的足球解說，一激動跟朗誦一樣。」

確實回顧今天的這場比賽EDG的表現的確相當的不錯，特別是上單xiaoxiang以及下路Hope的發揮，可以說打出了大腿級的表現，感覺EDG買這個青訓狀態上單確實是買對了，這樣的表現真的是未來可期，所以對於EDG戰勝TES以小組第一的戰績出線這件事你們怎麼看呢？
