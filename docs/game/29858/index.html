一起來捉妖：合區後QQ區玩家為何節節敗退？不僅僅是實力的問題

2019-12-24 2,204條評論 4,019次閱讀 762人點贊

在一起來捉妖遊戲中，自從QQ區與微信區合區之後，很多玩家都非常的不適應，因為拿不到擂台了。在合區之前，由於很多玩家都退遊了，擂台都空了出來，所以很多等級比較低的玩家都能拿到不少擂台，一級的擂台幾乎都沒人爭奪了。而合區之後，擂台數量沒有增加，但玩家數量突然多了一倍多，也就是僧多粥少。在此情況下，點一兩個擂台都比較困難。

最近幾天，捉妖玩家討論最熱烈的話題就是擂台全面被微信區玩家佔領的問題，好像很多城市都是如此，QQ區玩家一直節節敗退。那麼，QQ區玩家與微信區玩家之間的實力差距真的如此之大嗎？

從整體實力來看，微信區玩家確實要比QQ區的玩家強上一些，而且玩家數量也多一些，這一點可以從世界BOSS預約人數就能看了來，在前幾次的世界BOSS預約挑戰人數中，微信區玩家要比QQ區玩家多好幾萬。但能一片區域一片區域的搶佔擂台，不僅僅是實力的問題。

相信很多玩家都發現了一個現象，那就是那些圈地的圈子都統一了頭像，從山河圖上來看，一片區域全是一個頭像，看起來就像是一個玩家的，實際上是一個圈子的人。附近的玩家要麼加入他們，要麼就別想佔領一個擂台，因為會有很多人來打你，除非是中午去打，否則根本就拿不到擂台第一的獎勵。另外，他們還會拉附近打擂台的玩家加入他們的圈子，表示只有加入他們才可以分擂台，否則必須被打。在此情況下，一些QQ區的玩家為了獲得擂台，也加入了進去，讓那些圈地聯盟的實力更加強大。

當然，也有很多玩家不願意向邪惡勢力低頭，畢竟，他們在以前的圈子已經待了很久了，已經有感情了，自然不願意退出。只不過這些圈子的玩家都是分散在各處的，並不在一個區域，所以都是單打獨鬥，與圈地的圈子打，肯定打不過。哪裡有壓迫，哪裡就有反抗，也有很多玩家非常反感這種圈地的行為，所以成立了反圈地聯盟。

事實上，打擂台主要還是看玩家數量，否則圈地再多，到中午也守不住，會被短時間內全部打下來。只不過很多的好戰分子都被吸收進圈地聯盟了，而沒有加入的大部分都是佛系玩家，所以無論是從數量還是實力上比較，都是比不過的。
