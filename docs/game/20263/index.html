Uzi隊霸實錘！年年四保一未見成效，現在換成六保一劍指S10？

2019-12-17 1,422條評論 2,495次閱讀 910人點贊

截止到昨天為止，沉寂已久的LPL戰隊陸續將自家休賽期的人員變動狀況公布了出來。WE雙C離隊卻還無補充，EDG下路iboy加入VG、廠長退役轉型教練基本談不上有成功引援。反而是一直被嘲買不到人的RNG成為了休賽期LPL「御三家」中最成功的。

小龍堡擔任首發打野這是RNG眾所周知的秘密，隨後引進的幾個新人也沒能給予粉絲眼前一亮的感覺，要說這個休賽期RNG最讓人感到驚訝的操作還是把Mata簽下來當主教練。

昨天上午Mata宣布退役時所有人都在感慨三星白全員退役之殤，轉眼間他就變成了RNG的主教練；外加Tabe回歸擔任分析師，Uzi「隊霸」的稱號就此坐實!集齊「老中青」三代輔助的羈絆，下賽季RNG的傳統藝能「四保一」就要升級成「六保一」了！

RNG向來被人詬病不願意自己培養選手，去年曾試過一段時間二隊的走A怪，但最終結果還是表明他不堪大用。與之相比IG的眼光就老辣地多，The shy和JKL都是名聲不顯之時就已經引進，今年二隊又出了個蕉太狼讓人眼紅，自己培養好苗子自然比不斷買人要好得多。

而今RNG的陣容中上中下都是老將，明年可能將會是他們的最後一年，管理層一下就引進了小龍堡、Betty以及提拔了二隊的S1xu和yuekai看得出來他們也想培養自己的新人，以免未來陷入到無人可用的境地。

這幾個年輕隊員未來能打成什麼樣決定著戰隊管理層對自己培養選手這一計劃的實施決心。

雖說這些新人的實力還有待考證，但在下賽季上場的機會鐵定不會少，行不行至少先得跟粉絲混個臉熟，否則等到Uzi退役hz們也將消散如煙~

EDG為了留住粉絲讓廠長當了一年掛件而後轉為教練。Uzi基本沒有留隊任職的意願，因此管理層推新人的計劃勢必達成；到了下賽季Uzi可能真的要為新人做嫁衣了……
