我們還行吧？DOTA2 7.23版本強勢組合推薦

2019-12-12 1,746條評論 4,194次閱讀 1067人點贊

在7.23版本推出後，兩位新英雄的加入和眾多老英雄的改版都讓如今的對局中有了很多新的套路。兩位新英雄本身的技能和打法，都可以與一些英雄構成威力強大的組合；一些新改動的技能也能讓一些英雄煥發出新的活力。在當前版本中，有哪些配合，只需兩位英雄，就能極大影響比賽走勢呢？

【定點轟炸——電炎絕手&虛空假面】

隨著對電炎絕手的幾次技能加強和廣大玩家對這一英雄的熟悉，電炎絕手的勝率也是一路上升。電炎絕手以前期不俗的耗血能力和中期的團戰爆發成為又一熱門的輔助選擇，而與之搭配的英雄，一般都會強調線上強勢和團戰中的控制能力。雖然進行了一定的平衡，但上個版本大火的虛空假面，仍然是當前版本較為熱門的1號位選擇，無論是線上還是團戰，虛空假面都有著不錯的發揮空間。當比賽中同時選出電炎絕手和虛空假面，那麼無論是對線還是團戰，都有著充足的配合空間。

在對線期，虛空假面自身就有著不錯的對線能力，無論是時間漫遊躲避傷害還是時間鎖定帶來的耗血能力，都讓虛空假面可以與強勢4號位打出良好的互動。電炎絕手則能為虛空假面提供追擊、控制能力，讓本有著出色物理輸出能力的虛空假面完成擊殺。

進入到比賽的中期，虛空假面和電炎絕手的大招配合更是有著相當直觀、強力的效果。虛空假面的大招可以為電炎絕手提供定位，而超高傷害的電炎絕手大招本就有著很遠的射程，與虛空假面的配合可以打出非常可觀的傷害，一旦定位到多人，這場團戰將是一邊倒的場面。

【一往無前——虛無之靈&天涯墨客】

虛無之靈雖然已經被多次削弱，但他本身前期的強勢仍然讓他有著頗為可觀的勝率。但這位英雄本身也有著一些弱點：本身身板較脆，難以抵擋魔法爆發；缺乏穩定的控制能力，一旦一輪傷害未能擊殺對手，就會陷入技能的真空期、而這些缺點，也可以通過隊友的技能來加以彌補。比如天涯墨客，就非常適合與虛無之靈搭檔！

虛無之靈是一位主要依靠技能打出傷害的英雄，並不依賴普攻，因此，比較適合與虛無之靈搭檔的英雄，最好是可以補足控制、提高虛無之靈斬殺線的。天涯墨客的「墨涌」技能需要讓隊友與對方貼身才能最大化作用，而虛無之靈的「異化」和「太虛之徑」兩個技能都有著位移能力，可以讓虛無之靈輕易縮短與對方的距離，發揮「墨涌」的作用；在眩暈之後，虛無之靈不僅可以打出輸出，更是可以用「殘陰」補上控制；天涯墨客的「命運之筆」也可以補充傷害和提供減速，這些都是虛無之靈非常需要的。對於虛無之靈本身害怕的爆發，天涯墨客的「幻影之擁」也可以用沉默的效果來限制對手，兩位英雄的技能是非常契合的。

【我的手更長了？——狼人&聖堂刺客】

之前曾介紹了狼人與諸多英雄的配合，其中最為意想不到的便是與聖堂刺客的組合了。狼人在裝備阿哈利姆神杖後新增「餓狼撕咬」技能，可以將隊友變為近戰（射程150）；而聖堂刺客初始射程僅為140，變成近戰的聖堂刺客射程反而更遠了！變身後的聖堂刺客的「靈能之刃」技能仍然有效，聖堂刺客並不會因為變成近戰損失射程；而因為變成了近戰，聖堂刺客可以觸發迴音戰刃、碎顱錘（近戰）等，這是對聖堂刺客實實在在的加強。在7.23版本中，狼人的「嗥叫」技能重做為了周圍單位最高降低8點護甲，這與減甲流打法的聖堂刺客非常搭配。最重要的，無論是暴擊還是極速，都極大的補足了聖堂刺客自身的缺點……在這樣的加持下，聖堂刺客可以打出非常強勢的發揮！

當然，以上的英雄，也可以與其他的很多英雄打出極具威力的配合，文中所介紹的也只是一些較為常規、配合併不複雜的組合。也歡迎各位玩家分享在新版本中發現的強勢套路，並與好友一起在比賽中使用，努力發掘遊戲的更多玩法，這才是DOTA2的魅力所在。
