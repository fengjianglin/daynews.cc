懷舊服盜賊單刷血色這樣效率最高

天天要聞
2 天 ago

在魔獸世界懷舊服中，盜賊玩家作為整個遊戲中數量排名第二的職業，無論是團隊副本還是5人副本都是人滿為患的狀態，而且如果是非DKP團玩家，就需要在裝備上大出血。同樣盜賊玩家在打錢的能力還是比較出色的，雖然比不上法師，但是自給自足也是沒什麼問題的，本篇文章主要分享一下，盜賊玩家單刷血色的最大化。

盜賊單刷血色修道院是沒有任何難度的，但是通過怎麼樣的路線去刷產生的收益是很大的。關鍵點在於怎麼刷最效率，下面這個路線是我多次單刷總結下來一條最效率的方法

路線流程

血色墓地

主要目標：2個BOSS，3個草藥點，2個箱子（墓地苔，皇血草），稀有精英

路線：進門殺審訊官韋沙斯，大概率掉「染血的手指虎」價格40+Y，第一個箱子穩定刷在BOSS旁邊，走到墓地先看入口左右兩側箱子，如果沒有則會出現在墓穴右邊的小墓室里，每次副本必定會有3個草藥點，如上圖，大概11個刷新點，最後擊殺血法師

血色圖書館

主要目標：2個箱子，1個草藥點（枯葉草）

路線：進門走到上圖草藥刷新點採藥，然後開第一個箱子，穩定刷在之後第一個迴廊的小屋子裡，第二個箱子如果刷新點在上圖下個迴廊的屋子裡，開箱子後直接出本，如果刷新在最後BOSS口，則把BOSS打了

血色武器庫

主要目標：1個箱子，1個草藥點（枯葉草）

路線：采完葯開完箱子即可出門

為什麼要走這個路線

因為血色修道院特殊的原因，雖然幾個副本是獨立的存在，但是共享副本CD，所以重置一次可以刷3個入口。我測試過在血色圖書館第二個箱子沒有刷新在杜安門口的情況下，可以做到12分鐘/次，這樣可以做到5次/小時，然後回城清包。這樣可以讓收益最大化，開更多的箱子，采更多的葯。而血色教堂出產的活根草、金棘草價值不高，而且打BOSS需要清小怪所以放棄。血色軍械庫的BOSS曾經測試過加上，但是每次會增加3，4分鐘，只能達到4次/小時，5箱子+5草藥的價值還是大於4個小強光的。血色圖書館杜安必掉一把藍色武器，和一個綠裝，但是綠裝和藍裝價值經過調整，綠裝不到10Y，藍裝匕首30+Y，法杖40+Y次次都打不划算。訓犬者BOSS掉落的法杖雖然價格高，但是掉率低，而且掉落的狗哨唯一，無法多次拾取經常白打，所以放棄。

專業選擇

如果想要達到最大收益一定要附魔+採藥，因為墓地和圖書館4個箱子出產的綠裝都是介於26~30等級區間，賣商店價值很少，但是卻可以分解出大量的靈魂之塵，平均下來每個綠裝分解了也有40Y左右

出產變現

盜賊玩家按照這個路線的收益上下限比較高，保底收入是草藥，和墓地BOSS的掉落，浮動收益是綠裝的分解和拍賣，箱子出產的草藥，藍裝和圖紙。記住千萬別把所有綠裝都賣商店，包括圖紙

綠裝：所有雄鷹系列放拍賣行包括武器，現在法師小號多不愁賣，通常是裝備2~3G，武器5G，基本很少滯銷，靈猴系列裝備拍賣行，武器賣商店，銷路比雄鷹系列差點，建議1~2G之間，野熊系列裝備拍賣行，武器賣商店，建議1G左右。其它26~30區間裝備分解，31+賣商店

草藥：血色的箱子很容易開出野鋼花、雨燕草、枯葉草、石楠草

圖紙：圖紙是比較看人品的，附魔次級盾牌格擋，鍛造的鋼質武器鏈價值最高。當然銷售時間也長，次級格擋200+，武器鏈50+，其它的鹽爆炸彈圖紙，平衡錘圖紙也能賣上價

藍裝：基本上每小時能出1件，最多的一次5件/小時，價格也不一

總體來說50G/小時還是很容易達到的甚至更高。

注意事項

  • 此賺錢路線的收益大部分依賴於與玩家交易的價值，所以浮動較大，如果這麼刷的玩家多了，收益會下降很多，畢竟小號裝備以及低級圖紙屬於小眾需求，如果出產多了價值也自然下降的多。
  • 一定要按照固定路線刷本，然後重置，要不很容易亂，導致提前爆本
  • 建議關調自動拾取，灰色裝備選擇拾取
