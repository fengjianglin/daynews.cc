少見的任天堂外設！宮本茂的《塞爾達傳說》首發竟然在磁碟機上？

2019-12-23 513條評論 1,567次閱讀 84人點贊

命運多舛的紅白機Disk System

序章·磁碟機並未如紅白機一樣家喻戶曉

相信了解寶琳的讀者朋友們都知道，寶琳的小時候過寒暑假，紅白機絕對不能少，即便是如今，對於紅白機，寶琳仍然有一份偏愛。

相對於銷量超過6000萬台的紅白機家喻戶曉來說，它的外設磁碟機似乎就沒那麼有名了。因為寶琳一直想收一台一體機，所以也頗為關注相關的內容，那今天就拿出來和大家聊聊吧。

如果說對比銷量，紅白機6000多萬的數字足夠成功，並且還有非常多的盜版機，沒有被統計進來。但反觀磁碟機，卻只有444萬的銷量。究其原因，只在日本本土發售是一個，另外一個就是它的隕落速度太快了，這一點確實頗為遺憾。

不過，《薩爾達傳說》，《超級馬里奧兄弟2》與《銀河戰士》等名作，有足夠讓玩家們快樂。這麼說起來，還真有些矛盾呢。

那麼，今天寶琳就來聊聊紅白機磁碟機的興衰史。

紅白機外設·磁碟機

進化·容量更大且可存檔的Quick Disk磁碟來了

我們先將時間調回1985年，那時候的日本，個人家庭電腦發展的已經不錯，軟盤已經相當普及，另外電腦上也已經有了一些挺不錯的遊戲了。

此時紅白機Rom卡帶卻遇到了容量的問題，512Kbit（64KB）的卡帶容量已經很難再提升了。

任天堂社長山內溥當時心裡多多少少有些想法，尤其是面對著機制更為複雜一些的電腦遊戲，他想了很多。比如，是不是可以把電腦遊戲搬到自己的紅白機平台，電腦遊戲可以存檔，而紅白機則完全沒有這個概念，能用密碼選個關，就是極限了。

磁碟機首發遊戲·《薩爾達傳說》封面

思來想去的結果，就是任天堂在1986年的2月21日，推出了這款磁碟機外設，售價15000日元。當時的磁碟容量是1Mbit，也就是卡帶容量的兩倍，而價格呢，則基本上便宜一半。另外，這款磁碟機用的磁碟是可以雙面讀取的，如果玩過磁碟機的玩家肯定都知道，玩著玩著需要換面，也是很正常的操作。

首發護航的遊戲，紅白機有三款，《大金剛》，《大金剛Jr.》與《大力水手》，感覺簡直是宮本茂展銷會……至於這次的磁碟機，首發遊戲則增加到了七款。首先就是經典作品《超級馬里奧兄弟》，這款遊戲自然不比多說，大家也全都知道，任天堂也不可能不利用。還有一部作品，便是宮本茂製作的《塞爾達傳說》，這款遊戲內容之豐富，遊戲性之強，簡直讓人驚嘆，再加上可以存檔，的的確確算是大作無疑了。不過剩下的五部作品，明顯就寒酸了許多，簡直就是小遊戲合集：《高爾夫》，《足球》，《網球》，《棒球》和《麻將》。不得不說，感覺任天堂做這幾款小遊戲，充數嫌疑明顯。

廣為熟知的《任天堂網球》

矛盾·到底是要做大遊戲還是小遊戲？

這裡其實有個矛盾，因為磁碟機本來優勢是可以存檔，容量更大，明顯是要做RPG，做戰略遊戲這種的發展方向。玩家買磁碟機，不也是為了玩更大型的遊戲嗎？任天堂雖然也做了《塞爾達傳說》，但更多的小遊戲，到底是為了什麼？

其實這是山內溥設置的一個消費陷阱，眾多的小遊戲，可以催生任天堂的一個擦寫遊戲的服務。也就算是說，只要你購買了任意一張正版磁碟機遊戲，就可以每次花費500日元，將磁碟上的遊戲刪除，拷貝上另外一款遊戲。

每次花費500日元，就可以擦寫換遊戲！

由於任天堂在發售初期，任天堂的準備也並不充分，他們就先在一些實體店面裝了兩台機器，一台就是Disk Writer，用來支持上一段描述的服務，而這種服務不能由玩家自己操作，必須要店員給弄才行。另外還有一台機器叫做Disk Fax，用來記錄排行積分。

值得一說的是，任天堂在當時並不出售空白的磁碟，如果你想要擦寫遊戲，必須先買一張正版磁碟才行。

Twin Famicom

神來之筆·夏普的Twin Famicom一體機與積分榜

起先，磁碟機的噱頭加上電視的廣告，銷售量節節攀升。

與任天堂關係頗為不錯的夏普還特別做了一款兼容機，叫做Twin Famicom，主要的功能就是將卡帶插槽和磁碟機部分整合到了一起，並且還內置了AV輸出埠。

這台於1986年7月發售的兼容機，顏值頗高，寶琳準備收藏的，就是紅色的那款。

如果嚴格的說，初期符合磁碟機性能理念的遊戲，還真就只是《塞爾達傳說》，能存檔，內容多。

至於《銀河戰士》的發售日，則已經是1986年8月6日的事情了，也就是《塞爾達傳說》發售半年之後了。不過，在6月3日，《超級馬里奧兄弟2》發售了，為什麼要單獨提一下這一部作品，並非是因為主角是馬里奧，而是因為這款遊戲算是磁碟機的最暢銷作品，銷量為265萬套。

《銀河戰士》磁碟

在當時，有一些已經在卡帶上面推出的遊戲，為了彰顯磁碟機容量大的優勢，還特意加了一些內容，比如《綠色兵團》，就加入了隱藏關卡。而《赤色要塞》則正好相反，磁碟機版先發售，之後才有了卡帶版本，而那時候，卡帶的容量也不那麼拘謹了，結果反而是卡帶版的內容更為豐富。

我們再翻回來說說Disk Fax，這算是一個積分排行榜系統。

玩家們在指定的幾款遊戲里，獲得的積分記錄，都可以通過磁碟來Disk Fax設備上做記錄，任天堂呢，則根據統計積分的排名，給予名次靠前的玩家以獎勵，而這些獎勵通常都是非賣品。

《超級馬里奧兄弟2》磁碟機版

意外·卡帶突破瓶頸重回王座！磁碟機無奈命運多舛

不過，卡帶Rom在之前遇到的瓶頸，竟然在磁碟機上市不久，就被解決了……

甚至山內溥的很多想法都還沒有成型，卡帶就再一次回到了王位之上。當時卡帶的Rom容量迅速趕超了磁碟，2Mbit的Rom都已經被生產了出來，並且價格非常便宜。

更為關鍵是，製作商們通過給卡帶里裝入鈕扣電池等操作，改變了早期紅白機卡帶不能存檔的問題。

任誰也不會想到，本以為會前程似錦的磁碟機，竟然如此的不堪一擊。

可以存檔，容量也足夠大，甚至價格也更低了，卡帶完全抹平了自己的缺陷。

反觀磁碟機這邊，劣勢就更為明顯了。

首先是讀盤速度問題，卡帶這種形式，讀盤的速度非常快，而磁碟機就要慢得多。

NeoGeo-CD也被玩家詬病讀盤速度過慢……

這個問題，也曾經出現在SNK的NeoGeo主機上，當時他們為了讓遊戲售價降低，推出了光碟機版本，但光碟的讀取速度，在當時簡直難以忍受。其實，很多玩家針對PlayStation提出的意見，有一條就是讀取時間較長，而PlayStation使用的還是2X速的光碟機，可NeoGeo-CD使用的還是倍速光碟機，結果可想而知。

即便是如今，讀取速度，也同樣是主機們進化的一個方向，固態硬碟的成熟，也會讓玩家受益，在之後的主機上，這種可以大幅降低讀取速度的硬體，勢必會成為標配。

第二個特別明顯的弱點，就是磁碟容易損害的通病，這些問題，就算是電腦上用的軟盤，想必經歷過的玩家也深有感觸。

尤其是在我國，那個時候市場上魚龍混雜，電腦軟盤質量簡直是噩夢，據說還有非常多的假貨充斥著市場。軟盤說壞就說，在當時並不誇張，寶琳每次都把一份資料，拷三份存在軟盤了，這還有大幾率在一段時間後，來個全軍覆沒。

當時電腦上有3寸盤和5寸盤之分

寶琳記得，小時候為了拷貝一個遊戲，可能需要壓縮分卷，然後用6，7張軟盤，然後很可能某張盤，忽然壞了，就需要拿新盤，到店裡專門再靠那個分卷。要知道，離寶琳加最近的這樣的小店，騎著自行車也要十來分鐘，還是騎的風馳電掣的那種。

最後還有一個原因，就是任天堂用大量小遊戲來充斥市場，提倡玩家們反覆擦寫，來賺取500日元，甚至重複賺取，但這種思路，卻遭到了想要做大作的第三方廠商的反對，畢竟500日元就拷走，真不是什麼讓人開心的事情。

當時的任天堂社長山內溥

預見·山內溥似乎早就知道未來的模樣

陰差陽錯，時勢所趨，磁碟機最終只在日本本土進行了銷售，所以我國少見，也情有可原。

不過作為外設的話，超過400萬的銷量，也算不上什麼失敗，甚至在當時，還是值得肯定的一個成績。

但如果仔細想想，山內溥的思路也著實領先。

第一個就是用排行榜發獎的制度，來刺激玩家消費，這已經是目前很多網路遊戲的標準套路了。

第二個就是磁碟機後面的通信埠，這個埠是可以連電話線的，任天堂想做的應該是先在門店設立Disk Wrtier和Disk Fax，但對於距離太遠等不方便的玩家來說，可以通過網路，在家裡就實現去門店的一切功能。

基本上就等同於數字商城了。

現在咱們用的這些，任天堂在幾十年前就想到了，山內溥也著實有眼光。

有興趣的玩家，也可以選擇收藏這款

落幕·磁碟機上仍有不少好遊戲呢！

如果玩家們願意，寶琳還是特別推薦，耐心的去玩玩《塞爾達傳說》的初代，或許你能感悟到什麼也說不定，畢竟咱們現在太過浮躁，甚至我們玩遊戲的初衷。

任天堂的磁碟機雖然並未流行起來，但上面其實仍然有多達200款左右的遊戲，好遊戲也有不少呢。

那麼，今天的話題就到此結束，等到寶琳收到了那台紅色的Twin Famicom，再來給小夥伴們做彙報，Sayonara啦。
