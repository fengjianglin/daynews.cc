元氣騎士：聖誕節版本9大裝備調整解析，你的本命神器是否在內？

2019-12-22 59條評論 3,500次閱讀 1850人點贊
  • 雨傘：獲得冰系元素增益

雨傘在《元氣騎士》中隸屬白武領域，在老宅印象中它的傷害與手感並不突出，2段攻擊的特點雖然賦予它區別於其他武器的長處，但是較低的四維屬性卻限制了這件武器的實力。所以在歷屆版本中，雨傘除了玩梗並沒有多少用處。

調整解析：個人認為，雖然雨傘獲得了冰凍元素加成，可是元素的激發需要相應的暴擊率作為輔助，而毫無暴擊屬性加成的它，並不能讓脫離弱勢的範疇。

  • 魔法手套：降低傷害增加暴擊屬性

魔法手套是《元氣騎士》中第二關BOSS大史萊姆的專屬紅武，剛上線時給老宅的第一印象就是-30的暴擊幾率。由於本紅武自帶毒火雙元素，可是卻不具備暴擊效果，因此不能靠普攻激活元素傷害，雖然有第三段範圍元素附加，可是卻不夠靈活，這無疑是典型的自相矛盾。

調整解析：個人認為，用2點傷害的代價換來了60%的暴擊屬性提升，這對於十分依靠暴擊屬性的元素武器來講是十分划算的。高暴擊屬性在精準暴擊元素暴擊天賦的加成下，傷害至少可提升2-3倍以上。

  • 圓盾與布朗線圈炮：形態變化

在《元氣騎士》這款遊戲中，圓盾是唯一可以格擋彈雨的遠程武器，但由於傷害較低攻擊頻率低下所以不被玩家所鍾愛。而作為藍武的布朗線圈炮實力明顯是超標的，同時兼顧散射、鐳射、彈射的它隱約有媲美天堂之拳的實力。

調整解析：個人認為，圓盾由於霰彈加成可一次釋放3枚盾牌，不論是防守還是攻擊都獲得了質的提升，老宅十分看好這件艾德曼金屬打造的無堅之盾。而布朗線圈炮只是削弱了子彈的體積，對於整體實力來看並沒有多少影響。

  • 速射型武器

在本次調整中，白隱、紅龍、PKP獲得了少量的攻速提升，2次衝鋒槍的藍耗降低1點。這幾種武器雖然各具特色，但是本次的調整並沒有多少話題性，老宅就不再過多敘述。

  • 維C機甲：傷害提升

在遊戲中，維C機甲是公認的最強機甲，不僅擁有強大的火炮還同時兼顧導彈發射器的火力支援，單以實力來講，它並不遜色於任何一件強勢武器。

調整解析：在工程師2技能無限機甲上線後，以維C為主流的「次元斬」玩法也逐漸盛行起來，雖然本次的傷害提升僅為2點，但個人認為，它對於整體實力的增幅卻是強大的。
