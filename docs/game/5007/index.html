The shy轉會另有新瓜？ESPN記者爆料：IG俱樂部想賣，但他不想走

2019-12-03 0條評論 15次閱讀 0人點贊

摘要

在11月底曾有一段時間，在網路上曝出了The shy欲轉會SKT，LCK賽區開出5000萬天價高薪的新聞。對此IG俱樂部也是在第一時間作出了官方闢謠聲明，表示The shy仍在合同期內，並不會在短時間內進行轉會。並對於The shy留在LPL賽區的堅決態度，作出了高度的讚揚，那麼事實真就如此嗎？

就在剛剛不久，外媒ESPN記者在推文上連續發布了兩則有關The shy轉會的消息，表示「這個休賽期有兩筆差點成了但最終吹了的大交易，第一筆就是T1.TheShy，關於另一筆交易我這周很有可能會寫個報道，我們離瘋狂就差一步之遙。」

接著這位EPSN記者還暗示了IG俱樂部真動過賣上單的念頭「我最近幾周會寫一個更長的報道來關注IG和T1的故事，但是我覺得TheShy可能會是(或者就是)最終拒絕轉會的人。我認為比起錢$$$$$他更在乎在一個他舒服的地方打比賽。」

這兩條推文無疑將矛頭再次指向了IG俱樂部，如果轉會事項真就發展到了需要The shy最後決定的地步，那麼無疑在俱樂部層面上已經談妥了大部分有關選手去留的合約。考慮到The shy合同未到期的事實，SKT與The shy的洽談無疑已經取得了IG部分管理層的默許（合同在身的選手，私下接觸俱樂部視為違禁行為）。

對此各大網友也再次對於IG俱樂部的行為感到噁心，表示：「再次證明了，只要IG闢謠了的，就是真的。」「這就是真考慮過賣了。果然對於IG沒有無緣無故的闢謠，噁心吐了。」

當然此事也證明了The shy還真是一位「良心韓援」，面對高薪的誘惑居然在最後一刻堅守住了自己的選擇，留在了LPL賽區。小夥伴們對此怎麼看呢？歡迎在下方留言。
