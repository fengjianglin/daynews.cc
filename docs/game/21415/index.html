fgo聖誕三期復刻高難本詳解 多種方法3回合輕鬆搞定黑槍呆和梅林

2019-12-20 301條評論 1,611次閱讀 233人點贊

fgo國服聖誕三期活動隨著主線完全開放，估計不少玩家已經開始攻略高難本了，這次帶來比較詳細的攻略打法和思路，幫助大家利用多種方法3回合輕鬆搞定黑槍呆和梅林，一起來看看吧。

副本難點

本次副本難度在於梅林和黑槍呆的完美配合和擊破血條後附帶的不可解除的效果，梅林開場發動暴擊率提升技能，血條擊破後會給黑槍呆提供紅魔放和高達10W血量上限以及暴擊威力提升，同時梅林還有無敵等讓人膩歪的技能，非常難搞。黑槍呆更不用多說，擊破一次直接充能1格，擊破第二條血充能2格，就是追求3回合效率通關，至少也要吃一次黑槍呆的寶具，其輸出不但特別高，而且還帶有持續掉血的狀態。

打法思路

攻略思路有兩種，要麼直接硬打用光炮把兩個從者的血量都帶走，要麼逐個擊破，這個副本很吃隊伍的配置還有拐子的持有數量，如果思路不對打起來會很難受。

推薦陣容

1.飛哥+喬老師+雙孔明+CBA+梅林

對於拐子的要求較高，馬老師一定一定等級不能太高，1級毫無疑問是最好的，如果是滿破的話容易翻車，飛哥攜帶寶石翁開局BUFF滿上，喬老師給梅林上龍BUFF，飛哥寶具帶走第一管，喬老師下場。第二回合先讓梅林給無敵，然後換人上孔明，雙孔明給飛哥充能，一炮帶走第二管（飛哥集火黑槍呆）梅林肯定不會掛。黑槍呆發動寶具帶走一個孔明CBA頂上，直接給飛哥充能放第三次寶具，OVER

2.綠卡隊+拐子+單體劍階

這個比較吃配卡，可以考慮用伯爵或者狂蘭，個人用的是狂蘭攜帶50初始NP的禮裝（醉貞最好，個人這裡帶的是百級相撲，效果也一樣）常規綠卡隊配合CBA解決前兩條血，等黑槍呆放寶具送走前排，後排陣容上位直接秒黑槍呆最後一管血，這裡強力推薦用小莫。

另外的拐子或者工具人出不起四大天拐之一可以考慮保羅，或者武則天等。如果覺得前期太吃配卡的話，可以考慮帶一隻泳裝BB來鎖卡。

3.劍階+拐子+防翻車從者（比如海叔和黑狗）

這個思路是目前最穩妥的，不主攻梅林，以黑槍呆為主，現在有了CBA了，還有了很多強力的新劍階從者，打起來很方便，綠卡隊的話建議以總司為核心，如果是紅卡隊的話，西哥和武藏是最佳人選，這裡最推薦的是西哥配合梅林的組合墊底，前期可以用總司打穿兩管血。如果沒有後續防翻車的以黑狗和海叔為主，剩下個梅林特別好搞定，如果嫌效率不高可以考慮最後塞個梅芙。
