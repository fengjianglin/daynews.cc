經典街機《合金彈頭》奇特的死亡方式，就算死也要留下福利

天天要聞
2019-12-19

《合金彈頭》的畫風一直深受玩家們的追捧，遊戲的風格正是由「世界第一的像素畫師」Akio親自操刀，而音樂則是由冷牟田卓志來完成。其實很多玩家都注意到了，《合金彈頭》初代中的人物表情包和親和力，是後期的版本無法達到的。《合金彈頭2》之後的版本增加了各種戰鬥元素，將一些額外的人物表情包和動作進行了刪減。不過值得慶幸的是，後期的版本中也加入了大量新的搞笑元素，其中就包括人物的死亡方式。

玩家們在不同的場景中，死亡的特效都是不同的。當年我們常說的「有一百種死亡方式」，在這款遊戲中被體現得淋漓盡致。很多時候玩家在被打死之後，雖然會心疼一條命，但是看到這些特殊的效果，感覺死也是值得的。在模擬器時代甚至有玩家為了看到被腐化的場景，還專門跑到這些場景中去死幾次。反正命不值錢，隨便死。

這些看上去原本非常驚悚的場景，但是在合金彈頭Q版的畫風之下看不出有什麼恐怖，反而覺得非常有趣味。這就是屬於《合金彈頭》系列的魅力啊！

那麼，在遊戲中到底有多少種奇特的死亡方式呢？

大魚吞食

這個畫面僅僅出現在《合金彈頭1》之中，而且只能在《合金彈頭》第二關的大橋場景中才會出現。出現的幾率比較低，幾乎是四分之一。也就是說，玩家掉下去四次才有可能見得到一次。被魚吃這個畫面相信當年很多玩家都是經歷過的吧！甚至我們還曾經抱過僥倖心理，或許掙脫大魚就可以活下來。但是隨著大魚沉了下去，我們的希望也破滅了。

其實在《合金彈頭2》中的ROM文件中也是有這個大魚素材的，但是並沒有合適的場景可以使用。

電擊而亡

在《合金彈頭2》中加入了木乃伊設定，玩家被連續噴兩次就死了，特效是被紫色的火焰燒死的。在BOSS處，玩家要是被電擊中的話，就會出現非常搞笑的表情包。頭髮頭燙直了，眼睛一副驚呆樣。

估計很多玩家曾經被電死過，因為這一關是可以卡死BOSS的。但是技術不好的話，就會被電擊而死。

蝗蟲啃咬

在合金彈頭好幾個場景中都曾經出現過蝗蟲，第一個場景就是在輻射島上。這裡的爬行動物和昆蟲都變成了龐然大物，甚至可以將人直接生吞活剝了。

蝗蟲在核爆炸中倖存了下來，變成了巨大的飛行生物，大量吞食島上的生物。之後叛軍將抓捕的蝗蟲帶到了秘密實驗基地。到了四關之後我們就會發現實驗室的場景已經失控了，無數的食人花和蝗蟲被放了出來。

玩家被蝗蟲抓住之後，就已經失去生機了，就算是有隊友在也救不了。值得一提的是，這個場景中的G彈正好克制陸地和空中的怪物。

飛魚搶食

我們在合金彈頭3的第一關降落時就落在了大海之中，非常慶幸的是那裡沒有食人魚，要不然是不是落地就死了。在上船的場景中，玩家要是掉入大海中就會被一群食人魚搶食，在這種情況下估計連渣都沒有了吧！

這些食人魚擁有了飛行能力，在BOSS場景之前會拚命的向玩家飛來。玩家需要把握時機進行射擊，就可以得到大量的彈夾。

食人花

實驗室失控了，所有的食人花和蝗蟲飛了出來。附近的叛軍能逃的已經逃了，沒逃掉的則成為了食人花的養分。在實驗室外面到處都飄落著種子，這些食人花的種子同樣是致命的，玩家只要沾上就必死無疑。這些種子一旦落在地上，有可能迅速生根發芽，然後進化成為新的食人花。

玩家要是跳起來沾到食人花立刻腸穿肚爛，屍骨不存。

要是在地面被食人花種子擊中，直接就會被同化，身體的養分迅速吸干。整個身體直接變成了植物的根系，然後再生出一隻完整的食人花。這個場面被玩家們認為是最驚悚的死法。

食人花是不少玩家的噩夢，無論在幾代出現都是一樣的兇殘。如果是手槍子彈的話要打好幾槍才死，如果是用刀劃的話估計拼不贏食人花。而且一旦被一群食人花圍觀，甚至連站的地方都沒有了。只能通過跳躍躲避。運氣稍微不好一點就被咬死了。

食人花吞食的死亡特效非常細膩，只剩下一隻手還在外面，而食人花還要咀嚼。

食人花的母體是無法移動的，靠通過這裡的生物補充給養。

她們的觸手比較長，而且非常靈活，玩家只要走這裡過，沒先將其打死就有可能被抓住。而它們抓住人之後，有一個吮吸的動作，而且是將人物衣服脫乾淨了才吞下去。

腐化而亡

在《合金彈頭》中這種效果有不少場景都是可以打出來的，其中就包括蝸牛的酸性噴射和地下迷宮中的酸性溶液。如果是男性角色被腐蝕的話，就會現出頭蓋骨，場面非常慘烈。而要是女性被腐蝕的話，場面就稍微和諧一點了。

而且跳起來被擊中，和在地面被擊中的效果是不一樣的哦！哪一種方式才是你的最愛呢？

當年為了看這個效果圖，多少玩家曾經故意在這裡死幾次

另外，在《合金彈頭》最終關中，那些複製人也會出現這種情況。大量的複製人因為本體消失了，出現崩壞的狀況。然而有部分複製人繼承了主角之前中的喪屍病毒存活了下來，並且擁有了喪屍的能力。

同化金幣

這可能是最貴的死亡方式吧！只要死了身體就會爆出大量的金幣。而玩家要是還有命的話就可以直接增加十幾萬分。這算是對掉命的小小補償吧！

不過一般只有新手玩家才會被這樣幹掉，因為BOSS出招之前是有明顯暗示，而玩家也是完全來得及躲避。

燒成骷髏、凍成冰球

這或許是最慘的死亡方式吧！人物直接被活生生的燒成了骷髏而死。

出現在《合金彈頭4》中的死亡場面，不過相信很多玩家都沒有遇到過吧！到底被誰殺死才會出現這種情況呢？小編會在之後的章節中詳細介紹一下。

另外，在《合金彈頭4》中還有被冰凍的效果。這其實是出現在《合金彈頭2》最終關的死亡特效，在《合金彈頭4》中被繼續沿用。只要被凍成冰，就已經宣布死亡了，救不活的。

飛刀斬殺

《合金彈頭2》和《合金彈頭4》中都會出現的場景。在《合金彈頭2》中的阿拉伯人，善於使用這種武器。而到了《合金彈頭4》中出現了會使用飛刀的海盜，同樣可以讓玩家這樣被插死，場面看上去還是比較殘忍的。要不是這種Q版畫風，這估計就成為血腥遊戲了。

其實還有幾種死亡方式，但是和上面的比起來就顯得不值得一提了。《合金彈頭》至今已經走到了第七部，不過估計很多玩家都能感到《合金彈頭》已經失去了最初的原汁原味。
