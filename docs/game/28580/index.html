IG二隊帶不動小樂言！最差數據MVP彈幕滿屏問號，全靠同行襯托

2019-12-22 1,711條評論 2,652次閱讀 617人點贊

今天12月22號英雄聯盟德杯，由IG戰隊對抗DMO戰隊。IG戰隊組成了一個全新陣容，S9首發打野小樂言帶著前VG雙人組以及二隊中上新人出戰。前VG下路Puff不愧是韓服千分王者，第一把賽娜天崩局絕境翻盤，第二把金克斯也是打出了超強壓制力，看上去不遜色於阿水！但小樂言的發揮就很差勁了，連續兩把悶頭刷野，團戰下飯操作直接白給，被青訓隊員吊打，粉絲對他的耐心值已經快耗完了。這場對抗DMO戰隊的比賽，小樂言能否為IG戰隊找回節奏呢？讓我們看看雙方陣容！

IG戰隊選用了賽娜、塔姆、蜘蛛、佐伊、劍魔；而DMO戰隊選用了女槍、盲僧、泰坦、蛇女、青鋼影。

3分鐘DMO盲僧游中，但他Q空了，中了睡眠，還被蜘蛛反蹲，因此送給了IG佐伊一血。好在DMO青鋼影提前趕到戰場，擊殺佐伊，IG劍魔被迫交出TP保蜘蛛。

7分鐘DMO戰隊把控住河道視野，趁著下路推線擊殺水龍。8分鐘DMO野輔聯動，游到上路越塔強殺劍魔。此時IG佐伊及時趕到，遠程過牆睡住了盲僧，DMO盲僧交出閃現拉開距離，仍被佐伊QR閃現點燃，一套秒殺。

11分鐘IG蜘蛛在河道排眼，DMO三人蹲伏野區想抓他，泰坦閃現出鉤發起先手。但IG蜘蛛反應很快，飛天躲鉤，閃現拉開距離，塔姆跟閃將他吞下。隨後IG蜘蛛換掉了DMO中野，順便控下了峽谷先鋒。

12分鐘IG戰隊釋放峽谷先鋒撞了中路一頭，回頭再拿火龍。此時DMO戰隊趕來接團，戰場卻分離了，盲僧入場踢飛蜘蛛，其他三人卻在集火落單塔姆，把他賣掉了。隨後IG戰隊繼續開打火龍，DMO三人在青鋼影回去收線的情況下，仍在河道猶豫不決，三人只換掉了蜘蛛。

15分鐘IG戰隊越塔強殺泰坦，連推中路兩個塔。19分鐘小樂言控下第二個峽谷先鋒，推掉上路一塔，DMO戰隊趁機拿下土龍。之後IG戰隊持續推線壓制，又控了一條土龍。25分鐘IG蜘蛛進野區反紅BUFF，卻被女槍搶了，自己還被盲僧逼出閃現。同時IG劍魔在下路與青鋼影單挑，賽娜開大，塔姆開車支援，將DMO青鋼影擊殺。

28分鐘IG戰隊開始逼大龍，DMO青鋼影推掉下路二塔，再TP趕來接團。此時DMO泰坦閃現出鉤，再加上女槍大招，但其他隊友都跟不上，只能將他賣掉。隨後DMO青鋼影推掉中路一塔，沒有及時撤退，也被留下了。

於是IG戰隊以四打五，又開始打起大龍。此時DMO蛇女閃現大招，嚇出了小樂言的懲戒，DMO盲僧再入場搶下大龍，換掉小樂言，還拿下了第三條小龍！

31分鐘DMO戰隊在中路開團，強秒塔姆，再逼高地。此時IG蜘蛛強行出手，EQ打殘泰坦，然後被女槍大招掃死，IG劍魔也沒能砍起來。好在DMO青鋼影衝動懟臉，被IG賽娜反殺，使得IG戰隊保住了中路水晶。

35分鐘第四條小龍刷新，DMO戰隊佔好視野，直接開打。IG蜘蛛閃現入場沒搶到，直接被秒殺。隨後DMO戰隊追死塔姆，推平中路，回頭拿下大龍，又弄死了劍魔。

此時DMO戰隊已經擁有了土龍護盾，不怕佐伊賽娜的POKE，IG戰隊陷入絕境。DMO戰隊帶著大龍BUFF分推，推掉了下路高地，青鋼影獨自推掉了上路高地塔，還差點強殺了劍魔。

IG劍魔已經打不過青鋼影，也清不動超級兵，只能縮在基地前守家，其他四人想在中路防守遠古巨龍。等到超級兵走進高地的時候，DMO青鋼影突然入場，配合女槍TP，直接偷掉了IG基地，DMO戰隊贏下了本場比賽！

這場比賽DMO打野盲僧的發揮很糟糕，帶起反向節奏，甚至不如不做事的小樂言。但小鵬盲僧後續搶龍立下了大功，吹響了反攻號角，足以彌補前期的失誤，因此拿到了本場比賽的MVP。然而當小鵬被評選為MVP的時候，直播間彈幕全是問號，觀眾都在表達不解。

這個盲僧戰績0-5-6，傷害佔比10%，參團率只有50%，甚至落後小樂言一千多經濟，這可能是史上數據最差的MVP了。也有網友分析了一下，表示這是官方在搞事，暗示小鵬全靠同行襯托，其實這個MVP應該頒給小樂言的！

小樂言的蜘蛛每次團戰都被秒，要不是塔姆多次吞他，估計他都超鬼了。上單劍魔雖然對線崩盤，但他團戰操作還是蠻不錯的，輸出打得也不低；中路佐伊瘋狂Carry，幫助隊伍推塔滾雪球，卻因土龍護盾而報廢；Puff賽娜一次都沒死過，6-0-9戰績無奈躺輸……IG戰隊前中期佔據這麼大優勢，就因為小樂言這個突破口而被翻盤，塔姆保了他這麼多次都沒用，實在讓人絕望！

小樂言沒節奏就算了，還手抖瞎操作，被當場懲戒搶龍教學，丟掉了遠古龍魂和大龍，連紅BUFF都保不住，這才導致IG戰隊突然崩盤被翻。明年IG戰隊真的得好好考慮一下，到底小樂言還能不能待在一隊，也許他要回次級聯賽繼續磨練吧！
