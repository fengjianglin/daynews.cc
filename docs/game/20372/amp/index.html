雲頂來襲，影之魔法：六影極限輸出，點頭小法再現江湖，點誰誰死

天天要聞
2019-12-16

各位召喚師大家好，小皮雲頂來襲又來了，雲頂之奕9.24版本更新後，影羈絆得到更新升級，人數要求由原先的2/4調整為了3/6，羈絆激活人數得到提升，增大了獲得頂級特效的難度，但是也將傷害提升變更為了70%/140%，並在任一影羈絆英雄參與擊殺後刷新狀態時長。總體上來說這個羈絆激活難度被提高了，但是激活後獲得的狀態提升也得到了加強，今天小皮為大家推薦的這套陣容就是以六影為核心，利用影羈絆的高額傷害加成，開局清場流。

英雄配置：塞恩，小法，賽娜，吸血鬼，劍聖，千珏，馬爾扎哈，辛德拉

羈絆特效：

羈絆上六影是核心羈絆，140%的傷害提升，任何影單位的輸出都是不容小覷的，這次陣容主要是影羈絆與魔法師的搭配，同時雙海洋的法力值回復可以讓三位魔法師的法力值回復更快。

裝備配置：

小法：熾天使+熾天使+復活甲

賽娜：鬼書

劍聖：火炮+羊刀

小法師是主C，因為他既是影單位也是魔法師單位，三魔法師的羈絆特效讓其有50%概率再次施法，六影的高額傷害加成，讓其一個大招就是2.4倍傷害，觸發魔法師羈絆就是4.8倍，輕鬆帶走敵方前排與C位（同階水晶秒不掉），雙熾天使讓小法開完大後迅速再次滿藍，瘋狂扔炸彈，只要不鎖定三星蠍子，那麼輸出就是停不下來的，第三件裝備可以選擇一件防禦裝或者一個回復裝備，增加續航能力，追求極限輸出的話也可以考慮大帽子等輸出裝。

賽娜裝備鬼書，可以大範圍附帶重傷效果，同時為隊友增加額外傷害，鬼書對於賽娜是個神器，劍聖選擇羊刀加火炮，如果裝備允許再給到一個回復裝，劍聖一人就能晉陞為另一個主C。

陣容站位：

賽娜放最後面的角落，塞恩放在前排里賽娜最遠的地方，這樣賽娜每次釋放技能都會鎖定塞恩，能夠輻射到中間的所有友軍。

陣容過度：

這套陣容對於金鏟子沒有額外需求，因此不需要放血到最後一名，我們主要需要的裝備是藍水晶，先滿足小法的裝備需求，保持中等排名即可，使用海洋德魯伊打工，前期過度很強，中期五人口就能湊出海洋影法，陣容過度十分平滑，小法中期必須升二，因此存夠50利息後就用利息D牌，小法一旦升二就不要D牌了，存利息升人口，到八人口開始找劍聖，劍聖一到，陣容就穩了。

前期開局：吸血鬼，辛德拉，大樹，翠神

中期過度：小法，賽娜，馬爾扎哈，吸血鬼，辛德拉

陣容分析：

這套陣容不像其他陣容對金鏟子的硬性需求，裝備上的要求也不是很高，唯一的難點在於如何過渡升級到八人口找到劍聖，因此前期快速攢到50塊利息十分重要，連勝是個很好的金幣來源。前期德魯伊海洋的強度挺高的，因此這套陣容適合中期暗影牌刷新較多，且小法師快速二星的情況下可以嘗試該套陣容。
