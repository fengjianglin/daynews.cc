冠軍之夜 | RSG全員空降，12月27日19點冠軍之夜開啟

2019-12-23 1,044條評論 1,987次閱讀 394人點贊

虎狼之師，勢不可擋！恭喜RSG在2019S聯賽秋季總決賽中，以3:0的比分成功擊敗王者之師QG，奪得總冠軍！

在這場時隔了兩個賽季的宿敵對決中，無論是競速中RSG.東方和RSG.小鈺神勇的發揮，還是道具組的默契配合，RSG都展現出了虎狼之師的意志。

選手回顧奪冠時刻 慶祝2周年

為了慶祝新王加冕，12月27日19點，RSG俱樂部舉辦冠軍之夜線上見面會，屆時由解說游游和澤斌帶我們一起嗨翻冠軍之夜！

身為坐在台下或者直播間前的觀眾，看到RSG他們在比賽中每一幀的對抗，都能直觀地感受到那種競技的刺激，那選手們在當時又是什麼樣的心境呢？想知道他們當時語音里是怎麼溝通的嗎？12月27日晚，冠軍之夜告訴你！

當然啦~我們也準備了許多粉絲比較好奇的問題會在直播間現場問隊員們，俱樂部各種八卦趣聞，車手們確定不來一探究竟嗎？！

與粉絲玩轉歡樂小遊戲

冠軍之夜怎麼能少得了與粉絲互動的環節呢？當晚直播中，飛車知名主播杜辰浩還將作為粉絲代表，與選手們展開歡樂互動和小遊戲。

有遊戲當然就有獎懲，橘妹已經迫不及待想看到老司機們被懲罰的樣子啦~當然，觀眾的福利也有！直播期間都會有抽獎環節，有機會獲得豐厚的大禮，車手們準備好了嗎？！

2周年嘉年華版本登場

其實隨著這賽季畫上了圓滿的句號，我們遊戲也將迎來新版本更新，相信車手們已經在今天的全明星生存賽中體驗到了2周年慶版本全新玩法與賽道帶來的樂趣，刺激的航海賽道究竟有什麼「衝浪」秘籍？

屆時，選手們將會在直播間中與我們分享他們對新版本的看法與心得，讓我們相約冠軍之夜！

歡樂水友賽 趣味不停歇

了解完新的地圖和賽道，接下來也是到了大家最期待的環節了，RSG成員會輪番上陣與水友們一起開黑比賽。

道具模式、競速模式和邂逅模式統統都有，贏了有獎勵，輸了的話還有超級有趣的懲罰，據說還有選手撒嬌…橘妹已經開始有畫面啦！

最後，再次恭喜RSG俱樂部奪得本次S聯賽秋季賽總決賽冠軍，競無限速！讓我們相約12月27日19點的冠軍之夜！
