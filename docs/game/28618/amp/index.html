王者榮耀：曾經的“上單一哥”，S17成排位熱門輔助，玩法一覽

天天要聞
2019-12-22

大家好，這裡是鴿神。

王者榮耀S17賽季，曾經的「上單一哥」孫策在對抗路卻並不吃香，面對呂布、亞瑟、曜、老夫子等英雄，孫策很難打出優勢。然而，因為孫策的技能控制能力強，自身坦度不錯，所以走輔助位倒成了這版本孫策的主流玩法，在排位中也十分熱門。那麼輔助孫策的優勢和具體打法如何呢？跟鴿一起來看看。

輔助孫策的優勢

孫策這個英雄開團能力和抗傷能力極強，在選擇輔助路線時有著非常不錯的快速支援能力，並且可以幫隊友提供長時間的連續控制。由於被動效果的存在，在前期被動疊滿時直接多出200點物理護甲，配合被動的續航能力，在前中期讓孫策有資本瘋狂的騷擾敵方射手和刺客，幫助己方打野提供給力控制和補充傷害。

出裝銘文推薦：

銘文攜帶：10隱匿10鷹眼10異變。選擇常規百穿銘文，放大孫策前期對敵方英雄的傷害，讓其在前期反野抓人時能夠提供足夠強力的輸出。

出裝選擇：極影+抵抗之靴+極寒風暴+反傷刺甲+魔女斗篷+碎星錘。

由於選擇輔助路線，在裝備的購入選擇上一定要精打細算，前期選擇鳴鳳指環和抵抗之靴快速參與團戰，獲得一定資金後入手極寒風暴和神隱斗篷。將指環升級成極影后再將合成費用斷層比較厲害的反傷刺甲購入。後續慢慢補齊所有裝備。

玩法思路：

孫策的召喚師技能選擇暈眩，在前期時可以進行反野/或線上蹲人，孫策前期1~2級強度驚人，我們在輔助的時候不會被兵線牽制，因此可以大肆遊走，如果隊友英雄是橘右京、雲中君這類的強勢打野，我們可以肆無忌憚的入侵野區。也可以1級幫射手建立優勢，如果對面是當前版本強度比較超標的孫尚香、馬可波羅、成吉思汗等，我們用孫策去參與射手之間的戰鬥很容易就能幫射手拿到1血。

4級以後的孫策遊走能力更加驚人，在抓人時我們要注意大招和召喚師技能的情況，大招航行時間越遠，撞擊敵方英雄後能夠提供的控制時長就越久。因此我們的長距離突襲的控制效果甚至要好過一直賴在線上，在大招命中前盡量下車，因為貼臉撞人後孫策自身也會陷入一定的後搖僵直，提前跳車也可以保證有足夠的距離進行1技能的銜接。

後期團戰孫策就是團隊中的開團好手，也可以利用孫策的支援特性進行41分帶，在隊友被開團後直接大招沖入團戰中心提供控制。如果選擇打反手，孫策也有著非常不錯的保護能力，需要注意的是要和需要保護的隊友拉開一定的距離，讓你能夠在隊友被切時瞬間打出一個完美的1技能銜接。另外孫策使用1技能擊飛敵人後瞬間開啟大招，有可能出現直接穿過敵人開船走的現象，因此小夥伴們在保護後排時需要注意這個問題。

總的來說孫策這個英雄對抗射手還是非常有效果的，在對抗路和戰士對拼還是有一點浪費。因此如果想玩孫策針對射手時，用孫策打輔助或者打野位是現在這個版本不錯的選擇。

以上就是王者榮耀S17賽季輔助流孫策的玩法技巧啦。我是鴿神，一個專註遊戲的原創作者，想了解更多遊戲資訊攻略就點個關注吧！
