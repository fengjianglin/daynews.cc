9年成長為一線人族，浪漫表示奪冠的劇本

2019-12-23 2,570條評論 3,654次閱讀 2587人點贊

九年，對應普通玩家來說足以更換很多遊戲。但是放在職業圈的老將身上，卻是他們的全部。九年時間，足以讓他們從零啟動、步入輝煌，又或者是曾經閃耀、黯淡離場。

九年，也是浪漫從一個魔獸門外漢，成長為頂尖選手的時間。

在電子競技中，衡量一個選手有沒有天賦打職業，要看他從0基礎，打到成為業餘高手需要多長的時間。如果這個時間少於1年，那這個選手就及格了。「我從11年開始打魔獸，12年加入業餘戰隊，14年獲得NESO的殿軍。」如果套用上述公式，浪漫似乎也能滿足電競選手對天賦的苛刻要求。

當說到他是怎麼成長起來的時候，浪漫侃侃而談。

「我的偶像嗎？沒有吧，如果有的話，那就是肯哥，他讓我知道了我也是有實力打職業的。另外Happy也算，是他讓我堅定了方向」

「另外就是不斷的練習了，努力做好每一步，認真對待每一句比賽來提高自己的實力，不要太在乎外界的干擾。我的絕招的秘訣就是把細節練習到極致。」

看似平常的回答，竟跟倒油翁的「無他,唯手熟爾」如此相似。在戰隊賽啟動之後，各種戰隊都在招兵買馬。去年年底，Newbee就連續簽下了120、Lawliet和Lyn，銀河戰艦已經構築完畢。今年年初，RW豪氣衝天，一口氣簽下Fly、000和Life，打造全華班頂尖豪門。

而起步較晚的其他戰隊，看似已經很難在戰隊賽中擁有強力的競爭能力了。不過，只要世界上有伯樂，還是會不斷有千里馬的產生的。LP戰隊的經理呂峰，就是這麼一個角色。連續簽入贈品馬彩色，上等馬FoCuS，他又找到了浪漫。

「我們經理在大家都不看好我的時候給予我肯定，這個難能可貴。並且隊伍裡面有彩色和佛爺，他們實力都很強，我覺得這會是一個不錯的陣容。」

「隊長福克斯平易近人，彩色勇於承擔責任，k神是戰隊的大腦，各種細節和思路的發起者，15歲未來的新星。至於呂峰？輸了肯定他背鍋啊，贏了就他請客，哈哈。」

說到這，浪漫就如賽後擊敗了NB一樣露出了微笑，看來LP戰隊的氛圍，在眾多俱樂部里也算得上不錯。除了在平時嘻嘻哈哈，LP的凝聚力更表現在戰隊賽進行的那一瞬間。

「每個人當時的眼神都非常堅定，讓我來，交給我，我能贏。」浪漫說到這，頓了一下，「但是也有不太好的情況，就是我和FoCuS的交流還是有點困難的，不過正在好轉。」

戰隊發揮出色，但是浪漫的個人表現更加不容忽視。兩次擊敗頂尖亡靈單騎救主，對於這些評價，浪漫表現出了沉穩和謙遜

「總體而言人族打UD是偏弱勢的，要贏必須做到100分，而UD不需要100分就可以贏，人族容錯率較低。出坦克我是因為對手的戰術選擇作出的決定。」

關於贏RW的劇本，賽前我就有構想過這個劇本的存在，但是更多的還是希望聽一下浪漫的說法。「劇本嗎？贏RW的劇本就是好好訓練，做好充足的準備。」一句普通的話，透露出他們的信心。最後，我們一如既往的提到了讓選手放一句狠話。浪漫卻出乎意料的低調，浪漫表示「沒啥狠話啊，在我眼中每個對手很強，好好打就是了。」

結束了採訪，我特意去打開了一下大帝的空間，回顧當時浪漫被各種戰術花式養豬的視頻。視頻還在，可惜當年那個14級小人族已經蛻變成漫皇了。

現在，浪漫平時除了訓練，就是直播和賣大閘蟹。在廣告屆，又一顆新星在冉冉上升。
