隨著天氣越來越寒冷，作為最真實的江湖《逆水寒》也將迎來它的冬季更新。這次的冬季新資料片定為「踏雪尋梅」，和新資料片同時上線還有新伺服器的開啟。那麼在這次的更新中又會有哪些好玩的東西呢？小編今天就帶大家一起來看一看！

首先在這次更新中變化最大的就是「全新荒島玩法」，自從「吃雞」遊戲火了之後，很多遊戲中都引進了這一玩法《逆水寒》也不例外。那麼在這次更新玩之後整個「荒島」將會進入冬季，開局的時候大家都是一無所有，只有落地之後才能在偌大的荒島撿到一些物資裝備，而玩家則需要利用撿到的武器和敵人鬥智斗勇。並且除了要和對手對戰之外，玩家還需要時刻準備好迎接大自然的考驗，有時候自然的力量才是最可怕的。

對於不喜歡競技的玩家來說，除了荒島新玩法之外，原來的玩法也是會保留，並且這次還會加入更複雜的建造模式，希望各位玩家可以為荒島帶來文明的曙光。除了更新之外最近《逆水寒》在打擊外掛的力度上也是非常的強，甚至不小心誤封了很多玩家的正常號。不過官方打擊外掛也是為了能給玩家創造更好的遊戲環境，如果被誤封了記得及時向官方反應哦。

除了這些之外新資料片最大的一個「公益」更新值得大家注意，全新地圖「青天寨」將會解鎖並且會推出苗族主題時裝，該時裝賣出的錢將會全數捐給苗寨用於兒童教育事業，在遊戲中也能為扶貧盡一份力！想要了解更多的遊戲資訊那就關注小編吧！
