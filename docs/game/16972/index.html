英雄聯盟：碰見排位隊友秒選這4個英雄，你多半是遇到“代練”了

2019-12-15 3,069條評論 3,992次閱讀 1170人點贊

相信大家在打排位時最不喜歡的就是碰見「代練」了，我們這裡也不是提倡「代練」的，但是有些代練當我們排在隊友的時候還是挺爽的可以躺一趟，但是如果代練如果在對面的話就非常影響遊戲體驗。有些時候在排位時碰見一些玩家選接下來的這幾名英雄的話，多半是碰到「代練」了。接下來讓我們看看這幾位英雄都是誰吧。

吸血鬼

大家都知道吸血鬼是一個非常後期的英雄，只要到後期就是一打五的存在，前期在線上非常的弱勢，補兵都非常不好補，尤其是對線期，但是在低端局又很少看見吸血鬼的身影，因為這個英雄的操作難度也是一樣的高，一般的玩家也是操作不好的。

說到後期一打五，基本發育好的吸血鬼大招配合w技能進場，秒掉脆皮輕而易舉。但是很多玩家都不會玩吸血鬼，特別是用他來對線。如果是「代練」來操作吸血鬼的話，吸血鬼可不見得是後期英雄了，前期都有可能給你打崩對線，所以在中低端局能秒選吸血鬼的，要麼是熟練度很高的專精玩家，要麼就是「小代練」。

豹女

接下來就是豹女，相信大家都聽說過一句話「十個豹女九個代，還有一個代中代」，就可以知道豹女這個英雄多受代練的喜歡了，因為豹女是一名拿出來就要C的英雄，不讓豹女就沒有一點用，大家都知道這個英雄傷害非常高，一個q技能就能消耗你不少血。

並且這個英雄的操作難度還是很大的，不是專精的玩家都無法很熟練的操作這個英雄，而且豹女這個英雄gank的時候非常依賴q技能的命中，不過會玩的豹女是不會輕易放Q的，一般會用平A騙你的走位，但是一旦被Q到，就不好跑了，所以豹女是「代練」非常喜歡使用的英雄。

蛇女

蛇女在9.23征服者改版後更是迎來了新的強勢期，對於「代練」來說可是史詩級加強，當然對於主玩蛇女的玩家更是史詩級加強，以前蛇女在低中端的排位中是很少見的，現在可能因為版本原因玩蛇女的玩家會多起來，但是這絲毫不影響「代練」對這個英雄的喜愛。

蛇女被譽為法術機關槍，她的傷害十分可觀，在後期的比賽中有很強的統治力。但是這個英雄對於走位的要求很高，E起來就是「法術機關槍」中低分段少有能掌握的玩家，對於R的釋放更是難上加難，但是代練很喜歡選這個英雄，順風起來可以很快結束遊戲。

瑞茲

不僅僅蛇女在9.23征服者改版後更是迎來了新的強勢期，瑞茲也同樣是等於變相加強了。

瑞茲以前就有著雷電法王的名號，現在沒有以前那麼誇張了，但是依舊是非常的猛。瑞茲和蛇女很相似，順起來前期後期都能carry，熟練度不高的人玩起瑞茲，應該連招都摸不透，但是在代練的手裡，這個英雄真的是可以一打五，順風一挑五，逆風能和對方一換一，代練在操作這個英雄時的走位也是非常風騷的。

其實很多人都是抵制這款遊戲中有代練的存在的，但是那款遊戲沒有代練呢，我們能做到就只能提高自己的實力，在碰到代練的時候，能夠戰勝代練。
