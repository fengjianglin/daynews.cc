「2019德瑪西亞杯前哨站」21日A組賽事前瞻FPX、VG、WE、SN、LGE

2019-12-21 421條評論 1,483次閱讀 91人點贊

【2019德瑪西亞杯前哨站】A組FPX、VG、WE、SN、LGE

【FPX】選用了自己LDL陣容，FPB在LDL春季賽常規賽中14-10第9，夏季賽常規賽中14-10第11
但是兩季季後賽中他們都晉級到了半決賽，常規賽中唯唯諾諾,季後賽中重拳出擊，季後賽還戰勝了非常靠前的隊伍，這支FPX的青訓隊伍FPB實力不容忽視，特別是這些人員一部分都晉陞到一隊

【VG】是個不缺錢的主，絕代雙「蕉」+韓國「小酋長」+LCK訓練生，這個牌面實力可算中上游水平，如果你認為這支隊伍年輕那你就錯了，我們有理由相信世界上最頂尖的LOL教練「力宏」會將這支隊伍調教得很好

【WE】在新賽季各隊重新排列組合和補強的大環境下，WE算是一股清流，繼續沿用自己的青訓體系，甚至自己的雙C離隊，額外買來了一名LNG PLEX中單選手，這支隊伍每名選手都顯得沒有那麼有「名氣」，我覺得這真的可以算是比較年輕的隊伍，不知道有沒有化學效應

【SN】送走了雙C MAPLE和SMLZ，一名國產上單新希望「BIN」再加上越南「悍匪」SOFM的加入，我認為這個陣容並不能說SN變弱了，我認為他補強了上野，中單ANGEL一直發揮都還不錯，操作不輸MAPLE，頂多大賽經驗遜於MAPLE，這個版本對ADC本來就不友好，SMLZ的離開沒有想像中那麼對隊伍不利，所以我認為SN總體來說是補強了整體實力

【LGE】如果沒記錯的話這個隊伍雙C是打過LPL的，加上新任教練丁皇的加入，顯得在LDL隊伍中還是有點實力，這支隊伍在今年春季賽常規賽獲得13-11排名第13無緣季後賽，夏季賽獲得16-8第6季後賽止步首輪0:3輸給了FPB，但現在的陣容是補強了之前的陣容，不容忽視的實力
