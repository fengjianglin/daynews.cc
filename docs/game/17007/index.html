C羅很難抽？看了這個，助你輕鬆獲得C羅等黑金球員，征戰賽場

2019-12-14 239條評論 2,687次閱讀 658人點贊

《實況足球手游2020》終於是在各大平台開始公測了，對於很多小夥伴來說，如何能抽到自己喜歡的球員，估計是最令人頭痛的了，特別是像C羅、梅西這樣的頂級球員，更加難抽到。咳咳，小編不才，摸到了一點門道，給各位小夥伴分享一下。

看看小編的這個陣容，我自己的有點怕！好了不說其他的了，轉入正題。首先，各位小夥伴都知道，在《實況足球手游2020》中，想要獲得球員有兩種途徑，球探或者經紀人，哦還有拍賣也能獲得。想要獲得C羅梅西的關鍵點是，只要你擁有合適的球探，你就能穩定合出你想要的自由球星！當然，在球探與經紀人中，使用經紀人會更加容易獲得C羅。

不過，並不是每一個球探都能獲得C羅，要抽到C羅，對球探的要求也是有著非常苛刻的要求的。相信大家在打開球探項目的時候會有一個疑問，球探上面會有星星，有的多，有的少，這是為什麼？

額，小編有點非，球探今天沒來，只能用拍賣的球探來解釋了。這樣說吧，在上圖中，都是球探上面都是三顆星星，如果我用了他們，他們就只能給我符合這個標準的最多至銀球的球員了，那麼接下來的就很一目了然了，作為黑金球員的的C羅，五顆星是必不可少的。

不過，在平常我們所打的聯賽當中，所獲得的球探，能夠出現五顆星的概率是非常低的，那麼我們需要找高端的球探就只能去拍賣行看看了。拍賣行的規則很簡單，兩小時內出價最高的玩家就能獲得該球探，當然球探星級越高，價格也就越高了。

當然，在如此龐大的拍賣行數據面前，找心儀的球探也是非常麻煩的一件事，不過好在《實況足球手游2020》很好地解決了這個問題，我們可以看到在球探搜索條件這裡，我們逐一選擇相應的選項，就能夠更加精準的找出自己想要的球員。

打個比方，C羅是葡萄牙人，並且踢的是中鋒或者邊鋒（看選項選擇），身高在185以上，用這三個條件，就會有很大的幾率能夠找到C羅的球探。

不只是球探可以抽到C羅，經紀人也是一樣可以抽到的，不管是特殊經紀人，還是標準的經紀人，都有一定的幾率。當然兩種經紀人有不同的選擇，幾率也是不一樣的。不過在最新的《實況足球手游2020》中，C羅的獲取方式暫時只在標準中可以找到，這也為玩家節約了不少花費。

小編覺得，標準的是最精準的了，因為你無論如何都能得到自己想要的位置的球員。並且球員的能力都是很高的，在得到自己想要的球員前，也不失是一個過渡的好選擇。不過為了慶祝公測，官方也給玩家帶來了超多的福利！據說官方會有大行動來慶祝公測！13日至15日期間每晚20:20登陸遊戲，就能獲得公測大禮包，絕對有你心動的！

而且不只是有在線禮包喔，在12月27日至1月2日期間，玩家還可以通過歐洲豪門挑戰賽，只要踢進四球，參與抽獎，就能獲得簽名球衣！心動了嗎？還有更勁爆的！只要分享使用梅西的視頻和截圖，上傳到公眾號，就能參與梅西親筆簽名球衣！絕對是珍藏一生的選擇，快來踴躍參加！

《實況足球手游2020》正在火熱公測中，現在加入還能獲得各種大禮包，還能體驗刺激的公測活動玩法，快來和自己喜歡的球星一起征戰綠茵，享受足球至高的快感！
