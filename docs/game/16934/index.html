LOL：“無限R”套路火了！艾克十秒一個大招，拉克絲大招CD僅七秒

2019-12-15 1,734條評論 3,680次閱讀 2559人點贊

英雄聯盟里所有英雄的技能已經多達七百多個，而在這些技能里，大招通常是英雄最強的技能，作用也是各有千秋。LOL的職業比賽里，一個恰到時機的大招有可能直接扭轉戰局改變勝負。與英雄大招的冷卻時間相對應的，是大招的強度。一般來說越強悍的大招，冷卻時間也就越強，死歌、慎和崔斯特的大招就是如此。

LOL早期的暮光之眼

無限火力模式之所以受到玩家的歡迎，就是因為技能CD的冷卻時間大大降低，玩家們能夠品嘗到更加緊張刺激的對局。那麼在英雄聯盟的常規對局裡，有沒有可能像無限火力一樣擁有極短的技能CD，隨意釋放技能呢？

LOL「無限R」套路火了

菲奧娜老版大招：利刃華爾茲

大多數LOL玩家都有過爬段位、虐菜和想像遊戲套路的經歷。時光眨眼即逝，S10賽季也即將到來。對於大多數沒能打進峽谷之巔的玩家來說，爬段位再也沒有S4賽季的那種新鮮感了。許多玩家玩LOL不再是為了勝利，而是為了快樂。而最近，一個可以「無限R」的快樂套路火了！

拉克絲大招CD很短

每到LOL版本變更時，英雄們的平衡總會經歷一番新的排列，玩家們總能在新的版本找到隱藏的快樂套路和組合。9.23版本最大的改動莫過於小龍的屬性，四條小龍都有各自的作用，而其中最快樂的當屬風龍。

艾克十秒一個大招

兩個關鍵符文

要達到極限的大招CD，必須點出兩個符文：

  1. 終極獵人：通過參與擊殺來減少大招CD，滿層25%
  2. 星界洞悉：略微減少大招

冷卻元素龍是隨機刷出來的，因此進入遊戲之後就得看運氣了，風龍越多越好。外服可以看到一些玩家運氣不那麼好，打低端局連著七八局也碰不到一條風龍。但集齊了四條風龍的玩家，便能得到比五殺還要爽快的遊戲體驗！

艾克十秒一個大招

這位達成了「無限R」成就的國服玩家，艾克大招的冷卻時間僅僅十秒。如果這樣一個艾克再拿出金身等半肉的保命裝的話，十秒一個大招無限回血續命，如果不衝到人堆里，想陣亡都難！經過訓練模式的測試發現，擁有了四條風龍之後：

  • 死歌大招冷卻三十秒
  • 亞索絕息斬的CD不到七秒
  • 布里茨三秒一個大
  • 拉克絲七秒一個終極閃光

有意思的是，掘墓者不知是不是出了BUG，大招冷卻時間完全不受風龍的加成。

拉克絲

而拉克絲的話大招射程很遠，傷害也很高，七秒一個大招只需要站在遠處打消耗就行了。如果是在狹窄的、容易命中大招的野區地形，幾發終極閃光下去，團戰還沒開始對面就已經全部殘血。當然，這個需要運氣的套路也只能帶來快樂了，畢竟都拿到了四條風龍，何愁打不贏團戰？個人預測，過後幾個版本設計師將會削弱小龍的加成。對此，大家怎麼認為？
