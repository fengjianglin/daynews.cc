全明星Solo賽16強：小天能否解鎖“單殺Faker”成就？

2019-12-07 0條評論 17次閱讀 0人點贊

摘要

北京時間2019年12月6號，英雄聯盟全明星賽第一天的比賽已經全部結束。而除了表演賽之外，Solo賽無疑是本次全明星賽最大的焦點項目。

北京時間2019年12月6號，英雄聯盟全明星賽第一天的比賽已經全部結束。而除了表演賽之外，Solo賽無疑是本次全明星賽最大的焦點項目。值得一提的是，從第一天Solo賽中的結果來看，代表著我們LPL賽區出征的四名選手Uzi、Theshy、Doinb以及小天都順利贏下的比賽，晉級到了本次Solo賽的16強中。

不過，在這四場的比賽中，最為精彩的無疑是小天的那一場Solo賽了。因為他所面對的對手也正是在下個新賽季即將在LPL舞台上登場的選手，LGD戰隊的Peanut。所以，不少人也是將小天與Peanut之間的對決稱之為「前哨站」。不過，讓人沒想到的是，在這一場比賽中，小天與Peanut則是事先商量好，同時拿出了自己非常招牌的打野英雄盲僧。最後，小天更是憑藉著自己出色的操作完成了對Peanut的「雙殺」。正是小天出色的表現，不少LPL的粉絲們也紛紛對Peanut喊起了「歡迎來到LPL」的口號。

而在經歷了第一輪的對決之後，從這四位選手第二輪的比賽情況來看。首先是Theshy，Theshy即將在16強面對的是DFM的上單選手Evi。相信同為上單選手，Theshy的個人能力還是能夠讓許多人放心的。與此同時，一旦Theshy成功出線，他即將在8強賽中遇到的選手也並不是以個人操作著稱的選手。相信Theshy還是能夠在這個組裡輕鬆應對的。

另一組的Doinb選手則是在下一場將面對JT戰隊的中單選手Fofo。值得一提的是，這兩個人本身就在S9全球總決賽中交過手。相信對於Doinb來講難度並不是很大，而一旦Doinb成功戰勝Fofo之後，他也即將面對Pabu與Bwipo之間的勝者。值得一提的是Pabu在2018年的全明星Solo賽中就是一匹「黑馬」，不知道Doinb在這組中的最後是否能夠遇到他呢？

而與Theshy以及Doinb的分組情況不同的是，Uzi與小天恐怕將在之後開啟地獄模式。首先是Uzi，在16強賽中他就將面對自己的老對手Bang。而如果戰勝Bang之後，Uzi也有著很大的可能性會遇到天才中單Caps。而要知道的是，在2018年英雄聯盟全明星賽中，Uzi就是輸給了Caps而無緣Solo賽的「三連冠」成就。

最後，就是小天。從小天的分組情況來看，他即將面對的則是S9世界賽中的對手，G2戰隊的打野Jankos。而一旦小天戰勝了Jankos之後，他更是有可能面對Faker大魔王的挑戰。不過，值得一提的是，也正因為很有可能面對Faker，不知道小天是否能夠在職業生涯之中完成「單殺Faker」的成就呢？

各位小夥伴們，你們覺得本次LOL全明星賽的Solo King會是我們LPL的嗎？
