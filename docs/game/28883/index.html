KPL年度實時投票，夢淚穩坐第一，李九被瓶子反超，超玩會力壓E星

2019-12-21 112條評論 1,223次閱讀 316人點贊

KPL19年的兩個賽事，春季賽和秋季賽已經全部結束了，接下來就是一年的總結了，按照慣例是要搞一個年度人氣投票，評選年度最受歡迎選手，俱樂部，解說。每個人都是有投票的機會的，還沒有投票的小夥伴，可以趁著這個機會，給喜歡的選手投上一票哦。

這個投票是實時的，每一分鐘都在刷新票數，在我看到這個投票榜單的時候，夢淚是排在最受歡迎選手的一名。夢淚後面的兩個分別是貓神和一諾，而夢淚的票數要遙遙領先第二名的貓神。

不得不說，夢淚的人氣真的是高啊。明明秋季賽只上了一場比賽，人氣卻要遠遠領於打滿了的貓神，可以說夢淚的人氣比後三名加起來都要高。而前三名選手裡面，有兩個都是AG超玩會的選手，看來這次超玩會奪冠給了粉絲很大的熱情。

值得注意的一點是，往往排名很高的AT，這次只排在了第六名的位置，看來成績的不理想也導致了AT的人氣下滑。老帥和六點六雖然排名靠後，但也算是榜上有名，這次超玩會真是全員上榜啊。

最受歡迎俱樂部的第一名是AG超玩會，第二名是E星，第三名是QG，也就是大家常說的「御三家」。超玩會剛剛回歸KPL，就力壓E星和QG，成為了人氣第一的俱樂部，可見超玩會的老粉還是有很多。擁有AT的TS卻無緣榜單，這點還是讓我有點意外的。

再來就是最受歡迎的解說榜單了，這個不用看也知道，第一第一名肯定是李九和瓶子，因為其他的解說水平確實有差距，人氣也不是特別高。

這次是瓶子排到了第一，而且領先第二名的李九很多票，不知道是不是因為QG的事情，粉絲對於聯盟的處理方式不滿，才會用這種方式來進行抗議。

總之，如果不出意外的話，最受歡迎解說應該就是瓶子了，第二是李九，剩下的解說也只能爭一爭第三名的位置了。

那麼，在你心目中的最受歡迎選手，俱樂部，解說，分別是誰呢？
