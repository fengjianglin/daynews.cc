《七日殺》遊戲測評：七日一殺，能活到第七天嗎？

天天要聞
2019-12-16

生存遊戲作為大家喜愛的遊戲類型之一。經過不斷的創新發展壯大，已經衍生出了多種門類。今天為大家測評一款集合恐怖、角色扮演、塔防和第一人稱射擊為一體的生存遊戲——《七日殺》。是一款難度、玩法兼存的遊戲，曾經一度火爆於遊戲主播的直播間。

《七日殺》殭屍末日

遊戲測評：8.5/10.0

遊戲類型：末日生存

節節攀升的遊戲難度美妙。

《七日殺》的遊戲背景故事設定是第三次世界大戰之後。戰爭沒有贏家，損失與哭泣是雙方的。而經歷過災難過後也許是上天對人類的懲罰，災難過後是更大的浩劫。也許是因為核輻射，也許是源於變異。在地球出現了大量的殭屍。沒錯《七日殺》就是一款末日射擊殭屍類求生遊戲。比起生存來說求生更為妥當。因為《七日殺》這一款遊戲有難度、有操作。而節節攀升的遊戲難度正是玩家所追求的樂趣。

殭屍們

什麼是《七日殺》？

首先來說說《七日殺》最基本的遊戲操作。W、A、S、D鍵分別是前進鍵、向左鍵、後退鍵與向右鍵。E鍵是使用操作鍵、Shift鍵是保持行走、跑。M鍵地圖、C鍵下蹲、R鍵重裝武器。玩家遊戲時需要注意的以下遊戲技巧。《七日殺》是一款末日喪失題材遊戲，說到喪失。他們有著明銳的嗅覺，所以！身上帶生肉會吸引大部分殭屍的注意，吸引殭屍的還有聲音、氣味、火把光源等因素。玩家需要盡量避免。

就如《七日殺》的名字一樣，遊戲中的設定是殭屍會在七的倍數天數的夜間大量刷新攻擊玩家。在這一點上我猜連乘法口訣都不會的外國玩家一定玩起來倍感困難。而玩家就需要建造工事基地等躲避殭屍的圍殺。對於《七日殺》前期的過度方法那就是白天盡量多的收集素材。製造木塊、木刺等，而且鐵是非常重要的資源。前期一定要儘可能多的收集。夜晚就需要玩家狗在合適的住所裡面，這時候的玩家沒有什麼實質性的武器，盡量不要一個人夜戰到天亮。

排好隊一個一個來

《七日殺》牛皮的建築系統。

其實《七日殺》前期還可以找到馬路，沿著馬路會到達小鎮，不是那種大型的淪陷城市。而是馬路邊的鄉村，對於前期來說有著不錯的防禦力，足夠玩家抵擋一陣子。而且玩家自己建造房子的話不要懸空建造，會掉下來砸到玩家。一款求生類遊戲不是被餓死的，也不是被殭屍咬死的，而是被自己的房子砸死的。那可就鬧笑話了。在《七日殺》中勸各位玩家不要從高處跳下來。因為《七日殺》中真的有地心引力，很容易骨折。如果玩家想要拆房子的話可以直接砸地基，這樣整個房子會直接塌陷下來，方便得多。

打個鹿嘗嘗鮮

說到《七日殺》的建築就不得不說的是。建造容易，但是拆起來是真的難！尤其是對於一些有強迫症的玩家來說。一塊磚建錯了敲掉它要好久，所以建築的時候盡量用心一些不要出差錯。接連的差錯真的讓人想砸鍵盤。而且遊戲中有一項隱藏設定，殭屍在眩暈狀態下除卻槍支以外的武器會造成三倍傷害。建議遇到殭屍不要站擼，盡量放風箏。

謎一樣的合成系統。

可能《七日殺》中建築出錯確實處理起來很麻煩。但真要說讓人頭痛的就是科技創造系統。真心叫人眼花繚亂，不知所云。複雜的一塌糊塗。科技點的選擇性學習、物品的合成公式就像一團亂麻纏在一起。雖然沒有刀塔那樣的背裝備合成公式那麼誇張。《七日殺》的合成真的是那種麻煩的不想去合成的一款遊戲。人物死後會在提前放置好的睡袋旁重生，如果沒有放置睡袋則隨機重生在七日殺的開放性地圖中，一切都要重新開始。
