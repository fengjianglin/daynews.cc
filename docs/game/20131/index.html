爐石外域之聲：無面腐蝕者到底該怎麼削？

2019-12-18 1,672條評論 3,054次閱讀 1999人點贊

我不要你覺得，我要更多的粉塵～

●前言

大家好！我是掌游寶作者：落穎繽紛，很高興為大家帶來新一期的外域之聲，新版本剛出，相信大家都是比較痛恨新版本的薩滿的，國外的玩家們呢也是正在熱議薩滿的削弱和無面腐蝕者的削弱，那讓我們來看看外國的網友想讓無面腐蝕者削弱成什麼樣吧！（括弧中的話是作者的吐槽）

●給他一個3/3身材？

quacak：無面腐蝕者削弱建議：為什麼不給無面腐蝕者一個和他的無面兄弟們一樣的身材呢？（圖中的無面兄弟們分別是無面操縱者、無面潛伏者和奪心者卡什）

subsume_回復：我同意，現在的無面腐蝕者是4費7/7了。

san4ez13回復subsume_：你忘記給他加上了過載(2)。

Elrann回復san4ez13：不是Overload應該是Ogerload。（Overload是過載的英翻，ogerload應該是與怪物史萊克代言的麥當勞廣告有關）

看到很多國外玩家都用ogerload而不用overload可能就是因為這個廣告～但是可能也有梗，大家想了解的話可以找找～

lolwat1337回復Elann：這樣的話性價比就很高了！

SyhtheicMoJo回復subsume_：這樣的話（改成477的話）薩滿就解決了更多的麻煩了！

Cepstral回復：為何他不是牧師的卡卻要讓他擁有那麼菜（牧師卡）的身材呢？（暗示牧師的卡的身材都不行）

TheOneTrueDoge回復Cepstral：真相了

●給他一個477身材？（想peach）

eutjjkujl：無面腐蝕者削弱建議：為什麼不給無面腐蝕者一個和他的無面兄弟一樣的身材呢？（圖中是無面投火者4費7/7）

Impretendingtodowork回復：這也太強了（477身材的話），給他過載2（但不給477身材）的話還是比較平衡的。

imkappachino回復Impretendingtodowork：你沒看到薩滿的過載聯動嗎，就像穴居人，過載是正面喜效果。順帶一提，爐石傳說是免費遊戲。（F2p：Free to play）

●削弱突襲吧！

capitantelescopio：削弱建議：給這三張卡帕奇斯的待遇吧！（734突襲、無面腐蝕者和薩滿jk的突襲詞條全都砍掉，以前的帕奇斯是有衝鋒的，後來被砍沒了）

Sheepherder226回復:暴雪太喜歡突襲這個詞條了

PaperSwag回復Sheepherder226：Iksar（Team5首席平衡設計師）曾說過他有多愛這個詞條（突襲），但恐怕他還不知道突襲有多厲害吧！（突襲還是相當強的，在冒險中有一個寶藏是全體隨從獲得突襲也是一個特別好用的寶藏）。

這是第一個版本突襲卡設計成和普通模型一樣的身材（比如劍龍和426突襲野獸，一樣的身材卻多了一條突襲），有一些突襲卡過分強，以至於你不得不懷疑他們是怎麼通過測試的。

FrogZone回復：同意。在這個版本，突襲就像白給的一樣。426突襲嘲諷（角鷹獸）還有653聖盾魔免突襲龍（辟法巨龍）就是最好的例子。自從新版本出了，我就只玩競技場了。這些卡牌和其他的卡牌稍微配合一下就有很好的節奏，然後勝利的天平就穩穩的在有這些卡牌的這一邊了。

我認為突襲至少值一費。中立的衝鋒在往常大概會砍兩費，職業的衝鋒通常會削弱一費。我可以接受職業的突襲不被削弱，但是中立突襲不被削弱就過分了。

—Jester–回復FrogZone：聖盾突襲的隨從很需要狂戰斧和劇毒～（戰棋玩多的了）

swaggyrogers回復：我認為他們會這樣做的，但你需要保持冷靜。不要衝動～（rush一語雙關～）

Skelemania回復：實際上，我也同意這些改動。他們在做卡牌設計選擇的時候越來越依賴突襲了。雖然突襲不是衝鋒，但依然十分破壞體驗。

●結語

今天的外域之聲就到這裡啦，不知道大家認為薩滿和無面腐蝕者會怎麼削弱的呢？歡迎在評論中提出～如果被薩滿打得不要不要的，歡迎給文章點個踩表達對薩滿的憤怒！！！我們下期再見～（偷偷b一句，46級英語剛考完的小夥伴再看外域之聲會不會頭疼呢～～）
