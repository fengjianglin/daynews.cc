玩家自製「猛男」惡搞皮膚，獲得男性玩家一致好評！

有關注和平精英官方消息的玩家應該都知道，光子最近在搞一個時裝創作大賽。然而，遠在海外的國際服版本，也有玩家「亂入」了這次時裝創作大賽，並動用自己精湛的PS技術，對男角色進行了一番操作。簡單製作之後，便生成了一套「猛男」專屬時裝皮膚，「妖王」畫面感非常飽滿惡搞圖片一經發出，就在社交圈中獲得了大批男性玩家的一致好評。據了解，一般人帥、槍狠、喜歡剛槍的玩家，都喜歡這種類型，你是不是？

有一說一，這套皮膚還真的挺有韻味的。

體驗服更新，細心玩家發現了地圖「暗改」！「基地」疑似從海島中移除

自從老戰場更名為「和平精英」以後，老玩家應該在SS1初期體驗過「空軍基地」出生島。而當初光子為了響應空軍招飛的號召，特意在絕地海島地圖加入了很多軍事元素。再到後來，空軍基地便一直存在於絕地海島的地圖之中，可惜玩家卻繞不過光子的空氣牆阻擋，導致玩家一直登不上那個充滿好奇的空軍基地。

而近期的和平精英體驗服更新中，細心的玩家卻發現海島地圖中空軍基地疑似被光子「移除」了，不知道這是暗改還是後續版本的內容呢？

玩家化身「透明人」！藏在不為人知的地方，連暗算8人

在最近和平精英體驗服的更新中，也是進行了一次顛覆性的更新。玩家不僅可以化身成為特種戰士，還能掌握各種各樣的作戰技能，而最受玩家歡迎的莫過於裝配了「立體機動裝置」的突擊兵。

然而，在體驗服測試中，就有玩家利用突擊兵的專屬技能進行了一波「上樹」操作，化身成為一名「透明人」，藏在了大部分玩家都容易忽略的樹上。並通過假扮人機的方式，吸引路過的隊伍以及玩家，用守株待兔的方式連續淘汰了8名敵方玩家。

毫不知情的玩家在最後選擇觀戰的時，才發現自己原來這麼「冤枉」就被淘汰了，想想都來氣！那麼問題就來了，這到底是大樹建模存在BUG問題，還是新模式太過於「高能」才導致的呢？這是一個值得深思的問題。
