LPL教練爆料，PDD的戰隊陣容太差了！訓練賽被其他隊伍打自閉了

天天要聞
2019-12-19

LPL這幾天陸續有很多支戰隊都官宣了隊伍的大名單，OMG是第一個公布名單的戰隊，算是憋氣大賽的最後一名，目前有六支隊伍公布名單，還不到一半，因此憋氣大賽估計還要等幾天，等到德杯正式開賽之前所有隊伍的名單應該已經出來了。

在這些隊伍中，很多觀眾都對PDD的新戰隊ESTAR很感興趣，這支隊伍目前的陣容還是比較神秘的，目前只官宣了一個選手，就是之前在RNG替補過的Wink。

其他位置的選手還不知道，在此之前曾經有關於這支隊伍想要組建銀河戰艦的消息，PDD也曾表示要買Tarzan，事實上的爆料是這支隊伍其實在搶Chovy，但是搶不到，轉會期已經沒有很多大魚的情況下，這支新隊伍恐怕不能給出一個更好的陣容。

最近在V5執教過的教練涵藝就親自爆料，表示這支戰隊現在非常慘，PDD的隊伍配置很差，而且訓練賽打誰都是輸，現在基本處於一個自閉的狀態。

Wink的實力很不錯，但是CJJ沒了李小龍（RNG的小龍堡真名叫李小龍）有點難受。

總結來說就是這支新隊伍現在的情況比較差，估計明年的春季賽只能以一個墊底級別的隊伍配置打比賽了，成績也不會好，後面不知道還有沒有通過轉會翻身的可能。

事實上LPL在隊伍不斷增多的情況下，對於選手和人才的需要可以說是不斷增長，在這個過程中就很容易出現某支戰隊整體配置非常離譜的情況，類似今年的V5以及RW等等隊伍就是這樣。

不過LPL現在大多數的隊伍其實都是想要拿成績的，以RW和VG舉例，這兩支隊伍今年轉會期就有很多的動作，都是想要好好出成績的，因此相信類似V5或者ESTAR這樣的隊伍能夠抓到合適的機會通過培養以及轉會期來組成一支更好的隊伍。
