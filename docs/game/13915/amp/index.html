iG蕉太狼即將征戰LPL？國產中單又添一員悍將

天天要聞
2019-12-12

蕉太狼Forge這個名字相信大家都略有耳聞吧？他是IG二隊的中單，在Rookie因家事回韓國的那段時間頂替Rookie上場，難能可貴的是，他在場上的表現絲毫不遜色於一流中單，甚至單殺了國產中單的排面黃金左手Knight！

短短几場比賽，蕉太狼就證明了自己的實力，確實有征戰LPL的資本。

這樣一名出色的新人自然會是LPL各大戰隊爭搶的對象，畢竟一個實力一流的國產中單還是十分難得的！而根據可靠的消息爆料稱，蕉太狼在S10賽季已經確定加盟LPL了，並且還是一支「名字裡帶G」的戰隊！

這個消息一出，嗅到味道的吃瓜群眾頓時來了精神，不過僅憑這一點線索還是很難找到真正的去向的，畢竟LPL里名字帶G的戰隊太多了！不過根據粉絲層層推理，得出了兩個最有可能的戰隊：JDG、LGD！

先說說JDG，之前便有傳聞稱他們的中單牙膏是「中國阿P」，有轉ADC的可能，這時他們就需要一個強力的中單了，蕉太狼還是有一定可能性的！其次的LGD也有不小的可能性，因為該知名爆料前段時間還發了一條消息。

要知道，LGD剛剛引進了LCK一流打野小花生，如果又有蕉太狼的加盟，那麼在S10一定也會有很強的競爭力！不知道這樣一位橫空出世給觀眾帶來眼前一亮的國產中單小將，最終會加盟到哪一家戰隊，在S10又將會給我們帶來什麼樣的表現呢？
