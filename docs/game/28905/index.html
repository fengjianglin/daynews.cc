小花生加入LGD因不會中文被隊友嫌棄！德杯還未開打LGD已中野決裂

2019-12-21 2,723條評論 4,553次閱讀 1345人點贊

LPL冬季德杯賽在本周末開啟，各大戰隊經歷過轉會期的重組，都抓緊磨合新陣容，以求能在德杯賽中取得好成績，不過，LGD新賽季引入小花生，卻引發了隊伍內的選手矛盾。

LGD中單選手Yuuki在直播中公然Diss小花生，吐槽小花生加入LGD之後的種種不好事情，Yuuki表示，小花生不會中文，打訓練賽期間經常看不懂隊友的信號，一點用處沒有，和他打訓練賽起不到任何效果，在Yuuki認為，小花生還不如LGD之前的打野。

另外，Yuuki對於LGD引入韓援的事情，意見頗大，他表示，隊伍內五名選手，有兩名都是「啞巴」，打訓練賽雙方都聽不懂對方在說什麼，有事情只能信號溝通，可團戰中經常會因溝通問題，出現失誤，就很火大。

小花生加入LGD才剛滿一個月，不過說中文這個大家都理解，但Yuuki作為小花生的隊友，在直播中公開嘲諷小花生不會中文等於沒用，屬實有點過分，這件事在網上也引起很多人的討論，難不成德杯還沒開始，LGD就已經中野決裂？

LGD在S9的成績並不理想，S10又是在中國舉辦，引進小花生，也是俱樂部想要在成績方面有所提升，但現在看來，LGD首先要解決的就是選手之間因為溝通不暢產生的矛盾，然後才能再談S10的成績。

目前，LGD官方還沒有對此事做出回應，Yuuki也沒有向小花生道歉，有趣的是，小花生剛到LGD時，恰好碰上Yuuki生日，小花生為了能儘快融入到LGD戰隊中，還及時的給Yuuki送上生日祝福和禮物，但好像兩人的關係並沒有太好，你們對於Yuuki說小花生不會中文等於沒用怎麼看？
