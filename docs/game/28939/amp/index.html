守望先鋒：源氏就是弟中弟，終究成為版本棄子，鐵拳玩家笑了

天天要聞
2019-12-20

隨著新版本的更新，很多英雄都做了相對的改動，比如上版本強勢的雙盾被大削，還有查莉婭再次增強，幾個版本棄子源氏也是得到了一點增強，可是這點增強與上版本相比有什麼區別嗎？不就是把24的飛鏢增加到了30，其他什麼都沒有改動，真的還是版本棄子嗎？

守望先鋒發行與2016年5月20日，源氏作為元老級的英雄深受玩家們的喜愛，還有很多玩家是因為暴雪的CG《雙龍》去玩的源氏。源氏的飛鏢作為主武器，很多玩家都想體驗當忍者的快感。

源氏的技能有閃、影、斬作用分別是，格擋向你射來的飛射物並將其向你瞄準方向反彈回去，同時還能格擋近戰攻擊。快速向前衝刺，對遇到的敵人造成傷害。消滅敵人可以重置冷卻時間。源氏在短時間內拔出武士刀。在入鞘之前，源氏可以對敵人造成巨大傷害。源氏技能的屬性讓人覺得很有意思，正因為這樣玩源氏的人越來越多，同樣越來越多的大神級別的源氏玩家也是紛紛出現在大家的眼前。

很早以前的源氏大家都喜歡跳起來砍，這是為什麼呢？因為很早以前的源氏是可以砍出暴擊的，那時候的源氏已經成為了天國的源氏了，現在的源氏想在天梯打出非常優秀的一波是何其的困難，很多發視頻的源氏玩家，大多數都是在街機先鋒里的經典快速中打出來的，因為經典快速可以不用玩家強制222陣容而且玩法多變，這樣的情況下源氏才能真正的發揮出自己技能的優勢。

可是在天梯中源氏沒有資源，想砍出這樣的刀是何其的困難啊。沒有安娜激素的源氏，在對方滿狀態下能砍死幾個呢？小編有一個專精輸出鐵拳和源氏的朋友，在500強的高分局中沒有安娜激素是不會玩源氏的，有激素的情況下源氏才能像以前那樣一刀砍出180的傷害，沒有激素就只能砍出120的傷害，源氏的技能加上刀的傷害，在沒有激素的情況下，刀砍一下Shift一下最多只能打出170的傷害，況且在開大的時間裡是不能用飛鏢和近戰的，只能依靠刀和影打傷害。

雖然新版本更新加強了源氏的飛鏢，但是很多玩家都是因為源氏的刀才去玩的源氏啊，現在在天梯里，源氏的身影越來越少了，鐵拳逐漸變多了起來，如果源氏再不增強，那麼源氏始終只能是版本棄子。以上是小編的看法，大家認為現在的源氏是弱還是強呢？歡迎大家在下方評論、留言。
