騰訊宣布《雪鷹領主》手游將於12月17日在全平台上線，自創技能、短視頻社交等元素都令人期待不已。

《雪鷹領主》手游公布了效果震撼的CG戰鬥視頻。在戰鬥CG中，玩家可以看到人物細膩精美的建模，流暢酷炫的技能和爽快的戰鬥場面。

更重要的是，官方曝光了《雪鷹領主》手游獨特的自創絕學戰鬥體系。自創技能戰鬥系統將跨越原來MMO手游固有的職業特性，讓玩家能夠根據戰鬥需求進行技能的選擇和組合，開啟不同戰鬥特性技能，享受自成一派的戰鬥。

自創技能系統首次開放了炎龍、雷虎、靈鯤三大技能特效，玩家還可自主選擇技能的傷害效果和傷害範圍等，創造自己的獨有絕招，為戰鬥帶來更多的可能性。另外，炫酷的特效也讓人熱血沸騰。既實用又好看的絕招，誰不喜歡呢？

《雪鷹領主》手游的「短視頻社交」玩法也引人注目。玩家在遊戲里探索玄幻異界的同時，還能享受遊戲內「短視頻刷不停」的休閑體驗，爆笑段子、網路紅人、時事熱點都會在該玩法中體現出來，玩家將緊跟時代潮流，獲得豐富的遊戲樂趣。值得一提的是，短視頻社交玩法不僅可以看視頻，還可以與視頻創作者進行點贊、關注、訪問等親密互動，令遊戲體驗更為充實，喜歡社交的小夥伴務必細細體驗此玩法。

東伯雪鷹、余靖秋、司安、銅三叔等經典IP角色將等待玩家前去邂逅，他們身上的異聞傳記將會被玩家一一揭曉。另外，富有策略性、博弈感的自由戰鬥體系，多樣化的戰場PK和副本挑戰等內容能夠進一步滿足玩家的需求。


目前，《雪鷹領主》手游預約活動正在火爆進行中。玩家可通過多種渠道參與預約修行活動，預約成功即可在不刪檔開服後獲得豐厚的獎勵。當全服預約達到特定數量，還能領取相應檔次的集結獎勵。趕快預約遊戲，將預約獎勵收入囊中吧！
