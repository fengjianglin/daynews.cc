德杯首日看點：RNG小龍堡首秀，VG能否帶來驚喜？

天天要聞
2019-12-20

北京時間2019年12月21號，作為我們國內英雄聯盟最主要的杯賽，德瑪西亞杯將在這一天正式拉開帷幕。從此次的參賽隊伍來看，除了我們LPL聯賽16支隊伍之外，LGE、GD、YM以及SDX這四支次級聯賽的隊伍也將一同參加。該賽事的參賽隊伍一共分為了四組，而12月21號德杯首日的比賽，就將由A組跟C組展開組內循環賽，決出各自小組中的前二名，晉級到下一輪的比賽中去。

那麼，在本屆德杯首日將會有哪些看點呢？讓我們一起去了解一下吧！

看點一：VG新陣容首秀

傳奇教練Kkoma加盟到VG戰隊無疑是本次轉會期中的大事件之一。作為S世界賽的「三冠王」得主，很難有人相信他會加盟到一支曾經在我們LPL幾乎處於墊底的隊伍中去。不過，伴隨著Kkoma的加盟，一系列的轉會交易也是在證明著這支全新的VG似乎的確是有著比較大的潛力。而本次德杯也是這支全新VG戰隊的首秀，他們能否在德杯中一鳴驚人，或者是帶給我們更多的驚喜呢？還是非常值得期待的。

而在A組的組內循環賽中，VG戰隊最大的對手無疑就是SN了。而從雙方的勝率來看，SN也的確佔據著一定的優勢。不過，這也是在以往與VG交手中的數據。而新版VG是否會逆轉這種勝率呢？讓我們拭目以待！

看點二：RNG小龍堡首秀

相信在新賽季中，RNG最大的變化無疑就是打野位置上面的變化了。在此之前，不管是Mlxg還是Karsa都是個人實力相當不錯且擁有著豐富經驗的打野選手。如今，伴隨著Mlxg的退役、Karsa的轉會，RNG打野的重擔就全部落在了新人的身上。而出身自YM的小龍堡自然也成為了RNG新人打野選手中最值得注意的一個人。不知道小龍堡在RNG的首秀是否會很順利呢？

另外，從RNG與另一支隊伍OMG的勝率情況來看，雖然Smlz回歸到了OMG，但顯然還是支持RNG戰勝OMG的人更多的。

看點三：OMG能否夢回S7？

伴隨著Smlz的回歸，OMG戰隊似乎又讓人感覺到有些夢回S7賽季的感覺。雖說OMG並沒有在S7賽季打入世界賽，但至少S7賽季的OMG表現還是相當給勁兒的。特別是Smlz與小五的「心靈感應」組合，真的是好久不見了。

看點四：LNG能否起飛？

與S9賽季的陣容相比，LNG在這個新賽季最大的變化就是更換了中野。相信大家都知道，一對好的中野能夠為隊伍帶來的改變也是非常巨大的。所以，新賽季的LNG很可能會因為中野的改變，在戰術打法上面變成另外一支隊伍。但不管如何，由聖槍哥、Xx以及Maple所組成的上中野還是比較強勢的。再加上這一屆德杯，LNG是拿出了自己的首發陣容。所以，我們也能夠從本次德杯中了解LNG是否能夠在新賽季起飛。

看點五：LGE與YM能否成為黑馬？

與歷屆的德杯一樣的是，本屆德杯依舊有次級聯賽的隊伍。但是，在以往的德杯中或多或少都是會出現「以下克上」的「黑馬」局面。不知道在德杯首日的LGE與YM這兩支次級聯賽的隊伍是否能夠成為「黑馬」呢？

最後，除了這些看點之外，FPX全部啟用新人、WE的新陣容以及JDG等這些隊伍也都是充滿了很多的看點。究竟在德杯首日的小組賽中，哪四支隊伍可以脫穎而出呢？讓我們拭目以待吧！
