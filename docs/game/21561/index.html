大富翁新游《億萬富翁之路》PC版預告 4月17日發售

2019-12-19 2,104條評論 4,070次閱讀 1097人點贊

今日(12月18日)萬代南夢宮公布了《億萬富翁之路》PC版新預告，本作將於2020年4月17日在Steam上推出，遊戲支持中文，售價暫未公布。

Steam商店地址：

PC版預告：

《億萬富翁之路》已登陸Switch，是一款類似於「大富翁」的休閒遊戲，玩家在地圖中通過擲骰子決定前進距離，並通過購買房產等手段積累財富。遊戲勝利條件非常簡單，玩家要努力賺取比對手更多的金錢並搶先到達分散在日本全國各地的目的地。停留在增益格可以獲得金錢，但各地區的產業才是最主要的收入來源，購買產業便能實現不勞而獲。

舉例來說，購買東京銀座的壽司店、秋葉原的動畫周邊專賣店等店鋪或農場、工廠等產業，即可在每年3月獲得收益。在遊戲結束日期到來時，資產最多的玩家將獲得冠軍。玩家可以為淘汰對手而發動各種惹人生氣的搗亂行為。例如向對手送出散財怪物、用道具將對手隨機傳送到其他地點等等。用盡一切手段妨礙對手吧！

1.最先到達目的地可以獲得獎金，但僅靠此項收入顯然不夠，還要在途中投資各種產業。

2.遊戲中共有三十餘種道具和五十餘種怪物！玩家可以通過道具和怪物的作用獲得有利局勢。但有些怪物會擅自花費玩家的金錢，需要多加註意。

3.多種遊戲模式：億萬富翁之路還包含與淘汰賽模式等普通模式不同的特殊模式。其中有數種遊戲時間較短的模式，不妨在有興趣時嘗試。

4.多人對戰：最多可以四人對戰。與朋友們同場競爭，看看誰能成為日本第一的金融王者吧！

5.億萬富翁之路以日本為主舞台，同時還存在隱藏的特殊地圖。去尋找隱藏的怪物島吧！

6.兒童可以通過本遊戲學習到金融與投資的重要性！
