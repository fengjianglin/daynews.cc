動視暴雪擬與玩家合作 制定旗下手游營銷策略

2019-12-19 1,220條評論 3,466次閱讀 1750人點贊

一直以來，玩家們的聲音向來是許多遊戲公司們非常重視的，畢竟，做出的遊戲玩家們不買帳，無異於把錢扔進水裡。而知名遊戲公司暴雪向來非常貫徹這件事情，當年的《星際爭霸》一代就是因為市場反應不佳，吸取了玩家的意見之後回爐重造，最後才成為了經典的即時戰略遊戲。而最近動視暴雪正在考慮將玩家聲音加入公司的決策當中：他們希望從玩家群體中選出最活躍的人組成一個研究小組，來提高對於手游市場消費者的了解，制定更明確的營銷決策。

根據外媒報道，這個組織將被稱為King’s Council，將由玩家群體中較活躍者所組成，代表著玩家們的聲音，而在理想的情況下，這個組織會讓動視暴雪更快的制定更明確的營銷決策。以吸引更多的手游用戶。

動視暴雪的媒體總裁Brian Ames表示：「我們希望為社群注入更多的生命力，並讓我們的玩家在他們喜歡的遊戲與體驗中給我們更有意義的回饋，而King’s Council是玩家們分享他們回饋與觀點的一種方式，確保我們在開發新產品時，能做出最適合玩家和我們營銷合作夥伴的決定。」

這項計劃的重點將放在手游之上，因此像是主機或PC的遊戲玩家，可能就不會收到加入這個組織的邀請，然而，根據動視暴雪今年第三季度的財報，旗下手遊子公司King的活躍用戶在暴雪的每月活躍用戶中佔了一半以上，來到2.47億人，因此動視暴雪越來越關注旗下手游業務的發展似乎也並不是奇怪的事情。
