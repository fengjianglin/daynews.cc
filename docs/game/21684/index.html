國服本周俱樂部精選（12.16）

精選球員將分為俱樂部精選和每周精選兩種。入選每周精選的通常是最近在真實賽場表現優秀的球員。俱樂部精選則是各個俱樂部表現突出的球員。

國服本期（12.16）推出的俱樂部精選包含兩支俱樂部，分為利物浦和曼城兩個俱樂部精選卡包。利物浦俱樂部精選國際服目前推出過兩次，本期國服推出的是國際服10月14號推出的版本；曼城俱樂部精選國際服目前也推出過兩次，本期國服推出的是國際服11月4號推出的版本。

利物浦精選

利物浦精選（8人）：范迪克（初始93、滿級96）、馬內（初始92、滿級96）、阿利松（初始92、滿級95）、薩拉赫（初始91、滿級96）、菲爾米諾（初始90、滿級）、亨德森（初始89、滿級94）、阿諾德（初始88、滿級96）、張伯倫（初始87、滿級）

范迪克

精選范迪克有四個版本，其中三個初始評分94，國服本期推出的是最差的精選版本，初始評分93。最強版本的范迪克與本期相比，主要差別是滿級頭槌高3點（98）、身體對抗高1點（99），如果想等最強版本不建議現在氪金。

馬內

精選馬內目前就只有這一個版本，初始評分比20a高3分，滿級總評高1分。具體提升情況如下：

精選馬內差不多相當於現役馬內突破5級的水平，增強幅度一般，主要提升的就是射門，與20a基礎球員的區別玩家基本無法感知，直接使用20a馬內即可。

阿利松

精選阿利松出過兩個版本，滿級後的守門員數據一致，除解圍外其它4項數據均達滿值。與20a阿利松的主要區別就是臂展+1,20a基礎阿利松滿級後的臂展是98。說實話精選阿利松的數據難稱最強，而且狀態持續性也比較低（6），與其氪金不如直接用20a的特爾施特根或者埃德松，小獅子狀態持續性高，埃德松也是四項守門員數據滿值。

薩拉赫

精選薩拉赫有兩個版本，最強版本初始評分92，國服本期推出的是較弱的版本，初始評分91。二者的區別只有一個，最強版本的滿級進攻意識高3點（96），跑位更加積極。

本期推出的精選薩拉赫基本是20a薩拉赫各項能力+1，整體大概是20a薩拉赫突破6級的樣子，主要加強的弧度由於不是關鍵數據，雖然有一定作用，但並非決定性因素，所以氪精選薩拉赫的意義不大，20a薩拉赫和精選沒差多少，至少每項能力值只+1，玩家是感受不到區別的。

菲爾米諾

精選菲爾米諾推出過兩個版本，二者評分一致。只不過國服本期推出的控球更強（+3），另一個版本地傳更好（+3）。由於新版本十分依賴中鋒球員的對抗能力，而精選菲爾米諾既沒有對抗也沒有速度，實用性不會太強。

亨德森

精選亨德森只有這一個版本，一般金球變黑球精選的球員，其精選版本都屬於變態級加強：

本期推出的精選亨德森，差不多相當於20a亨德森突破15級的樣子，屬於提升幅度較大精選球員，基本上精選和20a是同一個模型的兩個球員，如果組隊套值得氪金。如果放在所有精選和傳奇里比較，精選亨德森也就是T2級別的中前衛，速度慢、對抗弱、靈活差是原罪。

阿諾德

精選阿諾德目前只有這一個版本，滿級提升幅度較大。與亨德森類似，精選版本和20a是同一個模型的兩名球員。精選阿諾德是新版本最強助攻型後衛之一，但攻強守弱，身體對抗能力較差。右後衛這個位置實戰效果最好的還是精選巴里奧斯、萬比薩卡這種防守見長的球員。

張伯倫

精選張伯倫目前只有這一個版本，跟20a比屬於史詩級加強。從實用性來講，175cm的的小模型要比數據顯示出來的對抗更不禁撞，雖然速度在中前衛里比較有優勢，但盤控和傳球數據較為平庸，而且新版本防守AI集體加強，盤帶過人極為困難，張伯倫這種推進型中前衛很難發揮作用。

曼城精選

曼城精選（8人）：阿奎羅（初始93、滿級97）、埃德松（初始93、滿級95）、德布勞內（初始92、滿級96）、斯特林（初始91、滿級97）、戴維·席爾瓦（初始90、滿級95）、奧塔門迪（初始88、滿級93）、羅德里（初始87、滿級95）、津琴科（初始85、滿級92）

阿奎羅

精選阿奎羅目前有兩個版本，一個初始評分94，另一個初始評分93，本期國服推出的是初始評分93這個版本。不過二者區別不大，94分版本進攻意識更高（99），但緊密控球稍低（90）。無論哪個版本的精選阿奎羅，速度都不快並且身體對抗能力較差，基本不可能在新版本中有所作為。

埃德松

精選埃德松目前有兩個版本，二者關鍵數據一致，守門員五圍均滿值，是新版本最強門將之一。目前為止，守門員五圍數據全部滿值的門將只有兩個，一個是精選埃德松，另外一個是國服已經推出過的精選特爾施特根，區別是精選特爾施特根的狀態持續性也是滿值。

德布勞內

精選德布勞內目前有三個版本，兩個初始評分94和一個初始評分92，國服本期推出的是初始評分92的這個版本，屬於最差的一個版本。與最強版相比，各項數據均不佔優勢，建議想要精選德布勞內的玩家等最強版本再氪。

斯特林

精選斯特林目前只有一個版本，與內馬爾類似，在新版本中面臨著「一碰就倒」的尷尬局面。由於20賽季身體對抗的空前強大，主要依賴盤控突破的小快靈集體迷失，精選斯特林的前景不容樂觀。雖然依然可以靠速度生吃，但在新版本中一旦被後衛貼身，基本無法加速，速度的優勢等於變相削弱。而且新版本球員慣性增大，可操控性降低，想要充當爆點也不那麼容易。

戴維·席爾瓦

精選戴維·席爾瓦目前也有兩個版本，國服本期推出的是低配版，最強版初始評分92。二者的差距較大，最強版在各項能力上均較本期推出的版本有所提升。不過最重要的問題是，雖然戴維·席爾瓦在現實中是一名中場大師，但在新版本中這種既沒有速度也沒有對抗的球員只能淪為雞肋。

奧塔門迪

精選奧塔門迪目前有兩個版本，初始評分相同，具體數據差異懶得看。因為精選奧塔門迪的缺點太過明顯，183cm的模型在中衛位置太過吃虧，並且速度慢如牛、轉身一萬年，沒什麼氪金的必要。

羅德里

精選羅德里目前就這一個版本，是精選中最強的後腰，在整個遊戲中也是僅次於傳奇維埃拉的存在。

精選羅德里基本在20a的基礎上各項能力值+1，但與眾不同的是新版本最重要的身體接觸，精選羅德里提升了4點，等於加強了一檔，關鍵數據的提升非常明顯，恐怕也是本周俱樂部精選里少有的幾個值得氪金的精選。

津琴科

精選津琴科目前只有這一個版本，雖然在20a基礎上有較大幅度提升，但身體對抗較差，速度偏慢，整體平庸無亮眼數據，能用20a球員替代的精選為什麼要氪？
