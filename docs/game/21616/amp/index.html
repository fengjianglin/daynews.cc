CSGO：shox——我們為應對本次震中杯已做好了萬全準備

天天要聞
2019-12-19

在昨日以比分2-0率先擊敗EHOME晉級到震中杯A組勝者組後，Vitality選手shox接受了來自HLTV的採訪。

採訪內容如下：

Q：Vitality上一次打線下賽還是在ESL職業聯賽第十賽季的歐洲區預選賽上，在這一段空白期里你們有大量的時間用來訓練，所以對於本次震中杯的備賽工作感覺如何？

A：我們在過去的幾場賽事里打得很艱難，因為這支陣容僅僅才在馬爾默大師賽前集訓過一次，而除了那十天的集訓之後隊伍再也空不出任何時間用來訓練。

不過我們對於本次震中杯充滿自信，因為隊伍充分的利用這段空白期來備戰這場賽事。我們在賽前除了有10天的集訓之外，還有大約5天的線上訓練。所以這一次的我們充滿信心，每個人的狀態都非常好，我們終於能夠至少有一些準備的來參加一場賽事了。

Q：考慮到你們對於本次賽事有了充足的準備，隊伍對於震中杯的期望如何？你覺得取得什麼樣的成績對於隊伍來講才算不錯？

A：我們其實還沒有聊過對於本次賽事的期望，但我們都對自己充滿自信，做好了完全準備。我們只有一個目標，那就是奪得冠軍。

Q：你是在9月末加入到Vitality的，自從入隊以來，你的位置有發生任何改動嗎？

A：肯定還是有變動的。當我參加完IEM北京站後，我被換到了一些我個人感覺非常舒服的位置。因為隊伍終於有時間進行訓練，所以我開始慢慢習慣隊伍的整體風格，對於地圖的節奏等。總得來說，我感覺一切都在朝著正確的方向發展，我自己每一天都在進步。

Q：在之前的採訪中你提到自己很難擺脫在G2當指揮時的思維方式，那麼現在的感覺如何？

A：是的，雖然花了一些時間，但是我現在更加適應當一名普通的選手了。唯一的問題是在比賽過程中我還是需要了解為什麼我們要這樣做。一旦我開始搞不清楚情勢的時候，我就會遇到麻煩，這對於我的個人發揮產生了極大的影響。

因為我們有時間去準備，我也利用了這段時間去思考每一件事情，並積極地與隊里的指揮進行溝通與提問。

Q：有關你搞不清楚場上的情勢，你能舉一個具體的例子嗎？

A：有時當我不明白比賽場上的形勢或者是我們在做什麼的時候，我就會問隊友，有時候甚至是有關戰術層面的。因為我們之前沒有太多的訓練時間，所以有些時候我很難去記住在一些特定的情況下我應該做什麼。這一切都是關於我理解自己角色遇到的問題，以及我們作為一個團隊整體需要做的事情。
