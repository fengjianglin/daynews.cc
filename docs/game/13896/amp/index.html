笑談風雲，一語定乾坤。

大家好，我是星仔醬。上期和大家略微講了一下火計，今天我們繼續探討一下火陷阱的用處。

堵路

火球陣堵路。

這個是利用ZOC機制來限制敵軍進入己方領地。操作的也很簡單，需要控制的路口旁邊建一個陣，然後在陣旁放上幾個火焰種。主要的原則就是，電腦部隊無法一回合直接通行。需要把火焰種破壞、或者把陣破壞才能過去。由於三國志11電腦在點火焰種無法燒到敵人時是不會點火焰種這個規則。利用這個規則，再加上陣的ZOC，就會導致敵部隊傻傻的站在火焰種旁邊望陣興嘆。當電腦研究出【射程】後對井欄、投石部隊無效，【遠矢】的弩兵部隊無效。對呂布、劉備無效。

防禦經濟區

火焰船防禦經濟區。

這個主要是用於防禦自己城市沿水的經濟區。根據上面電腦部隊點火焰種的原則，只要在經濟區能被攻擊到的區域範圍內安置上火焰船，就可以防止電腦部隊對經濟區建築進行攻擊了。這種做法對投石也有效哦。作用最大的當屬江夏，甚至可以防住江夏城來自南邊水域。

攻城掠地

火者，凶也。不可不察，不可不慎。

火焰球在攻城方面表現很優秀。甚至有部分小勢力的破局都需要靠他。別的不說，就單單說難度在三國志11所有劇本中數一數二的嚴白虎。251年超級難度下的嚴白虎，已經是大部分玩家心中的噩夢了。但是也有很多老玩家在經過無數次試驗後，琢磨出了多個破局的方法。而這火焰球破局法，就是其中一種了。具體怎麼破局，這裡先賣個關子，星仔下次詳細發文說一下。經過多次試驗證明，火焰球破局法對251超級難度的嚴白虎勢力有效，效果還不錯。

在研究完火四級科技後，火陷阱的傷害能力能在周瑜的火神下得到最大的發揮。星仔之前發過一個業火種能造成多少點傷害的視頻，初略估計的結果是五萬六千點傷害。再攻城方面，通過五個埋火部隊的配合，周瑜一回合能拿下城池的1400*4的耐久。遊戲中8成以上的城市都可以這樣速攻拿下。而且四級的業火種擁有炸死、炸暈的效果，傷害也非常高。就算是周瑜，也會被炸死。
