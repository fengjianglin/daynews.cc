DNF：新冒險者活動又玩坑，全勤也湊不齊材料，獎勵根本換不完！

以下為具體的內容詳情：

DNF在今天更新之後被下架刪除的那個星羅萬象活動，因為兌換商店獎勵的材料數量有獲取上限，很多在一開始不知情的玩家都白白錯失了一本增幅書，可以說特別的坑人。而隨後在今天策劃剛剛更新上線的「冒險者的遊戲」活動同樣也是如此，玩家們就算保持全勤也湊不齊材料，商店中的道具獎勵在活動結束前根本換不完。

冒險者活動的商店獎勵全勤也換不完？沒錯，小寶並沒有騙你，因為我們可以大致的來計算一下：在這個活動商店中的獎勵如果玩家想全部兌換出來（不包含藥劑），總共需要消耗180個龍之硬幣材料，180個！缺少一個都兌換不全商店中稍微重要一點的獎勵。

而這個新活動從今天這第一天上線，雖然說持續到1.13當天才會結束，中間總計足足有35天的時間，但是在這期間玩家們就算每天都保持全勤兩次，且每次都是第一名拿到3個材料，全勤最多也只能領到210個材料，嗯？210個！這不是剛剛好能將商店中的獎勵全換出來，而且還額外多出了30個材料沒地方用嗎？

針對於這一點，玩家全勤35天總計參與70次，次次拿到的都是第一名領到3個材料的概率，說實話比魔盒開出+12的鑽石增幅卷還低，是根本不可能發生的事情。因此，我們就取個中數，總計有一/二/三個名次，全勤70次你次次都可以拿到第二名，而第二名每次的獎勵只有2個材料，那麼全勤總計也才140個材料，所以根本不夠換完商店中的所有獎勵。

拿著到底該怎麼辦呢，小寶的建議是到時候玩家們只要湊夠一定的材料，優先一定要先將商店中的那兩個普雷寶珠給換出來，其餘的道具獎勵在活動結束前能換出來多少是多少，最大化的減少損失，切記，無論如何優先選擇兌換的一定要是商店中的那兩個普雷寶珠。
