《閃耀暖暖》一套8萬面建模的3D服裝，是如何製作的？

2019-12-17 3,231條評論 4,823次閱讀 2413人點贊

前段時間上線的次世代3D手游《閃耀暖暖》因為其優秀的遊戲品質以及美術表現獲得大批玩家的關注，上線當日即登頂IOS免費下載榜單第一。據了解，這款遊戲首日新增已達400萬，這個數據足以說明一切。

打磨細節 還原真實材質

《閃耀暖暖》遊戲高質量的遊戲品質來源於它的建模精度以及對細節的把控。在此前，《閃耀暖暖》就透露過遊戲內一套3D化服裝的整體建模面數在50000至80000面之間。這樣的水準甚至超過了端游的建模水平。

從很早的時候，小編就關注了這款遊戲。原因在於，不管是多麼外行的人，只要仔細看看這款遊戲的建模精度，就能明白它在技術上花了很大的功夫，在手游端實現了一件非常困難的事情。

具體來說，《閃耀暖暖》里製作一套3D化的服裝，整體建模面數能50000~80000面之間，遠遠高於此前水準較高的15000~25000面。這個數字可能沒法給人太具體的概念：做一套這樣的衣服到底有多難？。

“辭鳳闕”作為遊戲內歷史悠久的”雲端帝國”風格的服裝，糅合了中國古代戲服元素，也是雲端文化的代表服飾之一。這套服裝的整體建模達到80000面左右，並且為玩家展現出許多細節，如頭冠珍珠的光澤，多變的刺繡與花紋等。

套裝的整個設計流程如傳統3D遊戲一樣，從原畫設計開始，經過3D模型的製作與動畫動作效果的製作。但到具體的細節把控上，《閃耀暖暖》又與傳統遊戲截然不同。遊戲的製作團隊在對細節的具體實現上有極高的要求。2D原畫將服裝的具體細節繪製出來，而這些細節的真實體現需要在建模中實現。

製作團隊對細節的極致追求，讓服裝的褶皺、不同花紋的變化、刺繡的方式與針腳走向等等得以真實的展現給玩家。

更讓人驚嘆的是”辭鳳闕”套裝的頭冠。據了解，頭冠上面的珍珠大大小小共有717顆，還有複雜的珠簾裝飾。不僅每顆珠子都光滑圓潤，甚至還可以看到珠子表現出的真實的晃動效果。

這樣的細節不僅僅只體現在某套套裝中，也存在於遊戲內的散件、裝飾中。比如在不同布料上的針腳也會有不同的形態。

除了能夠在玩家直觀感受到的地方，製作團隊注重細節；在玩家平常看不到的地方，製作團隊也是十足的”細節控”。遊戲內真實的還原了不同的鞋底，在一些情況下，比如遊戲內的拍照模式下，當暖暖抬起腳的時候，就能看到不同的鞋子會有和現實世界中一樣匹配的鞋底。

正是因為製作團隊對細節的把控，讓《閃耀暖暖》成為一款高質量的3D手游，也讓玩家有極近真實的遊戲體驗，吸引眾多行業內外以及遊戲玩家的關注與讚譽的同時，也將國內的換裝遊戲水準帶上新高度。

所以說一款遊戲的成功，離不開建模師的技術，你技術高，製作的好，受玩家喜歡，走到哪都有人搶著要你。學習遊戲建模，你首先要找到適合自己的方法，再去學，堅持學，才能成功。
