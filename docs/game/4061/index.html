爐石傳說玩家提前開巨龍降臨卡包，這出橙概率，暴雪良心發現？

2019-12-10 0條評論 15次閱讀 0人點贊

摘要

小夥伴們好鴨，這裡是一樹一葡萄，一瓜吃一天的遊戲日報騎虎。想必小夥伴們都知道爐石傳說新版本“巨龍降臨”將要更新了吧！看著新版本超模到強無敵的新卡是不是已經蠢蠢欲動了呢？雖然新版本正式到來需要等到12月11日，不過一些小夥伴已經發現遊戲內的卡包可以通過特殊方式打開了！

小夥伴們好鴨，這裡是一樹一葡萄，一瓜吃一天的遊戲日報騎虎。想必小夥伴們都知道爐石傳說新版本「巨龍降臨」將要更新了吧！

看著新版本超模到強無敵的新卡是不是已經蠢蠢欲動了呢？雖然新版本正式到來需要等到12月11日，不過一些小夥伴已經發現遊戲內的卡包可以通過特殊方式打開了！

近日，騎虎所在的爐石遊戲群里，小夥伴們開始了新一輪的秀歐氣，各種橙卡發圖不斷，但是我尋思著雖然暴雪最近送了多次卡包，但是都是新版本的啊，同時新版本並未正式開啟，他們又是如何開啟這些卡包的呢？

正常玩家的卡包現在仍無法開啟

在騎虎的再三追問下，終於知道了方法。就是通過遊戲內的爐邊聚會，便能提前體驗到新版本，同時也就能打開這些新卡包了。而如何參加這些爐邊聚會呢？

一種方法是親身抵達爐邊聚會的場地，另一種方法便是通過手機定位到達聚會的場地，從而讓系統判定你加入了這場爐邊聚會。

當然，有的朋友擔心那些提前開啟巨龍再臨卡包的玩家，用新卡牌組卡組是否會影響現有遊戲環境平衡？

答案是完全不會，因為提前開出的卡牌雖然能夠組建卡組，但是被系統限定只能在爐邊聚會的模式中使用。不會擾亂現有的天梯以及狂野環境，沒有新卡牌的玩家完全不用擔心這些提前開卡包的玩家對於遊戲的影響。

巨龍降臨的卡牌無法使用與製作

所以說這次通過爐邊聚會提前開啟新卡包的玩家，可以說是一群小白鼠，測試新卡包的出橙概率。

而結果看來這次暴雪好像較之前良心許多，玩家秀出大量開出橙卡的截圖，想想當年騎虎開「奧丹姆奇兵」卡包時，預購僅出三張橙，眼角不禁流下一滴碩大的非酋淚。

最後還要說一下，雖然可以提前開卡包，但是預購贈送的1張橙卡仍要11號才能獲得，所以小夥伴們希望拿到一張怎樣的金色傳說呢？
