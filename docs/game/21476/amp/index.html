氣定神閑帶你走，說一說懷舊服戰場兇器奧火與火奧的細微差別

天天要聞
2019-12-19

上周魔獸世界懷舊服正式開放了奧山的戰場，想必大家都已經體驗到戰場的魅力了吧。對於老玩家來說可以說是找回記憶，對於新玩家來說則是全新的RPG遊戲體驗，那麼今天給大家簡單分析下，猶如戰場兇器的奧火與火奧的區別.

廢話不多，咱們先說下他們的相同點：奧法天賦的氣定神閑是必須的。例如氣定炎爆、氣定烈炎風暴、氣定羊，如果實在無聊可以氣定造水術等，總之氣定神閑不能少。

下面來說說區別，奧火是奧法最終天賦31點的奧術強化，一個可能很早就被人遺忘的天賦。30%的傷害加成，30%法力消耗，持續15秒。出到這個天賦的31/20奧火是真正的”瞬間戰場兇器”。在奧山中3分鐘CD一點感覺都沒有，一下就到CD冷卻時間。開奧術強化，火球+火沖+氣定+炎爆，後手可以強反（天賦強化反制），再接灼燒、吹風、奧爆都是不錯的選擇，另外超短CD的火焰衝擊，則是決定勝負關鍵，堪比戰士的斬殺一樣實用。

當然如果你想不開，可以先開奧術強化，再衝到人堆里氣定+烈炎風暴（做個按鍵宏就是了）+奧爆（奧術精妙+6%奧爆爆擊+30%傷害），只要走位風騷點，完全有可能用閃現逃回來，平均A下有400+爆擊600左右，現在裝備普遍奧抗低，如果弄2個減抗性的戒指，效果棒棒噠（奧山崇拜有個-20對方抗性的戒指，貌似還+1%暴擊）。

火奧就是不出奧法最終的奧術強化，火法點出衝擊波。雖然沒有奧術強化的瞬間輸出，但火法後期+6%火系法術重擊，+10%火系傷害的天賦都是無時無刻高傷害的保證。氣定+烈炎風暴+衝擊波2000左右的傷害絕對穩穩的，如果強化烈炎風暴這個天賦再加上人品爆發，人堆中總有幾個瞬間會殘血，心情好魔爆兩下再閃現回來，吃點東西喝口水看看剛才弄死幾個。這2下一點也不遜色於奧火和冰法凍住加吹風。不過生存能力稍微低了些，我看到過很多法師衝進人堆里冰箱，剛解的瞬間就中了N個強反，最終什麼都沒幹死於原地。在團戰中只有風騷的走位才能保命。不然即使是聖騎也是一樣，信心滿滿的剛衝進人群中就被強反，結果無敵都開出來就被亂刀砍死。

總結一下，奧火屬於瞬間5秒真男人型，過了3分鐘又是一條好漢！則火奧屬於全程高輸出，衝到人群中就像絢麗的煙花一樣。另外炫耀下，本人300不到的法傷這兩天奧山平均1小時能殺50來人，這也太刺激了。心動的話你也來試試吧，當然你喜歡冰法的話，我也是非常支持的！
