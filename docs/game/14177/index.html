長城削了，扶桑弱了，走投無路的我拿起了這套陣容

2019-12-10 408條評論 1,683次閱讀 250人點贊

各位模擬戰的小夥伴們大家好，《王者模擬戰》自上線以來，誕生了不少很強勢的吃雞陣容。最早是高端局人人都用的長城守衛軍套路，也就是俗稱的」養豬流「，之後又有了扶桑法刺等套路，但這些都無一例外的被官方制裁。時代在進步，咱們也不能因為陣容被削就止步不前，所以在多次的實踐後，便有了這套成型後很強的「稷下坦射」。

和堯天坦射不一樣的是，這套陣容加入了稷下羈絆來增強陣容實力，除了公孫離和黃忠這「兩門大炮」，在後期還把呂布也拉了進來，這樣一來，整套陣容的輸出是沒什麼問題了，剩下的看的就是防禦和控制了。廉頗、盾山和蘇烈組成的防禦體系還是很肉的，而弈星這種大範圍控制會讓對手不少英雄感到難受，再說廉頗的控制能力也是很強的。

在前期，我們依然可以採用3或者5長安來打工，雖然他們的屬性被削弱，而且還時長「打假賽」，但不得不承認，長安羈絆帶來的福利是別的陣容沒有的，前期用他們一點毛病沒有。如果過渡得當，多金的屬性會讓你中後期都是無往不利的狀態。

到達中期，長安棋子就可以慢慢被替換了，公孫離是陣容核心，搭配長安兩個射手可以組成3射手羈絆，而弈星可以和公孫離組稱堯天羈絆，然後在有2稷下的情況下，加入其它打工位，比如3長安就行。這樣既可以在對戰中不輸對手，還能繼續享受長安帶來的福利，一舉兩得。

到遊戲後期9人口，我們就可以慢慢祭出這套陣容了，公孫離盡量3星，其它棋子全部2星，如果有可能追3就別害羞，大膽的上。裝備給公孫離、黃忠和呂布，開戰後你就會發現，公孫離和黃忠的輸出是真心的高，而呂布的大招一旦出來，對面脆皮基本是會掉半血的。

總體來說，這個陣容的羈絆還是很強的，對戰能力也不錯，但唯一的難點就是棋子難湊，因為成型陣容中的5費棋子太多了。不過這樣的好處就是全2星也可以吃雞，畢竟他們的自身屬性在那擺著呢。

好了，本期的內容就到這裡，如果你對這套陣容還有什麼要說的，歡迎評論區留言，說出你的看法吧。
