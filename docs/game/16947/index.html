DNF100級裝備介紹，新增可交易史詩，神話裝備，裝備保值方法

2019-12-15 1,785條評論 4,004次閱讀 180人點贊
  • 2020年1月9日，更新後的各種地下城中，可以獲得各種100級全新裝備。
    根據角色的成長階段，可以自然過渡的新裝備獲取方式。

  • 100級神器裝備>100級傳說裝備>100級史詩裝備>100級神話裝備
  • 100級神器裝備：與當前相比擁有更強大的性能，可以為快速獲取裝備的角色提供支援。
    獲取地點：根特皇宮
  • 100級傳說裝備：與當前相比擁有更強大的性能，可以以此進入全部新增的副本。
    獲取地點：記憶之地，無底的坑道
  • 100級史詩裝備：武器74種 + 35套裝備，可以組合成各種各樣的套裝。
    獲取地點：奧核之眼，苦痛的地下室，暗黑神殿
  • 100級神話裝備：2020年1月新增的稀有度，可以升級獲得，提高100級史詩裝備的效率。
    獲取地點：奧核之眼，苦痛的地下室，暗黑神殿
  • 多種多樣的史詩套裝
    防具、首飾、特殊裝備各類別只需1個部位，即可組成混合型套裝。

特殊新增裝備

  • 可以交易的史詩「智慧的產物」
  • 首次開放可以交易的史詩裝備。
    不存在套裝屬性的單一部位，起到補足成長區間的效果。
    無法強化、增幅、鍛造，可以通過專屬的[裝備改造]系統，觸發特殊的屬性。

神話等級裝備

  • 新增的裝備稀有度，擁有更強大的性能。
    當各個史詩套裝屬性中，只要存在1件神話裝備，就可以附加神話專屬屬性。
    神話存在上衣、手鐲、耳環3個部位，每個角色只能穿戴1個部位的神話裝備。
    100級史詩套裝 + 神話裝備 = 強化特殊屬性

裝備增值

  • 升級

可以將當前的95級裝備升級為100級史詩裝備，提升裝備性能。
只能升級為部分100級史詩裝備。（武器除外）
超時空漩渦、普雷·伊希斯獲得的95級裝備 → 100級史詩裝備 （消耗100級新增地下城獲取的材料）
可以保留附魔，強化，增幅，鍛造！

  • 銘刻

可以將當前的95級裝備的附魔、強化、增幅、鍛造數值轉移到新增100級裝備。

95級裝備上的附魔、強化、增幅、鍛造 → 100級神器、傳說、史詩、神話裝備上的附魔、強化、增幅、鍛造（消耗100級新增地下城獲取的材料）

  • 繼承

神器等級以上的100級裝備可以自由轉移其裝備上面打造的屬性。

100級神器、傳說、史詩、神話上的附魔、強化、增幅、鍛造 ↔ 100級神器、傳說、史詩、神話裝備上的附魔、強化、增幅、鍛造（消耗100級新增地下城獲取的材料）
