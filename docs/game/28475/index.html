EDG上單德杯受認可，Puff表現超預期，小龍堡帶不動RNG

2019-12-23 44條評論 3,540次閱讀 1055人點贊

關注2019德瑪西亞杯的網友都知道，隨著今天EDG擊敗TES、IG擊敗BLG之後，這次德瑪西亞杯整個小組賽部分正式結束，還是要先恭喜EDG等8支隊伍成晉級，沒有晉級的隊伍也沒有太大的關係，畢竟德瑪西亞杯的比賽只是2020LPL春季賽前的熱身賽。不過，縱觀德瑪西亞杯這兩天結束的小組賽的賽況，剛剛經歷LOL冬季轉會期人員變動隊伍和選手表現各不相同，特別是對於各個隊伍的新選手實力（無論是轉會而來二隊提拔上來）表現，更是引起了眾人的關注和熱議，畢竟賽前很多戰隊粉絲對於新來的選手和教練都是抱有一些期待，無論是IG還是RNG，都是這樣。

首先來說說大家聚焦的幾支戰隊新來選手給戰隊帶來的改變，RNG打野小龍堡，就是目前四場小組賽的表現還是不錯的，能夠帶著隊伍Carry。但是對比新轉入的ADC的Betty，真的是表現一般，不過話說回來，小龍堡的加入並沒有改變RNG實質性的問題，那就是上中兩路的短板，尤其是中路小虎軟無力讓小龍堡帶不動了，就目前狀況而言RNG轉會期的人員變動是失敗的。

對於EDG而言，花了50W和RNG競價搶來了新上單xiaoxiang，這幾場的表現倒是讓EDG粉絲感到開心了，無論是他塞恩還是大鱷魚操作都是得到了網友的認可，儘管上單奧恩局是輸了比賽的，但是他的表現至少要比很多粉絲賽前的預期要高不少。至於退役轉型成為教練廠長，他的執教BP首秀還行吧。

和EDG新來的上單一樣，從VG轉會到IG的新AD的表現也是讓人眼前一亮，Puff選手這幾場的表現還是得到了網友點贊，就在很多網友認為阿水走了之後，IG的下路要徹底拉垮了，但是目前的Puff的表現告訴我們還行。倒是樂言的表現有點差，讓很多網友有點質疑。

對於如今全華班的TES來說，轉會而來的卡薩讓目前TES的上中野節奏狀態很好，可惜的是補缺Loken和Ben下路雙人組在場上似乎沒有聲音，以至於很多網友調侃：TES上中野不送，下路躺贏，如果TES能下路在春季賽中有所提高，這支全華班的TES真的是無限可期。

當然了，隨著小組賽的結束，即將上演的將會是淘汰賽的較量，德瑪西亞杯的後續比賽鬥魚直播將會全程高清藍光獨播，上鬥魚，看德杯，見證新生的力量，還是非常期待各支戰隊的表現。
