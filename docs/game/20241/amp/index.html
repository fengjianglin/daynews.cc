教科書級忽悠：狼人殺眾明星主播被他玩弄於股掌之間？

天天要聞
2019-12-17

要說什麼技巧在「狼人殺」中最實用，那莫過於「忽悠」。在14號當天由《狼人殺官方》舉辦的頂級狼人殺賽事：WPL2019主播邀請賽上，一名「忽悠」好手便在眾目睽睽之下騙過了另外11位頂尖明星主播，一番操作著實讓人五體投地！

比賽當天採用的是《狼人殺官方》特有的「血月獵魔人」板子，場上角色配置為3狼、4民、預言家、白痴、女巫、獵魔人、血月使徒。獵魔人可以選擇在黑夜裡獵殺一名狼人，錯殺會讓自己死亡。另外血月使徒如果是最後一個被放逐出局的狼人，他可以存活到下一個白天天亮之後才死亡。這個機制可以讓狼隊多刀一刀，所以「血月局」便衍生出一種專屬的「倒鉤戰術」，即狼人陣營犧牲另外三名狼人來掩護血月，通過瞞天過海來讓血月活到最後。

而在第一場比賽中，來自「三星飯」戰隊的蛋炒飯就完美地利用「倒鉤戰術」瞞過了平民陣營，和狼人們上演了一出「絕世好戲」，拿下當天比賽MVP。

剛開局，1號位狼人率先跳狼吸引周邊火力，為旁邊最為危險的蛋炒飯打好掩護。之後蛋炒飯和8號狼人的配合更是天衣無縫，8號自爆掩護蛋炒飯的同時，遺言自己是平民，這下獵魔人徹底被狼人陣營的大戲帶偏，最終獵錯平民而死。

獵魔人已死，蛋炒飯將忽悠技能點滿，警下被大部分人保下，最後直接票白痴結束比賽。身為血月的蛋炒飯存活到了最後，利用對全場優秀的掌控能力拿到了全場MVP！

不得不說，這場對局可以說是教科書級的倒鉤玩法，不僅狼人自己主動犧牲打掩護，血月也沒有浪費狼人的努力，成功的打亂了平民的思路。即使小編自己在看到最後的位置分布之前都一直覺得11是狼人，看來自己還是太年輕了！

這場比賽也讓「三星飯」戰隊成功保住了積分榜上第三的地位，如果錯過比賽直播的狼友們也可以前往虎牙，關注WPL狼人殺英雄聯賽直播間，觀看當天比賽視頻，或許還能學上一招半式呢？當然，小編現在已經忍不住要上《狼人殺官方》練練從蛋炒飯那學到的完美「倒鉤戰術」了，不知道大家會不會一樣被我忽悠到呢？
