爐石傳說：龍德的另類構築，這一腳下去你可能扛不住

2019-12-24 146條評論 2,810次閱讀 1072人點贊

我巨龍一腳是你能扛得住的啊？

●前言

大家好，我是不整花里胡哨的掌游寶作者：以爾賄遷。真的好久不見了。至於為何這麼久沒寫文，主要是上個版本後期實在太無聊了天梯，沒有什麼構築空間，看我文的人知道我推德都是自己構築的有強度的新意卡組，所以……

至於這個版本，還是非常好玩的，而且薩滿削弱之後的環境真的蠻好的，各種思路也在層出不窮。那麼今天給大家帶來一套55勝率左右的龍德（亮點就是打現在大火的T7獵勝率幾乎百分百）

●卡組截圖

●核心思路

一套牌組出來的價值就是怎麼贏。那麼這套牌的贏點是什麼呢？打快攻就是上牆疊個甲，打中速就是看誰站住場面，打慢速就是兩波伊瑟拉。

總而言之就是中速偏慢靠大龍站場獲勝的卡組。這個版本的主題就是超模，在人均超模的年代怎麼做到最超模，那就是提高自己的費用利用率。何謂費用利用率？比如我7費拍了個作戰傀儡，對面薩滿變JK兩個8-8突襲+5-2的刀，這就是費用利用率上的碾壓。

再舉個例子：JK薩、福報賊的低費給手牌buff，T7獵把每一點費用轉化為傷害它們都把費用利用率做到了最大化。那麼我們怎麼可以把費用利用到最大呢？夢境吐息，光巢，新伊瑟拉。這三張牌作為我們卡組的強點，胡起來也是有摧枯拉朽的力量的，下面詳細講講這些卡。

●單卡講解

◆夢境吐息

這個只要2費準時打出就是強無敵，回到當年費用壓著別人打，每回合都比對面大的德魯伊的年代，除了打一些極限快攻諸如t7獵這種會把手牌很快打完的職業可以在2費有其他2費生物的時候選擇壓場面。

另外，如果起手拿到這個並且配一個龍，就算再大也不要換，因為我們的龍濃度還沒有那麼大，可能就會拿不到龍。另外附一個小tip，如果手上沒龍，打出吐息先結算抽牌，抽到的那張是龍，再結算跳費，是可以跳成功的，各位歐洲人不妨試一試。

◆軟泥教授弗洛普

德魯伊這幾個版本最牛逼的卡沒有之一，而且一個版本比一個版本強。在我們這個卡組最強的就是複製光巢或者新伊瑟拉，前期雙光巢什麼卡組都給我們壓死了，後期雙伊瑟拉也是厚度拉滿，打快攻也可以複製其他的站場。需要注意的就是搞雙伊瑟拉盡量就不要上太多怪了，不然容易把夢境之門卡沒了。

◆香甜的靈力瓜

定向檢索新伊瑟拉和大表哥，可以提前給壓力或者下回合救命，殊死一搏。

◆觸手恐嚇者

另類構築點就在這裡。現在天梯絕大多數的卡組都是偏快的，這也意味著我們這張卡正向收益的概率是很大的，而且娛樂效果也拉滿，給對面t7獵一張7費的奧射我們再反手拍一個177，對面心態就崩了。

總而言之，是有可能一張牌就給我們創造出巨大優勢，但收益確實不穩定。如果不想要這麼娛樂的把他換成別的卡也是可以的。

◆死亡之翼

現在爐石把職業特色做的確實很明顯，德魯伊幾乎沒有解大場面和解大怪的能力，所以在我們沒有辦法的時候，大表哥就是殊死一搏的最佳武器。

但是要注意兩點，第一，最好上了伊瑟拉再大表哥，這樣我們就有持續給壓力的能力了。

第二，一定要到實在沒辦法了再拍，現在連獵人都帶致命射擊了，別的職業解一個大表哥還是很輕鬆的，除了德魯伊，而且就算解不了，上個牆還是可以的，反正我這種臉黑的從來沒有第二抓抓個解牌把牆解了大表哥踹死的經歷。

●起手留牌

起手大部分對局都是找夢境吐息和火樹巫醫，然後是瑞麟騎士，然後這些都需要龍，所以有大龍一定要留，如果打獵人可以考慮留兇猛咆哮。

●對局思路

◆T7獵

別的卡組不敢多說，打這個是究極穩定，幾乎不可能輸。如何不輸？上大哥拍他臉，上牆擋住他的怪，我們6、7費兩個牆，這個獵人是不可能過的了的，這樣他的衝鋒傷害就可以規避很多，然後我們中期的怪，4、5、6攻這些，打臉兩回合就可以把對面血線壓下來，然後我們隨便疊個甲，火樹巫醫找點奶（概率很大，德的回復牌還是多的），就很容易贏下來。因為這個獵人場面不是很強，主要都是直傷，所以上了牆之後走臉就完事了，獵人完全扛不住我們中號怪兩輪踹的。碾壓。

◆宇宙獵

這個就比誰胡了，我們順起來唯一怕的就是生平龍，這個有可能讓我們優勢消失的，其他的話就還好，暴龍王很多時候礙於場面得來解我們的，因為對面宇宙我們不是所以我們前期站場的概率是比對面大的。然後奶要留好，對面的傷害也算得出來，暴龍王，殺戮什麼的，然後對面還有一波紅龍女王比較難處理，這就需要我們先站場用場面來控，不然可能就得大表哥搏命了。均勢。

◆福報賊

比較難打的對局，我們就是做好自己該做的事，不要失誤不要貪，壓力拉滿，賊也是沒有什麼大解場的，除了悶棍要防一手，其他就沒什麼了。剩下的就看對面胡不胡了，胡起來就是比以前更快的任務賊，我們沒有反抗能力，不胡就是比競技場卡組還蠢的卡組，我們很容易就平推了。小劣。

◆JK薩

不管帶不帶任務，現在的薩滿都慢了很多，給了很多我們積累優勢的時間，有足夠的優勢就可以和JK沙導兩波抗衡，如果我們上過伊瑟拉，運氣不太差對面是解不過來的。實在不行就大表哥了。均勢。

◆JK戰

中後期傷害非常高，基本是看不懂了。所以前期我們能壓場的時候壓力還是要給滿的，這個戰士最多帶一張亂斗，所以我們是可以滾雪球的。後面要上牆，防對面看不懂的衝鋒斬殺。均勢。

◆海盜戰

打戰士可以留軟的，這個也是比誰胡了，我們的牆對面過不來，對面前期的場面我們看不懂。小劣。

◆元氣騎

我們沒有甲可以擋住他的傷害，所以就是全力進攻吧！我們胡起來，鬼都擋不住，是他一個小騎士可以控住場面的？巨龍咆哮！均勢。

◆JK園

鋪場賊快的動物園，但是我們的生物質量也很高，對面除了全體加一攻鋪那麼多也沒什麼威脅的。然後我們稍微打點橫掃，蜂群來襲什麼的也能和對面打個差不多。重點是後面，如果我們站場基本就贏了，如果是對面，盡量別交大表哥，等對面JK上了再打，對面就沒法處理12-12了。只要我們不是太黑，都是能贏的。大優。

●小結

總而言之，我們是一套比胡的卡組。胡起來2費吐息光巢加光巢，那真的是捏死別人像捏死螞蟻一樣容易了。這次我們不挨打，我們重拳出擊！希望這套卡組可以幫很多控制卡組玩家（比如我）改掉解場癌，我巨龍踢你一腳是你能扛得住的啊？
