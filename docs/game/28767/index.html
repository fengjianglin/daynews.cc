德杯：明明加賽2場能解決的問題結果又讓RNG變太子隊

2019-12-21 35條評論 1,298次閱讀 168人點贊

★遊戲馬蹄鐵原創

RNG「太子隊」的帽子請戴好

今天是德杯冬季賽的第一天，A組方面沒出太大意外，敷衍了事用二隊的FPX被淘汰，SNG和LGE也都無緣淘汰賽。轉會期變化較大的WE和VG最終晉級。而B組方面JDG很出色打出了4-0的戰績晉級，然而在其他3支LPL戰隊身上卻發生了相當不愉快的事情。開局0-2的RNG，連追2盤打出2-2，他們和LNG以及OMG最終都是2-2，但三方勝負是互咬的關係。

結果裁判的判定很奇葩，在明明可以用平均時長結束的判罰上加了一個150秒神邏輯，導致最終用時更長的RNG居然晉級了，而用時最短的LNG被淘汰。其實這明明是一個再加賽2個Bo1可以解決的問題，卻因為這個讓人看不懂的150秒規則而導致RNG再次被扣上了「太子隊」的帽子。

三方加賽最公平

最佳解決方案：這三支戰隊既然LNG用時最短，那麼就讓LNG等著，先讓RNG和OMG再打一個Bo1，輸了淘汰，勝者再和LNG打一個Bo1，最後獲勝的晉級。

這不是很簡單的方式嗎？偏偏搞一個什麼150秒鬼才邏輯規則，最後還得出一個RNG＞LNG＞OMG，感情今天OMG擊敗RNG的那個Bo1是白打了。如今這版本節奏也就30多分鐘一局比賽，兩個Bo1加起來最多90分鐘，8點多結束加2個Bo1不行？這判決下來，RNG也是莫名躺槍。能晉級固然是好事，但是這種讓很多人難以理解和苟同的方式，無疑是招黑。

德杯當個笑話看？

很多人也是吐槽其實德杯變得太無關緊要，從不少戰隊啟用二隊陣容就可以看出，他們根本不在乎德杯是個什麼戰績，不給參賽名額都無所謂。是「誰愛奪冠誰去奪冠，反正我無所謂」的態度，雖然這對於那些全核心參賽的隊伍是一種侮辱，但事實的確如此。加上今天判決RNG晉級這件事，不少人也是把德杯當成一個笑話看了。這樣的比賽最終的冠軍到底會是誰，從某種意義上，還真是別有一番看點。
