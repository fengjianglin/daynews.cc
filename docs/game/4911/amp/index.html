不經一番寒徹骨，怎來梅花撲鼻香。

我想用這句話形容RNG最合適不過了，就是去年輸給G2之後，僅是輸了一場比賽，這支站隊曾經帶給我們的驚喜和感動就彷彿在某些人記憶中憑空消失了一般，我忘不了那些美如畫的團戰，那些絕地的反擊，那些曾經帶給我無數驚喜的瞬間，這是英雄聯盟的魅力，也是競技的殘酷之處。

在輸掉G2的那個晚上，我有些難受，睡不著覺，轉折難眠，我本該打開手機朝著RNG官博傾瀉我的不滿，可是我又有些心疼，試想下18歲的你此時坐在那裡，坐在uzi的位置，坐在小虎小明他們的座位上，面對這麼一場失敗，你們又是什麼感受呢？失敗當晚，悔意纏身，誰能從這樣的挫敗中再次站起？

那時，我們感覺遭到了背叛，凡是有助於這一想法的謠言都順理成章，什麼打假賽、隊內不和、膨脹，都不過是我們自欺欺人跟自己過不去而已，誰有真正在乎他們？他們到底承受了多少？

今年，不是最有希望的一年，我依舊欣賞這支隊伍，欣賞他們的每一個走位每一次GANK，每一個開團的瞬間，那種跟隨場上局勢而起伏跌宕的心情，贏只不過是最後的甜點。

S9 RNG能進世界賽，在我心裡已經超出預期了，無論什麼成績都能接受啊……畢竟常規賽結束的時候，誰能想到RNG能成為S9二號種子呢？怕是那個時候大多數人都覺得RNG今年去不了世界賽了……

相信現在RNG的粉絲也已經無所畏懼了！只要RNG出現在世界賽，只要uzi還在打，我就滿意！沒有一個隊能夠永遠是冠軍，能永遠巔峰的。

RNG這一年熬過多少苦難從多少嘲諷質疑謾罵聲過來的，那些狂熱的冠軍粉早就在s8結束的時候離開了，留下的都是陪他們走過了這磕磕絆絆的一年的人。經過s8的失利，春季賽的絕望，夏季賽的絕地逢生，我們現在這群陪他們走過巔峰也熬過低谷的軟粉，還會在意八強嗎？

失敗不可怕，RNG最不怕的就是失敗，被狠狠打入泥潭中又在沒人看好的時候重新站起來的RNG，才是最可怕的。

縱有疾風起，皇族永不言棄。
