RW俠不敵KR隊伍CW提前出局，丟了kpl的人，改名“RW夾”算了

2019-12-24 29條評論 2,925次閱讀 85人點贊

王者榮耀冬冠杯預選賽的比賽進入了最後一個比賽日，在前面幾場比賽當中estar戰勝了XQ，Hero則是不敵TS，本來以為要等到最後一場比賽才能大結局。但是沒想到RW俠竟然輸給了KR的隊伍CW，使得這次的預選賽提前大結局了。這也是KPL隊伍第一次在正式比賽上輸給KRKPL的隊伍，不少網友都表示RW俠可以改名為「RW夾」了。

這場比賽RW如果獲勝的話，是有機會在最後一場比賽當中和estar一較高下的，誰也沒有想到他們會輸，但是他們卻和所有人開了一個大大的玩笑。他們在陣容選擇上就有些冒進了，前面幾手選擇劉邦加馬超的突臉陣容，射手位是虞姬。但是後兩手竟然拿出了楊戩和司馬懿，這兩個英雄都不是版本強勢的英雄，最致命的是他們這個陣容一旦逆風，清線能力非常差。幾乎沒有什麼翻盤點。

兩分鐘的時候渡難的楊戩就走位失誤，被趙雲大中交出一血。從此RW就陷入了逆風，就像之前對於陣容分析的那樣，司馬懿這個英雄逆風的情況下根本沒有清線能力，縱使有一身本事也要被沈夢溪按在中路。中期的時候RW也有機會，他們拿到了先知主宰，但是在下路又出現了大失誤，他們想要推掉CW的下路一塔卻被完美反打，CW一波一換四並拿到暴君。十三分鐘的時候RW竟然先手開了一個蘇烈，結果蘇烈反身一個大招，虞姬，楊戩和劉邦全部被秒。渡劫的馬超被曜牽制，而剩下一個司馬懿完全沒有清線能力，CW一波推掉水晶創下了境外賽區的記錄。

這場比賽RW失利很大程度上是因為他們的掉以輕心，這種陣容選出來明顯是沒把KR隊伍當回事，沒有考慮過自己前期陷入逆風的情況，而比賽當中也多次出現了失誤。境外賽區第一次在正式比賽中戰勝KPL隊伍，對於比賽的發展來說也是好事，也說明了不能小瞧任何一支隊伍。RW俠輸掉這場比賽之後不但提前出局，還會背負巨大的質疑聲，很多人都說他們太丟人了，倒不如改名「RW夾」。估計隊員們的心態也會受到非常大的影響，在休賽期RW應該會做出比較大的調整，目前的首發陣容應該會換掉1-2個人吧。

大家對RW輸給境外賽區的CE有何看法呢？歡迎在評論區一起討論吧。
