慶魚年！找回自我的楓哥終於打破連敗魔咒

2019-12-24 744條評論 1,861次閱讀 479人點贊

2019年12月22日絕對是值得銘記的一天。

在這一天VG完成敗者組一穿五成就，用冠軍獎盃撕開了聚集在西恩刀塔粉絲頭頂的陰霾；同樣是這一天，曾經的yyf現在的胖頭魚用一盤絕活拍拍，把纏繞在自己身上數天的連敗魔咒給無情拍開了。

雖然楓哥一直說這個版本感覺自己變強了，但殘酷的現實告訴他這可能只是自己的錯覺。這幾天的對黑中，不管楓哥這邊的班子如何變化，隊友換了一批又一批，但不變的是楓哥永遠贏不了。劣勢被碾壓，優勢被翻盤，就連五五開的局勢也可能因為一波團戰沒打過導致瞬間崩盤，這幾天時間楓哥嘗盡了各種各樣失利的滋味，把自己的頭打暈了，自信心也打沒了。

之後就陷入了惡性循環，越輸越不會玩，越不會玩就越贏不了，贏不了要被噴，被噴了心態還要炸。打到後面楓哥已經變得畏畏縮縮不敢說話，不能有自己的想法，跟著隊友沖就完事；不能去刷錢，一刷錢就是毒瘤沒有用。一個常年霸佔三角區的人突然不讓他刷錢，箇中滋味大家可以想像一下，那簡直比殺了他還難受。

這把軍團錢都不敢刷，經濟一直在倒數，但好在最後時刻連續幾波關鍵決鬥都秒掉了黑鳥，艱難翻盤拿下一局，為之後的「過年」作出鋪墊。

第二局楓哥拿出了3號位小小，又是熟悉的配方熟悉的味道，不能發育，不敢打錢，最終非常憋屈的輸掉了比賽，雙方重新回到同一起跑線。

然而到了決勝時刻，實在受不了的楓哥在遊戲開始前就跟鯛哥商量，自己來打1號位痛痛快快的刷一場，即便輸了也不留遺憾。於是鯛哥選出了發條技師，而楓哥掏出了這個版本非常強勢的拍拍熊。

雖然說著要痛快刷一場，但拍拍這個英雄是沒辦法當刷子大哥來玩的，沒有很好的AOE技能，也不會出電錘狂戰點金這樣的發育型裝備，拍拍真正強勢的地方在於中期打架非常兇殘，刷到後面還真不一定打的過別家大核。

最後的結果大家也知道了，楓哥憤怒的拍拍熊抓著對面一陣亂拍，最後拿下13殺4死12助攻的完美數據，全場拍出了4W1的輸出，親手終結了環繞在身邊數天之久的不勝魔咒！

fgnb！

在已經取得巨大領先優勢，破對手兩路時，彈幕上已經刷起了「YYF你準備好了嗎？」，不知道的還以為是TI6決賽在打呢。

在對手敲出GG，基地爆掉的時候，彈幕上已經被「fgnb」鋪滿，直播間禮物也爆發開來，大家等待這一刻已經等了很久很久了。

收穫勝利後，楓哥和兄弟們還專門出去吃了個「慶功宴」，這輪來之不易的勝利確實需要一頓飯來慶祝一下。

看看楓哥胸前的兩個大字，NB就完事了！

可惜的是對黑結束後，收官局的大聖沒能贏下來。大聖作為楓哥的野生絕活勝率一直不高，這個在topson手上秀來秀去的英雄楓哥玩起來總感覺哪裡怪怪的，不是在樹上see就是被暈下來落地see，究竟什麼時候大聖這個野生絕活能變成楓哥真正的絕活呢？

我們拭目以待！
