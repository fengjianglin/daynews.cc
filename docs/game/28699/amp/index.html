Roguelike遊戲的“精神內核”，讓生活中的選擇變成“Lucky開局”

天天要聞
2019-12-22

遊戲是一種把「現實藝術化並誇張表現」的表現形式，我們可以在其中體驗到只有想像中才會有的虛擬世界，可以盡情的釋放自己現實生活中的壓力，讓自己的精神得到休息和沉澱。而Roguelike類遊戲則是遊戲圈中的一個「另類」，每一次體驗都會給予玩家與眾不同的全新體驗，隨機和意外是Roguelike類遊戲里的一大特點，讓其一直都散發著獨特的魅力吸引著大批玩家。隨著近幾年遊戲圈的快速蓬勃發展，越來越多的遊戲都引入了「Roguelike」的元素，可以說Roguelike類遊戲對遊戲圈整體的變革有著非常重要的影響力！

何為Roguelike類遊戲？

Roguelike類遊戲是什麼？很多玩家對於這個辭彙比較陌生，但不能否認的是，幾乎每一個喜歡玩遊戲的人都或多或少的接觸過Roguelike類遊戲，並且對於這一類的遊戲都有著比較好的體驗。Roguelike類遊戲用簡單的解釋就是——它們是RPG遊戲的一類型，並符合幾個特點：「不可挽回」，一旦失敗就得從頭再來；「系統複雜」，遊戲內容設計的複雜，不僅限於裝備、技能、對戰體系等等；「極高的自由度」，在Roguelike類遊戲中，你往往可以做到很多其他RPG遊戲中做不到的事情；「進程單一性」，搭配不可挽回形成單一進程的遊戲模式，比如《元氣騎士》就是一個很好的例子。「隨機生成性」，現在很多遊戲都是學習Roguelike類遊戲這種隨機生成性內核，讓每一次遊戲體驗都有全新的感覺。比如暴雪旗下的《暗黑破壞神》系列遊戲，其中的地下城出怪與寶物掉落機制就是引用了Roguelike類遊戲的「隨機生成性」，從而大大的提升了遊戲樂趣和遊戲壽命。

優質的Roguelike類遊戲

目前遊戲圈裡不乏有許多優秀的Roguelike類遊戲，雖然Roguelike類遊戲的「特性」是固定的，但遊戲的全新思路設計就看遊戲設計者的「腦洞」有多大了。《元氣騎士》作為一款特別成功的Roguelike類遊戲， 正如製作它的涼屋遊戲所說「堅持做有節操有創意又能賺錢的好遊戲」一樣，元氣騎士堅持了「像素風」的美工設計，在這種設計風格的大框架下，儘可能的把遊戲人物刻畫的特點鮮明。在遊戲設計上，每一個遊戲人物都有專屬職業，這些職業擁有隻屬於他們的獨一無二的技能。關卡設計也是同樣秉承了Roguelike遊戲的精神內核，完全隨機的掉落與地下城房間分配，造就了你永遠猜不到自己本次冒險會有怎樣的旅程。

除了元氣騎士以外，由Motion Twin製作的《死亡細胞》也是一款Roguelike類遊戲的代表佳作！Motion Twin是法國著名的遊戲開發商，旗下的優質遊戲作品不在少數。死亡細胞幾乎涵蓋了所有遊戲平台，PS4和任天堂的NS（Swicth），包括之後的PC版本都備受玩家好評，而IGN也給出了9.5分以及TGA給出「最佳動作遊戲獎」的超高好評！死亡細胞的設計靈感來源於「銀河惡魔城」，並在此基礎上增加了更加豐富的色彩渲染，使得整體遊戲畫面給人一種「融合Roguelike樸素+色彩打擊絢麗」的優秀體驗。在遊戲操控方面，讓玩家能從無盡的隨機地下城中，遭遇各式各樣的敵人，並且極富有難度！

還有一個不得不提到的經典Roguelike類遊戲，它的誕生告訴了遊戲圈一個全新的認知，那就是Roguelike遊戲可以是任何形式的，並非局限於傳統RPG闖關類的，這款遊戲就是《殺戮之塔》。殺戮之塔是一款純正的Roguelike類遊戲，但它也是一款純正的卡牌類遊戲！殺戮之塔讓我想起了小時候和同桌特別喜歡玩的一個遊戲，當時我們會用一張作業紙給對方設計迷宮，然後把這張紙捲起來讓對方從一個入口進入開始隨機選擇，在重複了無數次失敗之後總有一條正確的路是通往終點的。殺戮之塔就有這個因素在，不過不同的是殺戮之塔可以縱觀整個迷宮的概況，路線選擇完全取決於你自己。不過至於你想選出最簡單的出路，那也是不可能的，因為Roguelike類遊戲的不確定和隨機性，讓你無法預知每條路線上會遇到的怪物或寶物。

隨機和意外是真正的Roguelike內核

不管是何種類型的Roguelike遊戲，他們的「精神內核」永遠不會變，正如玩家們給予Roguelike遊戲的定義，只要有一點不符合，那就不能稱其為真正意義上的Roguelike遊戲。隨機和意外永遠是Roguelike類遊戲的內核，大多數玩家第一次接觸這類遊戲的時候，都會很快的以Gameover而告終。也正是因為這種「Gameover」的巡迴往複，才讓Roguelike類遊戲更加充滿了挑戰。而每一次挑戰遇到的全新的問題，不管是技能搭配還是BOSS變化，每一次的變化讓我們擁有多一次的遊戲體驗，也讓我們的「遊戲功力」得到進一步提升！

獨特的遊戲魅力下給予玩家的「反思」

Roguelike類遊戲總有其獨特的遊戲魅力，難度高而生命只有一次是Roguelike類遊戲的內核，也正是因為如此，玩家從Roguelike類遊戲中得到了現實中的許多反思。我們的生命又何嘗不是一次Roguelike遊戲冒險呢？生命中所發生的一切，雖然會根據你的人生抉擇和努力程度發生不同的軌跡改變，但與Roguelike遊戲同的是仍然是充滿未知和不確定性的，並且你的人生軌跡也和Roguelike遊戲一樣只有一次。隨機和意外造就了Roguelike類遊戲每一次體驗的與眾不同，這種恰好是我們現實生活中不能體驗到的，因為生命不能重來，但遊戲可以。

把生活變成Roguelike似的「lucky」開局

在Roguelike遊戲的世界裡，我們會謹慎的對待每一次抉擇，每一場戰鬥，特別在你有「lucky」開局的時候，更會珍惜且謹慎的前行，因為失敗了重來，可能你就不會再次如此幸運。在生活中又何嘗不是這樣呢？我們越努力越珍惜的對待生活，那麼我們得到的也會是美好的結局，而人生雖然沒有再一次開局，但我們可以把生命中每一次抉擇都努力做到最好，那麼生活中的無數種「未知」，就會回饋給我們最好的那一個「lucky」開局。
