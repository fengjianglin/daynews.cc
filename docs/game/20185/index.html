部落衝突：女皇飛盾相爭一姐之位，蠻王閉門不出，詠王泡茶看戲

2019-12-18 2,851條評論 4,445次閱讀 1034人點贊

話說那《部落衝突》手游上市已久，一直以來在戰場上共有三位英雄各領風騷：大守護者為部隊提供後援，是士兵們最為信任的堅實後盾；野蠻人之王不僅是最早登場的英雄角色，而且以兇悍的近戰攻擊聞名；而弓箭女皇則以最高的傷害和唯一一名女英雄身份穩坐《部落衝突》一姐之名。三者一直也是互有千秋，彼此相安無事。

2019年末，這種微妙的平衡被打破，原來是《部落衝突》更新了十三本，同時引來了一名新的英雄登場：飛盾戰神。新來的妹子不僅擅長攻擊防禦型的建築，目標還可地可空、可上可下、可攻可……咳。不僅戰鬥力極強，還是個身材健美的火辣御姐！

看慣了面若冰霜的女皇，新妹子不僅面相親和，戰鬥力也不容小覷。不僅如此，官方還發布了一張蠻王和飛盾並肩看日出的浪漫插圖！一時間首領之間議論紛紛，士兵們也是一片嘩然。難道說一直以來蠻王和女皇只是假官配，飛盾才是正主？曾經的一姐女皇即將地位不保？

士兵議論紛紛，幾名英雄則是反應各異：身為身處八卦風眼最中心的蠻王，最近的日子不太好過，因為走到哪都能引來男兵們”原來如此”、”蠻哥牛B”的表情和女兵們”渣男”、”始亂終棄”的指指點點，已經被逼的弄得乾脆閉門不出；而大守護者還是每天抱著書本講課，貌似事不關己高高掛起，實則泡好了茶準備看韓劇情感大戲的樣子。

而本次事故的兩位當事人飛盾和女皇就有意思了，兩位女英雄各有千秋，身前也是聚集了不少簇擁：曾經士兵們在打仗時備受敵人防禦塔折磨，而新來的大姐大雖然初來乍到，卻能用自己手上的長矛和盾牌對對方建築予以打擊，一時間被前線士兵視作英雄。另一個則繼續面若冰霜、精神抖擻，保持一箭一個的兇悍輸出，準備為女皇之威正名，引來身邊的遠程DPS們陣陣尖叫。一時間前線戰場上花開兩朵，各有千秋。這邊首領剛剛收到女皇擊落了敵軍多少單位的戰功，那邊前線飛馬來報飛盾又率軍攻破了對方的據點堡壘。

隨著兩位女英雄的鬥爭愈演愈烈，部隊中甚至有了互立山頭之勢，首領們既高興又頭疼。要說作用吧，女皇在麾下效力已久，戰功卓著，是輸出部隊的頂樑柱；而飛盾初來乍到，卻能幫助首領們剋制敵方頭疼不已的防禦塔，屢立戰功。偏偏這個時候蠻王又慫的藏起來了不敢站出來表明態度，看來這個難題只能拋給首領們，只有經歷更多的考驗，才能證明兩人誰才是名副其實的”一姐”呀！
