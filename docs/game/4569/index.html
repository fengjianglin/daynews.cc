魔獸世界懷舊服教你如何在戰歌峽谷里用知識改變命運

再熬幾天就要迎來我們盼望已久的戰場了，12月11日魔獸世界懷舊服將開啟大家期待已久的戰歌峽谷和奧特蘭克山谷戰場，相信不少老兵萌新們已經按捺不住了吧，那麼我們來提前了解下如何打好戰歌峽谷的一些小妙招吧！

為什麼常常會聽到這句話，知識可以改變命運！沒錯，在魔獸世界懷舊服裡面這句話同理，沒錯就是工程學！工程學這個神奇的專業技能，還有兩個分支，分別為：1.地精工程學，讓你變得簡單又粗暴，2.侏儒工程學讓你覺得滿滿的科技感。有了它們的加入，讓你的PVP變得更有樂趣，讓敵人更加的畏懼！

沒錯這些裝備在這個60年代還是可以帶進戰場的，就讓我們充分的利用它們吧（正式服裡面已經體會不到了哦）

地精火箭頭盔

雖然戴上後看上去傻傻的，但是它的作用可不簡簡單單就是字面上的意思，讓你無論是什麼職業都擁有了一個強大的先手控制神技，盜賊的悶棍，法師的變羊，細想下要你要是個盜賊起手悶棍對面章解了，一個火箭頭撞過去，它就只能傻傻的看著您扛起旗幟，跟他打個招呼，悠哉的離去，他會不會吐血三升呢！（還可以利用他來移位哦，例如對面復活點剛復活，衝上來一個科學怪人，然後他就只能靜靜的望著你）

地精火箭靴

看樣子有沒有讓你想到星球護衛隊裡面的公爵，有了它就可以到處的飛來飛去的，飛的同時也是有著危險的哦，對沒錯”爆炸”，有可能會炸飛你！這就是地精學的簡單粗暴，讓你比別人多個疾跑的同時，也有可能帶你去見天使姐姐。

侏儒火箭靴

這個基本和地精火箭靴作用相同，但是副作用就溫柔太多了有沒有，只是熄火大不了就是沒加速而已。起碼不至於推你去死吧。

侏儒隱形裝置

讓你多一個盜賊的消失，讓你瞬間脫戰！就是技能CD時間有點不盡人意，一個消失讓對方失去目標，救了你一命啊，還要啥自行車啊。（注意自己身上的DBUFF，要是有DOT傷害的話，剛消失就被燒出來了，那你就有點浪費神技了）

貓眼超級護目鏡

抓賊神器有沒有，相當於一個小感知了，要是賊對賊，人類帶工程，這再不能不先手你都對不起這個遊戲了！當初我的野德也是靠它插旗戰場，讓對面的賊懷疑自己是不是忘記開潛行了的存在。

希望這些小技巧能幫你，讓你在戰場飛檐走壁，無往不利！昨天5號魔獸世界懷舊服又新增1組經典懷舊服伺服器。塞卡爾——PvP，沖啊，帶上你的好基友們！
