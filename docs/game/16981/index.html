夢幻西遊：殺雞儆猴還是割肉補瘡，封禁工作室對夢幻意義何在

2019-12-14 430條評論 2,187次閱讀 928人點贊

在159聯武神壇進行前夕，夢幻西遊對代練工作室進行了一系列的清理。有一部分工作室的賬號遭遇了七天限制登錄特殊待遇。其中包含一些「非知名伺服器」的服戰玩家，在與GM溝通後得到的答覆是武神壇可以正常參加，但打完比賽回到原伺服器後還是有7天的限制登陸時間。

和以往交仙玉贖身的封禁方式不同，這次的小範圍處罰似乎並沒有特別嚴厲，限制7天的登錄只是一個黃牌警告.也沒有去動紫禁城釣魚島這些大的服戰工作室賬號，實在是有點殺雞儆猴的味道。封禁的賬號也僅僅圍繞「代練」這個詞展開（不得不說下，見面會玩家提問題的環節是真的有托），因為在剛剛結束的廈門玩家見面會上，主策劃小奇明確表示不贊同「代練」這種遊戲行為。沒想到這麼快就動手了。對於這次封停風波，小兵有以下幾個觀點可以共同探討。

●夢幻西遊玩家的年齡結構

夢幻西遊已經正式運行16年之久，玩家結構也發生了重大變化。大部分玩家十七八歲接觸夢幻，現在30歲左右（我們幫的幫戰主力都已經超過30歲，最大的已經37歲接近40歲。服戰玩家因為接觸不多不太了解）這個時候正是處於事業的巔峰期，空閑時間一點點減少（越成功的時間越少）一些單純的工薪階層可以下班後五開來消磨一下時間順便補貼家用，有能力加大投入的其實都沒有什麼時間在遊戲里浪費更多的時光。這時候「代練」就起到了很大的作用，他們可以讓經驗跟上消耗。尤其在靈寶這個經驗大熔爐推出後，經驗顯得尤為珍貴。

在這個時候對「代練」動手，時間點卡的極為精準。如果下一步繼續加大打擊「代練」的力度，這一部分主力幫戰人員面臨的就只有兩個選擇1.賣號不玩。2勉強支撐直到有一天賣號不玩。可能有一部分玩家會把投入轉嫁到口袋版這個吸金利器，這可能也是策劃的目的。

●夢幻西遊玩家的玩家結構

現在夢幻西遊高端服戰人群除了極個別被忽悠來投資夢幻的「老闆」外，只剩下一些靠夢幻發家的高端牟利人群。某知名服戰玩家依靠直播平台短短几年時間靠直播和賣零食躋身服戰頂端，還有一些「服戰高端玩家「其實也是靠在夢幻倒賣裝備才積累起來的財富。真正捨得投入的情懷玩家數量日益減少，想在夢幻中撈取利益的越來越多。「為了掙錢而遊戲」幾乎成了夢幻玩家心照不宣的口號。這也是藏寶閣這個熱詞要高於夢幻西遊本身的原因。玩家更在意的不是遊戲本身，而是能夠把遊戲換成RMB的平台。

在這次的封禁工作室背後，其實透露著夢幻西遊這個遊戲的危機。如果只是單純的殺雞儆猴，告訴高端牟利人群你們在獲得利益的同時要兼顧一下遊戲的發展，讓遊戲更長遠。要真的是加大力度剜肉補瘡，相信等待夢幻的將是進一步衰退的結局。
