2019-12-05

摘要

DPS綜合排行

當前階段懷舊服的熔火之心和奧妮克希亞的巢穴兩個副本已經進行了三個多月，各項統計數據有了十分豐富的樣本，就算是小眾職業也已經有了數萬的統計量。那麼當前各大職業的傷害排行究竟如何呢？

我們先來看看綜合dps實力排行，該統計的時間為兩周內，統計範圍是MC的10個boss：

對比一下兩個月前的統計數據：

我們可以發現，隨著裝備的提升，絕大部分職業的平均水平都有了長足的進展。狂暴戰和盜賊雖然平均水平幾乎不動，但是最高傷害卻有相當可怕的提升！

圖中標註的近攻職業潛行者（也就是盜賊），平均DPS(75%傷害數值)達到了417.04，而極限DPS是1324。狂暴戰雖然整體比盜賊弱，但是上限卻高出不少，達到了1594！比起兩個多月前小O進行統計時的880高出了700多DPS，不得不說是相當恐怖。

被稱為開荒利器的獵人由於當前不存在開荒的情況，所以屈居第三，平均水平較高，但是上限甚至不如法師。

三大布衣差距並不明顯，分別排在第四到第六的位置。法爺最高數據為1204，是第三個dps過千的職業；術士最高為913，也很不錯；向來就上限很低的暗牧最高只有670。

其他混合職業——薩滿、小德和騎士由於樣本的增多，dps能力也反映了出來，但是平均水平還是不如術法牧。比較值得注意的是top99%的貓德玩家打出了較為亮眼的數據，不過期間付出的艱辛（用壞了多少手動懲戒器）只有自己心裡清楚。

懷舊服進行到當前階段，依舊還是物理系統治天下，不僅均值高，上限也高。

PS：圖怎麼看？

關於第一個圖，裡面其實一共包含了6項數據，從左到右分別是：10%排名數值、25%排名數值、50%排名數值、75%排名數值、95%排名數值和99%排名數值。

而排名的統計方式是按照「75%排名數值」的高低來進行的，因為這個數值最能反映職業的整體情況。

單個Boss的DPS排行與分析

1號BOSS：魯西弗隆

職業分化最明顯的一個boss，由於部分職業有其他任務，dps基本不具備參考性。狂暴戰與盜賊的dps最高值均達到了2000+。

2號BOSS：瑪格曼達

比起焚化者古雷曼格可以靠兩狗毛傷害，瑪格曼達可以說是比較公平的木樁了。我們可以看到基本上的排名和上一段分析的boss綜合輸出相近，狂暴戰的上限一騎絕塵，高達1477，盜賊也達到了1047之高。

3號BOSS：基赫納斯

和1號boss相似，1boss+2小怪可以一起aoe，狂暴戰和盜賊的盛宴。

4號BOSS：加爾

由於小怪數量眾多，當前很多碾壓團隊都採用直接強A的打法，雖然平均水平還是狂暴戰和盜賊，但是傷害上限最高的卻是具有最強aoe能力的法師，達到了2k+。而由於有多線操作，加爾也是暗牧表現最佳的boss。

5號BOSS：沙斯拉爾

同樣是物理系碾壓法系的boss，貓德表現亮眼。

6號BOSS：迦頓男爵

終於有一個boss是法系領先物理系的了，全賴boss的aoe機制導致近戰無法正常輸出。

7號BOSS：薩弗隆先驅者

同樣是強A的1boss+4小弟，法師盜賊表現搶眼，而狂暴戰上限和下限都很明顯。

8號BOSS：焚化者古雷曼格

本來應該是第一木樁的boss，因為兩狗的原因，導致很多職業可以毛傷害，所以這個boss的dps沒什麼太大的參考性。

9號BOSS：管理者埃克索圖斯

同加爾，由於小怪數量眾多，當前很多碾壓團隊都採用直接強A的打法，雖然平均水平還是狂暴戰和盜賊，但是傷害上限最高的卻是具有最強aoe能力的法師。電薩表現亮眼。

10號BOSS：拉格納羅斯

雖然最高紀錄還是狂暴戰，但是其平均水平實在比較低。因為對於普通團隊而言，大螺絲還是比較適合遠程輸出，所以該boss遠程輸出的平均水平較高。
