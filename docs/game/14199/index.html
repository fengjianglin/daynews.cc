夢幻西遊雄哥27天反超濤哥到達150層地宮 怪物難度恐超自在天魔

2019-12-10 249條評論 1,252次閱讀 152人點贊

前陣子西柵老街的雄哥通過了地宮142層，來到了地宮149層，但是因為賽季重置的關係，並沒有見到傳說中的150層。吃瓜群眾們議論的焦點在於濤哥的百萬129隊伍與雄哥的真愛夢幻隊伍，誰將在下一個賽季登頂。

珍寶閣的新賽季開啟的較早，濤哥的隊伍每天都會購買至少7個火把。濤哥一路過關斬將很快打到了110關。

此時雄哥也加入了冷酷攻略群,集中了冷酷攻略群的兄弟們的集體智慧，雄哥以全新的針對性陣容開啟了新賽季，雖然每天也會購買3個火把，但是因為賽季開啟較晚僅僅才爬到50層。

11月20日時，雄哥發來截圖，顯示已經通過139層，狂追60層差距！極限反超濤哥！

當天晚上，雄哥已經通過141層，並且告訴小編，前141層全部都是一次性過關！避免了火把的浪費。

11月23日，雄哥再創記錄通過了地宮149層！終於要揭開地宮150層的神秘面紗了。

地宮149層截圖如下：

149層的輸出並沒有142層高但是封系非常多，雄哥購買了8個逆境寶寶給一速佩戴，每回合一個逆境，天陣配合花果山和凌波城暴力點殺，19回合速推了149層。錄像因為某些原因暫時不能公開。

此時是賽季的第27天，離賽季結束還有25天，每天4個火把計算，雄哥有接近兩百次機會挑戰地宮第150層，小編也是非常的期待看到地宮150層的通關錄像，然後感慨一句：人類的智慧是強大的。

作為最終BOSS，150層的怪物分布表面上卻異常樸素，沒有花妖甚至連俳優都沒有，也沒有出現新型的暴力輸出怪，看起來好像甚至沒有142層難，但是直覺告訴我們，事情沒有這麼簡單。

怪物分布，2隻自愈雨師百姓，會失魂人物，場上怪物不滿則無限分身。

2隻虔誠千年蛇魅更夫：高神，第2回合後每回合會逆境解封全場，每回合會搖2個鈴鐺。

2隻暴怒的混沌獸捕快：靈身，高法爆，捕快會掛上搜查狀態並且攻擊帶有」搜查」（虛弱）狀態的單位傷害結果加倍。

3隻狂豹：酒鬼被打醒或被逆境後造成多倍輸出，富豪每回合神跡並反間隨機一隻寶寶，捕快會掛上搜查狀態並且攻擊帶有」搜查」（虛弱）狀態的單位傷害結果加倍。暴怒是狂怒+高必。

1隻自愈的金身羅漢護衛：每回合群血，如果出現殘血單位會活血回復大量氣血（這層的情況基本活血就是近1W），場上任意單位被物理攻擊時，會無限保護直到不能保護。

難點一：怪物血多回復超高

第150層出現了一個全新的怪物——自愈的雨師百姓，殺過地宮的朋友會在第65層初次遇到百姓怪物，特點是當場上怪物數量不滿的時候，會召喚一個與自己同樣名稱的滿血百姓怪物，並且召喚出的百姓也可以繼續召喚。

場上只有一隻百姓的時候，優先擊殺百姓即可。但是當場上有多個百姓的時候，就沒有辦法用簡單的辦法點殺了。

擊殺2個百姓辦法有2種：

第一種辦法：1回合擊殺掉場上其中一個百姓和7個以上怪物，並在後續的回合中每回合擊殺剩下的百姓並有多餘輸出打到新召喚出來的百姓，幾回合之後就會出現2隻百姓同一回合被擊飛的情況。這個辦法大多出現在地宮爭霸賽中。

第二種辦法：先清理場上其他有威脅的怪，讓百姓分身，然後將每個百姓打到殘血，然後同一回合擊飛掉對方場上的所有百姓。這一個辦法在地宮120層、地宮140層都會用到。

但是150層出現的百姓和之前的完全不一樣！兩個自愈的雨師百姓，雄哥集火點殺一隻都不能看到他自愈的氣血上限，經過計算這2隻自愈的雨師百姓每回合會自動回復8600點氣血！這意味著不管是磨血雙殺還是用輸出硬點都會非常困難！

為了過關雄哥特意測試了每個怪物的數據。自愈的和尚護衛會每回合自動保護被物理攻擊的怪物，推血回復的同時自己每回合還會回復6992的氣血，不受禁療瘴氣以及2級法寶「夢魂香囊」的影響，這讓同時擊飛雙百姓點殺難上加難。

難點二：怪物特別抗封

雄哥封印3次均未封住，所以決定使用「死亡召喚」技能來測試對戰方怪物等級。

測試原理：消耗的魔法-10即為對戰方怪物等級。測試結果為對戰方怪物全部為156級。

原本129級的封印在挑戰模式中就會受到「無忌」的影響降低16%封印命中，這下對戰方等級也不再是129級，而是直接高了27級，讓封印怪物變得幾乎不可能。

因為沒有175級的隊伍打到第150層，所以也無法測試175級面對的是多少級的怪物。

同時就算有封印命中，更夫後綴的怪物也會每回合使用逆境將大部分法術封印技能解除（部分封印例如鎮妖除外）。

難點三：怪物輸出非常高

地宮150層的狂豹因為等級提升，輸出也提升了非常多，結合雄哥過關地宮142層的錄像從數據上物理輸出看是地宮142層的2.5倍。（142層無虛弱飛機為700）

法系輸出大概是142層的1.5倍（142層有羅漢平秒秒1277,150層無羅漢平秒秒4000）。

第1回合如果沒有羅漢並且未封住怪物，不論物理還是法系只要出現一個暴擊就會倒人。

錄像只有2個回合，全地宮中傷害最高的酒鬼怪物甚至還沒有出手！酒鬼怪物初始傷害為富豪的2倍，並且每回合提升10%傷害結果，隨著回合增加最高可以提升到10倍的傷害結果，（85層肥胖的畫魂酒鬼測試數據）。如何抗住狂風暴雨般的輸出，並同一回合擊殺對戰方的雙雨師百姓呢？

難點四：召喚獸出手空間小

除了富豪會每回合反間召喚獸之外（可以帶神跡套解），場上的兩隻蛇魅每回合會丟4個鈴鐺給挑戰者的召喚獸，就算吃了同心肉脯和滿層凜冽氣召喚獸加上滿級馭獸術，也禁不住對戰方不要錢無CD的鈴鐺狂搖。

唯一克制鈴鐺的辦法是攜帶TG使用掌心雷，（因為對手是蛇魅更夫所以催眠無效），掌心雷與目標等級有關，高達156級的蛇莓能否在自己寶寶跑光之前封住呢？

難點五：百姓雙抗超高，神技力劈完全失效。

雄哥的人物不暴擊的情況下可以打4000傷害左右，但是大部分輸出都會被護衛吸收。類似女魃墓的法系門派大多沒機會見到這一層，所以也無法測試對方法抗如何。如果極限加點則一定扛不住這一層怪物的輸出。

經常打100層以上地宮的玩家都知道，力劈對地宮中非物理怪傷害非常高可以達到3000-4000，在地宮中比100%高連的傷害都要高，雄哥也依靠著五隻耐力劈成功秒掉了142層的混沌球捕快以及和尚更夫，但是力劈150層的自愈雨師百姓，卻只能打出1000左右的傷害。

雨師總血量有4W3，每回合可以回復8600血，還得同一回合擊殺掉2隻。這可能是這個遊戲最瘋狂的挑戰。

六、可行的過關思路

雄哥嘗試了好幾天之後並沒有太大的進展，首先頂尖的硬體都不能讓對戰方百姓磨掉一滴血不說，不帶血敏寵的情況下就連對戰方2狂豹2混沌的輸出都很難抗住3回合。

唯一的過關思路在於新的資料片「靈寶風雲」，經過冷酷攻略群的測試和數據推算，無底洞的靈寶煞魂冥燈是可以降低百姓的氣血上限，地宮戰鬥回合數上限為150回合，將雙百姓的氣血上限降低50%，變成21500，再配合靈刃真陽等靈寶出現暴擊，惡報無視護衛的保護可以直接同時擊殺兩隻雨師。一回合同時秒掉2隻滿血百姓。

剩下的問題就是如何在封印命中如此低的情況下，硬抗10倍傷害的狂豹酒鬼堅持112個回合併讓WDD使用2次靈寶？這恐怕得靠無限神佑了。

地宮150層不知何人能夠突破，但是玩家的智慧是無窮無盡的，希望早日看到玩家攻略地宮150層！不知各位豬友對於如何通過地宮150層有什麼看法呢？覺得濤哥可以過關地宮150層嗎？可以在評論區留言哦。
