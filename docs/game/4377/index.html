懷舊服8種最佳秘葯附魔 你知道在哪獲取嗎？

2019-12-07 0條評論 15次閱讀 0人點贊

摘要

一、 什麼是秘葯？秘葯是懷舊服中能為頭部或腿部附魔的一種道具，由瑪瑟迪斯·火芒和博學者萊德羅斯提供

一、 什麼是秘葯？

秘葯是懷舊服中能為頭部或腿部附魔的一種道具，由瑪瑟迪斯·火芒和博學者萊德羅斯提供的任務獎勵獲得。直到第四階段祖爾格拉布「原始哈卡萊神像」上線前，秘葯都會是你頭部和腿部的最佳附魔選擇。

二、 瑪瑟迪斯·火芒提供的秘葯

瑪瑟迪斯·火芒位於燃燒平原（65,23），會提供5種不同類型的秘葯獎勵。

  • 次級體質秘葯：使生命值提高100點。黑鑽石×1肺片雞尾酒×1體質聖典×1夜龍之息×130金
  • 次級適應秘葯：火抗提高20點。黑鑽石×1尖刺水晶×4燃燒精華×1恢復聖典×130金
  • 次級冥思秘葯：法力值提高150點。黑鑽石×1被折磨者的血液×1砂囊口香糖×1沉思聖典×130金
  • 次級堅韌秘葯：護甲提高125點。黑鑽石×1堅韌聖典×1防禦水晶×4卡加爾之眼×130金
  • 次級貪婪秘葯：敏捷、力量、智力、精神或耐力中的一種屬性+8。黑鑽石×1貪婪盛典×1鞭根塊莖×4精神水晶×430金

三、 博學者萊德羅斯提供的秘葯

博學者位於厄運之槌的圖書館內，提供了3種不同類型的秘葯。

  • 防護秘葯：躲閃幾率提高1%。防護聖典×1原始黑鑽石×1大塊魔光碎片×2磨損的憎惡縫合線×1
  • 專註秘葯：傷害和治療法術效果提高8點。專註聖典×1原始黑鑽石×1大塊魔光碎片×4暗影之皮×2
  • 急速秘葯：攻速提高1%。急速聖典×1原始黑鑽石×1大塊魔光碎片×2英雄之血×2

四、 秘葯所需材料獲取方式

  • 各種聖典
  • 每一種聖典都沒有固定的掉落方式，擊殺50級以上的精英怪物就有幾率獲得，刷高級地下城是最好獲得聖典的方式。
  • 原始黑鑽石
  • 原始黑鑽石有地下城或團本中的精英怪物掉落，在後期的祖格職業附魔都需要用到，非常稀有。
  • 黑鑽石
  • 黑石深淵大多數怪物都會掉落。
  • 大塊魔光碎片
  • 分解51級以上精良品質（藍裝）裝備可獲得。
  • 肺片雞尾酒、砂囊口香糖
  • 詛咒之地重複任務獲得。
  • 夜龍之息、鞭根塊莖
  • 費伍德森林「被腐化的草藥」任務獲得，詳細攻略可以查看懷舊服地圖賺錢攻略（2）：費伍德森林。
  • 尖刺水晶、防禦水晶、精神水晶
  • 安戈洛環形山收集水晶獲取，詳細攻略可以查看懷舊服地圖賺錢攻略（1）：安戈洛環形山。
  • 燃燒精華、被折磨者的血液、卡加爾之眼
  • 黑石深淵內的黑暗寶箱有幾率掉落。
  • 英雄之血
  • 可以在西、東瘟疫之地找到，有難度，詳細攻略可以查看懷舊服地圖賺錢攻略（3）：西瘟疫之地。
  • 暗影之皮
  • 僅通靈學院可獲取，擊殺精英怪及BOSS後有幾率掉落。
  • 磨損的憎惡縫合線
  • 斯坦索姆內的縫合怪掉落。
