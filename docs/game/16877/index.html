陰陽師決戰京都高分攻略陣容分享 這幾個式神搭配可打出千萬傷害

2019-12-16 2,878條評論 4,596次閱讀 1018人點贊

陰陽師近期開啟的決戰京都活動，估計不少玩家都在攻略中，而作為活動核心的攻略打法和陣容，不少玩家也在探索，試圖用最效率靠譜的陣容打出最高分，雖然不同的敵人設定和屬性不同，但依舊有很泛用和萬金油的陣容，本次就來提供幾高分攻略陣容，這幾個式神搭配可打出千萬傷害。

PS：以下式神速度分先後順序

群體階段：保底6000W傷害起步

火靈追月神+塗佛三生命小袖+雪幽魂土蜘蛛清姬+三生命書翁+星滅晴明+土蜘蛛兵主部茶几/針女歌姬兩面佛

群體階段最佳攻略陣容，保底6000W傷害起步，除非你主力輸出是暴擊不到50%，完全猜拳的御魂屬性，一般基本上沒啥壓力，群體階段的陣容基本上就是這個固定不變的。三個增傷神器可以將茶几的額輸出最大化的發揮出來。

如果覺得陣容不穩可以考慮選擇將書翁替換下來換成招財蜜桃，一速二速都OK，效率也不低，而且安全性更高。如果懶得養小袖之手，可以帶雙茶几來打輸出，當然前提是你有兩個茶几，茶几+兩面佛部隊件，追月神供不起這樣的火。

單體階段：

狂骨荒骷髏醜女+三生命書翁/招財久次良+狂骨/破勢雙酒吞

單體最佳陣容打法，目前離吞吞已經成為了主流的攻略單體BOSS的打發了，塗佛是增傷利器必須要有，至於破勢和狂骨帶什麼看玩家個人的儲備和練度，醜女御魂基本上就是卡在荒骷髏比較好，如果供不起可以考慮心眼或者破勢。書翁和久次良沒什麼好說的，兩個都是增傷的神器，如果你酒吞就練了一隻，那久次良和書翁可以一起上場。
