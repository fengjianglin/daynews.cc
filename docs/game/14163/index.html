本周更新內容搶先看，新系統、玩法開啟、驚喜不斷

2019-12-10 2,219條評論 3,280次閱讀 308人點贊

大陸的小可愛們，大家好呀~這次小狸用好吃的紅蘋果收買了風大製作委員會的某工作人員，成功得到了本周更新的相關爆料！
想了解本周究竟會開放哪些功能玩法的守護者們可千萬不要錯過哦~

一、魂卡-副卡系統開啟，打造自己的專屬卡集
魂卡是各位守護者在冒險旅途中必不可少的助力之一，而如何對已有魂卡進行篩選鑲嵌更是每位守護者心中的頭等大事，於是【魂卡-副卡】系統就這樣誕生啦~系統開啟後，大家可在副卡卡槽內裝備滿足相應條件的紅卡及閃卡並享受該魂卡已激活的部分屬性。多個卡槽，多種搭配，專屬角色等你打造！
二、全新頭銜開放。「神裔」降臨風之大陸
小可愛們期盼已久的新頭銜將在本周更新後開放啦，頭銜已經達到「聖者」的守護者們可以進一步向「神裔」邁進了。「神裔」共分為5個等級，激活頭銜即可享受對應屬性加成，頭銜達到「神裔Ⅴ」的玩家還可額外解鎖專屬頭像喲~

三、「永恆戰場」開放，組隊探索贏取功勛！
據小狸的可靠線報，本周將會開啟全新玩法「永恆戰場」，角色等級達到要求的小可愛們可以與夥伴們一起組隊探索哦~究竟是選擇不斷搶佔新的據點還是守住我方陣營的已有據點就看大家的謀略啦~參與該玩法還可獲得大量用於兌換機甲養成材料的功勛幣哦~
四、英雄蒼穹之塔第五層開啟，九尾妖狐「玉藻前」等你挑戰！
英雄蒼穹之塔開啟第五層，已獲得挑戰資格並成功通關前4層的守護者即可對九尾妖狐「玉藻前」發起挑戰。別看小姐姐長得清秀，實力可是不容小覷，快來與好友組隊一探究竟吧~
看完以上「偷跑」消息，各位守護者是不是已經按捺不住了呢~12月12日更新維護結束後大家就可以進入風之大陸一探究竟啦，祝願大家都可以愉快享受全新旅途哦~
