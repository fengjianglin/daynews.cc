爐石傳說：希爾瓦娜斯皮膚實際測評，獵爹還是獵爹

天天要聞
2019-12-13

隨著新版本的更新,希爾瓦娜斯女王的皮膚終於與大家正式見面了,不知道大家有沒有入手這款超值的皮膚呢?小編我第一時間購買了女王的皮膚,就讓我為大家帶來實際的測評。

女王的登場特效是被一團漆黑的死氣所包裹,並不算所么華麗,但很契合她作為被遺忘者的女王的身份,這點非常讓我滿意。

相比登場特效,女王的技能特效稍顯平淡,作為對比我覺得不如個人最愛的麥格尼和逐星,這點確實有些不足。

不過女王的皮膚最讓我驚艷的還是台詞,實在太贊了,冷酷中帶著些許慵懶的語調著實誘人,完美還原了遊戲中女王的風範,如果要給這款皮膚打分的話,說滿分十分還是有些誇張了,但九分的水平絕對是有的,非常值得收藏。

值得注意的是商店更新之後女王的皮膚不能直接用戰網點購買,而是需要花費符文石。

點擊更新後商店右下角符文石旁邊的綠色加號就能進行購買,比例是一元等於一符文石,而女王皮膚的售價是六十符文石。

符文石可以購買除卡包外的所有東西,而卡包則需要消耗奧術寶珠購買。

奧術寶珠能夠通過符文石轉換。

同樣是點擊右下角奧術寶珠旁邊的綠色加號就能轉換,而這次更新之後爐石傳說也是更新了新的任務。

任意模式完成三場對戰就能獲得一百八十奧術寶珠的獎勵!小夥伴千萬不要忘了完成啊。

當然還有不少已經入手了女王皮膚的小夥伴們很想知道新版本應該玩些什麼獵人卡組,在這裡小編非常推薦宇宙獵!

獵爹還是獵爹,新版本來了宇宙獵的地位照樣不可撼動。
