爐石傳說：狗賊被騷套路連打近兩條命，新版本上線套路就是多

天天要聞
5 天 ago

新版本開始了，很多玩家們紛紛投入天梯與狂野之中了，畢竟新卡肯定要好好試一試啦。而我們的狗賊叔叔卻在狂野模式里被一個全新套路給虐，而且那傷害將近打了兩條命了！今天我們就來看看我們的狗賊如何被虐的吧。

首先狗賊選用的是戰士牌庫，而對面走的潛行者。狗賊先看對面不叫無人生還，而是走的迦拉克隆就沒多說啥了。估計走的就是新​版本的迦拉克隆體系，變英雄牌。狗賊就很無所謂，覺得迦拉克隆打狂野太弱了。

在這之後，看到對面潛行者居然直接下了寒光智者，想這個盜賊走的是爆牌賊吧。但看對面牌都是什麼一些不屬於爆牌賊的體系牌，狗賊越來越弄不明白的這潛行者玩的是什麼！下背叛又拚命的過牌，而且隨從牌都是一些給的跟班，這玩的什麼亂七八糟的，難道是什麼OTK之類的？

然而謎底終於在潛行者到了10費後謎底揭曉了！先是直接下了鯊魚之靈，接著打下兩張跟班牌（騰出手牌位置），之後下幽影潛藏者，觸發兩次戰吼，給了5張幸運幣。用了5個幸運幣這後用暗影步將潛藏者拉回手牌，之後再下又獲得6個幸運幣，再用掉再用暗影步拉回了潛藏者，再讓潛藏者上場再獲得6個硬幣。這時候狗賊已經慌了，網友們說這個是大范，狗賊卻驚呼這個應該是鬼靈。此時狗賊還有29點血，狗賊覺得自己不會一擊必殺的。而對面的潛行者打出5個硬幣後，拿出了BOSS：鬼靈竊匪配和這鯊魚之靈的連擊觸發2次的效果打出了2次27點的爆炸傷害，加起來有54點傷害了，近兩條命。狗賊都驚了還能這麼玩！

可能還有些網友不明白是怎麼回事，小皮就給大家介紹下：首先先上了鯊魚之靈效果是你的隨從連擊和戰吼觸發兩次，再下了兩個隨從（騰出手牌）。下幽影潛藏者，戰吼翻倍就可以獲得6個幸運幣，用了兩次暗影步將幽影潛藏者移回手牌觸發了3次戰吼，瘋狂使用幸運幣，之後再打出鬼靈竊賊。鬼靈竊賊的效果是連擊：造成傷害，數值等於在本回合中你使用的其他牌的數量。這個回合一共打出了27張牌，雙倍連擊打出了54點傷害！所以網友們說想玩這個騷套路手速一定得快，一回合打出27張牌手速不行能玩嘛。
