想要馴服《方舟：生存進化》手游的班龍？一定要記得白天去找它

2019-12-07 0條評論 16次閱讀 0人點贊

摘要

世界之大無奇不有，在這個世界上存在的形形色色動物中不乏有一些&quot;夜行俠&quot;，它們往往會在晚上活動、白天睡覺，貓就是最為典型的代表之一，不過在遠古的恐龍時代中，會不會有這樣的夜行俠存在呢？

世界之大無奇不有，在這個世界上存在的形形色色動物中不乏有一些”夜行俠”，它們往往會在晚上活動、白天睡覺，貓就是最為典型的代表之一，不過在遠古的恐龍時代中，會不會有這樣的夜行俠存在呢？對此《方舟：生存進化》手游就為我們做出了一個猜想，它做出一個名叫班龍的恐龍模型，它就是一個名符其實的夜行俠，只在晚上活動，且實力堪比霸王龍，因此你在馴服的時候要千萬注意，只能在白天去找它。

《方舟：生存進化》手游的班龍的白天一般會在礦洞裡面睡覺，晚上再出來活動，因此你就需要在白天的時候找到礦洞進去看看，要是能發現班龍的話，就在它的身邊建造一個馴獸房，能控制住它的行為就行，然後餵給它們班龍喜歡吃的竊蛋龍蛋飼料便可以將它們給馴服，但是一定要白天去，晚上不管你怎麼暈班龍都不會被暈眩，你也沒辦法馴服它。

被馴服了的班龍依然保持著野生狀態的習性，你白天就算把它給叫出來了也沒有絲毫的作用，可一旦過了晚上的8點20（遊戲時間），班龍就會瞬間起了勁，它在晚上的實力還是很強悍的，甚至能和《方舟：生存進化》手游裡面的霸王龍拼個高低，而且班龍在《方舟：生存進化》手游中還能叼起來一些小型的恐龍，是整個遊戲裡面都很少見的存在。

總的來說，班龍在《方舟：生存進化》手游中還算得上是一個比較好馴服的恐龍，在遊戲前期也有比較大的幫助，只不過在你馴服它之後你也要和它一樣，變成一個夜行俠在晚上活動，這點不知道大家是不是可以接受呢？對於這樣天天在晚上行動的班龍，大家有什麼看法？可以在下方評論區留言討論。
