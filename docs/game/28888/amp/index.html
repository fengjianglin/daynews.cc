新版征服者皎月，搭配冰杖無限回血，對面集體自閉

天天要聞
2019-12-21

在當前英雄聯盟版本中，皎月黛安娜的熱度呈穩步上升趨勢。想用法師打近戰英雄的玩家， 皎月就是一個很有潛力的選擇。現在的皎月不管是出裝、對線、團戰、連招都和以前發生了翻天覆地的變化，設計師把皎月重新定義了一遍。通過把【月神衝刺】變為她的基礎技能，我們使黛安娜的核心機制之一得以提前解鎖，並且允許她在達到6級之前也能和對手進行戰鬥和互動。這個改動還讓她在打野時也能在遊戲初期發起有效的gank，這對於打野黛安娜的影響力來說是一個巨大的提升。

另外，現在黛安娜新的終極技能讓她隨著遊戲的進行有了更多的選擇，使她獲得了之前沒有的參與團戰及小規模衝突的能力。這還讓黛安娜在進入戰場時有了更多的變化，讓她的目的不再僅僅是一整套連招秒殺單一敵人而已。整體而言，這些改動的目的在於保存玩家們喜愛的特點，同時給予她應有的手段與變數選擇。我們希望黛安娜玩家可以繼續舒服地保持現有的遊戲風格，同時讓黛安娜的技能配置對她的新玩家們而言更加有趣且更加有益。
最近我發現新版皎月被動第三下攻擊會觸發冰杖的減速，而且還是範圍減速，於是我就有了一個大膽的想法，經過試驗給大家推薦一手，征服者粘人不死流皎月，肉的一批的同時傷害還巨高！

天賦：征服者、凱旋、歡欣、堅毅不倒。副系：護盾猛擊、復甦！

皎月的W護盾不僅有AP加成，還有10%的額外生命值加成！由於護盾有兩層，實際相當於20%的額外加成。顯然我們出一身血量法強裝，也能在保證輸出的同時自己還有很高的坦度，如此可以達到最大化收益！

裝備推薦 前中期：香爐、布甲鞋、納什之牙、冰杖

皎月的護盾可以觸發香爐的被動效果，加上自身的被動，攻速能突破2.0。平A停不下來，狂刷半斬！再配合征服者，回血能力非常強。而冰杖配合攻速裝，2.0的攻速兩秒一次被動，只要粘到就別想跑

後期：深淵、大面具

深淵的被動，附近敵方英雄多承受10%魔法傷害。自身承受來自英雄傷害時，會回復15%傷害值的法力，花費法力時，會回復20%法力消耗的生命值，每次施法最多回復25生命值！配合征服者回血能力超強！

而面具提供的額外生命值和被動的百分比傷害加成在保證傷害的同時也提升了站擼的能力。有了冰杖的強力減速，大面具就能始終保持最高百分比生命值的灼傷傷害，同時，香爐和復甦的護盾加成，讓皎月在後期每隔5秒多就有六百的護盾，達到一個不可思議的坦度，頂在人堆里拉大招的感覺那是真刺激啊。

  1. 前期對線，皎月在一級的時候一定要先出w，為什麼呢？因為w它不僅具有傷害，而且還具有保護性的功效，非常有利於自己去發育。在還沒到六級的時候一定要先慢慢發育，在和對面中單對線的時候，你站在自家的小兵的後面來進行補刀。補刀有個小竅門，那就是在小兵的血快沒有的時候，上去A小兵就可以了。還有如果對方英雄對你攻擊的時候，你可以開啟w保護的功能就可以。W開啟後即使被對方的傷害打到，自己的血不會掉的那麼快。
  2. 中期在出到冰杖香爐納什，此時皎月迎來了自己的第二波強勢期，冰杖香爐提供的血量和護盾保證了皎月的坦度，可以讓皎月站的更久。而納什之牙和香爐被動則給予了高額的攻速，配合冰杖可以瘋狂粘人。雖然不像純法傷那樣輸出爆炸直接秒人，但是這樣可以給團隊帶來更大的收益，面對這樣一個法坦，打也不是不打還又被粘著跑不掉，集體自閉。
  3. 後期補上深淵和大面具，這樣皎月的抗性和血量就足夠了，深淵和面具提供的額外傷害也很好的增加了皎月的輸出能力，當個法坦一樣有著不俗的持續輸出！

這套打法優勢在於，從對線到結束都能有著不俗的傷害和坦度，搭配冰杖的留人很是煩人！面對持續輸出的英雄可以一直風箏對面。而劣勢就在於不能像純法傷那樣傷害爆炸，一套秒人。當對面有強控英雄時要注意自己的進場時機，不要被對面控住集火秒掉。尤其是對面有小法這種，E技能的眩暈以及大招的百分比斬殺，都是皎月吃不消的

以上就是筆者研究的冰杖皎月，希望對您有幫助！祝排位連勝！夜裳花凰對鏡弄妝，晨香舞鳳也喜天翔，我是夜裳舞晨香，一個熱愛遊戲的作者！
