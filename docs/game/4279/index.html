決勝局中shy哥時選出了自己的冠軍皮膚英雄劍姬帶了屏障點燃，而Bwipo則是拿到了奧拉夫帶了屏障雪球。

在前期對拼中shy哥劍姬是佔據了優勢，Bwipo奧拉夫想要吃血包被shy哥抓住機會成功拿下一血。

但在大優勢的情況下shy哥劍姬也出現了同樣的失誤，想要去吃對方半區的血包，導致被Bwipo奧拉夫追著白打了半條血，加上最關鍵的大招沒能打出4破，被Bwipo奧拉夫拉開，最後一個E技能成功擊殺shy哥劍姬，最終Bwipo贏下這局solo賽，晉級到了決賽！
