史上最強山寨公司！從涼涼邊緣到年入10億，還被玩家瘋狂吹捧？

2019-12-12 462條評論 2,738次閱讀 1215人點贊

大家好，這裡是正驚遊戲，我是正驚小弟。

國行NS發售，機子單日銷量破十萬，任天堂股價應聲上漲，這一場持續30多年的入華之旅，終於開花結果。此刻的老任「熱淚盈眶」，畢竟這麼多年來頭鐵入華，腦袋都差點撞破了……

老任確實讓人心酸，遊戲的銷量、口碑、評分、影響力在業界都找不出對手。銷量前十的單平台遊戲，老任一人獨佔8個，史上評價最高的十部遊戲，它也包攬前5，走到哪，都要被叫一句：「大哥」！

在全世界呼風喚雨的它，唯獨無法在中國掌控雷電，還被傷得心都碎了……

今天就和大家嘮叨下，老任心塞的30年！

一、自家遊戲機被山寨個底朝天

1983年，老任發售FC紅白機，迅速在日本本土火爆，一年賣出300萬台，之後火到世界各地，順帶拯救了被「雅達利大崩潰」整殘了的美國。到2003年紅白機停產時，FC在全世界的銷量超過6000萬台。

當然，FC的影響力不只表面的6000萬，因為在老任「屆不到」的中國市場，幾乎家家戶戶的小孩都買過「山寨版FC」。

紅白機在海外大火的年代，28歲的「段永平」（現步步高集團董事長）負責的小廠年虧損200萬元，眼看就要涼涼，他敏銳嗅到FC的在中國商業價值，帶著工廠轉攻電子遊戲機市場，於是便有了後來的小霸王。

靠著山寨，這家小廠從年虧損200萬到後面成了了年產值10個億巨無霸！

二、看不下去了，老任第一次入華

小霸王在國內賺得盆滿缽盈的時候，老任卻沒沒半毛錢業務,心在滴血啊……

從1994年開始，老任就開始坐不住了，他們通過中國香港的萬信公司開始向大陸發售自家的遊戲機！先後授權了超任（SFC）、紅白機（FC）、掌機Game Boy……

但結果都不理想，即便是車輪戰，也都被盜版打得很慘。

當時的環境就這樣，國民的經濟水平一般，正版FC機子要上千，卡帶一兩百，還得靠關係才能買到，幾個月不吃不喝才能買台機子。小霸王就不一樣了，機子便宜，卡帶更便宜，5塊錢能買個8合一、16合一的。

最騷的是，小霸王冠以學習機的名譽推廣開來，那時候的小孩，有誰不想好好「學習」，學習使我快樂啊！（調查下，當年有誰藉以學習之名買小霸王的，舉起手來小弟看看！）

而萬信作為代理又無權本地化，沒法給英文、日文的原版遊戲打上中文，正版在盜版面前喪失了最後的競爭力，以小霸王為首的山寨盜版幾乎統治了此時中國遊戲市場。

三，藉機生蛋，老任第二波攻勢來了

第一次入華失敗後，時間來到2000年，一紙遊戲機禁令打得老任措手不及。

從外部進來行不通後，老任打起了從內部突破的注意。2003年，老任甩開萬信，找上矽谷大佬、美國華裔「顏維群」博士在大陸合作成立了「神遊」公司！

而且在同年，老任在大陸開始發售以N64遊戲機為原型改版的「神遊機」。

這次老任做足了功課，特地為大陸玩家量身定做了機子。不要998，只要498正版主機帶回家，相關大作也只要48，玩家用自己的「神遊卡」就可以隨時下載，可謂是充分考慮了中國國情！

可是誰也沒想到，這時候中國互聯網的發展如此迅猛，以《熱血傳奇》為代表的韓國網遊蜂擁而至，索尼PS2也接肘而來，並且向中國玩家瘋狂示好，各種打折促銷。

一下子平白無故多了那麼多對手，誰頂的住啊。

雖然困難重重，老任幹勁還是很足，又推出了改版的Nds掌機「 iQue GBA（小神遊）」，小神遊憑藉遠低於水貨的價格、便捷的服務在國內取得了些許成績（大約賣了30萬台）。

賣機子是不賺錢的，重要的是賣遊戲，苦逼的是老任的遊戲送審之路太過坎坷，神遊DS上的遊戲至今也僅過審6款！沒有了遊戲的支持，玩家選擇了盜版和水貨！

互聯網的發展，給盜版提供了肥沃的土壤，在那個正版意識薄弱的年代，如果你說自己玩正版，會被很多人嘲笑：傻不傻，有免費的還去玩正版……」

可憐老任，又一次撲街……

四：瘋狂蹭奧運熱度，老任第三次入華

沒有遊戲是原罪，神遊的沒落是註定的事，但這些都沒打消任天堂佔領中國市場的決心，這次老任聯合神遊放出了殺手鐧——Wii遊戲機！

老任的機子更新換代很快，Wii遊戲機是他們的佼佼者，機子在全球範圍內圈粉無數，老外都玩瘋了，連英國女王伊麗莎白二世、美國總統奧巴馬等大佬都被「征服」。

07年，神遊把Wii遊戲機拿去審核，為了送審之路過得順暢點，老任還蹭上中國全民熱點「08年北京奧運會」。

他們與世嘉合作，一起拿到了國際奧組委的授權，做出了奧運會唯一官方制定遊戲《馬里奧與索尼克在北京奧運會》！

說實話，這一招「討好中國玩家」的手段非常高明，如果順利的話，沖著這波唯一官方授權，指不定遊戲還能火一陣子。

然而，現實是殘酷的，直到北京奧運會結束，這款「國際奧組委唯一官方制定遊戲」也沒有過核，同樣沒過審的還有Wii遊戲機。

老任在中國又雙叒叕敗了，一口老血噴得老高……

五：牽手騰訊，1234再次一次！

前面接二連三的失敗，讓老任對中國市場心懷敬畏，這塊待開發的蛋糕還是依舊誘人，但老任已經學乖了，靜靜的等待市場變化，2013年，遊戲機禁令解除，老任也不急著出手（心理都玩出陰影來了）。

而後，隨著中國玩家生活水平的提高，正版意識逐步增強，大家轉變了觀念：玩盜版比遊戲是不尊重人行為，買款遊戲就跟吃頓飯一樣，吃霸王餐是不道德的。

2017年，任天堂新一代掌機switch出爐，銷量爆棚，口碑也很好。老任發現，機會來了！

這一次，老任穩打穩算，在中國找了個最強的合作夥伴——騰訊！「企鵝」的能量大家都清楚，QQ和微信的渠道能力、影響力在中國都是獨一檔的，是根竹子插在兩大渠道上，都能開朵花來。

老任看重企鵝的渠道，想要藉此打開大陸市場，騰訊看重任天堂的一大票優質遊戲，想要拓寬自己的遊戲業務。兩人把手一牽，推進了這次國行NS的發售。

不管從什麼角度來看，這次都是機會最成熟的一次。市場環境好，玩家有追求高品質遊戲的需求，又有騰訊「地頭蛇」牽線，一切水到渠成，國行NS發售單日就賣了十多萬台。

30年一朝圓夢，老任也算是了卻一樁心愿……

至於國行NS的整體表現，在小第看來，總體誠意還是比較足的，機子價格，售後服務也算給到位了，遊戲價格也算打了折，估計未來國行的NS遊戲只會比外面便宜，不會貴，畢竟好不容易進來了，總得把中國玩家「伺候」好了。

為了避免沒遊戲玩的尷尬，騰訊也做了三手準備。老任自家的遊戲、海外其他大廠的、還有國內精品獨立遊戲，三管齊下，管飽！

新人玩家可以大膽的剁手，國行NS整體還想蠻香的。早就買過港版的，日版的，小弟覺得不用重複買了……（不差錢的當小弟沒說）

小弟有話說：如果有人在擔心到時候想玩的任系大作又在審核的路上出幺蛾子，這個問題嘛，大家可以看看隔壁PS和XBOX，指不定未來NS也會出現個小「BUG」啥的，當然這只是小弟的猜測。所以，也可以先觀望，未來在做決定。

一個正驚問題：你最希望國行NS推出哪款遊戲？
