“吃雞”測試服上線“圈內圈”模式：官方制裁LYB？

2019-12-09 0條評論 3次閱讀 0人點贊

摘要

2017年的第一季度，一款大逃殺類型的遊戲《PUBG（PLAYERUNKNOWN』S BATTLEGROUNDS，絕地求生大逃殺，吃雞）》橫空出世，自此，這座名為Erangel的島和每局100個玩家的「傳奇」故事開始了。

玩家四處找物資點獲取裝備武裝自己，並在逐漸縮小的遊戲區域內彼此死斗，直至個人或者隊伍勝出，這就是一般概念上的獲勝流程。當然，也有其他別出心裁的戰鬥方法，比如在獲取了基本的裝備後，就提前前往遊戲的「毒圈」內部，在邊緣逡巡，甚至一直消極避戰，直至最後一刻才在圈內發起突襲。

這樣的打法的使用者一般會被受害者稱為「老陰B（LYB）」，就戰術意圖上來說，雖然看起來有點陰險，但其實在某些時候確實是行之有效的，可以彌補一些裝備和槍法上的不足。然而，官方看起來是不太鼓勵這一戰術的。

地圖出現藍色區域

在PUBG的實驗室官博上，官方人員表示他們這段時間正在測試一個叫「藍洞模式（BlueHole Mode）」的新機制，它會在地圖上提前顯示下一次縮圈大小和位置，而提前進入這個藍圈內的玩家在蹲點守人時會受到傷害（等同圈外懲罰傷害）。

官方表示，開發這一機制的原因有二，一是因為此前縮圈是隨機的，運氣成分可能導致遊戲的局勢變化甚至勝負之分；二是由於縮圈位置不可知，當玩家的跳傘位置不好，又沒交通工具，圈還縮得遠，很大幾率就會進入「長時間跑圈→沒機會發育→死了」的路線，最大的致命原因反而不是其他的敵對玩家。

在測試服TPP / FPP模式 Erangel 4人小隊的隊列里，這一模式將會開啟，從比賽開始直到倒數第二個安全圈，新機制會在原有的安全區大圈內再生成一個藍圈，代表著下一個安全區刷新的範圍，提前進入其中的玩家將受到與下一階段毒圈相同的傷害。
