換皮站擼肝又氪，《新笑傲江湖》，還是我們熟悉的國產武俠RPG

2019-12-20 2,542條評論 3,480次閱讀 2574人點贊

往上看，關注我。（￣︶￣）↗

年底了，國內各家遊戲大廠都開始打出了自己的”2019年度收官大作”，這一周，是完美世界的全新武俠RPG手游《新笑傲江湖》。遊戲測試期熱度就不小，Taptap上有將近30萬人預約，今天開啟雙平台公測後，這款大廠大作到底品質如何呢？一起來了解下。

略帶水墨風的畫面不算差也不驚艷，總體感覺很眼熟

現在的國產手游，頭一個拼的就是畫面。但這次的新笑傲江湖，從畫質上來說，並沒有什麼令人驚艷的感覺。畫質細膩度，光效豐富度等方面，相比此前的同類手游並無太大提升（甚至有後退，不少玩家認為就是低配版楚留香）。可能是優化不足，掉幀和鋸齒現象也時有發生，影響觀感。

放大後可看到人物服飾上明顯的鋸齒

而畫風方面，遊戲打出”國風神韻”的旗號，帶有一點水墨風格。但這個水墨風可能表現最強的地方就是過場動畫了，在實際遊戲中，看無論人物造型還是場景布置，都還是典型的國產武俠RPG風，對於玩慣了該類遊戲的玩家來說太過眼熟。同時在很多細節方面表現不足，比如劇情對話時的人物面癱臉，又令畫面體驗有所降低。

國產武俠RPG玩法套路，和沒有什麼打擊感的戰鬥

畫面上沒什麼大的突破，那麼玩法呢？就更套路了。劇情對話過任務，掛機打怪做日常，再來點社交結婚生孩子，就完事兒了。基本上沒什麼需要詳細介紹的必要，唯一能感覺有點亮點的，是在社交玩法中加入了類似”朋友圈”的江湖功能，玩家可以在其中記錄心情，發布語音，炫耀服飾等等，順便認識全國各地的玩家。

戰鬥方面，因為是MMORPG不是什麼動作遊戲，所以也就別指望什麼打擊感了。平A無法移動，迷之距離判斷，動作僵硬銜接不流暢，只有技能效果方面表現還可以。總體來說，站擼就行了，而等到開啟自動戰鬥掛機後，連站擼都用不著了，感覺彷彿去了一個假的武俠江湖。

品質不夠流量來湊，當紅炸子雞代言應該花了不少錢

無論畫面玩法還是戰鬥，新笑傲江湖可以說是一個徹底的換皮之作，無創新套路化。那麼，又是怎樣吸引了近30萬人預約的呢。除了完美老牌遊戲廠和笑傲江湖的IP外，功勞都要記在宣傳身上。在測試期，遊戲就放出了製作極其精美的CG動畫，圈了一批重視畫質的玩家關注（CG畫質≠遊戲畫質，請各位玩家牢記這一點）。

其次，請來了今年爆紅的流量小生肖戰代言，不僅各種花式宣傳，還特意在遊戲內加入了肖戰獨有劇情線，甚至於連很多伺服器名字都緊貼肖戰（肖煙四起、一世為戰之類），不留神還以為進了明星粉絲應援群呢。

當然，這些大幅宣傳和紅人代言耗費的錢，都需要在玩家們身上找回來。時裝、月卡、通行證、充值禮包、成長基金，反正氪金方面該有的全都有。不想氪就去肝，各種副本日常活動休閑玩法等等，不多說玩家們也應該都懂的。

總之，在同類遊戲中已經有了楚留香、天刀等產品的情況下，以新笑傲江湖目前的表現，真的找不出什麼可以玩的理由，新玩家嘗個鮮可以，老玩家們就不用去自找難受了。
