DNF：12月劍魂“大削弱”！一天之間，幻神職業跌成二線

2019-12-18 2,022條評論 4,626次閱讀 1650人點贊

DNF迎來了大洗牌更新！100級似乎夢回85版本，裝備百花齊放，職業CP外加三次覺醒，技能新效果配合三覺新技能，足夠玩家們體驗蠻久的，這就是地下城憋了接近1年的大改版，但國服玩家卻漸漸發現了職業平衡很不對勁，測試服在12月18號更新了100級版本，但劍魂和劍宗這些職業一看就想哭！你以為的特色加強，只不過是SPK的禮包套路而已。

12月劍魂「大削弱」！一天之間，幻神職業跌成二線

劍魂作為國服人氣最高，槽點也是最多的職業，是最先得到無我劍氣這個強大技能，因此受到了下水道玩家的集體抱怨，尤其是12月打樁比賽，多個跨區被劍魂屠榜，1000E傷害說來就來，太強了！可沒想到，劍魂沒爽幾個月就要涼了……

在100級開放的那一刻，所有職業都得到了「無我被動」，比如純C中最弱的審判，加了30級技能「堅強的意志」，與無我劍氣效果一樣，紅眼也是如此，得到類似的「無我暴走」提升，粗略估算是20%的傷害提升，哦吼，劍魂再也不能一騎絕塵！國慶套已經下架，劍魂的幻神體驗卡也到期了，是時候讓紅眼、天帝這些職業上位了。

在這樣的情況下，劍魂史詩級「大削弱」啊！不是你變弱了，而是其它職業變強了，DNF95級之後一直都這麼改，很少削弱某個職業，而是通過加強指定職業來變相削弱，禮包銷量輕鬆漲！

不過在18號的職業洗牌中，一些職業僅加強10%左右，因此劍魂沒跌太多！在女鬼劍三覺的背景下，從幻神跌落到了二線水平，大概是20~30名。

這麼一來，劍魂真是慘！完全是DNF的最強背鍋職業，熬了這麼多年，好不容易當幻神了，還被其它職業各種酸，而12月的100級版本上線後，直接打回原形，跌落到二線！要不是還留有斬鋼這個純C開頭，會跌落到40名之外。

劍魂玩家們，記住12月18號這一天，你的幻神寶座沒了，又回到了下水道附近徘徊！簡言之，被酸了3個多月，最後全是一場夢，還把面板弄丟了……
