9.23版本香爐大幅削弱，卻讓皎月，吸血鬼吃香，中單位直接起飛

2019-12-21 708條評論 4,411次閱讀 1484人點贊

馬上隨著新版本到來，在9.23新版本中香爐被進行一波調整，沒想到調整後的屬性和一些英雄完全符合，結果直接成了排位中的BUG。我們先來看看原本熾熱香爐的詳細數據。

這次的版本改動中熾熱香爐移除了讓你的隊友普攻回復生命值的效果。這是一個對於射手上的削弱，但是保留了額外攻速加成和普攻造成額外傷害的效果，皎月和吸血鬼依然是這件裝備的受益者。

吸血鬼第一件出香爐，也是性價比極高的一種選擇，移速的加成，10%的治療效果，10%減CD，60的法強配合吸血鬼，妥妥的穩搭，但是並不是就是第一件非出不可，不過大家可以試試這種出法，可能有更好的效果！

再說受益者之一的重做版皎月，在這次的改動中，設計師對皎月可以說是進行了一波史詩級的加強，大招的位移效果改到了E技能，使得皎月在前期就擁有了的突進能力，對線能力得到提升，同時抓人能力也在中期上升了不少！

對於法刺皎月來講，香爐僅需要2300金幣，價格極其便宜，可以為皎月增加10%的冷卻縮減，30%的攻速加成，60點法術強度，8%的移速，還可以增加皎月W的護盾值，50%的基礎法力回復也可以彌補皎月前期的藍量，可以說這些屬性全是皎月需要的屬性，同時性價比也是極高的！

在韓服高端局最近出現的皎月新打法，就是裸香爐，同時也是FPX中單在直播證實的一種玩法！

Doinb作為一個遊走型的中單，在一局排位中使用皎月也是選擇第一件裸香爐，表示這裝備真是太強了，數據直接超神，嘗試了下吸血鬼出香爐，發現也是很強。香爐自帶的10CD配合天賦可以在十級擁有30%的CD，所以出完香爐可以直接轉帽子沙漏法穿棒，增加中期的傷害，團戰基本就是無腦的。
