有關DNF12.10版本的活動內容，目前策劃已經在官網上發出了具體的更新公告，而在其中除了那些早在前幾天體驗服就已經陸續爆料出的各種新活動，比如史詩之路，獸語套禮包，漩渦副本的相關改版等等之類的外，小寶還發現了一個同樣在明天更新開啟，但策劃在今天12.09才剛剛爆料出的新活動。

這個策劃在12.09才剛剛爆料出的新活動，玩家們均可以到官網的版本界面，拉到最下面的這個福利活動中找到，如上圖所示就是這個「勇士集結」活動。但是目前還無法進行參與，玩家需要等到明天更新之後，此活動才會正式開啟，玩家到時候就可以直接到官網上面進行參與。

等到明天12.10更新活動開啟後，玩家們就可以直接到官網上找到此活動，無需上線遊戲等之類的就可以直接領到一個勇士集結禮盒，開啟後可從三種獎勵不同的勇士禮盒，成長勇士禮盒，資深勇士禮盒，精緻勇士禮盒中自選其一。

當然，至於選擇哪一個勇士禮盒最好，這裡小寶就將三種禮盒開啟分別可獲得的獎勵給你們說一下，以此來讓你們自己進行選擇。成長禮盒：5張升級卷、3個雙倍葯，15個大眼仔。資深禮盒：20個反物質或300個蒼穹碎片、15張史詩之路門票。精緻禮盒：一張普通~稀有裝扮兌換卷，外加6個品級調整箱。

相信到時候絕大多數玩家選擇的可能都是第三個精緻禮盒，必定開啟保底可獲得1張普通裝扮兌換卷，用不了拿來分解換成徽章也是相當不錯的，建議等到明天12.10此活動開啟後，玩家們都不要錯過哦，雖然說獎勵並不怎麼樣，但必定都是動動手就可以免費領到，不要白不要。
