尼爾COS現身爐石傳說雙蛋現開賽？這波操作可以啊

天天要聞
2019-12-23

哈嘍小夥伴們好呀，這裡是誠然。這聖誕節也是馬上就要到了，之後就是歡慶的元旦，在這即將告別2019年之際，爐石傳說也在用它的方式辭舊迎新。比如一年一度的艾澤拉斯專屬的節日：冬幕節。爐石傳說會開放限時活動用以歡慶和給玩家們發放福利，而隨著巨龍版本的降臨，今年這個雙蛋（聖誕+元旦）格外有情調~

除了冬幕節合集包的上線之外，爐石傳說巨龍版本的雙人現開賽即將開始，又是爐石主播歡慶一堂為粉絲們「盛裝」演出的日子。每次的雙人現開賽，分組和組內主題都會是玩家們最為期待的，比如在奧丹姆奇兵版本中的雙人現開賽「奪冠就回家結婚」的春哥奶粉組合雷人的新郎新娘主題裝扮就很讓人難忘，畢竟辣眼睛的東西記得還是很深刻的。

在巨龍的雙蛋雙人現開賽開始前（賽程為12月25日到2020年1月1日），官方就開始逐組公布這些組合的主題裝扮。本來以為「山姆」和龍媽的COS組合已經讓人大開眼界了，沒想到狗賊和啦啦啦組合的路飛和山治也是吸引了一大波海賊粉啊。

而官博逐組公布的第一天小丑自殺小隊的COS質量還是非常高的，昨日公布的B組全貌中除了雪諾和龍媽，最讓人驚喜的那當屬2B的身影了——楓雨無卒組合COS的尼爾機械紀元那真的是好看……我說的當然是女COS！眼罩之下紅通小嘴那真的是誘人啊，簡直吸取了2b的精華！只是有個疑問，比賽當日也是這個裝扮嗎？

當然其他組合的也都是好玩的，龜大仙還有紫霞至尊寶雖然看上去有些「低配」，但好歹也是能看得出來的。C組中最有聖誕氣息的那當數濤妹和鬼鬼組合了，冰雪奇緣請收好，感受下冬日裡的冰冷無情……

而對於賽制來說，雙蛋現開賽更加好玩。在巨龍版本上線之前，爐石傳說就迎來了全新的玩法——酒館戰棋。雖然剛上線四十餘天，也已經經歷了三次更新，並且最近一次更新還讓環境發生了不小的變化，相信很多小夥伴現在已經用上惡魔制霸酒館了吧。而這次的現開賽也將會和酒館戰棋有些關聯。

雙人現開賽都是現開卡包，而這次除了標準模式之外，在對抗之前選手們會進行一場酒館戰棋的排位賽，然後根據對局排名來確定開包的數量。這波操作可以說是給足了酒館戰棋粉絲的面子啊。就是不知道選手私下裡有沒有好好下棋做功課啊。

至於這一次誰能把「北海道雙人游」收入囊中，誰又會是「長白山雙人游」的幸運兒，那只有看完直播才能知曉了。小夥伴們記得25日開始觀看精彩比賽啊，好了本期內容就到此結束了，感謝小夥伴們的閱讀，那我們下期內容再見吧~
