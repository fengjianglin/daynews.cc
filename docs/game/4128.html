CFM中越丨亞洲杯小組賽階段戰罷 晉級半決賽隊伍名單揭曉

2019-12-09 0條評論 9次閱讀 0人點贊

摘要

隨著CFM亞洲杯小組賽的結束，AB兩組的出線隊伍也隨之揭曉。A組的eStar與AG晉級，而B組則是R.LGD與Q9晉級，四支隊伍將在接下來的半決賽中兩兩對陣，其中獲勝的隊伍將進入預選賽決賽爭奪最終的出線資格。

隨著CFM亞洲杯小組賽的結束，AB兩組的出線隊伍也隨之揭曉。A組的eStar與AG晉級，而B組則是R.LGD與Q9晉級，四支隊伍將在接下來的半決賽中兩兩對陣，其中獲勝的隊伍將進入預選賽決賽爭奪最終的出線資格。

【eStar如何迎戰強敵Q9】

由於eStar作為A組第一，而Q9作為B組第二，因此在半決賽中，二者將進行對決，爭奪進入決賽的資格。由於eStar與Q9在以往的比賽中有所交手，那麼這一次交手的結果是否會有所改變？

eStar在本次的小組賽中，憑藉著4勝0負的成績成功以小組第一的身份出線，在比賽中叮噹貓三歪以及雷靜等人出色的表現成為eStar未嘗一敗的關鍵，當然成功出線之後，eStar仍然需要提升自身能力。由於在職業聯賽常規賽以及NEST總決賽中，eStar都敗北Q9，因此當他們再次面對Q9時，需要進一步的調整隊伍，從而爭取比賽的勝利。

反觀Q9戰隊，在本次CFM亞洲杯的小組賽中，當他們再次面對R.LGD時卻再次以失敗而告終，從而導致他們未能以全勝成績出線。當Q9在半決賽中遭遇老對手eStar時，若Q9能夠保持穩定的發揮以及結合以往獲勝的經驗，那麼或許Q9便能獲得決賽資格。

【AG能否擊退R.LGD】

AG在本次CFM亞洲杯的比賽中憑藉3勝1負的成績，成功以小組第二出線，這對於AG而言無疑鼓舞了隊伍的士氣。在接下來的半決賽中，他們將面對B組第一R.LGD，那麼AG又能否獲勝？

對於AG而言，在本次小組賽中之所以能夠獲得比賽的勝利，是因為團隊之間的配合以及戰術的調整有進一步的提升，選手的出色臨場發揮為隊伍提供了有力的火力保障。在接下來迎戰R.LGD的比賽中，由於對手強勢的表現，因此AG需要通過更多變得戰術限制對方主力選手，從而掌握比賽得主動權。

反觀R.LGD，由於街寂林准晨銘等人得強勢發揮，令R.LGD在小組賽中佔得一定得優勢，再次擊敗Q9也彰顯了隊伍得強勢，當然在比賽中敗北KZ也需要R.LGD總結自身一些不足。在即將到來的半決賽中，R.LGD需要注重比賽的細節，狙擊手街寂需要為隊伍及時提供信息，保證突破手能夠更好的突破，從而保證R.LGD獲得比賽的勝利。

雖然目前CFM亞洲杯預選賽半決賽還未開啟，但是每支隊伍之間的火藥味已然讓我們感受到這次比賽的緊張，相信即將到來的半決賽中，我們將看到四支隊伍的強勢發揮。
