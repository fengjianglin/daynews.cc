KPL總決賽AG超玩會 VS QGhappy對位分析：打野篇

天天要聞
2019-12-08

摘要

​​2019年KPL秋季賽總決賽即將打響，久違的雙G大戰彷彿將我們的思緒拉回到2017年。但是如今這兩支隊伍無論是選手還是打法都有了很大的改變，處長將根據兩隊的打野、發育路、對抗路、中路、輔助五個位置的選手對位來進行分析，一起來看看究竟誰更勝一籌。

​​2019年KPL秋季賽總決賽即將打響，久違的雙G大戰彷彿將我們的思緒拉回到2017年。但是如今這兩支隊伍無論是選手還是打法都有了很大的改變，處長將根據兩隊的打野、發育路、對抗路、中路、輔助五個位置的選手對位來進行分析，一起來看看究竟誰更勝一籌。

總決賽對位分析–打野篇

藍領野

從季後賽開始，QGhappy的戰術核心更偏向於以Hurt為主C位的射手體系，這就需要一個既能吃土又能打出應有效果的藍領打野。Giao在藍領打野選手中算是較為優秀的一個，他的控龍能力在季後賽多次為隊伍帶來關鍵性的buff收益。

身為一名專職藍領打野，Giao對於節奏的把握遊刃有餘，在關鍵時刻他可以挺身而出，幫助隊伍打開優勢局面。

相比於QGhappy在打野位的固定，AG超玩會則走了一條截然不同的路。

在季後賽的幾場比賽中，AG超玩會將輪換戰術利用到了極致，起初由六點六和麟羽在藍領野位搖擺，但是在常規賽中後期，六點六的戰邊表現過於耀眼，AG超玩會最終將藍領野的重任交給到麟羽和藍柚這個「LY組合」身上。

不可否認，麟羽和藍柚在近幾場比賽中的表現都越來越好，他們逐漸可以承擔起藍領野所應承擔的責任。

藍領野的作用在於讓經濟給C位的同時能夠打出足夠的效果，如劉邦的大招保護、盤古的繳械、豬八戒的開團等等，在團戰中都起到相當重要的作用，對於藍領野來說，參團率的高低能夠在一定程度上反應其在比賽中的支援頻率和帶節奏能力。

Giao身為一名職業藍領打野，他在季後賽所用的藍領野英雄共有八名，其中蘭陵王和楊戩勝率為0，其餘英雄勝率皆大於等於50%。同時，他的參團率較為可觀，平均參團率在67%左右。

藍柚在季後賽共使用過四名藍領野英雄，其中在使用到盤古、趙雲、孫策這三名英雄時勝率較高，但他在使用藍領野時的平均參團率只有57%左右。

麟羽在季後賽共使用過三名藍領野英雄，平均參團率高達71%左右，這一點足以看出麟羽的參團程度。但是在他所使用的三名藍領野英雄里，其中盤古和孫策都與藍柚的英雄池重合。在全局BP的模式下，除非打到巔峰對決，否則每個英雄只有一次的使用機會，所以英雄分配問題是AG超玩會需要考慮的重點。

從數據來看，QGhappy的Giao在英雄池方面更佔優勢，8個藍領英雄即使打到巔峰對決也綽綽有餘。而AG超玩會的藍柚和麟羽的藍領野英雄池加起來一共只有5個，這是他們在藍領野英雄池方面的劣勢。但是，AG超玩會的戰術並不只局限於藍領野，他們在戰術方面較為多樣化，野核體系也是他們慣用的戰術。

野核

在季後賽，AG超玩會並不只堅持藍領野+射手的體系，他們經常會使用野核體系。一諾和藍柚在野核位置的搖擺，讓AG超玩會在戰術上有了更多的選擇。

一諾極高的上限，讓他在使用野核英雄時打可以打出極強的入侵性，同樣以打野出身的藍柚，在使用野核英雄也能夠打的更加得心應手。

QGhappy在常規賽中也曾讓1dao這位優秀的野核選手上場，但是在季後賽，QGhappy似乎已經明確了自己的戰術體系，他們大概率不會像AG超玩會這樣把輪換當做戰術。從野核角度來看，AG超玩會的藍柚和一諾更佔優勢。

根據綜合分析，處長認為在藍領野上，QGhappy略強於AG超玩會，Giao的控龍能力和英雄池都更佔優勢。

在野核上，AG超玩會因為獨特的「LY」輪換體系而略強於QGhappy，畢竟在季後賽還敢將輪換作為戰術的，恐怕只有AG超玩會了。

本期打野篇的對位分析就到這裡，兩支隊伍的發育路對位究竟孰強孰弱？敬請期待總決賽對位分析–發育路篇：Hurt VS 一諾。
