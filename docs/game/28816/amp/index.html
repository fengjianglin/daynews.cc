魔獸世界懷舊服：制衡黑手，改變命運的物品大集結！

天天要聞
2019-12-21

​

黑手，是魔獸世界中最強大的存在，直接影響著每個團隊甚至伺服器Raid水平的高低。

一名黑手可以一個CD之內讓一個職業的玩家畢業，例如一趟MC開出10件塞納里奧，不僅直接插畢業一個小德，還能剩下一些拿去賣錢支援團隊發展，可謂為所欲為。

好在為了制衡黑手的影響，魔獸世界的設計師們當年早有準備，設計了不少逆轉黑手能量、足以「改變命運」的物品，今天就由小O來給大家詳細介紹一下。（千萬要看到最後哦！）

歐皇圖鎮樓

幸運兔腳

幸運兔腳可以說是魔獸世界裡面最強大的灰色物品，從最低級的野獸身上就會掉落。無論是剛開始練級或是早已滿級的玩家，包包里都會常備一兩組幸運兔腳，就是因為其擁有的強大的「逆天改命」作用。

經過廣大玩家的實踐與開發，幸運兔腳目前的詳細作用與使用方法如下：

  • 放包里會出現任務物品掉率加50%的幻覺
  • 放在包里，一個可以提神醒腦，一組可以提高暴擊率1%
  • 放在包里最右下角，能增加MC摸到右臉的概率，同理想摸左臉就放左下角
  • 帶在身上打怪降低未命中和閃避，下副本要裝備增加掉率
  • 沒事備上一組，出紫裝幾率賊大
  • 如果你感覺自己一直臉黑的話，建議弄一個放在背包里，據說有奇效
  • 更多未被證實的用法尚在開發中

好運符

顧名思義，這是一個可以增加好運的符。身為中國人，想必大家都知道「符」是什麼東西，這也是一件可以改變命運的物品。

好運符相對於幸運兔腳來說，就沒有那麼容易獲取了，需要盜賊玩家對部分獸人、地精、食人魔或者恐怖圖騰的牛頭人使用「偷竊」（搜索）技能才能得到。所以與幸運兔腳的全民擁有不同，普通大眾對好運符知之甚少。

好運符基本上擁有幸運兔腳的所有關鍵作用，甚至效果更強，搭配幸運兔腳一起使用的話，將會獲得最佳效果！

《好運符的超級增強版——費爾斯通好運符》

綠色品質，一看就知道比普通好運符更加靠譜，這件超級好運符來自於西瘟疫之地的費爾斯通農場的一個長任務，起始任務名字是「遲到總比不到好」，任務NPC是詹妮絲·費爾斯通。

你需要要做完一系列奔波勞累的任務，之後接到最終任務「合二為一」，目標是殺死農場里的「喋喋不休的食屍鬼」（實際上就是任務給予人的丈夫），之後將兩半好運符合二為一。

然後從現在開始，你就擁有了一個完整的增強版好運符了，好運將會常伴你左右。

嗜血套裝

嗜血套裝是懷舊服世界掉落的一套綠色鎖甲套裝，不要看它的名字顯示是綠色，其實它實際上卻是紅色的！

只要我們副手拿上[紅玫瑰]，再搭配上[紅色冒險者襯衣]，全身可謂是紅的發紫。穿上這樣一身紅色去摸裝備，難道還能黑嗎？

這是黑手們不可或缺的一套裝備，最大的遺憾是只有鎖甲和板甲職業才可以裝備。

幸運長褲

這不得不說是聯盟玩家的巨大優勢，知道為什麼60年代聯盟玩家比部落玩家副本進度一直要快嗎？

沒錯！很大的原因就和這件[幸運長褲]有關，因為它是聯盟專屬任務的獎勵，部落玩家完全拿不到！

所以由此就改變了兩個陣營之間的運氣平衡，使得穿上這條褲子的聯盟玩家在物品掉落、DPS、治療還有MT的閃避幾率上都有所提升，從而降低了聯盟團隊的raid難度。

看到這裡，可能很多小夥伴都在擔心會不會在練級途中將這件超級裝備賣了。其實不存在的，因為這個任務的NPC巡山人沃爾班位於洛克莫丹的左下角：

即使是侏儒/矮人玩家都不一定會做到這個任務，更別說其他種族了。任務相當簡單，只需要殺死一些小石齶怪之後再做一個後續（殺三個有名字的石齶怪）就可以了。

但是最後在選擇任務獎勵的時候可不要手抖選到其他裝備哦，因為另外兩件都是沒有價值的！

煤塊

說到黑手，就不得不說煤塊。眾所周知，在我們進行MC團本活動的時候，偶爾都會撿到煤塊。煤塊的特點之一就是黑，比黑手更黑。

最近流行的一種說法是，在MC裡面撿到煤塊的人，千萬不要摸boss的屍體，並且在別人摸裝備的時候，最好距離boss40碼以上，這樣才不會因為煤塊影響到boss的掉落。

而更好的做法，則是直接爐石回城，將煤塊賣給商人之後再飛回副本。

黑手

終於說到了我們壓軸的終極武器——黑手。

如果你是一個萬（shi）人（e）擁（bu）戴（she）的團（hei）長（shou），黑龍只有怒風復仇無盡風暴+禁錮之戒/艾斯卡達爾，MC只會出塞納里奧和惡魔之心，老八永遠是爆擊獵槍，大螺絲長年累月的純焰精華/馬桶盾……那麼你就必須擁有這件終極殺器。

傳說中擁有最強的逆天改命效果的單手武器[黑手]，需要由瑟銀兄弟會崇拜的鐵匠才能製造出來，而且所需要的材料不少：熔岩之核*4、熾熱之核*7、奧金錠*12、黑鐵錠*8、艾澤拉斯鑽石*4。

當然，為了一件這麼強大的神器付出小額的金幣也是非常值得的。一旦擁有了這件黑手之後，你只需要一個步驟，就是在開裝備之前，站在boss的前面，使用「分解」技能將其分解掉便可大功告成！從此以後遠離黑手！

由於其效果太過霸道，所以暴雪考慮到平衡性，該武器當前階段尚未開放，所以要改變命運的黑手們，請期待下一階段的來臨吧！
