FPS VR遊戲《Boneworks》登錄Steam，首發同時在線高達8717人

2019-12-11 167條評論 1,595次閱讀 174人點贊

（映維網 2019年12月11日）對於本周Steam新上架的內容，最受期待的作品之一當屬Stress Level Zero的《BONEWORKS》。這款包含大量戰鬥元素的「實驗性物理VR冒險遊戲」作品採用了基於物理的武器，工具和對象，並且支持超級逼真的IK反向運動學全身系統，這一切都是為了呈現最為逼真地槍戰體驗。顯然，你將會在各種不同的地圖和建築中進行刺激的生死作戰。

對於憑多人射擊遊戲《Hover Junkers》和恐怖遊戲《Duck Season》等內容打響自己名堂的洛杉磯開發商Stress Level Zero，他們自2016年以來就一直在為社區帶來VR精品。這支團隊於2018年在Node YouTube頻道發布了一個關於基物（基於物理）射擊遊戲的視頻預告片，並再次在網上和VR社區掀起了一陣熱潮。在2018年的GDC大會上，他們通過一個簡短的演示內容進一步點燃了大家對這款遊戲的好奇心。

現在，遊戲已於12月11日正式登陸Steam，售價人民幣90元，包含37元的DLC。值得一提的是，《BONEWORKS》一上線就吸引了大量的玩家。截至發稿前為止，686篇用戶評測給出了「特別好評」的評價，而且Steam平台的同時在線數量達到5790（最高峰值是8717）。

對於《Boneworks》，Stress Level Zero使用的目標是將其打造成一款基於物理規則的作品，從而允許玩家能夠以任何符合遊戲世界規則的方式來自由處理謎題和戰鬥。

這非常特別，因為其他大多數VR交互設計都是採用「腳本」模式，這有時意味著玩家看似合理的舉動卻沒有任何效果，因為開發者從未編程相關的交互。例如，玩家希望用盾牌敲擊敵人頭部，但盾牌只能前後左右地移動防禦，無法用作武器進行攻擊。

與之對比，《Boneworks》納入了迄今為止我們在現代VR遊戲中見過的一些最為有趣的基物玩法。利用Valve尚未正式發布的Knuckle控制器（兼容Oculus Touch），這款體驗為用戶帶來了前所未有的交互水平。例如，支持壓力感應的抓握允許玩家伸出雙手並實際抓取和擠壓對象。儘管這可能聽起來很簡單，但能夠用手向上和向下靈活地滑動撬棍而非單純地咬合至指定位置，這可以大大增強玩家的沉浸感。玩家可以利用撬棍攀爬，打開門壁並攻擊敵人。

遊戲令人同樣印象深刻的是，其支持超級逼真的IK反向運動學全身系統，允許你非常準確地移動和彎腰下蹲。根據視頻，每一個肢體幾乎可以完全相等地映射真實動作，並且在「霸王龍」風格的短手基礎上增加了「火烈鳥」的大長腿。

工作室希望，這種自由交互可以擴展至遊戲的解謎元素，從而允許玩家尋找創意解決方案。

官方Steam頁面寫道：「《Boneworks》是一款敘事VR動作冒險遊戲，採用了先進的實驗物理力機制。你可以動態地導航環境，進行基於物理的激烈戰鬥，並創造性地通過物理解決難題。」

遊戲已經正式登陸Steam，售價人民幣90元，支持Valve Index，Oculus Rift，HTC Vive和WMR。
