最早的爆款遊戲《俄羅斯方塊》！銷量超過五億份幾乎無人能敵

2019-12-20 81條評論 4,334次閱讀 759人點贊

作為 有史以來最暢銷的遊戲，俄羅斯方塊自然有著超級多的故事，如果你們喜歡聽，那恰巧筆者也願意聊聊，希望你們能喜歡。

其實俄羅斯方塊也並不算年代久遠，她是在1984年6月發布的，製作者是前蘇聯科學家阿列克謝·帕基特諾夫，據說這個程序僅僅是他利用自己的業餘時間所編寫的，畢竟當時帕基特諾夫還在蘇聯科學院電算中心供職。

至於名字，Tetris是由希臘語的4和tennis（網球）組合而成，不知道大家有沒有發現，在標準版的俄羅斯方塊遊戲中，雖然形狀各不相同，但每個落下的形狀都是由四個方塊組成。

那麼，我們在下文中，就好好聊聊這個賣出了超過五億份的超級遊戲吧。

我們知道，在俄羅斯方塊的標準版本中，基礎形狀一共有七種，當然對稱的形狀分開計算。遊戲開始後，隨機的形狀塊就從遊戲區域上方開始緩緩下降。不過在下落的時候，玩家可以旋轉（90度）這些方塊，也可以調整所在的位置，當然亦可加快下降速度。

最終，當這個方塊落到無法在下落的位置，便由一個隨機的新的形狀塊出現在屏幕的上方，一直重複這個過程。當一橫行都被填滿時，這一行就會被消除，並且給予玩家分數，同時消除的行數越多，分數也越高。

如果你的方塊在區域頂端出現時，已經無法繼續下落，遊戲便宣告結束。大多數的版本里，其實還會有一個窗口，顯示下一個形狀，以便玩家進行規劃。當然，遊戲在進行當中，下落的速度也會逐漸增加，以此增加難度。

記得在小時候，筆者記下的技巧，最厲害的一條就是利用好豎條，但也經常因此被屈死。

就是這麼一個《俄羅斯方塊》，其實也是同樣有知識產權的，而我們小時候的那些小型掌機，其實都是沒有授權的盜版貨，所以說銷售量伍億，還真的挺委屈了這個遊戲的。

最開始的時候，《俄羅斯方塊》的版權是歸在了前蘇聯的外國貿易協會，後來才被俄羅斯方塊公司（The Tetris Company）繼承。雖然有不少的作品都是正版授權的，但也有很多公司製作類似的產品，這也包含一些衍生作品。

不過這類性質的遊戲，通過計算擺放，只要沒用Tetris，就算出了遊戲，也很難被判為抄襲，這其實也算是對製作方多多少少的一個不公平吧，所以這也造成了之後很多開發商對這類計算性質的遊戲都不感冒，畢竟很難擁有版權上的保護。

不過即便這樣，也無礙於《俄羅斯方塊》的成功，這款遊戲已經在超過了50種平台上出現，對了，最近的一款，應該就是Switch上的《Tetris99》了吧。這款遊戲把《俄羅斯方塊》與吃雞玩法相結合，也算是讓《俄羅斯方塊》又火了一把。

要說起來，任天堂平台的《俄羅斯方塊》可是有版權的。

這段故事也是挺有趣的，最初阿列克謝·帕基特諾夫將這款遊戲的版權交給了蘇聯科學院電算中心，這個版權前文也提到過了，算是公有財產，最終歸到了外國貿易協會。

羅傑斯在一個偶然的機會發現了這個遊戲，雖然他出生在荷蘭，但當時卻在日本從事遊戲方面的工作，他馬上聯繫了任天堂。任天堂聽完他的表述後，乾脆派他為代表，去了蘇聯，與政府的代表進行了協商。

結果大家也都知道了，1989年的時候，任天堂獲得了《俄羅斯方塊》在家用機上發布的授權。不過在這次談判中，羅傑斯卻和帕基特諾夫成為了好朋友，隨後他甚至還幫助帕基特諾夫移居美國。在1996年，帕基特諾夫拿回了《俄羅斯方塊》的版權，當然這裡面也同樣有羅傑斯的參與。

之後，他們便專門負責《俄羅斯方塊》的授權事宜，在2009年的時候，《俄羅斯方塊》已經傳播了50多個國家，玩家們都為這個小小的方塊而著魔，據說還有玩家在飛機上用手機玩《俄羅斯方塊》，被判了四個月的刑，當然理由只是忍不住想要玩。厲害吧，即便是在當時，《俄羅斯方塊》也已經賣了1.25億份，據說在北美和歐洲的網路版本上，每天都會有超過100萬人在玩。

其實關於《俄羅斯方塊》的版權之爭也並非那麼簡單，PC版的遊戲便引發起了不小的風波，據說Andromeda的遊戲公司經理羅伯特·斯坦恩在當時並沒有取得的帕基特諾夫授權，之後就以盜版的形式發售，結果是其實他並沒有因此發橫財就是了。

雖然《俄羅斯方塊》如此偉大，但創造者帕基特諾夫卻並沒有因為這個遊戲而一躍成為富翁，即便有機會得到很多，他還是選擇了放棄。

因為他愛遊戲，他為自己能做出這款世界聞名的遊戲而驕傲！

或許蘇聯科學院電算中心獎勵了新電腦給他，就足夠讓他開心的了，這或許就是一個真正的遊戲人該有的樣子吧。

也許我們現在可能不會天天捧著《俄羅斯方塊》這款遊戲了，但她也的確非常經典，這就足夠了，對嗎？親愛的玩家們。
