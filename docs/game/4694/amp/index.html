新手游戰雙帕彌什開服重大事故 9.5分暴跌至5.6 同行趁機落井下石

天天要聞
2019-12-05

摘要

新手游戰雙帕彌什，算是近期期待度不錯的一款新手游，因為整體風格和崩3類似，所以你懂得，在眾多玩家盼到好不容易開服的那一時刻，誰知道這遊戲開服就遭遇了重大事故，簡單來說就是運營錯發1W黑卡，且評分從9.5分暴跌至5.6分，甚至還有同行趁機落井下石，來看看是怎麼回事吧。

新手游戰雙帕彌什，算是近期期待度不錯的一款新手游，因為整體風格和崩3類似，所以你懂得，在眾多玩家盼到好不容易開服的那一時刻，誰知道這遊戲開服就遭遇了重大事故，簡單來說就是運營錯發1W黑卡，且評分從9.5分暴跌至5.6分，甚至還有同行趁機落井下石，來看看是怎麼回事吧。

不說遊戲自身素質和手感如何，說實話這種風格的遊戲個人還是蠻喜歡的，也第一時間等著開服上限體驗了一下，結果說實話體驗並不是很高，遊戲自身素質沒啥問題，但問題在於遊戲自身開荒資源過於摳門，百萬預約配合第一章通關資源，加起來連遊戲中一次十連都湊不出來，這麼摳門的手游，如今是打著燈籠都找不到了。

其抽卡成本是首次十連1500黑卡，之後變成2500，也就是說一單648頂多整兩次半十連的，這氪金成本摸著良心說和崩3差不多，比fgo坑多了。

結果開服的時候因為系統問題這個首次十連的1500黑卡沒改動，官方為此發發弄了1000黑卡作為補償，按理說這種操作是合理的，但萬萬沒想到負責發放郵件的運營多按了一個0，結果這一下就變成了1W黑卡，這下直接造成了重大開服運營事故。

對此官方緊急進行了關服，但誰都沒想到期間又出現了問題，部分玩家趁機會登錄了進去又一次拿到了1W黑卡的郵件，這一下就算是拿到了2W黑卡。對此在後續的開服中官方並沒有回檔操作，去回收那些因為失誤操作發放且領取的黑卡資源，導致趁此機會佔到便宜的玩家開始炫耀，節奏瞬間被帶了起來。

連續三次炸服和多次睿智操作讓這部手游的評分直接從9.5分暴跌至5.6，然而官方目前的處理還是非常的迷幻，讓眾多玩家抱怨。這種看起來很簡單直接將錯誤當福利發放的情況不少，遊戲本身出貨率就比較坑爹，就算給了1W黑卡相當於給4次十連，說不定也不會出貨，但官方最後還是在四次十連和遊戲可能暴死涼涼之間，選擇了後者，這就是讓人很難受了。當然這裡並不是去黑遊戲，只能說運營方面確實操作有重大失誤，說實話遊戲自身素質確實不錯，是很有潛力的一款，就算成不了崩3，達到當年一開始七都的水準也是沒啥壓力的。

最有意思的是可能是業界手游似乎一致商量好的節奏，在同一天一起發布新遊戲的活動和公告，這操作讓很多玩家瞬間笑瘋了，崩3放出後崩壞書大招，粥游第六章主線預熱和新五星走起，fgo當天晚上開1500DL活動四星從者54選1，閃耀暖暖馬上過生日，碧藍聯動火力全開外帶預告厭戰改造復刻，唯獨戰雙原地爆炸。這種同行趁機落井下是的操作也是沒誰了。真是全靠同行襯托。
