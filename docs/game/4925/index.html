DNF：體驗服12.3拆包，新增泰伯爾斯史詩之路！翻版大富翁來襲

2019-12-04 0條評論 16次閱讀 0人點贊

摘要

早在十一月中旬，策劃就一直暗示一個消息：12月10日會更新新版本！提前一個月就公布消息，而且12月10日是周二，可想而知這次更新不會太普通。

早在十一月中旬，策劃就一直暗示一個消息：12月10日會更新新版本！提前一個月就公布消息，而且12月10日是周二，可想而知這次更新不會太普通。（DNF的重大更新都是在周二）轉眼間已經到了12月，距離更新新版本的日子越來越近了，等了好久終於等到今天，今晚體驗服停機維護，共爆料出四大活動，有新穎的史詩之路，也有當年的熱門活動大富翁，就連哈林商人也加入其中！

體驗服12.3拆包，新增泰伯爾斯史詩之路！新版大富翁來襲

不管是在哪個版本，只要史詩之路一上線，過不了多久國服就會更新新版本，95版本也是如此！12月10日更新後，國服會上線一個叫做「史詩之路：星河裂縫」的活動。由於體驗服還未開服，具體獎勵為止，不過看樣子超界史詩肯定是有的，累積獎勵應該也少不了，如果能送幾個黑瞳石就好了。

一個全新的版本，光有一個史詩之路怎麼能行？大家可別忘了，前陣子史派克進行過一個趣味調研：「你認為DNF中的哪個活動最有趣？」在那次調研中，大富翁這個良心活動脫穎而出，這不，它馬上要回歸了！

翻版大富翁，擁有專屬活動頻道，玩法跟之前的大富翁活動一樣，獎勵得到了提升。在遊戲內玩家可以獲得大量的時空石和守護珠，而且還有累積獎勵，龍之硬幣你聽過嗎？

12.10版本有這兩個熱門活動，玩家又能開心一段時間了！大家在刷史詩之路的閑暇之際，可以去找哈林商人「賺」點錢，12月10日更新後哈林商人那裡有個新商機，具體內容為止，大萌猜測一定是跟回收遊戲幣有關的內容，有可能會是一次性增幅器，一次性強化器類似的道具。

最後一項活動：冬眠充能！十分應景的一個活動，活動插畫中只有一隻小熊在屋裡睡覺。看樣子該活動不會有太好的獎勵，像是常規簽到活動。因為體驗服還未開服，四大活動的具體內容未知，單純的看活動封面不難看出，大部分活動還是很給力的，你希望泰伯爾斯史詩之路加入黑瞳石獎勵嗎？歡迎評論留言哦。
