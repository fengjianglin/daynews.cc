今天小獃獃和大家聊聊關於王者榮耀裝備的那些事，目前大件小件的一共有98件裝備，其中有四件裝備最失敗，但是有些時候他們可以發揮出不一樣的效果！可能這些裝備只是給某個英雄量身定做的，今天我們就來看看這四件裝備到底失敗在哪裡！

防禦加攻擊——破魔刀

首先我們來看看「破魔刀」，這是物理輸出英雄的裝備，因為被動需要攻擊力才有加成，一般只有戰士出會出這件裝備，刺客並不適合，100點的攻擊力實在有點少，而射手更加不可能使用這件裝備。

如果對面法師傷害過高，在絕大多數的情況下都是選擇「魔女斗篷」，這就導致了這件裝備最失敗的地方就是被「魔女斗篷」取代了，真是百場難見一次！

法師保命裝——熾熱支配者

「熾熱支配者」看上去是法師的保命裝，大量的法力值加成，最大的亮點就是被動，這裡有很多讓不知道的冷知識，大多數玩家只知道「熾熱支配者」的被動只是增加一個護盾而已，其實在觸發被動的時候還可以解控並且提升自身30%的移動速度，這一點被很多人忽視！

看似很強，但是在嫦娥沒有上線之前，沒有那個法師會去使用這件裝備，而它的出現貌似就是等著嫦娥的到來！如今嫦娥出現了卻是一位冷門英雄！

輔助裝備——星泉

“星泉”剛上線不久的裝備，唯一一件可以為隊友抗傷害的裝備，上線至今千場難見一次，比起其他輔助裝，”星泉”真的太年輕，也被網友聲明：2019年最沒用的一件裝備！

如果瑤在使用大招附體的時候開啟”星泉”，隊友受到傷害的時候瑤並不會掉血，但是成功的為隊友抵擋了一半的傷害，也許這就是為瑤量身定做的裝備！

防禦裝備——衝擊鎧甲

「衝擊鎧甲」和”星泉”一起上線，也是唯一一件擁有主動技能的防禦裝備，但是上線這麼久根本沒多少人使用，大多數玩家還是喜歡以前的出裝思路。

而最近「衝擊鎧甲」出現在了馬可波羅的身上，很多玩家認為馬可的神裝，但是「純凈蒼穹」怎麼辦？使用這件裝備真的適合馬可？
