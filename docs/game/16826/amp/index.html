夢幻西遊手游：30萬金幣打出價值千萬武器？被網友評論搞崩心態

天天要聞
2 天 ago

在《夢幻西遊》手游里，強化打造裝備是一項非常刺激的玩法，尤其是強化打造100級裝備的時候，更是心驚膽戰（幾十萬金幣下去要是屬性不好那就打水漂了）。打造100級裝備除了需要對應的製造書與百鍊精鐵外，還需要對應的裝備元身。比如你想打造一把100級槍，那麼就需要用到100級槍·元身。元身可以通過消耗隕鐵重鑄來改變元身的屬性，不過重鑄元身有概率失敗，失敗會使元身消失，玩家會隨機獲得2-3個隕鐵。

需要注意的是，元身重鑄只改變元身的基礎屬性，不會改變元身額外的附加屬性。建議各位少俠在選擇元身時，盡量選擇帶雙加和藍字的元身來打造。

最近一位115級龍宮玩家就打造了一把100級槍，我們看到100級槍·元身的價格其實有點貴，好一點的槍·元身價格差不多就是20-40萬金幣這個區間。他為了降低打造成本，最終選擇了一件20萬金幣的槍·元身。槍·元身的屬性先暫時不給大家看了，畢竟最終打造成型的裝備屬性與槍·元身的屬性有差異。

100級製造書·槍的價格差不多8萬金幣左右，喜歡跑環的玩家在運氣好的時候，會有機會得到100級製造書·槍。

100級百鍊精鐵差不多就是白菜價了，2萬金幣一個。對於打造者來說，大概是除了打造符外最便宜的材料了吧。打造的材料準備好後，他果斷按下了強化打造鍵，一把新的100級槍誕生了。

這把槍是100級慈航普度槍，帶有34點單魔力加成，基礎法傷184，還差3點滿法傷。僅僅花費30萬金幣就打造出了一把不錯的100級慈航普度槍，這發打造怎麼看都是賺，只是賺多賺少的問題。但他不是很懂100級慈航普度槍的物價，於是發帖詢問槍的價格。

有網友稱為100萬金幣頂天，也有網友說200萬金幣頂天，還有人說窮人買不起，富人看不上系列。100級慈航普度槍有些輔助會選擇用的，不過他打造出的這把槍治療強度不是很高，並且是單加屬性，對武器追求高的輔助不會選擇用這種武器。況且，慈航普度目前PK的使用頻率非常低，即便是輔助也很少使用，更別說龍宮了。因此，如果龍宮用這把槍的話，慈航普度特技幾乎用不上。

他原本以為自己打造出的這把槍可以賣1000萬金幣，但從各位網友的評論來看，這把槍好像只能賣100萬金幣。如果他是跨服擺攤出售，最終到手只有82萬金幣，和自己預期的售價相差很多。看到眾網友的回復後，他似乎有些失落，更扎心的是有直男網友稱這把武器神威三線龍宮應該看得上。既然賣不了多少金幣，不如留給自己用，好歹這把武器的法傷不低。各位少俠，如果你打出一件這樣的武器，你會選擇留著還是擺攤出售呢？
