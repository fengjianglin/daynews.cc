德瑪西亞杯D組，IG恐遺憾淘汰

天天要聞
2019-12-21

大家好，我又來啦！今天咱們說完最後的D組。

縱觀本次德瑪西亞杯分組，兩個組神仙打架，兩個組臭魚爛蝦，神仙組是BC兩組，而臭魚爛蝦組則是AD兩組了。

有人會問，A組不是有FPX嗎？世界冠軍隊伍。但隨著FPX發布本次德瑪西亞杯的出戰陣容，本次FPX派出的人員全是新人，也就是說，這次的FPX與同為LDL的LGE無二，喪失奪冠的可能。而另一個弟弟隊伍就是WE，之前也說過WE目前就是練習生訓練班。一個小組3個弟弟，算魚塘了吧？

D組毫不遜色。D組5支戰隊分別是IG、DMO、BLG、RW、SDX。查德一看，這個小組就不強。首先是IG，IG日前也發布了本次德瑪西亞杯的參賽陣容，上中野均為新人，The shy，Ning，Rookie都不在出戰名單中，而下路的阿水遲遲去向未定，寶藍本次也沒有出場。單靠胖夫+南風並不足以支撐起這支戰隊，大家可以想像一下之前的VG，聯賽墊底的存在。

而RW，在初代俠客分道揚鑣之後，這支戰隊就持續低迷，縱使迎來張教主，搭配初代的奇犽，也無濟於事。本次轉會期，奇犽離隊，雖然同時也引入相當一部分的人員，但大部分還是青訓中提拔上來的，整體實力相比LPL的隊伍，還是稍遜一籌。可能比SDX強一點吧，大家可能對SDX不太熟悉，我也不熟悉。但SDX里有一個選手大家可能會認識，他們的下路ADC，就是此前OMG的ADC，小北。

這麼數下來，D組的弟弟隊伍也高達了3支。於是出線的局勢又很明朗了。BLG與DMO。那為什麼這兩支戰隊不算弟弟隊伍呢？

首先是DMO，DMO本次的陣容與此前的DMO差不多，只是補充了一個中單Xiye。但也正因為補充到了Xiye，讓這支戰隊看起來，至少不會那麼弱了。上單的Natual，一直以操作著稱，而打野的xiaopeng，也是實力很是不錯。加之下路的Gala與Mark也是很長時間的隊友。DMO這支戰隊雖然整體實力還不夠頂尖，但勝在磨合，大膽預測DMO春季賽在中游的位置，不會太后面，而幸運的是本次的德瑪西亞杯，DMO應該可以晉級8強。

BLG呢？BLG引入中單FOFO，打野國豪兩大超強戰力。加上ADC 金角，輔助Xinmo，上單的ADD。BLG一直都容易被人忽視。大家都知道BLG的實力一直不弱，至少都是上游的。但每次問起LPL實力比較強的戰隊時，總會遺漏掉BLG。OK，那這次德瑪西亞杯，是一次讓大家記住這支戰隊的機會了。在這樣的一個分組中，BLG就像鶴立雞群一般，周邊的隊伍沒有一支能對他威脅，想輸，不要太難。Flag就立了，怎麼的吧~

本次D組，我無腦站BLG與DMO。不要問，問就是沒腦子。

下一篇，咱們來分析一下A組的戰況，換個角度去看比賽，不舔不噴的說，哪裡做的不夠，哪裡是需要加強的。

臨走前，點個關注吧~這個真的很重要，跪謝~
