電競與飯圈的交鋒？微博年度投票UZI排第二！力壓肖戰，朱一龍等明星

2019-12-12 937條評論 2,843次閱讀 207人點贊

2019年已經接近尾聲，在這一年中電子競技也誕生了無數個精彩而難忘的時刻。S9奪冠更是成為狂歡。而電競選手們在賽場上刺激對決，為觀眾們呈現了一場又一場的視覺盛宴，讓人們留下了許多令人難忘的瞬間。在這2019年與2020年的交替之際，微博也發起了微博之夜的年度影響人物評選的活動，這次活動中著名電競選手Uzi在榜單中排名第二，緊追第一!

微博作為當下用戶極多的社交輿論平台，微博之夜年度人物評選的活動也是一個很有含金量的人物評選活動，無論是什麼圈子的角色，只要你的影響力夠高就可以參與活動的評選。按照往年的情況來看，霸榜投票榜單的人物都是娛樂圈的流量小生，但今年的這次榜單評選中出現了許多位電競圈的職業選手;其中，Uzi還拿到了第二的名次。

目前來看，位列榜單第一的是王一博，Uzi緊隨其後排名第二，在Uzi後面還有肖戰、朱一龍、迪麗熱巴等人。作為LPL的傳奇選手，UZI這個ID至少在LPL圈子裡，算得上人盡皆知。而隨著電子競技行業的蓬勃發展和年輕群體的湧入，電競選手在社交平台上的討論率也逐漸上升。每逢熱門電競賽事，有關電競選手的微博熱搜屢見不鮮甚至登頂。足以見得電競在年輕群體中的影響力日益提升。而作為電競圈的知名選手，UZI擁有著極其龐大的粉絲群體。故而此次投票力壓一眾流量小生也不足為奇。

除了Uzi之外，電競選手Theshy目前在榜單中以796392票排名第七，將505150票的第八名華晨宇甩到身後。不得不說電競圈的影響魅力越來越大，也越來越受到了人們的關注。除了Uzi和Theshy兩人之外，Jacklove、Clearlove、Rookie、Doinb等電競選手也入選到了該榜單之中，此次微博年度人物儼然成了娛樂圈和電競圈的投票之爭。

雖然這份名單的投票，更多的是來自於粉絲們的努力。但眾多電競選手出現在榜單之中，人氣與流量小生不分伯仲。也昭示了電子競技正在逐漸成為年輕人的文化現象。活躍在賽場上的年輕選手也開始被大眾認可接受並喜歡。我們也希望電子競技的社會影響力越來越大，成為全民接受認可的體育文化。
