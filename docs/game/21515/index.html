陰陽師無一速玩家鬥技沖分陣容推薦，不怕雙化鯨，式神獲取難度低

2019-12-19 1,302條評論 3,554次閱讀 663人點贊

這幾天談論京都決戰的相對比較多，估計各位陰陽師大人們也研究得差不多了，今天我們就來聊聊陰陽師鬥技玩法。雖然速度是陰陽師手游中比較重要的屬性，但頂級一速御魂可遇不可求，我相信還是有很多玩家沒有一速御魂的。那麼我們在沒有一速的情況下該怎麼打鬥技呢？難道沒有一速就沒法打鬥技了嗎？我認為是可以的，只要陣容搭配好，沒有一速我們也同樣可以打鬥技，今天就來給各位推薦一套陣容。

這套陣容的核心就是白藏主、匣中少女和櫻花妖，其餘兩個式神可以變動，一般情況下選擇熏和御饌津。先來說說這套陣容的御魂搭配，白藏主使用速生生的地藏像御魂，這樣能抵擋更多傷害。四號位不用效果抵抗是因為沒必要，我覺得效果抵抗和效果命中這兩個屬性在陰陽師里始終是個謎。經常會出現自己130命中的睡豬控不住對面沒抵抗的白藏主，反而對面沒命中的鐮鼬上來就把我放150抵抗的騎驢敲暈了，反正沒人看得懂這概率，還不如堆點生命來得實在。

既然沒有一速，不能拿到先手的我們可以在匣中少女身上帶一套蚌精，御魂選擇生生暴就可以了。這裡我覺得有必要和新手玩家解釋一下為什麼要帶暴擊，在陰陽師中不單只有輸出傷害可以暴擊，治療量和護盾量也是可以暴擊的，所以我們的匣中少女在堆滿暴擊的情況下儘可能地將暴擊加成往上堆。當然，你也可以選擇速生爆，堆一些速度屬性也是不錯的，堆了速度的她可以讓隊友大部分時間被回溯保護。每個人的玩法不一樣，這個要看自己選擇，我覺得生命會更適合現在的鬥技環境。

櫻花妖的御魂選擇速生暴珍珠，在暴擊屬性堆滿的情況下儘可能提高暴擊傷害，速度雖然不用刻意去堆，但是快一點也不會有錯。目的就是在治療的同時為隊友添加護盾，櫻花妖配合匣中少女可以為隊友提供高額護盾，在面對雙化鯨不知火陣容絲毫不懼。

至於御饌津我是選擇攻攻暴木魅，能夠限制部分陣容鬼火使用的同時利用狐狩界反擊打出可觀的傷害，在必要時還可以使用破魔劍讓對方減員。如果各位玩家實在沒有御饌津，那麼將御饌津的位置替換成久次良也是不錯的，久次良可以在保證輸出的前提下增加隊伍暴擊傷害屬性。即便隊伍中沒有不知火，久次良也同樣可以依靠協戰頻繁攻擊。

最後就是熏了，熏的御魂我選擇速抗生招財，這個既可以增加傷害還可以供火解控的式神又有誰會不愛她呢？配合雖說白藏主和御饌津都是用火大戶，不過陣容中有了熏的供火基本上不會出現斷火嚴重的情況。有了熏的解控，沒有抵抗的其他隊友也不需要太擔心被敵方控制，只要不是遇到大岳丸這種強制孤立的式神基本上沒有太大問題，即便是遇上了也有一戰之力。

推薦給各位是因為我覺得這套陣容的式神獲取難度相對較低，除了白藏主需要花點心思之外其他式神都很容易獲取，實在不行就去百鬼夜行砸禿頭嘛。對於御魂的強度也不會特別高，反正不會像一速陣容那樣追求極致的單一屬性，只要不是太雞肋的六星御魂強化滿級就沒問題了。可能老區玩家們用這套陣容只能打到6段左右，新區玩家快速成型拿這套陣容打八段基本上問題不是很大。如果你看完還有什麼不懂可以私信問我哦，我會給提問者做詳細解答的，愛你們！
