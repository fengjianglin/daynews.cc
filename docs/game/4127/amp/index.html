這個遊戲好玩嗎：森林裡面不僅有野人還有飛機和兩條劇情線

天天要聞
2019-12-09

摘要

可能說起生存類遊戲，大家總會想起這個或者那個。無論是PC端還是手游，生存類遊戲都是一個越發受玩家喜愛的一個遊戲門類。原因無他，另類於生活的設定、節奏，都能夠給玩家帶來新鮮感。奇特的世界觀、獨立的生物群，能給玩家帶來探索的樂趣。本期為大家評測一款生存類遊戲——《森林》。

21世紀的森林裡面還會有野人嗎？其實另有真相。

可能光聽名字玩家會認為這是一款佛系遊戲，那就大錯特錯了。在一個普通的名字下面隱藏著製作組不平凡的內心。遊戲裡面的主人公也就是玩家是一位著名的明星，他帶著自己的兒子T乘坐飛機因為一起不明的事故而墜毀。在模糊之間玩家可以看見是一個紅色的人帶走了自己的兒子T，本故事所有的一切都從這個飛機墜落的森林開始了。以尋找兒子T為目的，玩家不得不跟隨著劇情任務走，打野人、潛水、跳崖、進洞穴。這一切的背後究竟隱藏著怎樣的驚天大陰謀。本期將從劇情錢以及玩家遊戲玩法上面來解析這一款生存類劇情遊戲。

飛機殘骸

先來說一說《森林》這一款遊戲新人玩家會體驗到怎樣的玩法。經歷過遊戲的開頭劇情之後就可以在飛機墜毀的森林開始生存之旅。需要注意的是這並不是一個普通的森林，而是一個充滿了野生動物、野人以及生化怪物的森林。至於說為什麼都21世紀了還會有野人存在，這都在接下來的劇情裡面有了合理的解釋。《森林》是一款第一人稱視角的遊戲，這也就是說明遊戲中會存在很多的視野盲區。也許不經意間跳出來一個野人或是什麼可是會把人嚇一跳。當然這也是生存遊戲的樂趣之一，刺激也是玩家追尋的目標。需要提前和大家說明的是《森林》這一款遊戲一個人玩是生存遊戲，兩個人玩也是生存遊戲，三個人玩是RPG遊戲，四個人玩就變成了逗比遊戲。所以說《森林》這一款遊戲是推薦一個人玩的。

這野人，別看了行嗎？一會還要吃飯呢。

先不說劇情，新手玩家一定要生存下去。

在遊戲的一開始需要和玩家分享的就是。在普通等遊戲難度上，玩家可以放心的去拿武士刀，中途並不會有野人攻擊玩家。周圍的牆壁大家一定要留心注意，有木板的地方記得砍開會有手電筒。這是一個非常非常重要的道具，一定要找到。推薦新手玩家把建築破壞調節成否，除非你想野人把你的家破壞殆盡不留一物。雖然這一款遊戲推薦單人玩，如果實在有很多小夥伴想要結隊一起玩。偷偷告訴你們一個秘密，這也是無意間發現的。那就是多人遊戲的時候物品是可以刷的，隊友的物品放在架子上然後點擊主菜單退出遊戲。之後進入遊戲會發現背包裡面仍然會有那些物品並且架子上的物品也並沒有清空。而且八爪等變異怪的護甲是防禦力最好的護甲，如果有機會千萬不要錯過。能得到一塊對於遊戲的玩法上是有很大的幫助的，包括最後的Boss也很難打破這個甲。野人也會有很多不同的等級，越到後期野人每天刷出來的數量越多並且攻擊力非常高，大概打玩家兩三下玩家就GG了。所以前期一定要趁早獲得關鍵性的道具：武士刀、潛水器、登山鎬、斧子。這些道具一定要儘可能早的弄到手。

《森林》截圖

而且要提醒玩家的是一定要小心野獸！在海裡面有鯊魚，在湖裡面有的時候會有鱷魚，玩家一定要小心。這都是一些比野人還要恐怖的東西，往往一口半血。被鱷魚咬一口還好，還有被隊友救起來的希望，在海裡面被鯊魚咬一口就直接判定死亡了！這一點一定要小心。還有雪山上面會有野豬，被野豬擊飛會產生僵直，而且野豬的速度十分快，很可能玩家還沒有反應過來在天上飛的時候就又被野豬打一下，那就直接死亡了。這一點上表示無能為力了，只能穿上甲跑，能避免的戰鬥就避免。說到僵直這一點上。其實玩家也完全可以利用僵直這一點對付野人。玩家用火焰武器對付野人，野人同樣會產生僵直。在野人僵直狀態下玩家可以切換成其他武器例如武士刀還有斧子之類的，這時候攻擊野人他是不會還手的。等到野人僵直結束玩家可以在此調換火焰武器如此往複。

能用火焰武器對付野人最好了。

《森林》的戰鬥不簡單，需要玩家去摸索取巧。

當然《森林》的戰鬥系統做的很真實，玩家的血量對於眾多的敵對生物來說就是一個擺設。這也正是人類除了頭腦是優勢外身體在大自然與野獸面前根本不堪一擊。遊戲中還有體力值等諸多限制因素。這無疑都對玩家充滿了敵意。其實《森林》這一款遊戲的戰鬥系統做的很不錯。可能大家在看遊戲直播或者是看貼吧裡面的帖子的時候會以為打怪在《森林》這款遊戲中很輕鬆。其實不然，《森林》這款遊戲的打鬥系統機制還是充滿了對新手玩家的敵意。第一人稱視角可能有的方向感略差的玩家找到背後的怪物都要好幾秒的時間，有這時間已經被攻擊好幾下了。並且可能很多玩家並不會拉怪以及躲避攻擊。還有就是《森林》裡面狹窄幽暗的洞穴時不時的傳出幾聲怪叫，一些怪物被設計的慘不忍睹不忍直視，就是那種生化怪物看過之後連吃飯的慾望都沒有。這一點上還是很具有視覺衝擊。

這一個個的怪物離遠看像個人，近了還是別看了。

當然《森林》作為一款開放式生存遊戲，就一定會有建築系統用來充當臨時落腳點或是防禦野人野獸的攻擊。更主要的是給玩家一個心理慰藉，讓玩家在緊張的高強度遊戲下也可以放鬆片刻。這也是幾乎所有可以建築的生存類遊戲的目的。可能並不會給遊戲帶來多麼便捷的幫助，更多的是給玩家以心理慰藉。《森林》的建造系統相對來說還是很不錯的，只需要木頭以及一些可以收集到的物資就可以簡單的建造一個屬於自己的臨時落腳點。可能《森林》這一款遊戲的神器就是打火機，用處真的是太大了。每次看見生活中不起眼的打火機這一物件時我都會感慨一番，多麼一個小巧又實用的工具。在《森林》的世界裡打火機有著別樣的用處，不說幾乎很大一部分照明都是打火機的工作。那一系列的火焰武器更是打火機的專屬武器，一切的火焰武器都需要打火機去點火。所以打火機的重要性不言而喻。

陷阱抓了個什麼東西。

《森林》的劇情結局竟然有兩個？

可能玩家在玩夠了《森林》的生存之後就會想著去走劇情，接下來就為大家分析一下劇情。值得一提的是根據玩家不同的選擇操作最終會出現兩個劇情走向。接下來就逐個分析，究竟哪一個才是大家心目中的理想結局。

玩家剛建的落腳點

《森林》的兩個結局前期走的都是一條劇情線。玩家可以通過收集來的線索一步步的推進劇情以及來分析這整個事件的完完全全的經過。遊戲中玩家可以發現一些照片、塗鴉、甚至還有錄像帶等一系列的線索。有自己的兒子T留下的線索，還有M留下的線索。有一卷錄像帶我們可以發現有一個不為人知的工廠，裡面跑出了怪物。那是一個生化工廠，原來這森林裡面的野人以及生化怪物都是這個工廠的傑作。不過果然惡有惡報，整個實驗室都被一個怪物摧毀。玩家就是要通過種種線索到達那間實驗室，跳下天坑、漫長的潛水、經歷左一個右一個的狹窄洞穴、爬過通風管道、上下過很多根繩子。玩家終於到達了錄像帶裡面的那一間實驗室。中途遇到的一切野人能跑就跑盡量避免不必要的消耗，也不要受到多餘的傷害。其實這個階段玩家距離最終Boss已經不遠了。

還是在樹上有安全感

《森林》的結局又是選擇，也太虐了。

到達了錄像帶裡面的實驗室，大家就要小心了。因為裡面的那個怪物還活著。什麼？這個是Boss？怎麼可能！這其實也是一個受害者、是一個生化怪物。通過劇情線我們大致可以知道這是一個生化基因實驗室，很多的人甚至花錢來這裡接受改造。大概是為了長生不老之類的吧。不過最終都變成了怪物。進入實驗室玩家要小心，遇見那個怪物玩家可以退到門後扔一個炸彈就搞定了。如果你是一個好戰分子那就去打吧。打過之後，經歷一條一條的劇情線路就來到了一間屋子。值得一提的是劇情裡面的通道幾乎都是唯一的，也就是說玩家並不用擔心迷路的風險。進入這間屋子玩家震驚了，因為兒子T已經死了！悲痛的拿齣兒子放在病床上發現機器可以鏈接兒子並救活他，但是代價需要另一個人放進機器裡面。之前把兒子抱走的紅色人已經死了，被塗鴉筆插死的。看著自己身上因為劇情而塗滿的紅油漆，玩家也許明白了這是一個循環，為了救活兒子而不得不犧牲另外的人。最終打完Boss-M，玩家來到了一間屋子。這裡有兩台電腦，一台顯示器上面正有一架飛機可以操作擊落。一台電腦可以終結這一切、、、
