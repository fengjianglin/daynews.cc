隨著年關將近，不知道今年諸位玩了哪些遊戲呢？有通關么？錢包有沒有扁下去呢？下面就跟小編來看一下，12月小編比較推薦的幾款PS4遊戲吧~

《妖怪手錶4++》

推薦理由：日本國民級遊戲《妖怪手錶》最新作，首次中文化，首次登錄索尼遊戲平台，是一款集寵物收集、養成、對戰於一體的遊戲，核心玩法類似於《寶可夢》，卻比《寶可夢》有著更高的可玩性，該遊戲也登陸了Switch平台，相較之下，PS4版幀率更高、更穩定，畫質也有一定的優化，小編在之前的文章中也有吹過這款遊戲，喜歡收集養成、喜歡偏柔和的二次元畫風的玩家可以嘗試遊玩。

發售時間：2019年12月5日

《新櫻花大戰》

推薦理由：《新櫻花大戰》相較於之前的作品突出了一個字”新”。玩家所扮演的主角將帶領自己的小團隊參加世界華擊團大戰。來自各國的華擊團，將會在全新的舞台爭奪世界第一華擊團的名號。除此之外，以往遊戲的玩法特色依然保留，玩家可以再次打打魔物、開開新型靈子戰鬥機、和妹子談談戀愛。喜歡戀愛養成、休閑向的玩家可以嘗試遊玩。

發售時間：2019年12月12日

《無雙大蛇3：終極版》

推薦理由：相較於傳統的《真三國無雙》系列，《無雙大蛇》整體遊戲風格更加魔幻與超現實三國名將對陣戰國豪傑，東方神仙爆錘西方天神。眾多武將、全新劇情、全解鎖的模式定能讓喜歡割草的你殺個痛快。喜歡割草、喜歡刷刷刷、喜歡熱血沸騰感覺的玩家可以嘗試遊玩。

發售時間：2019年12月19日
