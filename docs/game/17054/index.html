王者榮耀：蒙犽模型一覽，新增十大陣營，10位英雄收穫新皮膚

2019-12-14 2,494條評論 4,219次閱讀 3256人點贊

王者榮耀體驗服在凌晨的更新中迎來了全新英雄蒙犽，同時排位賽背景也更改為弈星的新賽季皮膚，此次我們來看看這次更新後蒙犽局內模型，以及陣營變動和新增皮膚都有哪些？

首先小個子英雄正式降臨峽谷，蒙犽稱號為烈炮小子，其局內模型還算精緻，手持類似「加特林」的渾天武器是一大亮點，同時背部的機關構架與武器同為銀白色，通過發紅的槍管可以看出蒙犽火力十足，對新英雄感興趣的小夥伴可以前往體驗服試玩看看！

若還對蒙犽技能不熟悉的小夥伴可以通過下圖了解一二，小編第一時間體驗後發現，蒙犽的普攻傷害並不高，但是技能傷害爆炸，特別是興奮狀態下的1/2技能可以打出不俗傷害，除非能打斷蒙犽的1技能釋放，否則刺客還未近身，就被一輪射擊送回泉水了，總之蒙犽將會是一位主要依賴技能打傷害的英雄，小編覺得用其走中路也是很不錯的！

另外這次更新後英雄陣營不僅優化了英雄關係星圖，還細分了王者世界的勢力陣營，以及增加了多個新陣營，分別是海都家族（米萊狄、馬可波羅和狂鐵）、灞上（項羽和虞姬）、狼旗（成吉思汗和王昭君）、玄雍（嬴政、羋月、白起和扁鵲）、南荒（阿軻和高漸離）、倒懸天（楊戩、哪吒等神話類英雄）、日落聖殿（安琪拉、亞瑟和雅典娜）、鴻門（劉邦、韓信和張良）、金庭國（蘭陵王）、封神地（孫悟空、后羿、嫦娥、豬八戒和牛魔），合計新增或更改了十個陣營，重新劃分後一票英雄的勢力歸屬更為準確！

最後我們再來看看未來都有哪些新增皮膚登場，通過體驗服更新後的數據獲知未來還有十位英雄迎來全新皮膚，分別是武則天、大喬、小喬、妲己、魯班七號、李白、李元芳、劉備、沈夢溪和貂蟬，其中貂蟬有兩款新皮膚（含FMVP），當然以上皮膚推出時間還以官方正式發布為準，感興趣的玩家可以期待一下！
