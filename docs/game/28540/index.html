《永劫無間》玩法猜測：武俠版榮耀戰魂？還是武俠吃雞？

2019-12-23 529條評論 1,445次閱讀 137人點贊

隨著《永劫無間》宣傳片在TGA2019上曝出，這款即將在2020年發售的武俠動作遊戲賺足了玩家們的目光。除了東方畫風以及流暢的動作吸引玩家之外，對於遊戲玩法方面官方並未給出明確說明。

《永劫無間》官方給遊戲的定位是多人在線動作遊戲，那麼鑒於這樣的類型，遊戲中又會如何呈現呢？

由於《永劫無間》給玩家提供了多個遊戲角色的選擇，或許分陣營對抗的玩法在多人動作遊戲中能夠有所發揮，玩家在遊戲中會有不同角色的選擇，不同角色之間相互配合，不同門派的招式不僅會有相互克制，還有各種配合出現，這樣的玩法可以更大程度讓開發玩家們對於不同角色之間搭配的想像空間。多少讓人感覺有些武俠版榮耀戰魂的意思！

如果在武俠動作的基礎上，配合沙盒遊戲中經常出現的30V30大規模戰爭，或許可以讓玩家們的激情瞬間拉滿。不僅有小規模的以少打多，還有可能直接面對的就是一個幫派的敵人，這樣的打鬥場面，無疑是對玩家們的顯卡有著嚴峻的考驗！

在武俠動作遊戲中，暗殺玩法可以說是武俠中不可缺少的元素。夜黑風高之時，一眾黑衣男子出現在房檐之上。接下來是劫富濟貧順便為民除害，還是密謀暗殺完成上面的指示，相信這種暗殺玩法帶來的刺激感，在多人動作遊戲中，勢必會收到玩家們的歡迎。

除了上述的玩法外，預告片中各種飛檐走壁的操作同樣令人期待，或許官方為了讓玩家更好掌握飛檐走壁，這方面操作上弄個全民挑戰什麼也不說定！好了，看過了這麼多關於《永劫無間》玩法的猜測，你們覺得這款多人在線動作的遊戲《永劫無間》會主打哪種玩法呢？
