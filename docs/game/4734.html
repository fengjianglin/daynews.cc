人造人-念力震懾者

人造人間－サイコ·ショッカー（人造人-念力震懾者）效果怪獸 ATK 2400 DEF 1500

只要這張卡在場上表側表示存在，雙方不能把陷阱卡發動，場上的陷阱卡的效果無效化。

人造人-念力插孔者

人造人間－サイコジャッカー（人造人-念力插孔者）效果怪獸 ATK 800 DEF 2000

「人造人-念力插孔者」的②的效果1回合只能使用1次。①：這張卡的卡名只要在場上·墓地存在當作「人造人-念力震懾者」使用。②：把這張卡解放才能發動。從卡組把「人造人-念力插孔者」以外的1隻「人造人間」怪獸加入手卡。那之後，對方的魔法與陷阱區域蓋放的卡全部確認。那之中有陷阱卡的場合，可以把最多有那個數量的「人造人」怪獸從手卡特殊召喚。

人造人-念力歸來者

人造人間－サイコ·リターナー（人造人-念力歸來者）效果怪獸 ATK 600 DEF 1400

這張卡可以直接攻擊對方玩家。這張卡被送去墓地時，可以把自己墓地存在的1隻「人造人-念力震懾者」特殊召喚。這個效果特殊召喚的「人造人-念力震懾者」在自己回合的結束階段時破壞。

人造人-念力王者

人造人間－サイコ·ロード（人造人-念力王者）效果怪獸 ATK 2600 DEF 1600

這張卡不能通常召喚。把自己場上表側表示存在的1隻「人造人-念力震懾者」送去墓地的場合才能特殊召喚。只要這張卡在場上表側表示存在陷阱卡不能發動，全部的場上陷阱卡的效果無效。1回合1次，可以把場上表側表示存在的陷阱卡全部破壞。這個效果破壞的卡每有1張給予對方基本分300分傷害。

電腦增幅器

電脳増幅器（ 電腦增幅器）裝備魔法

「人造人-念力震懾者」才能裝備。只要裝備著這張卡，裝備怪獸的控制者不會因裝備怪獸的效果使陷阱卡的效果無效化。這張卡從場上離開時，裝備怪獸破壞。這張卡的發動與效果不能被無效化。
