《御龍在天》公測時玩的“文字遊戲”，你是不是因此入坑呢？

2019-12-17 526條評論 3,238次閱讀 1398人點贊

《御龍在天》是國內首款大型國戰網遊，遊戲以漢末亂世為主題，將三國中的各個人物當NPC，前期需要刷副本打怪升級，後期就是參與國戰的積分拿獎勵，玩家和玩家之間可以組隊互幫互助，一起探索遊戲中的神秘世界。但是《御龍在天》在公測時為了更好的運營玩的文字遊戲你知道嗎？

《御龍在天》於2012年6月16日開始公測，但是內側的時候幾乎沒什麼人參加，運營了幾個月後玩家慘淡，於是就搞起了一個「文字遊戲」的活動，很快就有大量的玩家湧入，這是個什麼活動呢？相信很多老玩家都知道，也因此入坑；這個活動就是每升一級就能得到10QB，升到50就有500QB，升到1500級就可以得到1500QB，這樣的活動是不是很誘人，小編就是因為這樣的活動入坑的。

當時小編日夜兼并的刷副本打怪升級，玩了將近一個月，也升到了130多級，然後小編很興奮，因為很快就能得到1500QB了，但是TX會讓我們白白得到這1500QB嗎？顯然是不可能，這就是TX玩的「文字遊戲」，為的就是讓《御龍在天》前期更好的運營，當然TX也不會做的那麼明顯，小編被坑後，仔細看了看活動，活動第二條中寫著：「本活動結束時間請留意官方公告！」這條條例很能說明TX的「陰謀」。

TX的算盤打的很好，他們會在後台關注著升級最快的那部分玩家，這樣玩家即將達到150級的時候，TX就會將活動關閉，我們就這樣被TX的圈套套住，已經花費了這麼大精力，我們能這麼樣，舉報、投訴TX？沒用的，活動最後一條中，寫著：「本活動最終解釋權歸TX所有」，無論我們再怎麼憤怒也沒什麼用。最終TX把1500QB發送到遊戲中使用，這樣也算是為了補償第一批玩家吧。

《御龍在天》小編又恨又愛，愛的是這樣的遊戲值得玩一玩，遊戲本身是好的，大型國戰新穎的玩法；恨的是TX用這樣的方法來套住玩家，這樣會適得其反，TX不搞這樣一個活動，這樣的遊戲遲早也會火熱的，TX只是太心急了，是金子遲早會發光的。小夥伴們覺得TX這種做法是對是錯呢？
