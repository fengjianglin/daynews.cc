RNG虛空冠軍不是夢？YM玄學發動，明年S賽穩了

2019-12-18 1,343條評論 4,724次閱讀 1543人點贊

RNG現在的日子非常的艱難，因為很多戰隊的人員都在變動，RNG的核心隊員也都面臨著離隊，但好在UZI還在，同樣留下的還有S9拉胯的小明和小虎。打野位置新人由小龍堡接替，這也是一個從 YM出身的打野，這讓大家從RNG身上看到了希望，要知道現在的所有LPL冠軍戰隊，必要條件就是打野一定是YM出身的選手。

除了留下的選手，已經表明離開的選手還有AJ和一手凱南驚天地泣鬼神的ZRR，他們兩人當時為了救場來到了RNG，但是後來都沒有什麼太好的表現，所以一直都是被雪藏，很少能有上場的機會。狼行現在的去向暫不知曉，RNG的上單位置還是非常的而空缺。

現在的RNG太難了，不僅要重金挖人，還要保證原有隊員的不流失。因為IG和FPX告訴我們，好的戰隊不是必須要豪門，黑馬也是可以奪得冠軍的，所以很多人都想去一個默默無聞的小隊，然後一年努力衝擊冠軍，去了知名的戰隊壓力實在太大，很容易背鍋挨罵，不如小隊那麼隨心所欲。

RNG這次最大收穫就是YM打野，現在以玄學的角度來講他們也有了獲得冠軍的資質，而PDD很早以前就說過，自己非常喜歡也非常相信小龍堡選手。不過看起來PDD的眼光屬實一般，因為小天和寧王都是走出YM以後才憑藉自己打出了一片天，而在YM期間都被埋沒，至於結果怎麼樣，讓我們S10見分曉，希望RNG還能給我們帶來新的驚喜。
