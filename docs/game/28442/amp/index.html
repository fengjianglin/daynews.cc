雲遊戲的前景，和未來究竟會如何？你還需要購買CPU和顯卡嗎？

天天要聞
2019-12-23

我們為什麼需要雲遊戲？

我們都知道爽玩大作需要好一些的顯卡和CPU，當然你通常並不需要那麼一塊RTX2080Ti煤氣灶，也不需要I9-9900KF。

不過要獲得看上去還行的體驗，一兩千塊的顯卡CPU基本都是標配，配上主板SSD，2K 144Hz顯示器也不可少，結果發現遊戲都還沒玩上，電腦倒是已經花了6000了，而你要玩的PC版吃雞隻要50塊，堡壘之夜和APEX英雄都是0元就能玩。

遇到打折你花五六千買不到100個遊戲，50個也是能買到的。

所以這頗有一種買櫝還珠的意味。

當然電腦這東西平時你也可以用，但用一台用8核CPU，3000塊錢顯卡的電腦幹點雜事，上網看視頻的話又頗有一種用牛刀殺雞的浪費感。

雲遊戲就是為此而生的，你只需要按需租用設備，不想玩了不付錢就行了，而你的設備在不用的時候給別人也可以用，這樣理論上利用率提高了之後設備的成本也會降低。

而且雲遊戲有個更大的好處是如果放在伺服器端計算，開掛的難度會高得多，這樣無論是飛天俠還是水泥人不說絕跡，一定會大大減少，聯機環境也會好的多。

雲遊戲有普通離線遊戲做不到的內容嗎？

有，以FPS為例的話，即使現在標榜真實向的遊戲例如使命召喚和戰地的多人模式

仍然做不到偏向於真實的場景破壞，戰地還稍微有一些，而使命召喚用大炮轟牆牆都不會碎。

其他遊戲類型的話比如GTA5的鐵樹也算是一個核心的問題。

計算物理破壞的算力對於單個客戶端機來說太高，但如果是伺服器統一演算的話代價會小的多，比如計算物理破壞需要20或者30個CPU核心的算力，這個在你自己本地跑需要一個非常高價處理器才能承擔。

但如果在一個線上30-50人的房間，則一個人實際只需要承擔1-2個計算核心的費用，這樣子在很便宜的情況下就能有效提升畫面的表現。

問題：網路延遲

和傳統視頻網站不一樣的的情況是，雲遊戲有他的即時性和中心性，不像視頻的內容相對固定——如果你要看一段視頻，很有可能只是從你家門口的緩存伺服器里下載的。

但云遊戲每一個人看到的畫面都完全不一樣，這樣只能從中心伺服器傳輸數據到你的終端，而這個對網路的要求很高——即使在國外，運營商之間的互通也很糟糕，這對網路延遲和互動的考驗非常高。

之前測試結果通常都是表示能玩，但是像FPS和動作玩起來並不是特別順暢，動作類遊戲操作跟不上需要導致各種莫名橫死，是個不小的問題。

問題：流量與資費與設備考驗

根據測試來看Stadia一小時大概需要7G的流量，先不論速度，這個流量已經不是絕大多數國家的廉價通信套餐能夠覆蓋的了。

而玩一個遊戲需要消耗的流量，甚至要高於下載這些遊戲到本地，使用固定寬頻的話，不少國家也是限流量的。

而如果你待在家裡的話，如此大的流量也很容易讓內網癱瘓——畢竟普通家用路由器並沒有針對雲遊戲的優化，之前測試的時候多家媒體也都在反映雲遊戲會遇到瞬間掉線，視頻傳輸模糊等問題。

挑戰：高性能移動設備

雖然台式機和手機的性能並不可同日而語，但實際上已經有不少遊戲可以在手機和PC上享受基本相近的內容，只是畫面不足，例如絕地求生，堡壘之夜等。

而像巫師3，守望先鋒等大作甚至已經登錄NS，雖然運行情況一般般，但至少是能玩的，NS的性能也不能和目前最新的旗艦手機相提並論，這實際上意味著移動設備跑大作已經不是問題了。

如果能在本地運行，則也沒有必要上雲浪費流量和享受延遲了。

訂閱制與雲遊戲：

最近一個對雲遊戲有利消息就是訂閱制已經被大量玩家接受：

無論是20美元一年可以玩稍老遊戲的EA ACCESS。

還是一直白給的XBOX GAME PASS。

藉此你都可以用很便宜的價格玩到一大堆遊戲。

而且大家接受度很高，這客觀上來說完全有利於雲遊戲的發展——你只需要在遊戲訂閱的價格上加一些雲服務費就可以爽玩，而不受時間地點的限制。

未來發展

微軟：

微軟這邊的雲遊戲服務XBOX Play Anywhere已經在全球多個國家地區公測了，介於微軟在多個地區（包括中國多地）有多個數據中心，再加上自家得天獨厚的陣容和遊戲訂閱，因此對全世界包括國內的玩家來說肯定是值得期待的。

索尼：

PSnow作為索尼的雲遊戲服務，最近已經開始直接提供給PC玩家使用，雖然使用的區域會會受到限制（目前僅限美國和歐洲部分國家），但以索尼的陣容而言可以說是頗具競爭力，但很有可能與國內玩家無緣。
