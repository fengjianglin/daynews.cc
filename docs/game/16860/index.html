《符文工廠4 豪華版》評測：獻給系列粉絲的一次華麗新生

2019-12-16 1,227條評論 3,857次閱讀 629人點贊

2012年MMV推出了旗下《符文工廠（符文工房）》系列的第四部正統續作，輕鬆可愛的畫風加之較高的可玩性一舉在玩家中獲得了不錯的評價，時隔7年的時間《符文工廠4》正式在Switch平台得到了高清復刻，增強的遊戲效果和全新的內容足以讓玩家再次體驗冒險與日常所交織的奇幻生活。

符文工廠4 豪華版 | Rune Factory 4 Special

開發商：Marvelous

發行商：SEGA

首發日期：2019年12月5日

平台：Switch

屬性：日常、冒險、奇幻

　　作為一款高清版的遊戲作品，本作在大方面上並沒有進行太過高清化的處理，遊戲整體無論是畫風還是建模等要素皆維持了3DS原版的樣式，卡通風的設計延續了《符文工廠》系列的整體風格，帶有半童話風的世界配合鮮艷的色彩讓達到建築場景小到傢具等方面的設計都讓人感到相當舒服。不過在這基礎上也有一些並不是那麼容易被人所接受的地方，例如在遊戲中角色、敵人以及道具等「可動」的建模依舊是原版中那樣有稜有角的形式呈現，相比3DS的小屏幕而言，此次Switch版便直接將角色建模進行放大，在原版中圓潤的角色建模一下子被打回了原型，簡單來說有點類似直接將原版的建模硬拉成了Switch的比例，如果你習慣了精細的建模刻畫，那麼這款作品在這一「起跑線」上就可能會稍稍輸了一成。

　　如開頭所提及的那樣，作為一款高清化的作品，本作在內容上完整沿用了原版的要素，同時針對Switch的平台特性進行了一定的調整，主要的改變來自於整體的UI設置，作為雙屏幕的3DS，原版遊戲中上下屏幕各自分擔了主遊戲界面和快捷菜單的功能，不過如果玩過原版的玩家可能會有所了解，在《符文工廠4》中以往下屏所分擔的一些便利操作被刪掉了很多，例如《符文工廠3》中玩家可以利用下屏快速進行耕地等操作，而到了四代這一切都要主角一下一下地通過按鍵的方式完成，在此次高清版里，Switch的觸屏功能在這些方面依舊沒有起到任何作用，通體的玩法操作上也延續了原版《符文工廠4》的全部，可以說之前的操作是怎樣的那麼這次的操作還是那個樣。

　　不過雖然在一些簡單操作方面，本作並沒有「太留情面」，但實際的控制上，本作對於按鍵的配置還算得上是相當合理的，通過L鍵的快速呼出菜單對裝備進行快速切換的設定得到了完整保留，呼出後可以通過很簡單的方式對當前的道具或是魔法等要素進行更換以及使用，在這種即時戰鬥的遊戲作品裡，這樣快捷便利的操作還是能為戰鬥帶來不少節奏上的快感的。

　　如果再說遺憾的話，可能就是本作依舊沒有支持全程語音，在對話過程中角色的語音只會針對該角色的一些經典台詞，甚至有時只是一句簡單的笑聲，不過基於原版的內容，一些聲優之間的演出就足夠帶來一些動畫情節上的代入感，戰鬥等方面的音效與前作並無太大的差異，這點高清版也得以原封不動的保留。對比語音本作的BGM就與場景達到了一個相當高的契合度，配樂並沒有加入什麼全新的曲目，但每一首都很好的渲染了不同對應的場景，從玩家生活國家的安寧歡快，到迷宮的深邃緊張，不同的場景呈現的效果配合音樂的渲染，讓代入感加深了很多，也讓冒險的過程顯得沒有看上去那麼的乏味單調。

　　作為《牧場物語》系列的姊妹篇，耕種系統在本作中被完整保留了下來，與牧場系列的基本方式沒有太大的區別，在日常生活冒險的間隙，玩家可以對自己後院的農田進行打理，通過「收拾雜物、耕地、播種、澆水、收穫」等一系列步驟，將不同的作物進行培育，不同的作物各有著自己的成熟時間，短則幾日多則半月，在完成了日常冒險之後，提起水壺來到菜園澆上一圈水，在遊戲發展到中後期時，往往會成為一個習慣性或是必不可少的步驟，當作物成熟後玩家可以將它們賣出賣錢，或是交給其他NPC提高不同角色的好感度。隨著對於系統教學方面的不斷深入，農場會逐漸壯大起來，除了常規的種花種草以外，玩家還會在農場里獲得一些類似柴堆、魔物棚等設施，讓稍顯平淡的農耕生活逐漸豐富起來。隨著劇情的推進，遊戲還在一些野外的位置提供了更多的耕種地點，也為冒險帶來了一些可以「提高生活質量」的探索性。

　　根據劇情的安排，玩家所要扮演一名王子/公主的角色，而在日常的愜意生活中，還要為國家的事情操勞一下，基本就是完善國家的建設、拓展一些新的項目以及開展不同的慶典活動等，而開展這些項目則需要消耗一定的素材，素材的來源就是要通過推進劇情、野外打怪收集以及耕種等方式來進行，國家事務也在很大程度上起到了一種鏈接所有事宜的線，讓所有的活動彼此環環相扣起來。另外在日版中以DLC形式呈現的「外章」部分，在此次豪華版中也作為了特典進行了收錄，在補完劇情的同時也讓玩家擁有了更多可以遊玩的空間。

　　戰鬥系統作為區別於《牧場》系列最大的關鍵，在本作中同樣佔據著很大的分量，相比較牧場的拓展，《符文工廠》推進劇情的更加重要的因素就是伴隨著戰鬥而逐漸刺激起來的探索要素，同以往的作品一直，《符文4》同樣是以迷宮的形式作為開展，玩家可以在城內至多帶上兩名隊友，一同探索尚未開拓的新區域，隨著區域的不同野外所遇到的敵人也會各不相同，敵人的攻擊套路基本沒有太大的區別，但以種類來區分的話即便到了遊戲後期也總能帶來一些驚喜。相較於原版《符文4》此次豪華版中，隊友AI似乎下降了很多，在戰鬥過程中經常會看到他們站在原地發愣，任由敵人宰割，加之戰鬥的手感判定等方面遊戲依舊維持著原版的形式，讓整體的戰鬥體驗表現得並不算太好，不過豐富的武器種類也在某些程度上彌補了這一遺憾，對於老玩家而言，在遊玩的過程中也多少能夠找回曾經的感動。

　　另外值得提及的是，針對一些以往的老玩家，此次的豪華版中遊戲還追加了全新的「地獄」難度，體驗上即便是路邊的小怪都能對玩家造成巨大的傷害，幾乎在冒險的過程中時刻都要小心前行，稍有不注意就會陷入很大的劣勢，但另一方面高難度的加入也為系列的老玩家帶來了一種別樣的挑戰性，有時即便面對幾個平時不會重視的雜兵都有可能會陷入苦戰。

　　在忙於國家公務的同時，本作同樣允許玩家與NPC進行戀愛，雙主角的設定即便是女玩家也可以利用女主的角度來交往一些自己喜歡的小哥哥，在遊戲的主菜單中玩家可以查看當前小鎮居民的好感度，當臨近成為結婚對象後好感度會以心形作為標誌，通過聊天溝通以及贈送禮物便可以直接提升好感度，每個角色都有各自喜歡的東西，針對性的贈送可以達到事半功倍的效果，反之則會降低好感度，這點與《牧場物語》還是比較類似的。在一些溝通中玩家就可以了解到一些角色喜歡和討厭的東西，這也讓和NPC的每一段對話都要仔細觀看，當成功與一名角色達成結婚條件後便可以觸發特殊的結婚劇情，隨之還會解鎖遊戲新加入的「新婚模式」的新劇情，新模式主要是講述兩人結婚後的日常生活，也算是在側面完善了整個《符文工廠4》的劇情，只不過新婚模式需要一一針對性的解鎖每一個角色，因此想要開啟全形色的新婚模式還是要稍下一番功夫的。

A9VG體驗總結

　　以本作的表現，它並沒有帶來太過於「改頭換面」的體驗，整體的玩法表現上和原版幾乎完全一致，簡單的耕種、戰鬥、探索、戀愛，雖然不複雜但卻頗有一番實感，只是在表現上它確實顯得有些粗糙，致使新玩家可能很難融入這個頗具童話的世界中，但對於喜歡這些元素的玩家或是系列粉絲而言，這場冒險反而會顯得相當有趣，在《符文工廠5》之前不妨可以拿這部作品先來作為一道開胃菜，重新感受一下那個幻想的世界。
