馬上2019年即將結束了，很多小夥伴們都在盼望著明年快點到了。本來以為年關將至英雄聯盟在出完塞娜和厄斐琉斯後就改收收心，為S10準備準備了。但近日拳頭又公布了新英雄的藍圖計劃，還說是一位近戰格鬥家，會拳術的那種！網友心想難道是JOJO？

在官方新發布的4張圖片中我們可以發現，如下，蓋倫、諾手、盲僧、大蟲子等英雄紛紛被揍飛了出去，尤其是蓋倫被一擊右勾拳打的賊慘。看情況這位英雄應該是有個擊飛技能的，那麼亞索又要快樂了？

後面也有網友爆料一些信息：新英雄賽特將會是上單，是一個近戰格鬥家，QWE分別為左勾拳、右勾拳、前踢（位移），三個動作。每2個或者3個技能一起釋放打出不同的組合技，比如沖拳、勾拳、抱、抱沖等等，按照順序的不同打出了很多個組合技。而大招是給自己加個BUFF，加傷害，然後所有小技能CD變很短可以在大招持續時間內打出很多的連招！按照這樣說的話這不就和拳皇一樣了嘛，把橫版格鬥遊戲帶入MOBA遊戲里這個創新道倒也不錯。想想大門五郎近身抱摔ADC的樣子想想就刺激！

網友都在紛紛猜測這個英雄是個什麼樣子，有人覺得格鬥家就該有格鬥家的樣子，必須是一身腱子肉。JOJO的支持者最多，那種拳拳到肉的樣子JOJO最能表現出來的。作為一個猛男格鬥家性感囚犯也是不錯的代表，滿身橫肉簡直就是力量的代表人物。也不知道為啥拳頭會出這樣一個英雄，可能是想在過年的時候再給ADC們一個新爹吧！

大家覺得新英雄賽特會以什麼樣的形式出現呢？是滿身肌肉男嘛，看來哲學將會進軍峽谷了！大家可以在評論區留言說說自己的看法呢！
