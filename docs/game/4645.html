王者小喬天鵝之夢重做吹爆！白天鵝黑化成黑天鵝，蘿莉身御姐心

王者榮耀計劃四款品質非常高的皮膚都將迎來優化，分別是呂布-天魔繚亂、李白-鳳求凰、諸葛亮-武陵仙君以及小喬-天鵝之夢。前三款皮膚優化的變化不是很大，相對較大是的武陵仙君，被動樣式也改成了桃花符的樣式，還增加了全新的回城特效。

不過最重要的還是小喬天鵝之夢的優化了，相當於全新重做的一個新皮膚。之前小喬的天鵝之夢一直在優化，但是玩家一直不滿意，儘管增加了很多的局內動作，優化了出場等等，所以官方索性完全重做了。

保留了原來芭蕾舞者的設計，但是形象已經大改了，之前大家不滿意的翅膀直接取消了，露出了美背，並且形象完全變化，雖然是蘿莉的身高，但是不論是氣質還是樣貌造型都很御姐。

公主受到詛咒變成白天鵝，只有王子堅貞的愛才可以打破魔咒，但是王子卻被邪惡的黑天鵝米索，所以白天鵝在無望中分裂出了黑化的一面。

看著這個背景就覺得很A，暗黑優雅華麗的黑天鵝人設非常可以。按照設計，小喬將會以黑天鵝的形式常駐，但是在出場展示動畫中，將會出現白天鵝退化成為黑天鵝的動畫，由天真轉化為暗黑。

這只是目前的初稿，經過仔細的打磨成品將會更加的好看，大家有什麼想說的嗎？
