假如LOL也有朋友圈，亞索很無奈，拉克絲最真實

2019-12-10 162條評論 2,115次閱讀 1172人點贊

在現在這個網路時代的社會，微信已經逐漸取代了大家相互之間使用手機打電話的位置，而且不少人都是十分喜歡朋友圈這個功能，每天不刷都會不自在，更是有著不少人一天都會發上很多條動態，哪怕就是起床刷牙都要在朋友圈表示一下。

那麼，如果在英雄聯盟這款遊戲中，所有的英雄也是有著朋友圈的存在，都會發生什麼樣的趣事呢？

亞索，這位在英雄聯盟中出現率最高的英雄，不管是路人局還是五黑局，絕對是最常出現在大家眼前的英雄，但是，如此受歡迎的一位英雄，也是有著不少玩家都是十分懼怕隊友選這位英雄的，原因很簡單，就是因為自己的隊友只要是選擇了亞索，就不要妄想著可以秀起來，畢竟托兒索的名頭可不是毫無根據的。

假如LOL也有朋友圈，亞索很無奈，拉克絲最真實

但是為什麼這樣的英雄反而如此受人待見呢？歸根到底，就是這位英雄實在是太能秀了，就算是前期0殺浪成狗，也是在後期最容易拿到5殺的英雄，所以歸根到底，亞索還是很無辜的，畢竟超神和超鬼只有一字之差，但是卻有著天壤之別。

小魚人的朋友圈當屬最搞笑了，圖片里的這位人物想必大家都是十分熟悉的，沒錯，就是大名鼎鼎的「海王」，在《夏洛特煩惱》中的袁華，這一出場自帶BGM的人物，讓其在一夜之間便是火遍了大江南北，更是被眾多的玩家稱之為「潮汐精靈」，與小魚人也是完美的「撞衫」，而下面的評論更是有意思，一時之間眾多的英雄都是已經找到了平行世界的自己。

假如LOL也有朋友圈，亞索很無奈，拉克絲最真實

而拉克絲的朋友圈就有點真實了，想必在季前賽更新的新版本，大家也是都已經體驗過了，不得不說，現在的新版本真的是對ADC這樣的位置太不友好了，不管是地圖的調整還是裝備的整改，都是將ADC打入了無底深淵，讓這個曾經的霸主跌落神壇。

假如LOL也有朋友圈，亞索很無奈，拉克絲最真實

拉克絲也是一位十分全能的英雄，能中能上能輔助，能控能跑能輸出，但是這麼好的優勢卻是被自己的ADC隊友拖了後腿，自己都可以在下路一打二了，真不知道ADC還有什麼用，這條朋友圈也是讓一眾輔助揭竿而起，勢必要把ADC拉下馬。

石頭人自打出現以來，就是十分的強勢，現版本打中路或者上路更是厲害，而且與自己隊友的亞索相互配合，更是最佳的搭檔，石頭人的這條朋友圈，也是自身實力的體現，但是當亞索看到之後，卻是表示這種事情別帶我！

最有趣的是卡特琳娜的朋友圈，雖然也是在調侃亞索，但是下面的評論卻是比這條朋友圈本身更加有趣，蒙多的搶鏡簡直就是這條朋友圈的精髓所在。
