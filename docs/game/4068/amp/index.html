200星王者全被封

前段時間大概在10月14號，我打開王者榮耀體驗服突然發現200星的王者大神，剛開始看見我覺得很不可思議，因為那時候的新賽季才過去不到一個月，居然出現了這麼多200星的王者大神，體驗服的人本來就少，不可能在這麼短時間內衝上200+，這裡面絕對有貓膩，而在這幾天我上線體驗服發現這些200+的王者都不見了，就連那些百星王者全部的不見了。這背後究竟有什麼？

其實這些人都是使用了第三方軟體，簡單來說就是外掛，如今他們終於得到了官方的封殺，現在全部已封。小編認為玩遊戲都是為了開心！為了遊戲的公平，大家應該好好維護王者峽谷，拒絕使用外掛。不然影響遊戲平衡，換來的也是封號處理，完全不划算！

送皮膚的背後有什麼？

不知道從什麼時候開始公屏的聊天都開始送皮膚。但是在體驗服的公屏都是外掛。

首先我們來看看體驗服的這位夢之淚「傷：「買掛自喵，透視，秒殺，無傷，英雄聯盟內測手游，要的加」

看見這種人小獃獃都是反手一個舉報，一般這種都是騙子，可是舉報過後也許這些人只是被禁言，根本治標不治本，回頭換個號繼續發，官方真的重視這方面的事情嗎？

在正式服到處都是送皮膚的，比如圖中看見的「有一起玩嗎？加我送皮膚」類似這種，在這些背後還有很多貓膩。這一類屬於騷擾信息，可一直以來都沒有得到解決！

最後我還發現王者榮耀體驗服更新之後騰訊遊戲的圖標變了，用心創造快樂這句話居然沒了，對此小夥伴們怎麼看呢？
