《戰雙帕彌什》遊戲公告可自己修改？S級露西亞真的是本人嗎？

天天要聞
2019-12-22

諸君安好，霧夏菌報道。

在近日，遊戲官方發布了最新的活動預告——極地暗流，而有玩家卻在不經意間發現，遊戲的公告竟然是可以自己修改編輯的。

這個事兒到底是真是假？有很多玩家專門登錄遊戲，結果並沒有發現這個問題，因此認為圖片是P的。實際上，這個問題的確很有可能存在的，只不過並不是所有的區服都是這樣，而且還有很關鍵的一點，這裡不是手機打開的，而是要下載模擬器，在電腦上運行遊戲。

本菌試過了官方服和B站下載的遊戲，都沒有出現這個問題。然而在使用其他渠道服下載的遊戲，出現了類似的情況，有了一個編輯框的顯示。

不過可惜的是，可能是下載的版本還是不準確，或者說遊戲已經修復了，所以這裡並沒有發現能夠修改編輯的文字，有興趣的玩家可以自己下載渠道服試試。

再一個就是關於最新的S級構造體露西亞·深淵之紅。但隨著預告片公布，也有人提出了質疑，這個露西亞真的就是我們的女主嗎？

在遊戲的劇情中，咱們的女主紅蓮露西亞，曾經有過一個與這位S級露西亞對戰的關卡，也就是B級打S級，除非是一些操作極度兇悍的大佬，否則基本上都無法以勝利的姿態通關。在當時，這名露西亞的形象就令人懷疑，因為後續在露西亞的故事中得知，露西亞有個妹妹，叫做露娜。

只不過，雖然露西亞不讓露娜一起接受改造，但露娜卻在第二天清晨，不知為何被帶走參與了改造。後來改造的負責人聲稱露娜的改造失敗，並且已經將露娜當做廢品其處理掉了，露西亞再也見不到露娜了。

然而有個鏡頭引人注意，那就是露娜被帶走的時候，黑衣人說如果改造成功，這群孩子也能擔任區域的防禦保護工作，而修女則是罵了聲道貌岸然的傢伙。

其次在S級露西亞的預告片中，明明講的是露西亞，卻插入了另一個引人猜想的畫面：

為什麼突然加入了露西亞與露娜的畫面？想表達什麼？還是單純的剪了個畫面湊時長？

一襲白髮、有著灰鴉小隊的標籤、格外關注紅蓮露西亞。被處理掉的露娜，並沒有見到屍體，按照一些劇本的發展，露娜很可能是以改造失敗的說辭，被秘密選中參與了其他的行動，而且這活動並不善良。

菌說

很多人可能刷遊戲，根本不看劇情，但其實遊戲的劇情還是挺不錯，值得細品。如果S級露西亞，後續牽扯出新的劇情，並且搖身一變，成了露娜，那或許會更有趣。
