權游手游步兵輸出主力的抉擇：錫娜拉VS巴里特

2019-12-04 0條評論 16次閱讀 0人點贊

摘要

隨著S級槍兵指揮官龍媽的上線，槍兵防守陣容逐漸成為主流。因此，領主需要強大的輸出型步兵指揮官來克制槍兵。此時，錫娜拉和巴里特成為領主們首選步兵指揮官，那麼今天就讓我們一起分析一下巴里特和錫娜拉的實戰實力吧。屬性數值分析：首先是基礎屬性，從基礎屬性上看，巴里特和錫娜拉的差距不大。

隨著S級槍兵指揮官龍媽的上線，槍兵防守陣容逐漸成為主流。因此，領主需要強大的輸出型步兵指揮官來克制槍兵。此時，錫娜拉和巴里特成為領主們首選步兵指揮官，那麼今天就讓我們一起分析一下巴里特和錫娜拉的實戰實力吧。

屬性數值分析：

首先是基礎屬性，從基礎屬性上看，巴里特和錫娜拉的差距不大。滿級滿品質後，巴里特最高武力值2644，錫娜拉最高武力值2657。因此從基礎屬性上來看，兩者差距並不大，但詳細屬性方面，錫娜拉和巴里特是有些不同的。

詳細屬性方面巴里特的生命值屬性、防禦屬性都高於錫娜拉。巴里特的生命值最高可達到27000+，錫娜拉的生命值最高只有16000。而防禦方面，巴里特比錫娜拉高出近400點。而錫娜拉的技能效果比巴里特高300點，技能輸出要略強一點。因此從屬性方面分析，巴里特的生存能力要略高於錫娜拉。

技能效果分析：

雖然屬性相差不大，但技能以減療和持續傷害為主。主動技能可以對敵人造成傷害，並降低敵人的治療效果。3技能可以疊加攻擊，6次攻擊之後，最高可增加近7200+的攻擊加成；4技能則是增加生命汲取，進一步提升巴里特的生存能力。由此也可以看出，巴里特是一個純普攻型指揮官，只是靠普攻輸出。

而錫娜拉卻有所不同，錫娜拉的1技能可以對敵人連續造成3次攻擊，每次造成一定的技能傷害。雖然看起來1技能不是特彆強，但錫娜拉還有2技能和3技能可以對1技能進行加持。

2技能觸發時增強攻擊，3技能每4次攻擊可增加自身技能效果，並使1技能造成額外傷害。按照演武場中指揮官的攻擊頻率和技能觸發頻率，錫娜拉在觸發1技能時，有95%的概率會同時觸發3技能，為1技能接下來的攻擊提供增幅，使得錫娜拉的爆發能力瞬間提升。

而錫娜拉的4技能屬於攻擊提升技能，最高可增加2491點攻擊力，配合2技能時錫娜拉可以增加4800攻擊。雖然攻擊力弱於巴里特，但是錫娜拉的爆發絕對比巴里特要強。

那麼結論就出來了，巴里特的是一個可以提供減療效果，並可利用普攻不斷造成高輸出的指揮官。而且巴里特具有一定的防禦能力。而錫娜拉則是一個脆皮爆髮型指揮官，配合技能可以輕鬆打出成噸的傷害。在實戰中如果追求輸出用錫娜拉，如果在追求輸出的同時求穩，則選擇巴里特。
