FGO強如隊長能否拯救人理，換成A組還會不會有異聞帶？

2019-12-24 256條評論 2,040次閱讀 206人點贊

FGO日服主線2.51劇情中，A組隊長沃戴姆中途出現給了我們一個下馬威，開寶具來了個劇情殺，讓無數玩家感到絕望，開始認可沃戴姆的實力是真的在神靈之上，對比這邊大批從者保護的御主還打不過一個單槍匹馬不帶從者就能打贏的魔術師，沃戴姆才是真正的龍傲天，玩家們也開始懷疑，是否從第一部開始，沃戴姆就有能力可以拯救人理，而是被咕噠撿了「便宜」呢？本期我們就來討論組長是否能夠拯救人理。

其實這個問題也沒有討論的必要，因為蘑菇早就已經給出了答案，沃戴姆和戴比特有能力阻止第一部的人理燒卻，其他人都沒有辦法，而第二部的異聞帶，沃戴姆應該能夠應付大半，只是能夠應付大半而已哦，並不能完全擊破異聞帶，另外蘑菇也說了，沃戴姆在身為隱匿者的情況下是註定贏不了的，所以你強任你強，誰才是真正的天命之子沃戴姆心裡還沒點赫拉克勒斯嗎？

這個時候可能會有人想問，如果換成A組去拯救人理的話，還會出現異聞帶嗎？答案是肯定的，不管A組死沒死，異星神的降臨都是必然的，劇情中的51區早在一年前就撿到了外星人，並且對外星做了實驗，企圖讓外星發出求救信號，從而獲得更多研究樣本，結果玩火自焚，外星人直接把地球白紙化了，而紫苑（席翁）也早在一年前就預測到了地球被白紙化的事情。

從這裡就得出兩個結論，一是假設是A組在拯救人理，在阻止人理燒卻之後依然異星神會降臨，並且就算沒有A組，異星神應該也能從地球上選擇其他人來作為自己的引路人，並且如果是這個結果的話，即便是強如沃戴姆也無法阻止異星神降臨，最終地球被侵佔。

二是紫苑知道地球會被白紙化，但是沒有提前通知迦勒底做好防護措施，是因為如果幹涉了時間線的話，後面的事情就會在自己的預知之外，無法掌控，也就是說，紫苑相信這個有咕噠的迦勒底能夠拯救地球，而不相信主角是其他人的迦勒底能夠拯救地球。

從這兩個方面來看，世界和紫苑都是選擇了勝算大的一方，就是咕噠來拯救人理，大家也可以用平行世界的方式來理解，所有選擇A組的世界線全部死光了，只剩下咕噠這一條世界線能夠續存，所以不太需要做如果是A組這樣的假設。

而關於咕噠的能力問題，最多討論的就是咕噠只是一個攝像機，無法為從者提供魔力，也無法參與「神仙打架」，甚至近期在2.3序章出現的時候，還出現了認同貝里爾所說的只會躲在女人背後英雄這個說法。

不過其實真正的聖杯打法，確實是魔術師在遠處指揮，從者上前線戰鬥，所以卡多克對咕噠明明很害怕卻還是站在最前線的行為還是有些欽佩，從這一點看，雖然是無奈之舉，但是咕噠的勇氣還是可嘉的，換做別人的話，可能會連攝像機都沒有了，而且主角像沃戴姆那麼能幹的話，那還要召喚從者做什麼，既然不用召喚從者，哪來那麼多從者實裝給玩家舔呢。

沒有優質魔術師刻印確實非常無奈，可那有什麼辦法，每當看到有人罵咕噠的時候，我就想到真嗣這個場景。

明明就是個普通人卻還要背負拯救世界的重擔是最累的，咕噠並不是自己爭著搶著要去拯救世界，全世界就剩下他一個人能當御主那有什麼辦法，好不容易打敗蓋提亞心想終於可以回到正常生活讀書工作了，結果迦勒底最引以為豪的A組全部反水帶外星人來打地球，而且大部分的原因是自己還沒拯救世界就死了而且還被一個無名小卒給救了我們好氣啊所以要報復社會的侵佔地球，然後全世界又剩下咕噠一個人，又要去拯救世界一次，這種事情全都不是自己自願要去做的，結果救了還要被兩個次元的人攻擊，只能說按照月球的設定，如果咕噠說我不幹了，迎來的結果只會是人類GAMEOVER，而不是有其他人來拯救地球。
