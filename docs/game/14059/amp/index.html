火影手游：帶查克拉打決鬥場，這三個忍者可以玩的很歡樂

天天要聞
1 週 ago

查克拉秘卷最早的只是增加一個奧義點，後來伴隨著仙鳴的上架才加強到增加兩個奧義點，沒想到，這個當初專門為仙鳴準備的秘卷，現在被各種歡樂忍者使用，除了早期的查克拉佩恩天道莽開大以外，其實，我還想到了下面這三個忍者。

絕

這個忍者的奧義比較特殊，直接釋放的話只有一個抓取效果，傷害也很一般，但好處就是只耗費兩個奧義點，所以，很多玩家喜歡用絕帶查克拉打娛樂場，管你對面是霸體忍者還是四豆雷影，只要我瞄準你開大，你就要被我抓進來，並且還能穩接技能，一般情況下就是奧義後接兩個技能，如果沒有技能可以留一手查克拉繼續開大，抓到對方想退遊戲。

團藏

這個忍者的奧義同樣比較特殊，第一次釋放會開啟伊邪納岐，然後再次點擊奧義的話才會釋放里四象封印術，當團藏處於伊邪納岐狀態的時候被擊殺就會直接復活，同時重置替身時間，有時候在娛樂場里團藏帶個查克拉可以實現多次復活的效果，也是異常的歡樂。

泳裝小南

這個忍者帶查克拉就比較正常了，奧義沒有太特殊的機制，但好在她的奧義具備全屏效果，也就是說當泳裝小南在中場位置釋放奧義的話是屬於全屏效果，任憑你逃到天涯海角也會吃到奧義的後半段傷害，這還不算什麼，最關鍵的是你不僅要吃部分傷害，在吃完傷害後還會被拉到泳裝小南面前來，這樣可以更方便的接技能，所以，當泳裝小南帶了一個查克拉就跟佩恩天道一樣可以各種莽開大。

除了上面這三個忍者，其實現在的忍戰我愛羅有的也會帶查克拉，因為奧義具備抓取效果，同時可以無腦接技能，一局兩個奧義傷害也是相當恐怖的事情，當然了，查克拉這個秘卷同樣也適用於其他很多忍者，我只是個人覺得這幾個忍者帶查克拉會比較好玩，你們也可以推薦一些你們喜歡帶查克拉的忍者，看看有沒有什麼新玩法！
