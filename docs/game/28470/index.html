Hero久誠幹將莫邪回歸，8/0/11，輸出40%還是熟悉的味道

2019-12-23 339條評論 2,749次閱讀 206人點贊

王者榮耀冬冠杯繼續進行，Hero久競在輸給了RW之後形勢危急，而當對手換成了KR賽區的NOVA之後，Hero再次發生了變陣，射手位的久酷被換下，換上了久星打射手。在陣容方面也有很大的改變，Hero的久誠選到了成名英雄幹將莫邪，並且換上了自己的FMVP皮膚，真的有种放手一搏的感覺。

前期的小規模團戰Hero明顯更佔優勢，7-0大順風開局。十二分鐘的時候久誠的幹將莫邪再次打出爆炸輸出，配合隊友打出了一波小團滅，NOVA只剩下了上官婉兒一人。最後一波NOVA放手一搏，馬可波羅想要上前直接秒掉久誠，但是久誠輝月在手，配合隊友反秒了馬可波羅。接下來又換了復活甲和熾熱支配者站到了最後。Hero直接打出一波零換五，孫尚香還拿到了四殺，直接推掉水晶拿下勝利。

這局比賽久誠的幹將莫邪手感還是不錯的，雖然沒有像以前一樣很多次的四劍全中，但是大部分的團戰輸出都是打得比較足的。因為Hero這邊的陣容並沒有東皇那樣的強控，所以說幹將想要純靠自己的預判四劍全中的話是比較難的。但是久誠的幹將還是有熟悉的味道，只要這個幹將在那，對面的馬可波羅根本不敢上前輸出。最終久誠的數據也定格在了8-0-11，輸出佔比還是熟悉的40%，頗有些巔峰的味道了。

這局比賽雖然說Hero久競拿到了勝利，久誠也有不錯的發揮，但是終究對手只是KR的隊伍。Hero目前在KPL內戰之中已經輸掉了兩場，已經沒有任何退路了。他們最後一場比賽的對手是TS，TS在本屆冬冠預選賽當中僅僅輸過一場，可以說是狀態最好的隊伍之一，對於Hero來說還是充滿挑戰的。現在的Hero其實已經有點亂了，久星換掉久酷之後雖然看上去戰績不錯，但是就像解說說的那樣，久星除了那一波閃現躲掉狂鐵的突進之外，其餘並沒有看到太多東西。久星大名單上報的位置是打野，本來應該是和塵夏輪換的，但是這次去到了射手位，明顯是對之前久酷的表現不滿意。接下來對TS的比賽到底該如何排兵布陣，也會是久哲最大的煩惱了。

大家覺得久誠的幹將莫邪有沒有巔峰的味道呢？歡迎在評論區留言交流。我是天黑君，期待下次再見！
