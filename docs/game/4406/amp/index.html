大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友刺激哥。提及《和平精英》新版本的內容，相信大家已經知道了大概的情況。

如：極寒模式與聖誕模式重新上線，除此以外又新增加了新玩法「特種作戰」。需要注意的是，《和平精英》在12.7又放了一個「狠招。」

話題：「吃雞」12.7不停服更新約144M新內容，上線4種玩法，光子太狠了

如圖所示，《和平精英》體驗服在12.7進行了不停服更新，本次更新內容約為144M。在閱讀文章的時候，希望大家可以耐心的看一下《和平精英》上線的4種玩法。

玩法一、元旦限時玩法，海島地圖新增加了，可以供玩家互動的雕像。

玩法二、春節限時玩法-海島迎春，升級版的年夜飯、海島廣場的鞭炮互動、隨機物資年獸（年獸雕像、年獸信號槍）、除夕夜煙花。

玩法三、春節限時玩法-演唱會，更新公告中提及，在遊戲大廳的入口可以找到。

玩法四、元宵限時玩法，海島地圖可體驗與隊友PK元宵大作戰。

截止到發文時間為止，刺激哥並沒有發現公告所提及的相關的遊戲內容。針對這件事，刺激哥還特意詢問了工作人員。得到的回應是：「這些玩法會陸續上線」。

看到這些更新內容，刺激哥一種有兩個想法。

第一個想法：光子太狠了，把《刺激戰場》的玩法，又重新上線了一遍。

第二個想法，光子這也太會「省事」了。直接把《刺激戰場》的創意，又拿出來重做一遍。

雖然刺激哥有這兩個想法，但也是要理解一下開發人員的。雖然，字面上的介紹，僅僅是把現有的玩法，又拿出來重做一遍。

實際上，還有遊戲開發人員，無數個日日夜夜默默的付出，這些都是我們看不到的。希望玩家們在評論的時候，可以手下留情。

除此以外，《和平精英》其他的變化倒不是很大。創意工坊的極寒模式與特種作戰模式，依然是存在的，並沒有因為這次的更新，出現修改或被移除的情況。

這144M更新的內容，主要涉及到的，就是我們所提到的4種新玩法了。

老實講，《和平精英》提前測試幾個月以後的遊戲內容（元旦、春節、元宵），這招真的是太「狠」了。

這個信息量，對於刺激哥來說，真的太大了……緊張激動的心情難以平復。

另外，這次的更新，把海島地圖的聖誕圖標取消了。其次，雨林、雪地這2張地圖，在上個版本更新的時候，刺激哥已經預下載了。

這次更新以後，想要體驗這兩個模式，又需要重新的下載。這點對於體驗服的玩家來說，並不算特別的友好。

最後，要提及的就是遊戲體驗的過程了。刺激哥在海島地圖測試了很多次，並沒有遇到更新公告提及的4個玩法。

所遇到的，全都是「聖誕模式」。預計，我們想要體驗升級版的年夜飯，以及其他的內容，還需要等待《和平精英》為大家開啟許可權才可以。

總得來說，《和平精英》這次更新內容的主題，就是「懷舊」了。光子應該是想借著「情懷」，來找回那些「走失」的老玩家吧。
