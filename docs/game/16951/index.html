LOL: 新版本上路新晉大爹, 勝率高達60%登頂第一, 玩家已經被打服

2019-12-15 2,680條評論 3,958次閱讀 1238人點贊

英雄聯盟在今天更新了9.24版本，其中對許多強勢英雄偶讀進行了削弱，每一次的更新都是一次洗牌，或大或小的區別而已，而這一次也是如此，今天丹姐在看勝率排行的時候，發現上路又一新晉大爹，勝率高達56.03%四捨五入就是60%登頂第一，而且是一位很奇怪的英雄，那麼到底是誰呢？他就是虛空行者卡薩丁。

卡薩丁大家應該都知道，是一位AP刺客英雄，而刺客英雄都是打中路的，上路幾乎玩不了，因為上路的坦克戰士都是刺客的剋星，但是就是這樣一位刺客卻在新版登頂，其實仔細想一下就能夠明白，卡薩丁雖說是一位刺客英雄，但是非常的吃發育，在前期的時候沒有什麼作戰力的，當對面打野入侵野區的時候卡薩丁幫不上什麼忙。

而且現在又是一個前期版本，中路的作用性放大，前期中單帶節奏是非常重要的，卡薩丁已經不適合中路，而上路的話，更加的好發育，很少會打架，因為小龍的作用性非常大，所以在前期主要是在下路進行大亂斗，而上路就是兩人孤單的對線發育，卡薩丁在上路又能夠得到進一步的安穩發育。

而上單笨重的英雄有比較多，在六級之後基本上不可能單殺卡薩丁，甚至還要被壓，發育起來的卡薩丁是非常的恐怖的，上路就是卡薩丁的溫床，卡薩丁就算只是正常發育，只要有裝備了傷害依舊爆表，秒人能力極強，在上單閉關15分鐘甚至20分鐘之後就是一位大爹，誰不服就打誰，玩家已經被打服。
