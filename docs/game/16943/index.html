王者峽谷全面調整，野區大量改動，成吉思汗再次加強！新英雄上線

2019-12-15 728條評論 2,558次閱讀 1528人點贊

大家好！我是小獃獃！王者榮耀體驗服13號全面更新，本次更新內容較大，主要分四段。

  1. 新英雄「蒙犽」正式上線體驗服，喜歡的玩家可以去玩玩！
  2. 王者峽谷全面調整！包括先知主宰/主宰和先知先鋒/主宰先鋒，風暴龍王和野區！
  3. 英雄調整！
  4. 王者模擬戰的大量優化以及改動！

今天主要說一下第二部分和第三部分的內容！關於新英雄的介紹小編之前已經發過，可以打開我的主頁看看！

英雄調整

本次英雄調整總共6位英雄！

  1. 東方曜的大招星削比例由原來的50%下調至40%
  2. 回調安琪拉大招的旋轉速度！
  3. 成吉思汗再次加強！被動技能和一技能的移動速度增加10%！一技能的印記傷害大幅度增加！
  4. 盤古三技能的傷害大幅度下調！
  5. 重做後的嬴政大招被加強！
  6. 魯班大師的基礎生命值增加，三技能的射程增加，放段時長較少至一秒！

以上就是6位英雄的調整內容！具體可以看圖中的數據！

對線調整

對線調整主要分為6部分！

  1. 對抗路的調整：遠程兵已經不在有血包，直接改為經驗兵並且額外增加65點經驗！
  2. 中路河道之靈的優化！
  3. 先知主宰改為2分鐘出現，並且基礎生命值和攻擊力全面下調！
  4. 生宰先鋒的基礎生命值和攻擊力大量消減！擊殺生宰先鋒獲得的經驗減少！
  5. 擊殺暴擊獲得的經驗大量減少！
  6. 擊殺黑暗暴擊獲得的經驗大量減少！

以上就是對抗路和暴擊主宰的調整，詳細可看圖！小編認為這次主要是為了降低前期被等級壓制方沒辦法打出優勢的調整！

野區調整

本次野區調整主要紅BUFF和風暴龍王！並且野怪的刷新時間增加10秒！

紅BUFF的傷害被加強，並且對野怪的傷害翻倍！

其次風暴龍王的調整感電效果的傷害由4%的上調至5%，對非英雄增加至15%！雷電護佑效果全面更改，由原來的50%護盾改成每次脫離戰鬥5秒後獲得自身最大生命值的30%護盾！

詳細可看圖！本次重大調整主要以上幾點！這一次的調整，不知道有哪些英雄又要崛起！

不過小編認為這次成吉思汗加強，可能會成為真正的版本之子！以及風暴龍王的調整，劣勢方想要贏就是難上加難，而對抗路的改動不和原來的版本一樣了嗎？對此小夥伴們有什麼看法呢？還有評論區留言！
