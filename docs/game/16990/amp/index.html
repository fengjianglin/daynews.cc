LOL：S10季前賽最終版本敲定！1位最慘英雄大翻身，ADC再迎壞消息

天天要聞
4 天 ago

最近英雄聯盟9.24版本終於是和玩家們見面了，在新版本中還是有不少值得大家關注的改動的，比如說新英雄厄斐琉斯登陸峽谷，皎月重做上線，版本大爹蓋倫、卡薩丁、賽娜的削弱等等。灰哥本以為9.24將會是S10季前賽的最後一個版本，可沒想到的是在今天（12月14日）LOL設計師是公布了一則讓玩家們激動的消息，那就是在S10賽季開始之前，還會更新一個9.24b版本，那麼究竟在這S10賽季開始前的最後一個版本中，有哪些英雄能搭上起飛的末班車呢？下面就一起來看看吧。

首先第一大改動是關於整個召喚師峽谷的，其實在S10季前賽的元素峽谷上線之後就受到了不少玩家的吐槽，大家都覺得新版元素龍的屬性過於變態，尤其是遠古龍BUFF拿到更是無人可擋，導致如今只要刷小龍就必定會引發團戰，大大加快了遊戲節奏，對於一些需要發育的英雄來說非常不公平。而LOL設計師似乎也注意到了這一點，在9.24b版本中小龍的屬性將會全面下調，其中水龍最慘，而風龍成功幸免於難。

接下來灰哥要說的這位英雄相信大家都不陌生，他曾是比賽和排位中的大熱門，並且還可以三路搖擺，這個英雄就是塞拉斯。雖說塞拉斯曾經風光無限，但在經過設計師的幾番削弱之後如今他的地位十分尷尬，中路和上路均排在T5，勝率只有可憐的45%左右，打野更是沒了蹤跡，如果不看比賽的話你很難相信塞拉斯是一個曾經統治過召喚師峽谷的男人，歷史幾個月就從天堂掉到了地獄，堪稱是近期以來LOL的最慘英雄。

在經歷了好幾個版本的低谷之後，設計師終於是注意到了塞拉斯，並且在9.24b版本中對他進行了大幅修改，下麵灰哥就簡單說說幾個重點部分，首先是被動破敵禁法的充能次數從2次加到了3次；Q技能鎖鏈鞭擊一段傷害降低，二段傷害提高但移除了減速效果；W技能弒君突刺藍耗翻倍傷害增加，並且不再重置普攻，治療效果大幅增強；E技能一段CD降低，二段傷害提升，但移除了原有的魔法護盾，並且不再能重置普攻，EQ連招被移除。

那麼這一波改動究竟是增強還是削弱呢？灰哥覺得毫無疑問是增強，這次設計師的改動方向基本就是把塞拉斯的各種操作機制都給移除了，大大降低了塞拉斯的操作難度，並且簡單粗暴的減少了技能CD和提升了技能傷害，尤其是W技能的新回血機制「治療效果會基於已損失生命值提升最多100%」堪稱變態，有玩家在經過測試之後發現新版塞拉斯中後期點氣定神閑增加法力值上限，然後滿CD情況下4.8秒一個W，回血量非常誇張，無限回血甚至比諾手還要恐怖。

簡而言之塞拉斯的這次加強就是把「機制怪」改成了「數值怪」，只要臉滾鍵盤就能打出爆炸傷害，眾所周知在LOL中數值的加強是最為恐怖的，經常會因為一個技能的加強就徹底改變英雄強度，灰哥認為這位目前版本的最慘英雄在S10賽季將有望重返巔峰。

不過正所謂有人歡喜有人愁，相比較塞拉斯玩家而言ADC玩家們就沒這麼幸運了，其實在近幾個版本中ADC的地位就一直非常低下，而在S10季前賽剛上線的時候ADC們更是遭遇當頭一棒，不僅下路經驗被削，還被一眾戰士和刺客輪流欺負，不少ADC玩家也是坦言：「S10的ADC甚至連輔助都打不過！」不過設計師似乎覺得ADC如今的處境還不夠慘，在9.24b版本中又公布了一個讓ADC玩家們絕望的消息，那就是設計師認為現在的穿甲裝太弱了，新版本中將會進行全面升級！

而設計師的這波操作也是引起了眾多ADC玩家們的不滿，紛紛表示：「拳頭設計師是真的不管遊戲平衡嗎？如今ADC的地位已經這麼慘了，居然還覺得刺客太弱？再這麼改下去乾脆以後也不用ADC了。」如果按照目前這種情況下去的話，可能又會出現之前那種下路兄貴畸形版本。

灰哥覺得ADC玩家們會如此憤怒也不是沒有原因的，看一下目前的中路英雄強度排行榜你就會發現在前10位中單英雄中除了蛇女和亞索之外有8個都是ADC的大爹，而其中受到穿甲裝增益的英雄就有4位，而設計師居然還覺得刺客們不夠強勢，確實有些難以理解。

以上就是S10季前賽最終版本的全部內容了，那麼大家對於9.24b版本的幾大改動怎麼看呢？一起來發表一下你的看法吧。
