DNF：史詩之路刷不起？這些時空石獲取方法你需要了解

2019-12-13 504條評論 1,472次閱讀 78人點贊

DNF12月10日史詩之路活動正式上線了，在這一次的史詩之路中，勇士們不僅可以刷深淵爆史詩，還能用材料兌換各種活動獎勵。不過想要將獎勵換全就必須要刷3個精鍊一次的超星河裂縫，那麼多的精鍊該從何獲取呢，下面就一起來看一看時空石的獲取途徑有哪些吧。

​

魔界和哈林地區的白圖通關後都可獲得固定數量的時空石，每10個時空石可以兌換一個精鍊，勇士們在刷時空石的同時還能順帶著搬磚，可謂是一舉兩得的美差。如果只為了獲取時空石而刷圖的話，90級的血色防線的收益是最高的，單角色在有黑鑽情況下可以獲得138個時空石，相當於4次還多的超星河門票，雖然數量不多但勝在穩定。

普雷每日

​

普雷每日任務的獎勵就是精鍊的時空石隨機禮盒，可以開出8~20個隨機數量的精鍊，按保底的8個計算也相當於刷白圖大半管疲勞的收益了，收益可以說是非常可觀。不過普雷每日也需要一定的門檻才能去打，因此勇士們能打則打能混則混，實在混不了的角色就只能另尋出路了。

災難領域

​

哈林地區的災難領域副本中也可以獲取到精鍊的時空石，不過這一副本更多是為了大號養小號獲取帳綁精鍊而設的，實際的收益也比較看臉，運氣不好的話可能比刷白圖的收益還低。災難領域對角色的傷害要求也不低，勇士們的小號也很難刷的動，唯一的好處就是疲勞值消耗較快，刷圖效率較高，因此沒有太多時間刷白圖的勇士可以嘗試一下。

活動獲取

​

目前遊戲中的周簽到、大富翁以及阿拉德探險記活動都能獲得不少精鍊的時空石，不過除了大富翁活動外其他兩個活動獲取數量都是固定的，因此想通過活動拿到更多的精鍊就得看骰子擲的好不好了。除了這些以外，DNF助手和心悅俱樂部等長期活動也可以獲取一些精鍊，勇士們遊戲之餘也不要忘了去簽到領取哦。

公會生產

​

新版公會系統可以通過守護精靈生產精鍊的時空石，不同等級的精靈能夠生產出的時空石數量也不同，不過生產需要消耗一定的公會硬幣，而且每周只能進行一次，因此這種方法也只能為勇士們額外獲取少量的材料。

那麼目前版本可以獲取時空石的方法就是這些了，勇士們刷到了門票就趕緊去刷史詩之路爆史詩換獎勵吧，祝勇士們都能化身歐皇把把出金！
