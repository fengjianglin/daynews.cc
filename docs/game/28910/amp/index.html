戰士唯一單刷神器被砍！《魔獸世界》懷舊修復DK劍快照BUG

天天要聞
2019-12-21

60版本戰士的終極武器排號，可能有人說到米錘、十字軍劍、眼球斧，甚至風劍、橙錘什麼的，但真正懂行的只有一件，而且來自5人本，就是斯坦索姆DK掉落的「瑞文戴爾之劍」，也稱BUG劍。原因在於其特效，自帶快照效果，賦予了戰士想都不敢想的能力：高級本的單刷!然而就在最近，這個過去走完整個60年代的經典bug，竟然被修復了!

先談談何為快照，這是軟體開發里的一個術語。魔獸效果，是這把劍穿上一刻，根據當前裝備固定起效(增加回血和移動速度)，不受後續換裝影響。例如我穿一身治療500+的裝備，再拿劍，後面把一身治療裝脫掉換成DPS裝，依然享受治療效果的加成。

暴雪的修復聲明

一套治療裝，AH隨便淘點綠，200G能搞定。於是三個能拿雙手劍的職業，都能憑藉它獲得夢幻般的回血效果。

首先是戰士，之前已有戰士單刷厄運東，深淵正義之手的視頻，無論是單挑水元素、還是將軍，都非常給力。我們知道戰士有狂暴天賦的回血，有了DK劍，配合500+治療效果的快照，每一跳漲血都有500多，非常可怕。如果單刷換成巨魔戰士，其自帶10%的戰鬥回血效果，將更加恐怖。

玩成回合制遊戲的厄運東

DK劍的另一意義是決鬥，不用500，只用400+治療，然後和物理職業對砍，對面根本砍不動，每5秒的回血有了永動的感覺，橙錘毀滅皆下品(盜賊可能要被迫掛至傷毒藥了，疊5層可一戰)。BUG終歸是BUG，整個60級沒有比這個更強的PVP武器，贏了也有點勝之不武的感覺。

不僅是戰士，獵人拿到這把劍也有不錯的發揮。對獵人來說，首先DK劍的邪惡光環可以和豹守疊加，同樣是高治療的快照5秒回血，一旦拉開距離，將令對手非常絕望。與戰士不同，獵人幾乎不看近戰武器的DPS，DK劍背著回血就好，遠程傷害不受影響，戰場開旗1V3不是沒有可能。而對於單刷之王來說，無限回血又帶來無限可能。例如完美厄運，拿了DK劍甚至可以硬抗觀察者。作為工具人，有沒有DK劍，風箏BWL小怪，也是兩個效果。總之，這把劍對於獵人的意義，就如瑪拉頓公主無盡黑暗之刃對於法師的意義。

當然，騎士可能更好，因為自身就有加血技能。DK劍是懲戒騎最好的黑科技武器，卡住治療之後，5秒回到500-700，打任何職業都毫無壓力。

雖然暗黑3就有快照的先例，但暴雪老版不修復的原因，是因為DK劍實在太稀有，可能比橙錘更稀有，還沒到嚴重影響遊戲平衡的地步。而現在，DK劍的優點人人皆知，新版玩家的反饋又過多，暴雪是不能無視的。於是最近的熱修中，DK劍已被改動，砍掉其快照效果，就是一般的回血加移速而已。

​

國服玩家已測試，確認效果去除，DK劍淪為徹底的玩具。

最後，我們盤點下其他典型意義的裝備，經過削弱的那些。

無盡黑暗之刃，過去的改版是增加內置CD，限制單時間的觸發次數，但新版暫時並沒有改版，依然神器。

愛德華之手，一件並不怎麼樣的裝備，過去的削弱方案是從下一個法術的瞬發改成冷卻減半。

正義之手，舊版傳言也是加入內置CD，新版也是沒有。
