《魔獸世界》8.3職業狀況-獵人：那個職業，好像條X

天天要聞
2019-12-16

大家好我是喃兌啊內。

獵人在8.3版本已經不是一個慘字就能形容的了。各種原因導致目前獵人三系在ptr都表現很差。這一點也基本是公認的了。Preach Gaming是國外的一個視頻主。

​

每個版本都會做各種副本攻略視頻。在最近一部視頻聊到團本各職業的情況。提到獵人時，他說是最弱遠程DPS。獵人的DPS不起眼，而且對團本也沒別的貢獻。除非你的團本無所謂或者覺得沒問題，帶個獵人。不然他不推薦帶獵人。

​

Preach Gaming有幾十萬粉絲。這個視頻也引起了很多國外獵人玩家討論。在國外discord獵人討論區就有很多人討論這個。

​

而討論區的管理員只加做了個自動回復的腳本。自動回復的意思是，Preach說的沒錯，下個團獵人已經完蛋了。如果你玩生存，那麼轉成貓德吧，如果玩的是獸王射擊，轉成法師吧。新版本開後，討論區和獵人一樣沒用，不如關了算了。很多獵人也在討論8.3獵人完蛋了。

​

Preach已經說了一部分，接著我們再詳細補充一點別的東西。

1，團本表現太差以往大家對獵人團本的印象是開荒利器，靈活移動無損。但是爭霸艾澤拉斯的獵人並不是這樣。最近幾個團本各大公會都選擇了堆疊強勢職業。而獵人都是首先排除的職業。
獵人的傷害不拔尖，功能性為零。拍不了BUFF，給不了團輔，控制能力也不如其他職業。永恆王宮4號之後很多團就不帶獵人了。雙目標，多目標，爆發AOE，壓本體能力都不突出。到了8.3的尼奧羅薩還是老樣子。尼奧羅薩這個本有很多雙目標，分散多目標，還有需要爆發aoe清場的BOSS。獵人在這種場合表現太一般了。

2，團本核心特質

​

之前我有做過一期視頻介紹下個版本的核心特質。簡單說就是團本特質裝，設計師幫你選好了核心特質。但是設計師給獵人選的核心特質，3系有2系是錯的。

​

給獸王選的是死舞，射擊選的是激涌，生存選的是荒野。激涌這個特質，射擊頂多帶一個，甚至很多人都不帶。生存的荒野也只有一部分場合會帶。有很多獵人反饋了這個問題，但是沒得到回復。

3，團本特效武器
8.3的新團本新增了一批特效武器。獵人的特效武器在3號出。

​

效果是，你的自動射擊隨機減獵人某個技能1秒CD。這個特效算是全特效武器最弱效果了。獵人的自動射擊不快，隨機減一個技能。你的假死，龜殼，後跳也會減。

​

其他職業團本拿到特效武器都有一定提升。獵人拿到這個特效武器要看看腐蝕值。考慮要不要凈化掉這個垃圾效果。所以獵人的畢業武器就是尾王恩佐斯掉的槍。然後拼臉看看能不能腐蝕出別的效果。

​

4，暮光炮/爆傷
上面已經提到腐蝕效果了。接著我們聊2個很強的腐蝕效果。爆傷，在ptr出現腐蝕詞條後，我就測試了。獸王的寶寶不吃爆傷，一直到這周都沒改。納沙塔爾的底棲鞋子爆傷，寶寶就不吃。國外也沒啥人討論，所以這個就是設定，不是bug。爆傷這個詞條很難弄到，但是弄到後獸王就是差人一點。射擊和生存沒問題，但是射擊和生存有別的問題。

​

然後是暮光炮，暮光炮是狂暴戰的團本武器特效。上周ptr把團本一部分武器腐蝕特效加入到了腐蝕列表。我之前有一期視頻有詳細說。也就是說你在其他地方其他裝備，也可以拿到團本的腐蝕效果。狂暴戰的暮光炮是一個比較好拿到的腐蝕詞條而且很強。但是全職業全專精，只有獸王和射擊觸發不了暮光炮。

​

近戰可以觸發，法系可以觸發，坦克可以觸發，連奶媽都能觸發。只有獸王和射擊觸發不了，生存可以觸發。因為觸發條件上上周改成了你的法術有幾率。但是獸王和射擊的技能不算這個武器要求的法術。就像獸王和射擊的技能不會被震蕩打斷一樣。

5，同甲互插
下個版本有腐蝕裝備，但是很難弄。最好的解決方法就是同甲互插。比如，盜賊不要精通可以給奶德，奶德不要全能腐蝕可以給盜賊。皮甲板甲布甲團本大米都可以輕鬆同甲互插。但是鎖甲就蛋疼了，鎖甲只有獵人和薩滿。獵人和薩滿現在團本大米都不算特別吃香。

當然還有一些別的問題，我們就不多說了吧。目前獵人在8.3ptr表現很差。等正式服不知道會不會buff。最近有奧山戰場和各種經驗buff。如果你追求進度的話建議換個職業。但是如果你還是喜歡獵人的話，就堅持下去吧。
畢竟強弱只是一個補丁的事，等獵狗變獵爹的一天。
