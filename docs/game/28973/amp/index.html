笑笑排位用羅漢絕食流套路翻車被隊友怒噴，網友：舒服了

天天要聞
2019-12-20

最近解說羅漢又研究出來了一個騷套路上單絕食流賽娜。

當初賽娜進入峽谷時，玩家都是將其用來打ADC的，嚴重違背了設計師將其定義為輔助的初衷。

果然不久後就被削弱，賽娜補刀被動掉落的食屍鬼概率大大降低。

因此賽娜作為輔助，在走下路就缺乏穩定控制和保護AD，遠不如泰坦、女坦這類輔助。

解說羅漢則想出利用出工資裝走上路的賽娜，這種流派就是不補刀，利用工資裝不停消耗敵人偷錢。也能達到快速發育的效果。

德雲色笑笑看了羅漢的視頻後，也想在排位中試一試，就在昨天的一場直播里，他就嘗試了羅漢的絕食流套路。

沒想到卻翻車了。

排位中笑笑按照羅漢的套路出了偷錢工資裝走上路，1級團時笑笑交了一個閃現換來了兩個助攻，團隊開局大好。

但是回到線上後，對線鐵男準備偷錢發育時，敵方盲僧偷了己方紅BUFF知道笑笑沒閃繞過三角草來GANK笑笑含恨死在鐵男的大鎚下。

8分鐘時，盲僧再次GANK上路，笑笑想利用走位躲掉盲僧Q技能，結果失敗，再次被盲僧拿下人頭。

10分鐘時，鐵男直接開大把笑笑拖入領域，上路沒有地形加上賽娜沒有位移只能挨打，被鐵男完成單殺。

中期笑笑打團時作用明顯，給隊伍帶來了好轉，一度讓隊友看到翻盤的希望，但是他的賽娜依然發育不良。

隨後的團戰被敵方亞索風牆擋掉了大招和控制技能，這樣一來，對線也陷入全面崩盤，己方機器人心態爆炸。直接開噴笑笑。

此時網友也紛紛表示，這下舒服了。自己在排位中也遇到過這種看了一些解說視頻後沒練習就去排位坑隊友。

對排位練習騷套路的隊友也是深惡痛絕的，即使是職業選手的笑笑，這下也是翻車了。

各位小夥伴對羅漢絕命流套路怎麼看呢？
