碧藍航線：冬活預熱開啟，肝帝獲大福利，多餘材料有用處了

天天要聞
2019-12-18

每個肥宅都有一個中二魂，嗨大家好，這裡是ACG次元飯君。

在經歷一星期的長草期後，相信不少《碧藍航線》的「禿子」們已經按捺不住對於冬活的期待了吧。日前，碧藍官方一頓聖誕活動宣傳後，我們終於看到了冬活的影子。隨著聖誕活動進行，冬活預熱「祝祭之始」也將同步開啟，僅從這名字判斷，相信玩家就該知道，這或許又是一次關於重櫻的大型活動。

這一點從預熱活動細節也可以看出，在「祝祭之始」到活動正式開啟這段時間裡，重櫻有四位艦娘可獲得一段時間的經驗加成，並且在戰鬥中攜帶她們，還能獲得點數獲得限定頭像框。對於在上個聯動活動中消耗大量物資的玩家而言，這又給他們提供了一段時間恢復期，去籌集物資迎接下一個活動。

當然就目前來看，活動具體消息還未有音訊，唯一知道的就是重櫻的有一位艦娘得到了改造的機會，這也是時隔數月後一次新的改造，這也上很多手握大量改造圖紙的玩家躍躍欲試（改造圖紙都快放起灰了）。整體來說，這次活動間隔相比夏日那種活動三連發來說，還是給鹹魚玩家留夠了緩衝期，現在大家只需要期待新老婆是誰就行了。

不過在次元飯君看來，這次活動其實最有意義的部分，還是在於科研藍圖的處理上。從一期科研開始，相信很多玩家的科研圖紙已經溢出，而這些東西實際上已無任何作用，而這次改動後，玩家可以用自己多餘的圖紙去換取定向圖紙或者是科研裝備碎片。相信那些肝帝大佬的艦隊，又將迎來一輪提升。

作為玩家而言，這次預熱活動還算得上是不錯，但唯一有一個問題的是，那個不知道意義何在的限界突破玩法也伴隨著新活動一併歸來。從第一期開始，次元飯君就完全找不到這個玩法的意義，除了獲取頭像框以外，好像就找不到其它點來突顯它的存在，難不成官方是想讓我們集齊7個頭像框召喚神龍嗎（滑稽）？

總的來說，這次活動還是很值得人期待，不知道各位希望哪一位艦娘能在新活動中登場呢？
