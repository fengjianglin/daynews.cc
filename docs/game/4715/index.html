魔獸世界懷舊服在沒開黑翼三本沒畢業，沒有血藤套增加命中的情況下，團本術士基本就是功能性職業，基本可以划水對自己要求別太高，術士的爆發期在開了祖爾法拉克開始。術士現在再努力打也打不過同裝備的法師和賊，所以現在就專心打醬油吧。手法的話沒有啥手法，團本怪基本上沒有時間等術士上完腐蝕術，痛苦和獻祭等dot再搓煤球。

大多數時候就是一個瞬發腐蝕術然後就一直暗影箭，團長讓你上啥詛咒你就上啥就完事了，拉人、綁好石頭定好小鬼、發糖，適當時候放逐元素怪最後划水，這就是目前術士在MC的主要作用。術士這個天賦，打小本其實看不大出來差距，但是團本就不一樣了，毀滅天賦裡面減少暗影箭讀秒的沒點差距還是很大。

術士團本主要輸出手段就是腐蝕術和暗影箭，別人法師和點了天賦的術士，都是2.5秒一個球出去。你一個技能比別人多0.5秒，一個MC打下來兩個小時左右，你得比別人少多少個煤球？個人認為，術士最強的天賦還是7-21-23的犧牲毀，痛苦點出換血就足夠惡魔點出吞噬惡魔，下本吃女人增加15%的暗影傷害，遇到需要aoe的地方瞬召小鬼吃掉增加20%的火焰傷害，毀滅必須出暴擊傷害加成OT了就喝有限無敵。

正常來說開黑翼有一定基礎裝備的T，術士都沒那麼容易OT，這套天賦除了注意5回和換血的時間，以及未命中注意補暗影灼燒不可能打不上前5，野外帶女人被先手也有一戰之力，一打二實在打不過，也能瞬召胖子犧牲後搞殘對面。惡魔術士要射程沒爆發，要爆發沒射程、沒吸血，刷怪極其傷沒回復，被偷襲就是慘死，真的除開打架什麼都不是根本不適合新手使用。

這個天賦唯一的缺點就是基本只能站樁輸出，沒出夜幕一些需要跑位的時候會有輸出空白。例如MC的老五，但是夜幕也是看臉，連續夜幕又出暴擊，你也不敢狂按暗影箭有得有失，還有一種天賦就是26-0-23的暗毀下本也不錯。出夜幕射程遠自帶10%的暗影技能傷害加成，但是沒有自身血量加成，頻繁換血容易挑戰治療的神經，野外也沒有保障只適合打團本。你們覺得呢？
