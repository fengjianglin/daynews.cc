G2雙C再度換位，Caps將出任隊內AD，為了徹底擊潰FNC？

2019-12-17 739條評論 1,983次閱讀 338人點贊

大家好，我是想稱霸海上卻沒有船的遊戲日報王路飛。近日G2的打野Jankos在直播中說他們戰隊雙C將交換位置，Caps去打ADC，阿P則回來打中單，難道是因為Caps覺得阿P下路太菜了嗎？就讓遊戲日報君帶大家看看是怎麼回事吧。

今年G2雖然未能獲得S賽的冠軍，但在19年連續打敗SKT兩次也證明了他們的實力是有多強。這次G2雙C決定換位置其實不是因為Caps覺得阿P下路菜，據Jankos說是因為Caps在擊敗FNC中單後想去下路打敗歐成，所以才決定換去打AD，這是和FNC有多大仇？

G2戰隊最初出現在LPL玩家視線中是在16年的MSI，結果首次國際大賽便大敗而歸，除了贏了外卡戰隊兩場外，其餘比賽全負。隨後S6、S7世界賽G2未能衝出小組賽，雖然他們在EU的統治力很強，但在國際賽事中卻屢嘗敗果。

18年G2決定重組，在S7發揮拉胯的阿P成為建隊核心，引進Jankos、Wunder等選手後G2在S8的世界賽中直接衝進了4強，而且他們還擊敗了當時的奪冠大熱RNG，這時候G2這支戰隊才被LPL玩家所熟知，但也僅僅是因為他們喜歡口嗨。

直到19年，原FNC中單Cpas離隊加入G2，阿P轉到ADC位置，G2戰隊發生了新的化學反應。在S賽之前，19年都是G2時代，他們拿到EU賽區兩連冠、MSI冠軍和洲際賽冠軍，包攬了除S賽以外的所有冠軍獎盃。

進入S9的他們成為奪冠大熱，小組賽後Cpas一度被大家認為是世界第一中單，而轉到AD為的阿P發揮也十分搶眼，世界賽上的霞令人聞風喪膽。半決賽上再次擊潰SKT後，G2彷彿都已經預定了冠軍之位一般，不過最後還是遺憾輸給了LPL的小鳳凰。

但不管怎麼說，S9後大家對G2戰隊的看法從「喜歡口嗨」變成了「實力強勁」，而G2能在短時間變強絕對跟Caps的加入離不開關係。這次中下再次決定交換位置不知道會不會又起什麼化學反應，或許明年Caps就將成為下一個「Czi」。

但其實很多人都不知道，當初阿P挖來Cpas是屬於違規挖人，雖然Caps也自願來G2，但是FNC卻強烈譴責了G2，從此過後這兩支戰隊也算是結了仇。Caps這次轉到ADC位置可能也是為了徹底擊潰FNC，證明自己當初的選擇是正確的。

大家對G2戰隊雙C換位有什麼看法呢？歡迎踴躍留言參與討論。
