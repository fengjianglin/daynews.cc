UZI成功超越迪麗熱巴，上升到第四名！LOL玩家集體暴走引發熱議

天天要聞
2019-12-12

​電子競技發展到今天，其實的很多領軍人物及選手的人氣甚至已經超越了很多演藝圈的明星，在去年的微博之夜中，英雄聯盟的很多大事件就入選了當晚的頒獎典禮，而在今年的微博之夜年度人物的投票中，裡面有6個電子競技選手。

其中包括UZI、The Shy、廠長、JKL等人，在這個活動剛剛開始的時候UZI的票數就一路高漲，直接衝進了前10，超越了薛之謙和蔡徐坤等人物。

而在這之後，LOL玩家開始集體暴走，為UZI瘋狂投票，UZI是陸續超越了華晨宇和李現進入前五，而就在剛才不久，UZI是成功超越了迪麗熱巴，排名上升到第四位，現在UZI的票數是接近70萬，在他前面的三個人分別是肖戰、王一博以及朱一龍。

由此可見電子競技選手的影響力有多大，很多觀眾都在紛紛在各種論壇上幫助UZI打Call，與此同時The Shy的票數也悄無聲息地上升到了第七位，廠長則排名第12，JKL排名第十六位，之後是宋義進、史森明以及Doinb等等選手也有上榜。

從現在的形勢來看，UZI是有機會超越朱一龍的，但是機會沒有很大，至於前兩名的明星，票數過於高估計很難超越了，相信很多觀眾都知道這兩個明星拍過的陳情令，其中主演之一的王一博曾經多次參加過LOL的活動，也受到了很多召喚師的喜愛。

UZI目前還有被迪麗熱巴反超的風險，不過只要LOL玩家能夠足夠給力不斷地投票的話，以LOL召喚師的玩家基數來說，應該可以拿到一個非常好的排名。

雖然UZI今年的成績沒有那麼好，但是這個選手作為LPL的領軍人物之一依然是很多老電競玩家心目中最喜歡的一個選手。
