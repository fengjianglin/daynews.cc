懷舊服節奏整體加速《魔獸世界》戰場降臨大元帥爭奪戰一觸即發

2019-12-03 0條評論 15次閱讀 0人點贊

摘要

可以說伺服器選擇一直是所有魔獸玩家進入遊戲後的最關心的問題，原因也是很簡答的。懷舊服的精髓是沒有飛行坐騎，PVP裝備和PVE裝備沒有分開，這就意味著不用打本的閑暇可以野外殺人，野外殺人在一起可是佔據了大部分人的空閑時間。

可以說伺服器選擇一直是所有魔獸玩家進入遊戲後的最關心的問題，原因也是很簡答的。懷舊服的精髓是沒有飛行坐騎，PVP裝備和PVE裝備沒有分開，這就意味著不用打本的閑暇可以野外殺人，野外殺人在一起可是佔據了大部分人的空閑時間。當然如果你的伺服器本陣營弱勢的話那麼野外的日子絕對不會多好受，同理優勢的話幹什麼事情都是順風順水。那麼在戰場即將開啟的日子，在12月6日懷舊服又開新服，時至今日伺服器已經突破了70大關，相信還會有更多的朋友陸陸續續的加入到懷舊服之中，而新的伺服器將是你們最好的選擇。

​

對於懷舊服的PVP可以簡單的總結一下——團戰最強是冰法，野外最強是盜賊。站樁最狠是火毀，有奶配合是戰士。將來想玩PVE的朋友，戰牧應該是你們最好的選擇，提前找好團隊支持，前期也不會太苦，raid一開始馬上紅到發紫，搶手的不行，最重要的是沒有什麼人和你搶裝備，只有你去搶別人裝備的情況。畢竟輸出戰士用皮甲這應該是所有玩家公認的觀點。如果你想玩PVP，在懷舊服中首選獵人，其次小德，因為懷舊服主要是戰場，奧山搶人頭，戰歌和阿拉希抗旗守點這兩個職業都很有優勢，反觀盜賊在奧山大團中根本不佔優勢，單獨的去偷點偷人對於沒有人頭機制的老奧山來說，其實並沒有太大的戰術意義。

​

作為玩家來說屈尊於戰士牧師的畢竟是少數，所有人都有一顆輸出的心，所以大多是的朋友會選擇法師，盜賊。這是兩個絕對需要操作的職業，這兩個職業擁有強大的逃生能力。傷害不低控制又足，特別在當年那個沒有徽章的年代，盜賊一套秒人是經常發生的事情，也就只有法師能夠用冰箱去規避那個爆炸傷害了。盜賊操作難度中上等，上手還是蠻簡單的，但是想要玩的精通就比較難了。同時這兩個職業也非常適合打錢，所以自然而然玩的人也就多了。至於三大輸出中的獵，玩家群體也不少，但是比起大頭賊法來說確實少了很多的競爭力。

​

部落奧山一般都比聯盟厲害，這也和一個職業有著直接的關係，那就是——薩滿。成型後的電薩是個恐怖的存在，傷害非常恐怖，並不依賴大招，不過想成型也不太容易，安其拉開放T2.5之前基本不太好混裝備，因為副本中沒有位置，MC這個副本也就進去插個圖騰然後撿別人不要的垃圾。雖然天生一副小短腿沒位移，但是元素薩在奧山的統治力絕對是驚人的。比起巨魔來說，肯定獸人薩滿PVP更強大一些，雖然種族也不會對傷害產生太大的影響。至於戰場究竟會怎麼發展目前還是個未知數，但是看到懷舊服玩家的熱情我們自然而然的對其充滿著希望。

​

最後總結一下，懷舊服的戰場一直都是一個比較特殊的存在，因為在這裡很難看到正式服絕對公平的設定，但正是因為這些不公平，我們對戰場的開放才如此嚮往，各位勇士們準備好守護艾澤拉斯了嗎？
