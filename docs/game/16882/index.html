KPL樊叔賽評：從1:4到4:1，AG超玩會做了什麼改變

2019-12-15 1,166條評論 4,946次閱讀 1785人點贊

12月14日，KPL秋季賽也迎來了最後的總決賽，經過5輪角逐，最終分出勝負，AG超玩會以4:1較為輕鬆的戰勝了QGhappy，最終圓夢銀龍杯。從這一場比賽中看，兩隊似乎相差較大，但其實在11月22日勝者組的比賽中QGhappy同樣以4:1的比分輕鬆戰勝了AG超玩會，那麼在這短短半個多月的時間AG超玩會究竟做了什麼改變？我們從雙方對戰的第一局詳細分析。

BP：AG超玩會前期快攻，QGhappy中後期保射手

第一局雙方可用英雄最多，拿出的陣容最能體現兩隊的特色。QGhappy拿出了一套中後期運營的保射手體系，前期防守，拖到公孫離裝備成型，依託陣容非常厚實的前排打團，AG超玩會則拿出了進攻性較強的推塔體系，前期要靠裴擒虎和周瑜的強勢打出優勢，不斷擴大優勢滾起經濟雪球，利用經濟優勢打團。

這套陣容似曾相識，在11月22日AG超玩會對陣QGhappy的第一場比賽中， 雙方也選出類似的陣容。特別是AG超玩會前三手完全相同，但不同的是雙方在後兩手的選擇，11月22日的比賽中，AG超玩會選擇先拿沈夢溪，希望用沈夢溪的遠程消耗克制射手，讓QGhappy的團戰輸出環境更差，最後一手補出大喬輔助，雖然陣容中沒有大喬常見的搭檔，但就是想利用大喬不錯的後期能力彌補陣容後期乏力的缺點，AG超玩會這套陣容看上去更加四平八穩。

比較兩局bp，雖然陣容看上去相似，但在總決賽上AG超玩會明顯吸取了上次失利的教訓，中路選擇上放棄克制對方陣容的沈夢溪， 而是拿出更加適合自身陣容的周瑜，輔助選擇上放棄後期能力強的大喬，拿到前中期發力的蔡文姬。相比11月22日的陣容，AG超玩會不再想拿出一套前中後期都可以打的陣容，而是更加突出前中期的進攻性，並且ban掉曜，將QGhappy陣容進一步向中後期擠壓，這就一來AG超玩會的強勢期正好碰上了QGhappy的弱勢期。

賽況：AG超玩學會了QGhappy式運營

由於BP設計，這一次的藍buff入侵QGhappy沒有任何防守的餘地，AG超玩會成功拿到藍buff，並且通過提前卡位拿到了2分鐘的暴君，AG超玩會拿到了前期的小優勢。按道理說裴擒虎+蔡文姬+周瑜是一套前期非常強的打團組合，再加上AG超玩會是一支非常喜歡打架的隊伍，按照他們之前的風格，拿到這種陣容和前期優勢後一定會和對方瘋狂打團，通過人頭擴大優勢，然而總決賽中AG超玩會並沒有。

AG超玩會的作戰思路完全不在打團，他們的眼中只有塔。比賽中多次可以看到AG超玩會即使能開的團，他們也選擇不開，退後拉扯，等到對方交完技能後捲土重來，接著利用周瑜和裴擒虎技能消耗防禦塔的血線。7分10秒，雙方人頭比0:0，但AG超玩會磨掉了QGhappy三座防禦塔。

AG超玩會這套打法思路非常聰明，但在之前的比賽中他們的風格完全不是如此。在11月22日第二局的比賽中AG超玩會曾取得了前期小幅優勢，但就是無法進一步擴大優勢，這是因為他們總是在急於打團，而QGhappy堪稱避戰大師，他們利用自身出色轉線讓AG超玩會每次開團全部撲空，在AG超玩會集結打團的這段時間，QGhappyC位無壓力發育，以逸待勞，AG超玩會不知不覺就陷入了劣勢。

然而在12月14日的這場比賽中，AG超玩會放棄了他們最為熟悉的團戰思路，改為運營推塔，直到7分30秒才拿到第一滴血。這個時候QGhappy成了疲於奔命的一方，他們沒辦法主動轉線換線，只能被動的跟著AG超玩會的進攻節奏在塔下防守，自身C位完全沒有發育空間。

之後的比賽中AG超玩會沒有給到QGhappy一點點機會，全程避戰，推塔拿暴君滾野區，將經濟差滾到將近1萬時強上高地，一波推平。AG超玩會整場比賽只打兩波團戰，全場也只爆發了6個人頭。如果把隊名遮住，AG超玩會這種贏比賽的方式像極了QGhappy。

從這場比賽中我們可以看出AG超玩會在這半個多月的時間進步神速，他們的BP思路更加清晰，進一步強化自身優勢，專攻對方弱勢，而不是在猶猶豫豫的騎牆。另外就是他們的學習能力，他們非常快速的學會了對方最擅長的套路，靈活的轉換了自身比賽風格，以彼之道還施彼身，用QGhappy最擅長方式擊敗了他們，極其靈性。

不是QGhappy不夠強，而是AG超玩會實在進步了太多。
