知情人爆料：繼Ray之後，EDG又有一人恐將離隊，S10可能真要涼了

2019-12-10 0條評論 16次閱讀 0人點贊

摘要

LPL轉會期將於19號正式結束，目前幾條“大魚”的去向也已經明朗，卡薩已經官宣去了TES，JKL大概率加盟SN，目前只差一個官宣，khan人已經在FPX基地，唯一還沒消息的就是兮夜了，就連靠譜點的爆料都沒有，讓人很好奇。而在近日，有知情人爆料了一則關於EDG的消息。

LPL轉會期將於19號正式結束，目前幾條「大魚」的去向也已經明朗，卡薩已經官宣去了TES，JKL大概率加盟SN，目前只差一個官宣，khan人已經在FPX基地，唯一還沒消息的就是兮夜了，就連靠譜點的爆料都沒有，讓人很好奇。而在近日，有知情人爆料了一則關於EDG的消息。

這則消息的原話是「一名國產AD轉會到VG俱樂部」，而在底下的留言中，大家都將矛頭轉向了EDG的iboy。首先從字面看，那就是「轉會」，也就是有合同在身的，那些自由人不算。再加上是國產選手，符合條件的也就iboy。

而且在很早之前就有傳聞iboy和小胡會一起加盟VG，因為這兩人關係特別好。目前小胡已經在早些時候加盟了VG，所以iboy很有可能也會加盟。最主要的是最近一段時間，iboy和小胡開始了雙排。而meiko則是和hope雙排，再加上iboy和meiko搭檔的這幾個賽季並沒有產生化學反應。所以iboy很有可能真的轉會VG戰隊。

如果iboy真的選擇轉會的話，那麼EDG在轉會市場的表現真的有點糟糕了，加上上單Ray已經離隊，EDG不僅一無所獲，自家陣容還走了兩人。之前有傳聞說TES打野熊熊可能會轉會EDG，結果人家去了LNG戰隊。

這樣一來的話，廠長直播時說的EDG在S10會拿到好成績的夢想，也就徹底破滅了。估計廠長心裡也很鬱悶，都跟粉絲說好S10賽季EDG要起飛了，結果一個轉會期搞成這樣。

而這次轉會市場不僅是EDG一無所獲，那些老牌戰隊都差不多。比如IG，We以及RNG，因為各種原因，導致他們根本買不到選手，除了錢以外，壓力也是很大一個因素。而LPL所有戰隊的新陣容，等19號轉會期結束就都知道了。
