廠長教練生涯首秀獲勝，EDG新人表現各異，狀元上單成驚喜

天天要聞
2019-12-22

德杯已經進入第二天，今天杭州分會場的首場比賽，是EDG對陣來自LDL的GD戰隊。EDG在這個休賽期最大的話題，應該就是Clearlove正式成為主教練了吧。而今天的這場比賽，就是廠長作為教練的首秀。比賽BP開始後，幾乎所有的鏡頭都對準了廠長，而明凱教練的首秀狀態似乎還不錯。

這場比賽EDG在藍色方，選出了船長、皇子、辛德拉、女槍和女坦。GD則是選出了鐵男、蜘蛛、女刀、霞和錘石。

​比賽前期蜘蛛沒有做到任何事情，到了7分鐘Xinyi的皇子來到下路，EDG這邊下路率先發難，霞遭到擊殺，Hope的女槍拿到一血。隨後EDG直接吃掉了第一條小龍，並且掌握了比賽節奏。10分鐘Xinyi收下峽谷先鋒，直接放到中路吃塔皮。EDG不斷滾雪球的同時，GD這邊的野輔也始終在遊走找機會。12分鐘終於在野區埋伏到了Scout，成功將其擊殺，GD順勢收下第二條小龍。

​在這波後EDG節奏有些中斷，反倒是GD這邊的霞14分鐘在下路單推掉了一血塔，因為他們的經濟也領先了EDG有近兩千。17分鐘，GD又強行拿下了第二條峽谷先鋒，但EDG卻趁機把中路一塔推掉，並拿下了第三條小龍。21分鐘，GD推掉EDG的中一塔，經濟領先了兩千多，而雙人人頭比這是依舊是1比1。23分鐘，雙方終於在中路打了一波團戰，EDG打成3換2。

26分鐘，EDG擺開架勢後忽然在中路先手，鐵男撤退不及遭到擊殺，隨後GD中路二塔也被破。28分鐘，EDG收下龍魂，GD這邊則是選擇打男爵逼團，EDG過來接團後打出了0換2，隨後男爵由EDG拿下。

32分鐘，EDG五人在下路抱團推掉了高地，在高地上擊殺了GD四人後，EDG一波結束了比賽，收下了首場小組賽勝利，而廠長也拿到了教練生涯的開門紅。

EDG的陣容相比上賽季變化還是比較大的，上野和AD都換了人。Xinyi沒有什麼太出彩的鏡頭出現，但是整體節奏把控還不錯。而下路Hope相比之前出場的表現，並沒有多大的進步，這場比賽EDG下路也是劣勢，他也落後了七十個補刀。不過EDG上單新人Xiaoxiang卻是意外驚喜，作為狀元的他這場比賽表現出色，使用船長打出6-0的戰績和全場最高傷害，最終拿下了MVP。
