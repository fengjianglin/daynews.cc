和平精英：“獵人晚宴”軍需上架，最後一款沒有皮膚的步槍消失了

2019-12-20 3,074條評論 4,757次閱讀 2907人點贊

大家好，歡迎來到由小魚乾開講的《和平精英新鮮事速報》，昨天和平精英的正式服進行了大版本更新，但小魚乾非常想體驗的特種作戰模式，並沒有第一時間上線，還是有些許失望的，所以只好先去體驗「極寒模式」了。

這個極寒模式給小魚乾的第一感覺就是：冷，尤其是暴風雪來臨時，屏幕上的血霧效果會遮擋視野，如果人物不及時生火取暖，體溫條會飛快地往下掉，下降到一定程度，血量也會快速下降。

光子為了遊戲中的人物角色，在冰天雪地中也能保暖，必定要做出一些冬季的軍需。果然在今天晚間，光子發布了「獵人晚宴」軍需的預告，這個系列的軍需，將會在12月20日零點上架軍需。

新軍需包括3款套裝、5個武器皮膚、1個降落傘皮膚、全套的頭盔皮膚和背包皮膚，下面就讓我們一起來看一看吧！

首先是頂級的套裝「荒野獵神」，這款荒野獵神套裝，應該就是和平精英第三款熊形套裝了，第一款和第二款是萌熊和酷熊套裝，它們同樣都是有熊頭套的。

但這款「熊裝」，可比萌熊酷熊要兇狠多了，這個荒野獵神的熊頭，有鋒利的獠牙，眼神也是非常兇狠。一對比起來，萌熊酷熊就是個「吉祥物」！

官方對這款荒野獵神套裝的描述是，這個「巨熊」造型的傳說獵人，能在惡劣環境中狩獵任何目標，大家稱之為「荒野獵神」。看這個品質，小魚乾初步估計，也需要1000才能抽出來。

另外的兩款分別是「幻世獵手」套裝和「森林獵手」套裝。「幻世獵手」套裝，看起來和傳統的刺客裝非常相似，和平精英之前也推出過類似的服裝，可以說是沒什麼新意了。而森林獵手這款，則是西部牛仔的風格，第一眼看也沒有特別亮眼的地方。

接下來就是武器皮膚系列了，本次「荒野獵神」系列軍需，一共是5款武器皮膚，包括UMP49、VSS、G36C、AKM、AWM，值得一提的是，G36C這把步槍，是首次推出皮膚！也就是說，步槍系列中最後一款沒有皮膚的步槍，也消失了，在本次軍需推出之後，所有步槍都是擁有皮膚的了！

G36C能推出皮膚，小魚乾還是挺開心的，自己在雪地圖，一般都是選擇G36C，感覺比M416更好用些，現在終於推出皮膚了，估計光子也是為了玩家們能在雪地模式中，在使用G36C的時候也能使用皮膚吧！

這5款皮膚，槍身上都包裹有毛皮，還有灰白色的塗鴉，擁有良好的偽裝效果，不算亮眼，總的來說還是看個人喜歡吧，實際上大家主要追求的，應該就是AKM和AWM這兩款武器的皮膚了。

而背包和頭盔系列，同樣是3種等級的皮膚都有，三級頭的造型還是不錯的，兇狠的獠牙直接包住了半邊面罩，戴上它能讓敵人感到畏懼，而背包則是皮革樣式，看起來還是比較現代，「四級包」這次的設計，還是能讓一些玩家買賬的。

光子也給這次的軍需起了一個比較霸氣的名字：「獵人晚宴」，這個系列的軍需將會在12月20日零點上架，小魚乾保守估計，本次的軍需也要一千打底。光子這次為了遊戲人物「保暖」，還是下了一番功夫的。

​那麼以上就是本期《和平精英新鮮事速報》的全部內容了，更多「吃雞」的最新內容，還請繼續關註：小魚乾遊戲！
