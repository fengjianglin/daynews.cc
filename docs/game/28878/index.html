《無主之地3》DLC評測：充實的內容，堪稱優秀的擴展

2019-12-21 65條評論 1,733次閱讀 52人點贊

【評測前言】

　　在經歷了一系列系統和玩法上的更新後，《無主之地3》終於在年底迎來了第一個大型DLC。和之前的「血腥收穫節」以及「突襲馬里旺」不同，《莫西的帥傑作搶奪大作戰》具有一條完整的劇情線，相當於一顆星球體量的地圖以及全新的支線、挑戰和裝備掉落。其性質類似於二代的「托格先生的大屠殺」等等DLC。而這一次的故事，玩家要再次幫助莫西去解決他的前任之一，二代的大反派帥哥傑克所留下的麻煩。

　　帥哥傑克的形象塑造，稱得上是二代的神來之筆。他在系列中的地位、人氣，就像弗利薩之於龍珠;小丑之於DC漫畫。雖然新DLC中無法再次與他正面對抗，但是從留在「帥傑作」中的其他角色和敵人、設施中，玩家能以不同的角度來了解這位人氣反派以及相關的故事，整個流程的視點和體驗都是相當新穎和有趣的。

【相當紮實的遊戲內容】

　　在DLC中玩家要前往新的地區進行探索，目的地就是名為「帥傑作」的賭場。這是帥哥傑克和酒吧老闆娘莫西熱戀時，根據她的創意利用一座空間站打造成出來的「賭場星球」。類似於本篇中的潘多拉、普羅米西亞等星球，「帥傑作」里也分了數個區域供玩家冒險，其中分布著大大小小的支線、挑戰和敵人的據點，仍然有著非常豐富的內容可供玩家體驗。

　　對比本篇里各個星球內部相對統一的地圖風格，「帥傑作」中幾個區域的場景設計則有著非常明顯的差異。例如「全境賭場」整個區域以深藍這樣的暗冷色調為主，搭配色彩艷麗的霓虹燈光和塗鴉，具有科技感的亥伯龍機器人遊走在繁華的廢墟之中，凸顯出了濃濃的賽博朋克味兒。而像「垃圾蘭蒂斯」則走的是廢土風格，有別於潘多拉上擁有大片的荒原。「垃圾蘭蒂斯」的面積要小很多，更像是個大型的垃圾填埋場。到處都是廢料、集裝箱堆起來的小丘，殘破的平房搭建在其中。

　　▲遊戲的場景構造還是非常多變、複雜的

　　搭配這些新區域的自然是全新的任務、挑戰和彩蛋。戰友挑戰方面，像「托格的營銷失誤」需要玩家憑藉眼力尋找射擊目標，而「市長的超帥裝扮」則仍是要玩家尋找對應的精英怪擊殺。另外，每個區域中也包含了數個風格迥異的支線任務，支線任務中會有不少有趣的新人物出現，劇情和流程上依舊極盡惡搞和無厘頭之能。這一部分內容的體量非常充實，豐富程度不亞於本篇。

　　▲在「帥傑作」中除了有不少帥哥傑克的雕像海報外，也有一些可以收集的全息影像，玩家能藉此來了解帥哥傑克的性格和事迹。

　　新的內容並不單單只有地圖及任務而已。《莫西的帥傑作搶奪大作戰》提供了很多新式的武器、職業模組以及外觀裝飾等。全新的職業模組必然會對四個職業的BULID產生非常大的影響。就拿贊恩的新模組「見證死亡」來說，它的效果可以讓武器擊中敵人時，有幾率自動激活所有殺戮技能，並獲得效果加成。新模組的觸發幾率不高但條件較寬，分身、無人機等傷害都包含在內，通過高射速，多彈片的武器或者一些天賦技能都能讓贊恩較好地保持殺戮技能。在穩定提升贊恩傷害收益的同時，贊恩的技能加點、武器搭配思路也隨之變化。作為遊戲發售初期較為疲軟的一個職業，贊恩最近更新的模組一個比一個收益高，也漸漸擺脫了之前的弱勢地位。通過新的裝備、機制來改善職業間的平衡和玩法體驗，我個人認為是非常良性的改動方式。這比誰強削誰，一刀砍沒的方式要來得人性化和靈活許多。 除此之外，特殊怪和世界掉落中都增加了不少「新」的武器，其中有比較多的是莫西系列武器，像比如霰彈槍「慢手」等等。

　　這些新的內容加起來，使得DLC的流程就有7,8個小時左右。而其中關於新模組、新武器的收集、研究則進一步擴展了遊戲的時長。和之前的作品一樣，對於《無主之地3》的第一款大型DLC而言，Gearbox可以說在內容上是展現了足夠的誠意。

【穩定發揮的劇情流程】

　　這次的DLC，玩家達到13級並解鎖了避難所3號的躍遷功能後，就能從莫西酒吧外的帥哥傑克海報處接下任務。DLC的劇情也和主線的聯繫並不大，並且敵人的等級、掉落也依舊是動態機制，所以玩家可以自由選擇遊玩的時機，包括以混亂模式進入。

▲從馬庫斯到帥哥傑克，莫西的前任們來頭似乎都不小。

　　帥哥傑剋死後，「帥傑作」賭場也處於廢棄狀態，但賭場內的設施依然受他遺留的AI繼續運營著，莫西打算組織一票人馬，洗劫這個被遺忘的太空站賭場。再次作為工具人的玩家會在遊戲中招募到幾個性格各異的角色一路與反派靚仔哥為首的亥伯龍部隊以及瘋狂的賭徒們作戰。故事流程大致上就是從各個區域尋找志同道合的夥伴，最後與反派進行決戰的傳統套路模式。

　　雖不算新穎，但勝在幾名角色人物的形象塑造依然出彩，能給玩家留下深刻的印象。例如，與帥哥傑克外貌極為相似的蒂莫奇，遊戲通過他的身份以及與莫西的關係，從側面反映了帥哥傑克的一些過往經歷。還有將自己身體進行改造、崇尚火焰戰鬥的安貝兒，光頭造型配上火辣的身材，和「全境賭場」一樣都是滿滿的賽博朋克味兒。這些形象鮮明的人物不僅僅是在主線里，支線中玩家同樣能碰到不少奇奇怪怪的傢伙，像賭命不夠還要賭靈魂的瘋狂賭徒、喜愛酗酒卻活力十足的音樂家、互相競爭但又在意對方的賭場姐妹等等，形形色色的人物和故事豐滿了DLC的故事流程，同時讓整個太空站賭場變得鮮活、有趣了起來。

　　和本篇對比，這次DLC中各類機械敵人出現的數量較多，腐蝕屬性算是成為了VIP。機械類敵人的斷肢效果也更為明顯，將它們的下半身擊破後，它們的雙手仍會繼續持槍朝玩家射擊，而新加的幾個Boss中基本都是常規類型的設計，倒沒有出現像食人獸、大樹這類偏機制應對型的BOSS戰，當然要提的一點瑕疵就是如本篇戰鬥時的一些「光效污染」依舊存在。

　▲小吵鬧大軍

【評測總結】

　　作為《無主之地3》的第一個大型DLC，《莫西的帥傑作搶奪大作戰》的質量和內容體積都基本令人滿意，新加入的裝備、模組也有一定的深度供玩家研究，《無主之地》標誌性的無厘頭故事風格也原汁原味地體現在了 DLC的格式任務當中。另外，相對自由的進入條件以及動態機制對於新老玩家都非常友好和方便。
