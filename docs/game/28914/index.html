Knight9衝擊韓服第一：讓Karsa準備個小號，這次勢在必得

2019-12-21 226條評論 1,875次閱讀 494人點贊

大家好，我是愛玩遊戲的小蘿醬，每天分享最新的遊戲資訊和有趣的遊戲內容，嗯哼~

自從Karsa到了TES的基地後，一對中野組合在韓服開啟了瘋狂上分模式，Knight9自然是其中的大腿，經常在獲勝的局中斬獲MVP，Karsa在感受到如此強勁的Carry型中單後，一轉身拋棄了Doinb，變成了左手的「寶貝」。看兩個人有沒有配合，雙排的戰績最為直觀，剛剛聯手就豪取13連勝，似乎他們倆天生自帶默契。之後兩人贏多輸少，Knight9也是在穩定的上分，目前排名韓服第四，Karsa也即將衝進前100。

不知道出於什麼原因，二人的雙排在四天前停止了，難道真的只是在互相磨練磨練，練習新的風格和戰術打法嗎？從英雄選擇上來看，Knight9不像排在他之前的三名選手那樣，基本固定使用幾個英雄，要知道一直使用相同的英雄，就會越來越熟練，上分也就相對容易一些。所以不少觀眾認為，Knight9花式使用英雄，更多的是為了和Karsa練習不同的中野組合，看看能打出怎樣的效果，沖分只是次要的。

Knight9畢竟是上過韓服第一的，莫非現在和Karsa練習的差不多了，韓服第一對他沒什麼吸引力所以就此停手了？當然不是這樣，Knight9在直播中表示，這次韓服第一是肯定要上的，並且勢在必得。由於自己和Karsa目前的分已經很高了，遇到的都是那些高分路人以及職業選手，上起分來的速度明顯慢了許多。所以Knight9決定，讓Karsa準備個小號和自己雙排，這樣更加穩定。

根據Knight9和Karsa雙排的勝率來看，韓服第一應該只需要時間，最主要是看有沒有人和他搶。讓我們來看看韓服排行榜前五，基本可以分為兩檔，第一和第二比起後三名高出100多分，這100多分衝起來並不容易，好在榜首Cuzz已經1個多月沒上遊戲了，可能是轉到SKT有很多事情，現在還騰不出時間打Rank吧。

第二是Dopa，這個多次霸榜國韓雙服第一的玩家，正如之前所說，發條是他的上分利器，其次是招牌卡牌。不過Dopa更喜歡在賽季末開始發力，近期只玩了兩天還都是大亂斗，不愧是theshy的好朋友，所以Dopa對Knight9的威脅並不算大。

第三是Kanavi，此次重回JDG立即作為德杯首發打野出戰，現在正在使用自己的小號和Yaogao在韓服雙排練習配合，很少出現在Doinb的直播間中了。不得不說這名選手rank是真的強，在前期非常能帶節奏，戰績也非常華麗，經常拿下MVP。Kanavi在夏季賽的幾次亮相，也都展現出了不俗的實力，如果和戰隊磨合得夠好，應該會成為一名強力的打野。

所以以目前的情況來看，Knight9上韓服第一的壓力並不大，多數都來自於排在他之後的玩家，況且即使這三個人加入到這場戰鬥中，Knight9也不怕，畢竟有Karsa小號的保護。最後祝願Knight9能在Karsa的幫助下，早日登頂韓服，並將這股氣勢帶到TES戰隊，帶到LPL的賽場。
