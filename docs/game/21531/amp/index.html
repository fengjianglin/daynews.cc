遊戲王最強三邪神，是否能與傳說中的三幻神抗衡？

天天要聞
2019-12-19

三邪神——在《遊戲王R》中，由貝卡斯·J·克羅佛多製造，擁有可以剋制三幻神力量的3張卡。後被天馬夜行據有，將其大量複製給下屬的決鬥者使用。

3張「邪神」卡都不能特殊召喚，必須使用3隻怪獸作為祭品才能通常召喚，作為與三幻神相對的存在，他們也曾在漫畫中佔據大量的戲份

這三張卡牌分別是「邪神恐懼之源」、「邪神 抹滅者」、「邪神 神之化身」。

下面萌主就為大家介紹一下三邪神，看看他們是否有能力和傳說中的三幻神抗衡？

「邪神 恐懼之源」

暗屬性惡魔族效果怪獸，攻防都是4000。

卡牌效果：這張卡不能特殊召喚。把自己場上3隻怪獸解放的場合才能通常召喚。只要這張卡在怪獸區域存在，這張卡以外的場上的怪獸的攻擊力·守備力變成一半。

在漫畫中，這張卡需要3隻怪獸作為祭品才能通常召喚。陷阱對此卡無效，魔法·怪獸效果僅維持1回合，此卡的控制權不能被改變。此卡在對方回合特殊召喚的場合，對方的攻擊·效果對象轉變為此卡。只要此卡在場上表側表示存在，雙方場上除此卡以外的怪獸的攻擊·守備力減半。

與之相對應的神之卡為「歐貝利斯克的巨神兵」。

「邪神 抹滅者」

暗屬性惡魔族效果怪獸，攻防為？。

卡牌效果：這張卡不能特殊召喚。把自己場上3隻怪獸解放的場合才能通常召喚。這張卡的攻擊力·守備力變成對方場上的卡數量×1000。自己主要階段才能發動。這張卡破壞。這張卡被破壞送去墓地的場合發動。場上的卡全部破壞。

在漫畫中，這張卡需要3隻怪獸作為祭品才能通常召喚。陷阱對此卡無效，魔法·怪獸效果僅維持1回合，此卡的控制權不能被改變。此卡在對方回合特殊召喚的場合，對方的攻擊·效果對象轉變為此卡。此卡的攻擊·守備力等於對方場上存在的卡的數量乘以1000。當此卡破壞時，將場上的卡全部破壞。可以選擇將我方場上的此卡解放。

與之相對應的神之卡為「歐西里斯的天空龍」。

「邪神 神之化身」

暗屬性惡魔族效果怪獸，攻防為？。

卡牌效果：這張卡不能特殊召喚。把自己場上3隻怪獸解放的場合才能通常召喚。這張卡召喚成功的場合發動。用對方回合計算的2回合內，對方不能把魔法·陷阱卡發動。這張卡的攻擊力·守備力變成「邪神 神之化身」以外的場上的攻擊力最高的怪獸的攻擊力+100的數值。

在漫畫中，這張卡需要3隻怪獸作為祭品才能通常召喚。陷阱對此卡無效，魔法·怪獸效果僅維持1回合，此卡的控制權不能被改變。此卡在對方回合特殊召喚的場合，對方的攻擊·效果對象轉變為此卡。此卡的攻擊·守備力變成場上除此卡以外攻擊力最高的怪獸的攻擊力加1。

與之相對應的神之卡為「太陽神的翼神龍」。
