黃金大獎賽正式開賽，然而賽前17和4AM戰隊鬧矛盾，韋神放話

2019-12-16 232條評論 1,974次閱讀 665人點贊

　　經過前兩天的solo賽之後，絕地求生黃金大獎賽的正式賽也拉開了帷幕，有了前兩天solo賽選手個人精彩的表現之後，很多人對於正式賽都充滿了期待。特別是A組的分組特別死亡，讓很多人都期待有多少隊伍能夠殺出重圍，可以說在這次A組中有今年PGC表現出色的四支隊伍，同時也有微博杯剛拿冠軍的RNG這樣的新秀戰隊，看到這樣的分組很多人都跟小編一樣期待的不行。

　　在這次比賽中有幾大選手十分的值得大家去關注，一個就是在Solo賽中獲得第三的堡壘，要知道天霸戰隊在PCL比賽中也是一個十分重要隊伍，只不過今年的表現並不是那麼的出色，不過目前看來戰隊選手堡壘似乎狀態有些回溫，大家可以期待一下之後天霸戰隊的表現。此外就是XDD選手了，雖然說在solo賽最後兩天的表現並不是那麼的出色，但是在微博杯中帶領RNG的表現大家也是有目共睹的，可以期待一下正式賽之後他會做出怎樣的蛻變。此外，就是4am這支隊伍了，很多人也是期待經過半個月的調整時間隊伍是否已經回歸過去的狀態呢?

　　然而沒想到的是比賽還未開始戰隊之間的矛盾竟然先一步引發出來了，要知道17戰隊和4am戰隊之間是歷來就有矛盾的，畢竟這兩支戰隊在pcl圈子裡面是兩大豪門戰隊，結果卻因為比賽產生了一些不可調和的矛盾，因此平時你就會看到選手選手在直播的時候直播間經常有人吵架，就在最近和韋神關係比較好的小魚在直播時候就談論到這個問題，他表示戰隊之間哪有這麼多的矛盾，其實站出來聊一聊就好了沒有必要這樣一直的鬧下去的，然而沒想到的是韋神表示和他沒有關係。

　　要知道小魚很早之前和韋神是好朋友了，在直播中碰到這樣的情況就覺得心裏面特別的不舒服，就希望這兩支戰隊之間能夠和解，不希望粉絲因為這些問題繼續的吵架，而面對這樣的看法韋神並沒有第一時間做出回應，反而是第一時間引開了話題，或許他本人也不想聊到這方面的東西吧。然而小魚想繼續將這個問題引回來時候，韋神很直接的表示玩遊戲就好了沒有必要管這個問題，關他什麼事啊?

　　其實在小編看來粉絲之間的矛盾並不是那麼容易調和的，畢竟中間還摻雜著一些黑粉從中作亂，此外小魚作為韋神的朋友，可能韋神並不希望他介入到這場風波中，因此面對這樣的情況韋神才在第一時間做出了這樣的反應。在小編看來兩支戰隊之間的矛盾是不可能通過一句話而得到解決的，為何不將這種矛盾轉化為比賽之間博弈的契機呢?

　　在比賽中誰能打得更好誰就成王，小編覺得韋神或許也是有這樣的想法，所以說這麼久了兩支戰隊之間還沒有得到一個解決的辦法。不過賽前發生這樣的的情況，小編對於這次的大獎賽越來越期待了(吃瓜心理)，不知道你覺得這次大獎賽誰能獲得最後的冠軍呢?
