皇室戰爭：總決賽5大核心卡牌出爐，跟著版本玩肯定沒錯

天天要聞
2019-12-13

曉東說遊戲，皇室天天見！

各位CR的小夥伴們大家好，一年一度的《皇室戰爭》CRL全球總決賽已經落下帷幕，來自世界各地的選手們為大家獻上了一場又一場的精彩比賽。大家知道，遊戲賽場就是卡牌風向標，跟著大神玩遊戲肯定是沒錯的，那麼在總決賽的25場1V1比賽中，他們都用什麼卡牌來做自己的核心呢？下面的內容，咱們就來通過數據了解一下吧。

TOP5：巨人

巨人在這次的總決賽中，共出場5次，勝率40%。這是一張從進入遊戲就開始陪伴我們的卡牌，可以說是個老朋友了。但也正因為太早出現，不少玩家就會從從心裡覺得他弱，不過在大神眼裡，巨人可是肉盾中少有的性價比卡牌，畢竟穩健可靠是它的代名詞。在比賽中，暗夜女巫和巨人的搭配成了看點，雖然不如石頭人那麼強，但多出來的3點費用還可以接個礦工，使用上更加靈活。

TOP4：超級騎士

現版本的超級騎士被官方譽為「地面一哥」，能打能扛，而且還有躍擊技能傍身，曉東最喜歡的就是橋頭砸野蠻人，那種感覺真的不要太爽。本次總決賽超騎出場6次，勝率高達66.7%，強不強看勝率就知道了。在卡組使用中，WEDGM的「超騎羊」以4勝0負的戰績可謂是出盡風頭，而其它套路還有超騎炸彈人、超騎毒礦等，都是很不錯的上分選擇。

TOP3：熔岩獵犬

排在第三名的是熔岩獵犬，它在本次總決賽中有8次登場，但無奈勝率僅有可憐的25%，6次的失利都是敗給了小費卡組。證明卡組本身沒什麼毛病，只是被針對了而已，不然為什麼力量哥、TNT等高手還要用它，並且還是在這麼重要的比賽。

TOP2：骷髏召喚

第二名是骷髏召喚，說實話，曉東玩這麼久的CR，對這張卡牌一直無感。但在這次總決賽中，它有10次的出場，而且勝率高達50%。平時我們遇到骷髏召喚，基本都是各種肉盾頂傷害，然後骷髏蹦迪拆塔。但在這裡，大神們的玩法就很多變了，要麼就是傳統的蹦迪，要麼就是抓機會一波，要麼就速轉蹦迪，一次又一次的耗你，反正就是驅使小骷髏們去找你。

TOP1：掘地礦工

核心榜第一名的，毋庸置疑就是掘地礦工了，這張卡牌不論是在什麼級別的比賽都是第一核心卡，理由實在是太多了，最重要的就是他的性價比，而且他還那麼的百搭。在這次比賽中，礦工登場20次，勝率30%，所有你能想得到的卡牌搭配在這裡都出現過。就像官方所說的，每位大神的絕活兒里，必有一套礦工體系，所以，傳奇卡優先培養什麼，就不用曉東說了吧。

好了，以上就是總決賽中排名前5的核心卡牌了，各位小夥伴們對他們有什麼不同的意見，歡迎評論區留言，說出你的看法吧。
