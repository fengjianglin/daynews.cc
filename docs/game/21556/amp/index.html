火影手游：往年聖誕忍者680點券返場，聖誕黑土要變直購？

天天要聞
2019-12-19

真的沒想到，保持了三年的聖誕忍者招募方式，在今年可能要打破了，我們先來看一下完整的木葉快報，我先說一下重點，快報里沒有提到聖誕黑土的招募方式，只提到了往年聖誕忍者返場的消息，每個忍者680點券，看起來比以往的點券B忍更便宜？其實，是更貴了！

我先來說一下往年的聖誕忍者招募方式，先氪50獲得購買許可權，然後用1980金幣購買，很多時候，這個氪50還能趕上續費月卡，所以這個方式一直都是被玩家肯定的，前年的聖誕天天延續了這個方式，去年的聖誕香磷也是如此，本來以為今年也會按照這個方式上架聖誕黑土的時候，魔方直接把往年的聖誕忍者變成了680點券購買的方式，現在至少確認這些舊聖誕忍者應該不能再用1980金幣買到了。

我有點後悔在去年沒有招募到聖誕雛田和聖誕天天，只買了一個聖誕香磷，1980金幣的價格是真的不貴，強度方面並不重要，聖誕忍者主要就是收集，當時可能有點腦抽，而氪50獲取資格我是正好趕上月卡到期，所以這個活動對於我來說非常良心。

目前聖誕黑土的招募方式並沒有公布，既然舊聖誕忍者都變成了點券購買，那聖誕黑土可能會是一個獨立的招募方式，而不是再像以往那樣氪50，然後再以1980金幣購買的方式了。

如果聖誕黑土跟返場聖誕忍者一樣是680點券，我覺得還說得過去，而如果直接升到980點券，或者是乾脆變成像疾風咒印佐助那樣的直購，那我覺得魔方這波操作是真的太秀了，本來很多月卡玩家攢出來的點券還打算留給金角銀角，現在碰到這波聖誕忍者又有點糾結了，我個人建議，聖誕忍者更主要的是收藏，而想要強度，肯定還是要留給金角銀角，我就怕在第二季忍法帖結束之前還會有新的點券忍者來干擾，那就真的是更困擾了。

前兩天的疾風咒印佐助，我覺得直購沒啥問題，價格高也正常，反正我是不會考慮，你隨便賣，但這個聖誕忍者招募方式保持了這麼久的時間，你說改就改了？聖誕忍者一年就返場一次，很多玩家都在等著她們的出現，現在倒好，又要讓部分玩家陷入兩難，是放棄，還是氪金，我的建議是，氪金肯定不要，就這麼簡單！
