魔獸世界懷舊服資深老兵教你如何做好戰歌旗幟的搬運工

摘要

隨著昨天新伺服器塞卡爾——PvP的開啟，12月11日魔獸世界懷舊服將開啟大家期待已久的戰歌峽谷和奧特蘭克山谷戰場，卡利姆多也將會迎來元素入侵事件。對於老兵們來說是個很振奮人心的大事兒哦，對於新兵來說我來奉上一些當年的一些乾貨！

作為一個當初60年代，R14大元帥德魯伊的我來說，表示德魯伊戰場插旗還是很有得玩的哦，對於變形金剛的我們來說，能抗能打能輔助，還想要什麼自行車呢哈，尤其是在戰歌峽谷號稱戰歌扛旗小能手，為什麼？因為我不吃控啊，速度快啊，抗揍啊！天賦樹是死機，我們可以根據每個場景和用途洗天賦，推薦一個自用戰歌天賦，優勢控制能力和反控制能力強，劣勢沒有那麼肉，但是你也要記住你不是一個人在戰鬥，足以讓你跑遍全場！

變形金剛的開局基本和其他職業是一樣的，頂門但是要走上邊的，因為不是無腦的上來就潛行過去，要以最快的速度，上坐騎可以單獨貼邊走，沒人盯住你可以直接跑到對面的二樓入口，因為是可以最大化縮短時間的路線！要是有人追你過來，果斷掉頭繞視角，變貓潛行！看他們的動向，既然被發現了，就可以看距離嘗試走中路，觀察下通道的鞋子，有的話一會可以看對面守旗的配置！考慮是否走右門還是通道，若是非潛行職業看旗子的話，可以直接突襲起手，強化過的突襲足以讓你拿起挑頭開溜，看對方是否章解抓你，這就體現出德魯伊變形不吃控制的巨大優勢了，不要著急交疾跑，因為上來就交的話，被控制了你會損失好幾秒的時間，物理職業的話，直接變熊先跑出屋子，出了野外你的自然直握就派上用場了，他要是開始用章解了你的突襲，這時候他就只能看著你與他拉開距離了，越跑越遠越跑越遠。你要是夠豪的話洗個工程會讓你更加的像個開掛的生物！友情提示：注意獵人的恐懼野獸哦，野性形態不吃悶棍你說盜賊沒鬱悶不鬱悶。

隨著懷舊服日益的火爆排隊成了我們最尷尬的問題，可能下了班回家就那麼一會時間玩，還都用在排隊上了，登上了還不敢隨意下線，不然就得重新拍，你說尷尬不尷尬！好在昨天5號魔獸世界懷舊服給咱新增1組經典懷舊服伺服器。塞卡爾——PvP，來啊約起來吧！
