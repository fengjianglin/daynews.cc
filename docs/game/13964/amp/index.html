對馬島之鬼公布新宣傳片，遊戲畫風突變，完整版將於TGA放出

天天要聞
7 天 ago

在去年6月的PS4遊戲發布會，索尼針對《漫威蜘蛛俠》、《死亡擱淺》、《最後生還者2》和《對馬島之鬼》這四款畫質極高的PS4獨佔遊戲進行宣傳，玩家們也因此將它們戲稱為「索尼四大天王」。

現在《漫威蜘蛛俠》和《死亡擱淺》都已發售，《最後生還者2》宣布跳票至明年5月，而2017年亮相的《對馬島之鬼》終於在本月索尼「State of Play」直播中公布了全新宣傳片。

不過這個宣傳片與之前的相比有非常大的區別，看看2018E3的《對馬島之鬼》宣傳片和實機演示，斗笠、蓑衣、楓葉、白馬、沉默寡言的主角（日本武士）、刀劍相接的打鬥，這些都展示出了一種獨特的俠客風格。

再看看最新的宣傳片，要不是聽到遊戲名字，那你可能就把它當成其它遊戲了，還有不少網友吐槽遊戲畫質大幅縮水。

2018E3實機演示

2019索尼年末直播宣傳片

在這1分鐘左右的最新視頻中，主角也沒有像之前一樣，憑一把武士刀和自身本領對抗眾多敵人，而是在敵方喊出「武士」這個詞後掏出了煙霧彈，隨後整個宣傳片就結束了。

1分鐘左右的視頻沒有什麼信息量，武士主角突變忍者的原因也不解釋，《對馬島之鬼》官方還再度畫餅，宣布將在2019TGA上放出完整宣傳片。

鑒於《對馬島之鬼》是一款開放世界遊戲，講訴了武士「准」在日本對馬島上對抗蒙古軍隊的故事（基於歷史的虛構故事）。

或許是考慮到讓玩家從頭到尾都以武士身份對抗整個軍隊會很無趣，《對馬島之鬼》官方才在遊戲中加入忍者要素，可與之前宣傳所造成的巨大反差引起了大量玩家的吐槽。

《對馬島之鬼》之前的宣傳片截圖

《對馬島之鬼》最新宣傳片截圖

還有網友猜測《對馬島之鬼》在跳票期間已經回爐重造，他們之所以會有這樣的想法，是因為《對馬島之鬼》在當年和《仁王2》以及《只狼》是玩家們最期待的三款武士遊戲。

現在《只狼》因基於格擋的大膽創新而大獲成功，《仁王2》也通過加入妖化要素而受到不少玩家的關注，《對馬島之鬼》值得拿出手的就只有高畫質和開放世界，因此廠商為了遊戲質量將其推倒重做。

不過這些都是基於新宣傳片的猜測，《對馬島之鬼》具體表現如何，還要等到2019TGA（12月13日）舉辦的那天才能知曉。
