玩了這麼久，你真的知道《明日方舟》中的泰拉是個什麼樣的世界嗎

天天要聞
2019-12-10

《明日方舟》是一款近期比較熱門的手游，很多玩家都在遊玩的過後都喜歡稱自己為失智博士。

至於為什麼會這樣，想必大家都耳目共睹，方舟的體力恢復太慢了，而每一把都會讓人消耗大量的體力，基本打一局就要等一個多小時，因此這款遊戲也慢慢被上班族所熱衷。

很多博士會在失去理智的時候，翻看幹員圖鑑，查看背景故事，那麼，你真的了解方舟中所描述的泰拉世界，是個什麼樣的地方嗎？

其實泰拉就是大地的意思，在遊戲《戰錘40k》中，人類帝國被稱為神聖泰拉。

泰拉其實是一個架空世界，有著與地球相似而又截然不同的環境，歷史和文明。這裡的主要居民是帶有各種動物特徵的亞人類，而是否存在純粹的「人類」，目前還不得而知。（很有可能博士就是古人類？）

但泰拉世界卻不像我們地球這樣和平（笑），那裡每天都有不可預知的天災，設定就是現實生活中的火山爆發、洪水、海嘯、地震、甚至天降隕石都歸禍於天災（天災：我太難了）。

而其中生存的亞人類，為了生存和躲避天災，便建造了移動城市，（想想巨型哈爾的移動城堡？）

雖然天災帶來了無盡的苦惱，卻又帶來了源石，源石在這個世界中不可或缺，危險，但又讓人趨之若鶩（忽然想到核能？）

源石會給他們帶來能量，卻也會因感染患得礦石病，會在體表生長出石頭，於是便有了感染者這一類人。

感染者自然是被人嫌棄的，就像你在大街上知道有個人得了流感，你自然會躲得遠遠的。

但是大部分感染者便會感覺到，自己被這個社會所拋棄了，一個兩個，慢慢的，感染者變多了，他們有了自己的隊伍—整合運動。

有人覺得自己被拋棄，想要報復，想要他們自己的一片天地，便會發起戰爭，自然也會有人去反抗。

而這其中，卻還有一支明明是感染者，卻向著未被感染的人類，他們在整合運動嚴重，就是叛徒，在正常的人們眼中，便是工具，但是在這樣難堪的立場上，他們還在努力地去戰鬥，為了和平解決感染者問題。

這邊是我們的立場—-羅德島。

羅德島的行動在大部分人眼中都是不智之舉，但這在我們看來，難道不是最正義的舉動嗎？（雖然現在不知道還算不算正義，阿米婭：？？）

之後泰拉世界還有各種各樣的種族與國家，城邦與地區，都等待著各位博士自己去了解，有興趣可以在評論區一起討論啊！
