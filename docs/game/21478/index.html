爐石傳說：獵人勝率緊隨薩滿之後，外服傳說12舒瑪任務獵來襲

2019-12-19 602條評論 3,444次閱讀 1779人點贊

我開的「垃圾」橙卡有用了

在新版本新卡預覽時，中立橙卡舒瑪的評價都不算太高。所以，當玩家們興高采烈地點開一張泛有橙光的卡牌，發現它是舒瑪時，心中難免失望！包括我，我開的第一張橙就是觸手怪？舒瑪雖然有單卡鋪滿己方場地的能力，但作為一張高費鋪1/1，難免讓人聯想到術士職業卡蝙蝠之神，1/1身材實在是過於單薄，威脅又不大，1/7的本體在高費又太好解，1攻也著實猶如針扎不同不癢。

然而實際上，我們都錯了。進入新版本後，任務獵重新嶄露頭角。雖然任務獵的構築在不斷的調整優化，但新橙卡在核心的位置是越坐越穩。獵人看重的無疑是舒瑪的單卡鋪場能力，而獵人任務獎勵的群體加攻，又能很好地彌補舒瑪攻擊力低，威脅小的劣勢。就是這樣，兩者優勢互補，一套強力的舒瑪任務獵就此誕生！

在這個迦拉克隆薩橫行的年代，外服玩家BoarControl使用一套任務獵打到了傳說12的高位，算是為獵人掙了一口氣！值得一提的是，在外服大數據網站上，獵人是除薩滿外，唯一一個勝率超過五成的職業。

下面是他的卡組牌表，慣例卡組代碼見文末~

任務獵

一句話攻略

-這是一套鋪場卡組，利用鋪場做任務，也利用鋪場給任務技能提供輸出環境。

-卡組最大的武器無疑是任務卡獎勵的英雄技能，儘管卡組裡有大量出隨從的卡牌，1費老虎送隨從，2費野豬亡語出魚人等等，但對於完成任務需要的數量來說還遠遠不夠。在多數情況下，你需要在6，7費回合才能完成20個隨從的任務。

-如果說舒瑪是任務獵是絕對核心，那麼法術飛蝗蟲群則是當仁不讓的二把手。6費出完成7/20的任務，很多情況下，第一張蟲群能幫你完成最後的任務進度，而第二張鋪場則會直接威脅對面的血量！

-說到鋪場卡組，沒人會忘記老朋友海巨人！在雙方都在竭力鋪場時，哪方能更早地打出海巨人，往往能決定勝負的走向。就跟薩滿的7/3/4。前中期8攻的威脅有多大，自不用多說。

-來說說薩滿，薩滿自然是現在最強的卡組沒有之一，如何抗薩是所有卡組無法逃避的命題。任務獵最大的優勢就在於鋪場，在對面兩個88出來之前能把血線壓制下去那是甚好。

-卡組的對局軌跡多數是前期突襲混任務，6費做完，過年瘋狂鋪，瘋狂加攻，對面一但留下隨從，兩輪就死。火車王可以投傷害，放狗一波除了做任務，還有機會懲罰那些喜歡鋪場的對手，比如薩滿！
