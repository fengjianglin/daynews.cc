稻葉敦志：想做《神奇101》和《瘋狂世界》續作

2019-12-18 192條評論 1,466次閱讀 336人點贊
作者：尼米茲

白金工作室（Platinum Games）在過去幾年為任天堂製作了不少獨佔遊戲，在「任飯」當中享有很高的聲譽。你可能知道白金工作室製作的《獵天使魔女》、《尼爾：自動人形》。但除了正在製作的《獵天使魔女3》之外，白金的開發者也有興趣為那些不太有名的遊戲製作續作。

白金工作室共同創始人稻葉敦志，在被媒體問及最有興趣製作哪個遊戲時表示，《神奇101（The Wonderful 101）》和《瘋狂世界（MadWorld）》是他最有興趣做的。

​

《瘋狂世界》

稻葉敦志先聊到了《瘋狂世界》。他說，這款遊戲是白金成立以後發售的首個作品，和他們作為四葉草工作室（Clover Studios）製作的《大神》截然相反。《大神》是一款色彩斑斕，充滿童趣的動作遊戲。但《瘋狂世界》完全相反，是黑白、暴力且殘酷的。《瘋狂世界》讓稻葉敦志意識到，即使有大量暴力和血腥元素，遊戲依然可以很有深度，非常耐玩。

​

《神奇101》

《神奇101》是稻葉敦志提及的另一款遊戲。這是他與神谷英樹合作開發的最後一款遊戲。他們二人都是非常有個性的開發者，容易產生碰撞。這種碰撞可以幫助他們開發出高質量的遊戲，但也讓開發過程變的非常痛苦。即便如此，稻葉敦志仍然認為開發《神奇101》是非常有趣的體驗。如果將來能與神谷英樹再次合作，他希望能製作《神奇101》的續作。

白金工作室於今年（2019）八月份推出了任天堂Switch獨佔遊戲《異界鎖鏈》。他們正在開發的《獵天使魔女3》發售日期未定。
