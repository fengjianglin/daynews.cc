盛裝登場殺完就走！論懷舊服法師如何攻克奧山各種保衛戰之細節

2019-12-19 611條評論 4,098次閱讀 2843人點贊

（如果你打戰場只想著下雪變羊寒冰箭，只為混聲望榮譽，請關閉此文）

現階段大部分奧山戰場BL苦不堪言，因為LM的戰術已經漸漸變為雷矛/大橋/墓地保衛戰，LM聲望榮譽刷的飛起蹭蹭往上漲，BL卻經常要花數個小時才能攻破，甚至拖到輸的節奏，這可怎麼破？

於是網上就出現了各種各樣的攻略，討論激烈。不過以我的經驗來看，無論用什麼攻略，要攻破保衛戰，最終還不是需要法師和戰士上前衝鋒陷陣？

本人打了二十幾場奧山，只輸了兩場，並且最長耗時的勝局也就打了一個多小時。也慢慢在不斷地送死中打出了經驗，相信自己是發揮了最大化的作用來取得勝利。

法師們請自信點，你只要夠勇敢，就能在攻破各種保衛戰中起到關鍵作用。

有人會想不就是衝上去無腦冰環吹風奧爆么有什麼好說的。

在此本人想說：請一定要帶著腦子沖，細節非常重要！

攻克雷矛還算容易，LM遠程在山頭陰人，你自然也可以去另兩個山頭陰他們。請注意陰他們不是為了人頭，而是為了驅趕他們，棒打鴛鴦。

當LM在雷矛的山頭山腳陰人怎麼辦？

最難的自然是大橋/墓地保衛戰，雷矛變紅之後就可以準備向LM人堆沖了。沖之前最基本的條件，一是觀察自己這邊人手夠不夠有沒有遠程，二是號召其他FS/ZS一起沖。可能當時會號召不動，畢竟總會有FS只想呆在後方下雪，但是只要你自己多衝幾次，自然會有人願意跟你一起沖。

最重要的幾點細節（滿血上盾騎馬無需多說）：

1、提前選好一個關鍵目標，首選奶媽。最好的時機就是她加血時，馬上閃現加反制一起按（反制無GCD最大的優勢）

2、準備好飾品，首推暗反加巴羅夫。衝進去馬上召喚巴羅夫，暗反也可以衝進去直接用，SS/AM基本上都會給衝上來的FS死纏/沉默；遇到FS強反賊昏迷請直接冰箱，LR的驅散射擊這類一碰就醒的控制技一般不用管。這裡要注意，FS60年代的徽章有很大的局限性，這種時候巨魔戴合適亡靈可不戴。

3、想方設法殺掉關鍵人物，不死也要盡量讓他殘血。很多時候這比群控AOE更加重要，逮到對方大奶就是一頓揍，儘可能送她回家。一定要注意空藍的奶媽不是優先擊殺目標。

盛裝登場殺完就走！論懷舊服法師如何攻克奧山各種保衛戰

有人會說FS爆發不夠秒不了奶媽，那是因為主流是冰法，但至少有兩個解決方法：

1、至少兩三個冰法一起進場，強反有控制的法系。冰法雖然缺了點爆發，但是優勢是有盾有強反，進去可以多爆幾下。

2、推薦洗成冰火元素法天賦，如視頻中本人進去，冰環衝擊波火沖吹風，一人的爆發足以擊殺目標。同時對敵方造成大量群體傷害。這是非常重要的，只要敵方殘血就一定有人想上去干，然後我方鬥志昂揚一鼓作氣贏得團戰，這樣勝利的天平才會慢慢向你傾斜。

如果你不是在攻堅大橋/墓地，而是在後方防守怎麼打？

道理大致相同，最好是在他們打霜狼要塞，也就是部落BOSS老巢里的JY時，衝上去反制/控制奶媽加AOE。老巢里的JY獸人有非常強力的順劈斬，配合他就能砍倒一大片，大家可以看下面視頻的最後一分鐘見識一下。

奧山霜狼要塞的JY有多猛？一斧子砍倒一大片

最終說明一下，在戰場被敵方擊殺時裝備是不會掉耐久的，所以如果你想儘快地獲得最終的勝利，就別在意自己的數據難看，勇於衝上去，發揮法師最大的優勢，才能為你的團隊做最大的貢獻。

最後祝大家能在奧山玩的開心。
