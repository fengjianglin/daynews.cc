教你如何花最少的錢 玩夢幻西遊性價比最高的09法系

2019-12-10 168條評論 4,806次閱讀 2737人點贊

夢幻西遊神威組別是X9中人氣最高的級別，因此吸引了大批玩家進駐，其中不乏土豪玩家和平民玩家，今天就來教下大家，如何花最少的錢，玩性價比最高的法系。

　　玩09法系要性價比就不能追求極限，8段即可，選擇一般的魔力套就好，便宜好用

　　武器嘛，一般雙加48以上就好，法傷140+的就要2000多，我這把法傷119的就已經足夠了，最重要是便宜好用

　　腰帶對法系來說沒啥要求，夠用就好，500血和600血差距不打，價格卻差了很多倍，所以我選擇80的腰帶，便宜好用

　　09法系，項鏈是關鍵，任務活動就要買220人造的，我這個219的和220的就差了1靈力，價格卻便宜很多，寶石嘛9段就夠，不值得打10段，打了就虧了

　　衣服是生存的根本，如果不選擇高防禦衣服，秘境后羿你都過不了還什麼，選擇這個9段高防禦的，便宜好用

　　鞋子對法系來說也沒啥要求，反正速度快不過物理系，不如隨便選個魔力套吧，8段就好，往上打就是虧

　　玩09法系，前排靈飾很重要，你可以放棄法傷選擇4法爆，這樣的話你暴擊才到2000(而且還看幾率)，平時秒都不過千，所以，果斷放棄法爆，選擇全法傷穩定輸出1800+，反正SML自帶4%法爆和法連，開心，人形須彌

　　後排的話就不需要什麼雙防1血了，畢竟不是69，也不是物理系，所以，最高的性價比是4防禦，如果是雙血雙防，你的防禦都到不了1400天地都殺不過，哈哈哈

　　浮石組合選擇秋風，白嫖24法傷

　　最後，神狗是信仰，我玩09就是為了神狗，畢竟神獸的屬性和輸出等級越高越明顯，可是175人物裝備都太貴，而且玩得人不多以後不好出手，69又無法發揮神獸的威力，129單機多，159人太少，唯獨09這個階段是全夢幻玩家和任務最多的，果斷選擇了09

　　我本來想賣連善+須彌，後來想想沒意思，畢竟69天地的時候一直這樣，而且違背了我的信仰原則，所以我就帶一條狗闖天下，人形須彌，狗型DT，無敵搭配

　　隨便秒秒2400+，秒死3600+，數據舒服

　　暴力美學？？1.5w秒
