三國志9：用最簡單的勢力統一還是最困難的勢力統一呢？

2019-12-07 0條評論 18次閱讀 0人點贊

摘要

相信光榮公司的遊戲《三國志9》有不少粉絲，那麼對於這個遊戲中的多個勢力，大家認為最簡單和最困難的勢力分別是哪個呢？

相信光榮公司的遊戲《三國志9》有不少粉絲，那麼對於這個遊戲中的多個勢力，大家認為最簡單和最困難的勢力分別是哪個呢？

要說哪家勢力在任何一個劇本都能混的風生水起，那麼我相信有一部分玩家會說曹老闆，但是曹老闆四面都是敵人，武將質量確實很好，可以說玩曹老闆前期壓力太大，所以我認為算不上有多麼簡單。

而玩起來最簡單的勢力非江東孫氏莫屬了，不管在那個劇本，孫氏家族附近可以說是沒啥敵人，附近還那麼多空白城市，前期只需要以逸待勞，安穩發展，收集江東人才，不時的找準時機搜集中原人才，等到北方多個城市都被摧毀的差不多的時候揮師北伐，輕而易舉就可以平定天下，簡直是在簡單不過的了。

那麼最困難的又是哪個呢？可能有人會說什麼嚴白虎，劉度之類的，這幾個人確實有點難度，但是都可以很簡單的破局的，德王只需要前期跑路去柴桑，收了柴桑的幾個人才，輕易破局，至於劉度什麼的，前期練兵，派兵去吧交州佔了，結交韓玄，收集人才，等待韓玄出兵趁機偷襲他的老巢長沙。使用SL大法收了魏延黃忠，就可以破局了。

不過這幾個勢力的難度其實還是比不上遼東公孫度，不用多說，挑戰過的都知道有多難，前期幾個劇本還能跑路去江東，可以有破局的希望。可是從群雄194的劇本開始，破局的希望就變得非常渺茫了，若是你打算跑路柴桑，怎麼跑？時間夠嗎？烏丸不打你嗎？公孫瓚也在旁邊虎視眈眈？

種種因素構成了公孫度急難破局，要想破局，靠運氣多一些，要在孫權佔領柴桑之前佔了，要和烏丸公孫瓚搞好關係，跑路去柴桑的路上別被德王，王朗什麼的出兵滅了，跑到柴桑還必須使用SL大法搞到周泰，呂蒙什麼的，要不然破局還是困難，結合以上因素，我認為最難破局的是公孫度。
