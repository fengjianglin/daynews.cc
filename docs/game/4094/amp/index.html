陰陽師隨著高稀有度式神越來越多還有環境的變動，對於式神的強度評價基本上2-3個月就要刷新一次，本次就來說說SP鬼切版本全高稀有度式神強度榜，因為陰陽師不同式神專精的領域不一樣，所以目前只能以主流版本的就業面和泛用度，以及是否擁有這些式神對於玩家遊戲體感的影響作為標準去評價，一起來看看吧。

T4（接近倉管）

桔梗、賣葯郎、一目連、荒川之主

這四位基本上一直就是墊底的命運，因為真的沒有他們能就業的地方或者說是可以用到的時候，基本上已經算是被官方策劃放棄的邊緣了，最倒霉的還是荒川和賣葯郎，鬥技單體輸出被一個小白掐得死死的。

T3（偶爾有用）

黑崎一護、鬼燈、SP蒼連、殺生丸、山風、妖刀姬、小鹿男、荒、雪童子

偶爾會有用或者作為奇策用的式神，大部分時候可能玩家想不到他們鬼燈目前的趨勢並不是特別的明朗，誠然鬼燈確實鬥技部分時候還有不錯的出場率，但不是主流，同理小鹿男需要等待回歸環境。殺生丸目前那個禁復活的賣點也就是雨女副本中用用了，這些式神論強度其實不算差，治國對比之下真的是那種偶爾用用，不到關鍵時刻想不起來的那種。

T2（特殊功能性）

陸生、SP鬼切/鬼切、SP妖刀、SP般若、SP荒川、SP小天狗、面靈氣、花鳥卷

功能性非常強力的式神，相對與偶爾用用來說，他們有時候會成為特定活動或者陣容中的最優解，比如說無腦掛機的國家隊等等，還有偶爾會在PVE中突然高光的SP妖刀等等，SP鬼切給這個位置衡量了許久應該是沒問題的，鬥技算不上頂尖，但絕對不弱，但怎麼BAN對面的式神了。

T1（常用+主流）

犬夜叉、SP酒吞、SP御饌津/御饌津、SP茨林、大岳丸、白藏主、兩面佛、閻魔、青行燈、茨木、玉藻前、酒吞、輝夜姬、彼岸花、大天狗

常說的人權卡，平常大部分的時候可能都會用到這些式神，包含有常規鬥技還有PVE活動中的主流選擇和打法，其中可能部分式神有爭議。燈姐這地位現在已經從薅過輝夜姬了，但是對於入坑時間不長而且湊不出強力體系的玩家來說，輝夜姬還是很硬核的。玉藻前和茨木的地位從沒有玩不下去下降到這個級別。其他的應該都沒有異議，SP酒吞併不是必須品，但他絕對是人權卡。犬夜叉目前已經成為了多段輸出的主流式神中僅次於SP玉藻前的了，給這個位置不過分。

T0（沒有別玩了）

SP玉藻前、瀧夜叉姬、不知火、八岐大蛇

沒這四位就別考慮愉快遊戲了，要麼就是PVE絕對核心的高輸出，要麼就是通吃一切副本能打能輔助能當工具人的，要麼就是鬥技獨立體系的。反正SP鬼切的版本，湊齊這四個能給你帶來最好的遊戲體感。
