10個月瘋狂沖星，他把冷門幻獸練成全服第一，輸了戰力贏了毅力

天天要聞
2019-12-11

每一個有追求的人，都會一直堅持自己的原則與理念。遊戲中也是如此，曾經《魔域》有這麼一位玩家，為了提升幻獸的質量，居然持續了長達10個月的地獄式沖星之路。這一舉動震撼無數《魔域》玩家！

對於「紅顏青絲朝暮」這個名字，許多玩家也許會感覺到熟悉，它就是當時霸榜多年的裁決騎士莎菲，主人為「泡沫」。這隻幻獸剛剛開始嶄露頭角的時候，還只有603星，想對於其他玩家來說，還算不上強力。但是泡沫對它確實特別的用心，一直沒有放棄對這隻莎菲的培養。

他曾經經歷過許多的困難，他軍團中的一名玩家透露說，他之前因為家裡的一些變故，很長一斷時間沒有上線。開始我們以為他都退游，後來兩個月後他又重新的上線了。看來沒有什麼困難是可以組織泡沫對魔域的熱情。

在泡沫消失的兩個月你，莎菲一度從排行榜上掉了下去，重新上線的他開始了收集材料，開始重整旗鼓經過一段時間的沉寂後，泡沫找了一個空閑的時間，把囤下的材料一次性砸了上去，一天就把莎菲幻化了十多二十多星，直接突破1000星的大關，升到了1112星。從603星到1112星，中間這509星的巨大差距，她只用了10個月就創造了記錄。

這不得不說是個令人驚嘆的成就，雖然裁決騎士莎菲相比魅影龍、紅蓮戰神這種熱門幻獸，確實不算很強，但論玩家所付出的毅力，卻是當之無愧的NO.1！

《魔域》作為一個熱血PK遊戲，戰力的增強是每個玩家追求的目標，但並不是所有人都有泡沫這麼好條件，但是只要你有了恆心，你就會得到你想要的，每一件事情的開始，都需要你的堅持。

如今，泡沫的聲音越來越少，可能已經隱退了。但是他的故事永遠留存在我們心中。如今的《魔域》不也正和他一樣嗎，為了玩家們的遊戲體驗，不斷的堅持，為玩家帶來更好的遊戲內容。就在這個寒冷的冬天，《魔域》公布全新資料片「御劍天下」將在12月21日公測，全新職業「御劍師」也將正式降臨亞特大陸。

《魔域》還在遊戲里為御劍師設計了專屬主線挑戰任務，可贏取大量福利。面對早已成名的7大職業。御劍師又將有怎麼樣的發揮了，讓我們拭目以待！
