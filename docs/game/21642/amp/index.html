神諭幻想12.24上線！精雕細琢的次時代畫質！定製你的幻想世界

天天要聞
2019-12-18

賞金獵人，為財賣命，不懼危險，在大家的認知中是遊走在危險邊緣的職業，而今天給大家帶來的這款遊戲，玩家所扮演的賞金獵人可不單單是個為財戰鬥的傢伙，而是富有正義而又足智多謀的戰士。《神諭幻想》原名：《代號:天啟》將於19年12月24日正式上線，敬請期待！

這款遊戲是一款高自由度3D西方幻想MMOARPG手游，MMOARPG這個類型大家其實也見得不少，近幾年不少由小說，電影或者一些文化改編的手游也層出不窮，那麼這款遊戲又有怎樣一個特點呢？

NO.1 恢弘磅礴的故事背景

遊戲講述的是身為賞金獵人的玩家，在接受一個賞金任務中，發現任務端倪，結識夥伴，找尋真相最後完成一場在西方幻想大世界裡的使命之旅。遊戲採用的是開放式劇情玩法，玩家在結束上一劇情時會觸發選項，這些選項將會影響到玩家之後所遇到的怪物，npc的好感度，也豐富了玩家的遊戲體驗。

NO.2 多武器自由搭配

在遊戲中沒有花里胡哨的職業限制，玩家所扮演的賞金獵人是近遠攻擊都十分強力的高高手，近戰持刀壓前陣，後可遠戰亂槍掃。遠有槍械，近有長刀，當然遊戲中也有多種多樣的武器和技能供大家選擇組合，玩家只需根據自己的喜好選擇角色的性別，並在遊戲中側重自己的戰鬥方式就好啦。

NO.3 次世代高精畫質

遊戲的高畫質一直是這款遊戲所在追求的目標，西方魔幻的風格，次世代高精畫質，全視角高清3D大地圖，呈現一個西方幻想大世界，如果玩家喜歡這種科技感與玄幻色彩並存的風格，這款遊戲不失為是一種選擇。

NO.4 自由捏臉定製主角

捏臉系統一直是許多玩家喜歡的元素之一，給自己的角色捏一個美美的樣子，在遊戲中戰鬥，完成任務，探索劇情相信也是許多玩家認為必不可少的設定，遊戲中也設定了捏臉系統，獨創專屬自己的人物風格。

NO.5 熱血戰鬥末世求生

戰鬥！恆古不變的話題，只有戰鬥才能體現賞金獵人的價值，遊戲中的戰鬥可以進行多角度調整，玩家可以根據不同的情況使用近戰或者遠程技能，遊戲中的打擊效果還是不錯的，技能特效乾淨利落不拖泥帶水，不過還是需要一些操作與走位，需要玩家進行練習哦！

總的來說，《神諭幻想》是一款帶有西方幻想色彩的MMOARPG手游，富有特點戰鬥模式，開放式劇情玩法，史詩級奇遇冒險，百態NPC可敵可友可攻略，打造專屬於你的故事軌跡。都是這款遊戲帶給玩家們的樂趣所在，相信在這裡，你可以定製自己的幻想世界！
