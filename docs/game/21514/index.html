DNF：100級新口號，是男人就下46層！旭旭寶寶也過不去

2019-12-19 399條評論 2,182次閱讀 900人點贊

地下城與勇士的100級如約更新，新史詩的屬性讓老玩家興奮，但普雷C們卻很難受，夜語黑瞳武器這麼快就涼了，雖然保留可以打造，但屬性就跟100級傳說一樣，辛苦這麼久是為了什麼喲！

100級的變化不僅僅是裝備更複雜，副本同樣變化太多了，搬磚都是普雷C的門檻，普通BOSS都是135級的水平，但還有更高難度的副本——「紅玉的詛咒」，別問有多難！一陣雨紅15的劍影都非常吃力。

100級新口號，是男人就下46層！旭旭寶寶也過不去

大家都知道，在DNF中，死亡之塔/ 絕望之塔曾經很火！後來遠古圖也出了一個無限層數的模式，超過20層就已讓普通玩家絕望了，防禦高到離譜，而100級這模式又來了，紅玉的詛咒這個副本，基礎為5層，平民玩家刷完就可以結束了，但層數越高材料獎勵越多！

一陣雨在測試100級副本的時候，就被這個地圖給虐了，紅玉的詛咒在46層以上會獲得「名譽稱號」，相當於安徒恩初期的作戰精英，擁有的玩家無一不是大佬（開G的除外），一陣雨以為是46層必得稱號，於是使勁刷……但根據副本介紹，是有幾率獲得！

如果能獲得這個稱號，價值會比心悅三的最強戰士還高，因為一般的心悅三也刷不到46層以上，一陣雨的劍影可是紅15的水平，紅玉的詛咒難在哪呢？

BOSS等級每層都+1，比如40層的時候BOSS是150級的，如果到了45層就是155級，現階段的普雷BOSS不過130級，就算是希洛克Raid更新，等級也不過超過150的！

其次就是DeBUFF，進圖就是減少50%左右的命中，這是副本效果，只有紅玉的詛咒掉落的藥劑能解，並且只是解除一段時間，而這副本，每個小房間都是BOSS！你能想像一個紅眼走位熬了幾分鐘，好不容易放個二覺然後MISS的感覺么（注意：不能用復活幣的）。稍微一個失誤，哦吼，涼了！

就算是旭旭寶寶的紅眼來了，也是過不去這個副本的，因為紅玉的詛咒根本沒有層數上限，就算是為了46層以上的專屬稱號，僅靠95B史詩都難以做到！因為紅眼續航太差了，武器都得修幾輪。

至於普通玩家，20層以上就很難做到！所以DNF出了135級的格藍迪，平民玩家就刷這個吧，最有趣的一點，紅玉的詛咒竟然被定義成了搬磚副本……100級新口號，是男人就下46層，可是增幅15都吃力，太套路了。更為過分的是，100級最難副本並不是紅玉的詛咒。
