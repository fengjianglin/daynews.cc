DNF體驗服：劍宗改版加強實測，皮甲短劍“封神”，傷害超越輕甲

2019-12-16 294條評論 2,464次閱讀 277人點贊

「地下城與勇士之小狐狸君愛談遊戲電競」第七百四十六期《DNF體驗服：劍宗改版加強實測，皮甲短劍「封神」，傷害超越輕甲》，體驗服本次維護後，五甲的地位變得非常的有意思了，之前輕甲一家獨大的情況，得到了很大的改觀。不過就目前國服特色屬強來看，輕甲的地位對於絕大多數職業而言，還是無法撼動的。不過本次劍宗的改版，開始出現了「奇蹟」。對於短劍馭劍士而言，皮甲的BUFF變得越來越有意思的，目前打樁實測，皮甲短劍傷害超越了輕甲巨劍，成為未來的打樁輸出主流裝備。不過劍宗的本命甲原本就是輕甲，如果不是大號追求極致傷害。輕甲還是首選，畢竟不用轉甲。但是佩戴短劍的劍宗，爆發能力能達到目前國服前三的水準！而巨劍輕甲達到國服前十的強度。其實就是幻神和超一線的差距。

DNF體驗服：劍宗改版加強實測，

其實本次體驗服的改版，對於劍宗而言有2大優勢：第一，短劍劍宗，1~30級BUFF提升巨大，即使沒有新技能劍魔覺醒，短劍也是大有崛起之勢，其實就是那句話——誰用短劍誰知道；第二，新被動魔劍覺醒，這個技能堪稱是繼無我劍氣之後，有一大國服特色BUFF，基礎5級上限15級，等到100級更新之後，這個技能就是封神的存在！玩家實測，短劍劍宗完全超越巨劍，成為未來版本的主流玩法。

皮甲短劍「封神」 ，體驗服排名第三：

其實皮甲在95版本末期，也是吃到國服特色最大的裝備之一。當然首當其衝的還是屬強白字——輕甲！改版之後皮甲提升29.6%，其餘提升21.06%！皮甲翻身超越了重板，巨劍下實戰可能不會有什麼優勢，但是短劍居然能反超輕甲成為第一甲！目前實測劍宗緊隨瞎子，成為此次改版的最大受益者之一，目前排名體驗服第三。

短劍皮甲傷害超越巨劍輕甲：

其實短劍和巨劍相比，最大的優勢就是附加傷害，就像是機械師的G2、高達一樣，自動追蹤傷害。而且觸發幾率非常的高。而且短劍沒有巨劍的僵直和釋放慢的缺點。所以未來還是要追求短劍的極限傷害！
