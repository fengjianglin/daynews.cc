王者榮耀冬冠杯大名單出爐，hero三虎登場，阿泰可惜了

天天要聞
2019-12-18

大家好，這裡是指尖上的王者，12.21日王者榮耀冬季冠軍杯正式開打，目前兩個賽區的戰隊大名單已經公布，本文為kpl賽區大名單選手。

AG超玩會：藍柚、MC、6.6、愛思、一諾、老帥、麟羽

本賽季mc加入大名單，或許這就是他的宿命，夢淚沒在名單中，或將從此告別kpl。

DYG.JC：清清、星宇、縱情、懟懟、青浪、ovo、易崢

縱情在大名單，不清楚是否能首發，而老王和evd或許也將難有登場的機會，或許轉會是個不錯的選擇，evd的射手很強，奈何隊伍更看重易崢。

eStarPro：諾言、cat、花海、無銘、alan、無心

e星是鐵打的陣容，萬年不變，連替補也只有一個人，但是沒有特殊情況基本不會變的，不過無心冬冠杯報的位置是邊路，或許alan狀態低迷的時候也是有可能上的，兮兮沒在大名單，可惜了這個邊路。

Hero：久誠、久羽、最初、久酷、塵夏、久年、久星

比較順心的是現役hero五虎三人都在大名單，而且久凡不在，那麼最初就是穩定的首發了，而久誠跟久羽要爭一下，而塵夏也要和久星搶打野，大概率三人都會首發，值得期待。

QGhappy：fly、770、hurt、妖刀、giao、snow、mojo

snow竟然也在大名單中，其實他們的首發沒有什麼變化，kpl拿到了亞軍，雖然距離他們粉絲的期望還是差了一步，但是亞軍已經是頂尖強隊了，沒有什麼必要改變陣容，有人說hurt會換成妖刀，我倒是覺得雖然hurt狀態不在巔峰，但是依然是大腿。

上賽季被瘋狂搶的139一次都沒能上場，這一次也沒能進大名單，不知道俱樂部買來幹嘛，以選代ban？

RW俠：djie、dnan、yguo、chengkun、huamanlou、huayun

rw俠的首發沒有變化，在世界冠軍杯的時候，rw俠應該是最耀眼的戰隊了，一路連勝沖向總決賽，雖然最後摔了一跤，但仍然讓人記憶深刻。

TS：詩酒、阿豆、千世、神人、暖陽、十三

首發基本沒什麼變化，不過AT沒有首發，kpl中表現也不好，很難想到他還有什麼理由回來，希望看到這些老人但好像不太可能了，at或許將永遠失去了這個舞台。

XQ：晨羽、綁兔、冰塵、釺城、九尾、Eleven

這支戰隊給了我們太多驚喜，九尾和綁兔真的是潛力股，而釺城的射手在常規賽後期也開始發力，很想看到這個隊伍在冬冠杯的表現，相信不會讓我們失望。

以上就是本屆冬冠杯kpl賽區的大名單。
