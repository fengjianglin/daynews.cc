稱霸中原《萬箭齊發》揭秘高級裝備

天天要聞
2019-12-21

《萬箭齊發》是一款基於H5平台的國戰策略頁游，主打跨服競技和跨服國戰。上百位歷史名將供您選擇，武將的技能特色更加豐富，與傳統SLG頁游相比，國戰時的策略和武將組合更加的複雜，跨服競技和跨服國戰的玩法也讓遊戲的競技味更重。

在遊戲中，紫色以上的高級裝備除了可以直接獲得外，還可以通過合成系統來合成。目前，合成系統中有兩種合成方式：第一種是隨機合成，即主公給出材料後，合成隨機部位的隨機裝備;第二種是普通合成，可合成指定的高級裝備。

兩種合成方式各有優缺點：第一種方式的優點是合成時只需要消耗一定數量的任意裝備碎片，當然這些碎片必須是相同品質的。而它的缺點是合成的目標不確定，很容易得到重複的裝備，造成材料的浪費。第二種方式的優點是可以合成指定裝備，對打造緣分裝備非常有用。它的缺點是合成需要的材料種類較多，主要耗時間去一一收集。

主公可以根據自己的需要來選擇合適的合成方式。一般來說，在等級較小、裝備較少時，可以使用隨機合成的方法，反正合成出來的裝備都能用。等到等級升上去之後，再使用第二種合成方法，合成出適合武將的裝備，將武將的能力強化到極致。

在競技場中，主公可與其他玩家進行競技，勝利後可獲得競技幣。如果勝利時，自己的排名低於對手，那麼主公還可以繼承對手的排名，並得到提升排名的元寶獎勵。除了戰鬥獎勵外，系統每晚還會根據當天22點時的競技場排名，向主公發放對應的排名獎勵，獎品通過郵件發送。

在競技場中的排名越高，每天的排名獎勵就越豐厚。那麼競技場排名如何才能快速提升呢?

方法有二：

選擇合適的對手，合適對手對名次的提升是非常重要的。在選擇對手時，建議主公先看看最左邊對手的排名和戰力，然後比較雙方戰力，如果己方戰力有優勢，那麼就選擇這位玩家作為對手，如果打不過，就往後看。這樣反覆操作。因為玩家的排名從左到右是逐漸下降的，所以戰勝最左邊玩家就能得到更高的排名。如果主公對當前列表的玩家不滿意，也可以刷新列表，重選對手。不過當前最高排名的對手不會變化，這點需要注意。

增加競技場的挑戰次數

競技場的挑戰次數增加後，主公就可以多挑戰幾個對手，可以加快名次的提升速度(當然前提是要有足夠的實力)。這個遊戲中增加競技場挑戰次數的方法比較特殊，主公只能通過提升VIP等級來增加挑戰次數。當主公的VIP達到VIP4後，可以獲得額外的1次挑戰機會，之後升級VIP會繼續增加挑戰次數，直到VIP滿級。
