酒館戰棋：版本一代之神即將墜落，新鮮血液即將加入

天天要聞
2019-12-05

摘要

今天是12月5日，就在明天，當前版本最強流派，混搭流，當中的關鍵核心卡即將被削弱！在這最後的十多個小時里，你還在等什麼？不僅如此，也有一些新鮮血液即將進入玩家們的視野，它們又是誰呢？下面我們就來看一看吧！

今天是12月5日，就在明天，當前版本最強流派，混搭流，當中的關鍵核心卡即將被削弱！在這最後的十多個小時里，你還在等什麼？不僅如此，也有一些新鮮血液即將進入玩家們的視野，它們又是誰呢？下面我們就來看一看吧！

光牙被削弱

請記住他現在的效果，+2+2將在十多個小時以後，變成+2+1，光牙之後還會屹立在酒館之巔嗎？我們拭目以待吧！

酒鬼觀點：如果說混搭流派在當前版本是T1隊列，即使光牙被削弱，酒鬼依舊認為混搭流不會跌落T2。畢竟+2的攻擊力在這裡放著呢，在面對很多魚人流的時候，很高的血量意義很大嗎？就像葉問師傅，在大後期碰到劇毒，再大的屁股也沒有用。但是如果面對亡語流、惡魔流等流派時，多一滴血就是多一點可能！

新隨從和新英雄

漂浮觀察者

酒館觀點：漂浮觀察者的效果就像憤怒編織者的效果一樣應該不會有變，2星科技應該能找到這個隨從，不僅如此，漂浮觀察者與憤怒編織者的還有一點不同就是，它是個惡魔！這就非常有意思了，惡魔流派會不會在節奏上有一些變化呢？

拜戈爾格國王（拜戈魯格國王）

酒鬼觀點：魚人又一次被加強了！這一隨從的出現，有效的彌補了魚人在中期的乏力。「媽媽再也不用擔心我在找到巨殼龍之前就被抬出去了！」

新英雄拉法姆

酒鬼觀點：這只是其中一個新英雄，另外還有2名新英雄準備登場。它們是誰我們現在暫不得知。不過，之前說過，有3名新英雄的加入，就會有3名我們熟知的英雄暫時離開，它們是誰我們同樣不得而知，唯一能確定的是「普崔賽德教授」在這次版本更新中是絕對不會離開的！為什麼呢？我們接著往下看。

老教授也被加強

這是當前版本的普崔賽德教授，在10多個小時之後，普崔賽德教授的技能將從+10攻擊力變為+20攻擊力！

酒鬼觀點：看似被加強了，但是我怎麼覺得意義不大呢？你攻再高，第一個碰到對面帶劇毒聖盾的隨從身上也很尷尬啊！如果是融合怪，劇毒聖盾嘲諷，那就更尷尬了。就算教授的中期作戰能力被加強了一點吧。
