部落衝突：為何資源可以選擇其他途徑隱藏？低本城牆技巧又是什麼

2019-12-17 79條評論 1,213次閱讀 103人點贊

《部落衝突》中的擺陣，既有高深的細節，也有適合萌新玩家的基礎技巧。

其實對於許多的首領來說，一些簡單、明了的基礎技巧才是耐看的。

如果你尚未升級到10本、11本，乃至12本等高階的村莊，不妨先了解一下《部落衝突》基本的擺陣知識。

如果你是一位老首領，也不妨來看看，說不定會感到挺有趣的。

1、萌新世界的資源。

曾經有玩家問道：什麼樣的陣型，可以把村莊守護的最牢固。

於是許多的首領，開始分享出去他們的陣型，其實最好的防守，便是把資源全部使用完。不管是高杯數段位，還是低杯數段位，別人掠奪你村莊的首要前提是看資源多少，沒有資源意味著沒有玩家願意理睬你。

所以，我們可以盡量把多餘的資源使用在建築升級上，或者城牆上。

但是如果自己的村莊沒有空閑的建築工人怎麼辦？

我們可以選擇其他的途徑來隱藏部分的資源：比方說首領短時間下線前，利用訓練營儲備一些需要大量聖水、黑水的兵種和法術，等下次上線後，再選擇取消這些兵種與法術，讓返還的資源回到的我們的資源罐中。

2、低本的城牆。

在9本以下的村莊，或許你的村莊會經常遇到高出數個段位首領的「光臨」。多麼精巧的陣型設計，在高階的科技等級面前都是不堪一擊。

但是我們可以利用的城牆外的不同建築位置的擺放，增強村莊的攻堅難度，儘可能地守護你的資源。

在村莊段位不高的時候，許多的玩家只是把防禦塔簡單的圍繞起來。其實這是一個錯誤的選擇，因為它只防禦了近距離的兵種，依舊讓城牆後的防禦塔，暴露在遠程火力的鎖定下。

所以，低本的時候，我們可以讓其他的建築物，墊放在城牆外，使敵方火力鎖定到這些建築的同時，讓防禦塔有更多的時間輸出與消耗。

在這裡城牆外的建築，也是有一些擺放的小細節。

比如，建築之間相鄰，沒有空格的擺放，非常的佔用數量，並且阻擋的效果與間隔2格的建築是一樣的。

如果，我們在兩個建築間，讓它們隔開2格的距離，且錯位的擺放開來，你會發現兵種走的路線偏向「Z」字，它們會多花費出0.5秒的時間在走位，所以城牆外的建築，最佳的擺放方法是：隔開2格，錯位擺放。

3、資源的細節。

然後是資源建築附近的細節。

當我們休息了一個晚上後，礦井內的資源存會溢滿出來。

比如上圖。這時候我們不妨把金礦，或者聖水井，每2個的放在一起。因為敵方首領通常都是從資源多的一側放下兵種。

如果此時在這兩個資源建筑前，隱藏一些陷阱，兵種一定會踩中它們的。這便是利用了玩家的「貪婪」心理。

4、低本的雙層牆。

而9本以下的村莊，開牆的方法主要還是依賴骷髏。

當你有不少的低級城牆，或者多餘城牆的時候，不妨充分的利用起來。怎麼去做呢？

首先，在陣型邊沿的城牆外，再鋪放一層牆柱，比如圖片中的例子。

這樣子因為有雙層城牆的阻擋，骷髏只可以多跑路線，從兩側的拐角點才能開牆。並且這不限於骷髏，胖子、武神、蠻王、石頭等近距離的兵種，都會繞過短路徑，選擇拐角點開牆。

雖然這只是一些低本村莊的技巧，但是你不覺得挺實用的么？
