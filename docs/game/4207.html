雙道合擊—吞噬沼澤：那時候合擊剛出來的時候，都覺得道道合擊傷害最高的，傷害力是非常足的，而且這個合擊技能特效非常好看，比較有震懾力，特別是在和戰士對戰的時候，那戰士簡直毫無招架之力，法師職業還是有盾可以抗抗，不過基本都是被熟練的道士所秒殺。

道法合擊—末日審判：我覺得這個不算最強，但有的人這樣認為，排第三四還差不多，不過這個搭配很火熱，打寶最佳拍檔，這個技能還是遠程的，比較噁心

雙戰合擊—破魂斬：這個組合比較少，因為戰士刷級本身就很難受，而且很重要一點，裝備很貴，這說明戰士很吃發育，所以玩家很少組雙戰士，雖然合擊技能還是很猛的，但是不划算，所以性價比不高

雙法合擊—火龍氣焰：這個組合也少，因為法師很脆，刷怪還可以，pk就不行了，都不能抗，雖然厲害，但是難以生存下來有什麼用，不過說看見這樣的組合的大部分是土豪級玩家，比較不在乎，就圖個甩技能的快感

戰法合擊—雷霆一擊：首先技能特效比較帥，看起來感覺毀天滅地，但是沒什麼用，因為戰起來不容易靠近，打架沒優勢，很雞肋

道戰合擊—劈星斬：個人覺得這個組合還是比較生猛的，用現在的話來說，就是莽夫，法師如果碰到這個組合，直接一個閃光就沒了，幾乎是秒殺，開盾都是秒殺，因為有道士毒，個人覺得這個是厲害的，無論是秒殺能力和生存能力
