絕地求生：XDD空降RNG強勢崛起，韋神開啟4AM練兵模式

2019-12-09 0條評論 18次閱讀 0人點贊

摘要

隨著絕地求生PGC全球總決賽的落下帷幕，國內絕地求生大賽再度強勢襲來，微博杯的戰鬥率先打響，包括4AM，IFTY，VC等國內強隊紛紛出征，但隨著賽事的即將結束，最讓國內玩家震撼的並不是在國際賽場發揮極其出色的4AM戰隊，也不是內戰霸主IFTY，反而是RNG戰隊了。

隨著絕地求生PGC全球總決賽的落下帷幕，國內絕地求生大賽再度強勢襲來，微博杯的戰鬥率先打響，包括4AM，IFTY，VC等國內強隊紛紛出征，但隨著賽事的即將結束，最讓國內玩家震撼的並不是在國際賽場發揮極其出色的4AM戰隊，也不是內戰霸主IFTY，反而是RNG戰隊了。

由於「三大未成年幻神」之一的XDD這名天才少年的強勢出征，RNG戰隊整體實力瞬間上升一個檔次，6局18殺在國內一眾頂級選手中脫穎而出登上戰神榜，就連對手都在賽後採訪中調侃XDD這人恐怖得有點像外掛了。這也難怪，畢竟當初XDD就是因為槍法與反應意識過於強大一度引來質疑，如今賽場上再度證明自己了。

至於4AM戰隊，他們在經過了PGC大賽的洗禮之後，國內賽事上似乎有點放鬆了。但也正如教練逍遙與韋神所說的一樣，PGC全球總決賽之後，我們覺得今年已經結束了，再加上時差等一系列原因，如今主要都以是調整戰術和練兵為主了。

隨著微博杯大戰的即將收官，在此後由鬥魚直播平台舉辦的絕地求生黃金大賽賽將會再度強勢襲來。眾所周知，絕地求生黃金大賽賽是當前國內絕地求生舉辦賽事歷史最悠久和最具影響力的平台級賽事了。當年大哥A+，韋神，NB211等人都曾在這一賽事中給到粉絲無數高光時刻。

此次鬥魚黃金大獎賽更是有著不一樣的創新，除了再度邀請各大頂級強隊與知名解說坐鎮之外，鬥魚直播平台還聯合華為、中國電信等一系列官方，運用5G直播技術，並推出4K/8K VR多視角流，讓觀眾身臨其境沉浸體驗電競氛圍。在決賽階段鬥魚更是下血本，首創5G+VR直播12月21號-24號上線，觀眾不用自帶VR眼鏡，用手機觸屏操作拉動鏡頭轉換視角，自主選擇觀看不同的現場視角。不僅僅是參賽選手，就連現場嘉賓的一舉一動都將一覽無餘，全部都呈現在VR直播流的畫面當中，這無疑是當前電競直播的一大創新了。

畢竟在絕地求生這一遊戲中，玩家們往往未能欣賞到自己喜歡的選手和隊伍的獨特視角感到疑惑，如今5G+VR直播怕是要讓黃金大獎賽再度掀起一番吃雞的浪潮了。
