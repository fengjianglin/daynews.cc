下半場Uzi是最後一名對戰選手，對陣的是18年獲得Solo King的Caps，掏出本命VN的Uzi前期雖然佔優，但Caps的圈子媽還是抓到一波機會，成功拿下一血。

隨後Uzi打得都比較穩重，畢竟再死一次就沒了，回家後直接摸出了紫雨林之拳。而Caps就比較激進，出完幕刃後還買了個大紅葯，不過最後還是Uzi技高一籌，成功擊敗Caps，挺進四強賽。

下半區的ThyShy在對上Mikyx後，TheShy的AKL全面壓制對面龍女，多次逼得對方守不了塔，最後TheShy在完成第一次單殺後成功推掉了敵方一塔，成為了首位爆塔結束比賽的選手，Shy哥不愧是操作怪。

目前挺進四強的選手分別為TheShy、Levi、Bwipo和Uzi，LPL賽區在4強中獨佔兩席，這次的Solo到底會花落誰家？TheShy能否有望奪得一次Solo King呢？
