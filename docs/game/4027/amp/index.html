羈絆詳解：

解析：4影陣容作為T1級別陣容一直深受大家的喜愛，但是很多人對於影陣容後期優化有很大的誤區，就是將影陣容優化成為3三召喚、3地獄火等無用羈絆。

影陣容最大的缺陷就是防禦力過低，所以影陣容後期應該將思路集中在如何加強防禦力上，而秘術雲霄就是彌補這一缺點的最好選擇。

前期我們正常拿4影3召喚（3地獄火）等羈絆過渡，隨後上8人口找JS，同時在這期間娜美、索拉卡、風女、亞索，經濟好的話就追2級，經濟不好留一隻當作掛件就行。

找到JS立刻優化為上圖中成型羈絆4影4秘術2劍士2雲霄，即可做到陣容攻防兼備，強力輸出的同時還能防止迅速被秒的情況發生，絕對是版本上分最佳選擇。
