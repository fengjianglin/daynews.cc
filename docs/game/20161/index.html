文字生存手游《黑色倖存者》12月18日開啟刪測！體驗另類大逃殺

2019-12-18 790條評論 3,540次閱讀 307人點贊

《黑色倖存者》是一款策略競技類手游，也被玩家稱為是一款文字恰雞遊戲，獨樹一幟的風格在海外受到大量玩家的追捧，值得開心的消息，《黑色倖存者》於12月18日開啟刪檔測試。

《黑色倖存者》的背景模式與大部分生存手游略微不同，玩家進入遊戲後，在不斷縮小的地圖中，尋找物資，同時擊敗其他玩家以及野怪，生存到最後獲得遊戲勝利，道具的配置是遊戲勝利的關鍵，但是，作為一款文字描述戰鬥的恰雞遊戲，更加註重策略以及隊友之間的配合，這就是文字大逃殺。

《黑色倖存者》可以說是一款將ACG跟文字大逃殺相結合的手游。這款遊戲的畫面，以及配音真的讓熊大感到驚艷，看過官方PV的玩家應該和熊大有同樣的感覺，這款手游無論是畫質還是BGM，都讓人眼前一亮。遊戲最新的對戰玩法，採取3V3模式，操作界面也十分整潔。遊戲對每個戰鬥角色的塑造都很豐富，戰鬥技能都各有特點。

進入遊戲，玩家需要在地圖中搜尋物資、擊殺野怪、擊敗對手。《黑色倖存者》區別於大部分生存手游，在遊戲中得到的戰鬥經驗，以及整個遊戲所得的活動經驗，都是團隊共享的，三人對戰比起單人匹配來說，更能顯示玩家的技術性，所以遊戲更強調於角色技能配合，以及玩家之間的交流。

在《黑色倖存者》里隊友尋找的物資，同樣也是共享傳遞的，三人對戰中，三個玩家可以同時使用一件回血的裝備，這對於生存遊戲玩家來說，意味著藥品的消耗不再是限於一次性了！所以隊友落單對遊戲的成敗也十分關鍵，因為可能存在關鍵的物資搜尋，導致反殺的危險。

如果是沒有戰略技術跟配合的新手玩家，很難玩的明白，同時相比同類文字大逃殺手游的無腦點擊，你會覺得《黑色倖存者》有著更多元的豐富性。不得不說這是很有新意的玩法，《黑色倖存者》中的槍械、道具以及角色屬性設定，也是我們可以去探究的重點。在遊戲里武器的熟練度，會決定武器的攻擊力，此外，補給類物品除了常規設定外，裡面還給玩家提供自己做藥品的合成設定。

《黑色倖存者》有著二次元類遊戲「標配」的精美立繪、優秀配音，又有著十足刺激且快節奏的生存遊戲玩法，在以眾多生存遊戲中堪稱一股清流。遊戲於12月18日開啟刪檔測試，快和熊大一起享受流暢的文字恰雞體驗!
