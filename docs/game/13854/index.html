Uzi力壓肖戰王一博，登頂微博年度人物，宅男終於能贏過宅女了？

2019-12-12 396條評論 2,152次閱讀 982人點贊

一年一度的「微博之夜」活動正在經歷緊張刺激的投票環節，相比起往年，今年的「微博之夜」顯得格外有意思，因為今年不再單純是娛樂圈的龍爭虎鬥，電競圈以強有力的姿態殺了進去。截止目前，英雄聯盟RNG電子競技俱樂部選手Uzi力壓肖戰王一博這對經典CP，成功登頂「微博之夜」年度人物榜首。

按照前幾年的經驗來看，這幾乎是一件無法想像的事情，電競圈居然能夠將娛樂圈的壓制住，難道Uzi的粉絲戰鬥力真的有這麼恐怖嗎？其實不然，我們來仔細分析一下會發現，目前「微博之夜」年度人物評選項目上，得票數前10的人物中僅有3位電競圈人士，其他均為娛樂圈明星。

雖然看起來電競圈佔據了很大一部分比例，但實際上從總得得票數來看，娛樂圈遠遠大於電競圈。而為什麼Uzi能夠力壓肖戰、王一博這些當紅小生成功登頂呢？其實這並不單純是Uzi粉絲的力量，而是整個電競圈一同發力，以Uzi為代表，把他推上了巔峰。

相比起圈子更大的娛樂領域，電競領域其實面對外部「敵人」的時候顯得更加團結一些。從人數上來講，電競愛好者與飯圈粉絲完全無法相比，但有一個很現實的問題就是，電競圈Uzi能夠成為代表，但是娛樂圈卻各自佔地為王，甚至明星粉絲之間還存在一些芥蒂，因此很難做到萬眾一心。

反觀電競圈就完全不一樣了，遊戲日報從各種渠道了解了一下，為了讓Uzi登頂這個榜單，不僅是普通電競玩家，甚至很多解說、職業選手都紛紛發動自己的力量，號召大家一起為Uzi投票。也是在這種全民參與的情況下，Uzi才能夠獲得如此之多的得票數，也不算是一件意外的事情。

正如王一博的粉絲在之前看到自家愛豆即將被Uzi超越時，就在超話中呼叫支援時的感嘆一樣：「這群宅男的戰鬥力太強了吧」。對，其實從本質上來看，這次「微博之夜」年度人物投票，就是愛好電競的宅男與喜歡追星的宅女之間的一場正面對決，當然這些不同陣營的人會不會反水，可能就只有他們自己知道了。

雖然目前Uzi成功登頂，但實際上距離這個「微博之夜」投票結束還有很長一段時間，到底是宅男們的戰鬥力更持久，還是宅女們更瘋狂，讓我們拭目以待吧。
