街機大神一人雙槍打兩台《化解危機2》

天天要聞
2019-12-23

上周，一位cosplayer兼街機大神Eddie Esguerra突然在推特上爆紅，DC漫畫的編輯Andi Khouri發表了一段視頻，視頻中Eddie Esguerra用矯健的身姿和嫻熟的槍法，一個人拿兩把槍同時打兩台《化解危機2》，中二指數報表，立刻獲得170萬點擊量。

Eddie Esguerra是一位光槍射擊類街機遊戲大神，不僅槍法准，在各處街機廳刷最高分，而且更重要的是玩法中二，看他打手槍比自己玩還帶感。

《化解危機》是1995年首次推出的自動前進式光槍射擊遊戲，獨闖的掩體機制讓玩家除了射擊以外還可以用腳步與遊戲進行互動。玩這款遊戲不僅需要反應速度，也需要背板，所以也非常適合用來裝逼，比如下面這兩位：

Esguerra第一次玩到《化解危機》還是在90年代，當時他大概只有5歲，跟家人一起去拉斯維加斯的「海市蜃樓賭場」（ Mirage Casino ），「一個陌生人給了我一個硬幣，於是我就玩到了《化解危機》的第一關。我記得我第一關沒過就死了。那款遊戲對當時的我來說很難，但也很刺激。從那以後我就開始常年玩。」

20年後，Esguerra終於有機會成為《化解危機》的大神了。他就讀了洛杉磯社區學院，發現馬路對面就有一個街機廳，正好有《化解危機》和其他幾款光槍射擊遊戲。「下課以後我就會去玩好幾個小時，玩《化解危機》《死亡之屋》《沉默的狙擊手》《Police 911》」他最開始給自己加戲是在《Police 911》里，因為這款遊戲是追蹤玩家動作從而換子彈躲子彈的。而在《死亡之屋》里，他為了給自己增加難度，一邊移動一邊設計。自從他能夠一命通關《化解危機》以後，就開始研究雙持了，同時也開始給自己加戲，增加挑戰。

「我認為大家願意看我打是因為他們從沒見過有人雙持通關《化解危機》，可能他們從沒想到過《化解危機》還可以一個人雙持並且不死通關。」

Esguerra雙持通關《化解危機》通常都是同一時間只在一塊屏幕上開槍。因為遊戲設計中兩名玩家通常有一名是被火力壓制的，這時候另一名玩家可以較為安全地瞄準射擊。所以雙持似乎可以稍微降低遊戲難度，但卻也增加了背板的難度。
