元氣騎士：2.4版本武器調整，雷劍上榜死星暗削，神器就此隕落

天天要聞
2019-12-22

距離上次《元氣騎士》大版本更新已經過了近三個月時間，作為一位熱衷於玩法開發的資深老玩家，老宅無時無刻不在期盼新版本的到來。可是自從十月份BOSS風格的英雄專屬皮膚上線以後，涼屋就發布公告表示準備進入不限期的「閉關修鍊」。

至此一晃就是將近三個月，正當老宅一度懷疑涼屋大大是否準備暫停《元氣騎士》一心準備新遊戲的時候，2.4版本隨著聖誕節的腳步悄然而至。本次更新的主題除了4款全新的日系皮膚以外，涼屋對11種武器的強度調整，才是玩家最為關注的焦點。俗話說得好「一代版本一代神」，那麼誰又藉此機會強勢出線奪得神器之名？誰又慘遭削弱至此一蹶不振呢？就請跟隨老宅的腳步一同進入本期：新版本武器調整解析。

  • 雷劍：屬性變動、強化模式更改

作為《元氣騎士》中的最強近戰，只要有雷劍在手，就可以無往不利所向披靡。雖然雷劍對於大多數玩家來講並不常見，可是對於那些家底豐厚的老玩家而言，也只是花費點材料的問題。所以就綜合實力來看這件武器的強度實際是超標的，削弱也是勢在必行。

調整解析：涼屋並沒有直接削弱這把神器，而是調整了它的使用方式。藍耗+2暴擊-10雖然降低了武器的使用頻率，但是雷劍對於天賦的加成卻又再次提高了累計傷害，因此個人認為，本次調整中上榜的雷劍仍然穩坐神器的寶座。

  • 死星鐳射：暗削

本次對死星鐳射的調整，從表面上看雖然只是降低了10點暴擊屬性，但是真正的暗削卻是傷筋動骨的。由於涼屋取消了鐳射武器的子彈彈射能力，所以讓原本可輕鬆達到2倍傷害的死星實力大減，所以個人認為，它已經無法回到往日的輝煌，一代神器就此隕落。
