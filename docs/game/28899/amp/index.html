王者榮耀：全新SNK限定亮相，玩家公布小技巧，首周僅需660點券

天天要聞
2019-12-21

在王者榮耀中，從很早版本開始，就有一些比較固定的設置，比如SNK英雄都屬於免費類型的道具，因此每次上架新角色的時候都備受期待，可惜的是相關的新英雄已經沒有繼續上架的意思了，也就沒了免費新英雄。不過最近官方再次公布了一個新的信息，已經拿下了SNK英雄的皮膚製作版權，由於之前的SNK英雄是免費的，很多玩家也產生了一種誤解，其實這些新製作的皮膚並非免費類型，而是需要像普通皮膚一樣消耗點券。

從目前公布的新皮膚信息來看，皮膚的製作還是非常不錯的，品質肯定比一般的史詩皮膚更高一些，不過皮膚的品質卻也只是史詩級的設定而已，因此皮膚性價比會高一些。在官方公布的信息中，認定娜可露露的這款皮膚是第一款SNK皮膚，這恐怕是忘記了之前已經出了一款宮本皮膚，也是SNK限定。

這兩者不同之處，應該就只有是否屬於免費皮膚了，但皮膚設定的不同也意味著皮膚的品質截然不同，宮本武藏的霸王丸皮膚雖然是免費的，但是皮膚的品質卻只有勇者級。確定娜可露露的新皮膚並不免費之後，很多玩家也比較好奇皮膚的價格，這也是如今已經確定的內容，最低660點券就能拿到手。

榮耀戰令中可以得到50點券的卡，然後史詩皮膚首周都會以710點券的價格上架，這樣一來最低只需要660點券，相比那些設定為588點券的勇者皮膚性價比也不算低。不過在選擇皮膚的時候，還是需要記住一個重要的點，那就是皮膚並不能增強玩家們的實力，因此千萬別有拿到皮膚會變強的妄想。

在娜可露露的SNK皮膚確定之後，不知火舞的皮膚也被曝光出來，皮膚的製作是旗袍風格的，外形的設計其實還算不錯，但這顯然不是真實的皮膚，很大概率是玩家自製的。因為從娜可露露的皮膚就可以看出，這種類型的皮膚為了讓背景故事很合理，必然不會出太奇怪的風格，三款皮膚的風格會接近。

但儘管這些皮膚並非官方製作，但有一點事可以確定的，那就是SNK限定皮膚總共有三款，而且會在最近幾個月陸續上架，皮膚的品質也可以確定為史詩級。根據目前公布的一些信息來看，這些皮膚很大可能會設定成有標籤卻不會限時上架的皮膚，也就是說隨時都有入手的渠道，當然最終的設定還是需要等月底娜可露露皮膚上架，才可以確認。
