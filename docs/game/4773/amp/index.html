遊戲中我們是朋友，聊天侃地，在這裡我們可以無拘無束的發言，不會有任何人阻撓，還有大家最喜歡吐槽的小編，請把口水收集好，隨時準備和小編一起吐槽！

魔獸世界懷舊服低級不能刷金？網友說這地方適合小號來，那就是去凄涼之地，裡面有一個坐享其成的任務，然後去刷貝殼陷阱，有很多額外的附加物品，雖然賺得少，但是對於玩家的等級沒有太大的要求。

一個小時七金，36級LR，感覺不划算，一個小時就是包里那些東西，貝殼任務交完、垃圾賣完，得了四小格經驗、10片薩滿下水呼吸用的材料閃光的魚鱗、11條血肚魚。至於魚竿，一個都沒有！

很久以前就玩過這個遊戲，以前就是玩到大半夜，懷舊版遊戲確實很好，場景畫面非常逼真。

其實我覺得如果大家是想要刷金，這個時候還不如最初練個法師，千金馬都不是問題。

32級LR，刷了一個小時，不到6G，主要是要清理巡邏的怪，或者開出來的怪，打怪慢，耽誤時間。其它還好，人也不算多加上我，4個玩家，一個聯盟小德。

其實想要刷金，還是滿級比較好，如果大家都滿級了，高級地圖隨便打打怪就不止這些了。不要在低級地方浪費時間 ，早升級，滿級很容易賺錢，我40多盜賊光刷怪練級都比這多。

魔獸世界懷舊服法爺掙錢最速度，手法差點的也可以帶血色，無非就是效率低點，一個小時30g，組滿4個寶寶也能掙120了。還有人說自己兩小時差不多300G，就是去教堂和血色帶老闆。不過現在血色是所有能單刷裡面最難的。
