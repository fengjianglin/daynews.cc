DNF打團為何要求越來越高，速度越來越快？

2019-12-23 1,657條評論 2,635次閱讀 1025人點贊

不得不說，現在玩家們的心態無論是對於小遊戲，還是團本，都是一心的想節省時間，追求速度。

這就導致了很多團長給團隊隊友設置了較高的團本門檻。絕大多數的細節都有，類似春節寵物，寵物裝備，光環和90版本畢業附魔等都已經齊全，不過裝備上全無紅字，面板較低，卻被定義為「划水」「醬油」「沒傷害」「沒打造」。

實際上，如果大部分細節都有的情況下，在普雷團本的輸出傷害其實是並不差的，就算不是強力，也不至於淪落到是一個「醬油」的身份。

但是因為玩家們都追求快速通關團本，所以這種明明可以參與團本的角色，卻在找團隊這關浪費了極其多的時間，甚至有可能兩個多小時才加入一個團隊。

其實，在早期的團本，安圖恩時期並不是這樣的。那時候的團隊講究的是隊友的配合，對怪物的機制的熟練程度以及副本打法的理解深度，當時對於裝備和傷害的要求不像現在這樣有著過高的需求。

從盧克版本末期開始，玩家們對於團本的通關時間都追求更快，對裝備都追求更強力，以至於接下來的每一個團本，加上遊戲本身的打樁機制，都是追求高打造高傷害，花的錢越多，打造才會更好，傷害也會更高，團本通關時間就會更高，這樣的理論慢慢演變為絕大多數玩家對於隊友的選擇的標準。

一、是什麼原因導致了這樣的結局？

細細想來，如今會出現這樣的現象，其實也是必然。這個問題從安圖恩時代就鋪墊好了，而後在盧克和超時空漩渦中逐漸爆發出來。

二、絕大多數還堅持這款遊戲的玩家都是老玩家。

玩家們都長大了，曾經那個對遊戲津津樂道的少年也成長為了青年，也許還是中年，對於遊戲，堅持著還未放棄的也許僅僅只是因為一份情懷，但是不能如以往一般把過多的時間都投入到遊戲里，絕大多數時間的對象還是工作，家人，對於團本，能越快打完自然是越快打完最好，節省出來的時間還可以用於其他方面。

三、角色數量的增多，導致團本所需要的時間增多。

安圖恩團本開放初期，當時能參與的玩家的角色並不多，可能只有一個，或者兩三個，玩家們對於團本充滿著興趣，即使角色不多，但就是感覺團本其樂無窮，也不會把團本當成是一個任務來看待。

對比當前，玩家們卻只是把團本認成一個任務來對待。

也隨著版本的更替，玩家們能參與團本的角色在不斷的增加，對於團本所需求的時間也是越來越多，玩家自然就會想要提高團本通關速度了。

就好比是小學的一道公式，速度x時間=路程，既然路程不變，速度提升，相對應的時間也就減少了。

過高的門檻也並非一無是處，最起碼可以有效降低炸團率，保證效率通關團本。

但壞處也是非常多，這裡就不一一贅述了，相信大多數玩家都是團本高門檻的受害者之一。
