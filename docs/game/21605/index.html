CFS丨全回顧：中國隊無緣總決賽 VG連勝奪冠 dgz一戰成名

2019-12-19 144條評論 1,966次閱讀 214人點贊

CFS世界總決賽從小組賽、八進四、半決賽到最後的總決賽，經歷了嚴酷的對戰，從小組賽的bo1賽制，到八進四、半決賽的bo3賽制，再到最終總決賽的bo5，數場酣暢淋漓的較量輪番上演。今年的CFS，隊伍CFer來說，有激動亦有不甘。最終VG戰隊成功奪得本次CFS世界總決賽的冠軍，今天我們就一起來回顧下今年的CFS世界總決賽。

【六大賽區十二支隊伍集結 為榮譽而戰】

11月25日，CFS世界總決賽小組賽抽籤儀式準時上演，來自六大賽區的十二支隊伍被分在A、B、C三個小組。分別是A組：ANUBIS GAMING、LCK、ALL GAMERS、BAISHA GAMING，B組：VINCIT GAMING、GOLDEN V、KINGZONE、SUPER VALIANT GAMING，C組：IMPERIAL、BLACK DRAGONS ESPORTS、LAZARUS、ALL TALENTS。而從小組分布情況來看，中國和巴西將提前遭遇內戰。SV所在B組對手實力十分強勁。開賽當日，土耳其的LCK戰隊因為簽證原因無緣本次盛會。

【小組賽精彩紛呈 八強中國巴西佔六席】

小組賽bo1的賽制讓人十分擔心，AG戰隊先後戰勝白鯊的ANB，成功率先突圍晉級八強。B組VG戰隊關鍵戰戰勝SV同樣率先晉級八強。C組BD實力強勁以小組第一身份晉級。隨後敗者組的較量中，白鯊、SV、IMP、GV和ANB戰隊紛紛晉級，八強誕生，中國和巴西作為最強的兩個賽區至此無一隊伍被淘汰，紛紛晉級八強。

【八進四白鯊無緣半決賽 中國巴西四強各佔兩席】

八進四的較量首先在AG和IMP之間展開，首局IMP率先拿分的情況之下，AG連扳兩圖，率先晉級四強。隨後白鯊在對陣VG的較量中，沒能戰勝VG，成為首支被淘汰的中國隊伍。第三場BD對陣GV的較量，BD穩紮穩打成功闖進四強。SV最後出戰對陣ANB，雙方實力差距明顯，成功晉級。

【半決賽中國隊失利 VG登頂奪冠】

半決賽兩場較量分別由AG對陣VG，SV對陣BD。這兩場中巴大戰引起了極大的關注。首場AG對陣VG的較量中，VG表現更加優秀，槍感在線，戰勝AG率先進入總決賽。隨後SV對陣BD，慘況再次上演，比分雖然最終差距不大，但SV同樣未能取得勝利。季軍賽AG對陣SV，經歷了失敗的兩隊打法都略顯急躁，最終AG獲得本次比賽季軍。總決賽巴西內戰，帶隊mNdS兩圖MVP實力強勁，dgz狙擊仍舊犀利，VG 3-0戰勝BD奪得冠軍。

CFS世界總決賽，對於中國的CFer來說充滿遺憾，中國隊無緣冠亞這個慘痛的事實也讓我們認識到了自己的不足。巴西的崛起絕非偶然，dgz這把狙擊也因為CFS世界總決賽而揚名。初心未改，使命不變，明年我們重頭再來！
