年關將至，對於lol觀眾們來說現在的感覺像過年了一樣，進入今年的下半年首先是FPX奪得S9的冠軍獎盃，奉獻了一場極為精彩的世界賽，隨後lpl的全明星賽也是讓很多觀眾看得大呼過癮，選手們紛紛開始調皮「整活」，不僅選手們玩得快樂觀眾們也看得津津有味，而lpl全明星賽剛落幕，馬上將要開始的是今年的拉斯維加斯lol全明星賽，真是一場接著一場眼睛都看不過來了。

在lpl全明星賽結束後，選手們經過短暫的調整也是火速飛往美國拉斯維加斯參加一年一度的lol全明星賽，本次S9全明星賽將在12月5日至7日舉行，如果說lpl全明星賽只是家門口的小打小鬧的話，那麼S9全明星賽將是「重頭戲」了，全世界觀眾們投票選出的選手匯聚一堂，他們會擦出怎樣的火花呢讓大家期待不已。

就在剛剛，本次S9世界賽的solo賽環節的對陣名單出爐了，Theshy和Doinb在上半區，而Uzi和小天在下半區。有趣的是首輪小天的對手很有趣，是剛剛加入LGD的小花生Peanut，這場下賽季兩位強強打野提前「對決」也是看點十足，另外Uzi不出意外的話將在第二輪與Bang提前相遇，這對「老對手」也是讓比賽充滿了火藥味。

在RNG官博曬出的圖中，我們可以看到「棗子哥」又是一套標準的穿搭短褲+拖鞋的組合，不得不說Uzi真的很喜歡這種「清涼風」啊，看到好幾次Uzi都是這麼穿了，對比下Doinb和小天的羽絨服，不得不感慨「咱就是脂肪多不怕冷！」

在RNG和FPX分別更新選手們到達拉斯維加斯的動態後，IG的粉絲們不淡定了，紛紛「怒斥」官博不負責，能不能及時更新動態，想要看Theshy的照片還要去FPX官博發布的動態中找，還只能看到Theshy的後腦勺，看來粉絲們真的「急」了，Theshy在lpl全明星賽上是solo輸給了Doinb，也被粉絲們調侃「solo賽零勝率玩家」，不知道Theshy這次能不能認真起來，不要再提前下班了！

最後Doinb和小天蹲在機場牆角的照片也令人心疼，對於職業選手們來說休息時間真的很少，在休賽期還要參加各種活動，希望他們能夠好好保重身體，最後在S9全明星賽上玩得開心！
