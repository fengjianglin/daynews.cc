CF選手自嘲：我的房管在打世界總決賽，我卻在家看比賽

天天要聞
2019-12-10

摘要

穿越火線CFS世界賽正在進行當中，來自世界各地的12支隊伍向今年的世界總冠軍發起衝擊，其中中國參賽隊伍有三支，他們分別是SV、AG和白鯊，其餘的隊伍分別來自歐洲、北美、巴西、埃及等地。

穿越火線CFS世界賽正在進行當中，來自世界各地的12支隊伍向今年的世界總冠軍發起衝擊，其中中國參賽隊伍有三支，他們分別是SV、AG和白鯊，其餘的隊伍分別來自歐洲、北美、巴西、埃及等地。

不過在場上有一支比較”特殊”的隊伍，來自北美賽區的冠軍KZ戰隊當中有一名中國人，據悉，這位選手曾經在國內打過TGA，後來因為留學之類的原因去到了北美，北美也有CF的相關賽事，憑藉國內出色的比賽經驗，ying2迅速成為了當地知名的選手，後來還加入到了KZ，並且拿到了今年北美賽區的冠軍，代表北美賽區回國參加世界賽。

對此，白鯊在解說的時候透露：這位ying2還是選手兮默直播間的房管，國兮默也常在背地裡自嘲：我的房管都在打世界總決賽，而我卻在家裡看比賽。

兮默，相信不少小夥伴也很熟悉，國內知名選手，S6賽季的時候在白鯊奪冠，因殘局打崩AG的even而獲封殘局大魔王；在轉會情久之後，於S12賽季第2次獲得聯賽總冠軍，S14回歸白鯊後又一次帶領白鯊戰隊披荊斬棘，拿下聯賽總冠軍，兮默這樣一名選手，能帶動隊伍氣勢，能幫助隊伍拿殘局，可以說是冠軍版圖不可或缺的一員。

但是在今年的席位制中，規定老牌戰隊只能保留三名選手，兮默被白鯊無奈放棄，又一次轉會來到了情久，在今年的CFS預選賽中，白鯊如願獲得CFS世界賽的名額，而兮默所在的情久無緣世界賽。

不過在CFS世界賽首戰中，白鯊被AG大比分碾壓（10-1）時，白鯊的粉絲就想起了兮默：這樣一個為大賽而生的選手，沒能繼續留在白鯊效力，不僅是兮默的遺憾，也是白鯊戰隊的損失，不過這也是賽制下的無奈之舉，據悉，明年兮默可能會回歸白鯊戰隊！

我的房管在打世界賽，我卻在家看比賽！對於這句來自兮默的自嘲，大家是怎麼看的呢？
