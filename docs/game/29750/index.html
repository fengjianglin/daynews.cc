王者榮耀：榜首大神打野選娜可露露，為何不選勝率第一的雲中君

2019-12-24 923條評論 2,179次閱讀 1152人點贊

這是我前幾天在問答上看到的一個問題。

某玩家發問，為什麼那些大佬玩家打野，選擇的是娜可露露這類勝率倒數的英雄，而不是選擇打野位勝率第一的雲中君。

那陣子剛巧撞見我在小號（王者10星）練打野，練得恰好是雲中君這麼個英雄。

因此，對於這個問題我多少有些心得體會。

1.

早期的雲中君強勢在何處？

在回答這個問題前，必須要說說雲中君在近幾個版本的強度所在。

早在S15賽季，雲中君初次上線正式服時，其被動擁有的高額傷害一度破壞了野區對拼的平衡性，且13技能的兩段不可選中，更使得雲中君在GANK中通過合理操作來規避塔傷實現越塔強殺。

其強勢引得眾多玩家不滿，隨後就是官方及時更新，將雲中君的基礎技能削弱了一圈。

雲中君於6月4日的正式服削弱

這一番削弱的勝率影響極為明顯，兩天內勝率下跌4個百分點，在當時也引得諸多玩家的討論。

其實現在回過頭來看雲中君當時的強勢，主要就在於最初的他能肆無忌憚地越塔，通過13技能觸發不可選中來轉移防禦塔的仇恨，並以此來降低越塔的成本。

張大仙視頻中提及的雲中君重置塔傷

這是最為關鍵的要點。

而當時S15賽季的主流環境，由於地圖依舊繼承早期的中心對稱，邊路依舊是戰士對線射手的打法；也正因如此，由於線上敵我戰力的不平衡，使得邊路的對峙線朝著劣勢方後移。

這樣的情況有好有壞，對峙線朝劣勢方後移，意味著敵方射手補刀時站位靠前，有利於遊走位進行線上GANK；而不好的情況是，敵方射手有更多的活動空間，若線上不能直接探明敵方射手的位置，GANK的覆蓋面太廣。

因此，在當時版本中，馬可波羅和公孫離這兩個位移繁多的射手英雄，通過位移技能來有效躲避GANK保證生存，從而避免了因站位靠前而出現「容易被GANK」的情況發生，並就此挫敗了眾多射手，成為了S15賽季的最大贏家。（另一個原因是公馬這兩個英雄的發育期比較短）

那闡述完當時的主流環境，再回過頭來解釋「當時的雲中君為何強度高」？

其原因主要就在於他越塔成本低，且前期傷害不俗；而射手在前中期主要依託於防禦塔來提高生存環境，換句話來說，防禦塔就是射手的最後防線。

可雲中君卻能通過技能來降低越塔成本，視防禦塔於虛無，從而提高GANK的成功率；而前中期射手位被GANK減緩了發育節奏，自然會影響中後期的團戰戰力。

為此，當時的雲中君一經上線就奪得54%勝率，其根本就在於他的技能機制；而官方策劃隨後砍下的那一刀，也可謂是一針見血。

失去了「越塔成本低」這麼一個最大優點，S15賽季末期雲中君一經削弱勝率大跌也是自然。

2.

時隔兩個賽季，被削弱的雲中君為何又重回勝率第一？

S17賽季最大的變動，就在於對抗路與發育路的設立，顛覆了傳統的中心對稱地圖，成為了以河道線為軸的對稱地圖。

這番改動，最明顯的影響就屬邊路的對峙線相比過往，越發向著河道線靠攏。

就發育路而言，敵我雙方的線上英雄都屬版本寵兒的射手位，因為輸出能力充足而在發育路中形成一個線上平衡。

也正因為平衡，才使得這份線上的平衡更容易被遊走位所施加的外力所打破。

而在S17賽季初期，正式服的一份改動將發育路的河道草縮短了350碼，也是為了進一步的保障發育路的平衡，避免外力過於輕易的施加。

這份改動的影響面覆蓋了絕大多數的打野英雄，唯獨雲中君例外。

在雲中君剛上線時，我寫過一篇文章闡述了我的觀點：雲中君最為強勢的特點在於他被動的無視地形，可以實現一些「詭異」的GANK方式。

但在當時，大家都被雲中君的傷害及他越塔成本之低所吸引了注意力，從而導致他的核心特點——「無視地形移動」，被玩家所忽視。

可隨著版本更替，雲中君雖然不再有早期那般低成本的越塔能力，但卻通過「無視地形移動」來實現更穩定的GANK，從而破壞線上平衡來引導對局節奏。

因此，雲中君勝率位居打野位榜首這個情況，也就顯得自然而然。

沒有哪個打野英雄能像他這般輕易的干預線上平衡，更沒有哪個打野能像他那般輕易越過視野分布來展開GANK。

這是雲中君於當前版本最為契合的特點，而他不高的操作門檻及收益，成就了他登頂打野位勝率榜首。

3.

那大佬玩家為何不用雲中君，而是用娜可露露？

不管在哪個區服，翻看榜首那些大佬玩家的常用英雄，都會發現他們在打野英雄的選擇上，更傾向于娜可露露，而並非勝率榜首的雲中君。

從他們的強勢點切入來思考問題，兩個英雄相比較，娜可露露強於爆發，雲中君強於GANK，這是顯而易見的。

而假如從他們的弱勢點來切入，娜可露露疲於前期發育，雲中君則在後期團戰中難以發力。

綜合來說，娜可露露屬於偏向於成長性的打野英雄，而雲中君則是屬於前中期強勢，越往後價值越低的「高開低走」型英雄。

結合英雄的特色，這兒就有必要再說說中高分段的最大不同。

高分段在前中期的對線中，人頭爆發相對少數，線上對線時雙方玩家都更加的謹慎；且由於優勢到勝勢的轉換集中於對局10分鐘的大小龍刷新，使得雙方射手都有機會發育成型。

而其中引導局勢的往往是那一兩波關鍵的團戰。

至於中低分段，在前中期的對線中，線上玩家在打法上容易腦熱，很容易出現線上崩盤的情況；而中後期也很容易因為個別玩家的不合作，導致節奏一拖再拖，最後在某一波團戰中被人抓住了空子，一波翻盤。

因此，中低分段的打野玩家，要想更好的穩住對局節奏，就需要在前中期擁有對線上的威懾力，決定勝負的往往在於「哪邊的射手先發育成型」。

那麼，前中期強勢的雲中君就更好達成這個目的，利用頻繁的GANK來限制敵方射手發育，從而引導局勢。

而高分段中，玩家的整體意識相對完善，在射手被頻繁針對時，隊友會將重心轉移到射手身上來確保他的順利發育，為此前中期強勢的雲中君很可能因為敵方過多的保護，而白白浪費了自身前中期的價值，從而拖到了自己相對弱勢的後期。

可娜可露露不同，就算到了後期團戰，他對於後排C位來說依舊是一個巨大的威脅。

因此，高分段選擇娜可露露的一個關鍵原因，在於高分段玩家更看重「團戰中哪邊射手的輸出環境更好」。

前中期的疲軟，可以藉由決勝點的推移（10分鐘大小龍刷新）來苟住發育；而後期的疲軟，原因在於英雄的選擇，這是沒有操作空間進行彌補的。

4.

當前版本的打野位更傾向於圍繞C位作戰。

就像雲中君的推薦出裝中，有一套藍領打野式出裝，以末世宗師為核心，加以反甲魔女提生存。

其中目的，無非就是提升在團戰中的存在感，避免一身輸出裝使得身板羸弱入場就消亡。

雲中君的推薦出裝

這也側面的應證了雲中君在後期團戰的乏力，必須要藉以防禦裝來體現自身價值。

而娜可露露是不存在藍領的可能，但他是從另一個角度圍繞C位來展開作戰，那就是先於己方射手陣亡前，將敵方射手扼殺於搖籃。

雖說現如今切後成本抬高，一切刺客類英雄都不如以往強勢。

但大佬玩家之所以稱之為大佬，就源於他的操作與意識比常人更加完善。

與其將勝負依託於隊友，不如合理髮揮自身的理解來決定勝負。

或許，這份大佬玩家的自信，也是他們選擇娜可露露而非雲中君的一點原因吧。
