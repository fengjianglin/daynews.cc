2019-12-16

壹周遊聞為遊戲日報推出的一周資訊匯總欄目，聚焦與遊戲行業有關的大事小情，旨在以簡短語言提供全面的遊戲資訊。主要分為遊戲概念股、行業要聞、遊戲資訊三部分。後續也會根據實際內容調整相關匯總，敬請期待。

【遊戲概念股】

1、心動網路正式於港交所掛牌上市，首日收盤價11.86港元，較招股定價11.1港元增長約6.85%。截至發稿前的最近交易日（12月13日），心動公司收盤價為12.96港元。

2、A股上市公司新海宜科技董秘在回答投資者問中表示，公司持有7%股份的遊戲公司深圳望塵科技的港股上市計劃正在如期推進，前期的ODI備案（境外投資備案）已經完成。

望塵科技成立於2013年12月，是一家體育遊戲和科技公司，核心成員來自夢工廠、微軟、騰訊等。公司2014年推出的足球經理類遊戲《足球大師》，截至發稿前位於iOS暢銷榜體育類遊戲第7名。

3、游久遊戲發布《關於公開掛牌出售部分房產的公告》：擬將其位於上海市浦東新區的8套房產公開掛牌出售，若在2020年前交易成功，公司將會新增8000萬凈利潤，從而有望實現扭虧為盈。

據游久遊戲財報顯示，在2017-2018年以及2019年前三季中，公司的虧損分別為4.22億元、9.05億元以及1127萬元。若2019年繼續全年虧損，依照相關規定，游久遊戲將會被暫停上市。

4、中文在線發布公告稱，公司擬將持有的晨之科100%股權出售給鋼鋼網，交易對價為3.24億元。而在此之前，中文在線獲得晨之科100%股權的交易總價超17億元。

5、上期的《壹周遊聞》提到，有消息稱創夢天地正就以約14億美元的價格收購同為港股上市遊戲公司樂游科技控股。次日創夢天地便發布公告表示將短暫停牌，並就有關樂游科技股東可能出售的69.21%已發行股，宜訂立諒解備忘錄。

6、港股上市公司雲遊控股預估公司2019年度將繼續產生虧損，金額不超過2018年度的虧損3.2億元。主要原因系收購空中網旗下的VR企業北京西瓜69.84%股權而產生商譽及可辨認無形資產的減值1.17億元。此外在雲遊控股運營的小額貸款業務中，有10筆企業貸款處於預期狀態，其中七位未能聯絡到的借款人貸款合計約4100萬元。

【行業要聞】

1、國家新聞出版廣電總局公布12月第2批遊戲版號審批結果，本次共有42款遊戲過審，均為移動遊戲，其中休閑益智遊戲21款，依舊佔據半數。

2、據北京日報報道，北京市委宣傳部副部長、北京市新聞出版局局長王野霏發布了《關於推進北京遊戲產業健康發展的若干意見》，確立圍繞將北京建成「國際網路遊戲之都」的總目標。力爭到2025年，全市遊戲產業年產值突破1500億元。

3、《上市公司分拆所屬子公司境內上市試點若干規定》於2019年12月13日正式發布，在設置7項硬核門檻的同時，明確分拆上市行為監管要求，打擊概念炒作與「忽悠式」分拆。

4、遊戲飾品交易平台C5GAME所屬公司杭州星巢網路科技有限公司已完成一輪千萬級融資，投資方為杭州道生投資管理有限公司。公司聯合創始人謝力揚對遊戲日報透露，公司的新一輪融資將會用在平台海外布局方面。目前C5的海外市場以東南亞為代表的亞洲市場為主，東歐與北美地區正在陸續推進。

5、光子（重慶）美術研發基地項目正式簽約落戶兩江新區，據悉該項目是騰訊遊戲深圳總部之外的最大研發支持平台。目前，光子工作室已在西南區域與30餘家遊戲美術內容製作企業達成合作。

6、蓋婭互娛與俄羅斯知名獨立遊戲開發商、《戰爭雷霆》開發方Gaijin Entertainment簽署戰略合作協議，雙方合併攜手尋求在國際資本市場上市。

7、互動影視內容公司互影科技完成億元級的A輪融資，由閱文集團領投，疊紙遊戲跟投。

8、任天堂Switch國行正式發售，首發當日，任天堂的收盤價為46370日元，創下自2018年5月18日之後的最高股價。

9、QQ小遊戲中心舉辦開發者峰會，據介紹，QQ小遊戲中心目前已接入500多家開發商，上線近3000款小遊戲，最新DAU為2000萬。用戶方面，男女佔比分別為57%與43%，二三線城市為主。此外，除了流量與買量等扶持，QQ小遊戲中心還表示會為開發者開放獨特的生態，如社交分享與邀請、好友PK榜、服務通知/訂閱消息、小遊戲禮包等。

10、針對「多玩論壇即將關閉」消息，多玩網回應稱，此次調整並非「關停」，而是全面改版升級。即將原有的遊戲資訊與遊戲專區等內容，改版為遊戲直播、遊戲視頻、開黑玩法、蝦哥論壇等。
