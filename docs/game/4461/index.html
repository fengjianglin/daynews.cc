DNF12鑽石增幅卷開出有多難？虎牙狂人：有土豪找我50萬元包

2019-12-07 0條評論 16次閱讀 0人點贊

摘要

DNF開盒子可以說是玩家們最喜歡的其中一個環節了，特別是盒子加入好東西的時候，玩家們更是熱情，希望自己能其中的幸運兒，能小賺一筆。最近盒子加入的幻化武器和白金徽章就大受玩家們歡迎了，吸引了一大波玩家去開盒子，有部分玩家則選擇了找DNF主播們包盒子，用一定價格開出自己想要的東西。

有很多水友就找到了虎牙狂人包盒子，這其實也是很正常，畢竟狂人是出了名的倒霉小王子，另外找狂人包盒子也是福利多多，出13的裝備強化卷就可以獲取額外的10000元獎勵，說實話出13強化卷的幾率還是挺大的，有水友找虎牙狂人包盒子就連續兩天都出了13的強化卷，小賺了一筆，號主可謂是笑不攏嘴了。想要包盒子的小夥伴，不妨關注下虎牙狂人。

除了13強化卷，出12的鑽石增幅卷狂人就以一萬元一張的價格回收，要是一次出雙倍12鑽石增幅卷就白賺了2萬元了，當然能獲取的幾率是比較低的。至於開出12鑽石增幅卷有多難，虎牙狂人是這樣說的，狂人表示有土豪曾找他包12鑽石增幅卷，並且是50萬元的高價，但是自己並沒有接，一個時間比較長，直播中沒那麼多的時間，另外一個就是50萬元未必能開出12鑽石增幅卷。

50萬元能否開出12鑽石增幅卷我們不知道，但是我們知道歐皇一個盒子就能開出，至於非酉玩家真的很有可能百萬元都未必能開出。各位小夥伴，你們認為多少包12鑽石增幅卷合適呢？
