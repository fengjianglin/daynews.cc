隨著時間的推移備受關注的英雄聯盟LPL全明星周末終於是落下帷幕了，回顧本次的比賽最讓人印象深刻的莫過於老IG與老WE的傳奇賽大戰了，由於賽前發生了那種事使得這次的比賽變得火藥味十足，最終在經過兩局的大戰之後，老IG與老WE打成1比1平，而主播姿態也憑藉著優秀的發揮拿到了本次傳奇的MVP，賽後也是引發了不小的熱議。

鑒於大家最近都在聊全明星賽這件事，近日德雲色主播笑笑小孩游神也是在直播中回憶起了本局的比賽，小孩游神表示這次的傳奇賽自己第一把確實打得不好，但是會造成這樣其實是上了姿態的當，因為在之前的訓練賽中自己和姿態已經約好了，自己玩盲僧姿態玩佐伊或者瑞茲的，結果姿態突然選了一個夢魘出來導致自己的局面變得很尷尬，就很難發揮了。另外小孩游神還表示其實第二把自己有機會拿下MVP的，但姿態一直在叫自己讓人頭，沒辦法只能讓給他，所以就無緣MVP了。

以下為德雲色直播時的原話：

笑笑：說實話偶像比賽沒拉垮的，第二把你把西瓜爆穿了。

小孩游神：一人C一把第一把我本來想玩酒桶的，我想了一下沒必要對面酒桶玩家萬一…我很想贏的話就會選酒桶，我第一局不是不想贏我就想著玩一把瞎子，就開心一下，而且第一局又不握手，嘿嘿嘿。

笑笑：確實，其實第一局NOC就是要配酒桶的。

小孩游神：不是第一把姿態說他要玩佐伊或者瑞茲，後面他上台了之後說了玩NOC，那我也變不了了，因為我前面練的就是盲僧啊，我又沒姿態那麼猛。

笑笑：啊呀，練的盲僧呀姿態變卦了呀。

小孩游神：你問孫哥，我們前面訓練賽的時候他佐伊瑞茲狂玩，我沒辦法兄弟，這個也不是甩確實也是沒玩好，但是我確實小上當了一下，然後第二局姿態很強勢的說要玩卡薩丁，我只能選個前期打野，當工具人而且他一直在喊讓人頭我也沒辦法，總不能搶吧，姿態一直就在那喊讓人頭，其實那把我就有機會拿MVP的如果搶的話，可惜，不敢搶啊我害怕姿態把我拉黑兄弟，他一直在跟我們兩個說讓人頭對吧，跟你也說了。

笑笑：說了說了一直在說。

小孩游神：我們兩個當工具人，而且第二把我不選挖掘機我還是一樓選的，我要是不選挖掘機對面是必拿，對面要是選了挖掘機你想一下我們會被抓成什麼樣，沒辦法的呀總要有人犧牲，總不可能我也選個後期吧，他選個卡薩丁我也選個後期一起在那前面司馬臉。

厲害了原來老IG選人還有這麼多的內幕，這樣看來真的是委屈小孩了，不過有一說一回顧這次的傳奇賽小孩游神確實打得不錯，雖說第一把盲僧一厘米摸眼很滑稽，但第二把的挖掘機還是打出效果了的，現在想想如果第二把小孩游神主動搶人頭的話，MVP還真有可能易主的，所以對於小孩游神直播時的這一番話你們怎麼看呢？
