魔獸世界：工作室紛紛湧入懷舊服，採取什麼措施才能限制住他們？

2019-12-12 356條評論 3,188次閱讀 555人點贊

魔獸世界懷舊服充了一個月卡，還買了一些遊戲幣，根本上沒有以前的味道。魔獸世界團隊副本機制非常消耗時間，和工作上班差不多，懷舊情懷只是想起以前一起玩的朋友之誼，現在大家都上了年紀成家立室，都忙著工作賺錢養家為生存，哪裡還有時間精力長期打遊戲。懷舊服懷的是什麼，就是那個時間環境下的朋友，事情。為了公會藥水在加基森海邊釣石鱗鰻，給兄弟籌錢買貓鼬圖紙，兄弟撥皮，練採集＋挖礦就是為了養一個附魔大濕。當年的大神法師三季稻，還有銅須門事件，開火車，打小紅龍，現在還有這些樂趣了嗎？

DKP團也是有親友，但是團隊有規定，只能進本天賦拍，不可交易，拍到的裝備必須立即裝備並讓團隊全員確認，需要跨甲的職業如狂暴戰，奶騎等必須在掉落本甲職業無需求的情況下才可以出分，且一個人一個次活動只能一需多貪。理論來說一個人一次活動能只能拿一件裝備，排除小德聖騎術士等一次活動畢業，或者沒人要的情況，裝備不好被按在地上摩擦。

大部分人只對個別裝備有執念，這遊戲對現在新玩家來說就是個帶老闆賺錢的遊戲，我帶了三個新人進來玩。現在兩個啥也不幹，每天就帶老闆和打金，周六參加一次公會團，純帶刷天賦什麼葯都不吃，四小時通關黑龍和mc，然後繼續帶老闆，還有一個只參加一次mc，然後就下線。

裝備難度獲取再降低，因為每個職業的很有特點，人們不斷練小號，才有興趣，才更有生命值。別搞得死難，掉落好少，該進場的，都進場了，不玩的也不會再來了，現在就是維持住，現有客戶，難度老高，體驗不好，人氣就沒了。也不見其他純點卡遊戲沒工作室，還是得看遊戲埠的監控和運營商的運營方針。

比如以後加好友一周後才能互相交易不被納入買金範圍，不然低於50g的交易一律當作工作室，加了好友每天也有交易限制，超過也當作工作室處理，雖然極端不過應該能杜絕很多工作室甚至金團。最好的辦法就是，裝備不能交易，金幣不能交易，取消拍賣行，裝備、物品掉落只能個人拾取可見。這樣的話能很好地減少工作室生存空間，但這樣的話遊戲性又會受到影響。
