DNF：旭旭寶寶要哭了，紅眼徹底廢了？劍魂和阿修羅都在加強

天天要聞
2019-12-16

最近體驗服出了一波職業的加強，這一次增強的職業有奶爸，驅魔，死靈，魔道，劍宗，阿修羅等等。值得關注的是這一次的阿修羅可以說是史詩級的加強了，根據測試發現阿修羅整體加強了30%左右，現在打樁排名中阿修羅已經是第一了，可以說阿修羅，天帝，劍魂可以並稱為鹹魚翻身三人組，這三個職業真的是迎來了新的春天。

但是對比之下紅眼和鬼泣就慘不忍睹了，國服中最受歡迎的職業莫過於紅眼和劍魂，可以說紅眼之所以出名也是因為旭旭寶寶的原因，而劍魂之前雖然是下水道職業，可是依舊受到了廣大玩家們的喜愛。國服中最受歡迎的就是鬼劍士了，而現在劍魂已經超一線了，阿修羅這次加強之後很有可能成為幻神職業，之後就是鬼泣這個職業，從幻神直接變成了下水道職業。劍鬼的地位一直還是很穩定的，屬於一線職業。

目前來說鬼泣應該是墊底了，排名老四的就是紅眼，要知道國服第一的紅眼旭旭寶寶可是用了幾千萬將紅眼全身增幅上16的，在明年的增幅活動中，紅眼還要衝擊上17。但是這樣下去紅眼這傷害這麼低，估計寶哥是真的要哭了。原本旭旭寶寶可以憑藉紅葯1755億的傷害排名國服第一的，但是因為劍魂進行了一波加強，導致狂人的第一劍魂的傷害超越了寶哥，雖然狂人的手法不行，但是技術好的玩家可以用劍魂打出2200億的傷害，這樣對比來看，紅眼真的是太可憐了。

不過因為100級版本還沒有到來，之後肯定還會有新的職業平衡，按理說策劃也不會讓紅眼表現太差勁，不要忘了國服的紅眼玩家多到數不勝數，喜歡紅眼這個職業的玩家更是數不清楚。現在來說紅眼這個職業也真的是比較委屈了，但是多數玩家還是很愛這個職業的。在100級版本上線之後一定會有新的調整的，到時候紅眼也有可能會翻身。

目前來說紅眼玩家中也就旭旭寶寶是最可憐的了，他的紅眼可是花費很高的價格打造出來的，結果淪落成為了現在這幅樣子，希望以後紅眼能夠崛起也讓寶哥安心，不必再擔憂之後紅眼的情況。還有就是馬上就要到年底的增幅活動了，在這次的活動中寶哥也要繼續增幅紅眼上17。
