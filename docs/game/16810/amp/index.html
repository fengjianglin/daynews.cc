DNF體驗服：瞎子實測增傷30%+，皮甲成大贏家，輕甲白字雙暴和諧

天天要聞
2019-12-16

「地下城與勇士之小狐狸君愛談遊戲電競」第七百四十五期《DNF體驗服：瞎子實測增傷30%+，皮甲成大贏家，輕甲白字雙暴和諧》，國服新一輪的職業平衡（史派克造神）再次上線，目前瞎子領銜的第四輪幻神職業，再次讓玩家大開眼界！瞎子原本打樁就是幻神級別的，這次再次提升30%可謂是如虎添翼。目前瞎子在體驗服無論是單刷還是組隊，都是排名第一的超級幻神！同時由於BUFF技能刻印，新增了技能攻擊力之後，皮甲成為大贏家！輕甲的白字雙暴擊和諧之後，和其餘的防具差不出5%的傷害了。部分國服特色的職業，皮甲的優勢就更加明顯了。老規矩搞快點，直接上本次體驗服瞎子改版和實測傷害的截圖展示：

DNF體驗服：瞎子實測增傷30%+，

其實從上個版本開始，皮甲就在崛起，當時輕甲和皮甲的差距，對於國服特色BUFF職業劍魂、鬼泣、劍豪而言，差距就在2%左右。這一次體驗服改版之後，所有的白字（普通附傷害、屬強附加傷害）都取消二次暴擊。輕甲和皮甲對於瞎子而言，幾乎不相上下了！而且皮甲的BUFF是前技能提升的，而白字的輕甲對於稱號的要求更苛刻。

皮甲成大贏家：

其實皮甲對於所有的職業，都有著天然的優勢：BUFF技能大幅提升。尤其是國服趨勢就是全職業加入新BUFF（劍魂、劍宗為代表）或者原有技能加入BUFF提升（瞎子為代表）。這樣一來30級的BUFF都能有不錯的提升。白金選擇BUFF，時裝換裝選二覺，保證最大化輸出！所以皮甲就成了本次改版的最大贏家了。輕甲也是變相削弱。

輕甲白字雙暴和諧：

這個和諧是遲早的事情，輕甲一家獨大，主要是國服的屬強要比韓服高出一倍。導致輕甲的屬強白字傷害非常高。這一次取消二次暴擊，後期應該不會改版了，100級就要來了，新的裝備搭配多的書都數不清，平衡不是問題，開發各種裝備之前的搭配，才是玩家的終極目的！
