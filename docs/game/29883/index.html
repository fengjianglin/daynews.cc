Steam 2019 大獎《縱使手殘仍大愛》《傑齣劇情遊戲》

2019-12-23 1,284條評論 2,734次閱讀 542人點贊

前面我們為各位讀者詳細介紹了今年入圍「傑出視覺風格」、「最具創意玩法」大獎的遊戲名單，不知道大家看完以後有沒有親自上手試試的呢？今天繼續為大家帶來「手殘黨」和劇情控的福音，一一介紹入圍今年「縱使手殘仍大愛」和「傑齣劇情遊戲」的那些精彩遊戲。

獎項：《縱使手殘仍大愛》

聽名字就有一種「縱你虐我千百遍，我仍待你如初戀」的味道，我們勤勤懇懇苦練，讓手指變得敏捷靈活，精準掌握時機和走位，在克服重重難關後，那種刻骨銘心的美好，強烈深邃的個人成長感受，讓玩家大受鼓舞。下面一起來品一品五款入圍遊戲。

《雷霆一擊（MORDHAU）》

《雷霆一擊》是Triternion製作發行的一款中世紀多人動作冒險遊戲。創建你的僱傭兵並在野蠻的戰場中廝殺，你將體驗快節奏的酣暢戰鬥。此外，城堡圍攻，騎兵衝鋒等中世紀常見的作戰方式都會在遊戲中一一展現。你將進入一個多達64名玩家的忙碌戰場，作為一個僱傭兵，你可以體驗到殘酷而令人滿意的戰役。許多玩家因為其蹩腳的機翻，遊玩的時候有許多困難，因此不知道怎麼玩。這個時候建議大家多上網搜一搜攻略再上手會容易許多。

《噬血代碼》

《噬血代碼》是由萬代南夢宮出品的一款硬核動作RPG遊戲，在崩壞的近未來世界，玩家將扮演擁有超自然能力的吸血鬼。遊戲中玩家可以選擇一個可以提供標記與支援的可靠角色作為夥伴。玩家的夥伴會追擊敵人並與玩家進行合體攻擊。當玩家HP歸零時，夥伴還可以治癒玩家。有趣的地方是，《噬血代碼》將有多種結局，玩家通關後便可從第二周目開始開啟多周目，隨著周目數的增加，敵人也會變強遊戲難度逐漸上升。

《獵殺對決》

Hunt：Showdown 的高競爭性競技式玩法，將PvP和PvE中的元素相結合，在遊戲中打造獨一無二的緊張氣氛，讓玩家無時無刻不提防著自己的生命、角色和裝備會灰飛煙滅。每場競賽中，最多可允許五支二人團隊出發追尋魔怪。魔怪一旦被尋獲並消滅後，玩家就可獲得一份賞金——但也將立刻變成地圖中其餘獵人的目標。一不留神，就會遭人伏擊，眼巴巴地看著賞金被奪走，慢慢死去。風險越大，回報也越大——走錯一步，萬事皆空。

《遺迹：灰燼重生》

《遺迹：灰燼重生（Remnant: From the Ashes）》是由Gunfire Games製作、完美世界娛樂發行的一款第三人稱生存類射擊遊戲。單人遊戲的難度要遠遠大於多人遊戲。單人情況下倒地=死亡，多人則可以扶起。多人多改裝技能多槍，能分工處理BOSS和小怪，單人模式下則對輸出、走位有較高要求。

《真人快打11》

《真人快打11》是一款血腥到極致的格鬥遊戲，同時它也是一款極具深度的格鬥遊戲，它被稱為「最難卻最適合新手的遊戲」。作為系列25周年的一作，它終於在易上手和有深度之間找到了一個絕佳平衡。再配合它流暢的打擊感和暢快的連續技，玩家很快就能享受到屬於格鬥遊戲的獨有魅力。同時「保姆式」的新手教程又能讓玩家輕鬆意識到從入門到精通的每一步自己應該做些什麼。無論是對於格鬥遊戲的高手，還是喜愛格鬥遊戲卻投師無門的新手來說，《真人快打11》都絕對是目前最好的選擇。

獎項：《傑齣劇情遊戲》入圍遊戲

有時候，只有劇情豐富的遊戲才能撥動心弦，下面要介紹的五款遊戲故事情節豐富無比，像肥皂劇一樣能夠引人入勝，和優秀劇本一樣精緻細琢，能夠讓玩家在遊戲之餘有所感觸。

《瘟疫傳說：無罪》

這款中世紀劇情冒險遊戲目前steam已有千條特別好評，MC均分也達到了不錯的78。遊戲里，我們扮演的角色——阿米西亞，是一位法國貴族，因某天與父親外出狩獵時發現了「老鼠惡魔」的蹤影，不幸引來了宗教裁判所對其家族無情的審判，宗教裁判所根本不聽人話，隨即對其滿門抄斬，阿米西亞悲慘的家破人亡。憤怒悲傷之餘她選擇堅強，帶著自己見面不多的弟弟雨果一起逃命，試圖揭開這一切事情的真相。

逃亡途中阻擋我們的不止有老鼠，還有宗教裁判所的走狗，面對著這般奪命的鼠群和嗜血的惡徒，正面交鋒自然不是明智之選，於是遊戲的玩法就選擇了「潛行+解謎」的模式，讓阿米西亞巧用智慧擊退敵人。

《極樂迪斯科》

首發即在Steam「特別好評」的偵探RPG遊戲《極樂迪斯科》獲得IGN 9.6分超高評價。當前Steam上好評率達到了94%。你將扮演一位落魄潦倒的中尉偵探，身處墮落的港口城市雷瓦科城，這裡腐敗橫行，謀殺案一波未平一波又起，青少年們也整天只想著蹦迪。你在其中或是搜尋線索，審訊疑犯，或是探索這座廣袤迷人的手繪城市，揭露它深埋的秘密。與此同時，局勢逐漸惡化，城中衝突加劇，危機一觸即發…

《孤島驚魂：新曙光》

《孤島驚魂：新曙光》Steam上的好評率達到了86%，不少玩家都誇讚了《新曙光》的進步，可玩性、好玩度上都要優越於《孤島5》，畫面也很好。《孤島驚魂：新曙光》的故事發生在前作的十七年以後。核爆席捲全國時躲入地下得以倖存的人們度過了六年暗無天日的生活，等到重返地面才發現曾經熟悉的世界已經不復存在，恣意生長的植物和異變的動物重新成為了希望郡的領主。經過數年的努力人們建造了自己的據點，然而本來欣欣向榮的繁榮鎮卻引來了瘋狂殘暴的不速之客。此時約瑟夫的時代也已經過去，這個全新的希望郡已不再是邪教的領地，支配這篇領土的是一夥由一對雙胞胎姐妹帶領的攔路強盜，大家必須拿起武器為了自由和生存而戰。

《戰爭機器5》

最近，《戰爭機器5》已經成為了一年多以來第一款擊敗《堡壘之夜》而成為Xbox Live上遊玩人數最多的遊戲。而現在，IGN為《戰爭機器5》打出了9.0分的總評分！再次證明了這款遊戲是多麼的成功。整個世界都在分崩離析，人類對技術的依賴讓他們更快地垮台，而敵人正聯合起來消滅所有殘存的人類。主角Kait必須武裝自己，穿越廣闊、美麗的《戰爭機器5》世界來揭開獸人的起源，並與小隊一起保護重要的事物。在村莊被Swarm襲擊之前，Kait和Del前往一個Outsider村莊尋求一位老盟友的幫助。可能玩家已經猜到了，一場激烈的交火隨之而來。

《貪婪之秋》

這款遊戲在steam上獲得了大多數玩家的好評，在856篇用戶評測中，好評率達到了81%。根據Spiders的介紹，遊戲的核心是角色扮演，並且他們將會傾盡全力來打造這個充滿神秘色彩的世界，塑造具有深度的人物互動，在完成人物和目標的選擇上擁有著極高的自由度，遊戲會提供多種解決問題的方式。據開發者透露，在《GreedFall》中，這個神秘的島嶼擁有自己的生命，並且在不斷發展，玩家們的一舉一動都會影響到同島嶼的關係。

好了，今天的十款入圍2019 The Steam Awards的遊戲就介紹到這裡，如果你對某一款遊戲感興趣的話，可以去Steam官網進行下載和購買，後續入圍名單我們將一一公布。
