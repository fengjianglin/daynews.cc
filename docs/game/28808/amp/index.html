LOL：德杯小組出線預測！RNG深陷死亡之組，EDG出線形勢堪憂

天天要聞
2019-12-21

2019年漸入尾聲，經過了長達一個月的轉會期，各家戰隊相繼確定新賽季陣容。而在春季賽之前的德瑪西亞杯也就成了隊伍們磨合新陣容的「試煉場」。

對於諸多粉絲來說，經歷了很長一段時間的「比賽荒」之後，終於又有比賽可看。官方也順勢推出精彩活動，而首先開始的第一輪就是小組賽競猜。20支隊伍4大分組，究竟誰能從小組賽中突圍成功？

A組出線預測：SN、VG

A組五支隊伍分別是FPX、SN、WE、VG，以及LDL隊伍LGE。

冠軍隊伍FPX並沒有動用冠軍陣容，由二隊FPB主教練yo和教練River帶領以年輕班底為基礎的陣容，參加此次杯賽。

VG作為此次轉會期的大贏家，出手闊綽。在2018賽季啟用新人效果不佳的情況下，轉會期連續引入三冠教頭Kkoma、前iG中單選手Forge、前BLG打野選手Chieftain、前LNG輔助胡建鑫、前EDG AD選手iBoy等人，紙面實力大幅提升。SN此次轉會期同樣收穫頗豐，引入前LNG打野選手SofM組成強力新陣容，兩支隊伍都將以全新陣容征戰德杯。

在當前FPX啟用二隊、WE轉會期雙C離隊，而LGE又是純LDL隊伍的情況下，所以A組大概率由SN和VG攜手出線。

B組出線預測：TES、LGD

B組五支戰隊分別是TES、EDG、V5、LGD和來自LDL的GD戰隊。

相對來說，B組戰隊出線形勢並非十分明朗，但可以確定的是，GD戰隊和V5戰隊出現幾率渺茫。TES、EDG、LGD三支隊伍主力選手均在出戰名單中。

一方面，雖然EDG轉會期動作不大，而且首發AD iBoy離隊，但是Scout和Meiko的存在，讓EDG依然保有一線隊伍的實力。另一方面，TES和LGD均在此次轉會期中收穫頂級打野，補強成功。尤其是TES Karsa的加盟，配合上黃金左手，穩穩的LPL頂級隊伍行列，甚至有望競爭LPL第一中野。可以說，TES此次啟用的陣容，完全就是為了奪冠來的，不奪冠，即失敗。而LGD Peanut的加盟，或許可以改變Kramer獨木難支的尷尬境地。儘管Peanut距離2016年全球總決賽時期的狀態有所下滑，但仍然不失為一名頂級打野。

值得一提的是，此次Karsa和Peanut並非首次交手。2018英雄聯盟季中冠軍賽時期，兩人曾在決賽碰面，Karsa四局「窺屏式」打野可以說是完爆了Peanut，因此，TES基本可以坐穩小組頭名出線。而另一個出線名額則在EDG和LGD之間搖擺。LGD Kramer頂級的後期能力，或許將幫助隊伍拿到這一個出線名額。

C組出線預測：JDG、RNG

C組五支戰隊分別是RNG、LNG、JDG、OMG和來自LDL的YM戰隊。

C組是真正的「死亡之組」，幾支隊伍都是LPL的老牌勁旅，RNG、LNG、JDG和OMG都曾經在德瑪西亞杯中取得不錯的成績，而這幾支隊伍在新賽季中都進行了陣容調整，競爭力很強。

RNG小龍堡的加盟填補了Karsa出走的空缺，在德杯Uzi休戰時期，新簽約的AD Betty出征的情況下，仍不失為一支德杯強隊。而且，儘管隊伍中單選手Xiaohu全球總決賽上的表現遭到詬病，但是其在國內聯賽尤其是德杯這種許多新人參與的比賽里，有一說一個人能力依然處於上游。

JDG和LNG同樣是主力陣容出征。JDG雖然轉會期沒有大新聞，但是憑藉上賽季的陣容，完全具備進決賽的實力。Zoom的頂級個人能力就是隊伍的王牌。而LNG則引入Xx、Maple，在加上聖槍哥坐鎮上路，實力同樣得到提升。

最後，OMG在此次轉會期期間，引入H4cker和Smlz，此前Smlz和icon的雙C組合曾在常規賽階段一度領跑聯賽，這次Smlz的回歸同樣使得OMG紙面實力有了明顯提升。因此，JDG、RNG、LNG、OMG四支隊伍誰出線都不會讓人意外。

而YM雖然貴為「黃埔軍校」和新晉LPL隊伍，但是在面對到經驗更加豐富的RNG、LNG、JDG時，出線形勢仍然不會樂觀。

D組出線預測：BLG、iG

D組五支戰隊分別是BLG、iG、DMO、RW和來自LDL的SDX戰隊。

轉會期期間，BLG宣布FoFo加入隊伍擔任中單。剛剛結束的全明星Solo賽上，FoFo還戰勝了冠軍中單Doinb，而今年的最佳新秀獎也是由BLG的打野國豪拿下，FoFo和國豪組成的中野組合在新賽季可以說是備受期待。

而DMO戰隊這賽季有望衝擊隊史最好成績，去年他們在季後賽曾3-0擊敗了EDG。Xiaopeng這位打野的潛力正逐步轉為賽場統治力，GALA的Carry能力同樣不容小覷，再加上轉會期Xiye的加入，讓這支年輕的隊伍有了有了更強的指揮能力和大局觀，所以，DMO很可能在德杯給粉絲一個驚喜。

而對於RW、SDX以及全二隊陣容出戰的iG來說，出線壓力很大。

註：此次德杯包括16支LPL隊伍以及4支LDL隊伍，分別是YM、SDX、GD和LEG戰隊。而16支LPL隊伍中，iG、FPX啟用全新選手陣容，首發隊員均未登場。
