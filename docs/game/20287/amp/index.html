王者榮耀：久劍不在Hero大名單，冬冠結束後，桃哥或重回YTG

天天要聞
2019-12-17

各位王者玩家大家好，我是小雨。隨著秋季總決賽的落幕，冬冠席位賽即將打響，並且2020年春季轉會期也越來越近了。本賽季沒有取得理想成績的戰隊，免不了會在轉會期有所調整。當然，人員變動不僅是選手變動，還有教練的變動。從目前來看，YTG望月草、RNGM老黑、WE.Dada、VG奶茶等多位教練已經官宣離職，佔了教練總人數的4層，並且這個人數肯定還會隨著冬冠的戰績而增加。

今天小雨想跟大家聊聊下一個最有可能離職的教練——Hero久劍（桃哥）。桃哥是原YTG主教練（據說還有一部分俱樂部股份），在18年夏季轉會期望月草來到YTG之後，桃哥升任賽訓總監一職。本以為桃哥以後就會在YTG養老了，誰知今夏轉會期，桃哥出乎意料地來到了Hero擔任主教練一職。其實桃哥剛去Hero，大家就為他捏著一把汗，因為都知道，在Hero有個隻手遮天的久哲，他才是真正說了算的人。

賽季初，Hero的戰績還算不錯，不過由於桃哥大膽啟用新人，也是遭到了久誠等老一批冠軍成員粉絲的圍攻。終於，在接連輸給XQ和RW俠之後，久哲坐不住了，Hero官方公告久哲出山，桃哥擔任助教。不過久哲此次並未能實現王者歸來，回歸首場比賽便2比3負於AG超玩會，後來的5場比賽更是只贏下聯盟墊底的WE，季後賽一輪游。

在Hero最新公布的戰隊冬冠大名單上，久哲依然是主教練，助教也不再是桃哥，而是變成了久豪。這就意味著桃哥在Hero的日子已經進入了倒計時。新上任的教練，全新的陣容，不到半個賽季的時間就被管理層（久哲）否定，這不怪桃哥。如果不是桃哥前期執教的戰績還不錯，Hero恐怕連季後賽都進不了，別說打冬冠了。在總決賽結束後，桃哥也是第一時間發微博稱：「誰說今年東強西弱。」有粉絲在留言板問了桃哥，望月草離開YTG了，他會不會回去。桃哥回復了一句：「下個月看情況吧。」

​關於桃哥的這句回復，小雨覺得有兩層意思：

1、如果Hero冬冠能取得好成績，久哲肯定繼續坐穩主教練，桃哥必走無疑，回YTG的可能性極大。

2、如果Hero冬冠拉胯，甚至拿不到冬冠門票（概率極大），久哲將永久退居幕後，不再擔任主教練，而桃哥重新回到教練席。不過以久哲的性格，也不排除他自己親自動手整頓。

所以綜合來看，桃哥離開Hero的可能性極大，而且大概率會回YTG。
