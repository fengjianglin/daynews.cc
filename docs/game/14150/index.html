DNF：大富翁活動第一名才能拿滿獎勵，低保沒希望，必須按計劃來

2019-12-11 470條評論 2,587次閱讀 483人點贊

時隔多年，大富翁活動又回來啦，獎勵照樣豐厚，兩發頂級的附魔，一個自選的神器守護珠，還有兩個兩個非常好的首飾寶珠等等。獎勵很誘人，但是這個活動只有一個月的時間，持續時間為三十天，年套一上架就沒了。活動時間短對於小編這種附魔基本上畢業的玩家來說問題不大，但是對於那些需要拿滿獎勵的玩家來說，這個問題就大了。

首先，每天兩次大富翁入場機會，用完了就沒了。第一名三個材料，第二名兩個材料，第三名一個材料。而把獎勵全部換完需要一百五十個材料，每天兩次第一名是六個材料，一個月是一百百十個材料，勉強能換完獎勵。小編第一次第一名三個材料，第二次第二名兩個材料，按照這個進度，要在活動最後一天才能拿滿獎勵。

但問題就在於，這次活動是三個人一同入場，必須分出一個一二三名來，按照機會均等的原則算，一個月三十天，假如你前十天天天第一名，中間十天天天第二名，最後十天天天第三名，那麼算下來，一共是一百二十個材料，根本不可能把獎勵全部換完，必須放棄一部分。如果天天低保的話，一個月六十個材料，那就更沒希望了。

所以必須把這個活動搞明白，任何打怪的房間都不要進，直接返回城鎮，馬上點篩子，所有遊戲里的獎勵統統放棄，一個都不要。兩輪遊戲的精鍊石頭最多不過十個八個，和價值兩千萬的頂級附魔比起來什麼都不是！快，一定要快！尤其是遊戲裡面有一個老異界三的空間支配者，別打，浪費時間的，直接返回！白圖可以打，只要是磨嘰的，都別打，獎勵最重要。

不得不吐槽一下策劃，出了這麼個活動卻不想讓大家拿滿獎勵！三個人進圖就肯定有排名啊，按照這個機制來，最後肯定就會有玩家拿不滿獎勵，這不是整人玩嘛！

總而言之，低保是一點希望都沒有的，這次活動和以往不同，必須努力再努力，步步爭先，天天拿第一，每次都拿第一，不然的話頂級附魔肯定和你沒緣分。需要換齊獎勵的玩家一定要注意了，活動里的精鍊石頭都是拖後腿的，最後的附魔最重要！
