與DOTA、LOL平起平坐的三國殺，依然令人矚目

2019-12-23 2,012條評論 4,515次閱讀 523人點贊

12月19日，歷時三天的2019年度中國遊戲產業年會在海南省會海口市落下帷幕，在壓軸的年度「遊戲十強」頒獎盛典中，杭州游卡麾下的《三國殺移動版》與《英雄聯盟》、《DOTA》、《守望先鋒》等行業翹楚一起榮獲「2019年度中國十大最受歡迎電子競技遊戲」獎項，或許對游卡或「三國殺」IP來說，獲得一些行業獎項只是一些常規操作，但是成功躋身和英雄聯盟、DOTA這些行業標杆平齊的業界地位，從某種角度來看，也是業界和玩家對游卡這家公司和《三國殺移動版》這個產品乃至「三國殺」這個國民現象級原創精品IP的一種認可。

今年的中國遊戲產業年會由國家新聞出版署主管，中國音像與數字出版協會、海南省新聞出版局等共同主辦；中國「遊戲十強」頒獎盛典、功能遊戲探索論壇、海南遊戲行業協會第一屆會員大會等多個官方組織；會匯聚了包括騰訊、網易、完美世界等業界大佬，堪稱中國遊戲產業影響力最大，最具權威性的產業大會。

其中中國「遊戲十強」頒獎盛典，旨在表彰為國內遊戲產業做出了卓越貢獻的企業和個人，並評選出最受玩家喜愛的遊戲產品，是產業大會上最受歡迎的活動之一。能夠獲得此項殊榮的必是玩家最喜愛、品牌口碑最好、遊戲受眾最多且影響力最廣泛的行業標杆產品，像是去年和今年連續蟬聯十佳的《刀塔》、賽事和規模強大的《英雄聯盟》、經典老牌之作《星際爭霸》等都在榜內，而《三國殺移動版》作為一款電子競技遊戲位列其中，足以證明游卡運營和製作的用心。

一款好遊戲離不開開發者的精心運營，作為游卡的代表之作，「三國殺」IP匯聚了游卡十多年的心血，從線下到線上、從頁游到手游，這樣立體化的生態布局、與遊戲載體變化同步的腳步，稱得上遊戲產業中的領航員。面對行業內的各種衝擊，「三國殺」並沒有被其他模仿者比下去，而是一直穩紮穩打一路走來，當然，這中間也沒有忘記在電競之路上的探索。

「三國殺校園行」活動是游卡走進年輕群體和全國高校社團的重要途徑，也是「三國殺」對電子競技的普及和探索之路。如今已有400多所高校社團加入「校園行」體系，通過高校之間的聯賽，為「三國殺」在遊戲競技推廣方面做出巨大貢獻。

「三國殺王者之戰」體系是與「三國殺」同歲的歷史悠久的賽事體系，也是目前游卡職業電競化之路的探索者和電競體系的重要一環。

「三國殺俱樂部聯賽」（SCL）則是游卡麾下一項專業性競技賽事，在國內各個三國殺俱樂部之間進行，不僅有成熟的賽事流程和規則，其獎項與賽事播報也是全國性，相當於「三國殺」中的LPL。同時遊戲內部也會進行各項比賽活動，由官方、公會或其他玩家發起，為此「三國殺」還專門建立了「自建賽」玩法模式，方便更多人自發性地組織、參與。

綜上所看，「三國殺」上榜今年的「遊戲十強」評選並不意外，這是「三國殺」十一年的發展歷程的一次小小見證，也是全國玩家對其成長的一次肯定。十一年，對遊戲品牌的發展並不算長，我們期待的是它下一個以及更多的十一年。
