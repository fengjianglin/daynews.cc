我的世界：四大遊戲時刻，會讓人感覺很爽，尤其最後一個更舒爽

2019-12-12 1,302條評論 2,450次閱讀 1132人點贊

經常玩mc我的世界的玩家，大概都有一些強迫症的感覺，就算是本來沒有，玩玩mc就會覺得有強迫症了。畢竟這裡的方塊都是很平整的，雖然有的時候會凸出來，還是會以很完美的正方體樣式凸出。對於其他的物品來說，也都很整齊。在玩遊戲的時候，大家經常會遇到一些比較舒爽的時刻，就是專屬強迫症玩家的時刻，以下這些情況你們都遇到過嗎？

1.草方塊的蔓延

在我的世界生存模式中，大家都需要圈養一些動物，因為它們存在於不同的生物群系，而我們的家園只會在一個地方安定下來，很難經常浮動於很多個地理環境中。因此，如果不把動物們養起來的話，大家的一些物資就沒有辦法得到，生存中也會出現困難。有的動物是食草動物，它們愛吃草，因此大家必須把它們放在草地上。而草方塊有一個功能，那就是能把周圍的泥土塊也給變成草地。看著它一點一點蔓延，感覺很爽。

2.水漫岩漿

Mc我的世界中獲得圓石和黑曜石的方法有幾種呢？除了自然界中生成，就是用水和岩漿的混合體來兌，岩漿遇到水會得到圓石，而反之就會得到黑曜石。在我的世界空島模式中，這些是很重要的，畢竟一開始沒有鑽石鎬的情況下，直接兌一個黑曜石門也是可以的。當我們在地下挖礦的時候，經常會有岩漿擋路，當把水倒上去以後，就會邊有滋滋的聲音，邊變出來一條黑曜石路，這情景很不錯。

3.附魔鎬挖礦

讓我們的生存變得更簡單，其實有很多種方式，其中一種就是給我們的武器附魔。附魔書的獲得方式有很多，比如找村民換，或者是運氣好了以後釣上來，再或者就是自己做。被附魔的武器和工具都變得更加高級了，當然也會帶來更多的便利。比如說，如果我們拿著效率超高的鎬子去挖礦，別說是圓石了，就是黑曜石，挖下來也是幾秒的事情。看著前方的道路一點點變得開闊，並且速度還那麼快，是不是很爽呢？

4.無限死亡

相信不少玩家都打開過「死亡不掉落」這樣的作弊行為，確實，一旦開啟了這樣的模式，大家甚至都不用穿鎧甲了，畢竟死了以後什麼也不會損失，連經驗都還在，簡直是野外回家的最佳方式。而無法在現實中體驗死亡的我們，一次次死亡或許很爽吧。
