DNF：體驗服春節版本內容前瞻，7大職業平衡和護石符文系統上線

2019-12-16 234條評論 1,972次閱讀 727人點贊

魔界體驗服目前更新了春節版本的測試內容，主要是針對系統和職業平衡的先行體驗，活動和商城暫時沒有更新，那麼提前更新的這些內容需要關注哪一些內容呢？

7個職業平衡改版

本期的職業平衡改版分別特色職業平衡和韓服同步平衡。

  • 聖騎士的主要平衡內容是優化了技能攻擊判定，修復大量的BUG，也就是和諧了猴戲，和諧大日如來，削弱輕甲的提升，所以這一個職業平衡對國服來說根本就是削弱，輕甲審判再次被削。
  • 驅魔師僅有3個技能優化，主要是無雙擊在火焰脈輪下攻擊範圍範圍和攻擊力增加20%；黃龍斧施放後僵直時間減少；
  • 死靈主要是增加了技能攻擊力，邪穢精通變更為常駐Buff效果，只需疊加一次，每級增加2%，9級基準為增加34%。
  • 特色職業平衡改動職業有阿修羅、劍宗、決戰者和魔道，四個職業和原先的職業平衡調整方法相似，均為被動技能的數值調整。
  • 阿修羅波動刻印增加技能攻擊力增加屬性，00級增加10%，30級精通增加25%，心眼16級後成長為2%；
  • 劍宗新增被動技能魔劍覺醒，1級增加5%技能攻擊力，每級成長1.5%，15級上限。
  • 魔道幸運棒棒糖新增技能攻擊力增加的屬性，每級成長是1%，5級精通增加12%，15級上限。親和法米利爾調整為每級成長2%，15級上限。
  • 決戰者長槍精通和行云：沐，技能精通後的成長為2%；行云：啟，16級後的每級成長調整為2%。

裝備欄新增護石界面

雖然CP副本魔界大戰還沒有上線，裝備系統欄已經更新，新增的護石界面就是玩家裝備CP技能的槽，中心兩個方塊是放置技能護石的區域，也就是每個職業只能裝備2個技能，每個護石可以裝備三個符文，神器符石裝備3個符文可以獲得護石的附加效果。

比如說稀有護石需要搭配3個符文才能觸發附加效果，而神器護石則有基礎效果，搭配符文獲得附加效果。

符文可以選擇自己需要的技能加成，分別為增加攻擊力、減少技能CD，增加範圍的屬性，根據自己的需要去選擇。

這些護石和符文均為魔界風雲副本擊殺領主隨機獲得。穿戴護石則無法穿戴異界套。

國服目前回歸系統中可以看到魔界大戰引導模式，所以國服上線直接上單人模式，並且很搭可能性會直接減負和贈送護石符文等。

攻堅戰系統優化改版

盧克攻堅戰創建界面UI同步韓服，並新增盧克困難模式，獎勵和常規模式一樣，並新增單人引導模式，目前體驗服入場單人模式就失敗。

此外超時空漩渦和普雷伊希斯的攻堅隊創建UI也重做了。

而普雷伊希斯還新增了一個述語者NPC，作用是普雷裝備的升級和轉換。

任務欄UI界面調整

任務欄UI界面調整，並新增冒險任務，接取冒險任務完成後可以獲得特殊道具，這一個道具送給NPC直接滿好感度。

新增人物詞典

菜單欄新增人物詞典，可以在詞典中了解到NPC的故事背景和好感度，NPC隨著主線劇情任務解鎖，並且可以點擊人物圖片上的星形圖標將其置頂，也看點擊右上角的位置指引尋找NPC。

卡片交易限制和新增卡片

國服終於等到了卡片交易限制系統，以後卡片只能交易一次，購買後變成帳號綁定。並新增了魔界大戰的卡片，其主要值得關注的是耳環150點四維，以及雙屬性武器附魔卡片。

最後還有回歸系統的改版，但是因為沒有回歸號測試，所以不再說明，畢竟大家也都是活躍玩家。
