阿水到底去哪兒了？理性分析，想要打世界賽，這三支隊伍很合適

2019-12-18 112條評論 1,655次閱讀 210人點贊

「阿水到底去哪兒了？」成為了轉會期，甚至是轉會期結束之後LPL中最大的「懸疑」。而作為目前我們LPL賽區最炙手可熱的新一代ADC選手之一，JKL的動向無疑還是牽掛著許多LPL粉絲們的心的。也正因為JKL個人能力出眾，他的轉會也必將為隊伍帶來質上的飛躍。

而身為我們LPL國產頂尖ADC選手，JKL的狀態依舊是保持得非常出色，再加上他年紀輕的緣故，相信在自己的巔峰期內，JKL也應該繼續向著S世界賽的目標進發。所以，從目前LPL戰隊的情況來理性分析的話，這三支隊伍無疑還是非常適合JKL的。

IG戰隊

Theshy、Rookie以及JKL所組成的「三叉戟」無疑是IG最具標誌性也最具進攻性的組合。雖說JKL目前成為了「自由人」，但還是有可能會與IG續約的。而一旦與IG續約的話，IG自然又是我們所熟悉的「完全體」。這樣的陣容，相信在新賽季不出意外的話，還是有衝擊世界賽的資格的。

EDG戰隊

伴隨著廠長轉型成為教練，Iboy轉會VG，EDG的確在ADC位置上缺少了一位具有豐富經驗且個人能力非常出眾的ADC選手。雖然EDG戰隊在S9賽季並沒有進入到S世界賽，但這支隊伍無疑還是眾多LPL戰隊中最有底蘊的一支。所以，JKL如果加盟至EDG的話，在整體實力上相信還是能夠提升一個檔次的。而且，從EDG整體的陣容上來看，Jinoo與EDG磨合還有提升的空間，打野Jiejie也是有著比較大的成長空間；中單Scout不用說，一直都是EDG非常穩定的Carry點；而輔助選手Meiko也是一位經驗老道的選手。如果再加上JKL的話，誰能夠說這樣的陣容進不了S世界賽呢？

TES戰隊

在迎來Karsa之後，TES的上中野無疑在LPL眾多隊伍中也有著非常大的競爭力了。事實上，從S9賽季的表現來看，TES的衝擊力也是非常強。而且，作為我們LPL賽區的新軍，TES也的確有著非常大的上升空間。但是，TES上中野一直強勢，雙人路卻成為了他們「弱勢」的地方。雖說在轉會期有兩位二隊的新人升到了一隊，但無疑還是需要一段時間的磨合的。所以，JKL能夠加盟TES的話，可以說是直接彌補了TES之前所存在的「短板」，直接讓TES成為了一支擁有S世界賽競爭力的隊伍了。

當然了，從轉會期中的一些流言中來看，還有消息表示JKL可能會加盟SN。但事實上，從SN目前的陣容來看，一個JKL確實會讓他們提升不少，但卻並不足以在競爭如此激烈的LPL中去爭奪S世界賽的名額。所以，從戰隊整體的情況來看，IG、EDG以及TES都可以成為JKL不錯的選擇。
