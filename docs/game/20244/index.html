王者：17號正式服更新，3大kpl熱門被砍，這個無腦法師卻更舒服了

2019-12-17 1,738條評論 3,421次閱讀 1323人點贊

大家好，這裡是鴿神。王者榮耀17號正式服更新，由於AG超玩會贏得了冠軍，遊戲內也增加了很多新的活動。除此之外，之前體驗服改動的幾名英雄技能也正式得到調整。具體有哪些英雄被改了？一起來看看。

三大KPL熱門被砍

盤古、馬超這兩個英雄在當前巔峰賽頂端局局和kpl的出場率很高，而且實用性很強，對射手和刺客也有著特別的針對性，明顯屬於數值超標的這一類英雄。這兩個英雄在普通路人局來說並不算太強，盤古是因為強勢期，而馬超又太難，只有KPL和頂尖的巔峰賽層面才能將他們的優勢發揮。

上次的體驗服更新中盤古就被削弱了前期的流血傷害，但是沒有上線到正式服，而這一次的正式服改版直接將盤古的大招基礎傷害砍掉200再減去了0.2的AD加成，這一改動可以說算上比較傷了，減少了4級盤古的爆發能力。而馬超這個改動倒是之前在體驗服有過測試，撿標槍之後移速加成下調為20~75，遠比之前少了不少移速，這讓馬超在前中期的傷害數值上下跌不少。

姜子牙這個英雄相信不少常看KPL的人都知道他的噁心，由於哪隊先拿到這個英雄哪隊就能先升到2、4這兩個關鍵等級，對於一部分前期強勢陣容來說，姜子牙是一個必備中單。這一次的調整，這個每10秒增加一次經驗的被動能力變成了120秒增加一次，雖然數值相同，但是在時效性上卻不是那麼簡單的。

除了這個降溫操作，姜子牙的大招基礎傷害和AP加成還再次得到上調，這一波下來感覺路人局的姜子牙可以算得上純加強。因為路人局不像五排，很多拿出姜子牙的時候隊友並不會配合，這讓姜子牙的經驗被動好像無用。而路人的大招傷害增加，讓後期的團戰作用變強，這樣一分析，是不是覺得姜老頭加強不少。

「無腦」法師更舒服了

安琪拉這個英雄在這個版本里已經超標了，由於她操作簡單傷害高，對比同類型中單小喬王昭君，核心裝備在版本變更中沒有降溫，只要技能放的准再加隊友配合，在kpl上也只能說真香。這次的改動增加了安琪拉的大招護盾量，是技能等級數值和AP加成的雙重提升，可以說非常恐怖。

不要認為安琪拉的護盾僅僅是護盾，由於大招時的安琪拉只要護盾沒炸，就一直擁有霸體效果，這一改動會讓很多脆皮刺客中單法師非常難受，前期對線難度提升。這個英雄現在在法師強度中算得上頂尖，還不需要太多技術支持，相比其他諸葛亮貂蟬秀兒英雄，直接說真香就夠了。

新英雄緊急優化

魯班大師上線峽谷後，雖不是太熱門，但強度真心還可以。鴿真的想不到這次的更新調整中居然會有他的身影。這次的改動中魯班大師的基礎血量得到提升，一級血量加強，每級升級血量加強，18級時比原先血量提升700+。3技能效果優化，釋放距離增加，大招釋放期間遭到中斷後，只需要1秒即可將附近被連中的敵人收納回來。

這次的血量提升和3技能優化勢必讓魯班大師的強度再上一層樓，未來的頂尖輔助中一定會有魯班大師的一席之地。

以上就是王者榮耀17號正式服更新英雄調整的有關內容啦。我是鴿神，一個專註遊戲的原創作者，想了解更多遊戲資訊攻略就點個關注吧！
