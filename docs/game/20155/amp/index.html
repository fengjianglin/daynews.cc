魔獸：部落如何指揮破解大橋局？

天天要聞
8 小時 ago

作者：NGA-滑溜大嬸

我的做法與思路(核心：風箏聯盟NPC)

兩天指揮破解了大約4次真正意義的大橋局，(一舉擊潰的不算)包括1個已經打了3個多小時的殘局。有破不了的嗎？有1場，拖太久，死活打不進去，但也不虧，那就止損，集體橋頭/再見，散夥下一場。

以下，很多細節不完美甚至有錯誤，請BL玩家探討，也歡迎LM兄弟諷刺。

LM守橋的利益點在於殺人繳護甲漲聲望。但其實最好的辦法還是20分鐘拿下比賽，出去秒拍下一場，大橋刷說明已經輸了，所以不要看論壇的「守大橋真香」，能速度贏哪個鬼才守橋，這也是「BL撕毀協議才會如此」的來由—他們也不想守，想速刷，但已經輸了沒辦法。BL方40分鐘排進來15分鐘1牌子滾蛋繼續等40分鐘，怎麼可能？點的啥菜啊喝這麼些？所以BL要贏，保持平和心態是首要的，對面一堆已經輸了的肉而已，需要指揮給予隊友心理引導。

進入大橋局怎麼辦？細節討論

1、首先，隊友們腦子大多數是不好使的，比較衝動，上論壇知道大橋局本質的很少。每一局我讓隊友停下，剛開始都會被質疑「你特么LM卧底吧」，還有密著罵「CNM」的，哈哈哈。為什麼，因為20分鐘一路推到家門口了，你讓他們放慢節奏，肯定不樂意，就是要衝，人之常情，和陣營無關。指揮別生氣，一般最多半小時後就老實了。我一般會說「我不說話了，你們沖幾波，我做我該做的事，需要我時候叫我」。然後就看各種頭鐵，「戰士上」「法師沖」「集體BUFF」。一般沖3、5波以後就開始有人回過味了：不對啊！進不去啊！咱回去吧？退石爐吧？然後就開始各種撕逼了，什麼「法師別下雪」什麼「戰士為什麼不沖」什麼「獵人在搞毛」…這時候才是指揮該說話的時候。

2、指揮方案。隊友冷靜後，應首先回撤橋頭，佔領雷矛墓地橋頭方向的三個制高點，分別是左中右，同時盜賊組3人繞後，去聯盟狗洞口小路潛伏，防止LM後面散人騷擾。LM會派獵人在狗洞復活，跳入BL復活點送死放陷阱延緩集合速度，幾個賊就能看住小路，且打大團時候盜賊無用—記得拿獵人身上的血並/感謝。

3、關鍵點(也是LM最怕泄露的秘密)：穩住後，術士做宏「/目標 莫高特」，其實是莫高特·深爐 ，那個點不好打。NPC位置在北塔往裡，靠近旗。找不到可以用牧師心靈視界，然後選牧師目標。用相位小鬼拉出來，沒難度，拉出來後不要擊殺，找無人角落，2人輪流拉走無限控制。這是LM守大橋的核心NPC，交護甲片，換聲望，一人交全團有聲望，還有BUFF。拉走這個NPC，LM和BL就一個起跑線了，甚至BL更優，因為BL可以輪流徽章回家交，聯盟沒了就是沒了，無限風箏不刷新。很多LM看著這局面就退了。
補充：壇友透露，也可以順路擊殺大德魯伊，省的對方召喚大樹，「/目標 大德魯伊雷弗拉爾」(或者/目標 大德魯伊) 圖見下，他的位置我沒記錯的話，在北塔的後面。

3、LM會不斷派人從橋上騷擾，並佯裝不敵回撤，誘導BL送人頭。BL指揮記得一個原則：休想騙我上大橋。我管這叫反守大橋，你送人頭我就收，你想過橋那不可以，想引我過去門都沒有。記得拿敢死隊員的血並/感謝。

4、守橋不是不攻，要拿下勝利。守好後多餘玩家3、4個人，開始做任務，召老大。我不推薦冰雪之王，因為第一，需要10個人點門，人手不足；第二，招出來他要溜達10分鐘，煩；第三，最重要的一點，他會被風箏。BL將會在各種召喚者的帶領下沖入敵陣，並取得勝利，剩下的空軍、突擊隊、狼騎兵，試一下就知道哪個好哪個差，老玩家都懂。細節方面，牧師注意第一優先是驅散地面召喚部隊的魔法，特別是冰霜新星。各種突擊部隊最好一起招，一波推，但我這幾次都是招一種，一發入魂。要記住一個要點：他們只是輔助，只是輔助，只是輔助，真正殺人靠的是玩家，不要指望召喚物打開局面，不可能的。召喚部隊騷擾作用和提振士氣的作用極大！當我們的狼騎兵、突擊隊、冰雪之王出場時，不斷會有人報點：「老大到平原了！」「老大到石爐了！」個個摩拳擦掌，憋屈了半小時士氣一下子起來了。

5、一次攻不進去失敗怎麼辦。這種事很常見，必須認慫，止損，回退橋頭，看對方反映。由於BL已經拉走了NPC，對方已經沒便宜可占，又看出誘騙不可能，很多LM會退，一算算賬就明白：等15分鐘坑下一波不明真相的BL比這麼沒任何收益的耗著強。LM退了就上，不退就繼續任務，等下一波NPC大哥，再來一次。看到這裡想看速推的是不是失望了？哈哈哈，大橋守衛戰，要有過日子的心。

6、攻進去了怎麼辦。這是本文一個重點。在騎兵、突擊、傻大個的帶領下，推入了，接下來怎麼辦？北塔是守橋核心，要不要拿下？還是直接拼旗？我說一下思路。
一旦攻入並拿下了墓地，LM將會狗洞復活，接下來必然是40人集體跳雷矛墓地。很多人想學聯盟守橋，這不現實。因為此時將軍墓地沒變紅，雷矛如果丟了，那麼部落復活點在石爐。而且聯盟有徽章，守橋不現實。如果BL大部隊馬上回防雷矛，LM有徽章，321集體回家，小部隊守不住救援站。直接將軍？風險很大但一旦成功戰鬥結束，一旦失敗退回石爐，重新開始。我這幾次的選擇是，大約4成回防雷矛，打斷開旗，拖延，一直拖到將軍墓地紅。有風險，但考慮幾種方案失敗的後果，這是我的選擇。

總結：戰術是退守，拉出LM方刷聲望的NPC(重點)，召喚戰場大哥，與大哥一起沖。細節很多，有不完善的地方希望多多諷刺，多多討論。
