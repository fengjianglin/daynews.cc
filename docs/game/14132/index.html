掉幀卡頓全拜拜！玩遊戲還是得要智能九代酷睿芯

2019-12-11 1,453條評論 4,481次閱讀 2038人點贊

[PConline 單機評測]遊戲，作為我們日常生活娛樂的一部分，一直以來都扮演著傳達觀點、愉悅心情、增進關係的角色，特別是3A遊戲，更是構建出了一個個豐富多彩，有趣動人的世界。

但精彩的電子世界，卻並不會憑空出現——它不僅凝結了開發人員的心血，也同樣是需要具有強大數據計算能力的硬體支持才能呈現的。在你看不見的地方，電子在飛速地急行，數據在不斷地並行交換，唯有可靠的硬體支持，才能避免卡頓和死機的發生。

三十幀都不到，中等畫質都開不了的體驗，又怎麼讓人能夠痛快地叫好？刺客信條、古墓麗影、戰地V，還有新怪談神作Control，哪一個都少不了對算力有所需要。

遊戲一時爽 一直遊戲一直爽

刺客信條

扮演Assassin，讓玩家體會與聖殿騎士團之間的激烈碰撞，發現二者理念差異的《刺客信條》，本就是一個以「硬體重塑世界」為核心元素之一的故事。映射到現實，自然也少不了對硬體的要求。

在飛行堡壘7上遊玩《刺客信條：奧德賽》時，能以40～65幀之間的幀率來領略古希臘的風光，簡直是這世上最為美妙的事情之一了。

中畫質下，遊戲日常是穩定在43幀左右的，非常流暢，行雲流水般的操作、砍瓜切菜般幹掉神秘組織殺手的體驗，正是一個動作遊戲應有的表現。

如果後台運算數據處理器無法及時響應反饋，那卡頓甚至崩潰自然也不是什麼稀奇的事情了——不過顯然，九代酷睿i5-9300H卻很好地維持了這種遊戲體驗，沒有什麼比順滑的動作遊戲更好的了。

古墓麗影

同屬動作遊戲，古墓麗影也同樣是一個對操作響應速度要求很高的遊戲，不論是什麼難度，低幀率的遊戲體驗都會影響遊戲的劇情推進——更不用說頻繁卡頓、失敗導致的讀檔有多讓人惱火了。

在以高畫質用飛行堡壘7遊玩古墓麗影時，平均45～80幀的幀率水平，也讓人直呼過癮。雖然一般而言30幀人眼就不再會感到太明顯的卡頓，但更高的幀率和順暢的數據響應仍然能帶來更為流暢的體驗，而這在高難度下也尤為重要——試想，按下了操作卻沒有得到相應的反饋，在高難度下要多讓人抓狂？

這當然少不了來自它所搭載的i5-9300H的功勞，及時的數據處理保證了遊戲的反饋和交互體驗能及時傳達給玩家，與此同時還能保住遊戲過程中的種種細節，這正是3A動作遊戲應有的體驗。

而在戴爾游匣G3上遊玩這款遊戲時，則同樣有著55～100幀的順滑表現，得益於i5-9300H2.4GHz的基頻，以及高達4.1GHz的睿頻，後台計算數據的工作，才能夠及時進行。

今日遊戲能實現如此豐富的多邊形而不卡頓，也得益與處理器硬體規格和技術的發展。

戰地V

重返戰火紛飛的二戰時光，戰地V想要呈現的，正是這樣一種體驗，但在電子遊戲的世界裡，廣闊的戰場上會有大量的多邊形和建模出現。


搭載了i5-9300H的飛行堡壘7在高畫質的設定下運行戰地V的時候，能夠維持在58～94幀的水平。

期間計算每幀所花的時間，大約是在6～34毫秒之間，並不會有絲毫的卡頓，無論是在單人劇情還是多人對戰當中，都保障了操作的有效實行。


同樣是搭載了i5-9300H的戴爾游匣G3，在運行戰地V時，也有53幀到71幀的表現。

8～57ms的幀生成時間，也是它能夠高效運行這款3A射擊遊戲的最佳證明，沒有卡頓，沒有響應延遲，對，沒錯，想要愉快地遊玩一款遊戲，這些都少不了。

Control

最後，在新怪談神作Control里，飛行堡壘7則維持了40～57幀的幀率水平——在這款難度頗高，需要精準操作的遊戲里，高幀率所帶來的遊戲體驗在Boss戰中相當有用，順暢的操作給遊玩帶來了更多的容錯率和反應時間。


7～38ms的幀生成時間，完全沒有出現那種高幀率下卡頓的情況（有時玩遊戲會出現高幀率下偶爾幾幀卡頓的情況），整個遊戲體驗穩定而順暢，非常舒心。


而同樣搭載了九代酷睿的戴爾游匣G3，在運行這款遊戲的過程中，也維持了50～63幀的水平，各種一遍過Boss的體驗，減少了大量無意義的重新跑路和嘗試，特別的棒！


3～67ms的幀生成時間，同飛行堡壘一樣沒有出現高幀率卡頓的情況——別看Control的場景狹小，但其中的物件和多邊形計算數量卻多得可怕，加上各種需要計算的物理效果——嘿，i5-9300H的性能還真強！

性能，一向重要

其實，說了這麼多，無非是想說明，在運行一款3A遊戲時，一個優秀的處理器在遊玩遊戲的過程當中，一直都是很重要的。就拿上文提到的兩款筆記本所搭載的九代酷睿i5-9300H來說吧，它2.4GHz的基頻，以及高達4.1GHz的睿頻，都能帶來數據的高速處理。

哦對，它還有8MB的三級緩存，並且支持英特爾傲騰內存、英特爾SpeedShiftTechnology、增強型英特爾SpeedStep技術等多項技術，持續穩定輸出算力，需要的正是這樣多種功能於一身的處理器——這也正是為什麼之前遊玩幾款遊戲，都沒有絲毫卡頓的原因。

當然，更好的處理器還會有更好的遊戲體驗，在遊玩遊戲這件事情上，追求上限是永無止境的。

尾聲

總之，玩了這麼多年的遊戲，硬體的選擇也一直都是其間重要的話題——想要好的遊戲體驗，一個強大的處理器是必不可少——及時輸出計算好的數據，配合其它硬體給玩家帶來一流的體驗，正是它在遊戲遊玩當中的功用。想要一個好的遊戲體驗，還是搭配一個強大又智能的「芯」吧！
