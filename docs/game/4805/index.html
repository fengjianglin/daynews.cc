hello大家好，這裡是朝汐，最近體驗服迎來調整，針對一些版本熱門強勢的英雄進行了調整，下面我們一起來看看吧

1.成吉思汗

一技能：被動：對同一目標的第三次攻擊會額外造成傷害 → 對同個目標的第四次攻擊會額外造成傷害一技能：印記傷害：200(+30/Lv)(+0.4Ad) → 240(+36/Lv)(+0.4Ad）

二哈這次調整，可謂迎來了冬天，在後期狼狗先發秒人可能將減少了，之前3次觸發印記，改為了4次，而作為補償，傷害增加了1級1技能46點，滿級1技能增加了高額的76點傷害，是的你沒看錯，是高額的76點傷害！（頂個鎚子用）。不過我想，有人歡喜有人愁，只要機制不改，還是能打的。

2.姜子牙

被動技能：每10秒為友軍增加14點經驗值 → 每120秒姜子牙為友軍增加168點經驗值 三技能：基礎傷害：450(+150/Lv)(+0.35Ap) → 450(+150/Lv)(+0.4Ap) 三技能：最大傷害：1350(+450/Lv)(+1.05Ap) → 1350(+450/Lv)(+1.2Ap)

首先說下，這是削弱，姜子牙作為kpl的常客，其被動增加經驗能夠幫助友軍更好的到4級，從而接第一波暴君團，然後滾雪球，但是此次整改，姜子牙沒辦法在2分鐘暴君團刷新之前給隊友提供經驗保證己方優先達到4級，不能再優先搶佔視野以及位置，可以說，滾雪球能力下降了很多。作為補償，3技能稍微增加了一點傷害。

3.馬超

被動技能：2秒內逐漸增加30~100點移動速度 → 2秒內逐漸增加20~75點移動速度

馬超這英雄，兩極分化可謂嚴重，至少我是不會玩，所以對這次調整不是很有感覺，但看得出來，因為傷害跟移速有關的機制，這次調整可謂大削，在這我就納悶了，勝率倒數第二名，還削?

4.曜

修復了曜三技能在部分情況下冷卻異常的問題

沒什麼好說的，優化

5.盤古

被動技能：流血傷害比例：1.3%(+0.05%*英雄等級) → 1%(+0.07%*英雄等級)

削弱了盤古前期的流血傷害，但我相信只要機制不改，盤古依舊強勢，打團一個大集體繳械。

6.安琪拉

三技能：轉向速度：45度/秒 → 60度/秒

安琪拉又雙叒叕增強啦！作為萌新最喜歡的英雄之一，經歷了n次的小加強，增加傷害，減少cd，增加護盾，技能速度，我的天，低風險，高收益，這也是小編最討厭的無腦英雄之一。

以上的版本爆料就到這裡，趁著小狼狗正式服還沒調整，趕緊抓緊上分吧！
