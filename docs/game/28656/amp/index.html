酒館戰棋：寒光先知被調回三星，盤點三星酒館那些必拿隨從

天天要聞
2019-12-22

隨著更新補丁的實裝，夢魘融合怪暫時被移除了酒館，而比斯巨獸和寒光先知也被調整為了三星。

作為酒館戰棋一個非常重要的節奏點，三星酒館能否拿到那些關鍵的隨從基本上可以說直接決定了整局對戰的走向，今天小編就來為大家盤點一下那些在三星酒館必拿的神仙隨從。

首先當然是寒光先知，作為魚人流前期戰鬥力兩大保證之一，寒光先知和魚人領軍是絕對不能放過的關鍵牌，看到就要拿，當然也有玩家會選擇加完buff就將他賣掉以增加繼續刷出寒光先知的幾率，這也是一種方法。

比斯巨獸的身材放到四星科技時已經沒有統治力，再加上負面的亡語效果讓他成了一個鐵憨憨，但是這次補丁把他調整為三星之後幾面可就反過來了，能在三星科技時達到九杠七的隨從可不多，用比斯巨獸打兩回合工還是很不錯的選擇，當然，再多就不行了。

族群領袖作為野獸流前期的親媽，重要性根本不言而喻，三星選到族群領袖基本可以奠定走野獸流的路線了，要是有老鼠配合那戰鬥力簡直粉碎一切！

和族群領袖相反，鈷制衛士的身份是打手，當然這個打手的地位幾乎不可替代，機雜流最強大的地方就在於不停地為鈷制衛士刷新聖盾白吃對方的隨從，拿到鈷制衛士即便不走機雜流的路線也能當做一個強力打手使用，可以說三星必選鈷制衛士。

百變澤魯斯，讓小編又愛又恨的卡牌，他總能創造奇蹟，但也會在不經意間扼殺你的希望，小編曾經跳科技七費就拿了一個澤魯斯，八費直接拍下熊媽媽，野獸流打的所有魚人沒有後期直接吃雞，然而更多的時候他能給你帶來的也只是一個憤怒編織者。

但是有夢想誰都了不起，哪怕明知不可為，小編看到澤魯斯依然必拿！

戰棋雖然好玩，不過小夥伴也別忘了關注天梯，年關將近，一年一度的暴雪打折季也將開始，《爐石傳說》打響頭波優惠，推出超值限定合集——冬幕節合集，整整三十個卡包的超值福利，再加上限定卡背和德魯伊樹人英雄皮膚——黑澤巴爾克！

這波福利絕對要給滿分！

除了冬幕節合集以外另一個重頭戲就是先前上架的女王皮膚，這款皮膚無論是特效還是音效都堪稱完美，非常值得收藏。

這款皮膚的售賣到明年一月六號為止，心動的小夥伴可別錯過了。
