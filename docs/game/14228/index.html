日本人花70億做的遊戲，20年後再出新作，僅賣了1萬份就涼涼？

2019-12-10 157條評論 1,357次閱讀 408人點贊

大家好，這裡是正驚遊戲，我是正驚小弟。

在現在競爭激烈的遊戲市場中，遊戲玩法更新換代已經是常有的事。即便如此，仍然有許多耳熟能詳的經典IP不願服老，通過續作來延續自己的生命力。也許他們曾經代表著一個時代，但現在他們卻代表了玩家心中難以磨滅的情懷。

今天小弟就來盤點一下那些在2019年那些時隔多年推出續作的情懷遊戲，看看有沒有你玩過的吧~

鬼泣5：同時代遊戲基本都涼透了，他還在不緊不慢地出續作

要說情懷遊戲的翹楚，《鬼泣》說第二，絕對沒人敢搶第一。作為一款早在2001年就已經發布的作品，憑藉著優秀的畫質和超前的遊戲手感，在當時吸引了一大批的玩家趨之若鶩，創造了216萬份的銷售業績。這份意外收穫讓CAPCOM看到了新的商機，當即拍板決定了：《鬼泣》正式成為一個系列，要出續作了！

然而作為出了名的「電影公司」，CAPCOM可沒說《鬼泣》系列要多久才會出下一部續作。於是玩家在等續作的時候等了一年又一年，基本上每年都有新消息，每年都會跳一次票。

就拿今年新推出的《鬼泣5》來說，距離前作《鬼泣4》，已經過去了整整六年！期間《鬼泣4》推出了手機版，特別版，外加幾款DLC，反正就是不出續作。能等到現在還默默支持的玩家，小弟真心敬你是條漢子！

好在《鬼泣5》還是可以讓忍受多年跳票的玩家來一句「真香」的。作為遊戲的靈魂，《鬼泣5》延續了前作硬核動作遊戲的定位，在畫面上相比之前有了極為明顯的進步。同時在劇情以及角色設定上，《鬼泣5》也做得恰到好處，沒有任何水分。這樣一份遲來的大禮，讓《鬼泣5》截至目前收穫了250萬份的成績。不管如何，《鬼泣》永遠都是玩家心目中的動作王者，至於那份無論多麼跳票都在堅持的情懷嘛，誰說得清呢？

無主之地3：七年磨一劍的3A大作，你見過沒有？

《無主之地》自從2009年發布第一款作品以來，一直被死忠粉奉為3A大作中的代表。表面上，這款遊戲的玩法僅僅是將RPG和FPS遊戲融合在了一起。但只要進遊戲一看，沒有十天半個月，玩家甚至都有可能連這款遊戲的邊界都探索不到。

《無主之地3》作為今年發行的又一續作，著實是給了玩家一個巨大的驚喜。此時距離前作發售已經過去了整整七年，這七年里，Gearbox給了玩家一個滿意的答卷。遊戲在主線任務和支線任務的基礎上，又額外加入了諸多細節滿滿的開放地圖，玩家在一張地圖上幾乎每走幾步就可以有獨特的事情要做，也算是給遊戲增添更多趣味性。

同樣，3A大作的質量也讓無數的死忠粉選擇了買單。在發售僅僅前五天，《無主之地3》的銷量就突破了500萬份，加上之前兩款前作的總銷量突破5500萬份，還是9月全球最賺錢PC遊戲第二名。據外媒估計，《無主之地》系列很有可能在四作之內就會突破過億的銷量，累計獲利超過50億美元。

莎木3：一款把日本玩家感動哭的真·情懷遊戲

說到《莎木》系列，許多當年的老玩家估計都特別激動。在99年PS2還沒出現的年代，大部分玩家審美還停留在《生化危機3》那種一臉馬賽克的畫面上，莎木可以說是為就像現在的VR遊戲一樣為大部分玩家打開了一扇新世界的大門，真的忘不了第一次玩莎木的那種激動的心情。

不說別的，《莎木》光是從開始到莎木一章橫須賀完成時就已經用了約70億日元，而《莎木2》亦用了約20億日元作為開發的成本，吉尼斯世界紀錄大全中亦曾有記載過此遊戲為「開發費最高的遊戲」，由此就能看出這兩作的用心程度了。

《莎木》還是如今開放世界設定和QTE玩法的先驅，遊戲場景對真實世界的還原與沉浸感在當時乃至隨後幾個世代都難有作品能出乎其右。

不過由於《莎木1》和《莎木2》的銷量暴死，《莎木3》一直遙遙無期，一直到了2015年才有了眾籌的消息。

而《莎木3》不斷跳票，從說好的2017年發行一直到了2019年的年末才正式發布，已經消耗掉了一部分粉絲的耐心，讓它的情懷光環褪掉了不少。因此，正式發布的首周《莎木3》僅僅賣出了一萬份遊戲，一度震驚了日本的遊戲媒體。《莎木3》出師不利的遭遇，可真的算是遊戲界慘案之一了。

大富翁10：最後，為情懷買單的玩家還是失望了

《大富翁》首作可以追溯到1989年，在高度競爭的遊戲市場中堅持推出自己的續作，一堅持就是三十年。彼時阿土哥還是年輕小伙，到現在都成阿土伯了，這中間的時光可是老玩家們滿滿的回憶啊！

最新續作的《大富翁10》，新作上線首日遊戲的銷量快速突破了五萬份，但也留下了接近1500條的負面評論。Steam上的差評率更是了66%。其中不乏吐槽遊戲畫面和語音，聯機卡頓閃退，大小bug不斷等等，更多的，還是認為自己的情懷被《大富翁10》給輕視了。

《大富翁10》也因此緊急做出了不少措施，在遊戲發售前就提前給打了個半價，並且在事後補丁不斷，企圖修復這些問題。即便如此，《大富翁10》的此番行為在玩家之間的印象也有了定格的趨勢，究竟未來能否改善，那就看他們怎麼做才能讓玩家說一句「真香」了！

小弟有話說：時間總是會篩選出最值得留存的東西。一款遊戲的火熱，興許代表的只是一個時代。而一款遊戲如果能從火熱堅持到現在，那麼它繼承的就是無數玩家的情懷。但願時光不老，你我不散，在未來競爭激烈的遊戲市場中獨樹一幟，長青不衰。

一個正驚問題：你會為這些情懷遊戲貢獻錢包嗎？
