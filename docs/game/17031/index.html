《獵人》手游最坑新手副本失樂美，收下這份攻略，拒絕忙目送人頭

2019-12-14 176條評論 1,169次閱讀 152人點贊

談遊戲風雲，評好玩遊戲。玩遊戲，歡迎關注紅燭鎮。

《獵人》手游因其使用了《獵人x獵人》的動漫IP，又深度融合了DNF端游的大量玩法，近期吸引了不少玩家的注意。紅燭最近也在肝這款遊戲。

遊戲在46級時，會開放「遺迹探險」功能，同步開放副本「初探失樂美」。順著任務引導，玩家自然會熟練的點擊進本開打三連來一發。

但進本後會突然發現，這個引導任務比起之前的所有新手引導，難度驟然提升。很多人不明就裡的在吃藥、復活一套之後，被BOSS擊殺送出副本。

而如果不清楚打法，組隊打失樂美，那簡直就跟葫蘆娃救爺爺一樣：成了盲目送人頭。

今天為大家詳細分享一下《獵人》手游遺迹副本：「初探失樂美」 的具體打法。

單人通關可能更容易

初探失樂美副本，每次挑戰消耗5點疲勞和20個遺迹捲軸，系統建議46~60級玩家挑戰。但鑒於副本實際難度，紅燭鎮建議玩家在50級以後，稍微提升下裝備再嘗試挑戰。

初探失樂美副本中，共有2個BOSS，每個BOSS都有自己固定的攻擊套路。摸清Boss套路後，戰鬥會隨著熟練度逐漸變得簡單。

如果想嘗試通關副本，建議熟悉BOSS機制的玩家先單人通關，難度會稍微低一些。

一來是《獵人》手游中，組隊本身會提升副本怪物屬性，多組一個隊友，怪物血量大概會上漲至少25%。

倘若再加上隊友對於Boss機制不熟悉，往往打一把就散夥了，成了白白消耗疲勞。

初探失樂美通關秘笈

初探失樂美副本共有2個BOSS，每個BOSS之前會有一場小怪戰鬥。小怪戰鬥很輕鬆，AOE技能聚怪之後消滅即可，就不贅述了。

接著為諸位客官送上副本的BOSS通關秘笈：

BOSS出生左上角，遠程技能狠命擾

扇形覆地毒傷害，靈活走位不貪刀

甩尾衝撞二連招，小心應對防擊倒

天降水彈傷害高，冷靜走位閃避好

見到老二快進圈，否則地刺不間斷

走位遠離他捶地，閃避背後偷傷害

BOSS蓄力快出圈，備好藥劑通關易

一號Boss：古螈布魯

玩家面對的第一個BOSS，是一隻水陸兩棲的大蠑螈。一號BOSS的戰鬥分為3個階段，每階段有不同的攻擊方式，大家需要注意3個輸出節點。

在大多數情況下，BOSS開場從水中跳出後，會刷在屏幕的左上角。而這也是要抓住的第一個輸出節點。

在BOSS剛出水落地的瞬間集中利用遠程技能輸出。大招、夥伴能用的招式都在這裡用上。

BOSS出水大概2秒左右，會噴出扇形的AOE毒素攻擊。外形上就非常類似LOL中蛇女的覆地，要儘快閃避躲開。

鑒於BOSS覆地攻擊範圍較大，而且幾乎覆蓋自己四周，所以這個BOSS盡量用遠程技能騷擾磨血，近戰上去很容易吃傷害導致陣亡。

不過紅燭鎮在個人單刷遺迹的時候，也遇到過BOSS刷新在屏幕中間，而非左上角的情況。這個時候，近戰打起來就輕鬆很多，繞到BOSS背後瘋狂輸出就好。

BOSS的第二種攻擊方式是：隨機從屏幕左邊的一個位置，向右側直線衝撞；在撞擊完成後，甩尾並噴出扇形的毒素覆地。

在衝撞前，BOSS會先就位暫停一段時間，所以只要注意走位，衝撞很容易躲開。

而BOSS衝撞後，甩尾覆地前，是第二個輸出節點，大概有1秒左右的時間可以甩個遠程技能偷傷害。之後切記不要貪刀，閃避遠離避免被覆地傷害到。

在衝撞甩尾後，古螈會跳回水中，並從天發射水彈。水彈落地前會有明顯的紅圈提示，這裡其實只要原地站立，等紅圈亮起，然後走出範圍即可。這一階段全程注意保命，無需輸出。

水彈攻擊之後BOSS會繼續從水中跳出，並重複此前的3個階段循環。

另外在第一階段，古螈累積受到一定傷害會呈現「疲累」狀態，不會釋放攻擊技能，此時是輸出的第三個節點，可以上去狠命偷一波傷害。

但如果穩妥起見的話，這個輸出節點也可以先忽略，畢竟保命優先。

注意這三個階段的循環，大概2分多鐘就能磨光古螈BOSS的血條啦！

二號BOSS：守門人古均

面對古均的整場戰鬥，都圍繞「進圈、出圈」來進行。

當BOSS刷出後，地上會有一個藍圈，此時需要全體隊員在圈內集中。

如果不進圈的話，古均會一直釋放地刺技能，導致玩家一直無法進行攻擊，而且即使真的放了遠程技能，也無法造成傷害。

進圈之後，開啟正常的BOSS戰鬥。古均在圈內只有2個技能。

第一個技能是砸地板，BOSS會呈橢圓形範圍進行捶地攻擊。攻擊範圍相對較大，基本覆蓋1/3個圈，這裡只能靠靈活的閃避到BOSS身後，或是吃藥來應對。

建議玩家站在BOSS圈最左側，吸引BOSS前來攻擊的瞬間，閃避至BOSS身後，規避傷害的同時，在背後偷一套輸出，如此循環往複。

古均的第二個技能是蓄力砸地板，BOSS會有明顯的蓄力狀態，此時請儘快遠離藍圈！

BOSS蓄力後會在圈內造成高額的AOE傷害，所以需要儘快出圈規避，等待BOSS技能結束後再返回。

二號BOSS的技能很簡單，靈活在圈內圈外遊走便能熬到將其擊殺。整個戰鬥過程主要是吃操作熟練度：如何儘可能的閃避普通砸地板攻擊。

如果前期操作不熟練的話，只能靠藥品補給來熬過去了。

紅燭鎮評價

遺迹副本是《獵人》手游中神器裝備（粉裝）的一個主要獲取途徑。不過通關並不代表會出貨，關卡評價和神秘的掉率才是影響一切的根源。

新手玩家看過本文攻略後可以嘗試體驗「初探失樂美」副本，打起來還是有一定樂趣的。但若是想刷裝備提升角色屬性的話，那遠不如組隊刷虛空王者更加實在。

另外要注意的是，在失樂美副本中，全隊是共享復活次數的。整場副本戰鬥，全隊僅能使用5次復活，所以切記還是先熟悉攻略後再嘗試挑戰吧！

而如果遇到遊戲中有想一起玩，但不清楚BOSS機制的玩家，也可以一句話簡單為他科普：老一遠程騷擾不貪刀，老二進圈背後打。
