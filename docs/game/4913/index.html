大家好，我是盟盟，這是第166期《盟盟與貓和老鼠》

貓和老鼠遊戲中S級知識卡的地位已經不用我多說了，3級無畏堪稱多了一個技能，配合國王傑瑞的技能，基本上能夠無傷救人，高端局沒有一個2級無畏可能都不好意思去玩鼠，同樣貓也是的，貓卡的暴怒、擊暈都非常強，其中暴怒應該算是很多貓玩家翻盤必備的神卡，那麼問題來了，獲得3級暴怒要氪金多少？要氪金2000嗎？一位土豪給出了答案，我們一起來看看。

3級暴怒

首先和大家說一下貓和老鼠遊戲中的暴怒知識卡，這是一張7知識量的S級貓卡，應該算是每個高級貓玩家都會帶的。

效果很簡單，就是牆縫出現後，貓咪能夠獲得一擊擊倒老鼠的效果，但是攻擊間隔會變長，也就是增加了傷害，提高了攻擊間隔，雖然不完美，但是後期翻盤真的就靠暴怒了。

而且1級暴怒和3級暴怒的差距不是一點點，3級暴怒只需要稍微破壞牆縫就能除非，而1級暴怒需要破壞很多牆縫，所以3級暴怒是必須要升出來的，對於貓來說是質的提升。

但是升到3級暴怒談何容易啊，有的玩家升到了3級擊暈都沒有一張暴怒，或者是3級猛攻沒有一張暴怒，想要抽到這張S卡都是要靠運氣的。

而且遊戲的知識點並不多，基本上是靠氪金來抽知識卡，所以問題來了，獲得3級暴怒要氪金多少呢？對此盟盟採訪了一位土豪，他目前的S級知識卡都滿了，土豪說出了一個答案，他表示當時抽暴怒這張S卡的時候，附帶還抽到了3級的猛攻，兩個一起氪金了2500左右。

不知道大家對於這個答案是否滿意，氪金2500能獲得3級暴怒，想必很多玩家都望而卻步了，還以為2000就行了，沒想到2500才抽到了，看來S級知識卡的獲取概率比S級皮膚要難。

這是一個悲傷的故事

都說貓和老鼠遊戲裡面黑貓已經變成了黑鼠，現在黑貓的出場率真的太低了，不管是經典模式還是天梯，能夠看到黑貓都算是運氣很好了。

其實黑貓也沒有大家想像的這麼弱，21知識點的黑貓，其實能夠橫掃泰菲這種鼠的，當然遇到一些比較厲害的陣容，如果黑貓的操作到位也是能夠拿下SSS的評分，比如這位玩家就遇到了海盜傑瑞+劍客泰菲+劍客傑瑞的組合，本來以為黑貓會被胖揍，可誰知這是一個悲傷的故事，黑貓最終放飛了4隻鼠，完成了逆襲，黑貓強啊。

泰菲的SSS級皮膚值多少？

貓和老鼠遊戲中有很多S級皮膚，基本上上線一款新的皮膚，就會下架一款皮膚，那麼問題來了，大家覺得泰菲的SSS級皮膚熊寶大冒險值多少呢？

這裡盟盟有話要說了，因為泰菲這款皮膚在以前的爆率實在太高了，盟盟一個648居然出了5個熊寶大冒險，而一個天王巨星都沒有，以前這款皮膚的爆率是最高的，所以大家覺得這款SSS級皮膚值多少呢？
