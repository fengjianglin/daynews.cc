《魔獸世界》海島打撈品100連開！坐騎玩具幻化輕鬆獲得

2019-12-14 92條評論 1,047次閱讀 58人點贊

作者：喃兌啊內

8.0推出了海島玩法，非常機械重複掉落又坑爹

玩家不喜歡海島但是又饞海島里掉落的東西

8.3算是版本末期了，設計師推出了打撈NPC

用打撈品讓大家更容易獲得海島的掉落物品

在8.3達布隆幣商人旁邊新增了一個打撈專家

他每周會賣幾種打撈品，

目前Ptr看下來每周都是隨機刷新的

有時候會賣3種，有時候賣五六種

紫色的打撈品，175海員達布隆幣

藍色打撈品，120海員達布隆幣

綠色的就2種，1個75，1個50

紫色的打撈品對應7個海島地圖

瓊花村，飛掠谷，腐化泥沼，約倫達爾

灣林鎮，潮落島，恐懼群島

藍色的打撈品對應4個地圖

低語堡礁青翠荒野安戈爾廢墟熔火海礁

綠色打撈品有風險投資公司和元素生物

​

也就是一共有13種打撈品，會在打撈專家這邊隨機刷新

打撈品可以開出2~4個物品，包含幻化，聲望物品，

任務物品，玩具，小寵物，還有最重要的坐騎

因為一下能開出好幾個東西，出貨率很高

之前海島你有缺的東西，可以囤海島幣等8.3換打撈品

那打撈品是不是紫色的最好？紫色藍色綠色有什麼區別呢？

海島地圖有很多原生怪物，這些打撈品也對應海島上的怪物

比如瓊花村，原生怪物是螳螂妖魔古和兔妖

那麼瓊花村打撈品，裡面能開出來的就是這些怪物相關的物品

​

我這邊已經把各種打撈品對應的怪物種類列出來

大家最關心的應該就是坐騎，我也列出來了

你缺哪種坐騎，需要買哪個打撈品就一目了然

我們上面有說打撈品出貨率很高，到底有多高呢？

下面是老外各種箱子100連開的數據，可以參考

​

可以看到幻化裝備特別多，都在200多件，也就是200%出貨率

聲望物品在30~50%的出貨率

小寵物在30%左右的出貨率

玩具種類少而且限定怪物，所以出貨率最少

坐騎出貨率也在20%左右

在ptr用6000多海島幣就基本能開到所有坐騎

當然萬一臉黑也可能要更多的海島幣
