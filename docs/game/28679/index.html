孤影撞車E星Alan虐國服老虎，但MVP都沒拿到，娜可露露成神秘大神

2019-12-22 362條評論 1,636次閱讀 470人點贊

苦瓜原創，翻版必究！

《苦瓜電競》–讓你體驗不一樣的電競主播圈！

​

在高端局中還敢時不時拿出后羿的玩家，那就非孤影莫屬了。都清楚的，后羿在高端局中可是太難有發揮了，更何況，孤影這玩的可是巔峰賽2100分左右的局啊，這個局的后羿生存環境就更差了，這個真的和操作無關，就是英雄特性問題，就連本命英雄是后羿的大神們也是無法完全C的，孤影對后羿的自信，也是沒誰了，作為虎牙王者榮耀牌面主播，孤影的全能已然得到公認了，十幾個國服稱號，不開玩笑。

​

這把遊戲中，孤影這邊的發揮太順了，隊友們也是各種給力，可以說沒過多久，這場比賽就宣布結束了，確實是太順了，孤影都是想不到，搞得自己還是死的最多的，難受！

看到這賽後結算面板，也是佩服孤影的好運氣了，要知道，孤影可是撞車了E星的Alan哦，這要是作為對手，這勝負真的還說不準的，還好是隊友，這劉邦可是幫助太多了啊，看到這完美KDA沒？職業選手確實是有點東西的。二人的聯手也是讓國服老虎直接吃癟哦，估計這國服老虎都不想玩遊戲了，相信看到這賽後ID之後，也是心服口服了。

​

不過這二人都是沒有拿到MVP，還是有點可惜的，最後被娜可露露拿到了，娜可露露的數據更完美一些，這絕對是個狠角色了，神秘玩家啊，在這兩位面前能夠拿到MVP，可以的！

我是苦瓜，我為自己帶鹽！（不帶油）
