不求人力薦光子刪除暗夜模式，新版本果然刪了，網友：一哥有牌面

天天要聞
2019-12-23

最近和平精英12.18的新版本成為網友們的焦點，很多網友開始對這次版本的熱議，大多都是感覺這次的規模確實有點大，直接出現了倆大模式，而且都是增加很多道具或者細節的版本，這次光子也算是圓滿完成任務，玩家們也都等了將近一年的時長，確是這回完全沒有白等。

大批新穎的道具出現，而且幸運空投裡面的裝備更是豪華，再加上藍色火焰的幸好槍，真的太吸引玩家了，但是這次光子居然要刪除陪伴玩家們多年的模式，直接在網上炸開，具體怎麼回事呢？

在此次更新的版本內容中，最後一條細心的網友們可以發現，在12月24號1點將會關閉暗夜危機模式，在此之前仍可以參與該模式，並且提及如果後續再次開啟該模式的話將會在官網通知，希望大家諒解，但是網友們卻都很開心，畢竟每次任務中都有擊殺800個殭屍，不僅有的玩家完成不了，還得消耗大量的時間，這次刪除掉暗夜危機，想必這個任務也將會取消吧，玩家們都開心的飛起。

這次不求人的粉絲們又出現了難題，就是如今的地圖就只有海島，但是雖然系統安裝已經完成拆分，但是仍有很大的內存空間佔用著，所以在直播間發些牢騷，不過也只能這樣了，光子改進的系統選擇安裝，已經減少了很多內存，如果還是玩不了的話，那隻能建議換手機了，而且官方公告新賽季馬上就要來臨，12月24號將更新ss5賽季，這段時間和平精英的熱度真的是只增不減，這回主播們可挺期待的，新賽季前幾天那可是最好衝刺的幾天，不求人早就等著了，直接打上國服戰神，12月版本即將來襲。
