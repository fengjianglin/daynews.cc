​《時代》評十年十大遊戲 19年僅《極樂迪斯科》

天天要聞
2019-12-21

《時代周刊》評選出了他們眼中近10年間（2009-2019年）最好的10款遊戲，並給出了簡單的評價。其中排名不分先後且不局限各年份雨露均沾，2011年有3款遊戲獲評，而2019年唯一入選的一款遊戲是《極樂迪斯科》。

《GTA V》（2013）

​

有史以來商業上最成功的娛樂產品，創造的營收超過60億美元，其文化影響難以量化，遊戲內容中大量的暴力內容引起了社會的廣泛關注。

《塞爾達傳說：曠野之息》（2017）

​

創造出同類中最好的開放世界遊戲，玩家可以不受線性流程，自由探索海拉爾，創造自己獨一無二的冒險經歷。其優秀的質量也支撐了任天堂Switch主機的銷售。

《堡壘之夜》（2017）

​

雖然《絕地求生》推廣了大逃殺這一類型，但《堡壘之夜》正式使其成為一種地位牢固的文化現象。遊戲巨大的營收成績助力Epic投入重金推廣自己的PC遊戲平台，Ninja藉此一躍成為超級主播，而遊戲對青少年的吸引力也引起父母們對遊戲成癮問題的關注。

《黑暗之魂》（2011）

​

《黑暗之魂》的影子在近10年里一直隱約可見。就像「地下絲絨樂隊」的第一張專輯，並非所有人都演奏過，但很多人受到了啟發。遊戲敘事和系統被一代代遊戲借鑒完善，直到今年的《星球大戰絕地：隕落的武士團》仍然深受其影響。

《英雄聯盟》（2009）

​

本作幫助多人在線戰術競技遊戲這一類型成為電子競技的黃金標準，遊戲的電競項目影響波及全球各地，吸引到的觀眾總數量超過超級碗。而開發商Riot Games存在的性別歧視問題，也是近10年中遊戲業界熱點主題之一。

《寶可夢 Go》（2016）

​

在遊戲發售的最初幾個月，本作吸引了全球的目光，這款遊戲每天都會創造新聞，涉及在現實世界中走動的玩家們的快樂、冒險與不幸。

《我的世界》（2009）

​

這是21世紀10年代的文化現象，有史以來第二暢銷的遊戲，藉助著不斷更新的內容和未來移動AR玩法，它仍有希望在下個10年繼續佔據主導地位。

《上古捲軸5》（2011）

​

這一系列屢登銷量榜首，霸佔一代代遊戲機角色扮演遊戲旗艦的位置，《上古捲軸5》巨大而精細的開放世界、開放的遊戲風格使之擁有持久的生命力，MOD製作者也給與了巨大的支持。其他遊戲開發者在10年中試圖追趕它的地位，但鮮有人能如願。

《傳送門2》（2011）

​

以成功的初代為基礎，《傳送門2》在各方面都取得成功，包括優秀的故事和更完善的玩法。然而這也是Valve在遊戲開發上走向低迷的轉折點，他們忙於利用其Steam賺取數十億美元，但再也沒有製作過《傳送門2》這樣的優秀遊戲。

《極樂迪斯科》（2019）

​

《極樂迪斯科》為這10年的優秀遊戲畫下一個完美的句號，這是一款站在過去卻展望著未來的電子遊戲，這種電子遊戲只會在時代過渡、新舊交替的年代出現。遊戲的玩法和畫面類似上世紀90年代的遊戲，但敘事技巧似乎來自未來，為舊RPG注入新鮮血液。本作再次證明了電子遊戲在敘事上擁有某些不同於電視、書本的獨特優勢，證明了電子遊戲是一種藝術。

從評價中不難看出，雖然這份榜單名為「最好的遊戲」（Best game），但《時代周刊》的評選標準更加側重於遊戲的影響力，而不僅限於遊戲本身的質量。《GTA V》的商業成功、《塞爾達傳說：曠野之息》對Switch銷量的支撐、《英雄聯盟》在電子競技中的地位等被加入了考量。
