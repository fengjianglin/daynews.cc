王者體驗服新增特有buff,哼哈二將相愛相殺，玩家：策劃是個明白人

天天要聞
2019-12-22

各位小夥伴們應該知道為了新賽季的到來，體驗服進行了一系列的整改調試。除了對峽谷內的分路和野區資源的調整，還對畫面展示進行了優化，更加簡潔美觀。但除了這些調整之外，策劃還藏了一些小彩蛋，你發現了嗎？

其實王者的策劃一直會藏一些小彩蛋等玩家去發現，比如模擬戰裡面特有的文字說明。而這次要講的就是特定的CP在局內觸發的特定buff加成。這個buff在之前其實就有了，不過很少有人發現，一般會在一些情侶檔之間出現。

比如嫦娥和后羿之間就有這樣的小彩蛋。當嫦娥和后羿在同一隊時，只要雙方靠的近，就會獲得一個buff，這個buff能給他們額外增加13點的物抗和14點的法抗，然後就湊成了1314這個數字，是不是有點浪漫呢？策劃的小彩蛋也是很別緻了。

除了嫦娥和后羿，還有一對擁有這樣的buff的情侶檔就是大橋和孫策啦~當他們在同一個隊伍的時候，孫策會增加兩點法抗，而大喬會增加兩點物抗。而且他們倆比較特殊，這個buff還有一個專屬的名字——海枯石爛。

除了情侶檔之外，還有一些特殊關係之間也有這樣的buff。不過其中有一對比較特殊，就是諸葛亮和司馬懿。在王者世界的故事裡，這兩者同在稷下求學，但終究因為目的不同而反目。所以他倆的buff就不是在同隊的時候出現的了。

而是在這兩人在局內對戰時會有一個buff。當他們在敵對陣營的時候，諸葛亮會加兩點法強，司馬懿會增加兩點法抗，而且這增加的效果對全隊有效。這麼算來，有司馬懿一方有點虧啊，法抗只是司馬懿一個人有，但是諸葛亮的法強對對面五個人都有用。司馬懿隊友表示：為什麼受傷的是我們？

大家在遊戲過程中還有沒有發現其他的小彩蛋呢？歡迎留言~
