魔獸世界懷舊服：最強矛與盾！盜賊一套打9800，惡魔術扛下反殺

2019-12-07 0條評論 18次閱讀 0人點贊

摘要

說起魔獸世界懷舊服的PVP，九大職業中已經呈現出極端分化的現象，部分職業就像一把尖刀，化作懷舊服最強的進攻之矛！也有部分職業捨棄更優秀的攻擊力，專門發展扛打擊能力，成為懷舊服最強之盾。然而就如古人所言，以子之矛陷子之盾，結果又是怎樣的呢？

說起魔獸世界懷舊服的PVP，九大職業中已經呈現出極端分化的現象，部分職業就像一把尖刀，化作懷舊服最強的進攻之矛！也有部分職業捨棄更優秀的攻擊力，專門發展扛打擊能力，成為懷舊服最強之盾。然而就如古人所言，以子之矛陷子之盾，結果又是怎樣的呢？

毀滅在手，傷害超前

現階段第一梯隊的公會早已把MC中各種極品都納入囊中，許多盜賊目前都擁有了一把毀滅之刃，再不濟也有血腥撕裂者和狗牙。本來匕首賊一套連招下來，不用徽章解控的話非板甲職業都是九死一生，現在配合毀滅這種跨版本的大殺器，盜賊所能打出的極限傷害早就超過了玩家的血量承擔極限。在懷舊服部落的一場插旗戰中，一個手持毀滅的亡靈盜賊就給他的對手造成了9800以上的傷害，而且還是在連續控制的情況下打出來的。他的對手只來得及利用極短的控制間隙釋放技能，最終承受了這近萬的輸出。

T1連接，減傷60%

一個大半身T1套裝的惡魔術，血量一般可以達到4800往上，因為魔獸世界懷舊服的術士T1其實是PVP套裝。術士T1是高耐力低法傷的套裝，還有15%耐力加成的頂層天賦，所以4000血在術士這裡屬於PVP入門級別。同時惡魔學識大師和靈魂連接天賦，可以為帶虛空的惡魔術提供10%物理減傷和30%傷害轉移，自己本身護甲減傷20%，加起來就是60%以上的物理減傷。面對純物理傷害，4800血的惡魔術實際血量超過了1萬，再加上犧牲護盾和治療石，理論上每個成型的惡魔術都是一個5人本BOSS級別的怪物。

9800傷害，依然不致命

在插旗中，毀滅賊的對手正是惡魔術，而且是帶狗的亡靈術士，沒有昏迷抵抗和10物理減傷。盜賊的控制出了些許偏差，致盲與鑿擊沒有完美銜接，讓術士扔出了死纏回血加控制。術士在這控制的間隙中把幾個DOT技能給上，隨後又陷入了無限昏迷的狀態。術士也不敢瞬招虛空犧牲，只是吃掉了治療石，因為瞬招惡魔需要0.5秒讀條，而且吃掉胖子後就沒有連接減傷了，很可能一招就躺。傷害統計上顯示盜賊一直控著惡魔術打到9800的傷害，但就在術士還剩200多血時，系統顯示術士獲得了決鬥勝利。原來DOT和地獄犬的輸出已經磨死了盜賊，決鬥中盜賊遵守規則沒有吃任何藥劑。

最強矛與盾，單挑都別惹

只看這一場決鬥，最強的矛與盾中，似乎還是盾更勝一籌，然而那名亡靈賊並未發揮到職業的極致。熟練的盜賊面對術士這種無法解除昏迷和癱瘓的職業，是可以做到無縫控制的。在兩套技能之內基本可以保證打出成噸輸出，惡魔術想贏盜賊，理論上必須保證自己的硬度可以扛過盜賊兩套，然後才可以反打。喜歡手游的可試試BT版的，神兵神裝和資源都是系統送的，而且還有用不完的遊戲幣，你可觀住工重晧樂嗨手游獲取。 d不過在實戰中，其實沒有幾個賊能做到完美控制，而且也無法保證自己全技能在手。上面的惡魔術在承受了9800的巨額傷害後，卻還是反殺了盜賊，就是最好的佐證。當然，其他人最好單挑都別惹這兩位！

當然，如果在決鬥中盜賊面對的是獸人術士，那又是另一個故事了。魔獸世界60級的最強矛與盾，匕首賊和惡魔術當之無愧，然而不管在戰場還是野外，基本都是團戰和小隊戰，1V1隻存在於插旗中。所以儘管懷舊服有著最強的矛與盾，卻並不意味著它們就是遊戲里最強的PVP職業！
