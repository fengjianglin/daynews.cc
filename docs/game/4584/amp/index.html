羅布樂思開發者日誌分享：遊戲也是表達思想的一種渠道

天天要聞
2019-12-06

摘要

遊戲之所以被稱為“第九藝術”，正是因為一款優秀的遊戲也能擁有在音畫以及思想內核方面的頂級藝術表現力。就我個人來講，陳星漢的《風之旅人》完全可以稱得上是藝術品。

遊戲之所以被稱為「第九藝術」，正是因為一款優秀的遊戲也能擁有在音畫以及思想內核方面的頂級藝術表現力。

就我個人來講，陳星漢的《風之旅人》完全可以稱得上是藝術品。《風之旅人》給了我一種很奇妙的感覺，整個遊戲都透露出一種無拘無束的禪意和一抹淡淡的哀傷，所以在得知羅布樂思這個「創意作品徵集大賽」之後，我便想通過羅布樂思來做出一款同樣具有禪意的遊戲。

由於我本人其實並沒有開發遊戲的基礎，所有的構想都只是在腦中預演而已，就好比早已在腦海中構建好一副油畫，卻不知如何下筆一般。

好在羅布樂思的官方教程十分詳盡，我在經過了一段時間的學習之後便略微掌握了一些門道，於是我便嘗試創造我的第一個作品。

這個作品我暫時將其命名為《空靈山谷》，這是一款類似於障礙跑類型的遊戲，玩家需要通過難度逐級增加的四個階段來到達終點。

第一階段我為其命名為初心，這算是難度比較低的關卡，而這一部分意為初臨世界的生命雖會遇到一些困難，但只要不太過偏離路線，都會穩穩的度過。

第二階段我則將其命名為勇氣，這一階段代表著少年與青年交替之時，就如人生許多影響今後人生走向的抉擇都會在這個階段發生一般，在遊戲中玩家也需要做出一定勇敢的決定才能找到通往下一階段的路。當你以為無路可走時，不妨換個角度來看，路其實就在前方，只需勇敢的邁出那一步。

第三階段我將其命名為力量，代表著而立之年，這時人生中年富力強的時刻，而這個時候人正當壯年，正在慢慢的往中年靠攏。

這部分的場景我設計得稍顯凌亂，因為人生中的這個時刻也是大家需要面對許多茶米油鹽日常瑣事的階段，往往中年危機也是在這時發生，唯有從瑣碎中摸索出生活之道，所以在這一關中我在加入了許多迷惑性的路徑，一步踏錯，便是深淵，玩家只有找到正確的路線才能通往下一階段。

第四階段我命名為智慧，這時代表著人生中「知天命」的階段，經過大半輩子的沉澱和閱歷積累，一些明顯的困難已經可以輕鬆的跨越，所以我在場景上將障礙設置的非常不顯眼，玩家需要仔細辨別才能通過障礙，抵達終點。

看完上面的圖片大家也應該能夠發現，我在關卡設計上借鑒了一些紀念碑谷的構造和元素，目前遊戲基本上到了後期收尾階段，希望完成之後在遊玩時能給大家帶來一些感悟。

同時也要感謝羅布樂思給了我這個機會讓我通過遊戲來表達我的想法和態度，也非常感謝官方提供的技術支持和教程，大家如果再開發上有任何問題都可以去官方QQ交流群提問，將會有專人一對一解答。
