FGO喀耳刻簡評：稀有紅卡寶具連發從者，連續控制能力強

天天要聞
2019-12-07

摘要

FGO國服1500萬下載活動開啟了五星限定從者阿比蓋爾的卡池，同時也UP了另外兩名常駐四星從者喀耳刻以及哪吒，那麼這次卡池的“副產物”如何呢？本期就來說一說四星單體術階喀耳刻的能力。

FGO國服1500萬下載活動開啟了五星限定從者阿比蓋爾的卡池，同時也UP了另外兩名常駐四星從者喀耳刻以及哪吒，那麼這次卡池的「副產物」如何呢？本期就來說一說四星單體術階喀耳刻的能力。

從者寶具：喀耳刻的寶具為紅卡單體輸出，賦予敵方單體豬化（無法行動）狀態一回合（對部分特殊敵人無效），敵方單體暴擊發生率下降10%~50%三回合（效果隨OC等級提升）。

從者技能：喀耳刻的一技能效果為增加自身60%~120%NP，充能時間9~7回合。

二技能效果為賦予敵方全體500~1000毒狀態三回合，防禦力下降10%~20%三回合，充能時間7~5回合。

三技能效果為提升己方全體30%~50%暴擊星掉落率三回合，解除弱化狀態，充能時間8~6回合。

職階技能【陣地建造】自身藍卡性能提升8%，【道具作成】自身弱化賦予成功率提升10%。

從者優點：①喀耳刻的稀有的可以進行寶具連發的紅卡寶具從者，帶上寶石翁搭配雙孔明即可輕鬆達成寶具三連發，即便不是最佳配置，寶具二連也是比較輕鬆的，並且進行寶具傷害的同時，還能有效控制敵人，將敵人變成豬仔，所以連發寶具也就意味著連續控制，效果極佳。

②喀耳刻的輔助作用也不錯，除了寶具有控制效果之外，二技能有敵方全體降防和毒狀態賦予，三技能有高幅度並且持續時間長的掉星率提升，以及弱化狀態解除，都是比較實用的效果，且自身能夠在開局不依賴輔助就迅速開啟寶具，也容易帶上嘲諷禮裝在完成自己的使命之後退場，是個合格的工具人。

③道具作成效果讓喀耳刻的控制效果更加容易命中，大量對敵人的debuff就更加有價值了。

④一技能升到8級就基本可以開啟寶具了，也算可以省一些重要的材料。

從者缺點：①變豬的範圍比較難確定，所以控制效果也存在一定的不穩定性，除了直接無效之外，也還是有被抵抗的可能。

②自身的指令卡性能很差，所以寶具幾乎得靠自身或者輔助充能，以及藍chain來達成，自身的輸出也不高，所以真空期還是有點乏力。

③又是一個釘子地獄，需要吃180個釘子，其他材料的消耗也不少。

整體評價：周回和高難都有一定的用處，整體來說還是非常不錯的從者，自充大過天的遊戲，意味著喀耳刻無論何時何地都不會被淘汰，一直可以憑藉自身優秀的充能處於高評價的地位中，並且喀耳刻至今還沒有被強化，未來也是還有機會繼續變強的。

從者卡池：本次卡池是喀耳刻唯一的單UP卡池，不過喀耳刻是常駐四星從者，在聖晶石卡池有歪到的機會，未來的職階日替卡池和四周年卡池也有多UP機會，從漫畫了解FGO第二捲髮售紀念卡池會有喀耳刻的雙UP卡池，當然了，在今年1500萬下載活動的54選1和明年1800萬下載活動的60選1都是可以選擇的。
