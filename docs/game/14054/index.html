雲頂之弈：沙漠4劍 vs 海洋4神，人數有差距，結局有點可惜

2019-12-11 133條評論 1,129次閱讀 186人點贊

在雲頂之弈中，最熱門的自然就是法師和掠食者陣容了，其次就是狂戰士的陣容，還有3魔法3召喚的陣容，而沙漠和森林等陣容就比較冷門了。不過，小編還是比較欣賞冷門陣容的，於是最近也嘗試了一波沙漠劍士的陣容。

4沙漠可以削減對手90%的護甲，直接讓對面護甲變成個位數，再配合劍士的多次攻擊，真是非常棒的組合。小編的運氣也不錯，開局就湊出了3沙漠加2劍士，之後小編又湊出了劍士裝備，組成了雲霄沙漠4劍士陣容。

這樣的陣容戰鬥力還行，起碼不是一直被吊打，而到了4-5的時候，小編也湊出了4沙漠的組合。4沙漠4劍士，在小編看來已經足夠了。接下來還是走精英路線，多刷新升3星比較好，反正6劍太難湊了，還是4劍吧！

在之後的戰鬥中，小編也是保持了7級的狀態，很快把戰爭跟鱷魚升3星了，整體的戰鬥力也還行。血量也是不錯，就這樣一路打入了決賽局，而剩下的對手，最強的則是一個還用4神的陣容。

4守護神在4沙漠面前，一樣是比較脆的，但對面是9人口的，而且還有打控制的存在。輸出雖然不是特別高，但小編一方特別脆。開打之後一波控制之下，小編就被輕鬆放倒了，尤其是3星的戰爭，作為主輸出，直接被對面岩雀彈起，然後輕鬆被集火秒殺，真是非常的悲劇。

最後，小編只拿下了第二名，真是非常的可惜。雖然後期小編跟對手一樣，都是3個3星英雄，但對面有9人口，還有打控制的，小編則是純粹的輸出陣容，又脆又沒有控制，人口還少一些，真是太慘了，果然4劍沒有資格吃雞嗎？

對此，小夥伴們怎麼看呢？你們覺得劍士陣容有前途嗎？
