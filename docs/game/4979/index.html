聖鬥士星矢：不要小看聖域統領，雖然是B卡，但也能打出爆炸傷害

2019-12-03 0條評論 17次閱讀 0人點贊

摘要

在聖鬥士星矢遊戲中，有一類卡片的鬥士是玩家們不被看好的。比如C類鬥士，還有一些B類卡裡面比較雞肋的鬥士。玩家們通常都把這些鬥士升級成長魔典書了。

在聖鬥士星矢遊戲中，有一類卡片的鬥士是玩家們不被看好的。比如C類鬥士，還有一些B類卡裡面比較雞肋的鬥士。玩家們通常都把這些鬥士升級成長魔典書了。其實像C類鬥士本身是最適合升級魔典書的，獲取比較容易，在商店裡道具選項中，在軍團和榮譽欄里就可以看到普通星石，普通星石可以抽到C類鬥士，運氣好的話也能抽到A類鬥士。

那麼在B類鬥士中，也有性價比超高的，甚至可以媲美S類鬥士。比如遊戲的主角星矢，就是其中一個。基礎屬性非常均衡，同樣也擁有四個技能，天馬彗星全還能施加虛弱狀態，對殘血的敵人能起到收割作用。

還有一些B類鬥士就很一般了，基本都是三個技能，玩家基本都不會拿來培養。但是今天要說的聖域統領，絕對不在雞肋的行列。

聖域統領雖然定位是為B卡，但總體屬性很平衡，沒有側重點。技能也少了點，才三個技能。但是技能提供的效果非常好，不僅有可觀的輸出還能配合特定陣容打出超高傷害。

一技能：切割

迅猛攻擊1名敵人，造成100%的物理傷害。

普攻沒什麼亮點，造成傷害百分比還是很高的。個人感覺同樣百分比的情況下，物理攻擊要比念力攻擊的傷害要高，朋友們有沒有感覺到？

二技能：敗血巨鐮

對一名敵人造成三段傷害，每段造成50%的物理傷害，若目標處於流血狀態下，會額外提升本次攻擊50%的傷害。

二技能是聖域統領的核心技能，個人覺得凡是帶多段傷害的技能，輸出都不會太低。流血狀態下還能提升50%的傷害，滿技能可以達到210%的總傷害，還不算出血狀態下額外的50%，可以說傷害相當可觀了。

三技能：血之嗅覺

暴擊時有50%概率對敵人附加流血效果。流血效果可疊加5層，分別造成目標最大生命上限1/6/15/31/57%的傷害，最高造成攻擊力70%*層數的傷害，持續兩個回合。

這個被動技能一看是不是舒服多了？首先可以確定他是打暴擊流的，因為暴擊能觸發流血效果。若果能疊加5層，最大能造成350%的物理傷害，而且不要忘了流血的傷害是基於敵人最大生命計算的，血越厚傷害越高，這樣一算起來，總傷害將大大超越350%的物理傷害，獲得的瞬間爆發力極強。

陣容分析

新手推薦：莎爾娜+六分儀+聖域統領+楓+艾歐里亞+艾爾扎克

莎爾娜開局提供速度，六分儀祈願聖域統領，待聖域統領打出出血效果的時候，艾歐里亞的光速拳能打出6段傷害（目標受控狀態必定觸發6段光速拳），期間楓可以協助攻擊敵人，運氣好的話楓可以在一個回合進行四次助攻。艾爾扎克召喚的觸角傷害也非常高，而且觸角也能誘騙一些技能。整體來說這套陣容攻擊高、爆發強，非常適合新手玩家們。

以上是關於聖域統領的一些總結。聖域統領絕對是B卡里不易發掘的人才，只要我們善於開發運用，一定能發揮他們的長處。朋友們不要猶豫了，感興趣的快來嘗試一下吧！
