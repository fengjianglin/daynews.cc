“吃雞”12.20的新軍需是否值得入手？這3個方面，應該考慮清楚

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友刺激哥。看到12.20上線的軍需寶箱，刺激哥也是猶豫了良久，內心也是思考了很久，這次上線的新軍需，是否值得入手？

其實，在入手新軍需的時候，我們應該從3個方面出發。如果這3個方面，我們能夠說服自己，那這次的新軍需完全是可以入手的。

話題：「吃雞」12.20的新軍需是否值得入手？這3個方面，應該考慮清楚！

我們在入手新軍需的時候，需要考慮到的第一個方面，就是性價比。首先要看一下自己的預算，如果您打算進行「十連抽」，則是可以進行嘗試一下的。

另外，也有部分玩家，有著「佔小便宜」的心理，會嘗試一下「單抽」是否能獲得一些比較好的物資。

根據刺激哥多年的經驗來看，絕大多數的「單抽」，並不會出現極品。單抽出奇蹟，只屬於少數玩家。

6R來隨機抽取一件，我們可能用不到的東西，基本上是比較虧的。

第二個方面，新軍需中的物資的應用場景，是否能夠滿足我們的需求。在《和平精英》這款遊戲中，有些衣服是具有天然的保護色的。

其中，上上個軍需寶箱中的「東方飛龍套裝」，是應用場景比較廣泛的套裝之一。

而本次軍需的物品的應用場景，其實並不是特別的廣泛。如果您是為了收藏，那確實可以入手這次的新軍需，如果為了應用到實戰。這次的新軍需，可能就有些差強人意了。

如圖所示，當我們戴著頭盔的時候，背後有一個高高隆起的熊頭，以及背包的。在伏地的情況下，加大了我們被對手發現的可能性。

第三個方面，新軍需中衣服的精緻程度。需要注意的是，遊戲商店中時裝的演示效果，與實際效果是有著很大的不同的。

比如，某些衣服在遊戲商店中，有著隨風飄動的效果，但到了實際當中，可能就沒有這樣的效果了。

除此以外，還要考慮新軍需中的時裝的「穿模問題」。如圖所示，這次所上線的新軍需的某一件服裝，就有輕微程度的穿模。

其次，某些擁有擺動效果的衣服，在實際應用中，就有著一個十分尷尬的情況。

在此，我們以SS4賽季手冊中的100級獎勵的紫色裙子為案例。如圖所示，當遊戲角色，處於趴下的狀態是，遊戲角色的裙邊，出現的情況，是違反現實生活的常理的。

除此以外，遊戲中的某些大衣，也是同樣存在著，這樣的「違和感」的。

這不得不讓人擔心，這次上線的新軍需的時裝，是否也會有著同樣的效果。

說實話，《和平精英》的遊戲品質，是沒有任何問題的。但是每次更新時裝，更新軍需的時候，總是會出現這樣或那樣的小問題。

其實，很多的時裝，在正常的跑動或移動時，都沒有太大的問題。有問題的，通常是一些小的細節問題，如使用背包的穿模，駕駛或乘坐雙人載具的穿模，遊戲狀態下的穿模問題。

這些，都是《和平精英》的遊戲美工，應該考慮的。在細節方面，遊戲美工做的還是有點不到位。

至於武器皮膚，這次上線的新軍需，並沒有太大的亮點。相對比較中規中矩，並沒有帶給玩家眼前一亮的感覺。
