又一款全新皮膚上線時間確認！微信區QQ區數據爆料，誰更強？

2019-12-23 3,387條評論 4,877次閱讀 3253人點贊

本文原創首發於公眾號：電競怪客

hello大家好~我是怪客君~關注並私信我「戰令」即可領取戰令進階卡！

眾所周知，王者榮耀分為微信區和QQ區，大部分的玩家應該都會選擇其中一個區作為自己的主玩分區，對於另外一個區的遊戲環境則知之甚少。那麼不知道大家有沒有過這樣的疑惑？有可能大家明明玩的都是同一個遊戲，但是因為分區不同，卻獲得了兩種完全不一樣的遊戲環境（體驗）。

有很多玩家認為，造成這種區別的原因是微信區玩家相比QQ區玩家遊戲素質要高一些，而QQ區玩家則比微信區玩家技術高一些。那麼事實究竟如何呢？今天怪客君就通過一些數據來給大家分析一下QQ區和微信區究竟有哪些不同之處吧~

玩家年齡

怪客君認為，微信和QQ的差別首先取決於年齡段，步入社會的玩家由於社交圈等因素普遍存在於微信，而使用QQ玩王者榮耀的玩家基本都比較年輕。

年輕的好處是反應快、操作高，但容易心浮氣躁，反觀微信區的大多數玩家由於大多身在職場，更多會注意遊戲整體運營與跟隊友的配合。

根據極光大數據的調研結果，手游用戶較為集中於青少年人群，半數手游用戶年齡為26-35歲，36歲及以上中老年人群佔比僅約16%，而大部分的王者榮耀玩家竟然集中於已經工作的群體！

玩家數量

怪客君對比了QQ區和微信區的小區搭建數量，發現微信區中小區的數量更多，這也可以猜測微信玩家數量是多於QQ玩家的。而從前文提到的玩家分布也可以看出愛用微信的上班族實際上佔了王者榮耀玩家數量的大頭。

消費能力

顯而易見，由於微信區的玩家年齡較大，普遍經濟獨立，所以消費能力也就更強。

同樣的，微信區的「老闆」數量也比較多，張大仙也曾在直播中吐槽了這一點。

遊戲水平

有玩家說，QQ在同段位的遊戲水平比微信高一些，所以有些代練在微信區開的價格也比QQ區低。怪客君去知名代練平台某游查詢不同大區在永恆鑽石段的上分價格發現，這種傳言不屬實，起碼大部分陪玩代練在不同分區的同一段位價格是差不多的。

當然，代練畢竟違背遊戲精神，怪客君不建議大家選擇這個上分途徑。

而怪客君在QQ區和微信區練了兩個小號都上到了星耀一，很明顯的感受就是微信區單排的隊友即使技術欠佳，但比較穩紮穩打也會聽你發的信號，而QQ區單排雖然會出現操作帶飛的隊友，但大家都比較看重自己的KDA而容易忽略了遊戲的整體運營。

當然，怪客君分析的只是一部分，每個區也是存在相似的小白或大神的，而到了高分段，其實兩個區的體驗是差不多的。

英雄偏好

微信區盛產輔助坦克，隊伍整體非常好，QQ區盛產刺客輸出，個人操作滿場跑。BP選擇也很不一樣，QQ區打野多，所以鑽石星耀局天天ban猴子虞姬瑤等英雄(這三個在微信區基本不上ban位)，微信區法師射手多，所以常ban婉兒蘭陵王李白等刺客英雄。

不過，微信區也常常出現全隊無打野還全禁打野的情況，張大仙在玩微信區時也碰見過一場BAN四個打野。

遊戲環境

很多玩家都反映，微信區的遊戲環境要比QQ區略勝一籌。QQ區的玩家很多容易情緒消極，遇到逆風情況，遊戲界面右上角的投降也是亮個不停，更有甚者直接掛機、不參團或者送人頭。當然微信區這種消極也肯定不在少數，但更多的則是默默無聞地坑著，讓人覺得他們的確是在很努力地戰鬥。所以這也是一些人覺得微信區更簡單的原因，其實，你的勝利歸功於你努力的隊友，而不是消極的對手！

怪客君認為，兩個區雖然因為其特性而存在一些微小的差別，但大體上還是差不多的，不必去爭個孰強孰弱，傷了和氣。

遊戲系統

其實不僅微信/QQ區存在區別，不同系統的玩家畫風也是不一樣的。不知道大家有沒有發現，在半夜單排總是容易輸？最近有網友通過觀察排位記錄發現造成這一現象的原因，竟然跟玩家所使用手機系統相關。

他發現0點之前單排排的單排隊友都是同一個系統的！而0點之後會單排到的單排隊友是兩個系統混合的！

也就是說零點之前單排，你基本上匹配到的都是和你同系統的玩家，而零點之後單排遇到的是安卓和蘋果兩個系統的遊戲玩家。而正是因為兩個系統玩家的遊戲風格不同，造成了勝率普遍偏低。

怪客君認為，雖然遊戲勝率或多或少的會受到區服和系統的影響，但最關鍵的因素還是在於個人的操作和意識，所以不要對不同區/系統的玩家有刻板印象。畢竟高手換了手機還是高手，菜鳥換了手機還是菜鳥，本質是不變的。

貂蟬FMVP

最後是在下個月即將上線的貂蟬FMVP皮膚，該皮膚的剪影正式發布，根據貓神之前的描述，這款皮膚是粉色頭髮，並且大招效果十分震撼。

冬季冠軍杯總決賽將在1月4日開啟，所以貂蟬FMVP皮膚預計將在1月4日左右上架，首周折扣1350點券，參與活動最低只需要962點券，該皮膚屬於普通傳說品質，擁有專屬回城特效和專屬頭像框。
