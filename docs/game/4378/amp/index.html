Steam秋促結束，尼爾年度版開啟特惠，SE依舊“保持初心”？

天天要聞
2019-12-07

摘要

SE是一家非常獨特的遊戲發行兼開發商，其它廠商推出好遊戲，換來的是玩家們的讚美，而SE無論做出怎樣的遊戲，玩家們都會用一些詞語來表達憤怒。

SE是一家非常獨特的遊戲發行兼開發商，其它廠商推出好遊戲，換來的是玩家們的讚美，而SE無論做出怎樣的遊戲，玩家們都會用一些詞語來表達憤怒。這其實跟SE的運營方針有很大關係，就像這次尼爾機械紀元年度版特惠一樣，玩家們買爆遊戲的同時，還不忘用「真有你的呀，SE」之類的話來表達自己的心情。

給好評也不忘懟SE

在Steam開啟秋季特賣時，SE就干出過一件讓玩家大跌眼鏡的事，其它廠商都在配合G胖搞活動，SE的最終幻想系列也參與秋促，卻並未針對國區進行打折，「最多立省0%」是真的秀。而現在Steam特惠活動都結束了，SE卻讓旗下高人氣遊戲尼爾開啟打折，這彷彿是在跟Steam作對一樣。

尼爾進入Steam熱銷榜前三

尼爾這款APRG遊戲在剛登陸PC時，就因412元的高昂價格引起玩家不滿，要知道PS4版尼爾的售價不僅比Steam版更便宜還有官方中文，可到了2年後的現在，SE依然沒有推出PC版中文補丁。也許遊戲做得好真的能為所欲為，無論SE使用怎樣的騷套路，尼爾總是能在特惠時進入Steam全球熱銷榜，這次也佔據了第三的位置。

畢竟錯在SE身上，尼爾這款遊戲以及製作組白金工作室依然擁有不錯的口碑，比起遊戲廠商的風評，玩家們更在意遊戲是否有趣。喜歡日系人設、劇情向和動作類遊戲的玩家依然可以考慮嘗試一下尼爾，2B小姐姐也的確是最令人印象深刻的遊戲女主角之一。

當年的傳奇公司走到現在被大量玩家討厭的地步，只希望SE不要再「保持初心」，能找回原來的自己吧。
