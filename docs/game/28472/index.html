法爺死守奧山引熱議《魔獸世界》冰環吹風阻隔大橋天塹

2019-12-23 47條評論 1,086次閱讀 69人點贊

不得不說，隨著《魔獸世界》懷舊服中PVP玩法的加入，自然也將懷舊服推向了新的高潮之上。看今朝，奧特蘭克山谷和戰歌峽谷早已被大眾瘋狂的刷、刷、刷，一來是為了拿到戰場裝備獎勵，二來也是為了更好的軍銜獎勵。

13級軍銜

毋庸置疑，聯盟和部落各有13個軍銜等級，不同的等級自然會有不同的裝備獎勵，其中聯盟最高軍銜為大元帥，部落則是高階督軍。要想刷到第13級當然是一件很困難的事情，這需要你每天都在瘋狂的拿人頭，但俗話說「一分耕耘一分收穫」，因為你肯定會拿到最好的PVP裝備獎勵。

在懷舊服中，PVP和PVE在裝備方面並沒有明確的劃分，也就說即便你拿PVP裝備去打PVE，同樣也是一件可行的事情。值得一提，PVP武器在造型方面可是非常拉風的存在，而且在正式服中已經絕版多時，獲得這些獎勵不僅僅是為了實用，也是為了獲得拉風的效果，也是為了彌補當年的遺憾！

武器極為拉風

大元帥的雙刃刀當然是一把拉風的雙手劍，甚至要比兄弟會之劍更為拉風，該雙手劍又大又長，明顯要比兄弟會之劍更為霸氣一些。而且藍白相稱，散發出藍光的特效，拿在手中當然是十分拉風的存在，不要忘記上面還印有暴風城的徽章，獲得大元帥的雙刃刀猶如你成為了暴風城的皇家衛士！

不僅是雙刃刀，大元帥的手杖同樣是一把非常拉風的法杖，猶如大天使的象徵，要比牧師的祈福更為拉風。不得不說，每一個牧師都肯定盼望著自己拿到大元帥的手杖，這意味著你將成為大天使的化身。

再者便是大元帥的戰鬥重鎚，這把雙手錘同樣印有暴風城的徽章，藍、白、黃三色相稱，一看就會令人想到了暴風城的人類聖騎士。拿在手中猶如享受到艾澤拉斯最強的聖光，對此你難道不會為之興奮？

不光是大元帥，部落一方的高階督軍同樣也有大量的拉風武器獎勵，不過不太符合「人類」的審美，但如果用獸人的眼光來審美的話，這些武器同樣也是奧格瑞瑪的象徵！

眾所周知，在正式服中儘管會有仿製的軍銜武器，但終究是不如原汁原味更好看的，也正因如此，大元帥和高階督軍自然會成為所有人的夢想。

獎勵多多

其實不僅是武器，軍銜也會獎勵4種坐騎，而且個個身披盔甲，比如說亡靈可以騎上作戰科多獸，這當然也是十分拉風的。更重要的是軍銜獎勵的坐騎售價都是十分便宜的，也不過區區幾十金幣而已，如果你不捨得買千金坐騎，刷軍銜當然也是一種省錢的辦法。

​不難發現，隨著懷舊服中加入了PVP玩法，不光是大眾有了更多的事情可做，同時也能獲得大量的獎勵，而這些獎勵的價值遠不是其實用性能夠表達的，更像是一種身份的存在。

話不多說，趕緊為了大元帥和高階督軍而戰吧！
