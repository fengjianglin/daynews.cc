火油桶守城測試 投擲類城防器械實戰分析

火油桶是4級城防器械中的一種，屬於三種城防器械中的投擲物，可以用來克制步兵。火油桶的攻擊力並不高，但是對於步兵傷害很大。今天，就讓我們一起測試一下，火油桶對於其他兵種效果如何！

測試前置條件：

為了保證測試效果，攻方和守方均為滿科技0天賦狀態，攻方準備1萬不同的7級兵種，守方以滿狀態城牆+1萬火油桶守城。攻方以不同的兵種攻城，並收集戰報數據分析。

實戰數據分析：

步兵：步兵雖然在四個兵種中血量、防禦都很強勢，但火油桶可以剋制步兵。在步兵攻城時，火油桶有50%的傷害加成。一萬步兵攻城時損失2082，輕傷士兵3122，存活士兵4796。

而城牆耐久值消耗195800，城防器械損壞1180，待修復1180。相對來說，步兵雖然損失較少，但受傷極多。攻城戰役中，步兵的整體損傷比例為53%。

騎兵：騎兵的攻擊力要高於步兵，但防禦、血量低於步兵。而且騎兵和步兵同為前排兵種，因此實戰中火油桶也可以殺傷騎兵。一萬騎兵攻城時損失1497，輕傷2245，存活6258。

而守方城牆耐久值消耗為228895，城防器械損壞1452，待修器械1452。從戰報可以看出騎兵攻城的實力還是很強的。特別是對城防器械的壓制效果，騎兵遠比步兵要強一些。

槍兵：槍兵的攻擊是四個兵種中最高的，但血量很低。不過在野火塔、地刺等城防器械的測試中得知，槍兵的攻擊距離很遠。而火油桶作為針對步兵的城防器械，對於槍兵無法造成傷害。

但是槍兵依舊對城牆、城防器械造成的大量的傷害，守方城牆耐久值損失225637，城牆器械損失1496，待修器械1496。

弓兵：作為遠程兵種，弓兵在面對火油桶時出現了一些損傷。但1萬攻城弓兵，只損失28個，受傷士兵40人。

守方城防耐久值損失175907，損壞器械1075，待修器械1075。相對於其他兵種來說，弓兵攻城效率最低。

總結：經過火油桶在實戰測試，領主可以得出2個結論。其一，火油桶對步兵確實有極大的剋制作用，雖然不能全部殺死，但是可以對步兵造成50%以上的傷亡率，降低敵方前排實力。

其二則是火油桶依舊是只能對步兵、騎兵生效，對於槍兵、弓兵很難發揮全部實力。因此領主在設置城牆器械時，火油桶放置一部分即可，畢竟領地遭遇純步兵攻城的可能性不是很大。
