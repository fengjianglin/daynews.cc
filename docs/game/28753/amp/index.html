三隊同分，最後RNG晉級！德杯規則引質疑：贏得太快會被淘汰？

天天要聞
2019-12-21

12月21日終於迎來了德瑪西亞杯比賽的第一天，最終在經過一整天高強度的比賽之後，前四支晉級德杯八強的戰隊已經誕生，分別是JDG、VG、WE和RNG。JDG四場全勝晉級沒問題，VG和WE也都只輸了一場，分數最高晉級也沒啥問題，最後引發質疑的是RNG。當然不是說針對RNG這支戰隊，而是RNG、LNG和OMG都是一樣的比分，而且都輸給了JDG贏下了YM，三隊又互為勝負關係，RNG贏了LNG，LNG贏了OMG，OMG又贏了RNG，最後的結果卻是RNG晉級，引發了不少網友的不滿。

​主要是根據平均賽事時長來看的話，LNG贏下比賽的平均時長28分05秒，RNG贏下比賽的平均時長是30分30秒，OMG贏下比賽的平均時長是30分53秒，按照獲勝時長來看，LNG用時最短應該晉級，但是結果卻不是這樣。

根據德杯的規則來看，如果出現了三隊同分數並且相互對照記錄也相同的情況，先用獲勝時長來進行排名，也就是LNG第一，RNG第二以及OMG第三，取LNG的獲勝時長來作為標準，如果其他隊伍獲勝時長和LNG時長相差超過了150秒，則被淘汰，這就是OMG淘汰的理由，如果在150秒之內，則進行加賽。

而加賽並不是真正意義上的加賽，又開始比較勝負關係了，因為最後一輪RNG贏下了LNG，所以根據規則RNG直接晉級。按照德杯的規則來看，的確應該是RNG晉級。不過這個規則真的合適嗎？

尤其是對於LNG來說可以說是虧大了，如果LNG在和OMG的比賽中故意拖一段時間，也就是讓OMG和自己的獲勝時長也低於150秒。那麼結果就是OMG先和RNG加賽，之後贏的戰隊再和LNG加賽決出最終勝者，而又如果LNG在和RNG的比賽中再猥瑣個11秒鐘，那麼晉級的就是LNG。也就是說贏得慢的隊伍反而更有機會晉級。

​這個規則自然是非常不公平的，因為lol比賽中必定會有前期陣容和後期陣容，兩個戰術都是合理的，但是勝負的時長可能差距就很大，後期陣容就是要拖到一定的比賽時長才有可能獲勝，前期陣容一旦拿到優勢，可以迅速結束比賽。對於lol這樣比賽戰術比較多變的戰隊來說，這個規則真的太蠢了。說白了這個規則只是為了不產生加賽而已。

所以RNG晉級是完全沒有問題的，有問題的是這個規則，先看勝負，不行再看時長，看完時長後又要再看勝負。如果LNG打OMG故意拖一拖反而能加賽，他們打OMG打太好了導致自己被淘汰了。找遍全世界職業比賽，估計只能在德杯看到這樣的千古奇聞了。

​那麼，對於這次德杯的規則導致RNG晉級一事，你們有什麼想說的嗎？
