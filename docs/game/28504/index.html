為自己，造不凡 2019電魂娛樂星賽季嘉年華完美落幕

2019-12-23 1,383條評論 4,752次閱讀 2898人點贊

2019電魂娛樂星賽季於12月22日正式在鄭州市體育館完美落幕。來自全國各地的玩家見證了如火如荼的電競賽事，戰火燎燃各職業戰隊，最終由三分月戰隊摘得星賽季年度總冠軍！

星賽季嘉年華重磅對決 年度冠軍火熱出爐

2019電魂娛樂星賽季嘉年華於12月21日—22日在鄭州市體育館完美收官，三分月戰隊橫掃對手斬獲《夢三國2》職業戰隊邀請賽冠軍，捧得100萬元現金大獎；手游《野蠻人大作戰》年度冠軍邀請賽中娛樂至上戰隊以最高積分獲得冠軍。下面就讓我們一起來回顧一下吧！

娛樂賽事嘉年華 群星閃耀

伴著玩家的歡呼與掌聲，電魂娛樂星賽季嘉年華在鄭州市拉開序幕。今年是娛樂嘉年華賽事舉辦的第八年。賽事規模逐漸成熟，賽事體系不斷完善，讓本次2019電魂娛樂星賽季更加燦爛輝煌！


本屆娛樂星賽季開幕式特邀《夢三國2》核心用戶代表，火燒赤壁大區的春天；MPL職業戰隊「詩月」戰隊隊長流星、ZIYO戰隊隊長雷一；《夢三國2》賽事經理 黃勝毅；主策劃張勝，一同為玩家揭幕。隨著五位嘉賓魔劍插入劍壇，炎魔封印被破，炎魔現身賽場！強大的氣勢震撼全場，引的玩家驚呼連連，與此同時，也宣告著2019娛樂星賽季，正式開幕！


夢三國2 群雄逐鹿 巔峰對決

在第一天的比賽中《夢三國2》的ZIYO戰隊和三分月戰隊爆冷跌入敗者組，詩月戰隊以勝者組第一的身份率先挺進總決賽；而在第二天的比賽中，遙遠止步殿軍，敗者組決賽中三分月戰隊擊敗ZIYO，與詩月會師冠軍之戰。比賽氛圍變得更加緊張，每位選手都為了冠軍榮譽全力奮戰。

在決賽開戰前，我們特別邀請了《夢三國2》的遊戲製作人：姚少龍先生為玩家致辭。姚少龍先生提到，在星賽季正式開戰前，《夢三國2》在網頁上開啟了星賽季的戰隊投票支持活動，玩家紛紛支持，滿含激情，各個戰隊的支持次數，累計達到近300W。《夢三國2》帶著各位玩家的心意和支持，在星賽季的現場宣布，在星賽季的百萬冠軍獎金基礎上，額外增加10W元人民幣作為冠軍獎金，希望星賽季能夠持續的走下去，也希望戰隊成員能給我們帶來更多精彩的比賽。夢三不老，我們不散！

最終由三分月戰隊獲得了「2019電魂娛樂星賽季《夢三國2》MPL職業戰隊邀請賽」總冠軍，詩月戰隊獲得亞軍，ZIYO戰隊獲得季軍，遙遠戰隊獲得殿軍，至此，《夢三國2》職業戰隊邀請賽圓滿落下帷幕，讓我們恭喜三分月戰隊！



野蠻人三方混戰 激情開打

在《野蠻人大作戰》年度冠軍邀請賽中，可愛多、百味人生、娛樂至上三隻戰隊分別以不一樣的戰術體系激情混戰，3V3V3的遊戲模式讓整場比賽格外精彩。

最終，娛樂至上戰隊奪得了「2019電魂娛樂星賽季《野蠻人大作戰》年度冠軍邀請賽」總冠軍，可愛多戰隊獲得亞軍，百味人生戰隊獲得季軍。

夢三國手游復刻經典 花樣百出

除了《野蠻人大作戰》外，《夢三國手游》同樣也作為表演賽，為大家帶來精彩的競技體驗！本屆娛樂星賽季，《夢三國手游》邀請到了詩月和遙遠兩支戰隊為大家帶來精彩的對決，在對戰中，遙遠戰隊的董卓以其「出神入化」的鉤子引得玩家爆笑不止，賽事其樂融融。

老朋友解說更激情 畫龍點睛

本屆娛樂星賽季，我們特意請到了《夢三國2》的老朋友，同時也是老牌職業選手，希望戰隊的亂迷魂，和奢華來作為現場解說，他們獨特幽默的解說風格，對遊戲內容的超強理解，團戰即時的講解，都讓玩家有了良好的觀賽體驗，將比賽氛圍推向了高潮！

合作夥伴助陣 星賽季如虎添翼

本次星賽季，特別合作夥伴戴爾電腦也為選手提供了極佳的外設體驗，對於《夢三國2》這樣的競技類遊戲來說，一次輕微的延遲或細小的卡頓都可能直接影響比賽的勝負，因此設備的性能至關重要。此外戴爾G系列遊戲本搭載了AWCC外星人智控中心，不管是RGB燈效還是遊戲聲效，甚至CPU溫度也能隨時監測，為玩家帶來從未有過的便利。還支持一鍵散熱的「G模式」，熾熱戰意極速冷卻，讓玩家可以零負擔盡情酣戰；最高可搭載第九代酷睿I7處理器和GTX 2070顯卡等強悍硬體，搭配144Hz刷新率的IPS防眩光屏幕。進入https://j.youzan.com/gez8dY即可體驗！

至此，2019電魂娛樂星賽季圓滿結束。八載征程，星賽季已化繭成蝶，但輝煌仍在遠方，我們當以時為翼，乘風破浪，奮勇前行！為自己，造不凡！
