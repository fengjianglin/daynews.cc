《爐石傳說》：比臟牧還要無恥，王師傅的塞翔牧玩得對方痛不欲生

天天要聞
2019-12-18

王師傅，又名「臟鼎」，以玩臟牧出名。但由於牧師的強度幾個版本都不高，搞得老王也不咋玩臟牧了。

然而最近王師傅終於又回到了牧師的懷抱，並且不偷不搶，還給對面送「溫暖」。

對面：謝謝你啊！

厄運信天翁出來時，大家都覺得這是張針對宇宙卡組的牌，一個單純的3費優質隨從。但它在狂野出現後，玩家的想像力就按耐不住了。

比如這套塞翔牧，往對面卡組裡塞一大堆1/1，想像一下對面抽到一堆翔的感覺，是真的快樂。

玩法核心很簡單，就是不斷地復活厄運信天翁，給對面卡組一堆1/1的信天翁，然後發表情就行了。

打得順的話，一會兒對面卡組裡會有幾十張信天翁。

不過到後期，對面的疲勞會領先我們很多，所以需要大主教將對面的卡拿過來。之後再用挖寶的黃金猿把那堆垃圾都變成傳說隨從，這樣就算打疲勞對面也不是我們的對手了。

和其他雜技卡組一樣，這套牌並不算強，主要還是以快樂為主。看著對面一回合摸一坨翔，再發一個「哇哦」，嘲諷度直接拉滿。

除了快樂，這套牌還極為克制那些想要把卡組抽空，或者無限資源的卡組。

比如青玉德洗了一大堆青玉護符，最後看著這一堆信天翁……嗯，他加你好友千萬別接受。

又或者是元氣騎想要把自己牌庫過光用元氣彈，這堆翔可以讓他永不疲勞，感覺自己胸前的紅領巾更鮮艷了呢。

不過臟歸臟，發發表情就完事了，咱得有素質，惡意燒繩這種事可不要干哦。
