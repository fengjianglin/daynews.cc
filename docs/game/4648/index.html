DNF：12.10版本商城更新分析，4900點券獲得幻影光環和一套裝扮

2019-12-06 0條評論 16次閱讀 0人點贊

摘要

體驗服12.5更新的商城內容已經維護結束，本期就給大家對這一個新的內容進行一個詳細的介紹。

體驗服12.5更新的商城內容已經維護結束，本期就給大家對這一個新的內容進行一個詳細的介紹。

第一個：冬至暖心禮盒

10號售賣到22號，禮包售價為666點券，無法使用代幣券購買，禮盒總共可以打開兩次，每次打開固定獲得1個白斯追擊湯圓，單刷超星空可以百分百預見白斯5次；30點疲勞餃子，但是限制95級角色才能使用；15張超星空裂縫通行證；所以這一個禮包實際上就是666點券買30張超星空的門票，這樣是不是覺得虧了，當然不會虧啦，當第二次開啟時可以額外獲得1222歡樂代幣券，所以實際上666點券換到了1222代幣券和30張超星空的門票。

禮盒無法交易，但是開啟的獎勵都是帳號綁定的，每個帳號限制購買一個。

第二個：白色獸語禮包

這一次沒有大家期待的雪人套，但上架了期待了很久的了白色獸語套裝，並且分為1000點券的30天體驗禮包和9900點券的永久禮包。限時禮包無法用代幣券購買。

1000點券的禮包所獲得的時裝和寵物以及稱號都是30天期限，此外額外獲得每日驚奇禮盒5個和每日心動禮盒5個，以及新春禮包9折券。

每日驚喜打開隨機獲得2/5/8/10個波塞姆的祝福花瓣，也就是超星河裂縫的入場門票，低保就是10個。

每日心動禮盒打開隨機獲得靜謐獸語裝扮套裝、純凈的增幅書，+7到+10的增幅券，2個增幅器。低保10個增幅器，狗拖一套裝扮。

永久禮包打開有裝扮一套，出皮膚外都可以染色，但是都不能分解合成，適用高級裝扮屬性。

稱號屬性為增加本職業的一級BUFF技能，禮盒打開後無法交易；寵物三種外觀自選，屬性都是增加1級的BUFF技能等級，所以裝扮給大號，其他的給小號打開。

第三個：魔盒更新

魔盒新增波塞姆的祝福花瓣禮箱，打開獲得100個帳號綁定的波塞姆祝福花瓣，也就是可以刷一百次超星河裂縫，相當於300個精鍊的時空石，大家猜一下會賣多少錢？禮盒是50個門票，都是可以交易一次的哦！

第四個：阿拉德探險記

全新的戰令玩法，當然沒有購買高級探險家特權的玩家只有初級冒險家資格，完成每日、每周和賽季任務獲得經驗，至於什麼任務以正式服更新為準，目前體驗服顯示不完整。通關這些任務獲得經驗升級冒險家等級。

購買高級探險家資格證需要4900點券，無法使用代幣券，每個帳號限制購買一次；購買豪華探險家禮包15600點券，可以直接將冒險家等級提升到40級。

高級探險家可以獲得更好的獎勵，並且完成每日任務獲得雙倍經驗值，所以想要快速拿獎勵的玩家需要購買豪華禮包，佛性玩家選擇4900點券就好。

冒險家等級總共有80級，獎勵內容非常豐富，這裡我們不一一的介紹，大量的徽章禮盒。、絕版裝扮隨機禮盒、絕版光環隨機禮盒、春節寵物、稀有裝扮隨機禮盒等級，而高級冒險家的終極獎勵才是最關鍵的，幻影炫光光環，一套未來科技套裝，可惜女聖職和槍劍士沒補全。

以上為體驗服更新，一切以正式服更新為準哦！
