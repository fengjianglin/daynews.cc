迷室系列新作：給你克魯蘇式恐怖下的極致體驗

2019-12-06 0條評論 3次閱讀 0人點贊

摘要

速評幫你快速找到合適的遊戲誰會喜歡TAThe Room系列的粉絲，喜歡密室逃脫、解謎類遊戲的玩家。T

速評

幫你快速找到合適的遊戲

誰會喜歡TA

The Room系列的粉絲，喜歡密室逃脫、解謎類遊戲的玩家。

TA的優勢

1．畫面與前作相比有明顯的進步

2．物品質感和光影效果在場景中的表現驚艷

3．機關解謎元素設計巧妙，互動方式有趣

4．3D建模呈現出超真實的效果

5．遊戲營造出了緊張卻不恐怖的氣氛

6．劇情通俗易懂

TA的不足

1．每次載入遊戲都要等待90秒的時間，優化明顯不夠完善

2．缺乏爆炸性的創新，與前兩作相比沒有太大的突破

3．目前只登陸了手游平台，沒有主流的主機版本

下載方式

深評

帶你深入了解遊戲

The Room（迷室）系列的第一部作品於2013年首次上線，由FireProof Games開發，至今為止已經先後推出了《迷室1》、《迷室2》、《迷室3》以及《迷室：往逝》，共四部遊戲。憑藉出色的環境氛圍塑造能力、精緻的機械機關設計以及出色的遊戲劇情，迷室系列也深受密室逃脫類遊戲玩家的喜愛，在目前眾多解謎、密室題材類的遊戲中有著顯著的地位。

另外，新作迷室：往逝也很好地繼承了系列的傳統，無論是遊戲性還是畫面，依舊是此類型遊戲中的佼佼者，而且迷室：往逝由網易遊戲代理髮行，因此遊戲內自帶官方中文。

集系列之大成，畫面效果堪比現實

迷室系列之所以能夠獲得如此高的讚譽，其中一個關鍵原因就是它的畫面。而迷室：往逝作為迷室系列的第四部作品，距離上一部作品的發售已有4年之久，經過了長時間的優化和積累，它的畫面也自然比前作更出色。

迷室：往逝依舊使用了第一人稱的視角。但本作的場景偏暗，在一開始進入遊戲的時候，小屋裡沒有燈，但是背景並非是純黑，而是介於灰色和黑色之間，隱隱約約能夠看到遠處的物品，這樣的畫面效果讓整個場景更有層次感，此時閃電從窗外照進屋裡，模糊的背景突然變得清晰，彷彿所有的物品都瞬間湧入了場景里，很有震撼感。

其實，這種表現手法在電影中是十分常見的，可以有效地營造出「神秘」的氛圍。而在迷室：往逝中也多次使用到了這種手法，使得環境的塑造比前作更加出色，由內而外的散發著氛圍感和真實感。

除了場景畫面外，迷室：往逝物品的3D建模也非常出色，表面的光澤感和質感都得到了明顯的提升，而且材料也豐富了許多，類似花梨木、銅、鐵、不鏽鋼、陶瓷、玻璃等，一應俱全。而且打在物品上的光線會根據視角的調整而發生明顯的變化，一件物品上囊括了明、暗、影子等多種效果，因此玩迷室：往逝的時候，常常會感嘆建模和貼圖居然可以做的如此寫實，跟現實幾乎如出一轍。

玩法可圈可點，但缺乏核心創新

從玩法來說，迷室：往逝跟前作沒有太明顯的區別，核心依舊是與物品互動，通過機關的解謎來驅動下一個機關的啟動，如此反覆。而操作方式也非常簡單，雙擊物品相當於移動並調查，兩隻手指同時滑向屏幕中心則是離開。

在與物品交互的過程中，視覺上可以獲得極大的滿足，旋轉、推動物品的操作動態非常逼真，幾乎與現實無異，大大提高了玩時的新鮮感，哪怕是老玩家也不容易產生視覺疲勞。

此外，迷室：往逝較前作有更高的自由度，在解謎的過程中，物品與物品之間沒有強制關聯，因此在本作中想要快速找到線索，解開機關需要更豐富的想像力。但總體的解謎難度與最初的兩部作品相比有所降低，機關也相對簡單。

在一段時間內沒有解開機關，還可以使用遊戲自帶的提示功能，但是提示並非直接公布謎題答案，而是給出一個引導方向，這樣的設置非常人性化，既不破壞遊戲本身的探索樂趣，同時又兼顧了新手玩家的遊戲體驗。

但對於老玩家來說，迷室：往逝還是存在著一點吹毛求疵的問題，那就是創新。儘管在玩法上已經相當完美，畫面從第一作到如今的迷室：往逝也都無可挑剔，但總讓人感覺缺少了一點突破性的創新，而且越玩到後面，這種感覺就越強烈。

克蘇魯、科幻元素下的愛情故事

本作的大多數故事背景都記錄在筆記本上面，其實這也是大多數解謎遊戲常用的方法。雖然本作使用了這種手法，但整體劇情其實並不是特別複雜，反而非常通俗易懂。

本作的主角是一位前來調查工程師「愛德華」與夫人「阿比蓋爾」失蹤案件的調查員，而主要調查地點是一間古怪的「玩具屋」。

隨著遊戲進程的不斷推進，調查員發現這位工程師並非簡單的科學研究者，而劇情也牽扯到了「圓環會」和「虛空」的元素，跟前作似乎多少有一點聯繫。

但劇情的核心依然是圍繞著愛德華與阿比蓋爾展開，除去科幻、克蘇魯的元素，迷室：往逝最終突顯的還是他們之間的愛情，至於具體的故事，遊戲里的筆記本其實已經記錄的非常完善，本文就不做劇透了。

攻略

小小心得，請笑納

由於迷室：往逝本身自帶「攻略性質」的提示，因此通關的難度其實並不大，不過，以下的一些小建議還是可以幫助大家提高遊戲體驗：

1. 這一作的自由度更高，意味著需要多在物品之間進行互動，發揮想像力，不要一直糾結在同一件物品上

2. 視覺的轉動非常重要，同一件物品可能藏著多個關鍵元素，換一個視角可能會有意料之外的發現

3. 拼圖類解謎可以多觀察周圍的物品，兩者之間可能存在某種聯繫也不一定

4. 左上角的「提示標誌」大概會在卡關1-2分鐘左右出現，追求挑戰性的玩家可以選擇忽略，但需要注意的是：看完提示也還是需要發揮想像力才能解開機關

有一處熱愛遊戲的優質玩家聚集地，不定時會發放福利和測評各種好玩的手游~

每周至少2次活動，群主心情好還會發紅包，玩遊戲還能有豐厚獎勵~
