拳頭設計師也會提出沙雕改動方案！S10曾設計小龍可出坑找人單挑

2019-12-25 1,154條評論 3,341次閱讀 962人點贊

英雄聯盟更新到S10版本後，整個召喚師峽谷大改，加入諸多新的元素，如地形變化、元素龍屬性調整、遠古龍斬殺效果、峽谷先鋒可多次重生等，在豐富玩法的同時，給玩家們創造極多不確定的翻盤因素。

不過，S10的最終版本，不是設計師一次性改動成功的，是設計師從無數的提案中，篩選出符合版本改進，不破壞遊戲平衡的合理提案。

近日，拳頭在外網公布設計S10版本的廢棄方案，引起外網熱議，這些廢棄方案中，有很多都是沙雕方案，與網友們平時調侃吐槽的無理改動要求一般模樣。

具體廢棄方案如下：

1.一血草叢：被拿下一血的人，死亡處會多一片草叢

2.通過推塔玩家可以摧毀牆壁

3.水晶被擊破之後，基地會被草叢遮掩

4.創造幼年小龍，給玩家提供一些小的BUFF

5.拿下大龍BUFF後，玩家擊敗野怪會讓野怪加入到推塔大軍中

6.如果長時間沒有人攻擊小龍/大龍，小龍/大龍會走出龍坑找人「單挑」

有趣的是，設計師把這些方案都放到遊戲中一一實驗，拿小龍和大龍走出龍坑的方案來說，只要召喚師長時間不去擊殺小龍和大龍，大龍和小龍就會走出龍坑找人單挑，凡是在龍坑附近的玩家，都會受到波及，大龍還好說，畢竟出生晚，小龍找人單挑是去到下路，或者被人引到中路，這對於本就不好過的AD玩家來說，無異於天災人禍，設計師多次試驗後，才放棄此方案。

南風倒是覺得，一血草叢、水晶被推基地長草的方案可行，一血爆發地出現草叢，能讓線上發生無數可能，基地長草也能增加玩家的翻盤幾率，只不過也會加快對方的推進節奏，可惜這些方案都被設計師拋棄，興許下次峽谷大改會實施，你們對召喚師峽谷有什麼提議么？
