《反恐精英Online》即將迎來11周年 新模式生化獵場獨家曝光

2019-12-20 43條評論 1,275次閱讀 142人點贊

12月23日，經典FPS網遊《反恐精英Online》將迎來11周年最新版本。全新模式「生化獵場」即將顫慄來襲！黎明前最後的黑暗，驚悚殘酷的生死追擊，手無寸鐵的人類如何才能擺脫殭屍的殘暴追殺？讓我們趕快來了解一下這個最新玩法吧！

新模式生化獵場介紹

該模式下每局遊戲最多允許5名玩家參加。回合開始之後，將會從所有玩家中隨機選中一名玩家成為殭屍，其他玩家為人類。人類玩家需要合作修復5台散落在地圖中的發電機，最後藉助出口逃出生天。而殭屍玩家則需要阻止人類開啟發電機，將這些人類玩家永遠留在這個暗無天日的閉塞莊園內。

在該模式下，人類是無法將殭屍徹底擊敗的，但是藉助地圖中散落的道具，可以對殭屍玩家進行牽制，讓其他人類同伴能夠更加快速的修理髮電機，從而獲得逃生的機會。地圖散落道具分為3類，武器道具可以幫助人類通過不斷攻擊來擊退或定身殭屍，讓自己獲得喘息和逃脫的機會，為人類隊友爭取時間。救助類道具既可以幫助人類進行療傷，也可以用來加快救治隊友的速度。地圖中還會隨機掉落工具箱，使用工具箱修理髮電機將會大幅提高修理速度哦！

而對於殭屍玩家來說，必須要不斷破壞人類修理髮電機的企圖。殭屍玩家可以對人類玩家和被修理過的發電機進行攻擊。擊中被修理的發電機將會大大降低修理進度。而擊中人類玩家2次後，人類玩家將進入瀕死狀態，如果一定時間內沒有得到隊友的救治，就將會被淘汰出局。如果被救後再次進入瀕死狀態，也會被直接淘汰出局哦！

究竟是人類團結起來力量更大，還是夜幕下孤獨抓捕逃竄人類的殭屍更加恐怖，讓我們一起拭目以待吧！

看了新模式「生化獵場」的介紹，是不是感覺已經躍躍欲試了呢？12月23日，《反恐精英Online》全新版本將正式降臨，不僅有最新的模式玩法，還有新角色新武器及大量的福利活動在等著大家哦！讓我們一起盡情期待吧！
