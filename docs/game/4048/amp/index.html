摘要

在這次全明星上，出現了一種嶄新的遊戲模式——雙人共玩，由兩名玩家操縱一名英雄，一個控制鍵盤，一個控制滑鼠，進行5v5的對抗。

在這次全明星上，出現了一種嶄新的遊戲模式——雙人共玩，由兩名玩家操縱一名英雄，一個控制鍵盤，一個控制滑鼠，進行5v5的對抗。作為一個娛樂模式，雙人共玩的節目效果極好，因為兩人不能很好配合，遊戲中經常會出現反向Q等滑稽場面，看到平時大神一般的職業選手也會做出黑鐵操作，任誰都會忍俊不禁。當然也不乏ZZR和Levi的盲僧這樣的默契配合，在這種模式下更是顯得難能可貴。

實際上，在LOL的歷史上，除了召喚師峽谷5v5等常駐模式外，也有很多限時的娛樂模式，他們有的過段時間就會復刻一次，而有的只開放過一次，就讓小編來盤點一下吧。 一、無限火力 無限火力是LOL定期復刻的一種特殊遊戲模式，在這個模式中，英雄沒有藍量和能量的限制，出門自帶80%減CD，玩家可以一直扔技能扔個爽，高額的減CD還能觸發一些平時看不到的場面，比如船長的7桶連炸。雖然無限火力已經復刻過很多次，但每次都針對強勢英雄進行了調整，讓這個模式每次復刻都能開發出新套路，比如最近一次的跑酷燼和貼膜瞎，快樂的不敢想像。

作為LOL最成功的特殊模式，無限火力已經成了部分半脫坑玩家的挂念，他們平時不打排位匹配大亂斗。但無限火力出了他們總會找上幾個朋友，開遊戲玩上兩把。每年總會有人催拳頭復刻無限火力，甚至不少人都希望變成常駐模式，可見其受歡迎的程度。 二、末日人機

末日人機是PVE對抗的模式，玩家組隊對抗被大幅強化的人機，沒玩過的觀眾可千萬別盲目自信哦，末日人機不僅AI和屬性得到了強化，技能也變成了魔改版本，比如亞索可以吹出n道風，團戰一打起來人機的技能就像彈幕遊戲似的，完全沒有躲避的空間。遊戲任務是殺掉最終boss——高達幾萬血的巨大化提莫。小編表示當年這個模式開放時，始終沒能打通最高難度。後來才得知，主播和職業選手的五黑隊都經常在最高難度翻車，末日人機算的上是最難的特殊模式了。 三、克隆模式

克隆模式是由一方投票選出一個英雄，5個人全部使用那個英雄進行的對抗模式。因為正常模式同隊不同人不能選擇同一個英雄，所以克隆模式和無限火力一樣，能打出很多平時見不到的連招，甚至比無限火力更有節目效果。比如石頭人和亞索的五連大，對面在天上根本下不來。克隆模式是最適合組隊開黑的特殊模式，因為如果不和隊友配合，體驗和普通的5v5也沒什麼區別。但一旦配合默契，歡樂程度更高於無限火力。 四、血月殺

血月殺模式可能是所有特殊模式里最特殊的那個，在這個模式里，只能選擇規定的21名刺客英雄，通過收集遊魂強化自己，打出秒殺級傷害。這個模式概括一下就是秒與被秒，遊魂地圖隨機刷新，讓比賽充滿了博弈，無論是隱身狀態完成刺殺，還是野區埋伏爭奪遊魂，都有一種難以言喻的刺激感。可惜的是，血月殺模式從出現以來就只上線過一次，至今沒有復刻。 還有什麼模式是你喜歡的呢？歡迎在評論區留言分享。
