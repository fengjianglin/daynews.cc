懟人懟恐龍？《方舟生存進化》硬漢玩家就愛“粗暴玩法”

2019-12-15 96條評論 3,311次閱讀 753人點贊

《方舟：生存進化》遊戲是個世界同服的Steam神作，在方舟世界，玩家可以滿足各種心理。最近小編關注的主播就不愛高科技玩法和策略套路，他就喜歡「粗暴玩法」。這種粗暴玩法可以盡顯主播的拳頭和實力，主播玩得不亦樂乎。

《方舟生存進化》遊戲有各種騷氣元素：比如猛獸，恐龍，野人，老外，戰鬥，馴獸等等，於是主播就可以大展拳腳。這次，主播看上了方舟遊戲的鐮刀龍，他想要馴服一波。《方舟生存進化》遊戲的鐮刀龍，它的身材是主播的8倍大，但是主播也是絲毫不發怵，拿著弓箭和流星錘就上線剛。相比鐮刀龍的巨大身形，主播就顯得走位靈活，他跳到鐮刀龍屁股後邊用力懟，這種感覺也是蠻刺激。

除了在方舟世界懟龍，主播還特別喜歡懟人玩法。《方舟生存進化》是個世界同服的遊戲，主播可以藉機挑戰各路玩家。這次主播就和敵人相約來了一場「男人之間的對決」：憑著各自的肌肉拳頭和原始武器，雙方面對面的對打，這種感覺也是蠻酸爽，拳拳到肉啊。

各位玩家大佬，大家在《方舟生存進化》遊戲喜歡哪種玩法呢？歡迎大家在評論區積極留言，小編期待你的撩騷。《方舟生存進化》是個遊戲元素極其豐富的沙盒神作，它總能滿足各種玩家的需求的，大家走過路過不要錯過哈。
