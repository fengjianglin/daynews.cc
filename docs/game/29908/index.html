Dota2本賽季第四個Major公布！震中杯莫斯科Major5月開戰

2019-12-23 2,805條評論 4,529次閱讀 413人點贊

今天凌晨，EPICENTER官推公布震中杯莫斯科Major將為本賽季第四個Major，主場館比賽日為5月8日至10日，地點為莫斯科CSKA體育館。

至此，本賽季剩下的四個Major已經全部出爐，分別是：

第二個Major：2020年1月18日-26日的夢幻聯賽S13萊比錫Major

第三個Major：2020年3月15日-22日的ESL One 2020洛杉磯Major

第四個Major：2020年5月8日-10日震中杯莫斯科Major

第五個Major：2020年6月20日-28日的ONE Esports新加坡Major

每個Major賽事的獎金均為100萬美元，DPC積分為15000點。

根據TI9的經驗來看，目前穩被直邀的戰隊有TNC、VG和IG三支隊伍。備受矚目的雙冠王OG連鴿兩個Major，TI9季軍LGD在第二個Major預選中淹死海選，目測後面幾個Major的爭奪會越來越激烈。

值得一提的是，震中杯Major的宣傳片做得非常精緻，頗有一種大片既視感。

希望在TI10的舞台上，西恩刀塔能帶給我們驚喜。
