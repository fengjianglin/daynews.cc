與電影一同腰斬 未公布Wii版《蜘蛛俠4》遊戲原型曝光

摘要

油管頻道Hard4Games在上個月末發了一則新挖掘視頻，首次向玩家們展示了一款從未公布過的《蜘蛛俠4》遊戲。該遊戲原型在Wii開發工具包中被發現，原定作為電影《蜘蛛俠4》（也沒拍成）的捆綁周邊產品推出。

雖然Sam Raimi導演，Tobey Maguire出演的《蜘蛛俠》系列在商業上取得了成功，但是第三部遭到了部分輿論和批評家的批評，因此原本索尼是打算拍攝《蜘蛛俠4》，且電影和遊戲的原型都已經存在。可惜最後關頭索尼還是決定直接重啟，推出了《超凡蜘蛛俠》。（後者如今也被放棄，現任蜘蛛俠荷蘭弟也算是再重啟。）

所以該《蜘蛛俠4》Wii版原型遊戲的開發完成度也一般，僅僅包含了兩個不完整的關卡，但是其基本的設計和機制卻在原型中完整的呈現出來。高樓躍下時的「跳水」動作，蜘蛛俠核心蕩鞦韆的連接（包括拐角）、對敵的打鬥連招都比較成熟；而整個城市地點的的搭建也不僅僅是個毛坯。

有趣的是，通過模擬器運行該ROM時，該遊戲被識別為《死亡空間：撤離（Extraction）》，顯然用的是同款引擎。
