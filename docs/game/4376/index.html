感受科技的力量，魔獸世界懷舊服奧山之終極奧義

摘要

12月11日懷舊服將開放奧山、戰歌峽谷2大經典PVP戰場。說起wow的pvp話題永遠離不開工程學。此物雖不穩定，經常惹出搞笑Top10之類的經典集錦，但是深玩pvp的玩家都毫無爭議的認同工程學才是凌駕於九大職業之上的pvp第一職業這種說法。

12月11日懷舊服將開放奧山、戰歌峽谷2大經典PVP戰場。說起wow的pvp話題永遠離不開工程學。此物雖不穩定，經常惹出搞笑Top10之類的經典集錦，但是深玩pvp的玩家都毫無爭議的認同工程學才是凌駕於九大職業之上的pvp第一職業這種說法。

以雷矛要塞進攻戰為例，部落方很難在均勢的對抗中討得便宜。然而，在雷矛墓地背後的高山頂上，如果你能集結起一批帶著工程學披風的突擊隊，那雷矛大橋的天塹地位立馬蕩然無存。從這個位置能直接飄進聯盟的雙塔，然後橋頭阻擊的聯盟主力部隊在懵逼中不知所措，一部分反應過來的人會回去攻擊已經被部落站穩腳跟的塔樓，從下攻上十分吃虧。橋面部隊在分兵和混亂中則很有可能被斜坡集結的部落主力一波衝散。

有很多新區的朋友會說，我沖級都累的要死，哪兒有空搞高等級的工程學啊？辣么重點來了，有個寶貝東西叫諾格弗格藥劑

在熱沙港的廣場中央，我們能夠輕易找到斯賓科爾.諾格非沸格NPC，這個NPC就是出售亡靈骷髏變身藥劑，我們直接與其交談就可以直接購買藥劑了，在包包中點擊使用就可以讓我們變身了，若是再騎個亡靈戰馬，效果更佳。

不過侏儒做的東西都有個特點，不穩定。這藥劑除了能讓你變輕，還有可能讓你變小或者變成亡靈。如果不幸觸發了其他兩種效果，輕滑鼠右鍵點掉buff重新吃。

工程學另外有個神奇的寶貝叫洗腦頭盔，應用場景是boss房間攻防戰。無論是攻方還是守方，在boss房間都會有集結期，在此期間攻方因為還沒集結完畢會小心翼翼不去開到boss。守方往往是應情報趕來支援的小股部隊，在boss仇恨列表沒有目標的情況下也不敢貿然以少打多。這個時候如果有個逗逼掏出工程頭盔的話，效果100分。想像以下以下場景，攻方小心翼翼的貼著牆壁生怕惹出什麼亂子，突然站在身邊的隊友變紅了開這30分鐘的大招追著大家打（請盡量控制戰士，因為他可能會被隨機開魯莽喲），直播效果絕對滿分。

還有諸如縮小槍、火鍵鞋之類的道具，也能極大的改善短腿職業、少控職業的短板。總之，筆者在此安利各位，如果喜歡pvp，請不要錯過工程學。並且官方也在前幾天剛開一組PVP伺服器，想在戰場一試身手的玩家可不能錯過。
