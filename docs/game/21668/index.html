我想一直贏，拿到每一個冠軍！eStarPro.Cat冬冠備賽期專訪

2019-12-18 2,438條評論 3,746次閱讀 828人點贊

昨天貓神還在成都天美工作室為自己FMVP的冠軍皮膚提供修改意見，今天已經在廣州番禺區的一個片場進行冬冠賽宣傳片的拍攝，全部錄製工作結束已經是晚上七點，沒來得及吃飯，就坐車趕往機場準備飛回上海，兩天三個城市一般人聽到還是會覺得有些誇張，但這就是一個職業選手的工作常態。

在貓神簡單的吃了些東西後，我們在趕往機場的車上開始了今天的專訪，這是我第一次近距離接觸貓神，陽光，愛笑是他給我的第一印象，對周圍的人都很有禮貌，讓你很難將眼前的這個靦腆的大男孩和KPL七冠王,總決賽MVP這些響亮的稱號聯想在一起。

「從未想過打職業會這麼累」

在成為職業選手之前貓神也和普通人過著同樣的生活，在課餘時間和朋友們打一些線上比賽，周末了一起坐綠皮火車去參加城市線下賽，一次偶然的機會在KGC拿了些成績，那次之後正好趕上各大俱樂部組建王者榮耀分部，機緣巧合就被eStar選中走上了職業道路。

「第一時間家裡是反對的，主要是怕我被騙，當時我在濟南，直接跑回家和我媽聊了一下，最後給了我幾百塊錢路費就來到上海打職業了。」貓神說姥姥卻對他還是很支持的，老人家想法也比較簡單，只要孩子健健康康的比什麼都強，只要不是違法的事情就好。而貓神進入職業圈的想法也很單純，每天打遊戲，還會有很多人支持和喜歡，所以就試一試。

貓神走上職業道路的經歷聽起來都是水到渠成，但也有句老話說的好，機會是留給有準備的人的，從萬千路人中脫穎而出被職業戰隊青睞這和選手自身的努力是分不開的。

這也讓筆者聯想起了十幾年前和小夥伴們去參加CF線下賽的經歷，雖然每次都是一游兩輪游，但是大家樂此不疲，那時候家裡人也覺得我是沒個正經，但有一次比賽得了第三，獎金1000元，我們每人稅後分到160塊，從那時起家裡人也對我的熱愛有了一絲絲的改觀。

選手們正常的一天是從中午開始的，起床後吃個午飯，下午訓練到五點，中間休息一兩小時，晚上繼續訓練到十二點。

但貓神說日常的訓練其實算最輕鬆的一環，只是加上周三到周末的常規賽，還有日復一日的拍攝、採訪、綜藝、直播等等各種工作的壓力，聯賽接著杯賽、杯賽接著聯賽，休息的時間少，感覺身體很疲憊，也根本沒有時間去發展其他興趣。

「那你平常比賽如果覺得太累了有什麼調節的方法嗎？」我問到。

「其實我沒有什麼特別的調節方法，可能就是養生吧。「說完這句話貓神靦腆的笑了。

「我2016年2017年的時候每天會訓練到兩點鐘，現在打到十一二點就感覺有點夢遊，18年剛轉指揮的時候很容易發脾氣，就是指揮的時候如果隊友做的不好我就容易急躁，前幾天去看中醫，醫生跟我說是肝火比較大，他還說女孩子每個月是看得見的出血，職業選手用腦很多嘛，我們這個是看不見的出血，氣血不足。他讓我喝西洋參，現在我就比較注意，沒事會和領隊打羽毛球，每天喝一些養生茶，再喝喝中藥調理一下身體。」

「在被JC橫掃之前，我們飄了」

eStar今年可以說是KPL聯盟中最成功的戰隊了，前10個月都順風順水，但是常規賽最後一場對陣DYG.JC的比賽被3:0橫掃，讓所有人都大跌眼鏡，春季賽和世冠賽的大魔王就此跌下了神壇。

「其實打JC那場有想過可能會輸，但是沒想過輸的那麼慘，我覺得我們當時的心已經不在那場比賽上了，對我們的排名沒有任何影響了，當時常規賽沒開始之前我們定的目標是進勝者組，但沒想到我們穩拿東部第一了，大家都沒有什麼心理壓力了，又剛拿了兩個冠軍，都有點飄了，所有隊員都是，那天主要還是第一盤沒有憋住，差一點翻盤，後面就很自然的輸掉了，那場比賽，太飄了。」說完貓神搖了搖頭。

「在上個版本我們的掌握度還是OK的，但前期我們經常送一血，打得不夠好，後面可以打回來，也一直都在贏，這個問題就被忽略掉了，剛開始定的目標是我們穩住前期，不送一血，但現在發現我們穩住前期了，被對面拿到優勢，C位經濟起的很快，拖後期會打的很累。他們很多隊伍是兩分鐘前基礎的東西做的比較好，我們是中後期做的比較好，像JC前兩分鐘就很棒。」

「還是王者榮耀最好玩」

現在成為電競職業選手是很多年輕人的理想和願望，在成為KPL職業選手的最低標準上，我和貓神也展開了探討。貓神說：「首先我認為在一個位置上，巔峰賽上穩定100名是基礎的標準，很多人都說說多少多少星，沒有用，真正的高手都在巔峰賽排名靠前的位置上，但畢竟排位賽和職業比賽是兩個遊戲，也有很多路人王在進入KPL後發揮的並不盡如人意，訓練賽和巔峰賽時是大腿，但真正比賽時候卻放不開，所以有一個大心臟很重要。」

KPL職業選手的人員流動和淘汰頻率是我在其他項目上從未見過的，去年還風生水起，今年就已經上不了場，甚至消失在了隊員名單中，再加上現在每年都會有選秀為聯盟不斷的輸送年輕的選手，電子競技拿成績說話，不努力很快就沒了。

能把興趣當成工作是應該是很多人都非常嚮往的，但面對日復一日的訓練，接連不斷的聯賽、杯賽，這對於每個職業選手來說也都是一個考驗，不能很好的自我調節很有可能就會心態失衡，誰都會有失去熱情的時候。

「就這次嘛，我們輸了，四強就被淘汰了，然後我們放假了，就想不碰這個遊戲了，但隔了一天之後，又回來開始繼續每天玩，覺得還是王者榮耀最好玩。」

「我拿個亞軍你給我一百萬我也不會高興」

在聯盟中貓神是第一批職業選手，和他同期的選手還在役的已經寥寥無幾，在我問到他為什麼能一直堅持著的時候，只得到了幾個字：

「因為喜歡啊！」

他覺得他自己的職業生涯一直算比較順利，又是自己喜歡做的事情，遇到挫折又能爬起來，就自然而然的一直走了下去。「其實沒有太多的想法，我的目標就是一直拿冠軍，除了冠軍其他東西都是附帶的，有冠軍就行沒獎金也行，我拿個亞軍你給我一百萬我也不會高興。」

這就是熱愛吧，當事人往往不會發覺，一個英雄玩不好的時候不停地練習操作，哪種出裝效果最好不停地實踐，甚至像他所說的隊友有一點點沒做好就想發脾氣，其實無處不體現著貓神對於這款遊戲的熱愛，和超強的好勝心。

貓神的口頭禪：我人傻了

在和cat聊天的過程中，他還跟我們分享了幾個小故事，讓我覺得他還是一個可愛的大男孩，也不過是會被「連輸」和「缺氧」打敗的普通人而已。

「今年在備戰春季賽決賽之前，我們BP打後手是非常厲害的，勝率很高，但是先手不行，就有一天訓練賽我們先手BP勝率是零，然後我們就一直練先手，和幾隻比較熟的隊伍就一直練第一局，想把這個問題給解決。」「那最後有解決嗎？」「並沒有，輸了一天，都輸傻了，然後就打開始練第二盤了，再輸就崩潰了，總決賽第一局我們先手BP也是輸給了RNG.M嘛！」

「還有上次去雪山上拍片子爬了三天，都倒了，大家都在那裡一瓶氧一瓶氧的吸。」

「那聽說你們打完比賽還有吸氧的習慣？」

「哪有！那是之前問過狂人，他不是運動比較多嘛，他說運動之後頭腦不清醒吸口氧很舒服。我覺得這個是同理嘛，所以我就買了罐氧氣。」「那你吸了嘛？」「吸了呀，就打AG的時候打完後台吸的，就沒什麼用，還有人說打完生病去吸氧，我人都傻了。」

「今年的願望就是過個好年」

本賽季止步四強是貓神今年最大的遺憾，這也是eStar小夥子們為自己的年少輕狂付出的代價，這是件壞事，也是一件好事，壞在破壞了所有盼著小E星大滿貫的粉絲們心中的完美劇本，好在讓這幾個小老弟能及時認清自己，重燃鬥志。快到終點的時候更要玩命衝刺，提前慶祝必定會被人超越。

到機場時我問貓神今年還有什麼願望嗎？貓神笑著說：「打好眼前的比賽，過個好年。」這次回到上海就要開始備戰冬冠賽了，沒能進入秋季賽前三的eStar只能參加選拔賽，在等待他們的是XQ，Hero，RW俠，TS，個個都是難纏的對手，希望他們能有一個全新的面貌來迎接這今年的最後一個比賽，為自己的2019畫下一個圓滿的句號。
