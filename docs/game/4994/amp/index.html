LOL9.24版本內容爆料，根據拳頭設計師Mark Yetter的透露，在將到來的9.24版本當中，一些英雄進行了加強或者是虛弱，來看下Mark Yetter爆料的匯總介紹。

1. 熱修復過後，滾雪球能力將會增強；

2. 征服者將適用於戰士和部分法師（暫未改動）；

3. 峽谷先鋒小幅度加強；

4. 一些改變對於悠米和布隆，這可以幫助他們適用於普通玩家（不僅僅只適合職業選手）；

5. 之後的版本里塞拉斯的機制將會有改動；

被削弱的英雄：賽娜、瑞茲、卡薩丁、龍王、阿卡麗、蓋倫；

加強的英雄：悠米、布隆、卡爾瑪、龍龜、阿木木、沙皇、飛機、希維爾。

對於此次的9.24版本的英雄改動，主要圍繞在阿卡麗方面，玩家紛紛對阿卡麗的削弱發表自己的看法。

網友：「每砍一次akl，akl玩家越是能打出更極限的操作，然後又砍akl。」

網友：「阿卡麗還削啊？」

網友：「阿卡麗再削弱能玩？」

網友：「阿卡麗到底做錯了什麼？」

英雄聯盟9.24版本更新時間未定，但是對於9.24版本的內容玩家已經開始了討論，那麼對於阿卡麗，你是怎麼看待的呢？
