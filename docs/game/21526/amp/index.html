擬養成與2.5D沙漠幻想RPG的結晶《部落與彎刀》，創意層出不窮

天天要聞
2019-12-19

12月初，一款名為《部落與彎刀》的作品悄無聲息地登陸了STEAM平台，我們在瀏覽STEAM平台資訊的時候發現了這款遊戲，並在看到了遊戲預告片後將其推薦給了各位柄友。初見這款遊戲，獨特的異域色彩以及畫風吸引了我們。武俠的遊戲題材再配上集團作戰的形式。武俠+開放世界+模擬，無形中拔高了地主對這款遊戲的期待。

通過多方溝通，我們最終與開發商漢家松鼠取得了聯繫，並獲得了這款作品的測試資格，親自體驗了一下這款看似很有潛力的作品。最後驚奇地發現，它似乎大大超越了我最初的預期。

標新立異，眼前一亮的遊戲系統

初見預告，我一度天真的以為這是一款刷裝備的遊戲。直到我打開這款遊戲後，發現這個世界，其實並不是我們認知中的模樣。

作為一款2.5D的遊戲作品，《部落與彎刀》也為玩家們提供了非常詳細的捏人選擇，雖然在跑圖時大家並看不到人物的臉，但一些身形以及髮型是可以在2.5D的畫面視角中顯示的，並且在與世界中的所有NPC對話時，對話框中也會顯示你角色造型的大圖。捏人部分主要分為髮型、面部特徵、身體以及飾品，雖然並沒有像一些海外3D遊戲的捏臉系統那麼細緻複雜，但其每個部位為玩家提供了大量預設的選項，讓你在捏人時也同樣可以玩出一些花來。

在創建人物的時候，會讓你選擇你的身份以及一些天賦。這決定了你的技能樹，也就是你的職業。同時也決定了遊戲的難度！（遊戲的戰鬥難度應該是固定的，不同職業在遊戲時的強度不同，從側面改變了難易度）於是，習慣以最高難度開始遊戲的地主也與往常一樣無腦點了最為困難的劍豪開始了第一次體驗，然而卻寸步難行！這裡就要提到遊戲中與我想像及其不同的概念：集團作戰！雖然遊戲中也包含一些單挑的boss戰，但大部分的戰鬥中你都將面對的是抱團的敵人。尤其是高難度下，敵人的兵種搭配組合以及走位讓形單影隻的我苦不堪言，可以說高難度非常具有挑戰，前期玩家需要儘可能地規避戰鬥，通過採集以及完成一些任務來招募更多的人手，組建自己的小部隊。才可能在這個殘酷的世界中生存下來。不同職業的難易度主要區分在於前期出生時身邊的幫手數量，以及地圖中各個陣營與你的敵對關係方面。例如在你選擇劍豪時，可能由於為人過於狂傲，以至於地圖中幾乎全部陣營均與自己敵對，身邊又沒有幫手，可以想像他的境遇。用一句過街老鼠來形容我的第一次劍豪之旅可能不算過分吧！

在遊玩了數小時後，我逐漸對這款遊戲的結構有了比較清晰的認識。遊戲擁有極其豐富的開放世界RPG養成元素，這些元素會影響到你在遊戲世界中的每一個方面，包括與各個勢力的嘴炮、與商人砍價能力、帶領部隊進行集團作戰時的領導力（光環被動）、以及在戰鬥中你最為英雄角色的戰鬥力。角色養成+部隊管理+ARPG與RTS結合的戰鬥體驗+開放世界多線程的敘事&任務體系構成了這款十分龐大的遊戲系統，可玩性十足！

思考&操作，缺一不可

前文中也有提及，遊戲中的戰鬥結合了ARPG的單角色操作，以及指令式的即時戰略特點。英雄單位（也就是主角）可以進行手控走位+主動技能釋放，並能對旗下的部隊發送一些簡單地指令，包括跟隨、自由攻擊、防禦等等。判斷敵我雙方的兵力差距，調整走位以及指令，並在合適的時機釋放技能，是打贏一場戰鬥的主要因素。

並且，由於你在操控主角進行戰鬥的同時還需要給予部隊指令，所以遊戲在戰鬥中的操作量並不像一般的策略遊戲那麼少，緊張感十足的戰鬥外加集團作戰的特色，讓《部落與彎刀》的戰鬥體驗非常獨特。

此外，由於遊戲的設定為沙漠幻想，帶有著不少魔幻的色彩。所以類似於法術、召喚魔物等等元素在遊戲的方方面面也均有體現。

極具深度的系統，值得深入研究

如果你自己上手遊戲，相信很快會被遊戲龐大的內容所震撼到，天賦樹、技能樹、陣營關係、隊伍列表等等一系列複雜的系統雖然都不算太過複雜，但想要將他們完全理解並試驗出一套適合自己玩法的套路來，也得花費數小時的時間。

並且對於不同的職業可以選擇的技能線路也完全不同。超大的地圖配上數量繁多的城市、陣營。你在遊戲中可以選擇專精的領域也會全然不同，究竟想做一個用武力戰勝一切的莽夫，還是利用合縱連橫等犀利的外交策略站穩腳跟，亦或是全靠一張嘴吃遍天下英雄為我所用？這都會成為你的選擇。

仍需進一步雕琢的璞玉

《部落與彎刀》雖然目前僅僅處於測試版本，但其本身的遊戲素質已經可以讓人安心。俗話說，三歲看到老！在整體框架出色，遊戲內容紮實的現況下，後期只要保證遊戲的更新以及故事線的完整性、合理性，那麼本作毫無疑問是近年來極為出類拔萃的優秀作品之一。

當然，作為測試版本，我們也發現了目前遊戲中存在的一些問題。上手難度不小，加上略微有些高的前期敵人強敵都會讓新手玩家在初接觸這款遊戲的時候展現出一些不適應。此外，任務的引導也做的不夠完美，一些諸如「去某些地方尋找逃跑的士兵」等任務，會因為引導的不足，導致我明明已經遇到了需要找的NPC，但卻會產生他們不是我要找的人的錯覺！最後，由於遊戲目前還處於開發階段，明年初才會開啟EA測試，所以UI設計等細節方面似乎還沒有花太大的精力來進行優化，廉價感略強。其實這些都是些不疼不癢的小問題，對遊戲本身的素質影響不大，但玩家的眼緣、上手感受卻很容易被這些因素所左右，故而容易讓人產生本不應該出現的情緒，得不償失。

國產遊戲中的又一款希望之星，下一個銷售奇蹟的潛力股

就是這麼一款，上手有些難、畫風奇特、UI設計又不是很高大上的作品，卻是我這些年來玩到過系統最完整，最具深度，並且整體感覺最棒的國產遊戲。個人認為，《部落與彎刀》會是國產遊戲中又一部長臉的作品，並且潛力巨大，可玩性與趣味性都十分令人滿意！只要開發團隊繼續打磨並且保持更新的話，很有可能創造又一次的銷量奇蹟！畢竟試問誰能拒絕如此出色的中國風沙漠幻想遊戲呢？

《部落與彎刀》目前已結束了內部測試，並獲得了大量玩家們的喜愛。為了回饋玩家們的熱情，製作團隊漢家松鼠也是非常合時宜地將遊戲反向跳票，原本計劃在1月16日登陸STEAM平台的搶先體驗版本正式提前到了1月3日！遊戲在公開售賣後會發生怎樣的變化，以及玩家們會發現各種怎樣的神奇套路，可能是筆者目前最為期待的事情了！
