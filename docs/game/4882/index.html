在魔獸世界中，黑石山茉艾拉的計劃非常完美，她只是通過謠言就擊敗了這裡的最大的BOSS之一，曾經統治著黑鐵矮人的拉格納羅斯就這麼失敗了，她繼承亡夫的遺志，讓黑鐵矮人獲得了自由，當然統一矮人的計劃，茉艾拉也沒沒有放棄，拉格納羅斯和他的僕從們都被放逐回了元素位面，在勇士們看來黑石山的危機已經被徹底解決了，他們開始瓜分戰利品。

但是茉艾拉知道事情並沒有那麼的簡單，黑鐵矮人也確實獲得了自由，但是他們都知道在黑石山還有一個更可怕的敵人，黑石山其實有兩股勢力，第一就是被勇士們消滅的拉格納羅斯，他控制著黑石山的一部分，而另一部分卻被奈法利安的黑龍軍團所掌握，當然他們都聽從上古之神的意志，但是上古之神之間也並不是那麼的和諧，甚至可以說每個上古之神的存在都是相對立的。

其實在黑石山中火元素和黑龍軍團一直在進行著戰鬥，但是最近一段時間他們卻出奇的和平，這樣的停戰狀態是因為黑龍軍團的奈法利安，他不想讓別人知道他的存在，所以就依靠這些元素和黑鐵矮人來作為屏障，其目的就是為了掩飾自己的存在。

在獸人第二次大戰結束之後，雷德帶著舊部落的一部分成員，一直拒絕新部落的命令，當然薩爾成為了部落的新領袖，他更加的不認可，但是雷德所帶領的舊部落實力一般，根本無法和薩爾的新部落為敵，所以他一直定居在黑石塔這座要塞當中，這些舊部落的成員在雷德的帶領下是忠誠於奈法利安的，因為這位黑龍王子給了他們一個承諾，那就是只要機會合適，奈法利安就會幫助雷德他們對付薩爾的新部落。

也許很多小夥伴都明白奈法利安的承諾，不過是為了安慰獸人的謊言而已，作為黑龍軍團的現任首領，他根本不想捲入艾澤拉斯凡人之間的戰爭，對此也毫無興趣，奈法利安最關心的還是自己現在的計劃，作為死亡之翼的子嗣，他一直在繼續著父親的工作，雖然現在死亡之翼失蹤了，但是讓黑龍軍團成為這個世界上最強的部隊才是他想做的事情，所以在黑石山他一直研究如何讓黑龍軍團如何變得更強而已。

各位小夥伴們，你們知道魔獸世界黑龍軍團奈法利安的歷史不？
