弓馬3MM套路將成主流？暗黑破壞神2弓馬搭配冰凍槍兵或能橫掃BOSS

2019-12-13 393條評論 2,716次閱讀 938人點贊

此前看其他作者的暗黑破壞神2文章，曾探討過弓馬的戰鬥問題。在暗黑中不管你承認與否，部分職業之間的戰鬥力差別的確略大。比如親兒子聖騎士要橫掃地獄難度，也許你只是需要一個CB頭外加一把還能看得上的武器便能通關。這裡不將最後的超級BOSS計算在內，分享一下此前的已經進入地獄模式的聖騎士。

其他不過多解釋，因為裝備略顯尷尬，除了CB頭跟吸血鬼的凝視之外，個人覺得這個聖騎士並沒有過多值得可炫耀之處。但只是憑這一身裝備，聖騎士已經足以橫掃地獄。個人來到了地獄難度下的第四關，因為地獄難度下的小怪作戰能力實在是太強，所以我們最後選擇放棄。

反過來，如果你想使用其他職業達到同樣的高度，這樣一來會略顯尷尬，比如亞馬遜。在玩家們的認知中，這個職業有三種不同的進攻流派，弓馬、標馬和槍馬，但是實際上走槍馬路線需要大量的裝備支持，因為長槍攻擊意味著你不會裝備盾牌，防禦屬性會少上一大截，所以一般情況下個人更建議走弓馬路線，畢竟個人不算一個操作非常精湛的選手。在不少暗黑達人的文章中，都曾出現過3MM打法。即女武神、弓箭手和弓馬三者合體，理論上是女武神往前做肉盾吸收傷害，本體跟弓箭手MM在後方持續不斷輸出。

3MM打法到底有何劣勢呢？因為個人對亞馬遜這個職業沒有太多的了解，目前只是探索階段。所以女武神成長起來之後到底能吸收多少傷害，這個暫時未知。如果女武神吸收傷害能力有限，那麼在打BOSS時就會出現一種非常尷尬的情況，女武神會被瞬間融化，我們的MM也會被瞬間融化。考慮到這種特殊情況，個人選擇用另一種打法即冰凍傭兵、女武神加弓馬。

長槍兵在暗黑2中的出現頻率非常高，因為槍兵非常萬金油跟每個職業都能搭配。作為一個弓馬，我們的身板非常脆，所以我們需要多個前排來幫我們吸收傷害。作為召喚物，女武神的吸收傷害能力有限，所以我們需要一個傭兵來幫我們吸收更多傷害。

永遠不要小看一個傭兵的戰鬥力，在必要時刻面對新人殺手大蟲子，他們依然能幫我們度過難關。更何況傭兵將會有裝備的支持，所以理論上一個防禦型傭兵的抗打能力更強。

其次使用冰凍光環的槍兵有一個好處，則是在打BOSS時傭兵的冰凍光環可以幫我們有效施加BUFF。從噩夢難度開始，部分小怪會免疫冰凍元素傷害，但是無論在哪種難度下BOSS永遠都不會免疫元素傷害。所以依靠傭兵附送給我們的冰凍元素BUFF，女武神跟小弟在前方吸收傷害，我們在後方可以肆無忌憚輸出了。

以上就是個人對弓馬的一些理解，也許3MM並不是絕對完美的玩法，更換下零件將會有意想不到的收穫。不知道各位讀者們對於本文有何看法呢？
