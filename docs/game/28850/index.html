DNF：多號黨的噩夢？100史詩刻印難度極高 全身11要刷1000次深淵

2019-12-21 91條評論 1,465次閱讀 144人點贊

DNF韓服體驗服最近剛剛更新了100級版本，關於100版本許多玩家最關心的就是現有裝備的打造繼承問題了，畢竟我們現在身上穿的這身超界+普雷很多都是從盧克版本的B套海博倫升級而來的，經過兩次增幅活動的洗禮，很多玩家的裝備都達到了增幅10以上，那麼繼承全身的打造需要花怎樣的代價呢？

首先給大家介紹一些關於100級深淵的小常識，100級深淵入場一次需要消耗8點疲勞值和18個新深淵門票，這個新的深淵門票又是怎樣兌換的呢？1狗眼=4個新門票（可交易）=20個新門票（不可交易），20邀請函=1個新門票（不可交易），2星空靈石=1個新門票（不可交易），也就是說100級深淵刷一把差不多要消耗一個史詩靈魂！

100深淵的出貨設定也非常有意思，在你打碎深淵柱子的時候會看到一個光球，如果是粉色的球那麼這把深淵基本上就只能出材料，材料可以用來跨界和刻印，按體驗服的爆率來看粉色的球只有非常小的幾率爆史詩。

而如果你碰到的是金色的球，那麼這把深淵100%會爆出史詩，也就是說100級深淵無需打完npc就能知道這把能不能出貨了。

下面我們再來看看100級史詩刻印需要多少材料。強化增幅10及以下時，需要50個小材料和1個大材料，大材料獲取難度較高，1個大材料差不多需要刷十幾次深淵才能湊齊。全身增幅10全部刻印差不多要刷100多次深淵，對於平民玩家來說尚可接受。

但是對於高打造的玩家來說就有些坑爹了，強化增幅11刻印一件需要11個大材料，比增幅10多了整整10個材料！也就是說刻印一件紅11需要刷100多次深淵，刻印全身12件裝備需要刷1000多次深淵！每天滿疲勞深淵需要兩個月才能繼承完畢。

增幅再往上消耗的材料就更高了，一件紅14需要21個大材料，不過需要注意的是，這個大材料是賬號綁定的，你可以多號一起肝來加快刻印進度。對於那些好幾個增幅11的玩家來說，完成多號的刻印是一項大工程。

最後關於裝備刻印還有幾點需要大家知曉，我們身上穿的超界和普雷可以升級成100級工作服史詩，然後當你爆到畢業史詩之後才能將工作服的打造刻印到100史詩上，而普通的泰波爾斯史詩不能升級成100級工作服史詩，但是可以直接刻印打造。

總的來說，100版本的裝備多得讓人眼花繚亂，大家又回歸到了全民深淵的時代，從目前來看，100史詩暫時沒有史詩碎片這種保底系統，所以刷深淵會非常昂貴，而且對於紅11以上的玩家來說，打造的繼承成本也非常的高！
