《饑荒》關於月島的詳細介紹 包含生物和科技

2019-12-24 1,008條評論 1,920次閱讀 811人點贊

哈嘍，大家好，今天給大家全面解析一下月島。

月島是位於遠洋的幾座海島，玩家只能乘船進入。只要玩家登上月島，啟蒙值將會替代理智值，併產生相應的效果。

當啟蒙值超過25%時，月靈就會出現，超過85%時，它們就會攻擊玩家。月靈的攻擊會增加玩家10點的啟蒙值，並使玩家移動速度減慢幾秒，如果玩家連續被數個月靈擊中，就會陷入睡眠6秒。

玩家可以通過食用減少理智的食物來降低啟蒙值，綠蘑菇就是不錯的選擇。玩家還可以通過走位來躲避月靈的攻擊。

月島上會出現敵對的碎片蜘蛛和它們的巢穴，使用十字鎬開採巢穴會掉落月石、蜘蛛網和月亮碎片。

碎片蜘蛛擁有250點的生命值，它們有兩種攻擊方式，近距離的撕咬會造成25點的傷害。遠距離時會產生地刺攻擊玩家，地刺刺中玩家會造成10點的傷害，幾秒鐘後地刺破碎還會對玩家造成10點的傷害。

在月島上同樣會受到獵犬的攻擊，當獵犬被殺死時會有50%的幾率復活成為恐怖獵犬。如果在獵犬死亡時燒掉它的屍體，可以避免恐怖獵犬的生成。

在冬季，月岩企鷗會登上月島，它們與企鵝類似，會在12天後因飢餓而死。它們會主動攻擊靠近的玩家，擊殺它們會獲得冰和怪物肉。

在月島上可以找到幾個溫泉。

沙羅曼蛇會佔領一個溫泉，然後會在附近入睡。如果其它的沙羅曼蛇靠近這個溫泉，它們會互相威脅來爭奪這個溫泉，幾秒後失敗者會離開，勝利者則擁有這個溫泉。

普通的沙羅曼蛇擁有900的生命值和30的攻擊傷害，如果擊殺它會掉落葉肉。它們會跟隨帶著熱的恆溫石的玩家。

如果將浴鹽投入平靜的溫泉，會導致溫泉沸騰。普通的沙羅曼蛇在沸騰的溫泉旁睡一覺之後，會變成紅色的成熟沙羅曼蛇。

成熟的沙羅曼蛇擁有50點的攻擊傷害，除了普攻外，它還會使用火焰攻擊，這會導致周圍的易燃物著火，擊殺紅色的沙羅曼蛇會掉落一個火龍果。

在滿月期間，所有被投入浴鹽的溫泉會生成月光玻璃。挖掘月光玻璃會掉落月亮碎片，並有幾率掉落紅寶石或藍寶石。被開採後，溫泉將在一段時間之後恢復正常。

在月島上可以找到海葵，當玩家或其它動物踩到它時，它會合起來並造成60的傷害。可以用鏟子挖出它們，移植到其它地方當作陷阱使用。

採摘石果灌木可以得到石果，把石果放在地上可以用十字鎬挖掘，有65%的幾率獲得成熟石果，34%的幾率獲得圓石，1%的幾率獲得發芽石果。

月島上有看似蘿蔔的植物，採摘時會發現它是個活物，就像老鼠一樣，可以使用陷阱捕捉，擊殺它時會獲得葉肉。

砍伐月樹會得到月樹花，月樹花可以用來合成浴鹽。大的月樹還會產生月娥，最大號的月樹有幾率生成2隻月娥。使用捕蟲網能夠捉到月娥，餵給它無毒的蔬菜類食物可以防止它餓死。

把月娥種在地上能夠得到月樹樹苗。由於月樹數量有限，並且只能通過月娥來種植，所以最好挑最大號的月樹砍，這樣月樹才能越來越多。

月娥和花瓣沙拉一起可以在寵物岩穴領取1個小飛娥寵物，小飛娥能夠提供微弱的照明，但在飢餓狀態時它不會發光。

月島有獨特的科技。玩家可以在月島上找到3個誘人的結構，使用十字鎬開採可以得到3個天體祭壇的零件，將它們按照正確的順序插入天體裂隙中就可以獲得天體祭壇。

​通過天體祭壇可以合成月鏡片斧、鏡片刀、月球環形山地皮、浴鹽、月娥雕塑草圖、月亮雕塑草圖。

鏡片斧的砍伐效率是普通斧頭的2.5倍，耐久度是80。鏡片刀擁有68點的攻擊傷害，耐久度是75，在攻擊暗影生物時耐久度是150。
