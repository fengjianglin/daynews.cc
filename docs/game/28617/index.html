德杯BLG終結DMO連勝，上中發揮出色，補強之後有望殺進世界賽

2019-12-22 1,622條評論 3,584次閱讀 1995人點贊

大家好，我是想稱霸海上卻沒有船的Aggro電競王路飛同學。德瑪西亞杯第二日，在BLG對陣DMO的比賽中，BLG更勝一籌，終結了DMO的兩連勝。

在這局比賽中BLG的上單選手發揮相當出色，Kingen武器前期就差點單殺DMO的阿樂青鋼影，雙方拼至殘血後己方打野先到收下了青鋼影人頭。雖然小彭後續趕到戰場擊殺了Kingen，但兵線是回推線，阿樂虧得比較多。

全面武器幾乎都是壓制著青鋼影在打，縱使因為一波上頭越塔給對面青鋼影送去了雙殺，但雙方的刀差一直保持在30刀左右。要知道阿樂可是曾經單殺TheShy的存在，S10的LPL感覺有得看了。

另外BLG從JT引進的FoFo選手在這局比賽也和兮夜打得不分勝負，他的妖姬雖然在線上沒能單殺兮夜的卡薩丁，但很完美的壓制了他的發育，讓他沒有遊走的空間。

照這局比賽來看，這次BLG的補強可以說是相當成功，上單Kingen是KT的原上單，在LCK這個盛產世界級上單的聯賽中也是排得上名號的，來到BLG後發揮也很好。

其實今年BLG就差一點進了世界賽，要不是管澤元奶了一口的話，這次看來他們S10能進世界賽的機會是相當之大。不過目前還是各個隊伍的磨合期，想要知道BLG的真正實力到底如何，還需等到春季賽，甚至夏季賽才知道。

大家覺得今年BLG能否衝進世界賽呢？歡迎踴躍留言參與討論。
