王者榮耀：QGhappy王朝沒能重建，復仇者聯盟，最終復仇成功

2019-12-16 1,339條評論 3,090次閱讀 65人點贊

認真來說，昨天的比賽應該是會看得人很生氣，畢竟這支QGhappy跟我認識地QG有很大差距，我不是很敢相信這是一支隊伍面對總決賽的狀態，我們期待的那個QG最終還是沒能到來，可以說昨天的QG是不及格的。

勝敗雖然是比賽場上常有的事，但勝者有勝者的氣勢，敗者則沒有敗者的水準，對於昨天的比賽不知道大家怎麼看，但是對我而言這場比賽甚至沒有四強的敗者組比賽好看，至少那一場比賽輸的人也是輸的拼盡全力，但在昨天的比賽中我沒有看出一點勢均力敵的狀態。

對於輸掉這麼大的比賽，作為職業選手他們肯定是要比我作為一個外人更難受，但是電子競技結果比過程重要得多，當然我相信作為職業選手他們更加比我懂得輸贏對於他們的意義，但就算是這樣，該說還是得說。

昨天的比賽相信看了的人都能夠明白是有多生氣的，因為在我看來這不是最好的他們，QG作為AG最大的對手，他們面對AG時應該是有心理優勢的才是，畢竟之前他們曾八次戰勝過AG，在勝者組第一輪他們也是以很好的狀態取勝的。

AG昨天的比賽的確是打得非常好，狀態以及手感都是非常熱的，但是反觀QG，他們似乎不知道應該怎麼玩才是對的，整個隊伍迷糊得不行，很多決策總是不合時宜，這是我作為觀眾最大的感受，再加上這支以運營在kpl成名的隊伍在昨天的比賽中被對面運營輸了比賽。

作為將運營帶入kpl的隊伍，他們一點一點將自己摸索出來的東西丟了，現在的他們不僅打得不行，就連運營都不是別的隊伍的對手，這算不算是教會徒弟餓死師傅？不，我認為不是這樣的，他們的失誤更多地來自於他們對對手的不了解以及對自己的不自信。

回首2017年那一年，那時候的他們絕不會這樣打架，在昨天的比賽中觀眾也都能看到，他們不敢打是整個比賽最大的問題，從第一局開始，他們在開局開始就幾乎是逆風的狀態，哪怕是拿到了前期陣容也會打得畏首畏腳，直到自己 陷入死胡同，直到自己沒有強勢期。

他們總是對自己陣容的強勢期沒有好好把握，這真的不是那支能打能拼的QG，只要回顧一下2017年的比賽，你會發現那個時候沒有他們不敢打的局，哪怕是逆風你都會覺得很安心，因為他們總是有機會找到最好的方法解決這個問題。

如果說昨天他們的打法是令人難過的話，那麼昨天他們的陣容也是令人很失望的，雖然知道這些陣容可能他們在訓練賽的時候應該是勝率很高的，但是同樣的中路法師不是應該及時調整嗎？昨天的這陣容他們基本被對面克製得非常狠，在BP上他們就先輸了。

雖然很多時候BP不能決定場上的局勢，但是英雄之間的剋制在BP階段的確是會帶來很多的影響。昨天五局比賽，他們沒有一局比賽的BP是佔優的，並且隊員之間的配合也不夠默契，這不應該是一支磨合了一整個賽季的隊伍應該有的狀態。

這支QGhappy在這個總決賽上的表現的確是太穩了，穩中求勝是好的，但是太穩似乎對於贏下比賽沒有任何的意義，每個陣容都有其強勢期，錯過了就是錯過了。這樣一支QG想要再次重建自己的王朝似乎是不太可能的事情，尤其是在現在大家都非常注重打架的版本。他們一直用這樣的方式是贏不了的，但是他們還是有機會的，只要他們能夠調整好，冬冠應該是他們的下一次機會。

QG王朝似乎結束了，但也似乎活在大家的目光里，他們任重道遠，他們也要負重而行，能不能重建王朝似乎沒有那麼重要，但他們能不能突破自己則是被關注的。恭喜AG超玩會拿到了自己的第一個冠軍。
