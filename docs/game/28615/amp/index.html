韓跑跑撞車KPL五大邊路的兩人，對方馬超玩得可以，但還是夢淚強

天天要聞
2019-12-22

陌陌聊王者，給您帶來精彩的王者碰碰車資訊！

王者碰碰車已經進行了一段時間，但是韓跑跑卻非常的鬱悶。不是因為自身實力不行，而是因為在匹配的過程中，很多時候都是遇到一些強大的KPL選手。

對手強大也不是不可以打，但是很多次都是匹配到了一些不敢打的隊友，這讓韓跑跑非常的鬱悶。終於在一局中，韓跑跑匹配到了不錯的隊友。

​

這一局，韓跑跑的對手可是非常強的。同平台的小浪浪就不用多說了，是一個強勁的對手。不僅如此，還有三個KPL的職業選手，TES的迷神，Hero久競的七殺和QGhappy的Q。看到這幾個人，小夥伴們的認識嗎？

迷神雖然轉會到了TES，但是迷神的邊路是KPL最頂尖的選手之一，七殺也一樣，要知道，在XQ最巔峰的時期，七殺的邊路是人見人怕的存在。

​

或許因為知道對手的強大，因此也激發了韓跑跑和隊友們的鬥志。這一局，大家的溝通是非常多的。開局不到1分鐘，韓跑跑就在隊友的配合下，擊殺了小浪浪，拿到了第一血。

這局比賽，韓跑跑這邊是打得非常的主動，中路的玩家是非常給力的，只要清理完兵線，就會馬上去支援隊友，在中單的帶領下，前期打得非常順。

​

這一局韓跑跑使用的是孫尚香，孫尚香這個英雄是一個能夠輕鬆創造奇蹟的英雄。如果是前期處於劣勢，隊友們可以圍繞著孫尚香來打，只要裝備起來之後，孫尚香能夠帶隊逆轉。

如果是前期有優勢的話，孫尚香能夠讓對手毫無反抗之力。或許是對面也怕讓韓跑跑的孫尚香發育起來，多次兩三個人去找機會想要將其擊殺，甚至還不惜越塔強上。

​

特別是迷神使用的馬超，可以說是孫尚香最大的隱患。如果讓馬超跑了起來，孫尚香是很容易被三四下就擊殺。在被迷神單抓了幾次之後，韓跑跑也學聰明了，不會第一時間暴露自己的位置，先是躲在草叢裡，找到合適的時機再出來進行輸出收割。

​

經過了長達24分鐘的拉鋸戰，在前期大劣勢的情況下，韓跑跑和隊友們講過默契的配合，拿下了這局擁有三個KPL職業選手所組成的隊伍。

不過韓跑跑認為，雖然迷神的馬超打得可以，但是感覺還是夢淚的馬超要強一些。不知道夢淚聽到韓跑跑這句話，會不會非常高興呢？
