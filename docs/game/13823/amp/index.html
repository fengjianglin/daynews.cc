三國殺：他們幾個身為史詩卻不如精品，沮授竟也藏身其中

天天要聞
2019-12-12

在三國殺這個武將也分等級的遊戲中，按照武將的三六九等分為史詩、精品、普通，一般平民玩家史詩武將的數量一般都不會太多，而在絕大多數人的印象中史詩武將就代表的是強力無比，可是並非所有史詩武將都是如此，有一些史詩武將的強度甚至連精品武將都不如，今天就和小編一起看看那些史詩武將中的敗筆吧！

強度甚至不如精品的史詩級武將，沮授竟然榜上有名！

TOP1:楊修

楊修作為一個史詩武將在我看來是其強度和等級是完全不搭邊的，3血2技能的楊修在很多時候的發揮並不能夠和他的史詩成正比。首先是技能雞肋，這個技能的效果和名字其實差不了多少是真的雞肋，首先說一下雞肋的技能機制哈，被聲明的手牌是不計入手牌中的，機制類似於董白的黑牌。這就有了一個問題，20牌有AK的神CC殺了你一刀，你要不要聲明？這個技能在很多時候甚至是負收益的一個技能，史詩這個等級對於楊修來說實在是名不副實。

TOP2：士燮

這個在遊戲中被玩家們親切的稱呼為土鱉的武將可以說是公認的史詩中的恥辱了，作為一個站在食物鏈頂層的武將最起碼要在某一方面及其突出才配的上史詩這個名頭，而土鱉這個武將呢沒有爆發，沒有奶量，除了別人都摸不到他以外沒什麼大用，別說史詩，甚至一些標將他都比不上，真的是應了他那句經典台詞：身處亂世，自保足以。可是隊友都沒了真的能自保嗎？

TOP3：王平

王平這個蜀國史詩的技能也十分簡單，你可以棄一張牌讓手牌比你多或者裝備比你多的人給你一張牌，如果是本輪第一次對他發動可以摸兩張牌，一回合限一次。為什麼說他的強度低下呢,拋開技能的前置條件不談，一牌一收益實在是對不起史詩的稱號，至於額外的兩張牌？在現在的軍爭環境下看看馬良馬鈞甚至是看看李豐，相比於這幾個人的過牌量王平真的是太弱了，史詩之名真是名不符實。(改版後的馬良小編一直覺得是堪比史詩級的強度）

TOP4:沮授

其實按照以前的武將池來說沮授是完全當得起史詩這個稱號的，其高昂的過牌量和回合外優秀的防禦力都是碾壓當時的一眾武將的，可是如果放到現在的版本來看，沮授真的算不上什麼強將，舉一個很簡單的例子：張松

張松的過牌能力是遠在沮授之上的這一點我相信是毋庸置疑的，且張松的回合外對隊友的輔助更是沮授所遠不能媲美的，更不要提開掛一樣的馬良了。另外是矢北的防禦力在當前這個群魔亂舞的版本里真的是負面大於正面，現在能都一輪二刀甚至多次傷害的武將在現如今這個版本真的是多如牛毛，昔日強力的沮授放到現在是絕對擠不進T1這個行列更不要說當選史詩級武將的。

TOP5：沙摩科

這個武將是在上一次版本更新中登錄移動版三國殺的，技能為當你在出牌階段使用或者打出第X張牌時摸X張牌，X是你的攻擊範圍。這個武將就是很經典的一輪將，打一輪爆發，然後就沒了，攻擊範圍大才能帶來摸牌可本身作為一個摸牌白一輪又能打出幾張手牌？這個史詩真不知道怎麼得來的，我覺得給李傕更好哈哈哈!!
