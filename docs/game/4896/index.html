estarpro雖然是四強，但是我們知道這一屆KPL秋季賽至少三強才能夠直接鎖定冬季冠軍杯的名額，四強剛好只差一點點，還需要打預選賽呢。如果estarpro連預選賽都不能出線的話，那麼他們就又得跟2018年冬季冠軍杯一樣無緣了。對於雙冠隊伍來說，這賽季四強已經夠憋屈，夠飽受質疑的了，萬一冬季冠軍杯連正賽都沒進去，這打擊就太大了。所以啊，estarpro選手不放假也不直播，毫無疑問就是在瘋狂備戰冬季冠軍杯了。

想必小夥伴們還聽說了吧？estarpro約訓練賽的對象不是別人，而是AG超玩會。estarpro跟AG超玩會這兩支隊伍上至管理層，下至教練組，乃至選手們，關係都是非常好的。雖然estarpro被AG超玩會淘汰了，但是雙方關係還是非常好的，場上是死敵，但是場下確實很要好的朋友。而這一次有網友甚至表示，看來estarpro這一次勢必要幫助AG超玩會破掉QGhappy的運營啊，幫助他們拿到隊史第一冠？

其實estarpro跟AG超玩會這一次約訓練賽真不是簡單的幫助AG超玩會總決賽特訓而已。因為再好的關係，大家約訓練賽肯定都是互幫互助的。我們知道estarpro季後賽輸了兩場，分別是被JC、AG超玩會3比4打敗的。這說明estarpro一些方面相比於JC或者是AG超玩會是有不足的。而且千萬不要忘記了，estarpro3比4憾負的JC，AG超玩會可是輕輕鬆鬆就零封戰勝的了，AG超玩會肯定有很多值得estarpro學習的點。

我們知道KPL四強中，estarpro算是比較克制QGhappy的，算是比較擅長打QGhappy的。而AG超玩會跟QGhappy就要在總決賽上相遇了，這時候AG超玩會找estarpro一起打訓練賽毫無疑問是最好的選擇。你幫我進行總決賽特訓，我幫你找問題備戰冬季冠軍杯，多好的良性競爭、互幫互助關係啊！

不過坦白來講，現在的QGhappy可不僅僅只是會運營而已啊，如果還用老眼光看待QGhappy那也是不行的。希望AG超玩會能夠好好備戰。也希望AG超玩會跟QGhappy雙方能夠為我們貢獻一場精彩的比賽，而estarpro冬季冠軍杯加油！
