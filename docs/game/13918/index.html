魔獸世界中的滅世者，風暴團隊的堅實壁壘

2019-12-12 528條評論 1,730次閱讀 94人點贊

自版本更新，死亡之翼上線已有好幾天了，相信大家對他已經有了一定程度的了解，龐大的體型、永久無法被阻擋足以彰顯黑紅領主的尊貴地位與強大實力。而終極技能撕裂大地超大範圍的減速與傷害更是坐實了滅世者這一身份。

但是在注重團隊配合的風暴英雄中，強大並不代表特立獨行。龐大的體型使得敵人的技能更容易命中死亡之翼，冗長的技能釋放時間使得死亡之翼空有巨額傷害卻很難給敵人造成致命一擊，無法享受隊友的技能收益又使得坦度高如死亡之翼也很難在敵方持續集火的情況下存活。那麼這位如此異類的黑龍領主究竟要如何融於團隊，給敵方帶來恐懼呢？

官方給予死亡之翼的定位是一個鬥士角色，但是作為一位雙形態的強大英雄僅僅一個鬥士定位顯然不足以將其定位表達完全。死亡之翼的技能範圍大且傷害高，對於兵線的處理可謂得心應手。極高的坦度以及永久無法被阻擋這兩大特點又使得其很難被對手抓死。而在單人作戰的時候又使得無法獲得隊友技能增益這一缺陷規避掉。綜合上述三點，死亡之翼的單人邊路帶線能力可謂得天獨厚。

當然，死亡之翼的強大能力不止於此。坐騎技能巨龍升空的存在使得死亡之翼可以極快的趕到團戰的爆發地點，給予隊友強有力的支援。這一特點在大地圖作戰時體現的將尤為明顯，試想敵方帶線英雄騎馬趕路的時候我方死亡之翼已經從天而降，足以使得團戰天平向我方傾斜。

死亡之翼最頭疼的地方莫過於團戰了。無法享受隊友技能收益會使得死亡之翼在團戰的拉扯期間不斷消耗血量卻很難得到補充，所以對大表哥的走位要求極高，能不吃的傷害盡量不要硬吃。

對此，除了在天賦樹中選擇一些續航類天賦以為，最重要的一點是盡量避免敵方集火從而白白損失大量生命值。在團戰前期不要直衝敵方陣型，依靠巨大的體型為己方C位吸收一些控制技能防止C位暴斃，使用滅世者形態的W、E技能提供減速與控制，幫助己方擊殺敵方冒進的英雄。等到敵方潰逃的時候切換毀滅巨龍形態同時開啟撕裂大地追擊敵方英雄。亦或者在己方英雄撤退時切換毀滅巨龍掩護隊友撤退，最後通過撕裂大地撤離戰場。

總體來說死亡之翼的英雄設定不可謂不用心，技能組合不僅較為完美的體現了滅世者這一英雄身份，同時無法享受隊友技能增益以及冗長的技能釋放動作也充分給予了敵方團隊反抗能力。作為一款團隊競技遊戲，在保證英雄特色的同時還能兼顧對局的平衡性，這點尤為重要！

當然死亡之翼不僅僅給風暴的英雄們帶來了恐懼，同樣還帶來了一大波福利。為了迎接死亡之翼降臨時空樞紐，官網特地為其推出了專屬的活動，只要完成3個任務就能得到6個稀有補給箱，要知道現在補給箱可以沒辦法氪金購買的，所以自認為是歐洲人的你還是去準備一手吧。當然，如果你有嘉年華虛擬門票的話，你就可以體驗到白嫖大表哥的快感了。
