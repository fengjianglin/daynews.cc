各位小夥伴們大家好啊！相信大家都知道LPL全明星周末吧！LPL全明星周末的舉辦就是為了讓玩家們感受一下不一樣的氣氛，而RNG小虎在Solo賽上的發揮大家都是有目共睹的，那一手黑科技大頭讓小夥伴們都看傻了！最近虎牙姿態也在直播中和小虎進行solo，看來姿態也要來找虐？

在第一場比賽中，虎牙姿態選擇了俄洛伊對陣小虎的鱷魚，俄洛伊有多麼變態相信大家都是知道的，而這幾場solo賽都是通過盲選的方式來進行，因此小虎也想不到姿態會選出這樣的英雄，隨後姿態便在線上展示自己的壓制力，而小虎的鱷魚也無法找擊敗姿態的方法，因此最後姿態在塔下一個Q技能將小虎擊殺掉。

在第二場比賽中，虎牙姿態選擇了蠻王上單，而小虎選擇了刀妹上單，隨後進入遊戲的時候小虎說道”你怎麼總是選這麼噁心的英雄啊！”聽到這句話的姿態也笑了，不過小虎的操作還是非常犀利的，只見姿態在三級的時候就被小虎壓至殘血，但是在隨後的博弈中，小虎還是被姿態擊殺掉。

在第三場對決中，小虎說道”只能用卡蜜爾禾劍魔！”聽到這句話的虎牙姿態馬上說道”那我用卡蜜爾！”隨後在遊戲中，姿態的卡蜜爾在一級的時候就將小虎的劍魔打成殘血，但是小虎並沒有選擇回家，因為此時兵線已經來到了線上，隨後姿態升到二級的時候，姿態直接用E技能預判到小虎閃現的位置，因此小虎再一次遭到姿態的單殺。

在第四場對決中，虎牙姿態選擇了銳雯對抗小虎的獅子狗，然而小虎屢次憑藉草叢來消耗姿態的血量，此時小虎說道”沒血還敢待著塔下？”隨後姿態說道”那你把我越了啊！”隨後雙方也達到了六級，此時姿態直接開啟大招閃現W控制住小虎，隨後一波完美操作將其擊殺掉，此時小虎說道”不打了！心態都崩了！”
