精靈、俠盜與裂谷：《傳送門騎士》最值得買的DLC

2019-12-20 184條評論 1,826次閱讀 834人點贊

新種族、新職業、新玩法，都提升了遊戲後期的重玩價值。

從《傳送門騎士》正式發售到現在，已經2年多時間過去了，這款遊戲的基本魅力至今也沒有過時：

一款結合方塊世界與RPG元素的沙盒遊戲，強化了戰鬥部分和線性引導，讓那些玩沙盒時無所事事的玩家都能快速找到明確的目標，而且還能多人聯機。

這一套設計方向，坦白說，並不是只有《傳送門騎士》想到了。類似的沙盒很多，但它是做得較好的那一款。一個證明是，今年8月，官方宣布遊戲僅在大中華區的全平台銷量便超過100萬。

在同類遊戲里，《傳送門騎士》其實相對特別，很大程度上是因為它加了傳送門。

傳送門連接了不同地形地貌、不同怪物配置的島嶼，像是把沙盒分成了一個個小箱庭。開發商確實說過，他們從任天堂的《塞爾達傳說》系列以及其他JRPG中獲得了靈感。

小島秀夫也盛讚過《傳送門騎士》的搶先體驗版本，說它當時就把《我的世界》、《勇者斗惡龍：建造者》和樂高的優點結合了起來。如今2年多過去，相比他玩的早期版本，這款遊戲也有了不少變化，門檻更低，引導更強，內容也更豐富。

就國服來說，最近剛剛更新的DLC，《精靈、俠盜與裂谷》（Elves, Rogues, and Rifts），也拓展了遊戲後期的玩法，是當前版本最值得購買的DLC。

這個DLC的名字很直白地涵蓋了其中的主要內容。

首先是新種族精靈。《傳送門騎士》本體僅有人類，DLC加入了精靈，作為可選種族。但這不僅僅是新增一類外觀，其實是新增了一整個精靈世界。

在本體中，玩家所在的世界是艾麗西亞，艾麗西亞有一顆衛星，費諾。現在玩家也可以登陸這顆月亮，在上面看到9個新的島嶼，有著精靈種族的風土人情。費諾精靈也會有一整條完整全新的劇情和任務線，它將與陰謀有關。

精靈俠盜

其次是俠盜，這是一個新職業。新增俠盜職業後，《傳送門騎士》將會有四種職業。新職業與本體自帶的前三種職業，也就是戰士、遊俠和法師，會產生挺大的差異化。

老玩家都知道，前三種職業里，戰士操作最簡單，輸出靠平砍，傷害也不低，副屬性加生命，所以又是最肉的，容錯率穩居第一，只是缺靠譜的遠程攻擊；遊俠與法師恰恰相反，各有遠程，但都比較脆，遊俠射程最遠，法師得熟練背誦屬性克製表，傷害浮動大。

而俠盜這個DLC限定的新職業會更進階，主屬性是靈巧和智慧，也很脆，且操作門檻更高。從我的試玩體驗來看，俠盜核心特色是伏擊，可以潛入陰影，出現在敵人後方，弱化敵人，增加輸出，看上去像近戰，但這個職業的所有技能其實都屬於遠程。

俠盜的天賦加點、裝備搭配、操作手法，在不看攻略的情況下，即使對老玩家來說也都有摸索的空間。而結合新DLC的新玩法，則會把遊戲後期的重玩價值最大化。

裂谷是《精靈、俠盜與裂谷》這一DLC最核心的新玩法。新加入的費諾星球中將會有一些裂谷冒險島，分低級、中級、高級三種難度。所有難度的裂谷都有一定的Roguelike元素，關卡隨機生成，並且進入裂谷即開始倒計時，玩家必須想辦法穩定裂谷。

這些裂谷的玩法邏輯都是比較簡潔的，基本思路是找到鑰匙，拿它開門，路途中會碰到具體關卡。大多數關卡都有解謎或戰鬥要素，可能需要你做3D平台跳躍，越過腳底的岩漿；可能會讓你炸開泥土，找到埋藏的開關；也可能讓你守住一波波進攻，用大炮遠程打碎對面的刷怪籠。

通關裂谷或者失敗都會傳送到裂谷鑄造廠，可以在那打造裝備、道具，消耗裂谷中可搜集的水晶與靈魂。這兩種裂谷資源都不能帶出裂谷，這意味著玩家如果有目標裝備，每一次冒險就需要儘可能多地搜集對應資源。

加入裂谷玩法以後，《傳送門騎士》的後期就更完整了。

裂谷比本體中出現的迷宮難度更高，不能光靠等級碾壓打過去，有更多解謎與打怪技巧，以及事先就告訴你這已經藏了多少個的寶箱。一層層關卡也比地圖上的主線、支線任務的目的性更強，就像一個個小傳送門。

開發者設計這套系統輕車熟路，玩家打完一個裂谷，按F進入裂隙，邁向下一個裂谷，這與玩家走入傳送門，到達下一個島嶼如出一轍，只是有不一樣的隨機關卡、高密度解謎與怪物以及時間限制。

除了DLC內容以外，國服本次更新也新增了一些創造模式的內容。比如玩家可以自由搭建場景，在場景中擺放任意怪物，好比一個關卡編輯器。自己創造關卡自己玩，或者邀請好友一起玩。

創造模式另外一項重要更新就是新增了大量邏輯功能組件，包括與或非門電路組件、觸發器、連接器、道具陷阱和音樂組件，可以用來設置陷阱機關，也能用於作曲。

這些內容使本次更新成為《傳送門騎士》國服上線以來，內容量最大的一次更新，對一款已經上線2年的中小體量遊戲來說比較難得。

目前新DLC已經在iOS版國服上線，12月19日還會更新PC版與安卓版。其中iOS和安卓版本正在促銷，本體原價30元、新DLC原價12元，在12月12日至25日優惠，分別僅需12元和8元，感興趣的玩家可以進入遊戲官網購買最新版本。
