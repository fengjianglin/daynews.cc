AHQ戰隊更新臉書，官宣新教練為GreenTea，並公布戰隊大名單。

【ahq x 英雄聯盟團隊公告】

我們熱烈歡迎 蔡尚精(GreenTea 綠茶)回歸ahq英雄聯盟團隊，擔起主教練重任！

綠茶於S3加入ahq擔任輔助位選手，S5轉後勤後，在電競圈內持續深耕。在帶隊風格上，綠茶整合隊伍時雷厲風行，但同時能亦師亦友地帶領選手不斷邁進，督促隊伍成長。期間前後在LMS與LPL隊伍不斷深造，終於在今年度，ahq有幸邀請綠茶回歸擔任主教練重責，接下來的一年，他將與副教練 鍾震華(Zero)一同引領隊伍，創造佳績、爭取榮耀。

我們同時亦要感謝主教練Domo與戰術教練NeXAbc長期以來對團隊的付出。兩位教練在任期間盡心指導、積極重整，在2019年ahq重返世界賽的旅程中，貢獻甚巨。祝福他們在電競路上持續追夢，能培育更多優異選手。

戰隊總監Polo表示：主教練綠茶的加入，是ahq S10英雄聯盟團隊陣容之中的最後一塊拼圖，亦有如隊伍的拱心石。選手出身的他，不僅深刻了解ahq體系，現在更成長為擁有豐富閱歷的優異後勤，此次回歸，相信綠茶對電競與執教的熱情，將能渲染團隊成員，凝聚隊伍目標，有望在最短的時間之中磨合隊伍風格、力臻成熟。
