《最終幻想7：重製版》新情報 新預告將於本月底公布

2019-12-04 0條評論 17次閱讀 0人點贊

摘要

近日SE公布了Jump Festa 2020參展陣容，其中包含《星之勇者斗惡龍》(iOS，Android)舞台活動，預告片；《最終幻想7：重製版》預告片和新內容；《聖劍傳說3》(PS4，Switch，PC)預告片等等。SE還公布了一張新截圖，展示了克勞德準備戰鬥的畫面，與之前公布的截圖相比，此次截圖細節更豐富，人物模型和光影有極大改善。

最新截圖

E3 2015截圖

新一期Fami通雜誌也公布了《最終幻想7：重製版》新情報，包括招式、武器成長系統、召喚獸和經典戰鬥模式。

克勞德的連招：

《最終幻想7：重製版》使用的是指令輸入形式的角色扮演戰鬥系統，輔以動作遊戲要素。玩家按下方塊鍵(PS4手柄)即可使出普通攻擊；按住方塊鍵會發出範圍攻擊；連打方塊鍵，克勞德會施展特殊連招。

按下三角鍵，克勞德則會是在行動模式和激進模式之間切換。進入激進模式的克勞德，移動速度會減慢，但攻擊會造成更多傷害。

武器成長系統：

本作中，玩家可以花費SP對克勞德的武器進行多種多樣的升級。從截圖中可以看到，克勞德的大劍可以解鎖提升物理和魔法攻擊力，以及最大HP的能力。每件武器還有獨特技能。這些技能都有熟練度(proficiency)，隨著使用次數的增多而提升。熟練度升滿後，玩家可以在不裝備此武器的情況下使用它們。


陸行鳥召喚技：

雜誌還介紹了陸行鳥/莫古力召喚技能。

莫古炸彈(MOG-BOMB)：駕馭著陸行鳥的莫古力突入戰場，使用莫古炸彈攻擊敵人！

驚嚇(Stampede)：陸行鳥/莫古力的終極能力叫Stampede，即召喚陸行鳥群碾過對手，和《魔獸爭霸3》獸王的終極技能視覺效果上類似。

經典戰鬥系統：

除了簡單和普通難度，玩家在《最終幻想7：重製版》中可選擇經典系統，即原版《最終幻想7》的ATB系統。角色會自動蓄滿ATB槽並行動，玩家只需選擇讓角色使用魔法、技能和物品。玩家可在任何時候下使用手柄重新控制你的角色，或在經典與普通戰鬥模式中切換。

魯德角色介紹：

魯德是神羅公司總務部調查科，隸屬治安維持部門的精銳部隊「塔克斯」(Turks)的一員，戴著墨鏡的光頭巨人。他沉默寡言默默無聞，會忠實地完成任務。魯德沒有武器，全憑強大的肉身和體術混吃喝。

魯德與雷諾是搞笑擔當，兩人雖然能力不是最強，但絕對是打不死的小強。而且粉絲中間一直流傳魯德暗戀蒂法。

《最終幻想7：重製版》將於2020年3月3日發售，登陸PS4平台。
