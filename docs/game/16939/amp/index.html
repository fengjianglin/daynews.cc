下面再來說晶元搭配：

1 前期推薦：因為前期刷圖難度不大，搭配的是藍色(最低級)晶元，注意一定要搭配成一套完整的晶元!!!推薦升到滿級，後期不用了也可以過渡給其他小隊提升戰鬥力。

2 中後期推薦：中期打到困難模式，難度提高很多，大部分玩家會卡關一小會(歐洲人/氪金大佬除外)，所以推薦推圖小隊的紫芯滿級，過渡用。打到地獄模式後，也可以過渡給其他小隊的魂姬提升戰鬥力去駐守。

3 各增幅晶元推薦：輸出角色你要是有死神鐮刀魂器(所有傷害附帶吸血20%)或者帶了琉璃，就可以選擇追擊套，不然你無腦嗜血套就對了，刺客射手注意看大招吃不吃暴擊
