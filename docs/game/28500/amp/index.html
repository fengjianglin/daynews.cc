LOL：馬哥德杯五殺語音放出，icon狂飆演技：沒搶到啊

天天要聞
2019-12-23

在12月21日德杯首個比賽日中，雖然說OMG無緣接下來的淘汰賽，但是smlz的回歸，加上隊伍磨合比賽發揮都還不錯，也讓人看到了以往的黑暗勢力OMG，並且在對戰YM的比賽，馬哥輪子媽斬獲了德杯首個五殺！

其實在這局比賽中OMG前期是打出不錯的優勢，上路Curse鐵男在面對YM3人包夾中，先是大招拉住卡莎，一路平A秒掉人出來後又是閃現平A收下奧拉夫，1V3反殺2人，讓OMG佔據了前中期優勢！

不過關鍵的大龍團戰中OMG是被YM打野選手Jzr奧拉夫給搶下了這條大龍，讓YM成功拖住了一段時間。

但在最終的第四條水龍魂團戰中，H4cker盲僧也反制了Jzr，搶下了水龍魂，OMG團戰大獲全勝，馬哥輪子媽一路追擊收穫五殺，OMG一波結束比賽！

細品最後一波追擊幫助馬哥五殺的途中，icon蛇女閃現腹地，細節毒牙想要K頭卻沒能成功，還有Curse鐵男的一個E都差一點，這一幕也讓OMG粉絲直呼「青春回來了」！

在今天德杯官方也放出了OMG五殺的隊內語音，正麵糰戰馬哥輪子媽三殺時，icon開始飆起了演技，先是「詢問」隊友：我要T中嗎？

嘴上說著T中，身體卻很誠實的從藍buff處開始包夾，然後閃現留人，在最後歐隊友狂喊五殺的情況下，Curse鐵男一個E差點搶了五殺，自己也忍不住笑了：差點！icon在語音中也承認了確實是想搶人頭，但很可惜的直呼：搶不到啊！

隨後幾個人也調侃了起來，Curse：sorry馬哥，差點把馬哥五殺搶了，我職業生涯沒了。H4cker：你應該今天走不出這個場館了！畢竟是馬哥的五殺，香啊！
