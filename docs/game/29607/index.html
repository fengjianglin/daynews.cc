絕地求生：黃金8決賽首局戰報 4AM驅虎吞狼8殺吃雞 VR視角歡樂多

2019-12-25 184條評論 1,651次閱讀 711人點贊

2019年12月24日，西方的平安夜在跨洋傳遞喜氣的同時，PUBG黃金大獎賽第八季的最終決戰卻殺氣騰騰。2019年的收官之戰黃金8的冠軍究竟花落誰家無疑是許多觀眾的話題之一，我們也將為大家奉上首局戰報；另一方面，本次黃金8開了VR視角，選手表現一覽無餘，歡樂蠻多的。這裡感謝競技寶JJBONE的資訊支持。

首局開始，航線從機場左邊野區飛，相對來說所有隊伍都飛得還算舒服。不過解說此時也是毒奶一句，這種沾機場邊的航線很容易刷機場圈。不過首圈卻刷了個東部圈，排名前列的TITAN在橋路過12宮時被OMG的西法掃射，這一掃就引發了兩邊的團戰。最後TITAN被打掉3人，OMG剩下兩人選擇不沖而是進行轉移。

二圈圍繞監獄開刷，SMG強沖EDG帶走一人。

三圈極限西北切，飛天樓倉庫都舒服了呀。此時RNG和OMG爆發小型團戰。最終RNG被帶走一人後雙方鳴金收兵，各自開車轉移。

四圈決定生死的一個圈他lei了。4AM在圈中心探信息，而SSS直接選擇撞SMG房區。雖然六神很頂帶走了兩人，但最終SSSNB211還是寶刀未老，進房兩槍頭直接帶走六神。不過遠點的VG也是瘋狂出槍，SSS也無力回天。

五圈，飛天樓基本吃雞好吧。圈外隊伍非常之多，大倉庫的D7G對積分肯定是最渴求的。SU4選擇沖飛天樓。VC被LSTAR加上遠點ARK芒果的槍線團滅。

決賽圈，僅剩4隊。4AM天命，但ARK位置更高，一切還很難說。而ARK近點的SU4依然也是隱患，鹿死誰手還不一定！

這時候4AM的靈性指揮來了，在ARK試圖壓近點的時候全隊選擇後撤拉開。而此時的ARK進退兩難，近點埋伏已久的隱患SU4直接與ARK交火。而SU4和ARK一打起來，右上角自然會有擊倒信息，4AM直接前壓將兩隊的積分全隊收走。一招驅虎吞狼用的真的非常6好吧，但ARK這局也是拿了不少分。

最後4AM滿編面對D7G一人，大勢已定。

恭喜4AM8殺吃雞，永遠頭上這個聖誕帽還是很可愛的。

但由於此前4AM前兩天的比賽確實有點拉，所以18分也只是讓他們來到了全場11的位置。前面分別是RNG和SMG。莫非又是最後一天玩心跳？那可真有點刺激了。

積分榜前列的競爭則是非常大，畢竟前兩天的比賽大家分差都很小。一旦一支隊伍爆發，那麼積分的總形勢將有非常大的改變。ARK憑藉這局的強勢表現來到第一位，魚人本局的栓狙簡直准到可怕，槍槍300米開外，太強了。而第二則是SU4，第三是苟到第二的D7G，BA緊隨其後。泰坦因為開局和OMG的混戰吃了虧，積分來到第六。

以上就是黃金8首局的戰報，接下來我們看看本次比賽的新玩法：VR視角。在本次黃金大獎賽中觀眾不僅可以看選手遊戲畫面第一視角，同時還能以現場工作人員的視角看選手動態，有句講句還是蠻好玩的。4AM王哥也是一直跟旁邊的D7G打趣，RYAN說要跳學校，王哥直接說：”居然還有第一威脅倒數第一的？”

另一邊，BA戰隊全員紙牌大師，在遊戲現場玩起了紙牌接龍。好了各位觀眾老爺們對此怎麼看呢？歡迎留言討論哦。
