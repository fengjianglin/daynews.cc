英雄聯盟：打野不抓人就是不會玩？錯！新版本發育才是王道

2019-12-08 0條評論 13次閱讀 0人點贊

摘要

我們玩LOL的時候如果選打野，肯定遇到過線上隊友崩盤怪自己不去抓的情況，莫名其妙背鍋，隊友經常會說：「玩個打野都不會抓人，你還玩什麼呢？」

但是S10季前賽更新之後，打野玩家終於能有理有據、心安理得地在野區發育了！那些拚命發信號讓你去幫他的人，壓根不要管他，否則只是在浪費你自己的時間。那為什麼新版本打野必須要多刷野呢？又有哪些英雄適合這個版本？有什麼比較好用的套路呢？

我們先來看看S10季前賽的更新公告，在這個全新的版本當中，每組小野怪的刷新時間縮短了30秒，而且無論你從上下哪個半區開野，都能在三組野怪後達到3級。接下來再去控河蟹和另外半邊的野怪，簡直不要太舒服。

每組小野怪也擁有了像buff和龍一樣的刷新倒計時，這更加方便打野玩家規劃自己的路線。

同時，新版本取消了打野的經驗追趕機制，如果你一直在抓人的路上卻沒有收穫的話，那麼你將會比對方打野落後非常多的等級，而且基本沒有可能追上！

更關鍵的是新版本的小龍和峽谷先鋒都變得更重要了。小龍前期更加容易擊殺，很多打野4級就能偷龍；峽谷先鋒最多可以打掉兩隻，撞翻一整條線上的防禦塔都不是問題！這就迫使打野英雄必須花時間和心思在地圖中立生物上，而不是一味的抓人。

打野由於6級前刷野速度較慢，兩分鐘就刷新的小野怪剛好夠他們不停地刷到6級，中途還要抽出一段時間去嘗試拿小龍或峽谷先鋒。如果你在6級前花費大量時間去gank卻一無所獲，那麼這一局基本你就走遠了。

於是在新版本中，刷野速度快、配合隊友能力強的打野變得非常吃香，特別是擅長和下路打小規模戰鬥的英雄，畢竟元素龍實在太強了啊！因此我推薦大家練一手奧拉夫和艾克。

奧拉夫和艾克刷野速度都是頂尖的存在，前者血越少越猛，後者自帶三環和斬殺，刷完第一輪野怪時對手可能才剛剛到4級，此時就可以回城補給直奔小龍了。只要己方下路有支援意識，很可能連小龍帶人頭一併拿下。

這兩名英雄的打野思路很像，在自家有野怪時絕不主動gank，在小龍和峽谷先鋒還存活時絕不無腦抓人，第一目標永遠是控制地圖資源。可能有人會問了，你這樣不幫隊友，他們怎麼玩啊？

其實很簡單，我們需要做的是把線上隊友召喚過來和我們一起控圖！由於新版本小龍和峽谷先鋒的重要性人盡皆知，對手很有可能會趕來交戰，這時候就能發揮艾克和奧拉夫強大的小規模作戰能力了。如果對手傻乎乎地放棄資源，那我們就能幫隊友在前期拿下很多優勢（鍍層、一血塔、元素龍屬性等等）。

這就是新版本的打野思路，一定要選擇刷野夠快並且前期也有一定作戰能力的英雄，謹記「刷野控圖至上」的策略，規劃好自己的每一步行動，幫助隊友取得更大的優勢！
