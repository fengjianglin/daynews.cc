我的世界：四種遊戲設定，對稱又美觀，能讓強迫症玩家更舒心

2019-12-14 2,679條評論 4,000次閱讀 2429人點贊

隨著時代發展的越來越快，我們身上總是會出現這樣那樣的「癥狀」，這是大多數人都存在的現象，雖然不是什麼大災大病，但是也會對我們的日常生活有一定的影響。比如「強迫症」「密集恐懼症」這樣的小毛病，相信大家都是會有的。不過呢，玩玩mc我的世界以後，大家要麼會變得更加有強迫感，要麼感受到了異常的舒心，畢竟方方正正的方塊們可以為我們帶來很多愉快的瞬間。那麼，什麼時刻會讓強迫症玩家感到十分舒心呢？

1.對稱的遺迹

Mc我的世界給我們呈現了一個非常大的地圖，如果安裝了小地圖的玩家就會發現，不管自己怎麼走，怎麼去探索，這個世界大到讓我們找不到邊界。除了使用指令或者是特殊的方法，光靠走路是不可能走到邊界的，因為當我們不斷前進的時候，地圖也是在不斷更新的。在這些不同的地形中，有幾率刷出來神殿，這裡埋藏著很多寶貝，當然也少不了陷阱。大部分的遺迹就是對稱的，看起來十分美觀。

2.草方塊的蔓延

我的世界生存模式中，大家一般都會養一些小動物，為了能夠隨時把它們身上的物資給收集起來。像羊、牛這樣的生物，能為我們提供很多幫助。想要圈養生物們的話，必須是在草地上，就連豬和雞這樣的生物，也是需要有草地才能長大的。草方塊有一個特性，那就是能夠把身邊的泥土也給變成草方塊，如果在一片泥土地上擺上一個草方塊，我們就可以看到驚人的變化。當草地漸漸蔓延的時候，就是玩家看起來很滿意的時候。

3.平鋪的水/岩漿

除了草地對我們很重要以外，水資源和岩漿也同樣重要。它們有自己的特性，發揮著方塊的作用。在現實中，水可以根據容器的形狀來改變自己，mc我的世界中也是這樣。如果把它們隨意倒在地上的話，就可以看到緩緩向外擴展的液體。這就和上面的草方塊蔓延差不多，都可以讓玩家心中得到滿足。

4.直上直下的通道

礦石對於我的世界生存模式的玩家來說也很重要，平時很多工具都是需要用礦石來當作原料製作的。不知道玩家們挖礦的時候會使用什麼樣子的方式呢？不少強迫症玩家都喜歡把礦洞挖的整整齊齊，哪怕使用了效率最低的一條路挖過去，也要保持整齊。

如果你也有那麼一點「強迫症」行為的話，在我的世界遊戲中，你會因為什麼樣的遊戲設定而感到愉快呢？
