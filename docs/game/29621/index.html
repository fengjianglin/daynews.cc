上線首周攀升到免費榜第二 這款號稱“戀愛寶典”的超休閒遊戲是如何做到的？| 遊戲茶館

2019-12-25 1,269條評論 4,038次閱讀 2291人點贊

《戀愛偵探》是一款由蓉暢遊戲開發的戀愛解謎超休閒遊戲，在12月中旬上線後下載量迅速攀升。據七麥數據統計，該產品在12月16日達到巔峰228350次下載，最高排在App Store遊戲免費榜第2,目前排在Apple Store遊戲免費榜單第4，休閒遊戲第2名。

以戀愛為主題的休閒遊戲並不是一個新的領域，以宮廷、偶像和換裝為主題的產品比比皆是，也不乏《愛江山更愛美人》、《女神之路》等有影響力的產品。但是在一周時間內，從上線到免費榜第2的攀升速度卻只有《戀愛偵探》完成了，這款產品究竟是如何設計的對玩家有如此大的吸引力，在沒有內購的情況下又是如何做好商業化的呢？

文字描述與多元素畫面構成關卡 添加人物形象增強故事邏輯

傳統解謎遊戲在畫面之外會安排題目，有些產品還會加入時間的限制，而《戀愛偵探》的解謎過程並沒有如此硬核，相對簡單的謎題設計和多條線索支援，能夠給到玩家遊玩的暢快感。

首先，作為以戀愛為主題的產品，每一關卡都是獨立的「愛情矛盾」小故事，每條線索對應了一段文字描述。比如一名渣男沙發上出現了女性的捲髮棒，遊戲文字描

在畫面元素的設置上為增添解謎樂趣，每個關卡基本都設計在房間內，會出現與生活相關的諸多元素，玩家需要從紛雜物品中甄選出有價值的線索。為協助玩家找尋真正線索，每個關卡對於主人公的人物性格都有提示，比如說不愛收拾房間的人，那麼整齊的衣架肯定是其他人協助整理的。

人物形象、文字提示和畫麵線索三個要素共同拼搭出每個關卡愛情小故事的全貌，從簡短的3-4個線索中，玩家可以判斷出到底發生了什麼，這個人是不是出軌，整體關卡故事流暢性和邏輯性得到了加強，玩家解謎過程中也有玩RPG遊戲的敘事感。

從時下熱點情感矛盾選題 激發更多年輕用戶的共鳴

一名休閒遊戲開發者在採訪中表示，《戀愛偵探》最成功的因素在於對於故事主題的把握，它選取了時下年輕人情感問題作為切入點，用接地氣的劇情激發了年輕用戶的共鳴。雖然休閒遊戲面對的是全年齡段用戶，但是15-30歲的青年用戶卻是核心主力軍，兩性情感話題更能博得眼球，且故事化的劇情也削弱了解謎的壓抑感。

以第一個「再見吧，渣男」主題為例，小關卡的故事選擇了目前常見的有婦之夫、情場菜鳥、媽寶男等典型形象，尤其是女性用戶可以在這類關卡中體驗到真實情感生活模擬，或者積累情感經驗。這款產品也不僅僅只為女性用戶設計的關卡，在後續的主題中有以性格瑕疵為例的「千面大小姐」系列和渣女系列「花心的她」，男性用戶可以體驗各類性格的女生，比如暴躁白領和電競少女，花心的她系列可以幫助男性用戶熟知各類出軌套路。

由於取源於真實情感生活的故事主題，且覆蓋了男女性用戶，《戀愛偵探》在休閒遊戲之外，也擁有了戀愛體驗和兩性關係教學的功能，一定程度上滿足了用戶的好奇心和求知慾，比普通的解密劇情類產品更加具有吸引力。

界面簡潔只插入視頻激勵廣告 添加增加爽感的獎勵關卡

值得注意的是，《戀愛偵探》遊戲中並沒有設置任何內購，遊戲的商業化只依靠廣告帶來收入。而廣告的設計並沒有安排Banner位廣告和插屏廣告，只設計了遊戲內的激勵視頻廣告，在玩家有需求的時候主動點開。

金幣作為遊戲內唯一消耗型代幣，功能為在每個關卡線索收集錯誤後扣除15枚，當金幣不夠扣除時將不能繼續遊戲。獲取金幣的渠道也是非常多的，通過每天簽到可以免費獲得一定數量金幣，此外可以通過觀看10次激勵廣告獲得1000枚金幣。另外在闖關過程中，通過一定數量的關卡會激活獎勵關，完成任務可以獲得一定量的金幣獎勵，觀看廣告可以獲得四倍獎勵。

獎勵關卡除了作為用戶獲取金幣的渠道外，還是提供爽感的玩法。在通過一定關卡後，玩家獲得懲罰渣男、勸誡渣女的機會，遊戲會出現渣男與渣女承認錯誤的畫面，玩家可以懲罰他們或是打擊第三者等方式獲得另一種遊戲爽感。

除了與金幣掛鉤的視頻激勵廣告設計外，《戀愛偵探》在解謎環節也插入了雙重廣告引導機制。傳統的解謎遊戲在玩家卡關時，可以提供廣告支援，觀看激勵廣告即可獲得正確答案，但在某種程度上也讓玩家丟失了解謎的樂趣。

《戀愛偵探》為了保留玩家充分的解謎樂趣，在激勵廣告答案獎勵上進行了雙重維度設計。第一個是文字提示，玩家通過觀看廣告或者使用提示卡，可以獲得與正確答案相關的直接線索，與關卡文字描述相比更加易懂。這樣設計的優勢在於，提供給玩家進一步的線索但並沒有直接揭示正確答案，玩家仍然可以依照邏輯進行推理。

而第二重激勵廣告插入在正確答案揭示中，玩家必須要觀看了文字線索之後才可以解鎖進階的正確答案。如此設計之下，只求正確答案的玩家必須要觀看滿兩段激勵廣告才可能，增加了廣告的曝光提升了收入。

而對於解謎遊戲本身，快速的通關和無門檻的解題往往會快速消耗玩家的興趣。《戀愛偵探》兩段式激勵廣告設置讓玩家必須接受第一段線索的提示，完整遊玩整個關卡謎題才能獲得最終的正確答案，增加廣告收入的基礎上也保障了整款產品的長線運營。

加入超休閒遊戲試玩廣告 通過實際體驗帶來更多下載轉化

一名休閒遊戲開發者表示，其實很多休閒遊戲的激勵廣告轉化並不好，一些對於實時競技或者限時對局體驗的產品，插入激勵廣告會影響玩家的體驗，玩家對於廣告內容天然存在排斥。而《戀愛偵探》並沒有加入時間限制，整體遊玩體驗偏向於輕度和弱時間化，即使玩家退出遊戲也可以隨機繼續闖關，且激勵廣告並不是自動展示，需要玩家主動點擊。

在廣告素材的選題上，《戀愛偵探》插入了多款超休閑產品的視頻廣告，尤其是《皮皮蝦傳奇》的視頻廣告在播放結束後，會進入到試玩環節。玩家會面對幾個有趣問題的選擇，會激活不同的劇情結果，最終給到玩家一個誇張的結局。

試玩結束之後，就會彈出遊戲的下載選項，通過試玩的設計進一步增加用戶的下載轉化。相比普通的視頻廣告，試玩廣告可以在較短的時間內展現出遊戲的核心玩法和獨特亮點，與試玩用戶產生互動，用實際體驗帶來更多的下載轉化。

宮廷、偶像與男友視角 豐富遊戲內容深度面向更多用戶

超休閒遊戲一直以輕中度遊戲體驗為主，重度產品目前比較火爆的只有Roguelike品類，而《戀愛偵探》在輕度的解謎遊戲體驗中設計了重度的閱讀內容，不斷增加故事主題，讓整體遊戲體驗更偏長線，從而增強了玩家的粘度。

目前遊戲的章節主題分為「再見吧渣男」、「大內密探」、「非熒幕偶像」、「千面大小姐」和「花心的她」。分別映射了現實存在的男女情感問題、在女性玩家群體中極有影響力的宮廷和偶像主題。「千面大小姐」和「花心的她」章節更是為男玩家量身定製，提供了男友視角進行遊戲，玩家需要安撫女友的情緒、解決戀愛中的矛盾。

從內容傾向上來看，《戀愛偵探》將目標用戶定位於對情感問題和流行文化感興趣的年輕人群體，用女性偏向性內容和男性偏向性內容分別取悅不同性別用戶，又創作了大眾流行文化內容來滿足泛娛樂用戶的遊戲需求，內容層面上的深度拓展給予了用戶閱讀圖文互動小說的創新體驗。

綜合來看，《戀愛偵探》採用了較為輕度的解謎玩法，搭載了更加吸引年輕用戶的戀愛主題，核心玩法上增加了線索與玩家之間的互動。關卡的設計上保持了連貫的敘事性，並在特定關卡中插入增加爽感的獎勵玩法，將金幣和線索與激勵廣告進行強關聯，並設置雙重維度不損失玩家的解謎樂趣，諸多優點或是該產品成為近期超休閑爆款的原因。■
