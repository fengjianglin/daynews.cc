4am黃金大獎賽第二日拉跨，反向吃雞引噴子來戰，高呼王欣退役

2019-12-24 780條評論 2,607次閱讀 522人點贊

這幾天成功晉級黃金大獎賽十六強的隊伍，也開始對冠軍發起了衝鋒，在為期三天的比賽中已經成功的結束了兩個比賽日，就等24號決賽的到來了，但是隨著賽事進展越發白熱化，一些戰隊也開始迸發出不一樣的風采，但是有些戰隊就開始莫名其妙拉了跨比如說4AM。

在第二天的比賽中4AM的狀態一直被大家所詬病，特別是在第六局的比賽裡面4am決策上面就出現了一些問題。從這一局的航線來看是濱海小鎮到東海岸一條比較偏僻的航線，因此戰隊在跳點上面就主動的選擇了左下角，然而僅僅開賽才三分鐘的時間一直和4am糾纏的SU4就成功的將韋神給淘汰了，同時之後其餘隊員在進行轉移時也遭遇了SU4最終王欣被淘汰，而小鹿以及永遠兩個隊伍中的核心則在房區戰中陣亡，最終4AM在這場比賽中一分也沒有收穫到。

但是反觀其餘戰隊，特別是SU4以及Ark等在第二日可謂是煥然一新，一改首日的低迷成功的逆襲到了榜首的位置，可以說針對這一次的黃金大獎賽戰隊也是第一時間進行了調整，然而4AM這支戰隊從首日到第二日的比賽中都沒有看到太大的轉變，因此最終連前十都沒有進去，很多人就感慨4AM這支戰隊每個人都十分的強悍，但是在比賽中磨合度卻不夠導致運營上面也出現問題。

比較明顯的就是能夠感受到這兩天的比賽中，4AM在前期是想通過運營進行發育的，但是只要進入到了決賽圈即使是滿編的情況下，就開始瘋狂的降智。其中一個問題就在於4AM進入到決賽圈的時間通常比較晚，但是又想要第一時間選擇圈內的制高點進行限制對手，然而這時候制高點通常都有隊伍駐紮了，這就導致戰隊在這個時候進行轉移開始出現各種問題，也就是大家所說的反向吃雞了。

對於這樣的打發在虎撲論壇上就有粉絲髮表了自己的看法，他們覺得4AM在這兩天的比賽中根本不認真，能夠感覺戰隊成員很歡樂但是對於比賽的態度就出現很大的問題，並且有一些噴子撿到了4AM這種狀態，也是第一時間帶起了王欣的節奏，表示前段時間王哥就說想退役，乾脆這一次退役就算了，這樣的節奏也是不斷的圍繞著4AM。

能夠出現這些節奏小編覺得還是4AM這一次的表現不是太讓大家滿意，要知道今年的4AM可是衝進了世界賽三強的，並且在PCL夏季賽上更是勇猛奪冠，然而只不過幾個月的時間在PCL的比賽中就出現各種拉跨的現象，讓一部分的粉絲也是有些失望，特別是王哥的狀態顯得有些低迷的情況下，越來越多的節奏就跑到了他的身上。

不過在小編看來4AM還是沒有足夠多的休息時間，畢竟今年世界賽還沒有結束多久就回來參加國內的三大杯賽，對於選手來說精神壓力還是很大的，同時選手狀態低迷也是挺正常的事兒，只要之後慢慢調整在春季PCL比賽上表現出色就行了。你是這麼覺得的么？歡迎在文章下方留言評論喔。
