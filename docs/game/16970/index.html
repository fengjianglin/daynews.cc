DNF：國服版本發布會定檔，曝光4個神秘數字，春節同步100版本？

2019-12-15 999條評論 2,113次閱讀 257人點贊

DNF：國服版本發布會定檔，曝光4個神秘數字，年底將同步100版本？

以下為具體的內容詳情：

DNF韓服12.14冬季嘉年華髮布會現已結束，從韓策爆料出的有關100版本的內容中，玩家們已經了解到了像三次覺醒，新的100級史詩「智慧之產物」，相比更為頂級的100級神話裝備，以及新的希洛克raid副本等等，而在這之後，有關國服20年版本的發布會時間也已經定檔了。

從策劃在官網上最新發出的公告中得知，有關國服DNF20年版本相關內容的發布會，正式開啟公布的時間定檔在12.30當天，而從公告中策劃除了公布發布會的時間外，其次還公布了一張預熱專題的圖片，在圖片上面有4個神秘數字。

且據了解，如上圖所示的這4個「100·3·2·1」數字組合起來的含義，就是在從四個不同的層面來預示2020年國服版本的更新內容，那不用多說，單從其中的那個「100」，玩家們唯一能聯想到的就是100版本，後面的3/2/1則是一種倒計時，也就是說等到今年年底春節版本，國服或許會和韓服同步更新上線全新的100版本。

春節版本同步上線100版本？這說實話並沒有什麼可奇怪的，必定策劃如果在春節單想靠一個魔界大戰副本根本不可能拖多長時間，還不如直接一次性來個痛快點的，春節直接同步開啟100版本，這樣也能增加玩家們的積極性，這種選擇對整個遊戲而言絕對是最有利的玩法。
