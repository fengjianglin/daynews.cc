Steam周銷榜：《大鏢客2》登頂《只狼》再度上榜

2019-12-23 996條評論 3,152次閱讀 816人點贊

　　V社公開了Steam平台上周（12月16日至22日）的銷量排行，其中《荒野大鏢客：救贖2》奪得冠軍。《光環：士官長合集》退居第二。獲得TGA 2019與GameSpot年度遊戲的《只狼：影逝二度》時隔多月再次回歸，並奪得第三名的好成績，此外該遊戲已成功入選Steam 2019年度遊戲候選名單。

　　V社VR套件從上周榜單（12月16日）的第五名下滑到了第九名；《星球大戰絕地：隕落的武士團》上周消失在榜單中，但此次成功回歸併位列第四名。老朋友《GTA5》從之前蟬聯第九名的成績中取得進步，並位列第六名。

　　*由於多個區域伺服器產品的影響，榜單上依然出現了多個《荒野大鏢客：救贖2》。

榜單詳情：

　　01、《荒野大鏢客：救贖2》

　　02、《光環：士官長合集》

　　03、《只狼：影逝二度》

　　04、《星球大戰絕地：隕落的武士團》

　　05、《GTFO》

　　06、《GTA5》

　　07、《怪物獵人：世界》

　　08、《PUBG》

　　09、Valve Index VR套件

　　10、《荒野大鏢客：救贖2》
