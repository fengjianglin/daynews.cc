《諾亞傳說手游》機甲碎片的產出途徑和合成的花費

2019-12-03 0條評論 17次閱讀 0人點贊

摘要

機甲是大家在諾亞里打架的小幫手，身為內測玩家，今天來給大家整理一下每種機甲的獲得途徑以及大概需要花費多少~不準確歡迎大家一起補充哈。畢竟我身為一個非洲人，也沒用過好機甲。

機甲是大家在諾亞里打架的小幫手，身為內測玩家，今天來給大家整理一下每種機甲的獲得途徑以及大概需要花費多少~

不準確歡迎大家一起補充哈。畢竟我身為一個非洲人，也沒用過好機甲。

稀有機甲：大部分玩家的常用機甲

（合成要10個，進階分別要3、6、12、24個，基本交易行直接買都是20~40W銀幣，所以只說怎麼產出了）

合金護衛產出途徑：

七天累計簽到、勇者積分兌換、探測器、主宰入侵

影豹產出途徑：

勇者積分兌換、探測器、主宰入侵

洞察者產出途徑：

新手歷練任務、勇者積分兌換、探測器、主宰入侵

魔能產出途徑：

首充、勇者積分兌換、探測器、主宰入侵

普通和珍貴機甲：吉祥物般的存在

合成5個，進階分別要2/4/8/16個

產出途徑直接綁銀20w一個，再就是時空漩渦、主宰入侵和探測器

史詩機甲：小R玩家可以考慮的高性價比機甲

（合成20個碎片，進階分別要5個、10個、20個、40個，以下均只算合成一階的）

刀鋒

產出途徑：充值648禮包，世界boss拍賣，金幣商城

大概花費：648免費獲得不算錢，商城25元（2500銀幣），交易行一般400W一個

這個就不算錢了，建議大家648直接充值就行，後面進階每天只要400W以內都可以在

團拍賣

暮光騎士

產出途徑：高探，軍團領主拍賣，星戰拍賣，星空探寶，星際商人拍賣

大概花費：這個不太好算，我們區前期一個碎片300~400W，現在只要200多萬就可以買一個了。金幣商城25元（2500金幣一個）。

按照均價300W一個算，20個就是6000萬，差不多400~450元。

守護巨猿

產出途徑：高探，軍團試煉拍賣，金幣商城

大概花費：我們團拍賣價一般都300W~450W一個，金幣商城25元（2500金幣一個）。

我覺得超過400W就沒必要交易行買了，不如金幣買。

按照10個交易行買，10個市場350W收來算，也差不多400~450元。

雷電狂徒

產出途徑：高探，王者試煉拍賣，金幣商城，歷練任務

大概花費：剛出的歷練任務免費送20個。金幣商城25元（2500金幣一個）。出歷練任前交易行是80~200W一個碎片，現在估計要漲價到200W一個

按不出歷練任務來算的，150W*20=2400萬銀幣，差不多170元。

傳奇機甲：大R專屬

（合成100個碎片，進階分別要15個、25個、40個、60個，以下均只算合成一階的）

女神蓋亞

產出途徑：星空探寶（500次必出5個），高探

大概花費：

抽500次星空探寶1500元（15W的金幣）得5個，也就是1個碎片300元

交易行沒看到有賣的，賣的話大概200~300W銀幣1個的樣子，

100個碎片，按20個收交易行，80個自己抽來算，整體花費24400元

毀滅祖隆

產出途徑：V12禮包，商城

大概花費：V12禮包3600元100個碎片。而後金幣商城買50元（5000金幣）一個。

大地泰坦

產出途徑：高探

大概花費：我們區交易行長期賣300~400W銀幣1個，金幣商城50元（5000金幣）一個

100個碎片，按350W一個算，3.5億銀幣，按1：17.5的比例來算差不多2000元。

末日先驅

產出途徑：高探，金幣商城

大概花費：我們區交易行長期賣90~200W銀幣1個，金幣商城50元（5000金幣）一個，100個碎片，按150W一個算，1.5億銀幣，按1：15的比例來算差不多1000元。

最後給大家幾點小建議

1、拍賣剛結束的時候經常交易行有便宜的史詩機甲碎片，一般都是小團拍賣產出的。

2、每次開等級的前幾天，機甲碎片會非常貴，因為大家都會買碎片進階，所以最好提前就囤著。

3、傳奇碎片不著急的話就在交易行收，史詩碎片超過400W銀幣就在金幣商城買更合適。

4、傳奇碎片和史詩碎片一樣貴，甚至更便宜是正常的，因為每個區能用得起的人少。挖高探挖到傳奇碎片可以留到開等級前兩天賣給用傳奇機甲的老闆，其他時候都賣不出去價格的。
