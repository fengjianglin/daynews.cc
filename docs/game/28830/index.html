雲頂之弈BUG，5級就能6人口，虎牙至強：拳頭就沒想過修復

2019-12-21 470條評論 1,629次閱讀 438人點贊

雲頂之弈進入到S2賽季以來，變化真的是非常大。不過經過這麼一段時間的熟悉，相信大家現在應該已經比較了解新版本了。不過，即便是迎來新賽季新版本，但拳頭對於某些早就存在的BUG，是都沒有修復的。近日，虎牙主播至強男刀在玩遊戲時候，就談到了這個問題。

​首先，大家先來看看下面這張圖。這是3個掠食者+2雷霆的羈絆，已經生效了。但是細看的話，還會發現有劇毒的羈絆，已經有2個人口了。而現在只是5級，所以正常情況下最多只能是5人口。但下圖中已經是6人口了，超出了等級。大家也應該能推算出來，是多了一個4金幣的老鼠。而且也能發現，老鼠的羈絆效果竟然生效了。遊俠和劇毒的羈絆，都生效了。所以說，這算不算是一種BUG呢？

​至強男刀認為，可能拳頭就沒想修復過這個BUG。而此BUG的產生的方法，需要達到2個條件。第一個條件，是人口已經上滿的情況下，人口放置區也需要放滿，一個空位置都不能留下。第二個條件，是要經過選秀環節。這樣一來，人口放置區無法擺放選秀拿到的英雄，所以就只能放場上了。然而放到場上以後，本以為不會產生任何羈絆效果，也不會進行作戰。但沒想到的是，不僅羈絆效果產生了，而且多出來的人口也會正常作戰。

​如此一來，應該就算是打破了公平性吧。即便選秀拿到的只是一星英雄，但其所產生的羈絆效果，也是很恐怖的。假如本來是蠍子和狼人2個英雄，選秀拿到了大嘴。這樣的話，就很有可能直接湊齊掠食者和劇毒的羈絆了。同樣是6級，為什麼你通過這個BUG就有那麼多的羈絆，而我這邊卻還在苦於升級呢？

​這個BUG應該很早就出現了，但至強最近沉迷雲頂之弈，所以在直播的時候說了出來。他雖然是國服第一男刀玩家，雖然是虎牙高端局玩家，但玩雲頂之弈也是牛人，而且非常細心。這次他就認為，這樣的設定是很不公平的，因為有了這樣的BUG，那要自然之力還幹什麼呢？所以至強表示，希望拳頭官方能夠儘快修復吧。

​那麼各位小夥伴們，大家在平時玩遊戲的時候，遇到過哪些類似於BUG的操作呢？
