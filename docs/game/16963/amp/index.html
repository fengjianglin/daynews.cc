《重裝機兵》：這兩款手游正在製作，紅狼戰車遙遙在望

天天要聞
2019-12-15

諸君安好，霧夏菌報道。

之前有讀者詢問，有沒有《重裝機兵》的手游，有。其中一款是《荒野方舟》的移植版，另一款是以紅狼為背景的《重裝無限》

1、荒野方舟

戰車對男生來說，似乎有著與生俱來的誘惑，而重裝機兵系列，橫空出世，走過了無數玩家童年的美夢。這一走，便是28年的光景。

如今，《重裝機兵：荒野方舟》確定將會引入國內，屬於續作系列。後續會講的《鋼之季節》就是屬於MS系列，而且還與MM1有著劇情聯繫。這一次，終於能讓手機玩家有的玩。早年這一些列的單機遊戲，在國內的價格讓很多玩家望而卻步，更多人接觸的可能都是早年的初代FC，而且大都還是盜版的。

日服的荒野方舟已經停運了，至於未來的國服究竟能扛到什麼時候就不得而知了。遊戲的內容方面，核心依舊是人與戰車，然後通過打賞金首賺取賞金。本菌個人不太喜歡這一款，日服的遊戲畫面是豎屏的，總是覺得這類遊戲豎屏差點意思。

2、重裝無限

這一款是國內的獨立遊戲，應該是重裝機兵系列的真愛粉做的。

因為只有一個人製作，所以他自己也稱之為有生之年系列，遊戲是過一段時間就要一次優化、修復。更猛的是，這個人還有在做其他遊戲，而且數量非常多。

遊戲是上帝視角，講的是紅狼的故事，駕駛著紅狼的戰車，在地圖上直接與怪物炮火對轟。沒錯，這款遊戲不是回合制，而是即時的。缺點就是有點費眼，因為要觀察遠處的小怪，所以需要視角盡量寬闊，這就要將整車縮放的很小。

再一個遊戲的小鎮，只是一個界面，並沒有提供具體的房屋，這個其實就丟失了遊戲本身的一個趣味。當年最大的樂趣之一，就是去一個新的小鎮，得知新的故事或者劇情，做一些很多日常或者其他有趣的小任務。

菌說

上面的這兩款手游，暫時可以玩一玩。荒野方舟暫時還沒動靜，只能預約，具體要多久不得而知。而另一款重裝無限，現在是可以試玩的，本菌玩了一下，感覺還是挺不錯的。不過因為模式更改，沒有以前的那個味了。

其實……到了這個年紀，很多人並不在乎遊戲咋樣咋樣，只不過想要回味或者留住曾經的那個感覺。哪怕現在遊戲畫面再精美，初代FC的重裝機兵，依舊有著玩家開開心心地玩著。我們是舊時代的殘黨，新遊戲中，沒有屬於我們的手柄……因為都在劃手機屏幕。

好了，這一回合到此結束，感謝閱讀，拜拜~
