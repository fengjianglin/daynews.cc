國人賺外國玩家錢有多簡單？這家公司出海收入第一，大廠都沒追上

2019-12-21 257條評論 3,183次閱讀 255人點贊

2019年即將結束，這對於遊戲圈來說又到了一次”大考”公布成績的時候了，對於國內遊戲市場來說三足鼎立的情況一直沒有出現，騰訊與網易兩大老牌遊戲廠商一直佔據著大量的市場份額，所以這對於中小遊戲廠商來說出海闖一闖倒是避其鋒芒的不錯選擇。

所以，早些年開始各家都摸索出了一條屬於自己遊戲出海之路的經驗，在這當中更是有不少的黑馬廠商出現，根據最近權威數據統計機構App Annie發布了10月份中國出海手游收入排行榜顯示，在這份榜單當中位居第一的是來自國內的莉莉絲遊戲，而網易騰訊則排在了第三四位。

你一定奇怪莉莉絲這家遊戲廠商在國內幾乎是默默無聞，為何到了國外能有如此耀眼的成績？首先，小編要提醒大家《刀塔傳奇》就是他們家的，之後像是出海的《劍與遠征》《萬國覺醒》都在海外獲得了不錯的收入成績，而根據Sensor Tower發布的2019年Q3中國手游在美國的收入排名當中，這兩款遊戲分別排在了第四六兩位，其中《劍與遠征》的收入漲幅更是高達60%。

那麼問題來了，《劍與遠征》這款手游為何有著如此厲害的吸金能力？其實說起來，小編作為一個白嫖玩家在玩國際服的時候還真沒有感受到氪金厲害的程度，相反其良心程度真的超乎想像，首先在鑽石獲取方面，像是完成日常、周常、主線等等都能獲得，甚至是凱旋豐碑（也就是排行榜）有了新的變動你都可以有鑽石領，更不要說平時的更新維護了，那都是大手筆的送，一次更新給個600鑽也是有的。

再者，作為一款放置卡牌手游，其玩法雖然主要是以陣容搭配為主的推圖，但是掛機的收益是提升推圖能力的主要方式，反正小編一旦推圖推不過去就掛機一波等資源到位了，在升級英雄以及裝備，平時攢攢鑽石抽十連，實在打不過就換陣容再去試。因為遊戲雖然有戰力值的存在，但是以自己的低戰力在陣容搭配合理的情況下完全可以打過一些戰力值稍高一點的對手，哪怕你是個白嫖玩家也能有這樣的操作。

所以，在遊戲的玩法上與傳統放置手游就有很大的區別，而海外玩家們對於這款遊戲的熱愛更是體現在方方面面。小編簡單的舉個例子，因為遊戲的畫風偏向於歐美玩家喜歡的風格，所以很多同人向的作品在國外社交平台上湧現，其中像是各類角色cos質量真的是非常之高。
