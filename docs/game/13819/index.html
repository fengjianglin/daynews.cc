孤影撞車eStar諾言，擊殺數都是諾言送的，純凈蒼穹后羿可曾見過

2019-12-13 707條評論 3,238次閱讀 1828人點贊

陌陌聊王者，給您帶來精彩的巔峰賽資訊！

隨著eStar被AG超玩會踢出了季後賽之後，eStar的選手們也放假休息了。雖然這一次eStar沒有能夠衛冕KPL的冠軍，但是輸一次對於他們來說也不是一件壞事，起碼能夠讓他們清醒過來，讓eStar知道他們並不是無敵之師，這樣能夠在接下來的冬冠杯中更好的迎接比賽。

在這段休息的時間，eStar的選手也會經常去打巔峰賽，畢竟巔峰賽玩家的水平還是非常高的，能夠讓他們保持手感。

​

在一局2000多分的巔峰賽中，孤影就撞車了打巔峰賽的諾言，不過這一局諾言使用的不是他在KPL擅長的邊路英雄，使用的是一個輔助的太乙真人。

諾言沒有使用平時常用的英雄，孤影也同樣沒有使用，這一局孤影使用的是射手后羿。要知道，在2000多分的巔峰賽里，后羿的出場率是非常低的，雖然諾言使用的是輔助英雄，但是對方還有一個國服的韓信。

​

這一局，其實孤影的壓力是非常大的，因為他面對的是KPL最熱門的「太香」組合，太就是太乙真人，香就是射手孫尚香。這兩個英雄的組合是非常強勢的，就算是逆風，也能輕鬆逆轉的神奇組合。

原本在對線上，后羿就不是孫尚香的對手，再加上一個太乙真人，孤影是真的難啊。不過好在其他隊友也知道不能讓孫尚香起來，因此一直支援孤影這一路，限制孫尚香的發育。

​

或許是諾言對太乙真人這個英雄使用的次數不多吧，在雙方射手和輔助2V2的情況下，「太香」組合是打不過后羿和廉頗的組合的，即使太乙真人能夠提供復活，但是依舊被孤影打爆。

不過這個時候，孤影這邊的老夫子被多次擊殺之後，心態開始不爽，一直抱怨隊友不去支援他，開始和其他的隊友爭吵起來了。要知道，這可是2000分以上的巔峰賽，隊友直接不和諧，那這局比賽就已經輸了90%。

​

雖然孤影的后羿打得可以，但是因為經濟被拉開，對方的韓信裝備起來之後，是能夠切進去一套技能將其秒殺的，為了不被韓信秒殺，又不想出防禦裝，孤影就想到了一件能夠減傷的輸出裝，純凈蒼穹！

純凈蒼穹除了能夠增加攻擊速度和暴擊流之外，還能減少受到的傷害，在韓信切進來的時候，可以保證不被秒殺。

​

不過非常可惜的是，這局最後還是孤影這邊輸掉了比賽，想要贏五個人就必須要齊心協力，這是一款五人制的團隊遊戲。

不過孤影這一局的擊殺數都是在諾言身上拿到的，看來諾言在玩輔助上還是一般般啊！
