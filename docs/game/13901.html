爐石傳說：全新功能即將上線，4大排行榜實時刷新

2019-12-12 60條評論 2,811次閱讀 1515人點贊

據ZlofanyHS透露的消息，爐石傳說排行榜功能將會再度擴充，將要加入競技場實時榜單。並且外服排行榜的官網已經加入了競技場榜單界面，不過暫時數據系統還沒搞定。

等到數據同步之後，屆時標準天梯、狂野天梯、酒館戰棋和競技場4大主流模式就集齊了。對於喜歡競技場的玩家來說自然是好事一件，不用再等N久更新一次的競技場勝率榜了。

一般來說國服會比外服晚一點，而目前外服還沒上線，所以國服可能還有一段時間，年底上線恐怕不太可能了。

排行榜功能可以說是每個遊戲都會有的一個非常基礎的功能，便於玩家展現自己的實力，增加相互之間的”攀比”，從而帶動頭部玩家競爭。在爐石傳說里的排名是實力的象徵，但如果是市面上常見的手游，那麼排行榜差不多就是戰鬥力了。

說起戰力，就不得不說一下競技場排行榜可能會呈現的形式。以往國服統計這個數據採用的都是當月整體競技場勝率，一方面會限制最少打的競技場局數，另一方面再根據勝率來進行排名，相同勝率看競技場次數，次數多的佔優。

大概率後續會同步這個演算法機制，當然也有可能會採用新的計算方法。希望競技場的玩家們敬請關注~

不過事實上現在競技場在各種模式中已經是玩家數量最少的一個模式了，此前國外數據統計顯示，僅有3%的玩家仍然經常玩競技場。絕大多數都在下棋，其次是標準模式、狂野模式、亂斗和冒險模式。

這主要是因為競技場發展至今遺留下來的一些問題，在酒館戰棋推出後又被無限放大了：

酒館戰棋上線後沒多久數據系統就出來了，競技場5年多都沒有；

平衡競技場的方法只是後台改一下好卡出現概率；

沒有任何一個與競技場有關的任務；

整體看下來，競技場模式就像是被玩家們拋棄了一樣，完全沒有重視過。

對此，各位爐石玩家朋友們，你們怎麼看呢？
