茂哥說夢幻第53期：為您解讀物價漲跌背後的原因（二）

天天要聞
2019-12-14

之所以寫這個系列是因為發現很多玩家，根本不懂夢幻西遊的遊戲原理，還特別相信那些帶節奏的弱智理論。

今天開這個系列，就來給各位盤點一下，我所知道的一些物價漲跌的原因。當然你們也可以在評論里說出一下你們想知道的物價漲跌的原因。如果我知道，我會選中在後續幾期解讀。當然，我也不是全能，不知道的我就不瞎講了。

第一期已經解讀了三個熱點問題了。是第二期了，本期主要解讀是跟速度相關的問題。

四、為什麼速度元宵在18年暴漲？

答：2017年的4月份的固傷大改，將夢幻西遊的玩法徹底進行了顛覆。自此以後，夢幻西遊進入了速度時代。所有的PVE玩法都跟速度掛鉤，而同時在17年的元宵節，推出了水晶糕，在當時PK主流玩法下，防禦元宵的價格達到了最高的價格，於是在18的元宵節大量玩家跟風兌換防禦元宵。同時，大量PVE玩家在經過一年的探索後，發現手上的須彌BB在速度達標後，只需要全魔加點就行了，那麼在這種情況下，加速度就等於加法傷，那麼速度元宵在這樣的背景下就開始暢銷起來了。再加上大量元宵積分換成了攻防元宵，市面上的速度元宵並不是滿足五開大軍的需求，漲價就是理所當然的了。

五、關於高敏跟高法連的價格顛倒問題。

答：其實原理跟上面的差不多。所有的玩法都是玩家一步一步的摸索出來的，當所有的玩法都跟速度掛鉤以後，並且隨著各大主播的直播，「效率」成了所有五開的目標。在探索完速度元宵後，效率黨們又在實踐中發現，法連連空氣的現象嚴重拖累了效率。高法連也變成了負面技能，而此時，在尋找替代高法連的技能的時候，高敏進入了效率黨的眼裡，既然加速度就是加法傷，那麼高敏可以增加20%的速度，然後再通過如意丹洗掉多出來的敏捷點用來加魔力，那麼高敏的作用就變成了加法傷的作用。在這樣的前提下，原來的四法書（高法爆，高法波，高魔心，高法連），也就演變成了（高法爆，高法波，高魔心，高敏捷）。所以你們看到其實就是高敏捷取代了高法連的位置，相應的高法連跟高敏的價格也進行了對換。原本高法連的上限變成了高敏的上限，原本高敏的下限變成了高法連的下線。

PS：我在18年年末的時候想到了這一點。於是我在19年的道人期間，收了80本高敏捷。再通過我個人的一點抬價手段，我在19年6月份的時候以1400-1450的價格全部賣給商人了。買個商人的我沒有記錄了，但是我有通過小號在藏寶閣以100的價格賣過十幾本。

你們可以看一下時間，是6月27日。正好是當時的價格最高峰的時間段。

六：如意丹價格的問題。

答：陣法的改動。18年的十月大改中，對陣法進行了改動。

我們可以看到改動最大的就是速度方面的改動了。

我們再回到問題的本身，有很多朋友就會問了，你說這個跟如意丹有雞毛關係啊。

其實我們很多五開朋友一天到晚的刷刷刷，可能覺得這個改動跟自己沒關係，但是碰巧，茂哥本人是一個幫戰混子，而且每次都是不會缺席的。所有參與PK的人員基本上有會有2個以上的配速BB，而這些陣法的差距有10%-15%的改動之後，就意味著很多人原先的配速BB會隨著陣法的改動而配不上自己的速度了，尤其以龍陣，鳥陣，在PK中更是大頻率被使用到。那麼所有X9的PK人員，包括像我這樣的幫戰混子人員的BB大部分都要洗點。這樣一來，如意丹的價格自然就瘋漲了。

PS：我當時在看完直播的時候其實我沒有立馬想到這個問題，但是當天晚上是有幫戰的，我一想就想到了，卧槽，這麼多速度位置的改動，以後的卡速BB會配不上速了啊。在想通這個問題後，我立馬跑到天台，直接把體耐敏如意丹全秒了，然後還私信了很多天台商人，把他們的如意丹也按照擺攤的價格收了過來，大概秒了四五車的樣子。後面等漲起來的差不多賺了6-8萬一個。

其實這裡還有一個原因，那就是全力BB的流行，很多人把血攻洗成了全力，這也是為什麼木如意丹（體質）是最貴的原因。至於火如意丹（魔力）為什麼沒漲嘛，因為沒有BB會刻意去加魔力，有也是吃千金露的時候隨機加上去的。
