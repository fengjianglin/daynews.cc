《我的坦克我的團》12月24日安卓首發，報告局座，請求開炮

2019-12-17 127條評論 2,501次閱讀 1229人點贊

大家好，我是小編阿茶，說到坦克遊戲大家會想起什麼呢，是之前大熱的坦克世界還是童年回憶里的坦克大戰。

阿茶倒是很喜歡坦克類型的遊戲，所以今天我要介紹的遊戲是局座（張召忠）代言的指揮類遊戲：《我的坦克我的團》。遊戲將於12月24日安卓首發！

《我的坦克我的團》是由湖南草花互動網路科技有限公司推出的一款具有軍事特徵的策略塔防類手游。

遊戲的名字很容易讓人想起熱播電視劇《我的團長我的團》，而遊戲中我們也是作為指揮官率領著自己坦克團征戰沙場。

遊戲背景設定為以二戰諾曼底登陸戰役為開端的架空的世界觀。遊戲內共有三個陣營共玩家選擇，分別是帝國、聯邦、共和。地圖上共有107座城池，任何城市都是可以佔領的，不同城市可以產出不同的資源。玩家可以通過不同發展方針來決定如何防守如何，是堅守當下的利益，還是放棄當下去獲取更大的勝利。

遊戲中復原了經典的二戰和現代坦克，讓坦克迷們盡情的培養他們所喜歡的坦克類型。同時遊戲中也投用了一些科幻坦克，可供玩家體驗復古，現代，和科技帶來不同的戰鬥體驗感。遊戲中收錄了自二戰到現代的百餘輛坦克和戰車。

經典戰役，經典事件真實復原，多個戰役能讓你親臨戰場。

各式各樣的活動層出不窮，軍銜提升不是問題！

最後，相信許多小夥伴都有一個軍事夢，渴望在戰場上建功立業，《我的坦克我的團》就可以滿足你的這個夢想，指揮著自己的坦克團，經歷著歷史上的經典戰役。遊戲機制看似複雜以及指揮的概念也很籠統，但是真正上手，看著各種各樣酷炫的坦克，就會真正愛上這個遊戲。小夥伴們可以點擊下方鏈接來預約這個遊戲啦，有第一資訊也會通知大家。
