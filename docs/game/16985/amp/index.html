街機遊戲《合金彈頭》主角感染喪屍病毒後，居然還有這麼多的天敵

天天要聞
2019-12-14

在《合金彈頭3》第二關中首次出現了人物的喪屍形態，這是繼胖子、木乃伊後的第三種變化形態。擁有傷害逆天的噴血技能，被認為是目前遊戲中殺傷力最強的攻擊方式。

《合金彈頭3》出現過兩次喪屍場景，第一處就是2關的摩羅十誡降臨地球並散播喪屍病毒，意圖將地球打造成為火星人宜居的環境；而第二處則是火星人抓捕了地球最強的人類進行克隆人計劃，企圖打造出最強的軍隊。而被抓住的正規軍身體中殘留了未徹底清除的喪屍病毒。因此在克隆人巢穴被摧毀之後，部分繼承了喪屍病毒的克隆人並未當即死亡，而是進化成無敵的喪屍。

《合金彈頭4》第四關遊樂場中喪屍再次出現。而這一次是人為製造的。一群科學家將遊樂場當成了實驗基地。在這裡釋放大量的木乃伊氣息和喪屍病原體，製造恐慌。

看過《生化危機》的朋友應該都知道，喪屍的生命力極為頑強，只要沒有被爆頭任何損傷都有可能活下來。在這款遊戲中沒有爆頭的設定，每一隻喪屍的血量都是非常厚的。玩家要是被感染了喪屍病毒並不會當即死亡，而是被同化成為喪屍。移動能力差，無法使用手槍以外的武器，無法正常跳躍，但是卻擁有了「無敵」的屬性，可以免疫叛軍造成的任何傷害，而且擁有整個系列中最強的技能。因此很多玩家非常樂意被變成喪屍，就算是BOSS也經不起噴血技能的摧殘。

除了病原體，似乎並沒有其他方式可以消滅到喪屍形態。叛軍的載具和武器，在主角變成喪屍後就成為廢銅爛鐵。想要殺死主角的喪屍形態，只有再中一次喪屍的攻擊。另外，病毒發布者摩羅十誡的守護者「莫羅之眼」同樣具備殺死喪屍的能力。

在喪屍場景的一處隱秘山洞中，竟然還有一種生物體質可以剋制喪屍病毒，那就是「雪怪」。

在滿是喪屍的環境中，只有雪怪場景還有半分的寧靜。如果玩家以喪屍的形態進入雪人場景，就會發現雪怪吹出來的凍氣就是破解喪屍病毒最好的方式。

然而在玩家返回的時候才發現，雪怪山洞被數量龐大的喪屍大軍圍剿，雪怪被屠戮殆盡。而在不為人知的角落中，一些膽大的科學家冒死收集到了雪怪之血。並在之後的版本中研發出了可以解喪屍病毒的血清。

在《合金彈頭4》中玩家要是變成喪屍的話，是可以直接被BOSS殲滅的，就是因為他們已經完全掌握了喪屍病毒。

喪屍在其他關卡中出現，是不是就沒有天敵了呢？

在《合金彈頭1》中還沒有其他形態的出現，要是此時出現喪屍的話，估計艾倫抵擋不住一次的攻擊吧！初代的BOSS和敵兵血量都是比較低的。

喪屍在遊戲中會失去進入載具的能力，水下關似乎根本就進不去吧！

在《合金彈頭3》中擁有那麼幾隻無敵的生物：海鰻。這些被輻射巨大化的生物一直都是以水母為食，不會主動攻擊玩家，而玩家只需要躲開就行了。因為無論用什麼方式攻擊它，都不會有任何效果的。海鰻堪稱是整個遊戲中最強的生物。

那麼，要是玩家使用遊戲中最強的喪屍噴血技能對上海鰻的話，會有什麼有意思的事情發生呢？

最終的結局始終是井水不犯河水，海鰻殺不死喪屍、喪屍打不死海鰻，甚至在這裡連分都刷不到。一直這麼耗下去，不會有一點的收穫。

喪屍不會有浮力，進入水中之後要是沒有踏腳的地方就會一直往下掉，直接到達海底最深處。在沒有場景限制的地方可以快速的降落到最下方；而要是有場景限制的話，掉下去就死了。不過我們也可以看得出來，喪屍形態是不會被淹死的。

在遊戲中，喪屍病毒和木乃伊氣息都是來自火星。然而這兩種病毒卻並不能相剋。

喪屍狀態的玩家，即使被木乃伊的氣息擊中也不會有任何反應。而且可以免疫各種陷阱帶來的傷害。

在高速移動的環境中，喪屍的跳躍能力就排上用場了。可以追上前面飛行的BOSS進行追擊。

其實很多玩家都發現了，喪屍的移動能力慢，不過可以利用跳躍能力代替移動能力，稍微要快那麼一點點。喪屍在其他關卡中都是無敵的存在，無論敵兵使用什麼炮彈，都無法傷害半分。

肯定會有玩家疑問，要是在《合金彈頭3》最終關，喪屍形態還有沒有可能被抓走呢？

小編嘗試了一下讓喪屍形態出現在第五關，在打敗假摩登之後進入劇情，喪屍形態還真就被火星人抓走了。如果劇情照這樣發展，是不是就可以填上「克隆人喪屍化」的坑了呢！那其他問題就來了，玩家到達克隆人巢穴之後，裡面關押的是不是喪屍形態的主角呢？而打造出來的克隆人是不是直接就是喪屍形態了？

答案我們應該都想像得到，劇情是不會這麼發展的。

《合金彈頭3》中出現的最終BOSS「火星之根」，其實並非是完全形態。因為在逃避金星人襲擊的時候「火星之根」已經嚴重受損。玩家在攻入火星人巢穴時，火星之根正在修復。也就是說，此時是火星之根最弱的時候。

在玩家逃出生天之後，剛好遇到了前來複仇的「火星之根」。此時的「火星之根」已經擺脫了修復裝置的束縛，擁有了活動能力，於是抱著同歸於盡的想法和玩家展開了最終一戰。如果玩家是喪屍形態的話，火星之根是完全沒有辦法的，無論什麼攻擊方式對喪屍來說都可以免疫。而喪屍的噴血能力可以在很短的時間內將「火星之根」殲滅。

經過多次嘗試，我們發現「木王星」和「太陽神」同樣沒有對付喪屍的能力。它們的所有技能包括「化金」和「鐳射眼」都無法傷害喪屍形態。

BOSS對喪屍形態束手無策，那麼火星人製造的輻射生物和火星蟲子有沒有辦法消滅喪屍呢？

喪屍病毒最初就是來自火星的摩羅十誡，因此同樣來自火星的部分蟲子，是可以消滅喪屍的。進入蟲穴之後，喪屍要是被巨型母蟲衝撞的話必死無疑，就算是喪屍形態也照殺不誤。

在第一關中出現出現了大量被輻射的蝗蟲，它們已經擁有了巨大的身軀和無限飛行能力。看得出來這群傢伙已經進化成為肉食動物，場景中只要出現的生物都逃不過它們的利爪。在蝗蟲眼中，只要是能夠移動的生物都是食物。不管玩家是胖子、木乃伊還是喪屍，都僅僅是它們的口糧而已。

就是不知道吃掉喪屍的蝗蟲，會不會進化成為更加恐怖的怪物。

在遊戲中還有一種非常另類的生物，那就是食人花。在《合金彈頭3》第四關出現，我們會發現火星人假扮的摩登，指揮著叛軍在這裡培育食人花種子。然而因食人花的繁殖速度過快，導致整個場面失控，就連叛軍都成為了食人花的食物。不過這裡出現的都是培植出來的成品，母體被安置在另外的地方。

玩家此時就算是喪屍形態，仍然免不了成為食人花的口糧，就是不知道能不能消化。

喪屍的噴血技能，一直以來都是玩家們最為津津樂道的話題。噴血技能是持續性的攻擊，小兵都是不堪一擊，而BOSS被攻擊部位並不能完全的覆蓋，因此在遊戲中使用噴血技能很難發揮完整的威力。因此我們也無法判斷噴血技能的完整威力。

如果喪屍技能發揮到極致，到底有多大的威力呢？

小編嘗試將攻擊改成全屏，這樣的話就可以將喪屍噴血技能完全的發揮出來了。如果是單人玩的話，太陽神的血量是500發手槍子彈、而雙人玩的話太陽神的血量是700發手槍子彈。

單人模式中「太陽神」甚至經不住兩次噴血技能就被殲滅了，這樣無法測試出完整的威力。而在雙人模式中「太陽神」剛好可以承受兩次噴血攻擊，也就是說噴血技能一次的完整威力是350發手槍子彈，就算有誤差也不會很大的。

「木王星」在遊戲中是數一數二的堅硬，血量甚至比太陽神還高。雖然它的攻擊套路很容易掌握，但是時間會耽擱非常久。當年我們將所有的武器和彈藥全都消耗完後，就只有等死了。運氣好還可以打出武器來，運氣不好就死一命然後繼續使用10顆炸彈。

BOSS的血量，單人模式中使用手槍子彈需要：650發

使用噴血技能的話只需要兩次就足夠了。

最終BOSS「火星之根」的血量是最厚的，畢竟是最終關BOSS。玩家就算是使用噴血技能，也需要五下（四下半）才能搞定，而且必須開起全屏攻擊才能做到。那麼它的血量到底有多厚呢？

小編嘗試了一下使用手槍子彈，需要1600發子彈才能搞定。這樣算起來的話，噴血技能的確是350發左右的手槍子彈

《合金彈頭3》之所以能夠成為該系列的巔峰，喪屍形態的出現絕對是功不可沒的。然而遺憾的是，在《合金彈頭》後期的版本中再也沒有出現這個設定了。
