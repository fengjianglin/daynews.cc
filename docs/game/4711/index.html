LPL又一艘銀河戰艦初成：SN雖錯過卡薩卻迎來了另一個強力打野

今年的LPL休賽期可謂風雲變幻，卻又波瀾不驚，除了阿水出走IG，卡薩離開RNG兩則消息在網上掀起了不小風浪外，其他戰隊搞出的動作根本驚不起絲毫響動：

LGD迎來LCK全明星打野小花生，但是一個打野位的加入在我看來並不能改變什麼；被稱為御三家的RNG、EDG、WE不但在轉會期沒啥收穫反而有重要人員的流失，可以說他們三支戰隊是賠了夫人又折兵，很可能在下賽季成為魚腩戰隊；IG保全了除了阿水外的所有人員，可我真不知道失去阿水並且還留住寶藍的IG能在下賽季打出什麼表現。

以上我提到的都是在休賽期運轉不靈的戰隊，而FPX作為今年的S9冠軍，不但保留了所有首發成員，還加入了SKT的上單可汗，這也是我為什麼在開頭說「今年的轉會消息風雲突變，但波瀾不驚」的原因，FPX的如虎添翼讓下賽季LPL的局勢顯得十分明朗，不管如何變化，我們也很難想像在下賽季會有什麼戰隊能威脅到FPX的衛冕。

但最近轉會期一直有大動作的蘇寧卻有了實質性進展，下賽季的SN很可能已經組成了一支能威脅到鳳凰的銀河戰艦，上賽季的SN成績差勁，很大原因是因為上中野的疲軟，而在休賽季SN放走了除了輔助蛇蛇外的所有成員，而關於SN下賽季的陣容到目前為止已經大致成型，讓我們來一起看一看吧：

上單：Bin；Bin是個新人上單，打法兇悍擅長戰士上單，目前已經打到了王者1216分，是個不折不扣的路人王，在韓服排位中多次單殺韓服第一上單牛國瑞，但是具以往的經驗，rank分並不能決定賽場上的成績（比方說小樂言），所以新人上單能否在賽場上有不錯表現要打個問號。

中單：Angel；Angel在上賽季是SN的替補，他的幾次替補出場也多有亮眼表現，目前已經被SN扶正了，當然不排除SN會招募中單兮夜作為首發中單，當然可能性不大奧。

打野：SofM；這名越南打野已進入LPL就把我們的野區給統治了，當時LPL幾乎所有的打野都經歷過被他入侵野區的恐懼，據可靠消息，SN下賽季的打野大概率就是這位離開蛇隊的越南打野了；當然本來也傳出卡薩要來SN的消息，但目前看來卡薩好像已經和TES走到一起去了。

ADC：阿水；之前阿水疑似簽約的照片就在網上被爆出，一度被大家重度懷疑的SN好像確實已經簽下JackLove了。

輔助：蛇蛇；蛇蛇是上賽季SN表現最穩定的一個點，所以最後SN也是成功留下了這個LPL前五的輔助，不知道下賽季和阿水搭配的他，能讓SN的下路達到一個什麼樣的高度呢？

雖然SN以這樣的陣容也很難在下賽季戰勝FPX，但是從紙面來說，SN在下賽季絕對會在LPL佔據一片立足之地，特別是在今年幾支強隊都元氣大損的下賽季，SN和TES很可能成為FPX最大的攔路虎。
