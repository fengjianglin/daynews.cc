雲頂之弈9.24b版本：地獄火、遊俠增強，夢魘、婕拉慘遭削弱

天天要聞
2019-12-19

雖然9.24版本是今年最後一個版本，但由於玩家的反饋以及數據顯示，拳頭設計師仍然選擇在今年最後的階段進行了9.24b版本的更新。並且特彆強調：這是我們今年的最後一次更新，一月份，我們將會帶來10.1版本補丁。

3星英雄一直是一個需要平衡的問題，因為他們的生命值和攻擊傷害的過渡公式很僵硬。我們想通過去掉這個公式，就可以把這個能力放到其他方面。

3星英雄成長生命和攻擊傷害與2星英雄定位：2x ⇒ 1.8x

例如血量600/1080/2160調整為600/1080/1944；

平衡調整：

我們對地獄火的調整很滿意，但6地獄火和9地獄火仍然沒達到我們想讓它具有的競爭實力水平，所我們給了它一個調整。雷霆的傷害也有點超出了我們的預期，所以我們稍微削弱了一點傷害（儘管2雷霆仍然高於9.23版本）。最後，我們很高興看到影的登場，但它現在很容易和其他英雄組成陣容，所以我們現在要降低它的影響力。

3影加成傷害：70% 降低為 60%；

雷霆傷害：100/300/500調整為80/250/500；

地獄火傷害： 70%/120%/180% 調整為 70%/140%/210%；

刺客：

3刺客：75%暴擊傷害 10%暴擊幾率 調整為 50%暴擊傷害 10%暴擊幾率；

Noc治療效果：40%/60%/80%調整為40%/45%/50%；

劫AD傷害由80降低為70；

劫護甲值由30降低為25；

新英雄：

賽娜和阿木木和預期效果比較接近，所以只需要一些小的調整：

賽娜為友軍提供的傷害加成：25/45/75調整為15/30/55；

法力值：50/150調整為50/125；

阿木木技能範圍：2/3/4調整為全星級3格；

遊俠：

艾希每支箭矢傷害：25%/30%/35% 加強為 30%/35%/40%；

圖奇AD傷害由60增加為65；

圖奇法力值：0/90調整為0/70；

一般調整：

婕拉技能傷害：70/80/90 降低為 55/65/75；

雷克塞AD傷害：65 降低 為60；

布蘭德法力值：0/90調整為0/80；

沉默與繳械：

沉默幾率：33%降低為25%；

繳械幾率：33%降低為25%。
