電競市場還會繼續火爆嗎？看螞蟻電競入局一年“總結”

2019-12-24 547條評論 1,817次閱讀 178人點贊

電競行業的發展帶動了一眾周邊產業的發展，特別是近幾年，直播、電腦硬體、外設產品等都迎來了第二春，甚至連不少學校也新增了電競專業，讓遊戲正式登上大舞台。

電競行業的崛起讓DIY行業看到了一線生機。眾所周知，前幾年，PC電腦的銷量一直費靡不振，而電競的到來讓硬體廠商、外設廠商再次興奮起來，見到了一絲希望。而就在2018年，螞蟻電競正式入局，今年是它兩周歲生日，在2019年12月21日－22日AMTGAMER螞蟻電競品牌在武漢理工大學隆重舉行了「肆青春 競真我，2019螞蟻電競嘉年華暨產品發布會」，在此次螞蟻電競發布會上，迎來了令人震撼的螞蟻電競新品。

一年間，看螞蟻電競如何入局

螞蟻電競是2018年誕生的新生品牌，其實他的「爸爸」是HKC（惠科），所以它並不缺做顯示器的基因。但入局電競並不是那麼容易的事情，畢竟各大電競顯示屏廠商實力都十分雄厚，那麼螞蟻電競又有何優勢呢？

其實螞蟻電競除了擁有出色的產品外，還一直注重於打造一個電競的生態，從玩家裝備、玩傢俱樂部、玩家體驗中心和玩家賽事四個方面滲透到電競行業中。在售後方面，「螞上修」做到5分鐘響應，2小時上門，建立全新服務標準，只換不修政策的保證讓玩家用得更安心。

而此次發布會不僅有螞蟻電競的新品發布，ACUL電競賽事決賽也是本次活動的一大看點。注重於電競生態的發展，從生態入局，這一年，螞蟻電競可以說收穫良多，也交出了一份十分優秀的成績單。

電競顯示器談「色」也是首例

這次嘉年華上，螞蟻電競帶來了兩款全新產品，一款是搭載IPS「實景屏」的ANT27VQ，另一款是搭載「月牙屏」的ANT32VQC。

這兩款產品最大的區別就是ANT32VQC是採用了31.5英寸1000R曲率，VA面板設計，而ANT27VQ採用的是27英寸IPS屏設計。

共同點方面，ANT27VQ與ANT32VQC在Adobe、DCI-P3、NTSC和SRGB四大標準色域上均達到了100%，而且醫療級灰階調校和ΔE<2以及165Hz刷新率同1ms響應時間的2K解析度。並且兩款顯示器均支持G-SYNC、FREESYNC技術，能夠有效防止畫面撕裂。還值得一提的是，兩款顯示器都支持旋轉/俯仰/升降/豎屏等多角度調節，無論你是設計師還是遊戲玩家，兩款顯示器都能給你最出色的體驗。

以往電競顯示屏一味追求高刷新率，追求最快的響應時間，而忽略了顏色方面，其實對於一般用戶來說，一款全面的顯示器才是用戶所需，既擁有精準的色彩還擁有165Hz刷新率，同時響應時間均做到1ms，可以說照顧到設計、畫圖、遊戲玩家的需求。

1000R曲率會帶來什麼體驗？三星告訴你

依稀記得全面屏產品是從從2015年至2018年陸續推出的，當時曲率4000R、3000R已有非常優秀的體驗，而近幾年全面屏曲率方面已經做到了1800R、1500R的曲面屏產品，讓用戶在遊戲或者 娛樂體驗中有更強的包裹感，從而有更優秀的體驗。

而ANT32VQC此次就搭載上了1000R曲率，螞蟻電競也親切地給它冠以「月牙屏」。三星與螞蟻電競繼續強強聯手，三星DISPLAY把最新研製成功的1000R曲面屏優先提供給螞蟻電競，目前為止最具沉浸感的ANT32VQC電競顯示器誕生了！1000R曲率會給用戶帶來什麼更震撼的體驗？值得期待。

除了新品發布，ACUL電競賽事也是本次活動的一大看點，同時還舉行了TT的MOD大賽，讓整個電競嘉年更多姿多彩，每一個主題MOD作品都收穫了不少玩家們的肯定和投票。現場也能體驗到很多TT家的產品，新發布的G821 USB/藍牙/無線三模鍵盤值得期待。

此次的螞蟻電競嘉年華就這樣結束了，期待明年螞蟻電競會帶來更多出色的產品，2020年螞蟻電競值得期待。
