目標，冰鳳凰杯！冬季冠軍杯KPL戰隊盤點

2019-12-20 2,069條評論 4,072次閱讀 3069人點贊

2019冬季冠軍杯即將拉開帷幕，KPL戰隊將與KRKPL戰隊共同爭奪象徵著至高榮耀的冠軍獎盃，這場境內外的激烈角逐一定會格外精彩。

晉級規則

本屆冬季冠軍杯共有八個名額，KPL賽區2019年秋季賽冠軍AG超玩會、亞軍QGhappy以及季軍DYG.JC這三支戰隊，將直接獲得淘汰賽資格，餘下的五支隊伍eStarPro、RW俠、TS、Hero久競、XQ將與KRKPL的三支戰隊共同競爭僅剩的兩個名額。

戰隊盤點

AG超玩會

作為2019KPL秋季賽冠軍，AG超玩會意味著KPL秋季賽最強戰隊。本賽季重新歸來的AG超玩會在常規賽以西部第一的姿態衝進季後賽，掉入敗者組後一穿四斬獲總冠軍。多變的戰術體系以及將運營和打架完美結合的戰術風格，是AG超玩會奪冠的關鍵因素。他們獨特的六人輪換體系以及難以捉摸的BP風格，在冬季冠軍杯恐怕會讓其他戰隊格外頭疼。作為新科冠軍，AG超玩會將是冬冠杯冠軍的有力爭奪者。

QGhappy

QGhappy善於運營和射手體系，他們在這個版本可謂如魚得水。常規賽以東部第二的成績晉級勝者組，接連戰勝AG超玩會與DYG.JC後晉級總局賽。雖然最終與冠軍失之交臂，但也正是這樣的失敗讓他們正視到自身的短板和不足，涅槃重生的QGhappy在冬冠杯想必會有很大的改變。不僅如此，善於野核的1dao也在QGhappy的出征大名單之中，這留給我們許多遐想和期待，在冬季冠軍杯，QGhappy是否會突破較為單一的射手體系，展現出更多的戰術向冠軍衝擊?

DYG.JC

「人均絕活哥」的DYG.JC，在秋季賽給觀眾帶來了太多的驚喜。常規賽以7勝7負的成績位列西部第二，在季後賽與上屆冠軍eStarPro和本屆亞軍QGhappy都戰至第七局巔峰對決，這足以證明他們的實力。在季後賽，DYG.JC曾多次拿出雙射手體系，這也能看出他們在戰術上尋求多變的決心。雖然DYG.JC遺憾止步敗者組決賽，但這磨滅不了他們衝擊冠軍的信念。在冬季冠軍杯，這支「絕活」戰隊想必會拿出更多的絕活向冠軍衝擊。

eStarPro

作為2019KPL春季賽和世界冠軍杯的雙冠得主，eStarPro的統治力在秋季賽常規賽中依舊存在，善於創新是他們的特色之一，他們以12勝2負的傲人戰績穩坐常規賽東部第一。不過在經歷版本變動後，eStarPro受到了不小的影響，這也導致他們在季後賽戰績不佳。從打法風格看，eStarPro與本屆冠軍AG超玩會類似，雙方都喜歡通過「打架+運營」的方式來掌控比賽的節奏，面對其餘7支參與冬冠選拔賽的隊伍，eStarPro的打法具備較強的優勢。在經歷過失敗的反思和沉澱後，eStarPro很有可能拿下晉級冬冠淘汰賽的資格，向冠軍衝擊。

RW俠

RW俠在季後賽的表現可圈可點，對戰RNG.M時的無法師黃忠體系驚艷四座，之後與eStarPro的交戰也相當精彩。渡劫作為隊內的關鍵先生，他極強的個人能力穩住了隊伍的下限，逆風時能夠挺身而出拯救局勢，楊過、渡難、花滿樓和成昆也都有不少亮眼時刻。不過RW俠在季後賽也暴露出了一些問題，在關鍵時刻選手的狀態不夠穩定，這需要引起他們足夠的重視。，畢竟不能只靠渡劫一人力挽狂瀾，在冬冠杯想要衝擊冠軍，必須要五人都扛起Carry的大旗才行。

XQ

XQ這支由新人組成的老牌戰隊在常規賽的表現獲得了很多觀眾的肯定，以10勝4負西部第三的成績衝進季後賽。無論是九尾還是釺城，亦或是綁兔、冰塵、晨羽，從他們身上我們能夠看出不俗的天賦，九尾在常規賽的KDA更是超過許多老牌中單選手，新人的熱血和衝勁在他們身上體現的淋漓盡致，然而大賽經驗的不足也是新人的致命缺陷。XQ雖然在季後賽第二輪就被淘汰，但正是這樣的失敗為他們提供了更多的經驗，也讓他們越發沉穩。在強隊如雲的選拔賽，XQ的競爭力依舊很強。

TS

如同他們的口號一樣，TS是一支不止奇蹟的隊伍，秋季賽從附加賽逆境逢生後又衝到敗者組第三輪，他們獨特的野核+射手體系效果奇佳。我們知道，在這個版本多數戰隊會選則野核+雙戰邊或是藍領野+射手體系，而TS則更善於使用野核+射手，這樣的陣容在處理好資源分配的問題後將具有很大的優勢，畢竟單核更容易被針對，多核則具有相對高的容錯率。總決賽雙G大戰說明了一個問題，體系單一很有可能成為致命缺陷，而像TS這樣體系獨特的隊伍，在冬冠杯的表現令人期待。

Hero久競

作為曾經的三冠戰隊，Hero久競在賽季初換上了五名新人，他們都有著不俗的實力。但是經過一個賽季的輪換調整，Hero久競卻越來越不穩定，在季後賽更是多次在順風局勢下出現致命失誤被對手翻盤。和XQ一樣，Hero久競這支換上新人的隊伍欠缺足夠的大賽經驗，團戰細節的處理和心態都還需要磨礪。不過久誠、塵夏以及最初這三名老選手都在此次冬冠的出征名單上，這也讓我們期待Hero久競能否在冬冠杯再現三冠王的實力。

KPL賽區的八支戰隊都具備極強的實力，除了晉級淘汰賽的三支戰隊，其餘五支戰隊在選拔賽中的競爭想必會非常激烈。KPL賽區究竟有哪支戰隊能夠成功晉級淘汰賽，向著冠軍的目標衝擊?我們拭目以待!
