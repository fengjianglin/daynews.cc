元氣騎士：大騎士舊劍和暗黑騎士之劍真的存在？真相馬上為您揭曉

天天要聞
2019-12-14

距離上次《元氣騎士》大版本更新已經過了近兩個多月時間了，在這期間除了在萬聖節來臨之際涼屋對角色大廳的界面主題進行更換以外，其他遊戲內容幾乎可以說是毫無變化。雖然涼屋大大也曾在10月份的官方公告中表示，要進行閉關修鍊研究全新的「牙膏配方」，可是老宅也沒曾想到會等如此之久，看來當前大家也只有耐心等待一個辦法了。至於小夥伴們最關心的下次更新時間，官方也並沒有透露任何消息，不過作為一位老玩家多少也應該對《元氣騎士》的更新周期有所了解吧，若根據去年的更新頻率進行推測，老宅估計會在1月份進行2.4版本更新，而新版本的預告內容大約也會在本月底下月初開始陸續公布，還是那句話「好飯不怕晚」大家就先稍等「片刻」靜候佳音吧。

本期將要和小夥伴們聊的話題雖然和即將來臨的2.4版本沒有多少關聯，但卻和之前的2.2版本有著很大的關係。因為這個話題正是從那時隨著新BOSS新裝備的上線開始衍生而出的，而且隨著時間的推移愈演愈烈，就算曆經數次版本更新仍然讓不少玩家為之糾結。那麼本著「有問題就解決問題」的原則，老宅就藉此機會和小夥伴們一起論證這個讓大家糾結的問題，那就是：「大騎士舊劍」和「暗黑騎士之劍」真的存在嗎？

大巫師

要知道，關於這個疑問的由來，不得不提到在2.2版本中榮獲新生的BOSS們，而其中最具有代表性的當屬第二關的守將大巫師。由於《元氣騎士》本身只是一款2D像素風格的小型單機遊戲，所以對於遊戲中的很多場景和角色在建模上都做得較為簡易。而大巫師的線條在一眾BOSS中無疑是最低質的，因此涼屋大大為了維護整體的協調性，對新版本關卡的美工進行了優化。毫無疑問在2.2版本中，大巫師以及其專屬武器都得到了質的進化，將原本手持火柴棒的「矮丑挫」形象變成了手持三尖兩刃刀的「二郎神」模式，而武器更是一分為二，強化的成為大巫師之杖，弱化的叫大巫師的舊法杖。

新舊法杖

其中紅武形態的新法杖除了外表的變化以外，攻擊模式並沒有產生更改，依然是初始三元素單次7發的散射神器。而舊法杖也得到了保留，屬性外觀仍然與之前的舊紅武一模一樣，只是將品質降階為綠武，子彈數量也隨之降低57%，成為初始只有3發的小型元素法杖。所幸舊法杖雖然降級，可是對實力的影響並不大，並且還增加了裝配工匠配件的能力，在領域中仍然屬於佼佼者。

而玩家也正是在這種一分為二的全新武器模式下，對另一位BOSS產生了聯想，傳言就此開始。

大騎士

在2.2版本中獲得新生的BOSS當然也並非只有大巫師一位，還有同關卡的大騎士也同樣得到了脫胎換骨，而且在某種意義上甚至更勝前者。大騎士的裝備變得更加堅固鋒利形象也變得更加威武霸氣，甚至還一度激活了體內的陰暗面，形成了另一種更為強大的暗黑大騎士。或許策劃在優化這位BOSS時的創意是出自《街霸》系列中的殺意波動也說不定呢。

那麼有新的BOSS就會有全新的裝備，究竟暗黑大騎士的本命武器「暗黑騎士之劍」是否存在，大騎士也是否有舊劍這種綠色裝備呢？答案馬上揭曉。

試煉之地是玩家接觸BOSS最多的場景，在這種模式下可以快速的累積紅武與擊殺率，所以暗黑大騎士也是其中出鏡率很高的BOSS，而根據老宅自己的親身實踐，多次暗黑大騎士後掉落的道具仍然是：大騎士之劍，因此並不存在暗黑大騎士之劍這種武器。

至於大騎士的舊劍，更是無稽之談，其最早源頭出自某些視頻愛好者手中，是PS出來的。如果覺得老宅說的沒有可信度，那麼咱還有第二種論證方法。那就是：地窖，作為統計遊戲中一切數據的檔案處，玩家可自行在武器列表中找尋這兩把武器的影子，其結果自然是沒有，而謠言也自然不攻自破。至於大家看到的圖片，當然也是假的， 圖1中老宅做的兩把武器還不夠逼真嗎？
