[摘要]相信很多小夥伴都是知道的，在排位賽之中，很多戰士是單挑能力強，團戰也是不弱的，而像橘右京這種，則是切人強勢，團戰一團糟的，那麼本期筆者就來說一下團戰強的英雄吧！

首先第一個就是我們的關羽了，關二爺在峽谷之中絕對是頂尖的存在，受到了很多小夥伴的喜愛，出場率的話還是挺不錯的，而團戰的時候他的大招真的是很強勢，用得好的話，一波就帶走敵人了！

第二個就是孫策了，這個戰士英雄在峽谷之中絕對是頂尖的存在了，大招開船，支援速度驚人，而且因為一技能和佩戴暈眩的緣故，其控制能力是真的強，一套下來，除了坦克誰都頂不住！

第三個就是李信了，這個戰士英雄在王者榮耀之中還算是很不錯的，光信團戰強，群體輸出不錯，暗信單挑能力強勢，並且一技能有霸體和續航，所以能抓死他的英雄很少存在！

最後一個則是呂布了，其不僅單挑能力強勢，團戰也是不錯的，大招一跳，敵方只能四處逃竄，分割戰場的能力不要太強，直接將敵方前後排給切割開來！對此，大家有何看法？歡迎討論！
