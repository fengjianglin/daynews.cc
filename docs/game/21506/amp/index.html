《蔚藍》各關卡隱藏通道位置 隱藏通道在哪

天天要聞
2019-12-19

《蔚藍》是一款像素風格的動作冒險類遊戲，地圖上有很多令人意外的隱藏通道不容易被玩家發現。下面請看由「meiwu」分享的《蔚藍》各關卡隱藏通道位置，一起來看看吧。

　　Celeste的關卡設計非常優秀，地圖上有很多令人意外的隱藏路徑，還有許多有趣的落腳點，下面就按關卡選挑一些難以發現地方圖文演示，每個類型演示一次，畢竟這種遊戲還是要自己探索更有樂趣。

第一章

　　像這種情況，只要用些心，還是很容易注意到的，牆面違和感很大，只要突進即可撞開，如圖這種類型只演示一遍。

　　但也有一種情況，單從牆面幾乎看不出來，我們就要從周圍的細節來判斷，比如下圖，這裡有個不太合理的落腳點，那麼旁邊就十有八九就可以撞開。

　　還有就是要注意觀察角落，Celeste的地圖中，不會無緣無故出現莫名其妙的東西，比如下圖左上角，只要上去就會有大收穫。

　　莫名其妙的「凸」或「凹」，很好判斷，如下圖，一般看見直接頂就行了。

　　像這種雖然隱藏的很好，但自己觀察，還是有很大差別的，遇到這種不要猶豫，直接撞，百分百有新路徑。

第二章

　　第二章其實沒什麼好說的，沒有新的特殊地形，跟著地圖跑就行了，基本都是這種。

第三章

　　這一章就會開始出現各種有趣的落腳點了，可能沒什麼用，但依然值得上去玩玩，如下面這兩張圖，遇到這種高台基本都是可以往上沖的，千萬不要錯過。

　　還有就是一些比較黑的地圖，很容易注意不到，這些地方一般都是捷徑，暗道之類的，除了可以吃到草莓，還有機會規避一小段很難走的路。

第四章

　　從這一章開始就很難通過凹凸，不規則等來判斷暗道了，很多時候都要親自走上去試試。

　　遇到河流也不能放過。

　　還要多注意牆面，比如下圖中，正確的路線是玩上，但右面牆卻有個裂縫，這一看就不對勁，所以必定要去撞一下。

第五~八章

　　後面的關卡節奏愈發緊湊，很多跳躍都涉及劇情，所以還是自己探索更好，基本前面幾招，就可以逛遍地圖了，無非就是跳撞，蹲撞的區別，當然也有幾種情況需要注意：

　　這種一上來身後沒有障礙物的，這時盡量玩後撞一下，很容易撞出新區域。

　　還有幾處劇情大跳，前幾章追你的那個人往上丟你，這時一定要注意周圍環境，先看看沒有可去的地方，因為一旦起跳就回不來了。
