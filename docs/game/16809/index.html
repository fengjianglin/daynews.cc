DNF：冒險團即將擴張至40級，額外290點四維加成，又要開始肝了

2019-12-16 76條評論 3,862次閱讀 91人點贊

​DNF：冒險團即將擴張至40級，額外290點四維加成，又要開始肝了！

以下為具體的內容詳情：

玩家們都知道，在上次DNF的95版本更新，策劃就曾對角色冒險團的上限等級進行了擴張，從原本的30級提升到了目前的35級，四維屬性的加成也從260提升到了275點，而眼看等到新的100版本又要來了（國服基本可實錘將在年底開放100版本），據了解，到時候角色冒險團的等級將再次進行擴張至40級，隨之四維屬性的加成也將隨著等級提升到290點，玩家們又要開始肝了。

據了解，等到新的100版本上線後，角色的冒險團上限等級將再次增加5級，提升至40級上限。而雖然40級冒險團的具體四維屬性加成暫時還未爆料，但按照25~30，30~35這樣每提升5級額外增加15點的四維來看，40級冒險團應該就是290點的四維加成。

40級冒險團的四維加成有290點，相比起來又能額外多增加15點的四維面板，到時候玩家們又要開始肝了。而這雖然對大多數玩家來說是一個福利，但對少數玩家而言，相信在現版本中還有很多像小寶一樣的少號黨，冒險團的等級連35級都還沒達到，這馬上又要出40級了，真的是感覺肝越來不夠用了。

當然，屆時隨之冒險團的更新改版，除了等級上限的擴張，四維屬性加成的提升只外，策劃還會在商店中加入各種全新的道具獎勵，比如90~94升級卷，cp護石/符文，100級新的神器裝備禮盒等，雖然基本都是一些蚊子腿，但有新獎勵總比沒有好，一個字「肝」就完事了。
