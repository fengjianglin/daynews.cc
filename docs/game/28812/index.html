lol德瑪西亞杯RNG首日賽程：小龍堡首秀迎戰YM，狼行單挑聖槍哥

2019-12-21 896條評論 2,189次閱讀 221人點贊

21號德瑪西亞杯正式打響，由於剛剛過轉會期，因此與之前不同的是大多數戰隊都準備派出新主力陣容來迎接挑戰。RNG戰隊Uzi由於傷病缺席，小虎、狼行、小明將帶領其餘幾名新加入的成員出擊，而比較受關注的打野小龍堡也將在此次德杯中首秀。

接下來讓我們一起看一看RNG在21號德杯首日的賽程安排吧！

RNG vs OMG 12：00-13：00

Smlz從SN重新轉會回到OMG，不禁令人感慨兜兜轉轉又回來了，但是這一次德杯Smlz的對手不是Uzi，而是原FW的Betty選手。Smlz能否通過數據再次證明自己，而Betty在RNG的首秀如果能壓制住Smlz，相信RNG的粉絲們肯定會對他刮目相看，將有力回應外界質疑。

小虎和冷少的中路對決也是非常令人期待，冷少也算是OMG發揮最穩定的一個點了。而小虎雖然在世界賽上表現平平，但是他天性樂觀心態良好，假如能把這次風波扛過去，相信一定會觸底反彈，重新找回虎大將軍的狀態。

RNG vs JDG 15：00-16：00

JDG在上中輔三個位置沒有做出人員調整，但AD位Imp選擇退役，Loken重新回歸，同時GRF打野Kanavi也來到了JDG。JDG的整體陣容較S8時僅打野位進行了更替，並且JDG在S9春季賽中也是表現良好代表LPL參加洲際賽。相對於RNG來說，JDG在陣容的磨合度方面應該會更好一些，不過在賽場上兩邊選手的發揮也是十分的關鍵。

RNG vs YM 18:00-19:00

這場比賽就有點意思了，小龍堡剛剛從PDD的YM戰隊轉會到RNG，而輔助小明此前也是YM戰隊產出的王牌選手。在此次德瑪西亞杯中，兩位隊員在面對上自己的老東家時，是會選擇重拳出擊還是會手下留情呢，讓我們拭目以待。

RNG vs LNG 20：00-21：00

最後一場RNG對戰LNG的比賽，上路狼行面對聖槍哥的較量肯定會十分精彩，狼行在此次世界賽上的表現讓粉絲十分欣慰，而狼行在面對聖槍哥時是會選擇抗壓，還是會選擇強勢出擊令人期待。而另外一邊，在LPL可以與The Shy一較高下的Flandre，相信此次對決一定會再次使用進攻性上單來為團隊爭取優勢。

另外小龍堡和小虎的中野組合在面對上XX+Maple時，究竟能產生什麼樣的化學反應，到底哪邊中單能夠幫助野區建立優勢尤為關鍵。

RNG德杯Uzi缺席，小虎能否扛起隊長大旗，帶著首秀的小龍堡愈戰愈勇，相信21號的比賽會給出答案。
