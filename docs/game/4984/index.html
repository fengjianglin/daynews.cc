摘要

LPL全明星賽在上周末落下帷幕，其中老IG和老WE的傳奇對決也吸引了不少人的關注，畢竟在傳奇對決之後雙方就曾掀起了隔空罵戰，直接將這場傳奇對決的節目效果拉滿。經過BO2的對決中，老WE和老IG最終以1:1的比分結束算是最圓滿的結局了。

在老WE和老IG這場對決之後，西卡在直播間中也是現場連線了笑笑，要知道笑笑在這一次全明星賽前一直不被大家看好，沒想到在賽場中卻有點超常發揮。西卡也是將彈幕上觀眾提出的疑問找笑笑一一答疑。

第一個問題就是為什麼老IG贏了第二局之後會過去和老WE握手，這一點確實有不少觀眾不解，畢竟雙方是戰成1:1的結果，一般主動過去握手的都是比賽獲勝方。笑笑解釋是裁判說的，第二局獲勝的要過去握手，如果第二把WE贏了他們也會找我們握手。西卡調侃原來是這樣，還以為是射可可提出的過去握手的呢!也就是說這一切都是官方事先安排的，所以大家也不要誤會從而帶節奏了。

接著西卡詢問笑笑為什麼在結束後和若風握手的時候鞠個90°的躬，笑笑直言當時對若風說「風隊我這算不算核心玩家」，隨後就鞠躬了，不得不說笑笑還是非常簡單粗暴的。隨後笑笑還談到在輸掉了第一把之後，第二把並沒有感到壓力，笑笑還表示第一把是娛樂的，不過旁邊的姿態馬上打臉笑笑，直言「娛樂個鎚子，娛樂」。

最後一個問題就是關於和下路微笑對線的看法，笑笑又一次開啟口嗨模式，直言微笑確實是強的，自己全方位認真還是可以跟微笑對線的，微笑確實太強了。不得不說笑笑這一波口嗨還是可以的，既承認了微笑實力很強，又暗示自己認真玩同樣實力很強，不得罪人的同時還能吹一波自己，這一波直接將西卡直播間節目效果提升到最高潮。也許是為了避免再次被問及一些尷尬的話題，笑笑果斷稱自己還有事情就先不說了。

總的來說，這一次的傳奇對決還是不錯的，雙方並沒有將之前的罵戰情緒帶到賽場上，尤其是老IG這邊，能夠和老WE戰平恐怕之前都沒想到。而笑笑在這場比賽中也證明了自己的實力，尤其是老司機的開車技術還是在線的。對於這一次西卡現場連線笑笑，大家有什麼看法呢?
