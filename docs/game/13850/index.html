大話西遊2平民大炮三件裝備屬性展示 搭配起來能否追趕仙器

2019-12-12 783條評論 4,480次閱讀 2909人點贊

​隨著對遊戲體驗要求的增強，大家對炮的要求也是越來越高了，當然希望秒的越多越好。通常情況下，有條件的話，肯定是會選擇使用4級或者5級神兵，如果經濟情況再允許一些，可能會選擇5階仙器或者6階仙器了。阿宅也經常能見到一些玩家自謙到，會拿著5階當任務號秒日常怪。今天阿宅就遇到這樣一位玩家，對傷害有著極品的追求，但他的武器既沒有選擇神兵，也沒有選擇仙器，而是選擇了一把十五級的武器。這把武器到底有什麼過人之處呢？一起來看看吧。

一把重鑄過的十五級武器，原始黃色屬性還是很平常的，忽視抗風26.6%，這個初始忽視並不算極品，而且沒有藍字屬性。但是綠色煉化還是讓人眼前一亮的，風系狂暴16%，強力克水29%，還有14點靈性，對於追求高靈性的炮來講，這個屬性是非常讓人心動的。開光屬性更是讓人驚嘆了，居然開光出了風系狂暴20.5%，這是多少5階6階仙器玩家都夢寐以求的開光屬性。

來看看項鏈，黃字底子靈性+27，煉化綠字克水21，風系狂暴16，再加上藍字有2.3的忽視抗風屬性，算是比較極品的項鏈了。

護身符也不簡單，992品質，黃字強克13.8，綠字忽視抗風5.5，克水20.8，風系狂暴幾率5.5，也算是比較極品的屬性了。

再加上寶石的屬性，這三件套下來，傷害不比神兵仙器差。怪不得這位玩家寧願選擇拿著15級武器，也不用仙器神兵呢。大家覺得，這把武器，能抵得上幾階呢？
