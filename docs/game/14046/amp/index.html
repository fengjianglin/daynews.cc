【只狼·薔薇之劍各階段及技能介紹】

地獄難度被動:修羅道：只狼使用修羅之力攻擊對手心智，造成持續傷害。

［三個階段］

第一個階段：

血條大於20管，因為血多，所以只狼喜歡玩點B格高的，主要技能有染色臟團、絕技·葦名十字斬EX、影流分身。招式華麗，但都沒什麼o用。

[染色臟團]

為了追求B格，只狼將自己所有的灰燼團統統染成了粉色，會先傲慢的告訴你「我要臟你了！」，然後扔出玫瑰色菊花臟團，形成aoe傷害，並且出現酷炫的特效。

應對方法：出現後人物走兩步離開菊花建模後摁下閃避，很好躲。就是讓你發動時停用的。

絕技·葦名十字斬EX：有三個動作，第一個動作抬手然後進行一次快速的X斬擊（這個動作很關鍵，是區分影流之主的關鍵動作）。第二個動作左收劍，向下刺擊。第三個動作順勢X斬擊。

應對方法：這一招判定範圍相當窄，看到第一個動作手放下來時側向閃避然後輸出到這一招結束就行了，雖然名字很酷炫但很好應對。

影流分身：只狼在砍殺無數小怪後偶然間領悟的技能，可惜並沒有成群的小怪拿來練手，用起來非常生疏。兩個動作，第一個動作雙手抬起，此時只狼身上會飄起櫻色螺旋光帶。第二個動作蹲下釋放12個影流分身，據說這招練至化境便可將所有能量集中在兩個分身上，從而對敵人一擊致命。

應對方法：第一個動作時間較長大概有2s的樣子，砍他，輸出，等砍得他低頭之後，隨便閃個什麼位置就行然後繼續輸出。由於沒有勤於練習這招，所以華而不實，並沒有發揮出應有的威力，可見其相當之蔡。

第二個階段：

血條降到20以下，只狼發動第一次回生，進入第二階段，由於死了一次，只狼開始氣急敗壞，決定使用吹火筒和拜淚進行攻擊，但還是有些放不下自己的B格。招式依然華麗，而且攻擊硬直大，抬手明顯，恰好這時」The World「的CD剛好轉完，隨便躲一個技能就可以砍得他馬都不認識，主要技能：炎上EX、秘傳·不死斬EX。

炎上EX：如果你離只狼較遠會大概率發動這一技能，只狼將吹火筒插入地下，在你腳下形成一片火區，短暫延遲後遠程炎上造成傷害，發動過程中會出現三次火圈。連續使用兩次該技能進入失勢狀態。

應對方法：靠走是走不出去的，只能在火圈出現後走一步接閃避，這樣就能觸發完美閃避。

秘傳·不死斬EX：不死斬真tm帥啊，可惜打不到人。但是沒關係，秘傳·不死斬EX 可以。如果你離只狼較近（正前方大概90°範圍內，距離3-5個身位）大概率會發動這一技能。第一個動作，雙手平舉佩劍，將不死斬中的黃色瘴氣激發出來。第二個動作發動三次大範圍斬擊。第一次斬擊判定範圍是正前方180°，第二次斬擊判定範圍是正前方270°，第三次斬擊是360°無死角攻擊，豈可修！卑鄙的外鄉人。使用該技能結束後必定進入失勢狀態。

應對方法：抬手很明顯，所以很好躲第一次斬擊閃避到他身後，第二次斬擊前會先面對你，貼身走到身後即可，第三次斬擊只能用閃避，因為動作幅度太大，打完這一套後只狼會因失勢而閃到老腰，蹲下歇會兒（大概5s多一點）。你懂得接下來要幹什麼吧，正義的葦名人？

第三個階段，血條降到10以下，只狼會發動第二次回生，浮空1s並穿模，因為馬上就要死了，在血皮之力的加持下，管家變得極為謹慎使用歸佛回到佛前潛心苦練，因為兩個世界時間流逝不一樣所以我們只能看到他浮空時穿模了一下，並沒有看到他使用了歸佛，ready prefect之後的只狼會擺出非常中二的姿勢，改用實用的秘傳葦名流劍法配合卑鄙的月隱糖進行攻擊。

特別注意:

地獄難度下，只會發動磕葯（偽）技能進入第三階段。

磕葯（偽）: 第二階段中，當進入失勢狀態，受到大量傷害且血條降至10以下時，會立刻拉開距離釋放一次炎上EX，由於玩家專心躲避技能降低輸出，得以有時間使用逃課領主:風靈月影賜予他的秘葯:數字鍵 1－無限生命。回復滿血並進入第三階段。不過由於進入下一階段血量鎖定在10管且會清除自身所有buff，由此可見即使是踏入了修羅道的也只是一個只會逃課的dd，所以請在思想上鄙視他。

主要技能：秘傳·飛度漩渦雲·月隱、一字斬·三連刺擊、秘傳·龍閃、絕技·巨型忍者突刺。

秘傳·飛渡漩渦雲·月隱：只狼以迅雷不及掩耳盜鈴兒響叮噹之勢吃下月隱糖，同時以及其詭異的姿勢遁入虛空（？），準備偷襲，但還是沒忍住你頭上留下了標記，在暗處伺機發動秘傳·飛渡漩渦雲，但還由於太蔡只能打中一次。不過好在可以連續發動增加了容錯率。

應對方法：只狼月隱之後，千萬千萬千萬不要攻擊，攻擊一次便會充能一次標記條，形成所謂的「快慢刀」，另外如果頭上有標記除非沒閃避千萬不要換人，因為換人標記會消失但招式依然不會停下。只狼從視野中消失並且頭上出現標誌時停止攻擊，白色標誌會從中心逐漸變大直到填滿標識框。填滿瞬間狼會從陰影中降臨並發動卑鄙的偷襲，在標誌填補完最後一圈的同時進行閃避即可完美躲過並發動時停將其吊在中打，雖然很有效，但我們並不是楊過。

一字斬·三連刺擊：由一字斬衍生而來，只狼經過極短時間的蓄力後（不是很明顯）發動三次刺擊，雖然是刺擊，不過此技能與一字斬一樣，削韌極強，被擊中一次便會發生很大程度的硬直，然後被迫吃下一字斬·四五六七八連。缺點是出招也同樣耿直，被打中換人+閃避可解。

應對方法：長時間蓄力和耿直到家的攻擊方式是致命缺陷，閃避到身側然後給他上一課，逃課禁止×。

秘傳·龍閃：當你因為只狼太菜想用遠程羞辱他，或者你太菜被一字斬·四五六七八連時，只狼很大可能使用的一招必殺技，具體操作就是先左手畫一道彩虹（只是蓄力沒有傷害，刀光是天狼星的刀光）然後右手甩條龍，雖然很耿直，但有點難躲，傷害還高，因為他實在是太快了……並且在你倒地時如果仍舊離太遠，他便會使用派生技能:絕技·巨型忍者突刺。

應對方法，抬手有點快而且不是很明顯，能反應過來就抬手閃避，不能就瞎G2亂閃或者凹他不用這招吧。

絕技·巨型忍者突刺：當你被龍閃打中倒地的時候，或者躲過龍閃但並沒有拉近距離，而是依舊傻fufu地在遠處嘲諷時，只便會使用這招教你做人。依舊兩個動作:先擺出極其中二的姿勢迷惑對手，當你好奇他在幹什麼的時候，只會以光速進行偷襲，傷害判定的範圍有一個火車那麼長，基本躲不開。

應對方法:如果看見三階段過程中只擺出極其中二的姿勢，請不要猶豫默數「1」然後閃避，運氣好說不定可以閃過。如果是被龍閃擊倒在地的情況下，或者被一字斬打中硬直的情況下，看到只擺出極其中二的姿勢，不要猶豫，雙手拿開，讓你的角色享受被二連擊的快感。

【打法詳解】

攻略一：

一階段：

首先我們如果正常走過去，只狼100%億會試圖用臟團臟你，不用管，走兩步向前閃避到他身邊，來一套qte連招（先a幾下，等到超算空間時間剛好結束或快要結束，再觸發三消通過意識被動進入第二次超算空間，a幾下繼續三消然後aaa…）大概輸出5s左右（結束時請務必處於只狼身後以便看清後續招式順便多砍兩刀）。然後只狼會轉身，隨機變招，左手背後為絕技·葦名十字斬EX，雙手抬起為影流分身，兩技能會先後使用，順序隨機，冷靜應變即可。戰力剛好夠的話放完三個技能剛好進入下一階段，戰力不夠再來一輪隨機應變。如果剛開始直接閃避到只懷裡，他會直接給你一個X斬接影流分身。

二階段：

二階段剛開始只狼一定會和你拉開距離，且大概率發動炎上EX技能，靠近的同時不要試圖走出火圈，正常走是走不出去的，一定要閃避，時間剛好過12s的話，發動The World，給他一套連擊，很大可能直接進入第三階段。

地獄難度下：

需要注意的就是二階段進三階段的時機，只狼放完兩次炎上EX後閃到腰會蹲下休息，放完不死斬EX後也會蹲下休息，這時很大輸出空間，是灌死只狼的最佳時機，如果死不了，只狼起身後會先和你拉開很大距離，再次釋放一次炎上EX，隨後開啟磕葯（偽），進入第三階段並恢復到10條血量，這段時間一定要找機會近身，否則只狼進入第三階段之後必定會釋放月隱，龍閃，巨型忍者突刺中的一個，處理起來會很麻煩。

三階段：

這一階段是最難凹的，最好的流程是只狼一直使用一字斬·三連刺擊和秘傳·龍閃（最好也不要有這個技能，因為出招特別快），然後我們閃到側面抓住機會攻擊，配合大招快速擊殺。如果不幸被只狼用出了秘傳·飛度漩渦雲·月隱，此時也不要慌，如果血條很少，切換輔助角色，等待標記填滿後閃避觸發時停，將只狼吊在空中qte擊殺。如果血條很多，那就視時間決定繼續打還是右上角，畢竟如果凹分的話時間也算在結算分內，並且比重更大。

地獄難度下：

1、只狼月隱之後發動飛渡漩渦雲時，一定、一定、一定不要攻擊，攻擊一次便會充能一次標記條，形成所謂的「快慢刀」，另外如果頭上有標記除非沒有閃避否則千萬不要換人，因為換人會消失但招式依然不會停下，這很可能讓工具人暴斃。

2、千萬不要和只狼拉開距離，否則會大概率釋放龍閃＋巨型忍者突刺二連combo，中了龍閃之後如果只狼沒有第一時間補刀，而是站在原地擺出非常中二的姿勢，就必定是在蓄力巨型忍者突刺，此時我們要做的就是停下雙手，等待的連招過去，因為此時人物倒地，是沒法換人的，而且人物剛剛起身過了無敵判定突刺的傷害判定就開始了，請繼續安詳地躺下，或者起身換個人安詳地躺下。

最後，走到只狼右邊，進行嘲諷。
