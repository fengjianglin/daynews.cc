Switch獨立遊戲日本直面會匯總 創意帶來新機會

天天要聞
2019-12-11

12月11日今天，任天堂舉行了新一輪獨立遊戲直面會 Indie World，通過這次直播，介紹了2020年即將在Switch平台發售的數款獨立遊戲新作，而任天堂日本也稍後公開了日本版直面會，與之前發布的遊戲的有所差別，一起來了解下。

·每年任天堂都會為登陸旗下遊戲機的精品獨立遊戲舉行總結大會，今年的《Indie World 2019》同樣帶來更多新老遊戲，而每一個獨立遊戲都是創意之所在，或許都為日後的大作提供素材。完整視頻內容

天穗的長命草姬

本作自2018年公布以來，多次跳票，如今再次亮相，預定2020年春發售。主人公是掌管「收穫」的女神「長命草」，更是一位可愛小蘿莉。女神長命草將於本作中一邊從事農耕生活，沒事去探索探索迷宮打打魔物。

what the golf？

預定今冬上線。本作將是一款掛著「高爾夫」之名的非高爾夫小遊戲合集，據稱集合了多達200種以上的各種路線，玩法多樣。

風來之國

預定2020年上線。來自心動網路負責發行的國產像素冒險遊戲，幻想風格的畫面帶來獨特感覺，玩家將指引主角們逃出地下小鎮，開始在這個被稱作「末日的時代」的冒險。

犬犬

預定2020年春季上線。講述了一隻（2隻？）雙頭狗拖著長長的身體冒險的故事，而這具身體極有彈性，關鍵時刻能化解更多危機。

Synaptic Drive

預定2020年春季上線。是一款可以自己打造機甲戰士展開火熱槍戰的遊戲，支持多人混戰，動感火爆的音樂像極了某些大型多人競技槍戰遊戲。

Killer Queen Black

預定2020年夏上線。可以多人混戰的遊戲，玩家扮演女王和隨從，搶奪關卡中的藍莓，同時也可以互相攻擊，想要勝利並不簡單。

奧日與黑暗深林決定版

已經上線。《奧日與黑暗森林》一款動作冒險類遊戲，畫面精美，音樂動人，故事情節生動感人。講述了圍繞著尼布爾山（Nibel）的森林中有個守護神「精靈樹」展開的感人故事。

去月球

2020年1月16日上線發售。《去月球》這是一款使用RPG製作大師製作的視覺小說。雖然是個像素畫微型遊戲，流程只有幾個小時並且沒有真正的戰鬥；但是本作憑藉精美細膩的演出效果，和感人的故事獲得了許多好評。

Coffee Talk

預定2020年1月30日上線，類似於人氣日劇《深夜食堂》的情感系遊戲，玩家可以和光顧咖啡廳的的各種顧客一起分享不一樣的故事。

螃蟹大戰

預定2020年夏季上線。公布之時就備受關注的奇葩遊戲，玩家將可以扮演各類張牙舞爪的甲殼類動物，包括螃蟹、龍蝦等等；經過巨大化後在餐桌、樓宇之間大打出爪。從宣傳視頻中還可以發現玩家能選擇各種武器，甚至還有激光劍和鑽頭！

守墓人

預定今冬上線Switch。《守墓人》（Graveyardkeeper）已經登陸PC和Xbox平台，是一款像素風模擬經營遊戲，同時兼備了沙盒探險和地牢戰鬥玩法。

ABZU

2020年2月上線。 玩家可以暢遊神秘的海洋，與魚兒作伴，欣賞美景。

Crystal Clash

預定2020年6月上線。可以對戰的益智消除遊戲。

孤山速降

2020年春上線。遊戲中玩家可以騎著自行車從山上速降，體驗各種地形和欣賞優美的景色。當然，最快最安全到達山底的才是真正的秋名山自行車神。

Baba is you

2019年12月16日上線。類似於倉庫番的解謎遊戲，搭載更多玩法，需要動一番腦筋。

芬奇家的古怪大宅

又名《艾迪芬奇的記憶》，已經上線。玩家將探索神秘的芬奇家的古怪大宅，發生了各種稀奇古怪的故事。

Tools up!

已經上線。玩家既可以單人奮戰，也可以4人合作，一起來裝修收拾房子，各種並不簡單的挑戰將影響你的最終成果。

陷陣之志

2020年春上線。陷陣之志（Into the Breach）是一款回合制策略遊戲，玩家控制強大的攻城機器，在隨機生成的棋盤地圖上與敵人交鋒。遊戲設定在外星人入侵的戰火地球，殘存的人類文明要與潛伏在地下的外星生物交戰。
