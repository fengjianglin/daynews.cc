LOL中船長吃的橘子到底是什麼品種的橘子，效果這麼強

2019-12-11 2,011條評論 4,290次閱讀 3362人點贊

但凡玩《英雄聯盟》的玩家，一定都對船長這個英雄不陌生，雖然幾經改動，但是在上單的位置上仍然有著不可取代的作用，尤其是帶了不滅之握的船長，更是能把人噁心死，再配合大招的全圖支援能力，船長的作用還是有目共睹。

​船長和大部分英雄一樣，也經歷過改版重做，但是無論怎麼改，船長的一個神技能被保留了下來，那就是當你各種控制扔給船長的時候，都會看到船長不慌不忙從褲兜里掏出一個橘子，一口吃下去，一切就當無事發生過。

這種感覺，就好像小時候看的《大力水手》動畫片，沒有什麼是一罐菠菜不能解決的，如果有，那就是兩罐。

​那麼，船長到底是吃了什麼品種的橘子，才能有這麼好的效果呢，今天小球球來不正經的研究一下。

首先我們來看看船長的W技能，也就是吃橘子的技能，技能全稱是「壞血病療法」。

​於是小球球上網查了一下關於「壞血病」的資料，發現「壞血病」主要是因為缺乏維生素C導致的，而且以海員患病的居多。

看得出來這個技能還蠻貼切船長的。

​我們再來看看船長的背景故事，發現船長的行動軌跡主要集中於艾歐尼亞和比爾吉沃特一帶，當然也曾涉足諾克薩斯偷走了斯維因的「利維坦」號戰艦。

​通過這張地圖就能發現，其實船長基本就在守望者之海遊盪，守望者之海位於瓦洛蘭大陸以東區域，按照傳統地理學，這裡應該屬於亞熱帶季風氣候，而隸屬亞熱帶水果和維C沾邊的，其實檸檬和橘子都算。

再細細品一下，地圖的位置其實偏北而非偏南，但偏北的話，橘子並不算真正的量產，畢竟好的橘子品種，大都來自南方。

有詩為證：橘生淮南則為橘，橘生淮北則為枳。

​所以，作為長期在這一領域遊盪的船長而言，顯然不會攜帶大量的味道並不怎麼好的橘子，而且橘子糖分偏多，海上航行不久就容易造成腐爛，這也並非船員首選。

​那麼，英雄聯盟中船長吃下去像橘子的水果是什麼呢？真相只有一個，那就檸檬。

檸檬同樣屬於富含大量維C的水果，甚至比起橘子來要高數倍，同時，相較於橘子，檸檬的儲存時間要更長，所以，一般經常出海的人，都會經常攜帶檸檬。

所以，從地理位置，邏輯以及儲存時間來看，船長吃的應該是檸檬而非橘子。

由此，其實也可以證明為什麼船長的W技能在《英雄聯盟》里堪稱神技，因為船長就是一個愛吃檸檬的檸檬精，而檸檬精無所不能。
