絕地求生：微博杯第五日戰罷 RNG升至總積分第一 XDD猛如虎

2019年12月5日，微博杯小組賽階段第五日比賽結束，對陣雙方則是A組和D組。在16支戰隊鏖戰六場之後，積分榜也水落石出。RNG狀態火熱，單日拿到71分，一舉將自己推到總積分榜第一，同時AG也不甘示弱以54分拿到單日第二。而知名主播XDD成年後的第一次大型杯賽似乎打得格外好。這裡感謝競技寶JJBONE的資訊支持。

我們來看看積分榜的大致情況，首先是單日積分榜。RNG以52個淘汰71分的爆炸數據豪取當日積分榜第一，AG以末局兩連雞的強勢表現拿到54分位居第二。VC不甘示弱，拿到53分位居當日第三，雖然新援VV的數據並不算太好。後續依次是NEXT、BA和YTG，不過第六名的分數似乎有些低，今天的比賽可能頭部隊伍吃分比較多。

在5天的比賽結束之後，我們也對總積分榜的情況做一個總結。RNG單日71分的表現確實兩眼，目前以總分168分位居第一。而第二名的ARK則更恐怖，ARK目前還有一輪沒打但已經有130分在手了，或許小組賽第一的名號不一定穩呀RNG。第三是YTG，小組賽依然強勢的一支隊伍，破綻樓的參與人之一嘛（手動狗頭）。

而剛成”鯰”的XDD第一次杯賽打得怎麼樣呢？兩個字，”爆炸”。總淘汰數上XDD目前位居全場第一，而總傷害量第一也是他，場均傷害甚至超過了400。或許曾有人說XDD開掛，也有人嘲笑他到了正式比賽就不行，但就這幾日的表現來說，XDD確實值得RNG的等待。當日來日方長，小編也不吹太滿了，大家看看就好。

次局：EDG5殺吃雞，NEXT淘汰17人猛如虎

第三局：BA9殺吃雞，EDG決賽圈一碰就碎

第四局：SSS一殺吃雞驚呆全場，獨狼苟雞確實狠

第五局：AG玄學圈8殺吃雞

第六局：AG繼續發威，7殺再次吃雞

最後我們來前瞻一下12月6日BC組的最終決戰。BC組這一對戰是本次微博杯小組賽的最後一場，對於排名分靠近16的戰隊來說無疑非常關鍵。目前11到16名的隊伍大概在88到95分之間，最後一日的比賽瞬息萬變誰也說不清楚。下面來看看對戰陣容。

第一個映入眼帘的戰隊就有點尷尬，他就是PGC還打得不錯的4AM。不過目前4AM只有43分，全場倒數第二。可能是體驗了0延遲環境後再回來打線下會不適應？具體原因我們也不知道。而IFTY和ARK基本是穩了，一個111分一個130分，完全可以沖著高分前進。SMG目前56分，要加把油了，還有希望。
