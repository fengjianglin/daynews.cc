我的世界：當MC不再自由，加上這幾個限制條件後，你多久能通關？

2019-12-14 1,121條評論 2,549次閱讀 1528人點贊

哈嘍大家好，我是地瓜！大家都知道，我的世界的自由程度相當的高，這也是玩家們非常喜歡的它的原因，那麼咱們「作妖」一次，當MC不再自由，加上了幾個限制條件之後，你覺得你多久能通關呢？這個通關，指的是打敗末影龍！

那麼為了不讓遊戲太難，我們所說的這幾個條件並不是疊加的，而是任選其一，那麼我們一起看一下吧！

條件一：不允許使用方向鍵！

上來之後，地瓜就提出了個非常嚴苛的限制條件。這個現實條件一出，就等於把史蒂夫的雙足給困住了！在現實生活裡面，如果你不能走路，那麼你還能做什麼事情呢？可能也有很多，比如玩電腦！但是在遊戲裡面，如果不能行走，那麼你基本上就被限制死了！不過MC也有例外的，因為總是有大神的！

國外一位玩家，就是用了這個條件，然後一直在玩MC，結果他還通關了，那麼他是怎麼做到的呢？其實稍微動一下腦子，能夠讓玩家移動的有很多，比如在前期，這位大神用的船，還有水方塊，慢慢的，他開始騎豬了，到了後來，他還用了末影珍珠，這都是非常的厲害的，而且這位大神還很快的通關了！沒錯，靠著不用移動鍵，成功的殺掉了末影龍！

條件二：不允許拐彎，只能直行！

在遊戲裡面，史蒂夫想要不斷的成長，那麼就需要不斷的東一榔頭，西一棒槌地去挖掘自己能夠用到的物品，合成道具，但是如果只允許你直行的話，那麼將會錯過很多東西！

比如剛好你左手邊有一個村莊，右手邊有一個遺迹，但是只能直行啊，這就錯過了很多的東西！而且最為關鍵的是，想要通關，你必須找到要塞，可是如果要塞在你旁邊呢？真的非常的難受啊！

條件三：禁止採集物品！

什麼？那我還玩什麼呢？不讓採集物品，我連殭屍都打不過啊！那麼，如果不允許史蒂夫採集，這個遊戲就沒有辦法玩了嗎？其實並不是如此！

因為我們可以借力打力，使用外力！所謂的外力，首先就是我們可以不斷的去挑釁苦力怕，這樣我們讓他爆炸！另外我們可以在一些遺迹裡面獲得一些物品，比如村莊的獎勵箱裡面就有黑曜石，還有一些遺迹裡面的物品等！

另外一方面就是，能夠接受這個挑戰的，肯定不會是萌新，因為萌新上來就會直接放棄，不會玩到通關的！

條件四：不能產生傷害！

大家都知道，在遊戲中，會有很多的敵對生物，如果史蒂夫不殺死它們，就會有很多的危險！但是不能產生傷害的話，也就是史蒂夫不能攻擊了！這可怎麼辦呢？

其實有個「漏洞」，那就是只要不是史蒂夫直接產生的傷害，就不算史蒂夫攻擊了！比如用床炸末影龍、用岩漿燒死怪物等等，想必之前的挑戰，這個還是比較簡單的！

地瓜都說了，這都是一些「作妖」的挑戰，如果以上限制條件任意一個讓你挑戰，你覺得你能通關嗎？歡迎留言分享！

我是地瓜，喜歡的可以關注哦！
