懷舊服：為何獅心是聯盟大服？——從伺服器名稱談陣營和種族徽記

2019-12-13 525條評論 1,676次閱讀 119人點贊

獅心，聯盟大服，部落基本絕跡。當然也不排除小範圍內的小團隊還在苦苦支撐，而且好多真漢子真男人正在去獅心服部落的路上，請耐心等待支援^_^；獅心成為聯盟大服的原因讓我想了好一會，又查資料才最終確定，看來我不是聯盟的真愛啊，我是在弓服倫魯迪洛爾玩聯盟,在火錘伺服器玩部落，兩邊都可以體驗，兩個伺服器基本都不排隊人又很滿陣營也基本平衡，很安逸。

聯盟LOGO自帶獅頭盾牌，象徵的是第二次獸人戰爭中的人類領袖、艾澤拉斯雄獅安度因·洛薩公爵以及暴風城在聯盟中的核心地位；盾牌兩側的獅鷲和角鷹獸都是聯盟專屬坐騎；上邊直立的鐵鎚和上面的鐵砧標誌象徵著擁有高超鑄造工藝的矮人種族和鐵爐堡；下方中間的齒輪代表擅長工程學的侏儒種族；交叉的弓箭則是暗夜精靈族的象徵。因此無論是從艾澤拉斯雄獅還是瓦里安國王的獅頭肩甲，還有暴風城的獅鷲，都會讓人把獅心和聯盟聯繫在一起！而實際上魔獸世界懷舊服中目前只有一個鍛造製作的頭盔名為獅心，並沒有陣營色彩。

說到聯盟自然也要提一下部落的標識。跟聯盟的高大上、金屬質感、充滿藝術美感比起來，就很粗獷豪邁了，我們來看看其中的元素：中間最醒目的部落標誌如盾牌一般的形狀，盾牌的四周伸出N顆獠牙，下面再掛上幾縷羽毛；上面巨魔族的圖騰看起來很像嗜血的圖標，兩邊再伸出兩根代表牛頭人的牛角，其中一根還被砍下半截，牛角上又掛了好多羽毛；盾牌背後插著六根雞毛令箭，哦不對是六根代表被遺忘者的骨頭並用鐵鏈串聯；骨頭後面一破損的圓盾，整體造型看上去就像長了好多羽毛式鬍鬚的一個大牛頭！而且獸族的象徵物很不明顯，是獸牙？是圓盾？我聯想不到獸人。整個標誌原始、野蠻卻也不失威武大氣。

還有一些標誌代表一個種族的，包括魔獸爭霸3中的四大種族標誌都是精心設計的傑作，值得我們去玩味和欣賞、解析。下圖中第一排第三個是WAR3冰封王座中不死族標誌，一把神器霜之哀傷為骨架，搭配交叉成十字的兩把戰錘以及一根串著骷髏頭的長矛，統統被冰封住，這不正是象徵著霜之哀傷俘獲了人類的靈魂，造就亡靈天災的首領——巫妖王么？

不死族標誌後面一個則是「大地母親忽悠著你」，即牛頭人一族；第二排第一個是WAR3中人族的標誌，劍、盾、錘的組合簡單明快、設計精美；第四個暗夜精靈的標誌也很招人喜歡：貓頭鷹、月刃、弓箭的組合讓人聯想到優雅這個詞；第三排第一個代表血精靈的血之標含義比較模糊，一眼看上去我只能聯想到銀月城的建築風格，看不出其他含義；第三個代表暗矛巨魔部族的標誌不錯，能讓我們聯想到電影中看到的原始食人部落村口基本都掛著類似風格的東西；最後一個代表被遺忘者的痛苦之標，破碎的人臉以及被箭洞穿我們能理解，但背景的鷹盾是什麼含義呢，哪位大神評論區給解釋下？

欣賞完種族的，再來點職業的。你看到了什麼？上圖中仔細看你會發現術士惡魔的雙翼，被腐蝕而成的骷髏；獵人的虎狼熊豹鳥等寵物；法師的奧術光輝、握著寶珠的龍爪法杖和法典；盜賊那陰暗中的雙眼和陰森的雙匕；戰士那掛滿倒刺的盔甲和盾牌；薩滿的幽靈狼和各種神棍；牧師的祈福和天使之翼以及下方代表暗影的骷髏；聖騎士的各種全是鎚子！當然也少不了盾牌和保護祝福；德魯伊的野性印記以及LM、BL兩種獵豹形態，哪個最漂亮？

這是死騎的職業徽記

說起獅心，相信一些小夥伴還會聯想到一個遊戲——史克威爾出品的FF8，最終幻想八，其中的男主角Squall的武器正是一把帥到掉渣的Lionheart——藍色水晶質感的槍刃！而男主的最終終結技也叫做Lion Heart，是我在RPG遊戲中見過最華麗的大招，沒有之一！沒玩過的小夥伴一定要去完整的體驗一把！
