慎入，前方高能，烏拉爾山的恐怖，根據真實事件改編的恐怖遊戲

2019-12-11 695條評論 1,700次閱讀 777人點贊

#遊戲趣聞##遊戲圈那些事兒##恐怖遊戲#

恐怖故事、恐怖電影大家都聽過或者是看過，那麼恐怖遊戲大家有沒有接觸過呢，尤其是那些根據真實的事件改編的遊戲大家都知道嗎，下面為大家盤點一下，希望大家喜歡！！(堅持看到最後有小活動呦)

《回家：精神錯亂》

它是由印尼開發商製作的一款心理恐怖冒險遊戲。該遊戲主要取自印尼「都市傳說」製作，可以說相當真實了。

該遊戲主要講述了一個貧窮青年為了獲取榮華富貴，然後在婆羅洲島上進行了致富儀式Pesuglhan。在儀式上他將自己的靈魂獻給了惡魔，通過這種方式他獲取了大量的財富，但是幸福的生活並沒有降臨，反而產生許多怪事。玩家將以第一視角來經歷這些恐怖故事，遭遇各種怪物，並直面心中的惡魔，僅看遊戲預告就足以嚇出人一身冷汗。

這種經典的恐怖套路可以說是百試不爽，配合著製作組所營造出的黑暗氛圍，黑暗燈光渲染出壓抑的氣氛，的確能感受出它是一款相當不錯的恐怖遊戲。無論是從環境還是到怪物，都給玩家帶來了不小的心理壓。

《kholat》——第一人稱恐怖遊戲《烏拉爾山》

在1956年的時候，在烏拉爾山脈發生了這樣一件事情。9位滑雪登山者一同去挑戰「死亡之山」。最後有八個人死亡，因為其中有一個登山者身體不舒服沒有參加登山才逃過了一劫。最小年紀的僅僅是21歲，有的還只是學生。救援隊伍在半個月之後發現了一個被遺棄的帳篷。奇怪的是他們發現了一些現象不符合邏輯，他們陸續都搜索到了各個登山隊員的屍體，但是死亡的方式都不一樣。

有的隊員是死於凍死，有的隊員的屍體有被燒傷的痕迹，身邊都是參差不齊的樹枝都是燒焦過的。還有幾個更是離奇，頭蓋骨碎裂、肋骨斷裂。從這些死亡特別離奇的走過的路上發現有很多他們遺棄的物品丟在了地上，最離奇的是一個女生沒了舌頭，但是全身都完好無損。

雖然場面聽起來是很慘，看起來是跟什麼搏鬥過才會致死。但是最奇怪的一點就是他們雖然都死的很慘，看起來也不只是凍死的。但是就是沒有找到證據可以證明，有其他人來過這裡和他們搏鬥。丟掉舌頭這一點就可以完全確定，這不僅僅是自然死亡那麼簡單。

而這款遊戲《KHOLAT》就是改編這個故事的，「你」在遊戲裡面扮演的其中一個的登山者。在遊戲過程中，要通過探索和看劇情。去找到一些物件去幫助你通關，和基本的恐怖遊戲都是差不多一樣的套路。但是恐怖遊戲都會著重於遊戲的氣氛渲染。

《無盡夢魘》

據說這個遊戲的創意來自開發者 Matt Gilgenbach 的真實體驗，因為他曾經患過強迫症和抑鬱症。Gilgenbach一直在和自己的強迫性神經官能症和各種壓力、沮喪做著鬥爭。結果就是，Gilgenbach有非常多黑暗的幻想，所以他決定把他們製作成一款遊戲，也就是《無盡夢魘》。於是這款遊戲里充滿了令人驚悚的另類心理描繪。遊戲的畫面是黑白線條打造，主體色調是黑白的，但是也有彩色的東西，那些都是可以互動的，比如門，部分畫 ，柜子等。猩紅的血液在黑白的氛圍中顯得格外刺眼，而且遊戲中的音效十分壓抑。此外遊戲貌似有3種結局，感興趣的朋友親自試試吧。

無盡夢魘是一款以2D黑白風畫面為主的冒險驚悚解謎單機遊戲。畫面是遊戲的一大特色，其中不乏腥紅鮮血乍現的多個場景，給人在視覺上造成很強的衝擊力。整個遊戲的基調也是異常黑暗詭異，背景音樂設計的也十分壓抑恐怖，玩上手的感覺令人倍感氣氛緊張沉重。

無盡夢魘的劇情也是這部作品最引發聯想為人稱道的遊戲特點。故事講述了一個患有心理疾病的男子，時常會在噩夢中驚醒，發現自己身處在一個黑暗恐怖的大廈中，各種恐怖的現象以及恐怖的怪物都不時出現。但這一切並非完全癔病和幻想，因為其中的角色人物事件物品，都是有名字、有來歷的，玩家要做的就是區分出哪些是真實的，哪些根本就不存在。也就是分清哪些是真實的，哪些是夢魘。
遊戲中會遇到諸多危險情和兇惡怪物，但遊戲並不主打打鬥元素，基本操作和遊戲性是解謎冒險，通過潛行避開危險，探索和潛行是該遊戲的主要玩法。

《瘦長鬼降臨》

2013年這款遊戲正式上線，在遊戲中你將不停地躲避著一個名為瘦長鬼的可怕生物的追擊，玩過這款遊戲的玩家都知道，那種無時無刻都感覺有一個人盯著自己，隨時準備逃脫的感覺真的不太好，瘦長鬼可能在國內知道的人並不多，但是它在國外知名程度達到了家喻戶曉。

他又瘦又高，並且常年穿著西服，這個生物身體後面長滿了觸手，並且你看不清的他的臉，或者說他就沒有臉，頭上並沒有五官，並且喜歡去尋找小孩子，每當有他出現過的地方都會有小孩子離奇失蹤，並且沒有人知道他的動機,也沒有任何一個目擊者活了下來，只知道一旦被他盯上就會出現失眠、噩夢、咳血等特徵，想一想當你玩完這款遊戲關閉了電腦會不會有什麼東西正在盯著你呢。

《斯蓋爾女僕》

這款遊戲是根據伊麗莎白·威廉姆身上的真實事件改編的一款遊戲，故事發生在1898年的斯蓋爾莊園中，這是一座顯示真正存在英國威爾士南部小鎮布里真德的住宅，貌似至今還是存在，並且是英國鬧鬼最凶的一處住宅，有著一些非常詭異的傳說，但是我們的男主在這個房子中遇難，女主踏上了尋找男主的不歸路。

深夜中去尋找，不過他發現房間中的燈總是忽明忽暗，並且自己帶的火把也是飄曳不定，在一片漆黑的莊園中憑藉著僅存的一縷光明去尋找，突然房間的燈全部熄滅，火把也不再燃燒，所有事物都憑空消失，就像從來沒有發生過一樣，這款遊戲還沒有上線，不過僅憑它的宣傳已經讓眾玩家毛骨悚然。

《三教》

這款遊戲是開發者根據自身事件改變的一款國產遊戲，講的是一名遊戲策劃參加某公司舉辦的比賽不僅因為黑幕而落選，並且自己所製作的遊戲的版權也被對方奪去，他非常想證明自己的能力，所以決定跟自己的獨立遊戲製作團隊一起製作出來一款恐怖遊戲。

但是由於他的開發經驗不足，策劃經常性的逼迫員工加班，突然有一天，程序員猝死在了崗位上，所有的職指責都指向了策劃，偏偏在這個時候，策劃的女朋友也離他而去，多種不安感交織在一起，讓策劃的精神世界出現了不同的化身。
