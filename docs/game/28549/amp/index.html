百聞牌不敢設計的式神，陰陽師里都成版本之子，來了影響平衡

天天要聞
2019-12-23

玄不救非，氪不改命，老鹹魚ACG次元飯君給各位陰陽師大人們請安了。
陰陽師百聞牌已經上線一段時間了，不知道有多少陰陽師大佬們去蜃氣樓衝天梯了呢？大家都知道，百聞牌是以陰陽師裡面的式神為基礎，來設計的卡面和每一張牌、式神的技能效果的。像大天狗在陰陽師里是強大的群體輸出，在百聞牌里一樣如此。但是百聞牌還是有很多陰陽師里的式神沒有設計，一些是因為還沒來得及，而另一個問題則是在陰陽師里有些是式神實在太強，不好設計到百聞牌中，比如下面幾位。

不知火
不知火在陰陽師里開創了一個全新的鬥技體系思路，十分強！要是被設計到百聞牌中，根據不知火在陰陽師里的技能，肯定有一個張形態牌是讓隊友普攻兩次，妖刀姬三勾的形態牌，到不知火這兒變成全隊效果，哪怕也是三勾牌也有點不合理，傷害直接翻倍，太BUG了。

而不知火最硬霸的地方則是在她的被動，在陰陽師中被擊殺第一次後不知火會切換形態，也就是說她有兩條命！這若是到了百聞牌中，可能就是不知火的式神技能了，有兩條命，現在式神血量第一點的大概是4點血（防止被大天狗一下子吹死），兩條命就是8點……

御饌津
御饌津在陰陽師里的特點就是，在開啟結界時有概率反擊並且對對手造成四封的效果。也就是說，若是出現在百聞牌里，形態牌肯定少不了開結界這個形態的，然後四封方面，百聞牌現在的控制就是眩暈，以此可以想到御饌津在百聞牌中的效果可能會有反擊加上眩暈。

不僅如此，御饌津也有傷害的能力，在陰陽師中反擊可以疊BUFF，然後增加自己的各項屬性。在百聞牌中也有疊某些數值最後增加自己能力的式神，比如犬神的羈絆，山兔的骰子，桃花的加血等，御饌津是不是也來一個每反擊一次疊「1攻」或者「1血」呢？又有控制又加屬性，這保下來真要無敵……

鬼切
鬼切是陰陽師里傷害最高的輸出式神，不僅有超強的單點爆發，還有穿盾的AOE攻擊。要將這個效果複製到百聞牌上並不難，依舊可以當戰鬥時，將對手的血線打到一定的值以下時，就可以被動再追擊一下並且造成群體傷害，也就是對對手備戰區所有式神造成傷害，哪怕這個傷害是1點，在配合其他輔助鬼火的式神下，打出3次群體傷害，很可能讓對手一波跪好幾個式神。

而且這種效果對牌手本身有用的話，那麼血量越低，鬼切終結比賽的能力就越強。到後期3勾左右的時候，保得好的話一般都至少是6、7攻左右，雙方牌手血量也肯定是在觸發鬼切被動的那個值，一次攻擊追加被動兩刀，跟現在的妖刀姬一樣，可能突然就把你一波帶走了。

當然，這些式神的設計只是根據陰陽師里的原版，再根據百聞牌目前式神設計的效果來進行猜測的，一張牌肯定不能過於影響平衡，至於以後設計到這些看似BUG的式神時，策劃怎麼弄，那就讓我們拭目以待吧。
