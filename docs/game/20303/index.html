上古捲軸 刀鋒戰士：2019谷歌年度最具創新力遊戲，畫面表現一流

2019-12-17 29條評論 1,141次閱讀 128人點贊

《上古捲軸6》還遙遙無期，近期，B社（Bethesda Softworks）又做了一個艱難的決定，宣布叫停已在多平台推出的《上古捲軸：傳奇》的開發工作，但仍會繼續運營維護。

這款以「上古捲軸」為背景的卡牌對戰遊戲，由於沒能複製當年暴雪《爐石傳說》的輝煌，現在基本被放棄了。

雖然這款遊戲表現平平，但史詩奇幻風格的《上古捲軸》系列，在全球範圍內一直人氣不減，風靡20餘年。2019年初推出的衍生手游《上古捲軸：刀鋒戰士》（The Elder Scrolls Blades），也獲得非常高的關注。

就在近日，這款手游還獲得2019谷歌年度最佳遊戲的「最具創新力遊戲」，國內安卓玩家如果感興趣的話，可以通過OurPlay加速。

《上古捲軸：刀鋒戰士》採用第一人稱視角，玩家將扮演一位刀鋒戰士，前往並重建被摧毀的家鄉。遊戲沿襲原作系列的中世紀畫風和經典配樂，喚起不少老玩家回憶，親切感倍增。

在視覺效果方面，《上古捲軸：刀鋒戰士》具有極為細膩的畫面表現，光影效果尤其出色，通過實時陰影和高解析度紋理的光照，遊戲場景變得真實生動，超越了當前很多手游。

有時根據劇情需要所營造的氛圍，還是有點驚悚的感覺。

作為一款RPG手游，經典角色的收錄養成當然是重要元素。像《上古捲軸》曾出現的種族，在遊戲中均作保留，包括亞龍人、諾德人、紅衛人、高精靈等共計十大種族可選。

角色的外型打造，雖然沒有強大的捏臉系統，但多種多樣的組合方式，也是能把各個人物變得鮮明獨特。另外，遊戲本身的人物模建也非常精緻。

眾所周知，《上古捲軸》系列的世界觀很宏大，遊戲內容繁多，如果全部移植過來，就目前手機的性能配置，是無法滿足的，所以這款手游的玩法做了適當的調整，少了玩家津津樂道的高自由度。

《上古捲軸：刀鋒戰士》目前有兩種玩法，一種為「城鎮模式」，即我們熟悉的做任務、殺怪升級。在建造城鎮方面，基本按照遊戲規定的方式升級解鎖，不可以DIY。

另外一種「深淵模式」，則是在地牢里不斷挑戰副本，闖關的難度越大，收益越多。

另外，回合制的站樁打怪模式，操作是變簡單了，打鬥畫面也很出彩，但激烈程度不夠。

這款手游在加速開箱和跳過任務兩方面，需要氪金。當然等2小時後，也可以打開寶箱。所以氪金與否，還是與玩家的耐性有關。

應該說，《上古捲軸：刀鋒戰士》是一款為手機量身定做的遊戲，或許是《上古捲軸》系列的端游作品起點較高，手游並沒有達到粉絲們的期待，但單從手游本身來說，整體品質還是不錯的。
