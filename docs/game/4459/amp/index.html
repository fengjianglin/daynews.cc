祖爾格拉布的難度低讓人瞧不起？等到了這副本，就瞧好了！

祖爾格拉布的難度低，CD短（三天），人數少，副本速度快。是為了給高難度的安琪拉副本做鋪墊用的，能讓玩家迅速提升自己，過度到安琪拉。

加上很多新玩家的湧入，再讓玩家慢慢從mc、bwl獲取裝備再去打taq不太現實。很多版本都有類似的做法，比較相似的比如巫妖王版本ICC開啟前的十字軍試煉副本，也承載著同樣的功能。

龍王披風沒人搶？現在除了黑龍披風還有啥是能用的？

有人說因為分留著更重要，而且馬上開的zug有兩個法系披風，一個藍色23法傷，一個紫色23法傷1命中！

以前玩的時候打副本穿一套靈風，思維寶石加淚，武器用龍仗，打納克薩瑪斯就得用火法了，比蟲子掉的法杖好用些！

思維加速寶石個人感覺是唯一的神器，當年開zg的時候大部分公會還沒打通mc，bwl更不用說了，好像國服最快進度也就剛過二號。所以開個難度低一點的副本來給大家消磨時間。

我這兩天一直刷的zul，說實話 練了差不多六小時，從開始要刷四波五波這樣子，到現在刷兩波。總結了一下，就是怎麼出冰箱，你要看怪的抬手動作，然後出冰箱，一定要走兩步冰環，這樣基本不掉血，然後就是抵抗了，正常一個然後就是兩個，有抵抗就立馬冰盾，讓怪打，不掉血的反正，如果在有立馬冰環，冰環不建議用，遇到抵抗更麻煩，調整好位置 繼續吹風，還有就是第一波吹風，一定要冰環結束一瞬間吹，你如果提前吹風，有的怪還有冰環效果！
