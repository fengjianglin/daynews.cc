“吃雞”12.11正式服迎來“大更新”，安卓1.57G，iOS1.97G

天天要聞
2019-12-11

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友刺激哥。提到吃雞手游的新版本，大家所關心的內容，一定是正式服更新的內容了。

雖然，吃雞手游體驗服的內容，距離我們並不算特別的遙遠。但是，因為某些特殊的原因，大部分的玩家還是接觸不到體驗服的，只能從別的玩家口中，來了解體驗服。

話題：「吃雞」12.11正式服迎來「大更新」，安卓1.57G，iOS1.97G！

相比之下，大家肯定都更關注正式服的更新。如圖所示，國際服《Pubg Mobile》的正式服在12.11，已經率先迎來了「大更新」。

此次更新，安卓的全新安裝包為1.57G，iOS的全新安裝包為1.97G。看到這裡，刺激哥有一個疑問，為什麼iOS的安裝包會比安卓多了300M。

另外，想必大家同樣關心《和平精英》正式服的更新時間。如果不出現意外，《和平精英》的正式服更新的時間，預計在12月18號左右。

《Pubg Mobile》正式服所更新內容，與我們之前所介紹的內容，相比之下並沒有太多的變化。

首先，《Pubg Mobile》的進化戰場，又新增加了疾速激斗玩法。我們會在一個圓形競技場，利用車載武器互相進行攻擊。

疾速激斗的玩法一共有兩種：一、陣營死斗，二、積分爭奪。

陣營死斗以淘汰分，作為獲勝的參考依據。積分爭奪，則是根據撞擊競技場積分寶箱的次數，來決出勝負。當我們被敵對的玩家淘汰後，會掉落一定的積分。

總體來說，疾速激斗的玩法還是挺有趣的。挺考驗玩家駕駛載具的能力，以及在乘坐載具時的瞄準攻擊能力。

在經典模式，《Pubg Mobile》新更新了冰雪模式。在這個方面，《Pubg Mobile》與《和平精英》出現了差異化的更新。

《和平精英》重點推出了極寒模式，《Pubg Mobile》則是把重心放在了海島。值得一提的是，這兩個遊戲在12月的版本，都沒有推出海島2.0。

《Pubg Mobile》冰雪地圖的核心玩法，就是雪地滑板、纜車、冰雪城堡等。唯一讓人覺得遺憾的是，《Pubg Mobile》的冰雪模式，與《和平精英》的聖誕模式沒有互相結合。

如果，光子可以把這兩個模式融合到一起，絕對是1+1大於2的趨勢。

在團隊競技方面，《Pubg Mobile》摒棄了以往的武器架，重新設定了戰備方案。我們在遊戲中，可以根據仔細的喜好，來裝備武器。

值得注意的是，只能裝備一把主武器、一把副武器、兩個投擲武器（煙霧彈不能使用）。

把敵人淘汰以後，可以撿取敵人的武器，裝備2把主武器。被淘汰後，則是恢復為默認裝扮。

其次，團隊競技的武器，新增加了熟練度。我們需要提高熟練度，才能裝備使用各種武器的配件。

不同的配件，需要達到不同的等級才能解鎖。針對這點，也不太好輕易判斷它的利弊。

另外，《Pubg Mobile》的增加了「第一人稱」切換功能，以及移動時打葯的功能。還有多種武器所造成的傷害規則的調整，這些都在跟進《和平精英》之前的更新內容。

除此以外，《Pubg Mobile》還更新了「背包掛件」，當背包掛件裝備在背包上的時候，會左右晃動。

其次，一個背包掛件只能裝備在1個背包，適用於背包1\2\3個狀態。國際服的背包皮膚是3個等級是一起的，不像《和平精英》需要3個背包皮膚才可以。

最後要提到的是，《Pubg Mobile》還增加了「主題圖鑑功能」，我們可以更好地了解《Pubg Mobile》歷史版本更新的皮膚。

至於一些戰鬥方面的變動，刺激哥就不再一一介紹了。大家可以根據遊戲更新公告的遊戲內容，到遊戲中自行體驗。

好啦，以上就是《「吃雞」12.11正式服迎來「大更新」，iOS1.97G，安卓1.57G！》的遊戲內容了，建議大家在Wifi情況下更新，我們下期再見，拜拜。
