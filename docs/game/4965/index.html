暗金兩件套到手！原來暗黑破壞神2中毀滅之王才是隱藏出貨大BOSS

2019-12-03 0條評論 16次閱讀 0人點贊

摘要

在暗黑破壞神2中如果只是從出貨率來討論，三魔王中公認的一般都是勞模大於大菠蘿大於巴爾。在地獄難度下，部分精品裝備有專門的出貨途徑，不過在簡單難度下我們暫時不能苛求太多。但在橫掃簡單難度之後，我們還是找到了一些好裝備可讓我們通關噩夢難度，這次我們來展示一下我們的出貨清單。

在暗黑破壞神2中如果只是從出貨率來討論，三魔王中公認的一般都是勞模大於大菠蘿大於巴爾。在地獄難度下，部分精品裝備有專門的出貨途徑，不過在簡單難度下我們暫時不能苛求太多。但在橫掃簡單難度之後，我們還是找到了一些好裝備可讓我們通關噩夢難度，這次我們來展示一下我們的出貨清單。

肩胛骨雙刃斧

暗金斧有一項非常出色的屬性，那就是能增加100%對於不死生物的傷害。這個數據看起來很美好，理論上你在遊戲中遭遇到的大部分怪物都是不死系，所以這個屬性很重要！但是反過來，在不同的地圖中你也可能遭遇不到不死系怪物，你覺得有用么？不，跟雞肋一樣！

金瑟的裂縫空間之刃

空間之刃出色的面板自然是關注的第一焦點，其二則是能減少11點法術傷害。這是什麼概念呢？一般的裝備最多減少1到2點法術傷害，這把暗金武器一下子就能減少11點，算是非常誇張的概念了。所以絕對值得啟用，不過非常可惜的是我們是亞馬遜，就只有選擇了放棄。

殘忍之串列者雙手劍

金色裝備一般都沒有太多的好選擇，不過這把雙手劍勉強算是能看的存在。基本屬性都滿足一個近戰職業的基本需求，不過猶如雞肋食之無味棄之可惜。看起來很出色，但是相比於上述的暗金裝，還有比較大的差別。

鏈枷

急速攻擊速度增加聖騎士技能等級，這兩點對於聖騎士而言簡直就是完美增益。只是這兩點，已經足夠為他打一個高分。不過因為面板太低我們只能遺憾選擇放棄了，畢竟對於近戰職業而言，面板才是重中之重。

不知道你們對此有什麼看法呢？
