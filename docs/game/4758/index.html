LCS解說點評2019年的獎項：Uzi和Jackeylove兩人應該獲得1V9獎項

S9世界賽落下帷幕之後，全球LOL職業聯賽的S9賽季已經收尾，2019年也即將結束。

在年底的時候，各大賽區的電競工作人員都開始對2019年作出點評和回顧。

其中有一個非常騷的點評，就是來自於LCS賽區的解說們。

在錄製節目的同時，他們也點評了2019年以及各個獎項的歸屬。

首先是最佳戰隊獎，在他們眼裡這個獎項毫無疑問地要頒發給FPX戰隊。

因為首次參加世界賽的FPX就發揮出色，打出了自己的風格，還拿下了S9世界冠軍。

雖然有不少歐美的觀眾提名G2戰隊，因為他們在今年的MSI季中賽上讓全世界印象深刻。

但無可厚非的是，S9世界冠軍的含金量顯然要大於MSI季中賽，這個獎項頒發給FPX更合適。

隨後是最快進步獎，三位解說都認為是FPX戰隊的輔助選手劉青松Crisp。

從默默無聞的LPL中游輔助到世界賽表現最好的輔助，Crisp就是最絕佳的答案。

這個獎項也沒有太大的爭議，最快進步獎頒發給FPX戰隊里的任何一個人都合適。

除了中路的Doinb，其他四個人在今年年初的時候都是個不被人看好的無名小卒。

最後是1V9的獎項，這個獎項可就不止有一個候選人，這也讓LCS的解說們相當糾結。

第一個候選人是Uzi，LCS解說知道很多人用數據來否定Uzi，認為Uzi的經濟佔比太高。

但LCS的解說Kobe認為，大家只看到了Uzi的經濟佔比以及自家打野頻繁的幫助。

但並沒有看到Uzi前期通過自己的實力來取得補刀領先，同時只要Uzi在下路。

對方隊伍的打野也會頻繁來下路gank，因為大家都知道Uzi的個人實力有多強。

第二個候選人就是Jackeylove，自從去年的S8世界賽上涅槃之後。

Jackeylove如今已經成為IG戰隊里很粗的一條大腿。

尤其是今年的春季賽以及MSI季中賽期間，在隊友發揮和表現都欠佳情況下。

Jackeylove可以說是一路將IG戰隊給抬在肩上往前走。

但帶編認為，如果這個1V9的獎項按照名字來定義，那頒發給水子哥和棗子哥都不太合適。

水子哥這裡有Rookie和TheShy，棗子哥那裡也有狼行和Karsa，並沒有真正的1V9。

這個獎項頒發給CG戰隊的ADC選手CodySun，帶編認為會更加合適一點。

那上單的Huni，那剩下的三個菜b，唯一孫哥是真的全程在1V9，雖然自己偶爾也會送。

各位小夥伴認為，這些獎項還有其他更合適的選手嗎？可以留下你的看法喲。
