摘要

這些名場景要麼是玩家們初入遊戲世界，周圍美好的景象讓人留戀往返。

也可能是某個事件後主角的心境發生了巨大的變化。

《塞爾達傳說：荒野之息》林克蘇醒

在遊戲的開場，林克從睡夢中被喚醒。

玩家們根據指示穿戴整齊後，一點一點朝著光亮的洞口跑去。

隨著節奏不斷加快的音樂聲，林克也不受玩家控制地向前來到懸崖邊望遠。

周圍隨著微風吹拂輕輕擺動著的綠草，眼前一大片對玩家來說相當廣闊的未知世界，讓人印象深刻。

從一開始看上去陰冷潮濕的環境，到朝著亮光走去的豁然開朗，也暗示著遊戲的進程。

雖然之後林克可能沉迷各種刷人馬和找呀哈哈，甚至忘了還要救公主那回事。

不過初期震撼的場景也足夠激起玩家們探索世界的好奇心。

《生化奇兵：無限》初入天空之城

某天，主角布克接到委託要去往哥倫比亞天空城的某地，帶走一位女孩。

玩家首先要乘船來到一座色調灰暗的燈塔，這裡壓抑的氣氛會讓人不由自主地繃緊神經。

主角坐上了去往天空城的座椅。

接下來在一片灰暗的雲層和節奏飛快的提琴聲中，玩家能夠感受到布克的不安。

隨後，突破了陰雲展現在布克眼前的，是一座彷彿天堂般的城市。

伴隨著瞬間變得輕柔舒緩的音樂，哥倫比亞各種建築從玩家眼前掠過。

在徹底停穩前玩家還看到了各種標語，也在暗示著哥倫比亞遠沒有想像中的美好。

《古墓麗影》勞拉爬上信號塔

在重啟版的第一部《古墓麗影》遊戲中.

尚且青澀的勞拉和眾人一同被卑彌呼的力量困在了某座小島上。

島上遠遠不止他們，還有想要抓住他們的勢力索拉瑞。

為了尋求幫助，勞拉不得不前往信號塔發布求救訊號。

然而在前往信號塔的路途充滿了艱險。

勞拉需要潛入索拉瑞的營地，眾多緊張刺激的戰鬥後才開啟自己孤獨的爬塔之路。

這裡的每一步看上去都讓人肝顫，尤其是越爬越窄的道路。

稍有不慎勞拉可能就要摔得粉身碎骨。

信號塔也是勞拉成長過程中不可缺少的環節，勞拉的孤獨和周圍空曠的場景相互映襯。

也是她突破自我的一次轉折。

《刺客信條：叛變》里斯本大地震

謝伊在教堂的地下找到了一片用途未知的伊甸碎片。

讓人沒有想到的是在觸碰之後神器散成了粉末，隨之而來的是一場破壞力極強的大地震。

謝伊拼了命地朝前奔跑，避免周圍的建築倒塌砸到自己。

街邊時不時地有路人來不及逃跑被大地震波及。

最後謝伊逃出生天，在船上望著里斯本，那種愧疚感讓人感覺到謝伊深深的無力。

也是謝伊人生的一個轉變。

許多玩家們印象深刻的場景來自於角色與玩家之間的共情。

玩家們從自己的角度去理解看待可操控的角色。和他們一起經歷各種冒險故事。

那麼，你記得哪些讓你印象深刻的劇情場面呢?
