FGO恩奇都簡評：強力寶具，單挑性能優秀的單體從者

天天要聞
3 天 ago

FGO國服正在進行聖誕復刻活動，開啟艾蕾卡池的同時也會開啟常駐從者恩奇都的單UP卡池，本期就為大家分析一下恩奇都的性能如何。

從者寶具：恩奇都的寶具為紅卡單體輸出，造成傷害前降低敵方單體20%~40%防禦力三回合（效果隨OC等級提升），並付與【神性】特性敵方單體眩暈狀態一回合。

小恩的寶具在未來將得到強化，寶具的傷害倍率提升，對敵方單體的防禦力降低幅度提升到30%~50%，並且追加對【人類威脅】敵人200%特攻狀態。

從者技能：恩奇都的一技能效果為自身紅卡性能提升30%~50%一回合，隨機提升綠卡或者藍卡性能30%~50%一回合，充能時間7~5回合。

二技能效果為降低敵方單體30%~50%暴擊發生率三回合，解除敵方單體迴避狀態，賦予自身迴避狀態一回合，充能時間7~5回合。

三技能效果為自身HP回復5000~10000，解除自身弱化狀態，充能時間12~10回合。

職階技能【對魔力】自身弱化耐性提升20%。

從者優點：①小恩的寶具有非常高的傷害，寶具前的防禦力降低效果和高幅度的紅魔放能帶來極高的輸出，未來強化寶具之後傷害更是進一步提升，雖然人類威脅對象不多，但是即便是沒有人類威脅特攻，小恩的寶具傷害也非常高，剋制倍率情況下，持特攻狀態下的寶具，斯卡哈傷害最高，小恩第二，無特攻狀態的寶具，小恩的寶具傷害是最高的。

②小恩具有較強的單挑能力，二技能有短CD的迴避效果，還能降低暴擊發生率，三技能有萬血回復加上解弱體，配合自身高等級的對魔力也很難被控制，自身的血量在全從者裡面也是排在前頭的。

單挑從者帶來的好處就是和指令紋章的相性很高，大部分的指令紋章打在上面都是不虧的，當然比較推薦打上集星、出星或者暴擊威力提升效果的指令紋章。

③二技能的解除敵方單體迴避，寶具對神性敵人的眩暈狀態，以及自身沒有性別的特性，在特殊場合也有奇效。

④小恩的配卡十分奇特，但是指令卡的性能相對來說也是很不錯的，在單挑場合能發揮出不錯的效果。

從者缺點：①和小恩特立獨行的性格一樣，小恩單挑的發揮要比在隊伍中的發揮更好，因為自身的配卡不利於隊伍的發牌，紅卡寶具三綠配卡不知道是該融入紅卡隊還是綠卡隊，這也是很令人頭疼的一點。

②小恩的技能也存在一定不穩定性，一技能的藍魔放和綠魔放不穩定，二技能的複合效果也不太兼容，雖然大部分情況都是用來迴避，但是在需要解除迴避的場合就需要思考一下了。

③三綠配卡加上一頭綠髮很容易讓人以為是綠卡寶具。

整體評價：特殊的設定讓小恩只在特殊的場合發揮比較出色，不過整體來說，可以登場的機會還是很高的，無論是刷柱子，還是部分需要單挑的副本，或者是神性單體敵人，都是不錯的選擇。

卡池評價：小恩是聖晶石常駐從者，有幾率在聖晶石卡池歪到，未來的絕對魔獸戰線舞台劇公演紀念卡池和絕對魔獸戰線動畫上映紀念都還有小恩的單UP卡池，白情卡池有雙UP，明年還有兩次職階日替卡池，槍階職階池抽起來還是不太虧的，所以小恩的卡池並不少，如果近期有其他安排的話，可以選擇等待小恩的其他卡池。
