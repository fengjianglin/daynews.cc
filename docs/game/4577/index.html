全明星引發“咸甜之爭”！艾歐尼亞地位不保，北方網通才是王道？

2019-12-06 0條評論 16次閱讀 0人點贊

摘要

今天天上午觀眾們迎來了英雄聯盟2019全明星第一日的活動，首先帶來了“快樂模式”無限火力，不到20分鐘的時間雙方共擊殺137次，冠軍中單Doinb憑藉著25-14-10、55.6%的參戰率以及24次的參與擊殺成為了本場當之無愧的MVP。

今天天上午觀眾們迎來了英雄聯盟2019全明星第一日的活動，首先帶來了「快樂模式」無限火力，不到20分鐘的時間雙方共擊殺137次，冠軍中單Doinb憑藉著25-14-10、55.6%的參戰率以及24次的參與擊殺成為了本場當之無愧的MVP。

但無限火力對局中的一個細節卻引起了網友的討論，不少網友對於幾名選手相當有默契的選擇了上路一級團產生了疑問——無限火力一級團不應該在中路打么？

其實早在之前，國內一些玩家對「無限火力一級團到底在哪打」這個問題就有過一番討論，根據網友的描述，比爾吉沃特等網通區的玩家們比較習慣於上路開團，以艾歐尼亞為代表的電信區卻更習慣於中路團戰——別問為啥，問就是大區傳統。

不過既然職業選手都統一選擇上團了，這是不是代表著網通區的遊戲理解更與國際接軌，而傳言中「蛤蟆吃王者，河蟹挑大龍」的艾歐尼亞即將走下神壇了呢？

有網友戲稱「南方電信」和「北方網通」的南北差異是導致這個問題的根本原因——的確，中路兵線狹窄的地形對於「北方大漢」們來說似乎的確施展不開手腳，上路的廣袤地形更適合大家拼個你死我活。

但從另一個角度來說，艾歐尼亞作為國服第一大區，玩家們想要上分就需要更加理智和謹慎地應對比賽，中路的短線程讓雙方都有了點到為止的餘地，而這種中路開團的方式也勢必會通過小號等形式影響到電信的其它區服。

至於電信和網通與國際接軌的問題，有一說一，近幾年LPL賽區已經通過戰績證明了自己「第一賽區」的地位，甚至許多外服選手也開始來國服打RANK了，現如今「國際與LPL接軌」才是英雄聯盟中的大趨勢。

不過說到底，無限火力本就是以娛樂為主，打的過癮才是王道，這麼較真的分析上路開團和中路開團的差異，反而會讓遊戲少了一份不服就乾的快樂吧。
