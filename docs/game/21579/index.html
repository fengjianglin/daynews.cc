喜愛福快嘴：世界狙神排名，誰敢稱第一？

2019-12-19 1,125條評論 2,387次閱讀 1156人點贊

CFPLS15全明星周末結束了，CFS2019世界總決賽結束了，但戰鬥還沒有結束，CFHD競技盛典將於15日開啟，CFPLS15季後賽也將在本月打響。要說最近最大的新聞是什麼，那非巴西賽區包攬世界冠亞軍莫屬，其中冠軍隊VG狙擊手dgz，人送稱號大公子，彷彿突然蹦出來的世界第一狙神。據說他在巴西狙擊只排第二，據說巔峰時期的70kg也打不過他。那麼，世界狙擊排名，誰敢稱第一？

Dgz，巴西VG戰隊新引進狙擊手，少年的他初登CFS世界舞台，一桿衝鋒瞬狙震驚四座，先後與中國狙神馬哲、田豐、綠豆對戰，大公子沒有絲毫劣勢，反而越戰越勇，對中國狙神進行了極大的壓制。大公子一戰成名，但據網友所知，大公子在巴西還只是第二狙神，第一名是來自BD戰隊的Goken。在冠軍賽中，巴西兩大狙神遭遇，dgz對陣Goken，大公子以絕對優勢拿下勝利。看來，以後巴西第一狙擊的稱號應該屬於大公子了。

大公子的狙擊以衝鋒、瞬狙著稱，這一次的CFS2019世界賽成為他秀翻狙擊的舞台。有很多CFer說，曾經巔峰時期的70KG也不一定能打過他，如今的中國狙神除了wolf還沒有出戰，其他全部落敗。這樣來說，大公子已經成為了世界第一狙神，而中國竟然沒有誰能出來一戰。嚴格來說，大公子的狙擊打法和之前的70kg非常相似，都有很強的進攻性，可惜不是同個時代，不然真可以期待一下他們的狙擊對決。如果真的對上了，你覺得70kg和dgz，誰更勝一籌？

今年的CFS世界總決賽，中國戰隊遺憾落敗。技不如人是真的，但沒有巴西年輕人那股衝勁也是真的。就如大公子的狙擊，自信敢打，就是瞬狙追求一發擊殺，有時候來手感了更加無視細節，雖然被認為操之過急，但越戰越勇這股氣勢很少有人能做到。反觀中國狙擊手，更多求穩，利用意識卡點等，當然這種打法沒有問題，但總是少了那麼點血性。難道是比賽打太久了或者年齡受限了？

世界狙神排名，你認為哪位是第一名？喜愛福快嘴，說你想聽，寫你想看，下期見。
