DNF：85史詩新增五段打造，屬性完爆超界，格藍迪深淵又來了

2019-12-19 749條評論 4,310次閱讀 690人點贊

炒冷飯也好，智窮慮竭也好，策划到底還是給玩家們整出來了裝備上的「百花齊放」，一百級的史詩裝備不僅限於大家認知中的團本升級獲得和深淵獲得，在未來的一百級版本，你還可以通過把八十五級的史詩防具升級，來獲得更加強力，也更加有特色的史詩。可以通過升段來變強的八十五級史詩共包括如下五套：板甲魔戰套，輕甲天御套，布甲天劫套，重甲嗜血套，皮甲殺意套。

現在來說，其中最值得稱道的應該就是魔戰套了。魔戰套陪著小編走過了不知道多少個版本，烏龜的時候，穿著魔戰打團，盧克的時候穿著魔戰做CD換裝，超時空以後，就穿著刷深淵刷白圖了。魔戰套的CD和速度優勢讓人上癮！在早期沒有加強傷害的時期，魔戰的傷害甚至不如假豬，但現在經過了幾次加強，魔戰傷害真的不虛其他四套，優勢巨大。

其實八十五級史詩防具，一共有十套，每種防具類型兩套，除了開頭說的五套，還有：板甲霸域套，輕甲球手套，布甲逝魔套，重甲千蛛套，皮甲戰術套。為什麼這五套防具不能升級呢，小編從防具特色來分析一下。

霸域套有全程霸體，而現策劃把霸體葯，某些職業的建築霸體都砍了，會讓你全程霸體嗎？球手套特色在於後跳前沖的六倍黃字，不好意思太變態了，瞎子六倍炸眼歷歷在目。逝魔套的特色在於傷害巨高，單純追求傷害有團本史詩就夠了。千蛛套曾經作為異常流毒王的一套裝備，現在毒王的異常大砍，沒用了。戰術套是光環套，和兵法套的作用重合了。總而言之，這五套沒能被策劃炒冷飯，不是因為策劃忘了，單純的是因為這五套防具的特色屬性不適合新版本，或者不被策劃推崇而已。

而魔戰的CD，天御的特效，天劫的減防，嗜血的死不了，殺意的背擊，這些特色在新的版本還是比較適用的，尤其是魔戰的CD，本身新版本的神話裝備就有減CD的屬性，兩者並不衝突。

另外還有個問題，那就是不能升級那五套，普遍面板加成很高，能多吃系統奶，不利於高增幅大佬。最後說一下這些裝備的獲取，別去格藍迪也別去時間廣場，只有寂靜城深淵才是單純產出八十五史詩的地圖！
