夢幻模擬戰：不為人知的小彩蛋，老婆會分身術的快樂，你想像不到

2019-12-17 1,202條評論 3,724次閱讀 1930人點贊

傳說君今天默默給列表好友點贊的時候，不小心手誤點開了其中一個的個人名片界面，沒想到的發現了一個非常有趣的現象。我們都知道，《夢幻模擬戰》手遊人物名片中有以下幾個數據顯示：賬號昵稱、巔峰戰績，總點贊數、個人簽名、個人統計以及最重要的角色展示界面。

我們也知道，一般來說，人物展示界面都是5個不同人物，玩家頂多可以對角色的朝向和動作進行微調，絕不可能出現像上圖出現5個蒂朵的現象出現。那這名玩家是怎麼做到的呢？在排除開掛改腳本的可能性後，傳說君通過多種渠道，終於找到了答案。

原來這是今年年初的時候，玩家探索出來的一個小BUG。通過對展示人物進行一系列操作，就可以成功在展示欄中，達成分身術的成就，而且只要是你擁有的英雄，都可以做到。下面就由傳說君用自己的小號來一步步講解，如何實現吧~

1.清空展示欄，不保留任何一個人物。

2.根據系統規則，在5號位放上想要進行分身的角色（這裡我們選擇萌萌噠的奶騎），1234位隨意設定。

3.把1234號位的人物選定取消，只保留5號位我們需要進行分身的。

4.然後點擊選擇人物界面的框框外面取消，不要點確定！

5.按照第2步的操作，在4號位擺上我們需要分身的人物，其他位置隨意。

6.按照第3步的操作，取消1235位的人物選擇，只保留需要的那個。

7.按照第4步，點擊選擇人物界面的框框外面取消，回到名片主界面。

以此類推，最後1號位保留我們所需分身人物，其他4個位置選定後再取消。

完成以上操作後，最後分身人物來到1號位時，我們再點擊”確定”按鈕，就能呈現我們所需要的形象展示了。而且在分身之後，並不是說5個位置都必須是同一個人物，我們可以稍作優化，具體的展示可以看文章最後的圖片。

從人物選擇界面我們可以看出來，完成分身術的奶騎，每一個都是1號位展示，可以說是遊戲的一個BUG，相信ZL策劃也是知道的，但是在不影響遊戲內容的前提下，這種小彩蛋無傷大雅，還能給玩家帶來不錯的驚喜，因而還是保留下來，沒有進行修復。

而從玩家帖子的回復中，我們也可以看出來，利用這個小彩蛋，有很多名場面誕生，像是四個白毛和一個馬修的故事（馬修昏倒在地就很有說法了）；5個光頭將軍翩翩起舞也不是不可能；又或者是5個黑騎士共處一個舞台，這可不是世界祝福就能滿足的~

那麼，大家在學習後，會擺出怎樣的畫面呢？歡迎在評論區貼圖告訴我們~
