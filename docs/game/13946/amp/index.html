DNF新版“兵法套”，5%減防算錯了，實際提升2%，玩家：立即刪除

天天要聞
6 天 ago

「地下城與勇士之小狐狸君愛談遊戲電競」第七百二十二期《DNF新版「兵法套」，5%減防算錯了，實際提升2%，玩家：立即刪除》，隨著國服推出新版的「兵法套」之後，許多玩家都在討論34仔打團和5%減防問題，更有玩家說34仔化身66仔。那麼實際提升是多少呢？5%的減防到底怎麼算才合適。畢竟韓服已經刪除了這個系統，國服史派克強制復活這個減防系統，勢必會給傷害帶來干擾。小狐狸君就實際測試一下這個國服特色減防系統的收益如何把。老規矩搞快點，直接上本次國服正式服實測的結果展示：

DNF新版「兵法套」，

DNF中百分比減防除設定是：對精英只有75%效果，以及對領主（也就是每個副本里的boss）只有50%效果，外加打折後的效果還要向下取整！所以最終的結果絕對不是玩家的5%+效果，相反是遠遠低於這個數值的。就算減防收益是最高的，但是實際上由於存在特殊的設定，導致玩家不得不考慮這個問題。團本基本上難點都在BOSS，而且34仔還存在掉線、死亡問題，所以34仔的輔助能力，還是當前版本不能接受的。

5%減防算錯了，實際提升2%，

所以韓服的設定在國服同樣奏效，這裡用修鍊場的普通沙袋和專門設置領主後的沙袋來做測試。和上述的2.5%、然後向下取整數完全吻合。普通沙袋的減防效果是5%，領主沙袋的減防只有2%。所以團本收益也是只有少得可憐的2%減防，所以增刪和減防還是有著很大的差別的。之前驅魔的狗也是不同怪減防不同的。

玩家：立即刪除

也就是說目前右邊6件的提升，還沒有左邊5件的提上大。那麼實際提升是18%，算到34仔的34%提升，只有58%！可見這58%的提升，還是不如一個3000面板的輔助C提升大。DNA若干2個純C+一個34仔比之前稍微高一點，但是最終的提升還是缺少輸出的！玩家表示這套新的兵法套，還是刪除了吧。
