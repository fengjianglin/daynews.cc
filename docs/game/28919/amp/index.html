《代號：C》12月26日開測！想變身小蝙蝠縱向惡魔城堡一日游嗎

天天要聞
2019-12-20

沉睡已久的魔王蘇醒，喚醒了充滿怪物的惡魔城堡，惡魔城堡及其危險，處處遊盪著怪物，人類獵人為了阻止怪物入侵，冒著生命危險進入惡魔城堡打敗魔王！這其實是個老套的故事，但在這款遊戲中卻顯得奇妙而又魔幻。今天給大家帶來的遊戲是《代號C》遊戲將於19年12月26日開啟限量不計費刪檔測試。

作為一款探索劇情類型橫版動作ARPG手游，遊戲的畫風以暗黑，灰暗為主色調，也映襯了遊戲的故事背景，魔王蘇醒，世界即將陷入黑暗中。

遊戲的人物設定為夜族，故事也是以夜族進行展開，這裡夜族也是我們熟知的血族，吸血鬼。夜族神秘男子身在黑暗心向光明，混血獵人艾維斯，使魔等元素給這款遊戲增添了不少神秘色彩。

不得不提的是遊戲的畫質十分精美，無論是遊戲中的過場動畫，還是戰鬥場面，或者僅僅只作為背景板的插圖都製作的十分細緻，所營造的魔幻暗黑風格也更加濃烈，部分角色還能化身成蝙蝠，獲得飛翔的能力。

在遊戲的劇情方面，採用的是多英雄的模式，在遊戲開始，玩家會隨著劇情操控不同的英雄角色發展劇情，在惡魔城中探索，解決遊盪在惡魔城的怪物。每個英雄所使用的武器都不相同，但在遊戲中即使是手持長劍的角色，在裝備弓弩武器時也會改變攻擊方式。

英雄艾維斯擁有超高的劍術，但在更改裝備後也可以進行遠程攻擊

遊戲的操作頁面十分簡潔，普通攻擊，奧義技能，輔具攻擊，閃避和跳躍，簡單的元素構建了一個完整的戰鬥體系，降低了玩家的操作難度，卻不降低玩家的操作上限。

【奧義】

遊戲中的奧義技能可以說是戰鬥時的亮點，不同的英雄根據自身的特性也擁有不同的奧義，不僅擁有強大的威力，還有著絢麗的技能特效。

【戰鬥劇情】

劇情戰鬥的也加入了交互按鍵，讓玩家更有參與感，戰鬥時的刀光劍影好似電影畫面一般，戰鬥效果做的極好。

【輔具】

遊戲中對於戰鬥的自由度非常高，除了帶有英雄鮮明特色的奧義保持不變之外，玩家可以根據調整裝備，輔具對自己的戰鬥進行搭配。輔具是這個遊戲中很有意思的設定，需要消耗紅心才能使用，紅心則可以通過探索城堡，擊殺怪物來獲得。

遊戲中的劇情也是隨著玩家的探索惡魔城的進度隨之展開，也增添了不少趣味玩法，占守傳送門，探索寶箱，選擇性劇情等等

巨大巍峨的惡魔城更是給玩家極大的探索空間，搭配精美的畫質和極具特點的風格，玩家即在做劇情時，也可以將其當做一個迷宮探索類的遊戲，在其中尋找線索，展開惡魔城的神秘面紗。

當然這款遊戲還可以進行組隊冒險，利用策略挑戰BOSS，擊敗強敵。總體來說，這款遊戲在遊戲畫面上做的還是相當不錯的，戰鬥機制不複雜，好上手，遊戲的故事背景劇情也是經典的魔幻題材，部分英雄的血族設定也是讓喜歡這類元素的小夥伴期待不已，如果喜歡這款遊戲不妨進行預約了解一下~
