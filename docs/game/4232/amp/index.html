Hello，各位魔獸世界的朋友們大家好！

魔獸世界懷舊服目前為止開服已有近三個月時間，大部分玩家也基本早已滿級，隨著近期榮譽系統、戰場等新內容的不斷增加，懷舊服給我們帶來的玩法及樂趣也越來越多，與此同時金幣的消耗量也越來越大，那麼今天小編就來和大家分享一下懷舊服商業小百科，讓你打G路上事半功倍。

材料篇

懷舊服初期，材料價格相對而言會比較堅挺，攢材料丟AH是一個不錯的選擇，這裡重點說一下價格相對可觀且獲取難度適中的材料。

首先要說到的是元素系列：一、元素火焰，在安其拉開放之前，分解火焰之心不能獲得元素火焰，只能通過掉落，所以元素火焰價格長期居高不下，價格基本在5金左右；二、元素之土，基本屬於不會掉價的材料，而且在黑翼之巢開放後期，價格也會水漲船高，平均在5金一個，而且大部分玩家都知道，元素之土在安其拉和納克薩瑪斯都大有用處，所以有多餘G幣的朋友還是可以囤積一部分。

其次，原始黑鑽石也一直是熱門之選，在祖爾格拉布開放之前會漲到接近80G一個，價格不菲，主要用於ZG附魔和聖騎千金馬，在1.10版本開放前，差不多會下跌到5G一個左右。

裝備篇

升級過程中刷到裝綁的紫色裝備，可以說RP爆發，懷舊版本前期紫裝稀缺，並且部分屬性逆天的價格更是居高不下。首先是抗性裝：主要包括綠色+20火抗戒指，在MC前期市場相對不錯，另外在安其拉開放前皮甲和布甲自然抗性裝價格也會不錯，同時還有+20自然抗性戒指，只能通過AH獲取。

其次，裝綁紫裝主要分析一下劍師護手和寒冰指環，這兩件紫裝屬性爆表，價格1000G起毫無壓力，出了你就RP爆發了，大賺一筆，千G馬直接出爐。

專業篇

首先是附魔，如果你是全服第一人出了+55治療或者30法傷圖紙，那麼可以告訴你，你賺大了，光代工費基本在100G起，而且在前期基本處於有價無市。

其次是煉金，圖紙價格基本如下：泰坦1000G、超能800G、精鍊500G，另外代工費基本在5G左右，當然最穩定的莫過於點奧術水晶，收入穩定。

最後說一下制皮，懷舊版本魔暴龍套對於狂暴戰及盜賊來說是保值度極高的裝備，如果你能有幸一直刷且無工作室打擾，基本可以大發一筆，另外早期可以收集厚皮製作夜色衣服，直接丟商店一次也可以賺個幾銀，掛機時刻的不二之選，不在電腦前，同樣能幫你賺錢，前提是收足夠多的厚皮，畢竟現在是月卡。

那些你得越過的坑

首先是AH這一塊，對於卓越巫師之油大家都不陌生，但是有時候大家會忽略它的使用次數，以為就是按照數量消耗，所以就會有「不法商販」把1次的掛AH按5次賣，當然也總會有人上當，大家注意逛AH時擦亮眼睛哦。

其次，付費取信，也有不少人上過當，被坑過，最常見的就是拿有效無敵藥水充當泰坦合劑，然後付費取信騙取G幣，總有些奸商屢試不爽。
