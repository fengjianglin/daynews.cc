雲頂之弈9.24b更新公告：三星英雄全削，眾多英雄羈絆改動

今日，雲頂之弈設計師在直播中預告了9.24b版本的更新公告，外服將於近日上線，國服具體上線時間待定。重大改動——三星英雄屬性全面降低原來雲頂之弈英雄升星的計算方式為：1星英雄升級為2星，生命值和攻擊力乘以1.8；2星英雄升級為3星，生命值和攻擊力乘以2。然在9.24b的改動中，2星英雄升級3星，也改為了生命值和攻擊力乘以1.8的計算方式。
