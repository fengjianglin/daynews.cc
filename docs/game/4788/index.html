Hello，大家好！我是大飛，很高興又見到你。

《我的起源》開服已經有近20天的時間，經過20天時間的沉澱，玩家們都找到了適合自己的遊戲路線。

氪金黨在《我的起源》當中，喜歡PVP！所以對於他們來說，打造自己想要的起源套裝，合成高資質神話、限定、起源寵物，使他們非常喜歡做的事情。

公會戰、據點保衛戰、搶佔遺迹、節點突襲、烈焰爭奪，這些日常活動是他們的最愛。

建造黨在《我的起源》，當然是喜歡建造了！在不同的地點，建造不同的風格的房子，是他們最愛做的一件事！

據點可以是他們的建造選址，公會也可以是他們的建造選址，當然也有不少微氪玩家，選擇在自己的庭院建造，因為自己的庭院，可以購買不同風格的家園戶型。

《我的起源》對於建造黨，還是比較友好的，唯獨一個缺點就是建築擺放和撤銷不方面，沒有藍圖，也沒有一鍵取消。

當然，在《我的起源》當中，不乏一些佛系玩家。以太幣商人就是佛系玩家的一種，作為一名0氪黨，大飛就是一個喜歡看著以太幣瘋漲的佛系玩家。

大飛在遊戲當中，利用了一個多星期的時間，就積攢了1000萬的以太幣。

大飛為何能留下這麼多的以太幣呢？原因很簡單，因為以太拍賣行中，對於平民最重要的「閃耀晶石」無人銷售。

作為一名佛系玩家，其實大飛還是有夢想的，爬塔層數更高就是夢想之一。

目前，大飛在區內100層大神的贊助下，爬塔到了76層。大飛本以為借到了戰力，就可以輕鬆爬塔。但是，整個女武神試煉爬塔的過程，給大飛帶來的卻是另一種感受。

這種感受就是：戰力可以借到，但是生存必須靠自己。

平民想要在女武神試煉當中，爬塔層數更高，除了提高戰力以外，還需要注意生存。在不同的環境下生存，最大化自己的輸出，是玩好《我的起源》的關鍵！

如何做到這一點呢？想要在不同的環境下做到最大化輸出，增加人物三圍屬性的戰鬥食物是必需品之一。

除了戰鬥食物以外，在寵物助戰界面，合理的搭配助戰寵物（寵物助戰有四個位置，四個位置適合不同元素的寵物，搭配合理的助戰寵物，可以激活元素之力，元素之力可以最大化人物在不同場景的輸出），也是最大化場景輸出的一種方法。

雖然，兩種方式都能使玩家更容易在不同的環境當中生存，但是後者對於女武神試煉並不生效，所以，在女武神試煉爬塔的時候，我們應該搭配戰力最高的寵物！

前面，我們說到了如何在不同的環境中，最大化自己的輸出，這裡我們講一下，如何在不同的環境當中生存。

想要在不同的環境當中生存，我們要準備3種食物和1種藥劑，他們分別是「辣椒肉排」、「解毒熱湯」、「西瓜沙拉」和「避雷藥劑」。

上述4種物品，使用後，可以使得玩家在冰、毒、火、雷四個區域當中無傷生存。所以，在女武神試煉爬塔的時候，我們必須帶上上述物品。

當然，除了食物和藥劑以外，我們在爬塔的時候，還可以使用不同的套裝來應對。3件同類裝備，可以激活套裝效果，套裝效果可以起到免疫環境傷害的作用，同時也可以起到克敵的作用。

如何分辨裝備免疫的環境呢？我們可以通過裝備的抗性來分析，冰抗性對應的是冰環境，毒抗性對應的是毒環境，雷抗性對應的是雷環境，在不同的環境下，我們應該選擇不同的套裝。

以上內容，就是《我的起源》的生存要素和爬塔經驗，希望上述內容對大家有幫助

本文到此結束，謝謝大家的觀看和關注！我是大飛，我們下期再見。
