DNF幻化自選活動，絕版武器也不值錢！虎牙狂人小號那把很拉風

天天要聞
2019-12-21

不知不覺又到了年關，普雷版本也走到了末期，官方通過各種活動給大家減負很多，相信很多玩家都已經畢業了，如今100級更新內容已經曝光，玩家們都比較期待，剩下的這些時間勇士們可以對自己的武器或者裝備進行打磨，用來迎接新版本的更新，但對於增幅達到普雷門檻的玩家來說，重要的已經不是打磨裝備了，而是如何拿到一把絕版武器來幻化。

​早期因為武器裝備的統一化，人們對於當前裝備設定不太滿意，所以官方更新了一個幻化系統，就是用來解決這種問題，也正因為這個系統，很多絕版的用不上的武器價值飆升，就比如魔劍、格朗等等，這些都是可以買到的，而那些史詩裝備就無法買到，沒有存貨的只能幹著急，看著別人拿著獨一無二的武器上街，自己很是羨慕。

​不過近期官方的一次更新直接讓幻化系統崩潰了，同時也讓之前價值飆升的武器變得不那麼值錢了，因為官方推出了一個新的活動，就是武器幻化自選活動，只要你升級到高級冒險家36級，就能夠開出自己想要的武器，很多絕版的史詩都能開到，魔劍和名刀之類的更不在話下，不過需要投入一定的金錢才能拿到這個許可權，但不管怎麼說，玩家們都能得到自己想要的武器了，現在一些絕版武器也爛大街了，想要獨特只能找一些別人想像不到的武器來幻化。

​玩地下城的估計沒有不認識狂人的，也是當前劍魂傷害最高的保持著，除了劍魂外他還有很多小號，其中一個小號幻化的武器造型很拉風，而且這把武器並不是我們常見的，它是一把紫色的短劍，名字叫做灼魂之魔劍短劍，一把70級的武器，造型較為奇特，刀把處比較窄，末尾寬大，看似就像是放大的尼泊爾，配合一身黑紅搭配的天空套，整體看起來很霸氣，如此另走捷徑的幻化方式讓人佩服。

​不過後續估計幻化的人就不多了，畢竟新版本更新之後出現了一種新類型的裝備，比史詩更加高端的裝備—神話裝備，而且新版本史詩可以交易，變得不再之前，算是對武器裝備的一種更新，另外神話裝備可以和普雷套裝隨意搭配，看來幻化不過是官方給玩家的一個裝備過度而已。
