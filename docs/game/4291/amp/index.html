貓和老鼠：多一個小號都有哪些好處？可以抽大號抽不到的皮膚

天天要聞
2019-12-08

摘要

大家好我是阿涼。不知道大家在玩遊戲的時候有沒有註冊小號的習慣，在貓和老鼠這款遊戲裡面註冊小號阿涼還是覺得有點好處的。不僅小號可以讓自己多一個活動的優惠名額，大家有沒有發現一個當代遊戲的“奇怪定律”？這個定律就是小號永遠都比大號歐，大號非常想要的皮膚小號總是可以輕鬆抽到。

大家好我是阿涼。

不知道大家在玩遊戲的時候有沒有註冊小號的習慣，在貓和老鼠這款遊戲裡面註冊小號阿涼還是覺得有點好處的。

不僅小號可以讓自己多一個活動的優惠名額，大家有沒有發現一個當代遊戲的「奇怪定律」？

這個定律就是小號永遠都比大號歐，大號非常想要的皮膚小號總是可以輕鬆抽到。

利用小號脫非入歐

阿涼玩過不少遊戲，真的碰到過不少奇怪的事情。

王者榮耀小號鑽石抽獎一次就拿到永久李白，陰陽師小號用系統贈送符咒抽到鬼切，貓和老鼠小號抽到細心。

不論阿涼大號是否是非酋，小號的運氣永遠比自己的大號高一個台階。

每次阿涼都知道要相信科學，但是回回都被小號更歐的定律治得服服貼貼。

之前一直有不少玩家，在阿涼的文章下面評論覺得自己太非了。

阿涼對此只有一個建議——開個小號重新試試運氣。

有小號的快樂是雙倍的獎勵

現在遊戲和以前的遊戲相比，最大的變化就是全民氪金。

誰玩王者榮耀還不首充買個趙雲了，誰玩貓和老鼠還不首充拿個國王了（肝帝除外）。

包括貓和老鼠推出的很多活動禮包，都包含了很多「新手福利」。

例如現在貓和老鼠遊戲裡面的萬聖節禮包，首次購買只需要花一元錢，還有概率能夠抽到永久皮膚。

不少玩家都說，自己充一塊錢啥也沒抽到太不服氣了。可是再繼續購買，就得花六元錢，想想還覺得有點浪費錢。

但如果有個小號就不一樣了，換個號再繼續購買萬聖節禮包也只需要一塊錢，就算沒抽到也不是很心疼。

吃貨要把斯派克一起推進洞口

昨天阿涼收到一張很可愛的投稿圖片，兩隻吃貨泰菲正在齊心協力推乳酪。但斯派克剛好躺在乳酪面前呼呼大睡。

而這兩個泰菲推乳酪的神情又十分認真，看上去是要連乳酪帶狗一起推進老鼠洞裡面的氣勢。

就是不知道，萬一斯派克突然醒過來會是一個怎樣的畫面？

今天的遊戲阿涼就分享到這裡啦！

既然今天的文章里，阿涼都分享了自己小號的「脫非入歐」經歷。大家今天也在評論區裡面分享一下自己貓和老鼠小號的神奇「歐氣」吧！

你們用小號抽到過最珍貴的東西是什麼？
