《最終幻想14》全球玩家破1800萬！5.2更新內容簡介

天天要聞
2019-12-21

《最終幻想14》是2010年發布的老遊戲，在不斷更新後逐步完善，如今依然很火熱，還在今年TGA上提名了「最佳持續運營獎」。而現在官方宣布：該遊戲的全球註冊總玩家數已經突破1800萬了！

　　在最新的製作人直播中，總監吉田直樹正式公布了這則喜訊，同時他也公布了一些5.2版本更新內容，以及2020~2021的一系列粉絲活動，和一個「暗影之逆焰」藝術集。

　　5.2版本「追憶的凶星」更新將包含以下內容：

　　· 新主線劇情：玩家需要前往虎人族的據點「波茲哈要塞」，完成一系列任務。

　　· 新四人副本：黑風海底「Anamnesis Anyder」，支持攜帶NPC助陣。需要注意的是，從5.2版本開始，開發團隊只會為後續的每一個新版本添加一個四人副本（之前一般是兩個副本）。

　　· 新討伐戰：紅寶石武器（極）破壞作戰。可在普通和困難難度挑戰隱藏BOSS。

　　· 新團本：希望之園伊甸（零式）：共鳴篇。

　　· 新編年史任務：維爾利得之傷戰役。

　　· 全新的裝備強化任務系統：將會不斷推出更新，提供強力武器作為戰利品。

　　· 以及伊修加德正教復辟、野獸部落任務、合成與採集系統重整、職業調整、PvP操作調整、額外的新遊戲+章節等內容。

　　《最終幻想14》可能在明年2月中旬更新5.2版本。
