DNF:搬磚裝備搭配從入門到精通講解，這份攻略請收藏好

2019-12-20 614條評論 2,635次閱讀 1173人點贊

上一期文章提到有關的搬磚冷知識說：「萬物皆可搬」，意思指的是所有職業都是可以搬磚的，既然如此，那麼有沒有一套適合所有職業的搬磚裝備攻略呢？聯繫搬磚的兩大要素，傷害與速度，如果裝備基本滿足這兩大要素的話，那麼制定一套適合全職業搬磚裝備未嘗不能實現。

考慮到裝備的成型難易度，再結合搬磚裝備的兩大要素，綜合來看，搬磚裝備的制定應該分為前中後期，三個分段的裝備制定計劃。

【前期裝備制定】

考慮到當前版本升級速度較快，一般當角色達95級時，我們才著手製作前期的搬磚裝備。這個時期的搬磚裝備因以製作全身聖物套為主，並且聖物套的等級為95級。

[裝備搭配解析]

初期裝備，傷害應放在第一位。95級的聖物裝備，相比其他聖物裝備，其效果更勝一層，不管在四維屬性的加成上還是在速度方面的加成，都是頂級的。更高的四維屬性可以為搬磚角色提供不錯的傷害加成，以穩定角色在搬磚時傷害趨於穩定，有利於更好的搬磚。而稱號選擇[無上傳奇]，角色覺醒時送的免費稱號，是根據此稱號的屬性選取的，可為角色提升物品欄負重上線值，就好比角色在搬磚時，把要裝物品的袋子容量進行了擴充，使得裝更多的東西，減少了處理物品欄的次數，也是間接的提升了搬磚效率。勳章選擇傳說，可以增加少量的傷害值，並且附帶的寶珠選擇加速度的，由於工會的改版，傳說勳章獲取可以說是沒有難度的，角色從1級升滿95級，期間足夠攢下1000工會硬幣，在用1000硬幣換取勳章後，多餘的可用來換取速度寶珠，運氣好得一個也不錯，沒有也不必強求。在前期時，時裝與寵物我們不必去考慮，因為前期的搬磚傷害已經足夠了。

這樣選擇的優點在於，裝備製作無成本，製作難度低，裝備成型快，缺點的話，雖然傷害是夠了，但是在速度方面還是顯得有些不足，但是前期的搬磚裝備做到這樣，也可以說是初級搬磚裝備畢業。當然，這樣的裝備，只適用于格藍迪，如果想用這套裝備去鋼鐵之臂試水的話，肯定是比較吃力的。

【中期裝備制定】

如上圖所示，中期搬磚裝備推薦【防具】哈林皮甲神龍之子套裝，【首飾】艾肯·守3件，【特殊裝備】艾肯·守3件，【武器】對應職業，哈林武器有有加速度的優先選擇，比如說鬼劍士就可以選擇哈林武器中的疾風劫，【稱號】選擇年套稱號，【勳章】保持不變。

[搭配解析]

中期裝備，應把目光從傷害轉移到速度上。根據裝備屬性，優選選取在速度加成方面較好的裝備，結合其裝備本身的成型難度，在防具的選擇上，哈林史詩裝備是一個不錯的選擇。哈林史詩製作相對比較容易，根據哈林史詩加移速最快的皮甲來看，總體有著34%移速加成，雖然移速加成效果非常不錯，但是不足的地方就是其觸發比較困難，在移速加成背後，它作為95級的史詩裝備，其也提供了不錯的傷害加成。製作難度、速度加成以及傷害加成3者結合，哈林皮甲值得選擇。

為了彌補哈林皮甲在觸發移速加成上的空檔期，把右邊的裝備全部換上艾肯·守，總件數達6件，為角色在皮甲空檔期中增加60%的移速加成效果，還是不錯的，考慮到這套裝備需要打哈林地區副本，所以把稱號也換上了加傷害類的，其中的年套稱號就是一個不錯的選擇。為了保證傷害的穩定性，我們更多需要把這套裝備進行適當的附魔打造一下。當前版本，附魔已經花不了多少金幣，所以，抽出個百把萬金幣附魔，我想應該是很容易滿足的。

選擇這套裝備的優勢在於，在速度還算不錯的前提下，可以進行哈林地區的摸金或者是搬磚操作，在搬磚副本中可進行多項選擇，不在單吊一個格藍迪磚廠。缺點，在傷害還是有點欠缺，不出意外的話，噩夢難度應該是不好刷的。在搬磚摸金結合下的撈金打法，傷害不足，也就意味著鳥背就不能就行單人摸金了，同樣的道理，單人旋渦、單人普雷也是只能放棄。

【後期裝備制定】

如上圖所示，後期裝備推薦【防具】精靈的榮耀套裝，【首飾】江山如畫套裝，【特殊裝備】還是選擇艾肯·守3件，【武器】和中期差不多，選擇疾風劫或者蒼穹系列武器，【勳章】不變，【時裝】為年套，【寵物】年寵。

[搭配解析]

後期裝備，追求的是速度與傷害的結合，一個也不落下。後期裝備更像是填補中期裝備的不足地方，哈林皮甲被換成了精靈的榮耀，從速度觸發不穩定的34%到單人觸發的22%，升級超界後，更是能把速度提升到32%，後者加上其它的攻速、施放速度以及全程霸體，為搬磚的舒適度又提升了一個檔次。首飾換成江山如畫，雖然比原來的艾肯·守3件少了15%的移速加成，但是確實滿足了在傷害方面的不足，為了減少速度被減慢造成的搬磚不良的體驗，特殊裝備處，最終保留了艾肯·守3件。年套的出現，也是確保角色的傷害，為了能去更高級副本摸金而做出的準備。當然，想打出更完美的傷害，基本簡單的附魔打造肯定是少不了的，附魔所花，與中期也是差不多的，這裡不要捨不得花金幣，年套都買了，一點附魔的錢也不算什麼。

這套裝備的優勢在於，搬磚體驗非常好，在施放速度以及霸體的作用下，搬磚手感是非常棒的，不足的地方就是裝備製作的周期很長，所以這套裝備才放到最後作為後期裝備來講的。

最後補充的地方：

上述是跟著裝備的成型難易度進行講解的，如果說還有哪裡沒說到，那麼就是裝備增加收益這一塊了。作為老玩家都知道，在裝備中，有一些裝備是可以增加搬磚收益的，比如說盜墓筆記-尋龍訣，85級超越巔峰稱號，以及貪婪戒指等，玩家如果有條件的話，可以在上述對應的推薦裝備中，把對應部位的裝備換成這些加收益的裝備也是可行的，具體看個人的搭配習慣了。
