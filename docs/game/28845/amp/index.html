因IP過於經典，內測前就被5星刷屏，夢工場大冒險終於開啟測試

天天要聞
2019-12-21

說到今年最成功的二次元手游，明日方舟這匹黑馬應該是避不開的，上線之後就備受關注，很多畫師也對其進行了再創作。

最近遊戲日報貓君就看到了有畫師將它跟《功夫熊貓》結合了起來，創作出了一些可愛的畫面，比如抱著阿寶的食鐵獸，學烏龜大師打坐的龜龜，教大龍叼尾巴的銀老闆……莫名的可愛！awsl！

圖片來源畫師：DullNoko

過了這麼多年阿寶他們依舊被粉絲銘記，可見《功夫熊貓》這部動畫的成功。而在《功夫熊貓》上映11年後則是又傳出了新消息，由夢工場正版授權的手游《夢工場大冒險》就在今天開啟了內測！

《功夫熊貓》便通過這樣的方式，與《瘋狂原始人》、《怪物史萊克》、《馬達加斯加》和《馴龍高手》四位小夥伴一起重新回歸！

於是，遊戲日報君在《夢工場大冒險》開啟內測的第一時間便下載了遊戲，想要一睹IP角色的芳容。

首先吸引我注意的便是《馴龍高手》中夜煞和小嗝嗝，作為遊戲中的SSR角色，無牙仔和小嗝嗝在一起沒有分開，就像動畫里的設定一樣。

畢竟不管是《馴龍高手》哪一部作品中，他們一人一龍都是不可分割的夥伴，遊戲這麼設計很細節很用心呀。

而在跑酷時，無牙仔更是無時無刻都在幫助小嗝嗝，比如飛在半空中用技能幫同伴清理障礙。

主動使用夜煞小嗝嗝的技能後，小嗝嗝還會騎到無牙仔身上叱吒風雲，召喚閃電……熟悉的動畫加上熟悉的技能，《夢工場大冒險》讓人倍感親切。

要說技能還原，遊戲日報君就不得不說自己最喜歡的神龍大俠了！在《功夫熊貓》中，阿寶在歷經磨難後，終於領悟了功夫的真諦，在與天煞的決戰中召喚出了金色的神龍。

而在《夢工場大冒險》中，則是將這條神龍完完整整的呈現了出來，霸氣無比。神龍大俠和天煞的大戰，也在遊戲開始的新手教程中以BOSS戰的形式出現。

而這個BOSS戰玩法也是《夢工場大冒險》的特色之一。在每一章劇情關卡的最後，都會有不同類型的BOSS出現，玩家需要操控角色邊跑酷邊打敗BOSS才能取得勝利。

以第一章最後的BOSS大龍為例，大龍的攻擊方式是「波動拳」三波齊發，還會突進騎臉輸出，繞到玩家身後，是個十分難纏的對手。

當然，玩家也不是只能挨打，可以通過撿取道具攻擊，或者觸發機關利用地形反殺。這中BOSS戰玩法操作拉滿，細節拉滿，為跑酷遊戲增添了無盡樂趣。

不僅僅是在不同的關卡最後還原了動畫中的大BOSS，在每一章冒險關卡中，《夢工場大冒險》都完整的將動畫的劇情加入其中。

以《功夫熊貓》為例，冒險關卡第一章 「大俠之路」便講述了阿寶跟著浣熊師父苦練武功最終成為神龍大俠的故事。

動畫中阿寶練功用的沙袋、木樁等道具變成了關卡中的阻擋物，還會從天而降和拔地而起！玩家需要及時反應採取行動才能跑得更遠。

現如今《夢工場大冒險》內測已經開啟，快去遊戲中重遇童年老友無牙仔、小嗝嗝、阿寶，感受這款情懷滿滿的遊戲吧！

《夢工場大冒險》安卓和IOS預約現在也正火熱進行中，各位玩家老爺們多多支持，說不定有望敦促項目組和《明日方舟》來一場真正的聯動哦！
