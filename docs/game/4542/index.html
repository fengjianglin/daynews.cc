小花生成全明星首個“自殺”選手 UZI坦言最想碰到shy哥

2019-12-06 0條評論 17次閱讀 0人點贊

摘要

在萬眾期待下，2019LOL全明星比賽在拉斯維加斯正式拉開帷幕。在經過一天的激烈角逐，我們見證了太多有趣的比賽，也見到了很多高光的表現。不過在首日當中，最受大家關注的應該就是個人solo的比賽了，畢竟走到最後的人，就可以成為今年的solo賽冠軍。

在萬眾期待下，2019LOL全明星比賽在拉斯維加斯正式拉開帷幕。在經過一天的激烈角逐，我們見證了太多有趣的比賽，也見到了很多高光的表現。不過在首日當中，最受大家關注的應該就是個人solo的比賽了，畢竟走到最後的人，就可以成為今年的solo賽冠軍。

大家都知道，在過往的全明星賽中，我們LPL賽區的AD選手UZI一連拿下了兩屆的全明星solo王的稱號，在去年也是由caps終結了屬於UZI的神話。今年誰能走到最後，也是非常讓人期待的。在首日的比賽當中，最受關注的應該就是我們LPL的四名選手了。

好在的是，在首日的比賽當中，LPL賽區的選手都成功的戰勝了對手了，獲得了晉級的資格。其中FPX的打野選手小天，在和對手小花生Peanut較量的時候，還發生了非常有趣的事情。那就是在賽前，兩人約定用盲僧這個英雄對決。

雖然天賦什麼的都相同，但是從出裝差異上，我們也看到了兩個選手的不同，小天則是一開始出了一個回血寶珠。並且在一開始的推線對決當中，小天是率先抓住了機會，所以也是拿到了推線優勢，不過不得不承認，小花生的表現也非常亮眼。

只不過兩個人都是使用的盲僧，在都主W的情況下，這場比賽還是進行到了最終的縮圈環節。整個比賽過程，相信大家都看到了，可以說是非常精彩，不過小天在整場比賽中確實佔據著主動性。在最後一波較量當中，隨著縮圈到來，小花生原本是想要利用R閃將小天踹回防禦塔下，但是效果並不好，之後小花生因為兵線劣勢而且殘血，最終在小天掛上點燃之後選擇在圈過來的時候自殺輸掉了比賽。

雖然小花生沒有拿下比賽，但是卻獲得了一個有趣的稱號，那就是全明星首個「自殺」的選手。不過大家都知道，那樣的情況下，小花生已經是必輸之局了，所以他的做法也是非常坦蕩。另外，在下賽季，小花生將要代表LGD戰隊出戰，也是非常期待在春季賽當中，看到他和小天的再次交手，到時候一定會更加精彩。

其實在首日的solo賽當中，並沒有出現冷門，諸如Faker、Bang、UZI、The Shy和Doinb等選手都紛紛晉級，而第二日的比賽將會成為最關鍵的存在。而且值得一提的是，在UZI拿下晉級資格之後，在採訪環節，當余霜問到在這次的Solo賽上最想碰到的對手時，Uzi很坦然的回應說是Theshy，因為大家都說Theshy很厲害，所以他想見識一下!

不知道UZI的這個願望都否實現，但是可以預見，在第二日的比賽當中，我們一定能夠見證到更多精彩的較量，也能看到職業選手們爆發更多精彩的操作。
