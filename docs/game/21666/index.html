LPL四大AD三位去向已塵埃落定，“失蹤”的阿水現身上海機場

2019-12-18 3,477條評論 4,645次閱讀 2016人點贊

熱血電競，品味人生。An妮遊戲說為您播報，期待您的關注！

12月16日，2019LOL冬季轉會期正式宣告結束。在今年的轉會期中，LPL四大AD選手中的三位去向已經塵埃落定。我們都知道LCK賽區盛產頂尖中單，例如Faker、Pawn等選手。而我們LPL則盛產頂尖AD選手，比如Uzi、阿水等選手。而Uzi、阿水、Smlz和iBoy這四位AD選手也大家被稱作「LPL四大ADC」。

Uzi與RNG完成續約

在12月15日，一直處於「憋氣」階段的RNG終於有所動作。RNG俱樂部官宣Uzi、小明，以及小虎全部成功續約，他們將以核心隊員的身份，身披RNG戰袍參與接下來的比賽。小狗、小明和小虎三人續約RNG，加上小龍堡的到來，相信RNG在S10會大放異彩。

Smlz回歸OMG

說完Uzi，我們來說一說Smlz。馬哥和Doinb是前隊友，兩人也被稱為俠隊一代目，而在S9賽季兩人分道揚鑣。Doinb在加盟FPX後奪得S賽冠軍，可謂是功成名就，而馬哥在SN卻並未取得理想的成績。為了站上世界賽舞台，也為了不讓自己的職業生涯留下遺憾，馬哥決定回歸OMG。因為這裡正是馬哥憑大嘴「湮滅」一戰成名的地方。這次馬哥重返OMG，相信他一定不會讓支持自己的粉絲失望。

iBoy加盟VG

小昭iBoy是LPL賽區出了名的天才型ADC，曾經一度被很多人認為將會超越Deft。在去年S8全球總決賽的舞台上，iBoy憑藉超強反應速度圈粉無數。不過可惜的是，他今年的發揮相比去年，不但沒有進步反而退步了，而且在關鍵局的發揮也不理想。EDG在官宣明凱轉型成為戰隊主教練後，隨後就官宣iBoy離隊。而iBoy在離開EDG後，選擇加盟了傳奇教練Kkoma教練所在的VG戰隊，希望iBoy可以在VG有所收穫。

現在LPL四大ADC中，Uzi、iBoy和Smlz的去向已經塵埃落定。唯獨冠軍AD選手阿水仍不知去向。在iG官宣VG下路雙人組加盟後，我們還是沒有聽到有關阿水Jackeylove的消息，之後「阿水去哪兒」更是衝上了微博熱搜，阿水的去向也引發了網友的熱議。

阿水現身上海機場

在轉會期結束後的第二天，阿水現身上海機場，還被被粉絲認了出來上前索要合影。而這個消息的傳出，讓網友猜測阿水轉會的事即將有結果，畢竟在轉會期剛結束的時間點，回到上海本身就是一個很不同尋常的舉動。

在轉會期結束後，各個戰隊都公布了自己新賽季的大名單，那些積極競爭阿水的熱門戰隊似乎都未能如願。在SN公布的大名單中沒有阿水的名字，TES也提拔了二隊的AD選手，而EDG公布的名單也沒有阿水，因此在網友的猜測中阿水剩下的選擇僅有兩個。

第一種可能是續約iG！因為在轉會期iG一次性引入了兩位輔助選手，先簽了VG的輔助南風，緊接著又把二隊的輔助Fate也被提上了一隊。要知道寶藍還在，而iG卻依然選擇增加兩位新輔助，這很有可能是為了挽回阿水的一種方式。至於第二種可能，則是近期由LCK記者爆出的轉會RNG，不過這個可能性相對較小。

今年的德瑪西亞杯將在12月21日正式開戰！最後，希望阿水早點現身，不要讓粉絲們再等下去了。那麼，大家覺得阿水會續約還是轉會RNG呢？歡迎大家在下方評論區留言討論，說出你的想法！
