●卡組推薦

◆手牌龍術

kolento的一套手牌龍術。術士迦拉克隆體系被K神證明非常弱，祈求牌質量太低，迦拉克隆還看運氣，前中期丟失的節奏和血量根本彌補不過來。但是術士新版本並不是沒救，k神組的這套手牌龍術戰績是非常可觀的。

構築上加入龍體系，二費虛空吐息有龍就是二費打四，瘋狂的靈翼龍有龍5/5自帶地獄烈焰，八費橙龍是卡組後期核心，配合過期貨物專賣商和護魂者拿個7-8個，就是一回合一個4/12+66，對面解不掉4/12還能繼續滾雪球。

瓦迪瑞斯、深淵召喚者、山嶺是中期質量生物，瓦迪瑞斯非常厲害，K神都是直接留。三費解牌黑暗天際也是超強，起手就有三費打7或9，防快攻一流。

◆樹人龍德


Kolento的一套樹人龍德。這次的官方欽定好像真的很強，也可能是K神構築的好，直播中感覺格外厲害。構築上龍與樹人結合，龍加入的是翡翠龍探險者，覺醒伊瑟拉，深藍系咒師和辟法巨龍這類高費龍牌，前期有樹人不需要龍，這些龍除了作為中後期質量生物外，主要作用還是保證手上有龍，來配合夢境吐息，二費能打出夢境吐息跳費的德魯伊都？因為有了成長，滋養也加入了構築。

樹人牌除了前期場面厲害，中後期配合樹人橙卡和暗影崛起的森林的援助也不錯，雖然沒帶咆哮與buff，八費五個3/3（需要打出樹人橙之後）還是猛的。

◆職業牌牌騎


kripp的一套職業牌牌騎。今天凌晨發布的新卡可能太多了，真正的聖騎士強卡大家沒有注意到，他們就是光鑄狂熱者和光鑄遠征軍，這才是真正帶動聖騎士套路的神卡，四費4/2牌庫沒有中立牌裝備大寶劍，7費沒有中立牌隨機給五個聖騎士牌（還可能給自己），因此職業牌牌騎出現了，全是職業牌（本身聖騎士職業牌質量就高）。

前期格洛頓、微型木乃伊、沙漠蜂后配合奧秘和低費法術站場搶節奏，四費光鑄狂熱者強無敵，三費空中飛爪也是猛的不行，鋪場還能給機械加攻（可能也沒啥人注意它），就這樣前中期節奏搶到，壓力給到，七費遠征軍補牌續航，佛丁一拍直接壓死就完事。

◆元素奇蹟法


龍王的一套元素奇蹟法。沒有前面三套卡組驚艷，是在原來奇蹟法基礎上的補強，滿編了兩個支線任務，學習龍語和元素盟軍，五費火球滾滾作為解場手段，灼燒、水元素、元素喚醒和希亞瑪特作為元素站場鏈。

核心思路還是巫師學徒、法力颶風和齊恩瓦拉的combo，今天還沒見主播玩宇宙龍法，明天有的話一定搬。
