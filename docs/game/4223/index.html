KPL職業聯賽發展至今，已經有幾年時間，參與其中的隊伍確實也不少，但能夠被長時間稱之為強隊的並不多，甚至說能夠一直在中上游水平的隊伍也不多，畢竟遊戲版本不斷的更新，所有的職業戰隊都要去適應新版本，一旦跟不上就會面臨墊底的局面，其實墊底倒沒有什麼，如果說一直在中上游，但總是不能夠捧起冠軍，對於戰隊選手也是一個非常沉重的打擊。

為什麼要說是打擊呢？因為他們站到了高處，距離登頂是非常近的距離，但一直都沒有突破到頂峰，正所謂希望越大失望越大，中上游水平會非常尷尬，在KPL的隊伍中，EDGM、RNGM兩個電競豪門，從加入到現在一直都保持著比較不錯的水準，尤其是RNGM戰隊，印象中他們狀態一直都比較不錯，但總是沒有奪冠的希望。

尤其是2019年，他們終於有機會奪冠，走到了總決賽的舞台，距離夢想只是一步之遙，但最後還是被eStarPro從神壇上拉了下來，強行讓RNGM變成了背景板，從春季賽之後，RNGM好像進入了一個分水嶺，狀態逐漸開始了下滑，隊伍也進行了各種嘗試，暴風銳都多次坐上了替補席，但效果一直不好，前段時間老黑教練也宣布離開了RNGM戰隊。

接下來不管是哪一個教練加盟，陣容肯定都會進行調整，只是目前並沒有準確消息傳出來，近段時間有消息曝光虔誠下個賽季，會掛牌轉會。一時間引起了很多網友的關注，紛紛猜測接下來的RNGM是不是會進行大換血，但之後RNGM的前輔助選手一笑，在直播中公開進行了澄清，稱消息並不真實。

但說實話也並不一定，虔誠目前在RNGM多次錯失奪冠的機會，如果轉會確實是一次改變的機會，這種可能性並不能否認，一笑站出來闢謠或許是因為接下來的冬季冠軍杯，冬季冠軍杯RNGM確實不可能有太大的變動，尤其是核心虔誠。

但如果冬季冠軍杯成績依舊不好，虔誠轉會的可能性會不會更大？當然轉會的風險比較大，如果一個不小心，整個職業生涯都有可能葬送，留隊也是不錯的選擇，比如說之前的刺痛，備受質疑之後，依舊堅守在QGhappy，現在不也是迎來了春天？轉會與否更重要的是虔誠如何去選擇，不管怎樣還是應該選擇支持他！
