《雨中冒險》遊戲測評：3D射擊與Roguelike的一次成功的結合

天天要聞
2019-12-16

提起Roguelike你們會想到什麼呢？，是硬核愉悅打法多變的《殺戮尖塔》？手感勁爆的《死亡細胞》？還是尋龍探穴轉手射爆的《以撒的結合》？但無論是哪款Roguelike遊戲，它們給我帶來的核心元素往往都是卡牌遊戲，2D射擊或者是2.5D。的確之前少有遊戲將3D、射擊、Roguelike這三種元素結合在一起，因為3D和Roguelike意味著太多不確定因素，難以控制的手感稍不留神就會被玩家噴的體無完膚，所以市面上基本沒有這種遊戲，而之前大火的《雨中冒險2》則讓玩家驚喜連連。

遊戲開始界面

3D射擊與Roguelike的成功結合

主打3D的Roguelike的遊戲之前的確少見，更不必說還有射擊元素的加入。《雨中冒險》在一代中採用的2D像素點陣的形式，而在二代可以說是跨時代升級。峰巒疊起的山包，沙塵縱橫的遺迹，隱秘奇妙的神壇，炫酷多變的天空無一不是採用對比鮮明的3D效果製作而成。

更新路線

在畫面升級的同時，遊戲並沒有放棄簡潔的戰鬥界面。金錢血條、時間目標、技能裝備遺迹怪物的血槽等元素都非常直觀的顯示在屏幕上，讓玩家能第一時間了解掌握自己的狀態和遊戲的進程。如果說《只狼》是把玩家直接扔進沸水中，那麼《雨中冒險2》則是溫水煮青蛙了。

遊戲界面

遊戲里可沒有時間讓你閑逛

遊戲每關的目的是找到場景中存在的傳送門，殺掉BOSS等待傳送門充能完成即可進入下一關，但隨著時間的流逝，遊戲的難度會慢慢變得越來越大，倘若你把大部分的時間用在散步、賞景而非尋找裝備提升自己能力的話，慢慢你就會發現自己的成長曲線遠遠落後於關卡難度，因此直接被強大的BOSS一擊秒殺。所以在這款遊戲中爭分奪秒的提升自己進入下一關才是最佳選擇。

隱藏的道具商店

但或許是因為畫面轉為3D的緣故，在新作中，傳送門極其難找且無任何提示，甚至連反光閃光效果都沒有，更不必說小地圖了。有的只是右上角的一句”找到傳送門”的雞肋提示，這確實是苦了玩家們，尤其是那些本來就路痴的玩家，畢竟找到傳送門是唯一的過關方式，在時間如金子般寶貴的遊戲中，又有誰會願意因尋路而浪費時間呢？

道具和裝備

FPS動作遊戲愛好者們不會失望

對於那些FPS和動作遊戲的愛好者們來說，《雨中冒險2》的表現並沒有讓他們失望。遊戲採用五種人物不同技能，綁定滑鼠左右鍵以及RQ兩鍵的設定，讓每一位玩家都能靈活掌控不同角色的特點。同時武器的打擊感也相當不錯，後坐力市中的射擊手感還有子彈擊中敵方的一瞬間，硬直以及爆炸特效都給這款遊戲加分不少。

士兵的針筒

遊戲雖然沒有多樣的武器設定，但擁有著拾取裝備提升武器傷害及效果的設定。例如幾率眩暈敵人，命中幾率吸血、幾率暴擊等，這些裝備大部分繼承了前作，也有部分是本作添加的全新裝備，還有主動技能就更炫酷了，新老玩家的體驗可以說是上升了一個檔次。當玩家成為多種道具為一體的裝備帶怪物的時候，那些開始對這款遊戲嗤之以鼻的玩家一定會說出一句”真香”！試問這爽快的感覺又有誰能拒絕呢？

爽快打擊感也掩蓋不了的缺點

但恐怕遊戲最大的魅力不止於此，而是當你和夥伴們一起坐在電腦旁”找門”的感覺。一大片怪物在你的身邊，你們沒有絲毫慌張，兩個默契極佳的夥伴背對背解決所有敵人的時候，就像《史密斯夫婦》結尾部分那令人難忘的屠戮情景，至少這一刻我認為自己是無敵的，你們隨便！

死亡後顯示本輪遊戲的數據

遊戲整體現正處於體驗階段，內容尚待充實，不過它已經成為了3D設計類Roguelike遊戲中少有的成功典範，甚至這隻小蝴蝶很有可能帶起這類遊戲的巨大風暴。就像當初的《絕地求生》一樣，我期待著製作商有一天可以帶來他們的承諾，期待著他們能夠帶來更多的角色、BOSS和多模式設計，期待著2代能開放創意工坊集結各國大神的創意，更期待著這個最初只是學生項目的《雨中冒險》能像遊戲中的角色那樣逐漸強大起來。
