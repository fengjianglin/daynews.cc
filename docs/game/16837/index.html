夢幻西遊：狙擊珍寶閣王朝9連冠霸業，釣魚島豪取161聯服戰冠軍

2019-12-16 3,007條評論 4,176次閱讀 2735人點贊

大家好，我是如風，（微信公眾號從「如風夢幻西遊百科」更名為「如風說夢幻」）！

回顧近2年的武神壇，神奇的12月份，一切皆有可能，2017年是拉閘門，2018年是天地門，2019年的12月註定不會平凡。在16進8的複賽中，珍寶閣、西柵老街、紫禁城、姑蘇城這4大豪門相繼出局，最終釣魚島和齊雲樓會師了最後的決賽，並且上演了久違的90068內戰！

周六小組賽的第一場比賽，釣魚島和齊雲樓就是對手，到了最後的決賽，對手還是他們，這是巧合，或許也是冥冥之中自有天意。無論是釣魚島還是齊雲樓，都擁有大量的粉絲。

比賽開打之前，釣魚島粉絲在老王的帶領下，紛紛向職業背鍋30年的呆瓜道歉，二狗的軟飯家族和渣男軍團，更是喊出了「一月兩冠，就是干」的霸氣口號。

釣魚島2億餘額醉奶先生，更是將賽前的氛圍帶到了最高潮，直接送出了5萬元的紅包。無論是誰奪冠，他們都將成為2019年武神壇全新的服戰冠軍。

在BAN選階段，雙方都盯准了「化生寺」這個門派，無論是咒師普陀還是花果山，都非常需要這個輔助門派的配合。釣魚島方面，化生寺和花果山被禁選，齊雲樓方面，化生寺和陰曹地府被禁選。從BAN選結果來看，釣魚島方面的最強國標點殺陣容被禁。

切入對戰之後，釣魚島方面復活了花果山，直接選擇了「花果山+神木林+女兒+無底洞+地府」的陣容進行衝刺，齊雲樓方面求穩，選擇了「花果山+女魃墓+女兒+盤絲+化生」的雙封陣容！

開局第2回合，釣魚島方面直接點殺女魃墓成功，高傷善惡寶寶的表現非常亮眼！

10回合之後，釣魚島憑藉瘋狂的連環點殺，導致齊雲樓方面4人倒地！

齊雲樓陣容方面被嚴重克制，如今的武神壇PK版本，地府依舊是絕對的王者！13回合之後，比賽直接結算，釣魚島在2019年奪得群雄逐鹿冠軍之後，再次奪得服戰冠軍!

2019年的武神壇，珍寶閣服戰隊一家獨大，在西柵老街之後，我們終於迎來了全新的服戰冠軍，時隔1年之後，釣魚島複賽中狙擊了珍寶閣，最終再次登頂，12月服戰王者必有一冠！

我是如風，我為自己代言，點點關注不迷路，聊夢幻，聊服戰，小編是最專業的，內容全部原創。
