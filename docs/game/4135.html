為求奪冠離開WE，Xiye遭各大戰隊哄搶，疑似選擇加入DMO

2019-12-09 0條評論 1次閱讀 0人點贊

摘要

S9世界賽結束後，為了能夠在家門口舉辦的S10世界賽中奪得桂冠，LPL各支戰隊都開始了針對性的引援計劃；只可惜lpl賽區的轉會期就像一個憋氣大賽，每支戰隊都恨不得“憋”到最後一秒鐘再宣布自己的下賽季陣容；比起尚未公布的引援，大量投身自由市場的知名選手更受到觀眾們的矚目。

S9世界賽結束後，為了能夠在家門口舉辦的S10世界賽中奪得桂冠，LPL各支戰隊都開始了針對性的引援計劃；只可惜lpl賽區的轉會期就像一個憋氣大賽，每支戰隊都恨不得「憋」到最後一秒鐘再宣布自己的下賽季陣容；比起尚未公布的引援，大量投身自由市場的知名選手更受到觀眾們的矚目。

英雄聯盟S10

近兩年來，隨著上單957狀態下滑，打野condi因劣行不改轉會離開，ADC Mystic狀態起伏不定，加之俱樂部引援不利等因素，WE戰隊的實力逐漸下滑，今年夏季賽在賽季中段發力的情況下，依然沒有能夠進軍季後賽。這也導致Xiye做出了不再續約的決定，希望能夠尋求有野心有目標規劃的戰隊衝擊冠軍。

xiye

夕夜宣布正式成為自由人，這個消息讓各大戰隊的粉絲們都是十分期待的，畢竟成名的國產中單本就不多，像夕夜這種在世界賽上證明過自己的就更少了。特別是RNG的粉絲，都希望戰隊能夠買來夕夜和小虎進行中單輪換，競爭上崗。畢竟在他們印象里的夕夜，還是那個斬獲S7春季賽冠軍，在S7世界賽場上敢打敢秀，單殺Faker的進攻型中單。

然而，冬窗進行至今，大部分缺乏中單的隊伍，都進行了強力引援。就連最接近引進Xiye的LGD，也在一番斟酌後選擇了前LNG中單Fenfen。直到今天，轉會市場上才有了夕夜去向的一點蛛絲馬跡。

微博爆料

在今年冬窗，爆料準確率極高的Score賽高在昨日發布了這樣一條消息，消息中明確表示DMO簽下了一位國產中單。結合目前市場上國產中單不多，以及賽高爆料相關新聞時一般會註明（是/不是夕夜）來看，DMO簽下Xiye來進行補強的可能性並不低。

目前，DMO戰隊方面還沒有對這條信息作出回應，夕夜的轉會下家依然不明朗。總之，關於夕夜的轉會，大家更期待他加入哪支戰隊呢？
