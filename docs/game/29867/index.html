修真江湖12月26日正式上線！一朝入道修坤乾，兩瓢濁酒盡余歡

2019-12-23 2,144條評論 4,736次閱讀 581人點贊

話說阿茶也算是文字修真遊戲的老玩家。像什麼指尖修仙，修真凡人傳說，永恆仙途，破劫成仙，放置修仙什麼的也玩了一大堆了。所以今天我要介紹的遊戲就是12月26日即將上線的文字類角色扮演遊戲《修真江湖》！

修真江湖一款快意江湖的掛機手游。遊戲中以經典的武俠戰鬥模式來進行。多樣的文字說明，讓玩家更加有趣的進行冒險。

遊戲特色

1.全程自動掛機

手動打怪、手動挖礦、這裡統統不需要。礦是自動生產的，怪是你吃飯，睡覺，看視頻時，就被收拾的服服帖帖。不知不覺中你就獲得千年修為，成為大神。而這一切你只需要偶爾來收收資源，上線體驗遊戲關鍵歷程就好了。

2.功法組合自由

多個門派各有特色，且都有自己的禁地鎮派之寶。渡劫生涯皆同道，修成正果各不同。你作為修仙界的後起之輩，是劍術合一，專註防禦，還是控法大師，全看你怎麼自由組合這些功法。說下門派吧，門派有七個，感覺有點少。

還有就是每個門派只有一個心法三個攻擊術法兩個防禦術法還是太少了。期待更多強大的功法！

3.遊戲內容豐富

洞府養成生產各種資源。三大副本（常規，劇情，秘境）等你來挑戰。斜背一把劍，你可遊歷天下，發生各種奇遇。說說靈獸功能，這遊戲靈獸功能使用門檻低，基本在副本得到靈獸蛋就能使用，這是我在別的遊戲沒遇到過得。再說煉丹煉器，總的來說學習煉丹煉器術，獲得丹爐器爐沒什麼難度，材料獲取也不太難獲取，獲取的量也適當，不至於丹藥什麼的太泛濫。

4.原創修真劇情

這裡不只有玩法，還有故事。你將經歷一場修真劫難，多少愛恨情仇，悲歡離合，皆呈現在你眼前。你的一個抉擇將影響未來。

5.靈根設定

靈根並不天生註定，遊戲里靈根均是可以升級的，學習功法時也有相對應的靈根需求，所以並不是修為足夠就可以學遍所有功法，如果靈根不跟上，也是很難成為門派的翹楚的。

本期介紹就到這裡啦，想要體驗的玩家趕緊來預約下載修真江湖吧！
