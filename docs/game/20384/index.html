LOL：LNG官宣原SN中單Maple、原TES打野Xx加入

2019-12-16 197條評論 1,582次閱讀 391人點贊

北京時間12月15日晚，在LPL冬季轉會期倒數第二天，LNG戰隊終於官宣了全新中野組合：原SN中單Maple和原TES打野Xx加入，隨後TES也官宣了效力兩年的功勛打野Xx斷開連接。

LNG官宣新中單Maple：

勤中得道，可鏤金石。

或許曾經的努力還未看到成果，但是持續的努力終將得到回應。

今天我們很高興的向大家宣布，前SN中單選手Maple（黃熠棠），正式加入LNG電子競技俱樂部英雄聯盟分部，與LNG其他隊友攜手征戰。

作為一名有著豐富大賽經驗的中單選手，Maple有著出色的團隊協作能力以及優秀的職業競技態度，期待新賽季他的加入，能夠讓戰隊的整體性和協同作戰能力發揮到最佳。

未來，Maple將與其他隊員共同努力，向全新的賽季和更高的榮譽發起衝擊，我們相信，一切皆有可能。

LNG官宣新打野Xx：

路漫漫修遠，千帆過沉舟。

未來還未來，未來在未來。

今天我們很高興的向大家宣布，前TES打野選手Xx（熊宇龍）正式以轉會的方式加入LNG電子競技俱樂部英雄聯盟分部，與LNG其他隊友攜手征戰。

在過去的幾個賽季，作為一名優秀的打野選手，無論是良好的野區節奏，還是冷靜的指揮能力，Xx都曾給我們留下了深刻的印象。期待他的加入，能夠激發隊友間全新的化學反應，也為LNG的新賽季帶來更多的可能。

未來，Xx將與其他隊員共同努力，向全新的賽季和更高的榮譽發起衝擊，我們相信，一切皆有可能。

隨後TES戰隊也官宣了Xx正式離隊：

【TESXx正式斷開連接】

滔搏電子競技俱樂部英雄聯盟分部打野選手Xx熊宇龍從即日起正式轉會至LNG電子競技俱樂部。

兩年間，Xx與滔搏相伴不斷進步，突破自身並帶領見證著隊伍的成長，感謝Xx為滔搏所付出的一切。

希望小熊未來的職業道路一切順利，永遠心懷赤誠，循夢前行！@Xx970618 再見祝好，下次賽場相遇都要展現更好的自己！

Xx作為近兩年來從LPL脫穎而出的國產強力打野，在TOP/TES效力時期，以肉眼可見的成長速度迅速躋身LPL一線打野，成功幫助TES獲得今年LPL春季賽四強的成績，並代表LPL賽區參加了洲際系列賽亞洲對抗賽。在他和Maple加入LNG後，目前LNG的陣容為：

上單—Flandre

打野—Xx

中單—Maple/Plex

ADC—Asura

輔助—Duan

Maple從LMS賽區的FW時期到今年LPL賽區的SN時期，是一位征戰職業賽場多年的老將，經驗豐富且風格穩健，相信他的加入能夠幫助LNG在明年LPL春季賽打出更好的成績。小夥伴們覺得現在這支LNG的戰力如何呢？
