LNG換對了人？新陣容德杯強勢2連勝

天天要聞

★遊戲馬蹄鐵原創

OMG對戰LNG

今年轉會期兩支變化挺大的隊伍，OMG清除了過去太多無用的名額，留下了Icon，招募回了Smlz，順帶著還從SNG簽下了打野H4cker，中下野的新組合讓OMG煥然一新。而LNG方面在中野上也進行了較大的變化，從TOP簽下Xx，從SNG簽下Maple，保留本賽季的上下三位選手。同樣都是在中野方面變化較大的兩支戰隊，德杯上的相遇戰果如何呢？

LNG神仙走位拉扯團滅OMG！

雙方在前期掌控小龍上都很出色，對於下半區的注重沒有讓一方太早拿到龍魂。打到20多分鐘各拿2條小龍，而LNG通過一波野區的團戰神仙走位拉扯團滅OMG拿下大龍，在經濟上有了較大領先。而且陣容方面，鐵男的關人和潘森的強開威脅也越來越大。

OMG失去被反打，LNG一波拿下勝利

OMG辛德拉在中路草叢埋伏擊殺了打野，但是想要更多強行越塔被LNG反打擊殺三人，隨後LNG趁勢一波結束比賽拿下勝利。補強了中野的LNG，在這次德杯上目前已經取得了2連勝，狀態不錯。Xx和Maple的入隊，的確取得了非常好的團隊效果。聖槍哥在上半區也終於不再是孤軍作戰了。加上原本阿修羅和小段就很出色，這支LNG值得期待！
