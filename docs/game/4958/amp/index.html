魔獸世界：風哥都玩不過的公會白打工兩月還被工會強制摧毀裝備

天天要聞
2019-12-03

摘要

想必懷舊服開了這麼久不少玩家已經充分享受到他的魅力了，作為老玩具齊聚的懷舊服雖然開服的時候顯得是“溫文爾雅”，但經過這段時間的發酵之後各種毛人毛團是屢見不鮮。最近一位用戶甚至表示自己在DKP團裡面白打工兩個月不算，本著好聚好散的原則在正分退會的時候還被管理要求摧毀裝備才能退團。

想必懷舊服開了這麼久不少玩家已經充分享受到他的魅力了，作為老玩具齊聚的懷舊服雖然開服的時候顯得是「溫文爾雅」，但經過這段時間的發酵之後各種毛人毛團是屢見不鮮。最近一位用戶甚至表示自己在DKP團裡面白打工兩個月不算，本著好聚好散的原則在正分退會的時候還被管理要求摧毀裝備才能退團。下面我們就來看看這究竟是怎麼回事吧！

一位牧師玩家在國慶後回歸了懷舊服，之後認識了一位《第二世界》公會的成員。在他人的利誘下（二團參加活動，進團送100分）加入到這個公會進行活動。在這段時間（10月23至11月26號內）跟隨工會2團進行了總共6周的團本。但由於特殊原因（掉線排隊）導致自己缺席了一次MC，但本著集體活動的原則在小號告知團長請假得到了允許。而黑龍則是由於公會公告沒及時更新才缺席的。

在最近一次MC里由於MS的肩膀被頂到85分自己有點不開心了，畢竟這個公會MC一次才能獲得17分。這樣一算自己一件裝備需要5個CD才行，畢業的話實在太慢了。在和管理溝通表示自己想先去G團買裝備，等提升後再報名公會BW首殺L團。但是公會管理安度因表示退團要白打2個CD不拿裝備，因為公會規定補夠100分（當時餘下78分）才算正分退會。並且表示入團加的100分屬於借貸，必須補齊才行。事後管理安度因甚至表示不管你接受不接受，認同不認同，這個就是工會規定，必須執行。

隨後在28號該牧師詢問毀滅之刃其他會在招收治療的玩家和工會管理層，均表示補貼的DKP是穩定後就送的，沒有借貸這種說法，只要你不負分退團沒事。最讓牧師玩家難受的還是28日晚上會裡的另外一個MS私聊表示你退團了就是退會了，我們不再是朋友了。然後該玩家情緒上來了就退會了。在該牧師玩家退會之後，拉他入團的管理表示5G1分他補償給工會，然後該玩家自己補償工會200G。

但《第二世界》在公會公告污衊該玩家否認DKP制度，希望大家注意此人。隨後在公會YY溝通的結果是在公會拿的兩件裝備摧毀掉，但公會需要按市場價補償該玩家5CD的G並且刪除公告，但事情結尾卻是按照1分5G補償該玩家245G。個人看完這件事情覺得這位牧師玩家確實有錯，入會前沒有了解清楚《第二世界》公會規章制度，尤其自己身為治療職業還去DKP團打團。更讓人接受不了的還是明知道一個MC攏共才17分還打了這麼長時間。

不知道大家對於此事有何看法呢？
