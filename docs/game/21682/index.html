陰陽師SSR雲外鏡技能速報：515防禦系式神，鏡像血條無法被治療

2019-12-18 1,173條評論 3,716次閱讀 863人點贊

周二深夜，陰陽師官方終於公開了12月新式神——SSR雲外鏡，以及對應的覺醒前後立繪、式神PV，周三更新後，這位新式神上線百鬼弈，讓大家提前看到式神面板和技能，雖然技能是罕見且「親民」的515，但從這描述來看，又是一場閱讀理解考試。

首先來看看雲外鏡的面板，SDSAS，其中攻擊、防禦和暴擊是S，從後面要說的技能來看，雲外鏡攻擊和防禦雙S不過分，況且他還能做輸出式神，所以暴擊S沒問題。

（提前說明：由於雲外鏡有兩種形態、兩套技能，所以下面技能後面的名稱，在前的是陰狀態技能，在後是陽狀態技能）

普攻「落影」/「曇無」

陰狀態的普攻一般，只是造成攻擊傷害，而陽狀態就不太一樣，傷害是看防禦，造成防禦的200%傷害（滿級360%），而且還會讓行動條最前端的另一友方增加20%行動條，相當於是一個拉條技能。

二技能「斗轉」/「昭回」

在說二技能之前，先把初始狀態如何判定陰陽狀態的方法說了，戰鬥開始時，如果雲外鏡初始攻擊高於初始防禦的500%（5倍），則進入陰狀態，否則進入陽狀態，也就是說，讓雲外鏡開局陰陽是可控的，只要配置御魂的時候，注意一下攻擊和防禦的倍數。

按照雲外鏡初始面板來看，不帶任何御魂的前提下，戰鬥開始會直接進入陰狀態。

雲外鏡這兩種狀態，陰狀態額外增加初始防禦200%的攻擊，陽狀態下額外增加初始攻擊10%的防禦，而且陰陽狀態生命值互補，如果在某一個狀態生命值降低，那麼另一個狀態生命值會增加，特別注意，陰陽生命之初始各一半，這也就是之前PV中看到的，生命條一黑一白的由來。

這裡生命條初始各一半，不知道是怎麼判定，如果算作只剩一半血，那麼對鬼切來說，豈不是美滋滋？技能一放雲外鏡可能沒了不說，隊友也得吃到鬼切的被動，這個問題可能得等測試了。

結合「鏡中世界」這個被動來看，雲外鏡無法受到治療恢復生命，只會將治療效果20%轉化為持續兩回合的護盾，陰陽狀態生命條設定來看，雲外鏡最佳的治療自己的方式就是狀態互換，然而並不是說殘血互換生命值是無限次數的，因為二技能限制了……

主動釋放這個技能之後，不管你是陰轉陽，還是陽轉陰，總之只要用一次，鬼火消耗永久+2，整個鬼火條才8格，說明一局中最多用5次（0、2、4、6、8）。並且轉換狀態，能提升自身100點速度，以及獲得鏡盾，也就是能吸收雲外鏡生命值上限25%傷害的，持續兩回合的護盾，又是盾。

三技能「雲岸凈空」/「苦海浮生」

陰狀態下的三技能是偏向於輸出，先是AOE131%（滿級：145%）傷害，隨後會對敵方目標造成單體131%傷害（滿級：145%）。如果陰狀態雲外鏡生命比例每降低30%，那麼會額外追加一次單體傷害，且有100%基礎概率附加「鏡怒」，在對方下一次獲得1個可驅散buff時抵消。

陽狀態下的技能更多偏向於輔助，能使1位友方獲得「鏡佑」，最多兩位，持續至回目結束，還能讓友方全體增加20%行動條。獲得鏡佑buff的友方，能在受到減益、控制debuff的時候，立即驅散1個（滿級：2個），並且增加10%行動條（滿級：20%），每回合最多觸發一次驅散，且觸發時候能治療生命值最低友方雲外鏡180%防禦的生命（滿級：360%），一旦失去鏡佑，就能獲得鏡盾。如果陰狀態雲外鏡生命比例每降低30%，那麼該技能消耗鬼火-1點。

總結陽狀態的技能就是，友方能獲得一個驅散，驅散起效之後能治療友方且拉條，驅散buff失效後還能補一個護盾效果。

綜合雲外鏡的技能，應該是一位能在PVE和PVP都有作用的式神，陰狀態的AOE+單體輸出，能帶狗糧（困28跳跳妹妹不成問題），魂十一配合鬼吞？PVP就沖這個驅散和拉條，不過致命的可能是初始一半血，鬼切重出江湖？

雖然這次的技能看似有些複雜，還有防禦數值摻雜其中，但或許最大的難點就是雲外鏡的初始生命，還得等後續特邀測試的結果。
