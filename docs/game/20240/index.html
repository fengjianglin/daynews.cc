選手明凱的路結束了，我還想看他在另一條路上追夢

2019-12-17 3,369條評論 4,274次閱讀 572人點贊

2019年12月16日晚，EDG官方微博發表消息稱，EDG的替補打野clearlove、LPL001號選手、被稱為「廠長」的明凱正式退役，轉型為EDG戰隊的主教練。一代傳奇落幕，那一個心心念念的S賽冠軍，沒有等到它落入囊中，追逐它的人，已經褪去了年輕的光芒。

「老將不死，只會凋零」

沒有人能夠頂替他的地位，沒有人可以選擇無視他給LPL帶來的驚喜與回憶。可以說，在我心裡，他就是一個平平凡凡的追夢人。在追夢的路上，他有過挫折與低谷，但是也有過榮耀和輝煌。

1.自我S5恍然入世，你便身臨巔峰

我是S5初期開始玩的LOL，那時候的地圖相對於以往來說發生了很大的變化，準確的說，我只見過一眼S4時期的地圖，是在LOL的官網查攻略時猛然發現的。玩了不久之後，便開始關注職業聯賽。那時候正趕上「S5最強野王」淘寶權（TBQ）和頂尖國產中單GodV在位時期，加上冠軍ADimp和輔助PYL（那個時候還不是死神）的下路組合，上單Acorn，替補是大公主（Flame）。

這樣的陣容，在LPLS5春季賽的決賽敗給了巔峰時期的EDG。那場比賽被解說娃娃米勒譽為當時「LPL最具觀賞性、最精彩的BO5」。當時的我還是白銀水平，可以說那場比賽是我第一次完完全全看完的整個BO5。我當時激動地從椅子上跳起來，並不是因為EDG贏了，而是因為我突然發覺LOL比賽如此精彩。那是我見證的廠長巔峰的開啟，也是他離著自己的目標最近的一次。

那場比賽結束以後，我並沒有查找LPL以前的戰績和歷史，而是繼續關注了EDG這支戰隊。自古觀眾愛強隊，就如同之前各國的觀眾喜歡如日中天的SKT一樣。也是在S5MSI季中邀請賽前期，我開始關注國際賽事，第一次看到當時的SKT。

當時的SKT已經獲得了S賽冠軍，比起EDG來可以說家喻戶曉。SKT對EDG這場比賽，值得被記入LPL的史冊。我看見廠長的努努追著Bengi的挖掘機在打。最後一場我看見了廠長的寡婦和Faker的妖姬。那一場比賽，傳奇對傳奇。一本永遠疊不上去的殺人書，百分百勝率的妖姬成為歷史。EDG贏了，我的LOL青春才剛剛開始。

2.回顧歷史，我知道了關於你的所有

很長一段時間裡，我都一直沉浸在EDG最強的幻想之中，因為我入LOL的時機和EDG當時的表現都太巧了。直到S5全球總決賽，我才看見了LPL脆弱的一面。當時被寄予厚望的LGD16強淘汰，韋神反向一箭成了所有LPL玩家心中揮之不去的陰影。那一箭本來已無關緊要，但是在那樣的環境里，任何失誤都會被無限放大。就這樣，一代頂尖國產中單沒落了。而那一年的EDG，也倒在了八強。當時我並不能接受這個結果，於是我開始查找LPL過往S賽的歷史，查找關於廠長的歷史，我才知道，原來LPL從未獲得世界賽冠軍。不過我卻看見了另一個傳說——WE戰隊。

LPL第一個世界性賽事的冠軍，來自老WE，他們創造的歷史不僅僅是一個頭像，更是LPL的一個時代。那時候的明凱，在世界第一卡牌若風、國服第一機器人捲毛、敵方塔下跳舞的草莓、以及「大王」微笑的光芒下，或許顯得不那麼亮眼。但是在WE土崩瓦解後，他還在另一個地方，堅持著他們最初的夢想。很多選手選擇在巔峰時期退役並不是沒有道理，他們為自己留下了後路，留下的人，還在追逐那遙遙無期的夢想。

3.光芒褪去，留下的只是後人的唏噓

S5是EDG戰隊的分水嶺，也是明凱的分水嶺。在S5之後，明凱的隊內核心地位彷彿被替代。儘管EDG在國內的表現依然夠看，但是EDG的後來者已經厚積薄發。從EDG三代AD的更換，小昭在EDG處於低谷之時獻祭力量。haro初來乍到並不顯眼，很多人認為他就是來和廠長輪換的。可是在S8賽季，haro用自己的盲僧把EDG戰隊踢進了全球總決賽。自此之後，廠長逐漸淡出了大眾的視野。留下的除了4396，馬踏飛箭。還有一個追夢人的身影，還有不賣肉鬆餅不退役的倔強。

他老了，這是不爭的事實，這是他不能再上場、不能再嘗試、不能再追夢的理由。

一代野王的沒落，是我自S5以來、自我開始玩LOL以來所有的青春和回憶。可能我的LOL年齡不算很長，青春的回憶也沒有各位S2S3的玩家完整。但是自我來到峽谷，他便是第一個闖入我的LOL世界觀的人。沒有人能頂替他在我心中的地位，可能只是時間和狀態上的巧合，我來的時候，正是他站在巔峰的時候。現在他走了，也可以說是沒走，但是他冥冥之中，還在我的召喚師峽谷里。

廠長，打野不出鳥盾了！你也不在了。

雖然轉型為EDG的主教練，但他還在EDG，他依然是EDG的精神核心。希望有一天，EDG可以在你的帶領下，站在你曾經想站在的領獎台的位置，捧起那座屬於你們的召喚師獎盃。

LPL001，斷開連接。
