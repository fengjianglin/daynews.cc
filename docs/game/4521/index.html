DNF：仔細算了一下史詩之路獎勵的成本，獎勵全換需要1.33億金幣

大家好，我是冷水，致力於給大家帶來第一手的DNF資訊和攻略，看完文章不要忘記點贊和關注哦。

經歷過最近幾天的信息轟炸很多玩家應該都知道12月10日更新後將會上線新版的史詩之路——超星河裂縫了。這次的超星河裂縫延續了以往史詩之路的優點，史詩爆率高、獎勵好，不管對於小號還是大號來說都是一個不容錯過的活動。

大家看上圖所示，不僅有寶珠大號畢業的黑瞳石和提高屬性的增幅書，還有幫助打造小號的泰波爾斯史詩。

但是大家有沒有想過，如果把刷這些獎勵所需要的材料成本換算成金幣的話各自有多少呢？反正我算完是直接想放棄這個活動了。

首先免費的史詩之路（星河裂縫）每天每個賬號可以打10次，每次掉落一個輝煌的金色氣息，活動從12月10日持續到1月14日一共35天，每天全勤一共可以提供350個材料。

其次收費的史詩之路（超星河裂縫）每天可以80個金幣獲得3個入場材料，這80個金幣忽略不計相當於免費了。一次可以獲得4個材料，活動35天一共可以獲得420個材料。

也就是說玩家全勤的話可以免費獲得350+420=770個材料，超過的需求部分除了少數氪金活動送的一點通行證外都得靠玩家自己買了。雖然從拍賣買玩家盒子開出來的可能要稍微便宜一些，不過為了方便計算，我們還是按3個精鍊的時空石=25萬來計算。

1.先算永恆的黑瞳石，因為很多人等著這些黑瞳石畢業呢。活動開啟是周二，如果緊接著周四刷新的話一共可以兌換6次，也就是一共需要600個材料，需要刷450個精鍊的時空石，換算成金幣是3750萬（當時算到這我就直接準備放棄了）。

2.增幅書，增幅書一個月換一本，在活動期間一共可以換2本，一共需要840個材料，需要刷630個精鍊的時空石，換算成金幣是5250萬。

3.跨界石，兩個一共需要800個材料，需要刷600個精鍊的時空石，換算成金幣是5000萬。

4.泰波爾斯首飾+特殊裝備一共需要200個材料，需要刷150個精鍊的時空石，換算成金幣是1250萬。

5.異界氣息凈化書需要100個材料，需要刷75個精鍊的時空石，換算成金幣是625萬。

如果想把除了反物質粒子之外的獎勵都拿完，一共需要2540個材料，除掉免費的770，還需要1770個材料，需要用1328個精鍊時空石來刷，換算成金幣就是1.11億金幣。如果加上反物質粒子則需要1.33億金幣。

所以我建議各位玩家呢，還是只拿黑瞳石和紅字書吧，這倆加起來需要1440個材料，除掉免費的770個還剩670個，需要用四千多萬金幣來刷，還算湊合，說不定湊一湊小號上的精鍊時空石然後從拍賣買點便宜的通行證還不需要太多錢。

反正我不建議普通玩家多刷這個活動，獎勵看起來很美好，但是仔細一算髮現成本還是比較高的。紅字書其實等春節版本就行，普雷材料的話都等幾次團本次數，這些金幣還是用在強化、附魔方面吧。

反正這個活動我就每天上去打下免費的，其他的是徹底放棄了，看完這篇文章的你看自己的氪金能力和對獎勵的需求度決定參與不參與以及怎麼參與吧。
