LOL新版皎月晉陞T0已無法針對？強勢中單全軍覆沒，玩家這招太狠

2019-12-20 338條評論 1,396次閱讀 97人點贊

目前英雄聯盟已經來到了9.24版本，在當前版本中有一位英雄是引起了玩家們的熱議，她就是新版皎月女神黛安娜，重做後皎月的E技能與R技能完成互換，玩法發生了巨大改變，吃著征服者的紅利，瞬間從一個冷門英雄變為了版本大熱。目前中單皎月已經成功晉陞T1，登場率足足比上個版本增加了10%，排在所有中路英雄中的第二位，並且不只是中單，上單和打野皎月也被玩家們所開發了出來，不得不說設計師這一個改動徹底改變了她的命運。

那麼為何新版皎月會獲得成功呢？舊版皎月與之相比究竟差在哪兒？眾所周知舊版皎月在6級之前的作戰能力幾乎為零，只能通過不斷推線、回家補給來度過前期，遇上對面強勢中野很容易就被軍訓導致崩盤，在小龍為王的版本中顯然有些太拖節奏了。而新版皎月直接把這唯一的缺陷給彌補掉了，現在她不僅可以在6級前就能單殺對手，配合打野遊走帶節奏也是一把好手，幾乎沒有任何缺點。

在對線能力迎來巨大加強之後，如今皎月在中路已經不可阻擋，在當前版本中除了皎月之外還有4個T1中單，他們分別是卡薩丁、蛇女、小魚人和亞索，然而灰哥看完他們與皎月對線的數據之後我驚了，這4個中路大爹平時在線上沒少欺負人，但在遇上皎月之後就只能認慫，被單殺率均超過了50%，卡薩丁和蛇女更是超過了60%，並且除了這4個T1中單之外其餘大部分中單英雄在對上皎月時的被單殺率也非常高，如果灰哥要說皎月是版本T0應該沒人反對吧？

許多玩家在被新版皎月支配過後紛紛哭訴：「這個英雄實在太無腦了，本來6級前還能小壓她一手，可現在6級之前就要處處小心，法師靠近A兵就有生命危險，玩刺客她回家裸個護臂根本打不過，難道真的沒有英雄可以制裁她了嗎？」

灰哥也是看了許多玩家的推薦，發現大家比較喜歡用近戰戰士來counter皎月，比如奧拉夫、鱷魚、諾手等等，這些征服者戰士1V1對線的確可以小壓皎月，不過有一個需要注意的是中路並不是上路，線比較短，想要單殺對手是比較難的，如果皎月帶引燃的話硬拼起來還真說不定孰勝孰負，並且這些上路戰士還非常容易因為過於激進導致被抓死，容錯率很低。

那麼有沒有那種既可以線上counter皎月、又不容易被抓死、操作還簡單的英雄呢？有大神推薦了一個「狠招」是引起了灰哥的注意，他就是波比。從兩人的對線數據來看波比是少有能碾壓皎月的存在，單殺率也達到了56%，算是一個非常不錯的選擇。

中單波比之前就有在比賽中出現過，用來針對突進型刺客尤為好用，而皎月恰巧就是一個非常依賴突進打傷害的英雄，並且前期E技能CD非常長，一旦QE連招被波比的堅定風采打斷就會陷入被動挨打的局面，在波比出完魔抗斗篷和小日炎之後皎月不管在站擼還是推線上都會陷入被動，而波比只需要不斷推線、用被動消耗、遊走就可以把皎月給打得非常難受，唯一的缺點就是後期單帶速度沒有皎月快。

不少玩家在嘗試過中單波比之後也是大呼太狠，有玩家調侃道：「對面皎月遇上我的波比一局至少要撞10次牆，頭都要撞破了，遊戲體驗極佳。」灰哥推薦大家如果不知道中路用什麼英雄好打皎月的話，直接秒選波比吧，對面絕對意想不到的。

那麼大家覺得波比算不算是新版皎月的最大剋星呢？還有哪些英雄在線上是可以輕鬆吊打皎月的呢？一起來分享你的經驗吧。
