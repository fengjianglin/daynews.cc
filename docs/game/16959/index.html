克隆技術發展到極致有多恐怖？《方舟生存進化》玩家靠自殺來趕路

2019-12-15 161條評論 1,282次閱讀 141人點贊

我們知道，現在的克隆技術因為涉及到倫理、技術問題已經被禁止了，但這也無法阻礙人們對它的猜想，在電影《致命魔術》中，魔術師用克隆技術演繹了最為完美的魔術，魔術師將自己給克隆出來，然後將本體給殺死，用克隆體代替自己完成了隔空轉移驚艷眾人。但小編認為電影裡面的克隆技術還不算最最完美的，在沙盒遊戲《方舟：生存進化》中的克隆技術才算是完美，完美到了玩家甚至能夠靠著自殺來趕路。

在《方舟：生存進化》中遊戲的每一個玩家都是克隆人，當克隆人玩家死掉了以後就可以將自己的意識轉移，從而達到永生和不斷復活的目的。和各種克隆電影不同的是，《方舟：生存進化》中的克隆人是沒有自己意識的，只要玩家不死去，這些克隆人就永遠不會蘇醒，這就避免了玩家看到有著相同記憶和意識的自己。

而《方舟：生存進化》裡面的床就是克隆人復活的初始，玩家可以選擇在《方舟：生存進化》的不同地方多拍幾張床，這樣當你想要前往相應的地方時，就可以直接自殺將對應床上的克隆體給喚醒，這樣就完成了瞬間轉移的目的，當然，玩家身上的物品和武器等資源是不能隨著克隆體轉移的，因此大家要在床的旁邊準備好相應的道具。

通過《方舟：生存進化》我們也能看出克隆技術為什麼被禁止，不管是復活、永生還是自殺趕路都與很多的宗教和倫理相悖，很難想像要是這項技術真的在現實中再現了這個世界會有多混亂。對於《方舟：生存進化》的克隆技術，大家有什麼看法？可以在下方評論區留言討論。
