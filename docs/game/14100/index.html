快手王小歪再度帶來騷操作，卡位置驚嚇敵人，帶來刺客流反面教程

2019-12-11 839條評論 2,640次閱讀 901人點贊

說到快手王小歪就想到了娛樂主播，實在是因為他的直播太有趣了。其實王小歪本人是一個技術主播，但是因為總喜歡搞一些搞怪的操作，讓大家誤以為他是一個娛樂主播。不過不管王小歪是娛樂主播還是技術主播，總而言之他的直播不僅僅能學技術還能收穫快樂。

王小歪和隊友一起追殺一個敵人，結果敵人逃到了房子裡面，將車子輪胎都打爆了。本來以為能夠阻止王小歪的，但是王小歪為了給敵人驚喜，直接就搞出了一手騷操作。在和呆萌配合卡位置爬樓失敗之後，又找了一個新位置，而找到感覺的王小歪是輕而易舉就上到了樓上。

在上樓發現了敵人之後，王小歪並沒有採取擊殺對面的操作，而是不斷的前進離敵人越來越近。如果是一般的主播，這個時候肯定就將對面擊殺了，炫耀自己這種方法如何如何出色。而喜歡搞怪和搞笑的王小歪，則為大家帶來了更加精彩刺激的”刺客”玩法。

估計對面應該是聽到了聲音，轉身一看發現後面居然有一個人，嚇了一條之後這個妹子玩家直接亂開槍。而刺客流玩法創始人王小歪，就為大家帶來了刺客流玩法的錯誤教學。不直接擊斃敵人，而裝B靠近敵人究竟會出現怎樣的後果！面對妹子的亂槍，以及三級頭三級甲，王小歪付出了血的代價！

因為樓上只有王小歪和敵人的緣故，對方護甲又太高。而且因為王小歪沒有抓出先機，導致直接就被對面干倒地了。之後王小歪還想苟延殘喘的逃走，但是敵人可是深知不能養虎為患的道理，直接就將王小歪給淘汰了！而王小歪也是發出了極其凄厲的吶喊，不過王小歪也算是給大家搞了一種新玩法。你認為王小歪為大家帶來的刺客流玩法可行嗎？歡迎留言評論！
