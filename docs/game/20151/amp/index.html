元氣騎士：BOSS無頭騎士的頭到底去哪了？見習傭兵了解一下

天天要聞
2019-12-18

「涼風有信，秋月無邊，我是愛玩遊戲的大鼻孔。」

元氣騎士的BOSS們形態各異，每個傢伙都有著自己獨特的攻擊方式，讓玩家又愛又恨，關於如何打BOSS的攻略相信各位小夥伴已經看了不少了，偶爾也要換換口味，了解一下BOSS背後的故事也很是不錯哦！大鼻孔一直覺得BOSS無頭騎士非常的神秘，今天，我們就來揭秘一下他的頭到底去哪了吧！

BOSS無頭騎士和他的馬，嚴格意義上來說，其實是兩隻BOSS，所以我們不能說BOSS飛龍幼崽是唯一的雙子BOSS哦！無頭騎士也是的！但是只有在進入殘血狀態時，他們才會分離；不過，它和BOSS飛龍幼崽還是有很大不同的，就在於無頭騎士和他的馬並不是同一個物種，因為他根本就是一隻被拼接起來的怪物！

故事是這樣的：那是一個殘陽照射下的戰場，橫屍遍野，折戟沉沙，突然，趴在地上的騎士動了動手指，他坐起了身，發現四周一片黑暗，當他試圖去揉一揉自己的眼睛時，才驚覺項上空空如也，頭！不見了！他開始慌了神，四下摸索，幸好！在他身後就有著他正需要的東西，仔細摸了摸，有耳朵，眼睛，鼻子，牙口還不錯！就是嘴巴大了點，也顧不了這麼多了！他立馬就把頭給套了上去！

後來，才知道這是個馬頭，也不知道是被施了什麼咒語，不到危機關頭，馬頭根本脫不下來！因為，它有自己的想法！所以，這就是為什麼，馬頭總是亂跑的原因，哪裡是為了幫忙，根本就是想跑啊！那麼，無頭騎士的頭到底去哪了呢？

這就要說到狂戰士的無頭騎士皮膚了，大鼻孔本來以為狂戰士也會頂著一顆碩大的馬頭出現，但事實上，這款皮膚是一個整套的盔甲，下半身和BOSS無頭騎士是一模一樣，那麼這套盔甲的頭部，應該就是無頭騎士曾經的樣子沒跑了！

如果故事就這樣到這裡，也就結束了！可是，有一天大鼻孔看到見習傭兵，突然發現為什麼他們倆長得這麼像呢？無頭騎士不就是見習傭兵的魔化造型么？雙角變長，皮膚變綠，配色變黑！妥妥的惡魔化啊！再看，見習傭兵慣用的武器是霰彈類槍械，無頭騎士呢？雖然拿著把大劍，但是，它掉落的國王的新槍，可也是一把霰彈槍哦！這麼多的證據下，大鼻孔大概了解所有的原由了！

讓我們來還原一下事情的經過：像往常一樣，見習傭兵陪伴著他的僱主，來到地牢的最後一關，這一次的BOSS是瓦克恩頭目，在拚死為僱主抵擋了BOSS的激光後，他倒在了瓦克恩製造的毒圈內，隨著地牢的通關，他本應該被系統復活重新出現在地牢的某個房間，等著下一位僱主的到來，但這一次，他不甘心！

他幹掉了前來接引自己的牛頭馬面，可自己也被當場斬首，頭被帶走繼續製造見習傭兵！但是，他的身體居然又活了過來！最終成為了BOSS無頭騎士和他的馬！

以上，就是大鼻孔對於BOSS無頭騎士的背景猜測；盡量貼近了真實的元氣世界，小夥伴們對這個設定還滿意嗎？或者，你有更好的想法，歡迎在評論區留言和大家一起分享哦！
