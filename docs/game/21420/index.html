CFML丨從AG到SV 無論身在何方青尤定當全力以赴

2019-12-20 888條評論 4,066次閱讀 1458人點贊

CFML2019職業聯賽秋季賽即將上演季後賽，率先上演AG與SV的對決，已然吸引玩家們的關注。對於青尤而言，上賽季從AG出道到如今成為SV主力輸出的關鍵，在季後賽中他又該如何表現？

【出道備受關注 青尤一鳴驚人】

上賽季的AG隨著人員的變動，其引入了新人晴宇與青尤兩名選手，青尤作為隊內小將選手，由於個人並沒有豐富的大賽參賽經驗，從而受到玩家的質疑。

當上賽季職業聯賽開賽後，青尤作為AG的新人小將，當其與隊伍迎戰多支老牌勁旅時，青尤個人的多次四殺也成為隊伍中的亮眼發揮。青尤個人的敢打敢拼也讓我們看到這名小將的潛力。

（CFML2019春季賽 青尤效力AG）

憑藉著青尤晴宇新人選手的強勢突破以及老將嗜血青菜等人的穩定發揮，AG成功從常規賽中殺出，最終在季後賽中闖入半決賽，雖然未能獲得決賽資格，但是四強的榮譽則證明了AG新陣容的強勢。

【轉會SV 青尤的新起點的開始】

隨著席位制改革的開啟，各個隊伍也開啟了選手轉會，青尤在新賽季中則加盟SV，對於青尤而言，這也將是其新的開始。

常規賽前三輪中，SV未能迎來隊伍的首勝，不過在與R.LGD等強隊的對決，也令青尤經歷了一定的磨練，無論是臨場處理還是個人的發揮，都讓我們看到這名選手的成長。正是青尤的關鍵發揮，也令SV在後期遭遇EDGM以及eStar等強隊時有一戰之力，青尤的穩定輸出成為SV獲勝的關鍵因素。

由於季後賽率先上演SV與AG之間的對決，那麼意味著青尤將在最關鍵的比賽中遭遇自己的老東家。對於青尤而言，當其面對AG時將全力以赴，在青尤的眼中只有發揮自己全部的實力，那麼才是對AG戰隊的尊重。

即將開始的季後賽中，當SV再次在關鍵賽中遭遇AG，相信兩支隊伍將全力以赴，同時相信青尤也將發揮出超高水準，從而在賽場中證明自己的能力。
