來自魔力寶貝老玩家的在線評測之魔力寶貝好不好玩

2019-12-07 0條評論 17次閱讀 0人點贊

摘要

關於遊戲到底好不好玩，下面就給大家簡單的分享一下個人的一些遊戲心得以及各種遊戲感想。希望可以給大家一些參考和幫助，希望大家能夠喜歡。日常活動簡易的做，不習慣做的就不做（如討伐魔窟﹑護送軍資這兩項固定不做，周活動亦如是。

關於遊戲到底好不好玩，下面就給大家簡單的分享一下個人的一些遊戲心得以及各種遊戲感想。希望可以給大家一些參考和幫助，希望大家能夠喜歡。

日常活動簡易的做，不習慣做的就不做（如討伐魔窟﹑護送軍資這兩項固定不做，周活動亦如是。由於手機玩所以不會長期掛網，在經驗值上像單車長途賽事那樣，算是在主車群的中後，前幾天，我服已有幾位九十一級和完成所有主線任務，總有幾個人會拋離主車群，這些人從六七十級就一直拋離一般玩家三至四個等級，差距也許在於騎士團任務恆常刷和不分晝夜式掛網打怪吧。

這網上早前爭議的問題和官方的紕漏和失當，最近似乎平靜了不少，七月初至八月最為激烈，也許不足三個月之期，明顯淡出﹑賣帳號和退坑的玩家人潮多了，隊不用排，世界頻道冷清了，只剩來來去去幾個玩家明明來自同一個家族卻當世界頻是家族頻的聊，和鑽商﹑平價收高級橘帳收三寶的宣傳廣告，各樣合併伺服器的流言像大勢所趨。

身邊有大課長玩家已在群組向那些意興闌珊預備轉玩別的遊戲，或不玩的族員玩家小抱怨，所課金付出的已足以買車和去一趟豪華的歐洲旅遊，別這麼快讓遊戲環境變得死寂。真的，從五六月時知悉橘子代理新的魔力寶貝手游，會有什麼樣的一番景象，不必和陸服作任何比較，畢竟十年多前，橘子已代理電腦網遊，在遊戲業內累積不少名聲了，會怎樣經營並不意外。

何況免費手游本身就是個需課金的代名詞，不課就不要比較和爭逐什麼﹑不要發聲和抱怨什麼，因為你不課﹑課得少你就沒資格爭取任何那些所謂一廂情願的遊戲生態，要虛心的感謝課長們撐起這一片天，使我們有遊戲玩，其他抱大腿的玩法在此就莫提吧。其次，要麼這樣不憤，就不如乾脆不要玩，不要趟這潭渾水吧。

玩傳教士主要的理由是不想大課去拼輸出或整體數值，很簡單，坑，並不是這一二款手游看出的事實，更不會管是由誰來代理哪款手游，會有什麼差別的坑，多年來已是這樣發展，說起來挺消極的。推薦和抓我一起玩這款遊戲的朋友已在遊戲初期退坑了，他們的原因是日常活動太固定了，還是現今這類ＲＰＧ和養成模式的遊戲太過普通了，這只是叫做魔力寶貝，聽著一樣的音樂和主要劇情，養成寵物，除此之外呢，也許，我也是沒當這是魔力寶貝，才能玩下去。

我想一些玩家們很早已經醒覺課和不課﹑微課，差距會在遊戲後期擴大，這是有沒付出所得合理的強弱之分，拼的是財力不是智力，所謂的各樣玩家之間的競技也只是斗敏捷出手快而已，所以周四﹑日的活動純粹刷刷家族的貢獻值就好了，不要太認真。

遊戲生態里，很難避開有沒動腦玩，有沒課金玩的這些門討論，我很欣賞在一些任務里王戰的設計，需要玩家們配合和策略；慶幸在養成寵物方面沒太大課金的元素在裡面，暫時也沒有出現什麼必要課金才能通過或獲取的必須品。更新九橘後的日子，好像有群中課長醒覺了，在思量值不值得再下一城的課金，因而退坑了。

我不會去評價哪一隻寵物是最優異和強勁，或是值得去練，只因為寵物純粹個人的愛好，偶爾會看見妖狐也有人改造有人會練，所以像螳螂和龍蜥的比較，甚至拿課金金卡寵物來作比較都是無謂。我看的純粹是數值和屬性的分布，如成長值﹑種族和料理才是考量因素，在有了改造之後，銀卡升上金卡的寵物自然升格了可以用３的天賦。在洗湅方面，像力寵，體攻防敏要洗滿；像體寵，體防要洗滿；像魔寵，防敏魔要洗滿，完美主義或洗湅藥水太多的話便可以追求更進一步的洗。這遊戲其中一個亮點就是寵物的特殊天賦，無論取名和定位也非常有創意，可以彌補不同類型的寵物能力先天的缺憾，或使寵物的戰力更上一層樓。總括而言，要完整全方位的練好只寵物，洗湅﹑天賦﹑改造和吃滿料理並不容易，但當你願意為那隻寵完成了這些事之後，這肯定是一隻強寵。

我並沒看到這兩個月以來官方在盜帳拉鑽﹑工作室和ＢＵＧ練等級等事件上有誠意麵對，有很多原因吧，這遊戲會否很快經不起風浪而被收割是一回事，不想得罪部份曾違規了的課長玩家是一回事，專註推出更多形式吸金虛寶的更新並無意解決那些解決不了的事，反正想繼續玩的玩家自然會留下來，課了金的中大課長更不會輕易放下，遊戲里還有很多進度需要時間去完成，所以對官方未能夠處理好的各樣不公睜一隻眼閉一隻眼，對官方來說至好不過，又是一回事。要不然，經玩家們檢舉再認真查證，然後判違規者永久停權，這是一件很簡單的做法。

在這我並不想輕言說其吃相太難看，畢竟正服魔力寶貝的大宇這些年來正正是這樣的吃相，奇怪的生態養下來一群有年資的課金玩家（身上神裝神寵，不甘但仍在苟延殘喘。盡然遊戲裡頭的東西如何變態，初代玩家極不接受的種種（抱歉，話題扯遠了。吃魔力寶貝的名字賺錢，對凡玩過初代魔力有感情的玩家而言，這是很悲哀的一件商業現實。

另外，每段時期的節日活動也不太困難，而且每時候也有不同的獎勵，如之前的３３０萬經驗值﹑幣換寵物料理﹑銀色封印卡之類，使遊戲的日常不會太寡悶，也可以簡易獲得一些實用品。

後段要感謝一些玩家吧，在周日無私奉獻精神時間去做地宮司機，苦口婆心想盡法子在王前叫醒一車睡眠艙隊員，日常大花高水不用強制互刷不用帶隊的專業高水司機，進副本等確認很難唯有扔下那具死屍吧，義工為野團副本和任務付出時間去幫別人完成的玩家，專業走地圖的野王司機。這些都是在遊戲里最好的感受和體會，不是每個人都會在意人家不開裝備檢視和藍裝備﹑不對等裝備，當你有了家族和固定的玩家群，偶爾在許可時間和電量下幫一幫無妨。（但當遇著一隊必然走向失敗的隊伍時自然例外的離隊別費時間吧
