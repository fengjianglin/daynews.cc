《我的世界》SCP收容失效 裡面怪物來到主世界後果不堪設想

天天要聞
2019-12-18

在《我的世界》眾多解密地圖中，由作者「我曾經跨過山沒海」製作的《SCP收容失效》是一個非常特殊的地圖。它有著完整而龐大的背景框架和故事結構，光是故事的前言就已經非常吸引人了。讓我們一起來看看這張優秀的解密地圖裡面藏著什麼秘密吧。

地圖下載

在模組中心搜索：SCP收容失效，即可找到本模組。地圖僅支持基岩版本，值得注意的是，本地圖已加入10鑽石計劃，短期內免費過後也僅需10鑽石。

遊戲背景

SCP收容失效有著完整的故事背景和世界觀，大意是：人類繁衍的百萬年間，僅有千年內是有歷史記錄的。而在更遠古的時期，有太多奇怪的怪物威脅著人類的生存。

時至今日，這些未知的生物依舊還存在於這個世上，於是SCP基金會便誕生了，他們負責尋找這些生物並將其收容起來。而玩家扮演的就是SCP基金會中一名最底層的人員。

遊戲劇情

當玩家開始進行遊戲後，就可以按照劇情故事來逐步解密，進而推動整個劇情的發展。我們扮演的D級工作人員將會一不小心捲入一場可怕的陰謀中，為了活下去，需要不斷逃跑和尋找鑰匙。

收集

在探索過程中，你還可以收集到各種各樣的物資，例如不死圖騰和金蘋果等。這些物品到地圖的後半階段將有大用處。在遊戲中，你還可以來到所謂的SCP中國分部，在這裡你可以觀看很多詭異生物的文字介紹。

戰鬥

在遊戲的最後，你需要和逃走的SCP682（其實是個凋靈）進行戰鬥，之前收集到的物資此時都有了用處。

異界之門

值得注意的是，這張地圖還有一些地方是不需要進入的，例如這裡的傳送門，它會將玩家傳送至下界，卻與劇情的推進沒有絲毫關聯，完全不用進去。

通關

擊敗SCP096後，就可以通關了，而事情的真相也會浮出水面。原來是SCP中的高層想將可以讓人長生的物資佔為己有，從而引發的怪物逃脫。
