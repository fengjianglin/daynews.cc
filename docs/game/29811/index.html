魔獸世界懷舊服獵人PVP弱爆了？先來上一課

2019-12-24 1,146條評論 2,347次閱讀 1161人點贊

魔獸世界懷舊服開啟PVP戰場後，我經常會聽到抱怨有人獵人PVP弱爆了。但我對這種看法不認同。說獵人弱的人，不知道你們真正了解過別的職業沒 ，尤其是其它職業的控制技能。其實獵人PVP關鍵問題就是如何應對對方的控制技能，因為獵人是要拉距離的。下面是對獵人PK 某些職業的一些心得 ，希望對不善於PK玩家的有所幫助。

盜賊，被賊偷襲，不要隨便用徽記解暈， 等你能動了，先驅散射擊在徽章解毒（減速毒），標記跑開，然後開始放風箏，同等級裝備 ，沒有哪個賊能暈你到死，中間不脫手的（這裡不考慮工程的其他輔助技能）。這裡要注意，等你開始放風箏，賊會開急跑追，你要看好距離，賊離你近了很可能致盲，所以在賊比較接近你前，假死放冰凍陷阱，冰霜也可以。在賊追上你之前或者至致盲前假死的最大好處就是，可以把你身上的星消掉（賊的連擊點數）， 這樣賊就不能上來就腎擊；等賊追上你了，近身必然會中陷阱，你再次跑開，然後瞄準加毒蛇、多重，然後繼續風箏，這個時候你的驅散射擊CD已經快恢復了。如果賊又2次急跑追上來，驅散他，然後繼續拉開距離。中間用震蕩射擊減速，記得維持標記 。如果賊消失了，而你閃光彈沒照到他，就立刻給自己繃帶，賊一般不敢在出來了。同等裝備，殺賊其實不難。

法師，首先要搞清楚一點，被法師冰環了要速度解，因為法師攻擊被冰凍住的對象，天賦支持暴擊概率很大。如果法師先手，一般都是閃過來冰你，等他放了冰環立刻驅散，用徽章跑開。然後立刻上吸藍箭，開急速射擊、多重 。這裡不要開豹守，平速跑就是了。法師閃現恢復之前可以保證他一直在你8碼外， 放狗咬他。等估計法師閃現CD快恢復的時候，把寶寶定住，然後法師閃上來冰你，等他冰後立刻放寶寶過來，寶寶是沒被冰住的，可以攻擊他，盡量打斷施法，解了冰之後立刻衝上去近身毆法師，法師這個時候沒有冰環，閃現只能跑，然後你反向跑，拉開距離打。如果法師冰箱了，冰箱等最後3秒的時候開始瞄準，他冰箱出來剛好被打，如果他半路出來，直接多重。

戰士，個人認為戰士是獵人最難對付的職業。裝備對戰士的提升遠大過對獵人的提升。打戰士關鍵不要讓戰士脫離戰鬥狀態，多給他次衝鋒機會，獵人徽記等戰士攔截後再用。一般戰士衝上來 第一時間會斷你筋，不要慌上摔絆（獵人近身減速技能）後立刻區散，然後跑出8碼，這個時候大家速度都很慢，你可以打到戰士，但戰士打不到你，一般他都會換姿態攔截，暈3秒後你立刻假死陷阱冰住他。然後徽章解斷筋減速，開豹守跑開。如果不是穿軍裝的狂暴戰士，你可以隨便虐待他 20多秒了，在此期間要盡量打到他沒血。

總的來說，獵人還是很強的，但獵人對操作要求比較高，現在的獵人就要少放陷阱，不要依賴陷阱，多練風箏技術，PVP才能更強！
