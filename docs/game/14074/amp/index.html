爐石傳說：天下武功唯快不破，新版本海盜戰快攻再度來襲

天天要聞
2019-12-11

爐石傳說新版本”巨龍降臨”全卡已經公布，明天新版本就將正式上線，新版本最值得期待的當然是迦拉克隆卡組，不過相對於偏中慢速的迦拉克隆卡組來說，想要快速上分依靠的還得是快攻卡組，今天筆者就來為大家推薦幾套新版本用於上分的快攻卡組，想必也有不少信仰快攻的玩家。

打臉薩

新版本的打臉薩加入了關鍵的一費隨從電涌風暴，雖然比以前的穴居人差很多，但是這也是一張元素卡，在這個版本的快攻薩中非常關鍵。風暴之怒是非常好配合流電爆裂和雷雲元素的法術，而且可以通過青蛙之靈過到。

無面腐蝕者作為中立卡基本算是無腦強卡，薩滿本身雜毛比較多，變一個形成兩個5-4站場非常強勢。

新版本的戰士新卡很顯然和防戰無關，那麼新版本的新卡組只有節奏型的戰士了。

海盜戰在沉寂許久之後再度歸來，新版本的各種卡都非常契合海盜戰的構築。新橙武海盜之錨可以帶動整套卡牌，抽上來海盜，下海盜還可以帶上022空降歹徒，另外還有藏寶匪賊防止橙武被軟吃掉。新版本的海盜戰體系算是非常完善了。
