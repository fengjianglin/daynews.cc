DNF12.17西海岸更新：

此次西海岸更新的活動，算是2020年春節套的最後一次預熱了，下一次就是年套和春節專屬活動了。當然，西海岸目前僅僅是更新了副本相關的活動，只有元旦氪金活動，就是下一次體驗服更新的內容了。有一點玩家需要注意：本次活動送出的獎勵都是打造大號的獎勵：紅字書、+11的卷子、商城魔盒……大號用也是有非常不錯的提升的。

國服2020年春節套更新預熱，

其實每年的春節活動都是少不了：簽到、罰站街、歸來勇士（新職業）……等活動。此次也是老規矩了，玩家和玩家之間的交流越來越少了，而玩家和氪金商城的交流越多了。這就使私服的特徵。LOL之所以成為遊戲界的扛把子，靠的就是玩家之間的交流多。DNF的團本也是一樣如此。而現在的活動，不管是春節活動還是預熱活動，都是為了騙氪的。此次送出的罰站街，活動就是送出了畢業獎勵：

黃金書+11增幅券，免費領取

活動獎勵雖然好，又是老套路——在線站街time！非常沒意思的活動，現在玩家一周上線時間不超過2小時，卻要每日在線2小時，顯然是強人所難的活動。但是強打書和+11的卷子，倒是不錯的。全勤就能免費獲取。
