巨龍降臨正式登陸，宇宙德全新卡組出爐，嘗鮮就現在

2019-12-13 27條評論 1,254次閱讀 174人點贊

從12月11日起，《爐石傳說》巨龍降臨擴展包即將正式上線。在萬眾期待之中，這一套早在半年前就開始醞釀給的卡組將要正式進入到所有玩家的遊戲之中，作為巨龍年的最後一個擴展包，這無疑會帶來又一波全新的衝擊。

在最新版本的《爐石傳說》中，官方將會推出140張全新的卡牌以及祈求的遊戲機制，與此同時，新英雄泰蘭德·語風以及希爾瓦娜斯也將正式上線。相信不少玩家早在消息發布之初就已經躍躍欲試，如今終於盼到了新版本的上線，又有什麼理由不去一探究竟呢？

對於大部分的玩家們而言，這一次《爐石傳說》的升級無疑再次增加了遊戲的可玩性。在巨龍卡組引入之後，玩家們將能擁有更多地卡組搭配於選擇方式。在這裡，小編就向大家推薦一套德魯伊的新版卡組。

相較於前一版本的宇宙德，這套新的卡組的最大優點就是填補了進攻不足的漏洞。紅龍女王阿萊克斯塔薩這一張新卡牌的引入，配合上啟迪者伊莉斯和軟泥教授弗洛普，就可以反覆利用其戰吼，創造出強勢場面，一方面大大提升了攻擊力，另一方面也保證場面的穩定。

當然，我們也不能把賭注只壓在同一個位置。如果強勢場面被破，我們也可以藉助瑪里苟斯和伊莉斯的複製順勢打出一波新的爆發。與此同時，夢境吐息的引入可以實現及時的跳費，而藉助著耶比托·喬巴斯的戰吼，我們可以提升卡組斬殺能力的上限。

另外，這一套卡組中的辟法巨龍、看台噴火龍、雙頭暴虐龍等都有著不錯的攻擊力，其中的辟法巨龍更是擁有聖盾與突襲，足以緩解緊張的局面。

你是不是也想上手體驗一下全新的卡組呢？對於這次的升級，《爐石傳說》官方可以說得上是誠意滿滿。無論是新玩家還是老玩家，只要在新版本的遊戲中登陸，就能獲得包括5張迦拉克隆傳說卡牌、2張迦拉克隆之盾隨從牌在內的總共7張新卡！，很多玩家都紛紛表示這個活動簡直太香了。

除此之外，在完成了傳說級的每日任務之後，玩家們還能額外獲得三卡包和金幣獎勵，可謂是香氣撲鼻。你還在等什麼呢？快快登上全新的《爐石傳說》，一起體會遊戲的魅力吧！
