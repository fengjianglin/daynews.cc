DNF體驗服：100級深淵套路多，學會這一招必出SS，姜策劃真機智

天天要聞
2019-12-20

「地下城與勇士之小狐狸君愛談遊戲電競」第七百六十七期《DNF體驗服：100級深淵套路多，學會這一招必出SS，姜策劃真機智》，100級版本上線韓服體驗服2天了，許多玩家也是漸漸地開發出其中的一些個套路和玩法，尤其是深淵的玩法，是玩家最為關心的。畢竟新的裝備產出都和深淵相關。不管是升級、繼承、跨界都離不開深淵掉落的材料。所以小狐狸君覺得，這個深淵玩法，就是希洛克Raid出現之前，100級最主要的玩法了。那麼100級的深淵和95級相比，除了背景怪物之間的差距之外，還有什麼新的套路呢？老規矩搞快點，直接上本次新套路的實際截圖展示：

DNF體驗服：100級深淵套路多，

此次100級的更新，許多玩家都表示，看到了85級史詩的影子（大幽魂套）！其實這一點也不稀奇，第一，85史詩確實做得好，姜策劃學習借鑒85史詩也沒什麼；第二，姜策劃就喜歡朝花夕拾，CP系統、魔界大戰都是前任策劃放棄的版本。不過此次深淵的套路倒是新的，玩家只要遇到金黃色的深淵柱子，就能獲得100級的史詩，掉落的幾率為100%。

學會這一招，必出100級SS，

也就是說，玩家只要遇到金色柱子後，就可以保底拿到1件100級史詩。雙簧的即裡邊也是很高的。但是紫色的基本上不會掉落史詩。所以說姜策劃這一次等於說是將史詩的掉落告訴玩家了。有的神豪玩家說，金色就刷紫色就退，別忘了100級的深淵材料非常重要，升級、繼承、跨界都是能用到的。所以這個100級深淵還是必須刷的。畢竟仔100級Raid副本沒放出之前，玩家就不得不刷這個副本。

姜策劃這一次真的機智：

所以就算是提前告訴你，這一次不出史詩，玩家也是沒脾氣不得不刷。看到金色的柱子就可以了。未來國服玩家刷喇叭，就是：祈願金色柱子！不要紫色的，可見姜策劃的設定還真是非常的機智。
