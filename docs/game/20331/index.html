移動端畫質最好的競速遊戲：GRID Autosport

2019-12-17 1,241條評論 3,941次閱讀 1444人點贊

手機硬體配置年年更新，在兩年後的旗艦機上能夠玩到的競速手游畫質和兩年前卻幾乎沒有多少升級。看到《極品飛車：無極限》在一台驍龍 855 手機上的實機畫面，我甚至會懷疑是不是移動平台競速遊戲已經摸到了一塊千斤重的「天花板」。

直到上個月月底，老牌競速遊戲廠商 Codemasters 的 GRID Autosport 終於正式登陸了 Android 平台。至此，當今主流的移動平台上終於有了一款稱得上是「主機級畫質」的競速手游。

是移植，更是一次降維打擊

GRID 在 PC 和主機平台上早已名聲響亮，絲毫不輸極品飛車與 GT 賽車。而 Codemasters 選擇將 GRID 系列作品移植到移動平台時的決斷也可以說是有勇有謀：拋開 2019 年剛剛推出的 最新作，移植到 Nintendo Switch 和移動平台的 GRID Autosport 正是該系列最近的一款作品。

這也是 GRID Autosport 移動版最令人折服的地方，看似不太可能的移植不僅沒有帶來任何肉眼可見的縮水，當年這款遊戲在 PC 和主機平台上的畫面表現也原汁原味地呈現在了手機屏幕上：

根據速度和方向的不同，碰撞、擦掛等行為會在車身上留下不同程度的損毀痕迹；

過彎時側面擦掛留下的痕迹

不熟悉賽道瞎跑留下的各種痕迹

在賽道上疾馳的過程中，除了沙塵和草皮，你甚至還能看見前車因碰撞掉落的碎屑再次揚起；

把前車留在賽道上的保險柜殘渣給揚了

陽光明媚時透過賽道兩旁樹木和建築的光線，同樣也會在車身上一閃而過；

拍攝於上面的事故發生前

只能從回放抓到這角度了

輪胎牆也並不是一個撞上去會減速的賽道障礙，牆裡的每一個輪胎都會因為你的一次意外而被撞飛出來……

此前，移動平台上與「真實」二字沾邊的遊戲或許只有 EA 的《真實賽車 3》——在 EA 和 Firemonkeys 持續不斷地迭代更新和精心維護之下，2013 年「出生」的《真實賽車 3》默認畫質放在今天依然能夠讓同門師弟《極品飛車：無極限》甚至 Gameloft 的《狂野飆車 9》感到汗顏。

《真實賽車 3》中的布蘭茲哈奇，卡通感十足

而當我們把 GRID Autosport 和《真實賽車 3》放在一起對比時，事情突然變得「殘忍」了起來。雖然在 PC 和主機平台上發布已經是五年前的事，但 GRID Autosport 下放到 Android 平台後的畫質，依然能一騎絕塵將《真實賽車 3》這樣的傳統競速手游遠遠甩在身後。損毀系統「千篇一律」、光影表現「聊勝於無」、賽道環境一片死寂的《真實賽車 3》，在 GRID Autosport 面前甚至會顯得有點「小兒科」。

GAS 中的布蘭茲哈奇，華麗的光影與動態模糊

不過這樣的畫質表現也為 GRID Autosport 設下了不低的兼容性門檻，官方支持頁面 中給出的機型列表中大部分 Android 設備都是最近 2~3 年發布的、搭載驍龍 8 系旗艦處理器的中高端手機。即便手機能夠以最高畫質運行，遊戲也會將最高畫質下的遊戲幀率限制為 30 幀。

但無論如何，對於那些在移動平台上也絲毫不想妥協的「畫質黨」而言，GRID Autosport 是一款不可多得的作品。如果用不久前手機行業頗為流行的一句話來形容：

GRID Autosport 這一記來自 2014 年的「老拳」，在畫質表現上竟然全方位打掉了當下所有同類競速遊戲產品。

P.S.要是你對自己手裡的設備有信心，文末還有解鎖最高畫質 60 幀率的小教程。

漂移還是走線，總有一種適合你

無論在哪個平台，競速遊戲都有「模擬」和「街機」之分：模擬類注重對車輛和賽道的還原，常見的過彎方式是「走線」，在移動平台上的代表是《真實賽車 3》；街機類則更加追求速度帶來的快感，漂移、氮氣加速這類元素才是常客，在移動平台上的代表是《極品飛車》和《狂野飆車》。

《極品飛車：無極限》官方的「買家秀」

因此不管畫質多麼誘人，在高昂的售價面前，我們必須得思考 GRID Autosport 究竟是一款怎樣的作品。

GRID Autosport 移動版發行商 Feral Interactive 對這款遊戲的定位是「融合了模擬與街機操控風格」。這種融合的遊戲風格在遊戲中則具體呈現為高度自定義的精細控制方案、多層次可調難度級別和多類型賽事三個方面。

從 PC、主機到移動平台，GRID Autosport 引入了其它競速手游中常見的觸控和陀螺儀傾斜操控方案。在開啟自動變速箱、牽引力控制、防抱死剎車系統、穩定性控制以及「街機」操控輔助的情況下（對應遊戲內難度「新手」/「職業」），我們完全可以用更熟悉的競速手游操作方式把 GRID Autosport 當作是一款畫質更精細、細節更到位的《真實賽車 3》來玩。

不同的難度和輔助設定決定了不同的遊戲操控體驗

而對那些要求更高或喜歡挑戰的模擬賽車玩家來說，GRID Autosport 不僅為觸控和陀螺儀傾斜操控方案提供了更加細膩的進階設置，包括死區、飽和度、轉向線性度、陀螺儀靈敏度限制等等。這些進階設置位於「主菜單 > 選項及其他>選項>高級選項」當中，不少名詞對競速遊戲新手或街機向玩家而言都有一定的門檻，但耐心調校卻能很大程度上提高模擬賽車玩家的遊戲體驗。

強大的自定義控制

值得一提的是 GRID Autosport 作為一款移植遊戲自然也有著出色的手柄操控適配，除了沒有振動反饋這一點非常可惜之外，使用手柄遊玩移動版的確更能找回 PC 版上那種熟悉的感覺。

完美適配手柄

最後，豐富的賽事類型相信也能讓不同口味的玩家在 GRID Autosport 找到屬於自己的那一份樂趣。遊戲內置了生涯模式、額外錦標賽、自定義杯賽、計時賽和快速競賽五種遊戲模式，每種模式下都包含直道加速、耐力賽、巡迴賽、衝刺賽、漂移賽、街頭競速等多種比賽類型。

走線是不可能走線的

直道加速講究起跑反應、轉速控制和檔位控制；巡迴賽通常由多場賽事組成，排位、時間和積分決定車隊的最終名次；耐力賽通過夜間長時間比賽考驗參賽者的意志與體力；漂移賽則注重技巧，車身角度、入彎速度和連貫性決定得分，觀賞性遠超一般的街機競速遊戲……

多樣的生涯賽事

作為一款移植遊戲，這些特色其實也正是當年 Codemasters 期望中 GRID Autosport 應有的樣子，即街機和模擬的完美平衡：在核心玩家手裡，真實的賽車和賽道還原有著天然的強大吸引力；而在普通玩家手裡，多層次、人性化的難易度和操控方案設計又能保證剛入門時不會跌跌撞撞，把 GRID Autosport 玩成一款毫無體驗的「碰碰車」遊戲。

漂移賽是我這種愛耍帥玩家的心頭好

不過對於移動版而言，我更偏向、也更推薦將它理解為後者，模擬賽車遊戲對現實生活賽車運動的還原本就有著難以逾越的限制，在沒有方向盤、甚至沒有手柄振動反饋的移動版上，我們能夠體驗到的真實賽車運動的魅力更是微乎其微。

另外作為一款移植作品，Codemasters 並沒有針對移動平台調整 GRID Autosport 的體積和玩法，它並不是一款能夠隨開隨走的小品遊戲，放在後台對大部分設備而言都會是個不小的負擔。

不容錯過的良心移植佳作

無論從畫質還是從玩法上來看，GRID Autosport 都可以說是一款滿分的移植作品。而從目標平台的角度出發，它又能夠稱得上是一款處處都有考量的良心佳作。

畫質黨直接買就完事了

就移動平台的特殊形態，GRID Autosport 做了不少針對性優化。原作細碎繁瑣的畫質設定在移動版中簡化為「節能」「性能」和「圖像」三檔，畫質效果從低到高，並且能夠在遊戲過程中隨時呼出菜單進行調整。

可隨時切換的簡易三檔畫質

同時，由於遊戲本身比較吃性能，GRID Autosport 移動版默認也在畫面右上角固定顯示一個設備電量指示圖標，一定程度上可以避免在外遊戲沒有注意電量消耗導致意外關機的情況。

最後，GRID Autosport 移動版採用的是買斷制，售價 $9.99，購買即可解鎖完整遊戲、所有遊戲 DLC 以及高清材質包。根據官方給出的數據，這些遊戲內容共計包含了 100 輛賽車與來自 28 個國家和地區的真實賽道。雖然價格稍貴，但一次買斷的收費方式加上翔實的遊戲內容，對那些受不了「又肝又氪」體驗的競速手游愛好者來說，GRID Autosport 還是非常值得購入的。

DLC 擁有大量的額外內容

最後，如果你手裡的設備性能充裕且不擔心續航，可以按照如下方法解除高畫質（「圖像」）模式下的 30 幀限制：

  1. 使用任意文件管理器定位至目錄：\Android\data\com.feralinteractive.gridas\files\feral_app_support
  2. 使用文件編輯器打開 preferences 文件
  3. 搜索並定位至 <value name=”gfxconfig_high_max_fps” type=”integer”>30</value> 一行，並將其中的 30 改為 60（你也可以嘗試改成 90，風險自負）
  4. 保存退出

實測在最高畫質 60 幀率下，Pixel 4 XL 一小時遊戲耗電 18%，還算可以接受。

你可以在 Play 應用商店 或 App Store 購買 GRID Autosport，遊戲售價 $9.99。
