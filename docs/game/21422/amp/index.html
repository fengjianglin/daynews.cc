陰陽師：大佬揭秘——嘴上喊著毀滅平安京，背地裡瘋狂推土機

天天要聞
2019-12-20

大家好我是阿涼。

每到新式神上線或者全服活動的時候，彷彿玩家才會意識到——原來這遊戲真有這麼多人玩。

就像最近的決戰京都活動，靜悄悄「炸」出來一群口是心非的大佬。

這些大佬揚言在決戰京都活動期間，要給玉藻前當兒作女，將京都做聘禮贏取鯨骨開。

甚至還有大佬帶頭，揚言活動期間就待在百聞牌蜃氣樓「鬥地主」，誰勸他回來都不好使。

但是這群大佬們，嘴上喊著毀滅平安京，背地裡卻瘋狂「推土機」。

活動一開始，玩家們就直接湧向了大江山，自動成為了一個沒有感情的推土機器。

前面說要去百聞牌蜃氣樓「鬥地主」的大佬們，也默默建立了一個大江山臨時指揮群，有序指揮大家開格子，保（搶）衛（奪）京（資）都（源）。

但是，隨著路線的推進，參與活動的玩家開始分化成三個陣營。

看戲萌新

不少萌新玩家表示——這個活動到底怎麼玩？

每天上線都是一臉懵逼，看見哪裡起火就打哪裡，完全看不懂開格子路線。

反正，湊熱鬧就對了，還挺好玩的。

吃瓜玩家

阿涼敢說，現在起碼有一半的已經準備好瓜子，打算坐看「京都陷落」的劇情了。

這些玩家，打從遊戲一開始就沒打算按照策劃的想法去守衛平安京。

看來看去，玩家只覺得格子和資源才是最有吸引力的。

反正，京都陷落的劇情應該挺精彩的。

守衛玩家

最後這一波的玩家，屬於「理智派」。

這批玩家打從一開始就在認真研究「進京路線」，計算各類buff加成，分享各種高分陣容。

但是，他們碰上了「瘋狂推土機」。

任憑這些玩家如何分析，如何權衡利弊，其他玩家眼裡只有起火點或者資源。

反正，辛苦這批玩家了，開心就好。

強行劇透

據說，策劃目前已經準備好了活動的雙重劇情。

京都如果守衛失敗，則雲外鏡上線幫玩家兜底。

如果京都守衛成功，則雲外鏡開啟新的故事篇章。

那麼阿涼又要開始提（搞）問（事）了（情），你是守衛京都派還是京都毀滅派？

這個活動難道不香嗎？

網易每次推新式神總要磨磨唧唧預熱上小半個月。

說白了，其實新式神活動就跟雙十一一樣。早點告訴你活動，其實就是提醒你趕緊準備錢。

所以，式神的集結期在阿涼看來，其實就是給玩家預留了「準備」藍票票的時間。

但是，阿涼提醒各位痒痒鼠玩家——別只顧著攢票。

這幾天百鬼夜行概率UP的活動難道不香嗎？

趕緊來砸豆子！
