一局遊戲7名SKT選手：Faker排位遭遇SKT五人組

2019-12-24 2,343條評論 3,450次閱讀 2079人點贊

SKT有個很奇怪的傳統，每當SKT選手在韓服中內戰時哪邊SKT的選手多哪邊便會輸，而如今T1新人似乎也繼承了這個傳統。12月22號SKT有7人在韓服中撞車，Faker大魔王也在其中，有趣的是，7名SKT選手中有5名在Faker敵方，估計喜歡「殺隊友」的Faker要笑瘋了！

和以前的SKT內戰一樣這次T1的內戰同樣十分精彩，青訓中單Gori在前期上路遊走拿到了兩個人頭，不過饒是如此依舊打不過Faker大魔王，Faker不僅補刀壓制，看著Gori肆無忌憚的壓線更是直接一套操作將其打至殘血，最終Gori被打野搶下人頭。

上單Roach就更加嚇人了，本來Faker方的打野和輔助一同抓上，和劍魔玩起了三包一，然而結局卻是慘敗！Roach阿卡麗先是秒殺賽娜，而後再秀死艾克，最終從劍魔虎口安然撤離，打出了一段能夠上集錦的操作。

接著是Canna的劍魔，Canna在團戰中打出了無敵的氣勢，一個Q閃擊飛三人，追著5名SKT選手砍，最終幫助隊伍打贏團戰，是名團戰很強的選手！

有趣的是，到了最後Faker還虐起了泉，看得出Faker是真的非常喜歡「殺隊友」，不過，上帝是不好惹的，最終Faker被上帝擊殺了。當然，最終結果依舊是SKT人少的一方獲勝，也就是Faker和Canna的一方贏了，畢竟是內戰2V5！
