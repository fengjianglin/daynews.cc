官方公布賽季末最佳上分英雄排行，射手就佔了倆

2019-12-18 321條評論 2,535次閱讀 283人點贊

S17賽季是一個節奏非常快的賽季，不僅遊戲的節奏非常快，賽季結束的時間也轉瞬即逝。年關將至，王者峽谷也即將迎來嶄新的篇章，S17賽季即將過去，新賽季即將來臨。此間依舊有許多玩家沒有登臨王者段位，不過官方對此也給出了一份S17賽季強勢英雄的名單，希望玩家藉此機會達到滿意的段位。

亞瑟

亞瑟雖然免費送，但大家可以不要小瞧了這個英雄。從最近幾個賽季亞瑟的表現就可以看出這是一個無視版本的英雄，說他是上分的首先戰士英雄都不為過。亞瑟雖然無腦，但技能機制卻應有盡有，控制上有沉默和擊飛，傷害上有物法混傷，再加上本身的半肉出裝和續航能力，能用抗能打這個詞來形容都不為過。

前期的亞瑟依靠一技能的加速效果在清完兵線之後可以快速移動偷發育，當隊友需要支援時也可以進行支援，同時憑藉續航能力也可以輕鬆消耗線上英雄的血線。在團戰期間亞瑟開啟控制直接衝鋒，就算是前排也承受不住亞瑟的沉默技能，開團也是一個好手，在如今的勝率榜上也有49.32%的勝率，這樣的英雄用來上分自然非常簡單。

趙雲

S17賽季野核時代已然過去，崛起的都是半肉流打野。趙雲在半肉打野英雄中算的上是佼佼者，不同於手段的劉備、典韋，趙雲的機動性非常強，同時傷害也不差多少，這樣一個英雄用來打野帶節奏自然非常不錯。趙雲不僅有兩段位移，被動還有減傷能力，同時大招還有範圍性的控制。

前期的趙雲刷野速度也非常快，一般大招擊中對方一套技能就可以將其帶走。由於二技能可以回血，所以完全不需要回家補狀態，不至於拖節奏。趙雲由於被動有減傷機制，所以在後期團戰中也能在萬軍之中取敵方C位的人頭而從容離去，這樣的一個英雄在賽季末充分完全是無人可擋了。

虞姬

S17賽季完全就是射手的時代，本身發育路優勢就大，而作為萬年強勢的虞姬自然也就非常吃香了。虞姬雖然持續輸出能力很差，但勝在爆發高，同時生存能力強。這個英雄前期只要猥瑣發育，裝備成型後打團完全沒有敵手。

由於射手對線射手，虞姬的優勢也就體現出來了。一技能傷害加強之後可以用來消耗，憑藉二技能免疫物理攻擊完全有可能在前期就擊殺線上的英雄，有這個技能在也不怕有人前來gank。若是再有一個輔助從前期就跟隨虞姬，那麼虞姬在賽季末充分幾乎勢不可擋。

馬可波羅

這排名第一的就是馬可波羅了，賽季末充分射手就佔了兩位。在射手中馬可波羅可以說是一個另類般的存在，別的射手都出輸出，而馬可波羅卻是一身半肉裝。這個英雄不僅靈活，輸出能力也非常強，在如今勝率榜上馬可波羅的勝率也非常高。

前期的馬可波羅足以自爆一路，真實傷害不管面對敵方的射手還是戰士都能打出應有的傷害。同時在兩段位移之下，馬可波羅也可以所以穿梭躲避技能甚至追擊。不僅如此，馬可波羅常規出裝是半肉，團戰也能吃下一定的傷害，這樣的一個英雄賽季末沖分還是比較合適的。
