燃燒意志：初始攻擊最高的5名SSR角色，有的開始守倉庫

天天要聞
2019-12-17

諸君安好，霧夏菌報道。

這一回合是趣味內容，看一下遊戲中初始攻擊高的角色有哪些，但是不要被這個初始數據欺騙，因為有不少角色實際上很撈。

1、馬爾高

以官網數據為準，目前初始攻擊最高的是SSR馬爾高。

馬爾高現在天梯依舊可用，但不好混。PVE刷劇情，則是當之無愧的輸出第一人，持續傷害量非常給勁。對於大部分普通玩家來說，入手小馬哥是非常明智的選擇，在很多的主線劇情，以及各類PVE的玩法中，都會有非常亮眼的表現。

2、紅髮

紅髮的初始攻擊有283，當前位列第二。

四皇之一，依舊很香。雖然黑鬍子的出現，也有一定程度影響紅髮，但是紅髮憑藉完善的機制，同樣能夠穩穩地佔據自己的一席之地。PVP與PVE表現都比較出色，比較推薦入手，而且可以使用很長一段時間。後續新路飛登場，紅髮還可再獲一波加強。

3、花劍

位列第三的是花劍，攻擊達到了266。

花劍現在更多的玩家會選擇助戰。不過這名角色實際上在PVE模式中，憑藉超高的暴擊，小技能的傷害量同樣不俗。唯一的缺點就是奧義是全體傷害，與PVE的優先單點集火有所相悖。不過艾尼路的懸賞模式中，花劍倒是可以掏出來刷一波傷害。

​4、黃猿

再然後是黃猿，初始攻擊254。

黃猿現在玩的人很少了，其實原本黃猿的確挺強勢，但小技能的搭配太受限制。其次能夠與黃猿配合切後排的角色也不給力，沒錯說的就是艾尼路，所有無論是PVP還是刷劇情，都不再推薦這名角色。而且三大將綁定的天賦，也一定程度坑了黃猿。

5、艾斯

嗯？是你！沒錯，又是艾斯。這貨的初始攻擊達到了248。

然而在遊戲中的實際表現，也不盡人意。速度慢是其一大無法抹除的硬傷，也是倉庫以及下水道常客。雖然後續憑藉100級天賦有了迴光返照的跡象，可惜只照了背面，很快又涼了下去。究其根本，一個是速度意志至高無上，另一個就是新卡不知廉恥地各種比老卡強。

​菌說

這個只是官網目前提供的數據，黑鬍子的初始數值當時沒留意，所以這裡就不曉得了。實際上，這些初始數據中，除了速度的確是先天優勢，其他的並不靠譜。遊戲中真正起到決定作用的，還是角色自身的天賦、被動以及技能機制。

雷利的初始攻擊也不算高，但在遊戲中，面板攻擊卻可以遙遙領先。好了，這一回合到此結束，感謝閱讀，拜拜了您嘞~
