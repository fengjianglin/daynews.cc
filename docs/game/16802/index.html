PVP相關如何打造幾乎完美的39級戰場小號

2019-12-16 142條評論 1,395次閱讀 106人點贊

我個人覺得30-39級最強職業該是騎士。

第一，雖然40級能穿板甲防禦高一大截，但是同期的薩滿和LR也能換裝鎖甲，對手提升更大，而且37級左右的血色出品鎖甲頭、肩、胸、腿、手簡直是40級前畢業級藍裝，一堆力耐爆AP，屬性完美！！(狗男女的項鏈也能讓騎士多很多藍，多放幾個審判)

第二，武器，厄運鐘擺，39級，史詩級的4.0攻速，熟悉懲戒騎的都知道命令聖印其實很吃攻速，我一斧子砍下去，觸發一個命令就可能砍掉法師一半的血；

第三，騎士剋星SS、FS、AM，都差一點點出終極天賦，沒有冰盾的法師、沒有暗影形態的牧師就是兩個職業，至於術士，雖然不用考慮終極天賦，但是受限於裝備，凡是法傷加成的都沒耐，造成血少

第四，命令聖印，20級就能點出，完美髮揮厄運鐘擺的高傷，而戰士的致死打擊要40級啊！！！

PS：鑒於厄運已經被地精炒成海景房了，個人建議拿血色任務斧(凄涼接第一步，然後殺血色4boss)

護腕：XX之艾隆納亞的護腕 37級 奧達曼女泰坦掉落(考驗人品/碎山護腕 36級(裝綁) 附魔：+7力量

披風：蠍蟲斗篷 39綠/可汗披風 39綠/XX之薄紗斗篷(樣子很好看)，附魔：抗性+5或護甲+50？

頸部：卡莉希的項鏈/聖使護符 39級(沒錯，騎士也是吃藍的，耐力也不低)，加茲魯維的護符 36級

戒指：1、突擊指環39級；2、克恩印記 31級=烏瑞恩的圖章>叢林之戒 31級/獵鷹之鉤 39綠

飾品：競技場大師飾物(JJG打架，大毅力，要通宵來搞)/潮汐咒符(阿拉希高地下方稀有納迦王子掉落)/閃耀徽章 (+15火抗，法師火球火沖，ss獻祭)
