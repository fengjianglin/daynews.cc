王者榮耀戰令皮膚對比 第一個模型很棒 是天美最良心的一個皮膚

2019-12-11 34條評論 4,399次閱讀 1111人點贊

王者榮耀戰令限定出的皮膚其實挺多的，這麼多戰令限定皮膚大家覺得哪一個好一點呢？我覺得模型和特效最棒的就是典韋的鍵盤俠皮膚，這個皮膚可以說是天美最良心的一個皮膚，各種特效感覺就像不要錢一樣，全都給典韋加了上去，所以每次玩典韋的時候，如果對方沒有用這個皮膚，都覺得對方的人好像應該不太會玩。要知道這個戰令限定典韋不僅模型比較棒，A出去的手感和特效都是頂級的皮膚。

嬴政這個英雄也有一個戰令限定皮膚，大家覺得這個皮膚怎麼樣？我自己只玩過兩次，感覺不是太好，這個主要就是因為我用慣了其他的皮膚。這個嬴政的皮膚給我的一個整體感覺就是特效不是太明顯，當然也是有一點特效的。而且模型大家可以注意一下，也是會覺得有點彆扭，有可能是因為以前的嬴政其他皮膚都比較好，而且天美還優化了一下嬴政的模型，所以說這個才會有這樣的感覺。

呂布有一個獵獸之王的戰令限定，這個皮膚從原畫上來看是一個非常不錯的皮膚，看著是一個非常帥氣的英雄，可是整體的模型和特效的話不算特別的出色，比起兩個比較好的皮膚還有點差距，比如說比聖誕皮膚的手感要差一點，比天魔皮膚的模型要差一點。作為一個戰令限定皮膚的話，只能說是一個合格的皮膚，其他的可能就稍微差一點了。不過這是一個可以拿出來用的皮膚，比嬴政的好很多。

魯班七號的戰令限定獅舞東方絕對是一個好的戰令限定皮膚，看看這個皮膚的出場率就知道了，喜歡魯班七號的玩家如果有這個皮膚的話，基本上都是會用的。感覺比典藏皮膚都是要好一點。當然這個皮膚也不是完全沒有缺點，唯一的一個缺點就是被動A出去的時候感覺有點輕飄飄的，除了這個之外就沒啥毛病了，是一個目前版本最好的皮膚，在戰令限定皮膚中僅次於典韋的皮膚。

女媧這個英雄的戰令限定其實還可以吧，主要就是因為女媧就兩個皮膚，伴生皮膚被很多玩家討厭，那麼自然就覺得這個戰令限定皮膚看著順眼一點。這個皮膚和呂布的戰令限定差不多，原畫都是非常的漂亮，可是其他的點可能就稍微的差一點，尤其是手感方面可能不如伴生皮膚，不過特效還是比較足的，比其他的兩個皮膚好看很多。
