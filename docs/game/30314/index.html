懷舊服盜賊全系天賦、裝備、PVP攻略大全

2019-12-24 874條評論 3,201次閱讀 623人點贊

盜賊經常從暗處偷襲，率先發動狠毒的近身格鬥。在持久戰中，他們會利用一套經過精心策劃的連續組合攻擊削弱敵人的實力，以便施行致命的一擊。盜賊在選擇目標的時候必須特別謹慎，使他們的組合攻擊不至於浪費。如果戰鬥對他們不利，則必須清楚何時該隱匿或逃逸。

一、 盜賊的基礎信息

Ø 可選種族：侏儒，人類，矮人，暗夜精靈，獸人，亡靈，巨魔。

Ø 作用：近戰傷害，持續的單體傷害。

Ø 資源：生命值和能量。

Ø 護甲類型：布甲、皮甲。

Ø 武器：匕首，劍，法杖，拳套，槍械，弓，弩。

Ø 屬性：敏捷，力量，命中和爆擊率。

Ø 優勢：潛行，單體傷害，開鎖。

Ø 劣勢：缺少AOE能力。

Ø 適合的專業：制皮、工程。

四、 盜賊練級天賦推薦

劍類天賦（19/32/0）：

如果你喜歡單人遊戲的話，推薦使用這套天賦。唯一需要注意的是盡量保持切割buff。

10-41級以戰鬥天賦為主，增強技能傷害。

42-60級，剩餘的點數加到刺殺天賦中，儘快點出強化切割。

匕首類盜賊天賦（15/31/5）：

相比劍類盜賊，匕首盜賊的技能在團隊中更有作用，如果你喜歡多人遊戲，推薦使用這套天賦。相比劍類盜賊，匕首盜賊能有更好的輸出環境，所以放棄了部分的防禦性天賦，以增強輸出。

10-19級，先點出強化邪惡攻擊降低能量消耗，然後再補充另外兩個天賦增強鑿擊的效果以及背後攻擊造成的傷害。

20-45級，全都點在戰鬥天賦，降低疾跑冷卻，點出大技能。

46-60級，這時候點刺殺天賦補充爆擊和爆擊傷害以及切割的持續時間。

十四、其它

1、 毒藥/菊花茶職業任務

當盜賊等級到達20級時，可在職業訓練師處開啟毒藥的相關任務，這個任務非常重要，學習毒藥能提高盜賊的傷害或者生存能力。在毒藥任務期間還會學習到如何製作菊花茶，這是一種盜賊專屬的烹飪配方，可以使盜賊立即恢復100點能量值。

2、 搜索/開鎖

盜賊在4級和16級時，可以學習搜索和開鎖兩個技能。

搜索：到你靠近怪物時可以選中怪物使用搜索，可以在不擊殺怪物的情況下獲取一部分戰利品，例如少量的錢、食物、藥水等，不過裝備這些東西就不要想了。

開鎖：開鎖則能開啟一些被鎖上的箱子或門，你也可以通過為別人解鎖箱子進行賺錢。與他人交易時，將上鎖的箱子放在交易欄中的不可交易欄中，你就可以進行解鎖，通過這種方式收取手續費進行賺錢。

3、 專業推薦

對於缺乏AOE輸出的盜賊惡言，工程是最好的專業。工程製造的侏儒作戰小雞、鐵皮手雷等物品，能很好地彌補盜賊AOE輸出的短板。

4、 武器訓練師

對於盜賊而言，最好的練級武器即是劍，但盜賊初始只能使用匕首，需要去武器訓練師處學習使用劍類。並且武器的提升對於盜賊的收益巨大，應儘可能的提升武器，使得練級更加輕鬆。

Ø 暴風城（57,57）：匕首、劍、弩、長柄武器、法杖。

Ø 幽暗城（57,32）：匕首、劍、弩、長柄武器。
