LOL 新英雄設計概念曝光，他是來自祖安的吟遊詩人，曾是布隆哥哥

2019-12-14 579條評論 2,296次閱讀 894人點贊

英雄聯盟最奇葩的ADC英雄——厄斐琉斯，已經即將來到LPL。雖然近兩個賽季以來英雄聯盟里射手英雄的處境都十分尷尬，但現版本物理刺客遭到了大幅削弱，ADC們又有了回暖的跡象。如今ADC單挑一些操作差的物理刺客，在領先一千多經濟的情況下甚至可能與之站擼，這在之前的版本是不可能的事情。當然，雖然物理刺客被削了，但常規戰士和法師依然能輕鬆擊殺ADC。

金克絲

當殘月之肅的技能曝光初次出現在玩家視野之中時，許多玩家見到這個沒有位移、保命技能和強力硬控的英雄時，紛紛預測厄斐琉斯必將成為其它英雄的小金庫，缺金幣買裝備的時候，直接跑來擊殺厄斐琉斯就是了。但就目前測試服的表現，厄斐琉斯並沒有玩家們想像中的那樣貧弱。

厄斐琉斯設計概念曝光

厄斐琉斯設計概念曝光

近來LOL的全明星上，UZI曾拿出厄斐琉斯大放異彩。與其奇怪、複雜的技能機制相對應的，是讓人看不懂的傷害。作為LOL有史以來機制最複雜的英雄，厄斐琉斯的背後究竟隱藏著怎樣的故事？這是玩家所不知道，卻又感興趣之事。好在最近，設計師揭露了厄斐琉斯的設計概念。

殘月之肅的設計概念

關於厄斐琉斯的誕生，LOL設計師大衛吐槽：英雄聯盟（包括正在設計的英雄在內）至今仍然沒有真正的魔法師，瑞茲雖然與魔法師非常相似，但他依然不是。那麼在大衛的眼裡，究竟什麼才算作真正的法師呢？

LOL真正的魔法師，厄斐琉斯誕生

厄斐琉斯被設計師遺棄的裝備和技能

大衛直言，真正的法師應該依靠的是過人的智慧，而玩家的敏銳操作和切入時機。簡而言之，這位設計師認為真正的法師最擅長的應該是智慧，如果法師過於敏銳，反而更像是刺客或者其它職業——厄斐琉斯也正是因此而誕生。

曾是布隆的哥哥

其實在厄斐琉斯的設計之初，他並不是ADC，而是一名AP英雄。厄斐琉斯曾是布隆的哥哥

在設計技能的同時，厄斐琉斯的背景也沒有落下。對於這樣一個技能多變的英雄，究竟應該將它放到LOL世界觀里符文大陸的哪個位置最合適呢？設計師表示，厄斐琉斯最初曾是布隆的哥哥，可怕而又強悍，能像烏迪爾一樣運用弗雷爾卓德的原生魔法變幻出各種武器。

各種版本的厄斐琉斯

設計師覺得不妥，那麼符文大陸還有哪些其它的地方能給予厄斐琉斯這麼多裝備？於是厄斐琉斯又擁有了幾個設定：

  • 德瑪西亞精通武器的戰士
  • 奎因的朋友
  • 恕瑞瑪的飛升者
  • 祖安的刺客英雄
  • 來自祖安的吟遊詩人

殘月之肅

毫無疑問，設計師最後選擇的是巨神峰的孿生戰士。至於其中的原因，大衛直言：因為皎月教派的英雄實在太少了，只有黛安娜一個人。簡而言之，LOL里皎月的設定一片空白，拳頭想要充實一下巨神峰的故事。這個理由也算是無可反駁了，個人覺得布隆如烏迪爾一樣能切換獸靈的哥哥，似乎更加適合厄斐琉斯的機制。不過一千個觀眾有一千個哈姆雷特，各位玩家們怎麼看？
