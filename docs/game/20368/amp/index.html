DNF：韓服100級發布會淺談，在新內容的支撐下，遊戲能否變得有趣

天天要聞
2019-12-16

本文由Sky灬素顏遊戲視頻原創，請勿抄襲或轉載，謝謝。

萬眾期待的韓服冬季版本發布會，總算是圓滿結束了。韓服在CP系統版本實裝之後，遊戲的在線人數可謂是創下了史上新低，韓服策劃團隊也曾在採訪中表示，大招要憋到冬季發布會才能放出。

此次發布會確實展示了很多新內容：100級等級上限、100級史詩裝備、新品級神話裝備、三次覺醒、新的日常地下城、新的團隊地下城等等。從數量上看，確實對得起「大招」這一稱謂，但真正的大招並非完全取決於數量，要在內容和玩法上體現出質量和創意，玩家們才會買賬。

在發布會上，韓服團隊也對這些新內容的玩法做了一些基本介紹，本文就根據已知信息，來對100級版本做一個框架式的分析。

[100級遊戲劇情發展，新區域出現]

等級上限的提升與遊戲劇情的推進，一直以來都是同步進行的，新區域和新副本也是由新劇情來引出的。

就劇情方面來講，因為第二次魔界大戰的次元風暴嚴重影響到了阿拉德與天界，所以兩個區域發生了比較大的變化，為了應對將要到來的危機，天界重建了在內戰中變為廢墟的根特，阿拉德的各個陣營也都聯合了起來，形成了西海岸聯合陣營。新的城鎮區域和副本區域，也都是圍繞「阿拉德」和「天界」來進行展開的。值得一提的是，在95版本只存在於世界地圖上的「德洛斯帝國」，將成為100級版本中的新城鎮區域「切斯特鎮」。

共有「根特皇宮、奧核之眼、暗黑聖殿」等8個新副本登場，分別坐落於天界、西海岸聯合陣營、切斯特鎮等區域。這些副本都屬於日常副本，功能均為100級裝備和材料的獲取，在性質和形式上，類似於現95版本的泰波爾斯副本、哈林災難模式、超星空裂縫。100級版本初期應該就是打這些副本，刷裝備和材料，為希洛克團隊模式做準備。換句話說，在希洛克團本登場之前，100級版本應該也只有這幾個副本可以玩（前提是普雷裝備和CP符文都已畢業）。

這幾個副本都是日常副本，難度必然不會特別大，如果想讓玩家們覺得副本有趣，就必須要在副本的掉落物品上下功夫，這些副本掉落的都是100級的裝備和材料，從某種意義上講，100級裝備的質量直接決定了100級遊戲前期的趣味，那麼下面我們就來了解一下100級裝備的變化。

[100級裝備系統]

100級的裝備共有四種品級：神器、傳說、史詩、神話。

「神話」是新增品級，定位高於「史詩」，但這並不表示史詩裝備會被淘汰，因為神話裝備只擁有「上衣、手鐲、耳環」三個部位，並且每個角色只能選擇一個部位進行穿戴，所選的神話裝備可以與其他部位的100級史詩裝備形成套裝效果（例如：神話手鐲+史詩項鏈+史詩戒指）。

「神器、傳說、史詩」三個品級之間的強度，仍然是層層遞進的關係，沒有任何裝備基礎的角色，可以先製作神器裝備（對應[根特皇宮]副本），在有了神器裝備的支撐後，再去製作傳說裝備（對應[記憶之地、無底的坑道]副本）。

傳說裝備在100級的定位，是足以進入除希洛克團隊模式之外的所有副本，在有了傳說裝備的支撐後，就可以進入製作史詩裝備和神話裝備的副本了（對應[奧核之眼、苦痛的地下室、暗黑聖殿]副本）。

而對於在95版本就已畢業的角色而言，在95畢業裝備的支撐下，可以直接開始100級史詩和神話裝備的製作。

★裝備升級、銘刻、繼承★

1.裝備升級：就發布會展示的信息來看，在100級製作史詩裝備的地下城中，會掉落95級裝備的升級材料，我們可以將超界防具和普雷首飾/特殊裝備，升級為部分100級史詩裝備，並保留所有打造。但100級史詩裝備的種類非常多，我們用95級裝備升上去的，不一定能達到最好的效果。因此又新出了「銘刻系統」和「繼承系統」。

2.裝備銘刻：可將95級裝備的打造數值直接轉移到同部位的100級裝備上，這裡的100級裝備包括「神器、傳說、史詩、神話」四個品級，出於這一點，我們可以非常靈活地去使用這個銘刻系統。

用法1：在刷出100級史詩或神話裝備後，直接通過銘刻來轉移打造，類似於現版本在翻到普雷裝備成品後，將同部位泰波爾斯裝備的打造，通過刻印進行轉移。

用法2：因為能將打造轉移到100級的神器裝備上，同時神器裝備跨界的成本很低，所以當我們不想玩一個角色時，可以把他的裝備打造全部轉移到神器裝備上，再把繼承打造之後的神器裝備，跨界給你想玩的角色（然後需要用到「繼承系統」）。

但無論是用法1還是用法2，銘刻系統都只能實現其中一部分功能，要完全達成目的，還要結合另一個「繼承系統」。

3.裝備繼承：同部位的100級「神器、傳說、史詩、神話」裝備的打造屬性，相互之間可以自由轉移。神器的打造可以轉移到史詩上，史詩的打造可以轉移到神話上，史詩之間也可以互相轉移。

此系統主要是對「升級系統」和「銘刻系統」功能的補充，比如我通過升級或銘刻所獲取的100級裝備，並不是本職業最好的選擇，那麼在刷出更好的史詩裝備後，我就可以使用繼承系統，將原來那件裝備的打造轉移到這件新裝備上。這樣一來，無論你想怎樣變換裝備搭配，也不需要對新裝備進行重新打造。

★新史詩裝備，混合型套裝★

除武器之外，100級一共新增了35套史詩裝備，並且出現了新的套裝概念——「混合型套裝」。

從韓服發布會所展示的信息來看，「混合型套裝」並不是說1件重甲加1件布甲加1件輕甲可以成套，它是有指定裝備部位的，例如：上衣+項鏈+左槽、下裝+手鐲+魔法石、鞋子+戒指+耳環。同時，傳統的「防具套裝、首飾套裝、特殊套裝」在100級版本中仍然存在，前面所說的「35種套裝」，應該是將所有傳統套裝和混合型套裝考慮在內的總數。

就這種情形看來，100級的裝備搭配確實會非常靈活，但每種套裝的屬性是固定的，並不存在任意搭配產生隨機套裝屬性的情況。所以就100級的裝備選擇而言，具體職業仍然存在最優解，只是計算量確實會比較大。

100級史詩裝備的屬性還沒有全部放出，目前已知的屬性，除武器和神話裝備之外，就只有一套混合型裝備[預視深淵之人套裝]，這套裝備在屬性設計上，參考了80級防具[幽魂套]的暗抗增益。

就這種情況來看，100級的裝備應該不會新增屬性詞條，韓服策劃的思路，是根據為每套裝備設定的預期提升數值，來隨機或是有主題地分配現有屬性詞條，最後再讓玩家進行排列組合。這樣的方式，確實提高了裝備搭配的多樣性，但本質還是在玩數字遊戲，一旦求出了最優解，那麼最終結果必定是人人都朝最優解的方向發展。

但100級裝備的種類較多，獲取最優解搭配的過程必定是很漫長的，同時在達成最優解之前，你也可以通過穿戴已獲取到的非最優解裝備，去體驗100級史詩套裝的快感，並且在達成最優解之前，可以通過「繼承系統」來任意變換裝備搭配。所以整體而言，100級的史詩裝備就像是為你規定了一個最終目標，但在達到最終目標的過程中，安置著各種各樣的彩蛋。個人認為，如果裝備的屬性設計都比較有特色，那麼用這種方式過渡到希洛克團本登場，是完全沒有問題的。

[100級職業三覺]

​根據發布展示的信息，韓服第一批進行三覺的職業是女鬼劍和男格鬥，女鬼劍的三覺已經完成製作了，就發布會的演示來看，三覺技能的演齣動畫非常華麗，但演出時間都比較長。如果所有職業的三覺技能都採用類似的模式，還是會導致許多問題的，但願韓服在後續的設計上能夠進行改良和優化。

另外，全職業是「分批三覺」還是「同時三覺」，成為了一個爭議很大的問題。就韓服的更新方向來看，「分批三覺」應該是跑不掉了，但國服和韓服之間畢竟存在著5到6個月的更新時間間隔，如果韓服能在這個時間間隔內，把所有職業的三覺做完，到國服更新時，一次性把全職業三覺都進行實裝，也不是什麼難事。畢竟就國服的遊戲環境來講，分批三覺確實會導致很多問題。

[總結]

韓服此次100級發布會所展示的內容，從玩法方面來講，基本就是這三大類別了。這次新版本給我的整體感受，確實要比普雷版本和CP系統版本好很多，雖然很多東西至少要等韓服實裝之後才能定性，但就目前所展示出的整體框架來講，100級版本的結構是比較完善和多樣化的，滿級擴張和職業三覺也是吸引脫坑玩家回坑的兩大情懷向內容。如果後續內容能夠設計好，沒有虎頭蛇尾，那麼個人認為100級版本會成為一個可玩度較高的版本。

其實相比之下，我反而比較擔心國服的春節版本。就正常的更新進度來看，春節版本只有CP系統這一個新內容，要用CP系統來度過100級版本在國服實裝前的空白期，恐怕是一件非常困難的事情，希望國服在這個問題上能夠謹慎處理。

能夠讓100級版本提前在國服實裝，自然是最好的，如果比較難做到的話，也可以酌情提前更新一些100級的碎片內容。比如因為有「霸王契約」的存在，95級角色本來就可以穿100級的史詩裝備，國服提前更新一部分強度適中的100級史詩裝備，再新增一個活動副本用於獲取這些史詩裝備，讓國服玩家們在版本過渡期提前體驗一下100級史詩的強度，既解決了版本過渡期無聊的問題，又提高了玩家們對100級版本的期待。當然我這裡只是隨便舉個例子，類似的方式是有很多的。

最後祝願DNF能在100級版本中煥發出新的光彩，本文對100級版本的分析就到此結束，感謝大家的閱讀。
