LOL舉報技巧，讓你的舉報不再石沉大海

2019-12-12 1,349條評論 2,486次閱讀 328人點贊

作為一名英雄聯盟遊戲玩家，我們在娛樂的同時，最受關注的無非是自己的遊戲體驗。但LOL畢竟是一個團隊遊戲，在遊戲過程中，經常會因隊友的一些消極行為，給我們帶來不好的遊戲體驗。對此很多玩家都會選擇通過裁決之鐮，來舉報有消極行為的隊友，但處罰效率卻不太讓人滿意，很多人因此對舉報系統失去信任。

其實不是裁決之鐮不完善，處理效率不高，而是您沒有找對方法。今天就讓我們一起來學會如何用正確的方式，維護自身的權益。

首先我們要知道裁決之鐮是如何運作的，裁決之鐮大致可以分為兩個部分，分別是玩家舉報系統和後台檢測系統，通過兩種系統的配合來給出最後的懲罰。具體如何使用，我們下面分開來講。

舉報系統

玩家舉報系統是我們經常使用到的，使用方式就不再贅述，我們重點說一下如何去正確填寫，來提高舉報效率。填寫方式根據具體情況可以分為三類：

1.不適言論：這種情況比較常見，我們只需在遊戲內複製聊天日誌，等遊戲結束後直接粘貼「聊天日誌」到備註欄中即可。

2.故意送人頭：此種消極行為的判斷是比較困難的（按照現有規定：只有鑽五以上的玩家舉報，才會被受理，並且只限排位局），如果你段位低於鑽石，只有當系統提示您「檢測到某遊戲玩家存在故意送人頭行為」時，您的舉報才會被裁決。所以一般如果系統沒有提示你，舉報基本不會被受理，會被系統默認為玩家「真的菜」。

3.掛機：對於掛機的消極行為，只有玩家超過10分鐘未曾登陸的舉報才會生效。並且處罰力度是疊加的，不會一次性生效。所以舉報掛機行為基本沒啥必要，除非是經常掛機的玩家才會被處罰。

最後對於使用外掛的行為，大部分玩家基本遇不到，這一塊系統檢測的很嚴謹。

系統檢測

對於檢測系統，與玩家沒有直接的關係。大概可以分為四類，我們只需要在舉報時盡量向這四個方面靠攏就行。主要包括即時反饋檢測、送人頭檢測、暫離檢測和裁決系統，分別對應辱罵、送人頭、掛機和外掛等四個方面。所以其他方面的舉報基本都是無稽之談，只有這四類舉報才會被真正的受理。寫舉報「小作文」時大家可以盡量向這四個方面靠攏一下。

而對於消極行為特別嚴重的玩家，我們可以選擇不同的舉報方式，下面我們再來具體看看，除了在遊戲內舉報，還有那些舉報途徑。

1.官網客服舉報，這種舉報方式是比較立竿見影的，也是處理效率最高的，只需要提供對局時間、玩家ID和wegame的遊戲截圖即可。效果也比較明顯，只要舉報合理合規，一般都會在24小時內處理完成。

2.TP騰訊遊戲安全中心，直接到安全中心舉報，有一個好處是你可以直接和遊戲後台工作人員對話，效率也比較高。並且對經常維護遊戲環境（也就是經常合理舉報）和遊戲信用分較高的玩家，會定期送出一定的皮膚獎勵。

至於其他的舉報途徑，就不一一列舉了，這兩種方式再加遊戲內舉報足足夠用。

小結

總的來說，相比於其他的遊戲裁決系統，英雄聯盟在這一塊兒做的還是比較好的。只要掌握了正確的方法，我們還是能輕鬆的獲得一個乾淨舒適的遊戲環境。當然最後還是要告誡一下大家，任何的舉報都必須是合理合規的，不能出現捏造事實的情況，系統對捏造事實的玩家也會處於一定的懲罰。

好了，今天就說到這裡啦，希望對大家有幫助，讓我們一起維護好英雄聯盟的遊戲環境，讓自己有一個舒適的遊戲體驗。
