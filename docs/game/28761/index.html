勿把CSGO當參照，CFHD應當發揮"長槍局"優勢

2019-12-21 1,017條評論 2,201次閱讀 799人點贊

FPS一直都是遊戲領域當中的翹楚，但是目前國內FPS領域還處於一個相對空白的狀態，國外的FPS一哥CSGO在國內並沒有那麼受歡迎，甚至連一個正規的職業聯賽都沒有；而國內有正規職業聯賽的CF又難進入主流媒體，現在的絕地求生又遲遲沒有國服，所以國內FPS領域這一塊兒還值得深耕。

這三個項目誰能率先發力，值得探討，絕地求生除非出國服，或許能梅開二度，但是在世界範圍內，這個遊戲熱度已經驟降，除非藍洞願意把遊戲交給小馬哥來運營，即便是今後出了國服，很可能就是下一個”穿越火線”，國內外的熱度冰火兩重天。

PUBG跌出twitch前十

CSGO這款遊戲發源於半條命、CS，對於早期的FPS玩家來說，這款遊戲才是”根正苗紅”的FPS大作，CSGO這款遊戲也確實很不錯，有著獨特的經濟系統，經典的射擊手感和投擲物設計，不管是競技還是娛樂，從遊戲本質上來說，確實算得上是FPS一絕；但之於國內來說，運營商孱弱，steam+國服出了這麼久，這款遊戲都沒能像CF和絕地求生一樣大火起來，除非發生一個什麼難預測的大事件，比如國內哪支隊伍在Major奪冠，CSGO很難在國內復刻國外的熱度。

在劉大大看來，目前最有希望吃這塊蛋糕的還是CF，首先這款遊戲的玩家基礎深厚，國內玩CSGO的，多數曾經或多或少都玩過CF，端游超5億的註冊量、三億滑鼠的槍戰夢也並非虛傳，在LOL沒有出來的時候，2009—2011年的網吧，幾乎被穿越火線霸佔！

當然，現在的穿越火線要東山再起，難上加難，一是畫面落後，二是付費屬性讓這個遊戲的口碑跌入谷底；能重回巔峰的是解決了這兩個痛點的CFHD，穿越火線高清競技大區，提升了畫面，去除了一切付費屬性元素。

說起CF，拋開英雄級武器而言，很多人都喜歡將其對比CSGO，在競技性上，CSGO的一個經濟系統讓比賽變得非常有看點，每一次比賽有數十種比賽輪番登場，觀賞性也很足；而CF裡面，國內比賽基本只會出現兩種槍：M4和大炮，AK偶爾會有人用，國際賽同樣只有兩種槍：AK和AWM。

經濟系統是CSGO競技上的優勢，手槍局打長槍隊能贏，看起來確實很棒；但沒有經濟系統也是CF在玩法上的優勢，但CF每一局都是長槍局同樣保證了單局絕對的公平；很多人看CF清一色的M4和AK，嘲諷其槍械設定無競技性，如果CSGO去除了經濟系統，哪個CSer又不願意把把起狙呢？

在CFHD測試的這幾天，劉大大看到很多說CFHD在競技性上沒有經濟系統的說法，以及很多CSGO有而CFHD沒有的元素，沒有才是正常的，如果有，那就是徹底的輸了，一模一樣的玩法，為什麼不玩CSGO呢？

劉大大個人認為，CFHD應當徹底發揮自己長槍局的優勢，CSGO有經濟系統作為牽制，讓很多槍都有機會登場，如果CFHD不引入經濟系統也想要有這樣的效果，那就需要在地圖和槍械屬性上面下功夫，霰彈槍、衝鋒槍在近距離的屬性增強，保證在近距離的公平對槍下，提高衝鋒槍和霰彈槍的獲勝概率。

除此之外，我還看到策劃在CFHD裡面做出了一個很有意思的改變：手持衝鋒槍可以奔跑。

這對於職業選手開局搶點很重要，手持衝鋒槍是遊戲中的最快移動速度，那麼衝鋒槍就有機會登場；

提高霰彈槍出場率也很簡單，地圖設計中把關鍵點位設計為”狹小空間”，同時好好設計霰彈槍的彈道散射和威力衰減，遠距離威力小，散射大，但近距離可以做到一槍斃命；此外，想要提高出場率還有一個方式，為這些比較冷門的槍械設計精美的皮膚。

有些人喜歡經濟系統，但也有人喜歡全程長槍局的酣暢淋漓對槍之快感，這只是兩種不同的模式，無所謂優劣，遊戲本身沒有高低貴賤之說，只有玩家競技水平和素質才有高低區別，CSGO很優秀，但CFHD不應以此為參照物，當充分發揮自己的優勢！
