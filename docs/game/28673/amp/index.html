王者榮耀冬冠杯：XQ，TS首日都是全勝，eStar輸掉一場後前景不妙

天天要聞
2019-12-22

王者榮耀2019年冬季冠軍杯第一天的選拔賽已經是全部打完了，其中有幾場KPL戰隊之間的內戰還是非常激烈的，之前也給大家發過幾場的戰報，這裡我們來看看後面幾場的吧。

XQ之前已經是贏了2局，這場和CW的戰鬥也是比較輕鬆，前期限制住了娜可露露的發育，拿到暴君後又拿1血，雖然CW先破塔並殺掉來先手的盤古，但XQ還是打出1換3，馬可波羅發育極快，很快就拿下黑暗暴君和主宰繼續推進，最後是靠太乙真人的神控，XQ擊殺CW多人後拿下比賽，選拔賽三連勝。

Hero久競和eStar一戰非常關鍵，因為Hero之前輸過一場，他們再輸的話就基本沒希望了，這次他們是上了久誠，前期就吹飛貂蟬並擊殺拿到一血，之後多次入侵野區搶藍buff，還拿到幾次暴君，慢慢有了優勢，推進方面也做得更好一些，最後是靠豬八戒的輸出和控制連殺eStar4人，然後一舉攻破水晶，保留了晉級的希望。

TS和RW俠一戰也是KPL內戰，前期RW俠拿到第一條暴君，西施也拿下一血，不過中路TS十三公孫離一波蹲草推到兩人，配合隊友殺掉扳回局勢，雙方又在中路打團，神人呂布直接大後排成功，TS打出1換4後拿下主宰。最後一波的團戰又是在主宰處，暖陽各種天秀再次打出1換4，TS這波是直接攻破上路高地拿下主宰後帶兵線點爆水晶，首日2連勝。

第一天的最後一把是eStar對陣Nova，eStar開局下路抱團拿到一血，暴君則被Nova搶到，中期雙方各種拉扯，黑暗暴君處eStar再逼團，花海李白各種消耗打出0換2，擊殺黑暗暴君後又在中路擊殺3人，eStarPro就靠這波直接推掉水晶，也是贏得了第一場勝利。

第一天比賽打完，XQ和TS是全勝，並且他們都有一局是在和KPL隊伍較量中獲勝的，晉級的希望要大一些，而RW俠，eStarPro和Hero久競都是輸掉了一局，但依然保留著進入正賽的希望，關鍵還是看後兩天的比賽吧，你們覺得晉級的是哪兩支隊伍呢？歡迎大家關注和分享，遊人將繼續給大家帶來王者榮耀冬季冠軍杯比賽的最新消息。
