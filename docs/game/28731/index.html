鬥魚德杯：廠長小弟成隊霸大哥，SKT教練成他最後救命稻草

2019-12-22 186條評論 4,215次閱讀 2294人點贊

2019年低最後一項賽事德杯如約而至，作為新賽季的季前賽，新組建戰隊的磨合賽事，大家都在嘗試新的陣容，所以各家戰隊都在積極備戰，想在新賽季打個一好彩頭，基本上LPL賽區戰隊都加入其中，不過也有一些例外，比如RNG的UZI就沒有參賽，這是對主力隊員的保護，除此之外大部分選手都已參加！

剛剛進行完一天的比賽，老牌LPL戰隊基本上沒有什麼太大的晉級懸念，WE、VG、RNG、JDG都輕鬆晉級下一輪，這幾支戰隊在這賽季，其實都有巨大的陣容變化，比如WE雙C離隊，VG下路組合去了IG，而EDG的小昭來到了VG，RNG則是卡薩離隊，下路UZI也缺席了比賽，而JDG在imp退役之後，也迎來自己的老朋友loken！

但是憑藉老牌戰隊的底蘊，他們還是有驚無險的拿下了比賽，這一次鬥魚獨播的德瑪西亞杯，呈現出了群英薈萃的局面，而且新人也都大放光彩，RNG的新打野小龍包，作為一個新人，並不怯場，打得十分自信和果敢，成為了RNG戰隊的主要戰鬥力，同樣問題小虎的低迷還沒有結束，輸出傷害再一次讓人無法直視，RNG新賽季能夠走遠，小虎的狀態很成問題！

作為VG戰隊的唯一大佬，目前的小昭，可謂是隊內的老大哥，這位2017年夏季賽冠軍，多次參賽世界大賽的超強ADC，在EDG一直沒有表現出自己的完全實力，總感覺他的實力被壓制無法展現自我，而如今在VG戰隊，他會得到了前SKT力宏教練的調教，在隊友的幫助之下，相信未來會有一個很好的表現！

而小昭在接受採訪的時候表示，自己並不是隊內大哥，依舊還是小兄弟一枚，對於這位有天賦、愛說騷話大心臟的選手來說，他要把握好這一次來到VG的機會，不然的話自此沉淪再也沒有重現輝煌之日了！
