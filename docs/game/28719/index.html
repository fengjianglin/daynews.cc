LGD賽後採訪：小組賽被打懵了，沒調整過來

2019-12-22 185條評論 1,804次閱讀 595人點贊

在One Esports失利之後，LGD教練和領隊採訪。

關於這次比賽失利的原因

首先是國內外不同賽區的隊伍風格不一樣。我們打完WESG之後，體系暴露得也比較多。到新加坡下飛機後又有一個小更新，會對打法和體系產生影響。這也暴露出我們的調整和應變能力不足。在國內兩三套陣容打起來比較順暢，來這邊之後就會被針對。

在這個版本更新之前，前哨周圍的團戰爆發比較頻繁。LGD是一個比較喜歡打架和打團的隊伍，我們會在前哨附近進行逼團和打架等打法。現在前哨的錢符和經驗都有削弱，逼團和打架都相應減少了很多。更重要的是經濟上的運營和地圖上的控制(都有很大變化)。

還有，國外隊伍的對線打得很靈活，他們經常剛三、優三或者劣勢路勾兵。包括我們和EG打比賽，我們賽後看錄像發現，他們ROLL點選天輝，然後游中就會非常方便，那一把Maybe的影魔被針對得非常慘。

明顯看出別人有備而來，研究得比我們透徹。他們有出門反視野，包括開霧游中等操作。而我們中國隊就太常規了，出門默認212開局，對中路的保護和對地形的研究都沒那麼深刻。

也可以說我們LGD不是一個天賦異稟的隊伍，相反，勤勉才能帶來力量。需要回去多多思考，多多訓練吧。

這個比賽小組賽只有兩天，前期準備不足，被打懵了，第二天也沒做出及時的調整。

以TI為周期的話，這個比賽是在容錯率之內的。我們需要加強自己的應變能力和調整能力。放低自己的姿態，在天梯和訓練賽好好訓練吧。

關於組隊時長對選手激情的影響

這是大家在一起的第三個賽季，我們這個賽季還能在一起打，大家之間肯定是做過很多考慮跟溝通，最後才決定下來的。五個人一起創造了LGD隊史最長組隊記錄，也在創造全球最長記錄，擔憂(激情減少)是多慮了。

關於TA這個英雄

我們知道他們很擅長TA，我們賽前也有準備幾套陣容去應對。但是第一盤又有選手整活，想嘛玩大屁股。那一盤效果是把TA壓製得很慘，然後我們團隊推進節奏也起來了。但是那一盤打起來和傳統的LGD和中國隊不太一樣，就比較耿推，我們自己也不太適應那個節奏。我們也在嘗試一些新的體系。至於帕克，帕克打TA…額….帕克削了甲和技能傷害，就會打得比較難受，如果是之前的話，會稍微好一點。

關於是否有找國外賽區的隊伍打訓練賽。

跨區打主要和東南亞的隊伍，東南亞的強隊也基本都在中國訓練。想要找國外的隊伍訓練的原因是覺得我們比歐洲或者獨聯體賽區的隊伍相比更落後。解決的辦法是以職業選手為基準，改善天梯環境，加強訓練態度和想法。

對粉絲說的話

大家的支持是隊伍前進的動力，有一些(尖銳的)聲音對隊員也是一種鞭策。我有時候希望隊員更加努力，有時候又很心疼他們，就比較矛盾。希望隊員和粉絲一起，都能享受這個遊戲帶來的樂趣。職業選手就以更加職業的態度去面對(比賽)。我相信我們肯定可以在之後的比賽中回饋給大家。
