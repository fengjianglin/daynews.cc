前言（懶人直接最後一張圖看結論）：

1.三個號都是頂級打造（附魔白金寵物時裝紋章等等）但是增幅有點不一樣的，皮夾手鐲耳環12武器鈍器 輕甲耳環右槽12 重甲全身11 但是通過拿收集箱寶珠卡片等可以調整到力量獨立都一樣 在保證技能順序以及跳X數量差不多的情況下 可以默認為三個號打造一樣 舉個例子：無論你是帶25力寵物裝備還是增幅提升25力 對於結果都是一樣的 所以裝備這塊就不要質疑了

2.目前普雷副本以及CP本都是磨血+打樁的輸出模式 所以測試環境選取

①20秒極限打樁②30秒無覺醒不平X跳X只放技能

雖然很多人反感打樁 但是沒有更好的方式能相對公平的測試 打樁是目前公認度最高的了

3.手鐲耳環均為大地 其中重甲右槽項鏈也為大地 皮夾號缺一張勞動收集箱粉卡 少了2屬強 收集箱也快到期了 春節套也快出新的了 但是勞動的收集箱估計一時半會續不上 也可能春節出新的屬性 所以三個號均不帶五張粉卡10屬強 結尾有爭議最大的輕重帶了10屬強收集箱測試的彩蛋

4.每一張打樁圖背後都有五張以上的測試 盡量排除白字暴擊的影響 所以不要質疑嚴謹性 一是國服快要取消這個設定了 二是雖然實戰也存在 但不是拿來炫耀的理由

最後結論：

個人評價：

1.不可否認屬強拉滿的輕甲在打樁方面確實很優秀 但是重甲皮夾也不差無論磨血佔二成還是三成 尤其是很大一部分帶10白字稱號的玩家重甲是最合適的 另外我打了不下500次普雷 滿屬強的玩家是真的鳳毛麟角

2.在此之前我也看過很多測試 但基本都是打造沒有嚴格意義的相同或者普雷沒升級完 他的結論只適用於他測試的普雷進度下的差距 大部分玩家關心的應該是升級完以後的差距吧

3.很多玩家沒有白字紋章覺得抬高了重甲 然而並不是 ①我雖然盡量避免了白字暴擊但是這麼多段的攻擊肯定是有技能白字暴擊的 這個設定取消以後輕甲會略低一點點②重甲的高額百分比力智屬性單人是要比屬性詞條略低的 帶上系統奶跟奶會無限接近其數值③CP符文的加入 都是非一二覺技能的增強對於整體提升肯定是重甲比輕甲高的

4.已經打造了的玩家沒有換甲的必要 這差距是真的不大 不到半年就開放100級 以及馬上的年套稱號只要換個非爆傷詞條都會重新洗牌 提一下布板：按照頂配 布甲是要比輕甲低5-7%的 板甲就算屬強拉滿不帶紋章也要比重甲低1%左右 而且屬強拉滿的玩家是真的少

5.雖然馬上又有新的風暴 但我還是花了幾個小時測試 算是對我剛開95版本就在群里說過重甲不差放心升級的負責吧
