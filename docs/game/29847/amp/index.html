簡談《控制》：“紅色”主題中的虛幻世界，朦朧面紗後的天馬行空

天天要聞
2019-12-24

人們從未停止過對未發生事物的幻想。從《回到未來》（1985年美國拍攝的穿越電影）在到當今火熱的穿越電視劇，劇情中充斥著各種遐想內容。這些飽含深意的構思引導著人類從不可能中探尋未來。而如今，正因為電子遊戲這個獨特的「形態」，更加利於思想家們表達著這一切。

隨著2019年進入尾聲，遊戲界各種獎項的評選也落下了帷幕。遊戲評分界大佬IGN也公布了本年度評選結果。其中，《Control》（控制）贏得了年度最佳動作冒險與藝術指導獎。對於這個結果是否公允暫不去討論。現在我們將一起，從遊戲玩法和內容出發進行相關簡談，來看看控制為何能奪得佳績。

《Control》

製作組的啟程

本遊戲PC端發售於2019年8月27日，由之前《量子破碎》的工作室「Remedy Entertainment」開發。這是一家來自於芬蘭的遊戲製作公司，由於規模原因，公司製作的遊戲目前為止並不多。但其發售的遊戲品質均為上乘，雖然在宣傳方面比不過其他3A級商業大佬，但依舊憑藉遊戲自身品質，為廣大玩仍舊帶來了精彩的遊戲體驗。

logo

該製作組除了近年的《量子破碎》和《控制》以外，還有其它幾款讓人有所耳聞的遊戲。其中之一的《心靈殺手》在MAX-FX和Havok與UOBooster引擎的協助下，把遊戲畫面在2012年時表現出極高的質量。配合遊戲以驚悚冒險故事為背景，電影推進式敘述為講述手段，將遊戲以一種極具氣氛渲染的方式展現在玩家們面前。而後遊戲本身還經歷了一段小插曲，由於遊戲中第三方音樂版權到期的因故，遊戲不得已進行了短暫的下架操作。

心靈殺手

除了玩家們熟知的大作之外，該遊戲製作組真正算得上是啟程之作的就要數這款名為《Death Rally》戰車射擊遊戲。目前我們仍舊能在Steam商店中尋得這款作品的身影，雖然在steam商店頁面顯示的是2012年發行，可實際這款遊戲早在1997年就被開發了出來，只是當時登錄的是MS-DOS系統，所以沒有被大眾所熟知。遊戲也不是單單的賽車競速，而是融入了車輛和武器系統的改裝系統。玩家能在遊戲中對自身的車輛進行改裝升級，該遊戲也於2011年時登錄過Android、iOS移動平台。

《Death Rally》

一路反觀，包括最近的《控制》在內，Remedy製作的遊戲為何都擁有出色的電影式故事劇情？這就要得益於製作組內一位叫做山姆·雷克（SAM LAKE）的故事編劇。包括《心靈殺手》和《馬克思佩恩》在內，遊戲內出色的劇情內容均是出於他手。我們不要簡單的認為「瞎編」劇本是一件容易的事，看似合理的背景、恰當的人物衝突和正確的故事走向等，之中包含的邏輯關係相當複雜。正因為有這樣的苛刻原因，造就了市面上繁多種類的遊戲。有的製作組擅長畫面；有的擅長戰鬥部分，而本作（控制）擅長的則為將天馬行空的故事講述給玩家們聽。

意念扔東西

☀戰鬥部分

遊戲屬於第三人稱射擊類。主角擁有一把多形態可切換手槍，玩家可按自己喜愛或戰鬥需要進行切換。因遊戲背景為科幻題材，所以主角的能力中就擁有了操控場景內物品以及瞬閃和浮空等能力。手槍在本作中作為唯一的遠程類武器，實際使用起來由於有能量的限制，當場景內敵人數量眾多以及出現擁有防護盾的士兵時，單純的利用手槍輸出則顯的略有疲軟。這時候就需要利用我們主角的超能力，「利用意念吸取場景內的物品對敵人進行攻擊」則能產生高額的傷害。

戰鬥

遊戲中在主角受到傷害後，左下角的藍色血條會減少，由於遊戲中沒有自動回血的機制，只能通過擊敗敵人後，撿地上敵人屍體掉落的藍色小光球回血。全程能對我們玩家產生較大威脅的大致可以分為二種敵人，一種會投擲手雷等爆炸物；一種是狙擊手，擁有單發超高傷害。

因為本款遊戲的核心並非為戰鬥，所以全流程的遭遇戰並沒有很難，只要玩家不盲目的扎進怪堆之中，BOSS戰時採取穩紮穩打的方式，是可以比較流暢通過遊戲流程中的戰鬥。

☀劇情（WARNING：劇透部分）

主角

《控制》作為一款以劇情故事為主要推動力的遊戲，和其他同類遊戲不同的在於遊戲開頭並沒有交待整個世界觀和背景，玩家將控制「傑西」（主角）一臉懵的走進一幢建築物中（聯邦控制局 FBC），還「莫名其妙」的坐上了局長的位置。

和老美科幻影視作品中的情節一樣，政府手下各種局子總是扮演者令人臆測不到的身份，其中就包括了本遊戲主要場景。我們全部故事也將在這座名為「聯邦控制局」（太古屋）的大樓中上演。這座大廈理論上空間無限大，並且能隨意變換。在這裡上班的人主要研究其它異世界對人類世界能產生的影響。而我們主角和她的弟弟在小時候就因為一個偶然的機會，在垃圾堆中翻出了一部投影儀。通過在投影儀中插入不同的幻燈片就能開啟通往其它異世界的通道。

異世界：即多元宇宙，也可稱為平行宇宙。因為宇宙由「原子」組成，偶然某次在科學家觀測到原子沒有固定狀態時得出的結論。既然原子存在各種形態，那麼宇宙也不會單單只有一個。

需要值班盯著看

在控制局中擁有著各種奇怪的東西，其中有一種必須盯著看的冰箱，這個靈感來源可能就來次網路上流行的SCP系列。

提到SCP這個組織，就不得不佩服創造它的人腦洞之大，簡直是胡編界的祖師爺。SCP基金會的口號便是：收容、控制、保護。最有名的當屬和這個冰箱有異曲同工之處的SCP-173，從你和它目標對接的那一秒開始，你只要眨眼，就會聽見脖子扭斷的咔嚓聲。

主角將從和控制局連接在一起的「靈界」中學習到各種強大的作戰能力，靈界中的原住民則被稱為「委員會」。他們會通過特殊的交流手段（黑色金字塔和紅色電話），將各種信息傳達給主角。我們可以把它理解為製作組為了讓玩家不齣戲所設置的教程。

委員會

之前我們提到的投影儀在姐弟的使用中，誤打誤撞的開通了兩個特殊的異世界。充滿善意的精神體「北極星」和極具攻擊性的「希斯」。而後由於投影儀被其他壞孩子搶去，壞孩子身體和精神成功被壞精神體腐蝕，而後趕來的姐弟兩也沒能有效地阻止這一切。為了防止災難蔓延，姐姐燒毀了除36號幻燈片（能通往北極星老家）以外的所有片子，但依舊沒能阻止「希斯」這種能控制人精神的敵人蔓延。

前任局長

姐弟所在鎮上的異常引起了控制局的關注，而後趕來的相關人員抓走了弟弟，姐姐則趁機逃跑。這個丟下親人的愧疚讓姐姐在之後的十多年中倍感內疚，所以也一直在尋找著抓走弟弟的聯邦控制局。

由於之前北極星的緣故，姐弟倆都擁有異於常人的超能力。姐姐一直在社會中正常生存，而弟弟被抓去後就變成了研究對象，並不擁有正常人的生活體驗，這也為之後弟弟暴走埋下了伏筆。

北極星精神體可視化

當年還是孩子的姐姐逃脫難道真的只是僥倖？實際則為控制局有意為之，這種兩手準備的好處就在於對弟弟的培養失敗後，還有姐姐能做備選人物。而我們主角「傑西」（姐姐）也正是第七任局長候選人，一切的一切看似雜亂無章，實則都據可循。當年燒毀的幻燈片中，有一張只燒毀了一半，那就是通往「希斯」老家的那張。控制局獲得投影儀後，手賤打開了通往希斯世界的通道，最終導致入侵事件的發生。而和主角進行精神連接的北極星，則更像主角的第二人格，我們能在過場動畫中頻繁的看到主角的「自言自語」，這種對能力的追求感，也驅使主角坐上新任局長的位置，所以解救控制局也變的順理成章。

希斯精神體

由於精神體無法被可視化，因此在遊戲的製作過程中製作組將這種「敵人」進行了實體化。類似於傳染病的「希斯」能通過精神上的控制，入侵操控沒有攜帶海德倫裝備的人類。在敵人被擊殺後，實體會化作類似於失去焦點的煙雲般消失。敵人出現時，也採用了相似的出現手段，從天空中出現紅光後降落。

海德倫裝置：拿到36號幻燈片的控制局通過對北極星世界的研究製作出了「海德倫」裝置，該裝置可以傳導和增強北極星的精神力量， 讓攜帶這種裝置的人類不被希斯所感染。

圈內標記即為海德倫裝置

慢熱的劇情是不想讓你錯過

遊戲對於開局時劇情的交代看似會讓人覺得無比糟糕。實情卻不是如此，毫無頭緒的開場激發了人類潛意識中的求知慾，難度適中（實則有點簡單）的戰鬥暗地裡起到可供玩家進行情緒釋放的地方。哪一種類似的情況會讓人很難受？無非就是即將得到而又得不到的時候。

除了本身就是電影互動式類型遊戲以外。劇情式推進的冒險探索遊戲，過多的劇情過場、情節交代、概念灌輸之類，易讓人產生煩躁感。為了達到「安撫」玩家情緒的目地，往往會在遊戲中加入「泄壓閥」（可以是戰鬥部分），讓玩家進行調節和釋放情緒。

以《古墓麗影》為例，「暗殺、槍戰、砸門、砸箱子、宏大場景長焦鏡頭等」。

場景

製作組故意的設置，迫使玩家只能通過場景中散落的檔案，和遊戲劇情的推進來了解這個遊戲的世界背景。對於劇情的解讀也需要玩家對各個已獲取的資料進行聯結，這些過程始終貫穿在玩家明知道這是一個科幻的遊戲世界裡，但它依舊激發起玩家對結局好奇的探索。

《黑衣人》

你會看到各種各種各樣的因素，都推動大都會隊取得世界冠軍，即使他之前每個賽季都墊底。——格里芬

本作的劇情借用同為科幻題材《黑衣人》中的一句台詞。從主角坐上局長那一刻我們就知道她能力不凡，一切朦朧的劇情也早已註定。如果你是一個細心的人，就能從中看出一定的端倪來。這樣一部看似沒什麼大深度的遊戲，卻會因為主角的內心對話；過場動畫時眼神不經意的閃躲；和幾句令人差異的台詞，改變你對這個「簡單」遊戲的看法。

主角台詞

主角在怪像叢生的太古屋內，孤身一人戰鬥的情況下說出了這句話。除了她已經知曉這一切或是對自己能力十分自信外，誰還能開心得出？是否北極星早已告知了主角一切我們無從知曉，但在這看似朦朧的劇情世界內，編劇天馬行空的想像力卻表現的淋漓盡致。

本作與其它同類遊戲最大區別就在於，別的遊戲是劇情講清楚遊戲，而《控制》便是典型的遊戲講清楚劇情。原因就是該製作組擅長藉助遊戲講故事，特意將一些看似無用的消息停留滲透在玩家腦海中，等之後再獲取相關材料時突然恍然大悟，原來劇情的交代是這樣為了某種目地的存在。這種需要玩家自身參入其中進行穿針引線的工作，便是本作最大魅力的所在。

結語

雖然遊戲按線性進展，也沒有重複遊玩的價值。但本作也一改筆者之前對於該類遊戲的認知，原來遊戲內信息的吐露與揭發也需要自身探入其中，一起跟隨主角獨特的內心獨白，一路拉開遮擋著劇情的「朦朧面紗」，探頭紗窗外、斜風細雨，一切都剛剛好。
