《初體計劃2》遊戲測評：品質在製作組的奮鬥中獲得提升

2019-12-24 1,150條評論 2,344次閱讀 355人點贊

2017年7月18日，一款國產”美少女”題材ACT遊戲《初體計劃》橫空出世，但可能是由於團隊首部作品的原因，加之遊戲品質也較為一般，當時並未激起太大波瀾。但遊戲品質在不斷的完善下，最終還是獲得了steam多半好評的成績，同時也斬獲了一票動作遊戲愛好者。而現在他們帶著《初體計劃2》又回來了。

Steam售賣界面

的的確確的”全面進化”

這次《初體計劃2》的副標題名為”全面進化”，實際上手後也的確能感受出，本作相較於前作真的是全方位進步了。角色設計方面不再走一代所謂的”美少女”路線，雖然還是標準的亞洲審美的角色面容，但這一代的人設更加合理也更具設計感，用開發者的話來說，本作的主角更像是個戰士了。

地圖場景布置

遊戲講述了一個超能力大陸發生的故事，菲洛索菲亞超能力學院的三名裁決者戰士，為了執行任務，獨自踏上了對抗了十年之久的塞恩提亞帝國的土地，玩家將會扮演其中的隊長和兩位夥伴出生入死並肩戰鬥，逐步揭開雙方勢力埋藏了十年的秘密。

豪華的音樂陣容，不輸大作的音樂體驗

遊戲在配音配樂方面可謂是全明星陣容，釘宮理惠和內田真禮參與的角色配音，小林未郁創作的主題曲，以及《尼爾機械紀元》、《死或生》瑪麗羅斯的音樂團隊的加盟，使得本作在演出方面的視聽體驗堪稱優秀。同時作為一款動作遊戲，動作方面的表現也算比較令人滿意，雖然沒有那些一線動作遊戲那樣豐富邏輯的連招，也沒有優秀的視聽回饋，但起碼對於國產遊戲來說，動作足夠流暢漂亮。

劇情

《初體計劃2》提供了4把武器——單手劍、巨劍、拳套和弓箭四種武器各具特色，用武器看時的動作乾淨利落，觸發斬殺動畫時的運鏡方式也十分乾脆，從中多少能看出一點《忍者龍劍傳》等頂尖動作遊戲的影子。但實話實說，遊戲的打擊感被視聽回饋方面拖了後腿。首先遊戲內敵人受傷硬直並不明顯，加上本作的打擊音效水準實在是稱不上優秀，這些導致了遊戲在打擊感方面在目前版本並沒有達到應有的水平。請允許我強調一下上一句話的表達，”目前”的版本並沒有達到該有的水平。

Bug仍然存在，但製作組的用心玩家看在眼裡

今天的《初體計劃2》的確有蠻多的bug，的確打擊感一般，的確有著這樣那樣的問題，但摸著良心說，雖然不推崇遊戲沒做完善就拿出來賣這種做法，但《初體計劃2》的開發組聽取意見方面的態度以及團隊執行能力方面，真的值得單獨提出來誇一誇。遊戲發售的前一個星期我提前拿到了遊戲最高完成度版本的DEMO，進入遊戲之後反人類的按鍵、字幕消失等小問題一個接著一個，它們接連挑戰著我對遊戲的耐性。

遭遇NPC

試玩過後一肚子憋屈的我也就毫不客氣的把成噸問題直接甩給了開發組，而令我沒想到的是，短短三天的時間，只有9人的開發組便解決了絕大多數問題，如今我看到的除了一個依然有著種種問題的國產ACT之外，還有一個態度誠懇，執行能力極強的年輕團隊，如果開發組能保持現在的態度和幹勁，再沉澱上那麼幾代，再積累上那麼幾年的技術力，《初體計劃》系列還是非常有潛力的。

動作模型還算可以

最後我們作為遊戲評測回歸遊戲本身，《初體計劃2》的確相較前作有著全方位的進步，但摸著良說，依然稱不上是一款優秀的動作遊戲。儘管如此，它依然坐擁日本頂尖配音配樂陣容，並且有著相較gameplay部分更優秀的劇本、演出，技術力和開發經驗可以慢慢積累，最重要的則是希望開發組能夠繼續保持勢頭，努力更新、優化遊戲。

拳套打擊感略差

以後保不齊初體計劃系列就能和《最終幻想15》一樣，靠後續的更新不斷完善遊戲，最終拿出漂亮的品質贏得市場的肯定呢？遊戲國區售價僅為68元，對於這樣品質的遊戲而言很具有性價比，購買過一代的玩家也都很期待遊戲之後的動向，這對於國產動作遊戲領域而言是非常不錯的發展，希望他們能夠堅持初心！

動作設計乾脆利落

豪華陣容配音配樂

製作組虛心聽取意見

缺點：

小bug仍然存在

敵人種類很少

敵人防禦建模有些僵硬
