王者榮耀：模擬戰終於出正式版了！你S0的段位承接下來是多少？

2019-12-23 448條評論 1,933次閱讀 821人點贊

哈嘍，大家好，我是小魚兒~

周末閑來無聊，小魚兒瘋狂排位了一下六連勝，美滋滋呀。不知道大家周末有沒有瘋狂和好朋友嗨皮嗨皮。今天小魚兒不講英雄，來講講這賽季新出的王者模擬戰，雖然現在大多數人已經熟悉玩法了，但是正式服相信還是會非常吸引人的。畢竟還是有所改變。

小魚兒一開始也是被王者模擬戰吸引到不想打5V5的排位賽了。現在王者模擬戰正式版本即將上線，今天小魚兒就來和大家說說，正式版上線後都做出了哪些更改吧~

大家都知道，王者模擬戰和排位賽一樣，也是有段位的，所以在新的版本上線後，段位也是會承接的，像小魚兒，目前段位是鑽三，等到S1正式服開始後，小魚兒的段位就會變成鉑金一。看看下圖，紛紛來對比下自己的段位吧~（圖片來源官網爆料站）

正式版變化之一：棋盤變大了！

原本王者模擬戰的棋盤是7×6，正式版變成了8×8。這樣看來，估計小魚兒以後得用平板來專門玩王者模擬戰了~本來手機就那麼點大，這也太為難手機了！看看圖片對比感受下~

變化二：新增陣營令牌

之前變化都是令旗，什麼輔助令旗等，大家都知道的，現在的變化是陣營令牌也就是說，如果你被卡牌抽牌制裁了，可以趁機選擇陣營令牌來給自己補充陣營人數。

但是這個新增的改變陣營的裝備，只有在萬物迴轉的回合才有機會拿到，並且！傳送帶只有傳到你面前才可以選擇，所以到時候的站位是佔一定的運氣成分的。

變化三：新增等級天賦

這個很好理解，就是每次升級選擇天賦時又增加了許多選擇，增加較多的基本都是cp黨，直接上圖，小夥伴們了解下，可以確定自己正式服時選擇玩什麼cp黨。

變化四：增加了天元賽

天元賽具體長什麼樣，小魚兒目前還真不知道，體驗服排位排了2分鐘都沒有進去，小魚兒真是太難了！

但是天元賽是有段位選擇的，青銅黃金鑽石王者，其實這個天元賽大概就是獲取天元幣兌換一些專屬的皮膚獎勵等。

小魚兒溫馨提示：王者體驗服已經開始了模擬戰的S1賽季，大家有號的可以趕緊去試試，大棋盤看起來確實是非常的爽啊！

好了，今天的文章就到這裡，喜歡小魚兒的點個關注喲。♥
