日本的中學生太可怕！不僅會拯救世界還會做遊戲

2019-12-17 1,488條評論 2,811次閱讀 1638人點贊

看動漫的都知道，很多日漫里都以高中生為主角講述拯救世界的故事，其實現實中的日本高中生也很強！

今日，一場面向全日本初高中學生的遊戲開發大賽的最終評選會在秋葉原舉行，出場的都是15到18歲的中學生，展示了他們通過自學和利用課餘時間獨立開發的遊戲，很多遊戲的質量竟然還很高！不信你往下看！

顛覆（北海道/函館拉薩爾高中隊「 Kats」）

這是一個簡單的規則益智遊戲，玩家可以通過操作可愛的等邊三角形角色來移向目標。特點是略有不同尋常的運動，其中角色可以順時針/逆時針旋轉並沿著牆移動，並且可以玩各種類型的地圖。

馬特瑪雷（東京/富山高中）

這是一款避免彈幕的遊戲，遊戲的主題是「數學」。所有的彈幕都有一個數學元素的階段，玩家可以任意指定生成彈幕的公式的係數，然後彈幕會發生相應變化，增加彈幕難度。

不知道早晨的星星（沖繩/ N高中隊）

這是一款動作類的角色扮演遊戲，男孩和女孩在遊戲中揭露黑暗世界的真相。這款遊戲從編程到背景建模，還有3DCG和效果的所有內容，都是由初中三年級的學生製作的，未來他們的目標是給遊戲加入情感，來達到更高的遊戲水平。

青蛙青蛙（東京都/豐島女子中學）

這是一款3D動作類角色扮演遊戲，玩家可通過魔術轉換為一隻青蛙，但僅能夠保持5秒鐘。遊戲過程中要通過合理轉換人和青蛙的角色，來通關遊戲。

自衛隊-免疫系統模擬器（千葉中學）

這是一款2D策略遊戲，其中玩家成為「樹突狀細胞」，作為免疫細胞的控制塔，並與入侵人體的細菌作鬥爭。通過稱作為免疫細胞的「嗜中性粒細胞」和「巨噬細胞」攻擊敵人。

協同機器人（廣島縣/廣島縣高中）

這是一款通過滑鼠操作移動，並通過單擊滑鼠左鍵來跳躍障礙的小遊戲，小心不要從舞台上掉下來哦！球加速的速度的時候還可能飛到特別的地方，會給玩家帶來令人耳目一新的遊戲體驗。

看了這些真是不得不給他們豎個大拇指！相信在未來，他們一定可以實現夢想，成為優秀的遊戲製作人。

曾經有日本媒體做過調查，日本學生們夢想中的職業是什麼，其中大部分男生都表示想做遊戲。看來在日本兩大遊戲巨頭的影響下，遊戲的魅力征服了不少人。而這些為了夢想在努力的中學生，說不定就是未來的遊戲巨頭呢！
