大家都知道，DNF的國服話語權越來越大了，作為SPK的年底壓軸更新，12.10的「全新起航」版本成了玩家們期待的改版，小夥伴以為會是全職業CP技能副本的上線，但體驗服12.3實錘了，並沒有新副本，看來要拖到1.14作為春節版本的主打內容，但作為史派克的特色版本，這一次專門為混子玩家和平民玩家著想！比如再次加入混子專屬裝備——輪迴史詩！

12月10號更新，加入6件新史詩！與兵法湊成95最強光環套

兵法套是95級唯一能使用的光環裝備，但在12月10號更新之後，他就不是唯一了，史派克再次為混子玩家專門製作了6件新史詩，不是防具部位而是特殊裝備和首飾，一共2套史詩6件裝備，屬性非常不錯，名字是都帶「輪迴」兩個字，SPK明顯告訴玩家，混子與BOSS同在，一直輪迴生生不息！

95新首飾的套裝效果是減少5%的防禦！千萬別小看5%的提升，驅魔、天帝、暗帝曾經因為擁有減防技能成了雙倍漩渦首選職業，雖然只有5%，但實際提升應該在10%左右。而首飾分別是增加5%的三速、100點力量和智力、400點HP和MP，沒有輸出能力，標準的輔助裝備。

而12.10新增的特殊裝備套裝效果則是減少10點抗性與附加霸體效果，可以給普雷主C良好的輸出環境，散件的效果分別是增加50點三攻、10點所有屬強、400點HP和MP。

以上6件史詩會在12月10號更新後加入，這難道就是史派克的全新起航！指的是混子玩家，再度起飛直接混普雷了，因為配合兵法5件防具，這的確是95級最強光環套。

在普雷副本中，如果紅眼主C組上一個擁有11件光環裝備的光環或者是帕拉丁，提升至少得70%以上！減防被動、三速、技能攻擊力BUFF、本身34輔助技能等一系列強力提升，混子玩家普雷輕鬆掛樹翻牌，只要跟著紅眼跑就行了。

12.10新增的6件史詩獲取方式和兵法套一樣，如果異形結晶不夠的玩家要趕緊刷黎明裂縫了喲，可惜SPK少更新了一件武器，不然這光環套的屬性會更強，比如來一個光環武器，被動是1~85技能+1的BUFF加成，比星辰套還給力，那麼何愁玩家混不了普雷呢！
