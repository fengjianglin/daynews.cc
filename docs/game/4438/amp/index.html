值得一提的是大家都沒有玩自己平常玩的位置，faker玩了打野盲僧，Doinb玩了上單努努，Theshy玩了ADC卡莉斯塔，Clid玩了上單龍龜，peanut玩了中單艾克，而NaRakyle玩了下路蛇女。Doinb雖然玩了上路，但是心在Theshy身上，開局直接藏在下路用努努推雪球。儘管沒有拿到擊殺，但是也打出了M神的閃現，而他自己則傳送上線。Doinb對於Theshy是真愛了。

這把比賽LCK隊伍玩起了LPL隊伍擅長的東西，4分鐘就開始五人包下了，直接打出了一波小高潮，而LPL隊伍的劣勢也由此開始。雖然LPL一方在後續依靠Doinb的推球和打野的控圖節奏打回一些場面，但三射手陣容在面對LCK隊伍以蛇女為核心的陣容時沒有翻盤能力。遊戲進行到20分鐘左右的時候，LPL一方已經在經濟上落後一萬了。而蛇女則是成為了這場比賽的真神級別的存在。

征服者蛇女在排位中的威力早就有所體現，而這次職業選手在娛樂賽中又拿了出來。在下路經過了幾次大亂斗之後，征服者蛇女直接統治了這場比賽，最終拿下了19殺0死的戰績。在遊戲即將結束的最後，蛇女Q到了Theshy的卡莉斯塔，隨後補了兩個E直接秒殺了卡莉斯塔。這就是征服者蛇女的威力，無論是上中下哪條路，蛇女的強度已經超出了正常的範圍，而9.24版本的削弱恐怕也是不痛不癢。

最後得談一談Cat老師了。這把遊戲Cat老師玩了新英雄賽娜，和Theshy配合走下路。也許是因為離開職業賽場已有一段時間的原因，Cat老師的對線處理實在是有點辣眼睛。最離譜的一波操作是對龍坑開大，被faker走位多掉了大招的範圍。Cat老師退役的原因終於找到了，可能就是年齡大了操作不來。這個時候貓媽媽們就可以保護起來了。
