自由而真實的江湖來了《我的俠客》帶來全新沙盒武俠遊戲體驗

2019-12-21 193條評論 1,889次閱讀 361人點贊

在國產遊戲發展的初期，武俠單機遊戲無疑是最具代表性的類別，仙劍奇俠傳、軒轅劍、劍俠情緣傳等經典作品為中國遊戲的發展培育了寶貴的人才，近年來武俠單機遊戲卻大都不怎麼樣，比如仙劍奇俠傳6就讓我大失所望。今年6月，電魂網路研發的沉浸式單機體驗的武俠RPG手游《我的俠客》突然聲名鵲起，都2019年了，到底什麼樣的國產遊戲能有這麼大魔力？我玩過之後，不得不說國產遊戲給我的驚喜越來越多！

我一進入《我的俠客》，撲面而來的就是濃濃的復古風，樸實的國畫風格立繪讓我想起了玩DOS版仙劍的時候，沒有人會抱怨李逍遙長得不夠帥。建立角色時回答問題能帶來初始能力加成，這也是90年代的戰棋類RPG中非常常見的設定，這讓我回憶起了伴隨80後長大的武俠熱潮，不假思索的選了行俠仗義！

在世界設定上，《我的俠客》採用了架空世界觀的設定，但無論是風沖還是五絕，這類角色名總能讓我想起那些經典武俠小說，華山派風沖這個名字有意思啊，莫不是風清揚和令狐沖合體了？在新手教學中你可以看到，主角的父母設定就像是加強版張翠山、殷素素，就人設來說，這款遊戲和俠客風雲傳的風格很像，無時不刻都在向經典致敬。

《我的俠客》的名字看起來是源自我的世界，沒錯，《我的俠客》主策劃何錚解釋過，這個名字確實就是想讓玩家感覺到這個遊戲非常自由，就像玩我的世界一樣，是一個真正的沙盒武俠RPG。雖然《我的俠客》初入遊戲時看起來和很多武俠作品類似，主角只是在一個偏僻小村中長大，做著大俠夢的少年或少女，也有主線劇情。你可以自由選擇下一步去什麼地方，雖然與NPC對話不會耗費時間，但探索迷宮是要耗費時間的。

在遊戲前期我習慣性的先找存檔功能，但是居然沒有！沒有存檔，意味著你並非一戰失敗就會看到”請大俠從頭再來”，但也意味著單機遊戲中最重要的SL大法不存在，因此探索迷宮失敗浪費的日曆時間將會積少成多，最終影響到結局。這就像真實人生，做錯事可沒後悔葯吃！

主角在設定上類似金庸群俠傳，開局只有最基本的武學，上乘武功全靠自己學，不過主角還有著情緣這種網遊中常見的項目。在《我的俠客》中，每個NPC的設定都很用心，你也可以和各個NPC通過不同的交流方式增加好感度，連村裡的狗都能刷好感度變成戰鬥寵物！不過既然是國產遊戲，應該不會有後宮結局吧……其實我也很想當韋小寶啊！跪求給設計些隱藏結局！

既然是開放世界，那麼正邪選擇全看你自己，不同的選擇都會對主角的仁義值造成影響，比如你面對地痞欺負小孩都無動於衷，那怎麼可能成為大俠？這個遊戲的劇情選擇點極多，雖然我沒有選擇成為邪惡boss的手下，但我用腳想也能知道要是這麼選擇，後果就是城裡NPC拒絕和我對話，還要被官兵通緝……

確實按武俠小說的規律，修鍊邪派武功能速成，但這樣我就沒有正義的夥伴了！我想像了一下，自己當邪惡boss這也是件很好玩的事，待我2周目後再試試。按照常識，東方不敗，哦不東方無敵那裡一定有更好的邪派武功秘籍，東方無敵又沒出現在拜月神教駐地，我先得和任窈窈搞好關係才能接近東方無敵，這些我得先拿小本本記下來，免得到時候忘了又沒有攻略可查。

江湖很大，我可以隨便遊覽各地的名山大川，有機會遇見奇遇提升武功，能學各個門派的武功，聽說這個遊戲比俠客風雲傳還要自由，我的多研究研究。送禮提升NPC好感度，學習他武功這個玩法我在俠客風雲傳中見過了，但是這個踢館玩法我可沒見過！只需隨便找個門派駐地，上去就是干，只要打的過，甚至能直接當上這個門派的掌門或是徹底滅掉這個門派？滅門還能拿到門派傳家寶，簡直太爽了！

要是我能滅掉江湖中全部門派難不成自動成為武林霸主？不過我多玩玩就發現，這個遊戲其實本質是個網遊，修改器自然沒法用，而主線任務的時間有限，顯然是做不到這麼逆天的事。雖然《我的俠客》玩法非常自由，但是世界設定是相當真實的，區區幾年時間你的武功怎麼可能超越張真人？也許藉助轉世，打個十周目大概有機會吧……

沒有修改器要變強只能靠自己，武功秘籍我所欲也，神兵利器亦我所欲也，武俠RPG中裝備一直很重要，在《我的俠客》中裝備製造帶有隨機屬性詞條，加上強化重鑄熔煉，神兵利器的打造也可以玩好多周目。只是通關也許並不需要極品裝備，但是和其他玩家打擂台時就需要了！這個遊戲儘管偏重與NPC的互動，但是也沒有忘記玩家之間的互動，這個設計我喜歡！

《我的俠客》中還有很多讓我驚喜的地方，比如武當派能號召江湖人士直接成為隊友，這在一般武俠遊戲的門派設定根本見不到。對於《我的俠客》這種博採眾多遊戲優點的創新武俠遊戲，如果你玩過傳統單機武俠RPG，那麼這個遊戲絕對不容錯過，即便你只玩過MMORPG手游，也可以來感受一下這個遊戲帶來的真實江湖感。
