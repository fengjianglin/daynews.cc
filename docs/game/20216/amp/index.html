電競·LPL|S9總決賽LPL官方紀錄片《英雄，新生不息》發布

今日（12月17日），《英雄聯盟》官方發布了S9全球總決賽LPL官方紀錄片《英雄，新生不息》，這部紀錄片記錄了FPX、IG和RNG三支戰隊的S9征程，並講述了諸多不為人知的故事。

在今年的S9全球總決賽上，FPX為LPL賽區奪得第二座冠軍S賽總冠軍獎盃，讓無數的玩家為之激動喝彩，而明年的S10全球總決賽也將再次落地中國，屆時希望LPL賽區的戰隊繼續努力，將冠軍留在家門口。

單機·家用機|騰訊旗下3A作《重生邊緣》上架Steam

由騰訊旗下NEXT Studios打造的科幻策略射擊新作《重生邊緣》（英文名：SYNCED:Off-Planet）開啟Steam介紹頁面。

在遊戲中，玩家將於瀕臨奔潰的世界中，面對敵人和納米人的威脅。《重生邊緣》將PVP和PVE通過”控制”元素進行巧妙結合，並重新定義了PVPVE遊戲的整體玩法。

在這場多人對抗之旅中，既有緊張刺激的近戰、各類型的納米人入侵，還能通過”控制”改變戰場形勢，從眾人中脫穎而出，創造屬於自己的故事。

《重生邊緣》運用UE4遊戲引擎，搭載NV光線追蹤技術，玩家將隨心所欲的遊歷2040年的未來世界，體驗古老的森林、輕科幻風格的建築物，未來與自然環境巧妙融合的獨特環境。

單機·家用機|Epic每日送遊戲活動即將開啟

從去年12月開始，Epic就不斷加碼他們每周的喜加一活動，截止本周Epic已經免費送出57款精品遊戲，總價值577.42美元，其中不僅有老牌3A大作蝙蝠俠系列，也有口碑神作《蔚藍》以及《深處》。

而就在上周，Epic又公布了一個更加給力的贈送活動——從12月19日開始，Epic將連續每天送出一款免費遊戲，至於將要送出具體遊戲的名稱至今還未公布。

不過也有玩家對此表示了調侃：Epic：確認您的訂單，支付0元 玩家：點擊支付 epic：謝謝。玩家：fxxk epic。

網遊·手游|過去十年最受歡迎手游是誰？

前不久，數據分析公司App Annie公布了一份數據報告，時間跨度為過去十年，在這份報告中，過去十年用戶下載量最多的遊戲為《地鐵酷跑》（Subway Surfers），因為其在印度市場取得了非常不錯的效果，為其貢獻了總下載量的15%，之後為《糖果粉碎傳奇》和《神廟逃亡 2》。

而在十年手遊玩家消費排行榜中，《部落衝突》、《怪物彈珠》、《糖果粉碎傳奇》位列前三，騰訊出品的《王者榮耀》排在了第六位，網易的《夢幻西遊》排在了第七位。
