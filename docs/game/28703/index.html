元氣騎士：春節版本最先了解！大廳花圃變大，14位角色新增裝飾

2019-12-22 1,041條評論 3,585次閱讀 949人點贊

「分享遊戲故事，彙集歡聲笑語」~大家好，我是菜鳥熊貓~

歡迎來到由菜鳥熊貓主講的《元氣騎士》大型欄目《元氣搞笑日常》第162期~

Hello，各位小夥伴們，大家好！我們國家一年一度的春節呢，即將到來，各大遊戲廠商的針對春節也都會有一些大的更新，其中的《元氣騎士》作為近兩年來非常優秀的一款單機遊戲，他的遊戲廠商涼屋遊戲工作室也是在春節的時候會迎來重大的更新，每年春節更新都是全方面內容的升級！今天熊貓我就看到了元氣騎士的春節版本的內容，下面我就帶大家來看一看會出現的好玩事情。

春節版本搶先看，機器人夢見機器羊，新增的彩蛋

​

你沒有看錯，春節版本機器人將會出現一個彩蛋！他將會在夢中遇到他的機器羊，不過涼屋將它的夢境變成了現實，打造了一隻機械羊放在機器人的身旁。你以為這是隨意打造的嗎？其實並不是，故事來源於美國作者-菲利普 迪克所寫的小說《仿生人會夢見電子羊嗎？》，不了解的小夥伴有可能看過世界頂級的科幻電影《銀翼殺手》~這就是涼屋設計關於機器人和機器羊的彩蛋

熊貓我認為機器羊將作為機器人的專屬寵物出現在遊戲中

大廳花圃變大，14位角色新增裝飾，花圃增加至8塊

​

在春節版本中，元氣騎士整個大廳的擺放都做了一個調整，總的來說大廳和花圃的空間的都變大了，可能是為未來推出新的角色而做的準備，除此之外，大廳當中所存在的14位角色，他們都有了自己新的裝飾，比如說牧師擁有了自己的牧師講台，遊俠擁有了自己的坐墊，鍊金術士有了自己的藥劑架~

​

在大廳當中也出現了一些新的房間裝飾品，比如說大廳左上角的藥劑售賣機，右下角的水族箱！它們既是裝飾品，也是新功能放置處！

在鍛造房間當中也出現了一個成就公示欄，玩家可以在這裡直接看到自己所獲得的成就。

​

在大廳的下方也新增了一個專門用來存放雕像等特殊玩法的房間。熊貓在這個房間當中也看到了一個占卜師的檯子，還有一些傳送門模樣的石門。這個房間應該就是春節版本更新之後出現新玩法的房間，應該會出現類似於副本的模式！

​

還有就是花圃當中的耕地總算是增加了，在原有6塊的基礎上直接增加到了8塊。但就目前看來，8塊花圃的仍舊是不夠用的！

得嘞，今天熊貓了解到的關於《元氣騎士》春節版本的一些內容就是上面這些了，我們現在還沒有實際上手，所以說很多功能究竟如何還未可知。不過花圃擴增是板上釘釘了~

PS：原創手打文章，杜絕抄襲轉載，你的關注就是我最大的動力
