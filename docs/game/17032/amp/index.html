懷舊服DKP奇聞！男QS拍下碧空，私下送女FS？QS：怎麼用是我的事

天天要聞
4 天 ago

DKP與GKP制度之爭從懷舊服開始前就一直存在，經過了三個多月的實戰，越來越多的玩家選擇支持GKP。最近又有一個奇聞，恐怕要加劇玩家們對DKP的反感了。

這個奇聞的主角來是一個小公會的騎士，由於是個小服小會，使用DKP制度，為了吸引混合職業，這個公會規定奶騎可以和法師一起出分競爭碧空。DKP制度下，法師肯定飈不過奶騎分霸，於是在一次真的出了碧空的時候，奶騎拍下了碧空。

畢竟公會規則規定了奶騎可以拿碧空，所以大家都沒發表什麼意見。然而此後這個奶騎並沒有換碧空，還是用的光環石錘，等到活動結束後，團隊里另一位法師裝備上了碧空。

經過了解過後，大家發現是奶騎私下把碧空給了法師。這個行為本身就已經讓大家感到很厭惡了，再加上大家都知道，奶騎是個男玩家，而法師是個妹子，這就更加引起大家的不適了。對此，奶騎自己的解釋是，按照規則，奶騎是可以和法爺飆分的，拍到了碧空以後，碧空也就成了奶騎的武器，如何處理也就只是自己的事情了。

相信這個解釋連我們這些吃瓜群眾這關都過不了，公會讓奶騎和法爺飈碧空已經非常友好了，結果奶騎還這樣噁心同會的法師。公會管理們在幾種處理措施之間猶豫，但這位騎士肯定是在這裡留不下來了。
