鬥魚黃金大獎賽首日：OMG登頂榜首！4AM狀態回暖，只有韋神在挨揍

天天要聞
2019-12-16

北京時間12月15日，絕地求生鬥魚黃金大獎賽S8迎來了小組賽正賽首日對決，第一天比賽將在A組間進行，不得不說A組可謂死亡之組，不僅有OMG、4AM、QM、iFTY等強隊，剛剛拿下微博杯冠軍的RNG也在A組。

從第一局比賽開始網友們就感受到了A組競爭有多麼激烈，在微博杯上表現不佳的4AM也是成為了玩家們關注的焦點之一。

前三局比賽中4AM整體還是有些拉跨，除了第一局拿到8分之外，後兩場都是只有3分，所幸的是第四局他們終於開始發力並先後拿下10分、13分和6分，其中第五局還以2人隊拿下一雞。

值得一提的是，今天4AM選手除韋神之外三人手感都恢復不少，狀態逐漸回暖也是讓他們在6場結束後拿下43分暫居第四。讓網友笑慘的是韋神今天的遊戲體驗可以說是極差，各種倒地各種死，真的是隊員都在秀，只有韋神在挨揍啊！

此外鬥魚簽約戰隊OMG今天的表現極佳，6場比賽中4場超過10分還吃下了一雞，最終他們以34個淘汰和59分暫居榜首，網友也是直呼夢回2018年，希望他們狀態能夠繼續保持。

接下來鬥魚黃金大獎賽比賽仍將繼續進行，明天比賽將在下午5點進行，哪支戰隊能追上OMG呢？各位千萬不要錯過拭目以待吧！
