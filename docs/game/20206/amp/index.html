QG冬冠大名單出爐，1dao和snow入選，總決賽失利或主動求變

天天要聞
2019-12-17

KPL秋季賽總決賽已經圓滿落幕，AG超玩會戰勝QGhappy拿到了隊史第一個冠軍。總決賽的舞台上有人得意就有人失意，AG超玩會的隊員們自然是喜上眉梢，但是QG的選手們就比較落寞了。尤其是對於刺痛和770的質疑聲一直都沒有停過，而QGhappy不能因為這次失敗就倒下，接下來還要備戰冬冠杯，大名單也新鮮出爐。

QG的冬冠杯名單顏色是一片大紅，頗有一點春晚節目單的意思，好像在給大家提前慶祝新年。可能這也是為了讓粉絲們開心一點，不要因為總決賽的失利而難過。大名單本身的選手並不算太意外，教練組還是那三個人。在隊員方面，除了總決賽的五個人之外，還有兩名選手就是1dao和sonw，都是比較常規的。

QG剛剛在總決賽失利，隊員們不受一點影響是不可能的，所以冬冠的比賽中也是有可能進行輪換的。C位的位置上1dao只是在常規賽有過登場，季後賽還從來沒有上過場。1dao的個人實力是毋庸置疑的，並且相比與刺痛，他的英雄池更加全面。總決賽放狠話環節老帥說「一個只會射手不會野核的隊伍不應該拿冠軍」，對於QG的粉絲來說聽上去很刺耳，但也是事實。刺痛的野核就是一直沒練出來，現在距離冬冠根本沒幾天的時間，1dao還是有可能得到出場機會的。

而輔助位的770總決賽的表現也不盡如人意，他是指揮型輔助，擅長運營。但是這個版本對於輔助位的進攻性和開團能力要求比較高，這些恰恰都是770不擅長的。770的加入對於QG是提升不假，但是他是個優缺點都很明顯的選手，年齡大了操作確實有點跟不上。當然，QG應該還會繼續給770機會，snow冬冠登場的可能性不會有妖刀大。就像是刺痛說的，雖然總決賽輸了，但是他們也是亞軍，也要比剩下的隊伍更強，冬冠並不一定非要作出調整，到底怎麼安排還得看訓練賽的狀態以及心態了。不過按照以往的規律來說，拿亞軍對於一支隊伍的心態衝擊是比較大的，在接下來的冬冠很可能不會有太好的成績，就看QG能否調整過來吧。
