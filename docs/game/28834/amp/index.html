騎射手養成手冊，最強輕騎兵是怎樣煉成的

天天要聞
2019-12-21

在《亂世王者》中，很多朋友都對曾經的”騎射大隊”記憶猶新，一度是人氣最高的兵種之一，今天我們就來盤點騎射手養成和用法的核心要點。

1、 兵種簡介。

騎射手自帶的技能中，【衝鋒】是輸出端的核心技能，觸發時可以優先攻擊敵軍抗性較差的後排單位，後續大部分加強都是以此為基礎展開的。

【遊騎兵】是騎射手優化後的技能，通過”減傷”來增強騎射手的生存能力，配合騎射手的站位，陣容合理的情況下可以極大程度減少騎射手的戰損。

2、 武將、技能、兵法和軍師計策。

1、 武將搭配。

騎射手的【衝鋒】最大的作用是選擇攻擊目標，所以比較依賴武將提供的輸出加成。當前版本中，”秦宣太后”羋月，依靠強大的傷害、暴擊加成和”蓮花箭”的攻擊效果，是當之無愧的騎射手武將核心，一隊輸出爆表的騎射手通常都是以羋月為核心進行搭配的。

羋月搭配韓信、花木蘭和趙雲都能打出不錯的輸出效果，其中三段傷害都可以觸發【蓮花箭】的韓信，是羋月最好的搭檔。

花木蘭和趙雲，可以組成騎射手的二隊，也有不錯的表現。這4個武將對於騎射手的作用是其他武將很難企及的。

2、技能搭配。

當前版本的騎射手技能搭配，是有標準答案的。核心武將主要學習【突襲】、【陷陣衝鋒】和【騎射手攻擊】。

即便要進行”資源戰”撞田，由於資源田敵軍實力往往不強，讓輔助武將或二隊武將學習【掠奪】或【殲滅】就已經夠用了。

3、 兵法。

相對於單人撞城，騎射手更偏向集結戰鬥。考慮到監軍府的加成，騎射手武將通常會領悟的兵法是：增加騎射手攻擊的【游擊】，增加騎射手生命的【跋涉】和增加騎射手防禦的【擊落】，以及增加全兵種攻擊的【雄兵】，雄兵對兵種的加成，也會受到監軍府的增益效果。

註：圖為槍兵兵法的武將，委任在騎兵府時，【雄兵】依舊可以為騎兵提供加成。

4、 軍師計策。

騎射手專屬計策【金蟬脫殼】，對騎射手進行了攻防屬性、輸出和減傷的全面加強，加成幅度也很大。

體驗服已經測試的新計策【胡服騎射】，加成幅度略遜於【金蟬脫殼】，單人戰鬥起效的限制比較大，有點是可以通過【點將台】研讀獲取，成本較低可以作為一個補充。

3、 龍脈、監軍、神獸和科技。

和其他兵種一樣，監軍、神獸注靈和科技，都屬於”內功”範疇，唯一的要點就是抓住一切機會進行提升，無比重要，也不存在方向上的選擇。這裡主要介紹一下和騎射手特色相關的部分內容和龍魂附加詞條的選擇。

1、 騎射手特色。

騎射手的很多加成，如龍脈六件套效果、青龍注靈100級解鎖的【宿敵】等都是針對弓弩進行攻擊的，這讓騎射在面對弓弩為主的陣容時，可以打出更高的殺敵數字。

騎射手承傷端，有減傷技能的加成，但同時需要注意他是一個後排的遠程騎兵。需要通過陣容搭配來合理承傷，曾經風靡一時的”騎射大隊”就是把這一點做到了極致。切忌把單排騎射手作為唯一的後排單位迎戰對手多排騎兵，或者在面對多排槍兵時讓騎射手作為唯一騎兵上場，被對手的龍脈”飛槍”、項羽”投槍”集中攻擊。

2、 龍脈附加詞條。

騎射手的龍脈中，騎射手攻擊、騎兵攻擊和騎射手生命是必選的，第四條，最好在騎射手防禦、騎兵生命和全兵種攻擊中選擇品質、屬性比較高的那個來進行搭配。

以上就是騎射手養成的要素，通過以上內容，配上合理的陣容，騎射手就是那個讓敵人無可奈何的最強遊騎兵。
