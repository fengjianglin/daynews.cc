創意周活動太坑爹，剛出皮膚就整體驗活動，這以後還敢買？

2019-12-09 0條評論 16次閱讀 0人點贊

摘要

最近王者榮耀里又出了一個新活動，叫王者創意互動周。雖說是這麼個叫法，但是實際上本次活動的內容跟四周年的活動內容基本相近。本意上都是想要回饋玩家，讓玩家體驗一下傳說級皮膚的手感，讓一些學生黨0元黨體驗一下傳說級皮膚。結果卻好心辦成了壞事，惹得大量網友對其不滿，究竟是怎麼回事？

最近王者榮耀里又出了一個新活動，叫王者創意互動周。雖說是這麼個叫法，但是實際上本次活動的內容跟四周年的活動內容基本相近。本意上都是想要回饋玩家，讓玩家體驗一下傳說級皮膚的手感，讓一些學生黨0元黨體驗一下傳說級皮膚。結果卻好心辦成了壞事，惹得大量網友對其不滿，究竟是怎麼回事？

究其原因，還是因為排位的問題！本身人家策劃的原義就是讓你試一下英雄的手感，讓你感受一下傳說皮膚的魅力，之後有可能你還會掏腰包購買皮膚。雖說購買的人數估計不會很多，但是能買一個算一個不是？結果時間設置剛好是周六日，一群小學生自然而然開始了自己的排位練英雄之旅，到排位中清一色傳說皮膚，清一色菜的摳腳，把原本的排位賽水平生生拉低了一個檔次，王者成星耀，星耀成鑽石！讓一些上分愛好者的遊戲體驗大幅度降低！

按理說，官方開啟這個活動還是希望大家應該是在匹配中來玩的，結果一大批「小學生」直接衝到了排位賽中開始練英雄了！比方說這位玩家遇到的這個后羿，面對對面有韓信，牛魔和亞瑟的情況下，依舊拿了出來。玩家還抱著后羿是大神的心態，結果對局中直接打到崩潰！后羿直接打出了0-7-1的戰績，對局時間13分55秒，后羿經濟才剛剛8000出頭！反觀對面的公孫離，經濟已經11000+了，雙方裝備根本不是一個水平！

不過這樣的情況也並非是壞事，就比方說下面這位朋友，充分的給大家解釋了一下什麼叫做射手榮耀！對面清一色五個射手，還是排位賽！除了魯班七號的皮膚是史詩級的電玩小子，其他的清一色都是限定傳說皮，充分給大家展示了什麼叫做只想體驗皮膚，去他媽的上分！為此老張也是表示實名羨慕，這局這位兄弟是躺定了呀！

但是像對面五射手的情況還是少見，更多的人經歷的依舊是己方無腦練英雄！因此許多暴躁老哥三天拒絕排位並開始了自己的吐槽：你說你10月26日就已經搞過一次周年限免活動了，結果這才時隔一個多月，張良皮膚剛出來就開始搞了又一波限免。這種操作真的是秀啊，讓一些剛買了張良皮膚的人做何感想？並且大量練英雄的玩家湧入王者峽谷排位賽，將整體的排位環境搞的烏煙瘴氣，不知道策划到底是怎麼想的，絲毫不顧及玩家的遊戲體驗嗎？

但是老張認為策劃也有策劃的難處，畢竟王者榮耀中充錢玩家占的比例還是很低的，大部分玩家都沒有怎麼體驗過傳說級皮膚，再加上玩家自製的源夢系列皮膚上架，搞一個活動慶祝一下也算是激勵廣大玩家吧。至於賣皮膚，又豈是用了兩把覺得順手就買了？就像那位老哥說的，傳說皮膚又非限定皮膚，是一直可以購買的，要買早就買了。如果以後再搞這種活動可以考慮限制模式，比方說排位賽不能使用這些皮膚就能很好的針對排位練英雄的現象。

這種活動還是不錯的，畢竟可以體驗一下傳說級皮膚。如果能更改一下模式設置那麼老張還是希望多出幾次這樣的活動給大家，讓一些0沖黨多體驗一下好皮膚的手感，指不定就能吸引人充錢購買了呢！你們覺得呢？我是老張，一名王者榮耀愛好者，期待你的關注！
