EDG德杯名單出來後，管澤元也忍不住吐槽了？網友：突然看到希望

2019-12-17 96條評論 2,763次閱讀 149人點贊

12月17日中午，EDG公布了出征德杯以及轉會期過後的人員名單。由於Iboy的離隊，加上廠長轉職教練因素的影響，這份名單也引起了大量粉絲的不滿，甚至遭到著名電競毒奶管澤元的炮轟！

EDG征戰德杯的出征名單如下

主教練為7醬明凱，戰隊監督是Heart，副教練是Corgi與Bibra，上單首發劍仙Jinoo替補Aodi與xiaoxiang，打野首發為jiejie替補為xinyi與JunJia，中單與輔助的情況並沒有改變，依然是Scount與Meiko，ADC選手則換上了之前出場過的Hope選手。

總的來看，與上個賽季EDG戰隊的強度相差無幾，但是青訓選手與LPL老對手的年紀與遊戲理解差距較大，隊伍給人一種青黃不接的感覺。由於出征陣容缺乏LPL人氣選手的坐鎮，使得大量EDG的粉絲髮出了「就這」的感嘆。

管澤元也在第一時間對於EDG出征陣容發表了自己的看法，表示「感覺明凱哥哥轉教練的第一個賽季有點難，太需要磨合了吧這個陣容！」

在電競黃大仙這口毒奶的作用下，大部分EDG的粉絲們紛紛表示自己又活過來了，甚至已經在考慮「廠長退役了，冠軍皮膚還能選寡婦么」這種想桃子吃的美好畫面。

當然大部分粉絲對於EDG在轉會期低調錶現，頗有一種恨鐵不成鋼的心態，只能在管澤元這條微博下方安慰自己「EDG起來了，好起來了，謝謝寧。」「EDG在下一盤棋呢。 既然D不出好人，那還不如就攢錢升人口。」

客觀來說，EDG的出征陣容距離頂尖強隊還是有一段距離的，對於明凱的執教壓力也很大，小編這裡也希望管澤元的毒奶可以成功吧！小夥伴們對此有什麼看法呢？歡迎在下方留言。
