地下城真的要黃了嗎？除了旭旭寶寶還準備了六百三十萬個矛盾準備在春節再沖兩件紅十七，其他一些知名DNF玩家似乎都有些底氣不足了，你賣號來我賣號，你脫坑呀我脫坑，像eiji這個以技術著稱的玩家也開始賣號了，大家都知道他的劍魂是全身紅十二的，附帶著還有紅十二的其他三個鬼劍士，以及一些其他的紅十一小號。價格是五萬人民幣。但是賣了好久，現在還是沒有賣出去，為什麼呢？是eiji的白手不值這麼多錢嗎？實話實說，確實不值這麼多！

這個紅十二的白手在eiji手裡確實強，總是能打出一些奇蹟來，百分之九十九的紅十二白手玩家做不到的事情，人家能做到，可以說是實力非常強了。然而紅十二是能買到的，那麼技術也能買到嗎？難不成買了eiji的白手，還附帶上一門「白手進階教程」，再培訓上一年半載，再造一個eiji二號？這五萬塊錢怕是不太夠吧。所以問題就是，紅十二能賣，技術賣不了啊。

然後就要說到一個名人效應了，大家首先要明白一個問題，號在名人手裡，那能體現他的價值。eiji的白手在eiji手裡，才能冠上「eiji」這四個字母，大家看了就能知道，這個白手很厲害，不是說叫這個名字的白手，給任何一個阿貓阿狗拿了，大家都會很佩服，那是不可能的。eiji換任何一個打造差不多的白手，都能做到他能做到的那些奇蹟。地下城的賬號沒有認主效果，也沒有什麼血脈認證，厲不厲害，全看賬號本人。

那麼按照賬號本身的價值來看，四個紅十二的角色，應該賣多少錢？一般來說，在保證安全的情況下，一個紅十二角色可以賣一萬，但是附帶的連體號只能賣半價，算上其他七七八八的東西，eiji的這個號估價應該在三萬左右，由於安全上有近乎絕對的保障，可以再提價百分之二十左右，eiji的這個號，要賣的話，不會超過三萬五千塊錢。

所以整體而言，eiji的這個賬號，名人效應是不能折現的，技術又不會隨著賬號的轉移而轉移，最後只有一個安全性可以稱道一下，賣五萬實在是有點多了，三萬五千塊錢以下，或許會有粉絲一類的玩家來買單。

但是有個大前提，就是現在的dnf玩家還對遊戲有點信心，要是遊戲真的過上一年半載黃了，誰還去買？那不成了接盤俠？
