魔獸世界懷舊服打本補助這件事從來沒有一個固定的說法，面對著全民打本的當下，各種各樣的問題都會湧現出來，今天我R了一個裝備，明天你多發了幾十G的補助，都有可能引發團隊的爭吵，下面984G十方（筆者）想說的事情，與黑龍TN補助有關。

由於魔獸世界懷舊服黑龍並不看重輸出，所以大部分團隊打黑龍都把它看作是磨過去的boss，給不給補助要看團長，確實有給的也有不給的，但是給一些屬於正常。如果一點不給，那麼團長就有一點「黑」的味道了，但如果開始就講好，自然也無可厚非。問題都在於黑龍目前已經被很多人征服，能夠成團就已經不容易，如果要求過多，那麼很容易就要再等個把小時。

而從沒聽說過黑龍有補助的玩家也大有人在，畢竟Mc錢多有TN補貼一點很正常，但是打黑龍的話，一般會給來救場的矮人牧師一些補貼，畢竟矮牧在魔獸世界懷舊服屬於比較稀有的那一類，打黑龍的時候如果有矮M那絕對是不一樣的遊戲體驗，如果再給其他人補貼也就分不到什麼G了。目前開黑龍團要數G團居多，進團之前就要有心裡準備。

關於以上兩種說法，其實各有各的道理，現在奶對於各個G團來說都屬於要搶的職業，如果你的團不給補助而其他團給，那麼不給的就慢慢等好了。984G十方一開始也是只給矮牧補助，但是根本叫不到奶，後來所有奶都給補助才叫夠人順利開組，這個情況供大家參考。

如果想順利開團黑龍，適當補助個10G或者20G的問題不大，主要還是看團隊賣了多少，由於黑龍不需要太多人打，2個T再加上5到6個奶就可以開團，再加上十幾個DPS，如果最終賣的少就可以不補，賣的多可以適當補點，看情況而定。

說到這裡984G十方想到一個題外話，曾經遇見過一個魔獸世界懷舊服大佬練了幾個滿級號，和朋友練級組AA隊的時候他常說，法師全程不要撿錢，法師後期來錢很容易，一天弄個幾百G很輕鬆，都讓給牧師。原因就在於牧師就是練到60也買不起千金馬，因為魔獸世界懷舊服牧師這職業不但升級慢，而且還窮，關愛一下很有必要。

最後再多說一句，MC是肯定有補貼的，雖然黑龍以前有補貼，但是現在魔獸世界懷舊服補貼的團確實不多了。
