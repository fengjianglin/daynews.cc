前言：昨天直播上承諾今天給大家帶來山脈法攻略，我特地加班加點，做了一下攻略。今天給大家帶來兩套陣容合集，都是用山脈法過渡的。雖然山脈法過渡高分段屢見不鮮，不過這種玩法確實是最容易上分的一種，本攻略屬於新手向，更新不易、高手勿噴，大家記得點點關注，點個贊哈。

山脈法不能算一個完全體的陣容，只能算的上一個很強的過渡陣容，後期上八我們可以根據來牌和決賽圈的對手，來選擇我們要走守護雷霆還是海洋守護法。
前期思路：前期拿大棒，優先做出鬼書和春哥，一般情況下用4海洋鬼書婕拉過渡，肉裝給錘石，可以走一大波連勝。記得奧恩狗熊各留一隻，岩雀是六人口是很關鍵的一環，可以補兩山脈，打劍刺。
中期思路：4海洋配合鬼書春哥婕拉，中期猛得一筆，順利的話能夠吃一大波連勝，我們需要注意的是劍刺發力的那個階段。這時候，小搜一波，提質量，搜石頭人，才能保住連勝。

後期思路：這套陣容最核心的一點就是速八，我們七人口不用停留太久，打完雞哥，一般來說都是賣血換經濟。上八後盯著橙卡，娜美、木木、寶石搜。運氣好來了劫，賣婕拉和錘石，把裝備分配給劫和娜美，就完成了守護雷霆的轉換。

沒有劫的情況，反而有兩星火男，那我們就可以走海洋守護法這條路線，不過要注意的是，現在的火男傷害遠不如之前，所以前排寶石和木木是否到二很關鍵，關係著爛分與吃雞。這兩套有血量優勢可以上九，如果對方是雷霆或者法師，那就補一手劍聖或風女，如果不是，那山脈法可以再補一個兩星火男，守護雷霆可以補一個布隆。

站位細節：守護雷霆劫的站位比較隨便，4個守護者都可以頂前排，可以壓縮對面站位，讓木木和寶石快速放大。劫盡量找軟柿子來切，只要能刷出技能，劫就是無敵的，娜美底格靠牆角與劫一側。如果是海洋守護法，火男居中，吸血鬼和辛德拉分居底排兩側靠牆，娜美可以稍微靠中間站，注意防刺客，守護就頂前排即可。

克制關係：這兩套陣容都比較怕劇毒，還有沉默刀，也都是劍刺的爸爸，隨便吊打劍刺。這要比的話，守護雷霆會比海洋守護法稍微強上一點，所以有劫的話，完全可以轉守護雷霆。
