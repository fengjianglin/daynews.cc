也許龍體系沒有那麼厲害，但是幾乎每一條龍單拿出來都很厲害

●前言

大家好，我是掌游寶作者：桃源。預覽時期大家都說這個版本不超模就是廢物，雖說也許不至於這樣，但是這個版本對環境的衝擊絕對是非常大。像外服卡組推薦裡面有一套龍牧就是26張新卡…

◆祈求突襲戰

女巫森林版本初，有人構築突襲戰失敗了，因為解場能力不夠，而且還沒有明確致勝手段。但是這一套突襲戰不一樣。這套是從奧丹姆版本的狂暴戰改變而來，整套卡組20個隨從，當回合不能動的只有8個，剩下12個被加拉克隆抽到手裡瞬間爆炸。祈求體系包括422當回合解場能力也是足夠的。

這套卡組的思路和任務獵相似，前期瘋狂解場，街道對面的出場速度比你還慢的時候，一個迦拉克隆直接勸退。迦拉克隆抽到的隨從能當回合拍下去還能動的只有廢墟之子，3費3個76突襲。3個衝鋒補充直傷，這套卡組不像節奏站那樣有受傷手段，所以並沒有選擇攜帶吼爹。

祈信狂徒是增加祈求濃度，歐洲人請換成民兵指揮官。

◆辟法巨龍

◆始生龍探險者

能把這套卡組寫出來，各位也能猜到桃源腦洞有多大……構成卡組的理由:1.夢境吐息太強了，2.翡翠龍探險者太強了，3.既然這麼多高質量龍不如外掛藍龍斬殺
