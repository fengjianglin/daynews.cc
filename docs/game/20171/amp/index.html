為遊戲而生，巴塞利斯蛇家族進化史

天天要聞
10 小時 ago

2017年，是FPS遊戲火爆的一年，越來越多的玩家投入到「吃雞」行列。為了讓更多玩家獲得更好的遊戲體驗，我們在當年推出了一款專為FPS優化的遊戲滑鼠，巴塞利斯蛇家族就此誕生。

Razer Basilisk
巴塞利斯蛇

一鳴驚人，電競玩家更優的選擇

巴塞利斯蛇使用了人體工程學設計，為玩家提供了舒適的FPS遊戲體驗，同時創新性的可拆卸式 DPI 切換側鍵和可自定義滾輪阻力的撥盤等特色，讓它一度成為「吃雞」神器。

Razer Basilisk Essential

巴塞利斯蛇標準版

人人都可FPS
在巴塞利斯蛇的爆火之後，為了讓更多FPS玩家體驗到巴塞利斯蛇的魅力，我們推出了巴塞利斯蛇標準版，它沿襲了巴塞利斯蛇的優秀血統，保留了出色的人體工程學設計以及可拆卸側鍵等特性，讓越來越多的玩家感受到了巴塞利斯蛇系列滑鼠的魅力。

隨著玩家對於滑鼠要求的提升，玩家對於旗艦級無線滑鼠的呼聲越來越高；同時電競裝備的趨勢也逐漸向無線化靠攏。我們聽到了玩家的呼聲，為玩家帶來了巴塞利斯蛇家族的無線成員——巴塞利斯蛇終極版與巴塞利斯蛇 X 極速版。

Razer Basilisk Ultimate

巴塞利斯蛇終極版

無線加持，性能巔峰

巴塞利斯蛇終極版作為2019最強無線遊戲滑鼠，採用RAZER HYPERSPEED 無線技術，相比其他無線技術，巴塞利斯蛇終極版的無線傳輸的速度提升了25%。此外，其高速傳輸、極低的點擊延遲等特性，延遲媲美有線滑鼠，讓玩家獲得超越有線滑鼠的使用體驗，玩家們也親切的稱呼它為「大巴蛇」。

巴塞利斯蛇終極版與以往版本的巴塞利斯蛇相比，採用了雷蛇獨家設計的Razer光學滑鼠微動。相比傳統的機械微動，從結構上改變傳統微動的工作模式，使其擁有更快的響應時間，並且壽命上更是高達7,000萬次。在多位頂尖的電競選手的測試下，均證明巴塞利斯蛇終極版在激烈的遊戲中能保持極佳表現。

除此之外，巴塞利斯蛇終極版燈光上同樣也由全局調整變成分區調整，14個可自定義的Razer Chroma RGB幻彩燈光區域都可進行單獨編輯；隨附的充電底座也能通過雷雲3進行燈光設置，讓玩家可在燈光中展現更多個性色彩。

玩家心水的可拆卸式 DPI 切換側鍵以及可自定義滾輪阻力的撥盤等巴塞利斯蛇家族的經典特色依然保留；側面防滑也升級為更加親膚的蜂巢式防滑側裙，能夠更好的貼合玩家手掌。

Razer Basilisk X HyperSpeed

巴塞利斯蛇 X 極速版

雙模無線，續航更進一步

與終極版一同發布的巴塞利斯蛇 X 極速版，讓更多的玩家享受到了無線巴塞利斯蛇的特色：同樣搭載RAZER HYPERSPEED 無線技術，並且還增添藍牙連接的雙模設計，玩家可在不同的使用環境下，自由切換連接模式，巴塞利斯蛇X極速版也被玩家稱為「小巴蛇」。

穩定的雙模技術也帶來了性能與續航的平衡：2.4G藍牙模塊模式下，可提供長達285小時的續航時間；藍牙模式下，滑鼠續航更是達到了驚人的450小時，完全滿足玩家長時間戰鬥的使用需求。
巴塞利斯蛇 X 極速版採用了Razer 5G光學感測器，為玩家提供高達16,000 DPI的靈敏度，給玩家帶來更好的速度與準度，使玩家在遊戲中搶得先機。此外6 個可編程按鍵以及板載 DPI 存儲等功能，同樣使玩家在遊戲中獲得更好的體驗，不論在什麼環境，都能在遊戲中盡情暢玩。

巴塞利斯蛇從誕生到進化，從有線到無線，無時無刻都在為眾多玩家提供更好的遊戲體驗。巴塞利斯蛇家族以其強大的人體工程學設計、極致的性能和出眾的外觀讓其一直在遊戲滑鼠中佔有一席之地。我們將不斷地把最新的技術融入滑鼠中，忠於「始於玩家，賦予玩家」的品牌理念，推出更多屬於電競玩家的遊戲裝備。
