玩遊戲什麼最重要，就是開心啊！只要開心什麼都好說，當然了你不能把開心建立在隊友痛苦的基礎上。今天給大家介紹一個非常恐怖的下路組合只不過這個組合需要一些操作並且兩個英雄還都是輔助，這兩個英雄就是牛頭+錘石。可能你們第一眼看上去我去這是什麼個垃圾套路！我和朋友第一次選出這個陣容的時候對方直接嘲諷，結果被我們直接錘掉線。

這個組合該如何進行操作呢？錘石作為ADC出AD裝備，而牛頭就是純粹的輔助。首先在前期對線的時間段，沒有3級最好不要和對方開戰，畢竟兩個人的技能傷害都不高。並且有一個非常需要注意的地方就是一定要把兵線控制在離自己家塔下比較近的位置才行，這一步要劃重點，怎麼個打法呢？進攻的開端由錘石發起，一記Q不論勾到的是對方的輔助還是ADC，別想直接過去！

這個時候向後拋一個燈籠放在牛頭的腳下，你去到對方英雄身邊的時候牛頭恰好來到，牛頭Q將對方震起來隨後走到對方英雄的身後直接頂進塔，如果閃現出來怎麼辦？這套陣容的恐怖之處就在這裡，將你頂到塔下，按照牛頭頂出的距離哪怕對方閃現也僅僅沒有脫離兩人攻擊的範圍，錘石可以直接一個E將他打回去，並且配合牛頭的E來打出眩暈，這一套下來對方死亡的幾率真的很大！如果到了6級別，那就更加暴力了錘石Q鉤住還是燈籠一放開，牛頭過來Q頂起來，這個時候錘石牛頭直接放大。

牛頭還是老操作直接把對方頂回去，直接撞牆的對手會被減速，加上錘石E和牛頭E是絕對走不掉的！不要小看兩個所謂輔助英雄的傷害，錘石有著積攢爆炸傷害，而牛頭的E暈住對面以後那一下的傷害非常痛。

在出裝方面，錘石最為ADC一定是多蘭劍加藥水出門，第一件最好出嵐切增加自己的移動速度，錘石也是可以走A的！鞋子選擇布甲鞋，隨後一定要出火炮，

。積攢被動用火炮砍下去的一瞬間對方會異常的酸爽！隨後就是無盡、春哥甲、要把自己的暴擊給點出來！你的第一條鞭子是你的主要傷害。牛頭在輔助裝備出完之後第一時間就出一個兵拳頭，為什麼呢？很簡單，減速，等你接過錘石的燈籠，過去將對方頂到塔下的時候，你的普攻擊是有減速效果的，特別方便擊殺對面。雖然這套陣容比較難，但是絕對值得你去嘗試，快樂就可以了！
