開放世界遊戲的腳步是否該緩一緩？

2019-12-16 820條評論 2,136次閱讀 145人點贊

說起開放世界遊戲，大家能想到什麼？龐大的地圖？充滿生機的NPC？自由的選項？眾多而精彩的故事？可互動元素多？老崔？GTA？（全靠同行襯托）

多年過去，GTA5依然是老大！

從2012年至今已經湧現了一大批開放世界大作，而3A級的開放世界作品在14年後更是呈現出了井噴的態勢！其中有育碧的野心大作《看門狗》與其後的《全境縮水》、波蘭蠢驢的《巫師3：狂獵》、夜晚大型跑酷聚會的《消逝的光芒》、B社《撿垃圾4》和《少女捲軸5》等等。一時間，圈內彷彿有種向開放世界看齊的態勢！

最近的斷點評價不太好，畫面還退步了！

在開發技術越來越高的當下，開放世界遊戲成為了時下火熱的題材，各大廠商紛紛入場，不落人後！不過近些年來貌似廠商一直都致力於開發更大的地圖，在內容上反而忽略了許多！像GTA這樣素質的遊戲在市場上依然不多見！

可惜PC平台沒有

如此密集的開發舉動無非是為了玩家的錢包著想！事實上，從市場反饋來看玩家們也確實吃這一套！不過，雖然遊戲數量上來了，但也只有少量作品質量達標，大多數還是些急功近利的副產品！我想，是時候放緩腳步填充一下空虛的開放世界了！

真三8…在內容上還是比較匱乏的！

一、關於地圖：一味的大也沒意思，不如把更多精力放在打造更加豐富細緻的世界中！

《GTA5》的地圖大嗎？一般吧，網上說什麼的都有，反正很大就對了！在當時的2013年算是個令人驚嘆的成就了！不過，在這部遊戲里當真可以什麼都可以幹嗎？答案是不能，因為你連大多數的門都進不去！當你把這些看似有形實則虛無的東西拋掉時，就會發現，即便是GTA5，在地圖的可探索上依然虛很多！

試想一下假如所有建築物都可以探索的話…….

那麼不難發現，現在的開放世界遊戲固然是開放了，可實際上供玩家探索的東西並沒有看上去的那麼豐富！地圖裡的許多東西其實充其量就是個好看的模型罷了！所以，大地圖固然好，但是也缺失了其中大部分的可探索內容。當然，想要兩開花就意味著需要更多的資源支持、更長的開發時間與更加難以捉摸的市場風險！但說到底，遊戲還是靠質量說話的！

輻射4在可探索上就非常好，當然，也犧牲了些東西！……老滾5不服

二、一款神優化的遊戲可以拉低電腦最低配置需求的下限，而渣優化的遊戲則是提高電腦最高配置需求的上限（上不封頂）！

一提到遊戲優化，我就非常想拿《絕地求生》這款遊戲來舉例子，沒辦法，畢竟作為常年在Steam在線榜前三的遊戲，其垃圾的優化堪稱教科書典範！多少網吧熬過了無數3A大作，沒想到在這個下水道翻了車！

誰信誰傻子！

遊戲優化的好壞，對玩家們的遊戲體驗起著至關重要的作用！而且一些玩家因為種種因素常年都在使用著低配置的電腦，假如遊戲的優化做得足夠出色，那這部分玩家也可能會慷慨解囊，變相的擴大了遊戲的市場，並且遊戲的口碑也會水漲船高！

我這邊AMD2400G無獨顯下R6顯存才佔了一半

生化奇兵-無限最高解析度+中特效材質，30+fps

三、總歸是要講個好故事的！

自由雖然是開放世界遊戲的最大賣點，但在這個世界中，即便給了你天大的自由，也終歸得有個目標驅使著你前進！絕對的自由沒有想的那般耐玩（MC警告），對於大多數玩家來說，恐怕更需要的依舊是這片土地上引人入勝的故事以及充滿活性的NPC了。

但絕不是這種的：「將軍，十松庄的馬桶又堵了，麻煩去通一下」

相較於更加專註於劇情故事的獨立遊戲作品來說，絕大多數開放世界遊戲在劇情方面其實談不上有多麼優秀！倒不是說這些遊戲的故事真的不行，只不過與其宏大的地圖空間相比，只不過其故事劇情在其中的佔比實在是太小了！另外，除了撐門面的主線之外，支線任務也都大多空洞乏味！

…….

支線任務不只是讓玩家打發時間的東西，它還是豐富遊戲世界的有效手段，更是可以讓本來略顯單薄的一眾NPC展現自己獨特故事的機會！反觀許多的遊戲作品其實並沒有好好的利用這點，只是機械性的添加，為了任務而添加！

巫師3在豐富NPC上可謂是下功夫了！

不管作為玩家的你我喜不喜歡，但必須得承認，開放世界遊戲已經成為目前最重要的遊戲元素之一！此前可能受限於硬體實力，開放世界遊戲的格局還不大，但如今已然沒有了機能限制（講道理，就目前而言，所有的頂級畫質下的卡頓都是渣優化的鍋）！讓我們期待一下未來開放世界遊戲的發展會是個什麼樣子的吧！

期待《賽博朋克 2077》！Ta可能會給我們帶來前所未見的開放世界遊戲！

還有5個月…………..希望別翻車！
