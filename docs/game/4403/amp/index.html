Miracle-重回中路！Nigma斬獲歐洲區Minor出線名額

2 週 ago

摘要

在今天凌晨結束的布克維Minor歐洲區預選賽上，Nigma戰隊成功再次從敗者組殺回，以一波5-0的戰績成功奪得歐洲區的唯一一個Minor出線名額。而在今日的比賽中，Nigma也做出了一項調整，這也成為了他們扭轉頹勢的關鍵，那就是Miracle-重回中路！

在今天凌晨結束的布克維Minor歐洲區預選賽上，Nigma戰隊成功再次從敗者組殺回，以一波5-0的戰績成功奪得歐洲區的唯一一個Minor出線名額。而在今日的比賽中，Nigma也做出了一項調整，這也成為了他們扭轉頹勢的關鍵，那就是Miracle-重回中路！在敗者組對陣Singularity的比賽開始，儘管Nigma還是選出了他們最近屢吃敗仗的卡爾，但這次人們卻驚訝地發現卡爾的使用者卻從w33變為了Miracle-，奇蹟卡再度出山，便以一一次超神殺戮終結比賽。這也幫助Nigma進入預選決賽，對陣曾經擊敗過他們的Vikin.gg，而這一次，Nigma沒有給對手任何機會，一個3:0將名額收入囊中。

對陣Singularity比賽數據

決賽首局比賽便是一場完勝，Nigma22分鐘便終結了比賽，Miracle-的藍貓在22分鐘時間裡便收穫了恐怖的12-1-12的數據；第二局比賽面對中單猛獁，Miracle-再度選出卡爾，並在前期打出完美節奏，然而比賽的形勢卻被A杖猛獁扳回來不少，但後期Nigma的團戰陣容還是更好發揮，憑藉一波謎團拉大取得了勝利；第三局的Miracle-操刀TA繼續走中，這局比賽Nigma打的更加自信，從對線開始就沒有給對手喘息的機會，在25分鐘以23-4的戰績碾壓了對手，Miracle-更是憑藉9-0-6的如神TA宣告了傳奇的回歸。

對於許多玩家而言，奇蹟卡就是他們心中最完美的卡爾使用者，而站在中路的Miracle-也是不少玩家的青春所在。儘管近年來漸漸轉型，成為一號位的Miracle-也越來越少出現在中路，就連招牌英雄卡爾在之前也是w33操刀，但冥冥之中似乎每一個人都有他獨特的位置，縱使版本更迭、歲月流逝，那份精巧的技藝也不會消失；同樣的，這支隊伍所具備的在逆境中俞戰俞勇的企業文化也不會消散，那份積極求變、「就算只有一把補刀斧也要砍向輝耀龍心」的勇氣也始終存在於他們的靈魂之中。希望這次的出線能夠幫助Nigma儘早適應新版本，將Miracle-與w33的位置磨合地更好，為玩家帶來更多精彩的比賽。
