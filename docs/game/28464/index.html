王者榮耀：冬冠選拔賽繼續進行，XQ5連勝，eStar也戰勝對手

2019-12-23 667條評論 4,587次閱讀 3214人點贊

王者榮耀冬冠選拔賽繼續進行中，今天的比賽可以說對幾支KPL隊伍都是非常關鍵的，尤其是RW俠，eSTAR和HERO，因為他們已經輸了一場，再輸多半就沒機會了，我們來看看這幾場比賽吧。

第1局是XQ對陣RW俠，KPL賽區的內戰，自然非常關鍵了，XQ前期一波埋伏拿到1換2並拿下暴君有一定的優勢，不過關羽被抓了一波，中期RW幾波開團打得不錯，扳回來一些，12分鐘時候RW俠進攻下路太急，XQ一波反打打出0換3，之後的決勝團，九尾關羽發威，連斬2人後團滅對手，成功拿下這一局，RW俠多半晉級無望了。

之後的一局是TS對戰CW，開局TS的百里守約就擊殺嫦娥拿到一血，然後幾次狙擊都很精準，還搶到主宰，TS的暖陽李白髮育起來後也非常無解，各種消耗輸出在團戰中打出0換4，擊破上路高地，決勝團中，不知火舞的傷害也打出來了，輕鬆擊殺對手多人後拿下比賽，三連勝。

然後是rw俠對陣EMC的比賽，EMC開局還想入侵野區，RW俠盾山開團反打，娜可露露拿到雙殺，中期EMC打得不錯，團戰中裴擒虎切入完美也拿到雙殺，雙方基本均勢，不過後面RW俠還是在幾波對拼中發揮更好，不知火舞各種輸出連連取勝，之後雙方在主宰處大戰，djie馬超的傷害都打出來了，RW1換4後強行衝上高地，豬八戒抗住傷害後擊破水晶獲勝。

XQ和Nova的比賽也是有點懸，開局Nova打得不錯，團戰中0換3並拿下暴君，不過之後XQ就拿得非常穩了，在不知火舞和孫尚香發育起來後，他們在團戰中不斷收割人頭，在經濟上慢慢反超，然後就一直抱團推塔，甚至是1塔未失的情況下破掉對手高地，最終團滅對手獲勝，已經是5連勝了！晉級很有機會。

eStar這場的對手是CW，開局貓神就支援上路壓制，按下一血，暴君團也佔盡優勢，之後下路又是打出0換3，雖然在推高地的時候被CW反打了一波，但他們調整之後再次進攻，這次是0換4輕鬆拿下，之後直接點爆水晶獲勝，這幾場大多數都是KPL賽區和其他賽區的對陣，全是KPL賽區獲勝，看來優勢還是非常大的啊，至於哪兩支戰隊出現，還是要看KPL賽區之間的內戰了，讓我們拭目以待吧，歡迎大家關注和分享，遊人將繼續給大家帶來王者榮耀冬季冠軍杯比賽的最新消息。
