《榮耀戰魂》第三年第四季事件「斬虎的策略」即日開戰

2019-12-22 59條評論 3,144次閱讀 1420人點贊

　　Ubisoft今（20）日宣布，《榮耀戰魂》第三年第四季事件「斬虎的策略」自即日起可於所有遊戲平台遊玩至1月2日。「斬虎的策略」將帶玩家深入探索孫達的旅程，以及他在武林與黑石軍團衝突中扮演的角色。玩家也將能夠遊玩一個限時遊戲模式，並享用與本事件有關的各式各樣自訂物品及獎勵，包括主題命令、戰場任務、服裝、武器、效果、表情動作和裝飾。

　在本事件期間導入的全新限時遊戲模式「帝王的逃離」，將讓玩家重現武林與黑石軍團於威遠關激戰過後，武林帝王逃離的過程。在這個四人PvE 模式中，玩家必須一路護衛帝王對抗敵方大軍，好讓帝王安全逃離。為了達成這個目標，玩家需要征服各區域以在地圖上推進。每次征服一個區域，帝王將進入該區域，並會解鎖另一個可征服的新區域。隨著玩家逐區推進，他們將會遇見愈來愈強的攻擊對手，包括有特殊調整數值的「被詛咒」敵人。玩家必須同心協力合作、使出渾身解數保護帝王，直到抵達目的地並擊敗黑石指揮官。

　　從現在起到1 月6 日，每位英雄能以45,000 枚鐵錢購買「斬虎的策略」同捆來取得全新服裝、效果、裝飾和表情動作；購買價格會因玩家已解鎖的同捆項目而變動。

　　繼三年來的持續改進後，《榮耀戰魂》最近確認將在2020年提供第四年遊戲支援、繼續開發更多內容給玩家。命名為「天理昭彰之年」的第四年將為遊戲注入額外一整年的免費內容，包括四個季度改版更新、兩名新英雄，以及全新護甲套組。
