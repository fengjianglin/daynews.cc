爐石新版本最騷的盜賊卡組！小精靈成核心，一套路登頂外服天梯？

2019-12-13 3,632條評論 4,555次閱讀 504人點贊

隨著爐石傳說巨龍降臨新版本的開啟，相信不少小夥伴們也都嘗試了不少新套路了。由於這次大量超模新卡的加入，也使得目前的上分環境十分混亂。各種職業與各大主播都在研究超COOOOL的構築。也在這個百花齊放的情況下，外服主播DeadDraw卻憑著一波加入小精靈的節奏亡語賊卡組強勢登頂了標準天梯。

卡組的構築曲線十分強調在前中期的作戰能力，滿編了背刺、悶棍、封印命運六張解牌，並攜帶了夢魘巨龍迦拉克隆，可以通過祈求關鍵詞來找尋跟班體系的卡牌，進一步提高對於場面的壓制力。只需花費較少的法力值，就能實現場面上的逆轉。

不過該卡組真正的發力點卻在中期階段，不論是小精靈加無面腐蝕者的5費雙54突襲的鋪場解場雙開花的操作。

還是小精靈跳連擊，排死金藥劑師抽取機械雛龍，再利用死金匕首觸發其亡語，四費鋪場一個2-5亡語與7-7機械巨龍的傳說級康波，都能讓敵方對場面壓力感到頭皮發麻。

而跟班體系與劫匪之王托瓦格爾也是老配合了，通過寶藏牌的強大特效來擴大中期的優勢；還能靠著手中眾多的一費跟班與0費解牌，直接合成出一個身材超大的范克里夫，一腳踹死對面。所以客觀來說，該卡組在中後期階段尚有奮力一搏，將敵方一擊斃命的戰力。

總結來看，目前節奏亡語賊卡組的形態十分完美，在各個對戰階段都有一定的發力點，出現鬼抽的概率並不大。根據跟班、連擊、祈求、亡語四大體系來取得場面上的優勢，十分適合喜歡解場搶節奏的盜賊玩家使用。

但是該卡組依然也存在著一些致命的缺點，首先是所有盜賊形態的通病，該卡組也不具備強大的回復與防禦能力，雖然攜帶了奇利亞斯稍微查漏補缺了一下，但依然容易在喪命於直傷傷害過多的卡組，像是某些戰吼薩滿就能在中後期，就能憑藉不講道理的斬殺一力降十會。

並且該卡組雖然找牌能力很強，但是卻沒有傳統盜賊強大的過牌能力，加上跟班體系隨機性的限制，在處理類似樹人快攻這類強鋪場的卡組時，很容易熬不到5費左右的正式發力期，生命值就已經陷入風燭殘年了。

不過對於目前尚未固化的天梯環境，這套節奏亡語賊對戰其他形態卡組的勝率都較高，這也是主播能夠登頂的關鍵原因，畢竟小精靈在這個版本中實在太超模了。小夥伴們對於這套陣容怎麼看呢？歡迎在下方留言。
