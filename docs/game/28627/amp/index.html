王者榮耀：狂暴重做後，能成功轉正嗎？別高興太早

天天要聞
2019-12-22

如果說哪一個召喚師技能最冷門，大家的第一印象恐怕就是干擾、弱化、狂暴這三個弱雞技能了，此次體驗服對這三個技能可謂是重點照顧，再一次史詩級地加強了他們。於是，我們經常會看到諸如類似這樣的論點。

這樣的觀點對嗎？至少我知道，之前弱化由降低30%輸出，提升為降低50%輸出的時候，狂暴由提供30%攻速，增強為提高60%攻速的時候，很多人也都說「太強了，肯定成大熱門召喚師技能」，如今呢，用的人還是寥寥無幾。加強了，就說無敵，這種事太多了。

那麼，究竟此次的召喚師技能調整，有沒有可能給對局帶來新的可能？還是說只是繡花枕頭，中看不中用呢？

弱化

弱化由原來的「降低身邊敵人傷害輸出50%，持續3秒」，調整為「降低身邊敵人傷害輸出30%，並增加20%的免傷率，持續5秒。」

改動有兩方面，一方面是減傷50%被拆分為兩部分，另一方面是持續時間增加了。持續時間增加，無疑提高了施放技能的容錯率，那麼減傷被拆成兩部分的意義在哪？

首先我們知道，老版弱化屬性其實已經非常不錯了，減傷50%，施放得當對方戰士、刺客的輸出會被大幅降低，保C位就多了一分勝算。但是，這是理想的情況下，真實的情況是，對方一先手，戰士、刺客跟上，你一個弱化，是降低了他們的輸出，然而卻被趕來的法師一套技能給秒了，減近戰50%的輸出，毫無意義。

新版弱化就不一樣了，對已經近身的戰士、刺客而言，依然會被降低等額的輸出，而20%的免傷則會減免那些來自遠程英雄的傷害，生存能力大幅提升。

因為新版弱化，對減少身邊敵人傷害輸出由50%降低為30%，保護C位的能力減弱，而提升20%的免傷率，對坦克、硬輔來說，又有點雞肋，所以這些英雄已經不再適合帶弱化了。

什麼英雄適合新版弱化？需要滿足兩個條件：

  1. 需要貼臉輸出，容易受到遠程傷害；
  2. 不吃閃現，打的是持續輸出；

滿足這兩個條件的英雄，只能在戰士和法刺裡面找，法師大多是遠程，射手太吃閃現，打野只能帶懲戒。適合帶弱化的英雄有哪些呢？

戰士邊路裡面，只有三個英雄比較合適，哪吒、曹操、程咬金。哪吒大招很容易被圍毆，曹操開大+弱化生存能力強，程咬金帶弱化，更能噁心對手。法刺裡面也只有三個比較適合，司馬懿、貂蟬、高漸離。

要特別注意的是，新版弱化成了貂蟬新的選擇！喜歡玩貂蟬的都知道，貂蟬從來不怕近戰，但比較怕遠程，尤其是射手，你負責秀，我負責A你很難受，而新弱化提供了20%的免傷率，重點是持續時間是5秒，貂蟬帶凈化外，還可以選擇帶弱化了。

狂暴

狂暴由原來的「提升60%的攻速和10%的攻擊力，持續5秒」，調整為「增加10%的輸出，25%的韌性和30%的物理/法術吸血。」

新版與老版的區別是，適用廣度得到了大幅提升，以前的狂暴，只適合普攻型的英雄，現在則適合除了坦克以外的絕大部分英雄了。老版狂暴的弱點，不是提供的屬性不好，哪怕他現在翻一倍，也沒什麼人用。比如射手並不缺這點輸出，因為活著才有輸出。而向老夫子這樣的英雄，又太吃閃現，狂暴適用性太差了。

新版狂暴，跟新版的弱化，其實非常相像。一個是通過吸血和韌性來提高生存（10%的傷害提升很有限），一個是通過減傷來提高生存，兩者一定會產生競爭關係。

在雞仔看來，狂暴不如新版弱化，狂暴雖然效果強大，但很怕被控，被控血就吸不上來，輸出也打不上來，25%的韌性+抵抗鞋，也沒想像中的那麼管用，畢竟不是單挑，你面對的，很可能是對方所有英雄的控制。相反，弱化只要用出來了，控制期間依然生效。更重要的是，狂暴十分害怕制裁和夢魘。

所以，如果要想使用狂暴，還是要滿足條件的，不是說喜歡吸血的英雄就可以了。

  1. 不太害怕被切，不吃閃現；
  2. 最好不容易受到制裁和夢魘的影響；
  3. 最好被控時，技能效果仍然生效；

滿足這三個條件的英雄，才能發揮狂暴的最佳效果。適合的英雄有：嫦娥、諸葛亮、王昭君、貂蟬、周瑜、高漸離、曹操。

雖然一些英雄可以帶狂暴，但效果很可能沒有想像中的好，所以這個技能雖然改了，也不要抱太大的希望……

干擾

新版干擾增加了一個效果，可以對我方機關使用，使用後，免疫所有傷害5秒。

老版的干擾，是一個越塔神器，順風特別無敵，但逆風真是雞肋，毫無用處，所以也就拆塔大師劉嬋會帶帶，其他英雄幾乎都不會用它。

新版干擾，更多是的戰略層次的改變。滿足以下其中一個條件，就可以考慮帶干擾。

  • 有開團能力的輔助
  • 帶線牽制型英雄，有較強的單殺能
  • 守塔能力弱，但越塔能力強

有開團能力的輔助，往往守塔能力很弱，新版干擾既可以塔下強開，也可以用來保護防禦塔。帶線牽制型英雄，可以用來塔下強殺敵人，迫使敵人回家防守，打亂節奏。

這樣看來，劉禪肯定是必帶的了，即便是逆風，也可以用來守塔，順風特別無敵，強錘塔還能回血，一個開干擾、開大轉塔的劉禪，很難被殺死。

總之，新版的技能效果肯定不是什麼「神技」「無敵」，不要聽別人胡吹。「閃現」仍然主流，不知火舞還是要帶斬殺、狂鐵還是要帶閃現、這些技能在橋，也很難改變很多英雄的。
