小鎮完成度很高！大神修改《PT》文件探索完整地圖

2019-12-21 188條評論 1,987次閱讀 531人點贊

YouTube 有一位名叫 Lance McDonald 的大神，他曾經通過技術手段對《黑暗之魂》系列和《只狼》等遊戲進行分析，發現了非常多開發商未公開的內容，最近 Lance 又再次完成了一項壯舉——他通過修改《P.T.》的遊戲內文件，讓這款遊戲未公開的地圖部分（僅出現在宣傳片中）得以出現，Lance 更是操控角色對小鎮街道進行了探索，令人驚訝的是《P.T.》在測試版時的小鎮部分就已經有不錯的完成度。

小鎮完成度很高！大神修改《P.T.》文件探索完整地圖：

Lance 在視頻中表示，他通過修改遊戲文件使得玩家在打開門的時候不會進入到輪迴走廊，而是可以看到房子外面的世界。不過在一開始 Lance 也遇到了一些困難，因為當他邁步走到小鎮的街道時就無法再向前走上一步，控制的角色會立即掉出地圖並被 Mary 殺掉。



但是這並沒有難倒 Lance，通過技術手段他讓角色得以在離開地面一段距離的方式下前進，這樣就避免了觸發遊戲中設置的保護機制，也讓玩家們得以能看到《P.T.》測試版中的小鎮究竟有多大。



在小島秀夫離開科樂美後，《P.T.》的開發工作也完全被擱置了下來，希望有朝一日能看到這款作品以它完整的面貌出現在玩家面前。
