雷吉：雲遊戲具有重大意義 未來十年是屬於雲遊戲的

天天要聞
2019-12-19

　　今年早些時候，遊戲業界的傳奇人物，前美國任天堂總裁雷吉（Reggie Fils-Aime）退休了，他曾經是遊戲業的風雲人物，而他也出席了今年的TGA遊戲頒獎典禮，並頒發了一個重要獎項。

　　雷吉最近接受了The Hollywood Reporter的採訪，採訪中，雷吉被問到了他對於串流雲遊戲這一「改變遊戲業」的最新技術的看法，讓我們一起來看看雷吉是怎麼說的吧！

　　

　　前美國任天堂總裁雷吉（Reggie Fils-Aime）

　　雷吉說：「隨著遊戲技術發展到了雲遊戲後，下載的速度會提升，這意味著你可以在任何時間、任何設備上玩任何遊戲了。這件事情將會在未來十年中發生，它對於玩家將會具有重大的意義。」

　　

　　我們都知道，谷歌的雲遊戲平台Stadia現已上線，但看起來反響並不是很熱烈。目前來說，雲遊戲還存在很多的挑戰和瓶頸，我們希望未來各廠商的雲遊戲能夠不斷完善自己，為玩家帶來更好的遊戲體驗。
