DNF：12.24新活動簡析，徽章轉換勿上頭，增幅11需謹慎

2019-12-24 1,263條評論 4,923次閱讀 360人點贊

本文由Sky灬素顏遊戲視頻原創，請勿抄襲或轉載，謝謝。

國服於12.24進行了2019年的最後一次版本更新，主要是上架了3種小禮包，新增了幾個小活動。雖然這次新內容的規模看起來很小，但其中包含了如黃力燦爛徽章、本職業白金徽章、+11增幅券等價值較高的打造物品，那麼這些物品需要通過何種方式來獲得，我們又該怎樣去合理地運用此次新活動，本文就來為大家簡單分析一下。

[五福俱全迎新年]

活動形式：商城禮包。[凱莉/克倫特/達芙妮/賽麗亞]福袋價值2888點券，[波塞姆]福袋價值888點券。

活動講解：每種福袋開啟後，都可從對應6種道具中隨機獲得一種，其中有4種道具是[帳號綁定]，有2種道具[可交易1次]。這次福袋的保底獎勵其實還不錯，如果你的角色存在對應道具需求的話，可以考慮入手。

5種福袋入手3種時，會獲得一個[瑞鼠獻福禮盒]，開啟後可獲得1~12期天空套自選禮盒（7天），並可在5種福袋的保底獎勵中（例如20個強化器）隨機獲得一種。

5種福袋入手5種時，會獲得一個[瑞鼠呈祥禮盒]，開啟後可再次隨機獲得一種福袋。

完成建議：首先根據自己角色的需求入手，需要強化裝備就入強化福袋，缺徽章就入徽章福袋，小號在刷史詩之路就入花瓣福袋……當然，如果自己角色對所有道具均無需求，也可以買來碰碰運氣，任何一個福袋開出可交易類型道具，都是比較值得的。

另外單獨說一下， [克倫特的增幅福袋]包含「可交易」的+11增幅券，雖然每種福袋一個帳號只能買一次，但隨著開啟量的增多，可交易+11增幅券的產出量肯定是會越來越多的，如果有玩家想趁這次活動置辦增幅+11，肯定是比沒有活動的時候輕鬆一些的。那麼這裡要提醒大家一下：

韓服100級版本已經開啟，並且國服也基本到了版本末期，我們現在做的一些事情，還是盡量把目光放長遠一點。在100級版本中，95級裝備可進行升級或繼承（轉移打造），但是增幅+11及以上的裝備在升級時會消耗額外的材料，並且額外消耗的材料數量，與增幅+11比增幅+10多出的提升程度嚴重不成正比。外加100級裝備的基礎力智都很高，增幅的性價比會因此下降一個層次。所以不建議大家在這次活動中置辦增幅+11，若有置辦的想法，建議等到100級版本的增幅大狂歡活動。

[絢麗多彩雪人禮包]

活動形式：商城禮包。分為666點券和2288點券兩種類型，「666」只能用點券，「2288」可以使用代幣券。

活動講解：666點券禮包只包含帽子和上衣，高級時裝屬性，不能合成或分解。

2288點券禮包除了帽子和上衣外，還包含一個[光環禮盒]，一個[絢麗多彩雪人禮盒]。[絢麗多彩雪人禮盒]開啟後，可從「1~3個雪晶材料、黑色雪人裝扮、紅色雪人裝扮、金色雪人裝扮」中隨機獲得一種。湊齊3個雪晶材料之後，可在NPC賽麗亞處兌換一套雪人裝扮。

完成建議：純粹的外觀向道具，基本沒什麼實際用途可言。喜歡外觀可以考慮入手，對外觀無感的話可以直接忽略。

[歡喜迎新徽章禮包]

活動形式：商城禮包。「隨機2個燦爛徽章」和「隨機1個白金徽章」，價值均為666點券，兩者合計1332點券。

活動講解：1332點券換2個隨機燦爛徽章和1個隨機白金徽章，就道具品級與數量而言，應該還是比較值的。但想用這個禮盒直接開出需要的徽章，肯定是比較困難的。這個禮盒的意義，更多還是為「徽章轉換活動」提供一個徽章胚子。

完成建議：直接開出所需徽章的幾率較小，如果不準備玩「徽章轉換」活動，或者不缺徽章胚子的話，可以考慮放棄。反之，如果準備玩「徽章轉換」活動，已有徽章胚子的數量又不夠的話，就比較值得入手。

[歡喜跨年倒計時]

活動形式：登錄領取禮盒

活動講解：活動期間登錄即可領取一個[歡喜跨年禮盒]，開啟後可獲得5個[賽麗亞的幸運]和88個[新年爆竹]。[新年爆竹]的用法與往年相同，在城鎮中使用，有幾率獲得道具獎勵。

完成建議：白給的，點擊即可領取，難道我不要？

[萌豬套娃歡喜開]

活動形式：累積在線時長

活動講解：萌豬套娃共5層，第1層可直接開啟，後續每累積30分鐘在線時長後可開啟一層，開啟5層共需累積120分鐘在線時長。每層開啟後可獲得消耗品、超星空裂縫通行證等小型獎勵；根據第五層開啟的次數（即完成120分鐘在線時長累積的天數），獲得對應大型獎勵：

累積5次（5天）：5個裝備品級調整箱；

累積10次（10天）：1個純凈的黃金增幅書；

累積15次（15天）：從「+8~+11增幅券」中隨機獲得一張。

完成建議：就活動獎勵的價值來講，個人覺得在線時間顯得稍微長了一些，現版本的玩法本身比較單一，除打團日外，其餘時間想保持120分鐘在線，還真的有點困難。

就獎勵的增幅道具來講，同期其他活動（史詩之路、南瓜球）也都有贈送，並且數量較多，應該可以滿足大部分玩家的紅字需求。所以對於這個萌豬套娃活動，個人覺得大家沒必要刻意去完成。當然，如果史詩之路和南瓜球活動的增幅道具還不能滿足你對紅字的需求，那確實就得委屈一下自己了。

[冒險家玩轉徽章]

活動形式：徽章轉換（洗徽章、扭轉屬性）

活動講解：玩家可在西海岸NPC達芙妮處參與活動；可進行轉換的徽章包含三類：白金（全職業隨機）、白金（本職業轉職大類隨機）、燦爛（隨機）。每種類型在達成對應的轉換次數後，下次轉換必定會出現一種特定徽章。

白金（全職業隨機）：轉換次數達到30次後（共1500W），下次轉換必定隨機出現本職業轉職大類的白金徽章（例如職業為阿修羅，將隨機出現劍魂/鬼泣/紅眼/阿修羅/劍影的技能白金徽章）。

白金（轉職大類隨機）：轉換次數達到20次後（共3000W），下次轉換必定隨機出現本轉職的白金徽章（例如職業為阿修羅，將隨機出現阿修羅的技能白金徽章）。

在進行「轉職大類隨機」轉換時，第一次放入的也必須是角色對應「轉職大類」的徽章，例如阿修羅可以把劍魂的白金徽章放進去，但不能把戰法的白金徽章放進去。

燦爛（隨機）：轉換次數達到30次後（共900W），下次轉換必定隨機出現「黃/綠燦爛徽章」，「黃」對應「15力量/15體力」，可鑲嵌頭肩和項鏈；「綠」對應「15智力/15精神」，可鑲嵌上衣和下裝。

完成建議：優先進行白金（轉職大類隨機）和燦爛（隨機）轉換；白金（全職業隨機）建議直接放棄，除非是缺少本職業轉職大類的白金徽章胚子。

白金（轉職大類隨機）相當於用3000W獲得一個本職業的白金徽章，但即便是本職業徽章，拿到畢業被動徽章或者Buff換裝徽章的幾率也比較小，所以這個方法需要一定的運氣和成本，建議玩之前慎重考慮。

燦爛（隨機）相當於用900W獲得一個燦爛黃/綠徽章，「黃」力量+15、「黃」體力+15、「綠」精神+15、「綠」智力+15，徽章共有四種。理論上來講，獲得所需徽章的幾率為四分之一，對運氣和成本的要求比白金徽章轉換低一些。

物理職業和奶爸對此類徽章的需求較大，從物理職業的角度來講，「15黃力」徽章是不可多得的傷害細節打造物品，但國服之前已多次上架或贈送過「10黃力」徽章，如果你的角色在對應部位已經鑲嵌了「10黃力」徽章，在資源不充足的情況下，個人建議不必刻意追求此次的「15黃力」徽章。

另外對於部分比較依賴攻速的職業（例如劍魂、劍影、漫遊、物理彈藥等），如果你的頭肩和項鏈已經鑲嵌了燦爛攻速徽章，從個人的角度來講，其實不建議更換。一方面是少了四顆燦爛攻速徽章，對手感的影響會比較明顯；另一方面，100級史詩裝備全是高力智，在那種層次的基礎力智加持下，「15黃力」帶來的提升，無論從面板還是傷害來講，都非常的微不足道。

當然，這些都只是個人建議，最後還是建議大家結合自己可投入的資源情況，以及自己對細節提升的理解認識，合理運用此次徽章轉換活動。

[神秘人員]

​活動形式：春節活動預熱

活動講解：每天可在賽麗亞房間NPC神秘人員處領取一個[神秘禮盒]，[神秘禮盒]開啟後可獲得2個星空深淵靈石和1個封印的攻略本。攻略本在1.14後才可開啟，1.14正好是春節版本實裝的時間，開出的道具，多半就是與春節版本相關的活動材料了。

完成建議：建議每天領取一下，相當於提前儲備春節活動材料。

[總結]

在此次更新的活動中，包含了較多的高價值打造物品；但在獲取方面，需要一定的運氣成分和金錢成本。最後希望大家可以根據本文的講解，再結合自身實際情況，對這些活動進行合理運用和靈活取捨，在儘力完善打造細節的同時，也不要做無意義的浪費。

12.24的新活動，就為大家分析到這裡，感謝大家的閱讀。
