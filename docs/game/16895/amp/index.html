死亡之翼強勢加盟風暴英雄，大菠蘿竟成最悲催英雄

天天要聞
2019-12-15

眾所周知，在風暴英雄之中存在著這樣一位英雄。他擁有著給予隊友安全感的龐大體型；在戰鬥中，他常常沖入敵方的陣型，並以強力的控制能力壓制敵方隊伍，讓隊友得以安全的輸出；必要的時候還可以吐出傷害爆炸的吐息來給予敵方隊伍強力的打擊。這個英雄便是——滅世者死亡之翼。

滅世者死亡之翼，江湖人稱大表哥。在近日的一次版本更新中，這位具有極高人氣的英雄。以其霸氣側漏的身軀，強力無比的技能組合，以及獨特的永久無法被阻擋這一超強特質，強勢入駐風暴英雄。為了迎接這個極高人氣的英雄加入風暴英雄的大家庭，官方也是毫不吝嗇，在活動中發放了大量的福利。

首先，曾經購買過暴雪嘉年華門票的小夥伴們，將會直接獲得包括死亡之翼及其附帶的三個皮膚在內的大禮包一份。要知道遊戲內大表哥的售價高達兩萬金幣，如此力度可謂誠意滿滿，然而更大的福利還在後面。自死亡之翼降臨之日起，官網中還會發布三個任務，完成任務的玩家將會獲得大量稀有補給包的獎勵。其中與大表哥進行組隊遊戲的任務，更是將大表哥的受歡迎程度推向了高潮。

然而，幾家歡喜幾家愁。大表哥一經登場便深受玩家喜愛，而昔日的強力坦克角色「大菠蘿」則顯得有些落寞。原本龐大的體型在表哥面前顯得有些「瘦小」；一身強有力的控制技能被表哥的無法被阻擋完美化解；就連強力大招也被表哥搶走，變成了表哥的小技能。除去無法改變朝向，其傷害效果可一點不比大菠蘿來的低。

面對大表哥迪亞波羅唯唯諾諾，而一旦兩位強力的鬥士角色出現在同一隊伍時候，咱們的大菠蘿則可以解放狂暴本性，重拳出擊撕裂地方陣型。當兩位鬥士同時存在的時候，大菠蘿可以作為團隊的主T先手切入地方陣型。配合毀滅巨龍形態的大表哥從天而降，打亂敵方陣型的同時配合己方C位擊殺敵方英雄；亦或者配合滅世者形態的死亡之翼，打出控制鏈並給予死亡之翼輸出空間。

滅世者死亡之翼作為新晉強力鬥士英雄，對傳統的鬥士英雄格局造成了強力的衝擊。再未來的戰鬥中，小夥伴們又會在什麼陣容之中選出死亡之翼這一強力英雄呢？
