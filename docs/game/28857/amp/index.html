雲頂之弈6影系魂陣容玩法推薦 6影系魂陣容玩法搭配攻略

天天要聞
2019-12-21

雲頂之弈9.24加入的系魂羈絆和該羈絆的英雄盧錫安、賽娜，徹底改變了T1陣容的排位，系魂羈絆搭配出來的各種陣容都有著不錯的實力，其中6影系魂更是吃雞上分的常客，那麼6影系魂怎麼玩呢？和小編一起來看看吧。

雲頂之弈6影系魂陣容玩法推薦

盧錫安、賽娜、千珏、劍聖

羈絆效果：

6影：影（部署多名影英雄能讓他們在戰鬥開始時及獲得擊殺時5秒內造成強化傷害。部署更多影英雄能在任意影英雄造成擊殺時增加全體影的傷害。）在任意影英雄造成擊殺時，在5秒內提升全體影英雄140%的傷害。

2系魂：一輪中首先陣亡的系魂單位會前往靈魂領域，變為不可被選取的單位，且在系魂另一單位存活期間繼續戰鬥。

詳細玩法以及陣容強勢點：

這套陣容靠的是影羈絆只要斬殺一人之後，就能夠使得影全體棋子都進入嗜血狀態，而賽娜自帶輔助技能，更是能夠讓影羈絆的效果更上一層樓。不過影在上個版本就幾乎沒人玩，主要原因便是因為影太難運營了，如果沒有強勢裝備和棋子，可能會陷入剛打死一枚棋子，身邊的其餘棋子都死光了的境地。這種情況，哪怕是賽娜加入也很難逆轉。

不過這並不是沒有辦法解決，玩家選擇影作為主羈絆的時候，可以選擇其他的打工仔羈絆，等到場上需要的棋子來到，就果斷把他們賣掉即可。這裡前期推薦大家使用3影3狂戰，前期低費的狂戰棋子就挺多，賽恩自身也是狂戰，正巧可以用來過渡。只需場上注意留核心棋子，以及所需的影羈絆棋子即可。

後期盧錫安一到，馬上就上場湊出2系魂羈絆，然後將盧錫安放在前排，方便對方馬上秒掉。只要對方秒掉盧錫安，就可以觸發系魂羈絆，接下來只需要欣賞盧錫安清場即可。而盧錫安在吃到影羈絆之後，會變得愈發的恐怖！

核心裝備：

巨人殺手、巨型九頭蛇、大天使、鬼書、鬼索狂暴之刃





裝備適配者：

巨人殺手、巨型九頭蛇、大天使三件套都是盧錫安核心裝備；鬼書給到賽娜；鬼索狂暴之刃給到劍聖。

最終成型陣容：

盧錫安、賽娜、千珏、劍聖、馬爾扎哈、小法、賽恩、（餘下選擇兩個輔助形掛件）

注意事項：賽娜一定要注意保護，否則盧錫安就不能保持靈魂狀態，這對於這套陣容等於廢了一半！其次盧錫安一定要放給對方先秒，否則沒系魂羈絆真不是很猛。
