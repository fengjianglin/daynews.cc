《傳奇》納入規範管理 傳奇創盟打造國內遊戲維權認證平台

自2005年，網路遊戲行業進入快速增長期，整個網路遊戲行業發展空前繁榮。近幾年，網路遊戲行業仍保持整體向好的發展態勢。

2018年，中國遊戲市場實際銷售收入達2144.4億元，遊戲用戶規模超過10億。

但是在網路遊戲行業繁榮發展的同時，各大正版官方遊戲公司一直被非法私服盜版遊戲的侵權行為所侵害。

近日，江西傳奇創盟科技發展有限公司(以下簡稱「傳奇創盟」)攜手國民傳奇產業園落地合法化認證，《傳奇》遊戲共同著作權人及獨佔權權利人向江西省宜春市遊戲產業知識產權保護協調領導小組(以下簡稱「領導小組」)遞交了《關於傳奇遊戲知識產權維權情況的報告》。

該報告中稱，正值宜春市國民傳奇產業園即將開園之際，傳奇創盟特向領導小組報告《傳奇》遊戲知識產權維權情況，懇請依法對《傳奇》遊戲合法權益進行保護。

2001年《傳奇》被引進上線後成為全球同時在線玩家數最高的網遊。然而2002年，由於著作權人之一的韓國娛美德公司過失導致《傳奇》源代碼泄露，出現大量非法私服。

2003年，新聞出版總署等五部委聯合發出《關於開展對「私服」、「外掛」專項治理的通知》，此後《傳奇》遊戲在多方大力配合下開展全國性、全面性的打擊私服活動，經歷了長達17年的艱苦卓絕反盜維權歷程。

據不完全統計，正版「傳奇」IP(包含官服、頁游、手游、H5等)所產生的合法銷售收入超過600億元。據估計，私服銷售收入有可能遠超正版，在促進市場宣傳的同時，也嚴重擾亂了市場環境、侵害了傳奇著作權人的合法權益。無疑凈化市場、保證著作權人合法利益為目的，以促進產業化綠色和健康發展為主題思想的認證平台實施意義重大！

因此，對於《傳奇》遊戲以及整個網路遊戲行業來說，保護網路遊戲知識產權、打擊私服盜版成為目前亟需解決的問題。

值得注意的是，隨著遊戲盜版市場的不斷發展，侵權盜版手法也在不斷演變，在傳統私服以及盜用官方遊戲人物角色、地圖、道具、界面等美術素材的新型私服之外，還出現了外掛等其他類型的侵權手法。

在此背景下，傳奇創盟及國民傳奇產業園所有入駐企業，懇請領導小組對國民傳奇產業園所有企業予以關注，並從多維度維權環境建設、全產業鏈侵權犯罪打擊以及國內外知識產權理論支持等方面依法加大對國民傳奇產業園知識產權的保護力度。

希望通過領導小組積極有效的知識產權立法、司法、執法行動，將國民傳奇產業園打造國內《傳奇》遊戲產業在知識產權保護方面的示範區，吸引更多遊戲企業入駐，形成地方支柱產業並形成蝴蝶效應，成為宜春市乃至整個江西省新的經濟增長點。

11月28日，宜春市宜陽新區管委會(以下簡稱「管委會」)對外發布關於加快「國民傳奇產業生態鏈」建設的公告，標誌著宜春市「國民傳奇產業生態鏈」建設正式進入加速期。

公告顯示，管委會收到盛趣遊戲承諾函，盛趣遊戲將為「國民傳奇產業生態鏈」建設提供以下支持：

首先，盛趣遊戲及關聯公司自承諾函出具之日起30日內將《傳世群英傳》遊戲著作權及版號給與江西省國民傳奇網路技術有限公司進行合作。

其次，盛趣遊戲及關聯公司授權傳奇創盟作為有關《熱血傳奇》和《傳奇世界》遊戲私服打擊及合法化排他性被授權方，授權期限至2024年12月31日止。

再次，盛趣遊戲及關聯公司全力配合提供有關《國民傳奇》遊戲的孵化、開發、發行及運營等支持。

另外，盛趣遊戲及關聯公司將全力配合建立「國民傳奇」遊戲著作權認證服務中心，並為該中心運營及發展提供有關《傳奇世界》和《熱血傳奇》等遊戲的認證服務支持。同意落地於宜陽新區的全資子公司擁有《熱血傳奇》和《傳奇世界》遊戲的對外授權權利及進行對外授權工作。

最後，盛趣遊戲及關聯公司同意將自己享有著作權的其他具有影響力的網路遊戲(包括但不限於《龍之谷》、《千年》等網路遊戲)的私服打擊及合法化權利排他性地授權給盛趣遊戲全資子公司或傳奇創盟。

目前，傳奇創盟在《熱血傳奇》及《傳奇世界》著作權人盛趣遊戲的授權下，擁有中國大陸地區傳奇IP以及授權與維權的獨有排他性權利，以及盛趣遊戲享有著作權的其他具有影響力的網路遊戲的私服打擊及合法化權利排他性權利，將全面排查市場上的相關侵權行為，採取多種舉措打擊侵權盜版，維護著作權人合法權益，並開闢IP授權服務通道，引導該類企業走向正版授權、合規運營的合法化道路。

另一方面，盛趣遊戲將不遺餘力的配合孵化開發、發行、運營和全資源支持「國民傳奇」原創IP的打造。

「國民傳奇」IP具有跨時代的意義，其是世紀華通攜手椰子遊戲孵化的第一個IP，也是國民傳奇產業聯盟第一個PC和手游產品，不僅僅是一個新傳奇IP著作權的誕生，更標誌著傳奇類三端互通產品時代的到來。

今年10月22日，旨在打造傳奇IP的產品研發、IP認證與版權監測、市場推廣和銷售、線下場景化體驗、衍生品拓展、文旅消費等全生態立體產業鏈條立體生態圈的「國民傳奇產業園」在江西省宜春市宜陽新區舉行了簽約儀式。

在簽約儀式上，宜春市委副書記、市長致辭稱，國民傳奇產業園孵化培育傳奇世界新生代的產業，與宜春市大數據產業發展的方向高度契合，將有利的推動宜春新經濟產業快速健康的發展，宜春市委、市政府將竭誠為項目落地建設提供高效優質服務。

領導小組在宜春市委、市政府的大力支持下，提供依法對《傳奇》遊戲的合法權益保護，打擊私服盜版等侵權違法行為，《傳奇》私服將納入規範管理，國民傳奇產業園將打造國內遊戲認證第一平台。

國民傳奇表示這套認證平台(系統)不只是適用於傳奇遊戲，其他類遊戲同樣適用。國民傳奇願將這套系統與其他遊戲IP共享。

一直以來，私服市場的繁榮作為硬幣的兩面，深刻影響著遊戲產業的進一步發展，私服從業者從傳奇玩家到創意者GM作坊，再到如今傳奇私服GM聯盟平台化、合法化認證系統的落地，傳奇IP「大一統」的市場格局逐漸成型。

目前市場佔比90%以上產值的108家運營傳奇遊戲的公司已開始註冊、落地國民傳奇產業園。圍繞傳奇維權、合法化授權及版權監測與認證工作的展開，知識產權維護以星星之火可以燎原之勢，拓展到《龍之谷》、《千年》等盛趣核心IP的產業鏈打造，拉開私服產業平台化、合法化，打造全生態IP產業鏈的新時代！

傳奇創盟將於2020年攜手「傳奇至尊」公司全面打造傳奇產業GM聯盟合法化、平台化！每個GM人的心中都有一個故事，一段記憶，那就是「我們的傳奇」電影，在玩家心中影游聯動，是傳奇20年以來的第一次親密接觸！

盛趣遊戲(原盛大遊戲)系全球領先的互聯網遊戲開發商、運營商和發行商。盛大遊戲成立於1999年11月，2019年正式啟用盛趣遊戲作為全新品牌標識，重組完成併入國內上市公司世紀華通體系。隨著重組及承諾業績的逐步完成，世紀華通已成為A股市場中凈利潤最高的互聯網公司。
