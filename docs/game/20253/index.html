雲頂之弈：法海永恆——前國服第一蝴蝶HDL新海洋守護法陣容推薦

2019-12-17 845條評論 3,360次閱讀 1729人點贊

大家好，我是S2賽季國服第一個大師，曾僥倖做過國服第一的蝴蝶HDL。在新版本更新後，很多陣容的打法有所改變，為了方便大家更好的理解，我以之前遊戲的熱門陣容海洋法師為例，給大家提供更新的思路。

前期和海洋法基本類似，三森林三法師（雙海洋)過度。到第4階段開始後加入泰坦-錘石等能抽到的守護羈絆英雄，湊齊四守護-三法師-四海洋的高質量陣容。8人口的時候如果有娜美或者阿木木可以還下吸血鬼或者辛德拉，之後根據其他人陣容調整秘術羈絆或者加強守護羈絆。最後如果有地獄火轉職書也可以給娜美或者泰坦等英雄。

整體來說這套陣容的發力期在後面。所以在遊戲過程中也有一些關鍵點需要介紹

3-2的時候即使質量不夠，也盡量小D。儘可能保持經濟完善。火男沒出現之前打工裝給辛德拉。

第4階段開始的時候，只要守護羈絆的質量足夠，憑藉岩雀+吸血鬼+辛德拉也還是有一戰之力的。

過度第4階段的時候，我們需要儘快收集2星火男替換岩雀，但是岩雀不要出售，和石頭人湊成的山脈羈絆也可以保護後排。

這套陣容的核心是火男的高傷害及守護羈絆的坦度和控制。岩雀和娜美也是為了更好的給火男提供輸出空間和時間，所以其他英雄盡量不要追3星。

在新版本新英雄的發布後，我會儘快發布更多的新陣容，喜歡的朋友可以關注我。下次帶來沙漠狂戰刺客的打法攻略。
