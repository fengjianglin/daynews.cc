左槍右劍，兩種武器自由切換，手游《神諭幻想》12月24日安卓公測

2019-12-20 812條評論 3,044次閱讀 1716人點贊

邪惡的鱗化病悄然擴散，疾病催化了大陸國家的全面戰爭。戰火之中，唯有格羅維亞漁翁得利，鑄造了一統天下的帝國。然而大陸並未陷入平靜，四地皆有名為黑鴉的反帝國組一般的遊戲，都會清晰的劃分職業特點。戰士大多血厚能扛，法師總是皮薄攻高，近戰和遠程徑渭分明。手游《神諭幻想》卻有些不一樣的想法，遊戲的主角是位背著長劍和槍的冒險家，左槍右劍攻擊方式切換自如。遊戲將於12月24日公測。

織。化身冒險家的你，正是黑鴉中的一員。

在《神諭幻想》的角色創建界面，我們看不到種族和職業的選項，只能選擇角色的性別。男性角色，從頭到尾，一身黑衣嚴嚴實實；女性角色卻是一位挑著一雙大白腿的小姐姐，只是衣服有些單薄，僅僅遮住了一些重要部位。

如果結合網上的理論，看起來更像一位強者。當然，男女角色只是形象有些差別，遊戲劇情沒什麼不同。大家進入遊戲之後，便會看到成排的西式中古世紀建築。可能是遊戲占內存並不大的緣故，《神諭幻想》的畫質並不精細。

不過，根據開發商宣傳及測試期間玩家爆料，《神諭幻想》有一個極大的亮點「三段跳」，每一次都能跳得更高，第3次會張開宛如天使一樣的翅膀飛躍而起。三段跳和遊戲中的複雜地形構成了有趣的互動機制，玩家幾乎到達建築上的任何一個地點，非常有趣的操作。

《神諭幻想》還開放了坐騎系統，在大陸的冒險中，玩家有機會收集各式各樣的奇幻坐騎。黑色機身藍色條紋的摩托車、頭有三角渾身電芒的雷獸、渾身紅紫色條紋的飛龍……，西方奇幻元素十足，極為炫酷。坐騎為主人形象加分的同時，還會加強綜合屬性，是玩家必備。

坐騎固然好看炫酷，可打架終歸還得靠玩家自己。在遊戲內，角色的技能就像手中的武器一樣靈活，玩家能自由組合戰鬥技能。後期的話，覺醒角色第二人格之後，還會激活終極技能。技能的渲染效果也和炫酷的坐騎設計一樣，非常拉風，但打的傷害高不高，可是要看你的裝備了喲。

《神諭幻想》，一款炫酷風的西式奇幻手游，12月24日安卓正式公測。
