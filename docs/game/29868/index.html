《獵魔人》播出，帶動《巫師3》玩家迴流！網友：那是玩遊戲洗眼

2019-12-23 469條評論 2,345次閱讀 231人點贊

上周五由網飛出品的《獵魔人》在大家的期盼中上線了第一季全集，在此之前無論是《巫師》小說的粉絲還是遊戲粉絲都懷著激動心情期待著劇版的到來，而在《獵魔人》上映之後Steam《巫師3》的同時在線玩家也有了非常明顯的迴流，達到了史上第二高的4.8w人。

這種遊戲帶動影視劇、影視劇反哺遊戲的操作算不算「套娃」對長不知道，但與其說這是《獵魔人》的熱度導致了玩家的迴流，不如說是玩家需要一個地方宣洩情緒。

因為劇版的選角根本比不上遊戲中人物風姿之萬一！

原本最受大家擔心的是由「大超」亨利-卡維爾飾演的傑洛特，未曾想全劇還原度最高的就是他這個主角了！外型契合且不說，本身就是遊戲迷的「大超」連遊戲中傑洛特低沉的聲線都還原了出來，對戰時用劍的動作也無可挑剔。

而剩下的角色就受到了來自於玩家們的猛烈吐槽，原因則只有一個字——「丑」

希里眉毛都禿了

特麗絲變成了黑髮大媽

葉奈法變成了印度裔

不僅如此，最雷人的是暗夜精靈居然真是拿黑人來演！所以全員「黑化」就是網飛想要達到的目的嗎？追求多元化就得加入無限多的黑人？在這不得不引用咱國內知名的老藝術家六老師的話：「改編不是亂編，戲說不是胡說……」這麼搞導演是要向全世界玩家謝罪的！

看到這裡大家想必也都明白了玩家迴流的原因之所在——不是劇集太好看大家要重溫遊戲找感覺，而是人物太辣眼要回去洗眼。

所以是遊戲不好玩了嗎,要去學別人追劇？
