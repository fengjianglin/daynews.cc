「777推薦」卡牌手游排行榜2019前十名，看看你是不是都玩過

廢話不多說，直接開整~

1、少年三國志2

《少年三國志2》由原班人馬打造！兵符合擊突破陣容局限！頂級畫師造就極致國漫畫風！神將傳帶你親歷三國戰場，變身張遼威震逍遙津，化作趙雲勇闖長坂坡！

2、閃爍之光

破次元放置手游《閃爍之光》開啟全新召喚！赫拉新裝甜美上線，百位神明小姐姐陪你浪漫冒險！結契精靈萌寵，解封五系元素！異界神域等你探索！

3、長安幻世繪

《長安幻世繪》是一款自由探索卡牌手游。自由探索，揭開長安古都沉寂千年的神秘面紗！自由漫步此間幻世，巧解機關，探尋秘密。

4、航海王:燃燒意志

航路不止，燃夢不熄，正版3D航海王手游《航海王：燃燒意志》一周年慶典盛大開啟！遊戲中草帽一夥、白鬍子、比斯塔、馬爾高、喬茲等實力夥伴悉數集結，全3D場景主機級表現，原版豪華聲優傾情獻聲，原汁原味體驗動畫經典劇情，實時真人組隊同屏回合對戰，完美立體呈現原作動畫。

5、霸王雄心

百位武將叱吒亂世，群雄並爭上演魔性亂斗。休閑豎屏掛機，沙場點兵半小時，每天上線收城池。主公之席虛位以待，快來打造屬於你的掌中三國！

6、小小三國志

玩家可通過收集、合成、羈絆組合，在50餘個特性鮮明的三國名將中搭配出千萬種獨具特色的連攜陣容，無雙神將親手培養，三消升級制霸全場，與全網玩家進行6V6對局一決高下！快來開啟屬於你的「自走三國新世代」吧！

7、一拳超人:最強之男

《一拳超人》正版授權手游——《一拳超人：最強之男》忠實還原動畫劇情、燃爆戰鬥、人氣角色、原版聲優，親身體驗熱血的動畫劇情！
8、三國創世紀

《三國創世紀》鬼畜來襲，收集卡牌不費勁，回合對戰有新意，精緻呆萌畫風的武將，專屬特定的戰鬥動作，傲嬌無節操的對話獨白，爆笑劇情拒絕雷同，演繹Q萌新境界。

9、三國殺名將傳

《三國殺名將傳》是一款6V6橫版動作卡牌，酷炫合擊大招，爽快戰鬥體驗，軍團BOSS廝殺，全服實時嗨聊，多樣武將養成，豐富陣容搭配，戰力蹭蹭往上漲！帶給你前所未有的卡牌樂趣！ 天下名將，盡在我手！

10、放置奇兵

《放置奇兵》是一款備受歡迎高質量放置RPG手游，派出你的英雄小隊，出 發，從碧綠蔥蔥的薩拉森林到神聖威嚴的至高天，同數百萬玩家一同踏上魔幻 旅程，帶領你的英雄小隊深入古老的廢墟，對抗邪惡的黑暗勢力！培養你的英 雄小隊為正義而戰！
