steam大獎提名，這個獎項的遊戲讓人看到愛情的樣子，你玩過幾個

天天要聞
2019-12-14

這兩天，steam商城的”2019年steam大獎”提名階段已經結束，官方公開了各個獎項入圍的提名作品。

提名《瘟疫傳說：無罪》、《極樂迪斯科》、《孤島驚魂：新曙光》、《戰爭機器5》為”傑齣劇情遊戲”。

除了這個獎項，steam還公布了一個聽起來不怎麼正經卻十分真實的獎項提名。沒錯，這個獎項就叫做”縱使手殘仍大愛”！真是一個又虐又愛的獎項名字啊！

老喵今天就帶著大家看看，哪些遊戲能有如此魅力，能讓玩家就算手再殘也愛慘的遊戲！

1、 《血腥劍斗（MORDHAU）》

《血腥劍斗（MORDHAU）》是一個中世紀的第一和第三人多人砍殺遊戲，在一個虛構但現實的世界裡，玩家以僱傭兵的身份進入一個高達64名玩家的繁忙戰場。

在遊戲里玩家將體驗到殘酷的近戰戰鬥，在殘酷的戰鬥中戰鬥，就需要經歷快節奏的戰鬥，城堡圍攻，騎兵衝鋒等等。

這樣的近戰戰鬥系統，就很考驗玩家對於全局的掌握力，手必須快啊！不然就被幹了！

2、 《噬血代碼》

《噬血代碼》是一款由製作《噬神者》系列遊戲的創作者們所企劃開發的完全新作遊戲，帶有濃濃的二次元黑魂風。

遊戲以以近未來世界為遊戲舞台，玩家要扮演一名吸血鬼，為了追求吸血鬼到底是為何而生，以及這個世界是不是有”出口”等真相，而踏上探索世界的旅程。

作為一款硬核動作角色扮演類遊戲，除了高難度的打鬥，還有高自由的捏臉系統，捏臉真的也很看動手能力，手殘點就長殘了……不少玩家也吐槽這個遊戲地圖繞，但還是擋不住真香……

3、 《獵殺：對決（Hunt：Showdown）》

《獵殺：對決》是一款來自《Crysis》系列製作團隊的第一視角競技式PvP賞金追獵遊戲，包含大量PvE元素。

《獵殺：對決》是一款十分與眾不同的喪屍題材遊戲，遊戲設於世界最陰暗的角落，將生存遊戲的刺激感注入競技式的玩家對決中。

遊戲同時包含了PVP與PVE的要素，遊戲機制也比較複雜，許多新手剛上手這個這個遊戲的時候都不知道該要幹什麼，但就很充滿吸引力。

4、 《遺迹：灰燼重生》

該遊戲是一款第三人稱的合作生存遊戲，背景設定在一個被異次元怪獸佔領的末世地球。

遊戲中支持最多4人組隊，將會擁有動態生成的關卡，並支持人物自定義和裝備武器製作，遊戲充滿激烈的射擊和近戰格鬥元素，玩家可以用各種方式來探索遊戲世界，並和致命的怪物、史詩級BOSS酣戰。

作為一款魂like遊戲，其難度就是”死亡總是無處不在地伴隨著你”。但是你可以通過不斷升級自己，來讓自己變得更強、更難以殺死。這種虐來虐去的感覺就好像莫名很受人喜歡？

5、 《真人快打11》

《真人快打11》（Mortal Kombat 11）是一款由NetherRealm Studios開發的格鬥遊戲，《真人快打》經典系列隆重回歸，是《真人快打》系列最新作。

在這款更為酷炫和華麗的新作中，全新”自定義角色模板”功能可讓你以史無前例的方式自定義戰士，打造完全屬於你的角色。

伴隨著一直以來膾炙人口的電影劇情模式，再加上一眾全新以及重磅回歸的經典戰士，在推出25周年之際，《真人快打》這部史詩系列將迎來其新的高峰。

除了已經公開的”2019傑出視覺風格獎”、”2019縱使手殘仍大愛”和”2019傑齣劇情遊戲”這三項獎項，還有尚未公開的5個獎項提名，它們分別為：年度最佳遊戲、年度最佳VR遊戲、愛的付出、獨樂樂不如眾樂樂、最具創意遊戲玩法。

對這些獎項，老喵還是充滿期待的，小夥伴們覺得哪些遊戲最有可能獲獎？
