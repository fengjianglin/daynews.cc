想打職業要有啥條件？李哥4個字刷掉99％玩家，香鍋竟無辜躺槍

天天要聞
2019-12-20

我們都知道，英雄聯盟作為一款電子競技遊戲，很多玩家其實都是普通玩家，只是抱著玩玩的心態。但是對於很多人來說，英雄聯盟很可能是一個職業，例如對於職業選手。職業選手在我們心中的印象基本上只有一個，那就是強大，這也是電子競技的殘酷性了，電子競技，菜是原罪，你的一切實力都擺在了上面，大家看的明明白白。那麼大家想要打職業，至少得是個什麼條件呢？

有玩家就問了faker，如果想成為職業選手，是天賦重要還是努力重要，李哥的想法是兩者五五開，或者說是兩者都要有，不然是打不出好成績的。之後李哥又說了一句話，那就是想要打職業的話，最低條件就是王者段位水平，如果低於這個水平，或者打上這個段位很吃力，不容易保持下去，那基本上就可以考慮放棄了。李哥這4個字「最低王者」就直接刷掉了99%玩家了，畢竟現在段位最多的還是白銀，黃金，白金這三個段位的。

王者對於普通玩家而言還是很遙遠的，就算是在國服，一個鑽石都是大家眼中的高手了，而大師，宗師更是遙不可及，更別說是王者了。不過想一下，這是李哥，就覺得釋懷了，畢竟是說出「菜的像個大師」的人物，還是韓服大師，這句話說得大家不得不服。不過李哥這句話就讓冠軍打野麻辣香鍋無辜躺槍了，畢竟鑽石野王。

當然了，香鍋這個鑽石野王也只是一個梗，以前香鍋也是在韓服打上過前十的存在，如果不是賽事比較多，香鍋應該不會掉這麼多分的。個人認為，一個人相打職業的話，第一要有天賦，第二得要努力，隕落的中單之光還少了嗎？
