摘要

《英雄聯盟》發展到現在，已經有了各種各樣的英雄存在，許多英雄都有著自己的特色，但是如果小球球要說有的英雄還能改變玩家的智商你信不信？今天小球球就來說說在英雄聯盟里都有哪些一開了大招，智商立刻就降低的英雄。

《英雄聯盟》發展到現在，已經有了各種各樣的英雄存在，許多英雄都有著自己的特色，但是如果小球球要說有的英雄還能改變玩家的智商你信不信？

今天小球球就來說說在英雄聯盟里都有哪些一開了大招，智商立刻就降低的英雄。

劍聖

首先不得不提到的就是這個易大師了，劍聖的機制其實就在於收割，尤其是大招開啟以後。

​每次看到易大師的高原血統一開，就宛如瘋狗沖入人群，迎來的消息也是不是敵人死就是劍聖死，一點點迴旋周轉或者拉扯的餘地都沒有。

開了大招的劍聖，就如同一個莽夫，總之今天必須要有一個人倒下。

​奧拉夫

說起莽夫，這就不得不提到一個真正的莽夫英雄了。奧拉夫這個英雄其實除了缺乏硬控，其他方面還是比較完美，有回復，雖然沒位移，但是有減速，還自帶真實傷害。

​沒開大之前的奧拉夫其實還是蠻正常的，有時候看隊友拿奧拉夫上單，6級之前，還別說，真的是細節拉滿，知道撿斧頭，控藍，回血，那操作堪稱行雲流水。

然而一到6級後，奧拉夫就彷彿換了一個隊友來打一樣，尤其是打團的時候，突然就聽到一聲怒吼，就看到自己隊友變紅了然後獨自一人衝進了對面的陣容。

真的是風蕭蕭兮易水寒，壯士一去不復還。

​諾克薩斯之手

嚴格來說，不是說諾手開大後變成瘋狗，而是諾手血怒一開的時候，大招會發生質變，所以準確來說是開了血怒的諾手。

​諾手血怒一開，就感覺整個峽谷都是自己的，自己就是無人能敵，不得不承認，諾手開了血怒以後的確很厲害，但是每次開了血怒的諾手就這麼直愣愣地往對面沖，然後還被隊手不斷風箏，不得不讓我懷疑他的血怒都是用智商換的。

​劍魔

自從Theshy劍魔天神下凡以一敵五以後，小球球發現峽谷的劍魔玩家多了不少，不過，幾經削弱的劍魔其實也早早不如之前了，然而，每當有看見我方劍魔開大的時候，都以為自己是無敵的殺神，沖入人群，還帥不過三秒，三段Q都沒打出來就死了。

​所以，劍魔這個大開起以後，也是讓人喪失理智的。

​以上幾個英雄，其實大招都是蠻強的，但是往往越強越容易讓人上頭，所以每每在遇到這幾個英雄，當他們開大的時候，我們總是感覺到他們的智商也在急劇下降。

那麼，在英雄聯盟里，你還知道其他哪些一開大招智商就降低的英雄，也可以一起來討論哦。
