雷霆羈絆AOE拉滿 三大變陣套路輕鬆上分

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

在這個海洋雲霄打遍天梯的版本，雷霆也悄然崛起，雷霆放出的AOE傷害能夠有效的針對刺客，而且雷霆本身和其他陣容很搭配，特別是雷霆體系中的劫，分身也能釋放雷霆的AOE，讓整個棋盤電閃雷鳴。

在這個海洋雲霄打遍天梯的版本，雷霆也悄然崛起，雷霆放出的AOE傷害能夠有效的針對刺客，而且雷霆本身和其他陣容很搭配，特別是雷霆體系中的劫，分身也能釋放雷霆的AOE，讓整個棋盤電閃雷鳴。

雷霆守護神

陣容構築

奧恩、錘石、狗熊、泰坦、奇亞娜(山脈)、石頭人、螳螂、劫

羈絆：4守護神、3雷霆、3刺客、雙山脈、

這套雷霆守護神是唯一一個成型不怕法海和雲霄劍刺的陣容，但是成型比較慢，要8人口才能和其他陣容拼後期，所以要合理的運營。

裝備選擇

狗熊是當之無愧的主C，雷霆提供的AOE效果能夠很快的壓低對方的血線，這樣狗熊就更加的方便斬殺，因為雷霆的傷害來自暴擊或者被暴擊，所以要儘可能的提升狗熊的暴擊率，魔暴手套和無儘是最好的搭配，還可以來一個防裝龍牙。副C是劫，同樣也需要無盡，再來一個復活甲。

運營思路

優先搶無盡的散件，有了無盡雷霆羈絆的效果才能夠不斷的觸發。前期就無盡的話直接給狗熊很容易就能打連勝。前期的陣容選擇4守護神2雷霆，即使是走連敗，也能保持一個非常健康的血線。

中期需要忍一手，只要血線不到3危險的地步就升人口，爭取最快速度到8人口，然後直接開始找劫，拿到劫把裝備給劫，陣容成型。剩下的就是找一個雷霆拉克絲來強化陣容了，成型的雷霆守護神直接鎖血，只怕一手4沙漠，不過這版本幾乎沒有4沙漠。

雷霆刺客

陣容構築

皎月、奧恩(幽夢)、雷克塞、夢魘、奇亞娜、螳螂、劫、拉克絲

羈絆：6刺客、4雷霆、2鋼鐵、1大元素使

裝備選擇

這套雷霆刺客裡面不存在絕對的主C，除了給奧恩分配幽夢外，還可以給奧恩離子火花。然後就是來什麼裝備給什麼裝備，有加攻擊力的給夢魘，能回藍的就給螳螂，無盡給劫，法強裝備就給拉克絲。

運營思路

前期賣血拿鏟子，拿到鏟子馬上走3德魯伊3刺2林地，在打石頭人之前適當的D一下，爭取D出3星的奧恩和皎月。6人口再上一個雷克塞，湊出雙鋼羈絆。這套過渡陣容至少在很長一段時間內是沒有敵手的，然後不斷的拿刷出來的刺客。再沒有劫之前裝備先給妖姬。8人口開始找劫和雷霆拉克絲，沒有雷霆拉克絲就用狗熊代替。

雷霆的AOE配合刺客的暴擊簡直是絕配，最完美的情況下就是劫和奧恩跳到後排，不斷的打出暴擊和傷害，對後排造成非常恐怖的威脅。

雷霆劇毒

陣容構築

狗熊、蒙多、風女、老鼠、煉金、劫、拉克絲

3雷霆、3劇毒、雙秘術、1大元素使、1煉金師

裝備選擇

主C為劫和煉金，劫的裝備選擇不需要贅述了，煉金的話主要是拿防禦裝備冰心加狂徒，有風元素地板可以給他冰拳，煉金還需要一本鬼書來打輸出。

運營思路

這套雷霆劇毒的造價非常的誇張，這也就意味著前期的過渡陣容都要換掉，前期先看看能走什麼體系，比如有電刀盧登就可以走劇毒掠食者，當然最好合一本鬼書給2星婕拉，做個地獄火召喚，後期鬼書還能夠給煉金用。賣血升人口只留奧恩和狗熊，中期拿到老鼠上三劇毒就能夠保住血量了。

後期就是慢慢的換上雙秘術，再找劫和拉克絲，成型的話也是沒有多少對手。難點就在於攢錢還有搜出這些英雄來。

總結

這三套雷霆體系後期成型都非常強，如果看到沒有同行的話，就果斷的走雷霆吧。
