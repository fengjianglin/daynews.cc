古神最後希望登場《魔獸世界》恩佐斯化身年獸降臨艾澤拉斯

2019-12-23 1,805條評論 2,731次閱讀 228人點贊

當腳男們在納沙塔爾擊敗了艾薩拉女王之後，新的故事已經拉開帷幕。我們都知道在8.3中團本的尾王是腐化者恩佐斯，也就是意味著這個從魔獸開篇就存在的角色，會在當前資料片裡面真正露出它的真容。目前的艾澤拉斯之上恩佐斯是僅存的最後一個古神，雖然說一直有玩家推測五大巨龍對應著五大古神，但是從已知的材料和暴雪爸爸透露出的信息來看四大古神應該不會中途出來一個新人，也就是說在陸續擊敗了克蘇恩、尤格薩隆之後我們終於迎來了最後一個古神的戰鬥，他的名字毫無疑問就是恩佐斯。目前新版本8.3的開放時間已經公布了為1月20日這個時候正值我們新春佳節，看來又到了一年一度的春節開荒階段了。

從劇情上看，恩佐斯絕對不是最強的那個boss，首先從力量上說亞煞極的力量是公認的最強者，甚至遭到了泰坦的嫉妒被直接人道毀滅了。另外三個古神的實力相當，從劇情上講虛空大君為侵蝕初生泰坦而投入艾澤拉斯的四大上古之神之一，也是繼克蘇恩和尤格薩隆之後，第三個出現在遊戲中的古神名諱。形象借鑒了克蘇魯神話的設定。之前的亞煞極已經被泰坦直接消滅了，所以也不存在突然冒出來一個副本亞煞極再臨。也就是說觸手怪可能會和巨龍一般從此消失在艾澤拉斯之中。

恩佐斯干過最出名的事情相信各位玩家都有所耳聞，在它被泰坦封印後，它仍利用陰謀將曾經無比睿智的黑龍之王耐薩里奧變成了死亡之翼臣服於自己，又在上層精靈們遭遇危機時出手相救，卻把它們都轉化為仇恨一切的娜迦海妖作戰。從此艾澤拉斯上有了兩個超級BOSS，黑龍死亡之翼和娜迦女王艾薩拉女王。恩佐斯在蠱惑人心上可以說是四大古神中的翹楚，而他一直也是以智者的形象存在於艾澤拉斯原住民的腦海中，誇張點說他甚至是《魔獸世界》裡面最聰明的怪物，有沒有一絲期待與它戰鬥的感覺呢？

恩佐斯被多次貼上最為弱小的上古之神的標籤，比起其他的古神恩佐斯比其他幾位顯得更為狡詐，目前他還從未與世界的英雄正面戰爭過，也許他是在尋找時機——一個能將艾澤拉斯徹底推向毀滅的大好機會。我們都知道在9.0中女王希爾瓦娜斯開啟了暗影界與艾澤拉斯的大門，這是我們第一次與虛空大君正面接觸，同樣新的故事自然要了卻舊日的恩怨，這將是腳男們的最後機會，那麼春節期間你是不是要來斬殺恩佐斯這個年獸呢？

既然暴雪爸爸要將恩佐斯設置成我們新年的年獸，貪婪的腳男們相信你們絕對也是能吃下這塊肥肉的。對了目前懷舊服也在如火如荼的進行中，大家都知道懷舊服中克蘇恩可能比克爾蘇加德還要厲害許多，在安其拉之門開啟之後，第一個古神與最後一個古神你會選擇誰呢？對於艾澤拉斯的勇士們來說肯定是——我全都要！
