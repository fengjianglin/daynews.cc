《諾亞傳說》手游測評：拯救世界末日，星際大戰一觸即發

2019-12-24 3,451條評論 4,542次閱讀 1688人點贊

見多了科幻片中的那些超出想像的高科技，總是會希望自己也能身處那樣的環境中，親自體驗高科技環繞的感覺。雖然現實中難以實現，但是我們還是可以在遊戲世界中體驗一把。今天要給大家帶來的遊戲《諾亞傳說》就是由經典端游改編而來的未來科幻團戰PK手游，下面筆者就給大家帶來這款遊戲的評測！

與《流浪地球》結成官方首席合作手游關係

手游《諾亞傳說》在繼承端游的經典玩法的基礎上，還將端游原先的2D畫面改成了3D呈現，使得遊戲整體畫面略有加分。並且還與同為”拯救末日”主題的電影《流浪地球》結成了首席官方合作關係，這點在手游剛開始時的界面中就可以深刻的感受到。

遊戲開始之前的背景說明

遊戲評分：7.9/10

遊戲類型：角色扮演

2D變3D，遊戲整體畫質升級

對於科幻題材不論是電影或是遊戲，都會給人一種高大上的感覺。雖然手游《諾亞傳說》改編自端游，但是卻不再是2D畫質，而是採用了最新的3D建模。遊戲整體畫質有了極大的提升，而且又是科幻主題的遊戲，我們可以在遊戲中清晰的感覺到爆炸所引發的巨大威力，還可以感覺在星際中戰鬥的那種科技感，這些對於玩家們在視覺上的衝擊還是很大的。

組隊團戰

當然，除了畫質，還有人物造型及色彩搭配上，相較於其他同類手游的寫實風格，《諾亞傳說》則顯得高雅的多了。每個角色都各有特色，不會讓人產生千篇一律的感覺。

主線任務中

職業系統更加豐富，玩家可任意選擇

無論哪款手游，玩家都必須面對角色以及職業的選擇，《諾亞傳說》自然也不例外。進入遊戲後，我們看到的便是角色的創建。手游《諾亞傳說》同樣也繼承了端游的職業系統，裡面有善於近戰物理輸出的劍武士、法術爆髮型且可以遠程大範圍輸出或緩傷的異能者、遠程物理輸出的槍械師，正面硬剛的肉盾格鬥者以及手游中全新推出的擅於治療的治癒者五種職業。

劍武士、異能者、槍械師

格鬥者和治癒者

而且每種職業都各有特色，擁有自己獨特的技能，在遊戲PVP團戰中，玩家們可以根據每個職業的特點，搭配出合適的出戰陣容，肉盾近戰在前，遠程爆發和治癒在後。強大的陣容搭配將擁有強大的戰鬥力。

星際大戰的多樣玩法是亮點

《諾亞傳說》既然是端游的傳承，在玩法上自然也是一脈相承，經典的PVP玩法也是讓玩家們倍感熟悉。除了前期開啟的1v1的PVP，後期還能開啟5V5的團戰模式，當然最主要的還是遊戲中最核心的星際團戰。

面對即將到來的戰鬥

但是這看起來似乎與市面上的國戰玩法又有點相似了，但是《諾亞傳說》中的星際團戰則更需要策略，陣容的搭配，適合的時機，都需要玩家有一個很好的敏銳度，這樣戰爭才會有勝算。還有一點對於玩家來說是非常好的，那就是在戰鬥中獲得的裝備無綁定，可以隨時交易，而且價格也是由玩家自行決定。

唯一美中不足的便是上帝視角下的自動戰鬥

原本手機遊戲便會有很多的限制，手游《諾亞傳說》在繼承的時候，同樣繼承了端游的俯瞰視角，這樣的上帝視角，而且還是自動戰鬥，憑白的讓遊戲少了很多的樂趣。

自動戰鬥中

在《流浪地球》的大IP下，科幻的星際大戰，應該讓玩家能夠感受到類似於電影中那般激烈的戰鬥場景，而不是以一個外觀者的身份，看著遊戲里在戰鬥，卻無法感同身受，，可以說是沒有什麼代入感的。

與怪物戰鬥

加分項：

3D畫質，視覺衝擊更強

職業多樣可選

經典玩法傳承

自由交易不受限制

減分項：

自動戰鬥沒有挑戰性

不氪金難玩

操作感，戰鬥感差

個人覺得《諾亞傳說》作為一款國產科幻手游，在繼承端游特色的基礎上，又加入了一些新的玩法，也算是用心的了，畢竟手裡還攥著國產大電影《流浪地球》的大IP呢？雖然在畫質和玩法上也都可圈可點，但是我還是覺得這種俯瞰視角下的自動戰鬥未免也太無趣了點。畢竟遊戲本身就是以科幻星際大戰，拯救末日為主題，應該讓玩家在遊戲過程中產生強大的帶入感，而不是簡單的自動戰鬥就完事了。在這裡，還是要提醒各位，如果對操作技能，戰鬥感方面有極大要求的，這款遊戲就不太適合你了。
