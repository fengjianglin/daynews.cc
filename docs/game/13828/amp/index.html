前期跟射手這邊4級一套123連招直接秒掉阿珂，劉禪操作很簡單，難的是你的打法思路，就比如這一局，對面有阿珂鍾馗我會全程保護我方C位，這個英雄非常克制鐘馗。

技能肯定主一，因為一技能會加移速搭配隱匿銘文，再加上極影，後期移速和技能冷卻都非常快。黃忠打我打得起勁，這就是他的錯覺因為我們一技能會加護盾，並且冷卻非常快。

我們1 2技能可以沉默防禦塔一秒，並且所有技能攻擊防禦塔都可以回血，我們可以利用這一點配合隊友越塔抓人。沉默防禦塔的時候最好連人帶防禦塔一起打，這個鐘馗這局守了至少4次塔 結果每一次都帶著防禦塔一起回泉水了。 這邊給大家看一下什麼叫輔助單吃射手我笑了，劉禪到中期是最噁心的，出了冷靜靴，極影移速快，冷卻快再加上一二技能循環控制，再搭配冰握的減速，只要黏上目標他可以被你一路控到泉水。這黃忠知道跑不掉了原地開大，加雙抗拖延時間 但是你的隊友好像在中路啊。

這裡假裝沒看見阿珂，等他近身直接就是一套連環控制加大招，玩劉禪你的一技能是可以存的，存一次一技能打團先普攻打出上一次控制，再按一技能打出第二次控制。說到底這個英雄的核心打法就是推塔、保後排、用超短技能CD 打連環控制留人。
