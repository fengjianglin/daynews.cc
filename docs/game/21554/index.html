OWL選手看待全球開賽，Surefour：上海觀眾很熱情，還能見到粉絲

2019-12-19 1,248條評論 3,921次閱讀 2022人點贊

《守望先鋒》職業大聯盟（OWL）將在2020年2月開始的第三季賽程實施全主場制，讓全聯盟30支隊伍在半年多的比賽中走遍全球。這樣的機制在社群間激發了不少討論，受影響最大的選手們倒是比較樂觀，OWL官方訪問了一些選手關於他們明年即將開始的旅程，多數都持著正面的看法。

歐洲成為想要多看看這世界的選手們的首選，休士頓有三位選手都想要去歐洲看看：輔助選手Boink表示「我從來沒有去過北美以外的地方，所以我特別期待在巴黎的比賽，更何況我們只有那個系列比賽不在北美。」輸出Blase也同意他的想法「我連美國一些大洲都沒去過，何況是國外。」作為隊中「扛霸子」的Danteh則對法國菜情有獨鍾「我很高興可以去巴黎，聽說那邊的食物超棒的。」

想要去一覽歐洲風光的不只美國人，紐約輸出組合Nenne、Saebyeolbe也對歐洲表達嚮往，SBB鍾情於歐洲城市景觀、Nenne則認為倫敦與巴黎是歐洲最值得旅遊的城市。

佛羅里達的射控專家Sayaplayer並不討厭前往大城市，但他更期待與隊伍一同在陽光明媚的佛羅里達州比賽；訪問當時還沒宣布退役的Jake（延伸閱讀：《守望先鋒》前美國隊長、休士頓神槍手DPS Jake宣布退役）喜歡自家主場的原因倒是很簡單：休士頓的烤肉是他的最愛。

加拿大在網路上被國外網友揶揄成世界上最友善的國家，來自韓國的Tobi與來自挪威的ELLIVOTE都想要造訪這個地廣人稀、對外來者相當友善的國家。土生土長的加拿大人Note最期待的卻是前往他無數隊友的故鄉：韓國，「我想要去韓國很久了，現在終於有機會！」

在韓語是「第二外語」的電競圈，韓國理所當然的成為選手們想要造訪的國家之一，除了許多出外工作的韓國選手能回到熟悉的環境以外，與韓國隊友朝夕相處的外國選手也對這個國家充滿好奇。來自西班牙的Harryhook、來自芬蘭的Shaz特別想吃韓國的食物（Shaz的隊友Biggoose甚至在學韓文）來自瑞典的Nevix仍然對他上次造訪韓國時熱情的粉絲念念不忘。

在中國擁有高人氣的Surefour則提到上海「我很高興我們會去上海比賽，聽說那邊的賽事跟觀眾都相當熱情，而且還能見到粉絲們。」

大部分選手對於這樣的賽程/旅程都是期待大於擔憂，能夠到訪許多不同城市、在家鄉比賽對於年輕選手來說一定是相當大的誘因，聯盟方也會想方設法讓隊伍與當地做更緊密的結合，2020賽季相當令人期待。
