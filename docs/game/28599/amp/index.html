夢幻西遊：主動技能加強，6種新寵物技能放出，大量寵物面臨淘汰

天天要聞
2019-12-22

在上一篇文章中，大熊給大家講了新資料片發布會上，策劃公布的一些新召喚獸圖片。當時大熊以為只有9種新召喚獸和2種新寵物技能，可沒想到真正的新召喚獸數量竟然有11種，而且新的寵物技能也有6種之多，沉寂了這麼久的煉妖行情估計要開始爆發了吧。那麼這11種召喚獸的遊戲內造型以及新技能都是什麼樣的呢，話不多說我們一起來看看吧！

首先是參戰等級為55級的小狐狸「花鈴」，光看普通造型似乎沒啥亮點，但看到它的進階造型後，真香！

我們再來看看第二隻新召喚獸，也是策劃發布會上沒有介紹的一隻，叫做「月影仙」，參戰等級是65級。

這隻同樣也是參戰等級65級的「獨角鬼王」。並且攜帶新技能「進擊法暴」。技能說明是提升10%的法暴幾率，也就是說高級法暴的15%加上進擊法暴的10%，一共就有25%的法暴幾率，法寵要大洗牌了。

日游神，參戰等級是85級。攜帶3個必帶技能的同時，資質和成長都完爆淚妖，大家懂我意思了吧？

狐不歸，參戰等級105級，這造型絕對是神威玩家的新寵兒呀。狐不歸也攜帶一個新技能「進擊必殺」，技能效果為提升15%的必殺幾率，與高級必殺搭配的話那就是35%的必殺幾率了，攻寵必備。

與進擊法暴相同，也是不知道會不會出現在低級獸決裡面，如果必須要煉妖才能有，那現在的任務寵全得淘汰了。

夜遊神，參戰等級125級。這隻新召喚獸也攜帶了一個新技能「勢如破竹」，保護單位分擔傷害減少30%，善惡、力劈必備技能。

月魅，參戰等級為飛升後125級，沒有攜帶新技能，但是不是看起來有點像夜羅剎呢？進階後的造型也是美到炸，帶個性夜羅剎的老闆們慌不慌？

無常鬼差（黑），參戰等級為渡劫後155級，攜帶新技能「高級遺志」。這個技能可以讓替換上場的召喚獸傷害提升10%，最多疊加兩層。這個技能搭配上10月大改後的高級鬼魂術，倒地就拉直接起飛。而且這個技能無論對什麼類型的召喚獸都有很大的加強。

無常鬼差（白），參戰等級和攜帶的新技能都與上面的相同，只不過是另一個類型而已。

塗山雪，參戰等級化聖175級，只可惜大熊沒有找到進階後的遊戲內造型圖。塗山雪攜帶了一個新的召喚獸金技能「出其不意」，只要是本回合己方召喚獸沒使用過的技能就能增加15%傷害，有主動類技能的召喚獸要直接逆天了。試想一下，4隻不同法術的須彌寵加上出其不意，輸出會有多麼爆炸。

最後一個新召喚獸「諦聽」，參戰等級也是化聖175級。也攜帶了一個新技能「觀照萬象」，可以在第十回合的時候，一次性使用出召喚獸身上的所有主動技能，包括善惡、力劈、壁壘、四大法以及小法等，輸出有多恐怖請大家自行想像。

以上的新技能是否為特殊技能暫時還不清楚，但如果6種技能都是在特殊獸決中，或者是只能通過煉妖獲得，那麼現在所有的召喚獸估計都要被淘汰掉了。這是在2014年須彌真言放出後，第一次全方位的史詩級增強，功法兼備的同時還提升了主動類技能的地位，煉妖想不火都難了呀。你們覺得這11種新召喚獸和6種新寵物技能，能夠讓夢幻重新火爆起來嗎？
