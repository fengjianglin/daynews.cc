刺痛成QG唯一背鍋俠？四大致命缺點盤點，告訴你QG為何慘敗

2019-12-16 313條評論 2,089次閱讀 913人點贊

「刺痛也太菜了吧？」

「數據好看有屁用，你看看他的表現」

「求他退役吧」

12月14的比賽結束後，除了對AG超玩會的恭喜以外，對於QGhappy的復盤、反思乃至責問和謾罵也都多了起來；而其中遭受火力最嚴重的，除了老將770以外，就是QG的當家射手Hurt了。人們覺得他德不配位，覺得他耽誤了其他選手，希望他退位讓賢，希望QG能換血重生。

曾經的第一射手，真的變成團隊「毒瘤」，成了QG最大背鍋俠了嗎？

飯飯不否認刺痛打的不好，但實際上除了他以外，導致QG總決賽慘敗的因素還有很多。這些問題不是一朝一夕產生的，而是經年累月印刻在QG的比賽風格之中。想要真正完成重建，改變這些問題可能更加關鍵。

BP：一味求穩

首先，一味求穩的BP和陣容選擇方式是他們目前最大的短板。

可能是因為早年間形成的打法習慣，QG有可能是KPL聯賽中最喜歡拖後期打運營的隊伍了；特別是這個版本射手至上，保後排拖發育似乎也暗合版本趨勢。所以他們在第一局AG放出公孫離的情況下，選出了一個偏poke和反手的3保2體系。

如果真的按照QG的思路，整個比賽進入發育團戰的節奏，他們這套陣容確實很強。結果沒想到AG五人組不走尋常路，利用裴擒虎的射程、周瑜的推進、關羽的高速機動和呂布的越塔強逼，將一個看上去好像要主打前期入侵和團戰的陣容硬生生變成了拆遷流！整場比賽下來總共就6個擊殺，QG可以說是眼睜睜地看著對手將刀子一點一點捅進了自己的心口。

如果說AG這是新研發的套路也就算了，但周瑜強拆體系是QG在對陣DYG的比賽中剛剛用過的啊（飯飯還專門解析過這套體系）！早在AG選出周瑜的時候，QG本應該就開始警惕了，或者中單補法刺，或者輔助or對抗路補開團，在保證前期團戰對抗的前提下依靠強開阻止對手的推進。

但結果，他們壓根沒有意識到這一點，最後一手拿的張飛雖然也有一定的強開能力，但面對AG這種高機動性陣容效果實在太差了。

都說月光教練後置位選了蔡文姬對抗沈夢溪是神來之筆，但其實在5分03秒AG推掉下路一塔並拿到先知主宰的時候，比賽其實已經就結束了。被自己剛剛用過的體系以這種毫無還手之力的方式擊敗，只能說是教練團隊的嚴重失職。

打法：固步自封

其次，打法上的固步自封，真的是QG被詬病已久的問題了。

第三局，馬超+劉邦+張良+香香+莊周，又是熟悉的3保1體系。這個陣容可能很符合QG的打法習慣，但問題真的很大。整個團隊的輸出重心都在孫尚香身上，單一輸出點一旦出問題，這個陣容甚至連接團的資本都沒有。

而且這個陣容還有以下缺點：

1、劉邦打野，清野速度慢發育慢，前期節奏會出現劣勢。

2、張良輸出環境太差。

3、整個陣容只有張良一個穩定控制，還是點控不是團控，前期團戰很吃虧。

4、因為第3點，Fly的馬超缺乏隊友呼應配合，打切入會很難。

QG在打法上喜歡放養FLY，給他一些對線比較舒服的英雄，給他自由發育的空間，這是因為Fly是QG陣容的X因素，只要給他一定的發育空間，他就能用操作創造奇蹟，就比如勝者組決賽第七局的曜和總決賽第二局的狂鐵。

但Fly真的不是神，沒有選手能夠在對手明顯針對的情況下還能一次又一次的完成一打五救主的壯舉。在他被針對，刺痛又啞火的情況下，QG本可以嘗試開發Mojo作為第三核心的戰術可能；但直到第五局比賽結束，我們也沒看到QG有這樣的嘗試，只能說很遺憾了。

770：水土不服背後的無奈

770應該被指責嗎？應該。

單就總決賽的表現來說，他真的遠遠不如對面的愛思。飯飯印象最深刻的就是孫臏這個英雄，在兩個選手手上甚至打出了截然不同的效果。除此以外，他的盾山和廉頗也有失水準，說他是整個團隊的突破口，不過分。

但從另一個角度來看，他之所以在總決賽這樣的舞台上表現掙扎，和他在QG的水土不服又有很深的關係。

我們都知道，770一直都是以「團隊大腦」而著稱，早在老BA時期，他就以臨場指揮和戰術開發而聞名整個KPL。但這樣的名聲，其實建立在另一位選手，也就是他的老隊友——六點六身上。BA時期的六點六作為聯盟第一坦邊，幫助770分擔了大部分做視野、吃技能和開團的任務，這樣770才能安心開著蔡文姬在大團後面邊奶邊指揮。

而在QG，卻沒有另一個六點六了。

Fly不能死，因為他的輸出和帶線牽制至關重要；Giao不能死，打野一死就要掉大小龍；Mojo和刺痛更不能死，就指著他們倆輸出呢。

在這樣的情況下，770探視野被抓，先手開團隊友跟不上就完全能解釋了。他畢竟是一個大齡選手，畢竟是在一支陌生的隊伍里，在這樣水土不服定位轉換的情況下還能幫助QG進決賽，飯飯私心覺得，真的不好對他苛責太多。

刺痛：轉型，還是沉淪

最後，就是刺痛。

其實刺痛的風格一直沒有改變過，早在MU時期，他的打法風格就是穩健發育穩健輸出。之後轉會QG，他也一直是這種打法，所以我們能看到三冠時期那個輸出拉滿的射手之王，也能看見野區自由人版本中那個艱難轉型的新人打野。

一個選手能一直堅持自己的風格固然是一件值得欽佩的事情，但想要成為一個偉大的選手，特別是王者榮耀這種版本變化非常快的MOBA類電競賽事選手，順應版本進行轉型才是真正考驗他們的地方。

我們不知道刺痛不轉型的背後，是自己的不願，俱樂部的不用，還是天賦上的不能；飯飯也很難針對刺痛的表現和技戰術特點再做什麼分析。

畢竟這道坎，只能由他自己跨過去。

總結來說，QG屬於選手的問題其實是最好解決的，但如果他們沒辦法解決印刻在自己打發深處的那種固執，距離冠軍可能就會一直那麼近又那麼遠。

冬冠開賽在即，留給他們的時間已經不多了。
