《FGO》Lostbelt 2.5 1池兩位五星：超人俄里翁、歐羅巴強度簡評

天天要聞
2019-12-22

作者：NGA-Asuka家駒

這次2.5的1池落地了三位從者，這裡從強度角度簡單評價一下，希望對各位有所幫助。
寫完俄里翁已經很晚了，歐羅巴明天見吧。

一、超人俄里翁

1.簡析
優勢：
1.暴力的技能組，超強的火力
2.優秀的白值
3.破格的藍卡性能，單尾藍配合大龍娘連發的潛力
4.對寶具等級的依賴相對較小
5.三回合弱體無效對特殊場景的應對
劣勢：
1.沒有保護能力，作為常規高難通關解沒那麼隨心所欲
2.對遊戲體驗基本沒有什麼提升
一個寶具無傷害的打手型從者，這類從者註定是在佔據絕大多數遊戲時間的周回副本中無法登場的。另一個常被提起的類似定位從者是始皇帝，始皇帝對遊戲體驗的提升是大於超人俄里翁的，因為始皇擁有超強的保護能力，作為常規高難解時發揮非常穩定，而超人俄里翁做不到這一點。
他更偏向於一個純粹的玩具，但在這個定位下，他的上限和潛力要遠遠高於始皇帝。
這是我對他的評價：非常非常優秀且有趣的玩具。有恐怖的火力和新穎的機制，對於高難沒興趣的玩家不推薦抽取，財力有限的玩家抽取一寶也能發揮其大部分潛力。

2.面板、技能與寶具90級12557的atk白值，超過此前弓階第一的阿周那200有多，接近1.12的總等效補正——白值方面完全無可挑剔。
0.96×3hits的藍卡，按面板來的話他應該是0.76×3的藍卡，廚放後是全游頂級，翹楚級的藍卡性能。
無任何buff下染色尾藍60.48，Ex 14.40的NP，寶具頻率相當有保障，組隊時也可以通過T1打寶BA，並且給到泳裝大龍娘的預告信Buff，T2繼續寶具。
綠卡性能不算差但也不突出，Ex卡的性能相當不錯。
指令卡方面，打NP的能力非常優秀，但由於擁有三張的B卡只有1hit，打星的能力比較一般，相當依賴寶具的出星。寶具方面，效果非常多，無敵貫通用到了就是用到了，沒什麼太多好說。
跟弱體無效一同持續3T，這個持續時間很有意義，結合他的寶具頻率基本可以認為是常駐的。弱體無效在當今高難中經常可以用到，對面動不動就給你加一大串debuff，BKB的意義無須贅述，而這是一個常駐的BKB。
攻擊力提升隨寶具30-50，寶具等級的提升僅帶來這一個效果，20攻擊力在buff堆疊後，在一般的場合都不算太重要。並不追求極限通關的玩家，只需要1寶超人俄里翁就可以打出很爽很好看的傷害。
暴傷隨oc100-200，OC提升幅度不小，但比起堆疊難度，基本就是個3T 100暴傷，幅度十分可觀，作為一個3B從者，紅爆是他最重要的輸出手段，梅林的爆傷僅有1T，爆傷的意義不容小覷。
出星3回合10個，小小的遺憾是並不是5T，也不隨OC走，數值不算高。前面說過他的出星能力比較一般，對這個寶具出星非常依賴，如果能再多一點就好了。
綜合來看是很暴力的buff型寶具，縱觀全游這種類型的寶具並不多，輸出端基本是寶藏院同款，副效果俄里翁要好出很多。
我給這個寶具較高的評價，作為一個輸出型玩具，足夠強。技能組
1技能1T 50紅放50猛獸魔性特攻，魔放中規中矩，這個副效果打從者的話跟紅字基本上是互斥事件，只有三個狂會吃。觸發的情況下由於他自身的暴傷buff極多，效果不是那麼明顯。整體並不是一個破格的技能。
2技能複合了蠻多效果，首先是唯一的生存技能戰續，這種類型的從者的確並不那麼需要生存向技能，在個別能吃對面開炮的情況下，戰續或可一用。
3T 20的加攻比較不錯，配合1技能來到了70的常規buff。
出星20堪稱雪中送炭，這個從者還是比較缺星的，儘管擁有集星，但集星綁定了暴傷首回合可能無法開啟，這20出星配合其他出星手段把星星堆一堆，對於啟動回合還是很有意義的。極限場合就靠這20星和弓階裸權重爆兩張卡也並非痴人說夢，非常有意義的效果。
3技能是我非常喜歡的技能，從機制到效果都又強又有趣。
首先集星500不算高但對弓階來說夠用了，旁邊站的都是拐漏不了多少。
重中之重是這個暴傷效果。B卡攻擊前置的付與暴傷，如何生效全看如何出卡，無論是打BBB分別擁有100-200-300暴傷最大化火力，還是分開打BQA等價連續3回合的100暴傷都十分美好，在單挑場景和組隊場景里都有無盡的潛力。
作為一個輸出玩具，最需要的就是這種東西，給了你無窮的上限，等待我們自己去探索。
我給這個技能滿分，創意、強度都非常非常棒。

具體應用
簡單說一下幾個可能的應用場景吧。
首先是單挑，俄里翁僅有的生存技能是一個戰續，從機制上講並不如其他常客一樣契合單挑，但火力太恐怖了，打一般血量的車輪一張卡一個人不是沒有可能。經過精心策劃未嘗不會有極為精彩的表演視頻。
不過對於一般玩家來說他的單挑沒有黑狗那麼好掌握，難以成為一拖五過高難的角色。
其次是組隊，組隊正常出卡攻略高難受限於藍卡只有一張，寶具每三回合出倆基本上沒什麼戲，帶倆梅林一個一個砍在當前環境下似乎也不那麼夠勁兒。這就是之前說的他對遊戲體驗的提升很小，不能幫你妥妥的腳打高難。
而考慮送人洗牌的情況是他能夠發揮無窮上限的場景，是一個玩具真正好玩的地方。
無論是第一回合直接BBB爆破，還是連續3回合的寶BA-寶BB-BBB爆發理論上限都非常恐怖，僅從Excel里看過去簡直是全游輸出最強的男人。
當然後者過於手操，在實際應用中很難遇到，但他的傷害恐怖到T3即便打BxB傷害也突破天際，這種表演還是可以期待的。
由於這種玩法比較小眾，這裡只提出一個輸出模板，供大家參考他的傷害性能。

如果有大量重開的決心，超人俄里翁是全游輸出天花板，當然這種操作多數人是不會去嘗試的，這裡貼出數據也僅僅是供大家參考，實際應用的時候不需要這麼極限的發牌，俄里翁的輸出一樣會非常爆炸。
一個這種類型的從者，能做到這些，足夠強了。使用他的直觀感受就是爽，1hit的紅卡超好看的暴擊數字，在重複勞動刷本的疲憊之中穿插著一兩次天文數字的暴擊傷害，我很喜歡這樣的感覺，不知各位是否也是如此。

二、歐羅巴

時間太晚了就只給一個簡析吧，後面再慢慢補齊。
優勢：
1.自充達到人權線，最起碼是可用的從者。無論孔梅起爆還是配小番茄吉娜可帕爾瓦蒂打兩面都可以應用。
2.生存能力非常好，真的非常好，有成為很棒的單挑從者的潛力。
3.火力可以接受，自buff單發紅字破9萬。
劣勢：
1.寶具buff無法堆疊，連續開炮火力沒有進一步的提升
2.沒有太明顯的長板，無論是指令卡傷害還是NP獲取，都沒有什麼亮點，即沒有那種「非她上班不可」的場合。
其餘的像NP能力-符合面板，可以接受但不破格。 白值很棒，跟低不沾邊，還有其他具體的後續再說吧。
整體的評價就是比較一般，如果生在舊紅卡時代有機會成為騎光炮主力，在當今環境下對遊戲體驗的改變較小，純強度的角度不推薦財力一般的玩家抽取。
沒有明顯的亮點和嚴重的缺點，創意和強度都不是很足。

中間許多思路受組內的各位啟發，難以一一感謝，這裡統一致謝。
