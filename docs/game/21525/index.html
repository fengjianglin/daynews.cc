2020春節版本前瞻 三分鐘看完更新內容

2019-12-19 975條評論 3,655次閱讀 2738人點贊

12月14日體驗服進行了維護更新，新增了一些內容，版本名為「決戰!魔界風雲」。

那麼接下來就讓我們看一看更新了哪些內容吧!

一、任務欄調整

1、任務欄右上角顯示已接任務類型。

2、任務分組細化，新增「主線」「活動」「冒險」「外傳」分組。

二、人物辭典

遊戲內新增人物辭典，可通過遊戲內系統菜單可進入。

辭典上的NPC會隨著主線任務解鎖，可以用於查看各個NPC的好感度及背景故事介紹等。

三、七大職業平衡

分別對「聖騎士」「驅魔」「死靈」「阿修羅」「魔道」「決戰者」「劍宗」這七個職業進行平衡改版，其中「魔道」「阿修羅」「決戰者」「劍宗」為國服特色職業平衡，「死靈」「聖騎士」「驅魔」為韓服今年8月份的平衡改版。

四、附魔卡/寶珠交易限制

所有的附魔卡與附魔寶珠均變更為僅可交易1次，交易後變更為賬號綁定，包括在更新之前已經獲取的卡片/寶珠。

五、魔界大戰CP系統

勇士們可通過副本內獲取變更職業35、40、45、60、70級技能效果的「護石」與「符文」。

除此之外，還新增了副本內怪物的附魔卡片。

六、社交優化

社交欄新增個人在線狀態設置，可設置為「在線」「忙碌」「離開」狀態。(參考QQ)
