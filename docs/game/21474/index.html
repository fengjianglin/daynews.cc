陰陽師2020年新SSR/SP式神衛星匯總 SP雪童子和UR大天狗預定

2019-12-19 200條評論 1,190次閱讀 274人點贊

陰陽師對於這次官方公開的決戰京都各個地方的線路中突發的各種劇情，估計不少玩家已經看出來了，這次的劇情基本上透露了未來2020年新SSR/SP式神的衛星，不過具體都有哪些式神估計不少玩家還不能確定，這裡就明確彙集和挑明一下這些式神衛星，一起來看看吧。

SP山風/SP薰

基本上已經確定了會出SP山風和SP薰，畢竟蒼風一目連直接給了山風力量讓其變強了，同時薰也去了森林中，這明顯直接挑明了這對CP會雙雙進化成為未來的新SP式神。

鬼域新SSR

劇情中大江山這邊瘋狂的進行了茨木鬼手原主人的信息，雖然這個設定有嚴重的吃書嫌疑，之前鬼手裡血液的主人，理論上已經被茨木給滅了，結果現在又出來了，那基本上就是明示了這肯定是個新SSR式神。

UR大天狗/SP輝夜姬

理論上常規遊戲中高於SSR稀有度的等級應該是UR，陰陽師可能一開始處於保守沒有直接挑明這個問題，所以出的是SP稀有度，但是目前劇情中明顯提到了自己目前有三道封印，而自己只解開了一道封印而已，也就是說大天狗的力量頂多也就發揮了三分之一，而接下來他要去望月之地——月宮尋找真相，這明顯預示著更強的UR大天狗和SP輝夜姬。

UR荒川之主/SP金魚姬

這個不需要多提了，官方已經明確了鹹魚王沒掛，只不過是區區肉身沒了，魂魄尚在，所以荒川之主明顯還是可以王者歸來的，到時候和大天狗一樣是直接UR荒川之主走起，然後SP金魚姬這邊順利登場，大功告成。

SP雪童子

劇情中挑明了雪童子體內埋藏著一股力量，只不過他自己都不知道而已，雪走是一把好刀，需要一個很好地契機來覺醒真正的力量，擺明了就是告訴玩家，明年SP雪童子穩了。
