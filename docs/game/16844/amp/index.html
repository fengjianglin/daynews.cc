王者榮耀：張大仙“冰心思想”深入人心，直播玩馬可差點也出

天天要聞
2019-12-16

王者榮耀在這一個版本稱為「射手榮耀」，射手的輸出已經過分的超標，成為了一場遊戲當中的絕對核心位置，只要是能夠在團戰當中保持射手的輸出位置，基本上團戰就能夠獲勝，不僅僅是在路人局中，像職業聯賽當中，射手也完全的固定為一個核心位置，野核已經鮮少露面。

而作為王者榮耀直播版塊知名度最高的一名主播，張大仙平時也經常玩一些射手，也因為張大仙的個人英雄池非常的深，平時各種英雄都喜歡玩，但是張大仙最喜歡的還是上單，相信「萬物皆可出冰心」這一個梗大家都非常的清楚，從側面也反映出來了，主播張大仙對於邊路戰士的理解程度還是比較到位。但是張大仙的一些射手玩的也是比較好，在此之前，主播張大仙一直都非常喜歡一個射手，那就是在前段時間大火的射手一姐孫尚香，主播張大仙也經常向直播間觀眾分享一些關於孫尚香的使用心得。比如前期第一件裝備出黑切，來大幅度配合孫尚香的二技能，從而打出高爆發的傷害，讓許多的粉絲們有所收穫。

但近期主播張大仙卻又盯上了另一個英雄，在直播排位的過程當中，張大仙的射手優先順序方面，馬可波羅的優先順序要明顯高一些，原來主播張大仙喜歡上馬可波羅這一個英雄了。其實一直以來在高端局當中，馬可波羅的優先順序並不比孫尚香低，甚至要更「吃香」，只不過馬可波羅要更考究一些細節操作。

主播張大仙喜歡上馬可波羅這一個英雄，原因是前段時間馬可波羅得到了一次調整，一定幅度的增長了，馬可波羅在前中期的技能傷害，但是也削弱了真傷的傷害，在後期的時候輸出相對於之前有所下滑，但是也正是因為在前中期傷害更高，導致馬可波羅發育的速度更快，所以這次調整嚴格意義來講，也不算是削弱，並沒有影響馬可波羅這一個英雄的選取優先程度。

在直播的時候，主播張大仙還向我們介紹了馬可波羅的出裝以及銘文攜帶，相對於其他的玩家來講，主播張大仙，紅色秘聞還攜帶了幾個紅月，更加的細節，還有張大仙也表示，馬可波羅的細節在於大招的釋放時機，千萬不要先手進場，開團先用一技能消耗，關鍵時刻再後手上去消耗，更搞笑的是，張大仙玩馬可出裝時，差點出了一個冰心。在連續好幾場直播過程當中，主播張大仙拿到了馬可波羅之後便大殺四方，連續拿下了好幾個MVP，充分展現了自己的實力。

作為王者榮耀人氣最厲害的一個主播，張大仙在努力的提高自己直播內容的同時，也不忘了提升自己的技術，將自己打造成一個全方位的主播，不僅直播風格幽默詼諧，而且能夠通過自己的技術，來給玩家們帶來一些真正有用的東西。雖然張大仙僅僅是一個主播，他也在用自己的行動，盡自己最大的可能來幫助他人，這一點是值得其他主播與之學習的！
