HSL：酒館戰旗巔峰戰，A組光牙黑馬第一，會長解說慘遭網友痛批

天天要聞
2019-12-22

最近虎牙HSL比賽正在火熱進行中，作為首場競賽爐石傳說比賽備受大家關注，在今明兩天舉辦的爐石線下決賽中，有不少選手為我們展示了非常精彩的瞬間，特別是剛剛結束的A組比賽的過程中，光牙如一匹黑馬殺出重圍，成功獲得小組賽第一的好成績，而本次作為B組的選手，沒有比賽的會長安德羅妮作為解說，卻在遊戲中遭到網友無限吐槽，這究竟是怎麼一回事呢？

其實從這次A組的名單中就可以看出整個A小組的人員都是非常有實力的，其中瓦赫拉、太極劍等等都是大家好看的奪冠熱門選手。特別是此前在直播中就為大家帶來的精彩的遊戲技巧，而在這一場小組賽中他們也為我們帶來了非常精彩的表現。本次作為B組的參賽人員，會長安德羅妮以及Kimmy作為解說為大家帶來精彩的講解。

《酒館戰旗》的積分規則是按照梯級積分制展開的，第一名選手獲得最高9分的積分，第二名選手獲得7分積分，別的選手積分依次遞減的趨勢，最低分就是遊戲中1分。同時遊戲中獲得第一名的玩家還可以在滿血狀態下額外獲得2分，並且掉血少的時候也會額外獲得積分。也就是說在眾多高玩對決的情況下，只有保持2場排名靠前的位置，基本上就可以穩拿晉級的資格了。

比賽一共進行三個回合，A組八位選手依次跟對手進行對決，最終第一的玩家獲得9積分。但是同時也意味著獲得第一的玩家需要與賽場上至少與3位玩家PK，如果不是實力非凡的玩家幾乎很難做到，更別說眾多高玩比賽的情況，一個回合就需要連續3場比賽同時取勝，基本上並不是一件容易的事情。而在這個過程中就看到一位選手脫穎而出，他就是光牙。在第一個回合首場比賽中，太極劍就遇見了光牙。兩個實力不凡的選手之間展開較量，比賽打的精彩紛呈，但是由於光牙實力太強大，因此第一個回合下來，太極劍就被直接淘汰了。

隨後經過幾場激烈的比拼，光牙以及變種屍新娘進入到第一回合決賽環節，兩人實力相當幾乎打成不分上下的局面。圍觀的網友也直接感嘆高手對決看的太刺激了，兩人都太強了。但是最終光牙技高一籌，一手惡魔牌直接刀掉對手獲得第一回合第一名。

然而就在這時，會長的解說卻遭到了不少網友的質疑。因為會長在遊戲中似乎吐槽了瓦赫拉的出牌，因此遭到不少網友吐槽，還有不少網友表示這組解說就是一個噴子組組合。

雖然第二個回合光牙並沒有獲得第一，但是他也以不錯的成績進入前三。之後開始的第三個回合，他再一次展現出了自己的高水平，進入到第三回合的決賽中，並且與未綿展開了激烈的爭奪賽，此時不管光牙是否獲得第三回合的第一，基本上A組第一的名次已經在他手中了。

經過激烈的對戰，果然光牙脫穎而出，成為A組第一併且順利進入到個人賽的決賽中。激烈的戰況還在繼續進行中，想了解更多虎牙HSL賽況，鎖定虎牙平台21日至22日進行的爐石線下決賽的精彩比賽。
