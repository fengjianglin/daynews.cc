騙老玩家幣的罪魁禍首！這些街機遊戲的開場動畫，你還記得幾個？

2019-12-16 3,222條評論 4,451次閱讀 262人點贊

提到當年街機遊戲，我們的思緒也不由得回到了童年的時候。其實我們起初並不知道哪些遊戲好玩，而真正吸引我們投下那神聖而又寶貴的遊戲幣的關鍵原因，除了那台機器旁邊圍繞著許多玩家之外，應該就是它們那閃著一行大大的「INSERT COIN」的開場畫面了。

因為幾乎每款街機遊戲為了吸引玩家的注意力，都會在沒有人投幣時出現的過場動畫上面下功夫，像《三國志》這類遊戲的開場畫面都是對劇情背景的介紹，而《拳皇》卻是將各個角色都走馬燈似的過一遍，《合金彈頭》更是用極具張力的畫面為我們刻畫出了一輛坦克。那麼說起當年那些經典街機遊戲的開場畫面，自稱街機遊戲老玩家的各位又記得多少呢？

一、《街頭霸王》

《街頭霸王》是由日本CAPCOM公司於1987年首次推出的格鬥類遊戲，也是後續所有格鬥類遊戲的鼻祖。我們現在熟知的許多格鬥遊戲的基本概念都是由《街頭霸王》首次提出並確立的，像輕重拳、必殺技、防禦、血條等等，可以說沒有《街頭霸王》，就沒有後續格鬥遊戲界的大紅大紫。

而初代街霸的開場畫面其實相當簡陋，只是在屏幕上出現了一堵有著「CAPCOM」字樣噴漆的牆，之後這面牆從後面被一個綠色頭髮的男子一拳轟開，之後該男子轉身露出了印在衣服後面的大字「STREET FIGHTER（街頭霸王）」，這也說明了CAPCOM當時其實並沒有對這款遊戲寄予太大希望。

二、《落日騎士》

其實這款遊戲在我小的時候經常被叫做《西部牛仔》，因為它主要還是為玩家們展現了美國西部的四個傳奇牛仔的故事。而我們所要扮演的角色，就是操控著幾個牛仔，去完成一個個懲惡揚善的賞金任務。

這款遊戲當時吸引我的點確實有很大一部分原因是在它的開場動畫上，經典的西部牛仔之間的決鬥場景，再加上一個個分鏡的切換，瞬間就能讓我們彷彿置身於當時緊張感十足的現場。

緊接著就是遊戲中的四位出場角色的特寫了，最後在落日餘暉下四人騎著馬緩緩遠去的背影，立即就能激發出我們中國人骨子裡的那種「俠者之風」。然後，一枚硬幣就從投幣口擲了下去……

三、《拳皇97》

作為格鬥遊戲的後起之秀，《拳皇》在《街頭霸王》的基礎上自然要取其精華，去其糟粕。在吸收了街霸一系列令人驚嘆的基本操作之後，《拳皇》自然不會像初代《街頭霸王》那樣，在開頭動畫上為人詬病。

尤其是真正打響了《拳皇》這一招牌的經典之作《拳皇97》，雖然他也是系列迄今為止BUG最多的一代，但是在玩家們的心中它依然是《拳皇》中的經典。而《拳皇97》能受到玩家們如此青睞的原因除了它暢快的打擊感以及風格各異的角色之外，其實就是它的開場動畫了。

直到現在，許多《拳皇97》的老玩家在聽到開頭時那熟悉的BGM響起的一瞬間，就會不由得回憶起自己的童年時光，那個與小夥伴一起偷偷溜進街機廳的夏天。也正因如此，《拳皇97》的開場畫面也被許多玩家認為是整個《拳皇》系列中，最為經典，也最令人熱血沸騰的開場畫面了。

四、《三國戰紀》

說到這款由咱們自主開發的這款國產街機遊戲「扛把子」，《三國戰紀》留給我們印象最深的恐怕就是開場動畫中的那一聲「三國戰紀」了。其實隨著遊戲版本的不斷變遷，《三國戰紀》的開場動畫也一直在不斷更新。

像遊戲的早期版本，如115、117當中，開場畫面的最後都是隱藏角色諸葛亮，但是在之後的《三國戰記·風雲再起》中，最終BOSS曹操則佔據了諸葛亮和關羽的「C位」。但是無論版本怎樣變遷，相信大部分玩家的印象肯定還停留在關二爺那迎面劈來的一刀吧！

五、《戰國之刃2》

無論玩家們如何辯解，我相信吸引各位站在《戰國之刃2》的機器前面的除了從一代過來的老玩家之外，更多的是被遊戲開場動畫中出現的小夜吸引過來的吧。

相信在看到開場動畫的時候，幾乎沒有人會將這位美女與一代中的「假小子」小夜扯上關係，還以為是本作中新增加的女性角色呢。沒想到啊，女大十八變，真的不是吹的！

好啦，以上就是小編今天為大家帶來的全部內容了，如果大家還知道哪些令你們印象深刻的街機遊戲開場畫面，歡迎各位來評論區留言，大家一起討論哦，謝謝大家！
