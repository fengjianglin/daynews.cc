明日方舟初雪新皮膚公開 路人女孩恩雅 看點全在新地圖和新敵人

天天要聞
2019-12-20

明日方舟有關初雪的新皮膚在近日正式透露和公開，這次的新皮膚看點蠻多的，除了作為核心的路人女孩恩雅之外，這次情報的看點全在新地圖和新敵人上，一起來看看吧。

本次獲得新皮膚的是初雪，設定上是薩維爾衣匠為初雪量身定做的服飾，開斯特之霜。全套純手工打造，完美貼合體型，增加了部分細節以最大程度貼合訂衣者的生活習慣。從這方面設定來看，這衣服真的是超級的昂貴，一般人是整不起這種私人訂製還是純手工待遇的，對此初雪還表示這裡沒有初雪，只有一個叫恩雅的普通女孩。還真就是普通女孩。

立繪形象方面沒啥好說的，初雪的服飾特色就是穿得賊厚實，精二之後可以看到一些比較媚宅的因素和畫面，不過立繪整體比較糊，這次的新皮膚對於立繪糊的問題有很大的改動，但整體還是超級厚的那種設定。

SD小人方面非常的萌，讓人很滿意，互動能力極強，對於喜歡初雪的刀客塔來說值得考慮一下，不知道後續會不會有銀老闆的系列皮膚，因為目前不知道這款皮膚是成套的系列還是單品，如果成套的話，那麼銀老闆的新皮膚也算是有盼頭了。

但這次皮膚的核心主要看點不再皮膚上，反而在這次透露的第六章主線的新地圖上，這個地圖，看看這位置，明顯是能讓小火龍和送葬人爽歪歪的地形和高台位，整兩個重裝幹員就能達到一夫當關，萬夫莫開的效果。

同時還有透露的新敵人單位，這種帶著兜帽和披風，同時持刀的地方單位，具體效果和設定是什麼目前還未知，但明顯目前的新敵人包括之前透露的冰狗，都是極地類型的地方單位，估計可能會有所謂的減慢攻速的效果。
