英雄聯盟玩家最開心的莫過於和朋友一起開黑了，特別是網吧5連坐的時候那是最快樂的時候，比五黑更加快樂的就是玩5排究極臟套路，但是有時候卻不知道有什麼套路可以使用，大魔王今天就給大家推薦一套超級強力，究極噁心的5排騷套路——趕屍流，該套路的核心就在於以人數為優勢一舉將對面擊潰，到底要怎麼玩呢？一起來看一下吧。

首先英雄選擇，上單約里克，中單塞恩，下路大頭、火女，打野翠神，首先上單約里克是一位對線超強的英雄，幾乎沒有他打不過的人，選他的原因就在於他玩遊戲還拖家帶口的， 老婆兒子加起來就有5個了，同時也是一位拆塔能力非常強的英雄，而中單賽恩在死亡之後還能夠詐屍，擁有很高的攻速和攻擊力，送死流賽恩在一個版本令人聞風喪膽。翠神是一位比較偏向於輔助的打野，選擇他的主要原因就是在與大招能夠召喚小菊，

小菊還能夠提供很好的控制效果，下路大頭最有特點的就是擺放炮台，加上大招的加強版Q，能夠擺放4個，火女就不用多說了吧，和翠神是一樣的能夠召喚提伯斯，整套整容不管是坦度還是傷害都是非常足夠，而且這套陣容的恐怖之處在與推塔，五個人集合能夠在短時間就將對面的防禦塔給強拆了。等全體到了11級的時候就可以中路集合，先是賽恩或者是小菊、提伯斯去抗塔，千萬不能被讓約里克的小鬼過去。因為約里克的小鬼的傷害還是非常的高的。

不過強拆的話，還是推薦11級的時候再去，反正這些英雄的對線能力都很強的，到了11級那就是誰都不能擋住的，直接形成以多打少，看一下到底是幾打幾，約里克5個，翠神、火女個一個，大頭4個，賽恩兩條命也算一個，那麼就是5+12個人，以人口優勢擊敗敵人，甚至可以強拆15分鐘就可以直接一波推了，甚至有玩家哭訴：5個怎麼打17個啊，不過可惜的是鐵男被改了，不讓賽恩換鐵男將會好好許多，搞一條小龍就是一頓亂噴。
