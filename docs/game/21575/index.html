王者榮耀：全新模式來襲，英雄技能進化，6個球的東皇你見過嗎？

2019-12-19 1,277條評論 2,647次閱讀 1601人點贊

繼推出新模式王者模擬戰後，王者榮耀下個模式的玩法如何備受玩家期待，目前官方僅預告了下個大版本更新後無限亂斗會有一波調整，並未提及新模式的消息，不過通過知情玩家透露得知，目前體驗服已經加入了一個全新模式，該模式的一大特色是英雄技能進化，一起來看看玩法如何吧？

據悉新模式當前只有28個英雄擁有強化技能，這些英雄的技能在原有效果的基礎上進行升級加強，以下圖中三位英雄的技能為例，其中呂布的1技能橫掃面積擴大1.5倍，預示著命中率提升，更容易打出真傷效果；諸葛亮大招元氣彈目標增加1個，等於是在同一時間自動打擊兩個敵人，數量增加後，敵人想要幫隊友擋子彈估計夠嗆；后羿分裂箭數量提升至5支，同時擴大了判定範圍，調整後一人站擼五人不是夢！

我們再來看看三個比較有趣的強化技能：

一、安琪拉的大招釋放後左右還會各出現一個熾熱光輝，且大招轉速加快，可以說面前的敵人想要通過天秀走位逃脫是不存在的，特別是在增加兩個混沌火種的情況下！（圖：纖歌帝）

二、白起的大招釋放後會出現刀刃密布的圓形刀陣，整體特效看起來像是放大版的天魔繚亂大招，釋放後對範圍敵人有效，稱得上是控敵利器。

三、東皇太一的1技能在正常模式中最多只能存在三個黑暗能量體，而新模式中經過強化後的能量體數量翻倍，達到了6個，不僅如此每個能量體丟出去後都可以對敵人造成暈眩效果。

可以看出新模式的英雄技能進化後，不僅對抗性變強，可玩性也變得更高了，小編還是很期待新模式能早點上線測試哈！那麼小夥伴們你覺得新玩法如何呢？歡迎留言討論！
