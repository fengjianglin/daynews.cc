明日之後最熱門的四種職業，機械工頗受歡迎，這個職業是土豪專屬

明日之後作為一款末日求生遊戲，自上線以來，就一直受到廣大玩家的好評。在遊戲中一共有10種職業，他們分別為：步槍兵，狙擊手，武士，病毒學家，機械工，護甲工，伐木工，采麻工，傢具工以及礦工。下面就來跟大家盤點一下當前版本最熱門的四種職業，其中這個職業是土豪專屬。

第一：狙擊手

大家都知道狙擊槍是遊戲中傷害最高的武器，不過這需要一定的金錢才能製造。所以大多數的狙擊手使用的幾乎都是鈔能力，這個職業也是土豪的專屬職業。當然也有玩家不甘心，也想玩這個職業，但事實證明，除非你是肝帝，不然平民玩家支撐不起這個職業的消耗。

第二：機械工

在遊戲中武器是影響玩家生存的一個重要因素，一把好的武器完全可以讓你的戰鬥事半功倍。所以機械工也應運而生了，在遊戲中這也是頗為熱門的職業。畢竟一把好的武器，不僅要屬性好而且特技也要好，這也是機械工要做的首要工作。

第三：病毒學家

新職業的強勢程度相信不用我過多的贅述，官方甚至還特意的給新職業做了一個特殊的武器，從這就能看出誰才是親兒子職業了吧？病毒學家不僅傷害不錯，最關鍵的是這個職業自帶治療效果，可以大大的提高玩家的生存率。

第四：伐木工

在遊戲中木材是最基礎的資源，不管你建造什麼東西，都離不開木材。所以在遊戲中木材的需求毫無疑問是巨大的，因此伐木工在遊戲中非常熱門。如果伐木工團結，甚至還能影響遊戲的經濟，這也是為什麼這個職業被稱作奸商的一個重要原因。
