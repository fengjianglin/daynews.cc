這個遊戲好玩嗎：高難度生存遊戲，冒險的旅途上總是充滿孤獨

摘要

《漫漫長夜》，了解的朋友一聽就會說，這遊戲不是早就出了么？你還吃什麼殘羹剩飯？確實，它第一次問世已經是五年前的事情了，但都只是EA階段而已，而在17年的8月1號，這款遊戲才算正式發售。

《漫漫長夜》，了解的朋友一聽就會說，這遊戲不是早就出了么？你還吃什麼殘羹剩飯？確實，它第一次問世已經是五年前的事情了，但都只是EA階段而已，而在17年的8月1號，這款遊戲才算正式發售。

《漫漫長夜》

在EA階段這款遊戲憑藉著生存挑戰模式，在求生遊戲愛好者的口中廣受好評，純粹的生存體驗也讓其與殭屍橫行的同類遊戲大相徑庭。在這裡只有探索、搜刮、製作、狩獵或者被狩獵，你永遠不知道哪裡才有食物、純水、火等生存必須用品，也永遠不會知道自己會在哪一個夜晚一個人孤獨的死去。唯一陪伴你的只有分分鐘都想要你命的大地母親。接下來我將對這款《漫漫長夜》進行一個全方位的評測，請耐心的讀者們一定要看到最後哦。

劇情

即使是生存類遊戲的通病但也值得一試

配合上冰天雪地的世界，與場景美術上的刻畫，在玩遊戲時，這幅廣袤的地圖中，孤獨的氛圍感一直環繞著你，你能做的就是竭盡所能的活下去。可以說《漫漫長夜》在EA階段所表現出來的遊戲性與玩法確實有自己獨到與個性的一面。雖然生存模式如果你砸進去大量的時間度過了中前期，後期就只能養老了，這就讓其失去了遊玩的意義。但對這種生存遊戲而言，這個問題早已是這類遊戲的通病，所以《漫漫長夜》還是值得生存控與慢遊戲玩家去嘗試的。

暴風雪

而今天，《漫漫長夜》已經發售了一段時間，玩過的人自然有著其獨有的見解，但我是第一次接觸這款遊戲，還是想說一說自己的感受的。這款遊戲是由一家加拿大遊戲公司開發的，發布至今，steam平台的評價都是”特別好評”，遊戲的故事背景設定在一次空難之後，男主和女主倖存了下來，需要在冰天雪地的荒原求生故事。

選擇主角

純粹的求生模式

這個遊戲著重強調生存探索，求生模式的確是純生存的遊戲，遊戲中沒有地圖，物資也是隨機刷新的，你不知道往什麼方向走才會遇到物資點，也不知道自己會在哪個夜晚孤獨的死去。畫風則採用了油畫的風格，配合上絕美的風景，還有好聽的背景純音樂，給人一種無法忘懷的意境美，也正是這樣的意境，才更加襯托除了遊戲里的那種孤獨感。

野外生物

因為這個遊戲不能聯機，只有單機，陪伴著你的只有大地與雪天還有周圍所有的危險，面對這樣的環境，你能做的只有求生。沒有建造系統的你只能孤獨的在暴雪中前進，竭盡全力地生存下去，一覺有可能就被凍死了，所以要非常注意保暖。當遊戲玩到一定程度，你甚至會像我一樣開始走馬燈，那種時時刻刻圍繞在自己身邊的危機感會讓你回想起自己的一生。當你感受到絕望的時候，睡一覺看到天亮了，暴雪停了，太陽也出來了，周圍的一切又都充滿著生機，新的一天又開始了。不管黑夜多麼漫長，不管前路多麼坎坷，你總有自己的信念支撐著自己活下去。我感覺，這款遊戲的作者如果不去當動漫電影導演，實屬屈才，畢竟在遊戲里能體驗到如此沉浸的意境，我還是第一次。

黑夜

遊戲的缺點真的很多

說完了遊戲的基本情況，我們再來看看基本玩法和遊戲的一些優缺點。遊戲的存檔方式我會在後面吐槽，而對於不同的玩家會給出不同的難度和地圖。每次睡覺醒來，進入房間或者離開房間都會自動存檔，存檔方式無疑給遊戲帶來了更具艱難的挑戰性，如果你在荒郊野外缺乏材料又下起了暴雪，這時候如果死了那真是從頭開始。食物、物資、材料、暴雪都是隨機出現的，唯一能給玩家帶來提示的就是建築物了，可問題是每次看到建築物的時候都不一定能有足夠的體力，在寒風暴雪中活著走到建築物內。

黃昏

所以每次看到房屋時都得預估一下風險，這個房子值不值得為之冒險前去。和絕大多數求生類遊戲一樣，求生模式的後期也讓玩家略感無味，因為沒有建造功能，你不能養老，即便是你獲得了所有的武器和道具，你的小名也時時刻刻命懸一線，一個不留神什麼東西都沒了，又得回檔重來。唯一可供玩家選擇的只有更恐怖的難度和地圖了，而且不要期待這個遊戲會添加什麼什麼新內容。

建築物內

劇情模式引人深思

雖然劇情模式的加入讓我對這個冰天雪地的世界產生了好奇，可在一個又一個漫漫長夜之後，這款遊戲的單人劇情遠沒有生存模式來的優秀。分章節式的劇情闡述，每個章節會給你一副新地圖，劇情整體故事還行，過場人物對話也都有，加持上本身就充滿遊戲性的玩法，經我這麼一說好像也沒什麼好擔心的地方，即使劇情遊戲性兩者兼備，但實際上玩家的遊玩感受並不是你說有就行了，”不要你覺得，我要我覺得”。遊戲整體應該怎麼去銜接，怎麼把劇情闡述與玩法相互結合，遊戲流程安排的平順性才是最影響玩家能不能玩下去的重要原因。

狩獵

而恰恰在銜接這一點上，《漫漫長夜》出現了斷崖式的遊戲體驗，如果你之前沒有玩過這款遊戲，我相信你永遠會被卡在遊戲的第一章節。後面的章節暫且不說，重點就放在第一章節，講述劇情故事開始，事故、墜機、掉落無人區、一個人求生開始基本教學，既有有指引也有有玩法介紹，再加上你對荒野的好奇心與對故事的好奇，以及整個遊戲的氛圍營造，一切都是那麼的美好，但前提是你沒有遇到比如瞬移、無法製作、儲物箱的東西無法放到背包等各種小bug。

讓玩家吐槽致死的自動存檔

而劇情剛剛推進到第一個鎮子，見到第一個NPC的時候，什麼孤島求生，什麼北極撿垃圾，氛圍感全無。一個煩人的老太婆一會讓你幫她把家裡的燃料箱補滿，一會讓你把她家裡的冰箱放滿食物，並且這兩項任務都足夠你忙活上好一段時間，非常讓人不爽。更有甚者的是遊戲採用自動存檔，如果你把某些重要道具給拆解了，這遊戲基本廢了要重玩或者重新讀檔了，但檢查點的存檔會距離你一個很遠的位置。

挑戰模式

然而這都不叫事，遊戲里的人物移動更是能讓你崩潰。就算你全精力，各項指數都封頂的情況下，按下加速跑，跑的快不快且不說，而是在地圖上你會發現很多時候你都需要來回跑圖，10分鐘遊戲9分鐘跑圖，運氣不好說不定就被狗熊一巴掌給拍死了，接著就是重複重複再重複的跑圖。我不管你這遊戲的遊戲性有多爆炸，就跑圖這裡，已經是足夠無聊到可以關閉遊戲了。

選擇地圖

劇情模式毫無代入感

接著就是之前還在緊張的求生，鋪墊劇情，解開謎團，這下好了，現在一個任務讓你老老實實刷刷刷，先前的恐懼與緊張感蕩然無存，在一個無人的小島上自己都吃不飽穿不暖的，一個NPC找你要吃要喝，分量相當多而且必須完成才能往下推進，講真的這孩子一定是我前世的情人，上輩子欠的。這樣的任務設計對於一個主講故事，求生，需要氛圍營造的遊戲來說實在是有些脫戲了。還有玩家表示，劇情根本沒有什麼代入感，玩劇情和玩生存根本就是兩款不同的遊戲，一旦玩家進入了生存階段，劇情所打造的意境美通通消失不見。

到此為止也只是說了這些僅僅只是針對於遊戲的劇情模式而言，同理可以反推，steam上有太多EA階段點子好到爆炸的遊戲，極具特色廣受好評，但是在正式版之後掉鏈子的也是數都數不過來了，《漫漫長夜》就是其中典型的例子。

物品是有耐久的

EA測試階段不代表遊戲最終品質

既有想法又有玩法的遊戲並不代表就是好遊戲，最終展現給玩家的具體流程安排怎麼融入，結合與銜接，如果沒有足夠強大的製作團隊是很難做好的。不僅針對於《漫漫長夜》而言，很多遊戲如果EA階段就已經是完整的遊戲售賣，後續再不斷更新強化的話，也許我會給出一個很高的評價，但如果只是畫餅，無限測試，最終成品與EA時的玩法衝突較大的話就很難給予好評了。

夾克

回到《漫漫長夜》，雖然話是這麼說，但是拋掉體驗並不是很好的劇情模式，生存挑戰模式還是很值得嘗試的，當然只是針對於喜歡生存或者慢節奏遊戲愛好者等少數玩家而言，你要是想通過遊戲自身的帶動，不自己給自己加戲的話，想體驗什麼荒野求生，北極孤島大冒險什麼的，還是洗洗睡吧！就算你強行想玩，那我還是建議你等他把遊戲中的中文翻譯bug修復一下再說吧。總的來說這款主打探索的生存遊戲不是不值得玩，而是要忍受它所有的缺點再玩，沒有過多的玩法確實是有點可惜，人終究是害怕孤獨的生物，不管是只有單人的沙盤模式還是較為雞肋的挑戰模式，它們都缺乏一個長久吸引玩家的理由。

物品欄

優點：

BGM音質上乘，與情境和環境結合恰到好處

寫實的生存系統，細節非常用心

整體美術效果很美

缺點：

地圖建模有些粗糙，影響人物動作

兩種玩法結合不恰當，BUG較多
