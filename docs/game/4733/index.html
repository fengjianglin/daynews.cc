《星球大戰：前線2》慶典版公布 售價約282元

2019-12-05 0條評論 13次閱讀 0人點贊

摘要

為了紀念《星球大戰：前線2》更新兩周年以及即將上映的《星戰9》，新的《星球大戰：前線2》慶典版將於12月5日（當地時間）在PS4，Xbox

為了紀念《星球大戰：前線2》更新兩周年以及即將上映的《星戰9》，新的《星球大戰：前線2》慶典版將於12月5日（當地時間）在PS4，Xbox One和PC平台上以數字版形式發布。此版本包含完整的定製內容集，這些內容可通過從《星球大戰：前線2》的原始發行版進行遊戲內購買而獲得，包括《星戰9》電影中的物品。

EA官方介紹：

《星球大戰：前線2》慶典版零售價為39.99美元（約合人民幣282元），其中包含：

·遊戲本體

·包括所有過去和即將發布的免費遊戲更新

·超過25個英雄皮膚：6部傳奇皮膚，以及受《星戰9》啟發的雷伊、芬恩和凱洛·倫的皮膚各一款，12月17日放出

·超過125款單位和補給物皮膚

·超過100個英雄和暴風兵表情和語音

·超過70個英雄和暴風兵的勝利姿勢。

已經擁有《星球大戰：前線2》的玩家可以將其當前版本的遊戲以折扣價24.99美元（約合人民幣176元）升級到慶典版，並獲得上面列出的所有自定義物品。

《星球大戰：前線2》慶典版將於12月5日登陸PS4、Xbox One和PC平台，僅支持數字版。
