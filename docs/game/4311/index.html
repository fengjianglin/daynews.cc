殘血才是完全體，遊戲中的背水佬角色，不是你死就是我亡

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

許多遊戲的數值中，往往會出現這樣一種情況，當血量低於多少的時候，角色的攻擊力會大幅度提升。沒錯，這就是傳說中的背水佬！！！高傷害的同時，意味著自己的血線也處在一個非常不安全的狀態，這種懸崖走鋼絲的感覺，就如同伊甸園中那條蛇的甜言蜜語一般，令人沉醉。

高傷害的同時，意味著自己的血線也處在一個非常不安全的狀態，這種懸崖走鋼絲的感覺，就如同伊甸園中那條蛇的甜言蜜語一般，令人沉醉。

《英雄聯盟》——蠻王

蠻王就是一個典型的背水佬。

在高端局和比賽中，他雖然沒有出場的機會。不過相信在許多玩家還是新手的時候，蠻王都曾經陪伴他們走過了許多場遊戲。

蠻王上手非常簡單，技能幾乎沒有什麼釋放難度，平A就完事兒了。

最爽的還是在蠻王大招開啟的五秒鐘之內，只要敵方沒有控制技能，皆是人擋殺人，佛擋殺佛，俗稱五秒真男人。

砍完之後，一口溫酒下肚，拂手離去，留下滿地屍體，像極了古代的俠客。

當然，大部分現實情況都是蠻王在開大之後被控到死。

《碧藍幻想》——佐伊

《碧藍幻想》中有著許多強力角色，也有著許多立繪精美的角色，但能夠將兩者完美合二為一的，暗水著佐伊絕對是其中之一。

此前在國外攻略網站gamewith的評分中，泳裝佐伊是唯一一個十分。

佐伊的技能組完美詮釋了什麼叫背水一戰，自身HP越少，傷害倍率就越高，最高可達16倍率的傷害。

擔心血量降不下來，發揮不了技能的效果?

不用擔心，佐伊還有另外一個技能，一鍵1血怕不怕？還是我方全體1血，瞬即將輸出拉滿，不是你死，就是我亡。

而佐伊在來到《影之詩》的是世界後，也是保留了一鍵背水的效果，在登場後直接將主戰者的血量變為1點。對於龍族玩家來說，最大的幸福就是佐伊三連了吧。

《最終幻想系列》、《怪物獵人》——貝希摩斯

貝希摩斯從《最終幻想2》開始，便是《最終幻想》系列的標誌性怪物之一。

在《最終幻想14》與《怪物獵人世界》的聯動中，這個巨獸也讓新大陸的獵人們狠狠地吃了苦頭。

難度之高，甚至超越了世界中原本的歷戰王。

招式什麼的先不說，普通貝希摩斯的血量就高達35000點，極貝血量則是51800點，除了古代鹿首精之外，是血量最高的怪物，包括了《冰原》部分。

什麼冥燈龍，天地煌啼龍，還是什麼最終Boss呦，丟死人了。

當然，貝爺還有一個技能，在《怪物獵人世界》中也是獨一無二的，這就黃道隕石，一個全地圖的秒殺技。

是「真」全地圖，即使玩家用回家玉回到營地，仍舊會受到影響。只能躲到地圖上的隕石背後，或者使用龍騎跳抵擋傷害。

至於為什麼說貝爺也是一個「背水佬」呢？

從某種意義上來說，貝爺是鎖血的，並非是被獵人們討伐。在臨死之前，貝爺必定會放一次黃道隕石，所以說，貝爺其實是被自己砸死的噠！

不知各位還認識哪些背水佬角色呢？不妨討論一下吧。
