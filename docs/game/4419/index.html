王者榮耀聖誕不出皮膚出星元，價格超史詩，玩家：免費星元不香嗎？

2019-12-07 0條評論 19次閱讀 0人點贊

摘要

相信每到年末，大家都能明顯感覺到新年的氣息，於是為了迎合聖誕節，雖然沒有聖誕皮膚，但出了貓狗日記的聖誕星元，不過玩家並不能滿意此次的活動，這是怎麼回事呢？在過去的兩年，每逢聖誕節，官方就會出配套的聖誕節皮膚，其中貂蟬的聖誕戀歌和蔡文姬的奇蹟聖誕更是廣受好評。

相信每到年末，大家都能明顯感覺到新年的氣息，於是為了迎合聖誕節，雖然沒有聖誕皮膚，但出了貓狗日記的聖誕星元，不過玩家並不能滿意此次的活動，這是怎麼回事呢？

在過去的兩年，每逢聖誕節，官方就會出配套的聖誕節皮膚，其中貂蟬的聖誕戀歌和蔡文姬的奇蹟聖誕更是廣受好評。但今年的運營和美術團隊似乎睡著了，並不打算出新的聖誕皮膚，而是又出了一款貓狗日記皮膚的星元。

自從大喬孫策這套皮膚出來之後，每逢重要節日總出他們的星元，之前的小紅帽和大灰狼星元、七夕的天使星元到現在的聖誕節星元，但玩家們的不滿情緒越來越大，一方面是因為貓狗日記皮膚本身質量就一般，尤其是大喬的建模和原皮根本沒法比；另一方面則是因為總出星元大家都已經審美疲勞了，希望能出一些新的皮膚。

我們可以來看看這次的星元效果，目前聖誕星元中貓和狗建模曝光，貓貓和狗狗都穿上了充滿聖誕氣息的服裝，狗的品種也從薩摩耶變成了雪橇犬，雖然看起來憨憨了一點，但能讓玩家心情愉悅。星元的出現讓眾多有收集癖的玩家很難受，一些星元需要點券才能購買，目前貓狗星元加在一起的價值甚至超過了一款傳說皮膚，果然是個大坑。
