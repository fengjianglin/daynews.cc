林煒翔在奪冠之後，眾人的目光開始聚焦於這個能在世界賽決賽中不死的傳奇AD。我們大家才驀然發現，這個人原來這麼皮啊！敢在巴黎公然嘲諷歐洲本土戰隊，敢在直播中的公然口嗨自己的隊友，敢在比賽中秀各種各樣的騷操作。就在最近，4AM的小醒目在直播中跟林煒翔峽谷之巔雙排，兩人全程騷話不斷，各種嘲諷，陰陽怪氣。這一盤林煒翔拿出了女警，小醒目掏出了瞎子，從前期就取得了巨大優勢。然後兩人就開始各種越塔，各種浪，險些被對面抓住機會扳回局勢。

在頻頻被對面控住殺掉之後，小醒目開始了：「你怎麼老被定啊？這就是冠軍AD么？」林煒翔作為大陰陽師中的一員，當然得把鍋甩出去：「別人玩的好，我有什麼辦法？」就連在一旁觀戰的韋神也吐槽道：「這LWX開始夢遊了」。直播間觀眾聽到韋神嘲諷LWX，就有觀眾搞事情說道：「人家冠軍AD,總決賽零陣亡，你呢，反向Q。」韋神看到後氣壞了！不過畢竟是小醒目的直播，醒目趕緊轉移話題，帶著隊友抓緊推塔拿資源。

在翔子哥開始抱團之後，對面基本沒啥反抗能力就輸掉了比賽。遊戲結束之後翔子哥打開數據面板，自己的傷害赫然遙遙領先，跟小醒目說：「看看你這低賤的輸出，再看看我的。」小醒目非常真實：「我MVP啊，你傷害高咋了，反正我MVP啊！」翔子哥當即反唇相譏：「你玩個打野MVP不是正常的么？你打野還不是MVP還打啥野啊？」大陰陽師名不虛傳，一張嘴小天遠在拉斯維加斯也得躺槍。
