《爐石傳說》：“酒館戰棋”MMR調整，5000分選手僅占玩家數3.3%

天天要聞
2019-12-06

摘要

隨著今天凌晨《爐石傳說》更新補丁的上線，“酒館戰棋”的天梯環境有發生了巨大的改變。不僅新增和回歸了五位的英雄，還有兩個全新的隨從加入酒館，相當多隨從的數值也有所調整。除了隨從和英雄的更新之外，官方還專門發了一個藍貼講解了關於MMR積分的調整。

隨著今天凌晨《爐石傳說》更新補丁的上線，「酒館戰棋」的天梯環境有發生了巨大的改變。不僅新增和回歸了五位的英雄，還有兩個全新的隨從加入酒館，相當多隨從的數值也有所調整。

除了隨從和英雄的更新之外，官方還專門發了一個藍貼講解了關於MMR積分的調整。

據藍貼稱，更新後當對局中其他玩家的MMR明顯低於自己時，吃雞後獲得的積分會被乘上一個係數。這個係數由MMR差距而定，有可能高達50%，也有可能僅為90%。

這個係數改動主要適用於那些MMR超過絕大部分甚至更多的玩家。如果你對手的MMR與你相當，就不會受到此次調整影響。因此大部分玩家不必擔心受此影響，減緩上分步伐。

有意思的是在這個藍貼中，數據分析師Tian特別公布了一個數據。那就是目前「酒館戰棋」中5000分以上的玩家可以排在96.7%玩家之上，而4300分的玩家可以排在80%的玩家之上。

看到這個數據後，大部分玩家都非常吃驚，表示之前看到《爐石傳說》論壇社區的曬分貼，以及各大主播直播間的「軍師」，原以為大部分玩家的平均分都超過了6000分甚至7000分，沒想到大部分「酒館戰棋」玩家都還在5000分以下掙扎。

仔細一想，就連多次在線下比賽獲獎的Wtybill，也會在直播「酒館戰棋」時被彈幕噴成「菜Bill」。甚至為了和諧彈幕氛圍，那個愛笑的針人都把直播間設置成了付費粉絲才能發言。

也許這就是這就叫網路上重拳出擊，遊戲中唯唯諾諾嗎？建議設計師以後多公布MMR分布統計。

其實這種現象在《爐石傳說》傳統對局中也非常明顯，就算你是線下比賽冠軍，已經打上了當月傳說段位，在直播時不也得虛心接受彈幕的批評。要知道在暴雪曾經公布的數據中說過，遊戲中僅有0.5%的玩家上過傳說，相信這些彈幕中幾乎沒有那0.5%的傳說玩家。

看了數據分析師公布的數據，曾經為自己只有5000分而感到卑微的小編，終於可以昂首挺胸地告訴別人「我超越了96.7%的玩家」。
