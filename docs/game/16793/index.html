國服韓信揭秘總決賽，MVP不給一諾因素很多，老帥獲獎感言有問題

2019-12-16 1,089條評論 4,797次閱讀 2597人點贊

眾所周知，AG超玩會以4比1的大比分戰勝了QGhappy，獲得了KPL秋季賽冠軍，這也是AG超玩會王者榮耀分部獲得的第一個冠軍。雖然說奪冠了粉絲們都很高興，但卻在MVP得主的這個問題上大家產生了分歧，讓我們來看看吧！

看了總進決賽的網友們，一定感覺出場的這五位選手其實表現都可以，但要說到最出色的，可能就是中路老帥和邊路一諾了，而當時在公布MVP是誰的時候，台下觀眾們喊的最大聲的也是一諾和老帥，所以說當時官方應該也很糾結。

當時八卦君是在國服韓信韓跑跑的直播間觀看的總決賽。因為每次有AG的比賽，韓跑跑都會解說，而且講解的很詳細，還會說出一些粉絲們不知道的事情，但是在當時韓跑跑也沉默不語了，可能他也不確定官方會給誰，自己的胡亂猜測會引起直播間的節奏。不過在主持人公布MVP是老帥的時候，韓跑跑拍手叫好。

在高興之餘韓跑跑表示「我是真的為了老帥高興，這個MVP給的太情懷了！其實一諾還年輕，但是老帥是真的老了，他打完這個賽季很可能就退役了，不過一諾還有機會。若是這個MVP給到了一諾，那麼老帥幾乎就沒有機會了！」

其實我們細想這個MVP確實摻雜的因素太多了，論實力其實老帥表現的也還可以，這四場勝局中，老帥一共獲得了兩次MVP，一諾獲得了一次（勝四局比賽，給出了三次MVP，最後一局沒有MVP），而且老帥也是AG超玩會元老級別的人物，再加上韓跑跑說的年齡因素，所以老帥獲得這個MVP也是理所應當的。

獲得這個MVP後老帥再次哭泣了，他表示「自己本身就是一個沒有天賦型的選手，我因為打這個遊戲太早了，所以才能成為職業選手，這一路走來確實挺不容易的。」可能是太激動了，老帥現場發言真的是語無倫次。聽了老帥的話語後，韓跑跑表示不夠官方，不夠嚴謹，甚至還自己模擬了一套。

韓跑跑表示，要是我一定會說「感謝AG超玩會，感謝《王者榮耀》給了我這個機會，感謝主辦方，特別感謝一路走來支持我們的粉絲們，沒有你們就沒有我的今天，謝謝！」但是網友批評韓跑跑「太過官方就讓人感覺是故意的了，要不然說你打不了職業呢！」（韓跑跑也是為了娛樂下直播間氣氛，只是調侃一下而已，網友們別當真）

好啦！以上便是關於國服韓信韓跑跑對於總決賽MVP的評價，那麼網友們，你們認為這個MVP給到老帥合理嗎？你們認為一諾還有機會獲得MVP嗎？
