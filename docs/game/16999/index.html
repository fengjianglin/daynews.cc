紅玉詛咒：暗黑城冷飯地下城，怪物更強，搬磚副本。
暴風之路：冷飯地下城，通過裂縫前往機械牛啊什麼的，也有新的boss，升級用副本。

裝備打造可互相轉移，展示中為+10 A甲→+10B甲。
