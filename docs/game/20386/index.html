休閑好去處《俠盜獵車OL》新增12款經典街機遊戲

2019-12-16 106條評論 1,094次閱讀 135人點贊

隨著新的搶劫任務「鑽石賭場搶劫」的上線，Rockstar為《俠盜獵車OL》新增加了即便是隊友失敗也可繼續的搶劫任務。另外，本次搶劫任務的基地街機廳也得到了強化。Rockstar為《俠盜獵車OL》的街機廳新增了12款復古風格街機遊戲，嘗試著用街機廳賺取更多資金吧。

《GTAOL》新增12款街機遊戲預告片：

本次新增加的12款街機遊戲包含：射擊、動作、競速等類型，詳細介紹如下：

《荒地復仇 II》：掠奪者、攔路強盜、小偷和稽查人員入侵了和平的邊境。明智地使用武器，除掉威脅您的生活方式的亡命之徒以及不法分子。

《毀滅巫師》：邪惡的巫師把格羅格綁架到恐怖的要塞之中。現在，索格必須穿越魔法森林、陰森沼澤和腐爛迷宮，用他的劍和路上找到的神奇藥水和可怕的敵人戰鬥，使出渾身解數救出落難兄弟。

《飆風追戰：街頭狂飆》：在這場競速，您將駕駛豪華跑車，穿越中心地帶以及美國最繁榮的大都會。把音量開到最大聲，突破各式障礙物，小心別撞到行人和高速公路上的野獸。

《飆風追戰：胯下火箭》：提升您超級機車的性能，狂飆上路，在城市和郊區與時間賽跑，沿途小心障礙物，別摔個狗吃屎！

《飆風追戰：卡車飆速》：狠狠踩下油門，和狗狗夥伴一邊閃避開太慢的駕駛和四處遊盪的北美野人，一邊狂飆穿越美國大陸，打破紀錄、光宗耀祖。

《太空猴 3：香蕉壞壞》：邪惡的丹克博士劫持了厄瓜多第五象限的所有貨運太空飛船，還利用他的變形雷射把香蕉變邪惡。當唯一愛過的事物與自己為敵，該如何是好？而今太空猴必須賭上演化的未來，與邪惡香蕉戰鬥並擊敗丹克博士。

《閃亮芥末小貓爪》：機械泡泡糖公主的新閨密閃亮芥末小貓，將她在動畫中古靈精怪的魔力帶到這個經典夾娃娃機，只需要一些技巧和運氣，就能得到夢幻的獎勵。

預言家：《娜札的預言》：永恆的預言者娜札女士善於靈界交流，準備好窺探您的未來和運勢。是否有巨大的財富等著您？愛情是否即將來臨？抑或命運之手會將您推入一場原可以輕易避免的異常事故中，讓您英年早逝？來一探究竟吧。

《愛情教授》：究竟是死黨還是柏拉圖式的愛情，「愛情教授」會診斷您和您的伴侶，看兩位的關係是離婚、曖昧中、炮友，還是搖滾芭樂歌才會出現的愛情。

《信仰戰士》（八位元經典遊戲）：這款經典街機遊戲新穎又錯綜複雜的玩法，讓許多後繼者爭相模仿。用超級飛天紅色方塊拯救綠色點點，確保您的不滅與來生的一席之地。

《猴子天堂》（八位元經典遊戲）：這款人猿模擬器在 80 年代風靡各街機鋪。用紅色方塊猴子擺盪至綠色點點，向世界展示究竟是誰站在食物鏈之巔。

《侵入者》（八位元經典遊戲）：在太空的黑暗深處，潛藏著威脅地球的邪惡勢力。射爆神秘紅色方塊內的綠色點點，拯救全人類。

《俠盜獵車OL》「鑽石賭場搶劫」現已正式上線。
