廠長走了很傷心？別忘了當初pdd，都是被你逼退役的

2019-12-17 3,455條評論 4,602次閱讀 1087人點贊

編者語：今天整個電競圈被廠長退役的消息刷屏，無數老玩家紛紛感慨青春不在。當年S2賽季，we和ig雙雄爭霸的場面，似乎還在眼前。

廠長是當年WE和ig最後一個退役的隊員，至此S2所有選手，都告別了英雄聯盟賽場，所有的一切都成為回憶。但看到許多玩家感慨青春不在的時候，大玩家卻有幾分不舒服。毀掉當初青春的，可能正是如今感慨唏噓的人。

當初WE 戰隊因為捧回了英雄聯盟第一個世界冠軍，而聲名大噪被玩家所熟知。自此才徹底打開國內電子競技的大門，結束了職業選手還在為收入辛苦代練的日子。如今，電子競技職業選手的收入已經直追一線明星，這個市場也越來越成熟。職業選手也可以專註比賽，而不為生活所煩惱。不得不說，英雄聯盟S2賽季對整個國內電子競技圈都有深遠的影響。

而整個S2賽季，幾乎便是we戰隊和ig戰隊的雙雄爭霸。其中的10個選手中，許多人的退役原因，都不是自願的。

作為當時世界頂尖的上單選手，雖然在S2賽季被we一直壓制。pdd本人也並沒有拿到世界頂級的榮譽，只有2013年SWL聯賽第二賽季冠軍、2013IEM新加坡站冠軍、2014LPL春季賽亞軍三個有含金量的比賽名次，但pdd卻是當時公認的世界最有侵略性上單。

在2014年退役後，便轉戰遊戲直播，開啟了自己的直播生涯。雖然pdd離開了英雄聯盟的賽場，但卻一直熱衷於電競事業。其組建戰隊出任老闆，繼續征戰英雄聯盟電子競技賽場。

雖然其戰隊成績並不理想，但卻培養出IG的寧，RNG的明，FPX的天和TES的Knight一批出色的職業選手，s8、s9總決賽都有其戰隊選手的身影。前不久更是宣布，和武漢旅遊集團合作拿下lpl名額，以另一種身份重新回到了lpl的賽場。

但許多老玩家可能還記得，當初pdd的退役可能並不是自願。甚至可以說，是被玩家噴退役的也不為過。許多老玩家相信還記得當時某AD女友在貼吧控訴PDD吃資源，被玩家「萬人血書」求pdd退役的事件。而這並不是孤立事件。

在當時的we戰隊，被公認的世界第一adc微笑，在當時接替若風成為we戰隊的新隊長後，卻因戰隊成績不佳，被玩家在網上瘋狂黑，在擔任隊長一年不到，便宣布退役。

其中也有因為廠長出走edg的原因。在s4比賽中，被廠長所在的edg戰隊打敗，倒在了通往S4總決賽的大門前。整個職業生涯也就三年時間，不得不說非常可惜。

而除了pdd、微笑、若風的退役同樣讓許多玩家不解，或者說整個we戰隊選手分崩離析，都是國內電競歷史中，讓人遺憾的事情。若風在圖退役之時，受到的流言蜚語同樣不少。當時英雄聯盟版本更新，對中路進行了增強。

而若風卻因為年齡偏大狀態下滑，幾乎被玩家噴成智障，即使退役也被玩家嘲笑，是所有退役選手中唯一常年鑽五的玩家。

而若風退役、明凱捲毛出走edg，也讓草莓也很快選擇了退役，其職業生涯也一直被「一血詛咒」這個梗所纏繞，輸了都是草莓的鍋，被玩家噴的體無完膚。而隨著草莓退役，整個s2的王者we戰隊，便只剩下一個空殼。

而ig戰隊的五人，其實一直都被網路黑子所包圍。Pdd的退役自不用說。當時的老隊員笑笑、姿態、影子、小孩幾人，也一直被黑粉所包圍。笑笑空大的梗，一直被玩家嘲笑了整個職業生涯。而影子在ig的生涯中，也一直被當稱作背鍋王，打得不好一定是影子的鍋。

小孩當時和笑笑被稱為ig雙子星，早期被稱為天才少年，但當ig戰績不好的時候，和這個名號同樣響亮的是一句順口溜，「兩人不滾，ig永斷腿」。

後期所謂的雙子星被秀成了背景板，被黑也自然是免不了的。而作為ig最後一個退役的姿態，由於轉會問題，直到退役的時候也依然沒被玩家原諒，被黑了整個職業生涯。

而此次廠長退役被一眾玩家，高喊青春不在的時候，卻忘記了自己可能是當初黑廠長最慘的那個。當時由於廠長的出走，直接導致了we的分崩離析。相信許多老玩家，還依然記得草莓哭著希望廠長留下來的樣子。但廠長還是走了，為了心中的一個冠軍。

廠長自此也被黑子黑上了天際。在老we、老ig戰隊中，廠長應該是心理承受能力最好的職業選手了，不然也堅持不到現在。

總結：如今看過當初we、ig退役的選手經歷，再看到許多玩家高喊的我的青春不在了，卻顯得有幾分諷刺。可能當初你的青春，就是被你自己親手毀掉的。在兩支戰隊中，如若風、pdd、笑笑、微笑的退役都或多或少和因為被噴心理負擔太重而退役的。

當時打的好了玩家就瘋狂追捧，打的不好了便被噴的抬不起頭。沒有一個好的心理承受能力，估計也只能如他們一般選擇提前退役了。現在你還會感嘆你的青春不在嗎？理智看待每一場輸贏，或許才是我們當下應該做的。
