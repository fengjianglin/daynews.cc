打野修鍊手冊：作為打野，如何呼風喚雨，掌控節奏，拿下勝利！

2019-12-09 0條評論 18次閱讀 0人點贊

摘要

大家好，今天正經遊戲呢，為大家寫一篇打野的文章，希望對大家有幫助。我從來不寫流量文，只做最有意義的講解、最硬核的攻略。

大家好，今天正經遊戲呢，為大家寫一篇打野的文章，希望對大家有幫助。

我從來不寫流量文，只做最有意義的講解、最硬核的攻略。

我們知道，打野是全隊第一節奏位，打野的發揮也就決定了隊伍前中期的節奏、優劣、順風和逆風；

很多小夥伴也想像韓信李白那樣飄逸，帶起全場節奏，今天我就來教大家如何成為一名優秀的打野。

我們分幾個階段介紹。

一、開局：第一個節奏點

打野開局怎麼打？紅開還是藍開？反野還是反打？

1.反野嗎？

如果己方陣容前期強勢，比如輔助是鬼谷子牛魔東皇太一、中單是諸葛亮司馬懿、邊路是孫策狂鐵橘右京這樣的強力戰士，而打野又是裴擒虎、劉備、楊戩這種前中期節奏的英雄，是可以去反野的。

但是，如果打野是娜可露露、趙雲這樣對四級要求很高的英雄，還是建議不要去反野，因為他們四級前作戰能力很差，反蹲即可。

2.紅開還是藍開？

如果打野是司馬懿或者露娜這樣的法刺打野，建議藍開，法刺對技能CD、回藍的要求很高，所以藍buff對他們的提升很大。

如果打野是裴擒虎、劉備這樣的打野，建議紅開，有真實傷害打野更快，像韓信、李白、趙雲這樣的打野，我建議還是紅開。

因為現在邊路對線都比較激烈，如果對方上單和己方射手對線激烈，比如橘右京前期壓制后羿發育，打野可以帶著紅buff二級直接抓下路。再不濟野可以控河道野怪，在前期，多一點點資源帶來的效果都會被無限放大，就比如，因為有河道野怪的經驗，我到四級，對面三級半，我就比對面多一個技能。

這樣往往有出其不意的效果，打野自身有紅buff減速效果，配合牛魔鬼谷子這樣的強勢留人輔助，gank成功率非常高。

注意：在打野紅開的時候，建議讓中單去看一眼己方的藍在不在，輔助去看一眼對方的藍在不在，摸清敵方打野動向。也可以讓己方輔助和中單去反三豬，也可以讓己方輔助防守己方三豬，如果對面有人偷豬，配合打野的紅buff可以直接黏住。

二、四級：第二個節奏點

四級了怎麼打？這其實是困擾玩家們很大的問題。有的玩家四級gank成功，進入順風，有的玩家被反蹲，直接陷入逆風。

所以，我建議打野四級去抓強勢邊/控制邊，如果下路強勢，就去抓下路，如果上路控制多，就抓上路。

1.反蹲

當然，假如對方打野紅開，那很大概率去抓己方下路，我們可以換個思維，反蹲對方打野，先讓己方射手（前期射手沒控制沒輸出）吃一套傷害，打野和輔助埋伏在草叢，在對方打野技能真空期收下人頭。

2.推塔

當然，在這個版本，抓下路是很好的選擇，因為射手普遍強勢，而且幫助射手快速推塔，能夠解放射手，讓射手更快發育，推掉下路，也可以順勢入侵敵方藍區，壓制打野和中單的經濟。

3.打小龍

如果你不推塔，就一定要拿小龍，用小龍的經濟彌補推塔的經濟。但是，在路人局，推塔帶來的效益，是遠遠高於小龍的，因為隊友可能不會利用打龍帶來的四級優勢打出大節奏。

gank成功後，直接推塔，推不了塔，建議拿龍，如果抓的不是暴君路，那就反野，如果野區沒野，就去壓制敵方中單，去磨中路防禦塔。

總而言之，在前期gank的時候，要注意隱藏自己的視野，不要隨意暴露在線上，當然也要合理蹭線，畢竟現在的版本野區經濟不足。

三、中期：第三個節奏點

推掉敵方下路外塔以後，該做什麼？

我們已經有了一定的突破，接下來該怎麼滾雪球呢？

1.控龍

要麼是控小龍，進一步擴大全隊優勢，要麼就是把重心放在上半區（為什麼？己方射手發育，就要少打架，多刷，己方打架射手磨塔吃線吃野發育，一個發戰爭財，一個拿峽谷經濟，打野是無法做到戰爭財和峽谷經濟通吃的，沒時間），在敵方紅buff打架，可以壓縮敵方射手發育空間。

讓對方射手忙於防守，無心發育。

2.推塔

所以，推掉下路外塔之後，要把集火目標放在上路外塔和中路塔，或者控制小龍。總而言之，就是要進一步擴大經濟差，壓縮敵方安全範圍。

總結：中期一定要把重心放在拆塔上，只有拆塔，才能壓縮敵方發育空間，己方刺客和戰士才能帶線，帶線了才能牽制形成以多打少的局面，牽制了才能擴大優勢，拿龍打團拿下勝利。

中期不拆塔，後期就可能面臨猝死局的情況，因為後期就是一波團戰的事情，前中期拆塔節奏起不來，後期就容易翻車。這也是很多玩家前期殺人後期翻車的原因。

四、中後期

當我們把三座外塔都推掉後，敵方的安全範圍就只剩三路二塔和高地的範圍了，這個時候，敵方野區基本上宣告失守，經濟差將會繼續擴大，兩條龍的視野也將基本丟失。

1.推二塔

中後期怎麼推敵方二塔呢？我們可以選擇牽制的方法：想推下路，先清下路兵線，不用帶到塔下，然後去中路或者上路打架，吸引敵方注意，如果敵方接團，那麼二塔就會被磨，如果對方少人去清線了，那麼團戰以少打多，就可能拿下勝利。

2.牽制

總而言之，敵人回防哪路，你就去相反的那一路清線，發揮刺客身法靈活的優勢。

那麼如何有優勢呢？野怪+兵線，一定要保證打野的經濟和等級是領先的，這樣才能有資本去做事。當打野領先對面兩級，基本上就可以呼風喚雨了，再不濟，也是一個人的順風，主動權依然掌握在自己手裡。

注意：打野作為前中期第一節奏位，千萬不能浪，吃了很多經濟，浪了就可能陷入逆風。一定要注意控制+傷害的銜接，先手後手和繞後入場的細節。

每推掉一座塔，敵方的安全範圍都會縮小。

五、上高地

1.什麼時候打龍？打龍怎麼總被搶？

這個時候，前中期推塔的優勢就顯現了。因為推掉了外塔和二塔，敵方的安全範圍縮小，得不到龍坑視野，而又被兵線牽制，疲於奔命，怎麼能顧得上大龍呢？

這裡給打架推薦幾個開龍時機：敵方團滅，己方無兵線；敵方陣亡一人以上；敵方打野陣亡；敵方兩人在暴君路（兩個人在暴君路，即使打龍被發現，三個人也打不過己方抱團，而兩人支援需要時間，即使龍被搶，己方也能留下敵方几個人，接下來以多打少，很容易拿下勝利）……

當打龍了以後，大家都會抱團推塔了。

2.如果對面高地防守能力很強，我們該怎麼辦呢？

拿下黑暗暴君，獲得AP和AD加成，依靠經濟和暴君buff優勢強開。很多人不注意黑暗暴君的加成，黑暗暴君的加成在中後期都是很恐怖的，後期大家都是六個裝備，一條黑暗暴君在後期的加成相當於半個破軍或者半個博學者之怒，這個時候，己方團隊的傷害是很高的。

除了打暴君，那就是控制三路兵線，中路兵線快，邊路兵線慢，我們可以放出中路兵線，讓中路兵線先自然消耗，然後讓打野和邊路利用機動性去清掉邊路的兵線，當邊路兵線前進後，再清中路兵線。這樣，三路兵線齊頭並進，敵方防守兵力分散，我們就可以集火一個目標，打開突破。

這個時候，就要給隊友發信號了，告訴他們，要控線。

只要推了高地，有超級兵的幫助，推水晶就很容易了。而敵方也更加會被兵線牽制。

所以，

假如你推掉下路高地，那麼把重心放在中路和上路即可，有超級兵在，下路兵線基本上不會陷入劣勢。

然後用下路兵線牽制上路和中路的兵線，這樣就可以分散防守，打開優勢了。

注意：不知道大家有沒有同感，一般先掉高地的隊伍，勝率都不高。所以，上高地非常關鍵！沒有準備好，己方千萬不能隨意在高地上開團。

總結：打野的優勢其實不來源於殺人，來源於推塔，推塔帶來的經濟和經驗優勢，對全隊的提升都有幫助。也能入侵敵方的野區，對面吃不到經濟，自己能吃經濟，這樣的優勢是雙倍的。

六、彩蛋：逆風怎麼辦？打野如何逆天改命？

1.如果隊友無腦上，明知道經濟落後還要強開，結果越來越逆風，我們該怎麼辦呢？

這個時候，打野就不要幫他們gank了，因為他們本身就經濟落後，越打越逆風，還容易搭上自己。這也是很多打野「幫優不幫劣」原則，所以，打野就要把重心放在運營和推塔上。

總而言之，少打架，多帶線，通過運營牽制來推塔（參照上文）。要利用刺客的機動性，全場牽制。

經濟可以慢慢刷，慢慢拖，但是防禦塔推了可就不能重建了。

2.如果己方因為決策失誤，比如被偷龍、被蹲導致經濟落後，我們該怎麼辦呢？

如果打團失敗，己方多人陣亡，可能被一波的時候，首先看看有沒有兵線可以斷！

如果有，斷線，如果沒有，單靠打野是很難逆轉局勢的，經濟落後，局勢逆風的情況下，以少打多贏的可能性很小。

如果己方能守住水晶，打野可以去偷龍，大龍可以強行續命。如果己方守不住水晶，其實單靠打野也是回天乏術的。

如果守下一波，在經濟落後的情況下，打團的勝率也不高，打野需要利用高機動性去斷線，保證對方的身後沒有兵線，這樣他們就沒辦法推水晶。總之，就是要不斷牽制兵線，補充經濟，爭取時間。如果兵線處理的好，甚至能帶掉敵方的防禦塔。

總而言之，就是要斷線，偷龍偷塔，多抓單。之後就要注意通過兵線運營，達到牽制效果，完成以多打少，多抓單！

3.如果隊友吵架、遇到演員，怎麼辦呢？

這個，我也沒辦法。可以關閉局內打字和語音。

我只做最有效果的攻略，希望對大家有幫助，祝大家玩得愉快！
