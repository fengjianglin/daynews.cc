顧雍不是廢將，當我拿到自書，牌堆都在燃燒

2019-12-15 610條評論 4,693次閱讀 3520人點贊

三國殺里總有些武將可能因為自己的技能強度不會被太多玩家在乎，或者出場性太低，這確實跟技能有一部分關係.

但是我們都知道技能的強大更在於的地方其實是一種配合，一個看似不厲害的技能如果能碰到一個與之契合的技能那麼所湧起的洪流將會遠超乎你想像.

簡雍，這個將其實一直出場都不高，1技能的2牌換1牌屬於那種破罐破摔的技能，2技能的免費看牌也一直受人詬病，也就順風時能群體閉月有點作用了.

因此即便在最近火熱的幻化之戰中單獨拿到這個技能也是顯得有些十分吃力，因為單兵作戰模式這個武將的技能實在有些不夠.

但你真的以為簡雍就是廢將了？他的技能一無是處了？

不，並不是. 幻化之戰雖然其他玩家也有多技能，但正是多技能才造就了更多的配合，因此簡雍的技能也能一飛成神.

慎行+自書，剩下的牌全都是我一個人的！

這個刷牌才是真正的bug，刷牌能力讓劉焉只能在一旁扣6，有這兩個技能，最後一個技能要不要都無所謂了.

慎行：出牌階段，你可以棄置兩張牌，然後摸一張牌.

自書：鎖定技，你的回合內，你摸牌時多摸一張.

先感受一下這2個技能，感受一下這其中暗含的、內斂的、強大的殺氣，試問哪個玩家不想試一試？不想上手感受一下牌堆瘋狂洗牌的快感？

丟2張摸2張，無限的制衡，莫得感情的刷牌機器，才是最讓人可怕的，至少劉焉的刷牌還是有節制的，只要摸到閃卡住就行，但是慎行就直接把手上不要的牌全部丟了，全部刷錦囊給自己屯著.

因此，幻化之戰存在著無限可能，但是如果你想體驗這個組合就可以去幻化之戰，雖然幻化之戰中技能搭配眾多，但是這兩個技能組合即便放在任何技能組合中都是十分亮眼的，絲毫不遜色任何搭配.

如果有想試試的小夥伴，趕緊去玩玩幻化之戰吧~超級刺激的~
