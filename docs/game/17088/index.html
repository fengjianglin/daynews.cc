魔獸世界：開荒打布胖有多難？治療和DPS缺一樣都會滅團

2019-12-14 1,521條評論 2,605次閱讀 636人點贊

魔獸世界的帕奇維克和布胖，這兩個boss是玩家們的噩夢。兩個BOSS開荒都是硬拼裝備，打戰神的時候最難的是湊4個能吃傷害的T，人數夠了就不太難了。布胖這個boss最難的就是雙刀秒人，運氣不好再加個腳踏，什麼T都不好使，所以那時候流行熊，血厚能硬閃一次。當年我們打布胖，我是3T，唯一的任務就是T死了以後能上去吃一次雙刀，基本上就是純划水，可以想像一下T的壓力多大。

記得開末日迴響版本前，能通SW的都是精英工會，每個服也就極少數工會能打。打布胖壓榨的是治療DPS，還有至少黑廟畢業的T。全團站位也很重要，指揮也很重要，在最後壓榨DPS階段，我們團都是組至少五個薩滿，輪流進DPS組開嗜血才勉強通過的。打布胖雙刀賊打2500是正常的，獵人要求才是1600-1800，而且那時候考驗的並不只有輸出，還有吃buff的層數和治療預讀，流程熟悉了就只剩運氣了。

2.4版本，布胖不倒T，輸出夠就過，戰士3圍加上格擋技能需要125，不然就吃碾壓和暴擊，2萬5以下的血都是被秒，還是熊T拉布胖好，吃碾壓暴擊就2萬2血左右，奶騎可以預讀大治療加得上血。布胖考驗的是兩個T的配合，治療的反應，DPS的輸出，缺一樣都會滅團，可不僅僅是飈DPS這麼簡單。至於穆魯，這麼說吧，沒削弱前的穆魯，只要進入P2階段，就沒有什麼走位技術這一說法，你的血就是不停掉，你要做的就是站樁最大輸出，在全團團滅之前把穆魯殺掉。

60級打布胖的時候，4對蛋刀，3賊1戰，2個人一組配個奶薩插風怒配個熊t。我秒傷2500，物理職業都是布胖前畢業裝備，法系基本都是差古頭狗杖，極限壓榨治療才過的。兩個奶德都洗了鳥，兩個牧師洗了暗牧，2奶薩1騎1神牧，沒踐踏的時候奶薩都是輔助輸出保持t激流搓閃電箭，第一次過純屬運氣好。

談難度要結合當時的時間，06年最難的boss顯然是小克、電男，其次4dk，老克和他的冰龍都要往後排，畢竟那個時代的網路加上40人團出意外的幾率，上網高峰期去打電男的下場穿T4也是送修理費。NAXX是40人副本，在容錯率方面高一點，沒有對於DPS的巨大壓榨，往後的25人10人DPS要求都高於之前。所以才會說布胖比戰神難打，40人去電男和跳舞機檢測一下看看，那才是難度。
