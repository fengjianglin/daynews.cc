暴走大俠：24日版本更新預告，新玩法或是披著PVP外皮的PVE

2019-12-18 289條評論 1,321次閱讀 338人點贊

氪金不暴走，大俠全靠肝，大家好，我是「爆肝大俠」小魚乾，歡迎來到由小魚乾開講的《暴走大俠新鮮事》。本期小魚乾將給大家帶來24日的更新預告，話不多說，抓緊時間和小魚乾一起來看看吧！

今天晚間，暴走大俠官方發布了12月24日的更新預告，PVP玩法終於出來了，趕緊來看看本次版本更新的內容吧！

新增內容：

  1. 新增PVP玩法「武林盟主」正式開啟，角色達到60級後即可解鎖
  2. 新增翅膀功能，消耗羽佛枝可提升翅膀等級，更可羽化激活更酷炫的翅膀皮膚
  3. 新增丹藥系統，玩家可通過煉製和食用丹藥來提升屬性

​優化內容：

  1. 查看他人信息界面，優化了境界部分的顯示
  2. 優化了物品（不含裝備）的分解體驗，支持可選數量進行分解
  3. 修復了登錄遊戲時會經常彈出歷史私聊信息的問題
  4. 優化了好友列表展示，現在可顯示好友離線日期
  5. 優化了遊戲體驗並修復了其他已知bug

更新獎勵依然是「老三樣」：200翡翠葉、3個魂鐵晶、3個免費復活。

本次更新的主要玩法，就是PVP玩法了，說實話很多玩家早就已經習慣把暴走大俠當成單機遊戲來打了，本次更新了PVP玩法，起碼能讓玩家們有些新內容可以玩，不至於除了闖關就是闖關。

但小魚乾還是有不好的預感，小魚乾覺得暴走大俠更新的「PVP」玩法，絕對不會是實時的，估計就是像王者榮耀那種「武道大會」，看似是PVP，其實還是看戰力的「PVE」玩法，真人去打一個複製出來的「機器人」，如果是這樣，實在是沒有什麼意思。

為什麼小魚乾會這樣覺得呢？其實很多玩家都能發現，暴走大俠的玩家數量已經急劇下降，小魚乾所在的大區已經是前十個，但現在可以看到世界聊天已經非常冷情，小魚乾這樣的「平民黨」，每天襄陽城隨便進去就送死拿個獎勵，居然都能排進前200，可見這個遊戲的熱度，已經涼得差不多了。

基於這樣的玩家數量，本次更新推出的PVP系統，能是實時的嗎？不出意外的話，小魚乾猜測應該就是由系統操控一個相同屬性的機器人玩家，與玩家進行對戰，然後進行排名。當然，實際玩法還沒有公布，小魚乾也無法確定是否就是披著「PVP」外皮的「PVE」模式。

再來看一下另外新增的兩個玩法，一個是翅膀系統，一個是丹藥系統。翅膀系統的描述是，消耗「羽佛枝」可提升翅膀等級，更可羽化激活翅膀皮膚。

得！原來就是另外一個「套裝覺醒」系統，消耗材料覺醒，提升等級，還能提升外觀，這不就一模一樣嗎？

再來看一看丹藥系統，通過煉製和食用丹藥，可以提升屬性。雖然看不出來具體的玩法，但可以知道這個丹藥系統就是用於提升人物屬性的，又是一個氪金大項。

而優化的5項內容中，也沒有關於關卡、技能方面的優化，也就是修復一些BUG。

總的來說，小魚乾對這次更新並沒有太高的期待，就看PVP做得怎麼樣吧，但各位玩家也要做好心理準備，畢竟暴走大俠現在的這個玩家基數，官方就算想做實時的PVP，也是無能為力，這個PVP，很可能就是另一個「戰力排行榜」！

​那麼以上就是本期《暴走大俠新鮮事》的全部內容啦！最後，感謝您的閱讀，各位路過的大俠給小魚乾點個關注吧~
