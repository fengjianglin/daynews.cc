《寶可夢劍盾》曠野地帶更改天氣方法

摘要

《寶可夢劍盾》的曠野地帶是一張巨大的開放地圖，有些寶可夢只在特定的天氣出現，玩家們想要捕捉到心儀的寶可夢的話可不要盲目跑圖捕捉。請看下面由「hunter94425」帶來的《寶可夢劍盾》曠野地帶更改天氣方法，希望對大家有用。

　　要捕捉曠野地區的寶可夢(非團戰)需要滿足兩種條件：

　　1. 徽章 –徽章數不夠的話抓不了等級過高的寶可夢，當然如果已經通關的話可以無視這個條件。

　　2. 天氣 –有些寶可夢只會在特定的天氣出沒於棲息地。

　　達成條件1不難，不過如果真的想要隨心所欲的抓自己想要的寶可夢，建議還是先通關(因為部分天氣跟劇情掛鉤)

　　首先打開圖鑑，並查看任意你想要的寶可夢的棲息地位置。

　　這裡以利歐路為例子，利歐路只會出現巨人之帽，並且天氣要求必須要是暴雪天。

　　這時候如果你的巨人之貌天氣不是暴雪天的話，你是絕對不可能遇到利歐路的。因此這時候我們要做的事情很簡單，就是按下你的Home鍵，去Switch的設定功能那邊調整你的主機時間。

已上是各個天氣所對應的日期

　　切換到這幾個日期後，整片曠野區的天氣都會是一樣的。

　　利歐路需要在暴雪天才會出現，所以我們調成2020/02/01後整片曠野區都會是暴雪天，兩個跟雪相關的天氣，大概會在第3-8個徽章內解鎖，確切的不清楚(因為我通關了)，但是濃霧天氣需要完成一周目(打完冠軍賽)才能解鎖。
