除了玩遊戲，PlayStation3還曾被用來計算宇宙的奧秘

2019-12-24 973條評論 2,349次閱讀 870人點贊

我有點相信GTX690是戰術核顯卡的傳說了

00 差強人意的第三代PlayStation

眾所周知，PS2是目前為止，世界上銷量最高的遊戲主機，整個遊戲史上也只有任天堂的NDS能夠望其項背，毫不誇張的說，PS2幾乎在全球範圍內統治了遊戲家用機市場。

數據來源VGChartz

但這樣的統治卻在PS3時代被終結，縱使這台PS2的繼任者在全球範圍內的銷量名列所有遊戲主機的第六名，但也正是在這一代，索尼PS2所構造的惟我獨尊的黃金時代一去不復返，這其中原因有許多，畢竟PS3在外界面對的是任天堂操作方式新穎的Wii和微軟財大氣粗的Xbox 360，但PS3本身從構架層面就充滿了劣勢，久多良木健帶領下的SCE曾經因為對技術的偏執而創造了PlayStation的奇蹟，而最終PS3又因這份偏執而將王位讓出。

但拋開那些老生常談的「黃燈問題」、「GeForce 7900GS」，這台初衷是為了遊戲的家用機，卻一度用來計算浩瀚宇宙中的奧秘。

電影《星際穿越》中的黑洞

01 失敗的PS2超級計算機

多年以來，科研人員一直都在考慮使用圖形處理器或是遊戲主機來提升實驗室的運算能力。按照這些科學家的構想，這些機器除了能夠渲染出《旺達與巨像》中的遊戲畫面，同樣也能進行大量的數據運算。而如果能夠用合理的方式將這些機器連接在一起，這個遊戲機群的運算能力將會大大提升，遠超單個遊戲機計算力之和，這就是集群計算。

2002年，索尼SCE為PS2遊戲主機發布了Linux工具包，這就好比在遊戲機和科學研究之間架起了一座橋，研究人員通過該數據包可以在PS2上書寫代碼，自此，遊戲機也進入了超級計算機領域。

如今的美國國家超算應用中心（NCSA）的高級研究員Craig Steffen可能是第一個實現「遊戲機超算」設想的研究員。2002年，他剛剛加入NCSA並參加某個項目，為了達到目的，他計劃購買一批PS2，並使用Linux工具包將PS2以及PS2上的「情感引擎（EE）」中央處理單元連接到一起，形成一種類似超級計算機的設備。

在最終的實際應用中，Steffen和同事們將六到七十台PS2連接在了一起，編寫了代碼還構建了資料庫，但最終的效果只能算是差強人意，因為這台「遊戲機超算」的內存存在問題，Steffen遇見了兩個無法解決的BUG——每當運行這台「超算」，都會使得所有PS2的內存進入一種十分古怪而又不穩定的狀態。

Steffen和失敗的PS2超算

最終，這一項目以失敗告終，Steffen在自己的辦公桌上留下了一台舊PS2當做紀念。游但戲機能成為超級計算機這件聽起來像是「天方夜譚」的計劃，已經照進了現實了。

02 連接PS3=超級計算機

PlayStation的超算冒險旅程才剛剛走出教學關。2006年底，PS3進入遊戲市場，彼時的PS3擁有著看上去十分強大的硬體（畢竟售價499美元，成本800美元），以及十分方便Linux系統運行的條件，研究人員發現，將遊戲機們連接在一起充當超算不再是一個有待驗證的概念，而是一項成熟可行的計劃。

馬塞諸斯大學達特茅斯分院的研究員Gaurav Khanna應該是第一個進行這類嘗試的科學家。Khanna和他的同事們研究的項目是宇宙中的黑洞，按照Khanna的說法，他們這種對於黑洞的純周期模擬研究通常申請不到太多資金，畢竟應用前景渺茫。

就在Khanna和同事們為了計算數據的超級計算機而頭腦風暴時，組內一名資深玩家提到了PS3遊戲機以及搭載的Cell晶元。雖然如今看來PS3的滑鐵盧很大一部分原因都要歸咎於這塊IBM製造的晶元，但在當時，類似的晶元可是用來製造最先進的超級計算機，於是Khanna產生了想法：「我們能將PS3應用於科學研究嗎？」

PS3搭載的Cell晶元

實踐出真知，這名天體物理學家立刻開始購買PS3，並組建自己的「遊戲機超算」。在購買完遊戲機後，Khanna還花了幾個月時間將代碼整理好，隨後程序的編寫又花了幾個月時間，最終這台特殊的超算搭建完成時，Khanna發現PS3遊戲機的數量從最初的8台一路增加到了176台。

馬塞諸斯大學達特茅斯分院的PS3超算

沒有大型空間的需求，也不用付費給超算中心，Khanna只是話費了相對於普通超級計算機微不足道的金錢，就能夠進行黑洞模擬的研究，或是輕鬆在同事間的密碼學競賽中取勝。

與此同時，其他的科學家們也有類似的想法。北卡羅納州的一個研究小組在2007年同樣製造了一台PS3超級計算機。幾年之後，紐約的美國空軍研究實驗室，計算機科學家家Mark Barnell開始了一個類似的項目，取名Condor集群。

但Barnell挑選的時機不是很好，2009年，索尼發布了最新的PS3 Slim，相比原版PS3，這台機器更加輕便，成本控制的更低，並且無法運行Linux系統。

這其中原因和遊戲機遭遇的「破解」有關，索尼在發現PS3遭到黑客破解後發布了固件更新，將PS3上的OpenOS進行了移除，這使得PS3無法運行Linux系統，能夠組件PS3超算的遊戲機開始變得稀有。

為此，美國空軍不得不向索尼進行交涉，要求對方將貨架上未進行固件更新的PS3下架，這其中美國軍方和索尼進行了多次會面，最終達成了目的，所有下架的PS3被存放在芝加哥郊外的倉庫里，一年以後，這一項目得以亮相。

整個Condor集群是由1700多台PS3組成，光是連接的電線就有5公里長，其一度成為世界上速度排名第三十五位的超級計算機，美國空軍利用它處理來自無人偵察機的圖像。

03 過時的PS3超算

為美國空軍服役可能就是PS3以及整個PS家族在超算歷史上最後的輝煌了。隨著科技的進步，超級計算機的運算能力變得更加強大，與此同時，遊戲主機也變得更加純粹，對於科學研究的幫助越來越小。

PS4在這一世代重新奪回了霸主地位，雖然沒有PS2那樣強勢，但仍然比Wii U和Xbox One高出一截，但對於研究人員來說，「PS4沒有什麼新奇之處，它只是一台普通的過時PC」。

美國空軍的Condor集群早在四年前就從部隊「退役」，1700多台PS3設備一部分被捐贈給其他項目，包括Khanna的那台；還有不少舊的PS3被當做舊遊戲機庫存出售給大眾，美劇《疑犯追蹤》（P.O.I）劇組搶購了幾百台存貨，並被媒體大肆報道。在該劇第五季首播時，觀眾可以在電視機上看到這些老舊的PS3，它們仍然被連接在一起，作為一台超級計算機出現。

而最早創建PS3超算的Khanna則將目光投向了英偉達的Nvidia Shield，這款電視盒子兼遊戲主機的設備運算能力是PS3的五十多倍，他計劃用這一設備構建自己的新超算。

Nvidia Shield

如今，在馬薩諸塞大學達特茅斯分校，數十台PS3遊戲機被放在一個冷藏的集裝箱里，運算著天體物理學的奧秘。這是該大學非常受歡迎的旅遊景點，校方試圖利用它向未來的大一新生學生和他們的父母推銷這所學校。

這些覆蓋在冷藏櫃，或是壁櫥後面，覆蓋滿灰塵的小盒子曾經是研究人員夢寐以求的東西，他們用這些遊戲主機來建造超級計算機。而有了這一台台遊戲機，科學家們能夠鑽研黑洞的物理學、處理無人機拍攝的畫面，或者贏得密碼學的競賽。

PS3超算只存在了幾年，科技就進步了，超級計算機變得更小、更有效率。但曾有過短暫的時間，世界上最強大的計算機可能還不如客廳裡帶來歡笑的遊戲機。
