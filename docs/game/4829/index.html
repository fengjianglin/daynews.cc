“王者榮耀一哥”以2億天價簽約，騷白人氣不斷下降原因何在？

2019-12-04 0條評論 17次閱讀 0人點贊

王者榮耀這款遊戲的火爆，帶動了國內手游產業的高速發展，該遊戲受到眾多玩家的喜愛，同時該遊戲也間接性的帶動了遊戲主播的熱潮，目前不管是大平台還是小平台，都有王者榮耀這款遊戲的主播，比如我們熟知的，騷白，張大仙，還有現任AG超玩會的隊員夢淚，他們都通過王者榮耀這款遊戲成為一名遊戲主播。

騷白大家應該都知道，曾經鬥魚以2億天價簽下這位遊戲主播，由於騷白的總給人一種神秘感，相信很多網友都有想過騷白到底長什麼樣子呢，為什麼不讓觀眾看到他的真面目呢，這個問題可能你得去問他自己了。

不過值得一提是騷白直播間的人氣在直線的下降，自從騷白簽約之後人氣可謂是一波三折，從最早有網友爆出代打風波之後，其實網友是有兩種的說法的，網友A表示，遊戲代打是目前很正常的一個灰色產業，裡面也不涉及什麼違法活動，只要你想升更高的段位，那麼別人替你代打，代打的人只是拿到他相應的勞動成果，所以你有錢你也可以找人代打。

網友B表示，作為一位王者榮耀一哥的身份，網友相信你有能力去衝擊更高的段位，但是找代打去提升自己的熱度，這就有點說不過去了，一位主播最起碼的責任沒有了，你感覺粉絲還會繼續喜歡你嗎？

我們都知道騷白在王者榮耀這款遊戲上面擁有333連勝的吉尼斯紀錄保持者，曾經也有網友各種的質疑，但是吉尼斯記錄是一個很權威的機構，隨之人氣得到回暖。

之前有人覺得是張大仙的人氣影響到了騷白的人氣，因為都在同一個平台直播，而且都是遊戲主播，相互爭粉是在所難免的，我們都知道大仙的直播氣氛一直都很好，而且處處都為自己的粉絲著想，感覺在大仙的直播間的氣氛很和諧。

之後張大仙離開該平台，並且自己也簽到了其他平台繼續直播，按照常理說騷白的人氣應該會有所好轉，但是事實並非如此，張大仙走後騷白在該平台的人氣依然低迷，可能是騷白目前還沒有能想到炒作人氣的方法，不過有網友說到，面具摘下來騷白絕對還能再火一把。

騷白的人氣在不斷的下降，不是他人的因素，而是要從自身找一下原因何在，一位想要擁有長久的人氣的主播，絕對不是一時的曇花一現，而是不斷地站在自己的粉絲多想一些問題，糊弄玄機只能讓自己越走越遠。

好了以上就是本期的全部內容，你認為騷白的人氣下降原因有哪些？
