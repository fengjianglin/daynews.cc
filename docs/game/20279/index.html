槍出如龍，掠敵先鋒，最爽莫過於開局一手殺閃的4血流氓雲？

2019-12-17 990條評論 3,824次閱讀 1946人點贊

「主公放心，我去去便回」

「龍升九天，馬踏飛燕」

sp趙雲一度在玩家心中是飛一般將領的少年將軍形象，不知道曾經打爆過多少玩家的心態，殺也不是不殺也不是，尤其是呂布、張角這樣的武將更是不堪一擊.

因此三國殺玩家愛稱「流氓雲」，偷的一手好牌，不靠別人靠自己，團隊中的主c經常帶飛.

因此拿到一手趙雲有多爽就不用多說了吧？

你想想3血的沖陣趙雲已經這麼強了，但如今鬥地主將池還加了sp趙雲該有多麼變態？

多1點體力上限暫且不論，可以多存一張牌.更核心的是每局可以出2張殺還能英姿3張，農民基本上就不用掙扎了.

槍出如龍，掠敵先鋒，4血並不是最爽，最爽的是開局一手基本牌.

開局一手基本牌二話不說上來先莽二號位兩下，先沖陣搶兩張牌再說，對方不僅少兩張而且還要掉2血，這種情況尤其是脆皮兩回合都抗不過去，連續被沖陣2回合，即便活著也是個痛苦.

這是淮竹鬥地主模式改版後第一次拿到的sp趙雲，先說感覺，簡直太爽了，無人能擋.

2閃1殺1無懈夢幻開局，不怕錦囊而且還有先手，這基本牌基本上穩定沖陣，所以2號位基本上是一個等死的位置，如果我是2號位，我直接刷一手五穀、借刀等著被沖.

如果地主拿到sp趙雲是一個絕大的優勢，一般的2號位幾乎毫無還手之力，如果自己再刷個樂、兵什麼的，2號位就可以直接託管了.

就從這局來說，界權死了後，骨頭難成氣候根本不可能單挑能贏sp趙雲，打的傷害打不出去、摸得手牌不屬於自己.

那如果地主是趙雲，有沒有什麼辦法呢？

如果地主是趙雲，沒有，等死吧！死在隊友前面，趁豆子還沒扣趕緊出去開下一把說不定還能贏回來.

因為一般的組合在sp趙雲先手的情況下真的很難有還手的機會，即便回合外sp趙雲憑藉4血血量優勢也是殺不動的.
