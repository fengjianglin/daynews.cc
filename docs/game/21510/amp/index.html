DNF：燦爛+15力/體黃色徽章又來了，930W一次獲取機會，太坑了！

以下為具體的內容詳情：

說起在目前遊戲中比畢業白金徽章還要難得的徽章，那答案只有在這11年的遊戲中僅出現過一次，就連我們國服第一的旭旭寶寶，最多也才擁有兩個的+15力量/體力的燦爛黃色徽章了，可以說特別的稀有，特別的難得，絕版道具中的絕版道具。

不過後續等到12.24更新後可就不一樣了，這個絕版了幾年的+15力量/體力的燦爛黃色徽章（可鑲嵌在角色的護肩或項鏈上）又來了，不過唯一比較坑的一點就是並不是直接免費送，而是需要玩家消耗金幣，平均每消耗930W金幣，才可以獲取一次獲得這兩種絕版徽章的機會，簡單來說也就是930W金幣可幾率隨機到一個，真是太坑了。

等到12.24也就是下周二更新後，策劃將會在達芙妮那裡加入新的徽章轉換功能。在其中玩家每消耗30金幣就可將一個燦爛徽章的屬性，隨機轉換成另一種燦爛的屬性。且當你轉換滿30次，在第31次進行轉換的時候，可以必定從+15智力/精神燦爛綠色，+15力量/體力燦爛黃色徽章中隨機獲得一種。

簡單來說就是玩家每消耗930W金幣，就可以隨機從綠色+15智/精，黃色+15力/體，這四種燦爛黃色徽章中隨機獲得其中一種，930W金幣，四分之一的畢業概率，嘖嘖嘖，不得不說策劃的這波春節前的回收金幣玩的是真心6啊。

不過儘管如此，這對於那些追求極限面板的玩家來說根本沒有什麼影響，比如說旭旭寶寶，不就是區區的930W金幣一次，四分之一的獲取概率嗎，只要準備個幾億金幣，哪怕運氣再怎麼差也能懟出來幾個吧，必定一個角色僅需要4個就能讓護肩和項鏈上的徽章達成最完美的畢業了。

但相反對於我們這些普通玩家就算了吧，930W金幣隨機一次真的是扛不住，反正慢慢等就可以了，絕版的燦爛黃色徽章既然又再次回歸了，那麼參考著之前的華麗黃色徽章，後續肯定還會陸續的出現，慢慢等活動白嫖就可以了，完全不用太過於著急。
