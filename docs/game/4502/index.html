Uzi渴望在單挑賽對陣Theshy，對於Shy哥，他用6個字評價

2019-12-07 0條評論 15次閱讀 0人點贊

摘要

有些新玩家可能不知道，在前幾年的全明星賽中，Uzi曾經兩次拿到了Solo

有些新玩家可能不知道，在前幾年的全明星賽中，Uzi曾經兩次拿到了Solo King冠軍頭銜。他是第一次在世界賽中，連續兩次拿到單挑王的男人。Solo King更是Uzi拿到的第一個冠軍獎盃了，可以說是有著歷史意義。全明星賽最近幾年都是娛樂為主，單挑賽可以說是唯一競技的賽事了。

畢竟，沒有任何選手想輸給對手，單挑賽則是他們證明自己個人能力的最好舞台。而在12月6日全明星賽的首日單挑賽中，四名LPL聯賽的代表選手，都順利擊敗對手晉級到後續的比賽。Theshy拿出德萊文輕鬆擊敗對手，Doinb依靠火圈完成了驚險翻盤，Tian在盲僧對決中技高一籌。

而Uzi則是和對手Nomanz都使用了小炮，小炮是這次單挑賽的強勢英雄，因為今年的單挑賽引入了火圈機制，在遊戲7分鐘後會刷新火圈，玩家在火圈外會立即死亡，並且輸掉這局遊戲。因此，不管你前面輸得多慘，只要你贏下火圈的死亡競賽，都可以反敗為勝。小炮大招可以將對手推出火圈，可以說是優勢巨大。

但Uzi則是依靠個人能力，直接打爆對手獲勝。雖然對手有草叢的視野優勢，但Uzi果斷地跳到了裡面，打出了E技能的傷害，刷新了W技能後繼續起跳。很多玩家都期待Uzi再次拿到Solo King的稱號，而他在賽後採訪時表示，自己最想在後續的比賽中對陣Theshy，畢竟Theshy是世界最強上單。

Theshy不只是上單英雄玩得好，其他位置的英雄同樣很強，被認為是LPL聯賽內部單挑實力最強的選手之一。Uzi則是很期待能夠和Theshy交手，想看看Shy哥的單挑能力有多強。手哥也期待Uzi和Theshy在Solo King比賽中進行對決，看看誰才是LP聯賽的單挑王。而兩位選手目前都晉級第二輪賽事，期待他們在後續比賽遇到。
