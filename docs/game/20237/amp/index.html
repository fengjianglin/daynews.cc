騷男試玩新英雄厄斐琉斯，二十分鐘偷大龍，騷男：看我單刷

天天要聞
2019-12-17

虎牙主播騷男，相信經常看直播的小夥伴都看過他的直播。騷男的直播一直都是非常搞笑的，騷男在直播的時候，也會做一些節目效果，每次都可以把觀眾給逗笑。騷男還經常跟粉絲互動，以及在直播的時候打Solo賽，每次的亞索也都非常的快樂。這次的騷男和青蛙雙排，騷男試玩無E凡，二十分鐘偷大龍，傷害直接爆炸，青蛙：沒了E還快樂嗎？

最近英雄聯盟又迎來了一位新英雄，那就是厄斐琉斯，江湖人稱無E凡的存在，他有五把武器，可以切換不同的形式，所以操作難度很高，而這個英雄卻天生沒有E，相信在全明星賽上，UZI的厄斐琉斯是非常的驚艷。這次的騷男也拿出了厄斐琉斯，並且是和青蛙雙排下路，青蛙則是錘石，下路前期打的有點小劣勢，因為對面是一個金克斯加大眼睛，不過在打野的幫助下，騷男還是成功的擊殺了對面的金克斯，下路逐漸好了起來。

而這時，觀眾都開始議論了，直接發彈幕問：沒了E還好玩嗎？因為騷男是玩亞索出身的，亞索最快樂的就是瘋狂的E了，而這個新英雄卻沒有E，剛好的阻擋了亞索玩家。不過騷男看見只是笑笑不說話，能殺人就行了。到六之後，青蛙先手鉤人，騷男在後面直接跟上輸出，然後切換成折鏡形態，直接貼臉打輸出，並且騷男帶的是致命節奏，直接拿下了雙殺！後續回家補過裝備後，對面已經完全打不過騷男了，下路直接開始了瘋狂屠殺。

騷男在前期打的非常的順利，而且在折鏡形態很猛，離敵人越近，他的攻擊次數就會越多，所以傷害是非常高的。這局打到二十分鐘，騷男直接跟青蛙說偷大龍，結果騷男直接跟青蛙一起偷大龍，騷男又換成了折鏡形態，幾乎無傷就偷掉了大龍，而且還很快。隨後的騷男就在中路開始瘋狂的屠殺了，並且加上青蛙的輔助，兩個人站在中路開始堵人了，看見誰就殺誰。不過在騷男偷男爵的時候，觀眾都看傻了，這傷害確實有點看不懂啊。

最後騷男還是非常輕鬆的贏下這局遊戲，不過沒有E的英雄，騷男玩的還是非常秀的，絲毫不影響操作，看來沒了E的騷男依舊很快樂。不知道大家對於騷男試玩厄斐琉斯怎麼看？歡迎大家在下方評論交流！
