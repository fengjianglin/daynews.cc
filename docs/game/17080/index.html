魔獸懷舊服戰歌峽谷，戰術與配合的極致體現

2019-12-14 3,721條評論 4,930次閱讀 3836人點贊

今天是魔獸世界懷舊服兩大戰場上線的第二日，因為昨天伺服器在上班時段都已經擠到爆炸，不得已起了個大清早開始排隊。進去之後驚訝的發現部落大廳早就已經擠滿了正在排隊的玩家。但是也不知道是凌晨時段人相對少一點，還是伺服器做了優化，今天感覺排隊速度快了很多，部落方戰歌峽谷幾乎只需要兩三分鐘就能排進去。

由於我算不上魔獸老玩家，對於經典版本的數據並不是太了解，開局只敢跟著大部隊走，當個邊緣摸魚的混子，原本心裡還挺沒底的。但所幸隊伍里有大佬，開局等待時便已麻利的分配好了隊伍職責。德魯伊開局繞邊路偷旗，盜賊開局跟隊沖一波，然後立馬回去守家，其餘的人留在中場接應德魯伊。

對面聯盟開局選擇十人抱團直接打正面，也因此第一面旗很輕鬆的就直接到手。雖然第二波對面開始警覺，但可能是散人隊的關係，雖然安排了盜賊守家，卻沒有安排人偷旗，打得比較散亂。所以我方正麵糰戰失利卻沒有丟旗，在回家守住了一波的情況下，還是勉強拿到了第二面旗。

第三波德魯伊偷旗被聯盟盜賊和法師抓死，我方剩下的人強推進去，雖然戰士拿到了旗，還是被對面堵門守住，最後反丟了一面旗。最後一波時隊里大佬看準了對面不打算偷旗這點，直接帶著全隊中場平推，趁著對面盜賊和法師還在守家，來不及支援中場，仗著人數優勢一波平推拿到最後一面旗。

雖說我全場幾乎都只是跟團躲在後面摸人，但一場打下來還是滿背流汗。中途一度陷入頹勢的時候，還以為這局要輸，但幸好抱了大腿，加上隊伍比對面更加團結，還是有驚無險的拿下了今天的首勝，也跟著大佬學到了不少東西。因為這把打得還算有來有回，並且除了PVP本身，和對面拼頭腦並且戰術成功實施的瞬間相當有成就感（雖說基本是大佬在指揮），一整把下來的體驗倒也感覺相當有趣。

或許是由於戰場才剛開不久的原因，個人感覺現階段純粹在體驗戰場樂趣的玩家要比效率黨少。即便不考慮聲望獎勵，戰場本身玩起來也挺有意思。
