開拓神秘島：12月19日即將登島！萌版魯濱遜，相約天堂島

2019-12-17 527條評論 2,539次閱讀 457人點贊

《開拓神秘島》12月19日即可登島！一段奇妙的冒險之旅強勢到來！荒島求生 萌寵加盟！帶你做一回Q萌魯濱遜~

《開拓神秘島》是一款來自於著名遊戲開發商開羅遊戲公司所出品的模擬經營遊戲。遊戲採用該公司一貫的像素風格，經營對象是一座海島，玩家可以通過種植作物、建設住宅和研究所等活動對海島進行開發。像素級別的遊戲在整體劇情和戰鬥設定上，也都保持著該公司一向的高水準。

遊戲的一大特色是可以捕獲和培養海島上各種神奇的怪獸，玩家可以利用這些怪獸進行戰鬥來拓展自己的疆土。如果看夠了自己的怪獸，還可以利用遊戲中所獲得的神秘咒文來跟朋友進行交換。不同的怪獸將擁有風、雷、土、水等不同的屬性，可根據其屬性相剋來合理的安排戰鬥，而大量成就系統的加入也使遊戲更加的出彩。

一開始，主角和寵物被流落到了一個荒島，什麼都沒有，手上只有很少的食物和木材，白手起家可以說是十分貼切的形容啦！初期會有三種寵物選擇1、狗土屬性血量高，速度一般.2、貓風屬性血量中，速度較高.3、鴨水屬性血量一般速度低但是MP高有點類似於法師。寵物的選擇完全看個人喜好！並沒有什麼強弱差距之說。

在開拓神秘島遊戲開局先建一個「牧場」給自己的寵物安個家，在布局時「牧場」請安放在樹木和山石之間，也就是「自然」屬性優先。「住房」請安放在沙灘上，一般在其四周建其他設施，以提高「文明」屬性的加成。當「牧場」建好後，系統會給我們提供新的探索區域，之後新的探索區域需要達到一定條件，才可增加探索的區域。

戰鬥是由寵物進行的，最多為3V3的戰鬥，當然自己開發了道具之後，可以有戰鬥中更換寵物了。遊戲中遇到的怪物可以使用道具將其捕獲，成為自己的寵物。有了新的寵物，我們可以安排更強力的寵物參加戰鬥。

初期我們擁有的道具有限，特別是捕獲道具僅僅只有三個，所以一定要節約使用，捕獲之前建議先存檔，如果多次使用道具沒有捕獲成功請讀檔重來。捕獲道具必須在怪物血量很低的時候，顯示「弱」，在圖像方面怪物開始出汗的狀態下進行捕獲，當然如果使用特定的屬性道具捕獲怪物時，會增加捕獲的成功率。

很快就可以解開島四周的區域，其中解救出了幾名特殊人物，當然先給他們建設新的「住房」，然後就可以改建了。改建出幾個特定的建築，為我們增加了「商店」和「研究所」等等特殊項目。曾經眼前一片荒蕪，如今萬丈高樓平地起！體驗非凡戰鬥的同時玩家還將在本島體驗到開羅最經典的經營要素，設施建造、改建相性、設施研究、土地擴建，玩法應有盡有！

本次的介紹就到這裡啦，《開拓神秘島》中依然保留了系列作中優秀的經營要素，在畫面風格上繼續沿用經典的像素風格，12月19日快來登島體驗Q版魯濱遜的荒島求生吧！快和夾子一起打造一個你心目中的天堂島~
