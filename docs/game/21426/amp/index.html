漫威也出手游啦？超級英雄齊聚集，鋼鐵俠還能發射激光炮

天天要聞
2019-12-20

一曲歡樂送，一場紅塵夢；大家好，我是電競歡樂送。

在如今，MOBA手機遊戲是非常火的，王者榮耀就是最大的一個代表。這種類型的遊戲玩法都差不多，最吸引玩家的一點就是英雄設計了，玩家最喜歡什麼樣的人物是遊戲公司必須考慮的。比如王者榮耀採用的做法是使用歷史人物的名字，還有的引入經典街機人物、聯名DC出超級英雄。這個做法還是很有用的，至少玩家看一眼就記住遊戲里的英雄叫什麼！

說到超級英雄，就不得不提到漫威，漫威里的超級英雄可遠比DC的超級英雄受玩家歡迎。小編想有不少玩家都希望能夠在遊戲中操作自己最喜歡的超級英雄吧，甚至有玩家自己自製漫威英雄的皮膚。最近小編看到一款遊戲，裡面的英雄聚集了漫威所有的超級英雄，你看右邊的英雄池，有綠巨人浩克、雷神索爾、快銀、黑豹，連格魯特和火箭等等，不禁讓人想問：漫威出手游啦？

來看看這款遊戲具體是什麼樣子吧。我們就拿鋼鐵俠來說明吧。從人物模型方面來講，這個鋼鐵俠還是做得很不錯的，王者榮耀的模型都沒有這個清晰，那這個鋼鐵俠在遊戲里是什麼樣子呢？

既然穿著戰鬥鎧甲，鋼鐵俠自然是不需要走路的，直接在天上飛來飛去。普通攻擊自然是遠程的，這個遊戲界面還是很熟悉的吧，和王者榮耀基本是一模一樣，就是圖標略微不同。這個飛在天上的鋼鐵俠和你們腦海里想的一樣嗎？有沒有覺得哪裡不好呢。

來看看鋼鐵俠的技能吧。鋼鐵俠一技能是釋放出一大串的導彈對敵方進行攻擊，在電影里鋼鐵俠好像沒有發射導彈的功能。這個技能設計得容易讓人齣戲，還以為是羅德上校駕駛的愛國者呢，不如改成強化技能。

鋼鐵俠的二技能就和電影能聯繫起來了，這是是通過手釋放出藍色的衝擊波。電影里鋼鐵俠的攻擊手段基本都是通過手釋放能量波，雖然不是藍色的，但是這個技能還是能讓人有那麼點電影里的感覺。不過這個技能的傷害倒是不怎麼高！

鋼鐵俠的大招就是沖胸膛中釋放出激光炮了，攻擊範圍還是非常遠的。其實釋放出瞬間切割的激光更好，就像電影里鋼鐵俠釋放的那個大招一樣，一瞬間毀掉了一大堆的機器人。既然設計漫威英雄就按照漫威電影里來不就好了，幹嘛自己設計技能呢！
