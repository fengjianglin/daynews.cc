摘要

昨晚萊比錫Major中國區預選戰罷，最後iG、VG和茶隊三支隊伍成功殺進了線下賽中，很不幸，EHOME成為四支隊伍中最菜的那個，4進3的賽制下都沒能把握住機會慘遭淘汰。

昨晚萊比錫Major中國區預選戰罷，最後iG、VG和茶隊三支隊伍成功殺進了線下賽中，很不幸，EHOME成為四支隊伍中最菜的那個，4進3的賽制下都沒能把握住機會慘遭淘汰。

EHOME和茶隊於敗者組決賽中相撞，本來還期待著名場面「無情握手」能上演一次，卻不料技高一籌的茶隊沒給塞老師任何機會，直接2:0就將EHOME斬於馬下了。這樣一來EHOME不得不去參加Minor預選，並且就算出現了還要再拿到Minor冠軍才能回到Major中來，這條路想要走成並非易事。

萊比錫Major預選全部結束後，Minor的四支參賽隊伍也全部出爐了。他們分別是Major預選淘汰賽沒能晉級的EHOME，小組賽兩組第三的LFS、CDEC，以及在加賽中戰勝Newbee拿到名額的RNG。

四支隊伍將通過雙敗BO3爭奪唯一一個Minor出線名額，不知道這次誰能笑到最後呢？

塞老師這邊沒能上演無情握手，但是馬桶哥那邊卻完成了這一名場面。馬桶哥所在的Secret昨晚於歐洲區預選中和Nigma遭遇，身具復仇之心的馬桶哥化身超級大腿，和新隊友一起2:1將Nigma眾人打入敗者組，之後Nigma又輸給了Alliance不得不去打Minor預選了。

馬桶哥：我要把負我的人全殺了！

布克維Minor中國區預選將於明天下午13:00開戰，為期兩天，想看比賽的同學別錯過開始時間喲！

今天V社把另一個新英雄電炎絕手也給削了，說起來也是挺有意思的，新英雄剛更新的時候表現很差，兩兄弟勝率排名倒數第一和倒數第二。V社看不過去了，於是趕緊在後續版本里加強了一波。

然而這波加強不得了，在玩家熟練使用兩個新英雄、並且強度提上來後，這倆兄弟的勝率又開始暴漲，再加上某天晚上那個男人使用紫貓把把超神掌控雷電後，V社對著兩個新英雄都砍了一刀，昨天是紫貓，今天是「老奶奶」。

電炎絕手：

霹靂鐵手魔法消耗從75提高至90

霹靂鐵手傷害從45/60/75/90降低至40/55/70/85

霹靂鐵手每次攻擊速度減速從20/25/30/35降低至15/20/25/30

龍炎餅乾冷卻時間從24/22/20/18降低至24/21/18/15

龍炎餅乾彈道速度從800提高至1000

龍炎餅乾自身使用的施法前搖從0.4降低至0.3

說出來你可能不信，在剛更新的時候楓哥就特別會玩電炎絕手，第一次使用就拿下了瘋狂殺戮，說是個小絕活不過分。畢竟看過=會玩，玩過=絕活，贏過=絕中絕。

楓哥今晚本來在激情對黑，結果打到一半OB幾兄弟集體掉線，一問才知道原來是基地電錶燒了。現在正在等維修工人過來檢修，至於晚上回不回來直播就不得而知了。

剛剛那把鋼背獸大家也看了，對線期被「狗男女」組合SK+火女打炸，場面劣勢巨大。結果最終還是翻盤成功，贏下了至關重要的BO3決勝局。如果今晚沒辦法回來直播，那這場比賽就是收官局了，贏了等於贏一天。

OB視角下雙方的經濟情況，龍楓雙雄高居前二，穩的不行！龍神的紫貓也是絕中絕之一啊！

如果今晚電修不好的話，那小殭屍們明天再見啦，鬥魚直播間9999精彩對黑不容錯過喲~

看著楓哥在遊戲中大殺四方，在屏幕前看直播的小殭屍是不是也手癢難耐，想要去全新的江湖裡闖蕩一番呢？！

那就趕快來玩我們為大家準備的精緻小遊戲吧，即便沒有楓哥的操作也能打出統治戰場的感覺，邊看楓哥直播邊玩小遊戲，體驗不同但同樣精彩的刺激世界，還有比這更幸福的事嗎？
