除了光榮，還有一家公司也曾涉足歷史類遊戲——世嘉的《天下人》

2019-12-04 0條評論 13次閱讀 0人點贊

摘要

如今提到以歷史為題材的遊戲，玩家們應該會首先想到光榮公司的《三國志》、《信長之野望》之類的名作吧？的確，歷史題材是光榮公司最擅長的遊戲類型，但是還有一家公司也曾經涉足過這個領域，比如今天我們要說的這款《天下人》，就是由世嘉公司一手打造的經典日本戰國題材遊戲。

如今提到以歷史為題材的遊戲，玩家們應該會首先想到光榮公司的《三國志》、《信長之野望》之類的名作吧？的確，歷史題材是光榮公司最擅長的遊戲類型，但是還有一家公司也曾經涉足過這個領域，比如今天我們要說的這款《天下人》，就是由世嘉公司一手打造的經典日本戰國題材遊戲。世嘉一直都是備受玩家們推崇的著名廠商之一，自早年的主機大戰失敗後，世嘉一直沉沒低調，將全部精力放在軟體開發上，推出了《如龍》、《三國志大戰》等優秀作品。本作雖然不像《如龍》和《三國志大戰》那樣著名，但也的確是一款很有代表性的優秀作品。遊戲系統與光榮公司的《決戰》系列類似，但同屏人數更高，並且在操作上保持世嘉一貫動作遊戲的良好手感，將戰火瀰漫的日本戰國時代從不同角度展現給玩家。

本作中玩家可以扮演的主人公多達五名，織田信長和德川家康自不用提，雄霸一方的島津義弘、獨眼龍伊達政宗、”日本第一強兵”真田幸村也都成為了統一天下的主角。在真正的歷史上，島津義弘雖然雄霸九州，但是卻深受領民的景仰愛戴，因此在本作中扮演的則是一位貌似兇狠但內心卻無比善良的明君，同時還著重描寫了島津四兄弟的關係。伊達政宗在歷史上則算得上是一位生不逢時的出色君主，雖然將伊達家的領地大幅擴張，使伊達家達到了頂峰，但卻因為天下已定大勢所趨，不得不放棄統一天下的理想，在本作中玩家則可以幫他圓夢。真田幸村在歷史上不過是小大名之子，但卻因為大阪之戰的出色表現而受到後人的敬仰，號稱日本第一強兵，在本作中玩家則可以讓他成為天下之主，用自己的熱血青春和忠誠英勇結束弱肉強食的亂世，讓百姓們重新安居樂業過上幸福生活。每名主人公的故事都是多結局並非一線路，根據玩家的戰鬥選擇，主人公們都可以進入不同的路線。如果想要達成全部結局，則要花費到玩家不少時間，也可以讓喜歡本作的玩家得到更長時間的享受。

本作的人設由著名畫家獅子猿負責，他的人設即尊重歷史又大膽新穎，讓玩家耳目一新。本作中最具特色的則是明智光秀和豐臣秀吉，前者一反明智光秀在遊戲中一般都是大帥比的設定，將他的睿智狡詐充分表現出來，後者豐臣秀吉雖然依然一副猴子神態，但在豪華的服裝基礎下平添了一副眼鏡，給玩家帶來此人深不可測的感覺。北條家自初代北條早雲崛起開始，威鎮關東數十年，但在秀吉小田原征伐時卻因為藐視農民出身的豐臣秀吉不肯低頭，但反抗戰鬥卻並不徹底，秀吉的大軍勢如破竹佔領了北條家的大部分領土。此時的北條氏政依然是猶豫不決，一直徘徊在抗戰與降服中，最終導致了北條家徹底滅亡。因此在本作中北條氏政的人物設定偏重醜化，把他演繹成極為滑稽的跳樑小丑，讓玩家一眼就能感覺出他的無能與狂妄。

本作的戰鬥採用即時戰略的遊戲方式，根據玩家下達的指令各部隊進行相應的行動。選定角色後將其移動到要進行攻擊的部隊或者是地點後，選定部隊會自動進行移動。如果選定敵人移動，我軍發動攻擊部隊則會自動追蹤。如果是遠程部隊，那麼在移動路上遭遇到敵人時，如果不選擇原地攻擊，則該部隊會無視射程內的敵人繼續前進。

影響各角色的攻擊力防禦力的是兵種和士兵數，士兵數量越多攻擊力防禦力越高，如果兵力顯示為零時，該角色則從戰場撤退。戰鬥中士兵損耗的數值有兩個，一是士兵上限數值，二是現有士兵數值，隨著戰鬥的進行士兵上限數值會逐漸減少，玩家如果不及時為部隊回復，那麼該部隊的士兵數量會越來越少直至撤退。現有士兵數值回復後最大人數為士兵上限數值，因此為了保證部隊的戰鬥力，要對損耗士兵的部隊及時回復。在戰場上會出現矢倉這一設施，設施周圍顯示為藍色，將部隊移動到圈內則會恢復士兵的數量，不過大部分矢倉要從敵人手中奪過來。除了回復我軍士兵數外，矢倉還可以對範圍內敵軍發動攻擊，雖然射程近殺傷力小，但卻可以通過工程兵改造得到提升，並且佔領矢倉或者敵人軍旗會提升軍隊的士氣，士氣越高攻擊力越高。在一些戰場上還會出現大型的武器，玩家只要將武將移動到這些武器上便可以自動使用，雖然大炮、石弓的發彈數量有限，但卻威力強勁，會給予射程內敵人部隊毀滅性打擊。同樣敵人控制著的矢倉和大型武器也會對我軍造成巨大的傷害，伊達政宗篇中敵人幾門大炮無限彈藥同時發射，簡直可以用地獄來形容。因此開戰以後玩家一定要快速佔領這些重要的設置，合理利用才是取勝之道。

在遊戲中出現的兵種共計25種，大致可分為八類，每類兵種都有自己的優點和缺陷，熟練掌握兵種相剋並靈活運用，在戰鬥中定會受益匪淺。槍兵雖然行動緩慢但卻防禦力極高，可以用來對抗敵人多部隊同時進攻，弓兵在初級階段遠程攻擊力高，但移動緩慢，經常會被敵人追殺，但轉職後攻擊速度和移動速度都會大幅上升；鐵甲騎兵雖然強勁但卻對敵人遠程射擊束手無策，面對高等級槍兵時也會損失慘重；醫療兵儘管手無縛雞之力，但卻可以隨時為周圍部隊回復士兵數；忍者的攻擊力防禦力一般，而兵種特技卻極為出色。根據各武將的特點，他們可以率領出征的士兵也不相同，隨著戰鬥的不斷進行，武將率領的兵種會轉職為新的兵種，武將可率領的兵種、士兵數也會不斷增加。

陣形也是影響戰鬥的重要因素，不同的兵種擁有不同的陣形，相同的兵種可以從四種陣形中選擇其一。每種陣形都有自己特殊的能力，玩家可以根據戰場的需要選擇最為合適的陣形。兵種的衝突還會導致部隊陷入異常狀態，比如說擅長快速行進進行攻擊的騎兵部隊與擅長遠程攻擊的弓兵系部隊交鋒後，弓兵會陷入混亂狀態，陷入混亂狀態的弓兵隊不但不能移動、攻擊、反擊，而且防禦力半減，很快便會被騎兵部隊殲滅。想要從異常狀態中恢復正常，就必須耐心等待或者是利用醫療兵的特技回復。異常狀態有混亂、逃亡、衰弱、睡眠四種，混亂狀態下移動、攻擊、反擊不能防禦力減半；逃亡狀態下該部隊士兵數量逐漸減少，當佔領，敵人矢倉時，敵人會進入此狀態；衰弱狀態下的部隊士兵數緩慢下降，攻擊力、防禦力、移動速度半減少、睡眠狀態下不能移動、攻擊，防禦力減半。

為了方便玩家遊戲，本作的五大主人公遊戲難度並不相同，玩家可以由淺入深逐漸挑戰高難度。如果在戰鬥時覺得比較吃力，也可以進入自由關卡中重複戰鬥，取得經驗值提升人物能力。當玩家對自己的實力很有信心時，還可以進入挑戰模式，利用自己在各主人公篇中招募到的武將，挑戰高難度地圖，挑戰成功後會得到優質的道具作為獎勵。

戰鬥中玩家派遣出場的武將們可以三人一組，將三支部隊合成一支大部隊。根據武將率領的兵種不同，組成的大部隊得到的輔助效果也不相同。大部隊的組建大大方便了玩家的操作，也可以使玩家能在最短的時間內選擇不同的兵種克制敵人的士兵。一般來說大部隊的第一名武將要選擇那些移動速度較快，士兵數量較高的武將，率領遠程攻擊部隊的武將要放在最後，便於保護自身的安全。

本作的特技與兵種緊密相關，各部隊可以使用的特技由兵種決定而與率領部隊的武將無關（主人公武將專有的御家藝除外)。兵種經驗積累到一定程度後，可以學會新的特技，並且會增加使用回數。特技不但可以提升我方士兵的能力、給予敵人強力打擊，還可以使敵人陷入各種異常狀態。

除了以上的諸多要素外，天氣、地形等要素都會對戰鬥有直接影響，想要完成各種挑戰，玩家就必須對本作的各種要素了如指掌，還要在最短的時間內應對敵人的行動作出最正確的判斷。喜歡戰略遊戲或者是對日本戰國歷史感興趣的玩家，一定會通過這款遊戲獲得一定程度的滿足感。總之，在玩過光榮出品的各式歷史大作之後，回味一下世嘉出品的歷史題材佳作，也許你會發現一片新天地。
