一夢江湖NPC俠緣上線後，原配俠緣都只能坐冷板凳了？

2019-12-13 268條評論 3,113次閱讀 147人點贊

現在市面上的遊戲非常多，玩家可選擇的範圍也很大，所以遊戲為了提高玩家和遊戲之間的黏度，增加了很多有趣的設計，其中的社交系統就是為了增加玩家與玩家之間黏度，進而增加對遊戲的活躍度而設置的。遊戲中社交性最突出的莫過於RPG類遊戲，擁有各種俠緣系統、師徒系統、好友系統等。

《一夢江湖》作為RPG類手游，自然也少不了這些社交系統，其中最受歡迎的就是甜甜的俠緣系統了。在俠緣系統剛上線時，整個江湖都瀰漫著戀愛的甜味，大家又是情侶裝，又是共享房產證的，街上還有好多乘坐花車出遊的情侶們，都以為這就是愛情了！

可沒想到，這樣的好景並不長，最近《一夢江湖》又上線了NPC俠緣系統，玩家們瞬間變心，都打算扔掉現有俠緣和NPC談戀愛。後來了解到有俠緣的也能和NPC談戀愛，就勉強把原配俠緣留下了，但很多玩家的日常就變成了和NPC秀恩愛，原配也只能坐冷板凳了！

不得不說玩家們和NPC之間的互動是真的甜，不僅能語音說悄悄話，還可以牽手、抱抱、舉高高，小編隔著屏幕都能感受到甜膩膩的味道。

不僅如此，從玩家們曬出的和NPC的聊天記錄里可以看到，他們之間的對話也相當有意思，當玩家說出”看山看海看某人”的時候，感覺方思明的醋罈子都打翻了，只能打出省略號，簡直就是還原了現實中男朋友有氣又不敢撒的憨憨樣子，著實沒想到高冷的方思明還有這樣一面！

雖說如塵是出家人，但說起情話來也是格外的熟練，一句”只要貧僧能給，便拱手送上”就蘇化了少女心，話說大部分原配俠緣都說不出來這麼浪漫的話，所以被冷落也是很正常的！

NPC俠緣系統現在也還在不斷地增加各種浪漫有趣的玩法，原配俠緣們可要注意了哦，可能一不小心做錯了點事就回歸單身狗的生活了！話說那些被俠緣冷落的內心都是什麼感覺呢？
