火影遊俠隨風起 雲頂八大S陣容盤點

2019-12-14 727條評論 2,097次閱讀 893人點贊

經過新一輪更新的平衡調整，雲頂之奕陣容玩法更加多變了起來。而在新版本中，更是有著8種陣容強度達到S級，掌握任意一種都能吃雞!

重要改動

首先說一下版本比較重要的改動，這是直接影響到版本走向的：

幻影之舞只能無效暴擊的額外傷害 ，不能閃避整個攻擊 。意味著刺客終於從這件裝備的壓迫中，抬起了頭!額外傷害閃避沒關係，我有普攻特效、有技能特效——雷霆的觸發、地獄火的觸發、繳械刀沉默刀的觸發……

雷霆傷害提升、光攻速加成提升、地獄火可以通過暴擊觸發。這些都直接間接影響到了整個雲頂局勢

召喚雷霆光劫

3召喚+3刺客+3光+2秘術+2雷霆+2海洋+2沙漠

很明顯，這個陣容就是全部羈絆服務於一人——光劫：

核心棋子

以上的7種羈絆，全都能作用於光劫，一旦抽出劫並給其轉職光，那麼場上電閃雷鳴，一劫可敵天下!

春哥甲防一手三星小法，無盡可以幫助劫更多的觸發雷霆來打出傷害。

而掘墓就是為劫提供儘可能多的光之工具，所以都做出防禦裝，保證自己的存活

陣形站位

娜美掛斜角，全體收縮盡量將敵方後排拉散，讓刺客可以更好的切入。

變動建議

當棋盤元素是風時，可以放棄2海洋，轉用2雲霄提升整體閃避。抽到雷霆拉克絲時，毫不猶豫拉人口或是換下奧恩。

火影遊俠

4遊俠+3影+3地獄火+2水晶+2守護神

大多數千珏都是走法師流，但此陣容里，物理暴擊流千珏更加給力!

核心棋子

千珏是此陣容的絕對核心。當前地獄火靠著暴擊也能觸發，所以給到千珏無盡來大幅提升輸出。另外羊刀+飲血來保證千珏的續航。

但比較怕三星小法，所以要麼儘早追三，要麼把水晶艾希養為副C。

陣型站位

三大近戰站前排，賽娜獨站角落好給最遠的隊友加BUFF。四大遊俠後排瘋狂輸出。

火影召喚

3影+3召喚+3地獄火+2守護神+2秘術+2海洋

此陣容打的就是人海戰術，靠著螞蚱瘋狂召喚虛空蟲靈，能打能輸出和。同時副C千珏在後排給到單體高額輸出。

抽中娜美之後，人海戰術的輸出強度會再度提升一級。

核心棋子

螞蚱當前比較好追三，兩把青龍刀能讓螞蚱的召喚速度達到一個恐怖的地步，搭配鬼書給每個人都套上灼燒重傷效果。

千珏主要負責打輸出，清理敵方較為難纏的單位。

陣型站位

幾大召喚師靠斜角站，讓召喚生物卡在前排吸收傷害。娜美對角大招能產生最大效果。千珏彈跳可以有效防刺客並反殺。

雷霆守護

3雷霆+2山脈+3雷霆+3刺客+2海洋

此陣容就是強化前排的坦度，讓雷霆屬性來儘可能的打輸出。活的越久，傷害越高。

核心棋子

劫是雷霆核心，做出防禦裝不斷分身，讓分身同時具備生存和雷霆的輸出兩種能力。

4守護神本身可以提升300%護甲，搭配反甲讓奧恩成為AD英雄的噩夢。同時龍牙+狂徒讓奧恩生存能力大幅提升。通過雷霆被動來造成高額傷害，成為前期的勝利保障。

陣型站位

守護神直接頂在前排，刺客掛後排。

提升建議

雷霆拉克絲換下狗熊，非山脈元素棋盤或是刺客不給力的話，換成兩大輔助。

極毒狂掠

2極地+3毒+3狂戰士+3掠食者

掠食者前期直接追高星，靠強度連勝。後期奧拉夫一人頂一片天!

核心棋子

大嘴通過電刀和回聲兩件濺射裝備來觸發掠食者，秒殺其他殘血單位，並通過幻影之舞防刺客強秒(法師多可以換成龍牙)。

後期主要還是看奧拉夫，大招一開，通過羊刀和技能大幅提升攻速，極地屬性眩暈敵人+沉默刀卡藍。配合狂戰士的錐形範圍打AOE控制。

陣型站位

兩個遠程第三排，兩個掠食者抗在前排，奧拉夫最後一排起跳。

提升建議

來煉金後直接換下老鼠，給其鬼書和春哥甲。為了強化前排坦度，可以換成水晶;或是拉一人口湊鋼鐵羈絆。

沙漠刺劍

4劍士+3雲霄+3刺客+2沙漠+2秘術

老牌強勢陣容，夢魘打輸出，輪子媽打控制。名場面參考全明星雲頂之奕那場以2敵N的爆炸輸出場景!

核心棋子

夢魘一定要搶金鏟轉職劍士，才能瘋狂輸出，靠著被動來打AOE傷害和回血，持續作戰。

而輪子媽就是繳械+沉默打控制，另外當正義之拳觸發回血效果時，輪子媽就可以以1v多!

陣型站位

兩劍在前排抗傷害頂過第一波，刺客直接切後排。讓輪子媽和夢魘進行瘋狂輸出。

極地遊俠狂毒

6極地+3毒+1煉金+3狂戰+2遊俠

前期利用掠食者過渡，來了高星棋子順勢轉型極地，瘋狂進行控制，讓奧拉夫輸出全場。

核心棋子

給老鼠轉職極地，湊高級控制羈絆，如果沒有金鏟子就先4極地。奧拉夫是核心輸出單位，吸血和高攻速加持可以打敗大多數陣容。

煉金給到鬼書全場高速掉血中毒，老鼠就負責掛AOE控制。

陣型站位

布隆在最前排抗傷害，煉金在第二排衝出給敵人掛DEBUFF，第三排遊俠進行輸出，第四排狂戰士跳臉。

極地雷霆狂毒

3劇毒+4極地+3雷霆+1煉金+3狂戰士+2守護神

此陣容對比之下核心點更多，單個棋子資源減少，但整體容錯率相對提升。除極地狂戰之外，多了雷霆作為輔助輸出。

核心棋子

此陣容依然要給到老鼠專職極地，否則就放下布隆先用7人口2極地。奧拉夫作為主輸出之外，劫也可以配合分身和雷霆輸出打出足夠的輸出。

陣型站位

第一排守護神吸收傷害，煉金依然第二排準備出手，奧拉夫和劫對角跳臉。

結語

版本八大S級陣容，你最喜歡哪一套?用你最愛的那套去吃雞登頂吧!
