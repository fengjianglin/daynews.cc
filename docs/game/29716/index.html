LOL日服才開設3年，比賽收視率已成T1級別，決賽近4萬人觀看

2019-12-24 499條評論 1,460次閱讀 453人點贊

眾所周知，日本一直不流行電子競技遊戲，畢竟他們有任天堂這個老品牌，導致掌機遊戲一直非常流行。最近十年主機遊戲和手機遊戲，又受到了很多年輕人的喜愛，所以，傳統的網路遊戲很難在日本打出名堂。拳頭遊戲也是到2016年4月時，才在日本開通了英雄聯盟的伺服器。

不過，英雄聯盟在過去幾年則是發展非常迅速，拳頭遊戲在日本做了非常多的推廣活動，使用了大量明星聲優來為英雄配音，再加上很多新模式都是在日本率先上線，比如雲頂之弈第一賽季。因此，日服逐漸吸引了不少當地玩家和外地人，就遊戲規模方面在本土形成了不小的氣候了。

在12月23日時，日本研究院發布了一份全新的報告，在最近一年中，日本國內觀眾收看直播的電競比賽，英雄聯盟顯然已經成為日本最受歡迎的電競賽事項目了。單純是日本的LJL聯賽，在今年就達到了數百萬小時的總觀看時間，比其他的比賽遙遙領先，堪稱是T0級別的電競賽事。

畢竟，拳頭遊戲在今年實行了特許經營，LJL聯賽從6個名額增加到8個名額，俱樂部沒有降級的風險了。LJL聯賽的舞台也變得更大，基本上和LCK新場館一樣，可以容納400人左右，而且是在市中心附近。這些大力改動使得LJL聯賽的人氣越來越高，今年決賽現場的觀眾人數也是非常多。

當然，日本畢竟是一個遊戲玩家比較少的地區，日服也才開設了三年，LJL聯賽想要走得更遠並不容易。不過，LJL聯賽平時在Twitch上的直播人數也不少，最近夏季總決賽接近4萬人了，如果拳頭遊戲認真經營的話，有機會變成一個發展前景還不錯的外卡賽區。尤其是英雄聯盟手游一旦發布，能夠吸引更多日本玩家。
