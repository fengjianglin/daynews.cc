“破冰流”關羽開始流行，一局打出17-1的戰績，這套路太凶了

2019-12-16 138條評論 3,353次閱讀 91人點贊

破冰流關羽是什麼？一種利用冰心的冷卻縮減加上破軍的高傷害，搭配全攻擊銘文無視隊友自己超神的打法。大家好，我是取名鬼才小y 。 開局直接頂飛上官婉兒堵住他回家的路，配合甄姬拿到一血。 這邊剛好有大，猴子剛好在打龍，大招推進去繞後，普攻頂回來繼續轉圈頂回來。跟大家講個連頂技巧，大招開完，先一技能轉刀然後繞後等能能量條滿，用普攻頂，千萬不要用一技能劈，這樣會把目標打遠，導致整套連招沒有連貫性。

聽說塔下有個殘血凱，這種時候用最保險的方式收人頭，二技能踩一下轉一刀 他連操作的機會都沒有。 觀察了一下蔡文姬已經奄奄一息了，這種時候還是同樣的操作， 跳進去轉一刀就跑。剛好開個大，又是一次連環頂，凱連技能都按不出來。然後我們再去切後排，調整好角度馬可波羅就是一刀的事情。這邊簡單拿個三連決勝。這種打法前期盡量少用滿能量的一技能劈 ，因為我們很脆，用普攻頂接一技能傷害更容易打出，你也可以滿能量用二技能踩一下再轉，這樣傷害也容易打出。這邊猴子躲草里，先普攻頂一下，迅速二技能解控脫身，等技能CD 繞到他逃生路線 一技能收掉人頭。

玩關羽思路要清晰，秒不掉就跑等技能，並且學會用最保險的方式打出傷害。只有前期發育起來，中後期才能浪，比如直接用一技能劈後排。

大招回推兩人，轉一刀 然後越塔兩發連頂帶走婉兒。有個細節就是連頂操作要學會用二技能去加速能量條回復，這種打法好處就是不依賴隊友，只要你出攻擊裝夠秀，一個打五個。小提示玩關羽不要強迫症，後期平A傷害也很高，千萬彆強迫症一定要跑起來。無論你玩什麼英雄，面對馬可波羅 出肉就是給他機會秀你，所以只要對面玩馬可 我從來不出肉。

銘文：隱匿、鷹眼、異變

裝備：抵抗之靴、霸者、暗影戰斧、破軍、冰心 、碎心錘
