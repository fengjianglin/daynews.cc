魔獸世界冬幕節活動16日到來 下周大地的裂變時空周常

2019-12-11 539條評論 3,960次閱讀 2146人點贊

12月12日-18日，在魔獸世界遊戲里即將發生的事件有：

  • 大地的裂變時空漫遊周常
  • PVP亂斗：六人戰
  • 冬幕節
  • 大秘境詞綴

一、 大地的裂變時空漫遊周常

86級以上的玩家可以通過隊伍查找器加入特殊的隨機時空漫遊地下城隊列中，在祖達薩和伯拉勒斯的記載者圖帕可領取「崩裂的時光之路」周常任務，完成5次時空漫遊地下城將可獲得一件415裝等的永恆王宮（普通難度）團本裝備。

二、 PVP亂斗：六人戰

迷你的戰場活動，在各種經典戰場上參與PVP，對戰人數被壓縮至6V6，部分戰場所需收集的物體和目標數量也會減少。在這個模式下，銀屑礦坑的礦車數量將只有兩輛，戰歌峽谷里的陣營旗幟之間的距離會離得更近，寇魔古寺中的神器數量將減少至兩個。

三、 冬幕節

一年一度的冬幕節即將於12月16日（星期一）開啟，活動持續至明年1月2日結束，共17天左右。

冬天爺爺正在帶著煙林牧場的禮物訪問鐵爐堡和奧格瑞瑪，整個艾澤拉斯到處都充滿著節日的喜慶氛圍。

依照去年的玩法，參與冬幕節活動還能獲得節日限定獲取的「寒冬之握」，還有節日限定坐騎「格嚕普斯的爪牙」，另外還有基本上白給的特效爐石「冬天爺爺的爐石」等多種物品獎勵。

四、 大秘境詞綴

下周大秘境詞綴是：強韌、繁盛、震蕩、迷醉。（遊戲內的日程表顯示有誤，不是激勵重傷。）

想了解下周大秘境排名嗎？記得關注YXRANK大秘境排行榜。

五、12月11日20點~20點30分懷舊服伺服器排隊狀況
