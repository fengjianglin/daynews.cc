攻略步驟：

1、 開局櫻雲賢者使用迅雷之刃，降低敵人雷屬性耐性。

2、 皇帝使用w黑魔法，2次爆雷，如果武器法攻不強可以選擇用君臨。

3、 拉薩姆使用英雄之歌，如果隊伍中有人等級不夠血少也可以用匱武降低物理攻擊。

4、 劫火使用防禦結界。

5、 羅塞莉亞使用T白魔法加血。

如果按照上述步驟來，第二回合就是可以完成戰鬥了。

6、 皇帝繼續使用w黑魔法爆雷，其他兵員重複上一步操作。

7、 櫻雲賢者迅終終焉之雷，結束戰鬥。

一回合破防和加buff

總結，這個討伐快速結束戰鬥的思路就是降低重魔導機甲的雷抗性和魔法抗性，最佳輸出優先選擇櫻雲賢者，同時具備破防和雷魔法最大輸出。Boss的攻擊以物理輸出為主，滿等級配合合格的白魔道士問題不大，想要完成4個挑戰的話還是需要一回合破殼、加buff，二回合開始就可以最大輸出了，5回合內條件不難達成。
