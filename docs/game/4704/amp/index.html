LOL新英雄神似SKT上單！因髮型太殺馬特，而遭到拳頭設計師修改

2019-12-05

摘要

LOL最新英雄厄斐琉斯的相關新聞相信大家都已經看了，這個英雄是引入了全聯盟獨一無二的武器庫機制，自己一共有五把不同的武器，每次只能帶兩把，然後循環使用五把不同的武器，這樣的機制可以說讓很多玩家都非常感興趣。

LOL最新英雄厄斐琉斯的相關新聞相信大家都已經看了，這個英雄是引入了全聯盟獨一無二的武器庫機制，自己一共有五把不同的武器，每次只能帶兩把，然後循環使用五把不同的武器，這樣的機制可以說讓很多玩家都非常感興趣。

同時也有玩家擔心說這樣的機制會不會導致這個英雄的強度過於強或者過於弱？現在不用擔心的是會出現凱隱那種兩形態不好平衡的問題，但是也需要擔心強度上的問題。

由於目前新英雄還沒有上線因此玩家無法得知上手之後的真正體驗。

不過最近新英雄卻因為一個特殊的修改而引發了網友的興趣，新英雄厄斐琉斯的模型形象被拳頭設計師修改，因為之前的形象似乎有點不太符合大眾的審美。

從這張圖可以看出，這個新英雄原來的髮型的確看起來有點彆扭，劉海偏向其中一側，另外一側的頭髮非常少，看起來有點偏比較彆扭，而從這張老圖很多玩家都紛紛表示，這個新英雄有點酷似SKT戰隊的上單Khan。

Khan的髮型的確與這個非常相似，也是偏劉海，而且五官形態也有點像。

同時也有網友表示這個新英雄還有點像模特張亮，以及LPL官方解說澤元之前的那張非常社會的髮型圖，僅僅只是髮型非常相似。

總之英雄與現實人物撞臉的事件比比皆是，這次新英雄的撞臉也是在意料之中，在髮型修改之後，很多玩家也表示好看了一些，但是也有玩家表示兩個髮型都很醜，都有點殺馬特，新英雄的形象看起來不太符合英雄聯盟的一個畫風。

總之這個帶有武器庫新機制的英雄的出現必然會引發非常廣泛的玩家爭議，目前所有玩家都在期待這個英雄，尤其是ADC位置的玩家。

因為這個英雄的官方定位是射手，所有武器的射程都是遠程，因此很多ADC玩家都期待這個英雄能夠拯救一下現在版本非常弱的AD。
