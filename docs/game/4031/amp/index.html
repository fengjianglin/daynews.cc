王者榮耀：虔誠50萬被賣！白菜價意味RNGM大變革！冬冠杯可期待？

Kpl秋季賽即將開打！

淘汰出局的戰隊也沒閑著，開始了人員調動！一些小道消息也「滿天飛」！其中，最勁爆的一條消息：RNG.M戰隊的打野虔​誠將要在下個賽季加入到RW戰隊！

爆料的網友稱：虔誠​直播時所講。出讓價為50萬元。

首先聊一下這消息的可信度！

一、虔誠轉會身價為50萬？這個價格猶如白菜價。是真的么？

在RNG.M，虔誠是打野位。技術水平在整個聯盟中算是一流。他與暴風銳有「寶貝雙C」之稱。

雖然今年RNG.M成績不佳。本賽季更是破天荒的被首輪擊敗出局。但虔誠的發揮還算穩定。多次獲得MVP的稱號。也是隊內絕對的主力。

如果他要走？一定有兩方面的原因：一是過得不愉快。二是工資不高。三是實現理想。我認為前兩方面的原因不存在。第三個原因有道理的。作為一個對事業有追求的競技選手來說，到更強的戰隊去，是常態想法。比如。國內很多足球運動員都想去歐洲跑球。一方面是賺更多的錢，另一方面是提高競爭水平。更重要是到更高的平台去實現自己。

虔誠有出走的理由。

轉會費50萬。對於kpl王者榮耀來說，這是一個極低的價格。2019年貓神的轉會費是1200萬；久誠的甩賣價在3750萬；老帥2018年轉會價1000萬；當然，這些人正當紅時。轉會費高可以理解。

可是虔誠只有50萬有這個可能么？

存在這個可能。一是打野英雄在當前版本中，已不吃香。目前打野位英雄需要兼邊路。如果是一味的打野，作用不大。在RNG.M虔誠與中路暴風銳合作，成為兩個最強輸出。可實際效果不佳。今年的成績就是一個很好的證明。二是RNG.M戰隊敗​績拖了虔誠的後退。試想一下，一支排名靠後的戰隊，能將隊員賣出高價么？水落石出就是這個道理。三是RNG.M存在重大變革，急於甩包袱。或許「寶貝雙Ｃ」的體系即將拋棄。對於重要性不大的打野位便宜賣也有可能。kpl王者榮耀人多粥少，有些選手壓在手中，最後轉會價可能更低。

二、虔誠被賣，RNG.M意在何處？劍指冬​冠杯決賽么？

RNG.M算得上kpl王者榮耀中的豪門，得過許多成績。一直來都能進前四。但2019年秋季賽敗得太慘。對RNG.M來說是個恥辱。對RNG.M俱樂部高層更是刺激。因此，RNG.M大洗牌不可避免。

變革的開始，第一步就是換人。有的戰隊大手筆。全換。比如，昔日霸主hero將老隊員全下，直接上新隊員。有些戰隊微調，比如，Ｅ星引進貓神。還有一些戰隊是重組合併。比如，ＡＧ超玩會，二個新人，二個轉會，二個冷板凳。

如果說RNG.M將虔誠賣掉。是改變當前作戰體系。就能理解了。因為新版本下，射手體系強勢。中單暴風銳不會動。

打野位虔誠地位就不保。與其坐冷板凳，還不如賣掉。免得麻煩。

其次，RNG.M這一次變革是痛定思痛的自救行為。再不變就沒市場了。至於是否劍指冬​冠杯，進決賽圈。就不得而知了。如果接下來沒有持續的大資金介入，就只是一種小幅度變化。如果RNG.M大資金引進新的隊員，就可判斷出RNG.M高層有新的想法。

畢竟落水的老虎，還是老虎。也有重歸山林的想法。

​對於RNG.M的未來，我挺悲觀。

因為RNG.M在今年秋季賽中的表現太差。這一種差不僅是隊員的狀態不行，更重要的是RNG.M打法戰術一點也沒有變。

對於一個不能與時俱進的戰隊，期望值不需要太高。

虔誠的轉會，是一種正確的選擇！莫要怕。在職業的黃金期，競技選手要大膽地去嘗試。一旦錯過，年紀漸長後，很有可能會和無痕一樣，沒人要。
