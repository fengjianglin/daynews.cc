大家好，這裡是指尖上的王者，王者榮耀中的活動越來越多，只是為了給玩家添加新奇的一些任務吸引玩家留在遊戲中，而王者榮耀中最好的獎勵莫過於皮膚了，擁有喜歡的皮膚是個開心的事情，因此王者榮耀最好的活動，可以說就是送皮膚的活動，且難度都不高，那麼有一個活動，堪稱王者榮耀最難做成的常規任務了，大家知道是什麼吧。

沒錯，就是17賽季全新增加的探索任務，該任務小編在25星的時候才完成，可見這個任務對玩家來說需要耗費多久的時間，不過獎勵還算豐厚，除了幾千鑽石外，還有一個回城特效，但只能讓賽季皮膚的英雄使用，也就是說本賽季的終極探索獎勵，是夏侯惇專屬的回城特效。

除了探索任務，本賽季更是添加了一個升級標籤的活動。

這個活動被稱作榮譽挑戰，可以說是王者榮耀中歷來最難做成的活動了，如果你單打排位，至少要榮耀王者才能獲取這個任務的獎勵，如果你的水平還不夠榮耀，沒關係，還有巔峰賽，打到40星，在把巔峰賽打到1500分就能獲取了，當然這需要的時間比直接打到榮耀王者更難，因為40星的玩家，只需要衝10星即可到達榮耀。

那麼很多玩家興沖沖的衝到了榮耀王者，拿到了獎勵卻不高興了，廢了這麼大的勁，竟然是這樣的獎勵？

夏侯惇的賽季皮膚，這個標籤從「鳳儀之詔」變成了「鳳儀之君」，且帶動態的特效，亞瑟變成了金色，但是除此之外，該皮膚的局內外特效絲毫沒有變化，也就是說衝刺到了榮耀王者，獲得的獎勵也不過是個標籤升級而已。

這個皮膚相信沒有幾個人會使用，有錢的買了高達，沒錢的如果喜歡這個英雄，至少也有海灘皮膚，而這個新的標籤必須要用這個皮膚才行，所以可以說是毫無價值。

當然了，話說回來，沒有人會為了這個東西而特意衝擊榮耀，肯打到榮耀的玩家肯定是自己有這個實力，就像我一樣，沖榮耀拿這個標籤，只是順手的事情，該打的還會打，打不上來的也不會有遺憾，你們覺得呢？
