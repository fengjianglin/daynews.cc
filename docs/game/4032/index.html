DNF：新活動冒險者的遊戲玩法詳解，不是要你玩得更好，而是更快

2019-12-10 0條評論 18次閱讀 0人點贊

大家好，我是冷水，致力於給大家帶來第一手的DNF資訊和攻略，看完文章不要忘記點贊和關注哦，您的支持是我莫大的動力。今天給大家帶來12月10日新上線的活動冒險者的遊戲的詳細攻略。

冒險者的遊戲就是之前大家經常說的「大富翁」，其實這個活動除了搖色子、走格子之外和大富翁真的一點都不搭邊。這個活動比的就是比誰先走到終點，所以這個遊戲不是比誰玩得更好、更全，別的是誰玩得更快，也就是鼓勵大家成為阿拉德最快的男人。

遊戲地圖

這個地圖算上起點和終點一共有32個格子，除了起點沒有任何作用外還有其他8種格子。

神器寶箱：進入地下城擊殺怪物後得到一個袖珍罐，打開隨機獲得一個稀有或者神器級別的守護珠，整個小遊戲只有第一個進入的時候有效。

巨龍地下城：進入巨龍地圖，擊殺巨龍後隨機掉落3~10個精鍊的時空石。

小遊戲：進入一個原來虛無的走格子或者巴卡爾城站格子的小遊戲，成功通關後隨機掉落1~5個精鍊的時空石。

黃金罐：隨機抽取一個高級至神器級別的守護珠。

黃金哥布林：進入怪物地圖，擊殺黃金哥布林之後隨機掉落2~10個精鍊的時空石。

跳轉：玩家瞬移到別的格子，後退、前進都有可能。

移動：按照格子上的指示（+是前進，-是後退）移動。

終點：進入領主BOSS場所，擊殺BOSS後通關小遊戲。

只要是轉移地圖的格子，在通關之後屏幕右下角都會出現一個「參與冒險者的遊戲」的按鈕，滑鼠左鍵點擊一下就可以回到小遊戲的地圖。因為在第一個玩家通關領主之後遊戲就會結束，所以在通關小遊戲、拾取獎勵之後儘快點擊這個按鈕回到小地圖才是玩家最應該做的。

這之中最浪費時間的格子有兩個，第一個是普通地下城，是的，這個普通地下城有時候比領主還浪費時間，因為這個圖的BOSS是特倫斯發電站的BOSS閃電之帕特里斯。如果玩家不能直接進圖秒了離得很遠的他的話，他就會瞬移並釋放技能，玩家必須打掉他周圍的四根電纜才能擊殺他，很是浪費時間，所以最好瞅准機會直接打死他。

第二個就是小遊戲環節了，不管是哪個小遊戲都有夠浪費時間的，等你玩完一個小遊戲別人說不定直接到終點了。很奇怪的是小遊戲可以通過返回城鎮快速跳過。一般格子，比如普通地下城，你回城的話會扣你一顆心（玩家在小遊戲中只有兩顆）並遣返你回到終點。但是小遊戲不一樣，你返回城鎮就跳過了，還是在原地，只不過少了小遊戲的獎勵罷了。

本來這倆小遊戲應該成為很多老玩家回憶老副本並在新玩家面前展示老玩家手法高明的地方，但是這種想法應該是實踐不了了，因為大家為了節約實踐都會直接跳過這倆小遊戲的，除非把從小遊戲回城、失敗也改成直接掉血然後回起點。

參與遊戲的三名玩家只要有一個玩家通關重點的領主房間遊戲就會結束。通關的玩家獲得3個龍之硬幣，然後其他兩名玩家按照離重點距離的遠近分2、3名，離得近的是第二名，得2個龍之硬幣，離得遠的是第三名，獲得1個龍之硬幣。如果後兩名玩家在同一個格子上則都是第二名，都獲得2個龍之硬幣。

至於獎勵嘛，我覺得兩個滿屬性的寶珠都挺好的，各需要40個龍之硬幣兌換。

以上就是這個活動的詳細玩法和流程了，雖然是個偏休閑的遊戲，但是玩起來還挺緊張的，因為慢了你材料就少，兌換獎勵就慢，一般想拿第一得往前跳才行，所以我就祝大家在玩這個小遊戲的時候多落在往前跳轉的格子上吧。
