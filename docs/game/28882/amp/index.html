《荒野大鏢客2》試著扮演一名在幫派生活的西部槍手，體驗熱血

天天要聞
2019-12-21

荒野大鏢客2

《荒野大鏢客2》是Rockstar製作並且發行的一款西部為題材的動作冒險遊戲，玩家在遊戲中需要扮演一名在幫派生活的西部槍手，體驗各種各樣令人熱血沸騰的故事。

遊戲背景：《荒野大鏢客2》將帶玩家回到美國西部的開荒時代，作為一個鏢客，玩家將騎馬馳騁於西部荒原，在荒野上搶劫貨車火車銀行，並和其他警匪周旋槍戰。通過搶劫的金幣購買強大的武器，挑戰被懸賞的通緝犯。通過搶錢提升等級獲得屬性點，在貨店升級技能槍支和支援召喚。每關分三種難度，能反覆練級。障礙能用槍打碎，連擊能追加金錢，金幣和獎勵錢幣連續命中能翻倍。推薦選開荒派的4號悍匪，或技能派的3號悍匪，優先升滿填彈技能，確保持續輸出。

人物

在本作中我們將扮演主角「亞瑟摩根」探索美國鐵路大發展事情的美國西部荒野，從崇山峻岭到邊陲小路、從茂密無垠的森林到無邊無際的沙漠，從牲畜無數的農村到現在化的都市。而我們所扮演的主角也不再是什麼光明偉大的英雄而是美國政府口中的不法之徒——牛仔。雖然是不法之徒，但是在遊戲中你卻可以根據遊戲中的大量劇情和隨機事件而自由的選擇成為一名好人或者壞人，你可以跟路人打招呼甚至是對目擊者進行恐嚇，選擇捲入事件或者是袖手旁觀，而這些選擇也會影響遊戲後期的劇情發展，你的意願將決定主角「亞瑟」成為一個什麼樣的人。

本作將不僅是開放的世界，同時也是一個互動性更強的世界，結合全新的動作系統和與GTA5相同的電影化敘事手法以及遊戲方式，你會身臨其境的體驗到一名西部牛仔、一名臭名昭著的亡命之徒為了生存而不得不戰鬥、逃亡的生活，你將見識到20世紀美國西部最黑暗血腥的時代，親身感受這一段變遷的歲月。而作為一個互動性極強的遊戲，本作的戰鬥系統也進行了徹底改進，戰鬥場面引人入勝，而每種武器也都有各自的特點。而在營地乃至遊戲中的每一位角色甚至是馬匹都有自己的性格，你也可以與他們進行互動。並且你在進行遊戲的同時也將要通過狩獵、交易等方式維持營地的運營。

而在之後將要公布的視頻中將為我們展示遊戲的任務、活動、敵對幫派、搶劫以及包括死神之眼在內的其他內容。
