世界第一上單TheShy的電競歷史，看完莫名有些心酸，真是太勵志了

2019-12-22 2,232條評論 4,248次閱讀 3317人點贊

2019年LPL賽區的全明星投票落下帷幕，TheShy以23.75%的票數位列第一，這也就意味著TheShy蟬聯最受歡迎選手獎項，為什麼TheShy會有這麼高的人氣，那麼今天小編就帶著大家翻開TheShy的故事，讓我們一起去了解這位喜歡發育，沒有自信的上單選手！

關於TheShy這個ID的由來，小編也是聽說網路上流傳著兩種說法，第一種說法是單純因為害羞，第二種是說他比較崇拜一炮四個PDD的Shy，所以給自己起名TheShy，小編個人還是比較傾向於第二種說法，因為小編懷疑Shy哥這個人害羞是裝出來的，小編認為真正害羞的男人是不會在採訪過程中狂笑的，沒有打職業的TheShy和大部分的職業選手一樣，早期都是活躍在韓服的高端Rank局，年僅十五歲的TheShy就已經在遊戲中個頂尖職業選手較量，有些甚至碾壓，這點並不是小編誇張，相信了解TheShy的玩家都知道！

TheShy的身影很早的時候就出現在各大集錦中，小編最早知道TheShy這個人的時候也是在操作集錦上看到的，讓小編印象最深的便是下路草叢一打五的瑞文，TheShy除了免費為精彩集錦的視頻作者免費提供素材外，TheShy還開創了屬於自己的獨立打發，TheShy的操作也被網友們稱之為TheShy combo，小編曾經看到一個TheShy集錦，視頻中的TheShy用瑞文僅用一秒鐘打完E+閃+RWA+提亞馬特+AR的操作，小編也將這位受害者翻了出來，正式前不久剛宣布退役的Imp！

想必有些玩家對TheShy獨創的豹女三分鐘最快刷野有印象，「十個豹女九個代，還有一個是TheShy」，這句話也是在民間廣為流傳，小編相信英雄聯盟玩家應該不會有人不知道吧，說到這裡就有許多網友問，TheShy天賦這麼高為什麼沒有就在LCK呢？很多網友不知道其實TheShy曾經短暫參加過LCK次級聯賽隊伍，並且還參加過SKT的試訓，那個時候的SKT則更希望TheShy能夠去打野位，我們的TheShy也是十分強硬，除了上單並沒有考慮其他位置，最終TheShy也選擇了離開。

小編記得那時候是2014年，那個時候的LCK正處於對英雄聯盟的統治時期，在LCK看來他們並不缺少天才選手，因此我們Shy也並沒有得到重用，小編猜想LCK萬萬沒想到的就是他們曾經放走的這個孩子，會在幾年後的世界舞台上綻放耀眼的光芒，在2014年末的時候TheShy來到了中國加入了WE，那時候由於規定，年幼的TheShy並不能上場進行比賽，TheShy只能作為青訓隊員終日打Rank開直播，就在TheShy與kid雙排的時候說出了自己內心的想法，小編認為Kid可能覺得TheShy很可惜，於是就將TheShy推薦給了自己所在的俱樂部IG。

2017年IG戰隊用輔助選手Ban與WE交易換來了TheShy，加入IG的TheShy起初雖然只是替補，但是好歹能夠登場，2018年我們終於等到了那個我們熟悉的IG，而TheShy也等到了那個願意為他抗塔的AD，春季賽TheShy帶領IG豪取18連勝，在此期間TheShy也為大家上演了很多驚為天人的操作，或許是隊伍的磨合不夠，IG在春夏季賽中都輸給了RNG，那個時候的RNG也迎來了最輝煌的時刻，S8全球總決賽IG作為LPL三號種子隊伍起初受到的關注並不多，隨著比賽的推移很多人也是愛上了IG的風格，IG的比賽總是能夠讓人感到酣暢淋漓，因為結束的太快網友們也給IG一個用不加班的稱呼。

小編認為一般選手在自己的職業生涯中打出一次兩次經典操作就已經很不錯了，但是TheShy就十分奇怪，越是世界大賽他越能打出讓人匪夷所思的操作，最後和隊伍一起成功拿到了LPL賽區的第一個S賽總冠軍，小編髮現奪冠後哭得最厲害的是Rookie，在隊友都沉浸在難掩的喜悅中時，TheShy第一時間去擁抱Rookie，這一抱在小編這種糙漢人看來都十分溫暖，奪冠後的IG戰隊也是吸粉無數人氣飆升。

小編認為TheShy完美的詮釋了電子競技不斷超越的涵義，小編覺得TheShy的操作像是一部看不完的電視劇，上一集是惡魔，這一集是天使，網友們永遠不知道TheShy在未來還會給觀眾們帶來什麼名場面，現在的LPL處於轉會期，小編相信IG戰隊也會有所變動，不管怎樣，小編相信TheShy在未來還會讓我們看到更多的英雄聯盟的魅力，小編認為這或許就是TheShy蟬聯最受歡迎選手獎項的原因吧！以上就是小編總結的TheShy選手的電競歷史，網友們還知道更多的嗎？知道的小夥伴可以評論留言告訴小編，更多遊戲話題以及遊戲熱點都可以在評論區中繼續交流與吐槽哦！小編每天分享精彩遊戲內容資訊以及好玩的遊戲推薦，感興趣的網友也可以關注一下小編哦！
