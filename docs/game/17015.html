王者榮耀：總決賽首發公布，麟羽重回首發，雙方對位全面分析

2019-12-14 559條評論 4,438次閱讀 3313人點贊

各位王者玩家大家好，我是小雨。2019年KPL秋季賽總決賽即將拉開帷幕，「四冠王」QGhappy與「銀河戰艦」AG超玩會之間將會上演火星撞地球般的激烈對決。從賽前的各方支持率來看，QG的勝算會大一點，畢竟本賽季他們兩次擊敗AG，僅負兩小局，而且雙方歷史對話，QG也從未輸過。目前雙方的首發陣容已經公布，麟羽重回首發，小雨按照對位來給大家全面分析一下。

中路 末將VS老帥 64開

這是一組新老頂級中路的對決，將與帥的較量。「海膽大魔王」末將榮獲本屆常規賽最佳中路，婉兒五殺令人印象深刻，順風秀的起飛，逆風可以穩住節奏，雖然季後賽的表現不如常規賽亮眼，但依然是隊伍的核心之一。老帥還是那個中單萬花筒，在季後賽中越戰越勇，優勢的時候也可以越塔強殺，但劣勢的時候往往比較乏力。這對新老對決末將稍佔優勢。

​對抗路 飛牛VS六點六 55開

本來兩個人不算是一個位置，一個戰邊一個坦邊，不過由於版本更新，一諾來到射手位，飛牛的對位大概率是六哥。4屆FMVP飛牛在本賽季季後賽重新展現出強大的統治力，局最佳源源不斷，六哥的曜和狂鐵等戰邊現在也是玩得風生水起。雖然小雨不太看好六哥能單殺飛牛，但是飛牛要想管住六哥也不是容易的事情，兩個都比較穩得邊路55開。

發育路 刺痛VS一諾 46開

刺痛也是QG的四冠功臣，不過他的位置沒有飛牛那麼穩定，還一度坐上替補席，後來刺痛憑藉自己的實力重回首發，並且在很多場比賽中都有亮眼的發揮。不過刺痛在打勝者組決賽的時候狀態不是特別好，這也是QG的隱患所在。他的對手一諾不用多說了，天秀之子，聯盟最強發育路，雙方都正常發揮的情況下刺痛也就能46開一諾，如果一諾再有驚艷表現，恐怕刺痛很難招架。

​輔助 770VS愛思 55開

兩位輔助不僅要自己操作，而且都身兼隊內指揮重任，到時候他倆的決策和應變將會主導比賽的走勢。大家也都知道，770一直比較擅長蔡文姬等軟輔，盾山等英雄的操作不算頂尖，不過愛思的操作也是時好時壞，不算太穩定。如果兩人都能發揮出正常水準，在指揮方面小雨還是更看好770的。

打野 GiaoVS麟羽（藍柚）73開

本場比賽麟羽重回首發，大概率會打藍領野，跟Giao對位。Giao哥本賽季的表現已經無需多言了，在當前的版本中，誰能控制住資源，誰就有更大的優勢，馴龍高手、最佳藍領野就是他。麟羽和藍柚兩人雖然也漸入佳境，但是在控龍方面對上Giao恐怕占不了什麼便宜，只能寄希望於他們多參加團戰抓抓人了。

總體來說小雨更看好QGhappy奪冠，大家怎麼想的呢？歡迎留言告訴小雨。
