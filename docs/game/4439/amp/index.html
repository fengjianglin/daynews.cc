哈嘍，大家好！我是九零小哥，私信「地獄火」，即可免費領取皮膚。一般情況下，打王者榮耀，只有在玩射手的情況下，玩家們才會選擇出攻速裝，並且這個攻速還是有上限的，所以當你攻速疊滿的時候，其實也不是特別的快。但是，這次小提莫給大家帶來了非常有趣的一個畫面，那就是把英雄的攻速提升20倍，看看會是什麼樣的效果。

首先我們看看伽羅的攻速提升到了20倍的時候，是什麼樣的效果？從圖中我們可以看到的時候，這個伽羅是開了1技能的。眾所周知伽羅開啟了1技能，攻速是會有下降的。但是提升了20倍之後，下降的感覺也體驗不出來了，那普攻的特效都快連成了一道線了，非常的厲害，像是一道藍色的激光。

相信玩過安琪拉的玩家都知道，她的普攻特效其實是一個火球，因為她是一個法師英雄，所以不能出攻速裝，而且傷害都是靠技能打出來的。當安琪拉的攻速提升了20倍的時候，安琪拉也可以出AD裝了，快速的丟出了多個火球，讓敵人猝不及防，看上去也是非常鬼畜。

眾所周知，后羿是出場率最高的一個英雄，且勝率也是蠻高的。這個英雄之所以這麼受歡迎，就是因為簡單暴力。原本出了攻速下的后羿，射速就是非常快的，當攻速提升了20倍之後，再開個1技能，直接是射出了三道黃色的激光，非常的勁爆。

當然，最有趣的其實還是妲己。眾所周知，妲己的大招是一個鎖定性的技能，然後丟出多個魔法愛心，給敵人造成高額的傷害。但是，當妲己的攻速提升了20倍之後，妲己的大招完全都可以省下來了。因為她的普攻是鎖定敵人的，並且普攻特效還跟大招特別像，如果出AD的話，相信傷害肯定不比她的大招要差。

相信在座的網友在看到了這一幕的之後，都會有這種感覺，這不就是妲己在手動放大招嗎？對於攻速提升20倍一件事，你們覺得哪個英雄才是最鬼畜的呢？
