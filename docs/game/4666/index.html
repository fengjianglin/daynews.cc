遊戲的發展，當年提出的構思總以為不能實現

2019-12-06 0條評論 20次閱讀 0人點贊

要說遊戲的發展速度還是很快的，從最早雅達利的4位機開始，經歷了任天堂8位機的FC，再到16位的世嘉遊戲機，後來的土星和索尼ps one為32位，又到了任天堂的64位遊戲機N64，到了次時代的ps2、ps3、ps4就不再以CPU時鐘周期來區分了，那麼我們的遊戲發生了巨大的變化是值得肯定的。

曾經記得在準備發售ps2之前，索尼就發表了未來遊戲的發展暢想，那時候聽起來很神奇，描述中提到未來的遊戲背景中的諸如天空中的飛鳥、陸地上的家禽，不再是按照設定好的軌跡規則運動，而是採用一片自由區域讓運動物體自己設定軌跡，那也就是說它們的行動都是自由的，不用認為設計它下一秒走向哪裡，多麼的神奇。

可是多年後的今天，遊戲中的背景設定早已不止是這樣了，它們甚至能根據遊戲主角的動作自動做出回應，你追趕它，它就會跑，就像古墓麗影11裡面的老母雞，或者巫師3中的雞鴨，更強大的還有GTA5，主角若追一個路人，那個路人會一直跑，甚至能跑好幾條街。雖然這些最初也是認為編程好的，但是不局限一個固定的軌跡運動。

未來的遊戲會更加擬真話，遊戲中的生活多多少少可能影響到我們現實生活，會不會有一天遊戲發展到電影《頭號玩家》那樣呢？

你玩過的最古老遊戲機是什麼？歡迎小夥伴們來討論吧。
