手掌模擬器生存登陸Steam，5鍵操控5根手指，聯機場面過於搞笑

2019-12-19 1,956條評論 4,143次閱讀 1550人點贊

喜歡沙雕遊戲的玩家們估計對《手掌模擬器》這款遊戲不會感到陌生，由於其極高的操作難度，玩家們聯機時的場景極為搞笑，手雷炸自己和手槍走火等情況頻發。

畢竟玩家需要分別用5個按鍵操控5根手指，在固定一隻手的動作後要切換到另一隻手進行下一步動作，這期間還要配合滑鼠進行各種操作，遊戲玩起來既搞笑又硬核。

雖然有些大佬玩起來行雲流水，但大部分玩家嘗試幾十分鐘都不一定能達成某項任務。

比如最經典的扔手雷，玩家首先需要用一隻手固定住手雷，然後切換至另一隻手去拉引線，但最後扔出去的步驟需要玩家迅速切換至最初的那隻手，揮動並鬆開手指將手雷扔出，期間所有動作都需要滑鼠的配合，只要一個環節出錯，那就會發生自雷的搞笑情況。

戰鬥場面過於沙雕

對於普通玩家而言，遊戲中最容易的動作大概就是按一個鍵將中指豎起來吧（滑稽）。

現在《手掌模擬器》的開發商又做出了一款名為《手掌模擬器：生存》的多人生存冒險遊戲並登陸Steam，想想在《手掌模擬器》中那些難以做出的動作，遊戲日報君認為這6元生存遊戲恐怕也相當難以駕馭。

玩家們一開始什麼都沒有，光是採集樹枝和打火石並點火都是一件需要大家協力才能完成的事，畢竟在現實中能一抓一大把的樹枝，在這遊戲中就像是大Boss般，能拿起一個並運送到指定地點都是一件很不太容易的事。

由於《手掌模擬器：生存》是一款生存遊戲，玩家要做的並不僅僅是生個火，來保證自己不會被凍僵。

伐木、收集石塊、採集食物和建築房屋，這些在普通生存遊戲中是最基本的操作，但在《手掌模擬器：生存》中一切都變得困難起來。

官方也是知道自己的遊戲到底有多沙雕和奇葩，玩家實際玩起來不用太擔心自己的角色會因各種外因而結束遊戲（如果你能撐過第一天的話），跟朋友們進行各種搞笑操作也挺有意思的。

房屋的每個部件都需要你自己加工

不過對於喜歡建造的玩家，《手掌模擬器：生存》並不友好，畢竟房屋的每個木板和石塊都需要你自己小心翼翼地組合，一不小心就會變成奇怪的東西。
