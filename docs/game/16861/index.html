備戰明日方舟主線第六章 這幾位幹員提前準備好 輕鬆腳打通關

2019-12-16 1,152條評論 2,308次閱讀 61人點贊

明日方舟隨著第六章主線活動的臨近，不少玩家在長草期選了積極的備戰和囤積資源，當然對於穩得住的玩家來說自然是到時候需要什麼急速養成什麼，對於憋不住的玩家，總會考慮先連他幾個幹員到時候直接腳打通關，本次就來推薦值得提前準備好的幾位幹員吧。

先說目前第六章可能會涉及到的敵人和特色，首先地形我們無法推測，所以並不知道這次高台位是多還是少，所以我們只能從涉及到的敵人來推測，通過能天使UP、葦草+布洛卡，還有夜鶯進商店這一系列操作，我們基本上可以得知官方是有預謀的，所以高物防法傷小兵+高血量+大量飛機是不可避免的，雖然這種配置對於新人玩家很不友好，但第六章明細那就是針對老玩家的，難度高是自然的。

角峰

面對高法傷的小兵，最靠譜的阻擋還是以角峰為主，雖然之前一直都吹角峰的魔抗盾能力，不過開服到現在其高光的時刻也不是很多，所以不排除這次輪到需要角峰去阻擋的時刻，四星幹員養成成本本身不算高，所以養個精二也不虧，當然這裡也不是強力推薦角峰，可以考慮提前攢好材料到時候再說，反正角峰養成需要的是紫薯條和裝置這種常規硬核素材，打不了到時候餵給其他需要速成的幹員。

白金

非常適合培養的一個幹員，打高甲不拋光的狙擊，針對飛機效果也不錯，和一流的差一點，但輸出手感很平滑，如果第六章敵人真的按照預測這麼出的話，那麼白金各種意義上都很吃香。不過考慮到白金暖機時間較長，所以需要高練度來彌補，養成成本也是很高，當然提前養畢業了日常用起來手感也很好。

藍毒/流星

理論上針對飛機除了能天使之外最靠譜的兩個選擇，考慮到這次飛機的數量應該不會太少，單單一個能天使肯定不夠，所以極其需要藍毒和流星這兩位副狙來減緩能天使的壓力，當然這兩位也算是典型的養了不虧的幹員，平常活動也會經常用到。

星極/拉普蘭德

危機合約的時候就已經高光表現過了，這次依舊不例外，拉普蘭德的混傷+遠程設定很吃香，這個不用多說，基建也是一把好手。星極的特色在於優秀的單切和近衛法傷能力還能玩空降，遇到高物防還有群體法傷的單位，多利用星極玩玩空降減輕壓力也不錯。當然如果這類敵人多了的話可以就不能這麼玩了，作為目前沒有很好上位替代的兩位幹員，早晚要練，現在集中精力去培養不虧。
