女媧補天都要砍它的腿，玩家表示《ATLAS》馴服的巨龜不要太帥氣

2019-12-14 360條評論 3,929次閱讀 1904人點贊

在古典神話著作《淮南子.覽冥訓》中曾經記載了女媧補天的記錄，其中有一段話是這樣記載的，”於是女媧煉五色石以補蒼天，斷鰲足以立四極”從這句話我們看出女媧補天是用一隻大鰲也就是大龜的腿，這隻烏龜到底要有多大才能讓女媧補天？對於這個問題，在Steam上線的海盜冒險生存遊戲《ATLAS》中就有答案，最近一位主播在《ATLAS》中馴服了一隻巨龜，和它的體型相比，人類就像是一隻螞蟻一樣渺小。

在《ATLAS》中，巨龜就像是一艘航母一樣，不管是體型還是血量都是《ATLAS》中頂尖的存在，且負重能力極其恐怖，有極端的玩家就把巨龜當船，用它來在島嶼和島嶼之間航行，不過小編倒是認為這是一件相當有風險的事情，因為巨龜的身上不能拍床，沒有這固定的復活點，一旦發生什麼意外會發生什麼事情都很難說。

不過巨龜的高血量、高負重還是有很大用處的，儘管玩家不能將巨龜當做船來使用，但把它當做貨架來玩還是可以的，只要玩家能在巨龜前放置一艘船當做牽引船就可以了，在牽引船內放上床，這樣玩家就能巧妙的避開巨龜身上無法拍床的問題。此外巨龜在水中的移動速度也不慢，當玩家需要在《ATLAS》中運輸大量資源，且自己的船負重不夠這種方法就可行。

值得一提的是，當玩家乘騎在巨龜的身上時，巨龜還能給玩家增加一定量的氧氣，這也讓巨龜在《ATLAS》中有著生物潛艇的稱號，玩家也可以用這個特點做點文章，比如說在PVP模式裡面靠著巨龜運玩家偷襲敵人。對於這個大傢伙，大家有什麼看法？要是你也想馴服它的話，可以一起來《ATLAS》馴服巨龜哦~
