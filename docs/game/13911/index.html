《荒野大鏢客OL》賞金獵人任務打法心得

2019-12-12 1,423條評論 2,422次閱讀 492人點贊

《荒野大鏢客OL》遊戲中玩家將會面對一些賞金任務，其中有些賞金任務是比較難的，可能會讓玩家面臨一些困境，下面帶來玩家「魯肅先生」分享的《荒野大鏢客OL》賞金獵人任務打法心得。希望能為各位玩家帶來一些幫助。

一個$標記，一個目標：

　　刷出黃點，提示前往「營地」「藏身處」，這種情況你抵達目標的時候目標就在營地里了。開槍驚動敵人之後目標一般會逃跑。如果逃跑不要戀戰，直接追擊目標，投擲繩索、困獸鎖捆綁之後再迎擊追兵。或者直接投繩索拖倒然後拖走。如果沒有逃跑直接清光敵人再抓，注意不要把目標打死了。

　　刷出黃點，提示前往任何不是「營地」或者「藏身處」的目標，這種地方會讓你先在附近埋伏，等待目標出現。可以選擇在主幹道等待，目標會刷在臉上。或者目標出現之前清光守軍，然後再抓目標。但如果選擇清光守軍，一定不要下馬，因為目標一般是騎馬的，而護衛一定會嘗試把你的馬射跑或者把你從馬上打下來。一次上馬失敗目標肯定會跑。

　　刷出紅點，這種情況一般是兩輛馬車，一輛全是人，一輛帶機槍，兩個騎馬的護衛。機槍射界很窄但也不是沒有，不要從後面接近硬剛。目標一定位於頭一輛馬車的副駕位置。從側面接近，然後馬上直接投出繩索，不要下馬，向垂直於車隊前進方向的方向拖拽前進，直到提示「死亡騎手」「擊暈」，然後方可下馬迎敵或者搬起來放在馬上

一個$標記，兩個目標：

　　此類任務罕見

　　黃點與單目標相同。

　　如果是紅點，一般是4個馬匹護航的一輛馬車，沒有機槍，一個目標是馬車主駕駛，一個目標是副駕。一個人全部活捉比較麻煩。費事錢也少。

$$，一個目標：

　　開局黃點：

　　現在開始會出現追蹤類型，新奧斯丁如果出現「仙人掌清泉」「賴利陣地」一般都是這種類型（但「岔路酒館」則是埋伏等人）。這種類型不需要戰鬥，只要開鷹眼，檢查線索-順著軌跡找就可以了。如果有多個線索、多條軌跡，每條軌跡都會計算一次「已用鷹眼追蹤賞金目標」的挑戰。如果沒有每日挑戰，其實可以圍著大黃圈騎馬跑，一般人都會刷在房子邊或者山崖下面。但如果拖到就剩五六分鐘還是找不到，還是按照軌跡找吧……

　　有時候也會出現埋伏（待確認）

　　開局紅點：

　　紅點出現一般都不會太遠，一般是由目標+兩個騎馬護衛快速移動，他們的移動目標一般是像布商堡這種地方，而且裡面已經有守軍。如果目標進入堡壘，戰鬥會很難打。而且像布商堡這種有前後門的，目標一定會從戰鬥一開始就向後門逃走。但是一般都可以抄近路，直接用套索或者困獸鎖截殺。不用戰鬥。

$$，兩個目標、四個目標：

　　黃點：一種也是探案，和一個目標一樣。兩個或者四個目標一般是刷在一起的。配合賞金獵人車可以單刷。

　　黃點：一種是埋伏，一下子會刷出來兩個人或者四個人，而且是從不同方向過來的。就算你是團隊也需要開語音密切配合才能搞定。單刷基本沒可能。就算打死幾個也極其容易逃掉N個人。

　　紅點兩個目標：和一個目標一樣，1個人+2個護航，區別是目標要前往的堡壘里會有另外一個目標。可以單刷，難度很高。

　　紅點四個目標：極為罕見，貌似我見過的就是2個人騎馬，前往另外2個人所在的堡壘。極為困難。

$$$，一個目標：

　　黃點：有可能是追蹤探案。

　　黃點：還會出現一種新類型「騷亂」，前往黃點之後再裡面殺人，殺夠一定數量之後目標會騎著馬和另外兩個騎馬的護衛一同出現。這兩種都很好搞定。

　　紅點：也會出現1+2，前往堡壘的樣子。

$$$，兩個目標：

　　黃點：和$$一樣。

　　紅點：和$$一樣。

$$$，六個目標：

　　黃點：追蹤探案。或者埋伏。

　　紅點：沒見過。
