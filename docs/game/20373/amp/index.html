好消息！電競奧運會來了？前奧委會負責人：明年1月在新加坡舉辦

天天要聞
2 天 ago

前段時間一直傳言說電競圈將有非常大的好消息公布，不少人猜測是什麼轉會以及俱樂部的事情，但是實際上是牽涉到了整個電競行業的發展。就在12月16日中午，毒奶王黃旭東剛在微博上爆料馬上好消息公布後，人民電競官博就第一時間發布了這個消息，原來是電競奧運會要來了。

​根據人民電競微博透露，全球電子競技聯合會將於明年1月舉辦全球比賽，國際奧委會前負責人將擔任這次比賽的主要負責人。

既然提到了國際奧委會，那麼這次的全球性質的比賽，大概率就是以國家的形式來參賽的。目前電競圈比較紅火的項目，比如lol、王者榮耀、dota2、csgo和星際等遊戲都有機會參賽。而上一次全球性質的電競比賽，還要追溯到2018年的雅加達亞運會。當時我們lol分部就拿下了世界冠軍，不過當年這個陣容在明年1月的比賽中應該看不到了，因為letme和MLXG都已經退役了。

​對於lol項目來說，大家最關注的肯定是陣容的選擇，這個應該會有LPL賽事官方的人來決定，相信選出來的陣容，也不會太讓大家失望，只不過大概率會出現撕逼事件，因為之前亞運會的時候，中單到底是小虎還是xiye，曾引發了粉絲大戰。以電競圈粉絲愛吵架的特質來看，這次應該也不會出現例外。

另外就是明年1月份就要舉辦了，也就是說距離現在只有一個多月的時間了？這麼匆忙的行程，不知道大家趕得急不，而對於lol國家隊的競爭力，目前來看估計也就中韓比較靠譜一點，LEC賽區那邊如果按照國家來比賽，實力會大大削弱，其實最後的看點也就是中韓大戰了。

​那麼，對於這波電競圈的大事件，你們有什麼想說的嗎？
