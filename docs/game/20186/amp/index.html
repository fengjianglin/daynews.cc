致敬三幻神的三幻魔，他們是否配得上遊戲王神卡地位？

天天要聞
2019-12-18

初代遊戲王動畫《遊戲王怪獸之決鬥》（簡稱遊戲王DM）中出現過的強大的「三幻神」——分別是天空之神奧西里斯的天空龍、大地之神歐貝利斯克的巨神兵以及太陽之神太陽神的翼神龍。

而《遊戲王GX》作為遊戲王系列的第二部動畫，也致敬經典的三幻神，創作了「三幻魔「三張同樣強大的怪獸，不過在知名度上，相比三幻神，則略有不如。

今天萌主就為大家介紹一下這三張堪稱能與三幻神匹敵的怪獸卡吧！

神炎皇烏利亞

動畫《遊戲王GX》登場的「三幻魔」之一。原型為動畫《遊戲王》系列中「三幻神」的「奧西里斯的天空龍」。

等級：10 種族：炎 屬性：炎

攻擊力：0 守備力：0

特殊召喚/效果：這張卡不能通常召喚。把自己場上3張表側表示的陷阱卡送去墓地的場合才能特殊召喚。

①：這張卡的攻擊力上升自己墓地的永續陷阱卡數量×1000。

②：1回合1次，以對方場上蓋放的1張魔法·陷阱卡為對象才能發動。蓋放的那張卡破壞。不能對應這個效果的發動讓魔法·陷阱卡發動。

在遊戲王GX動漫中：首次登場在影丸理事長與游城十代的決鬥中，影丸為了重獲青春而吸取卡牌怪獸的力量。在決鬥中影丸把場上覆蓋的「融合失敗」「懲罰遊戲」「黃金的邪神像」送去墓地特召這張卡，後來又通過把「連鎖炸藥」「強欲之瓶」「公墓炸彈」送入墓地讓這張卡反覆從墓地蘇生並提升攻擊力。無論從卡圖還是效果來看，完全是在致敬「內奸龍」之稱的天空龍，雖然可以用三張陷阱卡召喚，在那個沒有過多的套路，有大量陷阱卡作為「坑」的年代，召喚出來很是輕鬆，雖然現在也很容易，但是卡圖實在沒有天空龍傳神。

這張卡的原型是《聖經》提到的四大天使之一的「烏列（ウリエル/Uriel）」

降雷皇哈蒙

動畫《遊戲王GX》登場的「三幻魔」之一。原型為動畫《遊戲王》系列中「三幻神」的「拉的翼神龍」。

等級：10 屬性：光 種族：雷

攻擊力：4000 守備力：4000

特殊召喚/效果：這張卡不能通常召喚。把自己場上3張表側表示的永續魔法卡送去墓地的場合才能特殊召喚。

① ：只要這張卡在怪獸區域守備表示存在，對方不能選擇其他怪獸作為攻擊對象。

②：這張卡戰鬥破壞對方怪獸送去墓地的場合發動。給與對方1000傷害。

同樣出現在遊戲王GX中游城十代與影丸理事長的決鬥，雖然也是致敬太陽神翼神龍，但是效果卻沒有像另外兩張幻魔一樣與幻神有所聯繫。

這張卡的原型是所羅門七十二魔神的第七魔神「亞蒙（アモン/Amon/Aamon）」

幻魔皇拉比艾爾

動畫《遊戲王GX》登場的「三幻魔」之一。原型為動畫《遊戲王》系列中「三幻神」的「歐貝里斯克的巨神兵」。

等級：10 屬性：暗 種族：惡魔

攻擊力：4000 守備力：4000

特殊召喚/效果：這張卡不能通常召喚。把自己場上3隻惡魔族怪獸解放的場合才能特殊召喚。

① ：1回合1次，把這張卡以外的自己場上1隻怪獸解放才能發動。這張卡的攻擊力直到回合結束時上升解放的怪獸的原本攻擊力數值。

② ：對方對怪獸的召喚成功的場合發動。在自己場上把1隻「幻魔衍生物」（惡魔族·暗·1星·攻/守1000）特殊召喚。這衍生物不能攻擊宣言。

這張卡牌也是在致敬巨神兵，相對於前面兩張卡牌，可能是顏值相對前面兩張幻魔好看了許多。

這張卡的原型是《聖經》提到的四大天使之一的「拉斐爾（ラファエル/Raphael）」

神炎皇的效果相比動漫中可謂大打折扣，只能算作二流。而降雷皇雖然對應的是「太陽神的翼神龍」但卻沒有球體形態和不死鳥形態，技能也很雞肋。幻魔皇在動畫中號稱最強幻魔，但實卡後的效果，如果說降雷皇的實卡效果雞肋的話，那幻魔皇的就更加沒用。

相對於「三幻神」來說，無論從卡牌設計還是效果來看，「三幻魔」遠遠不如前者，所以知名度也不如他們
