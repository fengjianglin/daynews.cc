夢幻西遊62期：嘉年華曝光，新門派還是新寶寶？獸決C66或將暴漲

2019-12-03 0條評論 19次閱讀 0人點贊

摘要

各位觀眾大家好啊~【點關注不迷路】隨著資料的更新迭代，夢幻西遊的策劃們正在想盡辦法把遊戲變得更平衡一些。不管是之前的三界功績的大改–為了削弱工作室的收益，從而將工作室趕出夢幻。

各位觀眾大家好啊~

【點關注不迷路】

隨著資料的更新迭代，夢幻西遊的策劃們正在想盡辦法把遊戲變得更平衡一些。

不管是之前的三界功績的大改–為了削弱工作室的收益，從而將工作室趕出夢幻。

還是後來的千e獸決大改既打擊了刷子DL工作室，也讓武神壇的PK變得更平衡了

珍寶閣雙咒師PT一家獨大的局面已經在本次PK賽事中被「二狗」的獅駝，女魃墓陣容瓦解了。

那麼隨著千億獸決的更改，更多的神馬浮雲會被供給出來，珍寶閣的硬體也實際上是在被相對的弱化了。當然這樣的改動無疑受傷最大的就是剛剛入手神馬獸決的「爆爆總了」。

然而！！

新的維護想必大家也看到了，最最重磅的消息應該就是嘉年華的最新料片造型了。

新召喚獸造型or 新門派？

從這一張陰影圖來看。

這個造型非常的瀟洒，飄逸。

置於這是一個召喚獸還是一個任務造型，都有很多的猜疑和說法。

有的人說這是新門派月宮，仙族的法系門派。

也有人說這是第三隻天女，攜帶了新的技能（如類似物理須彌）

筆者認為，這個造型更加趨向於是一隻新的召喚獸。

為什麼？以下是筆者的分析。

1.首先，細看，你會發現這是一個女性的單位，長發有點類似仙女飄逸的感覺。

2.其次，這個造型雖然武器刻畫的很細緻，但是從整個單位的細節來看不是特別的細膩，可以說有些類似修羅傀儡妖的感覺。

3.而單獨出一個新門派和一個新角色，單從這一角度來看也是不太現實的，因為看之前新門派和新角色出來的時候都是可以供選擇的，畢竟如果只出一個門派那麼必然會導致蜂擁而上的節奏，這是不利於遊戲健康的。

4.從夢幻西遊的發展來看，打擊出去了DL工作室和刷子們，那麼剩下的任務就是把遊戲的庫存或者說積存的產出物品和遊戲幣消耗光了（如C66，獸決這些），當對來說，現在各大區的獸決價格都在上漲了，因為就是考慮到了新的召喚獸出來了大家要煉妖，C66的價格也沒有繼續無限制的下跌了，我們區穩定在了23W。各個任務中爆落小獸決的幾率是相對的越來越低了，所以理論上來說物以稀為貴，小獸決的價格必然是要上漲的。

5.最後也是最重要的一點，這個時間點在年尾，要放寒假了，這樣的新資料片出來，必然是回收物品的產出和夢幻幣的，是為了迎接寒假玩家的迴流，所以不管是從遊戲內在還是外在來看，這個新造型必然是新的召喚獸。

那麼！！你的獸決和C66都準備好了嗎？五開玩家現在C66和獸決就不用賣了，等著漲價吧。

深海狂鯊，同行禮幣~每一隻鯊魚都是親自打撈，禁止代捕~~=.=

本次的嘉年華禮幣取消了代領代購的現象。

可以看到本次的鯊魚，每一隻都是和個人綁定好的，需要二代身份證和賬號綁定的身份ID，人，證，ID三者一致才可以認領你的鯊魚。

那麼這樣的改動就再也不會出現水軍代領牟利和非夢幻玩家的黃牛們佔領嘉年華現場影響氛圍了。

本次的鯊魚售價也是非常的良心，只賣2580仙玉（258R）

需要注意的是以下這一段話：

「盛典共慶禮幣於12月03日-12月05日每日12:00和20:00會放出一定數量進行搶購。自12月06日不再每日增加禮幣數量，但如有未售賣完的禮幣，則會繼續進行售賣。」

可以看到的是，深海狂鯊實際上僅僅是在03日的12:00到05日的20:00，就結束了了。

那麼也就是說，一共只有6個時間點就完成了售賣。雖然說之後的天數可能會補售。

但是憑深海狂鯊的受歡迎度，幾乎是不存在沒賣完的情況的。

所以這裡說的沒售完的應該指的就是依靠腳本或者連點器獲得深海狂鯊的渠道的這一部分鯊魚的退貨了。

搶鯊魚！還是要趁早啊。

最後，不知道大家對鯊魚是否還有追捧的興趣呢？

新門派或者新造型的召喚獸你都會回歸夢幻來香一把么？
