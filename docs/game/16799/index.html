iG買到了好輔助，為什麼不把Baolan賣了？原因其實很簡單

2019-12-16 49條評論 2,614次閱讀 99人點贊

看了這麼久比賽，相信大家都明白，當一支戰隊出現明顯短板時，是不可能拿冠軍的。以iG為例，S8 iG首發五人都處於巔峰期，即便是個人實力最弱的Baolan，也沒有成為隊伍突破口，所以他們拿了世界冠軍。反觀S9，iG野輔拉胯之後，即使3C再給力，也不過是堪堪將隊伍抬進了四強，一遇到FPX，就將所有問題都暴露出來了。

S10全球總決賽在上海舉辦，粉絲非常希望，iG能在轉會期有所動作，不說野輔全換，至少也要把Baolan抬下去，給Jackeylove配個正常的輔助。結果iG管理層向大家證明了，他們在令人失望方面，從來不會令人失望，這一次他們做出的騷操作，堪比TheShy斷手事件！竟然在阿水給足了壓力情況下，強行留Baolan，導致阿水成為自由人。

Jackeylove的態度很明確，iG有Baolan沒他，有他沒Baolan，管理層沒第一時間做出正確抉擇，實在離譜。作為一名不佔用外援名額的一線選手，阿水在轉會期受到了各大戰隊追捧，這下輪到iG管理層傻了，本來他們還以為JKL只不過是鬧情緒，最後還是會續約。如今形勢逆轉，JKL掌握了主動權，以他的條件想找個人員配置好，簽約費又高的戰隊，不算太難。

形勢所迫，iG管理層不得不低頭，他們應JKL要求，給他找來個不錯的輔助，那就是VG輔助南風。目前iG官方還沒有官宣這個消息，但已經被實錘，因為南風突然關注了iG官博，並且到上海去了，還發表了一則動態。更為靠譜的是，圈內權威賽事網站OBGG，已然將南風歸為了iG戰隊選手，而且據蘇小落闢謠，阿水還沒有離開iG基地，他顯然是在等輔助到位，才肯續約。

既然iG買到了好輔助，為什麼不把Baolan賣了換點錢呢？這是很多人心裡的疑問，個人覺得原因其實很簡單，王思聰親自和Baolan簽了3年合同，哪支戰隊願意接盤呢？賣不出去啊！
