《陰陽師》文明迪賣號高達43萬！小伙充值200萬說小菜一碟，給跪

天天要聞
5 天 ago

大部分遊戲都是會有人民幣玩家。人家花錢買更好的裝備，或者是買更好的皮膚，或者是買通關秘籍等等。在遊戲中氪金也是一個非常常見的現象。而且做氪金玩家似乎比普通玩家更有優越感，所以很多玩家都會選擇在遊戲中氪金，來提高自己的遊戲體驗感。

《陰陽師》也有很多神壕，文明迪可以算一個。不過近日，該玩家的賬號上架到了藏寶閣中。藏寶閣是陰陽師的官方交易平台。而且該賬號的價值高達43W，真可謂是氪金玩家中的神豪了。

要知道，文明迪培養自己的這個賬號得花上百萬，而迪迪用43萬就直接賣出去了。其次，文明迪的行為也反映了他可能會退出陰陽師這個遊戲。這對陰陽師這個遊戲來說肯定是一個損失。畢竟這麼豪的遊戲玩家也是很少見的。而網易的官方也做出努力，他們將文明迪的賬號延遲上架，希望可以拖住文明迪回心轉意，繼續玩遊戲。不過文明迪也並不買賬，堅持了好幾周，總算將自己的賬號給上架了。

有不少玩家都去藏寶閣去圍觀文明迪的賬號。不出所料，文明迪賬號裡面基本上達到的數據量都是非常驚人的。基本上很多的角色都是六星式神，式神的收集度很高。除了在預防方面的一速二號位，稍微差一點之外其他地方都很強，基本上是屬於現版本中的最強的級別。相信這遊戲以後也很少會有再超過他的賬號了。因為這裡面不僅需要氪金，而且還需要花費很多的精力來培養。

不過像這個賬號一般的玩家，也就是看看了，真正要買的是土豪們考慮的事情。雖然網易在遊戲保值方面做得還可以，但是遊戲向來都是日新月異，等到再出幾個新的式神，再出一些新的玩法之後，這個賬號貶值也是很快的。

對於文明迪把自己的賬號給賣了的事，也不能從中反映出這個遊戲怎麼樣，或者是玩家處境是什麼樣的情況。因為對於土豪來說，這可能就是一件很普通的事情。

就像之前看到一個報道，一個記者採訪一線城市一個男生問他在遊戲裡面充了多少錢，那個玩家張口就說200多萬。記者瞬間驚了，說家裡人難道不會有什麼意見嗎？那個男生就說才200多萬，有什麼意見？行八，有錢人的樂趣我是不太懂啦……對我們來說很壕的事情，對他們來說可能就僅僅只是一件小事吧。
