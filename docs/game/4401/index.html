《黎明殺機》上線鬼武者，強大技能，讓獵物感覺害怕

黎明殺機新資料片目前已經上線，為玩家帶來新地圖怒火聖地，還有增加倖存者，還有鬼武者殺手等等，這款遊戲需要本體才能運行，國內的售價為28元，也可支持簡體中文，所以從目前為止還是備受好評的。

鬼武者殺手其實之前也是一名武士，但是在遭到腐化之後，也開始幻化成為心魔陰影，如此醜陋的憤怒生物也是殘忍和怒火的化身，讓所有的獵物都感覺到害怕。鬼武者在傷害倖存者之後，也能夠吸收受傷者的血球，一旦收集足夠之後就可以進入到血腥狂怒的階段。啟動之後，直接就可以把武士刀收集，並且拿出鬼精棒，還可以解鎖一些強大的技能，比如惡魔之機或者惡魔衝擊等等。

研發團隊也發現鬼武者殺手，本身就有著強大的誠心策略技能，可以看見附近的氣場，還具有鮮血迴音的技能，當倖存者被抓之後，在短時間內所有受傷者都會出現疲勞狀態影響。至於宿敵技能可以讓鬼武者殺手，直接就把眩暈的倖存者變成自己的祭品，而當新的倖存者直接變成祭品時，也可以在短時間內對鬼武者殺手所有的感知都變得遲鈍，氣場在短時間內就會直接被揭露出來。

木村唯也是一名摩托車手，為了生存而不斷選擇逃命，也具有很多技能，比如突破，不擇手段，還有好運連連等等。好運連連往往是在受傷的時候啟動，基本上一段時間內都不可能會留下血跡，而不擇手段也可以將掉落的木板重置，突破是當被抓住之後，可以快速的提升移動的速度。

新地圖怒火聖地也直接被設為遭遺棄的神詞，一旦陌生人進入後就沒有辦法控制，也會用力的砍向雕像，就算離開之後也會有著腐敗的一種味道。
