天天要聞
2019-12-20

小夥伴們大家好，我是小數點。

相信很多人都關注了今年lpl轉會期的這一場，《阿水去哪兒》年度大戲。水子哥jackeylove，可以說是憑著一己之力，讓這個lpl轉會期憋氣大會進化了一個層次。

是的，即使現在lpl轉會期都已經結束兩天了，一切已成定局，卻還是有戰隊扭扭捏捏不發布首發陣容。

而水子哥，也依舊不知道會去哪家戰隊。

​

不過網友們再著急也沒用，今年的水子哥表現得確實有些奇怪，關於他的轉會消息都可以單獨收集成冊了，但是誰也沒想到，就這時候，關於阿水的轉會動向還能有更離譜的消息！

沒錯，阿水可能哪都不去了！心神俱疲的他可能要直接休息一個賽季再打！

​

爆料出這個消息的人不少，其中就包括pdd。

12月19日凌晨，pdd在下播前就聊到了阿水的事情，而他表示阿水在經歷了s9的一系列比賽後，確實是感到有些累了，可能要休息一段時間才能考慮繼續打比賽的事情。

有一說一，職業選手的壓力確實是很大的，如果水子哥最後做出了這樣的選擇，確實也無可厚非。

你覺得呢？
