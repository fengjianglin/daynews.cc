天刀攻心決無用還有人信 新幫派技能解析

2019-12-25 127條評論 4,104次閱讀 1671人點贊

在新版本【禪宗少林】中，除了少林、蓮華鎮心魔、T9金裝這些大更新外，還有一些小更新也值得我們關注，比如本期要說的——幫派技能。

在版本更新後，「經武弘文—幫派能力」新增了兩項幫派技能：攻心、寧神。

攻心：比例提升攻心訣攻擊。

寧神：永久提升攻心訣防禦。


從技能效果來看，攻心和寧神的屬性加成都非常不錯，而且升級所需花費的修為與幫貢也都不算太多。如果說缺點嗎?這需要的碎銀也忒多了點!

攻心和寧神同等級下升級要求都是一樣的，僅僅是從20級升至21級就需要花費45金碎銀。與之相比，「驚怖」、「火花」這些技能所消耗的碎銀也就是洒洒水啦。

自「心劍戰境」玩法和「竅穴」上線後，攻心訣就成了我們一項非常重要的屬性組成。雖然還是有很多少俠覺得攻心訣沒有多少用處，除了副本和野外有概率觸發特效攻擊外，在論劍和戰場中又沒什麼用處。但目前來說我們對竅穴的開發還只是一個初級階段，效果還不是太明顯。但到了後期，攻心訣的作用一定會愈發明顯。

而且，即使是現在，如果你的攻心訣防禦屬性低於敵方的攻擊屬性太多，那麼你就會更容易被敵人打出特效攻擊。

但目前版本中增強竅穴能力的途徑很少，只有竅穴升級、天心鑄、砭石這三種。

竅穴升級沒有太多好說的，如果偏向PVE那麼就先升級攻心訣，如果偏PVP優先升靜心訣收益會更高一些。

砭石目前只有三種唯一砭石，選擇太少，大家基本也都是副本帶極陽/兩儀，野戰帶兩儀。

天心鑄和附魔類似，而且因為天心鑄屬性不可繼承，所以相比大家更換T9金裝後又要花費一些精力去打造天心鑄了。

而在【禪宗少林】版本中，除了新增攻心和寧神這兩個幫派技能用以外加強攻心訣屬性外，新神兵「百曉生卷」的也能夠增強攻心訣攻擊屬性。

百曉生卷的技能「萬雪之徵」除了可以比例提升攻心訣的攻擊外，隨著神兵傳記的解鎖還可提升風襲、灼燒、雷擊這三種特效攻擊的持續時間/傷害效果。

由此可見，官方其實也在一步步的增加攻心訣的提升途徑。雖然目前攻心和寧神這兩個技能都是在增強攻心訣，但相比隨著日後的更新，加強靜心訣的幫派技能也會陸續上線。
