機會從格鬥硬核概念開始：立回

當比賽開始時，兩名玩家之間的距離相當適中的。雙方都不是處在特別有利的位置。戰鬥開始後，雙方就開始不斷來回走動。這裡帶出一個重要的概念：立回。

立回：對戰雙方在攻擊或防禦之前所做的一切行動。其目的是使自己進入有利的位置，或者使對手陷入不利的情況。下圖右邊的白色豪鬼正在立回，而右邊的豪鬼已經做出攻擊了。

我們在看很多視頻中，硬核玩家一般都是在尋找適當的距離，同時又不斷地出拳腳。這種情況下，一般是在試探對方和找到距離的感覺。然後就會在安全的情況快速打出招數命中對方。這過程就是硬核玩家們所說的立回。這是在《街霸5冠軍版》中最重要的階段，也是你贏得比賽的基礎。這是比賽中最不浮華的部分。不是在沉默中死亡就是在沉默中爆發，每個硬核玩家都在這時候尋找一擊致命的機會。

立回具體的實施

知道你的活動範圍(用你的腳趾)。看起來是一句玩笑話。但實際上硬核玩家們卻真真實實地做這事。

啟用一個新角色首先你要熟悉他的所有動作，包括行走速度，拳，腳，跳躍等等。讓我們以嘉米為例來看看他們的動作。當你測試一個新角色的時候，你要做的第一件事就是檢查每一個動作，並試著去理解你何時何地會用到它們。你可以看到嘉米用她的腳向前伸了很長一段距離。

正如我們之前所說，立回是為了尋找一個相對安全的位置進行攻擊。自己的拳腳經常被用作掩護，這就是我們選擇嘉米的原因。當她伸出她的腿，攻擊判定是她的整條腿。她這種攻擊方法既是攻擊又是掩護。雖然這並不是完全沒有風險，但這已經是一種非常安全的攻擊方式。

最安全的攻擊，就像上面說的，用你最長的手腳——好像在使用維加的情況下，用他的爪子。當你使用這種方法的時候，試著從攻擊範圍的邊緣做出攻擊。這樣可以非常好地訓練你的距離感。優良的距離感往往比連招更具攻擊力。

最新的例子就是Punk用春麗敲擊少年桑的桑吉爾夫。眾所周知，如果你距離桑吉爾夫一定的距離你都可以吃上一套梅花大坐。Punk能做到控制好距離讓春麗的重拳剛好擊中桑吉爾夫。少年桑卻無法近身打上連段或者梅花大坐。街霸5冠軍版的連招等等高中級技巧都需要建立在良好立回的基礎少。那讓我們就重新重視每個招數的攻擊範圍。不斷地強化自己的距離感還有熟悉對手的攻擊範圍。

立回之後你還需熟悉差合。一旦你理解了這些基本招式的最佳攻擊範圍，你就應該開始掌握差合了。

什麼是差合

當你不斷去出招試探對方時，他其實也在做同樣的動作。當對手打出空招（沒有命中）你立刻做出反擊並命中對手，這就是差合。

兩個硬核玩家都非常了解雙方角色的性能。在立回階段尤其重視拳腳的攻擊範圍。他們將會來來回回的出招牽制對方。雙方都在爭奪有利的位置，並試圖引誘對方打出不會命中的一招（空振）。當對手打出空振的招式，收招硬直時間較長時（一般抓住中和重攻擊的差合比較容易）。你要不斷地訓練自己抓對手空擋的意識。先從抓重攻擊間隙開始，然後慢慢進階。

重視差合

街霸5冠軍版是沒有空中防禦的。當你選擇跳入攻擊時，會面臨極大風險。對手可以用對空技輕鬆將你擊落然後形成壓制。為了避免這種情況，硬核格鬥玩家會謹慎選擇跳入攻擊。雙方基本是以地面的立回為主。

地面立回時，只會發生兩種情況，

1）命中

格擋時，可以進入持續壓制或者繼續立回尋找機會。

擊中時，進入發動後續進攻的時機（目壓或者連招）

2）不命中

這就是打出差合的機會了。

  1. 如果你的拳腳夠不著對方，應該前前突進再進行攻擊。
  2. 如果你的拳腳夠得著對方，立即發動攻擊並送出一套連段。

立回與差合熟練地組合起來才能發揮最大威力

在《街霸5冠軍版》里，立回是基礎，差合是進階技能。可以說，差合是建立在有效立回的基礎上。立回要心細，差合要反應快。結合兩者才能在對手露出破綻時給與致命一擊。這時的你再不是上場將按鍵亂按一遍的初級玩家了。你更能從容面對對手的進攻了。
