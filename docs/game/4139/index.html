天龍八部：我超級甜正是脫坑！99萬入手賣價110萬，血賺

2019-12-09 0條評論 13次閱讀 0人點贊

摘要

喜歡玩天龍八部的玩家，大家對於我超級甜啦一定都是非常熟悉的，畢竟作為一個人民幣女神，當初為了玩遊戲，豪擲99萬來賣一個唐門，對於大部分玩家來說不得不說是大手筆了。並且我超級甜啦，先後轉過唐門、丐幫、逍遙等門派，頗受許多玩家的關注。

喜歡玩天龍八部的玩家，大家對於我超級甜啦一定都是非常熟悉的，畢竟作為一個人民幣女神，當初為了玩遊戲，豪擲99萬來賣一個唐門，對於大部分玩家來說不得不說是大手筆了。並且我超級甜啦，先後轉過唐門、丐幫、逍遙等門派，頗受許多玩家的關注。

最近就有玩家看見我超級甜啦正是脫坑，作為天龍八部裝備評分上160萬的超一流的大神，我超級甜啦的一舉一動都頗受大家的關注，從唐門到丐幫再到逍遙，我超級甜啦各個門派都玩得爐火純青，在體驗過各個門派之後最終選擇退出天龍八部的江湖，賣價110萬，一時之間成為許多玩家茶餘飯後的閑聊話題。

目前天龍八部官方更是更改了規則，以前的賬號捨得價格是100元到99萬人民幣，現在可以是999萬人民幣，有的天龍八部不仔細的玩家，還以為是最高為99萬，不知道我超級甜是怎麼設定的。

當然我超級甜啦的前身是一個164萬裝備評分的唐門，由於當初最高佘設定價格為99萬，現在沒有限制提高以後，我超級甜啦轉手售價110萬，玩3個月還賺10萬，立即就有天龍八部的玩家表示不理解，畢竟當初評分有164萬，現在僅僅只有160萬評分！對此，你是怎麼看的呢？
