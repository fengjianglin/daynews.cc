[摘要]在王者榮耀之中，有很多戰士英雄在後期強勢，前期需要發育，就像李信一般，中期特彆強，但也有一部分戰士是反著來的，那麼本期筆者就來和大家一起說一下前期很強勢的英雄吧！

首先第一個就是我們的鎧皇了，這個英雄在峽谷之中絕對是頂尖的存在，備受小夥伴的喜愛了，但是在前期的時候他雖然能夠壓制很多戰士，但是想要完成擊殺或者碾壓，卻基本上很難做到，尤其是四級之前！

呂布這個戰士英雄在前期的時候雖然還不錯，續航能力十足，而且手比較長，但是遇到一些強力戰士，依舊只能猥瑣發育，根本就打不出去，更何況是壓制了！

第三個則是我們的橘右京了，這個戰士英雄在前期的時候非常強勢，尤其是兩級的時候，普攻二技能加上一技能的兩段位移，因此在前期對線，消耗能力極強！

最後一個就是我們的狂鐵了，狂鐵對於能量的需求是非常大的，但是一級對面的戰士絕對沒有壓制的能力，因此能夠很好的利用兵線來積累能量，敵方敢靠近就給其一下，消耗一下到達兩級之後，就能擊殺了！對此，大家有何看法？歡迎討論！
