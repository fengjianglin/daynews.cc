遊戲王：最初的禁卡，你想不到這張卡也被禁吧？理由很多

2019-12-11 3,203條評論 4,590次閱讀 875人點贊

大家好，我是團團圈。今天我們來聊一下遊戲王最開始的禁卡，如果玩過的話，應該都見過，不過你想不到這幾張卡為什麼被禁。

注射天使 莉莉

注射天使 莉莉

卡牌效果：這張卡進行戰鬥的傷害計算時只有1次，支付2000基本分才能發動。這張卡的攻擊力只在那次傷害計算時上升3000。

首先讓你想不到的就是這張注射天使莉莉，為什麼這張卡會被禁呢？因為這張卡在那個時代支付2000生命，就可以達到3400攻擊力。要知道青眼白龍也就3000，而且這張卡也是少有的在對方回合戰鬥也能發動的卡，無比暴力。從2004年3月1日第一張禁卡表光榮入表以來，到了2004年9月1日才被改為限制卡，而且直到2007年3月1日才從禁限卡表走出。雖然莉莉是現在很多人看都不會去看的卡，但是她曾經的強大不能抹殺。也許多年以後，我們現在的禁卡，也會有出獄的一天，不過就是不知道效果還是不是原來的效果了。

八汰烏

八汰烏

卡牌效果：效果·靈魂：這張卡不能特殊召喚。召喚·反轉的回合的結束階段時回到主人的手卡。這張卡給予對方玩家戰鬥傷害的場合，跳過下次的對方的回合的抽卡階段。

這張卡的效果，打了對面就跳過對面下回合抽卡，想想就覺得厲害。你看那個刻之封印，從2006年3月1日進小黑屋就沒出來過就明白，沒有抽卡就相當於對面沒有了希望。

這張卡中途越獄出來過一次，出來之後就和混沌帝龍還有三眼怪一起靈魂舞蹈，混沌帝龍炸雙方場上和手牌，三眼入墓地到手八汰烏，攻擊，對方不能抽卡了，直接遊戲結束。就是這個原因，導致混沌帝龍和三眼一起走了，現在效果都改完了出來，不過八汰烏是不可能出來了。

閃電旋渦

閃電旋渦

卡牌效果：丟棄1張手卡發動。對方場上表側表示存在的怪獸全部破壞。

雖然這張卡只是萬年禁卡雷擊的一個超級低配，一是要丟一張卡手，二是只能破壞表側表示的怪獸，但是在那個時候，這張卡已經是厲害得飛起了，根本不能接受，所以就進去了。不過現在，有太多反破壞以及破壞抗性，這張卡早已經不足以站在禁卡表當中。

纖維壺

纖維壺

卡牌效果：反轉：雙方把自己場上的卡，手卡，墓地的卡和卡組合在一起洗切。之後各自從卡組抽5張卡。

只要沒掉血，就相當於重頭來過，一局遊戲我要看幾次你們纖維壺重來？這張卡不禁太噁心，弱勢方擺出來發出效果直接等於重頭來過，別說對手噁心，就是看比賽打牌的觀眾也噁心，所以這張卡在被禁之後也是永遠出不了了。

從上面幾張當時第一代禁卡來看，當時的製作師還是挺敢做的，什麼強大效果都做得出來。不過現在也不差，現在的卡各種強大，各種防止破壞，各種連鎖，已經不比當時的一些禁卡差了，不過雙方都一樣強，還是有策略可言，剩下的也有運氣了。

以上就是本期的初代部分禁卡說明，不管哪張卡被禁止都是有原因，總有無數決鬥者被其中一兩張坑的心態爆炸，然後最終導致這樣。感謝大家收看，如果喜歡的話，可以關注一下，我們下期再見，拜拜。
