和平精英：2個小冷門試驗，水下可躲避空襲，南瓜王不能被雷擊？

天天要聞
2019-12-15

各位親愛的吃雞小夥伴大家好，光子開放新模式之後，玩家對遊戲的探索一直沒有停止過。經過這麼長時間的遊戲，還是有不少人對遊戲中許多知識都保持好奇的態度。今天小編就給介紹2個冷門小知識，幫助大家在遊戲之餘也能了解遊戲中的一些小知識。

1、水下可以躲避空襲定位儀危害小實驗

大家都知道在火力對決中空襲定位儀的危害幾乎是非常大的，特別是對野區外地方的玩家來說殺傷力是非常大的。之前的試驗中曾經測試過在飛機上可以躲避空襲定位儀的爆炸，那麼在水裡究竟能不能躲避空襲定位儀的危害呢？帶著對這個問題的思考，玩家來到了水邊並在自己附近的水域丟了一顆空襲定位儀。

之後跳到水裡，到達空襲定位儀被丟棄的位置，等待著空襲的到來，探索試驗的結果。

經過試驗玩家發現，空襲只會對自己水域上空的地方進行轟炸，並不會對水下的人造成影響。並且附近的水域都在四處開花，而玩家則完好無損。可能有的人會問了，那會不會被水淹死呢？其實這個是完全沒有問題的。只要玩家保證自己血量充足，空襲定位儀的轟炸畢竟是有時間限制的。在這個期間玩家躲在水裡完全不會受到任何威脅。

之後玩家又嘗試了在水面的情況會不會受到空襲定位儀的傷害，結果玩家剛露出水面就直接被定位儀擊中之後就被淘汰了，也就是說只有在水下才能安全的躲過空襲的危險，這個知識點你學會了嗎？2、挑戰手雷轟炸南瓜王實驗

暮色鎮的南瓜王是大家都比較感興趣的物資，因為擁有非常豐富的物資，然而一般玩家對南瓜王的淘汰都是採用槍擊的方式，那麼手雷究竟能不能淘汰南瓜王呢？帶著對問題探索的心態，玩家來到了暮色鎮，並且挑戰用手雷淘汰南瓜王。

首先就是擊倒一些小的南瓜怪收集成為大的南瓜王，之後玩家就嘗試丟出手雷，朝著南瓜王炸開的嘴巴裡面丟出去。

這時驚奇的一幕出現了，被丟出的手雷在遇見南瓜王張開的嘴巴的時候，竟然被反彈回來了。而此時反彈的手雷直接在地上炸開了。這個實驗充分的說明了南瓜王並不能被手雷直接淘汰，看來在遊戲中還是需要使用子彈進行設計才可以將他淘汰的。
