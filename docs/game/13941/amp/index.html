陰陽師百繪羅衣：最有望實裝的四款神仙典藏皮膚 SP男皮大舅穩了

天天要聞
2019-12-12

陰陽師對於這次的百繪羅衣活動，官方公開了目前已經入圍的30個優秀作品，而其中獲勝的估計就那麼幾款，而要說人氣最高的，和最有望實裝的神仙典藏皮膚，那麼基本上可以鎖定這四款了，尤其是SP男皮大舅，基本穩了，一起來看看吧。

見切啾啾的這款SP玉藻前男皮的典藏皮膚，可以說是這次人氣頂級的，也是呼籲實裝呼聲最高的一個，完美符合了大部分玩家要求大舅媽出一款男皮的主流聲音，顏值到位不說，關鍵是主題立意非常完善，百鬼夜行的建模都是N卡拿來直接用，或者簡單修改一下即可，讓建模師也省事很多，各層面來說勝出的幾率都很高。

影法師太太這款小天狗就不用多說了，插畫收藏級的質量，在遊戲中還原起來不費事還好看美觀，完全契合典藏皮膚的要求，想做特效的話註定是朝著土豪金的華麗酷炫特效發展，結合小天狗的人氣，就算是不怎麼用估計也有不少玩家樂意入手這一款。

TaKeNo_竹野筆下的輝夜姬成功入圍，雖然不算是插畫級別的藝術作品，但是實用，讓人看著好看舒服，萌點十足，且看起來就有很大的概率實裝，對於建模師也非常的友好，需要完善或者修改的元素和細節不算很多，實裝概率非常的高。

親媽畫師繪製的親女兒追月神典藏皮膚，應該沒有比追月神原畫師更加了解追月神的了吧？要什麼有什麼可愛萌點還有額外的福利，這種緋月太鼓為主題的典藏皮膚還原和製作起來困難程度不算高，算是實裝概率很高的一款。
