Valve Index VR設備故障排查記錄和心得 「農步祥」

天天要聞
5 天 ago

維爾福 Valve Index 頭戴式VR虛擬現實設備

在體驗一段時間後，我對Valve Index給了很高的評價，由於是目前市面上唯一一款144Hz高刷新率的VR設備，在畫面表現上有了巨大的進步，在駕駛、運動類等對運動畫面要求很高的遊戲中表現極其出色，改善了長時間遊戲時小腦的感受。加上本月Valve還公布了SteamVR獨佔的劇情流FPS作品《Half-Life:Alyx》將在2020年發售，相信所有VR遊戲玩家都已經摩拳擦掌等待這款原生的3A級遊戲了。可我個人購買的Index在測評文章寫完一周後就不幸「變磚」，無法正常使用了。

癥狀表現

Valve Index 頭戴式VR虛擬現實設備 – 顯示異常

Valve Index 頭戴式VR虛擬現實設備 – 顯示器映射無法顯示畫面

在寫完Index的體驗文章後，就把設備大約放置了幾天時間，某天忽然想娛樂一下，開機後就發現顯示異常：畫面為灰色，而且很快進入休眠待機模式，SteamVR的顯示器映射功能也沒有畫面輸出。由於Index在中文網路討論極少，沒法指望百度知道來解答這玩意怎麼修了，然而在谷歌上用英文搜索灰屏也得不到什麼有用的信息，因此暫時判定為個體問題，只能自己解決。期間嘗試了一系列手段，例如更換電腦排除主機系統和硬體問題，另外也嘗試固件更新、SteamVR更新到Beta或者降級等操作，以及禁用USB電源管理等，但遺憾的是都無法改變現狀。在更換為HTC VIVE或者Pimax 5K時，顯示和使用正常，說明系統方面以及手柄、光塔等組件出問題的概率很小，大幾率還是頭顯本身出了故障。

對於Steam來說，Index的銷售區域僅限於歐美少數幾個國家區域，對於我們這些域外玩家來說，返修是非常麻煩的事情。理論上來說目前大廠的VR設備都來自歌爾或者深圳的專業廠商代工製造，但畢竟它們也無法對售後負責。在網上確實有一些提供維修伺服器的個人商戶，但需要跑路或者郵寄送修，不夠放心，本著折騰的原則開始了自行折騰的路程。

解決嘗試

屏幕發灰說明顯示功能還是工作的。因此初步判斷為個別感測器失靈，難道是玩體感遊戲太多，流汗和水蒸汽導致內部短路？於是首先請出售價40元的電吹風，在熱風吹一陣後果然顯示畫面了！然而在戴上數秒到數分鐘後，Index繼續陷入罷工狀態。而且吹的熱風溫度有點高，甚至導致塑料面板有些變型，顯然不是理智的解決方案。

Valve Index 頭戴式VR虛擬現實設備 – 拆開前面板[紅圈為前面板天線排線]

既然電吹風無效，就只好把前面板拆出來看看情況了，拆開前面板，發現隱藏的LED燈不少，但搜索網路和Index的英文說明書也沒有提供有用的技術信息，兩綠一紅也不知是好是壞，只能暫時擱置。在一個月後再度谷歌搜索Index灰屏關鍵字，發現Steam社區上有相關討論，但癥狀是干擾巨大，需要給數據線裝一個磁環來解決。難道是因為各種信號干擾導致直接沒法用？在京東購買了數個磁環裝上，結果是依然無效。

既然沒找到解決方案，就只好進行進一步拆解使用排除法一個個功能拆掉進行嘗試了。首先懷疑的是兩組前置攝像頭，但拆掉排線後問題依舊，最後只能把整個前面板寫下來試試。這時意外發生，在拆開面板瞬間，畫面居然再次出現了。驚喜的是在待機一段時間後畫面依舊正常，仔細進一步檢查，發現在拆前面板時外殼上天線的排線掉了出來，只要拆掉排線，就可以正常顯示。然而高興得太早，在裸奔的狀態下戴上進入SteamVR，很快發現了新的問題，遊戲中定位處於失靈狀態，只要手舉起放在面前，定位信號就會丟失，畫面就會強制停止並且不停閃爍。

通過進一步資料搜索，發現「天線」連接的感應器其實是光塔的信號接收感應器，在斷開狀態下自然無法正常接收光塔的信號了，但在連接天線的排線重啟後仍回變成灰屏，那要如何解決呢？最後，發現在拔掉天線排線狀態下進入SteamVR，再重新連接排線，就可以正常使用了。使用數小時後也並沒有遇到嚴重的問題。

Valve Index 頭戴式VR虛擬現實設備 – 光塔接受感測器數量[圖片來自reddit]

從網上的拆解資料來看，Index的頭顯布置了數量非常多的光塔接受感應器[天線]，前面板的數十個感應器通過排線並聯的方式傳輸同步信號。很可能是前面板的個別感應器出現了故障或短路導致定位信號丟失顯示異常，但無法理解的是為何熱插拔的方式卻能恢復正常工作。前面板接收器數量多，但最後都集中在兩根排線上，維修也需要進行整體更換。目前雖然需要每次開機都要拔一次排線，想要徹底解決必須找到替換件，但至少可以暫時使用，起碼可以熬到愛麗斯發售，無論如何縮小了問題範圍也是好事。

雖然VR在全球範圍內都是小眾玩具，但個人使用出問題比較罕見，網路上也沒有大規模討論Index的質量問題，中文網路上的用戶討論更是比大熊貓還稀有，我個人使用過多款VR設備（HTC VIVE、三星GearVR、Oculus Rift、三星炫龍MR、Pimax 5K等）也只有Index遇到了硬體故障，遇到這種倒霉的個體問題除了心疼和折騰外，還是值得寫文章分析和分享一下，即使對VR沒興趣的讀者也可以看看上萬元的設備變磚是啥樣子，看笑話圖一樂。然而，我們也可以看到VR還是一種相當複雜而且整合度很低的設備，晶元數量多，工作機制複雜，只是依靠著強大的工業和設計能力勉強將這些子系統壓縮在一個巨大沉重的頭戴式顯示器里，Index比HTC VIVE更重，廠商也不好意思在網站宣傳資料里提及這點。VR設備光是改善性能和畫質還不足以吸引更多的非專業消費者，在專業領域的VR用戶也更需要更輕盈和易於維護的頭顯。增加集成度，設計專用的晶元整合大部分功能，進一步減少重量體積改善佩戴也是非常必要的改變。
