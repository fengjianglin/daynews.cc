DNF：春節前重要活動評價，全都在為年套鋪路，人均四套已成必然

2019-12-11 535條評論 2,764次閱讀 277人點贊

很多人看了體驗服五號更新的活動了，可能都很高興吧，小編這個時候來潑冷水有些不合時宜。但是你想啊，從十一周年慶以來遊戲低迷，策劃提高了福利，也僅僅是一段時間內一本黃金書，而下一波活動直接就是四本！史詩之路上的史詩就不說了，大富翁活動居然還送畢業附魔，那些附魔的價格現在可還是一千多萬啊！但是想想馬上就要到來的春節套，你可能就釋然了。

這個送畢業附魔的活動好嘛？好啊！兩個二十屬強的寶珠，一個二十五屬強寶珠，還有右槽的二十屬強，左槽的七十物攻，拿去打普雷一點問題都沒有。一個傳說勳章自選，一個神器守護珠自選，總而言之，簡直就是巨型福利，自地下城開服以來，除了鎮魂末期的那波福利，可以說是無可比擬了。很好，很棒，大家都很喜歡。

再看史詩之路活動，普雷大材料，增幅書，鳥背裝備，還有兩個95的史詩跨界石！這個時間來說，很多第一梯隊的臉黑玩家普雷裝備都沒畢業呢，這材料就送上了。大家想想當年盧克半年以後的福利都是送的什麼玩意，對比一下現在，幸福不？姓曾啊。這麼好的福利，大家可以當作是遊戲人氣低迷時策劃的一種挽回策略，但策劃從來都不是僅僅挽回而已。

言歸標題，春節套快來啦，一般來說春節套應該在一月初上架，恰好是這波活動結束的時間，很多玩家通過這波活動，穿上了一身超界畢業的防具，打上了畢業的屬強附魔，打上了紅字，回頭一看，還缺什麼呢？不就是缺年套嘛！從進團的角度來看，沒有國慶套是可以勉強進普雷的，沒有年套也沒有py交易的話，進普雷就是做夢！

所以就要買年套了啊，肩膀寶珠要有吧，左槽寶珠要有吧，這就是兩套了，好嘛你玩了兩個角色，那就是四套，如果稱號不打算拍賣行的話，就是六套。策劃順便大大加強一波今年的年套屬性，比如說護肩寶珠加強一下，寵物加強一下，左槽寶珠再加強一下，這樣一來，你還有點沾沾自喜呢，畢竟大家都要換，你買一次就行，別人去年的年套好像是白買了呀，大家居然處在同一起跑線上了？

如果說你不買年套，那麼這波活動對你來說也沒什麼意義，普雷以下的副本隨便打，普雷你是休想了。
