程咬金榮耀典藏皮膚被吐槽，特效是唯一賣點！哪吒福利皮膚來襲

2019-12-24 226條評論 1,410次閱讀 486人點贊

新的一哥榮耀典藏皮膚到底花落誰家？這個問題一直備受玩家關注，之前也有很多網友紛紛猜測可能是李白的，然而就在剛剛，官方也正式公布了新的一個榮耀典藏皮膚，那就是程咬金的活力突擊。

本來公布新皮膚應該是好事，但卻遭到了很多網友們的吐槽，到底是怎麼回事呢？

這款皮膚還是蠻有意思的，以健身為主題，程咬金在不同的狀態下還會有不同的著裝，畢竟是榮耀典藏皮膚，每一處的特效必須酷炫。整個皮膚突顯的就是程咬金堅持健身，最終變成了肌肉型男。

玩家的吐槽也就在這裡，畢竟榮耀典藏皮膚絕對是遊戲中最受玩家期待的系列，帥氣酷炫絕對是首要條件。這款皮膚雖然非常有趣，但很多人表示外觀不夠帥氣，再加上這個英雄的人氣也不是很高，做成榮耀典藏皮膚不適合。當然官方也有自己的考慮，我們這裡也就不糾結這個問題了，不過皮膚出來到底如何一起來看看吧。

程咬金的技能就不多說了，相信大家也都比較熟悉。新皮膚的特效非常的有趣，2技能旋轉啞鈴和沙包袋的時候會有殘影。大招是在程咬金的頭上出現一個淋浴頭，一看小金金就是鍛煉過度了需要降降溫，不過這個特效也太醒目了，敵人一看就知道程咬金在開大，也不知道是好是壞啦！

除了程咬金的榮耀典藏皮膚之外，官方還爆料了一款福利皮膚，那就是哪吒的次元突破。這款皮膚是鑽石奪寶皮膚，聽到這個大家是不是很激動的，畢竟很多玩家鑽石非常多，王者水晶也是有很多的，所以這款皮膚上線之後基本算是免費送的了。

目前皮膚還沒有設計好，僅有設計圖公布出來，整體看上去還是相當的帥氣，難道官方想要依靠這個福利皮膚來挽回哪吒的人氣嗎？

那麼，大家覺得官方公布的這兩款皮膚如何呢？
