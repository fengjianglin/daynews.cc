日本最強動作遊戲，火18年後被中國廠商改成手游，卻被吐槽臉太崩

2019-12-23 1,365條評論 3,932次閱讀 1908人點贊

大家周末好，X博士又來了~

本周的熱門遊戲事件有點多，一句兩句無法說清楚，還是跟X博士一起走進本周的熱游情報看看究竟吧。

劃重點：EPIC送瘋了！12款遊戲免費送 《無主之地3》只賣9美元

好消息！好消息！EPIC商城大促銷，啥啥都打折，買啥啥打折，10美元代金券直接送，買多少送多少，另有12款遊戲直接白給。

前幾天X博士打開EPIC商城立馬就哭了，上線先送一張10美元的代金券，除此之外星戰、無主之地、大表哥2、地鐵、天外世界等大作統統打折促銷。

而且這兩個折扣還能疊加在一起使用，說真的當時我就懵逼了。

算下來《無主之地3》只需要66元，大表哥2隻要126元，天外世界賣87，地鐵就最離譜直接35塊錢帶走。

等於首頁上第一排大作，你只需要花424元就可以全部打包帶回家。

想像一下你424能買什麼遊戲，一款《生化3重製版》，然後餘10塊錢。

其實還有更爽的事情，X博士查看了代金券的規則，該券雖不可用於預購和DLC產品，但卻可以自己留到5月1日，也就是說理論上來講5月1日之前發售的遊戲，只要符合價格規定都可以使用該券。

看了一眼2077的發售時間，4月16日……

價格戰還請您繼續打下去。

劃重點：Steam2019冬季特賣開啟 生化3重製版上架登頂熱銷榜

與EPIC同時開啟活動的，是Steam的冬季特賣，雖然折扣依舊不錯，但看過EPIC的活動X博士還是要對Steam說一聲——

這還是X博士第一次看到Steam折扣香不起來的，G胖看來遇到對手了。

畢竟如果沒有EPIC的126元大表哥的話，X博士也會覺得199元的大表哥還不錯。

最重要的是，跟隔壁簡單粗暴的10美元相比，Steam5000代幣換35塊錢的代金券實在是沒眼看。

而且不光代幣的比例十分坑爹，換貼紙的玩法也實在讓人香不起來。

不過儘管EPIC的價格更加優惠，但Steam的遊戲陣容還是不容小覷的。

本周《生化危機3 重製版》就在Steam中開啟了預購，雖然414的售價並不便宜，但還是在第一時間衝上了熱銷榜榜首。

雖然Steam在遊戲數量上還是佔據很大的優勢，但這麼久以來打折搶錢的人設馬上就要被隔壁搶走了。

劃重點：國內廠商公布正版鬼泣手游 動作高度還原但免不了國產套路

本周卡婊正版授權、國內雲暢遊戲研發的鬼泣手游《鬼泣 巔峰之戰》正式開啟首測。

從遊戲的畫面和曝光的內容來看，主要是以《鬼泣3》為原型製作的。

遊戲整體給人的感覺其實還算不錯，無論是場景、怪物的還原，還是角色的動作設計，都還是有點內味兒的。

尤其是極為複雜的連招搓招系統，也能讓人看出廠商在還原動作效果上花費的心思。

只是這個但丁的長相實在是不忍直視。

（哥們，你誰？）

另外，X博士還在遊戲宣傳里找到了一些匪夷所思的東西，比如這個……十連抽得武器技能？！

簡單來評價一下這款遊戲，目前整體給人的感覺是極為撕裂的。

進入關卡後，遊戲無時無刻不流露出鬼泣經典的味道，而在你結束關卡的一剎那，它又高調地把極其頁游、手游化的UI按在你臉上。

（這UI做得實在離譜）

只希望官方趕緊給但丁和這UI界面整整容。

劃重點：LPL轉會情況基本鎖定 廠長退役當教練阿水不知去向

本周LPL的轉會期終於結束，各個隊伍的人員基本已成定局。

不少隊伍都趁機給自己進行了一波補強，VG簽下了曾經在IG表現亮眼的蕉太狼和EDG的ADCiboy，而且還簽下了SKT的教練。

FPX則是拿下了SKT的上單漢子哥，對上路進行了一波補強，估計是準備衝擊兩連冠了。只不過漢子哥的官宣竟然是因為餓了找Doinb點外賣。

除此之外TES則收穫了karsa，RNG也從PDD那邊拿下了「冠軍BUFF」小龍堡，老隊友mata也作為教練再次回到RNG。

最慘的莫過於WE和EDG了，WE的大舅子和兮夜全部離隊，粉絲直言「家都沒了「。

EDG則是送走了上單和ADC，甚至連LPL001號選手廠長都選擇了退役當教練，實在是有點可惜。

轉會期最大的疑案莫過於阿水了，最早官宣自由人的他至今都沒個下落，這也讓粉絲多次在微博上發起「尋找阿水」的活動，熱搜都上了好幾次。

就連外媒記者也在推特上尋找阿水的下落，可惜至今仍然沒有消息。

阿水也算是LPL頂級ADC了，雖然這賽季ADC不是很強，但也不能就這樣報廢了啊。
