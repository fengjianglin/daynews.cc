火影手游：穢土水門的首個技能已出現，空中丸子依然很帥氣

天天要聞
2019-12-22

最新的高招S忍穢土水門即將在2020年的1月10日上架，雖然之前拆包發現過相應的技能圖標，只知道有螺旋丸和飛雷神的技能效果，但至今仍然沒有看到正式的技能演示視頻，魔方似乎也正在進行緊張的測試中。

在今天，網路上流傳出了一個穢土水門的技能演示效果，雖然很短暫，但至少可以看到一些細節，關於這個圖的真實度我也去查了一下，確實是出自官方渠道，基本上可以認為這個技能是已經確認的，最多是會跟正式上架有一些細微的差異。

通過這個短暫的技能演示可以看到，技能圖標跟之前拆包出來的完全一致，而且這個演示裡面的二技能是長按，但是，在面具男身上並沒有看到類似標記的東西，而這個穢土水門還是直接從對方頭上出現，不知道這是距離問題，還是有什麼隱藏的要素。

除此之外，在釋放這個丸子之前，左上角的地方還有一個苦無，關於這個苦無的放置方式目前不太確認，但應該可以在釋放技能後點擊其他按鈕來閃回到那個苦無的位置，這個效果是類似目前新春水門的特性，只是不知道穢土水門會有什麼新的花樣出現。

當很多玩家看到這個動態圖演示的時候，除了說「真帥」「特效滿分」以外，還有一個擔心的點，那就是這個空中螺旋丸的前後搖有點大，你可以看到在釋放技能後會有一個明顯的搓丸子前搖，然後丸子打傷害的時候也是一個非常大的後搖，雖然技能後搖是比較常見，但我個人認為後搖估計可以通過閃到地面的苦無來取消，不然的話，對面都替身了，你還在耍帥的砸丸子？

其實這個空中丸子的效果在究極風暴里也存在，這個應該是瞬身閃現到前方的一段距離後把丸子砸下來，同時，在究極風暴里也有一個搓丸子的小前搖，這個也算是完美還原吧，很多玩家說，昨天巔峰對決剛出了一些BUG，這個技能是故意放出來吸引注意力的？我覺得這個倒無所謂，希望能夠出現更多的技能演示，滿足玩家們的好奇心，不要擠牙膏一樣吊胃口就好！
