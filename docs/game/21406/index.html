職業生涯差點斷送？theshy談S8手傷：醫生說我的右手可能沒法用了

2019-12-20 74條評論 1,200次閱讀 278人點贊

自從S8全球總決賽結束後，因為IG拿下了S8世界冠軍，收穫了大量的粉絲，人氣就得到了飛速的增長。其中上單theshy更是S8世界賽一戰成名，被不少人譽為世界第一上單，即使在S9世界賽上關鍵局中他發揮失常，但是一整年看下來，theshy的表現依然很出色，接連兩個賽季拿到了LPL最受歡迎選手的稱呼。在S9結束之後，theshy回到韓國休假的時候，也接受了韓媒的採訪，這次採訪中他第一次提到了自己S8賽季的那次手傷事件。

​根據國內網友的翻譯，theshy在採訪中表示因為突發事件，導致了自己的右臂嚴重受傷，當時醫生說「你可能再也用不了右手了」，theshy將此稱之為愚人節一樣的消息。對於普通人來說用不了右手都會對生活造成非常大的影響，更別說一個需要操作的職業選手了，關鍵是這才是theshy正兒八經打職業的第一年（S7賽季只有末尾才上場），之後theshy表示自己經過了幾個月的手術和復健，終於東山再起。

​這個手傷肯定是非常嚴重的，因為記得那時候theshy在S8春季賽表現非常出色，基本上是人擋殺人佛擋殺佛，但是突發事件導致了他缺席IG春季賽季後賽，也間接讓IG拿到了十八連勝總殿軍的尷尬戰績，而夏季賽theshy的復出戰是在洲際賽上，說實話表現並不是很好，這個應該和手傷剛剛痊癒有關係，一直到夏季賽結束其實theshy的表現都只能稱得上是中上游水平，但是沒想到S8世界賽他徹底展現實力，得到了世界的認可，但是也有很多IG粉絲認為，雖然S8世界賽的theshy很強，但是還是沒有恢復到春季賽的巔峰實力。

​現在theshy的很多定妝照上，一本都會遮住右手的手臂，應該是因為有一條很長很醒目的傷疤，而且theshy對於這段往事也應該比較介意，這是他第一次公開提到自己的手傷，雖然原因沒有說，但是IG管理層肯定是難辭其咎，一個隊內主力carry點在季後賽之前突然手傷，換誰都不能接受。最可怕的是，當時還有人借著這件事造謠，說theshy以手傷要了上千萬的身價，其實在採訪中，theshy也說了自己的薪資差不多10億韓元，摺合下來600W左右人民幣，這還是來到IG第三年的年薪。

​當然theshy的個人能力還是很強的，手曾經受過了這麼嚴重的傷，但是還能出現一打五天神下凡這種操作，相信他後面還會有更加精彩的職業生涯，所幸的是他現在恢復的不錯，如果真像醫生所說無法使用右手，對於LPL賽區甚至是LOL職業圈子來說，少了這樣一位優秀的上單，都是非常大的遺憾。希望theshy後面的職業生涯可以順利吧，這樣一位實力強勁又低調謙虛的選手，應該還有更好的未來。

​那麼，對於這波對Theshy的採訪，你們有什麼想說的嗎？
