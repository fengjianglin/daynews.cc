各位勇士好！在下遊戲日報風櫃君。說起DNF的土豪，很多玩家的第一印象都是旭旭寶寶。確實，作為國服第一紅眼，寶哥的大號裝備在國服無人可比：全身增幅16的超界，外加一把紅17的普雷太刀，這樣的裝備讓寶哥可以在任何副本里橫行霸道。

從60版本開始，旭旭寶寶就喜歡挑戰各種記錄。而這個愛好一直延續到了現在，無論是春節上線的雙倍鳥背，還是周年慶推出推出的普雷，寶哥都把這些副本玩出了花。不過在DNF遊戲中，除了旭旭寶寶外，還有許多神豪級別的玩家存在，比如國服第一驅魔B叔。

儘管B叔的名字不如旭旭寶寶那麼出名，但是他的實力卻不亞於寶哥。而B叔的代表作，就是一招秒殺各大副本boss，從宇宙惡魔羅什，到泰波爾斯最美狐狸，都是B叔的手下敗將。而在最近，B叔又用他的驅魔刷新了國服紀錄——5秒強殺第三使徒。

B叔之所以擁有如此恐怖的輸出能力，主要原因就在於兩個字「猴戲」。所謂猴戲，就是通過各種換裝，來提高自己的屬性。而B叔不僅自己要「猴」，而且他的隊友也要「猴」，在這樣的多重猴戲之下，B叔的傷害也在短時間內拉滿，所以他才能夠在5秒之內帶走普雷。

在之前旭旭寶寶舉辦的強者之路邀請賽中，B叔也有出場。那時候B叔就憑著一系列精彩的操作，讓寶哥和直播間間的水友們嘆為觀止。而B叔也被玩家們稱作是國服的隱藏幻神，如果旭旭寶寶是紅眼第一人的話，那麼B叔就是當之無愧的驅魔第一人。

在另一方面，旭旭寶寶和B叔同樣也證明了，在DNF中，並沒有弱勢的職業，只有「貧窮」的玩家。就拿驅魔來說，這個職業在遊戲中並不算強勢，甚至在今年上年被反向加強了一波，但是B叔卻用他的裝備和操作彌補了職業的短板，成為了隱藏的幻神。

各位玩家，你們對B叔的驅魔怎麼看呢？
