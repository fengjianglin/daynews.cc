在此次總結中，我們能看到廣東、浙江、江蘇、河南以及山東的召喚師人均皮膚數量排名全國前五，可以說是最愛打扮的五個省份。其實這個數據還是比較意外的，本以為北上廣會有更多的城市在線，最後只有廣東在線。身為北上廣和江浙滬兩個聯盟的一員，上海居然沒有上榜，也許是人們都太忙了沒時間玩遊戲吧？

男生和女生最喜歡的皮膚也不一樣，男生使用頻率最高的三款皮膚分別為李白千年之狐、孫悟空大聖娶親以及韓信的街頭霸王，而女生最常使用的則是小喬繽紛獨角獸、貂蟬聖誕戀歌以及大喬的貓狗日記，從皮膚的使用我們也可以看出不同玩家最喜歡的英雄，帥氣的小哥哥和美麗的小姐姐，誰會不喜歡呢？

同時官方還公布了最慘皮膚，那就是小魯班的福祿兄弟，每月被擊殺3億多次，這款碎片商城就可以兌換的皮膚見證了小魯班最多的陣亡，而大聖娶親則是每月擊殺貢獻最多皮膚，有接近6億個人頭，是非常誇張的數字了。
