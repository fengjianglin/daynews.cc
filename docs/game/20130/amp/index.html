DNF：體驗服春節版本回歸系統改版，登錄就送95史詩以及黑瞳石

天天要聞
2019-12-18

體驗服更新了春節版本的內容，但是並沒有全面的更新，為了吸引玩家的回歸，本次更新對回歸系統進行了一個改版，並且送的東西確實挺吸引人的，全新的任務玩法也讓回歸玩家需要更多的社交。

回歸登錄獎勵

登錄玩家即可獲得回歸稱號和賽麗亞寵物，一把95級史詩武器和一套95級的史詩裝備，裝備屬性和哈林強度差不多，但是均為30天期限史詩，所以並必要強調屬性。此外還會獲得20張超星空裂縫通行證。

每日好禮每天登錄領取，可以獲得1天的王者契約和藥劑禮盒，此外可以獲得一個自選門票禮盒，黎明裂縫，泰波爾斯，強襲模式或者是漩渦的門票。

全新的任務獎勵系統

採用了全新的冒險家任務系統，根據特定的等級完成任務獲得獎勵，所以95等級前的藥劑和經驗膠囊我們不細說了，這裡僅介紹95級的等級任務獎勵。

玩家達到95級時即可獲得一套95級的聖物套裝；

抗魔值達到4000/5000可以獲得鑲嵌欄開啟裝置；

完成深淵派對任務6獲得100張深淵邀請函；

完成超星空裂縫挑戰獲得10個星空深淵靈石；

通關強襲模式8次獲得400個黑夜穿行之證；

通關黎明裂縫4次獲得500個異形結晶；

通關6次泰波爾斯獲得200個蒼穹碎片；

通關10次災難領域獲得10個精鍊的時空石；

通關3次代號希望獲得100個黑暗之眼靈能碎片；

通關再戰魔界獲得15個勝利之義，5次獲得30個；

通關魔界大戰單人模式獲得符文原石禮盒1個，組隊3次獲得6個；

通關單人超時空漩渦獲得10個反物質粒子，通關3次團隊模式獲得30個；

通關1次單人普雷伊希斯獲得5個黑瞳石，3次團本模式獲得10個；

此外還有通關3次眩惑之塔獲得的材料，但是並沒有什麼重點內容。

任務總共分為1級、10級、20級、70級、85級、90級和95級不同等級的任務，但是很明顯前期主要是贈送經驗膠囊加速升級，而95級的獎勵，玩家需要完成對應的副本挑戰而獲得額外的獎勵材料，但這些任務都是一次性。

對比分析

當前版本的回歸玩家的獎勵是賽麗亞的熱忱，登錄即可獲得7天的結婚裝扮，完成7天的簽到可以獲得賽麗亞的熱忱稱號，遠古精靈的改良秘葯，復活幣，雷米的援助以及帳號銀質金庫升級工具，而完成使命任務獲得的獎勵也主要是雷米，復活幣，7天的結婚裝扮，靈越武器和傳承首飾套裝。

這一些獎勵對於當前版本來說根本用不上，所以這一個回歸獎勵讓很多回歸玩家上號就潑了冷水，而新版的回歸獎勵對於95版本來說還有一些作用，贈送的裝備也能讓玩家有了基本的副本挑戰能力，外加材料的額外獲得加速製作裝備的周期，對當前版本來說好了很多。

這一個改版是春節版本，而春節版本之後又將迎來一個100級版本，所以回歸活動也主要是想要吸引玩家回歸，如果春節版本上線職業預約活動以及回歸活動獲許可以回暖一波。
