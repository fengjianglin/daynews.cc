德瑪西亞杯賽果預測，最有冠軍相的曾是墊底戰隊

2019-12-22 2,082條評論 4,027次閱讀 1651人點贊

2019年德瑪西亞杯就要開始了，伴隨著開幕式，第一天就將進行10場小組賽對決，賽程從上午11點排到晚上11點，觀眾將一飽眼福。這次德杯的參賽隊伍，主要來自LPL，也有部分LDL的隊伍，加上部分隊伍沒有上主力，因此結果就非常難預料。小編收集了每支隊伍的參賽名單，發現有幾匹黑馬是冠軍的有力爭奪者，最有冠軍相的那位曾經還是LPL的墊底戰隊，話不多說，一起來看吧。

一、划水的老牌戰隊們

FPX、IG、RNG，都只上了部分主力選手，比如RNG的Uzi因傷休養，FPX更是不給面子，S9冠軍陣容五人一個沒來。而IG則相對好點，上場了新買的下路，主力與替補混搭，但三叉戟也是一個都沒到場。毫無疑問，這三隻老牌豪強都沒有爭奪冠軍的實力，權當是來給新人和替補練手，磨合陣容的。

二、摩拳擦掌的LDL隊伍

LDL和LPL的硬實力差距，並不如很多人想像中那麼大，LDL的頂尖隊伍，其實力能比肩LPL的中流隊伍，像是去年晉級至LPL的龍隊SDG（現已改名DMO），首年就打進了季後賽。而這次參加德杯的LDL隊伍，基本都是全主力上陣，說不定連LPL的頂尖隊伍都能幹翻。

三、紙面實力逼近一線——LNG

LNG在這次冬轉進行了大規模的重組，原核心打野Sofm出走，作為補償引入了實力更強一檔的原TES打野Xx，中單位置也換了人。

此前LNG被稱為”玩蛇隊”的主要原因就是戰術四板，打野Sofm一人獨享太多資源，在官方削弱野核玩法後，LNG依然採用逆版本的戰術，可以說天生吃虧。但隨著熊熊和Maple的加盟，LNG的中上野來到了一個較強的層次，這對於他們的戰術轉型是一個好消息。

四。臉上寫滿冠軍相——VG

要問整個冬轉提升最大的哪個隊伍，之前小編肯定會說OMG或LGD，一個召回馬哥，宣布要大規模重組；另一個口號喊得震天響，誓要以S10為目標發起衝擊。但轉會期結束後，小編將最終陣容統合起來，發現一直低調辦事，僅引入了三人的VG，反倒成為了小編最為看好的隊伍，因為引入的這三人都是經過時間檢驗的強者。

教練Kkoma，三冠王SKT的教練，水平無人能質疑，本人在加盟VG後說了一句特別狂傲的話：”如果VG的目標僅僅是進入S10的話，那我是不會來的。”這意思很好理解，就是沖著S10冠軍去的。

中單Forge，在LPL的幾場關鍵救火局讓他走進了觀眾的視線，根據幾場數據的初步判斷，他的實力至少排在LPL前五，一個硬剛Knight都不落下風的中單，小編表示很看好。

ADCiboy，原EDG的AD，在18年，他和IG的阿水被並稱為國產AD雙子星，雖然iboy的後續發展不如阿水，但這也和EDG的氛圍、俱樂部文化離不開關係，太嚴肅的環境並不適合他，從他零視野各種閃現躲技能就可以看出，這人依舊是可塑之才。
