盤點幻化之戰里你最不想遇見的技能

2019-12-15 2,285條評論 4,283次閱讀 1052人點贊

哈嘍各位小夥伴大家好又來到新一期解說了，剛剛結束的新模式「幻化之戰」不知道大家玩爽了沒，沒玩夠也沒辦法畢竟不知道還會不會再出，這幾天我也一直在玩幻化之戰，今天呢就來說說幻化之戰中那些最不想遇見的負面技能：

崩壞

第一個當然就是崩壞了，作為遠古時期的頭號負面技崩壞想必大家也非常熟悉了，在幻化之戰遇見這個技能就很尷尬了，畢竟大家都是4上限沒那麼多血來崩了

崩壞：鎖定技，結束階段，若你不是體力值最小的角色，你失去1點體力或減1點體力上限

詐降

界黃蓋是很多玩家特別喜歡的武將了吧，比起標黃蓋信仰跳躍界黃蓋無疑是穩定很多的，但在幻化之戰里遇見詐降就很尷尬了，畢竟沒有苦肉來掉血，詐降也完全失去了效果除非再來個配套的苦肉或者遇見個願意用絕情打你的玩家才能恢復往日雄風

詐降：鎖定技，每當你失去1點體力後，你摸三張牌。然後若此時是你的出牌階段，則直到回合結束，你使用紅色殺無距離限制且不能被閃響應，你可以額外使用一張殺

同疾

如果說遇見詐降還只是無用技，那同疾就真的是強行保護別人關鍵是這模式都是表面隊友啊，不過沒有了妄尊也不好發動同疾相當於觀賞技能，不過有個特別神奇的搭配武魂+同疾勸架全場非常OK

同疾：鎖定技，若你的手牌數大於你的體力值，則攻擊範圍內含有你的角色使用的殺，不能選擇除你以外的角色為目標

絕情

絕情如果全場很多賣血技能的話就很舒服了，但一進入遊戲「死戰模式」就尷尬了，因為死戰模式如果你的回合內不造成傷害要扣血的，而絕情註定了你的傷害都是失去體力也就是百分百沒有造成傷害的條件，那死戰模式的時候就很尷尬了呀

絕情：鎖定技，你即將造成的傷害視為失去體力

耀武

感覺策劃對於華雄真的是惡意滿滿，作為演義中僅次於呂布的董卓部下猛將這樣設計真的好嗎？都說華雄是最愛「吃菜」的武將，因為他根本不想要自己的二貨技能，但華雄的優勢在於他6點體力可以給他蓄爆，但耀武這個技能是真的沒有人會想要獲得吧

耀武：鎖定技，當一名角色使用紅色殺對你造成傷害時，該角色回復1點體力或摸一張牌

燃殤

說起突兀骨你最先想到什麼呢，他那無可比擬的15點體力還是秒天秒地的aoe？作為三國殺最肉的武將骨頭的負面技能也是很難受的，每次中火焰傷害都是致命的隱患，在幻化之戰中只有4血中一次火焰傷害基本就宣判死刑了

燃殤：鎖定技，當你受到1點火焰傷害後，你獲得1枚「燃」標記；結束階段，你失去X點體力（X為「燃」標記的數量）
