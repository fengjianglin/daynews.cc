DOTA2：一雪前恥！VG韌性拉滿2-1擊敗奪冠熱門TNC成功復仇

2019-12-22 1,217條評論 3,516次閱讀 349人點贊

DOTA2新加坡世界邀請賽淘汰賽的第二個比賽日中，來自中國的VG戰隊將要面對他們在Major決賽上面的老對手TNC戰隊，雙方究竟誰能夠拿到本場比賽的勝利呢？讓我們一起來看一下吧！

​第一局比賽，VG選擇了飛機和藍貓的組合，三號位則是人馬，醬油選擇了小小和巫妖。TNC則是選擇了小狗和船長，三號位則是劇毒，醬油選擇了末日和冰龍。

​前期雙方勢均力敵，TNC在線上比較有優勢。14分鐘，雙方在中路爆發團戰，互交一波技能之後，沒有爆發人頭；15分鐘，藍貓蹲符落單被擊殺。17分鐘，VG開霧先手劇毒，冰龍大招拖住，TNC趕來完成一波1換2；19分鐘，VG開霧技能全交擊殺船長，經濟領先擴大到5K；30分鐘，TNC再次擊殺藍貓，藍貓買活趕來戰場，雙方打成一波五五開；60分鐘，人馬脫節，TNC擊殺VG三人，人馬、小小交出買活，飛機沒有買活；63分鐘，VG選擇打三代Roshan，最後一波，TNC直接一波擊潰了VG拿到了勝利！

​第二局比賽，VG拿到了水人和伐木機的組合，之後又選擇了劇毒，醬油則是牛頭和復仇之魂。TNC則是選擇了帕克和猴子，三號位則是獸王，醬油拿到了小小和巫妖。

​前期VG在線上有了比較大的優勢。19分鐘TNC四人圍堵水人，牛頭關鍵扳救下水人；20分鐘，VG攜11K經濟領先輕鬆拿下一代Roshan，水人帶盾；22分鐘，VG中路上高，但只破掉中路高地塔；24分鐘，VG開霧追擊猴子，巫妖牛頭1換1，巫妖買活幫助TNC完成一波2換3；最後TNC眼看無力抵擋只能選擇放棄。雙方戰平！

​第三局比賽，VG選擇了毒龍和小魚人的組合，三號位則是亞巴盾，醬油則是拿到了VS和小魚人。TNC則是選擇了老鹿和水人，三號位拿到了酒仙，醬油則是拿到了墨客和樹精衛士。

​前期TNC戰隊劣勢很大，8分鐘，中路爆發團戰，水人拿下三殺幫助TNC完成2換3，隨後小魚人趕來擊殺水人將局勢穩住；19分鐘，VG拿下一代盾，但剛剛復活的毒龍再次被抓再送一個頭；23分鐘，VG帶盾上高破掉一座高地塔，經濟領先6K；28分鐘，VG上路推進擊殺對方三人，逼出兩人買活，但撤退不及時也被留下兩人；最後一波團戰，儘管TNC頻頻交出買活，但是依舊無力回天，只能遺憾落敗，恭喜VG2-1擊敗TNC戰隊！
