鬥魚黃金大獎賽：OMG王者歸來小組第一，SSS高開低走，VC拉閘了

天天要聞
1 天 ago

對於喜歡吃雞的網友來說，最近又有比賽可看了，鬥魚舉辦絕地求生黃金大獎賽正式開啟正賽的角逐，毫無疑問這次鬥魚黃金大獎將會非常的精彩，因為幾乎匯聚了絕地求生圈全部豪門戰隊，這也讓每天的比賽都是充滿著期待。當然了，關注比賽的網友都知道，就在次鬥魚絕地求生黃金大獎首日A組的比賽中，我們期盼已久結果出現了，曾經的世界賽冠軍OMG戰隊狀態回來了，在僅有4AM、RNG、iFTY、QM和SMG等強隊的情況下，OMG斬獲小組積分第一，打出了自己的氣勢，讓很多OMG粉絲感到開心。

隨著A組的比賽結束，今晚鬥魚絕地求生黃金大獎也是迎來了B組隊伍之間的較量，在B組之中又是猛男戰隊雲集，VC、tianba、ARK、VG、EDG和SSS等戰隊都在其中，相比於A組的競爭情況一點不弱，毫無疑問又是激烈的大混戰。有了OMG蘇醒的情況在前，對於今天B組的比賽眾多網友更加期待周莉莉的SSS的表現，希望看到二哥能帶領這SSS戰隊再次雄起。

B組的首局比賽，讓粉絲們開心的是SSS戰隊是火力全開，雖然最後沒能吃雞，但是以高擊殺成功拿到17分的高分，而且是打出了氣勢，也算是首局拿到了開門紅了。本以為後續SSS會越戰越勇，結果扎心的是SSS這是開了一個好頭，然後就是一路走低，後續幾局比賽表現不佳連續拉閘了，真的是非常的扎心，不過選手還是有亮眼表現，就比如小北吃毒繞後LGD身後就非常的靈性，不過好在SSS成功躋身前八。但是剛剛參加完國際賽的VC戰隊似乎有點慢熱，今天的表現完全沒有狀態，6局比賽下來只拿到15分，排名倒數第二，不知道後續VC能否及時調整過來。

當然了，對於ARK和Tianba這兩支猛男戰隊來說，他們的發揮還是非常的穩健的，一個位列小組積分第一、一個位列小組第四。當然了，後續的比賽還將繼續，期待更多精彩的比賽上演，喜歡的網友可以去鬥魚觀看比賽。
