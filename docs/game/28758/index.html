Q：軍裝以及奧山副手帶來的全新配裝

A：主副手的選擇變成了碧空+奧山副手，直接把統御(本人)黨拉開了一個很大的檔次，雖然嘴上說著將就用，但是心裡是真酸,魔杖更是點擊就送的16冰傷，法師們再也不用開組TL黑魔杖了

其次是軍裝帶來的混搭最優解，目前我個人經過深思熟慮之後，再除開1.火圈，2.大法師.3燃魔之後，最佳的PVE配裝是T1 頭 護腕 手套 +軍裝的肩膀+鞋子，這套在沒有秘術之心和JJC腰帶的情況下依舊保持著2點的命中，最極限的滿足PVE極限黨當前階段畢業要求，T1三件+軍裝2件都能完美的湊出效果，不佔用胸和腰帶的位置，在保證高法傷高命中的同時，智耐同樣不俗，如果有更好的搭配歡迎請留言指出

Q：R10太肝了怎麼辦？R7已經是我的極限了

A：的確，作為之前只在野外活動的人來說，沒開ZC之前野外靠榮譽打野小分隊的榮譽每周都能升升級，開了ZC之後突然出現了一群肝帝，奧山都崇拜了軍銜一結算髮現自己甚至在千名開外，軍銜結算過後發現一檔二擋基本都是馬雲黨或者穩定的戰歌國家隊，R10會有難度，R7可能已經是極限

軍裝只能拿手和鞋子，這時候只需要把手和肩膀的位置換一下，損失一點的命中，一樣的完美兩件套+三件套效果， (力量手套什麼的都見鬼去吧) R7目標黨的最優選擇

Q：為什麼一定要拿T1甚至軍裝？我三大畢業+碧空一樣的350+法傷，還不夠用嗎？

A：PVE來講，當前MC基本都是無壓力給人造成了法傷高於一切的感覺，關鍵的確是大家覺得最沒用的兩個屬性—智.耐，三大以及各種隨機附魔散件有著很高的法傷，但卻有著極低的智耐，三大畢業的撐死就2000出頭的血5000的藍，而一身軍裝T1卻可以輕鬆武BUFF3400血+6000藍，且不說PVP(開了榮譽各位野外自有體驗)，即將開放的BWL，面臨的會是法師數量的急劇減少，不會出現MC多則十四十五個法師的情況，法師屆時將會面臨嚴重的失業，團長都是擇優進團，什麼叫優？法傷合格的情況下智耐最優，我也玩N號打G團，很多時時候，你OT了能挨兩拳和三拳之間的差別，就決定了N能不能刷上一口快療，T能不能嘲回來，所以，各位法神們要有心裡準備，開荒就是新一輪的大洗牌，勢必有人會面臨淘汰

Q：你說的火法，MC真的好用(能用)嗎？怎麼個輸出手法？(個人再次安利)

A：能用，好用，特別好用，能讓你提前體驗到火法的快感，燃燒短暫火球三連爆甚至能點燃一跳1000+，且入門真的很簡單，火法的第一屬性就是暴擊，200+法強的情況下，你只需要吃個龍頭+費伍德花總共15%的暴擊，面板暴擊就能輕鬆地30%+，並且MC的火法一樣點出了所有的冰系單體輸出，你的寒冰箭輸出並不會有任何一點的降低，輸出手法也比冰法無腦111來的有快感，BOSS除開嘉頓老8螺絲，任何boss都是灼燒5層搓火球，開燃燒過了繼續循環，小怪灼燒灼燒火球火沖，石頭人就只用灼燒火沖。同樣的裝備本人多個CD證明，輸出比奧冰還高，且不需要深冬之寒工具人，有SS及時掛元素就行，現階段也沒人去搶你的點燃池，天賦如下，下個CD結束我盡量貼出WCL的統計，看看自己能到什麼水平(不好玩你打我)
