戰雙帕彌什再現神操作 修改條款變霸王 玩家：這操作勁兒真大

2019-12-07 0條評論 15次閱讀 0人點贊

摘要

作為近期爭議極高的新手游戰雙帕彌什，在經歷了自身開服的錯誤運營和補償策略之後，近日有展現了一波非常神的操作，其官方運營將優秀條款進行了為修改，直接變成了霸王條款，對此玩家紛紛表示：勁兒太大受不了。

作為近期爭議極高的新手游戰雙帕彌什，在經歷了自身開服的錯誤運營和補償策略之後，近日有展現了一波非常神的操作，其官方運營將優秀條款進行了為修改，直接變成了霸王條款，對此玩家紛紛表示：勁兒太大受不了。

戰雙帕彌什這遊戲具體的新操作是這麼回事，近期網友發現遊戲自身內部的協議條款被官方進行了人為的修改，而修改後的條約內容明顯是針對近期遊戲黑卡錯誤發放還有回檔事件進行的處理，對此官方對於回檔這方面的解釋大概如下

簡單總結就官方因為遊戲過錯需要回檔時，根據消耗情況按照一定比例為用戶進行還原，或者其他形式的替代，此外就不承擔額外的責任和義務了。

也就是說官方對於這次的回檔事件沒有完全回檔，導致部分領了1W黑卡獲益而沒有回收的情況，進行的處理方式就是不管了，回檔之後造成的錯誤官方給點補償或者道具這事就結束了，這期間的過程沒有遊戲運營的任何責任。

雖然其他遊戲也有類似的條款，這麼來看針對這個條款有點上綱上線的意思，不過仔細對比會發現敘述方式根本不同，別的遊戲回檔是將數據徹底打回，是還原操作，而這裡是比例替代，這個沒有可比性。所以不少玩家瘋狂吐槽這個合約的霸王條款的性質，嘲諷官方這一波神操作實在是勁兒太大，受不了。因為這感覺對比過於明顯

普通遊戲：在沒有錯誤的條件下，玩家可以怎麼怎麼樣

戰雙：在我們犯錯的情況下，玩家不可以怎麼怎麼樣

最後再貼一個協議規則，這個大家應該都看得懂，就是說你氪金了，我們遊戲就認這個兒子，沒氪金，愛玩玩，不玩趕緊走。差不多也是針對這次遊戲黑卡事故發出的聲明，意思就是說沒氪金還想要1W黑卡的資源？想桃子呢？

總而言之如果官方沒有這波操作可能也沒這麼大的風波，本來之前的風波就沒有平息，現在又整出這麼個操作，本著遊戲質量感覺還可以的基礎上，也沒什麼太多可以說的了，看這個遊戲後續發展和結局吧。至於一堆垃圾公告數據吃了500M流量這件事，不說也罷。
