DNF：100級版本魔戰套“復活”，屬性能進行改造，比超界防具都強

2019-12-20 639條評論 1,731次閱讀 406人點贊

100級版本新增「智慧產物」史詩，能在拍賣行上架，值得一提的是，當搭配85級史詩套，可額外的改造屬性。通俗一點說，魔戰套這類「過氣」史詩，又被強行的「復活」，在100級版本，改造之後屬性比超界都強！

「智慧產物」，比較的特殊，雖然可上架拍賣行交易，但是，無法增幅、強化、附魔和鍛造，這一點就很傷。畢竟我們知道，增幅附魔對一個裝備來說，這是提升的途徑。你不打造裝備，看起來跟白板一樣，檔次就顯得低一些，進團都較為困難。

不過，值得一提的是，「智慧產物」左槽裝備，搭配85級史詩防具五件，會觸發改造屬性！以魔戰套為例，其專屬的「智慧產物」左槽是凝縮的魔力漩渦，本身力智+146點，體精為46點，攻擊時附加40%的傷害。當玩家穿魔戰套，左槽是凝縮的魔力漩渦時，會觸發裝備改造屬性，魔戰套等級提升15級，變成100級史詩，且加成100力智面板！

而改造後，魔戰套1階段暴擊傷害+40%，3階段最終傷害+25%，三攻+10%，技能攻擊力+10%，力智+42%，5階段技能攻擊力+33%，最終傷害+30%。對比超界防具屬性，你會發現，在改造後的魔戰套面前，顯得十分脆弱，有種被「完爆」的感覺！

對於魔戰套史詩，玩家都非常熟悉，其刷圖CD技能很舒服，加上這麼高的傷害，毫無疑問會「碾壓」超界。當然，這只是舉例一個魔戰套，85級的史詩裝備，都各自有一個「智慧產物」史詩左槽對應，只要搭配之後，就可以觸發改造屬性。

比較「良心」的是，「智慧產物」史詩在拍賣行上可以買，不用「肝」副本。但是，這麼強的一個屬性，可見，「智慧產物」史詩裝備，一出來就會是「天價」，也只有土豪能買得起。一些玩家85級史詩，早就已經分解，而今能進行改造，變成比超界更強的存在，怕是後悔莫已！總之，魔戰套在100級版本，算是徹底崛起！
