魔域手游首個國風職業登場！御劍師降臨

2019-12-03 0條評論 1次閱讀 0人點贊

摘要

在去年這個時候，《魔域手游：精靈紀元》資料片正式公布，為各位玩家們帶來了集美型和強力為一體的精靈遊俠，引起了一輪”精靈熱潮”。今年12月，《魔域手游》將帶來全新資料片御劍天下。中國風新職業”御劍師”首度曝光，他將御劍乘風來，與玩家開啟一段全新的冒險。

劍是中國男兒的一種精神象徵，它不像其他兵器純粹代表著殺戮、武勇等，也是地位，身份，品味的象徵。古人認為劍為君子之兵，號稱百兵之君，其主要由於劍為直身雙刃，代表著君子正直，克己的美德。

也正是因為自古以來產生的”劍崇拜”，各種仙俠類作品裡總是會出現一位超凡脫塵的劍俠。《仙劍1》里”御劍乘風來，除魔天地間”的酒劍仙，是幾乎每一位80、90後男孩的童年回憶；戰國時，蘇秦身背長劍橫跨六國，促成了著名的”六國合縱”，迫使強秦不敢出函谷關十五年；《雪中悍刀行》里的那句：”天不生我李淳罡，劍道萬古長如夜！”代表了劍客放肆的張狂與囂張。

這些劍客無一不代表著飄逸和帥氣，而且往往還會擔當起戰鬥力天花板的能力。而今年年末， 《魔域手游》的全新國風職業”御劍師”也將跟隨新資料片的劃破虛空，御劍而來。正是《魔域手游》對玩家心中劍俠情結的完美致敬！

在西幻背景中加入御劍師是《魔域手游》進行的一種大膽嘗試，為玩家們帶來最熟悉的東方故事，也代表著《魔域手游》以”國風”為載體進行的一次多元化嘗試，讓玩家在更為豐富的遊戲體驗中感受更加深遠的文化魅力，也是遊戲心系玩家努力帶來新鮮內容的最佳佐證。

雖然目前有關御劍師的具體內容還需要賣個關子，但是劍出如虹，御劍飛行這些有關劍仙的瀟洒設定肯定是都會有的！加上新資料片本身會帶來的各種新內容變化，2019年最後的大更新一定能給各位玩家帶來全新全異的遊戲體驗，滿足你關於劍仙的一切幻想。

全新職業”御劍師”不僅將成為首個以劍為核心武器的職業，更是出場自帶國風劍俠氣質。相信這個代表中國數千年劍文化傳承的職業，一劍東來之時，定能給雷鳴大陸帶來翻天覆地的變化。
