摘要

神秘危險的克蘇魯風一直都有大量的熱衷者。而今，一款克蘇魯風的橫版闖關遊戲——《異化之地》已經誕生。在遊戲中，踏入此地的玩家會在rogue-like機制生成的隨機地圖上，並遇到種種邪惡畸形的恐怖生物，擊敗它們能夠獲得隨機的裝備、道具、神印。遊戲於12月6日開啟測試。

神秘危險的克蘇魯風一直都有大量的熱衷者。而今，一款克蘇魯風的橫版闖關遊戲——《異化之地》已經誕生。在遊戲中，踏入此地的玩家會在rogue-like機制生成的隨機地圖上，並遇到種種邪惡畸形的恐怖生物，擊敗它們能夠獲得隨機的裝備、道具、神印。遊戲於12月6日開啟測試。

當邪惡的黑魔法重現世界，恐怖的異類生物趁機污染了人間，整片大陸成了怪物橫行的異化之地。黑暗也變得不再純粹，踏足黑暗的人往往一去不復返，危險和恐懼成了這片大地的主旋律。而你，就是一位不幸踏足此地的旅者。

因為橫版打鬥遊戲一路闖關的直白戰鬥體驗，所以不得不說，《異化之地》很難帶給克蘇魯愛好者，那種詭秘且危險的純正體驗。但如果你不窮究百分百的克蘇魯，那麼從背景到畫面，都包含著滿滿克蘇魯元素的異化之地一定會讓你覺著這遊戲不錯。

當然，《異化之地》遊戲內容並不是只有戰鬥。在玩家闖入這片世界時，雖然是隨機生成的地圖，但還是會隨著闖關的次數增多，對遊戲劇情的了解越來越深，也會看到一幕又一幕邪惡異形怪物污染人類衍生的悲劇故事。

據悉，最初版的《異化之地》原本是一款單機遊戲。但由於單機手游太過容易破解，以至於遊戲還在測試期，破解版去賣得盆滿缽滿遊戲，所以開發者由於這些額外的麻煩，將原本純單機的《異化之地》改成了必須雲端同步的「聯機型」單機。

但遊戲的付費方式並沒有改變，依舊是一元解鎖一個角色。然後可以根據玩家的個人愛好，選擇是否解鎖另外三位角色。必須提到的是《異化之地》遊戲內有不少的其他內購，比如隨便購買、特級道具、特權……。

如果《異化之地》是一款網遊，遊戲內存在這樣直接破壞平衡的商店，無疑是自尋死路。可它是一款單機，除了雲端連接防止破解之外，遊戲內並不存在網遊類型的內容，所以目前為止，玩家對此並不反感。目前，《異化之地》正在火熱測試，各位想要嘗一嘗鮮的小夥伴可以一試。
