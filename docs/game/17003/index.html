DNF快速了解4大100級內容，三覺新史詩都在其中，再戰十年沒問題

2019-12-14 140條評論 1,378次閱讀 128人點贊

冬季發布會結束了，策劃說了一個多小時，內容非常多，可見其準備很充分。小編這裡給大家梳理一下發布會的內容。主要都是一百級的新東西，涉及四個方面，分別是一百級副本，三次覺醒，裝備繼承系統，然後是新的裝備系統。從發布會的角度看，誠意還是很足，讓人比較滿意。

一、裝備系統。

一百級的史詩防具一共有十五套，每一類防具有三套，屬性全面吊打九十五級史詩。武器每個職業有兩把，首飾四套，特殊裝備四套。還有比史詩更強大的「神話」裝備，可交易的粉裝和傳說重現江湖，簡直就是黑商的春天。然後就是裝備屬性全面可以繼承。最重頭的大戲是出了可以交易的史詩裝備，這個有點驚悚。

二、三次覺醒。

三次覺醒先行開放了女鬼劍和男格鬥家共八個職業，人物插圖和技能都做出來了，非常的炫麗，好看是沒問題的，就是有點燒顯卡。其他職業的三覺暫時沒有開放，女鬼劍三次覺醒將在一月九日實裝到韓服正式服，男格鬥三覺將在二月六日實裝到韓服正式服。為什麼沒有男鬼劍三覺？這個就要問策划了，不過即使開放晚，也不會很晚。

三、一百級副本。

總體上分為兩類，一類是非團本類型，可以單人或組隊進入，怪物血量挺厚的，甚至還能放紅眼二覺；第二類是團本，就是希洛克團本了，是八個人還是十二人，還是二十人的團本，暫時不知道，韓服應該還沒有完成製作，具體要等後面的更新。新史詩和神話裝備都由非團本副本產出，希洛克副本短期內不會實裝。

四、裝備繼承系統。

除了史詩升級時可以繼承屬性外，可交易的裝備也能繼承屬性，而且這個繼承是相互的，既可以把史詩裝備屬性繼承到粉裝上，也可以把粉裝屬性繼承到史詩裝備上。但是有一定的限制，比如說可交易的史詩不能打紅字，自然也不能繼承紅字屬性了。

最後說一下其他的一些小更新，比如說技能欄可以移動到屏幕中間了，城鎮間的移動可以使用飛行器瞬移，拍賣行取消，由新的交易系統代替，南部溪谷也重新做出來了，當然了這些更新都不重要，重要的就是上述四大更新。三次覺醒和神話裝備將是未來一百級版本的重頭戲，從發布會來看，挺不錯的，但具體怎麼樣，還要看以後的更新。
