夢幻西遊中大唐官府如果反間計可對人物使用，那麼大唐必將崛起

2019-12-11 338條評論 1,435次閱讀 431人點贊

​ 快樂遊戲，幸福生活！大家好，這裡是連載型頭條號《我說夢幻西遊》！致力於做有深度的夢幻西遊頭條號！這是小編第300篇原創文章！（這麼值得紀念的日子，各位還不點個贊~）

​大家都知道，大唐官府一直都是夢幻西遊的信仰門派，什麼骷髏大唐、血耐大唐、力敏大唐、全力大唐等各種流派都曾稱霸一時。但是大家還記不記得，最初大唐官府還有一個技能叫做反間計的，以前是特別丑的施法特效，現在更改過來了，變得有點酷炫了，但是現在限制了只能對NPC使用。

還記得小編以前的時候很喜歡在大雁塔上對著兩個羊頭怪使用反間計，看著他們互相拿著叉子戳戳戳感覺自己好開心呀，壞人打壞人，自己看戲，不用費一點血，最後還能獲得經驗，真的很不錯。但是後來自己變的厲害了，基本的人物都是一回合結束根本不給出手機會，因此慢慢的這種樂趣沒有了！

但是最近突然腦洞大開想到：如果夢幻西遊中大唐官府反間計可對人物使用會怎麼樣？會不會出現大唐官府作為封系的情況呢？PK場上會不會出現四打六、三打七，甚至是16打4的局面呢（包含召喚獸）？

如果大唐官府的反間計能對人物使用，那麼大唐如果作為封系肯定是一點力量都不加了，速度、抗性、大特技搞起，一直對對戰方的輸出單位和召喚獸使用反間計，哇瑟，想想那場面，對戰方窩裡斗，你打我一下，我錘你一下，場面一度陷入混亂，想想場面就非常有喜感！

大唐作為封系，首先是五龍丹的消耗會加劇！

因為大唐的封印不同於其他封印，他可以使得召喚獸亂打自己人，那麼就沒有機會和能力來對己方的蛇魅或者召喚獸造成傷害了！其次，對於物理系而言，如果打自己隊友還好，如果砍到了自己的召喚獸身上那就慘了，這才是真正的五打四呢！所以說，五龍丹和晶清訣一定要配合好，不然真的要被玩死！

大唐作為封系，還可以點殺呢！

大家都知道，現在的大唐已經不完全依靠橫掃千軍來完成點殺了，當靈源足夠，大唐的一個斷穹巨劍就是堪比橫掃千軍的存在，這樣一想，是不是無論是點殺還是絕殺，帶上大唐作為封系都是很划算的呢！畢竟絕殺肯定是佔上風的呢~

大唐作為封系，迷惑性極強！

如果你看到對面上了大唐、獅駝、女魃墓、地府/女兒村、化生寺，你是不是第一印象就是三攻隊伍了，是不是很慌，上去什麼放下屠刀啥的都丟給大唐了，然後大唐出手就是一把反間計，你的力劈壁壘啪嘰一下拍在你身上，你是不是有點蒙圈呢？

但是大唐作為封系也是有很多缺點的

最主要的一點就是抗性差，速度慢和封印比較單一！大唐的師門技能決定了他沒有那麼快的速度，而且封印只有一個反間計，但是遊戲嘛，快樂就完事兒了，你覺得呢？

好了，這一期就到這裡吧。歡迎關注《我說夢幻西遊》，連載型頭條號，致力於將夢幻西遊的趣聞講清楚，拒絕水文，吸引高品質讀者！記得往期內容更精彩哦！
