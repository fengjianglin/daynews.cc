大家好，我是想稱霸海上卻沒有船的遊戲日報王路飛同學。近日有玩家在論壇稱鐵男這個英雄簡直太OP了，自己完全限制不了他的發育。雖然現在征服者改回去後鐵男是很猛，但這幾個英雄卻天然克他。

這位玩家說他前期都壓著鐵男打，但是6級過後直接被打成麻瓜，己方打野來了也沒用，對面直接開大點名，搞不好就等於白送。很多網友猜測這位玩家可能選的亞索，不然不會被打得如此狼狽。

其實鐵男強的點就是單挑，那麼選個更強的單挑英雄豈不是就成了？有人說船長是鐵男的指定親爹，一手橘子隨時解大，但並非如此，船長單挑在前中期還真不好，鐵男冰杖出來後就算沒大也能追著船長打。

鐵男真正的親爹應該是奧拉夫和諾手，論單挑這兩個英雄就沒怕過誰，開大點名就等於自尋死路（雙方水平相近）。尤其是奧拉夫，一點不怕現在的征服者鐵男，開大後跟瘋狗一樣追著砍。

諾手面對鐵男也占很大優勢，主要是因為鐵男本身比較笨重，除了有閃現的時候幾乎都不好躲開諾手的外圈Q。雖然鐵男也有盾有回血，但是比起諾手的流血加大招根本不值一提。

說完親爹還有親媽，那就是劍姬，鐵男的技能前搖普遍較長，對上有W技能的劍姬完全不討好，劍姬很輕鬆就能反制鐵男的E。加入劍姬又靈活，鐵男的Q很難命中，這就直接相當於鐵男就好像沒有Q和E一樣，怎麼可能打得過？

LOL這款遊戲已經有10年了，因此不可能存在什麼無敵的英雄，整體的遊戲機制都是很平衡的，只要找對方法，總能和對面五五開。

而且就算有時某個英雄確實很OP，但過不了多久就會被削弱，就比如塞拉斯、AKL、刀妹等一時風光無限的英雄。

因此大家在玩的時候有空還是多看看攻略，這也是提升自己遊戲水平的方式之一。
