王者榮耀最貼心的設置來了，技能大小隨意調整，成就系統革新

天天要聞
4 天 ago

大家好，這裡是指尖上的王者，12.13日體驗服更新了s18賽季的內容，其中包含了英雄調整、峽谷格局調整、峽谷生物調整、模擬戰大改以及無限亂斗的更新，本文跟大家講一下絕大多數都會很期待的一些細節改動。

1.戰隊全國排名登場

戰隊可選擇戰區，其中包括【突圍賽】、【區級聯賽】、【市級聯賽】、【省級聯賽】、【國家級聯賽】，當獲得排名的時候，隊員在進入遊戲界面的3秒內會展示上圖，看起來有沒有很炫酷呢？

在獲得一血、超神、團滅、勝利後，會彈出上圖中的頭像，看起來逼格滿滿，關於戰隊賽，小編幾個賽季沒打過了，因此對這個也不是很期待，具體規則大家可以去官網看一下。

2.成就系統展示

在主頁被更改之後，成就系統只能被自己查看，哪怕小編全服第十也沒人知道，為此很多人還刻意開貼吐槽這個問題，而在s18賽季更新之後，你的成就就可以展示出來了。

成就系統會展示在個人主頁的選項卡上，點擊後就能看到你的排名以及成就獲得的數量，不再只是自己看，所以有了這個成就系統，會逼迫很多人自覺性的重視這個系統，因為會提升自身的存在感啊。

3.掛機不升星

掛機輸掉比賽會扣分掉星，但是被玩家拖著贏下了比賽，還要給他加星，很多玩家寧可輸也不願意讓這種人撿到便宜，s18賽季就沒有這個擔憂了，哪怕隊友掛機也要爭取贏下比賽，因為系統一旦判定為掛機（對局時間的一半以上），那麼即便贏下比賽也不會加星，還要扣掉積分，我只想說幹得漂亮。

4.更便捷的快捷消息

隊友發布指令後出現一個收到的按鈕，可以讓隊友知道你會配合他的行動，而不需要進入快捷消息界面去找，另外多了一些更簡單的戰術交流，拖動右邊的按鈕到草叢，會出現「和我一起埋伏」的消息，拖到自己技能上，會出現技能冷卻多少秒的提示，另外還能發送敵我經濟差，告誡隊友不要衝動。

5.按鍵布局調整

這是最重要的一項設定，早就饞哭了隔壁和平精英的按鍵調整，這次王者玩家終於能用上了。

很多人吐槽補刀鍵太小，技能位置阻擋視線，技能間隔不舒服，不用擔心，s18賽季將一切全部解決，按鍵的大小以及透明度全部能夠調整，補刀鍵小？那就調大，技能阻擋視野？那就變成透明，不得不說這個設定是千呼萬喚始出來啊。

這個調整可以在設置中隨意設定，在對局中同樣可以開啟，非常的方便。
