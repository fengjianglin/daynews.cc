CFHD全明星賽陣容已經公布 Lsn頂替寵兒出戰CFHD賽事

天天要聞
4 天 ago

在CFS世界總決賽期間，不僅有著精彩的CFS世界賽，可以看到各支國際戰隊和中國戰隊在一起交戰，精彩至極，然而除了精彩的CFS世界總決賽，還有CFHD中外全明星競技賽，據說中外明星競技賽的隊伍的陣容都非常的豪華，那麼陣容究竟是哪些呢？

CFHD中國全明星競技賽的陣容是根據票選結果出來的，根據結果來看，目前參賽陣容為SV戰隊的馬哲和念舊，AG戰隊的絕跡和寵兒以及白鯊戰隊的Even。在這方面來看，這套陣容是非常強大的，稱他們為史上最強中國戰隊也當之無愧。

但是，意外都是會來臨的，AG戰隊的寵兒因為個人身體原因無法參加CFHD的中外全明星競技賽，這是非常可惜的，畢竟我們想看到寵兒和念舊兩位絕代雙驕的發揮。可惜寵兒的退賽導致我們無法看到絕代雙驕的發揮，寵兒的位置由Lsn代替出戰。

接下來，再讓我們看一下外國全明星競技戰隊的成員，這裡的成員由兩支戰隊構成，一支是歐洲GV戰隊，一支是歐洲AT戰隊，成員分別是歐洲GV戰隊的bL4ME、phaze以及SEVEN和歐洲AT戰隊的MooN和nLs，也是一支非常強勁的隊伍。

比較遺憾的是，我們本以為外國全明星競技戰隊來源是多個戰隊的，不過均是來自於歐洲戰隊的，巴西戰隊那群槍男很遺憾沒能夠入圍，不過這支歐洲隊也非常的強大，裡面的選手都與中國戰隊交手多次，也有過曾經擊敗中國隊的戰績，不容小覷。

更為重要的是，這次比賽的領域不是CF，而是CFHD，是一個全新的領域，很多操作和CF是不一樣的，而且畫面也是不一樣的，我們不知道我們的中國戰隊和歐洲聯隊，哪一支戰隊能夠獲得首屆CFHD國際賽的勝利呢？12月15日，一同見證！
