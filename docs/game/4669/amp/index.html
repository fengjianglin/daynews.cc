英雄聯盟：前期掏出這幾位英雄，拿不到優勢的話後期堪比超級兵

天天要聞
2 週 ago

摘要

英雄聯盟這款遊戲上線已經有10年時間了，而設計師為了讓玩家能夠在遊戲中獲得更好的遊戲體驗，每位英雄的登場都是經過設計師精心雕琢的，各位英雄的技能也都是該英雄玩法的重要組成部分，每位英雄的玩法都是不一樣的，而設計師為了平衡總會有些英雄在前期強無敵，在後期就弱無力了。

英雄聯盟這款遊戲上線已經有10年時間了，而設計師為了讓玩家能夠在遊戲中獲得更好的遊戲體驗，每位英雄的登場都是經過設計師精心雕琢的，各位英雄的技能也都是該英雄玩法的重要組成部分，每位英雄的玩法都是不一樣的，而設計師為了平衡總會有些英雄在前期強無敵，在後期就弱無力了。

那麼下面就讓我來為大家介紹一下這些前期強無敵的英雄。

荒漠屠夫：

鱷魚這個英雄是玩家們公認的線霸，三級鱷魚在積攢好怒氣的情況下對拼不虛任何英雄，就是諾手也不是鱷魚的對手，E技能兩段位移和W技能穩定的控制以及紅怒Q的高額傷害和回復能力讓鱷魚在三級是絕對的大哥大，也就潘森能夠略壓一籌了，相信大家經歷過三級鱷魚到底有多麼恐怖。

三級的鱷魚只有攢夠100的怒氣，就可以之間秒掉對面英雄了，如果鱷魚沒有怒氣的話也可以差不多打到半血。如果選了鱷魚去打上單的話，基本就要打出單殺來取得前期的優勢，靠正常發育的鱷魚前中期都是非常頂的，所以選出了鱷魚就要前中期打出壓制的效果滾起雪球。不然後期的鱷魚會很容易讓人失望，後期鱷魚就如同一名超級兵一樣無力。

老版潘森：

潘森這個英雄強勢就強勢在前期，六級前少有英雄是潘森的對手，潘森的Q技能在前期的傷害是非常誇張的能夠利用Q技能不斷的進行遠程消耗，再利用W技能眩暈然後點燃加E技能完全可以帶走任何英雄，被動還讓潘森能夠免疫普攻和塔的傷害十分變態，相信都有過前期被潘森統治的經歷，六級之後擁有大招的潘森還能夠到處支援加快遊戲節奏給隊友建立優勢，一套技能基本都能帶走任何英雄，十分強勢。

不過越到後面潘森的作用就越小了，一套爆發秒不了人基本就要被反秒了，而且前期你隨便欺負的人都可以隨便欺負你了。

盲僧：

盲僧這個英雄的強勢點就是在前期，前期技能傷害的加成非常高，野區入侵能力非常強，在野區也是橫行霸道的，基本被Q到就要被打到殘血了，並且前期抓人能力也是非常的強，而且可以利用W摸眼進行追擊，E技能也可以打出不錯的傷害，基本抓人就是被Q踢上的話不交閃的話就會把命給交代了，到達6級之後的斬殺線又會提高很多，基本一個人就可以秒掉一個，也可以用R閃或者W摸眼R的迴旋踢把對方C位提到自己陣容面前，前中期滾雪球能力非常強。

如果盲僧前期沒有抓到機會的話，毫無作為就只能宣判是在慢慢死亡，因為盲僧後期除了比超級兵強的一點就是還可以用R踢踢人，但是踢不到的話真就和超級兵差不多了。
