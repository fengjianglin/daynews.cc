曾有DNF玩家8年搬磚換了一套房，現在還有什麼網遊平民能賺錢嗎？

2019-12-12 2,940條評論 4,956次閱讀 2736人點贊

2016年，在網遊搬磚界發生了一件震驚全圈的事情：一位DNF玩家曬出自己搬磚8年的積蓄，換了鄭州三環的一套房子，此舉秀出了網遊搬磚的至高境界，壕無人性、令人羨慕。

如今，時間馬上就要來到2020年，當手游狂潮席捲而來，端游的日子似乎越來越難，很多網友覺得當年電腦五開歡樂搬磚的日子恐怕一去不復返了。然而在2019年端游卻出現了復甦的跡象，這大概是經歷了手游狂歡之後，越來越多的玩家領悟到了端游品質更高、粘性更強的道理吧。因此很多玩家發出靈魂追問：2020年還會有DNF換房那樣的好事出現嗎？

搬磚戰鬥機DNF：收益下降，仍不失為搬磚聖地

能不能換房不知道，但是網遊搬磚賺錢時至今日還是可行的。就比如DNF吧，之前很多新聞都說DNF搬磚涼，不可否認DNF的搬磚賺錢黃金期已過，近年來DNF甚至還出現了大規模退坑潮，更是爆發了發票事件震驚圈內外，讓人覺得DNF已經不適合搬磚。但是百足之蟲死而不僵，DNF競技性強、有騰訊爸爸的BUF、更有八百萬勇士的情懷在線，通過DNF搬磚維持溫飽還是毫無疑問的。

老牌MMO：持續堅挺，迎來搬磚第二春

除了像DNF這樣的橫版格鬥，如果想玩角色扮演類的老網遊的話可以選擇網易的《夢幻西遊》、《魔獸世界》。夢幻作為至今堅挺的老回合制網遊，賣寶寶、打裝備賺錢都是常態，動不動就有玩家一把武器賣50萬元、一個賬號換個法拉利，都是令其他網遊玩家只有狂吃檸檬的份兒。

而《魔獸世界》本來就是帶起國內第一批打金狂潮的遊戲，在今年開了懷舊服之後引起端遊玩家瘋狂湧入，無論是魔獸老玩家還是90後、00後玩家都再次見識了這款經典MMO魔幻網遊的魅力不減，可以說：魔獸世界懷舊服又掀起了一股搬磚打金的熱潮，代練和賣金迎來第二個黃金期。

新游搬磚首選《逆水寒》：保底每月8000元？另類搬磚方式花眼

如果想玩新游的話就不得不提網易的武俠網遊《逆水寒》了，很多人一直說逆水寒會涼，但不可否認這款遊戲熱度一直都在，並且是現在新網遊中最適合搬磚的。

《逆水寒》在上線之初憑藉出色遊戲品質和高調宣傳吸引到王思聰為代表的眾多大佬玩家入駐，可以說是土豪最多的一款網遊；官方在2019年上半年又開啟了結構性改革給平民玩家減負減肝，花樣翻新地出新玩法，所以基數一直比較穩。逆水寒去年上線時候頻頻有玩家曝出月入幾萬、甚至為了搬磚辭職不上班的新聞。現在雖然收益有所減弱，但是老手搬磚玩家月入保底8000多元還是很輕鬆的。

在《逆水寒》中想要賺錢的方式很多，除了日常任務、在線押鏢、爆紫裝、養角色這些常規操作之外，鬼才玩家設計的各種風格的莊園也能賣出好價錢，甚至一張莊園圖紙都能掙個幾百塊；還有主播通過幫人砸裝備打鐵賺錢，比如虎牙浮誇兄弟，一單幾十萬替大佬打裝備是常有的事情。

此外，由於逆水寒風景美爆、角色精緻，還專門誕生了替人拍照的網遊攝影師通過接商單來賺錢，有技術過硬的專業跟拍一天也能賺個幾百塊，實在令人嫉妒。

甚至還有專業乞討發家致富的玩家……真是令人大開眼界。

除了以上提到的新老網遊，其實還有很多端游可以搬磚掙錢的，比如、《倩女幽魂》、《劍靈》、《完美國際》等等，只要勤勞活躍+思維開闊就會生財有道。當然，遊戲適度，不能為賺錢傷肝才好。

這些年，端游雖然一年比一年少，但是老網遊和精品新游的堅挺證明了端游並沒有被玩家拋棄。隨著手游亂象頻發，勢必還會有更多的玩家回到端游的懷抱中，所以就不要問”端游還值不值得投入時間和精力”這樣的問題了，2020年我們搬磚繼續走起！
