brollan狀態爆表提高Fnatic全隊上限，面對Liquid有恃無恐

解析：

今天的這場比賽推薦是有FNATIC戰隊對陣LIQUID戰隊的BO3對局。對於FNATIC戰隊來說，隊伍在上一場面對100T戰隊的比賽中，以直落兩盤的結果順利戰勝了對手，從比賽過程來看，FNATIC戰隊的隊員整體調整能力非常出色，在第一把比賽中期難解難分的情況下，隊伍在後期通過正面進攻和充分的溝通指揮接連贏下了關鍵對局，進而順利得贏下了第一局，而在第二把比賽中隊伍則沒有給予對手太多的機會，所以綜合來看FNATIC戰隊目前的狀態並沒有受到賽程的影響，緊密的賽程反而給予了他們保持狀態的機會，隊伍核心隊員flusha和golden狀態穩定，在隊伍的正面幫助隊伍取得優勢，而狀態爆表的brollan選手繼續發揮著個人實力去幫助隊伍提高上限，倘若隊伍能夠保持這樣的狀態，我認為FNATIC戰隊會走得更遠。

而另一邊的LIQUID戰隊，在上一場面對A隊的比賽中，隊伍再次以1：2的比分落敗，從比賽過程來看，LIQUID戰隊還是再次敗給了自己的狀態起伏，對於隊伍而言，過於吃槍法的他們在狀態上會顯得起伏，在上一場面對A隊的第三局比賽中，隊伍僅僅只有NAF和ELIGE兩位選手有著穩定的發揮，其餘三人明顯受到了地圖的換邊影響，所以綜合來看LIQUID戰隊更多的戰術還是要靠著自己的槍法，倘若狀態出現不穩定的情況，那麼隊伍整體的實力發揮將會大打折扣。綜合這場比賽來說，我認為兩邊的競技狀態保持都是各自比賽表現的隱患因素，看好這場比賽BO3打滿，推薦FNATIC戰隊+1.5.
