LOL-德瑪西亞杯：VG一波打死SN，iBoy拿下為新東家效力的首個MVP

2019-12-22 2,388條評論 4,608次閱讀 2448人點贊

德瑪西亞杯小組賽杭州場，賽前訓練賽超強的SN與重組後的VG迎來對陣。

SN

上單劍魔，打野酒桶，中單小炮，下路ADC女槍和輔助泰坦

VG

上單奧恩，打野奧拉夫，中單塞拉斯，下路ADC賽娜和輔助塔姆

本場比賽開局後雙方一直處於和平發育狀態，直到9分鐘，上路奧恩單殺劍魔拿下第一個人頭，隨後小炮和酒桶趕到將奧恩帶走。一分鐘後，下路女槍配合酒桶泰坦將塔姆擊殺。14分鐘，下路奧拉夫召喚峽谷先鋒幫助己方下路組合耗掉防禦塔的大半血量，隨即拿下小龍。隨後雙方開團，猶豫不決的SN終於趕到，但隨後女槍走位脫節遭到賽娜帶隊擊殺，隨後劍魔也在正面戰場被帶走。

16分鐘，拿下優勢的VG中野遲遲不願意走，在中路與對手繼續纏鬥，結果SN中野憑藉著一波2V2挽回局面。22分鐘，劍魔也憑藉著峽谷先鋒破掉下路防禦塔。26分鐘，迷路的SofM在上半野區逛街被抓，塞拉斯收下人頭後帶隊動手大龍順利拿下。

27分鐘，劍魔下路帶線卻遭到埋伏，SN四人從草里衝出將劍魔送回快樂老家。29分鐘，VG推上紅色方高地，破塔後直逼基地，隨後小炮帶隊防守卻遭到反打，VG將SN打成四人團滅，拿下了比賽的勝利。
