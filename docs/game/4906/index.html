第五人格：只因一個機制，屠夫“地窖”前投降，玩家不喜反悲

歡迎各位小夥伴來到本期的《第五人格不知道多少課》「遊戲機制」今天呢~公瑾就和大家來聊一聊《第五人格》中所謂的「遊戲機制」，在遊戲中官方設定了許多的遊戲機制，從而讓遊戲環境達到了一個平衡的效果，但是不少玩家卻對「地窖」前投降產生了疑問！

好啦，話不多說，下面趕緊和公瑾來了解一下吧！第五人格：只因一個機制，屠夫「地窖」前投降，玩家不喜反悲！

在我們的《第五人格》遊戲中，隨著玩家們的段位越來越高之後，匹配到的「求生者」或者「監管者」的實力也會隨之上升，但是在遊戲對局當中，不管是「求生者」一方或者「監管者」一方，只要出現過多的失誤的話，都會導致比賽的失敗。

所以說很多玩家在進行「排位賽」的時候，都是選擇一個「保平爭勝」的心態去進行遊戲，畢竟這樣一來我們才能夠有著更好的發揮，從而贏得比賽的勝利。

但是呢~隨著賽季未的來臨，遊戲對局當中「監管者」出現「地窖」前投降的情況也是越來越多了，不管是監管者放飛了3個求生者，還是求生者3出之後想要跑地窖。監管者們都會果斷選擇投降，那麼這樣的選擇到底是因為什麼呢？（相信小夥伴們也非常想知道吧？）

其實監管者之所以會在「地窖」前選擇投降的原理非常簡單，首先很多「求生者」在暫居了有利的地形之後會和監管者繼續僵持下去，從而拖延了比賽的時間，所以監管者們才會選擇投降的。

不過除此之外，其實還有著另外一種原因，那就是因為官方所設定的一個「遊戲機制」！當監管者在「地窖」前進行投降的時候，雖然說監管者沒有4飛的額外加分，但是「求生者」也會因為沒能夠從「地窖」逃跑而掉分，所以遊戲中才會頻繁出現「地窖」前投降的情況。

大家都知道，當我們晉陞到「殿堂級」之後，想要繼續提升段位是非常困難的，所以每一個玩家都會想方設法的去「增加」自己的分數或者是「減少」敵對玩家的加分等等~

而以上的這種情況，相信小夥伴們在目前遊戲中也會經常遇見吧？雖然說很多時候我們在使用「求生者」的時候，好不容易憑著實力「遛」監管者才來到「地窖」前的，但是這個時候監管者卻投降了！（對於這樣的事情，很多玩家並不會感到歡喜，反而會感到傷悲~畢竟遛鬼不易，加分更難嘛！）

好啦，本期的《第五人格不知道多少課》到這裡就要結束了，不知道大家覺得這個「遊戲機制」怎麼樣呢？或者是說小夥伴也覺得「地窖」前投降屬於一種「戰術」呢？接下來不如讓我們在評論區好好地聊一聊吧！
PS：最後非常感謝大家的閱讀與支持，喜歡公瑾的不妨點個關注喲！嘿~別忘了點贊、轉發喲！
