《小小夢魘》開發商Tarsier被THQ Nordic母公司收購

2019-12-21 189條評論 1,196次閱讀 244人點贊

平台解謎遊戲《小小夢魘》在推出後廣受讚譽。其開發商Tarsier Studios現已被Embracer Group公司以1,050萬美元的價格收購。

可能不少玩家對Embracer Group這家公司不太了解，但提到THQ Nordic相信沒有玩家不知道這家公司的大名，實際上Embracer Group的前身就是THQ Nordic AB，該公司現在擁有子公司發行商Koch Media和THQ Nordic。

這份報告來自Gamesindustry.biz，其中指出，「該協議包括保留所有65名Tarsier員工，並將其知識產權交付給Embracer。」

不過，《小小夢魘》現在仍歸萬代南夢宮所有，而開發商的另一款IP《Stretchers》歸任天堂所有。

當然，Tarsier仍將保持自主權，工作室是由一個Embracer旗下的集團、名為Goodbye Kansas Game Invest的團體負責運營。目前，開發人員將繼續集中精力為發行商萬代南夢宮完成《小小夢魘》的續作，之後才會為Embracer開發新IP。

《小小夢魘2》將於2020年登陸PS4、Xbox One、Switch、PC。
