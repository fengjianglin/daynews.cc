遊戲編年史：“OXO”電子遊戲歷史上最早開發的遊戲之一

介紹

《OXO》是亞歷山大·道格拉斯（Alexander S. Douglas）在1952年開發的一款井字遊戲，它是開發在電子延遲存儲自動計算器（EDSAC）上，EDSAC是最早的存儲程序計算機之一，具有可讀取或寫入的存儲器，有三個小陰極射線管屏幕來顯示存儲器的狀態，道格拉斯重新設計了一個屏幕來向用戶演示其他信息的描繪，例如一個圈和叉的遊戲狀態，所以就開發了《OXO》這款遊戲，與其他早期的電子遊戲一樣，在遊戲達到目的後，該遊戲被丟棄，《OXO》並沒有對大眾公開，只能經特殊許可在劍橋大學數學實驗室播放，因為EDSAC計算機過於龐大不能移動，並且計算機和遊戲都僅用於學術研究目的，《OXO》是目前已知的最早在電子屏幕上顯示視覺效果的遊戲之一，在某些定義下，它可能有資格成為第一款電子遊戲。

玩法

遊戲由一名玩家和人工智慧對戰，玩家使用一個旋轉電話控制器輸入，選擇他們希望下一步移動的九個方塊中的哪一個，他們的動作會出現在屏幕上，然後電腦會跟著移動，只有當遊戲狀態改變時，遊戲顯示才會更新。

結尾

早期的電子遊戲基本都是用於學術研究，在達到目的之後，就被丟棄，但是電子遊戲的種子，在那時就已經被種下，在多年後，經歷風雨，開枝散葉，最後成為一顆參天大樹。
