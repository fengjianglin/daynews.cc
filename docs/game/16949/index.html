垂死掙扎？絕地求生北美賽區建立新聯賽，但48支隊伍只有8千獎金

2019-12-15 2,073條評論 3,429次閱讀 877人點贊

2019年對於北美絕地求生來說無疑是異常艱難的一年，自FGS以後，不少俱樂部都解散了他們的吃雞分部，其中不乏C9等世界頂尖的戰隊。

本以為在北美舉辦的PGC全球總決賽會讓這一形勢有所好轉，但就目前的情況來看，形勢不僅沒有好轉，反而還更加惡化了。

C9原班人馬組成的Genesis在PGC後也幾乎解散了

PGC結束之後，FGS亞軍LZR（前STK）戰隊和第一屆SLI冠軍Ghost先後解散，Kaymind等知名的北美選手也選擇遠赴歐洲繼續職業生涯，北美絕地求生的前景看上去十分不樂觀。

值得一提的是Ghost教練曾公開懟藍洞高層，說他們只顧著出主播皮膚賺錢，根本不在意職業戰隊的死活，這或許是讓吃雞俱樂部連續解散的現實原因。

此外，近期藍洞在《2020年PUBG電競賽事規劃》中宣布會建立全新的賽事機制，PEL和NPL賽區的長賽制聯賽將會被解散。

就長期收益來看，賽制的更新很有可能提升這兩個聯賽參賽隊伍在國際賽場的競爭力，但卻又在某種程度上削減了他們本就不多的收入，這對北美俱樂部來說無疑是雪上加霜，不過他們似乎並不打算坐以待斃。

Jemmzz直播時曾證實PEL和NPL聯賽要解散了，新賽事即將建立

據愛吃雞的小女孩槐序所知， NPL賽區已經建立了名為「303ROYALE」(303 PUBG PRO/PRO-AM LEAGUE)的新聯賽以取代即將解散的官方聯賽，不過賽制還沒有公布。

聯賽將於明年1月4日正式開始，屆時會有48支北美隊伍參與到這個聯賽中，賽事的總獎金池達到了8000美元。

是的你沒有數錯零，48支參賽隊伍將為了瓜分這區區8000美刀而展開爭奪，摺合軟妹幣還不到6萬元。說實話，這種獎金級別的賽事充其量算是有收益的大型訓練賽，根本算不上是職業聯賽。

不過看到這微薄的賽事獎金，大家就不難理解為什麼北美豪門陸續解散了。

據稱一支北美絕地求生俱樂部每個月的住宿、選手薪資等基礎開銷成本在1萬美刀左右，按照最低標準來算，一年也得花費至少12萬美刀才能維持俱樂部基本運行。

聯賽可憐的獎金比起這筆支出根本不值一提，確實就跟此前Ghost戰隊的教練說的一樣，他們完全是在用愛發電。

不止NPL聯賽，其實PEL、PCL、PKL三大賽區的情況也差不太多，小俱樂部苦苦支撐，大俱樂部也只能靠贊助存活。

如果藍洞真的想讓把絕地求生的品牌賽事做大做強，他們也許應該學學隔壁拳頭和暴雪，在賽事建立前期主動出錢扶持聯賽成長，擴大聯賽影響力並吸引投資商，給參賽隊伍更多福利以鼓勵更多隊伍參與到聯賽中，大家覺得呢？

龍神絕曾說4AM在2020年的目標只是「活著」

如果藍洞始終對NPL賽區不管不問，不知道他們還能堅持多久呢？
