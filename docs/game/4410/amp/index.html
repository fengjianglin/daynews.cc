CFS丨最強教練團：AK教父Best&傳奇領袖Sir，強強聯合

明天，就在明天！CFS世界賽就要開幕了！在這個十萬火急，所有隊伍都在抓緊時間進行高強度訓練的時刻，我們的教練們也沒有閑著。這次他們破天荒的放下對手執念，精誠協作，坐下來同心協力的商討對付外國隊的策略。這對於即將出征賽場的中國隊還有廣大CFer們來說無疑是一個巨大的支持和莫大的心理安慰。在這個號稱「地表最強教練團」的陣容中，有兩位大佬最受CFer們的關注，他們就是來自王者之師SV戰隊的教練Best，和來自銀河戰艦AG戰隊的Sir！

【Best：穿越火線AK47潑射第一人，鼎力入駐最強教練團】

提到Best,關於他的傳說可實在是太多了，一時間竟不知從何說起。這樣說吧，但凡在CF這款遊戲中有些資歷的老玩家，應該都聽說過日照陸虎隊或者IG戰隊吧？沒錯，那就是Best曾效力過的隊伍，也是他職業生涯中最為輝煌的時期。當時作為一名職業選手的Best，在中國電子競技剛剛起步的那個階段，勤學苦練，憑藉日復一日的研磨和悉心鑽研，從AK-47這把槍械中得到天才領悟，自創了一套AK47潑射打法，從此開啟華麗麗開掛人生，幫助隊伍拿到多項賽事的冠軍和重量級榮譽。「AK教父」之名由此而來。結合當下的形勢，中國隊伍為國而戰，出征CFS世界賽，正是需要國內力量支持的時候。Best作為最強戰隊SV戰隊的教練，責無旁貸的需要肩負起訓練指導，為中國隊出謀劃策的重任來。從職業選手轉型教練的他對於比賽的理解和戰術的掌握都非常嫻熟完善，遊刃有餘，有他的支持，中國隊無論是在能力上還是信心上都得到極大的保證。

【Sir:CF傳奇領軍人物，為中國隊保駕護航】

Sir，本名李楊，在很多人的印象中，或許Sir都是以教練的身份出現在大眾視線里，卻很少有人知道他曾經也作為一名職業選手，戰鬥在CF的舞台上。為什麼說他是「傳奇領袖」呢？這麼說吧，70KG、Even、N1CE、林肯這些人曾經都是他麾下的隊員，Sir的地位可想而知。2011年他率領東珈AB斬獲了中國代表團在WCG世界總決賽歷史上的首枚金牌！而且，Sir還是第二期榮獲穿越火線終身榮譽會員的人物，與70KG是同一屆！關於Sir的介紹，我們就言盡於此，有興趣扒更多歷史的CFer，可以自行查閱。作為CF圈內人氣最鼎盛的AG戰隊的首席教練，Sir的身上不但肩負著把AG發揚光大的神聖使命，如今還順時勢，顧大局，主動請纓參與教練團為中國隊獻上錦囊妙計。有他們這些「大佬」在，中國隊何懼任何外來隊伍的挑戰？

人心齊泰山移，眾人划槳開大船。前有SV、AG、BS三大兄弟團努力訓練，認真備戰，後有地表最強教練團鼎力相助，指導有方，本屆CFS世界賽中國隊伍一定能夠揚鞭策馬，橫刀立威，創造傲人的成績，為國人爭光，為CFer提氣，我們就一起好好期待吧！
