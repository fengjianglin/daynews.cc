鬥魚黃金大獎賽戰況升級，前5名相差僅為1分，SMG依然有奪冠希望

2019-12-24 140條評論 1,607次閱讀 268人點贊

鬥魚黃金大獎賽已經進行到了決賽階段，在決賽第2日戰況再次升級，6場比賽結束以後積分排名前五的戰隊彼此之間僅相差一分，D7G在決賽第2日收穫2雞以總積分92分暫列第1，BA繼續保持了第2的名次，排名3至5名的分別是Su4、Ark以及TSG。

戰況的焦灼也讓鬥魚黃金大獎賽最後一天的比賽變得更加具有看點，其實不光是排名前5的戰隊有機會拿到冠軍，甚至排名第8的SMG也有奪冠的可能。目前SMG拿到了72分僅與D7G相差20分，20分也就吃一把10殺吃雞能夠解決的事情，況且從SMG前2日的發揮來看，如今的SMG已經不再拉胯，有了大京的穩定指揮再加上本就不弱的剛槍實力，SMG很有可能在老東家舉辦的比賽中開始復甦。

對於SMG在鬥魚黃金大獎賽的表現，玩家們也是十分的欣慰，此前SMG在國內的PCL春季賽以及PCM大師賽中發揮不佳，再加上17shou在場外的種種表現，使得SMG飽受非議。這次鬥魚黃金大獎賽17shou因為受到傷病的困擾無法上場轉而大京進行輪換，這一套全新的陣容也算是起了化學反應，雖然有一些名場面的發生，但終歸來說成績還是喜人。

作為SMG的「冤家」4AM在本次黃金大獎賽卻開始拉胯，雖然說小組賽階段4AM獲得了最高擊殺，但是來到決賽階段4AM卻開始反向吃雞，12場比賽下來4AM排名第14位僅收穫43分。回顧往屆黃金大獎賽4AM的成績向來不錯，不知道在剩下的6場比賽中4AM能否創造奇蹟呢？不過已經獲得德國柏林PGS門票的4AM早已走在了其他戰隊的前列，玩家們也無需苛責他們在黃金大獎賽的最終成績如何了。

最後你們覺得這次鬥魚黃金大獎賽的冠軍會是哪支戰隊呢？
