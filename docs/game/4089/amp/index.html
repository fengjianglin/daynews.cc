Dota2版本更新機制大改，更新只增加遊戲可玩度，版本卻充滿矛盾

天天要聞
2019-12-10

摘要

在7.23c版本更新之後很多人都感慨，這還是dota2么完全就是dota4，因為更新之後的dota2和過去有了相當巨大的改變。從遊戲的機制再到遊戲的節奏讓老玩家都需要一段時間去適應，那麼在新版本中有哪些值得大家去注意的重點呢？

在7.23c版本更新之後很多人都感慨，這還是dota2么完全就是dota4，因為更新之後的dota2和過去有了相當巨大的改變。從遊戲的機制再到遊戲的節奏讓老玩家都需要一段時間去適應，那麼在新版本中有哪些值得大家去注意的重點呢？

首先從版本的節奏上面來看似乎是加快了的，首先前哨的經驗已經從過去的每分鐘25變到現在的18，這也是變相的削弱了前哨在前期的作用，可能很多人會覺得這樣的更改其實是將遊戲的節奏放緩了，但是實際上你會發現你在遊戲中對於掉落的裝備以及前哨不再那麼的糾結了。其次對於這個版本是該刷還是該打架，目前都沒有一個準確的定論，畢竟在這款遊戲中關鍵還是看選手的臨場發揮，那麼大家可以看看冰蛙現在是有30級了，這樣看來一些大後期的英雄肯定都會選擇拖39級的天賦，這樣可能會讓你覺得節奏變慢，然而實際上遊戲中升級需要的經驗實在是太多了，不靠打架是很難到30級的。

此外在7.23版本裡面冰蛙提供的回魔以及攻速已經挺多的了，並且目前對於一些英雄的初始屬性進行了加強，因此五分鐘左右雙方互相博弈的情況下不會像過去那樣不痛不癢了，這時候上面提到的前哨改動就突出其作用了，因為經驗減少不得已開始抱團進行推進，在這個推進中必然就要打架，前期打架會更加的頻繁，並且單一的刷錢並不符合V社的初衷，對抗才是本能。

此外在新版本裡面防禦塔也再次進行了修改，不僅對小兵的攻擊降低到只有4s同時還能夠對玩家操控的單位有格擋機制。其次中立生物擊殺後的物品掉落也有特殊的設定，如果600碼的範圍內有人的情況下是不會掉落的，避免了被別人搶走的情況。這裡可以和大家聊一下更改後的掉落機制問題，大家對於dota2這款遊戲更多的印象是一款競技遊戲，然而V社現在更加偏向於趣味性。

要知道在所有的moba遊戲中偶然性幾乎都是選擇避開的，這也是為了遊戲的公平和公正，然而這一次的更新之後遊戲中物品的掉落充滿了各種的隨機性，並且一些裝備對於特定的英雄影響重大，這就導致很多老玩家有了怨言，勝負很有可能就是因為掉落的裝備決定勝負，相信官方對於這個問題也是經過了各種平衡的，但是對於小編來說隨機掉落雖然有趣，但是公平性卻大不如從前了，近期官方改動了200多件的物品足夠的良心，但是沒有太大的必要。

只能說從上面的改動來看，官方的確是為了玩家擁有更多的可玩性撒費苦心，可玩性增加的同時版本內部似乎也出現了一些矛盾，老玩家的不適應和新玩家的難以融入都是一個很大的問題，不知道大家在嘗試了7.23版本之後都有怎樣的感受呢？歡迎在文章下方留言評論喔。
