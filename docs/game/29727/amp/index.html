粉絲髮現4AM死亡區域，他們從未在這拿到高分，圈刷這裡必白給

天天要聞
2019-12-24

相信許多看了鬥魚黃金賽決賽第一天比賽的4AM粉絲都會有些難受，隊伍在第一天的表現只能用不如人意來形容。尤其是他們在最後最後一場比賽中在小E城加油站不理智地跟D7G戰隊開團，最後被殘忍1換4墊底出局，這令粉絲們相當失望。且不說這一場團戰完全沒必要打，他們近距離交戰的細節也做得並不算好。

賽後許多粉絲在論壇里就戰隊的拉閘展開了爭論，但這時卻有一名細心的粉絲悄悄整理了一下4AM在沙漠圖的歷史戰力，然後他就意外發現其實在沙漠圖中4AM有一塊白給區域，那就是小E城以及南部的大片山坡，尤其是小E東南角的那個高坡。只要安全區刷新到這裡，4AM從未拿到過高分，白給率接近100%。

PGC小組賽的一場比賽中，4AM也轉移到了小E城的加油站。雖然在那一場比賽中他們變現得相當高光，先是打掉了TSM，然後又在進圈途中開車撞死了Team Liquid的一名選手，但不可否認的一點是他們在那一場比賽中獲得的積分依然不高。要知道他們是跳小E城和水廠中間野區的隊伍，按照順位來說，他們完全有機會在更好的位置拿到更好的成績。

也許有很多4AM粉絲會不服，因為今年絕地求生最高光的鏡頭，醒目一雷四響獨狼吃雞的位置就是在這一片區域內。但大家也別忘了，醒目高光的背後是4AM自己走入了不利位置遭到了四周打靶，這才只剩下了他一個人。

不過愛吃雞的小女孩槐序覺得所謂的死亡區域並沒有什麼玄學成分，只是單純4AM對這片區域的地形還不足夠熟練，因此沒有布置出最合理的戰術。所以4AM的粉絲也不要太著急，此前4AM剛把海島圖的跳點換到學校時同樣出現過這樣的情況，慢慢把戰術調整過來就好了。

而且粉絲們能夠看清的東西，逍遙老師肯定也能看出來。之前復盤時他曾說這次黃金賽讓他看到了此前沒看到的問題，相信4AM很快就能克服這一戰術弱點，把死亡區域變成4AM的得分區域。

不過4AM在黃金賽決賽第一天打得確實十分辣眼睛，各位粉絲也不要否認，希望他們能夠趕緊找回狀態吧。
