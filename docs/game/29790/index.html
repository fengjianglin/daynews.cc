魔獸世界：洛丹倫往事，提米的天災噩夢

2019-12-24 478條評論 2,578次閱讀 1626人點贊

洛丹倫的廢墟了無生氣，斷壁殘垣講述著風霜下的故事，沒有”人”願意回憶那些支離破碎的片段。這裡的歷史未曾磨滅，跟隨經歷者的視角，打開洛丹倫往事，回顧提米的天災噩夢。

“嗨，你好啊！探險者。歡迎來到斯坦索姆，我的名字叫做提米“。

我和母親居住在這片土地，提瑞斯法林地是我和小夥伴最嚮往的地方，雖然聽說那裡有很多怪物，不過我相信泰瑞納斯·米奈希爾二世一定會好好的懲罰他們！感謝國王為我們築建了如此富饒的城邦。

對了，還有阿爾薩斯王子，他可是個大英雄！在我不小心闖入森林被那群卑劣的豺狼人囚禁時，是王子不顧生命危險把我送回家。等我長大了，也要成為像他一樣的聖騎士，用聖光制裁那些齜牙咧嘴的怪物。

最近我做一個噩夢，在夢裡大伙兒也都變成了怪物，那真是太恐怖了！

烈火焚燒著我們的家，濃煙快讓我睜不開眼睛，不過我依舊不斷地尋找著媽媽的蹤跡，心裡默念著”媽媽你不要害怕，提米馬上就來救你了”。你想聽這個故事嗎？也許我能看透你的心思，那我就跟你聊聊那夢裡的世界吧。

天災將至，危機四伏

獸人入侵的遺患依舊籠罩著洛丹倫帝國，泰瑞納斯疲於應對這些殘餘的不穩定因素，繼而導致潛伏的瘟疫肆意擴散。一群自稱詛咒教派的神秘組織，鼓吹著”真神”耐奧祖的力量，潛移默化地形成了”洛丹倫的腐朽統治是百姓的鐐銬，我們永不為奴，耐奧祖會賜予無上力量”這般的抵觸的想法。

瘟疫的傳播速度超乎任何王族的想像，無數口瘟疫大鍋如果繁星散落洛丹倫周遭的村落，腐爛的氣息源源不斷的鋪散，扼住每一個人的咽喉，即使是聖光的力量也無法挽救深陷痛苦之中的平民。然而，洛丹倫歷史上數次瘟疫爆發的記憶，讓泰瑞納斯僅將眼前的情況視為一次小規模疫情，卻並未意識到一場傾朝的災難已經拉滿弓弦。

死神終於到來，穿梭於城鎮之間卻杳無蹤跡。不斷有瘟疫感染者喪生，死亡人數迅速上漲，百姓含淚將已故的親友埋葬，帶著最好的祝福，希望逝者在另一個世界能開啟新的旅途。然而，靈異事件卻接踵而至，不少村民目睹那些已經埋葬的死者自己從土中蹣跚地爬出，踉蹌地徘徊於墓地當中。

危機爆發之際，詛咒教派依舊不忘煽風點火，”只有耐奧祖才是真正的救世主”。作為這場洛丹倫瘟疫的執行官，墮落肯瑞托法師克爾蘇加德愉悅地欣賞著眼前這般景象，哀鴻遍野，生機無存，連植物都在災難中凋零，落葉滿地、莊稼枯萎，肅殺之風摧枯拉朽如同奏響生命的輓歌。

“旅行者，故事是不是挺有趣的？那些怪物真是太恐怖了，居然能自己從土裡爬出來。不過這還沒完，接下來的夢讓提米更加害怕了。“

啊！安多哈爾

行屍走肉迅速覆蓋了曾經祥和的東威爾德，毫無意識地踟躇於廢土之上，盲目地向身旁的生物發起進攻，哪怕是曾經的親人也無一例外。洛丹倫統治者終於意識到問題的嚴峻，或許並未為時已晚，王子阿爾薩斯與肯瑞托法師吉安娜一同前往瘟疫肆虐的區域，探究這場災禍的根源。

縱橫的荊棘脈絡割裂大地，瀰漫在空氣中的腐敗氣息不禁讓人作嘔。看著眼前的瘡痍之景，阿爾薩斯深感自責，百姓身陷水深火熱之中而自己卻無能為力，即使是神聖魔法也難以遏制瘟疫擴散的步伐。也許找到這場災難的罪魁禍首將其連根剷除，才能還洛丹倫一份安寧。

天災的侵蝕不僅在人類之間迅速傳播，生活在附近的動物也紛紛遭到腐化，變得狂躁不安。考林路口已是人逝樓空，部分詛咒教派分子號令無腦的亡靈盤踞於此，築起天災部隊的營地。遊盪女妖的歌聲縈繞著廢墟上空，不禁讓聞者為之涕零。

通過對東威爾德的調查，讓阿爾薩斯察覺到所有感染村落的共同點，百姓所食用的糧食都源自安多哈爾，這個洛丹倫的樞紐區域。而現在安多哈爾已經淪陷，並由詛咒教派所掌握，其中一位名叫克爾蘇加德的墮落法師，或許跟這次災難存在莫大關係。

故此，阿爾薩斯與吉安娜迅速趕往安多哈爾，迫切尋找事件的真相，但接連不斷的陷阱卻等待著這隻待宰羊羔慢慢走入圈套。

面對來勢洶洶的阿爾薩斯，克爾蘇加德內心卻波瀾不驚，他深知自己的宿命，眼前的死亡對於自己腳下這條天災之路來說，不過是一場小小的獻祭，而更大的力量將屬於重生後的自己。瘟疫糧食輸送的任務早已按部就班，大規模的疫情爆發僅是時間問題，相信年輕的王子必定按捺不住內心的怒火，未來的發展盡在主人的掌握之中。

雖然克爾蘇加德殞命於安多哈爾，但這位墮落法師已在生前完成了最重要的任務，大量從此地運往斯坦索姆的糧食已被城中百姓食用殆盡，喪鐘將響徹城際。隨後，在恐懼魔王瑪爾加尼斯的挑撥下，阿爾薩斯愈加難以控制自己的意志，他心懷臣民卻無能為力，矛盾心態潛移默化地促使他攥緊武器，朝著斯坦索姆城中走去。

葯影子雜說

“阿爾薩斯王子真是一個勇敢的人！居然毫不懼怕那些怪物。我想，提米以後也會成為英雄，你說是吧？”

“你為什麼拿著武器指著我？想和我開個玩笑嗎？親愛的法師”

“啊！”

安息吧，可愛的提米，這個噩夢是時候畫上一個句號了。在新的夢裡，希望你能成為跟王子一樣的勇士！

作為《魔獸爭霸》戰役內，唯一一隻不會主動攻擊的食屍鬼，提米的故事並未用濃墨重彩渲染。而天災入侵的事件中，除了人人傳頌的英雄史詩，這些小角色的故事何嘗不是刻骨銘心？

終於決定寫一寫天災的故事啦！想了很久，打算就用”洛丹倫往事”作為背景吧。在這片土地上，既有不歇的輓歌，亦有不可磨滅的記憶。
