又可以看比賽啦！德杯戰鬥力指南

2019-12-20 542條評論 2,979次閱讀 421人點贊

21-22日，德杯冬季賽即將開賽，各大俱樂部也是陸續公布了自己的參賽陣容。這裡，我們對本次德杯20支參賽隊伍的紙面實力進行排序。特別說明，此次排名只針對德杯公布陣容，並不涉及隊伍明年LPL實力。

S級陣容：自己主力登場，隊伍陣容沒有大的變動

主力登場決定了隊伍的實際戰鬥能力，陣容的穩定性則意味著磨合期的減少。

TES：

TES本次不僅換掉了之前被大家詬病的下路組，更是拿到了前期穩定性更好的Karsa，大加強。

JDG：

與TES一樣，JDG以極好的陣容穩定性獲得了先發優勢。而且Kanavi訓了這麼久，應該已經可以使用了。轉會期結束後，JDG找回了曾經的隊員Loken，且Homme也沒有離開。陣容穩定，加強。

LNG:原班人馬加強出爐，被質疑的教練組進行了更換，大加強！

OMG：

來了嗨少，前期節奏加強。詛咒哥價值在，司馬老賊和小五組合回歸。只要爾康不犯罪，還是比較有競爭力的。

A級：陣容穩定，但丟失了核心

陣容看上去和之前差不多，但是少了靈魂人物或有不確定因素。同時，如果來的選手都是大家認可的角色，那麼也應當被視為是具備一定競爭力的。

BLG：把韓虎換了FoFo，上路加了一個上單。說實話，BLG之前的核心在我看來，其實就是韓虎。韓虎的穩健才給到了三條路發育的機會。國豪本來就是個肉食打野，Jinjiao也是個團戰期發力的AD，要不是之前Kuro比較能”油”，所以BLG拖著這麼多人才發育的起來。不知道FoFo這種快節奏打手會打出怎樣的感覺，只能看具體的表現了。

RNG：沒有小狗，丟了Karsa，但Ming+Xiaohu+狼行三人組在，可能會遭遇節奏問題問題。

EDG：雖然給打野位置找來了穩健的Xinyi，但魚男離開，馬Ray離開，希望皇之前的表現都不太好，所以可能有些難。

DMO：陣容保持完整，同時補強了Xiye，但人員競爭力不足，肉搏競爭力可能略顯不足

VG：VG屬於我們後面提到的典型，在召回了Zdz後，我絲毫不擔心這個陣容的強度了。上路Zdz，打野酋長，中單蕉太狼，下路iBoy+小胡，教練還是一個Kkoma，這個陣容的野心著實不小。但由於Kkoma並不會馬上登場，所以我們姑且認為，VG還是有存在磨合問題導致實力無法發揮的可能的。

B級：有那麼一丟丟強，但也有肉眼可見的短板

RW：在轉會期引入了大量選手，核心陣容雖然穩定，但是中路位置替換，輔助位置更換，待觀察。

SN：引入了SoFm進行補強，但丟失了Smlz，選擇讓之前的WeiWei轉型AD，這裡感覺有些懸。

LGD:在院長Kramer基礎上，他們又帶來了肉食性打野小花生。但比較遺憾的，小花生職業生涯處於一個低谷期，我們很難判斷出他在LGD里具體的表現。

FPX：純上了二隊，不予評價

IG：一個基本二隊的陣容：

V5：東北搖擺體系沒變，引入大量新人，效果未知

未知領域：這裡屬於素材太少，完全無法了解具體強隊類型

主要是幾個純新人隊伍和LDL戰隊

可以看得出，那些陣容保持相對穩定，陣容有競爭力的陣容，我們還是持有相對樂觀態度的。本次德杯你最看好哪支隊伍呢？不妨告訴我們吧！
