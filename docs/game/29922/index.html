DNF：韓服劍魔獲得無我式加強，開放三次覺醒能否成幻神？

2019-12-23 875條評論 1,879次閱讀 695人點贊

我們在前前期分別介紹了劍宗和劍帝的三次覺醒和平衡改版內容，本期我們將對劍魔在韓服100級版本的平衡和三覺內容進行一個分析。

喚魔：蛇腹劍調整

蛇腹劍轉職後新增技能攻擊力增加的屬性，一級精通增加12%的技能攻擊力，11級上限，和大多數職業的加強設定一樣，相當於獲得12%的提升率，其他需要通關寵物光環和裝備進行提升，每增加1級提升率為1.5%，

魔劍掌控調整

和劍宗的提升率有點像，但不是武器精通而是魔劍掌控，20級精通時增加30%的技能攻擊力，精通後由原來1.5%的提升率調整為2.5%，但是同樣的被動技能不吃寶珠，想要額外提升率就得看光環和寵物，以及裝備來提升，也就是說皮甲玩家獲得更多的加成。

一覺被動貪慾之燔祭調整

一覺被動調整和其他的職業加成一樣，調整為1級增加4.5%的技能攻擊力，每級提升由1.5%調整為2%，相當於16級後每級額外獲得0.5%的提升率。

一覺主動技能調整

一覺技能進行了調整，一覺在調整前有多次突刺攻擊的部分，調整後直接刪除了這一個部分的攻擊，所以調整了整體技能的攻擊力，也就是把刪除的部分傷害調整到後續的傷害種，整體演出速度快了很多，因為刪除突刺攻擊，原3級一覺增加突刺傷害和速度的屬性直接調整為增加10%一覺主動技能攻擊力。

這個改動很多玩家可能會覺得不太習慣。

95級三覺被動技能魔人女王

新增95級被動技能魔刃女王，和劍帝一樣是增加基礎攻擊力和技能攻擊力，1級增加20%，每增加1級提升2%。100級加契約可以學習4級獲得26%的技能攻擊力增加。

此外學習被動技能後還對2個技能進行了一些優化，蛇腹劍破地刺數量增加6個，蛇腹劍纏追擊一個下劈攻擊。

95級主動技能

血魔劍-閃是三覺後的95級主動技能，和塔莫斯融合插入地面生成大型蛇腹劍刃攻擊前方的敵人，因為技能的演出效果比較特別，以為是全屏攻擊，實際上只有對前面的敵人攻擊，背後的敵人範圍非常小，如果站在地圖中間，背後靠近屏幕邊緣的目標是打不到的。

施放的時候需要注意距離和方向，避免打空，技能分為4次劍刃多段攻擊傷害和終結攻擊傷害。

三覺主動技能

先不說別的，單看這一個劍魔的三覺插畫就已經打算把劍魔養起來了。三覺主動技能弒神劍 : 魔境，劍魔裝備魔人戰甲後，將塔莫斯變為巨型的蛇腹劍，在劍域內創造一個魔境，並對魔境內所有敵人進行攻擊。

僅能分為15次多段攻擊和終結斬擊，這是女鬼劍三覺技能中個人最喜歡的一個。

不過大家仔細看官方首次公布的立繪圖片已經是和諧過的，所以三覺技能實裝正式服有很大的可能性會和諧。
