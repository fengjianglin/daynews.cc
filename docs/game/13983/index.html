[實況談兵]沒有馬特烏斯的，精選阿蘭完全可以替代

2019-12-12 694條評論 1,684次閱讀 775人點贊

實況足球手游最新版本昨天晚上已經可以登錄。精選球員如約而至，我想大家都已經清楚精選球員的數據情況，估計也有很多玩家像我一樣滿懷希望的去用5個黑球兌換庫里巴利時的尷尬。說實話用35個黑球去兌換庫里巴利我是真捨不得，不知道有多少玩家使用35個黑球兌換了庫里巴利？

對於我來說吸引我的並不是庫里巴利而是阿蘭。兌換阿蘭「僅僅」需要18個黑球，沒錯這裡用僅僅來表達～，當看到庫里巴利需要用35個黑球去兌換的時候，能用18個黑球能夠兌換一個精選我反而不那麼感覺很貴了。本次精選球員庫利巴利和阿蘭都是非常優秀的精選球員，而阿蘭我認為值得擁有。理由就是：阿蘭的模板就是傳奇球員馬特烏斯，二者比起來真的是不相上下。如果你苦於獲得馬特烏斯而不得，完全可以拿阿蘭來體驗一下傳奇球星的風采。

讓我們來看一下阿蘭有多麼的優秀，我們用圖片來說明。

我們可以看到阿蘭的盤控領先於馬特烏斯，地面傳球比馬特烏斯高3點，空中傳球比馬特烏斯少了6點。雙數中的速度比馬特烏斯少2點差別不大。阿蘭的防守數據要全面優於馬特烏斯。算是一個防守加強版的馬特烏斯，可以說完全不輸於馬特烏斯，對於喜歡馬特烏斯這種攻防一體球員的可以考慮奢侈一把，忍痛自己18個黑球兌換一個。

查看了國際服出過的精選球員，阿蘭這種攻防一體的類型球員中，阿蘭算是佼佼者。當然不排除以後會有更出色的精選球員。

阿蘭值得擁有，大家可以根據自身實際情況選擇是否拿下！
