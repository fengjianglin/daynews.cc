JAMPPI受開掛風波影響難發揮，SJ和Nordavind恐將大分收場

2019-12-16 200條評論 2,508次閱讀 620人點贊

比賽： SJ VS NOR

買點：地圖總數大小大於2.5

盤口：1.99

盤口截圖：

解析：Nordavind這邊五位選手，HALLZERK，RUBINO，TENZKI，CROMEN以及H4RR3。其中狙擊手HALLZERK是絕對的大哥，個人實力超強，狙用的也非常好。TENZKI則是二哥，槍法個人能力都還可以。而CROMEN，原來在FAZE打的不錯的選手，以及RUBINO還有新人H4RR3，三個選手有點拖後腿，趕不上2個大哥的步伐，所以註定這隻隊伍不會強大。SJ目前是芬蘭三哥，隊員有SADDYX,ARVID,JAMPPI,KHRN以及XARTE。這個隊伍幾乎就是一神帶四腿的隊伍，只有狙擊手JAMPPI一個大腿，其他4個人實力都平均，並不突出，唯一突出的選手就是XARTE的鳥狙，他的鳥狙令人印象深刻。並且JAMPPI的鳥狙也非常厲害，他們2人一起很容易打出ECO翻盤的局面。但是他們最近的戰績並不好看，連續輸給了HAVU,ATN,WK以及UOL，狀態看起來非常差，估計是JAMPPI受到了開掛風波無法加入OG的影響，實力大打折扣。看看近況，兩隊的近況都十分差勁，NOR這邊被野雞隊SKADE直接2-0帶走，SJ這邊最近的狀態也非常差，誰贏不好說，但是看好菜雞互啄打滿三圖。
