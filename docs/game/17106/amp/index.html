森林續作森林之子，開頭飛機又失事，玩家調侃找完兒子找孫子？

天天要聞
2019-12-13

目前，2019年TGA年度遊戲提名獎已經全部公布。在頒獎期間，被譽為Steam最受歡迎的生存遊戲之一的森林，公布了新續作《森林之子》的宣傳片。

從視頻的表現來看，本作的畫質有了很大的提升，恐怖元素也有所增加，劇情似乎跟前作的結局有所關聯，具體信息未知。有玩家對本作的名稱調侃道，森林，森林之子，森林之孫，森林之曾孫，子子孫孫無窮盡也。

森林之子

對於5年前發行的原作，遊戲日報君相信玩過的人不在少數。遊戲開始後，從劇情得知，這對倒霉的父子遇到了空難，飛機墜毀於某神秘島嶼。

由玩家扮演的父親，親眼看到自己的兒子「提米」被神秘生物帶走了。蘇醒之後，獲得的第一把武器就是斧子。接下來遊戲就沒有任何的任務引導或者地點說明了，全靠玩家自己摸索，這導致很大一部分的玩家把遊戲玩成了荒野求生。

當然還是有很多人注意到兒子不見了這個問題，站在一位父親的角度，那首要任務就是找兒子。在這個過程中，大家自然會遇到各種野人和各種變異體，獲得更多的工具和物資，說起來剛玩的時候還是蠻害怕的，尤其是在洞穴里。

通過一二周目的同學一定知道，其中一個結局是玩家找到了兒子，但是兒子已經死在復活裝置里了；打敗BOSS，也還是復活不了。

最後只能選擇擊落一架飛機，用飛機上孩子的生命換回自己兒子的生命。隨後鏡頭一轉，到了你和兒子回到文明社會在接受採訪，但不幸的是，在這一過程中，兒子突然倒地開始抽搐，不省人事。這一幕跟小女孩變怪物很像，前作到這裡就結束了。

目前從《森林之子》的預告片里猜測得知，玩家在自己的胳膊上紋上了「Fight Demons（打敗惡魔）」的字樣，乘上直升機重返島嶼，這次飛機照常也受到了襲擊。

在往後的畫面中，隨行的特種兵被擊殺，前作中的怪物蜘蛛女，也變好看了不少；還有洞穴中的無臉野人也還在。由此我們可以猜測續作的劇情是在講述父親重返島嶼，尋找兒子變異背後秘密的故事。

目前遊戲的發售時間尚未公布，不知道小夥伴們覺得這次的故事會有什麼轉折呢？
