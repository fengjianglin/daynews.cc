DNF：這才是武器大師！Eiji劍魂裝備畢業，還升級了3把普雷武器

2019-12-04 0條評論 19次閱讀 0人點贊

摘要

之前傳出了Eiji要“脫坑”的消息，玩家一時間比較詫異，畢竟作為知名技術達人，都沒能堅持下去，也太過可惜。不過，Eiji經過澄清後表示，並非真的“脫坑”，而是主播時“口嗨”。Eiji對劍魂，研究非常透徹，其角色不僅畢業，還升級了3把普雷武器，這才是真正的武器大師！

之前傳出了Eiji要「脫坑」的消息，玩家一時間比較詫異，畢竟作為知名技術達人，都沒能堅持下去，也太過可惜。不過，Eiji經過澄清後表示，並非真的「脫坑」，而是主播時「口嗨」。Eiji對劍魂，研究非常透徹，其角色不僅畢業，還升級了3把普雷武器，這才是真正的武器大師！

Eiji的劍魂，雖說在裝備上，沒有一些主播打造的強。但是，也全身增幅12，算是土豪級別的存在。早先的時候，Eiji劍魂走的是重甲，不過，經過特色加強後，皮甲變得吃香，就打造了一身紅12。Eiji的劍魂，普雷7件完美畢業，傷害也達到極致，20秒綠沙袋，在修鍊場可以突破1000E。

值得一提的是，Eiji勵志成為「武器大師」，普雷裝備畢業後，除了升級光劍外，還升級了鈍器和巨劍。也就是說，Eiji的劍魂，目前手上有3把普雷武器，只差太刀和短劍！你不得不服氣，Eiji劍魂歐皇附體，別的玩家，普雷7件都沒有畢業，而他不僅畢業，還擁有3把夜語黑瞳武器！

當然，Eiji升級鈍器和巨劍，純屬為了打極限傷害，在20秒綠沙袋，可以進行切換，從而輸出最大化。不過，這技術要高超，一般玩家還是省省，升級一把光劍就行了！對於Eiji來說，劍魂還有太刀和短劍，正在慢慢打普雷團本，積攢黑瞳石材料升級！

紅12裝備打造的劍魂，修鍊場綠沙袋突破1000E，這個職業，還未達到幻神的境界！我們知道，目前幻神的定義，是紅11的打造，突破900E，劍魂還無法做到。像劍帝、召喚和武極等等，則是能夠做到。劍魂只能說，在超一線位置，幻神還是差了一點。要是當初武器奧義不削弱，那麼，劍魂妥妥的是幻神，這都毋庸置疑！

但就劍魂來說，光劍還是最適合的武器，傷害不一定是最高，卻流暢度很舒服。其他的武器，你需要比較高的技術，兩者完美的切換，才能得心應手。當然，這一點難不倒Eiji，畢竟他是知名劍魂技術達人！
