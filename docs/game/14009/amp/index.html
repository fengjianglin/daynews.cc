DOTA2：說好的世界強隊，怎麼開始紛紛掉鏈子，比賽都不想打了？

天天要聞
2019-12-11

DOTA2新賽季的比賽可太奇怪了，世界排名前列的幾支戰隊開始高掛免戰牌，首個Major賽事TI排名前四的隊伍居然連一支都沒有來。

這在以往是不可想像的，畢竟在DOTA2職業圈是沒有聯賽的，換句話說戰隊的收入基本上全靠杯賽獎金。

對於強隊來說，以他們的實力拿個名次豈不是很容易，為何強隊都選擇退賽不打呢？

首先最重要的一個原因就是怕針對，主要是害怕被冰蛙針對。

上個賽季Major霸主Secret戰隊在TI9之前橫掃一眾對手，那時候玩家都稱呼他們為大魔王。

結果這話傳到冰蛙耳中，他立馬就針對大魔王進行了一次版本更新，在TI9之前進行版本更新，其用心自然不用多說，而Secret戰隊在TI9舞台上也沒有達到玩家想像的高度。

相比起來TI8冠軍OG戰隊就顯得高明許多，TI9之前的Major比賽他們缺席了1~2個使得所有人對他們放鬆警惕。

而在TI9之前的Major賽事中，他們也保留了餘力，可以想像如果OG戰隊在賽前就將自己實力全部暴露，他們的TI9之旅肯定不會這麼如意。

所以TI9結束之後，強隊開始有意無意裝傻充愣，畢竟誰也不想辛苦一個賽季，到頭來被遊戲版本針對了。

如今LGD戰隊和Secret開始逐漸嶄露頭角，而OG戰隊還在休養生息，作為世界冠軍他們在新賽季連一場比賽都沒有打。

當然如此做法，也僅限於幾支頂尖強隊，畢竟其他戰隊萬一翻車了，可能連TI都進不了。

除了怕針對，另一個原因就是回報少，跟TI賽事獎金相比Major的比賽獎金可以算得上弟中弟了。

上賽季Major大魔王Secret戰隊在Major比賽里，拿下的所有獎金還不如漢堡貓首次參加TI國際邀請賽的獎金高。

在頂級戰隊眼中，他們犯不著費大力氣去奪取這麼一點獎金，當然賽事積分還是必須要拿的。

如今DOTA2職業圈，你不鴿幾個比賽的話，都不好意思說你是頂級戰隊。

不過這也只是在新賽季剛開打不久，後面還有很長一段時間可以調整，對於頂級戰隊來說，他們的目標也不是幾個Major冠軍，TI10的冠軍盾牌才是他們的目標。
