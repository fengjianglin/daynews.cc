英雄聯盟：Perkz為ADC排名，Uzi第一 JackeyLove第二

2019-12-15 196條評論 1,543次閱讀 480人點贊

Perkz在昨天的直播中應粉絲們的要求，為西方的ADC選手們進行了排名

在他的心目中西方ADC們排名如下：

1.Rekkles

2.Doublelift

3.Kobbe

4.Hans Sama

5.Upset

6.Patrik

7.Sneaky

他也為自己的排名進行了解釋，Rekkles選手排名第一是因為他可以使用各個版本不同的英雄，霞、卡莎、希維爾他都有著不錯的水平，而且Perkz認為Rekkles是西方ADC中最好的霞使用者。除此之外，Rekkles還可以玩蓋倫悠米的套路。

而Doublelift的技術和長處也非常明顯，他非常善於韋魯斯塔姆以及卡莎加里奧體系。而Perkz唯一覺得Doublelift排在Rekkles之下的原因是他的霞不如Rekkles。

Perkz還表揚了Kobbe從春季賽到夏季賽的進步，以及他認為Hans Sama是對線期表現最好的AD。

接下來Perkz還未自己交手過的ADC進行了排名：

1.Uzi

2.JackeyLove

3.Rekkles

4.Teddy
