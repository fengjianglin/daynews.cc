《荒野大鏢客2》：在自由狀態下探索世界，獨特設定帶來真實觸感

2019-12-04 0條評論 19次閱讀 0人點贊

摘要

遊戲是一種基於物質需求之上的，在一些特定時間、空間範圍內遵循某種特定規則，追求精神世界需求滿足的一種行為方式，從定義上可以看出，凡是遊戲都需要在一定的規則之下運行，條條框框成為了它的一部分。

遊戲是一種基於物質需求之上的，在一些特定時間、空間範圍內遵循某種特定規則，追求精神世界需求滿足的一種行為方式，從定義上可以看出，凡是遊戲都需要在一定的規則之下運行，條條框框成為了它的一部分。近些年， 「開放世界」成為了遊戲圈的一大熱門話題，無論是《刺客信條：奧德賽》還是《塞爾達傳說：曠野之息》、《GTA5》都是自由度極高的遊戲，在龐大的虛擬世界中，無論是細膩的遊戲場景還是與NPC之間的互動都顯得極為真實，彷彿給人一種置身現實的感覺。雖然說開放世界題材的遊戲仍在一定的框架之下運行，但是也給予了玩家足夠多的探索空間，作為該類型遊戲的經典之作，《荒野大鏢客2》絕對值得一試。

《荒野大鏢客2》是由Rockstar San Diego工作室製作的一款大型開放世界遊戲，它將目光聚焦於19世紀末的美國西部，豐富而又逼真的細節、精彩紛呈的可探索元素、跌但起伏的故事情節，以及脫離傳統遊戲框架的玩法成為了該作的閃光點。《荒野大鏢客2》完滿重現了獨具特色的「西部牛仔」文化，該作所帶來的沉浸感讓玩家彷彿在美國西部重生，遊戲體驗極好。Rockstar作為享譽全球的遊戲開發公司，它的代表作《俠盜獵車手》、《黑色洛城》、《荒野大鏢客》為玩家留下了深刻印象，從三部作品可以看出，Rockstar熱衷自由高的遊戲，它並不喜歡給玩家設置太多的條條框框。作為開放世界的集大成之作，IGN為《荒野大鏢客2》給出了10分的評價，可以說這是一個最真實、最開放的世界。

在《荒野大鏢客2》中Rockstar幾乎重現了真實的美國西部，從畫面、到音樂、交互都力求完美，正是憑藉精益求精的製作態度，該作才能取得如今的成績。《荒野大鏢客2》試圖脫離主角敘事，嘗試著向玩家呈現出一部「群像劇」，通過對不同角色的塑造，玩家將會遊戲劇情產生一個更加全面的理解與認識。遊戲中存在著眾多的支線任務與隨機事件，在體驗遊戲的時候，「不確定性」大大提升了遊戲的趣味性，帶給了玩家探索未知的樂趣，在遊戲中，玩家將跟隨亞瑟一行人在廣袤的美國腹地掙扎求生。對於這樣一款大作來說，《荒野大鏢客2》拿獎拿到手軟，在遊戲產業盛會The Game Awards上，它一舉拿下最佳敘事、最佳配音和原聲、最佳音效設計三個獎項，從它的身上，玩家可以深刻感受到開放世界的獨特魅力。

優秀的畫面與音樂設計，帶給玩家一場極佳的視聽盛宴

為了向玩家呈現出最真實的「西部世界」， 《荒野大鏢客2》在畫面設計部分下足了功夫。在遊戲之中，分布著數百種的動植物，對於細節的極致追求使得該作具備完善了生態系統，從鬱鬱蔥蔥的密林到荒涼的原野、熱鬧的城鎮都因為豐富多樣的動植物顯得更加有活力。在不同文化背景的城鎮之中，製作團隊還設計出了不同的建築物，力求重現當時的風土人情，就算在遊戲中什麼都不做，只是欣賞沿途的人與物就可以增加不少知識，可以說《荒野大鏢客2》充滿了文化氣息。在開放的世界中，無論是人物的神態還是隨風飄動的枝葉都極為傳神，遊戲中的花花草草彷彿擁有了生命顯得極為「靈動」，可以說優秀的畫面設計使得《荒野大鏢客2》擁有極佳的觀感。

在音樂設計方面，Woody Jackson親自上陣，作為一名優秀的音樂人，他曾參與《Red Dead Redemption》、《黑色洛城》、《GTA5》等遊戲的配樂工作，可以說經驗豐富。前前後後，曾參與《荒野大鏢客2》的音樂家達到了100多位，經過Jeff Silverman後期製作與編曲後，遊戲氛圍得到了極大提升。當玩家騎著駿馬遊走在西部狂野之上，極具風格的西部音樂進一步增加了玩家的沉浸感，隨著遊戲場景的不斷變化，總能響起適當的背景音樂，在遊戲的世界中，玩家可以盡情享受美妙的音樂。

高度自由的設定，推動玩家不斷探索未知世界

《荒野大鏢客2》主打的就是「開放世界」，遊戲之中並沒有出現硬性規則，玩家可以憑藉自己的意願在未知的世界中享受異域風景。一般來說，每一款遊戲都有自己的主線劇情，在《荒野大鏢客2》中，基本故事框架是圍繞亞瑟一行人展開，隨著幫派內部矛盾日益加深，亞瑟需要在自己的理想與效忠幫派之間進行抉擇，在跟隨亞瑟走主線劇情的時候，大量隨機事件成為了遊戲極為重要的一環。

「隨機事件」可以說是《荒野大鏢客2》的特色所在，同時也是非任務狀態下的重要系統。在遊戲中，充滿了各種各樣的角色，通過與他們的互動，玩家能夠享受到「真實世界」般的感覺。當玩家在地圖上隨機探索的時候，會遇見各種向主角求助的陌生人，有人可能被蛇咬了，有人可能遭遇了野獸的攻擊，當玩家選擇伸出援手的時候也會帶來一系列的連鎖反應，可以說隨機事件極大地提高了遊戲的自由度，在開放的西部世界，玩家可以盡情探索。

另外，《荒野大鏢客2》在「互動體驗」上面也進行了優化。在遊戲中，玩家可以主動與每一個角色打招呼，可以攔下馬車請求搭順風車，也可以試圖激怒一個看不慣的傢伙從而引發鬥毆。遊戲之中的角色都被賦予了「性格」，當玩家採取不同的行動時，他們也會給出不同的反應，這更加符合現實中的邏輯。置身於《荒野大鏢客2》中，玩家可以自由探索未知的世界，高度自由且符合邏輯的互動體驗極大地提高了玩家的遊戲體驗。

獨具特色的「榮譽系統」，讓玩家走向「除惡揚善」之路

在開放的世界中，玩家可以選擇做很多的事情。從蒙面劫匪到追捕逃犯，只要玩家願意就可以在「好人」與「壞人」之間進行轉換。對於玩家在遊戲中所做出的各種行為，《荒野大鏢客2》的「榮譽系統」會進行評判，正義的行為會增加角色的榮譽值，而邪惡的操作則會減少角色的榮譽值，榮譽值的高低則會影響主角亞瑟的走向，可以說遊戲的主動權完全掌握在玩家的手裡。

對於「正邪」之間的選擇，遊戲之中存在著一套評價體系，這使得虛擬社會也具備了社會運行的邏輯與道德評價標準，雖然並沒有強制要求玩家做一個好人，但玩家必須對自己的行為與潛在後果進行考慮。潛移默化之下，很多玩家還是願意成為「正義的夥伴」，畢竟劣跡斑斑只會增加遊戲的難度。

在開放世界中表現人性，唯有勇敢向前方能看到勝利的曙光

《荒野大鏢客2》對於每一個角色都進行了細緻刻畫，在這群人當中，並不存在十惡不赦的壞人，也沒有絕對意義上的好人。在動蕩的社會環境之下，他們都是基於自身的利益與環境不斷進行選擇，如果只用高高在上的道德標準去評價，那麼就顯得不合時宜，可以說「有血有肉」成為了不同角色的鮮明特點。在使用叢林法則的社會中，唯有勇敢地戰鬥下去，才能看到勝利的曙光，從《荒野大鏢客2》中玩家可以收穫很多。

結語

《荒野大鏢客2》為玩家提供了開放的遊戲世界，細緻的畫面與音樂設計帶給了玩家極強的沉浸感。高度自由的設定讓玩家跳出傳統遊戲的條條框框，在充滿未知的西部世界進行探索。其次，遊戲之中還存在一套「道德標準」，在體系之內，玩家需要對自己的行為負責。作為一款主打「開放世界」的遊戲，《荒野大鏢客2》在方方面面做到了極致，它成為行業內的一道標杆。
