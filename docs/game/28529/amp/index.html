魔獸世界大法師瓦格斯的備用法杖玩具有什麼古怪嗎？

天天要聞
2019-12-23

在魔獸世界8.2.5解密寵物傑娜弗爾的探索過程中，discord上的玩家們發現了在卡拉贊的塞巴斯蒂安身邊，使用大法師瓦格斯的備用法杖玩具召喚出大法師瓦格斯的影像後，再和塞巴斯蒂安對話偶爾可以改變對話內容。

新的對話內容和大法師瓦格斯的影像的對話似乎有一些關係，都提到了法蘭倫和虛空風暴。

大法師瓦格斯的備用法杖玩具是德拉諾之王資料片的產物，它是要塞任務尋找法蘭倫獎勵的神秘的閃光保險箱里開出來的。

上述發現很可能對解密沒用，不過這個玩具還有個小細節：對大法師瓦格斯的影像做出一些表情或者動作的話，瓦格斯會對其中的一些做出回應——先轉身面對玩家，再做出對應的動作。

聊天框里的表情有4種能讓瓦格斯回應。對著瓦格斯/招手或者/wave，瓦格斯也會對你招手。

/鞠躬或者/bow，瓦格斯也會對玩家鞠躬。

/跳舞或者/dance，瓦格斯會笑。

/指點或者/bow，瓦格斯會兩手一攤做個無奈的動作。

此外還有兩個表情能讓瓦格斯回應。/性感或者/sexy，瓦格斯會秀他的肌肉。

/耳光或者/slap，瓦格斯會左手叉腰，右手搖手指。

目前我沒發現其他表情能讓瓦格斯的影像回應。也許我知道的表情不多，大家可以試試看。
