宿舍開黑必備遊戲，四人一起打殭屍，時隔10年續作終於來了？

天天要聞
1 週 ago

DOTA2世外之爭、CSGO大行動和VR遊戲半條命愛莉克斯，誰也沒想到G胖在今年年末為玩家們準備了大量驚喜，其中最讓人驚訝的就是被稱為旗艦級VR遊戲的半條命新作。

而根據一位外國網友的爆料，V社為玩家們帶來的重磅消息似乎還遠不止於此，另一款宿舍開黑必備、四人一起打殭屍的遊戲在10年後或許也將迎來續作。

這位名為Tyler McVicker的外國網友也是油管主播，他曾準確爆料過A牌和CSGO新地圖的消息，其情報還是具有一定可信度的。

現在Tyler McVicker稱V社不僅將在TGA上公布更多關於半條命愛莉克斯的消息，還將公布另一款新遊戲-求生之路VR，G胖似乎跟「3」這個數字過不去了。

外國網友爆料的消息

考慮到此前不少網友猜測G胖要推出半條命3時，他卻公布了VR版半條命，這可以說出乎大量玩家的意料。

不過V社推出半條命愛莉克斯本就是想推動自家index VR套件銷量和擴大VR市場，那僅半條命一款經典遊戲還是有些不夠看，因此推出求生之路VR似乎也並不是不可能。

V社此前公布的半條命VR新作

其實求生之路製作組Turtle Rock早在2009年時，就留下已完成的求生之路2，離開V社準備單幹，它在之後找到了THQ作為新的發行商，並製作了一款非對稱競技遊戲-進化。

可隨著THQ破產，Turtle Rock的發行商又改為2K Games，因此G胖早已失去開發求生之路的原班人馬。

在V社將大量資源投入到VR領域的現在，G胖似乎在人力方面無法在Steam上推出半條命和求生之路等遊戲的正統續作。

那既然如此將這些經典遊戲做成VR遊戲，不也是一種有效的解決方法嗎？G胖或許正是這麼想的，才會一直致力於推廣VR設備和擴大VR市場，要是真成功了，V社也許會像半條命時期一樣再次引領遊戲業界。

不過求生之路VR始終是外國網友爆料出來的消息，是否真有其事還要等到2019TGA舉行的那一天（12月13日），沒準這個求生之路VR就是求生之路3呢？
