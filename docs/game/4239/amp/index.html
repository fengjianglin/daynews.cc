經典街機《懲罰者》防不勝防的腦洞，整個場景的箱子都可以舉起來

摘要

當年在遊戲廳火爆一時的《懲罰者》，相信是大家熟悉得不能再熟悉的遊戲了吧！然而就是這麼一款熟悉得不能再熟悉的遊戲，卻在二十多年後發現居然還有幾招隱藏招式和隱藏的暴擊。在網路沒有流行的九十年代，知道的玩家應該是是微乎其微吧！

當年在遊戲廳火爆一時的《懲罰者》，相信是大家熟悉得不能再熟悉的遊戲了吧！然而就是這麼一款熟悉得不能再熟悉的遊戲，卻在二十多年後發現居然還有幾招隱藏招式和隱藏的暴擊。在網路沒有流行的九十年代，知道的玩家應該是是微乎其微吧！

隱藏的暴擊相信大家都是知道的，在有限的時間內清掉小兵就可以對BOSS造成一定的暴擊，如果再加上滅火器和腳刀的組合，甚至可以秒殺2關BOSS。

而隱藏的招式當年在遊戲廳中並沒有多少玩家使用過

空中手刀：→↘↓B（一直按住↓不放）+A

迴旋擊飛：近距離→↗↑↖←↙↓↘→·拳（這可不是抓住人之後，直接點擊AB那麼膚淺的招式哦！）

這兩招隱藏招式在之前的文章中小編已經介紹了很多次，而且有不少玩家已經可以打出來了。在這裡就不再累述。

今天小編主要說的是遊戲中的另一個BUG，箱子的無限重疊。這個概念最早是我的好友@墮天使格鬥遊戲解說提出來的，並且做成了視頻在頭條發表。不過因為視頻被覆蓋的幾率非常高，因此很多玩家都是沒有注意到的。今天小編親自操作錄製並截圖，將這個BUG完完整整的展示給大家觀賞。

遊戲中，如果是有兩個箱子疊在一起的話，是絕對舉不起來的。

不過要是玩家技術到位的話，是可以利用手速來打破這個僵局。

看看下面小編的操作，是不是感覺很簡單？輕輕鬆鬆的就將三個箱子全都舉起來了。那麼是不是一會清完所有的小兵之後，可以將這些箱子全都帶到下一個場景中呢？

我們接著往下看

除了圓筒，正方形的箱子也是一樣的原理。玩家只需要通過極致的手速，同樣可以將箱子舉起來的。

相信當年在遊戲廳時期就已經有玩家發現了，但是卻並沒有多少玩家知道這個腦洞。三個箱子可以非常輕鬆的舉起來。現在已經舉起來了三個箱子，我們再試試舉起第四個箱子

是的，你沒有看錯，小編已經成功的舉起了第四個箱子。

然而因為屏幕的原因，上面的兩個箱子已經完全看不到了。

現在小編續幣將2P弄出來。但是因為箱子實在太高了。2P被卡在了上面下不來。

不過稍微跳躍一下，2P就可以從箱子上面滑下來了。

要是玩家有興趣的話，可以讓2P站在箱子上面掃射，然後由1P控制方向，看上去是不是很有意思。估計百分之八十的玩家都沒有體驗過吧！畢竟當年大部分玩家都喜歡單幹。

沒錯，2P的確可以通過兩個箱子的疊加，跳到1P舉的箱子上面去。但是在很多場合中並沒有那麼多箱子！如果只有一個箱子的話，想要跳上去幾乎是不可能的。

不過，我們仍然可以通過手速來解決這個問題。

在2P跳上箱子的瞬間，我們就可以將整個箱子連人一起舉起來。甚至2P本來就舉了兩個箱子的情況下，1P仍然可以將他舉起來。

我們在已經舉起幾個箱子和2P的情況下，再疊加箱子也是可以的。

不過要是讓小兵打一下，上面的箱子和人就會全部掉下來了。

那麼問題就來了，玩家要是在消滅所有敵兵之後，舉起了大量的箱子的話。場景切換時，是不是可以將所有的箱子都帶到下一個場景中呢？

很遺憾，似乎CAPCOM當年設計遊戲的時候早就已經想到了這一點！我們扛著箱子去下一個場景之前，所有的箱子全都掉下來了，只剩下手中的一個箱子可以帶過去。

看到這裡，大家是不是覺得當年白玩了，居然少了這麼多的樂趣。

另外，要是兩個玩家都舉起箱子的話，是可以相互調動的。

感覺是不是很過癮！
