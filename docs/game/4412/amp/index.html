摘要

不知道從什麼時候開始，“遊戲情懷”成了遊戲圈裡的一個重要風向標，而在這一波“懷舊情懷”做的最為成功的，就要數魔獸世界了。魔獸世界作為一款“年齡”高達15歲的網路遊戲大作，曾經經歷過的那些輝煌成就成為了一代人的美好回憶。

不知道從什麼時候開始，「遊戲情懷」成了遊戲圈裡的一個重要風向標，而在這一波「懷舊情懷」做的最為成功的，就要數魔獸世界了。

魔獸世界作為一款「年齡」高達15歲的網路遊戲大作，曾經經歷過的那些輝煌成就成為了一代人的美好回憶。大多數80、90後對於魔獸世界都有一種很深的羈絆，在艾澤拉斯的世界裡，擁有的不僅僅是遊戲回憶，還有一代人的青春。在玩家們熱切的期盼和建議下，魔獸世界懷舊服終於按時開放，但經過一段時間的「激情冷卻」，懷舊服里的諸多問題逐漸顯露……

「那些年」的魔獸世界，沒有金錢利益的糾葛

魔獸世界懷舊服雖然還原了當年你的遊戲核心玩法，但隨著近十幾年互聯網和電競的飛速發展，「那些年」的魔獸世界與現在早已不同！那些年，我們守護艾澤拉斯那是真守護，公會兄弟的情誼那是真情誼！當然了，並不是說現在的懷舊服就沒有真兄弟情，而是在這樣的大環境下，這種情誼在遊戲里顯得更加彌足珍貴更加稀有。

那時的魔獸世界裡，玩家與玩家之間並沒有多少金錢利益的糾葛，而現在的魔獸世界懷舊服，我們把以賺錢為目的的工作是去掉之後，還能剩下多少只為遊戲的玩家？

網路環境的快速發展，讓很多「能人」找到了更巧的賺錢方式，隨著直播、短視頻等一些列的網路新生產物的誕生，遊戲圈其實已經變得非常「透明」。比如我們用手機隨便打開某個直播軟體，你就能隨時觀看關於任何遊戲的視頻或者直播，隨著「快餐信息時代」的發展，自然會從這些熱門的IP中產生金錢利益的糾葛，而這些擁有龐大玩家基數的遊戲，就是所謂的「熱門IP」。

講一個挺讓我感觸頗深的例子，真正經歷過60、70級版本的玩家應該對「蒼天哥」很熟悉。蒼天哥可以稱得上是魔獸世界中的第一批「網紅」，如果按照現在時代定義來衡量蒼天哥，那我覺得怎麼也得是不輸於PDD、大司馬這樣的存在。要知道，在真正魔獸世界還不是懷舊服的時代，這款「熱門IP」的火熱程度是現代任何一款遊戲都比不了的，包括英雄聯盟。

也就是在偶然的某一天，我不經意間看到某一個直播平台有一個熟悉的ID，進去一看還真的是當年我熟悉的蒼天哥，只是再看他現在的熱度和影響力，早已物是人非，甚至其名氣還不如「黑了裝備」的「毛某風」。其實說這個例子我想表達的是，魔獸世界懷舊服的出現是沒有錯的，只是在十幾年後的今天，很多東西已經變味了，甚至被某些風氣帶的烏煙瘴氣的！

除了大量工作室，我們的情懷還剩多少？

15年前，我們是因為好奇走進了艾澤拉斯。

15年後，我們又因為情懷回到了曾經的艾澤拉斯。

短短几個月的時間之後，大多數老玩家已經膩了所謂的情懷，並萌生出「正式服他不香嗎」的想法，包括我。

短短几個月的時間之後，我們的情懷不知還剩多少，但可以確定的是，遊戲工作室已然佔據了艾澤拉斯的半壁江山！凡事不能一概而論，但現狀卻是遊戲工作室絕對比情懷多。

可能你會說，正式服也有數不清的工作室，魔獸世界也不是當年的那個魔獸世界了！我想說的是，首先我並不是要表達「正式服比懷舊服好」這樣的觀點，正式服養起來一個龐大的遊戲工作室體系是大家都知道的。但正式服的遊戲工作室與遊戲環境卻隨著15年的時間，慢慢的找到了一個合理共存的平衡點。舉個簡單的例子，萌新玩家在魔獸世界裡呆不下去的主要原因就是沒有「團體」，而不管在正式服還是懷舊服，沒有團體的玩家都是難以生存的。但在正式服中，遊戲工作室在某些方面成了玩家組成團體的橋樑，比如我們最常見的「免費帶成就團」。懷舊服與之截然不同的是，遊戲工作室因為「大熱IP」而大量湧入，用非常嫻熟的「賺錢套路」衝擊著60級版本「稚嫩的遊戲體系」，這帶來的結果就是災難了！

要知道，一個是經歷15年時間打磨達到平衡的，另一個是直接用成熟體系來衝擊稚嫩遊戲環境的，帶來的影響又怎會一樣？懷舊服一把風劍動輒拍賣上幾萬塊，這種在魔獸世界發展15年里也是聞所未聞的，但這種現象就這麼隨意的在懷舊服發生了。遊戲懷舊無錯，錯的是沒有一個適應當下時代背景的制度！

我對遊戲圈裡懷舊風的一些看法

遊戲圈懷舊風隨著時間慢慢的「冷靜」下來，遊戲懷舊，是對經典的一種致敬。比如星際爭霸重置版，比如魔獸爭霸重置版，還有玩家特別期待的暗黑破壞神2重製版。懷舊是玩家對經典追憶的一種期待，是一種「純粹的美好」，不容許參雜其他破壞這種感覺因素的事情。我期待遊戲圈的懷舊風，因為有了這些經典的存在，會推動和鼓勵未來的遊戲圈出現更加優質的作品。

魔獸世界懷舊服，養了一大批工作室之後，情懷還剩下多少？這個問題有無數種答案，但有一點是不會錯的，那就是破壞遊戲環境的因素一定要控制，我們不談「完全杜絕」，因為這是不可能的，也是遊戲發展不可或缺的。我們追求的是一種平衡，平衡達到之後不僅僅是為了讓玩家有一個公平的、寬鬆的遊戲環境，更多的是讓我們心中對於經典致敬的那一抹凈土得以保留。

經典的曾經是玩家心中最美的回憶，只可回味不能複製

經典的曾經不可能重現，只能作為玩家心中最美的回憶，只可回味不能複製。不能複製並不是指不能出現「懷舊服」這樣的遊戲模式，而是指的不可能複製出當年盛況的巔峰成就！而只可回味，是指我們目前要珍惜已經開放的懷舊服，在裡面盡情的回味只屬於你的那一份青春回憶。

對於魔獸世界懷舊服，以及遊戲圈裡陸陸續續可能會出現的懷舊情結，你的態度是什麼呢？
