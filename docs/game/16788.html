狂人幻化武器血虧，一把天脊乾坤劍上心悅3，倒霉小王子實至名歸

2019-12-16 1,378條評論 2,854次閱讀 209人點贊

又值歲末，想必地下城與勇士的策劃又想好了2020年的春節該如何撈金了，除了春節禮包，增幅活動，肯定還有各種層出不窮的活動，不過說回來，地下城與勇士這款遊戲的撈金能力還是非常之強的，不然也不會一直都穩居網遊戲氪金榜第一名了，就比如今年地下城與勇士上線的武器幻化功能，其實並沒有什麼實質性的用處，只是能讓玩家拿上自己的喜歡的武器而已，顏值那是地下城與勇士這十一年以來的巔峰時刻。

說到地下城與勇士，最關鍵的還是戰鬥力，所以裝備的打造還是重中之重的，現在的九十五級版本幾乎每個人的武器、裝備都沒有太大的區別，所以戰鬥力完全就體現在打造之上，現在最新的普雷副本主C最起碼也是要全身增幅紅12以上的裝備，而且最近韓服更新到了100級，國服應該也是相差不遠了，所以各位勇士們要抓緊攢材料升級裝備了。

地下城與勇士之中，強化和增幅裝備成功的概率都是非常之低的，越是往上就越困難，所以想要將一身裝備增幅到紅13以上還是比較困難的，但是從國服目前來看，還是有不少的神豪打造了一身好裝備的，增幅紅13的早就已經是隨處可見了，而且增幅紅14、紅15的也是非常之多了，但是增幅到一身紅16，甚至是紅17的就非常之少了。

一身裝備增幅紅16，大家肯定會想到DNF倒霉小王子，也就是虎牙直播平台的主播狂人，他的劍魂裝備就是一身增幅紅16的打造，而且還準備在2020年的春節之時增幅五件紅17的裝備，既然他裝備已經打造得這麼好了，那麼一身的時裝也是最高級的天空套裝，而且所有鬼劍士都是清一色的幻化無影劍，顏值是非常的高。

按照狂人幻化武器來看，開出一把無影劍也還是很容易的，但是在昨天直播中，他幫水友幻化天脊乾坤劍卻是血虧，直接幫號主上了心悅3，按理來說一把天脊乾坤劍，但是卻怎麼也出不了，硬是出了四把無影劍，就是不出天脊乾坤劍，可把狂人給愁壞了，他這個DNF倒霉小王子的稱號真是實至名歸。
