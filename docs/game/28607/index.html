王者榮耀：在峽谷里最大的錯覺就是，你以為他很肉，其實他很脆

2019-12-22 93條評論 1,120次閱讀 190人點贊

一曲歡樂送，一場紅塵夢；大家好，我是電競歡樂送。

在大熱的手游王者榮耀里，有的遊戲英雄看起來十分高大威猛的樣子，你們也被他們的外表所迷惑。就算連裝備也弄得很肉，但其實最主要的還是這些遊戲英雄的坦度不夠，承受不住太多的傷害，讓人會有錯覺覺得他們好像很肉，但其實這些遊戲英雄實際上是很脆的。今天，小編就給大夥透露一下這些遊戲英雄吧，讓大夥在選擇遊戲英雄的時候可以參考參考！

第一個是峽谷錘霸鍾無艷。鍾無艷前期的傷害是讓人畏懼的，小編每次前期遇到鍾無艷都容易被打成臭頭。特別是她的二技能震懾打擊，在一定範圍周邊的敵人的傷害是雙倍的，脆皮一個不小心就會被秒殺掉。可惜鍾無艷的坦度好像被削弱掉了不少，所以好像不太能在團戰中站得住腳，支撐不了多久，通常放完一套技能後就差不多可以撤了。

然後就是小編有點害怕的呂布了。他的大招是有範圍性的控制技能，每次小編的看到呂布的大招就覺得好像會有大牌的魔神降臨。雖然呂布的遊戲英雄形象看來非常的高大魁梧，全身就是鎧甲好像很威風的樣子，但是並不像他的身材那麼堅挺，如果在遊戲里，先跳大沒有命中敵人的話，就很有可能被敵人秒掉。

接著就是牛魔了噢。作為輔助的牛魔大哥，在打團戰里最大的作用就是橫掃峽谷戰場，然後為隊友們創造出一個很好的輸出環境以便於他們輸出。牛魔的身材看起來十分的壯碩，但其實真的都是錯覺。他的一技能雖然可以降低敵人的傷害輸出，可是面對娜可露露這類高輸出的遊戲英雄還是容易被秒殺掉的。所以用牛魔的玩家們還是得多加小心高輸出的遊戲英雄。

最後就是騎馬的關羽了呀！怎麼說呢，在小編看來關羽就是一個邊路遊戲英雄，但在大峽谷里好像一直挺熱門的，在團戰里也可以起到很重要的作用。但由於關羽一般都走邊路又是輸出裝，所以坦度也沒有很高。如果要說優秀的開團遊戲英雄，關羽一定是在榜上的，但是關羽開團的時機也是相當的重要，沒有把握好的話，也很容易破壞隊友的陣型和節奏，打亂節奏之後，也是很容易被敵人一套帶走的。
