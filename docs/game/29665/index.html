一個作者對工作室說的真心話：遊戲搬磚，思路決定出路

2019-12-25 1,041條評論 2,108次閱讀 279人點贊

一個作者對工作室說的真心話：遊戲搬磚，思路決定出路

這個月因為魔獸懷舊服這波大肉，我們工作室加班又加點，加上雙11新組裝了近50台電腦，實在忙的頭暈眼花了。

有個很奇怪的現象，現在魔獸吃肉這麼火爆的情況下，依舊還聽到有點說DNF混不下去了、天刀搬磚賺錢難了、閱讀新聞項目也嗷嗷封號，遍地哀號！我對此感到很奇怪。還真是，思路決定出路這句話不是白說的！

95%的小GZS在虧損，40%的大GZS沒開機項目，應該是這個行業的潛規則嗎？其實不是沒有項目啊，而是大家的思維沒跳出來，沒有敢於接受新的事物，嘗試新的玩法。

記得幾年前手游剛剛崛起的時候，我第一桶夢幻的金沒撈到，心疼了很久，但是我在劍俠情緣，誅仙，捕魚裡面著實撈到了很多很多，而且那段時間是我比較低落的時候，天刀和冒險島2的虧損單純依靠手游的補給完全不夠，手上的電腦也是清一色的I3，腳本那邊只能找網盤，各種渠道都沒有！捕魚大火的那段時間出彈頭被很多無良黑商欺騙，過年身上只有1000塊，年後甚至開機的慾望都沒有。

這時候還是感謝一個賣IP的好朋友，給我說DNF那邊有個客戶弄了很多IP，而且每隔幾天就要加，這時候才去研究DNF，直到從一開始的借號，租號開飛機，到後面租大號開飛機團本，再到後面的4開，16開，64開，中途也被一波絞殺過幾次，但是都在各種實驗測試中活下來了。

但是，這些一切的一切，都是自己玩的單人，一直到後面劍三，接觸的手動搬磚的比較多（可以說依靠一些小軟體半自動，但是人工成本高）才發現，這些所謂的半自動擋項目是真的穩啊，除了人工的損耗，各種利潤都不錯，然後和這些小夥伴，各自有各自的需求，就慢慢聚集成了團隊，負責各方各面，也就慢慢的，這個圈子裡面的人都開始走上正軌。

逆水寒的那一會，全民搬磚，GZS更加的瘋狂，我們從一開始的刷RC還有運鏢，到後面的賭百鍊等，都是用的單機手動，這時候我就開始尋找更高效的方法！

因為搞劍三熟悉按鍵這個，就實驗了一下，直接單機單獨做個按鍵，無腦跟隨一個號走，各種調試最後達成，可以無腦刷本，腳本這個不是沒用，封的厲害，沒必要給網盤作者做投資，這也是我想說的，別問我什麼項目行不行，你覺得行就行，你能有利潤產出就是項目，腳本這個東西，看你的需求，如果你的需求比較簡單，你甚至可以聯繫上作者，和他商量分成之類的。

沒有不封的遊戲！

什麼遊戲，包括魔力這種永不封號，或者TX和WY等1天死號的遊戲，都是會讓你有很多虧損的。

這邊順便回復下問我姿態的小夥伴，你說你問我這些，我不要錢告訴你吧，我自己投入過這麼多，我和你也不熟，為什麼無償教你，然後說給錢吧，我又不想被天天問來問去，現在魔獸單機100的收入，不說這些新進設備的投入，單獨說起號，我們工作室測試，調節，以及一些7788的東西，每個人的投入都在1W多的費用，你覺得我會給你說魔獸怎麼起號刷G才穩么？

然後DNF那邊，這麼爛的項目，防封手段NBE，包括搬磚吧都爛大街了，那些收費發你個文本的，無非就是去其糟粕取其精華罷了，甚至問我魔獸賺錢姿態的，兄弟，你找的網盤，或者你搜搜上個帖子，或者你去翻翻度娘還有NBE，都是有很多很多的！

有個小老妹，和我第一次說話就給我發了個200的紅包，前前後後一共發了800多吧，天天一堆重複的問題，問我這個問我那個，然後又是這個腳 那個的，錢花了不少，到頭來我說的一個都不聽，然後說虧了5000，不做了，我也是很服氣來的！

還有個逆水寒搬磚的團隊也是，問我魔獸的搬磚，我說要他們6個人起12個法師號，到頭來2個聖騎4個戰士3個DZ，我想問下你們是玩遊戲的還是來賺錢的。

方法我不是不說，是說了你沒有自己去思考，純粹靠我來教你，我累了，倦了，煩了就會不回，不說話。

當然，BB這麼多，也是想和各位說，賺不賺錢，看你自己的，我開始在銷售公司待過，別人一個月20W提成，我是個鹹魚，一個月2W出頭。

要清楚幹這一行，你不需要給別人打工，你就是老闆，你需要自己奮鬥一點拼一點，魔獸剛開的第一個月我每天就睡4小時，真的就是躺在椅子上太困了就眯一會，一個兩個小時的，驚醒過來就繼續弄，那時候真的是暴利啊，一天幾千多，現在利潤沒那麼高了，就得走量了，可是有沒有錢賺真的看你們自己了，很多東西靠自己，別人當不了你的導師。

不多說了，好好睡一覺，明天繼續努力賺錢。
