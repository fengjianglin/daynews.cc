CFML丨12.21回顧：SV潛艇發力難改天命 AG黑色城鎮獲勝成晉級關鍵

2019-12-23 1,268條評論 2,742次閱讀 1245人點贊

今日上演CFML2019職業聯賽秋季賽季後賽第一輪的第一個比賽日，AG與SV將爭奪第一個晉級季後賽第二輪的資格，對於兩支隊伍而言，由於今日的比賽至關重要，因此二者都將使出渾身解數爭奪比賽最終的勝利。

【SV率先發力 怎奈AG逆天改命】

SV與AG率先在港口中交手，在港口的比賽中，SV憑藉青尤阿軒等人的發揮幫助隊伍拿到比賽的節奏控制權，本以為SV能夠將比賽的優勢持續到最後，但是隨著AG的調整，雙方比分不斷縮小，雙方8比8的戰平，無疑令比賽進入白熱化階段。SV選擇暫停進行戰術變動，不過AG的冷靜應對最終將比賽定格在10比8的畫面，AG率先先下一城。

縱觀第一局的對決，AG之所以能夠反超比分，是因為青菜蕭雲總督三人的發揮，三人穩定的擊殺無疑保證了AG的整體火力輸出。

雖然SV的青尤有四殺的高光發揮以及阿軒為SV提供了穩定的火力，但是在最終的關鍵局中，SV的進攻未能突破AG的防守從而導致SV敗北。

【AG乘勝追擊 SV衛星基地難敵AG】

雙方第二局比賽來到衛星基地，在這幅地圖中由於地形因素導致保衛者略有優勢，不過作上半場作為保衛者的SV卻並沒有限制AG的整體進攻，AG的發力促使他們作為潛伏者時掌握比賽的節奏，當雙方攻守互換之後更利用比賽的地形加強防守從而輕鬆獲勝。

在這局比賽中，無論是AG的賭點還是個人選手的發揮，都有著亮眼的表現，而者也成為AG在第二局中掌握比賽的節奏。這局比賽中，青菜整體的亮眼發揮成為AG獲勝的關鍵，青菜也獲得本局MVP榮譽。

【SV陣容變動 成扭轉比分獲勝關鍵】

在第三局中，SV選擇派五柳上場作為主突破手。潛艇作為SV的一大優勢地圖，在今日他們再次展現了教科書般的發揮。SV對比賽地圖的關鍵節奏的把握，成為SV掌握比賽節奏的關鍵。寡婦率先上演的四殺發揮無疑提升SV士氣，雖然在下半場中SV已經拿下賽點，但是AG並未放棄，九天的四殺發揮推動著AG不斷爭取比分，不過最終本局以SV獲勝而結束。

對於SV而言，第三局的獲勝無疑令他們獲得喘息的機會，SV寡婦阿軒以及青尤等人在本局的強勢發揮為SV獲勝埋下伏筆。在本局中，SV對比賽節奏的掌握，以及多次率先撕開AG的防守成為SV獲勝的關鍵。

【AG黑色城鎮發力 最終強勢獲勝奪得資格】

在第四局的黑色城鎮中，作為潛伏者的SV頻頻發力突破，但是AG更為默契的防守，令SV未能獲得進攻的時機，而著也導致SV處於比賽的劣勢。當AG拿下7分之後，SV選擇暫停進行短暫的調整，不過暫停結束之後作為潛伏者的AG繼續發動猛烈的進攻，令SV未能招架，最終AG以10比4的比分戰勝SV，從而以大比分3比1獲得季後賽第一輪的勝利。

縱觀今日AG的整體表現，選手的穩定狀態成為AG獲勝的一大關鍵，當AG處於保衛者角色時，能夠利用地形以及完善的分配人手完成防守戰術從而限制SV的進攻，這便是今日AG獲勝的關鍵。

隨著AG以3比1擊敗SV之後，季後賽第一輪第一個比賽日也隨之結束，AG的獲勝令他們獲得季後賽第二輪的參賽資格，當他們迎戰老對手eStar時，他們能否再次獲勝，讓我們拭目以待。
