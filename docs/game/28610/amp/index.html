王者模擬戰傳統陣容變形,當刺客陣容加上攻速buff,切豆腐就完事了

天天要聞
2019-12-22

王者模擬戰幾經更改，有些傳統陣容已經不像以前那樣強勢了。像刺客這類陣容沒有攻速的的話，到中後期就會顯得很雞肋。而這套陣容用堯天羈絆的效果彌補了這一缺點，快來看看吧~

這個陣容最終成型的羈絆是三刺客、四堯天、二稷下、二輔助。由小明、阿離、老虎、弈星、婉兒、李白、廉頗、孫臏和宮本組成。

前期我們可以用三長安開場，然後再拿兩張堯天，在保證經濟的情況下在前十個回合升到5人口，這樣場上的陣容就可以由阿離、老虎、婉兒、程咬金和鍾馗組成雙坦克、雙堯天和三長安的陣容，這樣陣容既有一定的坦度，也能保證後期的經濟。三長安拿不到前面說的牌也沒關係，不必強求雙坦克的羈絆。婉兒的話，刷到再換上去就行。

如果前期陣容扛不住傷害，我們可以拿失敗的經驗來賣血保經濟，在50血左右就要鎖血了。五人口時不用刻意追求棋子質量，不用去搜牌，順勢拿就行，升人口比較重要。中期，盡量在17回合之前把人口升七，先湊齊四堯天，再拿李白和廉頗替換場上另外兩個長安。這樣場上的陣容的羈絆就是四堯天和三刺客。

然後，等到了8人口拿到孫臏時，陣容已經成型。如果暫時拿不到孫臏可以先拿墨子來代替，先湊齊二稷下的羈絆，鍾無艷也可以。等到了後期，我們可以拿一個宮本補全九人口陣容，因為宮本和李白有一個CP天賦雷霆劍道能增加傷害。

這個陣容主要就是拿刺客的暴擊傷害加上攻速打出更多的傷害，基本上就是速戰速決型的，但在坦度上有一定的缺陷，兩輔助的羈絆能增加一定的韌性，如果刷裝備的時候有戰士令旗也可以考慮拿一下增加一下防禦度。

稷下堯天刺的陣容就是這樣，大家有什麼問題，歡迎留言互動~
