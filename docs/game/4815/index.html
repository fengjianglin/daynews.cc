王者榮耀：因策劃嫌名字難記，被重新命名的5名英雄，阿軻露娜上榜

2019-12-04 0條評論 14次閱讀 0人點贊

摘要

峽谷很多英雄的名字都是手到擒來，比如：孫悟空、李白、趙雲，但是像一部分新創作的角色，比如：鎧、露娜、夢奇，起名字就要花點功夫，又要好聽又得好記。這些新創作的英雄，早已被我們熟知，但小夥伴們知道嗎？他們現在的名字，其實並不是本名，曾因為原名難記被更改過。

峽谷很多英雄的名字都是手到擒來，比如：孫悟空、李白、趙雲，但是像一部分新創作的角色，比如：鎧、露娜、夢奇，起名字就要花點功夫，又要好聽又得好記。

這些新創作的英雄，早已被我們熟知，但小夥伴們知道嗎？他們現在的名字，其實並不是本名，曾因為原名難記被更改過。

鎧

鎧曾用名：拉斐爾、魔鎧，剛開始爆料要出新英雄的時候，名字是拉斐爾，後來出模型的時候，又顯示是魔鎧，但最後上線的時候只有一個字「鎧」，讀起來挺酷，是王者榮耀第一個單字命名的英雄，改的算是比較成功。

阿軻
阿軻曾用名：荊軻，作為王者榮耀遊戲里第一位登場的刺客型英雄，在2017年4月，荊軻重做，並改名為「阿軻」。在王者榮耀背景故事裡，我們也可以知道「荊軻」這個名字就是一個家族代號，在阿軻哥哥死後，她繼承家族命運，但作為女性，這個新名字也更貼切。

夢奇
夢奇曾用名：夢見貓，這個稱呼比較有意思，還記得夢奇還正在製作的時候，被玩家利用bug把原型給曝光了，但當時很多人認為是假料，沒想到會出一隻貓，而且現在還成為了官方的吉祥物。剛出來的時候特別火爆，傷害高又肉，但後來調整後，大跌甚至無人問津，不過現在做了優化，情況稍有好轉。

露娜

露娜曾用名：紅佛，這個名字聽起來很奇怪，畢竟露娜原皮也沒有紅色，伴生皮緋紅之刃倒是沾邊，難道伴生皮其實是原皮，後面才調換的？不過改過之後的名字，的確比原來好聽，也更有仙氣。

公孫離

公孫離曾用名：離，王者榮耀里單字的名字並不多，鎧是第一個，很遺憾第二名與公孫離擦肩而過，被曜佔據了，而且第三名還有可能是曜的姐姐，鏡。但相比官方給出的公孫離，相信大家叫的最多的應該還是阿離。

有小夥伴說，「什麼華麗的名字最後都會變成俗稱，比如曜變成花里胡哨，蔡文姬變奶媽，上官婉兒變成秀兒。」最後，還是玩家的智慧更勝一籌。
