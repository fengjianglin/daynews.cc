這就是LPL好消息？起小點回應：我隨便說說，沒想到這麼大反應

2019-12-12 1,584條評論 2,792次閱讀 112人點贊

12月11日晚上的時候，突然知名電競媒體起小點在微博上發言表示：30小時內會有大消息公布，而且是一個非常好的消息，並不是關於選手的退役，而是可以讓明年S10世界賽更加精彩，LPL越來越好的消息。這個消息一爆料，就立馬引發了大量網友的好奇心，很多人猜測是S10世界賽名額要多一個了，或者說是有強隊合併。不過沒想到的是，12月12日中午官宣的所謂好消息就是estar和PDD聯手進入了LPL。

​這個消息對於PDD來說，肯定是好消息，畢竟他這麼多年了一直想要進軍LPL，但也一直失敗。但是真的有像起小點所說的那樣，會讓S10變得更精彩，會讓LPL越來越好嗎？好像的確還沒有到這個程度。就像是去年V5加入LPL，有人會說這是對LPL的好消息嗎？因此這件事情讓起小點遭到了收到了的質疑和批評，在estar官宣之後，起小點本人也做出了回應。

​起小點對於自己之前的爆料回應如下，他表示：estar和PDD都是這幾年為了電競做出巨大貢獻的人，所以我們應該對他們表示歡迎，而且有人說我之前在蹭熱度吊胃口，但是真的不是，我只是真心為他們開心，沒想到引發了那麼大的熱度，以後還是少說話了。

​雖然起小點解釋了，但是很明顯網友們並不買賬，在他這條解釋的微博下方，還有不少人在怒噴：又當又立！故意借著LPL大新聞的噱頭炒作，有熱度了之後就開始洗白說以後不說話了，真的是噁心人！LPL的好消息還以為是對聯盟或者戰隊有什麼新的好政策了，沒想到就是一支大家都知道的新隊加入了，炒作可以理解，但也沒必要上升到lpl的大好事這個層面吧？噁心人真有一手！為了熱度一點臉都不要了，不愧是PDD的舔狗。

​最後在評論區，起小點再次和微博的網友互動，他表示理解大家對這件事情的失望，自己就是隨便發了幾個字，沒想到大家互動和反應這麼大，帶了節奏，是我的問題吧！當然他這一波的回應，再次引發了大量網友的怒噴和攻擊。

​那麼，對於起小點的這波回應，你們有什麼想說的嗎？
