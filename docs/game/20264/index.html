我輩因何奪冠？月光神BP助力，4分換線運營成AG致勝關鍵

2019-12-17 2,642條評論 4,120次閱讀 2540人點贊

「讓我們，恭喜，AG超玩會！」

在周六，KPL秋季賽總決賽的獎盃被AG眾人捧起的時候，無冕之王的AG超玩會終於在月光教練的帶領下，擊敗了QGhappy，拿到了這個回歸KPL賽場後的冠軍。

可以說，影響這場比賽的因素有很多，從選手到教練，從台前到幕後，我們都可以看到AG真的是做足了120分的準備，來拿下這座銀龍杯的。雖然最終決賽的過程顯得那麼水到渠成，但為了達成這個局面，AG全員都為止付出了非常艱辛的努力。這些努力在飯飯看來可以總結成四個關鍵詞：

月之BP，英雄海，四分鐘timing，搏命式運營。

月光固然靈性，但英雄海才是王道

很多人都會說，AG這幾場比賽的BP做得真的是非常好。

第一局，月光靈性選出蔡文姬來克制QG沈夢溪；到第三局為避開保射手的莊周而選了一手超遠POKE流嬴政；到第四局開局選一個多體系的功能型輔助孫臏；再到最後一局宮本和瑤的選擇，都叫人拍案叫絕。

但是，不能忽略的是，月光能給出這麼多靈性的選擇，還是建立在AG各個隊員英雄池恐怖的基礎上。

要知道，在總決賽之前，AG的英雄池一直都是飯飯擔憂的點，甚至需要輪換隊員藍柚出來才能填補打野位置的空白。而且，六哥的藍領打野也並不出色，麟羽坦邊的表現平平，在打野和對抗路的抉擇上，AG可以說是需要面對一個十分尷尬的情況。

但就在總決賽第二場進行當中的時候，飯飯一度以為AG可以實現總決賽4:0零封對手的壯舉！在六哥祭出了打野盤古，老帥拿出了法刺火舞的情況下，AG還能一度佔據主動，只能說他們在賽前兩周內的成長太大太大了。

從這裡，如果QG還沒意識到，AG已經不是之前的AG，甚至不是季後賽的那支AG的話，那就真的是為時已晚了。

第三局，麟羽習得藍柚一手趙雲的精髓，打出了4-1-8的優異數據；愛思一手不常見的孫臏也是成功打出了戰術作用；第五局更是老帥的法刺上官婉兒和愛思的瑤引發全場尖叫。

如果不是英雄池的深挖，我們又怎麼能看到全局BP下AG這麼可怕的英雄海呢？

從原本需要輪換藍柚才能拿出來的趙雲，到不會輕易祭出的老帥的法刺，飯飯看到的，是奪取這個冠軍銀龍杯必勝的信心和氣勢，還有背後不為人知的刻苦訓練。

而反觀QG，仍打算按著自己一貫的節奏來，按著4保1的射核節奏來。結果，也是顯而易見的：只會射核的QG，終究敵不過英雄海的AG。

四分換線搶龍，timing節奏完全不虛

可能很多人會奇怪：明明是一支只會打架的AG超玩會，為什麼第一盤的運營就如此出色，拆塔破塔，一切都井然有序？

其實多看AG的比賽就會發現，AG在4分鐘的炮車出擊還有5分鐘的先知主宰的時間點是十分看重的，用星際2的話術來說，算是卡了「timing」。

所謂「卡timing」，就是讓自家射手或者團戰輸出點在先知主宰刷新之前換線，讓帶線的戰士能更好地帶線，增加5打4的幾率；同時，讓射手靠近主宰龍坑，在對面未換線前提前利用前4分鐘的高經濟壓制對抗路戰士，獲得對線和搶中立生物的優勢。

在第一把以裴擒虎為野核的運營戰中，AG選擇優先推下塔，給5分鐘爭搶先知主宰製造空間（方便關羽單帶）。在5分鐘的時候成功推掉下路一塔後，關羽光速換線，節奏一下就被提了上來。

而在第二局，一旦到了4分鐘炮車出擊，發育路失去金幣加成之後，一諾的成吉思汗就和麟羽的馬超實現了光速換線，並在4分30秒的時候換線成功，等待先知主宰刷新，而QG這邊直到6分出頭，才完成射手和戰士的換線。要飯飯來說，這第二局若不是最後Fly的救世狂鐵以一敵三，其實AG的贏面還真不小。

同樣，第三局，4分25秒完成換線，5分26秒拿先知主宰；第五局，4分26秒完成換線。可以說，AG這次總決賽的指揮和戰術貫徹上是十分到位，且目標明確——即使大家都知道4分鐘炮車出擊可以換線等著拿龍了，AG也能搶先製造換線空間，為後面核心搶龍和分帶，提供更完美的戰術支持。

推塔節奏得當，不光打架更有運營

前面「卡timing」如果算是AG在比賽中十分重要的戰略布置的話，那麼，在這個換線時間點之後，AG進步的地方就在其一整套運營。

要知道，AG在這次總決賽之前，但凡對上QG，可以說是灰頭土臉，毫無勝算的。

但是，AG這次，不僅打出了自己開團打架的氣勢，更是學到了一手運營的精髓，將新版本地圖的運營策略徹底消化，並化為己用。

最典型的運營要屬第一局的比賽了，用步步為營來形容AG的表現可謂再恰當不過。在人頭爆發數極少，且都是最後一波團戰才爆發的情況下，AG超玩會完全利用運營磨塔的策略，由裴擒虎和周瑜一起一個一個塔拔掉。直到推掉高地，QG這邊可以說是一點應對策略都沒想出來，只能遺憾輸掉第一局比賽。

而作為運營中的一環，AG騙關鍵性技能的戰術也執行得很徹底。比如集火探草的張飛迫使其交出大招，亦或是張良對廉頗的強行控制，還有瑤在中期騙孫臏大招吃傷害的操作，都是運營節點上十分關鍵的戰術執行。

再比如，在第四局麟羽的暗信被曜單殺那波，看似是個人操作的問題，但是為什麼暗信要這麼深入對面野區去搶資源和開視野？目的就是限制打團能力比自己強的曜去支援，讓飛牛無法參團。

在暗信帶線，拖住對手的運營策略下，直到10分鐘，AG的三座外塔都還佇立在場上。這樣讓戰火只燃燒在對方半場和中路，保全了自己野區，而且戰術運營後的效果也是顯而易見的。而反觀QG這邊，隊伍整體在廉頗被限制的情況下，帶線帶不過，開團打不死六哥的老豬，更是因為孫臏的拉扯而無法秒掉法師和刺客。

在打團方面AG固然有其能力去接下QG的團，畢竟戰鬥才是他們的強項。但是，能4-1戰勝對手，更多的是因為AG在運營上的出色。這樣出色的運營，使得AG在很多次團戰中，都能利用這或那的條件來扭轉局勢，甚至打出4打5反殺2人，打得QGhappy只會在中路唱愛的魔力轉圈圈……

總結：征程開始，引領新版本王朝？

正如AG教練月光賽後所說，關於他自己，關於AG的征程其實才剛開始，AG超玩會在這次KPL秋季賽的表現真的十分恐怖：士別三日，刮目相看。從讓人擔憂的英雄池，到任選任挑的英雄海；從只會打團的菜刀隊，到換線運營的強執行。AG的成長還有表現，大家都看在眼裡，他們對新版本的解讀和領悟力也是值得很多隊伍去借鑒和學習的。

飯飯有理由相信，AG月光新立下的Flag並不是空穴來風，AG開啟王朝的可能性也並非為零。最後，讓我們還是回到文章最初的那句「燃爆」了的話——

「讓我們，恭喜，AG超玩會！」
