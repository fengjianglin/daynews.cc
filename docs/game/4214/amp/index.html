相信入坑已久的元氣騎士的玩家們一定不拘泥於每天闖地牢打Boss奪回魔法石吧，在元氣騎士遊戲貼吧內無數玩家開始放出了自己在地牢闖關時遇到的沙雕瞬間，一年一度的沙雕圖大戰終於開啟了！讓我們來看看最受元氣騎士認同的爆笑場面究竟有哪些吧！

TOP5將要成為地牢海王的男人！

在地牢里的特殊房間玩家通過觸發陷阱擊敗怪物獲得強力的武器道具，然而這位玩家彷彿是天生的海王，一次直接開出了三把三叉戟，這概率可小的驚人，看來這位玩家把買彩票的運氣全部都花在了開箱子上，實在是令人忍俊不禁。相信只有成為地牢海王的才能有這種奇遇！

TOP4San值警告！

地牢中曼陀羅花因為其無法移動的特性和，特殊地形的保護！一直以來就是就是玩家屠殺的對象，而這個曼陀羅花死亡的位置和實際都太過詭異，從正面看彷彿就是從傳送門裡生出的無數觸手，這和克蘇魯神話里的不謀而合，不知道我們的小騎士會不會看到這驚悚的一幕，從而san值不夠而發狂失去理智！

TOP3誰才是地牢的真Boss！

這個比大騎士體型更為巨大，並且著裝還更加霸氣的聖騎士是不是擁有能夠取代大騎士，成為地牢里真正的Boss呢，畢竟只有真正的大騎士才能擁有自己的侍從小弟跟隨！而另外一位玩家則更過分了，直接把大巫師變成了我們的法師小姐，看來新版本的法師才是地牢中的真正BOSS呢！法師沙雕獃滯的外表配上霸氣的字樣實在是太有搞笑感了！

TOP2來自單身工程師的怨念！

不僅是在現實中因為工作原因程序員難找對象，就連地牢的工程師也難逃這個命運，於是我們單身工程師就設計了一個完美的圖案來表達了他對愛情的渴望，沒錯希望有情人能夠忠誠眷屬！來一場如TNT爆炸般轟轟烈烈的戀愛吧，哎果然單身技術宅的工程師惹不起，這個機關太險惡了吧！

TOP1來自官方的綠色暗示！

歐皇們在許願池裡不需要多少金幣就可以獲得各種各樣的優質武器，甚至珍惜的武器藍圖！然而這位玩家好像從許願池裡拿到了一個奇奇怪怪的圖紙，這波來自官方的綠色暗示是不是太過致命了呢？

當然還有更加過分的，有玩家一次從寶箱里直接領到了兩頂綠色的帽子，看來這位玩家是個有故事的人啊！好了以上就是元氣騎士玩家們展示的沙雕圖了，不知道各位玩家們還有哪些搞笑沙雕圖呢，記得點贊和評論哦，我們的斗圖大戰又將打響了！
