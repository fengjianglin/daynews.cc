Uzi的最強替補抵達！RNG戰隊遲遲不願官宣，選手親自改頭像暗示

天天要聞
4 天 ago

小狗的狀態是如今粉絲關注的焦點，畢竟新賽季將要開始，轉會期快要結束，而小狗還沒有宣布自己到底要不要退役，雖然說粉絲還是相信，小狗為了夢想能夠堅持下去的，不過letme說，小狗的狀態的確是已經到了極限，如果自己是真愛粉的話，一定會勸他不要繼續打下去的，而如今RNG戰隊也在做著兩手準備，一方面沒有公布小狗的消息，另一方面也為他買到了一個新的替補。

在轉會期期間有很多的消息，而RNG戰隊的兩個新選手基本上都已經有實捶的證據了，這兩個新選手一個是小龍堡，另外一個是betty，betty是閃電狼之前的ADC，只不過閃電狼的英雄聯盟分部如今已經解散了，畢竟lms解散的號角已經吹響，閃電狼也無法繼續支撐下去，他們的選手和教練都全部離隊了。

所以betty被RNG戰隊買下，只不過戰隊遲遲不願意官宣，想必是不想影響小狗的抉擇，還希望他能夠繼續打下去，如果現在就官宣了新的adc的人選，粉絲肯定會多想的，所以即使這位最強替補已經抵達，卻仍然不能宣布他的新身份，看到戰隊這個樣子，這位選手親自改頭像，把自己的頭像改成了RNG戰隊的標誌。

這位選手也真的是很會暗示了，看到他的這個舉動之後，大部分人都相信了RNG戰隊將會買入一位uzi的替補，而且這位選手實力還是很強的，可以說是閃電狼如今最能打的選手之一了，不知道他能不能成為下一個uzi，獲得粉絲的追捧呢？

不管怎麼樣，小狗肯定是要退役的，希望那一天到得越晚越好，不過粉絲們也需要有心理準備。
