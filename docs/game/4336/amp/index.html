在歐美種族歧視是絕對不可以觸碰的話題，這已經是一種政治正確，尤其是對於黑人！最近開發商呵呵魚工作室（Chucklefish）旗下戰棋策略遊戲《戰紋（Wargroove）》就因為遊戲角色聲優引發了爭議，最終呵呵魚也選擇了道歉！

《戰紋》是今年2月推出的一款像素風回合制戰棋遊戲，遊戲整體好評率85%為特別好評，而出現爭議的原因就是11月中旬，官方宣布將推出免費更新DLC「Double Trouble」，還加入了四位新角色，其中「Errol」、「Orla」與「Vesper」都是黑人角色，而聲優都是白人，這就引發了一些玩家的抗議，在推特和論壇上批評呵呵魚工作室有種族歧視嫌疑！

這些批評呵呵魚的玩家認為，既然加入黑人角色就應該找黑人聲優來為這些角色配音，這樣才是尊重角色設計。而玩家認為更不妥的是還在公布這些黑人角色的時候配上了白人聲優的照片！

雖然引發了爭議，但遊戲的口碑似乎沒有受到影響，近期還是好評如潮，好評率達到97%，不過在推特上還是不少玩家在爭論，為了不讓爭議延續，呵呵魚工作室也在推特上發布公告，為這一次引發的黑人角色配音爭議致歉。

呵呵魚表示此次遊戲配音是由外部團隊協助，選聲優的時候也是採用「聽試」方式進行，也就是只認聲音不認膚色，對於這次選角還配上照片出現的爭議表示道歉！呵呵魚還承諾未來在選聲優的時候會更加尊重並支持有色人種的表現機會！對於出現這種爭議，大家怎麼看呢？
