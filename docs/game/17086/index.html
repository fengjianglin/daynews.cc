《陰陽師》2400九十勝率上名士，17勝1負爆錘“鯨吞隊”

2019-12-14 1,337條評論 3,501次閱讀 1129人點贊

作者：skygtm

「一支穿雲箭，鯨吞白日來相見」

中午花了兩個小時從2400上到了3000，人生第一次名士好激動呀呀呀呀呀~~~

更讓我激動的是，用「離久白」僥倖以17勝1負的戰績優勝。不得不感謝這個離鯨吞遍地跑的版本

我只是剛好鑽到這個版本的空子。

或者換作以前離久橫行的時代，我也只有挨打的份了

(戰績、對陣截圖在最後)

十月份回歸買的號，閻兵切出身，拼出離後用過離鴆，一直在2200徘徊。

被離久爆錘後決心加入，硬著頭皮在繪卷期間六了久次良和幾個輔助。

師從X站「嚶嚶井」，學來「離久白變陣」後改良(鬥技自閉，幸得秘籍)

最近三周都可以在周末打上2700，打內戰、一速蛇茶都沒問題，但是和鯨吞打始終是46開甚至37

最為討厭的莫過於」帶小白、日和坊的離鯨隊「

」針女花鳥+地藏小白「雖然可以讓隊伍抗下六下齒甲，但久次良始終無法攻破這道」白日防線「，慢慢就會被補甲跳死

最後一手嘗試過各種變陣，大岳丸、閻魔、清姬、網切花、百目鬼等，要麼火量堪憂，要麼作用甚小

思來想去，決定今天用「騎驢小姐姐」試一試。

命中抵抗雙修驢——「吾乃神射手，伺機出手，出手必封，一封皆封，一箭一個木頭人」(主動被動御魂治療全封)

​

原本是用來封茶几的，結果今天打鯨吞出了奇效。於是想嘗試一下看看能不能上3000，結果樂開花了，剛好鑽到了版本空子輕鬆上分。(如果是下個版本估計沒有這般運氣)

鬼吞不推條了，鯨魚不翻身了，啊離不跳舞了，日和坊也不招手了，小白和大蛇更是像個木頭

只要對面白/日被封，點誰誰死，針女花鳥+老實人招財久次良，也可以輕鬆突圍

騎驢射箭偶爾會帶久次良一起跳，概率不高，一局能跳一次算不錯，當個彩蛋看哈哈

：哥，你齒甲沒了快補補啊哥：哥，你推推我嘛哥：哥，你動一動啊哥，一動不動是。。。

(補充：鑒於很多大佬談論運氣的問題，我也補充一下我個人對這個問題的理解吧

很多事情不是非零即一的，驢確實看臉

但是我堅定不移地相信，基數增大的同時，驢的射箭概率還是會和無限趨近於40%的

就拿招財來看，我是不相信有人不帶招財的，招財也就50%，每回合判定一次，每局判定的次數屈指可數，我還不是真香真香地弄了個五六套

只要判定的次數足夠多，容錯的次數也足夠多的，這便成為了一件低風險高回報的事情

如同針女也是40%，只要判定次數夠多，就有」賭「的空間

那麼我們粗劣且保守地假設，驢每局會對是否射箭判定15~25次，出手5~8次低於40%算是可以接受了吧

離鯨鯨這樣機動性和變陣空間相對比較低、無單秒能力、後續手段單一的陣容，驢哪怕每局只出手3次，也能成為勝負關鍵

對面不存在必須要封印的關鍵單位，所以驢的容錯率可以很高

下面的復盤也是以局均兩箭來考慮的，對面6個單位，我只射兩箭，低於40%了吧，不歐了吧，可以討論了吧。

每回合出手兩次，射誰都行。這要求真的很高很高嗎？高到你們都說我是在用壽命來打的？

我也不是喜歡拉出臉來讓別人打的人，也不蠢。原本以為，這個勝率已經可以盡量讓大家不去討論「歐不歐」的問題。

真的不需要歐的，歐的情況是少數，非的情況同樣也是少數。

希望可以理性一點)

驢的優勢在於「以一己之力，剋制對面六個單位」

對比傳統控制式神，優點有幾個：

1.作為尾速不費火

2.不用追著對面的高速的化鯨來控

3.封印周期長，從對方結束行動封印到對面下一次行動。而傳統式神的控制周期是和自身速度掛鉤的。周期越長，對付白藏主的時候越有利

4.可以同時封主動、被動和御魂

在對抗化鯨隊這種不怕控制的隊伍比較合適

在對抗化鯨隊這種後續手段單一的隊伍比較合適

在對抗不帶控制不帶驅散的隊伍比較合適

在對付那些想白日坐地的隊伍比較合適

無白日隊伍，2~3分鐘結束，最快的時候一分鐘，驢一回合射四箭，對面直接退

有白日隊伍，2~8分鐘結束，最長一局16分鐘，等了很多個回合驢才肯射對面日和坊，一直在奶，加上對面地藏不知火難秒

「這絕對不是壽命打法」，看戰績就知道了，如果只是其中一把歐可以理解。但這是連勝這十幾把，只要驢出手兩次以上，就可以打。

關鍵點在於「等候時機」

我下面按情況復盤一下：

(補充：復盤之前先說明一下，這套陣容，在第三回後開始斷火，需要考慮優先給誰開技能。也有大佬指正用蚌精匣子替換小白，這樣既穩妥又省火。下面的復盤是基於斷火的情況，也會說明火不足的情況下技能的取捨)

離鯨鯨白日：

久開局如果不觸發招財就不拉條，把火讓給驢，面對白日，前期搶血意義不大。對面單靠兩條化鯨六下齒甲，在沒推拉的情況下，是沒法讓我方馬上減員的。後續小白不用開技能，火留給驢和離。不用賭臉，驢出手兩次並不過分。

如果封鯨和離，對面後續沒傷害，繼續等機會

如果封日和鯨，對面後續沒傷害，我方壓對面小白血線，日和坊是尾速，我們打兩輪她才解控

如果封白和鯨，對面後續沒傷害，秒離或者鯨都可以，離二狀態的時候小白要開回技能

離鯨鯨吞白：

雖然對面爆發高，但是我方可以神樂疾風拉花鳥，抗住6下齒甲。久開局拉條。不用賭臉，驢出手兩次並不過分。火優先留給小白，以防鬼吞秒人

如果封吞、離、鯨其中一個，對面後續沒有能讓我方減員的傷害，我方高速花鳥有充足的時間來回血。接下來搶對面小白的血，逐個擊破

如果封白，直接秒其中一條鯨，因為吞的被動，對面全員長期處於半血狀態。只要對面白藏主被封，或者沒火開大，我方隨手就可以秒一個。(我們的是木魅驢，對面都是以普攻出手，很容易沒火)

離鯨鯨吞日：

誰脆秒誰，或者把阿離打出二狀態

離鯨鯨吞控/拉條/岳：

(補充：下面有大佬指正面對這種陣容我方會比較吃虧，確實如此，下面我再紙上談兵說一下我對當前環境的看法。

當下環境大部分人為了避免內戰吃虧，啊離都是帶地藏的，對面基本沒有帶火靈單位，而且我方有木魅。所以還是可以打的。如果對面是火靈久次良，火靈鬼吞，這樣的雙拉隊傷害將大打折扣，能否一波帶走真不好說。而且這樣調整御魂、變陣的雙拉鯨吞隊是否還能有效對抗其他隊伍呢？

另外，配好的擋刀星熊換火靈嗎？清姬換…？高速火靈大岳丸？這些都有待驗證。對面同時是個沒有任何保護的頭鐵隊伍，隨便減員一個就遊戲結束。

期待鯨吞的再一步進化。如果鯨吞不願意為我們這個小眾陣容作出改變，那這個版本的優勢可以吃到底)

有星熊秒吞，沒星熊秒控制。遇到一把大天狗，一把星熊。星熊那把他沒火開大，大天狗那把他被我花鳥針女打死了(尷尬)

有一把對面上尾速久次良，我花鳥觸發他反擊，他反擊後被我驢射了，我一臉蒙蔽，全局原地罰站

離鯨鯨蛇岳/白：

等變蛇就贏了，打了三把，大岳丸放逐我方哪一個都不好使，都剩20%血秒不掉，蛇被封了更是舉步維艱，唯有等待被一一擊殺

事後我又再分析了一下遍地走的「離鯨鯨陣容」

帶鬼吞的陣容

高速+推/拉條，儘快齒甲六次；鬼吞補刀，讓對面迅速減員；然後鬼吞回血穩健取勝。缺點在於沒有控制，沒有驅散，回血周期較長」

那麼只要頂住了一波攻勢，並且限制後續輸出，反打取勝易如反掌

帶白日的陣容

爆發一般，通過補甲慢慢壓倒

抗住第一波爆髮根本沒問題，只要能找到突破白日的辦法，就可以取勝

騎驢只是恰到好處，既可以突破白日防線，又可以減緩對面的攻勢，「以一克百」哈哈哈

我以前玩過一段時間的離鴆鴆，我一直感覺這種double式神的陣容機動性比較一般

化鯨優於鴆的其中一個點是「即使被控了也可以輸出」，但是同樣是多佔了一個位置，可以變陣的空間也變小了

這種不帶控制不帶驅散的隊伍太適合「雙修驢」上場了

我只是剛好鑽到這個版本的空子。

倘若沒有這一箭，或許我這周連2700都回不去

或者換作以前離久橫行的時代，我也只有挨打的份了

離久白今日射出了這一箭，期待化鯨隊接下來的回應

式神面板

這裡對式神面板補充說明一下

現在這套面板還可以優化

1.把啊離的速度提3點以上，啊離第一回合比驢先出手，剛好回火給驢開技能

2.把小白的速度降4點左右，花鳥第三回合超小白車，剛好回火給小白開技能

3.之後的回合會斷火，需要考慮優先開誰的技能。上文有寫，對陣不同陣容怎麼取捨的問題

4.小白用蚌精更優秀，下面有大佬指正。或者直接換成蚌精匣子。蚌精匣子的優勢在於開局不費火，而且不會像小白這樣為隊伍承傷過多而暴斃。中後速匣子可以讓花鳥不斷超車來為隊伍回火，而高速匣子會讓久次良多協站幾次。哪個更優有待各位驗證，畢竟要根據御魂的強度來。

5.御魂不好，驢迫於無奈六號位帶了攻擊加成，帶「生命生」更優。

這個思路很多有先見之明的大佬之前已經在研究了，我的配置還是比較粗糙的，有待優化。不敢作為攻略，只能算是一個小小分享。

我這裡算是拋磚引玉，找到適合自己的才是最好的。
