DNF：一陣雨舉辦打樁比賽，各大跨區前三誕生，劍魂成為贏家

2019-12-17 2,619條評論 4,252次閱讀 3220人點贊

在國服普雷畢業之際，一陣雨和大碩兩人，聯合舉辦了打樁比賽，各大跨區前三名誕生。不過，也被稱為「氪金比賽」，畢竟能打出超高傷害的，裝備都打造的十分強勢。其中，旭旭寶寶也參加了，紅眼突破1700E傷害。只可惜，狂人未能參賽，否則以劍魂的實力，妥妥的第一。玩家也發現，此次劍魂上榜人數居多，無疑成為了贏家！

在跨一中，最強的是一名紅眼「鹹魚娜娜」，紅14和15散搭，20秒綠沙袋打出1160E的傷害。其次是召喚，但穿的是布甲，倘若輕甲的話，「鹹魚娜娜」就會被超越！布甲比較「拉稀」，傷害力度不高，而且，跟第一名差距的傷害，也就一點點而已。第三名是劍魂，裝備紅13打造程度。狂人作為第一劍魂，也身在跨一大區，但這次打樁比賽中，並沒有出現。跨二最強的是劍魂，紅14裝備打造程度，打樁傷害高達1272E，而第二名也是劍魂。值得一提的是，特工排在了第三，且紅12打造程度！

​

跨三分為「A」和「B」兩個大區，跨三A整體實力較弱，第一名紅13和14散搭劍魂，不過，跨三B有「流年」鬼泣，紅18的聖耀短劍，在搭配14的裝備，雖然是34C職業，但實力非常強勢，打出了1242E的傷害，穩居這個大區第一名！

​

比較有趣的是，跨四大區的第一名，是稀有職業藍拳，全身紅14打造，綠沙袋突破1238E！國服遊戲中，這個職業玩的非常少，能脫穎而出，成為跨四第一名，刷新了對藍拳的認知，隱藏大神玩家頗多！

​

論神豪最多的，莫過於是跨五和跨六，簡直是「神仙打架」！跨五包含「江浙滬」地區，土豪玩家較多，前三名的職業，都是紅14打造。劍帝作為幻神，有這麼強的裝備加持，自然傷害飆升，打樁突破1500E。而跨六主播多，旭旭寶寶紅16起步的大紅神，傷害十分爆炸，打出了1700E左右。「亡誓」雖然脫坑了，但其號在別人手中，打樁比賽奪得第二，而第三是一名紅14的戰法！由此可見，跨五和跨六大區，才是真正的「豪區」。

​

跨七和跨八，玩的人不多，屬於典型的「鬼區」，被整合到一起，首次有阿修羅上榜，全身紅14打造，打樁傷害突破1112E。阿修羅比較弱勢，能在打樁比賽中脫穎而出，也出乎玩家的意料！

​

總之，這次舉辦的打樁比賽，以普雷畢業為基礎，涵蓋國服各個大區，將會決出最強的「打樁王」。不過，旭旭寶寶參賽的話，第一紅眼裝備打造太強，「打樁王」怕是已經被內定！
