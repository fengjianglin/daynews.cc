陰陽師：加強後也無人問津的三位SSR式神 都是茶几和大舅媽的錯

2019-12-15 2,838條評論 4,761次閱讀 2675人點贊

陰陽師對於老牌SSR的加強，官方可以說一直處於非常保守的狀態，近期獲得不錯加強幅度的SSR式神一共有三位，可惜加強後也無人問津，可以說都是茶几和大舅媽的錯，來看看是怎麼回事吧。

妖刀姬

正式服已經實裝的加強，但依舊不能改變現版本對於妖刀姬的評價，確實強度提升了，從過去的倉管變成了偶爾可以用的特化型輸出，妖刀姬現在依舊是多段輸出式神中最穩定的，因為6段傷害靠譜，而且攻擊速度的問題也通過姿度問題解決了，但根本問題沒有解決。還是因為現版本多段輸出式神有更好的選擇，相對於穩定輸出不會帶來驚喜的式神，玩家更願意選擇和運用上下限很高的式神，這也是為什麼開服時候會有猜拳茨木。所以妖刀姬就算是解決了攻速硬傷問題，也依舊無人問津，因為後來居上的多段式神已經不是短短一個攻擊時間的縮短就能彌補的差距了。

面靈氣

官方對於面靈氣的面具機制進行了大幅度的改動，這個經過分析後發現是非常靠譜的加強，可惜這個加強效果僅僅只是針對面靈氣的PVE加強，其增益BUFF效果還有慘劇輸出能力得到了非常不錯的提升。但是可惜現在PVE是茶几、大舅媽和兩面佛的天下，面靈氣就算是得到了加強，論AOE輸出能力不如茶几兩面佛，論對單體核心輸出能力比不過大舅媽，一個單純的增傷加強改變不了面靈氣的地位。如果那個加速BUG保留的話，那可能面靈氣還能掀起大環境變化，現在的話，正式服實裝後也不會帶來更好的效果。

山風

體驗服改動了山風的普攻設定，原本山風的普攻就是廢物，因為山風不用堆暴擊，但是改動後山風明顯是奔著最強收尾刀式神的定位去打造和加強的，然而山風的定位還是太單一了，導致PVP很難有上場空間，然後就是魂土當收尾刀式神，完全看燼能不能穩定收掉兩隻鬼切，這導致所有單體不敢輕易說能當穩定收尾刀式神，這讓加強後的山風也異常尷尬，所以以前該怎麼打還是怎麼打吧，山風算是一種另外的解法或者奇策，只能說老牌SSR式神無人權，都是茶几和大舅媽的錯。
