　　這些跟其他魚類不一樣在哪：

　　1、變色魚塘（4種）2、夜市魚（3種）3、洞穴魚（4種）4、特殊產出的魚類（2種、鱘魚和魷魚）5、不是「魚」但又能放魚塘（3種、螃蟹、珊瑚、海膽）6、秘密森林特有魚類（木躍魚）7、不能繁殖的魚（虎紋鱒魚）
