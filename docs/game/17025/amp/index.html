燃燒意志：100級沙鱷魚技能加點，名曰恐怖輸出，實為掛件之王

天天要聞
4 天 ago

諸君安好，霧夏菌報道。

這一回合要重置的是沙鱷魚的100級技能加點。早年在七武海版本中，老沙可不普通，而是掛件中的帝王，而在100級版本中，沙鱷魚更是承襲了這一優點，依舊是個優秀的掛件。

1、​角色介紹

老沙的定位是團隊壓制，前排優先。

在官方的描述中，老沙擁有著恐怖的範圍傷害，實際上也的確如此，但可惜的是，這貨身板脆，不扛揍，關鍵速度還慢，基本上輪不到自己出手，便被別人順手宰了。

後來有著一波加強，變成了堂堂正正的掛件一枚，並在之後的80版本以及現在的100級版本中，這一特性被無限地發揚光大，成為了掛件行業的霸主，也算是風頭無二了。

2、技能選擇

​技能：沙塵暴、化沙、沙漠寶刀、奧義

備用：沙漠金剛寶刀

一個是直線傷害，一個是單點傷害。PVP可以將沙漠寶刀換成金剛寶刀，開局放個化沙提升自身減傷，緊接著金剛寶刀來一波全體AOE補刀，且附加概率流血，然後等死就可以了。

老沙最大的優點，就是人死BUFF在，可以給七武海提供暴擊、暴傷BUFF效果。60級天賦開局降低敵方命中率、閃避率。此外新增的100級天賦，老沙又可以每回合清除我方全員一層減益效果，掛件之王，不負眾望。

3、​技能加點

1~10級：

幾個主動技能大都點到16或者17級，老沙的技能狀態很多，所以技能點並不充足。如果不點那個減傷技化沙，就可以將其他的技能全部點滿，根據自己的實際需求調整，這裡不再多說。

20~40級：

這裡的話，終結技新月形沙丘不用，主打一個恐懼效果，傷害也還可以，但面對諸多防禦力超高的敵人時，反而不如主動技沙塵暴來得划算。看個人喜好吧，如果自己覺得可行，那就用這個技能。沙漠金剛寶刀點16級，另一個終結技沙漠風暴點5級。

50~100級：

沙漠風暴的狀態點3級，原本這個技能還是很不錯，可惜開狀態7魂玉，不好釋放。其他的被動、天賦全部點滿即可。100的天賦就是上面提到的，每回合清除我方全員一層減益效果。100級的被動是暴擊傷害加成。

·菌說

老沙的機制同樣很複雜，要說打傷害吧，他還真的能刷出來一些，但自身保命差，速度慢是個無法抹去的硬傷，加上天賦偏於團隊輔助，所以目前的情況下，還是用來當掛件或者助戰更合適。

除非個人喜歡，否則不推薦用來當做主力輸出。
