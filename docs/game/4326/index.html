王者榮耀：這五個英雄前期忍辱負重，發育起來，後期到處找人復仇

2019-12-08 0條評論 19次閱讀 0人點贊

摘要

王者榮耀真的是一個挺神奇的遊戲，因為王者中的許多英雄，前期弱的誰都能欺負他，但是一旦被他發育到後期，裝備起來了，那就是到了他的復仇時刻了。這類英雄也基本就是後期英雄了，前期弱成狗，後期猛如虎。

妲己

前期經濟不夠，導致妲己的續航能力不強，經常被韓信、孫悟空、阿珂等高爆發英雄瞬秒，但後期的妲己那是真惹不起，一套秒脆皮，一套能讓坦克丟掉半血。妲己作為王者榮耀最初始的英雄，不管是高端局還是低端局，遇見她，一定要躲得遠遠的，等到妲己CD冷卻時，在來個出其不意，瞬秒妲己。

伽羅

伽羅前期也是很憋屈的一個英雄，雖然手長但是傷害確實不高，而且由於自身沒有任何位移和穩定控制技能，所以很容易被針對成為敵方的移動取款機，但如果熬到了後期那就是伽羅的天下了，超遠程普攻加上刀刀暴擊幾乎沒人扛得住，一個人足以統治戰場，追著你打想跑都難！

鎧

鎧這個英雄，前期抗壓能力極強，擁有一技能的回血能力，以及二技能的快速清兵，能夠在上路保持一個較好的狀態守塔。一旦到達四級，便是翻身農奴做主人，就可以嘗試開大強行單殺對方射手，如果讓其裝備成型，便化身峽谷砍王，拎著40米長的大砍刀，閃現追著對面後排，幾刀暴擊下去後排瞬間蒸發，這個英雄到了後期是個狠人，能成為對方後排聞風喪膽的存在。

曹操

曹操4級之前，非常脆弱，一技能雖然有三段位移，但是太短了，稍微厚一點的牆都穿不過，二技能就更不需要說了，就一個簡單的減速而已，最主要的還是大招了（解控、吸血、回血），前期猥瑣出些肉，後期配合大招簡直是無敵的存在，昨天打了一把，三個人都沒有把一個半肉曹操抓死，還被反殺了一個，可想有多「變態」。

典韋

​典韋在S12賽季時有著站擼王的稱號，雖說後面削弱了一些，但是憑著自己的被動，還是可以打出很不錯的傷害來，前期還是需要發育吧，由於典韋的被動可以疊加攻擊力，所以典韋出半肉裝更好，因為這樣有傷害也能抗一定的傷害，但是前期還是需要猥瑣發育，疊滿被動後就瘋狂的輸出吧！

其實，後期型的英雄還是很多的，我就不一一闡述了，那麼剩下的就交給可愛的你們了。最後也希望大家能夠多多支持我，謝謝
