揭開國風電競大戲序幕，看大話西遊2如何引爆國風文化新篇章

2019-12-07 0條評論 9次閱讀 0人點贊

摘要

“這是一場國風碰撞，也是新時代命題下，電子競技的全新嘗試”。在剛剛結束的《大話西遊2》冬季邀請賽上，洛陽市文化廣電和旅遊局古付先先生為比賽致辭時，面對國風與電子競技在洛陽這一文化古都產生碰撞與全新的融合時如是說。

「這是一場國風碰撞，也是新時代命題下，電子競技的全新嘗試」。在剛剛結束的《大話西遊2》冬季邀請賽上，洛陽市文化廣電和旅遊局古付先先生為比賽致辭時，面對國風與電子競技在洛陽這一文化古都產生碰撞與全新的融合時如是說。

十七年來，《大話西遊2》可謂一邊守護著鍾情於這片江湖的玩家共同成長，另一邊則始終守護著國風文化的初心，在前行的道路上尋求著屬於自己的創新與突破。在電子競技如火如荼的今天，如何更好地讓遊戲本身的特點與國風文化相結合，又成為了擺在《大話西遊2》面前的全新課題。

不積跬步，無以至千里

在正式回顧《大話西遊2》冬季邀請賽之前，請先與我們來看看洛陽是如何與《大話西遊2》吹響國風電競發展這一聲號角的。

作為見證了十多個王朝變遷的古都洛陽，無數文化、經濟、歷史、人文等等元素縱橫交織在這個曾經匯聚了無數文人墨客與江湖豪俠的古城之中，當一場場精彩的電競比賽在這座城市中盡情地釋放著熱情與青春的時候，就如同古付先先生在致辭中說的那樣：「很高興，今天能站在這場國風創新的現場，與眾多《大話西遊2》玩家一起感受『國風電競』的熱情，和《大話西遊2》一起為新時代命題邁出屬於我們的一小步。」

是的，亦如此次冬季邀請賽的主要舉辦地洛陽上陽宮文化園，作為建於唐代上陽宮遺址上的公園景觀帶，其定位於呈現盛唐文化背景，注重傳承創新、倡導融合發展的理念，正好契合了《大話西遊2》一直以來強調的傳承國風，打造盛唐繁榮勝景的初衷。

而為了更好地突出洛陽上陽宮文化園一直以來呈現出的獨特氣質與《大話西遊2》在電子競技比賽上的融合與交流，大話還特地邀請了北京奧運會表演隊——石信鼓樂團為現場的玩家帶來了國風節目《盛世鼓舞》，藉助世界非物質文化遺產中國大鼓這種脫胎於唐代燕月融合宮廷與民間音樂文化的大型民間樂種，用陣陣激昂奮進的鼓聲鼓舞參賽選手的同時，也奏響了此次冬季邀請賽的最強音。

不積小流，無以成江海

如何將國風與電子競技更好地融合在一起，選擇歷史悠久的洛陽舉辦一場所有玩家都關注的賽事，僅僅只是《大話西遊2》邁出的第一步，選取整個洛陽所特有的文化產業中的精華，使之成為冬季邀請賽中的點睛之筆也就此成為了此次活動中的另一大亮點。

除了一曲振奮人心的中國大鼓節目激發了玩家與電競選手們對於冬季邀請賽的熱情之外，《大話西遊2》還特別地選取了洛陽當地最具特色的傳統剪紙手工藝品作為受邀參賽的明星戰隊參賽選手的伴手禮。為此，網易特別邀請了洛陽當地資深的剪紙匠人，藉助洛陽剪紙的繁刻技法所講求的結構講究、精巧秀美等特點，讓充滿著霸氣與王者風範的年獸悅然紙上，也讓參賽選手們充分領略到了洛陽剪紙的精妙之處。

剪紙雖小，但它的背後是洛陽傳承中華文化的一個濃重的縮影，選擇將洛陽剪紙作為伴手禮中最為貴重的禮物之一送給參賽選手，更多的不僅僅是獎勵他們在比賽中的努力、堅持與戰鬥不止的電競品格，更重要的是《大話西遊2》多年以來堅持傳揚國風文化，堅守傳統之美的初衷，也與洛陽對經典與傳統文化追尋與傳承的精神不謀而合。

這種堅守與積累，不僅僅只體現在《大話西遊2》的遊戲文化當中，也體現在多年遊戲發展的大背景下，成為了《大話西遊2》探索電子競技市場發展當中的一個關鍵因素。在電子競技成為全民關注甚至上升成為國家大力支持的新興產業的大背景之下，作為一款運營了十七年的經典網遊，《大話西遊2》在電競市場的布局與耕耘是有目共睹的。無論是從天梯巔峰賽、無差別PK賽、比武大會等歷屆大話2已經舉辦過的電競大賽，還是今天我們所看到的這場與文化古都之間碰撞與融合所誕生的《大話西遊2》冬季邀請賽，都足以看出《大話西遊2》在不斷探索電子競技新方向與新思路時不斷探索與積累的嘗試和努力。

傳承國風，永無止境

這種努力與嘗試，在《大話西遊2》過往的發展歷程中幾乎貫穿了遊戲成長道路上的每一個重要節點。讓我們把鏡頭再拉得遠一點，看看《大話西遊2》這些年在傳承國風與展現遊戲魅力時的所做所想吧。

2018年，《大話西遊2》推出了一部名為《大唐國色篇》的紀錄片，將粵綉這一傳統刺繡工藝的獨特之美讓更多玩家熟知;2019年，《大話西遊2》與西安碑林博物館開啟了「漢字復興」計劃，通過「文化記錄、遊戲體驗和新文創」的形式將漢字書法的魅力呈現在玩家面前;而藉助皮影戲、京劇、風箏所獨有的文化呈現方式和魅力，《大話西遊2》將遊戲中的一些玩法內容提供給玩家的同時，也讓他們充分領略了這些中華傳統文化的特別之處;包括像與成都博物館、敦煌博物館進行深度合作，更加凸顯了《大話西遊2》立足遊戲，傳播文化的初心。

在一座文化古都的注視之下，《大話西遊2》冬季邀請賽的大幕已漸漸落下。賽事的成功舉辦、國風文化與電子競技的不斷碰撞與融合，勢必將持續推動《大話西遊2》堅持打造「國風電競」體系與文化的理念，而來自於市場和玩家們的積極反饋也將激勵他們在對中華傳統文化傳承與探索的基礎上，堅持那顆講求遊戲創新與不斷進步的決心。

十七年來，網易《大話西遊2》透過遊戲理念上的設計，始終將弘揚與傳承國風文化放在玩家體驗的核心位置上，不斷通過遊戲玩法上的進化與推陳出新，讓國風更好地融入到玩家線上遊戲與線下體驗活動當中去，為他們營造出一個更有互動感也更具體驗感的遊戲人生。在電子競技愈演愈烈的當下，國風文化與電競潮流的有機結合，不僅為電競圈的未來發展提供一個全新的思路，也勢必將會引爆國風文化傳承與創新的新篇章。
