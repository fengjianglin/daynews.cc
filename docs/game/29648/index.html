《爐石傳說》酒館戰棋名詞術語新手攻略，最強的必定是刮痧

2019-12-25 2,742條評論 4,236次閱讀 1701人點贊

信聖光者娼，不信聖光者不如娼。

歡迎來到聖盾戰棋。

《爐石傳說》的酒館戰棋上線這麼久以來，可以說是幫爐石立了大功，拉回了許多已經遠走他方的玩家。

對於剛剛接觸爐石又不太熟悉、套牌不齊全在天梯被捏著揍的玩家，酒館戰棋會更親民。不需要自己湊卡，不需要自己配置套牌。不會在湊卡上走彎路，不用拼出自己看著都愚蠢的套牌。

但是，不管這個模式的入門有多麼的簡單，行話、黑話、套路這三件事是必須要了解的。否則拿著同樣的牌庫、同樣的英雄，打出了反向操作，那可就要一頭霧水的捶胸頓足了。

沒有套路的英雄都是速八的命。

《爐石傳說の酒館戰棋》遊戲行話術語解釋：

一、理財☆☆☆☆☆

前期選擇購買具有召喚功能的戰吼卡牌。

理財卡牌都是1☆卡牌：【雄斑虎】＆【魚人潮獵者】

酒館裡僅有的兩張理財卡【雄斑虎】＆【魚人潮獵者】

理財的意思很簡單，就是投資：

這兩張卡召喚出來的隨從，是可以賣掉的！

在前期選擇理財卡，後期需要的時候賣掉召喚出來的雌老虎或魚人以靈活調整金幣數量。

比如在第三回合，如果你在前兩個回合有放置這兩章理財卡。

那麼在這個回合你可以選擇購買一個隨從→賣掉雌老虎→再買一個隨從。

這樣你就可以在第三回合最少擁有3個隨從，此時戰力將大大提升。

二、打工(過渡)☆☆☆☆☆

在前幾個回合沒有合適卡牌的情況下選擇的卡牌，到了中期再酌情選擇賣掉。

很多新手會在前期就執著於流派選擇，其實在前期洗牌不順利，不太容易形成流派的時候，最好的選擇就是「只選最大的！」

這一個回合的混搭，是為了不掉血。

下一個回合的變化，就有可能形成流派、套路

魯迅曾經說過：先擠進前四，再談吃雞！

活著比什麼都重要，只要血多，才有機會擠進前四。

三、搶星☆☆☆☆☆

搶星的意思是：

在場面上刷出第三張相同的卡，合體之後不扔出，先升級酒館再扔出。此時獲得的獎勵卡會是升級後更高一層的卡。

如：四星酒館時☆☆☆☆，合體了食腐土狼。升級酒館至五星☆☆☆☆☆，扔出土狼。獲得六星☆☆☆☆☆☆免費卡，開出六星隨從。

這一步操作在中期尤為關鍵，很有可能瞬間逆轉你的頹勢，將你送上前四。

永遠不要急著操作，多想一秒都有可能反殺。

四、鎖血☆☆☆☆☆☆

陣容各要素已經完備成型，拳打敬老院，腳踢幼兒園，不再掉血了，每個回合都取勝。

五、出血、入血☆☆☆☆☆☆

指這一下掉了多少血量，例：第一回合選擇2/4惡魔沒毛病，與其出2點血，不如拼2點血。還能穩住第二回合，直接升2☆。

六、狂戰斧☆☆☆☆☆☆

就是會分裂攻擊的隨從，在酒館戰棋里具備這個功能的就是多頭蛇和死神機器人了。名字由來是dota里的一把分裂攻擊的斧子【狂戰斧】，攻擊堆上去之後，一出手就是砍飛一片。

很多新手在初期會誤會成某種高端的套路，其實只是個稱呼。

七、速八☆☆☆☆☆☆

就是第八名，第一個被打爆的玩家。

被淘汰是很正常的事，心態要好。

酒館戰棋可是一個運氣第一、套路第二、技術第三的遊戲。

八、獵馬人☆☆☆☆☆☆

其實「獵媽人」。

斯里維克的攻擊被動使其可以無視嘲諷，搶先擼掉躲在後排一直+4+4+4+4的熊媽媽或者惡魔噴子，因此大家就稱它為「獵媽人」。

因為本身無種族，所以上場機會其實不多，只有針對個別陣容才會選擇。

九、吃雞☆☆☆☆☆☆

是的，就是絕地求生的那個吃雞，就是那麼沒有想像力。百人淘汰生存最後獲勝的那位即為”吃雞”。

酒館戰棋在設定條件上也差不多，都是拾拾取取湊成一套，最終獲勝的第一名稱作「吃雞」。

真的真的沒有想像力，吃點別的不行嗎？

十、.工具人☆☆☆☆☆☆

指那些只是為了扔個戰吼或者搏個三連合體獎勵而購買的隨從。

此類隨從一般都比較弱，但是擁有增加攻擊血量等強力的功能。一般扔完戰吼之後不會留在場上佔位置，會直接賣掉。

湊三連是為了搏三連之後的更高級免費卡，換算下來怎麼都是賺的，絕對不虧。湊完三連之後可以直接選擇賣掉，不要動感情。

十一、恰爛分(吃爛分)☆☆☆☆☆☆

「恰」是南方大部分省份方言「吃」的讀音，「吃爛分」表達的是吃雞而不得，儘力挨進前四的意思。

開局不順利，能預見自己無法吃雞甚至即將暴斃的時候，就應該選擇不符合套路但暫時能增加戰鬥力的路線。儘可能的保住血線，撐到決賽圈。只要不扣分，一切都好說。

合理的取捨，有時會帶來新的轉機。

連七八糟的打法，但撐進了決賽圈。

最後反而怒懟聖盾魚人取勝，成功吃雞。合理的取捨，有時會帶來新的轉機。

合理的取捨，有時會帶來新的轉機。

項羽曾經對李白說過：先擠進前四，再談吃雞！

活著比什麼都重要，只要血多，才有機會擠進前四。

十二、攀科技(攀星)☆☆☆☆☆☆

在前面幾局縮衣減食，儘可能的升級酒館，最後因為隨從過少，過於脆皮，最終導致暴斃的行為，稱為攀科技。

這是一個天胡才能為之的做法，因為攀科技的花費通常是2隻隨從的費用。過早的升科技會導致花費多大而戰力空虛，一但碰上對方有高級隨從存活，那一拳下去，天崩地裂無法翻身的可能性很大，甚至可能出現直接暴斃速八。

這個時候，你也許會需要一張「大屁股」….

但這並不會有什麼L用，「等死吧，凡人！」

十三、葉師傅、葉問☆☆☆☆☆☆

政治正確的俗爛港片《葉問》里主角有一句台詞：「我要打十個！」。

所以葉師傅這個詞一般是指強力的隨從一個單挑全場的情況。

而能做到這個壯舉的，只有安尼赫蘭戰場軍官和鈷制衛士。

安尼赫蘭戰場軍官就是十二項里提到的大屁股，搭配銅須的雙戰吼，有時確實可以做到一個站擼十個。

這個要求對方不是聖盾劇毒魚人流。玩聖盾劇毒魚人流的都是弟弟！！

呸！

而鈷制衛士則是通過其他隨從的亡語召喚更多的機器人來不停產生聖盾，從而做到一個打十個。

這個對位置擺放的藝術造詣要求很高。

用好了毀天滅地，用不好瞬間化為塵土。

十三、上當了☆☆☆☆☆☆

是的爐石傳說這個遊戲是個運氣排頭的遊戲，酒館戰棋里潛伏著許多許多的騙局，比如對子騙局、光牙騙局、轉型騙局、刮痧騙局等。

比如最喪的對子騙局，每次酒館老闆都給我叫來兩個一樣的隨從，而我又對自己的運氣有迷之自信，小手一抬就把兩個完全打亂自己布局的隨從買了下來，然後就沒有然後了。

第三個它再也不會出現了，大概是厭倦了當第三者吧，它再也沒有出現過…

愛情里都是騙子。

十四、刮痧☆☆☆☆☆☆

這是他獨有的流派。

刮痧流是視覺效果極佳的打法，也是前中期最強大的打法。

刮痧之意是有傷卻不傷身，打法簡述為：

酒館只剩一個隨從的時候，大量賣掉手上的隨從，通過被動給該隨從疊加攻血。

選擇一個喜歡的隨從，不購買並將其凍結。

將其身邊的其他隨從買光，手中囤積大量無用隨從。

等到酒館只剩該隨從之後，將手上的隨從全部賣光，通過英雄被動技能給該隨從疊加屬性。

因為技能描述是給2個隨從增加屬性，但如果只剩一個隨從，那該隨從是會獲得兩次屬性哦。

此法很容易製造出10+/10+的隨從。推薦選擇具有聖盾的隨從來疊加，效果更佳味道更好！

前期推薦隨從為1級酒館裡的1/1聖盾和2級酒館裡的2/2聖盾。

中期推薦隨從為3級酒館裡的6/3鈷制衛士、吵吵模組、量產機器人、鬧鬧機器人。

另外牆裂推薦多頭蛇，疊起來之後簡直就是葉問plus！

最適合刮痧的流派就是機械聖盾流，尋得伯瓦爾之後基本鎖血。

十五、套娃☆☆☆☆☆☆

無窮無盡的亡語，無窮無盡的奇葩死法。

套娃顧名思義就是猶如俄羅斯套娃一般源源不斷，打掉一個又通過亡語刷出來一堆。

最終把對手耗死。

套娃流就是亡語流，次流派不用考慮種族問題，只要有亡語，就能抓來用。

其中6☆巨蟒和和6☆伐木機搭配瑞文戴爾男爵，從而遠遠不斷的召喚隨從，這是一個充滿奇蹟的打法，分分鐘有可能打出超過30點傷害。

總結：爐石的酒館戰棋是一個運氣第一、套路第二、技術第三的遊戲。

想要玩的好，就要靈活思考，不要鑽進流派的牛角尖里去。懂得取捨，方能吃雞！

李白曾經對比爾蓋茨轉告過項羽的話：「先留在前四，再談吃雞！」

活著比什麼都重要，只要血多，才有機會擠進前四。
