《明日之後》“小金剛”即將賣斷貨，玩家入手被制止，一語驚醒

2019-12-11 1,988條評論 2,966次閱讀 1165人點贊

《明日之後》玩得久了，每次更新的內容總是在意料之中，又在意料之外。意料之中指的是各種傢具、時裝、集訓任務等，意料之外則與氪金直接聯繫，因為你永遠不知道，網易下一個氪金點是什麼。第二季迎來副武器系統、配方鑽研升星，又出了個鐵拳金剛掛飾，將「彩虹屁」市場小小地影響了一波，誰叫「彩虹屁」價格居高不下呢，很多人就退而求其次選購了「小金剛」。那麼當「小金剛」價格看似降無可降時，是否到了該入手的時候呢？來看看玩家怎麼做的吧~

不同區鐵拳金剛標記的價格也不同，據沫沫調查發現，掛得最便宜的區價格是48W，最貴的則在126W，沫沫所在區不高不低，售價72W，本來一開始最低價是56W的，沫沫想再討點便宜，期待著能降到50W以下，結果等了幾天，市場上的「小金剛」已經快賣斷貨了，價格也提高了，頓時腸子都悔青了。跟沫沫同樣後悔的還有一名玩家，看看到「小金剛」價格降無可降後，決定耗盡畢生積蓄購買，結果在入手前被營友制止了，為什麼呢？請接著往下看~

這名營友認為，網易之所以在一個月不到的時間上架兩款掛飾，只因為在「彩虹屁」那波嘗到了甜頭，於是趁人打鐵上線了「小金剛」，都是掛飾屬性卻各不相同，但結果是一樣的，都為網易賺足了金。那是不是可以猜測，再過不久遊戲又會出新的掛飾呢？屬性會不會是戰鬥時耐久消耗減免？不說多的，哪怕是5%就已經遠超「小金剛」了，畢竟武器護甲主要消耗是戰鬥時而不是死亡後。

馬上就要迎來聖誕節了，回想去年聖誕節可是非常隆重的呀，導致現在退遊玩家回想當初最熱鬧的時候，腦海里立馬蹦出的就是這個節日，想來網易今年也會好好「布置」，因此很可能又上線第三款掛飾。屬性即使與耐久消耗無關，但也不會比它的兩任「前輩」差，最起碼是平起平坐的，如若沒出，等聖誕過後再入手也不急。不用擔心掛飾絕版，看看噴氣背包，二次返場的時候價格不就跌落過谷底嗎？把握好時機才能盡最大可能節省開支。

看完營友的見解後，沫沫如醍醐灌頂般清醒過來，本來想著再關注一下「小金剛」價格，現在念頭也打消得差不多了。未來還有更好的等著呢， 好鋼用在刀刃上，花錢花在裉節兒上，本就為數不多的金條，更得好好籌劃~大家覺得呢？
