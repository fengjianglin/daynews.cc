三國殺：你敢對本公主無禮！是誰借給你的膽子？

天天要聞
2019-12-15

在三國殺中，很多菜刀武將是及其被玩家們喜愛的，刀刀到肉的爽快感是被很多玩家的喜愛的，並且這些武將的出場率確是也是及其高的，標準菜刀武將的代表就是界馬超和老馬岱了，他們非常依靠殺來進行輸出，另外還有界許褚，白馬等一系列暴力武將，這一類武將都需要合適的牌才能打出極高的輸出，但是這類武將大多都有這一個致命的缺陷，那就是摸牌白。他們很可能連續幾輪都摸不到關鍵的輸出牌，這時候今天的主角——孫魯班。

這是怎麼回事？我們先看技能吧：

譖毀——出牌階段限一次，當你使用【殺】或黑色非延時類錦囊牌指定唯一目標時，你可令可以成為此牌目標的另一名其他角色選擇一項：交給你一張牌並成為此牌的使用者；或成為此牌的額外目標

驕矜——每當你受到男性角色造成的傷害時，你可以棄置一張裝備牌，令傷害值-1

譖毀可以理解成另類版的求援，要麼給牌，要麼成為目標，這一點和皇后的技能機制是十分相似的，並且她和皇后還能打出一張殺3人的效果。不過需要注意的是這個技能在對方給牌後視為對方使用的這個效果，這個效果直接將孫魯班捧上了菜刀武將的最強輔助這一寶座，就拿馬岱來說吧，馬岱的潛襲效果是很強勢的，幾乎所有武將都不願意被馬岱砍一刀，一般在距離內的人更是會留一張閃作為關鍵牌，但是馬岱本身摸牌白的特性使得不能每輪都進行有效的輸出，但是如果有著孫魯班作為輔助就能夠達到每回合2次潛襲的效果，試問誰不怕？

而驕矜這個技能更是能夠給孫魯班帶來回合外的極強生存能力，一般來說能夠造成2傷害以上的武將還是不多的，這也使得裝備牌對於孫魯班來說成為了關鍵牌之一。我們要知道，驕矜並不是鎖定技，受到傷害時要不要發動驕矜是由自己決定。假如你受到的傷害不是致命的，你可以先考慮到你即將棄置的裝備重要嗎？棄置武器、-1馬可能會影響下個回合你的進攻；棄置防具、+1馬又可能會失去防守優勢 ，受到更多的傷害，反而得不償失。

另外關於這個技能還有這一個冷門知識：在被藤甲的火屬性攻擊的時候發動則不會受到一點傷害，這一點可能能救你一命。

適合位置

主公：3星

作為主公孫魯班如果隊友中有著強力的菜刀流武將，那麼孫魯班能夠配合隊友打出機器不俗的輸出，但是由於驕矜的特性容易遭受到選將上的針對。

忠臣：2—4星

作為忠臣的孫魯班是十分依賴主公的選將的，孫魯班作為忠臣不能給主公很大的幫助，屬於下限及低的忠臣武將之一，但是一旦主公是強菜刀武將孫魯班的發揮也會是極其出色的。

內奸：1星

不推薦使用孫魯班作為內奸，那樣你的遊戲體驗會很差，本身沒有極強的場控能力也沒有強大的單挑能力，作為輔助來說可能是一流但是作為內奸實在是不明智。

反賊：4星

作為反賊的孫魯班無疑是能夠給團隊帶來極強輔助的武將之一，甚至能夠讓主公在殺忠臣還是自己被殺中徘徊，這樣的一個反賊絕對是忠臣的噩夢之一！

關鍵牌：裝備、殺、黑色錦囊

最佳隊友：馬岱、馬超、黃月英等一系列吃指定牌的武將

最受克制：賈詡，徐庶等不受錦囊效果影響的武將
