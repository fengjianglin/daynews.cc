王者模擬戰：快攻傷害上調至4，扶桑真傷回調，群雄削弱呂布遭殃

天天要聞
2019-12-18

大家好，12月17日正式服也是進行了更新調整，我們王者模擬戰的調整改動也是有點大的，下面我們一起來看看。

一、天賦調整

1. 快攻的傷害從3上調到4，這也是為了加快遊戲節奏，且在擁有快攻天賦的情況下升級時不再會刷新到快攻天賦，這個改動就對稷下扶桑陣容有點致命了，只能拿到一個快攻了，所以喜歡玩稷下扶桑的同學要注意了哦。

2. 王者威嚴的效果持續時間從4秒下調到3秒，增加沉默效果，王者威嚴也是一個比較冷門的天賦，以前只有繳械效果，持續4秒，現在下調為3秒，並且新增沉默效果也算是變相加強吧。

3. 羈絆進階減少的輸出率從7%上調到8%，不出玩所料，策劃還是對這個神級天賦進行了削弱，直接上調了1%減少的輸出率，但是其實我覺得這個改動是可以忽略不計的，這個天賦依舊很強勢。

二、新增天賦

從上圖我們可以看見策劃新增加了5個天賦，而且都是cp天賦，這明顯是想增加模擬戰中各種各樣的新陣容，相信這次更新之後，又會多出來很多新的騷套路，到時候我會給大家一一更新的。

三、英雄調整

從上圖我們也是可以看見坦克夏侯惇又被加強了，策劃是要鐵了心的加強坦克啊，夏侯惇已經連續加強2次了。除此之外，武則天也是經過上次削弱後迎來了加強調整，上調了大招的傷害，因為上次削弱後的武則天1星實在是太廢了，基本上打不動人，這次上調傷害也應該是想加強1星的時候的作戰能力。然後我們還可以看到公孫離也是被削弱了，這個英雄是很多種陣容的核心，這次直接削弱了她的被動印記傷害，降低了輸出能力。

四、羈絆調整

這次更新對扶桑和群雄2個羈絆進行了調整

1. 扶桑真傷轉化比例從40%上調到45%，這個也是同步上次的體驗服更新數據，回調了扶桑陣營轉化真傷的比例。

2. 群雄1層增加的傷害輸出率從30%下調到25%，這次又對群雄進行了削弱，從一開始的50%削弱到30%再到現在的25%，這都是因為呂布的強度太高了 所以策劃這樣調整也是很有道理的

五、裝備調整

裝備的話就對冰霜長矛的屬性進行了改動，把40攻擊力加成換成了攻速加成，變的更適合射手使用了。

好了，本期內容就到這裡了，喜歡記得點個關注，我們下期見。拜拜！
