王者榮耀：第一印象惹的禍！坦度驚人的他們，在玩家眼中卻是脆皮

2019-12-11 83條評論 1,106次閱讀 137人點贊

Hello，大家好！我是大飛，很高興又見到你。

歷經了四年的風風雨雨，王者榮耀發展至今，已有近百位英雄。這些英雄有的出道即巔峰，有的形同虛設，有的因為「第一印象」不好被玩家所拋棄。

談到「第一印象」，大飛想說「第一印象」對於玩家的選擇，來說非常重要！因為「第一印象」會引導玩家的認知，如果引導正確的話，那麼這些事物就會被玩家很好的利用，如果引導錯誤的話，就會造成玩家認知上的錯誤。

在王者榮耀近百位英雄當中，有著那麼幾位坦度驚人的英雄，因為「第一印象」被玩家認為是脆皮。他們都有誰呢？下面我們一起看一下。

最肉的射手是誰？

如果問哪位射手的生存能力最強？每個玩家的答案不一樣，有的玩家會認為是位移多的射手，有的玩家會認為是物理免疫的虞姬，有的玩家則會認為是肉裝的射手！

對於上面的問題，玩家的答案形形色色，也有有理有據，沒有誰對誰錯！但是，如果問，誰是最肉的射手呢？相信有的玩家答不上來，因為在大多數玩家的「第一印象」當中，射手就是脆皮移動提款機。

那麼，射手當中有坦度驚人的英雄嗎？問題既然提出來了，肯定有！不過，要明白一點射手既是再肉，也不如戰、坦、輔。誰是射手中坦度最高的英雄呢？答案就是黃忠！當然，對於那些不會駕炮的玩家而言，黃忠的坦度是根本體現不出來的。

血量最少的英雄，坦度不高？

英雄的坦度，是否能夠用血量來衡量呢？從理論上來講，英雄的坦度，需要用血量來衡量，但是考慮到遊戲機制的存在，那麼血量就不足以決定英雄的坦度。

就拿王者榮耀中血量最少的英雄嫦娥來說，她就是一位非常肉的法師！然而就是這樣一位英雄，卻因為「法師定位」和「初始血量」，被玩家誤認為是脆皮英雄。

因此，很多玩家錯過了，一位坦度和生存能力極高的法師。這些玩家，有的是不懂「嫦娥」的遊戲機制，不知道嫦娥的坦度有藍量決定，在藍量較少的時候作戰；有的是因為不會出裝，不知道出加藍的防禦裝來強化「嫦娥」。錯誤的使用方式，錯誤的「第一印象」，使得很多玩家，錯過了一位可以輕鬆上分的法坦。

像「嫦娥」一樣的英雄比比皆是，比如說「機制怪獸」曜，比如說可以帶穿一路的「老夫子」。他們的機制，讓「較脆」的他們擁有充足的坦度，但是很多玩家不會利用他們的機制。

續航是否能夠改變英雄的坦度？

看似脆皮實則很肉的英雄，除了黃忠、嫦娥、老夫子、曜以外，還有孫臏和曹操。

孫臏作為一個輔助，在玩家的「第一印象」當中，並不肉！為什麼這裡大飛說他很柔呢？原因很簡單，他的被動續航，能夠讓他一管血當兩管血來使用。

一管血當幾管血使用的英雄，除了孫臏以外，曹操也是一位。曹操是少有的能夠站擼呂布的英雄，為什麼他能扛得住呂布呢？原因很簡單，他的續航保證了他的「坦度」。

目前，大飛研究一種「偽坦」路線的曹操，輸出裝只出末世，其餘裝備全部是肉裝，大家可以嘗試一下。

在王者榮耀當中，因為「第一印象」的原因，很多英雄坦度驚人，卻被玩家誤認為是脆皮，比如說上述的幾位英雄。在王者榮耀當中，還有很多被「第一印象」所「迫害」的英雄，作為觀眾的你，能否說出他們的名字呢？（大飛先來！在大家印象中傷害最低的廉頗，可以秒殺後排，是低端局上分克制射手的神器！）

本文到此結束，謝謝大家的觀看和關注！我是大飛，我們下期再見。
