多款手游斬獲大獎！第五年硬核聯盟黑石獎頒獎典禮拉開帷幕

2019-12-17 924條評論 3,328次閱讀 645人點贊

轉眼間2019年將接近尾聲，遊戲行業內各種獎項活動紛至沓來熱鬧非凡。硬核聯盟黑石獎也如期而至，歷時五屆的黑石獎在每年尾聲都會對遊戲行業的優秀產品進行表彰，同時總結一年以來的經驗與行業同仁分享，為來年的努力方向奠定基礎。

12月6日，一年一度的硬核聯盟黑石獎頒獎典禮在深圳拉開帷幕，硬核聯盟成員vivo、魅族、努比亞、OPPO、華為、聯想、金立、酷派八家廠商代表及大會嘉賓悉數到場。

從2014年成立至今，硬核聯盟一直保持著上升的趨勢，渠道滲透率過半，穩居安卓第一渠道的位置。以強大的安卓渠道分發能力為底，硬核聯盟每年頒發的黑石獎也成為了遊戲行業特別是安卓市場的風向標。

一、重磅大獎名單出爐！「硬核年度人氣遊戲」、「硬核年度最受期待遊戲」備受行業關注

12月6日，一年一度的硬核聯盟黑石獎頒獎典禮在深圳拉開帷幕，vivo、魅族、努比亞、OPPO、華為、聯想、金立、酷派八家國內一線智能手機廠商代表及行業內知名遊戲CP代表出席現場。從2014年成立至今，硬核聯盟一直保持著上升的趨勢，渠道滲透率過半，穩居安卓第一渠道的位置。今年的黑石獎是硬核聯盟成立以來的第5屆頒獎典禮，為表彰對手游生態發展起重要推動作用的各方夥伴，組委會設立了遊戲篇、行業篇、人物篇、明星篇、總共4大篇章、17個獎項。

「硬核年度人氣遊戲」是歷年來最受關注的黑石獎獎項，今年有10款產品贏得此獎，分別是《閃耀暖暖》《明日之後》《一刀傳世》《一拳超人：最強之男》《航海王：燃燒意志》《波克捕魚》《開心消消樂》《崩壞3》《問道手游》和《誅仙》。這些獲獎作品以龐大的用戶群、豐富的內容營銷、極高的用戶活躍度，盤踞各大廠商應用商店及遊戲下載中心Top位。

「硬核年度最受期待遊戲」共有9款產品獲得，包括《萬國覺醒》《哈利波特：魔法覺醒》《家庭教師》等等。這些將要推出的新品部分為大熱IP改編作，上線前就獲得了行業及廣大用戶群體關注，上線後的表現更是值得期待。

明星篇中的硬核月明星遊戲及硬核超級明星遊戲則是分別對應硬核聯盟「明星計劃」和「超明星計劃」設立的獎項。硬核月明星遊戲的獲得者共有9款，包括《狂野飆車9：競速傳說》《量子特攻》《龍珠覺醒》等產品，硬核超級明星遊戲則由4款產品獲得，分別是《閃耀暖暖》《神鵰俠侶2》《新笑傲江湖》及《戰雙帕彌什》。

近年來，硬核聯盟不斷挖掘精品，更是對黑石獎獲獎產品給予高度重視。相信未來，硬核聯盟將對這些優秀作品給予一定程度的資源傾斜，讓優質內容有更多機會、以更多豐富的形式觸達更多的渠道用戶。

二、黑石獎盛況的背後，是硬核聯盟對「明星產品」的追求

從獲得黑石獎的產品名單來看，各獎項得主都是今年表現出色的精品手游。據《2019硬核聯盟白皮書》數據顯示，硬核聯盟目前已經覆蓋了超過7成的熱門精品遊戲。在用戶對「精品」需求快速遞增的市場環境下，能夠囊獲這麼多的精品手游實屬不易。

實際上從2017年至今，更偏向於工具型的傳統分發渠道就已經開始自我變革，除了提供下載服務之外，還在內容、社交方面加大布局力度。硬核聯盟則在此基礎上，以「精品手游」為基準，通過明星計劃、超明星計劃等多個支持計劃，將市場上的精品攬獲囊中，由此形成了優質產品與優質用戶的良性循環。

比如「明星計劃」是硬核聯盟推出的月度扶持評選項目，通過各方評估每月選擇1到2款高品質手游作為當月明星產品，並給予資源、品牌等多種扶持。隨著用戶對遊戲品質的要求不斷提高，硬核聯盟還在「明星計劃」的基礎上推出了「超明星計劃」。

比起產品本身，「超明星計劃」更多聚焦於優質產品背後的團隊。硬核聯盟將與優秀的產品團隊進行深度合作，共同打造明星遊戲。除了給予計劃內的遊戲超S級的資源保障之外，硬核聯盟在品牌和配合上也進行了相應升級，通過渠道與CP的全方位合作，讓更優質的遊戲內容觸達到更多玩家。

比如攬獲「硬核年度人氣遊戲」及「硬核超級明星遊戲」獎項的《閃耀暖暖》，在上線前就頻頻出現在硬核聯盟各大手機商店的推薦位，首日新增用戶突破195萬，為上線後的用戶獲取打下了基礎。

在產品上線之後，硬核聯盟根據各時段的吸量情況，為《閃耀暖暖》進行持續的精準廣告投放，比如線上多種論壇活動及不同風格PR內容、短視頻內容、開機閃屏、動態icon、定製主題、精準push文案、會員定製禮包等，以多種形式觸達用戶。

三、鏈接CP、用戶、廠商，硬核聯盟如何建立完整分發鏈條？

本屆黑石獎也發布了《2019硬核聯盟白皮書》。數據顯示，硬核聯盟在遊戲用戶中的滲透率保持了穩定提升，從2018年的64.4%增長到2019年的65.7%，仍然穩居國內安卓渠道市場「霸主」地位。

要知道，在國內手游市場總體增長趨緩的情況下，硬核聯盟至今仍能保持穩定的持續增長，是十分難得的，讓我們從《2019硬核聯盟白皮書》一窺硬核聯盟增長的「秘訣」。

首先在用戶上，數據顯示截止至2019年10月，硬核聯盟旗下應用商店/遊戲中心活躍用戶總規模達到7.15億。不僅如此，隨著5G時代的到來，預計2020年安卓手機的銷量會進一步提升，為硬核聯盟的用戶量增長提供了支撐作用。

從用戶屬性來看，硬核聯盟的男性用戶佔比過半，並且以25-30歲的青年用戶為主，而這也是目前公認的遊戲認知能力與消費能力較強的人群。由此看來，硬核聯盟覆蓋了絕大部分高質量安卓用戶，而這構成了硬核聯盟分發遊戲產品的基礎。

不僅如此，不同於其他安卓渠道商，硬核聯盟由八大國內一線智能手機廠商組成，從手機硬體轉化而來的用戶群體，解決了遊戲分發過程中的流量真實性問題。

此外，硬核聯盟基於各大廠商資源，更了解手機性能和用戶的使用習慣，能夠提供符合用戶需求的遊戲推薦。硬核渠道在能夠保證內容的豐富性的情況下，用戶更少地跳出去第三方渠道需求內容，自然更夠聚集更多的流量。

最後在CP方面，硬核聯盟自成立以來，合作的CP眾多。比如《神鵰俠侶2》，硬核聯盟為其量身定製公測前一周的精細化預熱方案，取得了全渠道預約量300萬的效果。

憑藉優秀的分析預判和精準投放能力，硬核聯盟也形成了屬於自身的「精細化運營」競爭壁壘，形成鏈接用戶、CP、廠商三者之間的高速分發通道。而對於CP來說，藉助渠道的流量優勢和精細化運營，也能讓自身產品事半功倍地形成有效競爭力。

四、渠道迎來深化變革期，重視用戶、精細化運營將是大勢

在2018年的黑石獎頒獎典禮上，硬核聯盟就表示接下來要發展超級明星、月度明星、獨立遊戲和小遊戲這四大方向，用自身的資源優勢挖掘和扶持更多的優質產品和CP。在今年的黑石獎頒獎典禮上，硬核聯盟表示接下來將發力內容、精品、小遊戲、出海四大方向。

首先在內容方面，硬核聯盟將在原有的應用商店/遊戲中心的基礎上，大力發展資訊、評論、社區等內容生態板塊，在培養用戶質量的同時，也拓展自身內容營銷的發力點。

其次在精品遊戲方面，硬核聯盟將搭建遊戲評測體系，長期挖掘和扶持更多精品遊戲，形成用戶與產品之間的良好發展生態。

再次在小遊戲方面，硬核聯盟將通過快應用標準和小遊戲標準化，為小遊戲開發者提供統一標準，降低開發者的研發成本，方便開發者更好地與硬核聯盟各廠商進行對接，實現雙贏。

最後在遊戲出海方面，硬核聯盟接下來將進一步布局海外市場。目前硬核聯盟各廠商的海外商店已經初現雛形，海外聯盟開發者平台也在進一步優化和調試，未來將助力更多國內開發者走向海外市場。

當然，「精細化運營」仍然是硬核聯盟的核心策略之一。從市場角度來說，硬核聯盟堅持的「精細化運營」策略，實際上也是順應「產品為王」的市場趨勢。經過幾年的發展，手遊行業從增量市場轉入存量市場，在國內手游用戶被培養起來之後，對於「精品」的定義也不再僅是基於研發層面。用戶更看重從研發、到運營到內容推廣等整體體驗。因此，未來的手游渠道推廣必然是整合資源的時代，硬核聯盟也將繼續構建內容生態，並基於大數據提升用戶體驗，不斷自我完善，提升內容營銷分發遊戲的硬核能力。

綜上，重視用戶傳播口碑、加強精細化運營、提升用戶整體體驗將是未來渠道的核心發展方向。硬核聯盟已經著手布局，所獲成績有目共睹，收穫了遊戲生態夥伴的認可和支持。2020讓我們繼續關注硬核聯盟未來發展！
