魔獸世界：冬幕節正式上線，附魔幻化要塞裝飾裝綁坐騎不容錯過

2019-12-16 58條評論 3,553次閱讀 2592人點贊

各位夥伴大家好，我是扶鰩。2019年魔獸世界冬幕節已在今天正式開啟，一直持續到下個月2號，中間的31號還會舉辦一個煙花盛典，成就方面，沒有新的節日成就加入。

1、關於兩大經典附魔幻化；

主城裡面豎起了聖誕樹，但下面還沒有擺放禮物，玩家可以從哪裡接任務，任務完成後會獎勵包裹，被竊的禮物中有概率獲得「薩格拉斯火焰」及「寒冬之握」，前者最早源於周年慶，後者出貨量不愁。

效果就是這樣，左手是薩格拉斯火焰，右手是寒冬之握，感覺還不錯，尤其是薩格拉斯火焰，當你騎乘坐騎飛行的時候，後面會有很長的尾焰，很帥氣。

2、關於裝綁坐騎格嚕普斯的爪牙：

想要入手這隻裝綁坐騎，你必須擁有德拉諾要塞，在NPC哪裡接任務，上交後獎勵「歡樂補給品」，單天最多獲得5個，使用5個歡樂補給品能兌換一個「狂野禮物」，概率開出這個坐騎。

歡樂補給品也可以換點別的，如要塞裝飾，裝飾是永久的，不用擔心節日過後會被系統收回去，一共有5個，分別是彩燈、聖誕樹、小助手、節日盛裝以及老舊裝飾盒。

節日盛裝：為你的要塞守衛穿上應當季的節日盛裝。

小助手：為你的要塞增添一個冬幕節助手以及冬幕歡樂製造器，全稱PX-238冬幕歡樂製造器，站進去會變成一個冬幕節助手的樣子，持續30Min，很好玩的，也是常年有效。

癲狂殺戮戰車控制器：成就「鋼鐵大軍」5個玩具組件之一。

3、關於冬幕節成就

冬幕節是個大節日，魔獸世界千奇百怪的漫長旅行的中重要一環，如果你沒有紫色始祖幼龍坐騎，某些成就就一定得做。

沒記錯的話應該是16個成就，歸屬於千奇百怪的漫長旅行中的共11個，全部做完還給個頭銜，叫歡樂使者，裝等時代，似乎沒人在乎頭銜什麼的。

4、關於冬幕節寵物、玩具

禮物盒開出來的就不一一介紹了，早就爛大街了，這裡說一下「迷你雪人」還有「格嚕嚕」，前者在冬幕節期間，主城會刷新一個迷你聖誕裝飾場地，裡面是零重力，進去直接變地精，很好玩。

寵物「迷你雪人」就是在這個裝置里刷新的，CD是20分鐘，出來後呈瘋狂奔跑狀態！對戰，削弱他，抓住他，就是這麼簡單！他是人形生物，記得對戰時加入1級寵物，別一下子打死他。

寵物「格嚕嚕」是從雪堆里挖出來的，你去做任務的時候，能看到某些地形中有雪堆，能挖出雪球什麼的，概率出這個寵物，可以交易，也就是說，如果你懶，直接走拍賣行就是了。

冬幕節之後就是聖誕節，產出的玩具太多太多了，點擊SH+P，在搜索欄那裡輸入「冬幕節」，有的沒有的就一眼看清了，所有玩具都從禮物盒裡開出，有的可以交易，比如各色沙發墊。

最後要說的是，歡樂補給品是戰網綁定的，如果你想給某一賬號快速湊一套要塞裝飾，用戰網下的不同賬號買來郵寄過去就可以了，不要郵寄重複的，因為一旦郵寄就不能退還了。
