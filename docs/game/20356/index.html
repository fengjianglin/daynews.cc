28人押了15年的寶，成為今年TGA最大黑馬遊戲！明年初就有中文版

2019-12-16 3,234條評論 4,206次閱讀 694人點贊

大家好，我是X博士。

就在最近幾天，被譽為「遊戲界奧斯卡」的TGA頒獎典禮落下了帷幕。隨著硬核魂系作品《只狼》獲得最為重磅的「年度遊戲獎」，宮崎英高也在萬眾矚目下又一次站上了「神壇」！

不過，在這屆TGA上除了那些亮眼的遊戲外，來自世界各地的大小廠商也公布了一大波新作，其中不乏有很多值得玩家們期待的作品，一起和X博士來看看吧！

一、《地獄之刃2》

劃重點：獨立3A遊戲再出續作，明年登錄微軟新主機

在2017年發售的眾多遊戲里，由英國Ninja Theory組建20人團隊、耗時3年的《地獄之刃》或許不是最為出彩的一個，但這款遊戲極具衝擊力的音效設計卻在不少人心目中留下了極為深刻的印象。

在《地獄之刃》里，玩家需要扮演一名受到心靈創傷的維京女戰士塞納進行冒險，在遊戲過程中玩家能夠體會到來自塞納腦海中的一個神秘聲音一直環繞在耳邊低語，而這次的《地獄之刃2》很有可能將延續前作的故事。

雖然從短短兩分鐘的預告片中大家還不能獲取更多信息，但《地獄之刃2》那承接一代之後的故事無疑是富有吸引力的。

據悉，該作將登陸Xbox明年的新主機，帶領玩家再次深入飽受精神疾病困擾的塞納的內心世界。

二、《奧日與精靈意志》

劃重點：美術風格濃郁的2D作品，前作Steam高達96%好評

《奧日與精靈意志》作為曾獲得過50餘項大獎的《奧日與黑暗森林》的續作，自公布之初便受到了玩家們的大量期待。

而在包括本次TGA在內的多次大型活動中，該作都被微軟當做主要宣發對象。

從目前的預告片中玩家們可以看出，《奧日與精靈意志》最為出眾的特點便是奇妙的場景構造以及濃郁的美術風格，據說這個系列的靈感多來源於宮崎駿和迪士尼的動畫。

但是與夢幻的畫面形成強烈反差的，卻是足以令人抓狂的闖關難度，玩家需要用技能和場景中的各種道具，來通過危機四伏的道路。

值得一提的是，《奧日與精靈意志》其實並不是Xbox新主機的護航作品，在不久之後的2月11日，該作便能在Xbox、Win10商店以及Steam平台與玩家們見面了，因此是可以說是近期很值得期待的一部2D作品！

三、《對馬島之魂》

劃重點：有望超越《只狼》的和風佳作

即將到來的2020年是一個遊戲大年，對於索尼的粉絲而言，最為期待的3A大作莫過於兩款：《美國末日2》和《對馬島之魂》。

雖然在TGA上玩家們並沒能得知《美末2》的消息，但《對馬島之魂》卻放出了一段新的演示。

（這個場景怎麼這麼熟悉？！）

雖然《對馬島之魂》和《只狼》同為和風遊戲，但從目前的演示來看前者在遊戲畫面的整體顏色基調上要比後者明亮一些。除此之外，《對馬島之魂》也有相應的潛入暗殺元素以及打鬥效果，給人以無限的期待。

根據TGA上透露的消息，《對馬島之魂》將在2020年夏季發售，相信屆時這款飽受期待的和風作品能讓廣大玩家體會到一份「有別於《只狼》」的獨特感覺。

四、《極樂迪斯科》

劃重點：28人苦心研發15年，終成今年TGA最大的黑馬

作為一款兢兢業業開發十五年的新作，獨立遊戲《極樂迪斯科》一舉斬獲今年TGA四項大獎，成為當之無愧的「最大黑馬」。

該遊戲將黑色偵探小說和傳統的紙筆RPG融為一體，呈現出一款獨特的佳作。

《極樂迪斯科》除了超高的遊戲性外，還對遊戲中每名NPC都有著細緻入微的刻畫，是一款在細節之處極為精湛的遊戲。

正因如此，該作也擁有不亞於3A大作的百萬字文本量，這給該作的譯制方面帶來了很高的難度。

不過，據該作的遊戲開發商ZA/UM宣布，《極樂迪斯科》將在2020年初上架官方中文版本，這也為想要一睹該作風采的玩家提供了便利，因此成為了不少玩家心目中期待已久的佳作。

五、《最終幻想7重製版》

劃重點：這次的「冷飯」真香！

《最終幻想》作為SE旗下的老牌日式RPG系列，其各方面質量自然是毋庸置疑的。

而很早之前就放出的《最終幻想7重製版》，也在本次TGA上放出了一段新的宣傳片。

這段從宣傳片中我們可以看出，《最終幻想7重製版》不僅在畫質和特效方面有著顯著的提升，更是有著出色的人物建模和打鬥效果。

《最終幻想7重製版》將在明年3月3日正式發售，遊戲支持簡體中文，不過PS4平台將會獨佔一年。

課後總結：

除了這些遊戲之外，在2020年中還有更多已知或未知的大作與玩家們見面。相比於2019年的略顯平淡，明年的次時代主機大戰勢必會帶來新的遊戲陣容，屆時將會是一場屬於所有玩家的狂歡盛宴。
