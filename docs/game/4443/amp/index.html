全明星solo第二日的比賽已經開始，昨日我們LPL的幾位選手都成功晉級，今天的對決將會更加激烈與精彩。

而最為讓人矚目的莫過於今天的壓軸對決——uzi與bang的正面solo。這兩人的恩怨我想也不用多說了，就連解說都在瘋狂暗示：「這兩人對決是有名堂的，但是什麼，我不說大家也懂。」此時彈幕節奏早已帶的飛起「世界第一AD之爭？」

而在英雄選擇上，uzi選擇的是傳統AD維魯斯，想必是他覺得兩人都是AD選手，自然也會選用AD來做對決。可是bang卻不那麼想，反而選擇了亞索。英雄一出來，粉絲們開始為uzi擔心，就連解說們的語氣都弱了幾分，因為亞索在與AD英雄solo中優勢實在太大了！

而bang似乎真的很想贏下這場比賽，他深知拖到縮圈自己憑著英雄優勢可以輕鬆獲勝，於是全程都慫在後面補兵，硬拖時間，憑藉風牆可保自己不死。中途uzi顯然也被這種戰術搞了心態，接連中了亞索好幾個風。其中一位解說似乎很看不慣bang這種「分奴」行為，全程也在時不時bang。

到了最後的縮圈階段，uzi退無可退與bang進行生死相搏，此時bang卻犯了了一個巨大失誤，提前釋放了風牆後卻不撤退，與維魯斯正面硬剛起來，而uzi對攻擊距離把控的相當好，沒有被打掉太多血的情況下就輕鬆擊殺bang贏下比賽！

本身uzi獲勝是一件值得開心的事情，而解說卻似乎有些「興奮過頭」：「bang選擇了亞索已經有點吃相難看了！」旁邊的搭檔解說聽完連忙阻止他繼續往下說：「過了過了。」
