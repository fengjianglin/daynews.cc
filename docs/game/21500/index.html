PVP國產手游，比收費視頻網站吃相更難看

2019-12-19 929條評論 3,727次閱讀 742人點贊

最近人民日報炮轟某些視頻網站吃相難看，制定不合理的收費標準。但其實在我看來，人家這種一個月幾十塊的收費真的是很便宜啦。

中國那些手游公司才是正兒八經的銷金窟，喂不飽的餓狗，吃相要多難看有多難看。

不錯，策劃，建模，運營，客服，這些都需要玩家的人民幣來支持，但有的遊戲那種收費模式和效果呈現真的是噁心到了極點。

說實話，中國的遊戲公司垃圾得我不想多說什麼，幾乎沒有像上古捲軸，生化危機，寂靜嶺這種從配樂到故事背景，再到劇情深度都無懈可擊的優良大作。

沒有公司願意去用心的做單機遊戲，最常見的模式就是誘發玩家惡性競爭的PVP。

而PVP手游尤其是坑錢的東西，如果不是為了和朋友一起玩，我也根本懶得碰。

那麼，國產PVP手游為什麼越來越燒錢了呢？

噁心的抽獎模式

SSR，歐皇，非酋這類詞正是在這個基礎上誕生的，那麼，為什麼抽獎模式會被如此多的遊戲商使用呢。

首先 ，抽獎可以模糊遊戲道具成本。比如一個道具如果是300人民幣直接購買的話，那麼每個人心中都會有一個很清晰的界線，這是否在他的預計支出範圍內。

但如果設置成抽獎的模式，變成100到500的價格區間，那麼這其實是在模糊玩家的成本規劃，因為這是所謂看概率的。

很多人都覺得自己是歐皇，就像每個人都覺得意外是發生在別人身上的一樣。但最後還是很多人會被坑一把。

同時，這種完全沒有實際成本的虛擬遊戲道具是永遠不會弄虧的，吃虧的不是遊戲商，而是那些付出更大的代價才搞到同樣道具的其他玩家。

競拍道具

競拍道具是抽獎模式的一個加強版。某些遊戲會將強力道具以競價的模式發布出來，當然這基本也是土豪才有力氣進去摻和的了。但不管怎麼說，吃香相當難看。

不斷推出新道具，暗中修改舊道具

這也是一些吃相難看的遊戲商慣用的手段。

這就是所謂的充錢只能讓你暫時強大，一直充錢才能一直強大。

為了誘使新玩家充錢，遊戲商自然會推出各式各樣的新功能道具，某些屬性自然是可以壓制舊的道具。

最後，如果一些充錢玩家想保持優勢的戰力數據，就得不停更新裝備給遊戲公司送錢。

其實就和銷售一樣，每個公司都會不斷削弱銷售經理吃的老客戶的單子的提成，刺激他們不斷去開發新的客戶。

不斷開發新的氪金項目，最後疊加屬性，做成養成類遊戲

以某款戰艦遊戲為例子。那麼影響最終輸出的因素都有哪些呢。

船是肯定的，其次還有武器，武器有好壞，還有成員，指揮官，配件，寵物，皮膚等等，每一個都是需要玩家大量投入的。不管是金錢還是時間。

這就像CF其實本來做的好好的，但是英雄級武器這種東西出來後我就越來越提不起興趣了。

而PVP做成養成類的，也會導致老玩家不斷充值，而新玩家賬號上的數據和老玩家差異過大，新人都不願意投入。所以是新人不願意投入，老玩家騎虎難下，有的止氪退坑。

而這樣的遊戲，也無疑到了走下坡路的階段。
