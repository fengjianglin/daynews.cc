“吃雞”SS5賽季段位獎勵搶先看，SS5賽季的戰鬥服很好看

2019-12-23 421條評論 3,254次閱讀 1921人點贊

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友刺激哥。雖然，我們在數天前，就在吵著嚷著SS4賽季要結束了。

但這一次，我們要說的是，SS4賽季真的進入尾聲了。之前，刺激哥查看《和平精英》的遊戲社區時，還發現有玩家發帖問：「現在入手SS4的賽季手冊，還來得及嗎？」

針對這種情況，除非您的賽季手冊的等級，已經很高了。否則，真的沒有必要再入手SS4的賽季手冊了。

話題：「吃雞」SS5賽季段位獎勵搶先看，SS5賽季的戰鬥服很好看！

在印象中，好像是賽季手冊達到100級，再開通賽季手冊，好像是返賽季手冊的點劵了。這樣就相當於，《和平精英》免費送了我們一個賽季手冊。

提到《和平精英》的SS5賽季的段位獎勵，很多玩家都是比較關心的。刺激哥在查看留言時，就發現有玩家詢問：「能夠體驗透露一下SS5賽季的獎勵是什麼？」

《和平精英》的賽季獎勵，與以往的賽季相比，並沒有出現太大的改變。依然是賽季戰鬥服、超級王牌降落傘，以及無敵戰神的頭像邊框。

與國際服《Pubg Mobile》，《和平精英》的段位獎勵，顯得非常的沒有新意。

雖然，《和平精英》在遊戲玩法方面，一直都比《Pubg Mobile》超前。但在細節方面，《和平精英》就有點不太重視玩家的體驗了。

唯一值得欣慰的是，《和平精英》SS5賽季的戰鬥服，還是很好看的。如圖所示，女款的SS5戰鬥服，是一件修身的棉夾克，搭配了短褲、以及雪地靴。

除此以外，還有一雙黑色的手套，看起來非常的幹練。

而男款的SS5賽季戰鬥服，除了身材有些寬大，短褲變成了長褲以外，其他的變化，反倒不是很大。

還有就是，《和平精英》的SS5賽季戰鬥服屬於整體的套裝，無法與別的服裝進行搭配。

至於SS5賽季的頭像邊框，與降落傘皮膚，刺激哥就不再進行詳細地敘述了。畢竟，很多玩家的倉庫里，都堆積了很多的頭像邊框與降落傘皮膚。

說句實在的，《和平精英》真的應該像《Pubg Mobile》一樣，修改一下段位獎勵的設定。

與《Pubg Mobile》相比，《和平精英》很應該提升一下段位獎勵！

在此，我們以國際服《Pubg Mobile》為例。如圖所示，《Pubg Mobile》從黃金段位到無敵戰神段位，都一一對應了不同的段位獎勵。

而《和平精英》從黃金段位到無敵戰神段位，在段位獎勵方面，則是出現了斷層。

這也導致很多《和平精英》的玩家，在段位達到一定的程度以後，就不願意衝擊更高的段位了。

反過來再看，國際服《Pubg Mobile》，除了原有的段位獎勵以外，《Pubg Mobile》還對更高的段位，增加了更好的獎勵。

身份銘牌、特殊名字顏色、段位稱號，這些都可以提高玩家們的積極性。而《和平精英》的段位機制，反而挺適合「養老」的。

刺激哥在《和平精英》到了鉑金段位以後，就不願意衝擊更高的段位了。這大概是很多玩家，都正在面對的一個問題吧。

希望光子的運營的時候，能夠從多個維度出發吧。也希望《和平精英》，可以提升一下段位獎勵，不要局限於「氪金」。
