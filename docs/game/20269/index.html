《洛奇英雄傳》S4序章今日開啟 萌寵卡丹帥氣出擊

2019-12-17 238條評論 3,478次閱讀 408人點贊

12月17日，《洛奇英雄傳》的艾琳英雄們會在S4序章「時代的祭奠」中與艾旦相遇，善於操縱雷電的他將給予英雄重重打擊。不過，成功戰勝這位老隊長的英雄不僅可以收穫頭銜以作紀念，還能從戰役掉落中獲得「紋章：艾旦的玩偶」哦！在新版本上線之際，「為了不忘記」戰役出航次數限時加1，可讓英雄們更深地感受艾旦的在戰役之中複雜的心路歷程。

與此同時，羈絆系統將開放至第二階段，更精彩的對話，更近一步的交流，更多的頭銜等待英雄們開啟與獲得啦！在英雄們紛紛迎接新版本到來的時刻，有一位新夥伴悄然進入艾琳大陸之中，他，正是萌化的卡丹！現在，就讓我們看看本次版本將更新哪些新內容吧！

【2019女神的饋贈XI】

在2019年12月17日維護後至2020年1月21日維護前，四款「騎士卡丹」將入駐2019女神的饋贈XI，還有神靈系「瓦爾基里套裝」及「恩赫里亞套裝」限時加盟，驚喜多多，等你開啟啦！

「騎士卡丹」夥伴無疑是2019女神的饋贈XI中的王牌選手，身體雖然變小了，但實力和帥氣程度可是一點都不輸給當年的騎士團團長哦！「騎士卡丹」不僅不會進入瀕死狀態，還能幫助英雄拾起道具與採礦，在英雄生命值不足時，還會使用魔法幫你恢復生命值哦！如此實用的小可愛，你不心動嗎？



而「瓦爾基里套裝」與「恩赫里亞套裝」則為神靈系時裝中的高潔款，女性簡潔性感，男性魁梧英氣，將神靈的高貴很好的體現出來。


英雄們對死亡翅膀並不會陌生，雖然對於不少翅膀來說偏小，但配以暗黑系時裝又極為適合，此次全新上線的「紅色死亡翅膀」血氣十足，相信在英雄們的搭配下能展現出1+1>2的效果！

還記得秘密房間里的那位妖嬈的女子嗎？她不知在多少英雄的心裡印下了對《洛奇英雄傳》最初的記憶，如今，「女妖黑色套裝」重回艾琳，你的抵禦得了超短裙與緊身皮衣的雙重誘惑嗎？

【時裝沙龍】

冬天來啦，聖誕節也即將到來，「冰雪公主套裝」與「冰雪王子套裝」伴你溫暖過冬，想要入手的英雄可在商城或時裝沙龍中直接兌換哦！


【商城禮包】

在與艾旦激戰之際，五款禮包限時上架，「2019每日倒計時禮包」除了多款使用道具外，累計獲得15張「2019每日倒計時禮券」可合成為頭銜「再見2019，你好2020」，「2019聖誕節禮包」多款外觀兌換券伴你美麗過聖誕，還有「頭銜：特別的愛給特別的你」留作紀念喲！「不死鳥禮包」重回艾琳，四款不死鳥翅膀你會擁有哪種顏色呢？「頭銜：不死鳥」見證你的心動歷程！此外，還有「艾旦戰役出征禮包」與「8周年紀念禮包」登錄商城，為你的征戰之旅助力！欲了解詳情，還請進入遊戲中商城！

【守護聖誕節蛋糕活動】

在2019年12月17日維護後至2020年1月7日維護前，完成10級以上普通戰役即有大概率遇到假冒聖誕老人的地精，擊敗他後可獲得聖誕節糖果和假冒聖誕老人偷走的蛋糕（每日最多4個），將蛋糕交給奇怪的旅行者後可收穫「正義的禮券」，將12張「正義的禮券」製作為聖誕節禮物後，可從中獲得迷你聖誕熊、紋章：小鹿旅行者、聖誕節鹿角頭部裝飾！

此次新版本上線後，英雄們熟悉的「阿內斯特的採集活動」與「火眼金睛活動」將再度開啟，希望大家積极參与啦！2019年的末尾，就讓《洛奇英雄傳》陪你一同度過吧！
