《大鏢客2》PC版獨家內容更新至PS4，美顏拍照模式Get

2019-12-16 952條評論 4,743次閱讀 3633人點贊

《荒野大鏢客2》的PC版算是主機的移植版本，在上個月發行的時候為了能吸引更多的玩家增加了一些主機上沒有的遊戲內容，比如拍照模式和其他的額外任務。不過因為《荒野大鏢客2》之前早就在平台上首發過了，所以PC版本的銷量表現並不是很好。官方這兩天又把《大鏢客2》PC首發的一些遊戲內容更新到了PS4平台，也將會在明年的1月21號在Xbox One平台上更新這些內容。

　　這些更新內容包括：拍照模式——在遊戲中能夠自由移動鏡頭拍照，還能添加濾鏡、文字或者貼紙進行對照片的後期處理，並且可以把遊戲的照片直接上傳社交媒體分享。增加的故事模式賞金獵人任務是原有的故事線之外追捕通緝犯巴特·卡瓦諾、卡米爾·德麥厄蒙特和赫爾曼·親岑多夫。故事模式的藏寶圖會有富饒標識和一些元素的路線圖。故事模式中的幫派藏身處，玩家可以在規定的地點和兇猛的哮狼幫作鬥爭來完成任務。

　　在故事模式中新增的武器包括豪客左輪手槍、勒馬特左輪手槍和M1899 手槍等。故事模式中還增加了新的馬匹品種，比如稀疏斑點阿帕盧薩馬、紅栗色阿拉伯馬和其他的馬匹。連隱藏的小飾物都增加了諸如貓眼、海歸殼、鯊魚牙等種類。

　　《荒野大鏢客2》PC版本利用全新圖形和技術改進來進一步增強遊戲沉浸感，並充分利用了電腦強大的性能特點，讓這個龐大、豐富而細緻的世界的每個角落都栩栩如生。環境、光線和陰影有著更加接近於現實的效果，畫面中的植物、服飾以及動物皮毛的細節也更加讓人驚嘆。

　　《荒野大鏢客2》標準版在Steam平台上售價為249元，這兩天在進行折扣促銷活動，標準版折扣20%，現價199元；特別/終極版折扣25%，現價247/299元。Epic版約244元，打折後價格197元。估計接下來年底加上聖誕應該還會有折扣。喜歡的玩家可以抓住年底的優惠儘快入手。
