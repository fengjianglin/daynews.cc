美服奧山首日聯盟勝率80%，部落也能開心？

2019-12-12 892條評論 2,954次閱讀 1417人點贊

美服已於12月10日正式開啟戰場，作為萬眾矚目的奧山戰場，首日表現如何呢？我們來看一看。

戰場排隊情況

我們知道，美服聯盟與部落的陣營比例較為均衡，甚至聯盟玩家還會稍稍多一些，但是由於聯盟玩家喜歡PVP排戰場的人會少一點，所以導致當前聯盟基本上處於秒拍，而部落則需要大概5-10分鐘時間，其中戰歌排隊時間稍長，要10分鐘多。

所以有玩家分析，當前國服的聯盟部落比大約是4:6，如果再算上部分聯盟玩家不喜歡玩戰場，排隊玩家可能在3.5:6.5左右，那麼國服的部落玩家可能需要排隊的時間會比美服玩家長。但是由於當前都是對推，每一局結束得非常快，所以就算在國服，部落玩家也不會出現一兩個小時打一場的情況出現。

戰場對推情況

由於當前排隊時間都不長，並沒有出現大家擔心的部落玩家需要排隊一小時的情況，所以目前美服奧山戰場，絕大部分都是以速推的方式進行。

從一個聯盟玩家的主播視角來看，聯盟部落目前完全和諧，全部對沖，基本上10分鐘解決一場戰鬥，而這位主播一天不到已經刷了6萬多的榮譽，而且奧山聲望已經達到了崇敬！在這種效率之下，怕是沒有人會傻到去慢慢打。

有一個網友直言：看了挺久，都是直接對攻，彈幕有很多人問這是不是跑馬大賽，因為開門，上馬，下馬殺，上馬，下馬秒將軍，結束……就看誰跑得快，騎著小馬的別去了。

有一名在美服玩的玩家PO出了他的榮譽界面：

下面這位部落玩家則說他一場都沒贏過，但是他完全不在乎，因為他就算輸了一場都有2.3k的榮譽，而聯盟玩家贏了卻只有990。

出現這種情況和聯盟部落打法不同有關，因為聯盟為了速殺將軍，經常全員直接沖，路上的NPC都不殺，而部落則習慣分成小分隊殺光路上提供198榮譽的精英，所以導致了出場後聯盟贏了反而獲得的榮譽比部落還要少。

不過有玩家表示，對於要在奧山裡升級的小號而言，聯盟優勢就大很多，贏了三個牌子交掉就有3000多經驗，而部落小號每場只有一個牌子，升級效率遠不及聯盟。

聲望方面，基本上贏了有500，輸了也有300左右，所以肝的話一天到崇敬，三天到崇拜完全不是問題，特別是聯盟的人類玩家，有外交天賦加持，聲望的獲取速度相當恐怖。

美服首日奧山的狀況想起了前些天看到的一位玩家的言論，他說現在在野外刷榮譽或者對刷的都是過家家，幾天後戰場開放了才知道什麼叫做真正的刷榮譽。

目前按照美服的情況，如果國服部落排隊情況比較好，那麼奧山大概率也是以速度互推為主。而如果部落玩家的排隊時間比較長而不願意辛辛苦苦排一個小時而速輸的話，那麼就有可能出現經典的長時間拉鋸戰。
