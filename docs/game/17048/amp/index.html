《荒野大鏢客OL》賞金獵人高效率玩法分享

天天要聞
2019-12-14

《荒野大鏢客OL》遊戲中玩家如果選擇了賞金獵人這個職業，那麼玩家將會在各地抓捕通緝犯，但是可能抓到犯人後離交任務還有一大段時間，那麼怎麼度過這段時間呢？請看玩家「寒夜將曉」整理的《荒野大鏢客OL》賞金獵人高效率玩法分享，希望能為各位玩家帶來一些幫助。

　　話不多說切入正題，因為之前賞金獵人看書模式能提高金條的獲得量，自身測試後確有奇效，所以每次抓人之前都會身體力行研讀一番，可是看了幾次之後就發現每次等完15分鐘的看書期，繼續遊玩就已經提不起興緻了，這樣一兩次之後就會關掉遊戲刷手機去，這樣的遊戲體驗很差啊，總這樣等啊等的我還不如玩手游去呢！

　　為了擺脫這樣的窘境，昨天開了收藏家職業，剛開始對著收藏家地圖刷的很起勁，集齊了塔羅牌系列（因為開始比較窮，買不起挖掘鏟和金屬探測器，只有塔羅牌系列比較友好不需要其他工具），可是因為一開始得到收藏家地圖，加之對於這個職業的不了解把一些需要其他工具才能拿到物品也拿了，比如珠寶，傳家寶的，一想到第二天就會刷新地圖，強迫症就很難受了，然後就從昨天上午的九點一直刷到晚上十一點，收藏家一天內滿級，才終於把除了美國野花之外的收藏品都收集完了（傳家寶缺了一件，交了每周藏品了），可是這樣的忙碌也很累人啊。

　　於是今天上線就想著為什麼不能把兩種模式相結合，讓賞金獵人的等待變得分充實，讓收藏家枯燥乏味變的有趣，所以我就試著對照收藏家地圖來進行賞金狩獵，在規定的賞金目標路程中標註一些比較中意的收藏品(就是撿值錢的藏品收集，如硬幣，戒指)，因為有15分鐘的「自由活動時間」，所以可以先標記收集一些不太遠的收藏品，等收集到兩到三個藏品後就可以前往目標了，加上趕路的消耗應該是15分鐘應該是充裕的，如果對自己實力自信，甚至可以在抓到目標之後繼續找一兩個藏品，前提是不要遲到哦。

　　比如下圖，去時紅色箭頭將三個收藏品拿走，回來時如果時送到聖丹尼斯就走綠色箭頭，一旦聖丹尼斯的目標收集品被清空可以繼續千萬羅斯，瓦倫丁等地，這樣既提高了金條的獲取量，也能有額外的美元收入何樂而不為呢。
