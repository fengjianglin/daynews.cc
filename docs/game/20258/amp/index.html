三國題材新作《逆襲吧英雄》開放事前登錄同步遊戲特色及玩法介紹

天天要聞
24 小時 ago

　以三國時代作為題材架構的《逆襲吧英雄》營運團隊宣布，於今（16）日展開事前登錄活動，並搶先釋出遊戲特色及玩法介紹，供玩家參考。

【以下內容為廠商提供資料原文】

一款遊戲兩種玩法豐富你的遊戲體驗

　　在《逆襲吧英雄》的三國世界中，融合了大型國戰及自走棋元素，國戰系統將會以24小時全面開戰的方式進行，而隸屬於不同陣營的玩家們可以透過佔領城池來取得資源奪得大量資源；遊戲的戰鬥方式也採用類自走棋的方式進行，玩家們可以透過戰略布陣安排開戰的武將陣型，在遊戲開始後，武將們將自主進行戰鬥，如果情況危急，遭受突如其來的敵軍前後夾攻時，也可以手動施放武將大招，或是即時調整武將站位進行應對，繼續保持優勢位置並贏得勝利。

  • 一款遊戲、兩種玩法，豐富的遊戲體驗

地圖推進，即時操作軍團戰鬥，每個細節都不錯過！

　　在遊戲初期，敵我雙方會以正面交鋒的方式進行戰鬥，但隨著關卡的推進，戰局也將愈發詭譎多變，原先可受前排保護安全發揮強力輸出的攻擊手，有可能遭遇戰場後方突如其來的敵襲而身陷險境，善用隨機應變的能力即時操控武將移動位置，利用運籌帷幄的能力以天下英雄為棋、百城十三州為局，取得天下一統。
