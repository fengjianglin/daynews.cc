王者模擬戰：不僅在5v5中是“射手榮耀”，下棋也是

天天要聞
2019-12-17

王者模擬戰冬冠杯32進16淘汰賽開打，首先跟大家說明一下此次比賽的賽制，可能會有些複雜。3人一隊，在32進16階段，每個選手打要3局比賽，總積分加和進行戰隊排名。淘汰賽第一天比賽結束後，已經有4支隊伍進入了16強。

結合賽後戰況跟大家詳細解析下賽制，如圖上有8支隊伍，因為每支隊伍都有3名選手，所以要打3場比賽；在一場比賽中選手都要打3小局，這3小局的結果決定選手在本場的得分；最後3名選手在3場比賽中的得分匯總，即為戰隊總成績。第一天晉級的四支隊伍分別是Hero、WE、鬥魚和YTG。在這樣的賽制下比賽時間會變得更長，因為每一場要打3局，打滿3場就要9局，按照一局30分鐘來算，要打4個半小時，加上比賽間隙還有其他事情，所以從下午3點一直持續到了晚上9點多。

如果大家沒時間觀看，又想知道高端局大佬的戰術、套路等，可以關注我。接下來給大家帶來9局登頂陣容的介紹。

第一場

3局比賽最終都是射手登頂，可見射手陣容相當熱門且強勢，無論是走堯天坦射還是長城射手。其實這兩種射手體系都很強勢，在有羈絆進階這個天賦的時候，可以優先選擇長城射，也可看成是另一種「養豬」，主c是公孫離，其次可以選擇2星黃忠或者呂布。沒有羈絆進階就選擇堯天坦射，也有3長城的羈絆，效果也不差。

第二場

有趣的是在第二場的3局比賽中，前兩局依然是射手體系的勝利，而且都是堯天坦射。值得注意的一點是，坦射陣容大後期3星廉頗的作用是巨大的，在面對戰士陣容以及刺客的時候，會讓對方非常絕望。其次阿離或者伽羅追3，黃忠2星即可。

第6小局則是精英戰登頂，天賦有致殘之握和羈絆進階，最後是一個8戰的羈絆。

第三場

經典養豬陣容拿到了一次第一，升9人口的時候拿到羈絆進階，5長城進階7長城，加上典韋老夫子，以及黃忠這種「大豬」。其實這一局決賽圈遇到的是一個有著3星武則天和3星婉兒的法刺體系，最後養豬能夠吃雞還是因為準備了2個輔助令旗。

另兩小局則是同樣經典的蜀衛刺第一，9人口的蜀衛刺可以補一個呂布，那就更舒服了。不過雖然這兩家刺客都吃雞了，但是他們都沒拿到嗜血之影~

9局比賽，5局射手登頂，典韋老夫子養豬中也有黃忠以及3射手羈絆，2局蜀衛刺，而被視為版本之子的精英戰則只拿到一局第一。如果想獲得關於模擬戰賽事的更多信息，歡迎大家關注我。
