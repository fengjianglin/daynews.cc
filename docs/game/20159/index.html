3000次愛與恨——受苦遊戲《第3000次決鬥》評測

2019-12-18 2,917條評論 3,982次閱讀 1613人點贊

受苦愛好者，來接受挑戰吧，3000次夠嗎？

遊戲介紹

《第3000次決鬥》是一款橫板動作遊戲，如果你是一位老玩家，那麼聽名字，你就應該能夠猜出這是一款怎麼樣的遊戲，受苦又要開始了。本作的戰鬥節奏比較快，畫面也比較絢麗，人物刻畫得有些「壯實」，動作雖然比同類頂級作品《空洞騎士》略有不及，但是流暢度卻不遑多讓，本作不僅在戰鬥風格上獨樹一幟，在系統功能得建設上也比較齊備。在遙遠的《空洞騎士：絲之歌》還沒到來的日子裡，這款《第3000次決鬥》或許可以給我們帶來別樣的受苦體驗。

元素爆炸

評價一款遊戲的可玩度時，遊戲時長是一個直觀重要的因素，而能夠拉起足夠長的遊戲時長的概念一定會包含豐富的元素，我們可以舉一個很顯而易見的例子——《傳說法師》，遊戲無論在未發行、EA還是正式發行之後都反響不錯，但是很快卻脫離了玩家的視野，問題就在於遊戲提供的敵人種類過少，玩家在適應了所有敵人之後，就會漸漸脫離。

本作顯然認識到了豐富元素對遊戲時長的作用，提供了豐富了遊戲地圖和特定場景，另外武器技能也很豐富，最讓人感到興奮的是敵人的種類繁多，想要掌握打倒每種敵人的方式都需要仔細觀察，細心揣摩。

此外，主角還是一個「魔法劍士」，在闖蕩的同時，可以收集散布在不同區域的法器，這些發起能夠催動禁忌魔法，這樣一來，無論是本身所具備的解鎖元素還是收集元素都擴展開來，遊戲的可遊玩度也大大提升了。

叛教可恥

角色屬性的設定與概念，很多人是在玩網遊的時候第一次接觸，而大家在加點的時候基本都會在網上查一些高手攻略，能夠獨立思考的卻寥寥無幾，這樣就鑄成了很多人根本不會加點的現象，比如作為一名狂戰士，真的要所有點數都加在力量上嗎？或許只有「莽士」王老菊才會如此吧。

本作的加點同樣需要玩家細細品味，不然到了後期免不了成為叛教之徒，說到這裡筆者需要著重說明一點本作不合理的地方，遊戲中的很多技能是戰鬥勝利後獲取的，而某些核心技能卻需要每種屬性達到相應標準之後才能夠學到，這就有點被迫叛教的意思了，這樣的遊戲設定導致要麼點數卡關，要麼技能卡關，希望製作組能夠適當調整。

反人類BOSS

BOSS在每個遊戲中似乎都是必須的存在，區別僅僅在於難易程度，本作的BOSS應該是比較容易的，或者說出招動作很好區分，一旦掌握了敵人招式的起手特徵，你彷彿就把握了敵人的生死命脈，猶如武林高手一般，接下來需要的只是耐心，適當的等待與適時的行動會助你打敗BOSS，本作的BOSS大多如此，不過也需要交不少的學費，比如監獄騎兵這個BOSS，你如果盲目攻擊會發現它一直是無敵狀態，而交夠學費之後才發現——射人先射馬！

此外，遊戲的惡意還不限於此，因為我們在其他遊戲中所習以為常的異常狀態，在本作中卻非常致命，一個普通的灼燒Debuff可能會讓角色大殘，在很多關鍵的BOSS戰中可能會讓局勢逆轉，所以請務必在普通戰中節省血瓶。

武器大師與瘋狂解謎

遊戲中的每種武器都有特定的攻擊方式和優缺點，並且角色的某些被動技能和主動技能能夠調節武器的輸出。遊戲中的角色可以裝備多種武器，能夠拿到一款你喜歡的武器並不難，雖然沒有《黑魂》系列中那些稀奇古怪的武器，但是本作的武器還是需要玩家充分練習才能夠適應的。

遊戲中的謎題可能是在受苦遊戲中比較特立獨行的一個點，可能是因為這類遊戲的戰鬥難度就已經讓玩家遍體鱗傷，所以不會設置謎題或者謎題飛鏟簡單，而本作的謎題卻四號不近人情，比如有些融入場景的解謎關鍵，如果你沒有發現，會卡關很久。

嘰咪嘰咪，每天都好玩！

總結

本作是一款中規中矩的受苦遊戲，提供了豐富的戰鬥元素、技能元素與對戰要素之外，也提供了對玩家的滿滿的「惡意」，雖然可以讓一個熟悉敵人招式的你能夠毫髮無傷地打倒敵人，卻也能夠讓一次受傷就回歸新手村，遊戲目前的更新頻率還不錯，剛剛更新正式版本之後打了八折優惠，受苦愛好者可以嘗試，缺乏耐心的玩家還是視頻通關吧！
