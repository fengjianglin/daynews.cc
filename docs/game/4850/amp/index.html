「經典系」《熱血物語》：那道小時候過不去的大門

2019-12-04

熱血物語

英文名：Downtown – Nekketsu Monogatari

日文名：ダウンタウン熱血物語

中文全稱：市中心的熱血物語

這個遊戲是熱血的第一個Q版遊戲，之前的遊戲有全身版的熱血硬派，也有半身版的熱血躲避球。

本質上，這其實是一個ARPG遊戲。

打小弟和小BOSS賺錢，買食品和物品升級，買書看學招式，然後到最後的學校去大戰一場，這是閉著眼睛都能想到的RPG遊戲經典套路。

但是到了最後的學校，打完了小兵，大門仍然緊閉，你告訴我根本進不去？

一定有什麼地方出了問題。

流程功略

OK，我們從頭來看看這個物語到底說了什麼。

標題交待到：鮫島力的女朋友被冷峰高校的人綁架了，於是好打抱不平的國夫協助好朋友力踏上了英雄救美的路上。

遊戲開始在花園高校門前。

經過一些場景後，會來到一個修路的場景，這裡遇到第一個BOSS沢口靖夫（さわぐち やすお），來自白鷹工業高校2年級，雖然這是我們遇到的第一個BOSS，但這其實是一個非主線BOSS。

遊戲中有大大小小14個BOSS，其中大部門BOSS是與主線程有關的，但有些BOSS不一定需要擊倒。

兩三下就把沢口解決掉了。又走過幾個場景，在橋上遇到冷峰的長谷部長，國夫的老朋友，她會告訴國夫，力的女朋友真美就被困在冷峰的三樓。

下橋後會遇到兩個小混混，上條恆男（かみじょう つねお）和山本憲之（やまもと のりゆき），來自影村學園3年級。

這兩個BOSS也不是主線人物，是用來攢錢錢的，被打倒後還會在後面的浴室店門前登場。

輕易打倒這兩位後，過了橋就來到了倉庫，會一會來自白鷹工業高校3年的西村成孝（にしむら なりたか）。

這是一個主線BOSS，打倒他後，他會告訴主角，冷峰四天王之一的木下忠（きのした ただし）就在花園公園廁所等著。

這時候就不用一直向前，可以往回走了。

打倒西村之前進入這個公園廁所，是空無一人的，現在木下出現了！

作為冷峰四天王之一，皮厚，武器使用較強。

打倒了之後他會讓你去找其它四天王，再大步向前走，在倉庫之後（當然打西村不再是必須），分別遇到四天王望月 駿（もちづき しゅん）、平清文（たいら きよふみ）、小林政男（こばやし まさお），這些當然都是主線BOSS，

小林這個場景是沒有小弟的，意味著可以反覆擊倒他賺錢。

如果之前沒有在橋底擊倒上條和山本，在澡堂前是見不到他倆的，不過他們也只是給主角刷錢用的。

在冷峰高校門前清理完雜兵後，守門的豪田剛（ごうだ つよし），來自寶陵高校3年，出現了。

他有一招必殺技頭槌，是為這一作數不多有必殺技的角色。

這裡有一個有意思的小技巧：不需要打倒他，也可以直接進入冷峰高校大門。

進入後先來到一個體育館，清掉小兵後出現BOSS鬼冢崇（おにづか たかし），冷峰學園3年。

打倒後跳上籃球架，進入教學大樓。

二樓有BOSS五代獎（ごだい すすむ），谷花高校3年，有一招必殺旋風棍。也是非主線人物。

三樓清完小兵後終於見到了傳說中的雙龍兄弟，服部龍一（はっとり りゅういち）和服部龍二（はっとり りゅうじ），都會使用旋風腿。

解決後就可以進入旁邊的教室，解救真美了。

但事情還沒完，從龍一龍二出現那扇門出去，可以到教學樓的天台。

見到了整件事件的始作俑者，山田大樹（やまだ たいき）。

這一作他沒有山田之術，所以輕易擊倒。原來正是山田才利用了雙龍兄弟，綁架了真美妄圖打敗力，從而最終控制這一片地區。國夫和力擊敗山田後，這幾個學院終於恢復了平靜，大家都好好學習了。（高中了還打群架，這是什麼社會？）

沒有什麼結局CG，只有字幕交待了後來發山的事情，然後工作人員名單就出現了，遊戲結束。

短評

整個遊戲的音樂都非常優美，各個場景都有現代日本漫畫的風格，人物角色來去自由，是為數不多可以體現日本校園浪漫的FC遊戲。
