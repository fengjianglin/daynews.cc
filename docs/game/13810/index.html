網易想指望流失的夢幻西遊玩家去玩3D版夢幻，絕對是白日做夢

2019-12-13 477條評論 4,597次閱讀 604人點贊

當時新丁一直強調保值大家才有序安心加大投入，現在突然沒了保值這屬性馬上垮一半，如果不馬上止損夢幻就差不多藥丸了。說白了就是越改越沒人，你為了盈利可以，但是現在點卡模式情況下，物品開始不保值，那麼就離關服不遠了。老卓說一句實話，玩夢幻西遊前前後後也有13年了，我不知道是我的智商問題還是策劃的智商有問題，出的有些活動我是看不懂，根本不知道怎麼玩，我13年的夢幻西遊基礎都看不懂，我真不知道現在夢幻西遊的新人怎麼玩。

司馬奇你再怎麼修改，也不能去打擊商人啊，現在夢幻最主要的問題是沒人來收貨了，大家都在賣東西一下子市場大量拋售，玩家投進去的東西不貶值才怪呢，應該想個機制讓商人不受影響，他們才是夢幻這個市場的平衡點。司馬奇肯定是靠著數學模型預測的，靠著所謂的大數據來改遊戲，但是肯定沒想明白，【人心】，是他的模型預測不了的，他的大數據也預測不了。

誰去問問他遊戲到底玩了幾年？為什麼之前別的遊戲倒了，夢幻沒倒，現在改成這樣，還死不認錯。3D遊戲一直有，恕我直言，指望流失的玩家去玩3D版夢幻，絕對是白日做夢。國產3D遊戲不少，沒有一款像樣子。這次的3D版，也不會例外的。

司馬奇不是打擊牟利，而是要和玩家爭利，雖然賺著手續費和點卡，但是玩家的那塊蛋糕他們也想分，這塊蛋糕太大，以至於比網易賺得還多。所以未來儲備會越來越多，想在遊戲里搬磚賺錢會越來越難，搬磚的都滾蛋了，號和裝備的價格還得繼續回落，只花錢不出菜的玩家看到自己投的錢不斷蒸發，不管他們脫不脫坑，起碼不會繼續買買買了。等所有隔三差五充錢的玩家失去信心，剩下的就只有投資不大，每天玩2小時的娛樂玩家了，到時候全服合成一個區這種事是不存在的，只能關服了。
