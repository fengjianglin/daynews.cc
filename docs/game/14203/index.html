FIFA大神模擬孫興慜1V7天神下凡長途奔襲射門，遊戲里不如現實

2019-12-10 2,718條評論 4,913次閱讀 984人點贊

EA旗下的FIFA遊戲一直是全球足球電玩愛好者首選，該遊戲以真實程度高而聞名於世，已經使用了多年寒霜引擎的FIFA在引擎上就已經領先了實況足球太多，所以FIFA的用戶也越來越多了，再加上版權上的優勢，FIFA已經牢牢佔據了足球遊戲龍頭老大的位置。

前幾天孫興慜在對陣伯恩利的比賽中上演了一記像是當年羅納爾多在巴薩時讓主持人高呼外星人下凡的超級進球，孫興慜幾乎以1打7，並且最終將球打進，全程孫的帶球速度極快又好像是當年的卡卡附體，讓人看的大呼過癮！

於是本人也在FIFA20中用多次使用托特納姆熱刺，因為這幾年孫興慜的實力越來越強，我也越來越喜歡使用這位黃皮膚的男人。在昨晚的遊戲里，本人用孫興慜也嘗試了一番奔襲連過數人後射門，可以說孫在遊戲里還沒有他現實里跑得快啊，並且最後的射門由於角度太小也沒能將球打進。但是，從各方面而言，遊戲中這一番操作也算是比較接近孫的這個進球了，特別是這腳球同樣也是維爾通亨傳給孫興慜的呢。

本人將創作更多的FIFA模擬現實動作和進球，歡迎各位朋友多多來關注哦！

（邱翼曦）
