前言

最近很多玩家都表示雲頂之弈越來越沒意思了，一場對局下來對手不是海洋法就是毒水晶掠食者，玩其他陣容又打不過，加入他們又相互卡棋子，一點遊戲體驗都沒有。9.23版本更新讓雲頂之弈的可玩性再次被壓縮，成型簡單的海洋法成為了版本之子。

雲頂之弈S2賽季的改動十分奇怪，5費卡中只有劫和劍聖屬於C位，其他的都是輔助棋子。這導致遊戲節奏加快，增加了以5費卡為核心的陣容成型的難度，而且這些5費卡在沒有兩星之前，根本發揮不出陣容強度。唯一強勢的光劫還遭到削弱，這讓5費卡劫成為玩家口中的弟中弟，而雷霆召喚劫體系就成了為劫最後的倔強。

雷霆召喚劫體系可行性

冷門陣容，同行較少。棋子質量較高，完全可以彌補少羈絆陣容強度不足的缺點。劫在3召喚和2海洋羈絆的加持下可以有效的打出分身技能，憑藉劫和分身的離子火花和雷霆效果，打出爆炸傷害。另外娜美和風女非常強力的輔助棋子，雙團控大招可以保證劫能順利用出更多的分身，組成的2秘術也可以緩解海洋法施加的壓力。

雷霆召喚劫玩法總覽

8人口成型，3雷霆3召喚2海洋2守護神2秘術，所需棋子為：奧恩、錘石、狗熊、安妮、娜美、風女、牧魂人、劫。

陣容優缺點

優點：所需棋子的都不是熱門棋子，比較好拿到，同時也更方便追到2星。陣容中核心都是高費棋子，在這意味著前期可以更靈活的過渡。裝備上只要求劫有離子火花和復活甲，並不是很吃裝備。陣容成型後有較高的強度，2星劫完全有機會吃雞。後期可搜雷霆拉克絲進一步提升陣容強度。

缺點：8人口高費棋子陣容過渡難度大，需要玩家有很好的遊戲理解和運營思路。比較吃經濟，前期沒有拿到經濟優勢會嚴重影響陣容成型，7人口升8人口的階段會掉很多血量，甚至直接暴斃。雖然該陣容能與海洋法一戰，但面對劇毒掠食者依舊處於劣勢。陣容成型太依賴運氣。

雷霆召喚劫玩法詳解

理想的英雄裝備搭配：劫（復活甲、離子火花、龍牙），牧魂人（幻影之舞、冰拳、救贖），安妮（鬼書）。

裝備優先順序：大棒>負極斗篷>鎖子甲>反曲弓>腰帶>女神淚>拳套。

開局拿大棒或者負極斗篷，前期能合成離子火花或者龍牙都可以。前三波野怪把能拿的棋子都拿了，有2星棋子就上，沒有就用森林、掠食者或者2雷霆。掉落的裝備能合成上述裝備的都合成，不能和成的小件都先分開丟給打工仔，最大限度提升前期戰鬥力，保證經濟發育。如果前期運氣實在太差就要打假賽，確保拿到較多經濟。

中期在拿到50金幣之後就可以用利息上到6人口，這個階段一般來說陣容強度是跟不上的，所以可以D20塊錢提升陣容質量，避免被打掉太多的血量，影響後期的容錯率。6人口陣容可用2雷霆2海洋2守護神搭配其他羈絆，運氣好搜到掘墓和安妮就用三召喚，這個時間段安妮和掘墓都很強。之後繼續存錢上7人口，7人口屬於陣容危險期，血量高的話可以直接用利息上8人口搜卡，血量低的話就要開始搜卡，搜出2星安妮或者牧魂人才能鎖血，歐皇搜出劫就能直接存錢上8人口。如果前期過渡不出大問題，該陣容完全能夠以較高的血量過渡到8人口。

後期來到8人口，當後面兩個回合沒有野怪打的時候，可以D完全部的錢，搜陣容所需的5費卡。如果後兩個回合有野怪就先D30塊錢，留著一部份等打完野怪再繼續搜。8人口搜卡要確保陣容成型並且都到2星，期間搜到拉克絲可以替換掉沒有3星的狗熊。站位方面需要劫與娜美站到兩個底邊角落，牧魂人站第一排中間抗傷害，風女和錘石靠中保護。

總結

雷霆召喚劫算不上海洋法真正的剋星，但是在海洋法橫行的版本，一場對局必定會有很多家同行，經過同行削弱後的海洋法強度必定被削減，所以該陣容在不被劇毒掠食者針對的情況下有很大的希望吃雞。注意劫必須要2星，而且裝備一定要有離子火花和復活甲。
