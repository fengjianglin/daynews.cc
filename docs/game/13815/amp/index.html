皇族粉絲難以抉擇的事情，想看到小龍堡綻放光彩還是香鍋復出？

天天要聞
2019-12-13

說起今年的轉會期，有一個很有意思的現象，就是傳統老牌強隊在今年的憋氣大會中都出現不同程度的損失，最嚴重的就是EDG了，雖然廠長還沒有退役，但是不打比賽和退役已經沒有什麼區別了，上單Ray轉會至KT，下路小昭加入VG，現在的EDG就是靠小學弟苦苦支撐，不知道到讓人們期待很久的廠長能否在下賽季王者歸來，再現之前的光輝，畢竟當時的EDG強到讓兩個LPL隊伍打出一個電競奧斯卡之夜。

很多人都好奇為什麼損失最嚴重的是EDG而不是皇族，在一開始我也這麼認為，應為卡薩加入TES，雖然有小龍堡，但是畢竟是新人，而且也沒有替補打野，如果小龍堡真的頂不住壓力成為第二個樂言，那皇族下賽季就沒法看了，不過作為莽夫之王MLXG最近在直播間卻說出了這樣話，當被問及小龍堡頂不住了怎麼辦，香鍋卻是這樣回答的：無需擔心，俱樂部在失去卡薩引進小龍堡之後就已經想好了退路，最後加了一句你們應該猜到的。

這幾個字就很耐人尋味，之前卡薩在的時候一直是首發，但是鍋老師作為王牌總是在皇族危難之際上場拯救比賽，所以很多皇族粉絲都將香鍋當成皇族的救世主，雖然現在鍋老師做起了直播，但是還是在高分局保持自己的狀態，現在又在直播中說出這幾個字，意思很明顯：小龍堡實在頂不住，我就重回賽場，不得不說，皇族的幾個退役選手真的很良心，讓男在巔峰時候和皇族取得了極大的榮譽，眾生平等的Buff讓所有上單失去了光彩，再版本更替以及自身狀態下滑之後果斷讓位，給別的選手機會和發揮空間。

姿態退役之後，也是在皇族水深火熱的時候宣布復出，幫助皇族扛過了那一段艱難時光，當時為了這件事某些粉絲還和王思聰發生口水戰，現在香鍋退役了，但是在直播間表示小龍堡一旦頂不住你們應該猜到的，相信大家都明白什麼意思了吧，不得不說現在擺在皇族粉絲面前一個很難的問題，到底希望小龍堡在下賽季發揮得好還是不好，這真是一個問題，發揮好的話自然不必多說，前兩個YM打野各自收穫了一個Fmvp，如果小龍堡能收穫第三個，那PDD真的是電競圈的一個傳奇了，但是這樣就看不到鍋老師復出了。

發揮的不好也是一個問題，雖然鍋老師的意思是小龍堡發揮不好，自己可以再次打比賽，但是香鍋面臨和Uzi一樣的問題，身體不好，一次兩次可以，如果長時間打比賽訓練的話，估計香鍋也扛不住，那皇族打野還是一個問題，更何況相比香鍋能再次打職業，我更想看到他好好調理身體，因為這樣以後就再也聽不到香鍋因為身體問題進醫院了，不知道你們對於這件事有什麼看法，你們想看到哪種情況的發生？
