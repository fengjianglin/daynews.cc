爐石傳說：魚人體系削弱後怎麼玩？瓦莉拉的惡魔流打法很容易吃雞

天天要聞
2019-12-22

在爐石傳說的酒館戰棋這次平衡後，魚人體系可以說是受到了一個比較大的打擊，遊戲的整體節奏也更加迅速，另外的一些體系此消彼長的變得強勢起來了，比如慢慢崛起的惡魔流，這裡我們以瓦莉拉的攻略來談談這種打法吧。

惡魔流的核心自然就是憤怒編織者了，瓦莉拉推薦打惡魔流的英雄是巫妖、拉法姆、薩隆、阿凱，巫妖可以賣血賺費賺身材，拉法姆薩隆可以多搞隨從，阿凱也是前期強勢，當然其他英雄起手就是2個憤怒編織者或者是憤怒編織者+惡魔的開局也可以走這種套路，前期不要賣血太多，等到後期體系成型了賣血更划得來。

卡組的另外一個核心就是漂浮觀察者了，在憤怒編織者讓我們掉血的時候他也會成長，所以前期的任務就是找他，瓦莉拉的標準打法就是3費拿憤怒編織者，4費再拿憤怒編織者或者惡魔，5費再拿其中一個，之後刷，有碰的就鎖住，6費就是升級加碰了，這裡最好的3星隨從就是這個漂浮觀察者，另外如果有靈魂雜耍者或小鬼首領也是要拿的。

7費直接拿2卡保證戰力，有惡魔最好，沒有就拿當時最強的隨從打工，8費升3星繼續找關鍵隨從，就是上面提到的漂浮觀察者，靈魂雜耍者等，之後的費用也要盡量保證自己隨從的身材變大，這樣才能繼續獲勝，畢竟我們每一滴血量都能給我們帶來極大的優勢，一旦沒血就浪不起來了。

像這把瓦莉拉開始的這幾個隨從就非常適合玩惡魔體系，可以保證每次隨從的身材都在增長，不過後面比較穩了之後還是要去找一些5星關鍵隨從的，939和2王都是比較適合這套體系的，2王在13血左右可以下，但是如果血量在10以下就不用2王了，繼續賣血養隨從是正道，畢竟也都是一回合的事。

上面這個就成型後的陣容，惡魔體系主要是成型快，傷害高，要和成型的魚人打肯定還是打不過的，不過如今玩魚人的大多數也撐不到那個時候了，那麼你們覺得瓦莉拉的這種惡魔體系如何呢？畢竟是1萬多分的玩家，在酒館戰棋的理解上肯定比我們還是要高一些的，那我們就可以用這種打法愉快的吃雞了，歡迎大家關注和分享，遊人將每天給大家帶來爐石傳說的消息心得福利和環境強勢卡組及技巧哦。
