原JDG選手加入WE，千分選手值得期待，網友：這兩家還能做生意？

2019-12-18 50條評論 3,849次閱讀 1053人點贊

上個月中旬，今年的冬季轉會窗口正式開啟，LPL平靜的表面下實則暗潮洶湧，不過就之前透露出來的一些消息來看的話，在這次的轉會期最慘的戰隊應該當屬WE了，雙C出走，教練離隊，讓WE連最後的蔬菜大棚都剩不下了，不過在這個轉會期進入到最後的時間段之後，WE宣布了一個好消息，他們招進了一個強力外援！

這位外援是JDG在LDL二隊的一位上單選手，他的ID是Morgan，是一位零零後的小將，從LCK遠道而來，今年的上半年加入JDG二隊，後因出色的表現上調至一隊，但是並沒能獲得太多出場的機會，在經過這次的轉會期之後，正式正式加入WE，成為WE的一員。

可能很多人對這位年輕的小將還不是很熟悉，因為他在LPL打過屈指可數的幾場比賽，不過人家可是一位真正的實力悍將，在LPL出場的幾場比賽中發揮的都很出色，目前在韓服中的rank分也達到了1300分，目前排在第十四名，從他的rank記錄中可以看出，這是一位偏進攻型的上單選手，他的英雄池中幾乎沒有坦克英雄，他青鋼影的勝率更是達到了驚人的78%！阿卡麗也高達61%，目前的版本中上單也都是這些進攻性很強的英雄，所以這位小將也是比較契合當前版本的選手。

另外此前有人爆料說WE簽了一個韓服上單打到1300分，中單打到800分的小將，如果這個小將就是Morgan的話，那麼說明他在中單位上的造詣也是不錯的，而目前WE還沒有公布新的中單人選，說不定這位小將還能來一次救急呢，所以對於這位小將的加入，此前表示已經躺好的WE粉絲這下可算是高興壞了，畢竟這可是一位韓服1300分的高手啊！

不過看到這位選手是從JDG的手中籤過來的時候，不少網友的心中也有一個疑問，相信不少人都知道在經過今年的幾次交手之後，WE和JDG之間的關係可以說是勢同水火，彼此之間還進行過幾次海報與文案的交鋒，不過最終不管是比賽還是海報都是WE取勝，所以網友們對這樣勢同水火的兩支戰隊之間，居然還能湊成這樣一筆交易感到非常的難以置信。

說起來雖然當時JDG的老闆處處碰瓷WE，但是畢竟過去這麼長的時間了，JDG的老闆在這段時間也沒有掀起什麼風浪，再加上這本來就是俱樂部之間的利益交換，所以能促成這筆交易其實也不是很奇怪。
