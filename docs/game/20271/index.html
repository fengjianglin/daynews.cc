三國群英傳2，毒士賈詡，前期弱後期強

2019-12-17 1,947條評論 2,919次閱讀 145人點贊

賈詡，曹魏的主要謀士之一，從東漢末年到三國，賈詡為他的各個主公獻過很多的計策，並且計策都非常管用，曾獻計李傕郭汜反攻長安，打敗呂布，逼死王允，在張綉手下時，獻計奇襲曹操，導致曹操長子曹昂和典韋戰死，在曹操手下時，獻計力戰袁紹，離間計馬超韓遂等等，計謀高超，才智過人

群英2中，賈詡的頭像不是很好，一副小人摸樣，和一代智者的形象還是相差甚遠，最主要的還是在於帽子，這顏色怎麼看都充滿了詭異

遊戲中，賈詡的基礎武力50，非常低，文官的通病，智力97非常高，有一個很大的優點就是全劇本都有他的身影，下面說說賈詡的武將技

1級技能，地泉沖，清兵技能，效果非常差，清兵數量保持在個位數，很難找到一個比這技能清兵更少的，有沒有這個技能都沒差別

7級技能，火雷，清兵技能，效果比地泉好一點，運氣好至少能清10多個兵，不像地泉那樣始終保持個位數

12級技能，御飛刀，打將技能，前期用來撿漏追殺敵將是非常不錯的，如果敵將是滿血的話，那這個技能在前期作用就不大，畢竟前期賈詡打不過對面，能減少集氣可以當成前期的主要作用，後期的話可以當成主要打將技能使用，雖然後期有點偏弱

17級技能，炎龍無雙，清兵技能，效果還是不錯的，主要是清兵慢但是持續時間長，還是能清掉很多兵的

23級技能，雷擊閃，清兵技能，看起來雷電還是非常多的，但是實際效果卻不盡如人意，清兵數量在中期非常有限，可以說效果比較差，相對於群英1來說，這個技能明顯削弱了

28級技能，後伏兵陣，造兵技能，非常不錯，但是賈詡的伏兵如果是前伏的話效果會更好一點，算了，不能要求太多

33級技能，大地狂嘯，清兵技能，最好用最強的清兵技能，前期即使再弱，只要擁有這個技能後，都能一飛衝天，而且33級估計是最早出大地的人物了

40級技能，炎龍絕殺，清兵技能，雖然不錯，但是在擁有大地狂嘯後，這個技能就顯得多餘了，如果來個打將技能，那賈詡就非常優秀了

武將技方面，前期非常差，賈詡武力又低，前期真是弱到沒有話講，中期技能一般，也強勢不起來，後期出了伏兵和大地後終於可以揚眉吐氣了，喜歡賈詡的朋友，那就慢慢挨吧，最後會苦盡甘來的

軍師技方面，1級的鼓舞士氣（初),5級的快速集氣（初），12級的強化武將（初），18級的亂陣，23級的快速集氣（中），29級的減少疲勞（高），35級的恢復體力（高），40級的混亂敵軍（高），除了一個恢體還可以，其他基本沒用，可以說賈詡作為軍師是非常差的

賈詡在群英2是屬於大後期的武將，不能當軍師，前期中期又非常弱，要讓他強勢起來，確實需要很大的毅力去堅持
