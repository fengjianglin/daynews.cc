三國志11：有人說呂布適合弓兵？又有人說適合騎兵？都說錯了

2019-12-06

摘要

大家好，我是趙子龍。《三國志11》中呂布很容易被玩家弄到手，算是非常受玩家喜歡的一個武將了，天下第一的武力配上特技飛將幾乎是所向睥睨，而且本身適應性3個s一個A，算是全兵種適配了，那麼玩家在遊戲中搭配哪個兵種更好呢？

大家好，我是趙子龍。《三國志11》中呂布很容易被玩家弄到手，算是非常受玩家喜歡的一個武將了，天下第一的武力配上特技飛將幾乎是所向睥睨，而且本身適應性3個s一個A，算是全兵種適配了，那麼玩家在遊戲中搭配哪個兵種更好呢？

首先看看槍兵的效果，呂布的槍兵適應性為A，不是S算是一大缺陷，但這並不影響，因為呂布裝配槍兵已經可以發動螺旋突刺了，完全可以作為一個硬控點，效果還是非常好的。

其次看看戟兵的效果，呂布戟兵適應性為S，而且統帥也不錯關鍵是飛將特技無視陸地ZOC，在大規模作戰時可以直接進入敵軍內部發動旋風或者橫掃對敵人的多股部隊進行打擊，效果出眾；在我方不利的情況下，呂布帶領戟兵還可以在前排充當肉盾進行堵路的作用。帶戟兵的呂布是非常全面的，進可攻退可守。

再次看看弓兵的效果，呂布的弓兵也是S，帶弓兵是完全可以的，我們經常看到電腦帶弓兵，但飛將這個特技並不能像弓神一樣避免亂射時候的誤傷，大部分時候還要使用火箭，很多時候火箭完全可以派幾隊適應性為B的文臣進行齊射，效果也非常好，這樣看來讓呂布帶弓兵就有點顯得大材小用了。

最後看看騎兵效果，呂布的騎兵適應性S，加上本身飛將無視ZOC可以充當騎兵，對敵人後排的兵器（衝車、井欄）就行突襲，防止攻城兵器接近並可以避免騎兵正面作戰帶來的巨大損失。

筆者人物四大兵種呂布選擇槍兵和戟兵為最優，騎兵稍差，帶弓兵最差。不足之處還望各位看官之處，歡迎留下您的寶貴意見！
