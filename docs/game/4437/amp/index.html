再說一個和積分相關的事情，烹飪和3葯是要求品質的，品質低了會怎麼樣呢?

　　品質低了肯定是扣分，這還用樓主說?

　　但是到底扣幾分呢?請聽我慢慢說

　　低品質扣分是按照品質差來決定的，A=(要求的品質-交的品質)/2，A就是被扣掉的分數，存在0.5的時候回波動，有時候加1，有時候減1

　　有人說樓主講低品扣分的事情，沒有意義，我默默表示你確實不會跑寶寶環

　　舉個例子，人物等級159，第96環要豆齋果，品質要求73，積分為180分，這時候算一下：

　　100環積分會超過197分，拿120的嗎?5環17分，難度太高，幾乎不可能。

　　100環積分會超過187分，拿110的嗎?5環7分，可能係數90%以上。

　　100環基本沒超過187分，拿100的嗎?5環6分以下，除非後面4個都是找人。

　　如果你是商人就應該知道110的鐵比100的書略貴，110的書比100的鐵略便宜，而且110的書不好處理，特別是垃圾書，這時候低品質扣分的技巧就有用了。

　　下面樓主準備說說跑環的準備，如何幫你提高跑環的速度。有豬友想分享一下嗎?

　　提升跑環的速度，從三個方面入手：

　　1、找人的距離應最大程度的縮短，方法主要靠旗子和飛行坐騎。

　　旗子怎麼定，樓主晚上放圖;飛行坐騎是跑環必備，可以節約大量的時間成本。

　　門派的選擇也很重要，DF應該是公認跑環最快的門派，其次是TG、WZ這種門派附近有較多NPC的門派，想入還沒入坑的吧友，門派的選擇很關鍵。

　　2、找環的速度，長期跑環的朋友，我推薦收一些環在手裡，這樣不僅加快速度還能節約很多成本。錢多的吧友收678花樂，錢不是很多收60和70的環就可以了，畢竟80環很貴，省下的錢和60、70的是差不多的。

　　3、背包，背包很關鍵，我推薦放3葯和烹飪，有沒有必要開3個背包，我覺得沒必要，當然有錢的話，開了就是更上一層樓。

　　丟傢具這個事情，經過各路大神的賜教，樓主弄懂了。

　　1、每種2J傢具丟到3個才會被要到。

　　2、出現這種機制的原因是因為新區不一定會有人會做某種2J傢具，所以有人丟了才會被要的。

　　至於提高要2J傢具的幾率等作用，樓主還是那句話信則有不信則無，真有假有，我也無從考證。

　　下面說說買跑環號的休息事項。

　　跑環號

　　買號第一件事情就是號應該能保值，不能到手虧一半。

　　首先職業跑環號等級，從追求人品爆發拿150書鐵的角度來說，人物等級越高越高，渡劫以下的我就不說了，推薦這樣幾個等級：159，171，174，175。

　　175和159是從號的保值和脫坑角度推薦的，171-174的號是從性價比角度推薦的，當然號的基本條件是人修滿，這是最起碼得

　　其次是跑環號的門派，首推DF，其他的TG、WZ都是不錯的選擇，DF跑環速度杠杠的。

　　最後是技能，師門技能主要看賣符技能和靈寶神兵技能，175號賣符號肯定要滿技能，要是入手大唐和DF神兵和靈寶160最佳。

　　跑環號還要考慮一下有沒有飛行坐騎，沒有的話門貢有沒有600，輔助技能最好有烹飪或者3葯，這樣會很方便，劇情技能最好會買旗子和定旗子，當然有小號的忽略這一條。
