明日方舟第六章主線地圖詳解 小火龍：超神的位置已然決定了

2019-12-25 712條評論 2,516次閱讀 1435人點贊

明日方舟作為遊戲自身核心的玩法自然就是針對地圖的具體情況去布置幹員了，本次就帶來第六章目前透露的主線地圖的詳細說明，對於小火龍來說，超神的位置已然決定了，一起來看看吧。

6-1（異鐵+龍門幣）

突襲附加條件：可部署單位上限減少2

這個地圖無敵的小火龍的位置擺放非常簡單，盡量放下方格子保證能燒三面，不過因為高台位置設定非常不錯，所以最簡單的思路和方法就是放狙擊堵門口，這是很實惠的想法，追求速通的情況下沒必要追求小火龍

6-2（紫薯條+紫方塊）

突襲：敵方單位的攻擊速度大幅度提升，攻擊110%，生命110%，防禦105%，攻擊速度170%

到處都是小火龍快樂位，不過就目前來看這個似乎走三近衛堵門口的思路更好，比如說火神+赫拉格+幽靈鯊的思路，而左下角這個高台位置放一個莫斯提馬很容易爽到。

6-3（糖組）

突襲狂暴宿主士兵的攻擊力提升至181%

傳統塔防，輔助成排系列，這個你不想堵門口也要堵，可依靠的地面單位不多，高台一定要多準備一些，同時奶媽一定要靠譜。

6-4（聚酸酯組+聚酸酯塊）

突襲：狂暴宿主投擲手的生命值提升至181%

這關比較麻煩，因為投擲手的生命值不低，輸出也挺高的，所以需要靠譜的重裝搭配能天使這樣的狙擊才行，八方方面小火龍燒三格不算虧，想堵門口基本上沒戲，但是擺放難度不高。

6-5（固源岩組+聚酸酯）

突襲：敵方狂暴單位不再自然損失生命

壓力不小的一個地圖，正上方中間位置是不是家門目前不清楚，如果是的話那就坑爹了，那對於上方的高台位置的幹員出書要求極高，小火龍壓力倒是不大，在中間高台一放，向下燒三格沒啥壓力，後面補1-2個幹員守住一切好說，主要吃高台輸出的練度。

6-7

有浮士德+弩炮台

難度方面不好說有迫害整合運動的潛力，可以選擇守家或者堵門口兩種思路。

6-8（酮凝集組）

突襲：生命值120%，移動速度150%

分批次阻擊的思路，這個地圖設計的很有意思，簡單粗暴，但攻略起來難度也不低，因為看起來怎麼也攔不完敵人，尤其是突襲難度會增加敵人150%移動速度，需要注意和小心。

6-9（聚酸酯，中級作戰記錄）

突襲：攻擊110%，防禦110%，生命值110%，對凍結單位傷害提升至167%

小火龍，送葬人，黑快樂圖，這一眼就看出來了三個堪稱完美的直線和近距離AOE的位置和高台了，難度會低很多。

6-10（異鐵組）

突襲：鑿冰人攻擊速度提升至3，生命提升至150%，其他敵人生命提升至110%

這個圖不知道這個箭頭是什麼意思，代表什麼所以目前不好判定該怎麼針對

6-11（扭轉醇，裝置）

突襲：戰場中7個位置無法部署單位

難度問題不大，這種標準的地圖只要地面重裝幹員素質過硬，小火龍不論在那個高台位置都有絕佳的表現和少人的低點，如果這個地圖扭轉醇掉落效率較高，可以考慮常駐。

6-12

突襲：源石結晶間隔減少為50%，所有敵人攻擊110%，生命110%，防禦110%

手長甘願的快樂圖，這種直線類型的地圖最簡單了，中間最右側的高台還有上方的過道高台都是完美的部署和輸出位置。

6-14（研磨石）

所有敵人攻擊110%，生命110%，防禦110%，虛幻無人機的移動速度提升至150%

這關注意有冰凍的爆鴿，所以對於狙擊幹員的練度考核極高。

6-15（掉落海膽）

突襲：所有敵人攻擊110%，虛幻無人機的射程減少為90%，生命提升至200%，防禦提升至200%

敵方低星很窄，以長條為主，所以壓力不大，又是一張小火龍快樂圖。

6-16（新裝置，酮凝集）

突襲：禁止使用醫療幹員

有霜星的地圖，而且突襲模式還禁了醫療幹員，這就很考驗奶盾的練度了，難度會稍微有點高，提前做好心理準備。

S6-1（中級作戰記錄）

問題不大，能到我方老家的路線只有兩條，還被堵死了，送葬人和小活動在這兩個高台一方，來多少人都無所畏懼。

S6-2（固源岩，龍門幣）

注意這一關有冰爆源石蟲，所以近戰受到了威脅很大，不過左下方的高台位置很不錯，適合考慮一下，下方高台位置很多，能很好地處理特殊敵方單位。

S6-3（酮凝集，中級作戰記錄）

依舊存在大量的冰爆源石蟲，不過問題也不大，下方一個小火龍搞定一切，推理增大的這個地形可以適當利用一下來迫害敵方單位。

S6-4（龍門幣）

兩條直線，留給高台位置發揮的地方並不是很多合適的地方，也就是左下角這塊小火龍能燒兩個可以考慮一下。
