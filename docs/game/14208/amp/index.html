電競入奧？奧委會提交兩條電競發展之路，電競運動員不再是夢想

天天要聞
2019-12-10

電子競技算不算正式體育項目？能不能進奧運會？

首先第一個問題，百度已經給出了答案。

電子競技（Electronic Sports）是電子遊戲比賽達到「競技」層面的體育項目。電子競技運動就是利用電子設備作為運動器械進行的、人與人之間的智力對抗運動。通過運動，可以鍛煉和提高參與者的思維能力、反應能力、心眼四肢協調能力和意志力，培養團隊精神。

至於能不能進奧運會，有網友表示，「就連圍棋這種傳承了幾百年，有公平性保證的運動也沒能進奧運會，更別提電子競技了。」

但讓我們回顧這些年電子競技走過的歷程，也可以看出，電子競技正不斷的被人民群眾以及商業所認可。

  • 2003 年，國家體育總局將電子競技正式確立為第 99 號體育項目；
  • 2008年，國家體育總局將電子競技改批為第78號正式體育競賽項；
  • 2012年，電子競技被提名為2020年奧運會比賽項目；
  • 2013 年，中國電子競技國家隊成立，參加了在韓國仁川舉辦的第四屆亞洲室內與武道運動會；
  • 2017 年，國際奧委會第六屆峰會經過激烈討論，最終發表官方聲明，承認電競是一項正式的運動；
  • 2018 年，電子競技正式作為雅加達亞運會的表演項目之一。被視為這是電競向前的一大步。

2018年IG更拿下了英雄聯盟S8世界總決賽冠軍，央視也第一時間進行了轉發祝賀！

但是在近5年的時間電子競技入奧應該是不可能的了。因為在今年年初的時候巴黎奧組委提議在2024年的奧運會上新增四個項目，包括霹靂舞、滑板、攀岩、衝浪，並沒有電子競技。也就意味著即使這四個項目無緣2024奧運會，也不會有新的項目能被批准。

然而奧運會確實承認了電子競技是一項體育運動。

其實電子競技入不入奧並不重要，關鍵的是一個可以證明電子競技這個競技項目以及職業的機會。無數的電競人也是為此而努力著。

在最近的2019年12月8日，在瑞士洛桑舉行的第八屆奧林匹克峰會上，參會的各位大佬們討論了電子競技領域發展的機會。

事情的起源是國際自行車聯盟主席戴維·拉帕蒂恩特向大會提交了一份關於如何結合電競促進奧林匹克運動發展的報告，並提出了兩類可參考的路徑。

一是考慮如何管理他們的電子、虛擬形式的體育運動，探索與遊戲出版商合作的機會；

二是考慮打造體育利益相關者和電子競技社區之間的對話平台。

結果是奧委會將兩路並進探索電競「藍海」。

簡單的來說第一條路就是入奧的遊戲將不僅僅由遊戲廠商決定，需要更多力量匯入，來保證每次遊戲方向的更新與調整具備競技體育的公平性。

第二條路就是建立一個有能力的體育平台，利用更好的資源對接和溝通廠商和電子競技社區。

從這兩條有實施性的措施所展現出的國際奧委會態度來看，電競入奧也許不再是夢想~
