摘要

傳奇大陸最神奇的特殊套裝之一,裝備使用後果 :某個伺服器如果有人戴齊了祈禱一套,經常是有這樣的情景

傳奇大陸最神奇的特殊套裝之一,裝備使用後果 :某個伺服器如果有人戴齊了祈禱一套,經常是有這樣的情景,在外面練級的道士,突然發現才召喚出來的寶寶,不幫自己打怪物,卻拚命和自己親近,開始還以為是別人的寶寶發瘋了,定睛一看是自己的寶寶在砍自己,而HP已經見底了。有的寶寶還去砍自己組隊的朋友,而朋友還以為是道士不心指揮錯誤,或者是鬧著玩,結果躺在地上了回城了, 才明白。在外練級的玩家在彈盡糧絕時,回城補充就發現安全區已經是血流成河。和自己一樣躺在地上的人很多, 屍體在那層層疊疊一大片,人多，走在城市裡,慘叫聲連綿不絕,屍體在腳下橫七豎八，唉真是慘烈啊!惶如身處亂世。滿地寶物閃閃發光,可是大多數人都顧不上看清楚,急忙逃離城市去僻靜的角落練級。如果知道寶寶叛變了，道士如果有7級寶寶千萬別回城哦,不然肯定被城市裡的法師濫殺無辜。其實7級寶寶是不叛變的,失去理智的那些寶寶都是才召喚出的。

關注微信公眾號：復古傳世手游版下載遊戲，領取禮包

如今祈禱套仍舊在傳世大陸非常稀有,但是有一點需要注意,祈禱套穿戴身上後,如果玩家死了,祈禱套就會隨之灰飛煙滅,所以,警告玩家穿戴祈禱套後盡量不要去PK。

祈禱套裝有武器祈禱之刃、頭盔祈禱頭盔、祈禱項鏈、祈禱手鐲、祈禱戒指共五樣。
