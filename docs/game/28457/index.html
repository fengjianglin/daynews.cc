絕地求生黃金大獎賽總決賽首日：SMG和OMG狀態復甦 4AM依舊低迷

2019-12-23 2,396條評論 3,419次閱讀 2387人點贊

絕地求生黃金大獎賽總決賽首日結束，TSG戰隊憑藉著兩次吃雞的成績暫時問鼎榜首，而低迷了一段時間的SMG和OMG這兩大老牌豪門戰隊狀態有所復甦，並且目前排名暫時位居前十行列，依舊擁有衝擊冠軍的希望，4AM則發揮低迷，目前僅排名倒數第二。

　　這次絕地求生黃金大獎賽，在經過了六天的爭奪之後最終決出了晉級到16強的隊伍，4AM、SMG、omg、RNG、VC等老牌強隊紛紛晉級到這次決賽。在第一天的決賽爭奪中，TSG戰隊從眾多強隊中脫穎而出，並分別在第一局和第五局成功吃雞，目前暫時以55分排名第一，有很大機會衝擊這次比賽的冠軍。

　而TSG爭冠的最大對手將是BA和TITAN戰隊，這兩支隊伍在這次比賽的發揮也比較穩定，特備是BA雖然沒有吃雞，但卻憑藉著穩定發揮拿到53分，在積分榜僅僅比TSG落後兩分。而老牌豪門戰隊SMG也打出了應有的水準，目前以44分排名第五，也有很大機會衝擊這次比賽的第一名。最讓鬥魚水友欣喜的莫過於OMG戰隊，在低迷了很長一段時間之後，這次OMG終於在決賽開始蘇醒，目前也拿到38分暫時排名第七。

　　相比而言4AM在這次決賽的發揮並不是很理想，在跌跌撞撞晉級到16強之後，4AM在首日比賽僅僅拿到19分，暫時排名倒數第二，而這次比賽4AM也遭遇了狀態不穩定的問題，這也是目前4AM碰到的最大難題。
