LOL：轉會全攻略，RNG陣容完善，JKL去向成謎

2019-12-18 274條評論 4,087次閱讀 2899人點贊

大家好，我是承諾，很高興見到你。

轉會期的最後時間已經結束了，各大戰隊也是紛紛曬出了新賽季的陣容。這個轉會期，有太多我們意想不到的東西。廠長退役，MATA回到RNG，peanut加盟LGD等等，與正規賽事的激烈程度相比毫不遜色。競爭力越來越強的LPL聯賽里，每家戰隊都想在轉會期尋找到最合適的人選，衝擊即將在上海舉辦的下一屆世界賽。

有人歡喜有人愁

這是目前已經官宣春季賽大名單的隊伍，可以說傳統豪門的WE和EDG在新賽季並沒有太多的競爭力可言，尤其是WE基本是全新人出賽。SN則是引入了SofM作為打野的選手，前中單選手Maple離隊，總的來說改動強度不是很大，看新賽季的磨合程度了。

這一部分呢，是根據各大戰隊已經在官博公開轉會的選手和上賽季上場的陣容一起整理的一份名單，基本上變動不會太大。

大致調整：

LNG引入Xx和Maple在原有陣容上提升了深度，在新賽季有一戰之力了。TES引入Karsa和一對新人下路，意在全華班。LGD轉會期變化明顯，Peanut、Killua以及FenFen的加入對隊伍整體實力有很大提升。RNG方面在小龍堡替換掉Karsa之後再次引入了新人作為後備力量，基本同上賽季強度差不多。FPX在冠軍陣容上引入了LCK著名上單Khan增加了隊伍容錯率。IG在阿水合約到期後引入了原VG的下路組合。

這幾支隊伍陣容還未完全確認，尤其是新進的eStar目前官宣的只有wink加入，其他位置暫不可知。VG在轉會期活動頻繁，IBoy和蕉太狼的加入對隊伍提升很大，加上KKoma的加入，新賽季可以期待了。

以上是目前已經產生的各支隊伍名單，馬上德瑪西亞杯即將開戰，相信在接下來的幾天將會迅速的完善陣容。

JKL去向成謎

那麼問題來了，作為這個轉會期的大牌選手JKL的去向到底是哪裡呢？此前各路爆料說阿水會去SN和EDG，但是這兩支戰隊確是早早的更新了大名單，並沒有阿水。對剩下的隊伍而言，阿水還能夠去哪些隊伍呢？

從最後幾支不完善隊伍來看，V5今年因為各種問題大概率不會引援大牌選手了，eStar在剛剛簽約Wink後應該也不會再去追逐阿水，VG引入了Iboy就更加不可能了，那就只有在已經相對完善的隊伍中。

RNG可能性極小，FPX也會保持冠軍陣容，JDG剛剛擁抱Loken回歸，LGD上賽季Kramer打得也不錯，RW風格不太適合，BLG應該也不會放棄金角。那還剩下IG、TES、DMO和LNG了。這幾支隊伍其實都是有可能的，DMO和LNG都進行了重要位置補強，如果可以在增加一個核心位置會在新賽季擁有很高的主動性，不過財政方面是需要考慮的地方。IG和TES是可能性最大的，TES沿用的是一對新人選手缺乏穩定性，IG雖然引入Puff，但是在聯賽中表現其實一般，如果阿水可以重回IG將會在S10具有更大的可能性。

綜上，阿水回到IG和去TES組成銀河戰艦的可能性是最高的。

今天就跟大家分享到這裡，歡迎大家在評論區討論。我是承諾，我們明天再見！
