天外魔境1代

《天外魔境ZIRIA：遙遠的吉潘古》是《天外魔境》系列中的1代，是HUDSON開發在電腦上的一款奇幻風格RPG遊戲，而且是89年電腦上首款RPG遊戲！之後2005年在XBOX360上重置！

1代的遊戲劇情是圍繞三個主角展開，他們分別是：

天外魔鏡

​最後再給大家說說《天外魔境》系列的作品：

《天外魔境ZIRIA：遙遠的吉潘古》是05年1代的翻新版，主機是XBOX360！我沒主機，其實我很想玩這個試試！

《天外魔境MANJIMARU》是03年2代的翻新版，主機是PS2！

《天外魔境：南彌陀》是05年3代的翻新版，也是PS2！

《天外魔境：青天之外》是03年的GBA版！

《天外魔境：ZERO》是95年的SFC版，我試了試，感覺還不錯！

《天外魔境：真傳》95年的街機版，這個大家應該都知道！

《天外魔境：第四默示錄》97年的世嘉土星版，我還沒玩過！
