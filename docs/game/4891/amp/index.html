哈嘍，大家好！又到了每天的遊戲資訊時間，我是你們親愛的小白龍！

穿越火線官方將在今天推出新版本「竟無止境」，伴隨著新版本而來的還有30人團隊競技新模式、槍王排位的改動、無盡挑戰的改動。我相信，有很多玩家對於挑戰模式都跟小白龍一樣，只停留在了巨人城廢墟。不知道你們有沒有，反正我曾經可是通宵幾次都是為了刷個黃金箱或者刷個水晶箱。曾經的巨人城沒有神器，想要打通關少不了團隊的協作，記得曾經17、19、29關的A點或者B點集合嗎？那時候的我們，一把馬來劍、銀色殺手都能讓我們高興很久，現在雖然越改越驚險刺激，但是少了青春的味道！

想必這次加強的無盡挑戰模式很多老玩家都沒聽說過，但是對於現在還奮戰在穿越火線一線的小夥伴肯定知道是怎麼玩的。自新巨人城推出之後，也有很多人說那是天亮的巨人城，雖然通關難度變得很大，但也被實力強勁的玩家輕鬆突破。在10月31日上線的新挑戰模式——無盡模式，沒有挑戰難度上限，每張圖有六波怪物，只要在規定時間內擊殺便可獲勝，還可以無限復活，還推出了精華槽位和符文系統等等新玩法。但這個模式可給了眾多玩家當頭一棒，聽著可刺激，但是通關卻比「西天取經」還難，許多喜愛挑戰模式的玩家也是知難而退，喪失興趣。

但這次「竟無止境」版本來襲可謂是解決這一現狀，官方優化了通關率，通關獎勵也更加豐富。讓每次通關稍微輕鬆一點，那樣玩家才會有體驗下一難度的興趣，千改萬改可是要以玩家著想。此外還提高了無盡模式符文的產出率，讓新體驗玩家在剛開始就可以嘗試符文的作用。

這次改動最人性化的地方就在於匹配機制的改動吧，眾所周知，無論是在挑戰模式還是無盡挑戰模式中，階段越高，難度也就越大，於是便需要隊友的協作。尤其是在無盡挑戰中，在難度前期還是比較好打的，一兩個大神抱團在配合超強的挑戰武器便可通關，但越到後期越困難，那已經不是一兩個人能走過去的了，而是需要團隊的協作，也是這遊戲的初衷。但單人匹配到的總是一些水平參差不齊的玩家，更別說裝備了，或許前期能夠帶動，但是後期並不是一兩個人的力量就可以通關了，有些地方是需要整個團隊來共同維持的，但無奈碰到一些萌新或者一些拖油瓶，導致出現了很多中途退房的現象，這個模式中途退房還沒有懲罰，於是這種現象越來越多。

想必這次官方也是注意到這個點，將匹配機制進行優化，在新版本中你可以更快地匹配到實力相近的玩家，而實力差距太大的則不會出現在一個房間。那種大神帶萌新的現象，除非是開黑，否則就不會再出現了！
