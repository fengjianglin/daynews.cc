皇室戰爭：光頭策劃談調整，戰鬥天使很平庸，不會有所改動

2019-12-24 1,712條評論 3,323次閱讀 921人點贊

曉東說遊戲，皇室天天見！

各位CR的小夥伴們大家好，臨近元旦，《皇室戰爭》這邊也差不多要準備遊戲的平衡性調整了。自本賽季開始，很多小夥伴就已經開始預測下賽季要削弱的卡牌，其中聖水戈侖、暗夜女巫和戰鬥天使的呼聲最高。明眼人都看得出，這3張卡牌在天梯賽基本成標配了，隨便一搭配都是一套能打的卡組，風頭一時無兩啊。

不過，不管玩家呼聲有多高，最終的調整還是得靠CR策劃們，對於下賽季的調整，一向敢於擔當的光頭策劃這次又來了，他明確表示，以暗夜女巫和聖水戈侖為核心的推進太過強勢，所以在下次的平衡性中，這兩張卡牌會進行削弱或者一些小調整。總之，從目前形勢來看，肯定不會是增強！

而對於新卡牌戰鬥天使，光頭策劃覺得她並沒有玩家所說的那麼強悍，光從目前的使用率和勝率來看，也沒有什麼太突出的地方，所以在下個月的平衡性中，戰鬥天使並不會有什麼改變。

確實，在很多玩家眼中，戰鬥天使能打能抗，而且還能為團隊恢復生命值，4點費用就能做這麼多，當真是一張不可多得的輔助卡牌。一旦解不好，就很容易因為她的群體恢復力，讓敵方部隊越來越多，最終無法抗衡而被推。

但在大神眼中，戰鬥天使雖然屬性強悍，但也不是沒有弱點。首先是她的攻擊方式，雖然自身可以恢復生命值，但面對人海卡還是只能束手就擒。其次，她雖然可以飛行，但對於空中單位卻只能是被動挨打，並且地面部隊也可以打她，這也算是個弱點。而她的群體恢復也是需要打到敵人才能觸發。所以要解她，就從她的這些弱點入手就可以。

不過也有不少玩家反應，戰鬥天使融入卡組的能力太強，就算是狗球流也能塞進去。雖然大家所說都是事實，但不要忘了，戰鬥天使是張新卡牌。新卡上線，大家肯定是要拿她來做各種試驗的，那麼大家見到她進入各種卡組也就不奇怪了，慢慢的，大家發現她不適合某些卡組也就會放棄，所以這些根本不用擔心，所有卡牌剛上線都有個磨合期，過了就好。

最後，大家現在要面對的問題，就是聖水戈侖和暗夜女巫的削弱，各位小夥伴覺得CR將會怎樣對他們進行調整呢？歡迎評論區留言，說出你的看法吧。
