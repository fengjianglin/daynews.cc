DNF12月普雷全職業排名：爆發續航破招，劍魂僅排第7，紅神未上榜

2019-12-04 0條評論 1次閱讀 0人點贊

摘要

“地下城與勇士之小狐狸君愛談遊戲電競”第六百七十六期《DNF12月普雷全職業排名：爆發續航破招，劍魂僅排第7，紅神未能上榜》，雖然現在的版本普雷Raid已經進入了末期，單人模式也是陸續開放了，不過小狐狸君覺得，對於平民玩家而言，就算是單人普雷，還是一個交復活幣的副本。

「地下城與勇士之小狐狸君愛談遊戲電競」第六百七十六期《DNF12月普雷全職業排名：爆發續航破招，劍魂僅排第7，紅神未能上榜》，雖然現在的版本普雷Raid已經進入了末期，單人模式也是陸續開放了，不過小狐狸君覺得，對於平民玩家而言，就算是單人普雷，還是一個交復活幣的副本。那麼選定一個版本強勢的職業，可以節省不少的打造花費。尤其是當前版本史派克特色造神的時代，神豪打造的下水道職業，都比不過一般打造的超一線！那麼12月的版本幻神都有哪些，小狐狸君經過工會玩家的實測和玩家的反饋，得出了真幻神、幻神、超一線三個檔次的職業，當然不包括輔助奶爸和奶媽。老規矩搞快點，直接上本次國服全職業的排名截圖展示：

DNF12月普雷全職業排名：爆發續航破招，

此次普雷的排名，和之前東總的打樁理論排名，有著本質的區別。不再是以技能數據和面板的設定為基礎。而是綜合考慮了：續航、爆發、破招、容錯率、猴戲……等多個要素，比如：武極這個職業雙二擋提升巨大，實戰打樁能力複製修鍊場非常輕鬆。所以武極是當前版本最強的真幻神職業；然後是戰法、武神、四叔……等5個職業，雖然20S的數據不強勢，但是各具特色：續航、容錯率、抓破招（四叔專屬）非常不錯，碾壓超一線水準，是當前版本的幻神職業。

劍魂僅排第7，

最後是以劍魂、漫遊為代表的職業，雖然數據不錯，但是技能形態差、續航能力差……比如漫遊的和劍魂的一覺……導致這7個職業僅僅是目前的超一線水準。劍魂是目前超一線的分界點。強一點的就是幻神，次一點的就是超一線、一線職業。目前排名，在普雷的表現，僅僅是第七，勉強能夠用的職業，也是一個值得打造的職業。

紅神未能上榜：

紅眼這個職業，差不多實戰一線的職業。算是第四個檔次的職業了。神豪打造可以玩玩，平民玩家想靠這個職業進入普雷，在當前版本還是有一定的難度的。所以未能上超一線+幻神的榜單。
