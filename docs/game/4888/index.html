各位置歷史最強國產選手陣容，放現在世界賽起碼三連冠

2019-12-04 0條評論 14次閱讀 0人點贊

摘要

上單：Korol。原名童揚，前EDG上單，江湖人稱“童無敵”。S5的MSI上，決賽EDG大戰SKT，面對SKT三人圍剿，反殺一人全身而退，在2015年，納爾這個英雄姓童。提起妖姬超遠距離秒人會想起飛科，落地金身會想起鼻子，一炮四響會想起PDD，提起如來神掌，很多人都會想起扣肉。

上單：Korol。原名童揚，前EDG上單，江湖人稱「童無敵」。

S5的MSI上，決賽EDG大戰SKT，面對SKT三人圍剿，反殺一人全身而退，在2015年，納爾這個英雄姓童。提起妖姬超遠距離秒人會想起飛科，落地金身會想起鼻子，一炮四響會想起PDD，提起如來神掌，很多人都會想起扣肉。EDG有Korol在的比賽，基本都是你幫我我就carry，你不幫我就拖後期打團，是輸是贏全看廠長自己的選擇，有一句話很心酸，扣肉看到對面有人就要趕緊跑，因為他知道，對面藏著千軍萬馬，自己身後空無一人。線上江容易，團戰童無敵。巔峰期的Korol絲毫不虛LCK的Marin和Semb。

替補：PDD，Gogong，神超

打野：clearlove 。原名明凱，EDG打野，江湖人稱「廠長」、「諾導」。

曾經一次排位賽中，廠長的對手有職業選手(對的，就是我，Vdog)。廠長在成功將對手碾壓之後，在公屏打字:「玩的這麼菜，打什麼職業，回去養豬去吧」。一時間廠長大名響徹LPL。廠長可以說是LPL的牌面之一，也是榮譽最多的選手之一（除了S賽，別杠），從WE到EDG，廠長一直在用自己的方式統治著LPL，也影響了一代小豬崽，包括MLXG和TIAN都是廠長的粉絲。打野聖經「二級抓下就完事了」現在還在發揮著功效，廠長曆屆國產打野最強實至名歸。

替補：MLXG，Ning，Tian

中單：Vdog。原名韋朕，前DGL中單，樂觀家族族長。

韋神可以說是國產中單天賦最高的選手。S5是韋神的巔峰也是DGL的巔峰，春季賽橫掃OMG以及Snake。不過在決賽中以2-3不敵EDG，憾失冠軍。夏季賽捲土重來，3-2力克QG，獲得LPL夏季賽冠軍。韋神的發揮可以說甚是亮眼，皎月、狐狸、發條都是拿手英雄，和Pawn的「無天「大戰「佛祖」也是LPL的經典場面。可惜S賽的表現毀了一切，小組賽開局四連敗，LPL一號種子率先打道回府，維魯斯的「反向一箭」也帶走了韋神的天賦和未來。

替補：若風，五五開，無狀態，Knight（國產中單的希望）

ADC：Uzi。原名簡自豪，RNGAD。

出道即巔峰，S3S4連續兩年世界賽亞軍，也是S8IG奪冠前LPL的最高成績。可惜S5OMG「銀河戰艦的隕落」和S6QG「中野恩斷義」絕打斷了小狗的上升期，S7小狗回歸RNG捲土重來，一路殺進世界賽8強，可惜遇到大魔王。S8「一年6冠」，可惜世界賽抽到G2，這一笑，就笑到了S9。S9賽季止步世界賽16強。單論實力，小狗LPL第一ADC毋庸置疑，可惜命運不濟，可能會終生無S冠。

替補：微笑，JKL，LWX

輔助：Crisp。原名劉青松，FPX輔助。

17年和LWX一起加入NB戰隊（Doinb當年離隊），正式出現在大眾視野當中,那年LWX和LQS的下路組合是LPL下路計量單位。S8與LWX一起加入FPX，S9夏季賽FPX3:1戰勝RNG戰隊獲夏季賽冠軍，Crisp獲總決賽MVP。世界賽FPX一路殺進決賽，3比0橫掃G2登頂獲得2019年英雄聯盟全球總決賽冠軍 。

替補：捲毛，Ming，Meiko
