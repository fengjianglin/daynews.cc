DNF100級新增「神話」裝備屬性 - 發布會目前透露的部分

2019-12-16 1,059條評論 2,110次閱讀 855人點贊

神話裝備簡介

  • 神話裝備，是19年韓服發布會上公開的新等級裝備，稀有度位於史詩之上。
  • 在發布神話裝備之前，官方表示不會重置裝備，但緊隨其後就發布了新等級的裝備，使玩家一瞬間感受到了欺騙感，現場一度尷尬。但之後對神話的介紹，得到了玩家一定程度上的接受。
  • 首先，神話裝備只分為3個部位，即上衣、手鐲、耳環，分別對應三大類防具、首飾和特殊裝備。其次，每個角色只能穿戴1件神話裝備，更加印證了策劃極度想讓玩家放心，不會重置裝備這一點。
  • 既然神話裝備是凌駕於史詩之上的，那麼其屬性一定有特殊之處。首先， 神話裝備附加有專屬屬性，其數值是隨時隨刻在一定幅度之內浮動的，沒有固定的加成數值。舉個例子來說，如果裝備屬性為技能攻擊力 +x%(1%~6%)，那麼就會在這個數值區間來回浮動。另外，在專屬附加屬性中，只能隨機發動1種。所以究竟會觸發哪一種屬性呢，這也是要看臉的。
