QG向AG宣戰，770放言冬冠杯將復仇奪冠，老帥深情流露暗示退役

天天要聞
2019-12-17

AG超玩會戰勝QGhappy奪得秋季賽冠軍之後，雙方的隊員接受了KPL《榮耀大話王》欄目的專訪，月光豪言要帶領AG開啟一個王朝時期，而老帥似乎暗示自己即將退役，QG的隊員們對AG獲勝非常不服，放言將在冬季冠軍杯進行復仇，讓我們一起來看看他們具體都是怎麼說的。

首先接受採訪的是AG教練月光，他說到，這個冠軍是我在KPL執教生涯中拿到的第一個冠軍，意味著一個開始，我會在之後的冠軍杯還有明年的春季賽打造一個AG超玩會的王朝。

AG的輔助選手愛思表示，AG隊伍裡面很多人都是亞軍出身，這次奪冠也算是有了一個質的飛躍。

六點六採訪的時候非常淡定，幾乎沒什麼表情，他說到，打職業到現在終於有了收穫，自己很渴望這個冠軍，一個冠軍能給隊伍帶來很多好的改變，能提升隊伍的穩定和磨合，基本上後面的世界冠軍杯大概率還是冠軍。這裡六點六有個口誤，後面的是冬季冠軍杯，世冠是明年的比賽了。當然，我們可以看到六點六對拿到冬冠還是很有信心的。

作為隊伍的老大哥，記者讓六點六評價一下全隊的表現，六點六認為全隊的狀態都在不斷提升，一諾的射手和野核都很棒，藍柚和麟羽輪換的打法也非常好，老帥越來越穩，而且穩中還帶有點秀，愛思越來越有大將的風格，指揮節奏都不會亂。至於六點六嘛，還是一如既往的穩嘛，說到這裡六點六的嘴角露出一絲壞笑。

一諾和六點六一樣，也是AG陣中的核心選手，他對自己的要求很高，他直言自己還會繼續努力，要創立一個五連冠，拿到屬於自己的FMVP。之後一諾也對粉絲表示了感謝，並表示，AG也是今年才剛進來的一支隊伍，拿到了這個成績，那冬冠再拿一個好成績吧。看的出來，這次FMVP頒發給老帥，一諾心裡多少還是有點遺憾的，不過一諾才17歲，冬冠杯開賽的日子正好也是一諾18歲的生日，屬於他的時代才剛剛開始。

最後是本屆FMVP得主老帥，他說這次比賽的經歷很像是第一屆KPL，我們也是一路從敗者組打上來，通過比賽積累了更多的經驗，對賽場也更適應，最後也沒想到能贏。我自己的話一直有一個冠軍夢，今年終於實現了，我的職業生涯已經圓滿了。老帥這句話似乎有所指，一般說自己職業生涯圓滿了，都是暗示自己要退役了。不少網友也認為老師功成身退也是不錯的選擇，畢竟老帥的年紀也比較大了，以後轉型當教練或者主播都挺好。

QG的選手也接受了採訪，作為六點六和一諾的老隊友，QG的輔助選手770表示，首先要恭喜六點六和一諾奪得了冠軍，我們曾經也是為冠軍一起拼搏過的，我也為他們高興，但是就我個人而言，還是比較遺憾的，這次KPL的冠軍是AG的，但是沒幾天就是冬季冠軍杯了，下一個冠軍就是我們QG的！

SK表示，我們輸了之後，我一直在舞台下面等隊員，就是怕他們難過，我也是安慰他們，大家儘力了就好，我們下次會捲土重來！雖然說是成王敗寇，但是也允許有失敗，這次輸的是我們，但是下次就不一定是我們失敗了。看得出來，SK教練也想在冬冠杯與AG再戰一次，證明自己和隊員的實力。

冬冠杯還有幾天就要開賽了，預祝兩隊都能打出精彩的比賽，取得滿意的成績。
