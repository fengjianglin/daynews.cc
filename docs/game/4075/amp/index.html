玄不救非，氪不改命，老鹹魚ACG次元飯貓君給各位陰陽師大人們請安了。

繼決戰平安京之後，陰陽師又一款IP新遊戲百聞牌最近已經正式公測了IOS版，安卓預約人數也超過了48萬，加上tap tap 8.4的評分，可見其還是取得了一個比較滿意的成績。而在公測之後不久，陰陽師、百聞牌、決戰平安京就開啟了第一次聯動活動，不僅在遊戲中送出聯動福利，還進行了《一言不合百聞牌！？》的直播。

兩位老大哥帶帶百聞牌新出的小老弟也算是正常操作之內，但是讓人沒有想到的是，在直播當中出現了一件有趣的事情。百聞牌作為一款策略類的棋牌類遊戲，相比於陰陽師和決戰平安京，沒有那麼多涉及到角色建模，而這又是陰陽師IP的賣點之一，但失去了這一亮點並沒有讓百聞牌頭痛，策劃靠著精美多變的立繪抓住了很多玩家的眼球。

除了考智商的遊戲玩法之外，百聞牌最大的亮點就在於它每張牌的立繪。在百聞牌正式公測之後，為百聞牌畫過牌面的畫師們也都紛紛放出了自己的作品，客觀來講大部分百聞牌的牌面並沒有好看到讓人震撼的地步，但是其精美程度肯定是位於同類遊戲的前列水平。

說回最開始提到的直播，在看到這麼多好看的牌面之後，很多玩家開始對於百聞牌的主策劃產生了興趣，ta到底是一個什麼樣的人？會不會是一個美麗端莊的大姐姐呢？但在直播開啟後，大姐姐的想法瞬間破滅，但也並沒有讓人失望。直播中請到了這幾款遊戲的主策劃跟玩家見面（包括陰陽師主策劃「嚶蔚」），其中百聞牌主策劃「怪獸」讓人驚訝。

雖然「藝名」叫做「怪獸」，但是這位主策劃的顏值並不是怪獸級別的，相反還因為顏值過高受到了玩家的討論。在陰陽師這個「雞籠」滿天飛的IP環境下，百聞牌的玩家也開始喊起了「我可以」，不得不說，這樣顏值的主策劃貓君也覺得可以了。

而這次直播主角還是在於百聞牌這款遊戲上，策劃們也在直播在線打牌跟玩家們展示起了這款新遊戲。陰陽師主策劃「嚶蔚」在今年的520遊戲熱愛日上曾宣布，未來將會有多款陰陽師IP的新遊戲陸續上線，決戰平安京和百聞牌只是其中一部分，在測的還有更加休閑的妖怪屋，和還沒有曝光的疑似MMO的新遊戲。

不僅是遊戲方面，在動漫、周邊、實體店等方面陰陽師IP也在進行嘗試。百聞牌只是陰陽師IP發展道路上的一步，至於未來這個IP還能挖掘出怎樣的能量……以今年陰陽師前11月就已達到50億流水（去年全年流水48億）的成績來看，還是讓人期待的。

大家覺得陰陽師這個IP的潛力還有多少呢？
