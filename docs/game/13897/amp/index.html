深評

帶你深入了解遊戲

這次給大家推薦一款優秀的國產獨立遊戲——《失落城堡》。它是由Hunter Studio開發發行的可聯機類橫版動作遊戲。在遊戲中，玩家將化身為一名寶藏獵人，打敗惡魔，探索失落城堡，並逐漸變強。

雖然劇情啥的非常老套，但它不同於市面上的無腦刷刷刷，數值爆炸的遊戲。失落城堡中的寶藏獵人非常真實。

你僅有一條生命，各方面的屬性數值甚至比不上一隻小怪。你只能通過熟練的技巧，高智商的大腦來應對哥布林，蜘蛛等各種魔物。隨機性的武器道具需要你擁有作為寶藏獵人的嗅覺。

面對boss該如何站位，如何有效的躲開技能，如何游擊作戰等等，都是成為獵人那一刻起便需要思考的問題。

樸素不失細節的畫面

失落城堡的初始畫面可謂非常簡約樸素，咋一看好像就是十幾年前的遊戲，2D美術，濃厚的暗黑地牢風格，以及那一頭身的Q版人物造型，有沒有一瞬間彷彿回到了當初《冒險島》火爆的年代。

雖然畫風確實比較樸素，但遊戲在細節跟特效上面可是下足了功夫。角色吃了香蕉跟蘋果後會產生果皮，不小心踩到香蕉皮還會摔倒。（告誡我們不要亂丟香蕉皮哦）

遊戲的細節還表現在各個方面，比如角色跳躍，倒地，受傷之後的表情變化，怪物挨打之後的抖動反饋，場景內的寶箱，瓶子，雕像等等物品，被破壞之後都宛如真是場景一般，細節滿分。

隨著探險的不斷深入，關卡畫面越來越精細，第一關還是非常普通的哥布林跟城堡，第二關就變成了長滿藤蔓的森林，怪物的種類也開始變的豐富起來，石頭人，蜘蛛，野豬，骷髏兵等等。而到第三關畫面再一次有了質的提升，幽藍的骷髏鬼火相當有氣氛。隨著角色武器的不斷升級，武器的特效也變得越來越炫。

200多種隨機武器，可玩性極強

Rougelike類型除了只有一條命的設定外，最能讓人一眼認出的當然就是遊戲的隨機性所帶來的新鮮感。

《失落城堡》中武器道具種類多達200多種，而且現在還在不斷的出新武器。武器的種類大概可以劃分為三種：近戰，遠程以及兩者切換類型。近戰有刀，劍，槍，匕首，雙刀等等，遠程有弓箭，法杖，槍等等，不同武器的攻擊技能，外觀均不同，如果你想體驗完所有武器，那足夠你玩上玩上成千上百局了。

當然，除了武器之外，還有各種隨機道具，裝備鎧甲等物品，五顏六色的神奇藥水你甚至不清楚它們的功能，有些可以給你強力的buff，有些可能會給你減速，甚至偷走你的金幣。如果你玩過《以撒的結合》，那你一定對這種隨機設定非常熟悉。

玩法夠硬核，成就滿滿

在核心玩法方面，《失落城堡》不僅需要優秀的操作技巧，更重要的是需要策略。

遊戲中角色只要釋放技能，甚至跑動後的回頭，都會有一定的慣性，用遊戲里的話語說就是前後搖，熟悉這類操作方式你可得下一定功夫，不然可能會被第一關小怪血虐。

Boss的設計上也非常有意思。不僅在造型方面做得炫酷，在攻擊方式上也做得足夠多。擊敗boss可不簡單，失落城堡中的boss戰講究的是策略戰，你必須掌握合理的走位跟一定的攻擊節奏，切記不要貪刀，才能攻打下來。

每一次失敗或者通關都可以獲得一定的靈魂，靈魂可以用來點天賦，這方面還是需要慢慢肝，當然天賦對遊戲的加成方面做得比較平衡。前期關卡簡單，但後面的關卡依舊很難。（據說還有各種好看的時裝喔，）

氪金方面只要18元就能解鎖所有內容，可以說是相當實惠了。

攻略

小小心得，請笑納

具體攻略大家可以網上自行尋找，這裡主要針對新手玩家說一下操作技巧。

雖然該遊戲每種武器的攻擊方式不同，但還是有一些核心技巧傳授給大家，首先是45°角跳砍，這是最基本的操作。斜跳砍基本可以躲開大部分小兵的攻擊。

其次是加點建議，優先加滿：第一個房間鑰匙>第二個房間鑰匙>神偷>鍛造技巧>幸運>大工匠>折扣。

次級可選：第3個房間鑰匙>靈魂收割者>飢不擇食>恢復力>銅腸鐵胃>炸彈人>重甲專精

《失落城堡》由於角色攻擊，技能，跳躍等都有一定的前搖，後搖。所以一定要把握好釋放的時機。熟悉boss的攻擊機制之後玩起來會更加得心應手。新手初期建議拿遠程武器如：法杖，弓箭等等，近戰武器可以等熟悉了45°跳砍再玩。
