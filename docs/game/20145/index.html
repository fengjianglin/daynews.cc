懷舊服戰歌里一句“術士能拉么”，暴露你是雲玩家，還是老玩家？

2019-12-18 1,194條評論 4,908次閱讀 297人點贊

懷舊服開放了戰歌和奧山，雖然很多玩家第一時間都想沖奧山聲望，但其實戰歌的熱度也非常高，而最近有的玩家進戰歌隊伍以後，一句「術士能拉么」把隊友們的思緒帶回到了十幾年前。

在早期的魔獸世界當中，戰歌峽谷里存在一個BUG，那就是SS可以在戰場里拉隊友，這種打法被稱作「拉拉隊」。拉拉隊的原理就是在家裡直接拉人，被拉的人有2分鐘的時間點確定，只要在兩分鐘內保持非戰鬥狀態並偷到旗子，就能直接連人帶旗子一起飛回自己家交旗。拉拉隊支配了一段時間的戰歌，一場戰鬥變成了哪邊的拉拉效率高。所以顯然這位玩家肯定是當年體驗過戰歌拉拉隊的，才會在開局問術士能不能拉。

其實在魔獸世界早期版本的戰場里，還有很多的玩法套路，除了拉拉隊，還有一種戰歌「飛賊隊」， 效果是盜賊開了旗可以直接秒回到家裡的隊友身邊。當初有一段時間組戰歌國家隊喊得都是「ZG飛賊隊開組，來會飛的賊，會守的LR，SS」，其原理是設置一個焦點宏和一個目標暗影步宏，開旗以後先對一個敵人用焦點宏，再對一個隊友使用目標暗影步宏，你就能出現在隊友背後了。

這兩種都是在早期遊戲不完善而產生的BUG基礎上開發出來的套路，隨著版本的更迭都已經得到了修復。不過還有一種與BUG無關，但老玩家們都深惡痛絕的套路，那就是「飛機隊」。飛機隊簡單粗暴，直接就是使用變速齒輪外掛，飛機隊盛行於九城與網易交接的三不管時期，那時的戰歌里人均閃電俠。

所以小夥伴們以後進戰歌，看到隊友問能不能拉，盜賊會不會飛之類的問題，可別直接開始嘲諷人家哦，那可是真正的老玩家。小夥伴們還遇到過其它奇葩套路嗎？歡迎在下方留言。
