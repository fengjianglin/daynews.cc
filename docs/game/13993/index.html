王者模擬戰：模擬戰里最基本的知識點講解

2019-12-11 181條評論 1,286次閱讀 123人點贊

我相信還有很多人不知道我下面說的，這篇文章是給不太懂模擬戰基礎規則的朋友看的，希望模擬戰大佬們手下留情，謝謝支持，接下來進入正題

棄牌庫的作用

每一張牌都是從同一個牌庫里發給大家，且牌庫的牌的張數是固定的！牌的數量跟牌的質量成反比，也就是說牌的質量越高，牌庫中的牌也就越少！具體數量我會一一告訴大家！

白色卡牌-1金幣-總共24張

綠色卡牌-2金幣-總共21張

藍色卡牌-3金幣-總共18張

紫色卡牌-4金幣-總共15張

橙色卡牌-5金幣-總共10張

因為牌的數量是固定的，我們知道一個三星需要九張一星的卡牌，所以不難看出在一局遊戲里，橙色卡牌只可能有一個三星出來，所以如果大家怕他橙色牌出三星，只要自己手裡有兩張對手想三星的的橙色牌，那麼這一把他就不可能三星，其他牌同理，不過其他牌想卡也卡不住，這就是大家常說的卡對手的牌！

講了那麼多那麼棄牌庫的作用到底是啥呢，我們看一張圖

這裡我們可以看出這一局有遊戲里，呂布的牌還有三張，就是說我現在只可能在刷出三個呂布，可是如果我想三星我還需要8張，這可怎麼辦，這個時候我們可以去棄牌庫里看一看，看看自己丟了幾張呂布，這裡的你丟的牌包括你賣的跟你花金幣刷新的牌！

這個時候我們進入棄牌庫，點擊重新洗牌，那些你丟掉的牌，就會重新進入你的牌庫，我們點一下之後再去看呂布的張數

現在牌庫里有九張了，可是3+5=8啊，有九張的原因是如果其他對手淘汰，或者把他手裡的牌賣掉，那麼他賣掉的牌也會重新加入牌庫里，這把遊戲我淘汰了一個拿了一張呂布的，所以他的所有牌，包括呂布都重新進入了這一局的牌庫，要記住牌庫的所有人共用的，所以即使的把牌洗出來，也僅僅代表下回刷新，你可能刷到這些牌，並不是這些牌就只有你可以刷到，對手照樣可以刷到，所以棄牌庫的作用通俗來說就是你的後悔葯，他可以把你之前丟的牌刷新回來，可是能不能拿到還是要看你的運氣！

扣血規則

這是一局普通的對局，大家都知道左上角你頭像下的數字就代表你這局贏了回扣對手多少血，同理對手頭像的數字代表這局他贏了會扣你多少血，可是這個數字是怎麼算出來的呢？其實很簡單，就是你上場的牌的質量決定的，這一局我上場了婉兒，狄仁傑，程咬金，那麼我這三個牌的金幣加起來就是3+2+2=7，恰好等於我頭像下的數字，當我某一個英雄淘汰了，那麼就會相對應的減少那個被淘汰的英雄的數字，再來看對手的，對手有一個兩星的，那兩星是不是就是兩星所需要的錢呢，當然不是，兩星英雄的價值等於購買那個一星英雄所需的金幣+1，所以對手頭像下的數字等於（劉備）2+（李元芳）1+（兩星劉禪）1（+1）=5

相信大家都有過明明你可以打得過對方但是時間到了，你卻輸了，原因就是因為你的英雄價值沒有對方高，即使你打得過，所以如果時間到了，雙方戰鬥還沒結束，那麼系統就會默認價值高的一方獲勝

臨時購買區

遊戲時右下角的小齒輪，進入設置界面，裡面有一個臨時購買區是否開啟的選項，把它打開後就可以如下圖。。。。。

4.利息

遊戲里存金幣的利息跟你再銀行存錢很相似，錢越多利息越高，錢少於某個值，沒利息，錢高於某個值，利息還是一樣的，具體來說

如果這局打完你手裡有

10金幣—19金幣那麼你得到的金幣除了基礎收益，還會額外給你2金幣利息

20金幣—29金幣4金幣利息

30金幣及以上5金幣利息

所以如果你超過三十，不管你是30還是300，依然只有5金幣利息！
