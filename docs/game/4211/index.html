逗逗de魔獸世界懷舊服課堂之如何找回誤刪裝備

2019-12-09 0條評論 14次閱讀 0人點贊

摘要

昨晚打完熔火之心後，逗逗迷迷糊糊想幫自己的“巨人追獵者護腕”附魔+7耐力，結果喜聞樂見，狗爪一抖不小心點成了分解，等看到屏幕上提示“你得到了物品大塊魔光碎片×4”才發現為時已晚。

昨晚打完熔火之心後，逗逗迷迷糊糊想幫自己的「巨人追獵者護腕」附魔+7耐力，結果喜聞樂見，狗爪一抖不小心點成了分解，等看到屏幕上提示「你得到了物品大塊魔光碎片×4」才發現為時已晚。

杯具

正鬱悶時幸好公會小夥伴提醒可以通過戰網恢復裝備，趕緊跑到官網申請，順便將申請過程分享給各位，以備不時之需。

一、登陸戰網，點擊「支持」後進入下一個頁面再點擊「魔獸世界」

進官網，並登陸，點擊「支持」

進入下一個頁面點擊「魔獸世界」

二、新的彈出界面選擇「我想要通過問題分類尋求解決方案」，然後再點擊「遊戲問題’懷舊服’」>「想要恢復人物、物品、金幣’懷舊服’」>>「我不小心刪除我的物品（裝備、道具等）（懷舊服）」>>>「物品誤刪恢復」>>>>根據進一步提示選擇「點擊此處」

點擊「我想要通過問題分類尋求解決方案」

點擊「遊戲問題’懷舊服’」>想要恢復人物、物品、金幣’懷舊服’

點擊「我不小心刪除我的物品（裝備、道具等）（懷舊服）」

點擊物品誤刪恢復

按提示選擇「點擊此處」

三、現在可以在彈出的頁面選擇自己的伺服器（有下拉菜單）、誤刪時間（格式為年、月、日，舉例：2019-12-07）、需要恢復的物品（一共3個空可填寫，每個空寫1個物品/裝備）、聯繫電話、情況描述，建議如果能提供圖片證據的話將圖片作為附件上傳，全部填寫好後點擊「提交」

填寫工單

四、下圖是完成工單提交後戰網反饋的工單情況，以及今天系統提醒我裝備已恢復並通過郵箱寄給我。

發起完畢的工單情況

感謝客服，誤分解的裝備已通過遊戲內郵箱郵寄給我

五、注意事項：

1、恢復物品有3個月冷卻時間，也就是3個月內只能處理一次裝備恢復，處理後需要等3個月才能再次處理。

2、誤刪的裝備越快發起越好，客服是根據系統日誌來判斷你裝備是否被誤刪，以及誤刪的原因，時間太久的話如果系統日誌查不到不一定能幫助恢復。
