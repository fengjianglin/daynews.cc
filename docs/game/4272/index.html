英雄抗性的初始值和成長性各種各樣，先說魔抗，在小伽所了解到的全部英雄中，所有的魔抗初始值都是50，成長值都是每級增加8.5點魔抗，司馬懿例外他的魔抗成長值是17點每級，是其他英雄的雙倍，滿級司馬懿自帶288點魔抗，是全部英雄里最多的；而護甲值就多種多樣了，基本上都是坦克初始值和成長值都比後排法師射手高一截，初始值50-150之間，成長值15-30不等，這也是坦克和脆皮出相同裝備坦克更耐打的原因之一。

但無論是護甲還是魔的抗計算公式都是一樣的：

物理減免=護甲/(602+ 護甲)*100%

法術減免=魔抗/(602 + 魔抗)*100%

舉個例子，如果護甲值是900，得出的減免為59.9%，意思就是地方打出的物理傷害（包括普攻和技能）是100點，實際我們掉血為40.1點，減掉了大部分的傷害，小伽拿EXCEL做了一條函數曲線，來顯示隨著護甲增加，減傷是如何變化的，

圖上可以看出，護甲達到800，護甲收益開始變弱，為了更加嚴謹科學，小伽對護甲公式進行了求導，得出了減傷收益的變化率曲線=602/（護甲+602）^2,

到這裡有的小夥伴可能看不太懂了，表示一臉懵逼，通俗來講就是護甲達到800之後，再增加護甲，傷害減免增加的就越來越慢，達到1400增長就極其緩慢啦，舉個例子，護甲再800時減傷57%，而護甲1600時減傷73%，護甲性價比就變得極差，所以建議小夥伴在選擇裝備的時候，如果增加到1000左右，就可以考慮增加另外一項抗性或者增加血量啦。而魔抗則不需要考慮這麼多，因為目前魔抗的大件只有不死鳥和斗篷，再加上鞋子，三件件加起來也才700的魔抗，加上本身屬性大概850點，屬於性價比最高的階段，如果對面AP很強，可以毫無顧忌的堆魔抗。
