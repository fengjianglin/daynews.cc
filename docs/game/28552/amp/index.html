《鬥地主吐槽君》怎麼判斷一個人是不是鬥地主高手

天天要聞
2019-12-23

長久以來，所有的紙牌類遊戲都面臨著這麼一個問題，究竟什麼樣的選手才是高手呢？看勝率嗎？這是一個方面，當鬥地主當盤數達到上萬盤後，50％以上勝率的一般都是高手了，勝率更高也不是沒有可能。但是問題的關鍵在於，大多數人不可能一直那麼認真打，就算其技術可以達到60％的勝率，但他畢竟是人，人就有喜怒哀樂。

技術再好的人，要想贏，鬥地主時必須一直很認真打，但只要亂打，勝率很快就降下去了。要再打起來很難，”下去容易上去難”。所以說50％可能是高手，也有可能頂尖高手，只是他未必一直都那麼認真。那麼應該如何判斷一個人的鬥地主水平呢？今天我們就來給大家幾個參考的方向。

關鍵是看他會不會出牌。這個時候就有人要說了，這不是廢話嗎？出牌誰不會啊？我們這裡說的出牌是會不會出K、A這樣的牌。2和大小王，本身就是牌力非常強的定格牌了，打出來就是牌權，沒有什麼技術含量。太小的牌則是巧婦難為無米之炊，高手打也沒有用，只有K和A這樣的牌才能體現出高手的技術。在高手手中，K能夠很好地試探出對手大牌的分布情況是一張非常好用的探路牌。而A一般都是後手收割，等到場上的王和2都出完了，A的作用其實就和王、2沒什麼區別了！因此想要判斷一個人是不是高手，可以看他用K騙出了幾次A、2，用A獲得了幾次牌權就行了！
