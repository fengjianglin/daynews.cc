前言

競技場這個賽季環境不好，不知道幹嘛，由於只有術士沒有玩過了，所以在這個末期玩了個術士打評級，術士挺好進組，炮台隊伍多，寫一點經驗分享，也歡迎大家一起討論。

正文

天賦

1113223（大米也是這一套，沒有點爆燃衝刺）

PVP天賦：混亂聚焦，增加混亂箭傷害，必點

虛空守衛，法刀多/炮台隊伍，可點

烈焰纏身，打旗，近戰多可點

邪能裂痕，同上，後三選2個

精華

對面菜刀多，主衝突，多一個魔甲術，很有必要

副槽清醒夢境+完美願景

對面法刀AA+炮台，主完美(大米我也是主完美，副清醒+衝突)

個人建議3槽再去打，可以低配不用三級頂配(我只有3級衝突，2級清醒，1級完美，3級清醒觸發加全能，為了下個賽季，一定要做)但是沒有3個槽就有點坑隊友了，雖然2100分不高還是要對隊友負責

特質

裝備，特質只有一套，都是賭的

頭盔，靈魂功能頭飾，閃點+混亂獄火

肩膀，惡毒腐蝕護肩，閃點+粉碎混沌+回聲防護

胸甲，生靈背棄者法衣，閃點+混亂獄火+回聲防護

艾澤利特巨頭的襯裙，閃點+粉碎混沌+壓倒能量

囚徒的束袍，閃點+粉碎混沌

罪邪角鬥士的絲質長袍，粉碎混沌+閃點+回聲防護+壓倒能量（畢業極品，這個可以靠pvp征服拿，其他都是賭的，由此可見，每周打一個10+低保，可以賭兩次，先賭衣服，50%能出我們要的特質，由於我是末期小號，都是430特質，已經夠用，想要打上去就要賭445了）

3閃點+2粉碎+1混亂還是3閃點+2混亂+1粉碎就看個人洗好了，遇到對面法刀，可以選擇前者+主完美打爆發

飾品

PVP的，綠字全能，主動加主屬性，完美配合黑魂打爆發，可以寫一個宏綁定在一起開（大米我也是這個）

另一個看情況，對面法刀，帶全能觸發主屬性，沒有可以帶銘文做的，也很好（大米我也是這個）對面菜刀近戰多，帶護盾

其他散件要準備兩套

第一套，遇到法刀炮台對轟穿，極速全能/精通

第二套，遇到近戰菜刀隊伍，可能有人會問，現在還有近戰菜刀隊伍？有的，2刺殺+邪dk+dh+姑姑/暗牧這種(或許還有戰士)穿全能精通，我現在可以堆50精通+20不到全能，433.6裝等，常態34.6w血，牧師耐力+魔甲術，42w不到，能抗，活著才有輸出(毀滅精通減少傷害)，由於有3閃點存在，不用擔心缺急速

由於是版本末期，我是眼神上了1900+，然後進的YY隊伍，集火的時候浩劫給奶雙死纏奶媽做控制，騙斷比較隨緣，不滅決心可以打進攻，開爆發了地獄火+黑魂+飾品，一定要保重自己有章，不然對面控你就完蛋了。

作為一個術士，滿身大汗是常態，心態不能崩，控制吃完就可以打傷害了，多在yy說話自己大傷害讓隊友尤其是盜賊滿腎+煙霧彈秒人，也可以不打集火，壓制別人的血，然其他人轉火，大傷害爆發可以和隊友分開壓血，對面也會慌張。

結語

寫這個文章第一是分享交流，第二是給自己下個月攢人品抽幽靈虎。希望今年能抽到幽靈虎，集合石隨緣見了，戰友們。
