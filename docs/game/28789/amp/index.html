Xiaohu被粉絲噴上熱搜後，女友出來背鍋：要噴就噴我吧

天天要聞
2019-12-21

2019年冬季德瑪西亞杯今天開賽，在大量戰隊轉會期進行強化的情況下，RNG也在最近對陣容進行了調整。眾所周知的是Karsa離隊加盟TES，RNG的打野位置則是由來自YM的新人打野小龍堡頂了上來。本次德杯Uzi也沒來，下路則是由前FW的Betty選手頂上。這次德杯上，RNG上賽季的主力有上單LangX、中單Xiaohu和輔助Ming。

結果德杯小組賽的前兩場比賽，RNG先輸給OMG、再敗給JDG，吃下了新陣容亮相以來的兩連敗。第一場對陣OMG，RNG中下兩路都被爆。第二場對陣JDG，在優勢情況下被翻盤。這兩場比賽中，Xiaohu表現異常低迷，出現不少失誤，和隊友感覺也不在一個節奏裡面。

而Xiaohu本來就因為在世界賽上表現很差，成為現在RNG節奏最多的人。轉會期大量粉絲要求RNG換中單，但是德杯陣容宣布後，中單依舊是只有Xiaohu一人。在德杯開始前，Xiaohu在韓服中又衝上了一千分。結果德杯開始首日，他的表現卻很糟糕。這些導致他在今天下午被網友們噴上了熱搜，RNG官博下方評論區幾乎也都是清一色吐槽Xiaohu的。感覺Xiaohu今天的表現，讓他成為了RNG粉絲的眾矢之的。

沒想到Xiaohu女朋友隨後卻站出來背鍋了，他發出動態稱，這都怪他。因為昨天晚上他和Xiaohu吵架，而且是兩人吵得最凶的一次，而且最後還罵了自己。

不管是什麼原因吧，至少Xiaohu看起來確實有些不對勁，而且外界對他壓力現在非常大。對RNG來說，新賽季可能將非常艱難。對此大家有什麼看法呢？歡迎留言。
