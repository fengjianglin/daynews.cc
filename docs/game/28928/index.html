《只狼》過關技巧：如何巧過山內典善和鬼刑部？需要兩次忍殺

2019-12-20 2,259條評論 3,962次閱讀 2424人點贊

很多玩家在打完赤鬼後就不知道怎麼走，在赤鬼的上分有一個洞，用忍義手可以上去，但是過去之後會有一個BOSS山內典善。

這個BOSS周圍有6到7個小兵，這裡教大家一個技巧，當你已經響起來BGM時坐鬼佛是可以重置小兵和BOSS的，但是這個BOSS旁邊沒有鬼佛在門口外邊的一個懸崖邊有一個鬼佛。在懸崖邊的鬼佛重置敵人，由於山內典善周圍的小兵過多，小編建議不用上去硬鋼，當重置後所以敵人都屬於安靜狀態，你可以從左到右的順序進行忍殺，在最高的平台上，是有一個敲鑼打鼓的小怪，那個建議第二個殺，如果沒有擊殺你的位置將會暴露。殺完之後，來到二樓在山內典善的斜後方，在山內典善的旁邊還會有一個小怪，將他忍殺後就能對BOSS進行忍殺第一管血。切記在山內典善後方有一面旗子，千萬不能碰！！！一碰到他馬上就能發現。

接下來打山內典善的方法還是和赤鬼一樣，但是只能砍三刀第四刀的時候山內典善就能防住，而且山內典善的攻擊範圍極大，可血並不多，在燒他的時候一定要等到他揮刀之後在進行操作。同樣的打法對於很多BOSS都是有用的，殺完這個BOSS後玩家就集齊了三枚散落的佛珠了。

在前往主城的時候會遇見白素貞小姐姐，白素貞小姐姐雖然大，但是只要躲好了危險的就不是我們而是它。在鬼佛下來的地方有一片草地，從哪裡走到懸崖對面，穿過一個隧道還是一片草地，同樣只需要潛伏過去，到了草地的盡頭，玩家往前躲進一個轎子裡面，這就是為什麼小編說危險的是它了。它會把眼睛放過來，抓準時機插進它的眼睛然後拚命的往後面跑，穿過隧道就能到達葦名城主城外。

去到城門口會遇見一個騎兵BOSS鬼刑部，鬼刑部其實是比較難搞定的一個BOSS，在鬼刑部出來的一瞬間，是可以砍到4至6刀的，然後就一直防禦，當出現可以用鉤爪的時候就鉤過去，那時候可以砍兩刀，當自己的架勢條快滿的時候是可以砍4至6刀的，當血要空的時候，用爆炸進行干擾然後在砍幾刀砍滿他的架勢條，就可以第一次忍殺了以此類推第二條命也能怎麼打，但是這個BOSS與前面幾個不同，這個在空血的情況下需要忍殺兩次，有很多打鐵玩家好不容易砍死了鬼刑部，過於激動沒有進行第二次忍殺，導致需要從新忍殺。擊殺鬼刑部後會獲得一個戰隊記憶和一個可以用來升級忍義手的裝備。

對於鞭炮很多玩家都不知道在哪裡獲得，從虎口階梯往後面走，就是打第一個精英怪的方向，在一座山上有一個商人，在哪裡用500金幣就能買到爆炸。

以上就是從葦名城郊外到葦名城正門的所有BOSS和打法。如果大家有什麼補充可以在下方評論、留言。
