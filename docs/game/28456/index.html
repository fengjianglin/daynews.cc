魔獸懷舊服：要賺錢，就不能放過一切低級怪！

2019-12-23 1,094條評論 2,767次閱讀 827人點贊

目前懷舊服各種地精都頻頻出動就為了賺錢，其實原因也非常簡單因為下個月疑似第三階段就要更新了。不出意外的話到時候更新黑翼之巢，所以很多人為了開荒和進本都必然掃一大堆藥劑之類的等等。

但是在這個神仙頻出大家都認為好用的昂貴的才能賣錢，比如說什麼大家都知道的金珍珠這類材料。又或者什麼精華之類的？這類想必大家已經早就知道了。不過很多人對於到底去哪裡刷怪賺錢一直存在著疑惑，不知道到底去哪好。因為目前大部分已知的地方都很多人，甚至可能你跑過去還沒刷機個就被人亂刀打死了。所以推薦幾個刷錢地址。

首先是菲拉斯和塵泥沼澤這一帶，有著各種幼龍。這些幼龍有什麼特別呢？除了級別低基本在41~48級以外，最大的亮眼地方就在於它們都可以剝皮以及銀幣。這兩個基礎材料都是可以讓你一邊練剝皮一邊獲得基礎收入，當然重點是掉率有著10~13%的火囊了。火囊在目前而言唯一一個用處是用來製作火焰防護藥水，一般而言是小號去MC團打工或者混野的時候會買幾個防止暴斃。雖然一組大概就幾個金幣，但是勝在還有別的掉落以及兩個技能就能打死的快速。

其次就是精華了，元素精華可以賺錢這個大家都知道。大部分情況下目前的精華除了等元素入侵事件更新以外，基本上還是只能靠著野外打怪獲取。但是野怪的精華掉率基本上是個位數，如果你只是為了賺錢的話不妨去厄運東大海多斯伯恩，不僅不需要搶怪還有著11%的幾率掉精華，關鍵是不出的話還可以拼臉賭屠龍綱要。如果實在不行的話，那就去打東瘟疫算了，只要你不嫌棄無聊。

幼蟲酸液能幹嗎呢？能坐野性之皮這個57級紫色披風，提高42點治療效果，還是比較明顯的。除此之外，這個材料我也不曉得為什麼能價格那麼貴。雖說都是蟲子掉落的而且個位數的掉率，看著就很一言難盡。不過這玩意和元素精華不同的是瘟疫之地目前基本很少人過去，基本上你過去了還是沒人跟你搶，大部分情況下也沒太多人會去這邊也讓你基本不用太擔心被野外pvp打死。

然後就是鮮艷的羽毛，在未來暗月馬戲團也會更新。到時候暗月馬戲團是可以通過繳納這個鮮艷的羽毛來提高聲望值的，雖說目前一個羽毛扔商店大概也就幾個銀幣，但是說實話等到時候更新暗月馬戲團的消息出來以後，這玩意絕對就瘋狂漲價。最主要是鐵喙尖嘯者這個野怪有著29%的幾率掉羽毛平均3個就掉一個，即使不囤著拿取賣基本上也算是穩定的收入了。
