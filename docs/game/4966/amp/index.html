雲頂之弈9.24版本UI更新

另外遊俠也在持續加強

除了遊戲平衡有更新外

他在推特上說到

「雲頂之弈的用戶體驗設計師和大家討論了一下按鈕互換的問題

現在D鍵依然是刷新，F依然是購買經驗值

只是在UI界面上兩個位置互換了~

（上圖左邊為更新後，右邊更新前）

對於鍵盤黨來說可能不會有很大的影響

但是滑鼠黨就比較難受啦

說不定一個不小心就點錯了！

今日陣容推薦

目前版本也逐漸趨於穩定

今天小布給大家複習一下這套常規上分陣容~

推薦陣容：雲霄劍刺

推薦英雄：虛空遁地獸-雷克塞、疾風劍豪-亞索、永恆夢魘-魔騰、戰爭女神-希維爾、元素女皇-奇亞娜、無極劍聖-易、虛空掠奪者-螳螂、風暴之怒-迦娜

推薦裝備：希維爾推薦搭配颶風（斗篷+反曲弓）、沉默刀（斗篷+眼淚）、繳械刀（鎖子甲+斗篷）；夢魘推薦劍士轉職（金鏟鏟+反曲弓）、羊刀（大棒+反曲弓）、飲血劍（大劍+斗篷）

陣容羈絆：3刺客2沙漠2秘術2鋼鐵4劍士2雲霄

運營思路：

選秀優先拿鏟子，因為在這個陣容里螳螂變劍士是非常關鍵的，其次可以選反曲弓。前期用掠食者或者森林德魯伊過度，4、5人口可以加2星強力單卡例如皎月、夢魘。

因為這套陣容大部分都是3費卡，所以在6、7人口可以觀察對手拿牌多少，來決定要不要D。其次，陣容整體費用高，那麼就需要看裝備來決定誰追三星了。輪子媽主C、夢魘副C，如果夢魘的裝備不來也可以用亞索做副C，給肉裝輸出裝都可以。
