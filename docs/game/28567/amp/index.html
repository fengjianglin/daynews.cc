4AM小鹿決賽圈一穿四SMG，SMG的失誤在這個關鍵點

天天要聞
2019-12-23

黃金大獎賽F組的小組賽可謂貢獻了非常多的精彩鏡頭，其中第三局4AM戰隊和SMG戰隊上演了紫禁之巔對決，最終4AM兩人戰勝了SMG四人，決賽圈小鹿更是一穿四SMG讓人嘆為觀止，在這次黃金賽的小組賽中，SMG已經兩次被小鹿一穿三了，這次又被一穿四，果真4AM的4號位是SMG的剋星啊。

說了這麼多，我們再來理性地看一下，SMG戰隊在這個決賽圈的處理中，究竟出現了哪些問題，導致滿編優勢被4AM兩人逆轉，我們先來回顧一下完整過程：

在場上只剩下4AM和SMG兩隊之後，SMG滿編，4AM只有2人，人數上，SMG占絕對優勢，4AM有一個圍牆，SMG要出房區，在地形位置上，4AM佔小優勢。

由於要先動，SMG兩人封煙準備往圈內靠近，大京直接開車拉到圈內，這個時候已經形成了夾角槍線，怎麼看，4AM都沒有任何機會，但是大京在到點之後被永遠給擊倒了。

這個槍線沒有之後，4AM兩人直接拉出圍牆，一顆手雷往煙霧裡面去仍，正好SMG的兩人靠得比較近，直接一雷雙響，此時場面上瞬間變成了2V1。

此時永遠一馬當先，和堡壘進行1V1的battle，但是沒打過，永遠被放倒之後，堡壘決定把大京扶起來，在扶完大京之後，還準備扶第三人，結果小鹿一顆雷過來，補掉了倒地的人不說，還把剛扶起來的大京炸倒，堡壘也只剩下一絲血，最後煙散的時候，被小鹿一槍帶走。

SMG的四人當中，本來大京是被永遠帶走的，結果堡壘剛好把大京扶起來了，所以最終也成就了小鹿的一穿四。

看完了過程我們再來分析原因，為什麼SMG的這一波4V2會輸？

原因有三點：1大京開車進圈的策略沒有任何問題，但是點位沒佔住，這讓4AM能夠衝出圍牆扔雷；2小鹿一顆雷直接帶走了兩人，瞬間將2V4的劣勢打成了2V1的優勢；3堡壘扶人被炸殘血，血量不敵。

以上三點是我們都能看出來的表面原因和直接原因。但根本原因是SMG戰隊對於投擲物的理解不到位：

大京為什麼會被放倒？SMG為什麼會被一雷雙響？這就是煙霧的問題，如果SMG把煙霧封在4AM的一側，不僅可以干擾4AM兩人的槍線，還可以清楚的看到對方投擲物的軌跡，對方雷也扔不了那麼准，自己也不會踩雷，這是其一，如果封住4AM的槍線，大京也不會第一時間被擊倒。

最後堡壘為什麼要扶第三個人？確定了小鹿只有一個人，當時大京和堡壘扶起來之後，一左一右出煙包夾小鹿，基本是穩贏的局，想要扶第三人，就是在賭對手沒有手雷了，很不幸，賭錯了，SMG沒有把機會攥在自己手中，而是交給了小鹿，這一顆雷就讓他們絕望了。

這都是對於投擲物的理解不到位，我們平時玩遊戲時，都知道煙霧是保命的，但是決賽圈，煙霧是混淆視聽的，不僅可以混淆自己，也能混淆對手，決賽圈把煙霧封在自己這一側，無異於自斷雙臂，自蒙其眼，出煙都是劣勢，所以SMG戰隊在投擲物上的應用，確實還有待進步啊！
