魔獸世界：懷舊服要原汁原味嗎？關閉位面引起玩家的不滿

2019-12-16 501條評論 4,351次閱讀 3295人點贊

魔獸世界懷舊服，懷念的到底是什麼？其實魔獸世界剛一出來，那精美絕倫的畫面，那滑鼠鍵盤同時操作的新鮮，那開創先河的副本機制，那40人團隊同時副本的震撼，那野外pvp的緊張和投入，那現場pvp的策略和技術。這些內容在當時的遊戲環境中產生了驚人的魅力和吸引力，讓人耳目一新。

而反觀現在，不說其他遊戲的發展，就說魔獸世界本身的發展，我們並沒有看到暴雪對自己設計理念的堅持，而我們看到的是暴雪對於眼前利益的妥協。經典之所以能成為經典，是要用自身的理念去感染和同化別人思想。暴雪沒有堅持。現在的懷舊服，不過是再來撈一桶金罷了。想要找回當初的感覺和輝煌，已不可能。因為要回到過去的應該是玩魔獸的心情，而不是玩魔獸的內容。

記得最開始巫妖王之怒版本就是80年代，做任務會根據任務進度改變任務場景，比如我軍佔領一個據點，據點上的NPC和建築都發生了改變，想當年很強大別的遊戲沒法做到這點。還有比如遊戲BOSS有吸收，幻想，吃掉這些技能你會被傳送到另一個位面模擬以上狀態，再到不同伺服器不同大區玩家可以用位面系統一起組隊任務等等，想想看不管你朋友玩那個伺服器都可以一起組隊遊戲，這點非常人性化。最簡單的就是跨服組隊和交易，讓鬼服的玩家也有很好的遊戲體驗。

其實，懷舊服內容不變，有些東西優化一下還是不錯的，限時交易，Aoe摸屍體，幻化成就什麼的。該改的不改，比如陣營平衡，一方活躍角色大於55%就不準這個陣營建角色，這樣才能讓pvp血流成河，而不是漸漸變成單邊服變成pve。可以結合以前和現在的一些相對優勢的機制去稍加改變出一個新的懷舊服，而不是完全一樣的懷舊服。

有些改動是為了彌補當初的不足，避免引起遊戲外的糾紛，不影響遊戲平衡性和遊戲體驗的改動，相信大家都是歡迎的。主要是沒那麼多時間，40人團本，DKP制度，每天跟上班一樣，我覺得懷舊服打打戰場挺不錯，那時候的PK才是經典。世界在變，人也在變。以前的追風少年變成了孩子他爸。說魔獸不是以前的魔獸，那是因為人心在變。
