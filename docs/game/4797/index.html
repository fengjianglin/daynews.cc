無限進化自走棋：“銀幣大作戰”玩法介紹

摘要

前言：

「銀幣大作戰」作為休閑模式中的一種娛樂玩法，不僅能賺取銀幣兌換皮膚，又能「坐莊」「搞慈善」增加了遊戲的趣味性。在緊張刺激的排位賽過後，不妨來一把「銀幣大作戰」體驗盆滿缽滿的感覺。

模式特點：

①遊戲節奏快、時間短

②難度低、易上手

③銀幣獲取效率高

詳細分析：

這個模式通俗的來說就是賺取銀幣兌換想要的皮膚，可謂是眾多指揮官的福音吶！嘻嘻，我又能白嫖了呢！

首先在休閑模式中點擊「銀幣大作戰」了解一下吧！

模式分為：初級場、中級場、高級場3種不同場次，每個場次都需要達到相應的參與條件，花費一定的門票獲得相應的作戰基礎基金即可參與。根據個人實力瓜分相應場次的基金，據說第1名比較賺喔。你究竟是「送財童子」還是「賺錢高手」呢?我們拭目以待。

銀幣模式機制類似於快棋，不過對戰時間縮短了許多，時間就是金錢，效率就是生命吶！每回合同樣擁有免費刷新商店次數，切記不要浪費呦！卡牌刷新機制和排位模式相同。

進入遊戲後，會隨機給出8個英雄，根據個人需求挑選5個。並且右邊不在是血量而是改成了銀幣數量，這也意味著銀幣一但輸光你就出局啦，4人一局有點類似於鬥地主哈。對戰回合數為10回合，每一局會自動升級人口等級，達到10級後or所有對手銀幣為0時即遊戲結束，瓜分當前銀幣數量。

當玩家擁有的資金每超過10金幣時，可以在每回合額外獲得2金幣的利息，每回合最多為6金幣利息，但沒有連勝和連敗的戰鬥補貼。另外每局不再設有野怪回合，也不會有裝備掉落。對於陣容選擇，大家可以參考比較好成型的或者是容易上手的陣容。

看了以上的分析，你是否對銀幣模式有了些了解了呢？趕快打開遊戲去體驗一把銀幣模式的樂趣吧！賺個夠去兌換喜歡的指揮官、各種特效and英雄皮膚呢！
