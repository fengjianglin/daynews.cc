“遊戲X電視劇”！吃雞手游國際服迎來年度最強聯動，光子偏心了

大家好，又到了每日「鮮時刻」，本期為第433期！本欄目主打和平精英、國際服、體驗服、吃雞端游等伺服器的奇聞、趣事和技巧。廢話不多說，我們還是來看看最近的吃雞周邊，又有哪些新鮮、有趣、奇葩事，喜歡的可以先點個關注！本期話題，我們圍繞吃雞手游國際服來聊聊一些事。最近國服和平精英體驗服正在火熱更新中，而國際服同期也放出了驚人的消息，到底是什麼內容如此勁爆呢？

談到和平精英的「兄弟」伺服器，這個玩家早已聽聞的國際服，在版本內容上，一直都是稍落後於國服和平精英。但談到聯動、賽事活動這些部分，有充分的理由相信，國際服要甩國服好幾條街。

繼上次國際服與《生化危機2》進行了一波撼動整個遊戲界的大聯動，目前國際服再次攜手人氣美劇《行屍走肉》又來了一波「遊戲X電視劇」的跨界聯動。而上圖新角色皮膚就是《行屍走肉》中的Negan（尼根）。

當然，作為聯動也不會只有男角色Negan撐場。而女角色皮膚也由Michonne領銜登場，中文譯名也就是觀眾口中的「米瓊恩」。

這兩款聯動皮膚，不得不說，從人物刻畫的細節角度看來，效果堪稱1:1真實還原。在不考慮遊戲真實視角的情況下，游娛君看著就心動了一把。畢竟聯動這東西，可遇不可求。加上游娛君除了是一名吃雞發燒友，也是一名正在追《行屍走肉》的真愛粉。如果男主演瑞克和弩哥也登場，估計這次的聯動會帶來「爆炸性」效應。

不過話說回來，聯動也不可能只出兩個皮膚。因此游娛君也斷定，這只是光子在國際服暫時放出的「煙霧彈」，先放兩款角色皮膚作為鋪墊，後續說不定還會結合喪屍模式，再來一波內容上的聯動作為壓軸。習慣了國際服的更新模式，游娛君慢慢也得跟上了光子放「彩蛋」的節奏呀~！

除此之外，伴隨兩款人物皮膚的上線，他們各自使用的武器皮膚也跟著上線。這次《吃雞手游》X《行屍走肉》的應該算得上年度最強聯動了。有細節的還原，也有熱門電視劇角色捧場，光子真的有點「偏心」了。目前國服體驗服放出的版本內容比較局限，不知接下來和平精英又會給國內玩家帶來怎樣的驚喜？

對於國際服這次的聯動內容，你們又怎麼看？不妨在評論區留下你的看法，我們一起探討！原創手打不易，喜歡請點個贊、加個關注，冬日有你更溫暖！
