隨著新版本”巨龍崛起”上線時間越來越近，《爐石傳說》官方近日也是動作頻繁。近日，項目組就發布了一個令所有爐石玩家都心動的消息：牧師英雄泰蘭德只要登錄商店就免費送！不少玩家都表示，這活動真香！

#圖片來源於爐石傳說微博

除了這個令玩家們振奮的消息之外，爐石官方還發布了一項比較迷惑的改動，那就是商店和貨幣體系的改動。除了原有的金幣之外，新版本的爐石還將上架兩種全新的貨幣——符文石與奧術寶珠。從具體分工上來講，符文石是用來購買非卡牌包類的商品，比如英雄、冒險模式以及競技場入場券，也可以兌換奧術寶珠。而奧術寶珠則可以購買爐石卡包以及合集，金幣沒有變化。

#圖片來源於爐石傳說微博

要知道，如今爐石的金幣是可以購買卡包、冒險模式以及競技場門票三種物品的。玩家們過去只是對於金幣的獲取方式以及額度表示過不滿，對於目前的這種貨幣機制並沒有任何的不滿。但是在新版本當中，暴雪將原本屬於金幣的功能一分為三，那麼是不是就意味著，金幣以後只能用來買卡包，不能用來買競技場門票以及冒險模式了呢？

#圖片來源於爐石傳說微博

如果真的是這樣，那麼小編覺得一定會引起玩家們的強烈不滿。畢竟現階段的爐石傳說，玩家們只要足夠肝，即使不買預購包通過各種任務和競技場打金幣來賺卡包，也是能夠獲得足夠數量的卡包和組卡組的。但是，一旦競技場和冒險模式全都變成收費模式，爐石傳說的氪金成分將會大大增加，這顯然對於平民玩家們來說不是一個好消息。

#圖片來源於爐石傳說微博

如果金幣的作用保持不變，那麼暴雪費盡心機拆分原先的貨幣機制，只是為了降低玩家們購買流程的遊戲體驗？或者是專門為即將上線的希爾瓦娜斯做宣傳？小編實在是有些搞不懂。
