共由說《祖宗保佑》放置+國風元素觀感新穎 操作感不足成最大問題

天天要聞
6 天 ago

簡介

在近日的新游市場中，一款名為《祖宗保佑》的遊戲可以說是引起了不少小夥伴的關注。這款遊戲僅在試玩階段就已經攀升到了TAPTAP熱門榜第十位，評價更是已經破千。對於這樣的遊戲，我自然也是要嘗試一番。而在”並不簡單”的一番遊玩之後，關於這款遊戲的一些看法和評價也在我的腦中悄然匯聚，下面我們開始今天的共有說之路吧。

國風元素以及放置成現今不少手游的主題

實際上如今做放置和國風手游的廠家並不算少，幾乎每一周都會有幾款相關內容的遊戲面世。不管是與玄幻小說聯繫頗深的放置類修仙遊戲（放置修仙、武煉巔峰等）還是單純以國風為主題的《弈劍》，這一領域的遊戲在這半年內可以說是發展非常迅速。而按照這一市場的一般規律，一個遊戲類型的大熱勢必導致整個市場的作品質量良莠不濟。

當然在這一情況下，我們依然能看到不少廠商沒有去選擇玩”噱頭”玩”熱度”炒IP，反而是自己開動腦筋去尋找不同的元素進行創作。不得不說這一點比起08奧運年前後的不少遊戲廠商來說還算是進步了。國風趣味武俠遊戲里我們能看到漢家松鼠的努力，而《妙筆千山》也算是為不少廠家打開了一扇新的大門。

而正是這些廠商的共同努力，如今在國風以及放置類手游的版塊中我們看到的是越來越多的優質趣味作品出現，所以包括我在內的玩家群體對於這種情況自然更是喜聞樂見。就拿我們今天主要介紹的《祖宗保佑》來說，這款由ALFAGAME開發的手游以中華文化中的”家族祖先”為核心，用較為新穎的”放置”玩法為玩家再現了一個奇妙的架空時代。

“家族”這一傳統文化與放置遊戲的巧妙結合

那麼講完這一類遊戲的大背景之後，我們來著重談談今天的主角《祖宗保佑》。這款遊戲的玩法其實非常簡單，玩家進入遊戲後將隨機設定自己家族的姓氏以及”家風”（家風指一個家族的性格趨向，例如貪、怒、愛等等）。在設定完畢之後整個家族將開始模擬運行，玩家能做的只是在每4年一度的祭祖上對家族做出影響以及保佑指定的族人。

而家風這個設定算是這個遊戲的精華之一，在遊戲中玩家基本是做不出任何能夠實際上影響家族運作的操作的，只能通過”託夢””祭祖”等方式去改變這個家族的家風，而家族則會根據家風做出不同的反應，從而在一次次事件中生存。比如在上圖中，玩家家族中的”萬亞士”遭遇了沉迷煉丹的事件，但只有當”嗔”這個屬性大於22時才能實現加成。

而這一設定恰恰就給了這款遊戲非常多的趣味，因為就算玩家精心為自己家族設定了心中理想的屬性（仁愛勤奮等），但遊戲中偶發的事件卻會恰好沖著你的弱點來，最終的結果也是未知性拉滿。玩家能做的僅僅是作為一個祠堂里的”牌位”，在家族作出重要決定時進行干擾，在家風出現問題時進行託夢告誡。最終還是要看遊戲角色自己的行為。

那麼遊戲中玩家的目標又是什麼呢？其實大家也能猜到，無非就是把家族運作得好，代代傳承一直延續，有機會建立一個王朝什麼的。在一次次偶然事件以及家族的決定中，整個家族的實力也會進行變化。如果沒錢或者因偶然事件斷代甚至”搞事”起義導致滿門抄斬，那麼遊戲也就宣告結束。延續才是王道呀。

如果運氣比較好，那麼玩家可以不斷用子孫供給的香火去換來勢力聲望，這些聲望將在家族氣運積攢完畢之後發揮出巨大的能量，也就是”起義”。當玩家起義成功之後家族也就達到了最頂峰，剩下只需要考慮將王朝和家族延續下去即可。總的來說可玩性還是非常之高的，每一個選擇都會導致遊戲的結算不同。

而這種玩法相對於市面上的同類遊戲來說可以說是開闢了一種新的思路。”放置+國風”類遊戲的一般表現形式都是以修真、打怪為主，不少製作人跳不出武俠的坑。但這《祖宗保佑》毫無疑問就給了很多策劃思路也讓更多的傳統文化有了在遊戲中展現魅力的可能。這種創新的思考還是非常難能可貴的，整體遊戲節奏也非常好。

玩家可操作性不強成遊戲瑕疵，隨機性太高

但可能因為題材的限制，這款遊戲的”家風”設置決定了玩家並不能在遊戲中做出很多的有效操作。它不像武俠類放置遊戲，可以決定去哪裡刷裝備，去哪裡升滿級後再去下一張地圖闖蕩；它也不能自由選擇武學以對抗重重難關。所以這一設定在帶給玩家隨機的趣味性的同時，也讓不少玩家在一次次隨機事件中哭笑不得甚至直接關掉遊戲。

我們這裡可以舉一個簡單的例子。在《祖宗保佑》中，會有子嗣繁衍這一設定。每一代子孫都將擁有自己的天賦甚至會有”天命之子”的存在。但他們在遊戲中也會遇到各種各樣的事件，很可能玩家精心培養的繼承人最後落一個早早夭折的下場。而家族也可能不顧你的勸告進行分家，讓整個家族的人丁和財產極大地分割。最終氣得人砸手機。

所以落腳到這款遊戲的根本類型”放置”這一方面上來說，玩家的可操作性其實並不算高。這或許就是趣味玩法的雙刃劍，有瑕疵也是意料之中。

總體評價

《祖宗保佑》是一款可玩性極佳的遊戲，其趣味性與傳統文化所結合產生的家族隨機事件玩法是其魅力的重要來源；但另一方面，由於過多的隨機性以及刻意設置的玩家低干預機制，這款遊戲的操作感並不算太好。總的來說優大於劣，用瑕不掩瑜來形容也差不多。對於國風手游有興趣的小夥伴可以嘗試一下。以上就是本期共有說的全部內容啦。
