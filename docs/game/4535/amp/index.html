摘要

大家在入手新武將的時候特別是新上線的武將完全不了解他的強度如何，一般大家都會查閱武將的技能在腦海中設想他在各種遊戲環境中的表現，從而來評定是否是一個強力的武將。然而實踐才是檢驗真理的唯一標準，有多少人被自己的設想矇騙呢？

改動後看似強到爆炸實則弱到掉渣的武將

界夏侯淵——強度提升4分

我相信有很多朋友都想小編一樣被界夏侯淵帥氣的原圖蒙蔽的雙眼，如一人直面萬軍的霸氣，再看看他的技能介紹一回合三刀！那不得上天嘍！買買買！使用過後才發現掉進坑裡了。界夏侯淵確實能發揮三刀的高爆發，但是代價相當高，總共需要跳過摸牌、出牌並棄置一件裝備、跳過棄牌並反面。如果沒有能反面的隊友輔助，你很可能沒有第二次發揮的機會，得不償失，負面效果相當大。界夏侯淵的技能更適合場面靈活使用三種方式出殺，而不是破釜沉舟地搞一波大爆發。

界典韋——強度提升4分

界典偉說實話真心不推薦大家入手，雖然可以在一回合內對多人進行一個強制性地傷害打擊，但是標準典韋地兩個缺點也被完全地繼承過來。典韋和很多菜刀武將面臨著牌不夠用地問題，典韋更是刁鑽指定武器牌發動技能，武器牌不夠和血量不夠是困擾典韋一生地痛。界典偉可以在一回合最多砍七刀，但是他缺的是傷害嗎？不！他缺的是造成傷害的代價。如果能有包養流的隊友輔助外加一點歐氣，典韋的發揮不容小覷。

界小喬——強度提升3分

有些朋友看到這裡估計要給界小喬打抱不平了，界小喬可以把傷害設定成體力流失，對方獲得的牌由你給出，傷害反彈給對手不會出現弊大於利的情況。這確實是小喬改動的一大亮點，但是與此同時反彈的傷害設定成了1點傷害，無法再和李儒這樣高傷害技能的隊友配合，其次摸牌的上限也從原來的以損失體力值變成了最高五張，原來和董卓、兀突骨一把摸手牌的配合無法發揮全力。小喬更強了嗎？確實更強了，但她也變得更平庸了。

界袁紹——強度提升2分

界袁紹還沒上線的時候，許多人把他捧上了天，差不多就跟超神將一個地位了。理論上可以一次過牌七張，而且可以使用四次，一回合差不多獲得牌堆的四分之一。確實沒有說錯，但這僅僅是理論上的，在實際遊戲中，很難出現一波”萬箭齊發”沒有人受傷，就算有閃對手也會故意賣血影響你技能的發動。使用”亂擊”的門檻也有所下降，不在要求同樣花色的手牌，代價是一回合內最多發動四次。真不知道界袁紹算是增強還是削弱，多了一個實際上難以發動的技能，降低上技能使用條件，多了爆發上限，整體來看增強有限。

本期的界武將大盤點就到這裡，還沒入手的小夥伴趕緊避開這些坑，已經入手的小夥伴可以在留言區說下你的使用體驗怎麼樣，和大家分享一下。感謝您的閱讀，我們下期再見。
