揭秘：吃雞里最後兩人同時死亡會是平局嗎？光著腳真的不出聲？

天天要聞
2019-12-10

繼續有關吃雞的話題

把大型的汽車開到這個位置卡住門，然後跳上去就會出現在2層，應該是個bug,其實有時候用隊友站在門另一側也能達到同樣的效果。

​

像這種海港附近的大箱子是可以這樣踩著邊滑上去的。

樓房的建模其實是有bug的，比如我們上到2層站在這裡，蹲下來可以看到樓頂，哈哈

吃雞最近出了很多新模式，比如殭屍模式、冰雪模式。其中有一個是團隊賽，團隊賽開始時有20秒左右的讀取時間，這時候可以去中間拿走機槍和手雷，然後回到自己的陣地在倒計時結束前扔在地上，然後比賽正式開始時槍就在地上了。

之前有看過一些關於手雷和燃燒彈的傳聞，通過實測得到了些數據，比如這個，有人說如果被燃燒彈扔到了一直跳，騰空的瞬間是不掉血的，而蹲下或者躺下比直立行走傷害高，想想也有一定的道理，真實世界裡如果著火了肯定躺在火里燒傷面積大，而跳起來可能只是燒到腳。但經過測試發現不論什麼姿勢掉血都是一樣的。

再比如這個，我們以為手雷只要離敵人近就能造成傷害，但其實不是這樣，看這個實例，這個手雷被扔到樹的一側，另一側有兩個人，一個人被樹擋住無法看到手雷，另一個雖然離得遠但視野里能看到手雷，結果是看不到手雷的沒受到傷害，雖然感覺上這個人必死無疑，而另一個人受到傷害。所以說手雷的傷害是看人的前方有無障礙物阻擋的，有阻擋即使一片小的屋檐都不會受傷害，這點和CS一樣，多少次我以為要被炸殘了結果沒掉血，回頭一看原來地面有個小坑。

傳說光腳跑步沒聲音？其實聲音是一樣的

傳說在降落傘上的人打不死？其實不論是新模式還是老模式都能打死

傳說最後兩個人的時候你拉開手雷直到爆炸來同歸於盡結果是平局?親測結果是勝利。
