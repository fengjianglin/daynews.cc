但是我們第二期只介紹DNF的玩家充值情況

大家所熟知的《DNF》地下城與勇士

地下城與勇士已經不再是那個只要「勤奮」就能夠玩得很快樂的遊戲了。現在只有不斷的充錢去強化、增幅才能讓你在現有的裝備下得到提升，想要擁有一件高強、高增幅的裝備不充錢是不可能的。現如今的國服，你充的錢越多，你的裝備就會越好，想要變得更好，只有再往裡充錢。

遊戲充值榜第一： 旭旭寶寶 保守估計：一千多萬

旭旭寶寶，真名任怡旭，別名大旭、旭哥、寶寶，DNF國服第一紅眼，原龍珠直播簽約主播，現已與龍珠解約，於2018年7月16日正式加入鬥魚直播，首播熱度破4300萬人氣。

地下城與勇士資深玩家，國服刷圖競速第一人；使用狂戰士、武神賬號多次創造國服個人競速單刷記錄。

旭旭寶寶玩地下城與勇士的玩家心裡都有一個夢想，那就是去盜寶哥的賬號。寶哥的賬號可以說是一個鈔票粉碎機，就拿他的紅神來說，沒有一件裝備能下來幾萬塊錢，甚至幾十萬，特別是他的國服第一短增幅17的荒古遺精短劍。

DNF第一主播旭旭寶寶就在直播中透露，自己玩DNF花了差不多一千多萬，這幾年做直播賺的錢基本又投進去了，為此自己還被老婆罵個半死，不過錢雖然多但也還是無法在北京二環買套房。

遊戲充值榜第二： 土豪小偉 保守估計：一千多萬

他的裝備在土豪看來或許只能算一般，但在普通玩家眼裡，這樣的屬性已經是進入大佬梯隊了，再加上職業優勢加成，即便是漩渦本也足以拖動兩個醬油。站街3858智力，5.7W魔攻，不同於紅眼和劍魂這樣的土豪職業!

遊戲充值榜第三： 國服第一自戀狂 保守估計：800多萬

作為國服第一武極，帶領了一大波玩家進行秒秒秒，去年因為屯挑戰書的一個大手筆，直接賺了幾十萬，但是自己花費的，是賺的10倍還不止，現實中也是個超級有錢人，旭旭寶寶的好友，能用錢辦成的事，都不叫事。

國服第一自戀狂自己是開公司的，家裡十分有錢，多次斥巨資，讓寶哥幫他強化增幅裝備，看的水友們是一愣一愣的，有錢真的可以為所欲為。他的武極一腳可以毀滅阿拉德大陸，據說因為家裡生意上出了一點小問題，所以今年自戀狂把號30萬賣給了號商，充值好幾百萬的號，最後才賣了三十萬。

遊戲充值榜第四： 黑一阿旭 保守估計：700多萬

黑一阿旭國服第一個全職業盧克c的神豪玩家。每一個角色都有天空、13起步的武器，每一個角色的面板都可以與普通玩家的大號相提並論，可以說是達到了頂尖的程度。據說阿旭曾經上過心悅財富排行榜前三位，可見他充值了多少錢。號上不乏一些稀有天空，而且還不止一套。阿旭現在和寶哥一樣都在鬥魚直播。

遊戲充值榜第五： 土豪哥劍魂 保守估計：700多萬

雖然這張是地下城進圖後的面板，但是也是非常的驚人。三萬多的攻擊力，在當時那簡直是傳奇！要知道，有很多的平民劍魂到現在的90級版本攻擊力都不到兩萬的，人家早在8、9年前就達到這種程度了，真的是太豪了！

第二個欄目：當然還有我們之前的記錄 {DNF60級版本的時候那會土豪哥：}

1號土豪：凌晨4點

不知道還有多少老玩家還記得這個名字，當年DNF第一劍魂！看看身上的這些裝備，隨便拿出來一件在當時都是簡直連城的寶貝。站街力量能上一千的，當時整個DNF都找不出幾個來，連旭旭寶寶當時都達不到！

他身上的那幾件首飾，現在還認識嗎？

2號土豪：丶劍魂丶

雖然這張是地下城進圖後的面板，但是也是非常的驚人。三萬多的攻擊力，在當時那簡直是傳奇！要知道，有很多的平民劍魂到現在的90級版本攻擊力都不到兩萬的，人家早在8、9年前就達到這種程度了，真的是太豪了！

3號土豪：異様シ浩

這個玩家的面板圖找不到了，不過看看他強化的記錄，嚇死人！

骨戒，漩渦者之魂，王者之星，裂創，流星落，魔劍，無影劍…..

就算是現在也沒有多少人敢像他這樣強化的吧？何況是60版本，這上面的裝備哪個不能賣幾千塊人民幣的？

4號土豪：寶寶

旭旭寶寶當年這一身裝備，我猜百分之九十的人都說不上名字來了，因為拿的是無影劍不加力量，所以旭旭寶寶的力量在60版本沒有上千，並不是沒有這個實力，而是那個年代有把無影劍比多幾百點力量都強多了。

寶哥身上帶的稱號也是當年最貴的精·天下第一劍！

5號土豪：幾把劍

+17的朱諾星劍，骨戒、王者之星、墨竹，看看這個遊戲界面你就知道有多早了，我們還在4人組隊打殭屍的時候，別人已經單刷機械牛了。記得很多年前曾經看過他的視頻，刷起圖來真的好像是天神下凡，一個銀光落刃，整個阿拉德大陸都要為之一顫！
