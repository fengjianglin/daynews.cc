《戰雙帕彌什》比安卡.零度使用心得

天天要聞
2019-12-10

角色定位：遠程強力單體輸出、 (巨乳長腿大姐姐)

缺點：無屬性傷害、aoe能力弱

攻擊距離：

最遠感覺能有6~7米的樣子(4個比安卡的身高)，具體不詳，但是離得太遠進行普攻會打空氣，需要注意。

普通攻擊：

前兩下都是單根箭，連續普攻第三下會一個後空翻然後打出很多箭，因此站樁打不動靶，後空翻會使得比安卡與目標之間的距離越來越遠，因此偶爾需要主動拉近與怪的距離(太遠了打空氣)，個人推薦向前閃避一下，這樣不會打斷輸出節奏，但是如果閃避緊張，那就停止輸出往前移動拉近距離吧。

技能簡述：

消紅球：釋放單體傷害箭矢，釋放的時候會自動鎖定敵方，不需要注意人物朝向

消黃球：釋放扇形aoe箭矢，沒擊中一個敵人恢復1/2/3點能量(對應1/2/3消)，不過箭矢數量固定，飛行越遠越分散，不容易打到怪，有條件可以在離怪堆近的地點釋放

消藍球：加buff，使得攻擊額外附加箭矢

大招：只需60能量即可釋放(別的構造體大招大多需要100點能量)，單體傷害，一鍵釋放，自動追蹤目標，無需注意人物朝向

隊伍buff：當隊長的時候增加全隊10%暴擊傷害

核心被動：

每次3消增加2根箭矢(遊戲中對應的顯示是背上一根藍白色的光箭)，最多儲存6根，即需要進行3次三消，此時背上會顯示3根，如圖：

​

釋放大招的時候會把積攢的箭矢一起釋放出去，每根都會造成傷害。

比安卡的輸出思路：

消藍球增加普攻傷害，消黃球增加能量快速釋放大招(aoe傷害基本可以忽略…)，消紅球打出高額單體傷害，通過3消積攢光箭增加大招傷害

裝備搭配思路與推薦：

意識推薦：

最優先推薦 「每XX秒，下次3消進入超算空間」 的套裝效果。因為進入超算空間可以快速積累3消，積累被動光箭，從而增加大招傷害。

剩下的兩個意識位置推薦 「每消一個球增加X%全傷害」 (可提升大招、技能傷害)，也可以裝 「增加普攻傷害」 的套裝效果。

武器推薦：

用高星的就對了…

五星弓 獵隼E，特效是2米內沒敵人的時候增加10%暴擊率和暴擊傷害，大家可以用系統給的武器碎片去分解商店(如圖)換一個，分解商店很容易在前期被忽略，新玩家領完獎勵可以進去看看自己能換什麼，省著再花資源去研發武器了。

​

六星弓 雷米爾，肯定是最優選，特效能增加大招傷害，對於以大招輸出為主的比安卡來說，肯定是 「專屬武器」 的級別了。

操作心得：

非超算空間下，有3消優先3消，如果同時存在多個顏色的3消，則優先選擇能觸發QTE的3消，除此之外，優先選擇藍色3消增加普攻傷害

超算空間下，優先選擇能觸發QTE的3消，同時由於超算空間下，消除任意個球都算3消，因此可以優選選擇分布零散、位置孤立、難以形成同色3連的球進行消除，這樣既能觸發3消，也能給其它球形成同色3連創造條件。

關於鎖敵機制、以及如何切換攻擊目標：

普攻會自動鎖定一個敵人，沒法手動切換。

停止普攻到一定時間，鎖定會自動消除。

因此，如果要切換目標，方法是：停止攻擊，走向想要攻擊的目標，面對它，然後攻擊，這個敵人就會被鎖定了。 (視頻里有這樣的操作)

被鎖定的敵人，指向性技能會自動打到它身上(如消紅色球、大招等)。

其它小技巧：

大招動畫期間免疫傷害

大招動畫期間可以空下手來點擊QTE

由於比安卡是遠程單位，面對兩個怪，可以保持怪與比安卡成三角站位，使得怪都顯示在屏幕上，方便觀察它們的動作。(近戰經常打一個怪的同時，另一個怪就不在屏幕內，看不到了)
