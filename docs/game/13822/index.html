職業概覽

版本初期的職業整體勝率如圖。薩滿強度最高，按目前的趨勢，如果更多的薩滿嘗試放棄帶任務，那麼薩滿的勝率可能還會持續走高。

獵人雖然沒什麼新思路，但是在宇宙獵本來的框架上修修補補還是有很高的強度，這就是我們俗稱的吃老本，新卡補強了舊思路，但是並沒有成功開發出強勢的新思路。

說說倒數的四兄弟，我認為他們除了德好像真的沒救以外，另外三個職業都是還在開發的路上，經歷新版本的陣痛是無法避免了，除非你的新卡十分超模而且契合了舊思路（比如薩和獵就是兩個很好的例子）。

牧師倒數第一是因為他的迦拉克隆對比其他三張，對場面幾乎是沒什麼幫助的，但是牧師有三個流派等待開發，有一張不改的心火牧，有帶澤里克的克隆展的復活牧，有帶墓地符文和火車王斬殺組件的控制牧，首日還有兩套牧師登頂了呢，我相信在經歷卡組完善過程後牧師絕不是最差的那個職業。

騎士的勝率低的原因是沒多少人願意玩元氣騎（分奴除外），元氣騎的強度就擺在那，在現在薩滿卡組幾乎是掏不出來什麼直傷的情況下，擅長解場的元氣騎並不害怕薩滿而且這個版本目前來看甚至沒有出現高護甲職業的苗頭，元氣騎可以說是很好貼合環境的思路。

除了元氣騎，快攻形態的宇宙騎我認為也有開發的潛力，等到版本初期的新鮮勁過了，不在有那麼多人玩龍騎和職業騎，我相信騎士的勝率會慢慢爬升。

盜賊同理，我認為是構築還沒有開發完全，加上在初期遇到一堆海盜戰自然而然落下風導致了勝率的下滑。盜賊現在的思路現在無非是根據環境的整體節奏來調整思路，是迦拉克隆還是搶血的節奏賊，盜賊單卡的強度也在那擺著，加上喜歡玩盜賊的玩家有一大批，靜候卡組的完善就好。

德：沒救了，打擾了。新卡帶不出來德新的思路，龍體系顯得不是很厲害。而舊思路的任務德現在面臨迦拉克隆體系的衝擊顯得不是那麼超模了，還暴露出來德捉襟見肘的解場能力。我看了兩天的直播，感覺唯一有搞頭的德就是ray-c的諾米德（加基森低費法術刷牌的思路）。新卡保護甲板的獎勵三張爪擊對加基森的刷牌以及解場有一定的加強能力。等我看到具體的卡組我會在日報貼出來的。

HSReplay還總結出了新版本首日使用單卡之最，不多說了，正好一鍵分解出了bug，建議大家合理理財。

小結一下，這是在爐石歷史上為數不多設計師欽定體系真的很香的版本，就目前來看，迦拉克隆體系是真的超模而且強力，而作為對比龍體系就差了不少。前兩天的環境是薩滿英雄，但我相信後面會開發出越來越多的卡組能和薩滿一戰，現在外服很多人都在開發各種盜賊，大家可以移步至套牌廣場查看卡組，希望賊媽可以率先扛起抗薩的大旗。
