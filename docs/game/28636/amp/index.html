現在很精彩，未來更有趣——專訪《三國志14》製作人越後谷和廣

天天要聞
2019-12-22

感謝Koei Tecmo為我們提供了採訪《三國志14》的製作人越後谷和廣的機會，我們根據這款系列新作最近透露的情報，和過往作品上的一些特色，專門向他提問了幾個老玩家們會比較關心的地方。那麼越後谷製作人對這些問題作出了什麼樣的回應？趕緊來看看下文的Q&A吧！

●採訪對象：越後谷和廣（右）《三國志14》製作人

●Q&A

UCG：在《三國志14》最新的情報中提到，外交在本作當中很重要的一個元素，那麼請問AI的表現和外交關係之間會有怎樣的關聯？是和AI控制的勢力關係友好就不容易被攻擊嗎？還是說關係良好也會被攻擊？

越後谷和廣（後文以「越後谷」簡稱）：外交的作用，可以理解為和其他勢力關係良好的時候，會更容易結盟。至於AI的行為方式，是與遊戲中的各種因素都有關，不僅限於外交關係。所以即使是關係良好的時候也可能會被攻擊，但一旦結盟的話則不會再被盟友攻擊。

UCG：遊戲之前公開的情報中，提到想要使用外交中的「勸告投降「指令，則要求玩家不能被勸告的對手敵視。但歷史上也有很多通過武力使對方屈服的例子，那麼在本作中可以通過戰爭迫使對方和談嗎？

越後谷：沒錯，除了走外交路線，用武力讓對方屈服也是可能的，玩家可以在戰爭的時候選擇對敵人進行勸降。

UCG：本作當中武將會不會有自己期待的爵位？玩家是否需要在遊戲中去滿足武將對爵位的期待呢？

越後谷：有類似的設定，具體機制是這樣的：當玩家和武將關係變差的時候，或者武將的官職得不到滿足的時候，就可能會導致其忠誠度下降。但即使得不到滿足的武將，也不會因為嫉妒等情緒而導致不執行指令或不進行工作。

UCG：新情報顯示本作的自定義武將有很多要素可以設定，請問自定義武將的畫像是從默認的圖像庫當中選擇嗎？還是說玩家可以上傳自己的圖片來自定義武將畫像？

越後谷：玩家需要從圖像庫當中選擇武將的畫像，至於自定義畫像的功能，「現在」還沒有。（越後谷製作人在提到「現在」這個詞的時候刻意加重了語氣）

UCG：那麼本作當中會不會自帶地圖編輯器？

越後谷：很遺憾，本作不會有地圖編輯器功能。

UCG：《三國志14》之前在Steam上面推出了數據收集版DEMO，那麼在PS4平台上有推出體驗版的計劃嗎？

越後谷：鑒於遊戲的發售日已經非常臨近了，所以PS4不會再有體驗版的計劃。但遊戲上市之後我們也會繼續積極收集玩家的反饋數據和對遊戲的改進意見。

UCG：在遊戲當中，武將的個性里，會有一些負面效果，請問這些負面效果是否可以在遊戲過程當中消除呢？

越後谷：如果是說完全將其消除掉的話，那是做不到的，大家可以理解為那是武將性格的一部分。但遊戲中設計了與這些負面效果相對的一些正面效果，所以在實際玩的時候，玩家們可以嘗試將擁有互補特性的武將編隊在一起去行動，這樣負面效果會被其他武將的正面效果抵消。

UCG：最後確認一下，既然沒有地圖編輯器，那本作中會搭載事件編輯器嗎？

越後谷：就「現在」來說，遊戲當中還沒有事件編輯器功能。
