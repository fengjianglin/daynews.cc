摘要

老秦聊遊戲，每天一篇新鮮事。想必很多小夥伴都有過王者榮耀連敗的經歷，連敗是非常苦惱的，哪怕是在低端局都已經很難受了，更別提在高端局甚至巔峰賽。尤其是巔峰賽，連敗之後不知道要玩多少場才能打回到原來的分數。

想必很多小夥伴都有過王者榮耀連敗的經歷，連敗是非常苦惱的，哪怕是在低端局都已經很難受了，更別提在高端局甚至巔峰賽。尤其是巔峰賽，連敗之後不知道要玩多少場才能打回到原來的分數。

​老玩家都清楚，連敗之後最好的解決方法就是換模式來打，重拾信心，讓系統刷新並給你匹配給力的隊友。

但是絕大部分玩家都有著不服輸的精神和脾氣，孤影就是其中一位，哪怕連敗也要打下去，打到贏為止。日前，孤影的戰績就連觀眾都看不下去了，屢敗屢戰，而且幾乎每一場都是敗方MVP。觀眾讓他換模式來打，不要和系統對著干，但是孤影卻不信邪，覺得自己沒問題，不需要換。

​終於皇天不負有心人，時來運轉，在六連敗之後孤影匹配到了給力的隊友，而且連著三位都是國服在榜帶標的玩家。

如上圖所示，分別是國標嫦娥、國標大喬（頭像蓋住了）和國標孫尚香，就連楊戩都是省級的，孤影開了語音之後還發現，其中有一名還是自己的粉絲。

​遊戲還沒開始孤影就放飛了心情，覺得連敗六場之後，系統總算是放過了自己，匹配來了國服級別的隊友，這場可以放開了打，不用再這麼拘謹！

前面幾場就是因為打得太拘謹太謹慎，所以才畏手畏腳的，最終導致失敗。

國服在榜的隊友果然是有功夫的，才八分鐘就領先了敵方6000的經濟，其中孫尚香和嫦娥的數據比孤影還要好。

​孤影是越打越開心，越打越興奮，居然開始孤軍深入！但是哪怕自己一個人深入敵區，隊友也沒有放棄跟隨，大喬更是時刻注意孤影的狀態，準確的將回城機票放在橘右京的腳下。

或許是連續幾場失敗讓他憋屈了，這一場有了如此給力的隊友，孤影完全不帶慫的，多次果斷越塔！

讓人沒想到的是，國服大喬確實了得，看到孤影上前，自己作為輔助也完全不怕，放個圈圈就跟上去！

​觀眾也在感嘆，國服大喬和普通大喬真的是有區別的。普通大喬作為軟輔，都是躲在隊友後面放技能，只有會玩的大喬才敢主動上前抗傷害，並且利用圈圈回城。為啥我們就從沒遇到過會玩的大喬呢？

有了三大國標玩家護航，這一局孤影總算是贏回來了。

​不過各位小夥伴，大家可別學孤影頭鐵呀，要是連敗的話還是建議換個模式轉運，高端局巔峰賽的積分真的是很難才打得上去的呀！大家有同感么？
