航海王燃燒意志：當前版本刷劇情的主力陣容，各有作用

2019-12-15 230條評論 1,876次閱讀 398人點贊

諸君安好，霧夏菌報道。

PVP競爭激烈，普通玩家想要闖進去難度比較大，所以更多人會選擇主刷劇情，那麼這一回合就來說一下主線劇情方面的陣容。

1、輸出位

輸出角色目前有這幾位是優選，馬爾高、紅髮、新索隆。連續的輸出量與小回合的爆發都比較高，能夠應對大多數的主線劇情。最推薦的還是小馬哥，持續輸出量非常高。

之前也提過白鬍子，但實際上白鬍子主要提供傷害的技能是終結技一騎當千，單點爆發高，但是很難提供持續輸出。所以如果用白鬍子刷劇情，可以考慮帶上碎敵，有百分比的扣血機制。

另一個就是新烏索普，這個角色其實也能用刷劇情，傷害也不錯，也可以作為輸出位使用，如果能夠觸發終結技後的額外奧義，單回合的傷害量也會非常高。但他是優先血量最少，不太容易配合隊友集火。

2、肉坦或者半肉

優選：白鬍子、新索隆。

次選：甚平、卡普、戰國、喬茲

將白鬍子與新索隆放在了這裡，因為這兩名角色，在提供超高的輸出同時，也有著堪比純肉的坦度，白鬍子有中斷機制，能斷技能。新索隆站在前排有額外血量提升，配合自身的護盾坦度驚人。這兩個角色是比傳統肉坦還要好用的。

甚平、卡普、戰國、喬茲則是整體偏肉一些，其中甚平和戰國也能提供一定的傷害。喬茲則是依靠被動的反傷機制提供傷害，卡普屬於純肉，別指望他打傷害，不過這貨100級後是真的能抗，所以缺肉坦的玩家，不妨考慮一下，後期也是沒問題的。

3、輔助位

推薦：佳妮法、黑鬍子。

正常是優先佳妮法的，主線有一些任務戰鬥次數比較多，而且很多小怪、BOSS帶有負面效果，凈化治療就顯得非常重要，近乎是劇情必備。

再一個黑鬍子，能夠提供很多的增益、減益BUFF，可以降低敵方的傷害量，從而間接保護我方其他角色。在一些關卡中，可以考慮用黑鬍子，而且還能提供一些輸出量，但不推薦用他作為主力輸出。100級之前，黑鬍子的傷害量並不可觀。

因此，我們最後的劇情優選陣容是：馬爾高、白鬍子、紅髮、新索隆、佳妮法。備用：黑鬍子、新烏索普。

​菌說

實際只是單純的首發五人，並不足以應對全部的主線劇情，所以後續也需要多準備一些其他的備用角色。再一個就是上面提到的角色，只是比較推薦的，並不是說不可取代，也是可用其他的角色替代的，只不過相對來說，上面這幾位效果更好一些。

好了，這一回合到此結束，感謝閱讀，拜拜了您嘞~
