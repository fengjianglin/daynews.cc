魯班大師已經上線一個星期左右了，目前他在排位賽的出場率為2.4%，而勝率也僅僅只有35.64%，這個表現在新英雄裡面算是比較弱的了。不過這個英雄的上限和下限差距很大，孤影首次在巔峰賽2000分的對局中使用，便帶飛了隊友。魯班大師的操作相對複雜，而且很考驗配合，不太適合單排，所以35%左右的勝率是可以理解的。

既然魯班大師是一個上限很高的英雄，那職業選手是如何評價這個英雄的？一諾在接受的採訪的時候，被問到了魯班大師的強度問題，一諾先是笑了一下，可能覺得這個問題有點針對自己。隨後一諾表示這個英雄太坑了，他跟隊友三排了三局，MC玩的是魯班大師，結果直接連跪了三把，所以他才說大師很坑，老帥在一旁也表示同意。

不過一諾的話或許帶有開玩笑的成分，因為魯班大師的技能可以推人，這一點跟關羽比較像，所以在上線之前，不少粉絲都說一諾的英雄池裡又多了一員大將，還有粉絲調侃：「總決賽看一諾輔助魯班大師推人無敵，4保一愛思黑切宗師破軍張飛越塔五殺」。

個人覺得，一諾或多或少聽過類似的言論，所以他對魯班大師的評價不一定是客觀的。

另一邊，「老實人」刺痛也接受過採訪，談到魯班大師的時候，刺痛認為跟孫尚香搭配會有奇效，孫尚香可以一技能翻滾上前輸出，然後魯班大師再將她拉回來，輸出環境很安全。刺痛還表示跟黃忠、后羿搭配也很不錯，但跟馬可波羅就不太適合了。只能說刺痛還是太耿直了，決賽還沒打，他就把真實想法都說了出來。

近期Fly在直播的時候也提到了魯班大師，他對魯班大師的強度讚不絕口，認為非常有機會在總決賽登場。不過目前KPL的版本只更新到了11月20日，還沒有魯班大師，不知道在決賽之前比賽版本是否還會更新。如果魯班大師上線比賽服的話，相信會登上ban位。

個人認為，魯班大師在KPL的強度會非常高。不說一技能的推人，也不說二技能的瞬移，光是討論兵線運營，這個英雄就值得一個ban位。魯班大師一級便可干擾兵線，將己方兵線往回退，延緩對手的發育，推二塔、上高地、推水晶，魯班大師都可以加速推進，而他還能控兵，讓三路兵線同時到達，這都是職業賽場上非常需要的屬性。
