爐石傳說冬幕節合集20號上線，網友：白送的皮膚又丑又香

天天要聞
2019-12-20

在本月月初，爐石傳說官方就已經發布過關於冬幕節的相關信息了。活動時間從12月12號，一直持續到1月1號，時間算是比較長的了。而在20號，冬幕節合集就要正式上線了。大家從12號等到20號，是不是都特別期待這次的合集呢？

​

此合集也是限時出售的，時間從12月20日至1月7日，價格為1580奧術寶珠。使者認為，這次暴雪推出的合集，還是非常給力的。1580的奧術寶珠，就能獲得如下的卡牌包：女巫森林6包，砰砰計劃6包，暗影崛起6包，拉斯塔哈的大亂斗6包，奧丹姆奇兵6包。當然了，除了上面提到的這些卡包以外，還會獲得卡背圖案和德魯伊新英雄黑澤爾巴克。

​

說實話，當使者第一次看到這款皮膚時候，真的是被嚇到了。這竟然是暴雪推出的免費皮膚？這丑的也太令人印象深刻了。還有網友把托馬斯小火車P到上面，感覺毫無違和感，這看起來簡直就是托馬斯小火車的親兄弟啊。

​

下面就是托馬斯小火車，大家可以拿黑澤爾巴克來比較一下，相似度真的是很高啊。所以，使者懷疑暴雪設計師，是不是非常迷戀這部動畫片呢？但說實話，其實黑澤爾巴克也算是丑出新高度了，他現在應該算是爐石傳說裡面最丑的角色了吧？

​

我們再來看看爐石粉絲的對於這款皮膚的評價，估計暴雪那邊都不好意思看評論了。大家表示，這款皮膚丑爆了，讓人想自戳雙目。甚至有網友認為，這不像是托馬斯小火車，而是像兩個一大一小的電燈泡。大家也真的是腦洞大開，不過雖然都覺得皮膚丑，但是缺又覺得真香啊。因為有了這款嶄新的「丑」皮膚，「噁心對手」也算是效果拉滿啊！

​

不過總的來說，使者個人認為，冬幕節合集還是很值得入手的。雖然黑澤爾巴克皮膚有點丑，但那30個卡牌包它們不香嗎？畢竟只要158，過了這村就沒這店了。而且，有了托馬斯小火車的丑萌皮膚，用來噁心對手不覺得好玩嗎？另外，畢竟是絕版的限定皮膚，所以收藏價值也是有的。12月20號，托馬斯小火車皮膚等著大家領回家哦。另外新版本上線，超多免費卡包上線即送，泰蘭德英雄皮膚免費領取，小夥伴們還在等什麼？
