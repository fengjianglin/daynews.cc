在上周魯班大師終於登錄正式服啦，父子局終於圓滿，而官方也藉此炒作這一點。西施9月24日上線，魯班大師11月28日上線，中間空檔了兩個多月（一般一個月一個新英雄）。目前來說體驗服也沒有上新英雄，那麼下一個新英雄會是誰呢。

接下來將登場的三名新英雄猜測會是：蒙犽、鏡、王翦。首先是稷下星之隊的隊員，一共五人，孫臏、曜、西施、魯班大師、蒙犽。目前來說已經上課四位了，所以接下來最有可能登場的就是蒙犽。

首先這個英雄在官方設定中是一個15歲，身高150cm的小孩子，而且喜歡拿著火炮各種狂轟濫炸，妥妥的一個熊孩子。而且在背景故事中還特意強調了鄰城大將之子的身份，可見蒙犽的父親未來也有可能出現在遊戲中，算是一個伏筆吧。

定位是射手和戰士，這有點搞笑，這難道又是一個狄仁傑？特長是遠程消耗和團控，看起來有點矛盾，又有點有趣。生存、難度、技能指數都一般左右，但是攻擊力幾乎是滿的，所以是一個生存能力不錯的C位。

還有一個傳得很火的英雄就是曜的姐姐鏡，也就是傳說中王者榮耀裡面最難的英雄，比元歌還難，想當初元歌因為太難下線一年後才上線，那麼這個鏡的技能究竟是啥呢？定位刺客小編就知道與我無關了，但是這個英雄具體要等待多久還真不好說。

鏡是一個雙形態、四技能英雄。從分路站位介紹看出來有「鏡」和「鏡分身」這兩種形態。難度指數滿格，技能也幾乎滿格，攻擊過半，但是生存能力僅有短短一小格，可見它是一個技能非常難操作傷害可觀，但是身板非常脆的刺客英雄。

還有一個英雄可能大家不是知道，那就是王翦，定位戰士，之前也有曝光過相關技能。王翦是與白起齊名的戰國四大名將之一，其在王者峽谷定位戰士，王翦的被動與生命值掛鉤，即生命值越低，王翦的攻速和普攻持續傷害隨之提升，該BUFF最多疊加十層，另外王翦脫戰後呆草叢可回復3%已損生命值，這種自帶霸者的效果讓其續航能力提升了一個檔次，上單抗壓應該沒問題，據悉王翦的1技能和劉邦2技能一樣可蓄力位移。

另外兩款傳說皮膚即將優化上線，一款是李白的鳳求凰，一款是呂布的天魔繚亂，如果翻新上線將有大概率返場，尤其是鳳求凰，畢竟是一個賺錢的好機會。大家期待咩？
