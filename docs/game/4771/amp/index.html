1、傳奇怪物的AI值過低

AI，在遊戲中很多時候指的就是怪物的「智商」，要說遊戲中怪物智商比較低的，那傳奇絕對能算得上一個。傳奇中的怪物執行這麼一個套路「見你我上，你跑我追，你躲我跟，你死我停」。屬於各種遊戲中一個最簡單的套路，傳奇中怪物的智商遵循下列原則：

第一，怪物會優先攻擊那個先攻擊它的玩家，或怪物優先攻擊自己視野範圍內離自己最近的玩家。

第二，只要玩家不執行隱身操作或者下線，只要在怪物視野範圍之內，就會一直被攻擊到掛掉為止。

第三，怪物的「仇恨」判定簡單，傳奇中有一個仇恨值最高的技能，那就是道士的「施毒術」，反正筆者不知道遊戲設計者是怎麼想的，這種瞬時傷害最低的技能居然成為仇恨最高的拉怪神技……其次仇恨的判定就是單位時間內的攻擊傷害較高（這個單位時間大約在1-3秒不等），怪物就會轉向去追這個玩家。

引活怪就追

這三個「原則」使得傳奇中的怪物，都被賦予了最基礎的AI。就是這樣簡單的「套路」，使得玩家非常容易摸清楚怪物的習性，只要自身等級夠裝備夠好，就絕對能百分百戰勝怪物。不像有些網路遊戲，裝備只是通關的輔助，有良好的技術才是關鍵！（現在的網遊基本已經開始走這個套路了）

2、怪物AI值較高的幾個表現，傳奇都沒有或者AI不足

在很多遊戲中，怪物能分辨玩家隊伍中最弱小的那個，並且這個玩家會優先被怪物集合攻擊！此項設定多適用於遊戲副本，只要一隊玩家進入副本之後，遊戲系統則會直接分析玩家實力，牛一點程序的可以直接分析綜合戰鬥力，差一點的功能直接從HP上判定即可。即看誰血少先揍誰，這點絕對不會和遊戲本質衝突！因為誰都知道，一般遊戲中血量最低的角色，往往也是防禦最低，而且他們通常就是法師！傳奇作為這方面遊戲的鼻祖自然符合這樣的定律，看到這裡法師朋友又會覺得自己無端再次被針對躺槍……其實遊戲設定就是這樣，無可厚非。

法師卡位燒怪

另外，有些遊戲的怪物沒血會自己加血或者會逃跑，傳奇中的怪可沒那麼聰明，基本上玩家與怪就是死磕到底，只要能打得過，怪基本都是撲街「交貨」的。也就是因為這樣簡單的AI，傳奇到現在，也還是老玩家津津樂道並一直堅持玩下去的經典遊戲！
