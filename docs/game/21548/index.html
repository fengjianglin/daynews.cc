DNF：100級版本新平衡模式，三覺技能共享CD，玩家只能選擇兩種

2019-12-19 2,180條評論 4,911次閱讀 2866人點贊

韓服12.18體驗服，更新了100級版本，內容涉及的比較多。不過，玩家對於三覺技能，還是頗為好奇，畢竟韓服首發。值得一提的是，為了平衡三覺的職業過於強勢，技能實行共享CD！也就是說，一二三覺技能，玩家只能選擇其二。

​

韓服更新100級版本後，一陣雨就迫不及待的上線，為玩家展示女鬼劍士三覺技能。新增了一個被動，還有一個大招，以及三覺技能。當然，玩家也別擔心，技能欄不夠，已經額外增加了兩個。三覺後的女鬼劍士，都非常的帥氣，技能也超炫。而且，因新增了被動，加上主動輸出技能，在實力上，比沒有三覺的職業高。

​

韓服鬼才策劃，當初在冬季發布會上，就已經明確的表示，會進行一些「削弱」，讓三覺職業跟沒有三覺的，差距顯得不是很大。對此，一覺二覺和三覺，實行共享CD，三者只能選其二。不過，這個共享CD，是玩家進行選擇。比如，當你選擇三覺+二覺施放，那麼，一覺就會放不了。選擇三覺+一覺，二覺就放不了，選擇一覺+二覺，三覺就放不了！這個平衡方式，還是比較出色的，避免了三覺職業傷害過高！一般來說，三覺肯定是要保留的，畢竟傷害最高，而一二覺技能，玩家可以選擇捨棄一個。

​

不過，說實話，三覺職業雖然遭受限制，但新增了被動和一個大招，依舊傷害十分爆炸。在打團本的時候，玩家肯定優先選擇三覺職業，畢竟實力高一些。當然，等全職業三覺都來臨後，這個限制就會解除，三個覺醒技能，可以同時施放！

​

100級新版本的這個平衡模式，對其他的職業來說，實現了一些「公平性」，不至於差距過大。按照這個趨勢，在新一個團本希洛克上線，三次覺醒職業會全部達成。否則的話，一些未三覺的，那麼，希洛克團本都沒人會組，畢竟限制三選二，傷害仍然差距大！
