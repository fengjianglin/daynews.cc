遊戲推薦-既燒腦有可愛的回合制末日生存遊戲《活屍戰棋》

2019-12-06 0條評論 10次閱讀 0人點贊

Wanna Survive 活屍戰棋是一款極度燒腦的回合制戰棋遊戲。
遊戲結合了對抗殭屍以及回合制戰棋的玩法。
你將扮演主角阿火帶領一群能力各異的倖存者們，
在末日後的世界裡，想辦法殺出一條血路抵達倖存者的天堂「北市」。

遊戲的玩法非常好上手，教學模式也相當易懂。
基本來說，每個關卡都將由玩家先行。
在我方回合時，每個角色都能進行一次移動以及動作，
若是先執行動作，則無法移動角色。
動作則分為攻擊以及推開兩種，你可以在右下自由切換。
P.S. 若不想移動某些角色也可直接點右上選單後進行下一回合。

每個角色都有各自的移動距離以及獨有的攻擊模式。
像是醫生能使用手槍遠距離射擊或是近戰拳擊手大叔能藉由攻擊推開敵人等。
而推開則是用來移動場上的箱子或是幫助無法移動的角色掙脫。
當你把角色都動完一輪按下結束後，就輪到殭屍大軍的行動。

輪到殭屍後，他們會自動一個個朝最近的倖存者開始移動，
途中若是有箱子或是障礙物殭屍會直接對其發動攻擊。
當殭屍碰到倖存者後，你的角色並不會馬上損血，而是會被殭屍抓住。
除非有另一隻殭屍撲上來或是持續到下回合，角色才會扣血。

而就是這樣的抓住機制以及攻擊模式，讓你有了很多操作空間。
若是遠程角色被抓到，請安排隊友上前擊斃殭屍或是把角色推開。
若被抓到的是近戰角色雖說不能移動但依舊能進行攻擊，反手打爆殭屍。
因此請務必讓近戰殿後或是開路，讓遠程躲好啰~

成功完成關卡的任務後，玩家將能把地圖更往前推進並進入生存環節。
在這你必須分配在關卡中找到的食物給每位倖存者。
分配到食物的倖存者將能回復一滴血量並免於挨餓，
沒吃到食物的則會進入挨餓狀態，扣除一滴血量。
當然若角色原先只剩一滴血，則可以餓上兩回才掛這樣。

恩…提到了掛點對吧…這遊戲如同末日般的殘酷，
所有的角色死亡都是永久的，你將再也看不到他的任何故事。
因此非常建議初次挑戰的玩家遊玩輕鬆模式，
在這個模式下食物將會更好取得，讓你的角色能有更健康的血量，
但放心關卡中的難度依舊不變喔~
