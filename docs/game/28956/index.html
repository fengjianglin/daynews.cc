我還沒出力 你就倒下了？雲頂9.24召喚使陣容推薦

2019-12-20 239條評論 2,526次閱讀 102人點贊

相信大家都不會忘記

在雲頂S2賽季剛剛開始時

滿屏幕的劫唱愛河

打完直接換顯卡！

還有雙熊出征、一個掘墓的兒子暴打全場

滿屏幕婕拉的花和小螞蚱的畫面~！

沒錯~

今天給大家介紹的就是S2新增的羈絆召喚使啦

召喚使作為一個視覺效果爆炸的職業羈絆

棋子輸出主要依賴於召喚物的質量和數量

而且在一定情況下

例如鬼書、紅buff這類的特效裝備

召喚物是可以繼承裝備的！


今天就給大家推薦三套當前版本的召喚使陣容

除了視覺享受、質量也絕對不低哦！

在目前的職業羈絆排名中位居第二

刺客召喚使

推薦英雄：山影之焰-奧恩、沙漠皇帝-阿茲爾、元素女皇-奇亞娜（海洋）、眾星之子-索拉卡、虛空掠奪者-卡茲克、牧魂人-約里克、喚潮鮫姬-娜美、影流之主-劫

推薦裝備：約里克-冰拳（鎖子甲+拳套）、救贖（腰帶+眼淚）；娜美-鬼書（大棒+腰帶）；劫-復活甲（大劍+鎖子甲）、無盡（大劍+拳套）、光轉職（金鏟鏟+斗篷）

組成羈絆：3召喚使3刺客3光2秘術2海洋2雷霆2沙漠


陣容解析：

這應該是難得的7羈絆陣容了！當然這肯定不是小布亂寫的陣容，在許多的數據網站里，這一套都是S陣容！這套陣容以奧恩和約里克作為前排，風女、奶媽湊齊秘術做團隊輔助，而劫、奇亞娜、螳螂三個刺客可以快速切掉對面的後排。

如果運氣不好沒有遇到海洋地塊，在雲霄地塊里用雲霄奇亞娜，用風女替換娜美湊2雲霄羈絆也是可以的。如果有除了推薦裝備以外的輸出裝都可以給到螳螂或者奇亞娜，特別是奇亞娜~說她是一張超模的卡也不過分。

影召喚使

推薦英雄：虛空先知-瑪爾扎哈、永獵雙子-千珏、深海泰坦-諾提勒斯、黑暗之女-安妮、牧魂人-約里克、殤之木乃伊-阿木木、無極劍聖-易、喚潮鮫姬-娜美

推薦裝備：瑪爾扎哈-鬼書（大棒+腰帶）、青龍刀（大劍+眼淚）；千珏-帽子（大棒x2）、大天使（眼淚x2）；娜美-鬼書（大棒+腰帶）

組成羈絆：3影3召喚使3地獄火2守護神2秘術2海洋



陣容解析：

這套陣容以千珏和瑪爾扎哈作為核心，其中千珏的藍耗非常低，傷害十分可觀，且千珏的技能自帶80%減傷，也就省去了鬼書和紅buff這兩件裝備。大天使能讓千珏更快的啟動，而帽子也能大幅度的提高傷害。

瑪爾扎哈的小蟲子也可以附帶影buff的特效和鬼書效果，而且追三星的難度不會很大，加上青龍刀能更頻繁的召喚小螞蚱。

魔法召喚使

推薦英雄：荊棘之興-婕拉、虛空先知-瑪爾扎哈、沙漠皇帝-阿茲爾、邪惡小法師-維嘉、黑暗之女-安妮、復仇焰魂-布蘭德、牧魂人-約里克、影流之主-劫

推薦裝備：安妮-鬼書（大棒+腰帶）、大天使（眼淚x2）；劫-復活甲（大劍+鎖子甲）、離子火花（大棒+斗篷）、魔法師轉職（金鏟鏟+眼淚）

組成羈絆：6召喚使3法師3地獄火



陣容解析：

這套陣容就是傳統的魔法召喚師啦，相比前2套強度稍微低一些，而且需要金鏟子。不過在9.24版本，6召喚使能增加120%的召喚物的生命值和持續時間，還是值得一試的。

因為安妮在召喚出2隻提伯斯且其中1隻陣亡的情況下，將不再能獲得法力值了。所以這次的魔法師轉職優先給到劫身上，搭配復活甲和離子火花還是輕鬆分半個屏幕的身的~
