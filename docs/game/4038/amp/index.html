王者模擬戰：喜歡“打假賽”的英雄，沈夢溪太愛搶殘血人頭

1 週 ago

摘要

各位模擬戰的小夥伴們大家好，在《王者模擬戰》中，所有的英雄對戰都是他們自己在打，包括大招、連招、走位等等，玩家們是操作不了的。這就讓有些英雄在場上的表現有點“打假賽”的嫌疑，明明可以很快的解決戰鬥，他偏要不走尋常路，結果害己不說，還讓團隊整體失去優勢，真的是讓人恨得牙痒痒。

各位模擬戰的小夥伴們大家好，在《王者模擬戰》中，所有的英雄對戰都是他們自己在打，包括大招、連招、走位等等，玩家們是操作不了的。這就讓有些英雄在場上的表現有點「打假賽」的嫌疑，明明可以很快的解決戰鬥，他偏要不走尋常路，結果害己不說，還讓團隊整體失去優勢，真的是讓人恨得牙痒痒。那麼這些英雄都有誰呢，下面的內容，咱們就來一起聊一聊吧。

百里守約

百里守約是長安守衛軍中很容易出的射手，本來是可以當作陣營中的寶貝疙瘩的，但無奈這貨就喜歡打假賽。前方隊友打的火熱，他在後面一個勁地瞄，瞄半天也就算了，最後還老是打不中。有這時間多來幾次普通攻擊不好嗎？非得整的自己很牛似的，結果前面抗傷害的掛了，自己像小雞一樣被對手幾個英雄虐。哎，有這樣的操作，難怪只能當各個陣容中的掛件了。

沈夢溪

接著來說沈夢溪，這個英雄的大招還是很強的，往對手人堆里扔一個，可以他們造成大量的範圍傷害。但在實戰中，這傢伙釋放的大招怎麼就那麼愛搶人頭呢？並且還老是扔給那些殘血英雄，看的你是真心難受啊。要不是有和周瑜、不知火舞的CP天賦，鬼才想用他上分呢。

明世隱

明世隱在這裡的定位依然是個輔助，但他的技能卻老是把自己往輸出身上靠，上手就要和對面傷害最高的英雄牽紅線，在降低他物理和法術攻擊的同時，還要竊取他25%的屬性強化自己。你說作為個輔助，你要那點屬性幹嘛使啊，實在是想不明白。

花木蘭

花木蘭在峽谷中的表現還是很強悍的，會玩的經常粘的對手跑不開。但在模擬戰中，花木蘭自己的表現就有點太假了，每次出招都是在假裝蓄力，然後可能會用眼神告訴對手趕緊跑，接著冷不丁地給你放出大招，表演完這一套後，接著再去敵方英雄身邊蹭，也不嫌不好意思。

不知火舞

不知火舞在模擬戰中還是很厲害的，每丟一次扇子就會翻滾一次，對近戰英雄來說，碰到她基本都是被放風箏的。但這麼強勢的英雄有時候也會打假賽，那就是場上只剩下兩個不知火舞時，她們就會對著「自己」互丟扇子，但丟出去的扇子都會因為自己翻滾而放空。這樣的畫面一直會持續到本場時間結束，也不知道她倆會不會笑場。

好了，暫時就聊這5個英雄吧，如果各位小夥伴在遊戲過程中還遇到過哪些打假賽的英雄，歡迎評論區留言，說說他們的故事吧。
