用Python做個美少女大戰小怪獸

天天要聞
2019-12-13

對於飛機大戰，想必大家都一定不會陌生，但是對於美少女大戰小怪獸呢，大家是否玩過呢，今天小編就帶領大家從頭開始，製作一個《美少女大戰小怪獸》的初級版。之所以是叫初級版是因為這個遊戲只是實現了最為簡單的功能，後續的大家還可以繼續添加。

1

遊戲的界面設計

本次遊戲的編寫是基於pygame函數庫進行的，首先我們來看一下遊戲的界面，當我們運行程序後，會先彈出來一個前言界面，簡單的提示我們應該如何玩這個遊戲，然後我們便進入了遊戲的主界面，如下圖所示：

2

整個遊戲的角色，場景設計

接下來就是面對整個遊戲的角色，場景設計了，遊戲的設計主要分為四個部分，分別是四個類，對應著怪獸、美少女戰士、子彈和整個的遊戲。

我們挑選出一個類的函數來為大家稍作解釋，作為面向對象的編程，怪獸應該有什麼屬性和動作呢，針對於這個遊戲，小編想到它的屬性有位置，速度，是否存活已經剩餘的血量，外加被破壞時的樣子，而他的動作無非就是上下方向的運動。這樣一來，一個清晰的怪物形象就展現在大家的面前，怪獸類的程序如下圖所示:

上圖的程序中，小編按照上面我們所描述的定義了怪獸的血量、位置等屬性，並定義了其運動的函數。其他的例如子彈和美少女戰士都是按照類似的思想去設計，大家可以對照源碼去理解，這對於鍛煉大家面向對象編程能力的提高很有幫助。

3

整個遊戲的邏輯設計

然後就是整個遊戲的邏輯編寫了。這也是將各個對象整合到一起的關鍵點。在遊戲類的初始化定義里，如下圖所示:

小編定義了遊戲界面的帶線啊哦，遊戲界面的題目以及背景圖畫等信息，並實例化了美少女戰士、怪獸和子彈。

遊戲的主邏輯是這樣的，當我們運行程序後，pygame通過監測鍵盤的按鍵來判斷玩家對於美少女戰士的移動方向，與此同時，子彈被不斷的發射出去，程序如下圖所示:

當子彈打中怪獸時，此時子彈就失去了生命，同時怪獸也失去了生命，他們都被丟棄掉了，然後怪獸和子彈便被重置，我們的得分會加1000分，如果怪獸打中了美少女戰士，那麼怪獸和美少女戰士都被重置。

私信小編01 領取完整代碼

同時玩家的生命值也會減1。我們不斷完成這些邏輯判斷的同時，pygame也不斷的將子彈、怪獸和美少女戰士畫到界面上來，就構成了我們的一次循環。

當我們完成上面的操作後，程序便會不斷的再進入這個循環當中來，直到玩家的生命值變為0時，此時，便會出現下圖的界面。

程序會告訴我們的得分情況，然後提示我們是否要重新來一局還是直接退出該遊戲。如果我們選擇重新再來一局的話，那麼程序便會像我們前面所講的那樣，重新進入循環執行，直到我們想退出遊戲。

以上就是小編帶領大家對於遊戲的梳理，下面就讓小編為大家帶來展示吧，為了增加遊戲的體驗，小編還特地為該遊戲找到了不錯的配音，如果大家覺得配音不喜歡的話，可以自己選擇自己喜歡的配音進行更改即可。
