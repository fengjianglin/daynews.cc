共由說《閃之軌跡4》：粉絲狂歡 歷代主角高清出場 情懷真香

2019-12-10 1,162條評論 4,116次閱讀 2326人點贊

《閃之軌跡》系列雖然是《空之軌跡》系列的續作，但是並沒有像《空之軌跡》那樣優秀的口碑，再加上《閃之軌跡》系列的第三部和第四部沒有登陸PC，所以很多玩家對於《閃之軌跡》後兩部作品並不了解。之前也寫過《閃之軌跡》系列的前三部作品的個人測評，整體感覺下來雖然《閃之軌跡》系列的劇本質量不如從前，但是Falcom對於遊戲技術力的革新還是可以令玩家滿意的。

本人作為一個忠實的軌跡系列粉絲本來是中文版首發的時候入的《閃之軌跡4》，但是由於中間個人原因，最近才打通了《閃之軌跡4》。所以這裡也寫一個稍微遲到的個人對於《閃之軌跡4》的看法，希望可以和各位軌跡學家一起討論。

可控角色多 主角團人物刻畫不夠鮮明

《閃之軌跡4》應該是歷代作品中可操控角色最多的一代，舊七班新七班再加上各種協力者的加入都讓玩家玩的十分過癮。隨著劇情的推進，玩家可控制的角色越來越多的確是不錯的體驗，這樣玩家對於陣容的配合調控就更加自由，而且由於有考驗箱的存在，玩家也需要不斷的給這些人物更新裝備和迴路。而且人物的不同特性也使得零驅流，一刀流，閃避流等等玩法的出現在一定程度上增加了遊戲的可玩性，但是隨著角色的增加，不得不承認的是過多人物的出場很容易就導致對某些冷門人物的出現，以至於有些人物在最終Boss前我根本沒有怎麼使用過。

同樣伴隨著超多的可控角色，《閃之軌跡4》中對於人物的刻畫很不鮮明，新七組還好，但是到了舊七組以及其他協力者的時候，Faclom並沒有用太多的故事劇情去交代角色的轉變。個人感覺《閃之軌跡4》在人物刻畫上比較失敗的就是杜芭莉。從前作的感覺來看杜芭莉是一個帶有傲嬌且忠於主人的劍士，之前好歹也擊倒過劍帝維萊武力也沒什麼問題。而到了《閃之軌跡4》有點莫名其妙的加入了主角團，而且加入了主角團的戲份還少的可憐，其實Flacom可以花一些篇幅去講述這些角色內心變化的故事，可以是內心獨白也可以是回憶，至少讓玩家知道角色的想法而不是強硬的告訴你這個角色歸你了這麼簡單。

但是不可否認的是《閃之軌跡4》中給聖女或者是傭兵王這樣的反派都有不錯的刻畫，可能最後有強行洗白的嫌疑，但至少他們講述的故事可以讓玩家滿意。

遊戲系統更加人性化

之後再來聊聊《閃之軌跡4》的戰鬥系統，雖然《閃之軌跡4》的戰鬥依舊是圍繞著勇氣點數以及破防展開的，但是Falcom在《閃之軌跡4》中優化了勇氣點數（將最大5點改成7點），指令系統的平衡性等等，這樣都使得戰鬥的策略性有了不錯的提升。

但是由於《閃之軌跡》系列取消了之前《空之軌跡》中用迴路的元素點攢魔法的設計，很容易出現某些過於Imba的迴路，再加上Falcom一貫的數值不平衡”里一刀”的問題始終也沒有解決。我是最高難度通關的，可能在遊戲前期玩家會對不少戰鬥感到苦惱，但是遊戲進入後期Bug迴路使Boss戰依然變成黎恩開鬼化→里疾風。

在戰鬥系統之外Falcom還為《閃之軌跡4》的成就系統潤色了不少，很多可收集物品都會在地圖切換頁面顯示，這樣就會方便玩家確認收集品是否存在遺漏。而且在歷代最坑的寶箱成就上，Falcom還特意添加了開啟寶箱數量，這樣就可以及時的幫助玩家找到自己遺漏的寶箱。

最後再說說小遊戲系統，VM卡牌和波波碰其實作為小遊戲的出現都很不錯，Falcom平衡了一下卡片的模型。而且從《碧之軌跡》歸來波波碰的難度也恰到好處，敵方AI足夠強大，但又不至於毫無勝算。唯一想說的還是小遊戲的場合，在自由活動日玩玩小遊戲沒什麼問題，但是哈梅爾異變，你們在人家維萊的墳頭打牌是不是有點過分？

老掉牙的救場 爛俗的GAL情節

《閃之軌跡4》作為帝國篇終章的故事還是令我滿意的，雖然騎神的出現有點讓人咋舌，但是Falcom能自己圓回來也就不多說。這裡想要吐槽的還是那些老掉牙的救場。基本每個章節的最後都是一場大戰，而大戰過後，主角陷入困境，這時候就是公式化的救場：”沒那個必要！”之後救場的角色趕來，主角團突圍成功。其實這樣的橋段出現一次兩次還可以，但是一直出現是不是有點太蠢了？

而且由於自由活動日的出現，GAL情節成為了《閃之軌跡》系列很重要的一環，我並不是鄙視在JRPG加入GAL元素，畢竟人家P5做的也蠻好的。但是Flacom卻把養成環節做的很爛，顯而易見的就是黎恩這個主人公的輕小說化，女角色=後宮，而且幾個女角色的GAL劇情並不是相互影響，也就是說黎恩這個渣男剛和別人有過心靈交流後面就又和艾麗莎抱在了一起….真的希望後面的作品中Falcom不要再弄這種養成元素了，安安心心把感情放到主線上不好么？

老玩家的情懷

雖然上面也說了一些《閃之軌跡4》的缺點，但是我依舊認為《閃之軌跡4》是《閃之軌跡》系列作品中最棒的一部，不論是Falcom對於新遊戲引擎的掌控力，還是劇本的描述都有了不錯的提升，但是最令我喜歡的還是——情懷。

很簡單Flacom知道自己的玩家想要什麼，從《閃之軌跡4》的開頭就可以看出，《閃之軌跡4》徹徹底底變成了一部粉絲向的作品。小艾一家羅伊德一行人在開頭就亮了個相，看著曾經的三頭身小人變成了真實比例的建模，曾經的各種戰技都在次世代高清畫面的加持下演繹的淋漓盡致，這就是對於粉絲最好的遊戲。

而且Falcom在《閃之軌跡4》中還為老玩家們帶來了不少的驚喜，基神的回歸，安頓大神的戀愛故事這些前作中活躍的小人物的出場都很容易讓玩家回想到之前玩軌跡作品時候的感覺。老角色一批一批的歸來，之前Falcom挖下的坑也都不斷的在被填上，這樣的《閃之軌跡4》真的很過癮。

總結：《閃之軌跡4》天下第一 粉絲必如情懷之作

《閃之軌跡4》在我的心中應該是Falcom《閃之軌跡》系列作品中最優秀的一部，劇本雖然有輕小說爛俗的橋段，但是整體劇情並不拖沓，對於戰爭的刻畫也比《閃之軌跡2》優秀了不少。而且依靠著PS4的機能，Falcom給玩家們呈現的更加精緻的建模以及花哨的技能特效都足夠讓一票老玩家滿意。再加上前作主角們的加入也都讓《閃之軌跡4》這個大團圓結局更符合玩家的心意，因此個人十分建議粉絲入手。

最後說一句《閃之軌跡4》對於新玩家極度不友好，新玩家在看劇情的時候總是一臉懵，所以總能看到玩家問《閃之軌跡》從哪玩？個人建議有機會的話從《空軌》補，沒時間的話就從《閃之軌跡1改》開始玩。
