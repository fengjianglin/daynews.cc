盧本偉時隔六年重返賽場，打出全場最豪華戰績，幫助隊伍取得勝利

2019-12-19 663條評論 3,678次閱讀 2159人點贊

一提起被封殺的鬥魚知名主播盧本偉，相信很多小夥伴都知道。

在從事直播行業以前，其實盧本偉也是一位LOL的職業選手。

在2013年的時候，盧本偉所在的老皇族戰隊出征S3世界賽，一路過關斬將。

最終卻在決賽倒在了SKT戰隊的手裡，遺憾地只拿到了一個世界賽的亞軍。

也正是在當年的採訪當中，盧本偉面對鏡頭被問及Faker選手，他漫不經心的一句：

「我和他五五開吧，訓練賽我也經常單殺他的」，讓他至今還有一個「五五開」的綽號。

慘遭封殺之後，盧本偉作為一個前LOL的職業選手，卻對Dota2這款遊戲產生了濃厚的興趣。

之前就有不少的消息傳出：盧本偉痴迷於Dota2，不惜和女友鬧矛盾也要專心致志地上分。

沒過多久，又有消息傳出，盧本偉即將出資組建一支Dota2戰隊，開始征戰Dota2發展聯賽。

戰隊的選手陣容那也是相當豪華，兩個Ti冠軍加一個天才一號位選手，實在讓人無法不看好。

然而在12月16日舉行的發展聯賽當中，盧本偉戰隊當中的一號位選手由於家中有事，不能如約參加比賽。

身為戰隊老闆的盧本偉便決定向王思聰學習，自己挂帥親征，率隊迎戰擁有Ti亞軍選手uuu9的颱風戰隊！

時隔六年之久，誰都沒想到盧本偉會以這樣的方式再次出現在職業比賽的舞台上。

不過可惜的是，盧本偉這次的參賽項目已經不再是英雄聯盟，而是Dota2。

首場比賽當中，擔任一號位的盧本偉掏出了敵法師，多次打出精彩亮眼的操作。

最終敵法師的擊殺數冠絕全場，幫助自己的You Know Who戰隊率先拿到一分。

隨後的第二局比賽，盧本偉依然保持著火熱的狀態，憑藉著一手絕活火貓。

砍下了13殺的豪華戰績，再次幫助隊伍拿下一分，最終YKW以2比0輕鬆地戰勝颱風。

首次登場Dota2的職業比賽，還是和眾多的Ti明星選手同場競技。

盧本偉就能打出如此優秀的表現，這強大的遊戲天賦著實讓人有點佩服。

但新賽季的發展聯賽開始之後，盧本偉戰隊的成績並不樂觀，目前以2勝0平3負的成績排在倒數第二。

如果想要有更好的成績進入到之後的淘汰賽當中，盧本偉的戰隊得加把勁了。

時隔六年之久，盧本偉再一次重回職業賽場的舞台，著實還是讓人有些唏噓。

可惜的是發展聯賽也都是線上比賽，沒有露臉，大家也就無法看到狒狒式的招牌笑容。
