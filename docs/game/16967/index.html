王者榮耀KPL決賽飛牛極限操作，團戰狂鐵一打三，粉絲：永遠相信Fly

2019-12-15 1,365條評論 2,626次閱讀 1183人點贊

王者榮耀KPL秋季總決賽正在緊張刺激地進行當中，第一局AG憑藉優秀的運營兵不血刃拿下比賽，第二局QG憑藉Fly的高光操作扳平比分，趕緊來看看都發生了什麼吧！

第二局QG在紅色方BP劣勢的情況下，拿下婉兒和橘右京挽回陣容劣勢。開局1分鐘，Fly狂鐵中路配合隊友秒掉盤古，拿下一血，順勢拿下暴君。接著雙方和平發育，AG超玩會5分鐘一諾配合盤古擊殺Fly狂鐵後又抓住770盾山操作失誤集火秒殺，拿下先知主宰，拉回一定局勢。

在接下來團戰中，QG婉兒飛天打滿傷害，拿下兩個人頭，馬克進場收割，0換5直接推掉AG中路三塔，經濟一路領先。後AG超玩會愛思蘇烈大閃擊飛四人，0換3擊殺QGhappy中射輔，推掉中路高地，拿下黑暗暴君，經濟反超2千，QGhappy陷入劣勢。

在最後暴君團的時候，Giao哥馴龍高手回歸搶下黑暗暴君，在buff加成下，團戰完成反撲。雖然一諾已經有了復活甲，但Fly狂鐵天秀操作，不死戰神人群中一打三還切死一諾成吉思汗拿下人頭。在馬超有望收割的情況下，飛牛的大開了出來，簡直讓人絕望。最後QG2換5團滅AG超玩會，中路一波推掉了水晶拿下勝利，追平比分。
