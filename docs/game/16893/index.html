《對馬之魂》是SP工作室最大的作品 明年公布更多細節

2019-12-15 1,863條評論 3,307次閱讀 1262人點贊

在今年的TGA頒獎典禮上，《對馬之魂》公布了全新預告片，同時確認遊戲將在2020年夏季發售。Sucker Punch社區經理Andrew Goldfarb近日在PlayStation.Blog上談了該作的一些情報和計劃，一起來了解下吧。

Andrew Goldfarb首先向耐心等待這款遊戲的玩家們致謝：「我在今年3月份加入了Sucker Punch工作室並擔任社區經理一職，一直以來我都希望能夠向大家展示更多遊戲內容，如今我們終於做到了這一點。對於期待這款遊戲的每一位粉絲來說，這是一個漫長的等待，首先我要感謝大家對這款作品的耐心，因為對於工作室來說，這是一款非常、非常重要的遊戲，我們會花時間去做好它。」

隨後，這位社區經理透露了工作室下一步動向：「在接下來的幾個月里，我們將分享更多關於遊戲主角的更多信息，包括他所面臨的威脅、他在旅程中所遇到的盟友以及他必須要放棄什麼才能成為一名全新的武士。」

「正如我所提到的，《對馬之魂》是一款龐大的遊戲，甚至是工作室有史以來所開發的最大的一款遊戲。在新預告中我們展示了玩家將要探索的充滿活力、多樣化的開放世界，但這僅僅是一小部分，我們還沒有向你們揭露它的全貌，而在明年你們會看到更多的細節。」
