我的世界：關於1.15蜜蜂更新，你可能不知道的16個“秘密”特性

2019-12-18 1,934條評論 4,885次閱讀 2714人點贊

我的世界1.15蜜蜂更新可以說是「史上」關注度最低的一個版本，然而Mojang既然敢將其作為一個大主題更新，自然是尤其獨特的一面。

所以今天我們來聊一聊，關於1.15「嗡嗡蜂群」版本你可能不知道的16個事實！

​

1、戰鬥更新？1.15中鐵傀儡可以用鐵錠回血啦！這將開啟玩家在《我的世界》探險的全新戰鬥模式！如果說曾經的史蒂夫是個戰士，那這個版本之後你可以安安靜靜的做個「奶媽」了。記得多帶幾組鐵錠哦。

2、Creeper終於改名「苦力怕」；老玩家哭暈在廁所！Creeper終於在1.15更名為「苦力怕」啦！此前很多人都曾有的經歷：創造模式下，我們每次搜玩「苦力怕」，驀然發現應該搜「爬行者」。

​

3、傻子村民失去了職業徽章；村莊與掠奪版本更新之後，有個巨大的設計BUG：此前傻子雖然沒有職業，但卻擁有職業等級徽章，這給玩家辨別村民帶來了非常大的障礙。

4、找不到蜜蜂？手裡拿著鮮花跑到平原、繁華森林、向日葵草原，不一會兒蜜蜂都會跟著你哦。

​

5、發射器可以橫向放煙花啦！此前，煙花無論被發射器以什麼角度發射，它都會升到空中然後爆炸，但1.15之後，煙花可以被發射器橫向發射出去。同時發射器還可以發射盔甲架哦！

6、篝火現在可以被鍬右鍵熄滅；通過水桶熄滅營火被無數玩家吐槽，這一點特性的更改實際上是與基岩版更新保持了一致性。

​

7、鐵軌不再遵循東南原則；《我的世界》運用了長達9年的東南原則特性終被移除，如今放置鐵軌時會朝向玩家，而不總是南北朝向。

8、濕海綿放在地獄會變干；吸干海底神殿這件事，海綿將成為新一代的主力方塊，因為再也無需熔爐烘烤。地獄無法存水的特性會讓濕海綿的水蒸發；

​

9、幼年亡靈的誕生；創造模式下，在對著溺屍、屍殼、殭屍豬人、殭屍和殭屍村民的的成年個體使用「刷怪蛋」是，可以直接成幼年個體，再也無需看臉似的按概率刷新。

10、新的效果圖標；瞬間傷害、瞬間傷害、飽和三個效果狀態的圖示。例如吃下用蒲公英做的迷之燉菜，會出現一個飽和的狀態。

​

11、村民霸床怎麼辦？1.15給了一個新的解決辦法直接一腳把它踹下去！（對睡著的村民點擊滑鼠即可）這點類似基岩版的「守夜村民」原理。

12、蜜蜂可以加速農作物生長；農田上方放上蜂巢，周圍種點小花，你的農作物長得會更快哦。

​

13、鸚鵡學舌；但是自1.15之後，它將不會學末影人、北極熊、狼和殭屍豬人這四類生物的叫聲。同時，玩家在騎馬的時候，肩膀也可以坐上鸚鵡哦！（PS：此前鸚鵡不會在騎著坐騎的玩家肩膀停留）

14、殺凋靈新方法；在召喚凋靈的地方多養點蜜蜂，無需你動手，凋靈BOSS將會被蜜蜂蟄死；

​

15、如果沒有牛奶，《我的世界》1.15版本喝蜂蜜也可以解毒哦！

16、白天也可以睡覺了么；被怪物圍攻退無可退時怎麼辦？無論在何時，迅速放下一張床，右鍵點擊一下，均可以瞬間重置你的出生點，這是1.15改變戰鬥方式的另一個神級特性！

​所以你發現了么，《我的世界》1.15雖說不是一個多麼引人注意的版本。但Mojang在其中默默修改的小特性，有些卻足矣顛覆玩家的整個生存玩法！

​
