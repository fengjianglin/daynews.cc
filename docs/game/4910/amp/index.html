摘要

可能有的玩家已經知道了，在魔獸世界15周年期間，特別活動科爾拉克的復仇里獎勵400裝等武器的任務可以做兩次。除了自己陣營的任務，還可以當僱傭兵去完成對立陣營的任務。兩個任務先做哪個都行。對剛滿級的小號，尤其是雙持的專精來說，400裝等的武器還是比較有用的。

可能有的玩家已經知道了，在魔獸世界15周年期間，特別活動科爾拉克的復仇里獎勵400裝等武器的任務可以做兩次。除了自己陣營的任務，還可以當僱傭兵去完成對立陣營的任務。兩個任務先做哪個都行。

對剛滿級的小號，尤其是雙持的專精來說，400裝等的武器還是比較有用的。

殺科爾拉克的任務能做兩次的原因很簡單——聯盟和部落的任務不一樣。聯盟的任務「血怒者科爾拉克」的任務ID是57638，部落的任務「血怒者科爾拉克」的任務ID是57639。兩個任務是否完成的檢測宏分別是：

/dump IsQuestFlaggedCompleted(57638)

/dump IsQuestFlaggedCompleted(57639)

使用上面的兩個宏後，顯示false表示該任務未完成，true則表示已完成。

注意事項：

  • 當僱傭兵排戰場拿武器的話，接到的任務只能在同樣的僱傭兵身份下才能完成。玩家需要在持續1個小時的「僱傭合同」BUFF消失之前交掉任務。如果不小心點掉這個BUFF，接到的任務會立刻消失。
  • 離開奧山去祖達克打科爾拉克，然後再排奧山同樣能完成僱傭兵身份接到的任務，只要「僱傭合同」BUFF還在。

這個福利最大的缺點，是現在幾乎只有部落角色能排僱傭兵。在爭霸艾澤拉斯資料片期間，聯盟角色能當僱傭兵的時間屈指可數。
