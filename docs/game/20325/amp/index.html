小島慎太郎新作遠景破滅，其已從卡普空離職，未來職業規劃未定

天天要聞
1 天 ago

《怪物獵人》系列製作人之一小島慎太郎於2019年12月16日下午，在推特發布消息稱自己已經從卡普空離職。

小島慎太郎在推特中表示了對在卡普空的前同事們的感謝，並且提到自己將休息一段時間，再做一些自己認為有趣的事，但並未在推特中透露自己以後是否還會從事遊戲行業相關工作。

小島慎太郎推特

卡普空

據遊戲日報了解，小島慎太郎曾是《怪物獵人3DS》正統續作《怪物獵人X》的製作人之一，據2016年卡普空公布的數據顯示，該作品全球總銷量共410萬。

此後小島慎太郎再次與辻本良三一起負責了《怪物獵人XX》的製作，並於2017年3月18日發售。此外，小島慎太郎還曾擔任卡普空旗下文字冒險遊戲《大逆轉裁判》製作人。

《怪物獵人XX》遊戲截圖

自2017年起，小島慎太郎所在的開發團隊再未推出過任何產品。此前有傳言稱小島被調遣至《怪物獵人：世界》項目團隊，參與該項目的開發工作，也有人認為他正在帶領團隊開發新作。

《怪物獵人：世界》

不過據目前消息來看，小島慎太郎沒有開發新作，而且他的名字也或將不會出現在《怪物獵人：世界》的開發者名單內。
