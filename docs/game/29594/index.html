峽谷中他們兩個組合，還怕上不了分？

2019-12-25 1,926條評論 2,993次閱讀 880人點贊

王者已經火了那麼久了，各種玩法搭配也被大家組合起來，比如李元芳＋劉嬋的拆塔流，狼狗＋明世隱的快速節奏流，還有你強任你強，東皇＋張良的組合，今天我來交大家一種打野上分攻略，娜可露露＋瑤的組合。

作為SNK三個英雄中玩的人最多的英雄，高爆發和野區的機動性，使得他能前期帶動好的節奏，曾經的明世隱配合娜可露露是野區的霸主，明世隱提供的高額攻擊力導致娜可露露打脆皮一技能加個大，但是明世隱是沒有控制的，導致有時也會放跑敵人，自從瑤出了之後，這個英雄的勝率也是很高的，這個英雄配合娜可露露，使他輕鬆抓人，上分簡單！今天小編就來介紹下這個組合玩法。

娜可露露

娜可露露是一個刺客，高爆發能使他一套就秒人，同時2個位移技能也使得他的野區機動性很高，被動技能需要每次技能命中後下一次的普攻傷害會增加，使他每次放技能後接一次普攻使傷害最大化；一技能附帶最大生命值的傷害，命中後的普攻和技能都能觸發，配合裝備的穿透是他打坦克也是很疼的；二技能是一個位移技能，命中後能回血，且命中越多英雄回的也越多，對野怪也生效，導致他前期刷野也不會很傷；三技能也是一個位移技能，機動性很高，可功可守，高額的傷害，命中敵人後還能降低敵人的輸出，但需要注意飛行狀態中不能普攻和放二技能。

出裝推薦：

追擊刀鋒+韌性鞋+暗影戰斧+破軍+名刀+碎星錘

紅色打野刀加攻擊力和冷卻縮減，娜可露露怕控制出韌性鞋，娜可露露都是技能傷害，所以前期先出暗影戰斧，接著就能直接出破軍增加收割傷害，出名刀提高生存反殺率，最後出碎星錘，這樣超高額的穿甲，打坦克能力也很強。

銘文推薦：

七隱匿三奪萃；十鷹眼；九異變一禍源

隱匿增加移速和攻擊力，帶三個奪萃提高前期的回復能力，十鷹眼＋九異變接近百穿，大額度提高傷害，一禍源是他有機會打出幸運的暴擊。

玩法思路：前期讓輔助跟，快速刷野升到四級，四級後就有了爆發能力，配合瑤的控制和隊友可直接抓死，接著把龍控住，前期就直接帶好節奏，能不拖就不拖，連招技巧為312，中間穿插普攻就能秒人，一技能命中後會減少一技能CD，如果敵人沒死逃跑可直接一技能擊殺，達到遠程收割的效果。

瑤

瑤是法傷輔助，瑤的能力非常好，一是她的被動技能，被控制技能命中後會變成白鹿不受傷害控制，但也不能放技能，可用來探草叢或者騙技能；二是一技能可以擊退敵人且在最遠距離檢測敵人並飛向禁錮他，可用來探草叢；三是傷害技能，可對同時對三個隨機敵人照成傷害；四是她可以用大附身在隊友身上，給隊友增加一個什麼傷害都能抵擋的護盾，自己取消附身可使大招冷卻快，這邊建議盾快沒時就自己取消等待下次。

出裝推薦：

救贖之翼＋冷卻之靴+極影+極寒風暴+霸者重裝+魔女斗篷

瑤是一個上身英雄，她的機制是她出二個輔助裝是可以的，這邊建議出救贖和極影，是最適合的，鞋子出CD鞋，出極寒就能是技能CD減到上線，瑤是很靠技能的，同時增加護甲，魔女增加魔抗，霸者被動可以使她後期幾乎不用回城，魔女也可換成冰仗，放技能時減速對手，因為都是附身在身上，魔女就比較沒用。

銘文推薦：

十宿命、十調和、十虛空

這套銘文純肉的銘文，每種都加了最大生命的血量，是她前期就比較肉點，保證自身不死打出控制同時結合裝備的冷卻達到最高。

玩法思路：出門直接跟打野，使用二技能幫助打野快速刷野，打野升到四級時跟隨抓人，利用一技能眩暈敵人完成擊殺，自身四級後就可以直接附身打野身上，增加打野護盾，抓人時先用一技能眩暈，接著開二技能補充傷害，逃跑時可用一技能擊退。

組合套路：娜可露露配上瑤，厲害的地方就是瑤的gank比較強，利用一技能可以獲得視野使娜可露露的技能更好命中，娜可露露的爆發極高，但是卻很脆，進團很容易死，這邊瑤的大招可以給娜可露露增加護盾，配合救贖的護盾，可以使得娜可露露在抓人打團時不會直接給集火秒掉。

快拉上你的朋友或者女票玩下這個組合把，上分之旅開始咯~
