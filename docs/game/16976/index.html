CFS｜巴西馬哲不服？我歐少來碰一碰“巴西之矛”

2019-12-14 159條評論 4,620次閱讀 3238人點贊

12月13日就要迎來CFS2019四進二的對決了，雖然中國隊成功避開內戰，但中國軍團一路磕磕絆絆的狀態不由得讓很多CFer揪心，尤其是AG與VG的對決尤為讓人矚目。之前小編分析了綠豆與「大公子」的對位。其實除了狙擊手，突破手的狀態似乎更關鍵，幾乎決定了一支隊伍的上限。如果說到突破位，那說的就是歐歐和VG突破手Mazin了。

而歐歐作為AG的突破手，說實話最近的狀態一直不錯，從CFPLS15常規賽開始無論輸贏都能穩定發揮，突破效率極高，之後再奪NEST總決賽MVP，自從拿了全明星「最強AK47」之後槍法似乎更准了，就連以前 「殘局白給」而被人詬病，但最近拿下了幾個關鍵殘局，綜合看殘局完成率居然還挺高。歐歐在CFS開賽以來目前擊殺數高達73，KD1.14，發揮非常穩定，回合均傷害量也排名參賽選手前列，他被很多CFer親切得稱為「趙雲」（歐歐ID「ZY」，又有多次關鍵局「單騎救主」力挽狂瀾救AG所故）。可以說歐歐最近的表現讓粉絲們給他續了「呼吸權」，可以說如果歐歐在對陣VG這場站出來，穩定輸出，就徹底能摘下「歐大民」的帽子了。

Mazin一段時間內效力於BD，曾經隨隊受邀來中國參加PL聯賽，但實際上他擔任的是突破，他的拉槍很快定位特別准，而且經驗豐富殘局能力很強，因為他的ID類似於馬隊而聞名於國內，但作為BD的替補選手，讓很多CFer並沒有看重他的實力。後來Mazin加入了VG，並成為了VG戰隊的主力突破手，在國內聯賽中干翻BD拿到CFEL冠軍，之後又在巴西邀請賽上戰勝BS，成為了巴西國內炙手可熱的突破手。這次參加CFS，Mazin發揮穩中有爆，一直是強力輸出點，只不過風頭被「大公子」蓋過了而已。截止到八進四，作為一名突破手，Mazin的KD為1.21，可以說極其穩定了。尤其在對陣BS時第二圖黑色城鎮爆炸輸出18-10，各種連殺不斷，徹底將BS打崩。

我們發現其實歐歐的突破一向不錯，但最近表現更好的是他的防守，在港口B包以及異域A包樓梯和寵兒的「雙子星」防禦非常到位，寵兒吸引槍線，歐歐秒人，經常實現關鍵性刷屏，讓對手頭疼不已。而Mazin的特點則是突破極強，瞬爆閃運用非常出眾，進攻中槍線控得很好，所以希望歐歐能針對他的特點，做出一定的防禦改變。看看是「巴西之矛」更犀利還是「中國之盾」更穩健！明天就要比賽了，我們相信，努力終會有回報，歐歐訓練量超過所有選手的努力不是白費。12月13日，巴西馬哲，不服來戰，我歐少就要跟你「碰一碰」！
