《我的俠客》新手攻略，帶領萌新闖蕩自由俠義江湖

天天要聞
2019-12-20

一、遊戲簡介

《我的俠客》是一款自由開放的武俠沙盒RPG手游。其高度自由的遊戲環境能讓玩家體驗到不同凡響的江湖風景，裡面的人生百態，正邪善惡，都能由玩家自我選擇，形成自己的俠客世界。

之前一段時間的遊戲內測廣受玩家好評，相信有好多小夥伴已經熟悉了遊戲的基本玩法，而公開版手游也將在不久後正式上線，對於還沒有入坑的小夥伴，今天小編就來講一講這款遊戲的新手攻略，讓萌新們更好更順利的走上這俠客之路。

二、遊戲創建

《我的俠客》 這款遊戲的玩法非常之多，自由度極高，玩家可以根據自己的喜好來選擇劇情的走向及結局。

遊戲開始，在創建名字後，將會優先回答五個問題，而這幾個問題會對後面屬性玩法的走向有一定影響。

（1） 其對應的屬性為

木劍（劍法精通+25 刀法+25 奇門+25)

書籍（悟性+20 醫術精通+20）

銅錢（銅錢*3000 暗器精通+20）

蹴鞠（體質+15 拳掌精通+20 腿法精通+20）

（2） 其對應屬性為

迎面…(力道+20）

潛伏…（靈巧+20）

制止…（內勁+20 ）

逃離…（速度+20）

（3） 其對應屬性為

保護至親（仁義值+10 體質+15 鐵線衣*1）

行俠仗義（仁義值+30 悟性+15 高粱酒*5）

忠君報國（仁義值+15 精力+15）

天下第一（體質+10 精力+10）

（4） 其對應屬性是

毅力（反擊率+5％）

才智（破招率+5％）

武藝（連招率+5％）

機遇（暴擊率+3% 閃避率+3％）

（5） 其對應屬性是

他人情感（胭脂*5）

修鍊武功（真氣值+1000 小武學丹*5）

神兵利器（銅礦*10 銅錢2500）

活下去 （生命值+1000 白血饅頭*5）

估計看到這裡的萌新會一臉茫然，這些都是什麼鬼？但對於以前玩過類似武俠手游的玩家會感到比較熟悉，下面小編就來講一講這些屬性的用處和效果。

（1） 劍法/刀法/奇門/醫術/拳掌/腿法/暗器精通：顧名思義，就是角色對於劍法秘籍，刀法秘籍，奇門技巧秘籍拳掌秘籍等的運用，數值越大，則對戰中對應武學技能的傷害越高。

（2） 力道/靈巧/內勁/體質/精力/悟性

力道影響拳掌、刀法攻擊和連擊值；

靈巧影響腿法、劍法攻擊和命中值；

內勁影響內功，醫術，奇門，暗器攻擊和暴擊值；

體質相當於生命值上限；

精力相當於真氣值（俗稱藍量）上限；

悟性影響天賦點。

但具體要注重學哪些，專修哪些主要取決於你的門派和你所學的武學秘籍。每個門派都有相對應的（如拳掌，劍法刀法等）武學，而武學又有白色，綠色，藍色，紫色，橙色，黃色，暗金之分。白色為最基礎武學，等級依次提高，最高暗金為絕學，厲害程度不用多說。如”大力金剛拳”為少林寺藍色進攻武學，其與拳掌精通及力道息息相關，而”金鐘罩”為少林寺紫色防禦武學，其技能效果與內功值有關。如想加入丐幫，則注重學習拳掌，腿法武功，因為其頂級絕學降龍問天掌為拳掌精通，拳掌值越高，技能也越厲害。

（3） 鐵線衣/小武學丹/白血饅頭

鐵線衣為遊戲中的穿戴裝備，裝備上會加相應屬性；

小武學丹為消耗品，可加200點武學經驗；

白血饅頭也為消耗品，在戰鬥（迷宮）中使用，可恢復全體成員3000點生命值。

三、門派選擇

在完成了第一個月牙村主線任務後，將開啟門派選擇，而門派的挑選對於後面的發展也有較大的影響，下面就來看看這些門派都有哪些,都有什麼特點吧。

（1） 少林寺

包括拳掌，刀法，棍法三類武功

門派特色：忽視敵人防禦，造成更高傷害，適合打擊高防禦敵人

門派珍寶：珊瑚佛珠。闖王七寶之一，集齊可開啟闖王寶藏

絕學：大乘般若禪掌，燃木刀法，易筋經等

建議：上手難度低，強度高，適合打防禦力高的對手，推薦新手嘗試（燃木令為學習武學的鑰匙，盡量留著學習更高級的武學）。

（2） 華山派

包括劍法，拳掌兩類武功

門派特色：識破敵人的武學，大幅降低受到的傷害；附加內勁為基礎的攻擊。

門派珍寶：華山派鎮派之寶。華山劍宗祖師佩劍

絕學：九劍求敗劍訣，紫霞真經，太華君子劍

建議：華山派分為劍宗和氣宗，劍宗中的九劍對於殘頁的要求苛刻。不太適合新手，而氣宗的太華君子劍比較容易學到，推薦新手氣宗。

（3） 赤刀門

包括刀法一類武學

門派特色：對敵附加外傷，緩慢減少生命值；攻擊附加恢復自身少量生命值；通過決鬥擊敗他人，獲得他的所有物品。

門派珍寶：赤菩提

絕學赤刀神功

建議：其開局便擁有決鬥技能，擊敗他人將得到對手所有物品，對於新手的成長有很大的幫助，但由於其技能特性較為複雜，結合起來難度較大，因此不太適合新手玩家，但是老玩家的必入之一

（4） 丐幫

包括拳法，棍法，腿法三類武功

門派特色：連續不斷，綿綿不絕；通過乞討按鈕向他人乞討，獲得他的物品

門派珍寶：打狗棒，丐幫幫主信物

絕學：降龍問天掌，打狗棒法

建議：掌法厲害，前期劇情過度完善，殘頁好收集，適合新手玩家

（5） 拜月神教

包括劍法，腿法，暗器三類武功

門派特色：消耗生命值額外提升大量傷害，生命值越高傷害越高

門派珍寶：冰火蠶屍毒。服用後可大幅提高功力，若是隊友服用，其好感度變為99，但服用即中劇毒。

絕學：無敵斬，黑風腿法，日月黑光神針，日月寶典

建議：劍法，暗器較強，腿法一般，劇情很長，曲折離奇，不適合新手選擇

剩下還有

衡山派

包括劍法醫術兩類武功

門派特色：呼喚周圍的隊友一起攻擊；恢復自身或隊友生命值，解除負面狀態；製造恢復生命值回氣、提升屬性的丹藥。

絕學為七十二路飛仙劍，不老長生功；

恆山派

包括劍法棍法兩類武功

門派特色：替周圍隊友承擔部分傷害

絕學為十種自在法，百鳥朝鳳劍法

總而言之，新手推薦進入少林寺或者丐幫，前期強度夠，劇情完善，殘頁也很好獲得，並且少林寺是唯一一個隨時可以去學習也不會被踢出去的門派，但要注意燃木令的使用，盡量留著學習更高級的武學，低階武學可以通過請教門派相關人物學得。丐幫的降龍問天掌和打狗棒法很強，自帶乞討，歐洲玩家的福利，主線十分豐富，前期殘頁也容易得到，成型較快。而拜月神教則不太適合萌新加入的，可以等玩久了以後有充裕的時間再考慮進行拜月神教的劇情。還有些新玩家可能會問，如果我加入了少林寺後，又覺得丐幫的降龍問天掌很厲害特別想學，該怎麼辦？這很簡單，遊戲里的武學是可自由收集的，學習各門派武學不受限於拜師門派，各門各派幾百種武學也是能無限搭配的，所以玩家可以創造出專屬於自己的武學秘籍組合。

四、遊戲大概玩法及特色

（1） 劇情走向

該遊戲劇情主要分為五個分支：主線任務、歷練（支線任務）、角色自身任務、傳聞任務、委託任務（心愿）。

主線任務是整個遊戲的骨架分為三部分：宇文庄奇案、魔君亂江湖、掌門對決。而其中魔君亂江湖是最主要的劇情線路，也是玩法最多的。而在完成魔君亂江湖任務後可開啟自由模式，至此在自由模式中將得到沒有限制真正的自由！

支線任務大概是主線任務的十幾倍多，完成可得到等級經驗，武學等級，以及大量物品裝備等，是提高自身能力的主要資源來源，建議新手能做的都做，還可能觸發一些意想不到的禮物物品。

角色任務目前還很少，可能是小編還沒觸發。

傳聞任務大多太過於寬泛，不用刻意去做。

委託任務包括門派寄信的委託和人物的心愿，會加等級武學經驗和對應門派的好感度，也是值得一做的。

（二）戰鬥場景基礎玩法

這款遊戲為RPG遊戲，其戰鬥的棋盤當然與一些武俠手游不一樣。你將指定角色調移位置，釋放武學技能攻打敵人。其戰鬥布局如下圖

藍色方格為玩家角色可移動位置

橙色格子為敵方佔領位置，點擊可攻打敵方角色（前提在角色技能範圍內，或技能範圍內沒有隊友阻擋）。

下方為角色技能，點擊可查看技能詳情，再點擊敵方格子可用此技能攻打對面。

左邊為對戰信息，可查看雙方使用的技能和造成的傷害。

（三）送禮拉幫結派以及切磋技能

遊戲內的NPC大都十分鮮活，他會對你有單獨的好感度，而這好感度的升降可能是你送他東西，可能是你與他切磋，可能是做了相關任務，還有可能是你欺負了他門派的其他兄弟，或者半路上與其他商人交易而沒有找他交易，只要關於你和他之間的事情都會影響好感度。而當好感度到達60便可以邀請他加入自己的隊伍成為你的一個角色，幫你闖蕩江湖。切磋技能每兩天用一次，戰勝所切磋的NPC則可以獲得他身上的一件物品，是隨機抽取的。還有一個觀察按鈕，可以查看該NPC的所有基礎屬性以及所學武學和所屬門派，相對應完成一些門派委託任務。

這款遊戲有著極高的自由度，你可以跟著主線劇情一路發展下去，也可以去做支線劇情擴展武學秘籍和自身等級經驗，還可以去佔領各大門派，成為其掌門人，或者最後加入大反派得到武功的極大提升，但也會面臨眾叛親離的結局，就像該遊戲策劃所說的”你既可以做好人也可以做壞人，你可以做內心純潔的大俠也可以做內心有些陰謀的梟雄，都會獲得相應的反饋”，可玩性可以說是相當高，並且還有轉生功能，擁有一周目二周目等多種世界的玩法，這也是遊戲一大特色。

說了很多，這些也只是遊戲最基礎的玩法，只是稍微幫助萌新踏入這個陌生的俠客江湖。而對於整個遊戲來說，還有太多太多需要玩家自己去探尋去搜素去發現這遊戲的樂趣所在。
