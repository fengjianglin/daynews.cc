LOL：TES碾壓差點零封V5，貓皇：中野代練不送包贏

2019-12-22 1,254條評論 3,338次閱讀 84人點贊

12月22日德瑪西亞杯，在卡薩加入TES後，與黃金左手Knight的中野組合就備受關注，韓服雙排上分迅速衝進前十，如今德杯比賽也是檢驗訓練成果的時候，在今天第二局對戰V5的比賽中，TES拿到了奇亞娜酒桶的中野組合，而V5則是用塞拉斯皇子應對。

在開局TES設計入侵野區，一級團布隆給皇子掛上被動後果斷閃現點燃配合隊友打出控制，拿下xiaohan皇子一血反掉藍buff，野區開局就取得了很大的優勢！

隨後野區的優勢也逐漸滾到了線上，TES先是野輔聯動幫助中路Knight奇亞娜擊殺小東北塞拉斯，卡薩酒桶在掠食者好後再度軍訓中路，V5中野想要反擊的一波也被TES中野反秀，Knight奇亞娜一個大招定住兩人，遠程給到水元素Q定住皇子後交給卡薩收割，正面再將小東北塞拉斯斬殺，中野配合默契可見一斑！

在TES中野取得大優勢後，同樣關注比賽的解說貓皇直言：中野代練，不送包贏！

結果沒想到Knight奇亞娜就搞了起來，直接越到V5中路二塔後方單秒了皇子，但也被V5留下終結了大人頭，送了一波。

但是TES優勢實在太大了，裝備經濟差距已經拉開，TES的幾個大招一交正麵糰戰V5根本沒法打，差點就零封了V5，TES22分鐘就一波結束了比賽！

雖然K皇送了TES全隊唯一的人頭，但這局發揮太出色了，還是拿到了MVP，在賽後採訪中Knight也給自己的表現打分：7、8分吧，打得有點爛，就送人頭了！也提到了與卡薩的中野聯動：其實也沒什麼交流，打rank打多了，雙排的比較多，配合比較好。最後也解答了ID問題：因為馬上明年了，就想改個ID！大家覺得Knight會改啥ID呢？
