崩壞3：「黑暗破曉」主線第十四章即將揭幕

2019-12-19 220條評論 1,162次閱讀 191人點贊

為了阻止世界蛇在天穹市的計劃，琪亞娜與麗塔暫時化解干戈，攜手戰鬥。

周遭的黑暗仍在擴散，危險步步緊逼。她們能否為這座即將毀滅的城市帶來破曉之光？

——主線第十四章即將揭幕！

限時挑戰活動

3.6版本期間，參與主線14章限時探索活動，提升活動等級即可獲得4★聖痕「渡鴉·月夜（下）」、三分結晶、逆熵/天命·虛數核心、星石等獎勵！

特別說明：3.6版本後，限時活動獎勵將不再開放獲取，艦長一定要記得及時參與哦~（14章的女武神強化、章節任務等在後續版本會正常開放。）

4★聖痕「渡鴉·月夜（下）」

章節關卡

主線14章分為主線故事關卡和黑暗試煉關卡，試煉關可以幫助艦長獲得更多的活動經驗~

其中，主線故事關卡會根據時間依次逐步開放：

主線關卡14-1~14-5——2019/12/19 10:00開啟

主線關卡14-6~14~9——2019/12/26 04:00開啟

主線關卡14-10~14-13——2020/01/02 04:00開啟

真相撲朔迷離，潛行仍在繼續……

在探索真相的途中，新增的女武神刺殺表現，艦長也不要錯過哦~ 13和14章中，麗塔、琪亞娜、芽衣會有特殊的刺殺動作，艦長在潛行狀態下對敵人發動指定攻擊，即可觸發特殊的刺殺動作！

女武神強化

延續13章的挑戰征程，在14章的主線探索中艦長需要繼續幫助女武神進行強化，對抗強敵。

這次除了琪亞娜、麗塔，芽衣姐姐也會隨著劇情的發展，加入到隊伍中來哦！

艦長可以通過通關關卡、完成章節任務、參與限時主線挑戰活動等，來獲取「行動資源點II型」，提升女武神裝甲、武器、聖痕和輔助效果。

特別說明：強化過程僅在主線第十四章內生效，不會對艦長自己的女武神和裝備產生影響哦~

「行動資源點II型」

那麼，12月19日10:00，黑暗破曉，即刻出發！
