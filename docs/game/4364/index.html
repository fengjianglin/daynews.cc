“吃雞”12月新版本“瘦下來”了，這下再也不用擔心手機不行了

歡迎諸位小夥伴們來到本期天哥開講的《和平精英》「精英小課堂」~每逢新版本開啟測試的時候，天哥的後台總會有很多粉絲們的問題~為此呢，也集中起來寫出來告知各位小夥伴們~接下來呢，咱們就聊聊空中的一道光、出生島里的新彩蛋以及12月新版本「瘦下來」等有趣的「遊戲小知識」吧~

得，廢話不多說了，還是各位小夥伴們熟悉的「老配方」：有圖有證據，邊看邊聊天兒吧~

海島天空有道「流星」，倍鏡打開後看到了一個聖誕老人！

相信關注《和平精英》12月版本動態的老玩家們都已經獲悉，此次光子帶來了「聖誕模式」與「極寒模式」~天哥看到有朋友在評論區表示：這兩種模式不是在內測時期就已經出現了嗎？事實上呢，如今的這兩種模式與此前內測期雖然名字相同，但是玩法方面還是有很大的區別的！比如聖誕模式下，我們可能會在天空中看到一道「流星」，就像上圖中所示的那般，如果不打開倍鏡的話很難發現究竟是什麼「玩意兒」~

不過當我們打開倍鏡之後，就能清晰地看到這道流星的「真面目」啦~這是一個駕駛著聖誕飛車的聖誕老人~其實呢，這就是聖誕模式里的「空投飛機」，該模式中將普通的空投飛機更改了外觀，變成了聖誕老人！同時呢，這種新的空投飛機會有兩種形態，一種是同時降下4個空投箱子，一種是僅有1個箱子~就相當於我們在普通模式中遇到「雙黃蛋空投」那般，不過同時降下4個空投的幾率更高一些！

出生島的新彩蛋，雪人頭飾和聖誕樹！

在聖誕老人的空投里，我們可以撿取到很多此前從未見過的新道具~不過光子並非全部都是「升級」玩法，有些場景也特意致敬了經典~比如上圖中我們便可以看到聖誕模式下的出生島，此時玩家手中的投擲物變成了「小雪球」，投擲到其他玩家身上可以讓其擁有雪人頭飾，而且正前方還有一棵漂亮的聖誕樹，這些「元素」其實都是來自《和平精英》內測時期的經典玩法~換而言之呢，光子是在還原致敬經典玩法的同時，加入了新的道具和新元素，如此一來提升玩家們的遊戲新體驗~

12月新版本「瘦下來」了，這下再也不用擔心手機不行了！

天哥看到評論區里有玩家稱：「《和平精英》這款遊戲越更新越大，感覺手機都不行了，總不能因為一款遊戲而去頻繁的更換手機吧？」其實這種說法早已經存在了，確實也有玩家因為手機預留空間、內存等硬體配置的原因在不得已的情況選擇了「告辭」~或許是光子也考慮到了這點，在12月新版本使用「按需安裝」的方式來讓新版本「瘦下來」~

如上圖所示那般，我們可以看到每張遊戲地圖的大小，玩家們可以根據自己的需求，想要體驗的地圖以及相關的玩法進行「按需安裝」，比如說玩家平日以海島地圖為主，那麼其他模式就可以選擇放棄，如此一來就可以根據玩家自己本身的需求來自由的選擇相關的遊戲資源，從而減少手機的硬體配置要求啦~

得，本期的《和平精英》「精英小課堂」就和大家聊到這裡了，感謝各位的閱讀，接下來我們評論區里見，一起聊聊吧~
