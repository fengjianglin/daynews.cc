元氣騎士：傭兵也出“特性”，實力大幅加強，武器加配件玩法出爐

2019-12-07

特性核心

本期的遊戲特性，都離不開一個核心，那就是紅武：超級大寶貝兒。關於這件造型十分實物化的紅武，想必玩家已經從武器名字中發現了濃郁的梗點。只不過這件實力超強的紅武，最受玩家關注的地方並非是強大的AOE傷害，而是它的額外附加能力。

先不論其能對使用者造成損傷的武器特點，就是在剛上線時的極限「刺客影分身核彈流」玩法，就讓玩家領略到了它的「特性之力」。雖然之後涼屋進行了緊急修復，但是它的特性玩法卻遠沒有結束。

傭兵裝備加配件

工匠系統是《元氣騎士》中對武器強化的一種提升方式，玩家可以憑藉配件賦予的多元化特效，提高武器的實戰能力。而在遊戲中，能自由更換裝備的角色並非只有英雄一種，地牢中的傭兵也同樣可以根據等級的提升，自行選擇更為高階的武器裝備。那麼傭兵能裝備附加配件的武器嗎？

通常情況下是不能的！因為武器在丟棄的時候配件會自動分離，雖然傭兵會自動撿取武器卻不會撿取配件，除非加裝配件的武器在掉落的時候不會分離，那麼這種情況可能發生嗎？

當然可能發生！因為我們可以藉助超級大寶貝兒的特性能力，達成這種本不會發生的武器設定。

操作教程：提前準備2技能騎士和大寶貝，進入地牢後找到工匠與傭兵，完成武器加裝配件的準備工作。待所有條件達成後，將準備贈予傭兵的武器作為主武器，超級大寶貝兒作為副武器開啟2技能釋放核彈，此時手中的主武器會自動脫落而且與配件不會分離，傭兵自動撿取武器即達成「傭兵裝備加配件」BUG。
