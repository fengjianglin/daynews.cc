尋找全網最牛“諸葛亮”諸葛古鎮首屆王者爭霸賽盛大開啟

2019-12-13 957條評論 2,124次閱讀 1049人點贊

12月11日，諸葛古鎮景區推出尋找全網最牛「諸葛亮」，諸葛古鎮首屆王者爭霸賽活動，一石激起千層浪，吸引了眾多電子競技愛好者的關注，活動參與人數暴增，首開諸葛文化景區與電子競技跨界先河。

近年來，電競行業迎來飛速發展。2018年，中國IG一舉奪得英雄聯盟世界賽總冠軍，國內電競熱情空前高漲。當下玩家基數最龐大的要數英雄聯盟和王者榮耀了，王者榮耀作為MOBA類手游，競技性強，節奏更快，適合閑暇時間裡與好友進行切磋，深受當下年輕人的喜愛。而諸葛古鎮景區作為陝西省十大重點文化項目之一，以諸葛文化為依託，全景展現了諸葛亮忠貫雲霄的一生，景區獨有的諸葛亮IP與王者榮耀中英雄「諸葛亮」不謀而合，同時電競的年輕化、科技性、娛樂性高度契合了諸葛古鎮景區未來升級創新的方向，所以本次王者爭霸賽應運而生。

伴隨電競運動大眾化、城市化的趨勢日益突出，《王者榮耀》這款國民級手游以文化作為基石舉辦了6屆的《王者榮耀城市賽》，創造了「電競+城市+賽事」的全新普及模式，真正將賽事場景下沉，並與文創進行深度結合，極大提高了人們對城市電競的認可。而諸葛古鎮首屆王者爭霸賽正是依託於此賽事理念，跨界聯合，打造創新型「旅遊+電競」活動，充分挖掘和推廣諸葛古鎮的文化旅遊資源，向年輕人群體呈現景區獨特的諸葛文化與深厚的歷史底蘊。

據了解，本次大賽實行線上比賽模式，參賽者無須到場，線上使用英雄諸葛亮，採用1V1solo模式對戰，分設漢中組、陝西組、全國組三個組別，每組晉級前三名進入12.28—12.29總決賽角逐本次冠亞軍。零費用參賽，進群即可報名，大大優化了傳統比賽的參賽步驟。並且獲勝者可獲得豐厚的獎品，景區還將對部分獎品包郵到家，打消了參賽者沒有時間至景區比賽的顧慮，提升活動參與度，真正實現「全民電子競技」。

電競作為「體育+文旅」產業融合的新型主題旅遊形態，它的起步如同其他文旅產品一樣，首先就是要佔領年輕消費群體的心智，不僅僅是依靠景區的「勢」，還要在電競產品及文旅資源上進行梳理創新，體驗和服務升級。諸葛古鎮景區舉辦此次大賽，旨在充分利用旅遊和文化領域的融合優勢，挖掘電競產品的精髓和諸葛文化品牌優勢。形成「文化景區帶動賽事+賽事反哺文化景區」的新模式，迅速提升景區的品牌知名度。

2020年的諸葛古鎮景區，將陸續推出更多電子競技活動，真正服務於電競愛好者旅遊消費傾向，打造文化電競旅遊小鎮，倚靠其自身獨特的景區文化和運營模式，互促兩者融合，進一步豐富景區內涵。
