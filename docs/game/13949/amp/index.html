百人大亂斗，只為一隻咩，跑酷手游也能玩吃雞？

天天要聞
2019-12-12

「大吉大利，今晚吃雞」「我有8倍鏡，你有98K嗎？」這樣幽默的遊戲梗在生活之中隨處可見，但是大家知道其出處嗎？一部改編自日本作家高見廣春撰寫的同名小說電影——《大逃殺》，成為日後各種「大逃殺」題材影視、小說、動漫甚至是遊戲的開山鼻祖。在電影中，大家能看到現在各種大逃殺的橋段，比如放逐孤島這樣典型的玩法，為了能夠生存下來，在孤島上的人進行一場殊死搏鬥，直到剩下最後一人。

​

這樣大逃殺的場景也是被後來多部電影採用，比如之後的《飢餓遊戲》等電影。如今在遊戲圈中也採用大逃殺的玩法，其中以《絕地求生：大逃殺》為代表性，隨著大逃殺玩法的普及，越來越多的遊戲都加入了「大逃殺」的遊戲模式，甚至在一大批武俠大作之中有也會看到類似的玩法。

​

不過在遊戲圈裡，一款娛樂休閑性超高的跑酷類遊戲——《夢工場大冒險》，成為一股清流，百人在一張地圖上「廝殺」不再是為了吃雞，而是為了搶羊！在《夢工場大冒險》的搶羊大作戰中，玩家通過跑酷的形式競爭，最先到達終點或者存活到最後的隊伍便可取得勝利，贏得這一場搶咩大戰的勝利，當然了這裡的搶羊和吃雞一樣，不會真的給你一隻羊哦。

​

《夢工場大冒險》這個玩法叫做「搶羊大作戰」，一百名玩家以3人小隊的形式，在10條跑道上開始追逐亂斗。沖向終點的過程是最具趣味性的，玩家可以用沿途撿到的各種道具來攻擊所有競爭對手，比如雷雲道具可以砸中地圖上的第一個選手，哪怕是隊友也不能逃過這一劫，狠起來連自己都打！

​

還有雪球和陷阱等道具，都可以整蠱對手，另外還有增益BUFF道具，比如補充體力的血瓶，短暫無敵道具和霸體BUFF等等……道具這麼多，能撿到哪種卻是隨機的，那種體力槽馬上都要見底了，一個血瓶都撿不到的痛苦可以說是沒誰了。

除此之外，你還要面對賽道被隨機摧毀的難題，讓你感受什麼叫真正的「跑毒」！

​

值得一提的是《夢工場大冒險》不僅僅擁有夢工場旗下多部動畫電影的人氣IP角色和炫酷的遊戲畫面，還有各種有趣的玩法，隨著等級的不斷提升，將會解鎖BOSS戰、鑽石賽、公會戰等各種社交性玩法，在收穫快樂的同時，還會有大量的鑽石收入囊中哦。

​

作為夢工場和跑酷遊戲的忠實粉絲的你，是不是已經按奈不住了？遊戲即將公測，先和小編一起去官方預約一波吧，茶餘飯後到《夢工場大冒險》中一起來搶羊，享受當下快樂，追憶往昔經典！
