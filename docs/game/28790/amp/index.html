過去十年的10大遊戲！LOL成功入榜，巫師3卻輸給了一款小遊戲

天天要聞
2019-12-21

臨近2019年年末，各大媒體紛紛放出自己對遊戲業界的總結，作為美國三大時事性周刊之一的《時代周刊》也在這時放出了自己評選的過去十年的10大遊戲（2010年至2019年）十大遊戲。

不過遊戲日報君發現這份榜單的評選標準有些奇怪，甚至出現著名3A大作被一款在國內沒什麼知名度的獨立遊戲打敗的情況。

《時代周刊》評選21世紀10年代十大遊戲

外媒《時代周刊》公布的這份榜單並沒對遊戲進行排名，只是選出了10款於2010年至2019年發售的最具影響力遊戲。

顛覆玩家們對開放世界認知的《上古捲軸5：天際》和《塞爾達傳說：曠野之息》都進入榜單，讓玩家走出家門的現象級遊戲《寶可夢GO》也入選十大遊戲，V社的王牌之一《傳送門2》和R星的最高傑作《GTA5》都在榜單之中。

改變了業界和玩家對遊戲的看法並引領受苦潮流的《黑暗之魂》，以及現已火遍全球的吃雞遊戲《堡壘之夜》也被選為10大遊戲。

不過剩下的3款遊戲卻引起了網友們的熱議，於2009登場的《我的世界》和《英雄聯盟》出現在了榜單中，讓一眾網友吐槽「原來2009年不屬於21世紀初而屬於21世紀10年代」，雖然LOL和MC都在全球範圍內擁有極高的影響力，但《時代周刊》強行把超出時間範圍的遊戲算在十大遊戲中，就顯得非常奇怪。

而最後一款遊戲則出乎了玩家們的意料，一款名為《極樂迪斯科》的RPG遊戲把《巫師3》、《只狼》、《荒野大鏢客2》和《光環致遠星》等一堆優秀遊戲擠了下去。

雖然《極樂迪斯科》在「遊戲界奧斯卡」TGA上創下了「四提名四獲獎」的成績，但它並不是年度遊戲，在國內的知名度也遠不如其它大作。

《極樂迪斯科》看上去像1999年的遊戲

《時代周刊》對《極樂迪斯科》的評價是，這款遊戲的講故事技巧似乎來自於未來，不過外觀和風格類似於1999年的遊戲。

也就是說《巫師3》等外表光鮮亮麗的大作，輸給了一款擁有上世紀畫質的「小遊戲」，由於《極樂迪斯科》尚未推出中文版本，現在很難想像製作人要講故事講得有多好才能把一眾大作全部擠了下去，網友們也都在吐槽《巫師3》真的沒排面。

不過這21世紀10年代十大遊戲是由《時代周刊》根據自己的見解評選出來的，不知道對於大家而言2010至2019年十大遊戲應該是哪些？
