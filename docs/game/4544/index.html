今天將迎來絕地求生第三屆微博杯小組賽最後一輪賽事，本次作為第三屆微博杯小組賽壓軸的參賽戰隊為BC組，具體的戰隊是：

B組：IFTY戰隊、4AM戰隊、ARK戰隊、AMY戰隊、OMG戰隊、VG戰隊、1246戰隊、FTG戰隊

C組：Tianba戰隊、LGE戰隊、Weibo戰隊、SMG戰隊、TSG戰隊、LYG戰隊、CL戰隊、Newhappy戰隊

可以說今天的比賽應該是這幾天小組賽里最受人期待的，畢竟今天的看點多多。

現在就說說今天BC組賽事的看點吧：

1、4AM戰隊再次和SMG戰隊相遇，這個應該是很多觀看直播的人想要看的吧，到時候又有各種話題了。

2、4AM戰隊的上陣選手的調換，在前面的賽事里4AM戰隊的人員是有調換的，第一天的AB組賽事是韋神、王總、永遠、小鹿，第四天的賽事里是韋神、王總、小鹿、黑潮，那麼這個最後一天的BC組賽事會不會有新的調換呢？畢竟這次4AM戰隊的龍太子也在本次的參賽名單上。

3、名額上的爭奪，本次第三屆微博杯的總決賽名額只有16個，現階段上百分的戰隊有9支戰隊，也就是說現階段名額只有7個，且也有幾支戰隊在這個7名額之內，最後剩餘的名額蠻少的，不知道OMG戰隊、4AM戰隊和SMG戰隊能否成功晉級總決賽。

最後一天的賽事即將來襲，到底是哪些戰隊成功晉級總決賽呢，咱們拭目以待。
