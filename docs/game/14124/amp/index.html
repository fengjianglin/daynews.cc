WE兮夜下家或以出現，曾獲得冠軍，網友：從一個WE去另一個WE？

天天要聞
2019-12-11

英雄聯盟轉會期還在繼續，作為WE粉絲。在擔心WE下賽季陣容的同時，兮夜究竟會留隊還是離開，如果離開會去哪裡都是大家十分關心的事情。而從這兩天兮夜一直在與DMO戰隊的打野小鵬雙排，或許他很有可能會加入DMO。網友們看到也是紛紛表示：這是從一個WE去到另一個WE？

DMO戰隊的前身是SDG戰隊，在2018年斬下LDL冠軍之後成功闖入LPL。2019年春季賽常規賽第六，季後賽被TES3:0帶走。夏季賽常規賽更是排到第11。而且隊內全是新人，這樣一直戰隊。比之19年的WE還不如。

其實，兮夜目前的處境十分尷尬。LPL的強隊，比如IG，FPX，TES等都有著自己的當家中單，基本上沒有引進兮夜的可能。原本RNG會是兮夜最理想的去處，一是RNG會想要組建全華班。二是兮夜在國產中單之中還是數一數二的存在。但是似乎RNG並沒有想要換掉小虎的意思。

在之前很多人都傳言LGD會引進兮夜。這確實是一個很不不錯的選擇。LGD在轉會期開始沒多久就宣布引進了世界頂尖打野peanut。再加上隊內AD選手Kramer也是聯賽中頂尖的AD。雖然此前幾個賽季LGD的戰績都不太好。但是從今年LGD的動作來看，肯定是想要衝擊世界賽的。但是似乎LGD對兮夜沒有太大興趣。

而此前便有知情人士爆料。DMO戰隊將會引進一名國產中單。在LNG宣布中單FenFen加入之後，目前市場上成為自由人的國產中單也就剩下兮夜一人了。再加上兮夜和DMO打野這兩天的雙排記錄來看。兮夜去DMO的可能性還是很大的。

那麼問題來了。兮夜在成為自由人的時候就發微博說需要找一個對未來有規劃的隊伍。如果說以DMO如今的陣容，兮夜加入進去的話。似乎和留在WE也沒什麼區別。在其他戰隊都紛紛補強的情況下，DMO戰隊這一票新人似乎並沒有能去世界賽的兆頭。

不過如果兮夜去了DMO的話，那麼證明DMO對下個賽季肯定有著自己的規劃。很有可能DMO將會再引進其他選手補強陣容。JKL或許是一個十分不錯的選擇。在IG戰隊中，JKL的戰略地位一直得不到重視。而如果去到DMO的話，肯定戰隊將會以兮夜和JKL為核心制定戰術。

但如果DMO沒能爭取來JKL，以他們目前這個陣容。下個賽季很可能還是會在聯賽中墊底。即使兮夜能夠把自己的經驗帶給DMO戰隊的這些新人們。但個人實力上DMO戰隊比之聯賽前面的隊伍還是差了一些。

至於最終結果如何，還是靜靜等待官宣吧。
