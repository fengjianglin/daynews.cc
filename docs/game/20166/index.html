無限火力中誰的大招沒用？猴子開大成靶子，瑞茲組團送人頭

2019-12-18 1,070條評論 3,427次閱讀 2069人點贊

如今無限火力開啟之後，確實吸引到了一些老玩家的回歸，同時也有不少新玩家加入其中，想要感受一下LOL的魅力。但是就這個模式來看，因為英雄之間的差異比較大，一旦選擇很無解的英雄，遊戲的提現性就會大大降低。同時在遊戲中還有著一個很有趣的現象，那就是一些英雄原本比較有用的大招，在無限火力當中，就沒有原本那種作用，甚至還不如不放。今天就為大家盤點幾位無限火力中大招沒有什麼作用的英雄。

首先就是猴子，這個英雄一般模式中玩家們多會去打野，因為他有著出色的傷害和控制，還有著隱身，能夠打出一個出其不意的效果。猴子的主要核心技能就是他的大招，在一般模式當中，團隊的開團任務甚至都要依賴於他的大招。但是一旦英雄們的冷卻都被提升之後，會發現猴子的大招似乎沒有什麼作用了。

因為在冷卻時間降低之後，英雄們的單位傷害都得到了提升，猴子的大招卻並沒有能夠提升傷害。因此在遊戲中如果開啟了大招，就會發現自己很尷尬地在原地打轉，對面即便會被擊飛，在短時間之內就能夠脫身從而將自己秒掉。因此正確的玩法就是僅僅依賴於大招的擊飛，主要的輸出手段還是靠qe，這樣才能夠讓猴子的作用真正地發揮出來，如果太依賴於大招，就會出現被作為靶子的情況。

燼的大招在一般模式當中其實也能夠作為開團使用，同時還能夠進行收割，因為距離比較遠，很難在短時間之內就找到他的位置，對他造成威脅。但是在無限模式當中，這樣的局面就不復存在了，造成的結果也很尷尬，那就是燼在開啟大招之後，會被很快就找到位置。因為多數英雄的位移冷卻都很快，因此會導致燼在短時間之內就被秒殺。

最正確的玩法就是燼盡量打出暴擊，這樣能夠提升移動速度。通過這樣的方式也能夠幫助燼更快地打出傷害，加上移速提供的機動性，保證自己不會被直接秒殺。說到底，在這種模式中，盡量不要使用大招，因為太笨拙了，一旦被刺客類近身，像小魚人那麼靈活的英雄，就會被直接餵魚，然後一個E帶走，因此最好不要使用大招。

瑞茲也是在無線中不建議使用大招的一個英雄，可以說這個英雄在無限中完全解鎖了輸出的頻率。但是也有一個弱點那就是手短，因此可能會被風箏，其實阻擋瑞茲無限的最大因素就是他的大招。因為他的大招雖然可以位移，但是很可能就是直接被對面盯准，一套技能丟下來不死也殘血了。

雖然在平時的模式中，瑞茲可以通過大招來幫助收割，同時還能夠帶領隊友支援，但是如果在無線中開車的話，很可能會是組隊給對面送人頭。因此瑞茲的大招如果面對敵人的時候盡量不要使用，想要提高到線上的速度還可以，一般不要開團使用，不然就會被直接秒殺。

還有一個英雄就是波比，波比也是一個冷門的強勢英雄，鍾情於她的玩家，主要看中的就是她的控制性，能夠打出意想不到的效果。但是一旦到了無限模式中，就會直接將她的大招給「削弱」，為什麼要這麼說呢？主要就是因為她的大招前搖太長了，蓄力都要很長時間，玩家們可以通過位移來躲開不說，還能夠趁機打出一波傷害。本身波比大招主要就是打出控制，沒有什麼傷害，在這個模式中很吃虧，因此如果選擇了這個英雄，就不要再開啟大招了，不然就會拖慢自己發育的節奏，還很有可能被對面打爆，因為實在是太慢了。無限火力中誰的大招沒用？猴子開大成靶子，瑞茲組團送人頭
