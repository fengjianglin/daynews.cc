陰陽師：海國大局入侵，大佬們有心無力！平安京陷落成定局

天天要聞
2019-12-22

陰陽師12月15日正式開啟「京都決戰」活動。海之國傾巢出動，帶領大軍，發起對平安京及附近五個地點最後的總攻，不管之前海國經歷了多少失敗，只要他們成功了這一次，他們就是成功的。而平安京是所有陰陽師的大本營，如果被海國攻佔，後面會發生什麼誰也不知道（反正不會有好結果）。所以只有拼盡全力才能看到一線曙光。

但是這次的活動從一開始似乎就發現了沒那麼簡單。這次的活動類似於偽神降臨活動，上次的活動是因為大家對規則的不了解和一些人心的問題所以導致活動差點失敗。但是這次活動沒有那麼多的花里胡哨，只有單純的一個字：難。

活動一開始，玩家們就被分兵五路，需要攻破五個地方的陣眼才能接近京都。而每一路的攻打路線都不一樣，並且每一路的難度也有所不同，就算是最好打的「大江山」一線，也足足苦戰4天左右才攻破陣眼，其他線路所需時間更長。

當大家攻破五路陣眼之時，以為可以對京都的海鳴發起總攻之時，大家才發現，這僅僅是一個開始。京都的海鳴不僅有六位「大將」守護著，還有附近的「巨大海妖」給予的加速，減傷buff。如果不將海妖消滅掉，那麼不管是打附近的6位大將還是打京都的陣眼，都基本上是不可能的。所以大家不僅要對京都發起總攻還要開荒去打boss。一切的一切對陰陽師玩家們都是那麼的不利。

活動進入到最後一天，12月21日，而進度卻還遠遠沒到打海鳴的時候。中午11點才開啟了對最後一隻海妖的攻勢，解決掉這隻海妖，玩家們才能對6隻鎮守京都陣眼的妖怪發起攻擊。下午三點石距被擊殺，玩家對開始對陣眼附近第一隻妖怪——入儉師發起攻擊，可是玩家面對的可不僅僅只有這一隻妖怪。一隻入儉師有48007879082179的血量，每位玩家每次打3000萬的血量，都要打160萬多次，其難度可想而知。

這次活動很多的玩家都抱著失敗的念頭在肝，因為實在是太難了。但是大家都沒有停手，嘴上說著要投靠海國少主的玩家，打起怪來比誰都不手軟。作為痒痒鼠的一員，我也三開打著大岳丸，雖說兩個小號傷害不高，但至少能為守護平安京作出一份貢獻。

這次活動不管失敗還是成功玩家們都已經非常的努力了，就是不知道為什麼一個活動會這麼難（ch出來挨打）真的是有心無力，很佩服努力的大佬們。活動無非失敗和成功兩種結果，不過團結一起的痒痒鼠們真的是很有力量。最後祝每位努力的大佬都歐氣滿滿，一發入魂抽到自己想要的式神！
