爐石傳說：支線任務大盤點，獵人最受歡迎，最無人問津的是它

2019-12-24 1,346條評論 4,834次閱讀 3616人點贊

新版本巨龍降臨上線已經有接近兩周時間了，在這個版本里不僅延續了正反派對決的概念，而且也有相當多的創新。拋開5大反派的迦拉克隆不談，還有正派的支線任務。

版本初期支線任務無人問津，但隨著T7獵的崛起，很多支線任務也開始慢慢被玩家們重視了起來。今天筆者就來和大家聊一聊新版本支線任務的相關數據。

在這之前先來了解一下基礎信息，支線任務是對應的獵人、德魯伊、騎士、法師4個正派職業，每個職業2張支線任務，所以總共是8張牌，下面我們就按照出場率高低來排行。

TOP1——病毒增援

出場率：52.1%，勝率：57.3%

這張牌隨著T7獵崛起而崛起，最低下限也能打出6點傷害。在1費沒有隨從的情況下打出這張牌，後續就可以遵循少生孩子多射箭的原則，儘快完成任務，召喚3個小馬雲。

這張牌不僅出現在獵人52.1%的套牌里，而且平均勝率也是高達57.3%。

TOP2——學習龍語

出場率：27.3%，勝率50.8%

法師的學習龍語也是蠻給力的，只要多使用法術，就可以儘早召喚這條6-6的龍。這個身材其實有點類似於捕鼠陷阱，區別在於一個是主動觸發，一個是被動觸發，各有千秋。

TOP3——人多勢眾

出場率：25.5%，勝率：48.4%

與法師的學習龍語類似，同樣是消耗法力值，然後完成任務。由於任務完成是從牌庫里召喚隨機一個隨從，所以盡量少帶戰吼隨從，否則拉出來也就是個白板。TOP4——掃清道路

TOP4——掃清道路

出場率：18.1%，勝率：54.8%

這張任務牌其實挺好完成的，因為獵人有不少低費突襲隨從。當回合完成還能再招一個4-4突襲的隨從，基本可以在前期穩穩把控場面。整體來看勝率也是相當不錯，任務獵可以考慮一下。

TOP5——庇護

出場率：16.3%，勝率50%

後四名基本上使用率就不高了。不過貌似有元氣騎開始加入這張卡了，在有冰箱的情況下，2費穩穩出一個3-6的嘲諷還是挺好的。但是由於現在環境整體偏快，所以前期摸到手裡可能就會非常卡手，故沒有太多玩家選擇。

TOP6——正義感召

出場率：5.7%，勝率：49.1%

這張卡牌確實收益不大，因為使用1費換來的也就值1費，而前面幾個卡牌的性價比至少都是更高的，這樣的任務才有做的必要。

TOP7——元素盟軍

出場率：4.2%，勝率41.6%

這張牌筆者個人認為其實還是挺有潛力的，因為做完任務等於是1費抽3張，簡直就是奧術天才。不過還有待開發，目前版本還沒有元素法的一席之地。

TOP8——保護甲板

出場率：1.7%，勝率：34.0%

倒數第一名揭曉啦，他就是德魯伊的保護甲板。一看這個描述就沒有讓人產生想把它加入牌庫的衝動，甚至覺得太沒用了。

唯一有聯想的大概是和貢克德做一些聯動，不過目前也沒有大神去進行開發。
