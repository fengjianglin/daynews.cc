LPL人均打架狂魔？德杯LGD上單賽場亂秀，小花生看了都在笑

2019-12-25 384條評論 1,575次閱讀 447人點贊

　　作為英雄聯盟每年最後一場重要賽事，德杯對於很多戰隊來說都是十分重要的，一方面是來自轉會期之後對於新隊員之間的磨合，一方面也是為戰隊留下一年內最後濃墨重彩的一筆。對於觀眾來說，大家也是對德杯越來越在意，特別是在經歷了大變動之後。就像今年S9之後的LPL戰隊粉絲來說，德杯能夠看到新隊員的上場，能夠看看這些選手的真正實力。而今年備受關注的選手自然也是出現在德杯中了，不管是據說價值3000w的RNG打野小龍堡，還是來自LCK的小花生等，都讓大家格外的期待。

　　就在德杯第二天的杯賽中，觀眾們也是終於等到了小花生的首秀，可能很多人對於這名選手的印象都是來自當年和明凱的一戰，要知道當初的小花生也是天才打野級別的，只不過在被LCK打磨了幾年之後漸漸的失去了自己的魄力，變成了大家口中的小刷生，這一次來到LPL也是重盼所歸，大家都想看到這名昔日的天才打野能夠在LPL成長成什麼模樣。

　　不知道大家有沒有看LGD和V5的那場比賽，雖然說V5隻是一直下游的戰隊，但是這一場也是打的相當的有血性，也是在這一場比賽中小花生在賽後採訪中說法，是真的感受到了LPL的氛圍，那就是喜歡打架。首先是上單cult表現十分的亮眼，要知道過去的LPL上單都是走抗壓流的，自從有了the shy之後人人都是打架王，cult也不例外在和v5的對決中秀的對面亂飛。

　　前期的時候小花生就在幫上單不斷的gank，兩次越塔成功擊殺對方的建模之後cult就完全站起來了，不僅壓著對面打同時還在中期幫助隊友建立起優勢，其實V5也是發現了上單cult比較肥這個點，因此對面的打野盲僧和輔助也是第一時間選擇過來幫忙，但是此時的鱷魚在已經有了黑切以及提亞馬特的情況下，不僅先手將對方的上單打殘，同時在面對盲僧和泰坦的gank時遊刃有餘，擊殺劍魔眩暈盲僧最終拿下泰坦完成三殺，這一幕鏡頭也是給到了小花生，能夠看出小花生人都是懵的，隨後又笑了出來。

　　當然這一局亮點還是很多的，又比如說賽後花生在採訪時表示，對於第一波對方的抓人印象十分深刻，當時的花生是在大龍坑的地方寡婦輕鬆的越過牆逃之夭夭，但是V5的盲僧選擇第一時間交出閃現進行追擊，同時上單劍魔以及中單的妖姬也在第一時間趕過來了，並且妖姬還交了閃這就導致花生交出了一血，回想起這一幕花生也十分的感慨，表示自己還得好好適應一下LPL的節奏才行。

　　小花生這一次來到LPL很多粉絲也是相當開心的，因為從他個人的操作打法上面來看和LPL風格十分符合，對於LGD的粉絲來說，他的加入或許能夠讓這支戰隊產生質變吧，不知道大家在看完LGD同V5的比賽之後有什麼樣的感受呢?歡迎在文章下方留言評論喔。
