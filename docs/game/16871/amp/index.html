這個遊戲好玩嗎：略顯平庸的硬核roguelike——《太空羅賓遜》

天天要聞
3 天 ago

曾為我們帶來《旁觀者》系列、《別喂猴》、《我不是怪物》等獨游佳作的發行商Alawar Premium，這次為我們帶來了一款像素畫風的roguelike射擊遊戲——《太空羅賓遜》。這是一款雙搖桿垂直捲軸射擊遊戲，加上其核心玩法中的roguelike機制以及像素畫風，就不免讓人聯想到PC端的《挺進地牢》和移動端的《元氣騎士》這類遊戲。誠然，《太空羅賓遜》是一款中規中矩的roguelike射擊遊戲，難度也足夠硬核，但游遊戲體驗卻不是很出色，遊戲品質總體而言比較中庸。

遊戲開始的劇情

《太空羅賓遜》的設定並不複雜，玩家如果是鍵鼠操作的話就是左手通過鍵盤上的WASD來控制主角移動，右手使用滑鼠進行瞄準和射擊，和大多數雙搖桿垂直捲軸射擊遊戲的基本操作一樣，上手是比較簡單的。E鍵是交互鍵，用於拾取掉落物品，以及使用主基地中的設施。遊戲的開始，主角的飛船就遭遇了事故，主角不得不被迫降落，其間穿插一段簡單的新手引導。主角使用扳手進行近戰，使用手槍進行射擊，近戰使用滑鼠右鍵，射擊是滑鼠左鍵，掌握起來還是很快的。玩家面對怪物來襲時可以使用威力強大的扳手進行近戰攻擊，由於攻擊判定的範圍還是比較大的，所以近戰的上手度也不難，對付小怪完全可以近戰應付。手槍射擊雖然有遠距離攻擊的優勢，但威力似乎並不強，適合放風箏。當然，玩家在闖關過程中可以拾取補給箱里的槍械，如果能夠拾取到強力的槍械武器那自然就事半功倍了。榴彈發射器威力巨大，且帶有範圍傷害，但是榴彈發射出去後爆炸時間有略微的延遲，需要掌握好一定的提前量。榴彈飛行中碰到牆壁等障礙物會反彈，有時候利用好了會有奇效。不過，萬事有利有弊，榴彈由於是範圍傷害，如果主角自己也在爆炸範圍里的話會被自己的榴彈誤傷，這是需要小心的地方。除此以外，還有激光手槍、霰彈槍、衝鋒槍等不同類型的槍械可以在關卡地圖中撿到，各種槍有不同的利弊，比如衝鋒槍射速雖然快，但是威力較弱；霰彈槍威力大但是遠距離的命中率比較低，因此玩家需要根據自己目前使用的武器類型來制定不同的戰鬥策略，靈活走位才能克敵制勝。

戰鬥場面

關卡地圖中會有各種不同的怪物，擊殺怪物會掉落彈藥補給。玩家攻擊地圖中的晶石可以獲得晶石資源，用於升級主基地的發電機。地圖中會有一個讓玩家有機會選擇一件強力道具的機會，清理乾淨附近的怪物之後務必要去拿上這件物品，通常而言都是效果十分實用的道具或者武器，對接下來的闖關有很大益處。遊戲中有一個很有特色的系統——晝夜系統，玩家在地圖中冒險，遊戲世界的時間也會流逝，當夜晚降臨時不僅玩家的視野大大縮小，而且地圖裡會刷出大量的夜行怪，由於數量眾多，所以玩家在夜間基本上最穩妥的辦法就是在一個小範圍里跑圖，防止被夜行怪圍毆而死，不過這期間也不要去做別的事情了，因為地圖中本來就有的怪物加上夜行怪玩家是很難對付過來的，夜間一門心思應付夜行怪就可以了。只需要熬過了夜間時間，天亮後夜行怪就會自行死亡。

夜晚降臨

遊戲由於是roguelike機制，每一次死亡，玩家回到主基地之後都只能帶回少量資源，主基地的一些設施需要使用特定道具才可以解鎖，這些特定道具都是在關卡地圖裡隨機產出的，所以這款遊戲其實是有一定的肝度的——由於是隨機的，所以運氣也很重要，如果玩家想要解鎖某個設施但是刷了好多次都沒出某個道具的話就有點蛋疼了。遊戲的難度比較大，說實話對手殘玩家非常不友好，這一點在遊戲的steam商店介紹里倒是提到了，只能說誠不我欺。雖然玩家不一定要幹掉地圖裡所有的怪物才能進入下一關，只需要找到通往下一關的入口就可以，但即便如此遊戲難度依舊不容小覷，有時候茫茫多的小怪和彈幕實在是讓人不知道怎麼贏。

主基地解鎖設施需要特定道具

總的來說，《太空羅賓遜》是一款比較中規中矩的roguelike射擊遊戲，但品質比較平庸。典型的雙搖桿垂直捲軸射擊遊戲的操作方式，加上roguelike機制的隨機性與死亡懲罰，讓遊戲有一定的重複可玩性，但較高的遊戲難度恐怕會讓非硬核玩家難以體驗到遊戲的樂趣——畢竟就算喝了原諒藥水（遊戲里降低難度的藥水，但是喝了之後主角頭髮會變綠），也難免一次一次的死亡。
