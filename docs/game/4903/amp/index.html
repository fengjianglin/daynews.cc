別人家的遊戲公司！這家國外公司老員工可享無限期假期

天天要聞
2019-12-04

摘要

關於超負荷的工作和無限期的加班完美戳中了現代城市人的痛點，當你還在加班加班再加班的時候，紐西蘭一家公司為老員工提供的是無限期假期福利，今天有是化身檸檬精的一天呢。

關於超負荷的工作和無限期的加班完美戳中了現代城市人的痛點，當你還在加班加班再加班的時候，紐西蘭一家公司為老員工提供的是無限期假期福利，今天有是化身檸檬精的一天呢。

製作了生存模擬遊戲《DayZ》的主創Dean Hall自己創辦了遊戲工作室Rocketwerkz，已經出品了《彈盡糧絕》、《空間站工程師》等遊戲。除了遊戲挺受歡迎，最引人注意的一點是Rocketwerkz工作室幸福到令人咂舌的員工福利：老員工的年薪大約有80000紐西蘭元(摺合人民幣36萬)，更重要的是還能享受「無限期」地自由請假福利!

聽起來似乎不可能，畢竟遊戲開發公司要面臨的是時時刻刻的工作需求，Rocketwerkz能提供這樣的福利實屬難得。相比那些「996」的公司，它卻為員工提供幾乎無限制的休假時長，創始人Dean Hall談到這一點的時候說，這樣的福利制度是基於和員工之間的高度信任基礎之上的。

工作室中會有30多位員工共同負責2、3千萬的遊戲之作項目，這樣的大型項目的完全交託是工作室對員工在工作上的基本信任，進而他也說「既然都願意讓員工來負責這樣大的項目和這麼多的資金，有為什麼不相信他們會管理好自己的工作時間呢?」給予員工最大限度的自由，也是一種另類的員工激勵方式。

相應的，Rocketwerkz的員工也沒有濫用這樣的無限制假期，甚至很多時候他們上班時間比規定的工作時間還要多，放假得比以前還要少，完全地投入到公司的工作中。甚至公司還為了員工的正常休息，規定了員工每年的休假下限時間是4個星期。

Rocketwerkz的員工分為三個層級，所享受的福利也不同：剛入職的員工需要按照公司規定日程上班;工作時長達到規定時間後升到第二層級，可以擁有無限期的病假;再往後成為老員工就能擁有」無限期「假期。員工和公司之間的這種信任和關懷也能更好地增強員工的歸屬感和責任感，對於公司的長遠發展來說絕對是好事。
