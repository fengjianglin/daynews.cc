德杯最強戰隊逐漸浮出水面，場場碾壓，這裡可能最適合Jackeylove

2019-12-22 56條評論 3,562次閱讀 1458人點贊

2019年冬季德瑪西亞杯已經來到了第二天，今天是B、D兩個小組的比賽。經歷過轉會期的人員變動後，TES已經拿到了小組賽兩連勝，第三場比賽他們迎來了來自LDL的GD戰隊。這場比賽TES處於紅色方，選出了鱷魚、酒桶、夢魘、盧錫安和布隆。對手GD戰隊是蛇女、蜘蛛、刀妹、卡莎和鱷魚。

比賽4分鐘，雙方中單都來到上路打了一波上中2V2，結果GD這邊被雙殺，TES中上兩人一人一個人頭。5分鐘，GD野輔來到中路對左手進行gank，Karsa就在旁邊進行保護。兩人果然就地反打，GD三人全被殺，Knight拿到三殺，直接起飛。7分鐘Karsa來到下路，又幫助下路擊殺了錘石，隨後他輕鬆單人solo掉了第一條小龍。

比賽進行到了10分鐘，雙方僅僅通過對線，TES就已經拿到了三千多的經濟優勢。Karsa拿下峽谷先鋒，然後直接來到上路，一個大招炸飛蛇女，369的鱷魚W閃直接跟上傷害完成擊殺，峽谷先鋒隨後放到上路鱷魚單吃了全部鍍層。但是此時Knight單人在中路搶先其他路直接平推掉了一血塔。13分鐘TES拿下第二條小龍，左手隨後大招飛上，配合369越二塔擊殺掉了蛇女並在隨後推掉了上路二塔，TES經濟領先已經達到了八千，而此時時間才14分鐘而已。

15分鐘，鱷魚在上路單殺了蛇女，隨後刀妹來到上路帶線也被鱷魚打到殘血，Knight走過來一個點燃收下人頭。左手收下女刀人頭後，對中路飛一個大招，結果把GD中路三個人的所有召喚師技能全逼了出來。比賽到17分鐘時，TES經濟領先已經到了一萬多，兩邊的實力差距實在太大，TES又輕鬆拿下第三條小龍。

22分鐘，雙方在上河道相遇，GD仗著短暫的人數優勢先手開團，但是因為裝備差距太大直接被TES團滅，隨後TES順利收下男爵。在回家補給後，又吃掉了第四條小龍，成功拿到火龍的龍魂。TES經濟領先已經達到一萬六以上，幾乎就是想怎麼結束比賽都行。TES隨後開始中下分推，在高地上一波把GD團滅後拿下勝利，獲得小組賽三連勝。

其實這場比賽本來就沒多大懸念，但是TES能以這麼碾壓的方式獲得，確實說明他們的狀態現在非常好。轉會期TES送走了Xx迎來了Karsa，中上野瞬間升級成頂配。這次德杯中的幾場比賽，TES上半區基本就是無敵的表現，Karsa和Knight的配合也是天衣無縫，而在這場比賽中和369也有多次默契的配合。而Knight這場比賽再次打出絕佳表現，一人輸出都快趕上對面全隊了。最終10-1收下MVP。

雖然TES下路啟用新人組合，但是中上野實在太強，下路直接就能躺贏。通過今天的比賽來看，本次德杯最強隊伍逐漸浮現了出來，可能就是TES戰隊。雖然他們打得很具有統治力，但是Karsa和Knight打得感覺都比上賽季要輕鬆很多。就目前的情況看，可能對於還是自由人的Jackeylove來說，TES是一個比較不錯的選擇。
