陰陽師 | 強力SR分析，你不能錯過的非洲戰神

天天要聞
2019-12-10

與SSR/SP相比，SR是這個遊戲中相對好獲取的式神。眾所周知在陰陽師里稀有度不代表強度（看看就好，別當真），在當前版本下，依然有些SR卡在整個遊戲過程中是至關重要的，甚至是超越SSR的存在。

這篇文章里的SR卡大多是通過抽卡可以直接獲取的式神，臉黑的話砸百鬼慢慢攢也就有了，有些甚至可以通過妖氣封印等碎片副本獲得（日和坊、鐮鼬）。排名不代表強度，只是一個順序，大家可以根據自己的實際情況選擇培養，畢竟輸出和輔助很難放在一起比較。

1、鐮鼬

俗話說一代版本一代神，但是在陰陽師這個遊戲裡面，有一個式神是超越版本的存在，強度堪稱SSSR，從上古時期的鐮兔雙拉，到後來的鐮鹿中速、鐮卷輝、鐮御等，鐮鼬永遠是鬥技場上一顆不落的星星。作為一個拉條，雖然它的基礎速度不如面靈氣，但是鐮鼬自身的被動配合招財貓御魂，卻能打出「永動機」的效果。鐮鼬既可以用來搶一速，也能在面對速控隊伍時利用被動脫控反制，反敗為勝（有點看臉就是了）。

雖然在pve上，鐮鼬的作用相對有限，但是光憑他在鬥技場上的出色表現，就足以讓它穩坐第一的寶座，不管你是不是鬥技玩家，都建議你練一隻滿技能的鐮鼬。

2、追月神

在目前的遊戲內，主流的打火式神有五個，分別是SSR卡輝夜姬，SR卡追月神、薰，R卡座敷童子和天井下（古籠火、燈籠鬼和惠比壽這類非主流的不算），輝夜姬通過幻境來提供鬼火，薰用賣血來回火+保護隊友，座敷童子通過賣血來提供鬼火，天井下是在隊友行動後穩定恢復鬼火，而追月神則是通過推進鬼火條來為隊友供火。也就是說，追月神本身不生產鬼火，她只是鬼火的搬運工。

在大型BOSS副本時，追月神絕對是最實用的打火式神。因為其自身動畫快，給鬼火的同時還能為隊友增加傷害和速度，加快整體的輸出節奏，幫助隊友打出更高的傷害。不過她本身比較吃資源，建議有抽到追月神的小夥伴千萬別分解了，留著給她升級能。

3、書翁

書翁名為輔助，實際上卻是一個輸出，配合醜女和其他單體輸出，能將傷害最大化。在不知火出來之前，有大boss的地方，就有追丑書（追月醜女書翁），雖然離吞吞陣容（不知火+雙酒吞）在一定程度上會另書翁失業（因為雙酒吞+不知火一回合能打出的傷害遠超過書翁的爆炸傷害，所以書翁與離吞吞相性不太好），但是追書丑茨清依然可以面對大部分的副本boss。

書翁常見的御魂搭配是生生暴擊或者生生爆傷反堆暴擊，在隊伍中有清姬時候也可以生生生主堆生命上限。書翁的生命越高，能打出的傷害上限就越高，但如果主力輸出不夠給力的情況下，狂堆生命的收益就不如堆暴擊爆傷高了。所以書翁帶暴擊還是爆傷還是生命，一個是跟陣容有關，一個是跟主力輸出的傷害能力有關。在御魂套裝方面則沒有要求，有條件可以整兩件套蜃氣樓，其他散件即可。

在技能上，書翁的大招是一定要滿級的，而且因為傷害和自身生命掛鉤，因此書翁是一定需要六星的，以上兩個條件無法達成的話，書翁的作用就十分有限。

4、清姬

清姬姐姐在下水道呆了兩年多之後，終於在今年迎來的她的春天。清姬的增強主要在於她的大招和被動，在群體上毒的同時還能為對面施加破防的debuff，使輸出式神的傷害更上一層樓。而且在毒buff足夠多的的情況下，怪物的防禦可能到達0，此時間接傷害打這些怪物是必定暴擊的（主要是面靈氣和書翁）。因此在隊伍中有清姬時，書翁可以不用攜帶暴擊，面靈氣也可以不用堆暴擊。

除了在pve中配合輸出式神將傷害最大化外，攜帶雪幽魂的清姬也能在一定程度上克制驅散式神（因為清姬上的debuff實在是太多了），因此在鬥技中，也經常能看見清姬的身影，可謂是鬥技副本兩開花。

在御魂搭配上，清姬一般使用雪幽魂+土蜘蛛的組合，清姬五星也可以用，但是五星的話比較脆一點，二四六分別是速命生，如果是六星的話六號位可以換成攻擊。

5、櫻花妖

之前在寫ssr攻略的時候說到，櫻花妖的加強使得一眾受關聯的式神集體退群，其中包括花鳥卷、匣中少女等。櫻花妖與清姬的命運相似，都是一個低開高走的過程，從下水道式神逐步爬到版本之女的位置，成功幹掉了花鳥卷，成為PVP第一奶，還能兼職輸出/控制，即便是在pve櫻花的地位也相當高。不過由於不知火版本的到來，又把花鳥卷帶回了鬥技場，櫻花也迎來了一次削弱，且因為不知火帶動的化鯨體系，使得匣中少女也回到了人們的視野中，櫻花的地位略有下降，但是整體強度沒有受到太大的影響，依然是一個值得養成的奶媽。帶珍珠的櫻花在打彼岸花、化鯨等陣容時依然表現十分優秀。

御魂方面，除了無火的鬥技玩家外，一般建議攜帶珍珠，奶+盾，此外，招財、樹妖、地藏等也可以是櫻花的選擇。

6、日和坊

同樣作為奶媽，日和坊和櫻花的命運完全不同。櫻花在下水道奮鬥了兩年才勉強成為PVP第一奶媽，而且這個位置沒坐多久就被拉下來了，而日和坊則一生下來就是PVE第一奶，到現在也是。日和坊的技能機制比較特殊，她是靠對面的回合來奶隊友的，且根據隊友受到的傷害以及對手的治療而提升治療量，可以說是十分完美的機制，是大家挑戰秘聞不可或缺的夥伴。

在御魂方面，一般推薦攜帶薙魂、地藏或者樹妖，二四六分別為生生暴擊，有條件的話日和坊還是需要堆高爆傷的，和其他奶媽相比，日和坊滿暴再堆爆傷的收益會更高（因為吸收的傷害量和治療量也可以通過暴擊暴傷的計算轉化為更大的治療量）。

7、犬神

犬神的技能經過了多次的調整，目前已經基本穩定下來。在PVE方面，犬神可以配合其他反擊類式神，通關80%的秘聞副本（不過如果有猙鬼切的話是可以替換掉犬神的位置），在PVP方面，犬神能與白藏主形成良好的配合（噁心的陣容），能有效針對彼岸花、紅葉、鴆等陣容。

在御魂方面，一般建議攜帶地藏或者針女/破勢，就按常規的輸出式神的御魂搭配就好了。

8、弈

妖刀姬、二口女、傀儡師的上位替代，更多的段數和傷害上限使弈在許多場合都能找到自己的位置。例如打單體boss、御魂副本的收尾、御靈、陰界之門等。且不用吃技能也可以用（反正都是看臉觸髮針女來打傷害）。而且弈的被動能夠給對面疊加破防的buff，有概率打出爆炸的破軍傷害，是多段針女式神中強度較高的一位。

9、黑童子

pve高級玩家必備，不論是真蛇還是地域鬼王都少不了小小黑的加入，而且是不可替代的位置，常見的陣容一般有切黑、蛇黑、雙黑等。御魂方面曾經是針女+荒骷髏，現在鬼靈歌姬御魂出來後，針女+歌姬的小小黑能打出更高的傷害。前期玩家不推薦養，後期資源充足了可以考慮養1-2隻，

10、鴆

剛出來時不慍不火，經過幾波小加強以及間接傷害機制的引入之後，鴆就成了無比快樂的一個女孩子。不過鴆的快樂是有條件的，兩隻起步，且二技能要升滿。因此對於技能和黑蛋的需求比較高（一般不建議喂黑蛋，可以用多隻鴆進行反喂重新加點）。

在pve中，椒圖多鴆可以掛機大部分副本和地域鬼王，在真蛇十層也可以進行收尾，這時候的御魂選擇一般是速攻暴擊，套裝是猙，有條件可以搞兩件套蜃氣樓更好。在pvp中，由於鴆被動的毒傷讓鴆在面對當前版本火熱的化鯨陣容時能打出十分不錯的效果，配合不知火、小白、匣子等式神，可以輕鬆面對多化鯨的毒打，此時御魂一般選擇速生攻，速度越快越好，套裝選招財/火靈。除了面對化鯨之外，鴆配合鳳凰火、櫻花等式神，也能打出不錯的控制效果。

11、化鯨

與鴆相對的，化鯨就是當前版本的快樂男孩，當然，這個快樂的門檻也不低，同樣是兩隻起步，二技能最好滿上。與鴆不同的是，化鯨基本上是和不知火鎖死的，如果你沒有不知火的話，那你的快樂會減少50%。由於化鯨的技能機制，先機給自己上三層齒甲和體甲，動一次觸發一個齒甲的AOE，還不觸發對面的御魂和被動，即使被控也能打出傷害，再配合不知火的結界，相當於一個人能打兩次aoe，還是百分比傷害，很少有人扛得住。而體甲則能給化鯨或隊友減傷+治療，不知火多化鯨陣容 在面對大部分陣容時都不虛，也因此成為道館、突破中毒瘤的存在。

12、以津真天

同樣是不知火時代的產物，以津真天本身技能是普攻多段不消耗鬼火，且不像酒吞那樣需要預熱，配合不知火、大蛇、妖琴等，組成靈活多變的天火陣容，是鬥技場上十分強勢的一個流派。不過在pve中，以津真天的作用就相對有限，因此大家可以根據自己的需求決定是否培養。

13、匣中少女

不知火帶動了化鯨，化鯨帶動了一系列盾系式神，匣子也是其中一位受益者。被動0火套盾，大招一人兩條命，不論你是用化鯨還是打化鯨，都可以用到匣中少女。加上匣子本身基礎速度極快，配上高速的御魂，搶先給所有隊友套個回夢，然後就靜靜的看著對面表演了。在副本方面，匣子和日和坊、櫻花等式神作用相似，都是無火陣容中表現很好的奶媽之一。而且匣子本身不怎麼吃資源，不吃黑蛋也可以用，御魂選擇也很多樣，只要你開心愛戴什麼戴什麼，蚌精、地藏、薙魂、招財隨你心意，主堆生命就可以了（看情況選擇是否堆速度）。

14、久次良

真正的海國之主–久次良，同樣是不知火時代的一個受益者。久次良是一個十分靈活的式神，大招可以切換雙形態，在滿技能情況下，第一次使用大招會切換成進攻形態，並且有拉條效果、給隊友增加爆傷，並使自身獲得雙重暴擊的效果，且隊友普攻對方的時候有概率進行協戰。在此狀態下使用大招則會切換成防守狀態，給隊友套上一個高額護盾（參考一目連的大招），但同時也失去了協戰的效果。由於久次良滿一技能的雙重暴擊效果的存在（即暴擊時造成的傷害翻倍），所以久次良即便御魂較為一般也能打出不俗的傷害。

久次良的主要就業崗位在於鬥技場，配合不知火形成離久陣容，在pve中一般也可以作為拉條存在，但由於自身技能動畫時間較長，因此雖然拉條+爆傷效果很好，但是在御魂這種追求高速的場合一般也比較少見到它。

15、食夢貘

鬥技場上的常客，從遠古時期配合荒川、白狼的「睡殺」陣容，到後來的櫻豬、驢豬、閻豬等，食夢貘沒有經過任何一次調整，一直都是處於「強度合理」的範圍，也成了當前版本為數不多能在鬥技場上見到的控制類式神之一。食夢貘可以提供最多兩回合的睡眠控制，且不會觸發對面的御魂（骰子鬼、幽谷響這類抵抗禦魂除外），但是其弱點也十分明顯，害怕椒圖以及驅散類式神。在如今面對多化鯨陣容時，一隻高命中的招財/火靈豬依然能有一戰之力。它在鬥技中表現雖然很強，但是在pve時幾乎是沒用的，因為有很多boss是不吃控制的，且pve追求傷害和速度，而食夢貘的控制是「一碰就醒」的，所以註定了它不適合pve。

16、鳳凰火

般若時代的另一個神級控制，擁有著最強的群體不可驅散的硬控–「眩暈」（現在可以被鬼燈等少數式神驅散），但是因為傷害類的控制容易觸發對面的御魂和被動（返魂香、木魅、猙、反擊等），且由於整個大環境下抵抗的提高，使得鳳凰火的表現沒有之前那麼優秀。但是有一速的雨火/櫻火/鴆火陣容依然是一部分玩家眼中的毒瘤，你可能會被罰站到死。但是在當前版本並不是十分推薦培養，因為鳳凰火的控制需要前置debuff，且自身也容易被御饌津、彼岸花等式神反控導致控制鏈斷掉，在pve中她也並沒有什麼突出的作用，目前也就道館上偶爾能見見雨火了。

17、妖琴師

妖琴的核心技能就是二技能「餘音」，可以使一個隊友立即獲得回合，是隊伍中機動性的保證。並且因為自己天生腿長（基礎速度快），可以在鬥技上用來搶一速，可以拉晴明開盾或者拉輸出秒人等。妖琴的火也有很大一部分是八岐大蛇帶起來的，八岐大蛇拉隊友，妖琴再拉八岐大蛇，可以快速使隊伍全部變成蛇魔，最後就看大蛇吃掉陰陽師然後神憤之炎呸呸呸了。加上現在施法技能可選，再也不用擔心妖琴開大瘋狂輸出了，所以如果是鬥技玩家或者大蛇玩家的話，養一隻妖琴還是一個不錯的選擇。

18、白狼

核彈劍仙小白狼，終於從體驗服走出來了，白狼對於魂十、魂土的提升巨大，被動的增傷效果和以及額外的30點暴擊，能極大地降低玩家在魂土挖煤時的要求。常見的陣容一般有玉林狼、食茨狼等。雖然白狼在挖煤這一點上的提升十分明顯，但是如果你的練度還不夠的話，建議還是老老實實出兔丑、丑火當個司機，等以後自己湊出了面板合格的狂骨、破勢之後，就可以帶萌新刷御魂了~

19、貓掌柜

大佬的玩具之一，因其短暫的施法時間和不俗的傷害，使得貓掌柜經常活躍於各種魂十、魂土競速的排行，例如林貓貓、玉貓貓之類的高速陣容。此外，貓掌柜的大招還有推條/減火/額外打一次傷害的效果（隨緣觸發），在一些高難度副本用來減boss鬼火或者推條醜女的草人有奇效，但是如果是平民玩家的話，建議老老實實養一隻弈，動畫慢是慢了點，但是勝在傷害更高，能更穩定觸發鬼靈歌姬的效果。

20、蟹姬

另一位海國的居民，目前開發得較少。在測試服的時候，滿技能的蟹姬傷害係數可以達到驚人的414%，經過兩輪的削弱之後，蟹姬的係數逐漸達到一個「合理」的範圍（實際上還是屬於超模輸出），但是因為自身開大之後會眩暈一回合，所以不太適合打持續輸出，不過因為他的傷害係數實在是過於恐怖，因此養一隻來打結界、道館，不管是打肉還是打脆皮都會十分快樂。而且蟹姬絲毫不怕返魂香這種御魂（反正不管你觸不觸發我都會暈），如果你缺乏aoe輸出的話，養一隻蟹姬來打御魂、結界或是道館也還不錯，並且蟹姬攜帶蜃氣樓兩件套的時候，大招是不會錘暈自己的。

這個攻略其實拖了很久（一個多月），因為版本一直在更迭，一直在出新的式神，又有不知火這種版本之女存在，所以不好太快下結論。文章中有提到的需要不知火配合的式神，如果你沒有六星滿技能不知火的話，那優先順序就要往後放一放（例如化鯨、久次良、以津真天等），有些式神時純pvp的式神，喜歡鬥技的同學可以留一隻備著，有些式神是純pve式神，例如書翁、追月，基本就是每個玩家都要養的了。
