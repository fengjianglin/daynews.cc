《使命召喚》大殺四方的空中炮艇，在真實的戰場上，也就是脆皮雞

天天要聞
2019-12-12

相信只要玩過《使命召喚》的玩家，相信要說遊戲裡面最牛的連殺獎勵，很多人第一時間肯定會聯想到空中炮艇吧，如果遇到了空中炮艇，對面絕對會被打到懷疑人生，如果遇到一個手速快的，三種炮切著來瘋狂掃射，那麼，夠對面的人喝上那麼一壺了，而且你想用防空打它還做不到。

因為防空炮需要有一定的鎖定時間，在這個鎖定時間內，人家早就把炮口瞄準你了，而且就算你鎖定好了發射了，人家也會釋放干擾彈，將你發射的導彈無效掉，因此，面對這樣一個大殺器，大家只能乖乖躲在房間裡面別出去，或者想辦法把操作空中炮艇的人給幹掉。

但大家知道嗎？在遊戲裡面大殺四方的空中炮艇，可千萬別以為他們在現實中也這麼強大，事實上在現實中，空中炮艇也就是一個脆皮雞，但凡有點防空實力的國家，空中炮艇就會瞬間變成一個空中棺材，這究竟是怎麼回事呢？今天，我們就來好好說說。

單從裝備來看，這空中炮艇還真的不弱，裝備三種口徑的火炮，分別是25mm和50mm以及100mm，輪流切換著射擊的話，真的可以讓地面的裝備和人員感受啥叫槍林彈雨，再加上這種AC130炮艇機搭載的夜視系統，可以非常輕易地把炮彈砸在你的頭頂上，一點都不會打偏。

而且這種炮艇機製造難度並不是很大，甚至只要你有運輸機的話，稍微改裝一下就可以變成炮艇機了，再加上運輸機本身的體重也非常足，因此，炮彈方面還管夠，一個炮艇機就足以上地面上的一個連吃盡苦頭，可見這種炮艇機有多牛了，但也因為是運輸機的緣故，使得這種炮艇機的反防空能力十分脆弱。

但凡有點防空實力的國家就能夠輕易將這種空中炮艇給打下來，別看人家炮艇機火力兇猛到了極點，但與此同時尷尬的防空能力也是一個極端，就算一個單兵手持一枚攜帶型防空導彈就可以直接將空中炮艇給打下來了，更何況一些具備完善防空能力的國家，那簡直就是空中炮艇的噩夢。

這也就是為什麼美軍使用這種空中炮艇就是去打一些沒有啥防空能力的，比如塔利班和一些恐怖分子，如果像那種敘利亞的戰場上，美軍敢派這種炮艇機過去的話，那麼， 很有可能的事情就是敘利亞的S400防空導彈分分鐘讓駕駛空中炮艇的飛行員和機組人員去見上帝。
