CFS丨除了槍法，BD的細節處理也很可怕

2019-12-19 2,197條評論 4,972次閱讀 2694人點贊

最初對BD戰隊的印象，是在CFPL養狼計劃中，那個時候的BD看起來就是一群愣頭青組成的戰隊，沒有道具沒有戰術，見人就是剛槍，看不見人就找人剛槍。後來，經過幾屆比賽的磨練，非常明顯的感覺到BD的成長。從拿下CFSI越南邀請賽冠軍，到CFS2018世界冠軍，BD的改變是質的提升。CFS2019世界總決賽，BD戰隊進行了人員調整，並且以全勝戰績挺進冠軍賽。雖然最終敗給了兄弟隊VG，只拿到世界亞軍，但BD的整體實力，已經躋身世界前列。

回顧半決賽BD對陣SV，BD的勝利歸功於很多方面。首先，他們的槍法犀利，進入狀態很快，幾乎是開局就火力全開的勢頭。其次，他們的補槍配合太快了，隊友剛剛陣亡，第二人的子彈已經補上。最後，他們對道具的利用很精彩，不管是煙霧還是瞬爆閃，都發揮了很大的作用。當然，還有更多的細節值得一看。

SV常規包夾A區，pancada藏在樓梯附近先殺一人，此時SV已經全部進入包點，pancada從後方包夾，雙殺後，潑射拿下最後人頭，單局三殺。

SV已經下包，並且是以3打1的殘局，vianna1從包點樓梯摸上去先殺一人，SV快速補點可惜又被擊殺，vianna1拆包引出對手，再爆頭擊殺，拿下三殺殘局。

狙擊手goken以1打2殘局，C4已經安放，SV兩名殘血敵人回防。Goken換成手槍，在包點位置靈活走位，先擊殺一人，再快速回頭雙殺滅隊，單局三殺。

vianna1卡在小道近點，需要看A門和小道兩個位置，但他毫不慌張，先殺小道摸進來的敵人，再殺A門敵人，最後又回頭擊殺小道敵人，完成三殺刷屏。

再次恭喜BD戰隊拿下CFS2019世界總決賽的亞軍，期待下一次中國隊與巴西隊的較量。
