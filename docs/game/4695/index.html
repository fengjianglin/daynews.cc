LPL很多戰隊的職業選手，基本上很少有三次左右的轉會，基本上在一支戰隊都會征戰比較長的時間，比如在IG退役的小孩游神、WE的微笑、草莓、若風、皇族的盧本偉、OMG的大哥、小傘、EDG的koro1、等等選手都是從一傢俱樂部退役，而隨著職業化的進程，有不少選手也開始了逐漸走出原先的俱樂部！

怎麼這些選手當中，幾乎家家都喜歡，每一次都留下深刻影響的選手，就是來自前IG、前SS、前RNG的選手姿態，別人是換隊伍，而姿態則是換位置和隊伍，中單和上單經常變動，這也讓他成為了職業賽場上少有的萬油瓶選手，也因此在職業賽場上留下了不少佳話！

電子競技從來不缺少優秀的選手，而各種轉型都能成功的選手，其實並不多年，從位置上來說，上單和中單的互換，IG中單轉上單，從SS上單轉中單，RNG又是轉到了上單，再就是適應各傢俱樂部，再然後轉型成為主播，說實話老IG的選手，大部分轉型主播還是比較的成功！

在虎牙直播的姿態，已經是大主播，語言風格有趣，而且技術還高超，而且還可以和RNG各路選手雙排玩遊戲，能說會道還喜歡開玩笑，這些綜合優點 ，也讓他成為了，目前退役RNG選手之中，人氣最高的主播，在直播行業里，新人如同潮水一樣湧來，如果不加強自己的內容創作，等待著的就是被淘汰，顯然姿態明白這一點，內容上做到了突破自我，不管穿女裝還是打排位，還是上節目，姿態顯然是完成了自我突破！

直播行業就是如此，如果沒有一個敢於表現自己的決心，那麼顯然是無法做出成績，姿態已經完成了，從職業選手到主播的這一轉變，這一步操作其實很難，有很多人就葬身於此默默無聞！
