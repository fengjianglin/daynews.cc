王者榮耀冠軍杯hero五虎回歸，estar直接被打蒙，alan直接成替補

天天要聞
2019-12-22

hello大家好，這裡是指尖上的王者，12.21日王者榮耀世界冠軍杯開打，今晚最受矚目的一戰hero對戰estarpro，先輸掉一局給xq的hero換上了久誠首發，而estar是本屆冠軍杯的首戰，這局比賽雙方鏖戰至26分鐘才分出勝負，可以說精彩至極。

雙方陣容選擇如上圖，久誠放棄大法師，把核心給到了射手久酷，塵夏繼續藍領打野，而estar這邊貓神選到了自己最擅長的貂蟬，配合呂布打陣地戰，為的是後期的團戰，馬超切公孫離，裴擒虎打前期節奏，一切看起來好像都很美好。

開局estar因為裴擒虎的優勢準備入侵藍區，而hero知道estar一定會來換，因此開局以雙方互換藍buff開局，兩方打野分別拿到，而在estar呂布、裴擒虎、劉邦入侵的期間，久誠周瑜先手清掉兵線趕去支援，將拿掉buff的三人趕跑，貂蟬因為清線過慢，所以導致姍姍來遲，導致的結果就是被周瑜反吹，趕來的豬八戒與之形成包夾姿態。

貂蟬交掉凈化也無濟於事，被四人窮追，estar三人反應過來準備支援，但被豬八戒一屁股坐死，hero來到了前期的節奏，作為核心的貂蟬斷掉了節奏。

2分鐘，hero拿掉暴君，直接選擇入侵藍區，且在此控制第二個藍buff。

4分鐘，hero再次入侵藍區，但estar準備硬守，劉邦傳大給半血的貂蟬，藍buff卻被豬八戒懲擊收掉，貂蟬因為半血被劉邦傳大感覺能打，所以進攻的比較猛，結局就是被豬八戒再次擊殺，而劉邦也算是陪死了，好在呂布趕來支援，配合隊友擊殺了久酷的公孫離，但hero已經拿到了很多的優勢。

5分15秒中路貂蟬蹲到久誠周瑜，劉邦傳大輔助擊殺。

5分52秒，因為久誠周瑜剛剛復活，hero少人無法繼續強壓藍區，試探性的進攻了一下，被貓神貂蟬拿到第一個藍buff。

6分28秒貓神貂蟬中路肉身開團，劉邦傳大擊殺曜和豬八戒，至此estar終於找回了狀態。

7分30秒，hero先手開先知主宰，在僅剩一絲血的時候alan呂布跳大分割戰場，但應該是溝通失誤，estar沒有人跟上配合，以至於龍丟了，呂布的人頭也丟了，hero白撿個呂布，賺大了。

8分40秒，從下路回防的姜子牙被貂蟬蹲到，本以為能擊殺，在絲血的情況下被hero多人反被擊殺，不得不說這局貂蟬玩的真沒問題，但是就是少了一些運氣。

10分10秒，馬超蹲到公孫離，並配合劉邦成功擊殺，隨後趕到的豬八戒同樣被擊殺，本以為estar這波大賺，但上路曜在中路帶完線並成功讓兵線進到高低，而自己蹲伏在藍區守到了絲血的劉邦，成功擊殺後進行了止損。

11分30秒黑暗暴君被hero收下。

12分22秒呂布被消耗並逼出閃現，看到貂蟬過來並選擇開大，但落地成盒再次毫無作用的去世，而這一選擇導致了主宰再次成為hero囊中之物。

14分5秒，estar外塔全部被拔，僅剩三座高地，而hero防禦塔俱在。

14分45秒，hero控三龍其發，拆掉下路高地。

15分50秒，hero準備開暴君，但estar準備強搶，塵夏豬八戒先手發難開團，但貂蟬直接突到周瑜臉上，而呂布大招也跳在了周瑜身上，久誠輝月+復活甲+名刀，完成瞬間切換，而在期間，周瑜的火已經將呂布血量燒的很低，復活後自己反而逃出升天，一戰過後，hero剩下曜、周瑜、豬八戒，而estar吃了敗仗，好在貂蟬止損，剩下馬超和貂蟬。

此時因為貂蟬和馬超全部回城狀態極佳，剩下的三人不足以擊殺貂蟬，因此選擇了開黑暗暴君。

18分5秒有黑暗暴君的加持下，主宰也被hero收下。

18分50秒，estar在高地前背水一戰，馬超和豬八戒倒下，但馬超有復活甲並沒有減員，hero剩下的兩條龍沒有了用武之地。

22分50秒，estar龍王逼團，hero邊撤邊退，公孫離被擊殺後復活打出逆天輸出，久誠極限用一個復活甲換掉了馬超的復活甲，但塵夏豬八戒真的太肉了，團戰打完還是健康的血量，還追死了殘血的呂布，馬超也絲血逃生。

最刺激的畫面來了，超級兵進入水晶，馬超絲血被豬八戒追，久誠在泉水裡轉了一圈都沒回滿血就買了疾跑鞋沖向水晶，hero破水晶心切，以至於衝進了水晶殺掉了馬超，而馬超也完成了使命，清掉了最後的兵線，貂蟬復活，擊殺本就血量不多的周瑜，八戒撤回。

24分20秒，estar多人草叢埋伏，先手開到豬八戒，雙方四大四，呂布因為跳大太深再次被集火擊殺，這是輸掉比賽的前因。

25分，減員的estar伺機搶龍王，貂蟬在龍坑上草，而裴擒虎在清掉小鳥之後被豬八戒抓到機會，剛剛切換人形態的裴擒虎沒能逃走就被豬八戒大招圈住，絲血狀態劉邦傳大無濟於事，沒能傳送過來保下裴擒虎，而貂蟬必須要硬著頭皮上，再次被擊殺，後趕來的馬超也逃不開這個命運，就像葫蘆娃救爺爺一般相繼離開，僅剩劉邦一人無力防守，25分32秒，estar水晶告破。

總結：本局比賽estar貓神打的還好，但是缺少了一些運氣，失誤比之前的發揮多，alan一如既往的不穩定因素，也因在對戰nova時被無心替換。

而hero這邊，久誠多次團戰都能把火鋪滿打足傷害，久酷公孫離輸出逆天，天秀，但最終mvp給到的塵夏豬八戒我也覺得實至名歸，幾次團戰都是塵夏力挽狂瀾。

這局比賽非常的精彩，但estar問題太多，kpl沒能進入總決賽或許對他們的打擊不小，總之還是要加油啊。
