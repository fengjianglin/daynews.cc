在《只狼》中勸退大批玩家，卻始終逃不開嘍啰命的赤鬼

2019-12-16 93條評論 3,651次閱讀 1383人點贊

故事很精彩，主角很帥氣，瀟洒飄逸的通關之後，我們不如來回憶一下，關於那些死於刀下的魔物們，我們究竟讀懂了多少。

守關的赤鬼，悲哀的存在

12月13日《只狼》獲得了TGA年度遊戲大獎，作為第一個守關BOSS的赤鬼又一次被上萬玩家無情碾壓。身為許多《只狼》玩家的勸退boss ，赤鬼教育玩家們這個遊戲什麼是危字當頭。雖然在名義上只是鍛煉玩家墊步的精英怪而已，但是作為時常在各類遊戲作品與影視作品中出現的角色，《只狼》依舊賦予了赤鬼不同的意義。

從劇情上來講，赤鬼的並沒有在整個遊戲中佔有太多的分量，雖然關於他的描述寥寥無幾，但大部分玩家們根據後續的劇情還是可以得知赤鬼是道順用來做人體實驗的實驗者，一個戰爭下的試驗品與犧牲品。在NPC的對話中曾有提及「葦名有一壯漢人稱赤鬼。」一句話簡明扼要的概述了本作中赤鬼的由來。

由於只狼的故事設定在戰國末期，在這個時期鬼也用來形容不被社會接受的「賤民」。赤鬼之名也暴露了角色本身只是一個被社會所拋棄的下等人而已。

在日本有一句傳統諺語「鬼に金棒 （おににかなぼう）」便是對於赤鬼的形容，在日本傳統文化中鬼手上拿的武器各不相同，赤鬼拿著金棒，青鬼拿著桿鉤。這句諺語是比喻本來就強而有力的人得到幫助，使其力量變的更加強大。而放到這部作品的環境下則是與赤鬼的背景故事不謀而合，原本的壯漢在進行了人體改造之後擁有了極其強大的力量。

日本平安時代末期文學作品《今昔物語集》中曾有描述：「鬼身高八九尺開外，目如猿猴，一絲不掛，渾身長滿長毛。」這樣的形象與《只狼》中的赤鬼就極其相似。可以說《只狼》中赤鬼的存在是宮崎英高借鑒了非常多的日本傳統文化典故與傳說故事，並與戰國環境進行了融合的結果。

當一個被社會所拋棄的下等人在經過了肉體改造後成為了鬼。這樣的設定在戰國末期背景下其實是一種諷刺，更是只狼這部作品帶給人們的第一個思考，「賤民」的生死與轉變無人過問，他們生為鬼，死為鬼，並無任何的不同。甚至於在遊戲作品中他們也僅僅是主角的練手工具而已。不得不說，在《只狼》中赤鬼是悲哀的，他擁有「赤鬼」之名，但這卻是他恥辱的代號。

卑微的赤鬼，貪婪的嘍啰

當然像是《只狼》這樣以遊戲難度與整體品質為主的作品並未將這樣的深意很直白的展現在所有人面前，畢竟在日本的傳統文化中赤鬼是最為普遍的鬼怪，他們並不是單一的個體。

日本每年立春的前一天有一個叫節分（せつぶん）的節日，日本家家戶戶都會撒豆驅鬼，而這個被驅趕惡鬼的形象便是赤鬼，因此赤鬼其實可以算是日本傳統文化中最為親民的鬼怪，這樣的習俗在國內可能與新年的放鞭炮驅趕年獸是一樣的。而隨著現代社會的發展，曾經的赤鬼形象也是悄然的發生著許多的改變。

在早期的日本文化中赤鬼是披頭散髮，頭上長角，嘴巴里長著獠牙，手上拿著狼牙棒武器的野人。而到了平安時期和鎌倉時期，日本傳說中將鬼、靈、妖、怪混為一談，赤鬼變成一種可怕的妖怪。

以至於在當今的影視作品和文學作品在進行人物刻畫上，永遠都是把赤鬼當做反派來去刻畫，赤鬼與青鬼也成為了日本傳說中閻羅王的常駐幫手。

在早期作品《桃太郎傳說》中赤鬼還可以作為小頭目登場，而在後來的作品例如《七龍珠》之中，赤鬼則完全變為了沒有存在感的醬油，以至於在當今的許多作品中赤鬼之名常常提起，而已經不再是特指了，變成了紅色的那一類小鬼怪。

在日本的傳統文化中根據不同的鬼怪顏色，這些鬼也代表了不同的罪孽，其中赤鬼所代表的就是貪慾。因此在近現代的大部分作品中我們或多或少的都能看到赤鬼有著一些貪婪的屬性的隱喻。

近代童話小說《哭泣的赤鬼》中作者浜田広介雖然主旨是寫作青鬼與赤鬼的友誼，但是在側面依舊保留了世俗對於赤鬼貪婪的偏見，在得不到人類好感時拚命地想要人類對其產生好感，在喪失了青鬼這個朋友之後卻又嚎啕大哭想要友情。這部作品更是在改編後被搬上了大銀幕成為了著名的動畫作品《朋友：怪物島的納基》。

當然，這些隱喻放在當下社會發展的潮流之中並不能引起太多的共鳴，而赤鬼這樣的親民的又沒有太多特點的族群，則難逃成為綠葉陪襯的命運。

當下的赤鬼，無名者

當下，赤鬼早已經變成了各種動漫及遊戲中跑龍套的存在，《陰陽師》中天邪鬼赤是為數不多的N級卡牌，最低端的地位讓他們連人物傳記都不曾擁有。

在著名民工番《海賊王》中就曾以赤鬼為原型創作過一個巨人族角色赤鬼布洛基的形象，角色在太古之島小花園系列中出現，身材高大但卻沒能夠給觀眾留下太多的印象。

阿魚跟同事們做了一個小實驗，提起赤鬼之名他們的印象中似乎有著許多的畫面，無論是遊戲，無論是動畫，但是要真的列出其名，或者想出他們曾經在哪裡出場則是非常的困難。這種感覺就像是在人群中尋人，既熟悉又陌生。

他們的出現太過平常，平常到幾乎被人視而不見。眾多的影視化、動畫化、遊戲化的作品中，他們已經不再擁有姓名，他們沒有特殊的性格，沒有特殊的感情與故事。他們變成了生活中的符號，變成了代表日本文化的面具，變成了存在於我們生活中最為普通的存在。

歷史長河之下，總有些東西會被我們遺忘，但回到開頭提到的，《只狼》賦予了赤鬼不同的意義，他把歷史，把寓意，把人性再一次賦予在這個已經變成了標誌化的事物之中。

這也許就是真正好的遊戲能夠給我們帶來感動的原因：

「思考不限於未來，思考是讓我們可以回看過去！」
