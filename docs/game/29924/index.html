無限火力春節歸來，它與極限閃擊哪個更受歡迎？

2019-12-23 891條評論 4,038次閱讀 2700人點贊

無限火力作為英雄聯盟出名的娛樂模式，每次上線都會引起玩家的極大關注。之前一直是隨機英雄，上一次更改為自選模式，拳頭也希望大家能夠選到自己喜歡的英雄展開對戰，沒想到這一更新卻引發大家的吐槽。 但是好消息來了！美服已經上線無限火力經典隨機模式，預計今年春節時間國服正式上線。

這個模式大家都非常熟悉了，它出現的次數非常頻繁，上一次開放時間是今年的10月29日—11月9日，為了慶祝英雄聯盟十周年，推出了全英雄自選BP模式，誓要讓召喚師們嗨翻天。沒想到拳頭的一番好意卻引起了反效果。剛開始大家都為自己想玩哪個英雄就玩哪個英雄而開心，玩的次數多了以後所有人都發現了一個嚴重的問題，那就是「選來選去就那幾個英雄，毫無新意」。 全民無盡，上去就是一刀秒，大家都拼手速搶英雄，想贏的都拿雷同的強勢英雄，想快樂的，比如平常很少有人用的輔助貓咪，在這個模式里它能夠無限加速和回血，掛到人馬身上那簡直是光速，自選時間一長大家就出現審美疲勞，紛紛吐槽娛樂模式最終還是變成勝利至上，以往隨機還能出現一個驚喜，那就是平常很少玩的甚至看不起的英雄原來這麼好玩！

但在官宣無限火力的文章下面，還是有許多玩家還是更加期待極限閃擊的回歸。 這個模式已經將近一年沒有上線了，上一次出現是2018年的12月11日到2019年1月13日，持續了一個月的時間，遊戲模式與衝上去就是乾的無限火力完全不同。 在這個模式中，地圖是常規遊戲的濃縮版，野區的紅藍buff是平常的兩倍，並且每個參與擊殺的的玩家都能得到加成。玩家出門只能買一件裝備，你也不會遇見大龍，因為只有峽谷先鋒。最重要的是！遊戲過程會出現各種事件，這就是極限閃擊的精髓。

事件一、被迫打群架 遊戲進行到十八分鐘後，雙方的基地就會變得超級無敵大，然後自由地朝著對方移動。哪一方先擊潰對方的基地就能夠獲得比賽的勝利。這個事件出現就是為了限制遊戲時間，畢竟一局遊戲拖得太長就乏味了。通常情況下都會出現基地互毆情況，大家會選擇集火群毆的方式開干。值得一提的是，在這個事件中，不受玩家操控的小兵也會釋放相應的英雄技能，比如機器人的鉤子，並且小兵一次扔出的鉤子會多達六到八個。

事件二、絕地求生跑毒 這個事件是隨機觸發的，遊戲地圖上會出現一個火圈，當它出現，玩家就必須跑到圈子裡，否則就會遭到降速和越來越高的傷害，火圈也會不斷縮小，造成了大家為了生存躲在一個圈子裡打架的情況。這個事件的結束方法是某一方的英雄被全部擊殺，如果你中途陣亡了，那在分出勝負前不會重生，只能慢慢等結果咯。

事件三、推車 推車推車，推的是什麼車呢？原來到時候在地圖上會出現一個賽恩小車，它周圍會有個圈，如果範圍內只有一方的英雄存在時，就可以開始護送了。玩過守望先鋒的人會很好理解，如果雙方英雄在別人的車圈內，那就是爭奪時刻，車子是不會動的。通過把敵方趕出自己的圈子，塞恩小車才會一直移動到敵方防禦塔，然後將其直接撞毀，獲得勝利！

事件四、大炮 想要快速支援隊友？那麼當這個事件出現後你就可以輕輕鬆鬆坐上地圖上的大炮，讓它將你直接射到地圖中心，再也不用自己趕腿跑路啦，有些玩家把大炮當做玩具一樣玩得不易樂乎，打架與我何干，看我從天而降帥氣登場就完事了！ 其實極限閃擊還有許多好玩的事件，但它的出場次數太少，比起已經玩慣的無限火力，你更期待再次上線的是哪個模式呢？
