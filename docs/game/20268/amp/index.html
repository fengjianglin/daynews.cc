韓國電競協會把Doinb列入名人堂！比肩Faker

天天要聞
2019-12-17

★遊戲馬蹄鐵原創

韓國電子競技協會設立電競名人堂

韓國電競協會宣布將在明年的1月17日舉行2019年的電競名人堂頒獎典禮，並從今天開始設立網上投票。評選方式很簡單，主要針對2019年活躍在國內外的韓國選手，一類是退役的榮譽選手，另外一類則是在今年贏得各項賽事的職業選手。當然主辦方也有自己的評審委員會，粉絲投票只是數據之一。

SKT全隊獲得提名

SKT是今年在韓國LOL方面實現了常規賽2連冠的戰隊，同時在MSI和總決賽上都取得了四強的成績。主要首發選手全部獲得了提名，包含上單Khan，打野Clid，中單Faker，下路Teddy，輔助Effort以及Mata。這是很正常標準的結果，Faker和SKT在韓國電競圈的影響力非常大。即使到了現在，曾經3冠王的輝煌已經逐漸褪去，但短時間內很難有人可以超越取代Faker的電競一哥地位。不過今年也有例外，因為韓國電競協會把奪冠S9總決賽的FPX韓援列了進來。

Doinb和Gimgoon入選

作為在LPL效力的韓國人，雖然中文說得很好，甚至讓人懷疑不是韓國人，但是韓國電競協會還是把Doinb和Gimgoon列入了名人堂票選名單中。這也是讓很多韓國玩家感到驚訝，但畢竟他們的確是在海外聯賽活躍的韓國人，而且S9總決賽冠軍這個榮譽可要比常規賽大太多了。

在韓國跟Faker比人氣？

Doinb在中國LPL毫無疑問是超人氣選手，即使他是韓援，但是多年的中國生活以及在女友的助力下，讓他成為了LPL最出色也是最有成就的選手之一。能夠同時奪冠常規賽+總決賽，這樣的選手在LPL並不多。只是Doinb在中國和韓國的人氣是兩個概念，因為他從出道開始就是在中國LPL，跟韓國職業聯賽可以說沒有半毛錢關係。所以Doinb在韓國只是有人知道這麼一位選手，但人氣並不算高，沒有太多感情可言，這跟在中國的待遇天差地別。

無論是誰和Faker在韓國比人氣是沒有勝算的，從年年全明星票選率第一就可以看出來了。當然最終結果還是看評選委員會和觀眾投票的最終結果，如果Doinb和Gimgoon能夠取代SKT的選手入選名人堂，那就真的太有意思了。
