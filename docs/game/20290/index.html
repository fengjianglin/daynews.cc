陰陽師：決戰京都核心打手式神推薦 茨木強勢逆襲 雪童子迎來高光

2019-12-17 1,678條評論 3,813次閱讀 1476人點贊

陰陽師對於近期開啟的決戰京都活動，不少玩家都在考慮有什麼靠譜和穩定的核心打手式神值得使用，常規的增傷五大式神追書丑清袖基本穩定，基於這次的活動存在著額外機制的緣故，所以特別推薦幾個異常靠譜的式神，一起來看看吧，

兩面佛/茶几

活動前期針對群體陣容的核心打手，這兩位毋庸置疑是最好用的，茶几已經成為了目前PVE活動中最為主流的核心AOE式神了，尤其是攜帶土蜘蛛御魂後爆發出來的傷害，兩面佛都瀕臨下崗的邊緣，當然茶几因為推出活動時間的微妙，導致普及度也不是那麼高，所以沒有的情況下用兩面佛打傷害也足夠。（沒兩面佛去鬥技火速輸幾把刷分也成）

雪童子

活動本身涉及到了非常靠譜的各種式神祝福和BUFF，其中不僅有降低敵方效果抵抗，讓雪幽魂充分發揮控制能力的BUFF，還有目標帶有控制效果傷害提升100%的食發鬼BUFF，這BUFF明顯是提升雪童子的，這一頓破冰能力配合BUFF加成效果，雪童子的輸出傷害意外的非常頂級和靠譜，對單體方面的輸出更加強力。完全可以成為這次活動的主力輸出擔當。

茨木童子

雖然目前主流活動中表現有點過氣，不過這次活動再次迎來逆襲，主要依賴的是活動中茨木童子的祝福BUFF效果，對單體傷害時額外對其他敵人造成50%的真傷，茨木的單體傷害有多靠譜估計大家都清楚，這要是配合這個BUFF，對群體敵人輸出傷害瞬間爆炸，是時候再次讓茨木出山打這次的菊展京都活動了。

酒吞/瑰吞

單體核心輸出式神，最好有兩個酒吞，當然瑰吞也能用，這兩個搭配也好，還是單純的雙酒吞都OK，是一直可以用到活動結束的核心輸出。

燼天玉藻前

雖然針對單體式神方面明顯離吞吞的組合更好，但是選對了合適的BUFF燼天也能用，只要臉好燈姐夠給力的話，那麼燼天的實際傷害是不輸離吞吞的，另外因為這次的特殊BUFF，當大舅媽攜帶茨木BUFF時，實際輸出傷害對AOE比茶几還要高，會成為取代茶几的最優選。
