《天涯明月刀》手游到底啥時候公測？分析師：最晚2020年第3季度

天天要聞
2019-12-11

最近老是發《天涯明月刀》手遊玩家們的吐槽，負能量有點重，今天決定給大家帶來一個和《天涯明月刀》手游相關的分析資料。本文所採納信息，是國內遊戲行業相對權威的機構資料。大家可能不知道，國內遊戲產業有許多分析機構，對行業發函，行業趨勢進行分析和預判，比較有說服力的機構是伽馬數據，遊戲工委白皮書，各地方遊戲協會調研報告等。

主流的觀點認為：《天涯明月刀》手游會在2020年第3季度，也就是7月，8月，9月這三個月間上線公測。第一個主要依據是，按照騰訊遊戲IP改編手游的打法，一般會在內容開發量達到60%左右上線。從上一次測試及12月底的測試預告我們可以分析，目前天刀手游開發完成度大概在50%左右，未來半年，官方將繼續開發10%的新內容，並優化已經成熟的部分。

第二個依據，是騰訊的產品布局和企業盈利需求。2019年是遊戲行業一個非常難熬的寒冬，就連騰訊也陷入了增長乏力的狀態中。在手游領域，騰訊目前的產品整體下滑和老化，繼續優質產品來補位。目前，IOS榜單前30里，騰訊只有兩款遊戲，一款是《王者榮耀》排第11，另一款是《和平精英》，排名第13。這兩款遊戲都是老遊戲，已經氪不動了。

第三個依據，是接入外部IP合作將在明年年中觸頂。今年，騰訊遊戲引進了幾個重磅IP改編手游，一個是《劍網三：劍指江湖》，另一個是《完美世界》，最近一次合作是《天龍八部》。要知道，國內遊戲產業過去20年都在野蠻生長，有實力和精力培植IP的廠商不多，最多半年，國內優質遊戲IP就被耗盡。

毫無疑問，《天涯明月刀》是騰訊遊戲明年手中最重要的王牌，沒有之一。到明年的第三季度，《王者榮耀》最後一波散人玩家將被收割結束，騰訊遊戲需要新產品去接盤。放眼望去，《地下城與勇士》手游開發已在錘死邊沿，重頭戲只看天刀。

背負這樣的使命，《天涯明月刀》手游註定不能輕鬆。在全力提升品質，打造國風手游旗艦的同時，開發團隊不得不在天刀設定足夠多的付費點。所以，天刀手游必須成為叫座又叫好的產品。

明年，遊戲動漫產業的各種標準和要求只會更高，這個行業也已經進入平穩期，不會有太多資本湧入。所以明年的遊戲產業仍然會過的很艱難。《天涯明月刀》手游無疑是最受行業期待，也最受玩家期待的作品。希望這款遊戲不會辜負大家的等待。
