懷舊服25分鐘通MC！Progress公會帶18個狂暴戰一頓猛砍

天天要聞
1 週 ago

最近一段時間，WCL上MC競速排行的競爭真的是異常激烈。11月13日部落公會ONSLAUGHT的七薩滿純菜刀團以27:23的成績擊敗所有聯盟公會暫居排行第一，展現了在薩滿buff下的部落物理dps的兇猛。而全團僅有的一個術士，也是一個工具人的存在。

但是僅僅過了半天，聯盟傳統強會Progress便還以顏色，以18個狂暴戰的超豪華陣容，將MC整體的通關時間壓縮到了25分34秒，實在是令人驚嘆！

我們都知道，經過天賦的不斷修改，1.12版本的狂暴戰士是站在DPS排名頂部的職業，再加上很多極品畢業裝備（例如獅心頭盔）當前階段已經能夠獲取，導致強上加強。所以Progress這次帶上了18個狂暴戰完全是可以理解的，競速打法和普通打法畢竟不一樣，追求的就是一種極限，人員構成的極限、dps的極限、時間的極限。

但是令人奇怪的是，Progress公會這一趟MC還帶了一位平衡德玩家。不得不說這是一件令人匪夷所思的事情，難道就為了給幾個法系職業增加3爆擊么？但是從數據上看這位德魯伊玩家自身的dps卻是很低的，如果是為了爪子的話，是不是換一個恢復德然後增加一個強力dps會更好一點，不知道他們這樣做是不是有點得不償失。

值得一提的是，在打到加爾的時候，他們做出了一個出其不意的舉動，不少團員利用加爾小怪爆炸來進行趕路……在分秒必爭的競速賽中，每一個細節都能成為爭取時間的關鍵。而這裡平時我們都是要避免被炸到，所以說高端玩家有時候想法與常人真的是不一樣，這才能造就非同凡響的結果。
