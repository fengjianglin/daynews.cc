RNG與LNG新賽季陣容基本完成！Joker給出評價，前景並不樂觀？

2019-12-16 361條評論 1,796次閱讀 150人點贊

2019年12月15號，雖然距離轉會期截止還有幾天的時間，但是RNG與LNG卻不約而同的在這一天內官宣了新賽季的基本陣容。RNG這邊是成功續約Uzi、小明以及小虎並且成功簽下前YM打野選手小龍堡以及前FW戰隊ADC選手Betty，再加上上單狼行合同並未到期，RNG戰隊新賽季的陣容基本上已經完成了。

而LNG戰隊這邊，則是引入了前SN戰隊的中單選手Maple以及前TES戰隊打野選手Xx。雖然關於聖槍哥、阿修羅以及小段還並沒有官宣，但相信大概率還是會留隊的。所以，LNG這邊的新陣容為上單聖槍哥、打野Xx、中單Maple以及雙人路阿修羅跟小段。新賽季的陣容基本上也是完成了。

那麼，既然在陣容上有所改變，與之前的陣容相比這兩個隊伍會是如何呢？而在這兩支隊伍分別進行官宣之後，英雄聯盟圈內知名「毒奶」Joker也是針對這兩支隊伍目前的情況進行了分析。Joker表示，「RNG的引援差強人意，小龍堡也許是一個出色的打野，姑且認為能夠有接近小天的實力，但RNG最需要的其實是一個Doinb這樣的角色。」另外，Joker對LNG的新陣容也是進行了分析，Joker表示，「LNG難上加難，原本的上野體系又要回爐改造，Maple在S9賽季表現平平不知道明年如何？」

從Joker的分析上來看，他對新賽季的RNG與LNG保持著並不樂觀的態度。不過，因為Joker一向是擁有「毒奶」之稱。所以，不少LPL的粉絲們還感謝了一下Joker。但事實上，Joker分析的有問題嗎？其實，並沒有太多問題。相信不少人都已經見證過太多隊伍在引入強力選手之後，往往發揮不出這個陣容表面上的全部實力。這其實還是要跟隊伍的「化學反應」有關。個人能力雖然重要，但是否適應戰隊的體系，是否能夠與團隊很好的配合，這些也是相當關鍵的一部分。

所以，從RNG與LNG這兩支隊伍的整體情況來看。RNG因為失去了Karsa，可能在新賽季之初會遭遇到一些需要解決的問題，小龍堡能否很好地融入團隊，適合RNG的體系也就成為了RNG在新賽季最迫切的事情。而LNG戰隊，從表面上來看，更換了比較關鍵的中野則就意味著真的會在體系上發生改變。而這種變化是否能夠讓LNG發揮的更好，同樣也是需要等到新賽季開賽我們才能夠知道。

最後，有一句俗話，「是騾子是馬拉出來遛遛」。很多東西其實只有在賽場中才能夠看得出來，就像我們一直以來認為哪支隊伍是「銀河戰艦」一樣，但「銀河戰艦」翻車的例子還真的是不少。所以，RNG與LNG這兩個隊伍的新陣容是否能夠在新賽季有所作為，還是讓我們拭目以待吧！
