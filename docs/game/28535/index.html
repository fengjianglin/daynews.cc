堡壘前線：後坐力是應該消除還是應該避免？這是一個問題

2019-12-23 590條評論 2,583次閱讀 641人點贊

在《堡壘前線：破壞與創造》手游中的槍械可以說是數不勝數，而且絕大多數槍械都有著一個特點——具有後坐力，後坐力這個屬性在不同的槍械上表現的效果不盡相同，但是影響都是一樣的，那麼如何儘可能的消除後坐力帶來的影響也自然成為了眾多探險家討論的話題。

後坐力如何消除似乎也已經成為了一個重要的問題，但是我們可不可以換一種方式解決問題呢？先將消除後坐力放在一邊，而是來一起看看哪一種槍械的後坐力小，更適合玩家上手吧！

在眾多的槍械中，「步槍」這個種類可以說是十分的搶手，其憑藉著各項都十分均衡的特點深受廣大探險家的喜愛，不過雖說各項屬性都均衡，其弊端也是十分的明顯，那就是射速比不上衝鋒槍，威力比不上狙擊槍，突出特點有沒有特殊武器明顯。其中AK47就是處於這樣尷尬境地中的一把槍械。

AK47步槍的威力在步槍當中已經屬於中上等了，但是其缺點就是後坐力十分的高，如果不會壓槍的話一梭子子彈沒打完準星就已經飄到天上去了，所以如果不是壓槍大神的話還是少碰這個吧。

VSS狙擊槍在所有狙擊槍中應該是最特殊的一個了，因為它自帶消音，所以在探險家的眼裡也是十分的搶手，畢竟這樣的一把「神器」每個人都想佔有啊。

但是相對來說，VSS狙擊槍的後坐力也對探險家有著不小的影響，尤其是在開鏡的時候，如果想對一個對手進行連續射擊的話還需要練習一下點射的技巧，不然可是很容易打偏的。

你是否還在因為槍械的後坐力太大而苦惱？你是否想要一把沒有後坐力卻有十分強大的武器？別急！這些特殊武器都能滿足你！後坐力？光子發射器表示並不清楚後坐力是什麼，但是它只知道在近距離內鎖定對手就完事了，這樣的武器你是否想要呢？

M134加特林此時也站了出來，表示對自己的威力十分有自信：高額的載彈量加上微小的後坐力完全可以壓制任何對手（如果能夠忽落掉精準度這個問題的話），讓你擁有橫行啟明星之谷的實力！怎麼樣？有沒有心動呢？

好了，相信大家在看了秀妹兒介紹的關於槍械後坐力的講解後心中都有了如何選擇適合自己的槍械的想法了吧？是選擇後坐力大但是屬性均衡的AK47步槍，還是擁有不同屬性的特殊武器，大家肯定都有了決策，那麼秀妹兒就祝大家能夠拿著適合自己的槍械早日成為「破曉之星」哦！
