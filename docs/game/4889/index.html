《寶可夢劍盾》顫弦蠑螈特性介紹與對戰思路講解

2019-12-04 0條評論 15次閱讀 0人點贊

摘要

《寶可夢劍盾》顫弦蠑螈是本世代新加入的寶可夢，這隻寶可夢的使用性不錯，不少玩家不了解其對戰時的強度。下面就為大家帶來由「七巧蒙音」分享的《寶可夢劍盾》顫弦蠑螈特性介紹與對戰思路講解，一起來看看吧。

　　顫弦蠑螈，八代登場的新寶可夢，稀有的電+毒屬性。

　　種族值：HP75物攻98物防70特攻114特防70速度75

　　特性：朋克搖滾，正電/負電（此項為高音低音區分）。

　　夢特：技術高手

關於進化

　　性格外向（勤奮、固執、頑皮、勇敢、坦率、淘氣、樂天、馬虎、浮躁、自大、急躁、爽朗、天真）進化為高音形態（就是金色外形）剩下的性格則進化為低音形態（就是藍色外形）。很多人喜歡金色外形，但金色的都是物攻向技能，那我又想要金色又想打特攻怎麼辦呢，你按照金色所需的性格進化後吃個薄荷再玩特攻就行了。

　　技能池，以毒和電屬性為主，有一招普通系威力140的爆音波，配合特性朋克搖滾可以打出不錯的傷害。

抗性

　　很多人一眼看到這個種族值，肯定會想，哇，毒+電雙防那麼低還速度75，這鼴鼠一個地震你十萬條命也接不起啊，更何況鼴鼠速度88還有一堆人喜歡極速套圍巾，但我們再仔細看一看抗性，毒+電的弱點只有一個地面4倍和超能2倍，飛行，格鬥，毒，蟲，鋼，草，電，仙全都只有0.5倍，即使我只有75的hp和70的雙防，你不是地面或超能可能也沒法閉著眼一擊秒我，所以毒+電這個組合其實還是有著不錯的抗性面。

　　還有一個關鍵問題，八代的地面系非常吃香。

　　沙漠蜻蜓：「這麼說，加勒爾的地龍就是我……」

　　鼴鼠：「沒錯，正是在下烈咬陸鼠。」

　　在這鼠頭地龍橫行的環境里，頂著破格和高物攻種族值，什麼迷你q冰頭企鵝上去就一發鐵頭，什麼電烤箱洗衣機（洗衣機應該還能站得住）上去一發地震。

　　再看看我們的顫弦蠑螈，毒+電，雙本皆對地面+鋼的鼴鼠無效，剩下一個爆音波打鼴鼠還打不動（普通打鋼0.5），75的速度遇上鼴鼠瑟瑟發抖。

　　但顫弦蠑螈真的是拿這烈咬陸鼠一點辦法都沒有嗎？當然不是。

　　我看了看技能表，毒電有個非常強的buff技能——換擋（攻擊提升1級速度提升2級）。

　　首先，我是一名多拉貢盪死愛好者，這種速度還多+1的多拉貢盪死著實讓那些速度尷尬的龍舞眾羨慕。

　　但是有什麼用呢，毒電物攻只有98，大部分人只拿毒電打特攻，你會個換擋又有什麼用？

　　於是我又看了一眼技能池，居然有火焰拳！

　　這意味著什麼，這意味著我可以玩物攻毒電換擋後嘗試著對鼴鼠來一發火焰拳。

對戰思路測試

　　物攻+1的固執毒電使用火焰拳有75%幾率擊殺鼴鼠，最低傷害178最高傷害210。（此鼴鼠努力為252攻252速4hp爽朗or固執性格道具圍巾or磚頭）

　　就先不談那些71地鼠和拉hp努力的，換擋一發後的固執毒電（努力252速）百分之百快過爽朗圍巾的極速鼴鼠。

　　換擋後固執毒電速度254，爽朗圍巾極速鼴鼠速度231。

　　有人可能會說，你換擋有什麼用，第一回合還是鼴鼠快，你沒換擋出來，地鼠一發地震你豈不是爆炸？

　　對，它地震出來我的確爆炸，所以換擋物理流毒電推薦道具為氣球。

　　第一回合你震不到我，鐵頭打不死我（鋼打電0.5），我換擋出來下回合一發火焰拳錘死你，有本事你連續岩崩我害怕（雖然我還真被連續岩崩害怕弄死過）。

　　雖然即便這麼說，換擋後火焰拳也只是有75%的幾率能殺鼴鼠而已，但比起看見鼴鼠就嚇得瑟瑟發抖的特攻毒電，我們物理毒電有一轉攻勢的機會了。

　　雖然極巨化地鼠完全不虛你啥火焰拳。

　　還有死神板沙河馬等一票地面系毒電依舊不好處理，本文僅是提供一種思路。
