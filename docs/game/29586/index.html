懷舊浪潮來襲｜曾經的經典遊戲，哪些能戳中你的心懷？

2019-12-25 498條評論 1,812次閱讀 620人點贊

隨著前兩天上架的 Rewound 在 iPhone 上復刻了 iPod Classic為大家掀起一場懷舊浪潮，那麼除了 Rewound還有什麼經典？今天我們就來懷舊一下那些曾經的經典。80經典小霸王遊戲、俄羅斯方塊、windows95系統這些應用的時代，你是否經歷過呢？

先來一波經典的80後遊戲回憶殺，你準備好了嗎？

Tetris俄羅斯方塊 for Mac(經典休閒遊戲)

由小方塊組成的不同形狀的板塊陸續從屏幕上方落下來，玩家通過調整板塊的位置和方向，使它們在屏幕底部拼出完整的一條或幾條。這些完整的橫條會隨即消失，給新落下來的板塊騰出空間，與此同時，玩家得到分數獎勵。沒有被消除掉的方塊不斷堆積起來，一旦堆到屏幕頂端，玩家便告輸，遊戲結束。

俄羅斯方塊自然是俄羅斯人發明的。這人叫阿列克謝·帕基特諾夫（Алексей Пажитнов 英文：Alexey Pazhitnov）。俄羅斯方塊原名是俄語Тетрис（英語是Tetris），這個名字來源於希臘語tetra，意思是「四」，而遊戲的作者最喜歡網球（tennis）。於是，他把兩個詞tetra和tennis合而為一，命名為Tetris，這也就是俄羅斯方塊名字的由來。

FC經典遊戲600合集for mac(小霸王遊戲)

曾經的「小霸王」遊戲機，讓我們許多80年後、90後度過了一個歡樂愉快的童年，《超級馬里奧兄弟》《赤色要塞（Jackal）》《魂斗羅》《暗黑破壞神》耳熟能詳就。

經典中經典，雖然如今這類遊戲機和卡帶幾乎已經退出遊戲市場，但經典永遠留在了我們這一代人心中。小霸王FC經典遊戲600合集，帶你回顧童年，重溫經典吧！

魔獸世界懷舊服 (wow經典懷舊版) for mac

《魔獸世界》經典懷舊服忠實地再現了原版《魔獸世界》。戰鬥機制、經典角色模型和技能樹都能給你帶來真實的原版體驗。

在《魔獸世界》經典懷舊服未來更新的眾多內容中，你將發掘：

經典的團隊副本
投身於熔火之心和奧妮克希亞等史詩級的40人團隊副本。
經典的天賦
升級並使用來之不易的天賦點，要小心謹慎哦。
經典的世界
一個充滿挑戰的戶外世界正在等待著你。帶上好友——你需要他們的援助！

反恐精英 for Mac移植版(CS經典射擊遊戲 1.6移植版)

Cs 2.0 Mac版是大名鼎鼎的射擊遊戲半條命反恐精英（英文：Counter-Strike；簡稱CS）的Mac平台版本。反恐精英:《反恐精英CS 2.0》是一種以團隊合作為主的第一人稱射擊遊戲，它是由著名遊戲《半條命》（Half-Life）的其中一個遊戲模組（MOD）衍生而成的遊戲。雖然只是簡單的一個MOD，但是它卻在全世界風靡起來，WCG、CPL、ESWC等世界頂級電子競技大賽都以CS1.6作為比賽項目.　

Counter Strike mac版是一款基於macOS電腦開發的真人動作警匪射擊遊戲，在Counter Strike mac版遊戲中玩家可以選擇喜歡玩的地圖包括倉庫、船等等地圖，還可以選擇綁架人質的屠夫和警察哦，其中Counter Strike mac版遊戲的射擊效果還是很不錯的.

經典應用，什麼時候用都是一波浪漫的回憶！

New Paint X for Mac(經典的繪圖工具)

New PAInt X for Mac是應用在Mac上的一款經典的繪圖工具，New PAInt X Mac 像數字畫板一樣，你可以用它創建新圖像、編輯現有圖像，或者給照片添加標記，內置150種畫筆，具有不同外觀和紋理的線條。

支持png,tiff、bmp、jpeg、gif等常見的文件格式的導出或轉換，有了它能讓您創作出優秀的作品。

FlipClock X for Mac(翻轉時鐘屏保)

這是一款非常好用的時鐘屏幕保護程序，既防止別人亂看或者亂動電腦，看起來也乾淨利索些。主要以數字時鐘為主題。可以顯示秒鐘，它能保護屏幕安全，並且提供時間查看功能。

FlipClock X for Mac帶動態翻頁效果的數字時鐘屏保，和htc手機上的超大字體數字時鐘相似，可以讓整個顯示器顯示一個數字時鐘，時間改變時還會有翻頁效果，十分的典雅。

Winclone類似於Windows的Ghost，可以備份和還原系統或分區，下面提供Winclone專用的Windows10 1809 64位專業工作站版，系統內置BootCamp驅動，首次進桌面會自動調用驅動。安裝非常簡單，分割Fat分區，用Winlcone恢復進該分區，重啟電腦即可。

Microsoft Edge for Mac(edge 瀏覽器)

edge瀏覽器 mac版不僅擁有更好的性能、穩定性以及網頁兼容性，對追求效率的人士而言，最重要的是它還能「原生支持」安裝和使用 Chrome 插件擴展。從上手體驗來看，微軟對新 Edge 的性能優化調教得非常出色，Edge瀏覽器下載無論啟動速度還是打開網站的速度感覺都與 Google Chrome 幾乎一樣快，甚至內存佔用的控制還比 Chrome 要好。

edge瀏覽器 mac版不同於傳統IE的瀏覽器，功能很全面，不僅內置微軟Contana，可以為用戶帶來更多人性化的服務，而且Microsoft Edge瀏覽器還有著支持插件擴展、網頁閱讀注釋等特色功能。

windows95 for Mac(windows95系統體驗軟體)

Windows 95 發佈於 1995 年，在 Windows 家族中看，算是當時具有創新的操作系統。即使後來大多數人最初使用 Windows 系統的 Windows 98，也極大受到 Windows 95 的影響。如果你想再感受下 Windows 95 的使用，最方便的辦法是在瀏覽器中運行 Windows 95。

你可能聽說過 Windows 98，但你聽說過 Windows 95 嗎？
Windows 95 是微軟第一代 Windows 9x 系操作系統，同時也是奠定微軟操作系統霸主地位的重要產品。
時隔二十多年後，國外一名開發者 Felix Rieseberg 成功地將 Windows 95 操作系統做成了一個 app，並免費共享給所有希望嘗鮮 Windows 95 的用戶。
是的，你現在就可以在 Windows 10、Linux 以及 macOS 等平台上安裝 Windows 95，接下來，就帶大家搶先體驗一下二十年前的個人 PC 是什麼樣的吧！

Kaka for Mac(瑞星卡卡桌面小工具)

Kaka是一款macOS平台的多功能桌面輔助工具，去掉了瑞星無用的殺毒和防火牆功能，只保留小獅子，而且還有一些實用的功能，非常實用！

1. 返回桌面。
2. 隱藏桌面圖標。(對多屏幕支持)
3. 關閉顯示器。
4. 防止系統休眠,(合蓋也支持哦)！
5. 自動隱藏滑鼠指針。
6. 自動隱藏桌面。(雙擊桌面就可以顯示哦)
7. 根據屏幕亮度自動切換黑暗模式。
8. 增加自定義快捷鍵功能。
9. 支持Touch Bar。
