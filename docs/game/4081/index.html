全球僅一人擁有！束縛墜飾，魔獸世界60年代唯一橙色項鏈

2019-12-10 0條評論 16次閱讀 0人點贊

摘要

說起魔獸世界60年代的橙色裝備，那你肯定會想到風劍、炎魔錘、雞腿杖這3把武器。

說起魔獸世界60年代的橙色裝備，那你肯定會想到風劍、炎魔錘、雞腿杖這3把武器。

但，嚴格意義上說起，其實60年代還有一個橙色項鏈，這便是：束縛墜飾。

當然，該項鏈全球僅一人擁有，當Nurfed公會在美服公測4個月之後，一次擊殺熔火之心5號BOSS迦頓男爵之時，無意中掉落這個屬性絕對逆天的橙色項鏈，該公會戰士Noktyn最終將其拿走。

不得不說，橙色項鏈的屬性著實逆天，加13力量，8耐力，5敏捷，24火焰抗性，24自然抗性，被擊中之時還會對攻擊者造成4點自然傷害。

毫無疑問，這樣的屬性對於MT而言顯然有著質的提升。

不僅如此，帶著這個橙色項鏈，胸口還會閃爍著藍色的光球，其特效十分耀眼。

只是很可惜，暴雪立馬將其從掉落列表中移除，但Noktyn的橙色項鏈卻將其保留下來，於是乎，他成為全球唯一擁有橙色項鏈的玩家。

直到如今，我們還是在懷舊服中的資料庫中找到了束縛墜飾，只是很可惜，你是絕對無法拿到它了！
