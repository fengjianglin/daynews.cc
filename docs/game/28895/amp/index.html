黃金大獎賽16強即將出爐，RNG、EDG已鎖定席位，SMG能否衝出重圍

天天要聞
2019-12-21

目前大獎賽進行到了小組賽的最後一天，就在20號則會決出最後的16強名額，從目前總分來看的話Ark、EDG以及RNG這樣的隊伍出線幾乎比較穩了，而豪門戰隊4AM則以80分積分排在十四位，最關鍵的還是要看20號的比賽是否能夠打好了。

要進入到十六強，那麼目前積分總榜在這附近的隊伍都是很有希望的，只不過現在還有一些隊伍剩下六場比賽沒有打，因此最後的晉級席位還是撲朔迷離，這也是為什麼說4AM十四名位置不太穩的原因，比如說OMG這支戰隊目前86分的情況下還有六場可以累積積分，但是前方13到15位置的隊伍積分相差不多十分左右，但是這些隊伍都是已經打滿了十八場了，這就讓並沒有打滿場數但積分靠前的隊伍有了機會。

至少在小編看來，到20位置左右的隊伍都是有機會的，想要在六場左右拿到三十幾分還是很有希望的，不過在比賽中還是很容易出現各種意外狀況，特別是敗者組有4am以及smg這樣戰隊的情況下拿分還是相當困難的。不過對於4AM這支戰隊來說晉級還是很有希望的，目前的80分讓這支戰隊在剩下的比分中，如果能夠順利拿到二十多分左右都是能夠成功晉級的，目前並沒有什麼隊伍能夠威脅到他們。

小編比較擔心的還是SMG這支戰隊，可能很多人也注意到了，這一次SMG在小組賽的表現還是很不錯的，特別是和4AM的幾次對決中不管是拉槍還是位置轉移，都為4AM帶來了很大的麻煩。然而這樣一支隊伍目前只有60分的總積分，這就導致在剩下的六場比賽中最少也得爆發爆發才行，差不多要到一百來分左右才能夠穩定晉級，這關鍵還是要看20號比賽當天選手的狀態了。

在這些戰隊還在為十六強進行拼搏的時候，本次黃金大獎賽的全明星投票也正式結束了，並且在21號當天線下的全明星也會正式啟動，在這一次的投票中，玩家選出的人氣排名前十六的選手，會自動成為隊長並且組成隊伍進行比賽，據了解在這一次的全明星比賽中還會邀請國外豪門戰隊FAZE的選手參與進來喔。

小編在這裡只能感慨大獎賽的十六強將要決出的同時，人氣十六強也即將決出，這一次的黃金大獎賽可以說是下足血本了。從這一次的比賽中也發現了很多年輕隊伍的出現，比如說RNG戰隊在CDD的帶領下繼續向自己的第二個冠軍進步，而EDG也是穩定前進，雖然目前沒有保持在積分榜榜首的位置，但是排名前三的積分還是相當讓人欣慰的。同時在這一次的比賽中，能夠看到4AM以及OMG這樣戰隊狀態的調整，總的來說對於這一次的黃金大獎賽總決賽小編個人是十分期待的。
