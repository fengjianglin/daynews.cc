夢幻西遊：新召喚獸即將放出，C66獸決價格瘋長，當心糖桂花再現

2019-12-11 843條評論 4,265次閱讀 2137人點贊

​各位夢幻玩家大家好，我是你們的老朋友李叫獸，公眾號「叫獸說夢幻」每天為你帶來最新遊戲資訊

新資料片還有一個月就放出了，目前已經確定資料片會放出新地圖和新召喚獸，於是很多玩家開始囤貨，各個區的C66和獸決價格開始上漲，大家都盼忙著新召喚獸出來能發一波財。然而看到當前這一幕，叫獸還是要提醒大家，囤貨有風險，入市需謹慎。因為夢幻最近這一年發生了不止一次糖桂花慘案。接下來叫獸和大家聊一聊近一年來發生的囤貨翻車案例：

案例一：元宵節

以叫獸所在的區鎮淮樓為例，2019年的元宵節攻元宵商人剛開始90W收，後來降到了85萬，到元宵節活動快結束的時候80萬收。快一年了現在鎮淮樓的攻元宵還是賣90萬。造成今年元宵價格不升反降的原因，主要是因為今年延長了元宵節的時間，往年元宵節只有6天，今年延長到了10天，因此元宵的產量提高近一倍，產出過剩導致元宵價格漲不起來。

案例二：暑假活動松煙墨

今年暑假活動7月2日放出松煙墨，7月9日維護後，松煙墨可以通過擺攤出售，由此潘多拉魔盒被打開，各個區商人開始瘋狂囤貨松煙墨，價格一度被炒到10萬，個別區甚至漲價至15萬。8月13日系統維護後，提升活動中松煙墨的產出。8月20日維護後，系統開始出售松煙墨，定價8.8萬。隨後松煙墨的價格一瀉千里，最後跌到5-6萬/個。

案例三：重陽節細娟

今年10月1日，重陽節活動放出，在活動放出後的半個小時，個別區細娟的價格漲到3-4萬/個，商人開始大量囤貨，然而細娟的產出率太高，加上今年的茱萸香囊產出太垃圾，細娟價格一跌再跌，最後跌到幾百兩一個，更有商人直接擺1兩銀子，相當於白送，甚至有商人囤6萬個細娟虧到吐血。

案例四：囤神獸

11月25日，維護公告放出，神獸可以兌換神兜兜，神兜兜可以兌換千億獸決。藏寶閣的垃圾神獸從1100被炒到接近2000。當時論壇上的這位老哥由於得到的消息比較晚，看人家囤神獸也跟風囤，2000/只囤了20個神獸，當時信誓旦旦稱未來垃圾神獸的價格會漲到2500。半個月過去了，垃圾神獸的價格不漲反跌，現在已經跌到1500了。

和大家說這麼多，就是想告訴大家一個道理。看別人囤貨賺錢，自己盲目跟著囤，如果不懂其中技巧的話，最後只能做接盤俠。夢幻西遊中的物價千變萬化，影響價格的因素有很多，只有嗅覺敏銳，洞察先機，隨時了解市場變化，才可能在這場博弈中賺到錢。當然所謂富貴險中求，想賺錢不冒點風險是不可能的，也沒有穩賺不賠的買賣，叫獸只是奉勸大家，囤貨保持謹慎的態度，隨時關注市場動態，並作出相應調整。
