熱血傳奇：幸運9套裝的打造，不僅能刀刀最大攻擊，還強大到變色

天天要聞
2019-12-17

大家好，我是小逗說傳奇，今天小逗給大家說一說，關於我本沉默版本中，幸運+9套的打造與獲取。

運9套是熱血傳奇中最強大的套裝，不僅能刀刀最大攻擊，而且還強大到變色，一般來說，運9是由武器跟項鏈組成的，這裡就分開給大家說一說，武器跟項鏈的幸運達成。

第一：武器

武器的幸運打造很簡單，喝油就對了~但是想要喝出幸運+7，那也是需要運氣的。

一般情況下，能喝到+4或者+5算是非常不錯了，至於+7，就需要用元寶來弄了。

找到商人這裡，用30瓶祝福油+30W元寶，跟他兌換一瓶超級祝福油，然後就能愉快的合成了一瓶100%增加幸運的超級祝福油啦。

有了超級祝福油，武器就能輕鬆懟上7了。

第二：項鏈

武器上7就是頂級了，沒有辦法再提升，這時候我們需要弄項鏈。

項鏈的獲取是比較特殊的，首先要打怪獲取到精靈の翔，然後再去土城武器點找殭屍。

這次沒刷啊，有點坑

找殭屍，打碎片，然後把精靈の翔合成帶幸運的精靈の翔

再然後，找這個NPC合成。

帶幸運的項鏈合成成功，最後升級一下幸運就好了。

幸運+9之後，我變色了0 0

另外，這個幸運項鏈可以拆的，等你要換項鏈的時候，把這個項鏈給拆了，又能收穫到原來的材料喲。

沉默版本幸運+9的打造就是如此簡單，看清楚了的大佬，就趕緊去土城武器點打殭屍吧，幸運+9等你喲。
