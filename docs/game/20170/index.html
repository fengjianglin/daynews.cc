暴走大俠：寶石箱能出99個寶石？玩家測試10次，倉庫被寶石塞滿

2019-12-18 653條評論 3,834次閱讀 1407人點贊

氪金不暴走，大俠全靠肝，大家好，我是「爆肝大俠」小魚乾，歡迎來到由小魚乾開講的《暴走大俠新鮮事》。本期小魚乾將給大家帶來寶石箱開出99個寶石、兩個賬號99個「狂」、集卡「小訣竅」等新鮮事，話不多說，抓緊時間和小魚乾一起來看看吧！

寶石箱能出99個寶石？玩家測試10次，倉庫被寶石塞滿！

今天暴走大俠的限購中，上架了「稀有寶石箱」，200翡翠葉一個，開啟後可以隨機獲得一定數量1~3星的寶石，最大數量為99個。

但估計沒有多少玩家會相信，這個寶石箱子真的能開出來99個寶石吧，如果真的能開出來99個，都不求是2星、3星寶石，只要是1星就很滿足了，因為目前在珍寶閣，1星寶石每個是250翡翠葉，2星寶石每個是500翡翠葉。

我們就以2星寶石來算，單個1星寶石最少都值333翡翠葉，99個就是三萬多的翡翠葉，就算開到99個1星寶石，都直接賺翻，暴走大俠真的有可能這麼大方嗎？

一名玩家為了驗證這個寶箱能不能開出來99個寶石，就花了2000翡翠葉，把10次寶箱都開了，沒想到真的開出來了99個1級的赤炎石，看來真的是可以的！

但小魚乾總是發現，每次像轉盤之類的活動出來，就有一批到處秀自己單抽出來秘籍的玩家，真的不知道是不是「托」，現在抽寶石都有人出來秀了，實在是令「非酋」玩家咬牙切齒！

有玩家也曬出了自己抽到99個1級黃凝晶的倉庫圖，寶石都把倉庫塞滿了，而小魚乾連寶石合成系統都沒有試過，因為根本沒有3個相同的寶石，實在是羨慕嫉妒恨了。

寶石這個東西也是實實在在提升戰力的，但平民的話，小魚乾還是不建議去碰，直接買太貴，去抽這個寶箱，有人家這運氣嗎？

玩家兩個賬號都集了99個「狂」，活動結束後留下遺憾！

話說第三期的集字活動已經結束了，有多少小夥伴兌換到了竹子寶箱呢？小魚乾沒怎麼打，也就換了一個鑰匙和一些破鏡丹，反正自己有螃蟹弩，就不打算花時間去肝了！

不過話說回來，很多玩家都反應，這期的集字活動「狂」字掉率特別低，感覺比「感恩有你」的「有」還要低，有玩家購買體力都集不滿，估計也是運氣太差了。一名玩家兩個賬號，都收集了99個「狂」字，真的是氣死了，活動結束後，就只能兌換成9900金幣，遺憾地看著竹子寶箱的離開了！

玩家單抽出紅卡，玩家透露集卡「小訣竅」！

最近又有玩家曬出自己單抽出酒館紅卡的圖，運氣也是不錯的。這時候就有玩家問題，現在在酒館兌換處，基本上就沒見到紅卡了，就算見到也是比較後門派的紅卡，換了總覺得虧，難道除了抽酒館和限購英雄令，就沒辦法獲得紅卡了嗎？

有玩家也是透露了一個獲得紅卡的「小訣竅」，就是多存一些已經集滿緣分，可以分解掉的卡，不要存超過900的招賢令，這樣系統發現你的招賢令不夠，就會給你出紅卡，這時候分解掉多餘的卡，用來換紅卡就好了！

小魚乾也不知道這個「玄學」管不管用，因為自己已經存了三千多的招賢令，果然已經一個多星期沒遇到紅卡了，沒機會用掉，實在是太難受了！

​那麼以上就是本期《暴走大俠新鮮事》的全部內容啦！最後，感謝您的閱讀，各位路過的大俠給小魚乾點個關注吧~
