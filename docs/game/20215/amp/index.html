召喚師技能調整：狂暴新增法師屬性，干擾可使自家防禦塔無敵

天天要聞
18 小時 ago

一到賽季末，體驗服就成為了兵家必爭之地，別無其他，只有從這裡才能夠看到新賽季的大概走向。

畢竟遊戲策劃們也需要一個地方事先測試遊戲中的改動是否合理，比如說之前大家在體驗服看到的盤古的調整以及安琪拉大招的轉動速度，都是經過了幾番的調整。

所以在昨天，體驗服又經過對召喚師技能進行了調整。很可惜，之前在變身大作戰中的傳送還是沒有上線。

不過，卻有三個冷門召喚師技能脫穎而出，勢必成為峽谷中的黑暗勢力。

一、狂暴

首先在目前遊戲中，狂暴所能帶來的收益是：增加60%的攻擊速度和10%的物理攻擊，持續5秒。

在體驗服改版之後的狂暴所能帶來的收益是：增加10%傷害、25%韌性、20%物理和法師吸血。

為了方便大家判定狂暴調整的程度，用同裝備同等級的孫尚香打一套技能的傷害作為對比，得出的結論是，體驗服和正式服的傷害差別不大。

但體驗服中提供的25%韌性和20%的吸血效果，在實戰中的報命能力卻很強，尤其是吸血效果。

體驗服調整過後的狂暴適合不缺位移、缺傷害，還不走打野位置的英雄使用。

二、干擾

在正式服，目前干擾所能帶來的收益是：沉默機關持續5秒，6秒內疊加效果只有1秒的有效時間。

而體驗服在這個基礎上增加了一點：對已方機關使用，可使其無敵，60秒疊加原理同上。

與前面的干擾想必，干擾才是真正的史詩級加強。尤其是在逆風高地環節，一個干擾就可以使防禦塔無敵，直接推遲了雙方進攻的節奏。

把握好時機，一人極限守家也不是問題。

不過，目前看來這個技能除了劉禪、米萊迪等帶有推塔屬性的英雄適合。畢竟在逆風局才體現出實力的技能，在選英雄環節誰也不能提前預知局勢。

三、弱化

最後一個調整的技能就是弱化，正式服中的作用：減少周圍敵人50%，持續3秒。

在體驗服中調整為：減少周圍敵人30%，持續5秒，同時自身獲得20%免傷。

時間上從3秒提升到了3秒，減少敵人傷害從50%降到了30%，不過增加了20%的免傷效果。

對於不需要閃現操作的坦克英雄或是不過的效果，有些人也覺得走邊路的刺客也可以帶弱化突進切後排效果也不錯。

好啦，相信以上的召喚師技能調整對於目前遊戲中千篇一律的閃現、懲戒現象會有所改變。
