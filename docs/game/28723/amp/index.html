德杯：為什麼RNG、OMG、LNG三隊同分，RNG直接晉級？用數據說話

天天要聞
2019-12-22

英雄聯盟德瑪西亞杯出現了戲劇性的一幕，在C組的5支戰隊相互交手中，JDG以4勝的戰績脫穎而出，而YM以全負的戰績遺憾落敗，剩下的三隊RNG、OMG、LNG打成平手。但是根據比賽規則三支戰隊並沒有進行加賽，而是RNG直接攜手JDG晉級，這讓很多觀眾疑惑不已，解說在直播時的解釋也把各位小夥伴徹底聽懵了。

那接下來聯盟君為大家解讀，計算一下為什麼這輪交手中RNG最後出線了呢？先讓我們一起來看看德瑪西亞杯關於小組賽的決勝原則吧。

在德瑪西亞杯6.23小組賽決勝規則中，因為是三組同分，所以我們可以直接鎖定6.2.3.1規則。

a）當三支隊伍打平的時候，優先比較勝負關係。相互對戰勝場多則名次更高，但是RNG 0-1 OMG，RNG 1-0 LNG ，OMG 0-1 LNG。

由此可見三隊相互對戰的記錄都是1勝1負，因此規則6.3.2.2 a規則就分不出這三隊的名次。

那我們接著往下看b規則，根據隊伍小組賽中所有獲勝場次的平均用時（計算方式=獲勝場次總用時/獲勝場次，精確到秒）進行排名，平均用時越短的排名越高。

RNG勝LNG的比賽時長為33分40秒，勝YM的比賽時長為27分20秒，獲勝場次平均用時：（33分40秒+27分20秒）/2=30分30秒；

OMG勝RNG的比賽時長為30分55秒，勝YM的比賽時長為30分50秒 ，獲勝場次平均用時：（30分55秒+30分50秒）/2=30分53秒；

LNG勝OMG的比賽時長為29分15秒，勝YM的比賽時長為26分55秒，獲勝場次平均用時：（29分15秒+26分55秒）/2=28分5秒。

這裡要注意的是規則中說的是小組賽中所有獲勝場次的平均用時，因此有些小夥伴可能忘記把三隊和YM比賽的勝利時長算到平均值裡面了。

所以最後的結果是LNG 28分5秒 大於 RNG30分30秒 大於 OMG 30分53秒。

b規則第二條「但若以平均用時最短的隊伍為標準，其他隊伍與其平均用時相差在 150 秒以內，則這些隊伍需要進行加賽」，平均用時最短的隊伍為LNG，如果OMG、RNG勝利用時都和LNG相差150秒以內就加賽，但是LNG和RNG的勝利用時差為2分25秒，而LNG和OMG的勝利用時差為2分48秒，大於了150秒也就是2分30秒，因此三隊同分的狀況下就不用進行加賽了！

因此LNG直接把OMG給踢出局，在不打加賽的情況下RNG和LNG的勝負關係是RNG勝。因此根據規則最終晉級德杯8強淘汰賽的是RNG。
