改裝成戰鬥型汽車碰撞作戰，一款好玩停不下的手游

2019-12-19 709條評論 2,904次閱讀 388人點贊

【艱難道路】是我們推薦的第278款手游，感謝一直支持和關注我的小夥伴們。

號稱「毒廠」的111%工作室開發的一款手游，為什麼玩家要稱其為毒廠呢？因為111%開發的遊戲玩起來就停不下，經常就是再來一局到天亮的節奏。本次給小夥伴們介紹的這款手游，就是「毒廠」的一款改裝汽車類手游。

遊戲美術清新簡約，看起來是小製作，但是卻沒有粗製濫造的感覺。遊戲中沒有劇情，上手就是闖關戰鬥，直接了當。

操作也非常的簡單，玩家控制一輛汽車，汽車會自動前進，但是玩家可以通過點擊左右屏幕控制其轉向，可能前期玩家感覺操作會有一些彆扭，經常會撞牆，但是習慣之後就可以「大殺四方」了！

關卡地圖不大，開始的時候關卡會生成一些NPC汽車，玩家需要和這些NPC鬥智斗勇，通過撞擊它們的車身來減少它們的血量，最終消滅所有的敵人，進入到下一關。非常有意思的一點就是這些NPC相互碰撞也會掉血，這個時候玩家就可以慢慢把他們聚集到一起，突然幾個變向，他們就會「自相殘殺「。關卡中還有一些障礙物，毒水，滾石等等的機關，玩家不但要避開它們，還可以利用機關對付NPC，不過就看玩家的操作水平了。

消滅敵人之後，玩家會獲取經驗來升級，每次升級玩家都會隨機獲取一個技能，有增加血量、不怕毒水、撞擊增強等等，技能搭配的好玩家闖關會輕鬆很多。

當然NPC也不是一無是處，後面的關卡它們的數量不但會增多，還會增加一些特殊的技能，比如說有噴火的、帶鎚子的、帶機槍的，還有一些boss汽車，不但體型大，血量厚，還有一些特殊的技能，玩家一不小心就會掉血，最後等級清零，飲恨而終。

闖關之後玩家可以獲取一些資源，這些資源可以抽取裝備和汽車，汽車前面可以裝備裝甲，增加撞擊力和血量防禦等等，汽車後面可以裝備一些武器，類似鎚子、機槍等等。抽取的裝備如果有重複的，還能進行升級，算是一個小小的養成點。

如此整備好之後玩家的實力增強了不少，又可以重新開始闖關了。

這款遊戲就介紹到這裡了，整體上來看這款手游還是小製作，遊戲內的贏利點主要是通過看廣告的獎勵，當然有幾輛需要氪金的車輛，但是玩家不氪金也可以通過積攢資源抽取，並沒有太過逼氪。

但是遊戲玩法節奏較慢，給到玩家的目標感也較差，對比一下前一段時間我們介紹的同類型的【弓箭傳說】就可以看的出來。所以在關卡設計上【艱難道路】這款手游還是要再多多斟酌一下。遊戲中沒有中文，不過對於這類的小遊戲來說，並不算一個大問題，玩家稍微熟悉一下就知道玩法了。

總的來說這款手游還是非常不錯的，玩家可以隨時利用碎片時間玩一把，裝備車子肝一肝也全有了，感興趣的玩家可以去試試看。
