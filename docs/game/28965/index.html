《荒野亂斗》聖誕更新 海盜船上的新戰役開啟

2019-12-20 899條評論 3,776次閱讀 260人點贊

為了迎接聖誕節的到來，《荒野亂斗》進行了一系列海盜主題的更新。在卡爾海盜船長的帶領下，英雄們準備好揚帆起航踏上新的征途了。更新內容包含了很多方面，有兩個新英雄，五款新皮膚，新的對戰場景，還有新的遊戲模式。一起聖誕玩個嗨吧！

兩個新英雄

昆蟲學家貝亞是羅莎的助手，此次終於跟隨羅莎來到了荒野亂斗的戰場。喜歡穿成昆蟲模樣大概是為了能與昆蟲更加親近吧。

貝亞的普通攻擊是遠距離發射一隻機械黃蜂。命中敵人後，下一次普攻傷害會增加。

貝亞的大招能發射一群機械蜜蜂，這些蜜蜂能穿過障礙物使敵人減速並中毒。

麥克斯夢想是成為超級英雄，喜歡狂喝能量飲料配合渦輪增壓技術讓自己跑得飛快。

麥克斯移動速度超級快，普通攻擊能快速發射大量子彈。

大招會使身邊生成加速光環，增加光環範圍內自己和隊友的移動速度。對於行動緩慢的隊友來說真是一招神技。

五款新皮膚

看來貝亞是真的很熱愛昆蟲，瓢蟲貝亞能偽裝成一隻大號瓢蟲去接近真正的瓢蟲們嗎？

麥克斯只需要速度就行了，並不需要更多花里胡哨的裝飾。

波克憑藉自己的容貌順利成為海盜的一員，就從水手做起吧。

柯爾特憑藉多年從警的經驗，穩坐大副位置綽綽有餘。

船長卡爾甚至能開著小海盜船在場上滑行，拋錨手法極為精準。

新的對戰場景——海盜船

新的對戰場景是一艘滿載聖誕禮物的海盜船。這艘正在海上行駛的海盜船有時會隨著海浪的起伏而傾斜，此時會爬坡慢下坡快，增加了不少遊戲樂趣。

一個新模式

本次還更新了一個全新的限時遊戲模式——奪禮奇兵，兩隊互相搶禮物，將禮物放到己方的禮物圈裡，率先搶到指定數量禮物的一方獲勝。在拿著禮物跑的過程中還可以把它扔過障礙物。與亂斗足球模式不同的是，場上同時能有兩個禮物，一片混亂中與對方互相交換了禮物也說不定呢。

本次大更新一下出了兩位新英雄，大招攻擊能穿過障礙物的貝亞和跑得飛快的麥克斯都是很有特色的英雄，不知道你更青睞哪位呢？滿載聖誕禮物的海盜船新場景很有節日氣氛，海盜主題聖誕趴就等你來啦！
