DNF：100級版本又一次大轉移？次元風暴席捲全世界，泰波爾斯破碎

天天要聞
2 週 ago

摘要

韓服在上一周更新了一個新版本彩蛋，所有的NPC對話全部進行了變更，在12月5日的更新中，所有的城鎮背景都出現了更新，這一個更新對於經歷過第一次大轉移的玩家來說很熟悉，在第一次大轉移前城鎮背景都有了變化，所以這一次加上NPC的對話一起，很有可能再一次進行轉移。

韓服在上一周更新了一個新版本彩蛋，所有的NPC對話全部進行了變更，在12月5日的更新中，所有的城鎮背景都出現了更新，這一個更新對於經歷過第一次大轉移的玩家來說很熟悉，在第一次大轉移前城鎮背景都有了變化，所以這一次加上NPC的對話一起，很有可能再一次進行轉移。

韓服這一次背景改版主題是風暴，也就是暴風雨來臨前期出現的昏暗背景，其中主要內容是紅藍次元風暴的出現。

泰波爾斯破碎

在魔界集結地，我們看到背景是一個完整的泰波爾斯星球，這是我們當前版本所看到的背景。

韓服更新後，整個背景的顏色有了變化，天空變得更亮，地面變得暗沉，並且可以明顯的看到一個紅色的次元風暴，並且原來完整的泰波爾斯星球出現了破碎，碎片朝著風暴飛去，可能是為了更好的展示這一個背景，所以原來有的一顆大樹消失，並且滕蔓也消失不見，僅留下比較矮的植物。

因為韓服當前的劇情是魔界大戰，二姐打陰謀讓夏勒拿走了普雷伊希斯的力量，所以泰波爾斯失去了普雷的守護，有可能是因為夏勒無法控制這股力量加上二姐赫爾德的陰謀，所以才出現了這樣的變化，但是具體是什麼原因導致紅色次元風暴的產生，還需要等12月14日的韓服發布會才知道。

阿拉德又一次迎來災難

當年因為沒有錢守護好阿拉德，所以才導致阿拉德的毀滅，但是起源版本後阿拉德又一次在冒險家的努力下逐漸安穩了下來，但好景不長，這一次的次元風暴又一次席捲了阿拉德大陸。

我們現在站在西海岸港口，可以看到風和日麗，藍天白雲，但大家都知道一般風暴都是從海面襲來，所以韓服的改版中，這一個位置可以看到一個藍色的次元風暴。

韓服的港口天空背景變得昏暗，並且可以看到一個藍色的風暴已經席捲到港口的位置，整個阿拉德大陸的背景都變成非常出昏暗，如果藍色風暴登錄阿拉德大陸，那麼會發生什麼呢？

上一次的大轉移同樣是爆炸席捲了阿拉德，所以如果這一次的次元風暴登錄阿拉德大陸，那阿拉德又將迎來一次毀滅。

為什麼這一次的次元風暴會席捲阿拉德，而韓服新增的聖堂和五聖者後代，這些更新會不會有什麼相關聯，這些都需要等後續的更新才能了解新版本的劇情發展。不過話說回來，風暴來臨後，部分城鎮的背景變得挺好看。

比如斯頓雪域，原來的藍天白雲，現在的背景因為昏暗下來，整體看起來會非常的舒服，基本上本次更新都因為風暴的原因，地圖的背景都變得暗沉，不知道這些只是彩蛋還是之後的整體構造。

但是，之前大轉移還有這麼多勇士繼續守護，而現在大量的勇士脫坑，阿拉德大陸還能守護下去嗎？
