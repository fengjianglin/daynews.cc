夢幻西遊：五個女人的武神壇之戰——太和殿火鳳凰服戰隊

2019-12-11 147條評論 3,604次閱讀 828人點贊

快樂遊戲，幸福生活！大家好，這裡是連載型頭條號《我說夢幻西遊》！致力於做有深度的夢幻西遊頭條號！這是小編第209篇原創文章！

在之前的文章里，小編為大家講述了太和殿的夢幻西遊第一服戰女指揮——悠悠，夢幻西遊：溫婉妹子玩夢幻，暴力獅駝盛名傳！紀念太和殿暴力獅駝——悠悠，但是由於小編的疏忽將好幾次太和殿寫成了珍寶閣，因此在這裡跟一些粉絲真誠的道歉，非常低級的筆誤，希望大家原諒。

​在文章後面，小編提到了一個問題——為什麼悠悠一個容易情緒化的女孩子能作為一個服戰隊的主指揮呢？但是沒有朋友能夠回答我，今天小編就來揭秘這個問題：這是因為當時的太和殿是一個「陰盛陽衰，女權至上」的局面喲，下面請聽小編細細道來~

首先先明確，小編我講的是太和殿伺服器 從第11聯開始到28聯的這一段時間的故事，當時是以血戰天下、帥得坐牢、落葉仙子、朵朵、魔女小不點、魔女我怕誰、悠悠為主要成員的這一批太和殿成員。和這支隊伍一起參與過聯賽的總共有20個號23個人，名字有些小編記不清楚了，所以沒有一一列舉，所以不要抬杠哦~

大家從主要成員就可以看出其中的陰盛陽衰了。你應該知道，她們這幾位主要成員都是線下見過面的，而且都是朋友，因此不存在什麼人妖。其中悠悠名氣最大，她從官網公布玩家總經驗開始。當時悠悠54億的總經驗被驚為天人。大家應該都知道當初可沒有不像現在，這麼多任務可以刷。最初的經驗大多數還只能依靠燒雙和抓鬼等任務，54億經驗難度極高，悠悠也因此一炮而紅。而悠悠是女孩子這也毋庸置疑，而且悠悠也是從第10聯（也就是07年的3月份）開始就在太和殿征戰了，因此是當之無愧的元老，加上又是傲嬌可愛的女孩子，還是其他女性成員的大姐大，自然當上指揮就是順理成章啦~

​魔女小不點自然不必多說，應該是地府門派的首位武神壇冠軍獲得者（23聯，魔女小不點在太和殿以地府身份奪冠，是地府的驕傲），據說魔女小不點與魔女我怕誰這兩個ID都是同一個女孩子的，但是也有人說這是兩個人，但是關係非同一般，於是改的ID，具體的小編也不清楚，但是肯定至少有一個妹子啦~

​朵朵的龍宮和後面落葉仙子的普陀也是軟萌萌的妹子，這樣算起來，他們隊伍的主力裡面至少有4個女人，而其中只有最初的「血戰天下」和後面加入的真正指揮「帥得坐牢」（我呸i，怎麼可能比小編我還帥！臭不要臉）兩個爺們兒，而更可悲的是這兩個貨的女朋友也在他們的備選隊員當中，因此那是相當的沒地位呀！你以為小編在亂講？不不不，這可是「帥得坐牢」自己承認的。

​比賽隊員7個人，最少有4個女孩子，最多的時候六個，主要的男人只有「血戰天下」和「帥得坐牢」，你覺得在這樣「陰盛陽衰」的隊伍里很幸福嗎？不！一點話語權都沒有，一會兒要封印，一會兒要水清，根本控制不住局勢。而且悠悠還是這群女人的好閨蜜，她們只管自己玩的爽，也不管局勢好壞，優勢還是劣勢，只會問「為什麼劣勢」，標準的女性思維，雖然有點刁蠻，但是還挺可愛的~

​所以這兩個爺們不僅要當操作手，還要當觀察手，到最後這群女人折騰累了還要收拾爛攤子，所以呀，身邊女人多也不見得的是好事呀~三個女人一台戲，五個女人兩台戲，1:500隻鴨子的嘰嘰喳喳，真的不去拜師化生寺學習「嘰嘰歪歪」都可惜了~

18聯半決賽太和殿對陣齊雲樓

​但是，不得不說，這幾個女人在武神壇上還真耍出了一些成就，在一年半的時間裡，2次冠軍。4次亞軍，9次十六強，這些戰績在當初90支參加服戰的隊伍里都是數得著的。甚至到2019年多的現在，也沒見到哪一個服戰隊有這麼豪華的「女人戰隊」，記得在15年有一次採訪「血戰天下」，當時「我是特種兵之火鳳凰」大火，這貨就說「我覺得我們當年18聯的時候的陣容就是夢幻西遊中的火鳳凰戰隊」，當時不知道多少老太和殿的玩家有點淚目了~

​好了，這一期就到這裡了，算是為我的幾個粉絲加更的「太和殿」伺服器吧，喜歡你們能夠喜歡。喜歡那些老玩家或者故事都可以留言給小編，如果能找到資料或者聯繫到本人，小編會盡量寫出來分享給大家，也希望更多朋友能夠給我提供給素材，小編不勝感激~

歡迎關注《我說夢幻西遊》，連載型頭條號，致力於將夢幻西遊的趣聞講清楚，拒絕水文，吸引高品質讀者！記得往期內容更精彩哦！​
