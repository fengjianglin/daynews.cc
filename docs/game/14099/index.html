《寶可夢》八代好用的准神有哪些？新准神的對戰能力如何？

2019-12-11 2,147條評論 3,451次閱讀 2165人點贊

在八代還沒有完全出來的時候，有不少玩家推測八代的准神可能是鋁鋼龍，畢竟本身是龍系，外觀上用來當準神也多多少少有點接受，然後，等到八代正式出來時候，玩家才發現鋁鋼龍是沒有進化型跟退化型的，真正的准神是一隻叫做多龍巴魯托的寶可夢，外形上雖然有點詼諧但也博得了更多玩家的喜愛，總體來說是一隻可以很容易引起好感的准神，那麼，加入八代的准神可不少，在八代目前的環境下，又都有哪些准神可能在對戰中有著不錯的表現呢？就讓我們一起來看一下吧。

PS：排名不分前後；本文可能並不具有太大的嚴謹性，如有錯誤，敬請指正。

NO.1 班基拉斯

第一位就是萬年不變的老班，不得不說大佬還是大佬，在目前的八代對戰新環境下，老班依然是非常常見的寶可夢，整體用法來看的話和前代的老班其實差別不大，就是典型的開沙暴天氣的坦克，畢竟歷代老班就已經有著弱點多的劣勢，歷代都可以規避的問題那麼八代自然受到的影響也就不大了，而隨著像龍頭地鼠這樣的寶可夢也可以進入八代來看的話，老班不管是當一個開天氣的沙暴手還是自己做一個輸出坦克都是綽綽有餘的，在單打和雙打中都可以有所發揮，更可好好利用極巨化這個新設定。

NO.2 杖尾鱗甲龍

第二位比較受到關注的就是杖尾鱗甲龍了，關於杖尾鱗甲龍，有不少玩家推測說這一作因為沒有Z招式，七代能夠讓杖尾鱗甲龍火的杖尾鱗甲龍Z消失之後杖尾鱗甲龍可能會陷入不溫不火的局面，但是目前來看的話定論尚且太早，因為這一作中的杖尾鱗甲龍學會了一個新的招式，那就是魂舞烈音爆（好中二的名字），這個招式可以消耗少許一些自己的HP，提高自己的所有能力，總體來看有點像是翻版腹鼓，同時又有杖尾鱗甲龍Z的一點點影子，杖尾鱗甲龍應該是可以靠這招式玩出花樣的。

NO.3 多龍巴魯托

最後一位就是開頭以及標題都有所到的新准神，那就是多龍巴魯託了，因為是在試新階段，所以多龍巴魯托目前在對戰中的出現率非常高，整體來看的話是一隻高速的輸出手，幽靈加上龍這個屬性雖然不能算打擊面好，但也可堪一用，弱點相對來說難受一些，所以多龍巴魯托要注意保證先手機會，加上補盲招式還不錯的情況下，多龍巴魯托在做輸出手上還是非常不錯的，因為特攻學習面比較豐富，所以多龍巴魯托目前都是特攻輸出要多一些，這點有點類似七代的卡璞·鳴鳴，都是物攻高反而用特攻多。

好了，以上就是小二本次要介紹的三隻在八代可能會成為熱門的准神，當然其它的准神肯定也有屬於自己的表現，小二就不做一一的介紹啦。
