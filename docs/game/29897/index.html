專訪Peanut：有一波在LCK我是不會死的，但是我死了這就是LPL啊

2019-12-23 124條評論 4,889次閱讀 2293人點贊

德杯的小組賽已經全部打完，在賽後LGD打野選手小花生也是接受了專訪，來看看小花生都說了什麼。

Q：打野是特別需要跑圖的角色，所以需要和隊友們進行大量的溝通，那麼目前隊內的溝通是如何完成的呢？
Peanut：因為德瑪西亞杯很快就要開始了，要學習中文。但是現在是在比賽的時期，沒有很正規地學習中文，先從遊戲用語開始學習，也學習了簡單幾句，「可以」，「不可以」，「沒關係」這樣的句子。類似於「推完線之後走」的細節溝通暫時還做不了，一般都需要翻譯來進行協助。雖然溝通不是很好，但是現在的成績還是不錯的。

Q：到現在為止的磨合程度怎麼樣了？如果訓練賽過程中出現了不好的結果，你會怎樣和隊友以及教練團隊進行溝通呢？
Peanut：教練組，翻譯以及Kramer以及隊員們都給予了非常大的幫助，如果沒有去美國參加全明星的話，應該會有更多學習中文的時間。

Q：你覺得LPL與LCK的打法有什麼不同呢？
Peanut：現在是在尋找LPL和LCK打法中間點的階段，LPL的打法具有競爭性，LCK的打法比較慢，然而我們兩個都想要。雖然我們的前兩局比賽輸掉了，但是我們盡量會把遊戲拖到後期，訓練賽的時候我們一直在尋找兩種風格的中間點，那兩把其實我們拖的很好，剩下的比賽打的也很好。

Q：加入LPL後，和以前做對手時，有什麼不同的感受？
Peanut：打最後一盤的時候，有一波覺得自己不會死，因為LCK的話是不會這樣入侵的。結果那波我死了，但是死了之後，我笑了一下，覺得這就是LPL啊，我得適應了。

Q：VG官宣Kkoma加盟，後Kkoma之前在SKT有相處過很長的時間，這次在LPL也會有機會面對他，對此有沒有什麼想法。

Peanut：因為Kkoma教練的加入，VG會變得很強。但是我們要面對每隻隊伍，我們都要打贏他們，所以不會太在意。
Q：Peanut在來到中國後，有沒有感興趣想要面對的中國打野選手。
Peanut：雖然LPL的打野都很厲害，但是我想面對這幾個人，Karsa，國豪，Tian，Ning和Sofm。

大家對此怎麼看？
