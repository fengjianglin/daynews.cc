Hope臨危救主 EDG擊敗銀河戰艦滔搏 成功小組第一出線

天天要聞
2019-12-22

2019德瑪西亞杯冬季賽正在火熱進行中，由於B組中TES已經提前拿到三分獲得出線名額，所以杭州主場最後的收官之戰，成為了決定EDG去留的關鍵局。如果EDG能夠擊敗來勢洶洶的滔搏拿到一分，那麼他就會直接獲得晉級資格，但如果遺憾落敗，那麼2-2的戰績很可能讓他直接出局，所以這一局的比賽對EDG來說至關重要。好在ADC選手Hope及時站了出來，使用女槍打出高額的傷害，成功帶領隊伍贏下這場關鍵局。

從雙方的BP來看，EDG終於吸取了輸給V5的教訓，給新人上單xiaoxiang選出了強勢鱷魚讓他Carry，打野也選出了配合比較方便的雷克塞，雖然凱南的逃生能力很強，但EDG也不是沒有抓他的機會。不過EDG需要注意的是，TES上中野都六級的一波非常強，凱南配合卡牌一起抓下的話，EDG基本沒有什麼反抗之力，所以需要做好視野進行避戰，或者主動針對Knight的卡牌才行。

因為雙方陣容的先手能力比較強，所以這場比賽兩隊都打得很主動，不僅xinyi不斷來到中路想要針對Knight的卡牌，Karsa也在不斷利用中野聯動，以及輔助的遊走主動尋求擊殺xinyi的雷克塞。這導致了前期雙方在中路就開始了「搓麻將」之旅，但因為滔搏的支援比較及時，所以前期EDG一度陷入了極大的劣勢，讓不少EDG的粉絲揪心不已。

然而在經濟比較劣勢的情況下，EDG眾人在峽谷先鋒處居然打出了一波奇蹟團。在狹窄的河道處，雖然369凱南的進場時機掌握得非常好，直接限制住了EDG的三人，但Hope也不甘落後地用女槍放出了一波完美的大招，將369和滔搏眾人分割開來，從而讓Scout的蛇女在團戰中瘋狂輸出。在僅僅犧牲掉Meiko的情況下，EDG完成了一換三，隨後控下第一條峽谷先鋒，完成了局勢的逆轉。

後續滔搏雖然進行了一波反抗，但隊員們卻出現了一些小失誤，讓EDG不斷抓機會穩住了優勢。時間來到二十五分鐘，滔搏再次忍不住主動發起了攻擊，想要再來一場完美團戰，但萬萬沒想到EDG五人細節拉扯規避掉了滔搏的關鍵技能。沒有了大招銜接的滔搏，正面對拼起來根本擋不住發育良好的Hope，隨後EDG輕鬆控下大龍奠定勝局。

縱觀整場比賽，老將Hope的發揮最值得稱讚。在拿到女槍情況下，他團戰穩定輸出的優點被展現的淋漓盡致，每波團戰他都能找到最佳的輸出位置，不僅打斷了滔搏設想完美的大招輸出鏈，還讓切後排的凱南毫無作用，給滔搏造成了極大的困擾。觀看完EDG一整天的比賽後，小編認為EDG現在實力頗為不錯，每條線都能穩住對線，甚至還能小壓一點，只要再打磨一下細節，絕對是LPL的頂級隊伍。
