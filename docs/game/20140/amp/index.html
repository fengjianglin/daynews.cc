《FGO》官方發放超多福利，策劃是變良心了嗎？

天天要聞
2019-12-18

《FGO》國服官方給出的福利一向是「蚊子腿」，不過最近卻給玩家發放了60個聖晶石，對於突如其來的福利，估計不少玩家都處於納悶的狀態，策劃這是怎麼了？變良心了嗎？畢竟《FGO》的摳門是很出名的，像是在之前的活動中，沒有一點點的表態。

比如2018年《FGO》不參與B站九周年，在其它遊戲瘋狂給B站的玩家送福利的時候，《FGO》不為所動，當時處於莉莉絲和殺生院雙高人氣卡池期間，給玩家發放福利極大的減少了玩家拿到角色的概率，官方是不會錯過這個圈錢的機會，只能以日方監修為理由，拒絕發放福利。

還有一部分可能是日服不讓B站發福利，不過這個可能性很低，在雙王事件中，B站因為玩家帶節奏，無奈補償了30個聖晶石，在九周年的重要時刻，B站不會因為日服的強壓不發放福利，而這次發放福利確實有些莫名其妙。

官方給出的解釋很合理，也就是手游評選上榜和《FGO》第七章動畫的放送，為玩家送上聖晶石作為獎勵，不過今後會嚴格按照日服的更新頻率和進度補償聖晶石，也就是和日服同步。每次日服活動之後都補償玩家10個聖晶石，而國服就沒有這樣的活動，日積月累，國服的福利和日服發福利差距越來越大，不過這次能有20連抽的獎勵，可以在卡池試試水。

隨著聖誕活動和新章節主線的開啟，玩家可以選擇將所有的聖晶石攢起來，最保底的都有30抽到40抽左右，隨便哪個卡池保底都有四星保底，不過隨著新年的臨近，大招會在後面，也是可以放到新年和新年福利一起抽卡池，畢竟到時候拿稀有角色的幾率會大很多。

這次60個聖晶石的福利力度相比往常也是很足的，即使退游的玩家也可以回坑花光這20連抽，也許20連抽就出奇蹟也說不定，不過如何使用，就看玩家怎麼安排了，希望今後《FGO》的福利也像這次一樣多，那樣也不會導致玩家流失的那麼快了。
