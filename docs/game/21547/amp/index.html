怒焰三國殺符石系統被忽略的細節，所有技能每局真只能觸發一次？

天天要聞
2019-12-19

哈嘍各位好，今天依然是屬於怒焰三國殺的時間！前面幾期內容呢，落櫻分批給大家介紹了怒焰三國殺中的四大類符石，它們分別在各自所屬的領域有著優秀的發揮，是提升武將機能的重要道具。

然而，你以為這樣就完了嗎？當然沒有！因為關於怒焰三國殺的符石系統，還有一些被忽略的細節以及兩塊傳世符石沒有深入探究。現在，就請聽我慢慢道來。

傳世符石

天怒石

遊戲開始時，武將的初始怒氣+1

遊戲規則中已經說過，怒氣在武將受到傷害時獲得，也就是說開局時雙方都沒法使用強化牌，屬於從零開始慢慢發育。然而，如果武將鑲嵌了這個符石，就代表其在開局就能夠使用強化效果的卡牌，不管是回合內的輸出還是回合外的防禦都有了很好的加持。

天嗔石

武將的怒氣上限+1

同血量一樣，怒氣值也是有上限的，超過了上限再受到傷害時怒氣就沒法繼續增加了。一般情況下武將的怒氣上限是2點，所以想要積攢怒氣蓄爆是有點困難的事情，但鑲嵌了這個符石後，不光受到傷害後的隱形收益增加，也有了更大的空間蓄爆。

以上兩種符石呢，都會在”征戰天下”的隱藏關卡里掉落，想要得到的小夥伴們要加油闖關哦。

細心看過落櫻之前介紹的朋友們應該都有發現一個問題，那就是符石雖然強力，但每局都就只能觸發一次，不免會在心裡腹誹：這樣的符石真的有用嗎？不就是雞肋一樣！

對此，落櫻只能說，你們還是太年輕啊╮(╯▽╰)╭難道都沒有發現，各模式的獎勵里會有符石的碎片嗎？某種碎片達到一定數量就可以合成該符石，合成的符石會提高這種符石的星級，星級越高符石效果的單局觸發次數就越高！而每局能觸發二到三，甚至四五次的符石它還是雞肋嗎？

另外，武將只有單一的符石加持肯定是不夠的，所以如圖所示，隨著武將星級的提高，可以鑲嵌的符石數目也會隨之增加，最多能夠鑲嵌四個不同的功能符石，跟兩點初始怒氣和上限的加持。也就是說雖然怒焰三國殺中的武將技能都比較簡單，但通過不同符石的鑲嵌搭配，也可以出現千變萬化的效果。

那麼問題就來了，這麼多讓人眼花繚亂的符石，究竟該給武將鑲嵌哪一種呢？不要擔心，每個武將的詳情界面，都會給玩家推薦最適合這個武將鑲嵌的一系列符石，並且會註明這種符石在什麼地方獲得，讓玩家能夠有目標地培養一個武將。至於這個武將為什麼會推薦鑲嵌這些符石，落櫻將在之後的文章中為大家慢慢解析。

好了，關於怒焰三國殺符石系統的介紹目前就到這裡，如果落櫻有什麼遺漏歡迎各位在評論區補充，那麼我們就繼續明天再見吧┏(＾0＾)┛
