德雲色爆料IG阿水轉會去向，還談小龍堡三千萬的真正原因

2019-12-17 2,426條評論 4,154次閱讀 2814人點贊

每年LPL的轉會期都可以說是憋氣大賽，不到最後一刻沒有一支隊伍會透露陣容。但這一切都擋不住好奇的吃瓜群眾，每次都會有一些小道消息流出。如果說這次轉會期誰的關注度最高？那麼毫無疑問是JKL。

畢竟他是一個出道即巔峰的天才（S8登場就拿到了S賽的冠軍），而且還是國產AD，打法也比較百搭。給他資源就能C，即使不給也能苟，有了他隊伍的戰術就可以更加多樣化。這樣一個優秀的選手，受到多方關注也正常。

在轉會期的最後一天，德雲色的Kid就爆料了JKL的去向。他表示：就是你們猜測的那一支隊伍，娃娃爆料大概率是真的。根據之前的消息來看，大概率就是TES了。有了JKL的加入，TES下賽季真的可以說是銀河戰艦，各個位置都沒有明顯短板。

除此之外他也說了一下RNG的新打野小龍堡，因為S8以及S9的FMVP都是從YM出來的打野（寧王以及小天），所以也有了小龍堡是S10FMVP有力爭奪者的言論。

KID表示：現如今LPL的轉會費簡直可以媲美NBA了，即使是小龍堡這樣的新人也賣到了3千萬。彈幕不解地表示：為什麼他可以這麼貴？

KID也解釋說：之前YM都是便宜把選手賣掉了，包括小天以及明都是如此。以他們二個目前的實力，PDD可以說虧到姥姥家了。小龍堡的這一份價格，其實還包括了明的身價。只能說一個剛好擁有，另外一個剛好需要而已。不知道大家如何看待這次轉會，不妨留言討論。
