全境封鎖的真實故事背景 2代測評

天天要聞
2019-12-19

歡迎收看本期遊戲不止，我是喬伊。這次來給大家介紹下許多玩家期待已久的《全境封鎖2》

對於不熟悉全境封鎖系列故事的玩家，

我先簡單講一講全境封鎖的世界觀。

2001年6月22日，美國政府為了應對生化武器襲擊，在馬里蘭州安德魯斯空軍基地，進行了一次災難演習。定名為「暗冬行動」。這個演習是真的，並不是遊戲虛構的。

該演習模擬美國各政府機構在毫無防備的情況下，面對極為嚴重的生化武器襲擊。政府將如何應對，社會是否能承受。 最後測試和推演出來的結論，情況非常糟糕。政府無力對抗擴散的病毒和崩潰的社會秩序。

至於這場災難演習，美國政府有沒有什麼對策和預案，這個就不得而知了。反正只過3個月，就在紐約發生了震驚世界的911事件。

2007年，美國時任總統布希簽署了一項「51號政令」。這項政令就是指在國家面臨重大事件和威脅的一些應對措施。大部分是些中規中矩的內容，但有一些未公開的機密條款。

而湯姆克蘭西全境封鎖的靈感和世界觀，就來自於這個真實的「暗冬行動」和「51號政令」中未公開機密內容。

在遊戲的世界觀中，51號政令中那保密的內容，便虛構了一個叫國土戰略局的機密特工單位。

它們有獨立的運轉指揮和通訊系統。這些國土戰略局特工平時從事各行各業，有著各自的身份，但當發生重大，難以遏制的危機，政府其他部門陷入癱瘓狀態時，他們攜帶的手錶將會被激活，通知他們前往指定隱藏的裝備點，拿起武器來拯救國家和人民。

遊戲秉承著兩個真實的素材，虛構了在2015年的黑色星期五，在美國，黑色星期五可以簡單理解為雙11這種狂買東西的節日。在紐約曼哈頓地區，趁著黑五人們在商場搶購商品的熱潮，有人將一種名為「綠色病毒」的天花變種病毒，吸附在鈔票上進行傳播。

大家想必都聽聞天花病毒的惡名，曾經造成億萬人的死亡。如今真實的天花病毒已經在1979年被滅絕，只有一些可供研究的病毒樣本留在醫學實驗室。憑藉天花變種極其強悍的傳播力和殺傷性。導致紐約發生大量市民死亡。直接讓曼哈頓地區徹底崩潰陷落，美國政府不得已將該地區封鎖起來。並激活51號政令中的「全境封鎖」計劃。讓國土戰略局的特工前往被封鎖的曼哈頓調查真相，恢復基礎設施，拯救平民，遏制犯罪。

而由於紐約的病毒情況惡劣，第一批派遣的特工死傷慘重，並有部分特工放棄了自己的職責，甚至不滿上面的決策直接叛變失聯。玩家們便扮演的是第二批特工，再次進入在魚龍混雜犯罪頻發的曼哈頓執行任務。

這便是全境封鎖1的故事和世界觀。而這次剛發售的全境封鎖2的世界觀，則來自於紐約陷落7個月後，病毒仍然無法遏制，並已泄露出封鎖區蔓延到華盛頓特區。整個國家岌岌可危，空軍一號墜毀，總統換了幾任。而紐約的特工們，在暫時遏制住當地的惡劣形勢後，收到了華盛頓的求援信號。便啟程前往華盛頓進行支援。全境封鎖2的故事便由此展開。

2代的畫風和氣氛相比1代的陰冷肅殺，要緩和了不少。病毒的災難已經逐漸減退，但已被摧毀的社會秩序仍難以復原。在對抗綠色病毒的時候使用一種DC-62的化學消毒物質。一開始效果還好。但由於形式危急，沒有進行充分測試，結果DC-62暴露了一種致命的缺點，在某種條件下會變成劇毒物質，致使華盛頓不但被病毒破壞，還要被這種失控的消毒劑火上澆油。趁亂三個不同的犯罪組織在華盛頓畫地為王，搶佔各種資源。

而華盛頓的倖存的人們開始抱團取暖，重建家園。當你扮演的特工初入華盛頓的時候，第一步便是拯救白宮。和1代一樣，當白宮安定後就能成為你在華盛頓的行動基地。

花上幾天時間，做完主線和大部分支線任務就能升到30級滿級，清理掉這些犯罪分子的據點後。便有一個叫做「黑色獠牙」的准軍事化組織進入華盛頓，佔領之前你辛苦解放的區域。接下來又是繼續對抗「黑色獠牙」。

整個核心玩法和1代一樣，還是做主線任務，升級，滿級後刷裝備評分，提升世界階級。遊戲玩法本質上沒有什麼重大的改變。

在2代中，新加入了社區聚落的設定。這些聚落便是留在華盛頓的平民們賴以生存之地。玩家可以在不同地區的聚落中買賣裝備，接一些小任務，捐贈一些材料來幫助聚落的設施建設。你可以簡單理解為，白宮的行動基地為網遊中的主城，這些聚落則是遍布各地的小城市。

幫助他們的建設可以獲得圖紙配件經驗等獎勵。

這一代的有一個優點在於，地圖和1代差不多大小。但地圖細節變的更多更有趣了。有很多秘密的道具可供收集。ECHO等元素依然存在，地下世界變成了街道下方密布的下水道。有時候跑路從下水道走要安全許多，偶爾還能撿到能開出好東西的鑰匙。

對於裝備，作為全境封鎖的核心，就是刷裝備。2代的裝備在武器上，簡化了1代的武器配件。不用去收集各種繁瑣的武器配件。雖然簡化了，但缺少了一些武器的擴展度。而在裝備護具上，更講究套裝的配合和加成。除開511一些現實有的軍品，基本以虛構的軍品廠商進行搭配。甚至還虛構了中國的軍品企業的裝備。

在遊戲中另一重大改變，便是在30級滿級後，沒有大招技能，而引入了職業特長，添加了三種不同的專精技能樹和武器。

有用榴彈發射器的爆破專家，用爆炸弩箭的生存專家，以及使用大口徑狙擊槍的精準射手。這三種武器和技能樹的加入，使得滿級後遊戲有能更多的玩法和側重。

說實話這三個武器玩起來挺有意思的。就是子彈不好找。特工的裝備也保留了自動機槍，溜溜球等經典系列，還增加了更多有意思的東西。

1代中的特工獵人在2代中也有。不過這次以彩蛋的形式出現。在遊戲中某些地方，例如游泳池中進行開合跳，打開開關繞一圈聖誕樹，激活電腦打掉閃爍的燈泡等就能召喚出特工獵人，擊殺他們後能獲得裝飾面具和象牙鑰匙。一共有12個獵人和面具，收集這些面具還是非常有意思。具體召喚位置和方法網上有很多了，大家有興趣可以去找找看。

在遊戲中新添加了呼叫支援。當你在做任務，刷副本等時候被打死了，或者遇上難以擊敗的強敵。就可以呼叫支援。在附近玩的特工們就能收到信息，可以選擇加入支援該名玩家。這對於喜歡自己一個人玩的玩家，在遇上很難打的地圖的時候，就能讓別的玩家來幫你。

不過我還是推薦最好找點朋友一起來玩，單刷難度還是有點高，這一代的敵人明顯在武器裝備以及AI上有所提升。它們甚至有比國土戰略局還尖端的裝備。例如這個賊恐怖的無人機和有真實原型的大狗機器人。

至於遊戲中的衝突戰之類的，就是和其他遊戲一樣，兩隊玩家4V4，占點比人頭之類的。沒什麼特別新意。

暗區。暗區作為全境封鎖中最具魅力的一個區域，在於暗區的設定，暗區是遊戲中病毒破壞最嚴重的地區。因此裡面充滿各種病毒，但因為爆發迅速，導致軍隊的大量先進裝備和武器來不及撤離，遺留在裡面。讓各方勢力都想從中分一杯羹。

而暗區中的特工們，由於信號被屏蔽，因此可以啟用叛變條款，殺掉其他特工玩家來搶奪他們在暗區撿到的裝備。暗區撿到的好東西得通過特定撤離點，發射信號彈後等直升機來回收消毒，才能當做正常的裝備使用。因此每次收集裝備，在撤離的時候會吸引許多玩家和敵人打成一團，甚至剛才和你並肩作戰的戰友，在最後也可能叛變殺了你，獨吞所有裝備。

這種充滿爾虞我詐的玩法有著豐富的魅力。

而這一作的暗區相比1代，分割成了三個區域，同一區域最多同時存在12名玩家。進入暗區後裝備數值會進行標準化，不管你是萌新還是大佬，標準化能將裝備儘可能拉倒同一水平線。不存在裝備碾壓的情況。暗區門口還設置了自動機槍，可以防止叛變特工堵門的情況。

同時暗區新添加了一些潛伏叛變的機制。特工們可以黑入國土戰略局的網路，來盜竊本屬於國家的資產。並能通過多次三角定位，找到暗區中的賊窩。來進行黑市交易。

這種未殺人的輕微叛變特工標識不明顯，不容易被抓，適合獨狼行動。而殺人多的玩家則會明顯暴露自己的位置，成為全民公敵。他還能通過終端機提升通緝等級或者洗白，越高的通緝等級越危險，最後獲得的獎勵越多。

另外搶空投等吸引玩家聚集的機制也秉承著1代，有時候三個暗區中的某個區域限時會變成敵占暗區。敵占暗區可謂是最危險的區域，沒有數值標準化，友傷開啟，還會有更強力的敵人出現。和1代的暗區玩法差不多。不過越危險獎勵也會越豐厚。

總的來說這次暗區的玩法變的更加多樣化和有趣。

2代新添加了戰隊系統，更注重於社交和多人合作。戰隊系統目前功能還不是特別多，但能看到的是在未來的版本更新，更會注重戰隊的發展和一些戰隊特有獎勵出現。

最後來看看遊戲中的優缺點：

不得不說有一個東西育碧在2代，目前看來做的非常的好。最早期玩過全境封鎖1的玩家，可能都知道，當時育碧沒有做這種大型網路遊戲的經驗。導致遊戲裡面的數值設計非常的差。

甚至一個普通的暴徒可以打翻一隊特工的情況。再加上土豆伺服器時常全境掉線。導致大量玩家棄坑。隨著多個版本的迭代更新，1代的伺服器和數值平衡才有所好轉。但仍有時候玩起來還是不爽。

而2代就能明顯感受到育碧充分吸取了1代數值糟糕的經驗。把這一代的數值做的很好。

尤其是狙擊槍，終於有一槍爆頭的快感。霰彈槍的強大威力也能有良好體現。很多玩家不喜歡全境封鎖的原因，就是要花費大量子彈打死一個敵人。而2代則能感受到育碧在盡量避免這種挫敗感。

同時數值的良好體驗表現在多人遊戲。引入了標準化數值。例如高等級玩家帶你刷副本的時候，低等級玩家的數值會被平衡和補償。而不是呈現出如1代，高等級玩家碾壓低級敵人，低等級玩家打不動敵人，躲在後面無事可做的情況。同時敵人的數值也會改變，你裝備好等級高不代表你能浪。亂沖亂浪依然會被打死。

包括任務系統，許多玩家肯定遇到過，當等級高了，回到低等級區域清理一些早期沒做完的任務。由於沒有數值調整，導致清理那些小任務很簡單，得的經驗也很少，有時候因為強迫症，或者刷什麼東西不得不去做這種已經沒有挑戰性的任務。顯得無趣又繁瑣。

而這些大小任務都會隨著你等級進行數值調整，讓你不管多少級去清理這些支線任務，都有良好的挑戰性和收益。總的來說就是育碧花費了大量的精力設計了一套良好的數值體系，讓不同等級不同裝備的玩家，都能有一種良好的遊戲體驗。尤其是全境封鎖這種刷刷刷的，基本就是玩數值的遊戲，這一點上育碧目前做的非常的不錯。

而這個遊戲缺點，育碧的土豆伺服器有時候還是不怎麼穩定，BUG這種土特產我都已經見怪不怪了。經常還動不動跑著就摔一跤，整的挺真實的啊。

不過這種刷刷刷遊戲，後期就得等他不斷更新，添加新內容，不然刷到頭了就沒什麼樂趣了。拋開這些，全境封鎖2還是一個值得推薦和嘗試的遊戲，不過還有一點，不想肝的玩家可以單純做做主線體驗下劇情也不錯。如果想深入玩的玩家。你要是玩遊戲時間不多，還是少碰這種遊戲為妙。這種聖雄肝地的遊戲沒有大量在線時間積累裝備，是很難有好的遊戲體驗。

感謝收看本期的遊戲不止，喜歡的觀眾朋友們可以點擊關注，我是喬伊，我們下期再見。
