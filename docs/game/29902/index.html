霞都三件套了還不C？卡薩直言下路還是UZI香，這就是換中單的代價

2019-12-23 1,934條評論 3,185次閱讀 2165人點贊

Karsa在德瑪西亞的杯的賽後採訪時一時間還以為自己在RNG，不過事後Karsa也是馬上改口，不過經過最近的比賽來看，Karsa終於體驗了一把中路是大腿的感覺，而對於現在的TES來說中上野確實已經到了LPL的一級水平，畢竟現在TES中野堪稱是代練級的，但對於下路來說確實遠遠不如澡子哥香了，而Karsa在賽後採訪的時候也說道，現在我們中路是大腿，我要抱緊大腿，要知道之前Karsa在RNG的時候想抱也抱不著，不過TES的下路屬實讓人感到捉急。

ADC之所以被稱為是ADC的主要原因就是ADC後期可以提供恐怖的持續輸出，但英雄聯盟有一個前提就是活著才有輸出，而這次隨著TES出征的是Photic小應，而小應在LDL被稱為五殺之王，在LDL的ADC中也算是排得上號的，但面對了EDG戰隊的Hope的時候似乎顯得很是無力，尤其是後期的時候頻繁被秒，估計當時Karsa都在一直打問號，我們的ADC已經三件套了，可以出來C了，但是ADC你怎麼死了？其實之前的RNG不正是如此，別的路苟住，之後等UZI三件套。

不過值得一提的就是EDG現在是廠長上去BP，不過廠長的這幾局BP確實要比很多LPL教練好太多，還有就是很多玩家也在議論現在的Karsa轉會其實就是換了一種體驗，之前在RNG的時候UZI的下路就是爹就是大腿，所以後期指望UZI就可以了，沒有UZI的RNG在德瑪西亞杯的表現大家都可以看出來，而在TES呢？下路自然不是很強力，但是左手確實很香。

其實很多玩家也都明白無論什麼時候中單的存在遠遠比下路重要的多，畢竟中路可以輻射三路，左手在LPL國產中單中也確實是頂級的存在，前期Karsa也一直在圍繞著上中來打，小應確實沒有UZI那麼強力，但是細數整個LPL甚至各大賽區，能有幾位實力和UZI相近的ADC呢？這大概就是換中單的代價吧
