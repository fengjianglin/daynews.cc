銘文和出裝推薦：銘文就是10奪萃，10鷹眼，3無雙7貨源，第一件做電刀是為了提升清線跟對線手感，第二件無盡提高傷害，第三件黑切基本可以秒脆皮，第四件蒼穹可以提升持續爆發，第五件無盡提暴擊率跟攻擊力，全神裝鞋子換復活甲跟名刀這套裝備很便宜，10000開頭就可以神裝。但是我們要記得這套裝備便宜是便宜，大後期是打不動大肉的，所以我們要換裝備，即蒼穹換破曉或者黑切換破曉，而且小編我認為這套出裝會成為以後的主流出裝！至於末世的話，魯班不適合做末世，本身被動就是加強的末世，魯班吃100點攻擊力的整數倍，末世的攻擊力60點不夠用的，況且無暴擊 吸血過少，末世被動是當前生命值，魯班的被動是最大的生命值，所以不適合做末世，這個一定要分清的哦！

對線思路：技能一定是主一副二，魯班這個射手對線能力極強，一級打不過百里，其他都是穩壓，2級要拿一血，大家玩魯班的時候記住一定要卡位置，自己的每一個掃都要打到塔上，魯班是一個扛大逆跟逆風能力都很強的這麼一個射手，現在很多魯班的玩家有個致命問題，包括大部分射手玩家就是太想著去發育啦。

首先，發育並沒有問題，但是大家要記得這麼一個事情，你發育是在對等發育，對面也在發育，所以對線打爆壓制是主要的，對面爆了就相當於你發育了，魯班這個射手對線基本1分30秒左右大家卡住位置都可以推一塔的，這樣基本就一路盤活全隊了。
