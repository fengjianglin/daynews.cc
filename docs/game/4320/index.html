雪地地圖出生地一日游，你所不知道的中心公園

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

中心公園位於雪地地圖的東南角，也是雪地地圖中的出生點，想必大家肯定不陌生，每次進入遊戲都會在這裡轉悠兩圈，對於出生點附近的布局都十分了解，下面我們就來更加詳細的了解一下我們的出生點。

中心公園位於雪地地圖的東南角，也是雪地地圖中的出生點，想必大家肯定不陌生，每次進入遊戲都會在這裡轉悠兩圈，對於出生點附近的布局都十分了解，下面我們就來更加詳細的了解一下我們的出生點。

一、資源點分析及落地點選擇

我們平常在出生點活動的範圍就是在綠色的標記範圍內，很少有人會走出去好好的看看，其實中心公園是一個大型的資源點，房子很多，養活一個四人隊是沒有問題的，而且地形構造十分特殊，西北高東南低，也就是我們日常看到的地方是地勢最低的一部分區域。

一般玩家的落地點都選在了這個綠色標記範圍內或者附近，但是因為地勢低，被高處的人發現占不到便宜，所以我們的落點就選在了東北角的這個位置，從這個位置起手來進行後續的清點。

二、點內打法

我們從落地點沿著紅線在中心公園的外圍搜一圈，在紅線中心的位置有一個廣場，廣場下面是有一個通道的，這個通道裡面也會刷新一些物資，還有廣場上的小桌子上，搜索的時候不要放過這些資源，蒼蠅再小也是肉啊。

搜完這一圈以後我們本身的裝備肯定是已經成型的，其他的玩家也已經打的差不多了，不會剩太多人了，我們來到紅色五角星左上位置的山上先大致的觀察一下底下的情況，然後再下山來到紅色五角星標記的建築物樓頂，這個位置視野非常好，因為我們是從高處一路搜過來的，所以一段時間內背後還算是安全，我們可以安心的觀察點內我們沒有路過的地方，特別是綠色標記處和東邊的這些位置。

中心公園裡面也有很多可以通過三樓窗戶直接上屋頂的建築，我們可以根據敵人的位置靈活的轉移到這些點內進行近距離的埋伏。

三、後續打法

中心公園的位置算是比較偏的，也只有西北邊有一條公路，雖然中心公園的中心位置有一個車庫，但是並不一定就會有車，所以我們要確保在了解到車庫有沒有車的情況下根據信號接收區靈活的進行打法轉換。

在確保有車的情況下可以多搜一搜或者埋伏，車庫沒有車的情況下根據信號接收區的位置趁早離開去公路上找車跑圈才是王道。
