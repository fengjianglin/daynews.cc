黃金大獎賽S8A組海島圖結束，OMG暫居榜首，那支戰隊回歸了？

天天要聞
2019-12-16

絕地求生鬥魚黃金大獎賽第八季來了，在剛剛結束的A組小組賽海島地圖上，曾經的兩支戰隊都有好的表現，不知道這是不是代表著PCL賽區兩艘老戰艦再次容光煥發了？

現在就讓我們來看看今天A組的三場海島圖的成績吧：

OMG戰隊暫時排第一，而且從圖中我們可以看到OMG戰隊的得分非常的平穩（雖然第二場是因為吃雞才拿高分），可以說曾經的那支戰隊回來了。

最後在說下本次大獎賽A組賽事上OMG戰隊陣容是：小獅子、小海、T大爺、西法，這可是曾經拿下黃金大獎賽第六季冠軍的陣容。

排名第二的是TITAN戰隊，這名字很陌生，不過說起他的前身AMY戰隊大家應該是非常的熟悉了吧，畢竟這支戰隊在之前可是和IFTY戰隊搶橋頭的戰隊。

在說下SMG戰隊，雖然現階段SMG戰隊只排名第四，不過SMG戰隊在第二局禿頭山土路上的表現也是蠻亮眼的，對雷的使用也有了很大的長進。

堡壘保持SOLO賽上的狀態在對槍上終於不拉閘了，希望堡壘好好把握這次的狀態。

至於4AM戰隊的話，排名第九，不過相差第八的RNG戰隊只有1分，這表示這4AM戰隊沒有拉下太遠，只要後面3局不要拉閘，那麼就能穩定在晉級線作業，後面的賽事也不用那麼挑戰。

最後說說IFTY戰隊，今天上場的IFTY戰隊除了有在微博杯上上場的KK之外，其他的三名成員都非常的陌生，或許這次IFTY戰隊主要還是以練兵為主吧。（IFTY上場ID：Grayyyyyy、9nine02、GodXuan001、Kkai）

感覺這次賽事還是處於練兵狀態，很多戰隊能輪換的都輪換了。
