國行Switch首批用戶評價出爐：總體好評 無法與海外版聯機

2019-12-11 1,063條評論 2,161次閱讀 170人點贊

12月10日，國行任天堂Switch遊戲主機正式開賣，京東已有超過11萬人預約，相當火爆。

12月11日，國行任天堂Switch首批用戶評價也相繼出爐，目前國行Switch在京東擁有400+條評價，好評度96%。

絕大多數國行Switch用戶都給予了五星好評，評價內容包括「非常流暢沒有卡頓」、「畫面比想像中的清晰」、「續航加強版完美兼容老配件」、「幀數穩定」等等等等。

此外，國行Switch還有10條差評，除贈品、包裝、優惠券等糾紛之外，反饋的最主要的問題集中在無法與海外版Switch聯機上。

其中有用戶差評道：「如果你是要和朋友聯機遊戲的千萬千萬別買，國行只能和國行聯機朋友的全是日版，拆封了就不給退，一定一定要看清楚啊！」這位用戶甚至想直接在評論中出二手…

國行Switch包裝內包括主機、基座、Joy-Con左右腕帶、Joy-Con握把、電源適配器及HDMI線等，預裝了《新超級馬力歐兄弟U豪華版》體驗版。遊戲方面，目前上架的主要是《新超級馬力歐兄弟U豪華版》，未來幾周內還會推出《馬力歐卡丁車8豪華版》《超級馬力歐奧德賽》等，售價都是299元。
