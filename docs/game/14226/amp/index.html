Vstar慘遭Hope零封，kpl夢想破碎，vv難回AG，或就此退役

天天要聞
1 週 ago

KPL秋季賽季後賽已經進入尾聲了，除了總決賽之外其餘隊伍已經都被淘汰。現在的KPL發展越來越迅速，KPL的名額也就彌足珍貴，KPL席位賽正在激烈地進行當中，在敗者組決賽中，老將VV帶領的Vstar慘遭Hope零封，也失去了晉級KPL的資格，VV和他的隊友們的KPL夢想也破滅了。

這次的席位賽之前大部分觀眾都是看好Hope和KSSC的，對於VV的Vstar，大家都覺得他們晉級KPL有點難。但是Vstar的表現還是相當不錯的，在積分賽的時候排名第三，淘汰賽第一輪也贏下了大熱門Hope，一度讓人們看到了希望，雖然說不敵KSSC，但是也算是以逸待勞。不過在敗者組的比賽中Vstar面對連續征戰的Hope，再次碰面卻沒有一點還手之力。三局比賽幾乎每一局都是碾壓局，實力相差確實有點大。

而VV的個人發揮也不盡如人意。VV在Vstar隊中依舊擔任邊路位置，不過VV現在是戰邊坦邊都有玩，英雄池方面有了提升，但是作用相比於其他隊伍的邊路來說並沒有什麼優勢。尤其是Hope這場的劉邦和豬八戒發揮都很一般，而最後一局的呂布打得也有點慫，實在是有失水準。VV在預選賽中的整體表現已經成為了隊伍中相對薄弱的一環，不過也不能說VV拖累了隊友，畢竟是VV自己組建的隊伍，如果沒有VV的話，這群隊友可能也還沒有打職業的機會。

但是電子競技終究還是實力說話，VV現在的水平已經沒有KPL水準了，這支Vstar和KPL隊伍之間也有比較大的差距。雖然說VV帶領著隊友一直在努力訓練，爭取回到KPL，精神可嘉但是實力確實不允許了。有人會想VV回到AG超玩會，但是AG超玩會現在這個位置上是六點六，在整個KPL都是頂級，VV實在是擔當不起這個重任，AG是個有人情味的俱樂部，如果VV想回去俱樂部肯定能給他安排，但是可能是和蘭息一樣直播，想打比賽確實有點難。

這次的席位賽或許是KPL最後一次席位賽了，以後的KPL很有可能會固定名額，所以VV這次沒能抓住機會也就意味著隊伍將要解散了，而以VV現在的競技狀態也不會有隊伍買他。之後的VV很有可能會選擇退役，雖然曾經在KPL上創造過很多輝煌，但是終究敵不過時間的摧殘，VV也只能認命了吧。
