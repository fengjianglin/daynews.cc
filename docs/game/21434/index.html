《星際戰甲》攻略：洛基打法解析攻略 戰鬥配置推薦+復活方法解析

2019-12-20 2,262條評論 3,297次閱讀 706人點贊

《星際戰甲》中的邪神洛基Loki是玩家比較喜歡的角色，那麼洛基應該怎麼使用呢?戰鬥時如何達到自己想要的攻擊力呢，這裡為大家帶來的是《星際戰甲》洛基打法解析攻略，以及戰鬥配置推薦。

MOD搭配

　　LOKI：持久力，簡化

聖裝迴旋鏢：血壓點，四元素，快速收回，奮力一擲，旋風。

操作手法

　　離怪遠點，不然會把自己炸死，一下就死!

　　不用切換近戰武器(如果沒有奮力一擲需要切換出來，且不會自動爆炸。丟出去，在飛鏢回來之前左鍵引爆)，直接長按E蓄力丟出去，可以選擇怪的腳底下，也可以直接丟怪臉上，反正撞上就爆炸。

　　傷害的話，LOKI隱身可以1到2下把活動任務的狗秒掉，打I系防禦更是爽到不要不要的。

　　目前我弓的配置是：膛線、分裂彈頭、雷火、加射速的那卡等(其它不重要)

2.《星際戰甲》死亡了如何復活 復活方法解析

《星際戰甲》中一不小心死亡了怎麼辦，怎麼才能復活呢？

1、重傷倒地後你的隊友可以復活你

2、重傷後你可以使用副武器攻擊敵人

3、多名隊友同時復活你可以加快復活時間

如果你的生命值降低到零，你會負傷倒在地上。負傷後，你不能使用任何技能，也無法使用你的主武器和近戰武器。雖然你還能夠用你的副武器保護自已，但這將會並不是最好的選擇，由於敵方還能夠攻擊並加速你的失血率。有時，在救援抵達以前最好是平躺著。當你的隊友在時間用盡以前不可以趕來，你就沒法繼續作戰。

當你的隊友在作戰中倒下，它們的蓮花圖標會在你的迷你地圖上變成一個紅十字圖標，你能夠通過障礙物見到它們的名字展示為紅色，這也有利於你的救援！在負傷狀態下，你只能復生後才可以繼續作戰(你仍然能夠觀戰或和你的隊友聊天)當你的失血計時器達到零，隊友將沒法再復生你。幸運的是，任何的戰甲都具有在日常任務中讓自已復生的驚人工作能力；你能夠用這種工作能力重回作戰中協助你的隊友。

每種戰甲都會有每24小時自身復生4次的工作能力。如果復生次數用完，你能夠等到第2天自動修復或消費白金立即復生。當你選擇在死亡後捨棄日常任務，你的戰甲不會消耗復生次數，並且能夠用以其他更有價值的日常任務。了解何時應當使用復生，何時應當捨棄日常任務，是精通戰甲的一個重要小技巧！
