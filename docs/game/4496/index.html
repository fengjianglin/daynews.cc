我的世界：早期的村莊讓人壓抑？被稱作“鬼村”，連道路都沒有

小夥伴們應該都知道，在我們《Minecraft》我的世界裡，常見的建築就是我們的村莊了，但是我們都知道，村莊這個東西，其實在最早期的時候，是很多人都不喜歡的，畢竟對於我們的遊戲來說，方便才是最重要的，而早期的村莊，其實是非常不方便的，而且也不是非常好看的，這一點想必很多遊玩過來的小夥伴們都知道的，那麼具體是怎麼樣的，我們一起來看看吧。

在早期，我們的村莊其實並不是像現在的樣子的，我們能夠看到，在這個位置，出現了一些非常大的建築，而在建築的身上，使用的是圓石加苔石的一個建築體，而房頂的話，用的就是木板了，可以說這個村莊是非常的醜陋的，但是也是最早期的村莊了，其實一開始的時候，這只是想要給玩家提供的一個庇護所，但是由於很多人都覺得，這一些庇護所加起來，就如同一個小村莊一樣，所以隨後慢慢的就被人開始叫成村莊了。

村莊的生成地形有很多種，想必很多人最喜歡的就是在水周圍生成了，但是在早期，村莊在水周圍生成的話，是沒有道路的，這一點讓很多人都非常不理解，畢竟村莊按理來說應該是有道路的才是，這一個特性，其實也是集成了峽谷礦洞的特性，當時的版本中，這些地方，是不會自動生成道路的，所以沒有道路也是很正常的，但是這樣一來的話，就讓很多的玩家覺得，這個村莊並不方便，慢慢的就很少有人會去長期的在村莊生存或者在村莊待著了。

在村莊剛剛出現的時候，那個時候的村莊是沒有任何的NPC的，也就是說，我們的村莊雖然出現了，但是也只是表示一下，其實還是想要給玩家提供成一個庇護所的，所以在當時，這個村莊就是一個鬼村了，一個沒有人的村莊，就會讓人感覺到很恐怖，畢竟沒有人的話，這些房子是怎麼建設起來的，村莊的農田又是誰種的，可以說在當時，鬼村還是影響到了很多人遊玩的心情，隨後便出現了一些千奇百怪的生物，不過這些生物也是等於臨時替換村民的，隨後的正式版本中，村莊系統變完善了。

而現在的版本，則是以村莊與掠奪的主題出現在玩家的生活中來的，看來對於村莊這一塊，還是做了非常大的改變以及變動了，最後臨走前，不要忘記點一波關注啊！
