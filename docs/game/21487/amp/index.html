戀愛宮廷遊戲《深宮大燕》今日上線置身於大燕王朝享受後宮生活

天天要聞
2019-12-19

　由台灣淘米發行的最新戀愛宮廷遊戲《深宮大燕》今日在台港澳地區正式上線，官方表示，遊戲主打古風畫面、跌宕起伏的故事劇情以及多元的趣味玩法，讓玩家置身於大燕王朝，享受後宮生活。

【以下內容為廠商提供資料原文】

穿越大燕當​​女皇與他命中注定的相遇

　　你是一位美麗且智商高的職場女強人，一夕間穿越到大燕王朝成為了梁國的公主，又因聯姻必須嫁給燕王，你該如何在這波瀾詭譎的深宮之中過關斬將，與各具特色的男子們又是如何相遇的呢？就有待陛下們進到遊戲中各自探索了。

浪漫的書信劇情互動玩法

　　遊戲內浪漫的書信互動玩法，可以與男寵們飛鴿傳書互訴情長，幸運還會收到男寵們送的小禮物哦！主線劇情中也有精彩的互動式選項劇情，接下來的故事發展都由你來選擇，快來與他們談一場最醉心的後宮戀愛吧！

豐富的趣味後宮養成玩法

　　在後宮中陛下們可以建造自己的夢幻家園，招募的將星們也會在後宮中陪伴著你，可以派遣他們去種菜、伐木、煮飯、畜牧、釣魚等多種可愛的趣味玩法，如果將星累了還可以讓他們去泡溫泉或是到房間休息來恢復心情值哦！

將星養成掛機戰鬥玩法

　　招募可愛的將星幫助妳輕鬆登上女皇之路，輕鬆掛機戰鬥玩法讓陛下們不需花太多時間就可輕鬆升等，成為女皇的路上有可愛的將星們陪伴著妳，讓妳不孤單！

出訪遊歷挖掘大燕神秘奇遇

　　特別的出訪遊歷玩法，讓陛下們除了體驗宮中生活以外還可以探索大燕王朝的各個角落，探索的過程中還會觸發許多不同的有趣小事件，可以依據妳的想法做出不同的選擇，幸運的話還會遇見更多的藍顏知己並讓他們加入妳的後宮哦！

百變換裝系統成為最特別的女皇

　　多樣的古風精美時裝任陛下挑選，快來收集各種華美服飾吧！透過獨特的服裝彰顯妳獨特的魅力，妳就是這宮中最搶眼的大燕女皇！

開服命定之緣系列活動

　　誰是你的命定之緣呢….?搭配《深宮大燕》正式上線舉辦命定之緣限定活動，即日起- 12/27 (五) 23:59止，玩家只要到活動頁參與命定之緣的測驗，即可於活動結束後於遊戲中領取多項虛寶獎勵，如額外再參與命定之緣截圖活動，還可額外再獲得更多獎勵哦！
