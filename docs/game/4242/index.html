《符文工房4豪華版》上手體驗：為了下一代，含著淚也要補票

2019-12-08 0條評論 14次閱讀 0人點贊

摘要

《符文工房》是大名鼎鼎的《牧場物語》系列的姐妹篇，從2006年至今一共推出6部作品，橫跨NDS、Wii、PS3、3DS等平台，吸引不少熱愛RPG+種田玩法的玩家。時隔多年，MMV再度攜手Neverland，在Switch上復刻經典的《符文工房4》，讓一眾骨灰粉興奮不已。

由於近期大作層出不窮，游大師差點忘了數次跳票的《符文工房4：豪華版》，直到周末才有時間將遊戲拆封，結果一玩就停不下來。雖然本作移植的痕迹非常明顯，畫面、貼圖都散發著一股3DS時代的味道，奈何就是好玩！

就前面幾個小時的體驗來說，游大師甚至覺得比前一陣子復刻的《牧場物語：重聚礦石鎮》更有意思。相比起做一個平淡無奇的小鎮青年，扮演王子/公主明顯更帶感，人設和立繪也比較符合當代審美，不至於在萬花叢中選不出真愛。

此外，本作新加入的動畫和少量語音也比較有誠意，讓游大師對尚未解鎖的『新婚模式』（註：需在故事模式玩到結婚）充滿期待。根據日版的情報，新婚模式有各種福利滿滿的互動，並且還能夠在故事模式攢PT值，用來解鎖全部角色，光是想想就已經好了。

總的來說，《符文工房4：豪華版》在玩法上幾乎跟原作一致：戰鬥、種田、戀愛、結婚…喜歡這些元素的玩家一定可以玩得開心。提醒大家的是，儘管本作的解析度相較3DS有了長足的進步，但畫面仍與近幾年的JRPG有明顯差距，內容就是在冷飯基礎上添了少許香料。不是真愛黨的話，建議慎重入坑。

目前，與本作同時公開的真正續作——《符文工房5》尚未有進一步的消息，但願MMV和Neverland賺夠奶粉錢之後，能夠快馬加鞭把遊戲做出來，讓苦等7年的玩家，再一次重拾昔日的感動。
