2010-2019這十年下載量最高&最賺錢的移動遊戲是誰？

2019-12-17 563條評論 3,878次閱讀 237人點贊

數據分析公司 App Annie 近日公布了 2019 年度以及近十年間（2010-2019）全球手游下載量及收入排行榜。數據顯示，芬蘭遊戲巨頭 Supercell 出品的手游《部落衝突》成為了近十年來最賺錢的移動遊戲，截止到今年該作的營收已經超過約 64 億美元。而荷蘭開發商 Kiloo 和 SYBO Games 於 2012 年推出的《地鐵跑酷》則是下載量最大的遊戲。


從十年間的下載量榜單來看，《地鐵跑酷》在 2018 年中的時候累計下載量就突破了 25 億次，並且還成為了有史以來首個僅在 Andriod 平台下載量就達到 10 億次的移動遊戲。此後，該作的下載量還在不斷攀升。在 2019 年度十大移動遊戲下載量排行榜中，《地鐵跑酷》位列第三，僅次於今年的吃雞新貴《Free Fire》以及大熱遊戲《和平精英》。

從 2019 年度的下載榜單來看，排在第八位的《夢幻家園》在 11 月時已經達到了 2.23 億次的下載量。而第十位的《使命召喚手游》在發行首月下載量就達到了驚人的 1.48 億次，且它在剛剛結束不久的 TGA 上還摘得了「年度最佳移動遊戲」的桂冠。

在收入榜方面，卡牌收集類遊戲《Fate/Grand Order》依然穩坐 2019 年度收入榜首位，而在十年收入榜上，本作位列第五；Supercell 除了《部落衝突》之外，另一款遊戲《皇室戰爭》也進入了十年收入榜，其撈金能力著實很強；此外，國產遊戲《王者榮耀》和《夢幻西遊》也進入了十年收入榜，分別位列第六和第七。
