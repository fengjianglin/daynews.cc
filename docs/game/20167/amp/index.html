《滅神》新版本上線 開啟飛升成神新時代

天天要聞
2019-12-18

全新的境界，無盡的挑戰!每一名《滅神》玩家都有著一顆挑戰強者的心，但是在挑戰強者之前必須先要達到強者的境界。本次滅神開啟飛升成神版本，每位玩家都可以通過飛升進化成神。 本次開放的飛升系統，可大大提高玩家戰鬥力，相應地也更有開放神秘大陸「失落神界」供玩家探索挑戰!

飛升計劃啟動 全民成神新時代

飛升成神版本將從三大系統提升玩家戰力，它們分別是戰裝系統、飛升系統以及神鑄系統。飛升成神顧名思義重點必然是飛升二字!在玩家開啟成神系統後，將會啟動一個新的等級系統：飛升等級。玩家可以使用「飛升丹」來提升飛升等級，每個等級均可獲得一個神級稱謂!

當然，飛升可不僅僅只是讓玩家拿到一個稱謂那麼簡單，在成神後玩家的生命、攻擊、防禦以及天罰加成都會大量增加，並且加強的數值均受自己的飛升等級影響!不僅如此，飛升也還將開啟玩家的戰裝以及神鑄系統。

每一名玩家都夢想自己獲得一把「神器」，這一次戰裝系統將給予玩家披神甲揮神器的機會!在玩家開啟成神系統後，即可開始穿戴戰神裝備，擊殺飛升地圖的野外BOSS以及尋寶等途徑均有機會獲得戰神裝備，並且每一件戰神裝備都可以使玩家戰力暴增!

除此之外玩家還可以對飛升裝備的部位進行「神鑄」來強化裝備屬性，值得一提的是，如果各部位神鑄等級相呼應，還可觸發特殊機制，形成連鎖套裝再次額外提升屬性!

神秘大陸現身 失落神界挑戰武力極限

全新的地圖，全新的挑戰!本次開放的失落神界擁有兩塊飛升地圖，他們分別是無盡深海以及神艦沉船。正如前文所講，飛升武器的一大來源正是來自於本次的飛升地圖，而玩家也需要飛升等階達到需求後才能進入，本次更新的系統可謂是一環扣一環，玩家需先設法提升自己的飛升等級從而解鎖飛升地圖，隨後前往飛升地圖通過擊殺高級怪物入手戰神套裝，如果有一些用不到的戰神裝備，還可以通過分解來獲取神鑄石，並強化現有的戰神裝備屬性。

遺忘之塔解開封印 是男人就下500層!

作為這個遊戲的玩家，最不缺的就是一顆嚮往挑戰強者的心，這次開放的遺忘之塔必定能滿足玩家這方面的需求!同時遺忘之塔的階梯式難度還可以幫助玩家明確自己的實力究竟有多麼強大，畢竟只有足夠強的實力，才能保證玩家能夠進入到更深的塔層!本次開放的遺忘之塔一共有500層，每天玩家可以進行三次挑戰，最為良心的是玩家每次進入都將從上次挑戰的最高層數開始!這也將為玩家們節約大量的時間，每層首次通關時還會給予玩家首通獎勵!

本次滅神開啟飛升成神新版本，打造全民飛升新時代!戰神套裝將助你在戰鬥中無往不利，全新大陸給你成神道路更添無數挑戰，神秘的遺忘之塔當中無數秘寶等你闖關!
