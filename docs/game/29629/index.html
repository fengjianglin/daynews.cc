三國殺里那些被遺忘武將華麗轉身躋身T1級別，你還在禁用他嗎？

2019-12-25 895條評論 1,837次閱讀 307人點贊

看完標題可能會有很多朋友很好奇這個武將到底是誰,這個武將擁有者能夠每輪準備階段把自己的手牌補到4牌甚至五牌，還有著給予隊友兼顧著補牌和回血效果的能力，沒錯他就是祖茂！這個武將我相信是被很多人遺忘在角落裡面的一個武將，就連小編也是在偶然接觸後才發現這個武將改版之後強度原來這麼高，後來又試了幾局發現這個武將的強度完全不輸於那些熱門武將,話不多說上技能。

引兵

結束階段，你可以將任意張非基本牌置於你的武將牌上；當你受到【殺】或【決鬥】造成的傷害後，你移去一張「引兵」牌。

絕地

鎖定技，準備階段，你選擇一項：1.移去「引兵」牌，將手牌補至體力上限數；2.將「引兵」牌交給一名體力值不大於你的其他角色，其回復 1點體力，摸等量的牌。

祖茂作為一個四血二技能的武將來說技能介紹並不是那麼複雜，每輪結束階段可以把錦囊和裝備放在武將牌上，被殺和決鬥傷害就丟一張，準備回合開始可以給自己補牌也可以給隊友讓他拿牌回血並額外摸牌，能給隊友回血且最少給隊友2牌的武將，強度絕對不能被稱為「弱將」。

首先祖茂的兩個技能都是建立在引兵這個技能的基礎上的，所以如何才能避免殺和決鬥的的傷害是廣大殺友要去重點考慮的一個問題，個人建議決鬥幾乎可以忽略不計除非場上有著顏良文丑，主要還是要防止殺所造成的傷害，推薦開局刷一下加一馬和八卦陣這兩件裝備，這兩個裝備都能對降低你的嘲諷起到極強的效果。

早說說絕地這個技能，這個技能的強度是毋庸置疑的高，不論是把自己的手牌補滿還是給隊友回血補牌都是有著極高的收益，回合外沒人動你手牌消耗不大就把牌交給隊友(小提示，給隊友引兵牌之後隊友還會摸等同於引兵牌數量的牌）這麼強力的補牌居然還帶著回血的效果，如果不是因為有引兵的負麵條件，就憑藉這一個技能祖茂就能擁有這不低於史詩級武將的強度！

推薦思路：開局刷加一馬、八卦陣或是其他防具牌來降低自己的嘲諷度，每次引兵牌為了保險可放2張，這樣在無人打擾的情況下可以給隊友提供1血加4牌的補給。盡量降低自己的嘲諷度可根據形勢進行爆發。

推薦身份：忠臣 祖茂如果作為主反的話很容易遭到針對導致引兵牌被消耗一空，內奸則是缺乏必要的控場能力，但是作為忠臣的話反賊的目標放在主公身上能夠有效降低祖茂的存在感使得能夠更好的輔助隊友。

怎麼樣，這樣一個T1級別的輔助武將是不是一直被各位玩家所拋之腦後呢？趕緊去玩一局試試吧！本期內容到此結束，喜歡的話還請點個關注，你的關注是我更新的最大動力！
