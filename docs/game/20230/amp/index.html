雲頂之弈：9.24遊俠崛起：陣容成型即前三

天天要聞
2019-12-17

大家好，我是掌游寶作者：元芳。

給大家分享一個上分專用陣容，不管前期多爛，成型就是前三的強力陣容，地獄水晶遊俠。

遊俠在這次版本更新中加強了不是一星半點，又有新卡阿木木的加持，遊俠將是這個版本的前三的陣容。六遊俠的100% 2.5倍攻速加成絕對恐怖，搭配上寒冰和寶石的水晶羈絆，肉的同時傷害還高。

​

VN，千珏，維魯斯，EZ，老鼠，寒冰，寶石，阿木木

羈絆：

六遊俠三地獄火雙守護神雙水晶
裝備：
老鼠裝備：颶風+冰錘+羊刀or沉默or繳械or紅BUFF
寒冰裝備：羊刀+電刀+飲血
老鼠的裝備比較隨意，有鏟子給整個冰錘還能有個雙極地的羈絆，颶風可以充分發揮極地羈絆的效果，剩下一件有啥給啥吧，攻速最好，沒有給個特效裝也行，主C寒冰必須有電刀，電刀是寒冰的究極神裝，再有羊刀和飲血就是終極形態，搭配水晶羈絆很難被刺客切死。

六遊俠四水晶三地獄火雙守護神雙極地

這是這套陣容九人口的終極形態，老鼠帶冰錘，上個水晶拉克絲，C位和坦克的坦度增加一倍，要輸出有輸出，要坦度有坦度，要控制有控制，另外沒有水晶拉克絲有個極地拉克絲也可以上，搭配老鼠EZ的雙極地羈絆可以組成四極地，控制更上一層樓，這兩個陣容成一個必吃雞。

玩遊俠開局優先拿鏟子合冰錘，沒拿到或者沒希望就拿反曲弓，前期六人口之前可以用掠食者過渡，四人口三掠食者隨便帶個遊俠，記得掠食者里一定要有蠍子，是前期代替寶石很重要的卡，五人口掠食者帶倆遊俠，到六人口上布隆，蠍子，帶四個遊俠，盡量找到寒冰，沒有別硬D，布隆和EZ能組成雙極地，有寒冰的話寒冰和蠍子能組成雙水晶，算是完全體陣容的低配版，接下來就猛升人口，順風連勝打完鳥得到八人口，逆風小龍之前的選秀也得到八人口，順風可以留五十慢慢D，逆風要去世就留個十塊二十塊的得了，趕緊花錢D卡吧，如果D卡順利可以留錢上九人口了，九人口可以上兩個寶石或者寒冰，來極地拉克絲或者水晶拉克絲了都可以上。

阿木木寶石第一排抵擋傷害，EZ千珏第二排輸出，寒冰老鼠作為主C第三排輸出，VN維魯斯掛件靠邊站著吸引刺客，如果有特彆強力的刺客他倆擋不住了可以把阿木木放寒冰老鼠旁邊，至於具體那邊根據對面刺客的位置調整即可。
