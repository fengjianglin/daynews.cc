魔獸世界：別人謙讓是看不起我！負分拿完畢業武器就退會還有理？

2019-12-09 0條評論 15次閱讀 0人點贊

摘要

毛人毛事年年有，但是《魔獸世界：懷舊服》上線之後就特別多，好好的一個回味情懷的地方愣是被破壞的烏煙瘴氣。

毛人毛事年年有，但是《魔獸世界：懷舊服》上線之後就特別多，好好的一個回味情懷的地方愣是被破壞的烏煙瘴氣。有的時候想要找回以前玩遊戲的感覺，所以對於一起玩的玩家都是抱著良好的善意，可是就是有人喜歡利用這種善意，這不就有一個負分拿了畢業武器的法師轉頭就退會，這也就算了隔天竟然還假裝自己是新買的號。

這件事發生在「祈福」伺服器，在「祈福」有一個幫會組建了二團並且實行的是DKP加少量裝備稅的制度，第一件畢業武器CL，可能很多人看到這個制度都會覺得這個公會本身就有問題，DKP制度為什麼還要收裝備稅還要有CL一說，這也是那位法師拿完武器退會後用來污衊該公會的說辭。

首先我們先來說說這個公會的制度，裝備稅是普通散件50G，極品裝備200G，而這個稅的用途是給團隊買各種藥水以及給T奶DPS等各種的補貼，所有的資金流動都是公開的，該公會的玩家都能看到，並且等團隊磨合好穩定之後會取消裝備稅。據該公會其他成員所說每次下本所有的東西的確都是免費發放的，所以也能看出這一點公會並沒有瞎說。

那麼這個法師是怎麼負分拿到畢業武器的呢？那天公會組織活動，出了「統御法杖」然後團長說公會制度里有第一件畢業武器CL的規則，所以就問法師隊長要不要，而法師隊長本身也的確是缺少這件裝備的，但是想著先提升團隊於是選擇了謙讓。接著團長就說那就拍分吧，負分的不能參與，這位法師頻繁的表示自己想要，因為這個法師平時也不參與公會的MC小怪團，於是公會表示希望他可以以後多參加公會團這個法杖就給他了，該法師答應之後就以他的底分15分加200G拍了「統御法杖」。

等第二天公會管理髮現這位法師竟然不在線，於是私聊之後該法師表示我是新買的號？被無情拆穿之後表示自己是不滿公會GKP還要收G的制度。可是這個制度在群里已經反覆強調過了，如果自己不滿意可以不跟團，這個法師跟團也就意味著他自己是默認這種制度了，在他退會的時候還負分6分，當真是拿了武器就跑。

並且法師在拿到武器之後還找公會的人附了魔才退會，甚至還私聊公會的副T想要拉著一起走，一直在跟別人說這個公會制度黑，要分還要G，當後來別的公會的人問他為什麼退會時，他又說自己看不慣團長先問隊長，隊長不要才問他們要不要，心裡不爽就退了。

從整體來看這個公會並沒有什麼問題，就連可以被人詬病的制度也是早就公開的，如果自己不認同完全可以不上車，結果法師不但上車了還拿了畢業武器，拿了武器還找公會的人附魔之後才退會。別人想著團隊謙讓，到了法師這裡就是看不起他最後問他，也許魔獸就是被這種人破壞的吧！對於法師的行為你怎麼看？想要了解更多的遊戲資訊那就關注小編吧！
