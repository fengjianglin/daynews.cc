穿越火線到底怎麼了！爆破和生化模式開掛也就算了，現在就連

天天要聞
2019-12-12

穿越火線在以前是我最喜歡玩的遊戲，我相信很多人和我一樣。團隊競技的無休止的戰鬥還有擊殺敵方人物時的音效，往往令玩家們血脈噴張。但是自從CF氪金現象日益嚴重後，大部分玩家慢慢放棄了穿越火線，轉到了更為公正的英雄聯盟。也是從那時候起，3億滑鼠的槍戰夢想CF開始走下坡路。

現在，我又開始玩穿越火線了。官網上不斷的各種送神器活動，吸引了我的回歸。我又開始玩我熟悉的團隊競技模式。

然而近期卻發生了令我意外的事，團隊競技有人開掛。

打團隊競技這種娛樂模式也要開掛？我是真的服了。這些外掛玩家不敢開自瞄和鎖頭，單單只開一個透視，拿一把狙架住過點路線，一槍一個準。

看看我的舉報反饋。我在同一天舉報了四名玩家全部被封。圖二的外掛玩家甚至被封了10年。

現在的穿越火線不僅團隊模式有外掛，就連刀戰都有掛。我也是再最近發現的。

有增加刀戰武器攻擊範圍的，還有加快武器攻擊速度的。前不久就碰見一位外掛玩家。那天玩的是回合制的刀鋒寨，回合剛剛開始，無敵時間剛過。就看到提示說我們贏了這一回合。我還沒反應過來，看了看左上角的擊殺提示，對面還沒出寨門就被我方小刀全部團滅。不得不說這外掛真厲害。

最後還是標題的那句話，穿越火線到底怎麼了。以前爆破和生化模式開掛也就算了，現在就連刀戰和團隊競技都有人開掛。

到底是玩家的虛榮心作祟，還是被其他玩家虐的太慘了。歡迎大家留言評論。
