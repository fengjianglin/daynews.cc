爐石傳說本周亂斗又上當了！本以為合作結果被毒打？牧師花里胡哨

天天要聞
2019-12-19

終於又過了一周爐石傳說亂斗模式上新，本周亂斗的主題是「禮尚往來」，還是需要自己組卡30張！本以為又像上周一樣皮一下可能是合作模式，但進遊戲才發現是雙方互毆，直接被對方一頓毒打才發現原來這周又變成了對局，設計師真有你的！

在開局之後系統會給雙方玩家各一個冬幕節禮物，冬幕節禮物0/4亡語是使當前玩家獲得一個失竊的禮物，注意這裡的意思是只要在你的回合不管是打我們自己的禮物還是對方的禮物都能觸發亡語效果，而且只有在我們回合開始之前才會刷新，我們可以在我們的回合中打掉對方和自己的禮物，讓對方沒有禮物可打！

果然牧師整活是最厲害的，1費硬幣上哀黃然後兩個盾直接把哀黃變成了2/7，我的鬼鬼這怎麼玩。因為我們打掉箱子之後會變成一張1費的禮物法術，而哀黃的效果是使法術減1費，因此在這段時間內對方使用禮物法術全是0費，打禮物也會出現0費隨從，白嫖石錘了！

一頓操作之後他直接把哀黃養成了22/22的怪物，本來想一手130翻盤但最後還是事與願違，只能說牧師在本周亂斗中太厲害了！只要你前期沒能給到他足夠的壓力就只有被毆打的份，我太難了！

接下來有遇到一個牧師選手，這傢伙更無情直接把禮物變成4/4從開局就開始白吃我的禮物。由於我們沒有任何辦法只能讓他刷了很多禮物，本以為這把又要跪了。

不過還好這個牧師有點太得意忘形，直接下了868復活了一波禮物，我也不跟你多BB直接推掉對方送我的禮物搶佔優勢。

最後配合薩滿迦拉克隆的技能無限解他的場合刷箱子，對方也表示打不過就溜了。所以在任何模式中花里胡哨都不一樣能拿到最後的勝利，其實穩紮穩打也可以贏得！

最後給大家推薦一套該模式下十分厲害的術士卡組，想贏一把拿卡包走人的小夥伴們可以試試！
