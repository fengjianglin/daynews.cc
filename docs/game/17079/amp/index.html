LOL專扁近戰輔助的征服者塔里克，暈眩中的人是沒辦法說不的

天天要聞
2019-12-14

今天來向大家介紹9.23版本中，同樣受惠於新天賦系統勝率大增，但是出場率同樣是倒數的冷門強勢英雄，瓦洛蘭之盾－塔里克。

塔里克在9.23版本的統計數據中，絕大多數情況下都是擔任輔助的位置，無論是在OP.GG上的52.44%勝率還是Lolalytics上的52.88%勝率在輔助英雄中都名列前茅。如果加上其它位置的使用紀錄，塔里克的勝率僅次於上周我們介紹過， LOL勝率跟鬼一樣的宇宙大燒賣！現版本默默OP的龍王索爾！

從輔助位的數據來看，塔里克屬於前期表現中庸，但是中、後期擁有一定的裝備後，可以有效影響團戰勝負的輔助英雄。主要原因是塔里克的大招宇宙之輝（R）的冷卻時間較長，在前期小團戰碰撞較多的情況下，常常出現正在冷卻中的情況。

塔里克的優點是，在坦型開戰輔助當紅的現在，它能有效的壓制所有熱門的坦型英雄，像是：錘石、泰坦、布里茨、蕾歐娜等。堅毅壁壘（W）帶給隊友的額外物防也能讓下路雙人組在面對傳統射手時有更好的換血能力。但除了冷卻時間的問題外，造成塔里克不熱門的原因還有它有明顯的相剋問題，在面對迦娜、索拉卡甚至是巴德、賽娜這類AP遠程消耗型輔助，對線期會顯得很被動。

召喚師技能

閃現是必帶的技能想必不用多說；如果是對角色還不夠熟悉的玩家可以選擇虛弱；引燃比較適合對塔里克掌握程度較高，想在前期對線時製造擊殺的玩家使用。

天賦點法

天賦點法這邊會有兩種，同樣依照玩家的熟練度不同做推薦。

類型一：適合一般玩家

主天賦：堅決

守護者+生命源泉+調節/復甦之風/蛹化+復甦

雖然大部分人在堅決的第三層較喜歡選用蛹化，但是勝率最高的是調節，再都是復甦之風

副天賦1：啟迪

餅乾配送＋星界洞悉

副天賦2：精密（個人比較喜歡這種）

氣定神閑＋傳說：韌性

自選天賦：

第一層看心情任選影響不大，除非對線的Carry不是 AD，不然第二層都會選擇物防，第三層就是物防或生命任選

類型二：適合高端玩家

主天賦：精密

征服者+氣定神閑+傳說：歡欣/傳說：韌性+堅毅不倒/致命一擊

副天賦：啟迪

餅乾配送＋時間扭曲補藥

自選天賦：

第一層適應之力/攻速；第二、三層只要不是對方有大量AP英雄，大多都會選擇物防*2

技能點法

宇宙之輝（R）＞ 炫光（E）＞ 堅毅壁壘（W）＞ 星光之觸（Q）

不少韓服高端的技能升級順序會選擇：EWQQER EEEWR WWW

裝備選用

出門裝備一般來說推薦鋼鐵護肩＋生命藥水*2的配置

鞋子方面，塔里克需要一直貼緊敵人刷新被動，所以不推薦疾行之靴，高端玩家會為了粘著敵人選擇輕靈之靴。如果對手控制技能多選水銀之靴，如果對手的物攻傷害比重較高也可以選擇忍者足具。

輔助方面，塔里克的打法可以根據對手狀態調整成前排追擊或是保牌的打法，主要根據團隊的需求選擇適合的裝備，因此首推萬搭的基克的聚合，後續可以選擇一件增加回復能力的裝備，像救贖或是熾熱香爐。第三件可以考慮比較功能性的裝備，需要跑速可以選擇舒瑞婭的狂想曲或正義榮耀，怕對有被集火秒殺可以選鋼鐵烈陽之匣。

除了上面提到的裝備，像是冰霜之心、騎士之誓、蘭頓之兆、冰霜之拳等帶有物防的裝備都是可以優先考慮的目標。

玩法建議

炫光（E）的技能距離高達575，因此在面對手短的英雄時，往往能夠給予較大的壓力。堅毅壁壘（W）被動有物防加成，加上正氣凌人（被動）也有物防給予的傷害加成，在面對物攻角色時，塔里克擁有天然的優勢。

一般以堅決天賦為主的打法就是傳統輔助，只要好好盯著著隊友的血線，讓你的隊友不要在你還有技能時死亡即可。如果是採用征服者天賦的玩家，請務必在對手選擇出近戰輔助之後再鎖定塔里克。因為選擇征服者的理由，就是要依靠塔里克前期近戰無解的能力瘋狂粘著對手打，炫光（E）＋正氣凌人（被動）AA＋堅毅壁壘（W）/ 星光之觸（Q）＋正氣凌人（被動）AA可以在3秒內（甚至更短）輕鬆打出征服者的效果，幫助塔里克在影響力較小的前期建立優勢。為增加炫光（E）的命中率，搭配的英雄最好有一定的位移能力。而宇宙之輝（R）不僅僅是保護技能，如果搭配有突進能力的鬥士或刺客，一樣可以打出開戰的效果，這是一般玩家常常忽略的地方。

星光之觸（Q）因為耗藍高，前期光是利用正氣凌人（被動）不斷刷新層數，就能帶來足夠的回復量。
