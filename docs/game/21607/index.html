WE2020春季賽大名單出爐，並無阿水，阿水WE連接失敗

2019-12-19 641條評論 3,905次閱讀 600人點贊

WE2020春季賽大名單出爐，根據WE官方微博發布的消息，WE英雄聯盟分部2020 LPL春季賽大名單正式發布，來看下具體的介紹。

WE2020春季賽大名單：

經理：inkwan；

主教練：Domo；

隊員：Morgan、Poss、beishang、Plex、Yimeng、Teacherma、Jiumeng、Missing。

官方微博寫：「這個沸沸揚揚的轉會期，我們有很多不舍，但他們有了更好的去處，我們也還有機會在賽場上相逢，這或許已是最好的安排。他們的前綴變了，但我們希望他們達成夢想的希冀未變，我們共同奮戰數載累積下來的情誼未變。」

不過就在12月17號，也就是昨天，還是有很多玩家認為WE和阿水會進行連接，阿水加入WE被熱議。

但是現在WE戰隊的春季賽大名單當中並無阿水的名字，阿水和WE也是無法聯繫在一起了。

所以現在阿水去哪兒依然是個未解之謎。

阿水到底加盟了哪支新的俱樂部戰隊，亦或是繼續留在iG？只有等待官方宣布了。
