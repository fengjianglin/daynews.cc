國服第一小馬被路人叫囂，50一把solo敢嗎？打完後小馬懵了

天天要聞
2019-12-22

英雄聯盟這款遊戲至誕生至今憑藉其豐富多樣的遊戲玩法和變幻多端的對局形式而收穫了近億玩家的喜愛，每天都會產生成千上萬把遊戲對局。然而玩家們不是在每一場遊戲對局裡都能發揮出自己的最好水平，時常會發生自己狀態不好而被隊友謾罵的事情。為了能夠證明自己真正的實力會有部分玩家會邀請其他人進行SOLO對局以便證明自己實力。最近小馬便遇到了這樣的事情，兩人進行SOLO並以50元作為籌碼進行對決，沒想到這個玩家輸了比賽後竟然開溜了！

小馬同學作為PDD的小弟之一實力相當雄厚，曾經靠著一手上單人馬血虐聯盟高端局而受到玩家們的喜愛。奈何小馬距離職業選手有一段距離只好屈尊成為一名主播直播自己的高端排位對局並時常在遊戲中展現出自己出色的實力。在最近的直播中，小馬掏出自己新練的皎月上單進行教學，卻收到了一名心態爆炸的魚人玩家的SOLO邀請並以50元作為籌碼。小馬欣然接受邀請，卻不料這位玩家輸掉比賽後竟然跑了！小馬：做個人好嗎？

小馬在直播中，選擇了在版本更新中技能被修改的皎月英雄走上單進行排位。畢竟曾經是國服第一上單人馬實力不容小覷，遊戲開始便全方位碾壓上單厄洛伊並多次進行對位擊殺。遊戲正在進行，敵方玩家突然公屏說道：「我們這個小魚人自己炸了還跑來下路帶節奏，幫忙舉報一下謝謝！」小馬看到後微微一笑說道：「這把穩了」。敵方的小魚人卻不服公屏打字：「說我菜？誰敢和我SOLO一把？一把50元！」小馬見狀說：「50元呢！我參加！」

排位結束後小馬創建了自定義房間並邀請了小魚人玩家，小馬依舊拿出皎月敵方則選擇了劫進行對局。對局開始後兩個人互相不知實力深淺未敢輕舉妄動，升到五級之後小馬說道：「這人玩的一般感覺可以殺。」說罷小馬便率先發難釋放QE連招先手攻擊劫，兩人交手之後劫陷入劣勢。小馬心中對這個玩家的實力有了了解之後開始頻頻發難，劫在一次次攻擊後多次僥倖逃脫但是在最後一波交手後絲血不回家被小馬釋放引燃帶走。

SOLO結束後小馬退出房間想要回50元卻想起來還沒有加這個玩家的好友，小馬查找對局記錄找到了ID後添加好友但是過了很久依然沒能添加成功。小馬一次次刷新好友列表但是那個玩家一直沒有通過好友申請，小馬見狀無奈的說道：「這人肯定是跑了，剛才不還吹牛隨便來一個人就能打得過么，輸了比賽50元都不給人都沒了！求求了做個人行么！」看到這一幕觀眾也笑了：「是你輸了你肯定也跑了！」玩家叫囂SOLO一把50元，小馬獲勝後，人跑了！
