王者榮耀KPL全年數據，僅六個英雄從未登場，最短比賽不到9分鐘結束

2019-12-19 1,899條評論 2,880次閱讀 1489人點贊

王者榮耀的2019年KPL聯賽已經全部結束，有一些比賽趣味數據也新鮮出爐了，現在趕緊來看看都是哪些有趣的數據吧！

這一年的比賽中，由於引入了全局BP模式，因此眾多英雄紛紛登場，賽場百花齊放。在即使是這種情況下，依然有六個英雄從未登場，分別為劉禪、阿軻、女媧、韓信、劉備以及孫悟空，大部分英雄都是既不能在賽場出現也在路人局也罕有人選擇，可以考慮重做或者加強了，否則根本沒有存在的意義。畢竟妲己都能在賽場出現，官方應該思考一下這幾個英雄的問題。

在2019整個全年比賽當中，時長最長的比賽是AG超玩會和Hero久競接近50分鐘的比賽，相信大家對其都印象深刻。其實到了最後，雙方的隊員都已經無關緊要了，畢竟大後期的超級兵傷害實在太高，一錘脆皮半條血，當然擊殺總和最多的也是這場比賽，絕對會被所有人記住。

全年最短的比賽同樣也是由AG超玩會貢獻。那是對陣YTG的第二場比賽，通過前期優秀的入侵以及操作快速建立起巨大的人頭和經濟優勢，最後在不到九分鐘的時候就贏下比賽。能快速解決戰鬥，也能達到天長地久，不愧是冠軍隊伍。
