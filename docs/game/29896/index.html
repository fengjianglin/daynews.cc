為什麼《DOTA2》和《英雄聯盟》的控制時間區別這麼大？

2019-12-23 1,018條評論 1,987次閱讀 726人點贊

hello，大家好！這裡是鴨啤游訊室

MOBA類型遊戲這十年來風頭無兩，在整個遊戲圈發展史中佔據不容忽視的一席之地,並散發著它獨特的魅力。

鴨啤作為lol玩家，舍友玩的是dota2。我看他的各種控制長的有好幾秒。還有類似於黑皇杖撒旦之類的。。恐怖裝備。所以為什麼lol的控制時間和dota控制時間差別這麼大？


網友：Mino

因為LOL的設計初衷，就會演化成蒙著MOBA皮的格鬥遊戲。而格鬥遊戲就是不能有太強的控

DOTA的最大問題是遊戲機制複雜、上手難，LOL就是照著這個點優化的：

技能主要是各種形式的小位移、小控制和花式衝擊波，範圍不超過一屏幕（甚至有鎖定視角的設定，這在DOTA是不可想像的）

新手再菜也能感知「我死了是因為技能沒躲過去」「大招沒中不然反殺了」

類似於，拳皇哪怕你只會平A、打世界第一被連到死，最起碼也知道自己是怎麼死的

絕對沒有DOTA最常見的新手中血魔大走到死

但隨著玩家上手以後，因為機制發揮空間太小，博弈集中在拼操作，也就是格鬥遊戲了

（畢竟，不是人人都能像任地獄一樣，做到易上手難精通）

網友：Hapen

據說LOL設計師理念是:讓被抓玩家有逃命機會，不會一動不動地被打死。所以大部分技能需要預判。

DotA大多是指向性技能，天賦和神杖的存在使得部分單體控制技能只要點區域就能釋放變為群控。(比如萊恩羊和死亡一指~)

LOL面板高，速度快，技能CD短，注重微操，被控1秒就半血了，要是控制久基本就是死。DotA面板低，攻速低，依賴普攻輸出，技能大多是功能型，注重拉扯，團戰打得久。

LOL設計師從職業分路和時間節點上給玩家安排了最優方案，比賽也主要看操作和團隊配合，更多的是資源搶奪雪球滾大一波決勝負。DotA不規定陣容打法，玩家自我摸索，靠控制技能的無縫銜接帶走對面大哥實現翻盤逆轉成為美談，更多的是多次博弈拉開差距。
