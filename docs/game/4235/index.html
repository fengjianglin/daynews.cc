賽爾號年費種族值變化及與同期精靈的比較

賽爾號年費一直是賽爾號必氪的項目之一，因為年費的高強度與其一塊的福利，成了每個玩家的必氪首選，而年費的種族值也在成為每年精靈種族值的新標杆

聖光靈王體力139，速度120，攻擊130，防禦101，特攻70，特防100；2015年是賽爾號對戰體系轉變的一年，這一年的年費聖光靈王便是這一轉變的代表，他的種族值很好地體現了賽爾號時代的變化。

體力139，這在當時是一個很高的數字了，速度120，也算是在當時也是高速，當時速度種族值普遍還是以托魯克為標準，過了115就算高速，本攻130，這一個數字很長時間都引領了攻擊的潮流，現在還有很多精靈都是以這個數字為標準的，防禦特防也還行，這個數字到了16年年末也算及格，非本攻70很標準。

聖光靈王的種族值在15年上半年還算高，但是不是頂尖，14年的神獸超進化，15年的聖光奧爾德，可以說此時賽爾號種族值基本上保留在660這一水平線

聖光靈皇體力145，本攻135，非本攻70，雙防110，速度125，總和695，賽爾號690精靈種族值在15年下半年開始普及開來，聖光靈王也進行了相應的提升，各項種族值都進行了優化，在PVP對戰體系中695的種族值除了特訓譜尼也沒有哪個能高得過了，至於聖光靈神，是17年的精靈，那時已經不是年費了，我們暫且不提

16年重生之翼種族值體力體力145，本攻130，非本攻80雙防113，速度119，總和700，但是非本攻80，凈含量還沒有聖光靈皇高，16年PVP精靈種族值沒有太明顯的提高，只不過是690種族值精靈普及開來，所以，這也是當時標準的精靈種族值了

17年瀚宇星皇體力170，攻擊135，防禦115，特攻70，特防112，速度128，體力170這在現在也是一個非常高的數值，其餘數值放在現在（2019年年底）也是非常標準的數字，在17年上半年730及730以上種族值的只有神獸，異能王，索倫森這幾個，這時候年費的在人們心中的地位被人們抬高到新的第位，為了保持這一地位，使得淘米在之後一年多的時間讓劇情中精靈王級別的精靈限制在730

直到18年下半年，730種族值才開始在劇情中普及

18年天啟帝君體力175，本攻150，雙防120，非本攻75，速度140，這個精靈種族值基本是照著聖譜的模子做出來的，只不過體力少了5點，每一項數值在現在也都是頂級的，現在的730種族值精靈只是在某一項達到了帝君的標準，其餘都是過及格線，這些數值基本上是常規精靈所能達到的最高數值了

19年混元天尊，體力180，本攻150，非本攻80，雙防125，速度140，各項數值也是常規精靈所能達到的等級數值，只不過雙防提高到了125

對20年精靈種族值的猜測，體力185，本攻150，雙防125，速度145，非本攻75/80，在近幾年中，常規精靈的體力種族值極限是180，去年庫貝薩提高到了185，2020年很可能將常規精靈體力種族值提高到185，而本攻150，這個數字不需要改變，雙防改變的可能性也不太可能，速度種族值常規精靈的速度種族值極限是140，但是很明顯這個數值不可能長久。

如果年費種族值要提高，那麼這是最大可能性的一個突破口，接下來一年的時間裡速度超過140的精靈可能會大規模出現，而新年費，則是一個開始，非本攻，80的可能性最大，既給了聖譜排面，又在前一年精靈種族值的基礎上有了提升，但是75也是有可能的。

至於新年費強度，新年費出現應該不會將三黑屋格局改變，因為三黑屋成員任意一個實力都高於現在巔峰大害，如果三黑屋成員被放出來，會使現在巔峰格局進一步固化，但是也應該會對巔峰產生重大影響，至於特性，很可能不再繼承前幾年年費的免粉特性。

可能會對紅傷勢力進行抑制，比如鎖傷，免弱化，而不再免粉，而輸出，可能會混傷輸出，但是偏紅傷，對粉傷勢力進行抑制，而免控，則可能在技能上，但是有免斷，免控技能很可能帶有免斷回血，增傷等特性。
