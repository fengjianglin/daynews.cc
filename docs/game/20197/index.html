寡婦雷克塞強勢領銜 經典豬妹二級抓下 廠長五大招牌英雄盤點

2019-12-17 2,152條評論 4,594次閱讀 2039人點贊

LPL的001號選手廠長於昨天正式宣布退役，七年多的職業生涯中，廠長打出過無數讓人印象深刻的精彩操作，其中有很多值得銘記的瞬間永遠定格在了無數LPL觀眾的記憶之中。

今天我們就以英雄的角度來盤點一下廠長職業生涯的五大招牌英雄。

暗凱傳說：寡婦

廠長整個職業生涯最具代表性的英雄無疑就是寡婦製造者伊芙琳了，其生涯勝率為71.8%，2015年的MSI總決賽，廠長以傳說級別的寡婦擊敗了SKT，成功拿下世界冠軍。

當時廠長繞開開團的畫面依然歷歷在目，在與SKT的第五場比賽中，廠長的寡婦通過關鍵的兩波繞後開團成功擊敗了SKT。

符文出裝

寡婦在重做之後，廠長也曾在職業賽場上使用這個英雄，這個英雄的現階段定位其實是一個主打刺殺的刺客，而不是以隱身大招開團為主要任務了。

符文上自然要選擇突進刺客標配的電刑與猛然衝擊，副系點出兩個增傷符文絕對專註以及風暴聚集來提升寡婦的刺殺能力。附加屬性中由於寡婦主要依賴技能清野因此不需要點攻速。

出裝上伊芙琳以減速懲戒提升留人能力，之後選擇出巫妖以及鬼書，如有生存問題可以先出中婭，後期補高法強裝備。

打法核心

寡婦的技能加點選擇主Q副E，其在開局階段的打法以快速刷野提升等級為主，充分利用自身四段Q的快速清野以及被動的生命回復提升等級，控制關鍵的河道蟹，如果有可以繞後GANK的機會則可以選擇適當地幫助隊友，總之以自己的等級為絕對核心規劃好前期刷野路線快速到6是最關鍵的。

6級之後的寡婦來到強勢期，以W先手接E或Q再接AQ快速打出電刑並且跟後幾段Q的輸出以R收尾是核心的一套GANK連招，裝備發育成型後寡婦可以單人刺殺沒有位移或閃現的C位，可以利用被動打多次進場的偷人或者單殺。

團戰則需要找好進場時機或者在團戰前抓住落單點打先手減員。

逆天改命：雷克塞

如果說寡婦是廠長的第一招牌的話，那麼雷克塞就是廠長職業生涯中數據最華麗的英雄，廠長共使用雷克塞75次，勝率超過78%，兩項數據均是廠長所有英雄中排名第一的。

廠長雷克塞的經典瞬間非常多，其中最讓人印象深刻的應該是2015春決最後一場廠長雷克塞逆天改命，在前期幫助隊伍挽回劣勢之後搶下關鍵大龍帶隊完成翻盤。

符文出裝

雷克塞在經歷了多個版本的改動後，更加適合專精玩家使用，需要通過更多的遊戲對局來提升自己的熟練度與經驗。

符文上最主流的一套點法是選擇叢刃流，以叢刃搭配惡意中傷提升輸出爆發，配合一個無情獵手和眼球，副系則點出凱旋與歡欣進一步提升綜合作戰能力，附加屬性中點出攻速與護甲。

出裝上建議雷克塞選擇減速懲戒，然後核心裝備為重裝戰士三件套黑切復活甲以及血手，後續可以選擇一定的坦克裝或者在前期出提亞馬特的時候補出巨九。

打法核心

雷克塞的技能加點選擇主Q副E，使用這個英雄的時候要充分熟悉並了解雷克塞地底的偵測形態，在前期雷克塞就有很強的先手GANK能力，可以利用閃現W配合EQ和叢刃的加成快速配合隊友打出擊殺，同時在刷野的過程中充分利用鑽洞機制提升刷野速度，以及在入侵野區的時候規劃好洞口的逃生路線。

雷克塞是一個節奏型的戰士打野，需要不斷地通過觀察地圖和線上情況來尋找GANK機會，通過各種方式的GANK來幫助隊伍快速建立優勢，同時自己也可以更好地做出核心裝備並且佔據野區優勢，6級以大招收尾的強勢斬殺連招是雷克塞作為打野的優勢所在。

團戰期雷克塞可以選擇先手抓機會配合隊友打突進刺殺，進場後搖充分利用大招機制來刺殺或規避關鍵技能。

二級抓下：豬妹

廠長職業生涯中最為人津津樂道的操作可能就是二級抓下了，他曾多次利用這個套路GANK來幫助隊伍快速打開優勢，其中最經典的一波操作就是豬妹打出來的。

2017年夏季總決賽第五場比賽，廠長使用豬妹套路GANK下路的UZI，先是卡視野進草叢然後打先手配合隊友拿下了一血。

符文出裝

豬妹在任何不同版本中都有很強的一個綜合能力，且作為坦克型打野容錯率極高，上手難度也偏低，適合玩家快速地打出優異的實戰效果。

符文上毫無疑問選擇餘震坦克流，第二行點出生命源泉，以骸骨配合被動豬皮，過度生長用阿里提升成長屬性，副系則點出凱旋與歡欣，附加屬性第三行不需要點護甲，點出成長生命即可。

出裝上選擇減速懲戒的熔渣巨人，之後直接選擇出狂徒，後續的出裝以堆抗性和生命值為主，石像鬼蘭頓或者頭盔等等。

打法核心

豬妹的技能加點選擇主W副Q，一級點E，作為一個綜合能力很強的坦克型打野，豬妹可以靈活地改變自己的刷野順序來選擇針對對方的某一路英雄，在前期的GANK中豬妹的Q閃是一個非常好的先手手段，以Q閃配合W配合E的一個爆發和控制，只要黏住就能打出擊殺。

豬妹本身的被動非常適合用來吸收傷害和扛塔，6級之後豬妹可以以QR甩狙的連招以各種角度打先手突襲，越塔是一個很好的選擇，著重發揮豬妹的坦度和控制優勢來快速滾起雪球，等到中後期團戰豬妹則可以選擇以先手開團來配合隊友抓機會。

開團過程中豬妹可以選擇QR或者近身R或者Q閃R，根據不同的情況來選擇先手連招。

視野之外：酒桶

廠長職業生涯中共使用過57次的酒桶，勝率達到63.2%，是除了雷克塞之外廠長使用次數最多的英雄，同時廠長無與倫比的一個卡視野能力也在酒桶的某場比賽中得以體現。

在EDG與IG的一場比賽中，廠長的酒桶以一波教科書級別的卡視野操作配合隊友擊殺了對面的上單艾克。

符文出裝

酒桶在當前版本的打法以全AP裝輸出流為主，擁有非常恐怖的爆發能力，在符文上選擇點出掠食者提升自己的先手能力，配合猛然衝擊提升雙穿，副系點出兩個移速符文迅捷和水上行走來配合掠食者。

附加屬性不需要點出攻速，選擇兩個適應之力和一個護甲即可。

出裝上選擇減速懲戒的符能回聲，之後的核心輸出裝為鬼書巫妖，搭配中婭沙漏或者後期的女妖面紗，女妖也可以根據情況換成其他AP或者坦克裝備。

打法核心

酒桶的技能加點選擇主Q副W，開局階段可以根據敵我雙方線上英雄的屬性來決定自己的開野順序，選擇在3級之後重點照顧容易出現對拼的線上隊友，GANK過程中以E或E閃先手，也可以選擇快速刷野回家出鞋子解鎖掠食者先手打近身GANK。

6級酒桶的整體爆發與先手能力進一步提升，在小規模團戰中能夠發揮巨大的作用，除了以大招作為炸回敵人的控制之外，團戰中也可以以大招打AOE傷害或者打亂敵人陣型，在酒桶的裝備成型後，可以充分利用自己的高爆發打快速秒殺或者將敵方C位打殘。

5V5團戰過程中，酒桶可以選擇先手開團或者後手留大抓機會控制敵方關鍵人，一個關鍵的大招可以改變整個團戰局勢。

經典反蹲：努努

廠長在巔峰時期最擅長的英雄還有一個以食草為核心打法的努努，廠長在職業生涯中24次使用努努，其勝率高達84.4%，在S5時期，廠長的努努表現令人印象深刻。

MSI時期對陣SKT的比賽中，廠長用努努打出過一波教科書級別的反蹲，利用反蹲配合大招幫助Koro1擊殺對面上野順利取得了優勢。

符文出裝

在重做之後努努的核心定位其實沒有改變，依然是一個主打控圖的打野英雄，只不過新版努努的一個GANK能力是有了很大的提升，很適合玩家上手使用。

符文上選擇餘震流，與豬妹的符文點法類似，不過努努需要點出調節來提升中期坦度，最後一行點出復甦來提升自己的治療能力，副系則點出兩個跑圖符文迅捷和水上行走。

主流的出裝思路是先出一個火箭腰帶，可以提升刷野和GANK過程中的一個黏人爆發能力，然後出裝選擇坦克裝，亡者板甲很重要是最適合努努的坦克裝備，之後選擇振奮狂徒等。

打法核心

努努的技能加點選擇主Q副W，努努本身是一個刷野能力非常強的英雄，可以在前期的時候選擇快速控龍，利用真眼拿小龍，以控圖的方式幫助隊伍獲得優勢，GANK的過程中可以選擇線上推雪球或者河道推球轉彎打先手逼對面閃現，前期的GANK一般不追求成功率，只追求刷野過程中的順路GANK。

中期努努的主要任務依然是控圖控關鍵的地圖資源，小龍以及峽谷先鋒是努努需要重點刷的對象，在20分鐘之後努努配合一些打龍速度快的英雄可以快速偷大龍，而在團戰方面，努努的主要輸出以EQ為主，大招用來打群體控制，過程中可以不斷地推雪球來抓機會幫助隊伍拿下勝利。

結語

相信不同的觀眾對於廠長的招牌英雄都有不同的看法，從數據還是賽場表現的層面上我們可以得出不同的結論，但是不管怎麼說，寡婦和雷克塞這兩個打野都會是廠長職業生涯的代表英雄，是一直能夠被人銘記的帶有廠長個人傳奇色彩的英雄。
