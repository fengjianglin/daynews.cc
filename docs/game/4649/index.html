王者榮耀：楊玉環、姜子牙的調整，代表著法師未來的調整方向

2019-12-06 0條評論 15次閱讀 0人點贊

摘要

王者榮耀四周年版本，是對於遊戲的一個巨大變革的版本，隨著版本的更新，英雄調整也是一直在進行的，從最早的射手調整，前期影響力較大的射手降溫；到之後的戰士調整，前期爆發高的戰士降溫；一直都在平衡著四周年帶來的版本影響，英雄調整的周期比其他版本所經歷的時間要更久。

王者榮耀四周年版本，是對於遊戲的一個巨大變革的版本，隨著版本的更新，英雄調整也是一直在進行的，從最早的射手調整，前期影響力較大的射手降溫；到之後的戰士調整，前期爆發高的戰士降溫；一直都在平衡著四周年帶來的版本影響，英雄調整的周期比其他版本所經歷的時間要更久。

最近一段時間裡，王者榮耀似乎將英雄調整的重點放到了法師這個職業上，為什麼這麼說呢？因為目前的調整比較大的英雄體驗服有嬴政、姜子牙，正式服有楊玉環，調整的法師還有明確一個範圍，對於同時擁有爆發和團隊增益的英雄進行調整，單體爆發英雄或功能型法師不做調整，將多方屬性調整為單方屬性，這是目前法師隊列的一個調整方向。

中單職業定位分化：增益型法師降低團隊加成，爆髮型法師主爆發，功能型法師主團隊控制。

首先我們來看看楊玉環目前的調整和調整之前的區別：

調整之前的楊玉環缺乏的是清線能力，優秀的是他的後期poke能力和團隊回復增益效果，這也是楊玉環一直在低端局低迷，卻在高端局火熱的根本原因，調整的方向是將楊玉環這個英雄的一技能的基礎傷害值上調，提高楊玉環的清線能力；下調一技能附帶的強化普攻傷害，楊玉環的的單體爆發能力下降。

這樣的調整使得楊玉環的後期瞬時爆發沒以前那麼高了，楊玉環的的整體強度更加的平衡，高端局不會太過的強勢，低端局不會低迷，各個分段這個英雄都能有一個不錯的出場率，這樣的調整也讓楊玉環從一個增益和爆發兼備的法師轉變為一個主團隊增益的法師。

目前主要是一個功能型法師比較「吃香」的一個版本，調整後的楊玉環更加適合目前的版本，功能型法師的主要作用是打出控制，給隊友一個安全的輸出環境，楊玉環的調整，讓楊玉環不會低於目前的法師清線速度，基本保持在一個比較正常的清線效率，強化普攻傷害的下調，讓中單楊玉環團戰更加保守，更趨向於跟隨隊友作戰，楊玉環的大招，讓擁有楊玉環的一方，打團能力非常的強勢，調整後的楊玉環也是頻繁的在kpl賽場上出現，也代表著這個英雄對於當前版本的契合度是比較高的。

近期kpl比賽數據：

從近期的比賽數據中我們不難發現，楊玉環的調整也是獲得的很多職業玩家的認可，並且只要拿出這個英雄，基本都是勝利的，這種的調整強度，雖然降低了楊玉環的爆發，但是對於楊玉環的清線能力和團隊作戰能力都有了一個不小的提升，更加適合目前的版本。

這也代表著未來法師的一個調整方向，一個法師一定要有它的特色，並且這個特色是是極具代表性的，同時這個特色也需要成為它的一個「特徵」。這是最早的時候，我對於楊玉環頻繁調整的一個看法，那時候還僅僅是一個猜測，並不是十分確定，但是昨天姜子牙的一個調整，讓我堅定了我對中單調整的看法。

我們再來看看姜子牙的調整：

其中調整不是很大的是姜子牙的三技能，基礎傷害沒有提高，增加的是ap加成，也就是說提高了姜子牙的後期三技能，後期姜子牙的三技能更具威懾力。調整影響比較大的是，姜子牙的被動，從每10秒為友軍增加14點經驗值調整為每120秒姜子牙為友軍增加168點經驗值，這個調整也是引起了很多玩家的反響，有的人認為此次調整對於姜子牙和依靠姜子牙體系來打出團隊增益的組合是致命了，很有可能因為這個的調整，導致整個姜子牙體系的崩塌。這種看法不無道理。

這樣的調整對於路人局的影響似乎不大，但是對於高端局的出場率肯定是有比較大的影響的，大家都知道，在高端局或者職業賽場上，大家都是非常喜歡使用到姜子牙這個英雄的，在職業賽場上幾乎都是非ban必選的英雄，這個英雄的強勢點主要就來源於被動的團隊加成效果，每10秒的一次經驗加成，讓隊友可以在對線期一直保持著壓制力，減緩隊友對線期的壓力，提升隊伍的整體節奏，這都讓很多戰隊一直保持著熱度，但是總是常年存在於ban位。沒辦法，姜子牙在職業賽場上，被開發出來的體系非常的多，一旦拿到手中，勝利等於更近一步。

下面給大家具體分析一下：

第一點，開局120秒的時間，也就是第一波小龍刷新的時間，在這個時間點，如果隊友的對線不爆發人頭，有姜子牙的一方，基本都是快到4級而沒有到4級的一個階段，如果是以前的姜子牙，在經歷10秒鐘，除了輔助，其餘成員皆四級，大大的提高了第一波小龍的爭奪成功性，調整後的姜子牙體系，將不再具有同時間時的這種經驗優勢，對於姜子牙體系來說，是一個很大的衝擊。

第二點，隊友的邊路統治力下降，調整前的姜子牙的每10秒一次調整，無論是射手路、戰士路，還是中路、打野的隊友，基本能夠保證大部分時間的等級優勢，特別是一些射手，例如昨天聊到的孫尚香，快速到達2級的強勢爆發點，很有可能利用這部分等級優勢，直接擊殺對線的射手，進而壓制。姜子牙的被動調整後，無限的將這部分的爆發期延後了，很多搭配姜子牙快速搶二級的英雄，都基本失去了這部分的壓制力。

第三點，姜子牙的被動更趨向於單純的一個經驗加成了，對於隊伍的整體收益沒有以前那麼強勢了，以前是每時每刻的增益，現在是每兩分鐘的一次突發增益。強度是肯定下降了，姜子牙的團隊加持沒有以前強勢，但是也不用妄自菲薄，姜子牙的這種調整，會對順風局的加持更恐怖，同時大招的調整，讓姜子牙的壓塔能力更加強勢，輔助流姜子牙也是不錯的加成方式。

中單調整總結：

對於未來的法師英雄，調整的導向都會非常的明確。一、保證這個英雄的基礎強度，也就是前期清線的能力，保證清線的效率；二、明確英雄的定位，定位更加趨向單一，團隊增益型的中單，技能主要以團隊增益為核心，降低英雄的其他屬性；三、對於分化嚴重的中單，通過調整它的核心機制來降低她對於高端局的影響來平衡低端局和高端局的英雄出場率。

當然，每個職業英雄的調整，都是為了這個英雄更加平衡，這也是遊戲英雄調整的準則，這樣的調整符合需求。
