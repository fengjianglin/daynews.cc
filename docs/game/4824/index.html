陰陽師：新SSR/SP將於12月登場 SP金魚姬有望 死神二期聯動涼了

2019-12-04 0條評論 14次閱讀 0人點贊

摘要

陰陽師對於這次體服開啟的大型世界級活動，其中涉及到的一個細節設定，引發了很多玩家的討論，那就是官方再次對新SSR/SP進行了暗示，基本上可以確定將於12月登場了，來看看具體的情報分析吧。

陰陽師對於這次體服開啟的大型世界級活動，其中涉及到的一個細節設定，引發了很多玩家的討論，那就是官方再次對新SSR/SP進行了暗示，基本上可以確定將於12月登場了，來看看具體的情報分析吧。

具體的暗示來自這次的世界級活動中的商店，本次活動涉及到了隨機SSR/SP式神的召喚券，明確表示了不包含下期新式神，也就是說下期新式神將會在活動結束後馬上推出和登場。

藉助這個暗示我們可以算一下具體的新式神的登場時間，首先該活動持續12天左右，體驗服的商店明確表示是保留到15號的，而正式服實裝該活動會推遲一周，也就是說正式服的商店會保留到12月22日，那麼根據慣例的維護習慣，新SSR/SP推出和開始抽的時間大概率是在12月25日聖誕節當天，正好連著元旦一起過了。

雙旦提前的活動推出新高稀有度式神沒問題，但實在是讓人扛不住了，這速度未免也有點太快了。雖然說時間上是合理的，因為今年過年比較早，1月25日就是大年初一，所以按照一個月一個高稀有度式神的頻率來說，也就是聖誕節推出了。

本來以為會在中間穿插一點聯動，比如說死神聯動二期活動，但官方現在已經算是徹底明確表示了，這次的隨機召喚券不包括下期新式神，也就是說死神二期聯動涼了，12月底還真就是新的SSR/SP式神。

而到底是是新的SSR還是SP，目前還不敢確定，官方要真的整的話，無外乎是兩種方式，要麼12月底新SSR，春節直接雙SP，要麼12月底新SP，春節SSR+SP的模式，而候選人的話，估計SP金魚姬真的應該是快了，因為官方插畫圖把金魚姬的直接給整下架了，估計是擔心提前泄露的問題，等後續具體的情報吧。
