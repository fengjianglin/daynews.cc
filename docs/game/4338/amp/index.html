談遊戲風雲，評好玩遊戲。玩遊戲，歡迎關注紅燭鎮！

最近富堅老賊的《獵人X獵人》正式被騰訊改編成手游全平台公測。作為獵人老粉，當然第一時間就趕緊體驗了遊戲。

整體來說，《獵人》手游在許多細節上對原著還原還是不錯的！

比如在主線任務中，到達考場前會和原著一樣面對猜謎婆婆「兒子和女兒掉到水裡，先救誰？」的靈魂拷問。

老粉絲必然知道這個時候要選擇「沉默」。遊戲里還有許多諸如此類的細節設定讓IP獵人粉絲有瞬間回憶起淚點的感覺。

如果拋開IP，從遊戲玩法上談，《獵人》手游總給人一種《地下城與勇士》在移動端上做嘗試的感覺。畢竟有太多玩法很相似了。

就比如，今天我們要說的天空套！

從500到3000，天空套的氪金門檻均價？

「天空套」，是DNF玩家對遊戲中當前頂級屬性時裝的代稱。

從2009年DNF周年慶發布第一套時裝冥域天空以來，國服已經陸續更新了12套天空套時裝。入手途徑只有時裝合成，搭配當前版本頂級的屬性，天空套一度是DNF玩家身份的一個標誌。

《獵人》手游開服就學到了這一招。

遊戲中開放了諸多套換色的時裝，同時可以使用永久時裝合成「流雲芳華」套裝。簡單對比一下屬性就不難發現，流雲2件套就已經遠超普通時裝5件，天空套無疑。

《獵人》中一次時裝合成需要50藍鑽的商城道具和2件永久時裝，單件永久時裝售價500粉鑽。合成後保底出一件永久時裝，概率出天空。

折算下來同一部位天空合成的成本為1050鑽一次，1600鑽二次，每次550鑽遞增。

根據遊戲中提示各個部位的合成成功率，用很理想的情況來看。即使歐皇每次都能一次合成成功，需要至少5250鑽石，當然這個幾乎是不可能。

而如果取非常理想的平均值來計算，則需要大約20650鑽石來才能合完一套！考慮首沖雙倍以及一些開服獎勵，大概也需要1000元RMB左右的成本。

有些玩家擔心目前遊戲中粉鑽的獲取渠道極少，根本達不到合成需要。這其實有點多慮了，遊戲在粉鑽不足時會自動消耗充值的藍鑽……所以，氪就對了。

看著系統屏幕一直在提示XXX玩家運氣爆棚，獲得流雲時裝。趕緊嘗試在遊戲中私聊諸位大佬，了解下實際天空合成的成本究竟如何 ？

畢竟我充了98塊，血本無歸只剩下一個永久頭……

私聊了7位系統通告的大佬，有3位給出了答覆：一位氪金1000元左右，另一位合滿全套氪金約3000元，這樣看起來基本跟DNF中天空的獲取難度相似。

不過也有一位歐皇回復我當前充值400多塊，天空套時裝已經成功合了四件，算下來100塊一件……從非酋到歐皇，事實就是這樣的無情。

雖然樣本比較小，不過綜合3位大佬的回復，合成天空套的人均成本約為1500元左右。

《獵人》手游只是DNF的探路者？

天空套只是一個縮影。

就目前體驗到的玩法來看，《獵人》手游在諸多細節上確實滿滿都是DNF的影子。也有不少人直接認為這就是給DNF手游探路產品，於是玩5分鐘後就開始從氪金、玩法到天空一路噴過去。

但其實這個想法實在是有些幼稚了。

雖然騰訊有讓《刺激戰場》和《全軍出擊》兩款自家的吃雞遊戲火星撞地球，再用《和平精英》讓陪跑者下線的前科。但《獵人》和《地下城與勇士M》兩款手游面臨的情況卻又有不同。

一方面這並不是同一IP下的兩款競爭產品，兩個遊戲背後本身都有著巨大的IP效應支撐。

騰訊在2019年4月宣布與JUMP達成合作，拿下《獵人X獵人》的IP，如果只是為了做個陪跑產品，怕他是閑得慌了，有這點錢幹啥不好？

另一方面DNF手游已經雷聲大雨點小的喊了很久，究竟什麼時候上線仍未可知。而就算DNF手游上線以後，在當前玩法上能比《獵人》高出多少，其實個人並不做太高的期待。

畢竟在近期新發布的遊戲裡面，《獵人》整體來說算是做的還不錯了。

近期會陸續跟進這款遊戲的情況，後續也會為大家送上更詳細的評測。歡迎關注我哦！
