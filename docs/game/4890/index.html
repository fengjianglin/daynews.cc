主C英雄：

技能：

在石頭人法力值充滿的時候釋放，沖向一名隨機敵人，造成魔法傷害並擊飛附近所有敵人將他們暈眩，基礎傷害為125/200/275，暈眩時間2/2.5/5。

副C英雄：

技能：

在火男法力值充滿的時候釋放，發射一團會彈射的火球，對命中的所有敵人造成傷害，基礎傷害250/325/400，彈射次數5/7/20。

裝備：

大劍+女神淚合成青龍刀，攜帶者使用技能後每次攻擊可回復18%的最大法力值，大棒+腰帶合成莫雷洛秘典，攜帶者使用技能造成傷害時會燃燒目標，在十秒里持續造成真實傷害，總額相當於目標最大生命值的20%，暴擊拳套+大棒合成珠光護手，攜帶者技能可造成暴擊，三件裝備具備後火男戰鬥力大大提升。

陣容過渡：

前期使用三魔法師四海洋和兩守護神進行過渡，吸血鬼和石頭人在前排抗傷害，岩雀球女輸出。

中期在之前的基礎上加上兩山脈和四守護神，石頭人的裝備可以直接給到。

後期人口到八，風女和火男拿到陣容徹底成型，魔法師和海洋羈絆的存在讓雙C快速釋放技能來收割戰場。

該陣容有山脈和守護神的存在前排坦度足夠，海洋羈絆和魔法師羈絆觸發後是石頭人連續大招造成爆炸傷害，在此基礎上加上火男的大招戰場瞬間被收割。
