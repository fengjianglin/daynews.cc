DOTA2小小趣事：機智的網友已經發現了Ame的BP思路

2019-12-16 1,012條評論 4,864次閱讀 956人點贊

DOTA圈颳起了一波換LOGO的風潮

從LGD更換了戰隊新LOGO之後，國外的各大戰隊也開始了更換LOGO的狂歡，EG和Winstrike相繼發布了自己的最新的LOGO。其他戰隊也跟風開始惡搞，各種妖路的LOGO都出現在人民的眼前。雖然是惡搞，但是水友們覺得Liquid和C9戰隊的LOGO還有點可愛，紛紛慫恿兩支戰隊更新LOGO。

Ame的BP策略

看完WESG的採訪和比賽之後，泥潭水友們根據這些總結出了Ame的BP策略：由自己的大哥位置出發，然後開始衍生出一些列的英雄。還總結出了LGD的隊內地位：改版前：4215雞3；改版後：4215雞雞雞雞雞3。看來查理斯在隊內地位實在是太低了，每把都是疊最厚的甲，挨最毒的打。

B神在線想辦法

B神發微博稱自己花50塊都要糾結很久的時候，水友們給出了扎心的評論：當你捨不得多花50塊的時候，但是你可以花400萬買一個C。這個評論真的是戳中了淚點，不過Aster在連續的換人之後，開始有了不小的起色，反觀那個400萬C，倒是毫無建樹，在EHOME的新生活還是需要段時間來磨合的。

刪完了，全都刪完了

新版本發布後，水友們直呼這V社也太勤勞了，瞬間就從7.23a版本更新到了7.23e版本了，照這個速度更新下去，留給7.23版本的字母也不多了，說不定下一次更新就變成7.24了。再加上新版本的更新一直在刪除道具或者地形，從剛開始的每邊7個聖壇變成了現在每邊一個聖壇，照這個速度發展下去，聖壇也將會離我們而去。
