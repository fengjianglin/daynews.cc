王者榮耀：巔峰賽張飛上BAN位，切後成本抬高後的必然結果

2019-12-16 471條評論 4,949次閱讀 504人點贊

最近我在1800分的巔峰賽上發現了一個意想不到的操作——「隊友指名要BAN選張飛。」

本以為這只是極個別玩家的要求，卻不料在接下來的幾局遊戲中，連續碰見多個玩家如此要求。

為此，對於這現象我不得不重新審視一番：「是什麼讓張飛值得被BAN選？」

1.

早期針對張飛的削弱，導致了他對局思路轉變。

在S7賽季時，張飛曾遭到一次削弱。

由於憑藉著3技能的可攻可守，張飛總能在賽場上成為一個左右戰局的關鍵；因此官方針對張飛的3技能，從被動的能量積蓄上動刀，使得3技能的實際冷卻高於字面上的冷卻。

張飛的3技能CD為40秒，但由於能量條的限制，通常CD在1分鐘以上

而這一個調整的思路雖說是在施法頻率上動刀，但也直接的限制了張飛在對局中頻繁的利用大招作為開團手段。（如果貿然將3技能作為開團手段，單憑3技能變身後的技能組，張飛將很難保證C位的生存環境）

為此，削弱後的張飛在輔助定位上更傾向於打「反手」保護，而並非像早期的張飛那般可攻可守；（保護對於張飛來說是更容易完成的工作，也是一個下限的保障）

這直接的導致了在至今的近10個賽季中，張飛雖憑藉著自身的厚實身板依舊在路人局中保有可觀的熱度，但其對局強度卻遠不如那高光時期。

從「可攻可守」到「擅長反手保護」，張飛的適用面狹窄了許多。

在應對部分注重強開團的陣容時，張飛的保護能充分體現他的價值；但一旦遭遇擅長「消耗拉扯」的對手，張飛就會顯得極為被動。

我們腦補一下，在面對諸如孫臏楊玉環這樣的對手時，張飛除了給團隊提供護盾以外就很難有更多的突破；而中遠程的距離，又使得張飛就算想強行開團，依舊要考慮「命中率」與「隊友是否能跟進」兩個問題。

因此，張飛在應對強消耗陣容時（即拉扯陣容），他會陷入一個兩難的境地。

敵方的消耗就像溫水煮青蛙，火候到了想跑也跑不掉；而想避免消耗強行與之一戰，又有著諸多限制，一旦沒打出實際收益，己方就又陷入一個「張飛沒大」的更加被動的局面。

2.

S17賽季給予了張飛一個不被動的理由。

說來說去，又繞回了S17賽季最為關鍵的射手位。

在連續多個版本的扶持加強下，現如今的射手無論在發育效率，或者生存環境上，都有了顯著的提高。

拿當前版本的幾個主流射手來說，馬可波羅的一梭子子彈落在身上就是近半管血條，孫尚香的翻滾一炮就是毀天滅地，這都是近乎於核武器般的存在；而他們自身相對靠後的站位以及相對靈活的位移，更使得他們在團戰前的對峙中，就能打出可觀，甚至致命的消耗，從而決定了團戰的走勢。

同時，這也意味著，誰的射手存活的越久，對於敵方的壓力是成倍的增長。

為此，當前版本的團戰思路就由於射手這個關鍵點而定型：要不保護己方射手讓他相較對方有更好的輸出環境，要不配合己方打野快速的處理敵方射手讓他無處輸出。

在射手強勢的版本，團戰思路無外乎這兩點

可近幾個版本對於戰士與刺客這兩個位置並不算太友好。

作為射手的「對家」，戰士刺客與射手一直存在著直接的攻守關係。在射手獲得版本環境的大量扶持時，他們並沒有得到太多收益，甚至還變相的淡薄了自己的存在感。（野區經濟低迷，戰士被貶發配邊疆）

這樣的此消彼長，也就使得「切後成本」抬高，「配合己方打野快速處理敵方射手」這個方案的實施難度也就進一步提高。

而張飛在巔峰賽上BP率的大漲，也正是契合了「保護己方射手」這一點思路；

由於射手在後期的威懾力，也使得張飛的「反手」多了一個理由，逼迫敵方前來「強開」，使得自己從「被動」轉化為「主動」。

3.

那「保護」的收益就遠高於「進攻」么？

按照上文的講述，估計會有不少朋友認為我是推崇防守，貶低進攻。

但實則不然。

「保護」雖說契合了當前版本的主流環境，但不意味著只要徹底跟隨射手，從頭到尾無微不至的保護即可打贏比賽。（畢竟保護是建立在己方某個單位對敵方具備威脅的基礎上）

無論什麼樣的對局，其實都由「運營」及「團戰配合」構成。

其中，「運營」負責的是我們常說的「對局節奏」，也就是遊走位（中野輔）的主要工作內容。

藉由前中期的節奏掌控來佔據更多的資源，從而在經濟和等級上處於優勢，這就是「運營」的價值。

魯班大師由於3技能的存在，使得他在GANK時有極高的主動性

至於「團戰配合」，雖說在當前版本的對局中有著相當大的比重，但依舊建立在「運營」的基礎上；假若前中期處於過度的劣勢，在經濟上被壓制過多，就算擅於「團戰配合」也很難有著出色表現。

而節奏上的「運營」，其實就是張飛相對薄弱的一塊。

無論是1技能的擊退，還是2技能的護盾，都使得張飛在GANK上很難具備屬於自己的主動性（大招的施法頻率太低，很難在每次GANK中合理釋放來佔據主動）；那對於節奏運營，選擇張飛就更加依託於中野兩個位置在前中期的表現，而自身只能在他們的運營基礎上作出有限的配合。

張飛在GANK中很難展現價值

因此，部分擅長進攻的輔助（比如鬼谷子）不意味著在當前版本就弱於張飛，只是要懂得怎麼在前中期節奏上滾出讓對方難以招架的雪球罷了。

至於進攻防守誰強誰弱，這並不應該放在同一個層面進行討論。

一個擅於運營，一個擅於配合，兩者風馬牛不相及，自然也就沒有比較的意義。（這個觀點在前陣子的文章中也有所體現，草食性打野並不弱於肉食性，而是他們擅長的東西不同罷了）

4.

張飛BP率的上漲，還與巔峰賽的環境有關。

在「老劉備」的巔峰賽數據統計中，佔據榜首的依舊是露娜馬超與雅典娜。

這三名英雄向來都有著極高的上限，隨著分段的提高，玩家對英雄的理解越發完善，他們所能表現的價值就越高。

而他們的強度更多是源於針對射手，憑藉著強大的機動性來對敵方射手施加壓力。

也正因為他們「盯著」射手，才有了張飛的用武之地。

就像「保鏢」這個職業，他們被僱傭的原因，無非就是僱主受到了威脅，或者僱主樹大招風需要保障安全。

因此，張飛的BP率大幅上漲，一是版本扶持了射手使得他們值得被保護，二是巔峰賽的主流環境中依舊存在太多針對射手的英雄。
