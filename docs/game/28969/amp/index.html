測評《形骸騎士》：白虎王子異界遊記，可圈可點但仍待完善

天天要聞
2019-12-20

感謝今日頭條遊戲頻道提供的測評機會o(*^ ^*)o

《形骸騎士》

《形骸騎士》由無限飛遊戲開發，Gamera Game代理髮行，已於2019年12月12日正式發售。發售兩小時便已進入steam熱銷榜首頁，而早前發起的眾籌更是以參與玩家1188位、籌得款項236,963,.88元、達成度473.93%的結果圓滿結束，可見《形骸騎士》早在發售前便在大量玩家心中留下了深刻印象，而玩家對於這款遊戲更是飽含期待。那麼這款國產rough lite遊戲究竟如何呢？今天想就幾個方面與大家聊一聊。

註：測評文章主觀性較強，僅做參考；如有不同意見，歡迎討論。

劇情完整飽滿，人物活靈活現

在開始評價之前，我想首先放出官方關於《形骸騎士》的背景設定：

《形骸騎士》的故事發生在遙遠的過去，在一個神秘國度中有一個名為「塞勒菲斯」的城鎮，這座城鎮處於群山之中，核心的區域是當地的教會所在的城區，佇立著一個高聳入雲的塔樓和一座恢弘的大教堂。在沒有任何先兆的情況下，一場瘟疫在塞勒菲斯突然爆發。瘟疫肆虐之下，塞勒菲斯近乎毀滅，大量不知名的怪物出現在這座城鎮之中，居民生死未卜。

王子阿貝爾，與他的兄長凱因一起前往城鎮調查事件真相。就在到達塞勒菲斯之後，凱因突然變得行為異常，拋下弟弟阿貝爾一路沖向城中的高塔。因為擔心兄長凱因的安危，阿貝爾緊隨其後，沖向了城堡……

緊接上面這段劇情故事，便是遊戲的開頭，白虎王子阿貝爾來到城堡，一路殺到兄長所在的高塔。——令人匪夷所思，遊戲開場演出竟沒有前因後果。既然有著詳細的背景介紹，卻在遊戲中沒有一點體現，一上來就進入副本，讓我懷疑製作組是否缺少一筆製作開場演出的經費。不要小覷遊戲中開場演出的作用，優秀的開場演出不是簡單的操作介紹，它總領整個遊戲劇情，奠定故事情感基調，它更是吸引玩家深入遊玩和埋下伏筆的最佳選擇。而《形骸騎士》的開場卻有些突兀。

遊戲第一個boss

接著談談《形骸騎士》劇情設定中讓人欣喜的優勢。相比起我寫過的上一款rough lite遊戲《爍石物語》，《形骸騎士》製作組在這款遊戲劇情方面的打磨可謂煞費苦心。遊戲在宣發時主打的口號之一便是「用劇情銜接刷本」，為了能使劇情與rough lite遊戲本身「刷刷刷」的性質相輔相成又不顯突兀，遊戲將劇情點主要設置在重生時自動與npc威爾觸發的對話、副本中不具攻擊性的小怪、boss前自動觸發的對話中。其中與npc威爾的對話文本量豐富得讓人驚訝，重生多次對話竟少有重複。從上述官方背景設定中諸多原創人名、地名可以看出官方試圖營造的世界觀並非僅在這款遊戲為止。各個登場的人物——雖然有些目前都僅有一句台詞——他們的台詞中都隱藏著或多或少引人入勝的話中話，如主角的兒子，這一登場最讓玩家措手不及的反轉角色，讓人不禁發問他的故事究竟是什麼？如果在這款以白虎為主角的作品中沒有解答，我們可以相信無限飛工作室會把他的故事製作成下一部作品。

主角的兒子（…）

談到文本，不得不稱讚《形骸騎士》文案組寫作功底之深厚。作為一款平面2D遊戲，除了遊戲中寥寥可數的立繪，表達主角情緒、性格的任務便歸到了對話上。多虧了文案組的工作，《形骸騎士》的各個主要人物的性格特徵基本上都得到了很好的體現：主角白虎王子阿貝爾、雲遊商人威爾、劍術大師……主要角色詼諧的演繹很難讓玩家不產生喜愛之情，更何況還是furry角色設定，有誰不喜歡獸人呢？

然而豐富的劇情也給遊戲連貫性帶來了一定麻煩。遊戲設定中，一個大副本有多個小地圖，一旦在其中一個小地圖死亡進度便會清零，從第一個boss重新攻略起。這個設定在rough lite遊戲中並不少見，但遊戲劇情對話卻讓這個過程變得啼笑皆非。當玩家擊敗其中一個boss時，與boss的對話揭開了下一個小地圖的序幕，而重生時與威爾的對話也進一步推進了劇情。然而當玩家再次進入副本時，又要重新對戰原來那個已「改邪歸正」的boss。這個設定實際上嚴重影響了劇情的連貫性，摧毀了製作組試圖用劇情串起整個遊戲的設計理念。作為一款注重劇情的遊戲，比起劇情的豐富，「合理」是更需要追求的目標。《形骸騎士》由於其魔幻、平行時空的設定，諸多設定都可自圓其說，唯獨上述的情況，成了一個十分破壞遊戲體驗的突兀設計。要麼拋開劇情純粹打磨遊戲流程，要麼儘可能讓劇情更加合理，要想二者兼顧還需要更多的製作經驗。

小怪重複度高，打擊感較一般

遊戲基本架構簡單易懂，總體地圖設計類似惡魔城，每次進入副本地圖都會經過重新排列組合，小怪與boss、寶藏與商人都會被隨機分散到地圖的各個位置。遊戲總目標為擊敗每個地圖的boss，玩家需要一邊探索地圖尋找boss，一邊收集裝備強化自身。

《形骸騎士》的戰鬥系統中規中矩，沒有太多硬傷，也沒有太多驚喜。遊戲原始設置的按鍵模式操作起來比較麻煩，鍵鼠玩家建議調成自己順手的按鍵或使用手柄。遺憾的是《形骸騎士》身為動作遊戲，遊戲主角阿貝爾動作招式過少，主要攻擊與次要攻擊都只有一個動作，沒有連招系統，甚至普通攻擊由於幀數較低都顯得不夠流暢。外加遊戲音效與動作的搭配也比較粗糙，又幾乎沒有畫面抖動，使得遊戲打擊感整體觀感較差。

平a與技能

至於小兵的攻擊模式更是簡易，主要就是衝撞攻擊、投擲滾石、彈幕攻擊等，但作為小兵其實也不需要過分強求。讓人失望的主要是小兵重複利用率過高——在通關第一個傳送門後本期待看到全新的地圖，不想就只是第一個地圖完全複製再加上一些紫煙特效，甚至背景音樂也沒有根據氛圍變化——第二個傳送門中全新地圖只有一張。

BOSS關卡的設計相對更讓人驚喜，主要攻擊方式是「平A + 彈幕」的模式，但簡單的攻擊搭配根據劇情做出了調整，彈幕整體設計也比較有新意。然而BOSS難度可能有些過於簡單（相比起前期難度較高的探索過程），主要體現在boss血量較低，攻擊模式反覆單一，如果玩家身上有增加攻擊力的裝備，沒有幾下就擊敗了boss。

絢麗的彈幕攻擊

音畫精緻出彩，總體搭配不夠和諧

無限飛工作室為所有角色與怪物都使用了逐幀繪製的動畫，這一手法工作量之大令人驚嘆。《形骸騎士》製作人之一何一凡在接受採訪時坦言：「以一個boss戰鬥為例，我們大約需要繪製幾十秒的角色動畫，而就算我們使用了減少部分拍數的方法也至少需要繪製幾百張圖片，加上怪物角色與角色數量繁多，這確實是一個非常漫長的製作過程。」在此基礎上，上述的攻擊模式粗糙、戰鬥系統簡單的問題都有了答案——工作量實在是太大，以至於想要一個成員不到十人的小型工作室做到盡善盡美確實是強人所難。

在場景繪製上，遊戲建築融合了巴洛克與哥特式復古樣貌，同時參考了上個世紀初的新藝術運動風格，以幽深、神秘的色調為主，整體風格與遊戲魔幻背景銜接自然，其中遠景的繪製尤其精美，搭配合適的光效營造出了一幅彌散著魔法氣息的異世界圖景。然而遊戲選擇的音樂效果卻是老式電子遊戲風格，甚至有日系搖滾的氣息，反而使遊戲畫面所營造的氛圍大打折扣，著實可惜。

過場cg

同時，遊戲遠景的精緻放大了一些近景的粗糙。如道具：跟隨小幽靈的攻擊技能為發射數個箭頭彈幕向四周擴散，然而這個彈幕相比背景實在過於粗糙了，玩家在遊戲中可以自行心證。上文提及的小怪重複度過高，設計感較弱等問題也使其與精美的場景格格不入。

結語

忽略遊戲工作室能力範圍、國產遊戲等諸多帶有主觀意味的標籤，單看這款遊戲，他仍然優秀並值得一玩，甚至後續加以完善還能達到更高水準。他完善的畫面與劇情給了遊戲一個堅實的骨架，優秀的人設則成為遊戲的皮肉，讓它充滿溫度——我們已經可以看到許多遊戲的同人創作，若能將其變為一個流行ip，後續潛力更是不可小覷。略顯不足的部分主要還是囿於小工作室的製作能力，然而遊戲工作室明顯是以一種認真負責的匠人精神在對待這款遊戲——上線一周便根據社群反饋發布了接下來一個月的更新計劃，相信不久之後，《形骸騎士》會以一種更加完善的姿態呈現在大眾眼前。
