爐石傳說：惡魔就是機械的爸爸？安德羅妮倒數第一逆襲吃雞

2019-12-12 191條評論 1,401次閱讀 333人點贊

惡魔流在戰棋酒館裡一直都是以弟弟流著稱的，畢竟這個版本誰還玩惡魔流啊，太弱了！當然也不能說惡魔流就吃不了雞，只能說惡魔流現在吃爛分都難。而近日安德羅妮玩了一句局惡魔流告訴了網友，惡魔就是機械流的爸爸，倒數第一都能逆襲吃雞！

這局比賽安德羅妮開局就不是很樂觀，用的英雄是尤格薩隆（爛分率62.4%）還是蠻不錯的英雄，但其他7個人走的幾乎都是機械流！在4星酒館時被館長打掉11點血，還剩10點！自己隨從質量很差，可能下一局走的就是自己了，現在安德羅妮剛很焦慮。因為這局節奏很快所有對手都是在走機械，所以安德羅妮也無法轉換陣容了！

在升到5星酒館後再找到一個靈魂雜耍者，湊齊兩個噴子增加點輸出。這裡主要是靠噴子來破除對面機械的聖盾，因為自己又小鬼首領，讓噴子可以穩定輸出。就憑著兩噴子安德羅妮一路苟了上來，不得不說噴子的效果打機械是真的好用！

在這之後安德羅妮的隨從也是越來越好了，質量也提了上來畢竟都玩機械去了惡魔沒人拿！之後補了一手安尼赫蘭站場軍官，畢竟自己血量少（8點）配合戰吼組成一個大屁股站場很不錯尤其是打機械流。但對戰拉法姆時自己的雙噴被優先解決了，自己的穩定輸出沒了，結局自然是自己輸了！但對面的拉法姆居然只有3星，加上場上的怪就只有6點傷害，可把安德羅妮樂壞了！

安德羅妮又苟活了下來，畢竟拿到了二王和虛空領主，使自己的實力又上漲了。在打館長時，館長的隨從貼滿了護盾，而且融合怪有護盾有劇毒。就館長這樣的陣容硬是被安德羅妮的雙噴給打沒了，聖盾完全沒了作用，順利進入前三。本想賣掉雙噴的安德羅妮，現在知道了雙噴才是核心！

最後決賽是對戰鼠王，自己已經是有三連安尼赫蘭站場軍官和三連靈魂雜耍者了，自己的安尼赫蘭站場軍官擁有近160點的血量，還有兩個虛空領主。雖然鼠王有兩個貼了劇毒的融合怪，但依舊被噴子給打掉了，順利吃雞，倒數第一逆襲吃雞！安德羅妮還說了一句「惡魔就是機械的爸爸」，看來作為弟弟流的惡魔打機械還是很好打的，惡魔流並不是一無是處！

大家覺得惡魔流真的是機械流的爸爸嗎？還是說安德羅妮的運氣好才用惡魔贏的，其實惡魔還是弟弟？大家對此怎麼看，可以在評論區留言說說自己的想法！
