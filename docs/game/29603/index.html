王者榮耀冬冠杯RW俠在“演”XQ？保送eStar進淘汰賽

2019-12-25 3,322條評論 4,360次閱讀 2103人點贊

大家好，我是七七。王者榮耀冬冠杯預選賽已經結束，eStar與TS兩支戰隊突破重圍，會師淘汰賽，而KRKPL隊伍不出意外全部回家。選拔賽採用BO1，賽制相當殘酷，不到最後都不知道誰晉級，所以每一支戰隊都拼盡全力。本來以為會進行附加賽，沒想到出現了一個意外。

這個意外就是，RW俠生死局輸給了CW戰隊，這也是冬冠賽中，第一次KPL戰隊輸給了KRKPL戰隊，其主要原因就是輕敵。所謂驕兵必敗，雖然說KRKPL的戰隊實力有待提高，但是RW俠在最不該打團戰的時候接團，導致經濟差被迅速拉大，後續根本沒有繼續發揮的空間，最終輸掉了比賽。

而這場意外導致了eStar直接晉級淘汰賽，與TS攜手出線，而XQ躺著也中槍，連打附加賽的機會都沒有。

起初，XQ打得相當不錯，有著積分領先，狀態也不錯。 本來以為晉級淘汰賽是穩穩的，不過接下來連續輸給了TS與eStar。不過XQ以5分的成績，還可以與RW俠進行附加賽，還算有點機會，但萬萬沒想到，RW俠居然輸給了KRKPL的CW戰隊，我都懷疑RW俠是不是在演XQ，只是為了保送eStar進正賽。最後eStar與XQ同樣5勝2敗，不過因為XQ曾經敗給eStar，所以無緣冬冠賽。不得不說，這個結果有些戲劇性。

目前，冬冠賽八強抽籤已經結束，A組：SLT對戰eStar，B組：AG超玩會對戰ROX，C組：DYG.JC對戰TS，D組：QGhappy對戰GOG。如果不出意外，也不像RW俠那樣輕敵，那麼eStar、AG超玩會與QGhappy提前鎖定四強名額，而看點就在於DYG.JC與TS誰能笑到最後。

這場比賽將於12月26日正式打響，各位小夥伴最看好哪支隊伍呢？而接下來的比賽還會出現什麼意外呢？讓我們拭目以待吧！
