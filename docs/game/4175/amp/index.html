有玩過魔獸世界的都知道，在魔獸世界這個遊戲中，裝備是多種多樣，而且魔獸世界的裝備還有一個特點，那就是裝備保值，而且60級的裝備裝等差距不大，因此很多低級副本裝的裝能夠用很久很久，另外這其中也不乏有一些特效逆天的裝備，當然這些特效強大的裝備，暴雪本身就會對他們加以限制，比如觸發有內置cd等。下面就來和大家說說60級中，有那些特效非常逆天的裝備。

一：暴風雄獅號角。說到這套裝備，那就不得不提到這件裝備的特效。這個飾品號稱當初魔獸世界經典舊世的最強裝備，被目標被擊中時會有1%的幾率使得隊友的護甲提高250點。目前已經不能刷出來了，這件裝備的打戰場時候非常強大，全團保護效果啊。曾有玩家表示奧山見過，全團保護效果，一波打下石爐，LM表示受到驚嚇。逆天之處：全團保護祝福的視覺效果，音效雄壯。這簡直就像是「你們是被命運選中的人，去戰鬥吧。」

二：六魔包。這件裝備世界掉落，而且是裝備綁定，因此很好獲得。而且六魔包可以提供隨機的法術，並且傷害非常的高，魔包的特效可以觸發非常多種，比如傷害型的火球，寒冰箭，連鎖閃電等，控制型的變羊和包圍之風，召喚型的地獄火，虛空行者等。這個裝備每一次釋放的特效都讓人期待。逆天之處：曾經受100%法傷加成。那麼怎麼可以獲得這個裝備，玩家可以通過擊殺45級以上地圖上的怪物掉落，也可以通過盜賊開鎖秘銀寶箱、翡翠寶箱獲取，暗月馬戲團的高級暗月獎品中也獲取這件裝備。

三：無盡黑暗之刃。這把武器就厲害了，它的出處就是瑪拉頓公主，在懷舊服剛剛上線那段時間，有無數的隊伍去刷。這把武器，紫色品質，據說掉落率在0.12%，是法師的刷怪神器。因為以前回藍無CD，所以靠這個「神器」的爆怪，觸發概率相當可怕，有了它就可以輕鬆刷一些高血量的精英怪，如ZG的鱷魚。這個裝備，可以讓法師暴風雪時候可以不斷的回藍，讓法師續航無敵，但也因為太強被暴雪削弱了，目前法師暴風雪也只有第一次能夠觸發。逆天之處，法師單刷神器，無限回藍。

四：瑞文戴爾之劍。非常浮雲的一把劍，稀有程度堪稱神器，儘管官方給出的掉率是1.99%，但恐怕實際掉率要比這個低很多，在很多玩家看來，這DK劍的掉率甚至要比DK馬掉率還要低！它是索姆出品，人們過來刷DK，除了亡靈馬，就是為它。同樣小副本掉落，比通靈的院長之杖要更實用，和飾品「棍子上胡蘿蔔」一樣，是60年代少數能加移動速度的裝備，效果等同於War3閹割版的死騎光環。其最大的特點是，裝備能提高移動速度和生命值回復，這效果和魔獸爭霸中亡靈DK的邪惡光環效果很像。在當時，魔獸世界中有這種光環效果的武器是絕無僅有的。
