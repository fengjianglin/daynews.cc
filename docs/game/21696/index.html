真香！《無盡大冒險》即將迎來新職業

2019-12-18 390條評論 1,402次閱讀 351人點贊

隨著生活節奏的加快，能夠擁有大塊時間玩遊戲已經成為了一種奢侈的想法。手游利用玩家的碎片化時間可以說是抓住了玩家的一個痛點，這種隨時開啟隨時關閉的遊戲類型因收放自如而深得玩家喜歡。但沒想到的是，居然還有可以「懶」到極致的放置類手游出現的玩家面前， 《無盡大冒險》就是一款典型的放置類手游。

放置類遊戲是一種只需要把遊戲放在那裡，遊戲就會根據系統的一系列運算規則自動運行並得到遊戲結果的遊戲類型。這种放置類遊戲的原理其實類似於數學中常見的抽屜原理，製作者事先把眾多的結果與分支條件放在獨立的「抽屜」里，然後製作者會建立一個運演算法則，遊戲會根據公式給出相應的結果。這類遊戲對玩家的操作需求比較少，離線同樣可以進行遊戲邏輯運算，因此也獲得了不少玩家的喜愛。

《無盡大冒險》可以主動操作也可以掛機操作。主動操作就是指玩家進入遊戲後通過點擊界面下方的機能進行打怪，過關成功後獲得相應的道具和經驗獎勵，從而進一步培養自己的角色。

當玩家不能夠手動操作時，可以設置一些自動掛機的選項，比如自動釋放機能，自動出售裝備等等，這樣即便是玩家有事離開或者下線，遊戲依然能夠「進行」，等玩家再次上線後可以直接獲得收益，讓玩家更加自由。

《無盡大冒險》採用暗黑系列遊戲核心職業技能與裝備系統，豐富多樣的道具裝備可供玩家進行搜集，值得一提的是，本作遊戲中所有的道具裝備原型全部來自於暴雪經典遊戲作品《暗黑破壞神》系列，裝備也採用經典的白色、藍色、金色、暗金的顏色分法。

《無盡大冒險》通過深化技能和裝備的搭配來的加強遊戲的策略性，暗黑2最為精髓的符文之語和地下秘境玩法也被加入其中。通過挑戰秘境，玩家可以獲得符文和符文裝備，符文可以鑲嵌，也可以與其他符文一起合成高階符文，多樣的組合使得遊戲更加具有多樣性。

如果僅僅是打怪升級刷裝備這些，玩家很容易就會乏味，好在本作有著風格各異的二次元萌妹子， 著實為遊戲增加了不少色彩。本作中共有六個萌妹子可以使用，她們分別是：獵魔人、秘術師、召喚師、狂戰士、聖教軍和暗殺者。不同的萌妹子擁有不同的屬性和技能，每個萌妹子都有一個戰術技能、一個主要技能、兩個次要技能和三個被動技能，合理的的技能搭配可以打出非常可觀的輸出。

在獵魔人、秘術師、召喚師、狂戰士、聖教軍和暗殺者六位角色登場之後，即將迎來第七名可用角色——德魯伊。擁有著獨一無二的變身形態，可以自由控制冰、火、風和閃電四種能力的德魯伊究竟會給玩家帶來怎樣的體驗呢？讓我們拭目以待吧。
