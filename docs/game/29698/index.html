魔獸懷舊服：一大細節被開發，傻瓜式教程讓你享受“移動恢復”

2019-12-24 3,389條評論 4,400次閱讀 1498人點贊

【遊戲沒有圈兒】今天分享關於「走吃」與「走喝」的相關內容，顧名思義就是「一邊走一邊恢復生命值（血）/法力值（藍）」。好處是在恢復狀態的同時也能夠實現角色移動，比起傳統方式的脫戰恢復手段，它勝在靈巧與機動性，無論是PVE副本與團本，還是PVP拉開距離，它都應該被完全掌握。因為這是魔獸懷舊服中一個非常重要的細節，也是在近期美服5萬美金插旗大賽中（「走吃獵人」）再次重歸玩家視線，我們不得不說說它。

效果展示，請注意GIF左上角驚人藍量變化：

首先要詳細介紹一下背景，為什麼會出現「走吃」與「走喝」這種技巧？《魔獸世界》懷舊服所有職業都特點鮮明，對比正式服的職業角色，你可以理解為某種極端職業特色。既然極端，當然也伴隨弊端。例如影響法系職業的藍量上限，如果麻痹大意地使用技能，很快就會油盡燈枯。近戰職業也有著類似的弊端，例如戰士，在懷舊服中它是沒有強力回血手段的，必須依靠治療單位補足。結合魔獸懷舊服這一大特性，脫戰恢復手段就是各職業脫戰後的當務之急，而戰鬥中尋求脫戰來恢復更是家常便飯。

機制簡單介紹，嫌麻煩的可以跳過。它不是BUG，而是在重疊的技能共CD與恢復機制中有0.5秒的空隙，在恰當的時間使用，能夠實現邊走邊回血/藍。當然，恢復機制的前提也涉及到較麻煩的5秒脫戰等。當玩家脫戰後，即可進入2秒的回血/藍計時，而在這其中如果卡過共CD的1.5秒，並在2秒恢復效果之前使用恢復道具，就可以實現秒回血/藍。想完美掌握這個時機非常難，但經過個人多次研究，結合插件會變得更容易掌握。今天就分享給需要的玩家。

如何成就「走吃」與「走喝」的驚人恢復效果

首先，各大插件整合工具都有一個名為「FiveSecondRule」的插件，它的作用就是提供計規則，例如精神回藍監控、自回血監控、共CD等，它很好地解決了這一難題，推薦大家使用這個老插件。

安裝後可在界面-插件中找到設置面板（默認即可），「強迫症」玩家可以調整框架大小以及顏色等。開啟這個插件需要用到「/fsr unlock」命令，在聊天窗口輸入這個命令後，你的頭像下方便會有計時器標識（血與藍滿狀態不顯示），它能夠幫助你完美監控。

默認是藍色的。當玩家在5秒內沒有戰鬥行為時，它會進入2秒的循環計時。

重點，當進入2秒計時規則後，每完成一次計時玩家將自動恢復血與藍，而在自動恢復之前（到達圖中「臨界點」之前）使用恢復物品（如麵包、水）則完美觸發物品效果，從而實現邊走邊恢復。解釋：前1.5秒為共CD，後0.5秒也就是所謂的「臨界點」。通過這個機制的掌握，法系職業在副本中可以避免拖隊友時間等類似問題，A完一波小怪後也能快速跟上隊伍。而說到對距離極為敏感的PVP場景，它的作用就不言而喻了吧？續航、牽制的關鍵！

根據下圖可以看到，我們並沒有坐下使用「魔法晶水」，而是保持角色移動。而再參考圖中左上角藍量恢復，你會發現其本身質變的效果。

從暗巷丁字路口走到榮譽谷，我們的藍已經接近100%！

這個遊戲機制在《魔獸世界》早年其實已經被開發，但礙於傳播效果的局限性，知道的玩家並不多。我個人在早期也只是略有耳聞，而沒有具體使用過，今天花時間把它搞定，也算是圓夢了。最後，今天講解的內容不僅適用於法系職業回藍，也適用於物理職業回血。有疑問的玩家可以評論回復，知無不言。
