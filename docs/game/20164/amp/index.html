如何帶妹玩魔獸？壇友親自分享經驗

天天要聞
9 小時 ago

作者：NGA-winnilove

有很多大佬帶了妹子最後覺得妹子太菜帶不動了，那麼我就來分享一下，這五個多月如何把一個萌新妹子帶到現在拿了15限時的吧

首先，我忽悠她來玩魔獸，挑選職業的時候一定讓她自己選擇自己喜歡的職業。她說她喜歡玩遠程DPS，我就給她介紹了各遠程職業，最後慫恿玩了個XD。XD練級不會太抓狂，滿級之後大米切奶，團本切咕，野外飛行做任務體驗都非常的良好，這樣不會把她勸退。
其次升級過程，傳家寶、包包、金幣都是一次到位，然後讓她自己去練級了。沒讓她裝什麼插件(後來發了個背包整合和skada給她)並且告訴了她一句話攻略：出什麼打什麼，有什麼躲什麼，莫名其妙掉血看右上角DEBUFF。
妹子自己拍本做任務滿級了(其中德拉諾那一段是我和朋友帶她做的德拉諾19任務，差不多2小時就搞定了，軍團版本是帶她做入侵，也是很快搞定了) 我讓妹子切一會奶切一會鳥排本適應技能，然後仔細閱讀技能。
接著妹子滿級之後，先帶了幾個M本讓她裝等400+，接著帶著她打了一通10+的大米強X到了430左右，再帶她進了H團毛特質。這樣毛了一圈下來就有430的裝等，方便做任務了。
然後我讓她每天自己做世界任務+海底+麥卡貢，一是解鎖飛行，二是附帶可以弄精華和能量
在還不能飛的這段時間，她還是個小萌新，每天上線第一件事就是滿地圖跑任務做任務，我這邊空閑的時候(我當時每周有3天要打M團)就會每周帶她打1-2個15層以內的低保。不帶高層是因為不僅僅她自己要難受我們自己也難受。妹子難受了就會被勸退，我需要給她提供的是一個輕鬆愉快的遊戲環境。讓她覺得在艾澤拉斯打發時間是很輕鬆很快樂的。
在之後她解鎖了飛行，精華特質也都有的差不多的時候，也就是最近這個月，我就沒事喊她錘低層大米，熟悉副本的節奏。最開始是打12左右的，她總是很慌問是不是層數太高了，我怕我不會之類的。當然我就在YY里耐心的給她講解每個需要注意的小怪，BOSS的機制，BOSS怎麼打。新手萌妹當然會犯很多錯，給她說的東西肯定也記不住，但是我從來不給她壓力。並且鼓勵她，你這才玩幾天，現在已經做的很好了。
妹子自己也想努力提高沒事就在論壇看帖。我有時候看到那種大秘境的帖子也發給她，並告訴她，就看你熟悉的那幾個，其他的隨便看看，看了也記不住，這個是要你自己打，再結合看這個才有收益。沒事可以去10C以下的大米練手，再結合帖子看看
有的時候我喊打本妹子如果不想來打，說自己想任務我也不從來不勉強，她自己玩的開心就行。
我也不要求她肝這個肝那個，只要求每天上線用咕咕打5MIN的木樁。昨天檢查了下成果，1000W木樁打了4W+ 裝等440。
於是拉著她去混了一個野團M123，打的很糟糕，木樁雖然搓的不錯，但是實戰畢竟和木樁有些差距，這個問題我以後會每周拉她打個H練手。
前幾天最後一個15限時暴富礦區跳了成就，坦克說妹子你奶的不錯啊，我今天跟一個主播打16暴富都沒你奶的好。
看很多哥們帶妹都是一把屎一把尿拉扯大，離開你了妹子就沒辦法生存了，最後妹子啥都要來煩你，導致帶妹及其不愉快。
帶妹首先一點就是要教會她獨立，你能做到的只是給予她獨立的一些幫助，如果全部依賴你當然沒辦法成長。即便你給她弄了一套畢業裝，她也打不出應該有的輸出，奶不到正常的HPS。裝備只是一部分，比如她滿級我帶她強X到430是為了讓她做任務體驗更好一點。
帶妹打本，一定一定要教會她對應的技術。雖然低層本會很愉快，但是高層本還是得她自己能獨當一面。對應的技術就是在低層副本里一次次的熟悉總結練出來的。副本的細節要給她仔細講解，同時配合每次的大米 而且是一遍又一遍的重複一直到她能消化為止。每一隻需要注意技能的小怪，每一隻BOSS的機制時間軸以及加血注意點，這些我都是一個字一個字的教，說了一次記不住就說第二次，一直到記住為止。
大概就是這樣，好了，我去吃妹子給我剝的柚子了
