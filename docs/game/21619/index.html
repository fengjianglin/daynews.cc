VGAME消零世界世界卧底的堅持異聞攻略匯總 卧底的堅持怎麼完成

2019-12-19 85條評論 1,468次閱讀 341人點贊

VGAME遊戲世界中，有一個異聞是卧底的堅持，很多小夥伴不清楚這個異聞怎麼做，現在為大家帶來了VGAME世界卧底的堅持異聞攻略，如果你不清楚這個異聞在哪裡觸發，或者不知道在什麼地方尋找線索，那就一起來看看本期攻略吧。

卧底的堅持異聞攻略

1、首先在南郊記憶——一個男人易曉傅那裡，我們可以接到卧底的堅持這個異聞任務；

2、前往安提莫德中心，找到他並且對話；

3、前往海恩韋斯特，就在街角，找到他並且對話；

4、前往天橋，找到他並且對話；

5、上述三次對話完成之後，再返回南郊記憶，找到易曉傅，和他進行對話，然後就可以完成異聞了。

任務簡報：

那個叫易曉傅的男人手裡有關於卧底線索的情報，不過他好像沒有辦法解開情報的相關內容，因此希望有人可以幫助他找到情報對應線索的實際內容，我最好問一下他具體的情報內容。

關於卧底線索的情報總有3條，第一條是安提莫德中心隱藏的猛虎，千萬不要和他說「砰砰砰」；第二條是海恩韋斯特最誠實的草帽，千萬不要和他說「雙懲罰」；第三條則是天橋上沒有頭髮的湯達人，千萬不要和他說「後天船」。

我要好好想想這其中隱藏的信息，這三個人真的是非常奇怪，只要一和他們說那幾個詞，他們就情不自禁的開始告白了，我應該弄清楚有關卧底的線索了，現在就回去將線索告訴那個叫易曉傅的男人。

以上就是VGAME世界卧底的堅持異聞攻略以及詳細任務步驟介紹，祝大家遊戲愉快。
