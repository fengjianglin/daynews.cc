FGO2.5禮裝介紹，魔性菩薩上位禮裝來臨？

天天要聞
2019-12-22

FGO日服主線劇情2.5章的開啟不僅實裝了三位從者，也實裝了很多新的禮裝，本期就為大家介紹這些禮裝的效果。

祈願救濟者：常駐五星禮裝 HP+400（1600） ATK+250（1000）

效果為自身寶具使用時充能階段上升2級3（4）次

這張禮裝或許可以稱得上是魔性菩薩的上位禮裝了，畢竟魔性特攻的OC等級提升效果僅有1次，而這張禮裝光是未滿破就有3次，不過考慮到沒有初始NP以及非純ATK加成，使用還是有一定限制，至少對大英雄這種吃OC的一次性從者而言，魔性菩薩顯然要比這張禮裝優秀，不過對於像尼托這種吃OC，能連發，又不專註傷害的從者還是十分實用的，以及大部分OC效果優秀的輔助從者使用這張禮裝的效果也很好，一次中高速副本中輔助寶具的使用次數也大概是3~4次。

抽到的話建議不要滿破，畢竟滿破成本高，收益也不大。

野心的開端：常駐四星禮裝 ATK+400（1500）

效果為自身藍卡性能提升10%（15%），HP回復量提升10%（20%）

如今的四星禮裝能使用的不多，所以非特殊用途的禮裝也沒有什麼評測的必要了，但是年輕的達尼克還是非常帥氣的，不知道這會不會預示著未來的FA聯動復刻會實裝呢？

Noble Piece：三星常駐禮裝 HP+160（800） ATK+100（500）

效果為自身藍卡暴擊威力提升25%（30%）

阿爾戈號英雄：通關主線2.51贈送四星禮裝，在劇情池也可以抽取 HP+320（1200） ATK+200（750）

效果為自身三色指令卡性能提升6%（8%）

通關禮裝一直被稱為「遺照」禮裝，上了這上面的一個都不能活，這次不是像帕茨西和格爾達這樣的「土著」角色，而是在劇情中一起出生入死的英豪們，了解了劇情再看這張禮裝真的是要淚崩。

星之獵人：超人俄里翁的羈絆禮裝，超人俄里翁使用並在場上時，賦予自身每回合獲得8個暴擊星狀態，己方全體暴擊威力提升15%。

配合自身的寶具和三技能效果，可以大幅度提升暴擊威力，同時也滿足了出星的問題，還是比較適合自身的性能的。

白色公牛：歐羅巴的羈絆禮裝，歐羅巴使用並且在場的時候，己方全體的藍卡和紅卡性能提升10%。

戰後的斷劍：曼迪卡爾多的羈絆禮裝，曼迪卡爾多使用的時候，會在自身戰敗退場的時候提升自身外全體隊友20%攻擊力三回合，30%防禦力三回合。

作為一個會「自爆」的工具人，曼迪卡爾多也是把自己的優勢發揮到極致，配合自己二技能和三技能把自己賣掉的能力，這張禮裝也是能為隊友帶來很高的收益，30%防禦力以及加上一技能一共40%攻擊力。
