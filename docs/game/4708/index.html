第二日戰況

第二比賽日強隊也悉數取勝，但過程各有不同，只有North被ATK 2-0橫掃。昨晚A隊與G2的比賽實在太精彩，首圖G2拿下優勢圖dust2，次圖小鎮又順利拿下賽點，難道A隊就要被0-2帶走？結果A隊連追賽點並在加時賽逆轉。圖三雙方又是打滿30局，A隊才以16:14險勝，喜歡的玩家可以看錄像嗷。FaZe 1-2不敵100Thieves，銀河戰艦慘遭淘汰。mouz和液體的比賽也非常好看，結果mouz在決勝圖14:16惜敗。好在G2和mouz都是勝者組落敗，所以尚未被淘汰。而敗者組中落敗的FaZe、MIBR、Sharks和North就遺憾出局。
