轉會匯總：阿水仍下落未明！Kkoma蕉太狼官宣加盟VG！EDG官宣陣容

2019-12-17 2,091條評論 3,560次閱讀 295人點贊

昨日發完匯總版後，相繼傳出SMLZ回歸OMG和廠長的退役消息。伴隨著EDG官宣廠長擔任EDG主教練的決定，不得不讓人感慨青春逝去。今後有時間會做一期廠長的人物專欄，紀念這位曾經伴隨無數LPL觀眾走過這麼多年的歲月。話不多說，不少戰隊今天又宣布了很多消息，現在我們就來盤點一下。

1.kkoma加盟VG，同時引入蕉太狼

原SKT三冠教練Kkoma加入VG的消息此前已經傳出來了，而今天VG官方也是終於宣布了kkoma的加盟。VG昨日宣布EDG的ADC選手iBoy轉會至VG，還引入了iG二隊中單選手蕉太狼！並且從LDL引入了BLG.J的打野Chieftain和VG.P的中單Steel。看來VG雖然賣出了大腿下路和FPX替補打野Xinyi，但是仍然保持不錯的競爭力。

Kkoma的轉會被很多人認為是撈金，但是看完VG陣容後，感覺尚有一戰之力。尤其是蕉太狼在今年夏季賽的表現非常亮眼，是不少LPL隊伍競爭的對象。如今和kkoma、iBoy組成的陣容或許可以在新賽季帶來驚喜。

2.EDG官宣陣容，Aodi、Xinyi加入

EDG今天官宣了德杯的陣容，其中包括成為主教練的廠長，和此前的Heart教練。隊員包括上單選手三位：Jinoo、Aodi和xiaoxiang，打野為JieJie、Xinyi、JunJia中下：Scout、Hope、Meiko。Aodi可謂是LPL不少觀眾的”熟人”了，洲際賽上的”優異”表現和逃跑天賦讓不少觀眾都對這位上單印象深刻，今天Aodi的再度回歸也是讓很多EDG粉絲表示不解。

另外，此前就有消息傳出EDG和VG做了AD和打野的交換，今天EDG果然就官宣了Xinyi的加盟，作為FPX替補打野選手，雖然一直被小天按在替補席上，但是Xinyi的實力還是非常令人期待的，不知道今年的EDG能夠帶來怎樣的表現。

說一句題外話，今年EDG被網友調侃為轉會期最大贏家，截至目前，EDG通過轉會走了iBoy、Ray兩位選手，二隊通過轉會一個去了WE的Yimeng，一個去了LNG的lwandy，共計轉走四位選手，因此賺到的轉會費應該非常之多。

3.SN官宣陣容，SofM加入，阿水不在名單

SN也官宣了自己的德杯陣容，其中沒有阿水的身影。上單Bin、打野SofM、中單Angel、ADC：weiwei和Assum，輔助Swordart。其此前的打野選手weiwei是轉位來打ADC的，所以其實SN的ADC選手還是有空缺的。阿水目前的情況撲朔迷離，作為iG的冠軍ADC，至今都沒有公布自己的新戰隊，不由得讓人捏一把汗。

4.WE官宣：Plex、Morgan、Yimeng加入

WE官宣了三位選手的加入，一位是前LNG中單選手Plex、一位是JDG替補上單Morgan、還有EDG.Y的中單選手Yimeng。在昔日的雙C走後，WE陷入了很長時間的困境，從目前的引援狀況來看，WE可能新賽季的表現很難讓粉絲有所期待。

5.iG再添新輔助Fate

iG今天官宣了二隊輔助Fate的加入，並且官宣了蕉太狼Forge轉會至VG，二隊上單705轉會至JDG。不得不說這一決定還是很讓iG粉絲惋惜的，蕉太狼夏季賽在iG最艱難的時候幫他們抗住了，因而他的實力也被很多LPL隊伍和觀眾們所認可，甚至不少iG粉絲都期待蕉太狼將來能夠成為Rookie的接班人。

此外iG今年似乎也吃到了S9沒有引援的虧，不僅人員調度很困難，在隊員面臨壓力之際也容易產生心態問題。因此今日IG還宣布了二隊輔助的加盟，這是在針對誰其實已經非常明顯了。加上VG雙人組的到來，不知道新賽季還有沒有機會見到寶藍的登場。

6.其他消息

1）FPX官宣三位二隊選手上單xiao7，打野Mingjing，AD Aprill加入FPX一隊。2）RW官宣了明哲擔任教練，VG.P的Steel選手加入。同時RW宣布多人離隊，昨日RW官宣了Jiangqiao離隊，今天又宣布了huatian的離隊和隊長Killua轉會至LGD的消息。3）LNG二隊ADC選手Light調入一隊LNG。4）JDG官宣RWS輔助panfeng及iG.Y上單選手705加入

寫在最後

綜上就是今天的LPL轉會消息匯總了。阿水目前仍舊沒有消息。不過隨著德杯的到來還會有更多的戰隊陸續官宣的，你們看好新賽季的哪支戰隊呢？
