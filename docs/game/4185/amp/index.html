跳傘、開飛機、搞破壞，“爆炸就是藝術”的沙盒遊戲《正當防衛》

天天要聞
2019-12-09

摘要

當我們提起《正當防衛》這款遊戲時，玩家們總會覺得有一種既視感——這遊戲怎麼這麼像GTA？的確，同為自由度很高的沙盒遊戲，放到一起比較也是很正常的。

當我們提起《正當防衛》這款遊戲時，玩家們總會覺得有一種既視感——這遊戲怎麼這麼像GTA？

的確，同為自由度很高的沙盒遊戲，放到一起比較也是很正常的。當然《正當防衛》系列從各種角度上來說，都比不過R星的GTA系列，單從GTA5的銷量上來看，怕不是幾代《正當防衛》加一起還沒有人家一半多。但如果能夠拋開心中固有的成見，盡量以客觀的眼光去審視它的話，你會在其中發現其實這個系列還是有一些獨特的亮點，並且意識到它絕對不僅僅是一款對GTA簡單模仿的作品而已。這個系列發展至今已經出到四代《正當防衛4》了，儘管差評不斷，但這仍然算得上是值得一玩的遊戲。本期就讓我們一起來看看這個系列的初代究竟是怎樣誕生、有著怎樣的特色吧！

“真實感”是我們如今討論遊戲的時候經常掛在嘴邊的一句話。儘管遊戲的內容可以天馬行空，完全不受現實世界的拘束，但有一些規律、公理還是必須遵守。例如你可以把遊戲中的戰場設定在外太空，讓主角和外星人來一場你死我活的火拚，這擺明了就是不可能發生的事情（至少在目前來講完全不可能）。但即使是在這個虛擬的世界中，諸如人沒有氧氣就會死、挨了子彈會出血、失去了重力就會漂浮之類的物理定律還是不能忽略的。否則，我們就會說”這個遊戲的物理引擎太假了”。從這個意義上來講，《正當防衛》就堪稱一個蹩腳的作品。但有意思的是，它似乎完全不在意自己在物理定律方面的錯漏，而是將這種對定律的踐踏轉化成了頗具樂趣性的遊戲內容。如果你能暫時拋開心中固有的那些理念和成見的話，也許《正當防衛》能夠讓你獲得一些有意思的體驗。

《正當防衛》的故事發生在一個名為SanEsperito的南美洲島國之上。這個小島實際上算得上是一個獨立王國，島上的政權由名叫Mendoza的獨裁者把持，他勾結一支黑幫勢力，在島上實施恐怖統治。而玩家所扮演的主角則堪稱一位自由戰士，來到島上協助反叛軍推翻Mendoza的勢力。小島的設定是遊戲中的亮點之一。在這個面積1000多平方公里的小島上覆蓋著大面積的熱帶雨林，同時也有一個頗為現代化的都市，軍事基地、港口、機場等等在動作射擊類遊戲中經常出現的場景也一應俱全。儘管在開始遊戲時的讀盤時間比較長，但是進入遊戲之後你可以在島上的每一個角落自由行動，而不需要額外的讀盤時間。島上的風景迷人，光線效果也非常不錯，駕駛汽車四處遊盪也是一件十分愜意的事情。

之所以提供了面積如此龐大的場景，是與遊戲的進行方式相關的。《正當防衛》主要的遊戲內容與《橫行霸道》類似，同時也有一些類似於《僱傭兵》等遊戲的特徵。遊戲流程以任務的形式推進，主要不斷地完成主線任務，就能將你導向遊戲的結局。同時，你還可以在各處接到眾多的支線任務，完成這些支線任務可以讓你得到額外的裝備、道具獎勵，使得你在進行主線任務的時候更為輕鬆。不過話雖然這麼說，遊戲的主線任務本身的難度就不高。主線任務的數量不多，而且每個任務中都設置了很多檢查點。 如果你不慎失敗的話，就可以很方便地從最近一個檢查點接續遊戲，而不需要從頭開始。因此，如果你專註於主線任務的話，可以在五六個小時之內通關。而支線任務的設計也遠遠不如GTA系列那樣豐富多彩，而是與主線任務沒有太大的區別，無非是一些追逐、破壞之類的工作，給人千篇一律的感覺。這大概就是該系列比不了GTA的原因之一了吧。

看了以上的介紹，你可能會覺得《正當防衛》是一個毫無新意的遊戲。但實際上，本作中最為與眾不同的部分是降落傘的使用。在遊戲一開始，主角就是跳傘來到了小島上，而這頂降落傘還會在整個冒險過程中陪伴你的左右。如果你勇於嘗試的話，也許在第一次跳傘過程中就已經發現了這頂降落傘的怪異之處——你在空中居然可以自由開閉降落傘，隨時讓自己在懸浮狀態和自由下落中切換。降落傘的減速效果似乎也是”即時發生”，你可以從數千米的高空落下，並且一直不打開傘，保持自由落體的狀態，直到離地面只有幾十米的地方才猛然將傘打開，然後你會驚異萬分的發現自己居然毫髮無損的安全著陸……這降落傘還可以隨意穿越樹枝等障礙物，而不會被掛住——只能說細節決定成敗了。

遊戲開始後不久，你就能得到一把繩槍。有了這個道具，再配合上神奇的降落傘，你就可以體會到在《正當防衛》中最有意思的部分了。和GTA中一樣，玩家可以自由地駕駛能找到的各種交通工具，包括汽車、船、飛機等等。而不同的是在駕駛過程中，主角還可以登上車頂（或者跳出飛機外），然後利用繩槍懸掛住附近的車輛、飛機，然後展開降落傘，來一場爽快的空中滑翔。當然了，你會偶爾發現自己漂浮的方向好像和慣性原理背道而馳，有時候你還會發現自己可以從直升飛機高速旋轉的螺旋槳葉片中橫穿過去而不被切成碎片——如果你能夠容忍這類荒謬的現象，你會發現這種行動方式給你一種頗為自由愜意的感覺，甚至會開始沉迷其中，連任務本身都不是那麼重要了。

除了對物理定律的蔑視之外，《正當防衛》中還存在著一些其它的瑕疵。例如主角跑動的姿勢實在是非常滑稽，給人一種隨時有可能摔倒的感覺。而遊戲中的槍戰部分也過於容易了一點，你只要將準星移動到敵人附近，就可以自動鎖定，這使得大部分戰鬥都變得極其簡單枯燥。當然，遊戲中也還有一些值得讚賞的地方，例如充滿了拉美風味的配樂，雖然曲子的數量不多，但是非常符合遊戲的整體氣氛，讓人印象深刻一可惜的是，除了配樂之外，遊戲中其它的音效和人物的配音水平實在很難讓人滿意。

《正當防衛》不能算是一部大製作的遊戲，其中存在著諸多不合理的部分。不過你也可以從中找到一些亮點，並且體驗到一些奇特的樂趣。雖然我們這期介紹的是初代作品，不過後面的幾代總體上來說也沒什麼太大的變化，如果你感興趣，從哪一代入坑都可以。總之，如果你玩膩了GTA這種精品沙盒遊戲，來《正當防衛》中換換口味也是個不錯的選擇。
