後天11號懷舊服就要開啟戰場了，那就不得不提一下經典的戰歌峽谷了，作為WOW早期三大戰場之一，其高度戰術性和技巧性的結合讓這個戰場成為了WOW和其他遊戲的小型戰場的黃金模板。足以體現出戰歌峽谷的魅力。

不過今天我們從劇情角度分析下戰歌峽谷存在的意義，以便於各位玩家更好的了解魔獸的相關歷史，主要是從部落角度來分析。

最早的灰谷衝突爆發在獸人與暗夜精靈之間，薩爾帶領著獸人氏族到達卡里姆多後，需要大量的物資進行奧格瑞瑪的建設。相比較於資源貧乏的杜隆塔爾，灰谷豐富的木材和礦產資源對於奧格瑞瑪的建設大有裨益。所以格羅姆什·地獄咆哮帶領戰歌氏族在灰谷進行了大量的伐木工作，而這也激怒了致力於維持生物平衡的暗夜精靈們，第一次灰谷戰爭爆發。最終格羅姆什和戰歌氏族喝下瑪諾洛斯之血背叛了薩爾，擊敗了塞納留斯和暗夜精靈。但隨著泰蘭德和薩爾、吉安娜等人在麥迪文的勸說下組成聯盟後，灰谷的戰爭暫時停止下來。

從之前的劇情可以得知，灰谷的資源對於部落來說有著極大的吸引力，卡里姆多大陸資源相當貧瘠，而灰谷卻蘊藏著大量的木材和礦產資源，這對於剛剛經歷過戰火洗禮的部落有著極大的吸引力，而另一方面，灰谷作為暗夜精靈世代的家園，而部落的行為將給這片土地帶來巨大的傷痛，所以銀翼哨兵們不會就這一問題有著絲毫的妥協。

當然最重要的是，聯盟控制的灰谷嚴重威脅了部落的交通命脈，十字路口作為牛頭人和奧格瑞瑪聯繫的唯一路上紐帶，是連接奧格瑞瑪和雷霆崖的唯一交通路線，一旦聯盟突然從灰谷入侵十字路口，雷霆崖和奧格瑞瑪將徹底失去聯繫，而塞拉摩的聯盟部隊可以北上與灰谷的銀翼軍團會合，這樣卡里姆多的部落實力將被徹底清除。因此部落必須保證十字路口萬無一失，與其等待聯盟進攻十字路口，不如搶先一步在灰谷建立前線基地，以便於後續部落軍團在灰谷集結。

對於部落來說，能否在灰谷建立防線對於部落的安危十分重要。雖然灰谷屬於聯盟，但是部落為了生存，對灰谷的問題也將不會有一絲妥協。而如今戰火即將燃起，戰歌峽谷又將掀起血與火的浩劫。
