DOTA2：新加坡世界賽首日匯總 VG找回狀態 茶隊LGD成難兄難弟

2019-12-18 418條評論 1,802次閱讀 497人點贊

12月17日新加坡世界賽第一個比賽日的比賽已經全部結束，中國戰隊整體發揮不佳，其中茶隊1平2負，LGD2負，VG拿到1勝1負的戰績。本文素材來源競技寶，感謝jjbone的數據支持。

先說說A組的茶隊吧，其實一開始能感覺到茶隊第一天的對手都蠻強的，個人預測是1勝1平1負，結果被打成了1平2負。

感覺茶隊最不應該的是和VP的比賽，第一局輸了真的蠻難受的，但是看了看茶隊的比賽輸的也不是沒有道理，前期打出了優勢，但是節奏點的出裝還是中期節奏都給了VP太多喘息的機會。之後茶隊和TNC的兩局比賽感覺第二局比賽有點可惜，下路兩次沖前哨兩次白給，指揮問題真的有點大，同一個地方連著送了兩次。

第二天的比賽茶隊要做的一個是加強對線，一定要儘可能的保證線優，一旦茶隊兩路劣，Sccc再抗抗壓，這局就沒了，所以重中之重——線優。之後第二點就是節奏，既然xxs要刷，那就一定要讓阿銓玩節奏中單，三個大哥裡面一定有人要去找節奏。

12月18日早上10點茶隊小組賽生死戰迎戰J.Strom，這個隊伍真的不厲害，30歲的Fear，DC時候最菜三號位只會玩伐木機的Moo（現在還會玩德魯伊了），希望茶隊好好準備準備吧。

之後說說LGD和VG吧，感覺LGD應該是第一天比賽下來最令我失望的隊伍，一點沒有比賽的狀態。

不論是打Navi還是打EG都很迷，打Navi連著兩局玩單核陣容被Doom關，打EG中期亂沖亂開團。Maybe，Fy幾個人集體不在狀態，希望LGD趕快調整吧。

最後說說VG，VG打EG其實應該是互爆一局的節奏，結果第二局VG醬油看到優勢很大，出裝太浪了，Dy大牛做A杖，Pyw做阿托斯，其實這些裝備換成微光蓮花推推EG後面不太可能翻盤，但是現在說什麼也都晚了。這也算是給VG敲了個警鐘吧，醬油出裝真的要紮實，不能浪呀。

後面和液體的比賽基本就是碾壓了，可以看到VG在BP和戰術上的調整，希望VG可以在第二天的比賽裡面好好打，爭取到一個勝者組的名額吧。

之後再預告一下12月18日的比賽，上午10點A組茶隊打J.Strom，前面也說了出現關鍵戰，贏了還有機會，輸了就可以訂回家的機票了。

中午12點30，B組LGD打液體，現在LGD和液體都是2負，所以說誰能在這場比賽中獲勝，後面晉級的希望就更大。同時VG打紅星老師，紅星老師第一天是2勝的戰績，如果VG能戰勝紅星的話，那麼VG也就有了衝擊勝者組的資本。

希望這幾支中國戰隊好好準備一下吧，爭取來個觸底反彈，殺進小組前四。對於這三支中國隊的發揮你有什麼看法，可以在評論區留言，和大家一起討論，最後求評論，求點贊，求關注，謝謝大家的支持！
