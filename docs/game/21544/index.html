懷舊法傷騎pvp相關 提供給想法傷騎的朋友

2019-12-19 33條評論 1,168次閱讀 63人點贊

先就說為啥提法傷騎，還沒開T2一套，T2一套才是法傷騎巔峰的確，目前有些疲軟，但是馬上開榮譽，你想一套天賦又奶又PVP，只有法傷騎。其他奶職業應該更苦吧。雖然沒開T2，法傷低了點，但是其他職業傷害也低，血也少呀，法師一套T2不是分分鐘秒人。

本人60、70階段一直主玩騎士，而且只玩法傷天賦，80級不知道哪個策劃有問題刪除了法傷高加成，法傷騎消失，就AFK了。最近玩騎士小號逐漸回憶起一些經驗，分享給想玩法傷騎的人。

本人最推薦天賦如下：


天賦相關建議

1、天賦傾向是PVP，所以強化祝福、光環都不會點，王者當然也沒啦，留給其他全職奶騎點吧，盡量通過天賦彌補傷害不足的缺點，又不影響在副本的治療量為目的。

2、為啥不點強化正義聖印

因為強化正義聖印只強化基礎傷害，不強化法傷加成部分，這個技能在70級很厲害，60級真心是垃圾啊，對比下與命令聖印的審判傷害就知道了。命令聖印的好處，可以雙爆，PVP對面看你是奶天賦，很多人以為沒爆發，不注意血線，雙爆就是秒人的途徑之一，審判傷害也比正義高，所以法傷騎士主要輸出聖印是命令聖印。後面法傷加成部分會闡述我的觀點。

3、為啥不點強化十字軍

對我而言，這個技能除了懲戒騎團隊副本輸出就是練武器熟練度了。。。。PVP10秒的存在時間實在太短了，真心還不如審判一個命令來的直接。

4、為啥不點辯護

辯護是個神級天賦，但是是克物理系。法傷騎打物理系都是拿盾牌，已經是所有物理系的爸爸了，包括是懲戒騎的爸爸，不需要辯護來做爺爺，嗯?！如果你想做爺爺就減少三點定罪，點辯護也可以。不過還是推薦滿定罪，打法系可以用雙手武器，暴擊就是加法傷騎最缺的傷害量。(或者減少兩點正義追擊，鞋子附魔移動速度，這是回復中的好建議)

5、為啥建議點以眼還眼

高護甲又是法傷輸出的騎士，到後期真的很怕法師，法師的成長性很大，慢慢就會感受到一個羊逼出無敵的恐怖，只有一個神聖系被反制到了很可能沒有然後了。所以天賦要針對。

數據說話：以眼還眼，反彈暴擊法術傷害的30%。比如對冰法，法術暴擊效果200%。那麼法師對我們冰系暴擊造成200點傷害，就會反彈60點，從血線互相壓制的角度，冰法暴擊只對造成了40%傷害，如果對於其他類型150%暴擊效果的法術，造成150點傷害，就要反彈45點傷害，血線上講暴擊只造成了105%傷害。另外，以眼還眼是可以暴擊的。PVP中，騎士血量一般是大於法師的，而且可以加血，一次戰鬥騎士最多可以滿血四次之多，如果法師當前血量比騎士少的越多，法師越不敢暴擊，有這個天賦會讓對方很糾結。

總結：對於正義和命令的對比，普攻傷害，一個穩定輸出，一個看臉輸出(但命令有每分鐘觸發次數限制，不會黑到一直不觸發)，命令除了受法傷加成，還受到武器傷害的影響，另外正義聖印普攻傷害不能暴擊，所以我認為命令成長性更好，有驚喜，有期待。當然，普攻帶出的命令有一個缺點，可以被招架等。

對審判出去後的加成來說，強化正義聖印實際只強化基礎傷害，並不強化法傷加成部分，參見2.1.0版本騎士部分的更新公告，正義審判是50%，命令審判在未昏迷癱瘓目標是43%，對昏迷癱瘓目標是86%(目前我法傷太低沒實測，若我計算有誤可以指正)。

但是，暴擊倍率兩者不一樣(已實測)。正義審判是法術演算法，暴擊效果為150%，命令審判雖然是法術傷害但是物理演算法，暴擊效果為200%。暴擊時，正義審判法傷加成是75%，命令審判在未昏迷癱瘓目標是86%，對昏迷癱瘓目標是高達172%。

除此之外，命令審判可以觸發武器特效(暗影箭之類)，正義審判是否可以觸發未測試。

所以，建議初期裝備不好可以考慮正義來輸出，裝備越好，命令優勢越大，十分推薦命令。只有用命令才能看到暴擊的大數字，越後期越感受明顯。如果一直用正義，實際上爆發能力和純奶騎天賦差別不大，雖然傷害穩定，但是失去了爆發的可能性，無法體現騎士的第四天賦法傷天賦的優勢。

那麼命令雙倍傷害效果除了制裁還有什麼可以觸發？

目前已知：隊友的昏迷或者癱瘓技能+工程手雷+地精火箭頭盔，所以雙倍機會還是不算少。

專業選擇篇

土豪：鍛造+工程

沒大號的可憐孩子：挖礦+工程

騎士PVP工程必學，工程里很多傷害性，控制性玩具，很多都有副作用，騎士的無敵、自由祝福等技能可以完美化解。

裝備篇

選擇裝備原則：除非是工程學特殊裝備，一定要穿板甲，PVP非常不建議為了堆法傷跨甲。高護甲是法傷騎吃定物理系的根本之一。當然你要有時間做一身混穿火抗，和以後NAXX用的布甲冰抗打法師也可以。。。。。。

屬性選擇：法傷>耐>法術暴擊>智力>5回，其他屬性都是無所謂的啦。不需要命中，沒有神聖抗性，PVP很少抵抗，法術暴擊效果印象中是1.5倍，所以暴擊不是第一屬性。

現階段裝備推薦：

套裝：2件T2，頭腿，其他T1。

單手武器：碧空之歌 搶不過法師可以買愛德華之手，以前沒錢買，沒用過，聽說很好用。其實，最好的武器是風劍，附魔30法傷懷舊服全版本畢業，特效絕對的彌補了騎士傷害不足的問題，而且攻速高，騎士拿PVP威力才大，當然賊、戰士不會認同的，你買不起也當我沒說哈。

盾牌：力反饋盾牌(工程學)真的好用啊，打賊，帶小號，西瘟疫AA刷錢就好用。

雙手武器：隨便在MC拿個垃圾吧，不要浪費錢或者DKP了，或者開戰場拿個奧山的豬頭錘。

項鏈：火焰之王項圈

披風：薩菲隆斗篷

戒指1：鐵爐堡歌唱石(救公主任務) 現階段的戒指感覺都很垃圾啊，要不沒耐力，要不沒法傷，PK不好用呀。

戒指2：屠龍者的徽記(這個不是必須，會導致拿不到龍頭其他裝備，可以暫時用巨石指環或者物理系戒指補普攻傷害)

飾品1：戰場的徽章

飾品2：太多了，法傷騎士的優點就是可以有足夠的操作時間玩這些玩具，畢竟攻擊技能就那幾個嘛。。。根據要殺誰用什麼。推薦：六魔包(好用了，騎士完全可以應對這件裝備的隨機性，是否還受法傷加成暫未測試)、短暫能量護符、煙網蜘蛛的眼球、巴羅夫管家鈴、工程學的各種飾品、鍛造學飾品。

其實黑石深淵有一些法傷裝，但是穿T1奶高，PVP也可以的嘛哈哈。(實在沒時間貼圖。。。。)黑石深淵裝備:披風：靈法披風 鞋子：冒犯之靴(JJC出) 胸甲：巨蛛甲殼(JJC出)，戒指：巨石指環(JJC出)

等黑翼、ZUG出了，可以更新的裝備有

畢業套裝T2一套。

單手武器：洛卡米爾·伊洛曼希斯(碧空仍然好，因為劍帥啊)

盾牌：紅龍防禦者(打賊仍然用力反饋盾牌)

戒指：專註指環

披風：龍王披風

飾品：贊達拉英雄護符

一般都是單手拿盾牌打，打法系如果沒碧空之類的好武器就用雙手武器，上來就神聖震擊+命令審判，然後掛上命令聖印，早點把制裁用了。然後通過無敵、各種裝備、飾品玩具拖到1分鐘制裁CD好。找到敵人血線在一半的時候，並確保這時神聖震擊CD也是好的，然後制裁+命令審判+神恩術+神聖震擊+憤怒之錘斬殺。結束！

一定要注意飾品的使用，比如煙網蜘蛛的眼球防止賊消失、巴羅夫管家鈴防止無敵加血時候敵方繃帶吃喝、工程學冰霜、火焰、暗影反射器就不用說了，看時機開。

一次PVP中騎士最多可以滿血幾次？

對法系：本身滿血+無敵加血+地精火箭頭盔加血+聖療喝大藍=4次

對物理系：本身滿血+無敵加血+有限無敵藥水加血+地精火箭頭盔加血+保護祝福加血+聖療喝大藍=6次
