“吃雞”正式服上線“滑翔機”，詳細坐標公布，建議收藏

2019-12-21 3,588條評論 4,714次閱讀 1305人點贊

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友刺激哥。在寫這篇文章之前，刺激哥剛在端游《絕地求生》體驗過「滑翔機」。

在簡單體驗了幾局以後，刺激哥只想說：「這滑翔機，也太刺激了！」刺激哥在體驗滑翔機的時候，可謂是經歷了各種挫折。

話題：「吃雞」正式服上線「滑翔機」，詳細坐標公布，建議收藏！

廢話不多說，讓我們進入正題。在聽說《絕地求生》上線「滑翔機」的時候，刺激哥還以為是測試服的內容。

經過一番詢查以後，刺激哥才發現「滑翔機」是《絕地求生》正式服的內容。如果大家想體驗滑翔機，記得不要找錯地方喲。

如圖所示，這就是《絕地求生》正式服上線的「滑翔機」了。滑翔機的外觀非常的小巧，一輛滑翔機只要2個座位，這也意味著在四排的情況下，一輛滑翔機完全是不夠玩家們分的。

另外需要提醒的是，《絕地求生》的滑翔機，與《和平精英》的直升機，在設定方面有著很多的不同。

首先，第一處不同，就是乘坐玩家數量的不同了。在印象中，《和平精英》的直升機，最多可以容納8名玩家同時乘坐。

第二處不同，就是傷害規則的設定。在手游《和平精英》中，我們從直升機上直接跳落，會有降落傘。在《絕地求生》中，我們從滑翔機上直接跳下去，則是會直接摔傷。

其次，在手游《和平精英》中，直升機的螺旋槳，並不會對玩家造成實質性的傷害。而《絕地求生》滑翔機的螺旋槳，所造成的傷害，則是可以把玩家淘汰出局。

另外，「滑翔機」的耐久度，幾乎可以說是所有載具中最低的了。如圖所示，刺激哥因為操作不善的原因，導致滑翔機從高空中，墜落到了地面上，結果滑翔機的耐久就變得非常可憐了。

在飛行高速與速度方面，滑翔機的飛翔高度，基本上與《和平精英》的直升機是差不多的。至於速度，則是要比吉普車、摩托車的速度，要略慢一些。

在操作方面，《絕地求生》的「滑翔機」就要比《和平精英》的直升機，要難多了。刺激哥在遊戲中連續嘗試了很多次，都未能成功地掌握滑翔機的精髓。

另外需要提醒的是，滑翔機在水中是不能使用的。一旦滑翔機墜入水中，基本上就算是「報廢」了。

講了那麼多的使用心得，自然是要向大家分享一下「滑翔機」刷新的詳細坐標的。

如圖所示，刺激哥已經在海島與沙漠地圖，為大家標註出了20架滑翔機刷新的詳細坐標。

需要提醒的是，滑翔機的刷新位置是固定刷新的。所以，每局遊戲的競爭都是非常激烈的，希望大家在搶奪滑翔機的時候，提前規劃好路線，並把握好時機。

還有就是，好腦子不如賴筆頭，為了方便大家查到滑翔機的坐標，建議大家可以收藏一下這兩張地圖。

好啦，以上就是《「吃雞」正式服上線「滑翔機」，一共20個詳細坐標，建議收藏！》的遊戲內容了，更多吃雞遊戲話題，還請繼續關註：刺激實戰教室。
