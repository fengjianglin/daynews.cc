好評98%+爽快打擊，其實是帶孝子反抗親爹的故事#這個遊戲好玩嗎#

2019-12-16 735條評論 2,450次閱讀 980人點贊

由人氣遊戲作品《堡壘》和《晶體管》開發商的Super Giant Games開發的roguelike遊戲《哈迪斯》（Hades）於12月11日登錄Steam平台，遊戲一登陸就受到廣泛好評，斬獲98%的好評率。遊戲到底是怎麼好玩，來吃下小編這個安利吧。

由於這是一款砍殺型地下城遊戲，所以劇情並沒有很多。故事背景中，玩家扮演的冥界王子想要逃脫冥界，而來自奧林匹斯山的眾神給予玩家武器和力量，幫助玩家逃脫。

如果你玩過Supergiant眾多經典遊戲中的任意一款，都能在這款遊戲中看到一點他們的影子，先不說熟悉的畫風，遊戲融合了《Bastion》的快節奏，《Transistor》出色的打擊感以及《Pyre》一樣雖然不多但很有趣的劇情。

作為一款roguelike遊戲，玩家在進入地下城戰鬥前就可以選擇一個又眾神之一賜予的技能攜帶，在戰鬥中，每擊敗一個房間的所有怪物，都能得到那個房間給予的獎勵。因為房間給予的獎勵就在進入房間的門口標識著，所以玩家可以選擇合適自己的獎勵，進入對應的房門。

這款遊戲的難度還是有的，但對於手殘玩家來說，遊戲給予了不錯的照顧。遊戲中可以收集水滴（我也不知道叫啥，反正像個水滴），而這些水滴可以用於升級你的技能，或者是提高能力，而只要你不是究極手殘，一般情況下前期遊戲每一把結束都能升級一次。

除此之外，遊戲中還能得到鑰匙，要是可以解鎖強力的技能之外，還可以買到不錯的武器。各式各樣的武器可以搭配不同的神賜技能，加上遊戲中還有商店……這款遊戲的玩法真的可以說是千變萬化了。

除了戰鬥，遊戲中一個重要的場景就是冥界了。作為一個打遍冥界，根本不服自己老子的”帶孝子”，每次死亡回來都要受到自己親爹的一番嘲笑。除此之外，冥界的眾神也會告訴你一些新的故事，聽聽八卦也很有意思。另一方面，遊戲中還可以收集綠水晶（可能是吧，不知道是啥）來解鎖裝飾，一款roguelike地下城遊戲還能裝飾房屋，真的有點意思。

目前遊戲還在搶鮮體驗階段，頁面顯示下次重大更新在2個月之後。作為搶鮮體驗遊戲，遊戲的完成度各方面超出預期，遊戲性極高。唯一不足的是部分漢化機翻明顯，亂七八糟，甚至很多對話還是英語的，看不懂人物對話和物品介紹對於我這種英語殘廢來說極其痛苦。

總體來說，這是一款非常解壓，非常爽快的遊戲。不管是帶感的音樂，華麗的技能特效，還是流暢的操作和真實的打擊感都讓人熱血沸騰，一玩根本停不下來。希望之後的遊戲內容可以更加豐富多彩，要是能聯機就更好了啊！
