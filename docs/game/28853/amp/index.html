“吃雞”相對比較稀有的7套皮膚，全都擁有的是“老玩家”

天天要聞
2019-12-21

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友刺激哥。隨著《和平精英》重新上線了「極寒模式」，很多《刺激戰場》的老玩家，又重新回歸到了《和平精英》。

值得一提的是，這些玩家離開的這款時間，絕對是錯過了很多精彩有趣的內容，同樣也錯過了很多看的皮膚。

話題：「吃雞」相對比較稀有的7套皮膚，全都擁有的是「老玩家」！

在《和平精英》剛上線的時候，遊戲中上線了多款免費的皮膚。而特殊使命頭盔皮膚與特殊使命背包皮膚，便是其中之一。

只不過，很多玩家都不領光子的這份情誼，還是選擇了離開。所以，很多玩家都錯失了這2款免費的皮膚。

然而，這也是《和平精英》上線至今，唯一兩款免費的背包皮膚與頭盔皮膚。雖然，這兩款皮膚，看起來其貌不揚，但也是聊勝於無。

對於不願意氪金的玩家來說，這已經是很不錯的。如果您恰好擁有這2套皮膚，絕對是SS1的老玩家了。

至於SS2賽季的玩家，看到這兩套皮膚，也只能羨慕了。

第三套皮膚，旭日東升大衣，這款套裝是少有的《絕地求生》風格的衣服。另外一件絕地求生風格的短風衣，因為通過幸運空投又重新返場了，所以就沒有旭日東升大衣這麼稀有了。

值得注意的是，這件衣服同樣是一款免費皮膚。在印象中，只要邀請15位好友，並完成相應的任務，就可以獲得這套皮膚了。

第四套皮膚，冰凍風暴套裝。這是《和平精英》剛開放遊戲商店時，軍需寶箱里的衣服。在最初的時候，刺激哥雖然不喜歡這套衣服，但《和平精英》的遊戲美工的審美，勉強還算是比較正常的。

再回過頭看《和平精英》里皮膚，SS1賽季的皮膚，基本上算是巔峰狀態了。

現在所更新的皮膚的風格，最多也就能與SS1持平，想要超越SS1，估計還要很長時間的沉澱。

第五套皮膚，海濱突擊者。在印象中，這件皮膚同樣是一款免費皮膚，只要玩家在遊戲中累計登錄多少天，就可以免費獲得這件皮膚。

至於是5天，還是7天，刺激哥已經記不清了。這款皮膚唯一能擺得上檯面的，也就資歷比較老的。

在這款皮膚剛上線的時候，刺激哥不喜歡這款皮膚。現在，依然還是不喜歡這款皮膚。如果您也有這款皮膚，那肯定是放在倉庫「吸灰」的。

第六套皮膚，SS1戰鬥服。在談論到SS1戰鬥服的時候，有粉絲還在向刺激哥誇讚SS1戰鬥服比較好看。

於是，刺激哥直接把S1-S6賽季的戰鬥服，直接發給了他。在眾多賽季戰鬥服裡邊，SS1戰鬥服顏值，絕對是屬於倒數的那種。

但是，如果您擁有這件皮膚，絕對是SS1賽季的老玩家。

第七套皮膚，榮耀。老實講，刺激哥是完全想不出，光子為什麼要上線這款皮膚的，並且還把這款皮膚命名為「榮耀」。

估計，很多玩家的倉庫里，應該都沒有這件衣服。至少，刺激哥在遊戲力，從來都沒見過，有玩家使用這款衣服。

在這7套皮膚當中，榮耀這套皮膚的顏值，絕對是可以排入前3的。不知道除了以上7套皮膚以外，大家還有哪些比較稀有的皮膚呢？

如果您全都擁有以上7套皮膚，絕對是當吃無愧的SS1老玩家。

好啦，以上就是《「吃雞」相對比較稀有的7套皮膚，全都擁有的是「老玩家」》的遊戲話題了，更多《和平精英》新鮮事，還請繼續關註：刺激實戰教室。
