RNG走A怪又開發新套路，新版本AD也能凱瑞，網友：不當教練可惜了

2019-12-10 0條評論 19次閱讀 0人點贊

摘要

大家都知道，在去年的時候，我們LPL有著三大新生代AD，分別是EDG的iboy，IG的jackeylove，以及RNG的Able。其中Able也就是我們熟悉的「走A怪」，在S7的韓服rank中，他的操作給大家留下了深刻的印象，甚至連faker都直言這個人是不是腳本，而大家了解到Able是RNG的青訓隊員後，也是覺得Uzi後繼有人了。

不過隨著Able在RNG上過幾場比賽後，大家發現Able似乎並不適應LPL的比賽，發揮都比較一般，而隨著他被調到LDL後，大家對他的關注度也是越來越低。值得一提的是，最近這個走A怪又火了，因為他開發了一個全新套路，那就是女警出完無盡和攻速鞋後，直接就買3個暴擊披風，暴擊率直接打到了100%，前中期就有了不俗的傷害。

大家都知道，在當前版本AD的生存率變得很低很低，上中野甚至是輔助都能夠單殺AD。而走A怪這次開發的套路，卻是向大家證明了，這個版本AD也是能凱瑞的，不少試過的網友也是直言太離譜。有趣的是，對於走A的這一套路，網友也是感嘆：不去當教練真的可惜了。

其實走A怪在開發套路上，真的是一個天才。S7大火的香爐套路、ADC帶TP發育的套路、EZ雙女神淚的套路，全部都是由走A怪發明的，並且在賽場上也是大火。對此，大家對於這位走A怪有何看法呢？
