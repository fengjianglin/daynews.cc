「怪獵冰原」PC版如何備戰冰原？這15條Tips會讓你輕鬆很多

2019-12-10 2,789條評論 4,736次閱讀 1133人點贊

前言：距離PC版冰原DLC開放，剛好還有1個月。之前酋長就說過，會在這個時間點給大家帶來一期備戰冰原的專欄。現在也是到時間填坑了。這期可能會有些長，請耐心觀看。

1.護衛隊武器。官方最近也是更新了護衛隊武器系列，這一系列武器強度都非常高，基本可以用到冰原2星任務結束（其實可以一直用到比較往後，如果不怕打得慢的話）。酋長推薦冰原前期可以拿這系列武器進行開荒。

2.武器的準備。雖然說有防衛者系列武器，但是建議大家也不要太把它們當回事，該換的時候還得換。至於換什麼，酋長在【從過渡到畢業武器推薦】系列專欄裡面會有推薦。有興趣的可以去看看。

3.有用的防具。在冰原，基本上大部分上位防具，都只能在前期使用。但唯獨有一件上位防具，可以用到畢業，甚至於版本更新到12.0，還能繼續用，那就是歷戰王炎王龍的【帝王皇冠γ】：

酋長推薦大家盡量地在冰原開之前，把這個頭做出來。

4.其他防具。對於一些只有一個月時間準備的新玩家，在防具的準備上基本兩套可以滿足：騰龍套（來自貝希摩斯）和冥燈γ3件套（手腰腳，來自歷戰王冥燈龍），分別對應近戰武器和遠程武器。

5.護石。在冰原之前，根據不同武器需要，準備下列護石，並升到PC版當前版本滿級：【挑戰護石】，【冰，水，火，雷，龍屬性攻擊強化護石】，【耳塞護石】，【匠護石】，【鐵壁護石】。

上述護石在冰原可以進一步強化，且配裝里會比較常用到（尤其是匠護石），推薦大家都做一下。

6.需要準備的藥物。基本的藥物大家都懂，例如【秘葯】，【強走葯】，【力量種子】，【耐力種子】之類的，反正能加血加輸出加防禦的，推薦大家都準備上一些（開荒時期秘葯要大量）。唯獨需要特意提及的是【漢方葯】：

因為冰原的毒多了一種【猛毒】，一旦中了掉血速度非常快，不解毒滿血都不用10秒就能死。而冰原早期在打【痹毒龍】的時候，這種猛毒就會出現。如果不想冰原的第一貓交給【痹毒龍】，還是準備若干【漢方葯】吧。

7.需要準備的其他道具。關於陷阱什麼的，這些大家都知道要準備，所以酋長就不贅述了。【閃光彈】由於冰原閃光機制改變，所以並不好用，不過有時候帶上一些有奇效；爆彈桶因為冰原有新的開眠手法（怪眠之後鉤爪上頭撞牆），所以基本用的也不多，屬於不用刻意準備的道具。

下面說一個見仁見智的道具——【毒煙霧球】：

因為冰原對於各種異常狀態，都有加強，而且中毒在開荒時期特別好用（傷害湊合，還能加速怪物疲勞），所以如果使用一些非毒屬性武器開荒，可以考慮帶上一些，在開荒期效果不錯的。

另外，如果時間充裕的話，最好也準備一些【投射器異臭彈】，因為冰原怪物亂入概率非常高（大家可以簡單認為每個怪都像爆鱗龍一樣），一場狩獵亂入個2~3次很正常。不過由於冰原可以直接種植，也可以等冰原開之後再種植，會比現在準備要簡單一些

8.任務保留。在冰原，會有不少獵人犯一個比較常見的錯誤——將一些上位和下位的調查任務全部刪掉，用來為M級任務騰位置（因為冰原有了批量刪除的功能，所以酋長發現不少獵人都會這樣做）。

但是酋長推薦某些怪物的調查任務最好保留幾個，例如泥魚龍，眩鳥，搔鳥的上下位調查任務，尤其是泥魚龍的，這東西的鰭素材需求量極大，還不太好出，盡量多留一些調查任務（無論上下位）。

9.關於龍脈石和絢輝龍。如果是新手玩家的話，還是推薦去刷下龍脈石，至少刷到自己常用武器的相關龍脈石，用來客制吸血（最簡單的就是刷一個，然後將防衛隊武器客制個吸血），其他再多的也沒必要了，因為冰原龍脈石基本沒用。至於絢輝龍，不推薦刷了，有時間的話，可以將一些冰原有用武器的前置形態做出來。

10.珠子。可能有些獵人會想著在這一個月的時間裡刷刷珠子，但酋長個人覺得必要性不大，冰原1~3級孔的珠子都很好出。要是缺一些強弓珠之類的必要珠子，可以刷下，如果缺的是攻擊珠什麼的，基本不用刷，冰原不僅4級珠一大堆附帶攻擊珠，達人珠效果（還特別容易出），即便是攻擊珠本身，出的概率也比本體高很多很多，所以不用去刻意刷一些錦上添花的珠子（例如匠珠，攻擊珠之類的）。

另外，如果是單人開荒冰原的話，不屈還是非常推薦的（這個應該人手一個了）：

11.龍骨與礦石。各個地圖的上位龍骨素材和礦石素材，例如【扭曲狂骨】，【珊瑚紅骨】，【堅硬岩骨】，【太古大骨】，【大地結晶】，【龍脈結晶】，【深海結晶】，【獄炎石】等等，最好提前準備一定量，因為冰原裝備會在莫名其妙的地方用上，備上一些以備不時之需，免得到時候要花時間重新採集。

12.採集套。在冰原之前最好準備一套採集套，對主線劇情，甚至是主線結束之後，用處都非常大。雖然冰原會有更好的配裝，但需要比較後期才能做出來，所以酋長推薦一套在那之前使用的配裝：

13.武器選擇。總會有人問酋長，冰原用什麼武器開荒比較好，在這裡酋長統一回答：用什麼武器都行。開荒輕不輕鬆，不在乎武器，在乎人。

但是在冰原前，酋長還是推薦學一個重武器，因為鉤爪軟化上重武器有優勢，在組隊的時候也更為吃香——重武器鉤爪一次軟化，輕武器一般兩次（出某些武器某些招式之外），組隊的時候軟化的工作一般交給重武器。

PS：例如大鎚就很好玩（手動狗頭）~

14.各種族友好度。在冰原，有一個尋寶玩法需要用到各個種族的友好度，本體可以提升到5級（沒記錯的話），推薦大家都把這個友好度提一下，方法也比較簡單，帶上隨從貓，然後到各個地圖招攬各個種族，之後完成任務就可以提升得比較快。

當然，如果嫌麻煩的話也可以不用管，等到冰原再提升友好度也沒問題。

15.你需要群友。冰原在各種改動下，變成一個非常推薦組隊的遊戲。所以酋長建議，如果不是追求獨狼狩獵，還是推薦大家多組隊（尤其是在主線流程結束後）。這時候需要一種東西，那是群友，提前一個月加個群混個臉熟挺好的（也可以加酋長的群，群號在之前發的動態有，這裡就不一一貼了）。
