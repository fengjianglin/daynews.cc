LOL：穩健AP的必然選擇，時光杖的用處到底有幾種，刺客也適用？

2019-12-08 0條評論 1次閱讀 0人點贊

摘要

作為AP發育裝之一，時光杖的選擇率比大天使低，但如果想穩健發育過渡到中後期它卻是一件必然選擇，需要發育的卡牌、秒人的魚人，從傳統消耗流AP到爆發刺客，時光杖的適用範圍很廣，那麼時光杖的優點在哪兒呢？

作為AP發育裝之一，時光杖的選擇率比大天使低，但如果想穩健發育過渡到中後期它卻是一件必然選擇，需要發育的卡牌、秒人的魚人，從傳統消耗流AP到爆發刺客，時光杖的適用範圍很廣，那麼時光杖的優點在哪兒呢？

首先從裝備基本屬性來說，時光杖就是一件優秀的發育裝，300的生命值和法力值，60的法強，以及10分鐘疊加的屬性，總共在面板上能提供500生命值、400法力值和100法強。這是什麼概念呢，比我們常出的鬼書生命值和法強更高，是AP裝里撐血量最適合的一件。

而且時光杖還有一個被動也能提供續航和抗壓能力，變得更「穩健」——永動機之受傷回藍耗藍回血，1100的催化神石比起巨人腰帶或者小左輪槍這些可更為全面，那麼哪些英雄更適合它呢？理論上來說是所有的AP英雄，沒錯就是所有。

說到這兒就不得不提到老牌中單——崔斯特，雖然這個英雄線上並不強勢，還容易被亞索、劫、魚人counter，但他是有著翻盤能力的，被動的額外經濟以及大招，能讓他選擇單帶偷塔或者成型後團戰POKE，而為了彌補前期線上的弱勢，時光自然就成為了必選。

在各類刺客流行的版本，甚至出現了不滅時光卡牌，前期的要訣只有一個字——苟，紅牌+Q推線就站台下，做好視野報MISS，對方清線我大招支援，這才是分奴的樣子。而且時光比金身好的地方在哪兒，金身對AD刺客更有效，時光面對消耗、爆發都可以。

除了卡牌這類需要發育的法師，AP刺客其實也能選擇。例如妖姬、魚人，我就曾經對線過一個時光妖姬，當時我還在想這個B是真的混，玩個妖姬還出時光，真就不崩純躺唄。結果她反手出了殺人書，時光疊滿+本身技能讓她不容易陣亡，殺人書混著混著就25層了，估計對面看我也跟看傻子一樣。

當然時光杖也有著不容忽視的缺點，除了需要發育外，沒有CD這是硬傷，很多AP選擇回聲不僅僅是為了更多的藍，也是為了那20%CD，對於刺客來說CD也是很關鍵的東西，或許這也是時光不被選擇的原因吧。

不過對於某一部分英雄並不需要過多的藍量和CD，皎月就是其一，如今的皎月都是直接納什之牙+巫妖打爆發，需要出的裝備太多了。

歸根結底，拖節奏是它無法避免的缺點。
