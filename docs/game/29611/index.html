魔獸世界：黑鐵矮人一生都被炎魔之王束縛，他們是如何解脫的？

作者：林夫人的貓

還記得各位玩家在黑石塔的時候擊殺了索瑞森大帝，營救了矮人公主茉艾拉的故事嗎？其實矮人公主茉艾拉與黑鐵矮人索瑞森大帝是兩廂情願，兩人已經深深墜入愛河，當玩家們擊殺索瑞森大帝之後，茉艾拉公主會變得非常憤怒，而我們今天要說的故事，就是在索瑞森大帝死後，茉艾拉公主的故事。

茉艾拉強行壓抑了丈夫的死而帶給她的憤怒，大帝的死讓她的氏族陷入了一片混亂，但這樣的局面對茉艾拉來說卻是一個難得的機會，儘管拉格納羅斯手下的士官們對茉艾拉是否會忠於火焰領主抱有疑心，但此時他們都忙於重新控制黑鐵氏族，而無法密切監視她的行為。

茉艾拉充分利用了這一機會，她私下傳出流言，稱黑鐵矮人正在火焰領主的逼迫下組建一支大軍，索瑞森大帝的死延緩了這支軍隊成型的時間，但無法拖太久，隨著時間的推移，拉格納羅斯和他的部隊只會變得更加強大。為了吸引儘可能多的注意，茉艾拉還在流言中指出，在黑石山的深處有著世人無法想像的財富與神器，她以此希望那些勇敢又貪婪的冒險者們可以組建成隊伍，粉碎拉格納羅斯的防禦，將火焰領主放逐回元素位面。

最終結果甚至比茉艾拉自己期望的還要有效，還沒等聯盟和部落針對流言採取措施，海達希亞水元素卻立即開始行動，這些水元素生活在艾薩拉東部的小島上，他們天生就是火元素的死敵，他們願意向任何挑戰拉格納羅斯的人提供幫助與獎賞。

沒過多久，水元素便召集起幾十名勇士，前往火焰領主所在的山脈中心作戰，那裡就是熔火之心。他們小心翼翼地在這座充滿暴烈熔岩的要塞中前進，一路上與最強大的火元素交戰，並將它們一一殺死，在擊倒熔火之心的所有防禦者後，勇士們使用了水元素賜予的禮物，摧毀了拉格納羅斯的防禦符文，於是擋在他們與火焰之王間的屏障消失了，炎魔之王拉格納羅斯出現在了勇士們面前。

雖然它擁有傳奇般的力量，但仍然不敵這些入侵者，在戰鬥失敗後，炎魔之王被放逐回了元素位面，黑鐵矮人最終獲得了自由，而那些曾經懷疑過「茉艾拉」動機的人開始乞求她的寬恕，作為達格蘭索瑞森大帝的遺孀，她一直握有統治黑鐵矮人的權力，而作為他們的解救者，茉艾拉贏得了他們的忠誠，但這場勝利並不意味著黑鐵矮人的日子因此變得輕鬆起來，也並不意味著茉艾拉放棄了丈夫生前征服矮人的夢想。
