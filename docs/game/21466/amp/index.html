游我推薦：動作+酷跑=《刀光與少女》

天天要聞
2019-12-19

「游我推薦」是遊戲頻道推出的一檔全新欄目，每天，這裡都會向你推薦一款值得下載的好遊戲。關注這個欄目，在不知道玩什麼時，可以從這裡尋找靈感。

從遊戲頻道首頁的「游我推薦」，或本文末尾，可查閱欄目過往內容。

自從紅白機時代的《南極大冒險》以來，酷跑遊戲一直是遊戲界的一種特殊但備受青睞的遊戲類型。它們以角色不斷自動前進的設定和路上眾多的障礙，成為反應力出眾的玩家們心中所好。而另外一種類型的遊戲，場景宏大，謎題眾多，需要玩家們觀察環境讓角色在其中攀爬、解謎以及戰鬥，這類遊戲則是諸如《古墓麗影》一樣的動作遊戲。

那麼，如果我們把酷跑遊戲和動作遊戲結合在一起呢？《刀光與少女》給了我們答案。

（此處已添加小程序，請到今日頭條客戶端查看）

戰鬥

《刀光與少女》是由SunnyGame（陽光遊戲）所開發的一款手機酷跑遊戲，但在遊戲中，融入了大量的戰鬥元素。

我們將操控一個外形酷似英雄聯盟中Jinx（金克斯）的瘋狂少女，在不斷地奔跑中一路過關斬將。遊戲非常鼓勵玩家直面敵人硬抗，一旦躲避敵人就會受到懲罰。

因此，整個遊戲玩下來只能用四個字來形容我的心情——酣暢淋漓。

一直砍一直爽

在動作設計上，攻擊、跳躍、橫斬、落地斬等等招式，都可讓人感受到了一種戰鬥的流暢感，加上酷跑本身所帶來的速度感，讓人在短時間內，能把內心積壓的壓力都給釋放出來。

當然，若有時候殺紅了眼，一不小心手滑掉進坑裡也是時常會發生的事情。

砍怪雖然爽，不要手滑哦。

遊戲還設置有特殊的Fever模式，在這個模式下，會讓玩家進入「專心砍怪」的特殊畫面，如果順利通過，還可以將主角回復至滿血，所以一旦進入Fever模式，就放開你的一切神經，瘋狂釋壓吧。

手持妖刀進入水墨風格畫面

裝備與裝扮

在戰鬥之後，玩家將獲得一些物資獎勵：大部分都是女主的變裝道具。

到了我們的角色裝扮頁面，女主角的面容、髮型、武器、衣服都是可以進行自由更改的。

武器方面，每種武器的獲取方式都不太一樣。有些武器是直接用遊戲幣即可購買，而有些武器則屬於免費提供或者說活動提供的：比如妖刀村正，就是連續三次免費試用十五分鐘，就可以免費獲得，遇見這樣的獎勵千萬不要錯過哦。

琳琅滿目的武器等著我去解鎖

而類似鯊魚這樣威武霸氣的武器，則需要滿足攻擊力突破一定程度才可以獲取。

無法控制的大鯊魚！

當然，除了武器，衣服收集元素也必不可少，給了不少變裝愛好者一定的娛樂空間。
