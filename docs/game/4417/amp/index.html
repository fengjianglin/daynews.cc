CFML丨全明星成功衛冕金槍王 他是現役第一AK47選手

天天要聞
2 週 ago

摘要

在剛剛結束的CFML全明星“最強AK47”爭霸賽中，來自於情久戰隊的墨白擊敗各路強敵，成功拿到了“最強”AK47的稱號，同時，這也是他連續第二次獲得該榮譽，成功衛冕，經此一役，墨白捍衛了自己金槍王的榮譽，甚至可以稱其為聯盟第一AK47。

在剛剛結束的CFML全明星「最強AK47」爭霸賽中，來自於情久戰隊的墨白擊敗各路強敵，成功拿到了「最強」AK47的稱號，同時，這也是他連續第二次獲得該榮譽，成功衛冕，經此一役，墨白捍衛了自己金槍王的榮譽，甚至可以稱其為聯盟第一AK47。

在上一屆全明星最強AK47爭霸賽中，當時還在奇蹟戰隊的墨白出戰，在三張個人賽的地圖過後，墨白以20個積分位居個人賽的第三名，在接下來的單挑環節，先是擊敗了實力強勁的蠻蠻，在與總督的巔峰對決中，墨白以10-4擊敗總督，獲得了金槍王的榮譽。

來到本屆全明星賽後，墨白再次出戰，從臨場狀態以及個人實力來看，墨白都比上一屆全明星賽有了很大的進步，上一屆三場個人賽，墨白以20分位居第三，第四和第五名分別是Dude的19分和叮噹貓的18分，是驚險晉級到個人單挑環節中去的。

與上一屆比賽相比，本屆墨白在第一張地圖取得第二名，第二和第三張地圖均為第一名。所以墨白的綜合成績為28分，排名第二和第三的青尤和黑龍分別是15和12分，由此可見墨白的領先優勢是多麼的大，獨佔第一檔的存在，從比賽過程中看，即使墨白前期落後，也能後期發起，將戰績反超，這就是個人能力。

另外一點，就是墨白的單挑實力了，不僅身法運用得當，最令人稱奇的還是他驚人的超高爆頭率，即使在自己殘血的情況下，也能將對手反殺。在最強AK47爭霸賽的最終決戰中，當時比分為9-8，但領先的墨白僅剩3血，如果青尤將其擊殺，雙方就要搶10，在3血的情況下，墨白從邊橋跳下空中將青尤爆頭擊殺，取得了勝利。

賽後，也有很多說最後一槍有很大程度是運氣的成分，不過運氣也是實力的一部分，如果沒有強大的實力支撐，僅憑運氣，是無法獲得金槍王的，這就是實力，墨白以絕對的實力衛冕金槍王，是毫無疑問的現役第一AK47選手。
