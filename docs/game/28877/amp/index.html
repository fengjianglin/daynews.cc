電競圈2019年最令人失望事件

天天要聞
2019-12-21

掐指一算，距離2019年全部結束只剩最後11天了。

在這一年中，作為LOL的玩家們經歷了春夏聯賽、季中賽、洲際賽、S賽以及全明星賽。這些比賽中，我們有感動、有驚喜、有興奮、當然也有失望。

ESPN Esports在其官方推特就以此發起投票活動，來評選2019中最令人失望的事件。

選項分別有：

EchoFox戰隊解散（26%）

暴雪公司遭遇困境（26%）

世界賽上的北美隊伍（30%）

格里芬和Kanavi事件（18%）

投票截止時共有5901位網友參與，最終世界賽上表現慘淡的北美隊伍以30%得票率，當選年度最令人失望「冠軍」。

由此可以看出玩家們對於北美隊伍多年來世界賽成績不佳還是耿耿於懷的。

除此之外，國內的玩家們也對這個投票非常感興趣。只不過ESPN Esports給出的選項太少，不能滿足所有人的想法，於是玩家紛紛寫下自己心目中的【2019年最令人失望事件】

例如，小虎世界賽拉胯2200、WE2.0解散、ig季中賽負於liquid、4AM痛失PGC總冠軍、阿水離開ig等等，都是國內玩家選出的最失望事件。

那麼對於你來說，今年最讓你失望的是什麼？
