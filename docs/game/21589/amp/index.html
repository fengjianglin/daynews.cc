DNF12.13體驗服更新 阿修羅迎來史詩級加強

天天要聞
2019-12-19

　　職業平衡一直是玩家們熱議的話題，在9.24的平衡改版中雖然沒有對阿修羅的技能數據進行調整，但也優化了阿修羅的技能，使阿修羅在實戰中能夠最大程度的復刻打樁的傷害。而在12.13的體驗服更新中，阿修羅迎來了史詩級加強，在保證職業特色的前提下增加角色的傷害，真正成為了「完美職業」。話不多說，接下來讓我們一起看看阿修羅加強的幅度吧。

​

　　這次體驗服的職業平衡調整了阿修羅的波動刻印和心眼兩個技能。原本，波動刻印主要給自己增加大量的釋放速度，以及給隊友增加34%的傷害。

​

　　在職業平衡後，波動刻印額外增加了基本攻擊力和技能攻擊力，1級屬性+0.5%，20級精通，30級上限。這個技能是吃特色的，在普雷畢業的前提下，波動刻印可以額外堆7級(非皮甲無白金)。也就是擁有了20*0.5%+7*1.5%=20.5%的提升。如果是皮甲，則可以將波動刻印堆到滿級30級，也就是擁有了20*0.5%+10*1.5%=25%的提升。

​

　　心眼是阿修羅的一覺被動，在普雷畢業的前提下，非皮甲可以堆到21級。也就是增加了(21-16)*(2%-1.5%)=2.5%的提升，皮甲可以堆到24級，也就是增加了(24-16)*(2%-1.5%)=4%的提升。

　　則阿修羅總計提升了1.205*1.025-1=23.5%(非皮甲)，1.25*1.04-1=30%(皮甲)。而且波動刻印因為調整等級上限的原因，可以省下一百多SP點，所以也會略微提高一絲傷害。

　　接下來我們再說一下阿修羅的畢業白金和時裝上衣選擇的問題。

　　非皮甲

​

　　挫折意志滿級10級可提供33%的基本攻擊和技能攻擊力，10級之後的成長率是2%，普雷畢業的前提下，可以堆到13級也就是39%的基本攻擊和技能攻擊力。波動刻印在改版後可以堆到27級20.5%的基本攻擊和技能攻擊力，成長率是每級1.5%。如果雙白金鑲嵌的是挫折意志，則提升(1.43-1.39)/1.39=2.88%，如果雙白金鑲嵌的是波動刻印，則提升(1.235-1.205)/1.205=2.49%。則可得出阿修羅的畢業輸出白金還是挫折意志。

​

　　對於時裝上衣的技能選擇，目前大多數玩家選擇的是二覺被動雷神之息，雷神之息滿級9級提升24%的基本攻擊和技能攻擊力，如果上衣選擇雷神之息的話，可提升(1.26-1.24)/1.24=1.6%，如果上衣選擇波動刻印的話，可提升(1.22-1.205)/1.205=1.24%，如果上衣選擇心眼的話，可提升(1.35-1.33)/1.33=1.5%，可得出上衣技能選擇依舊是雷神之息最佳。

　　皮甲

　　皮甲的波動刻印是直接堆滿的，所以白金徽章不需要考慮波動刻印，依舊選擇挫折意志。對於每級技能的提升率來說，基礎越高，則提升率越低。也就是說，如果是皮甲的話，時裝上衣選擇挫折意志或波動刻印的提升率會比其他甲略小一點，所以時裝上衣依舊選擇雷神之息。

　　總結：12.13體驗服職業平衡中，阿修羅提升23.5%(非皮甲)，30%(皮甲)。輸出白金依舊選擇挫折意志，時裝上衣也是選擇雷神之息最優。
