峽谷之巔中不缺乏專精一個英雄的玩家，他們對於這一個英雄的理解非常通透，比如最近letme在和姿態雙排的時候，就碰到了一個有著百萬熟練度額卡特，不過和他對線的是姿態，而letme玩的卻是一個打野奇亞娜。

兩人很早就就開始了聯動，在三級的時候，兩個人就集火往上路走。對面的劍姬和己方的瑞茲他的不可開交，眼看就要跑掉了的時候，嚴君澤提前拿好了水元素上去斬獲了劍姬的一血。隨後因為換線的緣故，亞索在上路補線，瑞茲去到姿態的中路，兩人再次抓到了劍姬的一波機會，再次將其擊殺，但是這兩顆人頭都是由letme收下姿態這兩波的遊走確實有一點虧。

也正是這兩波，給到了對面卡特發育的機會，在16分鐘拿下風龍之後，letme趕往下路支援的，但是因為沒有大招的緣故，所以做不了什麼事情。本來以為就這麼算了，但對面發育了很久的卡特卻突然來到下路，直接一個大招加被動將letmeletme給秒殺了。

看到這驚人的傷害，不光觀眾們，letme自己本人都驚了：「這個卡特殺我給殺雞一樣」！本來這句話姿態並不放在心上，畢竟是和自己對線的英雄，看起來並不是那麼的強。但是隨著遊戲的進行，慢慢的。姿態在野區遇見了這個卡特，直接瞬間被融化，隨後才反應過來,對著letme講到：「這是啥啊」！

letme看到了姿態已經陣亡之後，也表示看到沒有，就是這麼的嚇人！於是兩人已經在開始商量對策， 等會打團還屬不要站在一起的好，不然怎麼沒的都不知道。看來對於這個百萬熟練度的卡特，letme還是有些許擔心呀。

隨著對面的滾雪球能力越來越強，letme逐漸陷入了劣勢，不過慢慢的，還是通過奇亞娜野區團戰的威力將整個局勢就賺了回來，看來letme對於這奇亞娜這個英雄的了解還是非常深刻的。

在最後拿了大龍之後letme一路高歌猛進，直逼門牙，最終通過一波漂亮的團戰，贏得了這一局的遊戲，在遊戲結束之後還暗自慶幸：「幸虧對面卡特沒轉起來，不然這一把就沒了」！看來以後對於卡特，letme又有一個心理陰影了呀。
