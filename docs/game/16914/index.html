IGN和Gamestop等滿分神作！最佳Switch遊戲TOP12

2019-12-15 1,073條評論 2,027次閱讀 561人點贊

或許你剛剛買了任天堂Switch、Switch lite，國行版Switch也於前幾日正式發布，而且任天堂自推出Switch後就一直保持穩定的輸送優質的遊戲，那麼問題來了遊戲琳琅滿目不知道玩什麼？近日外媒Kotaku發布了最好玩的12款Switch遊戲，一起來看看吧。

《塞爾達傳說：荒野之息》

《塞爾達傳說：荒野之息》是一項非凡的藝術成就，是一款極富創造力和充滿驚喜的遊戲，在未來幾年它都將會持續的受人關注，而且它也不同於以往的《塞爾達》遊戲。多年來《塞爾達傳說》一直在重新定義遊戲，《荒野之息》是迄今為止最好的《塞爾達》遊戲。

非常適合：所有喜歡遊戲的人。

不適合：不喜歡《塞爾達傳說》的人

《星露穀物語》

如果你有一台Switch，那麼《星露穀物語》就是一款必不可少的遊戲。通過這款2016年的農場生活模擬遊戲，你可以擺脫煩惱，在鄉村過上輕鬆自由的平靜生活。早上起來給農作物澆水，去打理農場。之後到鎮子上去買種子，然後和喜歡人聊天。如果你想，你還可以收集魔法材料，發現更多的秘密。這是一款看似無止盡的遊戲，非常適合便攜設備。

非常適合：喜歡輕鬆又容易上癮的遊戲的玩家。

不太適合：喜歡簡單遊戲的玩家。《星露穀物語》看似平庸而低調，但是他卻非常的複雜。

《任天堂明星大亂斗：特別版》

本作舞台103個，為《任天堂明星大亂斗Wii U》加上 DLC 後的數量兩倍，此外本作中每張地圖斗可容納八人亂斗，並可設置為「戰場」「終點」「大戰場」三種模式，等於說地圖數量超300幅。且玩家可設置戰鬥時雙舞台切換，可能性達300種。登場角色共74(不包含DLC)名，囊括歷代大亂斗角色。本作收錄音樂900首以上，播放時長 28 小時。《任天堂明星大亂斗SP》MC平均分93，IGN 9.4。

比較適合：喜歡競技遊戲的玩家，想要和朋友一起玩以及是大亂斗粉絲的人。

不太適合：討厭和別人起「衝突」的人，喜歡一個人安安靜靜享受遊戲的玩家。

《空洞騎士》

《空洞騎士》是一部微型的史詩，在其龐大的地下王國中塞入了大量的秘密、挑戰和獎勵。它有一點像《惡魔城》和《銀河戰士》，擁有廣闊的地圖和豐富的解謎元素。它又有一點像《黑暗之魂》，強大的boss，充滿捷徑的地圖，而且可能因為死亡而失去同步。但是它絕對值得你去花時間遊玩。

非常適合：喜歡充滿挑戰性的玩家，並且能夠深入進去，紮根於此。

不太適合：容易受挫。《空洞騎士》是一款殘酷無情的遊戲，充滿了挑戰，玩家很容易情緒失控。

《馬里奧賽車8》

馬車8很可能是你在Switch上玩到的最有趣的派對遊戲之一。而且最棒的是，《馬里奧賽車8豪華版》包括Wii U版遊戲中所有的DLC地圖和角色，而且支持Labo體感操作，玩家可使用Labo摩托車操控遊戲中的賽車。

非常適合：喜歡競速的玩家，喜歡路易基的玩家。

不適合：不喜歡《馬車》的玩家（這些玩家存在嗎？）

《星際鎖鏈》

白金工作室以創造敏銳、時尚以及最頂級的動作遊戲而聞名於世，旗下的《獵天使魔女2》更是各大Switch遊戲榜單的常客。《星際鎖鏈》也同樣的優秀，華麗的動作設計和出色的世界環境相結合，創造了與Switch上任何遊戲都不一樣的末世冒險。

非常適合：動漫愛好者，喜歡動作遊戲、探索解密的玩家。

不太適合：不喜歡性感的動畫惡魔警察。

《火焰紋章：風花雪月》

從任天堂時代開始，火紋之紋章系列就一直是玩家們喜愛的遊戲之一，戰棋的遊戲方式總是讓玩家沉醉各職業刷怪升級的喜悅中。《火焰紋章：風花雪月》戰鬥在優化策略性的同時也保留了深度；緊張的時間管理步伐靈活且回報豐厚；沒有相似的角色，玩家可以通過對話了解他們；每次通關都有多樣的內容，保證了新鮮感。

非常適合：任何喜歡動漫和燒腦的邏輯難題的玩家。

不適合：討厭動漫，傷心欲絕，玩遊戲三遍的人。

《傳說之下》

《傳說之下》可能看起來像是復古的JRPG，但它具有超前的思維。作為一個陷入怪物世界的人，你可以決定是肆意的暴力破壞還是基於情景的交際互動。《傳說之下》仍然是一個令人難以置信的溫暖、模糊和有趣的遊戲，無論你是宰殺還是與所有人成為朋友（或走中間路線）。遊戲中的文字都是一流的，配樂也是首屈一指的。

比較適合：顛覆期望的玩家，甚至是感到一絲孤獨的人。

不適合：討厭像素遊戲，不喜歡閱讀人物對白和艱難的BOSS戰的玩家。

《Baba is you》

如果你從未玩過《Baba is you》這樣的遊戲，你以後可能也不會再玩同類型的遊戲了。這是一個「簡單的」推積木的益智遊戲。遊戲中包括你在內的所有事物都會即時隨機定義，你要參破規則找到開門的方法。

適合對象：解決困難的人，因為他們喜歡痛苦，並且更喜歡從痛苦中跳脫出來的感覺。

不適合：何必為難自己呢。

《蔚藍》

《蔚藍》是一個非常難的遊戲，但是它也是如此的溫柔。當你幫助角色跳過山頂的時候，你會發現無論關卡看起來多麼複雜，其根本解決方法都是簡單的跳躍。純粹的設計與微操和迷人的故事相結合，使《蔚藍》成為一場成功、愉悅的體驗，而且音樂也很棒。


《超級馬里奧創作家2》

購買Wii U的理由並不多，《馬造》絕對是其中最重要的理由之一。無論你是想製作自己的馬里奧關卡，還是只是玩無數玩家們創作的關卡。《馬造2》在前作原有的優秀質量上更進一步，它還將多人在線對抗帶到了《馬造》上，這絕對是最有意思的想法了。

非常適合：渴望挑戰最精妙的超級馬里奧遊戲的玩家。

不太適合：想要玩《奧德賽》這種漫長的探索遊戲的馬里奧玩家。

《超級馬里奧：奧德賽》

與傳統馬里奧不同的是，這個戴著紅帽子的管道工不再一直往前走了。本作是睽違15年、繼承《超級馬里奧64》以及《超級陽光馬里奧》高自由度血脈的箱庭探索3D馬里奧正統續作。EDGE，IGN和Gamestop均給出滿分評定，媒體均給出極高的正面評價，綜評網站Metacritic《奧德賽》的綜評為97。

非常適合：箱庭遊戲迷、喜歡《超級馬里奧64》和《超級陽光馬里奧》的玩家、喜歡馬里奧的人。

不適合：討厭3D平台的玩家、討厭帽子的人。
