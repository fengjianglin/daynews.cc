又一個被策劃毀掉的遊戲，改版後再也不碰，本來挺有意思的遊戲！這個遊戲就是雲頂之弈，現在是第二賽季，改版太大了！

昨天5人口白刷一個火男一個安妮，速7開始d，d到經驗值24（十二回合，不知道多少錢了），一個火男和安妮都不來，我有一個妮蔻，就只需要一個火男就鎖血，可惜運氣遊戲。

上個賽季最後一個版本其實挺平衡了，任何陣容都有機會吃雞，相生相剋。吃爛分的話，斗槍騎槍，騎士遊俠，虛空刺，帝國，龍騎士這些都是能玩的。現在這版本真是不行。

雲頂之弈上個賽季，最後一個版本槍斗，虛空斗刺，忍刺，元素惡魔，狂野龍法，約法我都吃過雞，這個版本基本都是煉獄法海吃雞，也就比較胡的極地劇毒，劍刺能打一下，其他陣容吃雞更難，可以改名叫火男之弈了。

斗槍，九劍，帝國騎，狂野刺，虛空刺，動物園，元素，極地遊俠，龍護衛，龍法，約德爾，雖然有強弱之分，但是還是有的玩，現在來到s2，說是平衡，最多也就四五種陣容。

排位5個人開心，5個人失落，1：1的遊戲體驗。下棋1個人開心，7個人罵娘，1：7的體驗，而且運氣好的玩家得意過頭的時候，那場面，反正最後我吃雞了。

這還真是個人品遊戲，奧拉夫狼哥時就刷到8個，最後一個刷光了60金幣都沒一個，心態崩了！雲頂之弈這個賽季越來越複雜了，玩不了，那麼多裝備合成，羈絆記不住，裝備商店都不搞一個誰記得住裝備怎麼合成有什麼功能！
