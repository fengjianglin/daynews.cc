廠長宣布退役！8年老將終將離去，轉身成為主教練

2019-12-19 223條評論 2,159次閱讀 1150人點贊

2012年，第一次認識廠長，那個時候人們更喜歡稱他為諾言，那一年WE奪得LOL比賽冠軍，這是中國隊第一次奪取世界重量級LOL比賽冠軍！「恭喜WE」這句台詞至今成為許多LOL玩家眼中的經典。

2013年WE賽場上發揮欠佳，WE戰隊面臨重組，諾言和捲毛紛紛有意願加入了EDG，作為上單路人王的草莓曾聲嘶力竭的說道：作為兄弟和朋友，求求大家陪他打完G聯賽再離開。

12月16日，僅僅時隔8天，若風就宣布退役了。

2014年2月，捲毛和廠長離開WE戰隊，先後加入了EDG戰隊。

2014年8月28日，僅僅時隔半年廠長轉戰EDG半年，草莓也宣布退役，至此，老WE戰隊算是徹底解散。

轉戰EDG以後，廠長在LOL舞台上大放光彩，曾用男槍拿下五殺的成績，成為了中國第一個兩次奪得世界冠軍的選手。並且也幫助EDG成為取得LOL世界重量級冠軍最多的戰隊之一，比較可惜的是，廠長從未獲得S系列的冠軍。

在2014年夏季賽中，明凱創下了全聯盟76%的最高擊殺率。

2016年4月23日，明凱成為LPL史上唯一一名助攻數突破2500次的選手。

2016年7月30日，在EDG對陣SAT的比賽結束後，明凱的擊殺數達到1001，成為世界上首位擊殺數達到1000的打野。

S6總決賽賽場上，EDG對戰ROX的一場比賽，廠長的盲僧僅僅打出4396的傷害，從此以後4396更是成為廠長的代名詞。並且4396更是成為嘲笑的代名詞。

時隔一年，作為LPL賽區的人氣打野，香鍋經常被用來和廠長作比較，曾經香鍋發揮不好的時候打野傷害僅僅也只有2000多，廠長更是自嘲到一個諾言頂你兩個香鍋，可是人們卻只抓住4396這個梗。

當然作為一個LPL八年的老將，廠長留給了LOL玩家太多世界級精彩的操作，當然也有一些失誤。

皇子EQ閃成功留下uzi，兩段閃現也跑不掉

螳螂閃現留下rookie蛇女，直接讓蛇女空大

廠長放峽谷先鋒被酒桶大招打斷

廠長差不多也有一年沒有更新過微博，上一個動態還停留下2018年11月，他曾說過也許在舞台下，也許更能看得清前方的道路，早在去年就有許多粉絲擔憂廠長會退役。

然而僅僅時隔一年，萬千粉絲擔心的事終究還是發生了，2019年12月16日，廠長宣布退役。

廠長從EDG打野選手，正是擔任EDG英雄聯盟主教練一職，期待廠長在LOL比賽中更多精彩的表現。
