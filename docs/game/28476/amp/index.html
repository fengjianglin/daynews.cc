DNF：神話裝備到底有多厲害？一陣雨親身體驗，破千億很輕鬆

天天要聞
2019-12-23

因為最近DNF的活動比較多的原因，所以很多的玩家都開始對新的活動產生了好奇心，都想要知道這些活動的內容，而基本上現在也被玩家們爆料得差不多了，但是最近一陣雨又出了一個爆料，是關於神話裝備的，相信很多玩家也都對這個裝備產生了好奇心，因為這個裝備目前是超越史詩裝備的存在，所以在很多方面都要比史詩裝備強很多。

根據一陣雨的爆料我們可以知道，神話裝備其實很難曝出來，史詩裝備的幾率原本就要比其他的裝備低，但是這個神話裝備爆出來的幾率比想像中還要低，據一陣雨測試，要3000多次才曝出了一件神話的手鐲。得到了這個手鐲之後，一陣雨就迫不及待的用紅14的普雷手鐲繼承了，雖然按照正常的方式繼承下來了，但是一陣雨發現自己沒有什麼太多的裝備。

所以一陣雨打算隨便搭配上幾個裝備，然後測試一波，看看這個神話手鐲的效果。如果說比史詩裝備厲害的話，在傷害方面應該會有很大的提升，於是一陣雨就裝備上了這個手鐲，然後又加上了兩件的100級的裝備，不過，這兩件裝備都是白板，其實他就是想要裝備一下，然後就開始測試傷害。在裝備好了以後就開始去打樁了，也就是說現在一陣雨的裝備只有一件神話裝備，兩件都是白板的裝備。

但是在一陣雨用這個裝備去打樁之後發現其傷害直接破億了，這也讓大家覺得十分的奇怪，因為僅僅加上了這幾件的裝備就破億了，足以可見神話裝備的厲害。一陣雨這個主播原本就是韓服和國服都能夠玩的主播，在直播中也有很多人的支持和喜愛，關鍵是因為這個主播平時也會給大家帶來更多的爆料，因此也給大家帶來了很多的樂趣。

從中我們也可以看出來這一次所出的這個神話裝備有多給力了，估計一旦上線其傷害整體提升也會很多，關鍵是對於一些土豪玩家來說，一定會將所有的裝備集齊，對於多數人來說，這些裝備一旦使用好了就更加的重要了。當然不僅是這一個傳說裝備，100級版本還有更多的活動和有趣的內容會一一呈現在之後的遊戲中，因此多數玩家也都在期待100級版本的遊戲，這也是大家最關心的問題之一。
