兩開花即近，西遊熱度不減，這些經典西遊遊戲哪個最能代表西遊

2019-12-18 182條評論 4,586次閱讀 2532人點贊

轉眼間，都進入2019年的十二月下半月了，有部文體兩開花的電影也近了呢！相信大家知道老喵說的是哪部電影吧！

《西遊記》真是一個大IP，不管是電影電視劇等影視作品，還是遊戲作品，以《西遊記》為題材改編的還真不少！像今年的《大聖歸來》，雖然嚴重翻車，但還是能說明西遊題材的熱度。

在《西遊記》改編的遊戲中，也有不少經典的遊戲，今天老喵就帶著大家一起看看，有哪些經典的西遊題材遊戲，帶大家領略不一樣的西遊遊戲世界！

1、 《西遊記世界》

FC版《西遊記世界》是一款相當經典的遊戲，這是日本人用中國古代神話小說《西遊記》里的一些人物素材改編設計的，傾向於ACT類型遊戲的ARPG遊戲，這是紅白機少有的ARPG。

其實呢，這是個換皮遊戲，原作名叫《神奇男孩》 ，是世嘉和Westone的聯合街機遊戲，與西遊毫無相干。後來移植FC，因為版權的原因，才強行套上西遊記的劇情和人物。

遊戲的劇情與畫風雖然嚴重不符合中國玩家心目中的西遊記，但是如果無視劇情，把它純粹當作一款遊戲來玩的話，還是有很多閃光點的，也成為了很多人經典的回憶。

2、 《夢幻西遊》

提到西遊的遊戲，很多人應該就會想到《夢幻西遊》，《夢幻西遊》也是網易的經典遊戲代表作品，承載了很多人的網遊回憶。

《夢幻西遊》以《西遊記》為背景，透過Q版的人物，營造出浪漫的網路遊戲風格。嶄新的畫風、風趣的對白、優良的系統設計、謹慎精密的門派平衡、引人入勝的劇情任務貫穿整個遊戲，這在當時的網遊中是非常的新奇，給玩家打開了新世界。

《夢幻西遊》電腦版從2003年12月開始運營，至今註冊玩家人數突破3.6億，最高271萬同時在線，可以說是國內網遊的巔峰作品之一！

3、 《非常英雄》

《非常英雄（Unruly Hero）》是由法國 Magic Design Studios 遊戲團隊研發，完美世界投資的橫向捲軸遊戲。《非常英雄》以繪本一般的遊戲畫面、活靈活現的角色、打擊感和生動的音樂，為玩家帶來美妙而難忘的西遊體驗。

遊戲包含格鬥、解密、收集、跑酷、平台跳躍等豐富遊戲元素……師徒四人一路斬妖除魔，不斷前行。在某些關卡中，玩家還可以跟大聖一樣鑽進妖怪身體內進而控制妖怪進行過關，它們有著完全不同的技能，可以用它們玩到完全不一樣的過關體驗。

《非常英雄》從整體完成度上看，說是比較出色的《西遊記》遊戲也不為過，眾多用心的設計和有趣的細節，都彰顯著某種不凡的追求。

4、 《奴役：奧德賽西遊》

《奴役：奧德賽西遊》的遊戲背景雖改編自《西遊記》，不過設定卻在150年之後的未來，給我們講述的是未來世界的西遊記。

設定為第三次世界大戰爆發後，人類為了打敗對方而製造機器人。大範圍的衝突使得人類瀕臨滅絕，而大戰期間製造已經失去戰鬥意義的機機械士兵卻還在繼續獵殺倖存者。

遊戲里玩家可以操作的只有猴子，猴子沒有72變，有筋斗雲但是飛不高。還有一個一直跟著玩家的女Al（以三藏為原型），以八戒為原型的人物是打槍的，而沙僧和白龍馬並沒有加入該作品。

每個人小時候都有個西遊夢，不同人心中也有不同的西遊理解，就像那各式各樣的西遊遊戲，不知道在小夥伴心中，哪個西遊遊戲最能代表西遊遊戲的經典？歡迎一起分享哦！
