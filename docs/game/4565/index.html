12月已近年底，各種總結盤點都爭相發出，福布斯今年的各類榜單也已經出爐，無論是財富還是影響力的排行，都能為大家津津樂道很久，今天小編就帶大家來看看福布斯U30系列中的”遊戲界30歲一下傑出青年排行榜”，來了解一下遊戲界中的”精英才俊”們，其中不乏眾多玩家十分熟悉的遊戲工作室和戰隊等。

Brent Batas和Julian Gari，這兩位好朋友是AutoAttack Games工作室的聯合創始人，也做了一些受歡迎的遊戲，兩個人曾在高中時期就創作了《軍團TD》，這也是《魔獸爭霸3》目前為止最受玩家歡迎的Mod之一，可以說遊戲能力很早就顯露出來了。

Maxx Bruman，現年29歲，是KitBash3d的聯合創始人，而他聯合創立的KitBash3d為眾多的群體提供豐富多樣的藝術資源，為遍布世界75個國家的眾多藝術家提供藝術支持，其中就有不少著名的遊戲公司，例如育碧、EA和華納等等。

Justin Britch，現年27歲，是黑曜石遊戲開發公司的開發部部長，在13歲的時候就進入黑曜石實習，一路到現在的開發部長的位置，《永恆之柱2：死火》是他曾經主導開發的遊戲。

Joseph Bentley，27歲時任羅技超越娛樂部的主管，曾經是一名電競視頻和新聞撰稿人，之後個人主辦了Beyond Entertainment線上活動平台，因其表現突出被羅技收購。

Nicole LaPointe Jameson，現在25歲任職Evil Geniuses戰隊的美女CEO。今年五月的時候她接手了這支資歷長但暮色漸沉的俱樂部EG，接手之後就馬上把戰隊帶回了《CS:GO》和《英雄聯盟》兩個世界級的頂級電競聯賽，之後戰隊在紐約戰中奪冠一舉成名，她本人也被賦予了更多的期待。

Kezia Adamo，是SudioMDHR工作室的程序員，優秀的女性程序設計員，她本人之前也參與了工作室的經典作品《茶杯頭》編輯器的開發工作，負責了部分的編程內容。

Bonnie Burton，顏值與才華兼具的女性遊戲製作人，2003年在12歲的時候，她以其出色表現成為職業玩家，是美國Major League Gaming聯賽的首位職業女玩家，之後從賽場到遊戲開發，一路前行，現在任職Bungie，主導《命運》系列遊戲製作工作。
