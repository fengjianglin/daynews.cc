冒險島2去哪了？玩家記住的除了二，就只剩光速涼涼了

天天要聞
1 天 ago

縱觀遊戲風雲十幾載，好玩良心的單機遊戲有不少，可惜天不遂人願，種種因素讓單機遊戲的黃金年代轉變為網遊時代。無論老玩家還是新玩家，接觸到的網路遊戲多不勝數。要問誰涼的最快？可能玩家們一時間難以回答，但要問誰涼的最快而且測試得最久？那答案就顯而易見了，它就是咱們今天的主角——《冒險島2》

在一個陽光明媚的午後，Nexon宣布製作《冒險島》的續作《冒險島2》，玩家一看瞬間激情澎湃，對《冒險島2》是左盼右盼，就等著重回青春時代了。反正等著也是等著，玩家們開始盲猜誰會代理，但是說來說去，結論只有一個，別落在盛大手裡就行了，最好騰訊也別碰。

結果你看巧不巧，還真就讓騰訊代理了，這打臉打的真是疼。於是2015年騰訊宣布代理《冒險島2》，雖然玩家渾身難受，但也沒招，行吧，反正遊戲外掛是放心了，大不了再氪他個天荒地老。

2014年，韓服首先開啟測試，第二年迎來公測，這速度不服不行。國服慢了不少，2016年才開始首測，這一測就測到了2018年才以迎來公測，就問你見過測了這麼久的遊戲嗎？那麼為啥《冒險島2》測了這麼長時間呢？

根據民間說法，原因如下：

1：本地化難度頗大

每一個海外遊戲光臨神州大地都需要翻譯，還得修改一堆數據，工作量絕對不小。不過仔細一想事情貌似並不簡單。騰訊2015年代理的《冒險島2》，在2016年迎來首測，這前前後後也就幾個月，說明本地化工作應該做得差不多了，這首測到公測2年的時間都去哪了？修bug去了？反正我不信。

2：大型版本更新

民間還有說法，據說是韓服進行了大規模的版本更新，國服之前的活還沒幹完，一下子有來個大活，所以遊戲跳票2年。說的倒像是真事，轉頭咱們看看騰訊從代理《絕地求生》到手游版的上線，也就幾個月時間。騰訊做個遊戲都這麼速度，改個補丁就得2年？這上哪說理去？

3：不可抗力因素

據不可靠的小道消息稱，國服《冒險島2》早就搞完了，但是神奇的外星文明看上了這款遊戲，將遊戲代碼偷走了，於是騰訊不得已只能把已經完成的工作再干一遍。咱們不說這麼玄幻的事情到底會不會發生，但即使代碼真的丟失，騰訊能沒有備份？

4：賣激活碼變相賺錢

這個可能性我根本不信，我堂堂大騰訊，用心製造了多少快樂，在乎你這倆臭錢嗎？在乎！但就算利用激活碼來賺錢，這手段也太低級了，我們大騰訊不都是讓你氪的心安理得嗎？這有碼就可以為所欲為的作風可不像騰訊。

說了半天，發現案件的難度極高，估計只能請求柯南來破案了。咱們跳過這個問題，來說說不刪檔測試吧。

玩家們興高采烈的進入了遊戲，發現即使遊戲從2D變為3D，但顏色飽滿，音樂也讓玩家想起了自己在夕陽下奔跑，那逝去的青春。本以為憋的越久尿得越爽，結果玩了一陣後，發現這玩意把我的情懷摔在地上瘋狂摩擦。

遊戲一步一充值，咱們可以無視，但真煩人。調低爆率暗改數據，這就有點忍不了了，遊戲中的副本十分不友好，不光能打得玩家哭爹喊年，爆出來的道具也讓玩家哭爹喊娘，平民玩家拿不到裝備，天天拿根白色品質的木棒，自己都嫌丟人。

終於在重重壓力下，大量玩家自愧不如，紛紛退游回歸平凡的生活。你問這些玩家為啥退游，理由多種多樣，但你看他們離去的背影倒非常統一，都有一種失落感。

玩家們熱愛《冒險島》是因為情懷，他們追求的僅僅是簡簡單單的快樂。玩家們本想重新體驗那個純粹的遊戲世界，結果呈現在眼前的一切簡直比現實還現實。

很多玩家覺得《冒險島2》本身就不咋地，你看韓服不照樣涼涼。雖然這個說法比較正確，但最重要的可能還是前作太過經典，《冒險島2》沒有達到心裡的預期。很多因素的改變，讓玩家覺得這遊戲根本就不是我心中的《冒險島》，雖然《冒險島》里有的它都有，沒有的它也有，但是它就是爛。

都說是策劃毀了《冒險島2》，但仔細想想，玩家也有一份。如果玩家都不買激活碼，遊戲會測試這麼久嗎？如果沒有金團，工作室還能這麼囂張嗎？還真沒準！

可能我們的《冒險島》之旅已經迎來尾聲，即使它不那麼完美，即使《冒險島2》就是一個笑話，但能當一個反面教材也不錯了。
