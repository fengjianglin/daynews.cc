EDG官宣大名單：三名打野接班廠長，沒有JackeyLove

2019-12-17 1,897條評論 2,889次閱讀 1784人點贊

這裡有最新最全的英雄聯盟資訊，菌菌帶你看不一樣的召喚師峽谷~

在轉會期最後一天，LPL賽區的戰隊可以說是人均憋氣大賽冠軍，這次的冬季轉會期LPL賽區的變動很大，前世界冠軍輔助Mata加入RNG擔任主教練，EDG打野Clearlove轉型教練，帶領EDG征戰2020賽季，OMG戰隊首先憋氣失敗，率先公布了戰隊大名單，不愧是星球之光，Smlz的回歸讓OMG更有底氣了，而EDG在冬季轉會期結束的第一天，也是公布了戰隊大名單，三名打野接班廠長，中下輔沒有替補，也沒有JackeyLove選手。

EDG在中午十二點整公布了戰德瑪西亞杯出征陣容以及轉會公告，教練組：Clearlove、Heart、Corgi、Bibra，上單Jinoo、Aodi、xiaoxiang，打野：Jiejie、Xinyi、JunJia，中單：Scout，AD：Hope，輔助：Meiko，我們可以看到上單Aodi依舊在，首發應該還是Jinoo，打野選擇把S9世界冠軍替補打野Xinyi納入麾下，而中下輔都只有一名隊員，三人都是首發，沒有替補，這套整容很多粉絲都表示接受不了，至少在打野和AD位接受不了。

在公布大名單之前，EDG戰隊就宣布了隊內的iboy轉會至VG戰隊，這樣VG在缺少了泡芙和南風的情況下還不算虧，因為iboy在出道就被粉絲們稱為國內三大AD之一，個人實力很強，也是一位十分年輕的選手，年齡就佔了很大的優勢，不過在今年來iboy的實力有所下滑，有消息表示，iboy在考核中沒有EDG現在的首發Hope好，所以也是半賣半送地交易給VG，所以Hope也是坐穩了首發。

同樣也是在轉會期最後一天，IG宣布原VG戰隊南風和泡芙的加入，就有粉絲在底下詢問JackeyLove的去向，IG官博表示目前阿水還是自由人，有相關後續結果會通知大家，大家都在JKL去了SN或者EDG，因為SN的新上單很強，並且Smlz剛剛離隊，而EDG目前也需要一名實力強勁的AD選手，在EDG官宣大名單之後，阿水並沒有出現在大名單當中，也是讓很多EDG粉絲失望！

對於這份大名單，菌菌認為這套陣容配不上教練Clearlove，可能這屆會是廠子最難帶的一屆了，假如這套陣容能打出好成績，說明廠子的調教能力還是強，具體情況還得看德瑪西亞杯，期待廠子首次擔任主教練的發揮。
