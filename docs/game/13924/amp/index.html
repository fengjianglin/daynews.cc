夢幻西遊：目前CBG到底有多安全？陳無敵曾親自驗證，還真不賴

天天要聞
2019-12-12

如果在十幾年前甚至是五年前，你還在吹夢幻這個遊戲有多好玩之類的，那麼小明認為你還是一個比較理智的人，但是如果要是你現在還在說夢幻如何好玩的話，那麼小明完全可以認為你是想騙人入坑然後賣號，對於當前的遊戲環境來說，夢幻真的是成為了圈內人想出去，圈外人不明白還不敢進來的程度，當然有些忠於自己的玩家是不會考慮這些的，畢竟遊戲大環境真的不是我們一般人能夠改變的！

在夢幻當中很多極品裝備的價格都是居高不下的，儘管裝備的整體價格都是在下降，但是說實在的，那些極品裝備的價格還真的是沒有降多少，那麼最近有位玩家也是展示出了儘管等級低，但是依舊是高價格的武器，首先就是這把暗夜，這把武器的傷害其實還是一般的，但是因為無等和這把武器的造型，讓這把槍還是成為了上萬R的存在。

而這把天龍破城則更是一個有故事的武器造型了，因為對於這把武器來說，在當初的時候，就是第一把千傷武器的造型，一直到現在也是很多極品武器造型的代表，那麼這把150級的武器，屬性上面更是不用我們多提了，只要是有無等再加上這個造型的話，想便宜小明估計都很難，所以說這把武器估計在未來貶值的可能性也是很低的！

我們其實完全可以從夢幻當中的大事件當中，來評判一下夢幻這個遊戲內部的安全性，小明也是簡單的集合了一下夢幻最近發生的一些事情，那麼在這些事情裡面，最終還是回到了幾個月前陳無敵的那件事，他和晚秋的事情已經過去那麼久了，小明也不想多報道了，但是從他們這件事當中還是看出了一個問題，就是我們眼中的CBG現在到底安全不安全呢？

從陳無敵申訴找回的三件裝備來看，儘管陳無敵申請找回，但是最後並沒有回到陳無敵的手上，之後出現的凍結這種情況，而賣家最後又強硬的通過去WY總部出示了自己購買裝備的證據，拿回了這三件極品裝備，可以說在WY並不知情的情況下，凍結裝備真的是一個比較明智的選擇，所以不得不說CBG是夢幻的一大特色是沒有道理的，還是提醒大家正規交易還是走CBG吧，不要嘗試走線下了！
