王者：當你的本命英雄沒有手感時，這5個英雄在路人局絕對是首選

2019-12-12 431條評論 2,038次閱讀 703人點贊

我們都知道，在玩排位的時候，拿出我們最有把握的英雄是最穩定的，因為我們對這個英雄足夠了解。但是有時我們也會發現，打到一定的時候，會覺得我們的本命英雄越玩越菜，感覺到各種打不出優勢。當我們有這種感覺了之後，那就意味著我們需要調節一下，換一個英雄，因為玩同一個英雄太久，也會出現疲勞。那麼當前版本在路人局裡五個位置的強勢英雄都是誰呢？只要會這些英雄，上分也不是難事。

發育路：公孫離

因為現在是射手對線射手，所以有一些英雄會有天然的優勢，比如說虞姬，他有免疫物理傷害的二技能，在對拼中有天然的優勢，但是虞姬前期傷害不高導致她並不強勢。那麼這個版本值得練習的射手就是公孫離了，首先公孫離並沒有大家想像的那麼難，我們只要知道公孫離是要依靠被動輸出就可以了，因為沒次釋放和回收傘之後的普攻是可以額外附帶一次普攻，所以阿離每次放傘或者收傘在加上兩次普攻都可以打出一次被動，理想情況是瞬間三個技能可以打出6次被動，傷害非常爆炸，同時阿離也很難被抓，一個人走發育路可以很好的保證自己的安全。

對抗路：孫策

或許大家覺得現在對抗路曜相對比較厲害，但是曜想要打出理想的效果還是需要一點操作的，在別人手上或許技能根本無CD，但是在一個不熟練的人的手上根本是打不出這個效果的。我們玩對抗路要有明白一個道理，支援比什麼都重要，而孫策剛好是這種支援相當強的英雄，只要有大招就可以直接支援了，對面根本無法剋制，只要拿出孫策，打架的時候，可以保證己方永遠比對面多一個人，這樣隊友的優勢自然就有了，想贏也不是難事。

打野：雲中君

很多人覺得打野像是李白，猴子，盤古才是強勢英雄，但是我們在路人局中，像是李白猴子並不太強勢，盤古又需要配合才能起來。所以打野我推薦一位低調而又厲害的雲中君，前期反野無敵，只要成功，甚至可以四buff開局，一旦對面打野崩了，這局基本上也就穩了。

輔助：太一，張飛

這兩個輔助英雄在巔峰賽都是非ban必選的輔助英雄，相比那些孫臏，牛魔，這兩個英雄更加煩人，太一有控制，有復活，不管是保人還是進攻都非常厲害。張飛我們都知道，是專業保後排的輔助，一旦有一個會玩的張飛保護，射手基本上就可以亂殺了，因為那些刺客戰士根本無法殺死你。

中路：小喬

小喬可以算是中路萬金油英雄了，打誰都可以，可防守，可進攻，遊走支援速度很快，後期傷害非常高，一扇子能夠直接打掉脆脆皮三分之二的血量，打團AOE傷害也非常高，而且操作也不難，非常適合作為我們非專業中單玩家的上手英雄。
