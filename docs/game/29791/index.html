RW俠爆冷輸給CW，親手送EStarpro晉級

2019-12-24 3,998條評論 4,927次閱讀 952人點贊

王者榮耀冬季冠軍杯選拔賽正火熱進行中，出線隊伍也慢慢浮出水面，榜首的TS以及XQ有很大機會晉級。

但，比賽就是比賽，充滿懸念。在23日由RW俠對戰CW的關鍵比賽時，出現了爆冷對局。

開局前，所有人都看好RW俠，畢竟KPL隊伍肯定是要比KRKPL隊伍更強一點的，況且CW這支隊伍目前一勝難求。

令人沒想到的是，CW打的夠凶，兩分鐘的暴君團打出優勢，並且不斷擴大自己的優勢，最終居然擊敗了RW俠。

這一場的爆冷出乎所有人的意料，本來已經幾乎無法晉級的EStarpro突然形勢大好，而XQ卻出現危機。

因為RW俠輸給了CW，於是沒有機會爭取晉級名額，而EStarpro因為RW俠輸了，在它接下來對戰EMC跟RW俠兩場比賽中，只要贏得一場，積分便是5-2與XQ打平，但凈勝局卻會比XQ多一局，這便可以反超XQ然後晉級。

EStarpro 即使打不過RW俠，也是絕對可以戰勝EMC的，畢竟實力擺在那裡。

不得不說，比賽真是瞬息萬變，本以為淘汰的EStarpro卻因為一場比賽成為最大優勢方，本以為可以晉級的XQ卻成為及其危險的一方。

對於RW俠爆冷輸給此前一場未勝的KRKPL隊伍CW，大家有什麼看法呢？評論區討論討論吧。
