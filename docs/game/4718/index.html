食物語即將推出百日慶典限時活動，那麼這個活動該怎麼玩呢？下面就讓小編給大家帶來詳細的活動內容介紹吧！

百日慶典活動內容一覽

1.百日慶典活動召喚陣限時開啟：12月9日10:00至12月9日23:59,「佛跳牆」慶典召喚陣限時開啟；

12月10日10:00至12月10日23:59，「八仙過海鬧羅漢」慶典召喚陣限時開啟；

12月11日10:00至12月11日23:59，「吉利蝦」慶典召喚陣限時開啟。

「佛跳牆」「八仙過海鬧羅漢」「吉利蝦」將按期各自依次進入慶典活動召喚陣，對應召喚陣中食魂概率將限時提升，召喚概率占御品食魂總概率的50%!

2.12月11日當天登錄的少主，即可在郵箱中領取百日慶典專屬賀禮信件。此信件僅於當天有效，還請少主及時領取；

3.登錄成就升級，累計登錄達到100天/200天/365天，均可領取成就專屬特效頭像框及「調料」等獎勵；

4.2019年12月9日5:00至12月12日23:59，百日慶典限時商店開啟，金玉購買活動商品僅需半價；

5.百日慶典專屬直購禮包上架商店；

6.空桑【通訊】中新增了百日慶典專屬內容；

7.開服至今累計登錄滿100天的少主，將會在達成百日登錄的當天收到一封百日專屬紀念信件。

以上就是小編給大家帶來食物語中百日慶典限時活動的具體玩法了！
