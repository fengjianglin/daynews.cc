三國志11移動版，火計教學，當火神遇到藤甲，就像孜然遇到牛肉

2019-12-10 2,132條評論 3,382次閱讀 1507人點贊

笑談風雲，一語定乾坤。

大家好，我是星仔醬。今天星仔請到了幾位特殊人士，來和我們講解一下三國志11移動版中該如何正確玩轉火計。首先是縱火大隊長周公瑾，其次是縱火隊副隊長黃蓋，還有一位是新加入縱火隊什麼都不知道的後生仔星仔。

火分類

  • 計謀火，計謀中火計、落雷。
  • 戰法火，弓兵戰法、井闌戰法、船戰法。
  • 陷阱火，火焰種、火焰球、火焰船。
  • 假火，由攻城兵器火獸噴出的火焰。

這些火各自都有些區別，前面三種是真火，除了首回合會造成傷害，再接下來回合還會造成延燒的效果。而最後一種假火首回合沒有傷害，只有後續延燒的效果。業火種在研究了火科技第四層後，擁有了炸暈和炸死兩種效果。

火計的範圍

我們先看看計謀火有些什麼樣的發揮。如果沒有研究神火計科技的話，火計的初始計略範圍是1。當研究完神火計後，火計的初始範圍是3。落雷的計略範圍是3，研究神火計對落雷無效。當然，如果擁有鬼謀的話，火計的範圍直接擴大到4，還是相當實用的。

火計的成功率

火計的施展範圍很廣，可以對空地、部隊、城牆、壩、火陷阱、落石等地方使用。大部分場景下火計成功率是和自部隊智力成正比，小部分場景則受目標影響。一般來說，空地火計成功率要高於濕地、淺灘等濕氣很大的地方。對火陷阱施展火計則必成功。當擁有特技【火神】、【火攻】、【虛實】、【神算】，並且智力大於敵部隊智力的話，火計必成功。當智力低於敵部隊的話，火計成功率正常計算。同時也受【傾國】影響哦。如果目標隊伍有【洞察】特技，那麼火計成功率為0。

火計的傷害

火計的傷害倍率只有三種，一倍、一點五倍和兩倍。一般燒兵的話在四百左右，火神加持下能到八百多，火攻加持下是六百多。還有一種特殊情況，就是【藤甲】。當【藤甲】遇到【火神】，那真是一個慘字。星仔曾用周瑜對兀突骨使用業火球，一下造成七點多點傷害，簡直無情啊！

今天的火計課堂就先到這裡啦，下次我們講一講其他幾種火。如果你還知道其他火計的妙用，不妨留言一起討論一下哦~

漢化進度

菜單基本完成，現在正對特技進行漢化。
