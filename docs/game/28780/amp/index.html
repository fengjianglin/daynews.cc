2019冬冠杯即將開始 hero久誠首發 KPL隊伍能否拿下剩餘席位？

天天要聞
2019-12-21

大家好，我是七七。2019KPL秋季賽已經圓滿落下帷幕，恭喜AG超玩會終於拿下了夢寐以求的銀龍杯！秋季賽結束後，就要進入更為緊張的冬冠杯了。秋季賽我們面對的都是自己人，而冬冠杯則要面對來自全國各個賽區的選手。

2019KPL秋季賽和KRKPL秋季賽的前三名將會直接晉級，KPL的4-8名以及KRKPL的4-6名將會進入選拔賽，共同爭奪剩餘的兩個席位。這一次的冬冠杯選拔賽將採用BO1積分制。一共八支隊伍，每天十場比賽，持續三天，積分最靠前的兩支隊伍將贏得進入2019冬冠杯的資格。

在以往的印象中，KR賽區的隊伍向來都是比不上KPL賽區的隊伍，但最近有消息透露，在某次聯賽中，KR的幾支隊伍已經可以從KPL賽區的隊伍中拿下一兩場比賽。如果選拔賽是BO5賽制，我們還可以全盤壓KPL賽區的隊伍，但BO1的賽制，誰也不知道最後會不會有黑馬闖出來，而且近年來KR賽區的隊伍實力確實是有不小的長進。

作為上一任世冠杯的冠軍，eStarPro這次遺憾進入了冬冠選拔賽，將要與其他七支隊伍共同爭奪這兩個席位。eStarPro的實力放在這八支隊伍中，可以說是數一數二，但能否在BO1的賽制中脫穎而出，誰也壓不準。

秋季賽一直坐冷板凳的久誠這次冬冠終於被安排首發上場，想當初，久誠的巔峰時期，大部分玩家都覺得hero戰隊完全是被久誠踢進了三冠席位，不是貶低其他人，只是那時候的久誠實在是太亮眼了，這也導致秋季賽久誠幾乎坐了一個賽季的冷板凳之後，引起了許多粉絲的強烈不滿，不知道這次選拔賽久誠還會不會繼續給我們驚喜呢？

七七出品，必屬新品。以上就是本期王者觀點，大家覺得這次冬冠哪兩支隊伍可以拿得最後的席位呢？KPL賽區的隊伍能否將這兩個席位全盤收入囊中呢？
