《塞爾達傳說》續作能否超越前作？玩家表示這幾點一定要改

2019-12-24 1,450條評論 2,619次閱讀 432人點贊

在玩家之間流傳著一句話「能超越塞爾達的只有塞爾達」。《荒野之息》作為《塞爾達傳說》系列新的起點，一來就被捧上了神作的巔峰，而續作從官宣開始，就註定備受矚目。等未來續作發售後，它最大的對手就是《荒野之息》。

《荒野之息》雖被奉為神作，可它也不是完美的，很多玩家都覺得他還能做得更好。而那些玩家們希望得到改進的地方，是否會在續作中實現呢？

更多種類的怪物

在《荒野之息》中，怪物的種類不多，唯一有點挑戰性的就是人馬。可是只要玩家掌握了人馬的打法，就會很輕鬆，到了遊戲後期足以隨便亂虐它，這樣一來就失去了打怪的樂趣。如果續作中能增加怪物的種類，並且適當提高難度，一定會增加不少可玩性！

取消或增加武器耐久

在遊戲中最讓人煩惱的事情大概就是，打怪打得正爽，武器卻爆了。《荒野之息》武器的耐久值消耗太低，以及不可修復（除了大師之劍和英傑武器）這一點，一直備受爭議。有時候打一個精英怪就要爆掉幾把武器，實在讓人心疼。

所以很多玩家希望可以直接取消耐久值，或者增加武器耐久，再出一個武器修復的功能。小編覺得吧，取消武器耐久是不太可能了，後面兩個老任倒是可以考慮一下！

家園系統

還記得在《荒野之息》里買房的時候，真的很激動！想著要好好裝飾一番，畢竟周圍的樣板房都裝修得不錯。不過買完房才發現是自帶裝修，玩家只能掛幾個武器的展示框上去，除了睡覺也沒有其他的交互，確實讓人有點小失落。

而且房子明明是個帶花園帶池塘的小獨棟，卻不能像村民的花園一樣種植。玩家都希望能夠出種植系統，畢竟從村民菜地里摘蔬菜是要付費的。

潛水系統以及海底地圖

很多玩家都希望在續作里可以加入潛入海底的劇情，並且增加一個海底世界可供玩家探索，在《荒野之息》里見過了各種各樣的風景，唯獨沒見過海底世界！

對於這一點，小編有話說！《荒野之息》里真的存在海底世界！玩家只要去到「烏洛克海角」，並把自己卡在木板下，就可以看到未完成的海底世界！

卡在木板下

海底世界

看來老任一開始也想到了海底的劇情，並且開始了製作，至於最後為什麼沒有開發，我們不得而知。這樣一看，在續作里增加可探索的海底地圖，可能性還是很大的！

合作模式

在《荒野之息》的預告片中，我們看到了林克和塞爾達公主在一起行動，這讓很多玩家都猜測「會不會在續作里塞爾達也是可操控的角色？」。如果能出合作模式的話，做完主線後也不會無聊了，畢竟獨樂樂不如眾樂樂！

除了這些玩家們還有很多願望，比如更大的地圖、更豐富的劇情、更複雜的神廟迷宮、更多的武器裝備以及女裝等等。不過也有玩家提出「要是能登陸PC就好了！」對此小編覺得，這根本就是不可能的事！

你覺得續作要加入什麼創新才可能超越前作呢？歡迎評論留言，說不定哪天老任就看到了呢！
