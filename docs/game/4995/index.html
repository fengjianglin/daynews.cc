DNF：每人補償一件普雷首飾，官方BUG這樣的處理，你覺得如何？

2019-12-03 0條評論 17次閱讀 0人點贊

摘要

​針對11.21DFO國際服出現嚴重的翻牌BUG，國際服官方近日發出了一則公告公告內容是對沒有翻到超金的玩家進行補償，補償範圍為所有參與了普雷攻堅戰的玩家。原文：翻譯：-從11月21日至12月17日，至少通關一次普雷伊希斯攻堅戰的玩家。

​

針對11.21DFO國際服出現嚴重的翻牌BUG，國際服官方近日發出了一則公告

公告內容是對沒有翻到超金的玩家進行補償，補償範圍為所有參與了普雷攻堅戰的玩家。

-地下城與勇士在線運營團隊

眾所周知韓服和國際服策劃都是一個辦公室的，但兩服在福利上區別很大，這也經常導致韓服玩家不滿，國際服一貫以高福利、高屬性著稱，因此也被韓服玩家稱為地下城與勇士的官方私服。

從本次國際服策劃的應對策略來看，這樣的處理結果還是比較符合他們做事風格的，大手一揮眾人皆有。

用中國的一句老話來講，叫「不患寡而患不均！」，玩家們的憤怒於沒有公平的待遇，那麼策劃就予以公平。

一下就平息了大多數玩家的怒火，而沒有能夠參與普雷的玩家也並沒有什麼實質性的影響，對於經歷了超金BUG美服玩家來說，這個結果恐怕已經是最好的結果了。

相比之下，國服同樣出現過幾次重大的BUG事件，就拿前不久的王者禮包事件來說；

解決方案是沉默，一聲不吭，裝作沒有發生。

然後這次周邊商城的活動便直接把王者禮包移除了，既然出了BUG那麼乾脆大家就都別開了，咱們開開復活幣，強烈氣息，變身藥水又安全又不會影響到什麼，難道還不好嗎~
