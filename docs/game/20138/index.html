RNG沒想過留下Karsa換小虎，全程自導自演！FW二次就業服務中心？

2019-12-18 53條評論 1,098次閱讀 92人點贊

RNG官宣在很多玩家眼裡似乎已經結束了，和預想的一樣RNG需要留的人結果一位沒有留住，而需要補強的位置則一點也沒有補強，其次就是外界說小龍堡是RNG花3000W買回來的，但是這筆錢真的就留不住Karsa嗎？或者買不下其他的LPL中單或者強力的上單嗎？在很多玩家眼裡，其實RNG冬轉則是毫無作為，反觀整個過程都是管理層一次自導自演的鬧劇，所謂的花談價留下Karsa，什麼小虎續約遭到股東拒絕，小虎只能去轉上單，要買什麼C姓中單都是無稽之談！

留下三名老將其餘的人除了直接官宣離隊的直接就沒有下文了，其次就是在RNG公布了很多官宣後，反觀則是一直在說RNG這次是花了大價錢買的YM打野，直接花了3000W，所以花光的戰隊的預算，因此買下了Betty，話說你買下Betty幹嘛？每年都是讓下路打麻將，結果撐死就讓他們上兩場，還有就是RNG最需要改變的位置呢？則依舊沒有變化。

Betty年紀是很小而且經驗豐富，和Wink想必Betty確實很是完美，上過世界賽雖說和RNG成績一樣都是十六強，但Betty的大賽經驗算是比Wink多很多，其次就是Betty也可以好好和UZI學學技術了，要不然之後也是老是被壓刀，但是經過這麼一番操作，RNG變強了嗎？似乎沒有甚至沒有原封不動而是直接倒退了，拋開下路來看RNG可以說連三流戰隊都不算。

RNG的冬轉確實造就了很多流量，使得在沒有比賽的日子RNG依舊被很多人關注，在這關注的背後RNG則是什麼也沒有做，RNG最需要穩住的打野位置則被管理層拱手相讓，甚至Karsa連RNG給再多的錢都不會留，給了小龍堡3000W給Karsa的應該也差不多，這就足以見得Karsa對於RNG的失望，其次就是RNG最需要改變的上中則沒有任何動靜，反而別的位置變化很大，想必LMS賽區沒了，FW戰隊和RNG關係那麼好，怎麼也得安排一波！

RNG現在可以說是FW二次就業問題服務中心，甚至使得自己的粉絲都在懷疑，在質疑RNG管理層在冬轉的時候做了什麼
