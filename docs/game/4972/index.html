Steam超好評《熾熱熔岩》：非傳統線性跑酷佳作

2019-12-03 0條評論 18次閱讀 0人點贊

摘要

歐美曾相當流行的“地板上有岩漿”（The Floor is Lava），其玩法概念即是假想地板充滿岩

歐美曾相當流行的「地板上有岩漿」（The Floor is Lava），其玩法概念即是假想地板充滿岩漿並找尋立足點，如今這個概念成為《Don’t Starve》開發團隊Klei Entertainment的靈感，開發出第一人稱跑酷遊戲《Hot Lava》。

非傳統線性跑酷，日常感十足

本作的場景都與生活息息相關，玩起來相當有臨場感，像是客廳、學校、公園、批發商等，當然，你的立足點也會是桌椅、柜子、欄杆等常見的物品，不過只要是椅子這種體積小的立足點，站太久是會往下沉的喔！

以往跑酷類型的遊戲，到達「紀錄點」的路很「直觀」，路線通常只會有一條。然而本作中後期的關卡，到達紀錄點的方式有好幾種，玩起來有更多「動腦」的成份。

在這樣的情況下，影響成績高低的因素，除了操作能力以外，更重要的是對於地圖的「熟悉度」，即使是休閑玩家，多嘗試不同路線也能玩出好成績。

豐富的可互動物件

本作大多數的地圖物件都是可以互動的，除了「觸碰」會移動之外，也可以拿起來「拋投」和「放置」。像是公園與學校地圖，將籃球投到籃框中會有進球的特效。

另外也有西洋棋可以玩，不過因為角色本身會撞到棋子，所以玩起來不太方便就是了。

競速不是遊戲主體，收集要素才是

跑酷類型玩到後期通常會為了求好成績而充滿「競技」感，而導致多數休閑玩家退坑，不過本作整體氛圍相當休閑以外，還有造型、動作、貼紙與漫畫等收集要素待玩家集齊。

其中最值得一提的，就是散布在地圖各處的可收集「卡牌」。有些卡牌相當容易取得，不過多數的卡牌會是在比較「刁鑽」的位置，通常要費一些功夫才可以取得。相信對於許多玩家，收集所有的卡牌遠比跑酷的成績重要。

結語：推薦與好友同游的跑酷佳作

整體來說，《Hot Lava》是相當優秀的跑酷遊戲，地圖設計相當用心，配合價格來看物超所值，不過本作單人遊玩的話，很容易提早脫離嘗鮮期而感到乏味，相對的，與朋友同游不僅更有趣，也不容易膩，趕緊揪朋友一起玩吧！

《Hot Lava》已於9月20日在Steam平台上架，售價折後26元（-19%）。
