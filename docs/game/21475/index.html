AG奪冠瞬間各大直播間的反應，久誠迷之微笑，辰鬼激動到變太監

2019-12-19 66條評論 1,016次閱讀 69人點贊

AG超玩會奪冠之後，解說李九激動到失聲，他高喊：越北海、踏南疆、少年縱馬走四方！三載征戰，赤子AG，加冕為王！前BA選手小獸，也在決賽中客串解說，他同樣激動的說到，六點六一直在說他不行了，但是他一次次的堅持了下來，一諾的年輕和天賦，是許多職業選手都嫉妒的，認識770的時候，他已經27歲了，他去過無數次醫院，但是從來沒有選擇放棄，自己很遺憾不能陪同他們一起戰鬥下去，如果歷史還能重來，他一定不會選擇退役。

其實，除了官方直播間以外，還有很多職業選手和圈內人也在自己的直播間解說比賽，下面就讓我們一起來看看AG奪冠時，各大直播間的反應。首先是Hero選手久誠的直播間，久誠提起了自己往事，他第一次看KPL也是AG和QG的比賽，老帥當時用的是幹將莫邪，久誠在看比賽的時候還遇到了阿泰，就從旁邊路過，當時阿泰非常高冷，看都沒看久誠一眼，說完之後久誠自己也是哈哈大笑。

AG奪冠，宣布老帥是MVP的時候，久誠出現了一個迷之微笑，他說到，老帥獲獎是自己的功勞，因為前兩天自己剛剛把遊戲ID改成了老帥FMVP，沒想到老師真的當選了。巧合的是，久誠用這個ID的時候還恰好遇到老帥了。

再來看看前RNGM選手一笑的直播間，在最後一波團戰的時候，一笑比選手還激動，高聲的說到：要炸了要炸了，這把孫尚香還在輸出，完了呀，孫臏也殘了，很難打了，感覺要被一波了，要被一波啦！AG超玩會要一波了呀，就剩下刺痛和770了，感覺要沒了。讓我們恭喜AG超玩會，我就說這個陣容不行的。

AG成功推掉水晶之後，一笑也是激動到雙手捂臉，高喊道：ladies gentelman，觀眾朋友們，大哥大姐們，來一波恭喜AG超玩會，孫臏和孫尚香兩個傷害太高了，銀河戰艦，王者歸來！這個MC，我跟你們說，這買他誰就是冠軍！福星啊，真的是個福星啊！冠軍戒指非常貴，之前還有人想2，這是限量版，價值很高，替補也有，MC有七個！

最後是辰鬼的直播間，辰鬼曾經也是KPL官方解說，但是最近已經很少出場了，更多都是在自己的直播間直播。在最後一波團戰時，辰鬼是所有人中最激動的，他幾乎只用了一口氣在解說，到最後聲音都破音了，彈幕紛紛嘲笑他，聲音都變成太監了。辰鬼以超快的語速說到：可以打這波，呂布死啦，Fly的位置不是很好，上官婉兒23313直接上天，這波直接往裡面追，怎麼說曹操，孫尚香輸出無敵！老虎直接中路，上路有兵線，可以一波！AG超玩會離他們的冠軍只有一步之遙，這波上路有兵線可以一波！這麼多的文字，竟然是辰鬼在10秒之內說出來的，也難怪會變成太監音啦。

看到AG獲勝之後，辰鬼也開心得像個孩子，曾經的對手已經變成了最好的朋友！賽後，辰鬼在凌晨發了一個微博：可惜沒有如果，晚安，下面附了一張自己在AS仙閣時期的隊服。看得出來，辰鬼也很想再次回到賽場，再做一次追夢少年！
