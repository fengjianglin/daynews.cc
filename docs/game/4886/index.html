家養南巨戰五渣？《方舟生存進化》這款生物被它尊稱"大表哥"

不過，南方巨獸龍這種大怪物也是有幾門親戚的，尤其在《方舟：生存進化》遊戲中，小編還有更多驚喜發現！方舟遊戲的鯊齒龍，它有著食人鯊的鋒利牙齒，和南方巨獸龍是同科，最令人驚奇的是，這個鯊齒龍居然可以被南巨尊稱一聲”大表哥”？

這是為什麼呢？這就和《方舟：生存進化》鯊齒龍的優異屬性有關了。鯊齒龍和南巨是近親，流血buff自然也是少不了。並且，最厲害的是，即使鯊齒龍被馴服了，它依舊不會喪失流血buff功能。並且，獵物一旦被鯊齒龍咬傷了了，就會有10秒的”血流不止”的時間，會不斷損失生命值。光是這個強大屬性，鯊齒龍肯定是配得上南巨的一聲”大表哥”的！

各位玩家大佬，大家在《方舟：生存進化》遊戲還有什麼新發現，歡迎大家在評論區積極留言。
