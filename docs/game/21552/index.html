爐石傳說：T7獵殺入T1，勝率直逼薩滿，暴雪到底應該削弱誰？

2019-12-19 42條評論 1,229次閱讀 76人點贊

昨日，世界冠軍獅醬在微博發了一張圖，圖片中有官方人員表示平衡補丁將會於19日正式上線。

這是巨龍降臨的首個平衡性補丁，主要在於削弱JK薩。因為在上周，薩滿憑藉JK薩一舉拿下了30%使用率和55%以上平均勝率的誇張數據，甩其他職業好幾條街。

眼看著補丁就要上線了，但是天梯里冷不丁殺出來一位黑馬，他就是——T7獵。

59.9%的勝率直追迦拉克隆薩，HSreplay網站上直接將這套卡組稱呼為Face Hunter，意為只打臉的獵人。

話不多說，送上卡組代碼：

1費卡牌高達10張，其中6張隨從牌，灼光法師、矮人神射手、山貓起手都可以直接留。任務牌後手可以留，追蹤術不建議留。

2費由探險者、狗頭人沙漠步兵和相位追獵者6張隨從牌和4張奧秘組成。

相位追獵者是核心卡牌，可以起到過濾牌庫的作用，增加有用卡牌上手率。至於帶什麼奧秘，理論上是毒蛇和爆炸更好，毒蛇增加場面，爆炸降低血線。帶誤導其實大家都會防，很少有奇效。

3費則是獵人本職業的放狗、夥伴、殺戮和鷹角弓4件套，爆發能力非常高，鷹角弓還能配合奧秘養一養。

剩下來的就是吸血蚊和火車王了，蚊子可以過牆斬殺，火車王則是單體最高爆發。

這套卡組打卡德加召喚法勝率最高，達到80%朝上，最劣勢對局是機械騎。這是因為機械騎前期鋪場更快，中期還有奇利亞斯回血。前期如果T7獵搶不到血，基本上就宣告死亡了。

核心打法就是：“少生孩子多射箭，盡量走臉”，碰到劣勢對局也基本上是翻盤無望，所以趕緊下一把，打的快也是T7獵的一個優點。

所以問題就來了，明天的補丁還會是僅僅針對薩滿一人嗎？還是說帶上盜賊和獵人一起呢？各位玩家朋友們，你們覺得明天的補丁會是怎樣呢？歡迎在評論下方說出你的看法哦~
