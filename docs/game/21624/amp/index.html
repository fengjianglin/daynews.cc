《合金彈頭》將人類改造成怪物的科學家們，或許他們才是幕後黑手

天天要聞
2019-12-18

在歷代《合金彈頭》中都會出現一些比較怪異的生物，這些怪物的肢體和形態表明他們曾經是人類的身份。很明顯，「它們」是被某種基因改造之後的成果，而這一切的線索全都指向一群幕後的科學家。

從《合金彈頭2》開始，遊戲中經常都會出現一些看似科研人員的NPC。他們一般都是出現在一些實驗室的門口。要麼已經死亡、要麼正在倉皇的逃跑，而這一切的跡象表明前面發生了什麼不得了的事情。當玩家進入實驗室之後，就會發現大量令人毛骨茸然的「怪物」。而這些扭曲的怪物擁有可怕的能力，超高的防禦和超快的移動速度，甚至還會自爆和同化。

遊戲中的各種實驗室，曾經是多少玩家的陰影。沒錯，這一切的人倫慘劇都是一群科研人員造成的，事態的發展甚至超過了他們的控制範圍。

那麼，在歷代《合金彈頭》中這群科研人員到底幹了些什麼傷天害理的事呢？

「超級士兵計劃」

戰爭似乎永遠沒有盡頭，隨著戰事的進一步擴大，叛軍的首領意識總有一天自己的軍隊將會消亡殆盡，於是接受了科學家們的提議，打造「超級士兵」進行作戰。

「超級士兵計劃」就是將殘疾的士兵，或者已經失去戰鬥能力的人員進行人體改造，將他們的器官全都換成「機械化」，只要不擊中頭部就可以得到永生。而這一方案在叛軍中得到了大規模的實施，讓很多失去戰鬥能力的叛軍，擁有了再次戰鬥的能力。

這本來是一件好事，對於那些殘疾的士兵來說，總算有了心靈的慰藉。然而，人類的野心似乎並不僅限於此。

噩夢才剛剛開始…..

部分有理智的叛軍開始懷疑這一計劃，於是嘗試潛入內部搜索相關資料。他們在實驗室深處發現了隱藏的資料，上面記載了「超級士兵計劃」的最終目的。原來這次的「超級士兵計劃」並非僅限於讓受傷的士兵再次上戰場，而是另有內幕。其中相當一部分被帶走的士兵已經被當成了試驗品變成了怪物。

根據資料顯示，身體改造計劃僅僅是「超級士兵計劃」的一小部分，計劃的第二步就是「人類變種計劃」。然而目前計劃還處於實驗階段，通過將人類和動物的基因融合，打造出擁有動物本能和人類智慧的完美生物。

發現真相的叛軍最終選擇了逃亡之路，或許只有正規軍才能阻止這個瘋狂的計劃。

然而某一天，實驗室終於孕育出可以顛覆整個世界的「怪物」。「超級士兵計劃」第二部分宣布成功，然而這群科學家們似乎低估了它們的能力，以目前的人類科技根本無法控制這群怪物。隨之而來的則是怪物大規模的失控，整個實驗室淪陷了。這就是《合金彈頭2》地鐵秘密實驗室事件的源頭。

「人類變種計劃」

實驗室被毀，科學家們幾乎被屠戮殆盡，他們的屍體被這群飢餓的怪物分食。部分逃出來的科學家終於關上了實驗室的鐵門，然而受傷沉重的他們最終還是暴斃而亡，沒有一個人活了下來。

是的！沒有人知道這裡到底發生了什麼！

不過，根據逃亡的叛軍口述：在廢棄的地鐵通道中正在秘密的進行某種實驗，實驗一旦成功將會成為世界的災難。而根據正規軍情報部門得出的調查結果：在多次戰役中發現，部分實驗成品已經被叛軍運用於戰場上。

正規軍高層發現事態已經非常嚴重，這或許將演化成為人類的浩劫。於是，高層調回了已經遞交退休申請的馬克和塔爾瑪，並讓情報部分的英里和菲奧協助，提供詳細的資料。

於是，一場關係著人類命運的戰鬥開啟了。

「基因改造計劃」

「超級士兵計劃」的第二部分獲得了成功，然而實驗對象卻失去了理智，無法繼續實施。於是「超級士兵計劃」第三部分：「基因改造計劃」啟動，多年來一直都在秘密的進行。然而普通人的體質都承受不住藥物的摧殘，實驗期間的所有人幾乎都被藥力致命。

在多次的戰爭中，叛軍俘獲了不少的正規軍士兵，都被秘密送往實驗基地。當正規軍們將實驗基地攻佔下來之後，卻發現這裡只有一群「猴子」，而之前被俘虜的人早已不知所蹤。在科研人員眼中，擁有超強體質的馬克和塔爾瑪才是最佳的試驗對象。因此捕捉兩人也被列入了「基因改造計劃」之中。科研人員的原計劃是利用馬克和塔爾瑪的體質，打造出擁有翅膀可以噴火的怪物。

不過在劇情中馬克和塔爾瑪並沒有被捕獲。

在遊戲中，只要是馬克和塔爾瑪上場，科學家們就會變得相當的活躍，而且有可能一次性出現2-3個人，同時射擊帶有血清的針管。一旦中招就會變成猴子。

有意思的是，玩家要是近距離靠近這群科學家，他們就會抽出針管。玩家馬上閃開的話他們就會扎到自己，然後變成一隻猴子。而玩家吃到猴子的話，就可以得到1000分。

但是看看這隻猴子的形態，是不是從合金彈頭2代開始就一直出現呢！那麼我們是不是可以這麼理解，我們在遊戲中吃到的獎分猴子和跟班猴子都是人類變成的呢？

「病毒研究計劃」

火星人的最終目的就是將地球變成宜居的環境。於是在《合金彈頭2》時期就選擇和叛軍合作，並在多處場景中進行秘密的實驗，而這一切得到了叛軍的支持。隨著《合金彈頭2》劇情的深入，我們才發現叛軍被火星人擺了一道，看來它們的野心並不僅限於局部。叛軍的首領摩登在《合金彈頭2》時期被抓走之後一直沒有得到釋放，而結局中救回來的僅僅是火星人假扮的摩登。到了《合金彈頭3》時期，我們才發現，原來這一切的計劃都是由假摩登來實施的。一關的輻射生物、四關的異形蟲子都是來自火星人的殖民計劃。

另一邊，摩羅十誡被火星人召喚，出現在一個小鎮上並散播喪屍病毒，實行人類清除計劃。在這之前，摩羅十誡在下降的過程中擊中了一家飛機，並讓飛機上所有的乘客變成了失去理智的怪物，成為第一批病原體。

叛軍方面為了獲取喪屍的詳細資料，曾派遣軍隊進行駐守，並安排科研人員進入獲取病毒基因。

然而，當科研人員到達之後才發現，這裡的局勢早已失控。包括駐守的軍隊全都淪陷，士兵們變成了猙獰的喪屍。發現真相的科研人員四散而逃，可惜真正逃出來的屈指可數。

於是在第二關的開場我們看到了大量驚慌失措的科學家們。

在病毒源的深處，玩家們發現了大量的喪屍，其中包括：落難的乘客、駕駛員（胖子）、士兵和科學家，全都變成了沒有理智的喪屍，皮糙肉厚甚至連子彈都不怕。

而那些喪屍化的科學家甚至將腸子拔出來進行攻擊，玩家一旦沾上就會被感染。遊戲中唯一能抵抗喪屍病毒的只有山洞中的雪怪，它們雖然沒有變成了喪屍，但是卻被喪屍們消滅殆盡。

正規軍方面也注意到了小鎮存在某種變故，於是派遣了馬克、塔爾瑪、英里和菲奧潛入調查。這猶如煉獄般的場景，甚至讓久經沙場的四人都感到膽寒。在小鎮的盡頭，主角們找到了散播病毒的摩羅十誡，並和它的守護者莫羅之眼展開了激戰。這就是遊戲中的二關BOSS。在玩家擊敗它之後就宣布了火星人本次「人類滅絕計劃」的破產。然而，在不為人知的另一邊，部分不怕死的野心科學家們收集到了喪屍病毒，以及可以對抗喪屍病毒的血清：雪怪之血。

火星人的計劃失敗了，而叛軍的「病毒研究計劃」卻悄然開啟。

到了《合金彈頭4》時期，四關的遊樂場突然發生騷亂。有人在這裡投放喪屍病毒和木乃伊氣息，試圖將整個遊樂場打造成為地獄。（之前已經介紹過了，《合金彈頭2》中出現的木乃伊就是遠古時期火星人遺留下來的病毒造成的，只要進入其中的人都會被同化，目的是為了保護外星遺址。後期叛軍和火星人合作之後，遺址中的神兵被叛軍挖了出來。）

正規軍們來到這裡的時候，已經出現了大量的喪屍和木乃伊。玩家要是走上層的話就會面對各種喪屍的圍攻。

而玩家要是走下層的話，就會出現大量的木乃伊。隨著場景的深入，終於發現了正在投放木乃伊氣息的科學家，他們帶上了防護罩，將自己包裹得嚴嚴實實。而木乃伊並沒有攻擊他們的意思，看得出來已經已經擁有控制木乃伊的技術了。

沒錯，科學家們已經完全掌握了喪屍病毒和木乃伊氣息，並且已經擁有了可以解毒的血清，一切都在控制範圍之內。玩家在關底見到了BOSS，它仍然在嘗試散播病毒，玩家要是被擊中同樣會被變成喪屍。然而玩家變成的喪屍在他們眼中並不是無敵的，可以通過特殊的攻擊方式殺死。

此時，正規軍終於發現了這群科學家的幕後首腦，他就是阿馬迪烏斯，也就是《合金彈頭4》的最終BOSS。實驗室最終被瓦解、「病毒研究計劃」也宣布完結，然而BOSS阿馬迪烏斯卻逃之夭夭。或許在未來某一天他還會出現，那時候不知道他又會打造出什麼喪心病狂的計劃。

對於玩家們來說，這群科學家們僅僅是背景人物，然而《合金彈頭》系列的很多主線、支線都是和他們息息相關的。SNK遊戲對細節的把握，真的已經到了喪心病狂的地步了！
