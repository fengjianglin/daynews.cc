魔獸爭霸3重製版模型第11期：邪獸人怎麼看起來更有獸人的狂野呢

2019-12-17 3,193條評論 4,954次閱讀 2974人點贊

各位玩家晚上好，本肥喵今日繼續和大家一起分享遊戲熱潮！

近期文章起已經進入到新的模型分享板塊，即遊戲內的模型正視圖，只為帶來簡單直觀的效果。

上期寫完獸人的三期後，發現其實三期是不夠的，預計要好幾期，除了「正經」的獸人以外，還有「不正經」的邪獸人。邪獸人是被惡魔能量徹底腐化的獸人。

一、黑石劍聖

原版譯為「黑岩氏族劍聖」，都是翻譯的鍋……現在普遍說法是黑石，也就是奧格瑞姆-毀滅之錘的氏族，身上的裝備比對戰劍聖多了不少，背後的旗幟看起來可有可無，看起來更像是「天堂在左」的下半句成員。

黑石劍聖重製版模型

二、邪獸人苦工

對比略顯呆萌的正經苦工，邪獸人看起來倒像是個工作，加上那大腹便便的肚子，怎麼看都不像是「飽受折磨」。

邪獸人苦工重製版模型

三、邪獸人步兵

邪大G，與略顯呆萌的獸人步兵比較，邪獸人則較為兇狠，氣勢上完全壓倒，凶神惡煞，身上生出的各種畸形身體特徵，正彰顯著他們邪惡的能量。

邪獸人步兵重製版模型

四、邪獸人掠奪者

狼騎兵，與正經獸人可以說是完全一樣了，當然，那表情和膚色還是差在那裡，也許是錯覺還是什麼，肥喵我覺得邪獸人更有氣勢，更能凸顯獸人的狂野。

邪獸人掠奪者重製版模型

五、邪獸人科多獸

科多獸，別說是鼓手，就連座下的科多獸都染上了「愛是一道光」的軍團綠。

邪獸人科多獸重製版模型

六、邪獸人術士

術士，與正經的模型相差無幾。

邪獸人術士重製版模型

七、馬索格

獸人戰爭領主，出現在洛丹倫地區的小頭目，與上期的督軍、領主同一個模型。

馬索格重製版模型

今天對部分獸人模型放送到這裡就結束了，感謝您的閱覽，明天見！
