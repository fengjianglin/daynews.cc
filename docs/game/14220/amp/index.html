一如既往的慣例，每逢大型發布會前夕，各路「舅舅」也開始了。TGA現在也不僅僅只是一個晚會，也是新作公布，以及未發售作品宣發的一個舞台。去年的TGA任天堂就帶來了一些驚喜，今年的「傳聞」比以往來的都要兇狠，我個人也挑選了一些舅舅所透露的消息。不過還是要事先聲明一下：這些消息只屬於傳聞，沒被證實，具體情報還以TGA當天公布等信息為準。

《皮克敏》

《皮克敏4》其實是和《荒野之息》同時期立項的產品，但是經歷過一次推翻重做之後就了無音訊。不過舅舅們比起《皮克敏4》，更偏向於會先公布《皮克敏3 重製版》。《皮克敏 4》則會安排到明年春季的直面會發布，而且預計發售日為2020年夏季。

雖然《皮克敏》在任天堂社內地位並不算高，頂多算個第三梯隊級別的IP。起初也只是宮本茂玩票性質的項目，但是它卻成了宮本茂的心頭好，極端貫徹了貓叔「好玩就行」的理念。遊戲本身可玩性相當之高，樂趣拉滿。

《密特羅德 究極4》

關於《密特羅德 究極4》的消息，舅舅們也分成了兩派說法。其中一派指出的舅舅的說法是本作將會放出一段有實際內容的宣傳片，但不會公布發售日的消息。另一派比較不太靠譜的舅舅的的說法就比較含糊不清，他們的消息是《獵天使魔女3》和《密特羅德 究極4》其中一款會有消息放出。講道理，我覺得這派舅舅說的著實有些抽象，很難不讓人懷疑是蹭熱點式的發言。

不過自從播了個幾秒的片子之後，沒多久就推翻重做，甚至還讓高橋伸也親自出來道歉。結果到現在仍舊是毫無消息也說不過去，如果不是這期間有足夠重磅的作品撐著估計早就炎上了。距離完全重置，至今也過去了11個月，再怎麼樣這個作品也會放出進一步的消息，這一點各位自然不用擔心。

大亂斗新鬥士

這一點幾乎是舅舅們公認靠譜的消息，先前櫻井承諾會在2020年2月前完成第一季的鬥士DLC，那麼TGA顯然是最合適的一個宣發點。而且Joker參戰大亂斗，本身也是在去年的TGA上被宣布，當時公布的方式精妙算是相當有排面。

關於新鬥士的說法便是眾說紛紜了，一個現在看來很有可能的說法被逐漸證實，第一季鬥士通行證的主題就是「Smash」。S=Squre Enix（勇者）、M=Microsoft（班卓熊）、A=Atlus（Joker）、S=SNK（特瑞）。所以最後一個H很有可能便是櫻井的原點HAL研究所，所以頭巾迪在這種說法下顯然是最大熱門。

《拳無虛發》

新生代的玩家對於《拳無虛發》最大的印象應該就是大亂斗中的小麥克，畢竟《拳無虛發》的上一代作品已經要追述到10年前了。系列初代的Boss戰，小麥克對戰拳王泰森，是整個遊戲史上也是難度數一數二的BOSS戰。

而且關於本作消息最早的舅舅其實就是泰森本人，今年4月的時候泰森發推特表示任天堂在開發《拳無虛發》新作卻沒有通知他，這讓他感到很生氣。鑒於泰森一貫的大嘴巴，也沒人會太多在意他的話。不過最近一些舅舅們都開始提到了任天堂的新體感遊戲項目，那麼《拳無虛發》新作貌似也是有點靠譜。
