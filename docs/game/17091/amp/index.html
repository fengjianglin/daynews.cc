戰國指南丨沒有什麼戰鬥是兩個四魂解決不了的

天天要聞
2019-12-13

犬迷們大家好呀，我是小手靈活的七寶~自古站樁沒活路，唯有套路能輸出。

許多犬迷都問四魂的攻略，寫水文不是我強項，碼攻略硬貨才是專長，本著對犬迷們負責的態度，七寶將從全方位講解四魂，為大家帶來不走尋常路的四魂搭配詳解。

四魂分析

四魂位有荒、和、奇、幸， 犬迷們根據遊戲角色屬性配裝合適的四魂，才能發揮出最大的實力。

配裝同一套四魂達到2件/4件時，會觸發四魂套裝設定，提供額外屬性/效果。不同的四魂屬性會不一樣，犬迷們在搭配的時候能有更多的選擇，並且星數高的四魂屬性會更強。


強化可以提升四魂等級與四魂主屬性數值，當等級提升至lv3、lv6、lv9、lv12、lv15時，會生成新的副屬性或提升已有的副屬性。

四魂強化時需要消耗魂力與銅幣，強化的等級越高，需要消耗的就越多。


配裝分析

此次將要介紹的幾種四魂搭配，是七寶通過徹夜研究認為相當出色的，其中值得一提的是殺生丸-斗鬼神的「滿天」四魂套裝，既有雷屬性技能傷害加成，又能生命加成20%，是追求暴利輸出玩家的更好選擇！

殺生丸-斗鬼神是一個爆髮型角色，能造成巨額雷系傷害。

「雷珏」最高帶來對敵人1280%+9600持續的雷屬性傷害，「閃雷斬」能對周圍敵人造成最高1420%+12000雷屬性傷害；「滿天」的四魂套裝設定是配裝2/4件時，生命加成20%與增加雷屬性技能的技能50%傷害。


這樣在提升傷害的同時補足了生命值，而且技能具有範圍性攻擊，輸出暴力，是追求極致輸出的玩家值得入手的四魂套裝。

相對於殺生丸-斗鬼神的爆發輸出，月夜丸則是平穩輸出型，與「寶仙鬼」四魂套裝搭配後攻擊力會隨著防禦力增加而增加，與紫織搭配使用更能增強她的守護天賦。

「寶仙鬼」的四魂套裝設定是配裝2/4件時，防禦加成20%與戰鬥中，增加自身當前防禦值10%的攻擊力，不可驅散；防禦性的月夜丸與「寶仙鬼」的搭配非常強勢，並且隨著進階、升星等進化，月夜丸的防禦屬性還會得到大幅度的成長。


月夜丸具有非常好的平穩輸出，提升了防禦與攻擊的「寶仙鬼」四魂套裝再也不用費腦筋去想的選擇了，陣容中缺平穩輸出的玩家可以考慮培養一下。

陣容中有了爆發與持續輸出，還得增加全隊的生存能力，豹貓四人眾的春嵐就彰顯了奶媽系的特性，既能給予生命值的回復，也具備提供增益buff，是能夠增加自身和全隊生存能力的保障性角色。

七寶為春嵐選擇的「無女」四魂套裝，它的設定是配裝2/4件時，生命加成20%與提高攜帶者50%的治療量。與春嵐的主動技能「花開時節」可以使友方的生命持續5秒回復，每秒回復春嵐生命值3%，更為契合。

「無女」與春嵐的搭配七寶認為主要是給奶媽玩家設計的，治療能力非常強大，選擇輔助陣容的玩家可以仔細看看。

犬夜叉-鐵碎牙繼承了群攻系往日「當今之世，捨我其誰之風」，在戰鬥中的多段攻擊與範圍傷害，對有盾的目標能產生額外傷害，是個通用性很強的角色。

搭配上「邪見」四魂套裝，能夠提升大量傷害，在配裝2件套時攻擊加成10%，4件套時攻擊霸體單位時額外增加36%的傷害。是個攻堅玩家值得入手的四魂套裝。

紫織在遊戲依舊遵循了結界技能的風格，相對於輸出紫織更注重團隊的輔助。主動技能「守護結界」通過結界召喚可以保護友方不受傷害，並對敵方造成傷害總數為1000%+20；與月夜丸同時在場時，還能增加20%的效果抵抗，在面對多敵人情況下可以增加全隊的生存能力。

增加紫織的生存能力，才能夠給隊伍繼續提供輔助的結界技能。「龍骨精」的四魂套裝設定是配裝2/4件時，防禦增加20%與戰鬥開始時，增加15%的減傷，無法被驅散。四魂套裝特效大量減少了紫織受到的傷害，能夠持續釋放守護結界，更方便其他角色利用結界的能力進行輸出。

此外，「里陶」四魂套裝增加的攻擊加成10%與攻擊傷害亡靈角色，額外增加25%的傷害，為需要對戰亡靈角色的玩家提供了更多選擇；

「刀刀齋」四魂套裝似乎更適合水神，防禦加成20%與強化50%的奧義效果（傷害/恢復），使得水神的奧義技能「雨過天晴」帶來傷害與為全隊提供少量的治療量得到大幅度的強化。

不同的四魂套裝帶來的屬性加成是不一樣的，犬迷們在配裝時有更多的選擇，同時也豐富了遊戲的策略性。能夠提升角色的各種技能特性，配裝功能性更強，且設定更為適合角色的四魂，更值得擁有。

本期關於四魂套裝搭配的分析就放送到這裡啦，歡迎各位犬迷在留言區繼續討論。
