王者榮耀：軟輔遭人嫌，深究原因很簡單，下面五點完美解決

2019-12-07 0條評論 15次閱讀 0人點贊

摘要

在王者榮耀中，輔助是一個很尷尬的地位，因為大部分玩家對輔助的作用都不以為然的原因，導致原本就比較冷門的英雄定位玩的玩家更少，而最受大部分玩家討厭的，還是秒選的軟輔。

在王者榮耀中，輔助是一個很尷尬的地位，因為大部分玩家對輔助的作用都不以為然的原因，導致原本就比較冷門的英雄定位玩的玩家更少，而最受大部分玩家討厭的，還是秒選的軟輔。軟弱的功能性強，就意味著操作比較簡單，需要的多為對遊戲的理解，很多玩家沒有意識到這點，認為軟輔就是「混」，裝備和遊戲都亂出，想要改變玩家的印象，需要看看下面幾點。

1.別先做3級輔助裝

2級輔助裝和3級輔助裝最大的差別就是，2級的輔助裝是提高團隊收益的。在2級輔助裝中明確地寫著，被輔助的玩家會獲得兵線的全部經濟，而輔助裝玩家會獲得30%的收益，也就是說一波兵線的收益高達130%。而3級輔助裝就完全不同了，會直接平分掉被輔助玩家的一半經濟，也就是被輔助玩家一波兵線只有50%的經濟，這是最招人煩的一點。

2.確定英雄定位

很多軟輔玩家，真的就一個字混。每個軟輔都有每個軟輔的特點，比如瑤就是保護，孫臏就是AOE回血，蔡文姬就是爆發性奶+回血，莊周就是解控+扛傷切後，鬼谷子就是先手開團，大喬就是牽制運營，姜子牙是減抗留人抓人，這是所有軟輔的特點，一定要把握好精髓，起碼技能介紹看一遍，也不會體現的太「混」的。

3.別逛街，前期跟野/中都可

很多軟輔玩家還有致命性的錯誤就是整局逛街，全靠光環輔助裝蹭助攻，才讓戰績沒那麼難看，輔助才是團隊的節奏點，要帶起來而不是跟著射手逛。中路如果清兵速度比對面慢，那就幫中清兵，如果打野速度慢，就幫野，如果兩個都慢就看誰後期的傷害高，英雄操作的更好。輔助待在中路/野區不是不管射手，而是中路/野區線段地形靈活，無論支援哪裡都及時。

4.團戰技能別亂甩

團戰和對線，這是遊戲最重要的兩點，輔助不需要直接對線，卻需要不斷搶節奏，打控制，團戰更是如此。輔助不需要打傷害，但是很多團隊功能性都是無可替代的事情。比如軟輔孫臏，碰上敵人選了張良，就應該在張良大自己家前排的時候給出一個沉默，而不是大招減速讓自己逃命，2技能應該多給前排回血而不是加速逃跑。

5.輔助裝要選對

前文提到過，不要買個光環輔助裝混助攻，輔助裝的選擇其實非常重要。光環類的不可疊加，光環和主動技能的輔助裝可以同時生效，所以大部分陣容都推薦星泉+近衛榮耀。本身就是軟輔，射手是保護不過來的，因此近衛榮耀和星元用來保護射手，技能的機動性貢獻給前排，這樣既有保護也有英雄特色，有抗有控有輸出有保護，隊伍才是標準搭配！

看完文章，會不會對面前的軟輔玩家有一些幫助呢？軟輔其實是操作很難的位置，所以每一位軟輔玩家實力都不弱的！
