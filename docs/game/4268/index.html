CFM中越丨戰術講堂：戰舞4殺強勢追分 孤狼關鍵局強勢發揮獲最佳

2019-12-08 0條評論 16次閱讀 0人點贊

摘要

隨著eStar，EDGM，Q9以及KZ分別擊敗AG，SN，SV以及R.LGD，今天的比賽便落下帷幕，那麼在今日的比賽中，選手們又有哪些精彩的擊殺，誰又能夠入選當日擊殺TOP5？

隨著eStar，EDGM，Q9以及KZ分別擊敗AG，SN，SV以及R.LGD，今天的比賽便落下帷幕，那麼在今日的比賽中，選手們又有哪些精彩的擊殺，誰又能夠入選當日擊殺TOP5？

TOP5：槍法過人 青菜B大下壓收穫4殺

AG與eStar的供電所地圖比賽中，雙方經過漫長的心態比拼，最終AG決定從各個方位進行頂防，由於B道地形過於狹窄，潛伏者站位較為密集導致青菜輕而易舉收穫四殺，為AG拿到本局的第四分。

TOP4：強勢出擊 孤狼關鍵局一挑四帶領KZ取得勝利

在KZ與R.LGD的供電所地圖對決中，當KZ率先拿到關鍵局後，若KZ取得這一局的勝利則直接2比0結束當日比賽，為了贏得比賽KZ在防守上採取了32戰術，當R.LGD集火進攻B點時反被在B點防守的孤狼完成一挑四，為KZ取得今日比賽的勝利。

TOP3：借物使力 青尤中路跳板左右移動獲4殺

在SV與Q9的黑色城鎮對決中，身為潛伏者的SV選擇中路配合B橋進攻B點，但青尤在中路進攻途中恰巧被頂防的保衛者所發現，此時青尤藉助中路跳板不斷變換自己的站位，運用精準的槍法完成單局四殺。

TOP2：殘局大師醬油4殺完美處理殘局

eStar與AG的鷹眼對決中，處於劣勢的AG在一波進攻下成功將B點佔領，但由於進攻途中AG傷亡慘重，在守護C4時有些點位無人把手，而醬油則利用這個機會將躲藏在B點內的潛伏者全部擊殺，完成單局四殺。

TOP1：精準打擊 戰舞4殺強勢追分

在Q9與SV的對決中，當SV掌握到比賽優勢後，準備對A點發起進攻。當SV佔領中路後，Q9多名選手回防導致SV接連損失大將， 在此時戰舞利用精準的槍法接連消滅攻進前點的潛伏者，完成單局四殺，為Q9追回一分。

通過今日的選手精彩表現，可以看出每支隊伍的選手都有一定的亮點發揮，不知這5個精彩鏡頭是否是玩家心中的今日TOP5呢？
