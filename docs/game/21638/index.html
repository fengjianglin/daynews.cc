陰陽師SSR雲外鏡技能解讀：能驅散拉條能群體輸出 燈姐親兒子

2019-12-18 843條評論 3,054次閱讀 701人點贊

期待已久的陰陽師新SSR式神雲外鏡，其相關的技能效果在近日正式公開，本次就帶來詳細的技能效果解讀，一起來看看吧。

一技能

陰狀態為落影：正常普攻，滿級125%傷害

陽狀態為曇無：滿級造成防禦力360%傷害，行動條最前端友方拉條20%

解讀：這技能沒啥好說的，就是根據狀態不同造成不同的普攻，陽狀態比較實用，還帶有拉條的效果。

二技能

斗轉：進入陽狀態，翻轉到陽的生命值，1回合100速度，每次釋放後耗火永久+2

昭回：進入陰狀態，翻轉到陰的生命值，1回合100速度，每次釋放後耗火永久+2

陰/陽：陰陽生命值初始各一半且生命值互補，陰狀態受到傷害會讓陰生命值減少，陽生命值增加，反之亦然。陰狀態下額外增加初始防禦力200%的攻擊，陽狀態下額外增加初始攻擊10%的防禦。

鏡中世界：雲外鏡無法通過治療恢復生命，但是收到治療效果的20%會變成護盾，持續2回合

解讀：該式神的靈魂技能，通過切換形態來獲得1回合的加速，同時獲得對應的生命值，看官方描述的意思，就是雲外鏡吃不到奶，但是能吃到護盾，而且防禦力賊高（攻擊10%相當於800，也就是說陽狀態下雲外鏡至少防禦力是1300+起步的）陰陽血量是平衡的，但初始只有一半血量，那點血量快見底的時候直接翻面讓血量變成血最多的那個，不能讓血條見底（見底就GG了）所以官方為了保證平衡給了個鬼火上限永久+2的限定，切3次基本上就徹底歇菜了，因為達到了8火的上限，但是，因為有燈姐的存在，根據明燈技能效果，可以達到無消耗翻面，真燈姐親兒子，給你表演一個什麼叫極限壽命翻面復活。

三技能

苦海浮生（陽狀態）：友方全體拉條20%，並讓1個友方單位獲得鏡佑BUFF（最多兩個）持續到回目結束。進入陰狀態的時候鏡佑BUFF全部移除（滿級後耗火變成2點）

雲岸凈空（陰狀態）：131%全體傷害+單體131%傷害（滿級145%）100%基礎概率施加鏡怒，持續到回目結束，陰生命比例每降低30%，額外追加一段單體傷害。

鏡佑：受到debuff或者控制效果立刻驅散並拉條10%（滿級驅散兩個，拉條變20%）每回合最多觸發一次，並治療友方生命值最低的單位180%（滿級360%）雲外鏡防禦的生命值，失去鏡佑buff時獲得鏡盾

鏡盾：吸收雲外鏡25%生命上限的護盾，持續2回合

鏡怒：目標無法獲得1個可驅散的增益BUFF

解讀：分成兩方面去解釋

陽狀態是純粹的輔助狀態，普攻拉條，大招拉條，還能保護隊友不受控制+回血+護盾，且陽狀態下雲外鏡自身賊硬

陰狀態是純粹的輸出狀態，平A垃圾，大招給力，AOE+單體的茨林式輸出方式，而且陰血量越低後面這單段輸出追加段數越多，

這式神目前看起來還不錯，有點AOE版本SP鬼切，又有點SP御饌津的意思，可操作性非常的強悍，自保能力非常的強，很考驗玩家對於大局環境的判斷力。
