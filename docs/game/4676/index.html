一、攻速快慢的區別

慢速武器就比較適合瞄準射擊和多重射擊，DPH高，而且慢速武器對輸出循環更有利。另外，慢速武器在觸發強化雄鷹守護和使用了急速射擊技能後，收益相對會更高（僅個人看法）。最關鍵的是節省子彈和箭。

快速武器就適合放風箏，在練級打怪的時候用快速武器也是十分不錯，這樣即使是平射傷害打怪要比慢速武器要快，但缺點也是浪費子彈。

二、適用環境

獵人在什麼樣的環境中選擇什麼樣的攻速武器，這個就很有講究了。一般的來說，慢速武器，在RAID（團隊副本）中，產生的傷害會更多。而快速武器，一般在pk或者pvp中會更好，如果在跟法師pk時，雖然有些其他的操作，但是遠程速度夠快，就會影響法師的施法時間。在練級打怪的時候，最好切換成慢速武器，畢竟遠程武器速度過快會很浪費子彈/箭。

三、近戰武器的選擇

獵人的近戰武器再配合獵人的猛禽一擊使用，猛禽會增加20%的暴擊，且近戰的話，更注重於起手一刀，所以獵人近戰時，選擇雙手武器會更有優勢，雙持可以雙附魔，會更有優勢！
