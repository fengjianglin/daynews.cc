狼人殺中的大哥牌、狼族的預言家，你沒玩過石像鬼？

2019-12-12 1,014條評論 3,967次閱讀 1501人點贊

最早聽說石像鬼，還是在《英雄無敵》系列遊戲里，塔樓陣營中的二級生物，速度快、防禦值高、血量也不低，非常適合起手纏住對方的遠程攻擊單位。而石像鬼形象在歐洲、埃及起源已久，在龐貝古城、埃及的神廟、巴黎聖母院都有石像鬼的出現。

而狼人殺中的石像鬼，其實並不常見。

石像鬼在面殺中出現的幾率並不高，原因很簡單：大家對石像鬼怎麼玩並不熟悉。在《Godlie》中，石像鬼曾經出現過，尤其JY的一波狼人跳石像鬼騷操作，讓人嘆為觀止。

石像鬼也屬於狼人陣營，擁有一個非常強悍的技能——每晚查驗一個人的身份。但石像鬼與狼人陣營不能相認，不能每晚刀人，只有在其他狼人都被消滅後，石像鬼才能每晚出來刀人。

一般有石像鬼的局，是12人局，1預言1女巫1獵人，4村民，3狼人，守墓人和石像鬼。這裡再介紹一下守墓人，他的能力比較特別：能夠知道前一晚被公投出去的人是什麼陣營（好人或者狼人），堪稱後預言家，可以通過分析被票出局的人來分析場上局勢。

狼人陣營會安排一個狼人在前幾天悍跳，來保住真石像鬼。石像鬼也可以在查驗幾輪後，出來跳預言家，畢竟自己是真的能查驗到的，給神牌發發查殺。有些玩家給狼隊友發查殺，但目前看基本效果都不好，特別容易被狼隊友誤會，當晚被屠，畢竟現在平民跳石像鬼是非常普遍的。

另外有個比較有趣的玩法，就是跳守墓人，但如何脫掉守墓人的衣服，尤其是在守墓人普遍容易上警的前提下，這是一個問題。

話說回來，石像鬼最穩的玩法還是——苟著，苟住，能摸魚就摸魚，保證自己安全前提下可以適當帶一下好人的節奏，活到所有狼隊友都被消滅了，石像鬼獨殺時，一刀一個準，那才是真正的恐怖。

《狼人對決》中的石像鬼，取消了查驗，但增加了兩個很imba的能力：飛行（只能被飛行契靈選為驅逐目標）、護甲（抵消一次驅逐、消滅或契約值減損），生存能力極強。但技能和契約值不具備改變局勢的能力，最好能讓他多1-2點契約值，配合很強的生存能力，成為很好的查驗者。

畢竟你大哥牌，永遠都是大哥牌！
