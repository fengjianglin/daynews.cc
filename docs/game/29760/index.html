DNF：普雷團本末期現狀，“70仔”到處可見，帕拉丁競爭太激烈

2019-12-24 1,447條評論 3,330次閱讀 482人點贊

到了普雷團本末期，玩家裝備已經「究極體」形態，為了打出極限傷害，「70仔」光兵和帕拉丁孕育而生。不過，你突然發現，帕拉丁之間競爭太激烈！在一個頻道之中，「70仔」帕拉丁隨處可見，比正常穿輸出裝的都多！

「70仔」是普雷版本中，新定義的「混子套」，左邊是兵法套，右邊是新的史詩輔助buff裝備，搭配在一起，加上帕拉丁本身34C能力，提升差不多70%左右，因此，玩家就會親切的叫「70仔」。「70仔」沒有輸出能力，但輔助強大，在高端團中十分吃香。

如圖所示，在攻堅戰頻道，等候組隊列表中，玩家可以搜索「待機人員」。其中，有幾個破曉女神，也就是帕拉丁職業。打開裝備以後，你會發現，它們就是所謂的「70仔」。不過，值得一提的是，這些帕拉丁職業，並不止一個穿「混子套」，由此可見，競爭壓力還是蠻大的。

早先盧克團本，「混子套」比較火，但能夠進行搬磚，可用材料兌換一些卡片。然而，普雷不存在搬磚，這些帕拉丁角色，一看就知道，不會認真玩，穿「混子套」去打，純屬浪費時間。當然，不排除一些玩家，為了賬綁材料，從而進行混團。

總之，新的史詩輔助裝備上線後，讓「70仔」漸漸的出現，打團也時不時的能碰到。特別是到了普雷末期，一些玩家裝備已經畢業，為打極限傷害，就會組「70仔」。而旭旭寶寶，就是一個典型的例子，經常用紅眼打團時，會組這種類型的光帕。當然，旭旭寶寶的紅眼，搭配專屬奶爸「擠擠」，加上「70仔」光帕，傷害會非常恐怖，一兩個技能帶走伊希斯！

等到了100級版本，新的裝備更加強大，普雷成為「白圖」，「70仔」會雨後春筍般崛起。一個隊伍中，只要兩個輸出，「70仔」和奶進行輔助！不過，說實話，混普雷團本的意義不大，畢竟沒有穩定收益，無法進行搬磚，甚至會浪費時間！
