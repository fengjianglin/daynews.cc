搓核桃大漢赤腳過冬太悲慘《守望先鋒》西格瑪節日皮膚引熱議

天天要聞
1 週 ago

要說起2019年十二月最令人期待的活動，《守望先鋒》的玩家們肯定會第一時間想到”冬幕節”，這個每年都會在萬聖節之後，跨年之前的活動，被所有玩家所期待著。但對於今年冬幕節的”雪國仙境”活動，官網上卻沒有絲毫的活動預告或者隻言片語，這是怎麼回事？

而在今天的凌晨十分，《守望先鋒》的官方微博終於發布了”雪國仙境”的活動預熱，讓大家躁動不安的心情得到了平復(好像更加期待了)。
此次伴隨官宣的皮膚是西格瑪的”冰霜化身”皮膚，新皮膚有著濃濃的”魔獸”風格，讓大家不禁感嘆今年《守望先鋒》的美工團隊都是《魔獸世界》臨時過來客串的。

而在12月4日，網路上所傳言的1.43(即下個大版本更新)版本將會在明年1月更新，而《守望先鋒》項目部的遊戲總監Jeff隨後在官方論壇否認了這一說法，並且表示1.43版本將會隨著”雪國仙境”活動一同更新。為什麼大家對於1.43版本更新如此期待？

1.43版本更新了許多東西，例如所有英雄的護甲減傷從3改至5，T位的護盾屏障進行了大砍，奧麗莎的投射屏障技能從900減到600，西格瑪的實驗屏障技能從1500減到900，屏障恢復速度從每秒150減到120，大鎚的的屏障也立場從2000減到1600，舉盾減速從50％減到30％，可以說1.43版本大幅度改動了遊戲現有的主流英雄，對於現有的”雙盾，地推，放狗”玩法做了很多改動，網友們所戲稱的”破盾先鋒”時代也將會隨著1.43版本的更新而灰飛煙滅。

而今天官方微博對於”雪國仙境”活動的預告，意味著明天不僅有活動新皮膚，新的噴漆，動作，表情，還意味著雙盾時代的逝去，而這也讓大家對於明天的活動與更新更加的期待。
除開令人興奮的版本更新，下半年的暴雪對於守望先鋒可謂是十分上心，不僅在暴雪嘉年華上公布了《守望先鋒》的續作，使得守望先鋒的熱度再創新高。還別出心裁的為萬聖節設計了一系列的皮膚和表情。其中我最喜歡的就是小美的金秋南瓜皮膚和富有中國特色的殭屍跳錶情了。

本次”雪國仙境”活動圍繞著”雪”這個主題，自然是離不開小美這個英雄的，活動專門開通的兩個新的大亂斗”雪域狩獵”和”雪球攻勢”，其中是以猩猩和小美為主題的1V5狩獵行動，而”雪球攻勢”則是6V6的小美打雪仗模式，不得不承認暴雪對於中國玩家真是十分的友好。

明天就是守望先鋒更新的日子了，對於即將到來的新活動，大家已經整裝待發，躍躍欲試了，身為守望先鋒特工的你，還在等什麼？趕快登錄守望先鋒，和朋友們在”雪國仙境”中一起來緊張刺激的狩獵玩雪吧！
