玩不好新英雄高順，主要還是你不夠了解他

天天要聞
2019-12-10

摘要

在新英雄高順發布之前，舞文就有簡單為大家介紹過這個英雄登場後的一些玩法思路。可等到英雄實際發布之後，由於高順尚未登陸比賽服的原因，玩家們尚未在比賽中看到這個英雄的身影。不過，隨著路人玩家嘗試這個英雄的朋友逐漸增多，他的一些優劣勢情況也逐漸受到了廣大玩家的關注。

在新英雄高順發布之前，舞文就有簡單為大家介紹過這個英雄登場後的一些玩法思路。可等到英雄實際發布之後，由於高順尚未登陸比賽服的原因，玩家們尚未在比賽中看到這個英雄的身影。不過，隨著路人玩家嘗試這個英雄的朋友逐漸增多，他的一些優劣勢情況也逐漸受到了廣大玩家的關注。接下來，舞文就為大家簡單分析下這個英雄的優劣勢情況。

優勢

很多玩家對其不熟悉：雖說高順發布到現在已經過去一段時間了，但是作為一個新英雄，還是比賽服中尚未出現的新英雄，不少玩家不僅沒有玩過甚至都沒有見別人玩過。這種情況對於選擇了這個英雄的玩家而言，或許因為是新英雄的原因玩起來有些生疏，但別人的不了解其實也算是一個非常不錯的優勢。

不錯的傷害輸出能力：之前在為大家介紹技能的時候，就有為大家提到過，由於高順的E技能和R技能配合起來，可以讓他在控制到敵人之後造成極高的傷害的原因。實戰中，高順的傷害輸出能力也確實非常可怕。

極高的機動性：雖然高順沒有位移技能，但是在開啟大招進入加速狀態之後，移動速度會有大幅度提升。只要裝備移速超過65點（除了屬性鞋需要升級之外，其他鞋子做出來就至少65點），大招加速之下的移動速度都可以達到525（大多數英雄的極限移速）。

強大的突進能力：由於開啟大招之後高順的移動速度可以輕鬆達到525，且還能使用2段效果直接突進到對手臉上的原因，這個英雄的突進能力顯得非常強大。那些腿短且沒有位移能力的英雄，在面對高順的時候會非常難受。

強大的對線消耗能力：高順是一個近戰英雄不假，可因為E技能能夠提供不錯的控制效果不說，W技能還能對前方範圍內的英雄造成物理傷害且技能CD時間較短的原因，對線消耗能力還是非常不錯的。

缺陷

裝備需求較高：由於高順本身是一個近戰英雄，且技能特性註定了他在進場之後非常容易成為對手的集火對象的原因，自身對裝備需求還是比較高的。也正是因為這點，之前的文章中舞文才有提到高順這個英雄雖然輸出能力不錯，但是用作2號位相對而言會更好一些。

不怎麼適合做核心位：從天賦上來看，高順的天賦搭配還是非常不錯的，無論是輸出還是滾刀，都能夠輕鬆搭配出效果非常強大的一套。可是，正如上面提到的這個英雄極其容易遭到集火的原因，放在核心位置上效果只能算是一般。

先手能力一般：高順的E技能控制效果確實非常穩定，但是由於施法距離確實有些短的原因，在沒有足夠裝備支持的情況下，先手能力真的很一般。

綜合評價

整體上來看，高順還是一個非常不錯的英雄，無論是放在2號位上的團戰發揮，還是純輸出路線的抓人能力都不容小視，只是純輸出路線的團戰能力偏弱。最後，大家需要特別注意的一點是，雖然高順開啟大招後免疫減速，但並非免疫控制，而且自己在使用大招的2段效果突進的過程中，也會被控制。
