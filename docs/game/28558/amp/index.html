德杯八強出爐，最後1把IG戰勝BLG，按照規則IG被淘汰

天天要聞
2019-12-23

這裡有最新最全的英雄聯盟資訊，菌菌帶你看不一樣的召喚師峽谷~

12月21號到12月22號，英雄聯盟德瑪西亞杯小組賽程如期舉辦，這次的比賽所有戰隊都很重視，LPL的16支戰隊和4支新人隊伍參與進來，因為德杯的舉辦時間為冬季轉會期之後，在各個戰隊組建自己的全新陣容之後，急需一場大型賽事來考驗一下新陣容，順便磨合一下隊伍，所以德杯也被稱為「新人試訓杯」，而在兩天的角逐之後，最後八支隊伍晉級淘汰賽。

在第一天的比賽，A組分別是VG、WE、FPX、SN、LGE，VG在iboy和蕉太狼的加入之後，整體實力大幅提升，而最恐怖的是，主教練Kkoma還沒有上場，如果Kkoma上場bp的話，這支隊伍更加恐怖了，而WE戰隊在新賽季選擇了開展下一個五年計劃，培養一群新人，大家都以為WE的家倒了，其實WE只是拆遷住進了豪宅，在德杯上，這幾位新人無論是個人能力還是配合上，都十分優秀，WE未來可期！不過這次FPX沒有上一隊的原因，而是把二隊的弟弟們抬上去打，不然FPX應該也可以晉級淘汰賽，而在SofM加入SN戰隊之後，並沒有給SN帶來驚喜，全隊表現平平，而LGE作為次級聯賽隊伍，還需要磨練磨練。

第一天除了A組之外，同時進行的就是B組的比賽了，打野Kanavi在這次的冬季轉會期選擇回到JDG戰隊，可能因為JDG的伙食太好了吧，這位前GRF的打野，實力出眾也是個操作怪，中單牙膏像是被激活了一樣，於是帶領了JDG全勝小組第一出線，而RNG戰隊，沒有首發Uzi，因為Uzi的手還在治療，所以首發了Betty，沒有Uzi的RNG就暴露出很多問題，小虎表現不盡人意，小龍堡以為是來躺贏的，結果成為大腿，最終RNG、OMG、LNG三隊同分，但是按照賽事規則，RNG獲勝場次平均比賽時長都大於OMG和LNG150秒，所以RNG晉級，這賽制也被很多人吐槽！

第二天的比賽就來到了BD兩組，B組EDG參加這個比賽就像回到了家，十分兇猛，特別是中單小學弟，曾經50萬買的，如今打出了5000萬的效果，帶領EDG拿到了小組第一晉級下一輪，而TES在Karsa加入之後，全隊猛如狼，但是最後TES敗給了EDG，所以拿到了小組第二晉級，值得一提的是，LGD在Peanut加入之後，表現沒有預期那麼高，可能是因為語言的原因吧，還有就是V5戰隊真的也挺猛的，拿下了EDG戰隊。

而D組的話，大家都沒想到RW這麼強，居然3-1小組第一晉級下一輪，最後一場，IG對陣BLG，最終IG戰勝了BLG，BLG、DMO、IG三隊同分，但是BLG獲勝場次平均比賽時長都大於DMO和IG150秒，所以BLG以小組第二晉級，可能最不能接受的是，我剛剛打敗你，我卻被淘汰了，這讓很多IG粉甚至是觀看比賽的玩家很是不滿，不過這個規則是賽前就定好的，大家也就只能無言以對了。

最終晉級的隊伍是VG、TES、EDG、WE、JDG、BLG、RW、RNG，八強賽將會在聖誕節的後一天進行，也就是12月26號，WE再次遇到了EDG，不過這次都是全新隊伍，沒有恩怨情仇，你認為這八支隊伍哪四支能晉級半決賽呢？
