新英雄瑟提登場！純肉搏真男人，一個大招能秒ADC？

天天要聞
2019-12-12

新英雄要來啦！不就是那個厄斐琉斯嗎，不是都公布十來天了嗎，有什麼好稀奇的？不是ADC，是ADC殺手，有一個新英雄地下之王-瑟提！

沒錯，在國服玩家還沒有玩到上一個新英雄厄斐琉斯的時候，拳頭已經馬不停蹄的將下一名新英雄給公布了出來，他就是瑟提。在最新一期發布的王國機神系列皮膚當中，瑟提作為壓軸角色登場，和賈克斯、蕾歐娜等英雄一起成為了新皮膚的獲得者。

相比過去新英雄都只發布原畫以及特效不同，本次地下之王-瑟提的技能也在外服測試服一併更新，這名近戰英雄的所有技能效果以及數值也都得到了曝光。

從瑟提的被動技能可以看出，他是一個使用雙手進行搏鬥的近戰英雄，普攻中銜接著左勾拳與右勾拳，由於右勾拳附帶額外傷害，因此瑟提在擁有一定攻速裝備的前提下應該會有不錯的表現。

瑟提的Q技能看起來像是配合被動設計的，開啟之後可以快速貼近敵人，並且下兩次普攻傷害增加，在對線期這個技能可以加強他的換血能力，在遊走時也可以快速貼近敵人。

而W技能看起來是瑟提的主要傷害來源，他在戰鬥中受到的傷害可以被轉換成豪氣值，在積攢了一定量的豪氣值後瑟提可以主動使用一記重拳來攻擊對手，類似於蔚E技能的效果。而在這記重拳的直線攻擊範圍內，敵人會受到豪氣值加成的真實傷害，其餘敵人則受到物理傷害，如果出輸出裝瑟提在團戰期間可以打出大量的真實傷害。

瑟提的E技能比較類似扎克的Q，將其兩側的敵人撞到一起，如果成功拉到了兩名敵人，會對敵人造成1秒的眩暈。結合幾個小技能來看，瑟提無論是在上單位還是在打野位應該都能有不錯的發揮。

不過真正最有料的還是瑟提的大招，瑟提在使用大招後能將一名敵人背在身上進行一次摔跤動作，這次背摔會對背負的目標以及被砸到的目標同時造成傷害，而這個傷害加成中除了AD加成外，還包含了被摔者40%的最大生命值加成，假如瑟提背了對面上單6000血的大蟲子，一個背摔下去ADC可能還要倒欠瑟提1000血。

從目前公布的技能來看，如果瑟提的模型設計比較順手，他勢必將會成為上單戰場上的一大熱門人選，不知道大家覺得這名英雄是否強大呢？
