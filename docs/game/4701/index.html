8.25戒律牧師 解決按鍵問題實用宏分享

2019-12-05 0條評論 15次閱讀 0人點贊

摘要

大家有沒有覺得毒奶和懲擊佔位置?習慣把毒奶放在1，也習慣把懲擊放在1。大家有沒有覺得盾和痛佔位置?習慣把毒奶放在2，也習慣把懲擊放在2。大家有沒有覺得驅散和凈化佔位置?習慣把凈化放在t，也習慣把驅散放在t。大家有沒有覺得壓制和暗影魔佔位置?習慣把壓制放在r，也習慣把暗影魔放在r。

大家有沒有覺得毒奶和懲擊佔位置?習慣把毒奶放在1，也習慣把懲擊放在1。

大家有沒有覺得盾和痛佔位置?習慣把毒奶放在2，也習慣把懲擊放在2。

大家有沒有覺得驅散和凈化佔位置?習慣把凈化放在t，也習慣把驅散放在t。

大家有沒有覺得壓制和暗影魔佔位置?習慣把壓制放在r，也習慣把暗影魔放在r。
