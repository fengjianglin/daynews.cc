廠長帶隊出戰德杯！新賽季EDG能否讓廠長圓夢

天天要聞
2019-12-19

12月21日，本周六，一年一度的德瑪西亞杯將重燃戰火，闊別賽場許久的眾多戰隊在完成冬季轉會期後將迎來休戰多日後的首賽。德瑪西亞杯是國內除了LPL職業聯賽以外，最高級別的《英雄聯盟》賽事。這次的德瑪西亞杯在轉會期結束後舉辦，是每支隊伍都以新陣容亮相賽場，作為2020年LPL職業聯賽之前的一次練兵，究竟這些隊伍在人員變動後，能夠打出怎樣的成績，也是廣大觀眾關注的焦點。

而最引人注目的要屬由剛剛退役完成教練轉型的廠長帶領的EDG戰隊，Heart繼續擔任隊伍監督，此外還有來自LMS賽區的Corgi加盟教練組。

EDG德杯出戰名單

選手方面，此前曾在EDG待過的上單奧迪Aodi回歸隊伍，還有從2019英雄聯盟職業青訓營中脫穎而出的黃祥xiaoxiang也以上單身份加入隊伍，三名上單首發依然是劍仙Jinno。打野方面，此前曾宣布自己離開FPX成為自由人的Xinyi，如今也總算找到了下家，Xinyi曾隨隊伍FPX在今年奪得世界總決賽冠軍，因為小天狀態很好，作為替補打野Xinyi雖然沒有機會上場，但他穩健的比賽風格依然得到了大家的肯定。去年新人Jiejie成為主力，另外還有來自中國台灣的新人余峻嘉Junjia加入。上野都有多名選手輪換，也給EDG的風格與戰術帶來更多可能性。

廠長

而EDG前下路選手Iboy則選擇離開隊伍轉會至新戰隊VG，EDG下路由替補AD選手Hope升格為首發，中路與輔助則依舊是老將Scout與Meiko擔任首發。

EDG將在12月22日迎來自己在德瑪西亞杯上的比賽，下面是詳細賽程。
