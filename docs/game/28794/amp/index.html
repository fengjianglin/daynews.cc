《DOTA2》7.23版本強勢上分英雄推薦——劇毒術士篇

天天要聞
2019-12-21

曾今的劇毒令人聞風喪膽，它在路人局中出場率相當高，有AOE，有召喚物，還有無盡的減速，隨著7.23的版本更新，劇毒變得更強大了。它能在前哨與防禦塔之間利用蛇棒來回牽扯，也能利用回血抑制來克制像哈斯卡這樣的令人厭的英雄，它還能在到達30級之後成為一個絕對核心，即使你只是輔助劇毒的身份。我不知道其它否是最強的版本輔助，但若要談論契合版本，劇毒是當仁不讓的猛男。

劇毒與哨塔

一個團隊，資源的分配決定了玩家位置分配，而共享型資源會加速全隊成員的成長，使其更快速地從忙碌與沉重的輔助工作中脫離出，化身為核心來幫助團隊獲取最終勝利。在刀塔2中，哨塔的經驗加成就屬於這樣的戰略資源，如果不是全線潰敗，沒有任何一隻隊伍會放棄這堪比全團經驗書的資源，更何況在哨塔的附近還有閃亮的賞金符。

劇毒能在哨塔爭奪中做什麼？在我看來有2個要點，推進以及防守。劇毒是自帶推進能力的英雄，3個小技能都涉及到一定的帶線能力，醬油劇毒前期可以利用蛇棒來壓制兵線，中後期則可以利用1技能輔助清線。有一個小技巧是，劇毒的2技能被動會被蛇棒繼承，且能與本體疊加，所以帶線的時候記得操作蛇棒和本體攻擊每個小兵一次，然後收刀。

帶線的意義在於——在當前版本下，一旦涉及到哨塔的爭奪，擁有長且深入的兵線就意味著有哨塔的先手權。將兵線推過哨塔，我方任意成員都能在6秒時間內從容地單人控制哨塔，而提前點亮的哨塔即意味著我們在爭奪戰中提前掌控了哨塔周邊視野，相對的，獲得了戰術選擇的權利，是攻、是守？都由我們說了算。請注意了，因為蛇棒的存在，劇毒術士最好能做到提前30秒到達戰場，只需在哨塔周圍1000-2000碼布置3-5根蛇棒，我們就能給奔襲而來的敵人補上一個生命回復降低的Debuff的同時，掌控敵方動向，這是其他輔助很難達成的。

至於防守，蛇棒陣的作用大家都知道，如果明確的哨塔攻防，一個5根以上的蛇棒陣可太煩人了，經常性的看到了就會讓人產生一種無從下手的感覺，並失去進攻慾望。不幸的是哨塔不像高地是一個純粹的單面防守地形，而是一個被包了個圓的營地，使得劇毒的蛇棒發揮受到了一定的限制，但刀塔畢竟是一個講究平衡的遊戲不是么？即便如此，利用蛇棒配合哨塔視野，劇毒能在守塔守衛戰中發揮相當的作用，是一個守塔小能手。附贈一個技巧：蛇棒擁有獨立碰撞體積，所以在哨塔防守戰中我們是能利用蛇棒來堵住特定地形的路徑的，一旦堵住最近的路線，敵方自動尋路進哨塔的時候會直接繞路，而不是走最近的路線（這會讓對面很困擾，滑稽），當然我們也能利用蛇棒將衝進陣地的敵人瓮中捉鱉。

毒瘤輔助

由於新版本的特性，全體醬油在經濟層面上已經被徹底解放。但我們的輔助工作看似不多，實際上又需要顧及到方方面面，拉野、蹲人、符點、哨塔、真假眼、霧、粉，還有關鍵的找Gank節奏和帶線，這些工作做起來並不容易，這是版本對輔助提出的要求。但不管如何，對於劇毒本身而言，有幾點是不可忽視的。

劇毒擁有很穩定的刷錢能力，前期只要有足夠的被動和蛇棒等級，不論是幾波野怪都能清給你看。所以囤野工作就顯得至關重要了，由於自身是遠程的特性，其雙拉會變得非常簡單，只要有6-8波野怪，送給大哥也好，自己清掉也好，經濟就能瞬間起飛。因此，在度過對線期之後，劇毒需要在蹲人遊走的同時，找機會囤上一定的野怪，這一批經濟是我們中期發力的關鍵，能幫助我們快速出到推推棒、微光、紛爭面紗等團隊道具，幫助我們加重中期團戰的砝碼。

帶線的方面我們在第一部分說了，甚至在我們此前的劇毒攻略中《DOTA2英雄教學：我！渾身是毒，飛躍魚塘的劇毒術士進階攻略》也強調了，一旦進入到遊戲中期互刷環節，劇毒可以把帶線放到第一優先順序。注意不要搶團隊安全的線，那是給大哥的，我們要找機會深入敵營，利用蛇棒和1技能瘴氣一波一波有序的清理兵線，在劇毒擁有高等級之後，一個瘴氣+一根蛇幫就能清理一波兵線，也別忘記前文說過的本體與蛇棒被動疊加的操作。

劇毒是一個經濟戰鬥力轉化比低，又非常吃經濟的英雄，一方面其技能在常規輔助裝備的加持下，很難取得輸出加深的效果，另一方面，它又需要大量的道具裝備來幫助團隊做得更多，我個人的思路是，由於當前版本拔褲衩5號位近乎絕跡，包括Doom、斧王、天怒、宙斯都已經徹底適應5號位的角色，在裝備選擇方面，我們完全可以更加的奔放大膽一點，血精石、雙刀、羊刀、玲瓏心，這些法系傷害加深以及終極收益的道具，對於劇毒而言完全可以加入到我們的選擇項目里。這裡不要忽略了一個至關重要的道具，也是劇毒在當前版本中崛起的重頭戲——大骨灰（魂之靈瓮），該道具的生命恢復和治療降低效果的60%與劇毒自身的30%生命回復抑制完美疊加。一旦吃了劇毒被動、被貼上大骨灰，像下圖中這種6先鋒盾每秒回47+血的斧王回血效果降低到了4.8。換句話說，劇毒的毒刺+大骨灰約等於回血效果移除，就算對方回到了泉水劇毒在附帶上技能之後都能形成擊殺。

得益於劇毒2技能在新版中的變動，毒刺除了自身的傷害和減速效果外，附加了額外的生命恢復降低效果，在對付特定英雄、特定體系方面，一個大骨灰劇毒，能成為破局的關鍵性因素，譬如毒瘤中單哈斯卡，它是依靠被動吃飯的，碰上劇毒被動效果廢掉後，那就是一個鐵憨憨，一個靶子，除了一定的爆發力，完全沒有肉度可言。同理還有龍騎剛被這樣的肉核，劇毒只需要利用蛇棒就能在一定程度上削減這些英雄的戰力。

反制

劇毒並不是無敵的的存在，即便其綜合能力再強，也有反制手段。劇毒是一個非常懼怕團隊裝的英雄，尤其是笛子，對付有劇毒術士存在的法系陣容，我個人極其推薦雙笛子的團隊裝出法，近乎可以直接在戰略層面上達成巨優。只要遊戲沒有進入到大後期，一個笛子就能將劇毒的輸出降到最低，如果還有蓮花這樣的道具就最合適了，由於劇毒的1技能瘴氣、2技能毒刺被動在如今版本下都是能被驅散的，所以有一個蓮花這樣的道具或者神諭者這樣的英雄，就能保證我們的單體英雄能在劇毒手上存活下來。

另一方面，劇毒的回血抑制和大骨灰，都只是在恢復效果上對敵人達成壓制，撞見了吸血流，那是一點效果都沒有。所以，雙刀撒旦的流派是對劇毒極好的針對，如果可以，團隊中最好還能多出一個祭品來。所以道具層面上，有許多選擇都能剋制劇毒術士，畢竟除了上帝的小精靈，刀塔可沒有其他AutoWin的英雄。

在剋制與反克制方面，許多英雄都能針對到劇毒，而同樣有許多英雄畏懼劇毒。我個人玩TA（聖堂刺客）就很怕醬油劇毒，折光的傷害抵擋在其面前聊勝於無；拍拍熊就不用說了，劇毒是它的親爹，風箏起來不講道理的；還有哈斯卡，我們前面也說了，這些回複流的英雄在劇毒面前都是弟弟；最後還包括一系列的脆皮敏捷核心，如血魔、PA（幻影刺客）、黑弓等等，在BKB之前，劇毒的一個大招+蛇棒被動就能將這些英雄打成殘血。

劇毒被哪些英雄克制其實在前面的介紹中已經涉及到了，吸血流就是最好的解答之一，如小狗這種一個沖臉劇毒人就沒了，還有醬油宙斯，同樣是當前版本最閃耀的5號位，宙斯殺劇毒那就是宰豬，後期同樣裝備級別的宙斯多數情況下能比劇毒有更好的發揮空間。但話說回來，很少有人會選擇刻意的去針對一個醬油，基本上在天梯中1、2選的位置大家都已經能簡單判斷是一個醬油位了，除非是影魔這樣的個別非中路不可的英雄，所以當我們看到敵方陣營中有劇毒的時候，在裝備上去針對會是更合理的選擇。

最後，各路英雄於中新版本中大放異彩，輔助核心競相開花，不論是劇毒也好，宙斯也好，或者像大樹這種曾今無人問津的孤兒，只要有所付出，我們總是能在它們身上發現一些驚喜，畢竟這是刀塔有史以來最好的版本。當我們在談論刀塔的趣味性時，7.23已經給了最好的回答。
