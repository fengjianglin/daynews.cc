《極小噩夢》快來一場酣暢淋漓的夢境吧！「游范兒」

天天要聞
2019-12-25

相信很多小夥伴都做過一些奇奇怪怪的夢境，在這些夢境中，有時會遇到巨大化的怪物，有時又會進入循環往複的迷宮，而有時，我們會一腳踩空，在體驗短暫失重感後突然驚醒。
當然，噩夢最可怕的地方便是身臨其境的現場感，但如果讓你控制別人來完成他的夢境，又會是什麼樣的體驗呢?本期為大家推薦的便是名為《極小噩夢(Very Little Nightmares)》的手游，下面就來跟隨本菌的視角，體驗這款遊戲吧~

講講本作前身故事
在介紹這款遊戲之前，游范君要先來為大家講解一款PC遊戲，於2017年Steam上架的《小小噩夢(Little Nightmares)》。

這款由瑞典工作室——Tarsier Studios製作，萬代南夢宮發行的《小小噩夢》，一經發售就廣受好評，遊戲採用了橫版捲軸玩法，並採用了精緻的3D畫面，通過遊戲的潛入解密等玩法，慢慢引導玩家進入了主角的一個古怪噩夢之中。
在背景設定上，身披黃色雨衣的主角小六將在一個巨大而又古怪的游輪上探索，在尋找生機的同時，還要逃脫怪物的追捕，可謂是恐懼感拉滿。

手游版又有何差別
在遊玩過《小小噩夢》後，我們都會對小六的情況感到好奇，她的身上為什麼會發生這種故事——這樣的問題深深埋藏在玩家的心裡，於是在兩年後，《小小噩夢》前傳《極小噩夢》終於發行，講述了主角小六在發生《小小噩夢》之前的故事，相當於作品前傳。
只不過，這次萬代南夢宮並沒有選擇在Steam版發行，而是選擇了目前大火的移動平台，也就是手機版。
進入遊戲，我們看到小六急急忙忙的跑進一個小房間，彷彿在躲避什麼，氣喘吁吁的她連喘口氣的時間都沒有，就趕緊搬起一旁的木板頂住大門，過了一會，咚咚咚的砸門聲響起，遊戲也正式開局。

《極小噩夢》並沒有選擇與端游相同的橫版捲軸畫面，而是以豎屏視角展開，同個場景內可見內容更少，玩家無法一覽全局，只能慢慢往前探索，這種慢慢悠悠小心翼翼的遊戲方式讓這款遊戲的未知感更加充實。
縮減橫版視角，拉高豎版視角，讓這款《極小噩夢》更多的操作集中在了主角上牆下梯互動解密，看起來有點像《紀念碑谷》，在有限的畫面里利用視覺錯位來提升空間縱深。

玩法簡單上手，感官親臨其境
玩法方面，依舊是潛入與解密並存，為了方便豎屏控制，遊戲整體採用了點擊的玩法，只需要點擊想要去的位置，就可以控制主角移動，同時遊戲中的所有解密互動也由點擊來完成。

這款作品的優秀之處，不僅在於一環緊扣一環的解密玩法，同時還將策略及動作融入其中，例如在攀爬高牆時，首先要讓小六控制垃圾桶堵住裂縫建立落腳點，在攀爬過程中還需要躲避從上方掉落的岩石，不過於困難，也不太簡單。

美術風格則繼承端游的陰暗畫風，但可能是考慮到手遊玩家的年齡層偏小，整體亮度調高了許多，大部分時間都會有光源進行照明。

雖然畫風陰暗，但細節方面十分細膩，無論是空氣中的細小灰塵，還是角色道具映照出的影子，遊戲都能精細呈現給玩家。
除去前兩者，遊戲音樂也造就了靈魂，有點壓抑，但不致郁。大大增加臨場感，又不至於玩完就做噩夢~畢竟咱們還是要好好睡覺的~
總結
整體來說，這是一款質量上乘的恐怖解密遊戲，依靠精美的畫面及音樂烘托出了濃郁的氛圍，稱得上是一款不錯的遊戲。
然而，也正是由於Steam版由萬代南夢宮發行，時長僅僅5小時的作品定價卻高達168元，著實勸退了一大批玩家，所以並不建議大家去購買。

至於本期所推薦的手游版，海外已經上線，定價7美元，有興趣的玩家可以嘗試一番，安卓黨要記得魔法上網哦~
那麼，本期游范君就要跟大家說再見啦，祝你今夜晚安，好夢。
