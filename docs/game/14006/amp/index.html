DNF: 手裡還有這把武器的，現在可以封起來賣了，它比黑刀更稀有

天天要聞
7 天 ago

地下城與勇士的朋友們，大家好！

對於我們很多普通玩家以及一些佛系玩家來說，獲得金幣的難度還是比較大的，本來我們沒有特別多的時間，同時我們也不想做出一定的資金投入。因此對於我們很多朋友來說，就能夠依靠我們之前的一些存貨，或者是我們在平時刷圖的時候能夠獲得一定的金牌兌換一定的金幣。

對於我們很多老玩家來說，倉庫當中可能會保存曾經的一些武器，這些武器在當前的版本當中還是非常有用處的，因為我們的武器幻化系統已經出現，有很多朋友是非常喜歡曾經的一些武器，而且有很多武器在連環副本地圖當中是刷不出來的

手裡還有這把武器的，現在可以封起來買了，它比黑刀更貴更稀有！在我們的靈魂副本地圖當中，我們能夠刷出來神器武器以及史詩武器，但是稀有級別的武器是不可能刷出來的。在我們曾經的版本當中，我們很多玩家所佩戴的武器基本上都是稀有級別的武器，而且在造型上還比較不錯。

如果一些朋友還能夠擁有我們曾經60版本40級的武器血蓮鳴鳳劍的話，我覺得大家沒有必要去幻化，還不如封印起來把它扔到拍賣行去買。由於這把武器在外觀造型方面備受很多老玩家的喜歡，因此他在拍賣行的價格也比較高，如今已經被炒作了5000多萬。

如果一些朋友還有黑刀暗月的話，我覺得也可以封印起來扔到拍賣行去賣，因為這些武器對於我們很多佛系以及眾多的普通玩家來說，我幻化的意義並不是特別大，大家可以慢慢的去刷靈魂副本地圖。黑刀暗月同樣是一把在外觀造型方面比較不錯的一把武器，在價值方面也比較高，在跨五大區它的價格一般都是維持在3500萬以上。

我們看到的這一把武器，它同樣是一把稀有級別的武器，在我們的靈魂副本地圖當中也是一把不可能被刷出來的，僅僅只有一些老玩家的倉庫當中才會擁有。

對於我們很多普通玩家已經發現，玩家如果我們的倉庫當中還有這把武器的話，並不建議大家去幻化。我覺它賣掉會比較好一些，畢竟錢才是自己的，這把武器在拍賣行的價格也是非常貴，因為它非常稀有，單價也是在1100萬以上。

在我們眾多的武器幻化造型當中，泰拉石巨劍在整個造型設計方面還是非常不錯，我們也能夠看到有很多朋友幻化泰拉石巨劍。這把武器它在價格方面同樣是比較貴，外觀造型並不是特別看重的玩家來說還是可以把這把武器給處理掉了換成金幣打造角色不好嗎？
