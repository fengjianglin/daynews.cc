同樣是《妖怪手錶4++》，為什麼Switch上的玩家比PS4上更多？

天天要聞
2019-12-21

相信玩家們對於PS4和Switch的性能而言已經再了解不過了吧，這兩者的機能差距可以說是天差地別。但Switch對於很多玩家而言就只是一個掌機，機能差距從何談起？主機和掌機本身就有著天差地別的差距。

但是，要說兩邊玩家的用戶群體來說，相差還是不大的。任天堂和索尼都有不少的用戶群體。可是，最近小編卻驚奇的發現，《妖怪手錶4++》這遊戲PS4上玩家相當的少見，更多的則是Switch上的玩家。即便是貼吧，也幾乎都是Switch版玩家們的天下，PS4玩家想找人一起打特工任務都相當吃力，這是為什麼呢？PS4畫面流暢度更高，也更清晰，相比較Switch的畫面卡頓掉幀來說，PS4應該人會更多。

小編認為，PS4人氣低於Switch的原因並不是性能的區別。

PS4首次登入《妖怪手錶》

對於PS4的忠實粉絲來說，有一部分玩家幾乎完全看不上任天堂的掌機和主機的。所以，不少玩家對於曾經在3DS上堪比《寶可夢》的遊戲沒有太多的了解，並且首次登入PS4便是《妖怪手錶4》，而對於之前沒有玩過的玩家來說，怕劇情這些都過於陌生，所以提不上興趣。

而Switch玩家則大部分都是從3DS時期就已經開始玩《妖怪手錶》了，對於《妖怪手錶》系列還是相對比較熟悉的。況且曾經《妖怪手錶》沒有中文，大家玩起來都玩的不過癮，這次首次加入了中文，還是有不少玩家願意去體驗一下的。畢竟這是曾經堪稱和《寶可夢》有的一拼的遊戲。

主機便攜性

PS4和Switch最大的區別就是便攜性。PS4做不到和Switch一樣隨時隨地就能拿在手上玩。這一點多好的硬體都無法比得上的。說起《妖怪手錶4++》這款遊戲的表現，小編覺得它像是一款手游一樣，有自動戰鬥，自動尋路，每日丑扭蛋，還無法修改時間大法來作弊抽扭蛋。方方面面都有著手游的要素的遊戲，放在PS4上讓玩家就會感覺玩個遊戲好累。而不能像Switch玩家一樣，一邊玩著Switch一遍看著電視。必變了很多枯燥無味的跑圖劇情和內容。

主機面向玩家喜好不同

要說《妖怪手錶》的風格確實更適合Switch一些，但放在PS4上就顯得有些大材小用。畢竟《妖怪手錶4++》從畫面看的出性能要求並不很高，PS4完全可以勝任。並且，PS4的玩家更多的是喜歡發揮PS4機能，畫面偏向真實的遊戲。而不太能夠接受卡通畫風的遊戲。但Switch的玩家則相反，Switch玩家會更加希望遊戲性好的遊戲。因為隨時隨地可以帶出去玩耍的話，遊戲性還是相當關鍵的。

小編綜合下來，兩邊主機差距最大的原因還是這3點原因導致，與用戶量無關。PS4有著優秀的性能更適合在家玩大作，而Switch的便攜性可以玩一些相對比較有意思的遊戲。這並不是說PS4做的比Switch差，相反只能說現在的玩家對於便攜性還是相當看中的。

不僅如此，PS4的玩家基本都是正版玩家，而Switch的玩家中還是存在不少的破解玩家的。破解雖然並不是一件非常值得提倡的事情，可是破解確實帶來了主機的銷量。但破解終究損失了聯網功能。許多遊戲沒有了聯機，就基本沒有什麼可玩性了。

不過，小編建議，如果有玩家想加入《妖怪手錶》的遊戲中的話，真心建議購買PS4版本的，畢竟沒有魔改的妖怪聯機空間非常乾淨，而且遊戲打起來真心要比Switch更加流暢。小編入了雙版本後，再也沒碰過Switch了。Switch卡頓實在嚴重。

不得不說，現在《妖怪手錶4++》對於小編來說已經超越了《寶可夢劍盾》了。只是遊戲真的知名度不夠。
