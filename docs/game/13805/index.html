王者榮耀：31位“選手”為AG捧場，QG：我有世界冠軍，誰怕誰？

2019-12-13 307條評論 4,624次閱讀 178人點贊

「你有張良計，我有過牆梯」

AG超玩會與QGhappy宣傳戰已開打一段時間了！AG前隊員蘭息在直播間中表示：在總決賽那天，將會有超三十一位AG超玩會歷代選手在現場助陣！一起見證這歷史時刻。這一舉動立即引發了AG超玩會粉絲的集體回憶。一個字：美。

面對AG超玩會咄咄逼人的宣傳態勢！QGhappy終於放大招了。

一段由不同體育項目的世界冠軍、運動員及娛樂明星等組成的的慶祝視頻出現了網路。

視頻中張繼科、跳水奧運冠軍陳若琳、短道速滑世界冠軍石競男；擊劍運動員孫偉；女排隊員薛明；還有符龍飛、董岩磊、馮薪朵等都為QGhappy祝福。

他們齊聲道：祝QGhappy，在決賽中奪冠。

打出好成績。

生而無畏，戰至終章。

這一組宣傳攻勢，確實有意思！在這之前，QGhappy一直低調，默不作聲。任由AG超玩會上竄下跳錶演。隊內一片祥和之氣。

QGhappy隊員的直播也沒有停下。而AG超玩會則枕戈待旦。

不但請來了E星作陪練，而且還讓隊員接受了採訪，甚至放料出來邀請31位歷代AG超玩會的隊員出席決賽。

在總決賽那天，武漢成了AG超玩會的主場。

當然這一次QGhappy宣傳更上檔次。一方面參與祝福的人員廣，級別高，傳播效果更佳。比如，乒乓世界冠軍張繼科，要顏值有顏值，要影響力有影響力。

一下子就把QGhappy的高度拉升了。還有一些娛樂，競技，體育界的明星，更讓人覺得QGhappy人脈真強大。

在這之前，其它戰隊的競技選手，在接受採訪的時候，都一邊倒地祝福QGhappy！甚至說出了遇上QGhappy，六分投的話。

真是噁心死AG超玩會了。

兩軍交戰，勇者勝。

高手交戰，在於細節。

細節的處理，在於心態。

AG超玩會與QGhappy，都在用自己的方式影響著對方的心態。

這兩支競技選手，實力相當。雖然在打法上有所差別。但都是久經賽場的選手，甚至選手之間也有過私底下的交道。比如，老選手刺痛與老帥，就再熟悉不過了。

這也讓比賽有了更多的看點！在熟悉的人面前，如果戰勝對方，一定有奇招出現。作為月光主教練，對勝利的渴望是人人皆知，而「五冠王」的稱號，對QGhappy來說，也不容有失。只是筆者還挺好奇。

還有幾天時間，雙方就正式開始。

在開打之前，雙方會有什麼樣的宣傳，來影響對方競技選手呢？

不會爆黑料吧？
