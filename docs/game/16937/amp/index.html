DNF：瞎子加強百分之三十，無我波動一躍成神，給瞎子道歉

天天要聞
2019-12-15

無我波動真的來了，策劃還真沒有和你開玩笑，這次國服特色增強針對了好幾個職業，以瞎子的加強幅度最大，其他劍宗，魔道等職業也有百分之十幾的加強，不說幻神一線總是有的。還有那種修復了審判奶開鎚子時無敵三秒的「BUG」的改版。小編是一個瞎子玩家，這次就和大家聊聊這次改版，首先下結論，加強以後瞎子是幻神。

加強位置就在波動刻印上，也就是所謂無我波動，只是沒有改名字而已。改版後的波動刻印是這樣的，二十級精通，三十級上限，二十級以前，每一級提高0.5％的技能攻擊力，二十級以後每一級提高1.5％的技能攻擊力。這是純粹的傷害加強，因為現在的波動刻印是不加技能攻擊力的，多一點技能攻擊力就是多一點傷害，一點折扣都不打。

由於波動刻印是主動技能，所以能吃全國服特色，我們算一下。光環寵物稱號，大地手鐲，護肩腰帶鞋子寶珠，一共七級。也就是非皮甲玩家細節拉滿，應該是27級的波動刻印，實際上的增強幅度為10.5％，加上之前的百分之十，一共20.5％ 的提升，如果是皮甲玩家，那麼就再多三級的波動刻印等級，剛好滿級，實際傷害提升為25％。說好了百分之三十的提升還沒到，沒關係，一覺被動又增強了。

一覺被動現在的成長率為一級1.5％，改版後的成長率為2％，按照細節拉滿的21級一覺被動來算，又提升了百分之十，當然了這個是有基礎的提升，對傷害的實際提升率應該在7％左右。加上波動刻印的提升率，那麼非皮甲玩家傷害加強幅度為28％，皮甲玩家的提升則為33％左右。兩者差了百分之五左右的傷害，但是皮甲相對來說比較弱，也算是一種平衡。

傷害提高了百分之三十是什麼水平？現在打樁比賽中，瞎子的排名是第四，加強百分之三十以後，直接就是第一了，以前是紅十一一千億，改版後紅十就能一千億，這就是加強。

傷害提高了，爆發能力也就提高了，瞎子主要輸出技能能在十秒內放完，除非真的兩秒怪就死了，不然二覺錘下去的時候，就是瞎王爺DPS升天的時候。總而言之，無我波動強無敵，都給瞎子道歉！
