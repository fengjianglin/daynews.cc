DNF12.12韓服更新：魔界後院起火，颶風眼出現，100級呼之欲出

「地下城與勇士之小狐狸君愛談遊戲電競」第七百二十八期《DNF12.12韓服更新：魔界後院起火，颶風眼出現，100級呼之欲出》，隨著韓服12.14冬季發布會的臨近，DNF的新版本、新副本、新職業、新等級上限……等全新的地下城，也是即將到來！此次韓服經歷了近9小時的維護之後，阿拉德大陸也是出現了非常大的變動：比如西海岸、天界、魔界……等區域同時出現了暴風眼，魔界整張地圖爆裂、NPC裝扮和服飾都出現了變動、新的區域雷米迪亞大聖堂……都是一些彩蛋更新。那麼姜策劃能給玩家帶來什麼樣的驚喜呢？老規矩搞快點，直接上本次韓服大更新後的副本截圖展示：

DNF12.12韓服更新：

此次韓服正式服的更新，看似興師動眾——維護了9小時，其實不過是畫餅罷了。而且還是彩蛋類型的更新，並沒有什麼本質的爆料。當然還是就是10幾個職業的技能BUG修復更新，基本上韓服的職業改版和國服已經分道揚鑣了！國服的職業平衡就是特色造神，與強弱無關。倒是韓服的CP系統更加的好玩一點。所以此次更新對於國服而言，就更加的遙遠了。

魔界後院起火，颶風眼出現：

其實不管是過圖動畫，還是地圖特效，都是姜策劃放的煙霧彈，至於12.14放出什麼，真正的更新時間是什麼時候，還都是未知數！韓服更新一個BUG修復版本都要9小時，那麼開發新版本，真的是要到猴年馬月了。姜策劃夏季發布會都沒人手開了，開發100級、新版本……得要多少人手呢！颶風眼在所有區域都有，新的暴風大轉移也要來了。

100級等級上限，呼之欲出：

其實12.14能不能出新職業、100級都是猜測，不過新的副本是必然的。等級上限的提升倒是伴隨著顯得副本。但是為新添團本的開發，就比較難了。現在玩家對團本都審美疲勞了。反而是新的副本比較好開發，提升了等級玩家就要刷圖升級，然後是新裝備——100級的裝備，就呼之欲出了！
