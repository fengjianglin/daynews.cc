[摘要]相信很多小夥伴都是知道的，在峽谷之中，很多輔助英雄也是可以全輸出的，像張飛一樣，給其一個無盡戰刃，也是能打出不俗的傷害，那麼本期筆者就來說一下那些可以出輸出裝備的輔助英雄吧！

首先第一個就是我們的太乙真人了，這個英雄在排位之中雖然很強勢，大招的復活是很多玩家喜愛的，甚至他全法裝的傷害也是真的很爆炸，但是一般情況下沒有肉裝的支撐，你根本就命中不了敵人，因此全輸出也就沒有任何作用了！

第二個則是姜子牙了，這個英雄本來就可以走中路，就算走輔助位置也不用像太乙真人那般，近身靠近敵人，因此全輸出是很正常，這樣雖然會讓己方的抗揍度不夠，但是輸出絕對是很高的！

​

第三個就是莊周了，這個英雄在現版本之中是非常受玩家喜愛的，因為其不吃控制，但是他最好不要全輸出，半輸出就可以了，來上一件痛苦面具，只要疊加其了被動，就能讓脆皮痛不欲生了！

最後一個則是我們的蔡文姬了，剛剛推出新皮膚繁星吟遊的她也是峽谷中火爆的英雄了，但是筆者不建議蔡文姬出輸出裝備，因為很多玩家都戲說要來一個電刀無盡蔡文姬，拿五殺，這是夢想，但不現實，排位中這樣玩基本上是找死！對此，大家有何看法？歡迎討論！
