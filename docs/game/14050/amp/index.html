王者榮耀S17賽季末上分英雄，純凈蒼穹黃忠成真正坦克

天天要聞
2019-12-11

大家好，我是七七。王者榮耀S17賽季已接近尾聲，榮耀戰令也將於12月29日結束，那麼留給玩家上分的時間已經不多了。今天就為大家介紹一下幾個強勢英雄，比如純凈蒼穹黃忠，坦度堪比坦克，感興趣的話一起看看吧！

真正的坦克——黃忠

真正的戰士是狄仁傑，那麼真正的坦克就是黃忠。黃忠雖然是一個脆皮，但是他有點特殊，他是基礎坦度最高的射手，出門血量3226，比3204的白起還要高一些。在團戰的時候，他的2技能可以為他提供一層護盾，而開啟大招後，他還將獲得180-360點的雙抗加成。如果黃忠這時候再開啟純凈蒼穹的50%減傷，配合吸血，即使一身輸出裝，他的硬度也堪比一般的坦克，敵方刺客想要秒他還是有難度的。目前，黃忠的勝率為54.45%，位於射手榜第一，用來上分是不錯的選擇。

銘文推薦：3*無雙、7*禍源、10*奪萃（可帶適量狩獵）、10*鷹眼

出裝推薦：黃忠可以先出速擊之槍，可以觸發被動「精準」，後期可以合成純凈蒼穹。

黃忠開局學習2技能，在技能命中的情況下，1級單挑根本不虛任何射手。還有黃忠在草叢裡開大，敵方看不到大招的指示範圍圈，脆皮進來非死即殘。而黃忠開大的時候可以使用頭像鎖定，先解決敵方輸出，而不要朝著坦克轟。

免費抽獎——鍾馗

鍾馗最具有威懾力還是他的鉤子，在後期滿冷卻縮減的情況下，鉤子的冷卻時間差不多7秒一個，這樣即使命中率不高，可以依靠次數來彌補，一個好鉤直接翻盤。很多玩家鍾馗喜歡法裝，但是肉裝鍾馗收益更高，滿被動的血量可以達到一萬七，面對敵方的刺客、戰士或者坦克也根本不虛，直接站擼，容錯率更高。當然，如果能鉤到敵方C位，那麼敵方直接爆炸。目前鍾馗的勝率為52.95%，位於法師榜第二，僅次於武則天。

銘文推薦：10*調和、10*宿命、10*虛空

出裝推薦：出冷卻縮減與肉裝就對了，提高容錯率。
