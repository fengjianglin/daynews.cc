《荒野大鏢客OL》冬季節日福利上線 玩轉決戰 裝飾營地

2019-12-21 49條評論 1,048次閱讀 69人點贊

為慶祝即將到來的2019聖誕節，《荒野大鏢客：救贖2》在線模式《荒野大鏢客OL》冬季節日活動已經上線。在未來幾周歡慶年末佳節，等待您的不僅是即將到來的冬日天氣，還有限時節日決戰模式、營地和私酒棚屋裝飾等內容。

在旅途中，您可能會聽到酒館中鋼琴手演奏的節慶頌歌，並目睹各處居民快樂地沉浸在節日慶典活動之中。但是，請千萬小心，季節的轉變會為邊境帶來嚴酷的寒潮。據估計，最早將在明天降下大雪。想要準備過冬的玩家們請注意，未來可能還會有暴雪降臨，所以請維持篝火持續燃燒，並確保燉鍋中的食物持續熬煮。

遍布各地的神射手可在各種布滿積雪的血腥決戰模式中考驗他們的手眼協調能力，如黑吃黑、硝煙四起和極速佔領，而PlayStation 4的玩家也將可遊玩獨家節慶版本的快槍決鬥、招招命中和勝者為王。只需在「在線菜單」啟動系列賽即可馬上遊玩，截至1月6日。

為了歡慶佳節，12月23日至25日之間遊玩《荒野大鏢客：救贖2》在線模式的玩家將可獲得特別節日禮箱，裡面滿滿當當地擠滿了各種補給品和好東西，包括：

·坎卜斯版本雙管霰彈槍

·30 發燃燒霰彈

·100 發霰彈槍獨頭彈

·10 份百里香大型獵物肉

·10 份野薄荷肥鳥肉

·20 份胡蘿蔔

·20 份什錦餅乾

·50 份糖果

·10 瓶優質白蘭地

·10 瓶特效神奇補劑

·20 支烈性燃燒瓶

·2x 溪梅

·2x 十大功勞

另外，此期間內購買任何霰彈槍或外套，可享五折優惠，請前往「福利」選項查看。

我們已向在12月13日或之後登錄遊玩的所有玩家發放了一系列節日禮物。玩家們仍有機會領取以下禮物，只需在1月6日前遊玩 《荒野大鏢客：救贖2》在線模式並查看暫停菜單中的「福利」選項，即可兌換以下優惠券：

·3000 俱樂部經驗值

·200 RDO 遊戲幣，供您開始新職業

·1x 免費商販補給獎勵

·1x 免費自選武器部件獎勵

·1x 藏寶圖

·1x 免費私酒販麥芽漿補充獎勵

·以七折價格獲得一個馬廄槽位

在12月31日至1月6日之間遊玩Red Dead在線模式，即可獲得1000經驗值獎勵，Red Dead在線模式中的四種專家職業（私酒販、賞金獵人、商販和收藏家）可各領取此獎勵一次。

玩家可繼續以比原價少 5 根金條的價格獲得肉鋪桌，並在購買中型貨運馬車、賞金獵人版本斯科菲爾德左輪手槍和收藏家版本蘭卡斯特連發步槍時享有七五折優惠，截至1月6日。
