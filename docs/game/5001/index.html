百里玄策缺爆發，雲中君缺位移，11個刺客他最完美

2019-12-03 0條評論 16次閱讀 0人點贊

摘要

娜可露露是遊戲中一個非常強勢的刺客英雄，可以說其有著最高的爆發，當然這個爆發傷害往往與敵人的血量有關，因為露露的傷害主要來源還是百分比的一技能輸出，這就導致露露很容易就可以針對到對手，尤其是露露面對坦克時候秒殺能力非常無解，不過他自身沒有任何的控制能力，非常的吃輔助或者隊友留人。

百里玄策可以說是經歷過大起大落的熱門英雄了，不斷的改動使得他現在其實遠沒有看起來那麼強，而且玄策本身是一個沒有爆發的英雄，需要不斷地利用自身的普攻去打出傷害，尤其是二技能沒有命中敵人的情況下，很容易導致自身的傷害不足，從而出現無法擊殺敵人的目的。而且玄策的二技能沒有命中敵人的話，也會導致自身沒有位移。

雲中君是遊戲中非常熱門的一個刺客英雄。作為裴擒虎與百里玄策的替代品，可以說他的技能機制也是非常優秀的。自身有著被動流血的高傷害以及飛行之後無視地形的能力，可以說是面對敵方有著很好的剋制作用。突如其來的偷襲會讓敵方措手不及，但是自身沒有足夠的爆發能力，只有依賴被動的流血去打傷害。

所有的刺客中只有元歌是最完美的。首先這個英雄共有8個主動技能，但是很多玩家想要了解完這些技能並不是很容易。尤其是元歌的操作，很需要意識與手速的並肩。元歌有著強大的位移能力與突進能力，並且自身的爆發能力非常足，尤其是在傀儡三技能命中敵人的情況下，可以保證自身很好的控制住敵人，並且將傷害打足。
