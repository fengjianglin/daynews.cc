暗黑手游製作人都真香的手游，其發行商與暴雪有不得不說的過去？

2019-12-25 269條評論 2,480次閱讀 1143人點贊

哈嘍小夥伴們好呀，這裡是誠然。談起暴雪遊戲相信很多小夥伴們腦袋裡會浮現那麼幾個關鍵詞：精品，經典，高品質。旗下的遊戲也是涵蓋了多種遊戲品類，比如最近馬上要上線重製版的經典RTS遊戲《魔獸爭霸》，再比如火爆了十多餘年仍然話題火熱的MMORPG遊戲《魔獸世界》，再或者是開創了暗黑裝備系統的RPG遊戲《暗黑破壞神》，真可謂「暴雪出品，必屬精品」！

《暗黑破壞神》是我最喜歡的暴雪遊戲之一，強大的隨機裝備，豐富的詞條選擇加上架構完善的龐大世界觀，吸引了不少熱愛刷刷刷的玩家，它也成為了暴雪即將推出的第一款手游的IP。在暴雪嘉年華上，游研社曾問過《暗黑》手游製作人：「暴雪開發者自己會玩哪些手游？」而製作人Caleb Arseneaux的回答卻讓很多人感到吃驚。

「我在玩《AFK Arena》，很好玩。」製作人Caleb Arseneaux毫不掩飾的回答道。《AFK Arena》到底是一款什麼手游？能讓暗黑手游製作人愛不釋手？原來《AFK Arena》（因即將登陸國服，故以下使用國服版本名稱《劍與遠征》）的發行商莉莉絲遊戲曾經還和暴雪打過官司，而引起官司的手游《刀塔傳奇》也更名為《小冰冰傳奇》，《劍與遠征》就是莉莉絲旗下的作品。

也許正是《刀塔傳奇》經歷過的坎坷讓莉莉絲遊戲真正成長了起來，2019年莉莉絲遊戲就像一匹黑馬直闖入海外收入排行榜，甚至在10月份超越了海外中國發行商三大巨頭：騰訊、網易、FunPlus，拿下第一！

《刀塔傳奇》的誕生正是移動互聯網準備迸發的階段，之後莉莉絲遊戲打出海牌也是正好奠定了如今成績的基礎，而從《萬國覺醒》開始，莉莉絲在海外的SLG品類手游中站穩了腳步，而之後繼續大膽的創新，造就了今天的莉莉絲遊戲。

在我玩過《劍與遠征》之後（國際服，目前國服正在預約。2020年1月8日正式公測），我確實能夠深深的感受到莉莉絲遊戲帶給玩家「不一樣」的東西。在移動互聯網開始迸發之時，那時候的手游似乎會更單純，更快樂一些（我曾經因為抽到了一個摩托坐騎興奮的大喊大叫），而隨著移動端的飛速發展，廠商們看到豐厚的利潤便一擁而上，後來國內手游圈風氣越來越差，國內玩家又無可奈何對國產手游逐漸失去信任。

換皮一波又一波，韭菜割了一茬又一茬。

雖然作為遊戲公司賺錢無可厚非，但是「撈金=毫不注重遊戲體驗」的做法讓玩家們反感至極。而一次偶然的機會接觸到了《AFK Arena（國際服）》，讓我重拾了對手游的信任，而且了解之後發現這竟然還是中國遊戲廠商的作品。至於為什麼喜愛《劍與遠征》，我稍微總結了幾點。

◆其一個人原因。我本人是養成回合制遊戲的死忠粉，我覺得那種一隻手就可以輕鬆遊戲，但是又不缺乏腦力活動的遊戲真的太好玩了。《劍與遠征》作為放置類遊戲，更是解放操作，排兵布陣之後就能自動完成戰鬥。二來面對每日忙碌的生活，便捷的操作會玩得更加輕鬆。

◆其二是畫風。玩家們對一款遊戲的第一印象肯定是這款遊戲的畫風是否滿足自己的喜好和幻想。《劍與遠征》的畫風完全是「老少皆宜」的，雖然我不懂美術，但是我知道這畫風是真的好看就對了。

◆休閑養老。遊戲中的競爭壓力很小，反而遊戲還會用好友機制來減少遊戲壓力，比如可以選擇高戰玩家進行僱傭，幫助你推圖爬塔等等。所以《劍與遠征》是一款真正的休閒遊戲。

◆遊戲資源。資源豐富是其一，作為國內玩家玩國際服最糾結的是因為網路限制而不能充值。但是在我入坑的這段時間沒少抽卡，各種活動副本獎勵多的讓我覺得有點受寵若驚。其次是對比國內某些按照VIP等級送出的獎勵和天價充值梯度中令人瞠目結舌的「贈品」，在《劍與遠征》中氪金玩家和平民玩家之間差的真的只是遊戲時間而已。

每一次談到這款遊戲我都能說上很多，我相信作為這樣一個能夠真正為玩家著想的遊戲廠商，一定還會繼續優化遊戲體驗的。所以暗黑手游的製作人才能夠不避嫌的說他愛玩《AFK Arena》，因為這本來就是一款很好的遊戲。

希望，每一家遊戲廠商都能夠秉承遊戲人的初心。那本期內容就到此結束了，感謝小夥伴們的閱讀，那我們下期內容再見~
