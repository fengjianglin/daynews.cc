「19.11」決戰沙城h5全服白虎(壕區)榜

2019-12-05 0條評論 15次閱讀 0人點贊

摘要

今天更新11月的白虎榜，也稱壕區榜。這個榜單粗略的以目前最新大區風雲榜為基礎，統計所有大區前三名的綜合戰力進行排名，側面體現出各大區之間的實力。本期榜單為了更加直觀較上月做出微調。

今天更新11月的白虎榜，也稱壕區榜。這個榜單粗略的以目前最新大區風雲榜為基礎，統計所有大區前三名的綜合戰力進行排名，側面體現出各大區之間的實力。

本期榜單為了更加直觀較上月做出微調。首先是增加了每個大區的首區開服天數，這樣看數據將更加明顯有衝擊力；其次是將「區代表」的戰力值改為本月前三甲總共增長的戰力總和，可以讓大家對大區實力增加有個對比。如果遇到合大區的話，將以合區後新三甲增長為準。

本期榜單前十名專服進九個大區，混服進一個大區。

榜單第一名：來自皇族空空代表的混服四區，以總戰力34.1億高傲的成績佔據第一名。上期空空的大區還以26.8億排在全服第4，本期能以這麼強勢的勢頭進軍第一是因為上月混服空空大區、血帝大區（上期第25）、李洛克大區（上期第9）三個戰區進行了一次大區合併，原來三個獨立大區的扛把子並成新大區前三。目前以總戰力超第二名4.1億的成績排在第一位，也是混服唯一一個進入前十的大區。

榜單第二名：來自二把刀代表的專服六區，以總戰力30億整的成績佔據第二名。上期以26.5億排在全服第6，本期同樣是因為跟太子爺大區（上期第14）合戰區使得排名大幅度得到提升。當然這裡面也得力於傲視霸主和太子爺出色的1.7億月漲速度，使得大區排名以2000萬的總戰力領先於第三名。

榜單第三名：來自萬人敬仰代表的專服九區，以總戰力跟第二名相差僅0.2億的29.8億成績屈居第三。上月同樣是合大區了，但是由於自身的強大，合了大區之後並未對原大區三甲產生變化。並且同樣以超驚人的月漲速度，超越上期第1名總戰力0.2億，活生生將上期第1名拉到第四的位置。

榜單第四名：來自陸軍代表的專服四區，上期以28.4億的戰力佔據第1名，本期由於各種原因，被擠到本期第四名。但目前距離第五名還是有一段相對安全的距離，本月發力還是有希望可以跟前面兩名拼一拼的。

榜單第五名：來自峰仔的專服一區，總戰力28.8億，已經達到上期第一的戰力值。排名上除了被大區合併拉動增長的一、二名超越之外還是很穩，雖然從上期第3名降至本期第五名，但從增加速度來看，還是領先於下一名。

榜單第六名：接下來是以27.9億的總戰力並列第六的專服五區和七區。本期老馬大區得力於合大區，上月還跟大叔大區差距0.8億的總戰力，本月已打成平手。下個月鹿死誰手就更憑本事了。

榜單第八名：本期第八名總戰力26.5億，依舊是來自神永的專服三區，似乎在一旁看熱鬧般沒有產生排名變化。既沒有捲入前排的鬥爭，也沒有被後面合大區的戰區嚇唬到，月漲速度也是出色。但目前跟前面和後面都還保持一段安全距離，目測下期也不會有排名變化。

榜單第九名：本期最大的黑馬，來自村長的專服十區。同樣得力於合大區，以總戰力24.6億從上期第17名直接殺入到本期第九名。頂替了本期因大區合併消失的李洛克大區的位置，也扼殺了混服一區進入榜單前十的名額。強勢的把上期混服的名額給搶了過來。

榜單第十名：來自算命大師的專服二區，以漲戰力23.7億的成為排在第十位。目前距離下一名僅0.1億的差距，從增長速度上來看，下期極有可能被擠出第十。

下面的就不一一介紹了，各位看官自己看吧。
