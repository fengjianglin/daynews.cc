圈內人吐槽微博杯投票隊，百萬投票都沒進決賽，舒服了

2019-12-07 0條評論 17次閱讀 0人點贊

摘要

絕地求生微博杯的小組賽階段已經告一段落，經過總共6天的激烈角逐，最終16支隊伍從32支隊伍中脫穎而出，打進了最後的總決賽，其中不乏有Ark和RNG這樣小組賽極其亮眼的，也有4AM和SMG這樣低開高走，強勢逆轉的，這一次的決賽，一定會非常精彩！

絕地求生微博杯的小組賽階段已經告一段落，經過總共6天的激烈角逐，最終16支隊伍從32支隊伍中脫穎而出，打進了最後的總決賽，其中不乏有Ark和RNG這樣小組賽極其亮眼的，也有4AM和SMG這樣低開高走，強勢逆轉的，這一次的決賽，一定會非常精彩！

不過很有趣的是，在比賽結束後，某電子競技俱樂部的經理在論壇發帖稱：

100多萬票的一個都沒進決賽，舒服了！

該俱樂部經理吐槽的就是這次微博杯的邀請制下的隊伍，32支隊伍當中，PCM的24支隊伍直邀，5支隊伍是滄海遺珠：意思是微博杯的主辦方認為他們有著很強的實力，但是沒能在直邀名單中，所以主動邀請過來參賽；剩下的3支隊伍由網路投票篩選，最終通過觀眾投票，得票最高的三支隊伍獲得參賽資格。

說起投票，很多觀眾可能就會想到4AM戰隊，由於有著超高的人氣，在PCL勢力榜上，4AM戰隊是國內唯一一個可以破百萬大關的隊伍，但即便是4AM戰隊，破百萬大關也要大半個賽季的時間，但是在這次微博杯的投票活動中，Jstar、NEXT、FTG三支隊伍在短短几天時間內突破百萬大關，情形也很明顯：這個票數有科技幫忙啦！

儘管這三支隊伍沒能進入決賽，但至少這次都還是打出了自己的風格，FTG和Jstar這兩支隊伍都還是有一定實力的，也都成功吃過雞，在11月份的QESL聯賽中，這兩支隊伍還是獲得了冠亞軍，NEXT這支隊伍，觀眾們也很熟悉，打P城的，不管是打SMG還是打RNG，這支隊伍都不落下風，是一支切切實實的打架隊，團戰能力很強。

對於”刷票”，其實按照投票的這些隊伍，大家都沒有人氣，要人氣沒有人氣，要實力也沒有打過多少比賽，所以這個投票方式就很離譜，拼人氣，不可能，拼實力更沒得拼，那這個投票拼什麼呢？不就是拼誰願意花錢去投票？假如自己有一支戰隊，想要去大舞台嶄露頭角，或許也只能使點小手段了，除了這三支隊伍，後面的隊伍也有很多一部分刷了，只不過刷得多少罷了。

當然，就投票這種模式，今後主辦方就最好不要搞了，說好聽點是”眾望所歸”，說的不好聽就是”互相秀下限”，想要參加比賽，還是要切切實實靠硬實力打出來！
