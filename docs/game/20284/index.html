黃金大獎賽：EDG高淘汰吃雞，官方卻慘遭粉絲爆破

2019-12-17 536條評論 2,357次閱讀 180人點贊

吃瓜群眾不吃瓜，電競熱點看百花，絕地求生鬥魚黃金大獎賽激戰正酣，作為國內最早的絕地求生賽事，黃金大獎賽如今已經發展到了S8，給粉絲們上演了一次又一次饕餮盛宴。

目前比賽剛剛結束了第三場艾倫格地圖的爭奪，在這一局比賽中老牌戰隊EDG十殺吃雞，成功將自己推到了積分第五的位置。

這一局比賽航線由地圖上方飛往地圖右下角，EDG跳了自己的常用跳點大小電廠，而第一個圈形也比較眷顧，直接就刷了一個以農場為中心的安全區。

伴隨著安全區的一步步刷新，各支戰隊都在不停掉人，EDG成功運營進了安全區，雖然本身也掉了兩個人，但是也拿到了八個淘汰。

當比賽進行到了決賽圈的時候，場上還剩下四支戰隊，七個生存。當天霸和LYG打起來的時候，EDG戰隊兩名隊員果斷沖了LYG的房區，淘汰LYG的同時形成了最終二打一的局面。

最終憑藉著阿澤的出色發揮，EDG淘汰VC戰隊，成功以十個淘汰拿下本次大獎賽EDG的第一雞。

而在EDG吃雞以後，官方也是迅速發布了本場比賽的戰報，原本應該是很高興的事情，結果官方的這條動態卻慘遭粉絲們的爆破。

而在百花了解過事情的經過之後才知道，原來是為EDG英雄聯盟分部背了鍋。

作為國內老牌的電子競技俱樂部，EDG在LOL中獲得過很多不錯的成績，也吸引了大量的粉絲。但是伴隨著這兩年EDG成績不佳，粉絲們也是積累了滿腔怨氣。

如今LPL的轉會期已經結束，各大戰隊都有人事變動，基本都買進了一些新選手，但是EDG卻遲遲沒有官宣選手名單，很多人都想知道EDG有沒有買進新的選手。

再加上有人爆料，七點鐘EDG會有重大事件官宣，在粉絲們的一遍遍刷新中，沒有等到EDG英雄聯盟分部的官宣，卻等到了EDG吃雞的動態，粉絲們也是情緒爆炸，全都在官方的動態下面留言詢問LOL的相關事件，因此造成了這次官方被爆破的事件。

不管EDG目前LOL的情況如何，吃雞分部本局的發揮還是非常出色的，高大十個的淘汰也展現了老牌電競豪門的實力。讓我們每日17點，聚焦鬥魚，觀看更多黃金大獎賽的精彩戰鬥吧。
