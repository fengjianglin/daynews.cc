《末劍》遊戲測評：不一樣的武器操作向過關遊戲

2019-12-16 2,208條評論 4,104次閱讀 2166人點贊

《末劍》是由騰訊遊戲旗下的光子工作室設計完成的一款遊戲。雖然如今的各類手游大多數情況下如果玩家不氪金想要更好的玩通全部遊戲可能是會有一點困難，但是《末劍》這款遊戲就像是手游界里的一款清流，整個遊戲都不需要玩家氪什麼金，對於我這種貧窮玩家來講這種遊戲是再適合不過的了。

《末劍》作為一款橫向過關遊戲，整款遊戲中一直以武器的操作技能為核心，其實如果更形象化的比喻的話，就是《末劍》這款遊戲和當年火爆一時的《憤怒的小鳥》在遊戲玩法上非常相似，甚至還有網友把《末劍》評為”仙俠版憤怒的小鳥”。其實如果玩家仔細用心體會的話會發現這款遊戲與《憤怒的小鳥》在很多方面都會有著比較大的區別，《末劍》就是《末劍》，它不是任何一款遊戲的替代品。

遊戲的版畫風

【遊戲測評】:8.6分/10分

【遊戲類型】:仙俠風格橫版過關遊戲

基本玩法簡單易上手，延伸操作有點複雜

之前和大家提到，《末劍》這款遊戲在玩法上與《憤怒的小鳥》非常相似，事實上《末劍》的基本玩法也是玩家通過預設瞄準放出自己的仙劍，之後玩家可以通過滑動屏幕來控制仙劍的走位，利用仙劍攻擊敵人或者摧毀敵人的機關。遊戲中總共設置了四個大的關卡，每一個大的關卡中還會設有若干個小的關卡，粗略的計算大概每個大的關卡中會有將近20個小的關卡。其實我感覺這麼多玩下來也是有點累…

努力打怪獸…

這些關卡在遊戲中的難度是逐級遞增的，越往後玩家就會發現自己的仙劍似乎有那麼一點不聽使喚，總是懟不過對面的小兄弟，其實這都是正常現象，要是整個遊戲中沒有那麼一點讓你欲罷不能的關卡那這個遊戲也沒啥意思不是。

遊戲中玩家正在預瞄的畫面

另外每一個大的關卡中最後一關都是大boss戰，boss戰就是有點困難的了，boss們會有多種多樣的攻擊方式，各種招式讓你目不暇接，而且boss不僅僅是一個部位對我們展開進攻，它渾身上下的每一個位置都有可能是攻擊我們的利器，所以boss戰這一塊我感覺還是挺有難度的，雖然比不上《茶杯頭》那種難度級別，但是起碼我是也死了好幾次才過去…我覺得我可能不是手殘…（手動狗頭）

我個人覺得這個星空畫面非常真實

後期隨著故事的發展關卡的深入以及遊戲難度的增大，玩家的招式也會由最初的單純仙劍進攻變得會有更多的招式和技能。後期玩家的技能除了最開始的輕重劍之外還會有多種花招必殺技，比如附魔火和各種必殺技，這些名字聽起來就非常不一般的招式確實非常厲害，這些各具特色靈氣的招式在後期玩家攻擊boss的時候會有非常大的作用，但是如果說到實際體驗以我個人而言我覺得可能每個招式都是在某一方面會非常厲害，但是與此相應的也會有一個弊端，那種十全十美的招式現在看來想在《末劍》實現是有一點困難的。

正在進攻的主角

國風武打畫面，視覺感官效果非常不錯

《末劍》的整體遊戲畫面非常具有中國特色，遊戲畫面一開始出現的少年就給我一種仙骨飄然的感覺。遊戲整體畫面借用了中國的傳統藝術剪紙技術以及木版年畫。在玩遊戲的過程中整個視覺畫面就會給人一種非常精緻細膩同時看起來又沒有那麼繁瑣，整個遊戲畫面非常簡潔，遊戲整體給人帶來的視覺效果非常不錯，感覺就是像是在觀摩一副藝術品，是一種感官上的享受。

版畫風格畫面

沒有台詞的遊戲，配樂效果堪稱完美

《末劍》這款遊戲整個都是沒有什麼台詞的，與正常的手游相比如果沒有什麼台詞可能會感覺非常單調無聊，缺乏感官交互的感覺，但是我在體驗這款遊戲的時候一點也沒有覺得自己很空虛…我覺得這一切都是要歸功於這款遊戲強大的配樂。

遊戲中的每一首配樂都與遊戲場景密切相關，在恰當的時機遊戲中就會出現剛剛好的配樂，在需要你緊張的時候配樂幫你營造緊張的遊戲氛圍，在你休閑的時候讓你放鬆心情。另外最值得一提的是這款遊戲所謂的”古風”不僅僅是一個噱頭，這款遊戲的配樂也會有琴簫的結合以及鼓聲等各種中國傳統樂器，從細節上都在嚴格把控。

夜晚的畫面

【加分項】

完美的配樂

畫風非常養眼

奇妙的中國元素

【減分項】

遊戲玩法缺乏趣味性

各種招式會有重複的感覺

小關卡太多了…

多部位攻擊的boss

總之我認為這款遊戲的優點還是非常多的，我甚至覺得它可以當做中國風遊戲產品的代表作，當然遊戲畫面製作精良並不能將其所有缺點掩蓋，如果玩家的耐心不夠或者容易出現審美疲勞的人建議還是避坑，如果玩家和我一樣非常喜歡中國元素那麼一定不要錯過這款遊戲。
