device：只要我繼續保持這種狀態，也許我就能拿下今年的Top1

2019-12-03 0條評論 15次閱讀 0人點贊

摘要

在Astralis以比分2-1擊敗北美戰隊Liquid贏得本次ECS

在Astralis以比分2-1擊敗北美戰隊Liquid贏得本次ECS S8總決賽後，本次的賽事MVP獲獎者——device接受了來自HLTV的採訪。

採訪內容如下：

Q：最終的決賽就像過山車一樣刺激，Liquid他們首先選擇了Vertigo這張圖，你們當時的第一反應是什麼？

A：「故弄玄虛」

開玩笑的，其實這是我們在Major選這張圖時他們的第一反應。想想還是挺搞笑的，我們兩支隊伍都有能贏下這張圖的能力，不過我個人一開始的狀態並不是很好，遭遇了幀數過低的問題。我在開始時只有190FPS，這可不是我在給自己找借口，裁判也注意到了這一點，他們後來也在嘗試給我解決這個問題。我當時沒能在比賽初期進入狀態也是因為我一直在想這件事，有時候作為一名選手，你很容易被這些事情影響到從而失去專註力。

當然，儘管我們事先就有預料到Vertigo將會是我們要打的三張圖裡的一張，不過我一直以為他們會首選Dust2。Liquid在Vertigo打得也確實不錯，他們掌握著很多穿點的位置。

Q：你們在Vertigo作為進攻方的表現看起來有點不穩定，Liquid總是可以輕鬆拿到許多本不應該取得的連殺，隊伍問題出在哪裡？

A：我認為主要是因為我們打得毫無章法，當時都沒有展開自己熟悉的戰術，就只是在各玩各的。有一回合nitr0在中路拿到了一個四殺，但如果我們能夠守住那個角度，其實是可以擊殺他的。

同時我們輸掉了很多在他們起MP9這把衝鋒槍的回合，而奇怪的是我們只有在拿沙鷹時才能贏下比賽，我都不知道gla1ve在想什麼（笑）。要知道作為隊內的指揮，他需要在比賽中為隊伍指明一條方向，但問題是我們每次拿沙鷹就能贏，但起長槍又經常白給，所以這是一場相當詭異的比賽，完全不像我們之前和Liquid打得那場Vertigo。

Q：在輸掉圖一後，重新在Nuke上找回狀態是不是很難？

A：是的，更別提我們在手槍局裡還輸給了nitr0，這讓我們的心態產生了動搖。不過我們此前就已經遇到過很多次類似的情形了，所以問題並不是那麼嚴重。雖然初期的局面都是他們佔據著優勢，不過gla1ve指揮得非常出色，我們在隨後的比賽中取得了許多有利的開局，這也迫使他們作為防守方只拿被迫前壓外場或是大廳之類的，然後我們就一步步地誘導他們進入到我們所設的陷阱當中。

我覺得我們在進攻方打得很不錯，同時我們也放慢了作為防守方的節奏。之後局面就一點一點的朝著我們傾斜，我拿到了一個雙殺，gla1ve則依靠混煙取得了不少擊殺—就像平常一樣。我認為我們只是發揮了自己的強項，儘管當時隊伍的士氣並沒有那麼足。而在隨後的Dust2里我們則完全地進入到了狀態。

Q：在贏下本次賽事後，隊伍之後還有幾場比賽要打，你覺得Astralis今年總體來講怎麼樣？年初你們贏下了今年的第一場Major，然後Liquid強勢崛起，而現在又繼續輪到你們主宰比賽。你覺得今年仍會是A隊的王朝嗎？還是說已經沒有任何懸念了？

A：接下來我們還有兩場硬仗要打，如果我們都贏下的話，我認為我們絕對可以說今年是屬於A隊的一年，在今年裡我們贏下了兩屆Major，這可是觀眾心中含金量最高的賽事。此外，當我們在2018年建立了屬於自己的王朝時，所有人都想著我們不可能在複製那樣的成功了。不過我覺得我們已經充分證明了A隊仍可以做到這一點，我們晉級到了所有賽事的淘汰賽階段，打出了許多精彩的比賽，並且在上一次ECS和其他次級比賽失利後捲土重來。

這證明了隊伍中的每個人，不論是選手還是工作人員，大家都清楚自己的目標及前進的方向。

Q：這已經是你在ECS總決賽中收穫的第四個MVP獎牌了，是不是這個賽事有什麼特別的地方吸引到你？

A：你說是，那就是。但說實話其實並沒有什麼特別的地方吧，我覺得這場賽事的壓力並沒有那麼大，我們每天只需要打一場比賽，而且如果小組賽打得不錯的話還能休息一天，這樣的賽制就很舒服。其他就可能是一些隨機因素了，我覺得Magisk也完全配得上這個MVP，每當我們贏下一場比賽時，你要知道這不可能僅僅只是一個人的功勞，所以需要給他一點掌聲。不過我希望自己能在下次的ECS繼續獲得MVP，能夠贏下比賽還能獲得MVP，這絕對是一個額外的驚喜。

當然這對於成為年度Top1也是有著加分的，ZywOo的數據很棒，但他在年末並沒有任何的比賽，所以只要我繼續保持這種狀態，也許我就能拿下它。
