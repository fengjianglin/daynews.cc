魔獸世界懷舊服，探索那些常年不為人所知的神秘地點（上篇）

天天要聞
2019-12-18

｛本圖文皆【非正式攻略】原創作品，未經授權嚴禁轉載。｝

眾所周知，《魔獸世界》是一款大型多人在線角色扮演類的遊戲，在這個遊戲里有著完整的歷史背景時間線和豐富細膩的劇情設計，玩家們在魔獸世界中體驗各種冒險，既可以通過完成任務、征服怪物等來獲取升級和裝備，也可以一步步領略世界中的各種風景，探索未知的神秘地方。暴雪設計師在這款遊戲中滿了彩蛋，等著有心人去發現，從魔獸世界60年代到現在，玩家們從來都沒有停止探索。有許多神秘的地方大都隱藏在某些不起眼的地方，今天請跟隨我的腳步，去探索那些常年不為人所知的神秘地點，此文分為上下兩篇。

一、暴風城小巷。

作為聯盟主城之一的暴風城，以其輝煌宏偉的建築而聞名遐邇，在《魔獸世界》電影版本中也有暴風城的一個俯瞰全景特寫。從香草年代到如今正式服，遊戲版本經歷了無數次的更新，許多建築也都做了相應地更改，但是有一個地方被暴雪一直保留到現在。

在暴風城的矮人區，臨近運河岸邊有一排房子，幾乎所有房子的大門都是封閉狀態，而有一處房子是可以進去的，但是當你走到裡面的時候，你會發現，進到了一個顯示為「割喉小巷」的地方，並發現這條小巷道連接著另外一個屋子的後門，在二樓發現一張床鋪，應該是有人居住的。但這裡沒有任何NPC，周圍都是空蕩蕩的，聯想到暴風城特有的「軍情七處」，尤其是這條小巷的名字，聽上去就有故事發生的樣子，這裡或許是某個組織潛伏的據點吧。

二、暴風城一個不知名的副本。

暴風城有一個監獄副本，這個大家都知道，但在監獄的不遠處，又有另外一個被鎖起來的副本，外面有兩個衛兵把手在這裡，但是裡面卻不讓進！有好奇的術士通過基爾羅格之眼溜達到裡面仔細觀察，發現這個副本的構造和監獄副本一模一樣！不知道當初為什麼設計了兩個這樣的副本，難道是想說這個未開放的監獄副本里關押的犯人等級更高嗎？

三、阿拉希高地。

細心的同學觀察到阿拉希高地地圖的最東面，臨近無盡之海之處有一處陸地，但是這個地方到底是哪兒卻無人得知。從辛特蘭的惡齒村一路游泳過來，才能到達這個地方。當你來到這裡的時候，你會發現這裡竟然還藏著一個非常有意思的彩蛋。沒錯！當你來到這裡之後你會發現一個矮人農場，周圍有兩個矮人建築，據說以前在這裡曾居住著兩個矮人平民，一個在外面農場幹活，一個矮人在屋裡睡覺。這裡的確是一個隱居的世外桃源。

四、建設中的奎爾薩拉斯港口。

玩過魔獸世界70級版本的同學，肯定對這裡印象深刻，打開現在懷舊服地圖，在東瘟疫之地最北面就是幽魂之地，也是血精靈的大本營。同樣，這裡也有一個後期做日常的小島——奎爾薩拉斯島，島上有一個大型團本「太陽之井」。在60年代的地圖上，能看到幽魂之地還有奎爾薩拉斯的雛形，但是卻無法過去。所以我直接從提瑞法斯林地出發，沿著海岸線直接游泳到這塊地方，看看發現了什麼~

看著懷舊服這一張非常熟悉的地圖，總共是卡利姆多和東部王國兩部分，沒曾想這裡面隱藏了這麼多小彩蛋，今天就先帶大家探索這幾個地方，下期我們將繼續去探索其它未知領域。
