王師傅發布爐石新卡：臟牧神卡，重現上一回合 玩家：這不是diy？

天天要聞
2019-12-05

摘要

爐石傳說於今日凌晨開始的新卡發布會已經落下帷幕，除此之外，在昨天直播時還發布了一張新卡，號稱是臟牧的最後一塊拼圖？也就是牧師新版本的最後一張職業橙卡：永恆巨龍姆諾茲多。

爐石傳說於今日凌晨開始的新卡發布會已經落下帷幕，除此之外，在昨天直播時還發布了一張新卡，號稱是臟牧的最後一塊拼圖？也就是牧師新版本的最後一張職業橙卡：永恆巨龍姆諾茲多。

這張卡的名字好像非常熟悉，時光巨龍（北京霧霾龍）的名字的是諾茲多姆，非常相像，那麼這兩條同為代表著永恆與時間的巨龍是否有關係呢？

還真是有關係，其實諾茲多姆也就是姆諾茲多，作為時間守護者的他，具有穿越時間線的能力，但是他看到了自己的死亡乃至遙遠的未來中艾澤拉斯將要面對的無盡的磨難與毀滅，但是他無論怎麼做也無法改變自己的宿命和拯救艾澤拉斯，終於陷入瘋狂，亦或者說是他因為發瘋創造出了永恆之龍（自己變成了姆諾茲多），企圖改變宿命包括時間和命運。

我們說回卡牌本身，8費8/8的身材在龍卡中算是標準身材，然而作為牧師的職業橙卡當然要有牧師特色：臟，這張卡牌也是史無前例的效果，不愧是巨龍超模版本中的卡，設計師真是越來越大膽了！

戰吼：使用你的對手上一回合打出的牌，效果和苔絲類似，是不會觸發敵方隨從的戰吼效果的，法術也可以再次釋放但是目標隨機，直傷法術有可能坑到自己哦~

這張牌的最優向使用時機應該是對方使用了只對敵方生效的aoe或者是只拍下了一些身材好的高費隨從，比如敵方法師使用了呼啦、德魯伊的橫掃，或者是拍下了比如巫妖王、佛丁本身有亡語效果和不錯的身材等隨從，這時我們下個姆諾茲多就有相當高的正收益。

不過這種情況更加適合競技場，天梯中面對快攻難有作用，控制卡組之間的互相博弈比較有用，更多的玩法還是要等玩家開發，畢竟這麼髒的卡牌，快樂就完事了！

新版本馬上於12.11日正式上線，與此同時到來的還有一年一度的冬幕節以及大量福利活動，除了新的合集和皮膚之外，呼聲相當高的泰蘭德馬上就要回歸了！泰蘭德首次登場時就是免費贈送給玩家的，只要參與天梯對戰就能獲得，想必回歸也是免費的哦，沒有拿到的玩家可不能再錯過了！
