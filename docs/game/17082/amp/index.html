1、《正當防衛3》推薦指數：9.5

遊戲中玩家控制的主角Rico Rodriguez回到自己的家鄉地中海上的Medici島。這座島嶼被將軍所獨裁統治，Di Ravello要利用島上的資源來統治世界。而Rico的目標是推翻獨裁統治，並成為當地叛軍的首領。

在《正當防衛3》中，你可以不顧物理原則肆意破壞，毫無疑問，《正當防衛3》是系列破壞的最徹底的。在這個巨大的開放世界中，有著數十個城鎮等待你去解放，只是遊戲過程略顯重複，但拋開遊戲都會存在的BUG內容，這一次的瘋狂之旅一定會讓你獲得巨大的爽快感。

2、《狙擊精英4》推薦指數：9

《狙擊精英4》是Rebellion工作室於2017年2月14日發布的第三人稱射擊類遊戲作品。遊戲的戰鬥地點設定在1943年的義大利，玩家要扮演秘密特工卡爾·費爾伯恩(Karl Fairburne)，從法西斯主義的枷鎖中解放義大利，避免盟軍反攻歐洲的計劃泡湯。

《狙擊精英4》是一款聰明，富有策略性的射擊遊戲，它鼓勵玩家們用自己的方式解決處理問題。本作強有力證明了「玩你想玩」這個概念：將納粹的戰爭機器拆的七零八落，用你自己想到的方法來解決自己的問題。遊戲鼓勵你保持警覺，做各種各樣的嘗試。

3、《馬克思佩恩3》推薦指數：9

遊戲主角也就是英文名稱由來的馬克思·佩恩（Max Payne）是名紐約警探，曾經有過一段妻兒被毒癮患者殺死的悲慘過去，為了復仇他曾擔任卧底隻身與龐大黑幫組織對抗，不過內心的悲傷與痛苦持續糾纏著他，讓他深陷酗酒與藥物濫用的泥沼中，是個性格複雜的典型黑色電影角色。

無論選擇什麼遊戲模式，馬克思佩恩3都能帶給我美妙的時光。遊戲性方面並沒有將探索伸向新的領域，但能夠讓系列玩家滿足的親切感和魅力完全得以展現。本作對馬克思佩恩來說是嶄新的一天，也是向偉大過去的歸宿。遊戲採用了一些非常好的嘗試比如和電影結合，讓動作電影愛好者很喜歡比如經典的子彈時間。

4、《量子破碎》推薦指數：9

故事設置在Riverport大學城，玩家將控制傑克·喬伊斯和保羅·瑟蘭尼。這兩人原本是普通人，但在一次導致時間裂縫的意外中，他們獲得了操縱時間的超能力。

在遊戲過程中時間體系會隨時崩潰，導致環境中的時間暫停、斷續、倒流和凍結。玩家在遊戲中會與帝王聯盟展開對抗，喜歡《馬克思.佩恩》中子彈時刻的玩家，可以在放大時間因素的槍戰中尋找新的戰略。除了虛擬的故事情節，遊戲還採用了定製的實景真人敘事模式，根據玩家的不同選擇將產生不同的後果。

5、《合金裝備5：幻痛》推薦指數：10

《合金裝備5：幻痛》是《合金裝備》系列第五代正統續作，在遊戲中，小島秀夫想要讓玩家進一步感到戰爭的殘酷，每次戰鬥都很難全身而退，有些角色可能會在戰鬥結束後失去肢體。本作並非一個巨大的世界，而更像是世界各地許多許多不同的區域結合在一起的世界。遊戲中並非每個地區都充斥著大量的敵人。地圖上會出現各種動物，諸如山羊和綿羊等，看起來都非常真實。

本作首次實現了系列作品向開放式世界的轉化。大大的提升了遊戲的自由度，在繼承了原有的潛魂以外，也讓玩家有更多不同的選擇來進行遊戲。通過對任務的理解選擇適當的裝備、夥伴與路線來完成各種各樣的潛入行動。儘管對於本作是否真的實至名歸這一點眾說紛紜，但是只要玩過的人都能明白這是一款多麼出色的遊戲。
