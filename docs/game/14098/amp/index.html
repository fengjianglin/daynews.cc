3年英雄角色模型前後對比！露娜留出了長發，百萬玩家想讓她回歸

天天要聞
2019-12-11

王者榮耀這款遊戲到現在已經有了4年的時間，遊戲中出現了很多新的角色，這些角色在這麼多年的時間裡，也是在不斷的優化，模樣也是發生了很多的變化，那麼今天我們就來跟舊版和新版對比一下，如圖所示，露娜這個英雄在遊戲中是個優秀的法師刺客，外形更是特別的好看，新版本相信大家都見慣了，那麼小夥伴們是否能回憶出當年的舊版露娜呢？當時的露娜還是短頭髮，目前也是已經變成了長發更加的漂亮了。

后羿這個英雄在遊戲中是一個射手英雄，一直到現在都是深受玩家們的喜歡，從沒有做過冷板凳，這個英雄在這麼多年也是優化了不少，現在版本耳朵后羿也是看起來非常的帥氣，如圖所示，跟舊版的相比較精神了不少，舊版本后羿的臉上多出了很多的冷酷，特別的霸氣。

妲己這個英雄一直是很多玩家都非常喜歡的法師英雄，這個英雄傷害特別的高，外觀也特別的好看，在近些年這個英雄也是優化了很多次，這個英雄的外觀也是沒有特別大的變化，就是小細節方面有很多的改變，現在的新版本妲己比舊版本的妲己看起來更加的成熟和精緻了。

羋月這個英雄在遊戲中也是一個老牌英雄了，這個英雄在曾經版本中不是特別的強勢，但是經過優化和改版之後，也是直接變成了單挑王者，這個英雄的舊版本模型看起來也是特別的漂亮，新版本也是增添了幾分成熟，不過還是很多玩家喜歡舊版本的羋月，百萬玩家都想讓她回歸，這到底是什麼原因呢？歡迎小夥伴們，在下方評論留言，謝謝。
