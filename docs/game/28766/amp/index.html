大司馬鐵粉征戰LPL！WE中單起名馬老師，957：經常聽他講故事

天天要聞
2019-12-21

這裡有最新最全的英雄聯盟資訊，菌菌帶你看不一樣的召喚師峽谷~

12月21號，德瑪西亞杯如期舉辦，除了LPL的十六支戰隊參加之外，還有四支年輕的戰隊參與進來，這次德杯設有小組賽，賽程十分緊湊，同一個時間段將進行兩場比賽，在第三場比賽中，WE對陣LGE戰隊，WE五人全部為新人，對陣LDL的LGE戰隊，WE全程完美配合拿下了LGE，賽後採訪給到了WE中單Teacherma，自稱大司馬粉絲。

WE在經歷了1.0、2.0的時代之後，現在迎來了下一個五年計劃——WE3.0時代，WE在公布大名單的時候，大家就可以發現，基本都是00後的小將，WE最終沒有選擇高價買入強援，而是繼續培養新人，WE應該是整個LPL平均年齡最小的戰隊了，WE的青訓營在國內是很知名的，所以青訓營裡面的選手都不會太差。

在這場比賽中，上單Coten和中單Teacherma表示十分亮眼，阿卡麗開局兩分鐘就單殺了奧拉夫，而Teacherma的夢魘支援以及抓單都十分有衝勁，敢打敢沖，聽解說957說，全新的WE戰隊在打訓練賽十分強，菌菌聽到這句話就開始擔心了，因為打訓練賽強的隊伍，打比賽就會拉跨，但是這支WE沒有讓大家失望，整局完美的配合碾壓了LGE戰隊，拿到了德杯首勝，很多人都以為MVP會給到中單馬老師，但是MVP最終給到了上單阿卡麗，實至名歸！這兩人的表現都很亮眼，所以都是MVP！

而說到中單Teacherma，大家都說這不就是大司馬嗎？大司馬在直播界十分有名，曾經作為CC戰隊的教練，在2013年擊敗了如日中天的OMG戰隊，於是一戰成名，之後也是退役做主播，個人實力還是可以的，國服王者水平，擅長玩螳螂，被玩家稱為國服第一螳螂，一手正方形打野教育很多粉絲，在直播中經常和粉絲分享經驗與心得技巧，也因為其直播節目效果十足，也吸引著很多人前來觀看。

在WE拿下比賽之後，大家都想採訪到中單Teacherma，因為想了解為什麼他會起這個ID，主持人也是很懂，所以採訪到馬老師，主持人問：「為什麼取馬老師這個ID？」馬老師：「之前看大司馬非常多，我覺得他正方形打野比較有意思，於是就取這個ID，因為我是他的粉絲！」看來真的大司馬的粉絲，但是問題來了，看大司馬正方形打野有意思，所以選擇了打中單？不得不說大司馬的影響力來到了LPL，連鐵粉都來征戰LPL了，不過並不是大司馬常玩的打野，不然肯定打不好！

在轉播回到解說席的時候，957也表示今天的馬老師有點正經，因為他在基本是一個非常皮的人，經常講故事，大家在吃飯的時候就開始聽他講，菌菌覺得這不會真的是看大司馬老師長大的吧！

不管是WE的馬老師還是大司馬，WE在這次德杯的表現十分良好，讓粉絲們看到了希望，WE還是相信培養新人不會錯，在WE2.0的時候，隊內也都是新人，最後也是拿到了S7四強，希望在新賽季WE這套陣容能給大家更多驚喜！
