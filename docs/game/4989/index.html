守望先鋒：2500這個分段魚龍混雜，而且炸魚的基本三局遇到兩次

2019-12-03 0條評論 16次閱讀 0人點贊

摘要

守望先鋒我見過某500強源氏炸魚開大團滅對方，然而並不能改變戰場的勝敗，伊利奧斯燈塔，那個源雖然強行續點續了很久。然而還是輸了比賽。

守望先鋒我見過某500強源氏炸魚開大團滅對方，然而並不能改變戰場的勝敗，伊利奧斯燈塔，那個源雖然強行續點續了很久。然而還是輸了比賽。對方平均分1800分，己方也是平均1800，區別就是對方有肉有奶有配合，這還是6v6的情況下，己方雖然有個大神，但不開麥沒有任何配合，仍然被對方同樣1800分有配合的吊打。更不用說1V6了。

OW和CS GO還是有本質區別的。這個遊戲2500分只是一個入門階段，分段上去之後，比如4千以上，沒有合理的戰術配合，是不可能贏的對方的，尤其是這個遊戲可以中途更換角色，所以兩個實力相當的隊伍相遇時，又多了需要職業搭配的快速反應，暴雪遊戲嘛，都是入手簡單精通難。

這遊戲低分段，亂打。高分段水平都差不多，看哪邊決策，配合好。甚至可以說分高點沒有配合跟本玩不了。這也是很多人ow玩家退坑的原因之一。主要不只是技術的問題，我從1000分單排到1900，沒人在你沒carry之前願意換奶，當然t我當。當你c的時候，他們只會更理所應當地玩他們喜歡玩的摸魚，這裡面呆久了誰不暴躁，而且還單排。我承認勝負心有，但是不算重，輸個三四局心態還好，但問題是這種事情是每次玩都發生。而且炸魚的基本三局遇兩，在怎麼贏里誰好運抱到大腿也是一個大因素了。

2500這個分段魚龍混雜，作為一個普通路人，你碰到大神隊友和菜雞隊友，或者對面是大神對手菜雞隊友的概率幾乎是均等的。一個5歲小孩只要鉤得准一點隊友匹配得好一點，或許是可以在這個分段玩耍了，但是還是那句話，只要是單排，不是這個分段的實力，就不會長時間保持在這個分段，這個是所有天梯制競技遊戲的真理。很多人後來玩著玩著就不在意分了，在意分的可能等到了五百強也就去亞服了。當然了我們小於500玩家金武器多的很開心。

我覺得競技遊戲真的是天賦決定一切，守望先鋒我也是2600左右，但是我不會刻意的去上分，都是以玩的心態去大排位能加分最好不加分也是玩，LOL也是遊戲順序就是上野中下輔這樣打一套循環，英雄也是心情隨便選，但是上個白金也是可以的，但同時我也無法做到上王者下青銅。
