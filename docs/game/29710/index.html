王者榮耀體驗服12月23日更新，阿珂隱身已廢

2019-12-24 162條評論 1,441次閱讀 430人點贊

繼23日上午對蒙犽一技能加強、技能描述和顯示優化後，體驗服又於23日晚進行了第二波改動：阿珂平衡性優化、暴君主宰羈絆增強、覺醒之戰暫時關閉

阿珂平衡性調整

此次更新對於阿珂的改動不是針對技能係數以及英雄屬性，而是直接優化了阿珂各個技能的手感。

正式服的阿珂在二技能使用其他技能經常會出現「慣性漂移」的情況，致使走位難以控制，不但很難打出背刺，還會影響自己的生存。而手感優化後的阿珂則幾乎避免了這一情況，現在二技能使用後可完美銜接其他技能了，手感極佳。並且二技能則加入了時間條提醒，方便玩家及時打出減速效果，並且抬手似乎也更加迅速了一些。

大招提示效果出現間隔由0.75秒降低至了0.5秒，也就使得阿珂的切入以及逃跑會變得更加明顯，大招的隱身效果進一步下降了，由來無影去無蹤改為了一路烽火狼煙。

整體來說阿珂的改動算是平衡性調整，以切入難度提升的代價換取了手感的優化，這與蘭陵王之前提升視距並增大敵人感知範圍的改動有異曲同工之妙，整體來說還算得上是加強吧。然而姐夫認為阿珂的強度還是有待提高，畢竟這英雄既吃團隊節奏又吃個人能力，順風局收割有餘而逆風局連帶線都難以做到。

暴君主宰羈絆強化

自上次將先知主宰刷新的時間由5分鐘提前至2分鐘並加入了4分鐘重生間隔後，王者榮耀變成了雪球榮耀，優勢的一方能夠無限的控暴君、主宰壓制經濟經驗和生存空間。

而在12月23日的更新後，暴君以及先知主宰也可享受「魔種的羈絆」效果加持了，一旦擊殺了任意龍坑單位，都會降低對其他所有龍坑單位（先知主宰，則會導致對暴君、黑暗暴君、主宰、風暴龍王）50%的輸出。如此一來優勢的一方控雙龍的能力會下降很多。

同時，擊殺暴君以及先知主宰所提供的經驗也少了很多（實時經驗以及經驗成長均下降不少）；而擊殺主宰先鋒所提供的金幣則增加不少。

結合兩點改動的結果便是優勢方滾雪球能力下降，像之前那樣2分鐘控雙龍，利用等級經濟優勢打壓制逼對面投，或者等10分鐘主宰一波平推的局會越來越少。增加了遊戲的時長以及劣勢方的遊戲體驗。

覺醒之戰暫時關閉

廣受好評的新模式覺醒之戰暫時下架了，據姐夫分析還是由於目前的BUG過多。就我所了解到的便有：單位過少時，鎧皇多重回旋鏢回不上血；呂布一技能範圍時大時小；馬可波羅傷害溢出；貂蟬永久隱身等問題。

待修復完BUG該模式還是會重新上線的，屆時可能還會加入新的英雄或者新的效果，可以期待一波。
