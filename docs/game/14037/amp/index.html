王者模擬戰：刺客陣容的兩個提升方向，精英刺客與進階7刺

天天要聞
2019-12-11

這次來說說王者模擬戰刺客的一個陣容「蜀衛刺」，見下圖，英雄羈絆構成：3蜀國（趙雲、元歌、張飛）+5刺客（百里玄策、李白、趙雲、元歌、司馬懿/裴擒虎）+3衛（玄策、盾山、李信/花木蘭等）。

這個陣容是刺客比較強並且全面的搭配了，但是自從上一次更新後，發現刺客不太行了。上次更新下調了元歌和上官婉兒的傷害，並略微提高了曹操的回血。

元歌和婉兒都是刺客，這對於刺客的陣容肯定是有影響的。另外就是戰士曹操的回血加強，這促使玩精英戰的玩家更多了，而戰士克刺客，戰士強勢了刺客就不好生存。

所以在現在的版本，刺客已經不是一個上分的好選擇了，精英戰、坦射、扶桑法刺都是更好。

那麼刺客就完全不行了嗎？並非如此，最近排位的摸索發現，如果有這兩天賦，那麼刺客還是可以一戰的。這兩天賦一是「羈絆進階」，另一個是「精英」，下面來詳細分析。

1、羈絆進階將蜀衛刺提升為進階7刺

在說羈絆進階之前，先了解下刺客的羈絆。

如圖，刺客的羈絆分三個等級，3刺、5刺、7刺，很明顯，我們常玩的「蜀衛刺」則屬於5刺的級別，這並沒有將刺客的爆發發揮到最大，所以這也是不足之處。

但是如果我們拿到「羈絆進階」天賦就不同了，直接從5刺提高到7刺。

不要小看這個提升的作用，7刺和5刺的差別是非常大的，5刺是30%暴擊率和50%暴擊效果，7刺則是40%暴擊率和80%暴擊效果，單從傷害來說就提高了30%，這才是真正的秒殺後排脆皮。

有了羈絆進階的刺客，就不再是一般般的5刺，而是「進階7刺」。

2、精英將蜀衛刺提升為精英刺客

「精英」和「羈絆進階」同樣是屬於A級天賦，效果是上陣人數減少一個，全體英雄提高22%傷害。

「精英」一般在後期才會來，整個陣容成型需要8個英雄，所以要將自身等級提高到9級。有了精英天賦的加成後，彌補了5刺客爆發的不足，成為精英刺客，十分強力。

看我這把刺客，到最後只剩下2滴血，差點被抬走了，但是來了「精英」天賦給我，直接翻盤吃雞，第二名輸給我的是做扶桑法刺的。

這把刺客質量已經算不錯的了，有三星玄策、趙雲和司馬懿，但是沒來「精英」之前，都是被那個扶桑法刺吊打，來了精英後全部反殺。由此可見精英天賦的作用有多大。

3、嗜血之影只能錦上添花

除了上面說的那兩個很強的A級天賦，還有一個刺客專屬的天賦「嗜血之影」，效果是擊殺敵方英雄後回血13%。

這個屬於B級的天賦，只是擊殺回血但沒有傷害的提升，效果比「羈絆進階」和「精英」差太遠了，這點回血只能是錦上添花，並不能對勝負起到決定性的作用。

大家平時排位可以留意下，凡是能吃雞的刺客陣容，他們的天賦里是不是有「精英」或者「羈絆進階」的其中一個？個人統計看到的都是，沒有這兩天賦的都成了打工仔。

但是天賦技能是系統隨機安排的，玩家不是想有就有，比較需要運氣，所以玩刺客並不穩定，有時候運氣好能吃雞，有時候天賦沒來前四進不去，甚至還可能倒數出局。現在熱門的陣容，例如精英戰士和坦克射手，只要正常運營即可，並不像刺客這麼依靠天賦，所以更能穩定上分。

另外，不知道「精英」和「羈絆進階」能不能都擁有，目前還沒見過同時有這兩天賦的，如果可以的話那就無敵了。
