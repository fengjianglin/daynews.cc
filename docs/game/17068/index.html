王者榮耀：天克李白的英雄，老夫子張良不用說，他才是完美的天克

2019-12-14 1,669條評論 2,721次閱讀 558人點贊

王者榮耀裡面的李白是一名高機動性高爆發的刺客類英雄，也是王者榮耀早期的元老級別英雄之一，因為顏高輸出又強的緣故，所以李白的熱度即便是被削了好幾次也沒有絲毫下降，如果在對戰中對面出現了李白，而且還是有皮膚的李白難免就會心慌慌，李白最怕的是強控加上高爆發，因為李白本身是刺客類英雄，自身的血量並不太厚，所以一旦遇上強控配合高爆發的輸出，基本上是不死也要殘。那麼克制李白的英雄是誰呢？

老夫子

老夫子的單挑能力不用多說，站擼中的王者，並且老夫子會越打越強，被動一旦觸發，附帶真實傷害的普攻，想必任何刺客都招架不住。雖說李白的機動性強，老夫子是很難抓住，但老夫子並不會主動進攻，而是等李白先手，然後大招將其捆住，同樣是幾秒控制。而且處於大招範圍內的李白，輸出將大大降低，這時，李白敢於老夫子來一場1V1單挑賽， 基本上就是暴打場面

虞姬

虞姬是刺客最不喜歡的射手，她的2技能可以解除控制，並且閃避所有物理攻擊，只要預判到李白快要近身時，直接開啟2技能，然後李白的技能就白丟了。如果李白還是選擇追殺的話，虞姬顯然是不懼怕沒有大招的李白，雖然虞姬是射手，但其爆發能力足以和刺客相媲美。大招拉開距離，一技能消耗一波，之後單靠普攻就足以擊敗李白

張良

張良這個英雄許多英雄所懼怕的，而李白也最怕這個英雄，張良的控制可以說是非常到位的，而且技能還有真實傷害，只要被張良控制住的敵人，基本都是被打倒殘血或者直接被擊殺，張良的大招凈化是解不了的，使得張良在法師陣容里顯得非常強大，目前版本的張良表現還是很突出的，關鍵是張良的控制太強大了

宮本武藏

宮本武藏重做之後大招會有所延遲，但是這正是克制李白的絕佳技能， 一但李伯切入，直接大招套在李白身上。就算李白1技能返回原地,宮本武藏也能追蹤過去，大招下的宮本單挑沒有技能的李白，簡直就是爺爺打孫子-樣。李白只能使用2技能或3技能躲過宮本的大招，但是那時間必須算準，鑽石以下的玩家目前還達不到這個水平，而鑽石以上的玩家也不一定就能躲過大招。況且宮本可以等李白釋放2技能和大招再大他啊，真的完克

李白這個英雄的一技能是非常實用的一個技能，突進逃跑都非常的強悍。連續朝指定的方向突進兩次，第三次便會傳送回第一次突進的地方，這個技能是李白的特色技能，不管是打野，GANK，團戰都賦予了李白強大的機動性與追擊能力。所以在團戰看見李白突進進團時最好是瞬間控住李白，團戰被控住的李白可以說是會被秒殺。如果沒有瞬間能控住李白的技能，那麼最好趕快拉開和李白的距離，不要讓他蹭出被動，李白的大招傷害高範圍廣，基本上中招就差不多GG了
