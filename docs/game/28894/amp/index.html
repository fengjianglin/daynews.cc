魔獸世界：休閑玩家AFK的原因是什麼？取消點卡成為關鍵原因

天天要聞
2019-12-21

魔獸世界就是一個整體，有職業玩家，半職業玩家，休閑玩家，工作室，現在改的休閑和半職業大量流失，遊戲總體人數減少。魔獸本來就是靠人來維持的，拋棄一部分人，會導致惡劣的後果。現在暴雪已經嘗到了，月卡的出現導致假的在線人數。以前每個月用的點卡錢和現在月卡錢也差不多，但是開了月卡總有一種上班打卡的感覺，感覺不玩就虧了。更重要的是，任務繁瑣，技能越來越簡單，特殊性越來越少（所有獵人寶寶都一樣了，還有啥收集樂趣，換個模型的事）。

遊戲最開始目標只是裝備，小部分人追求打怪快感，後來加入競技，日常，成就，開始變得忙碌。後來坐騎獲取有點簡單了，再後來，副本變得簡單了，加速了寵物，坐騎很多了，舊世界能單刷了。直到現在，天天刷，副本也刷不完，一看還有那麼多寵物，坐騎，寶寶沒弄到，還有好多成就沒做，日常那麼多，裝備也沒法更新，沒裝備進現場都是被吊打，於是玩家AFK越來越多。

很多玩家AFK的原因，一是月卡，不是差這75塊錢，而是工作忙，花了75隻能玩十幾個小時，心裡不爽。二是pvp越改越沒法玩。三是整個玩家群體沒有激情了，當初天天組織野外團戰，隔三差五攻城，在yy上放著戰歌衝鋒陷陣的日子一去不復返。還有就是遊戲後面真的可玩性都沒了，副本變成跨服是最失敗也是最無奈之舉。60、70級的時候副本，戰場，是多有意思的事情。特別是10人戰歌，阿拉希的語音隊伍真是現在都能回味樂趣。

魔獸世界沒落的根本原因，是我們這群玩家老了，要為了生活，為了家庭去奔波，再也沒有那麼多時間去看攻略，去打木樁練手法，去通宵開荒了。而且這個遊戲沒有當年那麼火，95後都很少接觸這個遊戲了，散失了一群老玩家，新玩家又不像以前那麼多，所以這個遊戲就這樣了。

月卡驅逐了休閑玩家，現在還在天天玩魔獸的，大多都是工作室了，對他們來說成本低了。關於點卡，我有個建議，大家各退一步，恢復點卡但設置期限，從充了點卡的時間算起，半年或一年沒用完的時間就清零，這樣懷舊的可以是不是上來看看，真香玩家可以考慮點卡用完後改月卡，膩了再改成點卡。
