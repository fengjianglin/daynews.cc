龍之神途：細說那麼多年陪伴我們的井中月與裁決之杖

2019-12-03 0條評論 13次閱讀 0人點贊

摘要

1、井中月，可能是陪伴中高級戰士練級時間最長的武器俗稱“菜刀”，高級戰士的“身份武器”之一。正所謂有煉獄想菜刀！多少拿不起裁決的戰士把井中月列為自己傳奇人生的終極目標！“半月彎刀”技能加菜刀曾經是不少沙巴克行會的入會標準之一。

1、井中月，可能是陪伴中高級戰士練級時間最長的武器

俗稱「菜刀」，高級戰士的「身份武器」之一。正所謂有煉獄想菜刀！多少拿不起裁決的戰士把井中月列為自己傳奇人生的終極目標！「半月彎刀」技能加菜刀曾經是不少沙巴克行會的入會標準之一。

井中月的爆率在傳奇中屬於較難水平，以前掉它最多的怪物要數石墓七層的邪惡毒蛇，再次就是極品祖瑪！白野豬掉不掉筆者不知道，筆者是在1.70版本開放前一個月接觸傳奇的，白野豬後來是沒少打，見過掉煉獄和裁決但沒見過掉菜刀的。

升級7次的菜刀

早期「菜刀」的傳奇價位被定在1-2個金磚，屬於那種「只要賺錢就看得到希望」的全民武器，在後期裁決泛濫之後，菜刀價格一路下跌，淪落到幾十萬金幣就能收得到！很多大佬煉裁決時喜歡拿井中月墊刀，一把7-29的井中月其實並不輸裁決了，只是拿在手上給其他人的感覺不同。打架之時法師更怕拿裁決的戰士，直到被撂倒之後點開對方裝備才發現是把極品菜刀！這把武器煉高了真可以說是把扮豬吃虎的裝X利器！

另外，井中月被稱為老傳奇中練級性價比最高武器，用菜刀划出的半月總傷害是很高的！這把武器，基本上是陪伴很多戰士玩家練級時間最長的武器！

2、裁決之杖：從高不可攀到普及爛大街，但多數人的終極還是它

屬性：攻擊0-30，重量80，需要等級30級

江湖俗稱「棒子」，很早的時候，沒人見過這把武器！直到一個戰士與其他人不同，威風凜凜的站在土城，手中拿著裁決的時候，大夥才知道除了「菜刀」和屠龍，還有它這麼個「老二」！

別看它需要等級才30級，但是試問那個年代30級的戰士有幾個拿裁決的！裁決之杖掉落率十分低下，在1.70魔神歸來版本之前，只有極品祖瑪、祖瑪衛隊長和祖瑪教主能掉落裁決！另外還有傳言極少數的白野豬也會掉，能打到的真可以說是萬里挑一的幸運玩家！

裁決的爆率有多低？筆者猜測大約在1/1500-1/2000左右。這是個什麼概念呢？在盛大官服，大約清理5-10次祖瑪七1-3層房間，有可能掉出一根來，這就是它的爆率！

稀缺的產出使得裁決之杖成為了傳奇1.76版本以前的戰鬥次終極武器。那麼它的市場價格有多貴呢？裁決一出，基本上是軟妹幣交易，一把白的裁決交易價格都在幾千大洋，即使是在幻境開放之後，裁決遍地泛濫，它的價格也保持在百元上下。

黃金裁決

由於屠龍寶刀可遇不可求，很多玩家冒著碎掉的危險去沙巴克煉製裁決之杖，升級到0-37的裁決基本上就是戰士的「頂武」了。37裁決價值不菲，能有這麼一把武器，基本上是土豪戰士玩家的最終目標了。

傳奇中戰士武器爭論最多的就是井中月和裁決到底誰性價比最高，這個問題筆者會在後期文章中為大夥剖析，歡迎您繼續關注哦！感謝您的閱讀，更多傳奇資訊、攻略、心情、故事與您分享！
