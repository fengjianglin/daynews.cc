迷你世界：遇到這種餐桌要躲開，地下隱藏機關陷阱，大神都中招

2019-12-10 0條評論 20次閱讀 0人點贊

摘要

迷你世界作為高自由度的遊戲，不會阻止玩家製作任何道具。任憑玩家放飛自己的腦洞，創造屬於自己的道具。有的小夥伴對建築有著濃厚的興趣，他們製作歐式別墅、古典建築群、復原消失的遺迹等等，當然還有的玩家精通跑酷地圖、電路電能等道具。

迷你世界作為高自由度的遊戲，不會阻止玩家製作任何道具。任憑玩家放飛自己的腦洞，創造屬於自己的道具。有的小夥伴對建築有著濃厚的興趣，他們製作歐式別墅、古典建築群、復原消失的遺迹等等，當然還有的玩家精通跑酷地圖、電路電能等道具。這裡不得不提迷你世界另一部分大神選手了，他們最拿手的就是各種各樣的陷阱。

最近腦洞大開的他們再次發明了餐桌陷阱，很多小夥伴看到後會直接坐在餐桌上，但是一旦解除座椅狀態，就會觸發擁有自然武器的陷阱，大神看到都會上當。我們下面來看看這個有趣的陷阱是如何製作的吧，當然學會以後，看到這個類型的餐桌也要躲開走哦！

玩家在地面敲出一大塊5*5同時深五格的空地，如果餐桌的規格較大，也可以擴大空地的面積，接下來就可以倒入灼熱的岩漿了。

為了效率更高，可以利用複製方塊配合岩漿鐵桶，將空地中的所有方塊處都填滿岩漿，岩漿的高度在距離地面一格即可，因為後期還需要關鍵的道具。其實在遊戲中利用自然武器，是最方便快捷的。不需要複雜的電路電能，還能降低玩家的警惕性。目前自然武器最常見的就是岩漿和沙漠的仙人掌了，兩者都會讓玩家受到持續的傷害，並不能像炸藥桶一樣瞬間黑屏。

拿出方塊中的碎石塊覆蓋在岩漿的表面位置，與草塊平齊。這裡解釋一下為什麼要用碎石塊，因為碎石塊擁有坍塌的屬性，一旦石塊上方有道具，碎石塊就會瞬間掉落下去。細沙塊雖然也有坍塌的屬性，但是在這種大面積鋪蓋的環境中，細沙塊無法相互連接。

最後可以拿出地氈修飾下碎石塊，注意鋪設的時候不要踩到碎石塊，不然就會導致之前的努力白費，碎石塊坍塌後，重新擺放碎石塊。在地氈的正上方，拿出白楊桌子和白楊椅子組成一個餐桌組合，看到的小夥伴肯定會上前乘坐。從空中俯視就是一個典型的家庭布置，地上有地毯，地毯上有餐桌餐椅。

​

當玩家坐上去毫髮無損，但是一定取消乘坐狀態，就會瞬間掉落到地面上，從而引起碎石塊坍塌，掉落到最下方的岩漿中，5秒後就會黑屏重生。以後看到這種餐桌餐椅還是離遠點，這個有趣的陷阱你們學會了嗎？歡迎留言討論哦！
