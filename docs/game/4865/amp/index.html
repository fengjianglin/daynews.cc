第二：精品生肖：屬性6-6，3個普通合成

第三：完美生肖，屬性9-9，合成

第四：BUFF生肖，屬性9-9，合成

生肖可以合成，也可以出售，其中BUFF生肖是刷怪PK的神器，因為BUFF生肖會觸發不同的特效BUFF。下面就來看一看各種BUFF生肖的特殊效果。

攻擊的時候，10%減少敵人2000MP，這是要把法師打到沒藍的節奏！

2.蛛網の魂

攻擊時，10%幾率禁錮敵人5秒，法師道士的噩夢，跑不掉啊。

3.禁飛の魂

10%幾率讓敵人無法回程，無法隨機，這就是四戰到底的節奏，飛是不可能飛的，放開一戰吧。

4.嗜血の魂

10%幾率吸收對方2000HP，這就是傳說中的吸血。

5.和平の魂

10%幾率讓敵人處於和平模式9秒，這個很無恥，觸發之後你能打人，別人不能打你啊。

6.煉魔の魂

10%幾率讓敵人紅名，紅名死亡就是大爆，這是PK神器，看誰不爽就暴了他。

9.疾風の魂

10%幾率讓敵人看不到你，持續3秒，這是隱身啊，應該叫隱身の魂的。

10.永生の魂

10%幾率滿血復活，這個就非常犀利了，打著打著，血就滿了。

11.帝王の魂

10%幾率提升1.25倍傷害，這個PK打怪都是非常強力的。

12.冰凍の魂

10%幾率讓敵人冰凍5秒，PK中，5秒的時間基本上就被打殘了。
