冬季冠軍杯預選賽，XQ成為晉級熱門，eStar成為“最差”的KPL戰隊

天天要聞
2019-12-22

陌陌聊王者，給您帶來精彩的冬冠杯資訊！

2019年王者榮耀最後一項重要賽事，冬季冠軍杯在12月21日打開了戰幕。不過首先進行的不是正賽，而是由KPL和KAKPL8支戰隊進行的正賽爭奪賽。

這8支戰隊每一支都有7局比賽，採取的是BO1一局定勝負的機制，雖然兩個賽區各有四支戰隊參與，但是KPL戰隊的優勢還是非常明顯的，因此預選賽也被視為KPL的爭奪戰。

​

首日的賽程，XQ戰隊被安排了三局，是最多比賽的一支戰隊。不過XQ的表現非常出色，三戰取勝，暫時排在了預選賽第一名。

其實XQ是最被看好的一支戰隊，看好他們的網友甚至要比eStar多，畢竟這支XQ在此前剛結束的KPL秋季賽中的表現是有目共睹的。

​

RW俠首日獲得了1勝1負的成績，勝的是KAKPL的戰隊，這是在意料之中。但是他們在和同樣來自西部的TS比賽中，是不敵對手的。

TS同樣有兩局比賽，在戰勝RW俠之後，他們兩戰全勝，暫時排在第二名。不過TS能夠戰勝RW俠還是讓一些網友感到意外。不過TS在季後賽能夠以附加賽的資格一穿三，這實力還是很強的。

​

首日的焦點之戰是在兩支KPL戰隊，Hero久競和eStar的比賽。Hero久競在此前的比賽中輸給了XQ，如果和eStar的比賽他們還輸了的話，那麼他們就非常危險了。

不過這一局Hero久競的幾員老將是紛紛登場，在久誠的帶領下，Hero久競戰勝了eStar，獲得了一個寶貴的積分。

​

在首日的最後一場比賽中，eStar戰勝了KAKPL的一支戰隊，也同樣拿到了一個積分。但是不得不說，在首日的比賽中，4支KPL表現「最差」的是eStar了。

輸給Hero久競沒有得說，但是在和KAKPL的對決中，eStar被對手擊殺次數是所有KPL戰隊最多的。當然，這並不能說明什麼問題，畢竟過程再好看又如何，贏得比賽勝利才是最重要的。是強是弱，預選賽結束那一天就知道了。
