古風玄幻RPG《倩女幽魂2》雙平台正式上線 帶來不同遊戲體驗

2019-12-17 195條評論 4,482次閱讀 3303人點贊

艾肯娛樂12日正式宣布，旗下代理的古風玄幻遊戲《倩女幽魂2》正式於雙平台推出，代言人蔡依林也返台出席《倩女幽魂2》上市記者會，與艾肯娛樂董事長熊玉平共同點燃天燈，象徵《倩女幽魂2》即日起正式與玩家展開新的冒險篇章。

　　官方表示，《倩女幽魂2》除繼承前作《倩女幽魂》的經典要素外也同時進行眾多翻新，像是360度無死角的3D場景，10職業20主角的形象全面煥新，以及由新主角蒲松齡、阿初、嘯天犬所組成的全新主線劇情「青春尋妖天團」，提供玩家3大維度的全新遊戲體驗。

  • 《倩女幽魂2》遊戲中經典門派場景-逍遙觀

  • 《倩女幽魂2》遊戲中的經典門派場景-神機營

　　艾肯娛樂董事長熊玉平表示，「此次《倩女幽魂2》除了帶給玩家最經典的故事氛圍外，更在遊戲中的動畫、美術及故事線上有著全面的進化與革新，相信此次全新世代的遊戲內容，能讓玩家留下美好且深刻的遊戲印象。」

  • 艾肯娛樂董事長熊玉平

第1 維度- 全職業角色形象煥新

　　製作團隊指出，在《倩女幽魂2》中，5大門派里的10種職業共20種角色，全進行了形象上的更新，其中技術上更採用新的PBR渲染技術，讓人物從服裝材質、角色面容到整體身形，全面進行重新塑造提升質感，以更細緻的手法呈現遊戲內角色。

  • 《倩女幽魂2》全職業角色疊代更新-女偃師

  • 《倩女幽魂2》全職業角色疊代更新-男魅者色彩設計

  • 《倩女幽魂2》全職業角色疊代更新-女醫師色彩設計

第2 維度- 360 度全景3D 新世界

　　官方指出，《倩女幽魂2》將提供玩家相較於前作16倍大的地圖幅員，同時更進化成360度無死角的全景3D場景。此外，配合新世代場景升級也同步提升了社群互動系統「角色拍照系統3.0」，其中新增「風景相冊」功能，可自動尋路移動到指定地點；「視線追隨技術」，讓角色目光可始終跟隨鏡頭，拍出更動人的照片；其他如「九宮格」、「可調LOGO位置」等協助工具的增加，大大提升了遊戲內角色拍照及與即時社群分享的遊戲體驗。

  • 《倩女幽魂2》的360度3D全景呈現，讓經典場景更顯生動

  • 《倩女幽魂2》中同步提升角色拍照系統至3.0版本，提升社群互動體驗

第3 維度- 全新主線劇情「青春尋妖天團」正式成團

  • 《倩女幽魂2》推出全新主線劇情，由三位主角蒲松齡、阿初及嘯天犬共同展開

　　營運團隊表示，《倩女幽魂2》中除了「小倩」、「寧采臣」及「姥姥」的經典劇情外，這次更推出全新主線劇情，由「阿初」、「蒲松齡」及「嘯天犬」組成「青春尋妖天團」，與玩家一同展開三界探險。過程中，三人將不斷穿越前世今生、傾聽一篇篇故事，逐步揭開人、鬼、妖三界之中的各種隱秘事件。

  • 《倩女幽魂2》啟動儀式

　　《倩女幽魂2》今日正式上線，官方也同步推出一系列慶祝活動，詳情可至官方網站了解。
