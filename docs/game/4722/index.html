如上圖所示，魔界區域已經發生了一些變化，最明顯的就是地圖背景變暗了，與國服對比尤其明顯，原來泰波爾斯已經破碎了，原來這是次元風暴降臨了，雖然還未正式席捲阿拉德，但快了，12.14發布會就會再次打碎阿拉德了，這就是大轉移2.0！

比如西海岸，天空變暗了！火焰的碎片都開始從天空掉落，反觀國服西海岸的天空依舊是晴空萬里，次元風暴的威力太大，玩家們和NPC又要「搬家」了。但這一次會轉移到哪個世界呢？其實DNF大轉移很不錯，每次都讓玩家都新體驗。

雖然現在只是大轉移2.0的前兆，但玩家們都知道DNF需要重做了，這一點沒有爭議了，超時空漩渦和泰波爾斯耗光了平民的耐心，普雷無力回天，雙倍獎勵的國慶套都涼了，100級出任何團本都難以挽留平民，因為DNF的裝備、職業、副本、製作系統都需要改版，不然泰波爾斯為什麼才上線1年就碎了呢……

根據12月DNF劇情的透露，100級會有三次覺醒，同時會有一大堆使徒作為NPC或者BOSS跟玩家見面，而玩家們想要變強也有索德羅斯等職業鼻祖來領路，之前的爆料，還有超越史詩強度的新裝備，被命名為「起源」，同時12.14的發布會主題是「真·DNF」，看來一切將回歸最初的狀態。

總的來說，次元風暴帶來的大轉移2.0！泰波爾斯率先破碎，接下來就是全副本的重做，然後100級版本正式登場，小夥伴們做好準備了嗎。
