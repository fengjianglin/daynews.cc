陰陽師：痒痒鼠百聞牌聯動獎勵，我寮再添一隻豬，有望獲得頭像框

2019-12-09

摘要

《陰陽師：百聞牌》的降臨和全平台開服前的造勢不可謂是不火熱的，微博開屏廣告位置也買了，熱搜甚至佔了第三，話題度很充足。所以當知道《陰陽師》將會和《百聞牌》進行聯動的時候，似乎也就成了理所當然。

《陰陽師：百聞牌》的降臨和全平台開服前的造勢不可謂是不火熱的，微博開屏廣告位置也買了，熱搜甚至佔了第三，話題度很充足。所以當知道《陰陽師》將會和《百聞牌》進行聯動的時候，似乎也就成了理所當然。

值得一提的是，前不久《妖怪屋》剛剛發行，相比起《百聞牌》的造勢，《妖怪屋》可以算得上是默默無聞。她甚至都沒有和大哥痒痒鼠進行聯動，要知道，決戰平安京啊、百聞牌都和《陰陽師》聯動了，這樣一來彷彿《妖怪屋》不是親生的。

我們也知道，《陰陽師》每次與自己的遊戲IP聯動，如果完成聯動任務，就可以獲得聯動獎勵。猶記得當初《決戰平安京》與《陰陽師》的聯動獎勵有藍票、白蛋、頭像框以及雪女那款霜雪連蓮的皮膚。當初甚至還因為雪女這款皮膚的建模做得不甚理想而在平安京引起了一場混戰，最後因玩家的舌燦蓮花而勝利。

而這次，兩方聯動依舊是有獎勵的，目前沒有公開，不過已知的獎勵是又一隻小豬——小貓。這小貓可是很有來頭，和《百聞牌》遊戲內的指引形象”三目”擁有同樣的劉海貼，並且全身毛色和”三目”這條狗的花紋、顏色一模一樣，可以成為三目的貓版。

我們知道，在遊戲中，每隻動物每天的飽食度如果足夠，那是每天都能領取六分鐘的加成獎勵的，有100%金幣、經驗加成獎勵，也有覺醒、御魂加成獎勵。但是也有例外，比如那隻紅尾巴小白狗，就不能領取加成，當然，好的是它每天也並沒有吃我什麼東西。如果光吃不做，那就很遺憾了，我們寮不養這種。

所以，如果三目貓來到庭院里，儘管暫時還不知道它是否需要進食，也是否有每天的加成獎勵，你會選擇領養它嗎？

另外，頭像框也是可以期待的，當初《決戰平安京》與《陰陽師》聯動的時候，只要在《決戰平安京》中玩遊戲拿到一定的成就，在《陰陽師》中就能領取一款聯動頭像框。痒痒鼠與自己家的遊戲聯動情況並不多，但是根據已有的經驗，聯動必不可少頭像框。

誰叫陰陽師不僅是一款卡牌RPG遊戲，還是一款養成收集類遊戲呢？

此外，聯動皮膚也是可能的，至少上次《決戰平安京》是有雪女的聯動皮膚的。這次《百聞牌》的製作並不比決京低，而且宣傳力度也並不小，可以說聯動獎勵應該比決京較之更加豐富。

那麼這次的聯動皮膚可能會是誰的呢？小編認為，大概率應該是以津真天的，首先聯動皮膚不可能選擇SSR，如果讓你平白獲得一款SSR皮膚，不管付出的是什麼，只要不是錢，策劃組都自認為是虧的，所以SSR；聯動皮膚大概不用想了。

而在《百聞牌》中，遊戲內登錄初送的卡牌便是以津真天的，就如同《陰陽師》登錄就送雪女一樣，以津真天可能就是《百聞牌》中的狗糧隊長，帶領其他卡牌一點點成長，所以大概率，如果有皮膚的話，應該是以津真天的新皮膚，個人還是非常期待的。

但是，已經確認的信息除了三目貓便沒有更多了，想要知道確切的消息還是只能等到12日全平台開服了！
