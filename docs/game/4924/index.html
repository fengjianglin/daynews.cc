LOL吧熱議：新版本AP比ADC還慘？上單勝率爆表 兄貴聯盟回來了

隨著LOLS10新賽季的到來，整個LOL在遊戲地形，天賦符文和大小龍屬性都迎來巨大的調整，然而本以為玩家能夠享受到更加好的遊戲體驗，沒想到在S9版弱勢的ADC在S10變得更加慘不忍睹，而傳統AP法師也因為穿甲武器的加強變得太過弱勢，在貼吧里一場AP和ADC誰才是S10最卑微的位置討論就此展開！

由於在S10賽季許多天賦進行了調整，特別是征服者迎來了重做，征服者變態的攻擊加成和吸血能力，外加血色之刃的出現，導致中路變成阿卡麗、女刀、男刀等AD刺客的天下了，傳統AP在遇到他們時對線就很容易會被打爆，況且這些刺客推完線遊走能力還強，在誰先控到四條小龍拿到龍魂誰勝利的快節奏版本中，需要裝備支持的傳統AP位置無疑是個累贅，而ADC好歹還有輔助的保護，而AP則是既容易炸線又會因沒有節奏被隊友噴，遊戲里毫無地位可言！傳統AP也就剩下肉瑞茲以及球女能玩玩，剩下的位置全都被擠到輔助位置上去了！這無疑對喜愛AP中路的玩家是一大毀滅性的打擊！

當然也有ADC玩家們表示不服，認為傳統AP還能擁技能清完線，出個肉裝中亞混一混到後期補充點AP傷害就可以，而ADC才是真正的尷尬，前中期裝備成型太慢，真空期太長並且遇到刺客就會被秒殺，毫無操作空間不如直接放鍵盤，而且這個版本上單太猛了，自己裝備成型後被半肉戰士幾下給秒殺了，真不知道ADC設計出來是幹什麼的。可見S10版本的傳統AP和ADC面臨一樣的問題，那就是裝備成型太慢，前中期戰鬥力太弱，並且中後期Carry能力又有限，這次ADC和傳統AP比慘大會各有各的道理實在是難出勝負！

既然AP和ADC都吐槽勝率太低，地位太卑微那麼就讓我們就來看看新版本究竟哪個位置佔據勝率榜的前列，看看誰才是真正的罪魁禍首！

果然不看不知道一看嚇一跳，S10版本中排位勝率前五被上單全部包攬，除了蛇女外其他都是上單位置，而且都是重裝戰士，這些重裝戰士在搭配征服者後，只需要出一個黑切或者三項輸出就高到爆炸，傳統AP和ADC在遇到他們是跑也跑不掉，打也打不過，一旦被黏到迎接的就只有死亡，兄貴版本的降臨無疑讓ADC和AP們哀聲！

不知道各位LPL玩家們覺得現在版本ADC和AP哪個位置更尷尬呢，除了坦克上單過於強勢外S10還有哪些平衡問題影響了你的遊戲體驗那？記得點贊並且和玩家們分享一下你的觀點！
