如果刪掉所有英雄大招，典韋要哭，露娜得“笑”,射手或成最大贏家

天天要聞
2019-12-18

此文轉自：王者小信

王者峽谷中的眾多英雄，有不少靠著大招橫行霸道的，比如鎧皇，宮本等。但若將他們的大招全部刪除，哪些英雄最受震動？小信私以為典韋要哭，露娜得「笑」，射手或將成峽谷最大贏家。

典韋

典韋的大招能對鎖定目標發起跳斬，會造成百分比真實傷害和50%減速效果，再肉的坦克在他面前都不夠看。若沒有了大招，典韋估計得哭，不僅站擼王位置不保，從此還得繞著坦克走。

鎧爹

鎧是一個非常依賴大招的英雄。開了大招的鎧上去就是干，氣勢剛猛，猶如神助，一刀一個脆皮。但若沒了大招，鎧爹變鎧兒，也就只能打打小兵了。

露娜

峽谷秀兒的代表英雄。露娜大招可以無限刷新的機制雖然很好玩，但可將她忙壞了。若沒了大招，露娜將會變得平凡。不過，小信想，露娜應該不會介意的，畢竟再也不用被逼著練月下無限連了。終於能放鬆了，做夢都得「笑」吧！

妲己

妲己一直以簡單粗暴聞名峽谷，大招的瞬間爆發傷害特別高，是許多英雄都懼怕的技能。但如沒了大招，小妲己的人生將從此失去色彩，根本不能好好玩耍了。

宮本

宮本武士被天美一削再削，終於成了現在的超級兵。雖然如此，但其大招依舊人見人怕。如沒了大招，超級兵將變普通小兵，再也不能雄起了。

東皇

大招吸吸吸，專治花里胡哨。東皇的大招可是他能存在的根本意義。若沒了大招，他雖然依舊能屹立峽谷，但總歸失去了特色。

射手

在這裡，小信將射手劃為一個類別。因為許多射手都是靠平A攻擊打輸出的，大招對他們起到的是加成的作用，而不是決定性作用。比如魯班、后羿、孫尚香、伽羅等。就算沒有大招，他們的實力依然不容小覷，下降真心不多。

這種情況下，所有英雄大招皆被刪除，受影響最小的射手群體可不就得成為峽谷最大贏家嗎？對此，不知小夥伴們怎麼看？歡迎留言！
