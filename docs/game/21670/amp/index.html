守望先鋒：2位宗師VS 6位青銅，團隊合作誰會取得勝利

天天要聞
2019-12-18

守望先鋒是一款競技類型的遊戲，竟然是競技類型的遊戲肯定是有天梯排位的，同樣的也是分為好幾個段位，青銅、白銀、黃金、白金、鑽石、大師、宗師最後就是500強。

想上分其實並不簡單，要麼就是非常有天賦，或者就是從低分局中慢慢的往上爬，可是後者是一個非常漫長的過程，想要玩好一個英雄或者成為一個自由人，那麼就必須得要了解這個英雄怎麼玩，小編就是一名自由人，常年混居在鑽石和大師分段，什麼位置都會，什麼英雄都會，可是就是不能上分，因為自由人需要在適合自己的分段里慢慢磨練，所以自由人上分是比較困難的。但是有一些人就是專精那一個英雄，把英雄的所以技能的作用、CD都摸得一清二楚，這樣在加上隊員的配合，上分會非常容易。

那麼我們來猜測一下，2位宗師和6位青銅PK，就以團隊死斗的方式進行比較，到底誰會獲得勝利？有一句俗話說得好：「三個臭皮匠，頂個諸葛亮。」那麼6個青銅能不能頂2個宗師呢？

外國有一位網友就舉行了這一次比賽，請來了6位常在青銅分段的玩家，和兩位常在宗師分段的玩家，進行了一次團隊死斗。比賽分為兩場第一場是在艾興瓦爾德進行，陣容不定，雙方玩家可以隨意更換英雄。宗師方先是以雙飛陣容與青銅方的222陣容進行PK宗師方以巨大的有限拿到了一個較高的人頭比，然後宗師方換出了一C一T，拿下了第一局比賽。第二局是在好萊塢進行，宗師方還是一C一T進行比賽青銅方依舊是以222陣容。開始雙方的比分有些拉不開差距，但是領先的還是宗師方，最後還是以宗師方獲得勝利。

為什麼宗師方能獲得兩次的勝利？因為宗師方有團隊合作的意識，在第一局的時候宗師方之所以可以拉開很大的人頭比，是因為青銅方的人都非常亂，都不在一起，導致青銅方被亂殺。第二局青銅方意識到了這一點，開始抱團走，可是實力不如宗師方還是被拿下了第二局，但是第二局的人頭優勢就沒有那麼大。

所以宗師和青銅的對局是以宗師的勝利收場，6位青銅還是敵不過2位宗師。這個結果是不是大家心裡所想的結果呢？歡迎大家在下方評論、留言。
