最高攜帶2千金、間隔90天，《魔獸世界》懷舊官方轉服方案公布

天天要聞
2019-12-18

《魔獸世界》懷舊服國服付費轉服方案於12月18日出爐，網易暴雪放出了官方公告，確定將於12月19日維護之後正式開啟服務。此前關於付轉僅針對外服，不涉及國服的傳言也告破。

下面是詳細的付費轉服條件：

小編進行逐一點評。首先，這次並沒有公布付費轉服單個角色的價格，很多人擔心開付轉之後出現大量鬼服情況，以及不平衡的伺服器更加不平衡，小編覺得是多慮了。因為畢竟轉服是有成本的，即使出現極限情況，也可以通過漲價來進行調控。

限制角色等級

10級非常容易，但轉入的是單個角色而不是賬號，若有多個小號，或者單個賬號有多個60級角色，需要一個個轉，無論等級多少，價格同樣。

限制轉移時間

單個賬號，一次轉移之後，下次轉移至少要等90天!這是整個方案關鍵一點。唯一問題是不知是否可以批量轉移，例如我單個賬號下有多個角色，是否可以一起轉，不然每次轉一個角色，要等到天荒地老。

限制攜帶金幣

最關鍵的一項條款，目的限制工作室，以及那些黑人品的跑路人士。60級最多可攜帶2000金幣，對於目前動輒幾萬金消費，以及工作室的囤積來講，算非常少了。但有一點，轉服沒有限制物資，如將金幣轉成一些高價物資，如黑蓮花、奧金錠，甚至T1散件，都能一定程度規避，就看背包容量了。

限制伺服器類型

明面上不能從PVE服轉PVP服，但據大家討論，以及過往經驗，實際上PVP服也不能轉PVE服。總之只能轉到對應類型的伺服器。

限制角色陣營

不能雙陣營一直是魔獸官服的特點，雖然玩家可以創建不同的賬號輕鬆規避。

限制榮譽點數

榮譽點數僅限當周，意義不大。目前看來，在某村服(人口很少的服)打個高督或大元帥，然後轉熱服，賺中間價值差，還是有可能的。
