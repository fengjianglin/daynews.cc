各位好呀，我是不二貓。今天爐石一個大更新炸出了賊多的玩家，特別是這個全新的貨幣系統，可謂是一石激起千層浪哇。所以想要和大家就這件事情，聊聊我的看法。

●關於全新的貨幣

首先各位要知道，這個全新的貨幣系統屬於國服特色，外服是沒有這個東西的，全新的貨幣名為符文石和奧數寶珠，具體的價格是1RMB=1戰網點=1符文石=10奧數寶珠。

了解完這一點，我們在來說另一件事情，從年中開始暴雪就想要將爐石降低到E10+級，適合10歲以上玩家的標準。為此修改了大量的卡牌原畫，刪除了大部分卡牌原畫上的血腥元素，力圖為爐石帶來更多的新鮮血液，吸引更多的青少年來遊玩。

但是好巧不巧的，我國的中宣部在11月份出台了一個針對網路遊戲的專項文件，這個文件主要對未成年人遊玩遊戲時設置了大量的限制，其中關於費用收取問題有個很明確的收費標準，如下：

●費用收取

◆8歲以下用戶不得收費；

◆8歲~16歲之間的用戶，每周累計付費<=50元，每月累計<=200元；

◆16歲~18歲之間的用戶，每周累計付費<=100元，每月累計<=400元。

並且這個規定需要在今年年底，也就是12月31日前整改完成，當初我還在想爐石該怎麼辦，畢竟國家的規定必須執行，但上有政策下有對策，那這個就是貨幣系統就是非常完美的對策啦。

328元的預購禮包肯定是不會變的，但玩家尤其是未成年的玩家是可以通過每月積攢一定符文石的方式，在不違反國家規定的同時去購買卡包，你可能覺得戰網點是一個意思，這個貨幣系統替換的是遊戲內的直衝。和戰網點並不衝突。

這個做法和購買奧術之塵送卡包其實是一個意思。

遊戲內現在還是RMB直接購買

另一方面還有就是黃金線上賽的問題，黃金線上賽這個比賽也是屬於國服特色，官方為了推這個全民比賽可以說是煞費苦心了，拿出了很多福利。上一次的線上賽甚至打出了參賽就送戰網點的口號，說真的官方什麼時候宣布報名就送戰網點我都不奇怪。

但是戰網點這個東西是暴雪全家桶通用的。很多雙修甚至多修玩家，從爐石這比賽拿到的戰網點，跑去用到其他暴雪遊戲里也不是不在少數。長久下來肯定是個問題，但這個貨幣系統就是一個全新的解決辦法，比賽獎勵代替掉原本的戰網點，只限於在爐石中使用，或許下一次的黃金線上賽就會直接發符文石了。

說完了個人的猜想，再聊聊我的擔憂吧。

關於內置商場我最擔心的就是移動端適配的問題，在我的印象中，近半年來爐石每次版本的大更新都會導致移動端或多或少的出現一些問題，全新加入的這個商場是否會對移動端造成更大的負擔？如果真的出現BUG暴雪又該如何解決？這對T5來說都是很大的考驗。

●結語

當然以上只是我的一些個人猜想，各位看著樂呵就行，畢竟咱們也改變不了啥，樂觀的接受就行。
