燃燒意志：新活動「鑄造」攜100級新套裝上線，裝備工坊的變態版

天天要聞
2019-12-05

摘要

諸君安好，霧夏菌報道。

本周四的活動也已經更新了，其中有一項新活動，叫做【鑄造】，帶著100級的新套裝出現了，具體如何一起來了解一下。

1、福利活動

這方面就兩個，一個是秘傳、突破和訓練的雙倍掉落，另一個就是許願瓶活動。新版的許願瓶加入了很多資源，現在是非常給力的，包括黃金戰魂和搜魂券，都有機會拿到。

不過這兩個活動出現的頻率太頻繁了，導致很多玩家有點麻木了，但資源畢竟在那裡，該刷還是要刷的。

2、氪金活動

這裡主要包括了累計充值和限時禮包，以及最新的【鑄造活動】。之前有提過，很多玩家會存彩鑽，然後等著這類累計消費活動，一次性刷一波。那麼這次就可以考慮一下，彩鑽的消耗可以刷裝備，也可以買寶石袋。

此外說一下這個新陣型【冥想陣】，提供的是攻擊和爆傷加成，但是他那個站位，在PVP中如果沒有喬茲，也是屬於會被紅髮吊著錘的類型。

至於限時禮包，沒啥新奇的，也沒什麼優惠，不是特別急切的玩家，沒必要考慮。

3、鑄造新活動

重點說一下這個鑄造活動，新出的。

無數玩家苦等的【裝備工坊】還是不給，反而出了一個類似於變態版的鑄造活動。為什麼說這貨變態，因為這裡不提供任何的裝備分解，但卻具備了裝備工坊的抽裝備撈錢的特性。至於出貨率，參考之前的裝備工坊。

新的套裝就是100級的暴徒，套裝效果是提升15%的屬傷和一定概率讓屬傷傷害翻倍。詞條方面戒指和項鏈都有固定的速度屬性。200次的抽取可以獲得完整的4件套，也就是4萬彩鑽，4千塊錢。不是很推薦這裝備，這套裝備更適合屬傷角色，而目前可用的主流屬傷角色有新索隆和小馬哥。

但PVP速度的重要性太高了，所以如果入手，很可能會把速度拉下來，除非4件裝備都有速度詞條，且項鏈和戒指也都是雙速的（衣服有隨機速度）。此外就算運氣好能出來不少英雄套裝，還有鑒定這一關，因此整體的性價比太低了。

菌說

一些屬傷爆炸的角色，如果用上暴徒套裝，可能會有不錯的表現，譬如變身後的小馬哥，或者有玩家還想玩的一拳流艾斯。這要是錘出來一波屬傷翻倍，傷害量也是非常可觀的。

但是這價格太貴了，而且出貨率也一如既往的低。幾千彩鑽砸進去可能連個水花都沒有，所以一定要慎重，這貨比神秘商店還要殘忍。
