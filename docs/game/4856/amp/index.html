工作室自動腳本越發猖獗！魔獸懷舊服玩家舉報無果，大家說怎麼辦

2019-12-04

摘要

60懷舊排隊、太累？想玩70版TBC懷舊服、80版巫妖王之怒？私信發“懷舊服”，即為你推薦！

60懷舊排隊、太累？想玩70版TBC懷舊服、80版巫妖王之怒？私信發「懷舊服」，即為你推薦！

懷舊服開服三月，想必大家對遊戲中的工作室一定印象十分深刻，因為你幾乎在所有能打錢的資源點見到他們的小號，而工作室腳本這麼猖獗似乎也見不到網易與暴雪用些手段，玩家們紛紛表示心寒，先來感受一下下面這位玩家經歷吧：

講真的現在腳本猖狂很大原因就是因為網易的不作為，地圖小號大部分都是腳本升級，現在採集都有了，一個獵人，固定路線打怪剝皮，採藥，所有路線一次不變，打怪拾取剝皮以後退回原地，再繼續巡邏，每天24小時在線，舉報三天無結果，這是第四天，真心沒有見過這麼垃圾的運營商。這麼說吧，憑啥我辛苦打怪練級，人家輕鬆五開自動練五個？憑啥我辛苦採集，人家輕鬆十幾開採集，害的我千金馬都買不到？

網易無作為，不會封號的

這種掛機也不會太影響你的體驗，也就是金貶值快一點。我正式服自己寫腳本自動拍戰場定時跳一下掛機升級都沒事。因為這東西就是模擬鍵盤後台檢測不出來。按鍵精靈封號也只是掃描進程名而已。

得了吧，燃燒平原打黑龍，艾薩拉打藍龍，10多個獵人，都是差不多的名字，我舉報了1個月了，屁用沒有，很明顯的腳本模式。狗城唯一的黑點就是伺服器差，不過現在看來網易的伺服器也沒好到哪去，周二CD更新，黑石副本卡個逼形。拋開伺服器問題，網易就是一坨。

你們在講工作室的存在是合理的？純手動，你隨意，問題是這個年頭那個遊戲會有純手動的工作室？好聽點叫輔助腳本，直接點就是：自動打怪升級採集扒皮釣魚不下線防舉報懷舊魔獸外掛。

公共資源全被工作室佔用了，還問損害我們什麼利益了，他就是工作室吧。公共資源全被佔用，一些本來可以自給自足的東西，全部要買，金價掉物價漲，說到底還是工作室賺錢，你買金然後去買工作室的東西，然後工作室賺的金又賣給你，還有人說好的。

國內環境一直如此

輔助就是工作室··不是工作室開輔助！如果一個工作室·全都是人工玩··最多·雙開吧~！！~那你平啥說人家是工作室·就是因為湊一起玩就叫工作室？明顯問題就在腳本輔助上邊~！

月卡到底便宜了誰。仔細想一下也不怪工作室，畢竟人口這麼多，都要生存。但是破壞遊戲環境不對。我們區榮譽開了後 經常有人對刷….舉報沒人管，辛辛苦苦幾小時打個一萬的榮譽，別人一小時就刷到了。

知道隔壁逆水寒不？官方做的多開同步器價格最高，但是長期有效永不凍結永不封號，別家的三天兩頭出事抓到就封。都是網易的，雖然不是一個工作室，自己掂量吧。

為啥一說金掉價了就有這麼多人腦充血了？金的價錢高低，和懷舊玩家有毛的關係，在意金的價格的人，也就那幾種人吧？腳本號採集，一般AH放東西都會比最便宜的少那麼一點點銅幣，一般都會直接大幅度放一個（比如放個1-3顆草藥）來壓價，引誘腳本號，成組的放上放，這就可以低價收貨，做成藥水，降低成本。

魔獸世界腳本與工作室一直個困擾玩家與暴雪的難題，怎麼打擊工作室，如何才能做到讓玩家與工作室之間保持相對的平衡？你怎麼看呢？
