輸了比賽教練背鍋？真實的職業賽場上，教練BP究竟是否全說了算

2019-12-12 942條評論 2,668次閱讀 1113人點贊

時隔多日，小皮終於又回來了，最近閑來無事，看了許多之前世界賽的比賽。在觀看之前比賽錄像時，小皮發現了一個讓人感到很無語的事，幾乎LPL輸掉的每一場比賽中，都會有一些粉絲刷教練BP的問題，似乎在他們的眼裡，輸掉比賽的原因就是教練的BP做得不夠好，那麼問題來了：在職業賽場上，戰隊的BP是否由教練一人說了算呢？

其實關於這個問題，顯然是不可能的，教練雖然負責BP，但是也不能完全由自己的個人喜好，為場上的職業選手選英雄。畢竟受限於場上選手的個人英雄池，以及整個戰隊的風格。教練在BP時，必須考慮到這些問題。

一直以來，在比賽直播現場都會給大家一個錯覺：BP是教練的臨場發揮。其實真實的BP，選手們和教練已經在場下有一個大概的雛形了，也就是說，每一場的BP都不是臨場發揮，而是在場下已經做好了各種各樣的應對。而那些在觀眾們眼裡並不看好的陣容，也是因為你不夠了解這支戰隊，很多在你看來不行的陣容，卻是最適合這支隊伍的BP。

很多觀眾看到這裡一定會質疑，很多戰隊彷彿「鐵頭娃」一樣，即使輸了前幾場，為啥後幾場一樣會選出同樣的陣容呢？其實這就是每支戰隊的理解不同，在他們的理解中，拿到這個陣容是很適合他們贏下比賽的，輸了是因為沒有打出效果。所以大家以後再看比賽的時候，也一定要理性，輸了比賽教練可能會有問題，但絕對不是最大的問題。

一名教練在遊戲中，BP的影響力最多能佔到整場比賽的六，剩下的四還得看隊員們的發揮。想要完全依靠教練的BP贏下比賽，那就是痴人說夢。遊戲最終的結果，還是要依靠場上五名隊員的發揮。而教練決定的只是遊戲內的打法，或者是在這局遊戲中，哪條線是需要被迫抗壓的。做出的這個決定也是因為整個戰隊對於遊戲版本的理解，其實說白了，教練的場上BP可操作的空間還是蠻小的。

而對於觀眾們來說，場外BP也就是圖一樂。畢竟你是不了解這支戰隊的真實情況的。希望大家在看過這篇文章以後，也能夠更加地了解職業賽場上是如何BP的，總的來說其實就是一句話，場上的BP結果是場下教練和選手們共同討論出的結果，看比賽要理性，即使做鍵盤俠也要有理有據，千萬別被人稱作雲玩家哦
