20年前最流行的FPS遊戲，經典的真實戰爭遊戲《三角洲特種部隊》

天天要聞
2019-12-18

上一篇回顧了《紅色警戒》系列，我的網吧生涯與PC遊戲的啟蒙。正是1996年的《命令與征服：紅色警戒》開啟了我沒日沒夜的網吧遊戲之旅。

延續24年的經典，WESTWOOD即時戰略遊戲，《紅色警戒》全系列回顧

從那時起，我從一個堅定的主機玩家，變成了主機、PC兩手都要抓，兩手都要硬的全平台玩家。所以之後花費了大量的時間在網吧，惡補那些經典的PC遊戲。

97，98年的網吧有不少其實根本不能上網，這在現在看來簡直是無法想像的，不能上網那隻玩單機遊戲？其實當時有很多網吧主要是區域網遊戲聯機與一些盜版單機遊戲，只有個別幾台機器是真正可以上網的，但是對於當時的我們來說都不重要，重要的就是一起打遊戲。所以當時最流行的都是有區域網聯機對戰的遊戲。

這其中有不少第一人稱視角的射擊遊戲，ID的《毀滅戰士》（DOOM）、《雷神之錘》（Quake）等等，這中間又有一款以模擬真實戰爭的《三角洲特種部隊》（Delta Force）是最讓我難忘的作品。

經典的《三角洲特種部隊》

《三角洲特種部隊》是我的第一人稱射擊遊戲啟蒙之作，FPS（First-person shooting game）遊戲當年在網吧是除了即時戰略遊戲之外另一個最火的遊戲類型。

當時幾乎大部分FPS遊戲都是如《雷神之錘》這樣強調激烈，火爆，誇張的風格，遊戲設定上帶有科幻色彩，在高科技的敵人基地中，面對著一群群蜂擁而至的士兵。各種武器效果爆炸，畫面強調的就是重口味，你可以用各種武器把敵人射爆，就是為了一個字：爽！

《雷神之錘2》

《雷神之錘2》

然而進入《三角洲特部隊》的遊戲之後，畫面完全是另一個風格，一望無際的山坡與草地，藍天白雲的遊戲背景，連背景音樂都沒有。

沒有烘托氣氛的背景音樂，沒有威力無匹可以把敵人轟殺至渣的光線武器，沒有任何外星生物和高科技種族的影子……大多數時候，你需要徒步很長的一段距離，才會在遠方的山丘上或者建築中發現一個敵人的影子，他很可能只有幾個像素點那麼大……

這個時候你要做的並不是拿出你四十米長的激光大炮把對面轟成一堆血水，只是打開你的瞄準鏡，開槍之後一個表示真實彈道的弧線過去，敵人應聲倒地。

槍響之後，被驚動的地方人員就會向你的位置靠近，你並不是可以和敵人對射的超級戰士改造人，挨一槍就會斃命，你需要保持運動作戰，隨時觀察附近是否有敵方人員向你靠近。

遊戲中你所能感受到的是創作者在盡量還原一個真實世界中的特種部隊戰爭場景給玩家，這與當年充斥網吧的超級戰士「無雙系」射擊遊戲有著天壤之別。

說實話，《三角洲特種部隊》的遊戲畫面是無法與《雷神之錘2》相媲美的，我現在還記得同學當年配PC時賽揚333的瘋狂超頻加TNT顯卡，樂此不疲的優化著他的系統，就為了《雷神之錘2》的畫面效果。

《雷神之錘2》

而同時期的《三角洲特種部隊》，一塊英特爾的I740或者更低一些的顯卡都可以跑的十分流暢，雖然《三角洲特種部隊》遊戲畫面其實可以說是滿屏的馬賽克，但這依然無法阻擋國內玩家對它的熱愛。

但是從另一個角度看，正是《三角洲特種部隊》平易近人的低端運行配置，造就了它在國內網吧的火熱現象。98年有不少網吧還在使用MMX166的CPU，更別提專門的「3D加速卡」了，網吧出於經濟成本的考量而使用低端配置無可厚非，所以在低端配置上可以流暢運行的《三角洲特種部隊》藉助這一優勢更加風靡。

甚至還有專門的把畫面解析度調低，以更加容易發現代表敵人的活動像素點的專門的戰術。如果不是遊戲本身十分熱門，我相信不會有人鑽研這種特別的戰術。

《三角洲特種部隊》區域網支持32人聯機並且有多種不同遊戲模式，可以組隊對戰也可以多人亂戰，至今還能記得多人亂戰一開始就往沒人的地方跑，然後「貓著」陰人的情景。曾經有兩次我們組織了將近十幾人在網吧通宵分組對戰《三角洲特種部隊》，那種空前的盛況至今難忘，整晚的激烈戰鬥中伴隨著此起彼伏的「XX你們倆跟我從右邊包抄！」、「XXX他們繞到後面來了！」這樣的吶喊，一直持續到凌晨。

在《反恐精英》之前，《三角洲特種部隊》應該是國內網吧FPS遊戲的「一哥」。雖然《雷神之錘2》在玩家中的人氣遠超《三角洲特種部隊》，但是在當時國內網吧主流配置偏低的普遍情況下，《三角洲特種部隊》就好像是為國內網吧量身定做的FPS遊戲。走進任意一間網吧，都能看到在各種開闊地形中戰鬥的特種兵們。

真實的三角洲特種部隊

三角洲特種部隊正式名稱是「美國陸軍特種部隊第一特種作戰分遣隊」（1st Special Forces Operational Detachment-Delta，簡稱SFOD-D），所以才被俗稱為「D部隊」，這個Delta其實是指D，而不是指三角洲。他們還另外有一個稱呼，叫CAG（戰鬥應用大隊Combat Applications Group的縮寫），也會被人稱作D-Boys。

所以三角洲特種部隊並不是這支特種部隊真正正式的名稱，但是無論是在遊戲中還是大量的影視文化作品中都以三角洲特種部隊來稱呼他們，所以本文也依舊使用這個名稱。

我搜索到了兩種號稱是三角洲特種部隊徽章的圖案，鑒於我的軍事知識有限，希望知道的朋友們可以回復科普一下。

網路上可以搜索到的一些三角洲部隊的臂章，可以看出遊戲的標題畫面還是非常還原的。

遊戲中的三角洲特種部隊標誌。

對三角洲特種部隊描寫的影視作品中最著名應該就是《黑鷹墜落》了，電影中登場的D-Boys，三角洲特種部隊的隊員們。

電影中的三角洲隊員

電影中有一段劇情，是兩位負責在直升機內提供火力支援的三角洲隊員蘭迪.舒哈特與加時.戈登，在救援已經墜落的兩架黑鷹直升機時，得知地面部隊無法到達，主動要求下機支援。

但是面對不斷湧現的索馬利亞民兵，兩位三角洲隊員戰鬥到裝備物資耗盡，戈登陣亡後，舒哈特把戈登的步槍和最後的半彈匣子彈留給了直升機駕駛員杜蘭特，自己出去獨自面對眾多民兵武裝，最終犧牲。

這一片段是根據當時被被俘的直升機駕駛員杜蘭特的回憶改編的，杜蘭特最終獲釋，幸免於難。

獲釋的直升機駕駛員杜蘭特

兩位捨身支援戰友的三角洲隊員，在之後被追授榮譽勳章。

電影中兩位勇敢的三角洲隊員面對成百上千的民兵武裝，依舊跳下飛機，通過無線電向戰友傳達「我們正在構築防線！」的一幕雖然沒有口號，但確實很主旋律很能觸動人。看到這裡真希望我們的作品不要再無敵手撕鬼子……

影視文化作品難免有些藝術上的誇張，但是當年的《黑鷹墜落》根據真實歷史事件改編，又是拍攝的極其真實的戰爭影片，我對於三角洲特種部隊的印象來源基本源自這部2001年的著名電影。

哪怕是18年後的今天，再看依舊讓人感慨。

2003年推出的《三角洲特種部隊：黑鷹墜落》(Delta Force:Black Hawk Down），是系列中專門描寫這段故事的一部作品。

消失的《三角洲特種部隊》系列

《三角洲特種部隊》在國內絕對算得上是一款現象級的遊戲，但是這樣一款經典的遊戲卻在不知不覺中淡出了我的視線，其實《三角洲特種部隊》系列總共有8部作品，系列最後的《三角洲特種部隊：極限作戰2》（Delta Force:Xtreme 2）是2009年推出的。

《三角洲特種部隊：極限作戰2》

但是不管我怎樣努力回想，在我的記憶中，只有《三角洲特種部隊》的初代，之後的作品完全沒有一點印象。努力回想之後，捋出了我的網吧遊戲的代際關係，最初一代是《紅色警戒》，其次就是《三角洲特種部隊》，之後是《星際爭霸》，最後是《反恐精英》。

所以我查了一下，《反恐精英》1.0beta是1999年6月推出的，2000年發布CS1.0，2001年3月發布CS1.1，2001年9月發布CS1.3。而《三角洲特種部隊2》是1999年9月推出的。

接著我又搜索到了一則《三角洲特種部隊》被舉報的網路文章，具體的舉報原因可以看下圖。

還有一些報載文章的截圖，可以看到具體的舉報原因，以及當時媒體對《三角洲特種部隊》以及《文明2》這些遊戲的態度。

具體的新聞內容我就不再贅述了，如果有想了解的朋友們可以自行搜索一下，就可以看到描寫這次舉報的一些文章。

當時對於遊戲的主流聲音還是批判的聲音，以及「電子海洛因」這樣的形容詞，主機遊戲在當年也是不被接受的。

此時，我大概就知道為什麼我對《三角洲特種部隊》這款遊戲的記憶只保留在第一代了，在這些遊戲層面之外的因素影響下《三角洲特種部隊》給我留下的印象只在第一代。在這之後大環境寬鬆之時，《三角洲特種部隊》系列新作面對的又是下一個網吧時代的FPS王者《反恐精英》，慢慢被人遺忘也就是這個系列必然的宿命了。

寫在最後

最後能查到的，就是2003年《三角洲特種部隊》系列的開發商NOVALOGIC因為使用盜版軟體，被Adobe等7家軟體廠商聯合訴訟，最後賠償超過了15萬美元。

最後是NOVALOGIC被THQ Nordic收購，徹底壽終正寢。不過現在在Steam上還是有《三角洲特種部隊》全系列。

又一個經典的遊戲系列完結了，即便之後有《三角洲特種部隊》的最新作品，也不是出自NOVALOGIC之手了。

最後的最後，還是要感謝這款我的啟蒙FPS遊戲，以及那段奮戰在網吧的日日夜夜，時代變了，事物變了，唯一不變的，是記憶中那段簡單快樂的美好時光。
