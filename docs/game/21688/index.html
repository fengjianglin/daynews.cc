夢三國2：生活在陽光下的超強刺客——太史慈

2019-12-18 195條評論 1,492次閱讀 57人點贊

提到夢三國中的刺客英雄，大多數玩家首先想到的必然是卞玉兒、呂夢茹之類擁有極強秒殺能力且經常徘徊在對手視野範圍之外的英雄。不過今天，舞文將為大家介紹的並不是這些英雄，而是一個生活在陽光下，本身經常暴露在對手視野中攻擊範圍之外的”刺客”，他就是太史慈。

之所以舞文會說這是夢三國版的《刺客信條》，最大的原因還是太史慈本身是一個物理輸出英雄，並不能算是刺客英雄。但是因為自身有著極強的突進能力的同時，傷害也高得有些離譜，對手一個小小的失誤就有可能葬送自己的原因。所以，舞文覺得太史慈這個身居狂戰士與刺客屬性的英雄，真的就是在夢三國大陸上上演一場不一樣的《刺客信條》。

好了，話不多說，直接開始正題，為大家介紹下太史慈的刺客級玩法。

天賦選擇

在天賦選擇上，舞文還是採用了比較中規中矩的選擇，許願天賦選屬性，10級選20點攻擊力，15級選30%攻擊速度，20級選10點全屬性，25級選50%血戰天虹效果。整體來看，與昨天舞文為大家介紹的太史慈天賦選擇沒有多大出入。

出裝思路

屬性鞋（後面有錢的時候直接升級）、雷錘（增強打錢能力的同時強大的攻速提升可以大大提升太史慈的前期作戰能力）、bkb（因為玩法偏向刺客所以生存道具必不可少）、支配頭盔（進一步提升自己的生存能力，後期合成大血槍）、倍擊或聖劍（這個根據陣容選擇即可）。

打法思路

好事多磨，作為一個輸出英雄，太史慈即使玩法像刺客，也無法做到前期就依靠技能去無腦消耗對手，保證自己的發育顯得更加重要。所以，前期無論是選擇走線上還是打野都可以，我們需要的就是儘快做出雷錘這件前中期非常強大的輸出和打錢裝備。

一般而言，舞文比較推薦大家在10級左右開始自己的刺殺之旅，因為等到太史慈到達10級之後，本身已經擁有了不錯的裝備基礎不說，技能等級提升之後刺殺能力也會大幅度提升。好了，下面就是打法思路。

等級低的時候，血量需要在大半以上，同時儘可能保證自己滿藍或者有大半以上的藍量。這樣才能發揮出太史慈隱藏在輸出面具之下的刺客天賦。在看到敵方脆皮英雄的時候，直接利用E技能減速對手開始普攻，如果對手沒有位移技能基本會直接交代，如果有位移技能也不要著急，利用大招直接砸對手身上減速的同時直接銜接變身後的EW兩個技能基本就能輕鬆帶走。

對敵技巧

以上是基本的打法思路，實戰中我們需要面對的往往不只是一個敵人，雖說我們也有隊友，但在沒有足夠默契配合的情況下，刺客級的太史慈想要與隊友配合難度還是比較大的。所以，用好下面這些對敵技巧顯得非常重要。

1、 一定不要先手大招：如果我們遇到對手是多個英雄一起的，先手大招依靠極高的爆發傷害或許可以輕鬆帶走其中的一兩個人，但想要保住自己依舊很有難度。所以，盡量用E技能先手效果會好很多。

2、 用完大招後不要隨意切換形態：由於太史慈本身是個遠程英雄，但又可以變身近戰的原因，不少玩家在用過之後感覺不習慣就想要切換到遠程形態，其實完全沒有必要。在近戰形態的時候，我們完全可以直接E技能突臉攻擊對手，在對手準備逃離的時候用W技能拉回對手，拿下對手後利用大招拉開距離開始遠程輸出也是非常不錯的選擇。
