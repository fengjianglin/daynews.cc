德雲色征服者“血魔流”大頭，炮台邊打邊加血，人送外號永動機？

2019-12-11 2,520條評論 4,948次閱讀 97人點贊

最近征服者成為了英雄聯盟中最為火熱的天賦，雖然取消了真實傷害，但是對於眾多英雄來講，征服者都可以攜帶了。在疊滿之後，不但可以提供一定的適應力，還可以擁有不錯的回復效果。最近德雲色就玩出了一套新花樣，那就是”征服者血魔流大頭”。

這套玩法最主要的就是攜帶天賦征服者了，然後使用大頭走下路，這樣無論是推線能力還是大家能力，前期都非常的厲害。但是顯然德雲色還是一如既往的”憨”呀。在走到野區的時候，臉探草叢，被對面一級團抓了個正著，隨後德雲色也不約而同的笑了出來，讓觀眾們直呼”精彩”。

因為被對面的水友們認出來了，於是對面的打野就非常的針對，在拿完藍之後，直接從野區竄出來抓正在上線的西卡。給笑笑嚇了一跳。但是這個瞎子明顯有一些上頭，在打不死的情況下依然選擇了閃現上來點西卡，最終因為二級抗塔，被塔給點死了。西卡也因此收到了一顆人頭。

既然被對面認出來了，那麼照顧是必然的，在僅僅只有兩級的時候，對面的上單看到下路打起來了，便直接TP到下路，隨後看到殘血的大頭，直接閃現接平A擊殺掉，不過好在機器人有鉤子，將刀妹勾了回來，不然這一波就有點血虧了。

隨後下路爆發團戰，西卡的大頭絲血逃生，對面的佐伊還在追，但是一套技能下去之後，並沒有打死，佐伊本來沒打算追了，但是西卡卻還要接著浪。反而是被佐伊一個R接W三個火花給打死了，西卡都沒反應過來。

但是對面好像發生了一點矛盾，打野瞎子和對面下路的烏鴉有點不和。導致本來崩掉的局面一時間又好了起來。但對面的烏鴉+女坦組合確實很頂，女坦控住後烏鴉一拉，便直接開啟團戰。

不過西卡的大頭一套技能下去瞬間疊滿征服者，對面只要站在炮台的射程之內，大頭就能瘋狂吸血，「血魔流」大頭此時才展現出他的威力呀。最終笑笑憑藉著征服者提供的續航，在這一波大爆發中存活了下來，並且反殺掉了烏鴉和女坦，一切就都好起來了。

最終眼看局勢大好的西卡，直接出了一本殺人書，站在對面高地上面跳舞。對面的佐伊想要來騷擾一下，卻被這個征服者高的心慌意亂，最終德雲色一手征服者大頭，打出了8-3-2的戰績！
