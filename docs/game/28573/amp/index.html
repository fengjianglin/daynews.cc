寶可夢最強縫合怪鰓魚龍 重型超速炮台對戰分析

天天要聞
2019-12-23

純原創，請互相尊重【基於第八世代】

鰓魚龍理論上很強，水龍屬性抗性極佳，排除某些天敵之外能夠基本做到站樁輸出，特性各個棒呆，儲水雨天兄弟內戰，強壯之顎炮台點射，拔沙混沙暴隊優質打手（雖然現在暫時還無法獲得夢特），可以說在縫合怪家族中鰓魚龍理論實力優異，但實際使用過程中，由於鰓魚龍存在絕對天敵，在面對某些特定陣容時形同虛設很難有所發揮，所以我們還是需要理論結合實際，來好好分析一下鰓魚龍究竟對戰方面素質如何。

水桶身材種族值平均：

鰓魚龍種族值505且分布十分均勻，三防接近100配合水龍屬性帶來不俗的生存能力，速度75中規中矩，物攻略高為90，可惜特攻也被分配到70點種族值，這種分配導致有效種族值僅有435，略顯美中不足。正常情況下鰓魚龍還是走鰓咬物攻路線，但70點特攻種族值是否能夠用起來？給對手來個出其不意的龍之波動？也是留給各位訓練師的運營思考空間。

鰓魚龍的特性選擇分析：

鰓魚龍共有三種特性，儲水，強壯之顎與拔沙，綜合來說強壯之顎鰓魚龍的爆發力最強，儲水適合雨天內戰，考慮到沙暴隊中不乏拔沙打手，鰓魚龍在沙暴隊中的地位可能一般（但是沙暴下兩倍速還是非常厲害的），因此常規路線還是選擇強壯之顎，該技能使得啃咬類技能為例上升50%，效果拔群！另外在內戰中強壯之顎的適用場景也會高於儲水，因此依然推薦選擇強壯之顎作為鰓魚龍對戰特性。

鰓魚龍的道具選擇分析：

首選講究圍巾，該道具配合鰓咬能夠打出爆炸傷害，想像一下雨天之下超速對手引發鰓咬威力翻倍再加上強壯之顎帶來的威力加成，三倍威力的鰓咬誰能頂得住？（還真有天敵頂得住……）妥妥的就是一個站樁輸出的大號炮台，當然若是對速度還不夠有自信，大嘴鷗順風輔助鰓魚龍輸出也是一種不錯的攻擊組合。在講究圍巾之外，也可以選擇生命寶珠作為道具後補，雖然無法做到完美超速其他寶可夢，但生命寶珠下的鰓魚龍打擊面明顯擴大，也能夠更好的起到清理戰場的作用，同時在順風的作用下，鰓魚龍依舊有機會轟出三倍威力鰓咬。

鰓魚龍的技能選擇分析：

【鰓咬】是鰓魚龍的必要技能，通常來說圍巾鰓咬輸出就是鰓魚龍的標準打法，其餘技能在圍巾狀態下均效率低下，基本淪為雞肋，但是在生命寶珠加持下的鰓魚龍就可以通過不同技能組合來提高打擊面，以當前galar weekly中熱門寶可夢排行進行參考，為了應對幽靈與妖精屬性寶可夢的泛濫，【咬碎】與【鐵頭】都是不錯的選擇，另外若是對自己的運氣比較有自信，可以保留【龍之俯衝】應對第八世代崛起的龍屬性寶可夢，如多龍巴魯托。

鰓魚龍的努力值與性格分析：

既然通常來說都是走的炮台路線，為了儘可能打出三倍鰓咬這個終極目標，就需要將速度提滿，而在三倍威力的前提下，可能滿物攻努力值就不一定是鐵板一塊的選擇了，適當的彌補一下HP，使得鰓魚龍有更為持久的戰場能力也不失為一種運營策略。性格方面固執是第一選擇，如果對速度還是有所忌憚，可以選擇爽朗進一步提速。

鰓魚龍的天敵海兔獸：

正式由於海兔獸的存在，鰓魚龍才成為了理論中的對戰好手，引水，這個鰓魚龍怎麼樣也繞不過去的特性，猶如黑洞一般蠶食著鰓魚龍的生存空間，再強大的輸出能力，在引水海兔獸的面前也不過是為其提高特攻的食餌而已。因此在對戰時需要非常在意對方是否攜帶海兔獸，在雙打環境中若要使用鰓魚龍，建議放在六人大名單中並根據遇敵情況調整鰓魚龍的出場策略，避免偶遇海兔獸的尷尬。

總體來說鰓魚龍輸出能力超一流，同時能夠混跡與各種不同打法的隊伍，雨天隊，沙暴隊等等組合都可以打出自身價值，雖然海兔獸是一道邁步過去的坎，但鰓魚龍強大的攻擊能力依舊能夠給與對手極大的壓迫感，而特攻型鰓魚龍也許能夠使對方產生誤判，在對局中起到意想不到的效果！
