雲頂之弈9.24b版本更新

12月17日美測服更新了9.24b版本

這大概是今年最後一個版本啦

更新完之後設計師們就去過聖誕了

大家最近是不是都被雷霆打的不要不要的

沒關係！馬上它就囂張不了了

馬上給大家介紹預覽哦！

三星棋子改動

首先要給大家介紹的就是關於三星棋子的改動

因為當前版本有非常多的賭陣容

無論是賭掠食者、森林、光

當核心棋子到達三星後

有點過於強大了

於是設計師們決定削弱三星棋子

以前升到3星可以將2星時的生命值和攻擊力翻倍

現在生命值和攻擊力削減為二星的1.8倍

（上圖中以狗頭的屬性為例）

不得不說這對所有的賭高星陣容來說是很大的削弱

下個版本賭狗們還能繼續快樂嗎？

今日陣容推薦

今天給大家推薦一套狂戰士陣容

在9.23版本狂砍了兩周的奧拉夫

這個版本依然好使！

推薦陣容：極地狂戰士

推薦英雄：荒漠之咬-雷克頓、弗雷爾卓德之心-布隆、武器大師-賈克斯、雷霆咆哮-沃利貝爾、蒙多醫生-蒙多、探險家-伊澤瑞爾、亡靈戰神-塞恩、狂戰士-奧拉夫

推薦裝備：狗熊-鳥盾x2（大棒+鎖子甲）、EZ-大天使（眼淚x2）、奧拉夫-飲血劍（大劍+斗篷）、羊刀（大棒+反曲弓）、沉默（斗篷+眼淚）

組成羈絆：6狂戰士4極地

陣容分析：

這套經典的極地狂戰士陣容就不用過多介紹啦。6狂戰士的攻擊有100%的幾率對目標背後錐形區域造成傷害，而且在這個版本還可以額外提供給所有狂戰士25點攻擊力。搭配羈絆效果和飲血羊刀，奧拉夫真的是狂砍一條街。

陣容8人口成型，如果有鏟子可以優先考慮狂戰士或者極地轉職，那麼7人口就可以成型啦。站位狂戰士們基本可以懟臉站一排，開局跳上去就是干噢！
