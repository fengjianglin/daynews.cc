這個遊戲好玩嗎？曾引領潮流的《大金剛》，評價也會隨著時間改變

天天要聞
2019-12-13

今天的故事，屬於那些偉大的創造者們……

緣起·曾經的經典VS時代的變遷

有的遊戲很經典，但如今卻很少有人願意再玩了……

比如掃雷。就連筆者自己，恐怕現在也很難解釋當初Windows自帶的掃雷，是如何一夜一夜的出現在電腦屏幕上的。

比如俄羅斯方塊，那時候幾乎家家都有這麼一台小掌機，甚至全家人還都搶著玩。

隨著時代的變遷，這樣的遊戲雖然名聲在外，卻很難讓人提起興趣。

是什麼樣這些遊戲看上去不那麼好玩了？

《織夢島》的謎題方式，還適合現在的玩家么？

《塞爾達·織夢島》的遊戲機制，是不是不適合時代了？

宮本茂的《塞爾達·織夢島》多多少少也有這樣的命運，其實筆者認為，這些謎題的設計，即便到了如今，也並不過時，但玩遊戲的人卻變了。

他們不願意探索，更不願意為了某個線索找遍幾個區域。所以他們對謎題抱怨起來，認為這遊戲完全不符合潮流，也完全不好玩。最主要的是，他們不想徒勞的跑來跑去，他們需要直接的回報。

既然提到了宮本茂，那麼咱們今天同樣聊一聊一款這樣的遊戲，在當年，她風靡了全世界，甚至拯救了任天堂，山內溥也是因為這款遊戲，下定了決心，讓上村雅之開始研發紅白機。

這款遊戲最後也成了紅白機上的第一款遊戲作品！

可如今，儘管在Switch的會免遊戲里被複活，卻少有人願意再玩……

那麼這款遊戲，放到現在的年代，究竟還算不算好玩呢？或許這篇文章，會給你一個答案。

至於這款遊戲的名字，想必諸位也都知道了，沒錯，她便是《大金剛》。

《德軍總部3D》&《毀滅戰士》

湯姆·霍爾：只有宮本茂懂得遊戲的真正樂趣

《德軍總部3D》與《毀滅戰士》的主創湯姆·霍爾曾經說過，宮本茂是唯一懂得電子遊戲真正樂趣的人。

說起來，宮本茂並不是幸運兒。

他曾就讀於金澤美術工藝大學，專業是工業設計系。不過畢業後，宮本茂並沒有找到自己心儀的工作，成為了如今頗為常見的啃老一族。

關鍵時刻，宮本茂的父親還是動用了自己的關係，給當時的任天堂社長山內溥寫了一封推薦信。

想必我兒子的才華你們也有所耳聞，如今他正在家製作毛絨玩具，但國家的興衰匹夫有責，還肯老友助其一臂之力。

這是玩笑話，不過山內溥也看在宮本茂父親的面子上，收留了宮本茂，並讓橫井軍平帶帶他。

說起來，山內溥為什麼對宮本茂的毛絨玩具感興趣？

或許是因為橫井軍平當初的「超級怪手」給任天堂帶來了不菲的收入，讓山內溥對玩具有一種天然的好感。亦或是山內溥從宮本茂身上看到了光芒，就像當初看到了橫井軍平一樣。

總之，這個決定，又一次讓任天堂起死回生。

上村雅之意外，《Radar Scope》竟然失敗了！

山內溥想從街機領域撈金，卻出師不利

那時，山內溥幾次美國行，感受到了潮流所帶來的震撼，也想開始往電子遊戲這塊發展。

1977年，任天堂與三菱電機共同研發的Color TV-Game 6發售了，售價為9800日元，賣出了100萬台。同年晚些時候，Color TV-Game 15發售，遊戲數量從原來的6個提升到了15個，售價也增加到了15000日元，銷量同樣為100萬台左右。之後，這個系列還有後續，不過銷量都趨於平淡了。

這讓任天堂覺得，遊戲機和電子遊戲，是個可以發展的方向。

時間來到了1979年，街機的風潮席捲了大街小巷，此時市面上的熱門品類並不是CAPCOM主宰時的清版動作遊戲，而是以太東的《太空侵略者》和南夢宮的《小蜜蜂》為首的彈幕射擊類遊戲。

紅白機版本的《小蜜蜂》

山內溥也想做街機遊戲掙錢，便找來了上村雅之，在《小蜜蜂》推出之後沒多久，《Radar Scope》就出現在了市面上。這款任天堂與池上通信共同開發出來的街機產品，硬體性能確實不凡，運行《Radar Scope》流暢不說，對於偽3D的視覺感，更是有著不錯的表現力。

不過100萬日元的定價和同類遊戲的飽和，都讓美國市場並不認可《Radar Scope》，最終三千台街機基板，經過推銷人員的努力之後，仍剩下了2000塊。

翻過來說宮本茂，這幾年在任天堂也沒幹啥大事，反正自己是關係戶，也沒人說他。

當時橫井軍平和上村雅之算是任天堂的主力設計人員，而《Radar Scope》就是上村雅之負責的項目，當銷量出現問題的時候，任天堂一邊在想辦法補救，而另外一邊的橫井軍平則已經開始策划起了Game&Watch。

山內溥計上心來，乾脆就讓宮本茂試試看吧，畢竟這幾年，他啥也沒幹。

宮本茂最初的創意是大力水手三人組

天將降大任於斯人也……沒想到卻是天才！

山內溥看中了的人，似乎都有拯救世界的力量，比如岩田聰，也是當初因為Hal研究所瀕臨倒閉，他到任天堂求援的時候，被山內溥相中的。

言歸正傳，宮本茂接下了這個活，就開始琢磨。

起初，他讓人聯繫《大力水手》的版權方，希望用大力水手的形象做遊戲，以此來博得美國玩家的好感。

遊戲方式上，宮本茂則拋棄了時下流行的彈幕射擊，這多少還是有些冒險的。

遊戲中，跳躍和躲避成了主要的前進方式，再配上《大力水手》鐵三角的影響力，宮本茂此時覺得這遊戲已經成功了90%。

《大金剛》紅白機版遊戲關卡

不過意外還是發生了，《大力水手》的版權沒談成，宮本茂在權衡之後，決定自己來進行人設的工作。也正是因為這次不大不小的意外，森喜剛（中文官方譯名），馬里奧和寶琳都被創造了出來，或許就連宮本茂自己都不會料到，馬里奧會在以後爆發出如此驚人的能量。

《大金剛》做完之後，山內溥意外的滿意，儘管山內溥自己平時不玩遊戲，只在需要進行測試和驗收時，玩上那麼幾下，但他的洞察力卻絕對毋容置疑。

《大金剛》的Rom刷到原來的《Radar Scope》基板上，起初還有很多人質疑，尤其是相關的銷售人員。

據說只有兩個酒吧願意引進《大金剛》，本以為故事會以悲劇收場的美國任天堂卻在幾天後得到了好消息，那兩個老闆都過來要求接著購買《大金剛》街機。隨之而來的，就是其他酒吧的老闆……總之，《大金剛》成功翻盤，這也讓宮本茂的實力得到了認可。

關卡更為豐富的《大金剛Jr.》

後來，宮本茂又繼續推出續作《大金剛Jr.》就是大金剛的兒子去救爸爸的故事，馬里奧還做了回壞人。讓人沒想到的是，後來《大力水手》那邊又同意授權了，宮本茂就又做了一款《大力水手》，這三個遊戲，也成為了後來上村雅之開發的紅白機的首發遊戲。

據上村雅之回憶，當初山內溥想要進軍家用主機市場，提出的口號就是要把街機遊戲搬回家，這街機遊戲指的就是宮本茂設計的《大金剛》。也正是因為山內溥的這個口號，讓之後SNK在NeoGeo主機推出的時候不得不把宣傳口號改成了「將很棒的遊戲帶回家吧！」

任天堂還出過一款《大金剛Jr.》算術遊戲

上手難度過高，或許會讓玩家挫敗感太強……

說起來，在當時宮本茂算是開創了一個新的遊戲類型，這種創新，本身就是一種冒險。另外，遊戲中儘管看似操作不難，但宮本茂對於關卡節奏的把控，卻相當出彩。

你需要不斷的干擾下前進，這本身就是一個挑戰。

最為重要的一點是，宮本茂並沒有把遊戲的難度轉嫁給操作，這點也非常適合街機玩家的偏好，畢竟容易上手，就等於馬上可以掌握好這款遊戲。至於玩的好不好，或者能不能更快的發現規律，則是自己水平的問題。

同樣引領潮流的《Pong》

關於早期街機遊戲的上手難度問題，在這裡可以提出一個反例，就是雅達利的創始人諾蘭·布希內爾曾經在《Pong》之前所開發的一款叫做《電腦空間》的街機遊戲。

其實布希內爾在大學時候，接觸到了史帝芬·羅素的《太空大戰》，這款在PDP-1平台上做的電子遊戲很快就讓布希內爾如痴如醉。

在工作之餘，布希內爾對《太空大戰》念念不忘，加上之前他去投幣機店打工，便將二者結合。

但問題是，《太空大戰》的操作可不簡單，並且還有類似閃現的技能，可能對於大學生來說，適應起來並不困難，但對於剛剛接觸，甚至還沒接觸過遊戲的街機玩家而言，實在是太摸不著頭腦了。

操作複雜的《電腦空間》

所以布希內爾的《電腦空間》以完敗告終。

這多少也是他之後開發《Pong》的一個契機，他也發現了，操作簡單，上手容易，才是初期發展的奧秘。

在《大金剛》中，宮本茂也加入了道具，而這些道具也可以在某些時候發揮巨大的作用，並且讓遊戲內容變得更為有趣。

每過一關，地圖的結構就更為複雜，場景也更為混亂，也就是干擾元素會變多，但又並非難以企及，這都是宮本茂對於關卡設計的優越把控。

《大金剛》二合一的卡帶

遺憾，《大金剛》如今已經不能算是好玩的遊戲了……

在當時來看，《大金剛》的素質已經非常高了，但如果我們把這個遊戲放在如今的時代，那麼她是不是依舊出色呢？

筆者相信大多數玩家的回答應該是否定的。

不管關卡設計的多麼出色，過短的流程，都是始終無法避免的硬傷。

還有一點，就是目前的玩家，或多或少都希望在遊戲中有些養成元素，即便是TGA今年的年度最佳遊戲《只狼》，雖然是ACT分類，但也仍有養成的元素，也算是對玩家的一種另類的妥協了吧。

當初曾經風靡與拯救任天堂的《大金剛》，如今已經有些跟不上潮流了，就算是關卡長度不短的《超級馬里奧兄弟》，也被多次迭代，包括玩法上的更新，也有諸如《馬里奧·奧德賽》這樣的高水準新作品。

所以說，時代的進步，也會讓電子遊戲全方位的進化，《大金剛》也不例外，所以Switch版的《大金剛國度·熱帶寒流》便是最好的證明，這款遊戲同樣獲得了IGN9.0分的高分，可謂沒有給前輩們丟臉。

紅白機《大金剛》封面

結尾·有些私心的喜愛，向那些製作人致敬

答案已經很明確了，儘管《大金剛》是經典，也是曾經的寵兒，即便是現在，在Switch上，偶爾玩一下也沒問題，但年代的跨度讓其劣勢太過明顯，實在難以說好玩。

但遊戲裡面的跳躍元素，道具使用方式，以及關卡的搭建，同樣也被很多後來者參考，宮本茂的創意實在太過不凡。

所以如果要說這款遊戲不好玩，也有些不忍。

如果可以，我更願意把《大金剛》歸為好玩的一類，畢竟他所帶來的影響實在太大。如果沒有《大金剛》，也許紅白機都不一定能誕生。

綜上所述，請原諒筆者的私心，也請愛護這些曾經的經典，也向這些天才般的製作人致敬。
