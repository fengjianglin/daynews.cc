王者榮耀：來自我的王者人生以及意外擁有艾琳的詳細過程

2019-12-10 657條評論 2,868次閱讀 592人點贊

第一部分發現王者榮耀前言：2012年6月左右朋友拉我一起打LOL，看在大家都玩的份上我也不能掉隊啊，無奈此遊戲超出了我腦容量的理解範圍就是玩不懂隨之放棄，直到中間有意無意的打一下2104年成為一名網癮少年。到2015年的時候由於工作沒有時間玩電腦遊戲就四處找類似lol的手機遊戲，可是每一次體驗後的結果都是糟糕的最後放棄，15年上半年在偶然的一天作為善於發現遊戲的我無意間在qq的新游預約裡面看到了王者榮耀和全面超神的兩款新手游的預約。

第二部分從期待到失望：預約後一直有事沒事去看看等待開測的到來，直到2015年的8月某天提醒開測時間就拿著手機卡著點 在2015年8月18日還是19日的上午十一點左右（時間太久具體記不清了）開啟了不限號不刪檔的測試。立刻就登錄進去體驗了一下新手教程和新手人機玩了兩把就卸載了。找了這麼久期待這久跟我想像中想要的手機版lol不太一樣，在口中伴隨著一句LJ就卸載了。

王者榮耀app-蘋果商店

第三部分從不知名的遊戲到人人都玩的國民手游：時隔半年多左右16年初去北京後看我表哥他們都在玩王者榮耀說挺好玩的讓我也下載一起，當我已經忘記什麼樣子的時候我始終認為不管什麼遊戲只有夥伴一起玩的遊戲才是長久好玩的遊戲。那時候剛好是S2的亞瑟皮膚（記不清了）

亞瑟-獅心王

第四部分我的艾琳由來前言：跟我哥他們玩的時候都是用的微信所以我也是用的微信（不過後來換了微信啥也沒了以前的東西），當時玩了一段時間後來發現qq區還有好多朋友玩的時候我就想那就兩個區都玩吧。qq登錄進去的時候想到當初開服的時候只有三個區想到一區會不會太危險就進的最後一個三區，所以這次還是登錄之前註冊過的三區想著註冊過了直接進去就好了。當我登錄成功後發現好多好多的郵件等我領取，已經成為一名鑽石的玩家來說足夠讓我興奮了 畢竟還沒有現在的星耀（打到鑽石就不敢打了怕掉了）

一區和三區

第五部分我的艾琳由來後續：時隔近一年登錄qq區後發現大量的郵件就一直領取好多好多越領取越興奮 這下不懂擔心鑽石抽不到韓信了（哈哈鑽石那個數字對我來說就是土豪了）直到過程中一個郵件給了我艾琳的英雄和皮膚（當時不曉得就在想艾琳怎麼跟後裔一樣啊嘖嘖不過也好女的漂亮）。郵件領取完後分別為鑽石12000左右丶艾琳一套丶520點券。

第六部分為什麼會給我艾琳等疑惑：知道艾琳是絕版後就開始在想為什麼會給我那是什麼活動才能得到艾琳。然後就開始找，直到在官網公告翻到15年9月的時候看到了艾琳的活動。活動規則是登錄七天送艾琳滿15天送艾琳女武神皮膚。直到艾琳的活動後就釋然了要不然人家老是問我怎麼得到的什麼活動我一臉懵多尷尬作為一名老玩家。這個疑惑解開以後開始納悶我也不滿足當時的活動啊，九月的活動我是八月開服玩了兩局就卸載了。這個問題只有後來的傳言回饋老玩家贈送的才解釋的過去吧。

結尾：在網上看到各種各樣的說法，今天我的真實經歷給大家說了這麼多就是想關注點贊留一個不過分吧（開始是用手機打的打到一半軟體崩潰丟失了這是回家用電腦重新打得）
