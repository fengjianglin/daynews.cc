雲頂9.24全評級：盾盾流狂戰士崛起，火男跌至B級

天天要聞
2019-12-18

9.24陣容評級

雲霄劍刺本來在上個版本就是可以與法海相媲美陣容，經過全明星空門的奪冠之後，劍刺在國服的熱度也是一直沒有下降。不過其中也不得不說到幻影之舞削弱的影響，果然是一裝削弱，眾多受益。

盾盾流狂戰士是最近才流行起來的陣容，因為6狂戰現在額外增加25攻擊力，以往的掠食劇毒狂戰正在慢慢往6狂戰靠攏，這讓夜夜一下想起來遠古版本的盾盾法，不知道現在還有多少萌寶知道呢？

9.24英雄評級

得益於影羈絆從2/4改為3/6，5費卡的劍聖已經成為6影陣容的最後一塊拼圖。可以說劍聖的優先順序提高，完全是影和劍士火熱的結果。夜夜也在最近玩過不少6影，強度可觀極易鎖血連勝。

9.23的版本之子——火男，從S級一下跌至B級，其中不光有海洋羈絆和自身削弱的原因，劍士和雷霆的加強也是一部分原因。

9.24裝備評級

因為幻影之舞的削弱，無盡之刃終於翻身，暴風大劍也相繼成為S級裝備；鳥盾升至S級也是盾盾流狂戰士的崛起影響。

對了，9.24將是幻影之舞、反甲存在的最後一個版本。大家可以在這個版本玩一下懷舊的刺蝟流布隆這類的陣容，掉分怎麼辦？這個夜夜不負責~~
