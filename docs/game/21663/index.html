測試服前瞻鮑三娘：關索？不存在的！

2019-12-18 1,666條評論 3,200次閱讀 1246人點贊

在上一次的測試服更新中登錄了新武將鮑三娘，很多朋友可能沒聽說過鮑三娘，她是關索的妻子，其技能居然和關索沒有任何任何關係，這也是小編非常疑惑的一點，於是乎小編本著好學的心態去官網查了一下，結果自然是一目了然。

言歸正傳啊，首先呢鮑三娘如果真的是以這個面貌去登錄正式服的話可能會讓很多玩家失望，為什麼呢？我們先一起來看看技能吧！

姊勇：當你使用或打出【殺】時，你可以獲得一名其他角色的一張牌，然後其摸一張牌。

這個技能無疑是十分克制延時性錦囊牌的，並且能夠和很多武將產生聯動性，比如說陸遜，在鮑三娘有AK的情況下這個組合的強度是比王基加陸遜的強度還要更高的，一張殺可以使鮑三娘摸兩張牌，另外用於拆隊友的判定，拆敵方的裝備都是一個不錯的選擇。並且在一副牌堆中是有著44張殺的，這個技能的發動頻率也是非常可觀的。

許身：限定技，出牌階段你可以失去等同場上存活男性角色數的體力值；若你因此進入瀕死狀態，則你脫離瀕死狀態後，你可令使你脫離瀕死狀態的覺得獲得「武聖」和「當先」。

首先呢，在三國殺ol包括三國殺互通版中，鮑三娘的這個技能是沒有自爆向合格選項的，而是救起鮑三娘的角色可以自己選擇是否變成關索，就因為這個窮人小編才對鮑三娘的出現興緻勃勃，可惜手殺的策劃可能是怕鮑三娘身為一個土豪將卻是讓別人爽的這一點或者是為了對那些擁有關索的玩家保持公平。反正不管是怎麼樣，在測試服鮑三娘的許身是有所改動的。

再來說說許身這個技能的強度，這個技能在我看來不是很適合軍八，軍5倒是很有可能會大放異彩，首先軍八的將面一般男性角色最少在4個以上，這個技能的先決條件很苛刻，因為必須因此技能進入瀕死狀態才能夠獲得技能，這點對於鮑三娘來說算是小加強吧，畢竟選擇權在自己手裡。這個技能的爭議之處就在於要用很多血量去換取兩個技能，誠然這兩個技能確實是務必強力，尤其是對一些特定武將來說，但是代價也是非常大的，所以一定要選擇好發動的時機。但是毋庸置疑的是，這個技能是整個三國殺最強的輔助技能之一。

鎮南：當一張牌指定多個目標後，若你為此牌目標之一且此牌指定目標數大於使用者當前體力值，則你可以棄置一張牌對此牌的使用者造成一點傷害。

因為這個技能只能是大於體力值，所以這個技能的發動頻率是及其有限的，一副牌中的五穀，南蠻入侵，萬箭齊發，桃園結義，加起來都不超過十張，不過還是能夠去克制一些武將：大嘴，荀攸等。總的來說呢，鮑三娘在移動版三國殺中的強度是遠遠要強於十周年版本的鮑三娘，就是限定技的先置條件可以說是無比坑爹了，另外想一桃蹭關索的玩家們也可以收起自己的小心思了。
