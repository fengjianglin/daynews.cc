強力且厚重！盤點風暴英雄中那些手持戰錘的英雄們

2019-12-13 1,185條評論 2,484次閱讀 1105人點贊

戰錘，這一充滿了力量感的武器，再搭配其使用者強悍的身軀。在帶給敵人恐懼的同時給予敵人沉重打擊。這些戰錘的使用者無疑在魔獸的歷史中留下了濃墨重彩的一筆。而如今這些手持戰錘的英雄們紛紛加入到了風暴英雄當中，那麼在這款暴雪推出的MOBA類遊戲中，這些英雄在又會有著怎樣的表現呢。

提起戰錘英雄，炎魔之王格拉納羅斯可謂是家喻戶曉。作為艾澤拉斯大陸的火焰領主，江湖人稱”大螺絲”的格拉納羅斯唯一的目的就是用火焰燃燒一切。其手中的戰錘——薩弗拉斯，放眼整個艾澤拉斯大陸的神器中也足以排到很高的名次。現在這位火焰領主早已加入了風暴英雄的大家庭之中，他的火焰又會帶給風暴樞紐怎樣的毀滅呢。

說道了炎魔之王格拉納羅斯，不得不想到黑石山；而提起黑石山則不得不提起建造起整個黑石塔的矮人們。別看矮人族們身材並不高大，可精通鑄造的矮人之中絲毫不乏使用戰錘的高手。穆拉丁銅須，著名的三兄弟里的老二。他是一個善良的矮人，他喜歡麥酒和朋友。作為戰友時你會感受到他天神下凡般的英勇——他是一個堅定的同盟。而當你們敵對時，要小心他的風暴之錘，他將會是一個兇猛的敵人。

說過了矮人族，讓咱們聊聊獸族的戰士們吧。擁有強健身軀的獸族戰士尤為喜愛諸如戰錘、戰斧這類重型的武器。而作為獸族部落大酋長的薩爾更是其中的佼佼者，其手中的神器——毀滅之錘，在給予了薩爾強大戰力的同時，還賦予了法術的力量。在薩爾的帶領下，解放後的獸人們來到了一片屬於他們的徒弟，力圖恢復先祖的榮耀。無論現在還是未來，他都將是部落的酋長。

在魔獸世界的懷舊服中”奧特蘭克戰道”這一戰場也是即將開啟。作為部落聯盟的重要戰爭之一想必老玩家們對此也是期待已久。等待之餘不妨打開風暴英雄，體驗一下別樣的奧山之戰。恰逢風暴英雄全新版本死亡之翼隆重登場，曾經購買過暴雪嘉年華門票的玩家們更是會直接獲得這個強有力的角色。還在等什麼呢，選擇死亡之翼，加入奧特蘭克戰道的對局。撕裂往日仇人薩爾的陣地，抑或是加入，並與其一起戰鬥。
