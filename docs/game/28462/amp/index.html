5個人3個月做的遊戲，賣了2000萬份！15年後炒冷飯還能賣200元？

天天要聞
2019-12-23

新的一周又開始了，本周又會有哪些好玩的新遊戲等著我們呢？一起來看看吧。

【本周新游測試】

劃重點：天涯明月刀手游原定於本周的測試確認跳票

天刀手游原本準備在12月26日再開啟新一輪的測試「明月測」，並且要在此次測試中推出全新門派丐幫，只不過非常不巧的是官方發布公告稱測試將會跳票，而且並沒有公布具體上線時間。

跳票其實都是常事了，老實說已經測試了這麼多次都開不了服，麻煩送審請快一點。

目前測試延期的原因主要是因為畫面、引擎等優化問題，預計過不了多久就會重新開啟測試，而且本次公布的丐幫也是端遊玩家非常熟悉的角色，相信端遊玩家到時候在手游上也能玩的得心應手。

X博士點評：就問一句，丐幫是不是還那麼菜。

劃重點：來看看網易的哈利波特 《哈利波特：魔法覺醒》12月23日開啟首測

這款網易製作的《哈利波特：魔法覺醒》是原著小說正版授權的手游，在12月23日會開啟小規模IOS首測。

這是一款卡牌類RPG手游，遊戲的核心玩法是卡牌的收集和戰鬥，玩家通過收集「昏昏倒地、除你武器」等經典魔法，來與敵人進行戰鬥。

嚴格的來說這還是網易最擅長的那一套RPG遊戲做法，好在遊戲中對於魔法的還原還算不錯，原作小說包括「神奇動物」中也確實有很多值得一戰的怪物和巫師，如果能做好的話玩家可玩的內容還是非常豐富的。

X博士點評：只要不讓我氪金強化魔杖、給魔杖鑲嵌寶石、十連抽魔法卡片就行。

【本周新游發售】

劃重點：網易城建式沙盒手游《第九所》 IOS首發定檔12月27日

在騰訊籌備了一款主旋律遊戲《家國夢》之後，網易以國家科研奮鬥史為題材打造的沙盒遊戲《第九所》也終於要在本周正式上線了。

遊戲中玩家將以研究員的身份來到祖國的西北大漠，從零開始建設科研基地，並且體驗真實的「兩彈一星」以及「載人航天」的科研奮鬥歷程。

遊戲風格還是十分「復古」的，畢竟為了尊重並還原歷史，遊戲中人物的服裝、建築、環境都取材自真實史料，提供娛樂的同時也同時具備科普價值。

《第九所》的玩法有些類似模擬城市，玩家一邊進行城市建造，一邊也要顧及科研問題以及突發的危機，可玩性還是非常不錯的。

X博士點評：讓玩家一步步體驗這段艱辛的歲月，確實是非常有意義。

劃重點：《成人腦力鍛煉Switch》公布 12月27發售、實體版售價230元

在12月27日，任天堂將會在Switch平台推出旗下知名益智類遊戲《成人腦力鍛煉Switch》。

在當年的NDS上這款遊戲也是非常出名的作品，5個人只用了三個月開發出來的遊戲，在2005年發售後就直接賣出了2000萬份。

這款遊戲在Switch上一共有三種輸入形式，一種是用Joycon的紅外攝像頭讀取手勢，一種是通過按鍵輸入數字，還有一種則是手寫輸入，非常的方便，適用於各個年齡段的玩家。

該遊戲除了可以領思維活躍、鍛煉腦力之外，還可以多人聯機對戰，在玩中學習和鍛煉非常容易被孩子和老人接受，用來教小朋友學習也是不錯的。

目前遊戲公布的實體版售價為3480日元（230元），數字版則是2680日元（177元），這樣的益智小遊戲買數字版確實是個不錯的選擇。

X博士點評：不愧是家庭主機。

劃重點：《崩壞3》國際版將於12月26日登陸PC平台

國內知名動作手游《崩壞3》國際版即將於12月26日正式登陸PC平台，同時PC版支持跨平台登陸，玩家賬號可以在PC和手機上通用。

本次上線的國際PC版本還並不能確定是否與手機版完全一樣，只能說希望PC版本能夠儘可能的優化下遊戲的操作手感和實機效果，不然機能提升這麼多還和模擬器版沒差別就有點多此一舉了。

不過可以確定的是畫面肯定比之前更好一些。

X博士點評：希望國內的PC版本也能夠儘早上線。

【本周爛片警告】

劃重點：避雷警告！12月27日上映的《羅賓漢：起源》國外評分極低

本周國內會上映一部來自美國的片子，名為《羅賓漢：起源》。

該片並非是一部新片，而是在去年11月已經在美國上映的老片子了，目前豆瓣評分只有5.1分，看樣子拍的並不怎麼好。

X博士去國外的網站上看了一眼，爛番茄給出的新鮮度竟然只有15%，看來豆瓣用戶還是手下留情了。

而且據說該片投資1億美元，但在全球上映之後僅僅收回了2200萬，是一部又不叫好又不叫座的大爛片，希望朋友們別去浪費這個錢。

畢竟最近葉問、誤殺、半個喜劇等國產片都算是高水準作品，實在不行去看看冰雪奇緣2他不香么？
