DNF12.15體驗服爆料：魔界大戰最沒牌面，直接開單人，送畢業符石

天天要聞

「地下城與勇士之小狐狸君愛談遊戲電競」第七百四十九期《DNF12.15體驗服爆料：魔界大戰最沒牌面，直接開單人，送畢業符石》，隨著韓服發布會的發布：100級、新技能三次覺醒、新團本希洛克Raid，國服也是加快了更新的節奏和步伐。魔界大戰成為最沒牌面的副本，史派克直接開啟單人模式、新出的活動直接就能拿到畢業級別的符石，可謂是開服就能畢業了。當然，春節騙氪更少不了這些道具和獎勵！目前體驗服僅僅是開放了歸來勇士活動，新一輪的活動會在下一次更新放出，不過現在的活動就送出了符石盒子（隨機道具），歐皇玩家直接就能拿到了。老規矩搞快點，直接上本次活動送出獎勵的截圖展示：

DNF12.15體驗服爆料：

本次體驗服魔界大戰的活動還沒有放出，現在僅僅是細枝末節的活動就開始送獎勵了，可見本次體驗服的更新，直接向玩家宣布：魔界大戰就是過渡版本、拖一拖節奏罷了。如果開啟的相關的活動的話，隨機盒子自然是少不了的，最終要的還是自選盒子，畢竟61個職業，每個職業4~6個技能，近300多個隨機，能隨機到一個本職業的都不錯了。

魔界大戰最沒牌面，

其實國服12.30就會發布100級了，所以現在別說是魔界大戰了，就連超界和普雷Raid都是已經成了活動送出的道具了，更別說幾個隨機的符文和護石了。其實95版本末期送出的副本：勇士對決開放沒幾天就涼了，現在看來魔界大戰就是一個類似BUFF換裝的副本，加大玩家的負擔罷了。所以史派克也是為大家減負了。

直接開單人，送畢業符石：

現在基本上可以確認的是：魔界大戰玩家就算是不參加活動，也是能直接刷單人模式，拿到畢業的神器符石也是比較簡單的，再加上配套的活動，拿到一些獎勵也是比較輕鬆的！到了100級符文也是會再次提升技能的等級和數量的。
