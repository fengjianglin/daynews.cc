LOL：轉會期再起風波，王思聰親自出面挽留阿水，FPX簽下國產中單

2019-12-11 629條評論 2,948次閱讀 796人點贊

點個關注，每天和你聊聊電競圈趣事！


時間來到快12月中旬之後，2019年的LPL冬季轉會期也就只剩下最後半個月的時間了，雖然咱們LPL表面上還是一如既往的平靜，但實際上，各大戰隊為了備戰主場舉行的S10，已經花費了很大功夫尋找優秀隊員為新賽季做補強了，只不過他們繼承了一直以來的「憋氣」習慣，不到最後一刻是絕對不會官宣的。

但在機緣巧合下，各種圈內人士也是爆料了許多我們想聽到的「內幕」，其中流傳度比較高的消息已經大部分都已經得到實錘了。

畢竟天下沒有不透風的牆，從上周末開始，LPL的許多隊伍都陸陸續續地開始宣布轉會期的消息了，像滔博官宣Karsa入隊，BLG也是簽下了Kingen和FoFo。

在12月10號晚，同樣是一位轉會期炙手可熱的選手也確定了新賽季的歸屬，WE戰隊也是官宣合同到期的Xiye正式離隊，這位從打職業開始就在WE的選手選擇了和Mystic一樣的道路。

在WE官宣之後，很多人都已經開始把目光轉向兮夜的下家，畢竟在此時官宣離隊，大概率是已經有了新的去向。

而兮夜的好朋友，同時也是WE的上單957在直播時像觀眾透露了Xiye轉會的消息，根據他所說的，Xiye下賽季大概率會加入DMO，而且這支隊伍目前在很多職業選手的評價都是非常不錯的，雖然是剛進入LPL的新隊，但潛力不可小瞧，有了兮夜的加入，他們下賽季也很有可能躋身黑馬之列。

在轉會期間，「被動爆料」已經是稀鬆平常的事，除了WE和兮夜，S9的世界冠軍FPX似乎也間接透露了他們下賽季的新陣容。

在結束了全明星之後，Doinb也終於回到了基地，並開始自己的直播，而且他在直播時點了外賣，一邊吃著烤串一邊和水友聊天，然而坐在旁邊訓練的Khan聞到味道實在忍不住，於是就走到Doinb面前讓他幫自己點一份炸雞。當然，這一切Doinb的觀眾們可是看得一清二楚，看到沸騰的直播間，Khan和硬幣哥在交流過後也對攝像頭揮了揮手，不得不說漢子哥還是很可愛的。

看到轉會秘密已經被公開，FPX官博雖然沒有官宣，但還是發了條微博調侃一下。

當然，FPX的轉會消息可不止一個Khan，前不久就有人爆料稱FPX下賽季還要簽一名國產中單，Doinb也在直播時被問及此事，他也在直播間證實了這個爆料，並給了水友們一些提示：

1、進過世界賽，成績也還不錯

2、國產中單

3、比較擅長刺客英雄

看了這3個暗示，網友的第一反應是「盧姥爺終於回來」，但思考過後，都知道這是不可能的事情。

眾所周知，一直以來，LPL出名的國產中單很少，能打進世界賽的就更加少了，所以在這三條線索下，水友們也能非常快速地鎖定目標，那就是Corn，他曾經跟隨當時的皇族在S4打進了世界賽的決賽。

最後要說的也是本次轉會期最熱門的選手，Jackeylove。

他在本次轉會期很有可能是人氣最高且身價最高的選手，之前有很多爆料傳出他的下家將是蘇寧戰隊，甚至網上還傳出了他和蘇寧簽合同的照片。

但到了昨天，又來了個劇情大反轉的爆料。

在昨天Baolan的直播中，觀眾能很明顯地聽到Jackeylove的聲音，這很有可能就代表著他現在還是沒有離開IG基地。

並且也有消息稱，IG為了留住Jackeylove已經為他連續三次漲薪，最後就連老闆王思聰都出面挽留，況且阿水自身其實也沒有很想換隊，所以就續約成功了。

當然在官宣之前，所有的爆料都是半真半假，我們還是需要理性吃瓜的。
