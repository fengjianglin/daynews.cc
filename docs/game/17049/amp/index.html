街機《圓桌騎士》埋沒多年的隱藏玩法，這才是高分玩家的不傳之秘

估計全國的玩法都差不多，在三關和最終關可以砍出來兩個固定的「2UP」，以及三關出一個「1UP」和一支「魔杖」，其他時間就是保命和刷分的過程。在遊戲廳時期玩家們並沒有打寶的概念，寶物的出現全都是看緣分，因此那時候最流行的玩法就是貪分。《圓桌騎士》中的分數翻倍應該是大部分玩家都了如指掌，只要連續砍死相同的敵兵就可以累積翻倍，最多可以累積到四倍。
