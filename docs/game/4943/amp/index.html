虎牙三季稻為證明身份，PVP熱情程度不減，登錄原賬號曬歷史成就

2019-12-03

摘要

對於魔獸世界那個在艾薩拉伺服器留下各種PVP相關事迹的傳說人物三季稻，大家得知他在虎牙進行懷舊服直播時，彈幕上一直有著質疑的聲音。三季稻本人做了一定的說明，但是信不信的問題可以通過他的一些遊戲玩法來看。

對於魔獸世界那個在艾薩拉伺服器留下各種PVP相關事迹的傳說人物三季稻，大家得知他在虎牙進行懷舊服直播時，彈幕上一直有著質疑的聲音。三季稻本人做了一定的說明，但是信不信的問題可以通過他的一些遊戲玩法來看。果然在野外遇見紅名的三哥是不會手軟的，原來因為「擊殺小號」引發了很大的爭議，但既然是對立陣營的設定，這樣的情況也無法避免。

三季稻在野外行動的時候，看似是很普通的在前進，其實給人的感覺就是時時刻刻在觀察周圍的情況。這回是因為在一塊礦石旁看到了紅名，直接衝上去就是一刀，一時間不知道是為了人頭還是為了礦石，可能剛入坑的萌新遇到這種天降陣亡會有些無法適應，但其實玩久了以後，就已經見怪不怪了，大不了有能力的再打回去。

這樣的行動已經能夠讓大家回想起熟悉的場景了，紅名只要在附近，選擇到了就可以攻擊，大概是一個正在做「失蹤人口」任務的聯盟玩家，三哥在看到之後還是一樣毫不猶豫的擊殺了。原來做這任務前期可能需要組隊打洞穴的小怪，要真是護送的玩家多的話，可能三季稻還要思考一下怎麼對付，都要途徑這條路，發生衝突也是正常不過。

三季稻在副本門口遇到紅名的時候猶豫了一下要不要打，因為這裡隨時能傳送走，不過似乎雙方都有要攻擊的意思，二話不說就在這一塊小小的空間互相攻擊起來。對方是個術士，但是對上三季稻似乎也沒有辦法，讀條一直在被打退，最終讓三季稻獲得了榮譽擊殺。之前低等級的紅名是沒有這樣的獎勵的，還是得有軍階才行。

為了讓各位前來圍觀的小夥伴們信服，三季稻登上了自己的原號，現場展示了一下自己的各種絕版成就，聲望列表，以及他累計的擊殺數，最活躍的地方在東部王國。賬號是找回的，身份信息畢竟是無法改變的。當然在正式服的ID也貼在了直播間中，以後大家就能知道誰才是真正的三季稻了。
