《最終幻想7核心危機》回顧：在重製版來臨前，千萬別再想起它

2019-12-20 773條評論 2,570次閱讀 50人點贊

距離《最終幻想7：重製版》發售還有不到三個月，心急的玩家只能每天看著官方發布的預覽圖，盼望時間過得再快一點。與此同時，還有一部分玩家準備利用這段空檔時期，再一次重溫《最終幻想7》以及幾款衍生遊戲，為接下來的戰鬥做足功課，以免分辨不清錯綜複雜的人物關係。

在過去的22年里，SQUARE ENIX圍繞《最終幻想7》本篇，曾推出前傳《核心危機》（PSP）和《危機之前》（手游）、外傳《地獄犬的輓歌》（PS2）、CG電影《聖子降臨》等多部作品。對於大多數國人玩家來說，最熟悉的莫過於《核心危機》（FF7CC）和《聖子降臨》（FF7AC），這兩大經典是不少粉絲入坑的起點。

在PSP盛行的時代，《核心危機》憑藉高清化的CG和遊戲畫面，是當時最優秀的JRPG之一，因此幾乎每一個玩家都玩過或聽過。在這部以扎克斯為主視角的前傳中，補完他與女主角愛麗絲之間的愛情線。遊戲最後以註定的悲劇收場，將扎克斯的武器、夢想都託付給真主角——克勞德，讓劇情回歸到《最終幻想7》的開端。

平心而論，《核心危機》將本篇劇情的來龍去脈，以及角色之間的羈絆刻畫得愈加豐富。儘管遊戲的戰鬥系統比較蛋疼，老虎機玩法稍顯拖沓，很難稱之為好玩。倘若事先玩過《最終幻想7》的話，在本作看到各種熟悉的「伏筆」時，的確能夠收穫滿滿的感動，結局演出甚至算得上是系列最佳。

但是，如果此前沒怎麼接觸過《最終幻想7》，貿然從《核心危機》開始遊玩，人物關係將會變得十分微妙。在本篇中有著微妙情愫的克勞德和愛麗絲，一不小心就會腦補成「武松和潘金蓮的橋段」——打著失憶幌子接近大哥的女人，難怪扎克斯墳前的雜草看著有點綠，讓人忍不住想給編劇寄刀片。

根據目前的情報，愛麗絲在《最終幻想7：重製版》的命運依舊沒有任何改變，否則克勞德見色忘義的形象怕是徹底洗不動了。不過，本次重製至少要推出3部作品才會迎來結局，在沒有看到令人心碎的那一幕之前，一切皆有可能！
