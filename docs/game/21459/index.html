坦度更兼傷害！刺客奧恩自帶三把電刀 雲頂雷霆刺化守為攻

2019-12-20 2,612條評論 4,125次閱讀 1666人點贊

今天為大家帶來一套既有趣味又十分強力的陣容——雷霆刺客。普通的雷霆刺並不稀奇，而這套雷霆刺是將本來用作防禦的守護神奧恩轉職為刺客，把盾變成了矛。

陣容介紹

羈絆：6刺客&4雷霆&2鋼鐵

英雄：奧恩、皎月、挖掘機、夢魘、奇亞娜、螳螂、劫、雷霆拉克絲

雷霆和刺客這兩個羈絆可謂是絕配，雷霆的傷害觸發是被對方暴擊或者打出暴擊。英雄的初始暴擊率為25%，6刺客能夠再加20%的暴擊，接近50%，也就是每打兩下就能觸發一次雷霆傷害，還不包括被對方暴擊的情況下。

3雷霆能夠提供300點的AOE，4雷霆的傷害更是高達500，這相當於什麼概念?一把電刀需要普攻3次打出一共300點傷害，雷霆只要命中兩個英雄普攻兩次打出1000點傷害，相當於奧恩裝備了3把電刀。讓這個守護神爆發出非常恐怖的傷害，堪比上版本海洋法裡面的火男。

裝備選擇

既然要讓奧恩轉職為刺客，那麼幽夢肯定是必備的， 因為現在降低了選秀出現鏟子的概率，賣血需要謹慎，實在不行也可以不用幽夢，走刺客+雷霆的組合，畢竟陣容的目的是上分，如果賣血太過早早的出局反而失去了上分的本意。

主C是夢魘，夢魘的單卡強度非常高，而且來的比較。三件暴力的輸出裝備讓夢魘能夠橫掃對方的後排。如果前面運氣好直接D出2星螳螂，也可以用螳螂來做C位，螳螂需要的裝備是飲血劍和大天使，用螳螂就把挖掘機換成鱷魚/輪子媽，組成2沙漠羈絆。

劫是陣容裡面的副C，有刺客和雷霆的雙屬性在，劫也可以打出奧恩那樣恐怖的傷害，而且劫分身同樣可以繼承這一切，給劫復活甲和無盡，戰鬥時間越長劫的戰鬥力就越高。

運營思路

裝備選擇金鏟子>暴風大劍大于格斗拳套，金鏟子不用多說，大劍提升的攻擊力是刺客最需要的，也是幽夢的散件，兩把大劍更是可以合死亡之刃。格鬥拳套可以用來合成無盡。

前期過渡陣容用3刺客+任意羈絆，最通常的是艾翁、大樹和妖姬再加上兩個刺客組成林地刺，不過若是運氣好前期就刷出或者掉了千珏，也可以用千珏加韋魯斯，組成3煉獄3刺客2遊俠，板子上放奧恩和狗熊。開局拿到金鏟子直接啟動，沒有金鏟子適當的賣血，在打石甲蟲之前稍微的用20金幣去搜皎月和奧恩，能夠追3就追，追不到也無所謂。

中期刷出來的刺客都拿，然後升到7人口，上6刺客2雷霆，這樣就有了初步的戰鬥力，可以把一些輸出裝備給妖姬，等到劫來了賣掉妖姬就行了。這段時間鎖血上人口，沒必要強行的去搜劫，板凳上放一個2星挖掘機。

等到8人把全部的錢拿去找劫，拿到劫就換掉妖姬，把裝備給劫。然後繼續存20金幣吃利息找拉克絲，等到雷霆拉克絲出來，賣掉狗熊上拉克絲，組成完美的4雷霆羈絆。奧恩和劫突到對方後排一次打出一片雷霆AOE，夢魘憑藉技能也能打出非常高的輸出，而且雷霆英雄被集火也可以用雷霆的AOE來反擊。

總結

這套陣容在兼具坦度的情況下還能打出超高的傷害，而守護神奧恩轉職刺客就是最關鍵的一步，大家不妨在排位中試試這套陣容。
