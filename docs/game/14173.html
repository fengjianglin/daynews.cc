DNF：12.10更新完畢，靈魂武器禮盒實測，確定可自選任意絕版武器

2019-12-10 914條評論 2,629次閱讀 459人點贊

DNF：12.10更新完畢，靈魂武器禮盒實測，確定可自選任意絕版武器

以下為具體的內容詳情：

在昨天無論是從DNF的官網上，還是助手上面，全部都沒有有關新版「戰令」玩法的活動，本來以為策劃感覺有點虧直接閹割掉了，但實際則不然，因為在目前跨五已經提前開服，已經確定戰令活動上線到的遊戲當中，而有關戰令活動中相關可領到的獎勵，其中大多說玩家最在意的莫過於那個靈魂武器自選禮盒了，其到底是半自選還是純自選一把絕版武器呢？

針對於上面這個問題，小寶首先來說下，玩家們如果想領取到這個活動中的靈魂武器自選禮盒，前提必須要通過氪金才可以，無論是49的還是156的，兩者相比只不過是一個慢一個快。而小寶為了圖方便選擇的是156的豪華禮包，然後在今天這上線的第一天就領到了這個靈魂武器自選禮盒。

此禮盒從活動中領到之後為賬號綁定，且在小寶經過實測開啟之後，如上圖所示，玩家們可以看到確定可以自選任意一把絕版武器，比如絕版的史詩級武器，領主級武器，神器級武器等，那這不得不說就非常良心了。

以鬼劍士職業的絕版靈魂武器為例，外形比較好看比較帥氣的實在是太多太多了，比如說巨劍類的魔劍、罪惡之枝，太刀類的十字斬刀閃、血舞旋月刀、月之光芒，短劍類的無影劍艾格斯，沙凡特的冰齒劍等等，小寶在這裡也無法說出到底那把武器的外觀好看，必定這是因人而異的存在，結束。
