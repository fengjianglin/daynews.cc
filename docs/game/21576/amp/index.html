鬥魚峽谷棋王爭霸賽11月決賽：秋日斬獲月賽冠軍 射手陣容成主流

天天要聞
2019-12-19

鬥魚峽谷棋王爭霸賽11月份經過兩周周賽的激烈爭奪，終於決出了最後月賽的參賽選手，下面小編就給大家帶來月賽決賽的相關戰況。

小組賽

第一組：

滅神和GKE神在三輪比賽中各自吃雞一局，並依靠穩定的積分排在前兩位，愛晨瀟和墮城三局都有得分進賬，分別以第三、第四名的成績進入到淘汰賽階段。懶饞Q雖然在第二局成功吃雞，但是另外兩局都沒有得分進賬，也是遺憾遭到淘汰。

第二組：

264連續三局都拿到前三名，以28積分穩穩拿下小組第一，星辰、290和QG大牛分別排在第二到第四的位置進入到淘汰賽。風之藍以一分之差無緣晉級，略顯可惜。

第三組：

鈴鐺鐺雖然在首輪只獲得了兩個積分，但是後面兩輪分別拿到第一和第二名，最終後來居上位居第一。江流兒和葉凡均完成一局吃雞位居第二和第三位，永恆G憑藉三輪穩定得分進賬，以第四名的身份晉級。

第四組：

QG不加冰表現非常強勢，連續三輪都進入到最後的決賽階段，最終也是以30分遙遙領先其他人。QG一念、花謝和秋日分別排在第二到第四的位置進入淘汰賽。威廉比較可惜，以一分之差無緣淘汰賽。

淘汰賽

第一組：

秋日雖然在四輪比賽中均未實現吃雞，但是每一輪都有著穩定的得分進賬，這也幫助秋日成功獲得了小組第一。不加冰在前兩輪成功吃雞，無奈後面兩輪表現不佳，不過憑藉之前的積分優勢排在第二。鈴鐺鐺和滅神均以22分進入最後的決賽。

第二組：

264表現非常強勁，在四輪比賽中三次吃雞，最終以45分力壓眾人強勢登頂;QG大牛表現同樣不俗，在三輪比賽中都拿到了前兩名，最後位居小組第二;花謝和愛晨瀟分別以第三和第四名的成績晉級決賽。

決賽

進入到決賽階段的八位選手分別是秋日、不加冰、鈴鐺鐺、滅神、264、大牛、花謝和愛晨瀟，這八位選手將通過五輪比賽來爭奪11月月賽的冠軍。

秋日在前四輪中表現不錯，均拿到了前三名的好成績，不過在第五輪中想要選擇一個非常規陣容最終沒能獲得積分，但依然拿到了第一名的好成績。滅神雖然吃雞兩次，但是第三輪和第五輪沒有獲得理想的積分，最終以30分位居第二。愛晨瀟和鈴鐺鐺分別獲得了第三和第四名。

決賽中，選擇最多的陣容還是射手陣容，足以看出射手陣容在模擬戰中還是非常受玩家青睞的，感興趣的玩家可以試試射手陣容的效果!
