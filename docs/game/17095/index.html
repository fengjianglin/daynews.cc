神超雲頂之弈新版本評級：雷霆劫天下第一，劍刺第二，掠食者第三

2019-12-13 1,408條評論 3,711次閱讀 2748人點贊

現在雲頂之弈已經更新到了9.24版本，相信很多玩家都感到幸災樂禍吧？畢竟法海這個陣容，遭到了削弱，不再是一家獨大的局面了。不過法海是被削弱了，但是因為綠茶的改動，現在版本刺客又回來了！

為此，素有雲頂導師之稱的神超，在試玩了一天新版本後，也是簡單的分析了下雲頂之弈新版本。首先在陣容上，神超做了一個簡單的評級：雷霆劫天下第一，劍刺天下第二，掠食者第三，第四則是光影（盧錫安和賽娜的陣容）。

相信神超給出的這個排名，大家都非常認可吧？實際上雷霆劫在上個版本就已經很火了，誰也沒想到9.24版本中竟又加強了一波雷霆劫的傷害，讓不少玩家都有點一臉懵逼。為此在現今劍刺橫行的版本中，唯有雷霆劫方可一戰，這也是為什麼神超說「雷霆劫天下第一」了。

不過有點尷尬的是，雷霆劫和劍刺在前期度過的時候都非常難，特別是雷霆劫，前中期必須保持好血量和經濟才能上人口找劫，否則還沒有遇到劫就已經暴斃（神超遊戲過程中就有對手遇到這樣的情況）。劍刺的話，則是比較吃裝備，也就是鏟子這件裝備，一旦找不到鏟子，意味著劍刺無法成型，C位必須是帶劍士裝備的夢魘！

第三個掠食者的話，則相當比較簡單的，前期瘋狂D就完事了，然後在有三星大嘴的基礎上，去尋找煉金。但是最好要將鱷魚或者狼人也D成三星，這樣中期可以保證質量不掉血，然後備一本鬼書給煉金即可。第四光影這個陣容的話，神超表示還在研究中，盧錫安主C，賽娜給個復活甲保證生存率即可，不過目前這個陣容想吃分的話還是有點難的。

那麼最後，你們是怎麼評價神超這個陣容評級呢？
