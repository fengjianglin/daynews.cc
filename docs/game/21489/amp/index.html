Zard翔哥直播被人噴，惱羞成怒反擊失敗，還被叫“雙標人”?

天天要聞
2019-12-19

話說遊戲主播被人罵了究竟能不能反擊？

這事還真不好說，不過鬥魚DOTA2區的知名主播Zard，俗稱翔哥就遇到這個事情。

而經過不斷發酵事情變得愈發不可收拾，甚至還引起了DOTA2玩家眾怒。

整件事情經過並不複雜，近日翔哥在單排上分時玩了一把中單帕克，對手則是中單血魔，對線期翔哥帕克被血魔壓製得很慘。

被單殺2次之後，翔哥放話說在這麼打下去，自己要掛機鑽入野區了，此時隊友劍聖回應「你中路掛機都能帶你贏，別鑽野，把中路防禦他守好。」

並且這期間隊友還幫助翔哥遊走一波，成功擊殺中單血魔。

原本殺完血魔，翔哥帕克在中路應該可以混得很舒服了，但誰都沒想到，翔哥居然TP去上路跟大哥劍聖搶兵線補刀。

之後雙方之間你來我往爆發不少激烈團戰，雙方有輸有贏，而翔哥帕克則自顧自的在野區刷野，偶爾還送上一兩次。

這時候劍聖看不下去了，趕緊叫翔哥來參團打架。

翔哥則陰陽怪氣地回應到「你不是說，我掛機都能贏嗎？」後來嘛，劍聖直接開噴翔哥，罵得極其難聽，而翔哥自然不服將原話奉還，但翔哥卻不小心加了一個「也」字。

這個細節被劍聖抓住，大作文章，在文斗這塊翔哥只能敗下陣來，不過誰都沒想到打到比賽結束，翔哥這邊居然還贏了，賽後翔哥實在是氣不過，便舉報了劍聖。

舉報完成後，翔哥隨即委屈地說「是他先開口罵的我，我罵他不是噴人而是詛咒、我什麼時候主動噴過人別人，你們這是雙標。」

總之，翔哥用他那特殊的口吻，將整件事情說的相當委屈。

不過隨後便有細心的粉絲和玩家，找到了翔哥之前噴人得視頻，好巧不巧，翔哥說的話也跟劍聖說得差不多。

根據粉絲提供的視頻資料，翔哥在2018年11月左右的比賽裡面，使用中單SF，比賽開始1分鐘內翔哥SF還沒有補刀一個刀，不僅如此他還遭到對手發條和白牛的多次Gank，數次沖塔擊殺翔哥。

陣亡後翔哥開麥怒噴隊友，跟劍聖說得話如出一轍，甚至還質問隊友，你們憑什麼打DOTA，隨即秒退，整件事情發生期間，隊友沒有噴過翔哥一句話。

看到視頻之後，玩家徹底坐不住了，開始紛紛聲討起翔哥，說他雙標對人，說他是巨嬰，說不配打DOTA2。

眼看直播間彈幕越來越多，翔哥的房管們又開始整起活了，他們不僅封禁了不少玩家賬號，還在翔哥魚吧威脅彈幕玩家，說他們認識不少人，不怕被封號儘管來。

這下整個事情就變味了，本來一個簡單的噴人事件，變成了玩家與主播之間的對抗，翔哥被人噴了，原本他算是受害者，卻因為自己的一些操作，導致他成為了事情焦點。

另外翔哥的房管也因為手段太過強硬，不顧事情發展隨意封人，也讓玩家接受不了。

不過對於看戲吃瓜的玩家來說，他們可是樂意事情發展越來越大才好，就是不知道翔哥會如何應對了。

各位玩家如何看待此事呢，不妨暢所欲言。
