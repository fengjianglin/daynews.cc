王者榮耀射手一姐，孫尚香認第二誰第一？

天天要聞
2019-12-18

​相信每一個王者榮耀的玩家都對孫尚香這個名字不陌生，沒錯，就是那個一套技能秒一個英雄，不管是哪一個版本都能成神並且從未跌下過神壇的射手一姐。那麼，孫尚香的具體打法究竟是怎麼樣的呢?

孫尚香的定位是射手，而射手都是主打後期的，所以前期的時候一定要猥瑣發育，如果不是有必要，那就盡量不要去打架，畢竟傷害才是真理，只有裝備買好了才有傷害。

首先是出裝順序，第一件裝備是極速戰靴，提高移速的同時還有攻速加成，孫尚香不像虞姬和伽羅那樣完全是靠平a打架，所以第二件裝備是無盡戰刃。

雖然不是完全依靠平a，但還是需要宗師之力的，畢竟在技能用完的情況下也是需要平a傷害的，然後是暗影戰斧，如果對方太肉可以換成碎星錘，再是破軍，最後是破曉。

在有破軍和破曉的傷害加成下，孫尚香的暴擊效果可見一斑，出滿六神裝的孫尚香一套技能都出不完就能隨隨便便秒一個脆皮 ，就算是坦克，血量也是以肉眼可見的速度下降。

再是技能方面，孫尚香的一技能是翻滾後強化下一次平a，二技能朝指定的方向投擲技能，對敵人造成物理攻擊和減速，三技能朝指定的方向突擊，並造成高額的物理傷害。

一技能配合二技能的物理穿透可以對敵人造成更高的傷害。先點擊一技能翻滾，然後預判敵人移動的位置投擲二技能。

最後是召喚師技能，一般情況下是選擇閃現，但是也要根據敵方的陣容選擇召喚師技能，如果對方有妲己或者宮本這類英雄，那召喚師技能首選是凈化，畢竟妲己的控很傷腦，後期被宮本大到了也是毫無生還的可能。

不知道王者榮耀玩家們一開始是抱著怎樣的心態去玩孫尚香的？作為一個技術不怎麼好的王者榮耀玩家來說，我選擇孫尚香只是因為她有位移，逃跑的時候比較快，後來卻發現了她的爆發力極強，完全是一個上分小能手。

只要前期猥瑣發育，在經濟上領先，後期的傷害可謂是無人能敵。就算前期處於逆風，但是到了後期也一定有翻盤的可能。

孫尚香雖然沒有虞姬的免疫，沒有伽羅的手長，還沒有公孫離那樣的位移，但射手一姐的位置卻不容置疑的。
