《夢幻西遊三維版》主要玩法一個不落，首充小悟空，真香

2019-12-18 415條評論 1,567次閱讀 336人點贊

上次我們講到了完成了第一個組隊任務，以及對人氣值的一些介紹。那麼本期呢，我們繼續追尋著我小女兒的足跡，繼續探索這一片神奇而又迷人的三界大陸。看看這次《夢幻西遊三維版》又會帶給我們哪些驚喜呢？

日常上線之後，還是跟著主線任務跑地圖，在長安城簡單跑了幾步之後，我發現竟然沒有主線任務可以做了？！Excuse me？我現在才23級好吧，這下一個主線開啟等級竟然要足足30級？我可是足足差了7級啊！您這系統也不給我發發福利、漲漲經驗啥的？不過轉念一想，本身電腦版打經驗的時候主要就是依靠抓鬼這些的，轉型成為MMO之後，留下了這麼一塊等級空缺似乎也是想要玩家自己去做作別的任務，從而更好的體驗劇情？再想到似乎前期升級也不是那麼的麻煩，我就強忍住罵街的衝動，去做一做支線任務，看看有沒有升級快的方式。

打開行游錄的行歷之後，順利領取到了一個任務，下面我就做一做看看，這個每日領取數量都有限制的支線任務，會不會帶給我驚喜呢？

花了小半分鐘跑到任務地點後才發現，原來這個任務並不是那種直接給予海量經驗的任務，而是贈送給大家「生活」模塊的工具，並簡單教會大家如何使用「生活」模塊的一些功能。雖然經驗沒有撈到多少，但是又開啟了一項新玩法，我還是比較開心的。我這裡也推薦大家在完成20級的一系列主線任務之後，可以順手就把這個任務做了，浪費不了多少時間不說，還能開啟一項全新的玩法，可謂是一舉兩得了！

做完這個任務之後，我再去看看其他的任務，有沒有那種「量大管飽」，讓我一口氣升到30級的任務。不過話說，在此之前，我瞄了一眼各種活動，正好看到了首充送小悟空的活動，我合計著反正也不貴，買來看看的思想，就順手買了一個。

買之前擔心新寶寶等級不夠跟不上節奏，但是領取到了之後我才發現，這隻小悟空是20級的，不用擔心等級問題不說，雖然在資質和面板屬性上面並沒有高出最開始的泡泡多少，但是它整整比泡泡多了快一倍的技能。這樣一看，這隻寶寶在後期的時候肯定是沒法拋頭露面的，但是用來渡過前中期這一段時間絕對是夠用了，怎麼說呢，畢竟是首充6R免費送的，用起來那還真挺舒服的（真香）。

兜兜轉轉接了一個桃源村的拍攝任務，任務要求是要去桃源村的大樹下，拍攝一張跟給定照片相似的照片並分享。

這裡不得不感嘆一下，這個任務有點黑科技哦，它可以自動檢測出來兩張照片的相似度並給予評分。並且溜達溜達看看桃源村的美景的功夫，任務也能順手完成，這裡我讚美一波這個任務奧！完成任務之後給了一枚避水珠，並沒有經驗和銀幣的獎勵，所以這裡我推薦大家還是休閑的時候來做一做，忙著沖級的話還是先把這個任務放一放！（不過話說為什麼我拍這麼好看才給我36％的相似度？GM咱倆談談！）

做了一下唯一支線任務之後，發現給的經驗和銀幣還是挺多的，足夠大家升到25級，並且一整套流程下來也就十分鐘左右，對於前期沖級的小夥伴還是比較友好的，這裡強烈安利一波那唯一一個的支線任務。

逛了一圈下來，發現給經驗最多的任務還是師門和捉鬼，較比於捉鬼還有那麼一丟丟費力，師門完全就是放空自我的最爽辦法，不僅全程自動尋路、自動打怪，以我26級舉例，一輪五個任務下來，少說也有5W多經驗了，在前期升一級只需要2、3萬經驗的前提下，一輪師門下來就足夠大家升上個2級了。最最主要的是，一輪師門最多只需要5分鐘就可以完成，簡直就是不費時不費力的懶人必備選擇。

不過大家需要注意的是，《夢幻西遊三維版》裡面，這種高獎勵的師門任務一天只可以完成兩輪，這之後再完成就跟電腦版的一樣，會損失一定的收益了！不過《三維版》有一個額外的任務，那就是每日完成兩輪師門以後，會解鎖一個特殊師門任務，完成任務之後會給予相當於三個普通師門任務的經驗量，同樣一套流程下來也就3分鐘的事兒，所以說大家在完成高獎勵師門的時候可不要忘了順手做了特殊師門任務哦~
