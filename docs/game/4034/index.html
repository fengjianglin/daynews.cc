草叢三姐妹之一：安琪拉史詩級調整，大招旋轉180度，傷害爆炸

2019-12-10 0條評論 18次閱讀 0人點贊

摘要

安琪拉這個英雄相信大家都不陌生，畢竟都是從新手期就開始陪伴大家的老牌法師英雄，上次返場皮膚投票，小廚娘還一舉超過大聖，可謂是風頭正火。最近安琪拉在體驗服迎來了自上線以來第九次加強，這次的加強可以說是史詩級別的調整，趕緊跟老王來看看。

安琪拉這個英雄相信大家都不陌生，畢竟都是從新手期就開始陪伴大家的老牌法師英雄，上次返場皮膚投票，小廚娘還一舉超過大聖，可謂是風頭正火。最近安琪拉在體驗服迎來了自上線以來第九次加強，這次的加強可以說是史詩級別的調整，趕緊跟老王來看看。

​

官方給出的說明，指出了安琪拉二技能的命中率在王者的表現非常低迷，在上次調整整體移速後，這個技能飛行速度和其他英雄的移動速度比起來太過吃力了，在正面戰場上非常容易避開，所以混沌火種的飛行速度從原來的1250提高到1450，這個改變老王測試跟之前沒調整前的命中率應該相差無幾，大家也可以去體驗服測試一下。

​

最重要的是大招的轉速加強，一如所有問題，越簡單的語句發生的事情越大條。

​

體驗服這次調整，安琪拉放大的之後的轉向跟以前笨拙艱難的轉向不一樣了，可以迅速地轉出180度。以前放大後對手只要反向跟你繞圈就毫無辦法，現在對手繞圈可以快速調整轉向實現精準傷害。

​

不少人認為的安琪拉嚴重超標了，傷害爆炸，一套技能全中秒殺坦克不是難事。可以說坦克在安琪拉面前跟紙一樣，後期可能會對坦克進行調整，安琪拉應該不會削弱，畢竟只是增強了飛行速度和轉向速度，接下來看天美怎麼調整吧。

​

從正式服來看，安琪拉的勝率不算高，只能在中游的水準，比起其他草叢三兄妹成員，妲己，王昭君來說就略差一點，可見這次更新後測試通過移植到正式服，安琪拉的數據會得到一個回升。

小夥伴對於此次調整有怎麼看呢？還有哪些被躲在草叢埋伏的慘痛經歷呢，歡迎評論~
