網友熱議：移除LOL動態登錄界面，是拳頭今年最糟糕的一個決定

2019-12-11 306條評論 1,225次閱讀 180人點贊

或許很多國內的LOL玩家並不知道，外服的英雄聯盟原生登錄界面已經在幾個版本之前就進行了一次大的調整，國服玩家因為大部分都是常用使用TGP登錄遊戲，所以可能已經有好多年不知道如今的LOL登錄界面變成什麼樣了。

其實從這個版本開始，拳頭就開始有意弱化整個登錄界面中英雄聯盟的相關元素了，並且將原來的動態登錄界面移除，只用使用靜態圖片的模式。

而下面則是目前外服正式服的最新的登錄界面

（更新界面）

（登錄界面）

可以明顯的看到，就連之前的靜態圖片也被移除了，只留下這個大大的「PLAY」字樣。對此不少外服的玩家都表示不滿，尤其是對拳頭移除動態的登錄模式表示很滿意，有網友就指出，這應該是拳頭本年度做出的最糟糕的一個決定。

1、沒有了動態登錄模式，無法在界面中給新英雄給予更多的展示，當新的英雄以動態的形式展現來玩家面前時，玩家可以看出更鮮活的看到該英雄的一些特色和主題風格。

2、不能再展現英雄的專屬BGM了，很多英雄都有標誌性的音樂主題，這可以很好的將音樂與英雄進行聯繫起來，比如金克斯、盧錫安、妮蔻等等，另外還有一些主題活動內容，比如之前的電玩、星之守護等等，只要打開遊戲的登錄界面就能提前知道這些活動

3、對於重大遊戲活動的宣傳，在美服，不少玩家甚至不知道最近有全明星比賽，因為在新的登錄界面中，完完全全沒有該活動的任何內容，雖然遊戲內有相關活動的宣傳，但如果能將活動展示在點開遊戲圖標的那一刻，豈不是更好？

對此有外服玩家建議，拳頭應該在登錄模式上給玩家最大的選擇權，讓玩家自主選擇使用舊版還是新版，雖說新版的登錄界面在一定程度上可以讓遊戲的載入更順暢，但如果能將選擇權最大化的交由玩家，這無論是對於拳頭還是玩家都是雙贏的局面，當然最終怎麼樣還得看拳頭自己的選擇。
