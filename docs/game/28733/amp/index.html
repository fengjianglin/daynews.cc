LOL雲頂之弈第三賽季已在開發中，設計師分享內測趣圖

天天要聞
2019-12-22

在今年11月開始時，英雄聯盟迎來了9.22版本的更新，在這次版本更新中，雲頂之弈迎來了第二賽季的改動內容，元素崛起版本加入到遊戲中。畢竟，雲頂之弈模式是一個自走棋類型的遊戲，如果不頻繁進行遊戲改動的話，肯定是一個陣容一直強到爆炸，所有人都是玩那幾個套路了。

由於雲頂之弈第二賽季的更新內容比較多，絕大部分的英雄卡都換掉了，而且羈絆效果大洗牌，還引入了元素地形增加遊戲的隨機性。因此，有些玩家對於這次改動並不太滿意，不過，雲頂之弈第三賽季很快也會到來了。在12月21日時，設計師在推特上公布了消息，他們已經在研究第三賽季了。

設計師在半年前就已經說過，雲頂之弈的賽季時間比較短，一般是3個月左右，第一賽季是從7月持續到10月，第二賽季很有可能就是從今年11月持續到2月上旬。而雲頂之弈一般會提前半個月在測試服進行測試，所以在1月中旬的時候，我們基本上就知道第三賽季的雲頂之弈有什麼全新內容了。

手哥則是希望新賽季不要改得那麼複雜，畢竟玩家的學習成本有限，遊戲更新太快的話，不看攻略都不會玩了。而且，下個賽季雲頂之弈更新時，基本上手游版本也出來了，設計師很有可能會針對手游進行優化。除此之外，設計師也分享了雲頂之弈內測時的一些照片，看起來和現在截然不同。

在最早版本的雲頂之弈內測時，界面真的非常簡陋，甚至和其他自走棋有點相似。而且，這個內測版本似乎有BUG，有一個非常大的峽谷迅捷蟹，而這個野怪在後續開發時被移除了。有外國玩家則是覺得設計師可能會在新賽季增加更多的野怪，提高裝備的掉落率，並且給予劣勢玩家發育機會。
