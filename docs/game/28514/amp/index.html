VG奪冠隊員喜極而涕！他們這次CFS到底賺了多少錢？

天天要聞
2019-12-23

2019CFS世界冠軍歸屬巴西VG戰隊，奪冠之後，VG的小夥子們顯得非常興奮，mazin比賽結束淚流滿面，dgz也身披國旗跪地痛哭。世界冠軍的榮譽確實令人激動，但對於這些剛剛成年的年輕人來說，更重要的是他們獲得了一筆豐厚的收入。

下面我們就來一起了解一下他們在這次CFS世界總決賽收入了多少。本次冠軍的獎金為50萬美元，相當於350萬人民幣。而他們參加的別的比賽，CFEL冠軍獎金為2.5萬雷亞爾，相當於4.5萬人民幣。巴西邀請賽冠軍獎金1.5萬美元，相當於10.5萬人民幣。一個CFS冠軍，等於他們拿下77個CFEL冠軍或者拿下33個邀請賽冠軍！

那麼選手實際能夠拿到手的錢有多少呢？中國法律規定，境外企業在國內取得所得要繳納10%的企業所得稅，剩下的錢如果按照國內俱樂部的平均規定，要抽5成左右，這樣算下來，如果不算各種巴西的其他稅費，選手每個人能夠拿到手的錢大概在20-30萬之間。

其實這樣一筆錢對於中國的選手而言並不算多，也就是大多數選手一年的工資。像絕跡、馬哲、石頭這些直播做得比較好的，一年賺的錢恐怕是這些獎金的好幾倍甚至十倍。而且CFPL的冠軍為150萬人民幣，高達CFS的一半。

但對於巴西的選手而言卻是一筆巨款。因為巴西的選手非常年輕，剛剛成年並開啟職業選手之路，他們的工資很少，月薪只有兩千人民幣左右。巴西的直播環境賺到像中國這麼多的錢也不現實。所以他們賺錢的最好渠道就是比賽獎金。

說了這麼多，想必CFer們可以理解VG小夥子們激動地心情了吧！如果他們不是富二代的話，這個冠軍就相當於他們人生中的第一桶金，開啟走向人生巔峰的大門，換做是誰都會非常激動

最後，恭喜VG的小夥子們奪得本次CFS世界總決賽的冠軍。期待他們能夠繼續努力，在未來的比賽中拿到更多的冠軍，收穫更多的榮譽！
