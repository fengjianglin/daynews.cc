Nuguri傑斯1-11，被Doinb舉報了：這個上單在演！牛寶：我儘力了

2019-12-12 978條評論 4,766次閱讀 1481人點贊

相信很多觀眾朋友們都知道每次S賽過後都有一個Dade獎，這個獎項的設立就是為了頒給像Dade這樣的選手，賽前被寄予厚望，比賽時各種拉垮，各種掉鏈子，最後成績也很差這樣的選手。S5世界賽這個獎項頒給了韋神，S6世界賽頒給了廠長，S7世界賽的鼠王Pary，S8世界賽頒給了UZI，那麼S9打完之後這個獎項頒給了誰呢？相信很多觀眾們都已經知道了，這個獎項就頒給了DWG的上單選手Nuguri，原因就是比賽中太拉垮了！最近Nuguri可以說諸事不順，比賽中拉垮，排位中也是慘遭隊友舉報，牛寶要哭了！

最近Nuguri在直播中排位遇到了Doinb，本來是一場穩贏的排位賽，但是沒想到的是Nuguri又開始遭到針對了，一手傑斯在上路無限被GANK，被抓得Nuguri都想哭了！時不時就有三四個人來上路GANK，Nuguri被兩個人包夾往中路跑，但是在半路又遇上了亞索，絲血的Nuguri直呼絕望，這個遊戲怎麼這麼難啊？

相信很多玩家朋友們都知道，一旦傑斯這樣的英雄被壓制，遊戲中期本該傑斯站出來的時候站不出來，那麼這一局傑斯基本上就已經廢了！然後Nuguri就開始操作變形，各種被對面找到機會擊殺，最後戰績也被殺成1-11！我們都知道Doinb是排位的監督員，誰要不好好打排位必定受到Doinb的舉報，這一局Doinb也是直呼這個上單在演呢？怎麼打著打著1-11了呀！我出去之後必舉報他！

Nuguri和Doinb在一個平台直播，彈幕也是告訴Nuguri你被Doinb舉報了，Nuguri也是直呼難受！這一局真的不是我送的，確實是對方一直來抓我，我也沒有辦法啊！動不動就是四個人包上路，團戰時候對方打野酒桶一個大招，亞索接上一個大招，根本就沒有辦法打團戰，只能被對方殺，1-11確實不是故意的，已經儘力了！

看到牛寶都要哭了，觀眾們也是直呼太搞笑了！小將Nuguri能幫助DWG進入世界賽就已經很不錯了，下賽季繼續努力，希望能夠在世界賽上再次看到Nuguri的身影！小夥伴們對於硬幣哥這波舉報怎麼看呢？歡迎在下方留言。
