FGO弓大王簡評：周回高難、輸出輔助全方面發展，二號機已經就位

2019-12-14 1,714條評論 3,310次閱讀 2317人點贊

FGO國服正在進行聖誕節復刻活動，本次活動可以再次獲得四星從者弓階聖誕阿提拉，下面我們就來分析一下弓大王的情況。

從者寶具：弓大王的寶具為綠卡單體輸出，並獲得20~40個暴擊星（效果隨OC等級提升）。

從者技能：一技能的效果為己方單體NP增加10%~20%，暴擊星掉落率提升30%三回合，充能時間7~5回合。

二技能的效果為提升己方單體20%~30%寶具威力以及暴擊威力一回合，充能時間7~5回合。

三技能效果為賦予自身無敵貫通狀態三回合，暴擊星集中度提升300%~600%三回合，防禦力提升20%~30%三回合，三回合後賦予自身眩暈狀態一回合（副作用）。

職階技能【對魔力】自身弱化耐性提升15%，【騎乘】自身綠卡性能提升12%，【文明侵蝕】自身暴擊威力提升10%，【神性】自身傷害附加175。

從者優點：①綠卡寶具在當前有絲卡蒂輔助的環境非常吃香，攻擊buff有寶具威力提升和優秀的騎乘效果，加上活動從者保送五寶的傷害非常可觀。

寶具的產星效果和自身技能的暴擊威力提升效果以及集星效果，和絲卡蒂的契合度也很高，可以打出不錯的暴擊傷害。

②自身有充能效果，在周回上較容易開啟寶具，打道中的劍階或狂階大怪都有較高的價值。

三技能有持續三回合的無敵貫通和防禦力提升，在高難的作用也很突出，原本有個眩暈的副作用來牽制這個技能的效果，但是在指令紋章【白色器物的令印】實裝之後，完美解決了這個問題，眩暈延遲三回合，對指令卡的發牌要求也不高，可以說是非常契合了。

③弓大王的一技能和二技能的效果十分靈活，可以對己方單體釋放，也具有不錯的輔助效果，復刻活動可以獲得弓大王二號機，也就意味著一個弓大王可以培養完全體，還可以再培養一個一破的插件。

從者缺點：①弓大王二技能的效果只持續一回合，這也是唯一的提升輸出技能，所以持續輸出能力還是有點不足，但是這個劣勢在絲卡蒂落地之後緩解了很多啦。

②弓階的競爭十分激烈，不僅是光炮，單體也是一樣，輸出有小黑、崔斯坦這種開寶具更容易的從者，特殊情況下，三星的二姐也值得一提。輔助方面，「弓」具人也有非常多，其中喀戎、弓海倫娜、災星簡、聖誕南丁等也都有很高的輔助價值，弓大王僅有容易獲取的價值比較高。

整體評價：弓大王無論是周回還是高難本，當輸出還是當輔助都有較高的價值，也算是非常合格，非常難得的弓階從者了，缺乏弓階單體的情況下，弓大王是非常值得培養的從者。

不過弓階單體也不會缺乏，明年二月份的魔伊聯動復刻還有獲得小黑的機會，當然二者都是活動從者，培養也是比較容易的。
