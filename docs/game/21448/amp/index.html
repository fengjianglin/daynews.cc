大神爆料任天堂2020年將是瘋狂的一年！第一方遊戲數量很驚喜

天天要聞
2019-12-20

很快就要進入新的一年了，雖然任天堂之前就已經把2020年Q1的遊戲陣容安排了幾個，不過整體來說除了《動物之森》新作之外，好像沒有太多第一方的新作。

當然這並不是代表任天堂沒有新遊戲，而且任天堂肯定是在憋一個新的直面會了!因為具體上一個直面會已經過去好幾個月了!

最近爆料大神Emily Rogers終於出來爆料了!

雖然爆料並沒有列出詳細的遊戲名單，但是直白的表示2020年對於Switch來說又將會是偉大的一年。這個偉大是指任天堂的第一方數量方便，與此對比的是，他表示2017年和2019年是偉大的，而2018年則比較平庸一點，即使有大亂斗SP。

也就是說，2020年任天堂的第一遊戲數量應該和2017年和2019年接近，在個人喜好上，他表示2020年將會是瘋狂的一年，玩家將會收到各種第一方、第三方和獨立遊戲的轟炸。

還有他也表示，任天堂已經為《星際火狐》(應該是新作了吧)製作了一些demo，雖然目前不清楚是不是會最終變成一個完整的遊戲。但是證明任天堂並沒有放棄《星際火狐》這個IP，任天堂仍舊在嘗試一些新的創作和概念。

Emily Rogers的爆料一直還算挺準的，之前成功的爆料了《出發吧皮卡丘伊布》!
