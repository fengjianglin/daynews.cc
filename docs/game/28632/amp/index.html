DOTA2：中國隊伍最後的希望？VG擊敗Aliance晉級敗者組決賽

天天要聞
2019-12-22

DOTA2新加坡邀請賽淘汰賽來到倒數第二個比賽日，而中國唯一的獨苗VG戰隊依舊在堅持，他們此次面對的兩個對手分別是Secret戰隊和Aliance戰隊。第一場比賽VG2-1強勢逆轉了Secret戰隊，而在第二場比賽當中，他們將面對近期表現不俗的Aliance戰隊，雙方究竟誰能夠進入最後的敗者組決賽呢？

​第一局比賽，VG選擇了飛機和血魔的組合，三號位則是拿到了末日，醬油選擇了拉比克和冰龍。Aliance這邊則是選擇了水人和帕克，三號位則是小小，醬油拿到了冰女和發條。

​開局VG戰隊拿到了很大的主動權，小小配合冰龍多路開花，Aliance的多路發育陣容並不能和VG相匹敵。11分鐘，雙方下路交戰，血魔一馬當先擊殺了冰女，雖然後面帕克擊殺VG兩人，但是血魔復活歸來之後直接團滅了Aliance。17分鐘，雙方上路交戰，雖然擊殺了冰龍，但是依舊慘遭團滅。25分鐘，VG經濟已經全面領先Aliance了，並且超過的數量還很多，但是Aliance依舊不選擇投降。32分鐘，VG拿下不朽之守護，之後穩健的平推掉了Aliance。

​第二局比賽，VG拿到了伐木機和小魚人的組合，三號位則是選擇了亞巴盾，醬油出人意料的選擇了帕克和大樹的組合。Aliance則是選擇了中單猛獁，一號位則是拍拍熊，三號位則是選擇了獸王，醬油則是天怒和薩爾。

​第二局比賽Aliance一改之前的頹勢，在前期打出了很強勢的進攻來避免第一局的頹勢。10分鐘，VG拿到了雙前哨。13分鐘，VG擊殺了Aliance三人。15分鐘，Aliance抓住VG戰隊的失誤，直接一波團滅，並且收下roshan。29分鐘，VG瞅準時機秒了天怒，接著收下了二代roshan，而盾奶則是讓給了格子還沒有滿的樹精衛士。最後一波團戰，斯拉克率先陣亡，但是經濟領先巨大的VG早已經不在乎這些，最終買活回來的小魚人帶領VG擊敗了Aliance，他們也是進入了最後一個比賽日的比賽。
