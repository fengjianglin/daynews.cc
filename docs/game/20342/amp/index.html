體驗服評測：野核並非純加強，越窮的打野越難玩，打野經驗變少

天天要聞
2019-12-16

大家好，我是老劉備。昨天大寶備在體驗服對打野進行實戰，今天給大家分享下體驗服野區和遠古生物改動的想法。

打野效率

體驗服野怪表面上面加錢，但是你別忽視野怪也加了生命值，這也就衍生出一個問題，你會發現你中期以後打野效率比以前慢了許多。

而且經濟越差的打野，就越難打得動野怪。所以說，下個版本對於業餘的打野會更難，尤其是那些藍領打野，一旦野區被反以後，你的經濟會越來越差，你想打自家野怪也會越來越慢！

大寶備以前就說過人頭賞金的經濟是虛的，體驗服新版本的打野還是要依靠反野偷野，來控制雙方打野的經濟差，野怪刷新時間變慢，就給你更多時間去偷資源。雖然新版本給你更多GANK時間，但是我還是奉勸大家團戰能打就打，不能打就趕緊找資源！

打野收益

大寶備在實戰中，也發現一個問題，就是打野經驗明顯變少。

第1把8級多做出黑切

第2把剛好9級做出黑切

註：兩把打野刀層數分別是13層和12層。

單只野怪並沒有減少、增加經驗，問題是野怪刷新間隔增加，導致你原來相同時間內，刷野數量變少了，也形成現在經驗變少！

個人體感經驗大概在出黑切時候差了半級左右吧！如果以正式服來說，正常情況劉備10級出黑切，就算是很凱瑞的局，也很難8級時候做出黑切！

總的來說，自家野區調整影響的總結的話：

順風打野更容易打出節奏

平風打野刷野效率變慢

逆風打野坐著等死吧


雖然中後期打野效率變慢，刷新率也變慢，但是如果能進行三角形打野的話，收益會變大，這也是為什麼我說順風打野更容易打出節奏，我家刷新慢了？沒事，我去拿你家的！

2分鐘遠古生物

暴君的經驗降低，大家必須注意，拿下暴君只會給你更多的錢，但是不能幫助你到4級！

同樣是2分鐘的遠古生物，先知主宰能讓你的團隊到4級，同時產生的主宰先鋒也能壓制線上英雄的經驗！

所以說，對於打野來說，沒有必要再去強調紅開還是藍開了，就看2分鐘的目標是什麼選擇開局。但是對於大寶備來說的話，先知主宰的誘惑力更大！

總體來說，我認為打野只是加強上限，下限那部分打野，體驗服對他們沒有加強，因為他們自己都掌控不了打野的節奏！但是，這種調整很可能會導致兩極分化更加嚴重，低段位沒人願意搶打野，高段位個個想要拿打野！
