暗黑手游製作人也難逃“真香”！莉莉絲新遊戲成為他最愛的手游？

天天要聞
2019-12-22

前陣子，《黑暗破壞神：不朽》的製作人Caleb Arseneaux以及首席設計師Wyatt Cheng在暴雪嘉年華中所透露的關於暗黑手游的消息引起了不少人的關注。作為最為經典的遊戲之一，暗黑系列在全世界都擁有著相當多的粉絲。

不過，當記者詢問這兩個人，暴雪開發者到底在玩哪些手游，有沒有受到別的手游影響時，製作人Caleb Arseneaux卻回答了一個讓人忍俊不禁的答案。他表示，自己在玩《AFK Arena》（劍與遠征），很好玩。隨後，他還表示這是一款中國手游，來自《刀塔傳奇》開發商。

但事實上，為什麼會說這個答案讓人忍俊不禁呢？因為這款手游的製作方就是我們國內的莉莉絲遊戲，沒有錯，正是那一個在幾年前被暴雪告過版權侵權，最後打官司和解賠錢的莉莉絲。沒想到的是，在時隔幾年之後，這一家被暴雪告過公司所開發出來的手游讓暴雪製作人都感受到了「真香」的意味。

那麼，這款名為《劍與遠征》的手游到底有什麼讓暴雪製作人都著迷的地方呢？

世界觀與畫風

《劍與遠征》的世界觀是一個原創且獨立的世界觀，從「混沌之初」生命之神與死亡之神擁有著不一樣的意見開始，雙方展開了交戰，而這個交戰自然而然也是影響到了整個世界的格局。而在這款手游中，也擁有著「耀光帝國」、「蠻族部落」、「綠裔聯盟」、「亡靈軍團」、「半神」以及「惡魔」這六個種族。不過值得一提的是，從國際服《劍與遠征》新出現的「虛空來客」來看，這個世界觀仍在不斷擴大。

從畫風上來看，《劍與遠征》捨棄了我們相當熟悉的唯美或是Q版畫風，而是獨特的西式玻璃彩繪，這種跨越文化壁壘，完美還原歐式教堂風格的精美畫風，正是這款遊戲征服絕大多數西方玩家的原因之一。

玩法

《劍與遠征》是一款放置類的手游，但是與普通「放置」不同的是，它不僅擁有著放置類手游都有的玩法，更是擁有著許多考驗陣容搭配以及策略的玩法。

比如在「時光之巔」、「奇境探險」以及「異界迷宮」中，不僅僅只是戰鬥力能夠解決問題。他需要玩家們根據陣容去選擇Buff以及對抗不同的對手需要使用什麼陣容去克制。而且，這種玩法也並不是固定的，周期性的出現讓玩家們也保持著一定的新鮮感。

英雄升級系統

在很多放置類手游中，英雄的選擇以及培養讓不少玩家們傷透了腦筋。但是在《劍與遠征》中卻有一個「共鳴水晶」的存在。而因為這個機制的存在，玩家們只需要培養五個能夠升到頂級的英雄（紫色英雄）。因為「共鳴水晶」中最低等級的水晶祭司決定槽位中英雄的等級。也就是說，你只需要培養5個英雄，接下來你所獲得的任何英雄都能夠在「共鳴水晶」中「免費」升級。

當然了，除了英雄升級系統之外，英雄的質量還是需要憑藉著相同陣營或者相同的卡片進行提升的。而等級的上限也是與英雄的質量有關。

所以，《劍與遠征》雖然是一款放置類手游，但是它大膽的畫風，新穎的玩法以及「不肝」的機制，成為了目前手游市場上面最好的選擇之一了。這也就是為什麼暴雪製作人會喊出「真香」的原因了。
