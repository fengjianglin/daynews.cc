DNF：12.14體驗服各副本改版，全新的盧克單人和困難模式要來了

2019-12-14 198條評論 1,720次閱讀 655人點贊

DNF：12.14更新，各副本組隊界面變更，盧克單人和困難模式來了！

以下為具體的內容詳情：

有關DNF1.14春節版本的相關內容，在今天12.14體驗服維護更新完成之後，策劃已經提前上線了其中的一小部分，包括像魔界大戰副本，角色的cp護石和符文系統，國服特色化職業改版等等，而除此之外還包含了像各種有關副本的改版，12.14體驗服更新之後，在遊戲中各副本組建攻堅對的界面變更，同時單針對盧克副本還進行了一波大型的改動，新增了單人和困難模式。

此次體驗服更新出的有關副本的改版並不是很多，但針對的卻是遊戲中的所有副本。其一就是對遊戲中各副本組建界面的改動，如上圖所示玩家們可以看到，以普雷副本為例這是在目前遊戲中改版前的組建界面的外觀，之後就是下面這改版之後的組建界面外觀。

改版前和改版後的兩者對比起來，策劃就是簡單的提升了一下組建界面的逼格而已，並沒有什麼太過於實質性的作用，在其中的引導（單人）就是目前的單人模式，普通則是目前的團本模式。之後其二就是策劃單針對盧克副本進行的一個大改動。

在12.14體驗服不光對盧克副本的組建界面進行了更新，如上圖所示玩家們可以看到，在盧克可組建的模式選擇中，除了目前盧克團本所對應的普通模式外，策劃還額外新增了另外兩種單人模式和困難模式。盧克的單人和困難模式？在這其中的單人很容易理解，策劃早就該上線了，但另一個困難模式怎麼說，策劃這是又再走之前漩渦副本的老路？

而至於這個盧克單人和困難模式，相較於普通模式有什麼區別，比如難度低的單人能低多少，難度高的困難能高多少，通關之後的獎勵有多大的區別，以及困難模式會不會加入一些額外的新獎勵，等等之類的問題，由於目前體驗服還未進行完全的實測，玩家們也只能慢慢等待了，反正目前只要提早知道一點，那就是等到1.14春節版本更新後，盧克不光會迎來單人模式，額外還有新的困難模式這些就可以了。
