12.21主宰路T度排行｜曜依舊是最噁心的英雄，下賽季誰會崛起？

天天要聞
2019-12-21

大家好，我是老劉備。新的一期T度排行開始更新了，這期T度會是本賽季的最後一期T度排行。想必大家本賽季也感受到了，主宰路的英雄毫無參與感，而作為主宰路可以使用的英雄很多，但是強勢的英雄卻是一隻手可以數過來！

T0.5/T1

T0.5和T1我只安排了6位英雄，其中2個還是賽事專屬英雄。其實，你會發現很多英雄因為同職業內戰的關係，都沒落了。比如說，這個賽季你看得到幾次邊路橘右京？看得到幾次邊路孫策？

曜的門檻低，導致使用他的召喚師很多，而這個英雄也確實噁心太多脆皮，因為星削的關係，讓射手單挑他的時候很難打得動他。

（體驗服已經削弱星削，但尚未實裝正式服）

這是頂端局曜的召喚師技能數據，很明顯懲戒佔據90%以上，大寶備不能保證懲戒曜就是邊路，但是邊路肯定佔據大部分！

馬超85%是走邊路，而15%是走打野位（大寶備不相信帶懲擊走邊路），而你要觀察仔細的話，你會發現打野馬超的評分、KDA、擊殺、助攻都會高於邊路馬超（死亡低於）。打野能夠有更高的參團率，使得它的評分會更高，也更容易拿到MVP，但是打野馬超終結比賽能力並不比邊路馬超強多少。

T2/T3

T2和T3是最多的英雄，也是上面大寶備說的，主宰路的英雄對遊戲的勝負決定性太弱了。體驗服更新後，前4分鐘主宰路遠程兵增加經驗，預估計可以領先其他線上1級左右，但是我覺得這不是主宰路英雄所需要的。主宰路的英雄需要的是參團，為什麼這個版本坦邊這麼弱勢？就是因為清線慢、支援慢，等你清完兵線要去支援的時候，兵線又來了。

豬八戒自從上次平調以後，大寶備認為這個英雄慢慢升溫，但是還那個問題，作為坦克邊路，如果拿不到經濟的話，他就脆得和紙一樣，程咬金同理！

有同學一直說暗信這個版本很強，大寶備認為暗信這個版本非常弱，這不是帶線的版本，而是團戰的版本。以前版本大家上高地時候要帶3路兵線過去才好上，而這個版本只要帶2路就行了，帶3路兵線反而容易被抓，並且浪費很多時間。

夢奇

這個版本如果你想要玩夢奇，那麼一定不要玩AD夢奇！電刀、無盡等裝備都經歷削弱，而AD夢奇的天敵一直在增加及增強（說的就是你呂布）。最克制AD夢奇的卻是兵線，原本5秒清一波兵線，現在要10秒時間，原本你可以多5秒時間去反對面野區。

對了，下個版本時光法杖還加強了！

AP夢奇才是真正的夢奇！
