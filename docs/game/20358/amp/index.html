爐石傳說：“酒館戰棋”中為啥沒有龍？這些卡牌未來都可能加入

天天要聞
2019-12-16

想必大家初玩「酒館戰棋」時都會有一些疑惑，比如下面這兩張牌。

隨機使一個友方野獸，龍和魚人獲得……等等，哪來的龍啊？

沒錯，「酒館戰棋」如今雖出現了與龍族有關的效果，但卻遲遲沒有龍，大家一直都用融合怪來代替龍種族。

那麼如果將現有的龍加入「酒館戰棋」，哪些牌能有一席之地呢？我們不妨來大膽猜測一下。

先看看新擴展包「巨龍降臨」的牌。有意思的是，雖然新版本以龍為主，但絕大多數的牌都不適合戰棋，小編找了半天也只挑出幾張還行的。

始生龍探險者和翡翠龍探險者

這兩張牌都有強大的發現效果，第一個更小但具有劇毒，第二個很大還是牆。不過龍給自己Buff並不容易，所以作為4星及以上的牌，應該不會像發現魚人那樣過分。

幼龍飼養員

雖然不是龍，但主要還是跟龍聯動。可以快速湊三連，如果能加入戰棋，至少是一張5星牌！

辟法巨龍

在戰棋中可以看作5/3聖盾，應該能成為一張3星牌。

「巨龍降臨」的牌就這些了，再看看其他版本的牌吧。

其實之前版本的龍能用的反而比新版多（廢話，那麼多個版本呢），有些效果放在戰棋也很有意思。

考達拉幼龍

無限使用技能是個很有趣的機制，像是尤格薩隆，大螺絲等都有了新的戰術。

唯一的問題就是奧秘哥，有了它基本能穩定找冰箱。如果奧秘哥退環境了，可以作為4星及以上的隨從加入戰棋。

貪睡巨龍

龍在前中期很難像其他種族那樣互相Buff，把它作為一張4星牌或許可以挽回一下前期的劣勢。

夜鱗龍後

4星牌。單看這個效果好像沒啥用，如果後續出了可以在戰鬥中回血的牌，就能發揮不小的作用。

光明之翼

傳說隨從中既有澤魯斯，老瞎眼這種渣渣，也有獵媽人這種大哥，作為5星隨從補資源或許不錯。

沼澤飛龍

這個2/1劇毒是在酒館裡召，玩家可以再花3金幣把那個劇毒買下來。小編覺得這是個不錯的創意，3星隨從比較合適。

深藍刃鱗龍人

比較優秀的過度隨從，但3星感覺有點強，4星又感覺有點弱，很難抉擇啊。

夢魘之龍

無限增加攻擊，後期基本等於劇毒，4星感覺比較合理。

總的來說，爐石中龍的特效往往都跟「手牌中有龍」有關，這跟戰棋的思路產生了衝突。鼓勵買了棋子不下的話是一件很怪的事情，這導致現在能加入戰棋的龍並不多。如果要推龍體系，還得出惡魔3/3那種特殊牌。

嘛，小編對遊戲理解的深度有限，大家看個樂就行了。各位看官若還有什麼獨特的思路，歡迎在評論區留言，說不定設計師就跟你撞上了呢？
