德杯八強齊聚重慶，WE再戰EDG，全新陣容誰能登頂？

2019-12-23 86條評論 1,176次閱讀 107人點贊

【看菌菌遊戲錄 召喚師峽谷不迷路】

英雄聯盟德瑪西亞杯小組賽已經結束，德瑪西亞杯設立在轉會期之後，也是各傢俱樂部重視的賽事之一，因為轉會期剛結束，戰隊也是有人員的變動，全新的陣容急需一場賽事來進行考驗和磨合，所以德杯也被稱為「試訓杯」，還被稱為「EDG杯」，因為EDG拿過六次德瑪西亞杯冠軍，這次德杯八強已經出爐，WE再戰老對手EDG，每支戰隊都是全新陣容，這次誰能登頂？

德杯這次共有20支戰隊參與進來，16支LPL戰隊和4支新人戰隊，分為4個小組，每個小組5支戰隊，A組：VG、WE晉級，B組：EDG、TES晉級，C組：JDG、RNG晉級，D組：RW、BLG晉級，根據賽制，A組第一打B組第二，C組第一打B組第二，所以WE和EDG再次相逢，究竟是WE能幫前輩「復仇」？還是EDG繼續展現本名杯的實力呢？

首先說說26號的賽程，第一場VG打TES，VG在本次冬季轉會期買來了iboy和蕉太狼，VG絕代雙蕉就組成了，而iboy曾經作為LPL的頂尖下路，面對新人自然是沒有問題的，而蕉太狼是新次級聯賽皇帝，還上LPL打過幾場得到磨練，回到LDL把其他隊伍殺了個遍，但是這次他們的對手是TES，TES這次德杯沒有練新人，而是上了一陣的隊員，Knight+Karsa+369十分強勢，這個版本還是上中野的版本，所以個人覺得TES贏面會大一些，再加上主教練Kkoma沒有上場。

而第二場EDG打WE，EDG這次冬季轉會期做得最錯誤的一件事就是留Hope放iboy，聽聞在隊員考核中，iboy的表現沒有Hope好，所以也是半賣半送地交易給VG，如今VG的表現來看，確實是撿到寶了，擊敗了勢頭正猛的WE，而Hope的表現中規中矩，EDG也撿到寶了，上單50萬買的小祥也是一位個人實力十分強勁的上單，而中單小學弟依舊很頂，EDG的老闆挖掘新人真的有一手。而WE戰隊，今年大家都以為家倒了，殊不知是拆遷了蓋了棟別墅，WE3.0時代、下一個五年計劃宣布開始，兩隊新人較多，打起來還真的不好說，不過我覺得廠子打WE很有一套，EDG勝算會大一些。

個人覺得TES和EDG會師半決賽的可能性會比較大，畢竟VG和WE基本等於重組，還需要一段時間的磨合，在之後的比賽進步空間較大，TES在Karsa加入之後，也成為了一陣隊伍，EDG雖然這個賽季陣容大家覺得不太滿意，但是上單小祥在德杯的發揮讓大家心更穩了一些，何況教練還是那個男人，四支隊伍都加油吧！
