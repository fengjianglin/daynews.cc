德杯兩大野王迎LPL首秀！小花生戰Karsa世紀對決，勝率引網友熱議

2019-12-22 1,118條評論 4,126次閱讀 2826人點贊

北京時間12月22日，英雄聯盟德瑪西亞杯將進行第二個比賽日的賽程，許多LPL賽區耳熟能詳的戰隊將登場決出出線名額，而LDL新軍也蓄勢待發，欲在這個舞台上證明自己的實力，或是借這樣一個平台展現自己，以期在未來獲得加入LPL頂級聯賽的機會。

​對於觀眾來說，德杯第二日賽程里最受期待的一場比賽，莫過於TES對陣LGD的世紀大戰了。兩支戰隊在今年轉會期都是有大動作，在打野位置上引入強援。TES轉會簽下前RNG打野Karsa，而LGD遠赴韓國挖來了LCK頂尖打野Peanut，這場比賽將是兩大野王的巔峰對決。

​兩隊除了打野位置上的交鋒成為比賽取勝的關鍵外，LGD的戰術策略更傾向於下路Kramer，而TES則是主打上中野，靠著中路Knight的個人能力打開突破口。在獲得了Karsa這樣頂尖打野的幫助後，TES的前期中野節奏更加恐怖。

​讓人意外的是，專業電競平台給出的獲勝指數兩隊差距極為巨大。TES的勝率指數為1.32，而LGD則是3.34，這樣的對比下顯然勝率方面TES更受到看好，這讓普通觀眾十分不解。畢竟兩隊從紙面實力上來看在伯仲之間，唯一的解釋或許是Peanut此前的狀態太過低迷吧。

​在22日上午11點，將進行的是IG與SDX的對決。兩支戰隊對於LPL觀眾都很陌生，因為SDX是來自LDL發展聯賽的隊伍，而IG這次德杯是派出了替補陣容，因此這場比賽對於觀眾而言很難判斷勝負。

​這時就需要觀察第三方平台給出的評估了，在專業電競平台中，全替補陣容出戰的IG獲勝指數為1.37，而SDX則是高達3.06。顯然IG作為LPL老牌豪門，哪怕是全替補陣容，也依舊令人忌憚、畏懼。
