游漫談：《寶可夢劍盾》謎擬Q被評S級，種族值中庸的它靠特性翻身

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

寶可夢系列第八世代續作《寶可夢：劍盾》已經發行了快1個月，在這段時間裡大多數玩家們已經適應了新作帶來的各種改變。

寶可夢系列第八世代續作《寶可夢：劍盾》已經發行了快1個月，在這段時間裡大多數玩家們已經適應了新作帶來的各種改變。因為大量的前作寶可夢和技能被刪減，所以聯機玩家在隊伍編排和戰術上做出了不小的改變，排位賽的寶可夢排行榜名單也有了很大變化，比如在覺醒冰、覺醒火這類取消後《劍/盾》草鋼、龍系成為了大熱門。不過值得一提的是，在總評榜上大部分S級寶可夢依舊是舊作精靈，其中「謎擬Q」再次登上「物理攻擊型總榜」的榜首，成為與G8准神多龍巴魯托、暴鯉龍、伽勒爾達摩狒狒並列的S級寶可夢。

看著謎擬Q登上榜首，不禁有玩家就會問了「為什麼種族值平庸的它竟然能這麼熱門？」。誠然，謎擬Q作為第七世代《精靈寶可夢：太陽/月亮》首次登場的寶可夢，單從它的種族值上看完全是一隻中等偏弱的寶可夢：沒有進化形態，種族值總和只有476，除了特防剛剛破百（105）之外其他全部低於100，物攻也才90而已。但是無論第七時代還是第八時代，謎擬Q都是聯機對戰的大熱門，長期霸佔著TGL單打使用率第一的寶座不說，在雙打中也常見它的身影。究其原因，只因為它具有整個圖鑑890隻精靈中最強大的特性之一：畫皮。

畫皮特性是讓謎擬Q評為S級的根本原因，這個特性可以讓謎擬Q免疫1次任何攻擊招式的傷害（包括Z招式等混合攻擊），受到攻擊後謎擬Q損失八分之一的最大HP，並且該特性消失謎擬Q變為畫皮脫落後的樣子。說得再簡單一些就是「畫皮特性讓謎擬Q獲得一次無敵能力」。

在對戰中「無敵」可是相當於bug般的存在，但凡挨上點邊的寶可夢都會成為大熱門，比如只有1滴血卻可以無視大部分技能傷害的脫殼忍者。然而脫殼忍者有著各種致命弱點（撒菱等等）終歸不能完美戰場，但謎擬Q不一樣，它的畫皮特性可是做到了真正的無視傷害。這意味著謎擬Q的出場是絕對安全的，圍繞著這個角度玩家們就研究出了非常適合謎擬Q的物理輸出打法「劍舞三攻」。

劍舞是寶可夢中一個大幅度提升自身物理攻擊的強力BUFF技能，它令使用者的攻擊直接提升2級。劍舞提升物攻的方式是第一次劍舞物攻為基礎二倍，第二次變為基礎三倍，第三變為基礎四倍並且封頂。所以當謎擬Q擁有完美站場能力後劍舞至少可以使用1次，物攻翻倍後的輸出可就不是簡單的面板傷害了。

另外當謎擬Q攜帶氣息腰帶（鎖血1HP）時將大概率擁有3條命，敢操作的玩家完全可以賭二次劍舞，此時謎擬Q的物攻已經超過大部分寶可夢，輸出能力非常驚人。當然攜帶生命寶珠也是一種套路（招式威力上升30%）。謎擬Q的常見配招是「劍舞（核心）/暗影爪/嬉鬧/替身」，其中替身技能讓謎擬Q有了更多容錯率，也可以將其換為其他輸出招式。
