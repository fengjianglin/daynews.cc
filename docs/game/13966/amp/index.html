FGO國服2018年聖誕復刻活動一句話攻略

天天要聞
2019-12-12

大家好我是風叔，萬聖活動結束，大家經歷了一周的長草和攢石，然後就到了存蘋果活動，聖誕復刻了。

因為是復刻，沒有無限池，加上各位已經卯足精神等後面的正式聖誕活動了，這裡估計也不會太下功夫。但也請一定記得，之前打過聖誕活動的各位御主不要忘了，從這個活動開始，復刻的活動從者是可以賣金方塊的，要知道金方塊本身也是很難獲得的東西，別的獎勵不重要，這5個金方可以一定不要放過啊！

關卡方面，聖誕活動沒有傷害倍卡，不過並不影響FREE關卡3T爆破，這裡就不詳述了，反正FREE關卡也不多，跟著每天主線順序開放的。遇事不決狂蘭加特林就是了，怪不夠3記得穿個20充能御主衣服就行。最後的聖誕之塵關卡狂蘭換成塔喵就更簡單了。

就沒有我打不了的FREE（不要在意細節）

主線看上去血很多，其實大部分關卡都只需要等回合即可結束，畢竟艾蕾小天使才不捨得和咕噠子動真格呢~當然，除了艾蕾的關卡以外，其他幾關可能稍微會有點難度。這裡細數一下：

主線第三節，1面簡單，2面一個開局滿充的小安。然而這根本不是啥問題，因為是開局滿氣但不是開局寶具。。。用任何一個消氣技能就能讓他變成靶子，或者你直接用BB兩T摁死就可以了。

第五節，弁慶+狂牛肉丸，打過7章牛肉丸的都知道那個分身技能吧，這裡算是個弱化版，分一個不會用寶具的狂分身，如果先打死弁慶，牛肉丸會立刻再複製一個，複製體被優先擊殺也一樣，所以很簡單，直接起手轟死本體，剩下的隨便打了。

第七節 弓凜，第一條血因為有BUFF存在，紅卡打不動而且本身還特別的硬…帶人，時代變了。。。現在是綠卡時代。所以，整個槍奶光或者直接上斯卡哈一槍捅爆了。第一條血沒了第二條血直接變軟腳蝦，隨便欺負吧。

第八節第三戰，這是主線最終BOSS，技能14病魔會給我方單體丟一大堆DEBUFF，看著就覺得噁心，但這個可以被弱化無效直接抵消…另外炸血條會燒我方全體3000血，3條血固定燒2次，6000HP。然後技能一回合後我方眩暈稍微有點煩人，但這裡就可以用上之前活動給的刻印白杯了，所以整體並不複雜。

額外話，這次活動，弓大王全程有額外100%傷害加成，這是活動唯一的倍卡輸出了，算是復刻從者專屬的福利吧。

然後再說一說高難了…算了，這個就是一低難罷了。如果去年打過的御主都知道，這本之所以只能算低難，是因為一個設定，我們來詳細說一下：

配置：三血條黑槍呆+2血條梅林。開局梅林釋放技能，全體暴擊提升。BREAK之後會給黑槍丟個超級英雄作成。

黑槍呆常時普通技能，血條擊破後增加自身充能1-2格（根據血條先後順序），然後給予我方咒厄效果，提高她寶具的詛咒傷害，兩個都加上，寶具的詛咒一跳4000，抗是扛不住的。而梅林在場會經常給黑槍加充能，這使得黑槍頻繁使用寶具，導致我方快速減員，而且寶具以外，高概率的暴擊傷害也是很難抵抗的了的。但是如果你優先擊殺梅林，則會讓黑槍變成超人，更增加了通關難度。

然而……

這裡就說一下這個把高難變低難的設定吧。梅林在血條BREAK釋放的超級英雄作成…是可以用強化無效給頂掉的，僅僅要求的是在操作時，先給黑槍釋放強化無效，然後當回合擊破梅林第一條血，這樣他就會優先使用BREAK技能，然後就…3個NO EFFECT。由此一來，黑槍戰鬥力將直線下降，接下來繼續悶死梅林，黑槍就是個木樁了。

當然，說的簡單，操作起來還是有些運氣成分的，首先黑槍對魔力較高，很可能強化無效技能本身會MISS導致戰術失敗，而技能又不能SL，加上沒計算好導致這回合沒有擊破梅林血條也會浪費掉強化無效。基本出現這些個情況只能重來，但因為是高難，消耗AP很少，重來就重來吧。問題不大，只要成功一次，這關就輕鬆的1P.

然後活動獎勵，也就是一些不算常用的材料了，沒有指令紋章等必須兌換的道具。而唯一有價值的就是大王換金方塊了，而這裡只需要刷池即可，復刻只有8池，各位也不用太肝，休閑就好，畢竟真正要肝的還在後面呢。

我是新人編輯風叔，一句話攻略…好像有點長啊，大家來吐槽一下？
