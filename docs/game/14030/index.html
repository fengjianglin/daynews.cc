DOTA2：玩家發現新bug，黑鳥關人導致遊戲崩潰，這下終於清凈了？

2019-12-11 1,698條評論 3,266次閱讀 1480人點贊

大家好，這裡是包雞包眼包開心的A電競飛機。

DOTA2遊戲更新之後，玩家們對新版本的評價也是相當之高，不論是新英雄還是野區改變都令玩家十分驚喜。

不過V社的員工終究沒有讓玩家們失望，新版本的bug還是來了。

這個bug跟黑鳥有關係，有玩家發現只要在遊戲中其他人使用黑鳥的2技能遊戲就會崩潰，導致強制退出。

大家都知道DOTA2遊戲裡面有蓮花可以反彈黑鳥的關，還有敵法師以及拉比克等英雄可以使用或反彈黑鳥的技能。

而出現這種bug顯然會對玩家造成不好的影響，破壞玩家的遊戲體驗。

關於bug的具體原因目前還不得而知，而V社官方也沒有給出解釋，為了不影響遊戲正常進行，很多人都會避免選擇黑鳥。

講道理，遊戲出現bug其實很常見，特別是像DOTA2這種遊戲新版本剛開，肯定會有不少bug出現。

之前玩家們相當熟悉的水人bug就是如此，明明就存在這個bug情況，而且玩家們也在平日比賽裡面反映過，可V社就是置之不理。

最後這個bug更是直接出現了TI國際邀請賽的舞台上，實在有些不應該。

不過那時候V社倒是會找借口，因為水人的英雄機制相當複雜，很多問題不好解決，所以便沒有著手改動。

當然跟水人比起來，黑鳥的問題卻要嚴重不少，畢竟水人即使出現bug情況也只是影響他一個人，而黑鳥則會影響十個人的遊戲體驗。

對玩家而言，黑鳥的情況顯然要嚴重不少，當然不少人也在調侃，既然黑鳥出現了bug情況，那麼天梯可算清凈了不少。

玩家都清楚，新版本黑鳥改變不少，他的傷害得到不小的提升。

這也讓黑鳥成為了玩家們最愛的英雄，在天梯裡面叱吒風雲，如今bug消息一傳開來，選擇黑鳥的玩家自然就少了。

好在V社及時修復了bug，才沒有讓事情擴大化，看來V社還是時刻想著玩家的。
