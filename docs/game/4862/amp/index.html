王者榮耀最兩極分化的英雄，為什麼姜子牙在職業比賽上那麼強勢？

天天要聞
2019-12-04

摘要

大家好，我是掌游寶作者果果game。在KPL裡面有一個ban率極高的英雄，你說他是中單吧，主定位是輔助，你說他是輔助吧，他出場基本打的就是中路。

大家好，我是掌游寶作者果果game。

在KPL裡面有一個ban率極高的英雄，你說他是中單吧，主定位是輔助，你說他是輔助吧，他出場基本打的就是中路。

無論是哪兩支隊伍撞上，如何處理這個英雄，要不要放出來都是雙方非常頭疼的時期，甚至這個賽季的巔峰對決裡面這個英雄保持了百分百的出場率，他就是姜子牙。很多剛看KPL的觀眾其實看到姜子牙都不會理解，為什麼這個英雄在職業賽場上那麼強勢，其實這和版本有關。

姜子牙在職業比賽上強勢可以說有非常長的歷史了，不過統治KPL還是今年的事情。在這一屆KPL秋季賽，姜子牙數據爆表。常規賽姜子牙的出產率是9.5%，禁用率是88.7%，勝率71.1%；季後賽姜子牙的出場率是15.4%，禁用率是82.7%，勝率75%。

即使是大家普遍認為強勢的盤古，在數據上也大大落後於姜子牙。姜子牙的常規賽數據表明，姜子牙非ban必選的幾率是98.2%，季後賽的ban選率則為98.1%，換句話說，在職業賽場上，姜子牙不能放，放了就得拿，拿了就能贏。

姜子牙之所以這麼強勢，完全是依靠著他的被動-心魔。這個被動的作用是能夠給全隊增加經驗，這個經驗體現在兩個地方，一是正常發育的情況下，自己家全員到二級的速度比對方會快，二是全員四級的時候對方四級的人數一般就一兩個。

這個等級差放到職業比賽上是非常可怕的，極端點說，如果自己家全員二級而對面全員一級，那麼一波團戰自己能比對面多五個技能，而四級的時候，自己家能多五個大招（雖然實際上不會那麼極端）。這意味著二級和四級的團戰自己家獲勝的可能性非常大。

這是姜子牙的特點，在最近的幾代版本中這成了姜子牙的最大優點。因為最近的幾個版本都是重打架重入侵重資源搶奪的版本，一旦自己家取得優勢，滾雪球能力將會非常強，對面將難以翻盤。

由於姜子牙的技能機制，一旦姜子牙一方取得順風，姜子牙的破甲、控制、封走位逼塔的大招以及是不是超高額傷害的大閃會讓劣勢方極其難以防守。恰好，最近的幾個版本又都是以射手為主要核心，逆風射手的傷害有多低就不用多說了。

更重要的是，職業比賽不比路人，普通的路人局是不在意體系的，基本拿的都是對應位置自己比較擅長的英雄。即使是段位比較高的局，基本也只會選擇一些版本強勢T0英雄，而不考慮英雄之間的配合。但是職業比賽不一樣，他們每一局拿的陣容除了一定程度上能限制對面外同時會考慮自己家陣容的配合，因此職業的姜子牙被稱為姜子牙體系。

一般來說職業的姜子牙拿出來是打中路的，而其他位置也基本會選擇二級和四級質變的前期強勢英雄，比如狂鐵、豬八戒等。這就導致本就強勢的姜子牙的前期在隊伍中被進一步放大，這也是姜子牙體系那麼高勝率的原因。不妨想像一下姜子牙陣容配合狂鐵邊蘇烈輔助盤古打野，前期直接往對面野區鑽，這是多麼有壓迫力的場面。

雖然姜子牙體系很強，但是他也有兩個弱點：

一是姜子牙一旦死亡，在他死亡期間他的被動是不生效的。

二是姜子牙的後期傷害極度依賴大閃，單單一二技能的傷害在後期可以忽略。因此打姜子牙體系要麼選蘭陵王之類的英雄在前期殺一次姜子牙，要麼選一些清兵快的英雄拖到後期，職業上隊姜子牙的處理往往是雙管齊下。

不過即使如此姜子牙也保持著七十多的勝率，可以看出目前的版本的前期重要性是遠遠超過一兩年前的版本的，一個小失誤很可能導致了很長一段時間的逆風。從另一個角度來看，姜子牙的勝率和bp率越高，越說明現在的職業戰隊越成熟。

因為他們能拿到姜子牙，然後打出優勢，然後能很穩定地終結比賽，防止對方翻盤。拿到優勢後能滾起優勢，這才是一個成熟的職業戰隊應有的能力。

但是目前職業聯賽的ban位太少了，導致姜子牙成了bp時的毒瘤，誰都不敢放，放出來就是個未知的炸彈，這就導致了紅方bp陷入很大的被動，因為對面知道你有一個ban位必定會給姜子牙，從bp的角度來說，姜子牙的存在其實導致了很大的失衡。

說到底姜子牙的容錯相比之前的版本有點過高了，只要拿到姜子牙幾乎穩定順風，同時能反制姜子牙的英雄太少。如果不對姜子牙做出一定的調整，就需要對bp規則以及其他許多英雄進行調整，這是一件吃力不討好的事情。所以在這一屆秋季賽，姜子牙的毒瘤身份將會一直存在。
