王者榮耀正式服在今天上午九點進行了一波更新，本次更新的主要內容有：五位英雄調整、11款皮膚八折上架、完成任務兌換榮耀播報、聖誕星元下周上架、碎片商城更新等內容。王者模擬戰調整內容在今天第二篇推文專題解析。另外還有娜可露露新皮膚正式官宣，一起來看看今天的內容吧！

12.17正式服更新

一 英雄調整

▶魯班大師加強

怪客解析：基礎生命值加強，同時三技能射程加1000，這一改動會讓魯班大師的使用手感提升不少。另外防斷時間變成了1秒，這是對射程增長進行的平衡調整。總的來說魯班大師進行了一次全面加強，這也是因為他在中低段排位賽中的表現實在太差，很多玩法還有待玩家開發。

▶安琪拉加強

怪客解析：安琪拉的大招護盾變厚，三技能旋轉速度的調整沒有上線正式服該感到慶幸嗎？

▶姜子牙平調

怪客解析：經驗提升間隔增長，大招傷害增高。這次改動使得姜子牙在遊戲開始後的兩分鐘內幾乎毫無作用，但是兩分鐘之後，姜子牙的優勢將會體現出來。最近我在遊戲中也遇到姜子牙並且被制裁了，被制裁的主要原因就是對面姜子牙的大招放的真的准，在開團前我方都被打殘。所以這次調整，雖然被動經驗的增加方式發生了變動，但是可以頻繁的刷大招才是姜子牙最噁心的地方。

▶馬超削弱

怪客解析：被動移速降低，等於變相的傷害降低。路人局遇到比較少，對於大多數路人局玩家來說無事發生。

▶盤古削弱

怪客解析：三技能的基礎傷害、等級成長傷害以及Ad加成傷害都被削弱了，盤古在當前版本還是挺吃香的，但是盤古對於路人局來說是個冷門英雄，所以這次的改動並不會掀起波瀾。

二 11款皮膚八折上架

更新後至23日，也就是一周時間內，所有2019KPL秋季賽總冠軍精選英雄直降100點券，降價英雄有趙雲、曜、張良、孫尚香和蔡文姬。

冠軍戰隊精選英雄的所有商城直售皮膚享八折優惠。降價皮膚包含了趙雲白執事、忍·炎影、未來紀元、曜歸虛夢演、孫尚香末日機甲、沉穩之力、水果甜心、張良黃金白羊座、一千零一夜、蔡文姬薔薇王座、舞動綠茵，合計11款皮膚。

這些英雄分別是：嬴政、白起、曜、魯班七號、莊周

娜可露露新皮膚

王者榮耀中有三位SNK英雄，由於版權問題連伴生皮膚都沒有，如今官方化不可能為可能，正式官宣已經搞定SNK的版權問題，將會陸續推出SNK的新皮膚，第一款就是娜可露露的。

這款皮膚取名晚螢，定位史詩，直售獲取，價格應該是888點券。還好不是叫「娜可媽耶」
