德杯八強出爐 各戰隊新陣容實力分析

天天要聞
2019-12-24

這個周末，2019冬季德杯如期舉辦。我們在周五為大家推送了《德杯觀賽指南》一文，在兩天的賽程後，我們預言的S級隊伍，全部打出了不錯的效果（OMG和LNG比較遺憾，輸給了爆發後的RNG）。這裡小龍還是要小驕傲一下的。

這裡，我們再次將鏡頭聚焦在晉級的隊伍，看看他們在本次小組賽的發揮。

全勝之王：JDG

本次JDG以4-0全勝之姿出線成功，在本次德杯的賽程中，他們展現出比之前更強的前中期作戰能力。如果說之前的JDG只能苟住打後期，那麼現在的JDG在前期，就具備了一定的作戰能力。而Kanavi在經過了這段時間的磨合後，展現了自己被多方看重的價值。在本屆德杯，JDG展現出極高的競爭力。

VG：銀河戰艦

因為首場比賽發揮不佳，VG被調侃為「最快斷腿的銀河戰艦」。這支由多支隊伍潛力選手組成的隊伍，在轉會期被給予了極大的期待。在主教練Kkoma未加入的情況下，他們就以3-1的成績成功出線。對於一個以「成長」為主題的陣容而言，提前拿到這樣的成績，顯然是值得鼓勵的。但對於VG，我們始終期待Kkoma全面介入後，幾位潛力選手的成長幅度。那樣的VG才是更加值得我們期待的。

WE：純新人帶來意外

本次德杯，WE並沒有帶來他們的韓式雙C陣容，但他們依舊以3-1成功出現。

中路選手TeacherMa無疑是這種欣喜的製造者。從呼籲解散，到搖獎成功，WE粉絲經歷了過山車式的心境變化。

RNG：跌跌撞撞

RNG在本次前期大規模採用了無AD陣容，但恢復了傳統AD的使用後，整個隊伍反而開始贏了下來。目前來看，RNG確實沒有直接跌落過多，XLB雖說沒有給人感覺強的離譜，但整體期望值也在粉絲預期之上。總的來說，這支缺乏小狗和Karsa的RNG並沒有徹底跌出強隊行列。

TES：中野強勁

Karsa與Knight的組合發揮了足夠強勢的效果。

但369的骰子屬性還是不太穩定，在與EDG的比賽中，369被EDG新人上單XiaoXiang碾壓。

如果不是輸給EDG，那麼TES完全是能和JDG抗衡的不敗戰隊。

EDG：中獎達人

作為本次轉會期公認的「輸家」，籌備了巨額引援計劃的EDG並沒有如願。但在德杯上，EDG招來的50萬新人上單Xiaoxiang，展現了極為Carry的一面。

同時，Xinyi作為老挖掘機，也成功繼承了廠長偏為運營的玩法。儘管還有上頭的情況，但EDG之前熟悉的味道回來了。

RW：陣容帶來改變

本次RW在轉會期同樣有許多動作，他們也是進行大換血後短期內的受益者。新的RW一改之前後期迷惘的問題，在中後期處理中老練不少。對於RW這樣一個仰賴AD的隊伍而言，中後期表現的改觀，無疑能讓他們贏下更多本該贏下的比賽。

BLG：FoFo不穩定

BLG本次採用了相對穩定的陣容，但實際效果卻弱於他們本身的陣容。特別是FoFo，儘管他看上去沒有大的問題，但BLG原先Kuro式的中後期推進直接消失，BLG在中期顯得相當被動。
