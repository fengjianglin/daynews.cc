航海王燃燒意志：劇情通關近乎必備的角色，盡量入手，可舒服很多

2019-12-08 0條評論 16次閱讀 0人點贊

摘要

諸君安好，霧夏菌報道。

在主線劇情中，有一些角色作用比較突出甚至說必備，這種時候就需要盡量入手他們了，那麼這一回合就來看一下有哪些角色。

1、佳妮法

SR角色，主線通關的必備角色之一。全體解控+全體百分比回血，主線劇情非常好用。

佳妮法最大的作用就是提供全體凈化，在中後期，諸多主線小怪與BOSS有著各類控制，這個時候佳妮法的凈化能夠最大程度的解控。同時佳妮法的進化還帶有百分比最大生命值的回血效果，保證我們的續航。

2、白鬍子

白鬍子也屬於主線必備的角色之一，有沒有白鬍子，主線通關的壓力是天壤之別。

目前唯一可眩暈頂上戰爭之前絕大部分BOSS的角色。當初攔住無數玩家的【署長麥哲倫】，這位超級兇悍的主線BOSS，白鬍子都能夠控制。而且該控制時長達到了2回合，我方可以無壓力輸出。再一個頂上戰爭中，白鬍子的中斷機制極為重要，否則極難通過後續關卡。

3、馬爾高

當前主線劇情最合適的輸出位。

很多時候我們刷主線劇情，都會面臨一個尷尬的問題——輸出量不足。而馬爾高則直接解決了這一問題，其變身後的輸出量特別爆炸，罰站被打出來後，傷害量再次提升，這一階段的傷害量可以一回合輕鬆幾十萬，真正爆炸輸出，瞬秒主線BOSS。

以上三位是刷主線特別推薦的角色，哪怕只有他們三人，也可以清掃至少80%的主線劇情。那麼剩下的20%則需要一些比較特殊的角色。

1、羅&花劍

這倆選其一即可，個人更偏向羅。

就目前的主線劇情中，羅的作用是大於花劍的，雖然在海底監獄的四聖獸關卡，只能剋制1人復活，這一點比不上花劍的剋制4人，但勝在羅有著一手奧義的20%目標最大生命值傷害，且對頂上戰爭的青雉也有效。

2、紅髮&新索隆

這兩位都是攻守平衡的角色，雖然之前本菌提過，所謂的攻守平衡直白點就是啥都會一點但啥都不精通。但後續出的新卡開始打臉了，既讓這名角色非常能抗，也要讓他非常能打，沒什麼事情是一串代碼解決不了的。

不過這兩人相比，更推薦紅髮，機制更完善，每回合固定8魂玉，且有更穩定的保命機制，以及無法被中斷。這個在頂上戰爭中打青雉同樣非常重要。

菌說

以上提到的這些角色，在主線劇情中的表現更強悍，或者有著自己的特殊作用，替代品少，近乎必備。主刷劇情的玩家，比較推薦入手。而且現在白鬍子、新索隆都可以兼職一個前排的作用，所以沒必要再去苛求一名純肉了，當然有的話也很好。

好了，這一回合到此結束，感謝閱讀，拜拜了您嘞~
