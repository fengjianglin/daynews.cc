三國殺：小霸王孫策的前世今生，網友心中真正的強者

2019-12-17 890條評論 2,068次閱讀 527人點贊

孫策覺醒後作為一個4血5技能武將來說無疑是十分強大的，但是孫策和鄧艾鍾會這兩個難兄難弟一樣都是經常死在希望的田野上的武將，更是有著江東小白板54320的男人，因此也被廣大玩家戲稱為孫笨，江東小白板則是因為在沒覺醒的時候孫笨可以說是只有激昂這個一個技能，但是紅殺和決鬥的數量嘛大家都是有目共睹的，只能說江東小白板名不虛傳。這樣一個有著極高人氣的武將自然是有著各種各樣的改編版本的，今天我們就一起來看看吧！

TOP1:

這個玩家對於孫笨的DIY可以說是十分好玩了，所有的技能全部變成了技能的文字描述可真是太優秀了，想想一下這張卡面殺的場景，一個彪形大漢擺出了一個銷魂的姿勢激昂的說出殺！這個畫面感頓時就出來了，最後的技能是是和諸葛亮有了聯動，這真是太強了！(諸葛亮：關我啥事？）

TOP2：

這個玩家的DIY更是把孫笨設定成了一個四血13技能的武將，不過這麼多負面技能真的能玩嗎？崩壞仇海就不說什麼了，也算是勉強過得去，畢竟可以加快覺醒進度，但是持勇和耀武出現在同一個武將身上就有點過分了，華雄乘以二嗎?更狠的還在後面，鏈禍+燃殤+同疾這是要承包全場的火殺嗎？想想都可怕，雖然給了兩個正面技能但是這兩個技能確定不說為了更好的觸發同疾？1上限才能覺醒的魂姿英魂怎麼用。但是有一說一雖然這個玩家在黑孫笨這件事上是認真的，但是其自身對於遊戲的理解不可謂不深，每個負面技能之間幾乎都有著聯動效果，也是個老玩家了。

TOP3:

這個DIY可以說是真的把孫策的白班特性表達的淋漓盡致，雖說有著5個技能，但是仔細看看，給孫權加2血，在孫休範圍內，是孫皓的受傷角色還有袁術的偽帝，總結成一句話：技能挺多，但說的全是廢話，這不都是別人的鎖定技么？唯一有用的制霸還是經常被笨黑們嘲諷的每輪吳勢力角色可以對你使用一張不可被無懈可擊響應的過河拆橋，這個DIY武將可以說把孫笨的最大特性完美的發揮了出來，我願稱之為最強白板！

TOP4：

這個簡單粗暴的DIY讓我笑了有足足五分鐘，這恐怕是對孫笨技能最好的詮釋了，兩個沒什麼用的技能加上主公選項可以說是簡單暴力。想體驗白板打軍爭嗎？想體驗地獄級難度嗎？想的話請動動你可愛的小手選擇孫笨，絕對讓你有著獨特的遊戲體驗。這個小哥哥真的是太可愛了，這兩個技能說明能夠足以讓這張圖片成為我的快樂源泉了。

TOP5:

終於有一個DIY是不黑孫笨的了，不過賀笨是什麼鬼？笨笨聯合嗎？

這個覺醒技不可謂不強，激昂、馬術、英姿、英魂、閃襲、奮威、制霸、短兵，可以說只要能夠覺醒就是吊打全場的存在，吊打關索不是夢，但是前置條件就很有意思了，一血四裝備，花色還不能一樣，覺醒？不存在的，這輩子都不存在的，老老實實當個白板混日子吧，乍一看十分強力但是細想之下毫無亂用，這幫人可真的是太有才了。

好啦，看了這麼多孫笨的DIY你有沒有開心的笑出聲呢？不得不說孫笨為我們提供了很多的歡聲笑語，雖然大家都在黑孫笨但是他還是什麼強力的，軍爭場主公孫笨的勝率有著高到嚇人的百分之78！
