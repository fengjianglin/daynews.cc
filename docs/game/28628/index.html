高振寧在吐槽大會：LOL冠軍想和圍棋冠軍柯潔一樣，保送上清華

2019-12-22 103條評論 1,033次閱讀 63人點贊

愛打遊戲的人可以上清華大學嗎？IG-Ning高振寧在吐槽大會上表示：柯潔是圍棋冠軍，已經保送上清華大學了。而我曾獲得英雄聯盟總決賽冠軍……，我想上清華大學……。大家覺得玩遊戲的冠軍可以保送上清華嗎？看著高振寧非常誠懇，非常認真的推薦自己，旁聽席和觀眾席聽了高振寧誠懇的自我介紹，都在歡呼，校長，您就收了他吧。

圍棋蘊含著中華文化的豐富內涵，它是中國文化與文明的體現，圍棋至今已有4000多年的歷史。柯潔就是天賜給中國圍棋的領軍人物，他兼具了聶衛平的霸氣格局和馬曉春的聰明細膩，柯潔在多個圍棋界的聯賽乃至世界級比賽中獲得多個冠軍，有興趣的朋友可以自行搜索柯潔的事迹，這麼優秀的人才保送上清華大學大家都認可。那麼現在的問題就是：英雄聯盟LOL冠軍保送上清華大學是否可行？

電子競技和圍棋相比，電競更加具有商業價值：

大家都知道，現在是一個遊戲產業興起的時代，可以這麼說：遊戲誰人不知，誰人不曉。打遊戲的意義不僅僅在於娛樂，它還蘊含著巨大的商業價值，有商業價值的東西就有存在的價值，儘管打遊戲仍然和不務正業、不思進取緊緊捆綁在一起，但是現在的職業電競選手的的確確是在用打遊戲掙錢，高端職業電競選手甚至能養活一大家子，這比普通工人強太多。因此類似高振寧這樣的高端職業選手自然就成為了年輕人的偶像，他給予了年輕人很大的鼓舞，年紀輕輕就可以事業有成，就可以財務自由，可以擁抱自己所愛的事業，這正是年輕人想要的。為什麼要像老一輩工人一樣，辛辛苦苦幹到四、五十歲才混個高級工程師，微薄的工資不夠養活一家子，年輕人他們可不想走父輩的老路。

因此高端職業電競選手的價值就在於給年輕人鼓勵和希望，成為年輕人的偶像，從而讓職業電競選手的商業價值得以實現。有一點必須承認，儘管圍棋更加高大上，但是電子競技明顯比圍棋更加具有商業價值，圍棋是極其高深的運動，沒有受過專業訓練的人壓根就看不懂，為什麼高手過招總是東邊放一顆子，西邊又放一顆字，看上去像是均勻的把棋子放在棋盤上，其實這就叫做圍棋的「布局」。由於圍棋的高深，導致了圍棋在國內非常冷門，太冷了，商業價值較低。

和圍棋的難懂相比，LOL則很容易看懂，兩軍交戰，衝上去就是干，滑鼠點擊誰不會？於是LOL很容易成為全民運動。另外，LOL非常的精彩刺激！戰場之上，鬥智斗勇，一個不留神就被秒殺，年輕人樂在其中，甚至不能自拔，年輕人喜歡的東西商業價值極大，這就是電競的價值，也是職業電競選手的價值。因此從職業價值上看，頂級職業電競選手確實可以和頂級圍棋選手相提並論。

另外一個爭論的焦點就是：圍棋是中國文化與文明，乃至人類智慧的體現，那麼LOL是什麼：

圍棋非常講究思考，它也有很多定式，比如：立一拆二，立二拆三，三子正中，這些可以大大簡化思考的公式，然而實戰中是需要靈活運用的，圍棋選手可以隨時停下來認真思考下一步需要用哪一個定式，或者自創一個定式，但是LOL選手幾乎沒有停下來認真思考的時間，他們的反應在平時的訓練就已經形成，LOL比賽真正體現了「兵貴神速」，慢一秒就會被對面秒殺，LOL當然也需要思考，但是思考的時間不多，它也有很多戰術比如：全球流、四一分推、換線流、拖把流等等。當然這只是說兩種運動的不同，並不是說誰不好。

圍棋擁有著四千年的歷史，它的玩法和規則基本確定，各大聯賽的規則只會略有不同。而LOL的歷史還不到四十年，而且電子競技說變就變，一個小小的補丁就能扭轉遊戲的平衡性，另外還有更大的風險，人們的興趣愛好，眾所周知，十年前的星際爭霸是何等的流行，然而現在是LOL、王者榮耀、和平精英等等。那麼下一個流行的焦點是什麼？誰也不知道，可以知道的是，電競項目一定會變！

要說真正體現人類智慧，LOL和王者榮耀等一類遊戲還差得太遠，電子遊戲對手速要求很高，哪一個高手不是手指抽筋流？其實也有一種遊戲對手速要求不高，那就是回合制遊戲，只不過這類遊戲節奏太慢，無法流行。

另外，星際爭霸和魔獸爭霸等遊戲其實比LOL更考驗智慧，LOL一次只控制一個英雄，而星際爭霸一次戰鬥需要控制一隻軍隊，少說也有十種不同單位，另外還要考慮採礦，還要考慮造建築，還要考慮建築升級，還要考慮兵種搭配。正是由於星際爭霸太複雜，一般人很難玩懂，因此更加簡單的LOL才會流行，但是接下來呢？年輕人會不會挑戰更加複雜的即時戰略遊戲？很難說！

總之，圍棋和電子競技相比，圍棋更加註重思考，電子競技更加刺激，電競明顯更加符合年輕人的胃口，電子競技因此具有極高的商業價值。但是論智慧，LOL是沒法和圍棋相比的。圍棋可以流傳四千年，而LOL可以流傳多久，很難說，說不定幾年後新的電競項目就會崛起，就像LOL取代星際爭霸一樣。

電競頂級選手作為年輕人的偶像，寄託了年輕人渴望快速成長，快速成才的希望。最後，我們當然衷心的希望高振寧可以被保送上清華大學，也祝願他學業有成，事業蒸蒸日上。另外也要提醒一下廣大的還在上學的小夥伴們，高振寧的成功只是個案，絕大多數人還是要好好學習，天天向上才能上清華啊。
