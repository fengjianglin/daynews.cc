陰陽師決戰京都新CG詳解 雲外鏡和晴明拯救世界 SSR大妖聯手反擊

2019-12-23 792條評論 3,706次閱讀 1649人點贊

陰陽師對於近期決戰京都活動中平安京毀滅這件事，引發了不少玩家的熱議，而官方也儘快的發布了相關的新CG視頻來解釋這件事情，看得出來這一切都是官方一開始就已經設定好的，就是為了推雲外鏡拯救世界而特意製作的，來看看本次新CG的具體分析吧。

人家憨憨大岳丸少主不傻，看起來是佔領了平安京，但是晴明這一臉穩如老狗的表情，讓人不得不懷疑這是一個陷阱。

而阿爸也是非常的穩，穩得不行的那種，表示真正的戰鬥現在才開始，馬上就開了反殺大局的態勢。

拯救世界的就是SSR雲外鏡，通過鏡面翻轉，把原本已經被淹沒的平安京，直接翻轉成為了曾經一片祥和的現狀，真就是強推兩面佛拯救世界的節奏。

陰陽師和眾多式神再次集結於其，開始進行反擊，除了常規的四位陰陽師之外，包括茨木、SP酒吞、白狼、獸化的小白、SP大舅、大天狗、SP鬼切、姑獲鳥還有雪女。

之後就是SSR式神各顯神通，對海國來襲的雜兵發動各種攻擊。

小白這操作與其說是守護平安京，不如說是拆家更加合適一些。

SP鬼切CG表現真的非常好，而且輸出能力真不是蓋的，一刀直接砍船頭。

整體的高稀有度式神的表現看起來很燃，很有反殺的氣勢，真香的感覺，順便解釋一下為什麼大舅媽也在這裡，因為不像讓葛葉的孩子在這裡掛掉，所以參戰了。

但說實話這些式神的表現，與其說是守護平安京，不如說是毀滅平安京，而且放遊戲中，真該給清姬和書翁做點戲份，他們在吃PVE的主力啊。

最後是大岳丸被阿爸當面轟大招，整體來說這次的CG表現是真的香，尤其是SSR大妖聯手反擊的畫面，這說明後續活動還會再次開放，所以不用著急，現在該幹啥幹啥，做做日常活動，等待決戰京都活動再開啟就行了，估計應該是雲外鏡開啟召喚集結時活動會重開。
