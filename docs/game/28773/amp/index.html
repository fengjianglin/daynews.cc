再敗JDG，上中拉胯打野無聲觀眾失望，RNG到底怎麼了

天天要聞
2019-12-21

下午3點，RNG迎來了第二場比賽，對手是JDG，而這場比賽和第一場對陣OMG簡直如出一轍，究竟發生了什麼呢？和小編一起來看看吧。

先看陣容，RNG選擇了藍色方，JDG紅色方

RNG：劍魔、蜘蛛、潘森、大頭、泰坦

JDG：鐵男、盲僧、瑞茲、卡莎、日女

比賽開始，小虎的潘森在線上打得非常凶，但一波W上去打傷害時，不小心進入了塔的射程，隨後流浪反打，一路把小虎追到野區。此時小龍堡人傻了，我看你線上那麼強勢去放心反野，怎麼你突然就殘了？沒有狀態打2v2的兩人只能分頭逃走，結果小虎交閃成功逃脫，小龍堡則被盲僧拿下一血。

隨後兩邊互有來回，局面相對平穩。10分鐘，小虎來到上路gank，面對一個出了護臂的鐵男，錯估自己傷害深追，被趕來的盲僧收下人頭。

緊接著，狼行在越塔時空了關鍵的第三段Q，盲僧又收下一顆人頭。上中一套組合拳讓RNG瞬間陷入了劣勢。

此時下路站了出來，大頭憑藉良好的發育和英雄出色的防守性能穩住了局面，雙方發育到20分鐘，火龍區爆發了一波團戰，大頭爆發出超高的傷害拿下雙殺，RNG完成了人頭反超。

之後，RNG開始滾雪球，人頭差逐漸拉開，並在22分鐘時擊殺JDG的下路雙人組，一舉拿下了大龍，局勢一片大好。

但在隨後的推進中，RNG犯下了失誤，野輔率先被擊殺，後續中上也沒能跑掉，這波成為了整局遊戲的轉折點，JDG緊接著拿下了第二條火龍。

隨後，兩方在上路三角草交戰，Kanavi的盲僧踢出了非常漂亮的一腳，JDG完成了一波1換5，經濟反超。此時JDG手握三條火龍，第四條即將刷新，他們聰明地選擇了在中路與RNG頻繁拉扯，利用RNG不想放龍魂的心理找到一波先手的機會，再次打出一波1換5，結束了比賽。

結合上一場與OMG的對局我們可以發現，RNG的上中野都存在著不小的問題，中單天生對線劣勢，保證不被單吃都很困難，導致中野前期不敢打2v2，無法打開局面，所以RNG的戰術就定型為上路三包一。但狼行又是個Carry能力不足的藍領上單，幫了他之後他手握優勢，卻不能繼續擴大優勢，打到後期就和對面上單一個樣子了。

至於小龍堡的問題則是控龍差，兩把加起來才一條小龍。我們都知道季前賽的小龍堆疊到4條時有一個很強力的龍魂效果，誰拿了誰就有八成勝算。第一場控不住龍的原因可以歸結於下路劣勢、蘭博龍區太強，但第二把下路大頭是有優勢的，小龍堡還是控不住龍，這個責任就只能歸結於他自身了。

賽後，觀眾也是紛紛表達了自己的失望，原以為只有中單拉胯，結果這場看下來，上中野沒一個像人的，反而是下路的那個Betty選出了兩把法核，英雄池足夠深，說不定能成為Uzi的接班人。
