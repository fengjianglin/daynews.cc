天龍八部：全服第一卡級少林，百萬評分雙重樓，119都不是對手

2019-12-18 585條評論 4,387次閱讀 2252人點贊

說起《天龍八部》網遊江湖，卡級團體因為人越來越多，已經形成了自己獨立的江湖規則和遊戲體系。

而卡級大神也層出不窮，滿六卡級號已經不算是多稀奇了。

但是，眾所周知，即便是二進階的卡級號，也很有可能打不過滿級滿心法修鍊的119滿5號，卡級的天然限制，導致了卡級號的心法、修鍊等等都可能跟不上滿級號的腳步。

然而，聽香水榭的卡級號們，尤其是聽香水榭第一卡級號，卻顛覆了玩家們的這個認知。

百萬評分的少林，目測至少是2進階滿七號，小編雖然沒有看到這個號的面板圖，預估其血上限至少150萬+，主屬性攻擊1W6+。

惡眸在全服少林排行榜上，也能達到第八名，很是犀利，與第五名評分差距也非常小。

比較矚目的是，這個號還有雙重樓，雙重樓的控制效果可以說是杠杠的，少林的高血量高攻擊高防禦，在同等級卡級號中堪稱無敵。

而且卡92吃卡後，除了心法上有差距外，其餘裝備、修鍊、子女、寶鑒等等，基本無差距，對上119滿級滿心法滿修練的號，也並不示弱。

除了這個超強的百萬少林之外，聽香水榭的卡機大軍也是非常強力了。據小編粗略估計，40萬以上的卡級號越有20個，20萬以上的卡級號，能差不多達到50個以上。

這樣的戰鬥力，不愧是卡級神區，對上普通小區，也是毫不遜色了。
