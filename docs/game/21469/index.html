年度人物：UZI第一 shy哥第三 對於廠長網友卻說抬下來

2019-12-19 506條評論 2,043次閱讀 972人點贊

在全明星結束之後，雖然這段時間沒有什麼比賽，但是轉會期的新聞還是時刻牽動著大家的心。不過當轉會期結束，一切消息都將塵埃落定。無論最終的陣容情況如何，都期待他們在新賽季打出更好的表現。而除了轉會期之外，還有一件事牽動著大家的心，那就是微博之夜的年度人物評選。

其實在UZI開始出現榜單前幾的時候，大家都開始關注了這個評選活動。而且這個活動還出現了一件讓人沒有想到的事情，那就是LPL的召喚師們竟然團結一心，直接把UZI從榜單上不斷推進，最後更是以高額的票數穩居第一。

而在UZI穩居榜首之後，團結的召喚師們又開始把目光放在了後面的選手當中。其中Theshy的排名也是一直在不斷的上漲。UZI拿下第一之後，粉絲們又再次團結一心，開始不斷的把Theshy的排名往前推進。

經過粉絲們的不懈努力，一個意想不到的局面又出現了，那就是Theshy的排名一口氣衝到了第三的位置。可以說，在今年的年度人物評選活動當中，如果不出意外的話，那麼在前三名當中，將會有LPL的兩名選手。而這兩名選手也是LPL當中最受關注的。

大家都知道，除了UZI和Theshy之外，還有一個人的排名也在不斷前進，那就是一直受粉絲期待的廠長。其實今年的比賽當中，廠長久未登場，但是大家還是非常期待能夠看到他的身影。這次的投票活動，無疑是給了大家一個新的慰藉。

所以在把UZI和Theshy都抬上之後，大家又開始準備去抬一手廠長。但是粉絲們還有一個出奇一致的看法。那就是希望能把廠長的排位穩在第七，畢竟7的意志，大家都是了解的。不過因為大家發力過猛，目前廠長是第六名。

所以就開始有調皮的粉絲表示，因為把廠長再度推上去難度還是非常大的，所以大家還是把七醬給抬下來，畢竟放第七名才有儀式感。哈哈，雖然是把排名倒退，但是總覺得大家說的很對。

對於這次的年度人物評選，相信讓更多人的了解到電競的力量，也讓更多的人開始重視電競這個行業。所以希望能夠更多的人理解這個行業，支持這個行業。我們在過去的S8和S9拿下了世界冠軍，這是屬於電競的榮耀，也是屬於各位召喚師的榮耀，未來，我們繼續加油。

最後，大家可以登陸微博，然後搜索年度人物，就可以為我們的選手投票了、
