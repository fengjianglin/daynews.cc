DOTA2新加坡世界賽整體回顧：CN DOTA Never Die

2019-12-23 579條評論 3,391次閱讀 541人點贊

近日DOTA2新加坡世界賽落下帷幕，VG在決賽舞台3比0戰勝EG，成功奪下冠軍獎盃，幫助CN刀塔在年末畫上一個美好的句號。

縱觀本屆賽事，CN刀塔一共派出三支隊伍參加，但小組賽里，CN軍團明顯不在狀態，LGD與茶隊小組賽結束後被淘汰，唯有VG一路從敗者組殺出，接下來就讓我們一起來回顧下這段驚心動魄的旅程。

CN軍團開局不利，VG成為唯一希望
雖然本屆新加坡世界賽是非官方賽事，但是前來參賽的隊伍實力都不容小覷，LGD、Secret、EG、VP等等強隊紛紛登場，似乎在預示著一場好戲的登場。

小組賽階段CN軍團遭遇慘敗，首日比賽LGD和茶隊一場未勝，而VG則是平局無數，反觀國外戰隊，Secret和EG兩支老牌強隊實力恐怖，捷報連連，Navi和A隊更是將CN軍團暴打。

小組賽第二日LGD狀態依舊沒有好轉，他們繼續延續了自己不勝的傳說，而茶隊則在面對Secret的比賽裡面打出士氣，將大魔王囂張的氣焰壓制下去。

不過小組賽結束之後，他們依舊被淘汰出局，僅存的VG也進入了敗者組，身為全村人的希望，大家都在期待VG能夠挺身而出。還好，他們沒有令玩家失望。

淘汰賽階段，VG先是戰勝了老對手TNC，隨後迎來秘密大魔王，在背水一戰的情況下，VG眾人在第三局徹底蘇醒，僅用18分鐘就拿下比賽成功跨過對手。

接下來面對A隊他們輕鬆拿下，總決賽面對EG身為CN刀塔唯一希望的他們眾志成城，在戰隊BP和選手發揮上完爆對手，拒絕者小魚無人可擋，RTZ和Abed面對他沒有任何招架之力。

3比0用一場酣暢淋漓的勝利，VG宣告了CN刀塔的回顧，賽後ROTK面對採訪大聲說出了那句：「CN DOTA Never Die！」在2019年末用冠軍安慰了CN刀塔玩家。

LGD狀態不佳，曾經硬實力第一迷失自我
本屆賽事跟VG奪冠形成鮮明對比的，還有LGD眾人的發揮，同為CN刀塔希望之光，LGD在比賽中的表現實在令人失望。

曾經中路王者Maybe在對線找不到優勢，而AME只剩下刷錢了，隊內幾人打法落後，戰隊BP始終劣勢。

小組賽裡面他們連一場BO2勝利都沒拿到，而更令人絕望的是，LGD目前還沒有找到合適的教練。

對於想要在Ti國際邀請賽拿下冠軍的LGD來說，教練絕對是他們必不可少的對象，希望再經歷了這次失利後，LGD眾人可以找到新的方向，在2020年找到最適合自己的打法。
