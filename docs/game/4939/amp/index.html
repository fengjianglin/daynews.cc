元素崛起降臨，盤點新版本強勢打野（上），上分必看

天天要聞
2 週 ago

摘要

新版本上線將近一周，相信大家已經開始新賽季的定位賽了，這個版本改動比較大，很多人目前還不適應這個版本的改動，特別是打野玩家們，不適應節奏的話就會被版本所淘汰，野區經驗值和野怪刷新時間的改動對打野節奏的影響是非常大的，對版本理解的深度不同會嚴重影響到隊伍的勝利與否。

新版本上線將近一周，相信大家已經開始新賽季的定位賽了，這個版本改動比較大，很多人目前還不適應這個版本的改動，特別是打野玩家們，不適應節奏的話就會被版本所淘汰，野區經驗值和野怪刷新時間的改動對打野節奏的影響是非常大的，對版本理解的深度不同會嚴重影響到隊伍的勝利與否。

而元素崛起的核心就是小龍的掌控了，在拿到4條龍的情況下想翻盤幾乎是不可能的，因為龍魂的過於強大，這更加體現了打野節奏的重要性，如果沒有控到龍，線上優勢明顯也有可能讓對面集齊4龍以後翻盤，如果說上個版本是中野的節奏，那麼這個版本會更依賴打野的節奏了，今天我就盤點一下這個版本熱門打野。

盲僧，作為打野的常青樹，每年的世界賽上都有上場且有不凡的表現，特別是關鍵局，一腳踢出一個世界並不為過，猶記得S4的世界賽上dandy的一腳踢5個奠定了他世界第一打野的地位，而今年小天和寧王的盲僧發揮也非常亮眼，這個英雄也非常受大家的歡迎，作為前期節奏型英雄，技能也非常平滑，打野效率不低的同時抓人能力也不弱，野怪刷新時間的改動，讓盲僧的等級來得更快，節奏也會更好把控，同時由於q技能的斬殺特點，對中立生物的控制比較強，在新版本也會更加如魚得水。

皇子，作為賽場上的常客，皇子的技能機制非常不錯，e技能的攻速加成對團隊的提升也是非常不錯的，雖然前期刷野能略遜於盲僧，但是抓人能力比盲僧稍強，能帶給線上隊友更多的幫助，同時野區掌握度也不低，如果說上個版本是T2的打野，那麼這個版本應該會成為T1的存在，因為他對線上的幫助比較大，能更快的幫助中路取得優勢，掌控地圖資源。

酒桶，作為T0級別的打野，這個版本依然非常強勢，特別是經驗來的更快的情況下，對線上的威脅也會更大，同時被動會讓打野非常舒服，讓其野區的掌控力大大增加，同時，其技能機制對團隊的收益也是非常大的，可以配合中路刺客輕鬆完成斬殺甚至越塔，所以說，這個版本酒桶依然會成為非ban必選的打野。

奇亞娜，這個上個賽季剛出的新英雄迅速在版本中崛起，中野都有發揮，她的可怕程度在於出其不意的秒殺，讓你防不勝防，但是這個版本打野經驗和刷新時間的改動我覺得對奇亞娜是一個變相的削弱，眾所周知，奇亞娜的打野效率並不高，更多是找機會抓人，而且新版本小龍的重要性大大加強，在前期，奇亞娜的野區掌控能力並不強，所以說這個英雄在打野位置上的地位也會受到一定的影響，不會成為打野位上非ban必選的存在，後面應該會更多的出現在中路的位置了。

所以說新版本強勢打野要選擇野區掌控力度大，同時前期能給線上更多幫助的打野，會讓我們的勝率大大增加，所以說這個版本是打野的版本。那麼，你覺得哪個打野是這個版本最強打野呢，歡迎下方留言，最後希望大家定位賽都能10連勝
