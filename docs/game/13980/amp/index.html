部落衝突：13本到來，這些兵種等級率先提升，天詠野豬成了

天天要聞
2019-12-12

近日《部落衝突》13本實裝上線，而聯賽也剛好掐著點結束了。所以想必爆倉已久的滿12大佬們都紛紛搓起了興奮的小手手，擼起袖子準備干他一錘。那麼，面對13本新增的眾多建築和兵種等級提升，到底應該優先升級什麼呢？

有一說一，時隔12本更新已經過去了一年時間，所以庫存豐富的大佬們請隨意….

首先，大本是肯定要升級的。但是卻不用急於往上升星，畢竟升級大本營需要大量的資源和時間，但是卻要到三星以後才能擁有冰凍效果，對於村莊的防禦能力並不會產生質的飛躍。

其次，此次更新推出的戰爭機器大家應該也粗略了解其功能，僅僅多出一隻大皮卡和少量法師實在過於雞肋，因此不妨按照國際慣例，在造好新王和新的防禦建筑後，把資源和時間先用來提升兵營和科技~

值得一提的是，在13本實驗室達到11級後，除了治療、彈跳、骷髏三種法術以外，炸彈人、氣球兵、飛龍、掘地礦工、雷電飛龍、野豬騎士、巨石投手、天使等8種部隊也均有了等級提升。因此，各位首領針對自己平時的遊戲方式，也可以採取不同的升級策略。

聽說新防禦建築”投石炮”成為廣大主流配兵的剋星，部落戰一時找不到進攻的頭緒？不用擔心，因為它來了！全新的四級雷龍，升級只需一千六百萬聖水，幫您劃的更爽快，穩二追三暢遊部落戰！升它升它升它！

另一方面，詠王等級的跨度提升對於喜愛野豬流派的小夥伴來說無疑是一個驚喜。可以預料到，在此次版本更新後各種風格的野豬配兵必然又將回歸高杯大佬們的視野當中！都說遇事不決升野豬，部落中的老大哥果然沒有騙人，COC親兒子又實錘了~

雖然詠王的等級突飛猛進，天詠流派一時間躍躍欲試，頗有取代天女坐上COC頭把交椅的風頭。但是從前不久結束的COC世界錦標賽中不難發現，以天女為前戲的狗球流派依然是頂尖職業選手們必備的絕活兒。而炸彈人、天使、（對英雄奶量上升）氣球兵的等級提升也毫無疑問再次鞏固天女+氣球的主流地位，所以野豬哥認為短時間內天詠或許還無法威脅到女王一姐的地位~

最後，已經升本的老司機不妨透露一下，13本打魚的情況是不是一言難盡呢？
