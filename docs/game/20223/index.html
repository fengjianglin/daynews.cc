DNF：12月職業排名大洗牌 阿修羅榮膺幻神 武極CP自瞄大腳登頂

2019-12-17 392條評論 3,108次閱讀 754人點贊

DNF體驗服前幾天剛剛對瞎子、劍宗等7個職業進行了國服特色加強，其中阿修羅在加強23%之後20s打樁數據已經遠超劍豪成功登頂，不過驅魔和審判就沒那麼好運了，策劃用一波可有可無的小加強直接一筆帶過，不少審判和驅魔玩家最後一絲希望也破滅了。

從10月開始，國服已經經歷了3次特色化職業平衡，這些得到了特色加強的職業大多數都直接躋身超一線行列，近日有玩家排出了12月國服全職業綜合排名，雖有爭議但總體還算客觀。

其中武極登頂真神，因為在即將到來的魔界大戰CP副本中，武極將獲得自瞄大腳，自動鎖定boss，二檔大腳再也不會踢空氣了，這很大程度上提高了武極的實戰能力。

在幻神這一行除了四叔之外，戰法、劍豪、武神、召喚、魔道和瞎子全部都是國服特色加強的產物，而且女鬼和男格鬥將率先開放三覺，未來很長一段時間劍豪和武極都會排名登頂。

而在第一批特色加強中獲得無我劍氣的劍魂也在兩個月之後逐步下滑，技能數據被不少後來加強的職業趕超，紅眼更是不進則退，在短暫的輝煌了一段時間後現在只能處於純C中游水準，除了出手快之外，打樁傷害真的平平無奇。

當然，還有一部分職業被策劃徹底遺忘，在審判、戰法、特工和劍宗相繼加強之後，刺客和女漫遊已經穩居純C之恥的「寶座」，柔道、血法、締造者等冷門34C更是看不到任何希望。

雖說任何遊戲都不可能做到真正的職業平衡，但也希望策劃不要過度造神，多關注一下冷門職業，春節版本趕快都安排上！
