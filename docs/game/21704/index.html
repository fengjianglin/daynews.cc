廠長宣布退役，微博之夜隨即殺進前四，飯圈大呼：還能這樣拉票

2019-12-18 488條評論 2,673次閱讀 1317人點贊

2019年12月16日晚8點56分，也就是昨晚，EDG官方放出人員變動公告，封面上是一個與大家闊別了一小陣子的男人，這次回來他換了個新髮型，封面上的照片拍得也格外帥氣，不過沒想到，這樣亮相的廠長，居然是來和大家說告別的。

這對整個電競圈來說無疑是一個爆炸性的消息，廠長退役了？那個永不言棄的男人，那個一路堅持到現在的男人真的要退役了？峽谷的野區中，再也見不到77777的操作了？

這兩天如火如荼的微博之夜年度人物評選，本來廠長是位列第六，甚至很多朋友開玩笑說給李現投幾票，讓廠長保持第七的位置，因為7這個數字，實在對於廠長有太深的意義。

而隨後，就在廠長宣布退役以後，今天一早，廠長的排名已經超過了朱一龍、肖戰，直接殺到了第四，而且他的票數，還在飛速地上漲。

也許有更多的人，希望廠長能夠帶著光輝，再走一次紅毯吧。

繼UZI佔得年度人物第一的位置，爆出冷門後，廠長也終於發出了自己的聲音，飯圈女孩也許根本搞不清楚這兩天名次的變動究竟是什麼原因，不過我們一定懂得，沒錯啊，這兩個男人，不管背負過多少罵名，有過多少次失利，可最終堅持下來的，也只有這兩個人了。

心中的世界第一打野，諾言之後，再不會有人擔得起這名號和信仰。
