說起超級英雄改編遊戲，玩家們想要的究竟是什麼？

2019-12-20 924條評論 4,813次閱讀 2850人點贊

1938年，史上第一位超級英雄角色《超人》問世，初次讓人們接觸到超級英雄的概念。所謂超級英雄，應當是懲惡揚善伸張正義，擁有普通人所不具備的能力維護世界和平的強大人物，自這個概念誕生起便獲得了眾多擁躉，至今已擁有了絕對強大的影響力與商業價值。

超級英雄的概念誕生於漫畫，而後諸如與超級英雄相對立的反派和立場不同的「反英雄」也逐漸開始受到創作者們的重視，那個近在咫尺而又遙不可及的宇宙逐漸擴大與完善，相關的周邊產品以及影視產品也逐漸開始湧現，其中也必然少不了改編遊戲。

一直以來，人們對於「改編遊戲」這四個字向來不抱有什麼好感，彷彿這四個字本身自帶負面Buff一般誰都不願意接觸，諸多影視改編作品皆是如此，而更不用說受眾群體更大的超級英雄圈。而在最近幾年，市面上也不乏許多製作精良的超級英雄遊戲，這些遊戲提升了人們對於這個類別的審視標準，也充分讓開發商逐漸意識到一款好的超英遊戲應該具備哪些元素，一時間這個類別的前途變得逐漸明朗了起來。

摸索

超英遊戲的出現最早能夠追溯到上世紀80年代的雅達利時期，那個時候要說最受歡迎的超級英雄角色自然當屬「人們的好鄰居」蜘蛛俠，漫威漫畫公司靠著這個角色正式開始走進大眾的視野，而改編遊戲這份「榮譽」也自然是要頒給蜘蛛俠。1982年1月，漫威的第一個漫畫改編遊戲《蜘蛛俠》登錄雅達利2600主機，玩家們所期待的是能夠在遊戲中體驗與反派激情對戰、在大樓間隨心擺盪的遊戲，而在雅達利2600上的《蜘蛛俠》卻只是一款敏捷類的「爬樓遊戲」，玩家控制蜘蛛俠一邊向大樓上方用蛛絲攀爬，一邊躲避從窗口探出身子來的敵人以及大樓上的炸彈，在限定時間內爬上樓頂拆除綠魔放置的炸彈，這便是遊戲的全部內容。

先不說蜘蛛俠需要控制蛛絲才能爬牆，蛛絲竟然還是黑的

鑒於雅達利的貧弱機能，能夠做成這樣已經很不容易了，畢竟那個時候的大多數遊戲內容甚至還不如這款《蜘蛛俠》。而後時間進入由任天堂主導的FC時期，鑒於有漫威的改編遊戲在先，另一邊的友商DC自然也是不能善罷甘休，於1988年在全PC平台推出了《蝙蝠俠：披風鬥士》，這款遊戲在畫面的表現上要優於同時期的許多遊戲，遊戲本身採用了橫版過關的形式進行，甚至還提供了與小丑與企鵝人對抗的選項，但由於遊戲糟糕的操作以及奇怪的系統設計導致遊戲的體驗非常差勁，雖然遊戲中有玩家們想要的，但體驗太糟心也很難讓人有毅力將遊戲進行下去。

遊戲的想法很好，但理念還不夠完善

認知

要說史上最爛的一部超級英雄遊戲，非任天堂N64平台上的《超人64》莫屬，超人的能力可謂是人盡皆知，擁有絕對的力量與速度、刀槍不入、兩眼發出射線等等，他在漫畫中就是無敵般的存在，而玩家們在遊戲當中自然也是想要體驗這種無敵的感覺。而在《超人64》中，雖然遊戲有著3D的畫面，但超人的一眾超能力在遊戲中均未得到展現，反而在大多數時候玩家需要控制超人在天空中鑽圈圈，同時遊戲的操作還十分蹩腳，「需要在時限內鑽過所有的圈圈以到達目的地拯救你的朋友」便是遊戲中將近90％的內容。

即便是與其他N64遊戲相比，《超人64》的畫面也非常差勁

這樣一款遊戲自然是免不了玩家們的謾罵，讓一個無敵的英雄只是來回鑽圈圈成何體統，更何況遊戲中的超人就像是中了迷幻劑一般不聽控制，對於一眾喜歡超人的粉絲來說這款遊戲簡直堪比災難。就像上文中所說的那樣，早期的玩家們就僅僅是想在遊戲中體驗親身扮演超級英雄的快感，使用各式各樣的超能力打擊犯罪，但卻少有作品能夠實現這一小小的要求，當時的大多數開發商認為，只要遊戲當中包含了漫畫或影視作品中出現過的元素玩家們就能夠心滿意足，顯然他們理解錯了。

而在筆者的印象中，我曾體驗過的第一款感覺良好的遊戲作品，是動視發行Treyarch製作的改編自同名電影的《蜘蛛俠2》。這款遊戲幾乎是與電影同步上市，採用虛幻2進行開發，有著在當時看來非常出色的遊戲畫面，並且除去電影中的角色，還加入了部分漫畫中出現的角色以填充遊戲內容，雖然遊戲的流程不長，但蜘蛛俠該有的花式戰鬥、盪蛛絲爬牆等元素均有很好的還原，雖然遊戲的自由度並不算太高且可玩性很低，通關之後也基本無事可做，但遊戲本身卻營造出了一段十分良好的遊戲體驗，第一次讓玩家真正意義上扮演了一回蜘蛛俠。

遊戲流程緊湊，即便是不懂英文也能玩得明白

進化

市面上大多數超級英雄遊戲均來自漫威與DC，並且涵蓋了各式各樣的款式與類型，但對於玩家們來說，依舊是動作類遊戲最能讓人感到滿足。2009年動視（沒錯又是動視）曾推出過一款改編自同名電影的動作遊戲《X戰警前傳：金剛狼》。正是這款遊戲第一次顛覆了許多玩家對於超級英雄遊戲的認知，雖然遊戲中的主角與電影同為由休·傑克曼扮演的金剛狼，但遊戲中的金剛狼明顯比電影中的人設要野蠻許多，遊戲本身的玩法與《戰神》系列相似，通過各式各樣的組合按鍵釋放不同的招式，但卻能夠使用利爪將遊戲中的或物撕成碎片，場面異常暴力血腥，而主角受到攻擊後甚至能夠看到身體的內部結構，雖然有些顛覆電影中的人設，但遊戲中的戰鬥部分卻異常的出色，讓人大呼過癮。

即便放到現在，遊戲的整體表現也並不算差

而要說到迄今為止人氣最高口碑最佳的超英作品，還得數在遊戲與電影領域兩開花的《蝙蝠俠》系列。由克里斯托弗·諾蘭導演的《黑暗騎士》三部曲被譽為有史以來最出色的超英電影，而由Rocksteady開發的《阿卡姆》三部曲也被譽為最棒的超英改編遊戲，而之所以這個系列能夠被廣大玩家認同，則是由於遊戲本身不僅在還原蝙蝠俠的設定上下足了功夫，同時還在遊戲當中融入了些許唯有用遊戲才能實現的敘事手法，比如意料之外的場景切換，能夠是遊戲在遊玩過程中獲得優於電影的體驗。

《阿卡姆》三部曲中綜合表現最好的無疑是《阿卡姆之城》

簡單來說，遊戲在高度還原蝙蝠俠設定的同時，也大膽融入了許多開發者自己的想法，使得遊戲的表現力與遊戲體驗更上一層樓，同時遊戲的操作與系統等設定也識時務的與時俱進，即便不是蝙蝠俠的粉絲也能夠輕鬆讀懂劇情與背景，是一款能夠讓粉絲與路人都玩得爽快的作品。

結語

去年發行的《漫威蜘蛛俠》無疑為玩家們帶來了足夠的驚喜，即便明知這是一款商業氣息濃郁的作品，但對於玩家們來說市面上已經好久沒有出現過這般大製作的超英遊戲。這就好比之前天天吃稀飯饅頭，突然給你加了點紅腸榨菜，就會突然讓人感覺很滿足。平心而論，《漫威蜘蛛俠》的製作質量也是可圈可點，否則玩家們也不會這麼痛快的買賬。

唯一的遺憾是內容略顯敷衍的公式化開放世界

幾年前玩家們玩超英遊戲是想在遊戲中得到扮演超級英雄的快感，而在近幾年人們的要求也不再這般單薄，一款超英遊戲需要在滿足其最基本的內容同時也需要有優秀的製作質量、足夠的遊戲內容、以及適合超級英雄設定的獨特新創意，這樣的一款作品無疑會非常優秀，但從另一個角度來看也是玩家們的口味變得更加刁鑽了，而鑒於改編遊戲自帶負面Buff，稍有不慎就有可能成為玩家們口誅筆伐的對象，因此或許唯有對於玩家意願有充分了解、並且願意花重金投入製作的公司才能夠做出玩家們真正滿意的作品，但歸根究底，真正願意這般投入的公司又有幾個呢？
