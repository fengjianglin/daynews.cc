Clid接受採訪表示SKT氛圍差，被網友集體爆破：你這個白眼狼

2019-12-16 1,412條評論 4,806次閱讀 1191人點贊

LCK的冬季轉會期早已落下帷幕，在世界賽失利的SKT無疑是這次轉會期最引人關注的對象：逐漸老去的Faker是否能帶隊繼續保持競爭力？轉會期SKT不出所料對戰隊人員進行了大換血：上賽季的大腿上野組合均未選擇續約，Khan來到FPX，Clid加入Gen.G。上賽季的陣容僅剩李哥和下路兩位新人，多年過去SKT人來人往只有Faker一直坐鎮中路，不禁讓人唏噓。

轉會至Gen.G後的Clid最近接受了採訪，引起了軒然大波。Clid在SKT的一年中堪稱隊伍最粗大腿，在比賽里的影響力不比Faker小！作為與Doinb經常雙排的打野和前JDG隊員，中國觀眾對這位小胖子也比較熟悉，轉會Gen.G也算是意料之中，畢竟李哥狀態不復當年，Clid在當打之年也需要一份長合同。

在採訪中，記者問到Clid加入Gen.G的原因時，Clid回答道：「我說出來可能感覺有點怪，但是我希望在一個感覺大家都是朋友的隊伍中。像G2在季中賽奪冠的時候，還有FPX，因為他們是世界賽冠軍。像他們一樣的隊伍。這些隊伍給了我啟發。因此，我加入Gen.G是覺得他們是最適合我的隊伍。」

這樣一番話雖然沒有明說，但大家都看得出來Clid對SKT的氛圍不太滿意，甚至有粉絲直言：你這不就是在暗示李哥嗎？

這番話也得罪了韓國的粉絲們，紛紛跑到Clid推特的評論區對他進行攻擊！大家對Clid這番採訪也眾說紛紜，大部分玩家都認為Clid在離隊後還這樣暗示實在有些沒必要，氛圍不好還能拿兩個冠軍也不錯了！也有玩家認為李哥確實是在遊戲里投入了太多精力，不太喜歡交朋友！

確實，SKT一向以嚴格訓練和管理聞名，這也是這隻隊伍一直能保持高水準的基礎。Clid顯然更喜歡類似FPX這樣的兄弟LOL，正所謂「道不同不為謀」，兩隊下賽季賽場見真章吧！
