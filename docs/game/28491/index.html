《瘟疫》評測：與眾不同的非線性流程冒險遊戲

2019-12-23 3,063條評論 4,891次閱讀 3161人點贊

人能從潔白中拷打出罪惡，亦能從罪惡中拷打出潔白。——陀思妥耶夫斯基《罪與罰》

瞬息萬變的電子遊戲市場催生出一代代優質的遊戲作品，但總有那麼一兩款製作精良又相對冷門的遊戲淹沒於浩如星海的藝術結晶中，它們沒有使用出眾的物理引擎和精心錄製的音效，只用反覆打磨的優秀劇本來打動玩家。Ice-Pick Lodge是一家位於俄羅斯莫斯科的視頻遊戲開發商，自2002年成立後就致力於冒險類遊戲（ACG）的開發。該工作室的處女作《瘟疫》（Pathologic）一經上市就在俄羅斯斬獲數個遊戲類大獎，但這款在俄羅斯頗負盛名的遊戲在國內卻鮮有問津，這樣一款運用了三合一劇本及非線性流程的冒險遊戲到底有著怎樣的魅力？

故事發生在西伯利亞深處一個無人問津的小鎮上，這是由一群游牧民族遷徙而形成的聚落，最終不斷發展，成為一個具有相當規模的城鎮。這種設定不禁讓人聯想到陶淵明筆下那令人無比神往的桃花源，但永遠陰沉的天幕和刺骨的寒風都提醒玩家這裡將不再是幸福的天堂，而這場終將以悲劇收場的戲劇也緩緩拉開了帷幕……

在故事開場的劇院中，三位主人公正進行一場有關”拯救”的爭論，最後分道揚鑣，玩家就是在這個時候正式開始遊戲，而三位主人公則代表著不同的劇本——學者、占卜師和女巫，當玩家選擇其中一位進行遊戲時，另外兩位則會成為NPC，他們在遊戲中也會觸發關鍵劇情，甚至與玩家產生交流。

一場瘟疫毫無預兆就向這個與世無爭的小鎮席捲而來，當然，它並沒有像一支鐵血軍隊一般來勢洶洶，也不會像嗜血屠夫一般瞬間將小鎮變得屍橫遍野。相反，它隱藏於波濤洶湧的暗流下——百歲老人的死亡、村民焚燒屍體、屠宰場的關閉，玩家只能在不斷的對話中用自己的推理揭開它神秘的面紗。而這場恐怖的”沙疫”只是小鎮變故的導火索，是點燃炸藥桶的最後一點星火。

遊戲的任務系統，如若不能達成某些目標就會造成關鍵人物的死亡

在三位主角中，學者是小鎮里唯一的一位外來者，所以通過他的視角可以最真實還原故事的全貌。從遊戲的任務系統中可以看出如若不能完成當天的任務就會造成關鍵人物的死亡，從而造成劇情上的失敗。作為一名試圖拯救小鎮的冒險者，玩家的任務複雜且艱巨，不僅要保證在對話時盡量不觸發那些容易引起失敗（比如讓關鍵NPC對主角的好感下降）的選項，還要時刻在那些被感染的建築中搜集線索、與流浪的竊賊戰鬥、躲避被感染的白化病人。影響劇情走向的要素非常多，玩家不僅要慎重斟酌每一次對話的措辭，甚至在一些關鍵劇情中還要運用sl大法。

遊戲中一些屬性值——名聲、血量、疲勞度、飢餓值、感染度

屬性系統在一定程度上也達到了”擬真”的目的，對人出言不遜、攻擊NPC、盜竊都會出現名聲下降的情況。尤其讓人覺得驚喜的是藥物系統，在本作中出現了三種不同程度的藥物，止痛藥可以恢復血量，但會提高疲勞度，降低疲勞度的辦法就是睡覺，而在睡覺時恢復效果則翻倍，除此之外，抗生素類藥物還能降低感染度，但也有很大的副作用。

“堅果”作為遊戲中小孩子的貨幣，可以用來交換情報

麻雀雖小，五臟俱全。在遊戲中存在著各種各樣的小細節可以讓玩家在這個封閉的小鎮一探究竟，比如”堅果”這個小道具是小孩子間流通的貨幣，可以從他們身上獲取一些情報。另外遊戲中存在著比較完善的交易系統，同樣的物品在不同的人看來有不同的價值，所以玩家甚至可以在小鎮上扮演一個商人——當然，這只是一個娛樂項目而已，因為隨著疫情的不斷擴大，留給玩家的時間會越來越少。不過金幣在遊戲中的用處很大，不僅用於購買槍支彈藥和各類抗生素，還要用來在小鎮的臨時監獄中贖人甚至僱傭殺手。

精細龐大的地圖，上面的地區多以人體器官命名，血手印為主線任務，白手印是支線任務

這個與世隔絕的小鎮在設計上具有相當高的完善度。隨著劇情的推進筆者發現這個小鎮上存在三大家族，分別控制著小鎮的法律、醫療和治安。而隨著瘟疫的爆發家族之間的摩擦也愈發激化，切斷水源、關閉屠宰場、教堂成為避難所，就連神秘的多面體建築里也藏著很多秘密……每天玩家可以從一位NPC手裡購買到最新的地圖，紅色的部分象徵著瘟疫區，灰色的地方則是瘟疫過後恢復正常但罪惡滋生的區域，玩家需要橫穿這些危險的地帶，為整個小鎮的復甦奔波著。城鎮的地圖非常大，但與之相對對的則是玩家緩慢移動的速度，在遊戲中不能奔跑，只能步行，所以在大部分時間裡，與玩家作伴的是陰沉的天空和若有似無的慘叫聲，值得一提的是遊戲的背景音樂雖然製作上比較簡陋，但配合整個遊戲的氛圍來看還是十分詭異的。

隨著遊戲的不斷推進，主角與三大家族的接觸就愈發頻繁，這時，劇情的閃光點就顯現出來。小鎮上的百姓將瘟疫妖魔化，甚至燒死那些被認為是魔女的女性。管理治安的家族隨著手中權力的擴大開始瘋狂搜捕對他有威脅的人，那些染上瘟疫的人衝進教堂，試圖讓其他人都染上這種可怕的瘟疫……玩家在為了一線生機不斷奔波的時候，整個小鎮的人們卻只是把玩家當做工具，為了達成那些不可告人的目的，所以，雖然在第六天的主線劇情中就研製出了疫苗，但對於這個開始分崩離析的小鎮來說毫無用處。

為了研製解藥主角以身試藥的場景，那宛若鬼魅的白風正是瘟疫具象化的體現

自此，遊戲的走向又向一個極端滑落，隨著家族和家族間衝突的加劇，主角需要處理的事情也變得多了起來。為了拯救家族鬥爭中無辜的犧牲品，需要用大筆錢去贖人，為了拯救垂死的病人而不得已犧牲幾個無名小卒的性命作為交換……玩家在扮演一個與時間賽跑的”工具人”的同時，內心的掙扎也與日俱增。遊戲中一些支線和主線任務的交互也非常有趣，例如其中的一個任務是陪伴一位舞女去與她的繼父對峙，但主線的任務卻是在同一時間和另一位NPC交互，當玩家不得已選擇主線時，卻意外發現這位舞女作為犧牲品出現在主線劇情中。這樣的例子數不勝數，即使玩家只選擇一種劇本遊戲，也能從很多角度了解這個小鎮的背景。

遊戲的第七天，新的執法者上任，劇情更加黑暗

這種精彩絕倫的劇情交互模式如同將玩家親自置身於這個西伯利亞的偏遠小鎮上，二十個小時的獨立劇本也如同美劇一般絲滑順暢。遊戲中出現了很多詭異的事物——蟲人、倒懸多面體建築、魔女、鳥頭執行者、無臉悲劇作家，都暗示著這個小鎮的虛構性。但小鎮中那些”活生生”的人所相互之間產生的化學效應，卻無比真實——無論是那些勾心鬥角的野心家，還是殺人卸貨的僱傭兵、熱心腸的貴族二世，都在反覆印證《罪與罰》中那句名言：人能從潔白中拷打出罪惡，亦能從罪惡中拷打出潔白。

鳥頭執行者的形象讓人聯想到《哈利波特》系列作品中的攝魂怪

這部遊戲作品不禁讓人聯想到德國真實存在過的模擬監獄實驗，將二十名男子分別投進監獄中，一隊人扮成獄警，另一隊人扮成囚犯，在這個閉塞的環境下，擁有不同身份地位和權力的人所作出的事情超乎了所有人的意料……就像遊戲中開場出現的鳥頭人對玩家所說的那個單詞”play”一樣，玩家不是在”play”（玩）這個遊戲，而是作為一個活生生的參與者，真實融入進這場人性的”play”（戲劇）中。

電影《死亡實驗》中，扮成獄警的一方開始盡情施展對犯人展開攻擊

也許，就像《瘟疫》中表達的那樣，人性中”惡”的一面是禁不起考量的，而固步自封更是”惡”的催化劑。小鎮中雖然有著等級森嚴的制度和冷酷的法律，但矯枉過正的一系列措施卻適得其反，過渡的壓迫最終結出惡果。在遊戲過程中可以看到類似東正教的女性卻衣著暴露，被稱為”聖潔天使”的女性卻有著魔鬼的另一重人格。小鎮就像《銀河帝國》里川陀的麥麴生區一樣，雖然極力壓制著人類的本能，但人的慾望卻絲毫不減。最有趣的是，那群鳥頭執法者每次都以一個類似中間人的身份冰冷地注視著主角所做的每一個決定和小鎮的每一次墮落，像極了波萊德爾在《惡之花》中對信天翁的描述——他們慵懶地注視著旅人，而船則是在苦澀的深淵中滑行。

看看風景時發現這個遊戲遠景做得還不算太糟糕

令人遺憾的是，由於沒有漢化和建模粗糙等原因，這款遊戲界的”罪與罰”一直默默無聞，也許只有耐心的玩家才會在偶爾歇息的時候，從不起眼的角落裡發現這款神作的微光，抖落它身上的灰塵，然後沉浸在故事裡好幾天——麻木機械的小鎮分崩離析，劇場里上演又落幕的詭異啞劇，道貌岸然的貴族勾心鬥角，都從踏上對抗瘟疫之旅的那一刻就已經開啟，如果說這款遊戲除了午夜時的一身冷汗外，還讓我明白了什麼道理的話，那我一定要說——

但行好事，莫問前程。
