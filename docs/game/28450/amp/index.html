絕地求生黃金大獎賽首現女選手！還一穿三淘汰VG 十年老粉全來了

天天要聞
2019-12-23

最近絕地求生黃金大獎賽正在火熱開打了，目前決賽第一天的六場比賽已經結束，TSG吃下兩雞暫時排在積分榜第一的位置，緊隨其後的就是BA和TITAN這兩支隊伍。

在這次黃金大獎賽中，讓飯堂君最高興的就是OMG狀態終於回來了！雖然離巔峰還有點距離，但至少好起來了呀，小組賽壓線晉級到決賽，決賽首日也是吃下一雞排在積分榜第六的位置，OMG真的好起來了！

除了OMG之外，SMG狀態也逐漸回歸，感覺在黃金大獎賽，老牌隊伍的狀態都逐漸回來了，不過有一說一，手雷真的是天克SMG這支隊伍，尤其是小組賽最後一天的第二場比賽，本以為SMG能穩定吃雞，但沒想到被小鹿一雷雙響，二打四吃了雞，在決賽首日也是，也被王總的手雷制裁過，該說SMG是被手雷天克還是說被4AM天克呢？

當然，這次黃金大獎賽最吸引玩家注意力的就是有女選手上場了！在小組賽最後一天，CSG換上了一名女選手玥玥，這位選手一登場，彈幕瞬間出現了眾多CSG十年老粉，而且這位小姐姐不僅是顏值高，水平也是非常厲害了，VG心裡苦呀，被玥玥小姐姐一穿三淘汰，打出這個操作之後，玥玥的粉絲就更多了！

不過CSG最終沒能晉級到決賽階段，可惜了可惜了，要是CSG進決賽，就又能看到玥玥小姐姐了，不過隊伍雖然沒晉級，但是在評論席上也看到了玥玥小姐姐，啥也不說了，十年老粉不請自來（狗頭）。
