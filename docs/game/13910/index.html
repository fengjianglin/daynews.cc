王者榮耀：三年玩家1分錢未花，看到皮膚數量，這才是最強毅力帝

2019-12-12 744條評論 4,074次閱讀 795人點贊

隨著電競的發展，現在的遊戲種類越來越多，玩家在眼花繚亂的同時，關注的不僅僅是遊戲的可玩性，更關心的是氪金的程度。而王者榮耀作為一款5V5的moba手游，相對公平的競技模式，讓玩家即使不氪金，也完全不影響遊戲體驗和上分。這也是王者榮耀能火遍全網的重要原因之一。

雖說不氪金不影響遊戲體驗，但是面對精美的皮膚，很多玩家都忍不住試水。近日卻有一位玩家曬出了自己的資料圖，直言自己是三年老玩家從未花過一分錢，讓不少玩家羨慕的同時，也佩服這位選手的毅力。

可以看到這位選手的遊戲生涯達到了1228天，在峽谷中戰鬥了三年半的時間，英雄數量達到了92個，可以說能用金幣獲得的英雄都獲得了，只剩下武則天等英雄，皮膚數量更是讓人羨慕，達到了59款！除了賽季獎勵皮膚、周年限定皮膚外，玩家自己還積攢了大量的皮膚碎片用來兌換皮膚。

即使一分錢未花，也完全不影響玩家上王者，從S11賽季一直到S16賽季，一直都是最強王者。這讓貴族八還在鑽石的選手，只能默默的羨慕了。玩家親身證明氪不氪金和上王者完全沒有任何關係，系統也不會因為你不是貴族就給你匹配坑隊友。

一直以來天美每隔一段時間就會上線有趣的活動，玩家可以獲得大量的遊戲道具，還有機會獲得皮膚和英雄。這位玩家在擁有99%英雄的基礎上，還擁有17萬金幣和兩萬多鑽石，積攢的背包道具更是讓人羨慕，光改名卡就四張，還有眾多皮膚體驗卡以及英雄體驗卡，不過玩家在曬出自己的皮膚和英雄時，有細心的網友發現不對勁，因為該玩家自稱一分錢未花，但是玩家英雄中卻有趙雲。

眾所周知趙雲是首充才能獲得的英雄，玩家零氪是如何擁有趙雲的？還有網友調侃：趙云：看不到我看不到我！玩家在底下解釋了原因，是因為之前活動送的，並且自己主頁上還有首充的字樣。之前確實是有免費獲得皮膚的活動，除了武則天外都可以隨機獲得，例如春節活動，還有更早的修鍊之路獎池等等。

該玩家為了消除網友們的質疑，曬出了有力的圖片，這是玩家的個人生涯，可以看到最底部的我的資產：英雄92，皮膚比首頁的多了一款，也許是剛剛兌換的288皮膚，重要的是消耗點券數為0，各位小夥伴你們身邊都零氪的選手嗎？他們的皮膚都有多少了呢？歡迎留言討論哦！
