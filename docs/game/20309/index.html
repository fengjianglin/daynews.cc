暴雪遊戲烏鎮遊園會 站點打卡彩蛋大揭秘

2019-12-17 406條評論 2,155次閱讀 1132人點贊

當暴雪遊戲遇上烏鎮，是詩與遠方還是魔法與劍，下面有請我們的安度因與吉安娜為大家帶來今天的黃金聯播。

購買活動門票 暢遊烏鎮贏好禮

僅需188元即可購買2019黃金總決賽或黃金年度盛典單日票（含150元烏鎮西柵景區門票）

實體隨票獎勵：定製Q萌滑鼠墊、隨機祝福紅包、黃金鑰匙扣、定製禮品袋、觀賽手冊、應援棒、隨機主題明信片、遊園打卡活動獎品

虛擬隨票獎勵：《爐石傳說》勝利之悅卡背

購票及投票：https://gold.blizzard.cn/final

七大站點打卡大揭秘

在為期兩天的暴雪遊戲烏鎮遊園會裡，你不僅可以欣賞本年度最高級別的精彩賽事對決，還可以與眾多知名主播們，一起見證2019年發生在暴雪遊戲世界裡的輝煌與榮耀。

再悄悄告訴你一個秘密：酒館的鮑勃也來到了烏鎮！

只不過他現在似乎有點焦頭爛額：他不但弄丟了自己的棋盤，還接到了眾多僱主的委託——他真的有點忙不過來了。為此，他發布了懸賞任務，只要你動動手指幫幫他，他將會把自己珍藏多年的獎勵慷慨地贈予你！

當你完成任務後，記得要在烏鎮大劇院前的「暗月馬戲團」找到那個光頭，即可領取你的獎勵！

《爐石傳說》遺失的棋盤

粗心的酒館鮑勃弄丟了他的棋盤，這使得他無法在戰棋模式中給冒險者提供消遣，酒館的生意也一落千丈，現在他心急如焚。

親愛的冒險者，你願意幫助鮑勃找到他的棋盤么？只要你找到它，並留下你和它的合影，鮑勃就知道他的棋盤在哪了。據此前的冒險者透露，鮑勃的棋盤好像出現在一處叫做「露天影院」的地方。

《魔獸世界》遙遠的信件

曾經的你心比天高，心中秉持著正義與榮耀，那時的你是否還有很多未竟的願望，或是未了的遺憾？是否有未曾說出口的話，想要留給某個心愛的姑娘 …

據說，烏鎮郵局的位面與艾澤拉斯大陸產生了融合，展露出了艾澤拉斯的一角。現在， 你可以使用那裡的郵箱，向曾經和你一起討伐拉格納羅斯、抵抗天災入侵、在厄運之槌尋寶、浴血在奧妮克希亞巢穴的老朋友們發一封郵件，將你的思念與祝福，以跨越時空的方式告訴他們。當然，你也可以給那時的自己留言，將來自未來的忠告，留給曾經的自己。

《守望先鋒》半藏的召喚

來自未來的智械利用蟲洞打開了一條時空隧道，來到了這個守望先鋒還尚未集結的時代，他們的目標是那些未來會對抗他們的年輕的「守望先鋒」們，甚至要不惜一切手段殺死未來的領導者「死神」萊耶斯。覺察到了危機的守望先鋒派出半藏回到了現在，他在烏鎮開了一家射箭館，並在那裡召集他未來的同伴，指導他們成為未來的「守望先鋒」。

如果你能找到半藏，並和他合影，立志和他一起在未來應對智械危機，那麼你的履歷上（任務折頁）將蓋上來自守望先鋒的專屬印記。

《星際爭霸II》丟失的探測器

隨著埃蒙實力的突然加強，以及蒙斯克大帝的再次現身，讓宇宙重新籠罩了一層陰影。為了應對可能發生的戰爭，星靈指揮官們正在加緊鞏固自己的基地，並派出了大量的探測器去探尋新的疆域。

就在前兩天，酒館鮑勃接到了星靈指揮官的任務，在尋找新的資源過程中，一個探測器莫名失去聯繫，他們希望消息靈通的鮑勃能幫他們找到探測器。

HI，指揮官，你願意幫助鮑勃尋找到丟失的探測器么？據說，探測器好像曾在燈籠鋪短暫出現過！

《魔獸爭霸III》新的世界

無數個世紀以來，爭霸中的艾澤拉斯日漸殘破，來自遠古的泰坦正在安撫日漸狂暴的元素之靈，並想要重塑艾澤拉斯的山川海洋。

HI，冒險者，你願意幫助泰坦們重塑艾澤拉斯么？去尋找那些散落在烏鎮某處的拼圖，並將他們恢復成原本的樣子，泰坦們會根據你的提示來修復艾澤拉斯。然後，你會發現嶄新的世界比原來更廣大、更清晰。你會成為艾澤拉斯新的英雄，你的故事將會在魔獸爭霸未來的歷史中永遠流傳。

《風暴英雄》時空樞紐

新的時空之門已經打開！連結烏鎮與時空樞紐的橋樑已經出現，你還怎能停滯不前！

時空扭曲所產生的能量已經影響到了烏鎮的某個地方，那裡出現了原本只該在時空樞紐中的景象。據說…可怕的死亡之翼也可能會出現在那裡？

嘿，英雄，趕快去看看是怎麼回事兒！記得將那裡的景象拍下來，並收集更多的信息，越詳細越好，然後將你的發現告訴周圍的工作人員，他們會給你蓋上特殊的印記。

《暗黑破壞神III》墜落的天使

為了避免天堂與凡間陷入永恆的地獄，泰瑞爾在凡間發起了召喚，任何秉持著正義的奈非天，只要尋找到泰瑞爾的祭壇，並獻上自己的正義之力，幫助他恢復天使之力，就可以獲得泰瑞爾的祝福。

神秘的寶藏任務

由於發生了時空扭曲，在遊園會期間，烏鎮散落著許多來自暴雪遊戲世界的小玩意兒，可能是一個帶有印記的酒杯，一個噴漆，一個氣球，甚至是一個魔法棒棒糖！只要你能拍下任何帶有它們的照片，展示給位於暗月馬戲團的酒館鮑勃，你就能獲得額外的神秘寶藏！

獎勵領取

  • 按照任務折頁的提示，完成七個站點打卡主線任務，獲取工作人員的7個印章，即可到暗月馬戲團換取獎勵。
  • 任務折頁蓋章和社交媒體傳播（微博、朋友圈等）是領取獎勵的必備條件。
  • 發現並和活動區域內含有暴雪元素的彩蛋合影，發現越多，驚喜越多！
  • 換取獎勵後，你可以用手中的任意一枚徽章，和酒館老闆鮑勃換取一次抽徽章的機會，如果你足夠幸運，就有可能獲得暴雪限量獎勵。
  • 主線任務、彩蛋任務和徽章抽取，都有且僅有一次換取的機會。另外，玩家也可交換徽章。
  • 有關本次活動的相關信息，如有疑問，可以隨時向現場的工作人員進行詢問。

場地信息 浙江 · 烏鎮大劇院

烏鎮西柵景區以千年積澱的文化底蘊成為江南古鎮中的佼佼者，西柵毗鄰古老的京杭大運河畔，由十二個碧水環繞的島嶼組成，真正呈現了原汁原味的江南水鄉古鎮的歷史風貌。在烏鎮舉辦2019黃金總決賽暨黃金年度盛典是暴雪遊戲文化首次與旅遊文化深度結合的嘗試，屆時玩家不僅可以在烏鎮西柵景區領略數千年的烏鎮歷史文化，還可以感受到景區內濃濃的暴雪遊戲氛圍，「暴雪」降臨烏鎮，一起來玩吧。
