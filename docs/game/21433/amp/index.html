暴走大俠：修鍊過快怕“走火入魔”？玩家發現快速修鍊技巧

天天要聞
2019-12-20

氪金不暴走，大俠全靠肝，大家好，我是「爆肝大俠」小魚乾，歡迎來到由小魚乾開講的《暴走大俠新鮮事》。本期小魚乾將給大家帶來氪金兩千郵件、歷史私聊解決方法、修鍊小建議等新鮮事，話不多說，抓緊時間和小魚乾一起來看看吧！

氪金兩千後官方發來一封郵件，玩家看到表示再無下次！

不知道大家在暴走大俠里氪金多少了呢？小魚乾玩了3個月，也就氪了個首充，所以戰力提升的速度還是挺慢的，今天看了一下，自己的幫貢是31000，而少林寺升到滿級職位需要31500幫貢，加上500退幫貢費用，就是32000幫貢，看來明天就能拿到排山倒海了，還是挺期待的！

一名玩家氪金兩千之後，收到了一封官方發來的郵件，是氪金1000~2999元獲得的獎勵，裡面有15萬金葉子、一個2級玄黃玉、3個破鏡丹！

看到這個禮包，這名玩家感覺有些來氣，氪金兩千，就送這些「蚊子腿」？官方未免也太小氣了吧，感覺送過來就像在侮辱自己，還不如不送呢！這名玩家表示，自己絕對不會有下次氪金了！

其實這名玩家還是知足吧！大部分玩家所在的渠道服，連這樣的「蚊子腿」都沒有，有的玩家甚至氪了金還因為各種BUG，沒有收到活動道具。所以也有玩家說，暴走大俠是連最讓氪金大佬都受氣的遊戲了！

官方怕玩家修鍊過快「走火入魔」？玩家找出快速修鍊技巧！

大家在「突破境界」之後，有一大堆的修為，是怎麼處理的呢？是點擊「自動修鍊」之後，看著上面的境界亂跳，看自己運氣怎麼樣，還是先掛在那裡，過一段時間再回來看呢？

不少玩家都覺得需要等待它修鍊完真的麻煩，為什麼官方不能設置直接完成呢？非要浪費玩家們的時間，難道官方是怕一瞬間修鍊完成，人物會「走火入魔」嗎？

也有玩家找到了一個能快速修鍊完成的方法，但這就要考驗你的手速了，只需要瘋狂點擊「自動修鍊」，修鍊的速度就會快很多，因為每次點擊，會連續修鍊兩次，這個技巧對於那些屯了一大堆修為的玩家，就可以省些時間了。

登錄後一直顯示歷史私聊？解決方法公布，不建議嘗試！

不知道大家有沒有發現，在登錄暴走大俠之後，經常會顯示之前別的玩家發給你的私聊，雖然沒有什麼實際影響，但有些內容，有些玩家還是不願意看到的，小魚乾每次登錄，也會顯示自己的一名好友，在一個多月前發給自己的消息。

其實這個問題，是可以解決的，具體方法就是，點開世界頻道，再點擊右上角的菜單欄，就能看到給你發過私聊的玩家了，這時候，只需要長按這條私聊信息，就會彈出來一個對話框，問你是否刪除對話，並強調刪除後聊天記錄將會全部丟失！

但是，如果你有多條私聊信息，就需要全部刪除，才能解決登錄就出現歷史私聊信息的問題，只刪除第一條的話，還是會出現第二條歷史私聊，全部刪完才不會顯示。

所以，小魚乾是非常不推薦大家用這種方式來解決這個問題的，在官方預告的12月24日更新中，優化中修復了這一BUG，所以大家還是繼續等一等，等到24日，這個問題就能解決了。

不過話說回來，現在暴走大俠的世界聊天頻道，已經是極其冷清了，如果連這個私聊都刪光，在比較靠後的大區，估計都看不到信息了，感覺就像在玩單機一樣。
