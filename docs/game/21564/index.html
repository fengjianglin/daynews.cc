燃燒意志：“舊公告”開放新劇情，增加新角色？還要再添新玩法

2019-12-19 1,073條評論 2,517次閱讀 1104人點贊

諸君安好，霧夏菌報道。

本月18日晚，遊戲發出了新的公告與活動，只不過令人無語的是，公告的落款竟然是12月11日，成了作古的「舊公告」。而公告的內容卻都是最新的，具體一起來看一下。

1、新劇情

開放了最新的第50章劇情，以及相關的劇情回顧、支線劇情。經過了頂上戰爭，終於迎來了兩年後，後續也就要前往魚人島了。這裡吐槽一下支線劇情，有做的玩家都應該能感覺到，那個難度甚至比一些主線還要變態。

具體的地圖就是一個酒吧，一個白鬍子墳墓，要去緬懷一下這位舊時代的海上皇帝，以及火拳艾斯。

2、新玩法&新角色

新增的一個「大航海」的玩法，而且可能還是要後續慢慢開啟的，所以急不來。具體內容暫時不清楚，不過目前遊戲中的額外玩法已經足夠多了，血戰、海島爭霸、歷險。現在打開遊戲都頭皮發麻，肝不動。

新角色有點意外，按理說應該一個月一名新角色的。很少出現接連新SSR的情況（有連出過），但不要太激動，因為考慮到這份公告出自12月11日，如果不是數據錯誤，那麼不排除這裡的新角色就是指之前的黑鬍子，的確出過這種烏龍。再一個早前黑鬍子的限定活動，也是在12月11日公布的。

3、寶石等級上調

這個其實是針對一些氪金玩家的，因為大部分的普通玩家，可能根本湊不出幾套完整的10級寶石。而一些氪金玩家，可能會有富餘，這裡也就可以繼續提升等級，獲得更高的屬性，吊錘可憐的平民玩家。

而普通玩家的話，沒什麼影響，能收菜就繼續收菜，包括一些其他的玩法，有時間就刷一刷拿拿資源。因為現在的確是太肝了，有點讓人反胃了。

4、新活動

這玩意不用多提，一個是充值，一個是禮包，一個是幸運商店，再加上一個累計登錄。上次活動的累計登錄還沒結束，這裡又來。

幸運商店如果有一些打孔石，可以考慮入手。我記得以前還有能涌藍鑽購買的資源道具，現在不知道有沒有了。但陣法和戰艦這次會出現，到時候具體再說。

菌說

新劇情開放後，很多玩家又有得玩了。實際上還有很多人被卡在之前的劇情沒有通關，這個沒辦法，後面的劇情基本上默認地需要一些新限定，之前的紅髮就是如此，那麼這次的黑鬍子後續也有可能出個需要他的劇情。

至於其他的內容，有時間有興趣就玩一下，別太強求，搞得自己反而不舒服。好了，這一回合到此結束，感謝閱讀，拜拜了您嘞~
