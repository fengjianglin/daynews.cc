Clearlove宣告退役！新教練明凱帶領EDG能否打破8強魔咒？

天天要聞
2019-12-17

就在昨天轉會期結束的最後一天，所有LOL老玩家都得知了一個悲痛的信息，EDG官方宣布了一名選手的退役，他就是在LPL賽場征戰八年，七年前拿下了中國第一個英雄聯盟冠軍，六年前從WE轉會至EDG，四年前為LPL拿下了第一個官方賽事獎盃，三年前更是被打上了4396的烙印，在這之後兩年更是慢慢把C位傳給後輩，自己則是在台下看新人帶領EDG一次次衝擊世界舞台，他就是EDG的廠長Clearlove。

我見過諾言作為WE隊員 拿下IPL5世界冠軍，也見過廠長作為「導演」統治艾歐尼亞的時候，同樣也見過當年S5的MSI上，廠長的寡婦一戰封神，打破SKT不敗BO5神話，將FAKER百分百勝率妖姬斬落馬下的EDG盛世王朝。大家都以為廠長正值巔峰，之後的LPL更是最有希望的一年，然而由於EDG在LPL上的表現不盡人意，聯賽內部卻成績不錯。這形成了一個巨大的反差，被扣上了「內戰內行」的帽子。

雖然他內戰重拳出擊，外戰唯唯諾諾。但無法否認7醬一直都是我們LPL的靈魂人物，他可以說是此版本在LPL的活化石，見證了與培養了無數職業選手的成長，一手二級抓下打得Uzi束手無策，打野意識如同開了全圖透視，就連karsa和小兲都視廠長為偶像，因為廠長才喜歡上打野。什麼叫國際巨星？（戰術後仰）

儘管我廠子哥出色的地方特別的多，但他一直招人黑，無數經典的梗被網友們拿來借題發揮，如果英雄聯盟裡面所有梗共分10斗，那單拎一個廠長就能獨佔8斗，「廠長金句、4396、馬踏飛箭、7之意志、摔表位號、魚死網破、抑鬱刀法」多得數不勝數，甚至還有的噴子噴出了感情。我一直記得在一場Clearlove7的比賽中，有這麼一條彈幕「如果7醬真的退役了，那噴其他人感覺都沒那味。」

說實話按照七醬的年齡完全可以在最巔峰的時候選擇退役，那之後就不會有那麼多的梗了，更是不會成為全網黑的對象。他本可以功成身退，但是他沒有，他選擇了承受著質疑，失望與謾罵一直前行著，就是為了那個被無數黑子嘲笑的夢想。可惜的是他作為選手一次也沒有觸摸到那個想要得到獎盃，也許他看開了，S9的缺席一年把自己選手的位置留給新人，如今更是選擇了退役。

可能對於clearlove來說他的職業生涯已經結束了，但是對於明凱來說他的教練生涯才不過剛剛開始。他依然對英雄聯盟充滿了熱愛，即使現在自己操作有點跟不上了，但是他的頭腦還在，仍然在為自己的夢想而奮鬥著，只為了能站在比MSI更高的LPL舞台上拿下屬於他的榮耀。

說實話我也很期待廠長作為教練會有什麼樣的表現，他的意識以及理解的確是英雄聯盟職業選手中數一數二的，在男槍剛剛改版的時候他就開創了黑切打發，並且指出男槍打滿黑切全效果只要兩槍。如今他的確打不過很多打野，但是有廠長在的局對方打野想要GANK都會被他提前預知。所以作為教練來看，也許是clearlove離開最體面的選擇，也可以算是離夢想更近一步了，說不定還能打破EDG的8強魔咒。
