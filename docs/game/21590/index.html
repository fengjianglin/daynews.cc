新模式即將上線！全員擁有2條命，黃忠移動開大，東皇變身六道

2019-12-19 2,029條評論 3,170次閱讀 1359人點贊

本文原創首發於公眾號：電競怪客

hello，大家好，我是怪客君~

今天給大家帶來的是三個新消息，分別是王者榮耀有待上線的新功能，新遊戲模式以及最新官方答疑。答應我要看到最後哦！

有待上線的新功能

在王者榮耀上線的這四年時間，有許多的玩家或許因為連跪，一怒之下刪掉了遊戲，卻怎麼也逃不過真香定律。生氣了就卸載，反正還能下載回來。

不過在王者榮耀體驗服中，新推出了一項功能，如果你真的下定決定按下這個選擇，那麼你將和這個遊戲真正的一刀兩斷，這個功能就是「賬號註銷」。

賬戶註銷的功能應該不用我多解釋，大多數遊戲中都帶有這個功能，它意味著你可以將自己的遊戲賬號清空。平時說退游就卸載遊戲，但是下載回來還是一如既往的玩，而註銷賬號之後，即使再次登錄遊戲，之前在遊戲中的所有投入都將一筆勾銷，並且不會得到任何補償。

卸載王者榮耀，還可以隨時下載回來。但是一旦你註銷王者榮耀遊戲賬號，那麼就算你事後腸子都悔青了，你在遊戲中收集的金幣、鑽石、銘文和英雄，購買的各種皮膚等遊戲資產，以及王者印記，巔峰戰績等等遊戲記錄都將一去不復返。

試問誰忍心註銷自己的遊戲賬號呢？畢竟是自己的一份記憶，投入了那麼多的時間、精力和金錢，就算不玩遊戲，也確實不用走到註銷那樣的絕路。

不過「賬號註銷」是一個普遍的遊戲功能，王者榮耀還是有一些玩家正常性需求這種功能的，比如說很多玩家喜歡開小號，每次選區可能有好多個低等級的無用小號，平時不玩也沒時間玩。有註銷功能後，就可以把這些無用小號都刪除了。

最後註銷賬號需要用到身份證號，所以不用擔心借號刪號這種情況的發生，除非找你借號的是你的對象、家長這些特別熟的人。

目前這項功能只是在體驗服默默上線，連公告都沒有說，所以暫時不確定正式服是否會上線賬號註銷、刪號的功能。如果上線了，必然會有大量的玩家流失，但是作為一個正常的遊戲功能，我認為是有必要上線的。

新模式-技能強化

在體驗服12.17的更新中，還有一個新的遊戲模式也更新到了體驗服，但是由於新模式未正式開測，所以暫時無法正常試玩。這個新模式的規則簡單說就是這個模式下的英雄技能都會得到極大加強！

比如安琪拉的大招，強化後的效果是釋放三根火柱，且轉速更快，而其2技能則是增加兩個火球，且飛行速度更快。

再比如東皇太一，原本最多身上掛三個球，在強化模式下，可以有6個球！

這六個球同時砸向地面的樣子真的挺壯觀的，而且這控制範圍，估計有閃現也很難躲掉吧！這個控制強化的相當給力啊。東皇在這個模式下非常像《火影》裡面的六道仙人，太強了！

而且東皇的二技能數量也更多，範圍也更廣了。

是不是很有意思？再說一個，黃忠的12技能在強化模式下變化不大，但是他的大招在強化後，不僅範圍增大，還多了一個超級超級超級BUG的效果，那就是可以邊打炮邊移動，而且移動的速度還挺快的，這誰能躲掉啊？

目前新模式能強化技能的英雄一共有28個，除了以上幾位英雄，還有許多英雄的技能在強化之後都堪稱BT！比如太乙真人的大招，強化版的可以直接復活5個人！在這個模式下，選了太乙，不就等於團戰時己方全員擁有2條命？

這些英雄的技能會在原有的技能效果上進行升級加強，可以看出新模式的英雄技能進化後，不僅對抗性變強，可玩性也變得更高了。個人感覺這個模式還是很有意思的，滿足了各種奇葩腦洞，期待新模式能早點上線測試。

最新官方答疑

王者榮耀官方每周都會有一次老亞瑟答疑環節，是官方對於玩家反饋的一個正式的回復渠道。本周的老亞瑟答疑重點回應了關於武則天技能後搖的問題。

總結一下官方對於武則天這位英雄的態度：技能效果全面，強度符合預期，手感確實不好，但是不會調整。

有一說一，每個英雄都是獨具特色的，有強勢的地方也有弱勢的地方，每個英雄都需要保持在一定的強度，才能在這個遊戲中保持動態平衡。武則天之所以保持著靠前的勝率，本來擁有這個英雄的玩家就是少數，低出場率是導致高勝率的重要原因之一。

但是隨著版本的更替，各種花里胡哨的新英雄登場。武則天這個老牌法師，已經越來越不適合當前版本，笨重的手感讓她玩起來越來越吃力。所以我認為武則天也是需要得到更適應版本的更新的。

優化手感確實等於變相的加強了武則天，畢竟絲滑的手感才能帶來五殺的操作。所以就算要優化手感，那麼一定會砍其他的地方，這點犧牲在我看來是可以接受的。

在官方微博的下方，玩家也表達了自己的不滿。

不過現在官方已經明確說了不想動，奉勸各位也不要對反饋報太多的期望了，如果說對武則天這個英雄還有什麼期盼的話，那可能只剩下多出幾款皮膚，多優化模型特效，多加幾個標籤了。
