爐石傳說：HSR最新數據薩滿勝率暴跌！牧師成功擺脫墊底？

2019-12-23 566條評論 3,057次閱讀 320人點贊

前不久《爐石傳說》剛剛更新的平衡性補丁，其實大家都知道是針對新版本薩滿太強而來的。那麼補丁更新之後也是有很多玩家對於這次的補丁內容有著很多的評價，但是俗話說「不管是黑貓白貓，只要能抓到老鼠就是好貓。」，這次更新之後薩滿究竟有沒有受到影響呢？

數據最能反映變化了，自從平衡性補丁更新之後，薩滿的勝率從之前的穩坐第一變成了呈雪崩式下跌。因為在更新之前獵人就隱隱有要追上薩滿的趨勢了，於是在薩滿勝率下跌的時候獵人喜提勝率第一。並且有趣的是當薩滿的勝率下跌之後其他職業的勝率開始緩慢上漲，並且牧師的勝率也開始往上了甚至擺脫了最後一名。從HSR最新數據來看，每個職業的勝率還都是比較平衡的，這也就再次驗證了之前薩滿的超模。

這個數據的出爐可謂是狠狠的打了那些節奏黨的臉，畢竟昨天還有人在高談闊論這次的補丁什麼都沒改，今天薩滿勝率就跌落倒數第三了，小編就想問這個臉疼不疼。很多人都覺得削了533不夠還得削556狼魂，但是只有真正的薩滿玩家才能感受734這個點反場壓場有多強，更別說121削了，基本前期節奏全沒了。

或許目前來看雷克薩可能還是有點強勢，但是也不能滅了薩滿滿門呀對不對？如今的百花齊放已經比當初的一枝獨秀好太多了，對於更新後薩滿勝率暴跌你怎麼看？歡迎在評論區討論，想要了解更多的遊戲資訊那就關注小編吧！
