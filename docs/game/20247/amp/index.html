遊戲史最“快”的男人？連續20小時拿下50個遊戲，引來5萬人圍觀

天天要聞
22 小時 ago

大家好，這裡是正驚遊戲，我是你們的正驚小弟。

速通和收集白金獎盃一直以來都是PS4粉玩家熱衷於做的事情，而最近國外有個大神級玩家卻將二者結合，目標是20小時收集50個PS4白金獎盃。

這名被稱為「獎盃獵人」的玩家名為Brian，近日在國外視頻網站直播的時候就給自己定下了這個「小目標」。目標之初是24小時結束挑戰，結果只用了20小時就完成了。當時一共50000+人見證了記錄的誕生，不過小弟發現直播的錄播因為故障暫時無法觀看。

而且為了達到這個目標，Brian可以說是指定了非常詳細而且合理的計劃。其中就包括只限定PS4遊戲，允許在相同的獎盃在不同地區使用不同的獎盃列表，遊戲時間包括載入時間在內，但每隔幾小時會有不記錄在內的短暫休息時間等等。而且為了防止意外，遊戲清單中還預留了7個備選遊戲。

Brian的遊戲清單↑↑↑↑

最後這些遊戲平均的遊戲市場在28分48秒左右，根據Brian自己透露，這項挑戰中最困難的地方在於20小時保持不停的精神專註，而且還要配合肌肉記憶、興奮程度和熟練度等等才能完成這次挑戰，小弟感覺普通人應該很難做到！因此這項挑戰也創下了一個世界紀錄！

最後Brian也對獎盃的難易程度等等機製作出了一些點評，認為PS4在很多獎盃的獲取途徑設置的有些不合理。但是對於小弟這種上白金困難群眾來說，基本上沒有區別！那麼網友對於這個速度與質量結合的大神級玩家創下的成就又是怎麼看的呢？

——我5分鐘兩次我驕傲了嗎？

——這大概就是飯吃多了吧

——第一個想到的是迪拜黑眼圈肝帝

——好遊戲值得白金，白金過程就是一種樂趣。像MHW這種大小金冠杯堅決抵制！

——一個都沒白的我_(:з」∠)_

——原諒我見識短淺，這些遊戲我居然一個都不認識

看到這裡小弟跟你講，看到這裡雖然小弟覺得我上我也行，但是看到需要50款PS4遊戲，小弟看了看自己接近枯竭的遊戲庫想想還是好好工作比較現實！
