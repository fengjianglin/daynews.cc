國內二次元手游這個市場，時不時就會出個好遊戲，前有《崩壞3》，後有《明日方舟》。而最近，能讓這兩個熱門遊戲都緊張起來的新品，無疑要屬《戰雙帕彌什》了。三消+動作的玩法有新意，畫面精緻戰鬥流暢，在連續幾次測試後都維持著9分高分和極高熱度。就在昨天（6日），這款遊戲終於正式上線了。

原本大家都期待著帕彌什和崩3直面競爭天雷動地火，期待著又一款經典二次元手游的誕生，然而僅僅一天，帕彌什運營就用自己匪夷所思的烏龍事故，及後續更加匪夷所思的處理方式，讓自己的口碑暴跌至不足5分，堪稱2019年最經典的手游運營事件。

1、 手抖發黑卡，天降1萬！我懷疑這運營是個萌新

事件的一開始，是安卓全服玩家都突然收到了一封郵件，贈送1萬黑卡（價值近1000rmb）。在首充648元，獎勵才1萬左右黑卡的背景下，這天降資源就是白給錢！玩家當然要拿走！然後，伺服器就開始了暫停維護。

有小道消息說這是萌新運營給內部人員發資源，結果手抖發給了全服；也有小道消息說這是給全服1000福利，結果手抖多按了個0。總是，大錯已成，只能趕緊拔伺服器插頭，進行事後補救。

2、 回檔時間錯，再度天降1萬！運營腦子丟到太平洋了？

接下來，官方開始了第二輪神操作。一般來說這種錯誤就兩個解決方式，要麼硬著頭皮一萬黑卡全送了，後果是會引發氪金大佬不滿，以及遊戲經濟系統可能崩盤；要麼就全部回檔到發送郵件前，再給出一點小補償，後果是會引起普通玩家口碑下滑。帕彌什選擇了第二種。

原本很簡單的處理方式，運營的腦子卻再度消失，竟然搞錯了回檔時間！回檔到了剛剛發完郵件的那一刻，於是，全服玩家再度天降1萬黑卡。運營，就又拔了伺服器插頭，再度暫停維護。

3、 再度回檔再度錯，隨機回檔讓氪金大佬充值慘遭清零

這下，錯誤資源的金額擴大到近2k元，還能怎麼辦，繼續回檔唄。官方又繼續神仙操作，竟然選擇了隨機回檔？根據分析，造成隨機回檔的原因，可能是程序員沒有檢測是哪些玩家領了郵件內的資源，而是去檢測了抽卡情況和黑卡下降情況，有抽過卡消耗過黑卡的玩家，就回檔，沒有的就維持原狀。

但不少領了天降2萬資源的玩家還沒開始花錢，於是沒有被回檔；而很多氪金大佬早早就去抽了卡，於是被回檔了，充值直接白給，是個人都不能忍啊！

4、 緊急救場說錯話，再度讓玩家不滿，好在最終補償還算香

事態發展到現在已經幾乎無法收拾，官方也緊急在B站開直播和玩家溝通後續補償。可官方的一句話又讓玩家炸了，”就是把你們當人看才會出來直播啊”。玩家？？？

最終，官方給出了”自選S角色+限定皮膚一件”的補償措施，還算不錯，帕彌什的評分也從低谷4.7逐步回升到了目前5.5

5、 各路友商紛紛發來賀電，並表示我們送的1萬不回檔

這樣的運營錯誤，友商怎麼能放過呢，一大波友商的1萬正在趕來的路上。
