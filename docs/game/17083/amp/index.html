巨龍降臨：術士手牌上限增加！玩家：老師一定給暴雪寫信了

天天要聞
2019-12-14

爐石傳說新版本巨龍降臨開啟後，瞬間成為了最熱門的話題。新版本也極具特色，各式各樣的隨從讓整個對局充滿了不確定性。除了新的卡牌，新英雄死亡之翼和獵人新英雄風行者也將加入新版本。不少小夥伴都覺得戰士在這個新版本會崛起成為天梯的強者，也有獵人的忠實玩家覺得新版本配上獵人新英雄才是最強的。

新版本剛剛開啟，各類卡牌構築也不斷出現，究竟誰是天梯最強的英雄還不確定，但是不少小夥伴驚奇地發現，術士好像又被偷偷加強了。瓦迪瑞斯•邪噬，戰吼：將你的手牌上限提高至12張，抽四張牌。玩家紛紛調侃：異靈術老師一定給暴雪寫信了！

這樣的改動不可以說是不大，除了手牌上限可以提高，術士的龍屬性隨從牌也是非常強。例如扭曲巨龍澤拉庫，效果是每當你的英雄受到傷害，就召喚一條6/6的巨龍。這簡直就是一張嘲諷牆。它佔在場上會讓對手不得不想辦法先處理掉它，否則生出來的巨龍真的是源源不斷。

異靈術老師有沒有給暴雪寫信這個還不清楚，但是新版本的福利絕對是史上最超值的，這是毫無疑問的。登錄就是全部5張迦拉克隆傳說英雄卡牌，2張迦拉克隆之盾隨從牌，3包巨龍降臨卡牌包！這樣的福利真的是前所未有，這5張英雄卡牌是在收藏界面打開的，小夥伴們記得登錄領取。畢竟可以算是新版本非常核心的卡牌了。

雖說術士在新版本被格外關注了，但是戰士的新英雄死亡之翼應該是其他職業比不了的。死亡之翼也非常符合新版本的特性，從音效到質感都不輸任何人。12月11日，獵人新英雄希爾瓦娜斯•風行者和卡背圖案也將限時加入商店。往往在新版本開始的時候入手這樣的英雄是最超值的，喜歡的小夥伴千萬不要錯過這次的機會。

由於這次新版本的到來，商店也進行了一些靈活的改動。比如你可以通過暴雪遊戲點數或者現金購買符文石，符文石可以用來購買爐石中非卡牌包類商品，包括英雄、冒險模式或競技模式入場券。同時奧術寶珠可以通過轉換符文石獲取。1枚符文石可以轉換10顆奧術寶珠。你可以用奧術寶珠來購買各類卡牌包以及合集。

新版本的特點就是福利一定要是史上最超值的，所以大家登錄爐石進入商店，牧師英雄泰蘭德免費送給大家！薅羊毛的機會已經到了，小夥伴們趕快進入遊戲領取自己的福利吧！
