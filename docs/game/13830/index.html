「DOTA2」WESG：iG中期掌控雷電 2-0 KG晉級下一輪

2019-12-12 560條評論 2,038次閱讀 521人點贊

超凡電競12月12日訊 WESG大中華區總決賽 淘汰賽敗者組第三輪 iG 2-0 KG

第一局iG獲勝

雙方陣容如下：

開局不到5min，iG便取得了三路優，下路中路更是頻繁得手，6miniG經濟領先3K，人頭6-1領先。

8min，水人來到野區配合火女將陳帶走，iG經濟差拉開到了5K。

10min，下路KG集合五人擊殺iG雙醬油，水人則是一個人開了上路前哨。11min，到對面野區刷錢的小小被KG三人擊殺。

13min，iG全員來到KG下路野區，KG雙醬油陣亡後iG順勢拆掉下路一塔。

15min，KG上路開霧蹲開前哨的猴子，雖然冰龍放了一個不錯的大將斧王率先擊殺，但緊接著交過一輪技能的KG只有逃跑的命，iG追死四人後轉而控盾，經濟差來到了1w2。

20min，下路帶線的猴子被斧王和莉娜擊殺，同時上路野區做眼的藍胖被擊殺，iG拆掉下路二塔。22min，兩邊下路開團，iG水人更新虛空刀後擊殺猴子、火貓、船長。

之後雖然想買活守高，但無奈經濟差實在是太大了，KG只能敲出GG。

第二局iG獲勝

雙方陣容如下

​5min，馬爾斯到六後越塔強殺沉默。9min，上路老鹿配合全能擊殺了小黑，對線期結束後兩邊人頭KG4-3領先，但經濟卻是iG領先不到1K。

12min，KG來中抓人，馬爾斯先手大留到沉默，但沉默馬上反開大配合隊友反打，puck陣亡後選擇買活，但己方的大牛也陣亡，KG沒有輸出後選擇撤退。

16min，雙方中路爆發團戰，KG三核沖臉直接秒掉沉默和人馬，隨後小y和藍胖也被追死，KG打了iG波4人團滅。

18min，中路帶線的沉默再次被馬爾斯抓到，蹲人的藍胖也被擊殺，這段時間KG的戰鬥力卻是強。

19min，KG在iG高地下擊殺沉默和人馬，隨後全能買活，KG中路上高又擊殺小y，但殘血的老鹿則是被防禦塔砸死。

24min,iG在抓到puck後成功控盾。局勢又被iG緩了下來。30min,iG上路抓到機會在哨塔處團滅KG四人。這波打完iG反超了3K經濟。

32min，上路的老鹿和大牛再度被擊殺。34min，iG上路上高，在擊殺老鹿、puck、全能、馬爾斯後KG買活死，最終KG無奈敲出GG，恭喜iG晉級下一輪。
