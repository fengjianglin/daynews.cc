水晶商店大更新：榮耀水晶皮膚和王者水晶皮膚即將同時更新上線

2019-12-25 40條評論 1,121次閱讀 133人點贊

大家好，我是王者小喬妹，關注我，為您分享最新王者資訊~

據官方皮膚新爆料，兩款新皮膚即將更新至水晶商店，分別是：榮耀典藏皮膚——活力突擊-程咬金，以及勇者皮膚——次元突破-哪吒。

一款榮耀水晶皮膚，一款王者水晶皮膚，將在2020年一月中旬上線。

活力突擊——程咬金

程咬金一直是峽谷中邊路霸主，大招超級強力的恢復能力，以及血量越低傷害越高的被動，兩者一起配合，使得程咬金又肉又有輸出，噁心對手一條街。

這次居然優先李白上線榮耀典藏皮膚，也可見程咬金的火爆程度在峽谷也非同一般！

那小夥伴和小喬妹一起來看看這款榮耀典藏皮膚究竟有什麼炫酷特效吧！

局內展示

局內展示里程咬金由一個肥胖大叔不斷鍛煉成為一代型男，挺符合程咬金一貫騷騷的樣子。

一技能-爆裂雙斧

程咬金扛著他的運動器材從天而降，在地上砸出一個大坑，一看就感覺傷害很足的樣子。

二技能-激熱迴旋

程咬金拿著運動器材在「愛的魔力轉圈圈」，周圍擦出火紅色火光特效，感覺範圍增加了一倍。

三技能-正義潛能

大招開啟後，一個沐浴的水蓮蓬打開，程咬金也是由肌肉猛男穿回了肥宅大衣，這特效感十足啊。

也特別符合程咬金這個搞笑的角色！

次元突破-哪吒

哪吒這款皮膚一開看到模型以為也是一款榮耀水晶皮膚，有點酷似孫悟空-全息碎影，非常炫酷的新科技機甲類皮膚！

仔細查看品質後，是勇者皮膚，不過是王者水晶兌換哦~

大家的王者水晶存了嗎？鑽石還夠不夠？這款哪吒的皮膚可千萬不能錯過哦！

特效上肯定是和上面那款程咬金皮膚比不了，畢竟人家是榮耀水晶皮膚，這款皮膚沒有什麼特效，但是外觀模型上絕對秒殺大多數皮膚的！所以非常值得用王者水晶兌換。
