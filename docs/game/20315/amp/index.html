除了騰訊網易的大佬演講外，2019遊戲產業年會還有哪些看點？

天天要聞
2019-12-17

12月7日至12月19日，由中國音像與數字出版協會、海南省新聞出版局、海南省工業和信息化廳主辦，中國音數協遊戲工委、海南生態軟體園承辦，並由國家新聞出版署主管的2019年度中國遊戲產業年會將在海南海口市魯能希爾頓酒店舉辦。

中國遊戲產業年會與面向玩家的ChinaJoy一起被並稱為中國遊戲領域兩個最大的活動，該會議於2004年首次舉辦以來，都充當著全中國遊戲行業總結經驗、分析現狀、規劃未來的交流與合作的首要平台的角色。因此每年產業年會都少不了大佬們的演講，今年亦是如此。

2019遊戲產業年會嘉賓名單

那麼除了大佬演講之外，此次年會還有哪些看點呢？那就要從主辦方已公示的大會信息一一說起了。

2019遊戲產業年會

據2019年度中國遊戲產業年會官網顯示，本次大會除了產業年會主會程和中國年度遊戲十強頒獎典禮以外，還將由功能遊戲探索分論壇、海南遊戲行業協會第一屆成員大會、未成年人守護分論壇、遊戲產業國際化論壇、電子競技發展主題論壇等活動組成，詳細時間表如下圖所示。

2019遊戲產業年會日程表

從大會日程表可以看出，功能遊戲探索分論壇與海南遊戲行業協會第一屆成員大會、未成年人守護分論壇和遊戲產業國際化論壇以及電子競技發展主題論壇之間存在開展時間重疊的情況，那麼參會人員以及媒體們則需要根據需求，參加不同的主題論壇。

在功能遊戲探索論壇中，包括騰訊遊戲副總裁張巍、故宮新媒體團隊項目負責人盛馨藝、盛趣遊戲功能遊戲製作人謝剛在內的8位嘉賓，將分別從遊戲行業、數字化與傳統文化、功能性遊戲開發經驗分享、中國高校遊戲教育的思考等方面發表演講。

功能遊戲探索論壇日程

海南遊戲行業協會首屆成員大會將由宣讀海南省遊戲行業協會理事會成員票選結果、介紹人工智慧遊戲審讀及運營監控系統、海南網遊審核監管平台授牌儀式、科技與遊戲內容監相關管圓桌對話等活動組成。

海南遊戲行業協會第一屆成員大會日程

在遊戲產業國際化論壇上，來自騰訊的使命召喚手游負責人姚遠、QQ飛車手游製作人劉晶，以及完美世界副總裁伊迪、上海莉莉絲髮行業務負責人張子龍等7位嘉賓，將各自分享旗下遊戲在海外發行的經驗。

遊戲產業國際化論壇日程

電子競技發展論壇方面，騰訊電競品牌總監雲潔、FPX電子競技俱樂部總經理李淳、中國傳媒大學遊戲專業教師張姚弓等嘉賓，將就電子競技行業以及國內外電子競技教育的發展發表演講。

下午15:30分，上海網映文化CEO林雨新、成都龍淵網路CEO李龍飛、上海樂競文化COO鄭奪、上海度勢體育CEO陳維力以及完美世界公共事務總監喬婷婷，5嘉賓將展開關於「文化or商業 破局中國電競出海」話題的圓桌對話。

遊戲產業國際化論壇日程

在未成年人守護論壇上，將由中國音數協遊戲工委第一副理事長主任委員張毅君致辭，隨後由騰訊互娛、暢遊、創夢天地三家公司就未成年人守護話題發言。下午15:50分，包括暢遊、創夢天地、第七大道、網易、盛趣、樂元素、多益、中手游、凱撒文化、完美世界、騰訊、掌趣在內的遊戲公司為響應新規，集體發表《中國遊戲行業未成年人守護宣言》。

行業集體宣言結束後，《好媽媽勝過好老師》作者尹建莉、湖北省恩施市屯堡鄉羅針田小學副校長張玲、聯合國兒童駐華辦兒童保護官員蘇文穎、騰訊遊戲客服負責人張顥、哈佛爸爸，《世界名校學生家庭教育手記》主編王建軍將就「未成年人守護，家校如何協力共建」話題展開圓桌對話。

未成年人守護論壇日程

12月18日的各大主題論壇結束後，次日上午便是2019年度中國遊戲產業年會大會，年會結束後，下午便是2019年度中國「遊戲十強」頒獎典禮。

中國遊戲十強投票界面

從本年度大會日程可以看出，相比往年來說，2019年遊戲行業比較重視海外遊戲市場發展。互聯網科技在不斷進步，遊戲企業們也隨之開始注重功能性遊戲的探索，當然，電子競技行業也是無法繞開的話題。
