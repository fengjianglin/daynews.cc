懷舊服：分享昨天半小時拿厄運鑰匙+法師大水流程——獻給法師們

天天要聞
1 週 ago

大家好，我是T哥在三亞，今天和大家分享一下昨天我剛到60的法師小號做大水任務+拿厄運之槌月牙鑰匙半小時搞定的流程，方便新到60級卻苦於找不到隊伍刷厄運之槌的新手法師們。

58級左右，就可以著手做MC門和黑龍門任務了，做完基本也就60級了。在此過程中，我深深地感到喝水過程的痛苦！魔法蘇打水喝一瓶30秒藍還差一點滿，太慢了！因此必須先去做大水任務了！在網上查了下流程，首先必須60級才能接大水的任務，然後要先去厄運北的圖書館接任務，在此過程中注意一點不需要打怪，但需要厄運之槌的月牙鑰匙才能開厄運北副本外和圖書館兩個門，鑰匙需要去厄運東追逐一隻副本門口的小鬼普希林打上好多怪才能拿到（隊伍所有人都能拿到），此過程相信大家都熟悉，不再贅述。

這就給我們法師造成了困擾，難道我要先打一次厄運東拿了鑰匙，再接任務，再去厄運東打水元素BOSS？那樣實在有點漫長。我是這樣做的：單槍匹馬來到厄運之槌後我直奔北門，並一邊跑一邊喊5G求開圖書館的門，只喊了一聲就很幸運地被一位牧師組進了隊伍，並且當我跑到厄運北門的時候我被神眷顧了一下，有人開了副本外邊的門，而且竟然是聯盟。區區一個盜賊和一個獵人怎麼可能留得住法爺我呢？冰箱，閃現，進本。在此普及一下，厄運之槌原為上層精靈引以為傲的城市——埃雷薩拉斯，曾經是保護艾薩拉女王珍貴的卡多雷奧術秘密的殿堂；在第一次永恆之井爆炸的世界級大災變中毀滅，現在的廢墟被亡靈、薩特和食人魔佔據了！

如果各位兄台沒有被神眷顧，可以多喊一會並在進隊之後讓別人幫開副本外的門。大家也知道現在獵人，法師在厄運北的那是相當多，當然PVP也相當激烈，此處耽誤時間多少純粹看臉。進本後，看到這位可愛的牧師正在帶一個50多級的號在厄運北干不可描述的什麼事^_^，並且很慷慨的幫我開了圖書館的門沒有收錢，感謝並加了好友之後，我走進了圖書館漫長的通道，來到辛德加這個古老的部族NPC處接了大水任務「久違的法師」——這些愉快的經歷總是讓我們越來越喜歡魔獸世界了對不？

接了任務後，我就開始喊厄運東打月牙鑰匙+法師大水任務開組，畢竟這樣喊會比較容易組到人，然而事實非常打臉，組了個法師和獵人後又喊了十分鐘都組不到MT和奶媽，畢竟這兒不是三大副本也不是MC黑龍門任務必去的黑石深淵，相信各位小夥伴都有這樣的經歷，懷舊服的T和N非常難組！所以才有MT奶媽賣身一說。就在這時我再次被神眷顧！一位德魯伊M我說組不到TN照樣可以拿鑰匙，並且只收10G，路上還能順便做任務收回這10G的成本。將信將疑的我選擇了相信他。

他換號之後組進隊伍發現是潛行者，沒錯當你熟悉之後，厄運東可以說就是潛行者的後花園！他很專業的說需要5-8分鐘時間進副本準備，我和隊伍里剩餘的一個法師則聽從安排跑到莫沙徹營地北邊一點的高台上接任務——相信剛到60的小夥伴基本都不會注意到這個任務的：在一個高台上有一個暗夜精靈的NPC，並且高台的下面就是厄運東副本的入口（需要月牙鑰匙才能進）！任務獎勵有一把價值4G多的雙手劍，任務獎勵還有4G多。

我們一邊愉快的聊著天，一邊等盜賊同志打通關係，並在此過程中又有一位法爺和一位法奶奶進組，當門打開之後我們愉快的踏上了征程。進副本後繞開花朵怪，擊殺掉隱身的巡邏怪（強力一點的賊可以單殺），我們一路暢通無阻的上了2樓，那個可愛的小鬼已經在這裡等我們了！交了保護費10G後，盜賊對話激活小鬼，刷出了一堆小小鬼我們A完之後還可以恢復好，再開小鬼BOSS。我們每個人都愉快地拾取了鑰匙。

小鬼普希林掉落列表

由於協議內容並不包含大水任務，而且盜賊對這個BOSS也是束手無策的，我只得硬著頭皮試探性的問了一句，各位可以幫我殺下水元素做大水任務么？這位賊兄想都沒想，直接回復：「我想一下怎麼去水元素那裡」然後就帶著我們輕車熟路來到了目的地。在清旁邊的一朵花的時候我不小心竟然被它奧爆給爆死了——大家小心那朵花，傷害比較高要遠距離打。幾位可愛的小夥伴也耐心的等我跑屍，賊兄也熱心的到副本門口再次接我進來——此時已是後半夜2點多馬上3點的時間了。

我們幾個簡單商量了一下，決定採取術士單刷的辦法來擊殺。我們3法一起火球術起手，愉快的在柱子後跳起了華爾茲！事實證明，法師單殺水元素也能實現，只是比術士要累很多，每次BOSS過來只能讀一個灼燒+火焰衝擊，快出射程時再來個奧術飛彈！所以火法應該會殺得快一些。水元素在我們3法1賊的組合下沒堅持過2分鐘就被KO了！我們還順路去把女術士BOSS給做了，又順便完成了一個任務！有了鑰匙的我感謝了各位之後愉快地去交了任務。在此再次感謝這位為人民服務負責任公道價格掙辛苦錢的盜賊兄！

所以想做大水+拿鑰匙的新手法師們可以參考此經驗在主城喊一喊，兩三個人輕鬆半小時搞定。喜歡請關注，有用就收藏，歡迎評論關注收藏轉發！我是T哥在三亞，暴雪二十年鐵粉！
