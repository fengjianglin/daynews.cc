lol手游什麼時候上線？看這篇你就心裡有數了

2019-12-14 266條評論 1,778次閱讀 537人點贊

LOL手游什時候上線？有人說是2019年12月18日；有人說是2020年6月份之前；有人說是2020年3月份到6月份之間。那麼到底是LOL手游什麼時候上線呢？這個問題怕是只有開發和運營團隊知道了，但是我們LOL手游的期待者卻是可以一窺端倪，且聽我的分析。

一、LOL手游的消息

2019年10月6日，LOL十周年慶典時候公布LOL將發布手游，騰訊並開啟了在線預約；在此發布了試玩的英雄聯盟手游視頻，引發廣大moba遊戲愛好者的狂呼。

2019年12月11日，LOL手游將有第二批新英雄加入內測，共有27名新英雄。援引LoLWildRiftBRA在社交網站上的信息，LOL手游將有新一批英雄加入《英雄聯盟》手游的測試，可以看到包括錘石、皇子等比較靠意識和操作的英雄。

這其中已經經歷了兩個月多有餘，LOL手游還想不想出呢？等的眾多玩家花都謝了，但是想要知道LOL手游什麼時候出還是要站在手游開發商和運營商的角度來思考下。

下部分是新增手游內測英雄

二、LOL手游運營方的想法

LOL的直系所屬是拳頭老大；最終老大兼運營商是騰訊。那麼是誰來開發LOL手游呢？

國服開發有騰訊下屬的光子工作室來開發。端游和國際服則是由拳頭負責。稍微介紹一下，騰訊旗下有五大遊戲開發工作室，分別為天美、光子、北極光、魔方和波士頓工作室。其中波士頓工作室在是騰訊的海外工作室，所以在國內知名度不高。

那麼決定LOL手游什麼時候上線的只有兩個因素了，一是開發團隊光子工作室；二是騰訊遊戲運營策略。

1、開發因素

根據官方曝光視頻、官方內測視頻、流出自測視頻各個方面的視頻可以看到，遊戲的開發已經算是完成度接近99%，剩下的怕是英雄內部英雄屬性和細節微調了。

畢竟光子工作室的也不想砸自己的招牌，這是超越天美工作室的一個重要時機嘛！所以內測的完善也是需要不斷精進的，尤其是最近新加入的第二批的英雄試玩名單，也可以推斷LOL手游的整體完善度已經很高了，不然也不會繼續加入新的英雄測試。

有人要問了，既然已經完善了為什麼不趕快公測讓我們這些遊戲玩家趕快玩，這就要問騰訊運營了。

2、騰訊運營策略

那麼騰訊的想法是什麼呢？騰訊和拳頭因為什麼樣的判斷決定要推出LOL手游呢？

先說第二個問題。騰訊和拳頭無疑是達成了一致，意識到手游的流量已經成為遊戲流量中的第一位。當然一開始騰訊是最先意識到的，但是拳頭沒有把這件事放到心上，這裡暫未不管他是因拳頭的遊戲運營重心也好，還是遊戲的偏見也好。但是最終騰訊和拳頭達成了一致，想要讓LOL進軍手游界。關於為什麼要進軍手游界，就是說一點就夠了，哪怕端游再好玩、再沉浸，無可置疑總體流量和商機還是在手游上，商人聞利而動，你不動自有人動，這個道理大家肯定很清楚。

再說第一個問題。騰訊推出LOL手游的目的是什麼？是為了滿足國內廣大愛好LOL但沒有時間玩的遊戲用戶嗎？兩個原因，一是強化品牌。二是盈利。當然有時候這兩個事情不是衝突的。強化品牌？盈利？國內已經有了一個王者榮耀，而 且是自己的，這是鬧哪樣啊，要砸自己的牌子嗎？但這可不是競爭勝出的玩法了，王者榮耀已經是現象級了，LOL手游根正苗紅，這兩個勢必要爭奪流量啊，但是不出嗎？這又不是騰訊的尿性，肯定要出，但是有一些其他的目的。

那就是騰訊想借LOL手游出海，佔領國外手游moba市場。這次是騰訊的LOL手游最重要的使命。當然這也是給他的目的相關——品牌和盈利。那麼現在就已經清晰了。既然是為了國際市場，那麼國內市場的開發是什麼樣的情況呢？

三、最終會決定什麼時候推出呢？

LOL手游擔負著騰訊國際手游市場佔領的任務，那麼其爭奪國內市場的任務就沒有那麼急迫了，但是為了可以更好地發展和手游相關的電競等周邊事情，肯定也會把國內市場做大的。

或許可以猜想一下：王者榮耀和LOL手游就像DOTA和LOL的關係一樣

那麼對於國內市場的重視也是不容小噓，那麼國內市場要想佔據更多的人的時間並且打開知名度，會是怎麼樣的呢？

大膽推斷一下：

春節是中國人特有的傳統，這個時節意味著團聚、玩耍、休息。大家一定知道很多APP的起來都是極大的借春節的勢頭起來的，所以推斷，騰訊肯定會在春節前推出一款遊戲。這個遊戲是什麼？有兩個遊戲比較知名度高一個是DNF和LOL手游。而最看重的和擁有當前知名度的只有LOL手游，古話說的好「一鼓作氣」，相信騰訊會做出有利的決定來，除非開發會有問題

所以為了更多人了解增強人流量，另外已經有了兩個月的預約造勢了，有極大的概率是LOL手游會在春節前推出。
