《迸發2》（機甲狂潮2）全新裝甲套裝擴展包上線

天天要聞
2019-12-24

　　Game Source Entertainment (GSE) 宣佈，由Deck13打造、Focus Home Interactive發行的動作角色扮演遊戲《機甲狂潮2》(The Surge 2)為玩家推出的第二彈新DLC內容正式上線。最新推出的「Jericho』s Legacy Gear Pack」裝甲套裝擴展包，為玩家帶來3款全新裝甲套裝。「Jericho』s Legacy Gear Pack」已在PlayStation®4及Xbox One上進行配信，持有Season Pass(季票)的玩家不用再額外付費購買「Jericho』s Legacy Gear Pack」裝甲套裝擴展包，未有持有季票的玩家可以HK$39購入「Jericho』s Legacy Gear Pack」裝甲套裝擴展包。

　　全新「Jericho』s Legacy Gear Pack」裝甲套裝擴展包為玩家帶來的3款新式裝甲套裝，能為刺激的戰鬥增添爽快的元素，而且因應全新裝甲的特性，也為玩家們開拓了全新的戰鬥方法，能為玩家們建造全新的遊戲體驗。

　　「Jericho』s Legacy Gear Pack」裝甲套裝擴展包追加的裝備，包括有可以將敵人推開的「BEAST Tank Buster」(BEAST坦克殺手)、為水平攻擊增加攻擊距離的「Kyoko Hacker Suit」(京子駭客衣)，以及鼓勵「先擋後攻」的作戰方式的「Jericho Firetails Mascot」(Jericho Firetails吉祥物)。如果想獲得這些新裝甲，便要在遊戲內先找出裝備有這些新裝甲的敵人，只要將他們打敗，就可以立即獲得它們。

　　這次「Jericho』s Legacy Gear Pack」裝甲套裝擴展包是屬於季票當中的第二個包含項目，所以持有季票的玩家都不需要額外購買，便可獲得新發表的擴展包內容。未購買季票的玩家亦可以HK$39獨立購買「Jericho』s Legacy Gear Pack」裝甲套裝擴展包。

　　經過本次的更新後，《機甲狂潮2》將會在2020年1月迎來一個超大型DLC「Kraken」，是一個強化故事內容的擴充資料。內裡將包含有新的故事線、全新的敵人和BOSS，這些全新的東西都會在「Karken」的航空母艦上發生及出現。持有季票的玩家可以比其他玩家提早48小時率先體驗這個豐富的擴展內容。

　　《機甲狂潮2》已於2019年9月24日於PlayStation®4及Xbox One平台上推出盒裝版，遊戲以英語語音進行，並將附有中英文字幕。未持有季票的玩家可以$39購入「Jericho』s Legacy Gear Pack」裝甲套裝擴展包;或以$149購入季票，不單單可即時免費得到「Jericho』s Legacy Gear Pack」裝甲套裝擴展包，連同早前推出了的「大眾公敵」擴充內容，以及明年推出的「Karken」都會包含在季票之內。《機甲狂潮2》的季票現在可於PlayStation Store及Microsoft Store內購買。
