Falcom社長回應創之軌跡跨平台 繼續支持NS及PC

天天要聞
2019-12-19

在昨天下午舉行的股東會議上，日本遊戲公司Falcom正式發布了軌跡系列的正統續作《英雄傳說：創之軌跡》，並確定本作會在2020年夏季登陸索尼的PS4遊戲機平台。在這次大會上，並非僅有針對PS4平台的消息和情報。

除了發布面向PS4平台的兩款畫質增強遊戲，《零之軌跡·改》和《碧之軌跡·改》之外，Falcom社長近藤季洋在回答記者提問時，也提到了Switch和PC平台「軌跡」系列遊戲的想法。

有記者提問，在發表《英雄傳說：創之軌跡》時，畫面並沒有明確表明該遊戲的發售平台，這是否表示本作將是一款多平台遊戲。另外，雖然伊蘇和軌跡系列目前都以PS為主要平台發售，不過隨著Switch銷量越來越高，而且PS5的裝機量還需相當時間提升。接下來Falcom如何選擇遊戲平台？

對於以上問題，Falcom社長近藤季洋表示，《英雄傳說：創之軌跡》將於2020年夏季登陸PS4。過去Falcom一直專註於PS平台。也有討論推向任天堂Switch平台。隨著Switch在全球持續熱賣，Falcom也收到了「希望在PS4以外遊玩」的呼聲。

考慮到這一點，Falcom已經同日本一合作，為Switch推出了《伊蘇8》。Falcom也收到了許多來自STEAM平台的需求。雖然在日本核心玩家都專註於PS4，不過在其他地區則沒有這個問題，Switch和PS4能夠處於同一水平。

考慮到遊戲陣容，Falcom社長認為Switch將會繼續熱賣。因此該公司將繼續評估市場、收集數據，以確定後續支持平台。

Falcom已經確定與日本一再次合作，為Switch玩家帶來《閃之軌跡3》，該遊戲將於2020年春季發售。
