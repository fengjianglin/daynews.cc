回憶小時候，VR模式街機遊戲來了

2019-12-16 2,004條評論 3,436次閱讀 1719人點贊

12月15日青亭網報道，自從VR進入大眾視野，大家就一直在玩法上尋求突破，甚至也曾嘗試和接機進行聯動，當然也有不少有趣的案例。

近日，此前推出黑白風格驚悚類VR遊戲《Wilson』s Heart》的開發商Twisted Pixel Games，在The Game Awards上宣布將推出一款VR版的《龍虎之拳》（Path of the Warrior），並且為Oculus獨佔。

據了解，在這款遊戲中你將通過位於經典街機遊戲的場景中，尋找當年在遊戲廳的感覺，遊戲中你以第一人稱視角開啟遊戲，就像此前的2D場景變為三維場景一樣，你可以通過拳頭擊打npc，一路過關斬將。

Oculus Studios執行製片人Mike Doran表示：我們最初印象的接近是《雙截龍》《忍者神龜》等，後來才有《狂怒》、《快打旋風》等等經典作品。當我們有機會和Twisted Pixel進行和合作時，我們首先想到了街機風格的遊戲。

的確，VR版《龍虎之拳》有些懷舊的意思，但是整體也為VR模式進行了優化，玩法上也比較有意思。目前，該作已經登陸Oculus Quest和Oculus Rift平台。
