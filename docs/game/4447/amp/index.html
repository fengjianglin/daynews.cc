S9冠軍中野翻車，Faker爆冷被淘汰，Uzi的對手卻被解說現場開噴

2019-12-07

今天12月7號英雄聯盟全明星SOLO賽十六強，首先是FPX小天對抗G2打野Jankos，重現S9總決賽！小天在賽場上的打野節奏完爆對手，打得Jankos唯唯諾諾，昨天盲僧SOLO踢翻小花生，今天還能再暴打Jankos嗎？

前期小天阿卡麗沒兵權，但塔下補刀也穩住了自己的發育。Jankos諾手很急，總想塔前消耗，卻被防禦塔打了兩下。於是小天阿卡麗果斷大招出手，絲血擊殺諾手，成功拿下一血。

然而Jankos諾手打得很拼，沒買防裝，選擇買憤怒合劑撐起極限爆發。在縮圈決戰開始的瞬間，小天大意了，沒有回家補裝備，交出E技能跳臉消耗，被諾手雪球跟上。最終小天阿卡麗沒有拉開距離，被諾手大招劈死，大優勢翻車了。

賽後Jankos又開始說騷話痛擊隊友，表示自己1V1贏了小天，說明S9總決賽不是打野問題，是上中下三路差距太大了！

第二場則是Faker對上越南打野Levi，賽前大家都覺得這組沒啥懸念。因為Levi作為打野選手，對線能力肯定不如專業中單，更何況他的操作也是老化了。然而Faker阿卡麗秀出老年人操作，屏障開晚了，沒擋掉小炮火花，陷入了巨大劣勢。

好在之後Faker阿卡麗穩住局勢，找到機會越塔一換一，逼出小炮金身，還拿下了一血。

又是縮圈決戰，Faker阿卡麗技能全中，但是大招沒好，不敢E上去火拚。最後Faker阿卡麗躲在煙霧裡，大招仍是處於等待冷卻的狀態，還被打滿火花，E出火圈而死。Faker也翻車了，爆冷敗給了越南打野Levi！

第三場輪到TheShy對抗日本上單Evi了，昨天TheShy為Jackeylove選下德萊文，秀起操作，躲過刀妹每一個E，完虐菲律賓中單，並表示這一場要選Rookie最擅長的英雄了！而Evi是日本DFM戰隊的大腿，也是韓服王者段位，他能扛得住TheShy的攻勢嗎？

結果觀眾都上當了，TheShy選用了劍姬，這可不是Rookie擅長的英雄。畢竟Evi選用了塔姆，打近戰幾乎無敵，對脆皮也很有威脅，但這種偏坦的戰士很怕劍姬打破綻，更何況對手是TheShy。果然，塔姆前期對線被TheShy劍姬吊打，只能靠QW塔下補刀，一度被壓了三十多刀。

縮圈決戰TheShy劍姬三小件加雙多蘭劍，而塔姆只有日炎，也沒操作空間，根本沒有擊殺劍姬的可能性。最終TheShy劍姬把塔姆打成絲血，在草叢裡掛機，等著塔姆送死。

而接下來就是Doinb對上JT中單Fofo了，兩人在S9小組賽中曾有過交手，算是一勝一負，據說今年Fofo要來BLG戰隊打中單。Doinb用瑞茲來打刀妹，這是很不好打的對局，很容易被單殺。於是Doinb選擇純坦出裝帶徵服者，想用這個另類思路來進行對抗。

前期雙方和平發育，等縮圈決戰。但Fofo刀妹也很懂SOLO，摸出死刑宣告和飲魔刀，Doinb瑞茲中個E就涼了，毫無對抗性，輸掉了本場SOLO賽。昨天小天採訪真說中了，S9冠軍中野齊翻車，兩人都在做夢。

最具關注度的大戰來了，Uzi要對抗Bang！Uzi的實力有目共睹，而Bang作為兩冠ADC，以前在SKT戰隊完全不虛Uzi，被稱為打狗棒，只是現在去北美養老不如當年了。

而Uzi選下韋魯斯面對Bang亞索，一旦進入縮圈決戰就沒得打了，他必須在前期抓住Bang亞索的破綻，打出足夠大的優勢。Uzi韋魯斯對線拚命想打出壓制力，但還是沒能壓住亞索的補刀，兵線被反推到塔下。

此時解說車瀟開始放飛自我，表示Uzi想用ADC拼操作，但Bang選亞索就是在搞Uzi的心態，太想贏了，噁心人，吃相難看！這下連旁邊的解說雨童都坐不住了，趕緊打住，喊道：「過了，過了！」

縮圈決戰開打之前，Uzi拆了兩層塔皮卻沒回家補裝備，局勢非常危急。好在Bang亞索失誤了，亂交風牆，導致自己的位置進退兩難，被Uzi韋魯斯大招禁錮活活打死。

雖然Uzi贏下了比賽，但解說車瀟卻激起了網友的怒火。SKT粉絲自然是憤怒不已，Bang打SOLO賽為了贏而選亞索有什麼錯？解說車瀟現場直播開噴，罵Bang吃相難看，有給兩冠ADC一點尊重嗎？而Uzi粉絲也覺得車瀟是在噁心人帶節奏，又是給Uzi招黑，最後他連一句恭喜Uzi的話都沒說！車瀟這波開噴太敗人品了，得罪了兩邊粉絲，微博慘遭爆破，網友叫他趕緊走人！

今天SOLO賽三人翻車被淘汰，好在TheShy與Uzi都順利晉級了，沒準他倆真能會師決賽，讓我們期待下午開打的八強賽吧！
