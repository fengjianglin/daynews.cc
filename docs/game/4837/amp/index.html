摘要

在王者峽谷中，玩家是否擅長某一位英雄不僅可以通過熟練度體現，榮耀戰力值也是衡量大家對英雄掌握程度的標準，那麼遊戲內如何直觀查看所有英雄的榮耀戰力呢？

在王者峽谷中，玩家是否擅長某一位英雄不僅可以通過熟練度體現，榮耀戰力值也是衡量大家對英雄掌握程度的標準，那麼遊戲內如何直觀查看所有英雄的榮耀戰力呢？如下圖所示，大家都可以進入大廳左下角的英雄庫進入選擇榮耀戰力排序查看，然後我們再來看看官方最新公布的星耀以上段位的榮耀戰力排行榜都有哪些英雄上榜？

第六位李白，上榜戰力12796，本賽季李白成為了玩家排位上分僅次於猴子的首選刺客，勝率也超過了猴子高達52.1%，正因為擁有高機動、不可選中和高消耗等機制，李白的禁用率位列峽谷前五，畢竟打排位時脆皮最不想遇到的就是李白這類刺客！

第五位幹將莫邪，上榜戰力13037，身為高Poke法師的代表，幹將莫邪因為上手操作難度和技能命中率問題，人氣並不高，但在排位賽高分局幹將莫邪還是很吃香的，小編建議想要練習的小夥伴不要太過專註於雙劍命中，因為裝備起來後單劍也能打出高額傷害，而把握好與敵人距離釋放遠近劍才是關鍵！

第四位不知火舞，上榜戰力13052，與炮台法師幹將莫邪相反，不知火舞屬於法刺，機動性高爆發傷害足，需要貼臉打傷害比較考驗玩家的操作，同時因為gank能力強，可以配合野輔聯動帶節奏，成為了不少玩家的上分首選！

第三位露娜，上榜戰力13651，露娜這類可以秀翻全場的英雄通常是主播和職業選手的最愛，比如KPL選手VG.易瞳就是使用了露娜打上國服最強，正因為用其衝擊榜單的玩家非常多，導致露娜的戰力長期居高不下，雖說月下無限連很秀，但做不到的話容易被集火，不適合手殘黨使用！

第二位娜可露露，上榜戰力16121，娜可露露是野核體系的代表刺客，優勢是爆發傷害足抓人效率高，只要前期打出優勢，就能快速滾雪球讓對方熬不過發育期，小編建議使用娜可露露打野選擇一位好輔助組排配合，比如明世隱或鬼谷子！

第一位諸葛亮，上榜戰力16899，諸葛亮在星耀以上段位的登場率為1.7%，屬於上下限較高的中單法師，依賴被動和大招，諸葛亮不僅有著非常高的持續收割能力，還能完成刺客的反蹲和切後排任務，同樣考驗玩家的操作意識！
