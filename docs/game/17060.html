結束了EPIC限時獨佔合約，這兩款Steam獨立遊戲值得一玩

2019-12-14 443條評論 2,099次閱讀 728人點贊

去年年底的時候，博士和大家分享了文章《Epic Games旗下平台首批上架遊戲名單公布》，講述了以《堡壘之夜》與 Unreal Engine 聞名的遊戲廠商 Epic Games 旗下 Epic Games Store 上線之際，還公布了首批上架的遊戲名單。

如今，一年時間已經過去了，Epic Games 限時獨佔的遊戲相繼結束合約，可以登錄Steam了。今天，博士就分享兩款還不錯的、脫離了 Epic限時獨佔合約的獨立遊戲。

一款遊戲是《Ashen》。這是一款美術走精緻簡約風格的「魂類」Roguelike 動作冒險遊戲，編劇則是奇幻小說《破碎帝國三部曲》的作者 Mark Lawrence，玩法上類似《黑暗之魂》，但是關卡卻千變萬化。玩家可以自由的探索世界，並且允許多人合作一起攻略遊戲。

在戰鬥和遊戲情節推動方面，《Ashen》從黑魂系列那裡借鑒了很多，但它很好的利用了黑魂的模式，提供了令人滿意的戰略戰鬥和劇情進展。目前，遊戲在外媒網站上普遍獲得了好評，值得一玩。

另一款遊戲就是《Hades》。由 Supergiant Games 打造的《Hades》是一款45度俯視角 Roguelike ARPG，主打手繪美術風格，玩家將扮演冥府王子 Zagreus，忤逆冥王，試圖從地底世界逃脫。在遊戲中，玩家也能遇到熟悉的希臘神祇。

遊戲中有許多隨機生成的房間，玩家必須擊敗各個區域的敵人與 BOSS，也能從中體驗劇情與不一樣的路線，而戰鬥上則屬於熟悉的暗黑風格，玩家需要迴避敵人的攻擊與包圍，一邊打一邊跑。

另外，在海外備受好評的《無題大鵝模擬（Untitle Goose Game）》雖然還在Epic獨佔期中，但是這只是針對PC平台的，遊戲即將登陸 PS4 與 Xbox Game Pass。大家也可以留意一下這隻搗蛋的大白鵝！
