AG暫停Team radio還原！讓我們猜一猜隊員當時在想什麼？

2019-12-24 619條評論 1,922次閱讀 107人點贊

12月22日，在CFPLS15賽季季後賽八進六的第二場比賽中，AG戰隊3-1戰勝WE戰隊，成功晉級六強。在比賽的前半段，WE戰隊是一直佔據優勢的一方，第二局擔任潛伏者，WE一度取得4-2的領先，在拿下2V5的驚天殘局之後，AG心態崩潰，叫了暫停，Team radio完整的記錄下了AG在暫停時的交流，下面我們就來複盤一下。

絕跡：咱要麼就一起壓中路，要麼就別壓，別一個一個上。

寵兒：咱們能不能按照訓練的打，都不按訓練的打，咋玩嘛！

Sir：大家都聽我說，我們還正常打，訓練怎麼練的，我們就怎麼打。穿越火線都打這麼多年了，都按訓練的來，別一到得分荒了就都不會玩了！別急！心態都調整好！

LSN、綠豆、歐歐：迷茫沉默中……

從上面選手和教練的交流中，我們可以看出AG的隊員心態都比較急躁，也對自己的表現並不滿意，那麼他們心裡是怎麼想的呢？

Sir：這比賽打得也太亂了，都各打各的，賽前安排的挺好的呀，咋一到比賽就這樣？

絕跡：這打的啥呀，總有人不聽我指揮，真服了。不過WE這群小年輕，我們正常打，還有贏的希望！

寵兒：我太難了，都不按訓練打，隊友狂送，每次都剩我自己打殘局。

LSN：穩住別慌，我們能贏！

綠豆：求你們了，都別浪了，再浪我們真輸了！

歐歐：我還是別想太多，這賽季狀態還行，輸了也不用我背鍋，還是繼續把槍穩住，干就完了！

以上就是我們總結的AG在心態崩潰瞬間隊員們的心靈路程，好在AG最後有驚無險的逆轉了比賽，晉級下一輪。比賽雖然獲勝，但AG需要總結的地方還有很多，那麼CFer們認為AG的隊員們在暫停時還在思考什麼呢？歡迎在下方評論區與大家留言分享！
