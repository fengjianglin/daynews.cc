《方舟：生存進化》手游的這些生物水陸兩棲，樣貌都奇醜無比

我們知道，淡水和陸地之間的交接處是蚊子、虱子等小昆蟲最愛呆的地方，這些昆蟲不僅僅吸食人血，其他動物的血液它們也都很喜歡，為了抵禦這些昆蟲的攻擊，很多水陸兩棲生物都歪著長的，其丑無比，在《方舟：生存進化》手游中的水陸兩棲生物就是這樣，讓人們懷疑它們是不是長歪了，下面就讓小編來給大家對這些生物進行一次盤點，看看這些生物究竟長什麼樣子吧！

首先登場的就是帝鱷這個水中霸主了，相比於現代的鱷魚，帝鱷在體型上要大上許多倍，足足有45英尺以上，這樣的帝鱷在《方舟：生存進化》手游中當然也是橫著走的，甚至能捕食來河邊喝水的小型恐龍，而帝鱷的外貌就不用多說，鱷魚能有幾隻好看的？

然後則是玩家很喜歡馴服的生物——魔鬼蛙，《方舟：生存進化》手游裡面的魔鬼蛙就是一個加大版的蟾蜍，而且樣貌比蟾蜍還要醜陋三分，不過雖然魔鬼蛙的樣貌不好看，可還是有很多玩家馴服它的，因為它能夠通過吞噬昆蟲製作出水泥，這是玩家在《方舟：生存進化》手游裡面蓋房子的必須品之一。

最後再來和大家說說《方舟：生存進化》手游裡面的水、陸兩棲恐龍棘背龍，和上面兩種恐龍相比，棘背龍在外貌上算是過關了，只不過它背後鼓起的大大的脊背還是會讓人有點不舒服，而且棘背龍是《方舟：生存進化》手游最危險的恐龍之一，就連霸王龍在河邊都打不過它，因此大家要是沒有什麼特殊需要的話最好還是別去惹它了。對於這幾隻《方舟：生存進化》手游水陸兩棲龍，大家有什麼看法？可以在下方評論區留言討論。
