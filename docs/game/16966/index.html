DNF：阿修羅終於迎來加強了？體驗服更新了部分春節版本內容

2019-12-15 68條評論 1,095次閱讀 90人點贊

在12月14日的時候韓服進行了發布會，這一次的發布會中，重點的內容就是100級版本了，新的版本即將上線，同時國服體驗服也進行了一次更新。這次的更新內容主要是關於春節版本的內容，四個職業進行了一次更新調整，這一次更新之後這幾個職業的調整可以說都很厲害。

一．阿修羅
這個職業最主要的調整在於無我波動的更新，這一次增加了一個基礎攻擊和技能攻擊力增幅的屬性，主要是這種分等級的更新和升級其實還是很有趣的。例如波動刻印每增加一個級別，那麼提升率就能夠達到0.5%了，在20級之後每增加一級就能夠提升1.5%左右，之後就是30級了，這樣的模式說實話從很大程度上將阿修羅的實力提升了不少。因此阿修羅這一次的更新以後也將會成為很多人心目中的一級職業了。

二．劍宗
然後同樣做出了調整的職業就是劍宗了，這個幻神也是如此，也有了不錯的改動，但是和劍帝比起來還是差距比較大的，因為劍帝這個職業目前已經是幻神了，一般的職業和他簡直對比不了。30級技能，1級增加5%，五級精通增加11%，15級上限，可以說這些個職業的加強肯定要比之前更加的厲害了。

三．魔道
然後就是魔道的加強了，這個職業也進行了一次比較大的加強處理，具體的增長也算是不錯的，被動技能加強了1%左右，3級加強了10%左右，6級增強了13%左右，5級精通，而15級上限。現在雖然看著是這樣，但是畢竟還沒有正式上線，所以之後到底有什麼樣的改動就不得而知了。相信大家應該知道魔道的加強其實還是很厲害的。不管怎麼說也是有所突破了，我們也很開心能夠進行一輪加強。

四．決戰者
最後是決戰者的加強了，決戰者的加強一共分成了三個階段。這三個階段分別是這個樣子的，長槍精通從原來的成長1%變成了2%，一覺也有所改動，在16級之後每一級的成長調整成為了2%，這個調整其實還是很走心的。也得到了很多人的支持。

總的來說目前體驗服更新的調整大概是這個樣子，因為具體還是要看最終在國服上線的調整內容。對於多數的玩家來說，這次的調整內容也就意味著這四個職業進行了一次大的改動，還好是增強而不是削弱。
