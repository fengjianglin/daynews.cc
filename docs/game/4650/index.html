航海王燃燒意志：新100級套裝暴徒更名“殘忍”，新限定全靠猜

2019-12-06 0條評論 14次閱讀 0人點贊

摘要

諸君安好，霧夏菌報道。更新的【鑄造】活動中帶著一套新的裝備，叫做“暴徒”，然而今天卻有進行了一波更新，更名為“殘忍”。結合這次鑄造活動的出貨率，殘忍的這個名字，的確非常般配。1、裝備這套裝備主打的是屬傷，2件套提升15%的屬傷，4件套有一定概率屬傷翻倍。

諸君安好，霧夏菌報道。

更新的【鑄造】活動中帶著一套新的裝備，叫做「暴徒」，然而今天卻有進行了一波更新，更名為「殘忍」。結合這次鑄造活動的出貨率，殘忍的這個名字，的確非常般配。

1、裝備

這套裝備主打的是屬傷，2件套提升15%的屬傷，4件套有一定概率屬傷翻倍。這套裝備的價值，官方公眾號也在鼓吹，而且這個時候突然出一套屬傷裝，很可能為了迎合下一波的新限定。那麼這套裝備的價值究竟幾何？

老玩家都與一個感覺，那就是屬傷類的輸出角色，普遍比不上暴擊爆傷輸出。其中的原因之一，就是屬傷不參與暴擊。同樣的傷害係數，一個技能全是普通傷害，另一個技能一般普通傷害一半屬傷。前者一旦暴擊，配合超高的爆傷係數，最終刷出來的傷害量，自然要高過屬傷技能。

而這次的屬傷翻倍，從某種程度上說，相當於給屬傷一個暴擊效果，補足了屬傷原本的不足點，其實際造成的傷害量，也就有可能超越前者。不過有一點要注意，那就是這裡的一定概率，到底是多大概率沒有明確，這是個隱患。

再一個，這套裝備太貴了，40000彩鑽一套裝備，而且屬性優劣還不保證，這個隨機性太大。除非你覺得自己經濟優渥，願意做點善事，否則這套裝備，物如其名，殘忍。

2、新角色

新裝備，大概率與新角色有關，而且很可能是一名屬傷角色。這個時候，官方公眾號也有動靜，提到了後續可能出現的一些角色，包括有雷屬性的娜美、火屬性的山治、暗屬性的黑鬍子、毒屬性的凱撒。

但只是推測，他們自己也保證不了。之前也出過一些錯誤，因為遊戲本身就經常出一些錯誤，臨時修改都成了家常便飯。而且一些數據包，也都是在更新前再加入遊戲，玩家想要偷看也基本沒可能了。再一個還有魚人島沒做，暫時也急不來。

就當前來看，下一個角色最有可能是一名屬傷角色，早已做好的黑鬍子，備受期待的新娜美，以及索隆的好兄弟山治，都有可能。

菌說

其實對於大部分的玩家來說，不一定非要苛求新角色，目前的陣容已經足夠應對大部分的主線。這個時候最好是存券等自己喜歡的角色，否則按照當前的更新量，真的吃不消。至於裝備方面，遊戲中氪金的的確太貴，急不得。
