騰訊羅辛辛：優秀QQ小遊戲湧現令人欣喜，要重視5G變革

2019-12-12 650條評論 1,593次閱讀 347人點贊

2019年，遊戲行業迎來動能轉換穩定期，輕量級小遊戲釋放流量紅利，為開發者帶來巨大發展空間。作為行業新動能中的重要力量，QQ小遊戲中心於12月11日在深圳舉辦主題為「小遊戲，Q精彩」的開發者峰會，吸引了逾800家遊戲廠商及開發者到場參與。

在大會現場，騰訊PCG平台與內容事業群/平台增值中心高級總監羅辛辛做了精彩的開場致辭，其在演講中表示，「經過QQ小遊戲半年的運營，我們欣喜的發現已經出現了許多高留存、高DAU的產品，這些優秀產品擴大了QQ小遊戲的商業化價值。在明年我們將會大力擴展買量分發體系，實現雙頭並進的分發模式。」

對於小遊戲的未來發展，羅辛辛特別提醒開發者，「5G時代的到來，可能對小遊戲帶來非常劇烈的衝擊。小遊戲從業者應快速做出調整，我們也將會在未來嘗試更多的遊戲形態、分發模式，順應時代的變化。」

騰訊PCG平台與內容事業群/平台增值中心高級總監羅辛辛

以下是羅辛辛演講實錄：

羅辛辛：感謝各位來賓參加此次大會，這也是我們QQ空間小遊戲升級為QQ小遊戲後的第一次峰會。

回望2019年，雖然去年整個遊戲行業經歷了人口紅利下滑、遊戲審批暫停再到恢復，但實際上去年遊戲行業產值仍然在增長。2019年遊戲行業報告顯示，今年上半年整個中國遊戲行業產值已經達到1163億，同比增長10.8%，其中移動遊戲增幅更是達到18.8%，那麼是什麼推動了遊戲行業的發展呢？

整個行業還在不斷前進，輕量化遊戲也佔據著越來越重要的地位，小遊戲開拓了更大的市場空間，並且極大程度拓充了流量規模、以及可商業化的遊戲品類。2018年我們還是QQ空間小遊戲的時候，就已經擁有了超過3000款小遊戲，並且積累了接近4億的遊戲活躍用戶，在小遊戲上，我們已經有了豐富的經驗。

今年5月，我們決定把QQ空間小遊戲升級成QQ小遊戲中心，為開發者帶來更開放、更優越、更平穩的平台運營環境，我們在多個方面升級了QQ小遊戲中心，主要體現在流量、品類、技術這三個方面。

首先我們開放了QQ全量的用戶群體，各位開發者的產品將被更多的QQ用戶所接觸到，我們現在採用的是中心化的分發資源，在明年我們將會大力擴展買量分發體系，實現雙頭並進的分發模式。

同時我們的品類覆蓋範圍變得更加的豐富，精品化和社交結合的優秀遊戲是我們平台關注的重點，我們將對這類產品傾注更多的資源。

我們QQ小遊戲的產品團隊和技術團隊也為開發者提供了更多便捷開放的介面，更加可靠的緩存環境和更多的設備能力，這些都能為遊戲帶來更加安全可靠的運行環境。

經過QQ小遊戲半年的運營，我們欣喜的發現平台已經出現了許多高留存的產品，比如《斗羅大陸》、《胡萊三國》、《超級精靈球》、《天行道》、《少年封神》等人氣口碑雙豐收的產品；同時我們也欣喜的看到如《歡樂鬥地主》、《鋼琴塊》、《全民切西瓜》等高DAU產品，看到這些開發者的付出獲得了良好回報，我們QQ小遊戲中心的「中期考核」也算合格了。

正是因為這些高流量的產品依託於QQ小遊戲分發和裂變，這些優秀產品也擴大了QQ小遊戲的商業化價值，你們的收益是我們最好的背書，我很希望未來有更多的開發者加入QQ小遊戲平台，讓我們一起努力把大家的創意、想法變成現實。

談完激動人心部分，我也想說說對未來的思考，希望能給大家一些啟發。

未來小遊戲可能會遇到更大的挑戰，正如4G時代給小遊戲帶來的機遇一樣，5G時代的到來，可能對小遊戲帶來非常劇烈的衝擊。

為什麼這樣說呢？因為APP遊戲可能會以非常快速的方式下沉到小遊戲市場，比如現在需要花半個小時下載的APP遊戲包體，在未來5G時代可能只需要幾秒就能下載完，同時未來還會有新的雲遊戲形態，新的模式有可能如山崩海裂一般衝擊現在小遊戲運營模式和市場，那麼那時的小遊戲開發者、熱愛者該何去何從？

歷史總是如此，我們的時代在不斷變化，技術也在不斷創新，我們不應該去阻擋歷史的洪流，而應該更多的去擁抱它。APP遊戲有可能會下沉，但小遊戲也會有更多機會上浮、去與APP遊戲進行競爭。

小遊戲要上浮，我們就要理解它的「輕」到底輕在哪裡？它是因為「包」很「輕」嗎？還是因為技術架構比較「輕」？開發時間短、開發成本低嗎？是這些手段上的輕嗎？其實都不是。這些所謂的手段隨著新技術的來臨，很快會被5G所替代。

那所謂的「輕」在哪？它體現在我們給用戶帶來的便於理解的玩法體驗，是我們對用戶的無差別的體驗，這是拋開所有技術手段，在任何時代都會被認可的用戶需求，這個核心訴求，我們需要好好把握它，並且不斷更迭技術，調整我們自己的組織狀態，去嘗試更多的分發模式以及更多的付費模式，來順應整個時代的變化。

小遊戲也許在未來會變的越來越「重」，「重」在技術投入上、畫面效果上、玩法上，我們越來越需要強調小遊戲的精品化，也許未來APP遊戲和小遊戲不會再是物理形態上區分，兩者間界限會變的越來越模糊，就像在PC時代微端模式，微端是頁游嗎？其實它有可以下載的客戶端，但微端遊戲卻有媲美真正端游的能力，有著精美的3D效果、豐富的遊戲系統，那微端是端游嗎？微端的進入方式、覆蓋的用戶人群、玩法形態又分明是頁游。假如移動遊戲沒有這麼快到來，那麼在PC頁游時代、微端將佔據著很重要的地位。

在今時今日，我們小遊戲的從業者，面臨時代的變化應該快速做出自己的調整，包括QQ小遊戲中心也是如此，我們將會在未來嘗試更多的遊戲形態、分發模式，去改變我們自己、去順應時代的變化。

也許我們名字會改變，我們的遊戲形態會改變，但是唯一不變的只有不斷變化的我們自己。
