黃金聖鬥士系列皮膚已有三款，官方難道真的要湊滿十二個？天真

2019-12-05 0條評論 19次閱讀 0人點贊

摘要

專業瞎解說，我是大酋長～

好久不見，甚是想念。

前段時間，王者榮耀上線了張良傳說皮膚黃金白羊座，不知道大家有沒有上手體驗呢？沒有也沒關係，王者榮耀即將開啟全直售傳說皮膚限免，新上線的張良的黃金白羊座也在限免名單里，喜歡的朋友可別錯過。這期想說的是，張良黃金白羊座加上前期已經推出的后羿黃金射手座、達摩黃金獅子座，王者榮耀已經擁有了三款黃金聖鬥士系列皮膚，官方難道真的想集齊十二款嗎？

關於這個問題，玩家之間的猜測也有很多，甚至不少玩家都為官方物色好了各個黃金聖鬥士系列皮膚的英雄人選。酋長的看法卻並不樂觀，官方或許真的有這個想法，但未必真的能出完十二款黃金聖鬥士系列皮膚。

首先從視覺效果來說，黃金聖鬥士系列皮膚因為原本的設定就是如此，所以這個系列的皮膚必須全部採用金色特效，如果只是一兩款還好，但整整十二款都是金色特效，非常容易造成玩家審美疲勞。除此之外，王者榮耀本身也有一些金色特效皮膚，比如五款勝利之神系列皮膚等等，就會更容易加劇玩家的視覺疲勞感，而這也就直接關係著皮膚的銷量。

然後從歷年表現來看，王者榮耀從公測至今已有四年時間，挖下的坑可以說不計其數，但填上的卻不多。比如賽季皮膚系列，原本的設定是五個賽季推出一個系列皮膚，比如勝利之神系列和無畏之靈系列，但第三個系列隱龍之影只出到第三個，S17賽季就改成了鳳儀之詔，變成了每個賽季都有一個主題；再比如夏日皮膚系列，前年和去年的夏天都推出了夏日皮膚，玩家對其的評價也很高，但今天冬天都到了，也沒看到夏日皮膚的續作。諸如此類，還有不少，由此看來，黃金聖鬥士系列皮膚也極有可能走上這條道路，或許官方此時真的有湊齊十二款的想法，但未來的發展過程中就未必了。

綜上，酋長認為王者榮耀不太可能出滿12款黃金聖鬥士系列皮膚。當然，這個觀點僅限於酋長的猜測，被打臉也不是沒可能，那麼大家認為呢？
