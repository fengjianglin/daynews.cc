碎片商店更新，天美終於良心一回！網友：我要的街頭皮膚不是這款

轉眼間2019年就剩下最後一個月了，S17賽季距離結束也只剩二十天；大家都打到了自己想要的段位了嗎？

月初這幾天天美不但上線了魯班大師這一新角色來陪伴小魯班一起挨打，原定的活動也如期進行著，碎片商店的更新相較於新英雄顯然更能觸動玩家們的G點。

作為2019年最後的幾次碎片商店更新，本次天美上架皮膚商店的多款皮膚在質量上還是非常高的。亞瑟的心靈戰警首當其衝，夏侯惇的乘風破浪、嫦娥的露花倒影、楊玉環的霓裳曲都是一時之選（如果你不在意這些英雄的上場率的話）。

像以往那樣只靠一款皮膚撐門面，天美這回推出的一系列皮膚確實足夠良心。（當然只是跟往常對比，玩家永遠不會嫌皮膚太好）Pony馬這將再次達成血虧成就？

而本次皮膚商店的頭牌心靈戰警-亞瑟，無論是在外觀還是特效上都深得玩家們的喜愛，比其他皮膚更魁梧的身形及富有科技感的技能釋放過程都突出一個無敵炫酷吊炸天；唯一美中不足的也就只有那操蛋的手感了……

當然每一次碎片商店更新玩家們都會幻想著能不能出現自己中意的那款皮膚，擁有超高人氣的韓信每次都被謠傳街頭霸王皮膚將會上架，本次也不例外；只不過最終等來的確是賠錢虎的街頭旋風~

與此同時收款由玩家設計的皮膚繁星吟遊-蔡文姬也將以6元秒殺皮膚的身份在5號上線，屆時這個在排位賽中擁有超高勝率的輔hun助zi英雄勢必將引起諸多女性玩家們的另一場購買狂潮。

大家覺得此次的碎片商店更新的皮膚成色如何呢？
