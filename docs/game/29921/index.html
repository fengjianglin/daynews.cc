LOL德瑪西亞杯：A組御三家首日實力評價，TES為S級，RNG僅有A級

2019-12-23 1,759條評論 4,675次閱讀 308人點贊

雖說德瑪西亞杯賽事的含金量水平不是太高，但是作為總決賽和春季賽之間的過渡類比賽，在一定程度上，德杯賽事也能夠的展現出轉會期過後各個戰隊的實力。畢竟賽前的各種預測，各個隊伍的新選手實力終究是需要比賽來印證的。從第一個比賽日A組戰隊的表現來看，A組御三家也展現出了轉會後新隊伍的實力。本來賽前被認為新賽季低迷的戰隊，首日比賽中展現出”精裝修”的實力，是讓人有些驚嘆，出乎意料。

WE戰隊在德杯的表現讓人稱之為原地建別墅，在德杯中表現出的實力可以被評為S級別。賽前認為離開xiye和大舅子雙C之後的WE戰隊必定淪為墊底隊伍，可是賽場上一串三的壯舉更是讓WE戰隊的粉絲找到了早期的感覺。beishang也擔起了carry的角色，盲僧搶龍幫助WE戰隊力挽狂瀾，或許這樣全新出發的WE戰隊會有另一番成績。

RNG戰隊在德杯賽前被認為是老隊伍中轉會情況最好的一個，卻在比賽中證明一點RNG只適合四保一。從目前小龍堡和RNG戰隊的磨合狀態來說，沒有Uzi 的RNG在比賽中的表現只能被評為A級。小虎、狼行仍舊是RNG戰隊的兩大短板，沒有了小狗的RNG成為任人踩踏的一個戰隊，倘若中上兩路不做改變，就憑藉小龍堡一個人carry還是很難更進一步。

在二十支德瑪西亞杯戰隊之中，最讓人期待的莫過於全華班TES戰隊。雖說轉會期的TES並沒有進行大規模的變動，但是熊熊替換為卡薩讓TES戰隊實力飆升。熊熊在去年的聯賽中展示出個人不俗的實力，在英雄池和節奏的把控方面的能力仍舊與卡薩有較大的差距。再者，在洲際賽的時候，熊熊一度指責369選手，讓整個隊伍氛圍陷入尷尬的局面，如今換走了脾氣頗大的熊熊，迎來了卡薩和左手的中野聯動，肯定能夠在聯賽中稱霸許久。而且TES戰隊選擇放棄韓元選手loken，這就讓AD位置不會出現暴斃的局面。這樣的TES戰隊被評為SS級一點也不過分。

在第二個比賽日結束後，也能再次看出B組轉會後的隊伍實力。廠長雖成了B組的亮點，但是成績才能說明一切。本屆德瑪西亞杯比賽將由鬥魚獨家直播，賽事從12月21日持續到29日，和鬥魚一起見證新隊伍的力量，戰入新章。

《英雄聯盟》德瑪西亞杯賽事由鬥魚全程獨家直播。上鬥魚，看德杯，見證新生的力量。
