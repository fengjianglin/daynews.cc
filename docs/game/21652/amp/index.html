冠軍教頭執教魚腩！Kkoma加入VG遭質疑：帶好了血賺、沒起色不虧？

天天要聞
2019-12-18

專為LPL戰隊磨合陣容而舉辦的德瑪西亞杯在本周六就將開賽，屆時所有的戰隊都將亮出自己的底牌並測試陣容強度，這也是世界賽之後LPL觀眾看到的第一場正式比賽。

整個休賽期人員變動最大的戰隊非VG莫屬了，從教練到隊員都換了個乾淨，新的引援對於吃瓜群眾們也擁有足夠的吸引力。

戰隊最為核心的中、下雙C換成了夏季賽有過亮眼表現的蕉太狼，ADC則是從EDG轉會過來的iboy。不敢說他們是同位置最好的那位選手，但前五還是沒有問題的，最重要的是貴在年輕、潛力無限。

而VG其他選手的個人能力也不差，常規賽號稱「前10分鐘世界級強隊」，只不過比賽一拖就容易大腦短路突然暴斃。

觀眾都看得出來的問題管理層自然也不會放著不管，花重金從SKT挖來冠軍教頭Kkoma就能看出管理層想在S10出成績的決心。畢竟前面的FPX就是以新軍奪冠的，人總是要有夢想的嘛，萬一真成了呢！

理論上來說，用一位經驗豐富的冠軍教練來帶一支年輕的（魚腩）隊伍是再好不過的了。只不過，凡事也無絕對，里皮執教國足的故事就告訴我們有些隊伍不是靠教練就能改變的。

玩笑歸玩笑，但網友們對於Kkoma不遠千里到LPL來執教這樣一支弱隊的行為還是抱有疑慮的，而最為大家所接受的理由則是以下兩點——

1. 把VG當養老俱樂部了

2. 執教弱隊壓力小，帶好了血賺、沒起色不虧

後者的可能性更大一些，畢竟執教VG就是一個低風險高回報的活兒。

只能說以VG現在的陣容博一手季後賽還是沒有問題的，至於能不能像FPX那樣創造奇蹟就看磨合後戰隊內部能產生怎樣的化學反應了。
