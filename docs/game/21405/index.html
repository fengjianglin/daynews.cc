FGO超人俄里翁簡評，自buff型三紅輸出

2019-12-20 1,341條評論 2,586次閱讀 1183人點贊

FGO日服2.5章一期卡池實裝了五星弓階超人俄里翁，這裡我們就來了解一下新的人形俄里翁是怎麼樣的從者吧。

俄里翁的寶具為藍卡輔助寶具，效果為賦予自身三回合無敵貫通狀態，自身攻擊力提升30%~50%效果三回合（效果隨寶具等級提升），自身暴擊威力提升100%~200%三回合（效果隨OC等級提升），賦予自身弱化無效狀態三回合，賦予自身每回合獲得10個暴擊星狀態三回合。

一技能效果為自身紅卡性能提升30%~50%一回合，賦予自身對【猛獸】和【魔性】30%~50%特攻狀態一回合，充能時間7~5回合。

二技能效果為賦予自身1000~3000毅力狀態一次·三回合，攻擊力提升10%~20%三回合，獲得10~20個暴擊星，充能時間8~6回合。

三技能效果為自身暴擊星集中度提升300%~500%三回合，賦予自身【紅卡攻擊前暴擊威力提升50%~100%一次狀態】三次·三回合，充能時間7~5回合。

職階技能【單獨行動】自身暴擊威力提升12%，【海神的祝福】自身傷害附加175，傷害減免175,，【天蠍的詛咒】賦予自身毒蝕狀態（副作用，毒的傷害提升20%，沒有吃到毒的時候不起效果）。

超人俄里翁持有神性、天地、所愛之人和希臘神話男性特性，被EA特攻，配卡為三紅一藍一綠，hits數為紅1藍3綠3EX5，最大ATK為12557，最大HP為13494。

超人俄里翁是首個自buff型的弓階輸出從者，有持續三回合的無敵貫通效果和大量攻擊力提升效果，配合三紅配卡輸出非常高，自身有毅力狀態也比較適合單挑場合。

融入隊伍中的話因為自身有出星和集星效果，所以打暴擊也比較容易，不過因為目前的自buff輸出都是比較偏向高難的，所以能否被廣泛使用，也得看未來的高難情況，總之周回是不太能指望了。

俄里翁為五星限定從者，雖然可能肌肉男的形象並不是大家都能接受的範圍，不過稀有度也是存在的，只能說波塞冬的基因看起來要比宙斯強大啊，如果剛好戳中萌點的話，也還請不要錯過卡池。
