DNF：100級搬磚圖！金幣掉落+120%，格藍迪又火了

天天要聞
2019-12-20

地下城與勇士100級史詩製作路線已介紹完畢，那麼對於大家來說，還剩下另一個問題，100級的金幣夠花么？目前來看，比95消耗的少，因為刻印和升級免費，多個小號直接省下幾個E。當然你要是無限刷深淵，哪個版本都不夠花，為了讓大家轉移搬磚地圖，魔界在100級收益會大減！

100級搬磚圖！金幣掉落+120%，格藍迪又火了

在100級副本的定義中，普通玩家要搬的就是「風暴航路」以及「紅玉的詛咒」，兩者均產出金幣和深淵材料（100級深淵門票引導石），後者一陣雨展示了一波，平民5層止步，除非你是擁有F1秒殺能力的強大飛行員。

在100級更新後，哈林魔界的確的金幣掉落會下調，另外普通玩家最想要的一個設定來了，那就是副本難度越高，金幣掉落越多，在100級的風暴航路區域中，搬磚難度遠超魔界！普雷C的水平，漩渦C都吃力。

實測這就是新版本的「格藍迪」！但小怪和BOSS均升級到了135級，所以難度才會很高，一個拔刀過圖不存在，因為中間會隨機刷出各種精英怪，上千條血需要幾個技能（知道魔戰為什麼要升級了吧，搬磚都得100級裝備才行）。

在這100級的專屬搬磚圖中，依舊限制了GZS，通過三個方式：①難度高，組隊血量翻幾倍，想要搬磚先去達成普雷C或者刷出100級傳說。②隨機地圖，遇到不同BOSS！每位玩家搬這個風暴航線遇到的BOSS都不一樣，一些精英怪還需要走機制，多開簡直是災難。③金幣掉落是主要收益！和魔界差不多，翻牌+掉落兩者要組合。這樣普通玩家的收益就能變相提高了。

100級的搬磚圖相比95級魔界，有一個很大的變化，那就是金幣掉落+120%，BOSS等級高，掉的金幣就是多，一次6000金幣左右，而95魔界不過2600上下，相比提升了一倍還多，要是一次出3~4個金幣堆，玩家就算過了房間也得掉頭回來撿！其實還有另外一個收益，100級的引導石，下個等級大家可以靠深淵畢業，搬磚的同時還能畢業。

總的來說，等級擴張後！格藍迪又火了，但不是85級而是135級的，搬磚副本這麼改，小夥伴覺得怎麼樣呢？
