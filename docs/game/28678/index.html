三炮齊發，大鳥拐彎，射手強化誰將稱霸？

2019-12-22 1,835條評論 4,257次閱讀 1648人點贊

左輪變機槍——馬可波羅

強化分析

馬可波羅受到強化的是一技能：將一技能的子彈數提升至原來的三倍，然而不會疊加被動或提升能量

看上去馬可波羅的改動好像不痛不癢，然而實際上強化後馬可波羅的一技能傷害是特別高的，雖然不會疊加被動的層數也不會增加能量，然而卻可以觸發末世的破敗被動、純凈蒼穹的精準被動以及破曉強化普攻的效果，實戰傷害極高，如果一梭子全部命中，傷害不可計數。並且額外的子彈還具有穿透效果，這便使得馬可波羅的刷錢效率提升不少；額外子彈提供的寬度也使得馬可波羅的技能命中率大幅提升。

玩法：老老實實的利用強化後的一技能刷錢打輸出即可，大招盡量不要使用，因為真的很弱。

出裝推薦：還是以法球出裝為主，攻速鞋、末世、純凈蒼穹、破曉這四件套必出，除此之外還可做出一件冰痕之握以提升技能命中率，也可做出暗影戰斧以提升技能傷害、增加減速、降低冷卻。最後可做出一把破軍，將技能傷害拉滿，同時也能增加收割能力。

夢回當年——狄仁傑

強化分析：

被動層數提升至10層，並且每層所提升的攻速移速屬性也略有增加，這便使得狄仁傑前期對A能力提升明顯，中後期走打能力進一步提升，配合強化的一技能尤其如此一技能六令追兇所丟出的牌數由6提升至12，而被動效果也改成了狄仁傑每次普攻都會發出紅牌、藍牌或者黃牌，其中黃牌帶有0.5秒眩暈。有種恍如隔世的感覺，初代腦殘傑又回來了？只要我攻速夠快運氣夠好，你就永遠醒不來。大招由一張牌提升至三張並排，命中率大幅提升

玩法：與正常模式無二

出裝推薦：由於強化版狄仁傑一技能的被動有概率帶有眩暈，因此實戰中還是要做出幾件攻速裝以充分發揮作用，而大招命中率的提升又使得我們的輸出有了保證，因此實戰中可做：攻速鞋、電刀、無盡、泣血、破曉、逐日之弓這套出裝，無論是正面還是秒人均不弱，攻速移速極快能充分發揮兩個被動的作用，發育也比較快。

草叢殺手——成吉思汗

強化分析：

被動移速提升由35增加至70，並且發射箭矢由二連發提升至三連發（似乎被動持續的時間也略有增加）被動的強化提升了成吉思汗的機動性，並且能夠更快的打出一技能的爆發，實戰傷害十分驚人二技能百獸陷阱儲備時間減半，數量提升至5（似乎觸發範圍以及視野提供範圍也增加不少）強化後的二技能可在實戰中用來開視野、刷錢、也可疊加多個用以秒人大招可汗狂獵的儲備時間下降至3秒，並且至多能夠儲備10支箭矢（實戰中冷卻也由2秒下降至1秒）強化後的大招最為厲害的便是那只有1秒的CD了，配合被動以及一技能可以打出極為爆炸的傷害

玩法：蹲草叢，使用二技能大範圍的減速控制敵人，然後打出被動以及大招的傷害，實現光速秒人

出裝推薦：與正常對局無二，不過最好做出諸如暗影戰斧、宗師之力這樣的秒人裝，因為強化班成吉思汗整體強度一般，只能秒人而不適合站擼。

孫三炮——孫尚香

強化分析：

一技能改為三炮並發，並且命中後可降低更多的CD。實戰意義便是將孫尚香的爆發提升了不少，前期發育速度加快，而到達後期則有幾率一炮秒人。二技能命中範圍提升至1.5倍。改變了二進呢過難以命中的情況，更方便減速以及減甲了。大招也改為三炮齊發，雖然沒有一技能那麼好用，不過用來補傷害，清兵還是不錯的

玩法出裝：與正常對局無二，最好儘早做出暗影戰斧、無盡戰刃之類提升攻擊力的裝備 ，以增加爆發。

移動的炮台——黃忠

強化分析：

二技能可存數量上升至5個，提升了對線期的生存能力，並且根據姐夫實測似乎每顆雷所增加的護盾也多了不少，實戰中能增加媲美張飛的護盾量，極大的提升了黃忠的生存能力。大招的攻擊範圍提升至普攻模式的1.2倍，並且釋放期間可移動。黃忠由一個固定炮台變成了移動炮台，實戰中可操作空間大了不少。玩法出裝：與正常對局無二。

不過姐夫建議可做出一件逐日之弓，以提升自己的移速，打出更好的效果

拐彎狙擊手——后羿

強化分析：

被動懲戒射擊所射出的箭矢由3發提升至五發，變相提升了后羿的輸出能力，額外的箭矢也可暴擊並觸發法球以及攻擊特效還有吸血效果，實戰輸出提升明顯。大招可在命中目標後通過操作使其轉向。這個可以就厲害了，后羿的大鳥有了思維能力，即使第一次命中不了敵人也可通過轉向實現命中玩法:雖然后羿的被動傷害提升不少，然而實際上與其他射手相比優勢並不明顯，實戰中玩家通常喜歡利用大招來做些文章，配合蘭陵王、鍾馗之類的英雄實現定點擊殺的效果。

出裝：冷卻鞋、暗影戰斧、破軍、碎星錘、破軍、破軍。將冷卻縮減屬性以及技能傷害拉滿，充分發揮大招可轉向的特點輔助隊友。

這些射手強度各有不同，然而最為有趣的則莫過於后羿了，姐夫也實名推薦一波，希望大家玩的開心。
