上賽季的盾法重現雲頂？盾盾流陣容該怎麼玩？

2019-12-22 310條評論 3,879次閱讀 2101人點贊

經過了個把月的激烈角逐，大家終於摸索到了上分之路，然而，就在掠食毒和劍刺陣容大火大熱之際，被大家忽略的一件裝備——鋼鐵烈陽之匣 重現雲頂棋盤！悄然改變了雲頂近期走勢。

論S2的雲頂誰最適合此盾，狗熊當之不讓。噢，忘了，先來給不認識的小夥伴介紹一下這件裝備：

鋼鐵烈陽之匣，由無用大棒和鎖子甲合成，長的是這麼個鳥樣，所以眾人稱之為——鳥盾

屬性呢很簡單，開局施加三百點的護盾，左右各兩格，加上自己五個，持續時間長達8秒。

這麼一算，一件裝備300*5=1500，一個山脈羈絆到手！那麼三個……這誰打得動呀，關鍵是不是隨機得呀，是可以自己安排的，明明白白的！

好了，接下來討論幾個陣容搭配：

先來難度高點的：極地雷霆守護毒狂戰

陣容組成：

奧恩+布隆+狗熊+蒙多+奧拉夫+老鼠+煉金+劫

裝備很簡單，鳥盾有幾個做幾個，老鼠需要一個極地轉職，劫復活甲最好要有。

都是基於狗熊的陣容，先把陣容看完，咱們再來說說怎麼玩。

第二種簡單粗暴：六狂戰四極地

鱷魚+武器+蒙多+塞恩+狗熊+奧拉夫+布隆+EZ

除了鳥盾，EZ需要熾天使，如果有條件我喜歡加上法爆和無盡，奧拉夫就是標準裝備了，能有幾件就幾件吧

第三種：海洋極地守護雷霆

狗熊+EZ+奧拉夫+布隆+錘石+泰坦+奧恩+劫

裝備還是那麼老一套，一般玩這陣容，前期沒有很慘，都能夠上九，補個海洋光輝或者極地光輝都非常的舒服！

接下來就說一下該怎麼玩呢？

開局搶裝備，有這麼幾種：搶狗熊、搶大棒或者鎖子甲。如果是狗熊帶的這其中裝備，那肯定是必搶

前期，奧恩、狗熊不用說，牌來了肯定拿。像鱷魚、狗頭、武器都得留，因為這決定後面是走狂戰還是走守護。

誰兩星的快，不用想了，賣掉別的，吃利息，升級！

這些個陣容的奧義就在於速8！

因為你需要他：

當然走狂戰流，你不需要劫，但是也需要升級，

EZ這個英雄，三費，前期一隻停留是來不了的，當然發牌員是親戚的話，隨意。

運營方面，前期要是奧恩狗熊只有一星不要強上，隨意拿兩星的打工仔，升級也是為了連勝。

其實這幾套陣容並不需要怎麼過度，只要你的英雄兩星了，都能扛得住

所以難點在於沒有兩星英雄支撐，那就垮了。

所以在第二輪選秀後是一個關鍵點，如果這時候你沒有四隻甚至三隻兩星的卡，強行上場是不行的。

前期的掠食者手段極其殘忍！

所以選秀過後，視自己場上卡質量，適當的D一波，盡量都追到兩星，那就高枕無憂了，

實在背沒有搜到，也不要驚慌，千萬別一口氣把金幣D光了！別問我為什麼！

留至少20-30

後邊上八補齊陣容，OK，打得過，存錢升九，打不過，搜哈！把沒兩星的追上，能上三的追三，

追三基本上是狗熊、布隆，EZ，四塊以上的就不要想了！

希望你們能夠把握機會！估計下個版本，設計師又要進行調整咯！
