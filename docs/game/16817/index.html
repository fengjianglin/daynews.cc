這遊戲公司瘋了？12月19號開始連送12天免費遊戲，送完還跪謝玩家

2019-12-16 24條評論 1,103次閱讀 106人點贊

新的一周又開始了，本周又會有哪些好玩的新遊戲等著我們呢？一起來看看吧。

【本周熱點事件】

劃重點：Steam冬季促銷12月20日正式開啟 接連不斷的打折真讓人扛不住

G胖他又來了，來得如此的迅速和頻繁。

上一次的打折促銷截止於12月4日，這才剛剛過了兩個星期，steam又要開啟新一輪的冬季促銷了。

不知道在上一次已經打骨折了的遊戲究竟這次又要拿出什麼樣的折扣。

X博士點評：我反正已經被掏空了，畢竟兩次打折促銷期間還夾了一次雙十二。

劃重點：Epic送遊戲送瘋了！19號開始連續12日每天送不同遊戲

Epic商城自上線以來一直都很熱衷於白送遊戲，每周都會送出一到兩款作品，有時甚至還會打包送系列全套，這也讓很多玩家享受到了相當多的福利。

可由於獨佔了不少好遊戲的緣故，儘管它狂送遊戲，但在玩家間的口碑也依舊是普普通通，一邊白嫖一邊罵Epic的人更是數不勝數。

可在本周，這樣的局面或許將完全扭轉——Epic開啟了真正意義的新年活動！

這可比簡單折扣要爽得多，因為它將開啟為期12天的「白送活動」，每天都會有一款不同的遊戲白給。

X博士點評：我已經沒辦法說他不香了。

【本周熱門更新】

劃重點：年終最強版本！《流放之路》12月19日開啟S10賽季

2019年已經只剩下最後兩個星期，在多數大型遊戲都已經推出相應的年末新版本之後，《流放之路》也迎來了今年最後一個大型版本「異界災變」，也就是我們所說的S10賽季。

正如新版本的名稱，在S10賽季中異界地圖發生了極大的變化，玩家只能從異界圖鑑中心開始，並且最初只有91張地圖可供選擇，玩家需要擊敗征服者並將取回的守護之石插入圖鑑才可以解鎖其他地圖。

除此之外，技能寶石也迎來一波強化，如今升級版的強化寶石1級就比原先20級的普通寶石更強，玩家的戰鬥強度大幅度提升，不過相對的新版異界的難度也不可同日而語。

很明顯，此次更新的異界變化、新通貨、新寶石等內容其實就是在給之前公布的《流放之路2》做鋪墊，更高的難度以及更高的提升上限也讓玩家擁有更多可玩的內容。

X博士點評：現在倒是一個入坑的好時機。

【本周新游上線】

劃重點：魔性動物格鬥遊戲《動物之戰》上架Steam 12月19日發售

一款名為《動物之戰》的超級魔性遊戲在將在12月19日正式登陸Steam平台發售。

這款遊戲的畫風玩家看了想必都非常熟悉，這些角色都是之前在網上因為錯位而特別火的動物。

肌肉柴、王者狐、霸氣貓之類的名字雖然聽起來不是懂，但如果看了圖片大家肯定就能聯想起來。

另外遊戲採用了一鍵出招系統，玩法極為簡單，只需要按下按鍵就可以進行戰鬥，格鬥新手也完全可以駕馭這款遊戲，所有玩家都可以來體驗這個奇葩格鬥。

X博士點評：能把這些奇葩動物做成一款格鬥遊戲，這個開發商也挺奇葩的。

劃重點：《無題大鵝模擬》新宣傳片 12月17日登陸PS4/X1

本年度最成功的網紅遊戲《無題大鵝模擬》將會在12月17日正式登陸PS4和Xbox One平台。

這應該是開發商為了讓全平台的玩家都能夠感受到「鵝」的可怕之處。

不過若是想要遊戲變得更紅，不如考慮考慮移植手機版，畢竟遊戲操作並不難，移動平台的用戶也很容易接受這樣一款有趣的遊戲。

能把一隻鵝做到全平台就足以看出它有多火，難怪TGA頒個獎也要來蹭一蹭大鵝的熱度。

X博士點評：

只可惜大鵝雖然榮獲兩項提名，但卻沒能拿下獎項。

唉，只好自己動手搶了。

劃重點：街機遊戲《JOJO的奇妙冒險：最後的倖存者》12月18日上線

在12月18日，日本各大街機廳將會上線一台新機器，那就是JOJO吃雞《JOJO的奇妙冒險：最後的倖存者》。

其實JOJO的世界觀倒還真挺適合吃雞的，畢竟漫畫里一直就是能力者互相追殺。

在遊戲中玩家要扮演漫畫中的知名角色在地圖中進行戰鬥，根據之前的預告片來看，遊戲中每個人的替身能力都做到了極好的還原。

當然還原替身能力的代價就是人均掛B，每個人都擁有相當厲害的超能力，比如布姐的拉鏈可以穿牆、DIO的時間暫停等能力全都被搞了出來。

這可能是近幾年JUMP漫改出來最吸引人的遊戲了，可惜是街機。

X博士點評：別光想著街機啊，快出PC、主機版，我一定買爆！

【本周新片推薦】

劃重點：《星球大戰9：天行者崛起》國內定檔 Netflix《巫師》美劇正式播出

在12月20日，《星球大戰9：天行者崛起》將會在國內全面上映，12月18日晚上7點則會開啟國內大規模點映。

這一部星球大戰是全新三部曲的終章，將由J·艾布拉姆斯執導，由黛西·雷德利、亞當·德賴弗、奧斯卡·伊薩克等演員出演。

而網飛帶來的美劇《巫師》也將在12月20日正式開播，首季共有8集，不知道「大超」能不能演出傑洛特的味道，當然X博士更期待的是和葉奈法的戲，安排個獨角獸吧。

只要別整部戲都在打昆特牌就行。
