刺客最頭疼受不了的法師英雄，安琪拉都知道，他讓刺客有來無回

2019-12-15 190條評論 4,418次閱讀 2980人點贊

在王者榮耀這款手游裡面有著定位為刺客的英雄，刺客擁有著對對方脆皮超強的威脅力和致命的殺傷力，但是他們同樣脆，只不過相對於射手和法師來說要肉一點而已。刺客的高風險和高收益讓無數玩家沉迷，他也需要極高的操作，當他們建立優勢秀起來時簡直是神擋殺神佛擋殺佛。當然萬物都是相生相剋的，刺客也有害怕的英雄，因為這些英雄的存在，使刺客無法隨心所欲。接下來，就為大家細數一下遊戲裡面刺客討厭的英雄

妲己

在低端局用妲己的玩家還是非常多的，裝備起來的妲己真的是秒天秒地秒空氣。前期由於傷害太低對敵方造成的威脅也不是很大，但是由於二技能可以魅惑所以對面還是有些頭疼的。妲己的大招cd非常短，在中後期只要被姐己魅惑到就是一套技能送給你，面對刺客也是如此，想要突臉問問手中的魅惑吧。建議各位在團戰中如果對面刺客還沒有露臉盡量不要用二技能，一旦用了生存能力會大大減弱，如果刺客頭貼過來切，妲己直接反打將他帶走

安琪拉

安琪拉是一位沒有任何位移技能的脆皮法師，但是卻有2技能的眩暈控制，1技能的高傷害，以及3技能大招的爆炸輸出，如果有刺客敢來侵犯，反手2技能控住，1技能消耗血量，接大招超遠距離直線傷害，想跑都跑不了，而且安琪拉的大招免疫一段時間控制效果，想斷她的大都斷不了

幹將莫邪

幹將莫邪的大招，給你一下，如果是後期，基本一下就死了，特別是射手或者是一些脆皮的刺客。在團戰中一般都要先追殺幹將莫邪，因為幹將不死，團戰的勝率基本沒的說。幹將在遠程的輸出上面有一定優勢，而且全部招數用完，團戰基本結束，而且勝率絕對高！所以，這個法師確實傷害很高，很多刺客也都不敢輕易的靠近他

張良

張良的控制技能非常的多，一技能和大招都具有控制效果。敵方刺客英雄不敢靠近他的原因主要還是因為他的大招特性，選擇的目標受到強制性的壓制，無法動彈。如果一個脆皮刺客被張良控制住，那麼只有死路一條了
