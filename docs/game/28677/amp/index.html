DNF：韓服100級版本即將開放，國服當前版本玩家應該提前什麼？

天天要聞
2019-12-22

目前韓服體驗服開放了100級，我們在體驗了一段時間之後，覺得適合提前讓國服的玩家去做一些準備，國服的一百級版本最快可能是在三月份上線，所以現在做準備是最好的。

準備新版深淵票

觀看我們之前發布的視頻大家應該也清楚了新版的深淵票是什麼，時間的引導石是一個全新的100級深淵模式門票，但是產量非常低，擊殺100級新副本領主掉落，但是數量非常低，目前來說比較實際的獲取途徑就是兌換。

兌換有多種渠道，1個史詩靈魂可以兌換20個引導石，20張深淵派對邀請函兌換1個，2個星空深淵靈石兌換1個，而刷一把新版深淵需要18個引導石，所以從現在開始我們需要提前準備大量的門票，因為升級裝備需要花費很長的時間，而深淵看臉有可能更快的畢業。那怎麼準備門票？也就是準備可以兌換時間的引導石的材料：史詩靈魂、深淵派對邀請函和星空深淵靈石。

而這些材料最直觀的就是通過攻堅商店中兌換，那麼攻堅商店中兌換的優先順序別是什麼？因為每周獲得的不渝之詠上限是4000個，而很多普通單號黨玩家可能換不到全部兌換完全，而考慮新版的裝備升級刻印不需要靈魂之源，所以最優先順序別兌換的是深淵派對邀請函。

深淵派對邀請函兌換每周：25X16=400個不渝之詠，共獲得2560張邀請函，每360張刷一把新版深淵，所以可以拿到7次的門票。

史詩靈魂兌換每周：100×30=3000個不渝之詠，共獲得30個史詩靈魂，兌換共20×30=600個時間的引導石，可以刷33次新版深淵。

而剩下的600個材料則用來兌換星空深淵靈石即可。因為攻堅商店每周刷新，所以現在開始就要每周將獎勵全部兌換完畢，多號黨建議先給要玩的全部角色先兌換邀請函，因為性價比比直接兌換狗眼高，畢竟100個不渝之詠兌換640張票，差不多能刷2次新版深淵了哦！

此外有多去獲取史詩靈魂，這一個獲取的方法不建議當前版本再去刷深淵，可以刷刷每日2次黎明裂縫看臉掉落。

如果購買挑戰書目前價格需要100萬一次刷一把，而選擇使用異形結晶直接購買哈林史詩分解，那麼只需要40萬金幣，這樣的途徑更好，畢竟大部分玩家號里都有上萬個異形結晶了。但說到異形結晶，其實可以給100級做更好的準備，因為哈林史詩可以刻印。

利用哈林史詩為100級做準備

第一個部分就是哈林史詩武器也可以刻印屬性，雖然每個角色只能兌換一次哈林武器，但可以直接利用裡面的鍛造券和強化券用於打造，到時候直接刻印給新深淵獲得的100級史詩武器上，不過目前測試需要對應的武器類型才能刻印，所以武器的部分建議玩家在更新後再兌換打造。這一個內容大家在看蝸牛直播的時候應該看到了哈林光劍的打造屬性是可以刻印到100級的史詩光劍上的。

但是刻印屬性越高，需要花費的紅色材料數量越多，紅色材料叫做刻印之石，需要智慧水晶1:1兌換，而智慧水晶需要智慧水晶碎片1:100兌換，水晶碎片僅有新深淵模式產出，一次10個這樣。但是多刷深淵就能弄一把13的武器，我覺得可以了。

那麼現在的階段玩家可以多製作傳說武器，並且將傳說武器進行強化13，如果成功強化13後直接升級為哈林史詩武器，再直接刻印給100級的史詩武器，可以省下非常多的金幣，強化傳送需要的金幣數量比史詩便宜很多哦！

此外現在階段可以準備傳說耳環打造13，然後升級為哈林為100級的史詩耳環或者神話耳環進行刻印，這樣可以省很多金幣。
