王者模擬戰冬冠杯：hero晉級十六強，estar竟然倒數第二

2019-12-19 1,399條評論 2,629次閱讀 620人點贊

關於王者榮耀的比賽，大家都知道剛剛結束的kpl秋季賽，還有之後馬上就要開始的冬冠杯的比賽。其實在這之間，還有一個賽事也是備受關注，那就是王者模擬戰冬冠杯的比賽。如今隨著比賽的進行，首批晉級十六強的隊伍也是新鮮出爐。作為在kpl創下三冠王朝的hero久競持續領跑，讓人意外的是WE和YTG竟然也是成績不錯！

王者模擬戰

王者模擬戰A組戰罷，四支隊伍晉級十六強！

32強分組

在此次王者模擬戰的冬冠杯上，共有三十二支隊伍，分為四組，爭取十六強的名額。而在A組則是被稱為死亡之組。有kpl的兩支冠軍隊伍estar和hero久競，還有KrKPL的強隊CW和MVP。所以這一組的比賽也是十分的好看。其中estar作為kpl的冠軍，更是世冠杯的冠軍，在人員儲備上也是有著別的隊伍比不上的實力，所以也是被大家看好的種子隊伍。然而結果出來之後，卻讓人有些意外！

hero久競領跑，A組第一晉級！

A組積分排行

在有著號稱死亡之組的A組中，hero久競真的是一騎絕塵，以總分71分的成績，直接A組第一晉級，這也是讓人看到hero久競的實力。在kpl的舞台上，hero久競曾經上演了從預選賽冠軍，到常規賽第一，再​到總冠軍的傳奇，更是一年三冠，讓人刮目相看。來到王者模擬戰的賽場上，hero久競依然繼承了英雄精神，以遙遙領先的優勢，直接晉級四強。這就是我們記憶中的hero久競，榮耀非凡的三冠王！

WE讓人意外，拿下A組第二！

we

在晉級的隊伍中，WE有些讓人意外，因為大家都知道，在kpl秋季賽中，WE的成績一落千丈，成為墊底的存在，就連老闆在直播中，都拍桌子並表示要在下個賽季大換血。然而沒想到到了王者模擬戰的舞台上，WE卻一掃kpl的陰霾，實力大增，以54分的成績，排在小組第二，也是進入到十六強，進行下面的比賽。而排在第三的則是鬥魚，總分51分！

YTG實力不錯，三名小將越戰越猛！

A組晉級隊伍

除了WE的表現讓人驚喜之外，YTG的表現也是非常不錯，拿到50分，也是驚險的進入十六強。YTG的風格大家都知道，是一支喜歡越塔乾的隊伍，打法兇猛，十分的讓人佩服。這不在王者模擬戰的賽場上，淺墨也是完美的繼承了這一點，結果自己只拿到5分。不過安然和軒崽實力帶躺，YTG也是成功晉級。在YTG晉級之後，網友們紛紛表示，能在死亡之組晉級真的不容易，三名選手和教練是真的辛苦了！

estar遺憾出局，倒數第二讓人震驚！

estar

在A組中，之所以被稱為死亡之組，除了有hero久競之外，還有estar的存在。在kpl的賽場上，estar是當之無愧的霸主，更是進軍LPL勢頭正盛。然而到了王者模擬戰的賽場上，卻被吊打，總成績倒數第二，只比MVP高一點。這非常的讓人震驚，作為一個在強隊，更是要處處都強，如今在王者模擬戰的賽場上，拿到這樣的成績，真的讓人很意外，所以對於estar的小夥伴來說，真的要加油了！
