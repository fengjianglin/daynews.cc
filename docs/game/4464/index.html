9.19斯維因成下路新貴 力克三大主流射手

韓服高端局的節奏一直都非常快，在這個刺客中單與法系打野盛行的版本更是讓一眾ADC的生存環境差之又差，於是卡莎、EZ、盧錫安這三個前期有一定作戰能力，後期生存率又高的英雄則成為了韓服高分段的常客，於是9.19版本更新後又誕生了這麼個黑科技，成為了專治這三個英雄的存在，他就是我們今天的主角–斯維因。

斯維因走下路的優勢

斯維因之所以能夠在這個版本中崛起，是因為當前版本的下路格局基本是由卡莎、EZ、盧錫安這三名英雄組成，而斯維因又對這三名英雄有一定克製作用，於是斯維因這個”怪咖”開始當起了下路皇帝。

EZ、卡莎、盧錫安這三個都是普攻距離較短的英雄，而這個攻擊距離也是斯維因最喜歡的攻擊距離，不僅E技能更容易命中二段，Q技能也更容易打出多段傷害，所以斯維因在打這三個英雄時簡直不要太舒服。

符文搭配：主宰+巫術

電刑：斯維因的技能機制非常容易觸發電刑，我們選擇電刑來提高我們的爆發和線上消耗能力。

惡意中傷：W和E都能輕鬆觸發惡意中傷。

眼球收集器：眼球收集器比殭屍守衛和幽靈魄羅更適合斯維因。

貪慾獵手：配合大招，斯維因在團戰中能站得更持久。

靈光披風：開大後更快速地進入戰場。

超然：冷卻縮短是斯維因的剛需。

技能加點：主Q副E

一級點出E技能AEA就能觸發電刑，然後我們主升Q，Q技能是我們的主要輸出技能，W主要起到一個看視野和遠距離收割的作用，副升E可以讓我們更頻繁地進行摸獎，有大點大。

裝備選擇

作為一個法坦定位的英雄，時光杖當然是斯維因的首選裝備；其次我們要儘快地做出中亞，這是我們的核心裝備之一，有了中亞我們就可以肆無忌憚地開著大招衝進人堆中，不怕一瞬間被秒掉；就算血量很殘，我們也能利用金身這2.5秒的時間回一大口血；第三件裝備我們可以從鬼書和蘭德里的折磨中進行選擇，如果對面英雄的回復能力較強或者說沒有什麼抗性裝備，我們就選擇做出鬼書來提升我們的爆發；如果對面的英雄比較肉，不能一瞬間將其秒掉的話我們就做出蘭德里的折磨，保證我們的持續輸出能力。

盧登的回聲是一件劍走偏鋒的裝備，他可以進一步提升我們的爆發和前期作戰能力，但是這是在我們犧牲了坦度的情況下實現的，若是覺得對面的傷害對我們的生存空間構不成威脅的話，也可以選擇出盧登的回聲。

對線思路

斯維因前期對線就非常強勢，若是配上有控制的輔助那將如虎添翼。我們1級學E，對面補兵的空隙我們就可以用E技能進行騷擾，一旦命中敵人，我們EAA打出電邢的傷害，對面是完全吃不消的；如果輔助是末日這種帶有硬控的英雄我們可以等他Q先手，然後用我們的被動拉回來再E，這樣可以保證E技能的命中率，也能最大化控制時間，對面被控的這段時間只能活活挨打，沒有一點脾氣可言。

團戰思路

團戰開始前，我們可以用E技能進行摸獎，只要摸中一個敵人，WQ打出電邢很容易讓對方減員，這時候團戰也就開啟了，我們開啟大招衝進人堆，儘可能讓大招打到更多的人，同時要保證我們的血線健康，必要時候開啟金身保命，金身這段時間我們的大招也是可以幫助我們回復血量的，我們存活得越久，團戰打贏的幾率就越高。

連招技巧

1、E+A+A

1級時我們可以用EAA連招觸發電刑，這是斯維因對線最基本的連招。

2、E+W+A+Q

3級後，我們E到人時預判對面拉過來的位置放W，用平A把敵人拉到我們的W上，這時候的距離也足夠讓我們Q到人，一套下來，不死亦殘。

總結

以上就是斯維因走下路的玩法解析，賽季即將結束，希望這篇文章能對小夥伴們的上分有幫助哦！
