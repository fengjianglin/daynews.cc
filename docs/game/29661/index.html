沙雕網友你們夠了！火爆全球網紅表情包，台灣人竟拿來做遊戲？

2019-12-25 1,486條評論 3,723次閱讀 908人點贊

有圖有真相，沒圖說個蛋！大家好，這裡是每周一更新的《圖個好遊戲》，我是正驚小弟！又到了我們看圖挑遊戲的日子，廢話不多說，上圖！

《無雙大蛇3：終極版》：爽快割草的一鍋燉

遊戲平台：PC/XboxOne/PS4

《無雙大蛇3：終極版》上架了！大蛇系列一向以結合真三國無雙和戰國無雙為核心賣點的割草無雙類遊戲。前作《無雙大蛇3》是一款以奧林波斯的神明「宙斯」所創造出的異次元世界為舞台，《真·三國無雙》及《戰國無雙》兩系列的英傑們與眾神活躍的戰術動作遊戲。

此次無雙大蛇3會有170位人物出場，然而比起上作只有5個新增人物，此外不會有真三國無雙8的新增人物。遊戲還將加入新模式無限模式，以及此前作為DLC推出的挑戰模式。不過遊戲這價格，也是相當感人……目前主機版已經上架，Steam將於2020年2月14日正式發售。已擁有《無雙大蛇3》的玩家都可以購買「Ultimate升級包」遊玩。

玩家評論：

——就在普通版的基礎上就加了那麼幾個角色，升級終極版就要接近400塊錢，光榮吔屎啦

——給各位想買觀望想買的提個醒： 遊戲不包含季票、除新角色以外的武器包、服裝包和神器包DLC。也就是說縱使Ultimate加入了新的內容，這些增加遊戲體驗的東西仍然屬於需要額外消費的增值服務。

——這代應該是最爽的無雙，而且人物也極多，重複性並不高，新增的角色也很棒，人物可以9轉，屬性很誇張，反正肝的挺開心的，當然不喜歡無雙或無腦的可能一星都不想給。

《動物之鬪》：網紅動物表情包真被做成遊戲了？

遊戲平台：PC

這記得這些紅極一時的動物們嗎？台灣的一家公司，就把這些網紅動物做成遊戲，讓它們站上決鬥的擂台後，將會迸出何等激烈的火花呢？肌肉右鉤柴、絕對王者狐、霸氣貓、烏鴉猩、魔法松鼠…等待著你的參戰！

據官方說明，本作僅需四個按鈕操縱，並且採用一鍵出招系統，只要按著方向鍵加上技能鈕就能施展必殺技!完全沒玩過任何格鬥遊戲的玩家也無須擔心更可以透過 “簡易連段”的輔助模式、用最順暢的方式享受痛扁對方的過程！沒有複雜的操作、打起來輕鬆有趣！

玩家評論：

——遊戲很搞笑，很歡樂，角色少了點， 高難度有點難打，可以本地2人，也可以聯網，現在人少可能連不到， 另外遊戲很流暢反正我不卡

——很簡單、很沙雕、很好玩。雖然作為格鬥遊戲就很菜了，但是作為沙雕遊戲和朋友玩還是挺有趣的

《魔女之泉4》：「老父親們」又可以養新女兒了！

遊戲平台：手機

你們的女兒來了！《魔女之泉》系列是一款養成類RPG遊戲，系列在2016年推出第一部以來受到了很多玩家的關注，並且每部作品都保持著不錯的口碑。而《魔女之泉4》講述了小君主摩卡莫莉的故事，這位魔女君主統治著烏爾坡大陸，遊戲還未上架就有20多萬人預約，一上線就被蘋果官方推薦，可見遊戲火爆程度！

這一次和女王身份相匹配還有部下指揮系統：摩卡莫莉可以收服部下，給他們武器，讓他們為自己出戰。在魔力覺醒的時候，她可以自己使用這些武器，動畫也非常酷炫。這一次，魔女又會帶來怎樣的故事呢？

玩家評論：

——重點在於男僕啊！！！男僕也太香了！！！衣服做飯外加陪練樣樣精通 嫁人就嫁小男僕 你值得擁有！！

——年底大作預警？？？從潛力來講可以給4星。建模和貼圖都比上一代更精細，也還是那個認真講故事的遊戲，這一代是摩卡莫莉的故事，也有機會操作前3代的魔女一起戰鬥。

《泡沫冬景》：在自己的一生中，每個人都是主人公。

遊戲平台：手機

在1988年泡沫經濟下，看似繁華的日本東京，高樓疊幢，霓虹耀目，卻在不起眼的地方，消逝和更替著屬於舊時代的痕迹，所有人的境遇與無奈，都在人間浮華下顯得格外渺小。來自中國的落榜少年與日本鄉村少女相遇，誤解，語言不通，略帶苦澀，卻溫暖人心，這個發生在1988年冬季的故事，名為——《泡沫冬景》。

故事的主角，就如清晨擁擠公交上飽含倦意的少年，亦或超市兼職熱情洋溢的少女、職場里喋喋不休的上司，甚至是閱讀著這段文字的你，都有著屬於自己的故事。而在自己的一生中，每個人都是主人公。

玩家評論：

——比起遊戲，這更像是一部帶著配音的漫畫。畫面尚可，甚至有的景色畫面可以說上些許驚艷；劇情平凡的日常中帶著波瀾，有的角色切換個人覺得處理的有些僵硬，玩家和遊戲互動感極低，甚至可以說是十八塊錢買的不是遊戲，而是一個故事。

——連肝八個小時玩完了這部優秀的作品，不得不說這劇本寫的確實優秀，每個人物也刻畫的有血有肉啊。

《窪》：放空心靈的治癒之旅

遊戲平台：手機

騰訊極光遊戲出了一款音樂解謎遊戲——《窪》。這是一款能隨時陪伴你的音樂解謎遊戲，和簡單的生活一樣簡單的畫面，和平和的心境一樣平和的音樂。釋放一天的疲憊，靜下心來聽我的旋律，身居鬧市而無車馬喧，流水細語解謎闖關，放空心靈的治癒之旅從這裡開始。

遊戲的玩法也很「簡單」，好吧，其實有一定的難度。首先傾聽琴聲與和弦，感受旋律和節奏；再通過輕點水面以製造波紋；漣漪所經之處會激活琴聲，將琴聲按順序與節奏還原，即可過關。

玩家評論：

——整個意境和音樂完美配搭，讓人沉浸在一個優雅放鬆的環境中，後面幾關略有難度。如果你不喜歡這種佛系的遊戲的話，可能會覺得有點無聊。

——唯美的畫風搭配輕靈的音樂，是安然入睡的最佳伴侶！1元錢你買不了吃虧，1元錢你買不了上當！
