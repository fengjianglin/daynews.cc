爐石傳說新版本全員超模？王師傅打出超多5分！網友：我信你個鬼

2019-12-11 235條評論 3,111次閱讀 908人點贊

爐石傳說新版本「巨龍降臨」發布之初就有很多玩家表示這個版本將是全員超模的一個版本，整體的強度非常高，甚至有玩家表示你不超模個1-2費都不好意思說你的是新版本的卡牌了。從王師傅的打分情況也能看出，畢竟1分卡的比例大大減少同時5分卡的比例大大增加，但就九大職業的職業卡牌就有13張5分卡，簡直就是強無敵！

第一張5分卡是盜賊的橙卡隨從，6費4/4效果是消滅一個隨從極其所有的複製。講道理這張卡牌真的強，下限是消滅一個隨從但上限是直接摧毀對方的體系。再就是對於盜賊這種節奏職業來說，這張卡牌的戰略意義也非常大，5分必須的。

第二張5分卡是薩滿的藍卡法術，1費效果是過載1點使你的隨從獲得+1/+1。對於薩滿來說確實很好用畢竟薩滿的鋪場能力非常不錯，然後還能使我們的圖騰具有一定的攻擊力，過載1費也能配合部分卡牌打出combo，整體來看也還不錯！

第三張是薩滿的橙卡隨從，5費5/5效果是該卡牌在手牌中時會隨機變成一張傳說隨從的5/5複製。這就很強大了呀，畢竟是屬於裸下不虧若是能隨到其他有特效的隨從簡直血賺，搏一搏單車變摩托！

第四張是薩滿的迦拉克隆體系，該卡牌的強大之處在英雄技能的強大，薩滿迦拉克隆的英雄技能是2費召喚一個2/1的突襲元素！不管是解場還是鋪場都非常不錯，同時還有一系列的配合，有一說一薩滿的迦拉克隆算是最強的迦拉克隆了！

第五張5分卡是術士的橙卡隨從，7費4/4將我們的手牌上線提升至12張還抽4張卡。對比盜賊的疾跑簡直超模到姥姥家了，個人也是非常看好這張卡牌，只能說很強！

第六張5分卡是戰士的橙卡隨從，8費12/12效果是攻擊所有的隨從。王師傅表示第一眼看到這張卡的時候覺得一般但越看越強！特別是在特定的情況下比大部分的解場法術都好用，實戰應該還不錯。

第七張5分卡是戰士的橙卡武器，3費2/3效果是在英雄攻擊之後從牌庫抽一張海盜。海盜戰重回江湖？講道理只要這張卡牌不沉底是真的強，不僅能補充手牌還能精準的壓縮卡庫，等一個海盜戰！

第八張5分卡是牧師的橙卡隨從，8費8/8效果是使用你對手上回合使用過的卡牌。這張卡牌只典型的裸下不虧但只要觸發效果就血賺，而且我們還能權衡利弊自己去選擇到底用不用，最後王師傅表示這張卡牌打5分的另一個原因是這張卡牌是他發布的！

第九張5分卡牧師橙卡隨從，3費3/3效果是選擇一個對方隨從亡語是召喚一個全新的選擇隨從的複製。這張卡牌的強大之處在於費用低，我們在後期可以選擇自己主動觸發亡語，同時還具備一定的戰略意義。

第十張5分卡是法師橙卡隨從，5費2/8效果是若手裡有龍牌則發現一張升級過的法師法術。相信大家都看到了這些升級的卡牌是多麼OP，同時要注意這裡是發現加上費用不高，因此我們在很多情況下可以考慮是斬殺還是保命，實際的作用肯定非常大！

第十一張5分卡是小德的紫卡法術，0費使牌庫中的隨從獲得+2/+2同時消耗增加1費。這張卡的強大在於他本身是0費，也就是說我們不管在什麼時候都能使用而且不用佔用我們當前回合的費用，再者他這個效果在初期提升非常大，很強！

第十二張5分卡獵人橙卡隨從，6費7/6效果是使左右地方隨從生命值變為1。只能說如果不是獵人的橙卡那這張卡牌便是神卡，身材稍稍超模效果也很超模簡直就是這個版本的寫照，反正帶了也不虧我玩獵人肯定會帶！

第十三張5分卡是騎士的橙卡隨從，3費3/3亡語是使你手牌中的龍獲得+3/+3以及此亡語。大體的意思和橙錘一樣，我們可以持續的BUFF和亡語，同時這張卡牌本身的費用不高我們可以儘早打出，只要觸發就血賺！

不知道小夥伴們對於這些5分卡有什麼自己的看法呢？你們會合哪一張呢？歡迎評論討論！
