酒館戰棋：聖盾劇毒狂戰斧來了？玩家的DIY真是腦洞大

2019-12-10 1,109條評論 4,608次閱讀 2745人點贊

如今的酒館戰棋已經誕生出了各個流派，每個流派之間也各有優劣，總有玩機雜的望著玩魚人的全員劇毒眼饞，心裡想著要是能給狂戰斧貼個劇毒豈不美哉。

這麼bug的卡牌暴雪當然不會放出來，但架不住玩家們腦洞大，一個個的都設計出了各種強的沒天理的DIY，接下來就讓小編帶大家一起看看吧。

能給狂戰斧貼劇毒的隨從來了！這是由貼吧老哥設計出的戰棋隨從，戰吼效果也是簡單粗暴，使一個友方隨從獲得劇毒，這要是給狂戰斧貼上，一刀下去還不是毀天滅地！也只有玩家敢設計出這樣的bug牌了。

這是由小編我DIY的戰棋隨從格魯爾，延續了他原本的效果，每回合結束時獲得加一加一，小拳頭光加攻擊不加血量實在太弱，那就用格魯爾吧！當然六星的級別也註定這樣的格魯爾既不能打工也不能致勝，只是純粹的娛樂卡。

最近還有一位玩亡語流的老哥受女王的英雄啟發，乾脆把希爾瓦娜斯作為隨從DIY進了酒館戰棋，效果還是一樣，亡語隨機控制一個敵方隨從，金色加倍。那要是配合上瑞文戴爾，那真是把對面全家都偷空了！那場面想想就快樂。

當然，要是被偷的是自己那可就不一樣了。

高階祭司阿門特要是放到戰棋里確實有開發的潛力，只要想辦法把他的血量疊高就等於有了無數的大屁股，只不過這樣的隨從要是星級設定的太高就沒有意義了，後期誰看得上七血的屁股啊，要是把他設定成兩三星的隨從，那他一定會被瘋搶。

看別的DIY也就圖一樂，真要學習設計師還得看金手指納克斯！

金手指納克斯要是放到酒館戰棋里，殺傷力絕對和面對戰士的破甲騎士有的一拼。高貴的野獸屬性非常方便疊buff，很容易就能堆到超過四十攻，那這一拳要是下去，對面的英雄也就瞬間爆炸了！

恐怕這樣的卡牌也就會在酒館戰棋停服的那一天才會放出來讓大家過把癮，現在也就只能停留在玩家的DIY里了。

看了以上幾張DIY不知道大家有沒有感到自己的創意在燃燒呢？要是小夥伴們也喜歡這樣的腦洞的話不妨關注一下我們，每期都會帶來不同的精彩內容哦！
