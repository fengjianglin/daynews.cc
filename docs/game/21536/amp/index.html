CSGO-震中杯：JAME轉生VP，雙加時遺負Mouz

天天要聞
2019-12-19

震中杯的第二個比賽日，兩支本土作戰的俄羅斯戰隊——forZe和VP，在首輪比賽中分別同EG與Mouz狹路相逢。forZe方面，雖然這支隊伍在過去的幾個月中有所成長，但很明顯，現在的他們還不是EG的對手。其中比較詭異的是，forZer在EG挑選的Mirage上堅持到了常規時間的最後一個回合。而到了自己選的Nuke上，foZer在上半場就直接拉跨。比賽結束的時候，forZer僅有四分入賬，如此巨大的分差不得不令人懷疑，也許一切都在EG的掌控之中。

Mirage的上半場，forZe取得了三比零的優勢開局。不過在第四個回合，Stan便帶領隊伍展開了一連串的反擊。這個回合，forZe的幾個CT有點急功近利——喜歡中路前壓的隊伍不是沒見過，但像forZe這麼大膽的真沒幾個。偷了Brehze的替補「crush哥」後，forZe的人還不撤退。這個時候，Stan抓住機會在近點秒掉了FL1T，然後打死了拱門的facecrack。almazer想替隊友報仇，但自己的位置卻被Stan鎖定。憑藉著Stan的三殺，EG拿到了整個BO3的第一分。分數從三比零來到三比三，就在EG即將實現分數反超的時候，xsepower不樂意了！幾個月前的柏林Major上，xsepower的狙擊槍令人印象深刻。第七個回合，xsepower的A點警家大狙把EG打的四分五裂！因為道具上的失誤，EG的人從一開始就進入了forZe的包圍圈。一個、兩個、三個、四個，來自xsepower的瘋狂殺戮！forZe重新掌控遊戲的節奏。

上半場的最後時刻，forZe擴大了自己的優勢。這個階段，facecrack和Jerry先後有過兩次不錯的發揮——前者是M4的中路三殺，後者是AK的前點三殺。不過在上半場的最後一個回合，來自Stan的1v2殘局卻讓forZe嗅到了一股不妙的味道。上半場結束的時候，forZe領先EG三分。下半場開始後沒多久，Ethan和CeRq便突然絕醒！第21個回合，Ethan在殘局砍下兩個人頭。之後的幾個回合，CeRq的大狙讓forZe的A點戰術全部泡湯。一連八個回合，forZe雖然時不時地能打入包點，可他們就是無法得分。比賽結束，比分定格在16比14，EG獲勝！

如果說上個BO3是一場實力懸殊的對決，那麼到了今天的第二場比賽，VP卻讓我們見識到了俄羅斯男兒的血氣方剛——打住！這裡必須要糾正一下上文中的錯誤。三天前，就像當年EG簽NRG一樣，VP簽下了AVANGAR的原班人馬。現在的VP，從教練到選手都是AVANGAR的人，其實和俄羅斯沒有任何關係。回到具體的比賽中，就算換了一個隊伍，但Jame大神的遊戲風格卻依舊獨樹一幟。圖一Inferno上，在隊友的掩護下，Jame多次上演保槍傳奇。說笑歸說笑，在下半場的混戰中，Jame的狙擊槍沒有讓隊友感到失望。Mouz作為EPL新科冠軍，其實力自然是蓋過VP一頭。拿下VP選擇的Mirage後，比分被Mouz追平。

Inferno的最後一個回合，Mouz其實一度看到過勝利的希望——來自woxic的手榴彈和燃燒彈先後兩次幹掉了殘血的匪徒，Jame進入了1v3的不利殘局。因為勝利唾手可得，Jame難得地沒有選擇保槍！B點的回防戰，被閃光彈半白的Jame憑藉著意識和運氣打出了一槍完美的一箭雙鵰！兩個隊友突然就沒了，留下frozen一個人在包點凌亂。輸了這局就意味著輸了這圖，雷包是不拆也得拆！Jame正是抓住了frozen的心理，在真拆快結束的時候突然出擊。果不其然，frozen慌了！Jame果斷地收下人頭，將比分定格在16比11。當然，之所以VP能贏下比賽，也不完全是Jame一個人的功勞。上半場，「工具人」SANJI撿起對手掉落的553後化身EliGE，在A點的防守戰中連殺三人。到了第11個回合的關鍵長槍局，手持AK的buster在三箱後躲了一萬年——Mouz以為那裡沒人，剛準備下包卻被buster打了個措手不及！

賽前，估計Mouz萬萬沒想到，自己居然會在Inferno上輸的那麼慘。於是到了圖二Inferno上，他們打起了12分的精神——特別是woxic！作為隊伍里的頭號大腿，woxic在上張圖上有些夢遊。好在這一次，woxic將功補過。第26個回合，woxic用狙擊槍粉碎了VP的B點回防。站在二樓的woxic居高臨下，先打超市再打B連再打包點！三槍完美狙殺，Mouz手握賽點。

VP和Mouz的決勝局是今天晚上最精彩的一場比賽。地圖沙漠二，扮演匪徒的VP在上半場砍下10分。換邊後，輪到Mouz扮演匪徒，他們在frozen的帶領下將比賽拖入加時階段。第一個加時，雙方戰的難捨難分。直到第二個加時的最後時刻，Mouz以微弱的優勢鎖定勝局。第27個回合，就在Mouz的夾B戰術即將失敗之時，frozen挺身而出！中門警家附近的煙霧彈混戰，frozen在回頭的一瞬間用553打出了準確無誤的掃射轉移。一連拿下四個人頭，VP偃旗息鼓。

Mouz和VP在上場比賽中戰的酣暢淋漓，本以為今天的最後一個BO3也是一場火星撞地球的大戰，只可惜，EG的狀態並不如我所願。和forze那場比賽非常相似的是，在對手選擇的火車上EG反而拿到了更多的分數（11分），而到了自己選擇的Inferno上，EG在比賽結束時只拿到了可憐的四分。有一說一，Mouz不愧為EPL和巔峰聯賽的雙料新科冠軍，水平比少人的EG高了不止一星半點。而相比全員就位的時候，今天的EG也不得不就具體的情況重新安排了戰術——Stan的位置更加靈活，Ethan的擔子則變得更重。火車下半場的手槍局，Stan在隊友完成下包後為EG打贏了1V3的殘局。別的不說，至少Stan的走位非常刁鑽。開第一槍前，Stan繞到了Ropz的後面。緊接著，Stan又在正面槍法對決中搞定了Woxic。最後一個大表哥想拆保，但他也逃不出Stan的魔爪。不過在接下來的幾個長槍局，Mouz的人還是擋住了EG的進攻。如果不是Ethan在第25個回合用CZ製造了一波三殺，比賽會結束的更加迅速。

今天，就是本次震中杯的第三個比賽日，即小組賽的最後一天。下午五點這場，是EHOME和Heroic的中外對抗，祝EHOME好運。因為小組賽只會淘汰一支隊伍，只要EHOME打贏了這個BO3，就能獲得晉級淘汰賽的資格。接下來的一場，是forZe打VP，就今天的比賽來看，當然是VP的勝算更大。剩下來的兩個BO3和晉級無關，只是小組二三名的排位賽。就個人而言，我更看好NAVI和EG——Brehze的歸來將讓EG重回正軌！
