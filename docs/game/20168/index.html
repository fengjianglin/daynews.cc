電競行業中心亞特蘭大：AEV為亞特蘭大首支專業電競戰隊建新基地

2019-12-18 904條評論 1,863次閱讀 592人點贊

​喬治亞州州長Brian P. Kemp近日宣布，亞特蘭大電競投資公司（Atlanta Esports Ventures，簡稱AEV）將為旗下的職業電競團隊亞特蘭大君臨隊（Atlanta Reign）建立新的戰隊基地。基地位於亞特蘭大中城區，將創造45個新工作崗位。該項目總投資達1億美元，用於持續發展亞特蘭大的電競生態系統。

亞特蘭大君臨隊是亞特蘭大首支職業電競戰隊，是《守望先鋒》聯賽的參賽團隊之一

喬治亞州州長Kemp先生表示：「新基地的建立，將加快亞特蘭大君臨隊的發展步伐，我們對此感到非常興奮！」他強調：「亞特蘭大正迅速成長為美國電競行業發展的中心。這一項目的實施充分展示了喬治亞州在科技行業和娛樂行業的實力與活力。」

在亞特蘭大舉辦的一年一度的電競賽事

亞特蘭大君臨隊是亞特蘭大第一支職業電競戰隊，曾代表大西洋賽區參加《守望先鋒》聯盟(Overwatch League，簡稱OWL)的比賽。該隊所屬的亞特蘭大電競投資公司（AEV），是由總部位於亞特蘭大的跨國企業Cox Enterprises與金融諮詢公司Province Inc.於2018年成立的合資企業。

亞特蘭大君臨隊（Atlanta Reign）隊標

AEV致力於為亞特蘭大帶來最好的電競比賽，擁有《守望先鋒》和《使命召喚》聯賽（Call of Duty league）的特許經營權。今年5月，AEV曾表示將加入以亞特蘭大為基地的新城市《使命召喚》聯盟。在君臨隊新基地近日曝光的同時，AEV也宣布，該公司擁有的另一支電競團體——亞特蘭大FaZe（Atlanta FaZe）成為《使命召喚》指定戰隊。

亞特蘭大FaZe隊的隊標

「喬治亞州為亞特蘭大的電競行業提供了超乎想像的、大力的支持和幫助。」AEV總裁兼首席執行官Paul Hamilton表示：「我們與地區政府部門緊密合作，不僅創造了許多行業內的工作機會，吸引了眾多的遊戲玩家，也讓我們確信，對於電競項目來說，亞特蘭大是美國最好的發展基地。亞特蘭大欣欣向榮的科技產業和娛樂產業進一步證明，亞特蘭大是電競組織及科技精英能夠紮根的沃土。在這裡，我們更容易獲得成功！」

憑藉多年舉辦電競比賽的經驗，亞特蘭大在競技遊戲領域獲得了長足發展。伴隨著遊戲開發和科技產業的蒸蒸日上，亞特蘭大正成為遊戲開發者和遊戲玩家心中的嚮往之地。

亞特蘭大正在成為電競之都，將進一步豐富地區電競和遊戲活動，建立良好的電競生態系統

「亞特蘭大都會區的電競行業已迅速成為我們地區最熱門的科技領域之一，電競生態的發展獲得持續投資，這令人十分欣喜！」亞特蘭大都會區商會（Metro Atlanta Chamber）會長兼首席執行官Hala Moddelmog分享道：「 AEV宣布為亞特蘭大君臨隊和其他電競戰隊建立基地，這將幫助亞特蘭大在內容消費的供給方面保持領先，鞏固我們已有體育社群的強勢地位。

電競賽事現場

亞特蘭大投資（Invest Atlanta）總裁兼首席執行官Eloisa Klementich博士表示：「喬治亞州的視頻遊戲產業有150多個製作工作室，是這裡最熱門的產業之一，每年帶動的經濟效應達7.5億美元。」 他補充道：「由像亞特蘭大君臨隊這樣的團隊帶領我們的電競社群日益壯大，為城市新一代的數字化領域創新者創造了更多商機和就業機會。我們很高興看到亞特蘭大君臨隊能在新的基地訓練和成長，期待君臨隊在即將到來的比賽中擊敗對手。」

亞特蘭大君臨隊（Atlanta Reign）

喬治亞州經濟發展署署長韋宏森（Pat Wilson）對於AEV公司的投資決定表示歡迎：「恭喜AEV和亞特蘭大君臨隊將新基地落戶在亞特蘭大中城區。」 他表示：「亞特蘭大市已成為數字網路和科技行業增長的中心，也是發展電競行業的最佳城市。我也要祝賀我們所有的經濟發展夥伴，這些激動人心的成績因他們的努力才得以實現。」

關於喬治亞州經濟發展署（GDEcD）

喬治亞州經濟發展署(GDEcD)是本州主管經濟宣傳與推廣的部門。作為喬治亞州經濟發展的引領機構，主要負責吸引外商投資，促進現有企業發展，提供職業教育培訓，拓展本州產品的新市場，吸引國內外遊客，促進藝術、電影、音樂、動漫和互聯網娛樂業等文化產業的發展，並為本州經濟發展進行規劃和提供資源。www.GeorgiaBusiness.cn
