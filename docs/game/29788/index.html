王者榮耀冬冠分組抽籤確定，KPL“御三家”上籤，DYG.JC與TS內戰

2019-12-24 2,958條評論 3,938次閱讀 154人點贊

各位王者玩家大家好，我是小雨。#百度APP遊戲年度票選活動#冬冠席位賽最後一天的比賽正式落下帷幕，最終6勝1負的TS和5勝2負的eStarPro攜手晉級淘汰賽。TS輸掉的一場正是輸給了e星，而e星則分別輸給了Hero久競和RW俠，當然，輸給RW俠純粹是因為已經確定晉級，抱著隨便打打的心態。

XQ遺憾出局

最為可惜的便是前兩天比賽就拿到5勝1負的XQ，他們同樣是5勝2負的戰績，不過由於跟他們e星之間的勝負關係，無緣最終的淘汰賽。XQ在本次席位賽展現給大家的東西已經很多了，不過年輕的他們最終還是沒有抵擋住老道的eStar和更加年輕的TS。這兩支戰隊也正是最終出線的兩支隊伍，可以說XQ輸得也不冤。

RW俠輸給自己​

最令人無法接受的便是RW俠了，他們成為本屆賽事唯一一支輸給KRKPL戰隊的KPL戰隊。他們的大意和輕敵使得他們最終落敗，也可以說他們是輸給了自己。如果他們擊敗了CW，他們也是有機會爭取晉級的。可惜沒有如果，他們只能把晉級名額拱手送給e星。至於Hero久競，不知道是自己作死還是實力使然，不想評價。

在TS和eStar從席位賽突圍會後，冬季冠軍杯淘汰賽8個名額正式確定。跟足球比賽類似，冬冠淘汰賽也是用抽籤的方式來確定分組對手的。淘汰賽總共分為兩個半區和4個分組，A組和C組一個半區，B組和D組一個半區，然後兩個半區最終的勝者爭奪冠軍。在最後一場席位賽結束之後，英凱、瓶子和拖米三人進行了冬冠淘汰賽分組抽籤儀式。

大家都知道，英凱、瓶子和拖米三人都是KPL圈內有名的「毒奶」，對於讓他們來做抽籤嘉賓，觀眾們還是比較介意的。不過沒辦法，既然聯盟已經定下來了，也改變不了，只能期待強隊之間不要那麼早遇到。

​結果還算比較能接受，eStarPro、QGhappy和AG超玩會三支戰隊首輪分別對陣KRKPL的三支戰隊SLT、GOG和ROX。而剛剛晉級淘汰賽的TS則需要跟秋季賽季軍DYG.JC進行KPL內戰，這場比賽也將會是首輪最精彩的一場。

冬冠淘汰賽將於12月26日下午15點正式開打，大家記得觀看哦。
