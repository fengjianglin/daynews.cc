LPL選手韓服排位：Xiaohu狀態神勇沖千分，FPX雙上單或輪換出場？

天天要聞
2019-12-16

S9全球總決賽結束已經過了一個月，經歷了如火如荼的轉會期，大部分戰隊和選手的去向也已經塵埃落定。德瑪西亞杯開賽在即，各隊也將在德杯做出第一次的陣容試煉。

其中，S9新王FPX是最受關注的隊伍之一，其他例如S8冠軍iG、重磅引入Peanut的LGD等戰隊也是熱門話題。本文將從部分選手的韓服排位情況簡要分析其近期狀態。

第一位本地選手——國產中單Doinb

Doinb在奪冠後也在一如既往地用心直播，在直播中依然是喜歡和Canyon、Kanavi等「韓國野爹」進行雙排。Doinb的英雄池仍舊是難以預測的，除了比賽中常用的亞索、瑞茲外，似乎是為了符合自己「擅長刺客」的形象，奇亞娜、阿卡麗等刺客英雄在Doinb的排位中也常有登場，其中剛剛改版的皎月更是短期內達到了100%的勝率。近期比賽中，這個英雄應該也會是各個戰隊非ban必選的一手。

直播官宣的Khan，或與GimGoon輪換上場？

前SKT上單選手Khan經歷了一次獨一無二的官宣：因在Doinb直播時讓Doinb幫忙點外賣而出現在攝像頭中。

Khan在S8前半年一度被譽為世界第一上單，但洲際賽不敵RNG後便日漸低迷。來到SKT後，Khan似乎煥發了自己的第二春，在LCK聯賽、S9全球總決賽中也有超級Carry的表現，此次來到LPL的冠軍隊伍，與冠軍上單GimGoon之間又會做出怎樣的輪換呢？

從排位記錄中來看，正如同當前版本，Khan更加偏愛Carry型上單。使用的英雄大多為刺客、法師和戰士，相比來說純輸出英雄使用得比戰士更多，從戰績來看，也是阿卡麗、凱南、盧錫安等英雄勝率更高。

而GimGoon選手的英雄池和Khan雖有部分重合，但兩人的風格還是截然不同的，GimGoon最近與Doinb進行了多場雙排，近期使用劍魔也是打出了一波小連勝。

目前FPX的三人都是宗師段位。雖然兩位上單都是95年生的老將，但可以看出二人狀態都很不錯，再加上賽場上互補的打法，則可以形成一個幾乎沒有短板的上路格局。若是可以合理運用二人風格，相信會產生1+1＞2的效果。

Xiaohu：王者近千分，會再次爆發嗎？

S9小組賽上小虎可謂是經歷了一次職業生涯低估。飽受詬病的比賽發揮，以及小組賽第二輪中那兩個刺眼的傷害數值，都令粉絲不禁懷疑，那個MSI上風光無限的RNG不會回來了嗎？

然而從Xiaohu最近的排位來看，他的狀態似乎恢復了神勇，使用4個英雄在成功打出6連勝，且基本都是數據完美的Carry局，使用剛剛改版的皎月時，也有著出色的發揮。

RNG剛剛告別了Karsa，Uzi能否繼續首發也未成定數，新賽季陣容還未公布，粉絲們能做的只有等待和期許。但Xiaohu的幾頁連勝也給了粉絲不少信心，或許我們可以再相信Xiaohu一次？

LGD重磅引援Peanut：豹女盲僧能否帶領LGD闖入季後賽？

LGD在11月就官宣引入前GEN打野Peanut，算得上是相當重磅的引援了。上賽季LGD的成績難以令人滿意，Peanut在採訪中表示，自己的目標是首先進入季後賽，最終目標是進入全球總決賽。Peanut的到來會為LGD帶來怎樣的變化也是眾多粉絲期待的。

Peanut的排位一直是保持在高分的。目前以1235分排名韓服第24，常用英雄盲僧、千珏、豹女等，似乎又回到了S6那個意氣風發的銀髮少年。

Peanut的戰績較為起伏，連敗連勝交替。但大部分時候即使沒能贏下比賽，Peanut也可以打出較好的戰績。勝利的比賽中，則鮮有躺贏局，每次都對勝利做出巨大貢獻。這次德杯將是Peanut在LPL的首秀，曾效力於Rox Tigers、SKT、KZ、GEN等，都是拿過冠軍的豪門戰隊，此次來到目前形勢低迷的LGD，Peanut會帶來怎樣的活力，讓我們拭目以待。

因排位和比賽不同，會受到較多運氣、隊友等因素的影響，因此選手們排位的戰績也僅供參考，並不能完全代表選手的狀態，也不乏一些選手排位和比賽中表現相差極大的情況。德瑪西亞杯上，各大戰隊新陣容將首次在正式比賽中登場，他們究竟會打出怎樣的表現，是需要後續更多磨合，還是首次比賽就配合默契帶來勝利呢？也請大家共同關注德杯以及後續的春季賽。
