老秦聊遊戲，每天一篇新鮮事。

孤影從S17賽季開始之後，就一直在努力打巔峰賽，他的目的地是巔峰賽榜首的位置！雖然過程非常崎嶇難行，但是好在離榜首的位置越來越近了。也正因為如此，所以孤影經常在2000分左右的巔峰賽徘徊。

無論是哪個賽季，2000分的巔峰賽都足以稱為高端局，這個分段經常會和各種大主播或者職業選手偶遇。這不，本場比賽，孤影就撞車了有著王者榮耀第一狼狗之稱的虎牙林乖乖。

​林乖乖的大名想必常玩巔峰賽的小夥伴都不會陌生，這幾個賽季，經常會在巔峰賽前十的位置看到他的名字。林乖乖的特別之處就在於他擅長成吉思汗這名英雄，而且是在地形還未改動之前，狼狗還未強勢之時，他就已經登頂巔峰賽榜首，實力可見一斑。可別忘記，當時的狼狗還沒有增強，是一名冷門英雄。

​如果孤影只是撞車了林乖乖，也許勝負還是五五分，畢竟孤影也是祭出了自己最強的國服露娜。但沒想到，敵方不僅僅是林乖乖一個人國服而已，神秘選手橘右京的操作卻讓孤影感到絕望。

露娜最怕有控制的英雄，但是對於橘右京來說，其實並不是非常忌憚。可是這一位神秘橘右京卻可以把被動耍得出神入化，而且利用一技能位移可以極限躲掉孤影的標記，這就很可怕了。

​別忘記，還有林乖乖在一旁虎視眈眈，每次露娜空大都被敵方抓住機會。林乖乖可怕的地方不僅僅在於操作，還在於他的意識。居然只依靠一技能的視野就能判斷出露娜是否空大。

圖三所示，孤影偶爾一次空大就被林乖乖抓住機會，一直追著不放，最後只能飲恨黑屏。

打著打著，經濟就被拉開很遠，導致孤影看到橘右京落單都不敢上前。神秘橘右京居然一個人壓制露娜以及孫尚香和西施三個人打！

​在個人技術和經濟領先的情況下，神秘橘右京和林乖乖並沒有上頭，而是一直運營著兵線步步緊逼，使得孤影一方根本無法出塔。

出不來基地就拿不到經濟，於是在一波團戰失敗之後，藍方只能眼睜睜的看著水晶被推掉。

​賽後一看結算界面，原來這名神秘橘右京竟然是來自職業戰隊VG，也難怪操作和意識都做到這麼極致。近期，VG的職業選手一直在發力，現在巔峰賽榜首的位置同樣是來自VG的選手易瞳。

看來，職業戰隊都是卧虎藏龍的敵方，永遠不要小瞧任何一位職業選手哦。
