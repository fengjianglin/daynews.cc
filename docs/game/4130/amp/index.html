摘要

打開筆記本，裡面是否還有捨不得刪掉的遊戲截圖和心情故事？

究竟有哪些老物件，最能勾起夢幻老玩家的熱血青春回憶呢？

今天為大家帶來的內容，可以說是一發催淚彈，請各位備好紙巾——

玩家一夢十餘年：夢幻虐我千百遍，我待夢幻如初戀。與夢幻共同成長！

（這位老哥，你的將軍令都是批發的嗎？）

玩家小明小貓種魚：2004年4月1號申請的號一直在玩，最好的青春都在這裡，愛夢如初。

（厚厚的一摞點卡，見證了老哥從青蔥少年到長大成熟）

玩家二八一毛八：換了好多組號，這三兄弟一直在。人在，情懷在。

（一家人最重要的是整整齊齊呀）

玩家椒鹽小海豹：我的師傅小蛾子，現在都已經是個帶娃的媽媽了，這麼多年大家都從小孩子變成了大人。

（曾經善良可愛的小蛾子，一定也會是一個好媽媽）

玩家名字長就能中啊：用情懷伴隨信仰，讓虛擬照進現實。轉眼間夢幻西遊距今快有16年了，感謝夢幻西遊陪伴我走過高中三年、大學四年的青春時光。夢幻相伴這些年，有太多的回憶難以忘懷，課堂-宿舍-網吧，聊天的話題始終是夢幻西遊！2004年的楊千嬅代言-2010年的周杰倫代言，夢幻陪伴我成長，也讓我有幸見證了夢幻的征程。夢幻不倒，我們不散，80-90的夥伴們沖鴨！

（這篇獲獎感言給一百昏不怕你驕傲）

玩家小蜜心：回憶總是情懷，說好從東海灣一路陪伴到須彌東界的你呢？說好一起攜手江湖的你呢？我一直在原地，等你。

（時光不老，我們不散，願有緣之人常相伴）

一夢十年，當年偷偷攢錢買點卡的少年如今已經成家立業。

但夢幻曾經帶來的美好回憶卻不會隨著時間的流逝而消失，就像老酒一樣，歷久彌新。

夢回東海灣，老玩家的故事還在繼續……
