#這個遊戲好玩嗎#《失落餘燼》評測：離光之城還有一段路要走

2019-12-11 750條評論 4,077次閱讀 1941人點贊

和小狼相比，還是狐狸好些

在對《第一棵樹》里的狐狸和故事意猶未盡時，這頭小狼和紅球出現在了推薦列表裡，隨便掃了幾眼截圖和相關信息，他又被搬到了願望單，最後一直在這裡坐到了今年十一月底，他才成為了我的庫存遊戲。

老實講，從把他放入願望單的那一刻起，沒有太期望這款遊戲會多麼”成熟”，因為這一類型的遊戲總是顯得有些”門外漢”，一眼望過去就能看到其作為遊戲的條條裂縫，尤其是化身為鳥翱翔於天時，你甚至一眼能看到地圖邊界和簡陋的關卡。

有裂縫自然意味著有光流入，只要這款遊戲能帶來一段獨特的故事，能讓玩家多停留在滾動的製作人名單一會兒，至少能讓人鼻頭一酸，那麼這些光便有其意義，遊戲部分的裂縫也不再那麼重要。但《失落餘燼》沒有做的太好，套用遊戲主軸的故事情節，這顆靈魂離光之城還有那麼一段距離。

拿到這款遊戲，朋友告訴我”這是款藝術遊戲”，完整體驗了遊戲內容後，這句話後面應該加個問號。《失落餘燼》的表達流於形式，你可以看到場景和地形隨故事節奏變化，在每個小關中你還有慾望去截個圖，其美術功底可見一斑，在情節的關鍵點也特別設計了視覺和聽覺表現形式，使其能和對應的故事橋段匹配。

《失落餘燼》因其架空世界的特性，製作方用大量的篇幅去讓玩家了解世界和主要角色，在這期間主要矛盾出現太慢，故事節奏難以推進，導致劇情發展在很長一段時間內處於玩家的意料中。

而前期不算有趣的劇情自然讓玩家的關注點回落到了本作的玩法本身，這其實是《失落餘燼》的短板，體驗自然不算太好。

《失落餘燼》的核心玩法是解密和收集。玩家可以附身於任意動物，並使用他們的特點解鎖迷題。這個過程非常簡單，只要你找到了正確的動物，那麼迷題迎刃而解，所以在這個條件下，製作方將樂趣更多的集中在了解密之外。

變成一隻快樂的小鴨子

變成小豬吃野果，吃撐了還有成就，變成穿山甲偶爾能從泥土裡拱出蘿蔔，當然最爽的還是蜂鳥飛翔於天際，如果玩家附身於母鴨，還能看到身後緊跟的小鴨子。

除了動物本身的樂趣外，遊戲還設計了一些環境互動，例如吹散蒲公英。也就讓遊戲產生了一些若有似無的小關卡，比如在木樁通道前擺出一列蒲公英，玩家很清楚自己該怎麼做，撞完就是爽到，沒撞完也不影響接下來的流程。

這些小趣味不斷地裝點著遊戲本身，點點星光最終匯聚成為了亮點。而比起這些小趣味，遊戲里的收集要素就顯得非常多餘，玩家可以收集到各類蘑菇，或者和遊戲世界有關的諸多遺物。除非對獎盃頗有執念，否則這些東西本身的價值不足以讓玩家費盡心思尋找，蘑菇不會讓玩家多了解這個世界一點，部分遺物的物品描述也有些意義不明，還算值得的就是收集白色的傳奇生物，但新鮮感也會很快過時。

在收集的過程中，我們自然少不了BGM的陪伴，而本作的BGM比較令人遺憾，雖然在稍微關鍵一點的情節點和關卡都有BGM，而且效果還不錯。但因為BGM整體覆蓋率不足，所以在非必要時刻，玩家只能聽到乾澀的環境音，比如玩家變成各種動物漫步於世界時，卻沒有BGM進一步渲染氛圍，在體驗上打了些折扣。

但隨著遊戲推進到後期，劇情終於展開，核心矛盾開始對抗，本作的質量終於有了突飛猛進的進步，這種進步也是全方位的。

首先就是進步顯著的畫面設計。比之前要更加震撼和富有層次，關卡內過於缺乏指引的問題不再出現，他們似乎意識到這些問題，甚至用”迴響”的方式提示玩家該往哪個方向前進。而劇情經過之前漫長的鋪墊，種種謎團終於在這個階段內被揭開。

這是整個遊戲里非常漂亮的橋段，每走一步身後就會亮起來

在這裡我們簡單的總結一下《失落餘燼》的劇情結構。因其架空世界的特性，他在前期進行了大量鋪墊，這一過程過於冗長並且主旨不明，導致劇情很難吸引玩家，而在劇情進入正式的對抗階段時，終於去除了次要情節的干擾，伴隨著謎團的挨個解開，劇情重新富有了活力。

但這並不意味著《失落餘燼》在最後講出了一個好故事，他所要表達的核心還是讓人有些難以共情。不過在遊戲最後的階段，《失落餘燼》在解開所有謎團呈現主題後，結局成為了整部遊戲的點睛之筆，帶給了玩家不小的震撼。

所以總的來說，《失落餘燼》在開局階段因為”遊戲之上”的元素沒有處理的太好，使玩家的關注點回落到了”遊戲之內”。而在這部分內容中，本作又沒有使玩家折服，出現了或多或少的諸多問題，都影響了實際的遊戲體驗。好在遊戲進入後半段後，通過頻繁的劇情演出重新抓住了玩家，前半段的諸多問題也有所緩解和改善。

那麼《失落餘燼》到底算不算一部”藝術化遊戲”呢？我的答案在文章中間，他的藝術流於了表面。單看截圖本作確實有那麼點味道，但實際玩過之後就能發現，本作想要表達的核心難以觸及到玩家心中最柔軟的地方。

所以並不是每款不以遊戲性為主的遊戲都能被冠之為”藝術”，《失落餘燼》在試圖通過遊戲講故事打動玩家，他只成功了一部分。在作者沒有突破遊戲這個媒介前，所呈現的作品也自然難以跨越遊戲本身。

總評：7.8分

優點：

  • 玩法的Idea有趣
  • 遊戲里的小趣味不錯
  • 漂亮的畫面

缺點：

  • 故事本身一般，好在結局拯救了遊戲
  • 技術不到位
  • 操作體驗不是很好
  • 關卡缺乏指引
