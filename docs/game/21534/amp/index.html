爐石傳說：巨龍降臨最不受歡迎的5張卡牌，一頓操作花里胡哨

天天要聞
2019-12-19

新版本上線已經有一周時間了，在此前預覽新卡時，大家應該都記得新版本里有很多以前沒有見過的特效功能型卡牌。

其中有些已經被大家發掘加入了天梯卡組構築，但是有的缺只能在收藏里吃灰，下面筆者就帶大家一起來看看巨龍降臨天梯最不受歡迎的5張卡牌。

TOP5——龍眠凈化者

使用率：0.07%

勝率：33.9%

我們順著這張卡牌的效果反推，那就是帶兩張龍眠凈化者，然後選擇28張中立卡。最完美的操作就是後手起手就拿到一張這個，然後把牌庫全部變成自己的職業卡牌。

然後就聽天由命，給什麼玩什麼。聽起來其實有點像棄暗投明，但小明術好歹還能減費，這個也不能減費。小明術減費尚且只是娛樂卡組，這張卡最多只能算是比較弱的娛樂卡組。如果硬要說的話，那麼新手玩家缺卡，可以帶一張這個，然後其他都帶中立卡，開局變變變就完事了。

TOP4——活化龍息

出場率：0.06%

勝率：46.2%

這張卡其實身材挺優質的，但是效果時無法被凍結的光環。這是嘛意思，剋制我冰薩嗎？但是冰薩還沒有崛起啊喂！

目前天梯只有法師和薩滿會用冰系卡牌，但還遠遠沒有達到需要帶這張卡牌去針對的時候，除非我冰薩強勢崛起。

TOP3——恐懼渡鴉

出場率：0.05%

勝率：31.1%

在版本尚未上線的時候，有很多玩家都在圍繞這張卡牌做文章，各種花里胡哨的OTK都出來了。但是版本上線後，連T7獵都東山再起了，這張卡完全就不能作為核心去支撐起一套卡組。

主要原因在於難度頗大，需要通過各種手段洗多張恐懼渡鴉，再配合犀牛衝鋒，才能打出OTK的效果，有那時間T7獵早贏了第二把了。

TOP2——灰發巫師

出場率：0.05%

勝率：45.7%

問題就是交換了技能幹啥，現在標準模式也沒有戰士的開炮，換了技能沒啥意義啊！而且如果你覺得薩滿的雙倍戰吼不錯，那你自己去玩不就行了，為啥要靠這個巫師去騷一下？

TOP1——噴燈破壞者

出場率：0.03%

勝率：47%

以前也有針對對方英雄技能法力值消耗做文章的卡牌，比如說在58牧時期，有一張光環封技能的卡牌，可以較為有效地針對58牧。

這張牌與其有異曲同工之妙，但是問題是這個版本，沒有什麼特彆強力的英雄技能，還用不上他。

總結：

但是整體看下來，這些卡牌的身材都還可以，如果在競技場看到這些卡牌，倒是可以酌情考慮一下。
