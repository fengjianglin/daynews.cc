每年年底都感覺自己要裂開，玩個遊戲也“太南了”

2019-12-23 249條評論 1,198次閱讀 131人點贊

mumu丨文

2019年，我們遇到了很多困難。

——這可不是我瞎掰。幾天前，教育部發布了2019年中國媒體十大流行語。在一眾格調積極向上的辭彙中，「我太南了」實在顯得有些格格不入。

回顧2019，網路上確實發生了不少難事。

比如天災人禍，巴黎聖母院起火遭難，育碧大哥慷慨捐款，白送「大革命」。

比如流言蜚語，「雞你太美」在全網爆紅，成為B站鬼畜新寵，被律師函警告。

有些人不信邪，於是在一片叮叮噹噹的打鐵聲中，讓《只狼》教會了自己「蔡」字怎麼寫。（打了MOD）

有些人信了邪，於是堂堂圍棋冠軍柯傑，被阿爾法狗血虐後，在鬥地主比賽橫掃奪冠。作為普通人的對手可太難了。

今天BB姬就帶著大家以「太南了」為切入點，回顧一下2019年遊戲圈發生了哪些難事。

也算是對BB姬這一年的回顧——

首先說說直播方向的事情。

今年直播平台流行過的遊戲有很多，比如流星般划過的吃雞遊戲《Apex英雄》、讓全世界玩家沉迷「匡扶漢室」的《全面戰爭：三國》、期待已久的小島秀夫《死亡擱淺》、還有今年TGA年度最佳的《只狼：影逝二度》等……

但真要說，2019年中國直播圈最大的爆款遊戲當數「下棋」。

——《刀塔自走棋》，這是一個由國人團隊開發製作，並且在DOTA2的社區中發布的一個RPG地圖。

遊戲在今年1月上線後，立刻在刀塔、爐石直播圈引爆話題，颳起了「全民下棋」的風潮，甚至吸引了不少從未玩過Dota2的玩家慕名前來。

伴隨著不計其數的刀塔玩家卡在「尋找伺服器中」，自走棋巔峰在線人數破30萬，而Dota2時隔兩年又一次重歸百萬在線，這可太難了。

遊戲火了，各遊戲廠商順著這股「東風」紛紛入局，抱著各種IP大腿 「華麗上線」。

並且直接促成了V社自家《刀塔霸業》的誕生，以及之後LOL《雲頂之弈》、爐石《酒館戰棋》的流行。

說到刀棋，就不得不提正主刀塔。

關於電競方面。

DOTA2這邊有一個特殊的事件：今年TI9（第九屆國際邀請賽）在中國上海舉辦。

但很可惜的是，今年本土作戰的Ti9，中國戰隊全軍覆沒，LGD只拿到了第三。

如果說去年Ti8的敗北還可以歸結於LGD輸給了自己，那麼今年OG雙冠，讓更多的人看清了西恩刀塔的式微。

隨著很多老玩家開始步入社會，有了現實生活的瑣碎，大家早已是不打遊戲只看直播的「雲玩家」，要說和Dota還有什麼情感鏈接，可能就是Ti和小本子了。

電競沒有常勝將軍。DOTA2是一個非常看重成績的電競項目，所謂「沒有成績連呼吸都是錯的」。中國觀眾對Ti、Dota2電競的熱情，正隨著整體成績的變差而降低，這應該是不爭的事實。

我不想說什麼展望未來的空話，2020年的西恩刀塔，難，真的難。

好在今年夏天，對各位DotAer來說並不全是傷心事。

Ti9期間，每天激烈的淘汰賽打完，官方直播間就會播出一檔V社牽頭的深夜談話節目，《刀塔之夜》。

每期請來幾位有名的圈內人士，從梗文化出發，引申出諸多喜聞樂見的大「尺度」話題，被戲稱為「節奏之夜」。

「立蓁小辣椒」是刀塔之夜的主持人，真名張立蓁，土生土長的台北人。

BB姬有幸採訪到了「台妹」，了解到她的個人成長與節目背後的一些難事。

相比之下，LOL則要幸運的多。

S9是個全新的故事，一樣有著令人熟悉的結局。去年iG創造了歷史，今年FPX將這份榮耀傳承下來，這個冠軍得來不易。

FPX重組前什麼鬼樣？

原本因為傷病已經有退役打算的Doinb，被SN卡合同守了半個賽季飲水機的Tian，年齡最大已經有回國服兵役打算的Gimgoon，下路lwx跟lqs的組合在天才AD輩出的LPL也並不亮眼。

剛打完小組賽的時候，甚至有很多人說FPX是運氣隊，差一點點就進不了八強。誰能想到這群小伙竟然在決賽3:0擊敗G2最終捧杯呢？或許這就是鳳凰涅槃吧。

LPL連續兩次奪冠，證明了我們是第一賽區，也為這個老牌電競遊戲帶來了更多的話題度。過程雖艱難，至少有個好結果。

與此同時，國內騰訊在產業化的運營，比如綜藝節目、娛樂明星的各種聯動，都走在同行的前列。

而國外拳頭為了擴展符文世界的影響力，今年正式宣布成立自己的遊戲發行部門「Riot Forge」，與第三方工作室合力製作並發行LOL宇宙的遊戲作品。

一系列的動作，體現了英雄聯盟這個IP的生命力。「LOL要黃」這梗玩了這麼多年，怕是未來幾年裡都看不到咯。

關於其他的競技項目。

2019年中國《CS:GO》進入全新時期。本體免費後，遊戲經歷了一波人氣暴漲（隨之而來的還有愈加泛濫的外掛問題）。洗腦歌《EZ4ENCE》征服無數CS玩家，成為最流行的「背景音樂」。

各種直播梗的出圈，讓不少沒有接觸過的玩家開始嘗試這個老牌硬核FPS遊戲。我相信，2020年中國CSGO會更好！

隔壁《守望先鋒》玩家群體則相對穩定許多，雖然OWL在國內一直不溫不火，但是各種遊戲梗從來沒停過，今年OW最出圈的梗莫過於「末日鐵拳」了。

這個黑色光頭擁有屁股里少有的民工連，讓他成為團戰攪屎棍般的存在。

當一個輔助可太難了。奶媽們整天提心弔膽就怕人群里突然衝出來個黑色光頭，飛進來艹飛輔助再全身而退……

手游電競方面，走入平穩運營期的《王者榮耀》也在探索聯盟化的賽事運作。

今年6月《王者榮耀》KPL春季賽決賽於西安的大明宮遺址公園室外場地舉行，結果因為天降大風的「不可抗因素」延期。

誰能想到全世界最賺錢的手游之一、最高峰時用戶高達兩億人的《王者榮耀》，最高級別賽事的決賽舞台竟然被一陣「妖風」吹垮了。室外承辦電競比賽，可太難了。

現在電競類的遊戲實在太多，其他的吃雞、彩虹六號、堡壘之夜、使命召喚的電競項目就不一一展開了。

最後說說國內遊戲圈方面。

2018年的最後一個工作日，在經歷長達十個月的「寒冬」後，中國遊戲行業終於迎來了版號重啟。

然而版號審核重新開啟後，情況依舊不樂觀。突然而至的一頓鐵拳暴揍，一度讓國內大大小小的遊戲廠商全都懵圈，國內遊戲行業被迫進行大洗牌。

對於獨立遊戲圈而言，日子過得更難了。

儘管今年在B站、騰訊、椰島等一眾發行商的強勢投入下，依舊有不少國產獨立遊戲精品湧現，比如用聲音探案的《疑案追聲》、互動式電影《隱形守護者》等……

但你要說像2018年《太吾繪卷》、《中國式家長》那樣的爆款，很遺憾，基本沒有。

反倒是一些「國產大製作」引起了不小爭議。

比如米哈游的《原神》，比如其實是日本工作室製作的《大聖歸來》。


不少網友感嘆太難了，氣抖，冷哭，刷新了所謂「遊戲史上最黑暗的一天」。（你別說，我反而還挺期待原神正式上線的，怎麼回事？）

對於Steam而言，2019年同樣是難受的一年。

這一年Steam迎來了強力的競爭者Epic。

背靠《堡壘之夜》與虛幻引擎的成功，Epic手頭有著大量現金流，足夠通過買買買各種遊戲獨佔權，來為自己的新平台造勢。這招對於開發者來說確實很管用。

先是今年1月《地鐵: 離去》在發售前突然轉投Epic平台。隨後育碧、R星等廠商也逐漸脫離Steam，開始自立門戶，反戈G胖的勢力日趨壯大。

特別是Epic挑選遊戲的品味也相當不錯，限時獨佔的列表中不乏話題性慢慢的作品，比如——

時隔多年的無主之地新作《無主之地3》；

在TGA上被提名為年度遊戲的《控制》；

火出圈的年度沙雕遊戲《無題大鵝模擬》；

晶體管廠商的Roguelike新作《哈迪斯》；

超級食肉男的新作《超級食肉男：永恆》等……

對於玩家來說，每個星期的免費遊戲也確實有點香，很多像《信使》《蔚藍》之類的周免遊戲，之前都安靜的躺在我Steam庫里，品質非常OK。

而不喜歡Epic的人依舊對其深惡痛絕，彼此無法理解。

玩遊戲還要下各種平台，實在太難了。時候反倒是EA站了出來，宣布橘子平台的遊戲將會在2020年陸續回到Steam，不然真就裂開了。

相關閱讀（點擊可回顧）：

《大廠逃離，小廠背叛，Steam即將王朝覆滅？》

《玩家評分4.4分？《無主之地3》是糞作么？！》

《這款沙雕遊戲，讓玩家統統當上了帶「鵝」人》

（我的遊戲日常，有些人還不止這些）

2019年我們經歷了許多讓人難過的事情，不管是戰隊比賽的失利，遊戲平台的倒戈，又或是十連抽卡不出貨……

不少遺憾的事、悲傷的事、可惜的事似乎都涌在了年底，讓我們喘不過氣來。

這時你才發現第一批90後馬上要「三十而立」，第一批80後也要「四十而不惑」，於是又開始傷感起來。拼湊記憶，隨意在B站翻動，只想要在遊戲動畫里尋找自己「有閑沒錢」的童年。

「我又有了個製作大型藝術創想的靈感。」

說著，一個茶色短髮的大男孩睜圓雙眼，嘴角上揚，伸出食指向鏡頭示意，隨即在一堆垃圾桶中間翻找了起來。

你在《小神龍俱樂部》的「藝術創想」、啟動超級變換形態的《鐵甲小寶》、燒菜必放閃光彈的的《中華小當家》彈幕里找到了無數同好。

發現原來自己不是一個人。

《紅警死了十年，玩家們做的MOD卻一個比一個好玩》

回頭想想，嗨，人總得樂觀點，要向前看：

這不，「南」可不只是「難」，那也是太陽的方向啊。
