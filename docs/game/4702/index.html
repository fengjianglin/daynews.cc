「飯聊」8億買個血虧？深度解析B站購買S賽獨播權，睿總賺大了

2019-12-05 0條評論 13次閱讀 0人點贊

摘要

2017年1月23日，對於B站來說肯定是值得紀念的一天。就在當天晚上21點26分，編號為2233的拜年祭手辦，以98億7654萬的價格成交。

2017年1月23日，對於B站來說肯定是值得紀念的一天。就在當天晚上21點26分，編號為2233的拜年祭手辦，以98億7654萬的價格成交。在這樣一筆天價拍賣金和這樣一位資深真愛粉的支持下，我們的B站就此邁入超高速發展的道路，甚至在兩年後豪擲8億，一舉拍下未來三年英雄聯盟S系列賽的獨家轉播權。

哈哈，開個玩笑~

在B站拿下S賽獨播權的消息公開後，不少小夥伴除了「震驚」以外，也表示自己「看不懂」，覺得B站這次豪爽的有點過分，怕不是到最後要變成一筆虧本的買賣。

虧嗎？

其實在飯飯看來，B站這次的大手筆，不僅魄力拉滿，而且能夠獲得的回報也非常可觀。我們或許不知道馬總虧沒虧，但至少咱們bilibili最美的舞見，這次是賺大發了~

為什麼要砸這八億

2019年Q3季度的財報顯示，B站的月度用戶再次實現了高速增長，特別是移動端，1.14億的月活用戶已經證明，曾經的「小破站」已經穩穩站在了中國視頻內容平台的頂點，與老牌的「騰優愛」三家的距離在不斷縮小。

這樣的喜訊固然值得開心，但從另一個角度來說，B站的用戶增長也隱隱達到了一個瓶頸期。

我們都知道，以動漫番劇起家、以彈幕文化為核心社區內容體系B站，早早就被打上了「年輕」的標籤。這是B站能扶搖直上的關鍵，但當它的體量足夠龐大的時候，這樣的固化標籤卻很容易成為它進一步發展的掣肘。

另一方面，「難出圈」也是B站內容生態一直面臨的一個比較大的問題，包括B站最寶貴的財富——UP主們也一樣。蕾絲老番茄等鎮站之寶，在外部可以引發的聲浪相較於同等人氣的影視或體育明星而言，小太多了。

所以B站近年來大力推動生活區等非核心品類視頻項目的發展以及UP主培養，也在大力購入各種影視劇的版權，就是希望能讓站內內容的多元化梯次化，吸引更多不同年齡層次的用戶加入，繼續實現用戶增長。

除此以外，各種站外流量渠道也是B站近年來不斷挖掘的主要方向，就比如「星訪問」這個節目，創辦的主旨就是希望藉助明星本身的影響力吸引站外用戶。

在這樣的情況下，作為世界範圍內的重磅電競賽事IP，S賽及其附帶的巨大流量，自然就成了B站的追逐目標。

這八億，砸出的可不是水花

從多角度來說，S賽都是B站目前為止，最好的引流項目。

第一，足夠高的熱度。

S賽的熱度就不多贅述了，去年IG奪冠的轟動性還歷歷在目，今年小鳳凰涅槃的經歷也是吸引了不少圈內外觀眾的視線。更別說2020年的S賽剛好就在我們家門口舉辦，又趕上LOL十周年，其所附帶的流量預計會達到一個非常恐怖的程度。

第二，足夠好的契合度。

遊戲和電競本身在B站就有著足夠的受眾群。而且不僅僅是觀眾，大量圍繞這兩個主題產出內容的UP主，在賽事進行前後也可以貢獻足夠多的梗和社區話題。就算只有1%的站外觀眾因為這些UGC內容而轉化為B站本身的用戶，數字也會龐大地驚人。

第三，社會主流輿論的關注。

隨著整個電競行業的不斷發展，社會主流輿論對於電競的態度，也從漠視、不屑和不理解，逐漸變成了主動接納和深入了解，IG和FPX奪冠後多家主流媒體的報道就是最好的證明。要知道這些主流媒體所輻射的，往往就是B站尚未開發的潛在用戶群；如果能藉助這些主流媒體的高度關注，在廣大6070後間打開一道屬於B站的大門，那麼別說8億，18億都值得。

「我們也想搶，可他實在太有錢了」

當然，這樣的好處對於其他轉播平台來說也是實打實的，為什麼最終還是咱們B站一擲千金，拿下了獨家轉播權呢？

無他，唯性價比耳。

隨著鬥魚在今年正式上市，直播平台的野蠻生長階段正式宣告結束，這些最終存活下來的鬥魚、虎牙和企鵝電競們，如果不想重蹈熊貓TV的覆轍，制定長效性規劃，追求變現，保證盈利才是最關鍵的事情。

所以我們可以看到，鬥魚開啟一輪又一輪的粉絲節、嘉年華以及年度，眾主播為了一個年度十大爭的昏天黑地。更有甚者不惜打空了自己的身家，最後還是棋差一招沒有拿到那個名額。平台不惜犧牲粉絲和主播的好感也要舉辦這類金錢戰爭，足以見得其背負的壓力有多大。

在這樣的大背景下，幾大直播平台放棄對於S賽轉播權的爭奪無疑是非常明智的。

而且還需要注意的是，相較於B站而言，這些直播平台甚至都沒有完成自有內容體系的完善建設。你可以在B站看直播，看視頻，看飯飯的文章（羞澀），在評論區和同好交換意見一起玩梗；但你在鬥魚虎牙這些平台，最多是看比賽的時候和彈幕杠精打打嘴仗。

對於B站來說，8月份LPL夏季賽季後賽開打之日就可以炒作S賽的話題，12月LOL世界全明星賽的時候還能再帶一波回憶，整個賽事話題期可以長達四個月（甚至更長）；但對於直播平台而言，每年接近3億的價格只能買一個月不到的流量和熱度，性價比實在太低了。

B站之野望

綜上所述，B站在拍下S賽的獨播權以後，其實戰略目標就已經達到了。至於後面要不要開放二路OB許可權給一些知名主播，站內的預熱和跟蹤內容創作要怎麼組織，能否招商引資補貼一部分資金，以及利用S賽的熱度提升一下直播板塊的人氣，就是戰略目標之外的「小」驚喜，也都是後話了。

飯飯最後想說的是，這次事件的重點是「B站」買了獨播權，而不是「B站直播」買了版權。在B站的未來規劃里，它所對標的不是虎牙鬥魚這些直播平台，甚至都不是快手這樣的短視頻平台，而是騰訊視頻、優酷和愛奇藝這行業三巨頭。在更遙遠的未來，它所希望的是能夠建立一個完整的站內閉環內容生態體系，就像位元組跳動現在所努力的那樣。

飯飯很期待我們的「小破站」到底能不能走到那一步，你們呢？
