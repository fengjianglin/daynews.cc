LOL選手退役的新出路 Impact打造電競基地提供訓練

2019-12-03 0條評論 17次閱讀 0人點贊

摘要

從上周末LPL全明星周末的熱度就可以看出，電子競技職業選手的地位越來越高了，也被廣大人們和社會認為是一個有前景的職業。但職業選手的狀態很難始終保持穩定，這也使得他們的職業生涯具有極大的不穩定因素。

從上周末LPL全明星周末的熱度就可以看出，電子競技職業選手的地位越來越高了，也被廣大人們和社會認為是一個有前景的職業。但職業選手的狀態很難始終保持穩定，這也使得他們的職業生涯具有極大的不穩定因素。儘管現在電競選手的職業壽命比幾年前更長一些，但也很少有職業選手能夠在28歲後繼續活躍在賽場上。

​

職業選手退役後的選擇空間也不算多，一般來說，都是轉型主播或者教練分析師，少數可能加入拳頭遊戲和當地聯賽，擔任解說或者評論席。畢竟，職業選手們將最好的青春和時間，都貢獻給遊戲和比賽了，他們沒有更多的專業技能。但對於前SKT戰隊的老將、現在的TL戰隊上單選手Impact而言，他似乎有著更加長遠的計劃。

​

Imapct最近在個人推特上表示，他在韓國擁有一所屬於自己的別墅，他將其命名為「Impact Tower」。畢竟，Imapct是最早一批來到北美髮展的韓援選手，他在2014年的年底就加入北美LCS聯賽，他曾經和TL戰隊簽下了3年340萬美元的合約。Impact在去年已經拿到了美國綠卡，可以說是混得非常好的一位英雄聯盟老將了。

​

同時代出道的大部分選手都退役了，包括前SKT戰隊成員的福滿多、Bengi，還有上周末的Wolf。而Imapct顯然打算繼續在賽場上打拚，一方面是他在北美本土的實力能排在前幾名，加上被歸化成北美選手和能用雙語溝通自然加分。但隨著個人能力的下滑，Impact同樣為自己的未來做好打算，他將自己的薪資都用來打造一個電競基地。

​

「歡迎來到我的上路！」從2012年到2019年期間，Impact將自己攢下的錢都用來打造了這個訓練營，他為自己感到自豪。Imapct很快就和自己的現任隊友，來到這個Impact基地進行季前賽的訓練，他打算將這個基地出租給那些想來韓國訓練的選手。看起來，Imapct打算利用自己的資金和人脈，在韓國建立一個優質的電競訓練營。

​

畢竟，在過去幾年，很多歐美、外卡甚至LPL隊伍，都會在休賽期間前往韓國進行集訓，Imapct則是嘗試自己建造一個電競基地作為訓練營。這無疑是一個野心勃勃的項目，讓外國隊伍可以在韓國找到最好的訓練設施，而他本人可以作為連接韓國和西方的橋樑。Impact能夠協助前來訓練的隊伍或者選手，更容易約訓練賽對手和進行韓服練習。
