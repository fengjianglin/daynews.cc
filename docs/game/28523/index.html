拳皇志之哈維·D!打法極具觀賞性的一代拳王，可惜再會無期

2019-12-23 206條評論 2,288次閱讀 1370人點贊

大家好，我是零之筆記，歡迎來到零之筆記的遊戲領地。在拳皇界，美國運動隊是毫無爭議的元老級人物，自初代KOF94起便和草薙京、坂崎良等人一起打天下，連八神見了都得叫一聲「前輩」。但二十多年來，他們只在94和98中走過兩次前台（不算94RE和98UM），人氣之低可謂「凄凄慘慘戚戚」。不過沒關係，「拳皇志」一向有著扶貧濟困、敬老尊賢的傳統美德，哪怕再不招待見的角色咱也一視同仁。所以，今天咱們就來聊一聊美國運動隊中的拳擊手——哈維·D！（下文簡稱「哈維」）。

拳皇故事

哈維曾是一名美國職業拳擊手，但因為經常咬耳朵……不是，因為經常把對手打成半殘，導致再沒人敢和他比賽。還有一說是他經常失手把對方打死，這就有點太過分了，打了這麼多年難道手底下還沒點數嗎？

不管是打傷還是打死，反正哈維肯定有了一種「高處不勝寒」的寂寞感。恰好這時KOF大會給他送來招待狀，哈維一琢磨，既然在拳壇已無對手，不如去會一會其他格鬥領域的能人異士，便邀請了打籃球的洛奇與打橄欖球的布萊恩組成了「美國運動隊」參賽，並擔任隊長。

一進拳皇賽場，估計哈維等人的眼珠子都嚇掉了——有人手上能發電，有人嘴裡能噴火，有人轉大鐵球如溜溜球，有人飛來飛去如低配版蜘蛛俠……這哪是什麼能人異士啊？這明明是一幫牛鬼蛇神嘛！不過哈維他們還是憑藉運動員的強健體魄和敬業精神，一路浴血奮戰衝到了最終BOSS盧卡爾的面前。老盧一見三位體育明星便兩眼放光：「兄弟，能給我簽個名不？不簽？擺臭架子？那就休怪我把你們做成雕像收藏了，哇卡卡卡！」

經過一番苦戰，哈維等人擊敗盧卡爾，從游輪上逃出。三人覺得打KOF還是很有意思的嘛，比打拳打球打架都刺激多了，於是相約下次組團再來。

但非常遺憾，他們在拳皇95的預選賽中被八神率領的天敵隊擊敗，無緣正賽。拳皇96也毫無動靜，估計是在養傷，畢竟被八神的爪子撓一通還是挺疼的。到了97想要重新出山，結果又被七伽社胖揍一頓，招待狀也被搶走，再次無緣大賽。再以後，就沒有以後了……自KOF94後，美國運動隊在拳皇的主線故事中再沒有出現過（98無劇情）。最新的拳皇14，SNK寧可打造一個新人拳擊手尼爾森，也不給哈維重新站上賽場的機會。

雖然組織看不上他，但哈維對KOF卻是一往情深：既然不讓我參賽，那就安靜地坐在場邊，當一個美美的觀眾吧。拳皇95時，哈維與來自「餓狼傳說」的鴨王（Duck King）和來自「龍虎之拳」的藤堂龍白（藤堂香澄她爹）出現在「女性格鬥家」隊的比賽場地中，一邊飲酒一邊興高采烈地觀戰。不過我說哈維老兄，你真的是來看比賽的？不是來看妹子的？

拳皇97，在中國賽場上，我們再次見到三位「紳士」的身影，這次更是花里胡哨地跳起了扇子舞，不知是不是想吸引美女的目光呢？……

拳皇98，SNK大發慈悲，美國隊全員重裝上陣。這回龍白和鴨王沒有手拉手悄悄溜去看妹子，而是非常義氣地來到運動員隊的主場為哈維加油打氣。但我好奇的是，這三個風馬牛不相及的人究竟為什麼會變得比親兄弟還親呢？嗯，這是一個未解之謎……

02UM的義大利背景中，由於藤堂龍白去給女兒打下手了（藤堂香澄的MAX2超殺會召喚出老爸），這一次就只有鴨王與哈維搭伴觀賽。看哈維這一身腱子肉，顯然在平日里依然進行著刻苦鍛煉，期待著有朝一日可以重現往日榮光。

人物設定及彩蛋

哈維（Heavy·D！）的名字來自於美國著名說唱歌手德懷特-梅爾斯，其藝名便為「Heavy D」。SNK在角色名字的最後加了個嘆號，可能是怕完全相同會招惹官司。這位歌手已於2011年11月離世，享年44歲。我雖然對他不太熟悉，但知道他曾在我的偶像邁克爾·傑克遜的名曲《JAM》中擔任了RAP部分的演唱，在MV中也有亮相。在此，謹向這位傑出的音樂人表示哀悼與致敬。

哈維理了一個乾淨利落的莫西幹頭，看起來十分陽剛霸氣。這裡簡單說一下「莫西干髮型」。咱們通常理解的「莫西干髮型」就是指這種兩邊頭髮剪短或剃光，中間的頭髮直豎起來的樣子。

但其實這裡有一個誤區。莫西干（Mohicans）是北美印第安人的一個分支，但他們留的其實並不是這種髮型。「莫西干髮型」的英文是The Mohawk hairstyle，直譯過來就是莫霍克髮型，莫霍克也是一支古老的北美土著部族。由於翻譯上的以訛傳訛，錯進錯出，在國內我們就將本屬於莫霍克族的髮型安到了莫西幹人的頭上。「莫西幹頭」在國內廣為人知，應該是源於02年韓日世界盃上，萬人迷貝克漢姆就留了這麼一個髮型，當時引起了無數粉絲、尤其是女球迷的狂歡。但我斗膽說一句大不敬的話：我始終覺得小貝不太適合這個髮型……

哈維在戰勝對手後，勝利動作之一是脫下上衣，露出後背的鷹狀紋身。

這隻鷹是美國的國鳥「白頭海雕」，屬於大型猛禽，體長可達1米，翼展超2米，生性兇猛，捕獵能力超群，關鍵是長得還帥，活該被當國鳥。雖然哈維看起來像個弔兒郎當的混混，但既然能將國鳥紋在身上，至少說明他有一顆拳拳愛國之心吧。

拳皇中有不少戴著墨鏡耍酷的角色，哈維就是其中之一。但與比克、克拉克等「墨鏡長在臉上」的人不同，哈維在對戰前會小心翼翼將墨鏡摘下收好，嗯……或許他的眼鏡比較貴。

招式簡析

拳皇94沒啥可分析的，咱們只聊拳皇98和98UM里的哈維。作為一名有職業精神的拳擊手，哈維的四個攻擊鍵全是出拳，他那兩條粗壯的大腿除了走路就是擺設。眾所周知，除了劉備這種雙手過膝的基因突變型人類，一般人肯定是腿比臂長，所以不能用腿踢人就在攻擊距離上吃虧。好在哈維的個子夠高（2米08），臂展很長，一個站C能打小半個屏幕。

可惜除此之外，哈維就再沒有什麼距離遠、牽制強的普通技了。他的下C可以對空，性能只能說湊合；下D的距離和出手速度尚可，但不能取消；跳C有2段攻擊且可以打逆向，但並不穩定，且不像麥卓的跳B兩段有那麼強的欺騙效果。

跳B是98中一等一的打逆神技，只要跳過頭頂基本就能打到人，尤其在對手蹲姿倒地後，小跳B擇正逆是非常難以防範的進攻套路。只是需要注意一點，當對手站立時，如果跳B打點過高，不僅接不上2B連段，還有可能被反擊。

最常用的普通技當然是2A和2B，這兩招本身性能沒什麼了不起，只是由於出招與收招奇快，可以打出非常華麗的6-8hit目押連段。在卡「軟固定」的情況下，甚至可以打出10連以上。這也是很多高手愛用哈維的重要原因，雖然這個角色不強，但是可以秀手法啊。

在「軟固定」下打特瑞的極限目押

哈維的必殺技雖然不少，但多是雞肋，沒有性能超強的。輕「旋風正突拳」（26A）可以磨磨血；「組合上籃」（214A/C）有短暫無敵時間，且可以打個中/下段二擇，但這招出手較慢，不能形成連段；「投手攻擊」（623A/C）是個近身打擊投，綜合性能比較一般，偶爾可以用用。其他的就不說了，基本用不上。

值得一提的是「陰影」（22A/C），這招使用後會給哈維上一個BUFF，所有必殺技都會得到強化。聽著很強力，實際很無語。首先這個BUFF的條件太苛刻，強化效果只能使用一次，且若被對手打中，BUFF就會取消；其次說句不中聽的，大部分的必殺技即便強化了也沒啥大用。所以在對戰時，很少有人會去使用這招。

唯有「投手攻擊」這個打擊投在強化後確實很有威懾，它的最後一段會使對手浮空，不僅可以用必殺技追打，還可以連MAX超殺「銀彈終結」（2626A/C），傷害非常恐怖。當然，前提是你能摸到人且不被摸，還得有足夠的氣。

98UM里的哈維有一些細微的改動：26A/C可以直接將對手擊飛，雖然不能展開持續壓制，但也減小了對方強反的可能；「陰影」的強化效果不會因被攻擊而強製取消，實用性大大增強；再就是多了一個超殺「D.暗影」（2626B/D），其實就是「陰影」的超殺版，可以對必殺技進行強化。「陰影」只能強化一次，「暗影」可以強化四次，而MAX「暗影」更是無限次（不敢確定，但筆者在練習模式下強化了二十次後依然有強化效果）。

總的來說，哈維的實力雖然在拳皇98中只能位於中下游，但由於有變態的逆向攻擊策略，還可以打出非常精彩的目押連技，所以是一個有著鮮明個人特點的角色，玩起來不會覺得枯燥。在拳皇15中哈維會不會高調回歸呢？實話說，我覺得可能性無限接近於零。或許我們只能對他遺憾地說一聲：再會無期，望君珍重！
