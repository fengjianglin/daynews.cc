殺手47歸來！《代號47:出擊》今日登陸Wegame平台

2019-12-10 2,282條評論 3,596次閱讀 141人點贊

今天，由日本著名遊戲開發商SQUARE ENIX INCSE開發的一款以《殺手》為原型的衍生遊戲《代號47：出擊》正式上線Wegame平台，目前推薦率為84%，首發優惠9元發售。

遊戲延用了《殺手》系列經典的人物模型，畫面呈現出藝術感十足的立體效果，將動作暗殺元素轉變為在棋盤上的移動潛行，配合SE成名的BGM搭配。遊戲中你將化身鐵血殺手-代號47，在網格棋盤上選擇路線躲避敵人，滲入戒備森嚴的地點。遊戲在環境的刻畫上簡單到極致，一個剖面、一塊地皮、一個斷層就構成了一個任務場景，簡約而唯美。

官方介紹：《代號47:出擊》是由Square Enix Co.Ltd.製作，東品遊戲發行的一款有著布景模型式優美畫面的回合制策略戰棋遊戲。

控制特工，在網格棋盤上選擇路線躲避敵人並幹掉目標，或者滲入戒備森嚴的地點。仔細斟酌前行的每一步，一著不慎,將滿盤皆輸!潛行偽裝、干擾誘敵，利用狙擊槍以及標誌性銀彈槍，完成所有謎題挑戰，你能否成為殺手之王？
