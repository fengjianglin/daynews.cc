EDG阿布直播談廠長BP：他準備工作做得好 都在我們意料之中

2019-12-23 551條評論 2,317次閱讀 984人點贊

在之前英雄聯盟轉會期結束的最後一天，EDG突然官宣了一個重磅消息：LPL001號選手斷開連接，Clearlove(廠長)宣布退役。這也是不少粉絲猝不及防，當然廠長也並沒有離開電競這個圈子，而是轉型成為了EDG的新教練。

而昨天的德瑪西亞杯比賽也是廠長作為EDG教練的首秀，EDG最終在關鍵之戰中擊敗TES以小組賽第一成功出線，而昨天廠長BP選出的陣容也獲得了粉絲的一致好評，可以說這次廠長的教練首秀是非常成功的。

當然這只是粉絲對於廠長BP的評價，那麼專業人士對於廠長教練的BP評價如何呢?阿布作為EDG的前任教練，可以說是咱們國產教練的排名人物了，當時EDG巔峰時刻拿下15年MSI冠軍的時候，阿布就是當時EDG的教練。隨後在去年的亞運會英雄聯盟表演賽中，阿布也是咱們中國隊的教練，並且成功帶領隊伍拿下冠軍。

而昨天EDG成功晉級後阿布也開啟直播和粉絲們聊了聊天，也談到了廠長作為教練的首秀，阿布表示：你們想聽真話還是假話啊?明凱教練就是賽前準備工作做了很久，也做的很到位，所以在BP環節中也沒有出什麼意外，一切都在我們的意料之中。

其實廠長之前在EDG就是擔任的指揮角色，很多EDG的運營打法和關鍵決策都是廠長做的，而且廠長作為一個打野選手，對於視野非常的敏感。正如昨天Meiko在賽後採訪中所說的那樣，廠長的大局觀一直都不錯，這也是他能成為教練的原因。

接下來EDG將在淘汰賽中面對老對手WE，那麼EDG能否取得勝利呢?我們拭目以待!
