戰棋補丁：融合怪被刪，魚人先知回調，魚人流統治地位被推翻

天天要聞
2019-12-19

大家好！我是爐石閑說，為您分享爐石魔獸趣聞趣事！在今天凌晨，設計師在英文網站HS TOP Decks公布了新一輪的補丁（版本號為16.0.5）的詳情，其中在爐石傳說天梯方面，薩滿的3張職業卡牌以及1張中立卡牌遭到削弱。

而在酒館戰棋的模式中，上次由設計師放出風來表示即將把酒館戰棋中百搭棋子——夢魘融合怪，預削弱為2/2的身材。但是在這次的補丁中，融合怪卻被直接移出了酒館戰棋。還有其他的關於酒館戰棋的補丁詳情，接下來給大家分享下。

  • 英雄調整

在此次的補丁調整中，勝率足以排名榜首前幾位的新英雄布萊恩銅須，即將被移出酒館戰棋，代替的是前不久移出酒館的「資深調酒師」。這次的英雄輪換，想必也是因為布萊恩·銅須的勝率偏高，成為了設計師平衡的對象。代替的調酒機器人，不強不弱的英雄技能，也算是比較中和的平衡手段。

  • 棋子平衡

夢魘融合怪這個2星棋子實際上從酒館戰棋剛上線，就一直強悍到現在。在設計師下了將其削弱的決定後，終於不在妥協般地更具之前預調整為身材2/2，而是下定了決心直接將其移出酒館。之所以不採用之前的方案，具體的原因可以參照英雄館長的小融合怪。

其他對於一些冷門棋子的調整是波哥、機械蛋和比斯巨獸皆下調了1星，可以看出，設計師的意願是希望玩家們多使用這些棋子，然酒館戰棋的對戰中出現更多的面孔，而不是翻來覆去就是那幾個常用的強勢棋子。

同時，被削弱的棋子還有魚人流派的重要卡牌——寒光先知（由2星恢復為3星）。實際上，作為玩家的遊戲體驗，在設計師公布了寒光先知即將被調整的預告消息的時候，作者就覺得酒館戰棋的實際對戰中，包括寒光先知和蠻魚斥候等魚人流中重要的卡牌就沒那麼容易上手了，此次的回調，就上融合怪的刪除，預示著魚人流卡組終於被設計師動刀，魚人流即將走下神壇。就是不知道下一個「巫妖王」會是誰呢？
