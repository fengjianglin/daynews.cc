LNG公布德杯名單，聖槍哥小段留隊，全華班衝擊冠軍

2019-12-18 750條評論 4,181次閱讀 1265人點贊

12月18日，雖然轉會期已經截止，但是英雄聯盟LPL賽區各大戰隊官宣選手、名單卻是根本停不下來，RW戰隊一口氣宣布了多名韓援選手、教練的加盟，VG也在做最後的補充，而逐漸崛起的LNG戰隊也在這個時候宣布了戰隊功臣聖槍哥以及下路雙人組阿修羅、小段續約成功，這個消息可是為李寧的粉絲注入了一管子強心劑了，再起碼還是能看到李炫君在新賽季發光發熱的。

​隨後LNG戰隊就直接給出了德瑪西亞杯的陣容大名單：上單Flanders（李炫君）、chenlun17（周鵬遠），打野Xx（熊宇龍），中單Maple（黃熠棠），下路Asura（盧麒）、Light（王光宇），輔助Duan（段德良），主教練Cyo（林昕宥）。

​同時LNG戰隊官宣的還有他們將俱樂部主場正式更為蘇州陽澄國際電競館，坐落在太湖旁邊的主場，不得不讓人感嘆一句大（you）氣（qian）。不過這可苦了西南片區的粉絲了，因為傳言說OMG的主場也要搬走了，西南片區最近的主場還得是西安WE主場。

​話說回來，不少粉絲看到LNG這個陣容都感到非常吃驚，因為不僅僅是紙面實力夠頂，而且他們才是真真正正的全華班，主教練也是我們中華兒女。

​但也因為全華班這件事兒，許多粉絲開始擔心起了戰隊的成績。因為之前RNG戰隊也是信奉的全華班建隊（雖然主教練現在是韓國的），一直為人所詬病只要全華不要成績。如今LNG也開始了交易走韓援的全華班之旅，前路還是充滿了坎坷呀！

​朋友們，你們怎麼看LNG的陣容？你們覺得全華班有出路嗎？
