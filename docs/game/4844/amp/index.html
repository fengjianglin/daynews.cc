推薦一個快速實現創意的遊戲開發工具，不止適合新手

2019-12-04

摘要

我最近和朋友聊天，越發覺得但凡嘗試過自行開發遊戲的人，對開發工具應該都有相似的感受：UE4這類引擎上限非常高，但對於編程基礎薄弱的人，可能不是第一選擇；RPGmaker這一類工具我也用了一段時間，非常好上手，可惜2D畫面有些簡單。

我最近和朋友聊天，越發覺得但凡嘗試過自行開發遊戲的人，對開發工具應該都有相似的感受：UE4這類引擎上限非常高，但對於編程基礎薄弱的人，可能不是第一選擇；RPGmaker這一類工具我也用了一段時間，非常好上手，可惜2D畫面有些簡單。

今天我也想分享一個不錯的遊戲開發工具，它比較能滿足以下人群的需求，文章比較長，大家可以自行判斷閱讀：

  • 剛剛入門遊戲開發的初學者

  • 手頭上有個遊戲創意，想快速實現的人

  • 希望快速驗證遊戲玩法和品質的人

前兩天我看到微博上介紹了一個叫羅布樂思（Roblox）的遊戲引擎，據說它在全球範圍內已經有1億月活，超過400萬的遊戲開發者，和超過3000萬的作品。

雖然我猜測那是鵝廠做的一個推廣，但還是有點好奇什麼工具可以有這麼高的月活量，所以試著用了三天，發現它有幾個亮點。

引擎非常簡單易上手

這個引擎有多容易上手呢？簡單到我在沒看教程的前提下，就做出了一個遊玩時長不到一分鐘的橫版跳躍小玩法，因為這個遊戲大量的預設內容以及簡單的腳本語言。

羅布樂思是一個有很多預設模塊的3D引擎，比如我在這裡展示的城市場景模板。

除了場景模板，編輯器里還有大量預設好的模型和效果。預設內容大大降低了製作遊戲的時間和精力成本，假設你想做一個城市障礙跑酷小遊戲，那麼只需要打開城市模板，加入各種汽車、樓房甚至形狀部件，就可以直接跑起來了。

當然，這些預設內容也是可以修改的，以這輛車為例，右上角的資源管理器顯示，你可以控制的部分具體到某個車門、車窗以及乘客座位。再複雜點，你還可以實現對這輛車的加速速度、車輪扭矩、發動機動力等各種效果的控制。

如果你想做更多更複雜的玩法如捏臉、回合制、戰棋等等，通過腳本就可以實現，官網的教程里也有對RPG類等級系統、任務線、武器刷新、NPC等各種內容的設置教程，連重生點也只需要單擊就可以添加。

羅布樂思這個工具，可以說是把能簡化的內容都幫你簡化了一遍，非常友好。只要跟著教程走，很快你的遊戲想法就能讓人們玩兒上了。

並且，這個引擎用的腳本語言是lua。雖然相比起主流如C#，lua比較小眾，但事實上它作為一個輕量小巧又五臟俱全的語言，概念簡單，學起來是比較容易的。

對這一部分的實現有興趣，可以到羅布樂思的創作者主頁感受一下，在這裡泡上一段時間，簡單的遊戲製作應該難不倒你。（指路https://developer.roblox.com/zh-cn/learn-roblox/all-tutorials）

大量的免費美術資源

在工具箱的「商店」里，有官方提供的大量免費美術資源，涵蓋各種內容和類別。如果你在搜索框里輸入「hair」，那麼搜出來的不止是可以覆蓋在部件表面上的超多種毛髮材質，還有許多直接可以使用的髮型貼圖——一個換裝小遊戲的髮型選擇界面十分鐘內就可以完成了。

老實說，我身邊有不止一個朋友表示過，想參加遊戲比賽或是做遊戲，卻因為缺乏美術資源或隊友而難以實現。當然這個商店裡的美術資源主要滿足的是基本需求，勝在可以幫你快速地給自己的遊戲套好一個皮，不至於光禿禿的。

對於在畫面風格上有所追求的人，我在羅布樂思的中文論壇也看到有人做出過很不錯的畫面效果。

這是開發者Joseph155用四個星期時間製作的《23號公寓》，這種畫風和光影讓我想到了《我的世界》和《模擬人生》。

這是我在羅布樂思網站發現的遊戲《ROSES》，它的光影在網站里應該算非常優秀的了。按我的理解，這種效果能夠通過插件實現，同時選項卡里可以直接管理插件，如此一來美術效果上自然可以有更多的實現空間。

沒有伺服器成本，還能一鍵發布多個平台

但我認為這個工具最重要的兩個優勢，是沒有伺服器成本——也就是能夠零成本做聯機遊戲，以及一鍵多平台發布。

由於這款工具的客戶端和服務端是集成一體的，這使它不僅能夠免費提供伺服器，還能夠免費負責容災、運維等一系列網路端工作。

B站上有一位做獨立遊戲的Up主做了個試玩羅布樂思的視頻，他只用一小時就做了個簡單跑酷，發布之後，他還直接用PC和iPad登陸不同賬號並聯機遊玩自己的遊戲。

（視頻指路：https://www.bilibili.com/video/av76363666）

也就是說，這個一小時做出來的小跑酷已經實現了聯機功能，而且Up主並不需要操心伺服器問題。這是非常節省人力精力的，以前我還泡在MC某著名私服的時候，運營團隊有位負責網路的大神，因為熊孩子太多都被逼得退群兩次，更別提搭伺服器的成本了。

此外，根據我的理解和官網的說法，羅布樂思上的遊戲可以只通過一個按鍵直接發布在PC、安卓、iOS，甚至是Xbox和VR平台的羅布樂思客戶端，然後那些不同系統和設備上的玩家就可以直接玩到你的遊戲。當然這應該只是海外的情況，國內如何我不是很確定。

這也體現在測試選框里的一個功能上：點擊模擬器就可以看到在不同主流設備上的模擬畫面，這應該就是為了便於在各種設備上發布作品而設計的。做完之後直接點擊發布，你會得到一個網址，任何人在瀏覽器上打開這個網址，或是在任何系統與設備的羅布樂思客戶端上搜索遊戲名，都能玩到這個遊戲。

很明顯，這完全省去了中間漫長痛苦的各平台打包過程，想想只是把遊戲傳個Steam就需要多久，還得分成不同的操作系統，移動設備上也沒辦法玩到——在這裡只要點擊一下發布。

有了這些已經設計好的功能，剩下的就只是好好把遊戲創意實現出來，鑽研一個真正好玩的玩法。依賴這款遊戲的超高月活，遊戲創意與玩法的質量會很容易得到驗證。

不過這工具雖然很實用，但在國內很少人知道，網上關於這個遊戲的信息也很少。

所以我自行搜索過不少資料，發現YouTube上的個別Roblox視頻能達到7000多萬的播放量，社區（指路https://robloxforum.com；https://devforum.roblox.com）活躍程度也很高。

還有18歲高中生通過羅布樂思年收入上百萬美元的新聞，這是17年的事兒了。

但是我估計過不了多久，羅布樂思就會在國內變得更有知名度，因為那個所有遊戲都躲不過的鵝廠已經找上門來了。很明顯，羅布樂思很快就要進入中國市場了。

就看最近的新聞，它們推出了一個北極星計劃，準備花3000萬扶持創作者，另外還有技術和流量扶持，以及優先出海的名額；最近還有一個徵集作品的創意比賽。只是這部分內容比較多，我也不太記得，有興趣可以去搜一下。

我個人認為，如果大家都認可羅布樂思在開發遊戲方面的實用性，接下來鵝廠又能好好做國內落地運營，未來這可能是一個很不錯的社區。畢竟鵝廠要用心還是能做點東西出來的。當然，大家有別的想法歡迎留言討論。

官方也給創作者拉了個Q群，群里人還蠻多的，包括很多技術大神甚至是官方的技術團隊。如果大家對這個工具感興趣，或在使用過程中遇到什麼困難，可以進群來諮詢一番（群號：163421242）。

最後，祝大家做遊戲順利！
