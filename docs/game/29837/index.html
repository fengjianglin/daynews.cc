天美親兒子成功逆襲，當初你對我愛答不理，如今身價翻十倍購買

2019-12-24 2,175條評論 3,228次閱讀 786人點贊

王者榮耀到現在已經出來很長的時間了，還記得在剛進入王者榮耀的時候，首沖會送英雄，當時並沒有選擇的權利，大家都只能選趙雲！但是隨著時間的推移，天美開始了自己的另一輪新玩法，由之前的只能選趙雲，變成了三個英雄里選擇一個。所以有許多的小夥伴不喜歡趙雲，就選擇了其它兩位英雄。這裡要注意的是，其他兩位是可以通過金幣購買的。

但是趙雲如何獲得呢？畢竟當時除了首沖以外就沒有其它的方式獲得趙雲。所以為了改變這點，天美將趙雲上架了！可以直接通過點券購買，但是價格方面則有了一定的調整，不是之前親民的6元，而變成了現在的58.8元！足足翻了有近10倍，雖然最近為了慶祝KPL奪冠有一個活動折扣，但是價格還是很高，48.8元的價格讓許多玩家都接受不了！

雖然價格如此高昂，但還是有一些收集控想要湊齊全英雄，所以選擇在這個時期入手趙雲。畢竟按照這個架勢來看，趙雲想要6元購買是不可能的了，只能通過活動進行一定的減免！能便宜一點算一點不是？有一位玩家就是趁著這個活動入手了趙雲，心裡也是十分鬱悶。他進入遊戲的時間相對稍後了點，所以當時首沖6元的選擇是3選1，當時看不上趙雲，選擇了其它兩個。

結果當時看不上的英雄，現在一躍崛起了。身價足足翻了有十倍，現在想要購買可不是區區6元就可以搞定的，真的是應了一句老話：30年河東，30年河西，莫欺少年窮！當年你對我愛答不理，現在我翻了10倍你依舊要買。趙雲身為天美的親兒子，在天美爸爸的安排下這次總算打了一把漂亮的翻身仗，可惜苦的是玩家。

老張入手的時間較早，所以Q區送的英雄是趙雲。但是微信區是三選一，鑒於當時趙雲不強而虞姬很強的局面，老張微信區選擇的英雄是虞姬，而非趙雲。到了現在老張微信區依舊沒有趙雲，畢竟老張沒有收集英雄的慾望，並且這個價格真的是太貴了，而自己並不怎麼玩趙雲，趙雲的強度在這個賽季也就是一般般吧，拿幾十塊錢進階一個戰令，兩款皮膚加起來論性價比怎麼著都超過了趙雲吧？

​所以對於一些沒有趙雲的玩家，老張還是由衷的建議大家不要買趙雲，真的不值當。就算你買了趙雲，大部分還是認為你6元購買的，而嬴政不同，人家出來就是68.8元，相比起來哪個更有面子相比大家都心知肚明。而且這個賽季趙雲的強度也不是頂尖的，要不要都無所謂。當然，遊戲玩家少不了土豪黨，如果你的收集慾望很強烈，而自身又不差錢的話，就選擇趙雲吧。

最好在這個時間段內趕緊購買，畢竟還有一點折扣，能省一點算一點。那麼說到這裡，大家的趙雲是什麼價錢入手的呢？是當初的6元還是現在的58.8元呢？歡迎評論區留言討論！我是老張， 一名王者榮耀愛好者，期待你的關注！
