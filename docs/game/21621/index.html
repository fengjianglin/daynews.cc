明日方舟：重力控制，隨心所欲，安心院安潔莉娜大型角色攻略

2019-12-19 114條評論 1,077次閱讀 60人點贊

本期是第六期明日方舟大型角色攻略，也到了第六個六星幹員。

前五期的攻略大家可以在所長的主頁中查看，今天的主角是：安心院安潔莉娜。

本文將延續之前小火龍、莫斯提馬的風格，以角色使用技巧為主。

希望這期大型攻略，能為新玩家快速認識安潔莉娜，同時為老玩家提供一些新的思路。

安潔莉娜

寫在前面：如何評價角色

在這一期的大型攻略中，依舊會遵循以下四個部分去解析，分別是：

幹員面板和角色定位、使用思路指南、誤區和使用須知、常見問題答疑。

同時，所長會將角色使用心得糅合進以上四點當中，配合技能去解析。

本文中將不涉及幹員的打分，僅僅是為擁有潔哥的博士提供一份角色攻略。

安潔莉娜也是所長第一個滿潛能的幹員，已經放助戰了，證明練度和文章公信力。

所長的安潔莉娜

一、幹員面板和角色定位

面板數據：哪怕是精二滿級的安潔莉娜，面板數據也有些丟人。

安潔莉娜的面板

因為輔助的特殊性，麥哲倫雖然同為六星輔助但是是召喚系的。

所以這裡只能拿五星的真理來比較一下面板數據。

安潔莉娜：1385血、617攻、120防禦、25法抗。

真理：1280血、583攻、104防禦、20法抗。

觀察面板，可以發現安潔莉娜面板雖然略微超過真理，但不是絕對優勢領先。

對於像她們這樣的輔助幹員而言，面板數據的重要性並不高，只要別太低就可以。

安潔莉娜的攻擊範圍

其他面板數據：這裡再說一下攻擊範圍，以及攻擊間隔。

攻擊範圍方面，安潔莉娜跟其他輔助一樣，能夠打到身後三格。

這一點如果是熟悉輔助的博士們肯定掌握了，但是輔助用的不多的博士需要注意。

能夠打到身後三格的特性，類似於群奶，在一些部署位置有奇效。

59區廢墟

比如在剛剛過去的危機合約中，安潔莉娜展現了她攻擊範圍的部分優勢：完美契合角落。

無論是向左還是向下部署，能夠覆蓋到身後三格的攻擊範圍，都能照顧到整個角落。

在攻擊間隔方面，安潔莉娜是1.9秒，低於單體術士的1.6秒，高於群體術士的2.9秒。

但因為她「罰站」（技能未開啟時無法普通攻擊）的特性，常規形態下的攻擊間隔意義不大。

使用思路（一）：長時間的持續單體控制

適用於攜帶二技能「微粒模式」的安潔莉娜，能夠提供長時間的單體控制。

微粒模式：攻擊間隔極大幅度縮短，但每次攻擊只能造成相當於攻擊力40%的法術傷害。

這一點在危機合約的59區廢墟中，大家應該深有體會，請看下圖右上角：

微粒模式的控制力

可以看到，加滿了buff的紅刀哥，在安潔莉娜的控制下基本走不動路。

那麼這一份控制究竟是什麼效果呢？輔助能停頓，具體是停頓幾秒？

在明日方舟中，停頓是：減少攻擊目標80%移動速度，持續0.8秒。

而安潔莉娜的二技能，「攻擊間隔極大幅度縮短」其實是根據她自身的攻擊間隔計算的。

哪怕不計算攻速加成，微粒模式的攻速為：初始攻速1.9*0.15=0.285秒/次。

因為攻速加成比較複雜，這裡就先不計算進去了，可以發現0.285秒這個數字遠遠小於0.8秒。

而她的每一次攻擊，都將對敵人造成80%移動速度的減慢，所以可以理解為全程80%減速。

而這一份控制，時長足足有30-35秒（專三），因此安潔莉娜能夠將敵人「釘」在原地。

微粒模式：控制boss

關於微粒模式的單體控制，這裡介紹兩種思路：控制住boss，為我們騰出時間。

騰出的時間我們可以用來做很多事，比如等費用來部署重要輸出，比如清理乾淨小怪等等。

比如在上面的動圖中，我們通過微粒模式控制住了無敵的大亞當，讓他基本走不動道。

這為真銀斬騰出了寶貴的輸出空間，也讓塞雷婭緩過了一口氣，成功扛住了高壓波。

同時，這裡的動圖也演示了第二種使用思路：讓敵人更長時間處於我們的火力輸出區域內。

在上圖中，無論是真銀斬還是小火龍的灼地，走不動道的大亞當都不得不在裡面被毆打。

二技能微粒模式是安潔莉娜的核心控制技能，也是明日方舟目前最持久的單體控制。

35秒高達80%的減速，相當於敵人只向前移動了35*0.2=7秒的位移距離。

二技能起碼等級七使用，有條件推薦專精三，收益提升很大，SP消耗15持續30秒。

使用思路（二）：高覆蓋率的小火山

適用於攜帶三技能「反重力模式」的安潔莉娜，這是半個決戰技能。

為什麼說是半個呢？因為整體的傷害比不過真的決戰技能「火山」，但是卻有著其他優勢。

包括：持續時間更久，技能迴轉率/覆蓋率更高，也能更頻繁地使用。

5-10 反重力模式

比如在5-10圖中，反重力模式迎來了她最適合應對的場景：持續連綿不絕的中等壓力敵人波次。

秘杖·反重力模式：全場所有敵人重量下降一個等級，攻擊範圍擴大，攻擊力+105%，可以攻擊5個敵人

反重力模式在等級7時，消耗25SP，持續18秒（專三後持續25秒）。

所以在使用上我們完全可以把她當作小艾雅法拉來使用，廉價的火山，隨便開啟。

各位沒有小綿羊的博士，可以精二一個安潔莉娜用用看。

低部署費用的安潔莉娜配合低消耗的反重力模式，用過都說好。

使用思路（三）：特種部隊的打火機

適用於攜帶三技能「反重力模式」的安潔莉娜，主要是圍繞著重量減輕。

這個話題其實所長在上一期的莫斯提馬中就想說了，就是明日方舟中的力度與重量機制。

力度與重量參照表

力度與重量機制可以參照上面的這張表格，資料來源是PRTS資料庫。

概括來說：能不能讓敵人產生位移，是「力度」和「敵人重量」做減法得出的。

這個減法得出的數字越大，那麼敵人就會被推得越遠，同時位移時間也會越長。

被位移的弒君者

比如上圖中被迫害的弒君者，就是因為自身重量很低，導致「力度-重量」的數字很大。

而安潔莉娜的反重力模式，讓全場所有敵人重量下降一個等級，這意味著：

原本推不動的敵人，很多都能推動了；原本推得動的敵人，會位移更遠距離。

力度參照表

舉個實戰中的栗子，阿消的技能所長一直強調要升到四級，就是為了解鎖「中等力度」。

所以面對一般的敵方單位比如獵狗，重量為0，那麼「力度-重量」就是1，能夠產生1.6格的位移。

再比如面對屠夫這樣的重量級單位，重量為3，那麼「力度-重量」就是-2，不會產生位移。

但是如果配合上安潔莉娜的反重力模式，全場所有敵人重量下降一個等級。

那麼獵狗的重量變成-1，這個減法的答案是1-(-1)=2，能夠推出2.025格。

屠夫的重量變為2，這個減法的答案是1-(2)=-1，能夠推出0.1格。

別小看這個0.1格，只要有位移，就能夠打斷敵人的攻擊動作。

當敵人被擊退時，攻擊動作會因為超出攻擊距離而被強制打斷。

目前安潔莉娜能夠配合的不止是特種幹員比如阿消、食鐵獸等，也包括莫斯提馬。

這是一個非常有潛力的戰略用法，目前已經初步具備強度，以後只會更強。

安潔莉娜的其他戰略作用

安潔莉娜的兩個被動技能，都是全隊增強型的技能。

加速力場：全場友方單位攻速+8（+1）

加速立場能夠為全隊隊友提供攻速，根據大佬的測試，公式如下：

每秒攻擊次數 = [(100+攻速加成)/(攻擊間隔※間隔變化數)]/100

實際攻擊間隔 = 1 / 每秒攻擊次數

對於我們而言，其實不用計算那麼複雜，只要明白：攻擊間隔越大，攻速收益越高。

所以加速立場這個被動，對於攻速緩慢的群法比如莫斯提馬、天火，或者大劍選手比如斯卡蒂提升都不小。

兼職工作：技能未開啟時，全場友方單位每秒回復25（+5）點生命

第二天賦的治療量其實被低估了，並不是面板上的25點生命這麼低。

因為這裡的回復25點生命是秒傷（HPS），對比群奶2.85秒的攻速，能夠回復71點生命。

並且是全場回復，如果場上有六位幹員，那麼每次和群奶對比的治療量為426。

這個數據其實不止是蚊子腿了，治療量雖然也沒有特別可觀，但也沒有那麼丟人。

並且，安潔莉娜的兼職工作這個被動，能夠治療一些不接受治療的「絕食系」幹員。

首當其衝的就是火神、召喚物和赫拉格，都是吃安潔莉娜的奶長大的孩子。

安潔莉娜常見問題答疑

1、我的安潔莉娜該攜帶什麼技能？

首先我們基本不攜帶一技能，主要攜帶二或者三技能，這兩個都是神技。

如果需要安潔莉娜主打單體控制、打輔助，那麼就攜帶二技能微粒模式。

如果需要充當群法打輸出，或者組建特種部隊，那麼就攜帶三技能反重力模式。

2、技能等級需要練到多少級？

二技能微粒模式：七級，建議專精二，收益是最大的，最後一級減三點開啟SP和兩秒時間。

三技能反重力模式：七級，可以專精三，最後一級提升三秒持續時間，收益不如二技能。

一技能秘杖·速充模式：跟隨其他技能升到七級就可以了，用不太上。

3、什麼情況下都可以上安潔莉娜嗎？

只要高台位置充足，都可以上安潔莉娜。

但是要注意的是技能攜帶，要根據地圖情況自主選擇攜帶二或三技能。

如果想無腦攜帶，可以把安潔莉娜加入體系，比如：安潔莉娜+赫拉格/火神。

再比如組成控制鏈鐵三角：安潔莉娜、莫斯提馬、食鐵獸/阿消。

或者安潔莉娜+銀灰/艾雅法拉，組成大招流的循環體系等方式，都是可以的。

目前，安潔莉娜很靈活，絕大多數地圖都可以攜帶安潔莉娜，除非高台不夠。

4、安潔莉娜的加速立場，增加的攻速能夠疊加嗎？

能夠疊加的。比如配合赫墨的醫療幹員攻速提升12，能夠疊加到19。

5、安潔莉娜潛能提升收益大嗎？

作為把潔哥滿潛能的博士，不得不痛心地說：非常小。

其中，潛能3時第二天賦回血提升收益不錯，但是滿潛能攻速加成提升很小。

具體有多小呢？攻速加成從7變成了8，只加了1，說多了都是淚。

8、安潔莉娜適合搭配什麼幹員？

其實在攻略中都有提到這些幹員了，安潔莉娜在不同體系中有不同作用。

絕食系的奶媽：火神、赫拉格、召喚系幹員。

特種隊的打火機：食鐵獸、阿消、暗索、崖心、莫斯提馬。

輸出系：小火龍、小綿羊、銀灰。

之所以能配合輸出系，更多的是二技能的超強的持久單體控制。

能夠讓敵人長時間出於輸出系幹員的火力範圍之內，走不出去。
