我的世界：邊境之地有多遠？小哥徒步測量走了8年，只為找到終點

摘要

​邊境之地對於一些老玩家應該是非常熟悉的地方了，至少在mc的1.8版本之前邊境之地都是一個「著名景點」，不少玩家都希望去看看，當然大多數是用坐標定位 來穿送到那裡，可是這樣對於距離很難有一個明確的體驗，我們還是不知道邊境之地的距離究竟對於我們來說到底是個什麼樣的概念，假如說史蒂夫單純作為一個不作弊的遊戲人物，要去邊境之地只能徒步，那麼到底需要花多少時間呢？

不瞞你說，還真有人在試，這個人也是夠無聊的哈，這個人叫kurt，我們普通人可能不能理解他的毅力（無聊），但他是真的認真的在做這件事，從2011年3月開始，一直到今天還沒有走完！這就是那個無聊的小伙，表情是不是很滑稽

他只在有象徵意義的日子裡按f3，來知道自己走了多少距離，還有多遠，目前我們知道他在2015年3月31日看了一次，也就是走了四年後，他發現自己僅僅走了百分之十八的路程！那麼如果我們換算一下走百分之百的話他這個速度大概需要21年！相當於一個普通人壽命的四分之一，而且這還是在史蒂夫不知道渴和餓也不知道休息的情況下，看來邊境之地還真是遠啊。

不過這個小伙也受到了一點挫折，mojang在1.8版本中去掉了邊境之地，這導致他沒法更新後在1.8裡面繼續旅行了，不過他為了完成自己的偉大嘗試，決定不更新，繼續停留在1.7.3的版本旅行。看來他為了走路和我們現在mc玩家落後了好幾個時代呢，想想如果他走完，mc會變成什麼樣子呢？肯定他已經認不出了吧。

大人，時代變了：

有的玩家或許會問，他真的有這麼無聊嗎？就沒做點別的事？他還真做了，他把自己旅行的視頻都發了油管上，並且以此來發起了child’s play charity的捐款，看來他還蠻有愛心的呢。

Ps:還有沒有小夥伴沒去過邊境之地呢？我們有個好方法就是穿送，指令就是/tp 0 130 12550820，邊境之地也有個邊境之角，其景色比邊境之地更加奇特，這個地方在/tp 12550820 130 12550820，希望小夥伴們去看看哦。不過要注意幾點，首先這個坐標是手機版，其次如果手機配置不夠高可能會卡卡的哦~祝小夥伴玩的愉快。

最後要說的就是，小夥伴們覺得這個kurt的旅行最終會是個什麼結果呢？歡迎留言發表你的看法。
