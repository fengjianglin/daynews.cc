成功出海卻“浪”不起來的王者榮耀，究竟是哪裡得罪了國外玩家？

天天要聞
1 週 ago

《王者榮耀》——國內誰人不知、誰讓不曉的「國民手游」，上線以來穩坐王位一直至今，雖幾受衝擊，但就算你是手游界的當紅炸子雞碰上「頂流」王者營業，還是說被秒就被秒，排行榜更新分分鐘。

可是就是這樣一款國人慾罷不能的，遊戲影響力覆蓋全國各地、男女老少，收入盈利讓無數企業望塵莫及的史上最成功手游，為何在高調宣布要揚帆「出海」後，卻瞧不見太多的浪花蹤跡了呢？拋開表面的粉飾太平，發展真相其實早已是有目共睹。為何國內大家都說好，國外卻「不香」？《王者榮耀》的水土不服竟如此嚴重？

作為電子遊戲發展「先驅」的歐美國家，一說玩遊戲，絕大部分人想到的都是電腦端游或各種主機遊戲，手游絕對不會是他們腦海中出現的第一畫面，原因也很簡單！一是「歷史遺留」問題，從小到大的遊戲經歷讓他們已形成一套完整的遊戲排位，手游毫不保留地排在了末位。

二是歐美「大農村」，缺乏了讓《王者榮耀》蓬勃發展的基礎設施建設，出行大多得靠自力更生，導致公共交通玩手游的樂趣嚴重缺失；網路鋪設不到位，連4g都不普及的地方，聯網開黑說斷就斷，豬隊友此起彼伏；沒有微信和QQ，還不連FB、Twitter，想找人開黑也不簡單。對於這些，外國人想說《王者榮耀》愛你真的不容易。

就是以上種種如此，讓中國國民MOBA手游《王者榮耀》的出海一路險阻，與之形成鮮明對比的則是今年在國外異軍突起的《AFK Arena》，國內的名字叫《劍與遠征》，一款輕鬆策略放置類卡牌手游，準確命中了歐美玩家對手游的各種需求特性，成績一路飄紅，評分也是節節高。

作為一款放置掛機遊戲，AFK可以隨玩隨停，況且掛機也還能有收益，充分滿足了外國人更碎片化的手游時間需求；作為一款有策略的卡牌遊戲，AFK在手機屏幕5V5的小戰場里，為玩家提供了豐富多樣的戰鬥元素，屬性克制、能量恢復、時間流失、空間關係等等，遊戲可玩性十分之高；

作為一款輕休閑類手游，AFK英雄技能全程自動釋放，搭配上即時制的戰鬥，無需精妙操作也能讓手機屏幕有炸裂的戰鬥畫面；作為一款率先在國外上線的手游，AFK的背景故事是西幻題材、美術風格是精美的玻璃彩繪，都更能被歐美玩家所接受，評論區中關於畫風的精美和音樂的炫酷也是多次被提及。

所以就是這樣一款有國內莉莉絲遊戲出品的，《小冰冰傳奇》原班人馬創作的手游，在GOOGLE PLAY和FACEBOOK都拿下了4.7/5.0的高分，各種表情包輸出在Reddit小組（美版貼吧）、Instagram等各大社交平台上比比皆是，一舉成為了國內出海手游中收入最高的產品之一了，獲得了「牆內開花牆外香」的完美開局！
