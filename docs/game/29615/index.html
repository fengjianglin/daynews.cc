王者榮耀：皮膚簡介牛魔最短，宣太后最長，粉絲：都是我們爭取的

2019-12-25 2,465條評論 4,315次閱讀 2205人點贊

不知道有多少玩家注意到，其實大部分的皮膚在皮膚界面都會有對這個皮膚的大概解釋，一般包括了外觀，特效，語音，回城特效等。

孫尚香的殺手不太冷和劉備的黑幫教父算是半個情侶皮膚，對於殺手不太冷的簡介中提到，這個皮膚的外觀是一個冷酷又不柔情的殺手形象，回城特效是首個雙角色同場的回城動畫，隨皮膚會有頭像框，皮膚還有專屬的出生動畫。

孫尚香的末日機甲皮膚外觀是用機甲戰士風格，有數字流的技能特效，音效中有數碼音效，回城中有機車飛旋的特效，看過這個特效的玩家都覺得酷，皮膚的打擊感還不錯，可能這就是大部分國服孫尚香都喜歡末日機甲皮膚的原因之一吧。

夢奇的胖達榮榮皮膚絕對是王者榮耀裡面最可愛的皮膚，圓嘟嘟的身體，帶著臉譜，有著熊貓的外形，還有「火鍋兒好次」的四川話語音特效，還有好玩的「變臉」個性動作，是所有夢奇玩家的最愛。

最簡單的就是牛魔的西部大鏢客的簡介，標題就叫做嘗一下仙人掌的厲害，只有一句外觀介紹，我的仙人掌可比那隻猴子厲害多了，牛戰士看來是天生的不服猴子，雖然都是西部大鏢客，可能是因為鐵扇公主的原因，牛戰士對猴子有深深的鄙視，只想揍他。

虞姬的雲霓雀翎皮膚介紹中提到他的特效中有雲霓的雀翎飄動的技能特效，音效中有雀鳴的，背景用的是蜀綉中千里江山圖的屏風，回城特效中包括了湖光月色和翩然起舞的效果。

孫悟空的地獄火皮膚中出生動畫用了蹦出來的悟空，契合孫悟空從石頭裡面蹦出來的神話故事，語音特效中加入了地獄迴響的特效音樂，回城特效中有金箍棒插入大地和地動山搖的特效。尤其是孫悟空幽冥火的皮膚，更加酷炫。

簡介中最長的是羋月的大秦宣太后，這個皮膚是榮耀典藏皮膚，本身皮膚的等級就非常高，不過大秦宣太后最開始可沒有這麼多的特效，都是粉絲不斷地給官方反映之後，爭取過來的，現在大秦宣太后中加入了行走動作、皮膚專屬死亡特效等，總算是對得起她榮耀典藏的身份了。

以上就是豪哥總結的皮膚簡介相對最多和最少的英雄，不知道大家滿不滿意
