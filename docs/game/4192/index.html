夢幻西遊：還是策劃想的周到啊，這麼做又給玩家省錢了

2019-12-09 0條評論 16次閱讀 0人點贊

摘要

遊戲的意義就在於它能夠給人帶來快樂，如果過多的摻雜其他的東西就失去了其本身的意義，大叫好，我是小三，每天給大家分享遊戲中的八卦趣事。

遊戲的意義就在於它能夠給人帶來快樂，如果過多的摻雜其他的東西就失去了其本身的意義，大叫好，我是小三，每天給大家分享遊戲中的八卦趣事。

召喚獸進階的過程中會使用到兩個物品，一個物品叫易經丹，召喚獸使用五個易經丹能夠讓召喚獸的靈性達到50，並且讓召喚獸的形態發生轉變，召喚獸的靈性達到50之後就能夠使用另一個道具清靈凈瓶來進一步提升靈性，每一點靈性能夠給召喚獸增加兩點屬性點。

No1.恐怕這位玩家再也不想玩這個遊戲了

清靈凈瓶是通過清靈凈瓶獲得的，玩家開啟清靈凈瓶之後能夠獲得各種品質各種等級的凈瓶玉露，因為品質和等級的不同，價格也不一樣，高級八品的清靈仙露是最貴的，所以有很多玩家喜歡通過清靈凈瓶來驗自己的運氣，但是對於這位玩家來說他的運氣好像不怎麼好，大家看著玩家開清靈凈瓶整整開了一行囊的低級、中級的清靈凈瓶，而購買清靈凈瓶需要花費的成本卻高的嚇人，估計這位玩家以後再也不想體驗這個玩法了。

No2.不要輕易的開年卡

很多玩家都喜歡將點卡轉換成月卡或者是年卡，就像是這位玩家剛剛回坑，為了表明自己會坑的決心直接就開了一張年卡，整整1400個小時的遊戲時間足夠玩很長的時間了，但是這位玩家剛剛回坑，自己剛剛創建的角色才升到35級就堅持不住了，但是這位玩家不能退坑，要不然這140點三界密令和1400個小時左右的遊戲時間就真的全部都打水漂了，所以大家在決定充年卡之前先考慮自己能不能堅持住。

No3.還是策劃想的周到啊，這麼做又給玩家省錢了

在夢幻中每一隻成型的召喚獸都經過了很多次的打造，從胚子的合成，再到召喚獸獸訣的學習都有可能將一隻召喚獸給打廢，其實從側面想一下，召喚獸打廢之後也能給自己省一大筆的錢。

大家看這隻九技能的須彌持國巡守怎麼樣？雖然她的成長比較低，法力資質也不行，但是法寵是受成長和資質影響最少的召喚獸了，而且對於須彌真言而言對法傷影響最大的是法力點數，只要能夠將屬性點全部都加在法力上，依舊能夠打出超高的傷害，不過這隻持國巡守還少一個大法技能。

但就是這一本大法讓這個玩家感受到了來自系統的惡意，一隻擁有九技能的須彌草在打一本大法的時候正好將須彌真言給頂掉了，只能說這玩家的運氣太差了，1/9的幾率都被他碰到了，不過這樣下來也省錢了，要不然這玩家還得將這隻持國巡守給打成全紅。
