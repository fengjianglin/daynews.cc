「分享遊戲故事，彙集歡聲笑語」~大家好，我是菜鳥熊貓~

歡迎來到由菜鳥熊貓主講的《元氣騎士》大型欄目《元氣搞笑日常》第146期~

《元氣騎士》機甲體系出現的時間真不短了，許多玩家也就一時體驗一下，並不會去細心研究。玩家的態度在一定程度上也影響了涼屋遊戲工作室，這一年來新武器新地圖新BOSS都出現了不少，其中融合武器雷劍以及天堂之拳和光劍土豪金都受到了玩家的好評！但是，你見新機甲出現了嗎？難道機甲真的只有一時的快感？其實，深度挖掘一下機甲還是很有內容的！下面我就同大家分享狼人同機甲的奇妙之旅吧~

狼人手刀DZT模式，機甲+狼人=免傷，現版本可用

《元氣騎士》當中的狼人至今也沒有第二技能，僅有的一個技能就是狂化，狂化時狼人的血量越低傷害就越高，如果變身時的血量低於半月，開啟狂化回恢復血量。5星狼人也就是滿級狼人會額外多一個屬性，變身時會獲得傷害減免！這裡是重點，記住了。

​

機甲出現在《元氣騎士》的時候也是有護盾的，機甲的護盾同角色的護盾是2個體系。角色的護盾相當於角色的血量，能夠抵擋傷害。機甲的護盾就是護盾，如果說你受到了5點的傷害，機甲護盾為1，那麼你受到的真實傷害為4！這裡也是重點，記住了。

​

開啟技能的狀態下我們是不能夠穿上機甲的，但是《元氣騎士》的特色小夥伴們不能忘記了，「特性」多多BUG多多。選擇狼人角色，我們同時點擊技能按鍵和攻擊按鍵這一瞬間是可以進入機甲的。狼人的技能特性同機甲的護盾就解鎖了免傷！我們再進一步思考，狼人手刀通關DZT就不是夢了~

穿著天啟機甲可用光劍土豪金，厲害！

我們穿上機甲之後是不能夠使用角色手持的武器的，天啟機甲也不例外。還是上面說的，涼屋特色多啊，特產就是BUG。熊貓我就在穿著機甲的狀態下使用了光劍土豪金，這組合太完美了！有圖有證據，跟著熊貓我往下看~

​

具體操作：

我們必須同時持有光劍土豪金和天啟機甲，在地牢當中必須要遇到配件商人，讓他給你一個全自動裝置！全自動裝置同天啟機甲有概率會觸發「特性」~

​

特性狀態如圖所示，機甲攻擊的同時光劍土豪金也沒閑著~

騎士皮膚「電子騎士」和機器人皮膚「警備隊9102」免費獲得

《元氣騎士》中一些皮膚是充錢也買不到的，只有特殊渠道或者是官方贈送才能夠獲得。機器人的「警備隊9102」便是這樣的，曾經熊貓就分享了「警備隊9102」的獲取，許多小夥伴遺憾錯過了。

​

不要灰心，官方再次開啟了皮膚贈送，想要獲得機器人皮膚「警備隊9102」的小夥伴可以進入《元氣騎士》KT論壇參與其中，小錦鯉說不定便是你~
