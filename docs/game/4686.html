絕地求生：吃雞賽事驚現一殺吃雞 實力不行苟來湊

2019-12-05 0條評論 8次閱讀 0人點贊

摘要

絕地求生雖然是FPS類遊戲，不過很多時候玩家還是把它當作一款生存類的遊戲，畢竟吃雞遊戲最終的目標是吃雞，玩家怎麼玩都可以只要成功吃雞就可以了。很多主播為了一些娛樂效果經常進行一些吃雞挑戰，比如手槍吃雞、噴子吃雞、弩吃雞，更多的還是進行戰場記者性質吃雞。

絕地求生雖然是FPS類遊戲，不過很多時候玩家還是把它當作一款生存類的遊戲，畢竟吃雞遊戲最終的目標是吃雞，玩家怎麼玩都可以只要成功吃雞就可以了。

很多主播為了一些娛樂效果經常進行一些吃雞挑戰，比如手槍吃雞、噴子吃雞、弩吃雞，更多的還是進行戰場記者性質吃雞。

全隊1殺吃雞的情況或許在路人局上經常出現，但是也是不常見的，畢竟現在玩家的水平也都上來了，想要1殺吃雞非常的不容易。

而就在第三屆微博杯AD組的賽事上，在路人局都不容易的1殺吃雞出現在了電競賽場上，而且還是比較高層次的賽事，這就有點蒙了。（微博杯的32戰隊基本都是最高層次的戰隊了）

先來說說當時的情況吧：
第三屆微博杯AD組的第四場雨林地圖賽事上，SSS戰隊3-1分站，3個人守在房區，1個人在山崖上探點，不過房區很容易被LStars戰隊衝下了，SSS戰隊在房區的三名成員表現得非常的差沒有拿到任何淘汰分。
在只剩下一名成員的情況下，SSS戰隊的Brong直接苟著，而且運氣也非常的好周邊三支戰隊LStars、QM和YTG相互打著，最終只剩下LStars戰隊的一名成員，SSSBrong趁機起來淘汰LStars戰隊成功吃雞。

這或許就是實力不行苟來湊吧，畢竟SSS戰隊在微博杯AD組的幾場比賽上表現得非常的差，前三場海島地圖只拿到了1分，且還是有一局是天命圈的情況下拿到的分，而雨林地圖SSS戰隊也是各種的天命圈，可惜表現差了點沒有拿到分數，如果不是最後的苟吃雞那麼SSS戰隊最終的總分就尷尬了。

雖然感覺SSS戰隊這把吃雞有點拉低了檔次，不過這也算是絕地求生的魅力了吧，畢竟職業賽事上也能一殺吃雞，那麼我們這些槍法不行的人想要吃雞也是可以的。
