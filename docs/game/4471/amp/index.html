[摘要]相信大家都是知道的，在峽谷之中，像明世隱這類輔助搭配成吉思汗這種射手是真的很強勢，前期就有不俗的傷害，那麼在現版本該如何克制呢？下面筆者就來說一下吧！

首先第一個就是我們的白起了，他就比較簡單粗暴了，看到小明牽著C位站在一起了，那麼果斷不說二話，大招上去就是一個嘲諷，命中兩人，二技能回拉，配合隊友的傷害，直接拜拜！

第二個就是我們的達摩了，這個戰士英雄在排位之中絕對是頂尖的存在，受到了很多小夥伴的喜愛，出場率還算是不錯的，而且他極其克制小明組合，大招直接上牆，一套帶走！

第三位則是我們的幹將莫邪了，相信大家對其都很熟悉，手長並且傷害非常高，因此在排位賽之中，如果遇到小明組合，那麼不好意思了，直接開啟大招，裝備起來的話，四劍命中瞬間就秒殺C位了！

最後一位則是我們的盾山了，作為峽谷頂尖的輔助英雄，其也是比較克制小明組合的，因為盾山的過肩摔是能夠將其直接背回去的，就算是兩個人也一樣，二閃過去背回來，大招一開，隊友輸出跟上，你還想做啥？對此，大家有何看法？歡迎討論！
