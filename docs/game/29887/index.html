為何亞索不是第一，《英雄聯盟》年度掛機排行榜引熱議

2019-12-23 164條評論 3,246次閱讀 138人點贊

今年的《英雄聯盟》排位賽季已經落幕，官方的不少統計數據都已經完成公布。根據」League of Graphs」網站的統計數據可以看到，在2019年中的掛機率英雄排行榜。在峽谷中掛機率前五名的分別是：布隆以所佔比例2.28%的掛機率高居榜首，1.8%比例的亞索居於第二位，劍聖以1.73%的佔比緊隨其後，狗頭內瑟斯1.71%第四位，劫的1.69%排在第五位。（數據樣本不包含中國大陸地區）

　　布隆和內瑟斯這兩個英雄的掛機率還是比較讓人意外的，畢竟他們和那些很容易放棄或者極易暴怒的玩家常玩的英雄特性有很大不同，但現在的掛機率卻的確排在了前列。布隆的登場率很低，據統計顯示大約只有1.5%，他在輔助英雄裡面的出場率排在了22位，使用得並不頻繁。因此這個英雄有如此高的掛機率很可能是由於比較低的登場率所導致的。畢竟布隆的盾十分考驗玩家的實力，要是沒有發揮出應有的作用，隊友肯定會大說特說，心理承受能力差的玩家會選擇掛機的幾率就很大了。

　　而上了榜首的其他幾位英雄亞索、劍聖、劫可以說是玩家們討論聲望最低的英雄，玩的玩家數量非常多，在匹配中幾乎每局都能見到其中之一，再加上玩家水平良莠不齊，極易燃起怒火，然後掛機不幹了。其中劫和亞索完全就是潛在的孤兒隊友，相信大多數玩家都經歷過劫或者亞索掛機的戰局，戲稱「托兒索」和「兒童劫」。除此之外，偏向團戰而且操作易上手的英雄譬如娜美、澤拉斯等掛機率就很低，畢竟體驗感好。

　　在給出的單雙排為掛機率的統計數據中，掛機率第一位的是青銅段位佔比15.9%，然後是第二的白銀，黃金、鉑金和鑽石緊隨其後。

　　有網友評論「玩布隆的都知道，隊友太二自己什麼都幹不了，只能掛機」，看來是一把辛酸淚的真實經歷了，似乎也側面道出了布隆掛機是因為對內配合不好？

　　另外也有網友評論」我以為亞索是第一「，不少玩家看來布隆第一還是有點」意料之外「的意思了。
