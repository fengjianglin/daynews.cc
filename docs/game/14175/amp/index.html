八個人玩左慈會發生什麼？三國殺幻化之戰揭秘，活到最後才是贏家

天天要聞
2019-12-10

左慈，因為技能變化多端而一直受到玩家朋友們的喜愛，甚至很多小夥伴會一直點將左慈來玩。但是，一局遊戲最多只能一個人玩左慈，其他人就只有羨慕的份，可如果現在告訴你，有一個模式可以讓所有玩家都變成左慈，你想玩嗎？想的話就快去更新三國殺移動版379版本，來體驗活動場的全新模式——”幻化之戰”吧！

雖然說是八個人玩左慈，但為了區分，也不可能真的讓八個相同的武將上場，所以還是會給玩家選將框選擇自己喜歡的武將。但這裡選的武將是沒有他們原本技能的，可以理解為它就是一張代表玩家個人形象的”皮”，所以隨便你喜歡。

八個人全部選定之後，需要從三個常規技能中選擇一個作為初始技能，三個選項全場相同，所以會出現全部玩家同一個技能的情況。

之後遊戲開始，六、七、八號位會額外獲得一張手牌；七、八號位會獲得一個”靈力點”。靈力點的用途有兩個，一可以在出牌階段棄掉換取等量的牌；二可以在回合開始消耗兩個來為自己增加技能，由系統隨機給出三個技能選項，玩家在其中選擇一個。但每名角色最多只能擁有三個技能，所以後期想要擁有新技能就要在原有技能中選擇一個進行替換。而且因為是系統隨機的，所以可能會分到很好的也可能會分到”耀武”、”燃殤”等負面技能。

靈力點會在每一輪固定增加一點，除此之外擊殺其他角色，特別是左上角指示的擊殺目標時都能獲得靈力點獎勵。對指示保護的角色出桃等也可以獲得靈力點，而如果保護角色陣亡，則你也會受到失去一點體力的懲罰。

在一名角色的回合結束後，如果這個回合有角色陣亡，系統會進行一次”乾坤變化”判定，根據不同的卦象會有不同的效果發生，八個卦象對應的效果落櫻會在下期文章中為大家介紹，此處暫時按下不表。

判定之後，左上角的擊殺和保護任務會發生變化，小夥伴們要及時關注，避免出現失誤讓自己白白失去體力。不過因為這個任務也是系統隨機頒布的，所以有時候會出現擊殺和保護目標是同一個人的情況，稍微有點難受。

在存活角色減少到四人及以下時，遊戲會進入”死斗模式”，此時乾坤變化將不再進行，同時左上角擊殺和保護任務也會消失，並且如果不能在自己的回合內對其他角色造成傷害，回合結束時就會收到失去一點體力的懲罰。這個時候就非常體現實力啦，技能的搭配和控場都非常重要，膽大心細才能活到最後。

好啦，以上就是一局”幻化之戰”的大概流程，細節部分落櫻會在接下來的文章中為 大家慢慢探究，記得持續關注哦。如果你覺得這個模式很新奇好玩，那還在等什麼，趕快去關係三國殺吧！
