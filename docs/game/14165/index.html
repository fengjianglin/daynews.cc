CFS丨12.9回顧：八強誕生，中國巴西六隊全部晉級

2019-12-10 1,169條評論 3,602次閱讀 692人點贊

今日比賽結束後，CFS2019世界總決賽小組賽落幕，十二支戰隊經過三個比賽日的戰鬥，最終誕生八強隊伍。中國三支戰隊全部晉級，同時巴西三隊也進入八強之列。一起來回顧下今天的所有比賽。

GV狀態在線，KZ提前出局

這是今日的第一場比賽，B組敗者組，輸了就直接出局。這樣至關重要的一場比賽，兩支隊伍的狀態卻截然相反。GV開局氣勢如虹，主動掌控比賽節奏，KZ過於被動，槍感狀態受到很大影響，上半場GV大優勢，下半場KZ徹底崩盤，GV非常順利的拿下比賽勝利。KZ戰隊由於在敗者組中失敗，成為第一支被淘汰的隊伍。

後來追上，IMP驚險帶走LZR

這一場比賽非常激烈，戰況更是跌宕起伏。初期是LZR的節奏，IMP在很被動的情況下主動求變，通過調整小戰術追分，LZR的防守漏洞暴露出來，但這時候雙方差距並不是很大。來到下半場，戰況更加膠著，比分一度持平。關鍵時候，DANIMAL三殺滅隊，IMP拿到賽點，並且在最後一個回合拿下殘局，驚險獲勝。

白鯊火力全開，力壓ANB進八強

經過昨天AG的勝利，白鯊喘了一大口氣，今日對戰ANB，他們火力全開比較順利的獲得勝利，進入到世界八強。經典國圖黑色城鎮，白鯊率先擔任的潛伏者，一鼓作氣拿到領先分數，但因為失誤被ANB咬住比分，上半場沒有太大差距。來到下半場白鯊繼續積極作戰，主動頂防連續得分，ANB一分未得，白鯊獲得八強資格。

SV超強應變能力，穩勝GV

這兩支隊伍的槍法都很強，個人能力也非常突出，SV能夠順利晉級八強，主要是因為他們在臨場應變上做的很出色。前期GV進入狀態快，SV通過個人爆發咬住比分。但在SV摸清GV的作戰思路之後，SV掌控全場，取得優勢。後期同樣是頭腦大戰，SV保持優勢穩紮穩打，最終戰勝對手，晉級八強。

IMP乘勝追擊，兩連勝挺進八強

在上一場比賽中IMP較為驚險的戰勝LZR，這一戰對上AT戰隊，IMP明顯得心應手。上半場AT打法穩健，IMP同樣節奏，雙方分數相差不大。下半場戰局突變，藉助GDSZ的四殺，IMP突然火力全開，簡直勢如破竹。AT戰隊面對攻勢太強的對手，沒有很好的應對之策，最終遺憾失敗，IMP挺進八強。

小組賽已經全部結束，八進四的比賽將在明日打響，中國三隊雖然順利進入到八強，但真正的難關才剛剛開始。加油！
