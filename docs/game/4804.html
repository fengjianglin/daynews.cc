狼人殺和第五人格竟能合二為一？國產遊戲風暴島社交競技兩相宜

2019-12-04 0條評論 8次閱讀 0人點贊

摘要

在桌游領域，非對稱性競技遊戲早已非常流行，狼人殺等遊戲在國內受到了很多玩家的喜愛，而在手機遊戲領域，第五人格這樣的動作型非對稱性競技遊戲也讓人耳目一新。那麼問題來了，狼人殺和第五人格加起來會怎麼樣？

在桌游領域，非對稱性競技遊戲早已非常流行，狼人殺等遊戲在國內受到了很多玩家的喜愛，而在手機遊戲領域，第五人格這樣的動作型非對稱性競技遊戲也讓人耳目一新。那麼問題來了，狼人殺和第五人格加起來會怎麼樣？如今風暴工作室的新遊戲《風暴島》快要上線了，這就是一款創新性非對稱性競技遊戲，和第五人格不同的是，《風暴島》在遊戲剛開始時甚至看不出明確的陣營對抗，那麼這款遊戲到底哪兒好玩，小編下面就來詳細講一講。

《風暴島》是一款同時為PC和手機端設計的非對稱性生存競技遊戲，採用3D半寫實畫風，角色設定也各具特色，儘管客戶端不大但畫面表現還是相當真實。遊戲的背景設定是飛機失事導致你流落到名為風暴島的一個小島，連你在內一共有8名倖存者。天黑之後就會有巨大的風暴來摧毀一切，所以大家必須齊心協力造好船逃離這裡，但《風暴島》的詛咒讓兩個人迷失了自我，這兩個人就叫迷失者，迷失者認為留在島上可以永生，打算暗中破壞造船。唉，no zuo no die，這種人不可活……

遊戲的標準模式規則並不複雜，一局遊戲有8名玩家，其中有6名倖存者，2名迷失者，大家出現在一個小小的島上。倖存者10分鐘內造船成功則勝利，10分鐘沒造成功則迷失者勝利，當然一方玩家全滅的話另一方立刻勝利。每個角色的血量掉到0的話自然就掛了，不過由於大家從昏迷中醒來都餓得不行了，所以先得在地上找到椰子吃才有力氣攻擊別人，所以要直接用攻擊淘汰別人並不是一件容易的事。

我剛開始玩的時候光想找椰子吃，以為這樣就能當監管者想砍誰砍誰了，不料椰子刷新的速度還挺慢的，我一直找椰子老不修船，結果被眾人當壞人給圍毆了，可惜沒能留下遺言……嗯，說好的自由行動呢？你們咋不講道理啊？看來這個遊戲中雖然想幹什麼就幹什麼，哪怕是當狼也可以上來就跳明狼，不過幹什麼都要自己負責哦！

既然砍人是不能隨便砍的，那麼砍樹造船自然是遊戲的第一要務。不過和一般生存遊戲不同，《風暴島》中砍樹居然不能一個人砍！要3個人一起砍才能砍倒一棵樹，誒，這不是故意給壞人留下作案機會嗎？玩迷失者可以在砍樹時選擇搗亂，有人搗亂砍倒樹後參與砍樹的某個生存者就會掉血。下來這一局我拿到迷失者上來發現基本沒人聊天，我掐指一算，嗯本局必定新手很多，那我上來就搗亂，然後裝作小白不吭氣，果然不一會就看見一幫萌新倖存者互毆了，勝利so easy！

砍樹搗亂是一種非常明顯的破壞方式，但是《風暴島》也同時具備談話推理玩法，每個人都能發言讓全體玩家聽到，要是開局大家輪流有序發言，那麼大概率這一局都是玩過推理遊戲的玩家，這種情況下隨便搗亂，我試過了，剛砍3顆樹就被好人分析出是狼，死的那叫一個快555……所以遊戲還設計了很多其他道具讓雙方拾取，使得遊戲的玩法兼備操作性和策略性。再碰到這樣的局我當狼也得先裝好人，前期該修船修船，該把腐木燒掉燒掉，等隊友吸引了好人們的注意力，我再搞破壞不遲，沒錯，玩狼人殺的感覺來了！

桌游中的非對稱性競技遊戲都是基於情報不對稱方面來設計，壞人陣營的人少但是有著更多的情報，而《風暴島》也具備這一特點。雖然開局大家看起來都是好人，但風暴石這個道具可以拿到造船部件處召喚風暴。風暴降臨時沒造好的船隻部件一下子就被吹走，在風暴持續的一分鐘時間內，迷失者可以看見隊友是誰，倖存者則看不到其他人的任何信息，此時迷失者就能找到機會來搞破壞，在別人認不出你的時候使用炸藥、腐木將造船的部件毀壞，勝利就在眼前。

當然你演技出色，玩迷失者也可以在有人看見的時候搞破壞哦，再巧舌如簧栽贓好人，煽動倖存者互相攻擊，甚至可以幹掉同伴來騙取好人信任……哎呀你開局玩自刀？是個狼人啊！騙的我好慘！這種一局遊戲玩完還要復盤半天的感覺，完全能和當面玩狼人殺比！

而玩倖存者相對被動，但更重視團隊合作能力。如果沒人敢搞破壞，那造好船就是分分鐘的事情，但《風暴島》中其他玩家跑遠後你就看不見了，因此好人必須時刻提防有人搞破壞，大家一都散開去砍樹並不是什麼好的策略，實際上正確的方式是一部分人砍樹修船，另一部分人要專門監視別人的行為。玩倖存者就得合理分工才能認出壞人，確定壞人後並用漿果加以標記，被標記者即使在風暴中也能被全體玩家看見，很容易集火消滅之。

當然玩好人推理能力也是必須的，有些憨憨玩家總是會誤標好人，有的局我玩好人就看見有玩家莫名其妙的標記人，又不開麥說明原因，大家都不知道該不該集火，於是狼就渾水摸魚輕鬆摸到了時間結束，這種局就算我再會分析也帶不動好人啊！所以無論你玩的技術好不好，一定要開麥交流，就算在第五人格里，玩求生者也是類似的道理對不對？當我碰到好幾個不開麥的隊友時，只有祈願修船時運氣好掉落線索道具，直接驗出狼的身份了，不然就是欲哭無淚等著輸！

從之前測試的情況來看，《風暴島》還是很好玩的，操作簡單又具備多變的戰術，同時還具有很強的社交性，無論是推理愛好者還是社交愛好者，這款遊戲都是你不容錯過的驚喜之作！遊戲目前還在內測階段，隨著角色技能和其他玩法的豐富，遊戲的可玩性將進一步提升，有興趣的玩家可以關注taptap平台上的這款遊戲。
