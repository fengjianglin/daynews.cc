kpl預選賽再現老面孔：曾力壓縱情勝過Estar，如今再出發轉成射手

天天要聞
2019-12-05

摘要

2019年kpl冬季預選賽正在火熱的進行中，眼尖的小夥伴們也是發現，這屆預選賽中有很多的老熟人。他們從kpl賽場上下來以後，他們一直沒有放棄自己的夢想，一直在堅持著，從預選賽征戰，就是盼望著有一天能重新回到kpl的舞台。

2019年kpl冬季預選賽正在火熱的進行中，眼尖的小夥伴們也是發現，這屆預選賽中有很多的老熟人。他們從kpl賽場上下來以後，他們一直沒有放棄自己的夢想，一直在堅持著，從預選賽征戰，就是盼望著有一天能重新回到kpl的舞台。之前我們有過盤點，像VV、阿燦、笑煮等老選手都是這屆預選賽上的實力擔當。而除此之外，又一個老面孔出現在kpl預選賽的舞台，他曾力壓縱情成為青訓營的打野狀元，如今再出發轉型成射手，他就是諾龍！

2019年kpl冬季預選賽

kpl預選賽老面孔不斷，VV阿燦等回歸讓人期待！

VV

在這屆kpl預選賽中，有很多我們熟悉的老面孔，比如以前AG超玩會的上單VV，還有sViper的阿燦和笑煮，都是大家非常熟悉的老選手，尤其是VV更是kpl上的元老，和AG超玩會並肩作戰幾個賽季，是kpl上有名的五大關羽之一，更是有一手連勝的花木蘭讓人驚嘆。還有阿燦和笑煮，雖然sViper的成績不怎麼樣，但是在kpl上也是給我們留下很多經典的記憶！而如今他們征戰預選賽，也是為了重新回歸而努力，在比賽中給我們帶來不少的驚喜。

諾龍回歸預選賽，助陣DVG取得勝利！

諾龍

在DVG對陣到CGX的比賽中，DVG的陣容中出現一個大家非常熟悉的老面孔，那就是諾龍。諾龍曾經是sViper的打野新人，但是因為阿燦的原因，上場的次數並沒有很多。後來在阿燦轉到輔助位之後，諾龍才成為主力打野。在那個賽季，sViper的成績不是很好，難逃降級的命運。諾龍也是隨著隊伍來到預選賽，來到次級，逐漸淡出大家的視野！

曾經是青訓營的打野狀元，力壓縱情！

縱情

諾龍當初是青訓營的打野狀元，那時的青訓營和現在的選秀大會不同。現在的選秀大會狀元是指第一個被選中的人，而在當時狀元卻是靠自己的實力打出的分數來確定的。而在諾龍那一屆，縱情也只是榜眼而已，由此可見諾龍的實力。記得諾龍上場的時候，還贏下過Estar呢！然而讓人沒想到的是，當縱情在賽場上大放異彩，一手韓信迅速成名的時候，諾龍卻並沒有完全發揮出自己的實力，從而導致諾龍的實力被掩蓋，慢慢的談論的也少了！

從打野到射手，依然是最carry的點！

諾龍

轉到射手邊依然carry在此次預選賽中，諾龍的上場非常的讓人驚喜，而在位置上，諾龍從打野轉到的射手邊，依然是最carry的那一個點，他的成吉思汗9-1-2的戰績，真的是成為隊伍中當之無愧的核心。其實在2017kpl秋季賽的時候，諾龍就打射手了，那時主要是用射手打野，但是由於整個隊伍的問題，一直沒有很大的發揮，如今正式轉到射手邊，諾龍也是把自己的carry發揮得淋漓盡致，更是讓DVG成為本屆預選賽的奪冠熱門！

諾龍當初在kpl的賽場上匆匆而過，如今來到預選賽的舞台，也是再次重新出發，我們也是祝福諾龍和DVG，在預選賽能取得一個好成績！
