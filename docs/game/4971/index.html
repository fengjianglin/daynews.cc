《英雄殺》手游攻略，3V3排位上分套路組合

Hello~大家好，我是娛樂遊戲風，今天我們來說說三國殺。在《英雄殺》手游這款遊戲中，英雄的選擇至關重要，今天小編為大家帶來三套排位上分的陣容，希望對大家有所幫助哦~

【推薦套路組合1：宋江+褒姒+扁鵲】

這個陣容的主要輸出是褒姒，宋江的疏財配合褒姒和扁鵲的治療構成三回血陣容，回復能力非常強，宋江既能給褒姒武器牌讓她發動烽火打輸出，也能給扁鵲紅色牌發動回春和療傷，沒用的牌也可以給褒姒治療回血。

【武將選擇：宋江】

定位：輔助

血量：4

優先順序:S

選擇理由：血量充足，既能給隊友想要的牌，自身還能回血，續航能力強，有宋江的話一定要先選。

玩法推薦：當輪到宋江的出牌時，玩家可以選擇把自己手上的牌發給場上任意一個武將,比如諸葛亮（給錦囊牌）、關羽（給【殺】）、楊延昭（給【殺】）、時遷（給梅花色的牌）、扁鵲（給紅色花色的牌）、褒姒（給武器牌），如果宋江掉血的話，可以選擇當給牌的數量大於等於兩張時回復一滴血,每回合最多回復一滴血。

【武將選擇：褒姒】

定位：輸出

血量：3

優先順序:A

選擇理由：褒姒每回合可以選擇兩張手牌，回復自己和一個男性角色各1滴血，宋江和扁鵲都是男性英雄。有武器牌在手時AOE能力強，可以先群傷再加血，在這個陣容里擔任主要的輸出位置。

【推薦套路組合2：劉邦+虞姬+曹操】

劉邦主C，可以先釋放群體技能如烽火萬箭讓虞姬掉血給劉邦兩張牌，如果虞姬有【葯】一定要自己用，錦囊牌可以選擇給自己或者給曹操，其他的讓劉邦換牌，曹操打控制，在有【虎符】的條件下可以轉輸出。

【武將選擇：劉邦】

定位：輸出

血量：4

優先順序：A+

選擇理由：劉邦血量高，對局中可以把不用的牌換成其他牌，靈活性高，換牌好的話往往有奇效，獨立性強，可配合所有武將。

玩法推薦：當輪到劉邦的回合時，先出主動錦囊牌和殺再選出自己不需要的牌，然後使用技能更換牌堆中同等數目的牌，血量滿的時候可以選擇把其他牌都換了，掉血了可以留【閃】。自己或者隊友血量不健康的時候留【葯】，場上有虞姬的時候多留葯。

【武將選擇：曹操】

定位：控制

血量：4

優先順序：A+

選擇理由：血量高，控制里強，容錯率高，只要有一張錦囊牌，就使用技能選擇不用的牌重複這張牌，選擇多樣化，可以左右戰局。

玩法推薦：當輪到曹操出牌時，可以選擇任意一張牌，當作本回合內前一張使用的牌來打，技能優先選擇觸發【無中生有】和【畫地為牢】，【無中生有】可以在打完這張牌後再次觸發該效果，共計獲得四張牌，【畫地為牢】是很好的控制技能，如果能牢住對面兩個武將的話會給己方帶來巨大優勢，當血量過低時也可以利用奸雄使用兩個【葯】。

【推薦套路組合：任恆之+楊延昭+關羽】

高傷害高爆發陣容，優先進攻敵方脆皮武將，任恆之或者楊延昭出【殺】成功後，由關羽進行補刀，甚至可以秒殺脆皮，優先攻擊3血量低防禦武將如褒姒、荊軻、時遷。如果沒有楊延昭和關羽，可以用項羽、林衝來代替，當任恆之技能選擇友方時，隊友應該出自己最低點數的牌來打配合。

【推薦武將：任桓之】

定位：輸出

血量：4

優先順序：A+

選擇理由：任恆之兼備血量高和輸出高的優點，配合隊友可以快速壓低敵方的血量。

玩法推薦：當輪到任恆之的出牌時，玩家可以選擇一張牌和場上的任一玩家進行比牌的大小，勝利後本玩家可以不限距離的使用兩張【殺】，因為每張【殺】可以選擇兩個目標，所以優先攻擊敵方血量低的英雄，如果敵方有李煜的話，應該先判斷他是否有【閃】，如果李煜有的話，不能對他出【殺】。當指定的武將出的手牌點數大於或等於玩家出的牌時玩家該回合不能出【殺】，所以為了避免對面有【K】導致不能出【殺】的情況，玩家手裡有殺第一回合必須選擇和隊友比點數。如果玩家手中沒有【殺】，我們可以利用被任桓之技能指定的玩家必須出牌這個規則，選擇拿一張沒用的牌和敵方武將比點數，消耗他的手牌。

今天小編推薦的三個套路組合大家學會了嗎~希望對各位小夥伴們有所幫助哦~我們下期再見。
