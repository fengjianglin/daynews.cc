這些遊戲和最初的《Rogue》不盡相同，是殊途同歸還是獨闢蹊徑？

天天要聞
2019-12-14

近幾年《以撒的結合》、《挺進地牢》、《死亡細胞》等等優秀作品可謂路人皆知，Roguelike這個曾經小眾的標籤也越來越頻繁的出現在不同的遊戲里。然而在小眾變為主流的背後，這些遊戲似乎和最初的《Rogue》不盡相同，是殊途同歸還是獨闢蹊徑？關於什麼是Roguelike的爭論從未停止，那麼今天就讓我們借幾款知名遊戲，走進這一經典類型的前世今生。

故事的開頭要從1980年講起，這一年《Rogue》橫空出世，高死亡懲罰、隨機地圖、類似RPG玩法的冒險，使得玩家每次開始遊戲，都會得到全新的體驗。

我們可以看出，始祖遊戲其實已經擁有一套相當完備的玩法機制。但在當時想要引爆遊戲圈，只有這些還不夠。據開發者之一的邁克·托伊（Michael Toy）在採訪中介紹，本作之所以能夠如此成功，最終開宗立派，很大一部分功勞是因為它被內置在了1984年的BSD Unix發行版操作系統里，就像Windows內置的《掃雷》。

或許誰也不會想到，一款系統自帶遊戲重新定義了一種遊戲類型，並長盛不衰地發展到了今天。

《Rogue》的成功催生出了《NetHack》、《Angband》、《Moria》等等一脈相承的後續作品，其相關論壇社區也日益熱絡起來。時間回到1993年，這時類似的作品數量已是非常繁多，並且玩家群體也在不斷增加。

在Usenet論壇的news.groups板塊下，越來越多帖子表示他們需要一個獨立的遊戲板塊。那麼這個新興的遊戲類型到底應該怎麼歸類呢？是「地牢探險」？或是「RPG」？好像都不太準確。再三討論之後，玩家們決定為它取一個新名字。

當然我們都知道，經過社區投票後，「Roguelike」這個稱呼最終奪得桂冠。但進而又引發出一個大難題，應該如何去介紹這個類型的遊戲？到底什麼遊戲算是Roguelike？讓我們接著往下看看。

時光不會停轉，雖然大家還是無法準確形容出Roguelike的特質，但不可否認的是作為新興遊戲類型，它的發展從未止步。2006年《矮人要塞（Dwarf Fortress）》的發售無疑是Roguelike一次重要的傳承和發展，它依然包含《Rogue》隨機生成世界、 ASCII 字元畫面、高死亡懲罰的DNA，但三種不同遊戲模式，建造堡壘和探索世界相結合的玩法以及要塞的模擬經營要素都為這款遊戲披上了新的華麗外衣。

（《矮人要塞》計劃登陸Steam平台，包含新版本和經典的字元版本）

值得一提的是，自2003年立項研發開始，截止目前本作已持續開發了十六個年頭，並秉承著始終免費的態度，還曾被紐約現代藝術博物館收錄，獲得高度評價。歲月侵蝕了聯合製作人年輕的容顏，卻讓遊戲內容愈加豐富。

拋開榮譽加身的外殼，十多年的時光足夠玩家與《矮人要塞》的感情變得特殊。每次遊玩都是一部史詩的開始，每個矮人要塞里都有隻屬於玩家自己的故事。

「Dfstories（http://dfstories.com/）」是由玩家建立起的一個同人網站，許多該系列忠實的擁護者都會結合遊戲中的感悟，寫下屬於自己的小劇場。這裡就像一個超級終端，彙集了太多太多獨一無二的經歷，有為了反抗邪惡的統治，步步為營建造重鎚王座，抹殺邪惡吸血鬼的智者；有決策失誤，導致大規模惡魔入侵，最終城破人亡的愚蠢領主。

一部本身沒什麼劇情的遊戲能發展出這樣一個宏大豐滿的系統，不得不說是一件神奇而偉大的事，也可能正是因為原本的字元畫面的留白，才使得最終所有的一切都變成了玩家自己的傳奇。

兩年後，2008年Roguelike開發大會提出了「柏林準則（Berlin Interpretation）」，但強調包含或缺少部分元素都不能直接判斷該遊戲是否屬於Roguelike遊戲。

柏林準則：主要元素

隨機地圖和道具、高死亡懲罰、回合制、地圖基於網格設計、所有功能和戰鬥基於同一界面、系統複雜性、資源管理、對抗所有怪物、高探索性

柏林準則：次要元素

單人遊戲、怪物同樣擁有技能屬性等要素、注重戰術決策、ASCII字元顯示畫面、地下城不可相同、屬性數字化

或許當時的開發者們煞費苦心去定義Roguelike遊戲，只是為了讓玩家更好地理解該類型，並激發更多的創作熱情。但如此詳盡的劃分一定會帶來爭議，有的玩家認為準則非常酷，該類型就應該符合這些硬核特質；而另一部分玩家則認為柏林準則中有很多條款明顯落後於時代，讓製作者和玩家繼續為字元畫面、網格化設計買單，明顯不合乎歷史邏輯，所以想要「推倒柏林牆」的聲音也不絕於耳。

在各路爭論里，一個詞應運而生——「Roguelite」，原教旨主義者用它去形容那些不夠正統的Roguelike遊戲。Lite大概可以理解為輕量級，它沒有那麼嚴重的死亡懲罰，往往可以永久升級或解鎖技能，不會有真正意義上的刪檔重來；雖然依舊是可以無限重玩，但有一個明確結局；融合各種要素，任何你能想到的玩法同Roguelike相結合都會誕生出全新的遊戲模式。

《盜賊遺產（Rogue Legacy）》自2013年發售之初就自稱Roguelite遊戲，但很明顯，玩家並沒有因為輕量級而輕視它，該作一度成為Steam銷量榜首。

自那以後Roguelite好像就進入了井噴時代，無論是製作商還是玩家，全都心甘情願為之買賬。比如融合了彈幕射擊的《挺進地牢》、融合了卡牌構築的《殺戮尖塔》、融合了類銀河戰士-惡魔城的《死亡細胞》等等遊戲，都取得了不錯的銷量成績，原本硬核的遊戲模式，就這樣走入了平常百姓家。

Roguelike到底是什麼？Roguelite是它新時代的表現形式還是另一個分支？這些問題我可能無法給出如「柏林準則」一般詳細的答案，但喜歡lite的玩家或許某一天會想去看看《Rogue》到底是什麼樣子？喜歡like的玩家去試試《死亡細胞》也未嘗不可。這個類型魅力足夠我們去探索和發現，畫地為牢的意思不是停滯不前，而是「地牢以痛吻我，我卻報之以歌」。共勉！
