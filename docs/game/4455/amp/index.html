爐石傳說：競技場牌池更換，黑石山歸來，這17張看到一定要拿

天天要聞
2 週 ago

摘要

今日爐石傳說迎來了16.6更新補丁，除了預裝了備受關注的新版本&quot;巨龍崛起&quot;以及酒館戰棋的各種更新之外，爐石傳說還有一個模式，就是競技模式（已經被隱藏在其他模式中容易被人忽略的右邊）。

今日爐石傳說迎來了16.6更新補丁，除了預裝了備受關注的新版本”巨龍崛起”以及酒館戰棋的各種更新之外，爐石傳說還有一個模式，就是競技模式（已經被隱藏在其他模式中容易被人忽略的右邊）。

不過競技模式終於也迎來了卡池輪換，從12月6日開始，構成競技場牌池的6個系列將替換為：經典/基礎、黑石山、冠軍的試煉、卡拉贊之夜、冰封王座、拉斯塔哈，新版本上線之後，也就是12月11日開始還將加入巨龍崛起版本。

輪換以後的競技場卡池與之前相差很大，除了經典/基礎以及拉斯塔哈之外，其他四個拓展包的卡池均有輪換。下面，筆者就為大家盤點一下這四個拓展包中的部分強力競技場卡牌，選到它們，或許12勝就離你不遠了。

黑石山的火焰

盜賊

強力的群體AOE，黑鐵潛藏者配合盜賊的匕首很容易完成5費返場。

冠軍的試煉

中立

激勵機制的三張強力單卡，都具有單刷對手的潛力，解不掉這三張牌的後果是災難性的。

海怪作為中立的返場生物費用雖然高但是效果非常強大。

騎士

同樣是一張很容易單刷對手的牌，帶有激勵機制的牌一旦解不掉就會非常難受。

薩滿

薩滿作為在這個拓展包中崛起的職業，其單卡質量都非常高，與圖騰有關的隨從不僅在天梯的中速薩中表現優異，在競技場中的地位也同樣很高。

卡拉贊之夜

法師

火門作為一張技能返場又能打臉的法術牌在競技場中的地位非常高，幾乎是必抓的卡位。

薩滿

大漩渦傳送門是一張非常優質的aoe，同樣能夠完成返場加站場，對於薩滿缺乏的補刀能力也非常重要。

冰封王座的騎士

中立

白嫖buff怪，這張牌在競技場相當於貼聖盾，buff後交換場面相當賺。

之前版本最超模的三費競技場隨從，不過用這張牌的時候要謹記凍結的是一個回合的攻擊頻率（也就是這個回合不動，如果凍結，下個回合還能動）。

非常好的質量牌，在新版本上線之後超模龍牌更多，這張牌的發揮空間會更大。

德魯伊

德魯伊的終極感染即使放在新版本，這也是當仁不讓的T0卡牌。

術士

強力的AOE加站場，這張牌對於術士來說非常好用。

雖然說如今的爐石傳說中，競技模式的地位已經今非昔比，但是競技場仍然是許多萌新入坑的跳板，當然也有很多仍然熱愛競技場的玩家，相信大家看完筆者的這篇文章對於新版本的競技場也已經瞭然於心，十二勝在向你們招手。
