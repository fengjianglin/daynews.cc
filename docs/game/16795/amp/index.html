DNF：體驗服重拳出擊限制交易，黑商不幹了，再度叫囂遊戲要黃

天天要聞
2 天 ago

前陣子小編看拍賣行的時候，一度奇怪紅腿愛克托這個卡片怎麼這麼貴，坯子卡就要一千多萬，成品卡要兩千多萬，比二十五屬強附魔貴了一大截，堪比防具附魔了。後來想一想應該是有需求而且被黑商炒作了，畢竟之前左槽寶珠除了安徒恩就是春節，沒有春節禮包，紅腿愛克托就是最好的附魔，七十三攻更是面板黨的最愛。商人趁機炒作，沒年套的玩家爭相買單。討厭黑商，你也沒辦法，但是策劃有辦法。

體驗服這次改版主要針對各類附魔卡片，改版後，所有的附魔卡片僅限交易一次，也就是說從某個玩家手裡爆出來或者翻牌獲得後，僅有一次上架拍賣行的機會，黑商囤了就只能自己用，賬號綁定你休想賣出去。這次改版後，各類頂級附魔價格肯定要出現斷崖式下跌，然後各類白卡也要出現崩盤，這兩類附魔，正好是黑商最喜歡囤的。

頂級附魔是打造需求在，白卡則是戰令任務需要，一張貓妖卡被炒到五萬一張，你說它貴不貴？現在體驗服只是剛剛更新，跨三B的紅腿坯子卡已經猛跌四百萬，從一千萬變成了六百萬，等到這次改版實裝到正式服，紅腿坯子卡價格應該在三百萬左右，和一般的頂級附魔坯子卡同一個價格。玩家受益，黑商肯定不高興啦。

現在又出現了一些陰陽怪氣的人，說沒了黑商遊戲就黃了，黑商是大功臣啊！說商人囤貨能掙錢，也要承擔風險啊，對等的啊。這種謬論簡直不值一駁，不如告訴這些陰陽怪氣的人說：你去搶劫也要冒被抓的風險。什麼奇葩理論，承擔風險不是黑商擾亂遊戲內經濟秩序的理由，玩家深受其害，難道還是假的？說的好像沒有了這些囤積居奇的商人，地球就不轉了一樣。

而那些常年搬磚打團，攢了很多卡片的玩家，也要儘快把積攢的卡片處理掉了，現在白卡正是最貴的時候，傳說卡已經猛跌了，再不賣就要全都砸手裡，到時候哭都來不及。

這次改版以後，黑商能囤的東西大概也只有無色和矛盾了，但是這兩樣東西價格浮動從來都不大，產出又多，黑商願意屯就讓他們囤去吧，大不了自己搬磚自己用。
