DOTA2：新加坡世界賽半程數據分析 影魔真實弟中弟 聖堂auto win

天天要聞
2019-12-21

新加坡世界賽也過去了一半的賽程了，雖然中國戰隊表現不佳，但是其他參賽的隊伍還是給我們奉獻了很多精彩的比賽，這裡也對目前比賽的各項數據進行分析，看看職業戰隊是如何理解這個版本的。本文素材來源競技寶，感謝jjbone的數據支持。

勝率最高的英雄：TA（10場以上）

目前比賽中TA登場16次，有著恐怖的87.5%的勝率，這個勝率和上個版本的夜魔勝率基本持平。像EG，Navi，TNC這些隊伍都十分喜歡用TA這個英雄，TA在這個快節奏的版本依舊十分吃香，看了這麼多局TA的比賽，基本就是前期暗滅跳刀，中期補個BKB，沒到後期30分鐘再補一個大炮帶個盾就基本可以把對面抬走了。

而且TA高效的Farm效率很容易使自己升到25級，25級之後的TA有了天賦眩暈，切後排的能力更強。TA應該已經連續火了好多個版本，不知道下次大更新，冰蛙會不會對TA下刀。

勝率最低的英雄：SF（10場以上）

說過了勝率最高的英雄，我們再來聊聊勝率最低的，目前出場10次以上的英雄冢，影魔的勝率最低，只有可憐的27.8%。這個英雄在小組賽第二天B組的比賽中基本上局局都有，尤其是LGD的比賽，LGD和紅星，液體的比賽中每局比賽都有SF出場，而且不管是誰玩影魔，最後都輸了…..

儘管現在都是物理影魔輸出爆炸，但是影魔身板脆沒控制的問題還是正在比賽中顯得淋漓盡致，觸發有像VS這樣的輔助保護以及靠譜的前排支撐，影魔在後期BKB時間變短之後很難有足夠的輸出空間。（後面和VG打TNC，祈求Rotk千萬別給Ori玩影魔了…..）

BP率最高的英雄：大樹

說過了勝率再說說BP率，目前比賽中職業隊最尊重的一個英雄就是大樹。在所有進行的比賽中大樹的BP率達到了100%也就是非Ban即選。

所以說現在趁著冰蛙沒削，趕快練一手噁心人的大樹上分呀！再說說BP率前五的英雄，末日使者的BP率也達到了98.4%，目前Doom這個英雄就算前期很慘也可以吃兵慢慢補回來，而且現在焦土的傷害是真的有點高。

天輝勝率：62.5% 夜魘勝率：37.5%

最後說一下在地圖改動之後玩家比較擔心的天輝夜魘的勝率，目前天輝的勝率高達62.5%，而夜魘的勝率只有37.5%，基本上就是一個六四開的局面，看來目前天輝下路鍋點的消失對於天輝的”公平性”來說影響並不算太大。
