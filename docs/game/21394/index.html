鬥魚黃金賽數據一覽，4AM排名第10擊殺分恐怖，SMG場均吃毒僅3秒

2019-12-20 113條評論 3,683次閱讀 306人點贊

鬥魚黃金大獎賽小組賽還剩下2個比賽日了，由於本次黃金大獎賽採用了類似「瑞士輪」的賽制，因此最後2天的比賽日將由C、D兩組積分前八以及C、D組兩組積分後八分別進行比賽。

按照賽事規則總積分前16的戰隊將成功進入總決賽，並且積分不清零，這意味著能夠在小組賽拿到越多的分數那麼對於最後獲勝則越有利。從目前來看Ark以總積分101分暫列第1，並且從前幾場小組賽來看，Ark一直都是排名第1第2，也就是說Ark一直在勝者組中進行對決，能夠在一眾強隊中持續保持狀態實屬難得。

此外與Ark一樣保持了極佳競技狀態的還有EDG，EDG可以說是東邊不亮西邊亮，完美詮釋了什麼叫能量守恆定律，英雄聯盟那邊哭唧唧絕地求生這邊笑嘻嘻，僅僅落後Ark三分的EDG也是本次鬥魚黃金大獎賽冠軍的爭奪者。

相比Ark、EDG一直在打勝者組，4AM可以說是起起伏伏，第一天比賽打進前八後一天比賽就來到了第九，以80分的總積分暫列第10.不過玩家們也大可不必擔心4AM的狀態，至少從對槍來看4AM正麵糰戰的能力還是屬於無敵狀態，從已經打完的12場比賽來看4AM拿到了57個擊殺，這擊殺數量排在所有戰隊第4的位置。尤其是排名第九的C組比賽一共37分有30分是擊殺得來的，如果圈型對4AM友好點那麼4AM將不會止於這個成績。

至於在C組跟4AM一直有著刮擦的SMG，12場比賽結束後以60分的成績暫列第20名，不過由於SMG進入了敗者組因此還是有希望進入前16名的。雖然說SMG排名靠後但是有一項數據卻是無人能敵，那就是場均吃毒時間，僅3秒的場均吃毒可以說SMG要不就是能提前進圈要麼就是圈運好，不過如果是屬於後者的話那麼SMG才打出這點積分實屬不應該了。

鬥魚黃金大獎賽還在繼續，各位玩家們不如繼續鎖定鬥魚直播，觀看精彩的比賽見證16支總決賽戰隊的誕生。
