《光·遇》獲評年度遊戲，《三國志·戰略版》入圍“經典新生”榜

2019-12-04 0條評論 17次閱讀 0人點贊

12月3日，App Store公布了2019「遊戲年度趨勢：經典新生」榜，以及2019年度最佳遊戲和應用榜單。本次榜單由五大洲App Store編輯團隊共同評選，代表了最新的流行趨勢。

好遊戲不會因為時間的流逝而被遺忘，家喻戶曉的形象、獨特有趣的玩法具有超越年代的吸引力。因此，今年App Store遊戲年度趨勢的關鍵字是「經典新生」。

本次上榜的遊戲共有7款，均為過去數十年間經典遊戲的移植手游版本，包括《跑跑卡丁車官方競速版》、《軒轅劍龍舞雲山》、《最終幻想：勇氣啟示錄》、《輻射：避難所Online》、《三國志・戰略版》、《夢幻模擬戰》和《刺客信條：燎原》。

無論玩家喜歡哪種類型，都能在這些「新經典」遊戲中找到屬於自己的那一款：潛行玩法愛好者可以在《刺客信條：燎原》中躡足潛蹤；策略遊戲愛好者可以在《三國志·戰略版》中逐鹿中原，稱霸一方；戰棋遊戲的粉絲將在《夢幻模擬戰》中重新體驗原作一代到五代的經典劇情。

喜歡角色扮演遊戲的玩家可以在《最終幻想：勇氣啟示錄》中踏上拯救世界之路，《輻射》系列的老玩家們則可以在《輻射：避難所 Online》中打造安全舒適的庇護所，開啟廢土之上的新冒險。

值得注意的是由光榮特庫摩正版授權，靈犀互娛研發的《三國志・戰略版》。據了解，該作是蘋果設計該獎項至今，首次上榜的由國內團隊研發的策略遊戲。《三國志・戰略版》上線以來便在中國市場大獲成功，據Sensor Tower數據，10月期間，《三國志·戰略版》在國區App Store預估收入超過5200萬美元。

2006年，《跑跑卡丁車》被代理商世紀天成引進中國，迅速成為當年網吧最火爆的遊戲之一。在其手游移植版《跑跑卡丁車官方競速版》中，玩家不僅能聽到熟悉的音樂、駕駛熟悉的賽車，還能體驗到在移動平台上更為精緻的畫面、操作、以及無限氮氣等新玩法。

此外，進入App Store 2019年度遊戲榜單的遊戲共有五款，分別是年度iPhone遊戲《光·遇》、年度iPad遊戲《Hyper Light Drifter》、年度Mac遊戲《GRIS》、年度Apple TV遊戲《神奇小子：龍之陷阱（Wonder Boy: The Dragon』s Trap）》、以及年度Apple Arcade遊戲《再見狂野之心（Sayonara Wild Hearts）》。

在《光·遇》中，玩家要在廣袤的天地間自由翱翔，尋找先祖之靈，並將他們送回天空成為星星。根據蘋果評價，在過去的一年中，雖然湧現出許多優秀的遊戲，但難以找出比《光·遇》更能代表「遊戲美學」這個詞的作品。

恭喜以上遊戲。
