組建人情戰隊？Tabe、Mata回歸RNG，粉絲：還想不想出成績了？

天天要聞
16 小時 ago

近日，RNG俱樂部接連放出大消息，先是Uzi、小虎、小明三人續約，AJ和Zrr離隊，緊接著就是Tabe回歸，擔任數據分析師以及Mata回歸，擔任主教練。前面的消息都還挺正常的，畢竟AJ和Zrr上場機會渺茫，離開也可以理解。但後兩條消息就過於爆炸了，引來了不小爭議。

看到了消息後小編和粉絲的想法是一樣的：RNG又在搞情懷，玩運營，這兩人絕對不是出成績的選擇，讓我們逐條分析。

一、Tabe也敢要？

Tabe是老皇族SHR的一名輔助選手，因為在解說時向其他隊伍泄露戰術，鬧出過非常大的節奏，最終被迫離開大陸，回到港台發展，人送外號”電競汪精衛”。而他本人實力也就那樣，吹來吹去不過一手輔助安妮，S3皇族能拿到亞軍，跟他也沒太大關係。回到港台後，Tabe也沒能闖出什麼名堂，17年轉型教練後率領HKA征戰，戰績相當慘淡，這次回來擔任數據分析師，讓人不得不懷疑其專業能力。更重要的是，他到底有沒有改掉大嘴巴的毛病，我們還不得而知。把這人請回來，RNG真就不怕自己的戰術又泄露一次？

二、Mata語言不通

Mata是RNG於S6引入的一名韓援輔助，能力出眾，對RNG的轉型幫助不小，除了”風語者婕拉”事件也沒什麼黑點，回歸LCK後與RNG隊員的關係也一直不錯，經常保持著聯繫。看上去是比Tabe更靠譜的選擇。

但別忘了一點，Mata的中文並不算太好，在RNG曾經的比賽中，雖然他能夠大體上指揮戰局，但賽後無法獨立完成採訪，需要藉助翻譯。當教練需要的與選手的溝通，比比賽中選手間那點交流多多了，憑Mata的中文水平是肯定不夠的。何況他已經回到LCK兩三年了，中文還記得幾句呢？

最重要的是，RNG的新配置依然是全華班，沒有像Rookie和Doinb那樣能擔任翻譯的選手，所以這讓教練和選手間的交流又加上了一層隔閡。所以總的來說，請回Mata並不算一個太好的選擇。

三、RNG擅長玩運營這套

從S8之後，RNG就坐實了運營隊的稱號，成績不上不下，其他方面倒是搞得紅紅火火，這次S9年度頒獎，有關運營的獎項，包括什麼”最佳節目”，”最佳品牌營銷”等，全都歸屬到了RNG頭上。但電子競技成績說話，粉絲只看你的成績，並不關心這些花邊，拿這些獎反倒顯得尷尬，所謂”幹啥啥不行，營銷第一名”就代表了大部分粉絲的態度。

有無數前例在先，這次請回兩名老隊員，被很多粉絲懷疑成營銷也是難免的，這就和全華班一樣，是一張人情牌，目的是通過消費粉絲情懷來固粉。小編並不想以最大的惡意來揣測RNG的意圖，但如果真是如此，那也怪不得粉絲開罵了。
