《Star Trek：Bridge Crew》將登陸Oculus Quest

2019-12-19 1,283條評論 4,574次閱讀 1599人點贊

《Star Trek：Bridge Crew》是一款遊戲節奏非常緊張的多人合作探索遊戲，該遊戲最初於2017年發佈於Oculus Rift，HTC Vive和PS VR平台，並於同年登陸美國洛杉磯的IMAX VR 體驗中心。遊戲支持4名玩家組隊進行，去探索一個名為『The Trench』未知領域的任務，為瓦肯人尋找一個合適的新世界。

玩家可以在艦長，舵手，戰術官和輪機長中挑選一個角色扮演，每個角色都有一套特定的任務，包括從駕駛船隻到操作武器系統的所有事情。為了成功躲避Klingon艦隊的攻擊，團隊成員之間必須保持密切合作，在深空中並肩戰鬥。這款遊戲的特點在於一個完全開放的，動態的故事情節，玩家可以獨自享受或者與他人合作，也可以選擇遊戲時長更多的「Ongoing Missions」模式。

《Star Trek: Bridge Crew》現已在Oculus Quest平台開售，售價29.99美元（約合人民幣210元），價格比最初的Oculus Rift、HTC Vive和PS VR版本高出5美元。在去年7月,Oculus推出了一個新DLC包，玩家將有機會駕駛U.S.S. Enterprise-D星艦在整個太空中自由漫遊，以及兩種全新的任務類型：「巡邏」和「抵抗」。

在「巡邏」模式中，玩家將體驗隨機遭遇戰及任務帶來的無盡冒險挑戰。而在「抵抗」模式中，玩家將面對該系列粉絲最愛的反派——不可阻擋的博格人。玩家需要一邊躲避博格方塊的追擊一邊趕在最終決戰之前完成一系列的任務。

在本次更新中，遊戲還加入了一個新的玩家角色：Operations，負責管理船員和船舶；以及兩個新敵人：Romulans和Borg。

目前還不清楚Quest版本是否會支持DLC《The Next Generation》，也不清楚能否與PC VR或主機平台交叉運行。儘管如此，《Star Trek: Bridge Crew》代表著VR領域最受歡迎的合作體驗，對於追求AAA級多人VR體驗的Quest用戶來說是必買的一款VR遊戲。
