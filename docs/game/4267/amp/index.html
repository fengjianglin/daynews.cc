韋神吐槽導播：我沖樓的那波沒給我第一視角，你們不知道我多猛

1 週 ago

摘要

最近正在進行的微博杯，出現了很多讓觀眾吐槽的幺蛾子，說起來也是一言難盡，不過最大的槽點還是這次比賽的導播，導播幾乎承擔了微博杯90%的槽點，從觀眾們在論壇屠版嘲諷，到業內大佬海濤直言：導播下班去財務室領工資等等，這次的導播確實在業務能力上很差勁。

最近正在進行的微博杯，出現了很多讓觀眾吐槽的幺蛾子，說起來也是一言難盡，不過最大的槽點還是這次比賽的導播，導播幾乎承擔了微博杯90%的槽點，從觀眾們在論壇屠版嘲諷，到業內大佬海濤直言：導播下班去財務室領工資等等，這次的導播確實在業務能力上很差勁。在12月7日晚的直播當中，韋神也在直播中吐槽了這次的導播。

以下是韋神在直播中的原話：

感覺這次微博杯的導播很業餘啊，我那麼多精彩的操作都沒有被錄下來，很氣啊我，你看我滅Ark沒鏡頭，對不對，然後我單人沖樓也沒有鏡頭，然後我們吃雞那把，我反殺FTG，也沒有鏡頭，我很很寂寞，真的，沖樓那波沒給我第一視角，你們不知道我多猛，算了，說多了也沒意思

前兩天4AM表現不如人意，但是第三天的4AM豪取了68分，確實打出了非常多亮眼的操作，不過針對韋神所說的這幾個鏡頭，劉大大專門去錄像當中找了一下：

滅Ark導播殘局收尾

滅Ark的時候，沒有給到4AM的第一視角，這兩支戰隊前期的交戰信息，不過在快要打完的時候，在小鹿和by進行1V1單挑的時候，鏡頭給到了by，然後被小鹿淘汰，導播過來收尾了一下，至於4AM是怎麼淘汰Ark的，觀眾確實沒看到鏡頭。

航拍反殺FTG

決賽圈FTG兩人趴在同一個地方，導播給了韋神的第一視角，韋神擊殺了一個人之後以為那個地方就沒人了，但是那個地方還有一個，在韋神不備的時候，第二人偷襲，結果被韋神絲血反殺，這一波導播在航拍，沒能捕捉到。

第三視角單人沖樓

這一局可以說是韋神個人最精彩的一點了，單人沖了newhappy的樓，第一個人用手雷破點，導播沒有捕捉到，第二個人用三倍秒殺，此時的導播給了韋神的第三視角，在韋神擊倒這個人之後才給了第一視角，精彩的頭皮槍法，觀眾們沒有看到。

從這三個鏡頭看下來，觀眾們以及韋神所說的感覺不公平是不存在的。

因為這三個鏡頭導播其實都是給了視角的，不過他漏掉了每一個鏡頭的最關鍵也是最精彩的畫面，這一點還有Ark餘人的栓狙，在這次微博杯中，餘人栓狙放倒了不下二十人，但是觀眾一個都沒看到，所以說這次的導播不是不公平，他並沒有針對任何一個選手和戰隊，只是單純地業務水平不行。

微博杯的導播可能不太了解這個遊戲，他不知道什麼時候打架，不知道該給誰鏡頭，不知道怎樣算是精彩，導播特別喜歡給航拍視角，或許認為吃雞跟LOL一樣，想讓觀眾看全局，但FPS跟MOBA完全不是一個類型的存在，所以觀眾們也不要過多聯想，沒那麼多平台陰謀論，只是導播確實”不懂這個遊戲“。

如果我是導播，我會這麼跟選手說：兄嘚，不是我不給你鏡頭，我是不知道什麼時候給你鏡頭，該怎麼給你鏡頭，對不起！

有一說一，微博杯其他地方做得都還不錯，很希望微博杯決賽能請幾個有經驗的導播，給大家帶來更好的觀賽體驗吧！
