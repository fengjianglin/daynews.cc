王者榮耀：鎧為避免被控，不要出極寒風暴，更別出無盡，出他無敵

2019-12-16 619條評論 2,217次閱讀 332人點贊

哈嘍，大家好！我是九零小哥，私信「地獄火」，即可免費領取皮膚。王者榮耀是我們非常喜歡的一款遊戲，在王者榮耀中有著不少強勢的英雄，而鎧則是一位邊路上相當無解的存在。鎧的技能讓自己有著強大的續航能力，並且有著非常高的輸出，特別是在裝備成型之後，鎧將會成為王者榮耀中最為強大的英雄，對後排的威懾力非常高。

不過鎧的傷害雖然非常高，出肉也能夠打出不俗的輸出，然而這個英雄非常怕控制，容易被對方的英雄控制。那麼鎧為避免控制，鎧這個英雄應當出什麼樣的裝備？接下來就讓筆者給各位玩家說說筆者個人的理解。

首先我們不要出極寒風暴，這件裝備是很多玩家玩鎧的時候第一件出的裝備。極寒風暴的物抗以及冷卻縮減對於鎧來說非常重要，然而做出這件裝備之後，鎧本身的輸出並不高，對於對方的陣容的衝擊力度並不高，自己沖入到對方的後排也沒有什麼大的作用。因此在均勢和順風情況下，不要急著出冰心。

其次有部分追求輸出的玩家喜歡在第一件出無盡戰刃，無盡戰刃提高的高暴擊以及攻擊力能夠讓鎧成為脆皮殺神，只要能夠突到對方的脆皮身邊，那麼基本上是能夠百分百的將對方脆皮秒殺。然而這樣的鎧更加的害怕控制，通常遇到對面的控制，會被直接秒掉，相當的煩人。

而筆者個人認為鎧最好的選擇應當是暴烈之甲。這件裝備能夠增加自己的物理防禦和生命值。再來看看這件裝備的被動，能夠提升自己的移動速度、輸出能力。這也就意味著如此裝備下的鎧能夠有超高的抗性和輸出，根本不怕對方的控制。

那麼大家是怎麼看王者榮耀中的鎧了？歡迎各位玩家的點評，在文章下方說出自己的裝備，歡迎大家的點評，說出自己的看法或者想法，本期的文章分享到此結束。
