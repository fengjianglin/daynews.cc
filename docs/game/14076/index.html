兮夜發博不在和We續約！新戰隊也已找到，是LPL的一支“全華班”

2019-12-11 277條評論 1,361次閱讀 80人點贊

在本次的LPL轉會市場上，有兩個選手的去向備受關注。一個是JKL，不出意外的話，大概率會加盟SN戰隊，已經簽完字，等聯盟審核完合同就能官宣了。而另一個選手就是We中單兮夜了，在成為自由人之後一直沒有他的消息。不過在今天，兮夜的最終去向已經確定了。

在10號晚9點，兮夜正式發博宣布不再和We續約，因為自己打算接受新的挑戰。隨後We俱樂部也發博回應了這一消息，並向兮夜致敬，畢竟兮夜在We打了7年比賽，已經算是元老級選手，而隨著兮夜離去，這一代的We算是翻開了新的一章。

那麼兮夜在離開WE後，會去哪支戰隊呢？小米可以告訴小夥伴們，就是dmo戰隊。因為We的上單957在直播的時候明確表示，兮夜就是去了DMO，到時候官宣了大家就知道了。再加上兮夜最近一直和DMO打野小鵬瘋狂雙排，那麼去DMO基本是實錘了。

而這支DMO戰隊目前還是挺不錯的，雖然是一支新軍，但上賽季還是打的可圈可點，如果有兮夜這樣一名老將加入，想必他們能夠打出更好的表現。值得一提的是，DMO還有一個身份，那就是「全華班」，從教練組到選手全是國產的。

而隨著兮夜的去向塵埃落定，在新賽季大家又能看到他和小虎這一對歡喜冤家的交手了，作為當時LPL的中單雙子星，這兩人有著太多相同點，曾一起走過輝煌，也曾一起變撈。就是不知道新賽季兩人遇到會是怎麼樣的場景，是小虎繼續「吃飯睡覺打兮夜」，還是兮夜洗刷恥辱。就讓我們拭目以待。
