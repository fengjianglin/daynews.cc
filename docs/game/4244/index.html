第三屆微博杯：決賽來襲 4AM能否保持小組賽最後一天狀態繼續拿分

2019-12-08 0條評論 14次閱讀 0人點贊

摘要

經過一周的小組賽，絕地求生第三屆微博杯終於迎來了最終的決賽，本次進行第三屆微博杯的16支戰隊均是小組賽的佼佼者，現在讓我們來看看本次第三屆微博杯的決賽隊伍吧。

經過一周的小組賽，絕地求生第三屆微博杯終於迎來了最終的決賽，本次進行第三屆微博杯的16支戰隊均是小組賽的佼佼者，現在讓我們來看看本次第三屆微博杯的決賽隊伍吧。

參賽隊伍：RNG戰隊、IFTY戰隊、SMG戰隊、BA戰隊、TSG戰隊、Weibo戰隊、Lstars戰隊、EDG戰隊、Tianba戰隊、4AM戰隊、AMY戰隊、Ark戰隊、QM戰隊、YTG戰隊、VG戰隊、VC戰隊

比賽積分模式：

比賽獎金：

比賽時間：12月8日到12月10日每晚5點多開賽（官方是說5點，不過正賽基本會晚個十幾二十分鐘）

哪裡觀看：本次第三屆微博杯是鬥魚獨播。

決賽第一天的看點：

1、P城跳點

小組賽SMG戰隊P城讓給了RNG戰隊，自己跑去水城遺迹一塊，不知道決賽SMG戰隊還是否會這樣。

2、4AM戰隊的狀態

4AM作為國內最火的絕地求生戰隊，本次小組賽可以說是驚喜連連，本來以為晉級希望渺茫的4AM戰隊憑藉著最後一天的比賽狂拿68分成功晉級決賽，不知道今天4AM戰隊的狀態如何。

3、三大幻神的表現

雖然說是三大幻神，不過本次百分百參與第三屆微博杯決賽的應該只有小叮噹和小番茄，至於龍太子的話就只能看看4AM戰隊隊內的想法了。（4AMlong將在明年4月份成年那時候臨近PCL夏季賽）

至於其他的胖子也就不多說了，畢竟本次第三屆微博杯決賽的16支戰隊確實都很猛，每一支都是強隊，在比賽還沒開始之前胖子很難說哪只奪冠幾率更大。
