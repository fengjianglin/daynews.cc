12月1號的LPL首屆全明星賽無疑是吸引了所有玩家們的目光，各路明星選手時隔多日再次登場。除了由The Shy和UZI領銜的「至臻IG」對戰「至臻RNG」，最引人注目的那就是老IG對戰老WE的傳奇表演賽。不過賽後玩家們也表示，原本應該激烈的明星賽變成了娛樂大亂斗，表演性質的傳奇對戰卻認真了。

曾經的老IG和老WE向來是話題性十足，在比賽前若風和西卡就因「換人」事件在微博開團，引來各路老選手參加團戰，濃濃的火藥味也讓此次比賽被戲稱為「第二屆農夫山泉杯」。而比賽當天，捲毛的機器人，微笑的薇恩，若風的落地金身，笑笑的空大、以及小蒼、小米的解說加入，讓一眾玩家直呼「青春回來了」

兩隻戰隊在英雄聯盟早年各領風騷，無論國內國外都取得過很多榮譽，代表了中國戰隊的最高水平，貴為傳奇。而兩支戰隊也是多次對決，互有勝負，成為當年的一段佳話，今天小弟就給大家說說曾經的老IG和老WE。

傳奇的開始

IG和WE的恩怨最早可以追溯到2011年。因為SKY的WCG兩連冠，讓WE成為一方電競豪門，在2011年3月成立英雄聯盟分部，而若風和微笑是最早的隊員。

2011年8月，王思聰在微博上宣布進入電競圈，收購了當時國內一線俱樂部CCM戰隊，將其更名為iG（Invictus Gaming），成立英雄聯盟分部，成為國內最早的英雄聯盟戰隊之一。而IG也因為校長的資本介入，成為當時為數不多擁有自己的戰隊基地、幕後工作人員的「奔小康」戰隊。

曾經老IG基地↑↑↑

2011年10月在上海舉辦的英雄聯盟WCG2011挑戰賽上，WE和IG在決賽上迎來首次對決。而這個WCG挑戰賽有點類似於LPL的夏季賽季後賽，IG作為WCG的冠軍作為擂主，其餘8支戰隊需要經過層層選拔最終與IG會師決賽，最後的冠軍獲得S1參賽權。

在BO5的賽制中，IG因為是WCG冠軍先積1分，後連贏兩局以3:0戰勝WE獲得了S1參賽資格，也拉開了WE和IG恩怨的序幕，之後的IG在韓國釜山的S1上止步八強。

WE的制霸

到了2012年，兩支戰隊都引入了新的隊員。WE這邊引入國服rank王：諾言、草莓，IG這邊加入天才少年姿態、小孩以及PDD，雙方均達成玩家們最熟知的完全體，開始了一年的比賽。

2012年10月，韓國NiceGameTV主辦的韓國龍爭虎鬥邀請賽 ，WE和IG代表中國戰隊受邀參加，與另外5支韓國戰隊展開對決。比賽口號喊出：誰是亞洲第一，噱頭十足，也是中國戰隊早年抗韓的比賽了。

而IG和WE分別將各自的韓國對手擊敗，WE更是零封CJ，兩者會師半決賽。實力相當的兩隊打滿5場比賽，手握兩個賽點的WE在第三場被IG再20分鐘領先1萬經濟，無奈投降。最後一局兩隊都使用了中路雙人的奇葩打法，WE憑藉15分鐘rush大龍取得經濟領先，最終獲得勝利。

而之後的決賽WE對戰KTA上演了無恥的掉線時間，WE5人集體掉線，比賽非但沒有暫停，KTA還推掉了中路水晶，重連的WE集體開火，微笑拿下5殺奪得冠軍。

這一場比賽也是WE在2012年的縮影，WE5人超強的實力以及默契的配合讓WE在這一年完成了對冠軍的壟斷，在一年中取得26次冠軍，對35戰隊保持100％勝率，累計獎金超過100萬元！玩家們耳熟能詳的掉線門，IPL5世界冠軍均在這一年。實力與戰績為WE帶來大批粉絲，被稱為宇宙戰隊，「60e」的梗也是從此而來，寓意全世界60億人都是WE的粉絲。

而另一邊的IG同樣保持著國內一線水平，在韓國舉辦的擂台賽上，橫掃韓國群雄，並三殺clg.na，連續五周守擂成功。PDD打爆了所有的韓國上單，在韓國累積了大量的粉絲和人氣，使用各種奇葩中單的zitai也是名聲大起。

LPL百家爭鳴

到了2013年，我們擁有了自己的戰隊——LPL， 國內英雄聯盟的電競水平也不再是IG和WE二分天下，各支戰隊分庭抗禮。黑暗勢力OMG、擁有UZI的皇族、黑馬戰隊PE都實力不俗。

在2013年3月份舉辦的G聯賽全稱是全國電子競技電視聯賽，是遊戲風雲主辦的國內首個電子競技電視聯賽，比賽的舞台場地、燈光、直播設備都達到了一個很高的規模。在決賽上WE與IG大戰5場，雙方互有兩眼操作，草莓奧拉夫神勇開團、姿態死歌怒搶大龍、諾言趙信帶動節奏、PDD螳螂霸氣四殺等等，最終WE贏得比賽勝利。

而這一年兩支戰隊最激烈的比賽，要數技嘉SWL第二季的決賽，也就是農夫山泉杯。在這一季SWL上邀請了多家中韓頂級俱樂部，國內豪強悉數登場，包括S3奪冠的SKT。比賽持續兩個月，多家媒體網站轉播，冠軍獎金高達15萬，在當時影響力不亞於第一屆LPL。

當然決賽上IG和WE又一次相遇，在BO5的賽制上，WE因為冠軍組先積一份，但IG還是拿下3局，以3:1戰勝WE，本場比賽也爆發了許多名場面。在比賽第四場，使用石頭人的PDD一直分帶，而草莓的蘭博一直跟團，導致雙方補刀差距過大，最後草莓的補刀比PDD少了269刀。「草莓被壓300刀」的梗也由此而來。

同樣是在這場PDD的石頭人關鍵傳送繞後，配合笑笑的琴女大招穩定控制，一波團戰帶走了最後的比賽。而賽後同樣精彩，IG上台領獎時，底下WE觀眾仍朝IG選手扔了一瓶農夫山泉，瞬間讓戰鬥從賽場上轉到了賽場外面，PDD和笑笑立即呵斥台下觀眾。

「農夫山泉」事件讓本來名聲就不太好的60E更是受到了玩家們的指責，很多貼吧的吧友在這個時候看清了60E的嘴臉，之後就開始了反擊，徹底崩潰了60E在那個時候的話語權。

雖然兩支戰隊打出了精彩的表現，但無法掩蓋各路戰隊崛起的事實。2013年LPL春季賽雖然IG排名第一，但季後賽只取得第三。夏季賽季後賽也遭到皇族淘汰。S3結束後到2014年，若風、笑笑、PDD、草莓紛紛退役,諾言和捲毛轉回EDG，至此老IG和老WE的傳奇結束。

小弟有話說：周日的全明星過後，老IG與老WE的精彩比賽又增添一筆，幸運的是兩支戰隊的大部分選手還都活躍在圈內，能讓玩家們繼續看到他們精彩的操作。而他們曾經的取得的榮譽也是廣大玩家們青春的回憶。

那麼問題來了：下一屆全明星你想看到哪兩隊傳奇選手對決？
