DNF韓服“逆天”發布會：神話裝備上線，屬性超越SS，1件也成套裝

2019-12-14 31條評論 1,160次閱讀 206人點贊

「地下城與勇士之小狐狸君愛談遊戲電競」第七百三十九期《DNF韓服「逆天」發布會：神話裝備上線，屬性超越SS，1件也成套裝》，韓服本次的發布會，算是把過去一年的積累的更新內容都放出來了。尤其是裝備上限的提升、等級上限的提升、新的團本、新的副本、三次覺醒……幾乎是讓玩家目不暇接了。最重要的是，韓服的三絕在12.18登陸韓服體驗服。2020年1月9日就會正式上線正式服！國服就是在4~5月更新了。春節版本之後上線三覺！老規矩搞快點，直接上本次神話裝備截圖和屬性的截圖的展示：

DNF韓服「逆天」發布會：

本次發布會的水準，和之前的90時代發布會有一拼了。尤其是姜策劃，夏季發布會都「鴿子」掉了，這一次冬季發布會，簡直可以用逆天來形容了。尤其是此次韓服送出了15種史詩防具。95版本就有15種：哈林、泰波爾斯、超界……所以這15種目前還是屬於這兩個類型的吧，分為AB兩套和團本史詩套。

神話裝備上線，屬性超越SS，

當然神話裝備是非常不錯的，是在2020年1.09正式更新的，神話裝備的強大之處就是可以和史詩、神器、神話散搭，而且最重要的是1件也有套裝效果，獲取方式非常的簡單，100級的新地圖就能獲取，比如黑色神殿、痛苦地下城……等副本。神話裝備設計出了35種不同的套裝效果，也就是說這是一套催化劑裝備，可以自成一套也可以和其餘的裝備散搭出套裝效果。

1件也成套裝：

1件就能觸發套裝效果，2~5件都能出特效。不管是史詩裝備，還是神話裝備！都能觸發專屬的特效。而且100級的史詩都是可以通過95史詩升級的，神話也是不可以直接升級的，但是是可以繼承的，95級氪金完全有效。可見本次策劃還是非常不友好的，玩家氪金打造+15紅字裝備，算是非常的犀利的，神話只能帶一件和其餘的觸發特效。
