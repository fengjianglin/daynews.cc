☂榮耀戰力系統

只要玩王者榮耀的人都知道榮耀戰力系統。當你英雄的榮耀戰力變高，你這個英雄的排名也會隨之變高。

  • 當你的英雄戰力達到全縣（區）前100名時，你會獲得一個初級（銅牌）稱號；
  • 當你英雄戰力達到全市前100名時，你就會獲得一個中級（銀牌）稱號；
  • 當你英雄戰力達到全省前100名時，你就會獲得一個高級（金牌）稱號；
  • 當你英雄戰力達到全伺服器前10名時，你就會獲得國服最強的稱號。

那麼榮耀戰力是怎麼計算的呢？

官方給出的說法是榮耀戰力=（歷時戰績+歷史表現）*常用係數。

不過這段解釋其實已經過時了，現有的榮耀戰力並不是這麼計算的。

榮耀戰力=（巔峰戰力+表現分+戰績分）*榮耀係數*巔峰係數

☂榮耀戰力的組成方式

在榮耀戰力的組成裡面：

  • 表現分和戰績分是只可以在排位中獲得的，
  • 巔峰戰力是只可以在巔峰賽獲得的；
  • 榮耀係數是只要經常玩就行；
  • 巔峰係數是根據巔峰賽的分數進行額外加成。

❤ 表現分 ❤

表現分怎麼理解呢？

表現分的獲得取決於三個方面。

  1. 該英雄的歷史戰績。
  2. 全平台同段位該英雄的戰績。
  3. 該局勝負。

可以看到這張圖，我這一局玩的是孫悟空，我的本局評分是12.4超過了我的歷時平均評分8.6。與此同時，12.4分超過了同段位92.86%，所以這一局我的表現分就加的蠻多的，當然這也是我獲得金牌的原因。

表現分大概分了幾個檔次，我遇到比較多的是40+、50+、60+、80+、90。不過，我在諮詢了朋友以後發現，還有20+、30+的檔次，我倒是沒怎麼遇到過。

90目前應該就是封頂了，很難遇到，只有在幾乎完美的表現，又或者這個英雄你好久沒玩了偶爾拿出來用一把，才有可能獲得。

就比如我的老亞瑟，平時基本不玩，實在沒辦法，拿出來應急，結果贏了以後，就拿到了90的表現分。

另外，表現分根據不同段位也是有上限的。

  • 鑽石段位——上限1000分；
  • 星耀段位——上限1250分；
  • 王者0—10星——上限1500分；
  • 王者11—20星——上限1750分；
  • 王者20星以上——上限2000分。

❤戰績分 ❤

戰績分怎麼理解呢？

官方說戰績分根據段位不同也有上限，不過根據目前已知信息，並沒有一個明確的數值告訴我們它的上限到底是多少。

該圖片來源自網路

表現分打滿以後就會提示達到段位上限，戰績分在達到3300左右時就會特別難提升。我查了很多資料，也沒有發現有人能將戰績分打滿，大家普遍猜測，戰績分滿分可能是4000。

一般來說戰績分在達到3000以後就有點難提升了，因為贏一場從之前能加50-60分，直接變成20分左右。

❤巔峰戰力 ❤

在三個基礎數值中，巔峰戰力是最合理的一個。

首先，巔峰賽是單排，除開特殊情況，大家都是憑自己的實力打的；其次，只要分數不是太高，勝率超過50%，巔峰戰力基本只賺不虧。

玩過巔峰賽的都知道，低分時只要你贏一把，英雄的榮耀戰力都是100多的往上加，那為啥我這一把贏了以後只加了53嗎？是我玩的菜？

其實這一把我的典韋是MVP，之所以只加了53分，是因為巔峰戰力也是有上限的，它和表現分一樣，每個分段的上限不同。

  • 巔峰賽1200分時，巔峰戰力上限400分
  • 巔峰賽1300分時，巔峰戰力上限800分
  • 巔峰賽1400分時，巔峰戰力上限1200分
  • 依次類推……
  • 巔峰賽2000分時，巔峰戰力上限3600分

總結一下就是，巔峰賽積分每增加100分，英雄的榮耀戰力積分上限就增加400分。

在這裡說個笑話，我打到榮耀以後，想把宮本武藏打上金牌，結果我在1200分巔峰賽2連跪，第三把終於贏了，我以為宮本終於要加分了，結果並沒有！巔峰戰力想要加分，巔峰賽積分必須要超過1200才行~~

❤榮耀係數 ❤

榮耀係數是個什麼東東呢？

如圖，當一個英雄你長期不玩的時候，榮耀係數就會從1開始下降，並且會越來越低。不過，只要你保持每周玩一把，這個榮耀係數就會保持在1。

平時玩排位，一把榮耀戰力加個100多是正常的吧？不過，肯定有不少小夥伴遇到過，贏一把直接加三四百的情況，這又為什麼呢？其實，這就是因為榮耀係數過低，在你玩了一把以後，重新激活了這個英雄的榮耀係數，導致榮耀戰力爆增。

正常情況下，榮耀係數你看看就好，常玩的話，它肯定是滿值的，並不要擔心。

❤巔峰係數 ❤

相比較榮耀係數，巔峰係數就稍微有一點複雜了。

巔峰係數的秘密全再這張圖裡。

我的宮本武藏本身榮耀戰力不多不少，正好4000，贏了一把以後，由於這一把是邊路比較混，巔峰戰力加了96分，那為什麼最終我的榮耀戰力不是4096而是4136呢？這就巔峰係數的原因了。

巔峰係數有一個很簡單的計算方式。

每提升20分巔峰賽積分，巔峰係數就提升1%。
  • 巔峰賽1200分時，巔峰係數為0；
  • 巔峰賽1300分時，巔峰係數為5%；
  • 依次類推……
  • 巔峰賽1800分時，巔峰係數為30%；
  • 巔峰賽1900分時，巔峰係數為35%；
  • 巔峰賽2000分時，巔峰係數為40%。

    • ☂拿省排名的捷徑

    介紹完了榮耀戰力的計算方式以後，是時候來告訴大家拿省排名的捷徑了。

    還是拿我的宮本武藏來舉例子。

    目前江蘇最後一名是4739，我是4136。

    上面圖中有顯示，目前我宮本的榮耀戰力組成是994（表現分）+3006（近期戰績）+96（巔峰戰力）等於4096，然後4096再乘以係數1.01等於4136.96（數據好像有一點點偏差哈）。

    ❤捷徑的原理❤

    以我自己舉例子，我300多場打到了榮耀（50星就沒打了），那麼我的巔峰賽如果認真打，大概是個什麼水平呢？讓我們來看官方的提示。

    這個是S17賽季剛推出的榮耀挑戰活動。當你挑戰值達到120時，賽季皮膚就會升級。根據等價原則，我們就可以根據自己排位段位大概的判斷自己巔峰賽的水平（不過這裡有個前提，你要和我一樣絕大多數都是單排）。

    所以，我王者50星≈巔峰賽1800分水平。

    巔峰賽1800分可能確實需要很多時間，折算一下，按1600分計算好了。

    如果我現在巔峰賽1600，即使我宮本一把不打，你猜我宮本榮耀戰力會是多少。

    答：4096*1.2=4915.2。

    江蘇省宮本第67名時4914，我就算宮本一把不打，只要打巔峰賽上分，就可以輕鬆達到省67的排名，而這就是我說的捷徑。

    ❤捷徑怎麼走❤

    我們就按最按王者10星來（如果10星水平都沒有的話，拿省標確實有點困難了），再取江蘇省第99名宮本榮耀戰力分來計算。

    已知：第99名宮本榮耀戰力是4739，我們的水平是王者10星，求如何超過第99名。

    解：

    （表現分A+戰績分B+巔峰積分C）*巔峰係數＞4739

    第一步：已知王者10星≈巔峰賽1350分，所以巔峰係數為7.5%；

    第二步：4740除以107.5%=4409.3。

    所以，我們需要的是（A+B+C）≥4409.3。

    根據我的排位經驗來看：

    我的宮本武藏是我到了20星以後開始用的，本賽季排位一共打了57把，勝率是62%。

    可能和段位以及單排有關係，我的表現分打到1000就上不去了。倒是我的戰績分很輕鬆的到了3000+，不過到了3000+以後也明顯就有點入不敷出了。

    我擔心我統計的宮本武藏代表性可能不夠。於是在我拿小號去測試，再結合不少粉絲提供的數據，最後我得出一個大致的結論：

    • 表現分在低段位還是比較好刷的，不過普通單排玩家一般也不會超過1200；
    • 戰績分只要你正常玩，只要勝率超過50%，基本都能達到2800，勝率好一點點就是3000+。

    根據這個數據我們可以初步知道A≈1200，B≈2800，套到前面公式里，就是4410-4000等於410。所以，我們巔峰賽只需要410的積分就可以拿到省排名。

    410分很難嗎？從1200分打到1350分，只要你想沖榜的英雄多贏3、4把就可以完成任務，可以說是輕鬆加愉快吧？

    什麼？還是覺著很難？那麼我們換廉頗再看看？廉頗上榜戰力4000都不到！對於這種冷門英雄，理論上來說，你連王者都不需要上，巔峰賽也不需要打，保持住50%勝率，多玩幾把，你就可以上榜。

    ❤玩的真不好怎麼辦❤

    如果你是一個新手，又或者你確實沒有遊戲的天賦，但是你又想上排行榜，怎麼辦呢？

    縣級排行實在太low，我們看一下市級排行榜的要求，市級排名最後一名玩家評分是2770，還記得我前面測試過的嗎？一個英雄，只要你維持50％的勝率，正常玩下去，你的戰績分就能有2800+，表現分你總不可能玩成負的吧？

    所以我們可以得出結論：只要你認真玩，找一個稍微冷門一點的英雄，哪怕你是個新手或者真的挺菜，你也可以上到市級排行榜！（當然，你也可以換個冷門的地區，這和找冷門英雄一個道理）

    ☂萬分國服的最低要求

    雖然這和普通玩家不搭噶，不過為了滿足一下大家的好奇心，我還是來計算一下那些萬分國服究竟怎麼打上去的。

    國服趙雲，上榜戰力最低11220。

    這十位大神我挨個看了下，巔峰賽最低的是1950，最高的是2043。

    按照最低1950分計算，巔峰係數是37.5%。

    所以（A+B+C）*137.5%≥11220，（A+B+C）=8160。

    前面已經分析過了，表現分滿分是2000，戰績分滿分可能是4000，不過這6000分想拿滿，我覺著即使是五排車隊也很難完成。我們姑且就按照5000分（表現分1600+戰績分3400）計算好了，我相信這個對於那些大神車隊來說，還是比較輕鬆。

    反推，我們可以得出結論：這些國服大神，巔峰賽最起碼需要打到3100左右的巔峰戰力，才能成為萬分高手！

    萬分國服的最低要求（大概）：

    • 表現分＞1600（越高越好）；
    • 戰績分＞3400；
    • 巔峰戰力＞3100；
    • 巔峰分數2000左右。

    ❤總結❤

    排位是基礎，巔峰是升華！

    萬丈高樓平地起！不管是現實里的高樓大廈，又或者是遊戲里的榮耀戰力，想要高人一等就一定要打好基礎。

    如果，你想要榮耀戰力達到比較高的水平，一定要把排位里的表現分和戰績分打高一點，夯實基礎。

    當你的排位分比較滿意了，你才可以轉去巔峰賽，進一步的提高自己的戰力。

    ☂作為四年老玩家的一些肺腑之言

    一路風風雨雨，王者榮耀也已經陪伴我四年之久了，都說日久生情，王者榮耀我是看著它一點點火起來，又一點點衰落下去的。

    咱們今天說的是榮耀戰力，我就來說一說榮耀戰力這裡，我認為有哪些需要改進的地方吧。

    ❤規則老舊❤

    首先，這裡我看了很多遍，還是沒看懂，這個1.5.是個什麼意思。按理說，這麼大的公司，不應該犯這麼低級的錯誤吧？

    其次，在已經有了巔峰賽的情況下，至今沒有把巔峰賽的這一部分加進規則里，是不是確實有點不大好？

    ❤規則不合理❤

    玩新號或者練新英雄的朋友只要注意就會發現，一開始英雄的表現分會加的特別快，只要勝率過半，分數就會蹭蹭的漲，而要想加到同樣的分數，老號最起碼要保持66.6%的勝率才能做到。

    造成這一切原因其實是規則中的勝場數以及勝率。玩久了的號，達到一定場數以後，只要你還繼續玩，你的英雄勝率必然會越來越低。

    只要你有一兩場發揮的不好，你就必須要打三四場才能把分彌補回去。為什麼我會有這麼深刻的領悟呢？原因就是S16賽季，我的典韋。

    我的典韋一開始打到30多星的時候，單賽季勝率接近70%，在沒打巔峰賽的情況下，我成功拿到了省排名，但是越打勝率越低，最後到了榮耀的時候，我典韋的英雄戰力差點連市級排名都保不住（其他英雄玩的不好，只能玩典韋）。

    這就是很多人大號的縮影，越玩越難玩。而新號的狀況就完全不一樣了。

    上面說了，我的宮本是這個賽季剛練的，也可以算是一個新號，所以我在40場左右的時候，單排位宮本的榮耀戰力就已經達到了4100+。也差不多在那個時候，發生了一件事情後，我發現再打排位，宮本的榮耀戰力就越打越低了，哪怕我的總勝率最後還超過60%。

    什麼事呢？當我的宮本武藏的熟練度達到紫色，也就是宗師戰士後，排位再想加分就變得困難了，再結合官方規則看，我們就可以得出一個結論。

    • 勝場沒有達到要求前，只要你正常玩，你的表現分就會蹭蹭的漲，因為這個時候算是新手保護期；
    • 勝場達到要求後，就會改用勝率的標準來衡量加分標準，誰讓你是老手。

    如果你不是大神，你不是多排，你繼續玩大號，你的勝率總歸會越來越低，你的分自然也會越來越低。

    這合理嗎？？？？

    早在今年2月，就有策劃針對這個事情做過回應，很遺憾，目前我是沒有發現有任何調整的地方。

    給我的整體感覺就是，策劃其實是鼓勵玩家們多開新號。玩新號就可以更輕鬆的拿到排名，有一定基礎的甚至還能拿到國服；當然了，新號也代表著充值、皮膚、抽獎等等。

    王者榮耀玩家成千上萬，甚至以億計，我作為一個王者榮耀的忠實粉絲，在這裡給天美官方提一個建議。

    ❤做好切割❤

    賽季結束以後，榮耀戰力是會進行一定比例的折損的。那為什麼，之前的勝場、勝率就不能進行類似的調整呢？

    如果新賽季開啟，在計算榮耀戰力的時候，所有的勝場和勝率都會重新計算（歷史勝率不變），這樣不就可以讓大家都處在同一個起跑線上了嗎？

    為什麼玩王者榮耀的人越來越少，原因難道不是因為新陳代謝不對嗎？新人進入的速度遠遠跟不上老人退出的速度。

    不管什麼遊戲，如果連自己的根都沒辦法守住，只是一味的去追求一些短暫的利益（充值或者在線率），等忠實粉絲也忍受不了的時候，也許，這個遊戲就終結了。
