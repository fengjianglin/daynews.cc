《AI：夢境檔案》評測：不容錯過的推理神作

2019-12-15 63條評論 1,620次閱讀 398人點贊

原創： 庫巴姬

《AI：夢境檔案》是打越鋼太郎繼《極限脫出》系列後再次牽手Spike Chunsoft打造的一款偵探類文字冒險遊戲。作品延續了打越鋼太郎先生一貫的風格，在看似俗套的設定上大膽創新，玩家扮演一名特別搜查隊的警察，通過遊離現實與夢境拼湊出事件真相，解決各類離奇的殺人案件。相比傳統的同類作品，本作在風格上做出了一定突破，下面就一起來看看本作整體質量如何。

作品淺析

《AI：夢境檔案》這個名字本身就很奇怪，從夢境這兩個辭彙中很容易讓人聯想到《盜夢空間》，而剛好本作就是需要「潛入」他人夢境進行調查，遊戲過程中會逐漸加深這印象。而探索的過程，則會讓人覺得像《黑客帝國》這方面又把AI這個關鍵字引導了出來。

「AI」是標題最關鍵的部分，這既代表了我們常識中認識的「人工智慧」，也代表了日語中的「愛」以及英語中同音的「EYE」，可以完全指代我們主角的人工智慧義眼「AI-Ball」。而「AI-Ball」在日文的同音中也有「夥伴/同伴」的意思，是整個作品中最關鍵的破案道具。

這一切東西，都讓整個世界看起來非常浮誇，可是在打越鋼太郎的筆下，這個世界的推理卻異常的真實。沒有「超展開」、「神轉折」，所有線索、伏筆和路線一開始就放在你面前，只不過信息太多被覆蓋了，解開謎題靠的不是玩家的突發奇想，解開謎題之後的解釋部分也不會給人「原來如此」的錯愕，能解開謎題就代表你已經洞悉全局，雖然更加辛苦，但成就感絕對足。

遊戲體驗

接觸這款作品之前，首先玩家要明白ADV和普通AVG之間的區別。ADV雖然經常被稱為「日式AVG」，但是ADV通常是有大量行動需要玩家自己完成，並且整個過程沒有引導，更加強調玩家自己去挖掘線索，以及對於線索的分析與觀察能力。

與被稱為「視覺小說」的AVG在敘事和玩家浸入體驗都有著極大的區別，這段時間剛推出中文的《逆轉123》便是其中的代表，還有即將圓不下去的《命運石之門》、同樣為打越鋼太郎的《極限脫出》與Chunsoft的成名之作《428被封鎖的涉谷》，都是品質極高的ADV作品。

回歸正題，正如上面淺析所說，這是一部套著科幻、未來以及人工智慧等標籤的本格推理作品，因為玩家不同的選擇操作下會派生出大量的可能性，所以本作是一個註定會有多結局的作品。內容依然豐富得嚇人，推動一條線路玩家就會消耗10~20個小時，而且全程給人的懸念都特別足，只要沉浸體驗過就會覺得根本停不下來。

本作的搜證部分分為兩塊，現實世界和夢境世界，現實世界的搜證與傳統ADV玩法類似，大概就是這裡點點，那裡問問，換個角度再看看什麼的。而主角擁有的義眼「AI-Ball（瞳絆）」更是給玩家大大降低了探索難度，在各種證據中舉一反三給出提示，幫助玩家輕鬆解決許多無法完成的探索，難度非常親民。

而夢境世界的搜證則比較複雜，玩家要在6分鐘之內進行解謎，瞳絆在這裡可以隨意移動探索場景，但是每個動作都會消耗一定量的時間，而且搜來的證據未必真的有用，玩家要結合現實的證據和位置進行相關調查，除了考驗玩家的分析能力，同時也是在考驗玩家的時間分配能力。

不過限時搜證雖然讓人感覺到緊張又刺激，卻是讓許多玩家無法從中感受到樂趣，歐美媒體評測本作時對於這個時間限制非常不理解，不過玩家如果把劇情推進到最後，了解了整個事情的來龍去脈之後，你就會徹底明白為什麼ADV是日本人的專利，歐美玩家確實不懂這一套。

遊戲的人設是由《火紋覺醒》的畫師小崎祐介完成，剛好最近的《機甲戰魔》也是他進行的人設，可以感受到兩個作品的人設確實有點類似，習慣的朋友應該會覺得挺不錯的。作為一款首發就有中文的ADV，中文的漢化水平挺優秀的，本地化的內容也轉接得不錯，絲毫不會破壞整個遊戲的環境。

唯一讓人覺得奇怪的地方，大概應該就是作品中的角色，每個說起話來三句里至少有兩句話在「開車」。除了葷段子之外，還有很多日式笑話、冷笑話以及無厘頭廢話，在現實調侃主角這麼說就算了，到了夢境里居然調侃起了死人……也許這就是特殊的氣氛調整技巧吧。

總結

對於這部作品的劇情，我除了說一句「好！」之外真的不知道說什麼了，感覺隨便說說都會劇透，因為整個作品的邏輯非常嚴謹，只要一點點劇透就會讓玩家的體驗大打折扣。而本作設定在未來，偵破案件的手法也很特別，但這絕對是一款邏輯嚴密的傳統偵探類作品，對於喜歡此類作品的玩家而言，這應該是近五年來最不容錯過的推理作品！

最後一點小提示：ADV講究自己去探索，自己去摸索線索說的什麼，而不是盲目聽從指示去辦事，希望玩家有自己的判斷能力，不是每個NPC說的話按著做就是對的。如果發現有一個必死的局無法繞過去，那麼肯定是在曾經的一步操作中踩了伏筆，記住失誤的地方，其他就順其自然吧。
