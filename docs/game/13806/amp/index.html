Kpl三周年語音曝光，QG五虎重聚其樂融融，刺痛教老陽出裝反被罵

天天要聞
2019-12-13

KPL秋季賽已經進行到總決賽階段了，這個賽季的比賽可以說格外的好看，無論是常規賽還是季後賽都有很多看點。還有一個不得不提的就是KPL三周年活動，官方邀請了之前所有打過總決賽的隊伍，挑一局最經典的比賽對局重現，既有第一屆總決賽的仙閣大戰AG，也有第二屆的「雙G大戰」，也勾起了不少粉絲們的回憶。而QG五虎重現舞台效果也相當不錯，兩場復盤比賽都贏了，當時的語音也被放了出來，五虎還是其樂融融，並沒有像外界所說的那樣不和。

QG五虎重聚之後似乎並沒有因為分別時間過長而感到生疏，賽前的交流都非常自然。貓神讓老yang幫他清中路兵線，fly也表示可以幫他在中路劈一刀，貓神回應可以。而最有意思的還是刺痛在線教老陽出裝，他對老陽說得出輔助裝。老陽聽到之後忍不住罵了髒話，還真以為他不會玩了，好歹老陽現在也是QG教練組的成員，這種必要的東西還是知道的。在之後刺痛還建議老陽出小鎧加魔女，alan這個時候也忍不住對刺痛說「你教人家出裝幹嘛」。老陽也是調侃刺痛回去就教他玩遊戲，彼此之間沒有一點隔閡。

在比賽當中他們的交流也非常順暢，無論是掩護隊友撤退，發起進攻還是開團都還挺默契的，就算是不及當年，但是這麼久沒在一起玩了也非常不容易。能感覺到QG的幾位老隊員關係還是不錯的，只是粉絲們不和而已。包括這期的語音明明幾位選手交流的挺愉快，還是有一些粉絲覺得他們之間差了點什麼，內心戲確實是太足了點。選手們都是二十來歲的年輕人，彼此並肩戰鬥過，即便是現在不在一個隊伍了，也不可能苦大仇深的。電子競技轉會很正常，不是每個選手都能在母隊打完整個職業生涯，還是理性看待的好。

一些粉絲總是喜歡把自己的情緒帶入到選手當中，可能自己覺得喜歡QG就不能再喜歡貓神，總要做出抉擇，但其實既喜歡QG又喜歡貓神又喜歡estar的觀眾也大有人在，很多人看比賽就是圖個精彩而已。只能說粉絲們喜歡選手沒問題，但是強行替選手腦補一些事情就實在沒必要了吧。

大家覺得QG五虎的關係是不是依然其樂融融呢？歡迎在評論區留言交流。我是天黑君，期待下次再見！
