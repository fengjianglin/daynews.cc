火影忍者手游：19年限定招募忍者強度分析，你是否招募了？

天天要聞
2019-12-08

摘要

截止到最新的限定忍者——忍戰薩姆依上線，今年所有的限定招募忍者已經全部上線，不知道大家對今年的限定招募忍者是否喜愛呢？我是覺得忍戰薩姆依真香，傷害不俗且還有位移，大招還可以接技能，不可多得的忍者呀！

截止到最新的限定忍者——忍戰薩姆依上線，今年所有的限定招募忍者已經全部上線，不知道大家對今年的限定招募忍者是否喜愛呢？我是覺得忍戰薩姆依真香，傷害不俗且還有位移，大招還可以接技能，不可多得的忍者呀！

然後就是10月上線的六尾人柱力羽高，這個忍者就是讓人討嫌的忍者，這忍者完全就是來秀恩愛的，哪裡是來守護木葉的？站街還都是兩個人在泡泡里。把你招募來你就這麼對待我？看來你是要當倉管了！

接著是8月上架的首個限定A忍忍戰丁次，招募券也是要14張，那麼你的強度應該會比較高的，可是玩過之後才發現，這14張招募券花的真的太冤枉了，一個A忍強度還沒有B忍的強度高！太令人心寒了。好在我沒有花這個冤枉錢。

接著是6月上線的鷹重吾，該忍者強度還可以的畢竟是鷹小隊裡面的隊員，可以說不負玩家的眾望吧。重吾原本善良的面目下竟然會藏著這麼可怕的力量！

4月上架的忍戰井野，這個忍者上線後，我是被率過好多次了，一技能距離有限長，還可以釋放兩次，二技能轉圈圈時間很長，還可以移動，想跑都很難。不得不說該忍者很強！

以及3月份上架的忍刀七人眾系列——無梨甚八，該忍者個人覺得太過於笨重，強度不高，技能有點花里胡哨的，哈哈哈

最後就是1月份上架的尾獸系列忍者——五尾人柱力漢（差點想成了漢尼拔了）這個蒸汽忍者傷害不俗哦，雖然看起來笨重，但是操作起來還是挺靈活的！

以上眾多限定招募真正中，我覺得忍戰薩姆依強於其他忍者，其次是忍戰井野、漢。雖然有些忍者目前還沒有招募到手，但是魔方未來肯定是要的，況且周年慶的時候還會送老限定忍者呢！
