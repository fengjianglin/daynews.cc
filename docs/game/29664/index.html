《New GAME！》原作者助陣，芳文社手游“作家關”效果如何？

2019-12-25 3,039條評論 4,422次閱讀 2308人點贊

隨著手機遊戲的興起，好劇本越來越少。那些遊戲公司，紛紛邀請其他行業的能人助陣。

比如製作美少女遊戲出名的奈須蘑菇，就與Aniplex聯手，創造了大熱的《Fate/Grand Order》。由輕小說家寫劇本的手游《公主連結 RE：Dive》，最近也頗為熱門。

有一家出版社，就把這種「跨行業聯動」發揮到了極致。芳文社在官方手游《閃耀幻想曲/ Kirara Fantasia》中，特別設置了「作家關卡」，讓漫畫家寫手游劇情。

《閃耀幻想曲》：講述芳文社Kirara漫畫中的人物，穿越到幻想世界的故事的手機遊戲

以前，OurPlay也寫過一篇「得能正太郎」的遊戲關卡創作訪談。那次訪談之後，又有三位芳文社漫畫家參與了這部手游的劇本撰寫。我們實際遊玩了一遍，並為大家解說這些「原作者腳本」。

另外這些漫畫家原作章節，屬於遊戲的外傳。劇情開始的時間點，大多設定在遊戲中期，理論上可能會劇透。

但由於這款遊戲不是那種害怕劇透的嚴肅類型，所以玩家也不必過於在意。（想要自己體驗的玩家，可以使用OurPlay加速）。

1、《New Game！》：做遊戲的社畜，在夢裡打遊戲

章節名：PECO A trial version 測試版

作者：得能正太郎

章節長度：23節

劇情簡介：

穿越到《閃耀幻想曲》世界的葉月雫，不小心發動了奇怪的魔法。涼風青葉等人，被魔法送到了「飛鷹躍動」製作的遊戲世界。

出場人物基本，就是這幾位遊戲公司職員

遊戲世界裡，葉月雫變成了迷宮女王。幾位「飛鷹躍動」的職工，也被洗腦成了女王的部下。為了回到原來的世界，涼風青葉踏上了攻略迷宮的路程。

漫畫家得能正太郎，曾經在遊戲公司工作。他轉行當漫畫家後，在芳文社連載的漫畫《New Game！》，講的也是遊戲公司的故事。

作者為遊戲專門設計的「熊衛兵」

《New Game！》這部漫畫，曾兩次改編為動畫。然後這兩季動畫中，都有「漫畫作者寫劇本」的「動畫原創回」。

也就是說，得能正太郎有創作文字腳本的工作經驗。這些積累，在《閃耀幻想曲》關卡當中也得到了很好的發揮。

被洗腦的阿波根，提議用猜謎決勝負。八神光與遠山倫，依然在表演讓人臉紅的戀愛劇

畢竟是原作者劇本，人物性格把握得很好，一點也不突兀。這些遊戲劇情，當《New Game！》外傳看也完全沒問題。

2、《黃金拼圖》：金髮控的異世界冒險

章節名：黃金Darkness

原作者：原悠衣

章節長度：25節

劇情簡介：從小到大，大宮忍一直都非常沉迷「金髮」。來到魔法世界後，她發現可以通過召喚，來與更多金髮女孩相遇。這件事的背後，被冷落的愛麗絲開始對忍產生不滿……

《黃金拼圖》的原悠衣，曾經在訪談裡面提到，《閃耀幻想曲》的作家關卡，並沒有什麼特殊的創作限制。這種寬鬆的環境，讓《黃金拼圖》作者可以選擇「用漫畫講故事」。

《黃金Darkness》每一關的劇情，都是一個個的四格漫畫。這些全彩的漫畫，甚至給人「不花錢白看漫畫」的感覺。

另外這個篇章裡面，不只會出現《黃金拼圖》的角色。其他作品裡的金髮人物，也會在劇情中登場。不只是《黃金拼圖》的粉絲，其他芳文社漫畫的讀者，也能看到自己喜歡的人物。

3、《晴海國度》：小姐，你的畫風似乎有點不對？

章節名：晴海咖喱

原作者：大沖

章節長度：32節

劇情簡介：細野晴海，是一位來自外星球，會飛的超能力女高中生。來到劍與魔法的世界後，她依然還是那麼脫線。為了尋找自己喜歡的咖喱，晴海走上了搜集材料，在異世界製作咖喱的道路……

《晴海國度》這部漫畫，在2008年就已經開始連載了。但《晴海國度》沒有改編過動畫，與其他作品相比，這部漫畫的知名度並不算太高。

和《黃金拼圖》一樣，《晴海國度》也選擇了「用四格漫畫講故事」。但在《晴海咖喱》裡面，整個世界的畫風都變成了簡筆畫風格。

這些遊戲關卡的劇情，也變得非常脫線。主角晴海，知道自己是「從漫畫世界來到了遊戲世界」。這種「看穿第四面牆」的能力，讓人聯想到了漫威的死侍。

雖然《晴海咖喱》章節數很多，但多出來的部分，基本都是不需要戰鬥的劇情章節，所以並不會很冗長。在劍與魔法的異世界，這位脫線的晴美，依然創造了不少笑點。

4、《悠悠式》：稍顯嚴肅的劇情，和嚴肅不起來的「相聲段子」

章節名：林中小屋與悠悠布偶

漫畫原作者：三上小又

遊戲章節長度：24節

劇情簡介：情報處理部的三個人，聽到了Kirara世界的悲傷童話。然後進入傳說中的森林之後，發現了一個被詛咒的玩偶……

《林中小屋與悠悠布偶》的故事，從「情報處理部」開始。《悠悠式》的主角，也就是野野原柚子、櫟井唯、日向緣三人，聽Kirara世界的「蘭普」，講了一個關於「藝人、公主與國王」的童話。

之後，三人來到森林冒險。她們在林中小屋裡，找到了和童話相關的「詛咒布偶」。唯在兩人的攛掇之下，穿上了這件衣服……

《悠悠式》的關卡裡面，不但和其他章節一樣，畫了專門的立繪。戰鬥地圖場景里，也有穿著布偶的live2d形象。

當然，《悠悠式》的核心看點，也就是那種「聰明人才想得出的愚蠢冷笑話」，這次也充分地保留了下來。

本次考察就到此結束。在原作者的協助下，這些「作家關卡」確實都保留了原作的味道，值得親自去玩一玩。

另外這些關卡難度很低，並不需要很高等級才能通關。就算是新手，多續關幾次也能打敗BOOS，看全劇情喔。
