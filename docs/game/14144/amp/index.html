這5種遊戲有讓你玩到大崩潰的嗎？

天天要聞
1 週 ago

你是遊戲的愛好者嗎？從電腦遊戲到手游都有很多的愛好者和佼佼者。遊戲讓我們放鬆心情，娛樂、打發時間。有些電子遊戲卻可以讓人怒不可忍，恨不能將手邊能砸的，能破壞的全部破壞掉，以解心中的爆發情緒，玩了些什麼？的確會讓玩家紛紛崩潰。。。。。。手指無限制的按重來，讓自己，一小時為你。。。。。自己到底玩些什麼？！讓你大崩潰！

以下介紹5個讓你玩到大崩潰到遊戲~！

1.貓力歐

貓力歐，一個絕對會讓人玩到頭疼，玩到腦殘身體崩潰的遊戲。瑪麗歐惡搞有版本許多，這款貓力歐遊戲規則和原瑪麗歐遊戲規則相反。每個遊戲細節，都是陷阱，就像這裡的蘑菇是有毒的，進入水管後會讓你一飛衝天，還有隱藏的磚塊，就連遊戲的終點旗杆也是一個陷阱，而且玩家有時候生命值還會變成負數，這些陷阱遍布整個關卡，並且會在玩家接近它們或接觸它們時啟動，所以玩家難以完成遊戲，能夠玩到過關都必須超有耐心。

2. 小恐龍

斷線時可以玩谷歌送你的小恐龍遊戲

開心呀

斷線時小恐龍陪你 谷歌瀏覽器隱藏了一個彩蛋，那就是小恐龍遊戲。谷歌設計了一隻小恐龍來陪伴大家，在斷線的情況下用戶們可以和小恐龍玩遊戲。開始於2014年9月25日，遊戲里的小恐龍要跳躍各種仙人掌和障礙。恐龍奔跑的速度也會不斷增加，偶爾還會從白天變成黑夜，玩家們盡了力想玩到終點，但是這款遊戲沒有終點，儘管你花了十幾個小時在熒幕前，想終極大爆炸或者到遊戲的終點，但你不會遇到終極大boss或者遊戲的終點，遊戲的積分只會越來越高，但網路恢復了頁面就會強制性刷你的累積分數就。。。。。。

game over

3.魂斗羅

很熟悉的畫面吧

魂斗羅是80 90後門的經典遊戲，是日本科樂美公司在1987年發行的街機遊戲系列，後來一直到其他平台玩家們扮演特種部隊，面對大量的敵人，從人和機器到異形和外星人，這個遊戲和現實生活的人類一樣一顆子彈就必死無疑。三條壽命讓玩家闖關，要歷時10多小時才能打進大關，而且遊戲的難度非常高，斜向射擊操作更是難上加難，叫玩家怎麼能不崩潰呢？

4.像素鳥

在2014年2月9日有一款遊戲遭到開發者自己下架了，他就是像素鳥。別看畫面簡單，不僅無野心還吊你胃口，無限的虐下去，有些畫面你的小鳥會由左向右快速移動，玩家需要點擊屏幕，用點擊的節奏來控制像素形態大眼鳥上下移動，避開前方會出現的排水管，通過排水管之間的縫隙就能得分；碰撞到排水管，馬上game over，再重新闖關。這款遊戲剛登上各大遊戲排行榜時，開發者自行將遊戲下架。因為他發現像素鳥使很多人上癮。

5.光頭鎚子遊戲

2017年有一款遊戲讓全球玩家紛紛崩潰，主角是一個光頭泡在盆子里，手裡拿著一隻鐵鎚。看到這裡，你可能會覺得這到底是啥小遊戲，但操作過後會發現這個遊戲的概念其實很簡單，鐵鎚位置是跟著滑動滑鼠軌跡走，以此靠鐵鎚來翻越障礙物

，最後的目的是不斷的爬山脫離地球

。但當你好不容易手酸但爬到高處時，一個不小心摔到山腳下，你可能會覺得很崩潰，很多玩家都快氣到中風了。這一過程經歷一次次的挫敗後，還有作者的旁白出現，告訴你人生哲理，例如「重新開始比開始更難，只要你找個安全的地方停下，然後退出遊戲」等，看視很嘲諷的話。
