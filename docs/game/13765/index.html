雲頂之弈出現“白給戰術”，主C放在角落給刺客切反而更強

2019-12-13 1,056條評論 2,648次閱讀 210人點贊

出現「白給戰術」，主C放在角落給刺客切反而更強！

在雲頂之弈中，玩久了大家都會有一個自然而然的習慣，那就是會將自己的主C盡量放在一個安全的位置，讓他放心輸出，現版本來說，一般是放在中間位置，左右兩邊都有英雄，前面是坦克，主C在中間輸出。這種站位也是一種比較常見的站位方式，不會犯太大失誤。這種站位雖然有用，不過在劣勢局想要翻盤時，這樣的站位則起不到奇效。最近，盟盟在排位時，就採取了一種激進的站位方式，直接將自己的主C放在角落，雲頂之弈的「白給戰術」，主C放角落讓對面的刺客過來切，因為當前版本綠叉可以閃避一切暴擊傷害，所以很克制刺客。

於是我將角落的兩個英雄都給了綠叉裝備，另外火男還給了復活甲，進一步增強自保能力。這樣的站位可以儘可能讓主C躲避控制技能，然後又可以依靠對面刺客的攻擊更快的攢藍條。當然，如果有娜美這種英雄，一定不要講主C放在對角，不然還是會被控制到，而且這種站位不能一開始就暴露出來，只有等到決賽圈時出其不意來一下，才能攻其不備，取得奇效，主C放角落反而更強，大家學到了嗎？

玩家養了一個巨型掘墓人，一次能復活12個隊友！

四影玩法師雲頂之弈中一個相對強勢的玩法，中期強度非常高，只要全二星基本可以一路連勝，但是這個陣容想要吃雞還是挺難的。在大家陣容都已經成型高的情況下，四影的傷害就顯得跟不上了，所以想要吃雞千珏必須要三星，提高傷害。在一場對局中，盟盟也玩了這個套路，但是到最後千珏還是沒有三星，不過無心插柳柳成蔭，陣容中的四費卡掘墓人已經升到了三星，沒錯，我養了一個巨型掘墓人，一次能復活12個隊友，而且他還有復活甲、狂徒、冰拳這幾件裝備。

所以有了這樣一個大肉盾在前面，就算千珏輸出略顯不足，但是憑藉掘墓人的高坦度，硬是給千珏爭取到了收割的時間，而且掘墓人還召喚了很多小怪出來，分散對面火力。所以說掘墓人這個坦克英雄還是很厲害的，特備是面對刺客陣容時，因為是單體輸出，召喚的小怪都夠刺客打半天，所以還沒打完小怪自己一方的早已陣亡了，巨型掘墓人真的太強了，3星3神裝，就問還有誰？

法海新變種—森林六法師，強度不輸法海

在雲頂之弈中，森林羈絆在前期很好用，大家都喜歡用這個羈絆來打工，如果前期可以找到一個全二星的三星陣容，那麼基本可以保證前期一路連勝。不過有時候森林牌來得太多，就忍不住多留幾張，結果最後森林牌都三星了，都已經三星了，所以再賣掉自然很虧。這位玩家本來想玩地獄法海，但是由於森林牌太多，於是決定玩一套森林法。因為後面運氣不錯，所以找到了兩個的魔法師轉職書，於是最後湊齊了六法師陣容。這一套陣容和傳統的法海陣容相比，在回藍方面遜色一點，所以輸出肯定趕不上法海，但是因為森林羈絆都已經三星，所以前排坦度優於法海，兩個陣容各有優劣，這個陣容也不失為一個好陣容。
