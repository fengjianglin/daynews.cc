Steam周銷榜《光環》衛冕，多部新作登上榜單

2019-12-16 455條評論 2,271次閱讀 1135人點贊

V社公開了Steam平台上周（12月9日至15日）的銷量排行，其中《光環：士官長合集》蟬聯冠軍。上周發售的新作有多人合作射擊《GTFO》、VR射擊《Bnoeworks》、交通模擬《狂熱運輸2》、《晶體管》廠商新作《哈迪斯》，分別位列第3、4、7、10的位置，此外也有一些老面孔。

《Boneworks》IGN 7.9分：「有血有肉」的物理模擬效果

IGN為《Boneworks》打出了7.9分的評分，他們認為本作極具野心，對遊戲中一切事物的物理模擬結合玩家的創意能造就大量的VR遊戲樂趣，可惜你的點子並不總能有效實現。

在整個戰役流程、競技場、沙盒模式中，《Boneworks》提供了充實而多樣的「物理驅動」行為可供嘗試，它們在有趣的同時也兼具了娛樂性。尤其是當你完全投入到角色身上，有目的地「表演」出遊戲中所有搖擺、抓取、投擲時，遊戲中VR動作可以得到最佳的表現。雖說在劇情、關卡設計、甚至是其核心特色——物理效果模擬中，都有大量讓人意猶未盡地地方，但這不妨礙《Boneworks》表現出驚艷的交互性、迷人的體驗以及大量重玩潛力。

卡比奪得IGN日本角色人氣投票冠軍

據nintendosoup報道，IGN日本近期開啟了一場遊戲角色投票，而卡比最終過關斬將奪得了冠軍。開發商Hal研究所也在推特上發表了祝賀和感謝。
