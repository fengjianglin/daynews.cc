AG奪冠，老帥因放狠話被QG粉絲噴，Fly放狠話卻沒事，嚴重雙標？

天天要聞
2019-12-15

2019年的KPL秋季賽總決賽已經落下帷幕，AG超玩會終於克服了心魔，戰勝了此前從未贏過的對手QGhappy。在決賽的舞台上，AG超玩會明顯是狀態更好的一方，一諾、老帥等人的表現都可以打滿分，而教練月光、輔助愛思、戰邊麟羽等人的發揮也非常出色，AG超玩會全員皆兵，配得上總冠軍。

這場總決賽的FMVP是老帥，賽後老帥難掩內心的激動情緒，淚灑賽場。老帥這一路走過來確實太不容易了，出道便是KPL第一中單，但之後多個賽季都沒有上場機會，回歸AG超玩會後，賽季初他也經常被質疑，甚至被說成「混子」和「牌神」，但老帥在總決賽的表現無可挑剔，證明了自己才是最好的那張牌，老帥的歷程彷彿小說主人公一般。

在賽點局之前的放狠話環節，老帥說了這麼一句話：我覺得一支只會射手，不會野核的隊伍，不應該拿冠軍。這句狠話一出，現場反應十分劇烈。放狠話環節要有節目效果，而且往往要直擊要害。老帥意思是QGhappy一直是圍繞刺痛打射手體系，而基本不打野核，這是QG的一大軟肋，這句話確實夠狠的。

但AG超玩會奪冠之後，老帥說的這句狠話卻被挖出來了，部分QGhappy的粉絲開始噴老帥，認為老帥不尊重對手，還說這是第一次在放狠話環節聽到選手說另一個隊伍不配贏。但其實老帥說的是不應該拿冠軍，措辭並沒有那麼激烈。

在這條超話下面，也有很多QG的粉絲表示贊同，他們認為老帥的情商太低，QG的選手就不會說出這麼不尊重對手的話，甚至有的粉絲還覺得老帥人品有問題。有一個粉絲還說道：老帥混得了一時混不了一世，冬冠期待FMVP的表現。

不得不說，老帥也太慘了，本身就是放狠話環節，狠一點才有節目效果，這也被拿出來黑，實在是冤枉。而其實放狠話環節Fly也說了這麼一句話：AG超玩會，萬年老二嘛！之前是，現在也是！

Fly這句話也是戳到了AG粉絲的痛處，但由於是放狠話，根本不會有人去追究。老帥因放狠話被QG粉絲噴，Fly放狠話卻沒事，這是嚴重雙標嗎？

這兩支隊伍無論誰贏，都配得上總冠軍，從常規賽到季後賽再到總決賽，兩支隊伍都付出了100%的努力，都值得尊重。還是希望粉絲們不要因為放狠話這個環節，上升到選手的人品問題。
