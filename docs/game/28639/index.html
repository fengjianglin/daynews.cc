聖誕活動來啦！武將兌換、聖誕專屬皮膚應有盡有

2019-12-22 288條評論 3,438次閱讀 134人點贊

每當這個時節，又會想起我們的老歌 ：「single dog，single dog，why not be a gay~ single bell single bell … ~~~」

沒錯，又是我們最熟悉的聖誕節了！白鬍子老爺爺，紅色的聖誕帽又要出現了！同時，不可缺席的一定是三國殺的聖誕活動！

情暖冬至，繽紛聖誕！麋鹿陪你過聖誕！

以往的聖誕活動還沒有出現專屬的聖誕皮膚，但今年因為有香香、劉備、陸神的聖誕皮膚所以有了聖誕的牌面，那麼看看今年聖誕活動有什麼呢？

【皮膚兌換】

這次皮膚兌換活動是專屬的聖誕皮膚兌換活動，可以兌換香香、陸神的聖誕皮膚，但是劉備不可兌換，只能秒殺活動獲得.

這次聖誕皮膚都乘上了半價合成的飛機~所以都只要22222雁翎就能兌換了，比起平時雷打不動的4萬雁翎無疑好了不知多少.

【餃子兌換活動】

其實這個活動每次活動都會出，但是既然今天講活動，所以還是要提及一下的~

消費活動還是一如既往，消費得寶珠、活動物品然後去兌換武將，當然兌換條件和比例還是一樣，史詩武將依然得20寶珠往上走.

【兩軍對壘】

這次依然是每天只能玩2次的兩軍對壘，不過獎勵變成了勝者&銀幣福袋*1；敗者銀幣*100.

雙方對戰武將是諸葛、魯肅VS 于禁、張遼，但這次武將的強度十分不平衡，可以說拿到魏國基本就贏了，沒有為什麼，因為蜀國就是兩白板.

【半價武將】

因為最近還有一個武將半價合成的活動，所以肯定很多小夥伴就要問了到底換哪個武將比較好呢？在說之前，我們先看看相應武將需要的將魂.

界貂蟬*300

李嚴*225

黃皓*200

其實這幾個武將我也不知道該怎麼建議，畢竟半價嘛都不怎麼貴，所以全部換了放小黑屋吧（滑稽），畢竟加將魂不美哉嗎？
