《Pine》試玩評測：優哉游哉，日出日落，平民版的“塞爾達”

2019-12-12 218條評論 1,955次閱讀 625人點贊

自從兩年前「萬物起源」《塞爾達傳說：曠野之息》發售後，「類塞爾達風」這個詞語就頻繁出現在各種作品中，它們往往都有著「開放世界」、「畫風清新」等標籤，可質量卻是參差不齊。今天給大家評測的這款獨立遊戲《松木（Pine）》，就是一款類塞爾達開放世界新作，在斬獲了不少獎項之餘，它到底是否能稱上真正的類神作遊戲呢？希望大家能從本篇評測找出答案。

遊戲介紹

本作是由Twirlbound開發、Kongregate發行的一款開放世界冒險遊戲，此前已登陸PC、PS4以及GOG平台，NS版本於2019年11月26日發售，售價24.99美元，無中文。

遊戲背景設定在美麗的阿爾巴馬世界，我們主角Hue所在的部落是世界的最後一批人類，為了生存下去，它們躲避地面上的怪物來到了高山，搭建樹屋並居住在其中。在一次災難中，村莊被毀，可憐的Hue也失去了他的哥哥，為了幫村民尋找一個更安全的居住地，Hue必須走出大山，為人類的生存展開新的冒險。

本作包含獨特的AI系統，能根據玩家的習性和選擇不斷改變設定，確保每個玩家都能擁有獨一無二的私人體驗，

–優點：畫風清新、BGM優秀、獨特的交互體驗以及在獨立遊戲中少有畫面表現。

–缺點：敵人類型單一、優化較差、建模重複、生存機制概念模糊。

上手體驗

本作真正的核心亮點在於和NPC的交互體驗上，在這裡你會發現每個生物種群都是獨特而美麗的，即使沒有你的參與，世界也能照常運轉，形成一個完美的生態圈，遊戲擁有5個不同的異種族部落，要想讓村民更加的安全，玩家必須給這些部落提供資源及物品來獲得它們的支持與庇護，但只討好一個種族是不夠的，會引來其它種族的不滿甚至仇視，這種近乎於原始社會的樸素邏輯，相比其他遊戲少了條條框框及各種複雜的事件選項，但卻多了一些簡單的樂趣。

然而，本作的沙盒世界規模不算小，但在此基礎上沒有足夠豐富的內容，又會顯得相對空曠。推動主線的主要方式就是和人物對話、採集物品、製作物品等這些偏枯燥的任務，這就使得本該是亮點的開放自由地圖探索反而變成了弊端，一直跑來跑去找東西十分考驗耐心。

機制玩法

遊戲的基本玩法可分為解謎和戰鬥。對於新手或不精於此的玩家來說，本作的謎題足夠豐富且具有趣味性，內容也不會太過複雜，整體比較輕度友好。

而在戰鬥系統方面，近戰打擊感雖然不錯，但動作略顯僵硬，攻擊的後搖實在是太長，而且還沒有辦法主動取消，敵人打你只能看著，十分憋屈，尤其是在前期裝備不足的時候，只能戰術撤退，好巧不巧，這裡的怪物都擁有仇恨高，觸發快的特點，隨便一隻肥鳥就可以追你到天涯海角，於是「狂放風箏設陷阱，簡單粗暴用炸彈」的玩法就出現了。不過有失必有得，正是因為戰鬥系統做的不夠完善，所以才能更加凸顯出這些創意玩法的樂趣。另外本作還存在飢餓與健康值的設定，但補血道具同時也可以回復飢餓值，所以這個設定對遊戲的影響微乎其微。

遊戲的音畫表現可以用驚艷來形容，出色的場景畫面，歡快輕鬆的BGM及環境音效，彷彿真正置身於這充滿鳥語花香的幻想世界，很難想像這是獨立遊戲的水準。但缺點在於人物建模的重複以及不太理想的優化，村民的男性基本都長一個樣子……模型會經常會掉入地底，水底以及卡在山上的各種石頭縫裡。

總結

遊戲初上手時的確會給人一種眼前一亮的感覺，開放自由的世界、生動清新的畫面、實時的環境音效以及優秀的BGM都會讓玩家沉浸其中，乍一看還真有幾分「塞爾達」的韻味，但隨著遊戲的進行，任務繁瑣無聊、怪物類型單一、後期內容貧乏等問題會逐一顯露，但這些並不能完全否定它作為一款獨立遊戲的優秀。如果你能適應這種慢節奏的遊戲方式，那麼本作還是可以帶來很不錯的體驗。
