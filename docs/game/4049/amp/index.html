2019LOL全明星賽創下最低收視率，網友吐槽：這些主播都不認識

2019-12-10

從上星期五到周日連續三天的《英雄聯盟》全明星賽不知道大家看了沒？這兩年的全明星賽加入了遊戲主播，使得賽事更加偏向娛樂性，雖然在去年創下新高，但今年的全明星賽則是2016年來的最低點！這項2013年開啟的賽事，算是《英雄聯盟》年末最有趣的賽事了，玩家可以看到不同戰隊的選手組成一個隊伍進行比賽，SOLO賽、雙人聯手等都是諷刺有趣的比賽項目，選手們也都比較放鬆！

根據國外賽事數據網報道，2019全明星賽收視人數非常慘淡，觀看人數最高峰為293351 人，相比去年的 565909 人，下降非常明顯！雖然有三天的賽程，但累積的總觀看時長比 2018 年縮水了237萬小時，基本上今年的收視率只有去年的一半，玩家的討論度也不高，唯一有討論的也就是SOLO賽，畢竟不少知名選手都參加了！

那麼是什麼原因造成了這樣的情況呢？網友們普遍認為除了賽程安排不合理外，最大的問題就是全明星賽邀請的對象！全明星賽的改變是從去年開始的，拳頭除了邀請各賽區的職業選手外，還邀請了各賽區的遊戲主播，去年首次嘗試效果不錯，帶來了不少新觀眾，但在今天就大幅衰退！

賽程安排的問題就是連續三天，把賽程拉的過長，再加上時差的原因，在拉斯維加斯舉辦比賽，中國觀眾要想看比賽那都要早上八九點就要起床看，周末很多人都要休息，不少人根本不想早起看比賽！

而邀請對象的問題就是職業選手大家可能都認識，但對於其他賽區的遊戲主播，可能大家就不認識了，大家會關注其他賽區的職業選手，很多選手也都是參與過世界賽的，但賽區的遊戲主播基本上都不會關注！因此在雙人聯手模式中，不少人都會問「這是誰？？？」

再有就是今年還加入了新的項目「雲頂之弈」，這個模式自己玩可能很好玩，但比賽的觀賞性不佳，8個人的遊戲畫面不停切換讓觀眾混亂，而且一些不玩這個模式的觀眾還存在看不懂的問題！

對於出現這種問題，不少網友都希望拳頭遊戲能夠對全明星賽進行改革，縮短賽程不要邀請那麼多的遊戲主播，讓職業選手們好好玩遊戲！對此你怎麼看呢？
