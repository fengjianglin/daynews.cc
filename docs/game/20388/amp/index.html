力壓群雄斬獲TGA四項大獎：講講《極樂迪斯科》和製作人的故事

天天要聞
2 天 ago

文丨西夏Wayne@bigfun社區

在剛剛結束的TGA 2019頒獎典禮上，憑藉《只狼》拿下了年度最佳的宮崎英高，在現場激動地喜笑顏開。然而，除了「猶豫就會敗北」之外，一款不太一樣的獨立遊戲，風頭卻幾乎蓋過了包括《死亡擱淺》在內的一眾3A大作。

最佳角色扮演遊戲、最佳獨立遊戲、最佳敘事三項大獎，哦對了，還得算上開發商拿到的最佳新晉獨立遊戲廠商獎——沒錯，這就是《極樂迪斯科（Disco Elysium）》，和來自愛沙尼亞的ZA/UM工作室。而在TGA現場，製作組更是在獲獎感言里，表達了他們對馬克思和恩格斯的感謝。

那麼，這款體積僅有1.4G的小品級RPG究竟有何魅力，能夠獲得TGA評委會和玩家的青睞？它的開發者ZA/UM工作室背後，就有哪些不為人知的故事？

出身藝術之家的左翼青年

不論是《極樂迪斯科》，還是ZA/UM工作室，故事的起點都源於同一個男人：他就是愛沙尼亞人，羅伯特·庫維茲（Robert Kurvitz）。

1984年，羅伯特·庫維茲出生於愛沙尼亞塔林，父親Raoul Kurvitz和母親Lilian Mosolainen，都是當地小有名氣的藝術家。也許是從小受到家庭環境的熏陶，庫維茲從小就展示出了不錯的藝術天賦。

但估計父母沒想到的是，寫作、音樂和遊戲，將在日後成為這個年輕人生命的重要組成部分。而庫維茲對寫作、音樂和遊戲的深刻認知，日後也將在《極樂迪斯科》中體現的淋漓盡致。

時間到了2001年左右，此時年僅17歲的庫維茲對馬克思主義越發崇拜，於是化身搖滾青年，出任進步樂隊Ultramelanhool的主唱兼詞作者。隨後的2004年和2008年，這群以庫維茲為首的20出頭的年輕人，又兩度發布進步歌曲專輯，其中一首單曲甚至還被愛沙尼亞熱門連續劇《Ühikarotid》選為插曲。

2009年，左翼思想越發成熟的庫維茲，為愛沙尼亞的藝術家們發起了一項藝術運動，並創辦了一個進步網站「ZA/UM」。是不是覺得，這名字很眼熟？今天，當我們前往已經轉型遊戲工作室的ZA/UM官網，依然可以看到大量極具搖滾氣息的設計原畫。

2011年庫維茲與老爹合作，發布了樂隊的第三張專輯《 Forbidden to Sing 》，並成功拿到愛沙尼亞文化基金會年度大獎。也許是覺得功成名就，也許是覺得音樂無法唱出自己對於革命的嚮往——總之，這位投身樂壇10年的男人，又把興趣轉向了寫作。

從音樂到小說，只為傳遞思想的火炬

根據愛沙尼亞媒體的採訪，庫維茲早在童年時代就鍾情於桌游，並尤其喜歡有著「魔幻聖經」之稱的《龍與地下城》。受DND的影響，庫維茲在拿下文化基金會大獎短短兩年後，決心將自己的進步思想轉化為文字，嘗試傳遞給更多人。

於是乎，他在2013年出版了首部小說《 Püha Ja Õudne Lõhn》（注意封面左下角，青年時代發起的ZA/UM運動，真的影響了庫維茲一生），嘗試用音樂之外的角度，宣傳自己的信仰。

而從谷歌閱讀接近4星（滿分5星）的評價來看，這本小說的質量，應該算得上是中等偏上了。

當然，重點還是在遊戲。除了傳遞思想，《 Püha Ja Õudne Lõhn》其實還有另一個重任：那便是為開發中的一款桌游，創造對應的世界觀。也正因為如此，庫維茲和整個ZA/UM團隊，用了整整10年時間去構思。

也就是在這個過程中，遊戲走進了庫維茲的視野。「必須要承認，如今的年輕人已經很難接受小說，尤其是。而相比之下，遊戲要比任何一個傳統載體，更能打動年輕人。」這位信仰馬克思和恩格斯的男人，由此下定決心招兵買馬，用遊戲來傳遞自己的思想。

終於，在千呼萬喚之中，《極樂迪斯科》登上了歷史舞台。

極樂迪斯科，2019年最值得購買的遊戲之一

得益於藝術的熏陶，以及早年間對於藝術的追求，《極樂迪斯科》作為庫維茲和ZA/UM打造的首款遊戲，展現出難以言表的獨特魅力。

誠然，《極樂迪斯科》是一款小團隊打磨的獨立遊戲，但還是那句俗話「簡約不等於簡單」。僅僅是那如同油畫一般的超現代主義畫面，以及超贊的「迪斯科」BGM曲風，就足以收穫絕大多數玩家的芳心。

當然，不論畫面還是音樂，都只能算得上是錦上添花。一款遊戲是否好玩，關鍵還在於劇情和玩法。在這兩點上，《極樂迪斯科》交上了一份近乎完美的答卷。所以即便是已經在TGA斬獲4項大獎的今天，依然有很多人難以相信，《極樂迪斯科》竟然會是這支團隊的首部作品。

前面我們曾提到，《極樂迪斯科》與庫維茲的那本小說，採用了同一個世界觀架構。在這個未來世界中，到處都充斥著腐敗和犯罪，看不到出路的年輕人，沉溺於蹦迪不能自拔……等等，為什麼是迪斯科？不要在意這些細節。

在遊戲中，玩家將扮演一名窮困潦倒的失意偵探，在港口城市雷瓦科城揭開埋藏在這片土地最黑暗角落的秘密。《極樂迪斯科》高達100萬個單詞的文本量，無疑為遊戲提供了極為可觀的劇情容量。值得一提的是，遊戲里的所有NPC並沒有所謂的「路人」，而是都有其存在的意義和價值。

不僅如此，用官方的說法是，在這個擁有極高自由度的開放世界中，當需要解決遊戲里的一些問題時，玩家甚至可以無所不用其極。

如果說目前為止《極樂迪斯科》還有什麼不足的話，那麼暫時沒有中文，可能是筆者唯一能想到了。不過也不用擔心，ZA/UM工作室近期已在官網承諾，將會在2020年初開啟中文測試。

總而言之，大膽融入了黑色偵探小說（筆者註：一種流行於上世紀50年代前後的犯罪題材作品）和傳統桌游RPG元素的《極樂迪斯科》，成為每一位玩家不容錯過的經典。也因此，在《極樂迪斯科》發售後不久，便成功登頂Steam暢銷榜，並在IGN和Metacritic分別取得9.6和91分的恐怖評價。

《極樂迪斯科》的種種成績，不禁讓我們有些期待，這個有著無限可能性的庫維茲和ZA/UM工作室，未來是否還會給遊戲行業，帶來更多的驚喜呢？
