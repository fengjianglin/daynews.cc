還沉迷《第五人格》中的躲貓貓？這款武俠遊戲推出了進階玩法

2019-12-23 2,329條評論 4,970次閱讀 1568人點贊

作為一個喜歡懷舊的玩家，自己對於躲貓貓遊戲似乎有著一種偏執感，尤其是在網易的《第五人格》推出之後，將我對躲貓貓遊戲的好感度更提上一個檔次。這是一款1V4不對稱競技遊戲，將玩家分為兩個陣營，也就是一個搜尋者四個躲藏者，說的通俗一點就是貓和老鼠的關係。

在驚悚的背景音樂和搜尋者未知的情況下，躲藏的膽戰心驚感被無限放大，整個地圖上，也許下一秒轉角就和搜尋者的情況也時有發生，險中求生的刺激感為這個遊戲增色不少。不過《第五人格》是競技遊戲，對局時間過長，自己在喜好上面更偏於輕鬆愉快的感覺，而我最近玩的《新笑傲江湖》手游就是這樣一款遊戲。我玩過不少武俠手游，不過武俠躲貓貓遊戲真的很少見，在這款遊戲中，我找到了童年時代和小夥伴躲貓貓的感覺。

躲貓貓是《新笑傲江湖》手游中的一個活動，每周六限時開啟，每局遊戲中，共有7名玩家參與，其中貓隊有2人，鼠隊有5人，即使是作為搜尋者的貓，也有小夥伴可以商討對策並肩尋找躲藏者。

剛開始貓隊玩家熟悉地形，鼠隊玩家要在30S的時間內改變自己的外觀找到合適的位置，而貓隊需要熟悉場景內的地形和道具的擺放，以便於比賽開始後更好的找到鼠隊玩家，當然這時候兩隊之間是看不見彼此的。

遊戲開始，貓隊玩家通過不斷釋放抓捕技能淘汰混入場景或躲藏在角落的鼠隊玩家，不過貓隊玩家初始時只有五滴血且看不到鼠隊玩家的角色名，只能自行辨認鼠隊玩家的偽裝，若是沒有攻擊到鼠隊玩家將消耗自身的血量，所以盡量不要唐突出手，要是沒有抓到鼠隊成員，自己就因血量耗完淘汰出局，就不能繼續遊戲了。

因為對攻擊有要求，所以貓隊在開局的30S的時候一定熟悉地形和物件的擺放。鼠隊當然有著特殊的加成，那就是移速會是常規的70%，並且在看到貓隊玩家的時候，會處於敵對狀態，對自己操作、反應和躲藏能力有信心的鼠隊玩家，可以通過嘲諷、移動觀戰的動作戲弄一下貓隊成員，不過要小心不要被抓到。

遊戲的最後一個階段為瘋狂抓捕模式，在第二階段鼠隊玩家未全滅的情況下開啟，貓隊玩家可以隨意進行攻擊而且不減血，鼠隊玩家移速回復常規態，這就是雙方玩家拼手速和幸運的時候。雖然是休閑玩法，但獎勵可不含糊，獲勝的玩家將獲得銅幣、隨機染色禮包、綁定元寶等獎勵。首勝玩家將獲得「躲貓貓首勝禮包」，有機會獲得稀有坐騎和稀有外觀獎勵。

如果想體驗更加輕鬆愉快的第五人格式玩法，不妨可以試一試這款手游，整局遊戲下來，玩家只需要花費五六分鐘，快速便捷更具有娛樂性。這款遊戲將在12月19日正式公測，目前已經開啟預下載，快和朋友去《新笑傲江湖》手游中體驗武俠世界的捉迷藏吧！
