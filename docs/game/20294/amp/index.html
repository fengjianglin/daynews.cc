《聖鬥士星矢：重生》遊戲測評：戰鬥吧，喚醒內心熱血回憶

天天要聞
2019-12-17

對於《聖鬥士星矢》這部經典的動漫，即使沒有看過，大家肯定也都聽說過。尤其是80後，這部動漫可以說是陪伴了他們整個青春期的成長。而手游《聖鬥士星矢：重生》的出世，更是喚醒了玩家們的青春記憶。今天，就讓我帶大家去手游《聖鬥士星矢：重生》重溫一下那些屬於年少時的美好時光。

遊戲評分：7.5/10

遊戲類型：即時動作RPG

完美還原動漫中經典角色

作為一款經典動漫的同名手游，《聖鬥士星矢：重生》可以說是完美還原了動漫中的經典角色，比如帥氣的天馬座星矢，威風凜凜的天鵝座冰河以及擁有強大戰鬥力的天龍座紫龍，這些我們在動漫中非常喜歡的戰士。不僅如此，動漫的主題音樂《天馬座的幻想》還作為了背景音樂出現在了登入界面，讓玩家在進入遊戲初期，便能感受到滿滿的熟悉感。即使在進入戰鬥中，我們依然可以聽到屬於動漫中的背景音樂，那叫一個原汁原味！

天馬座.星矢

除了人物角色的100%還原，遊戲還向玩家們展示了原版動漫中的精彩劇情。玩家不僅可以作為一名星鬥士參與戰鬥，還可以在遊戲中再次回味動漫《聖鬥士星矢》的各種劇情。

天龍座.紫龍

遊戲畫面及特效表現異常出色

雖然遊戲只是2D畫面呈現，但是帶給玩家們的視覺效果卻是出乎意料的好。不論是遊戲角色聖衣鎧甲上的金屬質感，又或者是微風下飄動的頭髮，像這種細節上的刻畫，《聖鬥士星矢：重生》做的真的是很到位了。

雅典娜：微風中飄動的長髮

更讓我驚嘆的還是遊戲在技能特的表現。當角色釋放必殺絕技時，半透明的光影特效，讓人似乎隔著手機屏幕都能感覺到能量的波動，給人以強烈的視覺衝擊。

漂亮的光影特效：戰鬥中釋放必殺技

卡牌遊戲的全新玩法

《聖鬥士星矢：重生》雖然只是一款卡牌遊戲，但又並非是單純的卡牌模式，而是以動作卡牌出現在玩家面前。在戰鬥前，星鬥士都是以卡牌的形式存在，但是在戰鬥中，他們則是以真人出現，與敵人展開戰鬥。雖然釋放技能只需要玩家點擊屏幕即可觸發，但是精彩的打鬥畫面，還是非常過癮，而且整體的技能特效上也屬於高度還原動漫細節。

帶領星鬥士們與敵人戰鬥

遊戲中的”冒險”系統是玩家們的主要戰場，而在這個戰場中又分為了主線和精英兩種副本模式。主線副本主要是還原了動漫中的精彩劇情，難度並不是很大。而精英副本，難度就要大一點了，而且開啟也是需要一定條件的，並不是想玩就可以隨時玩的。遊戲中每次通關一個副本都會得到相應的星星獎勵，並且在星星獎勵積累到一定數量之時，還能開啟寶箱，獎勵也是非常豐厚的。

開啟寶箱獎勵

技能雖厲害，排兵布陣依舊很重要

雖然遊戲中採用的是自動戰鬥模式，釋放技能也只需要簡單點擊屏幕觸發就可以了。但是作為一款卡牌遊戲， 在戰鬥前還是需要進行排兵布陣的。因為只有5名鬥士可以上陣，並且同類型的角色最多上陣3名，所以在出戰前，安排好各位星鬥士的排列位置顯得非常的重要。一個強大的陣容，則將擁有強大的戰鬥力，在戰場上才能所向披靡。

戰前布陣

如果要提高整體陣容的戰鬥力，那就需要玩家通過不斷戰鬥，在戰鬥中獲取金幣，並且升級角色的技能和裝備。玩家若是能夠集齊特定角色還能激活角色相互之間的屬性，使其生命以及攻擊力都得到很大的提升。

星矢成功進階

加分項：

遊戲完美還原經典動漫

畫面以及特效變現優秀

玩法簡單卻有深度

減分項：

操作簡單沒有挑戰性

遊戲缺少參與感

雖然遊戲僅僅只是改編成一個動作卡牌而非動作手游，讓人略感遺憾，但是遊戲對動漫的百分百還原卻是玩家在遊戲的同時，又能重溫青春的熱血。除此之外，遊戲在畫面、音效以及技能特效上的表現，也足夠讓玩家忽略掉那些微不足道的不足之處了。由於是卡牌遊戲，所以遊戲與市面上的其他遊戲差不多，在操作上並不是很複雜，只是簡單的點擊或者滑動技能按鈕即可。這對於那些對遊戲操作感有特別要求的玩家來說，《聖鬥士星矢：重生》手游就會顯得沒有什麼挑戰性，以及在遊戲體驗方面也會差上一些，所以這類玩家就要慎重選擇，避免踩雷！
