最近賽博朋克2077有點火，主要是因為前段時間公布的賽博朋克2077演示片，但玩家們對於這個遊戲世界中的設定和觀念仍然不甚了解。所以遊戲關卡設計師Patrick Mills在採訪時為玩家透露了更多關於遊戲中人物NPC價值觀，人格物化。周遭世界當然會因為你外貌的不同而展現出不同的態度。

這是一個系統和權力已經將大家的人格物化（把人當成沒有權利或感情的物體）的世界，真的已經把人類當成了物品。許多時候人們也會為了叛逆或者適應潮流而把自己當做成一個物品，因此當然了，人格物化會是遊戲的一大主題。」

Mill表示：「無論是你穿著的風格，還是你自定義自己角色的方式，我們都想要這個世界對它們進行回應。」並確認了NPC會根據玩家們不同的打扮風格來對待V，他表示之前的遊戲中也出現過這樣的做法，但它們的表現方式都受到了限制。

玩家表示：如果捏個黑人，遊戲難度可能會變成地獄模式，敵人單彈夾彈藥多百分之五十，精準度提高百分之七十，敵人數量多百分之五十，敵人呼叫支援CD減少百分之二十，並且可破壞建築數量減少百分之十。但是黑人有種族特性就是速度加百分之百，終極技能是「你這個種族主義者」，當釋放終極技能的時候敵人全部硬直三十秒，自帶嘻哈說唱BGM，並伴隨嚇尿褲子的特效，武器隨機落地，附近平民NPC也會像主角靠攏並圍成人牆擋子彈。

此前CDPR就曾經對遊戲的成人元素進行過解釋，在他們看來，在《賽博朋克2077》的世界中，由於人們對「改造人類身體」態度的不斷開放，人們已經可以隨意對自己的身體進行改造，肉體也已經不再那麼隱私。同時賽博朋克風格中，人性和機械軀體的糾纏也是非常重要的一部分。
