我的世界：種子“火線妹”是奇種，掠奪者哨塔在村莊中央

2019-12-20 221條評論 1,132次閱讀 223人點贊

最近非常火的一位美少女主播「火線妹」是火透半邊天的，她的粉絲現在直接達到700萬，而她個人非常鮮明的標誌就是頭戴耳朵耳機，做出可愛搞怪的表情來。既然這麼肥嘟嘟可愛的女孩子，為何我們不嘗試用她的名字當作種子來探索一番呢？

不過你可別說，這一探索就探索出了非常有趣的東西，首先出生點的第一個村莊，不但擁有要塞而且還有地獄堡壘。不僅如此，在出生點附近的村莊，這掠奪者哨塔竟然就在村莊的中間。這簡直就是「燒炸腚」，這種村莊的村民可以說是內憂外患，外面是成群的殭屍，而村莊內部就是掠奪者！

那我們就開始探索吧，我們在基岩版當中輸入種子：huoxianmei。

進入遊戲我們就看到在出生點不遠處有一個針葉林村莊，這個村莊的神奇之處就是雖然是針葉林村莊，但是村莊周圍沒有多少針葉林樹木。當然如果僅僅是這個原因並不能夠算上神奇，我們繼續探索。

對！你沒有看錯，這個村莊下面就是要塞，好久沒有遇到這種出生點就是要塞的村莊了。因為現在的村莊比較多，但是要塞還是沒有變化。

要塞裡面有許多非常優秀的附魔書，比如彈射保護IV、鋒利III、效率III、釣餌II，又比如寒冰行者II、火焰保護III、精準採集I、力量III、正在治癒I。

並且還有一個驚喜，就是這個出生的村莊下面竟然還是地獄堡壘。所以這個種子出生的村莊，不但有要塞還有地獄堡壘！

下面這個景點會更加神奇，為什麼呢？我們來到坐標為：46.98.522的村莊，我們就可以看到這個神奇的景象！在一個平原村莊上，竟然出現一個碩大的建築，這個建築就是掠奪者哨塔。

正是因為這個哨塔，讓村民遭了殃。我們可以看到哨塔上面還有一群殭屍，而掠奪者的出現直接就把鐵傀儡困在其中。即便鐵傀儡物力高強、身強體壯，但是無奈雙拳難敵四手，何況更是一群亡命之徒。此時正值夜晚，所以村莊外面也是怪物叢生，可憐的村民四處亂竄，好像鬼子進村的場景。

我們就讓他們自生自滅吧，於是繼續往前走，在坐標：-631.123.894的地方，又發現另外一個有特點的村莊，這個村莊依然是平原村莊，不過不一樣的是這個村莊的鐵匠鋪是被樹林包圍起來的，如果不是因為在空中飛行還真是很難發現。

靠近海邊位置：-1091.107.980的位置，就來到靠近海邊的平原村莊，這個村莊並沒有什麼特別之處，不過在鐵匠鋪裡面發現了寶箱。

裡面有一顆鑽石，我們全部收走。

海中坐標：-2954.74.1310的位置就發現了海底神廟，大家不要忘記來拿海綿寶寶。

從海面越過，我們發現景色宜人的雪村莊，坐標為：-3813.87.1845的位置。這個村莊位置是錯綜複雜，又因為被大雪覆蓋，十分好看，這時候想起那首歌：我的老家，就住在這個屯……

在雪村的旁邊有一個雪屋，坐標為：-4316.75.2343的位置。

這不是一個普通的雪村，而是有地牢的雪村，是不是很驚喜？

本次種子探索完畢了，想不到就在出生點附近有這麼元素存在，特別是一出生的村莊，不但有要塞還有地獄堡壘，而不遠處的掠奪者哨塔村莊也是相當驚喜。想必「火線妹」這三個字真的是有毒吧
