CFer晉級賽丨SV攜手WE率先晉級 AG.Red爆冷不敵對手遭淘汰

天天要聞
3 天 ago

經過一天的激戰，CFHD-CFer競技賽首個比賽日即將告一段落，接下來將要進行的是CFer競技賽的三強賽，根據抽籤，AG.Red戰隊對陣KYEG，Team WE迎戰CG，小組賽表現強勢的SV.CFHD輪空直接晉級到了最終的CFHD-CFer八強賽中。

Team WE在小組賽中就難逢敵手，從擁有眾多強隊的小組中突圍晉級六強，六進三狹路遭遇CG戰隊，這是戰隊雖然名不見經傳，但是能夠淘汰來自情久俱樂部的戰隊晉級到六強之中，實力可見一斑，對於CG戰隊來說，有了此前的勝利，他們對淘汰賽充滿信心。

CFer們可以說對於這支WE戰隊不夠了解，此前他們沒有暴露自己隊員的信息，據了解這支WE戰隊中有的選手曾效力於其他FPS類遊戲的職業戰隊，無論是槍法還是大局意識都是非常強的，也是隊伍中的核心和領袖，最終首場淘汰賽中，WE戰隊以5-0和5-2輕鬆取勝。

與此同時第二場淘汰賽也在同步進行中，由TGA金槍王神勇鑫領銜的AG.Red戰隊將迎戰KYEG戰隊，和CG戰隊一樣，這支KYEG也是名不見經傳的戰隊，都認為AG.Red可以以2-0的比分順利擊敗KYEG，在首張地圖中，AG.Red順利拿下比賽，一切都朝著「劇本」方向發展。

結果賽場風雲突變，在潛艇這張地圖中，開局後雙方就陷入了膠著之中，比分一直呈現出交替上升的趨勢，在前八回合中雙方戰成4-4，在決勝局中KYEG反其道而行之以5-4擊敗了AG.Red，從而將總比分扳成1-1，淘汰賽懸念再起。最終KYEG在第三張地圖中以5-1擊潰AG.Red，晉級到八強之中。

SV.CFHD等三支豪門戰隊會師到了六強，SV.CFHD輪空直接晉級，WE戰隊也以秋風掃落葉之勢晉級，可惜的是AG.Red未能跟上前者的腳步，爆冷輸掉了比賽，非常可惜。接下來，這三支戰隊將參加下一個比賽日的八強賽，期待他們的發揮。
