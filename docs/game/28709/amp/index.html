王者榮耀：夢淚比人氣躺贏，KPL最受歡迎選手排名第一

天天要聞
2019-12-22

王者榮耀：KPL最受歡迎的kpl選手投票，夢淚只打一場比賽可排第一

其實這個一點都不搞笑，從第一屆KPL比賽開啟至今，夢淚一直都是KPL聯賽中人氣最高的選手。本次秋季賽上，AG超玩會史無前例的奪冠之後，夢淚和老帥這AG雙子星的人氣就一直居高不下。

給夢淚投票的大多是喜歡王者榮耀和KPL的老玩家，這是一份熱愛，也是一份情懷。無冕之王AG超玩會奪冠，隊內的精神領袖夢淚，主力選手一諾和隊長老帥成為最受歡迎選手，一點都意外。

為什麼大家都說夢淚是自己在王者榮耀中的信仰

說道夢淚，喜歡王者榮耀和KPL的老玩家，都會津津樂道的談起他一戰封神的故事。甚至還有些小夥伴會說，夢淚是自己在王者榮耀中的信仰。當然囧王者也屬於是，見證夢淚韓信無兵線偷塔的老玩家了，曾經也是為了他的巔峰時期的操作而驚嘆。在喜歡上了夢淚之餘，也隨之粉上了當年的AG超玩會，想當年的AG超玩會那真是群星璀璨，夢淚，老帥，傑斯，Vv，流蘇，蘭息等人的實力，絕對是冠絕全聯盟。可惜由於時運不濟，第一屆KPL中輸給了曾經敗給自己的AS仙閣。從此以後，AG超玩會就一直背負著一個「無冕之王」的「美名」。

隨著連續兩屆倒在了總冠軍的門口，有著冠軍夢想的隊員們紛紛另謀他圖，隊長老帥轉會，傑斯等人退役。AG超玩會成績一落千丈，甚至創造了KPL（含次級聯賽）26連敗的記錄。一時間網友都在說「銀河戰艦隕落」。但就算AG超玩會跌落次級聯賽，同時期的冠軍戰隊AS仙閣泯滅於眾人時，夢淚依然在堅持自己的冠軍夢想：他既沒有轉會，也沒有退役，依然在賽場上奮鬥。也許這就叫堅持吧。

重返KPL的AG超玩會，成為了真.銀河戰艦，夢淚成最受歡迎選手

2019賽季，AG超玩會藉助收購BA黑鳳梨的契機重返KPL，月光教練將AG和BA兩支戰隊的選手混編，利用三三輪換的特殊戰術，將AG超玩會打造成一支真.銀河戰艦。秋季賽上有驚無險的奪冠，就是最好的證明。雖然在整個秋季賽上，夢淚只登場了一次，但夢淚登場的這一次常規賽的熱度，僅次於總決賽。畢竟作為老玩家來說，還能看到夢淚在場上打比賽，其實已經足夠了，這是一份情懷。

從KPL官方推出的「最受歡迎KPL選手」投票的結果，就可以看出玩家和粉絲們，還是傾向於有情懷，堅持不放棄的夢淚的。在這次官方的投票中，AG超玩會的精神領袖夢淚以12000多票高居榜首，超玩會頭號實力選手一諾以2800多票排名第二，而總決賽MVP老帥則屈居第三。

這次投票，再一次證明了在電競圈，空有情懷沒有實力是不行的，電競圈菜就是原罪，拳頭大才是王道。AG超玩會，夢淚，老帥總算是圓夢KPL總冠軍，可喜可賀，今天你給他們投票了沒？
