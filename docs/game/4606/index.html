《噬血代碼》難度頗高，對於一些玩家來說可能不是很容易，在經歷了頻繁失敗後不妨看看《噬血代碼》刺刀射擊流Build思路與玩法心得，希望能為玩家帶來一些幫助。

BOSS抗性在0.9-1配裝：

被動：刺刀專精，生存本能，速度特殺，蓄力加速，兩個冥血上限。

主動：射擊強化，強化重擊，超負荷，殞命之旅，閃現尖牙，力敏的激發，減蓄力，換血，傑克煉血。

思路：開場不帶生存本能，上兩個上限和刺刀與速度，主動buff上完，換血替換為射擊強化後，應該留有15血，再將4點的被動上限替換為生存本能，做操的同時吃個減蓄葯，這樣開場第一槍應該在3W到3.5W左右，血少的BOSS基本可以打出轉階段，血多的補一槍，在轉階過程中，將6點上限替換為減蓄力，三到四槍左右結束，抓好boss的攻擊空隙，5S左右應該就可以結束。

BOSS抗性在0.75-0.8，血量10W以下配裝：

被動：刺刀專精，生存本能，速度特殺，蓄力加速，兩個冥血上限。

主動：射擊強化，超負荷，殞命之旅，閃現尖牙，力敏的激發，狂擊，換血，傑克煉血，絕對獵殺。

思路：開場不帶生存本能，上兩個上限和刺刀與速度，主動buff上完，換血替換為射擊強化後，應該留有15血，再將4點的被動上限替換為生存本能，做操的同時吃個減蓄葯，這樣開場第一槍應該在4W到5W左右，BOSS基本可以打出轉階段，在轉階過程中，將6點上限替換為減蓄力，絕對冷卻也應該好了，一道二槍左右結束（8W血左右的boss）。

冰火雙人：

被動：刺刀專精，生存本能，速度特殺，蓄力加速，兩個冥血上限。

主動：射擊強化，強化重擊，超負荷，殞命之旅，閃現尖牙，力敏的激發，減蓄力，換血，傑克煉血。

思路：總體思路和打火貓，艾娃差不多，因為冰基本0防禦，8W血左右吧，吃減蓄葯開場三槍直接秒，沒上絕對獵殺和狂擊，火10W血，五到六發子彈應該就可以死，視頻手滑蹭了下烤地板，打了八發。

魚塘：

被動：刺刀專精，生存本能，速度特殺，蓄力加速，兩個冥血上限。

主動：射擊強化，超負荷，殞命之旅，閃現尖牙，力敏的激發，加速蓄力，換血，傑克煉血，絕對獵殺。

思路：抗性0.75，絕對獵殺加成非常不錯，開場BUFF上完，等魚塘轉身，吃減蓄葯，換上被動的減蓄，4S10W血直接清空，施法不給自己上減前搖的法師不是好法師。

接來下介紹一種打雜魚使用的日常流派。

核心艾娃：

被動：速殺，竊取生命，體質，刺刀。

主動：凈化之光，超幅常理，厄運，超負荷，射擊，獻祭之血，封閉記憶，超能之血。

思路：簡單明了，增幅情況下的回血有效時間是90S，蓄力射擊消耗血量大概在80點不到，擊殺雜魚的回血量在200點左右，CD時間到了就來一發自閉，永動機誕生。

傷害在一發1.5W左右，雜魚的血量應該不會超過3W吧。
