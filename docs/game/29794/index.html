魔壇最專業的科普，2v2種族和選手戰力排行來了

2019-12-24 3,349條評論 4,415次閱讀 658人點贊

技術顧問：Xiaokk

文章作者+整理：老粉

魔獸2v2，一直以來都是魔獸RTS不可或缺的一部分。很多新手，都是從2v2入門，然後再轉型1v1。並不是說2v2比1v1簡單的多，而是2v2更像一個社交遊戲。感謝有人帶我，感謝有人包容我的菜，並且在輸掉比賽之後，打出一些鼓勵性的語言。簡簡單單的一句話，飽含著魔獸玩家的溫暖。畢竟因為隊友太菜直接謾罵的玩家還是相對比較少的。很多人，就是因為有那麼一個玩家把自己帶入門，從而堅持了下來，成為魔獸爭霸的老玩家。

說到2v2，近期接踵而來的多個22賽事（虎牙WFDC、伊詩蘭頓杯、魔壇杯、黃金戰隊賽）把魔獸2v2的熱度提升了2個檔次。場面震撼、魔法絢麗、配合精彩、選手話題，甚至有很多觀眾斷言，這可比1v1有意思多了。

在2v2中有一個說法，那就是2個30級的玩家，如果搭配一個15級的指揮，可以輕鬆擊敗2個25級的玩家，可見指揮在2v2的重要性。LF就是輸一手指揮和配合。操作、戰術、配合、思路和意識，2v2所考量的因素也許比1v1更多。2v2的水平也和1v1的水平有直接關係。95%的1v1高手，只要投入時間到2v2的研究中，也能變得很強。例如海牛，以前用ORC的時候就是蘭州冠軍，1v1巔峰VS23級~

打22就是打配合

在2v2中，也有一個說法叫搶節奏。在玩家裡有一個說法叫做3級出山，意思就是首發DK到3就去壓一波、首發AM到3壓一波，2發SH到3壓一波。但是在22的高端局裡面，大家對這種說法是不care的。只要有合適的時機，你就應該去賺優勢，或者去壓制對手。

關於種族強弱

Top1最強配合：ORC+HUM

上榜原因：獸族最強肉盾+最強群控TC。人族最強輸出火槍+有可能的AOE（暴風雪）+移動大血瓶聖騎士。在只考慮打正面的情況下非常全面，幾近無敵。

Top2：ORC+UD

獸族的前排+亡靈的後排輸出，近遠搭配契合度極高。DK+Lich配合BM刺殺+SH的變羊，對手很有可能開戰就少一個英雄了。O+U，不考慮戰損，也不考慮死兵，能懟死你的英雄就輕取，懟不死，那就再說。

Top3：ORC+NE

老牌2v2組合，在VS時期是平台最最最主流的22組合。每10盤裡，就有5盤是ORC+NE的組合。以前的戰術基本就是BM+SH+DH+NAGA，獸族萬金油+小鹿。但是這個組合放到現在已經廢了，現在還算比較好用的戰術就像FoCuS+Lawliet這樣，開礦後找機會跑家。

Top4：UD+NE

UD+NE打的就是一波流。以前Lucifer搭配木、饅頭和三蛋的NE，也都是這麼打的。當然，最近演示的最多的就是TeD+hainiu了。這招壓制的時機非常關鍵，精靈亡命爆精靈也很關鍵，打一波對手沒成型，屬於基本沒有後期的戰術（對手英雄等級起來了基本就意味著GG），畢竟NE只有一個後期殘廢的KOG在。

Top5：UD+Hum

如果覺得UD和人族無法互奶而覺得這個搭配弱，那我只能說太膚淺了。U+H的戰術，米四已經開發的非常透徹了。能用的基本都用過了，至於成績，也有黃金聯賽亞軍背書。厲害么，也不算特別厲害，但是也不菜。可行性戰術在於人族包養UD打速攻三本一波，又或者人族法師+亡靈後排。

Top6：Hum+NE

這個搭配，最近還拿了幾次冠軍，都是由蛋塔斬獲的。除此以外，也沒看到業餘或者其他職業選手用這個組合打出成績的。NE本來就不適合當前排，人族沒有前排就很弱。所以蛋塔能用H+N奪冠，基本就是個人能力的體現了。

關於選手強弱

T1級別：

TH000、Infi、Lyn、120、殺精、Lawliet勉強算T1的車尾。

蛋塔：11和22最頂尖的選手，實力不需要懷疑，妥妥的T1級別。他們擅長單人換族，靠小操作交換，用最合適的Timing賺便宜，幾乎很少有破綻和失誤。

Lyn：同樣是頂尖實力，在2v2中思路清晰，操作幾乎不犯錯，可正面可換家，是一個非常可靠的搭檔。

120：在tara521這個ID出現之前，他就叫120，在2v2里是一個獸族選手。他的搭檔叫110，是個NE。長期混跡2v2，加上他頂尖的個人實力，故能躋身於T1，在22中他善於配合隊友秒殺。

殺精：1v1實力不那麼突出，但是在2v2里屬於傳說級別的選手。菜菜組合的主力，蛇精組合的主力。和他搭檔的人只要實力不差，就能躋身於2v2一線。他對4個種族16種對抗和所有2v2地圖全線精通，操作從來不拖後腿，堪稱2v2的戰場指揮官。

Lawliet：大家不要被黃金戰隊賽誤導了。羅胖最大的能力，是在於他22里擁有No1的大局觀。之前搭配FoCuS把蛋塔跑死，搭配Sok決賽差點擊敗了飛翔，單靠這點就無法忽視他的實力。正面操作不弱，大局觀爆炸，這就是羅胖為什麼能勉強躋身於T1的原因。

T2級別：

TeD、Fly、Fortitude、Moon、海牛、Check、Zhouxixi、Xiaokk、51、狼、Yumiko。

TeD+海牛：不得不說海牛和TeD組隊之後越來越菜了。當然不是說近特者短，而是他們練得少了。他們巔峰的時候有T1的，相信我。

Fly：飛飛在2v2中以輔助的角色居多，較少Carry。由於不夠穩，所以只能在T2。

Fortitude：浪漫的基本功非常紮實，在2v2中不會拖後腿。單對單不劣，團戰中表現也算出色。

Moon：其實Moon的2v2個人能力是有的，但卻因為他的戰術和搭檔導致他在這個位置。在MYM時期，他就和GB兩人跑來跑去，後來在Fox搭配Lyn還是以換家為主。勝率只能說非常一般。

Check：和TeD拿過黃金聯賽的22冠軍。他無論和獸族搭，還是和亡靈搭，與生俱來的大局觀和正面硬剛的操作都讓他在22里有著不錯的表現。

Zhouxixi：和Fly一起拿過黃金聯賽22冠軍，在2v2里屬於什麼戰術都能和你配合的存在。西爺在22里很少雜技，特別擅長和獸族搭配，遠程火力覆蓋非常到位。

Xiaokk：K神V5！Xiaokk在22里能打能指揮，有著豐富的作戰經驗。

51+狼：51是PK的主力之一（當時的絕對主力是Clowns），狼神不太記得了。巔峰的狼神，Solo能力堪比ZDR，在2v2里和51搭配巨強，WFDC冠軍的競爭者之一。

Yumiko：當時和蛋總組的蛋米組合斬獲多次WFDC的冠軍，和FT（Fly+TeD）有來有回。玉米是一個非常有想法的選手，和蛋總一起開發了首發血法輔助，MK爆火槍搭配法師，和四哥一起開發出純包養亡靈，MK+Lich首發秒英雄等戰術。所以為什麼這麼多人喜歡看玉米比賽？因為有意思啊~

T3級別：

So.in、FoCuS、Sok、WFZ、Chaemiko、老蛇、Lucifer、ReMinD（實力退化）、Xiaokai、飛艇、7k、顏色組合（ReD+Yellow）、ZDR、Pastime等等（沒寫到也不要生氣，想不起來這麼多了）

當然T3這個級別，還有很多，我就不一一列舉了。能進T3級別的選手，一般都是22杯賽八強級別的選手，偶爾能進四強，和T1的搭配也許能進決賽。例如Sok+Lawliet。

當然在2v2里，留給我的回憶還有很多。當年07年，也就是魔獸最巔峰的時候，VS有好幾個專門以2v2為主的戰隊，我還印象挺深的。例如[Z.G]戰隊，這就是2v2里的皇甫軍校，隊里人不多，QQ主力群也就幾十個吧，但是個個都是猛男（當時大戰隊還有測試群來著）。比較知名的就是：殺精、海牛、大飛、明月、ReD、狼神、阿嬌、小貓、AO、18等等，很多想不起來了。ZG最強的那波人，真的就跟頂級豪門WE有來有回。當時TeD+Like還找他們練的，最多64開，不吹B。

當然2v2戰隊還有[D.A]，還有我的D.A.，最好玩的就是擠破頭去1房開黑，然後在房間里互懟。不過那些都是當年的事情了，想想也很懷念。不知道有沒有當事人在評論里露個臉的？

關於2v2的介紹就這麼多了，不知道有沒有刷新大家的認知呢？如果覺得這篇文章寫得不錯的，麻煩點個贊並且分享一下，感謝大家啦~
