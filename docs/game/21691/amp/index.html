笑笑開發奧恩新打法，把演員心態打崩了，頂級理解超越Letme？

天天要聞
2019-12-18

笑笑是一名LOL退役選手，曾在圈內有著6級前最強輔助的稱號，退役後的笑笑創辦了德雲色，因其獨特的遊戲理解和「偷梗」絕技吸引了一大批粉絲，雖然笑笑的技術不怎麼樣，但Kid的加入卻讓德雲色實力大增，甚至S9之後的笑笑已經開始階段鑽石局了！

對線理解不到位

作為職業輔助選手，笑笑的上單功底確實差了點，雖然對線期沒有被壓太多刀，但打天使還有些難度，這主要跟對面的中野一直抓上路有關，畢竟奧恩是一個相對笨重的英雄，一旦交完技能，被抓幾乎沒有活路，甚至天使搭配千珏可以輕鬆越塔，這也給笑笑帶來了很大困擾，即便對線單殺天使，但沒有支援的情況下依舊很難壓制天使，甚至打出了QE幫助天使逃命的下飯操作。

先做三項

但笑笑在裝備的理解上卻別出心裁，完全沒有按照深淵那套防禦裝走，而是第一件選擇了三相之力。作為一件純輸出裝，三相之力的屬性非常完美，奧恩的被動也可以給三相之力帶來最大的增益，先做三項的性價比應該算最高的！但奧恩作為一個坦克型英雄，幾乎不會有人選擇第一件做三項！

隊友開送

不過，這場比賽中的笑笑非常不走運，遇到了職業送人頭的隊友（己方瑞茲）。當笑笑等人拿完大龍後，瑞茲開大將隊友全部帶走，而此時的輔助則因信號問題果斷開團，進場瞬間被對手秒掉，但這一波還可以理解，畢竟瑞茲想的可能是保存最強戰鬥力！

瑞茲送出10個人頭

但接下來瑞茲連續迷路，作為鑽一分段的選手，瑞茲打出1-10的數據確實很難理解，團戰四打五讓笑笑和Kid壓力倍增。但因為笑笑獨特的遊戲理解，他的奧恩竟然有輸出！當然，沒有瑞茲也無所謂，畢竟瑞茲的戰績也沒什麼輸出。

笑笑完美開團

最終，德雲色二人組沒有選擇在高地坐以待斃，而是由笑笑強勢開團，緊接著輔助女坦進場，奧恩不僅輸出高，坦度也很高，在一波完美的控制之後，Kid進場收割，戰局瞬間扭轉，德雲色二人組帶隊四打五拿下勝利！在飯飯君看來，笑笑的奧恩和Letme的奧恩完全是兩種風格，笑笑的奧恩因為先出三項的緣故，導致笑笑奧恩輸出極高，而Letme的奧恩基本全靠大招，輸出則是比較考驗操作，但因為笑笑在Rank需要自己Carry，而Letme擁有UZI這樣的頂級隊友，兩人之間的差距也就可以理解了。你認為笑笑對奧恩的理解超越Letme了嗎？期待你在下方評論區留言討論呦！
