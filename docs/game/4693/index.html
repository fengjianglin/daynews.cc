王者模擬戰：新版本寵兒堯天刺客流強勢崛起，打不過就加入

2019-12-05 0條評論 16次閱讀 0人點贊

摘要

大家好，我是小非。王者模擬戰從最新版本可以看出，法師流進入了末法時代，隨著武則天大招的削弱，諸如扶桑法刺、扶桑法、長安流陣容風采不再依舊，反傷刺甲的加入預示著坦克流即將崛起，但是缺乏強力輸出點，依舊是“爛分”陣容，唯有堯天刺客流才是最新版本寵兒，打不過就加入！

大家好，我是小非。王者模擬戰從最新版本可以看出，法師流進入了末法時代，隨著武則天大招的削弱，諸如扶桑法刺、扶桑法、長安流陣容風采不再依舊，反傷刺甲的加入預示著坦克流即將崛起，但是缺乏強力輸出點，依舊是「爛分」陣容，唯有堯天刺客流才是最新版本寵兒，打不過就加入！

堯天刺客流陣容棋子：

明世隱、斐擒虎，公孫離、奕星、司馬懿、百里玄策、上官婉兒、李白、蔡文姬，組成4堯5刺2輔2魏。

堯天刺客流陣容站位：

前排弈星和明世隱，中排蔡文姬，全部刺客站後排。

堯天刺客流裝備解析：

堯天刺客流基本以物理輸出裝備為主，正所謂天下武功唯快不破，憑藉攻速加成下，再以一力降十會姿態去碾壓對手陣容。畢竟堯天刺客流陣容C位主要以司馬懿、斐擒虎，公孫離為主，所以需要的法術裝備比較少，基本以物理輸出裝備為主。

物理裝備：名刀、泣血之刃、無盡戰刃、影刃、制裁之刃、冰霜長矛

法術裝備：秘法之靴、輝月、迴響之杖

斐擒虎，公孫離這兩個棋子是最被低估的棋子，也是升3成型率最高的棋子，在堯天刺客流陣容體系，物理輸出裝備都優先給這兩個棋子，斐擒虎技能輸出為法術傷害，並且技能釋放後能獲得免傷護盾加持，之後有技能攻速加成，再配合4堯天羈絆攻速加成，成型後的輸出絕對不比百里玄策要差。

公孫離技能疊加印記之後有巨額的法術傷害加成，在堯天羈絆體系下，技能被動非常容易觸發，再裝備影刃、制裁之刃、冰霜長矛配合公孫離身法技能，絕對是風箏流傑出代表棋子。

司馬懿是刺客棋子中少有的aoe控制型棋子，技能特點就是範圍內沉默，自開測以來少有沒有被削的刺客棋子之一，是個人認為王者模擬戰中最被忽視的刺客棋子，配合秘法之靴、輝月、迴響之杖就對讓對手成為噩夢的存在。秘法之靴、輝月加快司馬懿的回藍速度和生存能力，迴響之杖則配合司馬懿aoe技能特點增加範圍內傷害，所以裝備成型後的司馬懿是絕對是堯天刺客流前中期的輸出核心之一。

堯天刺客流陣容運營思路：

前期快速上7，8人口，然後養百里玄策、司馬懿，如果對面很肉，就養公孫離。除了純稷下坦克流且三星多不好打以外，基本克很多陣容。一般刷到弈星、明世隱天賦技能後，前排都會針對性放倆弈星加強控制能力，之後就看司馬懿和百里玄策等刺客棋子滿天飛。如果有人卡牌玄策，就養李白或婉兒，前期靠司馬懿和中後期靠老虎和公孫離。

總結：

隨著當前版本的刺客流和坦克流強勢回歸，再有反傷刺甲這件裝備的加入，長安坦射流前排依舊很難打得動，但是隨著陣容核心C位射手棋子的削弱，堯天刺客流陣容絕對是你上分的最佳選擇，4堯天羈絆的攻速加成讓5刺客羈絆得到了最大強度的強化，再配合雙輔助羈絆加持，堯天刺客流比魔種流輸出更上一層樓。但是堯天刺客流核心Key卡比較多，前中期需要非常小心的運營快速升級到7、8人口。當你取得了人口優勢時，再排出雙奕星前排陣容，這就意味著你這局一定吃雞。
