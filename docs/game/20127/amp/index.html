國產動作RPG《破壞神傳說》將以手機遊戲 回歸首度公開最新情報

天天要聞
2019-12-18

《破壞神傳說》為大宇資訊1991年出品的動作角色扮演遊戲，如今睽違28年官方宣布續作將再度以手機遊戲之姿回歸。此次故事背景延續當初世界觀，劇情更加耐人尋味，並增加多元玩法提升遊戲豐富度。此外，官方今（13）日同步揭露遊戲第一手情報，讓玩家搶先一睹角色樣貌及遊戲內容。

  • 《破壞神傳說》首度公開遊戲畫面

挖掘災難背後的秘密真相撲朔迷離　

　　上古時期，大地曾經生氣蓬勃，機械文明高度發展。某日魔力機械卻突然背叛人類並帶來毀滅性的災難，倖存的人們在此艱難的時刻建立了新的堡壘- 奧斯城。數百年後，學者發現那場災難的秘密就藏在遺迹中，而奧斯城領導者卻在探查遺迹時意外失蹤，意外激化了奧斯城與敵方勢力，雙方戰火一觸即發。

  • 豐富的劇情帶領玩家深入探索

近戰、遠攻特色鮮明的三位主人翁

　　遊戲一開始玩家將化身為身材火辣的女主角- 莫妮卡，是個身負重責大任的皇家圖書館研究員，身手矯健的她擅長使用短槍。之後玩家將與聰明小天才- 莉蒂亞一同作伴，模樣看似嬌小可愛，但她的武器可是火力威猛的重炮。隨著劇情發展還會加入一位戰友- 雷歐，是個正義有為的年輕軍官，揮舞大劍的樣子瀟洒帥氣。三位主人翁的個性和戰力各有所長，玩家可依自己喜好操控不同角色，體驗截然不同的戰鬥快感。

　　

  • 遊戲內角色皆為真人比例

遼闊地圖等待玩家探索採集尋寶樂趣多

　　遊戲中有佔地廣大的平原，在這裡可以採集製作成武器的必要材料，選擇自動採集就能更有效率地搜集各種材料。此外，運氣好會遇到限時的寶箱可以撿取寶物，還有各式隨機任務可以挑戰，多元玩法想必能帶給玩家豐富的遊戲體驗。

  • 超大地圖讓玩家盡情體驗

　　大宇資經典續作《破壞神傳說》預計將於近日推出，屆時玩家就能重溫當年奮戰時光，如欲掌握第一手消息請關注官方網站。
