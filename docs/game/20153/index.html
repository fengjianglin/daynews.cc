黃金大獎賽：C組比賽4AM東部第一進入F組，戰隊淘汰數居前列

2019-12-18 57條評論 1,015次閱讀 95人點贊

保持在第一的位置，成功進入後天比賽前八組成的小組，E組的戰隊陣容當中，天霸也憑藉最後一局13殺吃雞直接上升了8名，進入C組的第五名，同樣參加E組的比賽。

註：本次賽制採取瑞士輪，32支戰隊第一輪按抽籤分AB兩組，AB組打完前八成為C組，後八成為D組，CD組打完前八成為E組，後八則是F組，共計三輪比賽。

4AM東部第一進入F組：

而4AM在今天的比賽中，表現不可謂不好，戰隊淘汰數達到了30個，相比今天的第一名EDG也僅僅只差了一個淘汰分，比第二名ARK戰隊的33個淘汰數也只相差了三個淘汰分，在東部榜單中也是遙遙領先，比SMG多出了10個，但是排名分拿的實在是太少了，所以拿了個東部第一，進入F組的陣容當中。

不過今天的表現確實穩定，每局至少拿五分，今天的得分分別為：5、6、7、5、5、9，但是也同樣的，絕不超過10分，說實話我都懷疑他們今天是不是戰術性的在演，想去後八組多拿分了（狗頭狗頭狗頭）！！

小鹿名列戰神榜：

儘管在東部第一的情況下，小鹿還是以10個淘汰的分數名列今日戰神榜第五名，也是東部戰隊唯一一個上戰神榜的選手啦，這個比倒心的選手你愛了嗎？？

4AM總分第十，積分80：

而在目前的總分榜單上，4AM暫時排名第10，積分80，按現在的情況來看，晉級應該是沒什麼問題吧，4AM本身實力強，狀態也穩定，還進了後八組，正是拿分的機會，希望4AM能把F組變成「死亡之組」吧，相對於其他戰隊來說。

D組比賽陣容：

從總分上來看，VC暫時排名靠後，不過畢竟還只打了一場比賽，所以明天還是可以期待D組各個戰隊的表現哦，比如結束了一波人青春的run哥微博戰隊，外星人龍四哥IFTY戰隊等等…
