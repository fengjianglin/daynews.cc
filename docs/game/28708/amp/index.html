魔獸爭霸3重製版模型第15期：聽說瑪維的副官是個高危職業？

天天要聞
2019-12-22

各位玩家晚上好，本肥喵今日繼續和大家一起分享遊戲熱潮！

近期文章起已經進入到新的模型分享板塊，即遊戲內的模型正視圖，只為帶來簡單直觀的效果。

上期已經開始進入暗夜精靈，暗夜精靈可以分為三部分，第一部分是以泰蘭德為主的艾露恩姐妹會及哨兵部隊，第二部分以瑪法里奧為主的德魯伊及塞納里奧議會，第三部分為荒野眾神生物，本期主要介紹瑪維為主的守望者組織。

一、瑪維-影歌

瑪維，暗夜精靈典獄長，守望者部隊的創建者及領袖，上古之戰後關押伊利丹為主，之後以抓捕伊利丹為己任出現在冰封王座戰役中。瑪維對責任的堅持無人可以動搖，對她而言，與邪惡鬥爭的一切行為都是正義的，儘管這讓她樹敵眾多，被認為是一種極端，但還是掩蓋不了她的魅力。

瑪維重製版模型

二、守望者（單位）

戰役單位，守望者是瑪維創建為抓捕惡魔而成的組織，其中大部分成員原先都是女祭司，在經歷過上古之戰之後，對惡魔的憎恨和獵殺成為瑪維建立該組織的初衷，旨在抓捕任何威脅到艾澤拉斯的危險生物。

守望者重製版模型

三、守望者（英雄）

對戰英雄，簡稱WD，俗稱「女王」，是暗夜中靈活性極高的英雄單位。根據背景故事，守望者（英雄單位）應該是守望者組織的高層人員，順帶一提，圖中守望者的名字塞拉-月衛，正是瑪維最信任的副官之一，但不同的是，WOW中的塞拉偏好使用的是雙稜錐，而不是傳統的暗影月刃。同樣的，守望者英雄名字中還出現過麥琳-刀翼，她是守望者的軍需官。

守望者英雄重製版模型

四、復仇天神、復仇之魂

提到守望者英雄就不能不說她的六級大招，復仇天神，以及通過屍體召喚的復仇之魂，俗稱「葫蘆娃」，不過數量上限為六個。重製版的復仇天神模型像是守望者卸下武器後的樣子，整體色調顯得夢幻，在黑夜中想必更為虛幻。

復仇天神重製版模型

復仇之魂重製版模型

五、娜薩

戰役單位，瑪維的副官，娜薩是瑪維最信任的手下，自從瑪維弟弟加洛德失蹤後，瑪維一直將娜薩當做親姐妹一般對待。但追擊伊利丹過程中，在薩格拉斯之墓娜薩為救瑪維被神器力量永遠地留在了冰冷的墓穴，雖說生死未卜，但以薩格拉斯之眼的力量而言，摧毀島嶼都不是問題。正是由於娜薩的死亡，使得瑪維對伊利丹的追捕變成復仇追殺，悲劇的小人物。
