Ps4新游《壁中精靈》發售，國外遊戲製作商圓夢神筆馬良

天天要聞
2 天 ago

前言

神筆馬良在中國家喻戶曉，馬良用神筆幫助村民，與惡霸鬥爭的故事點燃了舊社會苦難大眾對美好生活的嚮往，故事中神筆可以將繪畫的內容變為實物，讀過這個故事的人都曾擁有這樣一支神筆，而今傳說變為現實。國外遊戲廠商Pixelopus 發布的全新冒險遊戲《壁中精靈》在2019年10月18日正式上線，經歷了長達六個月的跳票，發售之日依然熱度不減。

劇情

遊戲主角艾希是一名經常被同伴欺負的男孩，他喜歡繪畫，素描本里有很多自己畫的小精靈，露娜 是他最喜歡的作品。 遊戲開始，艾希的畫冊被一群壞孩子（惡霸 ）撕毀，並被推到電纜車裡。在尋找散落畫冊的時候，他遇到自己畫的牆壁小精靈露娜，露娜給了艾希一支神筆並告訴他小鎮正在被邪惡勢力腐蝕，心魔已經滲入人心，而能拯救小鎮的唯一方法就是通過注入魔力的畫筆驅除黑暗。

畫冊被毀

玩法

壁中精靈一共分六章， 前面主要是探索繪畫和創造小精靈，隨著失落畫冊逐漸找回，繪畫元素和創作的小精靈數量也會增加。小精靈有多種屬性，例如被箱子擋住的大門，就需要火系小精靈摧毀障礙才能進入房間。遊戲中會遇到強大的黑暗腐蝕就需要超級顏料才能被凈化，超級顏料來自小精靈的喜悅度，滿足這群小可愛的要求便可以增加，畫一個蘋果或者是一棵樹，或者是玩遊戲互動都能增加好感度。當畫一個蘋果一口被吃掉的情形讓人忍俊不禁，我都是直接畫一大顆蘋果樹招呼小可愛們過來吃，過程非常欣慰愉悅。

獲得神筆

遊戲探索過程中會遇到撕毀艾希畫冊的惡霸，其所在地點會有稀有畫冊和任務，注意不要被他們抓到，如果被抓除了會被欺負之外，還會丟失畫筆，躲避惡霸的方法就是聲東擊西，吸引他們去別處然後迅速做完任務逃離。

收集書頁

本作同樣有戰鬥情節， 黑暗章節，畫筆被壞孩子毀壞， 導致被控制的黑暗勢力擴散，小精靈們變成邪惡怪物來到現實，那群壞孩子也被怪物吞噬。艾希找到露娜尋求幫助，但此時的露娜已經非常虛弱，從對話中得知露娜一族屬於光明系精靈，和黑暗精靈對立，此消彼長存在。露娜告訴艾希，拯救小鎮和同伴們的重任只能靠自己了，最後露娜犧牲自己用最後的魔法復原了神筆。 本章艾希開啟溜冰模式，依靠畫筆的魔力，可以在地面快速滑行追捕腐化的精靈，根據精靈的特性，需要用 風、電、火 三種技能分別應對，風系擊退減速、電系擊暈、火系範圍傷害，超級顏料可以釋放大型Aoe技能。追捕怪物的情節是本作最激動人心的時刻，有一種極地賽車的感覺。期間被拯救的壞孩子開始幫助艾希，整個過程既緊張刺激又溫暖團結，黑暗和光明之戰拉開序幕。

牆壁塗鴉

最後的boss戰難度大幅提升，除了不斷有小怪刷新之外，還有飛天、護盾和強大的傷害，就需要不同的法術應對，其中護盾有三種需要風，火，電三種法術擊碎。期間開啟溜冰模式躲避傷害，用超級顏料擊殺小怪，整場戰鬥熟練的話可能持續10分鐘左右。如果中途死亡需要重新開始戰鬥。 在凈化了小鎮之後，壞孩子和艾希成了朋友。最後關卡露娜復活，指引艾希來到了自己的密室，在這裡艾希可以隨意創作和擦除，大師級畫作任由發揮，完全屬於自己的一片畫室，至此整個劇情結束。

追捕邪惡

總結

總體來說，這是一款畫面非常精美的遊戲，雖然不是3A大作，但製作非常用心，IGN為《壁中精靈》打出了8分好評，通關僅約六個小時，但體驗卻非常甜蜜。遊戲本體突出了愛與包容的主題，還可以培養良好的藝術美感。製作方透露靈感源自神筆馬良，遊戲中塗鴉元素驚艷又美麗，玩家可以自由發揮，根據自己心情輕而易舉就可以描繪一副巨作。可能正是因為是一款治癒系遊戲，戰鬥模式在通關後就不能使用了，多少有些可惜。玩多了槍林彈雨的動作遊戲試試這款也是不錯的選擇，目前《壁中精靈》為ps4獨佔，並支持VR模式，開發商表示暫無DLC計劃。
