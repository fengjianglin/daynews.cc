換心靈戰警的玩家，你們上當了！請仔細看第4個活動的小字

2019-12-04 0條評論 18次閱讀 0人點贊

摘要

大家好，我是小魚乾，今天王者榮耀正式服進行了一波更新，除了常規的模擬戰更新外，還有兩款皮膚受到了玩家們的關注，一款是蔡文姬的“繁星吟遊”皮膚，這是首款由玩家自主設計的皮膚，只需要6元即可獲得。第二款就是亞瑟的“心靈戰警”皮膚了，在本次更新中，這款皮膚首次上架了碎片商店。

大家好，我是小魚乾，今天王者榮耀正式服進行了一波更新，除了常規的模擬戰更新外，還有兩款皮膚受到了玩家們的關注，一款是蔡文姬的「繁星吟遊」皮膚，這是首款由玩家自主設計的皮膚，只需要6元即可獲得。

第二款就是亞瑟的「心靈戰警」皮膚了，在本次更新中，這款皮膚首次上架了碎片商店。心靈戰警的特效不錯，加上這個英雄也足夠熱門，心靈戰警一上架，玩家們都紛紛第一時間進行了兌換，其中也包括小魚乾，自己也是毫不猶豫地兌換了心靈戰警。

兌換完之後，小魚乾還想查看一下有什麼新活動，但看到活動海報，才發現自己兌換心靈戰警，似乎是「上當了」！在今天更新完晚些時候，遊戲中出現了一張「王者創意互動周」的海報。

這張海報公布了「創意互動周」的7個活動，第2個、第3個和第7個活動，在更新公告中都有提到，包括心靈戰警上架碎片商店、繁星吟遊限時秒殺、全直售傳說皮膚限免，而第1個活動，目前也已經上線了，可以了解自己的皮膚數據。

但第4個活動，在更新公告中是完全沒有提到的，活動內容就是「亞瑟皮膚設計大賽」，開始時間是12月5日。看來，官方已經明確敲定了下一款玩家自主設計的「源夢」皮膚了，就是亞瑟。

官方在更新公告中已經明確表示，後續會為「源夢」系列皮膚做全新的展示動畫、技能特效和台詞，最終會升級成史詩品質，「源夢」系列皮膚均統一為史詩品質，並將一直以6元秒殺的形式推出。

這也就是說，下一款「源夢」系列皮膚，也就是亞瑟的皮膚，也將會是以6元秒殺的價格上架，這也是為什麼本次更新，官方會將「心靈戰警」上架碎片商店的原因了，為的就是在亞瑟「源夢」皮膚上架之前，把心靈戰警多賣出去一些。

所以，不少第一時間兌換心靈戰警的玩家，都直呼「上當了」，因為心靈戰警的特效本身不算很出眾，只是「充能復活」的特效亮眼一些，加上有不少玩家反映，心靈戰警的手感屬實一般般，如果不是看膩了死亡騎士，還是不怎麼想換的。

而官方更新前並沒有透露下一款「源夢」系列皮膚就是亞瑟，在更新後一段時間才放出活動海報，第一時間換了還真有些後悔。目前這張海報已經從遊戲中下線了，估計是官方也怕玩家們因為這個原因不去兌換心靈戰警，先暫時下線，等設計大賽開始再放出來。

在更新之後，諸葛亮的「黃金分割率」皮膚中，黑板背面也出現了新的彩蛋提示，本次應該是有關皮膚的，黑板上寫著「哪幾個英雄的新皮膚如此神秘？」，還畫著一個填字遊戲，小魚乾只看出有「五虎將」的皮膚，而後面的內容，則需要官方的揭秘了。

大家都兌換了亞瑟的心靈戰警皮膚嗎？如果兌換了其實也不需要太后悔，因為這個設計大賽還在預告階段，從開賽到玩家設計，再到選出獲勝作品，還需要官方轉化設計，最終才能在遊戲中上線，還是需要一長段時間，所以心靈戰警還是能用上個一年半載的，我們還是期待玩家們設計出比較精美的亞瑟皮膚吧！
