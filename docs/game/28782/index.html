"你C隊友崩，你混隊友C"，鐵律告訴你為啥有人老混

2019-12-21 820條評論 2,498次閱讀 84人點贊

當英雄聯盟的玩家談及自己最不喜歡的隊友或者對手是哪樣的，首先秒選怪和掛機怪算是頭牌，在第二位上佔據的，大概就是那種人家明明也有在認真對待遊戲，但你覺得他不夠尊重這款遊戲的混子玩家了，各種混分巨獸、混子泰坦充斥在峽谷的RANK對局之中，這些英雄玩起來一點也不快樂，甚至從頭到尾都保持著一種彷彿在掛機的感覺，團戰只需要出一個大招，死不死人無所謂主要是能夠給出控制，然後站原地不動好像也無傷大雅，這就是混子玩家們的日常，而大家都是花時間去遊戲中體驗快樂，為啥還有那麼多人選出這種”有手就行”的打法來上分呢？

其實流傳在上分界的一大定律就是，你C隊友會崩的，倒是你混的話隊友可以很C，不知道這是啥情況，估計在較為普通的對局中隊友們要是玩容錯率低的能C英雄，在得知邊路上的隊友已然殺穿了對面局勢大好，自己在這場遊戲中的鏡頭就會少之又少了，豈能眼睜睜看著自己操作的英雄毫無戰功就拿下勝利，必須有點聲音，之後反而自己的這一邊雖然本是優勢，但因為心智焦急，失誤不斷還老是愛上頭，所以成就除了定律的前半段，往往能C的時候隊友就莫名其妙的開送了。

全線開花的情況也有，但一般這種情況只有一種可能的前提，那就是打野是一個不折不扣的野爹，他能任意掌控整局遊戲的節奏，不然在很吃心態的普通對局甚至是”病友局”中，總有人會在大好局勢之下犯罪，要說不給對面機會是不可能的，拆個高地就行別拆家這種操作乃貨真價實的”享受遊戲”，無腦拆家不給對面機會的反而成了”分奴”。

確實，過於分奴是失去了部分的遊戲樂趣，但是每個人有自己的遊戲理解和思路，大家都討厭那些明明操作不行各種破綻的石頭人，卻總是能逍遙自在的擱那混，硬生生懟過來的一個大招還比自己累半死操作半天的連招來得有價值。但真要說的話，比賽會輸是要怪罪對面的石頭人不夠有水準，還是自家的泰坦太混了？人家混歸混，但是做到了自己份內的事，在對線期間並沒有養出爹來，在團戰的時候能抗能開團，是自己C不動了還是非要別人C你呢？

英雄聯盟這款遊戲吸引人的地方就在於每位玩家都能在峽谷之中找到屬於自己的答案，我見過從開始到現在幾年過去了，只玩煉金的”放屁人”專業戶，見過在如今ADC成弟中弟，還堅持選出常規射手的信仰玩家，有些人上分，有些人快樂，不是說坑不坑，大家都是認真對待遊戲，除非人家代練上來的與段位不符成為眾矢之的，不然的話無論是被打穿了，還是混了一整場都是正常的，理解就行啦。
