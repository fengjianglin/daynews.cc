夢幻西遊手游：69平民普陀身攜千萬金幣，還有兩件簡易裝

2019-12-24 994條評論 3,623次閱讀 773人點贊

在《夢幻西遊》手游里，很多平民玩家都希望自己能早日達成千萬金幣成就，但想達成這個成就卻有不小難度。比如合寵煉妖或者打造裝備時，就會在不經意間消耗不少金幣，如果沒出極品，那麼這波就賺不到錢。這次一位69級平民普陀曬出自己賬號的一些基本情況，她表示自己玩這個普陀號已經很久了。不知道目前處於精銳的什麼水平，現在我們就來幫她鑒定一番吧。

先來看她的人物屬性，氣血4718，物防1356，速度928。這號的物防算是不低了，普陀屬於輔助門派，不能在PK中被輕易點殺，否則隊伍會有崩盤危機。這位普陀的屬性在精銳組算是中上的水平吧，不算太好也不算太差。

再來看下她的法寶，她的法寶比較齊全，大多數法寶都被修鍊到了滿級。比如她的九黎戰鼓，是10級太陽屬性，即使她不佩戴九黎戰鼓，戰鬥中也可以發揮基礎的50%效果。可見九黎戰鼓的太陽屬性非常實用，可以讓她把空餘的格子佩戴其他法寶。

然後再來看下她的裝備，她的髮釵是件60級的玉清訣髮釵，鑲嵌了兩顆7級月亮石。玉清訣可凈化己方全體異常狀態，如果有憤怒腰帶的話，只需要100點憤怒就可以施放。有時候她用玉清訣群體解封，甚至比晶清訣還實用。

她的項鏈是條60級神農羅漢金鐘項鏈，神農加羅漢金鐘是60級項鏈里最好的雙藍字。或許60級項鏈的基礎屬性不如70級項鏈，但有這兩個雙藍字在，她佩戴這條60級項鏈的效果完全不輸70級簡易項鏈。

身為一個輔助在追求特技的情況下，她還需要考慮自身屬性強度。她的衣服是件70級體耐雙加簡易衣，衣服鑲嵌了兩顆9級月亮石。衣服的基礎氣血和基礎物防還有很大的熔煉空間，目前她只熔煉了2點物防。

她武器是件70級體耐雙加簡易飄帶，基礎治療強度161，寶石鑲嵌了兩顆9級崑崙玉，能有效提升人物治療強度。武器的治療強度中規中矩，依然有較大的熔煉空間。只是她目前捨不得花費自己身上的金幣，因為她好不容易才剛剛攢夠一千萬金幣。在身上有一千萬金幣的情況下，她的裝備里有兩件70級簡易，兩件不錯的60級特技裝備。對比同級69級普陀，她算是中等偏上的那一類了，好歹17217的人物評分擺在那。

她的寵物里有一隻不錯的7技能全紅隱攻吸血鬼，這隻寵物各項資質優秀，成長1.219。寵物的高級內丹用的是捨身擊，吸血鬼的專屬寵物內丹還沒有打滿。以上就是她這號的整體情況，她千萬金幣在囊中，簡易裝備、特技裝備以及全紅寵物統統有，這樣的69平民普陀你佩服嗎？
