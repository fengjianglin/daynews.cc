DOTA2邀請賽：末日克一切，LGD爆冷0-2不敵Navi

天天要聞
2019-12-18

ONE Esports世界職業邀請賽，剛剛斬獲WESG中國區決賽冠軍的PSG.LGD迎來了Navi。

第一局

天輝方 PSG.LGD：中單孽主，大哥小小，三號位亞巴頓，雙輔助劇毒和陳

夜魘方 Navi：中單TA，大哥螞蟻，三號位滾滾，雙輔助末日和薩爾

開局後，薩爾在上路率先送出一血，人頭被查理斯操刀的亞巴頓收下。隨後儘管Ame在下路被末日沖塔帶走，但PSG.LGD五分鐘領先三千經濟，掌握了在對線期的優勢。8分鐘，查理斯的亞巴頓在上路陣亡，但螞蟻在線上卻因為有劇毒的存在並不好受。10分鐘，Navi雙輔助入侵天輝野區，找到打野的小小後末日直接交大，隨後儘管亞巴頓趕到但還是沒能救下小小，滾滾前來收掉小小人頭，好在薩爾被亞巴頓生生點死。

14分鐘，TA帶隊偷掉不朽盾，姍姍來遲的PSG.LGD仍然選擇開團，在陳陣亡的情況下，小小瘋狂輸出打掉TA兩條命。16分鐘，孽主小小雙核帶隊一路平推，站上了夜魘上路高地。Navi末日先手動孽主給上大招，隨後孽主買活飛回戰場，PSG.LGD由此將敵方高地順利拿下。時間來到25分鐘，儘管孽主在下路遭到了Navi多人擊殺，末日交大，但在肉山處小小帶隊拿下不朽盾，PSG.LGD並不算虧。三分鐘後，小小孽主帶隊出擊沖敵方聖壇，雙輔助率先被擊殺後，末日交出大招鎖定孽主，隨後小小和亞巴頓在正面戰場硬剛，最終雙拳難敵四手，小小盾掉，Navi將PSG.LGD打成團滅，逆轉局勢。

33分鐘，雙方再度開團，劇毒被TA配合薩爾大招將其秒掉，隨後孽主再吃末日大招，陣亡後交出買活。此時PSG.LGD陣容笨拙打不到人的問題暴露無遺，最終僅僅留下一個輔助末日。36分鐘，夜魘入侵天輝野區，孽主又被末日盯上，TA跟上輸出將孽主人頭收入囊中。隨後正面戰場陷入了一邊倒，雙輔助先行陣亡，隨後亞巴頓為了救小小送出人頭。

39分鐘，Navi的重拳出擊再度對PSG.LGD造成了致命傷，陳、孽主、亞巴頓陣亡後小小一個人難擋Navi五人，也無奈倒地。隨後夜魘選擇推進高地，劇毒守塔陣亡交出買活，但還是沒能阻擋Navi破掉天輝兩路高地。隨著最後一波團戰小小和孽主的倒地，PSG.LGD翻盤無望，在Navi再度站上己方高地後打出GG。

第二局

天輝方 PSG.LGD：中單帕克，大哥TB，三號位蝙蝠，雙輔助海民和陳

夜魘方 Navi：中單TA，大哥螞蟻，三號位鳳凰，雙輔助末日和薩爾

3分鐘，陳在下路遭到圍殺送出一血，隨後TB和蝙蝠也分別遭到了Navi的擊殺，PSG.LGD以1-4的人頭比血崩開局。5分鐘，海民配合帕克動手TA，但此時鳳凰與末日趕到先是擊殺了正面戰場的海民，隨後將帕克連同趕到支援的陳也帶走，末日收下人頭。隨著海民在上路陣亡，螞蟻達成大殺特殺成績，而這才僅7分鐘。

10分鐘，PSG.LGD展開反擊，末日和薩爾雙輔助應聲倒地，但Navi三核的經濟仍然領跑全場。12分鐘，PSG.LGD選擇主動出擊找機會，但在末日交大、帕克吃下Doom後，PSG.LGD瞬間陷入敗局，蝙蝠和雙輔助為了掩護大哥紛紛交代在了戰場。三分鐘後，TB在上路遭到圍攻，薩爾瞥視留住TB後螞蟻跟上輸出完成擊殺。17分鐘，鳳凰帶雙輔助發起攻勢，隨後TB變身一套輸出將鳳凰擊殺，Navi雙輔助隨之敗退。

19分鐘，雙方在肉山附近爆發團戰，在雙輔助陣亡後，PSG.LGD雙大哥也被Navi帶走。這波過後，蝙蝠找到機會單提鳳凰，但隨後自己也被螞蟻收掉。這一波相當於PSG.LGD被打了個五人團滅，元氣大傷。22分鐘，TB在中路被TA盯上，儘管TB交出換血，還是被趕到的Navi眾人配合TA完成擊殺，順勢帶走了蝙蝠和海民。25分鐘，末日在上路交大動手TB，隨後開著變身的TB儘管以超高輸出將末日點死，但還是難敵螞蟻TA雙核，慘遭擊殺。隨後支援而來的陳被留，蝙蝠也在中路被鳳凰帶著雙輔助送回泉水。

29分鐘，Navi站上天輝高地，TB變身秒人卻被A杖鳳凰變蛋救下，隨後末日大招鎖定TB將其帶走，Navi由此破掉了天輝第二路高地。31分鐘，Navi推進上路高地，擊殺天輝雙輔助後鳳凰完成如神殺戮，PSG.LGD由此打出GG。
