童年噩夢——《英雄薩姆》，勾起80、90後玩家內心深處的“恐懼”

1 週 ago

摘要

每一個玩家心中，都會有一款遊戲是自己童年的噩夢，而《英雄薩姆》這款遊戲，我相信是很多玩家心中的童年噩夢。我們每一個人內心深處都有自己不願意麵對的恐懼，正如暗黑破壞神中演繹的那樣，恐懼魔王迪亞波羅可以用人類內心的恐懼來支配人類。

每一個玩家心中，都會有一款遊戲是自己童年的噩夢，而《英雄薩姆》這款遊戲，我相信是很多玩家心中的童年噩夢。我們每一個人內心深處都有自己不願意麵對的恐懼，正如暗黑破壞神中演繹的那樣，恐懼魔王迪亞波羅可以用人類內心的恐懼來支配人類。而《英雄薩姆》帶給我們的恐懼並不是像傳統恐懼類遊戲那樣，用「宗教性」的東西來「嚇唬」玩家。在《英雄薩姆》里沒有所謂的喪屍鬼神，有的只是更加「詭異怪誕」的奇怪科技。

三個時空的故事：最後的地球人

《英雄薩姆》在2001年發布與玩家首次見面，之後陸續出了第二部和第三部，回顧故事線我們可以發現是三個時空的故事，而薩姆則是為了拯救人類世界的「最後地球人」。遊戲故事背景設定在2032年（離現在也就還有12年……），外形侵略者對地球展開了毀滅性的侵略打擊，薩姆作為最強的英雄被送到外星人星球開始反擊，但在故事的最後雖然成功擊殺了最終BOSS，但一顆飛向地球的隕石不可避免的帶來了滅絕。所以就有了遊戲開始的出生點——時光機器，薩姆穿越到3000年前來改寫人類被滅絕的結局，所以他也是孤身一人的「最後的地球人」。

所以有了「最後的地球人」這個設定之後，遊戲裡帶來的緊張感就會特別明顯，試想一下，在一顆陌生的地方星球上，你孤身一人沒有援助沒有同伴，還要擔負起拯救人類族群遠離滅絕的重任，這本身就是沉重的。但遊戲策劃從設定上，把遊戲變成了快節奏盡情廝殺敵人的風格，這種風格正好可以很好的稀釋背景故事的壓抑，讓玩家可以痛快的傾瀉自己的情緒。

外星科技無不透露出怪誕的恐怖設計

遊戲雖然不是以恐怖題材為設計背景，但是外形文明的各種怪異設計和解謎摻雜在一起，造就了一個讓玩家極度緊張的遊戲狀態。比如《英雄薩姆》里最經典的怪物「炸彈人」，我們在遊戲中往往首先聽到「啊」這樣由遠到近的喊叫聲，隨著這樣越來越近的喊叫聲，玩家的神經也會越來越緊繃，因為遊戲中場景的詭異雨林設計，四周茂密叢生的雜草讓你不知道這個怪物會從哪裡蹦出來！還有那些解謎的關卡，你永遠猜不到密室之後會有什麼恐怖的外形怪物在等著。

解謎和盡情廝殺釋放壓力擺脫恐懼

雖然英雄薩姆帶給我們很多玩家童年陰影，但是不得不說，多少年後的今天我們再次回味這款遊戲，就又會有新的感覺和體驗。

英雄薩姆在遊戲設定上，本身就是一個節奏極快、包含了各種各樣的武器和怪物的第一人稱射擊遊戲，Croteam製作的英雄薩姆本意也不是為了讓玩家在恐懼中度過的，而是希望玩家在日常疲憊的工作生活中解放出來，在虛擬的世界中變身成為最後的地球人薩姆來拯救地球！

在遊戲中你會有這樣的感覺：明明怪物很多很恐怖，但作為體驗者我們卻能盡情的廝殺釋放自己現實生活中的壓力，而遊戲之後精神上卻是十分放鬆舒服的。遊戲的本質其實都是「英雄薩姆式」的，現實中的壓力在虛擬的世界中盡情釋放，然後用自己最好的狀態再次回歸生活。

作為被大多數80、90後玩家稱為童年噩夢的英雄薩姆，現在回味起來會有另外一種完全不同的體驗！而那種緊張刺激的快節奏遊戲風格，加上怪異有趣的異星生物戰鬥，還真的能讓玩家工作學習之餘盡情釋放壓力！你玩過英雄薩姆嗎？裡面哪種怪物讓你記憶最為深刻？
