火影手游：疾風咒印佐助確認周六上架，價格爭議仍在內部討論？

2019-12-13 2,062條評論 4,462次閱讀 1746人點贊

疾風咒印佐助應該是今年內比較受爭議的一個忍者，本來是原定10月份上架，後來直接延遲到12月，然後到了12月份，通過一次線下策劃活動中得知是12月13日上架，但一直到現在都沒有看到上架的公告，而且價格方面也一直是個迷。

然後，在今天的大版本更新後通過拆包發現了一張疾風咒印佐助的購買圖片，上面顯示的是328限時購買，這也恰好是應對了之前帕克提到的直購方式，這個價格的直購忍者應該是首次出現，即便是高人氣的佐助也難免受到了很多玩家的吐槽。

當很多玩家認為這個忍者會跳票的時候，官方發布了一個推送消息，上面提到是本周六上架，也就是12月14日，這個比原定的時間略微推後一天，時間上影響不大，但當很多玩家興奮的進去看價格的時候才發現，官方壓根就沒有提到價格，看來，這個價格問題確實比較敏感，魔方似乎仍然在內部討論具體的上架價格。

如果是按照官方確認的周六上架的話，那今晚12點或者明晚12點後應該會在公告里臨時加入疾風咒印佐助的消息，上架時間應該是在14號的中午12點上架，這也是除了高招忍者和限定忍者以外的新忍者常規上架時間。

我個人認為，魔方看到今天大多數玩家都在吐槽價格太高，按照正常的邏輯，魔方現在應該是會直接在328價格上打一個折扣，恰好是198直購，這樣既可以賺的銷量，又可以挽回一些人氣，簡直是雙贏，又或者是當初的328壓根就是魔方提前放出的煙幕彈，然後再以一個低價給玩家，這樣就會讓很多玩家覺得撿了便宜，本來不打算買的都有點動心了，總感覺原價是328，現在直接198，不買血虧？

當然了，上面只是我的個人猜測，魔方如果執意按照328上架的話也是沒啥問題，一個願打一個願挨，價格接受不了就不買唄，這個可以理解，但這樣的話肯定還是會讓吐槽聲延續下去，銷量上也不會太好，即便你強度再高，一個忍者也不會翻出太大的浪花，現在鷹佐助、曉蛇丸、穢土長門什麼的難道比不過一個疾風咒印佐助？如果真要出一個超過鷹佐助的直購忍者，那魔方真的是不打算照顧大多數玩家的感受了。
