《守望先鋒》全球總獎金池縮水？職業競技賽明年賽制變更

天天要聞
2019-12-17

《守望先鋒》職業競技賽官方網頁上周末發布了關於《全新的職業之路電競體系》公告，文中寫到關於2020年OC賽區的全新規劃並對其做更詳細的解釋，其中職業競技賽每年會改成兩個完整賽季、舉辦每周兩次的系列賽事，並採用門檻式的地區決賽晉級制度，且職業競技選拔賽和公開賽將同時進行，官方認為這將能創造關係更緊密的職業之路電競體系。

一般通則部分，北美賽區將再度把東西兩區整並成一個賽區、OWL培訓隊現在最多可以同時擁有4名雙賽型選手（過往限制為2名）、2020年開始移除對於居住資格的限制，官方也公布2020年《守望先鋒》職業競技賽總計有250萬元，只不過這個獎金池似乎比往年總額少了50萬美元，根據官方頁面公布的獎金，過去一年應為300萬美元。

詳細賽制部分，2020年《守望先鋒》職業競技賽會在全球七個賽區各舉辦兩個賽季，每個賽季會有4場職業競技選拔賽、4場職業競技賽，並在最後進行地區決賽，參賽隊伍只能透過職業競技賽累積晉級地區決賽所需積分。

根據官方之前在2020年賽事搶先看報導所述，2020年初將邀請所有2019年第二賽季職業競技賽隊伍參加雙淘汰制預賽以決定各隊在全新職業競技賽的初始種子序，預賽會在1月6日至1月26日期間舉行，前四強隊伍將會晉級首場八強職業競技賽，末四隊則將落至首場職業競技選拔賽；在此同時公開賽也將於賽季之初舉行，經過七周賽事後，將有八支隊伍晉級選拔賽。

緊接著展開的正賽部分，首先展開的是為期一周的職業競技選拔賽，預賽末四隊伍與自公開賽打上來的八隊、總計十二隊將在單敗淘汰賽中競爭，前八名將可以晉級第二周展開的首場職業競技賽，包括預賽前四種子與從選拔賽打上來的八隊、總計十二隊同樣會在單敗淘汰賽中對決，每隊在此階段賽事將可以累積分區季後賽積分，前四強隊伍可以直接晉級兩周後的下一場職業競技賽，後八強則會再度回到下一周的選拔賽，重新爭取前八名晉級職業競技賽席次。

從官方提供的示意圖可以看出，2020年賽季選拔賽與職業競技賽都將采雙周制，每兩周為一個循環並重複四次，最終單季將有4場職業競技選拔賽與4場職業競技賽，而在職業競技賽第四周賽事結束後，將由積分最高的八支隊伍晉級雙淘汰制的地區決賽，決賽采BO5、總決賽則是BO7，各地區優勝隊伍將以機會晉級國際性賽事，如歐美／亞太交鋒賽及全球交鋒賽。

公開賽部分，官方指出，全新賽季公開賽將採用新賽制，參賽隊伍每年有四次的機會可以晉級職業競技選拔賽。不論實力排名高低，所有玩家都可以參加公開賽，但是報名為限時開放。
