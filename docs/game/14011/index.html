火影手游：組織合併實測，飾品屬性選擇，其他活動及數據補充

2019-12-11 1,038條評論 3,350次閱讀 2089人點贊

>>>防禦，攻擊，生命

關於這三個屬性哪個好，一直有爭論，這兩個月就用實戰讓這個爭論終結吧。

  1. 我已經來到70級飾品的門檻，我這次70飾品，將會全部用防禦屬性，然後在天地戰場的時候記錄挑戰的對手，結束後去該區觀察對手的數據，親測防禦究竟強不強，具體情況會錄屏。
  2. 70飾品完我是直接跳100級飾品，100級飾品我將全用生命屬，重複70級時天地戰場的操作，親測生命屬性強不強。

>>>小隊數據會隨著等級的提升而遞增

昨天感覺數據不太對，做了下記錄，聲望過然是有變化，忍玉和密卷倒沒有。

3倍3915聲望，63級數據。

3倍3951聲望，64級數據。

>>>遊戲大更新分析

遊戲內已經更新了公告，今晚更新版本，改動還是挺大的，可以在設置-功能入口-遊戲公告中查看。或者看推文第二篇文章。

忍者改動其實也就3個人，就是砍了小自來也和柱間、三代秒殺流，柱間密卷雖然CD增加了，但一把照樣是能放兩次。不至於上不了場。

仙鳴這個我感覺還是等於沒加強，反正我現在決鬥場根本不帶霸體，也不用奇拉比，你愛咋破霸體破霸體。決鬥場現在破霸體、抓霸體、解術散泛濫，而且仙鳴這個加強可能就是開大的那一秒能破霸體，奇拉比那會可能還是無敵，搞不好還是能尾獸化出來把仙鳴打一頓。

說難聽點就是現在霸體都沒用了，你破霸體就更沒用了。

組織合併前面的區的大佬測試過了

合併後要塞不能疊加，想霸區刷要塞禮包是不可能的，而且直接會消失掉那個被合併的10級組織。

下次想辦法測試一下曉組織拍賣的東西會不會增加。

改動大的主要是這兩個：

組織回歸任務應該是加強了小號流，10個學員可以歡樂的刷起來。原價拍賣買1500金幣飾品給大號刷金幣。這個應該是對應超影加強氪金玩家，小號加強V0肝帝的。

通靈獸鮫鯊算是大加強了，沒咬到人會反向在咬一次。

增加了組織爭霸的報名和參與獎勵，這算是節日日曆沒有聖誕節的補償了估計。

>>>暖冬集祝福

本人實測：如果被禁言，你的祈福將消息不會在遊戲聊天頻道中出現，發的時候會直接提示你離解除禁言還差XX小時，XX分鐘。

此狀態下你只能等待活動界面內的人給你祝福，而公屏中你無法收到祝福。

暖冬集祝福的獎勵發生了變化，保護符變成了兩個，比昨天增加了一個。

不管怎麼說大家平時說話還是要小心，萬一被禁言的時候出現某種需要發言的活動就尷尬了。

被禁言後，私聊以及組織招人的那個公告也發不出去。

>>>新增自動回復

增加了「字體」「符號」「顏色」的自動回復，內容一致。優化精簡了文章內容。

今天晚上12點更新，任務記得接時間最久的，還有店鋪也是，記得搞完。

我玩的晚，不知道以前登錄界面有沒有我愛羅，如果這次更新完還是三代，那我愛羅就是真的沒牌面。

昨天微信玩家安和我說，佐井畫畫的時候召喚過老鼠，新春忍者可能是佐井，這個我看火影的時候還真沒注意過，如果佐井召喚過老鼠，那他也有可能。
