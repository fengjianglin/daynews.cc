LCK新一代銀河戰艦來了？ESPN爆料DRX購入Doran和Chovy，LPL難了

2019-12-04 0條評論 17次閱讀 0人點贊

摘要

大家好，我是想稱霸海上卻沒有船的Aggro電競王路飛同學。近日ESPN又曝出了一則非常勁爆的消息——DRX同時簽下了Chovy和Doran。

大家好，我是想稱霸海上卻沒有船的Aggro電競王路飛同學。近日ESPN又曝出了一則非常勁爆的消息——DRX同時簽下了Chovy和Doran。

在此之前，DRX也官宣了自己將在近期對外公布新一任的中單選手，另外DWG的牛寶也在直播中說過Chovy會加入DRX，結合這幾點來看DRX的新任中單應該是Chovy無疑了。

如果DRX真的能購入Chovy和Doran，那麼他們確定的人員名單就變成了麥哥、羊駝、多蘭和超威，加上前段時間Doinb在直播時還說Deft會迎來一個超級強的輔助，照這樣看來他們如今就僅差一個強力的打野位人選。

這勢頭看來DRX是鐵了心要組建LCK新一代的銀河戰艦，今年的Chovy由於世界賽和常規賽中的出色表現，前段時間據說身價已經漲到了很高，這次DRX也是下了血本。

今年轉會期LCK的戰隊都很猛，除了DRX以外還有GEN.G，買下Clid、BDD這兩位天才選手的同時還約簽了尺帝和Life，也是LCK的新一代銀河戰艦之一。

這兩艘銀河戰艦對於LPL而言都是非常難啃的骨頭，不知道明年他們會不會進入S10的世界賽。

圖片來自GEN.G官博

除開這兩支重組的隊伍，LPL潛在的敵人還有經歷了世界賽洗禮的DWG，對於這幾名小將而言明年可能會發揮得更好，因為不管是牛寶還是ShowMaker都是極具潛力的選手。

另外SKT雖然看似只引入了幾個青訓和一些在LCK沒什麼存在感的選手，但是因為有Faker和Teddy的存在，SKT依舊不容小覷，因為那個男人的ID便自帶了幾分霸氣。

圖片來自SKT官博

以上這幾支戰隊應該就是明年咱們LPL賽區最大的潛在敵人了，所以等到春季賽開打一定去好好研究下他們，爭取做到知自知彼百戰不殆。

不過其實也不必太過擔心，重組的隊伍始終需要一個磨合期，而如今LPL的隊伍也並不弱，比如FPX、IG和RNG，都是極具實力的強隊。
