元氣騎士：神器對決，導彈發射器惜敗加特林，差距終究難以彌補

2019-12-07 0條評論 17次閱讀 0人點贊

摘要

在《元氣騎士》這款遊戲中，初入元氣的萌新玩家追求的是更快的熟悉遊戲規則，而新手玩家追求的則是如何更加安全有效率的通關，至於大神玩家，他們最熱衷的事莫過於追求極致。不論是挑戰極致的通關速度，還是極致的武器傷害，更或者是極致的操作技巧。

在《元氣騎士》這款遊戲中，初入元氣的萌新玩家追求的是更快的熟悉遊戲規則，而新手玩家追求的則是如何更加安全有效率的通關，至於大神玩家，他們最熱衷的事莫過於追求極致。不論是挑戰極致的通關速度，還是極致的武器傷害，更或者是極致的操作技巧。總之一句話：不論你是何種定位的玩家，都能在這款遊戲中找到屬於自己的那份挑戰。可是既然提到了極致傷害，那麼就不得不和小夥伴們聊一下那些在元氣世界中大放異彩的「神器」了。

在《元氣騎士》這款遊戲中，裝備的數量共有近300種，每種武器之間的定位都是各不相同，有的側重於傷害，有的側重於防禦，有的側重於控制，可是若問起究竟以上哪種定位更具優勢才能稱得上「神器」之名？個人認為極致的綜合傷害才是不可或缺的主要條件。

紅武一拳之威，想必小夥伴們仍然歷歷在目，一秒999傷的雷霆一擊，也足以讓整個地牢的boss都為之膽寒。可畢竟一拳太過稀有使用限制又多，所以那些在平常寶箱中才會出現的常規裝備，才是玩家接觸最多的武器種類。除了像一拳這種初始極致攻擊屬性的武器之外，想達到與之匹敵的綜合傷害，那麼就只能退而求其次選擇與之對等的屬性，譬如散射與攻速。在遊戲中散射武器有多強老宅就不再過多描述，可是若問起誰才是這個領域中的翹楚，老宅首推「加特林」與「導彈發射器」，至於這兩種「神器」孰強孰弱？三言兩語也難以解釋的清楚，那麼本期就藉此機會和大家聊一下這個話題：神器對決。

武器特點

  • 導彈發射器

隸屬橙武領域屬遊戲中的高階等級，其武器特點個人認為可以簡單歸類成4點，分別是：散射、高速、自動鎖定、爆彈。其中最具代表性的當屬它自動追蹤目標的能力，即使彈道為散射但卻絲毫不用考慮命中率，在駕馭它的同時只需專註走位即可，畢竟它是必中的。

初始屬性攻擊2點藍耗2點再搭配極高的射速，所以導彈發射器也是一個妥妥的「藍零王」，而且爆彈特點的武器具備灼燒能力可對目標附加點燃效果，缺點就是無法暴擊。

  • 加特林（藍光）

在遊戲中加特林共有兩種，分別是：加特林與藍光加特林，兩把武器外形相似攻擊模式相仿屬性相近，若問及兩者之間的區別，除了藍光較前者遠程命中率更低之外，就要屬藍耗差距了。由於加特林的武器特點是射速極高，所以它比導彈發射器更加消耗藍量，因此藍量縮短一半的藍光則更受玩家喜愛。

初始傷害2點彈道4點的加特林遠距離命中率極低，想發揮極限傷害唯有近距離輸出一種方法。可是距離目標太近又無法保證自身安全，再加上加特林本身具有降低移速的副作用，所以輸出環境受限是它的最大缺點。

武器強化

導彈發射器的滿配公式為：霰彈增加、攻速提升、火盾、能量+100、狂戰士雕像、武器配件、咖啡（攻速提升）。

強化解析：由於爆彈型攻擊模式的武器定位特殊，不受多種天賦增益，所以導彈發射器對天賦需求並不嚴格，較加特林相比容易滿配，但是容易滿配也就意味著傷害提升幅度受限。

加特林的滿配公式為：霰彈增加、攻速提升、子彈彈射、暴擊穿透、精準暴擊、能量+100、狂戰士雕像、武器配件、果汁（暴擊率提升）。

強化解析：加特林具有散射武器的所有優點，單是暴擊能力就比前者傷害增幅巨大。而且子彈享受彈射效果增益，角度合適可以造成2次傷害，雖然有精準暴擊縮短彈道範圍，命中率依然是它的最大問題。

數據解析

滿配後的導彈發射器彈道為5道，提升傷害的方式唯有狂戰士雕像和工匠配件兩種，拋開與加特林相同的配件增益，那麼傷害增幅就只有2點，單次綜合傷害為：20點。

滿配後的加特林彈道為6道，受狂戰士雕像增益就是4點，單次綜合傷害至少為：24點。若受暴擊以及彈射效果加成，拋開子彈不能全額暴擊的限制，兩兩相加至少可提升一倍。

總結：導彈發射器弱於傷害強於命中率，加特林強於傷害使用難度更大，所以前者更適合新手玩家使用，而加特林想發揮最大威力，唯有老玩家方可駕馭。因此《元氣騎士》神器對決，導彈發射器惜敗加特林，無法暴擊的差距終究難以彌補。
