任天堂健身環有多火？賣800元還被買到斷貨，國產山寨卻只要200？

大家好，這裡是正驚遊戲，我是正驚小弟。

近日，小弟每天都在看著自己的體重嘆氣，平時工作忙，閑暇時刻又喜歡玩遊戲，哪有時間出去運動呢？

不過，任天堂前幾個月發售的《健身環大冒險》很好的解決了宅男宅女們的健康問題，既可以玩遊戲又可以健身，這絕對是一件一舉兩得的好事。

不過令人可惜的是，遊戲發售兩個多月，缺貨問題卻一直得不到解決，別說中國了，就連在日本國內這款遊戲也一直是供不應求，任天堂官方則一直表示歉意，從數據上來看《健身環大冒險》的出貨量確實一直在增加，但是問題還是沒有解決。

在10月時，《健身環大冒險》以13.25萬套的銷量力壓PS4上的《使命召喚》，是軟體銷售榜的第一名，而截至11月中旬，《健身環大冒險》的日本銷量就已經超過了23.6萬套。

看來日本的胖子很多啊！

雖然也有電子版遊戲的存在，但是由於《健身環大冒險》需要使用Switch的最新硬體：「Ring-Con」和「Leg Strap」控制器。由於這兩款硬體的備貨並不充足，所以導致了這次缺貨問題。

此時，我們正義的中國賣家站出來了！

日本遊戲公司一直缺貨，我們中國作為國際友人義不容辭！短短兩個月，淘寶上就出現了數不清的國產硬體！

正版健身環硬體已經被國內店家炒到了天價（原價500元，現在將近900元），國產硬體廠商站出來說不！

「要買就買我們的！只要288，運動健身拿回家！」

某寶上許多店家的月銷量都超過了三位數，並且看評價，產品確實獲得了絕大多數玩家的認可，看來國內硬體廠商的模仿能力真的是無與倫比！絕對的強大！

當然了，這種生意合不合法大家心裡都有數，任天堂法務部可是出了名的強大，後續會發生什麼小弟就不做推測了。

對於山寨健身環，網友們是怎麼看的呢？

——有一說一，破解玩家來說，現在買這個東西，溢價還是高

——正版健身環大夥不怕麻煩的試試在歐洲買吧，這邊一直都有貨的，價格比某些奸商低多了

——各位ns新老玩家注意，現在已經有高仿健身環了，注意區分，別高價錢買了仿貨。 ​

看到這裡小弟有話想說：雖然因為缺貨問題，硬體得不到解決，但是國內硬體廠商這種行為真的非常冒險，正版健身環加上遊戲原價也就500元，而國產的健身環不帶遊戲，如果再去買個遊戲的話，合起來也將近500元，相信如果備貨充足，不少NS的忠實粉絲還是更願意購買正版的吧！

一個正驚問題：你買《健身環大冒險》了嗎？
