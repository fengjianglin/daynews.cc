COD16白送雙倍經驗，新地圖專治“小聾瞎”，玩家又該整花活了？

天天要聞
2019-12-25

近期，動視暴雪的新作《使命召喚：現代戰爭》進行了一次更新活動，伴隨著第一賽季的進行，COD16又添加了內容。本次更新將會上線兩款新地圖，Vacant和Shirley（貨運），其中貨運地圖可以進行6V6和槍戰模式，Vacant則應該是一個重置的老地圖，COD4中出現過，它原版是來自戰地2的，不知道這次改成了什麼樣。

而貨運圖從演示中能看出是立體式陸空對戰，可以在集裝箱上戰鬥，搶佔箱頂能提供更好的視野範圍，同時也有無掩體的弊端，站在上面估計活不過三秒。這個地圖之前在幽靈和OL中出現過，玩家表示地面我都顧不了，現在還要顧天上；而且這種可以上房頂的打法，讓「聾」的傳人更加分不清腳步，到底是在地面還是上面，太難分辨。

Shirley

目前還不知道在COD16中有沒有好轉，大家玩了再評論。除了上線兩個新地圖，《使命召喚：現代戰爭》還加入了之前在《使命召喚：無限戰爭》中出現的Cranked模式。Cranked類似於傳統的快節奏團隊死斗，玩家擊殺敵人會獲得Buff，能提高自己的移動速度和填裝速度，但同時如果不保證高效的殺敵，導致計時器滿格，自己則會被炸死。

Cranked模式

另外，這次官方還給大家準備了一份節日禮物（聖誕節）和雙倍武器經驗周末（本周六到下周二），禮物現在已經能在商店中領取了。

還有大家一直很期待的鐵面男「尼克托」捆綁包也上線了，售價2400CP。根據介紹鐵面男是前俄羅斯FSB的一名資深特工，現在被指派到「阿爾法小組」參與戰鬥。「阿爾法小組」是前蘇聯於上世紀70年代為反恐而建立的特戰小隊，他們自組建以來在行動中還從未失敗，但它們也因為屢次違抗命令，引起了一些非議。

鐵面男上線之前在玩家群體中呼聲就很高，這次作為第二個幹員上線，想必很多小夥伴已經下手了，遊戲日報君也打算入手一個。畢竟帥才是最重要的，白給什麼的都不怕。

COD16這次的更新了這麼多內容，大家覺得怎麼樣呢？
