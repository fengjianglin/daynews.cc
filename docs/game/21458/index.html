阿水去哪了，EDG經理和PDD給出不同答案，到底誰在口嗨

2019-12-20 46條評論 1,016次閱讀 60人點贊

持續一個月的大型懸疑劇《阿水去哪了》還沒有結局，在各大俱樂部官宣德杯陣容的時間點關注更甚。PDD和EDG青訓經理老岳分別在直播中給出不同的答案。

PDD的說法，「阿水會休息三個月。」

EDG青訓經理老岳信誓旦旦的確認小道消息，「阿水和RNG簽訂了意向合同，TES、EDG因為這個合同需要付1000萬違約金放棄了簽阿水，SN考慮自家AD訓練賽效果好，雖然不用付違約金，但阿水工資太高，也改主意不簽。」

PDD休息三個月的說法沒考慮到一個頂尖選手長期不打比賽和訓練賽對狀態的保持，三個月時間2020年LPL春季賽早已開打，但阿水是自由人，隨時可以簽約，根據選手轉會細則，報備通過，次日可以上場，只是和隊伍肯定會缺少磨合。

EDG老岳的說法有點不合邏輯，RNG有Uzi在，阿水不可能簽過去當替補吧。即使雙方處於某種原因簽約，如果這個轉會期RNG耽誤阿水後續與其他俱樂部簽約，事情影響太大，怕是會驚動LPL官方出面。

目前各大俱樂部德杯大名單差不多官宣完了，阿水的去向越來越不明朗，又重新回到薛定諤的狀態，哪隊都有可能。

自從官宣IG.Jackelove合約到期，成為自由人，截止目前為止對阿水去向的猜測就千奇百怪，種種流言把阿水去LPL十六支隊伍都安排了一遍。有些小道消息前後邏輯矛盾，有些比較誇張不可信，比如LPL解說娃娃直播說SN給出3500萬簽阿水。

與之相比，EDG老岳和PDD的雖然一樣不靠譜，又還有一絲可能。然而提出這兩種說法的兩位讓人難以信任，老岳身為EDG青訓經理，Iboy被VG買了都不知道，PDD也是剛說完寧王世界賽上不了場，後腳寧王登場打臉。這兩位轉會期所謂爆料就沒一個中的，怕是最後都得再次失信。

阿水轉會消息傳得離奇，各種挑戰吃瓜群眾智商皆有，網友乾脆自己玩點花樣。IG.Jackrlove選手轉會洛杉磯湖人擔當控球後衛。

​

韋神從LOL轉到吃雞之後創立4AM戰隊，今年拿下絕地求生世界賽季軍，轉型非常成功，阿水也被網友安排過去，不過不是兩人聚首，而是互換隊伍。

其他還有諸如安排阿水進國家游泳隊、CSGO分部等，實在是大家被轉會期各種無厘頭爆料轟炸的頭疼。至於阿水去向如何，現在等待官宣才是最靠譜的。
