loken重返JDG！今年淚曬冒泡賽，發現還是綠毛適合自己？

天天要聞
2019-12-18

在很久之前就有小道消息爆料道，有一位LPL的AD選手將會重新回到他開始的戰隊。當初很多的小夥伴還在好奇，這位選手究竟是誰，沒想到如今卻終於浮出了水面。這位AD就是我們LPL自己培養的loken，相信很多的小夥伴對於他都非常的熟悉。

可能很多網友對於他的第一印象就是洲際賽上的軟腳蝦。在TES這支隊伍裡面，loken打的好像特別的差。一個賽季沒有一個MVP，沒有任何的高光操作。用AD被UZI花式教育等等。但其實這些都是大家誤會了，因為TES的打法不允許Lkoen打的非常的凶。

我們都稱TES是一個低配版的IG，那就說明他是主打中上野的，所以下路註定是抗壓， 因為吃不到什麼資源，並且大多數的時間都是黃金左手在carry。就算knight不能C，上單還有369的存在，所以這也是loken被稱為工具人的原因。

但是loken真的就這麼的差勁嗎？其實在去年JDG戰隊裡面，loken的表現讓大家都覺得非常的穩健，無論是春季賽還是夏季賽，對上當時如日中天的RNG，面對巔峰時期的小狗時，loken都臨危不亂，跟綠毛一起和小明+UZI打的有來有回，並且在春季賽的時候還一度雙殺RNG。這也是一種實力的證明呀。

而Loken這一次的TES之旅，肯定會是他人生中一段寶貴的財富。因為”韓式AD”普遍都比較穩健，屬於後期發力的那一種。在接下來的版本中，AD的作用就是穩健發育，等到後期的carry就行了。相信在這一點上面，loken肯定會做的相當出色。

在今年的冒牌賽上，TES因為輸掉了IG喪失了前往全球總決賽的名額，loken淚撒對戰席上，那一幕讓很多網友也動容。既然是打比賽的，誰會不想贏呢？以前大家總是說loken的技術不怎樣樣，但是在那一場比賽中，loken確實TES里發揮最好的一個。

如今loken重回JDG，並且碰上了自己的老搭檔綠毛，這一對下路組合曾經在LPL也是非常出名的，這一次有了loken的重新回歸。不知道明年JDG會不會成為那支逆襲的「黑八」呢？
