拒絕吃檸檬！《實況足球手游-2020》中國球員推薦：強到出乎意料

2019-12-15 535條評論 3,686次閱讀 285人點贊

最近這個大冬天的，想必大伙兒已經很少吃水果了，不過有人卻偏偏要讓咱們吃，還非要硬塞一個又大又酸的檸檬過來，這人就是南野拓實，最近已經幾乎官宣加盟利物浦的年輕球員。咱們的國足還在恨鐵不成鋼，人家那邊已經就已經融入五大聯賽了，眾球迷牙齒都快酸掉了吧…

不過這對於實況足球的玩家們來說，似乎沒有那麼酸，咱們國足的成績風雨飄搖，主要還是陣容不穩定磨合不夠的緣故，並不代表著咱們國內就沒有好球員。這不，最近遊戲《實況足球手游-2020》正式公測，增加了隊套玩法之後，眾實況玩家們紛紛在國內球員中挑兵選將，誓要把檸檬喂回去。今天咱們就來盤點一下，《實況足球手游-2020》里實力不凡的國足球員。

武磊

武球王在實況里速度優勢還是明顯的，並且作為《實況足球手游-2020》里唯一真實建模的國足球員，實力肯定是不容小覷。不過武球王的數據還有一點令我們驚喜的地方，那就是他的逆足使用頻率與逆足使用精度都是處在較高水平的。

艾克森

艾神的數據可以說是相當均衡了，技能也都比較適用，禁區之狐的風格搭配這樣的速度等於進球輕而易舉，而88的腳下力量有能夠讓艾神在遠射方面有所建樹。

韋世豪

韋醬的速度比武磊還快，雙傳、盤帶在白球里都屬於較高水平，逆足精度與武磊一樣，體力也不賴，主要是目前還24歲，據說下賽季有很大可能留洋，祝願他越來越好吧。

何超

何超的數據也是非常均衡，而國內雙方能夠上83的中場球員實在是太少了，這甚至是一個比後衛還高的數據，24歲的何超不僅數據均衡，風格也是十分討喜的全能中場風格。

張玉寧

張玉寧不愧是在歐洲聯賽鍛煉過的，不僅擁有白球很少見的85的身體接觸，腳下力量與射門數值都在較高水平，速度和頭槌還都不弱，只希望22歲的他將來還有留洋機會，能夠好好練練級。

所以說《實況足球手游-2020》對咱們國服玩家還是挺友好的，雖然咱們國足有那麼一點點不給力，但是球員們的個人能力還是有的，加上《實況足球手游-2020》新增的隊套玩法，這就給了玩家們「為國出戰」的機會，並向世界證明我們中國足球到底有什麼樣的實力。

目前《實況足球手游-2020》已經開啟了全平台公測，如果你也想在這一代實況里試一試這一批國足球員們的實力，不妨現在就上車，因為《實況足球手游-2020》最近有每晚20:20準時在線的福利活動，精選、傳奇、黑球球員唾手可得！而參與挑戰賽還有贏取梅西正版球衣的機會，還在等什麼，測測血統吧？
