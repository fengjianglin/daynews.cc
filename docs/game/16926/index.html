明日之後：玩家“更換包體”後發生多件奇事，飛天遁地無所不能

2019-12-15 57條評論 1,068次閱讀 152人點贊

歡迎諸位小夥伴們來到本期天哥開講的《明日之後》「生存那點事兒」~相信諸位倖存者們已經在遊戲中發現，網易CH帶來一個關於「更換包體」的新任務玩法！有些玩家順利的完成並且領取了相應的獎勵，但是有些玩家則遇到了一些讓人匪夷所思的遊戲經歷！

得，廢話不多說了，還是各位小夥伴們熟悉的「老配方」：有圖有證據，邊看邊聊天兒吧~

「一路同行感恩有你」的更換包體？倖存者有點「懵圈」了

在《明日之後》這款遊戲里呢，很多新任務玩法總會在玩家們猝不及防下忽然而至~比如當初的鋼鐵入侵、災情局等玩法都是在玩家沒有任何準備下，網易CH也沒有提前發布公告的前提下在遊戲中突然的出現~而現如今呢，這款遊戲里又忽然出現了一個網易CH寫著「一路同行感恩有你」的更換包體的新任務，美名其曰可以暢想優質遊戲體驗，並且有相對應的任務獎勵，比如配方殘頁、三圍等級經驗等等~不過網易CH並沒有公布為何會出現更換包體這樣奇怪的任務，所以倖存者們有點「懵圈」，不過在看到獎勵之後自然不能錯過~如此一來，就有了接下來發生在倖存者身上的奇怪事情了！

文字變成了「方框框」？顯示看起來有點奇怪！

由於此次「更換包體」後會得到相應的獎勵，所以很多玩家紛紛按照任務步驟更新了《明日之後》~而上圖中的這位倖存者玩家在更新之後呢，就發現自己的遊戲界面變得有點與眾不同了！任務欄、小地圖上方甚至是遊戲人物的血量、健康值等UI界面，出現了一種神奇的「方框框」，替代了原本正常的文字！

有趣的事情是，即便是打開任務欄、日程欄等系統任務的時候，也會出現這類「方框框」，似乎遊戲安裝有所缺失一般~不過呢，這種情況還是比較常見的，畢竟《明日之後》遊戲中已經不止一次出現過類似的狀況了！據天哥了解，這位小夥伴後來通過「修復客戶端」之後已經恢復正常的文字提醒了，所以說如果有倖存者也遇到了類似的狀況，不妨嘗試使用「修復客戶端」的方式來完成更新哦！

飛天遁地無所不能，新視角看著很奇怪！

接下來這件奇怪的事情同樣也是發生在「更換包體」後的玩家身上。如上圖所示，此時玩家的視角有沒有覺得很奇怪呢？嘿，這是宛如被「遁地」了一般，從地下探出頭往外張望似的，讓人覺得相當有趣~

還有類似這位玩家遇到的神奇遭遇，直接飛上了天空，而且還是「殘血狀態」，有倖存者調侃道：這樣的飛天是要表示玩家可以飛升上「天堂」嗎？好吧，天哥也忍不住的笑出了聲~事實上呢，這一次的「更換包體」後還出現了許多有趣的事情，但是大多數玩家都是與以往一樣的正常~不過講道理，截止目前為止也不知道為何網易CH會讓大家突然的更換包體，這就讓人難以捉摸了！

得，本期的《明日之後》「生存那點事兒」就和大家聊到這裡了，感謝各位的閱讀，接下來我們評論區里見，一起聊聊吧~
