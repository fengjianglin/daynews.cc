三國殺：歌神歸位，留贊重回鬥地主！金票商城更新

2019-12-14 3,375條評論 4,821次閱讀 3341人點贊

不怕告訴你們，最初的鬥地主模式剛出來時留贊就是在可選將里的，那個時候就已經是最強地主了，而且那個版本根本沒有禰衡、劉焉這樣的，但是即便現在他們依然沒有留贊強.

就是因為鬥地主太強所以留贊在鬥地主模式出來不久後就被禁了，歌神也暫時落幕了.

但是！如今！留贊又重新加入將池中，曾經的歌王又要重新回來支配戰場了.

奮音，每當使用一張與上一張不同顏色的牌便可以摸一張牌，本來無限續航的技能就讓人玩不停，更讓對手新生絕望.

如今地主又加了1點體力上限還能多出一張殺，直接讓留贊戰力翻番，即便沒有連弩也能大殺四方，可以說多出一張殺的優勢讓每個地主都不太受牽制，但是很明顯留贊的加持要明顯亮眼的多.

所以如果說有留贊的小夥伴一定要每天去鬥地主槍金票啊！但即便沒有留贊的小夥伴也沒有關係，說不定你的隊友就是有留贊的絕世高手.

在鬥地主里躺也是非常爽的，那種不用動手就能贏的快感真的會讓人上癮，別問我怎麼知道的.

還有一件更重要的事情就是金票商城更新了！

不止原來手氣卡、陶謙這類老古董可以兌換了，現在出了豬年盒子、大雪禮盒了！

300金票可以兌換豬年貳號盒子、大雪禮盒！

額外開啟的盒子兌換無疑是金票最好的回饋，招募令、武將、皮膚所需要的金票太多兌換起來比較多，可能會耗費玩家攢了很多天的金票，有些心疼.

但是盒子只用300金票，這個不高不低、恰好滿足所有玩家的標準的盒子無疑是最好的選擇，每天只需要贏一場300歡樂豆的鬥地主就能換盒子了.

不得不說，鬥地主這次的改版和金票商城的更新是非常良心的，如果有想嘗試最新版本鬥地主的小夥伴就趕緊去感受一下吧~
