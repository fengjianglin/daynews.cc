大話西遊2破日大仙屬性秀，誰說用14級武器輸出不強？

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

如今的仙族玩家最頭疼的事情就是如何提升自己的傷害，看著那些動不動就百萬級輸出的大神，估計無論是誰都得羨慕到不行。然而想要提升傷害就得投資各種裝備，甚至高階仙器都不在話下。不過小蘇這次要分享的這位大神沒有仙器、沒有神兵，甚至連武器都是14級，不過人家的傷害卻一點兒也不低。

如今的仙族玩家最頭疼的事情就是如何提升自己的傷害，看著那些動不動就百萬級輸出的大神，估計無論是誰都得羨慕到不行。然而想要提升傷害就得投資各種裝備，甚至高階仙器都不在話下。不過小蘇這次要分享的這位大神沒有仙器、沒有神兵，甚至連武器都是14級，不過人家的傷害卻一點兒也不低。接下來咱們就一起來看下～

討論傷害之前，咱們先來秀一波屬性和裝備。從圖中不難看出，玩家使用的武器是14級「破日」，也是大家經常調侃的類型。屬性方面以應付任務為主，混亂和遺忘抗性已經全滿，至於抗封印基本平時用不到。吃符文狀態下強力克木435，加強雷法131，忽視抗雷33.7，另外值得一提的是，玩家並沒有一點狂暴屬性。在如今的版本里，這樣「平秒」的面板屬性並不多見。

看完了屬性，咱們就來欣賞一波傷害秀，大家千萬不要覺得人家用14級武器就秒不出傷害，這是「域外風情」任務過程，一炮下去將近50萬的傷害，這數據在平民玩家裡絕對算得上優秀了。當然你也可以說某大神一炮可以秒百萬，甚至秒殺「鐵骨」，但是他們用的是六階仙器，相比之下14級武器能達到這個數據已經非常不容易了。

可能大家覺得域外的傷害是玩家多次截圖的效果，專門挑了一次最好的放出來。接下來這個「五環傷害」就應該沒的說了吧？進入戰鬥第一回合，直接就是「秒殺」，並且秀出屬性和上面的相差無幾。高強法再加上高強克的前提下，傷害絕對不容小覷。而且玩家是在孩子沒有喊話的情況下秒出來的數據，如果要是刻意截圖，再加上一點點狂暴和靈爆，估計突破百萬也不是沒有可能的。不知道各位對於這樣的數據是否滿意，歡迎在評論區留言，來說說自己的看法～

好了，關於「強法炮」的話題，咱們就先聊到這兒吧，我是小蘇，一個專註於大話西遊多年的老玩家。如果想了解更多遊戲資訊和相關玩法的話，記得關注一波，這裡絕對有你感興趣的內容～
