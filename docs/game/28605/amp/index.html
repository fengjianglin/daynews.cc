三國殺：這幾個武將在鬥地主中看到一次退一次！根本贏不了

天天要聞
2019-12-22

鬥地主是三國殺移動版中一個很有意思的玩法，兩個農民玩家需要想辦法殺死有著兩個額外技能的地主，在打倒地主的過程中，農民之間的配合也是必不可少的，那麼總有一些武將讓人看到就想口吐芬芳甚至於直接退遊戲的，今天小編就帶大家一起看看吧！

遊戲中的心態爆炸時刻

TOP1：農民張春華

張春華是一個很神奇的武將，作為地主的張春華屬於常年霸佔勝率榜的存在，而每次作為農民看到有隊友選了張春華心裡就會咯噔一下，根本原因還是因為張春華在三上限的時候的強度真的十分有限，一點體力上限的區別對於張春華來講就是天差地別的。更重要的是張春華屬於個人能力極強的武將但是在技能的聯動性這方面無疑是遜色了不止一籌的，很多時候地主在殺你之前都不會去動張春華，不動白又是信仰玩家就問你怕不怕?

TOP2:司馬懿

不得不說這個武將出現在鬥地主的將池中我覺得是一件很匪夷所思的事情，郭嘉春華之流也就罷了，一旦看到司馬懿出現在你的身旁的那一刻我相信你的內心一定是崩潰的。在鬥地主中除非遇到主公是大喬可能司馬懿還有一點發揮，其他時候他的作用甚至都不如四害，四害好歹偶爾還有著高光時刻，司馬懿就只能呵呵了。地主有著飛揚不怕判定的情況下試問誰願意要一個司馬懿做隊友？？

TOP3:鍾會

鍾會這個武將在身份局的發揮可以算的上可圈可點，其前期的低嘲諷加上覺醒後的穩定補牌讓其一度成為了內奸的代名詞，並且在內奸這個身份中鍾會的勝率一直是名列前茅的。可是身份場上適用的武將換到鬥地主中發揮總會有所差異，鍾會很不幸就是變的極差的那一批，了解鍾會技能的主公根本不會給鍾會覺醒的機會，隊友如果幫助鍾會覺醒無疑又會少很多輸出，這就造成了一個進退兩難的地步了，更別提沒覺醒的鐘會就是一個超大號的不動白，只能說誰遇到誰倒霉。

TOP4：土鱉

如果說前面的武將都是因為其不動白的特性榮登榜首，那麼這個武將絕對能刷新你對隊友這個詞的認知，沒過牌沒輸出地主要打他還沒距離，迫使只能把怒火發泄在你身上，這樣的隊友你見過嗎？小編見過，對面秦宓的手段可以說是十分殘忍了，碰到土鱉隊友既不能給隊友輔助，又打不出輸出，甚至還要把地主的火力全部推給隊友抵擋，這樣一個武將，我願稱之為最強坑爹。

TOP5:大喬小喬

如果說其他人的坑可能是因為各種各樣的外在因素，那麼大喬小喬的坑就是武將本身的問題了，天香和流離放在他們原本的主人身上強度都還算可以，但是組合在一起就變成了當之無愧的坑爹技能，首先是天香的問題，牌堆中的紅桃牌其實並不多。就算在軍爭場上小喬的發揮紅顏技能都能佔一半功勞，沒有紅顏的單天香根本沒有威懾力。流離這個技能更是坑爹，如果說土鱉是含蓄的坑隊友，那麼流離就是在光明正大的告訴隊友：你幫我頂著，碰到這樣的隊友極容易心態爆炸。

鬥地主是一個很有意思的娛樂場並且能夠通過贏金票來兌換各種各樣的好禮，為了您和您的隊友考慮，上面的坑爹將還是關了比較好，不然真的容易被打出內傷。
