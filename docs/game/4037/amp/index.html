王者榮耀：必出電刀的四個英雄，狄仁傑上榜，榜首竟是刺客

天天要聞
2019-12-10

摘要

王者榮耀中，無論是新手還是老司機，裝備的選擇都無疑是一門必修課，今天給大家講這件裝備，俗稱”電刀“，是大多數射手和暴擊類英雄都很實用的一件裝備，下面說的這幾個英雄，必出此裝備，逆風扭轉乾坤不在話下。

魯班七號

魯班七號是一個靠走A打天下的短腿射手，平A之後觸發的連射才是他的最大殺招，所以，魯班七號他需要攻速，移速，暴擊率，在破曉這件裝備沒有推出前，電刀是所有的魯班七號必出的裝備之一，即使現在破曉出來後，仍然有70%的魯班七號玩家還是選擇電刀和破曉的雙重組合，更能打出爆炸效果。

后羿

后羿定位射手，依靠出色的傷害和不俗的攻擊速度，在中後期可以虐殺一大批的英雄，配合大招和2技能的準確釋放有著很強的風箏能力，電刀這個裝備可以說是后羿的必選裝備，出色的攻擊速度和暴擊的加成，讓后羿在前期的發育可以得到穩定的提升。

狄仁傑

狄仁傑這個英雄在眾多射手中屬於比較偏前中期的，狄仁傑的被動技能普通攻擊可以增加狄仁傑的攻擊速度和移動速度，一技能是狄仁傑的主要輸出手段，兩次普通可以觸發一次令牌效果。二技能可以解除​控制效果，讓狄仁傑有一定的反手能力。而大招則是一個很強的單體控制，並且破甲效果可以強化狄仁傑的輸出能力。閃電匕首則非常適合狄仁傑出，無論是增加的攻擊速度還是暴擊率以及移動速度加成，都使得狄仁傑的走A時手感非常好，而且前期傷害很高。

孫悟空

猴子是王者榮耀中最受歡迎的打野英雄，沒有之一，其原因也非常簡單，那就是簡單粗暴，見誰都是無腦三棒，不是你死就是我亡！對於猴子來說，很多小夥伴都覺得宗師之力才是其核心裝，畢竟可以強化普攻傷害，但是現在的猴子更流行電刀攻速流，甚至有些大神玩家寧願不出宗師之力，也會為電刀留出裝備格！眾所周知，猴子需要快速打出三棒，並在期間穿插普攻，這時電刀提供的攻速加成就派上用場了。

還有哪些英雄出電刀更猛呢，一起討論吧。
