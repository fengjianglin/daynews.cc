Flawless：Doinb幫了我很多，因身體原因離開LPL

天天要聞
2019-12-20

在今年的轉會市場中，再中國已經征戰3年的選手Flawless決定重回韓國，並加入APK戰隊，以打野身份回歸。在這樣一支LCK新軍隊伍中，擁有五年職業選手生活的Flawless無疑成為隊伍的代表選手。

在LPL的比賽中，Flawless的表現讓人耳目一新，特別是在2019年春季賽中，隊伍獲得了第二名的成績，Flawless在野區的表現也給中國觀眾們留下了深刻印象，但成績只屬於過去，Flawless決定在2020年開啟新的挑戰。

作為挑戰者的立場，Flawless表示新賽季的目標是進入季後賽，並會為之付出最大努力。

在採訪中，Flawless回憶了上一次在LCK的經歷，那是2016年，成績略顯遺憾。2015年出道的Flawless所在的隊伍僅僅在LCK待了一個賽季，隨後便遭遇降級。

「從小雖然充滿自信，但感覺現在對遊戲的認識一竅不通。如果能知道的更多的話，那時成績應該會好一點吧。」

之後，將悲痛轉化為力量，Flawless將目標指向了LPL的舞台。對於2017年的這個決定，Flawless說：我很喜歡新的挑戰，我很希望能在不斷變化的環境中來展現自己真正的價值。

在LPL經過一年的適應之後，2018年所屬的RW戰隊獲得了LPL春季賽的第三名以及夏季賽的第四名。對此，Flawless表示：Doinb選手幫助了自己很多，剛來到中國時，雖然實力也存在不足，但是真正的意識到了溝通的重要性。Doinb選手幫助自己提高了對遊戲的理解以及中文表達能力。Doinb選手的領導能力也是頂尖的。

說到為什麼重返LCK，Flawless回答：從2019年夏季賽開始身體情況就不太好，在夏季賽期間必須得去醫院，給隊伍添了很多麻煩。感覺是在中國最遺憾的一個賽季。說到目標，Flawless表示，最終目標是進入世界賽，會以挑戰者的姿態一步一步往下走。這個賽季的目標是進入季後賽。

面對即將來到的KeSPA，Flawless表示，在中國接受採訪的時候一直會說，只要我表現的好，就能夠取得勝利，這次我的回答也是一樣。
