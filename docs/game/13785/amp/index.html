你以為次世代就只是這樣而已嗎？其實它是有分類的

首先，先簡單科普次時代，簡單科普次時代，次世代一詞起源於日本，通俗來講就是即將到來的時代。我們常說的次世代科技，就是技術不是很成熟，在社會上還沒有應用開來。由此衍生的還有:次世代遊戲、次世代主機、次世代計劃、次世代引擎等等衍生新詞。

可以把次世代定義為傳統和PBR兩種，傳統的只有顏色，法線，高光三張主貼圖，PBR你就記住，是顏色，法線，金屬度，粗糙度四張貼圖（根據目標引擎不同，輸出的略有差別）。

次世代人物建模

有什麼區別？

傳統次時代，高光主要是控制亮不亮，根本不考慮物體是怎麼個亮法？鏡子亮不亮？亮，而且會反射出周圍環境！一些嶄新的拉絲鋼之類的金屬亮不亮？會反射周圍環境嗎？傳統次時代，高光貼圖只能控制亮，不亮。而PBR則是把高光分成了金屬度，粗糙度兩張。

具體一些就是PBR是把高光範圍（Eccentricity）高光強度（SpecularRollOff）高光顏色（SpecularColor），反射強度（Reflectivity），反射模糊（Reflection Blur）統一合併為金屬強度（Metallic）和光滑強度（Smoothness），這個技術叫做 physically based rendering 基於物理的渲染。之前的做法類似於我們定義顏色使用「計算機語言」紅綠藍（RGB）定義顏色。PBR技術類似於使用「人類語言」顏色、飽和、度亮度（HSB）定義顏色。

Maya、3Dmax、Zbrush

有什麼用？

粗糙度控制的是反射不反射周圍環境（你這麼理解就行了）

金屬度控制的亮還是不亮（你這麼理解就行了）

有這兩個屬性，就能更進一步的區分你製作的玩意在觀看體驗。

次時代在國內國外的情況

受劍靈之類的遊戲影響，國內好多公司還在走傳統次時代，國外的一些3A單機大作，則是全面PBR，什麼虛幻4，寒霜3，什麼吊用什麼引擎。unity在這些國外單機大廠里，感覺被拋棄了（unity需要特殊材質才能支持PBR）。在substance painter里用2048做個東西在substance painter雖然看起來挺好，進了unity，還是貼2048的，視覺效果就很差了。所以，一定要學PBR，傳統次世代畢竟是傳統，我們是要取其精華去其糟粕的。
