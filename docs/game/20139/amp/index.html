DNF：燦爛徽章不要合成了！1月活動，25智力可變25力量

天天要聞
2019-12-18

大家都知道，DNF的低谷期活動獎勵尤為給力！更新都是慣著玩家，想要紅字書和燦爛徽章，一個字送，想要加強！無我加強全都安排上，體驗服也曝光了一部分新版本內容，盧克Hard、魔界風雲、天界戰記等新副本上線，劍宗、魔道等職業和劍魂一樣加強了，但未實裝的活動部分更為給力，史派克終於開始降低燦爛徽章的價值了！無論是過氣裝備的徽章還是現有無用徽章，都可以變為玩家需要的！（PS：因為裝備改造模式都來了）

燦爛徽章不要合了！25智力可變25力量，裝備上的也能摳下來

1月大版本的部分活動圖標已經曝光，包括涅槃之石、徽章轉換和屬性變更、武器特效解除和設置窗、新的封印界面等，涅槃之石屬性未知暫不介紹，但DNF出過功能相同的徽章道具，所以小夥伴們可要做好兩個準備。

（1）鑲嵌過燦爛的遠古裝備別分解了，比如有25力量的異界裝備（魔戰套）。大家可以通關徽章屬性變更這個界面，把魔戰套上的25力量徽章給摳下來，但只能回收一個，另外一個會損壞，燦爛等級的回收箱，5年前是80R回收5個。

（2）背包現有燦爛徽章，屬性用不上也不要合成了！比如劍魂分解時裝得到了25智力或者一堆魔法暴擊之類的，大家都是合成！基本都是浪費掉，但通過徽章轉換器的界面，無論是華麗還是燦爛徽章的屬性都能變更，但是隨機並非自選，曾經出過1次，是40R調整5次。

也就是說1月份燦爛徽章的價值會再度降低！難怪SPK最近活動各種送，至於白金徽章，不在徽章摘取和調整的範圍內，不然王室秘籍粉裝又崛起了！事實證明，DNF的左右槽白金就是平民最難打造，12月拍賣行就是最好的例子，1個隨機白金漲到了近7000W的天價，起碼在100級之後，夢想白金禮盒依舊只能靠禮包產出，魔盒都不會加入。

總結，春節版本過後就是100級了！既然徽章和增幅能復刻，那麼小號的燦爛也要跟上了，而史派克重新把徽章摘取和轉屬性道具拿出來，小夥伴們要提前準備好。
