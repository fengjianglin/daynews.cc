EstarPro幫AG“緊急特訓”？真相不僅於此！

天天要聞
2019-12-10

大家好，我是遊戲解說醬大橘，想必大家都對KPL很了解了。對KPL的各個戰隊也是非常熟悉了，在爭奪冠軍杯時EstarPro輸給了AG和JC，可是在他們打完比賽卻不見他們的蹤影，難道真的是因為與冠軍無緣所以放棄了訓練嗎？答案當然不是這樣的，除了他們自己不能爭奪他們也是很用心的在訓練， 除了在幫AG訓練以外他們也著手為冬季賽的到來，想必在秋季賽不能爭奪冠軍對於他們來講肯定是一件非常重大的事情，所以都不見隊員在直播，也沒有拋頭露面。

EstarPro為什麼要幫助AG呢？可能大家有所不知，EstarPro在四強里擊敗了QG卻無法進入冠軍爭奪，內心也是有所不甘的，想要幫助AG也是內心所向吧，不知道大家有沒有留意到，在一次直播中EstarPro的小怪獸公開咒罵了AG成員，雖然EstarPro沒有公開表示此時，但是想到是自己家隊員有錯於AG，這也是表示歉意的另一種方法吧。再者是這支昔日的王者—AG超玩會，大家都很想讓這頭沉睡的雄獅再次蘇醒，這種期盼是大家自發而成。

除了EstarPro幫助AG訓練外，也有傳聞出來說JC也在幫QG訓練，可能也是與EstarPro相同的境遇的緣故吧，AG打敗了JC內多少也會有所不滿呢，所以JC也在幫助QG訓練。不過我們拋開以上這些因素，戰隊與戰隊之間的訓其實是很正常的。

12月14馬上就要到來了，離秋季冠軍爭奪賽越來越近了，希望兩支隊伍都能夠發揮到最好，這樣才不讓買了票的觀眾們失望吧，也不要讓廣大的粉絲們失望，拼盡全力，不負眾望就好了。

現在兩支隊伍都是有人在做「緊急特訓」，你會覺得哪支隊伍的勝算更大呢？
