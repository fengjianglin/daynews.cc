哈嘍！大家好！我是你們最最喜愛的高高君,第一屆LPL全明星舉辦的非常出色，但是依舊出現了很多的問題令觀眾看得不是特別開心，其中最大的問題就是全明星正賽的機制引起了很多人的不滿，大家都非常期待看到BO3最後的一場徵召模式，但是非常可惜僅打了兩局這場比賽就結束了，其中最大的原因就是劉青松和小天的發揮太糟糕了。

特別是劉青松的表現簡直不忍直視，作為一個冠軍輔助我們一直相信他即使是娛樂也不可能出現寶藍這樣的情況，最後的結局居然是青出於藍而勝於藍，劉青松的發揮比寶藍還糟糕。對於LPL隊員糟糕的發揮，觀眾們一向是不手軟的，所以這次劉青松的發揮也受到了很大的爭議。

就和RNG小虎一樣，因為小虎的傷害特別低所以許多的玩家都把小虎調侃成了一個傷害計算的單位了，而如今劉青松或許要走上小虎的路了，因為他們的表現都是差到令人難以想像的。或許有人說劉青松這是消極比賽，但是作為一名職業選手，我們不可能相信冠軍輔助劉青松會消極比賽。

在這次LPL全明星之後，金貢在直播中遇到了非常能送的隊友，他直接開罵表示不開心：這個人一直在送送送劉青松，送的跟劉青松一樣。這一句話或許能夠引領LPL的潮流，或許以後有玩家在遊戲中特別能送，我們就可以稱他送得跟劉青松一樣。

很有趣的是有很多粉絲問金貢為什麼這樣說劉青松，劉青松他到底做錯了什麼？而金貢也沒有直接表示只是說我不知道，但是你們應該知道，那個事情你們都知道的。雖然這只是金貢對劉青松的一個嘲諷，但是劉青松這樣的行為確實引起了非常多玩家的不滿，如果這部分玩家足夠多的話劉青松或許真就能像小虎一樣引領潮流。

劉青松因為自己全明星的糟糕表現遭到了很多網友的怒罵，或許他也能和RNG小虎一樣引領潮流成為玩家噴其他隊友的一個單位，對此你們怎麼看？
