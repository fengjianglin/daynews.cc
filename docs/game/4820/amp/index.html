點個關注，每天和你聊聊電競圈趣事！

從11月19號冬季轉會期到現在，已經過去了兩周之久，雖然隔壁的LCK賽區在第一周的轉會期就如火如荼地展開了，各大戰隊的轉會消息來得非常猛烈，激烈程度讓人感覺下賽季的LCK會「大變天」，但咱們LPL賽區卻還是風平浪靜。

但這也都只是表面現象，畢竟2020年的英雄聯盟全球總決賽將在中國舉辦，各大戰隊都渴望著能夠在主場奪冠，讓隊伍的商業價值再翻上一番。

相信最近小夥伴們也吃了不少轉會期的瓜，因為在風平浪靜的表面下，各大戰隊為了備戰S10都是卯足勁在爭搶優秀選手，所以很多消息也還是偷偷走漏了風聲。

目前已經得到各位圈內大佬等無數人「實錘」的重磅消息有：SKT上單Khan加入了FPX成為首發，還有IG的ADC選手Jackeylove轉會到了蘇寧。

雖然消息的真實性比較高，但相比Khan選擇了剛奪冠的FPX，阿水選擇了在LPL聯賽處於中游水平的SN讓人感到無法理解，難道是他們給的錢實在太多了，實在頂不住誘惑嗎？

但如果你有細看目前轉會期的爆料，你會發現阿水選擇蘇寧其實是有道理的，因為目前的蘇寧光從紙面實力來看的話，已經成了LPL賽區的第一艘小型「銀河戰艦」！

雖然原來的雙C離開了隊伍，但根據轉會期的爆料，蘇寧的2020賽季新陣容可是遠超2019年。

上單：Bin

打野：H4cker；Sofm

中單：Angel

ADC：Jackeylove

輔助：SwordArt

雖然陣容第一眼看上去也沒什麼，出了阿水和越南野王Sofm以外其他人都不值一提，但通過這幾天微博的爆料，我逐漸了解到，這個上單Bin很有可能會在下賽季大放異彩。

該00後國產上單小將在2019年加入了SN的二隊，按照爆料所有，他會在新賽季成為一隊的上單。

雖然只是個新人，但Bin的遊戲風格是對線期壓制力非常強的那種，在韓服排位的這段時間，他已經在在高分段小有名氣，在排到DWG的Nuguri時，他兩場遊戲一共單殺了他8次，對線直接是壓爆了牛寶。

並且聽說他加入SN之後的試訓表現也是非常優秀，Bin連續三場都打爆了三支不同LPL隊伍的上單。

除了Bin以外，熟悉蘇寧的粉絲應該都知道，其實上賽季的替補中單Angel的個人實力也屬於非常優秀的選手，並且英雄池也是深不可測，雖然隨筆其他隊伍的頂級中單，他的比賽經驗還不夠，發揮時好時壞，但相信新賽季經過陣容補強後，他成為首發也能很好的適應隊伍。

綜上所述，上單是「國產小Theshy」，打野算是一個有勇有謀的入侵選手，中單也算是一個好苗子，下路是蛇蛇和阿水，如果蘇寧真能簽下這些選手，可以說是下賽季非常有看點的一支隊伍了。

但畢竟都還是爆料，在官宣之前，還需謹慎吃瓜。
