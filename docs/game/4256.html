續作有望了！《收穫日》開發商完成重組

2019-12-08 0條評論 8次閱讀 0人點贊

摘要

《收穫日》的開發商Starbreeze在推特宣布，他們已經成功完成了重組。CEO

《收穫日》的開發商Starbreeze在推特宣布，他們已經成功完成了重組。CEO Mikael Nermark在發言中表示過去12個月中他們經歷了一段辛苦的「旅程」。不過，好在最後他們成功獲得了債權人的信任，現在他們已經讓公司重回正軌，並且可以繼續專註於他們的核心業務「開發《收穫日》系列遊戲」。

聲明中還表示，他們希望在2020年上半年達成《收穫日》的發行協議，並且尋找合作夥伴開發系列續作。

Starbreeze是在《Overkill的行屍走肉》的發行上翻車了，由於《Overkill的行屍走肉》糟糕的銷量表現，《行屍走肉》版權方Skybound中止了和Starbreeze合作，導致遊戲只能停止銷售，登陸主機平台的計劃也得以落空。為了挽救公司，Starbreeze公司將《網路奇兵3》的版權賣給了OtherSide Entertainment公司，還把《十座王冠》版權賣給了開發商Mohawk。

Starbreeze指出《收穫日》系列是未來公司發展的關鍵所在，所以看來《收穫日3》出來應該是沒什麼大問題的了!不過還是期待一下不要翻車才是。
