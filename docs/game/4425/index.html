全明星solo：doinb止步十六強！FOFO刀妹好猛，全場壓制完成單殺

2019-12-07 0條評論 17次閱讀 0人點贊

摘要

12月7日早上的lol全明星solo賽十六進八的比賽，LPL第三位參賽的選手doinb登場。前面兩位選手的solo賽第二輪結果是，tian遭到jankos擊殺，慘被淘汰；而theshy擊敗DFM上單，晉級八強。

12月7日早上的lol全明星solo賽十六進八的比賽，LPL第三位參賽的選手doinb登場。前面兩位選手的solo賽第二輪結果是，tian遭到jankos擊殺，慘被淘汰；而theshy擊敗DFM上單，晉級八強。這場doinb的對手是FoFo，傳言也是一位即將加入LPL的選手，所以這場solo應該是兩位新老LPL中單的提前對決，看點滿滿。最終在LPL全明星上拿到了戰士組solo賽冠軍的doinb，在這次solo賽上不敵FOFO，和tian一起止步十六強！

雙方這一局的BP如下，doinb這邊ban了阿卡麗、盧錫安和亞索，選擇了自己的招牌瑞茲；而FOFO這邊ban了鱷魚、克列和夢魘，選擇了刀妹。doinb第一件就出的裝備是冰川護甲，以及一個小布甲，還是之前熟悉的內味，而FOFO則是直接出了雙多蘭。

剛開局對線換血FoFo這邊刀妹稍微有一點優勢，但是之後沒有什麼新的衝突，都在補刀發育，第一波雙方都是27刀回城，doinb做出了布甲鞋回線；而FOFO則是小飲魔刀回線。一波線推進去之後，doinb再次選擇回城，做出了小反甲。FoFo也在推線後再次回城，做出了死刑宣告。回來後雙方一波對拼，還是FOFO的刀妹優勢，doinb又被打成了半血。

FOFO再次回城做出了水銀鞋，而doinb得回城則被FOFO打斷了一下，做了一件遺失的章節，FOFO也選擇再次回城，做出了紫雨林之拳，doinb回城補充紅寶石。隨後開始縮圈，之後FOFO主動出擊，操作完美，Q上去兩段E接W直接大招，被動完美，成功單殺了doinb，拿下了這場solo賽得勝利。

那麼，對於doinb在這一局solo賽中的表現，你們有什麼想說的嗎？
