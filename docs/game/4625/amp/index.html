時空穿梭，巨魔來襲，重溫祖阿曼的4個記憶

摘要

70年代的兩個10人副本，真的給人留下了無數的影響，一個是祖阿曼，一個是卡拉贊。作為一個水T，真真正正的水T，曾經想堅挺無比，但祖阿曼的巨魔不答應。今天，我想和大家一起回顧一下祖阿曼這個副本給人帶來的感動。

70年代的兩個10人副本，真的給人留下了無數的影響，一個是祖阿曼，一個是卡拉贊。作為一個水T，真真正正的水T，曾經想堅挺無比，但祖阿曼的巨魔不答應。今天，我想和大家一起回顧一下祖阿曼這個副本給人帶來的感動。

印象一：這個副本是以巨魔作為主體怪物的副本，從60年代的祖爾法拉克之外，印象最深的就是這個副本，裡面的怪物騎著戰熊，駝著背，但當時打的時候卻很吃力，感覺那些怪物都好強力。

印象二：老一能夠變化成熊和人兩種形態，我記得他變成熊的時候給MT能上負面BUFF，我的戰士被他打死了好多回。最經典的則是他那句口頭禪：召喚野獸，你們都得死。

印象三：祖阿曼這個副本的音樂很好聽，在開門的時候，玩家要敲鑼，驚醒副本裡面的怪物，然後有一大波巨魔衝過來，感覺很壯觀。

印象四：這個副本的掉落還算經典，不過最值得紀念的當然屬於他的那頭戰熊——祖阿曼戰熊。當時，強力的隊伍在15分鐘之內，按照順序依次挑戰4個BOSS，就可以獲取這個坐騎，那時候騎著這頭熊，可真的是身份的象徵。

我的期待：也許是魔獸變了，也許魔獸沒有變，變的只有玩家，我現在每天上線，不管是正式服還是懷舊服，都找不到好的感覺，想想當初這樣經典的副本，為了這個副本，很多玩家都組的是10人固定團。青春不在，魔獸不老，我真的好希望能夠重拾過去的時光，讓我們一起再向祖阿曼進發。我是荔枝哥哥，我在地球問您服務。
