王者榮耀：更新帶來AG超玩會奪冠活動，聖誕和新版本到來也有福利

2019-12-18 712條評論 3,074次閱讀 1865人點贊

王者榮耀周2的例行更新也是來了，在這次更新中，有大量關於AG超玩會獲得秋季賽總冠軍的活動，我們一起來看看吧，

全英雄限免活動和巡遊活動是已經開啟了，然後還有冠軍戰隊的精選英雄和皮膚的限時折扣活動，這次的5名英雄分別是趙雲，曜，孫尚香，張良和蔡文姬，都是從588點券降到了488，他們的直售皮膚也是全部8折。喜歡的小夥伴可以這時候入手，時間是17號到23號。

然後就是對戰掉落冠軍紀念寶箱的活動，這個也是從17號到23號，只要參與對戰就能掉落「AG超玩會冠軍紀念寶箱」，每天最多3個，一共21個，裡面是碎片，鑽石，英雄皮膚的體驗卡等，當然運氣足夠好也有機會拿到永久的英雄和皮膚。

之後就是【為冠軍助威】消息的限時使用了，兩條冠軍戰隊的消息分別是銀河戰艦起航和AG榮耀歸來，可以說兩條消息都是非常霸氣的啊，特別是一些AG超玩會的粉絲，可以把它們設置在快捷消息中使用，時間是持續到31號為止。

聖誕的一些相關活動也是慢慢開啟了，從20號開始就可以在遊戲中通過登錄，參與娛樂模式等來收集聖誕拐杖糖，一共可以收集到40個，其中25個可以用來換取幸福聖誕的榮耀播報，其他的也能換取銘文小禮包等東西。

新版本強者之路就要來了，從18號開始也有相關的活動，分享版本解鎖圖片可以得到【探秘玄雍體驗卡自選禮包】，裡面可以選擇嬴政，曜等英雄和皮膚的體驗卡，另外還有台詞考驗活動，分別使用嬴政，白起，曜，魯班7號和莊周參與對局可以獲得銘文碎片和親密玫瑰。

最後就是金幣頭像框禮包和商城的更新了，前者是看運氣，商城的話積分奪寶中的司馬懿、關羽、冰冠公主換成呂布、李白、偶像歌手，鑽石奪寶的女媧、孫臏、典韋換成楊玉環、魯班七號、曹操，碎片商店則如上圖，還是有幾個稀有皮膚的，大家可以考慮一下，歡迎大家關注和分享，隨時給你們帶來王者榮耀的最新資訊心得福利等。
