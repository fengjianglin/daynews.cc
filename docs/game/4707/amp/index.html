電競·LOL|2019英雄聯盟全明星賽LPL選手參賽項目公布

2019《英雄聯盟》全明星賽將於12月6日-8日在拉斯維加斯火熱開戰，隨著比賽時間的不斷臨近，Thy Shy、Uzi、Doinb、四名代表LPL賽區的參賽選手將要參賽的項目業已公布。

其中The Shy將參加1V1單挑賽、雲頂之弈慈善賽、中韓對抗賽；而Uzi會參加1V1單挑賽、中韓對抗賽、雙人共玩模式、5V5表演賽。S9冠軍FPX的兩名隊員Doinb將參加1V1單挑賽、無限火力模式、中韓對抗賽、5V5表演賽，Tian則會加入1V1單挑賽、雙人共玩模式、5V5表演賽。

12月6日上午8點，屆時全球各大賽區明星選手以及社區名人匯聚一堂，將為全球《英雄聯盟》玩家帶來一場前所未有的視覺盛宴，喜歡的玩家一定不要錯過！

單機·家用機|PS4冬季宣傳片涉嫌抄襲下架

前不久，索尼互娛亞洲日本頻道為了宣傳冬季遊戲，發布了一個遊戲新作的混剪MV《夢の中へ》第二彈Lineup MV。

而在最近，新聞網站catsuka在官推中表示，這段宣傳片涉嫌抄襲了其他動畫，並直觀的指出了其中抄襲片段，包含了《宇宙小子》、《特別的她（FLCL）》、法國Gobelins動畫學院作品《MAILED IT》和《ORPHÉE》，廣告片《lune of ATlantis-moonlight》等多部動畫。

有趣的是，視頻的出品方Eallin Animation已經發布了公開聲明，將鍋甩給了這位導演：”我們並不知道Kevin Bao的作品是抄襲的。我們立即停止了任何與他相關的作品發表，並聯繫了Kevin請其作出回應。”目前索尼方面尚未發表評論。

單機·家用機|《暗黑血統：創世紀》IGN 8.7分

《暗黑血統：創世紀》是《暗黑血統》系列的衍生作品，將於當地時間12月5日正式在Steam平台和Stadia解鎖，2020年2月14日登陸PS4、Xbox One和Switch平台。

本作採用了上帝視角，玩法上與《暗黑破壞神》類似，玩家需要在地下城中消滅怪物收集戰利品。今日IGN為《暗黑血統：創世紀》打出8.7分，並表示該遊戲雖然有少許不足，但整體質量與主線作品不相上下。

在IGN編輯的心中，《暗黑血統：創世紀》在作品的打磨和潤色上相比主線作品有所欠缺，但它仍是該系列最好的作品之一。在15個小時的遊戲中，該作的戰鬥、解謎、豐富的探索以及進階機制，出色而且平衡，並為玩家提供了足夠的遊戲動力。

最後，該評測者還表示，《暗黑血統：創世紀》全新的遊戲風格或許不是該系列的新標準，本作還有很多可以挖掘的東西。

此外，在M站的評分方面，共有31家媒體對遊戲做出了評價，均分77分，其中好評22篇，中評9篇，沒有差評。

網遊·手游|《爐石傳說》或將迎來希爾瓦娜斯皮膚

據外媒報道，《爐石傳說》職業選手Rdu”Dima Radu “在近日的直播中泄露出，希爾瓦娜斯·風行者即將作為遊戲中的新皮膚推出。

作為一名職業選手兼主播，Rdu”Dima Radu “目前賬號上的所有內容已經解鎖，而在他瀏覽英雄時，出現了尚未發布的內容，希爾瓦娜斯的身影。

這也並不是希爾瓦娜斯第一次出現在《爐石傳說》中，曾在2017年時，它就以中立傳說隨從的身份加入遊戲，但在隨後的版本更新迭代中加入榮譽室，無法再次獲得。

而這次希爾瓦娜斯是否會以獵人的皮膚出現呢？雖然外媒PC Gamer已經聯繫了暴雪試圖確認這個消息是否屬實，但暴雪尚未回應他們。
