懷舊服天賦：冰法最強？火法並不差，也傷害爆炸！

天天要聞
2019-12-19

懷舊服到現在已經要半年了，距離第三階段更新也大概就一個月多吧？在這段時間裡面法師就業還算OK。但如何讓自己輸出更高同時又不會花費太多錢就是一個問題了，因為當前版本中冰法算是一個主流。

最基本的冰法套路就是這樣，也是主流的冰法天賦。這套天賦基本上算是標配，不但有著不差的生存能力還有續航。關鍵是有一個超大爆發可以瞬間重置所有冰法技能的CD，再來一波技能**。當然，最重要的還是因為冰法簡單，從升級到團本都可以說是傻瓜式的程度。也導致了很多增加冰霜傷害以及法術傷害的裝備部位價格非常昂貴，畢竟如今的法師都是冰法情況下就靠著冰傷以及法傷裝備來提高dps了。但你有沒有想過並不是很多人玩的火法，其實研究一下傷害也是非常可觀的？

和冰法完全相反，火法基本上是貫徹了暴力美學，而且放棄了一切的生存能力。當然火法之所以不受歡迎很大部分在於火球這個技能，大部分認為火法就如同冰法一樣無腦111寒冰箭一樣搓火球就完事兒。說是這麼說也沒錯，但是火球讀條即使天賦點滿也有3秒的讀條，耗藍也要接近400。傷害雖然很高，但是高讀條藍耗在當前版本中顯然是完全不適用的。

所以我們不如換一個思路，放棄追求傷害而追求暴擊來輸出。可能你又會說，暴擊變相提高了法師輸出也沒毛病啊？這樣一來和沒換個思路有什麼差別？重點就在於這個天賦上，暴擊以後會燃燒目標在4秒內承受此技能的百分比額外傷害。

然後再配合3級的火焰重擊，就是額外提高了火法技能的6%暴擊率。再配合3分鐘CD的終極大招更是可以再讓你的火焰法術暴擊幾率提高10%。雖然只有3次暴擊機會，但是毫無疑問這個技能效果還是非常不錯的。

而提高火法傷害的靈魂天賦就是這個強化灼燒，只要是可以灼燒的火焰法術都可以100%額外造成灼燒，關鍵是可以提供額外的灼燒易傷效果。每層3%最多5次那就是這個強化灼燒直接提供了15%的增傷！此外你可能會說這樣一來我的輸出不是很高，很容易OT嗎？而且沒有續航技能我不是很容易沒藍嗎？實際上完全不是。

火法的仇恨問題確實是很重，但是一般來說一場戰鬥下來你釋放大技能的頻率並不算高。大部分情況下你都是依靠放可以灼燒的技能來打灼燒輸出，而且天賦點了燃燒之魂以及元素大師以後再配合這個玩法，不但仇恨降低30%關鍵是火法暴擊就返還技能30%的mp，因此仇恨和MP問題這個事情基本上問題不大。

至於裝備選擇方面也是非常明顯了，雖說讓你放棄傷害但也不是說無腦放棄。畢竟你暴擊再高沒有傷害為基礎，即使能暴擊也不能打出很炸的傷害吧？所以選擇裝備方面我建議你直接用法傷裝備+暴擊裝備搭配。比如說大法師之袍這個衣服就非常不錯，不但加了法傷關鍵是還提供了1%的裝備，當然還有個聊勝於無的回藍主動雖然5分鐘的CD。
