PDD拿下LPL名額，eStar過去的“416事件”究竟發生了什麼？

2019-12-13 294條評論 4,738次閱讀 292人點贊

​​從昨天開始各個大V就在開始預熱今天的大新聞，起初虎喵以為只是最近的熱門話題「阿水去哪兒？」但是越來越多的大V發博預熱，虎喵又聯想到前幾天doinb直播說的新中單，按照他的形容實在不像是兮夜或者玉米。所以以為發展方向真的會是大家猜的外援身份的變化，沒想到竟然是PDD拿下LPL名額了！

PDD

新隊伍名為eStar，是PDD與武漢旅遊發展投資集團共同注資的戰隊，目前還未宣布首發名單不過應該是與YM合併可能性比較大。再想到前不久才買給RNG的小龍堡，PDD手裡還藏著更厲害的野王嗎？

今天的官宣因為經過各大V的預熱，所以大家都抱著吃大瓜的心情，也就是因為抱著很大的期待所以熱情並不是很高。

而且早在去年的四月份獲獎的eStar就與LOL圈有著不小的矛盾，也是為什麼大家都說是拳頭贈送的行竊預兆的原因，下面我們就統稱為「416事件」

事情還要從18年王者榮耀KPL聯賽戰隊eStarPro贏下了一場春季常規賽說起，一位電競圈大V發布了eStar諾言的個人照文字為「諾言顏值，可愛小奶狗風，繼草莓後又一位帥哥」

本來這件事並沒有什麼問題，只是重名以及帶上了草莓的名字，就算這種蹭熱度的行為引起了粉絲不滿也只是爭吵一番。

壞就壞在eStarPro俱樂部編輯了4月14日的一條博文，帶有「辣個男人」字眼，雖然在事後也發布了道歉也難免掀起LPL與KPL粉絲之間的「貼吧戰爭」。

當時的圖

雙方爆吧後KPL毫無疑問的輸了，也被大家扒出eStar諾言女友其實還是EDG粉絲增添了一些喜感。最後「416事件」以各俱樂部、解說以及電競媒體申請「諾言超話」結束，即使後面超話中還是有了eStar諾言的名人動態也經過溝通拿回來了。

官宣過後大家討論的點還有eStar俱樂部的背後是否為國企，虎喵特地找了一下並不是，而是國資。

不過這一查才發現原來人民日報投資了LDL，原來電競涉獵的範圍已經那麼廣了，好奇會在S10擦出什麼樣的火花~​​​​
