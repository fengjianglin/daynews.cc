亦正亦邪亦難分 淺談在陣營設定中加入善惡元素的影響

2019-12-06 0條評論 18次閱讀 0人點贊

摘要

PVP陣營簡介與善惡初談PVP玩法是MMORPG中非常重要的一個組成部分，指的是玩家與玩家之間在方方面面展開較量並決出勝負，而這些較量與戰鬥往往是發生在玩家群體之間的。

PVP陣營簡介與善惡初談

PVP玩法是MMORPG中非常重要的一個組成部分，指的是玩家與玩家之間在方方面面展開較量並決出勝負，而這些較量與戰鬥往往是發生在玩家群體之間的。這些玩家群體有的是自行組織，例如幫派或家族亦或是軍團；有的則是在官方的陣營設定下生存，不同陣營的玩家因所屬不同展開競爭和對抗，最終演化出了一幅幅遊戲戰爭群像。

而既然有不同的陣營，那麼官方自然就會給予他們不同的設定。就比如《劍靈》中的”天下雙勢”，武林盟和渾天教這兩大勢力都以”平定亂世建立新世道”為口號，但在具體問題上兩者有著很多衝突，最終這兩勢力作為遊戲中的對立陣營來到了玩家眼前。但既然有了設定和背景故事，那故事中的對錯善惡自然也是隨之而來。

雖然廠商並不會直接告訴玩家陣營的善惡之分，但玩家可以通過陣營的配色、背景故事以及主要人物的性格來判斷，最後玩家依然會對陣營的善惡做出結論。最終這些因素會對玩家PVP玩法的方方面面產生影響並且輻射到整個遊戲環境中。而我們今天的話題就是對這些影響進行拆分，從玩家選擇、玩家行為以及玩家體驗三個方面逐一展開。

玩家會根據陣營背後的善惡元素進行選擇，最終影響陣營平衡

我們首先來說說玩家選擇，通常來說，遊戲中的陣營是任由玩家選擇的。而玩家在選擇之前自然會考慮方方面面，比如所在陣營的強度，PVP體驗或氛圍，是否方便自己做任務等等。就比如魔獸世界60年代時，許多玩家選擇部落的原因有亡靈意志的強大、在部落跑副本更加方便等因素，畢竟當年聯盟跑副本確實頭疼。

而另一方面，陣營背後的故事也是吸引玩家來投的原因之一。劍網三中浩氣盟多以正派角色出現，從背景故事來看也是傳統意義上大門大派組成；而惡人谷在遊戲背景中則是由一群真正的惡人組成，谷主王遺風更是有屠城之惡名。那麼兩大勢力在傳統意義上的正惡就非常分明了，玩家在選擇時也會自己思考。

最終玩家的選擇會導致遊戲中陣營人數的不同，在PVP活動中人數始終是一個非常關鍵的因素，陣營的勝負也與人數息息相關。而獲得勝利的一方將吸引更多的萌新加入，最終達成一個循環。不過這裡多說一句，隨著時代價值觀的變化，善惡的內涵也有了區別，背景為”惡”的陣營或也有正道之處，而正派陣營背後或也有雞鳴狗盜之事。

陣營背後的善惡系統也將影響玩家的遊戲價值觀，最終影響玩家的行為

不過玩家與陣營之間其實也是個雙向的關係，一方面玩家會因為不同陣營間的各種元素做出選擇；而另一方面玩家所選的陣營也會通過背景故事、陣營任務等方式潛移默化影響玩家在遊戲中的價值觀，而價值觀自然也會影響玩家在遊戲中的行為。雖然做出特別”中二”行為的玩家不多，但陣營善惡因素的影響多多少少還是存在的。

就拿當年希爾瓦娜斯使用瘟疫來說，聯盟玩家站在自己的立場出發自然是大肆譴責並聲稱這與生化武器沒有區別，是無比邪惡的；而部落玩家則是以守城、常規武器、遊戲原本背景出發與聯盟玩家舌戰，兩邊可以說是罵出了不小的風波。不過說句題外話，這場爭論也讓人看到了網癮少年的知識水平還是不錯的，思辨使人明智。

而在這一場辯論中我們也可以看到，不同陣營確實能夠將自己不同的善惡觀傳遞給玩家並對玩家產生一定的影響。在現實生活中，如果問一個沒有玩過魔獸的玩家”使用瘟疫是否正確”，那麼答案自然大致相同。但從遊戲的整體背景以及玩家所處不同立場而言，這個問題的答案將五花八門。筆者本人也建議大家理性看待。

有利於豐富玩家PVP體驗，增強代入感

那麼前面我們已經說過了，玩家與陣營之間的是一個雙向的關係。那麼在玩家認可己方陣營善惡觀並受其影響的情況下，玩家對於己方陣營的認同度將大大增加。在這種情況下玩家很容易產生一種”集體榮譽感”，不少的玩家會因為這種情感而全身心地投入到與陣營相關的PVP活動中，最後不管輸贏，至少打了個酣暢淋漓、打了個無比爽快。

這種代入感拉滿的PVP體驗自然是非常少的，畢竟PVP體驗好且陣營設定有吸引力的遊戲不多，而相對老油條一點的玩家對這種體驗更是不屑一顧。但我們把時間線拉長一點會發現多數情況下老油條還是佔少數，大部分玩家都被這種極有歸屬感的PVP體驗所吸引。或許正在看文的你也經歷過一腔熱血不回頭，團長嘶啞不收聲且不斷跑屍的歲月。

善惡難尋，心正則善在，心不正才是惡之根源。
