《灰燼之人》—姍姍來遲的Lowpoly類魂作品

2019-12-17 450條評論 4,553次閱讀 1463人點贊

前言

「魂」系列可謂真的是遊戲界非常神奇的產物。它自誕生之初，被人視作難度到無法下手的奇葩作品，到被玩家們挖出來重新審視後發現了它並非像想像中那麼糟，再到從只有一小撮玩家星星之火可以燎原般燃遍整個遊戲界。眾多玩家從一開始的對它的嗤之以鼻，到自己的抖M之魂開始覺醒；宮崎英高從原本難以有公司願意接納的大齡誕生程序員，到成家立業並升為From software董事長；遊戲界對它的復古式設計思路的各種質疑，到將其各大各大廠商對其機制的學習模仿……

在經歷了《惡魔之魂》，《黑暗之魂》三部曲，已經《血緣詛咒》的洗禮後。From software不願自甘墮落為一個要將自己一個優點榨乾誓不罷休的辣雞廠商。而是逐漸開始嘗試挑戰自己的新的領域。TGA上《只狼》摘得桂冠不僅僅時處於遊戲自身的優秀素質，從某種程度上要可以看成是對於早已成為現象級遊戲卻一直只能陪跑「魂」系列的補償以及對From software以積極進取的認可。

From software雖然將重心放到了尋找新突破方面，但是「魂」系列自身優秀的模式以及作為遊戲界的一個熱點，依然有著不少廠商的依然要來蹭一蹭熱度。早期有著的《墮落之王》《迸發》這樣的迷弟之作，後面有著《暗黑血統3》《星球大戰絕地：隕落的武士團》這樣的機制借鑒之作，其間還穿插著諸如《鹽與避難所》《Unworthy》《神之褻瀆》《黑暗獻祭》等一大票2d獨立作品。

魂系的熱度不是一蹴而就的，它熱度散去自然也不會那麼容易。

這不，又雙叒叕一款類魂作品出現了。

遊戲體驗

作為類魂作品本作在公布之初同樣吸引了不少眼球，不過稍微可惜的是本作限時有些在epic平台獨佔，待獨佔期結束後才來到用戶基數要大不少的steam平台。如今《Ashen》姍姍來遲，終於讓期待的玩家們對其可以一睹全貌。

《Ashen》這款作品，關從名字上一聽就很容易然讓將其與「魂」系列聯繫到一起，明顯與《黑暗之魂3》中所著重體現的薪柴與餘燼對應。

一進遊戲後的立馬躥出一個無臉人估計會讓人不少沒有心理準備的人小小震一下。待稍微冷靜一下後開始懷疑是不是自己的電腦載入速度慢了導入角色的模型還沒有載入出來，然而事實上角色的模型就是如此。本作畫面表現方面採用了類似的《風之旅人》《BELOW》等作品的Lowploy風設計。玩家大體可以根據個人喜好設置一下角色的性別，職業等要素。在正式進入遊戲後，遊戲視角將被來遠，這時角色與同樣Lowpoly風的場景融為一體，將顯得不那麼違和，遊戲中一些遠景的所表現意境的震撼力也還是相當不錯的。

滿滿既視感的鍵位布局與操作

遊戲的操作與手感方面會比較接近與《黑暗之魂2》，但是與它有有著很多不同之處。LB舉盾防禦，RB輕擊，RT重擊打，這樣的鍵位布局對於「魂」系列玩家來說簡直再熟悉不過。不過不同於的地方在於。玩家不管裝備的是輕型盾還是重型盾，在舉起盾牌的狀態下LT所對應的是都將是重擊，而非使用彈反。也許本作是為了為了突出角色們的粗獷，玩家角色繞道敵人背後的背刺功能也是沒有的。不過本作所搭配的震動效果還是不錯的，玩家並不是只有在受到敵人攻擊時才會體驗到震動，當發動重擊，會從高處跳下也已經可以獲得震動反饋，可以給玩家帶來不錯的代入感的。

B鍵跑步，X按鍵使用裝備的道具以及讓人眼熟。不過Y鍵所對應的功能並不是雙持武器，單手持有武器的切換。而是對應攀爬/跳躍功能。玩家在本作中所操作的主角不會受到「不死人膝蓋無法彎曲」這一「詛咒」的影響。主角不僅可以攀爬，還可以自由跳躍，再結合上跑步功能，為主角在這個幅員遼闊的世界裡的趕路省下了不少時間。主人公也不會像魂系列主人公那樣，跳進水裡直接下沉game over。在本作中主人公會有根據角色的所剩餘的體力給出一段可供掙扎的時間。玩家如果看到跨度太的河流可以活用這個技巧渡河。不過玩家如果玩家在起跳前還請權衡一下角色的體力值以及河面的寬度，否則不小心該讓主角淹死就得不償失了。

本作的在死亡懲罰方面與也《黑暗之魂》系列比較相似，不要看角色跪了東西散落一地，但實際上只會讓主角的「金錢」掉落在原地（如果是掉下懸崖，或者掉進水裡淹死）主角的「金錢」會遺落在岸邊。本作的「金錢」為了與主題對於叫做「煤渣」。玩家在解決掉敵人後會獲取部分煤渣作為收益。在玩家在探索的過程中也會或裝有煤渣的小布袋，這個道具的設計與「魂」系列的可以用來捏的魂相對應。玩家在手頭需要有大部頭支持正好差點款項時，可為為玩家填補上尾款還是蠻方便的。角色不慎掛掉後重生點的祭壇與「魂」系列的篝火相對應，不過從祭壇在使用上也進行了一些小的改進，玩家並存一定要在祭壇初坐下才可以獲得血量的恢復以及血瓶的補足。玩家只需要從附近路過即可自動補足，這對於直接推圖，或者速攻的玩家來說還是可以省下不少時間的。

玩家在冒險中遇到其他友好的NPC可以帶會自己的據點，這一點是設計比較容易讓人想到《黑暗之魂2》的如蜜。據點隨著新NPC的加入也將解鎖物品買賣，打造武器裝備，附著符文等其他新的服務。

相似卻被大幅簡化的諸多要素

通過方向左右按鍵的切換左手，右手武器，通過下方向鍵切換以裝備道具。用來補充主角生命數值的每次樹汁液壺的設計依舊讓人既視感滿滿。不過本作有些同樣可借鑒的部分卻忽略了，這些要其實也是可以考慮加入其中的，讓遊戲操作體驗方面更上一層樓。除了前面提到省去的彈返要素外，本作中武器被直接被設定為了單手武器和雙持武器，玩家無法對喜歡的武器自由切換單手持有還是雙持。主角的攻擊招式方面相對魂系列也做了不少簡化，《黑魂之魂3》中的各種絢麗的戰技姑且不談。跑動攻擊的輕攻擊和重攻擊完全一樣，裝備雙手武器後直接無法防禦，這樣的小小細節其實還可以在稍做挖掘，從而將戰鬥系統打造得更加細膩一些的。

而負重數值，加點等原先需要考慮到要素則本作也沒有加入。正如同本作的Lowploy風的畫面那樣，也是由於團隊的規模有限，也許是製作團隊自己的設計理念，本作的整體的機制設計是相對要短平快不少。

種類不算多但還是有個性的敵人設計

本作登場的敵人雖然會讓人時不時有重複感，不過敵人角色的設計方面還算比較有個性的，常見的流浪者在見到主角前一半會席地而坐烤火或者休息。獵手會佔據高處準備通過標槍發動偷襲，手持碎金棒的大個敵人發現主角衝過來因為龐大的身體直接搞得地動山搖，醜陋且富含劇毒的巨大蜘蛛會攀附到高處趁主角路過時突然跳到頭上進行襲擊……

BOSS戰中的BOSS們在表現力上則更加豐富。例如拿吸光怪這個BOSS舉例，一身黝黑的他身形上雖然並不具備太強的震懾力，不過佔據黑暗地形優勢的他，結合自身超高的移動速度絕對是個難纏的對手，他可以靈活地躲避玩家的揮擊，同時進行反擊，通過上竄下跳讓玩家難以尋其蹤跡，再不時通過推倒石牆發起突襲，或者鑽入洞口恢復體力。即便是先前遊玩過「魂」系列的玩家在初次見到這個怪物應該也會被搞得暈頭轉向。

同伴系統

本作的難度雖不能算太低，不過「同伴系統」機制與可以很好的抑制初次嘗試本作玩家的受苦程度。玩家在探索的過程中會有一個與相關的NPC跟隨為玩家提供輔助。本作的敵人雖然喜歡抱團襲擊甚至懂得遠近配合。不過玩家與NPC通過親密無間的配合下通過彌補出招的空襲時間，同樣可以將敵人打的沒有還手之力。而通過搭建人梯的方式讓同伴到達高處的交互設計。作為同伴的NPC實力固然強力，不過玩家對於同伴也不用果斷依賴。AI所操作的同伴有時並非那麼智能化，玩家從剛剛從據點出來，或者是玩家通過地形較為複雜的地形後NPC往往是不能第一時間趕到現場的，如果是水面類的地形，同伴NPC甚至會自己掉下去，把自己給送了。玩家如果一心想著自己有著一個強勁的打手小弟而一路高歌猛進，那麼則經常得做好被被伏擊的和幹掉的準備。在面對BOSS或強力敵人時同伴NPC在戰鬥上也不少會有力有不逮，被打到跪地不起的時候。此時玩家還需要在其頭上的瀕死狀態圖標消失之前抓緊時間去救上一把。

地圖與任務機制

本作的任務有著較好的引導性，任務會分為主線和支線連兩大塊，每個需要完成的任務會通過黃色小標記在地圖上標註出來，而沒有完完全全把記路的任務完完全全交給玩家自己的大腦，玩家不需向以前玩「魂」系列作品時要把所有的路線動摸索地比自己家的後院還要熟悉，玩家之需要按照主線支線任務列表旁附帶的地圖自行路線即。遊戲的地圖則走了場景開發世界的路子，整體上一平鋪為主，再點綴上高塔和洞窟，為探索到的地方會被戰爭迷霧覆蓋，隨著玩家對遊戲的不斷探索，即可不斷解鎖出更多領域。

支線任務存在的另外一個好處在於給玩家一個類似於刷級的緩衝期。玩家如果如果暫時無法戰勝當前主線中的某個相當難大的BOSS部分，多做點支線，通過獲得的煤渣提升樹汁壺的攜帶上限，給自己多按幾個不錯buff的符文，或者尋找到幾件新的趁手兵器。玩家再次去調整先前的力所不及的攔路boss時，情況將大有轉變。還有一個比較有意思的設計點在於玩家沒完成任務後所獲得的貴重品並非像往常遊戲中那樣在交付任務時默認上交，系統將戰利品的去留交給了玩家自行選擇。

遊戲存在的一些小缺陷

  1. 本作當前存在部分卡bug的場景，如果不小心跳進不少山體的夾縫交錯出，那麼想要出來走幾乎是不可能的，這能陷在裡面各種打轉。當然玩家也並非是一點辦法也沒有，使用膽小鬼的骸骨這個道具即可將主角直接傳送。
  1. 遊戲地圖的引導性質其實還可以進一步的完善，遊戲中的地圖更偏向於是gps式的俯視圖，在某些山脈初有可以通過的通道之類的可以進一步引導玩家走正確路線的地方，本作並沒有加加以區分。如果通過顏色深淺等方式對地圖進行區分，那麼應該可以減少不是玩家繞路的時間。

總結

本作最為有一款類魂之作，它給人的總體感受上是一款簡化版的魂。可能由於時間成本有限等方面的問題，本作沒有在一些細節上作出太多的挖掘。不過對於不那麼喜歡去翻你那一堆堆的數值，或者對加點有強迫症的玩家同時也是福音。自帶的同伴系統進一步降低的本作的難度，則讓更是想嘗試類魂遊戲卻對較高受苦有抵觸的玩家有試一試的慾望。對於接受Lowploy風且喜歡魂系列遊戲的的玩家本作還是值得嘗試一下本作的。
