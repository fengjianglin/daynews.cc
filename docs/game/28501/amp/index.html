各位玩家晚上好，本肥喵今日繼續和大家一起分享遊戲熱潮！

近期文章起已經進入到新的模型分享板塊，即遊戲內的模型正視圖，只為帶來簡單直觀的效果。

上期已經開始進入暗夜精靈，暗夜精靈可以分為三部分，第一部分是以泰蘭德為主的艾露恩姐妹會及哨兵部隊，第二部分以瑪法里奧為主的德魯伊及塞納里奧議會，第三部分為荒野眾神生物，本期主要介紹瑪法里奧為主的德魯伊。

一、瑪法里奧-怒風

戰役英雄，大神棍…….咳咳，不是，是大德魯伊。上古時期暗夜精靈貴族上層精靈在研究永恆之井的魔法力量時，瑪法里奧卻向海加爾上的半神塞納留斯學習德魯伊之道，成為艾澤拉斯第一個德魯伊，而後言傳身教，教習暗夜精靈男性習德魯伊，在夢境之龍伊瑟拉在諾達希爾賜福後，暗夜精靈能夠自由出入翡翠夢境，瑪法里奧也帶著大部分男性暗夜精靈進入夢鄉，直到薩特之戰和流沙之戰。

瑪法里奧重製版模型

瑪法里奧的其中兩名弟子，萊拉爾-焰牙和范達爾-鹿盔，前者成為幾千年後影響吉爾尼斯的狼人詛咒源頭——頭狼，瑪法里奧因此建立塞納里奧議會觀察控制，後者則變成火焰之地的管理者，統領著背叛塞納里奧議會的烈焰德魯伊。這兩件事，瑪法里奧或多或少都有著責任（萊拉爾摯友的死，范達爾鹿盔兒子的死），然而慶幸的是另一知名弟子布羅爾-熊皮還沒搞事……

瑪法里奧（沒有鹿角）重製版模型

在原版中瑪法里奧有三個模型，名稱為瑪法里奧、瑪法里奧（沒有鹿角的）和法里奧，三個也各有差距，前兩個都是坐騎，法里奧則是步行，而前兩者的差距也在於頭部的差距。如今重製版也許是因為還未完全做完，當我拖出三個模型後，發現是一毛一樣，沒有任何區別，期待正式版的更新吧。

二、猛禽德魯伊

對戰單位，俗稱「鳥德」，暗夜戰術「吹風流」的核心單位，利用技能「颶風」控制敵方核心輸出單位，諸如劍聖、山丘、惡魔獵手等，而另一技能「精靈之火」也能減少敵方護甲，為友軍輸出提供便利。兩個形態，暗夜精靈和風暴烏鴉。

三、利爪德魯伊

對戰單位，俗稱「熊德」，暗夜戰術「熊鹿流」核心單位之一，擁有續航技能「生命恢復」和變成熊後的強力輸出能力。兩個形態，暗夜精靈和熊。
