王者榮耀：韓信的跑路秘訣，溜著五個敵人跑

天天要聞
2019-12-18

韓信被稱為「韓跑跑」，他有很強的位移能力，經常能溜著對方轉。

玩韓信就要四處搞事，給敵人帶來兵線壓力。甚至在KPL賽場也發生過韓信憑藉超高機動性，一己之力偷家翻盤的事。

在技能介紹過後，我會著重講講韓信溜人的幾個關鍵位置，大家可以仔細記一下。

技能介紹

被動技能|殺意之槍

韓信連續普通攻擊的第四次會將敵人擊飛零點五秒。

這裡要注意，普通攻擊的間隔超過兩秒，將不算連續普通攻擊。

此外韓信每次釋放技能命中敵方目標將會增加50%的攻擊速度，持續二點五秒左右。

一技能|無情衝鋒

韓信向指定的範圍內跳躍，並將範圍內的敵人擊飛，並且在五秒內可以再次釋放一技能。但是沒有擊飛效果。

二技能|背水一戰

韓信的二技能是向反方向進行一段位移，並且強化三秒內的下一次普通攻擊，強化普攻為橫掃狀態，距離非常遠，傷害非常高。擁有百分百的AD加成。

三技能|國士無雙

韓信的大招是站在原地揮舞長槍四次，第四次可以將敵人擊飛。

此技能的精華就是擁有霸體效果可以躲技能，並且可以減傷30%，還能控制敵人，大招配合和蒼穹使用可以減傷80%。

韓信的大招範圍其實是很大的，甚至連身後的敵人都可以打到。並且最後一下擊飛的作用範圍要比前三下傷害的範圍要大。

連招技巧

韓信連續普通攻擊時，釋放完一技能過後，他的連續普通攻擊依然存在。如果連續普通攻擊敵人時釋放了二技能，那我們的連續普通就要重新計算，這是韓信的細節，一定要記住並且掌握的細節。

如果距離夠短，二技能直接突臉橫掃一道，一技能挑起來，砍一刀，接大招。如果別人跑了，我們一技能追，如果打不過了，可以一技能用來逃跑。

如果敵人距離我們有點遠，可以二技能拉近距離，一技能挑起二技能的強化普攻橫掃依然能A出來，橫掃瞬間接大招。接著看情況，一技能追擊或逃跑。

這套連招的好處在於你快速打完一套的過程中，別人根本沒有機會把你控制住。

在前期傷害不足的情況下，這種連招在前期是很難秒掉一個人的，所以前期抓脆皮。我們推薦多打普攻。優先二技能近身接橫掃，一技能挑起後瘋狂普攻，別人逃跑繼續一技能二段追擊，爭取把第四下普攻擊飛A出來，然後再瞬間接大招。後續用二技能追擊即可。

但是實戰中也不要過於追求連招，畢竟三段位移，用起來還是比較靈活的。

跑路攻略

虐菜局韓信可以大殺四方，段位高了之後，韓信經常要帶線牽制。跑路是常態，溜著對方五個人跑，又不能死。你必須很熟悉峽谷的地形。

那些牆地方能跳過，哪些牆不能，要了如指掌。

比如上路一塔旁這個常見偷襲位，別人打河蟹的時候，直接一技能可以過去挑起。打完一套後還能一技能二段跳回來。

二塔位置的這堵厚牆，一技能可以來回跳，絕大部分英雄的位移都過不了，只能看著你絕塵而去。

狗熊這個位置也是，卡住牆角，一技能輕鬆來回，逃跑、偷野都很有用。

敵方一塔旁這個位置，可以偷襲塔下殘血，但是能去不能回。

龍坑旁的草叢也是能讓韓信在龍坑與草叢間來去自如。利用好了逃跑偷襲都有妙用。

這些位置都需要大家到訓練營多多練習，把逃跑練出正面剛的氣勢。

裝備及銘文

韓信後期要把打野刀換保命裝，或者鞋子不要都沒問題。保命裝的兩換三換要事前練熟。

韓信前期不要沉迷於抓人，能刷就刷，路過才抓。前期一定不要丟了藍爸爸，有了就刷。

偷野是韓信必須要做的，韓信清野快，並且逃跑能力強，一堆野怪是一百塊，那麼一來一去就是兩百塊的經濟，跟殺一個人頭差不多。

大小龍也要打，能偷則偷。韓信搶龍也是一把好手，二技能進去，橫掃的同時放懲戒，你對大小龍的斬殺線就能比對面打野高。把握好血量成功率不低，搶不到還能跑。
