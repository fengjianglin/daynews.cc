遊戲重製版是不是炒冷飯？玩家褒貶不一，廠商思路目的很簡單

2019-12-25 46條評論 1,512次閱讀 512人點贊

人都有懷舊情懷，遊戲玩家們也不例外，當這些老玩家們打開心中的那個神作卻會發現他不像你心中的那樣完美了，從畫質到遊戲引擎都遠遜於現在的遊戲。當然這也是有理由的，畢竟隨著科技的飛速發展有的時候差個十年技術就會差個3代。

不知從什麼時候起，遊戲廠商們或是為了滿足玩家的懷舊夢或是為了更好的圈錢，都選擇了開啟遊戲的重製版，那麼玩家對重製版的態度是什麼樣的呢？

面對那些為了撈錢而出重製版的遊戲，玩家們一向是抵制態度的，比如《獵天使魔女》這款遊戲，玩家們對這款遊戲的評價是相當高的，無論是可玩性還是劇情，每個方面都收到了玩家的好評。但問題在於，遊戲公司只是一昧的出重製版並沒有對劇情或者玩法進行更新，而玩家們對畫質的期待遠遠沒有對玩法和內容的期待大，這才是玩家們對出重製版抱怨的最大理由。

但對於那些真心為玩家遊戲體驗著想的遊戲公司開發的重製版，玩家們卻紛紛高呼真香！就在今年，魔獸世界就開放了懷舊版，讓不少AFK的玩家選擇重新回到艾澤拉斯的世界回到當年自己為之奮鬥過的事業當中。看到好友列表裡曾經灰暗的頭像一個個亮起,世界頻道中討論著的開荒攻略,無數玩家都與曾經的好友重新建立起了羈絆，玩家們的青春也回來了。

《劍網3》是當前少有的網遊開發重製版的遊戲，不同於其他網遊廠商選擇開發遊戲續集，畢竟開發遊戲續集對於遊戲廠商來說又是重新撈金的機會。西山居選擇照顧玩家的情懷，在原有的遊戲內容上重新換了個引擎，雖然在本質上這仍然是換了一款遊戲，但玩家的數據毫無改變，玩家愛著的劇情也沒有發生變化，反而隨著劍網3重製版的越來越火爆，西山居在劇情和玩法上面更新換代的速度也足夠讓玩家們滿意。

近期劍網3開啟的新資料片凌雪藏鋒中的劇情就讓玩家們對策劃的感情變得更加複雜了起來，在新劇情中，最受玩家們喜愛的兩個npc毛毛和莫雨都收到了來自反派謝採的傷害，玩家們一邊感動的同時一邊又對策劃「恨」得牙痒痒。這可謂是開重置版的遊戲中十分成功的例子。

對於遊戲廠家來說，開重製版是大勢所趨，畢竟十幾二十年前那麼多遊戲的遊戲如果因為畫風問題讓玩家對其卻步的話，確實是一件十分可惜的事情。如果開重製版能夠讓更多的玩家們玩到那些之前的好遊戲的話，那麼重製又何妨呢？

各位玩家們對重製版的看法是如何呢？歡迎來評論區討論一下。
