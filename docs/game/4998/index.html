王者榮耀：時隔8個月，諸葛亮黑板刷新，貂蟬新版回城特效上線

2019-12-03 0條評論 17次閱讀 0人點贊

摘要

關於王者榮耀版本動向和新皮膚的消息除了官方正常發布預告外，還可以通過諸葛亮黃金分割率的黑板提前獲知，而在時隔近8個月後，諸葛亮老師的小黑板再次更新，那麼這次更新帶來了什麼訊息呢？一起隨小編來看看吧！

可以看到這次更新加入了填字遊戲，並提問都有哪幾位英雄的新皮膚如此神秘？若想要找到答案需要完成填字遊戲，由於圖中只出現了「一二三四五」和「1234」這些數字，想必大家和小編一樣一頭霧水，不過官方既然這麼提問，那麼首先可以確定這次爆料會涉及多款皮膚，小編猜測三五那一列或許就是此前曝光的「三國五虎將」系列皮膚，當然大家猜不出來也別著急，官方會在近期發布提示哦！

另外此次更新後，貂蟬傳說皮膚仲夏夜之夢和星傳說金色仲夏夜的全新回城特效上線，兩款皮膚的回城特效加入了草地和水池等元素，看起來更加夢幻，也契合了皮膚主題，感興趣的玩家可通過皮膚展示界面的定製選項中的局內表現查看一二！

附上兩款皮膚全新回城特效局內展示：

最後官方還公布了一個非常給力的消息，那便是「王者創意互動營」將在12月5號正式上線，而互動營的首期項目便是邀請玩家為大家熟悉的一位英雄設計全新皮膚，這款皮膚將與蔡文姬繁星吟遊一樣加「源·夢」入系列。同時官方還表態「源·夢」皮膚的品質將全部定位為史詩級別，等於是60點券的史詩級皮膚，擁有專屬音效、特效和登場動畫。正因為統一為史詩級，因此即將上架的蔡文姬皮膚將在未來升級為史詩品質，感興趣的玩家可以期待一下！

而部分玩家等待的諸葛亮武陵仙君、李白鳳求凰、呂布天魔繚亂和小喬天鵝之夢的優化會在本周四公布，那麼小夥伴們你覺得諸葛亮黑板透露了哪些英雄的皮膚呢？歡迎留言討論！
