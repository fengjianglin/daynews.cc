RNG公布德杯名單，UZI缺席大家沒說啥，這個中單位是不是過分了？

天天要聞
2019-12-19

在經歷了一個月的轉會之後，LPL大部分戰隊終於湊齊自己的拼圖了。如果單從人員配置來討論這次轉會情況的話，RNG的或許只能算一個中上等：大名單上只換了一個打野，另外還留下一個AD做替補，提拔上來一個二隊的中單。RNG的大部分粉絲自然也是知足了，畢竟實打實的看到了隊伍的努力，現有的陣容勉強還算得上一個補強。關鍵是被詬病的虎大將軍終於有了替補，S10好像勉強爭一下冠軍。

直到18日晚，RNG官博發出的這位「德杯大名單」。

​

大名單上並沒有小狗，很多粉絲也並不意外，畢竟德杯這種賽事更多地重在磨合。小狗也捧過杯了，把機會留給Betty，自己多休息一下也還說得過去，往年的幾次RNG也都這麼安排過。

​

不過，當看到名單上的中單名字後，大家立刻就變了臉色。

「出征名單上，中單只有小虎一人，之前在二隊打中單的Yuekai被調到了上單位，和狼行互補。」

見狀，粉絲們紛紛表示忍不了。上百個憤怒的表情疊加在RNG的評論區。大家都開始質問；
「虎將軍真就給擋過子彈？」「終於知道Karsa為什麼要離開RNG了。」「這月開不是來一隊打中單的嗎？怎麼就去上了？」「德杯無啦！」

確實很難讓大家滿意，從去年小虎一年的表現來看。似乎完全可以用「欺軟怕硬」來形容。看見比自己弱的中單（兮夜除外），就能在中路打出天大的優勢。而一旦對面水平稍高於自己或者被出其不意地用了套路，小虎整局都會像夢遊一樣。直接「斷開連接」。

但也不是沒有這種可能（猜測）：月開上一隊是為了「終極搖擺」，畢竟有Mata這個搖擺大手子在。總是想玩帶你花的。說不定什麼時候，讓UZI去打中上也都實現了！
