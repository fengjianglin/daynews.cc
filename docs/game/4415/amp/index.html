陽光城玩家點化出首隻超級飛天！其施法就像天女散花

天天要聞
2019-12-07

摘要

“超級飛天”是夢幻2019線上嘉年華活動推出的全新神獸，因攜帶了兩個特殊技能“流沙輕音”、“凈台妙諦”而備受玩家矚目。在大多玩家看來，按照以往的套路，新神獸一般要在線下嘉年華現場首度抽出得主，再等周二維護後領取的。

　　「超級飛天」是夢幻2019線上嘉年華活動推出的全新神獸，因攜帶了兩個特殊技能「流沙輕音」、「凈台妙諦」而備受玩家矚目。在大多玩家看來，按照以往的套路，新神獸一般要在線下嘉年華現場首度抽出得主，再等周二維護後領取的。然而今年的線下嘉年華姍姍來遲，令人覺得應該有猛料要曝光的，往年的套路自然行不通了！

　　這不，來自陽光城的玩家就點化出了第一隻「超級飛天」，據說第一個祈福石就點化出來了，當系統頻道刷出這位幸運玩家時，大多群眾的反應是，狗托！而狗托自帶的屬性是也許會來遲，然而從不會缺席！

　幸運得主難以掩蓋內心的喜悅，於是將這圖發了出來。以前我們總說單身狗可以買個充氣娃娃，那玩夢幻的單身玩家買只「超級飛天」，精神上應該不會太過寂寞吧？

　　這三個高級技能挺搶眼的，不過最讓人看好的還是新特殊技能「流沙輕音」，按照技能描述，最多可以攻擊五人，而且法傷會衰減，這不由令人想起了同是秒5的門派–魔王及神木，其門派群秒技能都是秒5的，現在出了這隻新神獸，會不會有取締這些傳統門派的可能呢？以後是否會出現這麼一幕，帶個法系還不如帶只飛天！

　　emmm~~這一幕似曾相似，去年的超級神狗不就是這個情形么？當時的大唐活得不如一隻狗。。。現在貌似也是這樣，一隻神狗的價格約等於幾個打圖唐了，這回該不會要歷史重演了吧？

　　那「流沙輕音」的施法特效如何？當時我們只從策劃的爆料里得以一窺其站立原地的特效，那打在怪身上是怎樣的呢？請看大屏幕！就像是原地翩翩起舞的她分身跑到了對面，又舞動起來一樣，使人情不自禁地想起了「天女散花」這個成語，這一幕看上去很美！

　　不過因不知這個技能的法傷係數及群秒的實戰效果，現在很難定價。而幸運得主第一時間想到的居然是丟上藏寶閣，結果現在還不允許上架，看來夢幻玩家年紀大了，都比較現實，一切朝錢看！於是，這世間又平白多出了20個神兜兜？
