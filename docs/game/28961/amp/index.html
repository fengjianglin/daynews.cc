爐石傳說：冬幕節亂斗禮尚往來，帶上這套牌可輕鬆拿首勝

天天要聞
2019-12-20

爐石傳說本周亂斗依然與冬幕節有關，這周是多年的傳統玩法”禮尚往來”。

規則介紹：

狂野牌池

需要組牌

在每個回合開始前，系統都會在場上空投大禮包。每位玩家在自己所屬回合打破大禮包，就能得到冬幕節禮物。

獲勝可以得到冬幕節卡背，如果已有卡背則贈送一包經典卡包。

冬幕節禮物是一個0費0-4的隨從，亡語會給當回合打破其的玩家一個失竊的禮物，打破敵我雙方的冬幕節禮物都可以。

這裡的寶藏牌必定是控制方的職業卡牌（減少5點費用），控制方意為最開始落在誰的場上，就算誰是控制方。

比如使用術士，那麼自己場上大禮包給的就是本職業的卡牌，如上圖。

偶爾會出現特殊法術大雪球，隨機將敵方3個隨從移回其手牌（非常給力）。

經過一番博弈之後，筆者成功擊敗對手拿下首勝。

由於打開大禮包獲得的寶藏牌非常強力， 所以我們的核心思路就是打箱子，找寶藏。而箱子本身沒有攻擊力，為了加快寶藏獲取速度，擁有自殘手段的術士就成了第一優先選擇職業。

諸如力量的代價、殘酷集結、火焰之災禍、血色狂歡者等卡牌，都可以在前期就讓自己拿到寶藏牌。

迦拉克隆體系則是為了增加卡組的一些厚度，且其提供的眾多小鬼可以配合相關法術，達到解場等目的。

如果碰到對方也是會玩的，那麼在中期可能會有一波場面壓力，這時候需要高弗雷和扭曲虛空來控一波場，防止突然暴斃，所以筆者各帶了一張，使用下來效果還不錯。

筆者在拿首勝過程中遇到的是一個機械騎，瘋狂+buff那種，中期風怒飛行器騎臉壓力會非常大，解不掉可能就GG了。當然，在寶藏牌中也能遇到一些解場牌，遇到了也要考慮一下留與不留。

但是這個亂斗卡包，筆者總感覺每次都是這幾張卡牌，簡直就尼瑪離譜。

還沒拿到獎勵的玩家朋友們，不妨試一下筆者推薦的卡組。如果你有更好玩的卡組，也可以在評論下方分享給大家哦~
