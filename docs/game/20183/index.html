曾經火爆網吧的《跑跑卡丁車》，為什麼被《QQ飛車》代替了

2019-12-18 116條評論 1,644次閱讀 693人點贊

《跑跑卡丁車》是韓國開發的賽車競速網遊，由世紀天成代理中國伺服器，玩家可以開著卡丁車隨意在地圖中馳騁；曾經火爆在網吧之中，但是慢慢的淡出人們的眼前，因為琳琅天上遊戲工作室開發出了《QQ飛車》，QQ飛車的騰空出世，對跑跑卡丁車造成了威脅，但是世紀天成沒有做出任何防範，很快QQ飛車就代替了跑跑卡丁車。

為什麼QQ飛車在短短的時間內就代替了跑跑卡丁車呢？小編下面總結了3點。

1、TX廣告打的好。TX旗下的QQ擁有幾個億的用戶，每天都在QQ上投放廣告，在短短的時間內就吸收了很多跑跑的用戶，而且那時候每個人都在用QQ，飛車的登陸就是用QQ，跑跑用的是世紀天成的賬號，這就造成了玩家不願去註冊賬號，QQ登陸的方便已經聲入人心。

2、跑跑的玩法不如飛車多，操作難度比飛車大，遊戲本身就是來娛樂的，跑跑的上手難度很大，去跟那些老玩家玩根本就跑不贏，然而很多玩家都沒有那個耐心去練車；飛車的上手難度就簡單得多，隨便跑幾局就可以掌握了基本技巧，並且飛車的賽事很多，像WCG、SSC、TGC這樣的賽事是不是很吸引人。

3、遊戲質量跟不上，跑跑自從上線很少更新，跟隨電腦的不斷升級，顯示屏的解析度越來越清晰，而跑跑的遊戲畫質始終不見更新，這讓玩家放棄了跑跑；飛車的畫質從上線以來就很好，而且經常更新畫質、玩法；飛車的強勢而來，跑跑卻無動於衷，自始就毫不在乎，所以飛車很快就取代了跑跑。

跑跑本身是比飛車優秀的，但是跑跑高傲自大始終不理睬飛車的猛攻，可能是韓國對中國市場的自負，毫不在乎中國玩家，再加上飛車經常照顧玩家的感受，更好的優化遊戲體驗，所以飛車取代了跑跑，成為競速遊戲的第一名。
