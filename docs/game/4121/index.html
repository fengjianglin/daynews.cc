穿越火線CFS世界賽正在進行當中，在12月8日的比賽中，中國的SV戰隊在對陣巴西VG的時候，出現了爆冷，一向在世界賽上穩如泰山的SV戰隊不敵巴西VG，尤其是在作為保衛者的時候，居然被VG打了一個6-0，而讓我們國內觀眾感到最可怕的就是VG的那桿狙擊，全場戰績21-6，最後的四連殺更是讓SV膽寒。

無獨有偶，在今年十月份的巴西邀請賽中，中國的白鯊戰隊在面對這支隊伍的時候，最終也遺憾敗北，而當時觀眾們印象最深的就是這桿狙擊，在三局比賽中場均20+的戰績，簡直不要太離譜。

而白鯊的田豐、SV的馬哲在對陣這桿狙擊時，都沒能在正面壓制住他，任憑這桿狙擊在場上大殺特殺，一時間觀眾們又想起了當年越南戰隊的rudo在場上大放厥詞的時候：那個時候越南戰隊連敗中國隊伍，最後EP摩登在對陣越南隊的時候，田豐全方位壓制rudo，最終EP摩登大比分戰勝越南隊，守護冠軍榮耀，而田豐也一戰成名，被封”護國神狙”！

而今年，巴西的這位狙擊手”DGZ”再一次讓我們感受到了巨大的壓力，田豐、馬哲相繼敗北，而唯一還沒有和其交手的就只剩下中國AG戰隊的綠豆了！綠豆是國內出道即巔峰的職業狙擊手，是CFPL聯賽四冠王，是全明星兩屆狙神獲得者，被稱為”70kg接班人”，在本次的CFS世界賽中，三位中國的狙擊手當中，綠豆的表現無疑是最好的。

因此，在看到DGZ接連擊敗田豐、馬哲之後，觀眾們把希望寄託於綠豆身上，綠豆和DGZ，都算是各自賽區出道即巔峰的選手，兩人目前的發揮也差不多，就從目前的個人戰績來看，SGZ和綠豆應該是本次世界賽中表現最好的兩位狙擊手，這兩人的對決也註定會精彩萬分！

今年很多國外隊伍，如菲律賓、越南、俄羅斯等戰隊都沒來，再加上目前中國派出的是當下最強的三支老牌隊伍：白鯊、AG和SV，本以為今年的CFS世界賽會是有史以來最沒有懸念，最輕鬆的一屆，但是白鯊開局就被打自閉，AG差點輸給埃及ANB，SV直接小組賽翻車，看來國外隊伍的實力是一屆比一屆強啊。

希望接下來咱們國內的三支隊伍都能好好加油呀！
