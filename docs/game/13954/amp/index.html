LOL中最有趣的裝備外號 先知藥劑完克提莫 它卻讓因為劇情被刪

天天要聞
2019-12-12

英雄聯盟作為一款非常有趣競技遊戲在國內一直有不少的玩家，自從LOL在2011年登錄國服開始，就不斷有玩家加入到LOL的大家庭中來。而隨著版本更迭，LOL中的遊戲中的裝備也在慢慢增加，五花八門的道具讓LOL玩家無法一一記住他們的真名，因此在LOL中這些裝備誕生了很多有趣的外號。

1. 紅叉

紅叉原名叫做幻影之舞，是現版本綠色幻影之舞未改版時的樣子。因為外形酷似紅色的X號，在這件裝備誕生之初，就有不少LOL玩家將其稱為紅叉。

因為其唯一被動無視單位體積碰撞，在早期版本一直是ADC的必出神器。不過最歡迎它的並不只是ADC，而是當時喜歡出多把紅叉的劍聖玩家，因為高額的攻速、暴擊以及移速的加成，成為早期刺客流劍聖的必出裝備之一。

以外形來命名的LOL道具在現版本也是非常常見的操作，比如現在的綠叉、血手等等。

2.拐杖

拐杖是道具殘暴之力之力的外號，因為它圖標的樣子非常像拐杖的頭柄，因此獲得這個稱謂。在之前LOL版本中，殘暴之力是AD近戰英雄的利器，特別適合瑞文、潘森、盲僧等英雄使用。由於過於強勢以及合成平滑等原因，這個物品在5.22版本被移除，並被拆分為考菲德爾的戰錘和鋸齒短匕。

3. 糞叉

糞叉是道具阿塔瑪之戟的外號。至於為什麼是糞叉，不少LOL玩家認為他們單純長得像兄弟。但是還有一部分解釋是早期皇子喜歡被LOL玩家成為打團「攪屎棍」，因為阿塔瑪之戟是皇子的武器，是它就被稱為糞叉。不過阿塔瑪之戟的屬性非常不錯，因此在早前版本中早已被移除掉了。而對於移除的原因，拳頭還在英雄聯盟漫畫中解釋道是因為塞拉斯折斷了它，不過在現版本的閃擊模式中，阿塔瑪之戟又登場了。

4. 蛋刀

蛋刀是早期打野裝備蜥蜴長老之精魄的外號，而獲得這個外號的原因則是因為蜥蜴長老之精魄長得很像WOW中伊利丹所使用的武器。蜥蜴長老之精魄在S3中是一件超級熱門的裝備，不僅打野位置喜歡出他，很多線上英雄都因為它的被動而非常愛出。因此，出現了不少蛋刀流EZ、蛋刀流中單蔚的打法。

除了它以外，LOL還把另外兩件和它同類型的打野裝備稱為「蛋拳」和「蛋衣」。

5. 珍視明和春哥甲

不得不說LOL玩家們的腦洞真是大，最後居然把明星和LOL中的裝備結合起來了。春哥甲叫做守護天使，是LOL中後期神器之一。因為網上流傳的信春哥、得永生的梗，因此這件裝備被聯繫在一起，稱為了明星代言的LOL裝備。除了它以外，被稱為珍視明的先知藥劑也是明星代言的LOL裝備之一，每次遇到LOL玩家拿出先知藥水，不少就會想到郭敬明的代言詞。
