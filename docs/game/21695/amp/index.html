逆水寒有多沙雕？把王思聰蔡徐坤玩壞，還在雷峰塔前建了座遊樂場

天天要聞
2019-12-18

大家好，我是X博士。

要說國內網遊里哪一款最「沙雕」，那肯定就是《逆水寒》了。

這款遊戲從上線以來一直都以沙雕的風格著稱，不僅經常把表情包做成遊戲道具，甚至還讓坤坤、思聰等知名人物在遊戲里擔任NPC。

而後又常常化身小遊戲啟動器，動不動就搞一些下棋、打牌、打麻將的玩法，企圖取代4399。

如今《逆水寒》已經一發不可收拾，專註於沙雕風格的他們似乎又找到了新的方向，用極短的時間讓遊戲風格又再次發生了變化。

那麼今天就讓X博士帶大家來看一看，這款遊戲又帶來了什麼新鮮玩意兒吧。

劃重點一：玩家人均六歲 逆水寒開啟全民搖搖車競速大賽

自從十月底遊戲里安裝了搖搖車之後，逆水寒就彷彿改名成為「逆水寒幼稚園」，不少玩家任務不做副本不打，每天一心一意坐在搖搖車上晃悠。

而且該道具還會手動減齡，無論是PVP大佬還是職業打牌選手，只要坐上搖搖車統統只有六歲。

但逆水寒並沒有就此滿足，下定決心走沙雕道路的策劃最近就在遊戲里組織了一場搖搖車競速大賽，邀請所有六歲及以下的逆水寒小朋友參加，像X博士這種六歲零二百四十個月的小朋友當然是不會錯過。

別看玩家們都是騎著搖搖車，比賽的激烈程度可一點都不比正經的賽車差。

光是這些遍布賽道的水柱陷阱就足夠讓那些走位不慎的玩家耽擱很久。

萬一要是再撞上龍捲風就更別想前進一步了。

除了陷阱之外，該模式中還設置了大量的技能光球，觸碰光球會獲得加速、擊飛、互換、護盾、陷阱等多個技能，讓競爭變得激烈的同時也讓這場比賽更具趣味性。

玩家一邊互懟一邊競爭第一，頗有一種兒時的樂趣。

只不過唯一遺憾的是，搖搖車並不會播放音樂，賽車的背景音樂也是正常的古風，如果能夠一邊開搖搖車一邊聽「爸爸的爸爸叫什麼」，那就更有內味兒了。

劃重點二：幼稚園策劃接管逆水寒 全是小朋友愛玩的東西

可能是因為玩家人均6歲的緣故，這一段時間以來，逆水寒除了堅定地行走在沙雕道路上以外，也顯得越來越「低齡化」。

當初在六一的時候遊戲就推出了一套「童趣武器」，喵喵拳、螃蟹槍、鹹魚劍都引起了不少「小朋友」的喜愛。

而在前些日子的更新中，又推出了一套貓咪背飾。

更奇葩的是，他們竟然在遊戲里的雷峰塔前蓋了一座遊樂場。

在遊樂場里玩家可以玩到掃雷、雪人塔防、打地鼠（猴子）、配對、躲貓貓等各種各樣的小遊戲。

這些遊戲不僅免費參與，還能夠獲得大量的代幣獎勵和專屬稱號。

遊樂場中心還搭建了大型的旋轉木馬，讓無論大朋友還是小朋友都能夠在此處找到童年的快樂。

能有這種天天住在幼稚園的策劃，隔壁的小朋友都酸哭了。

課後總結：

看來逆水寒已經明白了遊戲的真諦就是快樂，讓當代快樂的主要源泉，就是沙雕。
