軍團出征中的帶線狂魔，最不怕輸出的法師非周瑜莫屬

天天要聞
2019-12-06

摘要

之前的文章中，有為大家提到過，在出征模式中所有物理&quot;低分惡霸&quot;都非常害怕法師。事實上，不只是&quot;低分惡霸&quot;在同等戰場情況下，同等裝備的物理英雄同樣非常重要被法師所秒殺。

之前的文章中，有為大家提到過，在出征模式中所有物理”低分惡霸”都非常害怕法師。事實上，不只是”低分惡霸”在同等戰場情況下，同等裝備的物理英雄同樣非常重要被法師所秒殺。不過，在物理英雄害怕被法師秒殺的同時，由於法師都是技能傷害，而出征中又有著bkb、天征劍這樣的魔免裝備存在的原因，大多數法師其實也都有些還怕物理英雄。尤其是在前期沒有足夠裝備支撐的情況下，雖然物理英雄大多數也都沒有bkb，但是由於法師傷害不夠的同樣非常害怕物理英雄。不過，今天舞文要為大家介紹的這個法師算是其中一個例外——他就是帶線狂魔周瑜。

大家都應該知道，在出征地圖中，一個裝備成型的周瑜，無論是在團戰、單殺還是帶線牽制上都有著極其強大的效果。他R技能的超大範圍持續傷害，配合上大招的刷新效果，無論是在團戰中限制對手發揮還是帶線實際作用效果都非常強大。可是，很多朋友其實都已經忽略了周瑜真正強大，也是他在前期就不懼怕任何一個輸出英雄的技能——W裁決之光。

仔細看一下技能介紹，大家不難看出，被周瑜W技能命中的所有敵方單位在接下來的一段時間內普通攻擊都會落空，對英雄的限制時間長達2.5秒之久。這也就使得了，所有依靠普攻輸出的物理輸出英雄在沒有bkb的情況下，只要被周瑜的W技能命中如果沒有bkb那麼就只能被動挨打或者選擇撤離。

也正是因為周瑜W技能對所有依靠普攻輸出的物理英雄有著強大的限制效果，所以舞文不推薦大家在玩周瑜這個英雄選擇肉裝。實際上，如果在裝備條件允許的情況下，可以選擇帶上肉裝翅膀（仙道）其餘裝備全部法術裝備。裝備不夠的情況下，直接全部法術裝備即可，甚至生存天賦中的狡邪（增加閃避幾率）都可以適當放棄。

最後還是提醒大家一下，由於周瑜這個法師與其他法師相比起來，整體爆發能力差了一些。但是擁有更強的帶線和支援能力，實戰中需要周瑜去帶線的時候也非常之多的原因。所以，舞文推薦大家在玩這個英雄的時候，一定要多買燈插燈，盡量避免自己傳送帶線的時候，出現落地就被帶走的情況。如果感覺自己落地的地方可能存在危險，可以選擇先用寵物探路，或者是直接傳送到自己實際目標地點附近自己控制好了視野的地方再想辦法千萬目標地點。
