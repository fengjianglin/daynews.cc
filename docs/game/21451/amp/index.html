人物辭典即將到來，勇士們喜歡的NPC是誰呢？

天天要聞
2019-12-20

作者：一位路過的假面騎士
阿拉德大陸的勇士們，談起DNF中的NPC,直接浮出腦海的是哪位呢?是初來阿拉德大陸時給予你第一份幫助的鐵匠林納斯?還是與你朝夕相伴的賽麗亞?又或是孜孜不倦傳授你技能的職業導師?
伴隨著下次版本的更新，全新的人物辭典系統也將降臨阿拉德大陸，玩家們與NPC之間宿命的羈絆也可以瞭然於目。

相信不少玩家有自己鍾愛的NPC，筆者也不例外，作為一名老玩家，心中難免有幾位魂牽夢縈的NPC，筆者就介紹幾位大眾「情人」吧!
1、鐵匠林納斯

作為一名鐵匠，林納斯指引每一位冒險家，給他們第一件武器，讓他們學習第一個技能。看似是一位平凡無比的大叔，實則臉上充滿著歲月的沉澱。 憶往昔崢嶸歲月稠,看今朝旖旎風光秀，年輕時與阿甘左並肩戰鬥，老來幫助青年才俊踏上旅程，可以說是林納斯的真實寫照。
作為玩家們冒險歷程中遇到的第一位NPC，其影響力是毋庸置疑的，相信不少玩家都熱誠的期待這位大叔能早日走出陰霾，重新亮劍於世間。
2、賽麗亞·克魯敏

只要玩過DNF的玩家，賽麗亞的大名可謂無人不知無人不曉，其高顏值以及甜美的聲音，令八百萬勇士深陷其中。 然而作為遊戲中人氣最高的NPC，其人物背景卻很凄慘;面對整個種族滅門，她成為了世界上最後一個精靈，但她並沒有陷入仇恨的漩渦，反而以真心對待每個人，這或許便是玩家們都喜愛她的原因吧!
時過境遷，DNF已經陪伴我們度過是十一個春夏秋冬，同樣我們也守護了賽麗亞整整十一年，這份執著在各類遊戲中也算是一個佳話了。想必不少玩家曾經幻想過，若有一種技術可以穿越到遊戲中，那DNF必將是眾多玩家的首選!
3、G.S.D

對於G.S.D，大家對他印象是阿修羅、鬼劍士職業導師。但其實G.S.D並不是單純的阿修羅，他身上的裝飾和技能其實都表明他精通鬼劍四系。 而他以阿修羅的印象深入人心的原因有二，一是其標誌性的眼罩，二便是當年帶我們天空之塔時使用的那一堆阿修羅技能。作為第一位帶我們刷圖的NPC，筆者對他印象深刻，不少裝備較差的玩家甚至選擇不完成任務，以此換得G.S.D長久的免費帶人。
G.S.D也是一位令人敬重的導師，無論死亡之塔中的殘忍領主，還是老版本迷宮boss，都受教於他，雖然走上歧途，但GSD始終未曾放棄對他們的期望，這種亦師亦父的NPC設定令人神往。往事如煙雲逝去，回首時已千帆盡過，這一句詩句可能是其滄桑人生的最好寫照。
4、達芙妮·馬弗羅

如果說感情的深度無法測量，那遊戲中好感度的出現改變了這一局面。雖然達芙妮的人氣不是最高，但是她收到的禮物一定是最多的!原因就是在以前物資貧瘠的年代，金色小晶體的價格高昂，而贈送達芙妮普通的小晶體便可獲得其回贈的金色小晶體，每個角色將近10w的收入令玩家們爭相前往，達芙妮也一度成為萬人迷。 可惜由於工作室的原因，NPC回贈系統被刪除了，但好在好感度系統保留了下來，但不知道達芙妮的人生巔峰何時才能再度降臨?
5、阿甘左

DNF四大劍聖之一，擅長使用巨劍。在希洛克一役中失去摯愛，從此兩耳不聞窗外事，遁入酒館終日與酒為伴，過著醉生夢死的生活。阿甘左的一生是悲劇的，擁有強大能力的他卻看著摯愛在懷中死去，自己卻無能為力，難免令人感傷。 不過其優秀的劍術依舊令廣大玩家著迷，猛龍帶風，劍舞秒殺是其強大的見證;在當年悲鳴洞穴還是超高難度的時代，他作為NPC帶玩家們探索悲鳴洞穴，一個劍舞+覺醒直接秒殺蟲王戮蠱，讓人望而生畏，冷汗直冒。
6、旭旭寶寶

悲嘆之塔65層怪物，手持魔劍喜歡放無色，難度極低，一身95級傳說裝備即可將其秒殺。就這樣一個小NPC人氣卻異常之高，不少玩家慕名而來悲嘆之塔只為將其秒殺，令人費解。
總結：
以上便是本文的全部內容，每個人心中的男神女神都有差異，心儀的NPC自然也如此。有趣的NPC有許多，筆者由於篇幅原因只能止筆於此，望各位看官海涵。當然若有不一樣想法的玩家歡迎在下方評論留言，筆者不盡感激。
最後感謝大家的觀看，祝大家早日追到心中的女神喲~
