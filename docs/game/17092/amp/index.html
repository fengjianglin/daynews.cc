TGA國產動作新游《永劫無間》曝光，高配版《流星蝴蝶劍》？

天天要聞
2019-12-13

在今天上午進行的TGA 2019頒獎典禮上，多家遊戲你方唱罷我登場，頗有些神仙打架的味道。在頒獎間隙，國產多人在線動作遊戲《永劫無間》的最新預告片也正式和玩家見面，這款在TGA2019開幕前就被主持人Geoff Keighley力推的遊戲，也終於揭開了其神秘的面紗。

在《永劫無間》的正式預告片中，開場出現的紅衣女俠在紅蓋頭的映襯下，暗藏殺氣。採用戲劇唱腔搭配女主的氣質烘托出了緊張的氛圍。在東方元素瞬間拉滿的同時，給玩家十足的代入感。

這款由24 Entertainment開發的動作競技遊戲預計將在2020年發售，光看預告片就有些按捺不住沖沖沖的慾望！新的遊戲上線，不免會被玩家拿來和老遊戲相比，東方背景，又是冷兵器之間短兵相接的戰鬥，不得不讓玩家有一種高配《流星蝴蝶劍》的既視感。

國產古風多人在線動作遊戲匱乏的今天，《永劫無間》的出現，無疑是豐富了玩家在這類遊戲上的選擇。

在打鬥過程中，除了玩家的硬實力外，《永劫無間》的飛索系統也可能會決定一場比賽的勝負。由於《永劫無間》中的飛索系統將採用無死角的設定，真正意義上實現了那句周董的經典歌詞「飛檐走壁莫奇怪，去去就來」。

登錄STEM平台的《永劫無間》，無論在遊戲畫面還是特效，在國產動作遊戲中都可謂上乘。遊戲中，玩家不僅可以通過熟悉的基本操作，還可以通過諸多招式的變換，完成一套套華麗的連續技。

在武器特效上的進步，《永劫無間》也在預告片中給出了答案，一招一式間，無論是行雲流水的打鬥還是殺氣騰騰的兵器，都可以看到24 Entertainment的用心。

《永劫無間》正式預告片剛剛在TGA放出，玩家就已經對這款遊戲的上線充滿期待。到底這款被稱為高配《流星蝴蝶劍》的多人在線動作遊戲《永劫無間》能否如玩家所期待，還要等到遊戲上線後小夥伴們親自感受。
