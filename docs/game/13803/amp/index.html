英雄聯盟：低端局最噁心的5個英雄，如果Ban位夠，真想全都禁掉

天天要聞
5 天 ago

英雄聯盟這款遊戲上線已經有10年時間了，而設計師為了讓玩家能夠在遊戲中獲得更好的遊戲體驗，每位英雄的登場都是經過設計師精心雕琢的，英雄們的技能也都是該英雄玩法的重要組成部分，每個英雄的技能也都是精心設計過的。

接下來給大家介紹的是低端局中最噁心的幾個英雄!

石頭人

在低端局玩家補發育能力差，裝備跟不上節奏，同時石頭人作為一個不是很吃發育的一個英雄，在低端局就非常噁心了，致使很多低端局玩家在後期面對石頭人束手無策，而且石頭人大招的控制效果更讓玩家沒有輸出環境，見了石頭人只能乖乖被秒。

石頭人大招超長時間控制，之後的貼身後E技能減少攻擊速度的限制效果讓低端局玩家頭疼，基本被撞到就要被秒，石頭人前排的控制能讓後排隊友有一個好的輸出環境，堪稱團滅發動機！

劍聖

劍聖，大招免疫減速效果，意味著劍聖可以無視對方的減速技能，一旦裝備成形收割能力大大增加，劍聖雖然免疫減速，卻不免疫控制，但是在低端局控住劍聖也不一定定會有人集火打死劍聖，在低端局的劍聖基本也很容易殺穿，1v5的場面也是非常常見的。

蠻王

一級蠻王二級信，三級鱷魚要你命，這句話可不是虛的，一級的蠻子全屬性是所有英雄中最高的，所以在一級的時候蠻王還是見面就開始打架的，所以還是要避免蠻王一級的時候，前期對線拼起來得避過這段爆發期。後期的高爆發和五秒無敵更讓玩家頭疼，對付蠻子最好的辦法應該就是放風箏。

提莫

玩過英雄聯盟一定聽過這句話，「團戰可以輸，提莫不必須死」，有數據顯示提莫每天得死200萬次，可見提莫有多讓玩家厭惡。整個地圖都有蘑菇的痕迹，提莫的蘑菇之所以如此恐怖，是因為低端局玩家沒有帶真眼的習慣，一個人踩蘑菇，整支隊伍遭殃！並且還可以Q致盲對手，提莫也是喜歡一個人玩的英雄，打團一般也沒有什麼作為。

亞索

LOL低端局最怕碰見的5個英雄，來自聯盟深深的惡意!亞索是全聯盟禁用率最高的英雄，勝率卻僅排在很低，很多小夥伴禁用亞索的理由無非就是怕遇到托兒索。在低端局玩亞索的一般不是托兒索就是快樂風男，只關心著自己的快樂E來E去，自己被殺穿了都不知道。簡直就是來噁心隊友的，所以不如不見，禁了就好！
