Rookie峽谷之巔被隊友狂噴！我冠軍中單沒牌面？

天天要聞
1 週 ago

距離德瑪西亞杯開賽還有一段時間，職業選手大多也選在這個時間進行直播補一下直播時長。但是在昨日，iG中單Rookie在直播打峽谷之巔時，被鍵盤俠隊友追著噴了20分鐘。

Rookie在這場Rank中使用的是阿卡麗，遊戲開局，Rookie就建立了很大的優勢。但是因為一波傳送繞後出現了失誤，雖然失誤了一次，但Rookie依舊非常自信，表示這局遊戲不會輸。

隨後上單隊友奧拉夫抓住了Rookie的失誤罵了起來，而且用詞相當難聽。這是已經有很多直播間的網友看不下去了，畢竟只是遊戲失誤，前期遊戲所建立的優勢也是Rookie打出來的，但這個隊友完全看不到。

這時上單奧拉夫的戰績為0/3，而Rookie在失誤後的戰績為4/1，奧拉夫從抓住一波失誤開始噴到了遊戲結束。

不僅如此，單單的是噴遊戲操作上的失誤並不會引發這麼大的反響，關鍵是這位路人隊友直接從噴遊戲操作失誤上升到了人身攻擊，說出的話更是極其的低俗：「噁心人，一輩子都是低能」。

這位奧拉夫玩家雖然遊戲玩的菜，但是罵街的能力確實是「超凡大師」。也不知道是哪裡來的自信，0/3的戰績強行甩鍋給4/1的中單。

不過Rookie對於這樣的謾罵並沒有在意，而是在第一時間主動背鍋，承認自己失誤。但是奧拉夫並沒有收手，素質簡直是沒有下限。

Rookie對此全程微笑面對，並沒有對奧拉夫的不禮貌行為做出回擊。Rookie也一直在直播中表示，是自己那波失誤了，自己沒打好。隨後還安慰直播間的粉絲：沒事，你們別生氣。

Rookie處理這件事的方式得到了網友的好評，對此網友們表示Rookie心態是真的好，遇到這樣的隊友並沒有給予語言上的回擊，而是把責任都攬到自己身上。網友表示不愧是冠軍中單，心態是真的好。

但是遇到這樣的隊友心態還是會受到影響，在結束遊戲後Rookie也是被這位路人隊友給噴自閉了，表示自己下播後要去韓服找回狀態。

遊戲中遇到逆風或者是操作失誤都是正常的事情，遊戲本身也是為了放鬆娛樂，但是稍稍遇到自己不滿意的就開始口吐芬芳，屬實是沒素質的行為。更何況自己的問題更多呢，隊友同樣不也是給予你包容了嗎？
