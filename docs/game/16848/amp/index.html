這個遊戲好玩嗎：“左右互博”的別樣體驗——《變數》簡評

天天要聞
2019-12-16

在塔防遊戲已經發展的比較成熟的今天，雖然《植物大戰殭屍》《kingdom rush》系列等經典作品都給人留下了深刻的印象，但似乎已經越來越少見到特別有創新的塔防遊戲了。不過，高手在民間，一些獨立遊戲開發者將塔防核心玩法與其他遊戲元素相結合，製作出獨具創意的塔防新作。今天為大家介紹的《變數》就是這樣一款將塔防遊戲與roguelike機制相結合，甚至將遊戲編輯器的核心思想加入遊戲玩法之中，玩家接下來會遇到什麼樣的關卡完全由玩家自己編輯，這種「左右互搏」的遊戲體驗可謂是塔防遊戲中非常少見的了。在這裡，需要感謝頭條遊戲騎士團提供的遊戲key。

主界面

在《變數》中，玩家面對的是一塊空曠的遊戲場地，塔防遊戲的重要構成元素——路線、出怪點、保護目標、建塔點等，一個都沒有。這恐怕會讓初次接觸這款遊戲的玩家感到十分詫異，難道這一切都由玩家自己編輯、搭建？

沒錯，在本作中，玩家所要作的不僅是建設防禦塔抵擋怪物的一波波襲擊，還得自己搭建路線，設置建塔點、出怪點和保護目標。但有一個極其重要的前提，那就是玩家必需保證出怪點到目標點之間的路線是暢通無阻的，如果你想在搭建路線時直接阻斷怪物的行進路線達到不戰而勝的目的，那是萬萬不可能的了。這是不是有一種自己給自己挖坑的感覺？更令人傷腦筋的是，玩家搭建路線所用的地板磚，每一關係統給出的數量十分有限，這就意味著玩家不可能隨心所欲地通過搭建來獲得曲折而漫長的怪物行進路線從而在戰鬥時取得優勢，可這樣一來出怪點和保護目標之間一馬平川、直來直去，豈不是對玩家阻擊怪物極其不利？還好系統也並不是如此不講道理，每一關開始前給出的道具還包括了各種建築。說是建築，其實是各種形狀的障礙物，玩家將其放置到之前鋪設好地板磚的地面上，就會對地面所在的那個平面產生分割，這樣一來，從出怪點到保護目標之間的路線就不再是直來直去的了，而是繞過障礙物的一條曲折的路線，這就是玩家製造有利於防守的路線的唯一手段。

利用障礙物來構建曲折的路線

這樣一來就非常有意思了。由於每一關給出的障礙物數量也是有限的，所以玩家如何合理利用手頭上障礙物的形狀，通過巧妙擺放來達到延長怪物行進路線的目的，就成為遊戲的精髓所在。障礙物不僅有延長路線長度的功能，防禦塔的放置點也是在障礙物上，這樣一來，障礙物的巧妙利用與放置就是本作最重要的一個策略點。玩家在放置障礙物時不僅要考慮儘可能讓怪物多走冤枉路，還要保持怪物行進路線的暢通，這就相當於玩家在自己編輯關卡地圖，然後在接下來的戰鬥中挑戰自己設計的關卡，這樣一種「左右互搏」的遊戲體驗不僅非常有趣，也讓遊戲作者節省了大量的開發成本。這可不是開玩笑或者調侃，其實這是一種非常巧妙的設計理念，將遊戲內容的創作權交到玩家自己手中，讓玩家自己產出內容，實際上遊戲作者只需要負責設計好這些供玩家差遣的功能元素就OK了，剩下的樂趣就靠玩家自己來尋找。這樣做的另一個好處是極大地延長了遊戲壽命，起到了集思廣益的作用，也讓玩家的參與感更強，成就感更強。加上本作的每一關出怪以及給出的障礙物都有隨機性，這就更加增添了遊戲的重複可玩性。

出怪具有隨機性，不要忘了看一下下一關出什麼怪再決定用什麼塔

玩家自己設計關卡自己去闖，這種讓玩家「自己給自己挖坑」的「左右互搏」式玩法的設計理念十分巧妙，同時又因為加入了roguelike機制，這就讓本作的遊戲性再次獲得了加強。玩家每一關使用的防禦塔是通過翻牌來獲得的，每一次翻牌玩家都是在9張牌里翻出3張，這樣的機會每關都會有一次。玩家可以選擇不翻牌，把翻牌機會留著。玩家在每一關消滅來犯怪物後會獲得金幣，金幣可以用來提升翻牌卡池的等級，產出等級更高的卡牌即品質更好的防禦塔。翻出什麼防禦塔很看運氣，歐皇次次出高質量防禦塔那就肯定有優勢，而且防禦塔還可以進行合成，如果是歐皇的話翻出的塔可以相互合成，獲得更給力的塔，那遊戲體驗毫無疑問上了一個台階。不同的防禦塔定位不同，有高攻擊但低攻速的，有大範圍AOE但傷害低的，也有對空十分牛逼但無法對地攻擊的，這些都需要玩家在遊戲過程中時刻關注下一關會出什麼類型的怪，根據形勢靈活變通。當然，防禦塔的擺放一定是在障礙物上，擺放的時候會顯示塔的攻擊範圍，不同類型的塔是不一樣的，需要玩家審時度勢地靈活利用。

翻牌環節

通過戰鬥會掉落一些裝備，把裝備裝到防禦塔上就可以產生效果。不過，很多裝備都是有利有弊的，比如增加攻速和攻擊，但是塔會失去自己的特殊能力。由於大部分裝備都有副作用，所以玩家不能無腦使用裝備，要根據戰場形勢以及自己防禦塔的特效來判斷，這個獲得的裝備是不是適合現在使用，給哪個塔使用。這也是一個比較有特色的決策點。

這就是一個有副作用的裝備

不過，本作雖然創意十足且重複可玩性好，但也不是十全十美的。新手引導比較淺顯，導致新人入門的難度並不低，有些基礎功能甚至沒有在新手引導交代清楚。比如防禦塔的裝備如何使用的問題，新手引導說得不夠詳細，我當時是在遊戲過程中找了好一會兒才發現裝備要怎樣放到塔的裝備槽上去。這裡其實也有UI設計的問題，個別重要的功能做得實在是太不顯眼。另外一點不足就是難度問題，其實本作的難度是很硬核的，即使是最低難度也對玩家的大局觀和障礙物的擺放技巧有比較高的要求，莽是莽不過去的，而且中途也不能有一點差池，容錯率很低。加上防禦塔和裝備的類型較多、特效各異，學習成本還是比較高的，新人初次嘗試有被勸退的可能。

除了塔，裝備也種類也很多

總的來說，《變數》是一款令人眼前一亮的國產塔防獨游。把關卡設計的權力交給玩家，讓玩家「左右互搏」，是本作最大的亮點，也讓玩法更具深度。其次是隨機性很高的障礙物、防禦塔產出等要素與roguelike機制的結合，這就讓遊戲的可玩性更上一層樓。雖然設計上的細節有待打磨，難度對新人恐怕不夠友好，但總體而言本作還是非常值得一試的。
