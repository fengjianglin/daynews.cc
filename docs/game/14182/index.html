《仙劍》官方發布調查問卷《仙劍奇俠傳》單機重製版或將發布

2019-12-10 357條評論 3,767次閱讀 1828人點贊

此前仙劍之父「姚仙」姚壯憲就曾透露將會對《仙劍奇俠傳》一代至三代進行重置，今天仙劍官方發布了「仙劍奇俠傳單機遊戲重置版問題調研」，由此看來仙劍奇俠傳重製版可能真的要來了。

官方稱：「我們了解到各位大俠近來對《仙劍奇俠傳》重置版的關注度越來越高，各種不同方向的建議也傳到了我們面前，為了讓遊戲能獲得更多人的喜歡，我們想了解一下您的看法，請依據個人的喜好作答即可，萬分感謝！」官方在問卷中針對多個問題進行了調查，問卷地址。

姚壯憲透露重製版主要是面向PC版及移動端進行開發，但並不排除登陸主機。官方表示遊戲將會是完整重製，包括人物模組也將重新製作，玩家不必去擔心遊戲會是二頭身小人的高畫質版本。而且大宇董事長塗俊光也曾在個人微博透露，《仙劍奇俠傳》前三部將進行重製，由大宇旗下的工作室樂多數位(《仙劍6》PS4版製作團隊)及阿爾特工作室打造，加入新的3D技術及新的配樂，重製原先的劇情跟遊戲。

《仙劍奇俠傳》發行於1995年7月，榮獲兩岸當時無數的遊戲獎項，被眾多玩家譽為「曠世奇作」。絕對是國產單機最為知名的遊戲。2018年10月25日，新華社下屬智庫機構瞭望智庫與北京文博會組委會聯合發布首份著眼文化IP的報告——《面向高質量的發展：2017-2018年度IP評價報告》，《仙劍奇俠傳》入選為《報告》榜單TOP20。
