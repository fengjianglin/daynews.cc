摘要

《貓頭鷹和燈塔》是一款以收集故事為核心玩法的放置類遊戲，講述了在森林深處一座神秘的燈塔里，一位孤獨的老人、一隻貓頭鷹與過路的旅人的故事。準備材料，玩家需要到後院劈柴，劈柴也是有技巧的，掌握技巧的，木材的產出量明顯較多。系統每小時恢復一次砍木頭次數，還可以通關看廣告來獲得木頭。

《貓頭鷹和燈塔》是一款以收集故事為核心玩法的放置類遊戲，講述了在森林深處一座神秘的燈塔里，一位孤獨的老人、一隻貓頭鷹與過路的旅人的故事。

準備材料，玩家需要到後院劈柴，劈柴也是有技巧的，掌握技巧的，木材的產出量明顯較多。系統每小時恢復一次砍木頭次數，還可以通關看廣告來獲得木頭。同時，木材也是遊戲中的貨幣，可以用來與商人購買道具、材料。

老人將木材製作成木雕，雕刻完畢之後，貓頭鷹將木雕丟棄到森林裡，接下來，你就無所事事地進入了漫長的等待，你也可以選擇關掉遊戲，一小時後，等待結束，途徑森林的旅人在拾到的木雕的指引下來到燈塔，敲響燈塔的門。玩家便能通過與貓頭鷹對話，了解這個旅人的故事。

根據木雕的品質與稀有度，匹配的旅人故事線也不盡相同，想要做出稀有木雕，掌握配方很重要，這是玩家總結的部分稀有木雕配方公式：

上百位客人，上百個故事，瘋狂、悲傷、暗黑，這些光怪陸離的故事訴盡人事百態 。

這個故事令人毛骨悚然

這些故事獨立成章而又相互聯繫，以「老警探」為主線，將線索串聯起來，就會發現，這些訪客間竟然有著某種關聯。

我們會發現，遊戲會給出現「倒一杯熱茶」與「什麼也不做」的選項讓玩家選擇。

但實際上這並不是一款互動遊戲，玩家的選擇並不能改變遊戲進程，只是一個觸髮結局系統的機制，玩家後續會收到一些信件，這些新聞一樣的信件側面揭示了某段故事的結局。

遊戲世界環境豐富多變，一年四季，日夜星辰，黃昏清晨，風雨霧靄，暴雪風沙……唯有這高聳削瘦的哥特式高亘古不變地矗立在叢林之中，彷彿已然被時間遺忘，神秘、詭異而又孤獨。

我們不必去糾結這個老人是誰，他為何孤身居住在這幽深的森林之中，他又為什麼成天雕刻著那些木雕。

大量的「留白」增添了幾分神秘感，無人知其來由去過往，這也是老約翰的孤獨吧。
