夢幻西遊：老王炸出100級第一戒指，最高傷害無人可比，起價40萬

2019-12-06 0條評論 14次閱讀 0人點贊

摘要

在夢幻西遊以前的時期，玩家們之間最喜歡比較的就是面板。想要加入一個隊伍，並不是像現在這樣發寵物給隊長，紅書須彌和連善就是通行證。那個時候隊長會問下你的傷害和靈力，這就代表著你能打出多少的傷害。

在夢幻西遊以前的時期，玩家們之間最喜歡比較的就是面板。想要加入一個隊伍，並不是像現在這樣發寵物給隊長，紅書須彌和連善就是通行證。那個時候隊長會問下你的傷害和靈力，這就代表著你能打出多少的傷害。這就造成在以前玩夢幻加點，大唐都是加力量的，獅駝嶺也是全部力量的加點，龍宮什麼的都是加魔力。玩家們更加喜歡做的事情就是堆高傷害，全服第一傷害一變化，那真的是艷驚四座。現在夢幻已經是一個新的玩法了，在增加傷害的基礎上，暴擊和穿刺等級也同樣重要。今天我們來看一個暴力的戒指。

在完成一個副本之後，每一位玩家都能得到一些副本積分。如果你有開月卡的信，那麼獲得的積分還能再增加一點。這些副本積分能做的事情也很有限，基本就是去建鄴城換看戲的戲票，然後就是去副本官員那裡換獎勵。如果想要求穩妥，那麼修鍊果和強化石都是很好的選擇。其實現在BT版的玩著不錯，畢竟無級別和神獸系統都送，而且還有用不盡的夢幻幣，這在叫樂嗨手游的弓中浩里能找得到。不過玲瓏寶圖肯定是最受歡迎，因為可以靠這個挖出靈飾的製造書。100級的戒指書絕對是最貴的存在，光一本書很多區就要賣800萬以上，真的是等於小賺一筆了。

最近，老王也是幫一位老闆鑒定了一波裝備。第一個出的140級戒指，就是傳說中的雙防禦加1氣血的，這種手鐲適用於任何的門派，它能帶來特別多的防禦和氣血屬性的增加。主屬性加28點的抵抗封印等級，也是能讓自身的抗封屬性增加。

還有一件150級的男頭，雖然不是很值錢，可是也展現出了手氣。它的總防禦是算高的，特技更是出了一個放下屠刀。這種小特技現在是高端任務和比武奇缺的，不過因為只有一條藍字，自然也不是特別值錢。

下面這個戒指絕對是重頭戲了，這是一件100級的戒指，亮出來一看就炸鍋了。它居然是傳說中的4傷害，這個就已經是鳳毛麟角了。戒指最為關鍵的是爆屬性，老王計算了一下子，下排一共是多爆了11點傷害，他表示自己直播了這麼多年，還從未看過總傷害比這個更高的，實在太暴力了。關於它的價格，老王還特意去藏寶閣上面比較了一下，它覺得如果要上架，真的起碼要擺40萬以上。
