CSOL限時周年特惠，開啟槍戰每一天

2019-12-10 240條評論 3,686次閱讀 2372人點贊

12月，經典FPS生化&競技網遊《反恐精英Online》將迎來十一周年大版本。在放出眾多獎勵的同時，官方還將限時開啟狂歡周年慶典優惠活動，凡是開啟禮盒的人都有機會獲得頂級超凡永久裝備！

除此之外，玩家還會在遊戲中偶遇黃金殭屍獲得不俗的獎勵！是不是聽著都留下了口水啊，那還不快隨我一起去康康限時福利活動！

入手頂級超凡永久裝備或進階強化武器

為了紀念《CSOL》國服十一周年的到來，官方商店開啟限時兌換費爾南多plus和【永恆】費爾南多plus的活動，並根據兌換人物的期限贈送不同數量的【周年慶典鑰匙】和【周年慶典尊榮鑰匙】。需要注意的是無論是哪一種鑰匙都有一定幾率獲得最高限制＋6的永久或期限制進階強化武器，更有機會獲得頂級超凡永久武器！（看臉的時候到來了！）該活動是12月4日版本更新後正式開啟，到12月23日截止，如果在此期間獲得了寶箱記得要及時兌換哦，不然在2020年的1月15日未使用的禮盒和鑰匙都會被替換為歡樂積分。

可以獲得的進階強化武器和頂級超凡永久武器的列表如下圖所示。這26款都是限定武器哦，走過路過可千萬不要錯過！

80餘款裝備隨機發送，小心偶遇黃金殭屍

如果說今年的官方的生化模式福利里，最吸引人的是什麼的話，那莫過於生化模式（英雄）隨機武裝活動地圖的限時開啟了。人類武裝、殭屍種類全部隨機分配，紅寶石之心、【輝煌】聖劍等80餘款武器裝備統統免費使用，11周年就是讓你爽！

當然作為慶典活動，只有普通的小殭屍那會顯得很無趣，各大地圖還會隨機刷出金燦燦的黃金殭屍！亮瞎你的雙眼，嚇掉你的下巴！當然風險越大，收益也就越大，擊斃黃金殭屍將會隨機掉落多個【1-5階隨機配件】和大災變專屬道具！！

同時官方還將將根據活動期間擊斃黃金殭屍的次數贈與玩家不同的裝備（每天最多計算三次擊殺，所以想要心儀裝備的人可要拼運氣和手速了哦）

戰場補給，配件獎勵

除了以上的獎勵外，每天累計在線10分鐘將獲取戰場補給——50換了積分兌換券，同時從11月20日起，根據累計登陸的戰場補給天數，還將獲得三階配件禮物箱的額外獎勵！

CSOL十一周年，讓我們再次在這片土地上開啟一場激烈的槍戰吧！
