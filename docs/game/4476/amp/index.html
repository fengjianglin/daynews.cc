小夥伴們好呀，我是貓幼，今天也是充滿希望的一天。

地下城與勇士在85版本末期到90版本中期可以說是最巔峰的版本，盧克頻道天天爆滿，進頻道都難別說打團了。但是自從今年開放95版本以後，地下城與勇士的人氣就一天不如一天，不過不管怎麼說，一款遊戲開服11年還沒關服，也是挺了不起的。從11周年慶版本過後，策劃要比之前大方了不少，各種福利道具都免費送。但是依舊抵擋不住玩家們脫坑的腳步，能不能挽回這個遊戲，就得看100版本能不能給大家帶來驚喜了。

還記得在90版本的時候，附魔天價，增幅書一年也就送個一兩本，那個時候玩家們都說策劃摳門，要脫坑。但是現在的版本，基本沒有天價的道具了，附魔和增幅書都是免費送。就拿下周即將更新的聖誕版本來說，增幅書就好基本，還有凈化書、扭轉書，跨界石、成品裝備，完美附魔等道具。這些道具都是直接畢業的東西，去普雷的門檻也更近了一步，但是很多玩家卻說遊戲要涼涼了。

就這一個月的獎勵，放在去年前年，一整天都不一定送這麼多獎勵了。現在多號黨打造小號也很簡單，裝備基本全畢業，紅字也打得七七八八了。不過也有別的小夥伴有不同的意見，贈送增幅書雖然好，但是增幅等級更好才更強，這不是騙氪嗎？滿級附魔我才買就送，這不是坑我嗎？裝備跨界石，現在裝備都畢業了，又不能跨普雷，有什麼用？你這不是叫我變相的創建小號增加氪金幾率么？

其實對於策劃來說，也是非常難做的一件事情，一百個人有一百個想法，更何況是八百萬勇士呢？有的外觀黨想要更多的時裝，比如說天空，絕版時裝。有的玩家覺得增幅已經足夠了，細節為重，想要白金徽章，燦爛徽章等等。所以說玩家想要的，策劃永遠都滿足不了，甚至玩家自己想要什麼也不知道。

好了，今天就說到這裡了，歡迎大家的評論點贊，跪求大佬們的關注。
