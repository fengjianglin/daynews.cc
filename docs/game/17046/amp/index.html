DNF：普雷武器“權能BUG”，技能可變大，鬼泣魚頭全屏裝不下

天天要聞
2019-12-14

隨著DNF的更新，不少裝備已經消失在版本的長流中，曾經出名的魔劍、名刀、無影劍也只能作為外觀而存在，而95級最強的1件裝備是夜語黑瞳武器，需要先製作再升級，周期接近半年，但由於全職業屬性一樣，為了區分出了一個權能系統，6件首飾裝備後可觸發，大家都能自行選擇比較有趣的效果，但有玩家在刷圖過程中，卻發現了一個有趣的特殊效果！

普雷武器「權能BUG」，技能可變大，鬼泣魚頭全屏裝不下！

普雷武器可附加一個「巨人化」的黑暗權能，但這只是拿來擺設的，連普通攻擊都做不到，可玩家們卻另闢蹊徑，既然權能釋放後不能丟技能，那麼技能和權能一起放不就行了，於是就發現這個「權能BUG」！可以改變全職業的技能大小。

下面用鬼泣的二覺技能來說明，如下圖所示，鬼泣的魚頭正常大小是半屏多的水平，這已經是很大的範圍了。

而如果觸發權能巨大化的效果之後，鬼泣的魚頭到了全屏都裝不下的地步了，最上面的技能已經溢出屏幕！技能屬性具體是加了50%還是翻倍真不少說，不過技能確實是肉眼可見的變大了！

「權能BUG」的觸發方式是鬼泣在釋放二覺的那一刻，普雷武器釋放黑暗權能！就可以達到脫手技能變大的效果。

其實不少玩家早就遇到了，因為95B超界防具史詩刷圖的過程中，觸發的時候釋放巨人化權能，套裝效果觸發的特效範圍也變大了。至於傷害的話，自然不會是加倍了！目前測試是僅增加技能範圍的大小，看來「權能BUG」只能作為修鍊場娛樂之用，普雷BOSS輸出作用不大，畢竟按了巨人化就等於OB了。

至於其它職業，比如劍魂，覺醒的時候釋放權能，也會把覺醒技能的劍變大！超大版的巨劍控制BOSS，專業輔助實錘了。不過紅眼的覺醒似乎不算是脫手技能，不能像鬼泣和劍魂的覺醒一樣變大。

總結，普雷武器的「權能BUG」，雖然可以增加技能範圍，但實戰作用很小，玩家們搬磚的時候娛樂一下就行。
