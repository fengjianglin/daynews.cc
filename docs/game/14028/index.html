萬戰神話締造者，全球第一幻獸，這位傳奇玩家的賬號到底值多少？

2019-12-11 869條評論 3,187次閱讀 2062人點贊

《魔域》玩家應該沒有一個不知道他的，他曾是第一個破8000、9000戰的玩家，還常年佔據幻獸榜第一名，他就是丿灬楓葉琴丶。

要知道，八、九千戰已經是非常非常稀有的事情了，但現在他又成為了第一個破萬戰的玩家，這讓普通玩家更是不敢想像。

當然，丿灬楓葉琴丶最輝煌的事迹還不只是戰績，還有他培養出來的幻獸，一直穩坐全球幻獸榜第一的位置，到現在，他的幻獸已經高達5799星了，比第二名足足超了1400多星。

全球第一幻獸+萬戰締造者，如此成就，很多人好奇楓總的遊戲賬號到底價值多少。對於這個問題，恐怕是很難統計，但可以肯定的是，曾經有土豪想用200萬買下楓總最愛的5000+魅影冥龍，被他果斷拒絕了……

在遊戲中擁有更高的戰力和榮譽一直都是許多玩家孜孜不倦的追求，很多人視楓總為英雄的同時，也在不斷嘗試超越他。隨著《魔域》新資料片「御劍天下」正式曝光，遊戲也帶來了自帶本命寵的全新職業——御劍師。

據說，這個新職業將很有可能成為未來軍團戰的主宰者，也許用他來趕超楓總也不是夢！

御劍師有強大的連擊技能，每次連招都會有三把飛劍，每把飛劍提供356%傷害，相當於一次連擊5142%，已經超越龍騎。

而和其他連擊職業不一樣的是，御劍師自帶連招免疫技能——護主靈劍，隨隨便便擋個9000血，還會反擊敵人，配合強大的輸出技能、短冷卻的群攻技能和大範圍的霸體技能，讓御劍師在打副本、PK的時候如遊刃有餘。

御劍師既有法師、龍騎的護盾類技能，又有亡靈職業的連擊免疫技能，還有強力的輸出技能，可以說是很全面了。

御劍師自帶的本命劍有三種形態：御劍、守劍、狂劍，每位玩家不同的修鍊方式將決定本命劍劍道的發展方向，御劍是提升傷害，守劍增強生命，狂劍則能在關鍵時刻爆發，一擊制敵。

算上本命劍，御劍師可是能一人帶兩寵的職業，橫行亞特大陸應該是不在話下。期待12月21號全新職業御劍師的上線，現在到官網預約還能拿專屬職業禮包哦！
