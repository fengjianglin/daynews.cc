城牆防禦指揮官搭配推薦 教你打造鐵血城防

城防建設的過程中，守城指揮官是不可或缺的。但是和出征指揮官不同的是，守城指揮官需要對應領地內全體兵種，而不是某單個兵種。因此在搭配守城指揮官時，領主需要注意很多細節。

首先最重要的就是兵種，領主需要根據領地內的兵種配比進行搭配。比如領地內的兵種是以步兵為主，則搭配指揮官時應該優先選擇步兵。但是如果領地內的兵力囤積以騎兵為主，則守城指揮官應該優先搭配騎兵。

但是守城指揮官只有三個，所以領主在選擇一個優勢兵力的指揮官之後，還需要考慮全局。因為守城部隊的戰鬥往往是全面攻防，不像兵團戰爭是以出徵兵力應對敵方全局兵力，所以守城部隊的加成要全面一些。在選擇一個優勢兵力的指揮官後，要考慮的便是次級指揮官和全局加成指揮官。

次級指揮官是指第二優勢兵力的指揮官，若領地內有100萬步兵，90萬騎兵，50萬槍兵，那麼騎兵就是次級兵種。搭配上要考慮到騎兵的作戰，所以城防指揮官可以考慮以步兵+騎兵指揮官的模式來搭配。而第三個指揮官建議選擇弓兵，弓兵作為輸出最強的兵種，配合指揮官後能在城防戰役中發揮出巨大的威力。

還有就是指揮官的具體選擇，選擇城防指揮官要衡量的是城防指揮官的專精屬性。而且不考慮指揮官的軍團帶兵量，只考慮全局屬性加成的多少。因此領主需要優先選擇加成高的，然後根據兵種搭配。當前性價比最高的城防指揮官便是龍媽，龍媽可以為全軍提供70%的屬性加成，而且不限制戰鬥場景，因此有龍媽則必選龍媽。

其次則建議搭配一個雪諾，雪諾可以增強步兵的屬性，而且沒有專精前綴。也就是說，雪諾的專精也可以在任意場景生效。而且城防戰爭中，最先接觸敵軍並戰鬥的兵種，便是步兵。因此步兵指揮官可以必須搭配。

最後則推薦搭配騎兵指揮官，當前版本中艾莉亞·史塔克也騎兵屬性加成，和雪諾一樣沒有專精前綴，適合各種情況下的戰鬥。如果沒有艾莉亞，可以考慮拉塞爾，雖然拉塞爾有攻城前綴，但整體加成很高。而其他兵種，則可以優先選擇加成最高的，如果加成相同，則選擇沒有專精前綴的指揮官。
