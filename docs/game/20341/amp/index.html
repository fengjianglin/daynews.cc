LOL：轉會期憋氣大賽即將結束，“阿水去哪兒”要上演大結局？

天天要聞
2019-12-16

2019年的LPL轉會期即將於16日結束。不得不說各路俱樂部這口憋氣屬實是長，可把不少LPL的粉絲們急壞了。不過，好在轉會期馬上就要結束了，馬上我們就能知曉這場憋氣大賽的結局了。下面指南君就給各位總結一下LPL部分戰隊的轉會消息。

RNG

RNG不可謂不是轉會期中的熱門話題俱樂部。自從官宣Zrr和AJ離隊後，就在15日也官宣了引入原FW下路AD Betty，原YM打野小龍堡的消息。這樣一來，RNG的S10名單應該已經敲定了。除了打野更換之外，其他位置的選手大概率應該會保留首發。只是不知道小虎，還能不能在新的賽季重新找回自己「殺瘋了」的狀態呢？

FPX

新科狀元FPX在轉會期比起其他戰隊來說，動靜就小了很多，輪換打野Xinyi離隊，中路替補選手Cepted合約到期，唯一大一些的瓜就是原SKT的上路khan加入FPX，雖然沒有官宣，但金咕咕在直播中都替他點外賣了，可汗加入FPX應該是板上釘釘了。

WE

要是在轉會期中讓指南君選擇一個最慘的俱樂部，這個稱號非WE莫屬。下路大舅子已經回韓國加入了非洲隊AFs，中路兮夜已經官宣加入DMO，再算上教練離隊，經理成為自由人，這個轉會期WE目前的情況可以說是只出不進，頗有一種潰散的感覺了。截止到指南君完成這篇稿子前，WE也沒有任何官宣加入的消息；下賽季的WE會以怎樣的陣容出現在LPL，目前還是個未知數。

EDG

EDG整個轉會期的動作並不大，除了原上路Ray離隊加入KT之外沒有任何人員的變動。但廣大粉絲顯然並不只是在關注這個，「辣個男人」究竟還能不能上場成為了粉絲群中熱門的話題。雖然廠長依舊「欠」粉絲們一個冠軍，但以目前7醬的狀態，恐怕很難再回到賽場上；至於廠長最後會選擇退役，還是轉行做EDG的教練，還有待官方的消息。

IG

整個轉會期，最能憋氣的恐怕就是阿水了。錦鯉duke離隊，下路AD小西west離隊，教練組也相繼辭職。但自從IG官宣阿水成為自由人後，粉絲關注的中點完全落在阿水究竟會去哪裡上了。除此之外，細心的網友還發現，原VG的輔助南風（SouthWind）最近發WB來到了上海，而且在韓服RANK查詢下也變成了IG的輔助，難道說寶藍首發沒了？阿水有望留在IG？

​

這一切的一切，都會在轉會期結束時揭曉，讓我們耐心等待「阿水去哪兒」的大結局吧！
