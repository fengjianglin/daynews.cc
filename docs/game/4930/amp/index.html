絕地求生SMG戰隊被爆卡隊員合同，隊內選手動手？大京和堡壘回應

2019-12-03

這幾日，絕地求生微博杯WEGL正在激烈的進行著，目前正處於小組賽階段。作為官方指定的三大賽事之一，自然也是吸引了很多玩家的關注，除了比賽本身之外，自己支持的戰隊能出現在比賽中，也是關注的重要原因。在這次比賽中，SMG就是備受網友們關注的一支戰隊。在經過第一日的比賽之後，SMG戰隊的成績並不理想，只拿到了14分，目前位列小組倒數第一。

作為一支老牌戰隊，SMG取得這樣的成績顯然不是粉絲們所期望的結果，所以也引起了很多人的討論，而討論最多的就是SMG這次比賽中啟用了新隊員Sixgod和Smallccc，放棄了一直隨隊征戰的老隊友大京和小魚。啟用新隊員無疑是一種新的嘗試，但是從結果來看兩位新隊員表現並不是很頂，這也形成了一種和大京、小魚的對比。儘管大京和小魚都沒有上場，但是他們兩個也是一直在直播打RANK保持狀態，而且狀態還不錯，小魚在Rank中就上演了16殺吃雞的精彩操作。狀態這麼頂還被換下場，這不合常理，所以就有人猜測大京和小魚不上場是不是俱樂部故意為之，不是因為狀態而是另有原因呢？

隨著很多人的猜測，有「知情人士」突然爆料，大京和小魚不上場是因為和17shou鬧翻了，原因是因為堡壘。大家都知道，堡壘是SMG花「400萬」買來的戰神，但是入隊後效果卻是一般，沒有打出戰神該有的成績，正是因為這，大京在一次訓練賽之後和堡壘吵架並動了手，之後大京聯合小魚找17shou要求換下堡壘，但是作為隊友17shou支持了堡壘，反對了他們兩個，這讓大京和小魚很失望。然後夏季賽戰隊繼續拉閘，大京和小魚就想轉會，可是17shou不同意，認為他們兩個出去，將會是自己以後的對手，想讓他們要麼退役，要麼就在俱樂部呆到合同結束。

由於這位「知情人士」之前爆過幾次料，也都一一應驗，而且之前有猴子和俱樂部事件，所以很多網友都期待這個瓜的真實性。而隨著這個瓜不斷變大，熱度持續上升，幾個當事人也坐不住了，紛紛出來開始闢謠。小魚表示「不要相信壞人瞎寫的話」。而隨後，大京和堡壘也是直接互動，大京表示自己打不贏堡壘，他們關係很好，造謠能不能有點真實性。堡壘看到後也是在下面直接回復：「我心思要真打，你倆也不夠啊」，最後大京表示「你打我的話，我直接跪下」。

造謠一張嘴，闢謠跑斷腿，看來這也是一場子虛烏有的鬧劇。不過話說回來，SMG之所以有謠言，歸根結底還是因為戰隊的成績一直不好。自從FGS回來之後，他們就一直拉閘，無論是換教練、輪換隊員、換指揮效果都是一般，而他們急需證明自己，所以這一次啟用新的隊員應該就是因為這個原因。但是新組合併沒有達到預期理想，第一天就拉閘了。不過比賽還沒有結束，希望他們在後面能取得好的成績，祝福他們吧！
