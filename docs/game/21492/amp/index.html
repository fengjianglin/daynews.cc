三國群英傳2，荀彧，在遊戲中當不起王佐之才的稱號

天天要聞
2019-12-19

荀彧，曹操手下重要的謀士，說是最重要的那也不為過，早期跟隨曹操，那時的曹操遠沒有想像中那麼強大，正是因為有荀彧的出謀劃策，曹操才一步一步的走上巔峰，迎奉天子，統一北方的計策都離不開他，最重要的是荀彧推薦人才的能力，戲志才，郭嘉，司馬懿等等智者都是荀彧推薦給曹操的

但是荀彧的結局不好，因反對曹操稱魏公而引起曹操不滿，曹操漸漸疏遠，最後因為憂鬱成疾而死，還有種說法是服毒自盡

在群英2里，荀彧的頭像還是非常靠譜的，並沒有出現奇形怪狀的頭像，這點就非常不錯了，如果再整一個奇形怪狀的頭像出來，那估計玩家就更不能接受了

遊戲中的基礎武力46，這可以忽略不計，文官大多都是手無縛雞之力的存在，智力98，非常高，整個遊戲中排第三，下面說說武將技

1級技能，木輪攻，清兵技能，效果不錯，這裡的不錯只是針對於大前期，因為兵少，敵兵全沖的話，一個技能可能就清完了，當兵多點時，效果就一般了

7級技能，炎龍，清兵技能，效果比較優秀，優秀是指在兵多成堆時，清兵非常不錯，兵少分散時，這個技能就不行了，最重要的一點就在於這個技能持續時間長，可以讓敵將的武將技長時間放不出來，敵將如果中途想跑的話也需要這個技能完了以後才能逃跑

12級技能，火箭烈，清兵技能，效果一般，非常普通的技能並沒有什麼值得說的

17級技能，炎龍無雙，清兵技能，炎龍的升級版，清兵效果不錯，其他作用和炎龍相似

23級技能，火雷爆，清兵技能，清兵效果一般，主要是火雷桶是隨機放置，總有一些是在範圍之外的，並且其他的太集中，不能對敵兵造成最大的傷害

28級技能，後伏兵陣，造兵技能，能造兵的都是好技，這也是荀彧唯一拿得出手的技能，其他技能雖然有些還是不錯的，不是不夠優秀

33級技能，雷光燋獄，清兵技能，清兵效果一般，並且需要敵兵高度集中，不然作用更小

40級技能，炎龍絕殺，清兵技能，雖然是炎龍系列的終極版，清兵還是不錯，但不能起到決定性的效果

武將技方面，前期武將技比較好，但也好得非常有限，加上武力低，前期是屬於偏弱的，中期武將技沒有什麼太大的改善，同樣中期也不強屬於非常普通，後期出了伏兵好點，智力高技力長，依靠伏兵堆，最後兩個武將技也沒能帶給玩家驚喜，總體來說，比一般的將領好一點，算不上優秀

軍師技方面，1級的降敵士氣（初），5級的增加技力（初），12級的喝阻敵軍（初），17級的亂陣，23級的恢復體力（中），28級的擾亂敵將（高），34級的恢復體力（高），39級的混亂敵軍（高），前期的增技非常不錯，後期的恢體也可以使用，但是沒有什麼可以支撐場面的軍師技，所以作為軍師來說，還是不夠的

荀彧在群英2中，基本和上場無緣了，留著守城還是不錯的，如圖，是答應大家兵滿的情況下用伏兵，天上掉鐵盆的畫面
