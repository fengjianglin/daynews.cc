游海拾遺：我在《搖滾樂隊不插電》中組了支樂隊，開了場世界巡演

2019-12-17 1,821條評論 4,773次閱讀 3629人點贊

引

電子遊戲誕生至今以年逾半個世紀，誕生了不計其數的經典名作。逝者如斯，個別優秀的作品有幸在歷史上留下了深深地烙印，被玩家傳誦至今，但絕大多名作都慢慢湮沒在時間的長河之中。

本系列筆者想帶領各位玩家老爺回顧一些已經、或即將被人遺忘的優秀遊戲作品。能讓這些覆滿時間沙塵的遊戲重新回到玩家的視野當中，被人拾起遊玩，想必也是一件非常不錯的事，故名為「游海拾遺」。

《Rock Band》

《Rock Band》系列是由著名音樂遊戲開發商Harmonix製作的一款音樂遊戲。該開發商此前最廣為人知的作品是音樂名作《吉他英雄》。Rock Band系列第一作誕生於2007年，在2008年、2010年和2015又分別推出了續作二代、三代以及最新的四代。

《Rock Band》

如今市面上大多數主機以及移動端的音游，採用的都是按鍵或者觸屏操作的方式。例如《太鼓達人》、《DJ Max》、《古樹旋律》、《Cytus》等等，都不例外。《Rock Band》系列則完全不同，必須購買架子鼓、吉他之類配套的外設才能遊玩，這使得該系列遊玩門檻極高，國內幾乎很難體驗到。就以最新的《Rock Band 4》來說，僅遊戲光碟國內電商幾乎沒有渠道可以購買，而外設更是少之又少，以至於某二手交易平台上一套Xbox360的架子鼓外設，標價高達一千元。即使在Harmonix官網購買，吉他同捆包的售價也要400美刀，架子鼓同捆包售價更是高達1400美刀。相信即使有強烈遊玩慾望的玩家，面對這個門檻都會望而卻步，畢竟不是每個玩家都如此硬核，願意為一款音游如此無節制腳花錢。

1399美刀讓人望而卻步

《Rock Band Unplugged》是Harmonix開發，由EA和MTV Games於2009年發行的一款PSP獨佔音游。Unplugged意為不插電，拔掉電源插頭，即用原音樂器演奏，是指不經過電子設備的修飾加工的現場化的音樂表演形式。不插電由演出於沒有重型音色的掩護，對於樂隊成員要求極高，只有技術成熟、現場控制能力極強的樂隊才敢挑戰這種表演形式。

不插電演唱會對樂隊要求極高

《搖滾樂隊 不插電》這款遊戲的獨特之處在於，它是第一款，也是唯二可以使用按鍵方式遊玩的《Rock Band》作品。另一款可以用按鍵遊玩的是NDS的《Rock Band 3》，但是礙於機能，遊戲表現力可想而知。可以說，《搖滾樂隊 不插電》是玩家能輕易玩到的表現力最好的《Rock Band》作品。

PSP獨佔《搖滾樂隊 不插電》

玩法

《搖滾樂隊 不插電》是一款傳統的下落式按鍵玩法音游。玩家需要在音符鍵下落到按鍵區域時按下對應的「←」「↑」「△」「○」四個按鍵，以演奏出一首完整的曲子。大多音游為了提高難度，會一邊把譜面做得眼花繚亂，一邊4keys、6keys、8keys如此增加按鍵數量。以致於音樂遊戲變成了一種難度極高的硬核遊戲。而本作只需要四個按鍵且只有四鍵模式，極大降低了遊戲的上手難度，即使困難模式，也可以取得不錯的分數，實數「手殘」玩家的福音。

僅有4Keys，上手難度相對較低

大體市面上的音游，大多都把整支樂隊演奏的成品——樂曲作為玩家遊戲的核心內容，玩家一邊玩遊戲，一邊聽音樂，考驗玩家的節奏感和手眼配合，以獲得遊戲的成就感。而《搖滾樂隊 不插電》則是將曲子進行分解，把樂隊細分為四位成員，每位成員演奏自己的樂器，與曲譜，四人配合成一首完整的曲子。玩家需要不停地在他們之間切換演奏：每演奏一個小節就必須手動切換到另一位成員。如果這一小節中出了錯，就必須繼續演出下一小節，直到一個小節完成，才可以切換。誠然玩家也可以在出錯後選擇「逃課」，切到其他成員來逃避這段困難的演出，但是會被判定為不計其數的MISS，非常容易造成那位成員的GAME OVER。但如果恰好遇到幾個非常困難的小節，連續失誤，被迫在一位樂手身上停留太久，會致使其他的成員由於你沒去操控而連續MISS，最終導致整支樂隊的演出岌岌可危。於是，玩家遊玩的核心內容就從聽歌遊玩獲得成就感轉變成如何努力使每個成員通力合作，完成一場精彩的演出。

演出失敗

其他音游一首歌往往只需要一個譜面。但是這款遊戲中由於玩家需要操控四名成員，每名成員的樂器不同，演奏的曲譜不同，所以一首歌就需要四個完整且獨立的譜面。加上本作僅僅本體就有40首以上的歌，就需要160個譜面，再加上後期發售的57首歌曲DLC，可以說工作量超乎想像的巨大。

此外，在演奏時有一共五級的連擊等級，每完成一個小節可以提升一個等級。等級越高，每次擊打音符時獲得的分數越多。

下落的音符中有時會出現白色音符。一個小節中至多出現一段，成功擊打它們則會積攢四分之一FEVER值，累計超過一半時按下「×」鍵即可進入FEVER狀態。在此狀態下，連擊等級可以翻倍，五級時可以翻倍到十一級，分數積累極快。所有的分數最後會轉換成星級評價，最高五星。評價越高，一場演出可以獲得的金錢和粉絲也越多，而足夠多的的評星也可以解鎖更多的內容。

連擊等級與FEVER狀態

總的來說，這款遊戲的玩法就是儘可能多地打出連擊，靈活運用FEVER，保證完美演出，積累粉絲和金錢，解鎖更多要素。

臨場感

本作的玩法基本上可以說與市面上主流音游大同小異，但它卻被人評價為「2009年最優秀的音樂遊戲」之一。那麼，除了玩法之外，它到底還優秀在哪裡呢？

筆者認為，《搖滾樂隊 不插電》最大的亮點在於：它大概是至今為止臨場感最強的按鍵式音游。

真正意義上的現代音樂類電子遊戲起始於1996年的《啪啦啪啦啪》，這一類型演化發展至今，玩法可謂五花八門、亂花迷眼，有的結合舞蹈，有的結合動作，有的結合故事，有的結合體育運動，但是鮮有把音樂遊戲與最能表現音樂魅力的現場演出結合在一起的。

《啪啦啪啦啪》重製

正如標題「不插電」所表示的，遊戲想要帶給玩家的就是親臨不插電搖滾演唱會，直面音樂、直面搖滾、參與演出的這種臨場感。

首先，本作選取了超過40首經典搖滾曲目，加上DLC，可玩曲目達到了驚人的97首。這些曲目幾乎都是節奏感極強、使人忍不住搖擺的音樂，極易上頭。如果在演唱會現場，每一首都可以燃爆整個場館。玩家在遊玩的時候，極容易受到音樂情緒感染，加之上手難度不高，幾乎不會感受到強烈的挫敗感，爽快度非常高。

遊戲的譜面設計水準極高，除了踩點準確之外還非常縝密地考慮了玩家手指的律動感和節奏感。有些音樂遊戲為了凸顯難度，經常把譜面音符設計得很散亂，很多玩家會手忙腳亂，往往連按鍵在哪兒都看不清，手指上下左右翻飛，結果還是瞎打一通。而本作則結合了樂曲的旋律，譜面整體設計得很工整，兩三種按鍵搭配有一定規律地出現，即保證了譜面的複雜程度，也保證了玩家能看清楚按鍵的排列，更使得玩家在遊玩時手指的動作能配合著音樂的節拍有節奏有規律地運動，讓玩家有一種彷彿是自己在演奏樂器的感覺。比如節奏感最強的鼓手，會出現類似「←」「△」「←」「△」、「↑」「○」「↑」「○」、「↑」「△」「↑」「△」這樣的有規律搭配，配合強有力的鼓點，玩家一點下按鍵就會覺得這勁爆的鼓聲真是自己擊打出來的。除此之外，由於樂器演奏中經常會出現多段短促的重複旋律，譜面設計還用了大量大段大段的重複連打譜面，踩著點按下按鍵，完成這樣的一段演奏成就感爆棚，可以說是「演奏感」極強。

工整、連續有節奏的譜面

另外，遊戲的音符設計得像是一個個小小的玻璃罐，裡邊裝滿了彩色的濃稠氣體。每當玩家按下按鍵擊打音符，小玻璃罐就會碎裂，裡面的氣體會隨即噴洒而出，「打擊感」極強，大大強化了玩家的「演奏感」。

音符碎裂的一瞬間

既然是現場演出，自然少不了舞台表現。遊戲中的樂隊成員一個個打扮得搖滾范十足，且被設計為「朋克」「哥特」「搖滾」「金屬」四種舞台表演風格，每一種風格都有獨特的演出動作。例如「朋克」吉他手會在彈奏時做踢腳動作，「哥特」鼓手會擺出一個宗教意味濃厚的姿勢，「金屬」主唱會在演唱時比出金屬禮。雖然動作不多，在遊玩的時候也就是幾個動作循環演示，但考慮到玩家專心演奏時其實並沒有閑暇欣賞成員的身姿，作為氛圍的烘托，事實上已經足夠有味。

不同演出風格有不同的表演動作

台下的觀眾和玩家也有一定程度的互動。玩家演奏時並非只有音樂的聲音，就如在搖滾現場一般，場下永遠都會有傳來觀眾各種嘈雜的聲音。但製作組只把這種聲音設定在了隱約能聽到的程度，既表現出了現場演出的氛圍，又不會干擾玩家遊戲。當操作失誤時，台下的觀眾會發出遺憾的聲音，甚至會傳來噓聲。當玩家的連擊等級達到最高時，觀眾則會報以歡呼。當進入FEVER狀態時，整齊響亮的歡呼聲會給予玩家鼓舞。這種互動讓玩家無時無刻處在演出現場的氛圍，而不是僅僅抱著小小的遊戲機，對著冰冷的屏幕在玩一款電子遊戲。這樣的「臨場感」是其他音樂遊戲中幾乎無法體會到的。

在遊戲模式上，為了配合現場演出的遊戲主題，除了常見的「快速遊戲」「訓練模式」之外，還有「樂隊生存」模式，玩家可以任意選取一首樂曲進行遊玩，每個成員的曲譜都會不間斷行進，不會因為玩家切換成員而短暫停止，玩家需要主動選擇切換時機保證樂隊的「存活」。此外遊戲還特別增加了本作主打的「巡演模式」，即生涯模式。玩家需要自己組建樂隊，可以編輯樂隊的名稱、Logo，編輯每一個成員的姓名、性別、外形、膚色、表演風格。筆者就很熱衷於用自己的家人或者朋友的名字來組建樂隊，根據他們本人的性格、造型給他們設定樂器、風格、長相。全部設定完後，就可以開始在全球展開聲勢浩大的巡迴演出，用音樂震翻全世界！這代入感、臨場感可真是太強了！

結

在如今越來越多的音樂遊戲想在玩法類型和主題上做出各種創新，意圖以刺激與驚喜贏得玩家目光，給玩家一種「音樂遊戲竟然還能這麼玩」的感覺時，這款發售於十年前的《搖滾樂隊 不插電》就似乎顯得純粹、質樸、踏實很多。可能在今後很長的時間裡，我們都無法再看到這麼一款能讓音樂歸於音樂，把演出交與玩家的傳統音樂遊戲了。

如果您熱愛搖滾樂，想要見證現場音樂的偉大，這款遊戲可以讓您得以窺探其中奧秘；如果您熱愛音游，希望重溫或領略當年傳統音游的魅力，這款遊戲足以讓您品味曾經的輝煌。如此一款優秀的遊戲不應該埋沒於時間的沙海，我將它再次挖掘出土，希望各位能有興趣一起品味。

謝謝各位耐心看完！
