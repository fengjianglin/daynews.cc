清華大學應屆畢業生轉戰電競 率領隊友扮演WESG黑馬

2019-12-13 1,833條評論 4,582次閱讀 1377人點贊

如果你是一名名牌大學的應屆畢業生，畢業後準備開啟怎樣的人生？繼續考研？對口就業？自主創業？

清華大學應屆畢業生魯巍揚做出了一個驚人的選擇——他將自己的愛好變成了職業，成為了DOTA2項目Ehome i戰隊(二隊)的職業電競選手，參加了今天（13日）在蘇州高新區文體中心舉行的WESG大中華區總決賽。

在以0-2不敵實力強大的LGD之後，魯巍揚接受了上游新聞·重慶晚報慢新聞記者的採訪，講述了一位清華學子和電競結下不解之緣的有趣故事。

▲清華大學應屆畢業生魯巍揚接受採訪

職業電競不認清華大學

昨日（12日），在以2-1擊敗TNT戰隊進入勝者組之後，剛組建不久的全新Ehome i戰隊成為了本次WESG大中華區總決賽的黑馬。

當粉絲們得知Ehome i戰隊的指揮是4號位選手insanity(魯巍揚)，魯巍揚居然是清華大學水利水電工程系畢業生後，直接給他起了一個新綽號——「清華哥」。

今天下午，「清華哥」 魯巍揚面對中國實力一流的LGD戰隊，沒能續寫神奇，以0-2被對手橫掃，一位觀看直播的觀眾在彈幕上評價：「這是一場教學比賽，『清華哥』被上了一課。」

顯然，在電子競技這個全新的舞台，魯巍揚清華大學畢業生的身份，並不能為他帶來勝利，他自己也很清楚這一點：「在DOTA2的賽場，我只是個新兵。作為一名DOTA2職業選手，Ti(DOTA2國際邀請賽)肯定是終極夢想，但要想實現這個夢想我還有很遠的路要走。」

▲魯巍揚在比賽中

回顧和LGD比賽的失利，魯巍揚坦言差距明顯：「LGD太強了，我們還要努力。畢竟我們建隊時間不長，只練了幾套陣容，結果被對手ban(禁用)光了;同時對線技術上他們也比我們更厲害，前期劣勢之後要想通過團戰挽回劣勢很難!這場比賽讓我看到了以後奮鬥的方向。」

所學知識仍有巨大幫助

為什麼會選一位清華大學的應屆畢業生進入職業戰隊?難道是為了噱頭!EHOME電子競技俱樂部經理戴健博否認了這個猜測：「不是的，我接過Ehome二隊招新人的任務之後，首先是考量天梯排名，其次是年齡。至於學校真不是我考慮的!我只能說insanity符合我的要求，同時我也看過他在高校聯賽的表現，他個人也有強烈的加盟意願，所以最後簽下了他。」

為什麼會清華大學畢業會投身職業電競?魯巍揚進行了解釋：「我是通過數學競賽保送清華大學的，因為高考的分數不算高，所以可以選的專業很有限，最後我學習了水利水電工程專業，這不是我喜歡的專業。恰好我在校期間，打DOTA2的天梯排名不錯，所以畢業後我想花一年時間嘗試實現我的電競夢想，如果行不通再回去做正常的工作。」

▲魯巍揚在比賽中

雖然現在從事的電競工作和水利水電工程專業有點風馬牛不相及，但是魯巍揚表示數學競賽的經驗對他還是有幫助：「數學競賽培養的邏輯思維，讓我在比賽中思路更清晰，加上四號位(輔助位)有更多的時間思考，比賽一般都是我來指揮。」

投身電競渴望得到承認

電子競技在清華大學是什麼地位?當記者把這個大家關心的問題拋給魯巍揚的時候，他選擇了實話實說：「說實話，清華大學是不支持電競的，我們學校沒有校方允許開辦的電競社，因此所有活動都是地下進行的。我們班30多個同學，只有4個人打DOTA2。不過因為我天梯的排名還不錯，校友們組織隊伍外出參賽的時候，我都會受邀參加，我記得當年還參加過清華大學和北京大學的電競對抗賽的。」

因為在大學階段魯巍揚就在清華大學的電競圈小有名氣，所以這次他表示這次到蘇州參賽得到了同學的支持：「同學們都給我加油打氣，我的大學好友還跟我開玩笑，讓我一定要努力，等我打出名氣，他才好在聊天時吹噓『嘿，你知道嗎?insanity是我同學!』這次比賽我沒想到能夠打進今天的主舞台比賽，不然我可能會邀請幾個好友來看我比賽了，哈哈。」

大學一畢業就加入了Ehome戰隊，是否意味著魯巍揚得到了父母的首肯?他苦笑著表示現在正在做工作：「我父母並不同意我打職業電競，還是認為我不務正業，讀了清華大學沒有學以致用。估計只有我打出了成績，他們才會同意。」

有趣的是，在採訪的最後，記者詢問了魯巍揚的工資，他的經理戴健博表示：「具體金額不方便透露，但是比畢業大學生還是高一點。」聽到這個說法，魯巍揚笑了起來：「可能比一般大學畢業生高一點，相對清華大學畢業生只是差不多，嘿嘿。」
