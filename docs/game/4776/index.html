蘭陵王馴魔獵人永久下架，限定皮膚返場投票開啟，李白典藏公布

哈嘍，大家好！我是九零小哥，私信「地獄火」，即可免費領取皮膚。王者榮耀這款遊戲還是很好玩的，相信不少人都是王者榮耀的忠實玩家，隨時隨地都能玩一把緊張刺激的排位。而王者榮耀最近也是不斷更新，曝光了許多新內容，新年馬上就要到了，按照慣例限定皮膚返場投票活動，也是即將開啟，另外還有李白的新皮膚公布哦！

S17賽季已經進行快兩個月了，過不了多久就又要賽季更新了，所以還沒上王者的同學可要加油了。有細心的玩家也是發現了賽季更新的世界，戰令系統早就標明了，12月29號左右就會賽季更新。屆時戰令系統也會關閉，蘭陵王的80級戰令馴魔獵人，將會永久下架，所以還沒獲得這款皮膚的玩家，可要抓緊時間了。

馬上就要過年了， 王者榮耀也是準備了不少新皮膚，畢竟到時候大家都有不少的壓歲錢，這麼好的機會自然不能錯過。相信玩家們最關心的就是鼠年限定皮膚了，目前也是確定五虎將，一款傳說加四款史詩。其中關羽的皮膚品質最高，售價也達到了1350點券，但皮膚看上去是非常不錯的。

同時過年的時候，還會進行皮膚返場活動，因此返場投票活動也會早早開啟，玩家可以自己選擇喜歡的皮膚返場。而周年慶已經返場過的3款皮膚，是不會加入返場投票名單的，呼聲最高的一款限定皮膚，就是呂布的天魔繚亂了，小編也是非常喜歡這款皮膚。

最後就是李白的消息了，李白的鳳求凰也是呼聲很高，有很大的概率能返場。不僅如此，李白還會推出一款典藏皮膚，這可是不少玩家期待已久的。新皮膚是大家非常喜歡的古風，終於不是常見的機甲風了，不少土豪也表示，老早就準備好了榮耀水晶，等皮膚上線就直接秒了！
