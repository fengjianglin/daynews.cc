王者榮耀：7個冠軍寶箱開出趙雲未來紀元，楊玉環即將下架奪寶

天天要聞
2019-12-22

王者榮耀在本周上線了一系列AG奪冠的福利，其中就有一個每日參與對戰掉落三個冠軍紀念寶箱的活動，該活動將在明日（23號）結束，期間大家最多可以收集21個，獎勵包含了冠軍陣容永久英雄/皮膚，雖說是概率獲得，但是首充英雄趙雲加入獎池還是吸引了不少玩家參與收集！

此次小編終於歐皇一回，在連續開啟兩輪寶箱後，第二輪一次開了7個寶箱得到了15鑽石、兩份張良一千零一夜體驗卡、永久趙雲未來紀元和5個銘文碎片，當然這次寶箱不乏幸運玩家出現，比如有小夥伴曬出了曜、張良和蔡文姬的288皮膚，不過目前為止小編還未看到有玩家抽到永久趙雲，不知你是否開出了呢？另外今明兩天還可以積攢最後6個寶箱，大家記得做任務獲取哦！

下周版本更新後王者峽谷將迎來雙節主題活動周，屆時不僅有娜可露露新皮膚上線，還將開啟年末最後一個給力福利，人人免費必得SNK英雄，而大家詢問的碎片商店則要等到下月初換新，不過每周一更的鑽石奪寶則會迎來英雄輪換！

那麼在鑽石奪寶獎池輪換之前，本期奪寶獎池加入的英雄是否值得抽取呢？其中曹操和魯班七號都可以消耗588鑽石獲得，還未擁有的玩家無需抽取，而13888英雄楊玉環自加強後勝率有所回升，屬於能奶、能控、能消耗的輔助型法師，同時也適合在開黑時作為核心體系對待，由於大號已經滿英雄了，本期小編依舊使用小號試抽了一波，並在三次五連抽後（810鑽石）後幸運開出！

在獲得楊玉環後如何快速上手呢？首先楊玉環在調整後清線和耗血能力提升，前期使用1技能+普攻額外傷害便能快速清掉兵線，清完兵線後便可中野輔、中野、中輔或單獨前往上下路gank和配合打野位入侵野區，中後期楊玉環除了打拉扯外，以保射手為主，畢竟裝備成型後奶量足，關鍵時刻給射手續命是拿下團戰的關鍵，另外楊玉環依賴大招可以騙敵人技能，因此依據戰場情況打的激進一些，越塔開團，賣自己一波也是可以的！

建議開黑玩家選擇楊玉環+孫臏體系，這個組合打拉扯戰無敵，可在保證自己血量健康的情況下壓低敵方血線，出裝銘文方面，可選CD鞋、面具、巫術、帽子、法穿棒和輝月的組合，銘文則選擇88法穿狩獵/調和的組合，攜帶狩獵不僅能更快打出平A，還提升了支援和2技能控制率。
