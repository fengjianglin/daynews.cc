FGO2.5衛星玩家猜想投票結果，大家都做了哪些猜測呢

天天要聞
2019-12-15

FGO日服主線2.5章即將開啟，前陣子日服媒體gamewith也發起了對四周年公開的2.5衛星從者的猜測，在近期公布了投票結果，雖然國內目前已經有疑似偷跑情報的流出，不過在遊戲正式公開之前，我們也可以看一下到時候有多少玩家奶中了新從者吧。

假面騎士猜想前三位

NO.1 宙斯（得票率28.7%）：在投票階段秋田君也是投了宙斯，因為在2.2的時候組長基爾什塔利亞也已經明確點名自己的盟友是宙斯，宙斯作為希臘神話中擁有最高地位的存在，這次PV的C位以及打扮最奇特的從者應該就是宙斯了，也很可能是本次異聞帶的異聞帶之王。

NO.2 哈德斯（得票率15.8%）：哈德斯是宙斯的哥哥，並且是冥界的主人，圖中這個從者的配色偏黑暗系，與代表光明（閃電）的宙斯相比，這個配色更加偏向哈德斯，不過哈德斯並未在之前的劇情中出現，只靠配色猜測並不太能立足，畢竟單憑一破立繪，大家也很難猜測其他幾位異聞帶之王的身份。

NO.3 珀爾修斯（得票率3.9%）：因為是蒙著臉，所以猜測珀爾修斯的玩家也存在，因為珀爾修斯也是老衛星了，寶具之一也有「披上後能改變使用者模樣的斗篷」，貌似也滿符合上面的設定的，不過從大致的劇情來看，珀爾修斯在這裡登場似乎也有點突兀。

PV Rider猜想前三位

NO.1 奧德修斯（得票率39.1%）：這個從者已經在PV中確定職階是騎階了，奧德修斯有使用棍棒的經歷與這個從者符合，同時奧德修斯在特洛伊戰爭之後乘船返回故鄉也是船長，符合騎階的設定，後來在夏綠蒂的羈絆故事也有提到奧德修斯這個名字，所以奧德修斯的可能性是最大的。

NO.2 奧吉爾（得票率13.1%）：白色挑染很容易讓人聯想到查理曼十二勇士，加上奧吉爾的慈悲之劍也有點像是圖中類似棍棒的武器，不過這個從者既然出現在希臘背景的異聞帶，自然是奧德修斯的可能性更高了。

NO.3 太公望（得票率6.7%）：很驚奇的是日服玩家對姜子牙的猜測非常多，不僅是gamewith，appmedia那邊也很多猜測太公望的，可能是因為武器像是魚竿的緣故？但是鎧甲也太不像了吧。

金髮女從者猜想前三位

NO.1 阿芙洛狄忒（得票率23.1%）：這個從者的裝扮幾乎完全符合阿芙洛狄忒在神話裡面的描述，並且因為本次的從者有了帕里斯和奧德修斯等特洛伊戰爭的從者，而阿芙洛狄忒也是特洛伊戰爭的始作俑者，所以大部分玩家都覺得會是愛神。

NO.2 歐羅巴（得票率20.0%）：作為設定集衛星，並且是地上的女神，歐洲的命名者，完全不需要考慮主神很難落地的問題，本身的神話也是藝術家很喜歡的話題，所以歐羅巴的可能性是很大的。

NO.3 赫拉（得票率7.3%）：因為身著很華貴，並且站在疑似宙斯的從者身旁，所以也有部分玩家認為是神後赫拉的，赫拉也是引起特洛伊戰爭的女神之一。

金髮男從者猜想前三

NO.1 阿波羅（得票率37.0%）：一頭金髮、頭戴桂冠，並且有彈豎琴的姿勢，很容易讓人聯想到阿波羅，加上劇情裡面也說到，組長有一騎可以預言的從者，而阿波羅正是預言之神，可能性也是很大的。

NO.2 俄耳甫斯（得票率12.6%）：阿波羅的兒子，非常有音樂天賦，也符合這個從者彈豎琴的姿勢，另外俄耳甫斯也曾經是阿爾戈號的英雄，和伊阿宋一起去奪取羊毛，用豎琴使守護羊毛的龍沉睡，作為這個從者的準確率也是很高的。

NO.3 赫爾墨斯（得票率6.2%）：赫爾墨斯在希臘神話中的人氣也是非常高的，少年的形象也是蠻符合的。
