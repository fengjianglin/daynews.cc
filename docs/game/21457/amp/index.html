王者模擬戰：扶桑三星武則天毀天滅地？玩家決賽卻敗得很慘

天天要聞
2019-12-20

在王者模擬戰中，品質越高的英雄越難到三星，比如黃忠、呂布、孫臏、蘇烈以及武則天，這些英雄都是五費英雄，平常能升到兩星就不錯了，三星千難萬難。然而，一旦到了三星，這些英雄也將會非常的恐怖，就拿呂布來說，三星呂布跳下去一掃就是一大片。再說武則天，在有裝備和法師羈絆加成的情況下，一個大招恐怕能清屏。

然而，最近有一位玩家，好不容易將武則天升到三星，卻發現並不是自己想像中的那樣一個大招就能清屏，碰到堯天射和坦衛射，居然一個都打不過。堯天射還好，至少還有一個三星的公孫離，可坦衛射一個三星英雄也沒有。再看看他自己的陣容，扶桑法師流，雖然只有五法，但是有羈絆進階，也就是七法，可以增加950的法術攻擊，在此情況下秒不掉一群二星英雄讓他覺得三星武則天並沒有想像中的厲害。要知道，平常大家只有二星武則天，一個大招也是可以將二星輸出全部打殘甚至秒殺的。

我們再來看看三星武則天的技能傷害，暈眩全部敵人並造成3000（+105%法術加成）傷害。也就是說，在沒有法術裝備的情況下，有七法羈絆的法術加成，三星武則天一個大招可以打出4000左右的傷害（當然，這是在沒有計算目標有魔抗的基礎上）。而一般二星射手的生命在4000左右，四費、五費射手的血量更高，有五六千。

如果有法術裝備，比如博學者之怒，不僅可以增加500的法術攻擊，而且可以提升100%的法攻，這就意味著能再增加武則天1000多的傷害；再比如迴響之杖，增加200法攻的同時，還有一個被動，技能命中時發生小範圍爆炸，造成200（+65%法術攻擊）傷害，又可以增加武則天1000多的傷害，如此一來，就算是兩星的黃忠也會直接被打殘，其他兩星脆皮肯定直接被秒。因此，這位玩家拿到三星武則天打不過坦射，恐怕是沒有合適裝的緣故。

畢竟決賽遇到的兩隊都是養豬流，一隊是三衛，一隊是五衛，長城守衛軍只要死掉一位成員，就能增加其他成員的基礎屬性。而且，他們的陣容大部分都是高費英雄，比如黃忠、呂布、蘇烈、廉頗、盾山，這些英雄就算只有二星，生命也是非常高的，再加上有輔助羈絆，三星的武則天也別想一個大招秒掉。
