Dota2：樹老師倒了（震聲哭腔）！新EH出師不利未能殺進官方賽事

天天要聞
2019-12-06

摘要

在剛剛的Weplay布科維爾Minor預選賽中，EH再次對陣RNG戰隊，卻被RNG連續兩局光速碾壓，連敗離開了2019-2020季度的第二個Minor/Major賽事。首局EH選擇了德魯伊、小魚人、小小三核，卻被RNG22手選到了幻影長矛手。

在剛剛的Weplay布科維爾Minor預選賽中，EH再次對陣RNG戰隊，卻被RNG連續兩局光速碾壓，連敗離開了2019-2020季度的第二個Minor/Major賽事。

首局EH選擇了德魯伊、小魚人、小小三核，卻被RNG22手選到了幻影長矛手。中期裝備散失的幻影長矛手如入無人之境，輕輕鬆鬆擊敗EH獲得勝利。

次局EH一崩到底，RNG在線上就取得了極大的優勢，憑藉藍貓、夜魔、火女的活躍發揮，迅速滾起雪球。EH在一波7換0後，局勢已經無可挽回，含恨吞下了苦果。

縱觀EH戰隊在第二個官方賽事上的表現，完全可以用高開低走來形容。

在小組賽階段，EH戰隊7勝1負，其中與VG戰隊戰成1比1平，擊敗了DK、Newbee、LFS戰隊，豪取小組第一。各路主播、解說把EH吹上了天，認為EH一定能夠進入Major。

然而進入Major預選賽階段，EH在第一局戰勝iG後，EH似乎被下圖的加油部部長一口毒奶奶得沒回過神。隨後EH被iG讓一追二擊敗，被茶隊2比0擊敗，失去了Major資格，今日又再次失去了Minor的資格。

EH在Sylar與NothingToSay加入後，整體風格偏快，xiao8也屢次為隊員選擇德魯伊、獸王、先知等英雄，在面對弱隊時確實有良好的表現。但面對強隊就好像原形畢露，怎麼打都推不起來。

在Minor預選賽中，xiao8的BP更是讓人摸不著頭腦，21手選擇小魚人打昆卡，卻破壞了全隊的節奏，耽誤了前期推進，被對方22手更適合節奏的幻影長矛手擊敗。這樣的BP也可以算得上「人類迷惑行為」了吧。

隊伍的狀態持續下降，不見好轉，3、4、5號位的表現遠沒有之前亮眼，經常在前期就會崩盤；塞拉狀態下滑也較為明顯，遠沒有了之前的銳氣；中單莫言也有經驗不足的問題，操刀宙斯的時候極為明顯。

樹老師也倒了（震聲、哭腔）！

從另一個角度來說，也算是好事，EH戰隊可以不用在過年期間到處晃悠了，可以回家陪陪家人過個年了。希望他們這段時間好好磨合，悉心調整，將狀態找回來，在年後的第三個Major打出好成績。

其他賽區中，Gambit、NiP已經確定殺入Minor，歐洲區的Nigma被Vikin.gg戰隊擊敗，落入敗者組。目前中國區RNG與CDEC正在進行決賽，大家可以移步直播間為喜愛的隊伍加油。
