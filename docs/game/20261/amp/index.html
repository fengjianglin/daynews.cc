夢三國：路人戰力匹配中最值得玩的輔助英雄推薦

天天要聞
2019-12-17

新一屆征戰天下任務的開啟，戰力重置之後想要上分的朋友自然增多了不少。而相對於休閑匹配而言，戰力匹配中大多數時候輔助都是必不可少的存在。這次，舞文就為大家簡單的推薦一些在路人戰力匹配中比較值得去玩的輔助英雄。

靈音

推薦指數：☆☆☆☆☆

有關注比賽的朋友，都應該知道比賽中的靈音一直都是站在輔助巔峰的神壇英雄。實際上，不僅是在比賽中，路人戰力中靈音這個英雄的可玩性也非常之高。進攻、防守能力都極其強大，大招與E技能的強大限制能力讓她可以剋制大多數熱門英雄不說，偶爾需要單獨或者協助隊友清理兵線的時候，靈音也是唯一一個能夠做到1秒清線的輔助英雄。

賈詡

推薦指數：☆☆☆☆☆

比賽很少登場，休閑對局中經常被用來當輸出玩的賈詡早已被很多玩家忽略了他本身是一個控制能力極強的輔助英雄。就目前的娛樂模式來看，擁有超強控制能力的賈詡，在輔助位置上的作用其實非常巨大。無論選擇什麼天賦，混到6級總控制時間長達5.5秒（E和F的控制時間）都是對手的噩夢級存在。哪怕你選擇了光頭走線上，W的強大消耗能力也能讓對手感到頭皮發麻。

劉璋

推薦指數：☆☆☆☆

準確一點來看，劉璋是一個4號位英雄不應該算是純粹的輔助。可是大家都知道，路人對局中，基本只有1、2、3號位什麼4、5號位的劃分本身就沒有那麼詳細，大多數玩家都將4、5號位英雄統一看做輔助的原因，所以說劉璋是個路人輔助英雄也毫不為過。作為一個打錢速度飛快的英雄，劉璋能比大多數英雄更快擁有恩惠（不考慮人頭爆發），而在擁有恩惠之後，利用恩惠的回復效果可以輕鬆推進不說，補出跳石之後配合隊友更是能夠輕鬆做到有大必殺，可玩性自然非常之高。

程普

推薦指數：☆☆☆

在寫這篇文章之前，舞文一直都在考慮路人局中的程普到底應不應該算是一個輔助。最後，考慮到這個英雄無論放在路人還是比賽中，最被看重的還是他的轉線支援能力的原因，所以舞文還是將其作為了一個輔助英雄。這個英雄的實戰作用和表現不用多說，最需要注意的是這個英雄雖然操作難度不高，但需要玩家隨時注意地圖動態有不錯的大局觀，所以平時不怎麼玩這個英雄的朋友需要多多練習之後才能打出效果。

曹植

推薦指數：☆☆☆

這是舞文最喜歡的一個輔助英雄，雖然腿短、進攻有餘而防守不足算是他最大的兩個缺陷。但是我們不得不承認，曹植只要配合上一些前中期傷害輸出較高的英雄，進攻效果往往都會非常明顯。哪怕是到了輔助最為脆弱的中後期，曹植在後排提供下技能消耗、控制，給給隊友禁錮什麼的效果也還算不錯的。

或許，在看到這裡之後，有人會問舞文為什麼沒有推薦糜竺、墨子。這裡簡單說明一下，墨子這個英雄不用多說前期強大異常，可隨著遊戲來到中後期實際作用會大大降低不說，操作難度也非常之大（舞文當然不會告訴大家我自己不會玩這個英雄，所以一直不怎麼喜歡他了，前半句是真的，後半句是開玩笑）。至於糜竺嘛，強確實是強，可相對於上面提到的幾個英雄而言，糜竺對英雄熟練度要求極高就導致了舞文真的不敢給大家直接推薦。當然，如果你的英雄熟練度足夠，這個英雄同樣是路人對局中的殺神級輔助。
