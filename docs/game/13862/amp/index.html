游我推薦：像散文詩一般的遊戲——劇情向佳作《看火人》

天天要聞
6 天 ago

“游我推薦”是遊戲頻道推出的一檔全新欄目，每天，這裡都會向你推薦一款值得下載的好遊戲。關注這個欄目，在不知道玩什麼時，可以從這裡尋找靈感。

從遊戲頻道首頁的”游我推薦”，或本文末尾，可查閱欄目過往內容。

題記—— 「初聽不識曲中意,再聽已是曲中人」。

《看火人》曾獲2016年unity最佳3D視覺體驗獎，在金搖桿獎中獲評最佳獨立遊戲，而其總銷量也超過100萬份。不論從那個方面來看這都是一款成功的獨立遊戲。

但這款遊戲在國內並不怎麼火，可能也就比前輩《史丹利的寓言》強上一點。我接觸這款遊戲其實挺早的，只是當時的我並不懂得如何欣賞這種「步行模擬器」，甚至一度覺得這種慢的要死的劇情肯定不受歡迎。

確實，和隔壁充滿想像力的《艾迪芬奇的記憶》相比，《看火人》是有些平淡。但它就像一杯清茶，如果品茶的人沒有一定的人生經歷是難以參透這清淡背後的深意。《看火人》的平淡實際上正是真實生活的展現，就像《父親寫的散文詩》一般把真實展現在了那些平凡的話語和行動中，如果不仔細去體會恐怕難以發現。

介紹一下遊戲內容吧。整個遊戲過程可以簡單概括為：玩家根據任務指示不斷地在地圖上走過去走過來。由於遊戲中沒有任何需要操作的地方，所以玩家的操作只是走路、看風景、聽劇情而已，稱之為步行模擬器非常形象。

玩家扮演的是森林中的一個看火人（護林員）亨利，在夏季中負責彙報森林的火災情況維護森林的安全。實際上玩家在遊戲中也不會見到其他任何角色，而角色之間的對話也僅僅只能通過對講機來完成。

對講機的對面是一個叫黛利拉的看火人，她沒事就喜歡用語言撩主角而主角其實也會撩她（取決於你的選項）。然而主角實際上是個已婚人士，只是妻子得了阿爾茲海默症回到了娘家休養。主角正是為了擺脫這些煩心事才來到這裡，而黛利拉同樣也有需要逃避的東西。所以這兩個寂寞的人在這寂靜的森立裡面竟然擦出了火花，說實話我覺得這有點「婊」。

搭訕能力滿分

但後來仔細想了想，在如此孤獨的環境下，只有彼此的兩個人發生點什麼不都很正常嘛。況且兩人充其量算是網戀，面都沒見過全靠語音調情。而作為玩家的你，實際上看上去有點多餘。因為我們很難融入到這個中年男人的角色。我們幾乎什麼都控制不了，甚至不同的選項也不會改變劇情的結果。

你肯定會問那玩家玩個什麼？其實一個文學作品本來就是仁者見仁智者見智，不同的玩家會從《看火人》讀出不同的東西。而如果你想知道這遊戲有什麼可玩性，我建議你自己去玩一玩體會一下。我相信那是只屬於你的獨特體驗。PS：遊戲美術很不錯，這也是我堅持玩下去的動力

主角和妻子的自拍

景色很棒

孤獨是我們永遠的夥伴

孤獨不是在山上而是在街上，不在一個人裡面而在許多人中間。——三木清

遊戲中要表達的情感是多層次的複合的，但有一個情感卻是異常的強烈那就是孤獨。

什麼是孤獨？相信各位都有自己的看法。在我的印象中感到最孤獨的時刻就是大學時候躺在操場上看著浩渺的夜空。看見在月亮旁邊不斷漂浮的雲朵，突然就感覺偌大的操場就只剩下我一個人。一個渺小的人面對如此廣闊的星空，那種孤獨感是溢於言表的。相比起大自然來說，我們人類就是最孤獨的最渺小的。

而在《看火人》中，玩家就可以真實地感覺到這種孤獨。在繁星點綴的星空之下，玩家可以隨意地走走，即沒有必須完成的任務也沒有任何野外生物，這個世界只有你以及對講機那頭的黛利拉。你大可以關閉那煩人的對講機，一個人在夜間漫步欣賞遊戲中那極為優秀的風景。

雖然由於經費限制，遊戲中的遠景都是貼圖而已但看上去還是別有一番味道。而當你在這個世界走上幾次之後，你才會真正體會到那種孤獨。遊戲中沒有任何一個可以和你互動的活物，哪怕是一隻熊、一頭鹿都沒有。雖然背景音樂中會有動物的叫聲，但在地圖上卻沒有。

這是作者偷懶不加嗎？我想並不是，因為加上幾個動物不過多幾個貼圖而已並不算費事。作者更多恐怕是希望玩家一個人呆在這個世界。我一路上驚嘆於風景的美麗，卻再無其他人可以交流，這種孤獨怕是最難受的。所以作者設計了對講機，以免玩家把遊戲當作了旅遊景區的宣傳片。

我說孤獨是遊戲的主色調，那麼對講機就是引導寂寞發泄的工具。初來乍到的主角開始還對對講機背後的上司有一定戒備。對於她過於熱情的問候顯得非常不適應，畢竟他也是有老婆的人意志還是比較堅定。但是隨著時間推移，在這人跡罕至的森林呆著也確實太過孤獨。因此從以前被動地接受黛利拉的調情，到後來主動找黛利拉說話。

在遊戲中，玩家遇到一些新奇的事物或者地點就可以主動和黛利拉對話。當然也可以不對話，反正完全不影響劇情發展。而這種可以對話的機會實際是隨著遊戲深入而逐漸變多的。雖然遊戲本身沒有表現出孤獨，但你可以通過主角的轉變來發現這種寂寞。從一個不怎麼講話的高冷男子，到最後任何事情都想跟黛利拉分享的痴漢。

能做什麼？成年人都懂

起初在b站看遊戲視頻的時候，我很難理解這種變化。但當我自己親自去玩的時候才發現，原來這就是寂寞。兩個被世俗所拋棄的人，在一個充滿了未知危險的森林。他們除了和彼此交流之外再無其他可以說話的人，而且由於黛利拉的形象全程沒出現所以玩家並不知道她長什麼樣。但玩家仍然會yy聲音的主人會不會是一個大美女，男人嘛在這種情況下總是喜歡幻想。

還好有一個對講機，我相信如果是你處在那樣的環境下就算對面是一個你最討厭的人你也會找他說話。不信你可以試試看，如果整整一周都一個人呆著不和任何人說話你會不會瘋掉。人類畢竟是群居動物，離開了群體那種孤獨感就不斷攀升。而這種孤獨隱藏在美麗的湖光山色之中，如果不是心思細膩的玩家恐怕很難發現，畢竟我們早已習慣了鋼鐵叢林的生活哪知道孤獨是什麼味道？

愛與責任

中年以後的男人，時常會覺得孤獨，因為他一睜開眼睛，周圍都是要依靠他的人，卻沒有他可以依靠的人。————張愛玲

或許有的玩家覺得這主角就是個渣男，老婆得病了不好好陪著躲到森林裡面和其他女人曖昧。但這樣的玩家顯然有些年輕，雖然以我的年紀來說要說清什麼是愛有點困難。不過至少我明白愛是有不同的等級和形式的。

主角是愛他的妻子的，所以他才會一直在遊戲中流露出對於妻子的擔心和不舍並在結局中選擇去看望妻子。但他的愛並不是那麼強烈，也做不到不顧一切。他不能犧牲自己的一切去照顧患病的妻子，他有著自己自私和懦弱的一面。也正因為這種人性的弱點，才讓亨利這個角色顯得真實。

他不是韓劇中那樣英俊帥氣又多金深情的長腿歐巴，他只是一個腆著肚子的中年男人而已。他愛他的妻子但他又沒有足夠的勇氣和毅力，因此在面臨人生選擇的時候他躲進了森林。他開始是希望在森林中能忘掉一切煩惱的事物，忘掉凡塵俗世好好過眼前的生活。但在最後那場生離死別的山火中，他才突然醒悟。自己與黛利拉的一切才是鏡中花水中月，於其說是愛不如說是寂寞時的生理反應。而自己真正愛的還是妻子，自己不應該再去逃避。

山火也有自己的名字「六月之火」

因此亨利最終還是拿出來勇氣去面對自己的人生，而黛利拉呢？其實我不太理解，我個人認為黛利拉還是選擇了逃避。她不願意和曖昧的主角見面，因為她並不想在現實中和任何建立這種親密關係。她因為自身的情感問題害怕愛情，但又喜歡享受這種寂寞時的曖昧。或許女人就是這樣的奇怪和矛盾，一面追求愛情一面又在逃避。而患有創傷後應激障礙（PTSD）古德溫顯然更為悲慘，他不僅失去了自己摯愛的兒子也丟掉了一個父親該有的責任。選擇了逃避的他自然也迎來了最為悲慘的結局。

其實每一個進入森林的看火人都是在逃避那些本該屬於自己的責任，他們像是不食周粟伯夷、叔齊，名義上是為了讓自己冷靜一下給彼此保留空間。實際上就是逃避本該盡的責任，害怕面對現實的世界。但這樣的逃脫畢竟是暫時的，當主角開始喜歡上了這種深山生活之後卻發現有更多危機潛藏在背後。直到最後那一場「六月之火」徹底摧毀了森林。《看火人》並沒有明確地告訴玩家是不是應該逃避或勇敢面對，它只是展現給了玩家一個中年男人的逃避與成長。

帶有懸疑味道的情節

情節必須是動人的，因為一切的心靈都要求受到感動。————伏爾泰

鐵柵欄之後到底是什麼?

《看火人》的情節實際上非常簡單，一遍看下來玩家也能理解的八九不離十。但簡單情節之下卻又包含了不少的信息，只是這樣的一些信息到底是不是真實的呢？

遊戲中的物品都是沒有任何實際作用的，只是包含了一定量的信息。比如一些照片、素描、字條等等。但讓人奇怪的是這樣信息卻都是無關緊要的瑣事，這實在讓人費解。我看過其他作者曾經說過，這種設計類似於反契訶夫之槍。這把槍實際上就是契訶夫的一種理論：他認為如果在小說的前文中詳細描寫了一柄獵槍，那麼這個描寫就一定是有用的。也就說在之後的章節中，或早或晚這把獵槍一定會響，否則前文的描寫就是毫無意義的。

原文如下：Remove everything that has no relevance to the story. If you say in the first chapter that there is a rifle hanging on the wall, in the second or third chapter it absolutely must go off. If it’s not going to be fired, it shouldn’t be hanging there. ——”Chekov’s Gun”

遊戲中有不少文本信息，但沒啥意義，只能算是在渲染氛圍

而我不認同這樣的觀點，遊戲中確實有兩個很詭異的支線。

一個是主角覺得森林中的科學家在秘密研究什麼土著圖騰或者神秘生物（有點克蘇魯味道），最後他前往營地並沒有發現什麼奇怪的東西，這個支線也就不了了之。

另一個是主角一直被監聽和襲擊，似乎背後是一個在研究他們對話的神秘組織，並企圖使用特殊方法控制他的行為（腦控？），最後你會發現那神秘人是前任看火人古德溫。他也是神神叨叨的，所以寫的東西也不可信，這條支線也就草草收場。

古怪的圖騰

在實際遊戲中，我確實被這兩條線索吸引，我一度以為這背後可能是一個類似於《克蘇魯呼喚》的故事。可惜我想太多，它就是一個步行模擬器而已。那麼到底為什麼要設計這樣的情節呢？是設計者的惡趣味還是經費的限制呢？

其實從整體基調來看，本作是關於孤獨和責任的，和這些獵奇故事顯然不搭。所以本作設計這些故事顯然不是因為經費不足才草草結束，而是能在某種程度上呼應主題。那麼如果出現這種獵奇的類似於小說情節的事件，你會認為是真的嗎？當然不會。

主角也是如此，但是隨著時間的推移和事件的不斷發生，可以明顯感覺到了主角的懷疑慢慢增加以至於在科學家營地中險些無法控制自己要放火燒山（牢底坐穿）。可奇怪的是主角剛剛一走營地就自燃呢？我傾向的解釋是，主角的內心狀態在不斷惡化。從開始的意志堅定，到疑神疑鬼，直到最後盲目自信。他的理性讓他無法相信外星人、政府秘密實驗之類的說法，但潛意識中卻早歇斯底里。我有理由懷疑他在科學家營地時就不受控制燒毀了營地，而自己的主人格並沒有發現這一切。這也對應了壞結局中，主角因為無法面對自己而走入火海。

到底是確有其事？還是妄想？

因此這些看似荒誕而誇張的情節並不是沒有意義，而是以一種隱晦的手法在展現主角的心理狀態。相信我，在那種深山呆上幾個月遲早會發瘋的，至少會變得神經質。這一點不知道是不是借鑒了《閃靈》中的那個情節，反正那把斧子很像。

情節上確實有不合理的地方

小結

當然即使沒有這些離奇的情節，《看火人》依然是一個很幽美的故事。

在這散文詩一樣的故事中，每個玩家都會感悟到不同的東西。是山火之下對於生命的認知，是深夜輾轉之際對於愛的渴望，亦或者生離死別之間對於責任的感悟。

每個人都會有自己的體會，其實很多玩家也評價這個遊戲除了風景好看以外情節一塌糊塗。實際上這也是很正常的，《看火人》裡面的內容是需要一定閱歷才能體會的。特別是其中的中年危機部分，我至今都無法體會。

但相信我，只要你慢慢去品一定會有所收穫的。
