全明星UZI捍衛ADC最後尊嚴，薇恩單挑琪亞娜，怎麼贏的我不知道

2019-12-07 0條評論 18次閱讀 0人點贊

摘要

LOL全明星已經打完了第二輪，其中SOLO賽是所有環節最為緊張的，強如Doinb以及FPX小天都沒能扛住，整個LPL就Theshy和UZI晉級了，可見Theshy和UZI的實力是多麼強，畢竟SOLO賽可都是一次性的，輸一次就直接淘汰。

LOL全明星已經打完了第二輪，其中SOLO賽是所有環節最為緊張的，強如Doinb以及FPX小天都沒能扛住，整個LPL就Theshy和UZI晉級了，可見Theshy和UZI的實力是多麼強，畢竟SOLO賽可都是一次性的，輸一次就直接淘汰。

而且UZI還在八強SOLO中捍衛了ADC最後的尊贏，以版本弱勢ADC英雄薇恩，毫不畏懼的單挑最強刺客琪亞娜，就連怎麼贏的我都不知道，下面就跟著甲子一起看看細節到底怎麼回事。

​

UZI這次的對手是Caps，一個非常強的中單選手，相信不少LPL粉絲都熟悉他，他曾經是擊敗過UZI的，所以UZI這次直接鎖定了薇恩，想要用自己的本命英雄來找回場子。說實話UZI這個選擇確實很冒險，因為S10版本的ADC真的很弱。

各種穿甲裝備和斬殺Buff讓ADC變得猶如紙張一般脆弱，基本就是到了碰一下就死的局面，很多ADC玩家應該感受到了這個版本對ADC有多麼不友好。

​

甚至不少玩家都放棄了ADC這個位置，就連LOL全明星的選手都很少選ADC了，比如BANG和UZI進行SOLO的時候，他都沒有選ADC。但是唯獨UZI一直堅持使用ADC，但是看到Caps拿出琪亞娜之後，很多人都覺得UZI必輸。

因為現在的版本琪亞娜可是最強刺客，秒人不用一秒鐘，一個Q就可以要ADC半條命，而在對線的時候UZI確實被壓制了，ADC果然還是不好打刺客的，哪怕UZI也是有點招架不住。

​

看到UZI被擊殺之後，很多人都說沒了，但是隨後UZI卻沒有放棄，他回去補了一點裝備，然後憑藉著自己的對線實力和距離把控，慢慢地撐到了最後。而Caps看到決賽圈開始之後，就直接突臉要秒UZI。

Caps的琪亞娜第一套突臉技能直接打掉了UZI半血，但是UZI趕緊大招加Q技能隱身拉開距離，反手就是一套三環逼出了Caps的金身。這時候Caps進入草叢等待UZI大招消失，但是UZI並沒有上去，而是轉身去A小兵回血，慢慢的把血量回上來了。

​

但是這時候Caps的技能已經冷卻好了，他還是有一套秒掉UZI的傷害的，Caps第一個Q技能打到了UZI，但是被UZI瞬間開盾給抵消了，之後的大招更是被UZI貼臉躲掉，所有技能打空之後，Caps就進入了被動。

而UZI直接控制距離打出三環反殺了Caps，整個過程UZI算錯一步就會死，全靠他躲技能和極限護盾以及吸血才反殺琪亞娜，在UZI贏了之後很多人都不敢相信，因為現在哪裡還有ADC敢反殺刺客的？

​

​甲子認為UZI這一波真是捍衛了ADC最後的尊嚴，哪怕版本對ADC不友好，在其他玩家和選手放棄ADC的時候，UZI卻堅持使用VN和ADC，還擊敗了很多選手，真的是讓人熱血沸騰。而且反殺的過程你看一遍根本都不知道怎麼打贏的。LPL的粉絲們，對於UZI這件事你們有什麼想說的呢。
