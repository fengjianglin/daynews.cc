2019-12-03

摘要

《英雄聯盟》S賽算是英雄聯盟職業聯賽中最具權威性的比賽了，而直播英雄聯盟S賽的直播平台也有很多，最主要的就是國內四大直播平台：鬥魚、虎牙、快手、B站。這四大平台是目前國內最有名氣的直播平台了，如果熊貓還沒倒閉的話，應該也能算上它，不過既然事情已經過去，我們就不再贅述了。

《英雄聯盟》S賽算是英雄聯盟職業聯賽中最具權威性的比賽了，而直播英雄聯盟S賽的直播平台也有很多，最主要的就是國內四大直播平台：鬥魚、虎牙、快手、B站。這四大平台是目前國內最有名氣的直播平台了，如果熊貓還沒倒閉的話，應該也能算上它，不過既然事情已經過去，我們就不再贅述了。

不過呢，這都是以前的事情了，就在剛剛，新京報獨家獲悉，小破站（B站）斥資8億，拍下了未來三年英雄聯盟（LOL）全球總決賽中國地區的獨家直播許可權。其他參與競拍的企業有上面提到的幾大平台，以及其他小平台等。

也就是說，包括2020年的S10在內，未來三屆英雄聯盟S賽，只會在B站直播，我們只能在B站看到英雄聯盟S賽的直播了。沒想到人們口中的小破站竟然如此財大氣粗，竟然花費驚天的8億拍下了LOL全球總決賽直播許可權。

對於這個獨家版權的價格，很多圈內人士表示給高了，5億以內是最為合理的，因為賽事的用戶打賞、轉化情況都不高，這應該也是其他幾大平台沒有繼續跟進的原因。不過就目前的情況來看，小破站已經陸陸續續買下很多東西了，難道小破站要有大動作了？

這回看誰還敢叫B站小破站，人家的財氣逼迫得其他幾大平台不敢競價，這種實力不可謂不強。只是不知道在B站看直播卡不卡，以前都在鬥魚和虎牙看S賽，在B站看S賽的小夥伴可以分享一下，你們在看直播的時候觀賞體驗好嗎？
