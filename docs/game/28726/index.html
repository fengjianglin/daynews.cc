Hero不敵XQ，塵夏最初首發，久誠未能登場，XQ九尾中單關羽都能贏

2019-12-22 2,130條評論 3,253次閱讀 1627人點贊

王者榮耀冬季冠軍杯預選賽全面開戰，5支KPL隊伍和3支KRKPL隊伍展開激烈爭奪。賽程是緊張刺激的BO1，誰都不容有失。在經歷了兩場KPL和KR之間的對決之後，終於迎來了KPL賽區的內戰，XQ對陣Hero久競。這場比賽XQ的首發陣容依舊沒有任何變化，而Hero久競這邊上了兩個老選手塵夏和最初，但是中路位置上久誠依舊沒有登場。

XQ這邊的陣容選擇和揭幕戰差不多，依舊是娜可露露的野核陣容，但是這次有了姜子牙體系的加持。Hero這邊是比較傳統的馬可波羅為核心的陣容，中路久羽並沒有選擇工具人中路，而是拿了沈夢溪這個大法師。而XQ這個姜子牙並不是中單，而是走到了輔助位。而九尾則是拿到了關羽中單，這也是九尾的關羽在賽場上的首秀。

因為姜子牙在改版之後沒辦法快速升2級，所以前期也沒有拿到什麼優勢，等到2分鐘的時候XQ四個人到達四級，但是暴君團反而被Hero打出了一個零換三。七分鐘的時候XQ在中路打出零換二，九尾的關羽連續救了姜子牙和娜可露露，一點都看不出這是一個中單選手的關羽。此後XQ利用陣容的突臉特性強行開團，最後一波雙方在暴君處打出激烈的團戰，場面非常混亂，XQ的娜可露露和姜子牙站到了最後，而Hero只剩下一個豬八戒。但是塵夏的豬八戒出現了失誤，不到半血的他並沒有選擇回家，而是殘血強行防守高地導致被娜可露露擊殺，娜可露露和姜子牙帶著主宰先鋒推掉水晶。

坦白來說這局比賽能感覺到XQ的實力還是要比Hero強一些的，包括BP上也要更好。Hero輸掉這場比賽也不是上不上久誠的問題，久羽打得並沒有什麼問題。隊伍的整體節奏不如對手，而不是在中單這一個位置上有差距，可以說Hero除了久酷的馬可波羅比較亮眼之外，其餘的人發揮都很一般。塵夏和最初兩位老將可能是因為許久沒有打比賽了，整體節奏上和幾位新人好像也不在一個頻道。至少從第一場比賽來看，Hero想要從這次的預選賽中脫穎而出難度很大。不過久誠這次是化了妝的，應該有輪換的想法，也還是希望久誠能夠在之後的比賽中有上場機會吧，不說固定首發，起碼還是應該有個輪換的資格的。

大家覺得Hero打不過XQ是因為久誠沒有上場的原因嗎？歡迎在評論區留言交流。我是天黑君，期待下次再見！
