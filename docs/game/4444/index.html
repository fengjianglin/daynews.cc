宮本武藏：超強控制，專切c位，讓敵方後排沒有生存空間

文|天雲解說原創，盜載必究

【摘要】小夥伴們大家好，歡迎閱讀本期的文章攻略!本期文章的主角是宮本武藏，這位英雄從上線峽谷之後，便是經歷了大大小小的多次改動，但即便這樣強度還是非常不錯的，在這個射手崛起的版本，宮本更顯強勢，並且現如今的勝率也是十分不錯的，而且宮本這位英雄的機制也是能讓他勝任多個位置，如打野或者邊路，強大的控制能力絕對是敵方後排脆皮的夢魘，那麼接下來讓我們一起來聊一聊他的一些作戰技巧！


召喚師技能–終結 懲擊

走邊路宮本可以帶終結整體更強勢一些，關鍵時刻可以利用終結彌補傷害，無論前後期一套的爆發傷害都能給敵人造成不錯的威懾力，打野位我們選擇懲擊這是毋庸置疑的。

出裝銘文

紅色十異變 藍色五奪萃五狩獵 綠色十鷹眼

急速戰靴 暗影戰斧 宗師之力 無盡戰刃 破軍 名刀司命

銘文方面還是百穿銘文，畢竟宮本這個英雄還是靠技能和被動的強化普攻來打傷害，高額的物理攻擊和穿透讓他傷害更大化，藍色奪萃增加宮本的續航，狩獵增加宮本遊走的速度和打出強化普攻更流暢。

裝備的話，鞋子我們可以看情況來，急速戰靴增加宮本的移速和攻速，如果敵方射手傷害高可以出影忍之足，對方控制多出抵抗之靴，暗影戰斧是最適合宮本的，所以第一件出，先預購隕星，隨後一把宗師之力提升宮本的全面屬性，無盡戰刃增加宮本的暴擊傷害，隨後再來個破軍再度增加宮本的傷害能力，名刀最後一件出作為保命裝，全輸出裝的宮本後期蹲到敵方後排基本上可以一套秒！

前期思路

宮本這位英雄大招是他的核心技能，在線上我們四級前也是以猥瑣發育為主，靠一技能清線，二技能也是可以利用好增加自身護盾與敵人換血，到達了四級的宮本才可以說是真正強勢了起來，在線上你也是可以掌握主動權，有打野前來宮本開啟大招還是很容易留下敵人的，但主要還是宮本自身多去遊走，有大招再露頭，沒有大招的宮本威脅能力很低，但是也要注意，支援看的是質量不是次數，沒有大招我們就老實一點利用被動去減少自身技能的CD再找機會遊走。

團戰打法

宮本的針對性很強，所以打團是以敵方後排為主，如果敵方後排敢單帶，那麼宮本是很容易將其擊殺的，但真正開啟團戰後，宮本建議等待己方隊友先手開團，這時繞後去觀察敵方後排的位置，只要宮本切掉敵方一位後排，那麼團戰基本上是很難輸的，如果己方處於一個劣勢，那麼宮本可以選擇留下來，利用大招去反制敵人的刺客，只要敵方刺客敢到後方來，那麼宮本開啟大招配合隊友迅速擊殺敵方刺客後，也是可以不斷尋找機會觸碰到翻盤點！

操作細節

宮本大招CD大家可能會認為比較長，但並不是這樣的，我們可以利用好被動去減少自己技能的CD，這樣大招CD短也是可以找到更多的機會，一技能由於過程緩慢，很難命中敵人，所以我們開啟大招可以立刻銜接一技能，而且一技能有個屬性，可以阻擋除防禦塔泉水外的所有飛行技能傷害，這一個技能釋放的好，擋下諸葛亮妲己這類英雄全套傷害也是不在話下，二技能位移距離比指示略微長一些，開啟大招後二技能還會獲得額外突進距離，而且大招會讓宮本進入雙刀流，這個期間的宮本技能或普攻都會附帶法術傷害，利用這個特性在這時間段內多去打一點傷害，對了，玩宮本注意開啟頭像鎖定，將大招拖動到你想要輸出的敵人的小頭像上，這樣才能大中你想要輸出的敵人！

英雄總結

（1）一技能可以擋下敵方的飛行技能傷害。

（2）開啟頭像鎖定讓宮本大招命中想要輸出的敵人。

（3）二技能命中敵方單位可以獲得護盾並減少50%的冷卻時間。
