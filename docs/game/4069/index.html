「地下城與勇士之小狐狸君愛談遊戲電競」第七百一十四期《DNF紅隊3500智力瞎子：團長真天帝，玩家看到獨立，直接退出隊伍》，關於面板團長問題，一直是玩家吐槽的對象！雖然神豪的面板一定很高，但是反過來面板高的不一定是神豪！有的玩家就是為了打團挖空心思，不去打造三攻和屬強，卻對四維面板情有獨鍾！更有甚者，獨立不到2000，面板卻堆到了3500左右。真是江山代有才人出啊。用3500的智力，成功的取得了團長的信任！老規矩搞快點，直接上這位瞎子的打造面板的截圖展示：

DNF紅隊3500智力瞎子：

這位瞎子的打造可謂是非常的奇葩，普雷Raid的固傷職業，獨立起碼要求是2300~2500之間。3仔要求低一點，純C必須達到2500才能打出必要的傷害！但是這位阿修羅玩家，獨立攻擊居然不到2000！但是智力達到了3500的高標準，不用輸附魔都是四維智力。就算是3500的智力，2000左右的獨立，黃隊養老的傷害都不夠，可見這位團長也是著了瞎子的道！雖然是全身超界、紅字+8~10之間，但是打超漩渦都比較困難。

團長真天帝玩家：

這位玩家的打造，可謂是非常的犀利，上衣65智力、首飾38~34四維、下裝55智力、鞋子50智力……基本上就沒什麼合格的打造，特殊裝備15的屬強——安圖恩時代的附魔，輔助左槽42三攻，真是連刮刮樂都不如。但是團長居然放到了紅隊，充當普雷的輔助輸出擔當。面板團長真是讓玩家哭笑不得！

玩家看到獨立，直接退出隊伍，

不過在甲板等待時間，玩家看到這個打造的瞎子在紅隊，就知道團本基本上過不去了。要嗎炸團、要嗎效率奇低。還不如乾淨去找下一個隊伍。於是玩家選擇退出了攻堅隊！畢竟現在團本，就算是狂人的16號，還是要輔助奶和輔助C的，這樣才能迅速和效率通關。常規的玩家，看到這個打造，還是知難而退的比較靠譜。
