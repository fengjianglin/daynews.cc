爐石傳說：樹多勢眾，鋪不完的場！外服傳說前百樹人德爆誕

2019-12-13 1,149條評論 2,393次閱讀 900人點贊

終於，小樹苗成蔭

眾所周知，暴雪想Buff樹人德由來已久。樹人一直都沒有」野獸「」惡魔「那樣獲得專屬的屬性詞，而且有著多個不一樣的樹人卡圖，但它們的屬性都統一是2/2的白板。幾個卡包以來，樹人相關卡牌都是走的鋪場路線，所以樹人也順理成章的成為了超生德/鋪場德的組成部分，比如單卡植樹造林，森林的援助等。

樹人在上上個版本曾今短暫出現」樹人德「的構築，以樹人鋪場，植被破碎機減費為核心，不過很快就被環境所淘汰，樹人德的組件也化作超生德的幫手繼續存活。進入《巨龍降臨》版本，設計師再次拾起樹人，德魯伊新卡中包含了多張樹人新卡，更是直接將一張橙卡分給樹人，可見愛之心切。

新版本第二天，樹人德不負眾望，外服玩家taraba使用一套全新的樹人德打到了傳說22的高位，強度可見一斑。下面是排表，慣例卡組代碼見文末~

taraba傳說22樹人德

一句話攻略

-樹人德的思路保持不變，依舊是鋪場，野蠻咆哮群Buff完成斬殺。不過新版的樹人德強度提升了一大截，鋪場能力是前所未有的強力，解不完的場能讓所有對手絕望。

-前期1,2費隨從鋪，中期植樹造林，園丁鋪，後期自然之力，森林的援助鋪，在去掉了群體Buff，比如遠古祝福之後，卡組裡幾乎每一張卡牌都為了鋪場而來。

-2/2的樹人再多，它終究還是略顯單薄，群Buff還是有必要的。這個時候就需要強力樹人格魯的登場了，樹人全體+1/+1，而無面腐蝕者將2/2的樹人變成5/4突襲則是中期十分強力的一波。

-新卡的加入對於樹人德的強化肉眼可見。2費盆栽投手，總身材3/3，重要的是送一個樹人，為後面配合強力過牌空氣盆栽提供了巨大的便利。多數情況下，0費-1費就能過兩張牌，源源不斷的鋪場帶的必然是巨大的手牌消耗，及時的補充彈藥，攻勢自然是一波強過一波。

-哦對了，樹人德現在有了自己的天胡。沒錯，卡組全體+2/+2,而且不要費用！森然巨化，不虧是紫卡品質呀！強度果然不凡，有了它，你每一個隨從都能比對手的大上兩圈，二王子的Buff前後的差距大家已經有所領略，這次無疑是強化版，即使需要多花費一個法力水晶。

-當然，樹人德離無敵還很遠，除了卡組本身還有優化的空間外，其最大的問題還是沒有賴場能力，也沒有返場能力。樹人都是2/2的白板隨從，沒有任何的亡語，直到現在可能還是賴場能力最弱的鋪場卡組，而返場能力為0，在被對手解光之後，只能硬著頭皮，閉著眼睛繼續鋪。雖然新版本樹人德的鋪場能力增強不少，不過弱點猶在。
