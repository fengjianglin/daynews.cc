懷舊服：不想練級？詳細分析從10級“躺”到60級到底需要多少金幣

天天要聞
4 天 ago

懷舊服到目前已經開放了三個半月（怎麼感覺像是過了大半年……），想必很多小夥伴都練起了小號。由於大家的遊戲時間有限，或者不想辛辛苦苦重複做大號已經做過的任務，於是很多人就想著「找人帶，輕輕鬆鬆躺」到60級。

本文小O將根據詳細的數據來分析一下找人帶刷副本經驗到60級到底需要花費多少金幣。

1-60級到底需要多少經驗？

首先我們需要知道從1到60級每一級所需要的經驗值，這個是計算的基礎，翻了翻資料，小O找到了這個表格：

  • xp：每級所需經驗
  • overall：當前等級所需全部經驗
  • xp/mob：當前等級每個怪物經驗
  • mobs/lvl：當前等級升級需要擊殺的怪物數量

躺到滿級的經驗分析

練級階段：1-10級

這個階段需要自己練級，因為10級是死亡礦井的門檻，好在前十級需要花費的時間是很少的。

第一階段：10-16級

副本：死亡礦井

分析：

  • 10-13級的寶寶經常有人免費帶的
  • 如果16-20級繼續呆在死礦的話經驗會比監獄還多點，但是死礦一趟得要30分鐘左右
  • 由於死礦刷一次時間長但是收費又低，幾乎屬於做慈善，所以帶死礦的人其實很少
  • 部落方面，10級就可以去影牙，然後一直刷到25級去血色

該階段總共需要經驗值為：65200

每趟經驗值：5000-8000，均值為6500

每躺花費金幣：3G

本階段需要刷副本次數：約10次

總計花費金幣：3*10=30G

如果勤快地拾取可以回本：約0.5*10=5G

第二階段：16-29級

副本：監獄

分析：

  • 20級以下的寶寶偶爾有人免費帶
  • 監獄可以刷到29級，再往上就可以考慮去血色
  • 監獄副本部落方對應的為影牙，刷到25去血色

該階段總共需要經驗值為：361900

每趟經驗值：5000-8000，均值為6500

每趟花費金幣：3G

本階段需要刷副本次數：約55次

總計花費金幣：3*55=165G

如果勤快地拾取可以回本：約1*10=10G

第三階段：29-41級

副本：血色修道院

分析：

  • 30級以下的經驗寶寶偶爾有人免費帶。
  • 只刷血色教堂的話，經驗值要看帶人的法師，不同的人帶的效率相差巨大：帶的好的話8000-12000，帶的不好可能就只有6000-9000。
  • 帶兩門的很多為了打快，可能兩門加起來才10000-14000經驗，主要原因是怪沒清乾淨。
  • 組的時候看清楚是不是全清，刷完看下經驗看看效率，實在太差就建議換人。

該階段總共需要經驗值為：790400

每趟經驗值：7000-12000，均值為9000

每趟花費金幣：5G

本階段需要刷副本次數：約88次

總計花費金幣：5*88=440G

如果勤快地拾取可以回本：約2*55=110G

第四階段：41-52級

副本：祖爾法拉克/瑪拉頓

分析：該階段有兩個副本可以供選擇，但是由於目前瑪拉頓的效率要高於祖爾法拉克，所以很多人都選擇瑪拉頓來升級。

該階段總共需要經驗值為：1375300

每趟副本經驗值與金幣開銷：

  • 祖爾 墓地加蟲子 6000-8000，5G
  • 祖爾 墓地加蟲子+前門兩波 10000-12000，8G
  • 祖爾 墓地+百人+前門兩波 12000-16000，10G
  • 瑪拉頓紫門 10000-14000，6-8G
  • 瑪拉頓紫門+瀑布 14000-18000，10G

下面的數據以瑪拉頓紫門+瀑布來計算：

本階段需要刷副本次數：約86次

總計花費金幣：10*86=860G

如果勤快地拾取可以回本：約4*86=344G

第五階段：52-60級

副本：斯坦索姆

分析：這個階段找人帶斯坦索姆需要花費巨大的金幣，但是幾乎沒有其他途徑，而且斯坦索姆很多時候都是不給自由拾取的。至於那些去厄運買莊嚴護腕之類的看自己需求選擇……

該階段總共需要經驗值為：1464300

斯坦索姆前後門每趟經驗值：18000-22000，均值為19000

每趟花費金幣：20G

本階段需要刷副本次數：約77次

總計花費金幣：20*77=1540G

如果可以拾取回本：約3*77=231G

總結

按照上一段的分析，我們從10級開始一直躺副本到60級的總共開銷約為：

30+165+440+860+1540=3035G

如果可以拾取回本的話，總開銷約為：

3035-5-10-110-344-231=2335G

而如果你只需要帶到52級之後不去斯坦索姆的話，那麼只需要1500G，勤快拾取的話大概只要1000G。

PS：由於每個伺服器和每人遇到的實際情況不盡相同，所以以上數據僅供參考。
