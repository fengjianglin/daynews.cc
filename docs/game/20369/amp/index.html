今天給你們講一下我最近剛研究的一種特別厲害的陣容，這個陣容非常厲害，成型基本上沒有天敵。這個以劍士為主，八人口就成型了。

二秘書所有友軍增加40的魔抗，這個非常克法海三光和四劍士非常像，死去都會加15%的攻速。而且四劍士攻速也特別快，所以我們給到輪子媽一個繳械沉默刀還有颶風。讓他直接罰站。

我們一定要把狗頭升到3星，不然這個陣容沒有坦度，所以說我們跟他兩個狂徒給他一個大帽讓他不死。

這個陣容剛開始的話會一直吃，連敗到最後八人口的時候就基本上鎖血沒有天敵。如果如果可以升級還可以上九人口。在拉一個沙漠的人沙皇或者螳螂。

如果我們有法裝的話給到劍魔，我們有眼淚或者其他東西的話，我們可以給到風女，我們選秀的時候先拿，魔抗還有反曲之弓把我們的鎖子甲也給拿上，這基本上沒有什麼人拿，所以我們裝備這方面就不用考慮了。以下是我裝備的圖。

我們一定要讓風女挨著輪子媽。這樣的話有刺客我們也不怕，有刺客來的話，我們有法裝在風雨身上，風女直接把他們吹走，所以說，輪子媽輸出環境是非常好的。

如果劍聖允許的話，我們要給劍聖一個火炮。在第二排，所以說這個是非常有坦度的，而且有傷害。以下是站位。

這是我研究一早上的套路，新賽季你剛出來，所有人都搶著盧錫安和他老婆，所以我們這個非常克他們，而且目前還沒有太多人知道。在高分段，基本上可以啃著雞屁股。

刀光劍影就是這麼玩的，就是玩的快而且是秒殺。你總能看見輪子媽在場上清場，讓強迫症心裡很爽。

我們前期可以用森林德魯伊來過渡，後來我們在上三光還有我們的建設，如果劍聖一直不來的話，我們可以搶鏟子去弄一個轉職刀。

我經常聽到我的朋友們說天天被法海克、被劍刺克、被掠食者克，但是這個我試了好幾天，基本上沒有天敵，這個是真的厲害。

秘書可以克法海。這個陣容是有風險的，但是成型以後是真的厲害，你們趕快去試試吧，我在這裡等著你們的好消息，加油！
