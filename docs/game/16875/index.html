部落衝突：現在可以輕鬆詳解，以前卻無人問津的天女完結篇是什麼

2019-12-16 2,914條評論 4,021次閱讀 2991人點贊

《部落衝突》中的天女是女王搭配天使的簡稱。

然而，天女的細節更多注重在滑步這個操作上，一個不錯的清邊滑步，更是COC思路的重中之重。

因此，本期的天女完結篇，將為首領詳細解析天女滑步的技巧、知識。不管你是萌新還是老首領，希望在這裡面，都可以讓你學習到實用的知識。

天女的滑步。

對於剛抵達9本的新手玩家，我們建議還是從簡單的滑步開始。選擇陣型外，只有一層建築的地方天女滑步，是一個非常好的練習。

一般女王面向的方位，便是天女即將滑步的一側，如果女王開始在單層建築間滑步，背後的天使先放下，再放下女王，可以避免後期天使貼牆飛行的情況。因為天使沒有目標之前是靜止的，讓天使與防空建築的位置相對的垂直，等天使跟著女王滑步的時候，可以讓天使與防禦塔之間拉開一個安全又垂直的距離。

雙層結構的滑步。

而對於一些需要天女操作進階的首領來說，陣型外的雙層建築乃至三層建築結構，則需要我們去深刻的熟悉與了解。

比如例圖中的雙層建築結構，如果想讓女王按照自己思路中的路線清邊、滑步，則需要外援的幫助。故此龍寶寶、亡靈、法師等這些兵種，在女王面對雙層建築滑步下，便顯得尤為重要。

比方說當龍寶寶拆除第一層某個建築的干擾後（阻止女王往這一側走），女王先後鎖定的順序便變得簡單又清晰：採集器訓練營加農炮/聖水罐/小木屋左側採集器。那麼，此刻女王的落腳位置，便應該稍微靠左側放下，便可以順利向左走動，不會走偏。

女王的清邊收尾。

當然，女王的滑步也有一個收尾的工作，借用同一個圖例。在女王拆毀加農炮後，及時在收尾的位置放下蠻王或者法師斷邊，讓女王開始往陣型內走去。這裡我們建議使用蠻王幫忙斷邊收尾，一來是因為蠻王的HP多，可以幫附近的女王抗傷害，並且接過女王往下的斷邊工作；二來是保證收尾建築必定拆除，若是附近有防禦塔，像法師收尾，可能就會折損好幾個，才可以完成斷邊。

收尾的細節。

我們再說下女王滑步收尾前的細節。女王想結束清邊之前，蠻王的落腳點是女王走向路徑上，未鎖定目標的後一個建築。比如圖例，如果你在女王鎖定的建築（1號金水罐）附近放蠻王，蠻王會和女王一起拆這個建築，但是女王依舊會往陣型外繼續清邊下去，而不會轉向陣型內。

到了這裡，天女滑步的篇章便結束了。
