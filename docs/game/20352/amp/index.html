懷舊服奧山LM一場聲望崇拜？BL只能橋頭跳舞：我們被坑殺了33小時

天天要聞
2019-12-16

懷舊服開放戰場不到一周，雖然時間不長，但雙方陣營的玩家們鬥智斗勇十分精彩，幾乎每當你一覺醒來，都會發現奧山的戰術已經發生改變，從第一天的對沖，到第二天的平推，現在流行起了大橋對峙。

大橋對峙迅速成為現在奧山的主流戰術，主要是由於這個打法對聯盟非常有利。因為大橋的地形易守難攻，60級奧山戰場沒有援兵系統，聯盟玩家可以完全放棄大橋外面的資源，把戰場完全變成玩家之間精力的對決，依靠守大橋的地理優勢擺鐵桶陣防守。

在雙方都有一定指揮和配合的前提下，作為進攻方的部落通常陣亡率會更高，而且陣亡的屍體全都在大橋里，聯盟可以扣完屍體轉身就交給NPC漲聲望。如果部落選擇掛機對峙，聯盟甚至可以去礦洞採集漲聲望。顯然對於聯盟玩家來說這是非常舒服的，有一場奧山打了33小時，從對峙熬到無限坑殺，據幾個一直堅持下來的聯盟表示，這一場幾乎直接聲望拉滿，沒滿的也差不多崇拜了。

而部落遠離基地，掛機對峙是玩家精力的純損耗，為了讓聯盟能出來打架，部落已經無奈到去兩軍陣前跳舞了。毫無疑問，現在這個大橋對峙的奧山環境對部落玩家來說是一種折磨，據有的玩家表示，自己周末上遊戲排隊5小時，上來了排戰場排了半小時，結果進來了還得在大橋邊上掛機……

這幾天里聯盟與部落在奧山的戰術花樣百出，螺旋上升，大橋對峙雖然讓部落玩家非常痛苦，但說不定一夜過後就有破解聯盟的大橋鐵桶陣的戰術出現了，小夥伴們對此怎麼看呢？歡迎在下方留言。
