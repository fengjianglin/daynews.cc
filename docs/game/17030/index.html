模擬戰新增2個系統，天賦重鑄很實用，拋棄無用天賦

2019-12-14 1,004條評論 3,590次閱讀 746人點贊

文|天雲解說原創，盜載必究

[摘要]模擬戰現在在正式服之中可以說是一個不輸於王者峽谷的模式了，因為其不需要太高的手速，更多的是運氣和腦力的對戰，因此備受玩家喜愛，本期筆者就來說一下這個的改動吧，下面一起來看看！

首先第一個是增加了萬物迴轉系統，當遊戲進行到第5，15，25，35回合的時候，場上剩餘的玩家將會傳送到莊周的花園當中，如同迴轉壽司一般，莊周會召喚出英雄，天賦，裝備，金幣等等道具供大家選擇，而座位排序是由血量的高低進行的，低的優先選擇，而是否能選中心儀的道具，就要看各位召喚師的戰術了！

第二個則是天賦重鑄系統了，當玩家的血量低於50點或者是25點的時候，將會觸發這個系統，獲取一次天賦重鑄的機會，將無用的天賦更換成戰鬥天賦，就能夠繼續撐下去了！

從上圖可以看出，若是你的血量低於50點或者25點的時候，選擇天賦重鑄，像這個天賦異稟肯定是無用的了，直接重鑄好么，而且不穩定的收益，金幣是已經獲得了，如果選擇重鑄的話，是否會扣除自己的金幣呢？這是個問題呀！

最後一個則是新增加了陣營令牌了，相信知道的玩家都是很熟悉，這個就是和法師，戰士令旗是一樣的，但是只能進入到相對應的陣營當中，像群雄令，只有一個貂蟬，有了這個令牌後也能加入到別的陣營當中了，真的是非常不錯的！對此，大家有何看法？歡迎討論！
