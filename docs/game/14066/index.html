一騎當千 傲視群雄！盤點十部Steam上的三國志遊戲

2019-12-11 1,599條評論 2,963次閱讀 242人點贊

「三國志」遊戲可謂在國內外都有著無可撼動的人氣，遊戲故事的趣味性和充滿個性的武將們的表現是其魅力所在。今天小編就通過「策略系」、「經典名作系」、「動作系」和「RPG/養成系」這四種類型來為大家來盤點一下可以在Steam上遊玩的有趣的三國志遊戲，一起來看看吧！

策略系

《全面戰爭：三國》

Steam價格：268元 支持簡體中文

本作實現了玩家「在全戰系統中玩三國志」的夢想。遊戲地圖是回合制，採用了《全面戰爭》系列的傳統遊戲系統，軍隊之間展開即時戰鬥相互碰撞。可以在敵人陣地所向披靡，也可以和敵將單打獨鬥，光是看著就可以享受到樂趣。戰鬥無論多少次都可以重新來過，也可以保存回放，為了打造出最精彩的一戰而往複戰鬥。


《三國志：漢末霸業》

Steam價格：68元 支持簡體中文

本作系統借鑒了光榮《三國志》系列（X以前）的優點，可以盡情地享受回合制內政和戰鬥，遊戲內的武將遵循史實，深度刻畫了三國各城市之間地域的差異以及每個武將自身能力和特性的異同，戰鬥方面突出結合天時地利人和的調配來進行策略作戰的特色。雖然本作目前還在搶先體驗平台，但與初上線時相比在視覺方面已經有了很大的進步，期待結束搶先體驗後的正式發售。


《中華三國志》

Steam價格：免費 支持簡體中文

本作是由國內民間團隊開發研製的一款免費、代碼開源的單機策略三國遊戲，稱為「中國人自己的三國志」，遊戲採用貼近史實內容的回合制策略，地圖上的據點也是根據史實設定。本作從2009年開始開發，默默發展近十年，乍一看畫面比較簡陋樸素，但可以使用皮膚來改變地圖畫面，或者更換武將面容和能力等。可以通過細緻的定製來打造只屬於自己的三國志。


經典名作系

《三國志9PK》

Steam價格：88元 支持繁體中文

光榮《三國志》系列首次在一張地圖展開戰鬥。軍隊編成後，決定了進攻的據點就會自動前往，並會自動使用單槍匹馬和計謀進行戰鬥。本作的隱藏要素尤其之多，內政不會很繁瑣，適合反覆遊玩。一邊說著台詞一邊出動的軍隊也讓人沉浸其中。只是因為遊戲在2003年發售，所以無法對應高解析度和窗口模式，期待今後能夠有所對應。


《三國志11PK》

Steam價格：116元 支持繁體中文

和《三國志9PK》一樣都是玩家喜愛的《三國志》系列作品，可以用桌游的感覺來體驗內政以及和敵人軍隊戰鬥，還可以預測敵人的前進方向，設下陷阱一網打盡。只是由於多個軍隊移動需要時間，所以可能不太適合像《三國志9PK》那樣反覆遊玩。本作對應高解析度和窗口模式，現在遊玩也沒有問題。


動作系

《真三國無雙7 with 猛將傳》

Steam價格：159元 支持簡體中文

該系列已經出第八部了，不過這次要介紹的是第七部的猛將傳。本作基本上是和7以前的無雙系列包含內容相同，不過可以使用的角色和舞台數量，根據條件分支而出現的IF故事等隱藏要素，還有建立自己據點的將星模式等，可以遊玩的內容非常的充實豐富，是其魅力所在。如果是沒有玩過無雙系列的玩家推薦先從本作開始遊玩。


《真・三國無雙7 Empires》

Steam價格：310元 支持繁體中文

這是一部將召集同伴、內政、攻擊敵人領土等戰略要素與無雙系列戰鬥相結合的作品。可以製作原創武將和原創勢力，只要有能力就可以構築完全不會出現三國志勢力的原創劇本，對於想打造架空戰記的人來說是再合適不過的遊戲了。可玩性很高，能下載其他玩家打造的劇本遊玩也很不錯。


《戀姫†演武 ～遼來來～》

Steam價格：70元 支持簡體中文

原作中的大叔們全部都化身美少女的「戀姬無雙」系列，以此為基礎的對戰格鬥遊戲就是《戀姫†演武 ～遼來來～》，與最近高速展開和長時間連擊為主的對戰格鬥遊戲不同，本作是側重策略的風格，取消了軍師所屬國限制，正統派，毫不陌生的遊戲操作加上一目了然的戰鬥體系，即使是初次接觸的玩家也很容易上手，開始格鬥遊戲所特有的引人入勝的戰略比拼。


RPG/養成系

《吞食孔明傳》

Steam價格：55元 支持簡體中文

本作是《吞食天地》的粉絲耗費10多年開發出來的遊戲，在地圖畫面上沿襲了與敵人戰鬥等類似《吞食天地》的系統，並且包含了眾多現代RPG要素。八個章節的正統劇本，兩百位武將登場，可加入玩家陣營突破五十名，二周目開啟，百種天賦技能及成就待解鎖。


《呂布夢工廠》

Steam價格：32元 支持英語/日語/韓語

相信大家都聽說過一個女兒養成的遊戲《美少女夢工廠》，本作可以說是其三國志版（？）。玩家將成為董卓來養成「女兒」化的呂布。安排行程，讓女兒學習武術和知識，和女兒交流，參加大會等，基本系統和《美少女夢工廠》是一樣的。不過結局可能會像史實一樣被殺，所以為了不讓女兒最終被殺，玩家要好好地養育她。
