SMG戰隊新陣容漸入佳境，新選手實力帶飛，那個17回來了

2019-12-07 0條評論 19次閱讀 0人點贊

摘要

絕地求生微博杯比賽的小組賽階段已經落下帷幕了，Ark和RNG的巔峰對決讓我們看到了兩支新豪強的崛起，同時4AM和SMG兩支隊伍的低開高走也讓觀眾們拍案叫絕，尤其是SMG，對於低迷了大半年的他們來說，這兩天的表現可以說是漸入佳境，慢慢的好起來了！

絕地求生微博杯比賽的小組賽階段已經落下帷幕了，Ark和RNG的巔峰對決讓我們看到了兩支新豪強的崛起，同時4AM和SMG兩支隊伍的低開高走也讓觀眾們拍案叫絕，尤其是SMG，對於低迷了大半年的他們來說，這兩天的表現可以說是漸入佳境，慢慢的好起來了！

在第一天的比賽中，SMG戰隊的表現非常不好，基本就是見面就跪，最終6局比賽僅僅只拿了14分，第一輪小組賽後位於所有隊伍倒數第一。

在所有人都對他們已經不抱希望時，第二天的小組賽中，SMG戰隊的新人smallcc站出來了，單人披荊斬棘，帶著隊伍一路前行，決賽圈單人6殺還擠進了前二，一局就為隊伍拿下12分，在第二天的比賽中，SMG拿到42分。

第三天的一開局，SMG的表現還是不是很好，也是正面接團有很大的問題，就在又有不少人放棄希望時，SMG連續三局來了強勢的天命圈，在第17局的比賽中，SMG憑藉著大E城外的麥垛天命，吃到了一雞，不少觀眾都調侃：17玄學局啊，第17局吃雞，唯一美中不足的就是拿了18分！最終SMG在第三天又拿到了46分，成功打進了最後的決賽！

三天的得分分別是：14分、42分、46分，這個得分趨勢還是步步高升，可以看出，現在SMG戰隊的陣容也是磨合得越來越好了，他們的狀態也很不錯，當然最關鍵的還是選手的個人能力很強，從選手數據來看，從大鵝引進的smallcc是隊內表現最好的，有了這樣一員猛將，SMG的實力自然也是水漲船高。

不過在比賽結束後，SMG的鐵粉還是對SMG的正面有很大的擔憂，目前SMG戰隊正面接團的能力確實有點不足，比如光明頂，滿編守著微博，按照常理，微博這四分他們應該是必吃的，結果反被微博低打高最終被零換四，在薩諾擁有天命房區，結果被對手零換三，即便是17局吃雞的那一局，占著優勢點已經被打掉一個，但凡對手還有點葯，這個結局都很懸。

所以SMG戰隊的正面能力還是需要加強，正如MsJoy老師所說，圈運是守恆的，小組賽圈運這麼好，決賽就不一定了，作為一支強隊，也不能全靠圈運吃飯，還是要自己能夠打出來的！期待SMG戰隊後續的表現吧！
