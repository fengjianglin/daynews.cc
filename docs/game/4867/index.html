LPL全明星賽結束後，姿態發文暗示自己想要復出？網友：劍指S11？

哈嘍！大家好！我是你們最最喜愛的高高君，今年的LPL全明星賽在前兩天也是圓滿的落下了帷幕，在選手以及觀眾們的歡聲笑語中打出了GG。雖然說這次LPL全明星賽看的歡樂，但有些觀眾卻是有點不太滿意。因為本次全明星過於娛樂化很多觀眾所期待的「至臻之戰並沒有到來，所以很多觀眾感到不滿也是正常現象。甚至很多網友都調侃說道今年的傳奇對抗賽才是重頭戲，正賽什麼的都要靠邊站了。而在全明星結束後姿態在剛剛也是發了一條疑似想要復出的動態，一時間也是引起了很粉絲的熱議。

在前段時間結束的全明星賽中，相信大家最為期待的就是傳奇對戰以及正賽了。傳奇對戰在開賽前火藥味就非常重了，當正式看開戰後雙方都是把這場表演賽當成真正的職業比賽來打。在老WE戰隊先下一城後老IG也是不甘示弱在第二局中強勢碾壓了老WE把比分扳平。

所以這次的全明星傳奇對抗賽也是由雙方打了一個平手而告終，礦泉水瓶事件沒有上演確實還是有點可惜的。但是姿態在這兩場的發揮可以說是相當出色了，可以說兩把都是靠著姿態在C如果第一把不是因為他的夢魘後期沒有太好的發揮空間估計第一把也是由老IG拿下的。

之後到了第二把姿態來了一把卡薩丁教學，在前期拿下三個人頭後直接起飛爆殺全場。之後更是殺瘋了殺進泉水中去，可以說第二把完全就是姿態的個人秀了。在看到姿態這麼C後很多觀眾粉絲也是紛紛在彈幕刷到「在？為什麼退役」之類的話。

而剛剛姿態對於網友們的調侃也是正式回應一下，在自己的超話中發了一篇常溫哦並且好像還在暗示將要復出的樣子。他在文中也是大致說道：「全明星賽之後感覺還是提得動刀的，並且非常享受舞台比賽的感覺，明年就等著我報備吧」

這是在暗示自己將要復出嗎？不過以姿態的表現來看真要復出去打中單問題也是不大的。畢竟作為曾經的天才選手訓練一段時間狀態就回來了，並且現如今RNG戰隊也是面臨著非常嚴峻的問題。那就是買不到人，如果這個時候姿態選擇復出去救場也是個不錯的選擇！但是之後網友的評論也是笑出了聲，真就征戰S11了唄？

對於這件事你有什麼看法？
