PEL賽場日記：恭喜4AM拿下冠軍！PEC世界賽隊伍名單出爐

2019-12-16 984條評論 2,520次閱讀 818人點贊

海島老司機，不浪不會玩斯基。各位好，我是難言的小跟班——浪斯基

在12月15日下午五點，和平精英PEL職業聯賽決賽階段最後一天的五場比賽準時開打，作為國內和平精英首屆職業聯賽，PEL賽事以「不由界定，百戰成名」為主題，在經歷了預選賽，晉級賽的對決後，十五支脫穎而出的隊伍，他們經歷了兩周六天共三十場比賽的精彩對決，最終由4AM捧起了首屆PEL職業聯賽的冠軍獎盃！

在PEL職業聯賽決賽最後一天，4AM雖然有較大的積分優勢，但並不穩定，尤其是第三場XQF吃下蛋糕後，拉近了和4AM的積分差距，但最終第四場比賽，還是4AM穩到了最後，吃下蛋糕提前鎖定了PEL的冠軍獎盃！而隨著比賽的結束，4AM奪冠之後，還將和XQF，DKG一起參加PEC世界賽！那麼最後一天五場比賽戰況如何呢？讓我們一起回顧一下。

第一場：XQF前期遭重被淘汰，DKG穩到最後拿下勝利

第一場比賽是FPP模式的艾倫格地圖，第一個圈是一個南部陰陽圈，一半是機場一半是陸地，在前期剛落地的時候，也是發生了小型交火，AgFox先掉一人。時間來到12分鐘信號圈刷新，XQF在過橋的時候先被前方的LK擊倒兩人，4AM也是開車堵在後面，隨後XQF選擇游泳過橋，但被4AM在水中擊倒一人，最後一名獨狼也是在進圈的時候被淘汰。

時間來到23分鐘，混戰開啟大量選手被淘汰出局，XQF和FPX獨狼先被淘汰，隨後4AM選擇開車強衝進圈，反遭到多方集火被淘汰出局。在最後的決賽圈，ELG先被送出比賽，隨後Tianba也是再將JCR淘汰出局，但最終被穩到最後的DKG淘汰，由DKG拿下第一場的勝利。

第二場：AG一路橫推不斷拿分，ELG完美配合拿下勝利

第二場比賽是FPP模式的米拉瑪地圖，第一個圈是一個南部獅城圈，在前12分鐘大家都以發育為主，並沒有發生激烈的交火。時間來到14分鐘，第一波小混戰開啟，左側不斷跳出淘汰提示，一波混戰後WF先被淘汰，隨後RSG也被LK趕走，僅剩一名獨狼存活。

時間來到23分鐘，此時混戰開啟大量選手被淘汰出局，SDR和AG在房區打了起來，SDR先被送出比賽；同時FPX，XQF還有RNG三方在圈邊混戰，4AM也在遠點勸架，一波混戰後RNG先被淘汰，隨後FPX也被送出比賽。在最後的決賽圈，場上還剩下五支隊伍，RSG獨狼先被淘汰，隨後Tianba也被ELG送出比賽。最後成為了二打四的局面，人數佔優勢的ELG拉開槍線，輕鬆拿下了第二場的勝利。

第三場：4AM遭強沖被淘汰，XQF狀態回歸拿下勝利

第三場比賽是TPP模式的艾倫格地圖，第一個圈是一個北部靶場圈，大部分隊伍都在圈外，前期各隊伍都以轉移為主，在前22分鐘，都沒有發生激烈的交火。在信號圈刷新後，混戰爆發，SMG在圈邊先將ELG淘汰，隨後4AM也是遭到JCR的強沖，被送出了比賽。

時間來到25分鐘，此時混戰爆發，大量選手被淘汰出局，SMG和JCR先後被送出比賽，隨後AG也是被WF淘汰出局。在最後的決賽圈，DKG和WF的獨狼先被清理，在和RSG的對決中，XQF佔據了人數和地形的優勢，最終拿下了第三場比賽的勝利。

第四場：XQF進圈遭集火出局，4AM吃下蛋糕提前鎖定冠軍

第四場比賽是TPP模式的艾倫格地圖，第一個圈是一個東部圈，在前期轉移的時候，ELG先被SMG淘汰兩人。時間來到20分鐘，第一波混戰開啟，RSG先被XQF淘汰三人，獨狼在轉移的時候，也是被其他隊伍淘汰，成為了第一支出局的隊伍。

時間來到24分鐘，此時混戰開啟，大量選手被淘汰出局，AG位置較差被打靶出局，隨後LK和FPX也先後出局，DKG在開車進圈時，也被RNG掃車全員被淘汰。在最後的決賽圈，場上還剩下四支隊伍，此時4AM還是滿編狀態，在清理了ELG和RNG獨狼後，4AM憑藉人數和地形的優勢淘汰SDR，拿下了第四場的勝利！

第五場：終局之戰放飛自我，XQF穩定出擊再拿一勝

第五場比賽是TPP模式的米拉瑪地圖，第一個圈是一個獅城圈，大部分隊伍都在圈內，前12分鐘大家都已轉移為主，雖然發生了小交火，但並沒有隊伍被淘汰出局。時間來到18分鐘，ELG在轉移的時候遭重創，剩餘兩人選擇游泳上岸，但還是在水中被淘汰。

時間來到21分鐘，此時信號圈刷新，大量選手被淘汰，Tianba和RNG先後出局，隨後XQF在圈邊和SDR也打了起來，最終SDR被送出比賽。在最後的決賽圈，場上還剩下五支隊伍，RSG獨狼先被清理，JCR和AG也先後出局，最終XQF憑藉人數和地形優勢，拿下了最後一場比賽的勝利。

隨著XQF的壓軸獲勝，和平精英首屆PEL職業聯賽也落下帷幕，最終冠軍是由4AM拿下，同時他將和XQF還有DKG參加接下來的PEC世界賽，在這次PEL職業聯賽的決賽階段中，4AM的狀態一直都保持得很好，尤其是第一周第三天的比賽，單日狂追105分，也為他們決賽奪冠打下了基礎，隨後4AM就一直高居榜首，最終拿下了冠軍。那麼最後讓我們一起恭喜4AM拿下冠軍！
