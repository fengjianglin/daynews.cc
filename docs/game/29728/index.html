爐石傳說：戰棋輪空就一定穩了？國服第一主播慘被淘汰者擊敗

2019-12-24 402條評論 3,014次閱讀 973人點贊

說起爐石戰棋，不知道大家最不慌的時候是什麼呢？是玩某一個流派拿到了核心牌嗎？或者是自己排名第一打倒數第一？其實都不是，最讓玩家安心的局面應該就是打死人了。畢竟輪空基本是不可能輸的，都是死了那麼久的人，戰鬥力一定會相差很大。

但在高端局中，還真不能以此來判斷。就算是死人，也不一定有那麼安全。而虎牙爐石主播夜帆，就為大家詮釋了，什麼叫輪空都救不了你。

現在這個時代，是魚人的時代。雖說每一個流派只要運氣好，足夠天胡，都能夠恰到雞。但在同等戰鬥力的情況下，其實還是魚人流最強。且魚人的前期非常的強勢，雖然中期比較弱勢，但只要活到了後期，基本就是無敵啊！

所以這一局夜帆就選擇玩魚人流，不過後續的運氣這的不好，除了開頭拿到了一個斥候，之後給的隨從基本都沒什麼質量，無奈之下，夜帆只要選擇轉惡魔流。不過因為後續拿到了魚人流的二星核心牌，所以看起來還是有轉回機會。

但魚人非常的吃酒館的等級，而惡魔在沒有核心牌的情況下，也要求快速的升級酒館。所以連續兩局升到了三星，但很可惜搜尋了幾個回合都沒有拿到惡魔流的三星核心牌。在拿不到任何核心牌的情況下，主播很慘的流落到了倒數第一。

或許很多人會說是主播水平不行，但要知道這個主播可是當年的國服第一，當時這一局時也快衝上一萬分了。對遊戲的理解，對運營的能力都是不會差的。加上這一局使用的英雄還是銅須，本就是非常強勢的英雄，卻還是打成了倒數第一，大家可想而知這一局有多倒霉。

不過好在中期的時候運氣不錯，拿到了機械流的核心牌。所以順利的衝到了前三，但即便如此局面還是非常的困難。在進入前三之後，便被排名第一的冰龍狠狠的揍了十二血。因為陣容差距還比較大，所以夜帆求了求系統，想要一輪空過。

不知道是運氣問題還是機制問題，這一輪還真的輪空了。在來不及高興的時候，居然被排名第四的死人給揍死了。其實從戰鬥力來說，兩者差距不大，但實在是流派比較克制，對面的惡魔流身材太高，所以機械流不敵。

事實證明，就算是空過也不能放鬆警惕，如果遇到比較克制的流派，還真不一定能贏。
