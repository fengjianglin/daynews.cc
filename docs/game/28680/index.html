夢三國2：玩法多樣化，很多玩家酷愛的馬超來襲

2019-12-22 3,531條評論 4,816次閱讀 718人點贊

昨天的文章中，舞文給大家留下的問題中提到，今天將為大家帶來的英雄，就歷史上來說很有可能是姜維的殺父仇人。這個人其實就是馬超馬孟起，雖然歷史上的他戰功顯赫，遊戲中也異常強大，但同時也是整個三國時期最強的坑爹達人。在介紹大多數玩家喜歡夢三國大陸上的馬超之前，我們先來解決昨天留下的問題。

在《三國志》等史書中，雖然都沒有介紹姜維父親的生平，只是簡單提到了姜維幼年喪父與母親相依為命。但是，年僅24就能獲得中郎將職位（算是地方高官了），且幼年喪父還能接受良好的教育等多方面因素來看，姜維應該算是漢朝根紅苗正的烈士家屬。而在姜維15歲之前（三國時期15歲算成年）導致姜維喪父還在天水附近的大型病變當屬公元212年馬超席捲隴右無疑。以此來看，說馬超是姜維的殺父仇人必然沒有問題存在。好了，史書什麼的我們簡單說下就好了，接下來按照以往的慣例舞文還是和大家一起去看看那些年玩家愛上馬超這個英雄的原因吧。

五虎上將誰不愛？

作為三國時期最知名的蜀漢五虎上將之一，馬超在來到夢三國大陸上之後會被玩家喜愛也在情理之中。實際上，不僅是在《夢三國》，在近乎所有的三國題材遊戲中只要有了馬超登場必然會成為很多玩家的大愛。

個人戰力不俗

無論是從歷史、小說還是從遊戲的角度上來看，馬超的戰鬥力都極為強大。對於所有喜歡玩遊戲的玩家而言，在選擇人物的時候沒接觸遊戲之前自然會從歷史、小說和影視作品中的印象出發。作為其他作品中戰力強大，在遊戲中表現也不差的存在，玩家會慢慢愛上馬超這個英雄也在情理之中。

顏值非常之高

有關注三國歷史、小說等文學作品的朋友，對錦馬超這個稱號相信都不會感到陌生。而這些作品中之所以在馬超的名字前面加上一個錦字，最根本上的原因還是為了形容馬超的英姿勃發（簡單點來說就是為了形容他的帥）。而在遊戲中，無論是基礎換裝錦公子、好感玉換裝五虎之威，還是最新推出的九天雷尊都將馬超的帥氣體現的淋漓盡致。

操作難度低

作為一個物理輸出英雄，只要站著就有輸出且技能又以被動為主的存在，馬超的操作難度還是很低的。這種低操作難度之下，自然也就使得了很多玩家在萌新時代接觸到這個英雄之後就有一種愛不釋手的感覺。（舞文絕不會告訴大家，當初之所以會愛上馬超是因為個人補刀功底差，馬超有大招之後打錢速度飛快。）

玩法多樣化

雖然從玩法體系上來看，目前的馬超其實也就只有肉裝、輸出裝和半肉半輸出三種玩法體系存在。但是，無論哪一種玩法體系之下，馬超都擁有不錯的生存和傷害輸出能力，這也直接讓很多玩家深深愛上了這個英雄。尤其是在肉裝玩法剛剛開發出來的那段時間中，更是娛樂比賽常客和路人超級霸主。

現狀分析

娛樂：馬超在如今的娛樂模式中，無論登場率還是生存都是非常不錯的。目前的娛樂對局中，喜歡純肉和純輸出馬超的玩家都非常多。

關卡：由於大招擁有不錯的範圍傷害能力的原因，目前的關卡地圖中馬超的登場率也還算不錯，只是因為沒有位移的原因登場率不算特別高。

戰場：馬超在戰場地圖中的登場率還是非常不錯，尤其是在出征地圖中強大到有些讓人看不懂的大招傷害更是大多數玩家深愛這個英雄的主要原因所在。如果不是坐騎和單手武器的限制，馬超在出征地圖中的登場率必然會有進一步提升，不過很明顯這種限制無法改變。

下期預告

今天的坑爹達人馬超介紹到此結束，結束就是新的開始。因為連續兩期介紹了蜀國英雄，第一期又是介紹的吳國周瑜的原因，所以下一期舞文準備為大家介紹一位中立英雄（那誰，別扔雞蛋）。沒錯，下次要介紹的絕對不可能是呂布，因為舞文準備為大家介紹下趙雲曾經的老大。
