外網討論：制裁末日鐵拳的英雄去哪了？

2019-12-19 340條評論 2,034次閱讀 375人點贊

鐵拳作為國服玩家的公敵，每次關於鐵拳內容的評論區是最精彩的部分，吐槽調侃等段子能讓人樂呵樂呵。但從平衡調整的角度來說，刪除是不可能刪除的，雙盾體系未形成前，從聯賽和外服的情況來看，鐵拳較為平衡，能夠大殺特殺，也會高風險直接白給，這符合他的秒殺定位。一些經歷過外服環境的玩家回到國服後都有這樣的困惑：為什麼外服的鐵拳沒有國服猛？這就涉及到國服環境的問題，在之前的分析里也提到了並不是所謂的玩家不努力，而是在團隊配合上存在差異從而影響到遊戲強度。像鐵拳源氏這種英雄，團隊配合上存在過多漏洞就是他們發揮空間的保證。

雙盾形成後，鐵拳取代了長槍帝國成為外服第一DPS，不過面對鐵拳橫行的情況，外服的討論更關注於雙盾，本篇內容也是翻譯於用戶ButterofBalls的reddit短文，他認為現在最大的問題是為什麼沒有玩家去使用能夠制裁鐵拳的英雄。搬運此文的目的也是希望各位盒友能夠從多方面去思考鐵拳的問題，平衡性改動、222機制甚至整個遊戲環境的變化等等。

以下是原文翻譯（有刪改）：

無論你如何看待鐵拳，他現在出場率如此之高的原因在於他是遊戲里唯一無視護盾的強力DPS，他的上勾拳和連招並不會受到護盾過多影響。其他的DPS位置只能面盾思過，帝國、雙狗都被雙盾所限制。除了鐵拳之外，更好的選擇則是死神和美，但這並不意味著死神和美能夠強力限制雙盾，作為使用者，雙盾會更讓他們頭疼，美如果越過護盾去凍結別人，就不得不離開隊友的保護與治療，而死神為了造成有效傷害，在西格瑪的靈活護盾前，死神必須不斷越盾。與雙盾對抗，他們只是鐵拳以外最不糟糕的選擇。所以簡而言之，玩死神和美抗衡雙盾仍然是一件苦差事，只不過這種苦差事比其他DPS位簡單了點，而能有效限制死神和美的麥克雷在雙盾面前毫無發揮空間。

作為一名在大師分段的DPS玩家，我受環境影響，被迫使用鐵拳，沒有其他的DPS位置能夠達到他的效果。只要避開西格瑪的隕石，鐵拳完全吊打西格瑪。與奧麗莎的對決，鐵拳可以輕易地迫使奧麗莎交出霸體，之後利用自身技能冷卻時間較短，打出連招將其擊殺。但是使用鐵拳並不像人們想像中的那樣簡單，而且很明顯有這樣觀點的人他們沒有在高強度對抗中使用鐵拳。

再看看其他能限制鐵拳的英雄，路霸曾經是最有效的重裝英雄，如果再有些小加強的話，他可以重新勝任這個角色。黑影和死神也是不錯的選擇，鐵拳的擊殺能力體現在連招上，如果敵方有足夠多的的控制技能，他就會顯得較為被動，進攻性也會更低。一個優秀的麥克雷玩家足以限制住鐵拳的進攻性。

我非常贊同鐵拳的秒殺能力讓人苦惱這樣的觀點，也許在以後的平衡性調整中可以將其修改並用其他的方向的改動彌補。在論壇中，大多數用戶最喜歡的英雄無疑是半藏和百合，在雙盾登場之前，帝國組合是最強勢的DPS位置。鐵拳也能發揮作用，但是無法有效限制百合，他們能在幾秒內進行多次秒殺，比鐵拳更加咄咄逼人。

雙盾環境下，鐵拳是為數不多可用的DPS，這是這名用戶的觀點。也正如他所說，外服在雙盾之前，主流DPS位置是帝國，DPS位置上並沒有進行削弱加強，而真正變化的是整個遊戲環境，這才是鐵拳在外服甚至比賽高出場率的原因。

在後續的討論中，不少用戶也贊同這樣的看法，其它DPS位置在大環境下過於乏力。末日鐵拳的強勢確實是版本的產物，他可以繞過目前遊戲最重要的一面，雙盾，雙盾阻礙了大多數DPS英雄的輸出。在此之前，百合、閃光等英雄能有效對抗末日鐵拳，可問題在於版本，列舉的DPS英雄處境糟糕，鐵拳成為了一個病人所展現的癥狀，病因在於雙盾。現在單純削弱鐵拳過於困難，而整個環境大變才會讓他被玩家所放棄。這篇文章指出的問題是玩家沒有充分考慮對於英雄有利發揮或不利的版本環境。問一個英雄究竟是強大還是脆弱的，不僅僅基於他們的設計或是平衡調整。以303時代的黑影為例。黑影的登場率和她的影響是通過多個階段才提高的。一些玩家喜歡將黑影稱為設計失敗、缺少遊戲樂趣的英雄。而事實上，黑影是非常適合303版本。303陣容一定是抱團並依靠技能開團，缺乏控制能力和直接秒殺能力，黑影的大招來得異常迅速能夠取得良好的效果。但是一旦脫離303的環境，黑影發揮的作用就要小得多。

如果玩家希望這款遊戲在平衡方面做得更好，他們需要進一步思考以下幾點。

當前版本的實際問題。

某個英雄存在的目的是什麼？

當前陣容的風格是什麼？

這個英雄為什麼能適合這種遊戲風格？

這個英雄的表現是否超出他們的預期？

此英雄是否對他們產生強大影響，如果是，為什麼？

一些玩家直言之前曾經是長短槍玩家，但是環境的變化讓他們此賽季不得不操刀鐵拳，不用堡壘的情況下破盾的效率低下，與其破盾不如用鐵拳去打雙盾陣容，越盾真香。如果敵方頭硬選取被雙盾克制的英雄，比如黑影麥克雷等等，鐵拳完全可以改變打法用右鍵壓制前排的血線，滿蓄力穿盾撞牆的250傷害是現版本槍位做不到的。打不過就加入，最終就變成了兩邊鐵拳飛來飛去的遊戲。

最後用一句話總結：鐵拳能打雙盾，但是能打鐵拳的都被雙盾按在板凳上。
