經典街機《合金彈頭》食人花進化的終端，人類不過是一道菜

2019-12-25 1,883條評論 3,353次閱讀 904人點贊

說到這個話題，不由得讓小編想起了《魂斗羅》中那些連續進化的怪物。魂斗羅整個系列都是圍繞著「天王鬼」和「天王龍神鬼」，以及天王鬼的妹妹「黑蛇」而展開的。期間它們孕育和分裂出了大量詭異的生物，並且隨著對地球環境的適應，這些怪物還在不斷的進化。一些前期看起來比較渺小的蟲子，隨著版本的增加不斷改變形態，甚至在後期成為了終極BOSS。

像是《超級魂斗羅》中的陰獸，在FC版本中就已經是最終BOSS，出場就能帶給玩家非常震撼的感覺。在通關之後，我們認為這個BOSS已經被徹底消滅了，不會再出現。但是到了《魂斗羅3》時期，玩家們才發現之前消滅的陰獸捲土重來，而且之前消滅的僅僅是其中一隻。在地底深處的巢穴中，還孕育著成千上萬隻未成形的陰獸，而此時的陰獸已經擁有了一定的進化能力。

陰獸有很多種進化方向，取決於他們吞噬的對象。最初的陰獸擁有人類女性的臉，就是吞食了人類。在《真魂斗羅》和《新魂斗羅》中陰獸成功的進化出了觸角，同時擁有了快速移動的能力，之後進入地底沉眠。當玩家再次遇到它時，已經進化出了翅膀。擁有了持續飛行的能力，而且仍然保留了部分人類的外型。直達人心的恐懼並不是怪物長得有多麼猙獰，而是這些怪物進化出了人類的特徵。

直到《3D魂斗羅》時期，陰獸終於進化完成，擁有了類似蝴蝶的翅膀和人類的智慧，這是陰獸最完美的形態，擁有史上最強的破壞能力。

合金彈頭系列中同樣出現過大量可以進化的怪物，不過估計很多玩家都沒有注意到吧！

在《合金彈頭3》的第四關中，玩家們首次來到了叛軍的秘密實驗基地。在這裡發現了大量的恐怖生物，其中就有食人花和火星蛆蟲。而培育這些生物的幕後首腦就是火星人假扮的摩登，它的目的在於通過培育火星生物慢慢侵蝕地球，並將地球打造成為適宜火星居住的生活環境。根據官方介紹，食人花是叛軍將火星病毒注入到某種植物後，基因突變的產物。然而令叛軍沒有預料到的是，食人花擁有驚人的進化速度和繁衍能力。不過，好在它們目前還不能適應實驗室以外的環境。

在之後的實驗中，食人花紮根將實驗基地附近的水土資源吸收殆盡，周圍的環境變得毫無生機。而在這期間食人花漸漸進化出了吞噬能力和咬合能力，甚至可以直接將人類吞食。大量的正規軍俘虜，成為了它們的口糧。隨著食人花對地球環境的適應，它們可以移動的範圍越來越大。食人花在吸收足夠的養分之後，就會紮根進化並散播種子，不過生命也走到了盡頭。

科研人員將種子收集，進行二次培育，意欲培育可以完美適應地球環境的食人花。

在第四關場景中，玩家會發現大量的食人花種子正在向全世界散播。而這種種子只要落地就會馬上生長，並迅速進化為食人花。要是動物觸碰到這些種子，身體馬上就會被吸收變成植物。不過，好在目前食人花種子只能在實驗室及附近一帶才能生長，散播到其他地方的種子無法繁衍。

玩家來到實驗基地門口時，發現這裡已經完全失控了。食人花佔領了整個實驗室和山丘，駐守於此的叛軍無法抗衡數量龐大的食人花，部隊幾乎被吞噬殆盡。玩家在門口可以遇到幾個倖存的叛軍，慌不擇路的逃跑。

整個部隊被屠殺，天知道他們到底經歷了什麼！

正規軍進入實驗基地後，局勢終於得到了控制。

然而，食人花的進化並沒有終止。

實驗基地的部分科研人員，在逃走之前帶走了培育數據和進化食人花種子。而這些種子在《合金彈頭7》時期成功的培育出了「食人花母體」。

「食人花母體」身軀極為龐大，不過好在沒有移動能力。然而卻已經進化出更加可怕的觸手，可以抓住距離很遠的生物。「食人花母體」再繁衍出來的食人花，已經擁有了完美的適應能力，可以在地球的任何地方出現。

在《合金彈頭7》中，這些食人花母體出現的場景比較多。它們有幾種攻擊方式，而最恐怖的則是被觸手抓住生吞。食人花母體在吞噬人類之前還會先剝乾淨衣物。

然而即使擁有如此能力的食人花母體，仍然不是最完美的。因為它們並沒有任何智慧，完全是出於本能的攻擊。

食人花還有機會進化嗎？進化的終端又是什麼？

食人花進化的終端，也就是食人花皇后：Bloom Metzelei

Bloom Metzelei並非是《合金彈頭》正統遊戲中出現的生物，而是來自《Metal Slug Attack》。也就是《合金彈頭》的官方手游。食人花母體在吞噬了大量的人類之後，竟然進化出了人類的特徵。並且通過上半身誘惑人類靠近。它的花粉對於地球生物來說是致命的，攻擊的範圍大大增加。

此時的食人花不僅僅擁有了移動能力，而且進化出了發達的大腦擁有接近人類的智慧。成為食人花最完美的形態

官方的人設圖中我們可以看到，Bloom Metzelei的下半身正是「食人花母體」，而上半身已經非常接近人類的形象。而且還是一個身材火爆的尤物。

如果有《合金彈頭8》還真希望食人花皇后可以加入，讓她擔任一個BOSS或者和主角發生一些曖昧，那就比較有意思了。

小編的圖文主要專註於街機遊戲和紅白機，希望可以喚起這代人的美好回憶。
