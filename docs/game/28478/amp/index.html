DNF：又一材料價格上漲！商人開始大量存貨，沒成想卻淪為接盤俠

天天要聞
2019-12-23

一直以來商人可以說是整個DNF的核心玩家了，不管DNF發生任何變動，都動搖不了商人對於物價的控制。要說商人控制物價那可是了如指掌的，眾所周知，國服的每次更新都是按照韓服的內容來的，而更新完成後所有的裝備和材料也是沒有任何變化的，所以說韓服只要是有一些更新的內容出現，就會影響到國服拍賣行當中的物價。就拿現在來說，100級版本的深淵才剛剛出現，商人就開始按耐不住了。要知道，這次100版本的出現很有可能是DNF再次衝擊巔峰狀態的機會，改版之後多了很多好玩的模式，就好比深淵，比現版本的副本還要強很多，因此商人們開始對深淵材料下手。

就在最近這幾天的時間當中，拍賣行中已經沉寂很多年的挑戰書價格開始上漲了起來。原本因為狗眼和不渝之詠的原因，挑戰書的價格開始大降，到後來價格雖然開始有些回升，但依舊是2000金幣左右。而最近100版本深淵剛剛出現，挑戰書的價格直接上漲1500金幣左右，目前跨六的挑戰書價格在3500金幣左右。而從整個拍賣行挑戰書的數量來看已經非常少了，目前才30多頁，看樣子商人已經開始行動了起來。但是，這次商人囤貨真的可以賺嗎？

到目前來看我們所知道的就是100級版本深淵是需要18個引導石才可以刷一次的，而一個狗眼可以換20個不可交易的引導石。而對於新深淵材料的兌換卻是20張深淵票可以兌換1個新深淵材料，也就是說如果通過深淵票兌換的話，那麼需要360張深淵票。按照目前挑戰書的價格來算的話，126w金幣左右才能夠刷一次深淵（這刷深淵的代價實在是太大）。但是照目前來看，玩家們最不缺少的就是狗眼，即使沒有很多狗眼現在囤還是來得及的。國服更新100級版本還需要大約半年左右的時間，如果通過這段時間刷副本和團本來兌換狗眼的話，也是可以兌換不少的。所以說，玩家們在打團的時候，首先考慮兌換狗眼，其次就是深淵票了。而至於商人的這次囤貨也是有很大風險的，到時候很有可能會成為接盤俠。
