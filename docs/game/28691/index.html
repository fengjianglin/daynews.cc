首席伍六七到底強到什麼程度，需要刺客聯盟表演這樣的一場戲？

2019-12-22 722條評論 3,995次閱讀 1502人點贊

顯然就是因為伍六七太過強大，強大得組織根本沒有辦法可以直接除掉他，只能夠用這種計策降低他的戒心才有機會。

十五歲成為暗影刺客

青風在21歲的時候打敗了眾多門派的高手，最終才被獲得邀請成為暗影刺客，顯然這一種是靠實打實的資歷累計下來才最終被承認。

而伍六七作為主人公一開始就各種光環屬性加身年紀輕輕就成為暗影刺客，而且還在一項任務都還沒有執行過的情況之下，這肯定會引來所有人的注意，畢竟這個年紀就與自己平起平坐其他人肯定心裡不滿，而赤牙就是最明顯的直接就找伍六七干架。

實力得到承認

青風對於組織首領賦予伍六七暗影刺客身份的決定顯然就是疑惑多於妒忌，於是暗中觀察赤牙和伍六七的戰鬥，站在一旁作為高級解說不斷的為觀眾呈現主角到底有多強，畢竟這一話看似是青風的個人秀實際上就是為了突出伍六七的強大，而且這還是還沒有得到魔刀千刃之前。

赤牙未必是暗影刺客中最強的存在，但能夠成為暗影刺客絕對是有相當的水平，伍六七在對赤牙底細毫不知情下，手無寸鐵依然可以非常冷靜的應對，真不愧是能說一口流利粵語的人。

的確正如青風在一旁觀察，並作出伍六七當時手中有足夠好的武器這一場決鬥在一開始的時候就已經決定了勝負的定論，只不過現在手中的武器根本就傷不了赤牙罷了。

作為首席實力自然在青風和赤牙之上，而最終還是差了一件好的兵器，於是清風就免費提供魔刀千刃的情報。

這裡主要是有兩個原因，其一是自己確實沒有這個能力得到魔刀千刃還不如做個順水人情日後好方便拉攏一下關係，其二是想知道這個新人到底能夠有多少的實力，雖然在某些方面確實是比自己要強但現在還沒有承認伍六七完全要比自己優勝，真不得不說青風確實是心裡盤算非常大。

擁有魔刀千刃加持成為首席

畢竟魔刀千刃被說成是天下無雙的神裝，就算要拿還是多少些難度，守護者所謂的千年惡靈的猴哥一定也很寂寞，因為被外界說得太玄了都不知道有多少年沒有被人來打搞，當然連刺客聯盟的人都覬覦的武器又不敢去搶，猴哥的實力可見一斑，估計放到組織里實力都是前列。

而魔刀千刃也就是面對斯特國王子的時候算是真正展現過威力，幾乎就是無法防禦的武器，王子的頂級裝備在魔刀千刃面前就是紙糊一樣一切就碎，另一方面確實也像青風說的那樣完全說只攻不守一碰就碎，基本上連格擋都做不到，但這也是青風夢寐以求的終極武器，畢竟自己當初沒有能力去搶，現在也就只好派梅花十三給自己搶回來。

伍六七與組織的關係問題

顯然伍六七並非像其他人那樣由首領身邊的使者邀請，而是由首領直接帶回來可以說首領說完全認可他的能力，而且在這個年紀估計上面的其他號人物還沒有什麼名氣，首領就是想把伍六七培養成為暗影刺客這個組織的未來核心。

雖然伍六七的實力毋庸置疑但太過於年輕就意味著難以控制，當中絕對是發生了一些事情而使得首領不得不剷除他，不過正面戰鬥的話肯定是拿他沒有辦法。

假如其他六個人一起上的話伍六七肯定是沒有能力可以應付，但就算其餘六個人合在一起也沒有能力可以留得下他，所以才被迫演出一場這樣的猴子戲來降低伍六七的戒心。

而現在的伍六七記憶逐漸恢復，身份也已經曝光遲早都會重新出現在名單之中，未來的戰鬥肯定是不斷有暗影刺客給找上門，這就可以呈現伍六七的真正實力。
