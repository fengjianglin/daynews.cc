熱血傳奇手游：瑪法大陸上最強的詛咒武器你知道是什麼嗎？

傳奇，想必很多人都聽過，玩過傳奇的人肯定也不在少數，隨著對傳奇的了解越深越覺得傳奇還是有許多好玩的地方，大多玩家玩過傳奇之後，都會對遊戲裡面的裝備系統有一定的了解，都知道武器有都詛咒這一說法，在遊戲里的武器隨著pk的次數增加，武器的詛咒值越高，隨著詛咒值的增加，武器的攻擊上限就會被限制，這樣的一把武器是無法發揮其最大的攻擊力，但是，有一把武器和其他的武器不同，它隨著詛咒值的增加，反而它的攻擊力越高，它就是瑪法大陸上最強的詛咒武器——羅剎。傳奇中的三大職業，特點都很鮮明，道士在三大職業中屬於輔助性更強的，主要傷害手段也是靠寶寶和放毒， 並且一手隱身技能也是非常秀的，團隊中有一個道士，也是標準配置，今天小編就來談談當初無數道士想要的武器——羅剎

羅剎，單看武器屬性，15-0的傷害並沒有什麼出奇的地方，但是這裡就要引入傳奇的機制，武器有幸運值和詛咒值，幸運可以影響武器的最大值，而詛咒影響的就是最小值了，羅剎這個裝備對於不死系的怪物是有傷害加成的，當你有一把詛咒+10的羅剎時，你的輸出是非常穩定的，所以有不少人也喜歡用羅剎練級。

但是如果僅僅是這樣，怎麼可能會讓傳奇玩家都趨之若鶩呢。

讓羅剎真正火的原因還是論壇里流傳的一則謠言，武器加了詛咒之後會提高爆裝備的幾率，而加10的羅剎在擊殺玩家之後必爆對方的裝備，一些相信這個傳說的玩家也紛紛前去購買，一時間羅剎的價格飛漲，大家都知道傳奇裡面的詛咒是根據pk值來的，一時間拿著羅剎的紅名到處殺人，玩家都聞風喪膽

具體這個謠言是真是假，現在也分辨不清了，但是小編個人看法是覺得並沒有這個隱藏屬性的

但是當時的傳奇中，很多玩家是相信這個說法的，羅剎的爭議不僅如此，關於羅剎要不要上詛咒這個問題也爭論了很多，還有玩家特意做過實驗

羅剎的初始屬性是15-0，如果一個35級的戰士只裝備羅剎，本身的攻擊是5-6，佩戴了羅剎之後的攻擊屬性是20-6，這時候，玩家的每次攻擊都是一個固定值20，羅剎每增加兩點詛咒，會增加三點攻擊上線，最多可以詛咒到10，可以得出羅剎的最高攻擊力是15-15，這樣使得使用加10詛咒的羅剎攻擊力比較平均，不會出現太大的起伏
