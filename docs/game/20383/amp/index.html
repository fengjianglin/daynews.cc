玩轉阿拉德探險記，了解這些即可輕鬆滿級

天天要聞
2 天 ago

作者：黑桃狸

DNF在添加了禮遇特權的「月卡」模式之後，12.10版本更新中，又添加了阿拉德探險記第一季(因為大部分小夥伴又調侃為戰令，為了方便稱呼以下簡稱戰令)，從獎勵上來看，本次戰令還是非常豪華的，無論新老玩家都對這個新模式的獎勵有所需求。

戰令共分為兩種獎勵三種參與形式，對於零氪玩家可以通過每日每周每賽季任務獲得一定經驗，拿到一定的獎勵，相比氪金大佬雖有一些差距，但是畢竟是正常遊戲之餘還能順便拿到不少獎勵，這樣看來其實還是非常不錯的。

高級探險家獎勵需要購買獲取，目前商城一共有兩種購買形式，4900點券的高級冒險家和15600點券的高級冒險家豪華禮包，最大的不同就是購買高級冒險家豪華禮包戰令直升40級，會省下不少升級的時間。 接下來從活動內容來分析一下任務經驗跟等級獎勵。

首先戰令第一季的時間為2019年12月10日6:00~ 2020年3月19日06:00 活動時長剛剛好100天的時間

第一季等級上限為80級，每級經驗都為1300，所以滿級一共需要102700經驗。

然後我們來看看一下每日、每周和每賽季任務經驗情況 每日任務分為兩種

普通探險家(不購買49和156)

每天3個普通任務，每個任務100經驗，每天共300經驗

高級探險家(49或者156均屬高級，除了會獲得直升這一點外基本無差別)

每天多1個高級任務，額外300經驗，每天共600經驗。

而每周和每賽季任務無論有沒有買高級探險家都是一樣的

每周4個任務，每個任務提供1000經驗，每周共4000經驗

每賽季18個周任務，每個任務提供1600經驗，所以每賽季可獲得28800經驗。

說太多了，來看最終結論 做完全部任務的話，賽季共可獲得經驗：

初級探險家(0氪)=300*100+4000*15+28800=118800經驗

高級探險家(49或者156)=600*100+4000*15+28800=148800經驗

由此大致可以看出，不買高級冒險家除了少了高級冒險家專屬獎勵之外，普通冒險家的獎勵還是能夠拿全的。

任務難度方面，每日和每周任務相對簡單，也不需要過多的投入。 所以為了最低投入拿全獎勵，就要在賽季任務上進行取捨，選擇最適合自己的。

賽季任務上通關副本次數相對不需要太大投入

所以可以選擇不做升級怪物卡片30次、合成獲得燦爛徽章2次、合成裝扮6次、分解裝扮6次，這4個任務，另外4個任務可根據個人無法達成的副本次數來進行選擇，但是這種方式相對獲取周期較長，需要到賽季結束才可以獲得全部獎勵。

對於購買4900點券高級探險家的勇士們，可獲得的經驗多46100，可以在賽季任務完全不完成的情況下拿到全部獎勵，非常適合不經常打團，總是單機的佛系玩家。 而如果賽季任務完成，那麼可以「曠工」5周半，或者滿勤提前5周半，也就是在2月7日或8日(具體看7日當天每日每周任務完成情況)即可拿到全部獎勵。

購買4900點券高級探險家的勇士們可以斟酌每周任務裡面，類似於不使用強化券強化裝備、在喬納森兌換精鍊時空石等消耗相對較大的任務可以選擇不做。

如果是購買了15600點券的豪華禮包勇士們，只需要額外獲得51350經驗即可達到滿級，如果全勤只需要6周半，最快在1月17日即可拿到全部獎勵，可以非常隨意的對任務進行挑選，或者在正常遊戲中順帶完成任務即可。

戰令經驗不僅可以從完成任務中獲得，更可以從點券消費中獲得，獲得相當於消費額度50%的經驗，戰令期間跨越2020年新年，相信非常多的勇士都會購買新春禮包。 按照往年價格為39900點券，按照50%來計算，一套新年禮包就可以獲得19950經驗，只要購買6套禮包就可以足夠戰力從1級升到80級的。

而購買了15600點券豪華禮包的勇士們只需要購買3套禮包就足夠滿級戰令，所以各位有打算購買新年禮包的勇士無需著急，可以放棄大部分消耗遊戲幣或遊戲內材料的任務，等購買新年禮包之後再填補缺少的經驗。

另外只有點券才可以獲得經驗，代幣券和歡樂代幣券消費是沒有經驗的。

各種情況下完成戰令周期已經大致清楚，那麼再把所有獎勵總結一遍。 最終結論：

初級冒險家的獎勵因為是0充就能獲得，性價比非常高，不過需要遊戲時間長，能夠完成絕大多數任務的小夥伴。

高級探險家的獎勵非常豐厚，有條件的還是推薦購買4900點券的，不過相對的其實也還是要花費一定時間來做任務。

如果時間實在是不多的話，那麼15600點券再加上後續購買禮包還會獲取額外經驗，幾乎不需要花費太多時間就能輕鬆拿到獎勵。

整體來看的話，本次戰令的設置還算合理，大家根據需要來進行選擇就行了
