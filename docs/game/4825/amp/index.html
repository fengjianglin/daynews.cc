新增三大屬性，想成為天使還是惡魔《暗黑破壞神4》物品屬性改動

天天要聞
2019-12-04

摘要

《暗黑破壞神4》首席設計師David Kim，近日他針對D4中的物品屬性改動進行了詳細地說明。相信我

《暗黑破壞神4》首席設計師David Kim，近日他針對D4中的物品屬性改動進行了詳細地說明。相信我，你看完不會覺得肝疼，因為他們總算認識到無盡的巔峰到底有多失敗了，一起來看看吧！

設計團隊在D4中認為，物品應該是角色強度的重要組成部分，而不是無限高的角色數值。玩家們有權利去決定自己想走向哪條最強的道路，而不是交給數值去決定。當然還是會有一個大的方向，但是你的「畢業裝備」絕對不是只有那一套。

D4新增了以下三個最新的屬性：

天使之力——增加所有增益效果的持續時間（例如自我增益或治療）

惡魔之力——增加所有負面效果的持續時間（例如減益或DOT傷害）

先祖之力——增加命中效果的觸發幾率

這些新屬性可以作為詞綴出現，比如+15 天使之力。除了提供上述的好處之外，這些新的屬性也可以作為觸發其他特定詞綴的先決條件。如果沒有足夠的某類力量，依然可以裝備該物品，但無法從到與該類力量相關聯的詞綴中受益。設計師給我們舉了個例子（他強調這不是最終版本，或許還會大改）：

稀有鞋子屬性：
·1448 防禦
·+15 天使之力
·3.0%移動速度
·毀天滅地 +1級（需要50點惡魔之力）
·+25%寒冷抗性（需要40點天使之力）

稀有項鏈屬性：
·15.0%暴擊傷害
·焚焦成灰 +2級（需要60點惡魔之力）
·25%火焰抗性（需要55點惡魔之力）
·10%壓碎打擊幾率（需要55點先祖之力）

這三種力量都會有一個與之對應的詞綴列表，所以根據你所關心的屬性，你可能希望專註於天使，惡魔或先祖的力量。在上面的示例中，你需要50點惡魔能量才能讓毀天滅地獲得額外的等級，需要55點惡魔之力才能得到25％的火焰抗性，或者60點惡魔之力就能讓焚焦成灰技能獲得的額外的2個等級。如果你要圍繞壓碎打擊進行構建的話，則需要堆疊至少55點先祖之力，而你需要40點天使之力才能獲得25％的寒冷抗性。

有選擇困難症的玩家可能就頭疼了，因為這讓詞綴本身也提供了更多有趣的選擇，因為它們的力量取決於在其他裝備上積累了多少相關力量。或許會找到具有完美屬性的項鏈，但是當你是專註於先祖之力的時候，這件裝備上的某些詞綴可能就需要惡魔之力。也許目前所用的項鏈是你的先祖之力的主要來源，所以裝備一個新的項鏈可能就意味著在其他地方要做出犧牲。真正適合你的角色的物品將取決於你當前所擁有的物品以及角色的構建，這就很難在網上去找到正確的答案。

以上就是《暗黑破壞神4》設計團隊對於物品詞綴、屬性設計的總體方向，當然這一切都不是最終答案，但是仍然讓人嚮往。D3中的裝備變數僅剩下一條傳奇特效詞綴+套裝體系，在這基礎上還要機制變化和提升強度。導致​大家更願意去選擇肝巔峰，結果只是考驗著廣大玩家對枯燥內容的容忍程度罷了。
