王者模擬戰：香香，不能再讓你當寡婦了！男子漢劉皇叔的崛起之路

天天要聞
2019-12-22

劉備，別號劉皇叔。在王者模擬戰中，經常作為一個掛件出現。更甚者，在」寡婦流「中被作為一個送命的悲催皇叔，突然有一種武大郎的感覺。

這怎麼可以呢？有什麼可以拯救皇叔呢？來看看」崛起流「劉備吧！

一、劉備作為主C的可行性分析

1、技能特點分析：

免傷護盾和每次攻擊增加自身雙防

讓劉備的生存能力得到了加強。隨著攻擊次數增多，越打越能抗，3星劉備如果能配合高速回藍，大招可以一個接一個無間斷釋放。

兩連噴攻擊

一定程度上加強了劉備的輸出能力，讓劉備具備了充當主C的可能性。

對比，同為戰士的曹老闆能作為主C，是因為回血和吸血，能站得住的英雄才有輸出。同樣，劉備大招的存在，讓其防禦和攻擊都有成長的可能，也是能抗能打，有作為主C的潛力。

2、劉備作為主C的優劣勢

這是臨時更換，陣容未成形的劉備主C，上2個輔助就好了

優勢：劉備是唯一一個價值2金幣的戰士。相對於3金幣的主C曹老闆，可以在前期更早成型。對比1金幣的戰士主C，站樁能力不弱於老夫子，攻擊力加成不像典韋那樣需要持續培養。

劣勢：相對宮本、呂布之類的爆發戰士，輸出較弱。

二、槍炮之戀天賦的加成讓劉備崛起

1、天賦激活條件：一是劉備、孫尚香同時在場；二是其中一方死亡。

Tip：「同時在場」是指要在場上參戰，在待戰區是不會激活的。

2、雙方受益：劉備死亡，孫尚香受益。孫尚香死亡，劉備受益。

對比紅葉相思天賦的單方受益：滿足公孫離、李信同時在場且李信存活條件下，只有公孫離受益。

小結：逆向思考「寡婦流」，可以讓孫尚香先送，劉備就可以享受槍炮之戀的增益效果。該天賦暴擊率和暴擊效果的增益效果，彌補了劉備輸出不足的缺點，讓皇叔可以作為主C來使用。

三、適合」崛起流「劉備的陣容和天賦

首先，戰士職業天賦——致殘之握是戰士作為主C的核心天賦，槍炮之戀是劉備作為主C的又一個核心天賦。其他天賦結合陣容一起來看。

1、七蜀陣容：七蜀+孫尚香+搖擺位。

優勢：3級蜀國天賦強大的回藍，讓劉備的大招能夠更快的釋放，而且釋放間隔縮短，這樣劉備就具有了更強的防禦和輸出。

具體英雄選擇：選英雄類似於選媳婦兒，要選擇心裡想要的。

如果不放棄黃忠，搖擺位可以是伽羅射手，激活了1級射手天賦。

如果對面法師陣容，搖擺位推薦輔助，激活1級輔助天賦。

如果要加強戰士輸出能力，可以用群雄貂蟬+呂布替換1個蜀國+搖擺位。

2、長城守衛軍養豬陣容：有點類似於張飛做豬苗，防禦稍弱，輸出稍強。

第一種：7長城+劉備+孫尚香。該陣容有點偏脆，本身孫尚香就是送的，再加上7個送的，有點極端，劉備如果不是最後一個存活，那就沒啥輸出了。

第二種：5長城+劉備+孫尚香+呂布+張飛。推薦使用這一陣容，激活羈絆有2級長城守衛軍、2級戰士、1級輔助。增加了雙防，可以站的更久，同時呂布也可以作為輸出點培養。

3、變種精英戰陣容：

英雄推薦：劉備+孫尚香+宮本+橘右京+盾山+木蘭+李信+呂布+孫臏。

孫臏的大招回血和移動速度加快，替代了原精英戰蔡文姬的作用。

Tip：如果有王者威嚴天賦，而對面法術傷害不足，可以用孫策替換孫臏。

四、」崛起流「劉備出裝推薦

劉備作為主C，由於沒有回血技能和天賦，泣血作為物理輸出回血，是必選的。根據情況推薦2種出裝：

1、防禦裝：泣血+極寒+爆裂或者魔女斗篷。

適合情況：對方物理輸出多選擇爆裂，法術輸出多選擇斗篷。控制多的情況下，選擇抵抗鞋。

2、進攻裝：泣血+破軍或者破曉+無盡。

適合情況：輸出環境好，沒有生命危險。

總結

劉備作為一個2金幣的戰士，本身站樁能力較強，大多數時間都把他作為一個防禦型戰士來用。如果想作為一個主C來使用，要選擇合適的天賦，致殘之握和槍炮之戀是必須的；合理選擇陣容，根據敵對陣容，從七蜀、養豬、精英戰等幾個陣容中選擇合適的；裝備的選取，要考慮劉備的輸出環境，也要注意調整站位，全面加強其輸出能力。

王者模擬戰中沒有無用的英雄，也沒有無敵的陣容。要根據敵我情況，靈活機動的進行選擇，合適的就是最好的，能打贏對面就是最好的。
