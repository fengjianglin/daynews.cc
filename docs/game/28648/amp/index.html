TheShy捐款一億韓元，就為改變路人們心中遊戲印象

天天要聞
2019-12-22

要說到LPL賽區誰對社會做出的貢獻最大，一千個玩家心中有一千個人選。像是我們熟悉的PDD，這兩年中做出了相當多的公益行為，引得粉絲和社會一片好評；肯定也有玩家覺得金咕咕doinb也是非常一個非常好的人選，要知道今年上半年的時候，金咕咕和他的老婆糖小幽也是為洪水受災地區做出了不小的貢獻，粉絲們紛紛表示太暖心了。

但是在LPL賽區中，the shy也是讓人非常感動的角色，在最近這兩天中還有一條新聞讓他的地位再一步的提高——捐款一億韓元做慈善！

這兩天中有爆料稱Theshy捐款一億韓元做慈善，其中的原因是在去年他的右臂受到了一次比較嚴重的受傷，而且當時，醫生那時候對他說：你可能再也用不了右手了。這彷彿是一個愚人節的玩笑，不能用右手，對於一個職業選手而言，彷彿被判死刑。雖然那時還在打常規賽，但是TheShy選手在兩個月期間反覆進行住院治療和手術。

於是the shy也思考了很多東西，他表示：人生不可能總是一帆風順的。在之後一定要儘快做完自己想做的事。如果現在不做，可能永遠都做不了了。於是，這一次TheShy選擇進行慈善捐款。

要知道，the shy在出道僅僅三年累計獎金就已經達到5億2千萬韓元，而且TheShy向忠清南道社會福祉部門約定五年內共捐贈1億韓元。也是職業選手中第一位在這個集團進行捐贈的選手。

The shy接受採訪的時候也表示是受到了父親做志願者活動和母親捐款的影響因此才決定捐出大額的錢資助那些貧困的人家，而他的父母則是受到了自己爺爺的影響。看來theshy會成為最受歡迎的選手也不是沒有道理，實力很強並且還非常善良的選手難怪這麼多玩家支持他，並且他還不在乎自己的年薪只在乎自己打得好不好。

這麼謙虛的選手，怪不得有如此之多的粉絲！那麼你是否也是the shy的粉絲之一呢？
