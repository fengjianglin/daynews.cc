而今天，小司馬就帶大家來談談當年熱血傳奇的故事，談談當年著名的沃瑪套裝，也就是傳說有隱藏屬性的那套沃瑪裝備。

熱血傳奇

紅寶石、思貝兒手鐲、生命項鏈……聽到這些熟悉的名字時，咱們不由得想到沃瑪名牌裝備，這套裝備主要是打沃瑪衛士爆出來，也就是刷圖沃瑪寺廟。

對於當時的玩家來說，攢足這些也是很不容易的，很多人都懷疑伺服器重啟之後，那個爆率就被官方調低了，沃瑪衛士就注水了。所以當年一開始的時候，並沒有太多人有一套。

熱血傳奇

至於隱藏屬性，那是有人做過專門測試的，結果據說是同樣的沃瑪套裝和魔法力同樣的極品裝備相比較，沃瑪裝備比極品裝備威力提升百分之五十。

可以說沃瑪套裝的法師跟極品法師，在同等魔法下站樁pk，沃瑪套裝將取得完勝。但是後來這個也被人質疑過，然而對於沃瑪裝備有隱藏屬性的事情，大家卻是都普遍承認，就像記憶裝備的天地合一功能一般，被大家所認可。
