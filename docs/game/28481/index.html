王者榮耀：eStarPro狀態爆棚，12分鐘解決TS，重現冠軍戰隊風采

2019-12-23 1,464條評論 3,387次閱讀 619人點贊

各位王者玩家大家好，我是小雨。#百度APP遊戲年度票選活動#王者榮耀冬季冠軍杯第二天的壓軸賽在eStarPro和TS之間展開。開賽之前，TS四戰全勝，與XQ並列積分榜第一，而e星則是2勝1負，並且贏的兩場全是面對KRKPL戰隊，負於Hero久競。如果本場比賽TS能夠擊敗e星，他們就有很大概率成功晉級。而e星在負於Hero的情況下，幾乎沒有可以再輸的餘地了，如果再負，2負的他們基本晉級無望。

就像貓神賽前說的，他們跟TS的比賽就是他們的生死戰。另外貓神說他們只是「小TS」，在面對同樣野核加射手的真正TS時，他們能有怎樣的比賽非常值得期待。在BP上，e星一ban就搬掉了暖陽的拿手英雄李白，而且後面繼續ban掉了娜可露露和蘭陵王，自己拿出了盤古和裴擒虎雙懲擊陣容，沒有使用射手。TS則是暖陽曜打野，十三繼續馬可波羅。

開局雙方平穩發展，都在等2分鐘的第一條暴君。2分15秒，e星拿下暴君之後再下路開團打出1換3,，奠定了前期優勢，並且通過這條暴君和這波團戰領先了1k5經濟。3分30秒，e星3人中路抓死千世的不知火舞，隨後e星下路繼續開團，但被TS打出1換2，中塔血線被壓到一半以下，略虧。6分16秒，TS中路開團1換1，拖住了e星的節奏。

7分半鐘，e星拿下先知主宰。10分鐘雙方在中路開團，e星先是擊殺TS的張飛和項羽，但是e星有點上頭，追擊TS剩下的3人反而被對方擊殺3人。不過僅僅一分鐘不到，TS開黑暗暴君被e星搶到，並被e星團滅。e星利用中路兵線一波推掉水晶，拿下比賽勝利，總戰績來到3勝1負，諾言的關羽拿到MVP。

本場比賽過後，eStarPro不僅保留了自己晉級的機會，也挽救了RW俠和Hero，剛剛打完比賽的RW俠全員變成e星粉，坐在台下為e星加油。只要e星剩下的比賽全勝，他們還是有機會晉級的，而TS和XQ也同樣如此，RW俠和Hero也並非沒有機會。貓神在賽後採訪中也直言，不打到最後一場，積分榜肯定會撲朔迷離，而且很有可能會打附加賽，也算是把懸念來到了最後。

祝幾支戰隊在最後一天的比賽中都能發揮出自己真正的實力，加油。
