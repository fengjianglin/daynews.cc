“吃雞”新增2個新遊戲角色，與2種近戰武器，角色模型很“傳神”

天天要聞
2019-12-07

摘要

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友“大忙人”刺激哥。有小夥伴私信刺激哥：“想預約個時間，一起打遊戲”。事業心比較重的刺激哥，直接拒絕了這次的邀請。這段時間太忙了，刺激哥進入遊戲，一般都是簡單的體驗一下，就繼續寫文章了。

大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友「大忙人」刺激哥。有小夥伴私信刺激哥：「想預約個時間，一起打遊戲」。

事業心比較重的刺激哥，直接拒絕了這次的邀請。這段時間太忙了，刺激哥進入遊戲，一般都是簡單的體驗一下，就繼續寫文章了。

為了不因為遊戲體驗，而破壞我們的友誼，還是等刺激哥忙完這陣，再一起玩遊戲吧！

話題：「吃雞」12.7雪地吉利服上線，幸運值可提升獲得粉色物品概率！

其實，有很多粉絲都好奇：「刺激哥在線的時候，究竟在幹什麼？」刺激哥在線的時候，所涉及到的內容包括：查看更新內容，尋找寫作素材，然後構思創作內容。偶爾，也會開啟直播，放鬆一下疲憊的身心。

如圖所示，刺激哥在《Pubg Mobile》查看更新內容時，發現國際服在12.7上線了「雪地吉利服」。

其實，這件衣服已經不是第一次在國際服上線了，這次僅能算是「返場」。另外，這件白色的雨衣，與「戰爭雪怪」相比，稍微還要遜色一些的。

另外需要提到的是，這款「雪地吉利服」所在的補給箱，特別註明了一行小字：「幸運值可以提升獲得粉色物品的概率！」

只不過，這款「雪地吉利服」明顯不是粉色品質的物品，而是更高品質的紅色物品。

那麼，粉色品質的物品指的又是什麼呢？如圖所示，98k武器皮膚、吉普車皮膚、飛機皮膚、聖誕老人套裝這些，都屬於粉色品質的物品。

至於聖誕老人的鬍鬚面飾，以及降落傘皮膚，則屬於紫色品質的物品。所以，補給箱右上角標註的「幸運值可以提升獲得粉色物品的概率」，自然是沒有多大用途的。

畢竟，刺激哥可不想穿著紅色的衣服，在遊戲中到處招搖。

提到載具皮膚，刺激哥倒是想要吐槽《和平精英》的。個人感覺，《和平精英》與《Pubg Mobile》相比，更新的節奏明顯太慢了。

運營了這麼久，《和平精英》的遊戲商店，只出現了一款吉普車的載具皮膚，至於比較炫酷的皮膚槍，我們也是只能望洋興嘆。

新增2個新遊戲角色，與2種近戰武器，角色模型很傳神！

值得關注的，並不是只有雪地吉利服與載具皮膚，還有新增加的2個新遊戲角色，以及2種新近戰武器。這兩個新遊戲角色，以及近戰武器的出處，來源於《行屍走肉》。

這次的更新，《Pubg Mobile》明確的告訴我們，光子與《行屍走肉》聯動了。

值得注意的是，這次新增加的遊戲角色的模型，與之前上線的遊戲角色模型相比，要更加的精緻以及傳神。

大家可以回憶一下之前上線的「弩哥」，以及另外一個叫「卡莎」的遊戲角色的模型，然後對比一下。

《Pubg Mobile》新上線的2個遊戲角色，無論是眼神，還是其他細節，都是惟妙惟肖的。與之前的相比，刺激哥反而感覺以前的有些「糊弄」了。

刺激哥都有點懷疑，《Pubg Mobile》的美工有「扮豬吃老虎」的嫌疑了。甚至懷疑，光子是不是又挖來了新美工。

希望，國際服可以把建模師借給《和平精英》用一下吧，讓《和平精英》也多更新幾件好看的衣服。

至於新的近戰武器，我們明顯可是看出是一個棒球棍，與一把武士刀，刺激哥就不再進行過多的敘述了。

好啦，以上就是《「吃雞」新增2個新遊戲角色，與2種近戰武器，角色模型很「傳神」》的遊戲內容了，更多吃雞遊戲話題，還請繼續關註：刺激實戰教室。
