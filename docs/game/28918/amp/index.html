出門裝

回家先買鞋子，然後升級打野刀。藍色打野刀抓人厲害一些，紅色打野刀更適合野區單挑。之後做出幽夢和暮刃，升級三級鞋。如果優勢比較大，選擇無盡和吸藍刀的雙暴擊組合。如果局勢不順利，則考慮飲魔刀和板甲。

輸出方式

主要依靠E技能加成下的普通攻擊打傷害，E踢到人之後使用Q和W。這套連招打完，正常發育情況下AD就只剩血皮了。

技能主Q副E，二級點W，我們需要W的回血效果打F6。

最強的一套是幽夢+疾跑+狼頭+E技能，四個技能都在可以輕鬆秒殺脆皮。

R技能前期可以作為控制。後期用E進場秒殺AD後，使用R技能回來，等E技能的CD準備下次切入。

打野思路

前期優先gank有隊友配合控制的線。正常發育情況下，不要去抓偏肉的上單，比如鱷魚，狗頭之類。一定要記住，我們是刺客。如果一套技能打不死別人，就會處於很危險的境地。
