DNF：最帥氣的武器特效，刷圖才能觸發，身後跟隨兩道殘影

2019-12-12 215條評論 4,410次閱讀 1292人點贊

帥是一輩子的事，傷害只是暫時的，當擁有界武器之後，你才發現，角色的檔次無形之中，又提升了一個層次。不過，普雷團本這把武器，並非每個人都能得到，只有歐皇玩家才能專屬。很多玩家以為，界武器只有站街幻影，以及權能變紅特效。其實，還存在著一個隱藏的特效，這是最為帥氣的，刷圖能震撼全場！

這個特效隱藏較深，一般玩家都不知道，而且，僅界武器可以觸發！對於黑暗權能「攻擊狀態」，大部分都賦予的是靈魂切割機生成效果，也就是我們俗稱的「大爪子」，可額外的增加傷害，副本清小怪最佳。但是，比較遺憾的是，你賦予「大爪子」效果之後，「攻擊狀態」就無法二次選擇！其實，在這個權能中，還有一個黑暗激活效果，並非看重傷害，而是其有帥氣的特效！

黑暗激活效果，你在發動這個權能時，可增加20%的移動速度，且持續15秒時間，在腳底生成小黑漩渦。在玩家眼中，這個權能效果，中看不中用，就沒人願意選擇，殊不知，把隱藏的特效埋沒！在界武器下，角色賦予這個權能效果，腳底小漩渦變紅色，也會出現兩道殘影，跟隨在人物後面，可謂是十分帥氣！

如圖所示的影舞者，就賦予的黑暗激活效果，在界武器之下，實現了殘影特效。你還別說，這個職業本身就蘊含著「影」，而今真正觸發了出來，甭提有多帥氣！

當然，這個特效不限職業，只要你擁有界武器，且「攻擊狀態」權能選擇的是黑暗激活，那麼，就會在刷圖的時候觸發。比如紅眼角色，打盧克時，那兩道殘影跟隨，讓隊友都無比羨慕！值得一提的是，這個界武器的特效，很少有玩家知道，畢竟賦予「大爪子」的居多！

總之，界武器對玩家的誘惑力很大，不僅站街有幻影，觸發的權能變紅，甚至刷圖還能出現兩道殘影！只是，這把武器「砸錢」都得不到，歐皇玩家專屬。旭旭寶寶和狂人，這兩位超級土豪，玩的大號角色，也都至今沒翻出來！
