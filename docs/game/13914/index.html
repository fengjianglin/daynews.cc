索尼全球工作室將繼續製作以劇情和角色為導向的遊戲

2019-12-12 471條評論 1,515次閱讀 93人點贊

本世代玩家已經見證了索尼的第一方作品，在全球工作室的「保護傘」下，開發者們推出了不少優秀的作品，很多甚至定義了這一世代。這要歸功於索尼公司對敘事遊戲和角色導向型遊戲的成功定義和堅持。儘管索尼全球工作室發生了領導層的變化，即將離職的Shawn Layden被Guerrilla Games的老闆Hermen Hulst取代，但他們的第一方團隊不會改變他們開發遊戲的方式。

在接受GamesIndustry採訪時，Hulst表示，未來索尼的第一方工作室將繼續像過去幾年那樣，「以劇情和角色為核心，創造宏大而壯觀的體驗」。

Hulst表示:「過去十年，我們一直專註於在全球工作室開發的遊戲類型：以劇情和角色為核心的宏大而壯觀的體驗。」「我們將繼續創造這些遊戲，因為我們喜歡創造它們。作為一個品牌，我們渴望拓展業務，並開始擴展和組織新一代的開發者，為新一代的玩家創造全新的、不同的體驗。」

儘管索尼的下一代計劃仍籠罩在神秘之中——至少就軟體而言是這樣——但他們仍有大量重量級作品繼續為PS4主機準備。《夢境》將於明年2月推出，《美國末日2》將於5月推出，而索尼公司也將在明天的遊戲頒獎典禮上首次發布《對馬島之鬼》的最新預告片。
