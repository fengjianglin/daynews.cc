DNF：12.24開服1億金幣試水，4個黃力+1個畢業白金，這波真不虧

DNF：12.24開服1億金幣試水，4個黃力+1個畢業白金，這波真不虧！

以下為具體的內容詳情：

DNF在今天維護更新完成之後，那個前幾天體驗服爆料的徽章轉換功能已經上線到了遊戲中，玩家均可到達芙妮那裡消耗金幣進行參與。且在這剛開服的第一時間，就有玩家拿了1億左右的金幣去試水，最後的幾個喜提4個黃力+1個畢業白金徽章，這波真的是不虧，甚至是血賺！

針對於這個徽章轉換的玩法小寶在這裡就不多做介紹了，對比起來最大的亮點當屬滿30次之後可以有幾率轉換出絕版的黃色燦爛力量和體力的徽章，這可是比白金還要稀有不知道多少倍的徽章啊，相信絕大多數玩家應該都不會錯過這個可以讓角色的細節打造達到極限的機會。

而這不就像上面這位玩家就是如此，據了解在今天開服後，總計花費了有1億左右的金幣去試水轉換的幾率；而最後的結果也證明了這位玩家的運氣非常不錯，1億金幣喜提一個奶媽專屬的勇氣祝福白金徽章，外加4個物理輸出職業專屬的畢業+15黃力徽章（轉換出的黃力/體，綠智/精徽章均為賬號綁定，且有刪除期限）。

1億金幣換來一個畢業白金+4個黃力徽章，這不用說絕對是血賺。但是在上面小寶也說了這位玩家的運氣是屬於狗托類的，在貼吧當中還有很多四/五千萬金幣下去一個黃力都沒見到，一兩個億金幣下去一個畢業白金都沒隨機到的玩家。

所以說別人能做到的事情並不代表著你也能做到，主要還是看運氣，小寶在這裡建議那些還沒來得及參與的玩家，一定要結合自己的運氣好好考慮一下在做選擇，不然……
