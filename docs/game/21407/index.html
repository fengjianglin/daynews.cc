你要的遊戲終於回來了！《最終幻想7：重製版》最新情報送上

2019-12-20 1,400條評論 3,131次閱讀 1609人點贊

日本知名遊戲雜誌Fami通上刊登製作方SE公布的《最終幻想7：重製版》的本月最新消息，其中包括遊戲的主視覺圖、新角色形象以及戰鬥技能介紹和遊戲場景的部分截圖，官方透露該作將會支持簡體中文的字幕。

《最終幻想7：重製版》是由Square Enix基於經典的《最終幻想7》所進行的的完全重製版。關於遊戲的故事背景，以星球的生命能源——魔晄為中心展開，米德加是一個擁有豐富的魔晄資源的階級城市，關於掌握了世界命脈的龐大企業「神羅公司」與保護星球挺身而出的與「神羅公司」鬥爭的「雪崩」之間的衝突。前神羅戰士克勞德，參與雪崩的活動與神羅公司之間的鬥爭。

遊戲角色：賽菲羅斯(CV：森川智之)，是實力最強的神羅戰士，曾在五台的戰鬥中立下了無數的功勛，被授予英雄的稱號，傳說他在執行某個秘密任務時喪生，許多後來人想要追尋他的足跡成為神羅戰士。

神羅公司組織：控制魔晄能源的巨大公司，以米德加城市為核心掌控了全世界，企業中包含兵器開發、治安維護、科學和都市開發等功能部門，分別由能力特色迥異的部長負責管理

神羅總裁(CV：若山弦藏)，精明而有手腕的商人，為達目的不擇手段。

艾莉絲的戰鬥介紹：艾莉絲可以發動遠距離的魔法攻擊，連續發動魔法技能可以對大範圍的敵人造成傷害。固有的能力是風暴，一直按著按鈕可以擊類能量，發動技能的威力會十分強大。

遊戲設計中增加了中心城市米德加的更多內部細節以及城市居民生活的風貌和細節，使故事背景更加豐滿和真實，遊戲中也加入了不少的新插曲。

在建設米德加城市時使用過的列車車庫，城市建設完成後把報廢的列車廢棄在這裡的地方，叫做列車墓地，堆積的列車錯綜複雜宛如迷宮的設置和遊戲原作中差不多。

角色裝備了召喚魔晶石之後，在戰鬥中積累了足夠的能量之後，還能召喚出各種召喚獸幫助自己戰鬥。
