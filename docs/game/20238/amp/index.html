9.24雲頂排位周報，四大陣容強勢出爐，網友：竟沒有雷霆劫？

天天要聞
2019-12-17

12月12日雲頂之弈更新至9.24版本，三位英雄空降雲頂之弈，阿木木撐下了地獄火的前排排面，盧錫安夫婦帶來的新羈絆更是讓新陣容層出不窮。來看看9.24版本第一周的排位周報，助你早日問鼎雲頂之弈。

陣容統計

只統計12月12日-12月14日最近的10場排位賽。

只統計進入前3名的局。

統計艾歐尼亞伺服器排名前10玩家。

統計場次100場

統計截至時間：2019年12月14日

陣容場次分析

統計一百場次，共計進前三場次46場。其中前三率最高的陣容為鋼鐵沙雲劍刺共11場次，佔比約23.9%。前三率第二陣容為海洋守護山法共8場次，佔比約17.3%。前三率第三的陣容為掠食劇毒鋼/水與6影秘術魂均為6場次，佔比約13%。

統計一百場次，共計吃雞場次24場。其中吃雞率最高的陣容為鋼鐵沙雲劍刺共6場次，佔比約25%。吃雞率第二陣容為海洋守護山法共5場次，佔比約20.8%。吃雞率第三陣容為6影秘術魂共4場次，佔比16。6%。

鋼鐵沙雲劍刺成為版本最強吃雞陣容；6影體系成為版本黑馬前三率超高；海洋召喚法退出統計舞台。
