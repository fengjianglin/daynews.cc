夢幻西遊：MHB跌破200，是否符合策劃預期？三界功績威力不再？

天天要聞
2019-12-07

摘要

各位夢幻玩家大家好，我是你們的老朋友李叫獸，“李叫獸談遊戲”每天帶你了解最新遊戲資訊。今年MHB的價格極其不穩定，春節過後開始一路下跌，再加上開放客戶端八開，更是加劇了MHB的貶值，策劃說是因為工作室大量湧入導致的，於是推出三界功績，抑制MHB的產出。

各位夢幻玩家大家好，我是你們的老朋友李叫獸，「李叫獸談遊戲」每天帶你了解最新遊戲資訊。

今年MHB的價格極其不穩定，春節過後開始一路下跌，再加上開放客戶端八開，更是加劇了MHB的貶值，策劃說是因為工作室大量湧入導致的，於是推出三界功績，抑制MHB的產出。然而三界功績實施之後，好像並沒有達到理想效果，如今剛剛進入12月份，很多玩家都發現MHB的價格又跌了，很多區的MHB比例跌到了三千萬不到200元的價格，甚至比三界功績實施之前還要低，究竟是什麼原因導致的這種狀況？

​

看到近期的MHB大跌價，各路玩家紛紛發表自己的看法，其中有人認為，是因為策劃打擊工作室不利，特別是腳本工作室，每天挖圖和刷金的產出太高了，單純的靠三界功績模式也許對於單人和五開玩家來說，受到的影響很大，畢竟大家都是手動刷的畢竟精力有限。

但是腳本工作室是機器，可以通過增加機器數量，來彌補利潤，說白了就是三界功績實施之後，腳本工作室為了彌補因為三界功績而減少的收入，反而加大了投資，刷金的力度並沒有受到三界功績的影響！

上圖為叫獸記錄的梧桐山金價數據，從今年8月份直到12月份，MHB的金價走勢。數據簡單明了，在7月底三界功績實施之初，金價短暫上漲，隨後下跌，而後的9月份雲遊道人出現，雲遊道人的威力，大家都知道，一天狂吸幾十億的MHB，成功將區內的金價拉高，在道人走後，MHB的價格曾經保持穩定了一段時間，直到最近這半個月，金價又出現了大幅度的下滑，目前梧桐山的金價已經是跌到了三界功績實施之前的水平。是三界功績威力不在了嗎？還是像有些人說的那樣玩家都退遊了，沒人消費了？

其實通過叫獸統計的最近三個月18門派闖關隊伍數量來看，10月，11月，12月份這三個月18門派闖關角色總數量是：159萬，149萬，153萬，總體玩家數量並沒有減少。而且三界功績的作用確實在的，假如7月份沒有出三界功績的話，現在的金價可能已經跌倒180甚至是170了，三界功績延緩了金價下跌的勢頭。

但是客觀來看MHB跌價就是供大於求的原因造成的，僅僅靠三界功績來限制產出還不夠，還要提升消耗MHB的途徑，讓系統來回收產出的MHB，形成一個循環。這樣才可以解決MHB跌價的根本問題。因此叫獸認為，今年的資料片肯定會出現一種全新的系統，大量回收玩家手中的MHB。

因此大家也不用擔心，相信策劃對於MHB的價格一直都有關注，肯定會出台相應措施解決這個問題。​
