王者模擬戰：武則天一發入魂，曾經霸主級存在的扶桑七法

2019-12-19 313條評論 1,485次閱讀 291人點贊

大家好，本期為大家帶來的是暴力上分陣容，成型之後基本上穩吃雞，不過基本上沒有成就分，加分比較少，個人不太喜歡。話不多說，直接開始教學。

陣容英雄推薦：

雙群雄（呂布、貂蟬）、雙扶桑（不知火舞搭配橘右京或者宮本武藏）、七法師（武則天、貂蟬、不知火舞、弈星、甄姬、小喬、墨子）

陣容理解：

由於呂布和貂蟬激活的群雄羈絆會直接加入扶桑陣營。變成4扶桑，然後激活全員轉40%真傷的效果，配合武則天大招可以打出非常可觀的傷害。

追三星的選擇：

前期不知火舞盡量追三星，打架很厲害。貂蟬出的多也可以拿一下，漏其他牌不要緊，刷牌的時候都能追回來。呂布和武則天后期臉白而且沒人搶牌可以試著追一下三星，希望不大，建議別嘗試。

裝備選擇：

優先拿回藍裝或者保命裝給武則天，讓武則天順利開出大招。剩下的好一點的法裝給不知火舞，打收割，出來破軍可以拿給呂布。反正記住一點：一定要讓武則天放出大招。

天賦推薦：

混亂之術，法師必備天賦。致命魔術，和混亂之術一樣也是必拿的天賦，效果帶來的暴擊有時候會有奇效。煉獄之火，這裡為什麼要推薦這個天賦呢，因為出來這個天賦我們就可以把小喬換成周瑜，墨子換成沈夢溪，前期傷害非常可觀，不容易掉血。致殘之握，拿這個天賦的前提是你除了不知火舞外的另外一個扶桑到了三星而且本局拿呂布的不多，可以合成2星呂布。如果再拿到一個戰士令旗的話這個天賦性價比就很高了，沒什麼好天賦可以拿一下這個。特價甩賣，有了這個天賦刷牌的時候，可以省很多錢。

陣容運營：

前期上長安牌快速拉人口，到8人口可以快速刷牌把陣容提到二星，保證自己不會掉太多血，或者直接拉到九人口找到呂布武則天陣容成型。如果有不知火舞、沈夢溪、周瑜的加傷害天賦可以將墨子、小喬換成周瑜、沈夢溪（上面已經說了）陣容成型全部2星的時候基本上就可以打過大多數陣容了。到後期所有牌能追三就追三，誰來的多拿誰，如果被卡牌還是按照我上面說的優先順序進行取捨，別到時候把自己卡死了。

好了，本期教學就到這裡了，喜歡記得點個關注，我們下期見。拜拜！
