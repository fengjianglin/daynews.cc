曆數CSOL十一年間的那些關鍵詞

天天要聞
2019-12-10

2019年是CSOL的第十一年，或許大家都沒想到一個簡簡單單的槍擊遊戲能夠在中國的遊戲市場屹立11年。讓我們把時間線拉回2008年12月16日，世紀天成宣布成功代理NEXON的FPS網遊《CSOL》，在那個網遊並不多的年代，一款畫質精良、玩法新穎的遊戲，剛上線就達到萬人空巷的效果，屬實罕見。那麼CSOL在這之後又創造了怎麼樣的輝煌呢，讓我們用一個個關鍵詞來曆數這十一年來的CSOL。

生化模式

CSOL是以V社《CS1.6》為基礎開發的一款遊戲。在那個時候，射擊類遊戲的目標還停留在人與人之間的對戰，而所謂的人與殭屍間的對戰，也只存在於部分殭屍服當中。CSOL吸取殭屍服的經驗，通過不斷的測試，將殭屍與人對戰的模式進行強化修改，最終迎來了CSOL開創的”生化狂潮”時代。在那個時候，網吧里連坐的十個有八個都在CSOL打殭屍，為的就是那種虐殺的快感。

左輪軍魂

作為CSOL國服的第一把人民幣武器，它有著里程碑的意義。雖然它面世最早，但卻一直是人民幣玩家心中無法磨滅的痛。因為在最早期的密碼寶箱，出現幾率最高的獎勵就是”左輪軍魂（3天）”。而所謂的骨灰盒玩家都是沖著永久武器去的，基本上每抽到一把永久武器就得抽到90天以上的左輪軍魂。

毀滅者

第一代神器，第一把永久武器。說到這把槍，新玩家可能很陌生，但對於老玩家，毀滅者就代表了生化模式。在曾經的生化1和生化2模式，人類方只要有了三四把毀滅者，那麼殭屍們就可以放棄遊戲了。那時候，帶上毀滅者打生化模式，會有一堆小玩家把自己的看家”武器”扔在你面前，並一個勁地希望能夠和你交換。那時候，擁有毀滅者就代表擁有了勝利的希望。時過境遷，雖然現在CSOL里的神器包羅萬象，性能傷害比毀滅者厲害的比比皆是，但誰也取代不了毀滅者的經典地位，因為它是一個時代的烙印，他是CSOL生化模式的代言人。

戰樂靈弦

說到這把神器，更多的是哀傷。那個夏天，邁克爾傑克遜離開了這個世界；那個夏末，多少玩家為了彈奏一曲”靈弦”而夢碎骨灰盒。當年貼吧里還有一位叫做”天熊丸”的土豪，為了一把戰樂靈弦，壕擲3000大洋，結果最終鎩羽而歸，而他也成為了貼吧的一代傳說（在那個年代，肯在遊戲里花3000元的已經算是中上水平玩家了）。

密碼寶箱

密碼寶箱（又名骨灰盒），是CSOL生化狂潮版本中添加的一項具有跨時代意義的道具。它的出現標誌著CSOL的”大菠菜”時代降臨。因為這個箱子里，蘊藏著無限的寶貝，各種稀有槍支、各色人物以及噴漆裝飾等。而且很多都是商城無法購買，只能通過抽獎獲得。那些年，多少玩家玩家為了能抽到一把在骨灰盒裡的心儀神器，連飯錢都省下來，就為了那可憐的爆率。雖然神器逐漸增多，爆率也稍微有所提升，但被骨灰盒坑走的玩家，卻再也不會回來了。

大災變

大災變是反恐精英中人氣最高的模式之一。該模式在秉承生化模式驚悚恐怖的同時摒棄了之前的遊戲機制。由傳統的PVP模式轉向PVE模式。只要技術夠好，玩家一個人也能在這個模式里獲得遊戲的樂趣。在這個模式里，你需要想辦法活下去，活著，就是你唯一的任務。在這個模式里，你需要風騷的走位，需要超前的意識，也需要精密的計算。

今年的CSOL發布了”槍戰熱愛，一直都在”的宣傳視頻，將這十一年來的CSOL的歷史用玩家的視角表現出來。讓眾多觀看此視頻的老玩家們都有一種滄海桑田，但是我們依然待CSOL如初戀的心情。因為不論我們的外表和生活如何變化，我們的內心依舊是那個喜愛CSOL入骨的玩家，我們會永遠都在，因為我們對槍戰愛的深沉。
