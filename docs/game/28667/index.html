陰陽師：海國篇落幕，大岳丸大獲全勝，玩家們卻不悲反喜

2019-12-22 1,032條評論 2,779次閱讀 1691人點贊

玄不救非，氪不改命，老鹹魚ACG次元飯貓君給各位陰陽師大人們請安了。
在經歷一星期的大決戰，陰陽師京都決戰終於落幕了，原本以為是「偽神降臨」第二期，卻不想最後來了個失敗的結局。在21日20點，活動結束，但是最後的大BOSS海鳴卻還基本上的滿血的狀態，最後玩家拿到了失敗的徽章。

可能大部分玩家都沒有想到策劃可能會真的來這一手，失敗的劇情居然也敢寫出來，於是陰陽師直接又被送上了熱搜。

但是讓ACG次元飯君好笑的是，熱搜裡面雖然有一部分玩家在噴策劃，但是大部分玩家卻在歡慶鼓舞。不僅沒有半分悲涼的氣氛，反倒是為反派大岳丸喝起了彩，甚至還表示自己之前參與戰鬥是被脅迫了，自己從頭到尾都是少主大岳丸的人……不得不佩服這些鬼才玩家，真的是說變就變。

不過仔細研究之後發現，這裡面其實暗藏玄機，大家都猜到了策劃這麼做肯定是為了宣傳新式神雲外鏡，而雲外鏡的能力跟虛實有關係，而且之前在SP大舅的劇情中，大舅也拿到了神器說要幫助晴明。所以最後劇情反轉，一切都是虛假的這樣的設計是可能的。

而且在大岳丸攻下京都後發現，庭院都變成了海國的妖怪，少主真好，來了還給我們換了套庭院皮膚，我永遠都是少主的人（破音）！但仔細看這些妖怪後會發現，出現很多模糊，按照陰陽師的遊戲設計水平不至於出現這麼劣質的BUG，那麼就只有一種可能，那就是這些BUG都是雲外鏡的環境照成的，這其實是鏡子里的世界，也就是說是假的！

這麼一看似乎一切都變得合理起來，這次活動出了兩種徽章，對於有強迫症或者收集癖的玩家來說只拿到其中一個肯定很難受，於是策劃就用這個設計巧妙地送出兩個！不知道大家覺得是不是這樣呢？（時間1221也是鏡子，滑稽）
