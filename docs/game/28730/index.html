《超凡雙生》：每個選擇導向不同結局，唯有堅強方能獲得幸福人生

2019-12-22 592條評論 2,637次閱讀 178人點贊

一款優質的遊戲應該具備什麼樣的特點？隨著遊戲產業的日漸興盛，不同種類的遊戲逐漸進入蓬勃發展的階段，因為各自的側重點有所不同，所以也無法用一個標準進行衡量。對於互動式電影遊戲來說，劇情無疑是最為關鍵的一環。這類遊戲「看」的成分與「玩」的成分幾乎不分伯仲，製作者通過電影鏡頭來表現遊戲劇情，玩家的一系列操作會引發事先設計好的劇本，從而產生出互動的效果。說起互動式電影遊戲，Quantic Dream所製作的《暴雨》曾經風靡一時，作為該公司的又一力作，《超凡雙生》能否帶給玩家極佳的遊戲體驗？

《超凡雙生》是由Quantic Dream研發的一款互動式電影遊戲，作為一家來自於法國的遊戲工作室，Quantic Dream以細膩的角色呈現與互動式的劇情故事聞名於業內。近些年，工作室推出了不少佳作，從《暴雨》、《超凡雙生》、《幻象殺手》到《底特律：變人》，Quantic Dream將大部分精力都放在了「互動式電影遊戲」的身上，在不斷嘗試中，他們也逐漸找到的一條適合自己的發展之路。隨著技術的大量積累，場景設計、人物動作表情、敘事結構都取得了較大提升，在製作團隊的努力之下，玩家對於互動式電影遊戲有了全新的認識，也看到該類遊戲所特有的魅力。

《超凡雙生》主要以劇情為主，在遊戲之中，玩家將要扮演一位名叫祖迪·霍姆斯的女孩。與普通人不同，祖迪能夠看到常人看不到的東西，並且她還能夠與靈體「艾登」產生心靈聯繫，從而獲得超自然的力量，在特殊天賦的影響下，祖迪接連遭遇了各種突發事件。作為一款互動式電影遊戲，《超凡雙生》在劇本設計上面下足了功夫，龐大的分支劇情為玩家提供多種可能，每一個不經意間的選擇都有可能走上截然不同的道路。在技術層面上，該作比《暴雨》更為精細，在畫面、人物、劇情的包裹之下，「懸疑感」被充分展現出來，最終也帶給了玩家極佳的沉浸感。

細膩的畫面設計帶給玩家身臨其境的感覺

想要成為一款優秀的互動式電影遊戲，自然需要在「互動」上進行細緻打磨。代入感與沉浸感是該類遊戲主打的特色，想要實現這一效果就需要在畫面進行精雕細琢。《超凡雙生》採用了真人拍攝的方式，玩家的選擇影響著遊戲中人物的一舉一動，從而「實時互動」的效果被完美呈現出來。最讓玩家所稱讚的就是人物建模，遊戲中的主要角色都進行了細緻處理，基本上接近CG級別。為了讓人物的動作更加流暢，製作團隊請來了好萊塢影星艾倫·佩姬與威廉·達福進行動作捕捉。通過演員精湛的演技，遊戲角色被賦予了「人氣」，約會時的傲嬌與害羞，看到奇怪現象後的惶恐與無助都顯得十分真實，角色的情感顯得更加有層次感，精細度得到了大幅提高。

《超凡雙生》對於遊戲場景也進行了細緻打磨，為了將情感正確無誤地傳遞給玩家，製作團隊對光影效果進行了藝術化加工，從而產生一種電影般的質感。另外，在鏡頭語言方面，《超凡雙生》也絲毫沒有含糊，場景的切換、動靜鏡頭之間的結合，平行蒙太奇的運用都使得遊戲觀感大幅提升。雖然說，在一些路人NPC上處理略顯簡陋，有的時候會讓玩家產生一種不協調的感覺；不過，從總體而言，該作的視覺體驗十分不錯，將互動式電影遊戲的代入感與沉浸感很好地表達了出來。

多線劇情的設計為玩家帶來了「多種」可能

劇情絕對算是《超凡雙生》的最為重視的一環，為了能實現「互動」的效果，製作組設計出一個長達2000多頁的劇本。遊戲擁有數量龐大的分支劇情，玩家所做出的每一個決定都會對主角的發展起到重要影響，可以說每一個選項都意味著玩家將要拿到不同的劇本，同時也會上演不同的人生。

《超凡雙生》將焦點放在了祖迪身上，整個故事線橫跨了她十五年的生命周期，通過不同的遊戲章節，逐一向玩家展示出來。在敘事手法上，《超凡雙生》並沒有嚴格遵循時間順序，而是在祖迪不同的生命周期間來回跳轉，這種手法在電影上運用的比較多，諾蘭的《記憶碎片》、昆汀的《低俗小說》都採用了類似的敘事結構。將敘述方式打亂之後，遊戲的懸疑感得到了進一步提升，因為玩家所獲得的線索來自不同的階段，所以會呈現出「碎片化」的狀態，只有完整走完遊戲流程才能得到真相，可以說，不到最後一刻，玩家是很難將故事線串聯在一起的。《超凡雙生》將一個完整的故事打碎，並通過碎片化的方式展現給玩家，從而大大提升了懸疑感。

所謂成也蕭何敗也蕭何，亂序的敘事手法也給玩家帶來了不少困擾。一般來說，電影時長大多控制在兩個小時以內，劇情之間的來回跳轉會讓觀眾注意力高度集中，最後的大結局也會帶來震撼的效果。然而，《超凡雙生》的整個流程大約為12到15小時左右，玩法很難保持長時間的精神集中，從而也會帶來一種分裂與疏遠感，最終產生不太連貫的感覺。雖然在敘事手法上有所欠缺，不過在故事完整性上，《超凡雙生》也算是取得了不錯的效果，祖迪與幽靈之間的故事深深吸引了玩家，一些日常發生的小事更是讓人記憶尤深，多結局的設定也極大地提高了遊戲的互動性，玩家的每一個操作都是在書寫自己的人生。

充滿懸疑色彩謎題推動玩家勇敢追尋幕後真相

作為一款互動式電影遊戲，《超凡雙生》沒有極限的操作，也沒有精彩的戰鬥畫面，它憑藉著優秀的謎題設計牢牢抓住了玩家的好奇心，為了尋找迷霧之中的真相，玩家必須勇敢探索下去。

祖迪從出生起便於一般人不同，她可以與一個名叫艾登的幽靈進行精神上的聯繫，從而獲得知曉他人靈魂的超能力。因為異於常人，祖迪從小就遭到了同齡人的排擠，為了弄清楚自己身上的秘密，她進入了特殊的研究所，後續又開始為CIA工作。《超凡雙生》在祖迪身上埋藏了一個個秘密，為什麼她能夠與幽靈共生？面對人生中出現各種的突發情況，祖迪到底應該如何去面對？遊戲埋下了無數伏筆，玩家需要逐一去破解。

勇敢面對人生道路上的辛酸與坎坷，積極樂觀方能獲取幸福人生

祖迪的一生並不順利，辛酸與坎坷成為了她的主旋律。因為天生異能，祖迪遭遇了排擠，而她的父母也沒有盡到自己的責任，在日常生活中也十分嫌棄祖迪。面對外部的各種壓力，祖迪沒有選擇認輸，反而變得更加堅強起來。為了能夠調查自己的身世之謎，她無數次將自己置於危險的境地之中，「殺不死我的，必然使我更加強大」，從祖迪身上，玩家能夠深刻理解這句話的含義。即便生活帶給了我們無數的辛酸與痛苦，也要積極樂觀地對面對苦難，這樣的人生才更加有意義。

結語

作為一款互動式電影遊戲，《超凡雙生》通過優秀的人物建模為玩家帶來了真實的觸感。雖然敘事結構上面存在著一定的問題，不過也將懸疑感很好地展現了出來，通過回顧祖迪15年的人生之路，玩家能夠體驗到人生的多變，面對各種各樣的磨難，唯有積極樂觀地去面對才能創造幸福人生。
