明日方舟公招系統調整詳解 斯卡蒂加入 近衛小車出貨率有望提升

2019-12-14 133條評論 2,703次閱讀 583人點贊

明日方舟官方近日發布了即將對公招系統進行調整的公告，對於這次的系統調整的內容，有很多地方非常值得具體分析和說明，來看看這次的公招系統調整的詳細解讀吧。

這次的改動分成兩個方面，首先是對於公招詞條的改動，移除了男性和女性幹員，先說這個改動帶來的問題，男性和女性幹員確實看起來比較多餘，但女性幹員移除了不錯，男性幹員的詞條移除了毫無疑問是重大損失。

原因在於首先玩家以後真的很困難拿到角峰了，之前男性+重裝出角峰的概率奇高，這下直接GG了，另外就是既然官方肯定會無限拓展公招系統，那意味著送葬人等男性幹員未來也會加入，而男性幹員詞條的消失，無疑是一個重大的損失。

第二個是加入了新的詞條「支援機械」，目前來看這個詞條明顯是針對兩個幹員的，就是兩輛小車，當然不排除官方未來有可能還會推出新的機械類型的幹員，畢竟是全世界最既有的鷹角，出什麼幹員你根本想像不到，當然還有個好處就是近衛小車概率大幅度提升了，不會出現卡詞條和3.5個小時還招募不到的尷尬問題。

之後是塞了一堆新的幹員，這個更新時間點可以說是恰到好處吧，包括六星近衛斯卡蒂、五星夜魔、四星獵蜂、三星月見夜和空爆，三星這兩位出貨率會大幅度提升，避免出現一直都是老三星幹員的問題。夜魔涉及到的詞條很豐富，卡輸出、治療、減速有可能會獲取。斯卡蒂能加入卡池自然是更好的了，就是容易稀釋高資+近衛出銀老闆的概率。

最後說一下更新時間，這次的更新時間卡在了24號，也就是說和第六章主線開啟差不多在同一個時間，所以一定一定務必要保證在24號之前用掉高級詞條，別到時候你24號出了個高資詞條你沒用，到時候系統直接強制刷新了，那到時候哭都沒地方哭去。

不過招募完成後不領取倒是沒事，能保留到實名制系統更新的時候，但試圖保留等更新後刷出新幹員這操作是沒用的，不可能說你整個高資+近衛保留到更新結束後給你蹦出個斯卡蒂來，只會出銀老闆，這點要注意。
