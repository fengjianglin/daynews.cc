有誰還記得，當年那個瘋狂的轉會期，「敗家之眼」的強勢入局使得剛剛拿到LPL名額的RW戰隊成為當年轉會市場的大鱷，很多知名選手都和RW產生過聯繫，因為這支隊伍成立之初就想拉起一支全新的「銀河戰艦」，最終他們拿下了Doinb和Smlz的雙C組合，當年的Doinb被稱為LSPL皇帝，JDG的「院長」，而Smlz早已威名在外，這對雙C組合非常令人期待！

可惜的是，當年的RW短板還是非常明顯，Doinb的英雄池並不太適應版本，而頻繁使用扇子媽、璐璐這種英雄打中單也被外界成為是「混子中單」，但去年洲際賽上Doinb用一手克烈制裁了BDD之後，很多人才開始正確認識到Doinb的實力，他的中單克烈、中單鱷魚，包括中單塞恩都已經在LPL打出了自己的風采！

然而，因為沒能進入當年的S賽，賽季末初代RW分崩離析，上單Mouse退役，打野Flawless去了JDG，Smlz也去了SN，而Doinb在官宣離隊之後一度傳出要退役做直播的消息！但最終他還是接受FPX經理的邀請，加入了小鳳凰，開啟了他的職業生涯「開掛」之旅！

如今的Doinb不但成為了世界冠軍而且榮譽滿身，誰能想到在一年前他的直播間的ID還是卑微的：金咕咕想去S8，一個差點就此退役專心做一個主播的失意者！

而Smlz在轉會到SN之後，SN的成績並沒提升很多，連S賽的門檻都沒摸到，出道多年的Smlz在LPL賽場留下了很多經典畫面，論實力，馬哥也是國產AD中的翹楚！WE青訓出道的Smlz在LSPL時就被稱為是下路皇帝，當年加入M3，和Looper、Dade是隊友！後來轉會至OMG之後也曾在季後賽大殺四方！再到去年的RW，今年的SN，馬哥似乎一直在為他的S賽之路努力著，但他的S賽依然沒有到來

有的時候選擇很重要，但運氣同樣也很重要，作為初代RW的雙C，如今兩人的差距也越拉越大，可憐的賊賊目前還沒有宣布下家，小道消息稱可能會回到OMG，但OMG這幾個賽季一直在重建，卻一直在爛擺！希望明年Smlz能找到一些像樣的隊友，能進一次S賽！
