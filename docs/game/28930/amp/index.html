《荒野大鏢客OL》邊境寒冬節日活動即將上線

天天要聞
2019-12-20

今日（2019年12月20日），Rockstar公司官方公開了旗下遊戲《荒野大鏢客：救贖2》在線模式《荒野大鏢客Online》的聖誕節特別活動情報。根據官方消息，遊戲世界不僅將迎來冬日天氣，還有限時節日決戰模式、營地和私酒棚屋裝飾、禮物及更多內容。

​

假日對決模式（Holiday Showdown）

隨著聖誕節即將到來，《荒野大鏢客Online》上線了多種新遊戲模式，如黑吃黑、硝煙四起和極速佔領，而 PlayStation 4的玩家也將可遊玩獨家節慶版本的快槍決鬥、招招命中和勝者為王。只需在「在線菜單」啟動系列賽即可馬上遊玩，截至2020年1月6日。

節假日禮品箱

在2019年12月23日-25日期間遊玩《荒野大鏢客Online》，玩家們將有機會獲得特別節日禮箱，裡面滿滿當當地擠滿了各種補給品和好東西，包括：坎卜斯版本雙管霰彈槍、30發燃燒霰彈、100發霰彈槍獨頭彈、10份百里香大型獵物肉、10份野薄荷肥鳥肉、20份胡蘿蔔、20份什錦餅乾、50份糖果、10瓶優質白蘭地、10瓶特效神奇補劑、20支烈性燃燒瓶、2x溪梅、2x十大功勞。

​

額外獎勵

Rockstar官方已向在2019年12月13日或之後登錄遊玩的所有玩家發放了一系列節日禮物。玩家們仍有機會領取以下禮物，只需在2020年1月6日前遊玩《荒野大鏢客Online》並查看暫停菜單中的「福利」選項，即可兌換以下優惠券：

  • 3000 俱樂部經驗值
  • 200 RDO 遊戲幣，供您開始新職業
  • 1x 免費商販補給獎勵
  • 1x 免費自選武器部件獎勵
  • 1x 藏寶圖
  • 1x 免費私酒販麥芽漿補充獎勵
  • 以七折價格獲得一個馬廄槽位

在2019年12月31日至2020年1月6日之間遊玩《荒野大鏢客Online》即可獲得1000經驗值獎勵，四種專家職業（私酒販、賞金獵人、商販和收藏家）可各領取此獎勵一次。

玩家可繼續以比原價少5根金條的價格獲得肉鋪桌，並在購買中型貨運馬車、賞金獵人版本斯科菲爾德左輪手槍和收藏家版本蘭卡斯特連發步槍時享有七五折優惠，截至2020年1月6日。

在2019年12月24日至30日之間，收藏家也可盡情享受節日活動，只需將節日盛宴收藏系列郵寄給納扎爾夫人，即可獲取豐厚報酬。找到一枚大雁蛋、一瓶干邑白蘭地和一份百里香肥鳥肉，即可集齊此收藏系列。

而在12月31日至1月6日之間，納扎爾將要求收取新年收藏系列，包括權杖首牌、蘇格蘭威士忌和鉑金懷錶。

《荒野大鏢客：救贖2》及其在線模式《荒野大鏢客Online》現均已正式發售，登陸PS4、Xbox One和PC平台。
