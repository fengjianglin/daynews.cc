爐石酒館戰棋：三星不止血四星無發現，惡魔雞雜凝視魚人體無完膚

天天要聞
2019-12-23

各位夥伴大家好，我是扶鰩。爐石傳說在本月20號發布了全新的16.0.5號補丁，在這個補丁中，最引人關注的內容某過於，酒館戰棋玩法中的夢魘融合怪直接被移除，沒有一絲猶豫一絲顧慮……

以下是關於本次酒館戰棋改動的全部補丁內容：

波戈蒙斯塔等級由5移4；機械蛋等級由5移4；比斯巨獸等級由4移3，寒光先知由2移3；蠻魚斥候等級由4移5；夢魘融合怪告老還鄉；布萊恩·銅須掛職深造；調酒機器人上崗。

可以說，作為平衡調整的犧牲品，夢魘融合怪的付出無疑是巨大的！在之前的戰棋對決中，融合怪一直是作為對決前期的萬用打工怪來存在的，磁力，BUFF或是劇毒等屬性通通都可以施加給它。

這將會在前期創造大量的效益，而場上融合怪的數量又可以和場面強度直接掛鉤，如此一來，玩家就會對其他路線的Buff怪選擇性無視，現在好了，夢魘融合怪功成身退，直接告老還鄉了。

有些戰棋玩家可能會說，不就一個夢魘融合怪嘛，走就走，大不了用別的。對！這就是設計師初衷，這邊融合怪一退，那邊直接原地跳出大堆BUFF牌，玩家最終只能退而求其次，矮子里拔將軍！

最終的結果就是，玩家將要花費更多的鑄幣用在洗牌上，來篩選有用卡牌，來完成套路構建，而這個過程是需要時間支撐的，相應的，遊戲節奏會被大幅延緩，對應的遊戲時長也將大幅增加。

正所謂槍打出頭鳥，夢魘融合怪從一出現就是悍勇無二的存在，被設計師盯上是早晚的事，但最初的第一方案是削弱而不是移除，這次直接移除，我猜想是因為那邊聖誕到了，設計師想節省時間多多悠哉！

夢魘融合怪的融合體現在了哪裡？它的卡面說得很清楚！這張牌是元素、機械、惡魔、魚人、野獸、海盜、龍和圖騰，說直白一點，是一個非常好用的強效版萬金油，它一退休大家都虧！

那為什麼說夢魘融合怪被剔除後最受傷的是魚人流呢？有玩家可能會說：融合怪和魚人流有嘛關係？魚人流靠的不是魚人的劇毒聖盾嗎？拿掉融合怪，其他流派就再沒有劇毒了呀！

直截了當地說，劇毒本來就是魚人的特色，之前節奏快，魚人流很容易速8，可現在融合怪剔除，加血魚上調，魚人流的處境就更顯窘迫，前期過渡會很難，中期又很容易被雞雜和惡魔吊起來暴錘。

再近一步說，就是魚人流活過中期的概率很渺茫！簡單總結就是”三星止不住血，四星毫無發現，五星直接翹辮子”，有些玩家說我看有人玩魚人流很溜啊，要我說，兄弟你不能拿小概率運氣當事說！

三星牌比二星牌少，難找，成型慢，中期可能會被血入，這是大前提！但不可否認，有的玩家就是歐皇轉世，人家要什麼摸什麼，你能跟人家比嗎？要知道運氣也是實力的一部分，而且還不容小噓呢！

聽人勸吃飽飯，小編覺得，當前的惡魔流就很有前途！你看吶，現在10費回合都能弄出個20/20的小紅小綠出來，更別說中後期半百血的惡魔了。

機械本來就被惡魔克制，魚人被削弱後更容易速8了，野獸本來就很難壓制惡魔，前後一衡量，就覺得惡魔流最香，風水輪流轉，沒有一條路能走到黑，魚人流走下神壇，註定是個時間問題。
