LPL花活失敗！Uzi薇恩“還在輸出”，M神錘石神鉤成亮點

天天要聞
2019-12-07

摘要

北京時間2019年12月7號，英雄聯盟全明星賽迎來了第二天的比賽。除了Solo賽之外，最讓LPL粉絲們激動的則是LPL與LCK之間的賽區對抗賽。畢竟，關於這兩個賽區之間還是擁有著比較多的“恩怨”的，而能夠在全明星賽的舞台上看到傳奇選手之間的正式對決自然還是非常不錯的。

北京時間2019年12月7號，英雄聯盟全明星賽迎來了第二天的比賽。除了Solo賽之外，最讓LPL粉絲們激動的則是LPL與LCK之間的賽區對抗賽。畢竟，關於這兩個賽區之間還是擁有著比較多的「恩怨」的，而能夠在全明星賽的舞台上看到傳奇選手之間的正式對決自然還是非常不錯的。

另外，從雙方的選手來看，除了現役選手Uzi、Doinb、Theshy、Faker、Clid以及Peanut之外，不少LPL粉絲們所熟悉的Cat、愛蘿莉以及Madlife也是紛紛登場。另外，微笑作為LPL的教練登場也是讓人獲得了額外的驚喜。

而另外一名LCK隊內的AD位NaRaKyle則是一名韓國非常著名的英雄聯盟主播。與此同時，在英雄選擇上面來看，LCK還是拿出了一套比較正常的陣容，而我們LPL則是上演了花活，不僅Theshy拿出了卡莉斯塔，就連Doinb也是拿出了在LPL全明星賽中所使用過的努努，可以說是非常快樂的了。

不過，或許也正因為太「快樂」的緣故，LPL這邊的雙人路Theshy與Cat則遭遇到了瘋狂的針對。特別是Madlife的錘石，讓LPL這邊傷透了腦筋。與此同時，在本場比賽中，Madlife最為驚艷的操作就是與Faker盲僧之間的神級配合。在Faker使用R技能的同時，Madlife錘石的神鉤居然在空中鉤到了Theshy的卡莉斯塔。

當然了，雖然LPL這邊一直處於劣勢，但是Uzi的薇恩在擁有了一定的攻速之後也是在團戰之中操作了起來。特別是在紅色方紅Buff野區附近，Uzi薇恩的一翻操作更是讓LPL官方解說長毛大呼，「Uzi還活著，Uzi還在操作！」不過，與Uzi薇恩成名之戰不同的是，經濟裝備差距過大，Uzi也僅僅憑藉著操作在對手多人圍剿的情況之下，極限拿下了Madlife錘石的「人頭」。

而要說到這一場比賽之中最無解的，則還是LCK的主播NaRaKyle的「征服者」蛇女了。不僅「殺人書」早早疊加到了25層，幾乎在遇到LPL脆皮陣容的時候，2-3個技能就能夠收穫到「人頭了」。最後，這位主播也是拿到了本場比賽的MVP，KDA自然也是相當完美了。

M神錘石的神Q，Uzi薇恩的極限輸出，LPL被LCK所治等等，這一切的一切也是讓不少LPL粉絲們表示，「青春又回來了！」當然了，這僅僅只是一場娛樂賽，「青春」回來的同時，看著過癮、開心就完事了。

各位小夥伴們，你們覺得呢？
