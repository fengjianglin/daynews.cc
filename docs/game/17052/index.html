FGO國服54選1投票結果出爐，第一是槍兵墊底也是槍兵

2019-12-14 1,134條評論 3,543次閱讀 1530人點贊

FGO國服在1500萬下載活動開啟了四星常駐從者54選1的活動，12月5日秋田君開啟了四星選擇的投票調查，截止12月12日投票結束，一共產生了17065票，將近兩萬票的投票，也數據也差不多能和日服的appmedia相比了，那麼國服的從者選擇情況如何呢？我們一起來了解一下吧。

NO.5 黑槍呆（874票，得票率5.12%）

黑槍呆作為歐證從者，至今卡池一直都很少，所以也是玩家們經常會選擇的從者，雖然沒有充能但是也有超高的輸出，選擇黑槍呆也算是廚力和強度雙得了，黑槍呆在日服54選1的排行是第九名。

NO.4 黑呆（951票，得票率5.57%）

黑呆也一樣是歐證從者，且基本沒有過單UP卡池，想要抽到也是很難的，加上有FSN的加持，黑呆的人氣也是非常高的，即便是經過了這麼久，黑呆的人氣依然很高，黑呆在日服的54選1排行是第8名，所以還是在國服的人氣比較高一點。

NO.3 狂阿塔（967票，得票率5.67%）

狂阿塔有這麼高的人氣確實是非常意外，在日服的54選1裡面狂阿塔甚至不在前十裡面，可見國服玩家對狂阿塔的喜愛程度還是非常高的，且狂阿塔也是比較晚實裝的從者，國服又有千里眼很多卡池都不可兼得，所以估計也很多玩家在彌補沒有抽狂阿塔卡池的損失吧。

NO.2 狂蘭（1626票，得票率9.53%）

狂蘭也是實至名歸的第二名了，在綠卡時代來臨之際，大部分玩家都想要一個強力的綠卡連發光炮來過活動周回，而狂蘭就是免費的最佳選擇，不管是NEW還是補寶都非常熱門，在日服第二名的也是狂蘭。

NO.1 瓦爾基里（2083票，得票率12.21%）

第一名的是瓦爾基里，傳說中的三倍快樂，並且一樣有著非常強的連發性能，在即將到來的聖誕節無限池也有著非常優越的表現，所以無論是廚力還是強度來將，瓦爾基里都是實至名歸了，在日服的54選1瓦爾基里也是第一名。

本次四星選擇墊底的從者是芬恩，僅有12票，同樣是槍兵，有的是第一名，有的卻是倒數第一，芬恩其實也算得上是藍卡隊的槍階連發代表，不過由於劇情的塑造失敗加上風向黑，一直都無法挽救回來，卻是是蠻遺憾的。

那麼以上就是本次國服54選1的投票結果，整體來說和日服當時的投票結果還是蠻相似的，不過國服這邊黑阿塔算是黑馬了，巴御前的票數也高得出奇。
