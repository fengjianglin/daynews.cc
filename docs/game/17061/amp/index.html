陰陽師里最依賴被動的四個式神，酒吞上榜，他沒被動還不如R卡

天天要聞
4 天 ago

在陰陽師中，有各種各樣的式神。根據在團隊中的定位，可以分為：輸出式神以及輔助式神。下面就來跟大家盤點遊戲中非常依賴被動的四個式神，如果沒有被動，它們可能連R卡都不如，一起來看看吧。

第一：御饌津

如果你是鬥技場的玩家，相信對她肯定不陌生。因為獨特的造型，御饌津被大家譽為騎驢少女，雖然名聲不好聽，但她在鬥技場是非常強勢的。被動高概率控制敵人，只要你的御魂還過得去，那麼在鬥技場就可以控得別人懷疑人生。而且當你御魂完美之後，就可以提高御饌津的輸出。相信不少玩家在鬥技場，被大佬的御饌津一箭一箭給消磨死吧？

第二：茨木童子

這是一個非常熱門的輸出式神，茨木童子的被動可以把溢出的傷害轉移到別的式神上。所以他也有一拳超人這個外號，如果沒有這個被動，那麼茨木童子也僅僅只是一個高爆發的單體輸出式神，並不會有那麼高的人氣。

第三：酒吞

跟茨木童子一樣，酒吞是一個人氣非常高的式神。因為他使用技能不需要鬼火，所以在團隊中基本擔當一個副輸出的定位。酒吞的被動可以讓他平A的傷害越來越高，如果沒有這個被動，那麼酒吞就僅僅只是一個單體輸出式神，也不會有這麼高的人氣。

第四：般若

鬥技場霸主一般的存在，在鬥技場中，大多數的式神之所以強勢就是因為他們的被動。但般若卻可以封印他們的被動，這個能力讓他在鬥技場非常的吃香。而且般若的輸出能力也不低，如果沒有這個被動，般若也僅僅只是一個普通的式神。
