Letme直播：The shy和Marin有相同點，人格魅力屬性點已都拉滿

2019-12-21 759條評論 2,129次閱讀 622人點贊

提到Marin，很多人第一印象可能就是15年那個強勢的上單。在當時的解說席曾有這樣的言論：Marin的小納爾要比大納爾還要厲害，這也足以體現出他對線細節的掌控。他的發育能力同樣首屈一指，在翻閱15年的所有比賽中，幾乎每盤都是很唐突的壓對方上單的等級，領先全場，可以說Marin擁有很強的carry能力。

而在最近Letme的直播中，他的一句提醒讓我們發現了Marin的又一個優點：心態夠好。雖然說Marin具備carry能力，但是他並不是無敵的，也有被單殺的時候。通常而言選手被單殺都會比較懊惱，在反思為什麼會被擊殺，接下來的遊戲該怎麼玩，心裡壓力不自覺的就自己增加重量。而Marin的第一視角，每次被單殺都笑的非常開心，心態好的令人羨慕！

除了Marin，LPL的選手the shy也有這樣的優點。IG.the shy的性格比較內斂，在很多次採訪中都是一臉害羞的回答問題，這也是羞男的來歷。不過雖然羞男性格內斂，打法確實異常兇猛，和The shy對線過的Letme就在直播中說過：和the shy對線，真的能感受到他的打法太凶了。是的，就這樣一個「兇猛」的男人，在觀察第一視角時和Marin一樣，被單殺了之後也會笑的很開心哦~

Letme也在直播中總結到：被單殺之後會在那兒笑。能夠做出這種操作的選手是真的太少了。哪有選手被單吃了之後在那兒笑的呀。而且還是那種狂笑，這種心態真的實在太好了。我當時人就傻掉了。

The shy和Marin的共同優點都是心態夠好，即便是被單殺了，讓了遊戲的大節奏你也不會在選手的第一視角看出他們的「難過臉」，絕對是全隊笑的最開心的兩位。而令人感到「驚悚」的是，在這兩位carry型上單笑的很開心過後，對手迎來的將是更兇猛的反擊。想想讓羞男究極進化，與打法變化非常大的穩健的背後，會爆發出怎樣的戰鬥效果？

不談結局只問過程，心態這麼好或許也是兩個人個人魅力的充分體現。祝願兩位能打出更好的成績~
