DNF：體驗服活動獎勵暗改，道具屬性大削弱，但屬實影響並不大！

以下為具體的內容詳情：

眾所周知，凡是沒有正式更新上線到DNF的正式服遊戲當中的活動及獎勵，在體驗服策劃隨時都可以對其進行改動。而這不針對後續即將在12.10上線的史詩之路活動，在其商店中玩家可用材料兌換的那個史詩跨界石道具，在今天體驗服再次更新後，策劃就對其進行了一波暗改，道具的屬性來了一波大削弱。

在之前體驗服的爆料中玩家們都知道，在這個活動商店中可用400個材料兌換出的史詩跨界石，其作用為使用後，可以將1~95的史詩級以下的任意裝備轉移到賬號金庫當中，且也有體驗服的玩家經過實測，確定為使用後可以跨界除了普雷裝備外，所有任意部位，任意等級的史詩裝備（包括超界防具）。

不過就在今天，策劃在體驗服對這個獎勵臨時進行了暗改之後，其一，原本這個名為「史詩跨界石（Lv95）」的道具，改版變成了名為「超界跨界石」的道具。而道具的屬性也從上圖，變成了如下圖所示的：可以將鳥背的任意史詩裝備和超界防具轉移到賬號金庫當中。

那這暗改前和暗改後的作用對比，之前的是可以跨界除普雷以外的所有史詩裝備，而現在卻變成了僅可以跨界鳥背和超界史詩。其作用的最高上限沒變，但下限卻來了一波大削弱，不過儘管如此說實話並沒有什麼太大的影響，必定在現版本除普雷外，最值得玩家跨界的也就僅僅只有鳥背和超界，其餘的史詩根本沒有什麼特別值得跨界的價值，所以暗改後屬實影響並不大。
