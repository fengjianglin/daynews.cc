新笑傲江湖手游是肖戰粉絲專屬？林平之成了主角充值才能得令狐沖

2019-12-19 98條評論 4,322次閱讀 256人點贊

12月19日，新笑傲江湖手游正式公測了。已經提前預下載好的老喵，等待著10點開服。原以為10點一到伺服器會因為湧入太多的玩家而崩潰，沒想到10點還沒到，大約9:50左右老喵就提前進入了遊戲，此時遊戲中已經有許多玩家在練號了。完美世界這一波提前開服操作，雖然有點騷，但是沒大影響，下面老喵就帶大家一起看看這款遊戲吧。

新笑傲江湖請了當紅人氣明星肖戰代言，肖戰的高人氣給遊戲帶了很大一波熱度，許多肖戰的粉絲為了支持偶像，紛紛進入到遊戲中。新笑傲江湖也很懂粉絲的心理，特意將一個區命名為”肖煙四起”，而”肖煙四起，為戰而來”正是肖戰粉絲們的應援口號。進入遊戲後，玩家可以選擇是不是肖戰粉絲，若是選擇了就會獲得一本”戰哥秘籍”，在秘籍中可以和肖戰的虛擬形象增加親密度，還可以和肖戰互動。

在肖煙四起這個伺服器中，肖戰的粉絲特別的多，許多幫會的名字都是和肖戰有關的。說真的新笑傲江湖應該是最懂粉絲的手遊了，把粉絲經濟玩得66的。

然後再來看看遊戲本身，創建角色時有日月、華山、武當、恆山、少林五個門派，有成年和幼年兩個體形。但是作為一個MMORPG遊戲，都9102年了竟然沒有捏臉。雖然說老喵也不是特別喜歡捏臉，但是新笑傲江湖裡的人物形象對比現在市面上的一些MMORPG手游著實是差了很多，老喵選的是恆山成女，奶媽向來都應該是MMOPRG里最好看的角色，但是新笑傲江湖裡能選擇的髮型只有兩個勉強能看，臉型只有6個可選，只有眼眸和頭髮可變色。膚色不可調，整個人物都顯得黃黃的。一夢江湖的捏臉系統都2.0了呀！可能是老喵被一夢江湖慣壞了吧。

遊戲劇情方面，老喵莫名其妙的加入了一個叫影宗的組織，然後收到密信去結交林平之，然後給林平之送禮，增加和他的好感度。這劇情完全不按笑傲江湖的來啊，這個影宗是什麼鬼？我不是恆山的嗎？雖然有點莫名其妙但還是在跟著主線走。此後的主線任務都是通過影宗密函來進展的，選哪個門派沒有差別。

遊戲的畫風偏卡通，不夠寫實，但不難看。操作時打鬥感和流暢度不錯，但人物穿模比較嚴重。有點煩躁的是每次放絕學時都會出現一個全屏動畫，播放時不可操作不可跳過。打鬥時的操作性不高，基本掛機就可以通過。

48級之後恆山才能開啟菩提系武學技能，48級前的萬花全是攻擊技能沒有治療技能。奶媽48級前輸出又低還沒治療真的有點慘，不過升級很快也不用擔心沒人組隊。

遊戲中的俠客玩法，就是一個妥妥的抽卡系統，玩家可尋訪俠客來助戰增加戰力，這也是遊戲氪金點之一。笑傲江湖的男主角令狐沖則需要充值6元獲得，而林平之則是我們獲得的第一個俠客，後面的劇情很多都跟他有關，只能感嘆林平之男配上位，令狐沖因充值慘變男配~

遊戲中會有彈奏和擼貓玩法，一隻東方不敗打扮的貓可供玩家擼，彈奏玩法類似於節奏大師，算是比較新穎的地方。

老喵一直覺得楚留香（一夢江湖）是現在最好的RPG手游，以楚留香的標準來看，新笑傲江湖沒有特別吸引人的和特別突出的地方，自由度也不是很高，就是流水線一樣的MMORPG玩法。肖戰的粉絲可能會堅持玩下去，成長基金等充值的地方也沒啥充值的慾望。不知道大家玩過之後有什麼看法呢？
