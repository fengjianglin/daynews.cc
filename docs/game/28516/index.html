冬冠杯選拔賽：諾言關羽成關鍵先生，E星擊敗TS，晉級前景留懸念

2019-12-23 111條評論 3,319次閱讀 2153人點贊

王者榮耀冬季冠軍杯選拔賽第二日最後一戰迎來KPL賽區兩支戰隊的交鋒，即eStarPro對陣TS，目前TS四戰四勝，而eStar兩勝一負，想要晉級此戰必須拿下，同時eStar無心替換下Alan，那麼雙方會帶來怎樣的精彩表現呢？

陣容方面，eStar針對暖陽禁掉了李白、娜可露露、蘭陵王以及姜子牙，拿出了關羽、上官婉兒、蘇烈、裴擒虎和盤古的雙懲擊強前期陣容；而TS禁掉了公孫離、盾山、貂蟬和周瑜，選擇了項羽、曜、不知火舞、馬可波羅和張飛的高爆發組合應對！

開局兩分鐘eStar四人集結入侵野區反藍，打出一波1換3，順利拿到第一條暴君，隨後eStar中野輔聯動在中路強行越塔擊殺千世的不知火舞，4分鐘eStar四人抱團抓下，雖然拆掉一塔，但也被TS反打0換2，諾言的關羽趕到擊殺阿豆的張飛止損，6分鐘TS拔掉上路1塔後，五人集結中路開團，諾言關羽及時開大援救張飛，雙方1換1，擁有雙懲擊的eStar搶到先知主宰，並拔掉上路1塔。

十分鐘中路一波5V5拉扯戰，原本打出優勢的eStar追敵過深，被TS反打一波2換3，11分鐘黑暴團，eStar諾言的關羽、無銘的蘇烈、貓神的婉兒和無心的盤古打出一波完美COMBO，瞬間擊殺四人，無心的盤古收穫雙殺，乘勝追擊的eStar再次擊殺十三的馬可波羅完成團滅，一波推上高地拿下比賽！

此戰MVP給到本場表現搶眼的諾言關羽，由於estar的取勝，讓兩個晉級名額要等到第三個比賽日決出，晉級形勢方面，如果XQ在23號戰勝eStar，以及TS戰勝Hero的話，那麼比賽將直接結束，反之，KPL賽區的戰隊就要互決生死了！

最後附上第二個比賽日結束後的小組積分榜情況，可以看到XQ5勝1負位列第一，TS4勝1負位列第二，eStar3勝1負位列第三，總之兩個晉級名額將在最後一個比賽日揭曉，大家可以期待一下！那麼小夥伴們你覺得哪支戰隊晉級的希望大呢？歡迎留言討論！
