航海王燃燒意志：可變身的SSR，有的變身後很猛，有的變身後很撈

天天要聞
2019-12-22

諸君安好，霧夏菌報道。

目前遊戲中SSR的數量，來到了28人之多，超越了SR和R卡。那麼在這些角色中，有哪些SSR擁有變身形態？

1、艾尼路

首先迎面走來的是我們的一號選手，他的名字叫做艾尼路，綽號雷神，俗稱倉庫守門員，下水道的種子。

起初艾尼路增強了一波，雷鳥變成了屬傷技能，並且憑此強勢了一段時間，但很快就被按下去了，因為這貨不會賺錢。後續遊戲發展，陸續開放了80級天賦、100級天賦，將艾尼路完全掃入了垃圾桶中。

艾尼路有兩大硬傷，一個是變身後的技能，消耗魂玉太多（雷神9魂玉）且沒有屬傷，第二個就是80/100級天賦太過敷衍，甚至變相削弱。

2、路奇

曾幾何時，綠色象徵著安全，香蕉主要是吃的，路奇作為第一名限定卡，當時是無數玩家眼中的香餑餑。

在變身後，路奇自身的攻擊、防禦、速度提升21%，且當生命低於30%，閃避、速度再提升50%，這樣的機制，讓路奇在PVP中逞凶一時，大殺四方。可惜，後續遊戲推出了個「開局增速型」戰士，無腦地開局加速，直接將路奇這名老限定，按在了馬桶里。

3、麥哲倫

肉坦版本的賊首之一，憑藉超高的降傷機制，讓遊戲第一次降低了速度的優先順序，更名「肉坦的意志」。

麥哲倫有兩個變身形態，一個是毒龍形態，一個是巨兵形態。毒龍形態主打降傷，巨兵形態主打AOE輸出，但無論哪一種，只要有人打他，就會被掛上降傷的debuff。不過現如今，諸多角色的爆發太高，完全不給麥哲倫降傷的機會，因此這位海底監獄的老大，現在蹲廁所的時間是越來越長了。

4、戰國

海軍曾經的希望，有海軍的增益BUFF，能回血還能奶隊友，並且錘出來的小拳拳傷害量頗高。

最初其實擁有的是「世界政府」陣營的增益BUFF，就是對七武海也有效的那種，後來被砍了。變身後有著當時最高的減傷係數，基本上沒人打得動，也是肉坦版本的核心之一。

現在的話，就算是變身的戰國，也很難頂住諸多熱門限定的一套輸出，瞬間就沒了。不過刷劇情還是挺不錯的。

5、馬爾高

廢掉肉坦版本的菠蘿頭。

馬爾高有著一個全體AOE，附加的狀態是全體角色降防70%，治療降低50%，直接把肉坦們給打得自閉了。變身後的小馬哥，全部的技能附帶額外的火傷，而且攻擊暴擊提升30%，被擦到就傷，碰到就殘。

罰站結束後還有沒有明確說明的額外傷害提升，能夠瞬間融絕大部分無鎖血、保命機制的角色。

​菌說

變身後最撈的就是艾尼路，其他人都是變身後增強，只有艾尼路是變身後比之前更撈，因此很多人直接把艾尼路的變身給關掉了。這名角色基本上就是被拋棄了，因此如果幸運地抽到，盡量控制一下艾尼路對資源的攝入量。
