老秦聊遊戲，每天一篇新鮮事。

在王者榮耀中，玩家經常會碰到這種情況，進入到英雄選擇界面的時候，隊友就預先指定位置，把輸出位打野位全搶完了，只剩下輔助位能選。通常來說，這時候為了顧全大局，玩家也會默默的接受，畢竟別人手快都搶了位置，那就委屈一下這局打輔助吧！

這時候問題來了，許多小夥伴是沒有輔助銘文的，經常莫名其妙的就把輔助英雄用成了戰士，會不自覺的主動清兵，而不是等隊友一起分擔經濟。

​在不得已必須選擇輔助的情況下，許多網友都會優先選擇牛魔、張飛這一類硬輔，畢竟他們的勝率是最高的。

但是這也會帶來一個問題，那就是當輸出打不好的時候，這一局很容易就會變成逆風，而且張飛和牛魔由於缺乏傷害，哪怕輔助玩家的操作和意識再好，也難以翻盤。

那麼有沒有辦法選擇一個既可以輔助又可以有傷害的英雄呢？孤影日前就遇到了這樣的局面，並推薦大夥選擇太乙真人，並讓玩家們別著急，這個版本太乙真人幾乎可以說是萬能英雄，選他絕對錯不了！

​孤影何出此言呢？當然是有依據的。

首先太乙真人是一名前期完美的輔助，不缺傷害不缺控制，而且這個版本跟著射手一起走可以最大程度的保證射手的發育，很快就能把經濟滾起來。

孤影為大夥演示了正確的打法，本場遊戲中孤影的太乙真人開局就先幫中路清線，隨後轉到發育路幫助射手推塔。

​熟悉太乙真人的小夥伴就知道，太乙真人的前期是很煩的，坦度高傷害高控制還強，沒有兩個人根本留不下他。

而且會玩的太乙真人還會很果斷的賣自己，通過越塔控制抗傷害，擊殺敵方之後給自己大招復活。如果是經濟壓制的情況下，這種情況在路人局幾乎是無解的！

​憑藉著簡單的套路，這一局孤影的太乙真人打出了輔助的真髓。雖然沒有擊殺但是16個助攻跑酷全場，贏得了觀眾的掌聲，最後幫助團隊獲得最後的勝利。

這時候孤影還解釋為什麼太乙真人是萬能英雄。退一步說，假如這一局隊友不會玩的話，那麼太乙真人是可以通過裝備轉換變成全輸出法師的。

​全法系出裝的太乙真人傷害同樣是爆炸的，有一段時間就非常流行這個打法！從全肉轉化到半肉再到全輸出雖然有一個過程，但是太乙真人卻是一名可以完美轉換裝備的英雄。相比牛魔和張飛多了操作空間。

各位小夥伴，假如你再遇到這種只剩輔助位置的局面，不如試試太乙真人吧，不會讓你失望的哦！
