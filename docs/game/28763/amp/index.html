maple真的不撈了？LNG打野XX前期節奏起飛，德杯輕鬆擊敗OMG

天天要聞
2019-12-21

12月21日下午結束的德瑪西亞杯小組賽LNG和OMG的比賽，兩隊都是轉會期變動比較大的戰隊，所以這場比賽的考驗也比較多。這一局中前期雙方非常膠著，但是LNG還是節奏更好一些，拿到了少許優勢，最終憑藉阿修羅出色的發揮，LNG成功擊敗OMG，拿下了比賽的勝利。另外，一直被詬病的maple，這次也表現不錯。而OMG打野hacker視野分數落後XX將近60分，也引發了不小的質疑。

雙方這一局的BP如下，OMG這邊ban了奇亞娜、莫甘娜、酒桶、佐伊和鱷魚，選擇了上單劍魔，打野盲僧，中單辛德拉，下路賽娜加塔姆；LNG這邊ban了蛇女、妖姬、瑞茲、阿卡麗和奧拉夫，選擇了上單鐵男，打野蜘蛛，中單潘森，下路女槍加泰坦。

3分鐘的時候，XX蜘蛛抓下，雖然擊殺了cold的塔姆，但是duan泰坦扛塔過多也被SMLZ擊殺。5分鐘的時候，中路1v1互拼，maple潘森表現出色，把icon打成了一絲血，還打出了icon的閃現。6分鐘的時候，OMG上單curse劍魔成功單殺了聖槍哥的鐵男。

7分鐘的時候，hacker盲僧抓下，擊殺了LNG輔助duan的泰坦。10分鐘的時候，LNG四人包上推塔，curse劍魔被泰坦Q到，直接被擊殺。11分鐘，SMLZ失誤在小龍坑被LNG眾人包夾擊殺。12分鐘，LNG推進過深，OMG上單curse直接TP繞後，包夾擊殺了阿修羅的女槍。13分鐘，原本是OMG想要擊殺XX蜘蛛，但是LNG支援非常快，反而反殺了cold塔姆。16分鐘，LNG利用峽谷先鋒推掉了OMG的中路兩座外塔。

20分鐘雙方野區交戰，阿修羅找到了完美位置輸出，最終LNG團滅了OMG，拿下了第一條大龍。22分鐘，maple下路單帶的時候，被OMG上野抓死。此時OMG打野hacker盲僧視野分12分，XX實也分數75分，視野上巨大的差距也讓不少彈幕紛紛質疑hacker的盲僧。

27分鐘，OMG找到機會秒掉了XX的蜘蛛。28分鐘，OMG想要越塔開團，但是LNG追擊反打成功擊殺了OMG三人，一波結束了比賽。
