陰陽師友人帳系統崩潰事件遭噴：300個拯救平安京的飯糰無情被卡

天天要聞
2019-12-23

對於陰陽師近期友人帳系統崩潰的事情，估計不少玩家都有不少怨言，這裡帶來比較詳細的分析和說明，來看看這次事件的前因後果。

和之前的偽神活動一樣，陰陽師決戰京都活動開啟後，大家的參與熱情還算挺高，雖然嘴上說著平安京毀滅，實際上一個個要拯救和守護自己的大寶貝兒式神，但這次卻出現了玩家被官方挖完坑然後往裡面跳，還萬劫不復的操作。

主要是這次拯救的世界增加攻略次數的飯糰一共有300個，這個說多不多，說少也不少，畢竟是每個玩家都有300個飯糰，玩家基數本身就很多，所以拯救平安京的關鍵的300個飯糰足夠了。然而這次官方給飯糰的方式不是通過郵件，而是通過友人帳，這操作就很迷了。

尤其是友人帳這個系統，在最關鍵的時候直接崩潰了，不少玩家壓根打不開這個界面，導致出現了非常尷尬的想拯救平安京，但是系統不讓救，300個拯救平安京的飯糰領不出來的尷尬局面，這樣一直拖到了平安京全面淪陷遭遇了海國的侵略以及毀滅的事故。

這其實就是官方玩的一種套路和手法，故意通過可以認為操控的友人帳系統，直接弄崩，讓玩家領不了飯糰，排除一切玩家可能瞬間爆肝拯救平安京的可能性，畢竟臨時增加NPC血量或者卡敵人血條這手法太愚蠢和明顯了，所以乾脆直接整這總系統崩潰的，美其名曰玩家參與遊戲熱情過為借口，輿論效果會更好。

雖然現在這系統已經修復過來了，但也已經於事無補了，順便還有一種可能性就是部分玩家以比較下作的理由出賣自身節操來換人氣的這種行為，確實是引發了部分玩家不適，但這個不是決定性因素，說白了就是官方設定好了這一段時間肯定要毀滅一下下平安京，刺激一下玩家，等兩面佛舊世界和翻轉。
