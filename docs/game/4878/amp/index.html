DNF史上最強力的散件史詩 削血裝備很無敵？紅眼因它成土豪職業

天天要聞
2019-12-04

摘要

在地下城與勇士這款遊戲中，遊戲人物的裝備絕對算的上是玩家間的永恆話題。雖然在95版本更新之後裝備系統已經變成了千變一律的畢業模板，但是早在60-90版本中，多樣和自由的搭配才是DNF的主流。

在地下城與勇士這款遊戲中，遊戲人物的裝備絕對算的上是玩家間的永恆話題。雖然在95版本更新之後裝備系統已經變成了千變一律的畢業模板，但是早在60-90版本中，多樣和自由的搭配才是DNF的主流。而除了套裝以外，遊戲中還有著不少強力的散件裝備，而今天我們就來說說那些曾經稱霸一個版本的強力散件。

五、三神器

三神器是DNF90版本中對於「英雄王的象徵」、「羅塞塔石碑」和「波利斯的黃金杯」三件裝備的統稱。作為DNF90版本最強的三個散件之一，不少玩家為了湊齊它們甚至不惜刷掉十幾萬票。雖然在盧克團本出現後海博倫套裝取代了三神器的地位，但是很多玩家卻依舊懷念當時的「三神器」時代。

​

四、黑色雙子戒指

黑色雙子戒指是DNF70版本中從頂級副本「痛苦之村列瑟芬」掉落的遠古神器。作為「削血」流派的標準裝備，因為強力的屬性還被炒出了天價。雖然在DNF80版本出現了龍神決這樣強力的削血史詩武器，但是因為70版本裝備相對80版本要弱一些，所以黑色雙子戒指的價值也更大一些。

​

三、精鍊的遺忘魔石戒指

在DNF85版本上線之後，一套精鍊首飾無疑是火山主C的象徵。尤其是精鍊的遺忘魔石戒指，因為其能重置技能的冷卻時間而成為了跨版本神器。甚至在DNF90版本之後，很多奶媽都會在盧克團中使用飛鏢來觸發CD重置的功能。

​

二、突擊隊長的機械腿甲

在DNF85版本中，除了有別雲劍、天叢雲這樣強力的史詩裝備以外，還出現屬性不亞於它們的傳說裝備。其中85級的傳說裝備突擊隊長的機械腿甲更是因為其對劍魔和狂戰士的強力加成而成為了土豪級別的換裝裝備。也正是因為它的存在，讓DNF狂戰士這個職業有了一個不小的門檻，甚至有了入坑紅眼就必須有機械腿的說法。不過在DNF95版本更新之後，因為深淵之鱗裝備的出現，橫跨了幾個版本的機械腿也遭到了淘汰。

​

一、靈魂獵者

靈魂獵者是DNF60版本出現的55級史詩裝備。作為DNF中只能通過材料兌換的史詩裝備，卻因為增加20%攻擊力的強力屬性成為了幾個版本的神器。尤其是在60版本出現之後，靈魂獵者的獨特屬性完全可以替代當時所有的史詩防具（60版本史詩更注重防禦力），直到DNF90版本大魂鏈的出現，靈魂獵者才退出了遊戲的舞台。
