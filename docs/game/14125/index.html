裝備方面時光的必要裝備是大天使-法穿鞋-中亞，特別是前期在女神之淚出來之後，親你盡情的對對方釋放技能！有些法力值回復的時光真的秒變傑森斯坦森！女坦也是鋼鐵護肩-五速鞋-騎士之誓，這樣出裝主要是為了配合，一個是提升兩個人的生存能力，其次呢就是女坦有些五速鞋的加成和時光老頭的加速跑快快，很容易就能在下路近身對面，甚至可以不用E貼臉上去就是Q更加方便後面的追殺。

團戰的時候這個組合的變態是你沒有辦法想像的，女坦只要大招給到了C位，時光先給一個炸彈在曙光身上，曙光EQ二連暈住對方，時光再來一個Q再次暈住對面C位，打出爆炸傷害。你要是覺得時光會進場就死？中亞加大招，不要太囂張！這個組合最重要的還是時光和曙光兩個人之間技能的銜接和時間差的掌握，畢竟要是控制斷鏈了，對方一個閃現那就不能達到理想的效果了。這個組合只要用的好，威力超乎你的想像
