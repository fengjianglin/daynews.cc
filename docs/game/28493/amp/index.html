爐石傳說：娛樂卡組不配出現在職業賽場？小鳥牧表示不服

天天要聞
2019-12-23

今年爐石傳說國服最重要的比賽——黃金總決賽已經完成了小組賽的爭奪，最終晉級到烏鎮總決賽的是深海羽翼、Leaoh、誠信和獸人之殤。這些選手也確實是全年發揮最穩定的選手，深海羽翼和獸人之殤是廈門黃金公開賽大師組和公開組的冠軍，Leaoh是上海大師組的冠軍，誠信是天津大師組的冠軍，這些人的成績也充分證明了爐石並不是一個運氣遊戲。

雖然經歷了重新遞交卡組的風波，但是最終各位選手提交的卡組也算是大同小異，只有極少部分選手攜帶了具有個人特色的卡組，不過令人驚訝的是，這些卡組還有不少最終笑到了最後。

這套牌在公布之初並不受看好，但是誠信證明了這套牌不僅可以針對像元氣騎這樣的otk卡組，在對陣快攻的時候也是遊刃有餘。即使是面對劣勢對局——福報賊的時候也可以依靠場面推死對手，直播場次就是最好的證明。

另外一套表現遠超預期的就是迦拉克隆賊，小組賽中只有深海羽翼和Leaoh選擇了純種迦拉克隆賊，而這兩個人都是從勝者組3-0出線，這足以證明迦拉克隆賊的強度確實非常不錯。

深海羽翼和Leaoh的構築區別在於Leaoh選擇了前期更加穩定的血帆飛賊和刺骨，而深海羽翼選擇了上限更高的任務達人，其他構築大體上都相差不大。迦拉克隆賊是一套非常秀的卡組，前期祈求之後利用劫匪之王托瓦格爾和迦拉克隆過0費卡牌，然後利用鯊魚開始各種操作。

不過要說黃總表現最好的卡組應該還是海盜戰，黃總選手幾乎人人都能準時海盜之錨，海盜戰基本都是一把跑，可以說從不背鍋。

這個版本的海盜戰在防戰勢弱，JK薩被削的情況下大有崛起之勢，本身快攻內戰優勢就非常大，現在面對中速卡組和控制卡組的時候也絲毫不虛，看起來海盜戰真的又回來了！

今年的黃總四強名額已經決出，接下來的就是1月3日在烏鎮舉行的黃金總決賽了。小組賽階段畢竟是新版本初期，即使是職業選手提交的卡組也難免會有不周全的地方，不過職業比賽總能拋磚引玉，筆者相信這些卡組是這個版本一個好的開始，作為巨龍年卡池最深的版本，接下來一定會有更多好卡組湧現出來，大家敬請期待吧！
