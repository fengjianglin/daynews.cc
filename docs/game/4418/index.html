Theshy和Uzi晉級四強！羞男賽後中文採訪太直接

★遊戲馬蹄鐵原創

Theshy晉級Solo賽四強

今天的Solo賽LPL賽區代表Theshy和Uzi晉級到了八強，兩人的對手分別是歐洲G2的輔助以及G2的中單Caps。而且這兩場比賽全都贏了，Uzi和Theshy也是全部晉級到了明天的四強中，另外也非常有機會在決賽相遇。當然他們的對手Levi和Bwipo在今天的表現也很出色。Theshy在獲勝晉級之後的採訪才是今天的重頭戲。

Theshy阿卡麗率先單殺龍女

Theshy的阿卡麗上來就耗殘了對手，而且全程保持著高強度壓制，這也導致對手在補刀經濟上落後巨大。最終在一波血拚中被Theshy阿卡麗率先單殺，結局也是非常戲劇性。

Theshy直接拆塔獲勝

Uzi對戰Caps

Uzi先丟一血，但是憑藉縮圈時的精彩操作，完美走位最終擊殺Caps獲勝！

Uzi是2016和217年的全明星Solo冠軍，而Caps是2018年的全明星Solo冠軍，他們之間的對決也可以說是最精彩的單挑較量了。可惜早早在八強相遇，沒能打Bo3，這次是Uzi最終獲勝晉級。

Theshy的賽後採訪：「他好像很怕我」

Theshy能夠用中文進行採訪了，在多年的中國生活，讓他現在基本上可以聽懂別人說的話，但是自己說起來還稍微有點費勁。不過差不多勉強可以溝通，聽到Theshy不太熟練的中文口語，很多人也是表示粉上了。Theshy在接受採訪的時候評價對手時不說人話了，覺得不是自己表現好，而是感覺對手特別怕他。而他的阿卡麗的確是一直壓著龍女不敢補刀的。Theshy和Uzi的強大對線能力的確可以在Solo賽中明顯感受到。如果兩人能夠在決賽相遇一定會非常有趣。
