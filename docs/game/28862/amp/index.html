kpl三名老將回歸，為何只有老帥一人成功，天時地利人和缺一不可

天天要聞
2019-12-21

2019年的kpl秋季賽，可以說是帶給我們驚喜不斷，尤其是AG超玩會戰勝QGhappy的奪冠，更是讓人熱淚盈眶。當在賽後，看到老帥的眼淚時，我們也是對這名老將非常的感慨。從第一屆kpl征戰開始，經歷了巔峰，低谷，經歷了轉會轉位置，老帥一路走來也是十分的不容易，當看到他舉起銀龍杯的一剎那，也是真的為他高興。其實在這個賽季，和老帥一樣回歸的老將有三個，但是同人不同命，只有老帥一人成功，究其根本，這三點是關鍵，天時地利人和，缺一不可！

kpl該我上場

kpl三個老將回歸，只有老帥一人成功！

老帥

在本賽季開始的時候，三名老將的回歸，引起大家的關注。老帥、偽裝和阿泰，都是很長時間沒有上場的老將，這一次一起回歸，也是非常的受期待。有意思的是，這三名選手都是kpl第一代的中單，所屬的AG超玩會、estar和XQ都是當初的霸主級別的隊伍。如今歷經沉浮之後，一起回歸賽場，也是勾起大家滿滿的回憶。然而在三人之中，雖然都飽受質疑，但是只有老帥一人成功登頂，而偽裝和阿泰也是再次留下了遺憾。

天時：錦上添花遠不如雪中送炭！

偽裝

三名老將的回歸，都是經歷了轉會，其中老帥是從RW俠轉到了AG超玩會，阿泰是通過自由人的身份來到了TS，偽裝雖然飽受爭議，但是在轉會的名頭上也是用了自由人。對比發現三名選手的轉會，其實差不多，然而細細分析發現，老帥重回AG超玩會其實更重要，從陣容上來講，老帥是kpl的頂級老牌中單，在AG超玩會的隊伍中有著舉足輕重的作用。從狀態上來講，老帥是隊伍的主心骨，有了老帥之後，整個隊伍不管是在經驗還是在打法上，都有最穩的一個點，所以說，老帥之餘AG超玩會是不可或缺的存在。而偽裝去到EDGM,阿泰去到TS，更像是錦上添花，在人家原有的陣容上，很難有更好的發揮！

地利：重回老東家比插班生的境遇更好！

AG超玩會

相比於偽裝和阿泰，老帥最大的優勢就是重新回到自己的老東家，AG超玩會在收購了BA黑鳳梨的名額之後，整個隊伍是處於重建的地步，老帥的回歸是AG超玩會重建的基石，所以也是AG超玩會重組的重要一環。而偽裝和阿泰去到新隊伍，就更像是一個插班生，在人家原有的陣容里，屬於外來戶，需要花時間去消除排外的影響，還要努力的改變自己，融合到新隊伍中。這樣對比之下，老帥更像是隊伍的主人，偽裝和阿泰更像是做客的，這樣的情況下，三者的境遇也是大不相同！

人和：同樣的發揮不同的隊友，也是會有不同的結果！

阿泰

說完天時地利，接下來說說人和，這也是他們三個選手之間最大的不同。老帥回到AG超玩會之後，地位依然是老隊長，在隊伍中有一定的話語權，不需要去遷就別人，也不需要委屈自己，可以很好的表達自己的想法，和隊伍一起成長，蛻變。而偽裝到來EDGM之後，在選手中或許沒有太多的變化，但是他的位置和無痕重回，隊友們也是更習慣和無痕的配合，所以偽裝也是很少有機會上場。阿泰則是更加的艱難，他去TS完全像是一個外來戶，根本插不進去TS的隊伍體系，甚至不斷的改變自己，依然也是沒能融入隊伍，因為TS的其他人已經習慣了彼此之間的配合，再加一個外人真的不會打了！

kpl老將

隨著kpl的一屆一屆舉行，kpl的老將也是逐漸的退場，在這個秋季賽之後，估計還會有一些老將退出我們的視野。老帥的奪冠也是代表著老將們最後的堅守，我們也是希望，kpl上的老將，能繼續加油，都能圓了自己的夢想！
