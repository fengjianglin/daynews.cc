王者榮耀：亂斗模式新增6個寶石裝備，人人自帶霸體和司馬懿大招

2019-12-19 1,238條評論 3,163次閱讀 1958人點贊

在王者榮耀眾多娛樂模式中，深受玩家喜愛的要屬無限亂斗玩法，通過體驗服可以得知該模式即將迎來整改，不過當前體驗服僅新增了兩個全新BUFF以及更改了BUFF圖標，關於這些調整小編已在早前有過專題介紹，我們再來看看官方最新公布還未上線體驗服的兩個全新變化！

第一個是新裝備，未來無限亂斗除了新增「月」（重置CD）和「光」（額外閃現）兩個BUFF外，還會加入第七件裝備—無限之石，該裝備的加入意味著無限亂斗比普通模式多一個裝備格子，那麼無限之石都有哪些效果呢？

無限之石選擇豐富，包含了6個種類，如下所示：

  1. 賢者之石，被動效果與復活甲一致（特效都一樣），英雄死亡後可立即復活。
  2. 熾熱之石，被動效果與熾熱支配者一致，殘血後立刻解控並增加護盾。
  3. 獵手之石，被動效果是英雄使用寒冰懲擊後，會獲得50%的移速效果。
  4. 免疫之石，擁有唯一主動效果，召喚師釋放後可以獲得3秒霸體效果，免疫控制。
  5. 守護之石，擁有唯一主動效果，召喚師使用後可以將敵人推開，並降低50%的移速，類似姜子牙的2技能。
  6. 隕擊之石，擁有唯一主動效果，召喚師使用後可以在短暫延遲後進行遠程衝擊，類似司馬懿大招。

可以看出6個寶石的作用包含了保命、位移、免控和增速等效果，要知道現版本無限亂斗是不能出復活甲的，有了賢者之石後大大提升了持續作戰能力，而獵手之石則提升了打野英雄的抓人遊走能力，最給力的要屬隕擊之石，攜帶後人人都可以化身司馬懿快速入場收割或是拿到擊殺後快速逃離戰場，可以說新裝備的加入讓無限亂斗的可玩度提升了一個檔次。

第二個變化是地圖，未來無限亂斗地圖的場景加入了東方韻味的自然元素主題包裝，不僅如此還會隨著Buff的改變顯示不同氛圍特效，比如出現「日」BUFF後，河道兩邊會有相應圖標顯示。預計優化升級後的無限亂斗模式會在春節版本上線，感興趣的玩家可以期待一下！
