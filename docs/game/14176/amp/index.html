《攻城掠地》覺醒孫策好用嗎？覺醒孫策解析

天天要聞
1 週 ago

攻城掠地覺醒孫策好用嗎?覺醒孫策的強度怎麼樣呢?下面來看看攻城掠地覺醒孫策解析吧。

開啟孫策覺醒的前提是曹植覺醒，其覺醒消耗為18酒/次或18000寶石/次(寶石上限)。孫策主動戰法為虎躍山河，攻擊範圍5排，釋放後、孫策全體部隊獲得霸氣效果(平砍減傷+戰法減傷+斬殺幾率提升，即效果「三合一」)、同時獲得初始10點霸氣值。

(1)霸氣值增加機制

孫策每平砍一刀可增加4點霸氣值，霸氣值上限為20點，達到上限後無法繼續增加。覺醒孫策的霸氣疊滿後，其效果幾乎等於未覺醒孫策3種霸氣都疊到滿。

(2)戰術變陣

A.選擇突擊戰術時，消耗8點霸氣值，同時釋放虎嘯傷敵3排。選擇突擊戰術的效果為：每次平砍時，孫策後排部隊進行輔 助遠射，遠射範圍2排(平砍排+後一排)。孫策最後1排上陣，此時後排已經沒有部隊，所以再選擇突擊戰術時無遠射效果。遠射傷害與孫策後排剩餘的部隊數量有關，孫策後排剩餘部隊越多、遠射傷害越高。

B.選擇攻擊戰術時，消耗12點霸氣值，同時釋放虎嘯傷敵3排。選擇攻擊戰術的效果為：孫策當前排對敵方前2排施加嘲諷buff，有嘲諷buff的敵方部隊無法釋放任何技能。若敵方攜帶嘲諷buff的排陣亡，後面幾排上陣時仍然可以釋放之前沒有放出的戰法。

C.選擇防守戰術時，消耗8點霸氣值，同時釋放虎嘯傷敵3排。選擇防守戰術的效果為：給孫策的所有排(含選擇戰術的當前排)增加1次格擋效果，格擋次數不累計、只刷新。

怒氣值不足時仍然可以選擇戰術，但沒有上述的3種效果。

(3)副將情況

孫策的副將享受孫策的霸氣效果(平砍減傷+戰法減傷+斬殺幾率提升，即效果「三合一」)，但不能進行戰術變陣。

孫策作為雲垂陣副將時，釋放主動戰法仍然有霸氣效果(三合一)，也有霸氣值，但無法進行戰術變陣。

孫策作為普通副將時，沒有任何效果。

(4)陣法精通

覺醒孫策可攜帶雁行陣或方圓陣，也可以攜帶通用陣法八卦陣。

若孫策不釋放主動戰法，則沒有任何特效;孫策釋放主動戰法的排、不管系統替它選擇哪個戰術，都只有霸氣效果(平砍減傷、戰法減傷、斬殺幾率提升)，沒有其他特效。

孫策的專屬寶物有江東虎符(35-35)、霸者雙棍(50-50);其專屬御寶為霸王槍，基礎屬性為青罡劍屬性。

孫策是目前最新的覺醒武將，戰力較強，可與聯合覺醒武將爭鋒，在比武、副本中起到重要作用，但它與曹植、孟獲一樣，各類效果必須通過釋放主動戰法才能生效，這是它最明顯的弱點。因此，覺醒孫策可稱之為「聯合覺醒武將之下第一人」(一定程度上甚至可以剋制3位聯合覺醒武將)，而覺醒曹植和覺醒孟獲曾經也享有過這個稱號。
