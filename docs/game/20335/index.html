雷霆劇毒：

推薦英雄：奧恩、布隆、狗熊、蒙多、奧拉夫、老鼠、煉金、劫

推薦裝備：奧拉夫-飲血（大劍+斗篷）、羊刀（大棒+反曲弓）；煉金-鬼書（大棒+腰帶）；老鼠-極地轉職（金鏟子+腰帶）；劫-復活甲（大劍+鎖子甲）

組成羈絆：3劇毒4極地3雷霆3狂戰士2守護神1煉金

而這套陣容的是八人口成型，需要的英雄有「奧恩，布隆，狗熊，蒙多，奧拉夫，老鼠，煉金和劫。」主C是奧拉夫，副C位可以選擇煉金，老鼠，劫都是可是的。而奧拉夫剛需的兩件裝備一定要有，其中飲血和羊刀是必不可少的。

這套陣容需要一個冰錘給老鼠轉職所以在開局選秀階段我們可以優先選擇拿金鏟鏟，拿不到的話可以選擇拿大劍之類的都可以，因為這套陣容是比較吃裝備的所以在第一波小兵沒喲曝出自己想要的裝備後可以選擇在前期賣點血搶裝備。

運營的話主要還是得吃經濟，前期如果有一到兩件自己需要的裝備就可以選擇拿出最強打工仔來進行過渡，不是鎖血吃連勝就是吃連敗拿裝備。一般在前期運營就是如此，不過前期我們選擇平滑一點拿雷霆羈絆來過渡也是個不錯的選擇，畢竟前期雷霆也是蠻頂的。

因為我們所需要的核心牌都是高費卡，所以也是優先選擇上人口。存錢在上到八級之後開始D牌，這個時候就不用考慮春存錢的問題了，不過在D之前你的確保自己有40-50塊錢。不然錢過於少的話可能會出現D不出自己想要的牌而暴斃！

6光系魂：

陣容組成

暗夜獵手-薇恩（或沙漠死神-內瑟斯）、武器大師-賈克斯、暗裔劍魔-亞托克斯、眾星之子-索拉卡、牧魂人-約里克、聖槍遊俠-盧錫安、滌魂聖槍-賽娜

羈絆效果（6光2系魂）：

6光：當一名【光】友軍陣亡，其他【光】英雄會回復陣亡英雄的25%最大生命值，並獲得25%攻擊速度

2系魂：系魂英雄其中一方死亡後會變為靈魂繼續戰鬥（不可選取），只有當所有系魂英雄死亡後才會消失

核心英雄及裝備推薦：

聖槍遊俠-盧錫安：巨九（反曲弓+腰帶）、巨殺（反曲弓+大劍）、殺人劍（大劍+大劍）

滌魂聖槍-賽娜：復活甲（大劍+鎖子甲）、狂徒（腰帶+腰帶）、綠叉（反曲弓+鎖子甲）或龍牙（斗篷+斗篷）

解析：【光】在新版本里有了一定的提升，在前期中期也比以前好用很多了。

開局優先拿反曲弓或大劍，沒有反曲弓就拿大劍，反曲弓會比較難拿，因為要和玩劍刺的人搶，大劍優先給賽娜留著做復活甲，這個非常重要，賽娜的一個復活甲可以說關係著後期陣容強度。

起手拿3光，盧錫安的裝備都可以給VN用著，注意靈活搭配，5人口時看手上用的3光是什麼英雄，盡量去湊羈絆，比如用著武器大師，那麼可以湊個3狂戰士，用著沙漠死神，就可以湊個守護神。如果VN來的比較多，拿兩隻2星暗夜獵手上場在中期是非常猛的，記得手上留一下賽娜。

6人口時還沒到達6光之前，有光英雄就往上丟，光英雄越多，攻速加成越多，續航更強。特別要注意拿牧魂人，牧魂人的小鬼對光陣容的提升是非常大的。

7人口開始找盧錫安湊到6光2系魂，一星的盧錫安也是非常猛的，並且死亡變成魂魄後依然是可以吃到【光】的攻速加成，攻速越高，盧錫安技能射出的子彈就越多。

7人口鎖血後開始存錢上8人口，補充小羈絆，最好是能拿到劍聖，可以補充到2劍士2秘術。

後期注意站位，注意賽娜不要被刺客切了，只有讓賽娜好好活著才能讓盧錫安在外面瘋狂輸出。
