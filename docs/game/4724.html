WEBL微博杯小組賽第二輪結束 Ark、IFTY、Weibo積分領跑

2019-12-05 0條評論 9次閱讀 0人點贊

摘要

12月4日WEGL（微博杯）S3小組第二輪結束，B、D組交戰參賽隊伍：4AM、IFTY、Ark、AMY、OMG、VG、1246、FTG、D7G、RNG、DAE、BA、AG、EDG、NEXT、Lstars

12月4日WEGL（微博杯）S3小組第二輪結束，B、D組交戰

參賽隊伍：4AM、IFTY、Ark、AMY、OMG、VG、1246、FTG、D7G、RNG、DAE、BA、AG、EDG、NEXT、Lstars

第一局：海島圖東部岸圈形，決賽圈Ark平推，淘汰BA、D7G、FTG，VG與IFTY聯合發難Ark，VG最後7殺淘汰，本局Ark_cobra單人10個淘汰，Ark以15個淘汰拿下本場積分第一20分

第二局：海島圖極限機場圈形，上演獨狼傳奇，1246獨狼被AMY帶走，RNG獨狼小叮噹被D7G淘汰，4AM獨狼小鹿帶走AMY一人後被淘汰，D7G獨狼被Dae淘汰。最後AMY以人數優勢帶走Dae，15殺吃雞。本局MVP：AMY_Honghong

第三局：海島圖東部極端圈形，EDG淘汰IFTY，Ark摸 EDG後翼，EDG淘汰LSTARS後被Ark突襲淘汰，Ark8殺吃雞，EDG本局終級打工助Ark吃雞。本局MVP靈性摸前Ark_by。

第四局雨林圖，西北島極端圈形，EDG被ARK獨狼收掉2人，損失慘重，且因為圈形被打靶出局，OMG、VG隨後也被淘汰。Dae被RNG與LSTARS兩隊聯合淘汰，RNG隨後淘汰LSTARS獨狼，6殺吃雞。

第五局沙漠圖南部圈形，決賽圈OMG清掉LSTARS獨狼，進圈被打靶淘汰，RNG淘汰ARK，隨後鎖定EDG小木屋兩人，跳跳虎清准手雷將EDG淘汰，7殺吃雞。RNG，BACK2BACK成功。

第六局沙漠圖西部圈形，決賽圈BA先被淘汰，IFTY正面推進，開開側身放倒FTG兩人，隨後淘汰FTG最終14殺吃雞。

DAY4小組循環賽結束，Ark、iFTY、RNG位列當日小組前三。

12月5日 17：00小組循環賽第三輪，本輪小組賽結束後，將會產出將下來晉級決賽的16支隊伍，佛系小編為所有隊伍加油！
