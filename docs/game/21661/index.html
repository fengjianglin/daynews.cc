《雪鷹領主》手游全平台上線 黑鯊手機為“指尖絕學”保駕護航

2019-12-18 517條評論 2,232次閱讀 1280人點贊

騰訊2019年壓軸大作《雪鷹領主》手游於12月17日全平台上線，跨界合作特別打造「雪鷹領主福利季」，為修行者送上滿滿福利。遊戲有著獨特自創絕學，主打自由策略的PK競技體驗，黑鯊遊戲手機「生為競技」，期望給玩家更優質戰鬥體驗。雙方牽手在「雪鷹領主福利季」中開啟福利派送第一彈活動，創建角色抽專屬稱號，豐富資源助力修行者超凡成長。

遊戲全平台重磅上線雪鷹領主福利季送禮

自《雪鷹領主》手游全平台上線，「雪鷹領主福利季」拉開了活動序幕。《雪鷹領主》手游有著諸多特色內容，如今都將全面呈現在玩家面前。自創絕學玩法讓玩家可以戰鬥不拘束，自由「加特效」組合戰鬥套路。融入「分屏設計」的短視頻社交，一邊可以激情團戰，一邊可以欣賞各類短視頻內容……「雪鷹領主福利季」同步開放，12月17日至31日期間，與黑鯊遊戲手機聯手送福利。

生為競技你指尖的絕學黑鯊遊戲手機來承包

《雪鷹領主》手游主打PK競技體驗，遊戲有著多個特色職業可以讓玩家自由選擇，獨特的自創絕學，玩家可以根據自己想要的戰鬥效果來創造絕學，搭配絕學。遊戲中有著眾多的競技戰場，涵蓋個人、組隊和大規模團隊競技，自創絕學自由布局戰鬥套路，讓每一場戰鬥都大有不同。黑鯊遊戲手機，卓越性能為《雪鷹領主》手游策略個性戰鬥「保駕護航」，流暢不卡幀，戰鬥更自在。

《雪鷹領主》手游攜手黑鯊遊戲手機，「雪鷹領主福利季」送上第一波好禮。活動期間，黑鯊用戶前往「發現好遊戲」下載並創建《雪鷹領主》手游角色，就可以參與抽獎活動，獎品有可以承包你的「指尖絕學」策略戰鬥的最新潮流款黑鯊遊戲手機2Pro，以及黑鯊移動電源、黑鯊HDMI線、黑鯊散熱背夾、雙翼手柄支架等精美黑鯊周邊，還可以領取讓玩家個性獨領風騷的專屬稱號「生為競技」以及遊戲壁紙等好禮。

《雪鷹領主》手游全平台上線，「雪鷹領主福利季」同步開啟。黑鯊遊戲手機與《雪鷹領主》手游合作第一彈活動開放，黑鯊遊戲手機、專屬稱號以及豐富獎勵呈上。更多活動內容後續到來，依舊會有滿滿福利送上。
