微博杯S3：睡醒的微博戰隊有多強？被3支隊伍包圍仍拿到5個人頭

2019-12-06 0條評論 16次閱讀 0人點贊

摘要

微博杯S3最後一天的小組賽已經正式打響了，今天過後，將角逐出16支晉級總決賽的隊伍，既然是比賽，就會有輸贏，16支隊伍晉級，當然也就有16支隊伍被淘汰。

微博杯S3最後一天的小組賽已經正式打響了，今天過後，將角逐出16支晉級總決賽的隊伍，既然是比賽，就會有輸贏，16支隊伍晉級，當然也就有16支隊伍被淘汰。今天之前，已經有多支隊伍提前晉級了，當然並不是走後門什麼的，而是這些隊伍在比賽還沒打完的時候，總積分就已經超過了及格線，可謂提前獲得決賽門票。

微博戰隊就是這麼多支隊伍中的一員。我們都知道微博這支戰隊，在以前的比賽中，微博戰隊往往屬於那種比賽前期全程夢遊的狀態，隊伍四個人每一局都死得莫名其妙，戰隊每個人每次都不知道自己會死在哪個犄角旮旯。

只有在比賽最後環節，微博戰隊才會覺醒並瘋狂拿分。所以微博戰隊也就成了「不確定因素」，比賽不到最後一刻，你根本不知道這支隊伍會被淘汰還是會晉級！但這一次的微博杯就不一樣了，這一次微博戰隊提前睡醒了，而這支提前睡醒的戰隊到底有多恐怖？

首先可以看到的是微博戰隊還差一天比賽沒打就已經提前晉級了，由此可見這支隊伍的強大之處。而在今天BC組比賽的第一場，微博戰隊再一次讓我們看到了他們的強大！在第一局比賽的決賽圈，場上還有4支隊伍，分別是微博、天霸、SMG、LGE，其中除了LGE之外，其他三支隊伍都是滿編！

當時天霸佔據了安全區非常有利的房區，微博在一個反斜坡，SMG和微博做鄰居，佔據另一面的反斜坡，LGE本來有兩個人，後來被SMG幹掉一個，最後只剩下阿航一個人。當時微博的位置是所有隊伍里最差的。

首先微博想要進圈，往右走就會遇到SMG，往左走會遇到天霸！可謂「左右為難」。本來微博想趁著天霸不注意從後面進到他們的房區，但是卻被天霸發現了，最後無奈只能回撤，可就在這時SMG已經從後面壓過來了，微博想回到原來的位置都難！

但這種情況必須做出選擇，面對這種情況，微博戰隊支棱起來了。他們不懼SMG的圍追堵截，勇往直前，微博067直接打掉SMG的smallccc，緊接著小李擊倒堡壘，場面局勢瞬間逆轉！敵方死一個倒一個，run哥帶頭衝鋒，直接擊倒17shou，至此SMG只剩下一個人了，最後的六神直接被sugary用四倍猛男槍爆掉！至此SMG被淘汰。

雖然這一役微博大獲全勝，但卻被聞聲而來的天霸偷了，但即使如此，微博還是在這被三面夾擊的情況下拿到5個人頭！最後阿航出手以一敵二，戰勝天霸的二人組成功吃雞！

雖然微博這局沒吃雞，但他們的表現極為精彩，這局拿到的積分甚至比吃雞隊LGE還多！以前的微博戰隊沒睡醒的時候有點拉胯，但睡醒之後的微博戰隊，恐怖如斯！
