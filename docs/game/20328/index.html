網遊二十年，畫風變遷史，你最喜歡哪款遊戲的顏值

2019-12-17 404條評論 1,564次閱讀 492人點贊

網路遊戲在中國發展也有了二十多年，隨著科技進步以及玩家數量的逐年增多，遊戲公司也對他們的產品越來越用心。從遊戲劇情到遊戲玩法，從遊戲畫面到npc顏值，無一不比20年前有了長足的進步。

單從顏值來說，二十年前的遊戲和當今的遊戲差別就足以讓人驚掉下巴，小編這就來帶你看看二十多年來中國網遊的顏值變遷史吧。

一、小霸王時代

上個世紀八九十年代，娛樂手段匱乏，孩子們心心念念的就是一個小霸王學習機，當然大部分的孩子們為的是裡面的遊戲，那些像素遊戲甚至黑白遊戲給當時的孩子們平淡的生活增添了一絲樂趣。不知你們是否還記得當年熬夜偷偷玩超級瑪麗的日子呢？

二、模擬器時代

當時，口袋妖怪的發售引起了全球性的轟動，正是口袋妖怪的發售拯救了GB這款模擬器，而任天堂也是憑藉著口袋妖怪在遊戲界站穩了腳跟，可以說是沒有口袋妖怪就沒有後面任天堂的一系列神作。

三、PC時代

當年風靡大街小巷的傳奇對於中國玩家們來說也許是他們記憶中第一款網遊，雖然他的畫風在當今的遊戲玩家看來實在是難以忍受的，但對於當年的玩家們來說只能接觸街機小霸王的年代，傳奇可謂是他們接觸過的第一款大型網路遊戲，那炫酷的畫風和人物都讓當年的玩家們對傳奇愛不釋手。

夢幻西遊在玩家心中可謂是精緻畫風的代名詞了，不僅npc做的好看，還做的細緻，把畫質調高的時候甚至能看到玩家身上衣物的細節。除此之外，光效方面夢幻西遊也是當年的業界翹楚，曾經三刀結束變身鬼將橫掃之下讓對面化生寺魂斷當場，滅門嶺飛機起飛，寶寶應聲飛出場外，這些炫酷的場景都是當年玩家心中的神作。

劍網三作為一款十年的老遊戲，在畫風方面一直是收到玩家們好評的。之前的風景就受到了玩家們的喜愛，不少休閑玩家在玩遊戲之餘就喜歡和親友一起去一些風景勝地截圖，而那時候的劍網三已經有了光影效果。

西山居為了玩家的良好體驗，在18年對劍網3進行了畫風重置，重置之後，劍網3更是在畫風畫質方面有了長足的進步。在人物上重置版也一改往日的略帶塑料感，添加了更加具有真人感的臉型

在風景上，除了往日的的風景截圖勝地如萬花的花海、明教的三生樹等地做了光影效果和貼圖的重置，西山居還開放了和現實風景勝地九寨溝一比一還原的地圖九寨溝，無論是花草樹木的數量還是水流的光影，在西山居運用了衛星航拍等數據下顯得格外逼真，真的做到了讓玩家們不花費一金一銀在遊戲里也能旅遊。

三、3A大作

進入了科技高速發展的時代，3A大作逐漸在遊戲圈裡崛起，他們的畫質顏值甚至可以被稱為顯卡殺手。

老滾5本身是一款畫質並不高的遊戲，但是萬能的網友們卻開發出了各種各樣的mod，而上古捲軸5變成了少女捲軸5，各種各樣的mod的層出不窮，光影效果別說有多炫酷，甚至連初音未來都加入了進去。

絕地求生作為近兩年大火的槍戰類遊戲，在畫面上卻是出奇的精緻，你甚至能在飛馳的汽車上看到輪胎的花紋，拉近視角，你也能看到樹葉的紋路，無聊的玩家甚至在房子里龜縮的時候數起了地上的地磚，讓人不得不感嘆這麼精緻的槍戰遊戲，怎麼讓人做一個熱愛大地的伏地魔啊！

回顧這些從二十年前到今天的網遊，玩家們可以清晰的感覺到畫風和顏值的提高，遊戲廠家們為了玩家的良好體驗，也越來越用心的在畫面等方向研發新技術。

那麼你還記得當年被畫風所驚艷的那款遊戲是什麼嗎？
