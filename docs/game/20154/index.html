心疼路人！輸掉決賽後刺痛打巔峰賽發泄，一路連勝衝到全國前五

2019-12-18 540條評論 2,726次閱讀 852人點贊

2019年KPL秋季賽總決賽已經落下帷幕，AG超玩會戰勝了QGhappy，獲得了隊史的首個冠軍。比賽結束後，AG超玩會的月光教練繼續立flag，表示要在2020拿到雙冠，他立志要成為KPL榮譽最多的教練。而QGhappy的目標也很簡單，那就是在冬冠杯中繼續衝冠，報這次決賽的一箭之仇。

QGhappy輸掉總決賽，隊伍中有兩人被黑得最慘。一是770，很多人都認為770的手速跟不上思路，操作經常出現失誤，但很多解說都站出來為770發聲。另一個就是刺痛了，刺痛在總決賽確實沒有什麼存在感，AG五局比賽都沒ban過射手，但拿到射手的刺痛卻沒有站出來，即使是QG自家粉絲，都表示QG要考慮輪換了。

輸掉總決賽的刺痛，心情是比較鬱悶的，而且全網範圍還有很多批評他的言論，刺痛肯定不好受。開播之後，刺痛決定打巔峰賽發泄，而且沒有理會彈幕的節奏。在賽場上失意，但在巔峰賽中他要重拳出擊。比如上圖中的這局比賽，刺痛撞到了自己的隊友1dao，而刺痛也贏得了這次的正面交鋒。

刺痛打巔峰賽的時候，真的非常認真，他拿的基本都是射手，而且每一局都壓著對手打，這可是巔峰賽2200分左右的對局！在一局比賽中，刺痛的孫尚香無人可擋，他還向觀眾秀了一手孫尚香繞後，成功切死對方的C位。大家看一下刺痛的表情，他似乎真的是憋著一股氣，想要發泄出來。

100%狀態的刺痛太恐怖，使用孫尚香的這局比賽，刺痛在巔峰賽2200分的對局拿到了15.0的評分。而對面的公孫離是國服，卻全程被痛打，對方的阿軻、楊戩、廉頗也全是省前五，卻也被打得毫無招架之力。賽後隊友看到刺痛職業標誌，也紛紛給刺痛點贊。

遇到暴走狀態的刺痛，只能心疼路人了！刺痛開播之後，直接在巔峰賽頂端局打了一波8連勝，排名一下子升至全國第五。而刺痛打得並不費勁，在總決賽被嬴政掃死的孫尚香，到了巔峰賽就是「狂扁小朋友」了。

個人認為，這也體現出了巔峰賽與KPL的差距。即使是巔峰賽的頂端局，跟KPL比賽差距還是非常大，總決賽刺痛的表現有些拉跨，但回到巔峰賽，他直接變成了大魔王，巔峰賽頂端局如打人機。
