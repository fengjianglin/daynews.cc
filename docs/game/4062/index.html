最近韓服王者排位出現了一位上單選手，在短短一段時間內就衝到了韓服第十的位置，並且是以刀妹、劍姬、阿卡麗等Carry型英雄制霸韓服，並且擊敗了各路王者上單，就連DWG的Nuguri選手都被制裁過，而這位玩家就是SN的新上單Bin，並且SN官方已經在微博上宣布Bin將正式加入LPL，隨隊征戰S10賽季。

這位新上單的出現也是吸引了不少觀眾們，畢竟國產Carry上單寥寥無幾，而這位小將的橫空出世也是引起了各方人士的觀眾，DWG的Nuguri選手也多次在直播時被Bin選手制裁，Nuguri對他的評價也是非常高的，他的實力不容小覷。目前，Bin在韓服的段位為最強王者1400分左右，也是闖進了韓服前十。

並且LPL解說Cat也曾在直播時透露Bin選手曾摩擦了LPL賽區參加世界賽的三位上單，強如TheShy這般的Carry型上單也被他制裁過，如果能在賽場上有如此發揮的話，SN戰隊明年的成績絕對是突發猛進的。

但是Rank重拳出擊，比賽唯唯諾諾型選手也是不在少數，也有不少觀眾對於這個未上場的小將也是存在質疑聲的，畢竟這種Rank猛如虎，賽場菜成狗的選手數不勝數，不少觀眾都表示等選手打出成績再議論也不遲到。

LOL玩家們的想法也確實如此，畢竟選手還未登場過，排位的發揮不代表比賽也有這種狀態，只有真正在賽場上打出名氣的選手，才是觀眾們所認可的。
