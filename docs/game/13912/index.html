LOL：一級可以單刷的三個打野，不插眼的話，你buff就沒了

2019-12-12 515條評論 2,950次閱讀 1688人點贊

打野開局一般會選擇和下路一起開，因為有了雙人組的幫忙一級就不需要用懲戒了，可以省道下個buff用，這樣對面來反野了也沒那麼怕，畢竟有懲戒可以搶還可以回血。不過也有錯開下路一個人單開紅的，這樣可以讓對面打野摸不清楚你的位置，也就可以更安全的刷野，特別是在面對強勢打野的時候，這種辦法更為有效。但是也怕碰到以下這三個打野，他們經常會選擇單開buff，要是你不放眼的話，要麼送一血要麼送buff。

​第一位：盲僧~李青

盲僧不管哪個版本都是熱門的打野，特別是現在遊戲節奏加快，前期誰能夠控龍就可以為後期的勝勢奠定基礎，像盲僧這種野區強勢的英雄就特別佔優勢。盲僧都不需要出吸血裝備，就可以將野區刷個遍都不回家，因為他的有W技能的護盾和吸血，野怪打他都不怎麼傷。一級的盲僧可以學Q技能快速清野，也可以學W技能有護盾和吸血效果保持自己的一個健康血量，配合懲戒的回復可以輕鬆反掉對方的Buff，就算來人了狀態也不至於太殘。

​第二位：格雷福斯~男槍

男槍現在的勝率也很高，有了血色之刃加持的他在野區可以橫衝直撞，見到人了就打，有回復有穿甲，刷野完全不用回家。男槍打野怪的時候有一個擊退效果，所以他在打BUFF的時候特別佔優勢，因為紅藍BUFF都是近戰輸出，被擊退後就無法對男槍造成輸出了，那麼他基本就可以無傷打野。一級的時候只要布好眼位，偷一個BUFF是輕輕鬆鬆的了，有的時候甚至是直接偷半邊野區，因為有可能對面打野刷過來的時候狀態還沒他的好。

​第三位：德邦總管~趙信

叢刃這個天賦就像是專門給德邦準備一樣的，衝過去快速三下普攻就可以直接將對方擊飛，這很大程度上提高了gank的成功率，在對拼中也是特別有優勢，快速壓制住對方的血線，所以一般來說也很少有人去反德邦的野。德邦Q技能的三下普攻都會造成額外傷害，而且被動還是回復效果，所以德邦在有懲戒的情況下也是很容易刷掉Buff的，一不留神有可能就被他偷了，而他還會躲在草里陰你。

​以上三個打野前期很強勢，刷野都不怎麼傷，很有可能就想著三BUFF開局，直接到你野區刷野了，所以說碰到了盡量還是叫上單放個眼。大家覺得還有哪些英雄是一級就很強勢可以反野的呢？歡迎在下方留言分享。
