賽高專訪LNG經理左霧：並非刻意全華 Maple不是“灣虎”

2019-12-20 352條評論 2,252次閱讀 683人點贊

「漫長」的轉會期終將會看到盡頭，伴隨著俱樂部的官宣接踵而至。各俱樂部的陣容名單也開始漸漸浮出水面。粉絲或是欣喜若狂，或是大惑不解，也有粉絲抑制不住自己的心情開始「口吐芬芳」，不同的戰隊對於轉會期有著不同的期許，自然，踐行的目標也會有所不同。

而在今年轉會期當中，LNG無疑是動作很大的一家。教練的更換再加上中野的大換血，接下來將向我們走來的是一支全新版本的「LNG」，但是在這場更迭之中，LNG的操作卻遭到了不少網友的「詬病」，甚至有人評價，今年的轉會期LNG就是在「反向補強」。

招入Maple、Xx入隊，LNG是出於怎樣的考慮？在經歷了2019年的故事之後，LNG戰隊在這次轉會期最看重的是什麼？「全華班」LNG的出世，真的是LNG的刻意為之嗎？帶著這些問題，Score賽高專訪到了LNG戰隊總經理左霧，請他談到了關於這次轉會期中，關於LNG的台前與幕後故事。

轉會期前·亟需全新面貌

對於左霧來說，2019年，是從SS到LNG的一年。

「我覺得（李寧進入）對整體隊員的信心，動力有了很大提升。我們內部也清楚，李寧來到了隊伍之後，目標肯定首先是要在成績上取得突破的，不管是冠軍啊，世界賽啊，肯定是以這個目標建立的。」同樣，左霧經理也談到，對於管理層而言，李寧帶來了「很大的幫助」，「體育運用的一些方法論，日常管理企業化的一些方法論，我覺得給整個團隊帶來了很大提升」。

2019年的LNG，夏季賽成功衝進季後賽之後，在首輪掀翻了iG戰隊，但是他們的季後賽之旅也就在此止步，1:3，夏季賽六強。對於過去的挫折，左霧經理也有自己的看法。「我覺得我們在底蘊上面還是有所不足吧，因為那個時候我們是個才……剛成立一個賽季的隊伍，只能是在原有的基礎上儘力去改良和升級，到不了一個說「更新換代」這樣升級的程度。」

於是在今年的轉會期，有了更加充裕的時間與選擇的LNG將正式開啟他們的「改革」計劃。2020是LPL激烈競爭的一年，不過左霧經理不害怕競爭。「本身我們的隊伍目標就是『爭取在世界賽上取得好成績』，當然明年很多隊伍都會有這樣的目標。」「在這樣的情況下，我們肯定要著手解決一些之前短期內無法解決的問題。一個整個大的框架要做一個改革，把地基建得更厚，用全新的面貌去衝擊明年的比賽。」

而要打好地基，首先就需要基石。而LNG希望在這次轉會期為自己找到的基石，正是「中野」這個位置的選擇。在左霧經理的計劃中，新一代的中野需要「很強的串聯作用和主導力」，在看到了FPX去年轉會期成功的案例之後，LNG也迫切需要中野這一方面「跟上強隊的步伐」。

Xx·能指揮的男子漢

伴隨著這個目標，LNG開始了他們轉會期的抉擇。而左霧經理在打野位置上，看中了來自原TES戰隊的Xx。

「這是一個很『Man』的選手」，在左霧經理的口中，我們聽到了一個平常不太用在形容選手時候的評價。「大部分的選手其實比較年輕嘛，他們相對會「可愛」一點、「安靜」一點，或者是「淘氣」一點，但Xx是一個非常「男子漢氣概」（的人），整個LPL我覺得是沒有人能超過他的」，而這種「Man」自然有他的好處——「他能給我們隊伍帶來的領導力顯然是不一樣的。」

談到為什麼選擇Xx，左霧經理給出了他的答案。「我們隊伍還是需要一些其他的東西，比如說指揮能力，我們需要幫我們的上單分擔一些包括指揮能力、開團能力這部分的壓力，這樣能讓我們的配置更合理，發揮更好。」而在指揮和開團的方面，左霧毫不吝嗇對Xx的讚美「（這部分）Xx是LPL最頂尖的打野之一」這種能給對面帶來更大壓力的打法，是能夠為隊伍帶來更好的戰術運轉的。

Maple·他不是「灣虎」

「我們之前出現的很多中路問題，其實主要集中在Plex的溝通能力，英雄池這樣的問題，我們肯定需要解決。」

面對著這個必須要解決的目標，LNG開始在市面上尋找能填補這一缺陷的中路人選。左霧經理坦誠，「今年轉會期，中單位置上能選的選手說實話是比較少」。不過Maple並非所謂「權宜之計」，他一直在LNG的考察範圍之中。

對於Maple有很多非議，尤其是在他經歷了一整個沒有多少發揮空間的S9之後。「灣虎」，「訓練賽之王」，網上關於Maple的段子一個不少。但左霧經理並不這麼看。「大家都這麼叫他，把他評估成了一個「比較差的選手」，其實沒有到那種。我們是想看看他是「不適應LPL」還是其他什麼原因，讓他從LMS最出色的選手變成了現在這麼一個評價的選手」

在LNG經歷了許多評估、了解與測試之後，左霧經理下了一個判斷。「我們其實是覺得他是有能力在我們這裡發揮出水平來的」。

而對於今年轉會期離隊的兩位中單，左霧經理也獻上了他的祝福。

「Plex的話，其實很遺憾之前沒給他配到韓國隊友，然後這一季我們的目標又是溝通需要加強。但是我們跟Plex做了個約定，就是他一定要在WE好好打比賽，然後只要不是打LNG，我們是會為WE加油的，因為Plex選手就在那裡。我們之間的關係是非常好的，說不定有一天Plex還會回到我們這裡征戰，我們也是很看好他的潛力。」

「Fenfen的話，特別感謝他一年來的這幾次「救火」，幫助隊伍進季後賽他是立下了很大的功勞的。希望他能在LGD那邊打出好的表現。然後到時候比賽場上遇到他，希望他——手下留情吧。」

全華班·並非有意為之


「我們其實沒有說想搞全華班。」

「那這次是偶然嗎？」

「對（笑）」

對於外界關於LNG「全華班」的事，左霧經理給出了他自己的解答。「我們其實是一個很包容性的國際化的團隊。未來也可能有比如包括韓國的選手、越南的選手、或者說世界上任何一個地方的選手來到我們LNG，都是有這次可能的。包括休賽期你也看到了，TSM，Doublelift有來到我們這裡訓練。」「我們一直在絞盡腦汁像怎麼匹配外援名額，但最後萬萬沒想到的是外援名額沒用完」

「但是在我們上個賽季溝通的方面的確出了很大的意外和狀況，也阻礙到了訓練的進展。所以為了訓練質量和比賽的流暢性。我們這次從教練組到選手全都是說中文的。這是根據我們的實際情況做出的調整，最後也的確就促成了現在的這個情況。」

在經歷了溝通問題的陣痛之後，LNG也在尋求自己的解決之道。或許不必太看重關於「全華班」的名頭，因為左霧經理也如此信奉：「電子競技，成績說話」。

教練·走出舒適區

在這次轉會期，陪伴LNG多年的朱開教練離開了隊伍。談及自己的這位「老同事」，左霧經理也送上了他的祝福。「朱開這麼多年，對隊伍起到了非常大的作用。當然他也……飽受批評。雖然很遺憾沒能取得更好的成績，但朱開的確是一個有才華的教練，也祝他在今後的執教生涯中能越來越好吧。」

「大家都一樣。不管是這次轉會期我們的大變動，還是整個教練組的改變，以及他個人尋求去其他地方的發展。大家在一個環境待了很久，會有很多的舒適區。或許不如大家換一個環境，去新的地方去挑戰，這對大家來說，發展都是更好的。」

而在這次的轉會期，新的教練從LMS賽區前來入隊。Cyo，原FW戰隊教練，也是LMS賽區解說當中的知名「名嘴」。在他離開教練席許久之後，將再次回到戰隊擔任教練一職。

「他（Cyo）是個專業性非常好的教練，他是個非常有計劃的教練，都是根據戰隊的實際情況去布置戰術。本身又有豐富的復盤經驗，就這麼多因素加起來的話，相對來說，是非常適合我們的選擇」再加上之前的解說經歷為他帶來了良好的團隊氣氛調節能力。在左霧經理的眼中，這對於一支嶄新起航的隊伍來說，這正是非常需要的一點。

未來·腳踏實地前進

「踏踏實實就好」

對於網上的評論，左霧經理倒也沒有迴避網友的看法。「當然有很多人覺得……我們陣容變爛了，甚至可能變爛了很多。但也有人覺得我們陣容變好了。我是覺得，評價這種東西，只有比賽結果才能夠體現。」

「德杯的話，我覺得我們先從小組出線開始吧，在這個過程中，我們會有短期的，一步步走的過程；也有圍繞我們長期目標。一步步走的這麼一個過程。」

對於這支新生的LNG來說，也許正如左霧經理所說的。路還很長，這場關於S10的長跑才剛剛打響發令槍而已。變強？或是削弱？或許要時間才能證明。但是追逐冠軍的心不應被低估，S10開戰在即，而LNG也正在蓄勢待發。
