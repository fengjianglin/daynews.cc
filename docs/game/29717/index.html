利用碎片時間刷刷刷！“暗黑血統4”推出，彌補整個系列劇情

2019-12-24 159條評論 1,325次閱讀 155人點贊

今天喳喳輝給大家帶來了暗黑血統創世紀的遊戲測評，這款暗黑血統系列最新力作最近登錄Steam，對於喜歡玩APRG遊戲的玩家來說，又多了一個全新的選擇，那麼這款遊戲到底好不好玩呢，會不會入手後發現物無所值？感興趣的各位粉絲不防跟著喳喳輝來一探究竟。

遊戲配置

喳喳輝表示12月份各個遊戲廠商都是打醬油的么？這個月發布的遊戲基本都是10G到20G大小的，是真的欺負我們玩家電腦硬碟空間不夠？這次這款暗黑血統：創世紀也是只有10GB的容量，而官方推薦的配置為8GB內存、i7-3930K處理器、GTX1060顯卡基本是常見的配置，所以玩家們基本不用擔心這款遊戲運行起來會很高配置。

測試機器

記得上次喳喳輝測試暗黑血統3的時候，在超高畫質的情況下即使是GTX1060的顯卡也只是達到60FPS（其實是被鎖幀了），所以這次即使是只有10GB大小，我也會對他表示出足夠的尊重。喳喳輝測試遊戲的電腦配置為i7-9700+GTX1660Ti，幾乎是2019年最高性價比的配置，我就不信搞不定這款遊戲！

測試結果

經過喳喳輝日日夜夜的奮戰之後，終於測試出來了，在暗黑血統：創世紀的最高畫質下，遊戲幀數穩定在110之間，作為一款APRG遊戲這個幀數結果是非常高的，除去一些Bug整個遊戲玩下來全程流暢。

遊戲畫質

如果單獨從遊戲畫質的角度來看，我個人覺得暗黑血統3和暗黑血統：創世紀其實並沒有太大的差別，背景景深效果非常細膩，很多小人物在背景中的戰鬥都顯得靈活靈現，路上建築物細節刻畫一個沒落下，還有人物的領巾、馬的盔甲也都非常的精細。從一款只有10GB大小的遊戲來看，這種畫質還是非常不錯，製作也已經非常精良了。

這款遊戲還有什麼特色？

雙角色玩法

這款暗黑血統：創世紀嚴格來說，並不屬於暗黑血統的正統系列，只是一個外傳，劇情延續的是開篇之前的事情。這次這個版本的玩法可以說是非常的簡單，沒有過多複雜的劇情和操作，打小兵也是基本處於虐殺的狀態，不僅如此這次1個玩家可以同時擁有2個玩家，分別為使用大劍近身作戰的「戰爭」以及首次登場手持雙槍的「紛爭」玩家可以自由在2個角色中自由切換，體驗不同的攻擊方式，另外他們本身的屬性技能還可以幫助玩家更快的通過解謎。

固定攻擊模式

如果玩家是為了追求比較高難度的挑戰，那麼在這次的暗黑血統：創世紀中就比較難以實現這個願望啦，2個可以自由切換攻擊玩法的角色全程基本全局遊戲處於吊打狀態，遊戲也沒有設定較為強大的BOSS存在，同時他們的攻擊模式固定，很容易就可以識別出。也因此如果是工作一族忙活了一整天的情況下，這種比較簡單的遊戲，可以有好的放鬆效果！

各種刷寶

既然是APRG又怎麼少得了收集各種元素、各種道具呢！這次的作品有大大小小十幾種的元素可以收集，包括寶箱、神怒之石、船夫金幣，種類繁多數量豐厚，打小兵、收集任務這些劇情玩法上就可以爆出。如果覺得這些沒用意思，還有一種元素叫做「生物核心」可以收集，打完BOOS後爆出的概率比較少，而這種生物核心的元素，可以提升角色自身的一些屬性效果，比如攻擊力+3、防禦能力+3，玩家可以根據生物核心的特性，給自己「創造」1個獨一無二的自己。

總結

這次的暗黑血統：創世紀屬於1個沒有什麼缺點但優點也不突出的作品，中規中矩，畢竟只有10GB大小！作為一款休閑放鬆的小遊戲，是比較值得入手的，但如果單獨購買好像並不是特別划算啦，你們覺得怎麼樣，可以下下方留言哦。
