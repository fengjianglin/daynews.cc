精彩遊戲，樂趣無窮無盡！這裡是夢幻西遊資訊集中營，我們每天為你分享不一樣的遊戲樂趣。

鑒定武器裝備是夢幻西遊中最熱門日常玩法。不少玩家玩家追求價值更高的武器裝備，不惜花重金橫掃全區未鑒定裝備來鑒定，砸進去的錢不少，但是收回來的卻寥寥無幾，真正有運氣的人沒幾個。人品爆炸的玩家，無需花太多的錢去鑒定武器，可能隨手一打造就出極品。閑話就不說那麼多，下面我們來了解一下本期的遊戲資訊，讓大家看一看什麼才叫做運氣王。

白菜價的無級別裝備，玩家鑒定出150級無級別項鏈卻開心不起來。

無級別限制特效本來是所有玩家夢寐以求的裝備藍字，為了它不知有多少土豪因鑒定軍火上癮過傾家蕩產。近日某玩家鑒定出了一件150級項鏈爆出了無級別，他卻開心不起來。因為他用一個175級角色去鑒定150級項鏈，以為彈出專用界面就可以逆襲了，結果裝備上還多了一個藍字無級別，這就尷尬了。作為一個最高級的角色，鑒定出專用無級別裝備，無級別相當於擺設，一文不值。

一個元身打造出靈力最高的項鏈，特技概率起作用了。

160級項鏈元身，靈力屬性187，附加特殊概率8%，然後打造出了下面這件神器。

160級法系的夢想項鏈，初始靈力266，附加特技凝氣決，花果山專用裝備。全服找不到初靈力266且附加藍字的項鏈裝備了吧？

玩家鑒定武器連爆羅漢金鐘特技，第一把大賺，第二把血賺！

據玩家本人稱，跑寶寶任務環好不容易遇到一次高積分的，本以為拿到150級獎勵會很開心，誰知道最後獲得了一本最垃圾的150級雙短劍。玩家一怒之下就直接把它打造了，結果在戰神山鑒定出了羅漢金鐘成就，一發即中！鑒定出來的武器如上圖所示，150級雙短劍，傷害544，命中561，體魔雙加62。屬性雖然一般，但是一個特技羅漢金鐘就可以賣到幾千塊錢，大賺一筆。

還是同樣的情況，跑環獲得了一本140級長杖書，同樣的地點鑒定出同樣的特技，又來了一把羅漢金鐘武器。不過第二把武器就不簡單了，除了單加耐力20之外，傷害爆到了568，這次就血賺了！

初總傷高達743的140級羅漢金鐘武器，最終以1.5萬的價格上架了藏寶閣出售，目前仍在公示期。
