俠盜飛車5這個遊戲雖然說配置要求很高，其實是因為高配置下，我們可以看到最完美的畫面，其實配置低的話，也是可以玩的。但是你必須把配置調低點才可以，但是體驗感就下降不少。那麼最高配畫面和最低配有什麼區別呢？

第一個就是很普通的大街了，最高配的話，我們看到路上的細節很多，地上的髒亂或者掉的東西都有。但是低配的話，地上是沒什麼區別的。然後就是這個摩托車在河水旁邊的對比了，這個可太明顯了。

高配我們看到摩托車的顏色分配很細緻了，而且這個旁邊的草地都是不一樣的，低配草地是沒有的，而高配的草地具體都能看的很清楚。而河水也是一樣的，低配沒有波浪，但是高配有。遠處的場景也是一樣的，高配的細節一直是最好的。

在下雨天的時候尤其是明顯的，遠處要是有雨滴下的特效，那就是高配了，而低配的話，遠處其實和沒下雨是差不多的。在車裡面也是一樣，低配你看外面，基本就是啥都看不清楚，地上細節你也不能看到。
