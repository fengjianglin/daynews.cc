DNF：劍宗新增三次覺醒和2個被動技能，超50%提升率成女鬼劍一姐

2019-12-21 805條評論 2,055次閱讀 926人點贊

韓服女鬼劍首次引來了三次覺醒，這也是二覺多年後的又一次突破，女鬼劍四大職業首先開放了二覺，而作為原來和劍帝相抗衡的劍宗在劍帝的加強後變成了下水道一般，同樣的打造傷害不如劍帝的一半，而韓服的這一次三覺，劍宗不僅是開放了三次覺醒，還有被動的新增，按照當前的提升率來說，劍宗將成為女鬼劍一姐！

特色加強部分同步

首先是國服在1.14版本將會對劍宗這一個職業進行一個特色加強，但是目前體驗服已經再次回檔，而韓服的100級版本直接同步了國服的特色加強。

第一個是武器精通成長的調整，在30級精通後每增加1級，技能成長由原來的1%提升為2%；

第二個是新增被動技能【魔劍覺醒】，30級可以學習的被動技能，5級精通，15級上限，5級增加5%的技能攻擊力，每學習一級提升1.5%，精通時增加11%的技能攻擊力。

第三個是一覺被動【斬魂術】調整，現在1級為1.5%的技能攻擊力，而每級成長由1.5%提升為2%。100級可以直接學到20級，所以這一個提升率等級越高提升率對比改版前就越大。

每個劍宗玩家都能獲得魔劍覺醒的11%技能攻擊力，而想要獲得更多的加成就需要時裝、光環和寵物以及其他的增加技能等級的裝備提升，在不考慮韓服三覺的環境下，國服的劍宗一樣是皮甲玩家提升率最大。

三覺加強部分

女鬼劍因為有三覺下篇，所以上篇統一的傷害加成就不再多加分析，僅分析下篇的三覺內容。

第一個是95級被動技能【劍心】，可以增加基礎攻擊力和技能攻擊力，1級增加20%，每增加1級提升2%，100級加契約可以學習4級，增加26%的技能攻擊力。

除了技能攻擊力的增加，增加1次御劍術的柔化次數，並對2個小技能的形態進行了變更，升龍劍施放時突進上挑後回扯；穿雲破空劍可以直接在地面發動技能，最後一擊附加當前屬性魔劍的劍氣攻擊。除了增加柔化，這兩個小技能的作用是什麼呢？

95級的主動技能【極馭劍術 : 蒼輪劍】，拔出強化過的雷沃丁沖向敵人攻擊，敵人會進入僵直狀態後產生爆炸傷害。技能分為三段傷害，分別是刺擊傷害，強化的屬性魔劍傷害和爆炸傷害，所以技能施放需要注意和怪物的距離，要不傷害低一半！

劍宗三覺主動技能【至高軍勢】，用雷沃丁指揮次元縫隙中的魔劍軍團對目標進行攻擊，技能分為強化魔劍攻擊力、下劈衝擊波、爆炸傷害，其中爆炸傷害為15段攻擊，一旦國服卡頓就很尷尬了，技能雖然非常的華麗酷炫，奈何動圖一聲「紫薇」讓玩家忍俊不禁，只希望韓服策劃可以將其迴路重做，讓劍宗回到一覺的感覺。
