風暴英雄大狂歡，玩具冬幕節帶你鬧翻天

2019-12-21 977條評論 4,770次閱讀 2365人點贊

寒冬來臨，天氣越來越冷。在這嚴寒之際，時空樞紐並未陷入沉寂，而是沉浸在節日的狂歡之中。雖然寒冷也不能阻擋大家慶祝冬幕節的到來。冬幕節活動—玩具大作戰現在已經登陸風暴。趕快加入狂歡的大軍，在樞紐中鬧翻天吧！

本次活動將一直持續到明年的3月份，可能是暴雪考慮到這次獲得成噸的獎勵不太好收集全，所以故意增加的活動時間，為了方便玩家獲取到他們心儀的獎勵。就讓我們一起來看看這次活動都有什麼內容吧。

玩具大作戰，顧名思義是玩具的狂歡。胳肢怪眼看著去年抱抱熊縫合怪搶走了所有風頭，吃了不少的醋。這一次趁著小主人上床休息，提前打開了玩具箱，將裡面的玩具分享給大家。

翻動玩具箱，最上面的便是兩款全新的坐騎，綠色小馬坐騎和玩具小火車坐騎。可別小看這兩款坐騎，風暴的設計師在他們身上下足了功夫。首先說說綠色塑料小馬坐騎。這個坐騎就像我們小時候常玩的綠色塑料玩具，有著厚厚的底座，我們將他們放置在「戰場」上戰鬥，風暴中的小馬坐騎也是一樣，蹦蹦跳跳的在樞紐中穿梭。玩具小火車坐騎就更加的神奇了！我方各個玩家的火車可以相互連接，組成列車組，坐上火車去團戰，那狀況定會把敵人嚇個半死。

胳肢怪墨菲托斯和宇航員琪拉是兩款全新的皮膚。科技風和玩具風都在這兩款皮膚上得以體現。琪拉的伸縮劍改由五角星組成，十分q萌。莫非托斯的下半身換成了彈簧，像是整蠱玩具一般，在樞紐中冷不丁的遇見他估計會被嚇一跳。和塑料小馬一系列的皮膚也被製作了出來，包括瓦里安，迪卡德，雷諾，他們組成了偵察兵小隊，在樞紐中遇見這個強力的小隊，那麼你可要小心嘍。

這次活動的模式依舊採取的是大富翁棋盤的形式，完成任務便可以在棋盤上行走，觸發獎勵，當你第一次到達終點時，就可以將塑料小馬坐騎納入囊中了，之後你每完成一圈就可以獲得一個稀有的戰利品箱子，還在等什麼，趕快上線，參與到大富翁的遊戲之中吧。

雖然天氣嚴寒，死亡之翼可沒想乖乖的躲起來「冬眠」，什麼樣的天氣也無法按捺他的怒火。作為全新的英雄，死亡之翼擁有極強的坦度和不俗的傷害能力，風暴中其他英雄見到他都會灰溜溜的躲開。，趕快化身死亡之翼，在冬幕節之中把樞紐鬧個底朝天吧！
