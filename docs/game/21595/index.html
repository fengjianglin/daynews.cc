職業選手評價DGZ：打了10年CF，從沒見過這麼可怕的狙擊手

2019-12-19 1,478條評論 4,763次閱讀 1330人點贊

今年的穿越火線CFS世界總決賽註定又是一屆難忘的賽事，三支中國隊全部倒在了決賽門檻之外，而來自巴西賽區的VG戰隊新王登基，相比於過去的幾屆比賽，今年的VG戰隊可以說把我們打得服服帖帖，三支中國隊全部被它一頓揍，全部零封，以往可以說是大意輕敵，今年是硬實力真打不過。

而VG戰隊中，給人印象最深刻的就是他們的狙擊手DGZ，據DGZ所說，這個ID的來源是他幾個好朋友的名字首字母，國內我們稱呼他為”大狗子”、”大公子”、”打工仔”等等，這個狙擊手在這次的比賽中給我們帶來了非常多不可思議，貢獻了非常多難以想像的鏡頭，有些鏡頭甚至被觀眾們懷疑是”透視提前槍”，但一幀一幀回放又發現沒什麼問題，最後只得感嘆一聲：年輕真好！

不說這些鏡頭，就單獨看他所對位的狙擊手，首先是小組賽對陣SV馬哲，再就是八進四對陣白鯊田豐，最後半決賽對陣AG綠豆，在單純地狙擊手發揮上，DGZ的發揮甚至讓觀眾們感受不到另外三人的存在，要知道，目前馬哲、田豐、綠豆基本上是代表了我們國內三種類型狙擊手的巔峰，結果被DGZ克製得死死的。

在巴西賽區中，DGZ本就是以使用AWM的天賦聞名，在他們國家，BD的Goken選手也是狙擊一絕，相當於咱們早期的70kg和白鯊吧！但是在這次世界賽的總決賽中，DGZ把Goken壓得死死的，Goken全場都沒怎麼發揮，而DGZ可謂是一戰成名。

這名選手特別喜歡瞬狙，有幾個鏡頭根本看不懂是怎麼擊殺的，劉大大隨便選取了兩個具有代表性的鏡頭，大家來看一下吧！

這樣的準度也得益於常年累月的勤加練習，再加上這名選手極其自信，非常相信自己的槍法，突破、殘局、斷後，幾乎無所不能，手裡的一把柯爾特比國內的COP打得還好，不管是手槍還是狙擊，在他手裡都是一把大殺器。

對此，不少職業選手在觀看直播時，都表示：打了十年穿越火線，從來沒有見過這麼可怕的狙擊手。

很多傳奇狙神70kg的鐵粉也大方承認：不得不說，比巔峰時期的70kg還要強勢，每一場比賽都會貢獻幾個驚艷的鏡頭。

更是有小夥伴評價：真乃半人半鬼，神槍第一！

要知道，全世界，能夠獲得這樣評價的，只有我國的超級英雄——燕雙鷹！

燕雙鷹極其自信，常常說：我賭你的槍里沒有子彈，我會在你開槍前讓你看到自己的腦漿。

而這個DGZ拿的就是燕雙鷹的台詞：我賭你打不中我，我會在你打中我之前一槍瞬了你！

而這名選手目前僅僅只有19歲，正處於電競的黃金期，可以預見的是，未來幾年，DGZ仍舊會給我們帶來最大的衝擊，而馬哲、田豐、綠豆這樣的選手全部被其斬於馬下，未來，又有誰能夠與之抗衡呢？
