明日方舟：煌與灰喉，新幹員強度究竟如何？最速簡評

天天要聞
2019-12-25

在明日方舟今天更新的新版本中，有三位新幹員上島了：煌、灰喉、安比爾。安比爾由於是四星，暫時就不多分析數據了，主要測評煌和灰喉。拋開廚力不說，但就數據而言，這幾位幹員的自身強度究竟如何呢？今天所長就和大家一起，簡單計算和測評一下這兩個五六星角色。

（註：下文中的數據計算為了統一標準，均按照幹員技能等級7，敵人屬性為300護甲、30法抗進行計算）

煌：六星群衛

面板數據：讓我們用煌的面板數據，來對比一下同樣是群衛的幽靈鯊和布洛卡，以及六星中的擋二近衛幹員陳sir。

煌：血2821、攻825、防415、法術抗性0；布洛卡：血2335、攻842、防366、法術抗性0；幽靈鯊：血2630、攻805、防355、法術抗性0；陳Sir：血2880、攻660、防402、法術抗性0。

通過面板數據，得出的結論是：煌的白值整體略高，血量、防禦這兩個屬性尤其領先，說明了角色擁有更強的生存能力，但攻擊力相對平庸。如果煌的技能能夠同時增加三維的屬性的話，那麼這一份的生存白值領先優勢就會進一步擴大，而攻擊力由於本身白值不高，則需要更高百分比的技能加成才行。

煌

強力擊·γ型（初始技能）：簡單粗暴地提升下一次的攻擊力，七級時需要3技力，專精三以後只需要2技力。在對單輸出的情況下，秒傷能夠達到870左右，是非常不錯的常規輸出技能。如果同時阻擋兩人，理想輸出能夠達到1740，完全阻擋三人時，秒傷突破了2600。

鏈鋸延伸模塊（精一）：二技能有個非常關鍵的屬性就是：增大了煌的攻擊範圍，但是需要暖機時間。這為煌的多人輸出創造了非常關鍵、且非常舒服的輸出環境。在計算時，我們也可以更多地看看面對多人輸出時的秒傷。

「暖機」是指未開啟技能時，需要積攢技力，可以參考白金之星的天馬視域。在暖機的環節，煌的常規輸出就是825的普攻，面對300甲敵人能夠造成525傷害，折算成秒傷就是437.5（同樣需要暖機的白金，暖機環節的秒傷只有280）。但是在80秒的暖機完成後，輸出秒傷能夠達到1100以上（白金完成暖機後秒傷為880），超越幽靈鯊的一技能攻擊力強化·γ型。

而上面的數據只是對單輸出。由於煌攻擊距離的加長，我們可以對煌的二技能接敵有著更多的期望，如果平均對敵兩人，那麼秒傷就會高達2200。二技能的數據非常爆炸，並且還能夠煌的防禦力，進一步她的提高生存能力。這個技能唯一值得顧慮的，就是長達80秒（專精三後70秒）的暖機時間，需要我們去儘早地部署煌。

沸騰爆裂（精一）：三技能有一些像星熊的「力之鋸」，並且最後有一發傷害超高的AOE物理爆發傷害。但是不一樣的地方是「逐漸升至」，而不像星熊是直接增加，並且持續時間只有十秒，遠低於力之鋸的25秒。

所長這裡著重想說一下煌的最後一擊：這應該是明日方舟目前版本的最高攻擊了（沒有其他幹員加成下）。煌自身攻擊面板是825，專三的技能加成後提高80%，達到了1485攻。最後一擊造成攻擊力400%的傷害，達到了5940點。哪怕是高甲專業戶黑，目前的最高破甲線也只有5264，低於5940。甚至煌這個還是個AOE。

天賦（嚴酷訓練）：第一天賦這裡就不細說了，主要是增強生存能力，這裡著重說一下第二天賦，很可能是此次冬季地圖的MVP。目前流出的新地圖新怪物，會給我方造成很多的「寒冷」和「凍結」屬性，而煌正是這樣一張針對牌。這裡說句題外話，大家的錫蘭大小姐也可以撿起來了，她的抵抗提升也會很有幫助。

綜合來看，煌的強度整體而言非常高，無論是技能倍率還是天賦潛力，都是非常值得培養的幹員。「打一不虧，打二血賺，打三上天」，這句口號大概就是所長的感想了。

灰喉：五星單體快狙

面板數據：灰喉作為一個單體快狙，主要的對比對象是藍毒和六星的能天使。灰喉：血1493、攻588、防152、法術抗性0；藍毒：血1230、攻610、防130、法術抗性5；能天使：血1673、攻630、防161、法術抗性0。

灰喉的面板數據並不出眾，尤其是最關鍵的攻擊力，也要低於藍毒和能天使。但是生存能力會稍強一些，比藍毒高了200血和20防。但是就單體高速快狙這個職業而言，攻擊力低是致命的，代表著角色更容易產生「拋光」。

灰喉

飛羽（初始）：這個技能目前看來是能天使一技能「衝鋒模式」的下位，但是在低練度時，卻不差能天使。同樣是專精三的話，飛羽相比衝鋒模式，連射次數少了1，並且倍率也低了5%。但是在練度不高的時候，7級的飛羽倍率有125%，衝鋒模式的倍率只有121%，反而領先了。所以這也告訴了我們灰喉的基本定位：在前中期過渡，尤其是新手期特別適合的狙擊幹員。同時，可充能也有著一些特殊的用法，在一些怪物刷新不快，能夠斷斷續續接怪的地圖，每次接怪都能打出一個三連飛羽的話，傷害還是不錯的。

迴流（精一）：這個技能對比的應該是能天使的二技能「掃射模式」。在技能開啟時，秒傷高達1860，雖然輸給了能天使，但是相差不多。全等級的攻擊倍率高於掃射模式，其實破甲線高於阿能，但是輸給了連射次數少一。

順風（天賦）：搭配上天賦，攻擊時的概率攻擊力提升，進一步提高了自身的破甲線，不會輕易「拋光」。所以當我們面對500甲，甚至600甲左右的敵人時，灰喉的輸出還能反超能天使的掃射模式。不過這裡比的不再是誰輸出高，而是誰更丟人了。

綜合來看，灰喉是一個對於新手非常友好的幹員，也有一定的後期能力。但是依舊可以理解成能天使的下位代替，雖然不是完全的徹底下位，但也相差不遠。

我是奇形怪狀研究所，專註於明日方舟、崩壞3的深度遊戲攻略。如果文章對大家還有用，就關注一下吧~
