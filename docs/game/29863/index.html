「DOTA2」VG奪冠賽後感言：敗者組1串5太爽了

2019-12-24 693條評論 4,327次閱讀 2081人點贊

超凡電競12月23日訊 隨著VG在昨天一舉奪得One Esports新加坡世界邀請賽金腰帶，VG已經拿到了他們今年的第四個國際比賽冠軍。賽後，無論是俱樂部還是隊員們都對能夠奪冠而感到興奮。

VG俱樂部：

原文如下：

突破心魔，四冠收官！

隨著EG敲出了GG，我們獲得了本次新加坡職業邀請賽冠軍！從小組賽瀕臨淘汰，到敗者組無人能擋，我們最終一串五成功舉起金腰帶。這是我們新陣容獲得的第一個國際大賽冠軍，我們用事實證明，CN DOTA NEVER DIE！

回顧本次比賽 ，有拒絕者Eurus 硬漢電魂力挽狂瀾；有@OriZz 靈動帕克秀翻全場；有@Yang_周海洋 團隊肉裝穩軍陣腳；有Xjh_Pyw 絕活小小游穿全場；還有勤勤懇懇為團隊做貢獻的MC子龍DY 與BP妙計安天下的rOtK白帆 。每一次擊殺，都是你們日夜訓練的回報；每一次勝利，都是你們齊心努力的成果。

今年我們從基輔Minor找回自信，到斯德哥爾摩Major一舉奪冠，再到震中杯Major享受金色的雨。今天我們拿到了今年第四個國際比賽冠軍，這不僅是簡單的奪冠，更是我們突破了敗者組心魔的一次升華，我們可以很開心的告訴大家，我們終於不是那個敗者組就GG的VG了！ VG奪冠！
