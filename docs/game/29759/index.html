火影忍者手游：影級大佬眼中的忍者強度排名，T0居然有五位

2019-12-24 1,021條評論 2,206次閱讀 364人點贊

《火影忍者》手游自初代的秘卷被大砍之後，老一輩「四大天王」的時代基本落幕，段位賽的忍者強度也變得相對平衡，越來越多的二番位忍者開始迎來屬於自己的時代。

​

下面是一位影級大佬對當前版本的忍者強度排名，並非權威，僅供參考，以下排名分先後：

T0：鷹佐、曉蛇、穢土三代、長門、柱間

T1：面具男、雷影、泳裝小南、千手扉間、金角銀角、百豪櫻、二代水影

T2：斑、照美冥、忍戰我愛羅、新春水門、水門

T3：奇拉比、神威卡卡西、小南、止水、仙鳴等一些無法上段位的老牌高招

​

雖然目前版本還存在鷹佐和曉蛇這兩大無解毒瘤，其實能明顯感覺到如今的決鬥場開始百花齊放了，長門、穢土三代、百豪櫻在段位賽的登場率越來越高，玩家們上分的陣容也越來越豐富，這都得益於魔方今年主抓的忍者平衡調整。

​

T0陣容中的五位忍者排名相信基本沒有人會反駁，這五位的確是時下最熱門的香餑餑，就連曾經霸道無匹的雷影都不得不退居T1。而且誰也不想到之前無人問津的穢土三代會成為PK秒人利器，而四豆長門的威名更是讓人聞之膽寒。

雖然，柱間的秘卷被削，但忍者機制毫無變動，整體強度依舊十分可觀，但比之前面四位新晉天王還是略顯疲軟。

T1陣容中，萬年國家隊替補的面具男依舊居於首位，雷影在經過版本變更之後也疲態盡顯， 不復當年威風。

​

泳裝小南的排名倒是意外的不低，不知道到時候會不會慘遭魔方毒手呢？金角銀角其實也是一直忽略的強力忍者，相信等到忍法帖結束之後，這倆兄弟將會在決鬥場大放異彩，策劃寧次製作的忍者質量就是硬。

T2陣容就沒啥好說的了，斑從來都是上忍守門員，照美冥雖然強度不錯，但機制並不出彩，排上T1略顯勉強。

​

而我們最新的高招忍戰我愛羅自發布至今，登場率可謂極低，不知道在穢土四代到來的新版本中，風影大人會不會香起來呢？這忍者手感不差，技能範圍也大么，對於忍戰我愛羅的潛力我還是極為看好的。

T3陣容中基本都是些快被版本淘汰的老高招了，最可惜的當屬奇拉比，當年制霸決鬥場整整一個春天，何其威風，可嘆現在只要新出一個抓取忍者就是奇拉比新爹，虎落平陽被犬欺，令人唏噓不已，不過奇拉比、神卡打打低端局還是非常舒服的，娛樂場中也經常能看到他們的身影。

​

所謂一代版本一代神，忍者強度變更實屬正常，如今決鬥場百花齊放，倒是一個不錯的好現象。
