姿態復盤老iG對陣老WE：微笑把我打自閉了

2019-12-04 0條評論 18次閱讀 0人點贊

摘要

LPL賽區這場舉辦的周末全明星可謂是精彩萬分，其中還有給老玩家回憶經典的老IG對戰老WE的傳奇挑戰賽，這場對決可謂是回憶青春之戰，不僅是老選手們的對決，還有遠古三大解說米勒、小蒼和小米給我們帶來精彩的講解，這場活動的舉辦可謂是心意滿滿。

LPL賽區這場舉辦的周末全明星可謂是精彩萬分，其中還有給老玩家回憶經典的老IG對戰老WE的傳奇挑戰賽，這場對決可謂是回憶青春之戰，不僅是老選手們的對決，還有遠古三大解說米勒、小蒼和小米給我們帶來精彩的講解，這場活動的舉辦可謂是心意滿滿。

對決結束後，兩場對決都發揮出色的虎牙主播姿態也是獲得了傳奇挑戰賽MVP，並且贏來了全場的掌聲。在結束為其兩天的全明星活動之旅後，姿態也是照常開啟了直播，在直播時姿態也給在直播間的觀眾們復盤了這場傳奇挑戰賽。

姿態在這次復盤中可謂是痛擊隊友毫不手軟，還特地把他們每個人的失誤都強調了一遍，並且委婉的表示自己想模仿Doinb的打法，但是野輔不是小天和劉青松，AD和上單也不是林偉翔和金貢，隨後哈哈大笑的問觀眾們：「懂我意思吧」，老IG表面兄弟石錘了。

姿態在復盤到PDD和草莓塔下被單殺時更是開心不已，還特地回放了幾次，嘲笑隊友毫不手軟。但是姿態也提到草莓的奧拉夫的確是很強的，並且賽前老IG眾人打訓練賽事還模擬過幾次對局，所以PDD被草莓奧拉夫單殺也是意料之中的事。

其中最有意思的就是15分鐘的WE下路一塔下的對決，姿態表示他的夢魘要大招飛下路時瘋狂向隊員喊到：「大微笑大微笑」，然後看見的是捲毛飛起來了，場面一度非常尷尬。

隨後姿態便孤身飛向微笑的薇恩，但是微笑反應極快薇恩E閃定將姿態定牆，姿態也直言到：「微笑這波操作真的特別帥，直接把我打自閉了。」

姿態一直非常尊敬自己的對手，但是作為隊友可謂是慘不忍睹，常常要受到他的重拳出擊，雖然如此，但是也絲毫不影響他們的感情，這兩場的對局也是精彩萬分，就讓我們一起期待下一次是否還有重演傳奇挑戰賽吧。
