當你打到第二個龍的時候，會出現龍的環境，剩下的龍都是那個環境的，比如水龍環境就會剩下的全部是水龍。

但是，小丑這個英雄是可以改變這一點的

比如，小丑可以選擇永遠不改變環境，或者你認為下一個環境並不好，那就讓他不變。

另外，小丑還可以改變龍的屬性，比如現在是水龍環境，你並不想刷水龍，你可以改變下一條龍的刷新屬性

是不是很神奇？想想就很牛逼？

不變地形

不！

不是！

完全不是！

這就是個坑貨的BUG！

以上的所有條件，在你用分身擊殺的時候都會實現
用分身擊殺的龍，不會改變地形

用分身擊殺的龍，即使是環境龍，下一條依然會改變屬性

用分身擊殺的龍，即使龍魂滿了，依然會刷新新的屬性龍，你可以無限的疊加龍屬性

但是！

用分身擊殺的龍，不會疊加龍魂！

也就是說，你即使殺滿了4條，依然不會有龍魂，坑的無以復加！

必須當真身擊殺滿了4條之後，才會疊加龍魂
所以，使用分身殺龍，輕易不要省懲戒，不要讓分身A最後一下

否則

這條龍除了加一點屬性，等於沒打

而且，4條龍之後是遠古

因為BUG，會繼續刷屬性龍

對面原本不能疊的龍魂甚至還能繼續疊

還有可能比你先拿到最終龍魂
