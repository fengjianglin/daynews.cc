爐石傳說：平衡組美女設計師直播答疑，銅須問題比光牙還要大

天天要聞
2019-12-16

在這個周末，加入爐石平衡組沒多久的Cora Georgiou開了直播，在直播中她一邊下棋一邊和觀眾們聊起了天，也就談到了一些有關酒館戰棋和新版本巨龍降臨的問題。

巨龍降臨平衡補丁

這裡的平衡補丁就是預計下周上線的那個補丁，主要是削弱薩滿的，但是截止本周，目前官方尚未完全決定好方案。

事實上，官方也希望每次都可以儘快解決問題。但平衡補丁往往需要走流程，所以有時候會看看環境變化。比如前段時間削弱的迴響，早在幾周前就確定了，不過一直憋著不能說。

官方對JK薩的定位本來就是很強的，一方面是新套路出來要讓人玩就不能太弱，另一方面他們”在給巨龍版本做（平衡）測試的時候薩滿並不強”。

結果事實上讓薩滿連續強了下來，如果換個別的職業新版本一上來太強情況又會好些……沒辦法，開發進度就是要超前很多，測試也永遠不可能達得到遊戲實際上線後的效果。

對於目前卡組整體強弱，Cora說不帶任務的JK薩比帶任務的要強，而且元氣騎確實打薩滿勝率最高，不過元氣騎並不是新卡組，所以平衡補丁還是要有。

酒館戰棋

在之前有一段時間，買入勝率最低的隨從是波戈蒙斯塔，現在哪張最差就不清楚了。

光牙改成+2/+1後就已經好了，相比之下還是銅須問題更大些。（現在的魚人流也基本上需要靠銅須來雙倍buff，才能支撐不斷發育）

館長確實很強，但印象里勝率是比拉法姆和銅須要低一點的。

鈷制衛士削成4星太傷筋動骨了，而且各個種族的主力往往都在3星。（言下之意不打算削弱）

漂浮觀察者用下來反饋還行，唯一不搭的只有二王。（惡魔噴子流其實也還可以，在二王上場之前儘可能把漂浮觀察者養大，然後雙噴）

認為玩家給澤魯斯的尊重還不夠。

奈法利安沒到要削的地步。（儘管天克喬治，而且技能非常強力，不過整體數據還沒到削弱的地步，和天梯的薩滿比差遠了）

巨殼龍作為6星隨從里唯一的buff牌，這是為了給混搭流一個繼續升本的理由，否則拿到銅須之後，瘋狂嫖buff牌就可以了。

測試過加入希爾瓦娜斯作為隨從，但是太強了。（其實可以想像一下，直接將女王放在第一位，死了之後觸發亡語，拉過來一個怪，這個越到後期就越離譜）

官方討論過要不要加入雷鳴刺喉龍，決定不加入是因為想讓各個種族的特色能有所區分，比如劇毒（作為進化選項）應該是魚人的特色；同理，毒鰭魚人雖強，但作為魚人一族也有它強的道理。

目前可收藏卡牌里的元素種族要想直接加入戰棋，還有很多機制是不適用的，如抽牌等。（不過大概率以後這些種族都會登場，只是需要做一些調整）
