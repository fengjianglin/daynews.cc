探索小樹林《卧龍吟》資源拿到手軟

2019-12-15 2,315條評論 3,895次閱讀 1364人點贊

在《卧龍吟》的主城內，有一個與世隔絕的小樹林，而這其中有多少的秘密等待你去探尋呢?古人狩獵，沒有槍炮，弓箭則是最好的武器，現在就暫時放下你在疆場上征戰的霸氣，走進這片充滿驚喜的樹林吧!

當玩家的主城等級超過20級時，你就能夠在自己的主城裡發現這片狩獵場地，每天凌晨狩獵場將會固定刷新3次狩獵次數，每次狩獵開始將會給你標配70支弓 箭。隨著VIP等級的提高會在開始狩獵的基礎上增加一鍵狩獵(花費金幣直接完成打獵)和雙倍狩獵(所得獎品加倍)。由於時間和弓箭數有限，所以快速射殺狩 獵場里的獵物能夠讓你獲得更多的獎勵哦!

狩獵場中的動物有大型的老虎以及野豬，黑熊之類，也有速度較快，體型較小的兔子，野雞等等，所以狩獵也許並不是你想像中那麼輕鬆，對於兔子這類體積小活 動迅速的狩獵對象，你更多的需要用弓箭提前預判它活動的軌跡，而對於較大型的獵物，雖然花費的弓箭會比較多，但是也更加容易命中。

更重要的是，只要射殺這 些獵物你就能夠獲得銀幣，軍令，戰功等等。對於剛剛創建自己的城邦的新手來說，每天的狩獵更是不能錯過了!
