55開買戰隊尋復出機會時，昔日隊友UZI卻在直播中錘他，多大仇？

2019-12-19 874條評論 1,958次閱讀 973人點贊

人世間最悲劇的事情是什麼，就是當你買戰隊尋求復出機會的時候，你昔日的隊友卻在直播中錘你曾經犯下的錯，這是有多大的仇呢？

距離五五開盧本偉的消失，已經有兩年了。雖然他已經不在電競圈出現，但是他的故事還在永遠流傳著。很多時候，我們在電競的直播中都經常聽到，比如無狀態和胖爹在中路的PK勝率是五五開，比如韋神率領4AM攻下OMG的房區幾率是五五開。太多太多，畢竟盧姥爺生前也是體面人，即使不在了，也有很多粉絲悼念。

不過，雖然盧姥爺如今已不在公眾的視線出線，但他對於復出直播和電競圈，還是抱著強烈的願望的，畢竟這是他心裡偉大的夢。無獨有偶，當Dota2戰隊You Know Who成立的時候，很多網友都認為這是盧姥爺組建的，而且裡面有2位冠軍選手，都是盧姥爺的好兄弟。外界認為，這是盧姥爺變相復出的信號，他將王者歸來！

有意思的是，他還真的回來了，但是很多粉絲都不知道。在近日的一場DOTA2比賽中，他的戰隊有一人臨時不能參加，於是盧姥爺便親自上陣，靠著風騷的操作，拿到了不少的人頭。最後，盧姥爺幫助自家戰隊拿到了這一輪比賽的勝利，證明了自己寶刀未老！雖然盧姥爺臨時客串這麼一把給了不少粉絲驚喜，但卻有很多網友不知道情況，因此並沒有引起太大的轟動。要是被媒體知道了，可能又要討伐一波了！

值得一說的是，盧姥爺的強勢復出，對大多數鐵粉來說，是一件挺不錯的事情。然而，他昔日的隊友Uzi，卻在直播中內涵他的過去，被網友誤認為用心險惡。具體是這樣的，從世界賽拿到冠軍回來的RNG選手Uzi，剛好是LPL的休賽期，於是他就在直播平台瘋狂補時長。當然，難得放個小長假，他肯定不會玩LOL了，於是玩起了其他遊戲放鬆。

Uzi選擇了玩絕地求生，大家都知道，這是一個吃操作的遊戲啊，Uzi不擅長，於是就連麥找了他的好朋友韋神，S5最強中單，如今PUBG項目豪門4AM戰隊的老闆。韋神帶著Uzi開啟了一段歡樂的吃雞旅途，當Uzi搜房子撿到一把神器98K的時候，他問韋神要不要，韋神對於這種燒火棍的東西，肯定是不屑一顧的，因此他回了一句不用，他要用SKS。

說實話，韋神這句話很有內涵。對於一個吃雞職業選手來說，SKS是連狙中最不受待見的，SLR比他好用百倍，更不用說超級穩定的MINI了。因此，韋神的言外之意是內涵五五開，而Uzi也很上道，直接回了他一句，說你行不行，那槍一般的人玩不來！韋神看到梗接到了，於是更起勁了，直接說放心，我也是打中單的！而Uzi就更直接了，回了句好像是哦，也是同一個時期的！

韋神和Uzi如此明目張胆的內涵五五開，讓他們的直播間彈幕瞬間滿天飛，直言他們在消費過世主播！而有的網友則認為，盧姥爺開科技的事情其實一直沒有真正的石錘，Uzi作為盧姥爺昔日的隊友，如此內涵盧姥爺，顯然是在實錘盧姥爺當年確實犯下了錯誤。不過，即使盧姥爺知道了這件事情，大概也會一笑置之吧，畢竟他和Uzi除了是昔日的隊友，還是如今的好朋友，他們還是偶爾會在一起玩的！

當55開買戰隊尋求復出機會時，昔日隊友UZI卻在直播中錘他，他們有多大仇呢？對於這件事，大家怎麼看？歡迎留下你的觀點，我們一起討論！
