總結一波休賽期LPL及LCK的一些情況：Deft可能是明年最強ADC

天天要聞
2019-12-04

摘要

LPL：1.BLG ADC小榴槤和他的輔助，DMO的GALA，包括希望老師打的都很亮眼。2.TES二

LPL：
1.BLG ADC小榴槤和他的輔助，DMO的GALA，包括希望老師打的都很亮眼。
2.TES二隊的ADC相對上一條這三位表現略差。
3.阿樂的排位打的很多，線上和團戰的表現都很強勢。
4.EDG的選手排位都挺多，jiejie、junjia和hope很勤奮，最近似乎廠長耶回歸韓服。

5.小虎確實打過幾天上路，不過最近還是在打中單位。
6.SNS的上單bin很C，十把裡面6把是刀妹，玩的也很好，刀妹人柱力。
7.韓服一位ID為juhan1的打野疑似國人，排位很猛非常C，暫不知道是青訓選手還是路人玩家。
8.lovezrr從11.19開始打排位(上次是9月30)，基本都是負戰績的情況下半個月打到了韓服700點，堪稱LPL反向上分第一人。

LCK：
1.Deft休賽期幾乎每天都在和keria雙排，排位數量讓人震驚，很可能成為明年LCK第一ADC。
2.DRX的練習生非常多，粗略計算估計超過10人。
3.最近和Chovy雙排的是他們青訓的打野Pyosik，極大概率是當年的韓服第一千珏。
4.Deft和尺帝撞車非常多，但是deft吃分的場次占多數，有一天撞了四把尺帝都輸了。
5.Clid狀態極好，雙排的人用腳玩也能躺上分那種。
6.BDD排位不太多，但是狀態也不錯。

7.Tarzan從排位來看最近狀態不怎麼好，也有可能是還在適應版本。
8.Viper很C，賽娜玩的尤其好，應該是這幾位排位的AD裡面玩的最好的。
9.韓華的中路lava這幾天排位一直在玩ADC，也和lehends雙排過，或許會有一位韓國阿P？
10.寶藍直播0-7下播7-0，可能是為了直播效果逗媽媽們開心。
11.Kiin醬狀態極好，一改夏季賽後半段的頹勢。
12.APK[剛升上LCK的隊伍]，他們的雙C排位挺下飯。
13.SKT青訓人數也非常多，SKT、GENG和DRX應該是幾支LCK隊伍里青訓最多的，前兩個隊伍佔大頭。
14.新選手中，目前來看Gumayus個人實力最強(就是星際呂布的弟弟)。
