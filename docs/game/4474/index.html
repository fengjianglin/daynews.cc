國產遊戲不需要垃圾，別再製作垃圾遊戲給國產遊戲抹黑了

2019-12-07 0條評論 16次閱讀 0人點贊

摘要

近段時間小編一直能在steam上看到一款叫做《王者英雄》的遊戲，想著現在國產遊戲正在往好的方向發展的我腦子一熱就花33塊購買了這款令我“開心愉快”的遊戲。偶，我親愛的達瓦里希，看看這令人無法言語的畫質與翔一樣的操作界面，怎麼說呢,我先吐為敬。

近段時間小編一直能在steam上看到一款叫做《王者英雄》的遊戲，想著現在國產遊戲正在往好的方向發展的我腦子一熱就花33塊購買了這款令我「開心愉快」的遊戲。

偶，我親愛的達瓦里希，看看這令人無法言語的畫質與翔一樣的操作界面，怎麼說呢,我先吐為敬。

隨後我就前往steam「快樂源泉」全部評測上觀看大家對這款遊戲的評價。

哇哦，看看這令人「羨慕」的特別差評，我都不知道製作組為什麼要賣慘討生活賣33元。

竟然還會有人購買只為差評，小編在這裡只對您說一句「牛逼!」

繼續往下看更是讓我看到了令我驚訝的超級關鍵詞！

我特么驚了啊！

這麼多數不清的關鍵詞，大佬牛皮牛皮。

小編經過截圖記錄計算所有關鍵詞加起來竟然有44612個字！

我特么裂開了啊

就算你要抄襲王者榮耀那你也起碼抄的認真一點吧，這抄的東拼西湊的都是啥玩意啊。

近年來，國產遊戲出現了大批良心製作，《神舞幻想》、《太吾繪卷》、《古劍奇譚系列》、《隱形守護者》。

它們各個都是近年來國產遊戲大有進步的體現。

《神舞幻想》從建模、貼圖、光照、粒子特效等等等等都不亞於頂尖遊戲，雖有差距但也可以感覺到製作組的良苦用心。並且遊戲劇情新穎，遊戲玩法獨特，有著自己的獨特風格尤其是《神舞幻想》的整個世界觀的構成都非常的好。

遊戲缺點就是劇情上是一大短板，總結來說製作組努力了，但還是沒有到達十分完美，但是值得誇讚。

《太吾繪卷》作為老污龜的沙盒類遊戲steam好評率有87%製作組在沒有任何盈利的情況下還連續更新了一年，就為這個小編就要瘋狂為製作組打call。

《太吾繪卷》豐富的遊戲內容包含十五個遊戲門派，並有中華上下上千種不同的功夫，並且遊戲中的NPC是會死的。

在這款遊戲中你必須要好好的去精心規劃建造、採集、經營等等等等。

遊戲還是非常值得推薦的

《古劍奇譚系列》

尤其是《古劍奇譚3》steam好評率93%，如此高的好評率足以看出《古劍奇譚3》的魅力有多大了。

遊戲雖然比不上國外的頂尖3A大作，但遊戲劇情一流，畫風寫實。

在遊戲中你可以欣賞各種古代美景，實在是十分好的遊戲體驗。

缺點是遊戲指引少，有的時候初玩者會陷入尷尬。

但劇情絕對一級棒！！！

《隱形守護者》打開了潛伏諜戰遊戲的新篇章，遊戲採用真人情景互動的方式，使得遊戲代入感極強，遊戲劇情一級棒，而玩遊戲的你總會陷入兩難的局面，並且劇情沒有抗日神劇那麼狗血，遊戲劇情真正體現了當年抗日戰爭時期的危險重重、爾虞我詐。

最後我想說一句：「庄曉曼真的太好看了」
