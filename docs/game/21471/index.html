KPL總決賽後，QGfly直播復盤比賽，網友：沒有fly，QG會輸得更慘

2019-12-19 858條評論 2,899次閱讀 1282人點贊

2019年《王者榮耀》KPL秋季總決賽完美落下帷幕，AG超玩會上演了一場決賽復仇之戰，以4:1的大比分擊敗QGhappy獲得了戰隊史上第一個聯賽總冠軍，也終結了AG無冠之殤。

一直以來，QGhappy都是《王者榮耀》中的豪門戰隊，連續拿下了KPL春季賽冠軍、秋季賽冠軍和冠軍杯冠軍，也成為了《王者榮耀》聯賽中的首個三冠王戰隊。在整個2019賽季，QGhappy也備受粉絲們看好，沒想到卻在總決賽中輸給了AG超玩會，還輸得那麼慘。

在比賽結束後，不少QG的粉絲都感到心痛，Fly也發文總結了本場比賽的，直言失敗就是真名實力不足，並且立下Flag，在下一年的比賽中會繼續努力不讓粉絲失望。2017年是QG的巔峰時期，但是2019年卻成為了QG的低谷年，冬季冠軍杯比賽就要開始，粉絲們也紛紛為QG送上祝福。

雖然QGhappy在KPL決賽中失敗了，但是Fly的表現卻非常的亮眼，對於失敗的QG來說也算得上是一種慰藉。比賽中使用曜更是打出了全隊最高的經濟，雖然最後還是失敗了，但是Fly贏得了觀眾和對手的稱讚，在一局比賽中狂鐵以一打三，更是帶領QG扳回一城。回歸直播之後，使用木蘭殘血回頭反殺王昭君，一整局把盤古打的不敢露面。

對於Fly在賽場上的表現，粉絲們也紛紛表示：

“要不是Fly，QG就被4:0了。”

“如果QG贏了比賽，那麼Fly一定就是FMVP了”

“fly大將軍牛皮！我的天，一打三，頂著復活甲和名刀，這都能切死。”

“Fly總決賽的表現可能是QG這次糟糕的總決賽之旅唯一的慰藉了。”

在比賽結束後，Fly也在第一時間回歸虎牙的直播間，和粉絲們講述比賽的心路歷程，復盤了比賽中的一些失誤，並且直言將會在接下來的冬冠會繼續努力，直指冠軍。不管是賽場上，還是直播匯中，Fly都非常努力，這就是成就了他在比賽中脫穎而出的原因。

努力的人，成績不會太差，天道酬勤，那個為了夢想全力以赴的Fly，一定會成為賽場上最亮的星。
