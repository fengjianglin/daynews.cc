KPL賽制被質疑不公平，JC背靠背高強度比賽，AG超玩會漁翁得利？

2019-12-04

AG超玩會這屆KPL已經超額完成了任務， 賽季第一場比賽，AG贏得非常艱難，不少粉絲甚至認為這賽季他們又要繼續連敗了。但AG超玩會之後的狀態越來越好，常規賽獲得了西部第一的戰績，到了季後賽，AG的狀態也是一場比一場更好，已經有強隊的風範了。

12月1日的敗者組決賽，AG超玩會成功戰勝JC，晉級到本屆KPL的總決賽，如果他們能復仇QGhappy，那便是今年的總冠軍了。

對陣JC的比賽，原以為雙方會打得很激烈，因為就在前一天，JC跟QGhappy打滿了7局，差點戰勝了QG。但這場AG超玩會卻4：0零封了對手，而且從場面上來看，JC被打得毫無還手之力，四局比賽裡面有三局都出現了一邊倒的局面。

AG超玩會戰勝對手，當然是實力的體現，但這場比賽卻打得並不好看，JC的實力明顯沒有之前那麼強。論壇發起了一項投票，有93%的網友認為當晚的JC是「殘血狀態」，沒有發揮出真正的實力。

賽後有不少網友質疑KPL的不公平賽制，認為JC是背靠背高強度比賽，會影響狀態，而AG有些勝之不武、漁翁得利的意思。

個人覺得進入到季後賽之後，比賽的強度還是很高的。Fly跟JC打完巔峰對決之後，也直接累到趴在桌子上休息。他表示比賽前一天太緊張，凌晨6點才睡著，再加上高強度的比賽，確實是沒有精力了。

JC在跟AG的比賽前一天剛剛輸給了QG，而且打滿了7局，最後只是棋差一著，無論是心理還是生理狀態都沒有調整過來，所以狀態確實比較差。

另外一點，賽程安排特別緊密的話，最多只能滿足選手的休息，而沒有充分研究對手的時間。比如在第三局的BP環節，JC禁掉了狄仁傑，直接把成吉思汗放出來了，結果一諾就直接起飛了。AG這賽季成吉思汗的勝率是8戰全勝，而狄仁傑僅僅只使用了一場。

個人觀點

在實力層面，AG超玩會還是要比JC強的，常規賽和季後賽AG都零封了對手，這足以說明問題。但JC也肯定不是100%的狀態，否則不會打得這麼狼狽。還是建議KPL在賽程的安排上，同一支隊伍不要背靠背比賽，至少要空出一天，這樣選手才有充足的時間去調整，畢竟總決賽在兩周之後才開打，賽程時間肯定是充裕的。
