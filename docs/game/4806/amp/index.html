陰陽師上線失敗劇情？玩家拿命肝SSR遭吐槽 大佬再不用負重前行

天天要聞
2019-12-04

摘要

因為陰陽師正式服的日常更新，體服也終於迎來新活動“決戰京都”。作為SP鬼切之後的又一次大型活動，因為酷似偽神獎勵的活動機制，所以在更新之後也被玩家戲稱為“偽神2.0”退治。根據陰陽師體服的更新內容，新的活動不僅比上期退治“陰陽之守”更肝，而且還設置了多條路線的獎勵。

因為陰陽師正式服的日常更新，體服也終於迎來新活動「決戰京都」。作為SP鬼切之後的又一次大型活動，因為酷似偽神獎勵的活動機制，所以在更新之後也被玩家戲稱為「偽神2.0」退治。

根據陰陽師體服的更新內容，新的活動不僅比上期退治「陰陽之守」更肝，而且還設置了多條路線的獎勵。除了玩家通過累積材料兌換SSR/SP式神以外，還有SP皮膚券、黑蛋、6星轉換券等珍貴材料。

在進入活動之後，玩家必須在五個陣營中選擇一個進行遊戲，並將所屬陣營的敵人全部消滅，才能解鎖勝利劇情。如果因為個人喜好導致陰陽師中有一個陣營防守失敗的話，那麼就會遊戲就會迎來平安京被毀滅的特殊劇情。

而之所以說平安京會被毀滅，除了遊戲活動的規則以外，在玩家的成就徽章中也新增了兩個絕版的道具，他們分別是「京都決戰敗局」和「京都決戰曙光」。而達成這個徽章條件，則是需要玩家在陰陽師新活動中獲勝或者失敗，如果全服玩家不一起努力討伐BOSS的話，那麼玩家就會獲得京都決戰敗局的徽章。

而對於這一情況，不少陰陽師玩家認為這次活動勢必又會出現大佬負重前行的言論，引起不少玩家的爭吵。不過個人認為，因為這次活動設置了失敗和勝利的特殊劇情，所以為了能夠成功的守衛平安京玩家勢必會多方面參與活動，不會出現之前負重前行的情況。

雖然這次陰陽師的偽神2.0活動確實很有新意，但是因為SP鬼切的活動才剛過，如此頻繁的速度自然也引起了不少玩家的不滿。其中一位玩家表示，最近遊戲連續多次的重肝活動不僅消磨了老玩家的耐性，更讓不少剛入坑的萌新望而卻步，加上遊戲式神的更新速度較快，頻繁的肝活動抽卡不免讓人失去耐心。

甚至有激進的玩家認為，接連兩次的大型活動不僅讓陰陽師勸退了許多萌新，更是無視上班族和學生黨的作息時間，讓這部分玩家熬夜拿自己的生命來肝想要的SSR/SP式神兌換券。不過個人認為，雖然這次的活動比較肝，但是因為年關將至，所以在時間上沒有條件的玩家完全可以放棄這次活動，畢竟新年馬上就要到來，所以遊戲後期的活動還是比較豐富的。
