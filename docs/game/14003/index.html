爐石傳說：魚人流進化選聖盾還劇毒？安德羅妮一朝選錯滿盤皆輸

2019-12-11 1,006條評論 2,046次閱讀 882人點贊

爐石傳說戰棋酒館這個模式一上線就受到很多玩家的喜歡，就連一些退游的老玩家都開始回歸。就連爐石大主播也開始不玩天梯了，就來下棋，像安德羅妮、瓦莉拉等人都是酒館裡常客了！而就在近日安德羅妮在玩酒館時，一個舉動卻迎來了玩家們的激烈討論，魚人流進化是先加聖盾好還是劇毒好？

事情是這樣的，安德羅妮在酒館中下棋，用的拉法姆走的是魚人流。而這場對局中有4位玩家走的是魚人流，其中也包括安德羅妮。前期安德羅妮走的還算順利，5星酒館時還有30點血，也摸到不少魚人！這裡就差銅須和巨殼龍了，之後碰到了亡語獵被血入18點血，這裡銅須頁剛剛來了！

這之後又來了一個玩魚人的銅須，這個玩家優先找到巨殼龍，魚人全部帶劇毒。安德羅妮此事還抱怨沒有聖盾打不了，的確魚人碰到劇毒不帶聖盾是打不了。這裡又被打掉了11點血，後面的不能再輸了，不然就GG了（之前找到3連找到巨殼龍了，但沒有劇毒和聖盾，拿了嘲諷和風怒）

後面升到6星酒館，順利拿到了巨殼龍，通過銅須獲得兩次進化，第一次是聖盾、+1+1、劇毒。安德羅妮果斷的選擇了聖盾，第二輪是嘲諷、聖盾、+1+1。這裡安德羅妮表情有點僵硬了，可能他有點後悔沒選劇毒了。這時候彈幕也在說為什麼不優先選擇劇毒呢？安德羅妮也回應彈幕說「先拿聖盾啊，拿劇毒神經病啊」，看來在安德羅妮的眼中聖盾比劇毒好用。可在下輪對館長的對局中，館長也是魚人流，而且也是個個帶劇毒，最後安德羅妮輸掉了比賽！

這裡也就引起玩家們的討論，要是安德羅妮先拿劇毒的話這局不就能吃雞了嗎？一部分玩家們後面還有一輪選擇可以賭一賭后面出聖盾，這樣聖盾、風怒、劇毒、嘲諷就齊全了。哪怕後面沒有聖盾也可以用劇毒和對面一換一啊，爭取再狗一把找到巨殼龍聖盾就來了。當然有些玩家認為拿聖盾比較好，因為自己魚人已經有風怒了，配合聖盾連打兩次，對面又劇毒也沒用。所以考慮當時對局選聖盾性價比最高，但對面的大哥屁股太大（血量）兩次攻擊都沒打死，所以才沒有贏，不然這局肯定吃雞！

面對玩家們之間的爭論，你們覺的是先選劇毒好還是聖盾好呢？小夥伴們可以在評論區留言，說說自己的看法！
