三國殺：沙摩柯李郭加入國戰，時代變了，你還在仇魏嗎

2019-12-16 3,199條評論 4,709次閱讀 2758人點贊

各位小夥伴大家好，我是”士別三日，即更刮目”的吳下阿蒙。

自從移動版三國殺上線的國戰2019以來，魏國的日子可以說是一步步江河日下，大魏核心太子與袁紹雙雙被削，而吳國新加入的凌統，蜀國的馬謖以及紛紛界限突破的菜刀們，標誌著”國戰不打魏，回家洗洗睡”的時代已經結束。

而最近新加入國戰的首位史詩武將沙摩柯，以及重置限定技的李傕郭汜，更是異軍突起。大人，時代變了，那麼就隨阿蒙來看看，這兩位會給國戰帶來怎樣的變化呢？

沙摩柯

首先是沙摩柯，俗稱老沙，四血單技能【蒺藜】：當你於一回合內使用或打出第X張牌時，你可以摸X張牌。（X為你的攻擊範圍）

需要注意的是發動技能的結算時機在使用本張牌之前，舉個例子，不能在使用第四張牌時裝備方天畫戟，需要在第三張牌時裝上，使用下一章牌時才能摸4張。

理論上可以摸1+2+3+4+5=15張牌，但極度依賴武器牌，一回合能過六七張牌就不錯了。優點是回合外也能發動，防禦力不俗，距離為1或2時最好。

即便是在身份局也尚有一戰之力，更何況是在爆發力更高的國戰模式里呢。沙摩柯的加入，給蜀國注入了新鮮血液，純粹的過牌能力比較百搭，配合一眾菜刀都能兼具過牌和防禦。搭配張飛新一任嘴臭組合，搭配皇叔散財童子，搭配小諸葛回合外白嫖無懈……

除開黃月英外，沙摩柯可以說是蜀國第二位過牌流的選手。但是由於自身史詩的緣故比較難獲取，所以出場率也不會很高。

李傕郭汜

李郭組合同樣只有一個技能，【凶算】：限定技，出牌階段，你可以棄置一張手牌並選擇與你勢力相同的一名角色，對其造成1點傷害，然後你摸三張牌。若該角色有已發動的限定技，則你選擇其一個限定技，此回合結束後該限定技可再次發動。

簡而言之，李傕郭汜的技能有兩個效果，一是對自己使用，一直重置自己的限定技，相當於國戰黃蓋的苦肉但不能多出殺，過牌能力不俗。二是對有限定技的隊友使用（也可以是自己的另一個將），重置這個限定技，然後失去凶算。

群雄一共有三個限定技，由於董卓的限定技要求為主將、出牌階段結束時發動、鎖定技，與凶算衝突，不能同時發動，所以只剩下甲魚的亂武和馬老闆的雄異。至於這兩個限定技的威力如何，想必常玩國戰的殺友們，腦海里能瞬間回憶起某一場殺局中亂武+完殺被支配的恐懼，或者被一個雄異翻盤的無奈吧。

那麼雙倍的亂武和雄異會有多快樂呢？那一天，人們終於又想起了國戰中被群雄支配的恐懼。不過兩次亂武或者雄異並不能在一回合內發動，也並不是那麼無解的。

總的來說，這兩位武將的加入也給國戰模式增加了更多的樂趣，也一定程度的影響著國戰各個勢力的強弱格局。各位小夥伴們對此有何看法呢，歡迎在評論區留言討論~~
