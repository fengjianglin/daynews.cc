新英雄秒傷2萬，躺著都能一打五！3個冷門召喚師技能調整

2019-12-19 304條評論 2,264次閱讀 388人點贊

本文原創首發於公眾號：電競怪客

hello 大家好，這裡是怪客君！關注並私信我「戰令」即可領取戰令進階卡！

蒙犽這個英雄在前兩天已經上線了體驗服，昨天怪客有心想體驗一下，但是奈何手速不夠，並沒有搶到蒙犽……

但是根據怪客在訓練營實測以及實戰中的觀察來看，蒙犽將會是一位非常強勢的射手，具體有多強？放一張怪客實戰的動圖大家可以感受一下…

這裡怪客有些網路波動，掉線了沒注意到對面蒙犽在草叢裡蹲人，因此才有了這臉探草叢的一幕。但是這傷害是不是有點過分了？動圖中是沒有進行加速處理的，實際情況就是這樣。

才1秒鐘，就把開著1技能的蔡文姬和另一位蒙犽給秒了！3級的射手直接秒掉對面滿血同等級的射輔？這種傷害正式服的哪個射手能做到？關鍵是小明的鏈子加成都沒給！這傷害實在是有些誇張了，對面的蒙犽也才3級！

不過這種不健康的傷害顯然不是策劃的本意。動圖中的蒙犽其實是利用了蒙犽一技能的一個bug，使用了連點器（相當於外掛）做到的超高傷害。因為蒙犽的一技能是在一段時間內持續開啟，一直點擊就會打出超高額的傷害，等於是手速越快傷害越高。

上方的動圖同樣沒有經過加速處理，只是開了連點器。可以看到訓練草人上顯示的秒傷已經高達2萬多，而如果是把連點器的速度往上提，還能更高！直接把草人秒殺了。而正常手動使用一技能的秒傷僅為2000多。

不過這一bug在最新的體驗服更新中已經修復了，目前連續點擊一技能並不能連續觸發一技能的傷害。

好了，說完了蒙犽的技能bug，再來說說體驗服的最新更新吧。在昨天的體驗服更新中，有3個召喚師技能的效果得到了調整。

干擾：

增加效果：可以對已方機關使用，使其無敵，持續5秒；60秒內疊加效果只有1秒的有效時間

怪客解析：以往的干擾只能對敵方機關（防禦塔、水晶）生效，但此次調整讓這個技能可以對己方的機關生效，那麼在面對敵方推進時，則多了一項防禦手段，給這個技能增添了防禦上的戰略意義。

狂暴：

舊效果：60秒CD：增加攻擊速度60%，並增加物理攻擊力10%，持續5秒

新效果：60秒CD：增加10%傷害並增加25%韌性、30%的物理吸血和法術吸血，持續6秒

怪客解析：狂暴改版之後移除了攻速加成，使得它不再是射手專屬召喚師技能，一些法師也可以使用了。比如貂蟬，韌性、傷害、法術吸血，這對於開大的貂蟬而言都是極其合適的屬性加成。

弱化：

舊效果：90秒CD：減少身邊敵人傷害輸出50%，持續3秒

新效果：90秒CD：5秒內範圍內的敵人減少30%的輸出，同時自己額外增加20%的免傷率。

怪客解析：新版弱化減少了20%的輸出弱化，但持續時間提高至5秒，同時給自己增加額外20%的免傷率。對坦克而言這個僅能改動無疑是加強了，更高的持續時間，雖然對周圍敵方的輸出減少削了20%，但同時也增加了20%的免傷作為彌補。

說完了體驗服的內容，再來補充一下模擬戰的正式服更新內容。

今天正式服更新後，模擬戰將新增5個S級別的新天賦。

鬼神降臨相當於加強了目前的戰士陣容，本來呂布就足夠強了，如果刷到這個天賦那吃雞應該十拿九穩。

女帝威嚴則似乎是給武則天的普攻增加了一點控制效果，算是彌補了武則天在大招放出來之前的作用真空期，畢竟是一張高費卡，但之前的武則天大招放出來之前基本上可以相當於沒有。

戰友之殤，感覺蘇烈和老虎的陣容不是很好搭，個人平時很少會把這兩個英雄搭一起。

稷下機關術，這項天賦實屬加強稷下陣容體系。墨子和孫臏都屬於稷下主力，2個稷下就能湊稷下天賦而且還能出機關術天賦，絕對的高性價比百搭天賦。

守衛者，這個屬於標準的坦射陣容，鍾馗盾山黃忠開局滿藍，意味著鍾馗開局就能勾一個過來，站位好了可以開局秒掉對方一個核心。盾山開大能抗，黃忠開局大招傷害直接拉滿，感覺這個天賦實屬最強。

天賦調整方面比較平衡，雙快攻被砍，也就是一些恰爛分的陣容不能玩了。王者威嚴雖然下調了1秒持續時間，但增加沉默效果，還可以接受。羈絆進階1%的調整，無傷大雅。

英雄調整方面加強了夏侯惇和武則天，削了公孫離。

扶桑傷害比例稍微上調，群雄稍微削了一下，大概是考慮到平衡新天賦鬼神降臨吧。
