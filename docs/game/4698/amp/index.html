一步到位！策劃見面會幹貨匯總上線啦

天天要聞
2019-12-05

摘要

光陰如箭，日月如梭。距離上一次暑假的策劃面對面活動已經過去很久了，恰逢逆戰公測盛典即將到來，今年的第五次策劃面對面活動終於在11月29日晚如期與廣大逆粉見面了。除了我們熟悉的主播夜崽崽外，這次見面會上之前從未現身的美女主播小遇見與狗頭策劃也驚喜登場!

光陰如箭，日月如梭。距離上一次暑假的策劃面對面活動已經過去很久了，恰逢逆戰公測盛典即將到來，今年的第五次策劃面對面活動終於在11月29日晚如期與廣大逆粉見面了。除了我們熟悉的主播夜崽崽外，這次見面會上之前從未現身的美女主播小遇見與狗頭策劃也驚喜登場!因此這次的面對面活動自然是乾貨滿滿，為了彌補有事未看的逆粉的遺憾，小逆我特意總結了四大亮點，快來跟我一起看看吧!

TOP1 獵場戰鬥不停歇

今年的公測版本逆戰首次推出32人獵場地圖可謂刺激無比，但你要是以為僅此而已那就大錯特錯了~以後還會出更多的32人獵場地圖，而且會放開限制，比如自帶槍械寵物等等，讓逆粉們耍得更加開心!針對另一個再臨大都會是否會出煉獄的問題策劃大大們表示再臨暫時不會出煉獄，但是不必為此失望，畢竟作為獵場模式的彩蛋地圖未來會製作和開放更多的區域供逆粉探索，當然比肩冥王骨弓的強力掉落也是不會少的(老Z：我太難了);

TOP2 塔防上新速不慢

作為逆戰PVE模式的另一頂樑柱，塔防模式的更新自然不會停下!明年塔防模式除了更新新圖外還會帶來一大批新的強力怪物供逆粉挑戰!新陷阱也自然不在話下，當然有了新朋友不忘老朋友，天網的升級版本正在緊張製作之中，讓我們期待它的王者歸來!

TOP3 逆粉要求全滿足

要說逆粉最愛的氪金活動換購肯定要算上一個!逆戰的策劃大大們也確認了換購活動已經在準備之中，與以往不同的是這次的換購會考慮出寵物和人物的換購哦!小逆我可以大膽預測未來獵場里初霜小姐姐會更多了呢!另一方面自選商店也會做出調整，很多曾經在網頁活動出現過的道具有可能上線，敬請期待~

TOP4 系統升級很給力

每一次大更新都會伴隨逆世界的系統升級!曾幾何時100級玩家在逆世界中可謂鳳毛麟角，如今一個房間一半的200級玩家都讓人司空見慣。不過逆戰的策劃大大表示，新的角色等級即將到來，讓我們儘管期待!同時刪除掉落道具功能還會繼續增加，媽媽再也不會擔心我的倉庫爆棚啦~

看到這裡你是不是豁然開朗了呢?就讓我們一起期待12月5日新版本的到來吧!小逆祝大家天天有金光，月月上王者!我們下期再會。
