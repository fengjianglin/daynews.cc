DNF：超星河閃光不斷 旭旭寶寶刷上癮 連這樣的小號也拿來刷

2019-12-11 585條評論 1,594次閱讀 406人點贊

眾所周知，像刷深淵這種枯燥無味的活旭旭寶寶都是安排大坤坤去做的，自己平時負責打幾個團再幫幾個水友上號圓夢就可以。但是這次新史詩之路活動出來後，旭旭寶寶刷超星河上癮，刷到沒號刷，最後連這樣的小號也拿來刷，玩得不亦樂乎！

這次新史詩之路，分為星河和超星河兩種，刷超星河有幾率遇到隱藏BOSS，只要遇到最低掉落1件史詩，最高可掉落5件史詩，而且沒有隱藏的時候閃光幾率也非常高，旭旭寶寶就是喜歡這種閃光畫面，非常刺激。

起初，旭旭寶寶本來只是想用紅眼給大家展示一下超星河爆率如何，並向水友挑戰滿疲勞挑戰10史詩，結果刷上癮了，刷完紅眼的疲勞還不夠，還要換號繼續。由於大坤坤白天將其他號疲勞刷完了，不過這沒有阻止旭旭寶寶想要刷超星河的心。

沒有大號可玩就拿小號來刷，剛開始刷的小號還有裝備，最後刷到連裝備都沒有的紅眼小號也拿來刷，徹底刷上癮，就是為了刷超星河碰見隱藏BOSS，爆出5史詩的快感！但是旭旭寶寶刷了這麼久，碰到隱藏BOSS最多掉落3件！

看完全文之後，各位小夥伴們，你們覺得新史詩之路好玩嗎？碰見隱藏BOSS最多掉落多少史詩呢？有什麼想說的話和建議，都可以參與評論留言！
