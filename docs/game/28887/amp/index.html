劍靈打臉所有武俠網遊，曾和lol兩分天下！為何火了半年就涼了？

天天要聞
2019-12-21

隨著大眾娛樂的方式越來越多，遊戲類型也越來越廣泛，遊戲也在隨著社會的變遷不斷注入新元素。新元素的增加並不會影響老遊戲的市場。比如說武俠網遊類型的遊戲，從網遊剛剛誕生到現在一直比較火。中國乃至世界有大量的武俠網遊粉絲。大陸有很多武俠網遊遊戲，但是它們的存活力不強，剛誕生就被社會所否認。縱觀遊戲來看，只有韓國的一款武俠遊戲，符合大眾的胃口。

中國和韓國的文化在一定程度上具有相似性，比如說遊戲，中韓兩國的遊戲一直在互相影響。直到現在，我們依舊可以看到很多玩家在網吧玩著韓國的網遊。雖然武俠是中國的體裁，但是韓國及時抓到了這個元素，他們創造出了很多款武俠類的網遊，但是大多都不符合中國網民的胃口，直到劍靈的出現，讓中國網民眼前一亮。

說起韓國的遊戲公司，我們第一個會想到頂尖遊戲公司ncsoct吧！成立於1997年，這家遊戲公司在韓國具有悠久的歷史，在大陸也具有一定的影響力，公司出品的天堂一，天堂二，激戰一，激戰二都受國內大小玩家的喜愛，因為遊戲畫面精緻，製作良心，給玩家們帶來的印象都是積極向上的。正如大家所料，這款優秀的遊戲就由這款有限公司出品。

但是天公不作美，Ncfoft公司的遊戲是屬於富人的娛樂，雖然天堂系列的遊戲已經有了高清畫面，但是他需要超高的電腦配置，在20世紀初中國大陸，就算是網吧也沒有那種條件，所以儘管這款遊戲受大眾追捧也成為流行。

劍靈的出現，讓很多玩家感到眼前一亮的感覺，可能是因為伺服器的不同，在中國劍靈是騰訊遊戲做代理。全新的遊戲體驗，超清的畫面，劍靈剛剛出世一個月，玩家數量就已經到達了100萬以上。剛剛推出的時候，在網吧中玩英雄聯盟的幾乎和玩劍靈的玩家對等。

當年玩家們對於劍靈的痴迷不亞於英雄聯盟，雖然才是一個體驗服，但是一個激活碼價格已經到達了幾千元，公測剛剛推出，就有很多玩家充了騰訊的會員，防止因為遊戲端擁擠難以進入遊戲。劍靈的出現讓騰訊掙夠了錢。

這個遊戲沒有辜負騰訊的期望，同時他也沒有辜負玩家們的期待。清晰的遊戲畫面，獨特的角色動畫讓我們看到了網遊公司的實力。雖然從遊戲的內容來看，劍靈的遊戲和中國傳統的武俠遊戲還是有一定的區別，但是這絕對不影響中國玩家對這款遊戲的熱愛和追捧。

再好的遊戲也都有缺點。劍靈剛一上線，給了大家一個驚喜，但是劍靈並沒有不斷更新，漸漸的人們對於他的熱情也消退了。看到劍靈的商機，騰訊也愈發過分，收費越來越嚴重。

就算是這麼優秀的遊戲，也是有缺點，未來中國的遊戲市場，武俠遊戲還需要努力。
