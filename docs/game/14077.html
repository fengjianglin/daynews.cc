上官婉兒如何打出最大化輸出？學會剎車，每段大招銜接普攻打傷害

2019-12-11 1,414條評論 4,446次閱讀 2306人點贊

文|天雲解說原創，盜載必究

【摘要】大家好！上官婉兒是一個法刺英雄，這個英雄有一定操作難度，很多玩家都會上天，但是卻只會上天一套打完殺不掉人被別人反殺，所以導致兩極分化，會玩的人這個英雄太強，但不會玩的打不出輸出老是斷大，下面給大家介紹一下她帶來一些連招技巧等。

技能詳解

被動：普攻第三下是強化普攻，是有時間的限制的，強化普攻是AOE傷害，傷害還是不錯的。筆力是會有小爆炸傷害，會回復一定的生命值，通過筆力可以接大招穿插平A使得傷害最大化

一技能：一技能CD短，但是飛行速度慢，一般配合二技能使用可以卡筆力爆炸傷害，打出被動回復效果，一技能也同樣能解鎖大招

二技能：二技能是上官婉兒的解鎖大招的關鍵技能，屬於上官婉兒的軟控制技能，傷害雖然不怎麼高但是減速留人解鎖大招是關鍵，一個二技能可以解鎖大招，線上注意使用二技能加一技能打出筆力使得傷害更高

三技能：大招可以最大飛行五次，當時必須有目標，敵方單位，或者可以用自己的一二技能解鎖，如果斷大只是消耗飛行次數X20%的CD，平時也可以作為位移使用，穿比較薄的牆，解鎖大招需要刷新四次，第五次無需觸碰直接可以上天，如果上官婉兒衝刺時被控制，大招是不會被打斷的，結束也可以繼續刷大，剎車每釋放一段上面都有進度條，玩好上官婉兒大招的方法是方向鍵施法，需要控制好方向，上天的上官婉兒是無法選中的這個期間是可以移動的。

上官婉兒一技能跟二技能可以成三角進行刷大，線上使用一二技能清線會比二一好用一些，卡一技能的爆炸。上官婉兒可以通過技能補充狀態的，玩好上官婉兒一定要學會剎車，因為殺人前對手往往會有一些保命技能，例如輝月或者宮本等無法選中的技能，剎車能夠很好的騙過這些

完美擊殺，在沒有藍的時候，可以看情況利用大招直接利用小兵或者敵人身上進行刷大，傷害也是很足的，玩上官婉兒可以開啟頭像鎖定，這樣你可以通過點頭像平A追蹤敵人這樣就可以實現剎車，但是記住不要動方向鍵盤，不然你就斷大了。

上官婉兒連招非常之多所有連招都是根據基礎連招改變衍生的，大招釋放有三秒的讀條時間停頓，每次刷大招都會獲得強化普攻所以我們利用三秒的讀條時間裡穿插普攻整個AOE的穿透型強化普攻造成更高的傷害。支援連招的手感和操作輪盤等大家要經常在訓練營等多練習形成肌肉記憶，這樣打起來就不會有緊張感就不會慌不擇亂。

小技巧

上官婉兒最遠的追擊連招，大概的距離為中路己方一塔到敵方一塔，這也是上官婉兒強勢的原因，只要對方中單四級不是滿血基本都可以依靠大招秒殺對手或者逼出敵方中單閃現。這套連招非常基礎，需要在訓練營練習到閉眼完成不斷大。

常見斷大原因釋放二技能後沒有走一步再接大招，導致大招夠不到二技能釋放一技能時沒有停頓或者沒有掌握好節奏，導致沒有成功釋放出一技能。在一級兵線到達前可以通過對普攻兩次攢兩次被動，當兵線到達時對準兵線就可以進行一次穿透傷害。

前期思路

上官婉兒前期的線上能力是很強的，一技能利用自己的強化普攻就可以快速清線，二技能有一定的小爆發所以對線期間建議打的凶一點，前期能遊走支援隊友就遊走支援，到達四級就是上官婉兒的強勢期，一套連招學會穿插普攻使得傷害最大化，那麼基本就能夠在線上製造單殺，這就是上官婉兒長時間被禁的原因。

銘文推薦

10狩獵、10心眼、10夢魘

常用法師的88法穿銘文，為什麼帶狩獵？是因為上官婉兒的被動強化普攻，所以需要有一定攻速加成，使得平A更流暢，上官婉兒需要移速，因為大招需要不同角度的刷大等，移速會使得手感不錯，前期支援也快一點。

出裝推薦

上官婉兒單殺能力特彆強，只要技能完備就能發動一波襲擊，所以為了更多次的發動奇襲，可以優先選擇冷靜之靴，而輝月會為其爭取刺殺後生存下來做出貢獻。

出裝細節

開局上官婉兒鞋子出門順序出裝為冷靜之靴、痛苦面具、迴響之杖、博學者之怒、虛無法杖、輝月。對手有回復能力強的可以出夢魘，如果對自己操作有一定的自信能夠最大化上官婉兒的傷害，那麼就可以出半肉，後期輝月換賢者的庇護和名刀司命。

最佳搭檔

鬼谷子：鬼谷子能幫助上官婉兒不用在遠處累積衝刺次數，並且給出的控制效果能幫助上官婉兒完成積累釋放。

劉邦：上官婉兒需要突入陣內進行刺殺，劉邦能夠藉助上官婉兒的高機動性順勢突入，上官婉兒也能在劉邦的保護下安全的展開刺殺行動。

瑤：持續輸出的上官婉兒在瑤附身後，全力輸出傷害並不用擔心突襲。

克制英雄：

張良：面對有2個強勢控制技能的張良，上官婉兒的靈動性會受到較大限制。

花木蘭：不僅僅擁有同樣靈活的身手，還擁有另大部分法師畏懼的沉默，上官婉兒很難找到空隙突入戰場。

楊玉環：面對楊玉環很難擊殺，你不釋放最後一段大招她也不放一直和你打消耗。


後期思路

後期的上官婉兒打法就可以更加激進一點，只要等敵方的控制一交，就可以刷大收割了，特別是在野區狹窄的地形中，雙方爆發和控制都不足，婉兒更加容易刷出大招。中後期小規模團戰，婉兒盡量不要用大招進場，單單”12連招”就可以打出很足的傷害了，大招進場一旦失誤，就是有去無回。只有當你確定敵方沒有控制和爆發時，才會考慮進場收割，收割的連招上面都有講到。值得一提的是，婉兒利用一、二技能經常可以空刷多段位移，追擊和逃生能力都非常強。

利用好剎車的上官婉兒更加恐怖，往往能夠打出更多傷害
