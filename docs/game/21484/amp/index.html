《落櫻散華抄Remake》公布三周年系列活動情報

天天要聞
2019-12-19

　由台灣團隊神嵐遊戲自行研發、營運的戀愛手機遊戲《落櫻散華抄Remake》，今（19）日公布了遊戲中12月份推出的三周年系列活動。

【以下內容為廠商提供資料原文】

一起在雪花中慶祝，耶誕、福音轉蛋期間限定！

　　於2019/12/25 00:00 ~ 2019/01/15 23:00期間《落櫻散華抄Remake》將於遊戲中推出耶誕、福音系列轉蛋。

禮物上門

　　發送禮物的任務自然是難不倒麗子，但是在圓滿完成任務準備要把自己禮物送出手的時候，麗子卻表現出一副心不甘情不願的模樣，這到底是怎麼回事呢？

  • 『快、快點收下啦，是要讓本小姐把手舉著等你多久？』

天降驚喜

　　為了分送禮物，悠理和佳奈分到了駕駛飛天雪橇的任務，只是悠理不小心從雪橇上掉下來啦！不過沒有關係，因為有緩衝的降落術式在，乾脆趁這機會一口氣抵達要分送禮物的目標家裡吧！

  • 『呀啊啊啊！佳奈～～～』

飛速雪橇

　　成為聖誕老人的佳奈自告奮勇的擔任起駕駛雪橇的任務，雖然是頭一回嘗試，但是配合著聖誕鈴聲的BGM，佳奈的駕駛技術表現得可圈可點！可是… 這雪橇速度是不是越來越快了？

  • 『目標是下個好孩子的家… 咦？悠理咧？』

聖誕夜話

　　夏希竟然接到了讓她扮裝為聖誕老人，和其他孩童互動的任務，雖然說這些小孩都不是普通人，而是年幼的善良妖怪，不過內向的夏希最後能否成功的完成工作呢？

  • 『齁齁齁… 聖誕快樂… 嗚，感覺這樣笑好奇怪…』

聖誕晚宴

　　烤火雞、肉腸卷餅、甜派… 做為聖誕節的習俗，大家要團聚一堂共享美食，而櫻華組成員中的綾乃毅然接下了替大家準備豐盛大餐的任務！

  • 『你說這身服裝嗎？呵呵，當然是配合氣氛特別換上的啰。』

馴鹿快遞

　　叮叮噹～叮叮噹～鈴聲多響亮，然而帶著禮物到來的並不是聖誕老人而是馴鹿？！興緻十足穿著馴鹿裝到處找有煙囪的屋子發送禮物的小葵難不成又弄錯了節日的傳統嗎？

  • 『咦～可是比起載人，肯定是送禮物比較有趣呀！』

聖誕壞公公

　　相傳在聖誕截期間，聖誕老人會給乖小孩禮物及糖果，他的隨從坎卜斯則會去處罰不乖的小孩。像這樣能以冠冕堂皇的名義欺負小孩子的任務黃泉最喜歡了，看她興緻高昂的模樣，不知道是哪個壞孩子會落到她的手中呢？

  • 『嘻嘻嘻… 壞孩子在什麼地方呢？』

服務精神

　　聖誕老人是給孩子們帶來夢想與快樂的存在，親和力十足的近扮演的聖誕老人完美無缺，只是… 為什麼這個聖誕老人比起男孩，更受到小女生的歡迎呢？

  • 『禮物一人只能拿一個，不過笑容可以無限量替各位小姐送上唷。』

栩栩如生

　　和近搭檔的真雲因為總是板著臉，加上生人勿近的氣質，成功獲得了會處罰不乖小孩的聖誕老人助手坎卜斯的扮演工作。雖然因為御影嚴重抵觸，最後只上了一層淡妝，不過還是一堆小朋友被嚇哭了。

  • 『喂，為什麼是我負責扮演這個角色？』

節前集訓

　　聖誕節是一年一度的大節日，作為靈揀會會長的八神也配合著節慶，換上了相應的打扮，只是想要從她手上拿到禮物，可不是件簡單的事情。

  • 『只有得到靈揀會聖誕突擊考核前三名的人才能拿到禮物！』

叮叮噹～叮叮噹～那回蕩在高空的熟悉旋律，正揭示著佳節的到來！

　　即將迎來一年一度的聖誕節日，靈揀會也為組織內的所有人準備了一份大禮。那份禮物就是…… 從平安夜到新年的連續168 小時集中魔鬼特別訓練營！

　　「因為連假快到所以就鬆懈下來了嗎？就讓我好好糾正你們這種不成熟的心態！」不過就在靈揀會內部一陣哭天喊地的同時，八神她也留下了一線生機。「不過我也不是什麼魔鬼，只要你們能在聖誕節之前完成我給出的課題，也不是不能批准你們的休假。」

　　而八神的課題赫然就是－－－靈揀會聖誕老人生死戰！

活動玩法

　　2019/12/21 12:00 ~ 2020/01/15 23:00 期間，遊戲中將推出三周年系列活動，活動分為上半活動與下半活動；上半活動包含「聖誕老人生死戰」、 「送貨開始」與「交通工具Get」共計三個關卡；下半活動包含「使命必達」、「一單都不能少！」與「一年的感謝」共計三個關卡。

　　上半活動詳細開放時間：

  • 「聖誕老人生死戰」開放時間：2019/12/21 12:00 ~ 2019/12/24 23:00，為不可重複進行關卡。
  • 「送貨開始」開放時間：2019/12/22 12:00 ~ 2019/12/24 23:00，為可重複進行關卡。
  • 「交通工具Get」開放時間：2019/12/22 14:00 ~ 2019/12/24 23:00，為不可重複進行關卡。

　　下半活動詳細開放時間：

  • 「使命必達」開放時間：2019/12/25 12:00 ~ 2020/01/05 23:00，為可重複進行關卡。
  • 「一單都不能少！」開放時間：2019/12/26 12:00 ~ 2020/01/05 23:00，為不可重複進行關卡。
  • 「一年的感謝」開放時間：2019/12/27 12:00 ~ 2020/01/05 23:00，為不可重複進行關卡；此關卡為高難度關卡，還請玩家注意。

　　本次活動除了上半活動中的「送貨開始」與下半活動中的「使命必達」關卡外，皆為不可重複關卡；通關上半活動中的「聖誕老人生死戰」關卡，可獲得經驗石、靈力精華與搜集品「聖誕扮裝組」，集齊一個後，可於活動兌換區兌換本次耶誕、福音系列任一張SSSR 角色卡

　　通關上半活動中的「送貨開始」關卡，可獲得經驗石與搜集品「耶誕送貨單」，集齊一定數量後，可於活動兌換區兌換強效復甦藥劑與SSSR 裝備卡「夢境雪橇」等獎勵；通關上半活動中的「交通工具Get」關卡，可獲得經驗石、靈力精華與SSSR 裝備卡「夢境雪橇」等獎勵

　　通關下半活動中的「使命必達」關卡，將可獲得經驗石與搜集品「聖誕好評點數」，集齊一定數量後，可於活動兌換區兌換本次耶誕、福音系列轉蛋券與重置印記券等獎勵，另外，搜集品「聖誕好評點數」有機率在「使命必達」關卡探索途中掉落

　　通關下半活動中的「一單都不能少！」關卡，將可獲得落櫻轉蛋券、本命轉蛋券、重置印記券、搜集品「聖誕好評點數」與搜集品「聖誕扮裝組」等獎勵；另外，通關下半活動中的「一年的感謝」關卡，將可獲得落櫻轉蛋券、本命轉蛋券、搜集品「聖誕好評點數」與SSSR 萬用卡等獎勵。

　　本次更新除了三周年活動外，還另有開放三周年任務，在2019/12/23 12:00 ~ 2020/01/02 12:00 期間完成任務即可獲得水晶、重置印記券、復甦藥劑與SSSR 應援卡「暖心雙傑」。

夢境雪橇

暖心雙傑

主線21 章- 神器試煉，正式開啟！

　　在櫻華組隊長進入萬有之源嘗試掌握更深層的力量同時，櫻華組的隊員們也在積極的準備著三神器的適性特訓；然而，既然被稱為神器，對於適格者的要求可是非常嚴苛的！究竟櫻華組的隊員們是否能順利的取得神器的認可呢？

　　三神器的適格者試煉，即將開始。

歡迎新生入學！讓我們帶你／妳快速的熟悉櫻華學院吧！

　　於本次更新後，將會新增常駐型新手任務，透過完成任務將可更快熟悉《落櫻散華抄Remake》的遊戲系統，讓各位新生能更快的融入櫻華學院；此外，完成任務後還可獲得SSSR角色卡「熱心情報商」、SSSR角色卡「親切的紅髮同學」與SSSR特殊卡「新生幹勁」等獎勵。

　　各位還在等什麼？趕快來加入櫻華學院的大家庭吧！

新生幹勁

2019 將邁入歷史，在最後一天，與落櫻一起迎接2020 的曙光！
