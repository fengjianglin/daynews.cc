Rng新陣容德杯首戰不敵Omg 網友振臂高呼：59e揭棺而起

天天要聞
2019-12-21

英雄聯盟轉會期已經正式結束，最能代表每年新隊伍新成員、LPL今年最後一項官方賽事，德瑪西亞杯，於今日正式開啟。德瑪西亞杯小組賽在LNG全新主場蘇州陽澄國際電競館和LGD杭州電競館兩處同時進行，LPL十六支戰隊與LDL四強同台競技。今日第二場比賽，小龍堡加入後的新陣容RNG居然不敵Smlz回歸後的OMG，引起網友們重點關注。

今年LPL轉會期各隊都進行了較大程度的成員變動，最終OMG上場的隊員為：上路Curae、打野H4cker、中路Icon、下路Smlz、輔助cold，RNG最終上場隊員為：上路Langx、打野XLB、中單xiaohu、下路Betty、輔助Ming。OMG戰隊Smlz回歸後與Icon再聚首，”冷老”組合再現江湖。RNG戰隊Uzi因手傷未上場，由新人AD選手Betty上場擔任射手位，同時新人小龍堡轉回後首次亮相賽場，兩隊紙面實力仲伯之間。

比賽初期對線階段，Icon在六分鐘時卡住升級時間，利用自己6級而小虎5級的優勢單殺小虎。同時下路Smlz的女警憑藉攻擊距離優勢頻繁推塔，9分鐘就拿下一塔。隨後OMG中下聯手，斬獲四小龍、大龍、遠古龍等成就輕鬆達成，僅僅用30分鐘就結束了比賽，最終OMG以9:6的戰績拿下RNG。

早期的OMG戰隊曾經為LPL觀眾帶來眾多熱血激昂的比賽，最著名的當屬”50滴血翻盤”，贏得了眾多的觀眾的心，被網友們稱為”黑暗勢力”，而粉絲們自稱”59E”，即比喻OMG的粉絲有59億人那麼多。然而自從老OMG隊員離開後，新的OMG戰隊一直戰況不佳，粉絲數逐漸減少，有網友稱粉絲數僅剩”0.59E”，但仍有一些忠實的老粉絲堅守著OMG。今日OMG戰勝RNG，老OMG粉絲戲稱自己”0.59E揭棺而起。”

OMG戰勝RNG賽後，Icon在採訪中承認再度與Smlz並肩作戰，老隊員聯手，默契仍在。希望Smlz、Icon雙C再度聯手後，能帶領OMG一鼓作氣衝出小組賽，直接衝到德瑪西亞冠軍獎盃的面前，讓OMG久違的”黑暗勢力”再度回歸，59E重建規模！
