歐洲、中東、非洲遊戲榜：全球玩家都愛玩它

天天要聞
2 天 ago

新一期的EMEAA（歐洲、中東、非洲）的遊戲周銷榜又公布了，《FIFA 20》延續了全銷量爆表的趨勢不僅統治英國榜單，在歐洲、中東、非洲周銷榜中也連續兩周霸佔榜首的位置，可以說是全球玩家都喜歡的遊戲。

其中實體銷量排名前兩位的是，第一名的《FIFA 20》和第二名的《使命召喚16：現代戰爭》。這也是《 FIFA 20》連續第二周位居第一，也是自發售以來第六周排名第一。

從數字銷量來看，《奇異人生2》第五章銷量在《使命召喚16：現代戰爭》和《鐵拳7》（從上周的60名到上周的第5名）之上排名第四。但是，僅憑數字銷售並不能使其躋身該地區前50名的總銷售排行榜。

歐洲、中東、非洲周銷榜TOP10

（截止12月8日）

1、《FIFA 20》

2、《使命召喚16：現代戰爭》

3、《路易吉洋館3》

4、《寶可夢：劍》

5、《星球大戰絕地：隕落的武士團》

6、《馬里奧賽車8》豪華版

7、《GTA5》

8、《舞力全開2020》

9、《寶可夢：盾》

10、《荒野大鏢客2》
