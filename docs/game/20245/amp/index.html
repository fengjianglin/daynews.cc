RNG聯合Uzi坑了Jackeylove？iG粉絲根據轉會期爆料，得出結論

天天要聞
2019-12-17

LPL觀眾曾天真地以為，在轉會窗口關閉的當天，年度懸疑大劇《Jackeylove去哪兒了》，會落下帷幕。然而當天晚上，確實出現了非常勁爆的消息沒錯，大家得知了RNG Mata重新連接、LPL 001號選手斷開連接、EDG Clearlove教練上線等消息，但仍然不知道阿水新賽季，將為哪支戰隊效力。唯一可以確定的是，iG JKL已斷開連接，因為iG官宣了S10首發AD和輔助。

自從Jackeylove成為自由人以來，他就成為了整個電競圈，討論熱度最高的選手，每天都有新的，關於他的爆料傳出，而且內容大不相同。在iG官宣VG下路雙人組加入自身之前，結合轉會期所有爆料來看，JKL最有可能去三支戰隊，Tes、EDG和SN，或者與iG續約。正當大家猜想，阿水會做出何種選擇之時，RNG突然插了一腳，有人爆料JKL被MING說動了，要去RNG！

RNG有Uzi坐鎮，Jackeylove怎麼會去RNG呢？有個iG粉絲根據爆料，得出了一個非常離譜的結論，他表示RNG聯合Uzi坑了阿水！他自認為很有邏輯的分析道：JKL之所以會和RNG簽意向合同，是因為JKL以為Uzi會退役，他想和MING搭檔。結果Uzi沒退役，導致其他戰隊想買JKL，就得額外掏違約金，而SN和RNG關係好，不用掏違約金，這也是為什麼有人說JKL去SN的原因。

不管是RNG粉絲，還是路人，看到這個帖子，都給發帖的人氣笑了！有人表示此事是真的，他就是Uzi，可以證明，還有人附和道「JKL為這件事跟Uzi打了一架，Uzi不僅踹了JKL一腳，還用凳子砸了他一下」，他就是那個凳子，也可以證明事情的真實性。有一說一，這個分析實在太離譜了，這要是真的，JKL把真相爆出來，RNG和Uzi還用在圈內混么？

據iG官方爆料，JKL仍然是自由人身份，但轉會期已經結束，各位覺得阿水，會如何抉擇？歡迎留言。
