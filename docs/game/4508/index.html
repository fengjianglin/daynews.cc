豬年就要接近尾聲了，為了慶祝鼠年的到來，王者榮耀會在鼠年出上幾款限定皮膚。在此之前早就有爆料稱鼠年限定皮膚一共有五款，所以很多玩家都想到了五虎上將。因為五虎上將乃是劉備手下五大精英，三國里都是出了名的，最主要的是數量上也恰好是五個，並且皮膚的外觀我們已經看到了，就是沒看到馬超的。現在官方給出了鼠年皮膚的主題風格，為五嶽的國風類型。這樣就基本上排除了五虎將的可能。因為五虎將皮膚是末日機甲類型的，不符合五嶽的風格。

那麼鼠年限定皮膚會是哪幾位呢?有不少玩家根據五嶽的由來和含義分析了以下幾個英雄：泰山-嬴政，華山-趙雲，衡山-曜，恆山-莊周，嵩山-達摩。在這幾個英雄中我認為達摩的可能性是比較高的。因為嵩山有少林寺嘛，達摩是開山始祖，正好對應。其餘的四個英雄就眾說紛紜了。

但是盤古作為五嶽的開山鼻祖，也不是沒有可能。因為盤古至今還沒有一款史詩皮膚。同樣的推理，涉及到盤古就會涉及到女媧。畢竟有盤古開天女媧補天的傳說，所以這兩位英雄的概率也非常大。

不管怎麼說，五款國風皮膚應該會很不錯的。不少玩家喜歡國風的元素，能充分還原當朝的文化色彩，更符合我們玩家的審美。朋友們你們心目中理想的五嶽英雄會是誰呢？
