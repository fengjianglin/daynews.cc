DNF：100級太難？嚇得狂人瘋狂囤墊子，準備打造5件17裝備

天天要聞
2019-12-22

地下城比較失敗的一個版本當屬普雷，因為是團本末期，雖然副本比較難，但是玩家不買賬，太折磨人了，很多人選擇了脫坑，為了留住玩家，官方更新了一大波活動，可以說從夏日套開始活動沒有停止過，主要就是為了玩家減負，讓更多的人儘快畢業，這樣好升級100版本，考慮到劍魂人口基數比較大，為了留住這部分玩家，策劃也提前更新了無我技能，之後狂人的劍魂也成為了國服傷害第一職業。

前段時間韓服100級見面會終於曝光了各種內容細節，其中包括女鬼劍的三覺，還有武器的更新策略以及難度比較大的副本，眾所周知，從團本做任務換武器開始，地下城所有玩家裝備就沒了特色，每次更新版本直接升級裝備即可，而新版本進行了改革，出現了一個更厲害的裝備，叫做「神話」裝備，此外史詩裝備也做了一個改動，可以進行交易了，也就是說現在的普雷畢業裝備在以後將不會是主流，史詩武器的地位進一步下降，和60版本的神器相仿。

除了武器的改動讓人關係的就是變態的副本了，早期團本模式取消了，新更新的地圖可以單刷了，不過難度比較大，普通玩家想要通關很難，就算現在已經達到了普雷門檻，在新版本也比較難混，要麼是抱團取暖，要麼是找一些大佬帶圖做任務，60版本困難時期又回來了，不過一些知名主播肯定又要徹夜肝神話裝備，因為曝光的屬性比較好。

看到新副本那麼困難，狂人也不淡定了，雖然擁有全服傷害第一的劍魂，一身增幅16，還有一把項鏈增幅到了17，但是狂人的技術太菜，只能依靠超高的傷害，想要刷神話裝備必須要下副本，所以只能對現有的裝備升級，根據狂人的透漏，現在已經開始大量囤墊子了，準備打造5件紅17的裝備，屆時整體實力會更上一層樓。

以狂人的知名度和金錢所言不虛，畢竟我們是看著他的直播把所有的裝備增幅到了16，而且為了一把項鏈上17，自己花費了一輛小賓利的價格，非常捨得投入，此次在打造四件17難道也非常大，前期的準備工作估計就要花費不少錢，屆時也期待他的直播，希望能成功，也為國服劍魂爭一口氣。
