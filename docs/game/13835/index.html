TOP2：有顏值、有個性也有實力，1601攻擊資質、1.3成長

貓靈在渡劫以上的寵物里，進階+染黑色顏值上更高一些。但是這隻169級的貓靈，非全力量加點就有2637攻擊、4254氣血，其他的屬性點都洗了出來，都是自然成長的面板屬性。攻擊資質1601附帶極限1.3成長，5技能不多但打造的很到位。高級連擊、必殺、吸血、夜戰和強力，再配個高級偷襲套裝，抓鬼用非常暴力。寶寶的等級越高，高級強力能附加的攻擊力也越多，平時打張天賦符相當於2950攻擊。還有坐騎上加成15%力量點的傷害結果，和5層捨身擊內丹，碰見抗物、抗法的野鬼，兩刀就能帶走。

TOP3：帶塔大龍的巔峰逆襲！掃塔獲得5級寵物裝，直接擺價8萬

夢幻西遊里，不僅有千速寵物打造的方式，更有速度不足50點的超級龜速寶寶。可以看到，上圖的這隻畫魂，速度資質就只有100點，面板上的速度只有16！加上高級遲鈍的效果，還剩下12.8！就是因為，它攜帶了一件5級的「神器」。
