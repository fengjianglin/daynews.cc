元氣騎士：聖誕版本預告來襲，涼屋改玩JOJO梗，4款全新皮膚上線

2019-12-15 356條評論 1,449次閱讀 507人點贊

「叮叮噹叮叮噹，鈴兒響叮噹……」當寒冬降臨，熟悉的旋律響起時，在一片孩童的歡笑中即將喜迎聖誕節的來臨。作為最具西方特色的傳統節日，人們除了準備豐盛的晚宴進行慶祝以外，聖誕禮物也在其中扮演著不可或缺的重要角色，據說只要小夥伴們向聖誕老人祈禱，都會在當天獲得自己夢寐以求的禮物。而作為大家熱衷的手游《元氣騎士》，涼屋大大們又何嘗不在扮演著聖誕老人的角色，在聖誕節臨近之際萬千玩家的祈禱中，大家也終於迎來了期盼已久的「聖誕禮物」。

圖片源自：涼屋遊戲

​作為《元氣騎士》的慣例，涼屋會在版本更新之前發布先行的預告片，讓大家對即將到來的全新遊戲版本有一定的了解。而作為熱衷於「玩梗」的策劃大大，最喜歡的事除了幽默搞怪以外，製造懸念也是他們的拿手好戲，所以本期預告片中出現的內容，很有可能只是大版本中的一小部分，相信在沉浸兩個多月後，涼屋大大們是不會讓大家失望的。那麼在本次預告中都有什麼樣的驚喜等著大家呢？就請小夥伴們和老宅一起，一探究竟吧。

全新的植物種子

作為聖誕節中必不可少的元素，聖誕樹自然首當其衝，而第1關BOSS聖誕樹人當然也就成為策劃大大們的重點照顧對象。可是僅有的一棵聖誕樹又如何能烘托節日的氣氛呢？所以聖誕樹就以另一種更加特殊的形態和大家見面了，那就是聖誕樹種子。玩家得到這個種子，僅需兩天就可在花園收穫躁動的樹榦一把，雖然這件紅武的實力及不上同領域的神器，但是作為涼屋的聖誕禮物，還是相當走心的。

全新的皮膚

  • 法師

本次皮膚的風格有著濃郁的和風，所以玩家可在遊戲中找到許多日漫的影子。而法師作為大家熟知的女性英雄，本次以「夜」為名的皮膚更能凸顯出法師妹子的端莊與優雅。並且激活方式十分親民，僅需8000寶石就能將其收入囊中，所以老宅也在這裡提醒下鍾愛法師的小夥伴們，趕快進入地牢為全新的皮膚積攢寶石吧。

  • 牧師

牧師在遊戲中的風格一直都偏向於莊重與聖潔，所以之前的皮膚設計都有些偏於模板化。而全新的日系皮膚「露」，以可愛的雙馬尾與學生裝相結合，向大家展現出牧師之前所不具備的可愛與靈動。雖然全新的皮膚並沒有附加特效，可是憑藉其「卡哇伊」的外形也足以「萌」翻眾人，那麼牧師粉絲們是否會將其收入囊中呢？

  • 聖騎士

聖騎士可是這款遊戲中絕對的硬漢，不論是技能機制還是初始武器都透露出一種堅不可摧的厚重感，堪稱絕對防禦的能量護盾更是保護著無數玩家在地牢中縱橫馳騁，深受非技巧型玩家的喜愛。

而本次涼屋為聖騎士帶來的日系皮膚與前兩種不同，不能靠寶石獲得，只有通過挑戰模式的試煉才能激活全新的皮膚。據老宅看來，難度也並不太大，如果可以搭配技能CD縮短50%，或者是攻速加倍之類的因子，通關什麼的並非難事。

只不過新皮膚的名稱是「苦」，這裡老宅就十分不解了，難道是準備告訴大家要想玩好聖騎士，必先「苦其心志，勞其筋骨？」那麼小夥伴們，你們認為這款皮膚的名稱究竟有什麼含義呢？

  • 狂戰士

本次更新預告的壓軸來了，狂戰士的新皮膚「思」。風格什麼先不用看，就單從皮膚的特效中大家就能看出，涼屋這次葫蘆里究竟賣的什麼「梗」！上圖中新皮膚的特效，相信熟悉某熱血動漫的小夥伴們一定會不會陌生，看來涼屋大大這次放過了火影梗，改玩JOJO梗了。

「木大木大木大木大……」作為《JOJO的奇妙冒險》中角色迪奧·布蘭度及其子喬魯諾·喬巴拿的口頭禪的諧音，一度被jojo粉絲們拿來表示搞笑，那麼有「木大」出現的地方，「歐拉」何在呢？

好了，以上就是本期的《元氣騎士》聖誕版本預告，共有4款全新皮膚上線，另有JOJO風格的「限定」皮膚等你呦。那麼上述皮膚中你更喜歡哪一種款呢？歡迎小夥伴們在下方留言，大家一起討論，咱們下期再見。
