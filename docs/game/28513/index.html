TES憾負EDG錯失小組賽全勝，網友：怪不得Jackeylove那麼吃香

2019-12-23 194條評論 3,388次閱讀 1685人點贊

S9德杯已經進入了第二日的比賽日，晚上B組已經到了最後一場TES和EDG的焦點戰，這場不僅是焦點戰，也是一場關係到B組三支戰隊的晉級的比賽。這場比賽如果EDG獲勝，那麼將會鎖定B組另一張晉級名額。TES前三場全勝，已經提前鎖定了一張晉級名額。兩隊都是新陣容征戰德杯，而TES目前狀態是所有隊伍中最好的。Karsa到來後和Knight配合幾乎是天衣無縫，兩人前三場都是完美表現，所以EDG要想成為B組第二支晉級到淘汰賽的隊伍，難度還是比較大的。

這場比賽EDG處於藍色方，選出了鱷魚、雷克塞、蛇女、女槍和泰坦。TES則是選擇了凱南、黃皇子、卡牌、霞和女坦。

比賽進行到了3分鐘，Karsa來到下路蹲伏，等了半天都沒有成功。而Xinyi同時出現在中路，成功幫助Scout抓死了Knight拿到一血。而TES這邊很快就做出應對，野輔在上河道抓到了皇子成功造成了擊殺，隨後順勢在中路又把Scout抓死。女坦和卡牌各拿了一個人頭。6分鐘時，Xinyi又來到中路，但是TES這邊早有準備，上野兩人瞬間趕到直接反殺，人頭被369收下。

​7分鐘時，雙方下路對拼，EDG先手，結果Knight的卡牌大招趕到，泰坦被反殺，人頭被霞收下。8分多鐘時，雙方開始爭奪峽谷先鋒，EDG這邊打出1換3，女槍大招掃到了多人，隨後峽谷先鋒被EDG收下。先鋒放到上路後，女槍單吃了一血塔。12分鐘，TES提前搶到位置，吃掉了第二條小龍。

16分鐘，EDG在上河道草叢埋伏，結果被Knight卡牌開打探出來，Karsa直接先手開團，369的凱南完美進場，這波團戰EDG直接爆炸，被擊殺三人後，中路二塔和第二條峽谷先鋒全部被TES收下。18分鐘，EDG多人在下路抓死了Knight，但369也用先鋒在上路破掉了二塔。20分鐘，Xinyi在野區排真眼，結果被TES四人強開擊殺掉。

之後雙方有來有回牽扯了一段時間，經濟基本上持平，EDG也是拿到了兩條小龍。25分鐘，兩邊在上河道開打，EDG打出完美團戰，三人被反殺。隨後EDG開始打男爵，TES雙C過來阻止也獻祭了。隨後EDG收下大龍，拿到了經濟優勢。借著局面優勢，EDG又收下了自己的第三條水龍。29分鐘，TES雙人組在野區被抓，兩人逃回中二塔下還是遭到強殺，EDG直接從中路上了高地。在高地最後的團戰上，TES被團滅，隨後EDG一波結束比賽。EDG用這場勝利成功搶到了晉級到淘汰賽的名額，並且淘汰了V5和LGD。

不得不說，這場比賽是EDG的生死戰，但是時間恰好是晚上7點鐘，不知道是不是時間到了，7點BUFF加持，EDG擊敗了德杯打到目前狀態最出色的TES戰隊，完成了翻盤殺進淘汰賽。這場比賽EDG新人上單Xiaoxiang的鱷魚表現又非常出色，多次突後排成功讓EDG獲得團戰勝利，這狀元上單的新賽季非常值得期待。

而TES雖然輸掉這場，也無礙於他們小組出線。但是在中上野全部加強的情況下，下路新人組合的發揮看起來似乎並不太理想。看完這場比賽後，網友直言，怪不得Jackeylove那麼搶手。
