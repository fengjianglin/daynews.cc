英雄聯盟：雙多蘭打野盲僧，韓服最新興起的套路

2019-12-04

摘要

不少玩家會提出質疑，打野盲僧出多蘭劍不是浪費錢嗎？可事實上，兩把多蘭劍的收益，高於虧的那點錢收益幾十倍不止，可以計算一下，兩把多蘭劍有6%的生命偷取，16點攻擊力以及160點生命值，這樣的性價比在前期能夠碾壓大多數裝備，再配上盲僧w本身吸血，在前期刷野就根本不掉血，而且征服者還有一定程度的吸血能力，也能增強盲僧前期的作戰能力，在1v1的情況下，能打過絕多數上單英雄。

但多蘭劍也不能太多，2~3個正適合，再多點的話就有點得不償失，而且還佔裝備格子，可能對女槍來說多蘭劍是越多越好，對盲僧就不是這樣，2個左右正合適，這方面玩家一定要記住，多蘭劍雖說對英雄有提升，可還是要懂得適可而止，這個套路整體的核心就是第一次回家刷到900塊，然後回家買兩個多蘭劍，再繼續去gank，在有雙多蘭的情況下，續航能力非常強，在不出裝備的情況下，甚至都不用回家補給。

其它裝備還是按照盲僧的正常流程就可以，通常來說盲僧會有兩種玩法，下面也給玩家好好說說，第一種是穿甲流，第二種則是提亞馬特黑切流，下面就給玩家好好說說，兩種不同出裝及不同玩法，穿甲流在裝備方面，是以幕刃、幽夢和夜之鋒刃為核心，這樣出裝的瞎子，一套技能的爆發可以秒掉脆皮，但相對來說身板就比較脆，這種出裝的瞎子更像一個刺客而不是戰士。

反之就是提亞馬特加黑切的這種出裝，這樣的瞎子就是個戰士英雄，一套爆發也沒有那麼高，核心三件套就是黑切、提亞馬特及復活甲，配合上兩個多蘭劍的吸血，在中期階段，這套裝備的瞎子是又肉又能打，整體來說的容錯率也比穿甲流更高，就算被集火秒掉，那也有復活甲可以站起來繼續輸出，但還是那句話，這套裝備的爆發很低，一套技能根本做不到秒殺敵人。

兩種方式都是靠兩個多蘭劍延伸下來的套路，其實雙多蘭就是所謂的過渡裝備，可以幫助盲僧很好的度過前期，而且售價也不是很貴，攻擊力吸血等等一系列東西都是瞎子所需要的， 可能現如今套路還沒在國服興起，玩家感覺有些無法接受，但等到已經流行起來後，玩家在想去玩就為時已晚了，所以務必要把握好這個機會，只要領先一步就是步步領先。
