王者榮耀：娜可露露新皮膚定價888點券，先別買，看這兩句話

2019-12-18 1,910條評論 4,580次閱讀 1656人點贊

各位王者玩家大家好，我是小雨。經過王者榮耀官方多天的預熱，娜可露露新皮膚「娜可媽耶」（不對，是「晚螢」！）終於官宣。這款皮膚從原畫到建模再到局內表現，都十分優秀，而且該皮膚也沒有大幅度改變娜可露露的原有形象，甚至還專門請來了《侍魂》中的娜可露露原版配音師——日本著名聲優中原麻衣為她配音，可見王者官方對於第一款SNK皮膚的重視程度。

皮膚出來了，效果也不錯，大家最關心的自然就是價格了，而官方也已經給出了回應：史詩級皮膚，888點券。而且根據以往的管理，新皮膚上線的首周都會打8折，也就說這款皮膚剛上線的第一周到手價格是710點券，相信這個價格很多玩家還是可以接受的。

不過這恐怕要讓一些指望「白嫖」的玩家失望了，因為在他們看來，SNK的英雄都是免費的，憑什麼皮膚不免費？小雨覺得這種想法有些不可理喻。首先SNK的幾個英雄都不是TX原創的，之前他們沒有版權，也不能利用這幾個英雄來盈利，這是大家都懂的道理。其次每一款皮膚都是經過人家畫師美工用時多日精心製作的，憑什麼就要白送給你呢？

所以小雨覺得有這種想法的朋友們還是要改變一下自己的心態，其實小雨也並不推薦大家買這款皮膚，原因在王者官方的這兩句話：

1、王者官方聲明：為了保持角色音色的辨識度一致，新皮膚上線後，娜可露露的英雄語音也會同步更新為中原麻衣（原版）配音的版本。

對於一些曾經的《侍魂》玩家來說，原版形象的娜可露露絕對要比有皮膚的好看，因為這不在於動畫特效，而是情懷。小雨便是其中一個，而且即使火舞和橘右京出了新皮膚，我也不會買，經典就是經典，改變不了，相信跟小雨一樣的玩家也不在少數。

​2、官方也說了：商城直售。

大家都知道，以前SNK英雄不出皮膚是因為沒有版權，現在TX買來了版權，必然要用皮膚盈利，所以皮膚收費是正常的。不過大家也都知道，版權一般都是有期限的，在版權期內，王者肯定不會只出這一款史詩級的直售皮膚，相信在接下來的時間裡，傳說、限定等各種SNK皮膚都會出現，大家不妨等一等更高品質的。
