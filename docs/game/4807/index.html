王者榮耀KPL超神實驗室：職業射手的基本素養

2019-12-04 0條評論 16次閱讀 0人點贊

摘要

本周的超神實驗室邀請到了KPL元老級射手QGhappy.Hurt!

本周的超神實驗室邀請到了KPL元老級射手QGhappy.Hurt!在過往中Hurt有著無數的精彩操作讓大家記憶猶新，尤其是那次孫尚香的一騎當千獨守水晶更是令人印象深刻，今天呢我們就通過一些測試和小遊戲測試一下刺痛的過人之處~

第一個小測試呢是讓解說天雲操作李信，刺痛操作孫尚香，只要被李信碰到一下，就算挑戰失敗！雖然李信的AOE範圍非常廣，但是幾分鐘的遊戲中李信無法碰到孫尚香，並且幾次反殺，可以說是刺痛在線調戲小姐姐了！！！

第二項測試是走位模擬小遊戲，進一步觀察刺痛尋找生存空間的能力，要求刺痛盡量長時間的盡量躲避子彈堅持生存，並且不可以使用屏幕中刷新的清屏道具，在如此條件下，刺痛的五局平均成績為181秒，最高成績為229秒，比工作人員利用道具的平均成績高出了3.5倍。

要知道未被使用的清屏道具是不會消失的，200秒後的屏幕內可生存空間不到10%。

之後對刺痛在KPL中的經典戰役進行了復盤，用紅點標註敵方英雄位置，紅色圓圈為敵方英雄的最大英雄攻擊範圍，藍點為刺痛英雄的位置。

其實這就和上一個測試有著異曲同工之妙，刺痛總能盡最大可能將自己保持在所有紅圈之外，並在尋找生存空間同時，在地方的攻擊範圍邊緣遊走輸出。

當然在使用特定的射手英雄，比如馬可波羅，刺痛也有時候會壓抑不住情緒，先手肉身開團~但這也要先考慮地方英雄的控制技能和己方馬可凈化和純凈蒼穹的CD情況~

刺痛個人推薦呢路人局比較好用的還是大小姐孫尚香和公孫離，然後交給了大家一個公孫離打線優的小技巧，就是一技能位移到塔下A敵方英雄的時候，由於被動會瞬間A出兩下，你可以再多A一下，然後再點一技能複位，塔是打不到你的~（但要注意對方英雄的控制技哦~！不然浪一下被控交個凈化好虧！！！）

最後刺痛的硬植入還做了一套廣播體操。。。。有興趣的小夥伴可以去看一下原視頻欣賞一下瘦下來的痛仔的美麗身姿！
