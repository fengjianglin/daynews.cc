三個人的時光，致敬永遠無法忘懷的《天之痕》

天天要聞
2019-12-20

小編寫過《軒轅劍四》、寫過《雲和山的彼端》，卻對於這款通關次數最多、印象最深的《天之痕》遲遲不敢落筆。因為這款遊戲在小編心中實在太過於經典，在小編心中已經超過了任何一款其他的遊戲，但是小編覺得讓這款遊戲湮沒在時間的長河中是一種遺憾，所以在這裡給大家介紹一下這款經典！

當年的大宇號稱雙劍齊發，雙劍指的就是《仙劍奇俠傳》與《軒轅劍》！當年的《仙劍奇俠傳》一經推出就收穫了大量的粉絲，而《軒轅劍》直到三代外傳《天之痕》的出現才真正意義上可以與《仙劍奇俠傳》並駕齊驅！

天之痕的故事，發生在《軒轅劍叄》的一百五十年前，背景是中國的隋朝。主角陳靖仇奉師父之命復國，尋找傳說中五樣中原上古神器：伏羲琴、神農鼎、崆峒印、崑崙鏡、女媧石，打算收齊它們，以它們的力量來複興故國。本作故事，就隨著這場艱辛的復國之旅，逐頁展開。

記得在月河村初遇小雪，她那滿頭的白髮刺痛的我的心，她的村子被村人祭祀的「河神」所摧毀，村人把責任全部歸咎於她，她因而被永遠逐出了自己的故鄉。後遇到拓跋玉兒，她冷靜聰敏，行事磊落，爽朗的性格又深深打動了我。

這款遊戲一進入就讓人眼前一亮！水墨畫風格的畫面在那個年代是十分驚艷的！那個時代並非沒有3d作品，但是當時的3d技術極其不成熟，大宇放棄不成熟的3d技術來討好玩家而採用自己擅長的2d畫面是非常明智的決定！另一個驚艷的地方應該是「耳前一亮」，十分應景的音樂讓人十分舒服！

劇情方面可謂代入感十足，讓人忍不住潸然淚下，雖然此作劇情沒有正傳大氣豪邁，但是其中的愛恨糾葛不正是當年的懵懂少年所期盼的嗎！

戰鬥方式依舊是回合制，但是充滿中國風的動作與技能也讓人愛不釋手！當時電腦並不普及，很多玩家都是剛才紅白機轉型過來的，對於這樣的戰鬥方式已經很滿足了。

本作還有一個極具特色的系統，就是十大神器之一的煉妖壺，在其中玩家可以將收服的妖物、得到的裝備或藥品合成出一個新的妖物或物品！這個系統在現在並不算什麼，但是放在當年就是一大創新！讓人慾罷不能！

最終赤貫星被眾人合力成功封印，大家回到人間，已是十年之後。本作為雙結局，根據人物的好感度來觸髮結局，但是不論是誰活了下來，我都很難接受另一位女主的逝去，每次玩到結尾，我都會有一種深深的無力感，只是懷念他們一起的三個人的時光。在小編心中，這款遊戲正是單機遊戲史上最巔峰之作！也是許多人的青春！
