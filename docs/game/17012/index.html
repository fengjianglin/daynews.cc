「遊戲推薦」爽快的2D橫板像素風彩虹六號--破門而入：行動小隊

2019-12-14 2,911條評論 4,216次閱讀 1144人點贊

遊戲名：《破門而入:行動小隊》（Door Kickers: Action Squad）

遊戲類型：橫板、動作射擊

警匪題材娛樂作品中的SWAT特警隊，經常是警察一方除了主角外的最強戰力。他們訓練有素，像機器一樣精準。攻堅，解救樣樣精通。可惜在戰術射擊遊戲中，我們在突擊耍帥之前往往需要進行大量的前期準備。有沒有一款輕量級的遊戲去除掉前期準備與行動規劃，把整個遊戲流程濃縮到踹門突擊這部分呢？

曾經開發了俯視視角戰術遊戲《破門而入》（Door kicker）的Killhouse Games工作室再度歸來，與Pixelshard聯合製作的《破門而入：行動小隊》在更新了多名可選角色和新的章節後，於近日結束了Early access。本作在最初揭露時玩家反映可謂是嘩然一片，因為前作《破門而入》可是正兒八經的回合制戰術遊戲。

精美的像素畫面第一眼看上去像是《Broforce》，實際玩起來卻像是橫板的《彩虹六號：圍攻》獵恐模式。

玩家將扮演SWAT小隊，從五名功能各異的特警中選擇適合自己的幹員。攻擊手裝備了自動武器，能用精準射擊狙擊匪徒、破門手使用威力巨大的霰彈槍，可以直接轟開要踹上好幾腳的安全門、盾牌手能為隊友提供足夠的保護、偵察員則善於發現敵人和標記位置，遇到室內情況不明的房間還可以用窺鏡偵察敵情、像是隔壁FBI部門過來的的鐵娘子菲亞探員還可以雙持武器，左右開弓。

通過完成任務，玩家可以根據評價獲得為幹員解鎖新武器和裝備用的星星。閃光彈，防彈衣，煙霧彈等五花八門的裝備對安全過關作用很大，隨著升級，玩家還可以點亮幹員自己的技能樹，讓自己的慣用幹員更加強力。

作為一個踹門遊戲，各種不同的門，與門互動的不同方式就是遊戲的主要機制。悄悄打開、用霰彈槍噴開、畫上點時間踹開、或者拿破門炸藥炸開，順便震暈屋子裡的歹徒，這些都需要觀察門後的情況來選擇合適的方式。

然而關卡中還存在少數封閉的小黑屋，後者裡面等待著你的可是未知的情況。拿著AK的暴徒，綁著炸彈的自爆客，還有可能只有某個倒霉的人質，一開門就被過度緊張的你爆了頭。

在經歷了前期誤殺人質，死於亂槍的挫折後，很快我就能享受其中。復古街機風格的橫板射擊玩法容易上手，配合精緻武器聲效與動畫，像素級的暴力表現讓踹門突擊的感覺加倍刺激。

在快節奏的80年代風格合成器朋克BGM的鼓舞下，我經常踹門，掃射，踹門，掃射……一不小心忘了下房間還有人質，為了獲得完美評價又要重來。不過遊戲優秀的聲效，極其精美的霓虹+像素風格畫面，讓失敗的挫折感化為烏有。

本地分屏或線上合作功能則讓這場快樂的踹門之旅更加不正經，和朋友在選擇幹員界面還討論著戰術和出裝，一副煞有其事的樣子。一進入交火就原形畢露，各行其是，輪流水槍，搶殺人質，互相丟鍋。好吧，又是一款「友盡」佳作。

因為武器可供選擇較少，遊戲後期解鎖裝備用的星星完全富餘了，讓人提不起打高評價的動力。總用一把武器也難免有些單調，可別的武器往往又差距過大，不值得使用。還好多種多樣的敵人類型豐富了後期體驗。前期雜兵一敲門還會傻傻的來查看，後期敵人一旦被驚動就會尋找掩體，架槍迎接你的到來。除了解救人質外，遊戲還有限時拆彈，清掃房間，抓捕嫌犯等任務關卡等待挑戰。在清掃房間任務中，你終於可以把人質忘在腦後，痛快的踹門突擊，感受像素級別暴力表現的刺激與幽默。​​​​
