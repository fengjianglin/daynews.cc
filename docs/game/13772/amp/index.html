爐石傳說：不止酒館戰旗能套娃！新版本紅龍無限套娃我生我自己

天天要聞
2019-12-13

在爐石傳說的酒館戰旗模式中，我們會將亡語套路稱作是套娃！特別是有時候我們的亡語隨從出亡語隨從再出亡語隨從，甚至能讓我們必輸的局起死回生，因此在酒館中關於亡語操作也成了大家調侃為無限套娃流！

而隨著新版本「巨龍降臨」的上線，有玩家驚喜的發現不止有戰旗中能出現套娃，現在天梯中也可以玩出套娃套路了，這時候你僅僅只需要一張新版本的中立橙卡紅龍女王。新紅龍身材還是9費8/8，效果變成了如果你的套牌沒有重複的卡牌並隨機將兩張龍牌置入你的手牌，而且其消耗降為0點，簡單點說只要你運氣一切皆有可能！

在實戰中，有玩家先下銅須讓紅龍的戰吼可以觸發兩次，給到的四張龍牌中有兩張是紅龍本體！也就是說我們在下回合還可以繼續置入0費的龍牌，紅龍找紅龍再找紅龍對方根本解不過來！所以只能說不管是天梯還是酒館戰旗，套娃套路都很強！

那麼問題來了，既然新版本的這些卡牌都非常強有木有一套卡組可以用來上分呢？下面就給大家推薦一套新版本的戰吼薩，這套薩滿的構成很簡單基本都是之前戰吼薩的套路只是加上了薩滿的迦拉克隆的體系。

玩法也非常簡單，前期我們就是依靠戰吼薩的資源來打用來完成任務。後期銜接迦拉克隆的資源來打，畢竟「祈求」機制的可以使完全體薩滿迦拉克隆召喚2個8/8的突襲隨從，加上我們的任務技能可以一次召喚4個8/8突襲，這什麼場面解不了？而且薩滿迦拉克隆變身的技能也非常強大，不管是解場還是鋪場都非常不錯。最後我們還可以使用沙德沃克放場電影，這一頓操作不僅實用還炫酷上分妥妥的！

而這些所有的基礎都是新版本的迦拉克隆體系，那為什麼迦拉克隆這麼強呢？個人覺得有幾點原因，首先是迦拉克隆自帶的「祈求」機制太強了，每一次的「祈求」所帶來的提升肉眼可見。再加上新版本眾多的「祈求」卡很容易將迦拉克隆進化到最強狀態，新版本收益最大的一個體系了！

迦拉克隆體系大多數都是戰吼而且迦拉克隆而是自帶戰吼，這一點又和戰吼薩這種成熟的套路非常契合。最後就是薩滿迦拉克隆的技能確實是非常實用而且每個階段都有用的技能，這一點使得薩滿的迦拉克隆再怎麼也不會虧。「祈求」機制的強大加上這些點結合在一起便促成了現在戰吼薩強勢的局面，實戰中這套卡牌算得上目前最出色的卡組之一！
