Karsa轉會仍被暴打，SN強到不需要JKL？

2019-12-12 461條評論 3,127次閱讀 901人點贊

轉會期各家歡喜各家愁，有些粉絲希望戰隊能保持現狀再創佳績，而有些就希望隊伍簽下強C明年再戰，那麼後者一定有TES的粉絲。

12月6日，滔博官宣前RNG打野選手Karsa加入戰隊，聽到這個消息的粉絲都樂開了花，評論下方一片祝賀，大家都覺得他的加入一定讓戰隊如虎添翼更上一層樓。 粉絲的期盼還是很有依據的，首先，戰隊在S9的表現不錯。 滔博的S9歷程止步於9月1日，那是一場與RNG對爭世界賽名額的生死戰。面對這樣的世界強隊，前期被2比0壓制，滔博依然迅速找回心態調整戰術在第三局中獲得勝利。雖然最後3:1落敗，但一路走到世界賽大門已經屬實不易，這隻隊伍的成長受到大眾的肯定。

其次，Karsa的S9戰績也令人滿意。 在世界賽與SKT的對陣中，雖然RNG戰績不佳，但憑藉著他盲僧多次優秀的迴旋踢，穩住了RNG的中期節奏，擾亂了SKT的下路，妨礙了對方抓下的計劃，更是阻止了SKT的幾波大龍推進。

總的來說，滔博現在的隊伍配置可以說是S10一流的；打野位置Karsa的加入補強，上中是國產頂級選手369和Knight9，就連二隊都有天才上單bin。 但是誰也沒想到，官宣沒多久，他們被SN打爆的消息傳了出來，粉絲的房子好像塌了。

在近期訓練賽中，滔博的現任成員，包括已經正式訓練的Karsa，對陣SN中單Angel帶領的二隊上單和ADC。新人上單完美壓制369，在中野配合方面，angle和sofm也與Knight和卡薩起鼓相當，最終SN不僅幾次都獲得勝利，甚至瑞茲還拿到五殺。

雖然是訓練賽，但其實兩隻隊伍在今年夏季賽中也有交手，結果就是滔博被零封了！ 第一局，TES紅色方，SN的奧拉夫在第五分鐘拿下對面狗熊一血，第八分鐘SN收穫第二個人頭，十三分鐘拿下火龍，不僅在隨後的團戰中二換三贏得勝利，三十一分鐘就推掉對方水晶拿下一比零，人頭對比更是慘不忍睹！

第二局，紅藍沒有交換，滔博的野區開局三分鐘慘遭入侵，第七分鐘塞拉斯在對面拉克絲和卡牌的圍攻下送出一血，沒過五分鐘，隊友寒冰再送人頭。SN的配合和支援能力在這場對局中展現得淋漓盡致，幾波團戰SN毫無收穫。三十分鐘對面就來到自家中路高地，三十三分鐘的時候下路也徹底崩塌，雙方經濟差過大已是無力回天，比賽毫無懸念拿下。

LPL官方解說CA也是對SN更有信心，更是在昨晚的直播中直言「SN現在已經很強了，不需要JKL，花四千萬買他不一定值」，他更是爆料有業內人士告訴他JKL以及和IG簽約， 不知道滔博上次對戰SN的戰績如此之差，簽下Karsa以後能不能在賽場上完成復仇呢？去向至今未知的JKL真的不值四千萬么？
