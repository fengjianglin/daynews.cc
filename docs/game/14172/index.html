射手之王UZI再度衝擊全明星冠軍，即將擁有三個獎盃的常勝將軍

2019-12-10 154條評論 1,575次閱讀 213人點贊

關注英雄聯盟賽事的小夥伴們最近可能都被全明星賽刷屏了，作為一年一度的大事件，全明星賽不僅代表著選手們技術的頂峰，也代表著人氣的頂峰。例如UZI這位蟬聯兩次全明星賽冠軍的超人氣選手。在今年的全明星賽上UZI也是發揮出色，今天嗶哩君就帶大家看一看這次賽事上UZI的發揮吧。

一、Uzi 韋魯斯 vs Bang 亞索

今年的全明星賽事可謂是看點頗多，無論是特殊設計過的「突然死亡模式」還是賽區對抗賽都有著精彩的鏡頭，今年的solo賽與往常不一樣，這次的「突然死亡」讓很多熟悉嚎哭深淵的觀眾們吃了一驚，第一次擊殺不算輸贏，只有突然死亡來臨時的第二次擊殺才算勝利，這讓職業選手們也重新考慮自己的打法，為了能夠獲勝而絞盡腦汁。說到選手們的1v1單挑，就不得不提我們的「射手之王」——UZI。

作為連續兩屆冠軍的得主，簡自豪可謂是這個行業內最有經驗的人。這次賽事上最讓人激動的就是UZI和SKT的冠軍AD——BANG的對決。在這次對決上BANG選擇了非傳統下路–亞索，而UZI還是堅持了自己的信仰，選擇了傳統射手維魯斯。在這次對決上我們看到了熟悉的身影，UZI的操作讓很多觀眾歡呼「我的青春回來了」。雖然兩位選手都有著很亮眼的操作，但是UZI最終還是以熟練的基本功戰勝了對手，拿到了通往最終比賽的門票。

二、射手之王—UZI的連勝之路

在這次SOLO賽中，我們的射手之王可謂是出盡了風頭。無論是對陣哪一位選手，他都臨危不懼，選出自己最信任的英雄。薇恩、維魯斯等英雄在他的手下展現出了驚人的動作，補刀細節、對細節的把控、技能的拿捏都讓我們看到了那個在賽場上叱吒風雲的小夥子，在對決中他戰勝了老對手BANG，新對手LEVI，再度打通了今年的冠軍之路，未來將如何發展，還是要看我們信任的UZI。

三、隨口說出的一句話引起節奏

賽後採訪時UZI一句無心的話引起了很大的節奏，在接受余霜的採訪時，他說到:在這最後一刻還能享受比賽的樂趣，是非常開心的。這句話頓時就登上了微博熱搜，希望簡自豪這句話並不是暗示退役，畢竟我們還是希望他能夠在世界賽上繼續證明自己，拿到自己的冠軍皮膚。

這次全明星賽讓我們看到了職業選手們的另一面，他們在賽場上認真拼搏的精神和在全明星賽上友誼第一的精神都激勵著粉絲們，讓粉絲們享受著這一年一度的盛事。希望各位職業選手能夠享受比賽，為明年的競爭做好準備，為我們帶來一場又一場激動人心的比賽。
