如果能重來，不一定會打職業，Meiko：這是一條付出青春的死路

2019-12-11 1,681條評論 4,001次閱讀 2030人點贊

大家好，我是愛玩遊戲的小蘿醬，每天分享最新的遊戲資訊和有趣的遊戲內容，嗯哼~

站在舞台上的職業選手是光鮮亮麗的，尤其是獲得勝利享受冠軍的那一刻，但在茫茫多的選手中，只有一小部分能完成這項成就，更多的還是在扮演著追夢人。對於Meiko來說，LPL的冠軍他拿了不少，也獲得了首屆MSI的冠軍，但在人人都嚮往的S賽上，他的最好成績只是八強，更加遺憾的是，Meiko所在的EDG缺席了今年的世界賽，粉絲們對這支戰隊的希望只能寄託到明年的S10。

付出青春的職業道路，不是孤獨的路，而是一條死路

在EDG俱樂部發布的《如果·EDG》Meiko篇的預告片中，簡單地講述了Meiko對自己職業生涯的看法，雖然預告片很短，所提到具體的東西也很少，但卻都是一些重點。「我已經付出我的青春了」，Meiko在一開場就說出了所有職業選手的心聲，他感覺自己所走的職業道路都不是孤獨的路，而是一條死路。

流水的AD，鐵打的輔助，初入LPL的Meiko就加入了EDG，如今即將迎來第五個年頭，自己也從年輕的小將成為了隊長。這麼多年的職業生涯一定是讓Meiko有所成長的，但同時也磨平了他的銳氣，現在的表現雖然也非常不錯，但總感覺失去了初登賽場時的那種驚艷。輔助的AD選手換了一個又一個，隊友也走了一批又一批，只有Meiko堅守在自己的崗位上，這也讓他在LPL成為了勝場次數和助攻次數最多的選手。

這些數據不僅表明Meiko是一名資歷十足的老將，更記錄了他的青春，一生中最令人回味的那幾年寶貴時間，Meiko全部用於去打職業。職業道路就是如此，成績決定著一切，敢走上這條路就要抱著走到黑的決心，像是參與了一場關於人生的賭博，只有成功者才能看到未來的曙光。幸運的是Meiko打出了不錯的效果，雖然沒有走到終點，但已經超過了很多的選手，那些人依舊在這條路上摸索前行。

要是能重來，不一定會選擇打職業

Meiko回到了自己的學校，舊地重遊自然感觸良多，如果當初自己沒選擇打職業，沒去EDG，現在應該還在上大學。打職業和上大學，這是兩種完全不同的道路，也將擁有兩種不同的人生，沒法比較到底哪個好，因為不知道在另一條道路上到底能走多遠。但有一點是可以確認的，打職業是比較枯燥的，尤其是在要求嚴厲的EDG，而大學生涯相對豐富多彩，將來的發展也有很多種選擇。

或許正是這些原因，讓Meiko在重來一次的問題上產生了猶豫，沒能堅定的選擇依舊踏上打職業這條道路。得不到的東西是美好的，就像很多玩家羨慕職業選手，幻想自己坐在電競椅上打比賽，會如何如何，但青春只有一次，不管怎麼做，選擇幹什麼，都應該付出自己全部的努力。

Meiko是眾多職業選手中的一個縮影嗎？或許是，很多人都像他一樣努力，為了最初的夢想苦苦堅持，又或許不是，Meiko的今天比大多數選手都要成功，未來的路也更加好走。

初心長遠的EDG，何時才能圓夢

最後要說說EDG，這支百戰千回，盛名難負的戰隊，曾經無數次站在LPL的頂峰，但如今的統治力早已大不如前。這個冬季轉會期也遲遲沒有消息，讓粉絲們不禁擔心，S10的EDG能力戰群雄，完成廠長給大家的承諾，打入世界賽嗎？

正如Meiko之前所說，「漫漫長路，甘為學徒」，EDG雖然不是LPL的新軍，但必須要以挑戰者的姿態面對任何一個對手。S賽的冠軍是EDG和所有隊員的初心，但只是想要觸碰它就任重而道遠，希望Meiko和EDG在S10加油，早日實現自己的夢想。
