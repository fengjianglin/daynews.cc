這個遊戲好玩嗎：互動電影遊戲《晚班》，新體驗！故事由你決定

天天要聞
2019-12-05

摘要

　　很多人接觸互動影視、互動劇或是互動式電影遊戲這個概念，可能是從Netflix的《黑鏡：潘達斯奈基》開始，你也許聽聞過大火的國產作品《隱形守護者》或者玩過《底特律：變人》。但這裡我們只探討真人實景，有演員出演的互動視頻作品。

　　我最早接觸的是這部《Late Shift》，中文譯名《夜班》，但更多人叫它《晚班》。我在2017年末，差不多是兩年前這個時候有幸在steam上看到這款遊戲，然後果斷玩了一夜。玩後興奮異常，覺得這就是未來電影遊戲化的發展方向。

　 《晚班Late Shift》是CtrlMovie製作的一款實時電影互動式犯罪解謎遊戲。該片預算150萬美元又叫《一個瓷碗引發的血案A Case of A CiWan》，故事圍繞一個晚明時期的瓷碗在拍賣會上被盜展開。無辜學生Matt在上夜班時莫名捲入其中。瓷碗看似成功搶劫，卻招來了殺身之禍。玩家必須在劇情中做出種種選擇，周旋在警方和各黑幫之間，發掘整件事情的真相併且努力活下去。

　　故事中，玩家就是主角Matt，為了洗刷冤屈證明自己的清白，需要通過無數次的選擇與判斷逐步揭開隱藏的真相，遊戲有數條劇情線，7個結局，最終收穫Bad ending 還是Happy ending全靠玩家做出的每一個選擇決定。

　　本片跟《隱形守護者》不同，是真人實景全動態影像，即使在做選擇中，電影也在流暢播放，不停頓不回放。

　　本作劇本由《大偵探福爾摩斯》（2009年電影）作者創作，取景於英國倫敦。 片中玩家將面臨180多種選擇，含中文字幕，玩家自行選擇偷車還是逃跑？是親她還是不親她？（還真有人選擇不親，哈哈）

　　總之，一個小時左右，你可享受到電影、遊戲、互動敘事三者混合的超凡體驗，以及扣人心弦的劇情故事。這就是互動視頻的最大特色，主宰劇情走向，你是這次故事的上帝。

　　影片在製作上，有寶馬贊助加持，場景華麗、氣氛到位、演技至少不掉線，還有航拍和撞車段落，頗有大片既視感。劇情上，有中國元素，還有餅乾塞紙這種小心思，就是高潮缺乏大場面，後勁不足。三刷四刷之後無法快進。如果單做成電影，毫無特色可言，但電影遊戲化，可互動之後，不乏是一次有趣的體驗。

　　如果你還沒有體驗過這類互動視頻，可以趁著秋促或聖誕打折季剁手。不到一張電影票的價格，在Steam、Switch、PS4、Xbox、 電腦、手機上都可以玩到《晚班》。

　　如果你很喜歡這類的互動視頻，記得關注我，告訴我你最喜歡哪一部作品，近期我會發片單、發視頻試玩這些互動影視。
