爐石傳說：新版本各大職業最受歡迎橙色龍卡盤點

2019-12-17 952條評論 4,507次閱讀 2650人點贊

Hello各位爐石玩家們，巨龍降臨版本已經更新了數天了。基本上現在的各個模式下的卡組也逐漸穩定了下來，不再有最開始的群魔亂舞奇招頻出情況出現。但既然是巨龍降臨版本，卡牌多少都是和巨龍有關係的吧？今天我們就來盤點一下最近這幾天的各大職業人氣TOP龍卡吧！

瑪里苟斯這張卡還算挺好用，畢竟如果手中有龍牌則發現一張升級過的法師法術牌。這也讓龍法的玩法更加可玩。在目前大部分情況下都是宇宙法的環境下，龍法這個隨從卡確實是非常超模了。5費的標準花費下，卻有著高達8血的身板以及白送一個強化法術卡。雖然說這個法術卡只能發現基礎法術，但是勝在是強化過的啊！因此各方面上都算是非常好用的存在，直接導致了這張卡加入構築的幾率超過了80%，當之無愧的冠軍。

德魯伊：覺醒巨龍伊瑟拉，9費4/12。進入構築的幾率比起法師就低了許多，不過也有著46%也算是人氣不低的存在了。這張卡主要是打後期的存在，後期只要抽到一個夢境之門就能召喚一條龍。而且7個召喚機會，召喚出來一個大表哥就是炸盤的存在了。但是弊病也很明顯，9費超高的花費以及需要抽到夢境之門才可以抽龍卡，這讓它必須考慮如果度過前期安全發育到後期。但是7次抽選的機會只要抽到王炸也讓這張卡有不少德魯伊選擇帶他，誰不想當天選之人呢？

獵人：維拉努斯，6費7/6，效果簡單粗暴但是究極炸胡。直接把敵方所有隨從的生命值強行變成1點。也因為這個超標效果，許多獵人都選擇地上這張卡因此有著41%的構築幾率。非要說的話這張卡無論是花費還是身板上都可以說是標準，而且具有強大的控場能力。你想想一上來對面全1血，不是逼著對面放棄原有的進攻手段轉成自保手段嗎？不然眼睜睜看著一個解場全炸不是哭死？無論是宇宙獵還是奧秘獵，即使是只帶上這張卡也是非常值得的。

術士：扭曲巨龍澤拉庫，8費4/12又是個後期卡。構築幾率是40%，也不算低。雖說這張卡效果也不錯，英雄每次受到傷害都可以召喚一個6/6的虛空幼龍。算是挨打就讓場上多一個輸出存在，你越打我我這邊輸出越多。但說實話我不太覺得這張卡效果有多好，因為花費太高而且需要挨打才有顯著效果。總體上來說還不如帶上動物園比較實際，畢竟這張卡還是比較吃站場。

牧師：永恆巨龍姆諾茲多，8費8/8身板。構築幾率就一般般了，也就30%吧。這張卡費用略微高一點，但是身板也算不錯有著8/8。關鍵是戰吼效果也很IMBA，複製上一回和對手使用的所有卡牌。先不提實用性，光是只要對方用過不錯的卡到你這邊就是血賺了。在對面用過法術和大卡的時候下一個，基本上可以讓對面懵逼。但說實話這版本牧師表現並不算很強，這張卡實際上能出現的幾率不高。

盜賊：蠟燭巨龍，5費7/5身板，構築幾率16%。不用說都知道效果很IMBA的存在，先不提高達7點攻擊5血標準身板就有多賺。關鍵是這個亡語效果，只要觸發了就會放入牌庫。只要你再抽到就能再度召喚，再死再抽。到底有多恐怖？只是這張卡還是比較怕沉底的，就即使如今勝率較高的亡語賊都不怎麼帶上這張卡。

聖騎士：4費8/8究極超模身板，構築幾率是16%。說實話戰吼確實是還挺好玩，把水晶強制重置為10個。可以打對面一個措手不及，特別是你這張卡是8/8相當於對面會有很多計划行不通，從而不得不改變計劃。但是說實話，有利就有弊。這張卡雖然可以帶節奏但是副作用也很明顯，容易把自己搞崩。因此大部分情況下，不是很多人會選擇選這張卡。

戰士：瘋狂巨龍死亡之翼，8費12/12，構築幾率13%。就身板而言確實是超標的存在，而且攻擊其他所有隨從。說白了這玩意就是個敵我不分的存在，但是打一口傷害有很高的。不過如果用在防禦戰卡組裡面倒是很不錯的存在，問題是這版本防禦戰被針對了多了一個摧毀護甲的卡牌。直接讓防禦戰這個玩法暴斃，因此這張卡也從超標存在變成了一個隨機炸彈級別的存在。也讓它很難加入構築裡面，畢竟12點攻擊力實在是太恐怖了。

薩滿：尼索格，6費5/5的卡，構築幾率慘不忍睹的3.9%，但是有意思的是帶上這張卡勝率有著53.5%。召喚兩個0/3的龍蛋，下回合變成2個4/4的帶突襲幼龍，效果上而言確實是不錯。 不管是軟嘲諷效果還是鋪場上而言這個卡還算OK，問題是這張卡實在是生不逢時。因為這版本的卡很多都比較超模，對比起來這張卡其實也就那樣了……
