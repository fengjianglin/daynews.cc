寥寥數年，彈指一揮，今朝江湖再會，自當把酒言歡！《笑傲江湖OL》年末回歸活動來襲，曾經的緣起沉浮，攜手相伴的朝朝暮暮，都化為記憶中你凌波而來的身影。12月18日年末回歸活動即將開啟，俠客夢長，你身邊是否有許久未歸的小夥伴？趕快喊上你的好朋友一起再戰江湖吧。

俠客回首，再戰江湖

活動一、超強屬性助力闖蕩江湖

活動時間：12月18日（維護後）至1月15日（維護前）

活動限制：不限等級，正式服與經典服皆可參加

活動內容：

活動期間內，現有角色與新創建角色在維護後登陸遊戲都可獲得全新助力Buff,從而全部提升角色的攻擊、氣血、治療等多種屬性，堪比一套3級寶石套戰力，讓您不管是回歸經典服重新入駐江湖打怪升級，還是一直奮戰在遊戲內的少俠都能有暢快的遊戲體驗。

獨孤求敗

活動二、緊跟大部隊，福利等級奉上

活動時間：12月18日（維護後）至1月15日（維護前）

活動限制：不限等級，僅限經典服新老玩家

活動內容：

活動期間內，經典服四個伺服器都會設有不同的伺服器福利等級，低於伺服器福利等級的角色（含新創建角色）在登錄遊戲後，都將自動獲得福利等級助力BUFF,從而獲得30倍經驗效果，直到角色等級提升至福利等級後失效。

志氣相投，江湖同行

福利等級限制：

經典一區【風雲再起】福利等級為89

經典二區【一戰成名】福利等級為82

經典三區【寵你所愛】福利等級為82

經典四區【桀驁不馴】福利等級為78

福利等級說明：

在福利等級Buff下獲得的經驗無法用於銘牌培養、萌娃養成，僅限提升人物角色等級使用。

活動四、回歸經典，專屬特權

活動時間：12月18日（維護後）至1月15日（維護前）

活動限制：

針對回歸經典服的老玩家，且之前正式服或經典服有創建角色等級≥20級，超過60天未登錄的賬號，可用之前的賬號中原有經典服角色，或重新在經典服創建角色登錄遊戲，都可獲得回歸專屬特權。

1、回歸專屬活動：

擁有回歸特權的賬號使用原有經典服角色，或重新創建經典服角色，均可在登錄遊戲後彈出專屬任務框，按照提示完成任務，即可得到以下專屬獎勵。

大俠回歸，領取好禮

專屬文字稱號：【摸爬滾打過來人】、活力草50*5、 三級大還丹*100、

回歸紅包*1（能開出40或888寶銀）

2、回歸專屬標誌：

滿足回歸條件的玩家擁有專屬的回歸標誌，頭頂角色名旁邊會有明顯的”歸”字，讓您一眼就能找到哪些是回歸的老玩家，讓我們一起看看裡面有沒有你曾經的戰友呢？
