LPL最菜戰隊提前誕生！訓練賽被打成自閉，沒有隊伍願意和他們打

2019-12-18 2,177條評論 3,636次閱讀 1209人點贊

LPL近兩年的戰隊數量可以說是越來越多了，在有了這麼多支戰隊的情況下，必然會出現一些實力非常差的隊伍，沒有辦法晉級季後賽甚至進入前十名，這樣的戰隊大多數是由新人組成的，因為轉會期一方面搶不過其他隊伍，另一方面可能也沒有那麼多的資金。

這樣的隊伍想要重新崛起難度還是比較高的，可能需要首先培養出幾個有潛力的選手在賽場上打出好的表現，然後再通過花錢來補強戰隊配置。

因為在英雄聯盟發展到S10賽季的時候，已經很難有一支新的隊伍直接花錢簽約五個頂級選手了，因為轉會期的頂級選手少之又少，就算是你有大把的金錢，也沒有辦法支付起這樣高額的轉會費或者簽約費。

今年轉會期中新加入LPL的隊伍Estar很可能就是這樣的一支隊伍，如果在今年的S9上尋找一支和Estar類似的隊伍的話，可能就是V5了。

眾所周知今年的LPL職業聯賽中排名倒數第一的是VG戰隊，然而VG戰隊現在已經開始組建新陣容準備起飛了，因此明年實力最差的隊伍大概率不會是VG。

昨天晚上，在V5戰隊執教過的暴躁教練涵藝透露，PDD的Estar戰隊實力很差，在訓練賽中被其他戰隊暴打後，甚至沒有隊伍願意和他們打訓練賽，這個爆料側面證明了Estar的確是沒有買到好的選手。

PDD這邊作為前YM的老闆，培養新人選手是有一手的，但是大多數的優秀選手都被他送了出去，現在的情況是只能從長遠的角度考慮，繼續以PDD的那一套來培養出更多優秀的天才選手出來。

因此明年的LPL春季賽，Estar的成績可能會非常差，不過相信PDD也能預知到這樣的結果，關鍵是要看在這樣的一個賽季中，選手能不能通過高強度的比賽快速提升自己，使自己成為一個優秀的選手。

如果能有1-2選手起來的話，Estar就可以在轉會期中局部補強慢慢地提升排名，這其實就是一支隊伍從年輕隊伍成長為頂級隊伍的過程。
