冬冠杯對陣出爐，KPL戰隊成為最大贏家，期待決賽AG和eStar會師

2019-12-24 1,240條評論 2,837次閱讀 459人點贊

陌陌聊王者，給您帶來精彩的冬冠杯資訊！

8支戰隊在經過了三天時間的爭奪之後，最終獲得兩個出線名額的戰隊是TS和eStar。在選拔賽結束之後，進行了本屆冬冠杯的抽籤。

5支KPL戰隊，他們分別是AG超玩會，QGhappy，JC，TS和eStarPro；3支KAKPL分別是SLT，ROX和GOG。最後的對陣是令KPL戰隊非常高興的。

​

作為KAKPL的冠軍戰隊SLT和KPL的冠軍戰隊AG超玩會，按照賽事的規則，他們在總決賽之前是避開了。最終的對陣形勢是：SLT對陣eStar；AG超玩會對陣ROX；JC對陣TS以及QGhappy對陣GOG。

從分組的情況上看，KPL的戰隊在每一個小組中都有戰隊，這就意味著在半決賽前KPL戰隊的內戰是分開的。很多程度上來說，四強基本已經是KPL內定了。

​

eStar有驚無險的進入到了正賽，可以這是所有網友們都非常希望看到的一幕，比較eStar的實力是有目共睹的。

如果他們能夠戰勝SLT的話，他們的對手將是KPL的JC或者是TS，這對於eStarPro來說都是可以接受的，他們很有可能保證了一個決賽的名額。

​

AG超玩會作為KPL的新科冠軍，首輪戰勝對手應該問題不大。在他們這個區里，另一個KPL的戰隊是此前KPL總決賽的對手QGhappy，這對於AG超玩會來說壓力還是非常大的。

雖然AG超玩會在總決賽的時候4比1乾淨利落的拿下了QGhappy，這一次如果兩支戰隊再一次在半決賽相遇，鹿死誰手還是一個未知數，畢竟QGhappy有著冠軍杯的經驗。

​

不過話又說回來，電競賽場上什麼事情都是有可能發生的，KAKAL經過了這段時間的發展，他們的進步是有目共睹的。在選拔賽中，RW俠輸給了CW就是一個很好的證明。

無論你是eStar還是AG超玩會，如果對你的對手不重視的話，那將有可能會被對手淘汰出局。不過呢，網友們還是希望在決賽中能夠看到AG超玩會和eStar的再一次交鋒，畢竟這兩支戰隊在此前的季後賽敗者組半決賽中打出了非常精彩的一場比賽。
