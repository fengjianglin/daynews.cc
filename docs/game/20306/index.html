TGA 2019 獲獎名單公布，宮崎英高榮登神壇

2019-12-17 1,668條評論 3,049次閱讀 1226人點贊

北京時間12月13日，享有「遊戲屆奧斯卡」之稱的The Game Awards（TGA）在洛杉磯微軟劇院舉行了頒獎典禮。獨立遊戲《極樂迪斯科》攬下了最佳獨立遊戲、最佳角色扮演、最佳新晉獨立工作室、最佳敘事四項大獎，而「年度遊戲」最後頒給了From Software的《只狼：影逝二度 》。

以下是具體獲獎名單：

遊戲類

年度遊戲：《只狼：影逝二度》

即便頂著「受苦」、「硬核」的名號，宮崎英高的新作《只狼》還是贏得了多數人的認可，開創魂系玩法的from software並沒有拘泥於已有的輝煌，《只狼》在給玩家帶來新玩法的同時，也延續了宮崎英高一徑的「痛並快樂著」的體驗。
