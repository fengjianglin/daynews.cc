王者模擬戰：長城被削至殘疾！蜀國崛起，或將成為一大熱門？

2019-12-24 818條評論 2,143次閱讀 837人點贊

本文原創首發於公眾號：電競怪客

hello大家好~我是怪客君~關注並私信我「戰令」即可領取戰令進階卡！

12月24日的全服更新對模擬戰也有不小的改動，怪客君就帶大家來一起來看看吧~

諸葛亮調整

怪客解析：諸葛亮驗屍官的身份一直以來是玩家所頭疼的，這次加快諸葛亮的回藍也許能夠讓他擺脫這個身份，成為致命收割機。配合蜀國羈絆，這回藍速度簡直猛如虎！

周瑜調整

怪客解析：周瑜本次大幅度增加了傷害，但因為他的傷害和控制太不穩定了，且法刺比他厲害的也有一大把，本次加強依然對吳國法師流沒有太大幫助。

裴擒虎調整

怪客解析：對冷門棋子的小幅度增強實際上對整個模擬戰流行沒啥影響，可以考慮加入刺客陣容。不過想玩魔種羈絆這樣「歪門邪道」的玩家應該會對本次加強感到開心，因為本次還對魔種的一員：玄策也進行了不小的加強。

百里玄策調整

怪客解析：小瘋子百里玄策在12月3日遭遇過一場大削，斷了他一臂。

（玄策12.3調整）

官方也終於發現玄策現在的尷尬處境實在很難對得起他三星牌的身份，這次給他增加了雙抗，提高了他的存活能力。這是對刺客陣容的一次加強，不過刺客陣容依然很弱，不只是玄策較以前相比依舊是削弱狀態的原因，還跟婉兒數次被削以及模擬戰當今戰士橫行射手遍地的環境有很大關係。

劉備調整

怪客解析：工具人劉備的攻擊得到了加強，不過幅度不是很大，依舊擺脫不了前期打工仔的地位。

長城羈絆調整

怪客解析：長城羈絆這一次可以說被削廢了，不過怪客君認為在當今這個拿到羈絆進階就想玩長城的時代，為了平衡與增加選擇，長城是真的該削。

總結

削弱：長城養豬流（大削）、長城坦射（小削）

增強（小幅度）：魔種、蜀國、刺客流、法師流

這次的模擬戰更新相比是給當前比較火熱的長城羈絆來了一記重拳，不過坦射依舊是當前版本的T0陣容。本次更新還有兩個蜀國的棋子都得到了加強， 是否會給蜀國羈絆增加一些熱度？怪客君接下來給大家推薦一個蜀國陣容：七蜀下天山。

七蜀天下流

1.陣容選擇

有青春之舞時：不要元歌、黃忠

沒有青春之舞時：不要劉禪，沒經濟拿元歌，有經濟拿黃忠。

核心：諸葛亮、張飛、呂布

2.必拿天賦

青春之舞：可以說是七蜀最核心的天賦了，高星蔡文姬的回血能力是大家有目共睹的，搭配青春之舞，可以讓張飛和諸葛亮如入無人之境。

天賦異稟：增加青春之舞的出貨幾率。

3.運營思路

前期先上三蜀國，最好五回合有二蜀國，因為蜀國不是版本強勢羈絆陣容，所以前期就必須有戰鬥力。由於核心牌都是高費，儘快提升人口，六人口之前不要搜牌，六人口時保利息搜到五蜀國，穩定自己的戰鬥力之後繼續升人口，到七人口之後升質量。核心牌在七人口有二星就可以升八人口成型了，升質量的同時找武則天，武則天有二星就可以升九人口了。

好啦，以上就是今天王者模擬戰更新的全部內容以及七蜀下天山的玩法，對這些你還有什麼想說的？歡迎留言評論哦~
