1. 【使命召喚】肥皂之死

相信在每一個男生的成長過程當中都會接觸一款槍戰類遊戲，而且我自己最喜歡的並不是多人對戰的CF之類的，而是單機遊戲，其中最讓我印象深刻而且建立起軍旅情懷的便是【使命召喚】這款，可以說直至現在都貫穿著我們的海，而且在所有的劇情裡面最讓人難以釋懷的便是裡面的好夥伴好戰友當然也是我自己【肥皂】的死亡，肥皂的死亡無疑是各個玩家心中的痛，肥皂與普萊斯在追殺謝菲爾德的過程中被謝菲爾德奪過軍刀一刀將肥皂刺傷，而後肥皂拼盡全力將胸口的軍刀拔出，並拋刀殺了謝菲爾德。但肥皂卻因傷勢過重險些喪命。而後在使命召喚8中，肥皂與普萊斯、尤里一同參加暗殺馬卡洛夫的行動，這段時間間隔只有幾天的功夫。肥皂的傷勢並沒有痊癒。到後來行蹤暴露。肥皂為了掩護尤里，先把尤里推了出去，自己卻被炸到，傷上加傷，因失血過多而犧牲在了布拉格廣場，貫穿了好幾部的使命召喚就再也見不到肥皂了。

2. 【戰地】遊騎兵之死

年老的遊騎兵，破舊的防守城堡，一小隊不知道自己塊成為炮灰的年輕人，即將轟炸全區域的軍艦，成為了整個任務的架構，一個年老的遊騎兵也是最後的遊騎兵接到了他的最後一個任務，固守堡壘三小時等待軍艦的地毯式轟炸，一個送命的任務，在達到指定目標後，解散了不知道會送命的年輕人之後一個老遊騎兵，一桿槍，一支煙，坐在城牆上等待著炮彈的到來，手中沒有絲毫的顫抖，彷佛與世隔絕一樣，沒有所謂的蕩氣迴腸，沒有所謂的大義凌然，死得其所，一切歸於平靜。

3. 【泰坦隕落2】BT之死

都說戰爭當中服從是軍人的天職，對人類而言這樣，那麼對與人類共同作戰的機器人也應該是這樣的，但是在泰坦隕落二中的BT相信大家都的印象肯定是深刻得無法捍衛的，特別是那句【有我在，很安全】一直都全程貫穿著玩家們的大腦，說好的一起下墜直至死亡，享受最後的幾秒鐘，但是BT最後一拋也是全篇遊戲最穩的一拋讓我活了下來【我從來就沒有想過會背叛你，只是在命令和你的生命這兩個選擇中我選擇了你】從此泰坦真的隕落了，你是我的泰坦也是我生的希望！
