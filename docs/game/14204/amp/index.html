爐石傳說：商店新界面將至，媽媽再也不用擔心我買錯卡包了

天天要聞
2019-12-10

在12月6日官方更新的公告中，已經預告了即將到來的全新商店界面。與此同時，還會有新貨幣系統和新英雄皮膚的上線。

從當時的公告中，可以看出商店的局部圖片。

標準牌池的卡包將會被放在一起，目前外服這個商店界面已經上線，國服暫時還沒有更新，預計明天凌晨會和新版本一起上線。

其他界面一覽：

以前的商店界面是這樣的：

從上圖左側可以看到，奧丹姆奇兵卡包和暗影崛起卡包的邊框都亮了。這時候如果沒有仔細查看右邊，那麼很容易就會買錯。

不過新版本商店就避免了買錯的情況，因為在點擊某種卡包之後，會另行彈出一個界面。在這個界面里，再選擇購買的卡包數量，這樣如果再買錯的話，那可就沒辦法了！

此外，還可以發現，狂野卡包也重新回歸了。儘管目前狂野模式式微，但總有一些玩家想要買狂野卡包。用玩家的話來說：”我可以不買，但我的有買的選擇”。

畢竟很多新玩家都沒有見過狂野卡包炸開的樣子，所以各買一包開著玩換換口味也是可以的。所以也就是說，大概率可以用金幣來購買狂野包。

外服雖然上線了新的商店界面，但是新英雄希爾瓦娜斯皮膚以及贈送的泰蘭德皮膚並沒有上線，應該也是明天凌晨全球同步上線。

各位小夥伴們，一起期待一下新的商城吧，屆時別忘記上線領取泰蘭德皮膚哦！
