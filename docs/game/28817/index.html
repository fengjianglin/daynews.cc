黑暗勢力回歸？RNG不敵OMG，小龍堡表現尚可，小虎狀態低迷

2019-12-21 963條評論 2,657次閱讀 897人點贊

北京時間2019年12月21號，德瑪西亞杯迎來了OMG與RNG之間的對決。值得一提的是，從雙方的陣容來看，OMG在Smlz回歸之後有些夢回S7的感覺，不少OMG的老粉絲們也是紛紛期待能否在德杯中見證「黑暗勢力」的回歸。而RNG戰隊則是因為更換了新人打野小龍堡也是成為了本次德杯中備受關注的一支隊伍。不過，在剛剛結束的OMG與RNG第一次較量中，RNG不敵OMG，敗下陣來。

從這一場雙方的BP情況來看，RNG拿出了一套偏向於四一分推的體系，而OMG戰隊則是因為Smlz的存在拿出了讓人印象深刻的女警推塔體系。但事實上，新賽季對於小龍的爭奪更加重要，所以在BP上，OMG其實是取得了比較大的優勢的。並且，這一場比賽的走向也直接向我們證實了，在新賽季的版本中雙人路擁有優先推線權是一件多麼重要的事情了。

Smlz與小五之間的配合依舊還是那麼默契，這自然也成為了OMG能夠拿下RNG最主要的原因之一。不過，從RNG這邊的情況來看，因為是新人打野小龍堡的緣故，可以看出RNG在整體戰術上面還是做了比較大的改動的。特別是小龍堡一直與狼行做聯動，讓RNG戰隊的戰術重心漸漸發生了改變。但可惜的是，小龍堡雖然成功幫助狼行擴大優勢，但因為雙人路太過於劣勢。上路的這些優勢自然也無疑於杯水車薪了。

除此之外，這一場比賽還有一個比較關鍵的點，那就是RNG戰隊中單選手小虎的狀態問題。在這一場比賽中，冷少的蘭博帶的是點燃，在小虎沒有閃現的情況之下的確是能夠配合上自己高額的爆發完成「單殺」。但是，在這一局中，小虎卻還是被OMG不斷尋找到機會。另外，在小虎最後一次「陣亡」中，擁有著遠古巨龍Buff的Smlz，直接配合上遠古巨龍Buff的「斬殺」效果，幾乎就是一下平A就帶走了小虎。可想而知，Smlz在這一場比賽中取得的優勢有多麼大，另外遠古巨龍Buff是有多麼OP的了。

不過，這也僅僅是兩支隊伍首次在德杯中的亮相。希望RNG戰隊能夠迅速調整好狀態，尋找到與這個版本相切合的打法吧！
