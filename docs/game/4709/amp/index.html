鯉魚王也有春天：有人僅用它擊敗了《寶可夢 劍/盾》的最終Boss

摘要

鯉命由你不由天。作為寶可夢系列中廢柴的代名詞，鯉魚王有著憨憨的外表和毫無用處的標誌性動作水濺躍，不論是在遊戲中還是動畫中都是搞笑擔當的蠢萌型寶可夢。在寶可夢圖鑑里，鯉魚王經常被描述成最弱最可憐的寶可夢。

鯉命由你不由天。

作為寶可夢系列中廢柴的代名詞，鯉魚王有著憨憨的外表和毫無用處的標誌性動作水濺躍，不論是在遊戲中還是動畫中都是搞笑擔當的蠢萌型寶可夢。在寶可夢圖鑑里，鯉魚王經常被描述成最弱最可憐的寶可夢。

而油管主播PokeTips，則要挑戰僅使用鯉魚王擊敗《寶可夢 劍/盾》的最終Boss丹帝。

多數寶可夢作品中，鯉魚王都不能學習強力的攻擊招式（活動強化版金鯉魚王除外）。而在《寶可夢 劍/盾》中，鯉魚王終於翻身了，它能夠學習水炮技能，戰鬥力大幅上升。

之前有人發現本作中NPC會帶著鯉魚王參戰五星極巨戰，還在網上成為了笑料。然而在玩了一段時間之後，人們發現這個鯉魚王實際上相當的靠譜，最開始笑話鯉魚王大姐的人，甚至感嘆說原來她是遊戲中最強的AI訓練師。

雖然鯉魚王獲得了史詩級強化，但僅憑一己之力擊敗丹帝還是有些困難的。畢竟它的基礎屬性不高，而且能使用的技能也有限，尤其在屬性被克時會異常脆弱。

因此PokeTips首先將鯉魚王升至100級，給它學了水炮作為主力輸出技能。然後通過吃藥將它的特攻、HP努力值和極巨化等級升滿。之後就可以挑戰丹帝了。

第一次嘗試，PokeTips決定先讓鯉魚王完全靠自己，不攜帶道具，也不在戰鬥中使用道具。

他的策略是第一回合就讓鯉魚王極巨化，一方面可以使用威力更大的極巨水流，另一方面也可以讓場地中下雨，增強水系攻擊力，同時也會觸發鯉魚王在下雨時速度增加的特性。

丹帝的第一隻寶可夢是堅盾劍怪，在第一回合有很大幾率使出王者盾牌，PokeTips藉機用兩發水炮無傷擊敗了堅盾劍怪。

丹帝的第二隻寶可夢多龍巴魯托是一個難點，首先它克制水系招數，同時還會使用十萬伏特。好在PokeTips的鯉魚王靠著極巨化狀態頂住了一次十萬伏特，並用兩個水炮擊敗了多龍巴魯托。

接下來丹帝的寶可夢陣容會根據玩家的初始寶可夢而變化，PokeTips初始選擇了炎兔兒，因此這裡丹帝會派出踏冰人偶，也會使用十萬伏特。

這裡鯉魚王的水炮沒能秒掉踏冰人偶，被十萬伏特帶走，第一次嘗試宣告失敗。

第二次嘗試，PokeTips讓鯉魚王戴上了道具進化奇石，可以提高寶可夢的防禦和特防，其他策略不變。這次劇本與之前大同小異，只不過鯉魚王用了一次水炮就秒掉了踏冰人偶，成功進入了下半場。

到達了第4隻寶可夢，丹帝派出了雙斧戰龍，不但克制水系，而且此時雨也停了。鯉魚王因為輸出不夠，被雙斧戰龍擊敗。

第三次，PokeTips改變了策略，以攻代守，將攜帶道具換成了講究眼鏡增強特攻，同時開場就使用特攻強化道具，讓特攻最大化。

在這種狀態下，鯉魚王的攻擊力暴漲，極巨水流直接秒掉了堅盾劍怪、多龍巴魯托，哪怕是極巨化結束後，鯉魚王也只需要一發水炮就能擊敗剩下的雙斧戰龍和千面避役。就這樣，鯉魚王一路橫掃，最終面對丹帝的超極巨噴火龍。

能夠撐到這裡基本上就等於勝利了，因為屬性相剋，噴火龍應該是鯉魚王最輕鬆的一戰。但PokeTips的水炮卻在關鍵時刻miss了，被噴火龍反手一個極巨草原秒殺。

想一想也有些神奇，畢竟噴火龍超級巨化之後有半個賽場那麼大，想miss實在不容易。

因為水炮只有80的命中率，即使鯉魚王可以依靠數值壓制秒殺所有丹帝的寶可夢，也隨時可能會被對方秒殺。要想擊敗丹帝還是需要些運氣。又嘗試了幾次之後，PokeTips終於用鯉魚王打敗了丹帝。

最後，主角和鯉魚王在會場中央接受人們的歡呼，這奇特的場面證明了哪怕是像鯉魚王這樣欽定的廢柴，也能夠逆天改命，成為冠軍。

難以想像，號稱不敗的冠軍丹帝，被一個才出道的小孩用鯉魚王擊敗，該是多麼屈辱的場面。
