2019春節必玩遊戲：準備好預算，年底一波大掃蕩這些遊戲

2019-12-03 0條評論 8次閱讀 0人點贊

摘要

離2019年春節只剩下一兩個月的時間了，到了年底豐收的季節，想必很多小夥伴已經準備好了預算，大算年底一波大掃蕩，然後宅在家裡沒日沒夜的在遊戲的世界中蕩漾，聽起來就已經心花怒放了，到底年底有哪些值得期待和體驗的作品呢，小游給大家整理一下。

離2019年春節只剩下一兩個月的時間了，到了年底豐收的季節，想必很多小夥伴已經準備好了預算，大算年底一波大掃蕩，然後宅在家裡沒日沒夜的在遊戲的世界中蕩漾，聽起來就已經心花怒放了，到底年底有哪些值得期待和體驗的作品呢，小游給大家整理一下。

遊戲

最近一段時間，遊戲大作都將不斷上線，也是在不斷刺激著玩家們的神經，下面就來看看新作有哪些吧。

荒野大鏢客2

1、荒野大鏢客2

遊戲平台：PS4、XboxOne、（PC版預計明年4月發售）

上線時間：2018年10月26日

這款遊戲剛剛發售，三天就獲得了7億多美元的銷售額，著實讓人震驚了，新作在自由度方面做得非常不錯，真實性很高，你能想像主角調戲路人被警察追趕的情景嗎，或者拍馬屁被踢翻在地的畫面，統統能夠在遊戲中實現，沒有做不到的，只有你想不到。

這款遊戲對善惡的定義不是很明顯，你可以做一個善良的壞人，或者做一個徹徹底底的惡棍，但不管怎麼樣，其實主角還是一個比較壞的人【手動斜眼】，打家劫舍什麼事都能做，做還是不做，關鍵看大家的選擇，想讓主角成為什麼樣子，想必大家心裡都有自己的一桿秤。

輻射76

2、輻射76

遊戲平台：PC、PS4、XboxOne

發售時間：2018年11月14日

輻射系列是非常經典的科幻射擊類遊戲，遊戲劇情還算不錯，隨著新版本到來，輻射76也是比較受期待的，遊戲地圖比前作大了很多，徒步穿越遊戲需要半個多小時。當然跟前作一樣，遊戲也加入了幀數和遊戲速度的功能，幀數越高遊戲速度越快，能夠大大節約徒步的時間，所以電腦配置才是關鍵。

這款遊戲同樣也是自由度比較高，可以通過培養任務獲得強大的屬性，但是也需要時刻有充足的物資保證生存，多人聯機模式能夠讓大家體驗到相互合作生存的樂趣，在核污染的世界如何生存下來呢，期待一下這款新作的上線吧。

精靈寶可夢 Let’s Go

3、精靈寶可夢 Let’s Go

遊戲平台：switch

發售時間：2018年11月16日

從GB、GBC、GBA、NDS、3DS，再到switch，精靈寶可夢已經成了任天堂獨樹一幟的吸金IP，最早在黑白掌機機上，精靈寶可夢給玩家們留下了非常深刻的印象，隨著遊戲一代代的發展，遊戲內容也更加豐富，switch平台獨佔的精靈寶可夢Let’s Go讓人又愛又恨，恨的是遊戲花費了我很多時間，愛的是又有可愛的小精靈能夠捕捉了，小可愛們乖乖到精靈球裡面來吧。

古劍奇譚3

4、古劍奇譚3

遊戲平台：PC

發售時間：2018年11月23日

作為國產遊戲，古劍系列雖然還存在很多不足，但是進步也是有目共睹的，特別是劇情的設定國風純正，玩古劍也是沖著劇情去的，新作加入了一些新地圖場景，前作留下的謎題真相也在這一作一步步被揭開，古劍的粉絲不要錯過。

暗黑血統3

5、暗黑血統3

遊戲平台：PC、PS4、XboxOne

發售時間：2018年11月28日

這是一款老牌動作冒險遊戲，時隔六年，再度上線新作給玩家們帶來了很多期待，開放式的動作冒險世界，加入探索解密元素，對於喜歡解密遊戲的玩家來說也是非常值得體驗的，最重要的是暗黑式畫面，能夠給小夥伴們帶來很多回憶，喜歡的話可以期待一下。

正當防衛4

6、正當防衛4

遊戲平台：PC、PS4、XboxOne

發售時間：2018年12月4日

正當防衛4已經進入壓盤階段了，所以12月4日發售已經是板上釘釘的事情，不會再出現跳票的可能。同樣是一款自由度超高的遊戲，新作加入了全新的天氣系統之後，將會帶來很多奇妙的體驗，比如說路上遇到龍轉風，或者是遇到冰雪天氣凍的發抖，新增的遊戲特性能夠讓玩家們體驗到更多的樂趣，當然遊戲畫面也是一大特色，可別光顧著觀賞風景，忘了做主線任務哦。

國夫君：世界經典收藏

7、國夫君：世界經典收藏

遊戲平台：PC、PS4、XboxOne、switch

發售時間：2018年12月4日

這是一款情懷遊戲，收錄了國夫君系列18部遊戲作品，其中3部暫未公布，不知道大家看到這款遊戲是否已經淚流滿面，相當高純度的情懷，不知道陪伴了多少小夥伴的童年，有些小遊戲的遊戲性還是非常高的，至於是否值得為情懷買單，那就要看大家的了。

目前涼風就給大家帶來這幾款遊戲吧，都是非常棒的，並且即將發售遊戲作品，當然還有一些遊戲也會在年底之前發售，至於是否值得體驗，大家可以根據自己的喜好去選擇。
