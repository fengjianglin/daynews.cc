你說的這個浮島物語，它，好玩嗎？#這個遊戲好玩嗎#

2019-12-09 0條評論 16次閱讀 0人點贊

不隨波逐流，只推薦自己喜歡的遊戲
大家好，我是右語rxc
有一說一，前陣子steam秋促買了個《浮島物語》，點擊安裝後顯示竟然不到200mb，我驚了。這個遊戲以前聽我朋友給我說過，當時沒注意，秋促的時候看見在推薦名單上，就想著入手看看。

《浮島物語》是一款2D開放世界像素放置型的沙盒類的生存遊戲。超受歡迎的古怪作品，「一款讓你停不下來的放置型遊戲」。探索，製作，收集並管理資源，尋找秘密並從零開始建造你的基地！買下土地進行探索和擴展吧！遊戲簡介如是寫到。
進入遊戲以後就被這個簡陋的世界震驚了，我就一個島？？？說好的冒險呢？？？是要我造木橋出去尋找新的大陸嗎？？？一開始島上就是石頭，樹木，礦石這些，這強烈的既視感是怎麼回事啊！我第一反應這不就是《我的世界》《泰拉瑞亞》嗎！3D《我的世界》2D《泰拉瑞亞》2.5D《浮島物語》，齊活了！

從玩法上來說，有新也有舊，《浮島物語》的玩法其實挺像《泰拉瑞亞》《我的世界》的，擼樹，挖礦，捕魚，造工作台等，還有體力值和能量條，挖礦，砍樹這種動作會消耗能量，能量用完就會體力加能量，吃烤肉烤魚就可以恢復體力值和能量條。這熟悉的操作，這熟悉的玩法（這熟悉的界面，好吧沒有界面）不過有1說1，玩過mc和泰拉瑞亞的玩家應該都可以快速上手。

秉承著要想富，先擼樹的優良傳統，開始了在浮島上罪惡的一生。仔細看了下系統，發現一開始就一個島嶼，其他的島嶼要用金幣購買，那麼問題來了，我們的第一桶金哪裡來呢？答案就是浮島物語第二個系統，等級系統。當我們扮演的小白通過砍樹，挖礦，都會帶了經驗，經驗條滿了就能升級，升級就會給技能點，有了技能點才能解鎖新的建築和能力。

然而成也系統，敗也系統，雖然前期這種升級買島點技能很有新鮮感，但是到了遊戲中期，我個人感覺太肝了。萌新期過完，後面到第13個島差不多就要3-4k金才能買，而小白掙金的手段還比較慢，就是不斷的挖礦，而且收益還不高，也可能是我沒找到正確的掙金方法。最絕的是你把能造出來的都造出來了，然後還有好多技能沒解鎖，看著需要升級解鎖新建築和需要4k+金才能買到新島嶼，一陣肝疼。
冒險的話就是遊戲中的草地，沙漠，墓地，雪地，地獄場景的島嶼，這五種場景的島嶼上有一個對應的副本，這些副本也許就是玩家們肝爆的安慰了。

音樂的話和《泰拉瑞亞》風格挺像的，晚上還會掉星星。而遊戲畫面還是蠻不錯，有種別樣的可愛。相較於《我的世界》和《泰拉瑞亞》，《浮島物語》沒有明確的任務，就是探索，製作，收集並管理資源，尋找秘密並從零開始建造基地，其中島嶼上npc的互動以及五大遺迹可以看成遊戲的主線了。

《浮島物語》總的來說是款還可以的遊戲。前期的新鮮，中期的肝爆，後期的無聊，以及平衡性的問題，有的道具存在意義就特別的低，前期碰到神殿，直接獲取神器，基本武器都不用做了，一把神器打天下，像捲軸炸彈生命藥劑一類的根本都不用做，製作麻煩，還沒有烤魚實用。而且《浮島物語》的70元定價相較於《泰拉瑞亞》的36元，還是稍顯貴了些。
