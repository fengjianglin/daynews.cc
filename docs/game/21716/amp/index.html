DNF：85史詩套現在先別分解！85級史詩戰術套可升級100級版本史詩

天天要聞
2019-12-18

韓服在發布會中公布了100級的裝備系統，其中非常突破性的不僅僅是神話級別的裝備，還有一個可以交易的史詩裝備，這一個史詩裝備叫做「智慧的產物」，沒有套裝屬性，同樣不能進行強化增幅和鍛造，但是可以利用專屬的【裝備改造】系統來觸發特殊的屬性。

目前在韓服的官網中也沒有非常詳細的介紹這一個可以交易的史詩裝備帶來的屬性效果，只是稍微提到這一個可以交易的史詩可以通過改造而獲得屬性。

但是當我們在發布會中去找這一個裝備的說明，卻發現了一個非常有趣的事情，這一個史詩居然是搭配85級史詩獲得額外的屬性效果！

發布會中可交易史詩的裝備內容分析

在發布會中主持人介紹了這一個可以交易史詩裝備，但是並沒有提到具體的屬性，不過我們可以在屏幕背後中看到特殊屬性。

右邊的史詩裝備名字叫做【威斯利的戰術指揮棒】，是100級可以交易的史詩輔助裝備，這一個在裝備就是第一張圖中的第五個裝備，裝備這一個輔助裝備時，並裝備五件套威斯利的戰術套裝，這一個國服翻譯就是戰術之王套裝，同時穿戴時將會獲得這樣的效果：

  • 裝備戰術套時將發動裝備改造屬性
  • 裝備戰術套裝時額外增加抗魔值+235
  • 裝備戰術套裝時，戰術套裝備穿戴等級+15

這一個怎麼說呢？如果玩家裝備了【威斯利的戰術指揮棒】，那麼原來戰術之王套裝的等級是85級，會直接變成100級的史詩裝備，而抗魔值也因為等級的提升而增加。此外還會獲得屬性改造效果，原來的戰術之王屬性不再生效。

根據基哥翻譯的改造屬性為：

  • 1階段屬性：最終傷害+32%，500px範圍內所有隊員全職業60-80技能+3（該輔助效果和裝備+15級的屬性不適用於奶類職業，也就是說這一套裝備對奶媽奶爸和奶蘿無效）
  • 3階段：三攻+40%，技能攻擊力+20%
  • 5階段：技能攻擊力+10%
  • 固有屬性：暴擊傷害增加42%

為什麼說這是戰術套的屬性效果呢？

韓服查看戰術套的名稱和發布會上五件套的名字一樣，並且戰術之王是85級史詩，和裝備史詩時增加15級等級正好符合，其次戰術之王是光環套，正好和一階段屬性增加隊友的技能等級相似，不過這一個區間的技能好像並不是很理想。

分析

韓服體驗服將在12月18日開啟測試，所以具體是不是這一個玩法可以在更新後進行測試，如果確實是這種類型的裝備玩法，確實讓100級有了更多的裝備搭配選擇，即使是85級版本脫坑的玩家在這一個版本回歸，拍賣場直接購買一件可以交易的史詩裝備也可以更輕鬆的挑戰100級的副本製作裝備。

現在發布會上僅看到了85級的戰術套，而玩家更希望有魔戰套、天御套這兩套史詩也能通過這一個史詩裝備升級而獲得額外的效果，增加裝備玩法的趣味性。

有人問是不是國服的兵法之神而不是戰術套？韓服是沒有兵法的，並且對比了韓服戰術套的名字和發布會的圖片對比，確實對應得上。

如果真是這樣，100級的魔戰套也太快樂了！這樣原來小號上的魔戰套又可以愉快的玩耍了。
