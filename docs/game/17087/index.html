該來的還是來了！《火影忍者究極風暴4》將登陸SWITCH

2019-12-14 3,041條評論 4,015次閱讀 2481人點贊

Jump周刊全新一期爆料，已經登陸Switch的《火影忍者究極風暴》系列最新作品《火影忍者究極風暴4：博人之路》確定將會登陸Switch平台。遊戲將於2020年4月23日登陸Switch平台，售價6680日元(約合432元)。

另外，所有版本的遊戲都將迎來一個售價700日元(約合45元)的DLC。此DLC包含大筒木桃式和大筒木金式兩名可玩角色，並為豬鹿蝶等11名角色提供博人時代的服裝。此DLC同樣適用於原版《火影忍者究極風暴4》。

火影忍者究極風暴的前三個作品都已經以單獨和合集的方式登陸了Switch平台，唯獨就是缺一個《火影忍者究極風暴4》，現在終於也是補齊了冷飯全合集了!

《火影忍者究極風暴4》承接《火影忍者疾風傳：究極忍者風暴3》的延續劇情，設定了由初代火影千手柱間和宇智波斑的決鬥開始，接著橫跨到第四次忍界大戰。同時還將加入最終劇場版THE LAST的劇情。本作最大特點就是混戰破壞效果更加真實，官方甚至用「天崩地裂」來形容查克拉爆發後的場面。
