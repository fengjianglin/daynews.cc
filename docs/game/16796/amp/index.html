畫面風格討人喜歡——《墮落軍團》

【遊戲介紹】

　　《墮落軍團》將RPG元素通過一種獨特的戰鬥系統與格鬥元素相結合。遊戲一開始，主角Fenumia王國的公主Cecille站在戰場上，身後是她的大軍，而他們面對的是洶湧而來的怪獸大軍。而公主的父親，Fenumia國王卻去世，留下了一本能夠賦予武器生命的魔法書，而這本魔法書卻知道整個王國的真相——Fenumia並不像它的臣民們想像中的那樣美好。

【難道是被冷落的佳作?】

　　當筆者看到擁有漂亮2D畫風的《墮落軍團》時，讓筆者想起了一年前在PSV上面玩到的《奧丁領域》，華麗的BGM、激烈的戰鬥和曲折的劇情，以及細膩流暢充滿美感的2D畫面，這兩款作品都有異曲同工之處。也許是受到首發在PSV平台的影響，遊戲發售後反響平平，在百度上幾乎搜索不到《墮落軍團》太多信息，即便是後來移植PS4似乎也同樣很冷門。

　　不過正因為此，喜歡類似香茶社精美2D畫風的PC玩家才能體驗到《墮落軍團》PC版。本次推出的PC版包含了之前一直分開銷售的兩個版本：《墮落軍團;王國之罪篇》與《墮落軍團：逆軍之炎篇》。一位是為了拯救陷入亡國危機的王國公主塞西露，另一位是為了欲排除塞西露的軍團「墮落軍團」的首領，兩個款作品之前不光分開售賣，同時也被分開到了兩個平台(PSV和PS4)，這次能買一份就包括兩部作品，完整體驗正反雙方的不同情節。

【還算有趣的戰鬥系統】

　　遊戲的戰鬥方式是即時半回合，只是在玩家不輸入任何指令的時候，己方隊員是不會自動攻擊的。操作上面還是比較順手的，沒有繁瑣的指令菜單，不用在繁多的技能當中去選擇，需要操作的隊員基本對應個別單獨按鍵，只需要在有行動條的情況下發動攻擊即可。

　　如果只是這樣的操作遊戲顯然會就太單調了，實際上玩家需要同時向四位角色輸入指令，由於遊戲採用了半回合制戰鬥，敵人可不會等著你的攻擊，所以需要很快速的做出判斷。遊戲的四人隊伍中，主角和另外三名「武神」的操作方式又不相同，可以通過上下選擇不同的技能，除了輸出，最主要的功能還是輔助作用。再次基礎上，遊戲增加了格擋要素，除了能防禦部分傷害外，在準確的時機格擋還能完全反彈傷害，在戰鬥中起著至關重要的影響。

說了這麼多，其實戰鬥訣竅還是在於完美格擋，很多時候敵人都比我方人數更多，攻擊頻率更快，能不能戰勝敵人基本就看能不能完美格擋，剩下的亂按也能過。當然，遊戲里還有貢品，寶石，獲得更多不同的「武神」搭配等等，整個遊戲系統並不複雜，且充滿樂趣。但是遊玩一段時間後，很明顯的感到重複度太高，這個系統怎麼看都像一個手機遊戲….

【特有的抉擇系統】

　　遊戲有一套類似塔羅牌的抉擇系統，這些抉擇會出現在戰鬥與戰鬥之間的行進過程中，下圖是STEAM頁面對這套系統的說明。

　　除了要應付激烈戰事外，玩家還要在激戰前的瞬間為國家作出困難的抉擇並承擔其後果。你會要脅某王子增強士兵的能力?還是要向飢餓的村民徵收賦稅來換取隨行士兵的性命?每個決定會提升玩家不同能力的同時，也會改變故事的發展。

實際上這些抉擇遠遠沒有影響劇情走向的威力，它們最終只會影響玩家的士氣值，這個值會影響玩家每關結算的收益，且每張牌會有不同的增益效果，想法還是蠻好的，但實際上不管是對劇情還是戰鬥影響都不太大。

【蒼白無力的劇本】

　　《墮落軍團》顯然借鑒了《奧丁領域》的敘事方法，一個世界的故事，使用多方主角的視角去敘述，讓整個故事和世界都顯得更加豐滿。遊戲的故事由帝國國王的突然去世作為開端，一個是身體里流淌著過去王朝餘暉的將軍的背叛，另一個是王國的正式繼承人公主，兩個人曾是並肩作戰的戰友，由於王國的變故反目成仇。

　　是的，遊戲一開始給了筆者一個比較好印象，讓筆者對故事的發展有著較高的期待，但是隨著時間的推移，遊戲蒼白的的世界觀開始逐漸暴露無遺，劇情的發展有一個好的開端，但是推動遊戲進程的情節部分顯得十分零散瑣粹，並沒有錯綜複雜的故事線，也沒有特別的愛恨情仇，很難帶動筆者情緒。

這樣的故事本可以是充滿史詩與欺詐、愛情與羈絆，一個類似《奧丁領域》般悲情歌劇般的故事，然而本作劇本顯然沒有達到這些高度，推動遊戲進程的劇情甚至讓筆者有一種在玩手游的錯覺，平鋪直訴的敘事模式，沒有太能帶動筆者情緒。

【評測總結】

　　《墮落軍團》是一款在美術以及劇本上借鑒了很多《奧丁領域》的作品，雖然在這兩點上都無法超越《奧丁領域》，但是本作原創的戰鬥系統還是值得嘗試的，曾經分開出售的兩部曲《墮落軍團;王國之罪篇》與《墮落軍團：逆軍之炎篇》，現在也只需要幾十元的價格而已，喜歡香草社氣質的玩家不妨一試。

休息會，掃碼玩玩遊戲吧~
