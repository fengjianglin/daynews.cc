PSN新增多款遊戲加入，體驗依舊不如意

天天要聞
2019-12-18

在新世代即將到來的時候，不論是玩家還是廠商都有一個共鳴，那就是未來的雲遊戲將會成為遊戲市場一大重要組成部分：

雲遊戲平台蓬勃發展

因此很多廠商都開始布局雲遊戲，谷歌率先成立了專門的雲遊戲平台stadia想要與御三家在新時代展開競爭。御三家這邊也是沒有閑著，微軟繼續穩步推進自己XGP的戰略路徑，xcloud也是成功推出，索尼這邊也是推出PSnow服務，除了內容為王的任天堂之外幾乎大部分廠商都在雲遊戲方面多有涉獵。

索尼宣布多款獨佔大作登陸PSnow

在近期stadia發售之後市場受到了冷遇，卻並沒有擊退很多廠商的熱情。索尼方面就宣布了在2020年1月份即將登陸PSnow的雲遊戲陣容，其中包括了多款PS獨佔大作，比如《神秘海域：失落遺產》《地平線：黎明時分》，索尼的PSnow也是逐步向著微軟的XGP發展。

只不過對於國服的玩家來說，總體上PSnow的體驗服務並不是很好，因為各種方面的原因制約也是很難體會到原汁原味的感覺，看來索尼要走的路還有很多。
