PDD攜手E星進軍LPL，駱歆人氣不敵Mayumi，寶藍透露IG將引進新人

天天要聞
2019-12-13

賽事圈

PDD聯手eStar正式進軍LPL

北京時間12月12日，由知名電競人PDD(劉謀)與武漢旅遊發展投資集團共同注資的eStar電子競技俱樂部，參與英雄聯盟職業聯賽(LPL)的席位競標，成功獲得英雄聯盟職業聯賽參賽席位。據悉，eStar戰隊將於2020年春季，正式開始征戰英雄聯盟職業聯賽。

LGD英雄聯盟分部官宣：教練UDJ離任

昨日，LGD俱樂部英雄聯盟分部通過微博宣布，原LGD教練UDJ正式離任，不再擔任LGD的教練一職。LGD稱，十分感謝從2018年就加入的UDJ教練為團隊的付出和貢獻，真誠祝願他的未來職業生涯能夠順利。

電競名人

Mystic就Doran禁賽事件發表不當言論，戰隊道歉

近日，DRX選手Doran被禁賽的事情引起了不少職業選手的關注。11月中旬，Doran因為在一局排位中用亞索打出了7-15的戰績，被拳頭官方以「妨礙遊戲進行」處罰停賽一場和罰款。包括PraY在內的多個選手都認為此次處罰欠妥。而Mystic則在社交平台公開發表了對Doran的不恰當評價，用詞十分不客氣。昨日，Mystic所在AF戰隊公開發布了聲明，為選手Mystic的不當言論向Doran道歉。

寶藍直播透露：新賽季IG要來好多新人

在最近的一次直播中，寶藍透露了一些關於新賽季中IG戰隊的動向。寶藍說：「下賽季你們肯定會看到新人的，也不算新人，反正肯定會加不少人。」不過對於隊內不和的傳聞，以及包括自己在內的隊員的未來去向，寶藍並未提及。

駱歆發起微博投票「硬剛」Mayumi失敗

昨日下午，LPL官方主持人駱歆在微博上發起了一項名為「選一個美少女給你打輔助，你選誰？」的投票活動，欲和巴西賽區INTZ戰隊的女輔助Mayumi「正面硬剛」。截止目前，Mayumi以4萬1千票遙遙領先。

新游快訊

《騎馬與砍殺2》新開發日誌公布，加入「王國決策」系統

昨日，《騎馬與砍殺2》的官方中文站通過微博公布了新的開發日誌。官方稱，遊戲內加入了新的系統——「王國決策」，該系統給遊戲提供了更多的戰略博弈空間，玩家可以通過外交手段扭轉局勢，達到「兵不血刃」的目的。同樣，該系統也會讓遊戲增加新的挑戰。

《最終幻想：水晶編年史》延期發售

近日，《最終幻想》官方在Twitter上宣布，原定於2020年1月23日發售的《最終幻想：水晶編年史》，將延期至2020年夏季發售。官方表示，本作還需要更多的時間進行內容上的打磨。

遊戲新鮮事

《時尚先生》發布Tian短片：打破人們說的不務正業

昨天，邀請Tian拍攝封面的雜誌《時尚先生》官方微博發布了一條Tian的短片。在短片中，《時尚先生》和Tian傳達了這樣的概念：電競總是拋棄掉隊的人，時代也是如此。Tian要做的，就是打破人們所說的「不務正業」，做那個創造新時代的中國新青年。

英雄聯盟新英雄瑟提曝光

近日，拳頭在測試服曝光了峽谷最新英雄瑟提的一些相關信息。

被動技能「沙場豪情」：普通攻擊交替使用左重拳和右重拳。

Q技能「屈人之威」：普攻附加額外傷害並增加移速。

W技能「蓄意轟拳」：向一個方向積蓄重拳造成範圍傷害並為自己提供一個護盾

E技能「強手裂顱」：將兩名敵人撞在一起並暈眩

R技能「嘆為觀止」：扛著一名敵方英雄前行並猛摔，著陸點敵人受到被擒英雄40%的額外生命值傷害並減速99% 。

TGA送福利：限時48小時，Steam十餘款遊戲免費下載

為回饋廣大用戶的支持，今日凌晨，TGA創始人Geoff Keighley在推特上宣布了一項名為「TheGameFestival」的活動。活動期間，十餘款遊戲將在Steam上架，玩家們可以進行免費下載。本次活動時間為北京時間12月13日凌晨2:00至12月15日凌晨2點，限時48小時。
