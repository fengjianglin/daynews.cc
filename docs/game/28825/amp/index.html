《鎮魔曲BT》夜狩職業高玩攻略

天天要聞
2019-12-21

關於夜狩這個職業，由於物理、魔法雙天賦的存在，導致屬性方面有特殊的地方主要有以下幾點：

(1)生存天賦物轉法的天賦，導致切換生存天賦的時候所有物理屬性(攻擊、暴擊、暴擊傷害、命中等等)全部會轉化為魔法，所以屬性追求方面魔法攻擊、暴擊、暴擊傷害同樣有很大的收益。

(2) 寵物搭配方面更加的靈活，物理增傷光環和魔法增傷光環雙疊加，所以光環的收益會顯的異常的高 其次由於生存天賦提供的魔法傷害為火屬性且會給當前目標施加火屬性易傷，那麼寵物選擇則優先考慮 翼翅、鳳翎兒的組合。控制方面當然是大聖+大象，輔助花妹妹。由於天降的難培養性(精修)，我只選擇 了翼翅。有條件的可以考慮翼翅+風翎兒+神將+大象+落英的組合。當然最好是神將擁有大聖的專屬技能，畢竟控制太強大。

(3) 生存方面;優先考慮一些紅色翅膀和時裝，提供大量的物理魔法減傷讓你屹立不倒。紅色的坐騎也是一個必不可少的選擇，可以提供大量的移動速度加成。

(4) 魂石培養：魂石提供了很好的屬性而且容易獲得，我們可以通過捐獻和造物主得到後面幾個魂石的屬性優先建議升級消耗少的魂石，這樣可以加速培養。建議先將每條魂石屬性升級到10級 可以更好的為你提供屬性

夜狩PK攻略：

關於夜狩這個職業由於天賦(生存)的存在導致魔法傷害也是一個很重要的選擇，在PK的時候表現的異常突出。 高機動與高爆發確保的穩定的生存與擊殺。由此可以看出魔法火屬性傷害是一個不錯的選擇所以在屬性選擇方面我更偏重命中、暴擊、暴擊傷害這幾個屬性方面考慮，當然前者都是從輸出方面出發。 生存方面的著重點：物理傷害降低、魔法傷害降低、閃避著手，神級翅膀與坐騎給予的傷害減速不容小覷。 有鑒於個人感覺生存天賦使用時技能會略微卡手所以最好具備2個條件：

1攻速與施法(寵物的光環)

2魂石萬法緣這2個方面著手。魂石萬法緣可以通過逐鹿捐贈獲得，每天捐贈超過4000就可以獲得3個。 關裝備選擇方面由於武庫的開啟首先是確保每個部位都擁有幾件紅色裝備，其次凡元武庫也是一個很重要的選擇在高於150級得橙色裝備某些特殊部位的特殊屬性，其價值可能超過高等級的紫色裝備，選擇的時候須有一定的考究。

夜狩靈獸攻略：

在夜狩生存天賦的條件下物理傷害100%轉換為魔法傷害，所以說寵物增加魔法傷害，魔法暴擊等等一系列增傷光環都有不錯的效果，值得注意的是魔法傷害為火系，所以瑞獸鳳鈴兒是一個很不錯的選擇，另外本體會給目標疊加火屬性傷害的易傷BUFF，其次就是翼翅(最全面的法系寶寶)這個必備，原因就不用提及了。物理系的寶寶主要有猴子(強大的控制)+大象(增傷和起到MT的作用)，治療當然就是落英萌MM了。在這個組合的基礎上再增加一個特殊的寵物技能(猴子的分身、兔子的爆炸葫蘆、魚人的橫掃)都是一些不錯的選擇。這個組合相對於其他的組合來說有更好的成長性，有鑒於神將的稀缺我們只能2選1 這樣翼翅就是不2的選擇。瑞獸2個都選，培養難度不會很大。落英作為唯一的治療神獸沒得選擇。猴子可以說是最好的選擇 培養難度低，易精修，關鍵是很全面。所以對於夜狩個人推薦寵物組合;猴子(控制+物理輸出)+大象+翼翅+鳳玲兒+落英的組合。瑞象和落英就沒有介紹的必要最強防禦神獸和最強治療神獸沒有別的選擇，有鑒於夜狩是個魔武雙修的職業所以輸出方面沒有必要全部物理或者全部法系，那麼萬金油孫小聖是個不錯的選擇，畢竟在擁有強大的控制天賦，分身也能提供不俗的傷害。當然神將和犬神是最強大的存在，但是考慮到寵物的培養問題後2者精修起來的難度太大，遠不如使用匯靈法卷扣取技能，翼翅最強大的雙屬性輸出寶寶沒有之一。最後一個是個風翎兒，有鑒於夜狩生存天賦為火屬性傷害，戰鬥中會給目標施加火屬性易傷BUFF，所以這個寵物無疑成為最好的選擇。 寵物培養方面 儘可能的讓每個寵物都擁有12個技能，然後分別學習有特點的技能。這裡需要注意的是各個寵物的群體光環技能是必備的，某些寵物的特殊技能也是一個不錯的選擇例如猴子的分身，個人偏愛。進戰鬥就是10+個寵物，對手很慌有木有啊。
