失落城堡：無限技能！原來有一群人是這樣玩遊戲的

天天要聞
2019-12-22

近日，有粉絲向我發來一個失落城堡的安裝包，說是有「特異功能」，讓我體驗一下。懷著好奇的心理，我在一部舊手機上安裝了這個安裝包（因為小編怕有病毒，所以特意換了舊手機）。

進入遊戲之後，根據遊戲的說明，原來第一個「特意功能」就是可以「無條件使用靈魂升級」，是的，你不需要再辛辛苦苦的積攢靈魂了，想升級哪個技能，直接點升級就可以了，幾秒鐘就可以達到110級！然後呢，第二個「特異功能」是無條件解鎖執照，這，好吧，我當時買執照可是花了十幾元錢呢！

暴力升級技能，直接110級

進入地圖後，一個更厲害的功能出現了，無限技能！任何一款武器的技能都沒有時間限制，這也太無恥了吧！

無限播種

無限召喚骷髏兵

無限水晶牆

無限回復生命

無限收穫

而且，當你遇到商人買東西的時候，金幣不減反增？好吧，這些破解別人遊戲的人也太厲害了。當我以為這僅僅是一個破解遊戲時，我的粉絲告訴我，這個安裝包的數據與正版安裝包是互通的！納尼？這不是坑我嗎！懷著緊張的心情，我來到了正版遊戲中，果然，我的技能已經從幾十級變成了110級，好吧，看來粉絲說的是真的，哎，真是坑啊，這個不會被封號吧！最近遊戲一直在打擊外掛，希望我不會被封號吧。這時候我也明白了網上那些幫人升級技能的人，說是只要提供賬號密碼，兩分鐘就能升到110級，我估計用的就是這種安裝包吧，所以大家千萬不要找這些人升級技能了！

最後呢，呼籲大家支持正版遊戲吧，本身也就是一個娛樂遊戲，有必要開掛來玩嗎？對了，也不要找我索取安裝包，我已經刪除了！
