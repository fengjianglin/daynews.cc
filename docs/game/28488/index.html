王者榮耀冬冠選拔賽，一戰定勝負，Estar、Hero等戰隊形勢不樂觀

2019-12-23 58條評論 1,072次閱讀 55人點贊

　　秋季賽結束之後，大家期盼的冬冠也是隨之而來，這場比賽不僅是對於國內的KPL玩家有很大的意義，同時也是全世界王者榮耀玩家的盛宴，因為在這場比賽中你可以看到海內外的戰隊齊聚一堂共同比試。這場比賽從選拔賽開始就已經充滿了濃厚的火藥味，因為選拔賽是一局定勝負你需要更多的拿積分才行，這對於每個戰隊來說是沒有了解敵方戰隊的時間，只有掏出自己最拿手的陣容才能夠抓住機會。

　　因此在這樣的比賽中經常會出現爆冷的狀況，即使你是看起來表面實力並不強的隊伍也會有獲勝的希望。特別是在今年的冬冠上想要晉級難度是很大的，因為這一次首日參賽的隊伍中有五支都是來自KPL隊伍，剩餘三支則是境外KRKPL的戰隊，如果不出意外的話應該都是KPL戰隊之間的內戰，畢竟KRKPL距離KPL戰隊還是有很大差距的，更別說面對五支戰隊的圍攻了。

　　但是這並不代表KPL的戰隊可以放鬆，因為是五支戰隊之間的內戰，想要爭奪兩個名額，在內戰的時候，關鍵分一定要拿到才。沒想到的是，隨著第一天的比賽打響，雖然說KPL的五支戰隊都有拿分，但是對於E星、俠隊以及hero來說情況並不是那麼的樂觀，因為他們在內戰比賽時失去了一分，同時在接下來這三支戰隊之間還有比賽，這就導致淘汰的隊伍很有可能是從這三支隊伍中決出。

　　目前比較樂觀的就只有XQ以及TS戰隊了，因為他們都是三個積分在手了。不過XQ的好處就在於有一分是從內戰KPL隊伍中拿到的，因此如果XQ戰隊在接下來的內戰中還能夠穩定的拿到兩分的話，那麼出線就很有望了，不過比賽都是瞬息萬變的，在比賽沒有結束之前我們都不能妄下定論。

　　在昨天的幾場比賽中應該屬XQ同Hero之間的對戰最受大家的關注了，對於hero這支戰隊來說，久誠應該算得上是核心人物了，因此上了大名單之後，大家都覺得他會在同XQ的對決中出現，然而他並沒有上場，讓粉絲有些失望。

　　不過更失望的還是在同XQ對局中Hero出現的那些失誤吧，其實前期的時候都打的很不錯，甚至在爭奪暴君的團戰中打出了零換三的戰績，但是在中路hero出現失誤後，優勢被逆轉。這場比賽中XQ的關羽以及娜可露露發揮得相當出色，並且在團戰中憑藉裝備優勢幾次獲得勝利，在拿下主宰之後成功團滅Hero，最終是獲得內戰的一個積分。

　　從目前來看，輸掉內戰的三支隊伍目前的形式都不是特別的樂觀。不過好在接下來還有兩天的比賽可能會發生一些變數，因此各位粉絲也不要特別的擔心，他們都還是有很大的機會的。小編在這裡自然是希望這三支戰隊能夠好好打，獲取各自隊伍的勝利，同事呢也希望KRKPL戰隊能夠在一年之後給大家帶來驚喜吧。你對於今年的冬冠有哪些期待呢?歡迎在文章下方留言評論喔。
