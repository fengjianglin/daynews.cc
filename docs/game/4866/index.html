王者榮耀：官方發布KPL季後賽第三周回顧，一諾公孫離成亮點英雄

2019-12-04 0條評論 21次閱讀 0人點贊

摘要

王者榮耀的2019年KPL職業聯賽秋季賽只剩最後的一場總決賽了，之前可以說大家已經欣賞到了很多精彩的比賽，官方也是發布了這一周的季後賽頭條新聞和亮點英雄，我們一起來看看吧。

王者榮耀的2019年KPL職業聯賽秋季賽只剩最後的一場總決賽了，之前可以說大家已經欣賞到了很多精彩的比賽，官方也是發布了這一周的季後賽頭條新聞和亮點英雄，我們一起來看看吧。

首先就是總決賽兩支隊伍的晉級路程了，QGhappy和JC的一戰非常精彩，雙方鏖戰到最後的第7局巔峰對決才分出勝負，而AG超玩會的晉級路線就要困難一些，因為他們開始是不敵QGhappy進入了敗者組，所以這周也是打了2場，其中和eStar的一場真的是驚天地泣鬼神啊，同樣是打到了巔峰對決，其中的第3局更可以說是本年度的KPL最經典對決了，之後和JC一戰就有些過於順利了，可能是JC之前輸給QG還沒緩過神來吧。

從各大戰隊的晉級路線也可以看到，最開始在敗者組的隊伍要想成功殺入最後的總決賽要困難很多，最多的要連勝5場才行，而勝者組連勝2場就行了，所以常規賽的戰績也是非常重要的，畢竟在勝者組的話還有一次失敗的機會，因此到目前為止，所以的總冠軍都是勝者組的，不知道哪一屆能出現一批真正的黑馬？

亮點英雄的話，AG超玩會一諾的公孫離可以說是當仁不讓了，雖然在好幾局巔峰對決中都是雙方都使用公孫離當射手，但能把這個射手玩得跟刺客一樣的就只有一諾了，尤其是AG超玩會和eStarPro的巔峰對決中，一諾從草叢閃過去直接把花海給秒掉絕對是天秀，也讓AG超玩會把握了比賽的主動權，最終獲勝他是功不可沒啊。

另外有一個亮點英雄則是FLy的曜，飛牛每次在季後賽都有完美的發揮，畢竟幾個FMVP不是開玩笑的，曜這個英雄在他的手中不僅能打出一定的傷害，還能抗下無窮的壓力，好幾次都是在幾人圍攻中不死，等到隊友來了後一波反殺，也是再現了FMVP的風采，總決賽馬上就要來了，歡迎大家關注和分享，將繼續給大家帶來王者榮耀職業比賽的最新消息。
