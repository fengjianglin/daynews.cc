小智入駐短視頻行業，那個我們熟悉的小智又回來了？

天天要聞
2019-12-12

LOL已經陪伴了我們九個年頭了，其衍生的相關產業也是愈發完善和多樣，從一開始百家爭鳴的遊戲視頻，到現在如雨後春筍般冒出來的直播，玩家們學習技術、消遣娛樂、觀看比賽的途徑也是愈發多樣化。

而眾多遊戲視頻的作者中，其中一個叫小智的，就頗受大家喜歡，其巔峰時期隨便一個視頻在平台上都能有數百萬的播放。

可以說，小智是做視頻最成功、最出色的那一批人。

後來，隨著直播行業的興起，小智也漸漸地不再做遊戲視頻，雖然真正的粉絲還是能通過直播的方式觀看小智打遊戲，但是直播和視頻到底還是兩種不同的傳播方式，總歸是有所差別的，很多人還是懷念小智做視頻的那段日子。

不過沒辦法，動輒半小時以上的遊戲視頻，已經不再適應時代的發展，現在是短視頻興起的時代，不到幾分鐘的視頻，恰好能滿足人們休閑的需要。

而小智就是看準了這一點，所以開始以短視頻的形式為大家帶來新的遊戲內容。

今日，小智許久未更新小智在自己的抖音賬號「LOL超級小智」上再次發布了新的視頻，並且表示自己會出LOL雲頂之弈相關的視頻。

難道，那個我們熟悉的那個段子張口就來的小智又回來了？
