帥的人沒有我有才，有才的人沒有我帥。大家好這裡是二哈講遊戲，每天花1分鐘聽二哈推薦一款好玩的遊戲！

身處囚禁

《INSIDE》是一款老遊戲了，可是這次的投票仍然能獲勝，說明這個遊戲真的是非常優秀的。

還記得去年，我也曾下載過，但是玩了一會兒就刪掉了，因為是實在是手太笨了。

這次我們再次購買這個遊戲，我也靜下心來去仔細試玩了。

《INSIDE》是著名的解謎大作《LIMBO》的續作，也延續了《LIMBO》的風格。陰鬱可怖的世界，反烏托邦的設計，依舊還是黑白灰的畫面，無時無刻撲面而來的壓抑感。

遊戲的背景也很簡單：一個男孩發現自己深深陷入了一個黑暗計劃，孤身一人逃跑並遭到追擊。

《INSIDE》的操作非常簡單，就是左右上下。全程遊戲並沒有對話，所以導致沒玩過的朋友，會覺得有點不知所措。沒關係，玩下去你就知道怎麼操作了。

《INSIDE》遊戲的畫面也做的非常用心。雖然是一款2D遊戲，但是採用了3D立繪，使得遊戲的代入感更強。無論我們處於什麼場景都會給我們最佳的視角，有時候覺得自己是不是處於一場精美的電影當中。大家可以留心觀察遊戲的場景，無論是跑動時濺起的水花還是浸濕的衣服在奔跑中水分的蒸發，都展現了一個用心作品。

無獨有偶，音樂的處理也是恰到好處，玩遊戲的時候建議大家佩戴耳機。《INSIDE》的配樂也非常能烘托遊戲的氣氛。

《INSIDE》遊戲想要進行下去，就要不斷試錯，全程無對話，無提示。我們要做的就是一次又一次地試錯，直到解開謎題。當然試錯的代價就是死亡，但是這個遊戲並不像《掘地求升》那麼變態，死亡了就回家，我們還是會回到上一個記憶點的。小男孩看似堅強，實際上非常容易死亡：摔死、淹死、被狗咬死、被豬拱死、被搶打死……

所以我們要做的就是怎麼樣避開陷阱，到達下一個解謎點。另外我們在向前奔跑的同時，也要注意周圍的隱藏點，因為這個遊戲不僅僅有主線，還有支線加彩蛋的。只有留意了，並且抓住了支線，才能觸發隱藏結局。

《INSIDE》是一款動作解謎遊戲，我也不能透露太多信息，解謎遊戲還是需要大家去體會的。

不過很多網友都說，就算看了網上的攻略，以及劇情，在自己玩的時候也很難講清楚這個遊戲講了些什麼。

就如莎士比亞所說的「一千個人眼裡有一千個哈姆雷特」，每個人對《INSIDE》的理解肯定都不一樣。但是我想不管大家的理解有多麼不同，認同他是一個偉大遊戲的觀點都是一樣的吧。
