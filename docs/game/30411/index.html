DNF：韓服100級深淵實測，炫彩柱子必出神話，金色柱子必閃史詩

2019-12-24 306條評論 1,300次閱讀 220人點贊

韓服體驗服開放100級深淵已經五天了，而我每天刷200把的深淵只為了看到神話七彩柱子的閃光，但可惜的是一個神話史詩都沒有閃光，千分之一的概率都沒有，看來策劃是真的打算讓神話史詩成為100級版本最強的史詩裝備，所以獲取的難度極高，而千把深淵的成果是我到現在還沒有一個腰帶，所以根本沒辦法湊成套裝。

不過100級的深淵設計感覺非常的巧妙，也非常的有趣，並且也是100級版本全員必須刷的副本，不管你是打算升級裝備還是做裝備，深淵副本產出的材料是100級版本的關鍵性材料。

為什麼必須刷深淵？

100級版本的深淵沒有掉落史詩碎片，但是會掉落智慧水晶碎片，困難模式掉落12-14個，普通模式掉落8個左右，智慧水晶碎片是一個基礎材料，可以通過NPC洛巴赫商店使用100個兌換1個智慧水晶，而智慧水晶的作用就是在諾頓商店兌換【刻印之石】這一個材料的。

刻印之石的作用是什麼？目前刻印之石的作用有兩個：

第一個是95級的畢業史詩裝備升級-升級需要50個堇青石和1850個黑水晶，此外還有一個刻印之石，也就是說大家升級1件裝備至少要刷10次深淵，所以一個大號最少也要100次深淵才行。

第二個是95級的史詩裝備刻印到100級裝備-刻印需要50個堇青石和刻印之石，在強化和增幅11前固定1個刻印之石，但是在打造11之後就需要額外的刻印之石了。

所以就算玩家要升級裝備，為了刻印之石也需要刷深淵。

有趣的柱子設定

100新深淵模式打柱子出現第一波小怪，清理後精英怪出現，柱子的顏色會跟著變化，一般情況下遇到的都是粉色的柱子，這一個柱子是隨機掉落史詩的，但是這一個隨機的概率非常低，甚至可以說刷100次粉色的柱子可能才掉一個史詩，反正直播的時候掉的還都是其他職業的武器。所以看到紫色的柱子就知道這一把黑了。

如果擊殺小怪後出現金色的柱子，那麼這一把深淵必定會閃史詩，只要是金色的柱子一定會出史詩，目前測試的過程中每次遇到金色的柱子就直接閃，可以說是100%出史詩的徵兆。

而如果有幸遇到炫彩柱子，那麼這一把深淵就是你最歐的時候了，因為這一把必定會閃神話裝備，但是目前我已經刷了1000把左右的新深淵，就是沒有看到炫彩柱子閃光，並且這一個柱子打的時候會強制擊退玩家，看來裡面的裝備真的太強了，這和其他的柱子完全不一樣的感覺。

這樣的有趣設定其實還不錯，可以提前預知自己是不是可以出貨，是不是可以閃出神話裝備，不過神話出現的概率也太低了。
