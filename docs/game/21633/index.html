LOL：G2宣布Perkz重回中單位，Caps或將接力轉型ADC

2019-12-18 73條評論 1,724次閱讀 436人點贊

2019年初，Caps加盟G2、Perkz轉型AD的消息震驚了很多人，而在懷疑和調侃聲中，G2戰隊收穫了前所未有的一系列成功，最終獲得S9亞軍。不得不讓人們敬佩Perkz為了勝利讓出中路位置的決定。

不過，隨著新賽季的臨近，G2戰隊又宣布了一條新消息——狂小P斷開連接，歐洲法王阿P回歸賽場！Perkz將回到他熟悉的中單位置，繼續帶隊征戰S10！

G2戰隊配文：今天讓我們告別Uma Jan（Perkz打AD時的排位id），歡迎新中單Luka the Bazooka（Perkz現在的排位id）！Perkz也在自己推特上表示：感謝Uma Jan今年的付出，不過現在已經不需要你了，等以後有需要的時候再叫你回來。

而結合之前G2官博的騷話以及Caps曾表態新賽季要在AD位置上戰勝Rekkles，G2戰隊新賽季的ADC選手應該就是與Perkz互換的Caps了！坐擁兩位歐洲頂級中單的G2戰隊又要在LEC掀起一陣血雨腥風了。
