Shy哥公開破壞IG鐵規則，阿水或無辜慘成背鍋俠，引發玩家們熱議

天天要聞
2019-12-15

在英雄聯盟職業聯賽圈中有一條不成文條例，就是每當世界賽打完後，便迎來了職業選手轉會的窗口期。這時英雄聯盟所有的遊戲俱樂部都不約而同地開始了憋尿大戰，不到最後一刻誰也不知道花會落誰家，誰是全新宇宙戰艦戰隊。最近網上盛傳IG下路Jackeylove要走，因為野輔演他？至今仍無定論，不過可以確定的是Jackeylove要續約IG早就續約了，阿水大概率要走。而且IG這個戰隊挺有梗的，比如IG用不加班以及小IG不斬無名之輩等等，這都是不成文規則。而就在剛剛！Shy哥公開破壞IG鐵規則，阿水或無辜慘成背鍋俠，引發玩家們熱議！

在國服英雄聯盟職業聯賽還處於S3之前的遠古時期時，以PDD為首的老IG便已經成為了一支老牌豪強戰隊了，只不過並沒有打出很好的成績。隨著時間的流逝，長江後浪推前浪，IG俱樂部官方宣布Jackeylove來到IG一隊，於是IG終於組成了完全體：勸架型上單Theshy、討飯型打野寧王、翻譯型中單Rookie、抗塔型射手Jackeylove以及逛街型輔助寶藍！完全體IG喜歡打「四賣一AD以及上單一保四」戰術，而且網上還流傳著IG永不五殺的魔咒！最近英雄聯盟上線了魄羅大亂斗，特別熱愛的Shy哥便立馬去體驗了一番。

這局Theshy玩的是瑞茲，Shy哥使用瑞茲時頻繁利用變態走位來EQ二連消耗對面，搞得對面很不高興。對面五人好像只認準了Shy哥是最厲害的那個，打起架來對面阿木木和大樹只追著Shy哥瑞茲打。有次對面阿木木看到Shy哥瑞茲沒血沒藍，果斷選擇Q上前來單切Theshy，但被Shy哥極限地打出一套EQ二連，再加上隊友奧恩的傷害，瞬間秒殺對面半狀態的阿木木。Shy哥成功拿下開局三分鐘的第一個擊殺數。即使對面瘋狂針對Shy哥瑞茲，但不知不覺間，Shy哥瑞茲還是發育起來了。或許這就是自稱愛發育的男人的美麗啊，屬於知識盲區範疇。

當Shy哥以及三件半套大裝備時，全局除了他之外都是兩件套或者兩件半套，所以Shy哥瑞茲打團有優勢。Shy哥隊友奧恩使用雪球砸中對面小丑，隨後果斷選擇飛過去用肉身開團，但Shy哥他們跟不上啊，結果奧恩白給了。對面五人突然發現這波五打四可以打，便讓阿木木上前開團，然後就來到了Shy哥公然破壞IG鐵規則的表演時間了！Theshy抓住機會先打一套EQ二連帶走對面阿木木，而後對面盧錫安犯了一個大失誤，E上前卻吃了Shy哥瑞茲一發Q技能而被擊殺！

對面在看到己方頓時兩員大將後，即使面對著只有一個人的Shy哥瑞茲，卻開始猶豫了起來。但Shy哥抓住時機，果斷閃現引誘對面開自己，對面沙皇先上當慘被EQ二連暴擊而死，對面大樹和小丑想擊殺Shy哥卻也無能無力，最終Theshy憑藉自己過硬的操作拿下五殺，畢竟Shy哥有著多年的以一敵五的經驗！大家都知道，IG流轉著一條不成文的鐵規則，就是IG永不五殺，但這次Shy哥卻做到了！為什麼呢？或許就是Jackeylove先帶的節奏吧，阿水一手卡莎對陣TES拿下五殺並成功助力IG挺進世界賽！

IG永不五殺似乎是條不成文的鐵規則，但只要是職業選手或者說是位熱愛英雄聯盟遊戲的玩家，都渴望著五殺，那些過激的玩家說阿水壞了規矩應該背鍋簡直是無稽之談。就Shy哥瑞茲這波五殺引發了玩家們的熱議。

最後不知道大家就在剛剛！Shy哥公開破壞IG鐵規則，阿水或無辜慘成背鍋俠，引發玩家們熱議！歡迎大家的評論和關注喲。
