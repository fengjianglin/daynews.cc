總決賽QG全場被碾壓，飛牛為何沒能站出來，張角早已看破一切

天天要聞
2019-12-17

KPL總決賽是超玩會取得了勝利，而且是以4比1的大比分戰勝了QG，很多人說總決賽還不如季後賽有意思。確實是這樣的，AG超玩會和E星的那場比賽，兩支隊伍最起碼是打到了巔峰對決，每場比賽都是勢均力敵，打得有來有回，其中還多次上演了逆風翻盤的局面，整場比賽也要好看的多。

反觀AG和QG的總決賽呢，完完全全就是一邊倒的局面，AG超玩會單方面碾壓了QG，第二局比賽如果不是飛牛站了出來，那麼AG很大概率上就是4比0了，藍柚都沒能獲得上場機會。那麼在其餘的幾局比賽中，飛牛為什麼沒能站出來呢？張角早已看破一切！

看總決賽的時候，不知道大家有沒有一個感覺，那就是AG的BP突然好起來了，再也不是之前的「光之BP」了，甚至可以說在BP上壓了SK一頭，這才使得超玩會每一局前期都能拿到優勢。

主要是BP的方向進行了轉變，所有人都知道QG打得是四保一的陣容，就是要保著刺痛這個核心點發育，以往的所有戰隊都是針對刺痛，在BP上就開始限制刺痛。但是月光沒有這麼做，他這幾局比賽很少ban射手，刺痛每一局都可以拿到自己想用的射手。

月光針對的是QG的另一個點：飛牛。看過比賽的都知道，超玩會每局的四個ban位，至少有兩個是給到了邊路英雄，這樣在很大程度上就限制了飛牛的發揮，正如張角在紀錄片中說的那樣，就是要盯死飛牛這點，不讓飛牛拿到拿手的英雄，到了第四局第五局的時候，飛牛幾乎就沒有什麼英雄能用了。

在我看來，QG唯一不穩定的點就是飛牛，他是一個典型的大賽型選手，越到了這種重大比賽，飛牛的表現就越好，QG取得前幾個冠軍的時候，都離不開飛牛的出色發揮。他也是FMVP選手中，唯一的邊路選手，可見他的邊路英雄真的是獨一檔的。

但是到了這一屆的總決賽中，張角早就看破了QG這支隊伍，超玩會直接在BP上就開始限制飛牛，不讓他拿到非常擅長的英雄，這樣在QG處於逆境的時候，飛牛很難站的出來，而QG的其餘四個人，可以說AG根本沒有放在眼裡，在BP環節都沒有加以限制，五場比賽打下來行雲流水，整局都在壓制著QG。

只有拿到狂鐵的一局飛牛站了出來，其餘四局飛牛都是無力回天，從這一點也可以看出，AG超玩會對於飛牛的針對有多到位了。
