國內玩家萬人求出，出海遊戲亮點不斷，為其點贊

2019-12-23 309條評論 4,516次閱讀 1701人點贊

最近一款國產遊戲在國外火了，聽到這樣的消息我還是發自內心的高興的，這證明我們的國產遊戲正在走向強大，走出國門，得到其他國家玩家的認可。這款遊戲雖然在國外發行，但是被中國玩家發現後，也同樣贏得中國玩家的喜愛，並自發變成自來水，吶喊助威並希望這款遊戲在國內上線。

說實話一款手游能夠出現自來水玩家還是十分少見的，於是我就查詢了一下這款叫做《劍與遠征》的遊戲，讓我有點為自己的無知感到羞愧，這款遊戲還真是憑藉自己的實力獲得玩家的喜愛的。

這款遊戲在國外的影響力還是非常強的，在19年11月中國手游收入榜中居然排第九，能夠排在他前面的都是網易、騰訊之類的大廠遊戲，不過莉莉絲的實力也不差，大家都知道的，之前的《刀塔傳奇》也是火遍遊戲圈。

但是這款遊戲的亮點究竟在哪裡呢？我們一起來看看自來水們的評價吧。我在tap上找了半天，感覺基本都是5星，終於找了個4星，看看這評論，依然和其他玩家看法差不多，就是喜歡這款遊戲顏值高、實力強，唯一對這款手游不滿意的可能是因為這是國際服，沒辦法氪金十連抽，這要求真是有點膨脹。

當然除了這些國內氣人的自來水玩家之外，一些國外玩家也是對劍與遠征十分喜愛的，很多粉絲玩家也是自行組織建立了網頁供大家討論，還有很多顏值粉也是紛紛cos劍與遠征的英雄，在官方萬聖節上就有很多粉絲亮出了自己cos的圖片，不得不說遊戲設計的美，玩家的cos更美。多點這樣的遊戲好友，這遊戲我也玩。

我的執行力還是非常強的，為了更好的跟大家揭秘這款遊戲，我也是快速的下載了這款遊戲進行了試玩，經過了幾個星期的體驗，我很負責任的告訴大家，這款放置卡牌遊戲千萬別玩，因為上癮了怕你戒不掉，原來這款遊戲的顏值只是假象，真正的樂趣在於如何取得勝利。

這款遊戲中的卡牌是有派系的，不同的派系都有自己各自的屬性，合理的提升屬性和排兵布陣就能讓你輕鬆的獲取勝利，有了勝利才會變得更強。不說了，我又遇到瓶頸了，去找國外大神探討一下陣容了。
