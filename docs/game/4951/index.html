LOL：你的青春從哪裡開始？從S1到S10，盤點記憶中的召喚師峽谷

2019-12-03 0條評論 16次閱讀 0人點贊

摘要

​不知不覺，LOL已經陪伴我們走過了10年。十年前那個熟悉的登錄界面應該還能喚起很多人的回憶吧？不僅僅只是這個登錄界面，聯盟中最重要的地方——召喚師峽谷十年來的變化也多得完全數不清。跟著指南君一起，從最初的地方開始，回憶峽谷的今生前世吧！

​不知不覺，LOL已經陪伴我們走過了10年。十年前那個熟悉的登錄界面應該還能喚起很多人的回憶吧？不僅僅只是這個登錄界面，聯盟中最重要的地方——召喚師峽谷十年來的變化也多得完全數不清。跟著指南君一起，從最初的地方開始，回憶峽谷的今生前世吧！

S1-S5 前世：初代召喚師峽谷

這版的召喚師峽谷放到今天，應該只能是很多老玩家的回憶了。自從S5季前賽中將召喚師峽谷改到今天這個模樣，不知不覺就已經過了5年，已經趕上了這版峽谷的壽命了啊……

這版峽谷充滿著一股濃濃的刀塔氣息，不僅是峽谷的材質，英雄的面板，各種裝備的圖標，整體UI的風格都和隔壁的另一款MOBA端游相似得不行。

不僅如此，還有許多現在已經成為回憶的裝備道具都來自於這個時代，殺人刀，自然之力，龜殼，洞悉重生，可以說，很多老玩家的青春都存留在了這裡。

S5-S6 今生：你我熟知的那個峽谷

其實自從S5將召喚師地圖重製為現今這個模樣之後，地圖本身變化算起來不算太多。之所以把這個時期單獨拎出來，是因為這一時期不僅修改了召喚師峽谷，更將一眾英雄圖標，召喚師技能，裝備圖標等等更新到了他們今天的樣子。新舊閃現圖標，新舊冥火之擁，老玩家曾經大呼變態的「五龍buff」，會給附近友軍套盾的二塔，激光炮一樣的高地塔水晶樞紐塔，高地新開的兩個出口等等，都來自於這一時期。也許現今很多改動看起來都有些摸不著頭腦，但不得不說，這一時期的改動很多都讓玩家們眼前一亮，新鮮感是真的非常足。

S7-S10：來世：未來的峽谷走向何方

這一時期，峽谷本身已經沒有多大的變化了，更多的改動是針對峽谷內的地圖資源。從S7紅藍buff失去小怪，加入元素龍；S8整合了符文天賦；S9引入鍍層；到現在S10元素峽谷的誕生，都是峽谷配合地圖資源而進行了改動。偷buff再也不能留一隻延遲刷新，下路再也不能開局換線到上路發育，前期再也不能輕鬆快速抱團拆中塔……那些熟知的套路也許今日的你還能回憶起來，但今日的峽谷可不會再讓你這麼玩了。這一時期峽谷為了迎合各種特殊模式和節日，曾變得恐怖陰森，也曾披上過銀裝，末日人機，血月殺，冰雪節特別峽谷，我們應該很快就會再見到他們。

也許，在將來的某天，峽谷還會再迎來一次修改，也許很快它就會讓許多許久沒有打開過聯盟的玩家們完全認不出來。但只要那個身披藍色皮膚，手持大寶劍轉轉轉不停，一口一個德瑪西亞一個正義的憨憨還會出現在那（也許正追著一個你不認識的英雄），你還是會微微一笑，心裡感慨一句，

那是我的青春。
