被罵作狗策劃卻不還口？這群遊戲策劃內心到底有什麼秘密？

天天要聞
2019-12-03

摘要

&quot;我氪了十單居然全沉船？&quot;&quot;禮包這麼多，你游吃相這麼難看的嗎？&quot;&quot;你游零氪低氪就是就是給大佬提升體驗的人柱力？

“我氪了十單居然全沉船？””禮包這麼多，你游吃相這麼難看的嗎？””你游零氪低氪就是就是給大佬提升體驗的人柱力？”

……

相信刷過遊戲論壇的小夥伴都曾經見過類似的口誅筆伐~

任何一個遊戲，一旦出現了問題，第一個被罵的顯然就是那群”狗策劃”~

但是，以小刀常年混跡各大遊戲圈的經歷來看，在這些看似嚴重的問題背後，卻是隱藏著”狗策劃”們的難言之隱，那些不能說出來的答案也逐漸變成了秘密~

下面小刀就來分析一下文首提到的四個問題，一起來看看策劃們的”難言之隱”吧~

在小刀的理解中，抽卡大概算是玩家們最關心的一個玩法。

而提到抽卡，必然少不了概率這個難過的關卡。

歐皇們月卡養號一發入魂，酋長們十單猛氪不見分文——這種現象首次被曝光似乎是FGO和崩壞3？

而隨著這種現象的”普及”，一些罵策劃們暗改概率之類的說法幾乎成了業界潛規則~

也有小天才推論：UR概率3%，那不就是抽一百發必出三個UR？

對此策劃們可是哭笑不得，要知道3%抽到的另一面可是97%的概率抽不到！試想，強化武器的時候，成功率97%，小夥伴們是不是下意識認為強化必成功？

簡單的道理，3%的出貨概率是非常低的，低到別說一百發，就是一千發不出貨也是有可能的~

而為了照顧玩家們的體驗，很多遊戲的策劃便進行了”暗改”，比如設置保底，不論是明面還是暗地裡的保底，當玩家抽卡累積到一定次數時，下一發抽卡必定出貨這樣子……

想想都是一股血淚……

除開抽卡，各大手游的最主要盈利方式便是讓人看得眼花繚亂的遊戲禮包了。

但禮包一多起來，作為玩家便經常會有兩個感受，不氪吧肯定會被大佬吊打，變成提升他們體驗的人柱力，氪吧禮包這麼貴，這麼多都氪起來一般人哪承擔得起……

對於這個問題，小刀的理解是：遊戲策劃畢竟也只是為遊戲公司打工的存在，首先他們頭上頂著一個叫做KPI的東西。

為了完成績效，”狗”策劃們也不得不掏出千萬種促進消費的禮包，其內容從捆綁銷售、飢餓營銷等等比比皆是。

魯迅老先生有句話怎麼說來著？”策劃做過的遊戲禮包比咱玩過的遊戲還多~”

正所謂人非聖賢孰能無過，各大UP主們如今都紛紛開始恰飯了，策劃們也是一樣的嘛！

但是！即便是這樣，咱這種平民玩家也不應該成為大R們的遊戲體驗人柱力！

本來玩遊戲就是享受一個慢慢變強的過程，但這一點放到以服務大R玩家為目標的遊戲中，即便是努力過、變強了，也完全不會得到一個好的結果！

這也就使得業內那些一次性買斷、終身暢玩的遊戲變得難能可貴。玩好一款網路遊戲，從幾萬塊到幾十上百上千萬都是有可能的，但玩好一款買斷制遊戲，在小刀能接觸到的領域中，最多也不過四五百塊~

而在氪金元素泛濫的手游市場中，同樣也是有著與主機遊戲一般清流的存在。

《魂之刃：巨龍城堡》，一個以硬核單機為賣點的手游。

其遊戲風格貼近於暗黑西幻，故事背景為典型的神界、教廷、人間、地獄四方角逐。

得益於遊戲本身主打操作難度的設計，在《魂之刃：巨龍城堡》中玩家幾乎看不到任何騙氪元素——除了買斷遊戲要花的30塊錢。

在其即將到來的版本更新中、遊戲新增了一個擁有全新技能和特效的時裝武器，也可以通過遊戲中的裝備碎片來合成。就算是不想刷材料，直接買時裝也只是幾杯奶茶的價格而已。無論是氪度還是肝度，都算得上是業界的良心~

而說起這個良心廠商的良心操作，小刀不能不點評一句：奇葩。

想恰飯卻又怕口碑下滑，於是就把遊戲內的血瓶（類似魂遊戲中的原素瓶）做成了氪金道具，整個遊戲內購商城只出售這一種道具——

這操作，真的怪。

由於這時候遊戲還只是停留在小眾的階段，若是遊戲走紅，玩家群體一多起來，勢必會有人拿這一點做文章！

於是在決定前往參展2019TGS之前，早在兩年前的TGS展會就獲得「最佳獨立遊戲」殊榮的《魂之刃：巨龍城堡》，便做出了一個極為”睿智”的舉動：刪除遊戲商城。

這下可好，遊戲除了買斷需要花的錢之外，便沒有了任何充值渠道。

發現這個事實的策劃立馬慌了：”沒收入怎麼恰飯？”

於是整個項目組立馬投身開發，連夜加班，最後終於討論出一個決定：做收費時裝。

沒錯，為了不影響這款遊戲的實際體驗，策劃們決定以氪金時裝的形式來收費，氪金內容與遊戲體驗幾乎不重疊。

充錢之前打不過的BOSS，充完錢買下所有的時裝你還是打不過。

怎麼樣，是不是很真實？

這般操作說良心也確實良心，不過小刀作為一個優秀的人類，必然會在這裡點評一句：真香！

藉由氪金時裝這個發展方向，遊戲中也就出現了上面提到的全新技能武器——熔火之錘。再加上原本有的大斧武器，遊戲便擁有了不同技能的兩種武器。

儘管這些內容在目前看來還略為不足，但根據官方公布的消息，後續除了武器之外，還會推出全新的可操作角色，遊戲中主角的老婆伊絲特將會化身可操作角色，供玩家征服整個神界！

想想都覺得刺激！（我在想Peach）

對此小刀只能說，祝貴遊戲越做越好！畢竟現在不騙氪的良心手游，實在是難得啊！
