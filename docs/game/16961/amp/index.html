《雲頂之弈》新羈絆“月”前瞻，綠叉削弱，法爆、刺客登頂T1

天天要聞
2019-12-15

隨著不久前的《英雄聯盟》9.24版本更新，《雲頂之弈》也迎來了最新一波的改動，期待已久的盧錫安夫婦終於正式登場，此外地獄火羈絆的重做以及新成員阿木木，也都對之前的玩法有著不同程度的影響，不過就在近日官方又再一次的放出了即將在雲頂10.1版本出現的新羈絆「月」。

如果大家聽到「月」這個字就想到皎月可能會加入其中就太天真了，聖槍遊俠不是遊俠、沙漠死神不是沙漠、影流之主不是影，按照這個邏輯皎月女神也不會是「月」，果不其然，在即將到來的《雲頂之弈》10.1版本中，設計師並沒有改變現有皎月屬性羈絆的打算，而是選擇加入了兩個新的棋子「卡爾瑪」和「蕾歐娜」，你沒有聽錯太陽的代表日女加入「月」了，而官方則是義正言辭的告訴我們羈絆是根據英雄皮膚來定的，本次加入月的是「月蝕蕾歐娜」，我一時間竟找不到反駁的理由。

羈絆「月」的成員只有「卡爾瑪」和「蕾歐娜」，但是buff效果卻是可以施加全隊的，隊伍中的成員每隔七秒就會獲得20%的暴擊率，20%的暴擊傷害，20%的法術強度（最多可以疊加4次），這個新的羈絆效果雖然聽起來比較墨跡，但實際上因為蕾歐娜和卡爾瑪分別是1費和3費卡，所以性價比方面絕對划算的，幾乎是全屬性的增加不僅可以使得刺客鹹魚翻身，就連法爆流的陣容得到的提升也不止一點兩點。

不過這畢竟不是第一季的《雲頂之弈》了，失去了火炮的加持，似乎滿暴擊也不一定是一件好事情，畢竟4費的螳螂都能被一把綠叉秀到死，如果暴擊都滿了，對面哪怕只有一個一星的英雄帶綠叉，那不等於是自絕後路嗎？事實上，在這次9.24版本更新時，官方就已經做出了對綠叉的削弱，現在版本的綠叉只能抵擋掉來著暴擊觸發的額外傷害，攜帶者還會受到來自攻擊者未暴擊的基礎傷害，並不能做到完全的閃避了，而在10.1版本中設計師還會再次的對綠叉進行削弱，以此來鞏固「月」的暴擊性價比。

一方面是綠叉即將迎來又一次削弱，而「月」的羈絆雖然不是由皎月組成但對皎月的增幅也非常大，要知道皎月不僅可以享受到刺客帶來的爆傷和暴擊加成，同時法術加成的護盾才是皎月能夠攜帶科技槍1v5的核心，帶上科技槍和法爆的皎月在享受著來自「月」暴擊、爆傷加成的同時，法術強度也能夠提供護盾厚度和護盾傷害，可以說是最大受益者了。以外，大家認為新羈絆除了對皎月非常「友好」，還能讓哪些英雄強度有質的飛躍呢？
