1、命運之刃能否換得極品

這也是有關這把傳奇武器的一個傳說，因為這個任務之前是不限等級做的，一般會玩的大神30級出頭就可以打過，而命運之書則需要35級才能換到。一個非常誘惑人的傳言也就來了：換取命運之刃的屬性是隨機的，最低上限屬性為16，可是玩家有機會換得最多加7點屬性的命運！也就是說拿到手的白刀，最好可以到達12-23（筆者覺得這不就是不帶準確的怒斬嗎……），並且等級越高，換到極品的幾率越大！

人性自然是貪婪的，傳奇玩家追求極品裝備的「恆心」恐怖至極！於是乎，在這個任務留存之際，不少老玩家手握命運之書，一路把自己等級憋到40級，才去換取，有的甚至到任務不能做了之後，還沒有把書用了……

真的是很可惜……本來有機會摸一下這把稀世武器，只因為自己有點貪，最後與它失之交臂了……

最後至於換沒換到，基本上無從考證，如果真的有換到23攻擊的命運之刃，再拿去沙巴克升級七點的話12-30的攻擊，在老傳奇中是一個何其恐怖的存在，它絕對可以秒殺全場！

背包里的命運之刃

2、命運之刃無視防禦之說

戰士職業相傳能無視其他職業防禦的道具有兩個，一個是騎士手鐲，另外一個就是神器命運之刃。法師只要帶盾之後，就成為戰士職業的大敵，在刺殺外掛沒有普及之前，大多數戰士PK還是很菜的，甚至有人不會利用刺殺走位PK，近身硬懟成為了最常見的PK方式。

這種戰士遇到會盾的法師，基本上就頭疼之極，因為找不到克制他們的方法，衝上去砍也砍不動，法師可以直接頂起戰士近身吃紅硬扛！

這時候就有人冒出來云云：命運之刃雖然其貌不揚，攻擊力只有12-16點，但是它擁有一個其他武器無法比擬的最強隱藏屬性，無視被攻擊者的物理防禦，在這把刀面前，你就是一張紙，刀刀見肉，次次見血，命運之刃發揮12點攻擊就砍你12點血，發揮16點就砍你16點！當然它的作用肯定不是在自身的發揮上面，它可以帶動使用它的人「全身發揮」，如果有聖戰一套的玩家，40級戰士攻擊力可達到50點左右！

拿到手上，去掉穿戴裝備，提升不夠攻擊力

看到這裡有的玩家可能會笑！50點左右？有什麼用？當然50點攻擊對戰士來講，真的沒用，50點攻擊揮出的烈火劍法能達到多少傷害呢？無視防禦的情況下約為125點左右！在補一刀共計傷害170點，法師40級只有193點生命值，也就是說38級開3級盾的法師，一刀烈火加一下平砍，直接帶走……但事實真得是如此嗎？
