《毀滅戰士：永恆》支持中心武器位置經典模式

天天要聞
2019-12-16

貝塞斯達日前宣布，《毀滅戰士：永恆》將支持經典的中心武器位置模式。換句話說，玩家可以像傳統的《毀滅戰士》遊戲那樣，把武器放在屏幕中央進行遊戲。發行商也在最近的直播中分享了一些新的遊戲片段。

在《毀滅戰士：永恆》，玩家將繼續扮演毀滅戰士在各種光怪陸離聞所未聞的世界裡使用各種強力武器擊殺惡魔。玩家將在超前的第一人稱視角戰鬥中體驗速度與力量的終極結合。

這款遊戲將使用id Tech 7引擎，並承諾將呈現我們在《毀滅戰士》遊戲中從未經歷過的新關卡/地點，包括比之前的遊戲高10倍的幾何細節，以及超過之前id Software所有作品的材質保真度。

英偉達和貝塞斯達已經宣布，這款新作也將支持實時光線追蹤。然而，這些效果在剛發行時是不可用的，與之前的《德軍總部：新血脈》類似。

《毀滅戰士：永恆》將於2020年3月20日在Xbox One, PS4和PC上推出(Switch版具體發行日期待定)。
