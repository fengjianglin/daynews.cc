爐石傳說：158的冬幕節合集香不香？2個細節表明：暴雪這次很大氣

2019-12-24 565條評論 3,551次閱讀 954人點贊

作為巨龍年的”壓軸好戲”，巨龍降臨資料片已經上線了有幾天的時間了，相信80%的小夥伴都被薩滿和海盜戰折磨得死去活來。這不，由於薩滿太過強勢，暴雪也直接對無面和元素師等卡牌進行了削弱，從而緩解了一下薩滿泛濫的局面。在薩滿被削弱之後，T7獵和樹人德的出場率也大幅提升，德魯伊的勝率也跟著上漲了不少，然而對於德魯伊資深玩家來說，最近又傳來了一個好消息，那就是新英雄黑澤爾巴克也正式到來了。

可能有小夥伴會問了，這算是什麼好消息，黑澤爾巴克這麼丑，我才不會用！對此濤咪心中再次閃過了兩個字，那就是”真香”。吵吵騎士和摩戈爾剛上線的時候大家也覺得這種奇葩英雄怎麼會有銷路，結果後來大家發現了他們的”精神污染”屬性之後，又紛紛悔恨當初為什麼沒有買。然而這次黑澤爾巴克的獲取方式是購買冬幕節合集然後就能免費贈送，只需158，30卡包和黑澤爾巴克帶回家！

另外說到這個冬幕節合集，158RMB就能獲得30卡包，單卡包的價格低至5.2，可以說是名副其實的冰點價了，一共包含5個版本，每個版本6卡包。對於剛剛回歸的老玩家來說真心是再合適不過。再加上免費送黑澤爾巴克，單從這兩個細節來說本次的冬幕節合集還是值得入手的，暴雪又大氣了一次。

尤其是小德玩家，就沖黑澤爾巴克就值得來一發，畢竟樹人德現在這麼強勢，用上這個皮膚真的可以讓嘲諷力瞬間拉滿。另外大家有沒有注意到，黑澤爾巴克有點像卡通人物托馬斯小火車，可以說是”一萌遮百丑”了。而且是限時出售，過了這村可就沒這店！

另外對於最近沉迷於酒館戰旗無法自拔的小夥伴，濤咪還有2點要提醒，首先是冬幕節亂斗千萬不要錯過，除了獲得卡包之外，還能白P卡背。另外就是獵人女王皮膚也在商城上架，現在T7獵這麼強勢，喜歡獵人的小夥伴完全可以考慮入手。而且在濤咪看來，女王皮膚可以說是獵人最帥氣最飄逸的皮膚了。

那麼大家認為，本次冬幕節合集是否很良心呢？
