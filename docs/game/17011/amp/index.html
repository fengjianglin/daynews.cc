破壞平衡的最強卡組，迦拉克隆薩削弱前的絕唱

天天要聞
2019-12-14

隨著爐石傳說新版本巨龍崛起的上線，各種套路也是層出不窮。多樣的卡組讓這個遊戲重新恢復了生命力，在新版本上線的這幾天里，各個主播、職業選手、高手玩家都在探索誰才是新版本第一卡組，都想要用最強的卡組來戰勝對手，輕鬆上到傳說段位。我們今天要講述的這套卡組以百分之七十的勝率統治了新版本，讓設計師看到了它的不平衡性，慘遭熱補丁削弱。

迦拉克隆薩的崛起

新版本上線之前，當我們在查看新卡的時候，就發現了薩滿最新英雄牌-風暴巨龍迦拉克隆的強度。與其他的巨龍牌不同，薩滿的巨龍化身注重的是解場和節奏。祈求的作用是召喚一個2-1的突襲隨從，這一點在對局中非常重要。眾所周知的是，爐石傳說對戰模式最重要的就是前期的節奏，如果能拿到前期攻勢，對手就很難翻盤。薩滿以其獨特的突襲祈求讓他在新版本上線後登頂第一，同時薩滿這個職業的傷害、回復、資源都很多，完美的搭配讓薩滿在九職業中脫穎而出。

迦拉克隆與薩滿任務的完美配合

這套迦拉克隆薩登頂的原因除了強度高以外，還跟上個版本的任務有著莫大的關係。上個版本末期，戰吼薩也是統治了當時的天梯，這次新版本出現後，迦拉克隆的新卡讓戰吼薩重出江湖，一方面新卡的補充讓薩滿更加簡單的完成任務，另一方面任務的獎勵讓加拉克隆薩的強度得到很大的提升。配合雙倍戰吼的效果，我們可以在9費打出四個8-8突襲，一把5-2的武器，一般的職業看到這一幕基本就投降了，就算沒有直接投降，我們還有導演”沙德沃克”進行第二次返場。

迦拉克隆薩在天梯上的肆虐

在這套卡組出現之後，天梯上很快就變為了薩滿內戰，一眼望過去都是薩滿。因為其強度和出場率的不平衡，讓設計師們對薩滿進行了緊急削弱，雖然還沒有確定具體的削弱方案，但是這一舉動肯定會對目前天梯的形勢有很大衝擊。個人認為這次削弱會對滅世巨龍英雄牌或者任務的獎勵進行調整，雖然將要削弱，但是目前迦拉克隆薩還是天梯第一卡組，我們可以放心的繼續使用，就算進行削弱，也無法改變它的強度，合一套薩滿卡組絕對是目前最值得的投資。

這次新版本的上線讓爐石傳說恢復了生命力，很多退遊玩家看到如此豐富的套路和卡組紛紛回歸，讓爐石傳說重回卡牌遊戲前列。希望設計師們可以以此為依據，多推出新模式和新套路，讓玩家們得到不同體驗的同時，讓遊戲獲得更高的生命力。這次對於薩滿的削弱其實也是對遊戲生命力的保持，否則天梯成為薩滿內戰，也會大大影響玩家體驗，像生物圈一樣有著相生相剋才是競技遊戲的本色。
