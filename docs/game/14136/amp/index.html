LOL新版本預告：1個新英雄1個史詩級加強12款新皮膚

天天要聞
2019-12-11

★遊戲馬蹄鐵原創

本周美服將更新9.24版本，也就是在全明星上玩的那個擁有新英雄的版本。這個版本除了包含慣例的一些平衡之外，新英雄和皎月的改動以及12款新皮膚是最受關注的。很少有某一個版本能夠一口氣上12款皮膚，由於數量太多我們之前也沒有預告過，其中包含常見的海克斯限定新款，新主題Sugar Rush，以及Dawnbringer和Nightbringer系列（也就是之前的黎明銳雯和黑夜亞索同一系列）。

新英雄

全明星上已經看過很多職業選手玩過這個英雄了，相信大家也都了解過了。很有特色的一位ADC英雄，擁有5把武器，沒有E技能，最常使用的就是Q和W，同時可以裝備主副2把武器。

無E英雄，技能複雜程度卻是史上之最！

雖然缺少一個E技能，但是由於他的5把武器主副搭配都會有不同的Q技能和大招效果，因此他也是史上最複雜的英雄。另外由於同時只能裝備主副2把武器，什麼時候該用什麼武器，每把武器50發子彈該如何利用，什麼時候切換充能，這個英雄的強度和你的腦容量成正比。

皎月史詩級加強

這個版本其他英雄改動可以忽略，但是皎月你是必須知道的。全明星上Faker玩過這個改版後的皎月，她的E和R得到了互換。這也是將皎月的高爆發大大提前了，以前6級後才可以施展QR這種月光突刺的能力，而現在你只需要2級使用QE連招就能打出過去QR的效果。月光重置CD效果依然保留，而過去的E技能也是變成了一個更大範圍AOE控制R，還給予了不小的傷害加成。總之新版皎月是得到了一次史詩級加強。

12款皮膚

Sugar Rush系列（伊芙琳，布隆，炸彈人，基蘭）

海克斯限定烏鴉斯維因

黎明使者和黑夜使者（圖為至臻版盲僧）

黎明使者：卡爾瑪，豹女，索拉卡

黑夜使者：盲僧（包含一個至臻版），吸血鬼，索拉卡（沒錯，索拉卡擁有一個黎明和一個黑夜，兩種使者身份皮膚）

除了新英雄自帶的皮膚之外，這個版本一共有12款新皮膚上線。以上都是在9.24版本的內容，本周將在美服更新，國服也即將更新。
