三國殺鬥地主一大批武將新加入，地主喜添仨惡霸

2019-12-14 418條評論 1,474次閱讀 374人點贊

相信只要是在12月13日下午和晚上斗過地主的人都是一臉懵逼：卧槽發生了什麼？是不是出了BUG？我怎麼碰到了留贊？我怎麼碰到了小學雲？我怎麼碰到了沮授？我怎麼……

好了，你不用怎麼了，這次還真沒出BUG，又有一大堆新武將加入鬥地主將池了：

這一次還真是一點風聲都沒收到，三國殺的策劃大大們就給了我們一個大大的驚……驚喜還是驚嚇？這一次加入鬥地主將池的總計有十三名武將，不過粗略看一下能改變當下鬥地主環境的也就三個武將：留贊、沮授和小學雲。次一級的能夠成為常駐農民的大致也就華佗、界周泰了。總體來說，玩家們的情緒是「悲喜交加」。

悲什麼呢？自然是悲傷留贊和沮授這兩個土豪將的加入。在之前的版本中，土豪將就已經是佔據了高勝率地主的大半江山了，什麼禰衡啊、麹義啊、許攸啊，已經把平民玩家們虐得懷疑人生了。沒成想，而現在轉頭過來，又要多一個留贊和一個沮授了，平民玩家真是太南了，男上加男……

此情此景，我想唱歌一首：「沒有一點點防備，也沒有一絲顧慮，你就這樣出現，在我的世界裡……「

有人可能覺得子佩說得太誇張了，危言聳聽。子佩這樣來舉個例子吧：眾所周知，馬良在當下的鬥地主環境中依靠自書刷牌硬是擠進了地主第一梯隊，而留贊呢，他刷牌的能力完全是吊打馬良，還先天比馬良多了一體力上限。至於沮授呢，也許刷牌能力跟馬良不相上下，但是！他的保命能力直接能讓大部分農民武將絕望……

當然，也不用那麼絕望，以上只是悲傷的內容，這不還有高興的內容嘛：小學雲即將成為平民玩家鬥地主又一神將。雖然小學雲被罵得厲害，但在鬥地主模式里，小學雲有望在繼馬良、孫魯育之後，成為第三個平民神將。

小學雲的價值在於能攻能守，當地主的時候兩次沖陣，要是再配上個過河拆橋或者順手牽羊，農民一輪下來非死即傷，然而即使挨了一頓毒打之後，也並不意味著能反擊了，你打他還要考慮會不會反被他摸牌……

這一次的大規模鬥地主將池更新確實是令人感到非常意外，不過還是像子佩說的，其實也就留贊、沮授和小學雲這三個武將有實際操作意義，其他都可以先關小黑屋。除此之外，鬥地主商城也更新了，現在可以用金票買盒子了哦~
