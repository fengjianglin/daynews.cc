魔獸懷舊服：你選愛情還是選擇狂暴戰？磚叔KBZ全面攻略

2019-12-17 588條評論 1,752次閱讀 334人點贊

關於ZS去打DPS，這在15年前魔獸60版本的時候幾乎無法想像，那時候只有混跡戰場的武器戰和副本里的1、2、3、4、5、6T，戰士幾乎從沒以狂暴姿態存在過，好在經過正式服長久的洗禮，版本末期戰士長期霸佔PVE團本輸出第一梯隊成為常態，懷舊服里玩家對KBZ普遍認可，於是，三項全能的戰士成為了懷舊服里最耐玩的職業，沒有之一！

然而基於暴雪對魔獸世界60年代的職業設定，戰士輸出裝匱乏的問題幾乎跨越60代所有階段的團本。狂暴戰只要選定，就必須去跟盜賊獵人甚至增強薩和懲戒騎搶裝備，這就讓戰士成為懷舊服里異常爆肝的職業，即便是土豪玩家，依然有許多裝備掉率非常渺茫，每天泡在副本里消費都不見得能看到裝備的影子，比如黑石深淵競技場的野蠻角鬥士胸甲~

三個月來，懷舊服里的老朋友們已經接連有幾位曝出情感危機，目前上線時間穩定減少，選擇狂暴戰，也許就要面對老婆的狂暴的姿態，愛情和信仰你到底作何選擇？

好了，進入正題，先給大家普及一下戰士的基礎屬性運算規則：

1力量=2AP

19.5敏捷=1%暴擊

8.6AP=1DPS(雙持)

14AP=1DPS(單手武器,雙手武器)

副手傷害懲罰50%。

種族選擇上：LM方首推人類，耍的一手好劍；BL方則是獸人戰，掄的一手好斧頭。

命中達標：當前階段，團本KBZ命中8%以上視為達標，在此基礎上可以追求的屬性為暴擊和AP，暴擊收益高於AP，計算公式就不說了，磚叔可不是什麼數據狗。TAQ和NAXX命中要求11-12，這是後話，先不提。

天賦一般選用標準的17/34，但是！

保證狂暴雙持以持續保持怒氣循環是狂暴戰的基本操作，不過由於部落方薩滿風怒圖騰的存在，雙手武器削骨劍猛擊流狂暴戰在很多情況下DPS要超越雙持狂暴戰，當然，斬殺階段悄悄換上兩把匕首刷15怒氣斬殺也是必備操作。猛擊削骨戰的好處在於適應性極強，點出猛擊天賦把這種戰士丟進戰場，依然一刀一脆皮，狂暴的很~~

裝備選擇概述：

眾所周知，狂暴戰在目前團本MC中除了武器，其他裝備沒有什麼選擇餘地，套裝都是防戰專屬，散件有三：烈焰衛視護手、衝擊腰帶、穩固護腕。

項鏈選擇黑龍頭任務項鏈，戒指則要跟盜賊和獵人們飈起埃古雷亞指環和迅擊戒指。上述散件相對廉價，只有KBZ有需求，戒指武器和其他鍛造裝基本都是天價購買，飾品黑手飾物任務獲取，屬於最容易的，而正義之手則極度看臉，刷吐將軍的玩家們都懂。

值得一提的是狂暴戰的現階段榮譽軍裝屬性極為優秀，除了命中短缺，各種暴擊和攻強加成全面碾壓MC裝備，奧山戰場崇拜的戒指也是DPS夢寐以求的極品，然而軍銜到底有多難說，作為BL絕對優勢服的磚叔來講已經難以用爆肝來形容，公會裡日夜在線已經連續肝了一周的老大哥今日結算軍銜竟然只有3級中士，路漫長而修遠兮~~

目前可以獲取的軍裝如下，大家量力而為，泡戰場之餘別忘了養肝護肝~~

懷舊服現在屬於MC階段版本末期，BWL開放在即，還有不到9個CD，大家所能提升裝備的時間已經不多了，藍裝方面日常需要獲取的磚叔就不再過多介紹，現版本畢業裝整理一下送給大家，氪金就好，想自己搜集材料去鍛造…….還是算了吧~

獅心頭(鍛造)，奧尼克西亞項鏈(黑龍頭)，強擊護肩(黑上)，黑爵士披風(斯坦索姆尾王)，野蠻角鬥士胸〔黑石深淵競技場獸人BOSS)，穩固護腕(MC)，烈焰手(MC)，衝擊束帶(MC)，踏雲護腿(鍛造)，驅逐者脛甲(世界掉落),埃古雷亞指環(MC),迅擊戒指(MC),殘忍利刃(MC),犬牙(MC),速射強弓(MC)，正義之手(黑石深淵),黑手飾物(黑下打信任務)。

當然，追求完美還需要奧山崇拜戒指，BL方還可以多備一把削骨劍，以及……你可以無限去刷雷德黑手的惡魔之擊，這玩意才是現階段真正意義上的副手畢業和斬殺階段畢業武器，你值得擁有！嗯……對了，還有橙錘和提布了解一下？

輸出循環如下：

衝鋒撕裂,切狂暴姿態,嗜血和旋風斬誰亮按誰,同時亮先按嗜血,保證嗜血技能不斷檔。

英勇只有在嗜血和旋風斬同時進入CD且怒氣大於50的時候使用。B0SS進入斬殺階段後,放棄循環,開魯莽磕怒氣藥水，切雙快武器！無腦斬殺。（風怒削骨戰用猛擊替代英勇，作為泄怒技能卡cd使用）

合劑方面：巨人藥劑或者魂能之力（冬泉谷好運任務後續）、貓鼬藥劑、冬泉火酒、元素磨刀石以及下階段開放的沙漠肉丸子，這些是狂戰追求極限DPS必備手段，其他可以在RAID前獲取的BUFF還有龍頭BUFF、雷德BUFF、完美厄運BUFF、費伍德風歌夜曲（費伍德任務後續）。

看完有沒有覺著，狂暴戰玩起來真的貴？這還不算，由於戰士是DPS里唯一沒有保命技能的職業，Raid途中，你還必須帶好大紅、有限無敵藥水等，近戰OT轉頭的瞬間秒喝，以防止猝死到來~~

總體來說，追求極限DPS的狂暴戰給人的直觀印象就是一個字——壕！壕無人性那種，當然壕戰在每個版本末期都是PVE天梯第一梯隊，只要你能放棄愛情，狂暴戰的天坑絕對歡迎你的加入~

最後，給磚叔點了贊再走！魔獸懷舊幫留言區見，拒絕白嫖~
