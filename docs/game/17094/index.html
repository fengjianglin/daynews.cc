AG戰隊0-2不敵巴西VG，遺憾止步CFS半決賽

2019-12-13 291條評論 2,491次閱讀 322人點贊

在剛剛結束的CFS第一場半決賽中，來自中國的AG戰隊以0-2的大比分輸給了巴西VG戰隊，其中異域小城和港口，兩張地圖均是8-10輸掉，遺憾止步半決賽，接下來他們將和另一場半決賽（SV和BD）的敗者爭奪季軍榮耀。

輸出主力低迷 絕跡成Carry核心

熟知AG戰隊的都知道，在國內的比賽中，AG戰隊的主力輸出是寵兒、綠豆這兩人，綠豆的狀態一如既往，兩場比賽表現都還算好，但主要是寵兒這個點沒能Carry起來，或許是更加擅長M4，來到了國際服之後並不適應AK。

相反，一向都是AG最弱的點蛇哥在這場比賽又站了出來，正如網友們所說的那句：世界賽蛇哥絕不拉胯，從整場比賽看下來，在槍法上，似乎只有蛇哥能夠頂住，其餘人幾乎一碰就碎。

對面狙擊手超神秀

其實從VG戰隊一漏面開始，觀眾們就注意到了DGZ這把狙擊，從CFSI巴西邀請賽開始，這把狙擊就絕殺了白鯊，在這次的小組賽中，這把狙擊又打得SV還不了手，在對陣AG的這場比賽中，這把狙擊多次不可思議的瞬鏡讓人嘆為觀止，我們只能嘖嘖讚歎：這狙真的不講道理。

槍法和細節壓制

槍法咱們前面也說了，除了絕跡，其餘人真的頂不住，在世界服上，AK選手確實更佔據優勢，而我們國內的比賽多數都是英雄級武器，我們的CFPL也基本成為了”M4PL”，去到了國際賽場，自然是不行的，只有長期堅持AK的選手才有一戰之力。

在細節上，國外的瞬爆閃和背閃堪稱一絕，尤其是瞬爆閃，幾乎沒有辦法躲避，而這個技巧我們國內很少有戰隊使用，其次是背閃，在國服根本沒有這樣的設定，所以我們在使用閃光彈的時候，很多時候對人家一點作用都沒有。

AG整體打得很好了 除了殘局

其實這一場比賽，整體上看下來，不看輸了的這個結果，我單方面覺得AG進步挺大了的，他們的戰術意圖很明顯，相較於之前的猶猶豫豫、磨磨唧唧來說，少很多了，戰術執行也很流暢，打法也很果斷，確定打一個點之後，就能完全執行下去。

但是就是殘局打不過，總是2V2、3V3這樣的殘局打不贏，甚至還有5V2的殘局打不贏，如果AG在殘局上不說戰勝VG，只做到一個五五開，恐怕AG都能戰勝VG，但沒有辦法，殘局同樣要槍法說話。

AG的這場比賽打完，我才真正理解白鯊戰隊在輸了之後，白鯊所說的那句話：我知道他們怎麼進攻，也知道他們怎麼防守，但就是干不過！

確實，這就是一個槍法的碾壓，殘局也不是說AG總被人打側身，而是真的槍法上打不過！

如果AG輸給了IMP或者ANB這種隊伍，我肯定會長篇大論地罵他們一頓，但輸給了這支VG，我啞口無言，不是AG失誤太多，也不是過去那種不敢打。

真的是單純的實力上打不過，技不如人！

最後，我們也只能把希望寄託於最後一支中國隊——SV，希望他們能夠順利戰勝另一支巴西隊BD，並且在總決賽上手刃VG，守住我們CN.CF的榮耀！
