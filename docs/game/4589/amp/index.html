“四聖獸”對於很多玩家來說，都不是一個陌生的設計，很多遊戲中都用到了它們。不過很多玩家其實對於”四聖獸”的了解僅限於它們是”青龍、白虎、朱雀、玄武”。其他就知之甚少了。那麼，今天我們來好好說說這四聖獸。

所謂”四聖獸”其實是古人用來描繪掌管星象的四神。與希臘神話通過星座區分不同，我們對於星象也有自己的解讀。這就是”二十八宿”的由來，上古時期，人們把黃道附近的星象劃分為二十八組，俗稱”二十八宿”。而四聖獸就是統領二十八宿的角色。對應道家理念中的”四象”。

到了現在，”四聖獸”的形象在很多遊戲中都有設計。比如很多人的童年回憶《數碼寶貝》的原作遊戲中就有”四聖獸”的設計，他們是守護數碼寶貝世界四方的究極體數碼寶貝。每一隻都具有著強大的力量。而在《軒轅劍5》也有四聖獸的設計，可以幫助玩家在戰鬥中提供更多的便利。《魔獸世界》的資料片潘達利亞之謎由於採用了東方背景，也使用了”四聖獸”的設計，不過在名字和造型上做了修改，只保留了青龍和白虎，另外的朱雀和玄武被修改為朱鶴和玄牛。

之所以有這樣的修改是因為，潘達利亞大陸其實是被神真子這隻巨大的海龜馱在身上的，如果用玄武的話就容易讓人產生誤解。而在西方神話中有巨牛馱起大地的傳說，因此做了這樣的修改。其實這種改變在遊戲界是非常常見的。比如在最近《神武3》手游的十一活動中的天降祥瑞活動中，玩家可以挑戰四聖獸來獲得獎勵，這其中朱雀用的就是鳳凰。

在活動中，每小時的10分和40分在大唐國境、烏斯藏、花果山、昆崙山地圖將會隨機刷新祥瑞聖獸，等級大於30級的玩家就可以組隊三人以上的隊伍就可以挑戰他們了。戰勝後會獲得相應的獎勵，不過一天只有三次機會，大家可以把握好。

除了”祥瑞聖獸”的活動外，這個假期，《神武3》手游還準備了”歡樂金蛋”、”歡樂煙花”、”金秋佳宴”、限時奇遇”國泰民安”等諸多活動等待著玩家們的參與，這次活動福利多多，大家不要錯過了。
