夢幻西遊手游：法連龍宮已死，傷害暴跌，一夜之間裝備虧損一半

2019-12-11 117條評論 4,266次閱讀 2627人點贊

一直以來，龍宮這一門派的玩法都很多，其中法連龍宮被不少玩家所鍾愛。但在新法寶上架後，法連龍宮似乎將遭淘汰，有玩家在網易大神發文道，自從新法寶上架後，法連裝備的價格直接降了一半，裝備直線貶值。那麼如今法連龍宮究竟有沒有出路呢？

圖片來自網易大神，作者櫻櫻櫻櫻桃啊

日前，同人娘在遊戲中遇到玩家討論法連龍宮的問題，據玩家所述，在新法寶安魂咒上架後，因為其效果可以減免受到相同攻擊的傷害，這也就意味著，法連龍宮的第二次龍捲雨擊的傷害將會受到大幅度削弱。所以不少玩家認為法連龍宮已經沒有了出路。畢竟打不出傷害的龍宮，又有什麼隊伍會要呢。

圖片來自網易大神，作者櫻櫻櫻櫻櫻桃啊

其實，若只是法連龍宮沒法玩了，那問題也不大，更新下裝備換個玩法就是了。但有玩家表示，遊戲中帶法術連擊特效的裝備已經開始跌價。原本帶法術連擊特效的裝備價格基本上是帶忽視法防裝備價格的一倍，而現在法連的價格已經與忽視法防裝備的價格持平，等於是法連龍宮裝備虧損一半。

雖然法連龍宮受新法寶影響，裝備價格大降，但不少龍宮玩家覺得這並不是什麼壞事。龍宮又不是只有法連龍宮這一個玩法，直接換上忽視法術防禦的裝備，龍宮依舊可以打出不錯的輸出，並且還能限制對面四件佩戴的法寶，也不錯。龍宮作為勇武組法系的扛把子，怎麼玩都可以，沒必要被法連龍宮限制死了。

此外，對於該玩家所言的法連龍宮已死，有玩家持不同的意見，並覺得該玩家對新法寶安魂咒的理解有誤。安魂咒的效果只有在隊伍中有相同門派時才會生效，若隊伍中只有一個龍宮，那即便觸發了法術連擊，傷害也是不會遭到削弱的。更有玩家覺得，法連龍宮不能用又咋了，龍宮只有龍捲雨擊這一個技能？直接龍騰不就行了，固定傷害，穩到不行。

回到法連裝備的價格上，安魂咒確實讓法連裝備的價格下跌，但同人娘覺得，法連裝備的價格應該不會有太大的影響。不如先看看行情再說，到時候，如果真的覺得法連龍宮玩不了了，直接換裝備、換玩法也不算晚。

門派調整與新法寶的上架，讓不少門派的玩家都憂心忡忡，擔心自己的投入付諸東流。但同人娘覺得並不需要過多的擔心，正所謂沒有垃圾的門派，只有垃圾的玩家嘛。那麼，對於法連龍宮，大家是怎麼看的呢？不如在評論區留言，與同人娘分享下自己的看法吧。
