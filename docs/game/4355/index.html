DNF手游：公測開啟，紙片人畫風太挫，玩家：阿拉德之怒都比他強

摘要

「地下城與勇士之小狐狸君愛談遊戲電競」第七百零四期《DNF手游：公測開啟，紙片人畫風太挫，玩家：阿拉德之怒都比他強》，DNF手游終於開始公測了，好多玩家都收到了官方的推送。小狐狸君也是領到了本次公測的資格。不過體驗只能說是一般，尤其是鬼劍士的畫風，濃濃的紙片人的感覺！就連阿拉德之怒的畫風，都趕不上！這建模是真的，太low了……感覺還不如玩家自己開發的那一款遊戲（dnq）。測試了這麼多年出了個這！老規矩搞快點，直接上本次DNF公測的實際效果截圖展示：

DNF手游：公測開啟，

此次手游終於開始了第n輪測試，不過之前測試過的玩家都知道，只不過是等級和職業數量提升了而已！遊戲的模型完全沒變更！之前玩家就吐槽過這個問題，也在官方反饋了。但是第一次一次的公測，僅僅是借口罷了。這一次還是紙片人外觀，可以說每次都升到55級，然後就刪檔重新公測！目前每個職業開放2個角色，基本上都是物理+魔法職業。

紙片人畫風太挫：

雖然DNF一直自稱是2.5D遊戲。主要是增加了一個Y軸的移動。但是手游的建模和陰影特效，真是做得太差了，整個就是一個二次元的紙片人，完全沒有任務的飽滿感和立體感！當然老玩家都知道DNF在60版本的時候，畫風也不是很好，但是現在是10年後了，DNF的美工都歡樂好幾任了，渴死手游還做得這麼差，果然是「用心策劃」開發商啊！

玩家：阿拉德之怒都比他強

其實阿拉德之怒的人物模型設定、遊戲界面畫風、遊戲的操作手感……都比DNF手游好很多。小狐狸君之前就公測過一次。也玩過阿拉德之怒，其中瞎子的手感非常的出色。但是DNF手游僅僅開放了紅眼和鬼泣！可見開發職業的數量，遊戲的畫質……完全是不及阿拉德之怒的。不知道手游什麼時候正式開放，但是如果是現在這個版本，涼的可能性會非常大。
