2019全明星賽LECvsLCS對抗賽剛剛落下帷幕，來自兩個歷史恩怨糾葛深厚的賽區的明星選手們帶來了一場精彩紛呈的對決，新英雄厄斐琉斯也迎來了全球比賽的首次登場，

NA：塞拉斯、酒桶、刀妹、獅子狗、艾翁

EU：炸彈人、皇子、船長、厄斐琉斯、泰坦

EU開局五人抱團入侵NA野區，但反而被NA包夾，匆忙下EU方連交四個閃現，仍然付出了兩人陣亡的代價，3分鐘皇子Gank下路擊殺艾翁，結果刀妹TP趕來加上酒桶，EU又付出了3人陣亡的代價，開局大劣。

Yassuo的刀妹在中路如魚得水，連續兩次單殺了船長，EU野輔包中將無解肥的刀妹擊殺，隨後又在下路的亂戰中，利用NA的上頭打出了一波0換5，一舉將局勢扳回。

兩邊你來我往達打成了回合制遊戲，在18分鐘就爆發了43個人頭，20分鐘左右，NA的獅子狗突襲失敗被EU反打，炸彈人和皇子的傷害灌死NA三人 ，隨後EU在21分鐘就將第一條大龍成功拿下。

連續幾波大節奏以後，EU在團戰傷害方面已經完全碾壓了NA，直接在23分鐘推上了NA的高地，擊殺四人推掉雙子塔後，EU突然選擇給了NA一次機會，拆基地拆到一半便集體撤退了。

重整旗鼓後，EU從上路再次推進NA基地，這次他們沒有再給機會，直接將NA團滅後推掉基地，恭喜LEC取得對抗賽勝利！
