德杯第二日看點：教練明凱迎來首秀，兩支前德杯冠軍能否順利出線

天天要聞
2019-12-22

北京時間2019年12月22號，英雄聯盟德瑪西亞杯第二天迎來了B組與D組的組內對決。那麼，讓我們一起來看一下，這一天的比賽中有哪些看點吧？

看點一：教練明凱首秀

從選手轉型成為教練，明凱換了一個身份繼續自己的職業生涯。這無疑也是這一天LPL粉絲們最為關注的地方。當明凱是職業選手的時候，為我們帶來了許多美好的回憶，那麼當他成為教練之後又會為我們帶來哪些驚喜呢？

看點二：LGD的新陣容

在轉會期Peanut加盟LGD可以說是相當重磅的轉會新聞，也有人因此表示LGD在新的賽季恐怕不會「樂觀」了。不過，在面對擁有著TES與EDG的B組，LGD的新陣容是否會收穫到奇效呢？不過有意思的是，Peanut的老對手廠長如今卻成為了教練，不能夠再次看到他們交鋒，對於LPL粉絲們來講還是有些遺憾的。

看點三：TES的Karsa

Karsa轉會到了TES，不僅帶去了個人能力，同時也帶去了自己比賽的經驗。確實，對於TES這支充滿天賦的隊伍來講，他們唯一不足的地方就是賽事的經驗。而Karsa去到TES很好地彌補了這一點。另外，TES新的雙人路究竟如何，也是值得期待的地方。

看點四：DMO兮夜首秀

其實與TES有點相似，作為新軍，DMO確實是擁有著不錯的天賦，但是在賽事經驗上面還擁有著比較大的不足。兮夜的到來也是很好地彌補了這一點，這種「一老帶四新」的方式，相信也是目前讓新陣容快速成長的方式之一了。

看點五：前德杯冠軍隊伍EDG與IG能否順利出線？

雖然德瑪西亞杯如今來到了第六個年頭，但是產生的冠軍卻只有三個。他們分別是EDG、RNG以及IG。而從德杯首日的表現來看，RNG已經順利出線；而在德杯第二天登場的EDG也是拿出了自己的首發陣容，看得出他們想要出線的決心；而IG戰隊則是派出了新人來迎戰德杯，不知道這樣的改變是否會對IG的出線造成一些影響呢？

最後，從專業數據給到的情況來看，雖然IG戰隊派出的並不是全部的主力，但在面對次級聯賽的隊伍SDX還是有著比較大的優勢的；另外，TES與LGD之間的對決，更多的人也是看好TES能夠獲勝。

各位小夥伴們，你們覺得哪四支隊伍能夠順利晉級到德杯的八強呢？
