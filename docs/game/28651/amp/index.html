元氣騎士：聖誕新版專享，紅武傷害飆升21億，這算史詩級增強嗎？

天天要聞
2019-12-22

隨著聖誕節的逐漸臨近，涼屋遊戲也在百忙之中為諸位元氣玩家帶來了精美的聖誕禮物。這份禮物雖然並不貴重卻同樣意義非常，在某種意義上甚至可以好比雪中送炭，而它就是大家期盼已久的《元氣騎士》2.4版本。本次更新作為聖誕新版專享，不僅元氣大廳為之煥然一新，就連地牢中的魔物都褪去了萬聖節的陰寒之氣，集體換裝向大家展現充滿歡快喜悅氣氛的聖誕氣息。可是當所有玩家著手開始探索新版本的時候，問題也接踵而至。

如果說《元氣騎士》中的版本更新是維持遊戲熱度的絕佳方法，那麼最新的2.4版本就是打破自己良好口碑的開端。因為這次聖誕版本是實打實的「翻車」了，僅僅數天之間，這款遊戲就從一個高達9.7分的精品小遊戲，成為一個在及格分數線徘徊的「渣作」。

而造成這種情況的主要原因，個人認為當屬那些在新版本中讓玩家苦不堪言的各種BUG。而糟糕的遊戲體驗更是讓各大論壇中的玩家怨聲載道，口碑嚴重「撲街」，《元氣騎士》成了名副其實的「買BUG送遊戲」。

當涼屋大大們了解到玩家的這些訴求後，並沒有閑著，馬上開啟《元氣騎士》緊急維護機制，對現有的問題進行修復，畢竟涼屋不能真的「涼」。而小夥伴們也從涼屋態度誠懇的道歉信中了解到，造成這種情況的主要原因是配合有關規定對遊戲進行合理化調整，而導致工期不夠強行上線的後果。

找到問題就解決問題，《元氣騎士》緊急機制的啟動也極具效率，僅用一天時間就進行了2.4.1版本的推送，在上版本中出現的BUG大部分都得到了修復。可是大家在高興之餘也不要忘記涼屋的另一種隱性規則，那就是：解決舊BUG出現新BUG，我們將其稱之為，涼屋特性！

21.4億屬性的神器

在最新的2.4.1版本中，不僅是精靈的天賦技能得到解決，死星鐳射的激光彈射能力得到恢復，機錘的玩法得到保留，就連被老宅戲稱為「可以坑隊友」的近戰反彈天賦也不再對玩家造成損傷，據不完全統計共有大大小小十數種BUG得到修復，為涼屋的工作在效率點個贊。

可是當老宅準備進入地牢享受完美版本帶來的愜意體驗時，卻無意間發現了一個讓人驚訝的問題，那就是某些紅武的傷害居然飆升至21億，老宅不禁想問一下涼屋：這算史詩級增強嗎？

據老宅所知，在整個《元氣騎士》中得到這種「21億傷害強化」待遇的紅武並非只有一種，而是兩種，它們分別是：死星鐳射與妙妙槍。

死星鐳射在老宅之前的文章中曾經詳細點評過，在2.4版本中它的傷害足足衰減了近1倍之多，直接從最強武器的寶座跌到2流武器的水準。而在2.4.1版本中涼屋恢復了鐳射武器的折射能力，所以從理論講死星鐳射應該是恢復實力了才對，可是當再次看到它的攻擊屬性後老宅震驚了。高達2147483647～0的攻擊屬性簡直要「逆天」，那麼問題來了：這件武器算史詩級增強嗎？

老宅對這件「逆天級」武器進行了實戰評測，結果在輸出時並沒有出現高達2.1億的傷害反饋，依然和原版持平。而且從地牢中切換撿取武器時出現的場景來看，面板仍然是原版的4維屬性，看來這飆升至21億傷害的死星鐳射，只不過是一場大大的烏龍而已。

可是既然死星鐳射並沒有獲得史詩級增強，那麼作為涼屋招牌的的妙妙槍是否定位與前者不同呢？

在《元氣騎士》中，妙妙槍是一把可隨機釋放攻擊模式的紅武，所以並沒有固定的傷害指數反饋。而在新版本中它的攻擊屬性雖然與前者相同，但是實際傷害仍然沒有達成21億的超額數值。看來這兩種武器的「逆天級」屬性，又是涼屋大大手下的一種全新特性，毫無可信度。

另外關於死星鐳射的傷害反饋數值也同樣出現疑問，元素武器不能暴擊可是傷害顯示又並非是屬性額度，計算方式真是讓人摸不到頭腦，老宅估計這可能又是程序員大大手滑的傑作吧。
