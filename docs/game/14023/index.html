王者榮耀：e星輸給AG原因找到，貓神坦言被張角看穿，邊路失守

2019-12-11 923條評論 3,248次閱讀 970人點贊

各位王者玩家大家好，我是小雨。時隔多日之後，KPL官方終於放出了AG超玩會在敗者組半決賽對戰eStarPro比賽的語音交流。在選手們的交流中，大家不僅可以得知他們在比賽中的戰術安排，還可以了解他們的賽前心態及互動等。良好的心態是取勝的第一步，兩支隊伍成員也一直都在互相鼓勵。大家跟小雨一起來看一下吧。

e星跟AG的第三場比賽開打之前，貓神問老帥在幹嘛，怎麼一直不進場，Alan說他在調楊玉環銘文。結果貓神打趣道：「那你喊他一下，你是牌兒，我是漏兒。」Alan忙說算了，和平一點。AG這邊愛思對六點六說：「六哥，諾言靠你了。」六點六淡定回答：「諾言還是蠻好抓的。」一諾則說大不了輸了放假，贏了後天接著打。一諾弟弟還是挺看得開的。

雙方經典的第三局，麟羽下路收割四殺之後全是來自隊友的誇讚，愛思說：「你就是天秀之人。」花海極限守家之前，貓神等人已經放下了手機，等著比賽結束，結果花海四殺成功守住，隊友們也是一波贊。不過AG老帥淡定地說：「翻不了，不著急。」果然後面在老帥殺死花海的李元芳之後，AG勝利拿下比賽。

巔峰對決賽前，貓神坦言：「他們這兩盤學著搞邊路了，我們也要搞邊路，張角看穿了局勢。」不過貓神這時候也體現了領袖氣質，Alan說六點六自信曜，貓神說沒關係，我們自信老夫子。另外他還對無銘說愛思的盾山很菜，這一局就是你的表演秀。這個時候得到隊長的鼓勵，無疑會增加選手們的信心。

​AG的選手們雖然心態也都比較放鬆，但他們一直以為會被4比0帶走，根本沒想到會打到巔峰對決。六點六也直言這是贏了血賺，輸了不虧；一諾也覺得輸了也沒事，反正都巔峰對決了，輸得也不難看。老帥對一諾說：「一諾這把公孫離肯定是天秀之局。」結果還真的被老帥一語中的，一諾的公孫離大招開團秀得起飛。也正是這次射手開團，奠定了AG的勝局。

縱觀這七局比賽，e星的邊路確實被AG抓了不少次，張角眼光還是蠻準的。
