比火圈還貴！魔獸懷舊服法師彪3300買到這衣服，你覺得值嗎？

摘要

60懷舊排隊、太累？想玩70版TBC懷舊服、80版巫妖王之怒？私信發「懷舊服」，即為你推薦！

懷舊服雖開服已過三月，但遊戲熱度並未隨著時間推移而有太大起伏，這次小編要講的是一位玩家在G團中花了3300G買了一件T2靈風腿，竟比火圈還貴，這位法師事後有些後悔，問道這裝備到底值不值這個價？一起給這位法師分析分析吧：

這個cd是我第五個cd，boss出了T2腿，我一上頭和一個fs彪了，3300拿的，就像問問大家這玩意比我的火圈還貴，到底值不值這個價！買過身上就剩三千G了，這個cd dps第二才補貼50G 人均345G！花得我肉疼，你們看賺了還是虧了？我們區G價0.35。:我不敢不拿，畢竟概率低，p了可能下次便宜也可能到黑龍都不一定能出了，屆時靈風套還貴漲價。

小編知道這位玩家現在想看到的一定是不虧二字，所以：不虧，可能下個靈峰腿到mc停活動都不會出了，我60就見過7件缺腿mc在活動的。競拍時確實容易上頭，還望各位玩家把持好自己，切勿意氣用事。來看看其他玩家都是如何評價的吧：

不虧，休閑玩家畢業褲子

如果你以後休閑的玩，這個腿基本畢業了，畢竟T2.5褲子要打安其拉了，廢墟褲子沒套裝屬性，所以我覺得不虧，靈風8件屬性真的逆天，關鍵是帥啊。不知黑翼難度啊，反正T2腿是必拿的，法師除了腰帶項鏈戒指，還是穿套裝好，兼顧PVP和PVE啊，另外老奈的暗影烈焰法杖一定要拿到，到NAXX的硫磺法杖才能換下。

正常一千多你們團法師的有多窮？我們工會最高的一團7000G拿的，我是四團，三千G拿的靈峰褲，和火圈一回事，只要有兩個非拿不可，或者看見你想要死命頂你的，就便宜不了了。

進本之前想要什麼一定提前想好，包括想要這件裝備的心裡底線價格最高能接受多少，也有個數兒，做到這些以後，我就特別喜歡有人故意抬價，那樣最後肯定是對方買單，自己還能多分點錢，何樂而不為呢。又不是馬上afk了，缺幾樣裝備還能有個繼續玩的動力，裝備一旦拿齊了的人就真的afk了.

個人建議：在有能力且在接受範圍內的拿下就完了，早拿早爽，晚拿不一定便宜。見過100的葉子，也見過3000的葉子，3000葉子那個就是想等後面低價拿，結果等著等著坐不住了，大佬的小號起來了，起手就是1000的那種，最後搶的頭破血流。

這麼說，開了bwl這腿還是畢業腿，火圈還是畢業項鏈，但碧空統御這些就要換成暗影烈焰了。保值程度上還行。

價格太高，幾百被拍走的大有人在

這玩意1000多差不多了。。火圈剛出了個。別人叫了4000，我想繼續叫的，他M我讓他。。想想就沒叫了。不存在的。套裝遲早能拿到。而且跟職業掛鉤。極品裝備大家都盯著。沒那麼容易拿的。

公會gkp團我都從來不跟任何人交流我想要什麼東西，因為即便不是dkp，也是有小團體存在的，人家微信群交流，故意頂你，你提前透出來迫切想要什麼，不往死里頂才怪。誰都想多掙錢.

你這就離譜了啊 現階段湊不齊套裝。如果每個cd都打，這玩意遲早爛大街。我們工會一般1500左右⑧。拿了就別想太多，錢都花了爽就完了。

​

火圈1000碧空能頂到3300。。。你們團里這群法系DPS是多不會玩啊，碧空開了bwl就換，火圈是畢業，根本不是一個等級的東西。

只要是在自己能力接受範圍之類都不算虧，這位法師能問出「虧還是賺」這樣的問題其實在他心中已經有了答案，另外切記別再將自己想要的裝備告訴別人了，G團小團體比比皆是，多得是人想故意抬你價格多分點錢。你有什麼看法呢？

歡迎發表自己的見解，優質評論將獲得置頂露臉機會哦！
