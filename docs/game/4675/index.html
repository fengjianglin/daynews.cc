文明系列的遊戲至今已經出了五部了，文明系列目前全球發售已經超過了4700萬份，而今天文明六也在任天堂Switch上正式發售了，支持中文。

但近期發現文明六齣現了一個逼和平主義者打人的 bug，在獲勝的時刻遊戲會直接崩潰，除非你在這一回合宣戰。

在 Reddit 上，一個人在抱怨崩潰的事，隨後就變成了一場大型同類病患交互療愈現場，他們都有著相同的境遇：在遊戲中，不管你已何種方式達到勝利條件，在最後一回合你馬上要走向輝煌的榮耀之時，遊戲就會崩潰你一臉。

一些人分享了 2K 客服給他們的標準答案，而其中一個叫做 Throwaway489er 的人或許道破了天機：「在結束回合之前對那個致使你遊戲崩潰的文明宣戰，實測在很多人那都能解決崩潰問題。想知道是哪個文明讓你遊戲崩潰的，你需要仔細觀察崩潰前到底輪到誰了，一旦發現，讀盤，對丫宣戰，然後再點 『下一回合』。」

這個 bug 教會我們，沒有戰爭的文明史是不存在的。
