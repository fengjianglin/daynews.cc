大家好，號主來啦

今天也快速地給各位帶來了國服12.16利物浦&曼城的俱樂部精選數據

本數據選取於10.31的國際服曼城精選數據，10-10的利物浦套數據其中比較具備性價比的球員分別是：

曼城羅德里，利物浦羅伯遜，利物浦薩拉赫

下面我們來看看這一些數據如何吧！

利物浦精選：

范迪克：畢業級

此卡對比普卡，總體強化不多，但抽到了就穩穩畢業

阿里松，畢業級

對比全99的小獅王，還是沒辦法全部99，不過在國際服實測效果還不錯

薩拉赫，畢業級，T0右邊鋒

對比普卡，強化不少，是2019賽季地位的薩拉赫，喜歡利物浦的球迷不要錯過

馬內，畢業級

對比普卡，強化不少，薩拉赫左邊路版本！

菲爾米諾，准畢業級

總體數據還不錯，算是個挺好的全能中鋒，但一般玩家玩不來

阿諾德，畢業級

本卡在國際服不少人是作為畢業右後衛的選擇，特別是你需要傳中需求的話，本卡是一個非常非常好的推薦，如果你是要讓右後衛專心防守，不準備傳中戰術等的話，那麼後續還有其他更加適合你的右後衛

亨德森：強力級

肯定沒辦法達到維埃拉的高度，但是作為一個後腰和中場，防守數據和傳球數據都很好，速度82也算中等，體力優異，抽到了可以上，但是盡量不要讓他帶球和前叉，速度和靈活不太足夠

張伯倫，准畢業級

一個比較不錯的邊衛，其實踢中場模型一般，但踢邊衛反而更加合適一些

曼城：

阿奎羅，准畢業級

對比普卡，提升不多

德布勞內，畢業級

雙傳這次再加強，效果不錯，你喜歡傳控型的前腰，值得推薦

埃德森，畢業級

全數據99，跟小獅王一樣，空中傳球也不錯，抽到了穩用一年沒問題

斯特林，畢業級

左邊鋒速度狂魔的代表，非常靈活，射門也不錯，狀態也可以，如果你喜歡右邊路薩拉赫的類型，那麼斯特林也非常適合你

大衛席爾瓦：強力級

席爾瓦雙傳極為優異，盤帶也很好，但在遊戲裡面模型和身體素質較為吃虧，比較適合高檔技術型玩家，不推薦普通玩家和非曼城隊套需求玩家入手

奧塔門迪：強力級

數據不夠畢業，模型不夠畢業，喜歡曼城的球迷可以考慮入手，不推薦普通玩家入手

羅德里：畢業級

沒有維埃拉的玩家，非常需要這個羅德里！這個是沒有射門版本的維埃拉，而且傳球非常棒，抽到了，可以打上頭槌等技能，可以讓你後腰有強力引援

津琴科：強力級

傳中優異，防守可以，但速度和身體條件較為一般，總體來說不一定能比普卡阿拉巴強，非曼城球迷不推薦入手

全部球員就在這裡，大夥可以結合自己的需求進行選擇，號主祝你歐氣滿滿！
