最偉大的科幻遊戲TOP20：你是一個合格科幻遊戲粉嗎？

2019-12-20 2,972條評論 4,792次閱讀 2056人點贊

「科幻，用幻想藝術的形式，表現科學技術遠景或者社會發展對人類影響。」這是「科幻」一詞最被大眾接受的說法。「只要故事中含有超現實因素，便可算作科幻作品。」但是要做到真正的科幻，又談何容易。今天小編為大家帶來的是20大最硬核的科幻遊戲，評選的標準是科幻元素>遊戲性，所以最好玩的遊戲不一定在最前面，但是最科幻，最未來的遊戲會名列前茅哦！N7號飛船，弗里曼的撬棍，恐怖的「石村號」……這些都是屬於科幻玩家最經典的記憶！你是一個合格的科幻遊戲粉嗎？

《沙丘2》

不僅是爺爺級別的科幻遊戲，也是爺爺級別的即時戰略遊戲。遊戲根據赫伯特同名小說改編，當時盛極一時的Virgin與其全資子公司westwood合作。把整個遊戲系統改為即時方式(儘管依然需要輸入文字命令)開創了即時戰略的時代，現在無論是媒體還是玩家對沙丘2的評價，都是即時戰略的鼻祖，任何美譽用在它的身上都不過分。

這對部隊的移動能力造成影響，而且在這個星球上找一片適於建造建築的石頭地表也很困難。沙層之下活動的沙蟲感受到沙子表層的震動後，就會浮出，將在沙漠表面作業的採集車和採礦部隊吞沒。遊戲中要派出採集車去採集香料，建造各種建築物，開發更強大的單元以消滅敵人。

《輻射3》

也許是所有玩家最喜歡的一部科幻類遊戲了。

2077年地球引發的核戰爭導致了地面的荒廢，人類在被稱為 「Vault」 的地下避難所生活、生育、死去。其中一間避難所編號為101，就在如今「廢土」的首都華盛頓DC附近。遊戲背景是2277年（核戰200年後）的未來世界。避難所101被譽為廢墟中的珠寶。在200年里，101號避難所一直忠誠地侍候著華盛頓與其郊外的倖存居民。雖然2077年的全球原子戰爭留給美國的只有廢墟，但是101號避難所上的居民很享受逃脫外部世界壓力的自由生活。巨大的變異昆蟲，盜賊，奴隸販子，還有甚至超級異形全部都無法對抗超級避難所技術工程。然而一個命運的早上，當你醒來時，你會發現你的父親公然頂撞監督（Overseer），沒有什麼理由離開了舒適安全的101號避難所。離開你熟悉的避難所，從中走進了太陽底下荒蕪的廢墟，來找尋你的父親，還有真相。

《太空入侵者》

「第一款真正意義上的遊戲」，「史上最偉大遊戲」……《太空入侵者》擁有太多的美名了。遊戲的劇情很簡單，一群邪惡的異星人，有組織有秩序地排成了11*5的大方陣,向地球襲來。玩家的目標就是在它們衝到屏幕低端前消滅它們。隨著你打掉的異形越多，它們的速度也會越練越快。如果你被擊中，就會丟掉一命.更要命的是，只要有一架異形飛船飛到了屏幕的低端，就意味著你的星球遭到入侵而立即game over。

現在看來簡單到令人髮指的遊戲，在1978年造成了全民狂歡。

《殺出重圍》

《Deus Ex》的方法是說它是一款完全開放遊戲，可以讓玩家自由選擇遊戲進程直到結尾的一種經歷。遊戲是一個講述未來的黑暗的故事，遊戲以我們現實世界中早已存在的陰謀論為基礎引申出一系列的複雜劇情，令人嘆為觀止。

而且遊戲中涉及了大量關於哲學，政治，以及神話等多方面的知識，這些可能會讓一些低水平的玩家摸不到頭腦，再加上較複雜的操作方式，很容易讓人望而卻步，可遊戲本身所創造理念在當時是非常先進的，而且此作品還被科學界評為最有科學引導價值的遊戲之一。

《銀河飛將》

一個陰險惡毒的外星族人發現地球的星座位置，計劃入侵毀滅地球，地球上的星河戰隊應徵而戰，在地球和外星人之間的太空展開了激烈星際戰鬥，捍衛全人類。克爾瑞奇族-貓人派遣一支艦隊奇襲，轟炸帕格薩斯小行星。
