荒野大鏢客2新職業換湯不換藥？季票獎勵真香

2019-12-14 909條評論 3,711次閱讀 1280人點贊

★遊戲馬蹄鐵原創

荒野大鏢客2的新職業私酒販在昨天上線，其實R星家的遊戲現在也有了自己的一套模板，所謂新職業無非就是給你一個生意，然後【搞原料-等生產-最後運送】，這個公式是萬能的，未來會套用於所有R星遊戲。私酒販也是如此，稍微不一樣的就是他有一個獨立的故事內容，另外做任務就等同於搞原料，價格低下來之後直接生產，最後等待完成去運送，幾乎和商販一樣。

新的生意夥伴瑪姬

私酒生意

如果不做任務的話，成本會非常高，你幾乎沒有利潤。但是通過做任務也就是各種破壞對手生意和清除禁酒偵探，可以達到最低價格買入原料，這樣就有利潤可圖了。和其他職業一樣，想賺大錢還是需要職業達到最高等級。

黑屏Bug

Bug不是育碧家專屬，R星的遊戲也有Bug。例如你好不容易搞到貨了準備銷售了，結果一直卡轉左輪圈圈黑屏…幾分鐘甚至幾十分鐘，無奈強制關閉重新進入後，發現貨沒了…錢？貨都沒了哪來的錢！相信不止一個人遇到過這樣的問題。

職業新馬匹

私酒販職業獎勵中，這匹馬在速度屬性上是神駒。因為它的起始屬性是速度7加速度6，這是目前市面上最高屬性的馬。作為參考阿拉伯黑白是6-6，而純血馬和其他多用途馬最高是7-5，這個7-6是目前最高速度屬性馬。另外體力優秀，血量稍微少點，體型也比純血馬要小一點。如果不是外觀黨，這匹馬在速度屬性上是目前遊戲最好的，帶上裝備可以達到9-8。

新的俱樂部獎勵

不知道是不是因為上一季有人吐槽俱樂部獎勵內容太少太短的原因，這次季票給的內容有100級，售價40根金條。最終獎勵的服裝是故事模式亞瑟的哈拉維套裝。這100級里包含很多食物彈藥等各種補給品，新的馬面具，各種折扣券，返還的金條，武器外觀等等。新職業是套用公式沒什麼亮點，但這個季票內容是真香。而且現在有挖寶的經驗在，還在等什麼？拿上你的鏟子和金屬探測器，幹活吧！
