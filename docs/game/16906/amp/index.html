昔日霸主黯然退場“抗韓奇俠”閃電狼LOL分部宣布解散

天天要聞
3 天 ago

今年LOL的冬季轉會期對於FM閃電狼來說可謂是異常艱難，轉會期開始後閃電狼戰隊的多名選手陸續離隊轉會；從上單到下單，打野到教練，一位又一位成員從閃電狼戰隊中離去。距離LOL的冬季轉會期的結束時間僅有一周不到了，本以為在轉會期的最後時刻閃電狼將會招納新的選手，但不曾想到閃電狼戰隊在這個時候官宣了LOL分部解散！

閃電狼戰隊在LMS賽區是最有話語權的一支隊伍，閃電狼戰隊在S5賽季官宣成立英雄聯盟分部，成立後的閃電狼就在賽場上為觀眾們呈現了一場又一場的視覺盛宴，甚至還拿到了LMS七連霸的優異戰績。但是隨著時間的推移，閃電狼戰隊的實力開始有所下降；在今年LMS賽區的夏季賽中，昔日七連霸的隊伍僅僅拿到了墊底的排名，因此也無緣拿到S9世界賽的參賽門票；這對於閃電狼戰隊以及隊員來說無非是一個巨大的打擊。

自打S9世界賽結束之後，閃電狼的內部成員就開始出現心態不穩的情況，很多成員陸續轉會離開了隊伍，戰隊中甚至還出現了無人交戰的狀況。這種情況的發生想必是很多粉絲們所不願看到的，可這仍舊不是最壞的結果，閃電狼在今日官宣了將不再參與明年的PCS事件。

PCS賽區是由LMS賽區和LST賽區組合而成的新賽區。在沒有PCS賽區之前，LMS大師賽是LMS賽區戰隊想要拿到世界賽門票的唯一渠道，每年只有三支戰隊可以拿到該門票。LMS賽區與LST賽區合併為PCS賽區後也就意味著前LMS賽區的戰隊想要參加世界賽只能在PCS賽區中拿到優異成績。

如今閃電狼戰隊官宣不再參與明年的PCS事件，這也就意味著閃電狼戰隊正式宣布解散！現在來看，閃電狼剩下的LOL選手將會全部轉會；其餘分部依舊正常運行。畢竟天下沒有不散的宴席，很感謝閃電狼戰隊能夠在成立後的2000個日子裡為大家帶來的精彩表演；以後還會有很多種可能，祝閃電狼戰隊好運！
