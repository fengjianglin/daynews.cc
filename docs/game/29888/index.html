新模式覺醒之戰來襲，宮本武藏重回巔峰，網友：我選白起

2019-12-23 558條評論 1,677次閱讀 577人點贊

王者榮耀S17賽季即將結束，新的一年即將到來，天美又開始放大招了，除了會推出大量系列皮膚以外，又有新模式襲來，它就是我們今天要說的覺醒之戰。之前大家開玩笑說的讓鍾馗的鉤子長度翻倍，鎧皇的大招傷害翻倍如今在新模式覺醒之戰中都可以一一實現了。

所謂覺醒之戰，其實與常規模式的地域以及勝利方式是相同的，不同點則是每個英雄都進行了大幅度增強，讓玩家可以在遊戲中感受酣暢淋漓戰鬥，當然，相比較普通模式，覺醒模式由於技能釋放更華麗，以至於在團戰中對手機還是有一定要求的，如果手機處理器較差，很可能在團戰中出現460或死機現象。

值得一提的是，喜歡宮本的網友可以開懷大笑了，無敵的那個男人又回來了，宮本一技能空明斬擁有擊飛敵人以及命中敵人後獲得可以疊加的護盾，宮本武藏的二技能「神速」由之前的一段位移改成了三段，並且可以疊加護盾，這麼一來宮本及時不開大招，也非常強勢，面對技能都增強的英雄，宮本的空明斬作用被無限放大，加上三段位移，團戰中禦敵保命兩不誤。

當然相比較宮本武藏，還有一位英雄的人氣更高，在體驗服基本上是必選英雄之一，他就是白起了，白起在覺醒之戰中一二三技能都得到了大幅度提升，最帥的要屬白起的大招了，鐮刀翻了3倍大小，在他方大招周圍額外生成整整一圈鐮刀，所有鐮刀會在一定時間後自動收回。

不知道各位網友是否覺得有些過於花哨，反正本人是白起這華麗的大招折服了，正式服上線後，必須首選白起！
