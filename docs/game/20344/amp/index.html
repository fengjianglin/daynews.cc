S18前瞻：六位英雄調整，對抗路有經驗加成，野區迎來巨大改變

天天要聞
2019-12-16

本文原創首發於公眾號：電競怪客

hello，大家好，我是怪客君~

12.13深夜王者榮耀體驗服進行了一波大更新，這次的更新內容不僅上線了新英雄蒙犽，還對六位英雄、野區和對抗路的環境進行了優化，另外還包括S18賽季的系統升級優化。內容有點多，且聽怪客君一一道來。

一 新英雄蒙犽上線

稷下星之隊的最後一名成員蒙犽上線了，上面這張圖就是蒙犽原皮膚的海報，他的技能和之前爆料的內容一致。

二 英雄調整

▶曜-削弱

怪客解析：曜原本所受傷害的50％變成40％轉化為星削結算。我把曜的星削比例做成表格形式方便大家理解。

星削對於曜來說相當於是免傷，所以這次的改動簡單說就是曜受到的傷害會變多，對曜來說是一波不小的削弱。

▶安琪拉-削弱

怪客解析：三技能旋轉速度又回檔了，不過之前加過傷害和護盾了，所以相對於上次調整是削弱，體驗服安琪拉對比正式服還是加強的。

▶成吉思汗-增強

怪客解析：上上次狼狗的技能機制進行調整，將3環額外傷害變成4環。接著上一次體服更新全面增強了所有傷害，這次又提高印記傷害，同時提升移速作為補償，所以相對於上次調整是增強，但是對比正式服來說屬於上限削弱！

▶盤古-削弱

怪客解析：三技能的基礎傷害、等級成長傷害以及Ad加成傷害都被削弱了，盤古在當前版本還是挺吃香的，在上線正式服前抓緊時間玩。

▶嬴政-增強

怪客解析：大招傷害得到略微加強，注意這次調整依舊是針對的體驗服重做後的新嬴政。所以具體強度等上線了再討論。

▶魯班大師-增強

怪客解析：基礎生命值加強，同時三技能射程直接再加1000！不過防斷時間變成了1秒。總的來說魯班大師進行了一次全面加強，這也是因為他在中低段排位賽中的表現實在太差，很多玩法還有待玩家開發。

三 對線分路調整

1. 對抗路回復兵改為經驗兵

怪客解析：對抗路的回復兵變成了經驗兵，無疑會讓對抗路在線上的優勢提升不少，同時也對一些比較吃4級的英雄來說是一波環境加強的調整，比如花木蘭曹操這些英雄都將成為新賽季邊路的受益者。同時也讓打野的第一波抓人有了更多的選擇。

2. 中路河道之靈生命調整

怪客解析：中路河道之靈的經驗和經濟都將高出邊路，這也是一種分路平衡，但同時這個河道之靈對於打野的吸引力也是很大的，就看誰有本事先搶到咯。

四 野區野怪調整

1、先知主宰/主宰調整

怪客解析：先知主宰將和暴君一樣為兩分鐘刷新的生物，另外，先知主宰還下調了血量和攻擊力。這樣一來打野的選擇更多了，同時也會加快整個遊戲的節奏。

2. 先知先鋒/主宰先鋒前期削弱

怪客解析：10分鐘前的先鋒的血量攻擊經驗有所降低。不過金幣倒是漲了不少。

3. 暴君

4. 黑暗暴君

怪客解析：暴君的擊殺會讓己方獲得少量經驗和大量金幣，黑暗暴君同理。這樣的調整讓打野有了兩個選擇，一個是選擇全隊金幣，一個是全隊經驗。

5、中小型野怪刷新間隔增加

怪客解析：小型野怪刷新時間整體加長了十秒鐘，雙buff的持續時間延長，野怪的刷新間隔增加讓打野有更多的時間去搞事情，加快遊戲節奏。

6、野怪成長收益增加

怪客解析：野怪的金幣獎勵和血量攻擊力隨時間變動增加許多，但是後面野怪也會變得很強，這是對野怪刷新間隔加長的一種平衡。

7、紅BUFF加強

怪客解析：紅buff的灼燒傷害大幅度提升，並且可以對野怪造成非常高的傷害，這讓打野刷怪的效率更高。

8、風暴龍王BUFF史詩級大加強

怪客解析：風暴龍王buff提升了傷害，清線速度和護盾的持續時間，如果是拉鋸戰絕對比現在強！同時擁有這個buff非英雄單位傷害大幅提升。

整體看來，野區的收益得到了提高，同時對打野的判斷能力也提出了更高的要求，作為前中期的節奏帶動著，打野在新賽季將會面臨更多的選擇，是選擇金幣還是經驗？是選擇抓人還是控龍？哪些資源該讓，哪些資源該搶？這些都需要打野根據具體形勢立即做出最優的判斷。
