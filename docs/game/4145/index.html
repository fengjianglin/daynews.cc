魔獸世界奧山戰場，竟在風暴英雄中得到神還原

六十年代的魔獸世界是玩家心目中不可磨滅的經典版本，諸多設計至今仍讓人念念不忘。也因此近期上線的魔獸世界懷舊服才能大獲成功。近期魔獸世界懷舊服終於打算開放玩家們翹首以盼的奧克蘭特山谷戰場了。

筆者仍記得當年在奧克蘭特山谷戰場奮鬥了一天一夜，終於拿下聯盟將領范達爾·雷矛首級的激動之情。而今懷舊服打算重啟奧克蘭特山谷戰場時，公會裡的玩家們也自然是開啟了積極備戰模式。畢竟除了可以拿到裝備獎勵外，這個模式本身的樂趣也絲毫不亞於一場完整的MOBA。

有意思的是，集眾家之長，將暴雪旗下各路經典遊戲的元素融於一身的風暴英雄，竟然也出現了奧克蘭特山谷戰場。雖說風暴英雄的地圖眾多，魔獸世界要素早已不算稀奇。就例如風暴英雄中出現了黑心灣這張圖，完美繼承了魔獸世界8.1版本更新海島玩法時，玩家可以通過達布隆幣兌換獎勵的元素。

不過在風暴英雄中達布隆幣並不是用來兌換聲望，而是用來僱傭地圖中央的黑心船長。雙方玩家必須在地圖中爭奪散落的達布隆幣，或是直接搶奪其他玩家已經攜帶在身上的。當湊齊一定數量並成功交付到黑心船長手中後，他便會一邊發出豪爽的笑聲，一邊將敵方玩家的基地轟炸至渣。

而奧克蘭特山谷戰場這張圖，則將魔獸世界的元素更加濃密的融入到了風暴英雄的MOBA玩法中。由於兩者本身契合度就相當高，風暴英雄的奧克蘭特山谷戰場，幾乎可以稱得上就是濃縮的原版。

相信玩家都還記得，魔獸世界的奧克蘭特山谷戰場勝負條件是擊殺敵方基地中的首領，聯盟方首領是銅須親族，矮人范達爾·雷矛，而聯盟方首領則是霜狼氏族的領袖德雷克塔爾。於是乎風暴英雄的奧克蘭特山谷戰場十分大膽的將主機概念給取消了，雙方營地的核心不再是一座建築，而是活生生的兩位傳奇將領。

並且奧克蘭特山谷戰場中拔掉敵方一座據點，首領便會喪失一名貼身侍衛的設定也得到延續。風暴英雄的雙方將領初始都擁有三面護盾，每次拔掉敵方一座據點，便會使得敵方首領喪失一點護盾。這也與當年的”剝洋蔥”戰術如出一轍。

為了迎接風暴英雄第88位英雄死亡之翼的登場，暴雪推出了專題活動，近期只要上線打遊戲，就會狂送戰利品箱，打開便有可能獲得各種史詩級皮膚和英雄，而死亡之翼更是只要購買暴雪嘉年華虛擬門票便會免費贈送。想要入坑的玩家絕對不能錯過這個機會。
