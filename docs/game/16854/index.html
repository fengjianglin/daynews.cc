EA Motive總經理：EA現在很不同 非常專註於品質

2019-12-16 360條評論 2,232次閱讀 731人點贊

由前《刺客信條》女製作人Jade Raymond於2015年成立的加拿大工作室EA Motive，在去年10月失去了自己的領導人——Jade加盟了谷歌，成為谷歌Stadia第一方開發團隊負責人。本周早些時候，外媒GamesIndustry發布了一個對EA Motive工作室新任總經理Patrick Klaus的採訪。Klaus也是一名前育碧高管，負責開發過《刺客信條4：黑旗》、《刺客信條：大革命》和《刺客信條：奧德賽》。

在採訪中，Klaus談到了EA。他表示EA現在感覺是一個不同的公司，非常專註於品質。

「感覺像是一個不同的公司，一個非常專註品質的公司。這也是我這次回歸非常興奮的原因之一。我們的首要任務是開發優秀的作品，而EA高管真的正在授予EA蒙特利爾這種權力，通過招攬大批優秀的人才，使其在為EA開發一些出色的遊戲中發揮非常重要的作用。

這是一個不同的時代，不同的領導，不同的信息。這在我們每天的討論中被強化。感覺這是我們的絕對首要任務。」

EA Motive工作室成立的初衷是為EA打造媲美《刺客信條》的動作/冒險巨制，讓EA成為動作遊戲類型的有力競爭者之一，這類遊戲正是EA多年來所欠缺的一個重要細分市場。

Kolaus表示他們正在開發一個新的動作類遊戲IP，已經開發了幾年了。這個新IP並不是星戰遊戲，儘管他們也在致力於打造一個「非常獨特的星戰體驗」。值得一提的是，這裡的「獨特星戰遊戲」並不是EA溫哥華前一陣子從Visceral工作室接手的同一個遊戲，因為根據Kotaku爆料，這款遊戲已經被取消，被EA溫哥華重啟，變成了一個規模更小的開放世界遊戲，同時保留了Visceral工作室開發的一些藝術素材。

EA Motive目前最好的聲譽是為《星球大戰：前線2》開發了單人戰役，期待他們能有更好的作品。
