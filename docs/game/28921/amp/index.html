黃金大獎賽S8F組：VG三連雞奪得榜首，4AM第三，玥玥拿到4淘汰

天天要聞
2019-12-20

絕地求生鬥魚黃金大獎賽S8小組賽最後一場的賽事終於結束了，在今天F組的比賽里出現了很多的名場面，相信很多看直播的小夥伴看得很開心，除了名場面之外本次F組比賽驚喜一名小姐姐，CSG戰隊上了一名女性選手玥玥，這讓彈幕更加的熱鬧。

其他的也不多說，現在讓我們來看看今天比賽的結果吧：

VG戰隊牛逼，雖然前期被小姐姐玥玥一串三，不過後面知恥而後勇連續吃雞，完成三連雞的壯舉，希望後面的決賽VG戰隊也有這樣的成績。

4AM戰隊最終拿到55分，記得賽前胖子說過4AM只要在拿20多分就能成功晉級決賽，55分的話更是沒有問題，而且這積分在小組賽總分榜也是前列。

SMG戰隊成績雖然不錯，不過胖子還是感覺SMG戰隊今天好慘，各種被雷炸到，先是第三局因為小鹿的雷痛失一雞，第六局更是被LYG戰隊一雷三響，這種場面實在鬱悶。

賽場上唯一的女性選手CSG戰隊的玥玥在今天拿到了4個淘汰分，未完成今天的目標，主要還是因為後三場CSG戰隊實在是太拉閘了。（中場休息時候宅宅採訪了玥玥，玥玥表示今天希望能場均一個淘汰。）

黃金大獎賽S8小組賽已經結束，明天是全明星賽，不知道哪些選手會上場呢？
