KPL年度最佳評選開始，AG超玩會入圍6項，但很可能顆粒無收

2019-12-18 2,084條評論 3,939次閱讀 1265人點贊

AG超玩會奪得今年秋季賽的總冠軍，這意味著今年KPL的比賽已經落下帷幕了，至於冬冠杯，賽程要安排到明年，冠軍2020年才會揭曉。KPL年度最佳評選現在也正式開啟，獎項由專業評審團投票產生，專業評審團由KPL固定席位俱樂部+KPL合作媒體+KPL賽事委員會共同組成，評選結果將在冠軍杯團聚夜公布。

此次的年度評選，一共有9項大獎，分別為2019年度最佳選手、年度最佳打野、最佳輔助、最佳中路、最佳發育路、最佳對抗路、最佳教練、最佳解說/主持，以及年度最佳新銳選手。這裡面，AG超玩會入圍了6項，而eStarPro則入圍了8項。

AG超玩會王者歸來，他們曾跌入最低谷，但僅僅只用了一個賽季，又登上了最高峰，這賽季的AG超玩會是最讓人驚喜的一支隊伍。而老帥獲得了FMVP，一諾更是被所有玩家和職業選手認可，很多人認為，從實力上來說，這賽季的一諾堪稱第一人。

AG超玩會入圍6項評選，這也是對他們實力的認可，但他們很可能一個獎項都拿不到，為什麼這麼說呢？

先看下2019年度王者榮耀KPL最佳選手的評選，入圍選手分別為一諾、cat、Fly、渡劫。按這賽季的表現來說，最佳選手毫無疑問是一諾，但這獎項評選的是年度最佳選手，而貓神在2019年獲得了兩個總冠軍，並且拿到了FMVP和常規賽MVP，從榮譽上來說，最佳選手應該是cat。

最佳教練的評選同理，月光也入圍了，但T將軍的榮譽明顯更勝一籌。

AG超玩會入圍的其他四個獎項為：最佳中單老帥、最佳發育路一諾、最佳輔助愛思、最佳對抗路六點六。而老帥的最大競爭對手為貓神，一諾競爭對手為花海、愛思的競爭對手為無銘、六點六的競爭對手為Fly和諾言。

從榮譽上來看，兩個冠軍畢竟大於一個冠軍，所以今年大概率會是eStarPro收割獎項的一年。AG六項入圍，很可能顆粒無收。

個人認為，客觀評選的話，今年大部分的獎項都屬於E星，但AG超玩會今年的進步實在是讓人驚嘆，賽季初沒有人認為這支隊伍能奪冠。考慮到AG的亮眼表現，最佳教練、最佳發育路這兩項AG還是有希望得獎的，畢竟花海也入圍了最佳打野，發育路這塊可能會給到一諾。
