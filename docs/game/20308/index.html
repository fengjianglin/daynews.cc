遊戲裡面玩街機？GTA5遊戲套娃

2019-12-17 340條評論 1,304次閱讀 400人點贊

近日，GTA5線上模式更新了新的玩法：「鑽石賭場搶劫」任務中隊友死亡其餘隊友也可繼續執行任務以及玩家所經營的街機廳更新了12款新的街機。說起來，在GTA5OL里玩街機並不是什麼新鮮事，甚至有許多玩家將其當作街機模擬器/啟動器（不會真的有人這樣吧），這種情況我們姑且稱之為遊戲套娃。

說起套娃這個梗，許多朋友的第一反應就是「禁止套娃」和「禁止禁止套娃」，不過這些更多的是語言上的文字遊戲。而套娃可不單單體現在這方面，在遊戲中、電影中、書中都會多多少少出現類似的「月讀」時間，那麼我們前面所說的遊戲套娃究竟體現在哪些作品中呢？讓我們一起來看下吧！

既然說了GTA5OL中的街機，那麼今天的第一款套娃遊戲也從街機說起，那就是大名鼎鼎的「背鍋俠」系列——《如龍》。《如龍》作為一款黑幫題材的動作冒險遊戲系列，除了打打殺殺，自然是要有一些放鬆的休閑小遊戲。

特別是被稱之為黑道活雷鋒的「背鍋之龍」桐生一馬被人陷害後，找個撫慰一下自己受傷的心靈這不過分吧。沒錯就是那個懂得都懂、只有男人才會去的地方，神室町的世嘉遊戲廳！

在《如龍》中，去世嘉遊戲廳不僅僅只是為了玩那兩款遊戲，更多的其實還是解鎖房產完成支線，不過那兩款遊戲確實有些意思。一款是世嘉被認為是遊戲史上最出色的街機賽車遊戲之一的《Outrun》，而另一款則是憑藉天馬行空的世界觀以及超前於時代的，被稱之為一代傳奇的《太空哈里》。

這兩款大名鼎鼎的經典之作，成為了不少玩家當初在這裡與女店員比拼分數時永遠的痛……

除開街機，打牌這一元素在遊戲中也是體現的淋漓盡致。說起打牌你第一時間想到的是哪款遊戲？鬥地主？彩虹六號？陰陽師？行了行了，不逗你們了，其實我們要說的是那個即將登上大銀幕的男人，白狼——傑洛特。

在遊戲《昆特牌》中，你不僅可以與來自世界各地的旅者進行昆特牌對局，並且還能去打怪打樁找女兒。作為一款卡牌遊戲，內置如此豐富的遊戲內容實在是讓別的遊戲自嘆弗如，不過帶惡人波蘭蠢驢竟然還讓巫師系列鳩佔鵲巢，使得《昆特牌》獨立出來成為一款遊戲，實在令人深痛惡絕。

當然，提到了打牌，就不能少了我21點，那提起21點，就繞不開最近剛上PC端就打折的作品《荒野大鏢客：救贖2》。這款遊戲其實筆者本身對其賭場並不是太感興趣（主要是不懂規則，玩不好），但是後來看了寅子的直播之後，頓時體會到了賭徒的樂趣，什麼德州、21點，贏了算我的，輸了要你命。

在遊戲《神秘海域4》中，頑皮狗設計師以一份PS1的《古惑狼》賺足了玩家們的情懷，甚至有玩家說，「拋開神海4的高質量，僅憑這個《古惑狼》就已經值回票價了」。不得不說，頑皮狗把玩家們吃的死死的，玩家想玩什麼、看什麼它似乎都知道，只可惜那個劇情關卡只能打一次，再打就只能SL了。

不知道在座的有沒有《最終幻想》系列的忠實粉絲，在FF14中其實就有許多有意思的小遊戲，在這當中最為玩家所熟知且最經典的還是以遊戲標誌擔當陸行鳥為主角的《陸行鳥競賽》。

其實早在1999年這款小遊戲的前身《陸行鳥賽車》便作為一個獨立出來的遊戲風靡日本玩家間，而我們所最先接觸到的跑跑卡丁車則是在七年後的2006年才開始發行公測。

只是可惜，後來SE不知道為什麼不再繼續進行這部作品續作的開發，而是將之移至《最終幻想》系列作品中，作為一個內置小遊戲。雖然玩法將賽車改成了騎乘陸行鳥進行競速，但是其核心玩法包括道具等都是和前身一樣的，即便畫質比之以往有了很大提升，但是還是有不少玩家在翹首以盼續作歸來……

作為復古狂魔任天堂的休閒遊戲，《動物之森》很好的繼承了休閒遊戲內置小遊戲的傳統，而這些內置的遊戲，便是我們曾經愛不釋手的各種FC紅白機遊戲。

其實不論是《敲冰塊》還是《大金剛》，都是大家曾經玩過無數遍的遊戲，內置到《動物之森》中後，不僅使得遊戲更具趣味性，同時也順便推銷了一波自家的老遊戲。要知道，時至今日任天堂還時不時在自家商城上大力宣傳售賣曾經的經典之作。

總的來說，遊戲中進行這種類似的套娃行為確實是十分普遍，除了上述之外更有《我的世界》這種BUG級別的套娃遊戲和往遊戲中加MOD的行為來實現一款遊戲（比如曾經國外有玩家做了一個在DOTA2中玩CS：GO的MOD，引領了一陣熱潮）。

說起來，有時候正是因為這些遊戲的堆砌使得玩家們在遊玩時獲得了不止雙倍快樂，達到了1+1＞2的效果。不知道各位是否還有知道什麼有趣的內置小遊戲，一起來分享一下吧！
