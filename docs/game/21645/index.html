不要夢淚也得帶上他，AG冬冠名單公布，月光：讓廣州出現那片紅海

2019-12-18 889條評論 2,828次閱讀 1209人點贊

各位王者玩家大家好，我是小雨。雖然秋季賽已經落幕，但是過幾天冬季冠軍杯即將開始，由於冠軍杯的特殊性，每個隊伍除了必備的教練組之外，只允許派7名成員出征，這就造成了很多替補隊員無法隨隊一起征戰冬冠，只能繼續營業自己的直播間或者參加一些官方活動。

作為新晉的冠軍戰隊，AG超玩會也公布了他們的冬冠大名單，除了教練組3人之外，還有奪冠的5個功臣以及最佳第六人藍柚，這也是他們固定的6人輪換陣容。日前有傳言夢淚將會在冬冠登場，不過大名單公布之後，可以確定他不會出征冬冠了。因為AG剩下的唯一一個位置不是給了夢淚，而是帶上了「7冠王」MC謝濤。

MC這名選手的生涯非常傳奇，17年他在QGhappy整整坐了一年替補席（射手出身，QG有刺痛不可能給他機會），技術有沒有提升不清楚，但是躺了3個冠軍。18年轉會到AG超玩會，結果AG超玩會降級了，但是他又隨中國隊征戰奧運會，以輔助的身份再獲1個冠軍。18年夏季轉會到eStarPro之後，依然是替補，到了19年春季賽初期短暫作為首發登場之後，又被無銘死死地按在替補席上，不過他又隨e星躺了2個冠軍，這樣算下來他一共有了6個冠軍了。

今年夏天，AG超玩會重組之後，把在eStar坐穩替補的MC買回來。當時就有人說AG是看中了MC的福星體質，買回來好拿冠軍，其實並不是這樣。AG本來是打算賣MC回來當首發輔助來用的，結果老帥從RW俠回歸後把他們的原首發輔助渡厄（愛思）一同帶了回來（據說是買老帥附贈愛思，RW俠是否該哭了？），結果MC又成了鐵打的替補。

​但是誰曾想AG在剛回歸就拿下了冠軍，這一下MC的「錦鯉」特質得到了所有人的肯定。3年換了3個隊伍，各個都是替補，卻拿了8個冠軍。這讓我想起了NBA曾經的球星「羅伯特霍里」，霍二叔也是換了三個隊，拿到了7枚總冠軍戒指。

月光此次帶上MC卻不帶夢淚的目的很明確，反正都是上不了場，希望MC能夠給俱樂部帶來好運，拿到冬冠冠軍，月光也是親自發聲：大家的起點都一樣，我們勢必讓廣州出現那片紅海！加油！
