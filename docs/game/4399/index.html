遊戲王：科學怪人頭髮少，實驗成功率0%，他找到手術醫生看牙口

大家好，我是弗蘭力。本期繼續給大家講遊戲王卡牌的故事。這次要講的是一名科學家的故事。

「魔導科學家」是一名中年男子，他在一家研究所里工作。他的特點就是那可以用「崩潰」來形容的髮際線，也不知道是不是對頭髮充滿執念，他將後腦勺還沒有掉的頭髮留得很長。他有兩名同事，其中一名是「平庸鬼」，另一名則不知姓名。

「魔導科學家」的效果是：支付1000基本分，就可以從自己的融合卡組中特殊召喚1隻6星以下的融合怪獸。這隻融合怪獸不能對對方直接攻擊，回合結束時回到融合卡組。

這天，「魔導科學家」和往常一樣，與兩名同事做著實驗。他們抓了一隻小龍，將「核成獸的鋼核」與之融合。如果沒有出錯的話，這隻小龍將會變成「核成龍」。可「魔導科學家」的屏幕上顯示著大大的「0%」，似乎在告訴大家，這場實驗不怎麼順利。

「魔導科學家」似乎並沒有在意屏幕上的「0%」，而是面帶笑容地看著他左手邊的照片。照片上是年輕時候的自己，那時候他的頭髮非常濃密。

「鋼核合成獸研究所」的效果是：這張卡的控制者，在每次自己的結束階段，將手卡的1張「核成獸的鋼核」給對方觀看。不給觀看的話，這張卡就被破壞。每次場上存在的，名字帶有「核成」的怪獸，在結束階段時被破壞，那些怪獸的原本持有者可以從卡組把1隻名字帶有「核成」的怪獸加入手卡。

「成功確率0%」的效果是：從對方融合卡組隨機選擇2張融合怪獸送去墓地。

後來，實驗果然出了問題，「核成獸的鋼核」能量失控了！「魔導科學家」與不知名的科學家緊急退出實驗室，那位不知名的科學家連鞋都跑掉了！看來確實很可怕呢。過了沒多久，「核成獸的鋼核」就爆炸了，爆發出了絢麗的光輝。咦，平庸鬼呢？

「失控能量」的效果是：把手卡1張「核成獸的鋼核」給對方觀看發動。對方場上蓋放的魔法和陷阱卡全部破壞。

「強制退出裝置」的效果是：雙方各自選自己場上1隻怪獸，那些怪獸回到持有者卡組。

「鋼核的光輝」的效果是：把手卡1張「核成獸的鋼核」給對方觀看發動。對方的魔法·陷阱卡的發動無效並破壞。

「魔導科學家」在這次失誤的實驗中受到了驚嚇。正好，他的定期健康診察的時間快到了。於是他前往了自己經常去的診所。診所里只有一名醫生和兩名護士，可看起來它們並不是人類呀，它們有學過人類醫學嗎？從醫生的姿態看來，似乎還是挺專業的，不過本力覺得它不應該看「魔導科學家」的牙口，而是應該看看他的髮際線以及受驚的小心臟呀。

「DNA定期健診」的效果是：選擇自己場上里側表示存在的1隻怪獸發動。對方宣言2個怪獸的屬性。選擇怪獸翻開確認是宣言屬性的場合，對方從卡組抽2張卡。不是的場合，自己從卡組抽2張卡。

好了，本期「魔導科學家」的故事就到這裡結束了，喜歡的朋友記得關注、收藏、點贊。我是弗蘭力，咱們下期再見。
