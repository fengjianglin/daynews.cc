魔獸世界：最令魔獸玩家吐槽的BOSS戰

作者：林夫人的貓

DPS打到瞌睡的龍背之戰，要說起龍背之戰，肯定令很多WOW玩家心生厭惡，但是不得不說，耐薩里奧的背脊是整個巨龍之魂副本當中設計的最具有特色也是最有難度的。

在這個副本的中間，大哥薩爾用寶物巨龍之魂成功擊中死亡之翼，智慧與力量兼具的黑龍王感覺不妙準備撤退，腳男們登上飛船開始追擊，在打退了攔截力量之後跳傘降落在了死亡之翼的背脊上，開始了這場頗具難度的背脊之戰。

我們需要拆除BOSS背上的源質鎧甲，為薩爾的致命打擊提供條件，但是這場戰鬥並不簡單，打黑龍雖然沒有痒痒撓，但是他會一刻不停的進行側身翻滾，當感覺到玩家在身體左邊的時候就會往左側翻身，在右邊的時候就會往右側翻身，被翻身甩下龍背的結果就是必死無疑。

戰鬥機制同樣比較麻煩，總結起來就是打爆觸鬚出火人，打爆火人出小軟，打爆小軟出火攤，拉著火人吃火攤，吃夠火攤炸甲板，炸開甲板打連接，打完連接甲板掉，接著循環下一輪。

簡而言之就是利用火人的爆炸來拆掉盔甲，同時需要時刻注意保持左右移動以免被側翻摔死，這場戰鬥的規則是所有單刷者都必須遵守的，哪怕你是120級的大羅金仙，一發火球成噸的傷害也沒有用，盔甲只有火人爆炸才能炸開，你也會被一個翻身瞬間摔死，沒有任何餘地。

終於，經過一番小心翼翼的操作，在連續拆掉三塊盔甲之後，死亡之翼的要害暴露在薩爾的超能打擊之下。

被擊穿的死亡之翼掉入大漩渦之中，竟然還沒有死透，因為恩佐斯賜予的力量，他的周身開始腐化並爆發出諸多觸手，已經完全瘋狂的他開始催發生命之力，妄圖製造下一個大災變與這個世界同歸於盡。

這場戰鬥的規則是玩家需要在各個平台之間轉移，逐一擊破BOSS的每一隻攀在岩石上的巨爪，四色巨龍也在燃燒生命之力阻止大災變的引導，戰鬥氛圍營造的非常的出色，身處無盡之海的我們彷彿站在世界的邊緣，狂風驟雨各種能量亂流不斷沖刷視線，黑龍王巨大的軀幹在身邊震顫，四色巨龍的嘶吼不絕於耳，伊瑟拉的夢境力量將玩家籠罩，彷彿暴雨中的透明雨傘，為我們再最後時刻提供微弱而又至關重要的保護。

最終，四色巨龍將所有神力灌注於巨龍之魂中，化為守護巨龍之心的力量貫穿目標，地表最強黑龍之王死亡之翼終於被徹底擊敗了，在無盡之海的大漩渦中粉身碎骨。

這場BOSS戰在85級別的時候，每根爪子的血量都上億，整場戰鬥雖然精彩，但是在社區裡面被玩家一直吐槽，BOSS血量高的離譜，又臭又長的戰鬥過程必須保持神經緊繃，跟當時的25HICC有的一比，不知道各位觀眾小夥伴們，你們怎麼看呢？
