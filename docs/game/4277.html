絕地求生：微博杯小組循環賽最後一輪，總決賽16支隊伍塵埃落定

2019-12-08 0條評論 3次閱讀 0人點贊

參賽隊伍：4AM、IFTY、Ark、AMY、OMG、VG、1246、FTG、Tianba、LGE、Weibo、SMG、TSG、LYG、CL、Newhappy

第一局海島圖聖山極端圖形，決賽圈WEIBO淘汰SMG，TIAN與WEIBO猛烈交火後只剩兩人，最後被LGE獨狼阿航帶走最後兩人，LGE7殺吃雞。本局MPV：LGE_AhaNg313

第二局海島圖東部圈形，決賽圈TSG被首先淘汰出局中，LYG清掉NEWHAPPY獨狼，4AM、LYG、TIANBAN混戰，FTG最後坐收漁翁之利淘汰LYG，最終3殺吃雞。

第三局海島圖西部圈，決賽圈4AM拉開戰線，四處開花，以一敵四，淘汰FTG、IFTY後，小鹿淘汰AMY獨狼，隨後韋神放倒1246一人打開突破口，前壓吃下1246最終16殺獲得大分吃雞。

第四局雨林圖左下圈形，決賽圈IFTY淘汰AMY後被VG架死。VG隨後淘汰SMG後15殺大分吃雞。本局MPV：VG_Cold9o9

第五局沙漠圖西北部左上的極端圈，決賽圈ARK強行下山被打靶淘汰，SMG和NEWHAPPY兩隊對決，SMG最終淘汰NEWHAPPY，8殺吃雞。本局MVP：SMG_17Smallcc

第六局沙漠圖東北部右上極端圈，決賽圈4AM獨狼韋神淘汰CL一人後被清掉，CL帶走NH獨狼，最終被TSG淘汰，TSG3殺吃雞。

最終，12月6日小組循環賽全部結束，16支總決賽隊伍新鮮出爐：Ark、RNG、IFTY、Weibo、YTG、VG、VC、Tianba、AMY、QM、EDG、4AM、BA、SMG、TSG、Lstars。

DAY6小組賽戰況激烈，4AM最終一改前期的頹勢，衝進總決賽；SMG抓住機會突出重圍殺進總決賽；AG 、JSTAR、NEXT遺憾沒有守門成功沒有晉級，SSS、FPX、OMG、LGD表現不佳無緣決賽，希望在後面的杯賽里能調整好狀況，拿下好成績。
