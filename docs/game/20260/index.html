《神田川JETGIRLS》海量情報公布 中文版發售日公開

2019-12-17 416條評論 4,354次閱讀 210人點贊

今日（12月17日），世嘉官方宣布：Marvelous旗下PS4平台全新動作競速遊戲《神田川JETGIRLS》將在2020年3月12日針對中國香港、中國台灣以及東南亞地區推出（這意味著中文版將於2020年3月12日發售）。此外，官方還公開了該遊戲的海量新情報，以下為具體細節。

《神田川JETGIRLS》遊戲背景介紹

『閃亂神樂』『VALKYRIE DRIVE』製作陣容隆重巨獻，全新「動作競速」遊戲將在PS4平台登場！本作描述為了賽艇賭上青春，熱血少女們的故事，由鳴子ハナハル老師繪製的魅力角色們因賽艇而相遇， 並在一次又一次的比賽中， 逐漸加深彼此羈絆的青春群像劇。

兩人一組同心協力進行賽艇比賽，大家都能輕鬆上手的戰略性動作競速！操縱各式各樣的武器以及自己的賽艇，在距離現在不遠的架空未來世界，以東京為舞台，展開火熱激烈的噴射競速！！

《神田川JETGIRLS》遊戲特性介紹

「騎手」×「 射手」 前所未聞的動作競速

所謂「賽艇」，是由操控賽艇的「騎手」以及使用水槍攻擊對戰對手的「射手」兩人一組同心協力，乘著賽艇在水面上飛馳的競賽。駕馭兩種模式，爽快地在神田川盡情馳騁吧。

使用推進及甩尾賓士於賽道的「騎手模式」！



取得及使用強力武器道具的「射手模式」！
