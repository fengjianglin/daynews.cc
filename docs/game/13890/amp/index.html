「饑荒生物介紹篇」海難鵜鶘，生物介紹

天天要聞
2019-12-12

大家好，這裡是巳寶寶，今天為大家帶來海難「切斯特」（鵜鶘）的生物介紹，話不多說，我們直接進入正題

鵜鶘（Packim Baggims）是船難DLC中的一種跟隨玩家的生物，與切斯特的功能相似，也擁有幾乎相同的數值，比如每3秒恢復22.5 的能力。請牢記，他會吃掉玩家放在他嘴裡的所有魚類。除了玩家開始遊戲時所在的島以外，召喚鵜鶘所需的魚骨可能在任何陸地生態群落或島嶼上生成。（大部分是沼澤地形）

圖中可見，他的魚骨和切斯特的眼骨很像

鵜鶘有和切斯特明顯不同的地方，鵜鶘擁有飢餓值（飢餓值沒了，不會怎樣，不會扣血）

鵜鶘有著150飢餓值，可以吃掉絕大多數魚類食物。（切記，如果吧魚類放進他的嘴裡，再關閉就會被它吃掉）他的飢餓值會以0.1 /秒或48 /天的速度降低。這個數值會被中毒影響，並和人類玩家一樣在中毒時減緩20%的飢餓速度，也就是38.4 /天。他在飢餓的時候不會丟失血量。

鵜鶘會吃掉以下放進他嘴裡的物品：

當他吃掉狗魚或者劍魚後他會吐出腐爛的魚。他不可以被冰凍法杖凍結，而且需要3發麻醉吹箭才能使他睡著。

這裡可能就會有小夥伴疑惑了，它會吃魚我可以理解，可是吃了魚有什麼作用嗎，有，往下看

胖鵜鶘是一種鵜鶘的特別版。如果鵜鶘的飽食度被玩家餵飽到 120 以上的話，他就會由普通鵜鶘變為胖鵜鶘並增加3個額外的物品欄，和暗影切斯特差不多，但不需要等到滿月就可以變身。他會在飽食度降低至 0 後變回原樣，因此玩家必須經常給他餵食才行。變回普通形態之後，鵜鶘會掉落之前放在他的額外物品欄中的所有物品。鵜鶘也和切斯特一樣擁有兩種形態

火鵜鶘

火鵜鶘是另一種鵜鶘的特別版。只要在他的每個物品欄中放置至少一塊黑曜石（海難版本特有的石頭，在火山裡面），鵜鶘就會立刻變為火鵜鶘。這個機制和切斯特差不多，但不需要等到滿月就可以完成。火鵜鶘對所有火焰傷害免疫，並會使用遠程火焰（可能導致火災蔓延）（第一次玩海難的時候，吧鵜鶘變成火鵜鶘，家裡來了獵犬，它就突然吐火給獵犬，順便幫我把家也給燒了QAQ，血的教訓）主動攻擊範圍內的所有怪物和敵對生物，包括韋伯。他還可以烤熟放置在嘴裡的所有食物，包括他不再會吃掉的魚。任何放進他嘴裡的可燃物都會在他重新張嘴後變成灰燼。

火鵜鶘形態下，它就不會吃魚了，但是把魚放進去的話，他會幫你烤熟

遊戲錯誤BUG

  • 如果玩家穿越濃霧的話，鵜鶘可能會跟丟玩家並呆在原地不動，玩家必須回到之間的位置才能讓他重新跟隨。把魚骨放在地上再撿起無法修復這個錯誤。
  • 如果玩家乘坐快船（如裝了帆的貝殼船）或乘著海浪前進的話，鵜鶘可能會因為跟不上而停在原地，玩家需要返回去取回他。他的圖標在地圖上是可見的。
