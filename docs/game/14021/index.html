《絕地求生》明年賽事：一年四場世界賽，獎金眾籌預計超過600萬

2019-12-11 62條評論 2,078次閱讀 595人點贊

今日《絕地求生》遊戲官方發布了關於2020年的電競賽事安排：

《絕地求生》遊戲2020年——全球官方賽

隨著《絕地求生》遊戲2019 PGC全球總決賽的結束的成功，藍洞在明年將會引入該賽事結構進入PGS全球系列賽中，甚至推廣到全球賽事。

在2020年，《絕地求生》遊戲將會舉辦四場高規格的官方世界賽。

其中三場為區域性世界賽 PGS，PGS全球系列賽將落戶到全球的各個主要城市。

時間：4月（柏林），7月,10月。

一場為PGC 世界總決賽，2020年年度最高級別賽事依舊為PGC全球總決賽。

與以往不同的是前三場《絕地求生》預習賽事除去規模會向PGC看齊外，也將有PGC晉級的名額產生。

在每一場吃雞比賽中將會有32支戰隊入圍。

2020年中每一次 PGS 都會有類似 PGC 2019 的賽事專屬皮膚和競猜活動推出。

於此同時，每次比賽所推出的對應皮膚銷售的部分收入也同樣會納入獎金池，和參賽戰隊進行分成，獎金池也將和以往一樣繼續開放眾籌模式。

在2019《絕地求生》遊戲 PGC全球總決賽積分排名獲得前四的戰隊，將自動獲得2020年PGS賽事的參賽資格。

換而言之，PCL賽區中國4AM戰隊已經獲得首站PGS的入場券。

提前祝4AM能取得好成績！

另外在《絕地求生》遊戲玩家的大力支持下，2019 年PGC全球總決賽冠軍競猜活動取得了非常好的成績，總金額達到600萬美元。

在後續的全球總決賽中，藍洞會繼續增加更多賽事專屬的道具，擴大獎池在2020年PGC全球總決賽中獎金總金額預計超過上一年的600萬美元。

那麼，作為《絕地求生》遊戲的老玩家對此你怎麼看？
