三國殺：這次兩軍對壘誰設計的？請你告訴我吳蜀要怎麼贏？

2019-12-23 735條評論 1,925次閱讀 431人點贊

小夥伴們大家好，歡迎來到深淵三國殺，不知道小夥伴們這兩天有沒有玩《三國殺移動版》的活動模式兩軍對壘？深淵為了白嫖一些銀幣，可是每天都準時打完兩局，但是這次的兩軍對壘不同於以前的幾次。

我感覺我非常的膈應和難受，我就想問問這次的兩軍對壘是誰策劃的？也太鬼才了吧？完全是想把吳蜀聯盟往死里整啊！技能搭配奇葩到我不得不吐槽一下！讓我深刻的懷疑策划到底會不會搭配技能。

其實嚴格來說，其他三個武將的技能我覺得都還行，就是這個卧龍的技能，讓我非常非常不滿意，感覺和鬧著玩是的。

挑釁＋八陣＋飛影，我就想問問這些技能有什麼用？尤其是挑釁＋飛影，你告訴我加了飛影后的挑釁有什麼用？我克我自己？

其實這三個技能正常拎出來沒什麼問題，飛影可以讓別人打不到自己，挑釁可以挑釁掉別人的攻擊距離裝備，八陣可以可以防止別人挑釁出殺後自己受傷。可是，策劃給對手的技能是不是太過分了？

於文則地將馳可以很輕鬆的就不用裝上裝備殺到卧龍諸葛，完全沒有要被挑釁的苦惱。最多就八陣難受點，實在不行不想殺，我摸三張牌苟一苟就行了，有距離了，膽守騷擾一下，而且因為摸三牌不能出殺，所以手上留殺的概率就大大增大，直接天然免疫了挑釁。

說完了卧龍諸葛的技能其實對於文則沒什麼用之後，我們再說說對敵方另一個武將張文遠的影響，對於張文遠來說那就更沒有影響了。

奔襲一個技能直接克制了”飛影＋八陣”，如果摸不到至少三點攻擊範圍的裝備，直接天然免疫了挑釁。你告訴我卧龍諸葛的三個技能到底對敵人有什麼影響？感覺就是用來搞笑的。

除此之外”推鋒＋奔襲＋傾襲”輸出是不是太爆炸了一點？打的好直接一輪就可以帶走對面兩個人啊。

卧龍諸葛的技能除了對敵人毫無用處之外，最絕的地方在於他竟然還一定程度上的影響了和隊友的配合，這就很靈性了，這讓我深刻的懷疑策劃你在搞我。

你見過給”曹昂”的隊友硬加”加一馬”的嗎？你讓我的”慷愾”給誰牌去？我一人我飲酒醉啊？

說句不好聽的，卧龍諸葛這匹飛影，裝了不僅什麼用都沒有反而還給隊友添堵。一句話來說就是”成事不足敗事有餘”！

而且策劃你真是想給卧龍加上一匹”飛影”保護一下卧龍，那你是不是也應該給魯肅配置一個”馬術”搭配一下呢？

其實吳蜀方除了卧龍的技能處處是槽點之外，魯肅的技能貌似也不是很強。

一個”慷愾”只適合給自己用，一個”甚賢”貌似強度也不高，熟悉三國殺的小夥伴應該都知道”甚賢”這個技能人越多越厲害，人越少越沒用。這局遊戲中，卧龍是個摸牌白，可能牌連自己都不夠用，更別說丟了。

張文遠又是一個用牌大戶，想”甚賢”撿他的牌，難度也是比較大，那唯一克制的也就於文則的”膽守”了，因為”膽守”要丟牌發動，所以可能會有一點收穫，但是如果遇到像深淵這種就不丟基本牌的老狐狸怎麼辦？

除了技能安排上這次的兩軍對壘有很大的問題之外，血量問題也很嚴重，明明魏國比較強勢，他們卻都有四滴血，吳蜀已經夠弱了，結果都是三點血？Are you kidding me？

還有就是位置上的問題，打過排位賽的小夥伴應該都清楚，正常來說除非一號位是強控，否則二三號位完全更佔優勢，你把比較好的位置給技能更強大的武將，這是不是也是在搞笑？

所以這次兩軍對壘深淵認為吳蜀就是兩個沙包，還是比較不抗揍的那種殘次品。

要是想贏，完全只能寄託於閃電和樂不思蜀，少了這兩張牌，深淵認為吳蜀聯盟除非牌運逆天，否則完全沒有任何機會。

除了勝利的這兩局魏國，深淵的兩局吳蜀都輸了，深淵玩的時候都覺得自己快秀飛起來了，結果還是扛不住技能壓制，讓我感覺非常的憋屈。

最後小夥伴們認為這次的兩軍對壘公平嗎？是和深淵一樣覺得不公平，兩方勢力懸殊，還是覺得其實是公平的，只是深淵沒有找到吳蜀勝利的方法，歡迎評論區留言分享，說說你的看法！
