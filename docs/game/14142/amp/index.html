二、門派屬性

屬性面板中：左邊是人物形象，右邊是詳細的數值參數

右邊詳細的數值參數包含：

1. 陰陽冥3種屬性：根據技能會發生變化。

2. 基礎屬性：從基礎屬性看出隱墨宗是法術攻擊為主，追求的是控制強度，通過升級加點基礎屬性會發生變化。

3. 高級屬性：主要通過攜帶裝備、強化裝備、煉化裝備獲得。

三、流派選擇

2. 當然大家也可以根據自己的喜好進行搭配，一共可以添加6個技能
