AG蓮即將回歸大名單？一笑坦言被騙，老帥：這牌贏不贏我說了算

天天要聞
2019-12-07

摘要

就在KPL秋季賽總決賽來臨前夕，網上突然流傳出一個消息，AG超玩會的選手蓮即將在下個賽季回到AG超玩會的大名單之中。大家都知道，蓮曾在一年前加入了AG超玩會，但是命途多舛，AG超玩會因為連敗降級，蓮也一直未能在KPL的舞台上展示自己的實力。

就在KPL秋季賽總決賽來臨前夕，網上突然流傳出一個消息，AG超玩會的選手蓮即將在下個賽季回到AG超玩會的大名單之中。大家都知道，蓮曾在一年前加入了AG超玩會，但是命途多舛，AG超玩會因為連敗降級，蓮也一直未能在KPL的舞台上展示自己的實力。之後，蓮還留著AG俱樂部內，從事直播工作。令人驚訝的是，蓮在直播中一共打了22個國服法師，可以說是中路法師的英雄海，玩家們也一直期待蓮能夠代表AG出場比賽。

所以網上傳出蓮回歸的消息之後，網友們都非常激動。這條消息是前RNGM選手一笑在直播中透露的，一笑作為業內人士，經常會爆料一些不為人知的業內趣聞，比如QG的Fly是如何一步步加入QG的，以及Fly和女朋友的關係等等。一笑的消息一向是比較準確的，所以直播的時候，觀眾們都更喜歡聽他講故事，一笑也打趣的說，直播打遊戲都沒人看，一開始講故事觀眾數量直接翻了好幾倍。然而，這次一笑卻被忽悠了，AG蓮回歸的消息是假的。

原來，一笑這條消息並不是從圈內人那邊聽來的，而是他自己從快手號中看到的，他關注了一個AG藍柚的快手號，正是這個號透露了蓮回歸AG的消息。不過很快就被證實，AG藍柚並沒有開通快手賬號，一笑關注的號是高仿的！常言道，老在河邊走，哪有不濕鞋，作為爆料大王，一笑這次也是栽了跟頭。其實，關注過蓮的網友們應該知道，蓮自己透露過，他對打比賽並不是很感興趣，他更喜歡從事直播工作，畢竟直播的收入更高，而且面臨的壓力也沒有打比賽那麼大。

AG的粉絲們知道這是假消息之後，很多人都表示非常失望，他們還是很期待蓮能夠出場的。但是也有一部分網友表示，目前隊內的中單選手老帥也很不錯。其實老帥在本賽季充滿爭議，在常規賽階段老帥的表現較為一般，他的打法比較保守，導致一些網友噴擊他打的太慫。常規賽結束之後，老帥再次引發爭議，因為他的場均輸出非常低，在中單選手裡面排名倒數。所以不少人說，老帥是這支AG最大的短板，AG想要取得成績，必須換一個中單。

但是季後賽階段，老帥就好像被重做了一樣，表現有了極大的提升。大家發現，過去被認為很慫的老帥，居然打的非常犀利，經常出現在關鍵位置收到人頭。而且老帥的數據也提升很大，對戰eStar的比賽中，老帥使用沈夢溪打出了41.7%的超高傷害，對戰JC的比賽中，老帥也多次打出隊內最高傷害，兩奪MVP。不僅是傷害高，老帥的站位也非常講究，永遠處在非常難抓的位置，對面想切他都找不到人，所以他的承傷非常低，通常在5%左右。網友們直言，季後賽的老帥，穩中帶秀！

戰勝JC之後，KPL官方給AG每位選手都寫了一條評價，老帥的評價是，我命由我不由天，這牌贏不贏，我說了算！曾經網友們把老師調侃性地稱為牌神，如今，狀態漸佳的老帥已經成為了自己掌控牌局之人！AG粉絲們也許看不到蓮的中路英雄海，但是他們得到的是昔日被譽為中路萬花筒的老帥！屬於老帥的榮耀，也許才剛剛開始。
