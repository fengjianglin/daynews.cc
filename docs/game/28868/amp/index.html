9.24b雲頂改動前瞻！山脈法領銜強勢崛起

天天要聞
2019-12-21

羈絆改動

　　地獄火、雷霆、影、刺客

　　3影的額外傷害由70%下調到60%

　　雷霆的傷害：100/300/500 調整為 80/250/500

　　地獄火的傷害：70%/120%/180% 調整為 70%/140%/210%

　　3刺客：75%暴擊傷害/10%暴擊幾率調整為50%暴擊傷害/10%暴擊幾率

　　英雄改動

　　

賽娜技能的額外魔法傷害由25/45/75 調整為 15/30/55

　　

阿木木的初始/總法力值由50/150 調整為 50/125

　　阿木木的技能範圍由2/3/4 調整為 3/3/3

　　

艾希每發箭的傷害由25%/30%/35% 調整為 30%/35%/40%

　　

圖奇的攻擊力由60調整為65

　　圖奇的初始/總法力值由0/90調整為0/70

　　

婕拉的技能傷害由70/80/90 調整為 55/65/75

　　

雷克塞攻擊力由65下調到60

　　

布蘭德的初始/總法力值由0/90調整為0/80

　　

魔騰技能的治療量由40%/60%/80% 調整為 40%/45%/50%

　　

劫的攻擊力由80下調到70

　　劫的護甲值由30下調到25

　　裝備改動

　　

肅靜的觸發幾率由33%下調到25%

　　

折劍者的觸發幾率由33%下調到25%

　　簡評：遊俠在9.24版本已經獲得了加強，而9.24b中艾希、老鼠再次獲得加強，遊俠羈絆在雲頂的強度再次提升;而刺客羈絆及其夢魘、劫，都遭到了削弱，加之特效裝備的削弱，這樣一來劍刺陣容的強度就大打折扣了。反觀火男、阿木木則加強了，看來9.24b的玩法將受到不少的改變。

（9.24b新晉S級陣容）

　　陣容羈絆

　　2山脈(岩雀、石頭人)：戰鬥開始時隨機賦予一個友軍1500點的護盾

　　4海洋(辛德拉、錘石、泰坦、娜美)：每4秒為隊友回復30點法力值

　　4守護神(錘石、泰坦、石頭人、阿木木)：守護者們提升300%的護甲

　　3魔法師(岩雀、辛德拉、火男)：法師有50%的幾率在施放完技能後再施放一次
　　解析：山脈賦予的高額護盾，無論是放置在前排，還是我方後排輸出，都能起到不錯的保護效果，加之9.24b版本的火男、阿木木加強，作為法海玩法的變種，目前山脈法的強度不俗。主要C位還是火男，再搭配一些防禦裝給前排，打出AOE控制的同時，火男技能進行輸出。

　　英雄裝備

　　熾天使提升藍量和回藍，保證對局中可以儘快放技能，而後續的裝備大帽子、珠光護手、鬼書都是不錯的選擇，依據實際情況選擇合成。而作為陣容的主要前排和AOE控制，石頭人和阿木木也需要一定的裝備加持，龍牙、冰拳或者伏擊之爪。

　　玩法細節

（具體站位以實際格子位置調整）

　　山脈的高額護盾用以應對刺客體系，是非常不錯的選擇，尤其是刷新到我方後排輸出時，往往使刺客無法儘快秒殺，而火男、石頭人、阿木木的AOE能力就可以大顯神威。由於陣容的主要輸出為火男，所以儘快升級人口，拿到2星火男後的陣容強度就大大提升了。作為海洋法玩法的變種，依舊比較適合在當前版本使用，尤其是9.24b的加強，以及有關劍刺的削弱，但後期海洋法還是被劇毒煉金克制的。

　　陣容羈絆

　　4守護神(錘石、泰坦、石頭人、塔里克)：守護者們提升300%的護甲

　　4遊俠(老鼠、EZ、千珏、艾希)：遊俠有80%的幾率在接下來的3秒內攻速翻倍

　　2水晶(艾希、塔里克)：水晶英雄承受的單次傷害不會超過100點

　　2海洋(錘石、泰坦)：每4秒為隊友回復15點法力值

　　解析：作為4遊俠玩法的另一套思路，則是比較傳統的前排坦克+後排輸出的陣型，艾希作為遊俠中技能提升傷害高，單點爆發高的英雄，是該玩法的核心點，再搭配另一位遊俠即可，再輔以一定的前排裝備即可。

　　英雄裝備

　　EZ技能會對敵方血量低的英雄釋放，具有一定的斬殺效果，更為頻繁地釋放技能就可以打出不錯的輸出，盧登+熾天使提升傷害和高藍量。而艾希則是傳統的攻速裝羊刀+電刀的AOE輸出+巨人殺手的額外傷害，或者是颶風和特效裝。

　　玩法細節

（具體站位以實際格子位置調整）

　　前期低費守護神+低費遊俠開局，可以給裝備打工，主要是中期高質量前排的提升，和千珏、EZ這樣的遊俠能夠合理打出輸出，用以保持安全的血量，最好是人口6-7人口就保持不錯的裝備加持，這樣可以快速升8，拿塔里克換低費，而2星艾希+2件裝備也基本可以打出巨額的輸出。中期的過渡極為關鍵，不然會導致陣容缺乏輸出，很難打贏對局而損失過多血量。

　　總結：本期針對9.24b雲頂的改動以及相關玩法的推薦，期待國服的更新。看完文章記得多多練習哦……
