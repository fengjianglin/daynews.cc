2019年度電競選手之Godv：凡是不能將他殺死的，都會讓他更加強大

天天要聞
2019-12-06

摘要

導語：用圖片記錄過往，用文字書寫傳奇，這裡是Aggro電競為大家帶來的2019年度盤點系列之電競選手篇。Aggro電競將會以選手所在領域內的成績和影響力為基礎，回顧過往展望未來。

導語：用圖片記錄過往，用文字書寫傳奇，這裡是Aggro電競為大家帶來的2019年度盤點系列之電競選手篇。
Aggro電競將會以選手所在領域內的成績和影響力為基礎，回顧過往展望未來。前四期的年度電競選手分別為英雄聯盟的Uzi、Faker、Perkz和DOTA2的Ana，而第五期我們將以絕地求生4AM電子競技俱樂部的隊長Godv為目標，回顧他的2019。

縱觀韋神的整個2019年，可以用跌宕起伏四個字來形容。這一年韋神和4AM經歷了不少磨難，孤存離隊、FGS拉閘、PCL春季賽失利……每一件事都曾將韋神推到風頭浪尖。但凡是不能將他殺死的，都能讓他更強大，這一年裡韋神帶領4AM完成了轉型、升級，也帶著這支自己親手組建起來的隊伍，走上了一個高峰。

寄卿以明月，棄之於溝渠
今年年初的時候，4AM陷入了一個低谷期，然而在這個時候隊內核心選手孤存卻選擇了離開，這對本來就成績低迷的4AM來說無疑是雪上加霜。但比起戰隊實力的削減，更讓韋神難受的是那種被辜負、被背叛的感受。就像那一句詩里說的：「我本將心向明月，奈何明月照溝渠。」
孤存剛到4AM的時候，大多數人都不看好他，那時的孤存能力稍有欠缺，而且不愛交流。但韋神頂著壓力把他留了下來，一點一點把他培養成了後來的戰神。在某一次直播中韋神曾告訴觀眾，孤存這個小孩挺努力的，自己很喜歡他。後來韋神甚至為孤存拿到了一份850W年薪的合同，然而孤存卻在戰隊最低迷的時候選擇了離開，這怎能不讓人寒心呢？

失之白眼將，收之中興郎
核心選手孤存離隊之後，4AM也進入了一個重建期，小將Forever被樹立為隊內核心。此外韋神做了迄今為止最重要的一個決定——買進KG的冠軍指揮王欣，他成為了4AM復興的核心隊員。隨後的鬥魚黃金大獎賽第六賽季，新4AM以第四名的成績剛好拿到了參加FGS的名額，韋神個人在36場比賽拿到51個擊殺，一切似乎都好起來了。

巔峰造偽誠，黃昏證信徒
隨後的FGS和PCL春季賽，現實又給了Godv一個教訓。FGS上的諸多豪強讓Godv清晰地認識到了，自己的個人能力還不夠強，雖然比賽中出現了小醒目一雷四響的高光畫面，但那並不屬於他Godv。而隨後的PCL夏季賽，他們甚至沒有進入前五，連打MET亞洲賽的機會都沒有，韋神第一次意識到，隊伍確實需要一個教練了，自己對戰術的理解確實還有些欠缺。
恰好那段時間4AM還與17戰隊鬧了矛盾，17粉絲多次利用4AM沒有FPP冠軍來攻擊韋神。4AM再次陷入了低谷，不少成績粉、個人粉離開了，但留下來的無一不是真愛，他們也在戰隊的這段低谷期不停地鼓舞著Godv。

鬥魚黃金賽，雙料戰神王
春季賽結束後，4AM正式引入了前OMG戰隊的冠軍教練逍遙，Godv被徹底釋放了出來。不再需要分心研究戰術、分析數據的Godv得到了大量的練槍時間，那段時間韋神每天除了練槍就是練槍，CS:GO、單人四排Rank、訓練賽，他只想變強。短短的一兩個月時間，韋神就完成了蛻變。
夏季賽之後的第一屆杯賽鬥魚黃金7，4AM雖然最後關頭拉閘沒有能夠守住冠軍，但他們的表現依然足夠優秀。尤其是向來被看做隊內短板的韋神，這一次憑藉18場比賽33個擊殺、5124點傷害的戰績在擊殺榜、傷害榜雙榜登頂。此後的兩屆杯賽4AM都帶了新人打，但韋神依然表現得可圈可點，他的成長肉眼可見。

怒漢斬惡龍，阿妹冠業成
2019年9月29日，4AM終於舉起了那座PCL夏季賽冠軍獎盃。這是4AM建隊以來的第一座FPP模式冠軍，為了這座獎盃他們戰鬥了整整483天。在整個PCL夏季賽季後賽的30場比賽中，韋神一共拿到了37個擊殺，僅次於風頭正盛的外星人Longskr和隊內大腿王欣。
但比起槍法的提升，更為恐怖的是韋神突飛猛進的大局觀。對手搜集信息反饋給他之後，他會迅速對場上已知的所有信息進行整合，對場上形勢進行判斷並做出合理的決策。夏季賽第二周的周決賽，韋神完美判斷出了場上所有人的位置，如同開了外掛一般，隨後4AM也在那場比賽成功吃雞。在那之後粉絲們送給韋神一個新外號——「高德韋地圖」。
也正是得益於韋神這一強大的信息搜集能力，4AM在最後的季後賽力壓IFTY成功吃雞，但這只是4AM建立王朝道路的第一步。

技驚洛聖都，惜敗奧克蘭
PCL夏季賽奪冠後，4AM將下一個目標定在了PGC全球總決賽。老實講雖然4AM粉絲們一直期待著韋神帶回一座冠軍，但他們深知Faze、GEN.G等強隊的實力，奪冠談何容易。即使在許多圈內人眼中，最有希望奪冠的中國隊伍也是IFTY，而不是4AM。在這樣不被看好的情況下，韋神和他的4AM反而釋放了壓力，展現出了最好的自己。
從小組賽到後來的半決賽，4AM展現出了統治級的實力。韋神的許多戰術安排就像開了天眼，半決賽第一場決賽圈，4AM、T1、WCK三支滿編隊混戰。韋神利用兩枚燃燒彈和幾枚煙霧彈將WCK全員驅趕到了T1一側，隨後趁兩隊交火的時機全員壓進成功吃雞，這一場比賽韋神的驅虎吞狼戰術和壓進時機是完美的。
但可惜的是，4AM又一次倒在了距離冠軍一步之遙的地方。PGC決賽最後一場，韋神駕駛著燃燒的蹦蹦沖向安全區的最中心然後被淘汰的畫面，被很多4AM粉絲稱為2019年最讓他們心碎的一幕。

回國後Godv很少和粉絲談到PGC上的失利，畢竟這是他離世界冠軍最近的一次。但希望這次的失利不會成為他心中的一根刺，而是他繼續前行的動力，繼續爭冠的動力。既然他能幫Forever完成救贖，那他也能為自己完成救贖。加油吧韋朕，凡是不能將你殺死的，都將讓你變得更強大。

本期絕地求生知名選手Godv的內容就到此為止了，之後Aggro電競將繼續為大家帶來2019年度電競選手的回顧，還請大家持續關注。
