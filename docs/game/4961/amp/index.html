魔獸世界懷舊服：製造業可直接做的7個極品裝備，3個可用到naxx

天天要聞
2019-12-03

摘要

很多玩家認為魔獸世界懷舊服好玩，其中有一個原因就是裝備保值，因為60級的裝備裝等差距不大，因此很多低級副本裝的裝能夠用很久很久。在懷舊服中，有幾件裝備非常好用，而且是製造業直接做的，其中更是有幾個能夠用到naxx。

很多玩家認為魔獸世界懷舊服好玩，其中有一個原因就是裝備保值，因為60級的裝備裝等差距不大，因此很多低級副本裝的裝能夠用很久很久。在懷舊服中，有幾件裝備非常好用，而且是製造業直接做的，其中更是有幾個能夠用到naxx。

第一件就是獅心頭盔，這件裝備鍛造可以直接做出來，ah價格在1000g-2000g。這件裝備就是為狂暴戰打造的，再加上戰士的套裝都是防禦的，因此這件裝備用到naxx也不必換，基本上是一次投資終身受用。

第二件裝備雖然是一個綠色裝備，但它可是60級版本貓德的唯一指定頭，很多人可能很驚訝，一個40級的藍色裝備這麼牛b？在60級貓德的設計並不完整，副本非常少見，套裝也都是治療裝，不然就只能去搶一些皮甲散件。而這個頭的特效變身後回復20點能量，再加上天賦的40點，等於德魯伊每次變身都能夠回復60點能量，這可是非常恐怖的，一些專治貓德的玩家想打輸出就是靠這個頭配合天賦全程變身回能才可以打出比較理想的dps。

第三件裝備是濃煙山脈之心，單看屬性感覺就是垃圾裝備，然而這卻是熊t的畢業飾品，因為熊t裝備團本中非常非常少，而且熊t抗怪就是需要護甲，這個飾品增加護甲還有抗性都比較適合熊t。這件裝備是由附魔專業製造，由於是拾取綁定的，所以需要去學附魔。

第四件裝備就是大法師之袍，是裁縫製造的，但也是拾取綁定。好在大多法師都會學裁縫專業。這件衣服就是前中期的畢業裝，甚至單看屬性一點也不比t2胸差，當然一般玩家有t2肯定會穿t2衣服，不過還要記住一點，那就是法師玩家實在是太多了，競爭不是一般的大，所以這件衣服穿到naxx都是有的。

然後是魔暴龍套裝，這兩件套也是初期的極品裝備，一般狂暴戰，盜賊，獵人都會做一套，價格也不是很貴，當然這一套也只是前期比較極品。

最後一個就是黑鐵板甲，是一個火抗胸，很多人可能覺得沒必要。目前來看mc的難度確實比較低，mt也完全不需要這東西，但是黑翼開放之後難度還不好說，而mt火抗足夠的話開荒也會比較簡單。
