王者冬冠杯：兩大賽區混戰，Krkpl成長迅速，kpl隊伍或強勢不再

天天要聞
2019-12-22

2019年王者榮耀冬冠杯的比賽已經拉開大幕了，在選拔賽中，兩大賽區將進行BO1的對決，角逐出兩個晉級名額。注意，此次冬冠杯的選拔賽是兩大賽區合在一起進行的。KPL賽區的四至八名加上Krkpl賽區的四至六名，共計八支隊伍同台競技，也是成為整個比賽的焦點。大家都知道Krkpl的歷史不長，在和kpl的隊伍交手中也是一直處於被碾壓的地位，然而此次冬冠杯中，卻傳出Krkpl的隊伍成長迅速，或許不再被kpl完虐，而KPL的隊伍也是不能大意了，因為Krkpl或許可以和他們分庭抗禮了！

王者榮耀冬冠杯

世冠杯成KPL獨秀賽場，Krkpl被打得毫無還手之力！

nova

冠軍杯的比賽逐漸的向著國際化發展，兩個賽區之間也是交手過好幾次，但是從實力上來看，依然是KPL賽區碾壓。在上屆的世冠杯中，兩大賽區的比拼也是尤為激烈，Krkpl賽區雖然還是難贏，但也開始反擊，各大krkpl的隊伍紛紛從kpl的隊伍手裡拿到小分，也是說明他們開始向kpl的隊伍靠攏，實力有很大的提升。當然，從戰績上來看，krkpl的隊伍依然被打的毫無還手之力，這和整體兩個賽區的差距有一定的關係。

再戰冬冠杯，krkpl和kpl一起爭奪寶貴的名額！

冬冠杯選拔賽賽程

此次冬冠杯的選拔賽中，是兩個賽區的隊伍一起打，不再是像以往一樣，各個賽區選出幾個名額。而是所有有資格的隊伍一起打循環賽，最後選出強者晉級。說實話這個賽制對於Krkpl來說並不是一個好消息，面對實力強勁的kpl隊伍，他們依然處於劣勢，雖然此次參加選拔賽的是除開各自賽區前三甲的隊伍，但是在實力，kpl這邊匯聚了estar、RW俠和hero久競等，對於Krkpl來說也是不可逾越的大山。

KrKPL成長迅速，已經開始和Kpl隊伍分庭抗禮！

EMC

經過時間的洗禮，Krkpl的隊伍也是成長迅速，據說目前的Krkpl隊伍在訓練賽中，不再是一味的被碾壓，隱隱有著與kpl分庭抗禮的趨勢，這也是讓人們對他們產生不小的期待。在這個賽季，Krkpl的隊伍紛紛引進kpl的選手，這也是他們成長的一個個關鍵。在此次冬冠杯中，Krkpl隊伍的表現，真的讓人十分的期待。比如有著橘子的EMC，上一次就曾在QGhappy手裡拿下小局分，這一次有了橘子加盟之後，或許可以拿下大場的勝利。

兩大賽區之間的差距在縮小，KPL的隊伍也不能大意！

estar

王者無界，聚有榮耀，krkpl和kpl雖然有著時間發展的差距，但是隨著一屆一屆聯賽的進行，Krkpl也是逐漸的向著KPL靠攏。如今在冬冠杯的舞台上相遇，相信Krkpl的隊伍也是會給KPL的隊伍造成不小的麻煩。在kpl這邊，五支隊伍都不是吃素的，而krkpl這邊也是如出山的猛虎，八進二的殘酷賽制，究竟誰能突圍也是充滿懸念。整體來說，Krkpl正在縮小與kpl之間的差距，不管在實力還是在打法上，Krkpl都開始逐漸形成自己的風格，所以，kpl的隊伍真的不能大意！

hero久競

在上屆冬冠杯中，hero久競一路過關斬將，舉起冰鳳凰杯，這一次八支隊伍同台競技，角逐出兩個名額，再匯聚兩個賽區的前三甲，進行後面的比拼。對於預選賽的八支隊伍來說，他們的征程會比較艱難，而最後到底誰能突圍也是讓人期待。對於Krkpl的隊伍來說，這次的比賽雖然困難重重，但是只要堅持到最後，或許就會創造奇蹟，他們正在奮力追趕kpl的腳步，也許這一次就能和KPL的強隊並駕齊驅！
