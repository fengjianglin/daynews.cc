鬥魚絕地求生小組賽完結，韋神卻拒絕參加全明星賽，網友：人品差

天天要聞
2019-12-21

12月20日，由鬥魚舉辦的絕地求生S8黃金大獎賽，正式結束了小組賽。本次比賽，一共有32支隊伍參加，但最終晉級的只有16支隊伍。

​晉級隊伍名單如下（按照排名順序）：TITAN、EDG、4AM、Ark、VG、RNG、BA、D7G、VC、Lstars、Su4、SSS、SMG、OMG、DAE、TSG。這次晉級的隊伍，也都是大家非常熟悉的。比如鬥魚簽約戰隊RNG、SMG、SSS、OMG等等。當然了，PGC隊伍4AM、VC也都晉級成功了。而從這張榜單上也能看到，4AM雖然排名第三，但也是也拿到了88個淘汰，應該是全場最高的。

​未晉級隊伍名單如下，同樣按照排名順序：天霸、FPX、QM、WeiBo、KG、CBT、LYG、YTG、RushB、1246、LGE、獨角獸、CSG、iFTY、LGD、Sxg。這其中，天霸、QM、WeiBo、KG、1246、LGE、iFTY都是很強的隊伍。天霸被很多網友看好，而QM和iFTY是PGC隊伍，KG是S7黃金賽冠軍，但如今都未能晉級成功。

​在12月21號，要迎來黃金賽的線下全明星比賽。在黃金賽開啟後，鬥魚就開始了全明星的投票通道。最終選取16名選手成為隊長，自行組隊出戰。而投票結束後，4AM韋神和王總，分別拿到了冠軍和季軍。

​但比較遺憾的是，韋神和王總拒絕了全明星比賽，反而是派出了4AM二隊成員參加。所以，21號的全明星比賽，大家是看不到4AM一隊成員了。下面是完整的鬥魚S8黃金大獎賽全明星出戰隊伍，大家看看有沒有你喜歡的新隊伍呢？另外，大家注意看OMG小獅子的戰隊，其中也包含了FaZe選手。所以說，這次鬥魚為了黃金大獎賽，也是付出很多的，竟然邀請到了FaZe戰隊的選手。

​

​而韋神拒絕參加全明星賽的原因，目前我們還清楚，不過4AM也派出了二隊參加。而有些網友認為，辛辛苦苦投了這麼長時間的票，他們竟然不參加，這票是不是算白投了呢？其實大家有這樣的想法也對，畢竟為4AM成員付出了，如今竟然是這樣的結果。但是有些網友太過分了，他表示：不想去讓人替，還有賊搞笑的言論，沒人願意和4AM玩，這人品該有多差！

​無論怎樣，目前全明星陣容已經出爐。所以，還是希望大家好好觀看21號的全明星比賽吧。12月21日，鬥魚絕地求生S8黃金大獎賽全明星賽來襲，大家千萬不要錯過哦。
