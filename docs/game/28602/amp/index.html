S18賽季即將到來，看到王者榮耀每個位置的吐槽，真的太真實了

天天要聞
2019-12-22

王者榮耀即將進入S18賽季，S17賽季雖然被很多玩家嫌棄，但依舊還是目前MOBA類手游一哥，其實版本的變更，只能讓大家自己去適應，一味的退游並不會讓你技術得到提升，還不如在新版本的王者峽谷找出最適合自己的玩法。

真實吐槽

當然每個位置都有自己的吐槽點，一起看看各個位置心裡的真實吐槽，是不是你在玩遊戲的時候也是這麼想的，網友直言太真實了。

王者榮耀每個位置心裡真實吐槽

  • 打野

S17賽季打野位一直不吃香，如果打野核的情況下，己方野區剛開局就被反，隊友開局心裡只有自己的兵線，沒有跟著打野一起守著野區，輔助也沒有幫助打野去入侵對面野區的話，那麼這個版本野核前期基本就炸穿了，只能通過在其他線上蹭經驗跟經濟，來彌補自己損失的資源，這樣勢必就造成每條路都發育不是很好，除了發育路。

打野位吐槽

當遊戲進入中期的時候，射手要紅、法師要藍，野核就更加難受，所以S17版本打野位一般更喜歡用藍領打野，即使自己野區爆炸，中期不要紅藍BUFFY，只要己方其他路發育好，也沒有太大的問題。

  • 中單

這個賽季的中單，還是選擇清線快的會更有優勢，在前期如果輔助不跟你一起清線，是沒有辦法遊走起來的，而中路兩邊的草叢更容易讓自己被頻繁針對，打野刷過一波野覺得血量不足還要吃你的血包，這個版本前期的血包奶量很足，隊友有事沒事還要過來中路蹭吃蹭喝，真的是讓人難受。

中單位吐槽

中單選擇個清線快的，就能上下路遊走，但是沒有抓住對面的情況下，隊友又嫌棄你吃兵線，或者你支援上路，造成下路被抓，隊友也要說你，而且支援或回來的途中還容易被抓，真的是遊走也吃力不討好。

  • 上單

S17賽季上單抗壓會比之前版本好很多，但是碰到強勢戰士邊路的對手，還是很難受的，比如曜。如果被抓的多，那麼真的要增強自己的意識了，輔助開局要是幫忙看視野，也不容易被抓的，除非對面集合幾個人強殺你。

上單位吐槽

當然如果上路抗壓，下路推塔不成還被反殺，那麼這局遊戲就越來越難了，當然S18體驗服對於對抗路的經驗更改，對於一些上單這個位置還是很好的。

  • 射手

射手強勢的版本之子，經濟發育就是比別人高，當然射手還是很依賴輔助保護對的，輔助如果跟了打野，射手就容易被抓爆，畢竟這個版本抓爆對面射手是最好的，前期沒人保護，後期沒人管，那麼射手的處境是很難的。

射手位吐槽

當團戰爆發時，射手肯定是敵方集火的第一目標，所以保護能力差的或者開團型的輔助，己方射手就很容易直接被切死，「團戰我先死，輸出靠你們」。

  • 輔助

輔助永遠是第一背鍋俠，所以喜歡玩輔助的玩家越來越少，對於先手開團的輔助，如果隊友不在身邊去開團，也不能怪隊友跟不上，還是要多注意小地圖。打反手的輔助，只要射手不被秒，自身有控制技能，還是有機會保住隊友輸出的。

輔助位吐槽

不過輔助容易背鍋，還是輸出不夠理解自己吧，有些輔助保人能力確實不強，這種情況下，還是注意自身走位，如果一味的讓輔助背鍋，那麼這局遊戲基本就涼了。

結語

不管怎麼說王者榮耀還是目前最受歡迎的手游之一，每個人對於遊戲的理解不一樣，在遊戲中最好還是不要一直責怪隊友，贏下這局遊戲才是最好的，對於遊戲中的逆風，不要著急逆風翻盤才能讓人遊戲越玩越好，希望看完這篇文章小夥伴在S17賽季末上分大吉。
