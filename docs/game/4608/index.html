CFPL丨S15常規賽大盤點：SV綜合排名第一，DBQ獲MVP

2019-12-06 0條評論 15次閱讀 0人點贊

摘要

常規賽結束已經有一段日子了，距離12月21日季後賽開幕也是時日無多。在上周的全明星頒獎禮上，揭曉了S15常規賽MVP，常旭獲得殊榮。另外來自eStar的JIE摘得新人王桂冠。下面我們就一起再來回顧下常規賽的幾個大數據吧！

常規賽結束已經有一段日子了，距離12月21日季後賽開幕也是時日無多。在上周的全明星頒獎禮上，揭曉了S15常規賽MVP，常旭獲得殊榮。另外來自eStar的JIE摘得新人王桂冠。下面我們就一起再來回顧下常規賽的幾個大數據吧！

【SV戰隊九戰八勝，綜合排名第一】

除卻在最後一場對陣KZ的比賽中失敗之外，SV在整個常規賽都保持勝利的姿態，實力的確是強。我們驚喜的發現，經歷席位制改革後的新五人組，不僅延續了以往SV的精髓，在風格上更具突破性，迎來了全新的開始。

【常旭獲S15常規賽MVP】

SV戰隊的常旭，在常規賽創立的數據如下：總擊殺199，MVP次數52，場均擊殺22.11，場均MVP5.78，由於他在比賽中出色的表現和為隊伍作出的巨大貢獻，榮獲S15常規賽MVP稱號。

【JIE.eStar獲封新人王】

雖然eStar戰隊整體表現不佳，最先被淘汰出局，但是小將JIE的個人發揮卻很出色。總擊殺269，回合MVP60,98個爆頭，KD高達1.08！看來是金子總會發光的，JIE的個人能力得到了人們的肯定。

【Li.WE以281個人頭斬獲最多擊殺數】

比第二名的1998多出3個，比第三名的寵兒多出6個。Li在常規賽共創造了14次三殺，5次四殺。真是英雄出少年啊！

【eStar和EDG無緣季後賽】

因這兩支隊伍在常規賽都只取得了一場比賽的勝利，積分墊底，所以非常遺憾的沒有進入到季後賽。新隊伍在陣容磨合和臨場心態方面都需要投入巨大的工作，希望他們在下來儘快查漏補缺，總結經驗教訓，下屆聯賽能夠有全新的面貌。

輝煌也好，榮辱也罷，總之常規賽已經成為過去，所有走到季後賽的隊伍，都要珍惜這個機會，爭取打出更好的水平來。12月21日，我們季後賽再見！
