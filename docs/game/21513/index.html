兵器格鬥遊戲巔峰新作《侍魂 曉》，小談見識及實戰招式技巧篇

2019-12-19 488條評論 4,607次閱讀 1335人點贊

序

名震天下的《侍魂》遊戲系統界面

1993年出版的《侍魂》兵器格鬥遊戲，迄今已問世近26年頭，它作為最早衝破經典《街霸2》規則的格鬥遊戲之一，也是當時最有創意的格鬥遊戲，其採用了不同的格鬥系統，對很多操作進行了改進，如：圖形效果、拉鏡頭效果等。它的續集《真侍魂》也受到遊戲界和玩家們的狂熱推崇。作為加強版的《侍魂》，《真侍魂》在玩家們細玩後有種「特殊」的感覺，它除了保留「格劍」、「怒氣槽」等系統之外，還追加了大跳躍、前後翻滾、伏地閃避等新動作，必殺技更是花樣百出，在普通必殺技外，還有超必殺技的武器破壞技、隱藏超必殺技、Q版變身技等，追加的要素極多。但該作的畫面是在《侍魂》的基礎上重新描繪的，華麗無比，1994年的《真侍魂：霸王丸地獄變》增加了前轉、後轉、伏地、下段躲避、受身反擊等躲避系統及武器破壞必殺技；1995年的《侍魂3：斬紅郎無雙劍》出現了雙向選擇修羅、羅剎劍質的系統。此外，還增加了武器破壞必殺技、見切等招數，三級級別選擇；1996年的《侍魂4：天草降臨》新增加了派生一閃和連斬的怒爆發系統。此外，還增加了連續斬、出現了多種打擊力強大的連續技和跌倒反擊等；1997年《侍魂64》增加了S·C·S和「一閃的極」等防禦、躲避的連續技能和跑動衝撞、體力槽、橫向移行等項目。其3D畫面，令人耳目一新。《侍魂》系列的不斷更新和完善系統，這才是讓玩家們一直致力於青睞，雖然自從2008年以來，其系列開發速度放緩，甚至達到沉寂狀態，但《侍魂 曉》於今年6月正式上架，該作即是該系列的前傳，又是重啟之作，成為近十年來該系列首款講述主線軸劇情的《侍魂》遊戲，同時能夠重現90年代初期的輝煌。

遊戲簡介

《侍魂 曉》遊戲海報

《侍魂 曉》是SNK旗下的兵器格鬥遊戲《侍魂》系列最新作，距前作《侍魂 閃》出世，已經歷經了10年的歲月，作為重啟作閃亮登場。霸王丸、娜可露露、加爾福特等歷代人氣角色將以嶄新的視覺表現開展激烈的戰鬥。

見識

《侍魂 曉》遊戲畫面風格

本作繼承了《侍魂》系列的獨特3D畫面風格和2D玩法，恰恰的原汁原味還原了老玩家知足的感覺。雖然遊戲的手感與最經典的《真侍魂》有較大的出入，但那緊張又刺激的感覺能讓玩家夢回20年前，其該作遊戲系統集歷代之大成，一刀一式的節奏和強威力斬殺效果都在，完成度和意外素質頗高。但是，該作缺點也是大有存在的，單機內容偏少、部分的招式判定有誤和讀盤時間偏弱，需要官方有待進行修正和完善。

操作說明

本作遊戲體驗的重中之重無疑是它的遊戲系統，玩家們需要花一些時間來適應，建議先進行修鍊指南項目，然後循序漸進的掌握遊戲的全部系統。

  • 按鍵

本作採用了格鬥共用的四鍵制，即ABCD四鍵來對應按鍵指令：A輕斬、B中斬、C重斬、D踢擊。

  • 通用出招

向前衝刺：66

後退閃避：44

實戰技巧

  • 跳躍篇

切記謹慎使用起跳，因為起跳破綻較多、風險較高，地面空投技和後跳空中迎擊無法得到便宜，飛行道具技能較弱；連續跳躍有非常明顯的停頓，但可以在起跳後落地的瞬間快速輸出7/8/9來消除停頓。

  • 前衝刺

衝刺的作用就是快速接近對手，躲避部分飛行道具。快速輸出66指令可以向前跨步，輸出66+6（不放）則可以向前衝刺。輸出66+A/B/C/D也可以向前衝擊，在衝刺狀態中按A/B/C/D也可以向前衝擊。向前衝擊的作用不同，下段攻擊、中段衝擊還可以接續必殺技甚至武器破壞技的銜接攻擊。

  • 看破防禦和還擊

當被對方即將攻擊命中的瞬間，輸出防禦指令並防禦成功，也就是及時防禦。看破防禦對幀數要求較低，所以成功率大。注意看破防禦前提要防禦成功，不要拿這技能來乾糧，對於投技、防禦不能技是無法使用的。但它的好處有很多：增加怒氣值、不吃削血傷害，還可以輸入A+B發動還擊。但是還擊超長距離的招式會顯出破綻。

  • 彈回、奪白刃

彈回和奪白刃的輸出指令都是236A+B，區別就在有沒有武器。兩者都是能將對方的武器直接打飛的技能，其效果非常精彩，但是兩者都不能造成傷害，只是將對方陷入無武器的狀態。注意，彈回和奪白刃僅對斬擊（除空中斬擊外）有效，對空手、踢擊、投技和飛行道具無效。

  • 投技

投技不具備攻擊力，只是將對方推或拉從而產生硬直。其共通指令：C+D向前，4C+D向後。投技具有不可防禦的特性，對空中的對方無效。在版邊可以接重斬，一些角色還可以接彈飛武器必殺技；其無法拆解，可以被閃避過去。

  • 怒氣頂點和爆發

當怒氣槽蓄滿能量時就進入怒氣頂點，同時角色的攻擊力大幅度上升，並能使用彈飛武器必殺技，在維持一定時間後自動解除，注意：彈飛武器沒有命中對方，可以反覆發動。而怒氣爆發則並不需要進入怒氣頂點狀態，只是角色持有武器且在怒氣槽就能發動，在沒有被對方擊倒、擊飛，在連續技中可以發動怒氣爆發，從而消除硬直避免被追擊。怒氣爆發和怒氣頂點一樣可以大幅度提升攻擊力，但是怒氣爆發十分強勁，遊戲系統有限制，當其狀態結束後，怒氣槽自動消除。

  • 秘奧義

秘奧義技能對於本作所有角色的輸出指令是共通的（641236C+D），此技能隨時可以發動，與氣槽無關，但是一場比賽只能使用一次。其威力與剩餘體力成反比，沒有無敵時間，很少適用於連續技能中。

  • 彈飛武器必殺技

彈飛武器必殺技主要是能將對方的武器打落在地，具有一定的威力。在怒氣頂點或爆發中輸出236B+C，就可以使出其技能。其技能出招速度相對較慢，可以接中斬或投技後面，實用性較高。注意沒有擊中對方，可以在怒氣頂點或爆發的狀態中反覆使用。

結束語

總之《侍魂 曉》對於火熱的Kof、街霸等格鬥遊戲而言，它更注重對戰雙方的博弈技巧。雖然其觀賞性並不完美，缺少專研連技的樂趣，但是對戰中的局勢和格鬥場面更為精彩、刺激。此外，本作的格鬥效果在虛幻4引擎的支持下有著不錯的表現力，整體的品質還是值得可觀的。

關注拆拆的青蘋果，跟著拆拆的青蘋果了解經典遊戲里的精彩故事，感謝大家的閱讀！
