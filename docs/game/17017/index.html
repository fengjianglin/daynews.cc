《問道》八仙獻寶火爆來襲，超多福利等你來拿

2019-12-14 675條評論 3,384次閱讀 1773人點贊

寒冬臘月，又迎來了每年的最後一個月，這個月可真是驚喜多多，光宇遊戲旗下的《問道》也迎來了八仙獻寶的盛大活動。如何參加這場活動呢，很簡單，玩家使用手中的元寶就可以啦！

使用元寶抽獎，就可以得到像神獸啊金元寶和一系列的重大道具獎勵，今天就一起跟著小編來看看八仙獻寶的正確的玩法吧！

要想參加這場活動，有一個小小的門檻哦，那就是真身（元嬰）等級需要大於四十級，才能參加到這場活動中，玩家們在登錄遊戲後，畫面都會提示「八仙獻寶」活動按鍵，通過此按鍵都能去參加這場活動的，在整個活動畫面上，有六個小柜子可以打開，每個小柜子里都隱藏了大寶物，還有一些代表了八仙的道具，像是漁樵唱晚、仙花朵秀等等各種道具。其中還有一切數量及其少，若是能抽到真是手氣棒！各位玩家加油哦，爭取抽到！

小柜子里除了有寶物，還有不計其數的金銀元寶，玩家每次開箱都需要二百元的金元寶。要是選擇雙倍獎勵，同樣開箱費用也要付雙倍的。各位玩家在遊戲畫面中可以自己選擇，可選擇只抽一次或者連續抽，箱子都會更新並且進行下一輪，更為大家關注的是，如果選擇連抽的各位玩家，連抽十次，同樣也會分十次獲得獎勵，也會每抽一次更新一次的。而且中途不能退出活動畫面，退出後抽到的道具獎勵也會損失。一定記得不能退出哦。這是此次活動中的一個重點。

除了這些，各位玩家們還能選擇使用一百個或者七百個元寶偷偷看箱子里有什麼獎品，使用一個百可以偷看其中的某一個，七百個就可以選擇都偷看，偷看結束後，還可以繼續選擇使用雙倍元寶開箱的，這樣就可以獲得各種心儀的獎品哦。當然如果偷看後沒有玩家們需要或者喜歡的獎品也可以選擇置換按鈕，這樣箱子就會更新一批。在八月份的時候，這個遊戲也迎來了夏季活動。上次的活動是通過煉丹，隨機活動各種道具，而此次的八仙獻寶比上次的活動力度大很多哦，各種獎勵也是進行了翻倍，這次活動有一個重點，那就是活躍在經典一區的玩家是不能參與的哦。

在參加這場獲得道具後，玩家們可以通過使用道具獲得等價值數量的元寶。也可以組合在一起使用，在活動大使處也可以兌換價值非常高的道具，如特技鑽石、特技寶典等等。當然這些道具都是限量的，要用搶的哦，而且寶物是不會重複出現的哦，看好就一定要下手哦，每天零時會更新的，所以時間快，任務急，玩家朋友們一定要記好時間定好鬧鐘哦！　

這就是這場活動所有的活動內容啦，主要是以抽獎為主，展現運氣的時候到了，大家快快行動起來吧！
