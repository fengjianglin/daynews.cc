王者榮耀冬冠杯：八強抽籤分組結果完美，KPL戰隊晉級形勢一覽

2019-12-24 1,985條評論 4,139次閱讀 3005人點贊

隨著選拔賽的落幕，各戰隊的積分排名情況也最終確定，TS和eStarPro分別以6個和5個積分位列前二，一起拿到了2019冬季冠軍杯正賽的最後兩個名額，此次選拔賽不僅帶來了一局定勝負的精彩對抗，還首次出現了「父子」組合—魯班七號和魯班大師（花海/無銘）！

另外非常關鍵的抽籤分組也在選拔賽結束之後正式開啟，那麼本屆冬冠杯的八支戰隊分組情況如何呢？一起來看看吧！

此次參與抽籤的依舊是KPL兩大毒奶—解說瓶子和主播拖米，經過緊張刺激的抽籤環節後，最終的對陣情況分別是A組—SLT對陣eStarPro（12月26號）、B組—AG超玩會對陣ROX（12月27號），C組DYG.JC對陣TS（12月27號）、D組—QGHappy對陣GOG（12月26號），可以說這次抽籤表現堪稱完美，不僅最大程度避免了KPL內戰，而且KPL人氣最高的三支戰隊分別迎戰境外賽區戰隊，大家都知道KPL代表著當前王者榮耀最高實力水準，因此estar、AG超玩會和QG翻車的幾率極低，這三支戰隊基本已經提前鎖定四強名額！

而八強賽唯一一場KPL交鋒在DYG和TS間展開，可以說TS運氣不夠，畢竟剛從殘酷的選拔賽晉級，而第一個對手便是強敵DYG，不過既然進入正賽，想來雙方也會全力以赴，最終鹿死誰手還未可知！對此瓶子也第一時間Po文表達了歉意，不過小編覺得瓶子和拖米這波分組堪稱鮮奶，絕大多數粉絲都是滿意的，畢竟無論怎麼分，都至少會出現一組KPL內戰。

若無意外的話，四強交鋒會是E星對陣TS和DYG的勝者，而AG和QG將重演秋決賽，此戰將是一大看點，大家可以期待一下，那麼小夥伴們你更看好哪支戰隊捧得金鳳杯呢？歡迎留言討論！
