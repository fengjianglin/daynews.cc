色字頭上一把刀！老玩家幫派兩周被搞散，罪魁禍首居然是一個妹子

天天要聞
2019-12-04

摘要

投稿玩家：花間狸

如果現在問你，作為一名遊戲玩家，最痛苦的是什麼，你大概會回答斷網。如果問你最幸福的事是什麼，那應該就是找個志同道合的女玩家告別單身了。

然而作為一個開起荒來六親不認，打起排位廢寢忘食的深度遊戲患者，我深知道一個事實，像之前說的幸福的事，只存在於歐洲人身上。而非酋，只有兩個選擇，成為單身狗，或者舔狗。

講實話，現在的遊戲圈子，女生是越來越多，遊戲奔現成功的也不那麼稀有，不過十幾年前那個時候遊戲女玩家真心比恐龍多不了幾隻，就拿魔獸來說，男女比例54比1 平均排兩場半戰歌，你只能遇見一個女玩家，還不一定是哪個陣營。

然而那時候我玩的遊戲，女生就更少了，那就是天龍八部，這遊戲我足足玩了一年多，留下滿滿的回憶，不過有一件大大糗事讓我漸漸退了坑，就是因為一個所謂的「妹子」。

其實這遊戲當時能玩下去是因為幫派，最開始就看著新鮮，下載下來建個號，玩了個少林，本來以為玩不長，早知道能玩一年多，我高低選個帥點的門派，當然我也不是什麼高手。

當時剛建完號，就看見有個人沒完沒了的喊收人，我當時也是好奇，想看看他裝備，這下壞了，追著你入幫，足足倆小時，簡直死亡小窗口，幾乎掛著機在那干聊啥也不幹，後來實在抗不了，入了。

剛入坑的我也看不懂什麼裝備好壞，幫主是個天津人，不管說話多正經，都像講相聲一樣，頓時讓我感覺，其他遊戲都黯然失色，聽他說話一天笑到晚。

那時候，遊戲內活動其實並不多，整天就是上YY吹水扯皮，搞事情，雖然基本就是掛機，但是網遊往往就是玩個社交，那時候正趕上寒假，幾乎天天上線刷一條龍。直到有一天，發生了一場意外。

幫派有個小夥伴沒上線，我們這固定隊少個人，還沒開喊呢，有個峨眉求組了。進了隊以後還沒等咱幫主死亡連CALL，這人自己問咱們有沒有YY，幫主樂壞了，以為幫里又能多一口人，還沒等咱們張嘴那，那個峨眉出聲了，竟然是個女玩家。（在此不爆ID）

卧槽了，女玩家，我的天哪！聲音真特么甜，瞬間幫里的大老爺們荷爾蒙都上來了，本來都以為軟妹能給遊戲帶來一絲色彩，誰知道，後來全是奇事。

有天我上線，在幫派頻道喊了聲一條龍來人，結果沒人愛動彈，讓我開車。這個時候還差個人，正巧趕上峨眉妹子上線，就組滿了，誰知道峨眉妹子說要帶個朋友，問能不能讓人換車。結果這幫沙雕，吃飯的吃飯，拉屎的拉屎，都不在電腦前邊。

我尋思等他們回來吧，看看誰能換車，等人都回來了，大夥一看妹子在組，那聊的一個投機，感覺幫主的段子都比平時質量高，結果說到換車的事，全爭著搶著開車，最後竟然給我踢出去了！

說實在話，那個時候天龍玩家平均年齡已經算大了，不過還是一群屁孩，搶個怪開個紅，口吐芬芳這都是正常事，不過自從峨眉入幫以後，打架的次數明顯增多了，倒不是說不愛打，就是頻繁的噴人也影響發揮不是？

這峨眉三天一小嚶，五天一大嚶，嚶完了幫主肯定是世界頻道開始口吐芬芳，約地方開打。有一次打完一回合，對方派個小號入幫對質，說是我們有個峨眉殺他們幫小號，還口吐芬芳，怎麼說是他們先挑事的。幫主一個反手就給那個人踢，這一波操作的確高能，估計放現在的劍三里，直接就能上818水到12級。雖然沒給人解釋的機會，但是自那之後，屬實沒幾個人天天妹子長妹子短了。好像只有幫主執迷不悟。

經過這次白蓮花事件，很多人都勸幫主把那峨眉踢了，咱們不是怕打架，但是讓人當槍使的感覺的確有點噁心。結果這傢伙根本聽不進去，動不動就買點卡，給峨眉買東西。幾乎只要峨眉那妹子在幫派頻道一個哥哥，一個嚶嚶嚶，裝備就能提升一點。

不過這轉變不了幫里兄弟對她的印象，幫里老人開始有意無意的疏遠她，有一回組一條，幫里一個新來的逍遙開車，我進了組以後，這峨眉也要打，逍遙不了解這些爛事，把她組進來的，本來五個人都要開車了，瞬間退了倆，說是突然不想打了，其實我也沒想到幫派里的氣氛已經惡化到這個程度了，當然我也退了組。

本來以為這峨眉識趣趕緊退幫得了，沒想到一直死賴在幫里，而且還出現一個新的惡習：空手套白狼。

說實話那個年頭沒有舔狗這個詞，但是伸手黨可是有的，幫派里人多，活躍的小20號人，還不斷有人被幫主的天津段子拉進來。這些新人有的是老玩家，這峨眉在YY上動不動就作點什麼幺蛾子，要個石頭，代打個一條龍啥的，因為幫里老人不愛組她，她就總讓新人替她打，一連五天不見人，新人啥也不知道天天雙開，估計還有點別的想法。這事直到幫里一哥們看不下去了。

這哥們也是個少林，那天峨眉又沒上線，有個新人替峨眉一條龍，結果讓這少林哥們一通臭罵，把峨眉的一堆爛事抖了個乾淨，我看的這個過癮。

不料幫主不樂意了，倆人就在YY掐了起來，噴的那叫一個芬芳，各種畫面層出不窮，他倆嘴裡那場景要是出現在影視作品裡，100%全是馬賽克。

吵到最後，少林就一句話，今天你要是不給這個碧池踢了，我就退幫。幫主當然也沒讓著他，就讓他滾蛋，結果那一下退了7個人，我尋思著這幫派一天被這個峨眉控的死死的，一起退了算了，結果幫主真的來了句「你也覺著她有問題？」，我覺著他現在還執迷不悟，沒救了，乾脆直接給遊戲和YY退了。

這回氣的我兩天沒上線，本來都打算退坑了，想起來這幫派就鬧心。直到有一天我上QQ，發現好幾天沒聲的QQ群聊起來了，都說被踢出幫派了，也不知道咋回事。那天津幫主晚上有自習還聯繫不上，大夥一碰頭得出個結果，幫主為了和峨眉一起玩，給大夥全踢了。我這一上線，發現事情還是想簡單了。

號里所有能賣錢的東西全沒了，讓人扒了個溜乾淨，好友一個沒有，而且也被踢出幫派，雖說號里東西，值不上幾百塊錢，但是心也涼透了，知道我號的人就幫主一個。莫非這兄弟瘋了？

給他QQ留言，等了半個小時，收著倆字「嘛事？」

因為誰都有偶然上不了線的時候，老人混久了，賬號互相都知道，幫主幾乎知道所有老人的賬號。為了這事，咱們又在YY上集合了，碰了頭才知道。峨眉找幫主要東西的時候，幫主不在線，結果就把所有幫會兄弟的賬號密碼的txt給他了。估計是頂其他人沒頂下來，都在線，就我和另外一個沒上線的哥們加幫主被盜了，臨了還把所有人都踢出了幫。

咱仨人在YY里啥也說不出來，我就在那草草草的連續七八聲。說實話雖然這事讓人心涼，但是實在沒想到幫主竟然哇的哭出聲了！

問他也不出聲，就在那乾哭。弄的YY里靜悄悄的，過會就啪啪的聲音傳出來，估計是在抽自己的臉。好多人都勸他算了，就當買個教訓，我和另外被盜那個兄弟也承諾不怪他。整勸了他一分來鍾，他才停下來，說了句讓我靜靜，然後就下了。

差不多有一個禮拜吧，幫主M我，問我回來玩不，重新來過，這回肯定聽兄弟們勸。我尋思玩吧，這一禮拜也試了不少遊戲，都沒內味，也沒有人在新手村纏著你入幫，號雖然被盜了，但是也沒多少值錢玩意。

但是天津人就是天津人，我上號重新加他好友，他說ID換了，之前那個號不要了，發現他又建了個一級少林，就叫聽人勸，這啥意思？雲出家？於是我們又走上天天一條吹水聽相聲的生活。

這事的經過雖然有點扭轉曲折，但我相信，國內網遊大環境下，比這種事情還精彩的「毛人毛事」絕對不在少數。我想，這些「毛人毛事」也算是我們當年的遊戲情懷之一吧。
