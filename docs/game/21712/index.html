DNF：關於搬磚裝備以及裝備處理問題的詳細介紹分享

2019-12-18 362條評論 2,783次閱讀 52人點贊

首先說一下，本文章僅適合搬磚玩家觀看，肥宅我也是癱瘓在床有三四年的時間了，這期間也是靠搬磚來賺取一些零花錢的，所以我發的文章也是對於一下喜歡搬磚的玩家看的，對於那些想說教的，我只想說，出門看哪棵樹下陰涼就去哪坐著吧，謝謝您了

進入正題，最近直播的時候，好多玩家問我裝備怎麼搭配，收益怎麼樣，裝備怎麼處理，這幾個問題是最常見的，今天我就勁量詳細的給大家介紹一下

肥宅醬的裝備搭配

全蝕市場裝備搭配，首先看看我的裝備吧，我的是最笨的搭配方法，首選哈林史詩皮夾，套裝屬性增加30%的移動速度，武器也是哈林史詩武器或者蒼穹，當然90級的史詩武器也可以的，我是花時間做的鳥貝首飾，如果沒時間的可以用哈林史詩首飾湊合著，輔助裝備我是首選尋龍訣，既然刷全蝕市場，我肯定追求最大收益了，魔法石和耳環是艾肯三速，當然不願意花大價錢帶尋龍訣的也可以換成艾肯三速的，這套搭配下來打噩夢級別沒問題

如果手法好的話可以把防具五件也適當的換幾件艾肯三速，然後打王者級別，因為現在發視頻大多都是打王者級別的，很少有噩夢級別的，我手法不好，所以我優先追求傷害，其次是速度，我知道我這套搭配周期長，而且代價也大，但是沒辦法，我是長期搬磚的，不是搬一下就算了的，所以我必須這麼干，當然我這也是僅供大家參考

全蝕市場噩夢級別有尋龍訣一個角色的收益在160萬到180萬左右，平均在160萬到170萬上下浮動，沒有尋龍訣的收益在120萬到130萬多左右，藍白分解，紫裝賣商店或者拍賣

肥宅的鋼鐵搭配，右邊哈林史詩首飾或者鳥貝首飾，特殊裝備都是哈林史詩傳說，武器哈林史詩或者蒼穹武器，左邊有霸域鞋子最好，沒有的就全部艾肯三速，刷鋼鐵的是七個角色，還有一個修羅沒發出來，裝備一樣的，第八個角色也是修羅，裝備正在做，鋼鐵一般都是速刷，路上的東西能撿就撿，撿不了的就不撿了，收益大概在100到120左右，平均在110左右，手不殘的基本上20分鐘一個角色妥妥的，藍白分解，紫裝賣商店或者拍賣

上面看完了搭配，下面給大家看看藍白裝怎麼處理比較合適，下面以生鏽的格斯泰爾短劍為例，分別用系統分解機和11級分解機測試一下看看能分解多少東西，短劍價格是8618金幣

這個我之前是測試過的，無色價格在120以上分解裝備是穩賺不虧的，這次我們再來詳細的看看

系統分解機

系統分解機得到78個無色，按照120算是9360金幣，再扣除5%的手續費是8892金幣，原價是8618，凈賺274個金幣，如果有驢子的話能再多賺點

11級分解機分解機得到94個無色，按照120計算是11280金幣，扣除5%的手續費是10716金幣，原價是8618金幣，凈賺2098金幣，有驢子的話是2323金幣

所以藍白分解是穩賺不虧的，有時候無色可能沒120，但是大家可以等漲價了再出手也是一樣的，這東西放著又不會過期是吧

最後就是刷哪裡合適，沒有尋龍訣的強烈推薦刷鋼鐵格藍迪，當然有些賬號不適合刷鋼鐵格藍迪會被制裁，那就沒辦法了，有尋龍訣的就優先推薦刷全蝕市場，當然這些全是肥宅我個人的搬磚建議，僅供參考，還有就是奶蘿別刷鋼鐵，搬磚的時候盡量退出WeGame，肥宅以前搬磚被制裁的時候說是第三方，後來不用WeGame後就沒被制裁過了，這也建議僅供參考

好了以上就是我這次整理的東西了，各位看看還有哪裡漏了得歡迎評論補充
