★遊戲馬蹄鐵原創

Impact購房產

近日SKT前上單，Faker老友Impact宣布自己個人在韓國購置了一套房產，這是他用自己從2012年出道以來賺取的錢進行的一筆投資。他希望這棟房子未來可以成為戰隊基地，Impact是韓國最老的一批上單之一，在2013年加入SKT的K隊並奪冠S3，在S5開始效力於北美，近兩年在Liquid拿下了LCS的4連冠。

Impact投資房產購買了一棟辦公大樓（寫字樓），並命名為「Impact」

這棟樓雖然不是特別高，但對於一般人來說同樣價值不菲。Impact做這樣的投資算是比較有遠見吧，畢竟如果以後電競發展下去，某支戰隊需要用到基地的話，這棟大樓是可以進行租用的，而且房產也是存在升值空間的。

職業選手到底能賺多少？

這個其實因賽區而異，傳聞中四大賽區里LPL的薪資是最高的，其實就是北美，然後才是歐洲和韓國。目前LOL的職業選手還賺不來像NBA和歐冠那些傳統體育明星的錢，別聽那些亂七八糟媒體的口嗨過億報價，當然電競選手賺的錢絕對要比普通職業高很多，因選手價值而已，從百萬到千萬不等一年是有的。

Faker也很有錢

LOL職業圈中像Faker這樣級別的選手，他每年賺的錢是非常多的。早前Faker也是在直播的時候調侃粉絲的打賞「我不缺錢，我很有錢」。只是比起一些明星選手和主播喜歡秀豪車泡美女，Faker比較低調，也許他的私生活中也有很多大筆投資，但至少目前為止從未有過Faker相關大手筆花錢的確切消息爆料。相反，關於Faker和錢的問題，大多數都是調侃李哥太吝嗇，摳門，一毛不拔，活脫脫的一隻電競鐵公雞形象。例如最常見的玩笑就是：「李哥請喝可樂，全隊一瓶，一人一口」之類的。

電競選手屬於高收入群體，Impact買大樓並不讓人感到意外，他也是為數不多能夠在海外聯賽混得非常好的韓援選手。雖然不像Faker能一直保持高強度紮根在SKT，但某種意義上Impact也是長壽型選手。
