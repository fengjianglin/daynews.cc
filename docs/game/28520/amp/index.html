爐石傳說：酒館戰棋惡魔起飛？魚人不胡可走雞雜，吃雞還是多流派

天天要聞
2019-12-23

自從爐石這波平衡補丁實裝之後，酒館戰棋的環境迎來了很大的改變，其中有些人心態都崩了，且不說銅須的退環境以及寒光先知從2星又重新回到了3星，光是融合怪的”去世”就對一眾玩家們來說打擊實在是太大了。

那麼融合怪為什麼在之前的版本當中如此吃香呢？小編只能說你沒有體驗過這位打工仔的能力實在是太可惜了，因為不管是磁力、BUFF又或是劇毒等等屬性都可以貼給它，這對於前期來說有很大的收益，同時站場的融合怪數量與強度是直接掛鉤的。

但是現在不一樣了，融合怪沒了這就意味著你一旦確定某個路線發展之後，其他路線的buff怪就都沒有用，那麼酒館當中就會出現大量所謂的”廢牌”，這直接造成的後果就是你要花更多的費用去洗牌，從而導致整個節奏被拖慢。

那麼此前非常強勢的魚人流現在又怎麼樣了呢？整體上來說應該算是一次大幅度削弱，雖然寒光先知變成3費，發現魚也要5費，再加上融合怪被移除，現在魚人想要成型是真的有點難，現在的魚人如果不是天胡還可以走雞雜/套娃，畢竟沒人拿而且成型快，運氣好的話還是一樣能吃雞，如果這樣也不行，那對不起速八歡迎你。

所以我認為魚人流依然不會被放棄，畢竟寒光先知的調整讓前期的強度有所減弱，但是別忘了劇毒魚人只要在，吃雞還是有可能的，但是這一次的削弱倒是讓惡魔流順勢崛起了。

說起惡魔流，現在酒館戰棋當中吃雞的流派也發生了很大的變化，如果用百花齊放來形容一點也不過分，而惡魔流當中小紅小綠成了爸爸，葉師傅也挺起了腰杆子再也不怕劇毒了，畢竟現在只要養小紅小綠就可以了，這強度可不是一般的高。

那麼此次的平衡補丁大家是如何看待的呢？小編看到了一個比較慘的案例就是這位老哥，酒館戰棋已經玩了2000多把，但是這幾天從8000分掉到了4000分，因為融合怪被移除所以成不了體系的情況下分數更是”腰斬”。

雖然此次平衡各家都有說法，但是小編最後還是總結一下自己的看法，本次融合怪的移除其實最大的受益者就是館長，畢竟他給的迷你融合怪現在是中期當中唯一一個穩定的劇毒，這讓館長對惡魔有一定的剋制能力，所以平衡後的環境下並不是單一的流派崛起，更多的應該是各有所長。

PS：小編想問最後一個問題：”你們有沒有發現巨殼龍的出率被改了？”
