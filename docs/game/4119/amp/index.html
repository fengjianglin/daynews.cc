魔獸世界是一個充滿回憶和青春的遊戲，一代人在裡面投入了時間和精力，是一代人的記憶！當然隨著懷舊服的開啟很多的老玩家漸漸回歸，也有很多萌新的加入。使得魔獸這款遊戲熱度又上去了，但問題也漸漸變多了，很多事都在改變讓玩家們心寒，近日某網友爆料了某些玩家在野外互刷榮譽值！

事情的原因是在一個夜晚仲夏夜之星巡邏隊在午夜巡查，自己探子發現了互刷榮譽的毒瘤在貧瘠之地！大家都知道榮譽值是可以換裝備武器的，想獲得榮譽值就要殺死對方陣營的玩家，所以就導致了一些玩家走上不光彩的路，互刷！仲夏夜眾人優先看到的是太乙公會的巫師，被人發現後他們也有點慌了！還有一個璀璨星辰的官員法師讓他們自己組團去刷，後面又說好溝通。版本來仲夏夜眾人就覺得這樣的很不恥，又怎麼會去做呢（事情如下圖！）

後面仲夏夜的網友接著爆料了後面，「因為事情一直解決不了，後面部落叫來一個團殺了他們，之後就開始世界頻道刷屏洗地，污衊仲夏夜之星的成員是GZS的，是他們嫉妒紅了眼」這件事就變得越發的有趣了，後面雙方更是在NGA論壇進行互撕。先是刷榮譽值那邊有人跳出來說「散人夜裡野外沒有人，白天肝帝殺人，榮譽超過了你們工作室，來這裡屋面搗蛋，想讓你們接單」。後面雙方都說對面的是工作室的，想刷榮譽值換裝備升軍銜，後面都在指責潑髒水。那麼這次的互刷事件到底誰誰對誰錯呢？

對於這件事先不管誰才是工作室的，首先互刷榮譽值就很不光彩的事情了。小皮覺得肝可以刷榮譽值沒有人會反對，但互刷榮譽值就破壞了遊戲底線了，互刷的榮譽值還算榮譽嗎？先不是說仲夏夜之星眾人是不是工作室的人，難道互刷榮譽值就沒有錯嗎？也可能是遊戲風氣變了吧，以前的信仰漸漸的無關緊要了，就像刷榮譽值的玩家說的一樣，「遊戲！我們想怎麼就怎麼玩」

所以大家覺得這個刷榮譽值有沒有錯呢？到底是誰才是工作室，是嫉妒？是眼紅？大家可以在評論區留言說說自己的看法！
