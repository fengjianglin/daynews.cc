《新笑傲江湖》12月19號燃情公測！絕美國風手游再續經典巨作

2019-12-10 339條評論 1,635次閱讀 638人點贊

完美世界遊戲自研的新生代國風武俠手游《新笑傲江湖》12月19日10:00燃情公測。金庸正版授權，全劇情動畫呈現，構建風格化國風視覺，經典職業搭配，體驗全新劇情爽快PK！更有超人氣青年演員肖戰化身令狐沖與你開啟一段奇妙邂逅，結締江湖羈絆。

在場景部分，遠處的景色儘是山水，洛陽、福州、華山等地的獨特風光都被用水墨畫的風格所構築出來。遊戲大膽採用國風與流行相結合的美術風格，更首次展現出一個凝聚了中國傳統文化的匠心江湖。

在不同的情節中，每一個玩家可以按自己的想法在多線路的選擇中觸發新的故事。同時隨著主線劇情的推進，自由切換各種江湖身份，探索大地圖中各個區域的城鎮，也將體驗到或善或惡的新鮮故事與奇遇事件。

《新笑傲江湖》手游以第一人稱視角展開對故事脈絡的精妙解讀，伴隨著劇情的不斷演變，玩家將依次與曲非煙、林平之、令狐沖等不同經典人物相遇，親歷並參與他們跌宕起伏的傳奇生涯。這樣設計的好處在於能夠在不同階段以不同視角來更深入的觀察這些熟悉的人物，並以主人公的身份參與其中，成為江湖的一部分。

操作上採用了傳統的左搖桿右輪盤的操作方式，在非戰鬥狀態下，玩家右邊是輕功、坐騎、打坐、切換心法與普攻按鍵，按下普攻鍵即可切換到技能輪盤。

除了本來的職業技能外，玩家還可以通過參加各種活動獲得「絕學碎片」，集齊並完成相應任務後方可學習小說中的各種絕學，從《吸星大法》到《獨孤九劍》應有盡有，只要玩家勤奮一點，多參加活動早晚會把絕學收集齊全。

作為一款MMO遊戲，戰鬥系統的策略性與可操作性也足以滿足戰鬥玩家的需求。而對休閑黨來說，一些輕鬆的日常活動也能給他們帶來不少樂趣。

這些豐富的活動能讓不同的玩家都施展身手的地方，也能夠讓玩家們在這新江湖裡的生活不再那麼枯燥。踏入此間江湖，即可觸摸古韻國風的精髓與魅力，感受沉浸式新武俠體驗。
