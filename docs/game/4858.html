韋神成絕地求生國服獎金收入最高選手 4AM全員集體排進全球前十名

2019-12-04 0條評論 8次閱讀 0人點贊

摘要

在已經結束的絕地求生全球總決賽，代表國服出戰的4AM在最後一局表現拉閘最終遺憾的獲得第三名，雖然無緣奪冠但憑藉著這次打進前三的成績，目前4AM的隊員在絕地求生賽事獎金排行榜上已經全部躋身前十，成為全球收入最高的絕地求生職業選手之一。

在已經結束的絕地求生全球總決賽，代表國服出戰的4AM在最後一局表現拉閘最終遺憾的獲得第三名，雖然無緣奪冠但憑藉著這次打進前三的成績，目前4AM的隊員在絕地求生賽事獎金排行榜上已經全部躋身前十，成為全球收入最高的絕地求生職業選手之一。

　　在目前的絕地求生賽事獎金榜上，作為4AM的隊長，韋神目前已經累計獲得超過43萬元美元的獎金，成為目前國服獎金收入最高的選手，在全球排名第五。而隊友永遠、王欣和小鹿也分別獲得38萬美元左右的獎勵，在全球排名第六到第八名，這也可以看出這次全球總決賽的獎金還是相當高的。

　　曾經獲得世界冠軍的OMG小獅子也在榜上有名，憑藉著 20萬美元的收入在榜上排名第16。而目前在國際賽事多次取得好成績的GEN戰隊成為目前獎金榜的最大贏家，憑藉著這次拿到全球總冠軍，GEN的隊員已經累計拿到了64萬美元以上的獎金，並成為目前比賽收入最高的職業選手。

　　不過這僅僅只是比賽獎金，對於選手來說還有直播收入，也是真正收入的大頭。比如之前4AM的隊員孤存，在跟隊伍鬧翻之後，就曾經被爆直播收入高達千萬元，這可比比賽獎金高得多了，而像很多頂級職業選手，都能通過直播賺到更高的收入，這也使得一些選手在訓練之餘也要一直堅持直播，並從中獲得更高收入。
