盤古出不出末世？夢淚和麟羽看法不同，夢淚覺得出麟羽則認為不出

2019-12-07 0條評論 13次閱讀 0人點贊

摘要

陌陌聊王者，給您帶來不一樣的王者資訊！各位小夥伴，2019年KPL秋季賽季後賽大家都有去看嗎？相信看了的小夥伴都覺得非常過癮吧？這一屆的BO7比以往的多得多，BO7最精彩的就是第七局巔峰對決了。

陌陌聊王者，給您帶來不一樣的王者資訊！

各位小夥伴，2019年KPL秋季賽季後賽大家都有去看嗎？相信看了的小夥伴都覺得非常過癮吧？這一屆的BO7比以往的多得多，BO7最精彩的就是第七局巔峰對決了。

在這幾局的巔峰對決中，盤古這個藍領打野英雄成為了各種戰隊都非常喜歡的「香饃饃」，其他位置或許會不同，但是在打野位都會選擇盤古。

​

在總決賽開賽之前，AG超玩會的麟羽和夢淚進行了一次雙排，在他們的雙排中，麟羽使用最多的還是盤古，畢竟這個英雄肯定會在總決賽上場，麟羽也想時刻保持對這個英雄的手感度。

不過在兩個的排位中，他們對盤古的出裝有了一定的分歧，這個分歧就是盤古該不該出模式！夢淚個人認為出，而麟羽則剛好相反，認為不用出。

​

在KPL賽場，為了穩定，盤古基本上是不出末世這件裝備的。就拿麟羽自己來說吧，在KPL賽場上，麟羽的盤古出裝是打野刀（肉刀），攻速鞋，暗影戰斧，反甲，破軍，最後一件裝備在名刀和復活甲之間輪換。

畢竟KPL賽場的首要任務就是拿下比賽的勝利，只有穩，才能保證這局比賽能夠拿下來。因此麟羽個人覺得出末世沒有用，打不到人。

​

我們先來看看末世的屬性，末世能夠增加攻擊速度，也有吸血的效果，被動是能夠附加敵人當前生命值8%的物理傷害。

夢淚認為，出末世能夠將盤古的傷害最大化的展示出來，雖然說很難切到對方的射手，但是對敵方的前排傷害也是非常大的。當然如果能夠粘住C位，基本就是三四下就能解決了。

​

不過出末世只是夢淚的建議，至於麟羽會不會在總決賽上出，那就要看他個人的決定了。除了盤古的出裝，夢淚還給了麟羽曜這個英雄的一些出裝意見。

夢淚給麟羽看了兩局曜不出暗影戰斧而是率先出碎星錘的操作，效果還是非常不錯的。在麟羽使用曜的時候，他也嘗試了曜出碎星錘的打法，看來對於夢淚的一些建議，作為小師弟的麟羽還是聽得進去，並且會去嘗試的。

​

各位小夥伴，大家覺得盤古這個英雄在KPL賽場上是否合適出末世這件裝備呢？在平時排位賽中，是經常看到有玩家還出的，但是畢竟KPL賽場不同排位賽。

最後也希望，AG超玩會能夠在總決賽中戰勝自己從來沒有贏過的對手QGhappy，不僅終結對QG的連敗記錄，同時也完成自己的夢想！
