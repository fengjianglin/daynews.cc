游漫談：遊戲王Sevens公布，畫風玩法全線崩塌，核心玩家慘遭拋棄

2019-12-24 1,371條評論 4,570次閱讀 3200人點贊

《遊戲王》系列自從第一部《遊戲王DM》發行之後，就引起了動漫界和遊戲界的軒然大波。這個自帶遊戲玩法系統的漫畫經過高橋和希的設計與創造、科樂美對其實體化與遊戲化的升華再創作，成為了全世界首屈一指的實體集換式卡牌遊戲。在其巔峰期間，索尼PSP時期的《卡片力量》系列幾乎成為每一個卡牌遊戲玩家的最佳遊戲，即便是現在仍然有大量玩家活躍其中。

然而好景不長，《遊戲王》無論是畫風、劇情、遊戲平衡和設計都一代不如一代。如果以初代《遊戲王DM》的人氣為參考依據，那直至第六代《遊戲王vrains》的人氣已經跌至不足10%。在加入同調、超量、link等越來越繁瑣的玩法後，不少玩家無奈退坑，現在還在圈子中的玩家基本全部是核心資深玩家。卡池深度逐漸加深就像一把雙刃劍，一方面為遊戲玩法增添了更多的可能性和趣味，另一方面無形中陡增的難度時刻在勸退萌新玩家。

毫不誇張的說，無論是動漫亦或是電子遊戲，《遊戲王》的輝煌早已不在，已經窮途末路的IP很難再被挽救回來。然而「禍不單行」，《遊戲王》第七部新作《遊戲王Sevens》即將發行的消息再次為其跌入深淵助了一份力。完全崩塌的畫風和人設，一下回到解放前的「村玩法」都讓玩家們無力吐槽。

首先從這份宣傳海報中我們不難看出，《遊戲王Sevens》的主角「王道游我」徹底成為子供向畫風的犧牲品，從初代開始鋪墊的暗黑、硬核且略帶顏藝的畫風人設形象從此灰飛煙滅。取而代之的是一名染著黃毛，拿決鬥盤有種玩「戰鬥陀螺」既視感的小毛孩兒。有一說一，單看畫風確實與《戰鬥陀螺》中的木之宮龍有些相似，甚至還不如木之宮龍來的帥氣。

除此之外，劇情設定上《遊戲王Sevens》以未來都市「豪派市」為中心展開，從學校到都市的遊戲規則全部由全新企業「豪派社」管理，而主角「王道游我」則想打破這個默認的規則，堅持相信自己的發明，創作出Road：Rush Duel奮鬥！這份設定雖然沒有表明過多細節，但是字裡行間透露的中二氣息讓喜愛遊戲王的玩家當真無以言表，不禁想念初代遊戲的埃及故事、二代的黑暗遊戲、三代的地縛神……唉。

最重要的是，該作將推出全新卡牌玩法「rush duel」！決鬥盤上只有三個卡片位置，直接將遊戲王打回村規原形，目前可知的規則如下：遊戲使用rush duel專用卡片；1回合之內可以通常召喚無數次；每回合抽卡補充到5張手卡為止。

不得不說這個規則真是充滿了想像力，完全拋棄掉所有老玩家企圖重新開拓市場，將遊戲重新放在新一代小朋友身上，卻不知道在《遊戲王》IP上究竟是那群玩家在苦苦支撐著它。最後不得不說上一句：科樂美，真有你的。

關注游漫談，御三家攜手Steam，遊戲資訊搶先看。您的點贊與評論是對老談最大的支持。
