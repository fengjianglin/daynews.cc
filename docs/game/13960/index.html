PDD直言UZI巔峰期很強 另外想打職業起碼要國服前十

2019-12-12 2,535條評論 3,712次閱讀 1969人點贊

隨著轉會期進入到白熱化階段，大家最關注的應該就是各大戰隊在夏季賽的陣容情況。不過距離轉會期結束還有一段時間，所以大家只需靜靜等待最後的結果就可以了。不過要說的是，每年的轉會期對於一個職業選手的生涯是非常重要的，因為他決定著你在明年的努力能否得到回報。

諸如Doinb這樣的選手，他在LPL拼搏了很多年，在去年更是萌生了退役的想法，但是好在在周圍人的影響下，他堅持下來了，拿到了世界冠軍。但是對於大多職業選手來說，想要打出好成績真的很不容易，而且能夠在職業比賽中脫穎而出的人真的是太少了。

近日，PDD在直播的過程中，有很多粉絲問到打職業的情況，PDD也是給粉絲們進行了一波回復。畢竟作為第一批的職業選手，還一手創立了戰隊的PDD是最有發言權的。不過從PDD的口氣來看，他個人並不是很支持大家都去選擇打職業，因為就現在的電競圈來看，如果單純的只是王者段位就很難衝上職業舞台，如果能夠衝進國服前十，還有很大的機會。

因為就現在峽谷之巔的情況來看，如果達不到前十的話，就只能去青訓隊裡面當個混子，當然這沒什麼用處。而且PDD還直言，如果衝進前十，哪怕不去打職業，還會有很多其他的出路，這樣不香嗎?

而且對於職業聯賽，PDD也是直言，現在打職業的選手都是非常猛的，但是這也是一個非常需要運氣遊戲，就UZI來說，他個人實力在巔峰期的時候是非常強的，很多人都覺得他那個實力應該拿個S賽的冠軍。但是至今他仍然沒有拿到。所以說，這是五個人的遊戲，能不能有一個好的隊友也是很重要的。

所以說，打職業這條路並非有想的那麼簡單，哪怕有很強大的實力，如果不能拿到好的成績，也是註定碌碌無為。而且能夠站到LPL舞台上的人更是鳳毛麟角，打職業的人那麼多，LPL轉來轉去還是那麼多人，新人想要加入都是很不容易的。

對於打職業這個事情，PDD也是說出了一些很直觀的感受，那就是除非你真的非常有天賦，這樣最好再去打職業，不然還是別考慮這個行業了。因為現在想打職業的人太多了，如果沒有天賦，那麼你在青訓的隊伍當中就很難出線。

其實PDD的這番話，對於現在的很多年輕人來說，也是一種警鐘，雖然職業舞台上的選手們看起來星光閃耀，但是他們背後所經歷的也是別人難以想像的。如果各位召喚師真的有打職業這個想法，在去做之前，一定要好好考慮，自己到底適不適合這個行業。
