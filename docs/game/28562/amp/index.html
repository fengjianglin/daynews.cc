熱血傳奇：沉默嘟嘟的八大頂級武器，大道最強，玄冰最酷

天天要聞
2019-12-23

大家好，我是小逗說傳奇，對於沉默嘟嘟我想很多玩家都玩過，今天呢給大家介紹一下，沉迷嘟嘟中的九大頂級武器。

第一把：九彩分神刺

九彩是高級武器中的第一把，等級為64級，這邊武器的顏色跟名字是非常符合的，因為他無時無刻不在變色，因此被稱之為「九彩分神刺」

第二把：永恆玄冰

玄冰是67的武器，這把武器就像是冰打造的一樣，外觀非常酷，可以說他是沉默嘟嘟中最酷的一把武器了，其中衣服的話，也是異常的炫酷喲。

第三把：毒龍之身

毒龍是70級的武器，第一次看到毒龍武器的時候，我以為他是衣服呢，畢竟毒龍之身這個名字感覺很像衣服啊，毒龍可以在地獄副本裡面出，爆率還算不錯的喲。

第四把：魅影之刃

魅影之刃是一把非常經典的武器，這個武器從我本沉默開始就有了，玩家也是非常熟悉的，在沉默嘟嘟中，魅影之刃屬於72級的裝備，也算是頂級裝備中的入門裝了。

第五把：五嶽游龍

第一次沙城的獎勵中，就會有一件游龍，當然，給的是隨機的，如果運氣好的話，就能隨機到這個頂級的武器-五嶽游龍，這個武器的外觀非常炫酷，金黃色劍身讓看起來特別霸氣喲。

第六把：盤古大帝

盤古大帝是76級的武器，這把武器外形有點想燃燒的AK47，我個人覺得這個武器有點搓，不過其屬性還是非常給力的。

第七把：次元須彌斬

次元武器跟怒斬是一個樣的，手持中間部位，二端是劍刃，我個人不太喜歡這個武器，因為我喜歡玩戰士，帶這個武器總感覺砍不出傷害，但是這個武器卻是次終極裝備，很稀有。

第八把：大道無形斬

這把武器是85級的武器，也是沉默嘟嘟的終極武器，目前我還是沒有獲取，所以我就找到一個大佬，然後偷偷摸摸的截圖一張。

大道武器的外形那是非常炫酷的，也是非常霸氣的，這武器感覺才是戰士的武器啊，次元太不爽了。

這八把武器的就是沉默嘟嘟中的頂級武器了，我個人最喜歡的外形還是玄冰的，不知道你們喜歡哪一把武器的外形呢？
