夢幻西遊：我究竟是買了個什麼角色？全服性質的成就都讓他拿了

2019-12-11 607條評論 3,229次閱讀 661人點贊

我的世界因為有你而精彩，喜歡的可以關注我！

真的非常佩服當初開發出祥瑞和錦衣系統的那位策劃，因為祥瑞和錦衣系統的上線給官方帶來了非常大的收益，玩家從此之後也走上了外觀攀比的道路，而且現在限量祥瑞和錦衣的種類越來越多，刷任務的時候沒有限量的祥瑞和錦衣想要混隊都變得非常困難了。

No1.我究竟是買了個什麼角色？全服性質的成就都讓他拿了

在夢幻中有很多限量祥瑞是真的已經絕版了，在夢幻中並沒有相應的祥瑞禮包來獲得，玩家想要獲得這些限量祥瑞有兩種途徑，一種是直接在藏寶閣上購買擁有這種祥瑞的角色，另一種就是使勁的刷成就，當成就點達到一定限度的時候，就能夠在成就寶盒中購買十種限量的祥瑞，這也是很多玩家拚命刷成就的原因。

有很多玩家為了能夠快速的刷成就會將自己的號交給別人來刷成就，這些專門接成就單子的人在接單的時候也有自己的規章，那就是對於別人家過的二手單子自己不做，因為成就黨有一套專屬的刷成就套路，能夠以極低的成本獲得某些成就，但是在夢幻中有很多種成就是沒辦法獲得的，因為整個伺服器只允許一個人獲得此類成就。

這位玩家最近就買到了一個這樣的角色，從他的功成名就中能夠明顯的看到夢幻中大多數難度副本的首次通關都是由他完成的，而且在本伺服器很多高等級的寶石成就也是由他完成的，這位玩家在購入這個角色之後就想知道這個角色的主人之前是什麼樣的存在，如此多的伺服器成就讓他的號達到了什麼樣的位置？又是因為什麼讓這樣的一位神豪脫坑的？

No2.這樣的妙華天女只能用奇蹟來形容

妙華天女是夢幻中唯一一隻沒有變異現象的召喚獸，因為它的造型和其稀有程度讓很多化聖大佬非常的追求不過因為她產出的數量比較少，而且沒有必帶技能多技能的妙華天女的數量也非常少，像這樣的一隻12技能的妙華天女就是最頂級的存在，而且這隻妙華天女還達到了1.3的成長，技能方面更是擁有三個特殊技能，這三個特殊技能中的凝光練彩是煉妖合出來之後就存在的，至於另外的善惡有報和壁壘破積有可能是打書完成之後又拍上去的。
