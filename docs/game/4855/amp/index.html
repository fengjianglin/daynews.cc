2級GANK一旦出問題死亡就會大量喪失節奏

打野為何會實現等級的壓制

通常在高分段來講，打野雙方水平一致的話，會就雙方策略的不同，差距1個等級，比如GANK思路與刷野思路的區別，GANK如果沒成功，就會被壓制1個等級。

但是這件事情如果放在雙方打野有本質水平差的時候，等級會擴大到2-4級，因為現在等級追趕機制沒了（即當等級小於平均水平時，刷野會獲得額外經驗）

所以，現在的打野對於刷野與GANK的平衡非常的重要，雖然這種事情我也會講很多遍，但是，這其實是一件很難理解的問題。

等級領先是打野最大的優勢，可以儘可能忽視隊友作用

刷野是一件動態邏輯的事情，你要有地圖思維

就好像思考一個地方的路線，腦子裡會出現動態的地圖，然後根據地圖規划行走的路線

而如果說到打野，你不僅要有地圖，還要有路線，甚至還要規劃刷新的節奏以及GANK的節奏，這個實時變化程度是比較高的，所以線上基本上都不懂打野的節奏，尤其小丑這種節奏本身就和大多數打野不一樣的英雄。

3+2也可以從GANK成功轉變為3+4，而且會變成領先2級

從開局的刷野講一講野區極限效率與GANK平衡

比如F6-紅-石頭的極限效率速刷設計，會讓你領先等級，類似這樣的極限效率思路是打野取得開局優勢的關鍵

這也是為什麼紅-藍-蛤蟆接著去GANK基本上就是純GANK思路而不是刷野思路了，因為野區不集中，你會浪費大量的時間在走動上而做不到極限效率。

極限效率通常是以2個為一組起步的，3個就代表多了。

比如極限效率刷完紅區3組，再極限效率藍區4組，這就是開局最好的純刷線路，而如果你採取GANK思路，你就必須刪掉起碼2組野怪，採用3-2的組合，來去獲得GANK的時間，這中間還要涉及到一次河蟹的爭奪博弈策略上。

但是，你不應該以刷多少組野怪為依據，而是以等級為依據，你的目標是到達4級，不論你設計什麼路線，你的目標都是這個。

這也是為什麼會出現3-4的極限速刷組合和3-GANK-GANK-2的（後面統稱3-2）組合，純刷野3-4到達4級，而GANK因為會蹭到線上的經驗，就只需要3-2的組合即可，如果GANK成功，拿到人頭3-2的經濟就會優於3-4 100，不成功則會少200，助攻打平，基本上就是這樣

低分段如果三路壓線，高手就會選擇3-4，保證自身的等級和經濟。

而高分段，即使沒有機會，也會選擇3-2，因為3-2會給隊友帶去優勢，加快遊戲的節奏，位置自身隊伍的穩定性。

而不會的打野呢，就可以根本沒有設計路線，或者會出現路線被打亂就不知如何繼續下去的問題，結果就變成了逛街型的1-2-GANK-博弈失敗，或者3-GANK-博弈失敗，就變成了被壓制1級，差距就從這裡開始產生了。

突然發現這樣發展會很長，我們下篇再敘。
