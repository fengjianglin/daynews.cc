「Steam推薦」魔法學校養成+卡牌策略對戰遊戲：Rocwood Academy

2019-12-16 2,145條評論 3,638次閱讀 733人點贊

Rocwood Academy（羅克伍德學院）是一款校園養成和卡牌策略對戰結合遊戲，玩家可以扮演一名學生，從一年級入學開始，結交朋友，學習魔法，並揭開學校後花園下面的秘密。本作由美國開發商Openlab Games出品

遊戲的故事背景設定在20世紀初的美國，有一個與現實社會隔離的魔法世界，玩家將扮演魔法學校「羅克伍德學院」的一年級學生展開自己的魔法師修行生活，學習過程中你發現了有學生消失在後花園的傳聞，解開這個秘密的重任意外落到了你身上

本作中校園養成是核心玩法之一，作為一名新生，你會面臨很多選擇，和誰交友？學哪種法術？選擇哪個老師？你做出的決定將會影響到故事的發展，每個人都會有完全不同的遊戲體驗

收集並組建魔法卡組是遊戲的戰鬥方式，你可以收集多種多樣的魔法物品，還能從其他NPC手中學會法術，組建起符合玩家風格的卡牌組，與隨時面臨的各種神秘生物進行戰鬥

遊戲在Steam售價15元，支持Windows，暫時沒有簡體中文
