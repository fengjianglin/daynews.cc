全明星六場雲頂之弈拿第一的陣容！原來神超早就預料到了一切？

2019-12-09 0條評論 9次閱讀 0人點贊

摘要

伴隨著2019年全明星賽的結束，相信不少英雄聯盟玩家們還對此次的表演賽充滿了意猶未盡的感覺。與之前全明星賽不同的是，在本次的全明星賽中還新增了一個雲頂之弈的模式。這六場比賽不僅讓許多雲頂之弈的愛好者們大飽眼福，更是讓我們了解到了當前版本中能夠拿到第一的陣容。

伴隨著2019年全明星賽的結束，相信不少英雄聯盟玩家們還對此次的表演賽充滿了意猶未盡的感覺。與之前全明星賽不同的是，在本次的全明星賽中還新增了一個雲頂之弈的模式。這六場比賽不僅讓許多雲頂之弈的愛好者們大飽眼福，更是讓我們了解到了當前版本中能夠拿到第一的陣容。那麼，也讓我們一起來回顧一下，這六場比賽分別是哪些陣容拿到了第一吧！

第一場：四水晶掠食毒

關於掠食毒這套陣容，相信不少雲頂之弈的玩家還是非常熟悉的。不過主播Becca則在構建這套陣容的時候拿到了水晶元素的拉克絲，一舉將這套陣容構建成為了四水晶的掠食毒。最終，獲得了這一場比賽的勝利。

第二場：四影三召喚

四影三召喚也是當前版本非常主流的陣容之一。雖然Ceros並沒有給主C千珏拿到熾天使之擁，不過拿到了劍轉與劍聖組成雙劍士也是很好地增加了攻速，算是變相增加了回藍的功能吧。

第三場：雲霄劍刺

正如同當時官方解說所說的那樣，雲霄劍刺的確是我們國服使用的比較多。而這一場比賽中，也是由我們國內的主播小澤所拿下。當然了，這套陣容最重要的還是夢魘的劍轉，在沒有劍轉的情況之下還去玩這套陣容的話，威力還是大打折扣的。

第四場：煉獄法海召喚劫

終於是到了很多雲頂之弈玩家們所熟悉的煉獄法海了。不過最終，因為裝備非常符合劫，所以這位主播最終使用劫來取代了約里克，並以一波連勝拿到了最後的勝利。

第五場：四影雙劍雙秘

Bwipo可謂是全明星賽中的最大贏家，而在雲頂之弈中，他的陣容也是非常奇怪。因為一般情況下，四影還是搭配三召喚比較合理，但是Bwipo卻證明了「沒有最胡，只有更胡」，在神裝三星千珏、三星塞恩以及兩星劍聖七人口的情況之下拿到了第一。

第六場：掠食毒

在沒有水晶的情況之下，空門憑藉著三星鱷魚、三星大嘴以及三星蒙多很好地鎖住了血量。在最後一次對決中，擁有著二星劫並能夠組成三刺的情況之下，因為對手綠叉的存在，最終將劫換下，捨棄三刺使用兩沙漠最終取得了勝利。當然了，這套陣容我們也可以理解為是掠食毒的變化版。

從這六場拿第一的陣容來看，也的的確確是我們所熟知當前版本最容易上分四套的陣容。他們分別是雲霄劍刺、煉獄法海召喚、掠食毒以及四影三召喚。但事實上，在9.23版本剛上線不久，神超就已經分別出過雲霄劍刺、煉獄法海召喚以及掠食毒這三套非常詳細的攻略。甚至四影三召喚這套陣容，神超也在排位中完成過滿血拿第一的「名場面」。

所以，本次全明星賽神超沒有參加還是讓人覺得有些遺憾的。畢竟，相信不少雲頂之弈的愛好者們還等待著神超能夠在全明星賽中表演自己的「法術」吧？

各位小夥伴們，你們覺得呢？
