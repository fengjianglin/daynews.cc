論三國志中的張遼和關羽以及陣容搭配的重要性

2019-12-11 2,283條評論 4,078次閱讀 1710人點贊

論三國志中的張遼和關羽以及陣容搭配的重要性

大家好，我是愛玩遊戲的夏目，上期咱們聊到曹操的五子良將和劉備的五虎上將誰更厲害，這期我們來講一下在三國志這款遊戲中，張遼和關羽的一些信息以及陣容搭配的重要性，如果你也喜歡遊戲，可以關注我，跟我一起分享快樂的遊戲時光。

在三國志中，張遼的屬性還是不錯的

張遼基本屬性

武有85，統有73，謀有67，各項指標都很平均，就和在演義一樣，張遼是個全面發展的將軍，在哪裡都是一樣

再看關羽

關羽基本屬性

武有98，統有82，謀有75，二爺的武力值是毋庸置疑的高，就跟演義里一樣，面屏的MVP可不是開玩笑的，這也就證明了上一期提到的張遼和關羽打一場的話誰會贏了，基礎屬性就已經決定最後的結果了。

第二天玩這個遊戲，我發現每一個英雄都有著自己的特色，在爭奪城池的時候，如果我們能夠搭配好一個陣容，讓每一個英雄的優勢都能發揮出來，那麼一定是可以事半功倍的。目前，英雄屬性有qiang兵、弓兵、騎兵、謀士、戰車這五種

英雄屬性

不管我們玩什麼遊戲，前排是肉的這個定律都不會變，不然還沒打一會就扛不住都歇菜了，那就沒意思了，所以在這裡的肉就是由戰車英雄來擔當，我現在有的戰車英雄比較厲害的就是馬超了

馬超（圖片來自三國志）

然後就是中排的輸出了，在這裡就是qiang兵了，一般戰士類的英雄會多一些，我有的就是關平了

關平（圖片來自三國志）

今天也是剛升到四星（但是朋友說關平不厲害，我好不容易升上去的，差點哭暈在廁所）

接下來就是最重要的後排輸出了，就是弓兵和騎兵了，弓兵我有個三星的貂蟬

貂蟬（圖片來自三國志）

騎兵的話我就只有一個張遼了，一般我都不用張遼，用蔡文姬代替，蔡文姬也是弓兵，輸出的話也是夠用的

張遼（圖片來自三國志）

最後就是輔助了，也就是謀士，奶媽，加血的，我有一個曹植

曹植（圖片來自三國志）

沒錯，就是那個寫了七步詩為自己謀得一條生路的可悲文人。

現在爭奪城池的時候，一般這幾個英雄的組合基本都是逢關必過的，陣容我覺得肉的話用一個就行，其他的看星級，如果有兩個三星或者四星以上的弓兵，那麼捨棄掉一個兩星的騎兵也未嘗不可，要把資源最大化，1+1>3也不是沒可能，輔助還是要有的，玩過遊戲的玩家都知道一個奶量足的奶媽是多麼的變態，簡直就是隊友的再生復活甲啊；qiang兵的話，選擇一個技能能給隊友帶來增益的就最好了，既有輸出又能給團隊帶來效益。像這樣的一個陣容我相信不管是攻哪個城池都一定是手到擒來。

剛玩這遊戲沒幾天，還有好多英雄都沒有，如果有對陣容方面有獨特見解的玩家可以給我留言評論，大家一起探討，看看什麼樣的陣容最厲害，最奇葩，最有可玩性，最後祝大家在遊戲中玩的開心，遊戲嘛，開心最重要，我也要去玩啦，下一期見。
