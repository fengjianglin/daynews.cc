王者模擬戰奶坦陣容，青春之舞是關鍵，長大的蔡文姬真猛

2019-12-10 924條評論 3,486次閱讀 617人點贊

本周以及上周的模擬戰周報中經常出現了一個高端局不常存在的陣容就是六坦克，相信大家萌新的時候都玩過稷下坦克，而這套陣容配合青春之舞在頂尖玩家手裡變成了榮耀局的吃雞陣容！

這裡強調一下開局至少拿到一個青春之舞天賦再去玩坦克。

一、棋子選擇

1、主選

一費卡一張（劉禪）；二費卡兩張（程咬金、鍾無艷）；三費卡一張（蔡文姬）；四費卡兩張（孫策、廉頗）；五費卡兩張（蘇烈、孫臏）

2、 備選

夏侯惇、老夫子、墨子、鍾馗等

這套陣容的核心特色就是靠坦克肉死對面，蔡文姬作為奶媽，鍾無艷是輸出之一。

二、羈絆效果

推薦六坦四稷下兩輔

三、裝備天賦

1、裝備選擇

出肉即可，都拿肉裝

推薦神裝

鍾無艷（爆裂之甲/紅蓮斗篷/極寒風暴、血手/霸者重裝、魔女斗篷/不詳徵兆）

孫策/程咬金（魔女斗篷/不詳徵兆、爆裂之甲/反傷之甲/紅蓮斗篷/極寒風暴、霸者重裝/血手）

2、 天賦選擇

推薦天賦：青春之舞、荊棘之盾、致命魔術

四、運營思路

1、前期（1-10回合）

長安開局：前由於我們陣容坦克中有兩張是長安也就是程咬金，鍾馗所以我們可以三長安開局三人口。前期注意用李元芳拿連勝，如果順利我們五人口可以上五長安，這裡注意我們三長安開局的話經濟比較多，所以我們可以留意程咬金、鍾無艷把這兩個刷到三，鍾馗太貴不需要三級，夏侯惇三星也不是很強，然後我們六人口轉型四坦三長安就行，如果期間拿到青春之舞，我們也不一定要上畢竟陣容沒成型強度不高。

坦克開局：正常情況下我們前期白嫖到的坦克比較多我們可以組成兩坦兩稷下的羈絆四人口，也就是鍾無艷＋一個稷下＋一個坦克，也比較好過渡，把裝備給鍾無艷就行，遇到鍾馗不要，夏侯惇、程咬金、蔡文姬、劉禪可以留。

2、 中期（11-17回合）

中期我們六人口可以上四坦外掛一個蔡文姬一個坦克，七人口就可以六坦外掛蔡文姬。中期我們如果有機會讓低費坦克三星可以適當搜一搜，畢竟我們陣容的核心輸出是鍾無艷，這裡注意中期的裝備十分關鍵，我們都拿肉裝即可，比如新出的反傷刺甲、暴烈之甲等，以後可以拿復活甲和霸者，鍾無艷也可以選擇暴力輸出破甲弓、破軍這些裝備。

3、後期

後期我們八人口成型組成六坦兩輔的羈絆外掛蔡文姬二星。後期我們低費卡可以三星的話就留，比如程咬金、夏侯惇沒機會三星我們可以換高質量的二星蘇烈、孫策代替，這裡注意一下蔡文姬是必須追三的看到一定要拿。廉頗、孫臏、劉禪二星即可，其他有機會都可以追三。

三星優先順序：鍾無艷＝蔡文姬＞其他坦克

大後期九人口推薦：

我們九人口可以上四稷下進一步提升戰鬥力外掛一個老夫子墨子就可以，也可以外掛一個二星呂布或者武則天。打法師陣容我們可以用四輔助，只要拿到一個令旗即可。

推薦陣容評價： ☆☆☆☆（滿星為5顆）

A、 前三指數：☆☆☆

原因：坦克沒有青春之舞的話其實陣容強度很低，我覺得目前版本這套陣容還是娛樂為主，但拿到兩個以上的青春之舞還是很強的。

B、吃雞指數：☆☆☆☆

原因：後期的話只要不是打扶桑陣容一般都是拼質量，我覺得五五開。
