《饑荒》瞬移法杖、喚星法杖、銩礦王冠等八種遠古物品的介紹

2019-12-22 1,750條評論 3,708次閱讀 1492人點贊

哈嘍，大家好，今天給大家介紹一下之前視頻里沒有介紹過的遠古物品，這些物品只能通過靠近遠古偽科學基地才能合成。

眼球雷射塔

眼球雷射塔需要1個犀牛角、1個鹿角怪眼球和5個銩礦石合成。

當眼球雷射塔被放置後，它會攻擊除了暗影生物外，所有對玩家有仇恨或玩家想要攻擊的生物。

眼球雷射塔擁有1000點生命值，每次能造成65點的傷害，每秒能回復12點的生命值。玩家可以使用一些非食用性的治療物品對它進行治療。

眼球雷射塔不能使用鎚子來拆除，在被其它生物殺死時，它就會消失。

瞬移法杖

瞬移法杖需要2個橙寶石、1個步行手杖、2個噩夢燃料合成。

當玩家拿著它行走時，可以提升玩家25%的移動速度。使用它可以讓玩家閃現到某個指定的位置，可以用來躲過敵對生物的追殺，或者用來穿過海灣和深淵。每次使用，玩家會損失15點的理智。

製造瞬移法杖時，最好使用建造護符，這樣可以節約珍貴的橙寶石。

銩礦皇冠

銩礦皇冠需要4個噩夢燃料和4個銩礦石合成。

它能夠吸收90%的物理傷害，單機版擁有1200點的耐久度，聯機版擁有840點的耐久度。受到攻擊時，它有33%的機率生成一個免疫傷害的力場護盾，代價是降低少量的理智。

護盾會持續4秒，之後9秒內不會再次發生，不能免疫火焰傷害，但可以防止被擊暈。

銩礦護甲

銩礦護甲需要6個銩礦石和4個噩夢燃料合成。

它能夠吸收90%的物理傷害，單機版擁有1800點的耐久度，聯機版則擁有1260點的耐久度，每分鐘能夠恢復3.3的理智。

銩礦棒

銩礦棒需要3個活木、4個銩礦石和4個噩夢燃料合成。

它能夠使用150次，每次攻擊在造成59.5傷害的同時，還能增加10%的移動速度，還有20%的幾率生成一個持續9秒的暗影觸手。

斧鎬

斧鎬需要2個銩礦石、1個金斧頭和1個金十字鎬合成。

它既可以用來砍樹，也可以用來採礦石，是個二合一的工具，還能提升33%的效率，比普通工具要快一些。

喚星法杖

喚星法杖需要2個黃寶石、2個活木和4個噩夢燃料合成，使用它可以召喚一個矮星，每次使用會降低20的理智。

單機版中矮星能夠持續2分鐘，熱量相當於三級營火，能夠烹飪食物，靠近它每分鐘還能恢復25的理智。在聯機版中，矮星能夠持續28分鐘。

在聯機版中，還可以通過月亮祭壇把喚星法杖轉換成喚月法杖，這個我在後面的視頻中會講解。

魔光護符

魔光護符需要1個黃寶石、2個銩礦石和3個噩夢燃料合成。

​它能夠提供20%的移動速度，這個移速加成可與其它裝備疊加。它每分鐘能增加2點理智，還能提供防止查理攻擊的少量光源。

魔光護符能夠使用8分鐘，1個噩夢燃料可以補充3分鐘的耐久度，當耐久度降到零時它就會消失。
