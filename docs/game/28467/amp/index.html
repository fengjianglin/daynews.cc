DNF暗帝VS天帝：百分比與固傷職業的碰撞，淺談版本末期入手指南

天天要聞
2019-12-23

鬼劍士這個角色在DNF里是非常受廣大勇士喜愛的，不管是男鬼劍士還是女鬼劍士，每個職業本身都擁有著很強的「職業個性」。暗帝是女鬼劍士里的純百分比職業，使用「暗」的魔法力量來攻擊敵人。而天帝則是男鬼劍士裡面的典型，純固傷職業並使用光的魔法來攻擊敵人。這一光一暗、一百分比一固傷的兩個個性完全相反的職業，常常成為玩家們喜歡對比的「cp」。在新版本即將開放的前夕，我們本期來一起淺談分析一下暗帝與天帝的入手指南。

暗帝的職業特性分析

暗帝雖然是純百分比輸出職業，但她也是女鬼劍士里唯一一位「25仔」。很多玩家入手暗帝都會進入一種誤區，他們會抱怨為什麼同樣打造下的暗帝，傷害卻打不過同為百分比輸出的劍魔！這就是對暗帝這個職業的特性不了解所致，暗帝的特點是「強控制」、「提升團隊傷害」、「補刀主C傷害」這三點，她的特性決定著這個職業不可能如同純C一樣打出全隊最高的DPS。

PS：目前DNF里的「25仔」還有另外一個名字——「34仔」，因為技能改版之後，這些「25C」對於團隊的傷害提升也從25%增加到了34%。

暗帝技能與打造誤區

【技能取捨分析】大家要記住一句話：DNF里沒有最強的技能加點，只有最適合的技能加點！DNF里的技能現在可以讓所有玩家免費重置加點，這個設定其實就是告訴我們一個事實，那就是DNF里的技能變化需要隨著不同的戰鬥環境，選擇最適合的技能組合模式。所以如果你們想要從本文中得到一種「完美技能加點」，那是不可能的。我們只能分析技能取捨的思路，從思路中大家自己選擇適合自己的加點模式。

控制技能和輸出技能我們要有所取捨，比如「暗影纏襲」和「暗影漩渦」，往往我們只需要1級的纏襲帶來的控制效果即可，多餘富足的SP點數可以給暗影漩渦增加持續輸出。

中段技能就比較明朗了，「控制型技能」依然建議大家選擇1級達到控制目的即可，而「輸出型技能」或者技能形態好的技能要優先點滿。這裡需要注意的是，什麼叫做好的技能形態？好的技能形態並不是一成不變的，而是根據目前遊戲版本環境隨動的！目前大環境下，BOSS幾乎都存在有限時間的破防時間讓我們輸出，在這麼短暫的輸出時間內，瞬間爆發和脫手輸出技能就可以被定義為「優秀技能形態」。所以我們技能加點的原則要遵循「技能形態優先」！

值得一提的是，暗帝的二次覺醒技能還是十分優秀的，技能形態優質，技能百分比加成都屬於全職業上流！雖然雙覺醒技能的形態略顯拖沓，但作為高百分比的加成技能，依然推薦毫不猶豫的加滿！

TP技能根據自己的技能樹來選擇，但要遵循：優先技能形態優質的高等級技能。

【打造誤區要知道】暗帝不是純C，不要去嘗試追求這個職業的「傷害上限」！裝備打造的優先順序推薦為： 強化+12武器優先順序最高、增幅推薦6~10都可以，因為作為「25C」，暗帝本身對於增幅的門檻就比較低，加上新版本即將開放，高增幅已經不適合現在這個階段。附魔選擇至關重要，但考慮到目前已經進入版本末期，選擇雙屬強20的寶珠即可。趁著史詩之路活動，可以迅速給自己的暗帝拿到一套「版本畢業裝備」，不需要考慮本命甲與否，因為新版本開放後都會被淘汰。

天帝的職業特性分析

天帝這個職業玩家基數巨大，其主要原因還是因為這個職業的特性所致。天帝幾乎是DNF里的一種「全能萬金油職業」，要控制有控制，完美的技能循環讓他也兼顧了持續輸出與瞬間爆發輸出！但也請大家記住，天帝與暗帝一樣，同屬於「25C」，這樣的基本設定下決定了天帝不可能像純輸出職業那樣打出傲視全隊的傷害。但天帝又有與暗帝不同的地方，天帝屬於純固傷職業職業成長線性屬於「前段高中段平穩後段低」的特點，天帝的門檻打造傷害收益十分明顯，但完美打造下傷害成長就有所不足。

天帝的技能打造與誤區

【技能取捨分析】天帝的技能相比於其他職業，爭議性是很低的，玩家對於技能樹的選擇無非在「鬼印珠or不動明王陣」、「無雙波and邪光波動陣」、「大火滿不滿」這3個方面。本文我也無法給出大家這三個爭議問題的「正確答案」，因為每一種技能combo組合都有其優缺點，只能根據大家自己的操控感覺來做出適合自己的優化。

鬼印珠與不動明王陣的優劣分析：鬼印珠可以讓我們的技能循環更加優質，並且可以「粘住BOSS」給團隊提供更加穩定的「破防控制」，但傷害可以用「毛毛雨」來形容。而不動明王陣雖然傷害更高，但需要消耗掉更多的SP點數，並且技能傷害成長線性很差，技能形態也不好，不過卻是阿修羅的招牌技能。

在「無雙波」與「邪光波動陣」的選擇上，我會毫不猶豫的選擇滿無雙波並點1級邪光，原因很簡單，無雙波技能形態優質可以打瞬間爆發，而邪光波動陣點1級作為穩定控制還是不錯的。至於大火要不要點滿就仁者見仁智者見智了，我推薦的選擇是點滿，SP不夠的從鬼印珠/不動明王陣上分攤點。

TP技能爭議不大，大冰基本上都選擇捨棄，因為並不增加實際傷害，並且實戰意義不大。小冰與邪光斬根據喜好點滿，其餘的基本上是定式。

【打造要點】作為固傷職業，強化武器對於我們的提升是「0」，鍛造武器增加的獨立攻擊力才是提升我們的主要手段，所以想要入手純固傷職業，第一件對於你提升最大的打造就是把武器鍛造到8！ 除了鍛造以外，與暗帝一樣的是天帝也可以適度增幅，畢竟增幅對於固傷職業的提升要大於純百分比職業。我們推薦的天帝增幅數值達到7就足夠了，並不需要全部追求一身「紅10」。而附魔的選擇仍然建議「門檻級」的雙屬強20首飾寶珠即可，版本末期不需要花費大量的氪金量來打造即將過時的史詩套裝。

結語

阿修羅和暗帝雖然一個是純固傷一個為純百分比，但從團隊定位上來看，還是有非常多的相似之處的。不管選擇哪個職業入手，我們都要保持清醒的打造思路和團隊定位，技能選擇上要努力與團隊傷害最大化的利益相結合，而打造也不要「上頭」去追求所謂的完美，畢竟新的版本即將到來，新的裝備與內容也等著我們去體驗。
