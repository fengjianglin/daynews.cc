FPX公布德杯出戰陣容！看不到khan了，也看不到S9冠軍五人組了

天天要聞
2019-12-19

馬上年底唯一的比賽就是德瑪西亞杯了，因為是轉會期結束後的第一個比賽，所以很多戰隊都會選擇在德杯上來鍛煉和考核自己的新隊員。不過全員換新人的還是沒想到的，在12月18日晚上，FPX官博公布了德瑪西亞杯的參賽陣容，基本上就是全員新人，一個S9世界冠軍都沒有，大家期待的Khan也沒有登場。

​FPX官博公布的德瑪西亞杯出戰名單如下圖所示，基本上就是自己二隊LDL聯賽的首發陣容，連教練也不是戰馬，是一個全新的陣容來打德杯。

​對於很多期待khan上場的觀眾來說，肯定是有遺憾的，畢竟大家都很想要看到khan和FPX其他隊員能夠擦出怎樣的火花，以及FPX是否會將戰術核心轉移到上路，不過因為S9世界賽長時間的征戰，再加上前不久的全明星賽事，FPX的隊員還是挺累的，高強度的比賽太多了，而且拿到了S9世界冠軍，讓他們休息一樣大家應該是都能理解的。

​因為FPX的缺席，這次德瑪西亞杯的冠軍似乎變得撲朔迷離起來了，按照之前傳言的爆料消息，大概率會在SN和TES兩隊之間決出，SN聽說訓練賽很猛，TES則是補強了打野，而其他戰隊比如RNG這邊主力UZI休息，IG這邊德杯陣容還沒有公布，以及EDG這邊感覺賬面實力不是很夠看，再加上FPX全新人出戰，這次德杯的冠軍應該會證實至少在明年春季賽成績也不會太差。

​FPX雖然是二隊出戰，但其實二隊也不算太弱，至少二隊在LDL拿到了夏季賽四強的成績，所以如果沒有遇到強敵，應該不會太慘。只是可惜了那些買了現場票去看德杯比賽的FPX粉絲了，在沒有自己喜歡的選手出場的情況下，就好好享受比賽吧！

那麼，對於這波FPX德瑪西亞杯的陣容，你們有什麼想說的嗎？
