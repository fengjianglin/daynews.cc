頭號攻略：懷舊服冬幕節全任務完全攻略，讓我們一起打雪仗吧

天天要聞
2019-12-15

今天大家上線會發現鐵爐堡和奧格瑞瑪多了幾個npc，還有一些類似聖誕節的裝束，因為冬幕節到了！

這期我們就來說說懷舊服中冬幕節都有哪些有趣的事情可以做呢？

冬幕節的小任務

冬幕節開放時間：12月15日-1月2日

活動地點：聯盟鐵爐堡，部落奧格瑞瑪

活動NPC：冬天爺爺

玩家可以在主城接到關於冬幕節的任務，通過交付相應的物品可以獲得一些禮物，禮物裡面有著材料、食物或者是寵物等一些小東西。也可以通過節日禮物獲得一些提高屬性的恢復糖果。

任務都很簡單，聯盟部落的任務都一樣，小編部落就以部落為例

冬幕節的意義

冬幕節

鐵爐堡門口

奧山隨處可見的雪堆

在尋找雪人的路上，奧山附近可以看到很多雪堆，可以開出雪球，在大戰場裡面也可以看到

雪堆

戰場中的雪球可以扔向自己人或者敵人，並且產生擊飛特效，大概能擊飛10碼左右。

大家可以拿一些，可以打斷敵對施法，還有盜賊近身什麼的，挺好玩。

在小編印象中，雪球在聯盟的大橋保衛戰中有特別重要的作用，可以把敵對勢力擊飛到橋下。。。

具體怎麼玩看大家的腦洞呢！
