頭號攻略：懷舊服冬幕節全任務完全攻略，讓我們一起打雪仗吧

2019-12-15 964條評論 2,543次閱讀 950人點贊

今天大家上線會發現鐵爐堡和奧格瑞瑪多了幾個npc，還有一些類似聖誕節的裝束，因為冬幕節到了！

這期我們就來說說懷舊服中冬幕節都有哪些有趣的事情可以做呢？

冬幕節的小任務

冬幕節開放時間：12月15日-1月2日

活動地點：聯盟鐵爐堡，部落奧格瑞瑪

活動NPC：冬天爺爺

玩家可以在主城接到關於冬幕節的任務，通過交付相應的物品可以獲得一些禮物，禮物裡面有著材料、食物或者是寵物等一些小東西。也可以通過節日禮物獲得一些提高屬性的恢復糖果。

任務都很簡單，聯盟部落的任務都一樣，小編部落就以部落為例

任務：冬幕節的意義

冬幕節的意義

到薩爾的房間和NPC對話即可完成任務，隨後又給一個任務

冬幕節

會給你一個任務道具，是一本書，右鍵可以觀看，講述冬幕節的由來

冬幕節

然後需要把書教給雷霆崖的老大凱恩血蹄，任務獎勵雷霆崖聲望200

對話完成任務

任務：馴鹿梅森

馴鹿梅森

這個需要去加基森羅帆海灣

任務地點

到裡面會看到馴鹿

馴鹿

和它對話即可完成任務。

獎勵是變形樹枝，應該是可以把坐騎變成馴鹿，如下圖效果

正面效果

感覺和亡靈很違和。。

再來張屁股，好醜。

背面效果

丑萌丑萌的。

再次和任務NPC對話，他表示石中鹽+5金幣就可再換5個。

大英雄

直接點旁邊的冬青樹貯藏器兌換，等於說可以有10個樹枝。

大英雄

任務：被偷走的冬幕節禮物

被偷走的冬幕節禮物

東西在奧山上邊的雪人洞附近

雪人洞

會看到這個

奇怪的雪人

對話後續是去讓你去搶回禮物

搶回禮物

按照插件提示區域，來回找一個36級精英雪人，下圖4個刷點

活動區域

找到了，迅速殺，因為做任務的人挺多的，聯盟部落都有。5分鐘刷新1隻

格林奇

拿到任務物品，部落回ag，聯盟回鐵。吐槽一下任務說明，部落的上面寫的去鐵爐堡交任務。

坑爹呢

給個箱子，我開的是冰霜之力，直接學了

冰霜之力

隊友開的

冬天爺爺的手套

做出來應該跟人物身上差不多

看起來很紅的手套

手套

建議手黑的團長找人做一個帶上。

任務：暫無內容（感覺像是翻譯錯誤）

暫無內容

交任務又給個冬幕節禮物任務

冬幕節禮物

需要小薑餅和牛奶，小薑餅需要自己做

旁邊的商人可以買到2個節日食譜

節日食譜

小薑餅和蛋奶酒。

薑餅需要小蛋和節日香料，節日香料節日npc賣

薑餅

小蛋很可惜，ah看運氣，我這個服沒有，應該是10級左右的鳥會掉落，現在小號都不打怪你懂得。想要做的話可能需要自己去打。

最後也是換一個禮物盒。

還有變形器，一秒變侏儒

部落在奧格的飛艇這裡

變形器

變成侏儒，持續30分鐘

變成侏儒

可以滿足很多人侏儒騎亡靈馬、侏儒騎科多獸、侏儒騎迅猛龍、侏儒騎狼的想法。

侏儒騎馬

聯盟的變形器在鐵爐堡門口

鐵爐堡門口

奧山隨處可見的雪堆

在尋找雪人的路上，奧山附近可以看到很多雪堆，可以開出雪球，在大戰場裡面也可以看到

雪堆

戰場中的雪球可以扔向自己人或者敵人，並且產生擊飛特效，大概能擊飛10碼左右。

大家可以拿一些，可以打斷敵對施法，還有盜賊近身什麼的，挺好玩。

在小編印象中，雪球在聯盟的大橋保衛戰中有特別重要的作用，可以把敵對勢力擊飛到橋下。。。

具體怎麼玩看大家的腦洞呢！

這期冬幕節攻略到這裡就結束了，大家還開出什麼好東西一定要在下面評論留言告訴我，懷舊服不知道能不能開出聖誕帽子類似好玩的東西？
