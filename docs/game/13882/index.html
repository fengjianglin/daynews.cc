雲頂之弈9.24版本前瞻：盧錫安有兩套打法，阿木木會成為主C

2019-12-12 662條評論 2,600次閱讀 1667人點贊

大家好，我是盟盟，本期和大家聊聊雲頂之弈9.24版本前瞻。

目前雲頂之弈的遊戲版本是9.23版本，在9.23版本裡面，我覺得最厲害的四套陣容是火男主C的法海，夢魘主C的劍刺，劫主C的6光，還有劇毒掠食流，這幾套應該是版本的主流上分陣容，但是雲頂之弈馬上就要迎來更新了，也就是9.24版本，對於9.24版本，大家需要了解三件事，一起來看看。

一、新增英雄

在9.24版本，拳頭新增了3個英雄，分別是賽娜、盧錫安和阿木木，賽娜（影，系魂）是2費卡，盧錫安（光，系魂）是4費卡，阿木木（地獄火，守護神）是5費卡。

根據這3個英雄的羈絆，我們可以得出以下結論：

1、盧錫安和賽娜可以組成獨特的系魂羈絆，系魂羈絆的作用是只要賽娜和盧錫安有一人在場，他們就不會消失，就算血條空了，依舊能夠打輸出。也就是說如果給賽娜升到3星，然後給一身坦克裝備，盧錫安一升輸出裝（盧錫安的技能是大招，傷害高），那麼盧錫安就能在場上無止境輸出，這也是盧錫安的一套簡單打法，簡稱「送死流」盧錫安，賽娜和盧錫安的系魂羈絆保證盧錫安的二次輸出。

2、盧錫安除了上面的系魂羈絆，大家應該要注意一下盧錫安的光羈絆，這也是盟盟要說的盧錫安的第二套打法，6光盧錫安主C，這裡說的6光很簡單，一定要有盧錫安和掘墓人，其他隨意，不需要找劫變成光劫，只需要給盧錫安拿裝備就好了，盧錫安最好是回藍和傷害裝，再搭配一個賽娜，湊系魂6光羈絆，盧錫安主C，這樣不需要5費卡的一套陣容就出來了，畢竟盧錫安的技能是聖槍洗禮，別人的前排根本頂不住，這也是9.24版本註定要火的一套打法，系魂6光盧錫安主C，盧錫安和賽娜堪稱絕配，先給大家介紹到這裡。

3、根據阿木木的羈絆，一套全新的地獄火打法來了，阿木木是5費卡，我們知道目前遊戲中地獄火的英雄只有7個，想要湊齊9地獄火，需要有地獄火拉克絲，而阿木木的加入，使得9地獄火變得更加可能，而且阿木木是5費卡，技能自然不會弱，據了解，阿木木的大招類似寶石的大招，星級越高，大招範圍越大，也就是說如果是一個3星阿木木，那麼基本上全屏能夠禁錮造成傷害，然後全屏地獄火的二次羈絆傷害。

目測在9.24版本，3地獄火應該是滿足不了大部分玩家的需要，至少就是6地獄火，或者就是法海體系帶阿木木，個人是兩套都比較推薦。

首先6地獄火的阿木木只要是兩星，基本上一個大招就能清屏，傷害非常高，因為他的大招傷害在2星的時候能夠佔據3/4個棋盤，敵人無所遁形，而且地獄火的羈絆效果就是釋放技能以後，技能區域範圍裡面會有持續傷害，只要阿木木出傷害裝，先手一個大招，配合一個火男，敵人還怎麼玩？所以在9.24版本，盟盟大膽預測阿木木會成為主C。

二、英雄的增強和削弱

上面是拳頭給出的9.24版本最新的調整，顯示了英雄的增強和削弱，這應該是9.24版本近期最後一次調整，所以正式服更新的時候，這才是大家最需要了解的。

拳頭給出的英雄調整當然有增強和削弱，單單看圖，想必大家不知道增強和削弱了哪些地方，且聽盟盟一一道來。

增強的英雄有亞索、妮蔻、劍魔、石頭人、寶石和拉克絲，亞索基礎攻擊調高，寶石開大的藍量調低25，拉克絲開大的藍量調低5，而且二星拉克絲的傷害提高100，石頭人整體傷害提到，三星石頭人的大招面板傷害變成1000，應該算是巨大提升。妮蔻的攻擊範圍提升一格，劍魔的技能釋放藍量減少15。你以為只有這個6個英雄增強了嗎？其實還有輪子媽基礎攻擊增加5點，索拉卡攻擊距離提升1格，三星掘墓人的召喚物生命值提高600，變成2000血。

既然有增強的英雄，自然少不了削弱的英雄，因為在9.23版本的雲頂之弈，盟盟前面也說過了，火男主C的地獄法海強度很高，煉金主C的劇毒掠食也很強，還有就是前期用三森林兩德魯伊過渡的體系也不錯，所以都被削弱了。

艾翁的護盾在2星的時候減少50，三星減少100，大樹的被動回復整體下調一個檔次，一星下調75，兩星和三星下調50，鱷魚的治療效果在2星的時候下調25，火女的提伯斯生命值減少200，護甲減少10，兩星火男傷害下調25，三星火男傷害提升200，兩星煉金傷害下調100。

很明顯，在9.23版本強勢的體系在9.24版本都被削弱了，而且削弱的比較狠，因為都是在2星的時候傷害或者恢復能力被拳頭大打折扣，這也就造成了一星C不動，兩星傷害低的局面，整體平衡了體系，使得9.24版本很多陣容都得到了出場的機會，不會出現一局遊戲7家法海或者一局遊戲7家掠食的局面，遊戲的多樣性變得豐富，其實還和一點有關，也就是我接下來要著重講的一點。

三、羈絆和裝備的調整

先來說一下裝備的調整，在9.23版本有一件裝備太強了，那就是綠叉，能夠100閃避暴擊傷害的綠叉，基本上現在玩刺客都沒人敢出無盡，因為出無盡之後，別人只要有綠叉，你這個英雄基本上就費了，就算升到3星也會變成一個給別人充能的鐵憨憨。

因為綠叉獨特的閃避暴擊的機制，在9.24版本終於被拳頭給削弱了，應該算是重做，在9.24版本綠叉的效果改為暴擊傷害變成非暴擊傷害，也就是不能閃避暴擊，而改成不能讓對面暴擊，這一改動是不是有玩家看不懂呢？其實很簡單，假如你有一個3星夢魘，在9.23版本遇到一個3星皎月，他有綠叉，而你的暴擊是55%，那麼你只有45%的概率能夠打出傷害，因為他的綠叉會抵消你暴擊的傷害，但是在9.24版本不一樣了，同樣的場景，你有100%的概率打出傷害，其中55%的暴擊傷害會變成普通攻擊的傷害，相比之下，同等條件下，9.24版本對於刺客英雄真的是天大的加強，直接提高了一半的輸出，而且不會再給別人充能了。

還有一件裝備是巨人殺手，這就是那件加強的裝備了，其實也就加強了1%的傷害，從8%當前生命值變成了9%，小幅度提升，大家就當是削弱綠叉的贈品吧，最後還有就是在9.24版本擊敗龍和峽谷先鋒掉落鏟子的概率下調到了10%，而且他們不會掉落自然之力、巨九和竊賊手套，這個改動其實都是玄學，畢竟這和玩家的運氣有關係。

再來就是羈絆的調整，雲霄羈絆整體提升5%閃避率，雷霆羈絆整體傷害提高，尤其是2和3雷霆的時候，傷害分別提高30和50，地獄火羈絆調整為技能和暴擊可以對敵人造成後續傷害，但是傷害整體調低，由80/175/275變成70/120/160（百分比），影響範圍有1格，變成了1/3/5格，個人感覺地獄火羈絆被削弱了，可能是由於阿木木的上線，官方為了不讓阿木木主C的地獄火一家獨大，所以調整了地獄火羈絆。光羈絆攻速在3/6的時候增加5%，4海洋羈絆法力恢復從35變成30，影羈絆有2/4變成3/6，畢竟增加了一個新的影英雄，在9.24版本就會有6個影英雄，分別是賽娜、千珏、小法、狂戰士、瑪爾扎哈和劍聖，當然最愉快的還是找到影拉克絲，雖然變成3/6的影羈絆，但是傷害提升直接翻倍，由原來的40/85直接變成70/140（百分比），這難道不強嗎？

最後就是職業的調整，召喚職業削弱，3召喚：生物生命提升降低10%，6召喚提升20%，說白了就是削弱地獄法海裡面的3召喚羈絆。遊俠職業在2/4的時候攻速整體提升，6狂戰士還會提高25點攻擊（個人感覺沒什麼用）。

以上應該是9.24版本的最全前瞻分析了，總結一下，在雲頂之弈的9.24版本，打法變得多樣性，不像9.23版本一樣，強勢的體系只有那麼幾套，「地獄法海」「暗影召喚」「沙漠劍刺」「掠食劇毒」，在9.24版本應該會百花齊放，個人感覺「極地狂戰會崛起，6影召喚會風靡並且持續一段時間，阿木木主C的地獄火應該很強，盧錫安主C的6光和夢魘主C的森林刺、沙漠劍刺都能玩，4遊俠體系也會得到開發」，所以雲頂之弈9.24版本什麼體系都能玩，就看你會玩哪套體系了。
