Ps：插個題外話，燼燃大招耗能100點，2阿獃芙可以使大招在80點能量的時候釋放，但還是會扣除100點能量（這句話的意思是80點能量放大後，能量為0點；90點能量放大後，能量為0點；99點能量放大後，能量為0點），所以裝上2阿獃芙的話，有大就視情況儘早脫手

看到這裡的朋友可能有個疑慮，為什麼漢娜和莎比我都分別共鳴了意識，因為我就是開頭說的特別糾結的那個

正文開始

就幾組對比下來，充能後普攻自不必說，莎比普攻傷害鐵打的就是比漢娜高。

但除去普攻，充能後三消紅球黃球對比，傷害其實是差不多的（濾掉各自+1紅黃球共鳴意識），由此可見，漢娜三消增傷25%與莎比四件套三消的傷害，基本可以持平。

大招對比，毫無疑問，大招火系傷害加成，漢娜略遜一籌。

由此得知，漢娜三消增傷25%和莎比四件套傷害差不多，在普攻與大招都稍遜莎比的情況下，能與莎比抗衡的就是迅速三消。莎比四件套中的灼燒傷害，拿圖1普攻來說，疊1層就是1秒跳1次193，疊3層就是1秒跳3次193，還有攻擊20%幾率爆炸傷害，爆炸傷害2000左右，差不多抵一次普攻。

咋一看下來，莎比好像強於漢娜，單單靠漢娜的迅速三消是否足以抗衡，請看下圖，現階段樁王（卡穆）對比（Ps：建議官方在木樁訓練里出個傷害統計，或者來個20秒內傷害統計，不然打樁只能打爆跳蚤卡穆）

在樁王二號里的表現，漢娜場均基本在180秒左右，莎比場均在150秒左右

圖已放完，那麼單靠漢娜迅速三消是否足以抗衡莎比呢，答案毫無疑問，是肯定的。

在能快速結束的戰鬥中，漢娜迅速四次三消接大招的黃金體驗（我剛說了jo）是莎比所不能比的，日常刷圖漢娜就是爽到不能呼吸，但相對的，在不能儘早結束的戰鬥中，莎比優勢會慢慢擴大。

那麼，漢娜/莎比如何選擇？我個人見解，都一樣。

在兩分鐘以上的戰鬥中，莎比優勢才會體現，但在此之前，漢娜三消加大招唰唰唰已經爽飛，要手感與還是要愈戰愈高的傷害，這其中的取捨，看個人，仁者見仁。
