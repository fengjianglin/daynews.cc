韓服更新後，半年前曾對DNF未來版本的猜想居然成真了

天天要聞
2019-12-23

2019年6月，我曾在掌游寶發表了一篇投稿，題目是《從安徒恩時期的玩家成長淺談DNF的未來》感興趣的朋友可以去看看。吃級

2019年可謂是DNF岌岌可危的一年，當時我從玩家成長的角度分析了導致DNF可玩性降低的部分原因，

最後總結了三點建議：

1.裝備方面

2.技能方面

3.團本方面

當時對策劃會意識到這些問題並不報多大希望，

然而後面半年韓服卻真的對其中裝備成長和技能成長兩方面進行了重大變革，這讓我感到驚喜和欣慰的。

當時我是這麼建議的：

而剛在韓服發布的100級版本中，我談到的幾個點得到了印證（儘管稍有偏差）：

1.粉裝傳說增加可玩性較高的專屬地圖。

2.增加可交易強散，平穩過渡。

韓服100級版本的可交易史詩以及強力傳說，提供了新的成長路線——拍賣行戰士，可交易史詩甚至讓曾經的經典85史詩套復活了！

3.混搭玩法，增強裝備功能性

100級史詩可以有超多種組合的可能，且每種裝備都有各自的特色，增強了裝備功能性。

這個自不多說，在發表這篇文章後，也就是在2019年9月，韓服果真把鴿了許久的CP系統發布了！

儘管很多玩家反饋這個系統和相關副本並不是那麼的好玩，也並沒有達到我所預期的「又一縷曙光」，但起碼策劃願意嘗試，這個系統和副本也並不算失敗。

如果考慮到和100級裝備以及希洛克團本進行組合，說不定到那時才能展現出CP系統的真正光芒。

現在最關鍵的希洛克團本還沒有出現，但根據韓服姜策劃的開發者筆記

姜策劃將對副本中玩家的輸出方式進行改變，不在是全部的怪物都是虛弱打樁輸出，或者單一的控制輸出。「我們希望會有一種在特定情況下才會出現虛弱的系統，比如雖然沒有虛弱但是也不限制控制怪物」

這是姜策劃的原話。當然這跟我所說的「互動性」稍有區別。

最後大膽猜測下希洛克團本將會是真正意義上的最大型副本，參與人數可能在20人及以上。

經歷了坎坷（gou shi）一年，韓服策劃（如果國服策劃不能保證儘快更新到100級版本跟韓服同步，那國服策劃還是shi）已經逐漸意識到提高遊戲可玩性的必要，希望DNF挺過這一年後能夠越來越好，至少未來兩年更好一些吧。
