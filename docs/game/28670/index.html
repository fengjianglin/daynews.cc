LOL-德杯：“絕代雙蕉”大殺四方，VG摧枯拉朽擊敗WE取得第三勝

2019-12-22 256條評論 1,366次閱讀 332人點贊

2019德瑪西亞杯小組賽杭州站VG對戰WE的比賽剛剛結束，VG野輔雙游打出完美節奏，層層壓制碾壓擊敗WE，拿下第三勝。

VG：瑞茲、酒桶、刀妹、女槍、曙光

WE：鱷魚、皇子、辛德拉、霞、泰坦

刀妹在中路對線上打的非常主動，利用前期的高傷害給辛德拉帶來了很大威脅，一波比翼雙刃命中讓刀妹打掉了辛德拉很多血量，迫使辛德拉交出虛弱。隨後不久，VG野輔遊走到中路，交出兩個閃現後擊殺辛德拉拿到一血。

中路遭到擊殺後，皇子來到下路想要博取優勢，但酒桶及時趕到反蹲，一個精準的大招救下隊友同時將霞控住擊殺，VG再次打出1換3。掌握了前中期的主動。

發育順利的女槍和刀妹讓WE非常難以處理，酒桶更是屢屢用大招分割戰場贏下小團戰。接連幾輪進攻獲勝，VG凱歌高奏，拿下了近4千經濟領先。

20分鐘過後，雙方圍繞著河道開始激烈爭奪視野，曙光率先發難先手開團，而想要趕來支援的WE被女槍大招洗臉直接陣亡二人，VG打出1換5後拿下大龍。酒桶在拿到大龍一瞬間被大龍換掉。

陷入絕境的WE又在小龍河道處試圖集火擊殺女槍，但不受限制的女刀從側翼殺出已是無人可擋，VG再次團滅WE，破掉兩路高地。重整狀態後，VG在中路再次正麵糰戰擊潰WE，一波結束比賽，恭喜VG！
