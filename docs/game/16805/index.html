王者榮耀：雙G大戰，誰贏誰稱王，QG到底輸在哪裡？

2019-12-16 2,071條評論 3,064次閱讀 749人點贊

不論什麼比賽，似乎大熱門的隊伍到最後都會垮掉， 兩支隊伍的焦點之戰在12月14號開打了，兩支隊伍都是實力很強的隊伍，畢竟都打到了總決賽，然而這次與以往不太一樣的是AG以較輕鬆的姿態贏下了比賽，4：1帶走了QG，這次比賽QG到底輸在了哪裡？下面帶來五局比賽的分析。

第一局AG拿到了野核，並且配上了陣地戰較強的英雄，而QG則是拿到了更趨近於四保一的體系，在第一局的BP階段，QG利用自己在藍色方的優勢為hurt拿到了熟悉的公孫離，而AG則是將姜子牙以及fly的曜鎖死在BAN位上，並且還拿到了比較針對射手的英雄關羽。

AG在前三樓就將自己的雙邊優先拿下，在第二輪選擇時則是將輔助位留在最後的針對位上，在對面選擇了沈夢溪作為消耗位的時候，AG拿出了蔡文姬，這個英雄拿出來就為前三手的英雄帶來了錦上添花的感覺，陣地戰陣容拿到蔡文姬是非常好的選擇。

蔡文姬在前期雖然身板太脆，不足以站視野，但是其前期的一級團團控非常充足，在一級團的情況下，他的控制還是非常充足的，並且前期因為有周瑜以及老虎的存在，蔡文姬不用孤身一人站視野，這也是給這個陣容帶來了很強的壓制力。

而到中後期，蔡文姬有了一定的裝備之後，團戰可以說非常輕鬆，因為蔡文姬的無限補給，就算對面有沈夢溪這樣的高消耗英雄也能保證自己家在團戰時的血量，充分將自己陣容的優勢發揮到最大。

而QG在總決賽的第一場比賽就拿出了非常類似四保一的陣容，並且他們拿到的英雄都是非常偏後期的，在面對AG的體系時，AG的前期進攻他們基本毫無辦法。而公孫離在前期的屬性基本決定了他在前期做不了什麼事，這就給了蔡文姬很好的發育空間。

從兩邊的陣容來看，QG的陣容太偏後期，而AG在拿到陣地戰超強的英雄並配上蔡文姬之後給整個陣容在一級團的成功率上又增加了很多，再加上QG的陣容決定了他們在前期面對AG時只能選擇避戰。

在開局時，AG利用蔡文姬一級團很強的屬性帶上前期也很強的裴擒虎及周瑜開始入侵QG的野區，並成功地反走了野區生物，這時候可以看出QG只是嘗試性的守一下，但覺得 守不住就放棄了，之後AG回到自己野區收完整片野區的資源。2分鐘小龍刷新時，QG雖然率先開小龍，但是最後還是沒能從AG手中搶下小龍。此時雙方經濟並沒有拉開。

但是從拿到小龍開始，AG就找到了推塔的節奏，利用 周瑜搶線快的優勢，幫助周瑜拿到線權，從而解放周瑜，讓周瑜跟著隊伍一起遊走，開啟了隊伍的推塔節奏，QG疲於防守，五分鐘就掉了兩條邊路的一塔。AG發現了對面清線困難的問題，就選擇避戰推塔。

利用自己的優勢開啟了推塔的節奏，第一局比賽雙方几乎沒有爆發人頭，AG利用自己的優勢一直運營，直接在20分鐘前就結束了這局比賽。這局比賽可謂是教科書級別的運營了，將對手在頃刻之間就解決了。

第一局比賽AG將自己的戰術執行到底，並且穩當的贏下了比賽，他們在抓住對方的陣容漏洞時立刻轉變思路，改變打法，這等應變能力的確是讓他們有了很好的節奏。

第二局比賽是QG唯一贏下的一局比賽，在hurt拿到公孫離還沒能贏下來後，QG變換了體系，雖然也是給hurt拿的射手，但是給他們的中路換了一手較強的法刺上官婉兒，並且給fly拿到了狂鐵這個在邊路較強的英雄。

在這一局比賽中，大家不再是圍繞hurt打，而是讓hurt自己在邊路和成吉思汗對線，這一局陣容和上一局相反，QG這一局陣容是比較偏前中期的。兩分鐘暴君團時，因為橘右京和婉兒的傷害都要高於對面，QG有驚無險的拿下了第一條小龍。而在這個時候，AG主動選擇了換線，將射手換到了上路，率先推了塔。

雖然在陣容上來說，QG是更佔優勢的，但是從場面上看AG佔據了主動權，還將經濟優勢掌握在自己手中，AG在 這局比賽中，知道自己陣容前中期不足以和對面硬開團，則是選擇了用運營來緩解自己的劣勢。

AG在前面的八分鐘里都做得非常好，但是在八分鐘的時候給了對面機會，把自己家的優勢完全抹殺掉，甚至還率先丟了一座高地，而後面他們錯誤地估計了對面狂鐵的傷害導致給對方抓住了機會，翻盤成功。

QG雖然贏下了這局比賽，但是他們贏得非常艱難，而AG雖然輸了但是他們在這局比賽里還是像我們展現了成熟以後的AG強大的團戰能力以及運營能力。

而第三局、第四局以及第五局對AG來說都是一個大順風局，我們並不做過多贅述，主要來說一下QG在這三局中存在的問題，QG在這剩下的三局比賽中基本是出於打團打不過，運營也出問題的局面。尤其是第三局以及第五局比賽。

在第三局的陣容中，孫尚香作為主C位的確是應該受到好的待遇，但是在選擇輔助時使用的卻是莊周，這一手莊周可能是為了能夠針對到對手的中路位置，但是AG卻沒有選擇讓拿一個有控制的陣容，而是在見到對面拿出張良的時候拿了一手嬴政。

QG的整體思路是好的，但是對面的陣容就決定了他們的陣容將會很被動，孫尚香也找不到輸出空間，這一局是整個QG打得最艱難的一局，但是此時他們已經用完了這個版本比較強勢的射手。

而第五局因為是對面的賽點局，QG拿出了孫臏加楊玉環的陣容，AG則是拿出了瑤加孫尚香以及宮本武藏的組合，這個陣容將楊玉環配孫臏的陣容完美的化解了，不得不說這次的AG準備非常充分。

整體而言，這場比賽觀賞度並不是很高，但是AG單方面表現出來的極高的運營水平以及團戰能力都是頂尖的，而QG在這段時間的備戰中似乎沒有找到正確的方法來應對這場比賽。從細節來說，總決賽上的AG做得非常的好。AG現在整體狀態很好，而QG則是還需要更多地調整才行。
