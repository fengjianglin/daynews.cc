不玩就虧了？全球總銷量第一的遊戲，2020年終於也要出續作了？

天天要聞
2019-12-10

大家好，這裡是正驚遊戲，我是正驚小弟。

2020年即將到來，而這看似平凡的2020年似乎就是許多玩家心目中的「有生之年」。因為許多遊戲大廠都表示將要在2020年推出打磨已久的大作，這其中就有著玩家們期待了許久的《騎馬與砍殺2》、《賽博朋克2077》等遊戲。下面小弟就來詳細的給大家講講2020年即將發售的遊戲，看看2020年到底是不是你們心目中的「有生之年」。

看門狗：軍團——黑客無孔不入，黑客就在身邊

預計發售時間：2020年3月6日

《看門狗》系列是育碧以「黑客」題材為核心打造的一款動作冒險式遊戲，距離前一作發行已有四年。雖然遊戲中黑客的風格與現實相差甚遠，但是各種炫酷技術的展現反而更受玩家們的喜愛，能真切的讓玩家體驗到一種類似《黑客帝國》中電影主角的感覺。

而相比前作遊戲中的大開大合，《看門狗：軍團》中一些技術的展現則體現出了一種「入微級」的狀態。玩家不用再像前作一樣只能利用技術入侵各種系統展現出炫酷的操作，而是能通過技術去影響NPC的生活。最終，通過改變NPC的生活來改變NPC的各種選擇，從而達到自己的目的，令玩家有一種「黑客上帝」的既視感。

彩虹六號：封鎖——被殭屍感染的經典神作

預計發售時間：2020年第一季度

作為最經典的FPS遊戲之一，《彩虹六號》系列始終圍繞著「反恐」題材來做文章，前一作《彩虹六號：圍攻》遊戲累計收入已經超過了10億歐元（約78億歐元）。該系列區別於一些無腦「突突突」的遊戲，整體極為重視細節，為玩家打造出了一個極為嚴肅的戰場。玩家必須靠著大腦、眼睛、耳朵以及手部的緊密協調才能獲勝，最後獲得的快感自然不言而喻。

而距離《彩虹六號：圍攻》發售三年之後，本次即將發售的《彩虹六號：封鎖》卻一反常態的打造了一個純PVE世界，玩家可以在這款遊戲中享受到經典的「打殭屍」模式。遊戲將採取3名玩家合作對抗電腦的形式，讓玩家們深入被外星寄生蟲感染的封鎖區共同執行任務。

賽博朋克2077——科幻版《巫師3》

預計發售時間：2020年4月16日

《賽博朋克2077》是著名遊戲《巫師3》開發商「波蘭蠢驢」的最新大作，遊戲2012年5月就公布了，終於在明年有望看到了！遊戲沿用著《巫師3》的製作風格，以龐大的開放式地圖、海量的劇情任務、大開腦洞的世界觀及玩法吸引著玩家。

遊戲中玩家可以化身各種職業去盡情的探索這未來的世界，在這個過程中玩家將面臨和《巫師3》中相似的道德層面上的抉擇。這會再度加強遊戲的真實性，同時也增強了遊戲對於玩家的吸引力。

我的世界：地下城——縱向玩法的擴展，《我的世界》版的《暗黑》

預計發售時間：2020年4月

大名鼎鼎的《我的世界》想必就不用仔細的介紹了，作為全球總銷量第一的遊戲，本次的續作更像是《我的世界》IP的一個衍生作品。遊戲告別了生存、養成、經營式的傳統玩法，向著動作冒險式玩法轉變。玩家可以在遊戲中盡情的打怪、探索，在地下城中進行冒險。

從遊戲的玩法看來，《我的世界：地下城》更像是經典的《暗黑》類遊戲。比如玩家在遊戲中可以消耗符文來解鎖不同的天賦，從而打造出具有職業傾向的角色。除此之外，遊戲也繼承了一些前作的細節，比如羊駝見到狼就吐口水的特性，這讓能讓玩家在本作中體驗到一些感覺。

光環：無限——最先進的引擎打造出的經典FPS遊戲

預計發售時間：2020年第三季度

《光環》系列作為最經典的FPS遊戲系列之一，本身就擁有著大量的粉絲。而本次即將於2020年發售的《光環：無限》還搭載了據說是目前最先進的遊戲引擎「Slipspace」，其表現出來的效果更是引導玩家們的期待。

雖然目前遊戲並無更詳細的資料放出，但就畫面、特效、動態物理效果等「硬體」而已，《光環：無限》絕不遜色於同類遊戲。同時，遊戲製作人也表示，《光環：無限》將回歸《光環》系列的精神核心，這意味著《光環：無限》將再現希望、奇蹟、英雄、人性、社會等極具魅力的文化屬性。

精靈寶可夢Sleep——任天堂帶給玩家們的一場美夢

預計發售時間：2020年

自《精靈寶可夢GO》之後，任天堂似乎把關注的重心都放在了玩家們的健康上。先是發布了需要靠玩家手動扔球捕捉小精靈的《精靈寶可夢 Let’s Go》系列，又即將在2020年推出一款靠睡覺來玩的《精靈寶可夢Sleep》。

同時新一代遊戲外設「pokemon go plus+」也將同步推出，這款外設能監控玩家的睡眠時長、睡眠質量並反饋到手機當中，以此來達成「通過睡覺玩遊戲」的成就。而目前遊戲並未透露出這種反饋將以怎樣的形式展現出來，所以遊戲具體呈現出來的品質還難以確定。

消逝的光芒2——更自由、更真實的末日世界

預計發售時間：2020年4月30日

《消逝的光芒》是一款經典的第一人稱生存類遊戲，該遊戲靠著比同類型遊戲更為優秀的劇情吸引著玩家。而本次的續作將加強這一優勢，在劇情中交予玩家更多的選著，讓玩家能夠更自由的按照自己的想法來控制遊戲世界的發展。

玩家可以在遊戲中扮演著好人、騙子、內奸、強盜等各式各樣的身份，而玩家做出的行為不僅能影響整個世界的運轉，整個世界還會根據玩家的各種行為做出不同的反饋。簡而言之，該作比起前作來說更為自由，並且世界運行也變得更為真實。

漫威復仇者聯盟——全新的故事背景，未減員的《復仇者聯盟》

預計發售時間：2020年5月15日

《漫威復仇者聯盟》這款遊戲基於《復仇者聯盟》IP構建了一個全新的故事背景，「復仇者聯盟」將再次集結，共同對抗一個名為佐迪亞克的古老組織。為了構建好遊戲的劇情，官方甚至還專門請人執筆撰寫了一本小說。

遊戲具體為動作冒險的模式，玩家可以根據主線的推進去進一步了解劇情，也可以通過「戰地任務」 來探索地玩英雄養成。從目前官方所放出的內容看來，玩家可以操控的英雄至少有美國隊長、鋼鐵俠、雷神托爾、黑寡婦和綠巨人，所以遊戲的時間線應該在電影《復聯4》之前。玩家可以再次看到復仇者集結的樣子，這對於某些復聯迷來說也算是好事。

廢土3——末日下的養成，遊戲里的道德考驗

預計發售時間：2020年5月19日

《廢土》系列屬於比較經典的末日角色扮演遊戲，與同類遊戲比起來，《廢土》系列遊戲的自由度不算很高，但是其劇情對於人性的拷問卻有著一種另類的魅力。這末日帶來的不僅有崩壞，還有感動。玩家可以在遊戲中看見人性的醜惡，也能看見醜惡里那一絲的微光。

而在即將發售的《廢土3》除了在畫質上進步以外，還強化了系列的玩法，讓玩家能控制更多載具。同時，遊戲中的「戰役模式」還加入了多人合作的機制，玩家可以與自己的好朋友一起去解決困難，共同通關。

騎馬與砍殺2：霸主——跳票多年的大作，真實又硬核的中世紀世界

預計發售時間：2020年第四季度

大名鼎鼎的《騎馬與砍殺》估計許多玩家就算沒玩過也聽過，畢竟身邊多多少少都會有些人在抱怨《騎馬與砍殺2》的跳票，不過哪怕如此也依然有不少玩家在苦苦等待。其魅力就是源於模擬的中世紀世界與真實的物理反饋。

喜歡硬核格鬥的玩家可以在遊戲中體驗到冷兵器的魅力，喜歡策略的玩家可以在遊戲里謀劃天下，喜歡帶兵打仗的玩家也可以體驗到戰場的廝殺，除此之外還有著種種不同的遊戲方式。《騎馬與砍殺2》在繼承前作基礎的同時還深度優化了畫面、動作細節等，讓玩家的代入感更為強烈。

小弟有話說：2020年雖然有許多「有生之年」系列面世，並且還有多款佳作，但是想想也能知道這些遊戲不可能太便宜。所以大家在購買的時候可要看看自己的錢包，量力而行。
