王者榮耀賽事精彩總是一波接一波！這個對於所有王者榮耀玩家來說已經是很熟悉了，都清楚的，在剛剛感受過秋季賽的緊張刺激之後，這王者榮耀冬季冠軍杯可是要來了哦，還沒來得及從AG超玩會奪冠的喜悅中走出來，這大戰又是要開始了，確實是一刻不容鬆懈啊！此次王者榮耀冬冠杯虎牙直播也是會全程直播，藍光快人一步的觀賽體驗，不容錯過哦！

​

對於KPL賽區來說，這將冬冠杯的獎盃留在KPL那是必須要做到的事情，這可是整個賽區的榮譽啊，就不單單只是戰隊榮譽了，很關鍵的。作為昔日冬冠杯冠軍戰隊，Hero久競依舊是備受關注的，雖說這個賽季這支戰隊經歷了太多，但是看到這Hero久競曬出的冬冠杯大名單後，感受得到，對於衛冕，他們依舊做好了準備，新老搭配走起！

特別是在看到名單中的Hero久誠的時候，網友們也是開始熱議起來了，都清楚的，這個賽季對於久誠來說可是太難了，粉絲們也是替久誠難受，現如今看來，這冬冠杯，久誠是要發力了，這個恐怖的輸出機器，大概率首發了，那一手幹將依舊印象深刻啊！

​

冬冠杯在即，為了讓玩家們看得更有方向性、針對性，居居也是將開啟虎牙直播詳談此次冬冠杯的出線形勢，競爭還是相當激烈的，E星、Hero久競等強隊的對決，名額有限啊，特別是對於Hero久競來說，居居也會仔細談談，畢竟久誠再現賽場太讓人期待了，作為虎牙籤約的KPL高人氣選手，久誠究竟會有怎樣的發揮呢？

​

​

另外，真的也蠻好奇居居的預測能力的，他也有這個實力，畢竟職業選手經歷在那裡，也是實力解說，還是很有參考性的，期待居居今晚的虎牙開播詳細解讀吧！
