三國殺：界山包再添四將，看我還敢捂住我孫策的臉

2019-12-25 250條評論 3,985次閱讀 2607人點贊

各位小夥伴大家好，我是”士別三日，即更刮目”的吳下阿蒙！

三國殺公測版在今天又又又開啟了，這次新上線的武將居然是四名界限突破的山包武將，其中就包括最受期待的界左慈和界孫策了，那麼就隨阿蒙一起來看看，究竟誰是最大贏家呢？

界左慈

【新生】的效果不變，【化身】大幅度改動，除了開局獲得三個化身之外，在回合開始和結束階段，可以選擇更改化身牌或者制衡化身牌，但制衡化身牌的條件是至多兩張、並且是沒有使用過的，總的來說和ol版的界左慈幾乎是一樣的。

那麼這個界左慈實戰強度如何呢，就阿蒙玩了幾把來看，還是有一些瑕疵的，因為想要重置化身牌的時機與更換化身牌的時機是衝突的，意味著你只能選一種效果。比如你在回合結束時重置，那麼就得保留回合內的制衡、離間這種技能直到下個回合開始。如果你在回合開始重置，就得保留遺計、奸雄這種白板效果到回合內。

這種情況就導致了在節奏比較吃緊的對局裡，很難抽出機會重置，如果你回合開始重置，就會帶著防禦技能淪為不動白，如果你回合結束時重置，沒有防禦效果又很容易淪為團滅發動機。縱觀目前的軍八局環境，實在對左慈不是很友好呀。

不過制衡化身的效率還是非常高的，即便沒有受傷，只有開局的三個化身，也能較快的換出華佗、郭嘉等關鍵化身（也可能是測試服只能化身標將的bug）。另外，在測試服界左慈居然是精品武將，不過我想上架到移動版之後，應該會改成史詩的吧。

總而言之，阿蒙認為界左慈的加強幅度不是太大，但是更能雜耍了，左慈大仙已經在在神棍的道路上越走越遠。

界孫策

插畫中規中矩，原畫中的細節只剩下了用父親的鮮血染紅的紅領巾，搖晃著雙節棍，您這是要上天嗎？

技能上面，【激昂】效果不變，主要效果依舊是打牌時你可以很激昂，差評差評。

【制霸】則是小幅度加強，針對拼點時點數相同這種情況，孫策也能獲得兩張牌了。只要捏著一張K，就再也不怕假忠臣強行打劫自己了，就算他有K也不行。

【魂姿】大幅度改動，首先是兩點體力及以下都可以觸發覺醒，而覺醒後的英姿變成了界英姿。這是最核心的改動部分，兩體力及以下覺醒，相比於一體力覺醒，遠不止多了一點體力那麼簡單。

可以這麼說，孫笨的覺醒難度下降了一個檔次，兩點體力意味著更多的手牌用以防禦，更安全的覺醒環境，更容易打破54320的詛咒！那麼以後評價孫笨的時候，可能更多的時候是評價覺醒後完全體的孫策，而不是只有激昂的”孫笨”。

以前的孫笨，主技能是激昂，約等於沒有。但是改版後的界孫策，更多的時候會以覺醒的姿態在場上，主技能是英姿和英魂，這完全是兩個武將好嗎？

從今往後，看誰還敢捂主我江東小霸王的臉！

今天的內容就到這了，下期阿蒙再來分析界二張和界劉禪吧。各位小夥伴對界左慈和界孫策有什麼看法呢？歡迎在評論區留言討論~~
