這款遊戲告訴你，為什麼遊戲能被稱為“第九藝術”

2019-12-10 1,437條評論 3,019次閱讀 141人點贊

歡迎收看遊戲不止，我是喬伊。這次給大家推薦一款神作。

這是今年我玩到的第二個讓我記憶深刻的遊戲，第一個則是之前給大家介紹的《奧伯拉丁的回歸》。而它的名字也和《奧伯拉丁的回歸》風格一致，名為《艾迪芬奇的記憶》。

其實我在STEAM上多次見到這款遊戲了，但每次都止步於遊戲商店頁面那毫無亮點的封面，以及不知所云的宣傳片。

直到不少觀眾一直私信說讓我給大家講一講這款遊戲，耐不住大家的熱情，我去嘗試了一下。

一個短語總結遊戲給我帶來的體驗：平靜而震撼。

藝術，綜合建築、音樂、繪畫、雕塑、詩歌，文學和舞蹈。之後將電影加入了藝術的行列。名為第八藝術。而無數的電子遊戲前輩和工作者，將遊戲推上了「第九藝術」的行列。

我們見過很多3A大作，它們帶給了我們刺激而有趣的遊戲體驗，我們也見過很多粗製濫造，騙錢氪金的劣作。但能被稱之為「第九藝術」的遊戲，少之又少。因為能被冠以這樣頭銜的遊戲，絕對是真正意義上的佳作，不但在遊戲中給人心靈上的感觸，在遊戲外也能帶給玩家無盡的回味和思考。在我心中，像《風之旅人》，《上古捲軸5》等佳作就是「第九藝術」。

而這次，我要給大家介紹的《艾迪芬奇的記憶》，也能承載「第九藝術」這一頭銜。

遊戲的流程很短，如一部電影時長一樣，剛好兩個小時。遊戲劇情講述了一個華盛頓州名為芬奇的家族，出現的種種怪談。

說實話遊戲的第一印象，畫質有點糟糕。即使是最高畫質，也不理想。

略帶陰暗的畫風，讓我一度感覺像是在玩《生化危機7》，去一個詭異建築探索的恐怖遊戲。但女主的自白和平靜的音樂，以及獨特的字幕呈現方式，打消了我的疑慮。

遊戲的操作體驗很不錯，在主機上玩過《底特律：變人》的玩家就能明白。例如在打開蓋子，旋轉鑰匙，翻書等，是需要你的滑鼠進行動作上的配合移動。意思就是有點帶體感操作的反饋，而不單純是我按個F什麼的就完了。這樣的設計加深了遊戲的互動體驗。

隨著我們跟隨女主自白的介紹，我們開始逐漸了解，玩家扮演的是一名17歲的女孩，艾迪芬奇。而這個看起來多次違章搭建的詭異建築，便是她童年生活的家。如今早已空無一人。她的家族成員因為某種詛咒，接連不幸去世。為了逃離這詛咒，艾迪被母親帶著離開了家。直到她成為最後一個家族成員，多年後回來探尋家族秘密。

由於芬奇家族的家很大，大部分家族成員都在裡面生活過。隨著詛咒一直籠罩著整個家族，讓家族成員總是意外的死去。而每一個家族成員的去世，剩下的家人便將他曾經住過的房間封鎖了起來。那名去世的家族成員留下的遺物，和曾經在這個家生活的氣息以及故事，便被封存在了各自的房間中，甚至長達幾十年都不會有人涉足。

為了給你流暢體驗整個故事，遊戲基本不會存在失敗重來的情況。而她攜帶的一個筆記本，在族譜樹上有每個家人的位置，以及生辰年月。玩家就能直觀的了解所探尋的每一個家族成員他們之間的親戚關係。

遊戲的玩法，像聽故事一樣。跟隨艾迪通過窗戶，密道等進入每一個家族成員的房間，來調查他們的死因和故事。由於家族成員去世後，房間被封存，所有的東西都得到了有效的保存，因此非常具有年代感。每一個成員的故事，則是遊戲最精彩的地方。無論是敘事手法，還是遊戲模式，都是經過精心的設計。

為了不給大家劇透的太多，我就選出兩個比較典型的家族成員故事，來告訴大家這遊戲的敘述設計有多麼用心。

例如當艾迪進入Barbara的房間，我們可以通過艾迪的自白了解到她的這名家人，出生於1944年，由於天資聰慧，在當時是紅極一時的童星。出演了不少電影。她的房間也充滿了各自電影元素。不過有意思的是你們看牆上的掛鐘，是不是有點眼熟。和我之前給大家講打鴨子那期遊戲中，那小男孩家裡也掛了同樣的鐘。這個鐘名為KIT-CAT KLOCK，是美國一個歷史悠久家喻戶曉的掛鐘。現在網上還能買的到，不過價格不菲。

Barbara的房間有一本驚悚漫畫。我們能通過這本漫畫，來了解她發生了什麼故事，以及她的死因。

通過一個將平面漫畫轉變動態，還讓玩家能在漫畫中操控角色。當邁克爾·邁爾斯的音樂響起，將氣氛推向高潮，短短几分鐘就講述了一個一波三折驚悚的故事。當我們知道Barbara的死因後，便在族譜樹上畫上了她的畫像和生辰。

如果你覺得這個設計還不是特別出彩，那我就給你看下一個。

艾迪的一個哥哥發生的故事。她的這個哥哥，有點精神問題。被安排在了魚罐頭廠工作。具體工作內容就是在流水線上，等魚來了，然後放入切割機切掉魚頭，進入下一步工序。大家都知道這種工作肯定枯燥乏味，拿著魚，咔，然後扔到下一個流水線上，就這樣一直重複。作為正常人大家在這種情況下有時候都會走神，而她的這個哥哥再加之有點精神問題，便開始了宏偉的幻想。幻想自己是一個勇敢的冒險者，在奇境王國冒險。

這個時候的設計特別的贊，你需要右手控制滑鼠一直反覆又麻木的完成切魚頭的工作。而左手操作鍵盤，控制冒險者在奇境王國的探險。一開始只是簡陋的線條式走迷宮，接著隨著她哥哥越來越走神，幻想越來越逼真。畫面從2D開始轉入3D俯視角。同時右手還一直在切魚頭，連我自己都控制滑鼠進入麻木的機械化操作。甚至我自己都開始走神。而思緒這一直盯著左邊的冒險歷程。隨著冒險的深入，她哥哥也和我們一樣，已經開始完全進入幻想世界了，早就忘了自己還在流水線上切魚頭了。這時候的幻想畫面也完全進入3D了，連我自己都感覺在玩一款新的遊戲。最後一路冒險，她哥哥已經完全入戲了。在萬眾矚目下，等待著奇境王國國王或者王后的皇冠加冕。但加冕前的斷頭台，暗示了他最後的結局。

我們得知了這一家族成員的故事和死因，再次記載上了族譜樹。

遊戲的玩法就是這樣，通過一個個房間探尋。每一個家族成員都有自己獨特有趣的故事，像是玩不同的小遊戲，你需要做的就是一步步把他們的故事和死因洞悉，然後記載到你的族譜樹上。最後調查出整個家族的故事。

因為遊戲的流程並不長，我不想給大家透露的太多。我希望如果你看到這裡感興趣了，非常建議你體驗一下這款遊戲，在一個寧靜午後，或是寂靜的深夜。感受這款遊戲帶給你的奇妙體驗，像是看了一本精彩有趣的小說和電影，讓人久久難以忘懷。

當然嘛，如果你不是對這類遊戲特別感興趣，或者趨於很多原因無法體驗。我也可以給你講述遊戲的完整劇情和結局。

500年來，在芬奇家族憑藉龐大的財富和不幸的詛咒，在挪威一直很有名。家族成員接連不幸死去，無一善終。迫使Odin Finch在1937年，無奈連房子一起連根拔起，試圖帶領家人離開此地逃離詛咒。不遠萬里來到大洋彼岸的美國。結果房子被海浪掀翻，Odin Finch不幸去世，只有部分家族成員倖存。從這就能看出，他們的家族詛咒並非是在哪個國家住，或者他們居住的建築造成的。應該是姓氏詛咒。只要你姓氏是Finch，你就會被詛咒所籠罩。哪怕倖存的家族成員在新地方重新建造家園，女性嫁入家族被冠以Finch的姓氏，就會接連死去。具體表現都是如死神來了一般，全是致命的意外。父親帶著艾迪去打獵，結果被垂死的獵物推下懸崖。哥哥放風箏，結果被海風吹翻的帳篷打死。小到甚至只有2歲的嬰兒都難以倖免。最後艾迪的母親帶著她逃離了家，但依然難以逃離這一悲慘的命運。

直到遊戲最後我們才知道，艾迪在遊戲中不斷稱述的自白，不是給玩家們聽的，而是給她即將出生的孩子說的，她在懷孕後後回到家裡，探尋了整個家族成員故事，包括自己的，一一記載了下來。

當孩子出生，艾迪也難產而死。所有姓Finch的家族成員徹底消失在整個世界上。我們才突然發覺，其實整個遊戲流程，不是艾迪調查了家族其他成員的死因故事，而是遊戲開篇坐船的場景，是艾迪的兒子。有一天他拿著母親的筆記本回到家，來到母親墓前緬懷。開啟了整個遊戲的所有回憶。

讓我們唯一值得欣慰的是，她的孩子不姓Finch，徹底終結了這一數百年來的詛咒。

感謝收看本期的不止遊戲，喜歡的觀眾朋友們可以點擊關注，我是喬伊，我們下期再見。
