打完結局想繼續玩？《外部世界》來年推出DLC延續劇情

天天要聞
2 天 ago

劇透警告 如果有人是開放式世界RPG的粉絲的話，那麼想必應該還記得兩個月前著名的黑曜石（Obsidian）工作室推出的最新力作《外部世界》（The Outer Worlds）。作為末日類RPG的元老級別工作室 ，這款以外星殖民作為背景的RPG遊戲推出之後也是很受歡迎。

雖然在畫面不是很突出，有的地方看起來很像十幾年前的畫質，但是在故事性以及撿垃圾方面，可以說是把某社的老冰棍按在地上磨擦了。不過很多玩家包括筆者自己都覺得主線有有點短 ，有一種「這就完結束了？我想要更多啊！」的感覺。而黑曜石應該也聽到了玩家的渴求，在Reddit上面表示明年會推出「富含劇情故事」的DLC。

圖片來源：《外部世界》官網

在剛剛過去的The Game Awards中，《外部世界》獲得了最佳敘事、最佳RPG、年度最佳遊戲以及最佳表演（Pravati的配音員Ashly Burch）共4項提目。雖然最終沒有獲獎 ，但是也絲毫沒有減低黑曜石的慶祝熱情。

圖片來源：《外部世界》官網

在Reddit的一篇帖子中，他們的社交經理表示，《外部世界》的劇情還沒結束，將會在2020年以DLC的形式推出，同時也表示感謝玩家的大力支持。雖然帖子中沒有DLC的詳情 ，但他們表示將會有之後的時間公布。考慮到《外部世界》將會有2020年初推出Switch的版本，所以有可能其DLC也會在這個時間段左右推出。

圖片來源：《外部世界》官網

以下為帖子原文:

我們想藉此機會感謝《外部世界》背後的不可思議團隊。正是他們的辛勤工作和對這個項目的奉獻精神，讓我們在The Game Awards中獲得了最佳敘事、最佳表演–Ashly Burch、最佳RPG和年度最佳遊戲提名。

對於所有投票支持我們的人，你們是最棒的，我們非常感謝你的支持。《外部世界》的服歡迎程度令人難以置信，甚至僅僅只是獲得提名就已經對我們來說很重要了。但是，這段旅程還沒有結束，我們很高興地宣布，明年我們將通過DLC擴展故事！詳細信息將在以後提供。

現在，我們的遊戲總監想和大家分享一則消息：

我們只想親自感謝我們的團隊的出色工作，以及我們的粉絲的大力支持。

Tim想說和我一起工作是一種榮幸。– Leonard

Leonard，現在我們的工作完成了，根據我們的協議，我希望你可以銷毀某些照片。– Tim

對此，筆者只想說，我那堆到2500+攻擊力的輕機槍已經準備好了，就等DLC推出了（笑）。
