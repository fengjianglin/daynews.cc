JKL的身價一路暴漲的背後，選手的實力和人氣哪個更重要？

2019-12-24 353條評論 1,870次閱讀 89人點贊

S9世界賽，FPX代表LPL賽區拿下冠軍之後，間接證明了LPL是世界第一賽區的實力。而明年的世界賽，將要在中國上海舉辦，這讓各大LPL的俱樂部都重視了起來。所謂強者如雲，只要戰隊能夠打好這一次比賽，對於俱樂部的發展是有非凡意義的，這也是為什麼S9的轉會期要比以往都更加激烈。

今年LPL俱樂部的轉會期的動作都很大，不少戰隊都想簽到更加優秀的選手，備戰下個賽季的常規賽，衝擊世界賽名額。而在這些職業選手的名單中，阿水作為一名自由人選手，又拿到了S8的世界冠軍，離開IG之後自然而然地遭到了各大戰隊的瘋搶。每個戰隊都想簽他，阿水也是試訓了所有隊伍，但是直到轉會期過去，都沒有傳出阿水具體的簽約消息。

目前最靠譜的猜測，是阿水想要通過此次德杯，來看清楚各大戰隊的實力，然後再選擇隊伍。畢竟阿水是一名自由人選手，可以在轉會期外加入任何一支戰隊，所以他擁有自由選擇權，完全不著急提前找隊伍。

不過最讓觀眾們不太明白的是，LPL的優秀選手那麼多，強勢的AD選手也不少，為什麼偏偏只有阿水遭到了各大戰隊的瘋搶。JKL的身價一路暴漲的背後，實力和人氣哪個更重要？首先，我們來看看阿水到底哪裡吸引了各大俱樂部的眼光。

實力強

首先，值得肯定的是，阿水是目前全世界頂尖的ADC選手，他的個人能力甚至不在Uzi之下，是國產AD的牌面。S8的冠軍AD，S9打進了四強，世界賽表現一直非常亮眼，哪怕是IG打得最差的S9夏季賽，也保持了良好的競技狀態。因為阿水是IG戰隊自己培養出來的選手，作為戰隊的「大腿選手」、「非賣品」、「代名詞」，原本是不可能被IG放掉的。只不過今年IG世界賽的成績不太理想，輔助寶藍一直被觀眾們所詬病，可是IG偏偏就沒給阿水找到合適的輔助，再加上阿水剛好在今年成為自由人，所以阿水就暫時離開了隊伍，這塊「香饃饃」流入轉會市場，不被各大戰隊瘋搶那是不可能的事情。因此阿水的身價被一路抬高，以至於高到有些俱樂部想買都買不起了。

人氣高

其次，阿水有著傳奇的經歷，小說都不敢這麼寫。原本是一名小主播，被粉絲一語驚醒，決心去打職業「揚名立萬」，17歲天才少年AD出道，在聯賽中豪取33連勝，只可惜隊友手傷飲恨春決，夏季賽終於殺進決賽，可惜倒在了如日中天的RNG面前。終於來到世界賽，又被分在了死亡之組，四強賽鏖戰五場，打敗了冠軍候選隊KT，一波閃現秒雙C奠定勝局，也打出了自己的名氣。結果還沒高興起來，又將面對老對手RNG，但是沒想到RNG倒在了八強最弱的隊伍G2面前，一波三折不知道是哭還是笑。後面的半決賽不出意外地呈碾壓之勢，先後戰勝了G2，FNC，出道即冠軍，還是S賽的世界冠軍，一瞬間就站在了世界之巔。至於阿水的稱號，源自於出道前，他直播時收穫了一大批技術粉，這些粉絲在貼吧到處說阿水有多麼厲害，貼吧老哥們以為是他請了水軍，就把JKL冠名了一個阿水的名字，水子哥的稱號也就是這麼來的。

年齡小

阿水是一名2000年出生的職業選手，從2016年加入到IG以後，在職業賽場上打拚了三年，都是待在IG戰隊。如今，阿水也才19歲，年紀也不算大，可持續性發展的空間非常足。一名職業選手的黃金年齡是16至20歲，不過也有些一些25、26歲的老將，依然堅持在職業賽場上，並且取得了很高的成就。原SKT的上單Marin，以及原SSG的打野Ambition，都是25歲才取得了S賽的世界冠軍，因此阿水起碼還有四五年巔峰期的職業生涯，這個是所有俱樂部可遇而不可求的事。如果不是今年IG戰隊的表現差強人意，阿水斷不會離開IG戰隊的，他可能會像聖槍哥一眼在一個戰隊堅持多年。而且明年中國本土的S賽，阿水依然在職業選手的黃金年齡之中，不會出現狀態上的起伏，加上他S9世界阿斯的穩定表現，也是各大俱樂部老闆所看重他的地方。

國產選手的身份

作為一名優秀的國產選手，阿水LPL國產選手的身份就決定了他的高身價。目前，LPL各大隊伍最缺的是優秀的中上野選手，國產的中上野又少又貴，而外援的名額只有兩個，在LPL賽區盛產優秀AD的情況下，一個戰隊性價比最高的配置，就是中上外援，打野和下路雙人組啟用國產選手。這也是這兩年LPL在S賽奪冠的標準配置。哪怕是堅持全華班的RNG，也買不到全LPL明星選手的豪華陣容，只能引援LMS賽區的選手或者是從LDL青訓營中提拔新選手。而且國產明星選手的身價狗很高，只要隊伍簽約了阿水，哪怕是後期再賣給其他戰隊，也能輕鬆賣掉，並且收穫一大筆轉會費，是一筆穩賺不賠的買賣。

最後還有一點，就是全球電子競技聯合會的成立，預示著國家性質的全球性英雄聯盟賽事要來了。在之前的亞運會比賽上，代表中國出戰的RNG、EDG和WE的選手，收穫了LOL首個冠軍金牌，這個冠軍的含金量可不比S賽低。而這個新的全球電子競技聯合會，正是連接英雄聯盟官方賽事和國家體育賽事的中轉站，以後就不光是賽區和賽區之間的比賽，而是國家隊與國家隊之間的比賽。因此，像阿水這樣又有實力又有名氣又是中國籍的選手，必定受到俱樂部老闆的格外關注，阿水本身就是除了Uzi之外最優秀的國產AD。

而IG儘力挽留阿水，也是在等這個全球英雄聯盟比賽，廠長轉型做教練也是有這個比賽的原因。全球電子競技聯合會的成立也說明了電子競技正被大眾所接受，是走向傳統體育賽事，走向亞運會和奧運會的里程碑。當國家隊招募國家選手時，首先是年齡過關，必須是中國籍的選手，其次就是實力強勁，人氣高不高只是他本身價值的一個佐證，反而沒有那麼重要了。

結合了眾多因素，阿水能夠達到這麼高的身價，每個方面的因素都佔了一點。在今年轉會期，同樣高身價的選手，Karsa他的人氣可能沒有阿水那麼高，年輕也偏大了，但是他的技術確實LPL的頂尖打野選手之一。而同樣粉絲眾多、人氣頗高的寶藍選手，如果把他放在轉會期市場上，估計沒有俱樂部願意要吧。電子競技從來都是以實力說話，成績能代表一切，阿水作為全球頂尖AD之一，兩年一冠一四強，這才是各大俱樂部瘋搶阿水的原因吧。
