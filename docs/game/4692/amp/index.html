官方暗示毒池上線？雙生視界更新泳裝池保底 冬泳怪鴿竟是福利

天天要聞
2019-12-05

摘要

隨著雙生視界開服活動的結束，遊戲官方也終於確定將在明日上線名為“獻給愛麗絲”的泳裝活動，同時開啟了新一期的泳裝卡池。不過作為在常駐女僕池和戰術池之後的第三個卡池，因為泳裝朱諾的強力表現而讓不少玩家開始擔心起卡池出貨幾率起來。

隨著雙生視界開服活動的結束，遊戲官方也終於確定將在明日上線名為「獻給愛麗絲」的泳裝活動，同時開啟了新一期的泳裝卡池。不過作為在常駐女僕池和戰術池之後的第三個卡池，因為泳裝朱諾的強力表現而讓不少玩家開始擔心起卡池出貨幾率起來。

而從這次的卡池來看，泳裝朱諾、泳裝伊珂絲以及泳裝蘇小真無疑是很多抽卡的目標，但是因為之前戰術池經常會50抽保底出現「女僕時無暇」的大獎，所以這次瓜雙生視界官方也更新了全新的保底機制。

根據雙生視界的官方公告來看，遊戲將在「獻給愛麗絲」的卡池中增加了100抽的全新保底機制。除了保留原本50抽必出4星人物的機制以外，在玩家達到一百抽之後，則會觸發新的保底機制，此時出貨的4星人物必然是泳裝朱諾和泳裝伊珂絲，可以說是對之前戰術卡池女僕時無暇保底的一個優化。

不過對於這次新加入的機制，部分玩家表示非酋仍然會出現沉船的概率。尤其是對於遊戲中的強度黨而言，泳裝朱諾的獲取幾率依舊是很低，甚至有玩家的覺得雙生視界在12月份開啟泳裝活動就非常離譜，完全有了「冬泳怪鴿」的味道。不過個人認為，雖然這次泳裝卡池開放的非常不合時宜，但是在卡池保底的機制上官方卻做出了比較良心的修改，獲得了不少玩家的點贊。

除此之外，因為雙生視界節奏榜的出現，不少玩家開始吐槽遊戲中存在星級欺詐的現象。尤其是在戰術千姬上線之後，關於三星私服時無憂和四星戰術千姬的破盾爭論就沒有停過。加上版本中輸出第一的人物是戰術時無暇，更是讓不少剛入坑的玩家感到疑惑不已。

不過個人覺得，在雙生視界中四星卡僅僅是屬性比三星卡更高一些，具體人物是否強力，還需要應對地圖的怪物屬性以及武器種類的選擇來看。無論是「戰術三巨頭」還是「T0時無憂」的說法其實都過於片面，通過不同的搭配來攻略副本和養成人物才是該遊戲的本質所在。
