19賽季單人天梯周報vol.4：請天使開始你的表演

2019-12-24 1,448條評論 3,452次閱讀 1512人點贊

今天為止，暗黑3的19賽季已經到了第四周，遊戲中也有四個職業單人打穿了150層，其中的野蠻人更是在天使的幫助下取得了8分40秒的極限成績，其餘職業天梯榜首也大多採用了捨棄受罰利用賽季500連殺天使擊殺BOSS的極限思路，一起來看看本周的天梯變化吧。
野蠻人：再快點都可以打兩把了

本周的野蠻人天梯新增了7位突破150的強者，排名首位的 松鶴灬 更是以8分40秒的極限成績穩居榜首。

相比之前的荒原主動撕裂沖層思路，松鶴灬 選擇將撕裂換成祭刀符文用以維持生存，將戰鬥怒火換成提供暴擊幾率的怒火中燒最大化輸出，配合捨棄受罰帶上賊神的取捨，最大化了進度階段的輸出能力，從而更快的打死小怪觸發賽季BUFF清場。
聖教軍：並不是最快的，卻是最多的

和有著鉤子聚怪拉怪非常方便的野蠻人相比，聖教軍在本賽季的天梯極限成績略遜一籌，但該職業通關150層的人數卻是全職業最多的——畢竟基礎傷害擺在那裡，聖教軍也是全職業中惟一一個帶著受罰打150層的職業，畢竟就算天使沒來，他也有機會靠自己的輸出擊殺150層的BOSS。

當然，聖教軍中排名第一的 Rhapsody 還是選擇了捨棄受罰帶上增痛，最大化進度能力，而BOSS則交給500連殺的天使處理。
聖教軍大神 諧星 也利用天使順利通關了150層，他遇見的BOSS是急凍魔，天使在陣亡之前足足幫他打掉了BOSS80%的血量。

獵魔人：賽季BUFF的極限運用

本周獵魔人的天梯榜首依舊是 淺唱大大 他利用散件連射打穿了150層，這也是賽季獵魔人天梯唯一的150層成績。

在上周的周報中我對她的成績表示了懷疑——當時正處於賽季BUFF遭到惡意利用的時間段，而她當時的成績過於碾壓。本周這位玩家在凱恩分享了她的150層實戰視頻，對於之前的懷疑本人表示致歉。
武僧：猴王還是正義，誰才是最後的勝者？

本周的武僧天梯變化不大，首頁玩家依舊使用著猴王風雷衝進行沖層，而亞服玩家在上周則利用正義風雷沖打穿了150層。

國服天梯第一的依舊是 重新來過 ，這是他在上周留下的記錄。
亞服玩家則利用正義套可以兼容增痛的特色，用這顆寶石換下了受罰，順利通關了150層。

由於攻速可以提升風雷沖的傷害，增痛帶來的額外攻速加成大大提升了武僧的輸出能力，而大量的攻速意味這大量的內力消耗，此時武僧新套裝正義套的50點內力恢復就顯得尤為可貴了——正是因為恢復抵不上支出，猴王才難以帶上增痛來最大化輸出。目前在普通模式下由於必須攜帶受罰的緣故猴王還佔據著優勢，而賽季模式下控BUFF召喚天使殺王的思路使得帶上增痛的正義套表現更好。
死靈法師：新星綻放

本周的死靈天梯榜首玩法換了人：在賽季BUFF的支持下，散件新星玩法憑藉強大的進度能力順利的衝到了無法順暢利用賽季BUFF的散件冰槍頭上。

成功通關135層的 kk86 使用的是主動散件新星的玩法，19賽季500連殺的天使為這個進度階段強大而BOSS階段羸弱的玩法創造了通關的可能，在散件冰槍不太方便利用賽季BUFF的當下，也許新星才是死靈法師的出路？
巫醫：活下去才能更好的輸出

本周的巫醫天梯依舊是層次分明的天梯，首位的 涵雪飄綾 在改變打法之後順利擊破了142層。

和上周相比，他用畢業的收割刀換下了怒火回蕩，而後將奧吉德套裝換成了賦歸+拉昆巴的腕飾的組合，提供了更高的固定減傷，看來活著才是輸出的保障。
魔法師：口徑即是正義

本周的魔法師天梯迎來了一個大躍進：在兩位奧隕選手參與天梯競爭之後，原本困頓於136層的魔法師在本周打穿了150層，這使得魔法師成為了國服第四個打穿150層的職業。

排名第一的 Goout 使用的是散件奧隕，和以往不同，這個玩法最大的短板BOSS站被賽季的天使彌補，而奧隕本身強大的輸出力度也使得他在觸發賽季BUFF方面沒啥難度，去掉受罰換上的活力寶石更是保證了法師不會在丟隕石的時候被怪物突然打飛。
19賽季的天梯衝層註定是和之前完全迥異的節奏，強力的賽季BUFF成為了奈非天們手中的利器，只要有擊殺小怪的能力，你就有機會觸發賽季BUFF，從而打出進度通關大秘境，這也使得一些AOE的玩法得以重獲新生，死靈法師的散件新星玩法就是最好的證明。
