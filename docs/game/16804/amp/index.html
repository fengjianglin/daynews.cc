魔獸世界懷舊服：盜賊主手越慢越好？其實先看秒傷 然後再看速度

天天要聞
2 天 ago

魔獸世界懷舊服影響盜賊單體DPS一直是盜賊玩家驕傲的點，但是有不少玩家發現面對同等級精英怪的時候怎麼自己卻變成了怪，全程切割也提前開閃避，甚至還要吃大紅葯，卻依舊難逃倒地的命運，更為嘲諷的是，精英怪還還剩下一半的血。出現這種情況的原因除了天賦之外，主手武器的選擇也是重中之重。盜賊主武器該看什麼？速度是一方面，主要看上限。

盜賊主手越慢越好？

其實是先看秒傷，然後再看速度。這個問題不在於不是主手武器越慢越好，是主手上限越高越好，但問題來了，上限高的主手武器就必定慢，而慢的主手武器卻不一定上限高，速度並不是決定盜賊傷害的關鍵。比如有一把的匕首和一把劍，都能打出100傷害，那麼是匕首好還是劍好呢？很明顯是匕首，因為匕首更快一點。

我們換個解釋

相同秒傷下選速度慢的主武器准沒錯，說上限也沒問題，只不過因為盜賊主慢副快的原則大家都知道，這樣說起來更容易被大家接受。100傷害的匕首和劍打出來的白字傷害不一樣，問題在於受到ap加成不一樣，相同傷害下也是速度越慢普攻傷害越高，速度越慢ap加成越高，不過加成一般，所以100傷害的匕首要比劍好。

再來說說天賦

盜賊從開始就需要優先點戰鬥，一直奔著開劍刃亂舞去點，這樣下來才能打過同等級精英怪，而且練級路的時候建議點出還擊，在打帶武器的怪的時候非常有優勢，並且還擊這個技能本身也是個只需要10能量便能打出優質輸出技能。武器本著主慢副快的原則，主手痛擊之刃、副手征服者之劍的配置，可以用到雷德雙刀或者克羅與米拉之歌。

再補充一點

拍賣裡面值得入手的武器有屠夫，不僅好用而且很實用，後面就是克羅。升級的過程中帶著屠夫，征服者和預兆，還有痛擊加預兆，再到後面就是克羅和半藏，克羅與米拉之歌。有一點要注意的是不要在武器和天賦上弄亂了自己，痛擊和克羅之前的武器都無所謂。
