VG德杯迎來首勝！IBOY三槍一人頭，SN失利只因少一個JKL？

天天要聞
2019-12-21

萬眾矚目的2019德瑪西亞杯近日如期舉行，作為轉會期後的第一場賽事，本次德杯也是受到很多玩家的期待。在輸掉新賽季的第一場比賽之後，重建的VG戰隊遇上了同樣陣容改變較大的SN戰隊。

不知道是不是因為首場比賽的失利，VG戰隊在這場比賽中拿出了一套偏發育的英雄。不管是上路奧恩還是下路的賽娜，前期作戰能力都偏弱。反觀SN這邊，雖然中野的對抗性並不如對面，但下路由於泰坦的存在，很容易配合酒桶打出優勢。

讓人意外的是，在大家普遍看好SN下路組合時。Iboy在這場比賽上站了出來，前期不到7分鐘的時間裡，對線直接打出了20個補刀的壓制。我們都知道，新英雄賽娜的消耗能力非常強，iboy也是利用這個優勢不斷進行消耗。完美的走位和精準的技能，讓女槍和泰坦這對強勢組合，直接被壓制在塔下。

除了下路表現出色之外，VG戰隊的上單在面對劍魔的時候，利用對方的失誤成功拿到了一血。要知道劍魔對線奧恩優勢很大，在這樣的情況下被單殺失誤確實比較大。隨後VG戰隊不斷找到節奏，利用團戰前的消耗，成功拿下四條小龍獲得龍魂BUFF。

最後憑藉大龍團戰的勝利，VG戰隊攜帶大龍BUFF直接一波推掉了敵方水晶，拿到了新賽季的首勝。在這場比賽中我們不難看出，VG能夠獲勝除了整體節奏完美之外，iboy更是功不可沒。

不得不說，兩場比賽下來雖然VG有輸有贏，但iboy卻有著驚艷的發揮。不得不說，在離開EDG之後，這位年輕的小將或許將迎來自己的第二春。

但對於SN戰隊而言，兩場比賽打下來，下路的劣勢暴露的非常明顯。在這樣的情況下，不少粉絲直言SN要想在新賽季獲得好成績，真的缺少一個阿水JKL。如果最後JKL沒有選擇加入SN，那麼這支隊伍在新賽季的成績不容樂觀。小夥伴們覺得JKL最後會加入哪支隊伍呢？
