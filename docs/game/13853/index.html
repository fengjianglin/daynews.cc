當LOL新英雄瑟提大中我方坦克時，後排脆皮跑吧！否則瞬間躺地板

2019-12-12 62條評論 1,028次閱讀 82人點贊

LOL新英雄瑟提在今天凌晨正式發布了，關於他的背景、技能以及玩法機制相信大家已經通過各種渠道了解一二了，然而對於瑟提來說，大家還是很在意他的大招效果，國服將他的大招命名為「嘆為觀止」，大致的效果就是將人扛起來然後重重的摔到地上，著陸點會有AOE效果。

動圖演示：

根據技能描述來看，這個技能的落地傷害是根據被擒英雄的生命值來計算的，那麼也就說說被擒的英雄生命值越高，落地的傷害就越大。

為此外服UP主就進行了一次實驗，分別扛起血量不同的科加斯，看看落地傷害究竟有什麼不同。

第一次測試，科加斯血量6745

第二次測試，科加斯開啟石像鬼板甲，血量飆升到12300

第三次測試，科加斯開啟石像鬼板甲，血量2162/12340

第四次測試，石像鬼板甲開啟，科加斯血量10450

從幾次的測試結果我們不難發現，瑟提的大招傷害確實跟被擒英雄的最大生命值有關，但是為什麼在最後兩次的測試中，血量在相差了1800左右，傷害卻差了很多呢？

其實這主要是因為技能描述中的這麼一段話：

也就說是當瑟提抱起一個英雄之後，摔他的著落點離敵方其他英雄越遠，那麼傷害也會越低。所以這一條也讓我們謹記住了一點：當瑟提進場對你的隊友使用大招後，儘快遠離被擒的隊友，尤其是血量如此高的前排坦克。

在過往的團戰中，ADC英雄大都是站在坦克身後輸出，但是隨著瑟提的上線後，ADC英雄的生存環境就更加堪憂，一萬血的大蟲子在前面幫你阻擋無情的炮火，結果瑟提一個進場過街摔，把大蟲子摔到你頭上，然後你就沒了！

當然這個大招能不能規避，如何解掉？還需進一步的測試。
