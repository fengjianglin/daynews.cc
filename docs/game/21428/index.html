江湖再無傑克，江湖卻少不了傑克的傳說｜《無主之地3》DLC體驗

2019-12-20 2,554條評論 4,853次閱讀 438人點贊

作者：大巴車司機

《無主之地3》的大型DLC「帥傑作搶奪大作戰」（Moxxi’s Heist of the Handsome Jackpot ）已經在12月20日正式上線，在這次的故事裡，莫西叫上我們的密藏獵人前往帥傑克的太空站賭場，奪走帥傑克留下的遺產。

量有夠足

作為《無主之地3》的大型DLC，內容的量還是足，整個主線故事的時長約6~8小時，加上支線任務、新的挑戰收集與新武器，在傑克哥的賭場里估計得呆10小時以上那是輕鬆得很。要注意的的一點是，DLC「莫西的帥傑作搶奪大作戰」至少需要到達13級，故事流程解鎖至避難所3號。

由於相當於來到了一個全新的星球，整個空間站並不是只有賭場，賭場內部、紅燈區、地下垃圾都市、工廠區等等相互連通，雖然缺少其他星球中寬闊的戶外荒野，但也正是因為在空間站內部，所以沒有了開車趕路的煩惱。作戰區域密集緊湊，咱們基本端著槍一路RUSH就完事了。

江湖再無傑克哥，江湖卻少不了傑克哥的傳說

簡單說說故事與流程，在避難所中可以看見帥傑克賭場的海報，靠近後可以看見這次的DLC主線任務目標，任務描述中莫西說，我們要做的是闖進帥傑克的賭場，發大財，然後溜之大吉。真的會有這麼輕鬆的活嗎，這事，肯定沒那麼簡單。

出發來到帥傑克的賭場「帥傑作」後，立刻就會有一段由帥傑克親自出演的賭場廣告，相信此時此刻，傑克哥的音容笑貌與過去的美好回憶，已經出現在了各位腦海中。由於傑克哥已死，傑克哥的賭場自然也成為了無法地帶，迎接我們的除了困在這裡的法外之徒，還有數不勝數的守衛機器人。

那除了能在廣告與石碑上看見傑克哥的那張臉，我們還有其他見到傑克哥的機會嗎？就是那種又走心還玩真實的那種機會。相信很多朋友也是抱著再見傑克哥一面的想法，來到了這個DLC。故事開場沒多久，就能為大家揭開這個心中的疑問，這是個暫時還沒有出現在DLC預告中的人物，他是傑克又不是傑克，也是這次故事的核心角色。

很快我們就會面對這個DLC故事的反派「靚仔哥」。在傑克哥不在的日子裡，靚仔哥掌控了大半個賭場，想要奪走賭場，突破重重難關，我們還得在賭場空間站里找到新的夥伴，打造特攻小隊。

特攻小隊的幾位夥伴已經在之前的預告中露面，他們分別是來自紅燈區的縱火大師，打扮辣眼的機器人工程師，還有地下垃圾場中烏托邦小城的市長。

DLC的簡體中文配音依然充滿了粗鄙之語。個人感覺，相比本體似乎更加放得開素質更低，崽種、牛逼之類的辭彙不絕於耳，非常契合整個DLC的氣氛，一些武器的紅字特效同樣看著令人印象深刻。

一杯二鍋頭，嗆得眼淚流

再說點有的沒的，在逛傑克哥的賭場時，這個傑克哥的大頭經常會讓我想起《猛獸俠》里威震天的大腦袋；賭場的紅燈區中不僅有男有女，還有機器人；垃圾山谷中會出現成群廢棄小吵鬧們；賭博性質的21點寶箱······當然還有很多有意思的細節和任務，等待各位去體驗。

小結

DLC「帥傑作搶奪大作戰」充滿了歡樂、江湖恩怨以及小小的感傷，對於喜歡快意恩仇外傳故事、想知道更多關於帥傑克小秘密的朋友絕對值得體驗。有一說一，帥傑克確實是《無主之地》系列中的經典人物，這回我們在帥傑克的賭場中冒險，今後會不會再有機會去探索帥傑克留在宇宙中的其他遺產寶藏？
