狼人殺遊戲中炸彈人有多bug？別惹我，我生氣起來自己都炸

天天要聞
2019-12-09

摘要

自從經典版狼人殺火了後，很多綜藝節目都在引進這個群體性質的社交遊戲，以增加節目的娛樂性和收視率。相應的狼人殺也延伸出很多不同的板子，比如公認的狼人殺2.0版，即帶有新月事件的那版，這也是官方承認的一款，其中加入了很多娛樂性角色，比如丘比特（愛神）、狼美人、白痴等。

自從經典版狼人殺火了後，很多綜藝節目都在引進這個群體性質的社交遊戲，以增加節目的娛樂性和收視率。相應的狼人殺也延伸出很多不同的板子，比如公認的狼人殺2.0版，即帶有新月事件的那版，這也是官方承認的一款，其中加入了很多娛樂性角色，比如丘比特（愛神）、狼美人、白痴等。

後來這個版本繼續擴展，便有了炸彈人這一角色（一說炸彈師），作用就是在白天如果被票出局，可以帶走所有投他票的人，從技能上看著絕對是一個強神。狼人和好人都可以跳炸彈師來保命，不過也許容易被女巫和獵人在夜間帶走，如此一來炸彈人就無法發動技能了。因此證明了那句話，神職有風險，悍跳需謹慎！

從角色劃分上看，炸彈人是屬於好人陣營的；從技能上看他又像個不分善惡的流氓，誰票誰死，從而延伸出了一種特殊的勝利規則，即當在場所有人都將炸彈人票出局，那麼炸彈人獲勝，不分好人壞人。

這麼看來炸彈人似乎是個「局外人」，也很少有正規狼人殺比賽去使用這個角色，但炸彈人的出場率在《狼人對決》中卻很高。作為一個中立勢力陣營角色，他可以和任何一個勢力組卡牌搭配。不過小編還是建議把炸彈人加入到白鴿盟勢力。

理由有2點：

1. 都是無腦式擊殺流，同根同源，用起來很有帶感。炸彈人本來就是出場即亡，被翻開後拼點意義不大，且配合其他勢力如黑袍會這樣的獻祭流就很虧，完全發揮不出技能優勢；

2. 能夠和小女孩這樣的粗暴尋狼牌結合，威懾對面。小女孩是犧牲自己使對方一個狼人暴露，炸彈人是犧牲自己驅逐對方一個潛伏牌，看起來都是一換一，可用起來明顯對面更難受。尤其是同時上2張小女孩和炸彈人，在不被翻開暴露的前提下，至少對面兩頭狼不愁找不到，而且炸彈人隨機帶走2個潛伏契靈，就讓你的對手哭會吧。

遺憾的是《狼人對決》的炸彈人明顯和狼人殺里的級別不同，前者只能一換一，若是運氣好懟到狼牌身上絕對是賺了，或者帶走一個4、5點這樣的高契約值契靈也不虧，因此炸彈人的使用多少看運氣。不過純粹地削減對面手牌，或者針對那些利用預備守衛反覆蓋回暴露牌的玩法，炸彈人絕對是個不錯的選擇。

怎麼樣，狼人殺和《狼人對決》中的炸彈人，你更喜歡哪個呢？
