聊聊關於街機廳的故事！即便是SNK如今也沒了當初的雄心壯志

2019-12-23 2,061條評論 4,078次閱讀 1927人點贊

街機廳一角

可能現在提起街機，多多少少有那麼些過時，畢竟現在的街機廳實在太少了，即便有也都充斥著抓娃娃機，捕魚達人這些。或許最街機的，反而成了擺放在門口的那台跳舞機，只要有小姐姐在上面跳，還是蠻養眼的。

有了網路遊戲，足不出戶就能享受社交的樂趣。有了主機遊戲，各種獨佔大作，也能讓玩家大呼過癮。PC遊戲玩家人口基數眾多，並且現在除了Steam，其他平台也逐漸走上正規。至於掌機，儘管索尼宣布暫時退出競爭，但Switch依舊錶現出色，給掌機留下了希望，儘管不那麼純粹就是了。

就連手機遊戲，都已經發展的如火如荼，這不禁讓我們懷念起來，街機怎麼就沒落了呢……

說起來，曾經的街機廳有一種頗為美好的社交方式，玩的人和看的人，可以面對面的交流，誰玩的好，自然眾星捧月，玩的不好的，也會通過指點慢慢進步。

SNK的眾多街機遊戲

那時候，卡普空的動作清版遊戲，SNK的格鬥遊戲，都還算受歡迎。說起來，駕駛，光槍，也都算是百花齊放。

記得類似《名將》、《三國》、《圓桌騎士》的各種技巧啊，秘笈啊，真的滿天飛。尤其是《圓桌武士》，在當時真的不知道那些小朋友是怎麼知道的，甚至有的地方還要先損一條命，接下來才能出寶，真是難以捉摸。主要在當時，消息也非常不流通，可不像現在，雜誌都不用買了，直接在網上就能搜到答案。

還有一點，就是大家喜歡在平時聊這些遊戲，這多少也與消息不太流通有關，隨便一點小事，就能成為新聞。有人說什麼遊戲過了第幾關，別人不信，就打賭的事情常有發生。至於打過了，還是因為緊張等原因沒打過，都成了日後的談資。

隨著街機廳的沒落，那種美好的社交方式也隨之消失，你再愛玩，玩得再溜，旁邊也很難有觀眾了。你轉過頭，望著空氣，似乎下次也不太想要來這裡了。

電競格鬥遊戲也算延續了街機的種子

當然了，不得不說，街機的消費方式，是大多數玩的不太好的玩家的噩夢。街機的難度，往往都被設定的比較高，如果細算下來，其實並不省錢。

問題是，缺失了話題性，缺失了氛圍，玩家也很難再不停的用金錢去磨練技術，選擇模擬器，似乎也穩妥了很多。

還有一點，就是街機遊戲的品類，基本都是快節奏的遊戲，也沒有存檔和養成的機制，這對於如今的玩家們，多多少少還是難以接受的。所以，儘管都是沒落，街機廳沒落的太徹底，而網吧仍然還有其可以施展的地方。

至於在當時，不管是任天堂，還是NeoGeo，初衷都是要把街機遊戲搬回家，這多多少少，也讓街機遊戲和家用機的界限，沒有那麼清晰了。至於任天堂當時的社長山內溥，據說還下令任天堂全面停止街機相關的研發，認為街機的生命力並不會太久。

NeoGeo

至於NeoGeo，個人認為，是把街機帶到家裡最為徹底的執行者了，因為這台主機不但用了當時SNK的街機基板MVS，就連操作設備都選用的是街機搖桿。

紅白機當時把街機遊戲帶回家的說法，山內溥的心思是《大金剛》這樣的作品，另外，也對研發者上村雅之提出了價格上的要求。也就是說，價格不能高，還得能實現設想。所以任天堂的紅白機，其實是放棄了很多的，相比NeoGeo來說。

不過不放棄，帶來的自然是成本的上揚，在當時這台主機的日版售價為58000日元，即便是任天堂的超級任天堂主機，在當時的售價也只是25000日元。如果說，主機貴，算是一次性投資，後面卡帶的價格低，玩家們多少也能買賬。但沒想到的是，NeoGeo的卡帶，一盤要賣到30000日元，也就是說，一盤卡帶的錢，就足夠買其他品牌的主機了。

這個問題，其實SNK也意識到了，後來他們用CD-Rom當成媒介，推出了NeoGeo CD版本，並且操作設備也有一定的縮水。但問題並沒有徹底解決，儘管卡帶的價格下降到了7000日元檔位，但由於CD-Rom完全不具備卡帶的快速讀取優勢，這大大影響了玩家的心情。

SNK的侍魂當初也頗有人氣

也就是說，玩家並不認可這次改良，而之後SNK為了賺錢，也不得不將自己旗下的遊戲搬上其他主機平台。

和眾多廠商打算在主機領域掀起風浪不同，SNK仍然在為了街機的事業努力，他們做的新基板，以此來實現3D顯示的功能，不過最終，SNK自己都不怎麼願意開發3D遊戲，就更別提其他廠商了。

在印象中，街機流行的時間真的非常短，如果諸位誰有關於街機的故事，也來聊聊，或者是喜歡哪款街機遊戲，自己家門口的街機廳狀況，都歡迎分享。
