CFS丨從小組賽到淘汰賽階段 雅哲的華麗蛻變令我們驚嘆

2019-12-12 3,005條評論 4,080次閱讀 2927人點贊

CFS2019世界總決賽在第四比賽日已經進行到了淘汰賽階段，中國的SV戰隊第四個出場，他們的對手是埃及ANB戰隊，這兩支戰隊此前在CFSI埃及邀請賽中交過手，此次又在CFS世界賽八進四中遭遇，最終SV戰隊以2-0擊敗ANB晉級四強當中，恭喜SV戰隊！

SV在對陣ANB，首圖打得非常順利，次圖雖然遭遇ANB的頑強抵抗，但是仍然獲得了勝利，晉級四強，最大的功臣不是隊長馬哲，不是念舊，而是加入到SV戰隊沒多久的雅哲，兩張地圖一共完成36個擊殺，其中首圖就拿到了21個擊殺，是SV挺進四強的絕對功臣。

對於雅哲來說，這是他的第二次世界賽，上一次世界賽之旅還是2018年的CFGI國際邀請賽，時隔一年半，雅哲再次登上世界賽的舞台，雖然有過參加世界賽的經歷，那也是很早之前了，在世界賽面前，雅哲就是一個新人。SV勝者組遭遇VG，那一戰相信雅哲一直會記得。

勝者組，SV和VG狹路相逢，獲得的晉級到八強之中，然而SV卻以5-10輸掉了比賽，雅哲的戰績全場為5-13，尤其是下半場的0-6格外扎眼，很遺憾沒能直接通過勝者組晉級到八強，小組賽雅哲的狀態不好，也難免會被議論，背受著質疑，好在SV最終還是成功晉級到了八強。

進入八強後，每一場比賽都可能是最後一場，在八進四中的比賽中，從一開始雅哲就很快進入到狀態之中，撐起了隊伍的輸出任務，第一張地圖可以說是雅哲的封神之戰，21個擊殺，幫助隊伍以10-4取得勝利，次圖ANB狀態回升，上半場比分膠著，攻守互換後，又是雅哲頻頻打開局面，幫助隊伍連續得分，總比分2-0擊敗ANB戰隊，取得八進四的勝利。

在觀看比賽中，我已經數不清解說說了多少次「又是雅哲，又是雅哲」，在隊伍需要他的時候，作為新人他能夠挺身而出，這就是蛻變，這就是成長。賽後採訪，馬哲也說到，令自己開心的是，隊伍中的新人雅哲今天打開了，無論是解說還是隊長，都給予雅哲充分的肯定。接下來的半決賽，雅哲，繼續加油！
