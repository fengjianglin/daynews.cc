《拳皇2000》還有這些不為人知的秘密？當年的隱藏角色你都見過嗎

天天要聞
2019-12-21

2000年，老SNK已經進入了關閉倒計時，而本年度推出的經典格鬥遊戲《拳皇》的第七代作品——《拳皇2000》，則更像是SNK跟拳皇玩家們之間的一種特殊的告別。現在看來，這部作品給玩家們留下的，並不是在遊戲性上做出了多麼大的突破，也不是在畫面上有了多麼大的革新，而是在援護系統中為玩家們帶來的那份驚喜。

之所以這麼說，是因為本作在《拳皇99》的基礎上對援護系統進行了一個較大的改動，考慮到SNK當時已經發布了許多經典的遊戲形象，所以在原本單個援護的基礎上，SNK為每個角色還單獨添加了一個「第二援護」，他們都是曾經SNK旗下的經典角色。

但是如果你以為老SNK只做了這些的話，那你就大錯特錯了！遊戲與玩家們最大的默契是什麼？不用說，也是遊戲中的隱藏和彩蛋啊！再加上SNK的經典角色太多，第二援護根本放不下，這也使得「隱藏援護」應運而生了。那麼小編我今天就來帶大家一起看看當年《拳皇2000》中出現過的那些隱藏援護角色吧，不知道這麼帥的陳國漢大家有沒有見過呢？

對了，忘了告訴各位隱藏援護角色的選法了，其實我們只需要在選擇援護角色時，先調到該人物對應的第二援護下，然後依次輸入指令「↑←←←→→→↓」，就會出現該角色的隱藏援護啦。

一、拉蒙——Neo&Geo

Neo&Geo是SNK於1991年發售的智力解謎遊戲《解謎大搜查線》中的2位主角，他們在遊戲中是以私家偵探的形象登場，並熱心幫助人們排憂解難。

二、坂崎亮——G-Mantle 小丑

這個角色最早出現在老SNK的家用機「NEOGEO」的宣傳海報上，之後一直扮演著其形象代言人的角色，而且在SNK的一眾遊戲中，G-MANTLE其實是一個著名的龍套帝。

三、藤堂香澄——藤堂龍白

藤堂龍白是《龍虎之拳》中的角色，與極限流的坂崎琢磨是老對頭。而他也是藤堂香澄的父親，在踢館時被坂崎亮打敗之後意志消沉，認為自己的實力還遠遠不夠，於是給家人留下了一封信，之後便不見蹤影。

四、陳國漢——Smart-Chang（型男陳國漢）

五、蔡寶健——Cool-Choi（變高的蔡寶健）

六、草薙京——草薙柴舟

前任草薙流繼承人，人氣角色草薙京的父親。《拳皇94》中被盧卡爾擊敗並洗腦，在之後的《拳皇95》中作為小BOSS出現，結果被兒子草薙京輕易擊敗……

七、八神庵——風衣八神庵

八、KULA——盧卡爾

Emmm，我是實在想不出庫拉和盧卡爾究竟有什麼關係。

不僅如此，在《拳皇2000》的PS2版本中，官方依舊「加量不加價」，為我們奉上了10個新的隱藏援護角色：

特瑞——布萊恩·巴特勒

初次登場於《拳皇94》中，是美國隊的參賽選手之一，可惜在《拳皇95》開賽前被八神庵的宿敵隊擊敗，並搶走了邀請函，《拳皇96》還沒緩過來，《拳皇97》又被七枷社領銜的新面孔隊搶走了邀請函，真的好慘……

安迪——洛奇古洛巴(LUCKY)

原為職業籃球員，後來因對籃球失去興趣而學習空手道，並挑戰美國各地格鬥家。現役全美空手道冠軍。與布萊恩·巴特勒一樣，就一個字，慘……

東丈——哈維·D!

美國隊隊長，黑人拳擊手，在一次拳擊比賽中失手殺了比賽對象，令很多拳擊手都不敢和他對打，因此非常寂寞，想要到拳皇大賽中尋找真正的對手，至於在遊戲中的經歷嘛……同上，就不贅述了。

紅丸——暴走八神庵

拉爾夫——暴走莉安娜

不知火舞——克勞薩

沃爾夫岡·克勞薩，是《餓狼傳說》系列的BOSS人物，還是另一名經典反派吉斯·霍華德同父異母的弟弟。初次登場於《拳皇96》中由吉斯領銜的老闆隊中。

坂崎琢磨——Mr.BIG

《龍虎之拳》中的經典反派，南鎮最大的黑幫組織的幹部之一，和吉斯常年明爭暗鬥。是綁架坂崎由莉的幕後黑手。與克勞薩一樣，在《拳皇96》的老闆隊中初次作為可選角色登場。

薇普——哈迪倫

莉安娜的養父，初次登場於SNK的一款動作過關遊戲IKARI（怒）中。

莉安娜——大蛇

「地球意志」，「拳皇最強」

K`——古利查力度（Krizalid）

NESTS篇中登場的第一個BOSS，一直以為K’是自己的複製體，但事實剛好相反。他也是草薙京複製人計劃的幕後黑手，在《拳皇99》中被擊敗，後被真·ZERO救走，但也失去了記憶。
