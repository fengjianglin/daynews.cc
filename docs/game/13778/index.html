這廠商目標有多大？做個社會模擬器卻叫遊戲，無數人因此成大佬

2019-12-13 280條評論 1,521次閱讀 541人點贊

在大家的印象里，遊戲就是娛樂的產物，是用來玩的對吧？雖然這個理念大家都懂，但並不是每個廠商都會願意這麼做的。有些廠商的產品雖然叫做遊戲，但玩起來可真的一點都不像遊戲。就好比之前比較火熱的《太吾繪卷》，這玩意你說它是遊戲吧，它玩起來賊費勁，但你說它不是遊戲吧，渡過新手期後又賊好玩。

然而《太吾繪卷》雖然稍微奇葩，但總的來說還是屬於遊戲範疇的。而有個遊戲則不同，這玩意表面上看是遊戲沒錯，但玩起來那可真不是遊戲，簡直都快成社會模擬器了。這個遊戲是啥呢？它就是《EVE》。一個影響之後星戰類MMORPG發展的遊戲，像清潔員玩的《第二銀河》就一定程度受到了它的影響。

《EVE》的開發商CCP雖然是冰島一個名不見經傳的小廠，但它的發展方向和其他公司完全不同。其他公司都想著怎麼讓自己的產品火起來，而CCP不同，它則是想到什麼就做什麼，而《EVE》就是其代表作品。

大家認為，一個遊戲做到”真實”需要做到什麼地步？是開放世界無空氣牆？還是死亡懲罰調高？確實，這兩個辦法都能提高遊戲的”真實性”，然而CCP表示這些不夠還要繼續加，最後不僅死亡懲罰設置的很高，而且還添加了很多其他的東西。

比如取消任務改為自由沙盒，配件系統增加重量和耗電這兩個”debuff”，甚至交易市場在原有的基礎上增加交易曲線圖，朋友們，你們見過哪個遊戲的交易行弄的跟股市似的，然而CCP它就敢這麼搞。最後的結果就是，光遊戲UI上的功能，都要學個一年半載才行。

Pong友們，功能學個一年半載什麼概念？大家玩其他遊戲了解系統功能的時候，基本都知道ESC菜單，I背包，M地圖之類的，記住這樣簡單的功能撐死一星期搞定。而這遊戲不僅功能多，而且裡面的子選項還特別多，點開後由於它涉及的東西比較多，所以你還得了解每個子功能都有啥用。這一整套系統，想完全記住，沒個一年半載能行么？這直接導致遊戲的上手難度變的奇高無比。

不過有畢業有利，這遊戲雖然上手難度高，但它裡面很多東西都比較貼合現實。最典型的就是軍團系統。這裡的軍團不是說讓你沒事和其他軍團掐架那麼簡單，這裡面軍團需要運營，只有運營才有良好的發展。

就好比軍團長，你的任務不是想一出是一出，而是要考慮周圍環境，資源產出，人際交往，所有事情都要考慮利弊，不然一個決策失誤就有可能導致軍團陷入尷尬的地步。

所以能把這個遊戲玩好的玩家，那都是人精中的人精。這樣的人放到現實里，那也是非富即貴，也就不難理解為啥有人調侃EVE玩家都是大佬了。
