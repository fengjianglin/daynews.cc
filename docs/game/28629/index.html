Clearlove—有些故事還沒講完那就算了吧

2019-12-22 3,237條評論 4,237次閱讀 2120人點贊

很多人在惋惜，很多人在嗟嘆

退役的這一年，他已經26歲了。

這篇故事在休止的時候，我們開始從影像里重新鋪開他八年的職業生涯

故事從WE的諾言，到EDG Coach：clearlove。

從遠古的貼吧時代開始，到微博之夜打榜結束。

所有當初鍵盤上的愛與恨，最後都同化成為了致敬和懷念。

諾言，MSI，德杯，LPL三連冠，EDG時代，KDA之王

2013年的8月，湖南經視的知名娛樂脫口秀欄目《越策越開心》迎來了一批年輕的節目嘉賓，被汪涵拉上來一臉青澀的小夥子有些拘謹的打著招呼「大家好，我是WE的諾言」 台下響起一段短暫的歡呼。 當時他形容自己的位置就像是「籃球比賽里的小前鋒」

上一年拿到IPL5冠軍的WE，很多新生玩家在未來戰士的半價慶功活動，和那個WE頭像中認識了他們。彼時的諾言和其他隊友相比，無論是若風還是草莓亦或是微笑，諾言並不是最耀眼的那一個。

但對於當時的WE來說，這檔綜藝結束後他們的豪門就開始崩塌。歡聲笑語的他們與一個月後無緣S3全球總決賽時的窘迫相比顯得分外刺眼。

次年2月，教練阿布、打野諾言、輔助捲毛宣布離隊加入新成立的EDG，WE的分崩離析讓初代電競圈上演了史無前例的聲討大戰，隊員互撕，粉絲參與，各方勢力打的不可開交。新賽季EDG戰隊背著罵名誕生了。而這一切對於EDG clearlove來說，他的時代才剛剛開始。

儘管諾言的比賽ID是「clearlove」，但在Phoenix和WE的時候，他用的最多的ID為「Troll（菜鳥）」

舞台一旦開始，就是輪番好戲。如果這段歷史擁有BGM，那麼一定是燃點十足的樂章。

14年春夏兩個賽季，EDG用兩個3:0橫掃IG與OMG創造了首個LPL春夏雙冠歷史。clearlove夏季賽創下了全聯盟76%的最高擊殺率。同年的德杯戰勝WE也開啟了他們德杯五連冠的第一步。

2015年春決，EDG與LGD給LPL的舞台帶來了一場酣暢淋漓的對決。第五局的大龍池雷克賽搶龍，明凱和他的EDG將聯賽雙冠的記錄改寫為了輝煌的三連冠，也握住了MSI的門票。

而這段樂章，也在MSI的舞台上奏到了最High點。面對世界賽上恐怖的SKT1，明凱在總決賽第五局祭出伊芙琳，一場團戰的致命繞後，讓六名身綉神龍的身影定格在漫天金雨下。這是LPL隊伍第一次在拳頭遊戲官方舉辦的全球性賽事中摘得桂冠，我們現在回頭來看，依然是一段值得眼淚和讚歎的劇本。

2016年至2017年，EDG不再常勝，但依然兩度在夏季決賽中戰勝RNG斬獲冠軍。

從建隊伊始，明凱和他的隊友創造了一個屬於EDG的時代。從OMG，IG到LGD再到RNG，賽場上的宿敵不斷輪換，但每一次的巔峰之戰都有EDG。那幾年EDG是所有LPL隊伍都視為頭號強敵的對手。對於明凱來說，這也是他職業生涯最意氣風發的幾年。

縱有失意，依然豪邁。

2016年6月9日，明凱的登場次數（280場）以及獲勝次數（200場）創下了LPL的歷史最高記錄。2016年6月，外國Reddit用戶EsportsDS對全球LOL職業選手的KDA進行了一次排名，明凱以6.6635的KDA排名第一。這是他交出的答卷，亦是他自己職業生涯的前半段答案。

廠長，內戰幻神，年年八強，4396，

在2015年MSI上Faker妖姬的不敗傳說被破後，他說出了一句令很多人熟悉的名言，而造成faker說出這句話的明凱，李哥的感慨之詞也是他的夢魘：

是的，我們在上一段並未提及明凱的世界賽成績。網路時代從來沒有一邊倒的讚美，批判亦然。

從WE到EDG，有關於諾言的爭論從未停止。喜歡他的強調著他的努力和執著，批判他的人則抓准了他在世界賽上充滿爭議的表現。如果說電子競技成績論，本身就是建立在泡沫上的建築的話，那麼諾言無疑就是這其中錯綜複雜的最實際體現了。有多少人喜歡支持他，就有多少人討厭他。

八年職業生涯，據不完全統計明凱累計拿到至少28個冠軍榮譽。但在討厭他的人眼中，依然不如他在世界賽上的膽怯表現來的深刻。從2014年開始，明凱的世界賽最好成績就是八強。最壞成績是小組賽止步。每一年都重頭來過，每一年都陷入輿論漩渦。

我們總是對不喜歡的人報以最深的惡意，所以世界賽的慘淡成績成為了明凱一生的「黑點」

不知從什麼時候起，「梗文化」迅速流行。而每年備受關注和承載LPL粉絲期望的S賽，也成為了造梗的集中地。因為EDG世界賽「永不缺席」，每一年也都被觀眾嘲諷出新的花樣。

「內戰幻神」

「年年八強」

「馬踏飛箭」

「4396」

失敗者的說辭的確很蒼白無力。鋪天蓋地的嘲諷之下，是每一年觀眾對於EDG報以的最高期望。

八強，八強，八強，十六強，八強

作為很長一段時間LPL在世界賽的標誌性隊伍，一號種子席位的「常青樹」

S賽的「外戰」賦予粉絲的強烈榮譽感讓他們「愛之深責之切」

最熱切的希望落空後必將招致最洶湧的輿論。而這些「梗」很好的承載了觀眾所需要去宣洩的情緒。明凱作為EDG的建隊元老和靈魂人物，更是歷遍了最猛烈的浪潮。

而這一年又一年中，少年逐漸不再是少年。

夢想，clearlove7，

從什麼時候開始，明凱這個ID開始和夢想兩個字被人們說起的呢？

海賊王里基德說過一句話：

「如果沒有膽量說出這個，就無法在前面這個海域生存下來」，

彼時正逢S7賽季伊始，明凱在S6世界賽被小花生錘的灰頭土臉後，在2017年改換ID為clearlove7。然而在很多人的眼中，戲謔的眼光遠遠多於對他的讚賞和鼓勵。很多觀眾並沒有因為這樣一個舉動而改變對於他的印象，「7777」不脛而走。夢想卻成為了很多人在社交平台和彈幕上陰陽怪氣的談資。

然而此時的EDG已經不再是LPL各家隊伍為之畏懼的對手，以麻辣香鍋為代表的食肉型打野的出現，讓明凱在打野位置的統治力出現危機。曾經意氣風發的劍手，似乎意識到他的劍不再所向披靡。

那一年的春季賽，明凱上場時間並不多，總共只上場了8次。 在春季賽半決賽中，EDG以1:3不敵RNG，無緣決賽。

就像是星爺在《少林足球》里的經典「人沒有夢想，和鹹魚有什麼分別」，

觀眾們看廠長的心境，亦如劇中人當時看阿星的眼光一般，戲謔而又歡樂。

逐漸的，他開始越來越少出現在賽場之上。

2018年LPL夏季賽，明凱和隊內打野陳文林進行輪換上場，

S8冒泡賽明凱缺席，而入圍賽登場四次幫助EDG進入正賽後，小組賽上場兩次再無身影。當然，S8未有奇蹟。

當這份夢想不再出現的時候，所有的惡意又似乎化成了期待。

人們開始期待著他的回歸，所有有關廠長的官方動態里，評論區也少了很多的鋒利話語，多了諸多溫柔詞句。

而真正喜歡他的人也慢慢不得不接受一個事實，他已經不復當年。

這片舞台終究是屬於年輕人的，

勇者一次次在塵與土中爬起，又一次次歸於沉寂。

每一次爬起都需要更多的勇氣，但勇氣沒有化為勝利

勇者最終也將歸為塵土。

LPL走過漫長的年頭，無數選手如同長長走廊里一盞盞明燈。前方總有璀璨亮起，身後亦是謝幕。最終明凱在一個所有人都明白的時間節點，選擇不再亮起，但與此同時，卻在同一個舞台用一種方式繼續點亮著。

其實觀眾們在故事結束後，都會趨向於對回憶寬容。2019年微博之夜年度人物票選活動，

明凱的票數在退役消息宣布之後，短時間裡躍至第四。這或許是網友們對於這位選手最善意的寬容，一如S5世界賽後論壇上鋪天蓋地的聲討。也許，很多人都是同一批人。

在我們喜歡的劇本里，英雄向來都是喜劇收尾。所有的血淚和積澱都是為了最終登頂巔峰的情緒爆發做鋪墊。而廠長的劇本里，只有鋪墊。這就是故事和現實的區別。

有些故事，

沒有講完，

那就算了吧。
