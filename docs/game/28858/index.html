最期待武俠網遊！一份策劃書價值100萬，玩家苦苦等待4年

2019-12-21 178條評論 2,875次閱讀 281人點贊

編者語：一款遊戲怎麼才能火？可能100個玩家，便有100個說法。總結市面上出名的網遊，能夠得到玩家歡迎，其實都是這幾個原因——畫面、玩法、不氪金，甚至因為代言的明星。

但是在國內網遊圈，有一款網遊從來沒有上線，甚至都沒有進行過測試，但卻一直是國內最期待的武俠網遊。
其實相信許多玩家已經猜出來了，這款網遊便是蝸牛遊戲旗下的武俠遊戲——九陰真經2。在國內遊戲圈有一個頒獎典禮，號稱遊戲圈奧斯卡。每年都會針對國內的眾多網遊，根據玩家投票選出最受歡迎遊戲、最受期待網遊等諸多獎項。

在2019年的金翎獎獲獎名單中，九陰真經2再次成為了最受玩家期待的網遊。這並不是九陰真經2的第一次上榜。在CGWR2018上，九陰真經2便已經斬獲「年度十大期待網遊」。甚至從九陰真經2被蝸牛正式立項進入玩家視野後，九陰真經2便一直掛在各大遊戲門戶網站，最受期待網遊排名的前三。

而從2016年立項到現在將近四年時間，九陰真經連一個正兒八經的宣傳視頻都沒有。九陰真經2的官網從最開始放出的幾張概念圖，幾篇採訪稿，也已經很久沒有更新。但依然成為了2019年最受期待的網遊。

到底九陰真經2有何魅力？靠幾張圖片便能讓玩家心甘情願等待四年時間。其實了解蝸牛這家公司的玩家，相信並不意外。在遊戲製作水平上，可能蝸牛現在連三線廠商都算不上了。但是蝸牛ceo石海畫大餅的功夫，幾乎和史玉柱不相上下。

這一點相信玩過九陰真經的玩家深有體會。當初的石海在研發九陰真經的時候，其實也並不順利。中途多次出現了資金緊張，遊戲開發進度慢的情況。

但石海卻靠著自己的一張嘴，讓玩家心甘情願的期待九陰真經。甚至開創了國內網遊的先河，直接遊戲官方賣激活碼。

在當時免費遊戲正如日中天，別說激活碼就是點卡錢許多玩家都不想出。但石海卻給玩家畫出了一張牛逼的大餅，讓玩家連遊戲收費模式都不清楚的情況下，以149元和249元兩種價格賣出了幾萬個激活碼。

這不但解決了當時蝸牛的資金緊張，而且成功地進行了一波營銷，提前圈出了目標客戶，可謂遊戲營銷史上的經典操作。

九陰真經2的營銷和當初的九陰真經如出一轍，直接放出沙盒武俠網遊的概念，給玩家創造了一個未來的次世代武俠網遊。

不得不說，蝸牛在遊戲的創意上，的確非常有想法。整個九陰真經2到如今也依然只是一個策劃書，但其中體現出來的玩法，卻深深吸引了玩家。

沒有時代背景、遊戲食物鏈設置、生存玩法、無縫地圖等等概念，放到一款武俠網遊中，立刻煥發出不一樣的感覺。

蝸牛就靠著這份策劃書，幾張截圖便將玩家的胃口吊了四年。其實，我們如果仔細研究其中的核心，無論是九陰真經還是九陰真經2，放出的諸多玩法，其實一直在滿足玩家的一個點，那就是一個開放式的武俠世界。

當初九陰真經所宣傳的無等級、奇遇等玩法，也只是開放世界的具象化而已。而九陰真經2放出的玩法設定，也是在武俠開放世界上的完善和具象化而已。玩家之所以買單，說到底還是因為市面上沒有出現類似的武俠網遊。

即使過去了四年時間，真正能夠做到開放世界的網遊，至今還沒出現。幾款大名鼎鼎的開放世界單機遊戲，都是3A大作級別的單機遊戲。

耗費金錢無數，才在某一個方面達到了開放世界的要求，如今年最有代表性的主機遊戲大表哥2。而大表哥2的研發成本高達1.7億美金，耗時8年時間。

九陰真經2想要在一款網遊中，做到真正的開放武俠世界，難於登天。只能說，蝸牛此次將這張餅畫得有些大，也讓自己騎虎難下。以至於四年時間，連個視頻都不敢放出來。

但不得不說，這張餅很香，只是當初的簡陋策劃，最少就給蝸牛創造了大量人氣關注度。這份策劃書最少也要價值100萬。

以至於就是大玩家也不免存於幻想，也許真的有一天國內遊戲公司能夠做到這種水平呢？但估計真的能做出來了，各大遊戲公司也早就開始立項了。

到時候可能不就是一個九陰真經2了，什麼九陽神功、易筋經、百冥神功、乾坤大挪移都可能紛紛出現。這在遊戲圈已經被玩家習以為常。

這也是為何九陰真經2看上去這麼香，但卻一直沒有遊戲公司借鑒的原因。但大玩家還是會將最受期待網遊投給九陰真經2,只因為這已經成為一個期望。
