到底發生了什麼讓遊戲開發組做出了這樣的操作？讓我來告訴你

2019-12-17 123條評論 2,025次閱讀 712人點贊

我和大家一樣都是凡人，而凡人誰都會犯下各式各樣的錯誤，被各式各樣的慾望驅使，當你在遊玩遊戲的時候，總會接觸到不少內容：無論是色慾，憤怒，還是嫉妒與傲慢。

無論如何，但過往的君子姑且請聽我一言。

色慾

讓我們先來看兩張圖：

這兩張圖是《野良與皇女與流浪貓之心》PS4的「健全版本」（聖光是後加的）和經過第二次修改之後的版本，作為一款Galgame和擦邊球福利遊戲，上如此重的聖光，甚至穿的十分健全實在是有違買這個遊戲的初衷。

而最關鍵的問題是隔壁任天堂NS和PC上的版本，不但沒有健全修正，且聖光還少得多，這樣的做法顯然引起了玩家的不滿。

玩家詢問索尼和任天堂的時候得到了兩套不太相同的回復，任天堂那邊的說法是：「只要能夠通過遊戲分級，在我們這邊就可以上架」

雖然是一些正確的大話，但是大家至少知道標準是什麼。

而詢問索尼的時候卻得到的是這樣的回復：「索尼對於遊戲的審核有著一套自己的標準」

等等，你的標準是什麼?

無獨有偶，之前在PS上有系列發售的閃亂神樂系列因為其有爆衣等系統，在新作的審核上遇到了很大的困難。

製作人高木對媒體說：

「目前的還是在於內容上，現在有關性感的描寫在全世界都受到控制，我們需要在不被誤解的前提下做東西。因此確實有一些東西很難做，這是以往所沒經歷過的。可能還需要時間去研究，想要實現的東西該怎麼做，目前正在思考。」

而他的新作閃亂神樂7早早公布，至今還沒有製作完成或者要發售的消息，有可能已經涼了。

實際上索尼突然收緊審核政策也不是沒有由頭。

姨夫擔任索尼社長時代，因為他就是索尼互動娛樂總裁出身，且十分支持遊戲事業，所以在社內說一不二，當時日本各類遊戲只需要通過CERO分級就可以直接上架。

而姨夫在去年退休之後SIE接任者新官上任三把火，需要表現一下自己，正好遇到女權勢力崛起以及Me Too等運動發起，日式遊戲正好成為替罪羊——既然ESRB的老頭子對這些二次元小姑娘都很反感，不如順水推舟，整個PS都採用一樣的審核標準，而只會得罪一小部分的日本客戶而已。

實際上這種收緊表現在很多地方，湯不熱已經禁黃，推特則是很快就要禁黃，西方對於此方面的保守肯定不僅限於目前這些，只允許神父喜歡真實蘿莉和正太，卻不允許死宅喜歡紙片人，這確實可謂雙標了。

暴食

R星作為一個家用機起家的遊戲廠商，不但在PC上出遊戲不太積極，且在PC方面的優化一直都挺糟糕的，當年GTA4的時候，不但畫質不太好，而且只要你CPU不夠好就一定無緣60幀率。

GTA5雖然稍微好一些，但本質上來說他也只是一個上代的遊戲，再加上吃了本代硬體更新的紅利才算是運行完全流暢。

而在荒野大鏢客2下也繼續著R星優化不太行的老毛病，想要全部開中以上60幀就需要一塊6/8核的CPU以及1063以上的顯卡，基本就是吞硬體的怪獸。

遊戲優化糟糕是一個問題，且還有連帶bug的問題，這個遊戲PC版本發售時，家用機版都已經發售了一年了，為何還有如此多的bug？這個是一個世界難題。

PC初版還有一個搞笑的BUG：

PC版本按照30幀為基數計算遊戲內人物的飢餓度，這意味著當你120幀的時候，你甚至能以400%的速度餓肚子，而在這速度之下，你只能不停吃東西才能緩解你的飢餓，而這就正可謂暴食（物理）了。

強欲

重生娛樂從動視獨立之後便一直過得不太順利，他們的第一作泰坦天降在不太成功的平台XBOX ONE上獨佔，且只有多人模式，很快就涼了。

而第二作泰坦天降2為了服從於EA的品牌戰略，和戰地1一起推出試圖擊敗COD，COD當然沒有被擊敗，戰地1熱度也很高，但泰坦天降2這個品質還可以的遊戲，卻因為之前知名度不高而變成了犧牲品。

當然EA也並沒有放棄重生娛樂，因為DICE在《星球大戰：前線2》中飽受惡評，而EA旗下能做好這類遊戲的開發組，實際只剩下重生娛樂了，因此EA在當時應該就已經將《星球大戰 絕地：殞落的武士團》交給了DICE。

而APEX英雄則是他們獨走（EA反對）意圖證明開發實力的項目，但作為一個不太大的組，在短時間內連開兩個規模不小的項目顯然超出了他們能夠應付的範疇。

雖然這兩個項目完成度都算不錯，評價也很高。

但APEX英雄上線伺服器有問題，且外掛太多影響體驗，沒有足夠人力導致更新太慢致使玩家流失的很厲害，只火了幾個月熱度就已經消失了，未能夠像堡壘之夜和PUBG一樣維持長時間的熱度，這可以基本上可以歸咎於急於成功而想要了更多自己不能完全應付的東西。

懶惰

今年的懶惰應該應該發給卡普空的生化危機2重製版，不止是因為卡普空愛做冷飯。

還更因為他們做冷飯的時候都沒有好的態度，生化危機2雖然使用了新的遊戲機制，但連原作的表裡關卡和對應的內容互動都沒有做全，我也是第一次見到做重置實際內容量甚至還不如20年前的原版的。

有一個好消息是：他們至少還捨得用新機制翻新那麼一個遊戲。

有個壞消息則是：

這麼做的生化危機3重製版已經在路上了，明年又可以貢獻一個名額了。

憤怒

NBA2K20作為一款年貨籃球遊戲，不但有被人質疑為換皮的嫌疑：運行遊戲之後任務欄圖標都顯示2K19（大概是以為大家都會全屏看不到這東西）

而且Bug還非常多，粗製濫造的很明顯，一個換皮遊戲還Bug很多本來就不可理喻了。

而大量模式需要在線才能使用，而聯機效果非常糟糕。

以至於評價里充滿了負面評價：

要知道一個遊戲爛並不困難，頂多就是幾條差評之後加上沒人玩。

而不好玩到幾千條差評，從契丹到愛爾蘭無人不知無人不曉就很難了，而2K20成功的做到了這一點，併當選當年最值得讓人憤怒的遊戲。

嫉妒

Epics憑藉堡壘之夜賺足夠了錢，所以開始嫉妒steam上的已有的遊戲，並開始不停的買斷遊戲獨佔權。

從莎木3的退款風波，到地鐵離去不再在Steam上發售，PSN漲價到638港幣，逼得玩家只能去EPICS，到惡搞遊戲無名大鵝EPICS獨佔引發論戰，EPICS的嫉引發了無邊的論戰。

但無論是真香還是抵制，這些口水戰都引發了效仿和玩家的擔憂——例如育碧就已經開始完全獨立的在自家平台上發售新遊戲，雖然價格通常真香，也並沒有受到太多指摘。

但不可避免的，仍然是PC應用商店的碎片化，這也許某種意義上有利於價格競爭，但對玩家而言總體來說還是弊大於利更多。

傲慢

寶可夢承載著千萬大小朋友的夢想，但作為一個打底就有千萬銷量的作品。

本作寶可夢劍盾做出來的質量卻完全對不起千萬大作的內容，不但寶可夢數量減半，從800個減到400個，地圖上的內容也難言樂觀。

甚至有一些小鎮的有效建築只有一個道館和一個PC房，如此缺失的內容甚至都不能比上好幾代之前的寶可夢地圖，而曠野地帶糟糕的畫面甚至一度被人黑成N64。

作為一個躺拿千萬銷量的大作公司，卻不能提供對應的遊戲質量說實話是一件挺令人沮喪的事情，但之前幾作至少不會遇到寶可夢大腰斬，以及遊戲內容缺失的問題。

說白了，以目前gf的開發實力，即使再等3年，下一作的口袋妖怪可能也不能做到有多好，而你到時候可能還得真香，說這玩意不知道比劍盾高到哪裡去了，而GF也還能繼續談著賺錢，而這種傲慢卻應該會一以貫之的留下來。

不過說了這麼多，雖然該噴的也噴了，該說的也說了，不過作為一個玩家說到底還是希望廠商，尤其是知名廠商能夠做好遊戲，這樣我們玩家才能有福。
