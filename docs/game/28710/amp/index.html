Hero、eStarPro冬冠杯首戰，久誠回歸助隊伍取勝，eStarPro飄了？

天天要聞
2019-12-22

KPL秋季賽已經結束，現在新的杯賽冬季冠軍杯，選拔賽已經正式開啟，選拔賽有些特殊，所有參賽的隊伍會進行BO1的對決，最後決出兩支隊伍晉級正賽，參與的隊伍與往常一樣，一共有八支隊伍，分別來自KPL與KRKPL賽區，如果不出意外，最後晉級的隊伍，應該全部都是KPL賽區的隊伍，畢竟兩個賽區的實力不對等，從前幾屆的冠軍杯明顯可以看出來。

觀眾對於冬季冠軍杯選拔賽，關注點肯定也都在KPL隊伍中，參賽的隊伍人氣都比較高，實力也都在線，RW俠、Hero久競、eStarPro、TS、XQ，前三支隊伍都是目前較強的隊伍，而後兩支隊伍，都經歷過調整，XQ狀態還算可以，TS則是挺不錯的，他們在整個秋季賽一直都在調整，一直到最後階段，才確定下來陣容進行磨合，冬季冠軍杯距離時間不長，所以狀態值得期待。

冬季冠軍杯中Hero久競與eStarPro的首戰，是比較讓觀眾關注的，Hero久競的亮點是久誠回歸，久誠對於Hero久競的意義非常大，在巔峰時期Hero久競體系打的最多的就是中核體系，而中核就是久誠，儘管之前有人否認過中核，但久誠無疑是KPL聯盟中很強的中單選手，長時間坐替補席，狀態如何大家都會比較關注。

而eStarPro冬季冠軍杯狀態怎麼樣，其實是決定他們能否再次回到巔峰，2019年想要成為eStarPro年，那麼冬季冠軍杯他們應該去爭奪到手，本年度兩個冠軍到手，到秋季賽失利，外界對eStarPro的印象，已經覺得他們膨脹了，如果有足夠的準備，秋季賽還是很有希望爭奪一下冠軍，畢竟整個常規賽都是積分領先。

秋季賽被淘汰之後，冬季冠軍杯選拔賽是eStarPro的首場比賽，他們會不會走上狀態下滑的路子，肯定很讓人關注。以前已經有太多隊伍，最後都比較玄學的迅速變撈，想要打破這個局面，冬季冠軍杯無疑就是一次試煉，與Hero久競的首場比賽，結果是eStarPro失敗，久誠的回歸幫助隊伍取勝，證明了自己的能力。

而eStarPro失敗確實可以看出他們飄了，有些朋友覺得是BP上的問題，尤其是中單選擇貂蟬對上周瑜，有了貓神的eStarPro，在陣容上本來就是敢於冒險的隊伍，拿什麼陣容肯定是有想法的，但這種冒險精神，就必須要承受失敗的結果，打得好網友會覺得有亮點，而不好則會成為背鍋俠。不管如何eStarPro在準備上肯定是不足，說他們飄了也不為過，秋季賽失利之後，冬季冠軍杯他們應該重視，在選拔賽中每一小場的比賽都至關重要，他們需要迅速調整狀態！
