王者榮耀：6件裝備模型，三年前VS三年後，無盡讓人很懷念

王者榮耀自公測到現在已經四年多了，目前也來到了S17賽季，相信大家都對這款遊戲非常熟悉了吧？我們都知道，為了保持遊戲的平衡性，天美除了大版本之外，平時也會不定時的對遊戲進行一些優化和更新。而在遊戲中，對於英雄影響最大的應該就是裝備了，不過，不知道大家有沒有發現，其實有些裝備在經過優化和更新後，已經跟三年前的不太一樣了。下面，小精靈就為大家帶來6件裝備，三年前後的對比吧。

1、反傷刺甲，這件裝備不僅了高額的物理防禦加成，而且還有攻擊力的加成，最關鍵的就是它的被動了，在受到物理傷害的時候，會將傷害的20%以法術傷害的形勢回敬給對手，這也可以說是對抗射手或者刺客的神器了。三年前的反傷刺甲上面還有很多尖尖的東西，不過現在的之後肩膀上有了。

2、魔女斗篷，三年前叫魔女面紗，這件裝備不僅有高額的法術防禦加成，而且還有生命值的加成，這件裝備也可以說是對抗法師的神器了，特別是對面有高爆發法師的時候，這件裝備就能很好的剋制他了。三年前的魔女面紗像一個掛墜，而現在的就是一個非常漂亮的斗篷了。

3、聖杯，這件裝備不僅有法術攻擊的加成，而且還有減CD，最重要的就是它有很強的回藍效果了，這也使得部分法師都會選擇它。三年前的聖杯看起來像獎盃，而杯子里的液體像血液，而現在的更像是夜光杯里裝滿了葡萄酒了。

4、痛苦面具，三年前叫折磨面具，這件裝備不僅有法術攻擊和生命值的加成，而且還有減CD，而最關鍵的就是它的被動了，有法術穿透效果，這也使得大多數法師在前面兩件基本上就會出了。三年前的折磨面具看起來還是有些嚇人的，而現在的痛苦面具看起來就好看很多了。

5、無盡戰刃，三年前叫無限刃鋒，這件裝備不僅有高額的物理攻擊加成，最關鍵的就是它的暴擊率和暴擊效果了，現在也幾乎是所有射手必出的裝備之一了。三年前的無盡看起來還是比現在的炫酷很多，也是讓人很懷念啊！

6、暗影戰斧，三年前叫黑色戰斧，這件裝備有物理攻擊和生命值的加成，也有減CD，而最關鍵的還是它的被動，有護甲穿透，現在也基本上是戰士英雄的標配了。不過對於模型來說，小精靈還是比較喜歡現在的。

以上就是小精靈為大家介紹的，6件裝備三年前後的對比了，話說，小夥伴們，你們覺得是三年前的好看，還是現在的好看呢？歡迎留言討論！
