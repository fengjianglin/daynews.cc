魔獸世界懷舊服：唱衰的已經說到年卡了 然而新服依舊要排隊

2019-12-09 0條評論 14次閱讀 0人點贊

摘要

魔獸世界懷舊服從開服開始，便存在著一群看熱鬧不怕事大的人，有一點風吹草動就來喊涼涼，從開服一個月、兩個月、三個月、半年、一年，可以說這群唱衰的人是越來越沒有底線，984G十方（筆者）此前還願意和他們理論一下，後來發現你真的叫不醒一個裝睡的人。

魔獸世界懷舊服從開服開始，便存在著一群看熱鬧不怕事大的人，有一點風吹草動就來喊涼涼，從開服一個月、兩個月、三個月、半年、一年，可以說這群唱衰的人是越來越沒有底線，984G十方（筆者）此前還願意和他們理論一下，後來發現你真的叫不醒一個裝睡的人。

不是第一次，也不是最後一次

回想魔獸世界懷舊服第一周的排隊情況，唱衰的聲音就開始了，甚至還有說故意製造火爆的假象，等進去之後發現，是真的火。到處都是人，並且大家還一團和氣，這樣的氛圍已經很久沒在網遊中出現過了，於是這群人開始找其他毛病。

矛頭指向GZS

魔獸世界懷舊服排隊快了不行，慢了不行，副本人多不行，人少也不行，指責了一圈之後，當遊戲內容沒有什麼可以指責的地方之後，這部分人開始找客觀存在的事情了，那就是GZS。其實GZS從魔獸世界誕生的時候就一直有，甚至其他的網遊和手游也都存在，遊戲涼了的責任往gzs身上推就可以。其實理由本可以選得更好一點，比如等關了位面，野外也別想做任務了，到處都在打架。

懷舊玩家如是說

不管版本如何，別人怎麼說，來懷舊的玩家表示每天上線兩到三個小時，時間完全不固定，團本更是沒打算去，練了不少於3個號，一天一個號清雙倍經驗，這樣玩遊戲難道他不香么，所以遊戲涼不涼關我什麼事呢。因為遊戲進度較慢，很多內容足夠他們再體驗很久，當然，對於一切遊戲目標都以稱霸伺服器為主的玩家來說，無論新老，都與懷舊二字無關。

有必要說幾句了

涼啊涼啊，魔獸世界在一片涼涼聲中進入了2019年。2019年夏天傳出魔獸世界懷舊服到來的聲音，那時候炒冷飯撈一波情懷就走的看法就不絕於耳。8月份都說魔獸世界懷舊服一個月就涼透，因為月卡到期。9月份都說魔獸世界懷舊服三個月就涼透，因為季卡到期。最近又有人說魔獸世界懷舊服半年後必倒，但開新服的時候排隊依舊讓人絕望。

寫在最後

剛開始說月卡，接著季卡，然後半年卡，現在到了年卡了，984G十方覺得魔獸世界懷舊服頂多撐到下個世紀，為什麼？因為魔獸世界太省錢了，之前玩其他遊戲每個月上千根本不稀奇，現在懷舊服半年卡才幾百塊，請問怎麼涼？
