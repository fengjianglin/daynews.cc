有生之年！經典暗黑職業"德魯伊"上線《無盡大冒險》

天天要聞
2019-12-18

說起”德魯伊”，不禁讓人想起那一段逝去的光輝歲月，那矯健的身姿，能打能抗；那廣闊的胸襟，能法能奶。可惜很久都沒在暗黑正作里見過了，但是”德魯伊”的故事並沒有結束。

【背景故事】

至高天一役，東部森林的德魯伊與戰友們一起從惡魔的手中拯救了這個破碎的世界，而後又歸隱於自己的家鄉，繼續做著那些不可思議魔法和技術的深入研究。這些德魯伊們崇尚自然的偉力並擁有著獨一無二的變身形態，對自然堅定的信仰賦予他們自由控制冰、火、風和閃電的能力，維護自然平衡和保護生命的使命也允許他們支配很多的野獸和植物，在經過二十多年的鑽研之後，德魯伊們甚至掌握了上古機甲的尖端科技。

德魯伊教徒相信總有一天惡魔們會重新威脅整個大陸，當這一天到來時他們會繼續堅持自己世界守護者的信念，因為他們見過自己所發誓保衛的東西被惡魔殘忍的摧毀，他們理解每一個奈非天所經歷過的痛苦，在任何一刻，德魯伊們都做好了與混沌的僕人決一死戰的準備。

【職業介紹】

如今”德魯伊”上線《無盡大冒險》，以一種全新的姿態，以一種不一樣的形象，閃爆全場。新職業德魯伊有三種形態：人形態、貓娘形態、機甲形態，可通過技能學習解鎖，自由選擇切換。可選擇自由搭配——近戰，召喚，法術三種成長。比如人形態的召喚搭配機甲形態的召喚，人形態的召喚搭配機甲形態的法術，人形態的召喚搭配貓娘形態的召喚，人形態的召喚搭配機甲形態與貓娘形態，……在此就不一一列舉，搭配組合多樣。

如果夠敏銳，就會發現，以上介紹這個遊戲的德魯伊形象是女性，跟常見的遊戲中德魯伊形象不同，難道是”娘化”？其實，一直以來，許多人都以為德魯伊都是男性的。然而，許多歷史記錄證明，一開始只有女性德魯伊，後來才出現了男德魯伊。在中文裡的”巫”這個字原來就指女巫，男巫另有叫法”覡”（Xí）。這是來自母系社會的產物。她們同樣也是智慧之人，通常精通草藥學，在部落的醫療、占卜中作了很大的貢獻。

【創作背景】

那麼為什麼這個遊戲里會出現”德魯伊”這個職業呢？這就得追溯到許多年前，《無盡大冒險》是一款15年就上線的暗黑佛系放置遊戲，是由三位暗黑2 的忠實粉絲深感玩家肝到窒息，累到抽筋，不禁渴望有一款可以佛系的暗黑遊戲。於是三位大佬一合計，《無盡大冒險》就那麼誕生了。而”德魯伊”作為暗黑遊戲的經典職業，理所當然的就被做進來了。

這麼介紹下感覺好草率啊，但是這其實還是很棒的一個遊戲啊，不然也不會去年出新職業，八月重製又請王境澤代言，現在還能出新職業。感覺有點迫不及待新職業德魯伊 1月8日上線了。
