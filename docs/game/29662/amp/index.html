季前賽未實裝的改動：獲得男爵之力後，擊殺的野怪會變成己方生物

天天要聞
2019-12-25

大家好，我是愛玩遊戲的小蘿醬，每天分享最新的遊戲資訊和有趣的遊戲內容，嗯哼~

英雄聯盟中最強力的兩個野怪應該就是大龍以及遠古巨龍了，擊殺遠古龍後可以讓戰鬥力得到巨大的提升，更適合於打團戰。而擊殺大龍最明顯的是對兵線的增強，有助於推進或者防守，這也是玩家為什麼如此看中大龍的原因，它可以將優勢轉化為勝勢，加速遊戲的進行，也可以幫助劣勢方撐過一段難度的時間。當一方積累了一定的優勢後，拿下大龍資源帶著增強的小兵一起將對手的基地推掉，是遊戲最常見的獲勝方式。

雖然野區分敵我，但是野怪都是中立的，唯一有關聯的地方就是擊殺遠古龍會使獲得的小龍屬性進一步增強，畢竟是一個坑裡生長出來的，不過大龍可能和野怪也產生關係了。英雄聯盟美服官方公布了原本對S10賽季季前賽的一些改動，其中提到獲得男爵之力的角色，所擊殺的野怪將會變為己方生物，跟隨小兵一同推進。

從視頻中了解到，被男爵之力持有者擊殺的野怪，會向距離自己更近的道路前行，比如F6會去中路，石頭人（紅色方）則會去到上路。雖然視頻中沒有提到，但是史詩級野怪應該不會受到影響，否則先拿大龍再拿遠古龍，雙龍會的威力加上遠古龍和兵線的進攻，對手可就真得雙手離開鍵盤了。

擁有支配其他野怪的能力，對於拿到男爵之力的一方來說自然是不錯的增強。通常來說優勢方更有機會拿下大龍，並且掌握的地圖面積更大，野區資源獲取的更多，再加上增強的兵線和大批的野怪，就能拿下更多的防禦塔和水晶，獲得更多的經濟優勢，甚至一舉推掉敵方基地。同時劣勢方如果能搶掉大龍，就能爭取到更多的時間，翻盤的幾率也就更大。

至於丟掉了大龍的另一方，就要第一時間解決掉能力範圍之內的野怪，防止其變為敵軍幫助對手，如果有野怪在前面扛住，對手以及他們的炮車就可以盡情去攻擊防禦塔。不過可能有一件「好事」，就是可以通過擊殺對方的野怪，獲得新鮮的Buff，但總感覺有些無濟於事。

「野怪攻城」這件事並不是在英雄聯盟中第一次發生，在末日人機模式中，電腦方在一定時間後，會派出源源不斷的野怪大舉入侵玩家的防禦塔以及高地，其中還會出現峽谷先鋒和小龍。說起來還有些想念末日人機了，不知道什麼時候能上線，再次體驗一下被電腦虐的快感。

不過這一改動究竟什麼時候能出現在正式服還是未知數，這是季前賽原本的設定，或許是因為過於強力導致沒有實裝，不過機會還是有的，既然設計師想過這個方案，就有可能會在某天實現，不知道大家期不期待呢？
