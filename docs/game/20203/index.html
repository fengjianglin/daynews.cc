12.17日正式服更新，五大英雄改動，姜子牙體系不復存在

2019-12-17 345條評論 3,466次閱讀 1438人點贊

最近KPL秋季賽已經結束了，在這裡呢恭喜AG超玩會取得了最終的勝利。為了慶祝AG超玩會獲得冠軍，正式服也是進行了一次更新，商城內所有的直售英雄限免，並且冠軍英雄和其皮膚也是進入了折扣環節，算得上是誠意滿滿。不過除此之外最讓玩家在意的還是一些遊戲內部的改動，比方說英雄的調整。畢竟這可是關係到我們的上分之旅！接下來老張帶大家一起來看一下吧。

盤古（降低AOE爆發）

盤古是一個前期極強的英雄，類似於裴擒虎一樣具有很強的入侵能力。憑藉著前期的流血傷害和繳械，在前期基本上能夠一錘二。本次的調整並非像之前一樣降低其被動流血傷害，而是降低了大招的傷害，但是其釋放機制沒有變化。

本次的調整不是特別大，還在可以接受的範圍之內，雖說降低了一定傷害，但是大範圍的繳械減速讓大招依舊有著很強的壓迫力。前期盤古還是入侵的好手，後期對於後排的威脅可能會小一點，但是機制沒有變化，依舊不容小覷。雙重繳械對於射手而言仍然是噩夢，頂多是利用平A多補充幾次傷害罷了。

馬超（降低移速加成）

本次對於馬超的調整就是四個字：簡單粗暴！馬超的機制很有趣，就是移速=傷害，這兩者本身沒有什麼交集，結果在馬超身上得到了完美的結合。本次降低了馬超被動增加的移速，也就相當於降低了馬超一部分的傷害，不管是對新手還是大神，遊戲體驗會有一定降低。

不過沒有改動移速對於馬超的傷害加成，讓馬超帶疾跑的收益依舊很大。對於大神而言，疾跑開啟之後的10秒時間內，馬超依舊是之前的馬超，機動性強傷害高。

姜子牙（被動進行平調）

姜子牙一直是KPL中非ban必選的存在，有了姜子牙的隊伍不論是二級的野區入侵還是四級的暴君之爭都會佔據明顯的優勢。本次調整了姜子牙被動觸發機制，由之前的10秒觸發變成了現在的兩分鐘觸發一次，同時作為補償提高了大招的傷害。

姜子牙本身是一個法師，但是在普通玩家的手裡基本上變成了一個輔助。此次調整了姜子牙的被動，讓他的定義更加明確，本次調整之後主要走的就是中路位置。對於四級的暴君之爭姜子牙依舊是不容忽視的英雄。

安琪拉（上升大招護盾）

安琪拉從改版之後一直處於優化的狀態，每一次調整都是小幅度的上調，本次依舊不例外。這次提高了安琪拉大招的護盾數值和AP加成，開啟大招後的安琪拉顯然是更具坦度的。並且玩起來也更加順手。

安琪拉的大招有一個特點，就是在護盾持續期間內不會被控制所打斷。提高了護盾的數值，也就意味著安琪拉在開啟大招後能更好的輸出，對於安琪拉的生存能力而言算是一個不錯的提高。

魯班大師（增加大招穩定性，提高基礎生命值）

魯班大師作為一個新輔助，在沒上線之前就被許多玩家所看好，其自身所攜帶的交互機制更是讓一些笨重的英雄變得更加靈活。按理說這個英雄上線之後強度會很不錯，結果勝率到現在才僅有37%，是輔助中的最低勝率。

本次提高了魯班大師大招的長度，並且減少了防斷時長，讓魯班大師的大招能夠形成一個較為穩定的控制，同時增加了基礎生命值和成長強度，魯班大師也能更好的充當一個前排。但是在老張看來，大招主要原因還是釋放的機械臂太短了，很難抓到人。如果能提高一定的長度或許會更好呢。

那以上呢，就是本次正式服更新的全部內容了，你們覺得本次調整如何呢？歡迎評論區留言討論！
