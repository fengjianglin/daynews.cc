夢三國2：軍團出征中法師最討厭的那些裝備

天天要聞
2019-12-21

雖說物理賬號只需要簡單的普攻就能大殺四方，但如今依舊有很多喜歡玩軍團出征的夢三國玩家對法術裝備情有獨鍾。這自然也就使得了那些對法師有著很大限制作用的裝備成為了眾多主法玩家異常討厭的存在。這次，舞文就和大家一起來看看那些在軍團出征中讓所有法套玩家都非常討厭的那些裝備吧。

黃書（太平要術）

對於護甲值本身就不算高的法師而言，被撒上黃書之後非常容易被秒殺不說，即使別人沒有攻擊你，一不小心被人撒上黃書持續8秒的藍量削減也非常讓人頭疼。尤其是對於那些技能加成高的玩家而言，前期在藍量不多的時候被撒上黃書，持續時間內基本沒有了任何戰鬥力可言。

禁魔甲（黑暗玄冰吞魂鎧）

雖說如今的軍團出征中，很多需要進場的法師都會選擇做出禁魔甲這件裝備。但我們依舊不得不承認，對於所有法師而言都非常討厭遇到這件裝備，畢竟只要一不小心中招無法使用技能與物品對於小身板的法師而言絕對是噩夢級的待遇。這裡額外問一句大家最討厭的是哪位英雄做禁魔甲呢？作為一個出征小法師，舞文最討厭的反正就是做禁魔甲的張郃了。

天征鎧、神羅天征（天舞天征鎧）

就這件裝備本身屬性上來看，對法師影響並不算大。可是，它的被動效果每隔5秒清除自身的負面狀態，對於大多數都需要在控制到對手之後才能打出傷害的法師英雄而言都是非常噁心的一環。畢竟，在對手沒有這件裝備的時候，你可以直接跳上去利用羊刀或者冰扇打出控制後一擊必殺。可一旦對手有了這件裝備，而且自己運氣不夠剛好控住對手就觸發了被動，那麼往往交代掉的都是法師自己。

禁錮（封印禁錮法杖）

相對於娛樂模式而言，出征地圖中禁錮的登場率低了不少。可是一旦這件裝備登場之後，對於對手的所有法師而言都是異常噁心的存在。畢竟，在出征地圖中，大多數時候都是極難處理的前排肉盾英雄和後排輔助英雄才會去選擇。法師在控住對手主力輸出之後，一套技能打下去被對方禁錮輕鬆化解的時候還是比較多的。而沒有完成擊殺還浪費了大量技能的法術英雄，輕則狼狽逃竄重則丟掉人頭。
