哈嘍，各位玩家大家好，氪金是遊戲運營的一種十分直接的獲利方式，所以目前幾乎所有遊戲都會從不同的方去引導玩家氪金，甚至很多遊戲不氪金基本就沒辦法玩，而主流的套路都是賣皮膚、賣人物、賣裝備，所以氪金玩遊戲在當下其實真的是非常正常的一件事情，比如這幾年流行的《王者榮耀》就有大量的氪金大佬存在，但是氪金大佬不一定真是一個土豪有錢人，甚至很多氪金大佬連收入都沒有。

一、氪金玩家其實大多沒有經濟來源。

原因其實也很簡單，遊戲玩家大多數都是低齡選手，甚至很多玩家都是未成年的，14、15歲的玩家並不少，而往往這類玩家氪金起來卻是最瘋狂的，我一個表弟就在王者榮耀里沖了超過1萬多，王者榮耀140多個皮膚，全英雄，還不包括送同學的皮膚，他不但自己氪還幫同學氪，但是他家庭卻是縣城普通家庭，作為一個學生更加沒有經濟來源，錢自然都是他父母承擔的了，所以說氪金玩家一半是窮人還真沒有誇張。

二、氪金真的能變強嗎？

其實氪金在手游里並不能顯著的提高實力，但是的確能更快的變強，比如購買銘文，購買銘文可以算是氪金作用發揮最強的方面了，而如果想要不氪金攢齊銘文，消耗的時間是非常久的，甚至很多玩家一整年下來也不過是兩套150級的銘文，實在是太久了，而沒銘文的英雄和有銘文的英雄的確有著非常大的差距，比如韓信、猴子等英雄，沒銘文基本上是沒法玩的。

三、玩家直言：氪金其實都是虛榮心在作怪

另外不少英雄比如趙雲，武則天，都是要充錢才能獲得的，其中作用最小的是皮膚，但是氪金購買皮膚的玩家卻是最多的，有很多氪金大佬就直言，皮膚的作用最大之處其實是滿足自己的虛榮心，類似於自我催眠大法，充錢能給一部分玩家莫名其妙的自信：反正我充錢了我是大爺，爾等窮鬼還想打過我氪金大爺？所以菜出現很多玩家寧願吃泡麵也要攢錢買皮膚。

綜上所述玩遊戲氪金的玩家其實真的又很多人都是窮人，他們氪金只不過是為了滿足自己的虛榮心罷了，畢竟現實生活中想要假裝成土豪代價是非常高的，土豪一餐飯就幾萬塊，而遊戲里就不一樣了，沖個幾千塊就能成為土豪了，完全能滿足自己的虛榮心。

但是說句個人真實的感受，王者榮耀里充錢真的不管用，你該菜還是菜，畢竟是5V5對戰遊戲，隊友一旦掛機你穿什麼皮膚都沒用，你皮膚在牛你能1打9嗎？而且皮膚的那點屬性加成作用真的不是特別大，更重要的還是意識、操作、走位和出裝。
