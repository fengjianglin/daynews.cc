潛行遊戲

軟體名稱：死亡點 (Death Point)

大小：1.600MB 原價：68元現價：12元中文界面無內購

一款潛行射擊遊戲，遊戲以上帝視角進行了一場精彩絕倫的硬核冒險故事。

遊戲除了具有射擊元素還具備了動作格鬥、角色解謎等遊戲要素，玩家扮演一個從監獄出逃的特工，潛入敵人內部救走你的同伴講解員，她被隱藏在一個神秘的地方，找到她並帶她離開這裡。

遊戲採用雙搖桿操作，支持手柄會滿足潛行遊戲愛好者所有要求，聲音吸引、腳印痕迹、暗箱隱藏、吸引敵人、背後暗殺。遊戲中遇見黃色提示方塊即可開門，如果有攝像頭提示說明此處需要長按才能解鎖，潛行遊戲中最重要的元素暗殺在遊戲中會發揮極致，慢慢靠近敵人不要放出任何聲音從他背後會有提示操作一擊必殺解決掉他。

遊戲畫面渲染和音效處理的都很棒，爆炸效果非常驚艷，建議戴上耳機遊玩，聽腳步聲音是這款遊戲必需要掌握的技能。

評價：遊戲比較有難度，遊戲開始會有教程讓玩家掌握基本操作，如果喜歡太空刑警的朋友肯定會很容易上手。
