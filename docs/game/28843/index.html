得不到的永遠在騷動！論魔獸世界懷舊服橙色武器為何如此值錢？

2019-12-21 642條評論 1,744次閱讀 179人點贊

眾所周知，在魔獸世界懷舊服中，橙色武器肯定是最極品的存在，也肯定是最值錢的存在，如果要問為什麼？

那恐怕一句話可以回答：得不到的永遠在騷動！

不可否認，都知道風劍是身份的象徵，又大又拉風，手持風劍絕對是非常顯眼的，絕對可以引來大眾的目光，於是乎，眾人肯定都想得到。

既然都想得到，偏偏只有少數人能夠得到，那麼對於大眾來講，肯定就是基本得不到的，既然大眾想得而得不到，那麼少數能夠得到的人自然就成了值錢的東西。

儘管風劍、炎魔錘、雞腿杖肯定都會是早晚過時的，即便明知這一點，諸多「土豪」玩家依舊在拚命的追求，這說明什麼？

另一方面，當你真正得到風劍、炎魔錘和雞腿杖之時，真的還會騷動嗎？不得不說，你其實並不是想得到橙色武器，而是在追求那種得不到的感覺！

得到了炎魔錘，你就想著風劍，得到了風劍，你就想到了雞腿杖，但真正得到以後，你又會想到更多的東西。於是乎，得到的玩家並不會騷動，得不到的玩家卻依然在迫切想要得到，這正是所謂的道理。

其實不僅是風劍、炎魔錘、雞腿杖，黑色其拉作戰坦克同樣也是對99%的玩家在騷動著，因為肯定會至少99%的玩家得不到它！

不禁試問，究竟是因為其稀有，還是因為你因得不到而在騷動呢？
