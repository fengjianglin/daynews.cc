CFS世界賽結束：官方或因此作出重大賽制調整

2019-12-21 428條評論 1,442次閱讀 418人點贊

今年的CFS2019世界總決賽，中國隊這邊可以說是輸得非常徹底，三支中國隊全部折戟，成績最好的AG也僅為季軍，冠軍和亞軍都是兩支巴西的戰隊。而在這場失敗後，官方也是很快發布了官博表示，將會在接下來進行一些列的調整計劃，不少網友紛紛猜測，很可能要進行賽制改革了。

​

從微博的內容上來看，官方主要提到了兩點發展或是說改革的方向。

​

第一條是在未來推動賽事變革，進一步優化遊戲競技化內容，提高中國國際賽場的競技水平；第二條是有關青訓隊的，將會啟動CFDL發展聯賽，保證有更多新鮮優秀血液進入。官方此舉足以看出，今年的戰績對他們的打擊還是挺大的，畢竟不僅僅是冠軍，就連亞軍都沒了，變成了兩支巴西隊的總決賽，作為一直冠軍拿到手軟的賽區，實在有些丟臉。

​

針對這一比賽結果，不少粉絲和玩家們紛紛積極出謀獻策，大家也都看出來了，因為在國際服的比賽中，選手們都是裸裝上線的。而這一點在CFPL的比賽就不一樣，英雄級武器的使用習慣導致CFPL的選手在與外國選手正面較量的情況下難有還手之力，畢竟國內選手的主要精力還是放在CFPL了，一旦轉換環境就沒有戰鬥力了。

​

不過不管怎麼改革，都必須要執行了才會有效果，希望不會像當初非常有意義的席位制一樣雖然提案出來了，但是實際卻一拖再拖難以實行，你對明年的世界賽抱有希望嗎？
