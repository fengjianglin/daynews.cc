IG公布德杯大名單！puff南風即將首秀，S8冠軍陣容集體缺席

天天要聞
2019-12-19

馬上年底的德瑪西亞杯算是今年最後一個比較有意義的比賽了，這次德杯因為在轉會期結束後舉辦，所以會有很多戰隊選擇來檢驗新人的實力，比如說FPX就派的是二隊登場，而像是RNG派出了betty，TES派上了Karsa，新加入選手到底表現如何，將在德杯上完全展現。作為人氣戰隊的IG也終於公布了自己的德杯名單，意料之中的也是清一色的新人登場。

​IG的德瑪西亞杯大名單具體如下，教練是丁波，上單和中路都是二隊新人yi和captain，打野還是樂言，下路就是剛剛加入IG的VG下路雙人組。而大家熟悉的S8冠軍陣容則不會在這次德瑪西亞杯上亮相。

​雖然是全新陣容，但是看點還是很多的，首先就是樂言這邊，在經歷了LPL冒泡賽證明自己失敗，S賽證明再次失敗，以及LPL全明星周末依舊杯質疑的情況下，這次德杯全員新人，沒有了老大哥的壓力，能不能打出自己Rank的水平和實力，不然真的要讓眾多IG粉絲對他失望了，如果真的證明自己就是下一個kid，那麼這個職業生涯說白了，盼頭也不大了。

​除了樂言之外，最大看點就是前VG下路雙人組puff和南風了，這兩人的韓服Rank分數都非常高，基本上都是1000分左右的，個人實力沒得說，只不過這是第一次代表IG出戰，能不能打出自己的水平，以及和IG其他隊友之前配合是否 默契將是這次德杯上的看點。

另外就是很多IG粉絲買了德瑪西亞杯的現場門票，現在來看似乎有點可惜了，因為論人氣還是S8冠軍陣容最吸粉的。而IG派出這套陣容，也算是證明了自己對於這次德杯奪冠並沒有野心，只是想要考驗一下新人的實力。不管如果，還是希望這些小將們加油吧！

​那麼，對於這份IG德瑪西亞杯的出征名單，你們有什麼想說的嗎？
