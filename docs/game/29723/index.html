絕地求生：黃金8決戰日前瞻 OMG仍有奪冠希望 SMG爆發即可翻身？

2019-12-24 1,948條評論 4,471次閱讀 1829人點贊

2019年12月24日，鬥魚黃金大獎賽即將迎來決戰的一刻。全國最頂尖的16支戰隊將在絕地島上展開激烈的角逐，64位選手之間的大戰可以說是令人期待無比。而就目前的積分形式來看，前8名的分差可以說是咬得非常緊，基本都有奪冠的可能，所以這樣緊張刺激的大決戰怎麼能不賽前分析一波呢？這裡感謝競技寶JJBONE的資訊支持。

我們來看看目前的總積分榜。D7G戰隊憑藉第二日的爆發順勢來到排行榜第一位，總積分92。而後面則是兩天表現都可圈可點的BA，BA只差一分。到了3/4名，SU4和ARK的分差也是非常有意思，都是跟前一名差一分。整個第一集團的分數基本都在5分之內，所以對於這幾支戰隊來說，誰能穩住拿分誰基本有更多主動權。

那麼誰能穩住誰又說得清呢？確實，沒人敢打包票的，畢竟PUBG也是個吃狀態的遊戲。我們這裡看看跳點好吧，D7G在東側橋頭的M城，TITAN在weibo出局之後直接進入了機場。而OMG則是老跳點十二宮加橋頭野區。這三傢俱樂部都在積分榜第一頁，如果前三局海島圖刷了艾倫格機場圈，那麼他們的分數會相應的舒服很多。

而按照黃金8之前的小規律來說，3局比賽很有可能會出現一局機場圈。雖然沒有科學依據，但小組賽6天18場艾倫格5個機場圈還是很能說明問題的好吧。所以一旦出現一局機場圈，這三支隊伍的機會無疑是最大的。那麼OMG能不能抓住機會就看這一下了好吧，D7G和TITAN之間很有可能圍繞優勢地形展開攻防。想想蠻刺激的。

既然說到了OMG，其實這次沒有人跟他們打橋頭還是很舒服的。目前OMG與積分榜第一集團僅差10分左右。不管是刷機場圈還是刷主島，西側橋頭這麼好的位置去哪都還算方便的（除非極限圈），所以能不能抓住機會一舉”破而後立”，完全就看他們自己了。老實說拉胯也快拉大半年了，年底還是給OMG粉絲一點盼頭好吧。

好消息是，獅子小海的手感真的還算不錯。

另一邊則是SMG，雖然他們比之前我們分析的OMG僅差5分，但這5分對於第一集團來說就非常可怕了。目前SMG與第一集團基本上差了15-20分左右，如果最後一日他們想爆發，那就意味著每一局SMG都不能白給，至少拿到5分以上才基本能保證追到前面的隊伍。而前面的隊伍也是要拿分的。這就要求他們還需要爆發一兩局拿高分。

這兩支G系戰隊任意一支奪冠，對苦了大半年的粉絲來說都比較安慰。畢竟不管是OMG還是SMG，從6月開始幾乎就打得一團糟，SMG甚至還出各種風波，不管是夏季賽的”燈”還是後來的隊內訓練賽吵架，輿論眼始終看著他們。OMG則是成績一直拉，跳點問題也是一直掰扯，核心小獅子又是跟A+起了一波節奏。所以這個冠軍還是很重要的。

各位觀眾老爺們對此怎麼看呢？歡迎留言討論哦，最後求關注求點贊謝謝大家啦。
