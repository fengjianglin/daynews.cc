EDG下路被點名吐槽，米勒感慨廠長執教，管澤元調侃被警告

天天要聞
2019-12-23

英雄聯盟2019德瑪西亞杯12月22日，在第一天的比賽中，許多LPL隊伍的新人和新組合都已經悉數亮相，而今天B組和D組的十支隊伍也將爭奪最後的四個八強席位。

德杯第二日EDG的首秀就格外引人矚目，廠長Clearlove轉型教練之後，穿上一身帥氣西裝登上了BP台，而此前同為選手的老將Pyl也站上了解說台。連解說米勒都不禁感慨：明凱當教練了，Pyl當解說了，我的青春結束了。

在昨天德杯首日的比賽中，A組和C組率先決出了四支晉級八強的隊伍，分別是：擁有「絕代雙蕉」組合的VG、失去了大舅子和兮夜的WE、四戰全勝的JDG，以及沒有Uzi的RNG。

而在今天B組和D組的對決中，另外四支挺進八強的隊伍也已經出爐，分別是：廠長帶領的EDG、中野代練的TES、花名別緻的RW，以及最後憑藉勝場平均時長晉級的BLG。

其中EDG在分別戰勝GD和LGD之後，在第三把中輸給了V5，不過又迅速調整狀態，在最後一局中力克三連勝的TES，最終憑藉勝負關係以3-1的戰績小組第一出線。雖然廠長的執教能力和50W冠軍上單的精彩發揮得到了不少網友的稱讚，但是ADC選手Hope卻不被看好。

特別是在EDG輸給V5的那一局中，Hope的表現更是被粉絲點名吐槽：「ADC真的菜，LPL能被壓刀，LDL也能被壓刀，團戰永遠祖傳閃現，總能率先陣亡，這就是綜合評分高於iBoy？」、「我今天就噴Hope，拿了一血打成這樣，女警開大滿血不擋槍，高地一波好團大招開成這樣，這AD你告訴我比iBoy強？」、「高分低能，回去打你的排位吧」……

雖然遭到了很多網友質疑，但是阿布卻依然很看好Hope，在EDG的比賽結束後，阿布開啟了直播，並且回應了許多粉絲對於Hope的指責，阿布表示自己從Hope身上看到了兩個字，希望。不過到底是阿布獨具慧眼還是真的高分低能，只能看EDG之後的表現了。

順便一提，在解說米勒感慨完廠長執教之後，管澤元也進行了轉發，並且還順帶皮了一下：我當觀眾的時候喜歡的米勒還在解說，我的青春還在……不過這句調侃卻引來米勒夫婦的「親密互動」，米勒口吐金蓮，米嫂緊隨其後，我酸了……

言歸正傳，本屆德杯小組賽已經全部結束，御三家也全部晉級，「家沒了」的WE最讓粉絲驚喜，廠長不變的眼神也讓EDG的豬仔們欣慰，但是沒有uzi的RNG和薛定諤的小虎就顯得不那麼讓人安心。

雖然德杯只是LPL隊伍新賽季前的試煉和磨合，甚至像FPX和IG都只是上了二隊過來練兵，但是既然來了，誰不想順便拿個冠軍呢。之後的淘汰賽都是BO5賽制了，比賽也將會更加精彩，只是解說能再專業一點的話那就更好了。附上八強賽賽程：
