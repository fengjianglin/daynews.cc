完美世界牽手中青社取勢“新主流”共推遊戲“青年標準”

2019-12-18 575條評論 4,350次閱讀 258人點贊

12月17日，完美世界控股集團有限公司與中國青年出版社正式簽訂戰略合作協議。未來，雙方將共同成立「中國青年遊戲電競戰略發展研究中心」，聯合研發和制定符合《中長期青年發展規劃（2016-2025年）》國家戰略、四億中國青年核心需求、權益與未來發展趨勢的遊戲、電競產品「青年標準」，並在全行業試行和推動「青標」，共同促進遊戲電競職業化、行業化、產業化和國際化有序發展，引導和規範中國青少年健康「遊戲」與電子競技「智慧」發展。

圖為完美世界副總裁伊迪（左）與中國青年出版社副編審、中國青年智庫論壇執行秘書長庄庸（右）簽約

當下，世界處於百年未有之大變局之中，「青年為世界立法」，已經成為席捲全球的時代潮流。四億中國青年是互聯網社群治理和社會治理體系重建、國家治理體系現代化和治理能力現代化、全球治理體系變革的生力軍——是謂「主流新青年」。

在此大格局之下，中國青年正在重新定義文學、影視、動漫、遊戲、電子競技、文創集群等新的形態與業態、技術標準、創新機制體制，甚至重塑整個生態系統——比如從「網文」到「網劇/網片」再到「網漫網遊新文創」，四億中國青年正在制定文影漫創「超級IP宇宙」架構設計的新標準，以此為好槓桿，撬動話語權、輿論權和文化領導權的「青年權」之爭。

遊戲和電競成為中國青年文娛風尚、青春潮流、新輿論場和思想生態重塑的時代切口。中國青年對遊戲和電競的熱衷，已經不僅僅限於遊戲與電子競技內容的娛樂消費，而是將其作為新的社交、社群和社區生活方式，作為其「年輕的力量」（為世界立法的青年權）聚集、分布、引爆社會潮流並成為時代風向標的新公共話語空間。在全球市場，各類遊戲中30歲以下年輕用戶均是核心玩家，佔比均超過60%；而在中國市場上，中國青年甚至正在開啟迥異於過去的「玩家時代」以及未來已來的「智慧競技時代」。

小切口，好槓桿，撬動大格局。提煉、研發與制定遊戲電競的「四億中國青年標準」（簡稱「青標」），已不僅僅是規劃整個遊戲競技業職業化、行業化、產業化和國際化的業態發展，而且切中四億中國青年核心權益、需求與未來發展趨勢的潮流涌動，更是在學習、貫徹和落實《中長期青年發展規劃（2016-2025年）》的國家戰略。

在此大勢和趨勢之下，研發符合國家戰略意志、國民價值取向和四億中國青年核心權益訴求的「青年標準」勢在必行。遊戲電競領域的「青標」更是一個前沿、前瞻和前衛的重大現實攻關問題、時代課題和未來發展趨勢問題。

遵循問題導向的思路、邏輯和智慧，近年來，為了促進遊戲行業的健康有序發展，使得遊戲行業肩負起文化傳承和傳播的責任，國家相關部門一直在不斷加強行業的規範化發展管理。尤其是在保護未成年人健康遊戲發展方面，不僅推出了《兒童個人信息網路保護規定》《網路生態治理規定》等政策法規，還在新的《未成年人保護法》（修訂草案）之中，特別提出對未成年人使用網路遊戲實行時間管理。在社會主流輿論和媒體領域，人民網推出了《遊戲適齡提示草案》，並號召全行業進行遊戲適齡提示。

此次完美世界牽手中青社，迎潮而上，正是基於宏大的願景和共同的意願，「聆聽時代的聲音」，取勢而為、順勢而行，聯合提煉、研發和制定中國遊戲和電競的「青年標準」，旨在推動中國遊戲及電競行業良好發展，服務於四億中國青少年，引領中國青年遊戲競技領域健康發展。

據了解，中青社是面向全國青年的中央一級出版社，直屬共青團中央領導。自1950年建社至今，出版了一大批高水平的、反映時代精神的、具有重大影響的書刊，影響了幾代青年讀者，創造了「服務青年，引導青年」的中青品牌，在中國青年中間具有巨大的影響力。在黨和國家發布了《中長期青年發展規劃（2016-2025年）》之後，按照共青團中央「構建四億中國青少年思想引領平台」的重大戰略部署，中青社加速進入轉場、升級和跨界戰略軌道，整合資源、戰略卡位、全版權運營，旨在重塑中國青年新文創集群的形態、業態和生態系統，引領新一輪「主流新青年」的青春潮流、輿論情報和思想生態重塑。

完美世界則是中國最著名的數字文創企業之一，不僅擁有眾多贏得青少年用戶喜愛的數字文化產品，還特別關注對青少年的保護和中國傳統文化的傳播。在青少年保護方面，完美世界擁有強大的防沉迷體系。在日前《遊戲適齡提示草案》的推行中，完美世界作為遊戲企業的代表之一，與人民網和多家企業共同發起遊戲適齡提示，同時也是第一批參與草案修訂、第一批進行適齡標記的遊戲企業。在文化產品打造方面，完美世界始終注重對傳統文化的繼承和發揚，完美世界許多產品都取材於中國傳統經典。

兩家在中國青年特別是「主流新青年」之中都具有相當的品牌知名度、美譽度和專業度。中青社具有豐富與完備的青年經驗和組織系統，洞悉並掌控中國青年需求潮流和核心權益驅動之源；完美世界則有技術優勢和產業優勢，具有研發、供給並滿足「主流新青年」需求的產品、項目運營和增值服務實力。兩者攜手，可以研發和制定出符合國家戰略意志、國民價值取向和四億中國青年核心權益與需求的遊戲電競「青年標準」。

同時，根據協議，雙方還將合作，開發以遊戲競技為創意窪地，貫通出版、文學、動漫、影視、遊戲競技等全形態鏈、全產業鏈和全價值鏈的「完美少年IP宇宙」和全版權產品與項目等，形成「新主流青年」需求-供給聯動互動、同頻共振的機制和模式。

以此為基礎和前提，兩家將進一步合和推動「遊戲電競超級IP」成為兩岸三地年輕人樂於共享的話題和內容，建構與打造中國青年和全球青年文化交流的新領地，切中中國-全球青年的青春潮流和思想脈動，製造和引領席捲中國-全球青年的新文藝潮流、新文創符號（文創集群）和新文化運動，書寫為美好生活奮鬥、構建人類命運共同體的新史詩。
