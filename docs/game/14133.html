琳琅滿目的衣櫥中尋覓亮點 換裝小遊戲如何讓女性用戶“為愛發電”？| 遊戲茶館

2019-12-11 273條評論 1,319次閱讀 116人點贊

早在Flash和頁游時代，換裝遊戲便是熱門品類，手游時代更是出現了暖暖系列等爆款產品。小遊戲時代來臨，簡單易操作且對女性用戶有著極大吸引力的換裝玩法成為許多開發者選擇的方向。

近期，《愛江山更愛美人》、《暖暖的衣櫃》、《菲夢少女之偶像物語》等產品在QQ小遊戲平台上表現優秀，在休閑類熱門榜單中佔據前列。《愛江山更愛美人》僅QQ小遊戲平台上玩家就達到696萬，據阿拉丁指數統計目前該產品排在小遊戲榜單第21位。

頭條和抖音小遊戲平台上，圍繞《葉羅麗精靈夢》IP打造的系列小遊戲也累計百萬級別用戶。百度小遊戲中心上，3D社交換裝養成產品《心動練習生》也獲得了「少女心萌游」版塊的重點推薦。茶館綜合分析了上述多款產品的玩法與商業化設計，圍繞其針對女性用戶的設計進行分析，希望給到各位開發者幫助。

《愛江山更愛美人》：換裝、宮廷、戀愛與戰鬥的集大成小遊戲 設計豐富沖榜活動

《愛江山更愛美人》由奇妙互娛研發，目前已經上線了QQ、微信等小遊戲平台，各平台註冊玩家總數應該已經達到千萬級別。從遊戲品類的定義上，可以稱之為宮廷模擬小遊戲，但實際玩法中又包含了換裝、戀愛、戰鬥和養成的多元素內容。

世界觀上主角作為為科研工作穿越清朝的現代女性，代替了去世的陳婉君的身份，在宮中與各方勢力展開角逐，不斷提升地位的同時解鎖更多的服裝和藍顏（男性知己）。從畫風選擇上看，這款小遊戲非常重視女性用戶的審美，主要界面選擇粉紅底色，搭配花瓣和流光設計，人物採用較為細膩的現代大眼畫風，服裝設計突出奢華和古典。

換裝作為遊戲中最重要的養成元素，從第一個關卡就要求玩家改變著裝，通過關卡的通過和金幣的積累可以換取到更多外裝。外裝除了是通過劇情關卡的必然要求外，在後期的一些沖榜活動中也需要比拼，遊戲還設計「時裝周」玩法，是通過玩家自行外裝搭配進行比拼的模式。

在換裝之外，遊戲還設計了主線與番外劇情，玩家會逐步解鎖隨從、藍顏、宮務、蜜會等進階玩法，可以發展與各類NPC的能力。雖然遊戲本身的數值設計略顯複雜，但是為了符合小遊戲平台的輕體驗，劇情都添加了自動和跳過選項，玩家的偏向選擇並不會對最終劇情走向產生影響，也減免了玩家的遊玩壓力。

值得一提的是，《愛江山更愛美人》選擇了以內購為主的商業化路線，由於玩家的最終能力與隨從、外裝、藍顏等能力有關，所以圍繞這些要素設計了VIP等級和獎勵。首充會贈送玩家一套稀有風格外裝，達到68元贈送有特殊能力的隨從，高級隨從在劇情戰鬥、特殊人物會成為玩家的助力。如此設計內購，可以讓玩家得到特殊隨從的優勢回饋後產生繼續充值的動力，解鎖更多特殊隨從，為增加用戶粘性，內購上也設計了月卡和年卡。

為了更好刺激消費和回饋付費玩家，遊戲不定期的推出各種各樣的「爭妃榜」，比如和藍顏角色的親密沖榜、與藍顏角色約會的蜜會沖榜以及遊戲內資源消耗沖榜，刺激玩家進行消費的同時也給到付費玩家大量遊戲資源反饋，形成良好的持續生態。

《暖暖的衣櫃》：多部位合成換裝 添加性格測試等吸引女性用戶的玩法

《暖暖的衣櫃》是一款合成換裝的輕度產品，相較《愛江山更愛美人》的重度玩法，這款遊戲更適合放置休閑。遊戲由南京紅豆公司研發，目前在QQ小遊戲休閑類熱門榜單排名第9。遊戲的美術設計比較精緻，外裝按照頭髮、外衣、頭飾、鞋子和手持掛件，每個部位都有專門的合成欄。

南京紅豆CEO秦川表示，《暖暖的衣櫃》並不是對標宮廷換裝類小遊戲，而是對標合成類產品。這款遊戲設計上的亮點，在於更加順暢的合成體驗和數值設計。很多合成類小遊戲合成基礎元素的獲得就需要觀看廣告，但在《暖暖的衣櫃》中初始合成材料的獲取沒有太大門檻，通過掛機金幣積累可以源源不斷的添加。

而廣告的設計在於每次合成高級服裝後，主界面會出現Banner廣告和分享選項，玩家也可以選擇觀看廣告解鎖更高級的初始服裝。在基礎外裝外，還有更高品質的稀有和天賜外裝，需要玩家使用特殊勾玉兌換，勾玉可以通過觀看視頻和分享遊戲獲得抽獎機會獲取。合成行為和服裝提供的掛機金幣收益，都可以通過觀看廣告進行自動合成和收益翻倍，該處使用了廣告置換玩家勞動力的規則。

除了上述設計，《暖暖的衣櫃》設計了更加偏向於女性用戶的功能，遊戲主界面提供了拍照功能，方面玩家截圖保存自己的穿搭並分享。還添加了根據外裝搭配進行性格分析的功能，同時在邀請好友數量滿足2位的情況下，會開啟人物的前世今生分析，加入性格測試和玄學內容更容易提升女性用戶好感。

《女神之路》：用網紅主播與渣男背叛等劇情吸睛的合成換裝小遊戲

《女神之路》選擇的是與《暖暖的衣櫃》同樣的合成換裝玩法，只不過由於劇情元素的加入，這款產品更偏中度一些，目前排在QQ小遊戲休閑熱門榜單第12位。主角是一位遇到渣男前男友的女服裝設計師，在遭遇背叛和創意被盜後，勵志通過服裝設計和直播進行創業。

核心玩法設計上，玩家在合成區域進行上衣、下裝、鞋子等多部位的合成，由於並沒有按品類分開合成區域，所以合成體驗上會有一些混亂。隨著高級服裝的合成，逐步增加星級可以解鎖進階劇情、支線劇情，並提升自己的咖位（賺錢能力）。

商業化設計上，通過觀看廣告可以解鎖更多的合成位，獲得額外物品升級獎勵和翻倍簽到獎勵。通過邀請好友、社交互動和觀看廣告，可以獲得高級服裝的抽取金幣，可以抽到非合成服裝。與《暖暖的衣櫃》最大不同在於，《女神之路》加入了很多戀愛劇情上的設計，加入了男友系統。玩家將經歷主線遇到渣男、艱辛創業、獲得幸福的劇情，也會經歷被各種名人帥哥約會的支線劇情，就鎖限定男友角色可以看男友劇情。

通過簽到獎勵、合成服裝升星，可以解鎖遊戲內的追求者角色，追求者會在主界面顯示外觀，並給予玩家賺錢能力的加成。除了加入渣男背叛的吸睛劇情外，遊戲將主角的身份設置為網紅主播，支線劇情上也設置了聯合直播選項，主界面會飄過很多網路熱梗彈幕，營造真實的網紅體驗，這對於女性玩家更有吸引力。

《葉羅麗精靈夢》：打造多款IP換裝小遊戲 吸引低齡女性用戶

前面三款QQ和微信上的換裝小遊戲面對的用戶年齡階層是比較寬泛的，而接下來這一些列頭條和抖音上的換裝小遊戲，或許對於低齡女性用戶的吸引力要更大一些。由成都雷獸科技發行的《葉羅麗精靈夢》小遊戲，選用了3D國產動畫片《葉羅麗精靈夢》的IP，以換裝為主體衍生出 《葉羅麗美顏公主》、《葉羅麗娃娃店》等系列小遊戲。

《葉羅麗精靈夢》主要面向青少年群體，以各位仙子在葉羅麗仙境發生的正能量故事為藍本展開，向觀眾宣揚善良和美的教育，近年來已經成為優酷、騰訊等視屏平台幼兒頻道的熱播動畫片。小遊戲的製作中，選用了各位仙子的形象，逐步解鎖更多的仙子可以積攢更多金幣，用金幣可以抽取到高星級外裝，玩家可以搭配套裝和自由搭配，在仙子之路模式中與其他玩家進行評分對戰豆兒提升自己的等級。

《葉羅麗美顏公主》在原IP偏寫實3D畫風的基礎上進行了Q版畫風改造，以更接近低齡女性用戶的2D Q版畫面為主。核心玩法為收集仙子能量，逐步解鎖更加高級的裝扮與美顏功能，將金幣抽獎改為了點擊製作，加強了玩家的參與感，抖音小遊戲顯示有66萬人在玩。

商業化設計上，兩款遊戲所需要的金幣和仙力都可以通過觀看激勵廣告獲得，在裝扮比拼玩法中，玩家裝扮、抽獎和比拼的過程都可以錄屏成為小視頻，通過抖音和今日頭條平台發送出去，自動帶有小遊戲的標籤，天然幫助小遊戲進行了二次推廣。

另一款IP小遊戲《葉羅麗娃娃店》則是模擬經營和換裝玩法的結合，玩家需要經營一家葉羅麗仙子娃娃的售賣店，需要按照用戶的需求打扮自己的葉羅麗娃娃，通過積攢金幣獲取更多的外裝，並通過外裝比拼提升等級，目前該遊戲正獲得抖音小遊戲今日最佳的推薦。

換裝小遊戲要圍繞女性用戶需求設計 QQ小遊戲更加傾向該品類

一名女性小遊戲開發者在採訪中表示，換裝小遊戲的品類目前屬於比較火熱的方向，尤其是年輕用戶和女性用戶佔比高的平台，比如QQ和抖音。目前來看與合成玩法的結合，是換裝小遊戲一個比較優秀的方向，QQ小遊戲也在加大對於合成換裝類產品的推薦力度。

他認為從根本上來說，換裝類小遊戲還是偏向女性用戶的方向，遊戲的核心玩法設計必須要關注女性用戶的真實需求。比如對於多種多樣華麗服裝的需求、對於遊戲操作簡易的需求、對於遊戲界面是否簡潔美觀的需求。

這些都是由於女性用戶一些偏好取向而決定的，女性用戶對於一款遊戲的付出和熱愛從根本上來源於「為愛發電」，所以從「愛」的角度研究更多玩法將會收穫更多女玩家的好感。比如《愛江山更愛美人》中與藍顏角色的蜜會模式、《暖暖的衣櫃》中的性格測試與前世今生、《女神之路》中頗為現實的渣男與網紅劇情，都以愛為核心進行內容設計，引起了女性玩家的共鳴，以換裝玩法為引導讓其「為愛發電」。■
