《玫瑰與黃昏的古城》—遊戲簡評

2019-12-16 1,813條評論 3,486次閱讀 202人點贊

遊戲介紹

《玫瑰與黃昏的古城（A Rose in the Twilight）》是Nippon Ichi Software, Inc.製作NIS America, Inc.發行的一款恐怖平台解密遊戲。講述了女孩在一座沒有色彩和時間的城堡里醒來，身上有著荊棘詛咒。在遇到一個巨人後，兩人合力逃跑。玩家幫助他們解開荊棘詛咒和陰暗城堡歷史了解真相的故事。

畫面玩法

本作採用了卡通畫風的遊戲畫面，齊耳短髮，吸取血液而綻放的紅色玫瑰開在小女孩身上 ，而不行動的時候就像是一個蝸牛殼的巨人，Q版的人物設定十分有意思。舒緩沉重的音樂聲下，幾滴鮮血「滴答，滴答」的掉落在沉睡的小女孩頭頂，這是一個沒有色彩的世界。2D的遊戲場景一片狼藉，只有身負荊棘的小女孩因為鮮血而有了顏色，她前進著，不小心從上面掉下來趴在了一塊大石頭面前，巨石攔路無法移開也無法跳過，只好一路向下走，期間還不小心因為跳躍摔倒了幾次，找到了一個捲軸。了解到了荊棘詛咒的存在。荊棘會長在受詛咒的人類身上，並且會吸收所有的時間和顏色，而物體有著自己的時間和顏色，在被吸走後，留下的就是靜止不動的軀殼，並且如果在半空中被吸走，那麼它就會停留在半空中。靜止的物體是無法移動的。吸血血液後會得到血液原本主人的記憶，遊戲畫面就像是坐在觀眾席看一場舞台劇。

系統玩法

遊戲的操作方式十分簡單，並且支持滑鼠點擊，遊戲玩法很有特色，控制荊棘吸取顏色，並且把顏色轉移懂沒有顏色的物體上使它的時間流動，從而達到解決謎題繼續前進。遊戲中以不錯文筆來描述遊戲的操作方式，比如「重生之力」自然就知道有著這個東西的地方自然就是死亡之後的重生點了。遊戲有著自殺的選項，可以在無法移動或無處可去時使用，可以切換小女孩和巨人來進行遊戲場景，比如用巨人夾著小女孩行走，可以保護小女孩避開落石，直接把小女孩扔到去不了的地方，之後再合作一起過去等等。

遊戲感受

本作給我的感覺有點像《螢火蟲日記》剛剛玩過不久所以印象頗深，卡通化2D橫版的場景移動方式，禁止時間的城堡，而雙主角切換的操作玩法讓遊戲有了更多的趣味性，居然可以抓住主角前進，可以抓住有顏色的道具，比如讓主角給凳子或者石頭移動顏色，我們的居然抓起來扔出去或者找一個地方放下，這裡要注意，有時候居然扔出的東西有可能把主角主角砸死。。。或者把主角扔出去摔死了。所幸遊戲中設置這重生點，我們可以在一個發光的物體那裡復活重生，再次進行我們的旅程，當然你們要小心像我一樣，無意間把自己卡在縫隙中出不來，捂臉，只能使用自殺選項重生，該說製作者是真的設置了一個好功能嗎！遊戲的畫面整體看上去很舒服，配合舒緩沉重的音樂給人一種陰森感，而吸取鮮血有時會讀取到鮮血主人生前最後的記憶，通過這些死亡片段來尋找接觸詛咒的辦法並且得到真相，使人更加有進行下去的慾望，遊戲中可以隨時選擇自己去過的地牢在地圖上進行傳送，解密設計也十分巧妙，拿到鑰匙開了門但是主角死了重生之後門還是開著呢，不用繼續去拿鑰匙了，不過要注意，並不是要一直是帶著紅色玫瑰的狀態，留著位置吸取別的物品血液會是明智的選擇。遊戲移動依然過於緩慢，並且沒有官方中文。

遊戲特點

非常不錯的遊戲氛圍

小建議

希望簡單難度不要限制存檔，最好隨時都可以存檔！

總結

本作採用了卡通畫風的遊戲畫面，Q版的人物設定十分有意思，沉重的BGM給人一種沉浸的感覺，稍微可惜的是本作尚不支持官方中文。推薦解密類遊戲愛好者嘗試本作。
