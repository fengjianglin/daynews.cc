灌籃高手手游：新手玩家入駐攻略，學會傳球，別做獨狼搞隊友

諸君安好，霧夏菌報道。

本月5號灌籃高手的手游已經正式上線了，國內很少見到灌籃高手的遊戲，那麼這一回合就來看一些這款遊戲，有哪些需要注意的地方。

1、遊戲操作

遊戲匹配成功後，會進入選人階段，選人結束後就會進入遊戲。首先是一個跳球，正常會由中鋒進行爭奪，有櫻木花道的一方，有可能會由櫻木花道跳球。

1、遊戲中的細節操作有很多，持球和無球會有兩套分別不同的技能，需要注意的是，普攻按鍵投球，會有一個起手動作，也就是需要長按才能將球投出去，這個抬手動作也是給了對方防守蓋帽的機會。

2、再一個一些投球類技能，需要達到規定的技能釋放區域，才可以釋放，點擊對應的技能釋放按鍵會有提示。大招的話需要積攢能量值，這個能量值可以通過遊戲中的一些操作獲取。但不要以為有大招就可以為所欲為，中後期的櫻木花道，連一些大招都可以蓋掉！

3、在籃下的時候，一些高個子或者彈跳力驚人的角色，普攻能夠觸發灌籃特效，具體觸發機制本菌也不清楚，正要研究的時候，遊戲又炸服了，所以只能等一段時間（現在開局不炸個服，都不好意思說自己做遊戲的）。

4、遊戲打的是半場（也有全場），所以當在對抗中，另一方搶到球時，需要走出三分線外才能進行另一方的進攻。

2、角色陣容

雖然是半場，但是因為每個位置的角色，都有自己的特性，所以也需要一定的陣容搭配。

正常的情況推薦的是一人控球帶球，串聯全場，推薦的是後衛角色，控球或者得分後衛都可以，主要是他們的帶球速度快，靈活，不容易被阻撓搶斷。

再一個需要一名投手，可以是小前鋒，也可以是得分後衛。他們可以承擔隊伍的得分主力，當然，也可以用大前鋒湊，增加籃板能力。

最後是防守位置，主用的是中鋒角色，大前鋒也可以偶爾客串，譬如遊戲中主打大前鋒的櫻木花道，雖然身高在中鋒中不佔優勢，但憑藉一手超高的彈跳能力，比一些常規的中鋒還要猛。

3、其他

主線劇情基本上都是在過以前的動畫，特別有那個味兒，回憶感滿滿的，適合PVP打累了休息的時候看一看。

再一個遊戲中一定要積極傳球，不要做獨狼，那個真的很噁心。當然這個傳球也不是亂傳，看好隊友的位置，跑動傳球找機會，既不能無腦突內線，也不要不看彼此的距離亂傳球。

菌說

說起來很搞笑，遊戲之所以炸服，是因為遊戲登錄的玩家，遠超他們的預料。原本能容納幾萬人的伺服器直接被懟翻了。可能他們自己也習慣了，因為早前他們也做了很多動漫手游，但至少有一半都涼了，所以出現了這次的尷尬事件。

好了，這一回合到此結束，感謝閱讀，拜拜了您嘞~
