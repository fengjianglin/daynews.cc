德杯八強賽程公布：VG與RW恐成黑馬，廠長能否再創佳績？

2019-12-23 184條評論 4,481次閱讀 2752人點贊

伴隨著英雄聯盟德瑪西亞杯兩天小組賽的激烈角逐，本屆德瑪西亞杯的八強名單與賽程也已經全部出爐。在上半區分別是由VG對陣TES、EDG對陣WE；而在下半區則是由JDG對陣BLG以及RW對陣RNG。

從這八支隊伍在德瑪西亞杯小組賽中的表現來看。

VG戰隊：「絕代雙蕉」果然名不虛傳，成為了VG戰隊最為穩定的Carry點；

TES戰隊：Karsa與Knight的中野聯動成為了這支隊伍制勝的「武器」；

EDG戰隊：新人上單小祥表現出色，廠長執教之下的EDG又打出了很典型的EDG風格；

WE戰隊：應該是在小組賽中最讓人感到意外的一支隊伍，新人選手，特別是新人上單選手表現都非常出色，有衝勁；

JDG戰隊：作為德瑪西亞杯小組賽中唯一一支全勝晉級的隊伍，伴隨著打野選手Kanavi的成長，JDG應該也是本屆德杯中配合最為默契的隊伍；

BLG戰隊：國豪與新中單FoFo的中野聯動表現可圈可點；

RW戰隊：有很多人把目光放在了EDG新人上單身上，但RW的新人中單Ruby也是在小組賽中表現最為出色的新人選手，連續三把穩定Carry的中路將RW帶入了八強賽；

RNG戰隊：沒有Uzi的RNG更多是要憑藉著上中野來Carry，從小組賽中「跌跌撞撞」進入八強賽的RNG想要走得更遠，上中野必須要站出來了。

那麼，從八強賽這四場比賽的對陣情況來看。

VG對陣TES：

雖然TES的中野在小組賽中的表現幾乎呈現出了「碾壓」之勢，但是VG戰隊的雙C表現卻也並不遜色。更何況，主教練Kkoma很有可能會在八強賽開始之前趕回VG戰隊的基地。所以，在擁有Kkoma的情況之下，VG即使年輕，卻也有資本可以與TES一戰。如果說，VG能夠贏下TES的話，Kkoma也將會開啟另一段傳奇。在「絕代雙蕉」出色的表現之下，並不妨礙Kkoma將VG打造成為一支「小SKT」。

EDG對陣WE：

從小組賽的表現來看，EDG與WE表現都相當出色，而且兩支隊伍也都擁有著出色的新人上單選手。但是，從WE在小組賽中的對手來看，並沒有像EDG這種比較「老辣」的隊伍。所以，我們暫時還不知道WE真正的極限在哪裡。而EDG很明顯就成為了WE最好的「試金石」。當然了，作為廠長轉型成為教練的第一個賽事，EDG也勢必會全力以赴的。

JDG對陣BLG：

不得不承認，JDG不管是在戰術上還是隊伍的磨合程度都是要優於BLG的。所以在對陣的結果方面，我們也是可以大膽預測JDG能夠拿下BLG。但基於八強賽還有段時間，而在這段時間之內，BLG能夠提升多少，也將成為他們能夠在德杯中戰勝JDG最後的時間了。

RW對陣RNG：

RW在小組賽中表現最穩定的就是他們的新人中單選手Ruby，而RNG在小組賽中表現最不穩定的就是中單選手小虎。就像之前所分析的那樣，在沒有Uzi的情況之下，作為隊長的小虎明顯是要承擔更多的壓力與責任。而小虎發揮的好壞也將成為是否能夠戰勝RW的關鍵了。

最後，如果把S9賽季稱之為「新生不息」的一個賽季的話。那麼，在這個新賽季中，新生恐怕已經開始茁壯成長。就像那句經典的名言一樣，「未來，始終都是他們的！」
