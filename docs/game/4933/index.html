DNF：官方最不喜歡的BOSS話題投票，玩家為什麼最討厭人類背叛者

2019-12-03 0條評論 14次閱讀 0人點贊

摘要

DNF官方在wegame的主頁上開啟了一個話題投票，參與投票的玩家還能隨機抽取30Q幣100名、20Q幣200名，10Q幣300名的獎勵，目前已經有2萬七千人參與了投票，為什麼這麼少人？不是說800萬勇士嗎？

DNF官方在wegame的主頁上開啟了一個話題投票，參與投票的玩家還能隨機抽取30Q幣100名、20Q幣200名，10Q幣300名的獎勵，目前已經有2萬七千人參與了投票，為什麼這麼少人？不是說800萬勇士嗎？怎麼才2萬多玩家投票，其實大部分玩家並不喜歡用這個登錄遊戲，因為會黑屏很久，而用這個登錄遊戲的玩家也不會去注意這一個資訊內容，活動時間是11月25日至12月10日，反正是隨手一點中了也還行。

話題投票是玩家最不喜歡的BOSS是誰，投票中給了8個領主作為選擇，分別是：普雷伊希斯（3237票）、宇宙惡魔羅什（1660票）、光之惡魔歐格里（3000票），星之守護者扎迪亞克（1261票）、甘霖之雲（2506票）、天弓亞莉珊德拉（5004票）、暗之惡魔艾森以及人類背叛者賓柏（7859票）。其中位居榜首的是人類背叛者和弓姐，其次是伊希斯和光惡魔，那麼這些排名前面的英雄為什麼這麼討人厭呢？

最不受歡迎的BOSS-賓柏

從超時空之戰，超時空漩渦，單人超時空以及強者之路，賓柏的身影一直相隨相伴，一個這麼勤懇的領主為什麼會成為玩家最討厭的怪物呢？主要是因為他的攻擊方式太「損」了。

賓柏最常見的攻擊方式就是追著玩家下劈攻擊，這一個攻擊在普通模式中傷害不高，但是在強者之路被碰到一下就沒了，並且因為這樣的追擊方式導致很多缺少位移的職業不得不吃藥，儘管在單人模式弱化的情況下，很多玩家也能被賓柏各種戲耍。

此外團本作戰是投擲的暗蝕槍全中秒殺，烈焰衝刺的連續攻擊，這些對於很多職業來說非常的不友好，很多玩家都說明明可以單人漩渦不用復活幣，卻總是被賓柏給破了，所以從90版本到95版本，我最討厭的 就是這一個怪物。

當然，一個掛這人類背叛者的稱號，能不被人討厭嗎？

弓姐

如果說普雷中你最不想打的怪是誰？當然是花姐和弓姐這兩個領主了，因為都是這兩個怪物都長得這麼美，但是花姐打起來簡單，而弓姐卻讓人非常的頭疼。

和其他怪物不一樣，弓姐是一個邊快速位移邊施放技能的怪物，玩家需要不斷的去追擊還需要時時刻刻躲避弓姐的各種屬性箭矢，失明冰凍應有盡有，並且和其他的怪物虛弱機制不一樣，組隊的時候弓姐需要引導五次金箭攻擊才能破防，這需要每一個玩家都非常的熟練才能完成破防機制，雖然在當前版本玩家傷害溢出已經不需要虛弱機制了，但在單人模式下還需要引導3次才能破防，對不熟悉的玩家來說非常頭疼，對缺少位移技能和無敵小技能的職業來說每次引導都是一次自殘，並且弓姐的快速位移，眼看就要砸下來的時候她就飛走了。

在這一個排名中普雷伊希斯和光惡魔的投票數量差距不大，而我個人覺得伊希斯其實更簡單好打，但是光惡魔比暗惡魔更討人嫌。

光惡魔

光惡魔和暗惡魔都需要打字，但光惡魔的打字是有讀條的，如果在讀條沒有成功的打字破防，那光惡魔就會進入狂暴，狂暴時有短暫的再次打字時間，如果還沒有全部打完就會召喚大量的光束攻擊玩家，最後直接爆炸秒掉一些脆皮玩家，再從新開始打字。

單人模式下比較容易，但是組隊時需要隊友也能熟練打字才能輕鬆通關。

在普雷版本來說，其實弓姐我個人覺得鬼劍士偷學個小抓就能非常的輕鬆打字難度不高，星光聖地的領主其實也不是很難，而最讓我討厭的怪物是七色鹿。

七色鹿

和墮落聖地和星光聖地同樣是擊殺分身的破防機制，但七色鹿作為一個阻擊部隊的怪物，破防難度比聖地的還高，玩家需要分別擊殺7次艾伊克的分身，並且還需要將分身控制好在合適不重疊的區域，擊殺7次分身後需要按照羽毛的擊殺順序依次觸碰才能達到破防。

相比其他的阻擊怪物，七色鹿可以說是最令人討厭的一個，需要花費更多的時間，並且攻堅戰時還需要隊友共同的合作才能輕鬆的破防。

那麼你最不喜歡的是哪一個BOSS呢？
