閃耀暖暖官方公開7R玩家流水 紅包系統暫時關閉 氪金大佬遭遇戲耍

天天要聞
1 週 ago

對於近期鬧得沸沸揚揚的閃耀暖暖7R玩家事件，近日官方終於站出來發布了公告和聲明，針對這一事件有了比較明確的答覆，並公開了7R玩家流水，且做出了紅包系統暫時關閉的決定，不過這件事情帶來的影響遠遠超過了預料之外，來看看是怎麼回事吧。

首先是官方公開了之前7R玩家的流水，雖然不夠完全，不過基本上算是證實了該玩家確實沒有走太多的非正規渠道，也就是說數據上是一切正常的，還了人家一個清白，當然對於這件事情耿耿於懷的玩家還是不少，但起碼從目前的數據和簡單分析計算，似乎看不出太多的問題。

然後針對這次的事件官方決定暫時關閉紅包系統，當然只是系統的一部分而已，包括首次通關、成長大事、榮耀成就這三個紅包，保留了氪金還有套裝系列的紅包系統，也就是說試圖走紅包盟來獲取大量鑽石的手法，基本上宣告破產了。

對此部分玩家反對不滿，也有玩家支持的，當然這個是指暫時的，後續官方還會進一步進行優化，並作為補償，發放500鑽石+5個幻之券+200體力+10W金幣。原本按理說官方從這個處理事件的流程和操作上沒什麼問題，證明了玩家的清白，也進行了看似妥當的處理，同時也發放了補償試圖來一波無事發生，但是這波操作從根本上造成了無法估量的影響。

因為這相當於是官方變相承認了，閃耀暖暖就算是7R的氪金，高等級、高收集度、圖紙刷光、jjc復甦還能囤花囤鑽石，完全無壓力，歐皇玩家一毛不拔直接要什麼有什麼非酋玩家氪金和往水裡扔錢一樣。雖然7R玩家本身是無辜的，但架不住直接把這遊戲本質最容易讓人酸的地方給刺激出來了。

憑什麼運氣好可以省這麼多氪金的資本，遊戲本身氪金與否帶來的直接體感差異過大，直接出現了氪金大佬遭遇戲耍的嚴重問題。也就是說氪金後的遊戲體感反而不如BP，這就太諷刺，如果一個遊戲本身可以做到這種程度，無疑是非常失敗的。
