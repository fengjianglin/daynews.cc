LOL：WE新陣容豪取兩連勝，解說台腿哥嘴角止不住的上揚

2019-12-21 21條評論 1,031次閱讀 116人點贊

12月21日英雄聯盟德瑪西亞杯正式開賽，在今年的轉會期中WE主力大舅子選擇了回歸LCK賽區，WE的新陣容也是由新人組成，說實話很多人並不看好WE新賽季的表現，但首個德杯比賽日的小組賽中WE這套新陣容是豪取兩連勝！

在比賽中展現了以往WE賴以成名的團戰支援體系，第一局對戰LGE的比賽中，對於LPL觀眾而言新面孔的上中兩人，Teacherma中單夢魘以及Coten的阿卡麗都給人留下了深刻的印象，LGE組織的包下瞬間被上中雙TP打破，唯一的機會被兩人扼殺在搖籃中，WE輕鬆拿下開門紅。

第二局中WE對戰FPX二隊也進行了換人，中單Yimeng首秀登場，選到了發條，而FPX二隊則是選出了一套熟悉的陣容組合！

作為如今隊伍的老大哥打野悲傷，在這局拿到了自己招牌的奧拉夫也是帶動了全隊的節奏，前期的gank是將FPX下路給殺穿了

在中期的團戰拉扯中Yimeng的發條接連幾個大招都是完美的大到了多人，搭配Jiumeng賽娜的大招，讓FPX陣型被打散，沒辦法反打，April卡莎剛進場就被集火秒掉，這飛行的軌跡有內味兒了！

​

最終推進下路高地時，WE野輔強行開團，傷害方面WE全面碾壓，剛想回泉水的艾克和卡莎也被Yimeng發條大招拉出秒掉，拿下雙殺後，WE一波結束了比賽，拿下了兩連勝！

在賽後解說米勒也是用一個「猛」字評價了這支新WE！

而今天在解說台上解說的腿哥，嘴角一直止不住的上揚，再說「恭喜WE」時笑容完全抑制不住，粉絲在看完兩場比賽後也是揭棺而起，大家覺得這支新WE怎麼樣呢？
