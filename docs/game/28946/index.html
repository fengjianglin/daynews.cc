想不想修真：前期需要知道的小知識，別只知道提升境界

2019-12-20 217條評論 3,468次閱讀 565人點贊

大家好，這裡是汪先生，感謝大家觀看這篇文章。

作為一款修真萬年還只能是新手的遊戲，汪先生作為一個新手中的新手，小白中的小白，要為更為新手的初入遊戲的小夥伴們帶來攻略了，本文就講講初入修真界有哪些小知識需要了解，本文專為小小白提供，各位大佬們可以移步了。

首先初入修真界，既沒有宗門家族庇護，亦沒有親朋好友作伴，開局鍊氣期，功能靠修鍊解鎖。前期最主要的應該就是通過升級洞府來獲取提升靈根的資源了，洞府中分別有食物，木材，靈草，玄鐵以及靈石五個生產目標，其中第一個食物是一切的根源只要有食物儲量就算增長為0也是可以提供生產的，此外靈石是最沒用的，所以不用升級，因為我們要是靠這裡獲取靈石會餓死，另外前期大家都會生產食物招募僕役，而其實在後面通過地圖後就可以有源源不斷的僕役了，所以不需要一直用食物買僕役。

接下來是來自官方的溫暖，雖然孤苦無依，但是官方並沒有放棄了，只要達到一定的目標就可以在下面的入口領取獎勵了。其中在道友有何指教中有一個沒有要求人人都可以領的禮物，裡面有5000靈石以及靈草木材玄鐵各1萬，在前期還算是非常充足的獎勵了。

第三點，關於宗門。我們入宗門並不是為了學習他們的常規功法，也不是為了門派寶庫內的資源，煉器術煉丹術同樣不是我們的追求，更不是為了成為宗派供奉而努力。前期進入宗門最主要的就是學習他們的鎮派絕學了，基本貢獻夠了學完就可以跑了，這樣有助於你在境界低的時候快速學完1-3星門派的鎮派絕學，目前1到3星中斗狂，黃泉，逍遙，仙侶，天海，青羽這幾個門派都有鎮派絕學，如果錯過了大家可以去坊市購買信物。總之這是一個前期不斷叛教的遊戲，大家不要有心理負擔。

接下來補充第三點，前面我們說進宗門不是為了學習常規的技能，事實確實如此，我們前期學習功法的途徑並不是靠進入宗門學習，因為技能點實在有限，而功法卻是幾百上千部起步，所以其實我們前期學習功法的最主要途徑是坊市裡的仙書坊，第一層的書都很便宜，如果有靈石的話，建議每次都買空。

最後整理一下零散的點，首先裝備的話前期十件裝備都帶防禦，一直到中品法寶破魄袍就不要再去更換上品法寶了，而是應該等有玄武破魄袍圖紙後將裝備煉器之後再進行提升，免得後面還得從新再刷一遍裝備。此外門派寶庫內前期值得兌換的東西只有醉仙釀，而且換完就放著不要兌換道童什麼的，至少分神期沒到不要換，而且有些東西對境界要求就更高了前期簡直占空間，畢竟前期儲物空間有限，其他東西後面再說，此外門派獎勵沒啥好領的，也就一點靈石。

好了，以上就是修真小白帶來的一點前期比較實用的小知識了，不知道大佬們還有什麼補充的嗎？仙界的就先別提了，還早呢。
