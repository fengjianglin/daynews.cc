AG戰隊季後賽之旅開啟 既要復仇又要去證明自己

2019-12-21 600條評論 1,564次閱讀 120人點贊

CFPL S15季後賽即將拉開序幕，在首輪比賽中，AG、WE、KZ以及SN四支戰隊出戰，他們將打響CFPL季後賽的首槍，在這四支戰隊中，要說最受期待和關注的隊伍，莫過於AG這支戰隊了，無論他們他們的表現是好還是壞，他們都是場上最靚的仔。

AG戰隊在常規賽中的表現可以說是用糟糕來形容吧，尤其是常規賽中段的表現，更是以一段連敗而告終，最終經過九輪常規賽得到比拼之後，AG以四勝五負結束了常規賽征程，幸好後期發揮穩定，從而以第六名晉級到了季後賽。

AG季後賽首輪對手是WE，正是AG常規賽第九輪的對手，在第九輪中，WE和AG酣戰三圖，最終AG以2-1力克WE，獲得了一場非常關鍵性的勝利，正所謂不是冤家不聚頭。以現在AG的狀態，相信擊敗WE戰隊應該不成問題，問題就是之後的比賽。

AG若是擊敗了WE戰隊，將迎來兩個對手，分別是白鯊戰隊和R.LGD戰隊，要知道，這兩個對手可都曾在常規賽中給自己造成非常大壓力的對手，當然了AG對陣R.LGD的比賽是存在爭議的。常規賽第二輪，AG對陣白鯊，在這場比賽中，AG以0-2不敵白鯊戰隊，輸掉了比賽。

緊接著第三輪又面對著R.LGD戰隊，雖然最終以AG戰隊獲勝而告終，但這是一場充滿爭議的比賽，即使AG獲勝也不能說明就一定強於R.LGD，所以如果AG想要證明自己，就要去和R.LGD戰隊交手，去堂堂正正的擊敗他們。

可是這對於AG來說並不是一件難事，擊敗WE應該問題不大，問題就是在於白鯊和R.LGD，要想能夠成功復仇並且證明自己的實力，甚至於打進總決賽，就要跨過兩座山，只要能夠跨過白鯊和R.LGD，對於AG來說就是勝利。

12月21日，CFPL季後賽即將開啟我們期待著比賽的開啟，AG將於12月22日對陣WE戰隊，我們期待著AG戰隊打出我們所期待的表現，為AG加油，希望他們能夠翻過這一座座的大山！
