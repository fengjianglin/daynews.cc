夢工場大冒險測評：在跑酷的世界裡 再現正義與反派聯盟大對決

天天要聞
2019-12-23

作為世界三大動畫工作室之一的夢工場創造了許多深入人心的英雄故事。在《功夫熊貓3》里，大反派天煞橫空出世，以超強的實力擊敗這世間的眾多高手。而我們的英雄阿寶在關鍵時刻，成功召喚出神龍強勢逆襲，完成了拯救世界的重任。只是從烏龜大師手中接過權杖的它，命中注定還要對抗更多強大的敵人，迎接更多未知的可怕危險與重大挑戰。

▲《功夫熊貓3》宣傳海報

夢工場正版授權英雄冒險跑酷手游《夢工場大冒險》於12月20日正式開啟首輪安卓渠道測試。玩家可在遊戲中扮演阿寶、小嗝嗝等英雄，也可以選擇成為天煞、杜布瓦隊長等實力強大的反派，重現正義軍團與反派聯盟之間的對戰，書寫出專屬自己的英雄故事。

在昨日的評測中，我們已經為大家詳細講解了這款遊戲的相關特色，而在參與了兩日的測試後，我們對遊戲所主打的「多英雄冒險跑酷」主題也有了新的感受，今天就和大家分享《夢工場大冒險》中各類有趣的「英雄設定」。

▲《夢工場大冒險》全渠道測試於12月20日開啟

正反派大亂斗？ 正義軍團PK反派聯盟

首批開放的35位英雄中，不但有神奇組合夜煞小嗝嗝、絕世高手神龍大俠、紐約之王艾利克斯這樣廣受歡迎的正派英雄，還有不少出身反派的角色，包括寡婦製造者天煞、辣手女王杜布瓦隊長、大反派魅力王子等。大家非常熟悉的銀幕英雄，幾乎都能在本作中捕捉到其身影。而正反陣營的劃分和清晰準確的英雄定位，也讓大家在挑選收集英雄時，幾乎不需要花費太多的時間成本。

▲你最喜歡扮演哪一位英雄？

如來自《馴龍高手》的神奇組合沒牙仔與小嗝嗝，兩者之間相互扶持共同成長的真摯情感曾讓大家熱血沸騰，如今在遊戲里，他們依然是最佳夥伴，同樣呈現出人龍之間的精彩互動。遊戲中，伴隨著小嗝嗝與沒牙仔的共同飛翔，絢麗的閃電縈繞其中，似有萬鈞之勢的雷霆大作，將敢於阻擋在前面的一切敵人和障礙物化為烏有，攜手作戰的深厚友情可謂是破屏而出。

▲沒牙仔與小嗝嗝同屏作戰

同屬正義一方的紐約之王艾利克斯，出自電影《馬達加斯加》，在遊戲里同樣有了生動鮮活的演繹。在釋放出為它量身訂造的專屬技能「Showtime」時，它完全能成為全場最帥的仔，走出「六親不認」的步伐，不僅可瞬間進入熱鬧的馬戲團場景，更能跟著音樂節奏開始即興表演，不愧是中央動物園裡最受歡迎的明星。

▲艾利克斯不愧是帥爆全場的紐約之王

當然，官方對反派英雄的打造也是不留餘力。曾在《功夫熊貓3》大出風頭，擁有「苦難之師」、「復仇之獸」、「寡婦製造者」等稱號的超級大反派天煞，在遊戲中同樣延續了武功最強的設定，全屏破壞型技能可為後期困難關卡提供強大助力。不僅如此，天煞的形象也延續了翡翠宮的的設計風格，不僅是手裡的雙刃是由綠色翡翠製作而成，技能釋放時也是滿屏綠光，你品，你細細地品。

▲天煞技能爆屏效果無與倫比

相較天煞，《馬達加斯加》杜布瓦隊長的出場更顯威風霸氣，一身帥氣合身的警服，在惡魔口紅的映襯下，女梟雄本色顯露無疑。並且她可發動急速追擊，在摩托車上瘋狂射擊，傷害值超MAX。真的有點擔心，艾利克斯和它的三位好朋友能否順利逃脫杜布瓦隊長團隊的圍追堵截。

▲杜克瓦隊長的絕招十分厲害

在遊戲里，每位英雄都有最佳搭檔，與之組合作戰能產生強大威力。比如美食狂熱愛好者阿寶的最佳搭檔是超級煎包，其作用不僅僅是美味的靈魂美食，更有起死回生的效果。而隨著阿寶晉陞為神龍大俠，最佳搭檔馬上變為神龍捲軸，這時已可召喚神龍助戰，對抗一切強大的敵人。

▲阿寶技能擁有太極陰陽之道

▲神龍大俠無需集齊七顆龍珠即能召喚神龍

▲遊戲還設有貼紙功能，配合使用可獲得加成，並釋放專屬技能效果

回顧夢工場經典動畫電影，懸念叢生的感情羈絆總讓人心生嚮往。遊戲同樣融入這龐大而豐富的羈絆系統，當玩家集齊了同一部電影里的幾個特定角色，就會觸發羈絆，發生不可思議的效果。比如當史瑞克碰到費歐娜、魅力王子，會觸發「美女與野獸」羈絆，生命值獲得加成。而艾利克斯碰到杜布瓦隊長、企鵝軍團，則會觸發「頭號通緝」羈絆，獲得破壞值加成。

▲羈絆效果還能升級

這種羈絆關係，又讓我們恍如置身夢工場電影世界，正義聯盟協同作戰，反派集團全軍出擊，激動人心的大戰一觸即發。

創新生存競技玩法 英雄對抗來襲

在此基礎上，《夢工場大冒險》引入生存競技玩法——百人「搶羊大作戰」的跑酷新模式，將英雄間對抗的樂趣進一步提升。

▲搶羊大作戰十分考驗大家的反應

如之前我們了解到的一樣，本次測試上線的搶羊大作戰模式，主打跑酷與吃雞玩法融合的休閑競技，致力於讓玩家感受極速競技的快感以及多人競技的緊張刺激。在該模式下，十條賽道會隨機被毀，不同功能的道具會隨機出現，千奇百怪的障礙物會隨機變化，一切都特別隨機。

▲遊戲過程中可隨意變化賽道，避開可能的危險

豐富的局內挑戰已經非常考驗玩家的臨場應變和操控能力，但在這個基礎上，我們還可邀請好友們一起組隊，劃分陣營，開啟互相關愛互相坑害的非常之旅，這種體驗該有多刺激好玩，可想而知。是成為傳說中的豬隊友，還是變成能操控全場的實力王者，都在我們指尖之間。

▲搶羊大作戰道具效果多種多樣

此外，遊戲還提供公會對戰、BOSS戰等多種跑酷玩法，這種正派與反派的較量，只會來得更兇猛。

▲和小夥伴們一起對抗超強敵人

總結：

《夢工場大冒險》英雄聯盟與反派軍團的恩恩怨怨，在競技生存冒險等流行元素的放大下，變得更為有趣吸引。這也像極了一對歡喜冤家，始終相愛相殺，又別有樂趣，讓人慾罷不能。

目前看來，《夢工場大冒險》獲得的反響很不錯，測試服在taptap上的評分在9.0左右，此前還獲得了金翎獎和黑石獎，並先後被評為「玩家最期待移動單機遊戲」和「硬核年度最受期待遊戲」，綜合表現可圈可點。
