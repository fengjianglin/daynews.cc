《葫蘆娃》手游測評：令人回味的兒時味道，卻逃不過玩法單一

2019-12-24 161條評論 1,068次閱讀 112人點贊

今天筆者要不正經一回，給大家帶來的遊戲評測很特別，就是這款90後共同的童年回憶——《葫蘆娃》的同名手游。86版的《葫蘆娃》，不要說是80,90後心中的經典，就在當代，也是備受小朋友喜愛的。

葫蘆娃故事背景

看到這張圖，是不是有一種兒時動畫片的感覺呢？沒錯，這就是動畫正版授權的手游《葫蘆娃》，遊戲不僅原汁原味還原動畫中的經典劇情與人物設定，而且還增加了許多創新玩法，經典動畫在手游中得以延續，下面跟筆者一起去體驗下這款遊戲是否真正的繼承了我們心中的經典？

遊戲登錄頁

遊戲評分：6.5/10

遊戲類型：回合制

經典畫面再現，竟是回憶滿滿！

進入遊戲，我看到了熟悉的葫蘆山，熟悉的爺爺，熟悉的穿山甲，還有爺爺的茅草屋和葫蘆藤，一股原汁原味的感覺就這麼撲面而來了。《葫蘆娃》遊戲繼續延用了動畫中的特色畫風—”剪紙風”，高度還原動畫劇情、人物等，而且遊戲還在原有的基礎上進行了畫面升級，讓玩家們在遊戲時，會有一種看動畫片的既視感。

葫蘆娃初長成：一根藤上七個娃

遊戲的背景音樂和動畫如出一轍，在配音上也是採用了”童音”，如果不仔細分辨，完全聽不出和動畫中有什麼地方不同。看到這些，玩家們會不會也會回憶起和兒時玩伴一起看動畫片的場景呢？

大娃出生

遊戲不僅能打妖怪，還能開荒種地？

除了按照動畫劇情打妖怪救爺爺，玩家們應該會更好奇，遊戲究竟還有些什麼玩法呢？這就要說到遊戲中的DIY養成了，也是遊戲的特別之處。葫蘆兄弟們在救爺爺的過程中，必須不斷的獲取碎片，進行自我升級，不然後期會在戰鬥中有所乏力。但是碎片除了打怪掉落，還需要花銀幣去購買。別急，我們現在就需要拿著”搖錢樹”的種子去種植，這樣我們就能夠獲得足夠的銀錢去購買碎片升級了。

大娃升級：階級提升

另外，穿山甲兄弟在遊戲里會成為我們荒地的開荒者。當葫蘆娃從妖精手中救出被困的穿山甲，作為報答，穿山甲將會把玩家農場沒有開墾的土地開墾出來，以便於種植。而種子的獲取，也是需要我們在打怪的過程中得到，然後繼續回來”播種”賺取銀幣。

穿山甲兄弟幫忙土地開荒

種子成熟，收取錢幣

玩法太過單一，略顯無聊

雖然遊戲中的”副本組隊”玩法比較過癮的，當碰到的妖怪太過強大時，我們就可以召喚社交系統中好友一起下副本打”BOSS”。但是就是玩法上太過單一，基本沒什麼技術含量可言。就跟個人闖關打怪一樣，只需要等葫蘆娃怒氣積滿之時，在屏幕上點擊釋放及可。整個關卡，你只需要做的只有這一個動作。而且當你看到葫蘆娃血量過低時，就只能是逃跑，然而卻並沒什麼用，因為根本沒有其餘地方給玩家操控的空間，玩家只能看著葫蘆娃血空而亡，未免太過無奈。

三娃即將血空

另外，還有一點讓人比較無語的那就是農場玩法。若是沒有種子，玩家必須去刷本獲取種子，然後再去播種獲取錢幣，之後才能去升級。過程繁瑣且無趣，但是卻是必要的操作，不然玩家無法進行其他操作。另外，當你在通關後獲取到了足夠的種子，遊戲則會提示你去進行播種，而且此環節同樣無法跳過，也正是因為這種強制性的設定，讓遊戲的樂趣少了許多。

玩法太多單一，沒有技術含量

強制玩法導致遊戲體驗差

葫蘆娃大戰妖精

個人覺得，《葫蘆娃》作為一款情懷遊戲，在畫風，畫面，劇情的還原上可以的。從進入遊戲，到開始闖關，不論是劇情還是配音，都讓我們產生了”動畫的既視感”，真的是記憶中該有的樣子。但是作為一款手游，要想吸引玩家，遊戲的玩法如何還是占更大的比重。說實話，遊戲操作太過簡單且單一，只是粗暴地釋放技能，讓玩家的遊戲體驗一下子就降低了不少，雖然有副本組隊刷怪的玩法，但是也同樣難以掩蓋玩法很無聊這個事實。如果你只是對此抱有懷舊情懷的話，《葫蘆娃》手游還是不錯的。但是如果玩家更加註重遊戲性的話，那麼，這款遊戲顯然就不適合你了。
