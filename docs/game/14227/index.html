12月還有這幾款手游上線！卡牌、MMO、武俠、IP...最吸引你的是啥

2019-12-10 38條評論 2,848次閱讀 777人點贊

大家好，這裡是1號玩家小編-游小妹，將給大家持續帶來新鮮的手遊資訊！喜歡看的話不如點個關注噢~

在2019年的最後一個月，回顧一整年，哪款手游把你成功留下了呢？在這即將敲響2020鐘聲的本月，還有幾款熱門手游在本月上線呢！快來看看吧~

《陰陽師：百聞牌》-CCG卡牌-12月12日上線

《陰陽師：百聞牌》是一款無套路妖怪卡牌對戰手游（CCG）。
逾百位頂級畫師，以匠心繪製的精美卡牌，演繹《陰陽師》中耳熟能詳的經典角色。
置身穿行三界的秘境樓船中，化身為妖，與百鬼共游，以全新的原創規則展開燃情對決。
豐富的策略、公平的對戰、精緻的和風，這是你前所未見的CCG新體驗。

陰陽師IP加持下的CCG卡牌手游《陰陽師：百聞牌》，特點除了是打牌之外，手游里的精緻和風場景、配樂、卡牌立繪都是相當優秀的，即便是對打牌無感，你也會被這漂亮的動態卡牌迷到。

開頭的引入劇情也通通配上了日語配音，可以說是相當優秀可愛了。

網易的野心除了是擴展「陰陽師」IP宇宙之外，也是進一步網卡牌品類進軍。除了手上的《影之詩》、代理暴雪的《爐石傳說》之外，百聞牌就是下一步嘗試。不過據很多試玩玩家的說法來看，《百聞牌》的玩法並不像《爐石傳說》。

小編也曽在測試中成功擠上船進行體驗，要是頭回體驗的話，得花上些時間了解規則：比如為自己上場的四個式神挑選卡組，卡組裡包括普通技能牌、形態牌、覺醒牌，光看卡面描述可能就有點蒙；

然後是在打牌過程中對形態牌、覺醒牌、出牌順序、瞬發、貫穿等等概念的不理解，這些可都得好好了解，不然可能連劇情都會卡住。

目前百聞牌的蘋果端已經上線，安卓端將在12月12日上線，近日還公布了《陰陽師》X《決戰平安京》X《陰陽師：百聞牌》的聯動決定，真·我聯我自己，看來網易是十分重視這款百聞牌了，最近也是各種廣告都在大力宣傳這款《陰陽師：百聞牌》。

△可能出現的雷點：由於卡牌死亡後等待三個回合即可復活，加上其他必須攻擊放在陣前的卡牌妖怪等等機制，牌手那30點血真的還挺難耗死的，前提是在你還不熟悉機制的情況下，單局時長可能會耗時很長。

前期主要的玩法就是推劇情以及PVP天梯，如果你對打牌沒什麼耐心、又或是不喜歡PVP、畫風又不對你胃口的話，那麼入坑可能有點難。

《夢幻西遊三維版》-3D MMORPG-12月18日上線

《夢幻西遊三維版》是由網易夢幻西遊金牌團隊研發的次世代3D即時制角色扮演手游，為夢幻西遊品牌在3D領域的全新拓展。作為網易旗艦級遊戲，延續了夢幻西遊的經典玩法、獨特的國風美學以及強大的社交功能，並塑造了一個萬物有靈的西遊大世界。

作為網易遊戲老IP，夢幻西遊無疑是成功的，這次的《夢幻西遊三維版》也是延續經典，從2D到3D，不知道夢幻西遊的老玩家對這個三維版又有什麼樣的新感受呢？

首先最直觀的就是美術畫風的不同啦，Q版3D畫風建模，和以前完全不一樣的感覺和操作，捏臉、輕功、看風景，熟悉的職業又有沒有讓你想起當年的情懷？

還有一個經典元素：就是寵物泡泡了，你也可以在三維版看到他的身影。

《夢幻西遊三維版》宣傳師門、法寶、抓鬼、幫戰等等熟悉的玩法在三維下的體驗，指向的就是以前夢幻老玩家，以及新生代對這個IP感興趣的新玩家。

不過在大家對MMO有點疲倦的時候，單人師門任務、科舉、組隊抓鬼、普通副本、幫派周常任務、寶圖任務、保衛幫派、幫派跑商、地宮、押鏢等等常規MMO元素是否還能抓住你的心呢？

△可能出現的雷點：如果你不是夢幻老玩家，或者是散人玩家，可能會對老玩家的一些梗、術語、組隊方面有點困惑，其次如果你對這種卡通風不感冒，或者是對MMO常規元素感到疲倦，那麼這款夢幻西遊三維版對你來說就不那麼吸引了。

《新笑傲江湖》-3D MMORPG-12月19日上線

清韻國風武俠手游《新笑傲江湖》，是完美世界遊戲里程碑式的自研國風新品。遊戲取材於金庸先生的國民武俠經典，並且呈現上高品質且有辨識度的清新國韻視覺，如同從國畫中走出的旖旎山河。

作為金庸著名武俠IP授權手游，《新笑傲江湖》自然吸引了不少玩家的目光，最近也是定檔12月19日正式公測，還請了明星-肖戰來代言。

首先和其他MMO不同的是，它不是Q版不是寫實，而是偏水墨的國風國漫畫風，在上次測試中，捏臉的選擇也並不多，只是選擇五官和衣服，如果喜歡這種畫風的話，這會是一個亮點。

而《新笑傲江湖》宣傳還原武俠，還原江湖，並對金庸筆下的江湖進行了考究和深挖，不知道你又對這個江湖有什麼樣的期待呢？

從上測的體驗來看，首先劇情中人物配音表演都很到位，有些NPC還使用了方言，劇情方面有選項分支，不同選項影響玩家的性格。

後續還可以認識不同的你或者熟悉的NPC原著角色，像是東方不敗、令狐沖等等，當然還有其他玩法：遊戲也有寵物系統、裝備系統、坐騎、時裝染色、家園、輕功、師徒系統等等，不知道公測後《新笑傲江湖》還會給我們什麼驚喜呢？

△可能出現的雷點：在上測時模擬器上的畫面還比較模糊，不知道正式上線會不會更好，不過如果你對畫質、建模很挑剔，且不喜歡這種國漫畫風，那麼玩起來會比較難；

同樣是MMO，官方稱不設一條龍，自由江湖，看看上線後的日常肝度吧。

《雪鷹領主》-3D MMORPG-12月17日上線

以團隊PVP為核心戰鬥體驗，並重點實現「自創技能」戰鬥模式，玩家可以融合動作、特效、傷害等創造自身絕學，更將遊戲中的捏臉、時裝染色、獨創輕功、境界、法寶等系統玩法與原著完美融合；同時獨創短視頻社交功能，可隨時分享遊戲內大招決勝瞬間，成為網紅博主與百萬粉絲交流，結交同好征戰玄幻異界。

小編在整理消息時還發現了這一款騰訊出品的小說改編手游《雪鷹領主》，由《天涯明月刀》團隊研發，改編自【我吃西紅柿】同名小說，而且這個《雪鷹領主》早已在台服上線，但是目前宣傳力度並不是很大，還沒上線就達到了6.0分。

《雪鷹領主》的主要特色是：還原IP原著、自創技能、短視頻社交、團隊PVP等等，不知道公測當天能吸引多少玩家前捧場呢？

△可能出現的雷點：這款在台服上線已久的遊戲，主打社交交友，如果你只是來玩遊戲的可能會覺得有那麼一點奇怪，參考目前還沒上線得TAPTAP評分，你自己考慮一下咯~

以上手游你期待哪一款呢？2019年的你遊戲荒了嗎？玩的最久的又是哪一款手游呢？
