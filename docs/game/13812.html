《光環：致遠星》太火爆，微軟稱已有300萬玩家戰鬥過

2019-12-13 623條評論 2,833次閱讀 543人點贊

IT之家12月12日消息 《光環：致遠星》PC版已在12月4日凌晨2點解鎖，經過差不多一周的時間迎來高光時刻。

微軟Xbox遊戲營銷總經理Aaron Greenberg表示，期待已久的《光環：致遠星》重製版上周進行了「重大發布」。該遊戲於12月3日在Xbox One、Windows 10和Steam上發布，並且還可以在PC和遊戲機的Xbox Game Pass上使用。

下面是一些《光環：致遠星》的戰績亮點：

  • 在發布周內，PC和Xbox One上的MCC玩家接近300萬

  • 在發布周內Steam上排名第一，在Steam上獲得Xbox Game Studios冠軍頭銜排名最高的遊戲

  • 發行當天在Twitch上排名第一的最受關注的遊戲

如你所知，微軟不會分享其第一方遊戲的銷售數量，而是強調突出參與度。的確，在發布當天在Xbox Game Pass上擁有第一方遊戲肯定會損害銷售量，但是《光環：致遠星》是Steam最暢銷產品（與《帝國時代II：最終版》一起）表明，微軟沒有將其作為Windows 10商店獨佔的PC遊戲絕對是正確的選擇。
