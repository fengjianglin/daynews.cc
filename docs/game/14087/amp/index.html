12月PC遊戲前瞻，掏空你的錢包絕不手軟 主機玩家暗自抹淚

天天要聞
2019-12-11

遊戲玩家千千萬，主機玩家掉眼淚。身為一個主機玩家一定知道11月份在PC平台取得巨大成功的遊戲《荒野大鏢客2》。這樣一款投資巨大製作頂級的遊戲，主機玩家們想必不會錯過，儘管價格略高但也阻止不了擁有它的心。

《荒野大鏢客2》在PC端上的成功可謂是羨煞那些遊戲廠商，這不12月的移植遊戲強勢襲來，種類繁多琳琅滿目，不僅讓你眼花繚亂之外更是讓你的錢包跟你say goodbye！今天讓我們來瞪大雙眼來看看12月份有哪些精良的主機遊戲吧！

《勇者斗惡龍建造者2》

《勇者斗惡龍建造者2》將於12月11日登陸steam平台售價355元。日本國民RPG遊戲勇者斗惡龍是第一次登陸steam國區。該版本結合了主機板全部的DLC以及自帶繁體中文。一上來就是355的售價確實是讓不少玩家打了一個激靈，價格確實有點貴，但是一分價錢一分貨，看完介紹還是蠻讓人心動的。

建造者系列咋原本的RPG玩法中還加入了《我的世界》方塊建造模式，這樣的一種設定就使得遊戲變得具有非常高的自由度以及可玩性。非常適合那些喜歡探索建造又可以在閑暇之餘打怪升級的玩家。想要在深入了解的話可以去看看遊戲實況，對遊戲有一定了解後在購買也是非常理智的選擇。

《底特律變人》

如果你作為一個B站資深用戶的話那一定聽過《底特律變人》這個遊戲，作為B站最強的雲遊戲將於12月12日登陸epic平台售價280元。在雙十二這個日子登陸而不打折未免也太不地道了。不過也正常，畢竟這個遊戲鎖國區，而且之前5月份在俄區上線只需要37元。總感覺有點黑心有點坑，建議還是別購買了，280拿來買排骨他不香嗎？

《暗黑血統：創世紀》

《暗黑血統：創世紀》12月5日登陸steam平台，首發折扣價84元。玩家們對黑暗血統系列的遊戲想必並不陌生。作為業內最能作的大廠THQ旗下的暗黑血統每一代都是全新的題材，比如說一代是正統act，二代是arpg，反正每一代都有不同花樣嘛。

從目前的預告片來說看起來還是不錯的，希望不要”圖片僅供參考”吧！最主要的是相較於前面的動輒兩三百大洋的遊戲，這款遊戲的價格就比較親民了，買下來玩玩體驗一下也是不錯的呢。
