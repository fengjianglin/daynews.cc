戰爭來臨之時戰略物資倍速增長？《ATLAS》：看我如何漁船變艦船

2019-12-08 0條評論 15次閱讀 0人點贊

摘要

不知道大家有沒有發現，不管在什麼時候，只要一發生了戰爭，不管士兵還是戰船等戰略物資都會以很恐怖的速度增長，這很讓人懷疑，為什麼平時的時候這些資源都那麼稀有，一旦發生了戰爭就突然蹦出來了呢？對於這個問題，要是你玩過在Steam上線的海盜冒險生存遊戲《ATLAS》就能理解了，在《ATLAS》中有很多的船一開始也不會用來作戰的，可一旦展開了戰爭，就算是漁船都能搖身一變成大戰船了。

和一般的航海遊戲不同，在《ATLAS》中你不僅僅能航海，還可以根據自己的用途來造一艘大船，《ATLAS》還是一款沙盒遊戲。就比如說你需要往返《ATLAS》的各個島嶼採集資源，那麼你就會想要一艘運輸船，在造船的時候你就會注意這點，將船體改造的大且負重能力強，這樣你才能儘可能的發揮出這艘船的運輸作用。

而當海戰來襲之時，戰船損耗的速度可能會遠超你的想像，這個時候之前所打造的運輸船就能發揮巨大的作用了，玩家只要再對它稍微改造一下，放置幾個炮門，加固一下船身就可以讓它上戰場了，憑藉著超高的負重能力，運輸船能攜帶很多的《ATLAS》火炮，也就能比較輕鬆的將敵人給消滅。

同樣的，《ATLAS》的其他船比如說漁船、速度型帆船等等也都能在戰爭到來之時變成戰船，耗盡所有的資源和敵人展開生死格鬥。《ATLAS》中發生的現實中也一樣會發生，假如戰爭真的來襲，那麼全國的船隻說不定都要被徵用和改造，最後在海上和敵人展開殊死搏鬥，大家對此有什麼看法？要是你也想在《ATLAS》中造幾艘船進行改造的話，也可以在Steam上進行下載和體驗。
