1-10慘遭AG血洗！白老闆：這一年錢也沒少花呀

2019-12-08 0條評論 19次閱讀 0人點贊

摘要

12月7日，2019CFS世界總決賽展開首日爭奪，A組小組賽上演中國軍團內戰，由AG對陣白鯊，結果白鯊全場沒有狀態，反觀AG則全員爆發，結果白鯊1-10慘遭AG血洗，跌入敗者組！

12月7日，2019CFS世界總決賽展開首日爭奪，A組小組賽上演中國軍團內戰，由AG對陣白鯊，結果白鯊全場沒有狀態，反觀AG則全員爆發，結果白鯊1-10慘遭AG血洗，跌入敗者組！

正在直播的俱樂部老闆白鯊心態爆炸，本次比賽是白鯊重組之後首次征戰CFS，白鯊對隊伍的期待值很高，但今天白鯊戰隊的比賽表現讓他感到絕望，在直播中感嘆這一年錢也沒少花呀，咋就打的不行呢？

一句「花最多的錢、挨最狠的揍」最好的概括了白鯊此刻的心情。不得不說，有時候運氣真的很重要，隊伍重組的首個賽季就憑藉史無前例的五步槍陣容拿下CFPL冠軍，白鯊S14的運氣實在太好。但盛極必衰，奪冠後碰上席位制，這是白鯊運氣不好的地方。

如果沒有席位制，白鯊不可能更換陣容，相信新賽季的表現也不會太差。但正是因為席位制的施行，白鯊不得不打破原有陣容，引援也不給力，子豪上賽季狀態不佳，田豐休戰了一個賽季，白鯊戰隊整體表現的下滑也在情理之中。

回到本場比賽，白鯊一上來主攻A包點，但道具打得不好，導致進攻有些脫節，比分一直落後。再加上殘局處理的一些失誤，導致上半場1-8落後，心態全無，基本提前宣告了比賽的失利。

接下來他們需要從這場比賽中走出來，全力準備下場比賽。AG與埃及ANB的勝者組比賽懸念不大，白鯊大概率會與ANB爭一個出線名額。八強是由三個小組的前兩名加兩個凈勝分最高的第三名組成，由於白鯊1-10輸給AG，比凈勝分毫無優勢可言，下場比賽再輸就將出局回家。

最後，請CFer們停止對白鯊戰隊的指責，繼續為他們加油，我們相信白鯊會努力調整，也還有機會。期待中國的三支隊伍能夠進入八強甚至會師四強，共同捍衛世界冠軍榮耀！
