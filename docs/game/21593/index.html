CFS丨半決賽兩圖唯有馬哲KD超過1，網友感慨老馬太難了

2019-12-19 3,091條評論 4,008次閱讀 2769人點贊

BD認為SV是最強對手，SV認為BD是最大難關，這兩支戰隊曾經交手過很多次，並且各有勝負。這一次的CFS2019世界總決賽，BD以2-0戰勝SV。BD的槍法戰術的確很優秀，但SV自身的問題也不能無視。希望這一戰之後，SV能夠痛定思痛，重回巔峰。

在徹底失敗的那一瞬間，不只是選手失落，觀戰粉絲也很難過，看到默默不語的參賽選手，這一刻感同身受。但還是要回顧一下整場比賽，因為什麼失敗了。先來看一下選手數據，馬哲在異域小城kd1.33，在黑色城鎮KD1.23，他的發揮沒什麼可說，作為指揮有這樣的數據很強。但是，除了馬哲，其他四位選手在兩張地圖KD均沒有超過1。難怪網友都在感慨：老馬太難了。

除了馬哲，還有一個人有些艱難，他就是SV的突破手念舊。其實在之前網友就看到這個隱患，也可以說是SV的陣容體系問題。之前的SV突破手至少有兩位，念舊完全可以和另一位突破配合起來。如今的SV突破手只有念舊一人，偶爾雅哲會配合，但更多時候念舊自己沒人補槍，贏了就進，輸了就退。即使最初有很好的手感，也被磨沒了。每次看到念舊孤單的身影，就感覺難受。

這一回合，馬哲其實已經狙擊殺死敵方3人了，無奈隊友一一陣亡，馬哲還需要1v2，BD兩人形成雙卡，絲毫不漏點，馬哲無力回天。

SV組織進攻B區，排點都沒有問題，但是在突破的瞬間，除了雅哲第一個跳到B門被擊殺，其他人都擠在門口，被BD集火刷屏，丟失這一回合。

BD手握第一張地圖賽點，殘局4me要1v2。Pancada看到4me進入B區，但沒有開槍，可惜被4me回頭擊殺，4me假拆發現敵人，換手槍主動擊殺，驚險拿下殘局。

這是一場5v5回防戰，AG已經下包，BD回防。能看出SV的思路很好，一人繞酒吧橋偷背身，包點藏一人，其餘三人在附近防守。但是，包點先被擊殺，BD一人直接偷包，其餘隊友配合阻截，SV還有人存活，可惜BD已經順利拆包。

雖然比賽失敗了，但真正的強者永不服輸，也相信SV戰隊會重整旗鼓，重新站回世界巔峰！
