從端游黑馬到FSPL《街頭籃球》為何經久不衰

2019-12-09 0條評論 14次閱讀 0人點贊

摘要

隨著EHOME在杭州拿下冠軍獎盃，第三屆《街頭籃球》職業聯賽（FSPL）正式落下了帷幕，作為國內首屈一指的籃球類遊戲職業電競賽事，FSPL的規模或許比很多路人玩家想像的要更為龐大。

隨著EHOME在杭州拿下冠軍獎盃，第三屆《街頭籃球》職業聯賽（FSPL）正式落下了帷幕，作為國內首屈一指的籃球類遊戲職業電競賽事，FSPL的規模或許比很多路人玩家想像的要更為龐大。

　　2019年的FSPL一共邀請了12支職業隊伍參賽，其中有EHOME、AG這樣橫跨多領域的傳統電競豪強俱樂部，也有彩虹、頂級榮耀這樣始終紮根於《街頭籃球》的專一戰隊，而這12支戰隊要歷經八周的循環賽制才能決出最後的6強人選，其專業程度完全可以比擬目前各大一線遊戲的主流賽事。

　　從14年前籃球網遊中的一匹黑馬到今天三屆FPSL的圓滿完成，十餘年歷史，生命力絲毫不減，《街頭籃球》能在坎坷的體育遊戲發展之路中走出一條平坦的大道，實為不易。

　　▌夾縫中的體育遊戲市場

　　《街頭籃球》發展之不易自其誕生起就已經註定了，在遊戲產業份額佔比本就不算大的體育遊戲類目里，足球遊戲要佔去一半，而所有的籃球遊戲則還要在剩下的市場裡面臨一座大山——《NBA2K》。

NBA2K初代（1999年）

　　這款由國際頂級廠商R星、2K聯合製作的真實競技籃球遊戲以籃球屆頂級殿堂NBA為背景，於1999年發布初版作品，到《街頭籃球》誕生已經有了6年的積累。

　　頂級的技術、頂級的IP、長久積累的口碑，種種優勢讓《NBA2K》無情的收割了籃球遊戲市場絕大部分的份額，而所有的後來者都要給玩家一個理由，「為什麼我不去玩《NBA2K》而要玩你的籃球？」

　　頂著這些壓力，《街頭籃球》橫空出世，而街籃給出的理由就是他的英文原名——《Freestyle》，自由。

　　《街頭籃球》沒有選擇在真實競技上和《NBA2K》硬碰硬，而是另闢蹊徑選擇了嘻哈漫畫風格，遊戲定位也十分明確，不再是一個玩家操作一支球隊，而是只操縱某一位置的一名隊員，不再有複雜的遊戲規則，自由易上手、門檻低、節奏快，讓玩家能快速的體驗到籃球遊戲蓋帽、搶籃板、灌籃等核心樂趣，同時還能感受到團隊配合的激情。

漫畫畫風現在看也並不過時

　　這些突破創新讓《街頭籃球》在2005年成為了一匹黑馬，成功和《NBA2K》兩分天下，甚至街籃還更受國內玩家歡迎，一度在網吧都具備一定統治力。

　　此後，諸如《熱血籃球》《自由籃球》等所有非寫實籃球遊戲又再次迎來了《街頭籃球》這座大山。

　　但一時火爆並不能一勞永逸，《街頭籃球》隨後開始出現新玩法、角色匱乏，匹配機制落後等諸多問題，這也是所有網遊都要面臨的問題，很多優秀的網遊之所以銷聲匿跡並非是輸給了遊戲本身，而是輸給了時間。

　　《NBA2K》抵抗時間的方式是每過一年出一款新作，每過兩年就關閉前兩代的網路伺服器，就在今年《NBA2K20》發售後的不久，《NBA2K18》就宣布關服，但這種操作顯然是《街頭籃球》所不能模仿的。

　　那麼，《街頭籃球》如何破局？

　　▌街頭籃球的電競之路

　　所有屹立不倒的老遊戲都離不開老玩家們的支持，時間會在遊戲和玩家之間形成難以磨滅的情感羈絆，從而讓玩家在若干年後仍對當年的遊戲念念不忘。

　　如何讓老玩家們始終能與遊戲產生聯繫則是一款經典作品的必修課，而天游則很有遠見的開啟了電競計劃。

　　電競對遊戲正面影響不用多說，《英雄聯盟》就是一個很好的例子，哪怕玩家幾年不玩，但他們仍然會去關注最高級別的S賽，當玩家們看到iG奪冠，也會熱血洶湧，恨不得馬上回歸遊戲打上幾局乃至買上一套冠軍皮膚。

　　《街頭籃球》作為一款籃球競技遊戲，也有著天然的電競屬性，和LOL等電競遊戲相比，街籃比賽有著時間短、節奏快、強度高的特點，雖然一局只有短短5分鐘，但這五分鐘就能讓觀眾體驗到酣暢淋漓的反敗為勝，大部分比賽最後1秒才能決定勝負，即使不懂技術的玩家也能感受到那份扣緊心弦的緊張，這是其他遊戲給不到的獨特觀賞體驗。

　　而《街頭籃球》的電競歷史比《英雄聯盟》更為長久，在2005年，《街頭籃球》就舉辦了首個電競賽事——七省爭霸賽，為眾多街籃愛好者提供了交流競技的平台。

　　在2010年，全新的《街頭籃球》全國超級聯賽刷新了參賽人數，獲得「世界吉尼斯紀錄」榮譽證書，成為世界參與人數最多的電子競技比賽項目。

　　《街頭籃球》的電競之路一直走的相當順利，在其誕生的14年里，遊戲曾一共舉辦了10屆《街頭籃球》全國超級聯賽和6屆國際大師賽，為遊戲打下了堅實的電競基礎。

　　十年磨一劍，到了2017年，深知電競潛力的天游CEO宣布進軍電競職業領域，創辦了FSPL。自此，街籃的電競基業已經徹底穩固，在頂級賽事的作用下，老玩家們與遊戲的羈絆終於有了最牢固的紐帶，根據官方數據，2019年FSPL杭州總決賽現場VIP門票一經開售，便快速售罄，其影響力在體育遊戲圈裡已經可見一斑。

2019FSPL座無虛席

　　至此，有了成功的電競賽事加持，《街頭籃球》在體育遊戲領域就穩穩的佔下了一座山頭。

　　▌在電競之外

　　還值得一提的是，除了成功的電競，《街頭籃球》還具備一些其他體育遊戲所不具備的優勢，那就是情懷。

　　作為一款曾經的現象級端游，《街頭籃球》巔峰時期有著空前的熱度，作為大量Fser的共同回憶，《街頭籃球》的懷舊服在籃球玩家群體中幾乎擁有著和《魔獸世界》懷舊服一樣的潛力。

　　甚至作為競技端游，《街頭籃球》的回歸門檻更低，而天游在今年11月底開放的懷舊服無疑又是一招殺手鐧。

重返自由廣場

　　在技術上有頂級比賽可看，在情懷上又有懷舊服可去，走過14年的《街頭籃球》如今說是全副武裝也毫不為過，在找到正確道路後，街籃無疑能得到玩家們多方位的支持，而這些支持，將支撐《街頭籃球》有條不紊的走完更多個十年。
