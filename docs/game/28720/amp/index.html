VG高歌猛進打破WE連勝，Forge徹底起飛了，和Iboy雙焦組合太猛

天天要聞
2019-12-22

2019年的冬季德瑪西亞杯今天正式開賽，首日進行的是A、C兩組的小組賽。A組的戰況發展有些出乎意料，在失去了Mystic和Xiye的WE戰隊，以全華班的全年輕人陣容出戰德杯。本來以為他們是著重練兵，沒想到卻接連戰勝了LGE、FPX和SN，排到了A組第一名。第二名則是休賽期變化也非常大的VG戰隊，擁有新組建的Iboy和Forge的雙C組合，他們也是打出2-1的戰績。最後兩隊都在首日各自的最後一場比賽中相遇。VG處於藍色方，選出了瑞茲、酒桶、女刀、女槍和女坦。WE則是掏出了鱷魚、皇子、辛德拉、霞和泰坦。

5分鐘時，辛德拉被抓，蕉太狼收下了一血，同時WE雙人組越塔強殺女槍失敗。8分鐘時，皇子來到下路，WE這邊先手開打，Iboy的女槍殘血被酒桶大招救下後，VG反殺兩人。泰坦最後衝上來強換掉女槍，自己也送掉了人頭。10分鐘，皇子又來到下路，WE雙人組再次雙雙被擊殺。WE這邊Beishang的皇子只能被迫拿下先鋒，隨後順手來到上路和鱷魚一起把瑞茲抓死了。

​12分鐘，WE在上路推掉了一血塔，隨後鱷魚TP下路又救下被包圍的雙人組，同時也保住了下路一塔。15分鐘，WE忽然集結多人來到中路抓死了Forge的刀妹。在連續吃虧後，VG開始主動找了幾波節奏，先是在野區抓死了皇子，隨後在中路團戰中打出了0換2，又把優勢拿了回來。

18分鐘，VG忽然集結五人，在河道抓死了Beishang。隨後Iboy推掉了上一塔，酒桶還吃掉了峽谷先鋒。20分鐘，VG把先鋒放到中路，推掉了中一塔，經濟領先來到了四千。21分鐘，雙方在上河道又打了一波團戰。VG打出1換3，隨後開始打男爵，WE雙C過來阻止也被雙殺掉，隨後VG成功拿下了男爵。24分鐘，兩邊又在野區開打，WE又一次被擊殺掉四人，蕉太狼的刀妹也是殺瘋了，隨後WE中下兩路高地塔都被破。26分鐘，兩邊在中路又打了一波團戰，結果WE再次被擊殺掉了四個人。VG隨後中路一波結束了比賽。

​看了這場比賽，可以感覺到Iboy和Forge兩人表現非常猛。這組VG新雙C，兩人全場都是各種細節操作和carry表現，Forge打出9-1的戰績，Iboy則是8-1。在雙C如此發揮下，VG成功斷掉了WE連勝。LPL眾解說在比賽結束後，紛紛發出動態，感慨VG新雙C「絕代雙焦」。
