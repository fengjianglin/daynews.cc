《偏執狂：幸福是強制的》2020年登陸Steam平台

2019-12-11 1,853條評論 2,777次閱讀 521人點贊

CRPG遊戲《偏執狂：幸福是強制的》已于于2019年12月5日在Epic遊戲商城發售，但是現在《偏執狂：幸福是強制的》將於2020年登陸Steam平台，具體日期未知。


《偏執狂：幸福是強制的》這個 CRPG 遊戲發生於人類最後避難城市 Alpha Complex，整個城市在「計算機朋友」控制的下顯得快樂祥和。所有市民都快樂地生活在整潔的城市中。然而質疑「計算機朋友」就是反叛行為。取得超過你安全許可層級的情報就是反叛行為。拒絕喝每日劑量的跳跳汽水就是反叛行為。犯下反叛罪的人將就地正法。

玩家在遊戲中將扮演警戒人員，要注意隨時可能出現反叛，並計劃摧毀「計算機大人」的危險分子。但是，要時刻保持警惕，不要相信人任何人，甚至是自己的夥伴。在執法過程中，「計算機朋友」會監視你的一舉一動，警示任何反叛跡象，對你迅速保衛 Alpha Complex的英勇行為作出嘉獎，對你的你的反叛行為嚴懲不貸，甚至會拿你當反應爐屏蔽物。
