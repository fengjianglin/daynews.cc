王者榮耀：e星和AG在加緊訓練，QG卻在搞娛樂，或將雙線顆粒無收

2019-12-04 0條評論 9次閱讀 0人點贊

摘要

各位王者玩家大家好，我是小雨。距離KPL秋季總決賽開賽僅剩十天的時間，可以說目前的每一天都彌足珍貴。尤其是對於AG超玩會來說，能夠在兩年半後再次進入總決賽的舞台，他們比誰都珍惜。

各位王者玩家大家好，我是小雨。距離KPL秋季總決賽開賽僅剩十天的時間，可以說目前的每一天都彌足珍貴。尤其是對於AG超玩會來說，能夠在兩年半後再次進入總決賽的舞台，他們比誰都珍惜。功勛隊長老帥在戰勝JC之後也表示：「本來都打算放假了，沒想到進了總決賽，那大家也別放假了，辛苦一些加強訓練，我們去武漢拿冠軍！」

的確，AG的成員們也是這樣做的，目前他們正在跟eStarPro聯合訓練。自從e星被AG超玩會淘汰之後，他們也沒閑著，一直在跟AG打訓練賽。大家都知道QG最怕的對手就是e星，而e星總是能擊敗QG，也必然有他們的優勢和QG的弱點，他們既然能跟AG一起打訓練賽，恐怕也不會吝嗇把QG的弱點告訴AG。

說起來e星跟AG兩個俱樂部也是頗有淵源的。e星的老闆T將軍（xiaoT），以前打魔獸的時候出道的俱樂部就是AG，雖然現在的AG和以前的AG已經不是一個老闆了，但俱樂部名字沒變，還是有些情懷在裡面的，而且T將軍和AG現任老闆菲菲關係很不錯。另外AG現任教練月光、分析師兮夜、打野藍柚全都是從e星轉會到AG的。

不過淵源歸淵源，e星和AG一起打訓練賽可不光只是友情贊助，他們也有自己的目的，那就是為了冬冠做準備。輸掉巔峰對決之後，隊中的每個人都憋著一口氣，發誓要在冬冠王者歸來。所以他們每天都在加緊訓練，從和AG的訓練賽中找自己的不足，也算是互相支持互相進步吧。

​反觀QG，可能以為自己四冠在手，能不能再拿一個冠軍無所謂；也或者是他們覺得總決賽對手是自己的手下敗將AG超玩會，總冠軍已經是囊中之物了，放鬆了警惕。飛牛大將軍最近幾天一直在帶女朋友和暴風銳一起歡樂排位，昨天更是和隊友末將一起與汪蘇瀧、李九五排，而且基本不練自己位置的英雄。

可能有的朋友會說了，直播玩自己拿手的不是暴露嗎？但是打把的時間直播，不玩自己位置的英雄，總決賽不會手生嗎？自古以來驕兵必敗，如果QG一直這樣「娛樂」下去，不僅總決賽結局難測，等到冬冠e星歸來時，恐怕雙線都會顆粒無收。

記住科比的那句話：不要以為自己已經穩了。美國籃球尚且做不到，何況你QGhappy！努力加油訓練吧，不要讓粉絲們失望。
