爐石巨龍降臨：一邊娛樂一邊上分的燭龍賊詳解

天天要聞
2019-12-19

●前言

大家好，我是沉迷新構築的桃源。

新版本上線之後，我嘗試了很多個看起來腦洞很大的卡組，包括我之前發表和沒發表的好多套預購築；最終我選擇認真構築的第一套卡組便是燭龍賊。因為燭龍賊的超展開模式最出其不意，最具有節目效果，而且本身強度不低。但是我還是要強調，這套卡組暫時是娛樂卡組！

●卡組介紹

這套雖然帶的最高費用為6費，但是實際上是一個中速卡組；各種神奇的combo都至少需要3費跳幣死金藥劑師，然後死金匕首觸發亡語來操作。後續要養出更多的燭龍，則需要更長的時間。

●核心思路

首先，超展開模式有以下幾種：

1.偷渡者穩定抽兩張蠟燭再抽2；

2.實驗室招募員洗場上的燭龍再死金藥劑師；

3.伺機待發燭火吐息1費抽3。

由於現在本身環境就偏快，所以這裡只有超展開模式，而沒有穩定的致勝手段。而且上面幾種超展開模式，極其容易打出，我們只要比對面的卡組快就行了。

如果防戰還存在的話，那麼這套燭龍賊應該能輕鬆地把防戰打得喊爹。

●防快攻組件

背刺、悶棍、刺骨、背叛、日怒保衛者、末日預言者、甚至深潛炸彈；

這套卡組滿編了賊的幾乎所有解場卡，只因為前5費接近空過。至於鋪場卡組為什麼帶兩個末日…現在的版本前期卡組太多，5費可能不到10血了。

日怒給燭龍架牆非常厲害，75的牆還是很難受的，當然如果對面是薩滿就另算了。

●節奏體系

火樹巫醫、銳鱗騎士、死金匕首、雛龍巨嬰、龍喉噴火者；

由於卡組本身就是一個中速卡組，所以除了主要核心的部件以外選擇的是節奏卡，龍體系本身的節奏很不錯，一定程度上能緩解前期壓力。

這套卡組原本是從加拉克隆體系延伸出來的，迦拉克隆可以攜帶535龍，盜賊的1費法術池都比較優質；而且還可以給祈求卡觸發連擊。但是如果放棄了加拉克隆體系，可選擇的優質龍就變少，手裡有龍的時候更需要謹慎。

◆其他選擇

其實有些小夥伴會問，這裡實際上有時候還會出現過排不足的情況，為什麼不帶賊的加拉克龍？理由還是很簡單，前期卡組太多了。加拉克隆雖然能提高卡組的上限，但是幾張祈求卡都是虧節奏的卡，一兩張跟班很有可能救不回來丟失的場面。

那麼這裡給出帶迦拉克隆的版本。

迦拉克隆體系有一個很不好的地方，那就是手牌很難傾泄出去，卡手牌連偷渡者都不敢亂拍。

●對局分析

◆對陣快攻劣勢

薩，賊，德，戰，獵，術；

這些職業都有快攻形態，默認快攻。

起手留所有抗快攻組件，3費以下節奏卡，還有425抽亡語。

沒辦法，現在的標準環境極度缺牆缺奶，掉了一滴是一滴，t7獵都大殺四方，日怒在危急的時候給兩個1-1架牆也能護一下臉。但是如果末日和深水炸彈把對面的前期場面給解掉了，對面在6費內無法返場，那麼就到了我們的表演時間了，這個時候再給燭龍架牆，直接勸退。

◆對陣控制卡組小優

法，戰，牧；

起手直接找燭龍找過牌，儘可能早地營造出一波對面完全無法處理的場面（也就是兩個燭龍，不要多）。手上可以留著一兩個偷渡者防止對面一波解光，解光我們還能重來。現在控制型的卡組帶的大aoe可能也就兩個，針對薩滿的兩波88；然而我們的卡組光是偷渡者就有兩個，何況還能洗更多的燭龍。

但問題在於，控制型卡組都可能有斬殺手段。我的宇宙法經常9費斬殺，帶加拉克隆的防站拍下加拉克隆三個回合之內可以打出30點傷害。牧師還有火車王加墓地符文的combo……所以我們要做的是，6費做到兩燭龍，然後每回合t兩腳。控制卡組還是很難做到，同時兼具結場和收集組件。

●結語

感覺燭龍賊暫時已經沒有多少的開發空間了，最終也還是缺乏快攻抗性，但是燭龍賊對於控制卡組的威懾力還是很強。

燭龍作為外掛組件放入宇宙賊的構築中感覺效果更好，在宇宙賊中配合偷渡者也是錦上添花。
