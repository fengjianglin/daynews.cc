在魔獸世界懷舊服里，戰士應該如何打好奧山

馬上12月11號懷舊服就要開放奧山戰場了，作為多年戰士的我在這裡分享一下戰士在奧山裡的打法和小技巧,希望能對今天剛加入艾澤拉斯世界的新人戰士們有所幫助。

戰士在線性平推這個最重要的一路對弈中扮演著非常重要的戰術角色。 大家一定經歷過在漫長的山路上聯盟部落兩軍對壘，一兩個小時無法取得戰果的焦灼狀態，兵法上說以正合以奇勝，擁有無條件衝鋒技能的戰士顯然就是正面主力兵種了。

兩個陣容集團互A的時候，最重要的就是哪一方可以破壞對方的陣型，衝散對方的輸出集群，戰士職業曾經在初版中擁有不限目標上限的破膽怒吼，可以瞬間把整個輸出集群驅散，可以說所有奧山指揮都會把這我們戰士的這個特性擺在重點地位來設計進攻方案。

裝備上我推薦用血裝+盾，當然一鍵切換雙手武器宏也是必不可少的，每次復活後請各位輔助職業幫忙刷滿buff。進入戰線之前，請務必上馬，選好目標，然後聽指揮安排進入衝刺。另外，做個紅字的宏提醒後排跟上效果更佳。可以寫的燃一點效果更佳 /yell Today is a good day to die! For the horde!!!! 一波有組織的4人以上戰士衝鋒往往在焦灼戰局中最有效的推進方式。

第一波的戰士往往只能突進到對方的前排，但是不要慌，你也往往是最容易活著回來的，驅散以後，快速選一個安全方向的目標攔截回家就行。如果你被安排在第二波衝鋒，目標是對面後排的話，就勇敢面對吧。破膽怒吼，刺耳怒吼，但凡是群體減益的技能，用一個是一個。這時候你往往是被集火的，記住直接切狂暴姿態開狂暴之怒然後再切回防禦姿態。別傻了吧唧被恐懼飛出去了就很尷尬了。我們精英戰士可以被集火而死，可不能被風箏飛出去射死啊，太丟面子了。

總結起來，戰士在推線中幾乎不起到輸出作用，致死打擊那點戰術價值其實也微乎其微，因為幾十個人的大團，血量是抬不起來的，你上不上致死效果都那樣。不過即便如此，最適合奧山的天賦仍然是武器31狂暴20. 當你在山地攻防戰和房間戰鬥的時候，輸出和致死rebuff就非常重要了。戰士在一個健全團隊中，他永遠是以霸道的位移技能，致死rebuff為立足根本的。第二作用是會快速切姿態，用拳擊或者盾擊打斷別人關鍵施法。這就非常考驗玩家的小規模戰鬥的戰術意識了。

一個優秀的戰士，永遠是在觀察別人後排單位誰的關鍵施法最有威脅的。不過最重要的還是要結合實踐操作去不斷練習，多的內容不說了，我們11奧山號見。
