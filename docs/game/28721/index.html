陰陽師SSR雲外鏡實戰強度詳解：這個小和尚明明很強但過於脆弱

2019-12-22 356條評論 2,143次閱讀 1082人點贊

陰陽師有關SSR雲外鏡的實戰強度和遊戲體感，估計不少玩家很感興趣，因為測試服對於這位式神有了一波不錯的加強，所以現在評價主要以加強後的為主，這裡就帶來相關強度的分析，一起來看看吧。

式神定位

這式神目前很有趣，在實戰中不論是PVE還是PVP都有極佳的體驗，因為輸出能力和上限很高，加上輸出機制很茨林類似，但鬼火的消耗很常規，外加攻擊面板很靠譜，所以常規的PVE對於群體輸出上來說很不錯，有望成為新一代刷狗糧的最佳人選和候補，這個還有待實裝後來下定論。而PVP中其作為輸出式神的作用被放小了很多，實際上沒多大作用，主要是要其陽面的輔助功能，防控能力等等很不搓，鬥技也有很大的就業空間。

優缺點

優點在於全能，這式神的特色在於不偏科，技能看著複雜實際上用起來簡單，最大的優勢就是掌握好被動的反轉技能會很方便，驅散機制過於給力，開了技能就能完美的剋制對面的控制式神，輸出也很靠譜。

缺點過於明顯，這式神的核心硬傷就是太脆了，沒見過這麼脆的式神，開局直接出發鬼切被動效果的，一旦被輸出式神針對直接當場就掛了，這個小和尚明明很強但過於脆弱是原罪，雖然加強後很好地改善了這個問題，但血薄依舊有硬傷。

實戰體感

操作比較複雜，很吃隊友和配置，這個有點像御饌津但又完全不同，很考驗鬥技環境中對於局勢的判斷，強度方面絕對是有的，但是目前測試中體感不出來，個人很看好這個式神，硬核狗糧魂土挖煤都OK，鬥技也有不錯就業面，實裝後總有一種會成為神卡的潛力股感覺。這個小和尚明明真的很強的，個人很看好他的實際強度。

總結

抽是一定要抽的，因為這不是SP式神而是SSR式神，對於玩家來說，SSR的全圖鑑優先度可是高於一切的，所以和強度沒什麼太大關係，另外官方對於SSR的態度也是很慎重的，目前的新SSR至少要有一個穩定的就業面或者作用，SP可以不人權，但SSR至少要有點特色，所以對於其具體的強度方面其實不需要擔心太多。
