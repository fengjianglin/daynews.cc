大話西遊2最強的2級忽視混紫雲劍 煉化讓多少玩家羨慕

2019-12-16 525條評論 1,726次閱讀 116人點贊

​在大話西遊2中，1-10級的裝備基本上是十多年的老玩家才會用的，伴隨著11-16級裝備和仙器神兵的湧現，1-10級的裝備漸漸已經退出了歷史舞台，當年抗物理75的石頭裝備已經沒了，甚至當年的1-13級寶石都已經沒了，被新的寶石系統所替代。但是還有玩家依然用著2級紫雲劍，到底是咋樣的紫雲劍才能留在現在呢？一起和阿宅去看看吧。

哇塞，2級紫雲劍煉化直接忽視混10，一個7級寶石，一個8級寶石，總體屬性強混8.4忽視混15，真的厲害了。不過2級武器應該沒辦法開煉器系統，要不然還可以繼續提升忽視混屬性的，屬實給力。

再來看看這個十二級項鏈，藍字忽視混0.9，煉化忽視混2+強混5，如果那一點忽視火也是混的話，這項鏈真的無敵，太猛了。

再來看看這把十五級武器，原始忽視風20+強水49.9，妥妥的沒啥用，藍字屬性強防0.3，也是毫無作用，一看煉化，震驚了，克火28+三屍蟲狂暴27，多少男鬼做夢都想得到的屬性，就這樣浪費了。

最後看一張粉絲投稿的圖，原始忽視風30，藍字忽視風5.5，煉化忽視風5+克金29+風狂暴17,3個6級寶石，煉器忽視風3.6，風狂暴22，這真的也太猛了吧。

在大話西遊2中，大家還見過哪些神奇的煉化呢，歡迎分享分享，阿宅和大家一起討論。
