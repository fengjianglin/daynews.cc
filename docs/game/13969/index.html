選對就是可以為所欲為，魔獸世界懷舊服里最適合PVP戰場的種族

2019-12-12 1,040條評論 4,715次閱讀 1809人點贊

魔獸世界之所以陪伴了廣大玩家15年仍然經久不衰，除了豐富的遊戲劇情故事，設計理念出色的團隊副本外，還有一大原因就是對RPG遊戲PVP的貢獻，WOW可以說是第一個能讓玩家公平競技的RPG遊戲了。除了這一點之外，魔獸世界的種族特性也十分有趣並且充滿差異化，讓種族不單單是一個模型的區別。那麼趁著戰場開啟之際，我們來簡單說說比較適合戰場的幾個種族。

我們先說說雙方的種族中，公認PVP最有優勢的兩個種族：

人類對於聯盟來說是一個相當全面的種族，個人認為是人類最適合的職業是戰士，尤其是感知天賦，提高偵測潛行的能力(感知），在PVP插旗的決鬥中，對於潛行者和德魯伊來說簡直是無情的技能。3分鐘的CD可以說是讓潛行職業失去了先手的機會，戰場中也是攻守兼備的神技之一。其他種族盜賊的話，基本上也不願意去挑戰人類玩家。與此同時，人類種族聲望的獲取速度（外交）也遠超同職業其他種族，讓玩家們付出同樣的時間和努力，永遠是人類能最快拿到聲望畢業的獎勵，簡直是護肝天使。

獸人作為部落的經典代表種族，不止是外表兇悍，強大的血性狂暴也讓他的敵人們心驚膽戰。無論是PVE和PVP都能比其他種族帶來更高的傷害與治療壓力，其特性就是在帶來成噸傷害的同時，也會降低自己被治療的效果，會給自己團隊的治療帶來不小的壓力（被集火和抗BOSS的時候慎用）。

獸人還有一個最逆天的種族天賦堅韌——對昏迷效果的抵抗能力提高25%。有了這個天賦對靠控制技能吃飯的盜賊來說，簡直是一個噩夢！在PVP的實戰中，經常看到獸人抵抗了潛行者的關鍵控制技能（例如5豆腎擊），然後殘血翻盤的案例。所以一旦MISS了，那就基本GG了。我自己就為此鬱悶過好一陣，有一次去東瘟疫之地做任務，碰到了一個部落沒有滿級的小號，果斷下手！當時我是人類戰士，結果衝鋒被抵抗，攔截被抵抗，對方連坐騎都沒有下，等我上馬再追，人家就直接跑到了部落的營地，沖我跳起了舞來，然後坐鳥走了，感覺七竅生煙被小號調戲了有沒有！

還等什麼？喜歡PVP的玩家應該知道哪些種族更值得考慮了吧。這些只是給大家作為一些參考，其他種族也是各有各的優勢的哦，回來自己體會下吧，明天戰場就上線了，PVP的玩家的春天馬上就要來了。
