難言與職業選手組隊吃雞，接受全網水友狙擊，依然輕鬆吃雞

天天要聞
2019-12-21

虎牙的高人氣主播難言，大家都非常熟悉了，他的外貌非常的帥氣，聲音也很好聽，還很會撩小姐姐，所以喜歡難言的網友，也是非常多的。最近，難言還舉辦了一場特別直播，那就是跟職業選手一起組隊，接受全網水友們的狙擊。這樣的挑戰還是非常有意思的，而難言邀請的職業選手分別是XQF的傘兵、4AM的33、ELG的大將軍，都是實力很強的職業選手。

這樣的一支隊伍，實力還是很強的，而四人的穿著也非常明顯，更加方便了水友們狙擊。在比賽的準備階段，也是有不少水友加入其中，就想幹掉難言或者3位職業選手一次，場面也是非常的火爆。不過，這樣的強力小隊，戰鬥力還是很強，雖然水友們努力進攻，但依然被一一淘汰。

難言小隊從海島圖打到了沙漠圖，都是輕鬆吃雞的。路上遇到的水友小隊，戰鬥力還是弱了點，也是深刻的體驗到了職業選手的實力。雖然正面打不過，但水友們對於這樣的活動，還是非常感興趣的，而之後難言又開始了近戰武器的模式。

所有的水友和難言幾人，都用近戰武器來戰鬥，看到這樣的模式後，水友們再次激動了起來。槍法比不過，砍刀難道還打不過？為了激勵水友們，難言還表示擊倒自己的水友，可以獲得30q幣的獎勵。可惜，最後水友們依然不敵難言小隊，無論是難言還是3位職業選手，刀法都是非常強悍的，就這樣一路砍到了決賽，最後還成功吃雞了。

看到這樣的結果，網友們也是表示驚訝，職業選手和難言的槍法厲害，大家都知道的，但沒想到他們的刀法也這麼強，居然能砍到吃雞，真是太厲害了！
