遊戲王：從口胡到神抽，我的遊戲王不可能這麼歐

天天要聞
2019-12-05

摘要

遊戲王可以說是陪伴很多人成長的動畫了。甚至不單單是動漫，遊戲王這款集換式卡牌有些，也曾經風靡過一段時日，哪怕是在現在依舊擁有著不小的受眾。正如羅馬不是一天建成的一樣，遊戲王的規則也不是在一開始就完善的，而是隨著決鬥者們的智慧和劇情的推進產生的。

遊戲王可以說是陪伴很多人成長的動畫了。甚至不單單是動漫，遊戲王這款集換式卡牌有些，也曾經風靡過一段時日，哪怕是在現在依舊擁有著不小的受眾。

正如羅馬不是一天建成的一樣，遊戲王的規則也不是在一開始就完善的，而是隨著決鬥者們的智慧和劇情的推進產生的。

因此在早期的遊戲王當中，甚至連一個詳細的規則都沒有，哪怕是遊戲王原著當中的口胡都隨處可見。攻擊魔法卡的岩石巨人，能夠依靠口胡增加怪獸卡的效果。實際上開始的遊戲王就是一個不夠成熟的產物。

當時的遊戲王不成熟到什麼程度呢？就算是遊戲的先後攻順序也是在現世與冥界的之逆轉這套先攻無敵的卡組出來後才誕生的產物。因此可以說，開始的遊戲王，的確是一個不夠完備的產物。他作為一個作者在漫畫當中構思出來的一種遊戲，一個人的力量終究有著照護不到的地方。

幸運的事情是，有人看出了遊戲王這款遊戲的閃光點。這才讓這款遊戲沒有成為現在通常所見的召喚抽卡類的氪金遊戲的怪圈當中。

這樣的卡牌對於單純的收集慾望來說已經足夠了，但是如果想要讓遊戲王成為神奇寶貝那樣的大IP，這樣粗製濫造的規則是遠遠不夠的。正是出於這方面的考慮，遊戲王的規則在歷代不斷的完善。而且隨著靈擺怪獸，同調怪獸以及NO怪獸等等反人類的怪獸卡出現，這個規則也越來越複雜。

複雜的規則帶來的就是主角口胡的餘地的減小。但是為了劇情的精彩程度，製作組有時候不得不將主角逼入絕境。但是沒有了口胡，主角又要依靠什麼來翻盤呢？

當然是神抽了！

我認為最適合神抽狗這個稱號的，就是游城十代了。每次只要說出來相信卡組精靈的話，他總能抽到自己翻盤的關鍵。當然了，這裡的神抽狗不是貶義。卡片精靈的存在既然遊戲王脫離了單純的遊戲，又給了游城十代的外掛找到了絕佳的背書。不再將他們當作冰冷的卡片，相信自己的卡組，正是決鬥者的奧義啊。

（雖然我的卡組總是背刺我）

同樣的，神抽的存在也在動漫方面極大地豐富了遊戲的趣味性。正是因為有著動漫當中的種種例子，在我對戰的感覺當中，只要還有一點翻盤的希望，決鬥者就一定會死纏到底。而這種精神的出現也導致了遊戲王的傳播更加的火熱。至今為止，遊戲王在多個地區的賽區都證明了他的火熱程度。

所以說神抽才是遊戲王的奧妙所在。就算是這種起手直接勝利的局在遊戲王的歷史上也並非沒有。在沒有將對方的LP徹底的消除乾淨之前就絕對不算是勝利。取消了口胡的遊戲王終究憑藉著自己複雜但是嚴密的規則，得到了世界各國人民的喜愛。而這無疑證明了製作組當初對遊戲王從口胡到神抽的轉型的用心良苦。
