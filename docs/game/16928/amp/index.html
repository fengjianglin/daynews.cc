絕地求生中，學會如何快速開鏡，才能秒人於毫秒之間

天天要聞
2019-12-15

在直播中我們經常看到那些職業的大神級玩家，擁有剎那間開關鏡，秒人於一秒的特殊技能。他們躲在遮擋物的後面，突然探出頭打開鏡殺完人再拉回來，喘氣的功夫就完成了，讓普通玩家驚為天人。其實這個技能不難，只要跟著我做就能輕鬆學會。

第一步，打開設置，找到遊戲內容，選擇按鍵方式，把瞄準視角改成保持，最後應用即可。

第二步，打開設置，找到控制，選擇戰鬥裡面的/瞄準射擊視角，把這裡修改為滑鼠側鍵，如果沒有滑鼠側鍵就改成鍵盤上的鍵，這個鍵得是你平常射擊常用的鍵，把瞄準視角改成滑鼠右鍵，最後應用。

按照上面的設置，從此你會發現自己也能做到瞬間開關鏡，大大節省了戰鬥中所需要的時間。畢竟實戰中，哪怕你只比對方快了0.01秒，也能處處搶到好處。後期通過反覆的練習，學會快速的瞄準，提高自己的準度，就可以打出和大神們一樣的招數，你也就漸漸成為神級人物。

​
因為遊戲中一旦開鏡，人就基本無法移動，所以這一招快速開關，還能大大提高自己的靈活度。帶著妹子一起玩遊戲的時候，亮出這招你就是全場最佳。
