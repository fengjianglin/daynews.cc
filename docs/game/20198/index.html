游漫談：盤點2019年度最差評分遊戲，索尼PS4竟上榜6款：秀的飛起

2019-12-17 591條評論 2,638次閱讀 249人點贊

每逢年底，遊戲界就會迎來各種各樣的評選和總結。今年遊戲界最大獎項TGA年度最佳遊戲，已經花落《只狼》家，小島秀夫充滿爭議的《死亡擱淺》只能淪為陪跑遊戲，混得一個玩家票選獎和最佳音樂安慰自己。從TGA的候選名單上來看今年的遊戲質量明顯有所下降，回想起去年各路神作百花齊放，《戰神》《大鏢客2》打得不可開交，而今年卻又顯得毫無懸念。

在各路媒體的正面評選結束之後，他們又開始了新一輪的年度最差遊戲評選。這些評分低下的遊戲大多數都有著致命缺點，但相比於其他小遊戲來說卻又還有些知名度。簡單說就是那些看起來很想玩，但等入手後才發現是垃圾的遊戲。根據M站（Metacritic）的評分得出2019年年度最差10款遊戲分別是：

Top1：《永恆：最後的獨角獸》

該遊戲是一款登陸於索尼PS4與PC的一款RPG遊戲，其中PS4平台評分36分，當屬年度最差遊戲。玩家扮演一名精靈女神或兇猛的維京戰士，需要與被自己解救的最後一隻獨角獸一起踏上驚險的冒險之旅。整體上遊戲並沒有太亮眼的地方，動作僵硬，畫風精緻但卻充滿年代感，不知為何總有一種在玩十年前遊戲的感覺。可能這也是讓玩家們最無法接受的缺點吧，畢竟誰也不想在3D遊戲上去追求「懷舊」。

Top2：《Left Alive》

也不知道SE究竟還想不想在遊戲界混了，他們又做出了一款口碑炸裂的遊戲《Left Alive》。這款動作遊戲評分37分，登陸於PS4/PC雙平台。雖然遊戲在題材上比較吸引人，遊戲從三位不同主角的視角，講述了在新斯拉瓦一場殘酷侵略中，一個關於人類艱難求生的故事。遊玩時可以選擇不同的遊戲風格。但遊戲整體上給玩家一種強烈的PS2年代感，不自然的動作和場景的單調讓它像是個半成品，曾經一流大廠的遊戲竟變成這樣，真是寒心，我們還是期待著FF7吧。

Top3：《時光之刃》

Switch版本的《時光之刃》獲得了38分，成為年度倒數第三名的作品。該遊戲是科樂美公司發行的動作冒險遊戲，遊戲擁有十個章節的流程長度。遊戲內沒有地圖，全憑藉主角的羅盤探路，進入射擊狀態時遊戲變成了FPS遊戲，但這其中還加入了各種魔法rpg的要素，總體給玩家一種四不像的感覺，可能GE工作室並沒有做好充分準備。

Top4：《魂斗羅：RC聯盟》

Switch版的魂斗羅僅僅得到40分，曾經在FC時代的那個系列神作跌落神壇，以後不要再講魂斗羅出現新作就會封神，我們本以為FC時代是它的起點，沒想到卻是它的巔峰。3D斜俯視視角配合渣到爆的畫面，科樂美最終還是親手毀掉了他們一款又一款被封為「神跡」的IP。

Top5：《玩具屋》PS4平台，41分

Top6：《WWE 2K20》PS4平台，43分

Top7：《FIFA20傳承版》Switch平台，43分

Top8：《淹沒》PS4平台，44分

Top9：《零世代》PS4平台，45分

Top10：《毒梟：卡特爾崛起》PC平台：46分

以上便是此次M站所曝光出的年度最差10款低分遊戲，其中6款遊戲均來自索尼PS4，不乏一些玩家非常關注且期待的IP續作/新作，只可惜種種原因下讓玩家感到失望。

關注游漫談，御三家攜手Steam，遊戲資訊搶先看。
