DNF：界·武器的100級後續！被舊版屬性取代，普雷特效作廢了

天天要聞
7 小時 ago

隨著DNF的100版本曝光，地下城與勇士重回巔峰似乎又穩了，因為裝備系統目前來到得到了解決，不再是枯燥的一套史詩升級幾年！而是重回85時期的「百家爭鳴」，並且開放了免費轉移打造，平民玩家樂開花，其次就是職業方面，三次覺醒能拖至少1年，只要再解決「團本無聊」這個問題，那麼100級的DNF還是很不錯的，但新版本更替，95B史詩怎麼辦？

界·武器的100級後續！被舊版屬性取代，普雷特效作廢了

DNF玩家不用擔心的就是95B的打造，包括普雷武器在內，均可轉移到100級裝備，不少玩家有一個疑問，那就是「夜語黑瞳武器」與「界·武器」的後續，在100級發布會中，絲毫未提兩者的區別，僅介紹+12夜語黑瞳武器可以把強化等級轉移到100級武器上，其實已經告訴玩家了，武器不能升級，但12強化可轉移，界·武器的100級後續就是淘汰。

所以翻到界·武器的玩家，不要妄想可升級成一把特殊的100級武器，結局和普通的夜語黑瞳武器一樣，全都淘汰，100級回到同一起跑線，想要新武器？去不同的100級副本自己刷！

其實界·武器最特殊的點就一個，那就是普雷權能特效比較華麗，但在100級之後，全都作廢了，就算普雷暫時不換100級武器，但普雷首飾少1件就少一個特效。

至於新武器的屬性，又回到了舊版史詩的屬性，比如100級的巨劍，屬性主要是三次覺醒+2級，屬強和武器特效這些屬性又回來了，而100級的太刀則是「陰劍」和「陽劍」的合體，不再是全職業一樣的屬性，但希洛克武器屬性暫未曝光，不排除100級也有最強的版本武器。

所以在12月之後，小夥伴們如果普雷Raid再次翻到界·武器的話，選擇黑瞳石是最優方式，當然95B已經升級完畢的話，再考慮選界·武器，短期內擁有一把外觀不一樣的武器也是不錯的選擇，但黑瞳石能換自選華麗徽章，或者是卡片冊，都比一把註定要淘汰的界·武器好一些，尤其是女格鬥和暗夜使者這些職業，兩種普雷武器外觀幾乎看不出區別。

總結，界·武器和夜語黑瞳在100級沒有不同的升級路線，玩家們再次回到一個起跑線，想要新武器，重新刷深淵或刷副本得到100級武器來繼承打造，在屬性上沒有任何區別！
