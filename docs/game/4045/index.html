王者榮耀：12月迎來皮膚雨，貂蟬FMVP不值一提，6元虞姬官宣真香

王者榮耀這是一款非常受歡迎的手機遊戲，目前在遊戲中的皮膚數量是非常多的了，在進入到12月以來，天美也是曝光了很多的新皮膚，就連貂蟬的FMvp都是不值一提的，下面就讓我們去看看吧！

貂蟬Fmvp，貂蟬的FMVP是一款傳說級的皮膚，它的上線日期也是越來的越近了，之所以玩家們不喜歡貂蟬，是因為它實在是太秀了，操作起來是非常困難的，你有使用過貂蟬嗎？

東方鏡，東方鏡是下一個即將要上線的英雄，它是東方曜的姐姐，在它背景故事當中我們可以看到東方鏡彷彿化身成為了復仇女神一樣，但是卻有著非常美麗的外貌，真的是讓人非常的喜歡。

廉頗，廉頗在遊戲中是一個非常老的英雄了，天美最近終於記起了它，而且給它製造了一款非常好看的皮膚，可以說是秒殺那些史詩級的皮膚，渾身上下冒著火焰，超級的酷，真的是讓人迫不及待了。

亞瑟，亞瑟在最近的活動中也是要上線一款新的皮膚了，雖然那說目前還沒有曝光原畫，但是只要天美下定了決心，相信很快就會回來了，這位從S1就上架的英雄，一直以來都是遊戲中最厲害的戰士了。

虞姬滑板鞋，最後一款就是虞姬的皮膚了，蔡文姬的繁星游吟已經上架了，作為同款製作的虞姬滑板鞋相信也是離官宣不遠了，作為一款6元的皮膚，僅僅只二技能是讓人很喜歡了，相信上架也是會引起玩家們的喜歡的。
