DNF新史詩之路星河裂縫來了，不需要深淵票和靈石，獎勵異常豐厚

天天要聞
2019-12-04

摘要

最近在體驗服更新了將要在12月10日登陸國服正式服的內容，其中最大頭的就是新版本的史詩之路——星河裂縫了。這次的史詩之路完全拋棄了之前需要深淵票的玩法，甚至95版本新出現的星空深淵靈石也不需要了，而且獎勵異常豐厚哦。新版本的史詩之路分為星河裂縫和超星河裂縫兩種。

最近在體驗服更新了將要在12月10日登陸國服正式服的內容，其中最大頭的就是新版本的史詩之路——星河裂縫了。這次的史詩之路完全拋棄了之前需要深淵票的玩法，甚至95版本新出現的星空深淵靈石也不需要了，而且獎勵異常豐厚哦。

新版本的史詩之路分為星河裂縫和超星河裂縫兩種。

星河裂縫每天每個賬號限進入10次，每天6：00刷新進入次數，入場不需要材料。而超星河裂縫進入次數沒有限制，但是進入需要消耗10點疲勞和一個波塞姆的祝福花瓣。

（分兩種模式和去年的史詩之路類似，大家應該還沒忘記去年的史詩之路吧？）

那怎麼獲得這個波塞姆的祝福花瓣呢？大家可以到活動NPC畫質女王波塞姆那裡去換，不過兌換的方式有兩種（順便給大家看看材料兌換的獎勵）。

右下角那種售價80個金幣的其實就是策劃白送給大家的，每個賬號每天兌換一次，打開可以獲得3個波塞姆的祝福花瓣，可供大家挑戰3次超星河裂縫。不過大家也明白，白送的都是限量的，還想繼續刷的就只能用三個精鍊的時空石來兌換了。

給大家算一下啊，3個精鍊的時空石就是30個時空石，如果是從NPC那裡買的話一個大概要8300金幣左右，30個的話大概是25萬金幣，這樣一算挑戰一次的成本還挺高呢。

所以想要打新版本史詩之路的話，最近就要把普雷每日和搬磚拾起來了，爭取自己多產出一點精鍊的時空石。

在通關新史詩之路的時候還會掉落權能碎片和輝煌的金色氣息這兩種道具（星河裂縫只掉落輝煌的金色氣息，而且掉落的很少），這些材料都可以兌換不同的道具。上面和波塞姆的祝福花瓣一起的那個是輝煌的金色氣息的兌換列表，大家可以看到有異界氣息凈化書、泰波爾斯首飾禮盒、泰波爾斯特殊裝備禮盒、純凈的增幅書、史詩跨界石、永恆的黑瞳石（一周8個）、反物質粒子（一周16個）等獎勵，其中跨界石說所有史詩品級的裝備都能跨，不知道能不能跨界超界和普雷裝備，畢竟普雷和超界都屬於史詩啊。

而權能碎片呢則可以兌換一些適合增幅用的東西（這是史詩之路的配套活動——南瓜球的新商機）。

大家看上圖，這裡兌換的獎勵中前綴全部是「奇怪的」什麼什麼，那它奇怪在哪呢？這些獎勵只能用於95級的史詩裝備，其他裝備是用不了的。

這個奇怪的增幅材料則可以當作矛盾來用，只不過只能在「奇怪的增幅器」上面使用。

這樣呢，新版本的史詩之路星河裂縫及其配套活動就介紹完了，大家可以看到整體的活動力度還是可以的，就是沒法清之前攢下的深淵票和靈石了，反而這個活動可能會消耗掉大量的金幣（用於購買精鍊的時空石），總之對於大號和小號來說都還算不錯吧。
