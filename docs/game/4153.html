滔搏打野Xx怒噴Uzi接班人：心態這麼差，活該打LDL

2019-12-09 0條評論 9次閱讀 0人點贊

摘要

時值英雄聯盟LPL賽區冬季轉會期，各大戰隊還在為新賽季的陣容招兵買馬，想要做好最完全的準備來面對即將在中國國內舉辦的S10全球總決賽，這也讓得這個轉會期充滿了腥風血雨，大家等轉會消息等得是辛苦又振奮。

時值英雄聯盟LPL賽區冬季轉會期，各大戰隊還在為新賽季的陣容招兵買馬，想要做好最完全的準備來面對即將在中國國內舉辦的S10全球總決賽，這也讓得這個轉會期充滿了腥風血雨，大家等轉會消息等得是辛苦又振奮。然而就是在這個時候，TES滔搏戰隊的打野選手熊熊在自己的微博上直接點名噴了RNG的一位隊員，一時間讓不少網友都覺得熊皇好剛，感噴RNG的人。

​事情是這樣的，12月9日凌晨，Xx在微博上說到：「RNG Able，怪不得只能打LDL，心態這麼差，打個毛的職業？」我們也曾見識過許多選手掛人，但像熊熊這樣指名道姓地點草的，還真的不多見，而且掛的這個人還是LPL老牌戰隊RNG的人，這確實讓人感到比較驚訝。

​有網友貼出熊熊和Able的戰績圖，我們可以看到他們這邊有著四名職業選手——熊熊，滔搏二隊1024、SN中單天使以及RNG的Able。劣勢對局對於召喚師來說再正常不過了，但是從大家的解釋中我們能知道，Able看到這場對局稍顯劣勢就開始送人頭了，這真的不能忍。

​早在這之前，Able的風評就一直很差。從最早的大發明家出名，到後來「走A怪」變成了「掛機怪」，RNG也只能將他從Uzi的替補下放到了二隊，韓服掛機被多名職業選手舉報。電競圈的方仲永也不過如此了。之前RNG二隊教練還出面為他開脫過，該罵的罵該噴的噴，但屢教不改是真的讓人覺得噁心了。

​身為職業選手卻沒有職業選手應有的素養，也難怪Xx會指名道姓公開處刑。朋友們，你們怎麼看Able這個曾經的Uzi接班人？
