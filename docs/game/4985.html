《荒野大鏢客2》Steam版本很快將要發售，根據Steam的提示，該作品將於2019年的12月6號正式解鎖，但是目前Steam版本的價格未知。

玩家打開Steam，搜索「Red Dead Redemption 2」就能進入到遊戲介紹界面了。

不過很早的時候，《荒野大鏢客2》PC版本已經在Epic商店上架了，所以很多玩家已經在Epic商店購買了《荒野大鏢客2》。

但是此前的《荒野大鏢客2》PC優化並不出色，出現了BUG、掉幀、卡頓等現象，是新的顯卡殺手。

雖然現在PC版本的《荒野大鏢客2》BUG修復完成，但是此前卻造成了玩家不友好的體驗，現在在Steam上發售，也是讓玩家有所顧慮。

不不過現在Mod管理器正式發布，實現Mod一鍵添加。這對玩家來說簡直就是福利，因為主機遊戲與PC遊戲最大的區別就在於Mod可添加。

到底《荒野大鏢客2》Steam版本會得到怎麼樣的評價暫時未知，但是作為一款大作，可以在PC上面進行遊玩，首先就是一件讓PC玩家感到開心的事情。
