國產整機不行？中國長城硬核支撐2019國際女子電競大賽，燃爆

2019-12-17 774條評論 1,784次閱讀 493人點贊

作者｜市界 沈淼

編輯｜老拿

2019成都國際女子電子競技錦標賽落下帷幕，中國長城高性能國產整機為比賽提供硬核支撐。

據悉，針對此次賽事，中國長城組建了強大的設備調試團隊，十多位工程師歷時十天十多款遊戲，7X24小時不間斷的調試和完善，實現了國產整機軟硬體體系與比賽遊戲的完美兼容和穩定運行。

來自全球的多位頂級電競選手和戰隊，在中國長城高性能國產整機上進行各項遊戲的角逐，戰況異常激烈。

「第一次在國產整機上打遊戲感覺非常不一樣，跟自己以前用的設備比起來，完全感覺不到任何區別，為國產整機點贊。」一位參賽選手對中國長城高性能國產整機評價到。

來自俄羅斯的選手於麗亞是第二次報名參加WEC比賽，她感嘆道「今年的電競設備較之去年有明顯提升，軟體的反應速度更快了，現實感更強。進入遊戲後，會給人身臨其境的感覺。」

賽事現場，中國長城還設立了高性能國產整機展示區和體驗區，供現場觀眾體驗國產整機的實力與魅力。

經歷28場精彩激烈的角逐，2019年12月15日，冠亞軍正式誕生。皇室戰爭的冠軍由黃星倩奪得，周玥玟拿下亞軍，蔣麗獲得季軍；QQ飛車的冠軍由周雯昕奪得，亞軍為章鈳鑫，季軍為Yilia；穿越火線-槍戰王者的冠軍由惟瑞戰隊奪得，亞軍菲悅戰隊，季軍青禾戰隊；爐石傳說的冠軍由吳雨潔奪得，亞軍卿媛，季軍楊瓊。

此次賽事中，中國長城高性能國產整機全程強力支撐，提交了一份完美的答卷。大國重器，風雲再起，中國長城，王者歸來。

公開資料顯示，中國長城創造過多項第一：曾經研發出中國第一台高級中文計算機、第一塊硬碟、第一台顯示器、第一台光筆圖形顯示終端、第一款終端ASIC晶元等；中國長城擁有60餘款系列產品、近100款自主研發的計算機主板、近200項國產整機領域專利。在黨政機關電子公文安全應用示範項目中市場佔有率排名第一，在檔案、金融、互聯網等多個重要行業國產化上，銷量遙遙領先。
