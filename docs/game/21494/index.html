小時候玩的FC打鴨子遊戲竟變成了恐怖遊戲？

2019-12-19 702條評論 2,476次閱讀 94人點贊

歡迎收看本期的遊戲不止，我是喬伊。小時候不少觀眾肯定見過這款遊戲吧。FC上的打鴨子（Duck Hunt）

任天堂的打鴨子光槍遊戲，在1984年4月首次的日本發售。隨後又憑藉NES在北美髮售。

遊戲的玩法很簡單，便是有鴨子從草叢中飛出來，利用光槍對準電視機屏幕瞄準開槍，擊落鴨子便可。要是飛走了，便能見到一隻賤賤的狗在嘲笑你。

這裡我簡單給大家講一下，這個光槍遊戲的原理。光槍槍口裡有個感光晶元，然後槍和手柄一樣是連線接遊戲機上的。

在玩像打鴨子這種光槍遊戲的時候，扣動光槍的扳機瞬間，光槍發出信號讓遊戲畫面瞬間變黑。但只有鴨子所在位置的區域是白色的。由於這一瞬間只有幾十毫秒，玩家並不容易察覺畫面的異樣。畫面一白一黑，明暗對比極大，光槍槍口的感光晶元只需要識別，是否瞄到了那個白色發亮的區域。就能判定光槍有沒有擊中鴨子。

原理就是這麼簡單，所以這也解答了很多老玩家的疑惑。當年我的光槍對著發亮的電燈開槍，為什麼也能打中鴨子。

然而相信有不少觀眾和我一樣，小時候沒有紅白機，只有大家更為熟悉的小霸王學習機。我的小霸王學習機的光槍被搞丟了，導致我偶爾選中這個打鴨子的遊戲，拿著手柄不知道怎麼操作。也只能無奈的看鴨子飛走，還要被那傻狗嘲笑。

那這次我要給大家講的遊戲便是《獵鴨季節》（Duck Season）。這款遊戲便是致敬當年任天堂FC的打鴨子。遊戲作為一款VR遊戲，在兩年前就已經上市了。但VR設備嘛，絕大部分玩家也只是體驗過，自己有的很少。所以《獵鴨季節》作為小眾VR遊戲，並沒有被多少玩家熟知。

前段時間這遊戲開發商，把他從VR上弄到了PC上。也就是說，現在只需要鍵鼠就可以玩了。這樣便能讓很多玩家可以接觸到這款遊戲。於是為了報我當年沒有光槍，一直嘲笑我的那隻狗的仇。我便搞了這款遊戲。遊戲畢竟原作是基於VR設備開發的，現在用鍵鼠操作沒多大影響。但還是能感受到強烈的VR遊戲的架構元素。

劇情很簡單，在1988年美國的一個夏天。一個男孩在家百無聊賴，他的母親便給他帶回來了打鴨子的遊戲。於是他非常開心。我們扮演的便是這名小男孩坐在電視機前玩遊戲。

不得不說遊戲很多細節做的非常好，遊戲主要是打鴨子的遊戲，其實電視機前的其他遊戲卡帶，錄像帶都是可以遊玩和觀看的。像這個卡帶便是一個忍者對打的遊戲（《Black Ninja》），玩法和猜拳一樣。或者像這些錄像帶也是一個小短片。細節上做的非常到位。

不過既然打鴨子，主要玩的便是這款遊戲，插入卡帶進入遊戲後，彷彿真正的來到了野外，作為一名獵人來打鴨子。而那隻傻狗依然在那。這次我終於能抬手給它一發了。雖然打不死它，但還能混個成就。

玩法和當年的遊戲一樣簡單，瞄著鴨子打就是了，沒什麼特別的。

然而當我玩了一會，我轉過頭的時候，便發現有些東西好像不對勁了。為什麼我能見到坐在電視機前的小男孩，看著它隔著屏幕玩遊戲。遊戲每玩一關，便會從電視機中退出來回到現實世界。看牆上的鐘，一局遊戲便會過了一個小時。隨著遊戲一關一關的發展，你能開始感受到一種說不出來的異樣。總覺得哪裡不對勁，尤其是那隻傻狗。而天色也逐漸變暗。

直到一次關卡結束後…

我才驚悚的察覺，這遊戲貌似披著打鴨子那休閑娛樂的名號，實則是一款驚悚恐怖遊戲。而這張詭異的遊戲卡帶，無論怎麼扔都會回到我的面前。

母親依然在忙著她的家務，我又硬著頭皮進去玩第八關，這一關和其他關卡沒什麼區別，依然歡快的打著鴨子。但慢慢我發現了一個問題，前面幾關打到一定分數或者達成目標。就能結束關卡退出遊戲，但這關怎麼都結束不了，音樂也消失了，狗也不在了。困惑許久後，直到我回了個頭。

母親倒在了血泊之中，而那打鴨子的遊戲，也變成打狗子的遊戲。在遊戲外，這隻傻狗變成了一個恐怖的殺人犯。在遊戲中，它變成了一個披著狗皮的怪物。當我在遊戲中戰勝了這隻巨狗，而遊戲也徹徹底底變成了恐怖遊戲。

遊戲的最後，我也被這狗給殺死了。有意思的是這是遊戲中七個結局之一。不過我估摸著這肯定是最壞的結局了。而讓這一款輕鬆休閑的遊戲，變成一個恐怖遊戲的原因。竟然在於，我對著狗開的那一槍，它便來報復我了。

所以對於喜歡懷舊，休閑，或者驚悚，恐怖遊戲的玩家，可以去嘗試這款《獵鴨季節》，這個遊戲可並不算單純的一個打鴨子的遊戲，實則是一個解謎求生的恐怖遊戲。

一共七個結局可以挑戰。希望你們別像我一樣搞個母子雙亡。

如果你不想玩這款遊戲，或者確實挺害怕恐怖遊戲的，可以聽我把剩下幾個結局和劇情講完。

遊戲從休閒遊戲，變成恐怖遊戲，就取決於你對不對著那隻狗開槍。如果開槍，那他將會來報復殺害你。因此：

結局一，便是全程不打狗，狗也不會來報復你，最後玩一天遊戲，乖乖被你媽叫去睡覺。得到完美結局。

結局二，因為我打了狗，狗來報復我，便是遇到母子雙亡的結局.

結局三，在最後它出現的時候，用光槍戰勝它。保住自己的性命。而可惜的是母親已經遇害，狗殺了我媽，而我殺了狗。

結局四，在遊戲的最後的一關，它變成大狗用激光攻擊電視屏幕，如果你沒有及時打敗它保住屏幕。你將會永遠困在電視機里。

結局五，還是最後打大狗的時候，由於槍變成了黃金武器，這時候如果對著太陽開槍就能出發核爆結局。但結局依然是一個壞結局。

結局六,就比較有意思了，在當時發現那狗用攝像機拍我的時候，就應該察覺危險選擇報警。等你再玩一局遊戲出來，它本該來殺害你的時候，結果剛好被趕來的警察擊斃。而且還被警察暴打，也算是一個比較好的結局。

每當你完成一個結局，你左邊的柜子上會有一個相應的結局獎盃。

最後一個結局，你完成以上所有結局後，找到錄像帶觸發該結局。這個結局比較複雜，並不單純那麼簡單。以及你找到那本白皮書，還能開啟一些隱藏的劇情。甚至通過一些觸發條件可以達到一些特殊場景，有遊戲製作者給你闡述遊戲的設計理念。

總的來說，遊戲表面是一個簡單致敬FC打鴨子的遊戲，實則遊戲細節和彩蛋特別多，這個就需要玩家們自己去探尋和摸索。

感謝收看本期的遊戲不止，喜歡的觀眾朋友們可以點擊關注，我是喬伊，我們下期再見。
