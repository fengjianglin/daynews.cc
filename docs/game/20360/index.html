雙C離隊沒關係，WE下賽季陣容爆炸，這才是電競該有的樣子

2019-12-16 448條評論 2,668次閱讀 277人點贊

英雄聯盟轉會期馬上就要結束了，WE戰隊在官宣了經理，雙C以及教練離隊的消息之後完全沒有任何補強的消息流出。眼看著一個個強力選手都紛紛選好了戰隊，作為WE粉絲的我們卻是憂心忡忡不知道WE以後的路在何方。

下個賽季，WE基本上會是全新人的姿態出征。大家也都不太看好WE的表現。但是誰說新人就一定不行？下賽季WE的陣容或將爆炸，而且這才是電競該有的樣子。

自從15年大量韓援來華之後，每年LPL的轉會期就掀起了各大資本家砸錢買賣強力選手的熱潮。而WE作為國內最老的一批英雄聯盟俱樂部之一，或許沒有這些資本家們那麼有錢，或許無法從他們手上搶下強力選手。但是WE的青訓體系一直是所有戰隊裡面最為成熟的，也是WE一直致力去做的一件事。

作為國內最老的一批俱樂部，WE一直都在尋找著電競發展的方向。從最初的幾個路人心血來潮便組建戰隊開始，隨後漸漸完善自己戰隊的體系。WE一直都在按部就班地努力著。而LPL創立之後，資本的大量注入導致WE戰隊這樣沒有雄厚財力的俱樂部舉步維艱。

2015年開始，LPL各大戰隊的老闆們都在砸錢購買強力選手。而那時，WE也曾這樣做過。曾經三星戰隊的Looper，Dade等都是當時世界上數一數二的選手。不過來到WE之後卻是由於自身的發揮以及隊友經驗不足的原因未能取得好的成績。春之大帝夏之阿珍的梗也從那時候開始傳了出來。

自那之後WE便一直沒有高價簽約過什麼強力選手。這也是資本不足帶來的無奈。而強力引援無果之後，WE便放棄了一技之長克敵之短的方法。更加著重於培養新人選手，18年入駐西安之後除了買下IMP更是簽下了很多新人。網友們也戲稱WE是足球隊。而這便是WE在這資本林立的LPL立足的最好方式。廣撒網，多嘗試，培養新人。

如今賽場上很多的強力選手都是從WE走出來了，如Theshy，Khan，SLMZ，Xiye等等。但這也暴露了WE的一個很大的問題，就是留不住人。可以說如今LPL的戰隊，每一支都比WE有錢。每個人有成績的人都能拿到比在WE好的薪資待遇。而這也成了WE留不住人的根本。

資金問題或許比較難，但是WE還有一個問題如果能解決的話資金便不再是問題。那就是戰隊的管理方面。誠然WE戰隊致力於培養新人但是戰隊管理跟不上的話，就很可能會錯失很多有天賦有夢想的年輕人。PDD的YM戰隊一直被人調侃「聚是一坨屎散是滿天星」，這多少也是因為戰隊管理經驗的不足。無法將隊員的天賦發揮到極致。

比拼財力，WE或許會稍遜LPL其他戰隊一籌，但也不是真有那麼窮。不過如今LPL資本入駐越來越嚴重，WE的財力才會顯得越來越捉襟見肘。沒有財力很可能會損失很多有能力的選手，但是只要WE能把戰隊管理體系做得足夠成熟這個問題便能迎刃而解。

資本入駐的戰隊或許能夠通過買人瞬間增強自身戰隊的實力。但是如今選手們的身價越來越高，資本終究會有個極限。而新人的培養卻沒有，無數個有夢想有天賦的新人等待著大家去發掘。而那些靠著高價購買選手的俱樂部明顯不會將重心放在這上面。而這便是WE的機會。

17年WE戰隊奪得了春季賽冠軍，隊內的幾人幾乎都是在WE成長起來了。Xiye、Condi是WE青訓時期培養的，957是WE二隊提上來的，Mystic也是在WE迅速成長，Ben在IG的表現也遠沒在WE的表現好。而這些人的身價在當時都不能算高。

下個賽季，WE基本上確認都是新人出戰，但新人的潛力才是最大的。957昨天也表示WE明年中游的實力肯定有。那很可能便是如今WE的這個新人陣容是有一些化學反應的，平時訓練的表現應該也不錯。距離明年開賽還有時間，而且世界賽要等到下半年。快有一年的時間，誰能說WE的這群新人就不能一鳴驚人呢？

可能在即將開始的德瑪西亞杯和2020年春季賽剛開始的階段，WE戰隊的表現不一定會很好。但是他們有成長的空間，而且雖然主力和教練都走了。但是別忘了，957也是很有比賽經驗的選手。微笑更是對WE不離不棄。兩人都曾登上過那個最高的舞台，只要新人們能夠聽從教導，這兩人的經驗完全能夠讓他們沖向更高的地方。

下賽季WE的成績只能說是一個未知，但是完全不用像這樣一直唱衰。或許很多人會覺得希望越大失望越大，但是作為WE的粉絲。誰對戰隊是不抱有希望的呢？誰都是從新人開始成長的，IG奪冠，FPX奪冠不就證明了這一點嗎？他們所有人奪冠時都是第一次參加世界賽。在世界賽上，他們是最新的新人。

以後WE如果能夠優化自己戰隊本身的管理，能夠同時發掘更多有天賦的選手。那麼他們肯定就能迅速打出成績，而有了成績留人也就不再變得這麼困難。職業選手都是熱血的，他們更多的還是希望有成績。金錢到了一定程度便就成了一堆數字，唯有成績才是可以不斷追求不斷突破的目標。

我想，只要是WE戰隊的粉絲，肯定都會對戰隊一直抱有期待。即使他們會因為探尋道路而遇到些許挫折，但我們依然會與他堅定地站在一起。也希望WE能夠真正找到自己的道路，不再成為所謂的「蔬菜大棚」。2020，WE加油！

小夥伴們對於WE下賽季有什麼看法，也可以在評論區討論。
