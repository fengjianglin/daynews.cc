路人意外選出職業陣容，夢淚連著說出八個優點，直言在我眼中無敵

天天要聞
6 天 ago

夢淚的直播選擇大多都是巔峰賽，打得緊張刺激，網友們看得也比夢淚本人還緊張。所以夢淚時不時也會來幾次五黑調劑一下直播氛圍，相對於事關積分的巔峰賽，五黑就來得隨意許多，也更加歡樂搞笑，深受觀眾們的喜歡。

遊戲的選人階段隊友們就玩了起來，在一樓選下鍾馗之後，便直接要求和四樓的魯班大師一路，揚言對面只要被勾到就是個死，非常自信。而夢淚本打算打中單，飢餓在隊友提醒有中路之後選下了李白去打野。不過看到對面陣容後夢淚嚴肅地說：「對面前期要反我野啊！」此時隊友們紛紛表示保證會幫助夢淚，場面非常歡樂。

進入遊戲讀取界面後，看到了對面的完整陣容，夢淚和隊友紛紛驚嘆對面的陣容太強了，可以說是要什麼有什麼。夢老師直接「導師模式上線」，在線點評對方陣容。接連指出了對面陣容的八大優點：有前排牛魔和孫策；整體輸出很高；POKE有幹將莫邪；清線刷野有曜；帶動全場節奏有裴擒虎；個別英雄單挑能力強；而且跑得快，機動性好；前期打架能力也好。直言這套陣容在他眼裡幾乎是無敵的。

不過羨慕歸羨慕，在絕對實力面前是不存在任何技巧的。果不其然，對方前期就開始搞事情。雖然對方打野裴擒虎沒有反紅，但是卻選擇了抓下路，而且中路幹將莫邪拿到了鍾馗的一血。不過之後鍾馗加魯班大師的黑科技也終於成了，移動的鉤子厲害的很，把幹將莫邪勾到中路一塔下將其擊殺。

雖然夢老師李白刷得還算順利，但卻一直沒拿到人頭，眼看就要將裴擒虎收掉卻被關羽一刀推開並搶了人頭。被搶了人頭的夢老師大呼：「兄弟你這個推也太無情了吧!我去，人頭都拿不到。」惹得隊友一陣大笑。

除此之外，公孫離也非常秀。在中路河道，夢淚開玩笑讓公孫離將對面推回來，結果公孫離信以為真真將兩人推了回來，結果對方的孫策及時趕到給出大船，夢老師一方慘遭團滅。夢老師非常無語，稱沒想到公孫離真敢推。

雖然大家都怎麼歡樂怎麼玩，但明顯實力是要比對面高出不少的，尤其是公孫離，打出了超級高的輸出，夢淚一方不顧對面「給個機會！」的苦苦哀求，團滅對手後一波結束了比賽。這麼完美的陣容還是打不過夢淚他們，可見王者榮耀的陣容固然重要，但也要有與匹配的實力才能取勝呀！
