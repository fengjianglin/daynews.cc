雲頂來襲，法師水晶蠍：超厚“白盾”，輕鬆過千，能抗能輸出

天天要聞
6 天 ago

各位召喚師們大家好，雲頂來襲快樂來襲，今天將為大家帶來這套陣容目前已經被廣大召喚師們使用，前段時間有為大家推薦過法師亞索雙重飛天雙重快樂，而今天的主角依舊是一位「不屬於法師的法師」——水晶先鋒蠍子，蠍子是水晶掠食者雙羈絆的二費英雄，技能水晶蠍甲是為自己增加護盾的同時提升自身的攻擊速度，而裝備法師羈絆後的蠍子就能同時釋放兩層護盾而且獲得雙倍的攻擊加成，下面就讓我們一起來看看法師水晶蠍的獨特玩法吧。

陣容配置：

蠍子，寒冰，遁地獸，大嘴，吸血鬼，火男，辛德拉，妖姬，小法

整套陣容九人口完全成型，人口要求是很高的，因此對於玩家的運營能力要求很高，否則很難活到九人口的大後期，因此這套陣容如果想用來上分是不太現實的，但是只要成型，蠍子一人就是爸爸。所有的其他英雄都是給蠍子的掛件，因此可以選用其他的替補英雄，例如法師可以用岩雀代替，水晶也能用寶石代替，掠食者也能換成狼人，原則便是誰先來就用誰。

羈絆特效：

裝備配置：

蠍子：反甲+法師帽+大帽子

沒錯，我們核心的主C選手就是蠍子，擁有水晶羈絆的蠍子能夠完美的抵禦法師的技能傷害，因為我們一次最多只會受到100的傷害，不過我們依舊會害怕地獄火的持續灼燒傷害，這一點需要注意，因為我們沒有出任何血量裝，兩星的蠍子血量還是不足以完全頂住傷害的，因此蠍子如果能追三，那麼這套陣容成型後吃雞就是妥妥的了。大帽子的技能強度加成能夠讓我們的護盾值直接上千，再搭配六法師的雙重施法，不好意思我的白盾你沒法想像，同時反甲能夠觸發掠食者的效果，讓我們哪怕只是被動挨打也能直接彈死敵方英雄。

除了以上蠍子需要的裝備以外，其他裝備直接就給我們前期的英雄裝備上，肉裝給吸血鬼，法強給大嘴，保證前期的血量，因為我們需要九人口才能完全成型，因此良好的血量對於我們來說很關鍵。

陣容站位：

站位上蠍子和遁地獸放前排，大嘴和火男放後面的中間，寒冰和小法師放一個角落，妖姬和辛德拉可以放一個角落，盡量分散站位，避免被狂戰士以及夢魘這樣的突臉英雄直接切到一堆人。

陣容分析：

這套陣容是一套對玩家運氣運營都很看重的陣容，因此如果前期裝備太差，英雄完全沒有自己需要的就不要嘗試了。前期採用三掠食者打工，掠食者前期作戰能力很強，且我們有裝備直接就上了，因此前期能夠很好的保證血量，大選前適度賣血輸掉比賽，讓我們能夠優先選擇自己需要的裝備，最主要的就是一個金鏟子，等蠍子的法師帽到了，就用三掠食三法過度，等到其他的水晶單位來了，就補齊雙水晶，最後補上最後的三位法師，達到我們陣容的最終形態六法掠食水晶。

以上便是本期法師蠍子的全部內容了，最後再提醒下各位召喚師喔，陣容雖強，但是要求也很高，不過想看下法師蠍子的召喚師還是可以試試喔，相信不會讓你失望的，雲頂來襲精彩不停，咱們下期再見。
