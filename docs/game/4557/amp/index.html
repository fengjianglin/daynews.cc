今天的SOLO賽中，Uzi和小天都擊敗對手晉級下一輪。賽後他們兩人接受了採訪。

問：1V1的時候為什麼要選擇小炮出戰呢，有沒有想到對方也copy了一手小炮出來？

Uzi：沒有想到他會選吧，在AD裡面小炮是一個不錯的選擇。

問：可以看到今天一級的時候你是想上去點他，但是他猶豫了一下回來點你的時候有沒有感覺有點意外呢？

Uzi：我是有點沒有想到吧，因為我是在草里先A了一下然後他沒有A到我嘛，然後他還敢跳過來還是在我多他一個屏障的情況下，有點把我嚇到了。

問：可以看到你們兩個人出門的出裝就完全不一樣，你是全進攻，但他還出了一個布甲，你覺得這是什麼樣的區別呢，思路有什麼不一樣？

Uzi：因為他練習得比我多那麼幾盤吧，因為小炮打AD英雄回家買一個布甲鞋就可以直接跳到對面臉上，會比我買一把長劍更好一點。

問：我發現你帶的召喚師技能是點燃加屏障，這在1V1當中是不是要有利一些呢？

Uzi：還是要看雙方英雄吧，我覺得兩個相同英雄在技能沒法躲的情況下帶屏障會更好一點。

問：最後一波為什麼會選擇鋌而走險上去把他給秒掉呢？

Uzi：那一波其實已經有一點失誤了，因為我的屏障基本等於空掉了沒有擋到技能，但是我機會抓得比較好吧，

我多平A到他兩下然後他還要跟我對拼，我就直接跳上去跟他打了，因為我裝備也比他好嘛。

問：這一次SOLO賽中你最想遇到哪一位選手或者說最不想遇到哪一位選手呢？

Uzi：最想遇到Theshy選手吧，因為大家都說他SOLO很強所以想見識一下。

問：這次全明星中你最期待哪個模式想去體驗呢？

Uzi：本來比較期待雲頂之弈，但是因為自己太菜了所以沒敢去報名，還是有點可惜。

問：這場比賽之前是和Peanut商量好了用一樣的英雄一樣的符文進行男子漢的對決嗎？

Tian：這場比賽之前我們商量好了都玩盲僧都點不滅之握。

問：可以看到你們的出裝思路是完全不一樣的，為什麼你會選擇出一些防禦向的裝備比如反甲呢？

Tian：開始之前我就知道這把會打很久，因為這英雄SOLO都是主點W互相吸血，反甲又是很克制吸血的，所以我就出了反甲。

問：為什麼盲僧SOLO的時候不帶閃現而是選擇其他召喚師技能呢？

Tian：閃現躲一個Q沒什麼區別，因為大家都是主W。

問：當時點燃過後為什麼先瀟洒回頭，後續又用Q技能把人頭收掉呢？

Tian：當時我給他點燃是想讓他回家，不敢追是因為我怕他R閃把我踢進塔里，我沒帶閃現出不來。當時沒想殺他，但是他好像沒太注意，就把他收掉了。

問：可以看到後面他也用R閃把你踢進塔里，但是你好像沒太在意很自然就走出來了，當時是什麼情況？

Tian：當時我已經很肉了，而且身上還有個盾，就不太怕那個防禦塔。

問：這是你第一次來全明星，這局過後會比較期待什麼模式呢？

Tian：一開始比世界賽還緊張一點，因為是同英雄SOLO。無限火力和雲頂之弈都挺有意思的，但是這兩個項目都沒有我。

問：之前Doinb在採訪的時候也說了一些……對吧，你有什麼想回應的，你會想在SOLO中遇到他嗎？

Tian：我覺得他很難遇到我，因為他那組有很多厲害的人，我這組我也很難出線，其實大家都在做夢。
