國產遊戲推薦：《艾希（Icey）》

2019-12-19 1,728條評論 3,133次閱讀 1387人點贊

《ICEY》是一款2D橫版ACT+META遊戲，跟隨旁白的指示，通過ICEY的眼睛去看，去戰鬥並發掘在遊戲中世界的真相。你可以通過對抗旁白的指示，思考何為遊戲、何為真相等等問題。

在國產手游群魔亂舞的時代，換皮遊戲和打著大IP坑錢的遊戲層出不窮，《艾希》更像是其中的一股清流，18元的買斷顯得誠意滿滿遊戲中也無任何內購。

和著名的《史丹利寓言》一樣，整個遊戲充斥著旁白的解說聲，幽默風趣的解說讓旁白甚至搶了機器人妹子的風頭，讓人有一種旁白才是主角的錯覺。此外如果玩家不按照遊戲設定的箭頭進行遊玩的話，會看到旁白君的怒吼和無奈，這種說話的方式讓整個遊戲不再單調和無聊。

遊戲的高速移動，將敵人玩弄於股掌之中。華麗的連招系統，操作簡單，稍微練習即可打出華麗連擊。 不同技能各有特性，靈活組合才能順利過關。

我對這個遊戲的評價還是很高的，認為它國產最好的動作獨立遊戲之一。畫面打擊感不錯，操作也很有意思，音效也很好。很多技能連招，沒有內置的充值系統，完全是一次付費買斷。3個小時基本就能打通關（有些boss要打幾次才能打過）。旁白的話語很親和，詼諧。

Meta元素配合出色的劇情掩蓋了作為ACT遊戲不足，以及豐富的內涵加上需要深度挖掘的遊戲內容使其增色不少。

一款好的動作遊戲需要很多要素來支撐。艾希的建模，動效，立繪，操作感完全稱得上是一流水準，它的特色也很符合遊戲的主題，高速移動可以通過距離控制豐富你的攻擊方式和回合流暢感，連招的設置很人性化，只需要練習幾遍就可以打出華麗的連招，不同的連招技能給出的反饋也很多樣。此外，打巨型Boss也給玩家帶來了視覺和手感上更一步的刺激。

在國產手游與單機不景氣的今天，我們都希望有更多像《艾希》這種優秀的獨立遊戲可以頻繁出現。總之是一款不錯的國產單機，所有的方面都不弱，但是也不能說是最佳，都有可圈可點的地方，最出色的設計還是屬於旁白君，是他給了這個遊戲以好幾倍的活力。最後祝國產遊戲越做越好！
