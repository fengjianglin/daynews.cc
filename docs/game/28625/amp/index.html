王者模擬戰：7法難吃雞該怎麼辦？試試這套長安打法以錢壓敵

天天要聞
2019-12-22

隨著版本的更新，王者模擬戰中的法師，地位也是一落千丈，雖然不至於墊底，但之前那種7法輕鬆吃雞的壓迫力也是沒有了。比如小編最近的幾局對戰中，就算是有了3星的法師，也是很難吃雞的，所以也是開始摸索別的陣容了。

在小編的摸索中，發現長安陣容還是很有潛力的，這個陣容以攢錢為主，最大的特點就是攢錢快。初期如果就走長安陣容的話，那麼基本全局都可以穩壓對手1級到2級，達到以錢壓敵的效果。

那麼這套陣容該怎麼運營呢？首先自然是3長安起步，然後轉5長安，這個過程中主意存點錢，英雄也只選一個射手，其餘的就拿長安和盾英雄即可。在轉5長安之後，攢錢速度進一步加快，然後就是湊出3射手，之後疊加盾英雄即可。

盾英雄，首選自然是稷下雙盾了，這樣還能組成個稷下的buff，至於7長安，能出也可以出，出不了也無所謂。這樣的盾加射手的陣容，戰鬥力還是挺強的，可以讓我們保持賺錢速度的同時，還能打贏對手，真是賺錢戰鬥兩不誤。

在快速升9級之後，剩下的就是給英雄升3星了，這個過程中注意選擇需要的英雄即可。如果對手不是很強，那就保持在50塊錢吃點利息，反正有了利息和長安的加成，每回合也能刷好幾次池子，儘快給英雄升星。

這樣的打法，還是簡單粗暴的，只要前期有長安英雄就可以走起，後期能不能吃雞，就要看玩家升3星的情況了。如果給對手時間成長起來，那就比較難打了。

那麼，對於這樣的長安打法，小夥伴們怎麼看呢？
