《這是我的家園》—少見的印第安題材開放世界遊戲

2019-12-25 2,495條評論 4,225次閱讀 2469人點贊

《這是我的家園（This land is my land）》是一款以印第安土著為題材的角色扮演遊戲。玩家將在這個故事中扮演美國原住民部落的一位首領，帶領部族來抵抗侵略者的殖民統治。遊戲畫面十分精緻，在世界塑造上絲毫不輸那些本世代大作，而且遊戲有著超過一百平方英里的廣闊世界，玩家可以在其中肆意探索發現，自由決定你的道路。雖然遊戲目前並不是完全版本，但是已經放出的內容也十分的充實，唯一可惜的是還沒有中文本地化。不過對於喜歡自由探索和對畫面要求較高的玩家來說非常推薦。

少見的題材

很少有遊戲選到印第安與殖民統治的題材，所以對於玩家來說新鮮感十足。本作的故事就聚焦與土著印第安人與外來的白人殖民者的對抗中。在這個故事中我們要扮演一位印第安人部落的酋長，玩家要統御自己的部族奪回原來屬於自己的土地。與此同時玩家還要不斷地對抗迅速擴張的殖民者，報復並趕走他們。遊戲採用了開放探索的模式，讓玩家的體驗更加的舒適和有趣，在廣袤的美洲大陸上來去縱橫。

豐富的玩法

由於遊戲的自由度十分之高，因此衍生出來的遊戲內容也很充實。在遊戲里，玩家扮演的印第安人可以使用小刀近戰武器來小心謹慎的暗殺敵人，也可以使用弓箭和步槍來用火力吞噬敵人。說實話，在潛行的時候還真有一種印第安刺客信條的感覺~除了需要對抗殖民者的動作內容外，還有壯大發展部落的經營要素。玩家可以控制部落的子民來收集製作物資，還能去尋找更多的土著來加入自己的部落。而且遊戲里還可以狩獵野獸，馴服馬匹，並於商人來交易你的戰利品。

遊戲畫面

怎麼說呢，即使在大作頻出的當下，本作的畫面也可以稱的上是優秀，不過在體驗過程中較為令人滿意了。遊戲完美的搭建了美洲大陸，從主題到場景細節都很用心，彷彿真的置身於那片土地。而且遊戲整體演出十分流暢，並不會卡頓之類的問題，除了載入過場的時候顯得久了一點。

總結

一款優秀的開放世界遊戲，豐富的內容，少見的題材都讓我對遊戲多了幾分期待，而良好的畫面和流暢的動作也讓玩家體驗良好。遊戲整體玩下來問題也並不多，只有一點小bug並沒什麼影響，希望遊戲能早日推出中文本地化，讓更多玩家了解這款遊戲。
