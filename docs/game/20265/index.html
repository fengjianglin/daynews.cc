《碧藍幻想Re:Link》新預告 四騎士聯手戰巨龍

2019-12-17 40條評論 1,240次閱讀 209人點贊

在《碧藍幻想》FES 2019活動中，Cygames在現場帶來了一段14分鐘的《碧藍幻想：Relink》實機演示和最新預告，官方頻道隨後也公布了10分鐘的官方實機演示，一起來欣賞一下。

本次公開的遊戲畫面中，蘭斯洛特、范恩、帕西瓦爾、齊格菲，4名玩家可使用的角色將在此登場。而視頻中各角色將由4名不同的玩家進行操作，並介紹遊戲流程。四騎士將施展獨自的技能和奧義，並在充分的團隊合作下與充滿魄力的Boss「遠古巨龍」展開激烈的戰鬥，過程中亦充分展現多人遊戲特有的魅力。

Cygames並未在本次活動中公開發售時期，官方表Cygames大阪團隊在過去的一年裡都在努力提升引擎和遊戲品質，但他們並不想催促團隊推出遊戲。

在直播活動中Cygames還公開了一段短片向玩家們預告在遊戲中出現的敵人，具體的情報將在未來公布，敬請期待。

據介紹，這款遊戲是Cygames與白金工作室合作開發的PS4平台遊戲，不過在今年二月份時，白金工作室宣布推出了這個項目，之後的開發工作完全交由Cygames承擔。與此同時，Cygames之前公布的格鬥遊戲《碧藍幻想Versus》也在製作當中。
