fgo有關五星弓階超人俄里翁的相關資料和設定，估計不少玩家都非常感興趣，來看看這次的相關技能組和基礎信息吧，簡單來說這次的新五星是三紅配卡的近戰超級肌肉弓兵。

先吐槽一句，沒想到這頭熊還真就實裝了，還真的實裝了！

基礎資料

聲優：神奈延年、NP獲取率：0.96、特性：神性、天地從者、所愛之人、希臘神話男

面板：筋力A、耐久A、敏捷B+、魔力C、幸運A、寶具EX

數值：攻擊12557、生命13494

配卡：3紅卡（1HIT）+1藍卡（3HITS）+1綠卡（3HITS），EX5HITS

技能組（滿級）

被動：單獨行動（12%爆傷）+海神祝福（175附傷+175減傷）+天蠍詛咒（20%自身蝕毒）

一技能：1回合50%紅魔放+50%魔性猛獸特攻，5CD

二技能：1次3回合的3000毅力+3回合20%攻擊BUFF+20暴擊星，6CD

三技能：3回合集星+3次3回合的紅卡攻擊前爆傷威力提升100%1次，5CD

寶具效果

純BUFF類的寶具效果，3回合無敵貫通、3回合攻擊BUFF（30%-50%，寶具等級決定）OC效果是3回合爆傷100%-200%、3回合弱化無效、3回合持續出10個暴擊星。

這弓兵真的有意思，可以說純粹的平砍類型的從者，有大量的爆傷BUFF不說，而且還有產星和集星的能力，這真的是一套下去毀天滅地，單體弓兵的設計基本上已經進入死胡同了，寶具傷害不敢太離譜，不然太打教授的臉，居然整出這麼一個單體近戰弓兵出來。
