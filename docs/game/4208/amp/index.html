王者故事，你我心事，我是鴛北，與美女做鴛鴦的鴛，與兄弟闖南北的北。作為一個堅持奮鬥在峽谷排位第一線的上分機器，我也收到了很多小夥伴的私信。問我有沒有什麼適合上分的技巧，其實上分真是三分天註定，七分靠打拚，接下來九十分就要看這次我給大家分享的上分攻略了，無論你玩什麼英雄，無論你目前的段位是什麼，只要你想上分，我們首先就要做一個合格的上分機器，來看看你達成了幾個標準呢。

一、戰前準備

現在局內的快捷消息添加了很多有意思的語音，相信很多小夥伴喜歡發「幹得漂亮」和「I』LL CARRY YOU」，但是這兩句話除了暫時的鼓舞隊友並沒有實質作用，想當合格的上分機器，你的快捷消息里應該有著實用的信號，例如「敵人消失！」「小心草叢！」「大招已經好了！」「等等我馬上到！」和隊友打出配合永遠大於一個人的優勢，無論你選擇了什麼位置，只要有配合意識，打出引導隊友的信號，你的上分之路一定比每場舉報隊友來得更順利。

二、支援意識

支援有三大原則，優先跟著打野或者輔助走，優先支援優勢路，優先清理兵線後支援，這是一個推塔遊戲，也是是團戰遊戲，優先跟著打野和輔助，才能讓打野先行取得優勢，優先幫助優勢路，是因為收益遠大於幫助劣勢路，並且前期可以很好避免經濟差距被拉開，優先清理兵線，這個非常好理解，防禦塔現在是非常重要的，遊戲中一人換一塔都不算虧 。

三、學會針對出裝

出裝從來不是固定的，也不是國服最強的出裝攻略就是最好的，有的時候一件裝備就可以影響一場團戰的局勢，輔助對輔助裝的選擇就很重要，如果隊友的陣容偏前期且需要攻速，可以選擇極影，但不要忽略一種情況，當對面都是高爆發輸出，或者經濟普遍高於我方傷害爆炸的情況下，可以選擇近衛榮耀加救贖之翼的雙重保護，輸出們需要看情況出重傷裝備，以及最後一件裝備應該選擇容錯率較高的，可以選擇輝月、名刀司命這樣的保命裝備，或者視情況補肉裝。

其實上分沒有什麼可以保障連勝的技巧，大家匹配到的隊友很多時候水平都是相近的，今天給大家分享的上分技巧，其實對很多高分段的人來說都是常識，當你發現排位的時候你總是把把評分不錯，卻就是打不贏遊戲的時候，不要只怪隊友，也應該反省自己有沒有未注意的細節，只要把以上這些做好，你差不多就是一個合格的上分機器了，哪怕這屆隊友再難帶，你也會是一步一個腳印走上去的真正王者，在這裡祝大家排位順利，上分大吉。
