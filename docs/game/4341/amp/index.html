首先我們來看看這位女玩家曬出的裝備，雖然這位女玩家僅曬出兩件裝備，但個頂個的強力。第一件是帶有破血狂攻、物理暴擊的鞋子。破血狂攻作為物理職業必備的特效，已經足夠令人羨慕，更不用說還帶有物理暴擊特效。另外在這件裝備上，也打上了十三級的黑寶石，這應當是入駐武神壇的標配了。

第二件裝備是帶有憤怒及破碎無雙特效的腰帶，依舊是打上了十三級的黑寶石。僅從這件裝備就能夠一窺該女玩家裝備的豪華了。憤怒、破碎無雙，雙藍腰帶，這是一般人能擁有的裝備么，果然在夢幻西遊手游中，女性玩家才是隱藏的大佬么！

然後我們再來看看該女玩家曬出的第一隻寵物，這是一隻全紅十技能的耐攻羽雲仙子，攻擊資質1519，體力資質5214，速度資質1305，1.258的寵物成長，光看面板就知道這絕對是大佬才能擁有的寵物。高級必殺、高級神佑復生、高級偷襲、高級強壯，攻寵該有的全都有，這樣的寵物著實令人羨慕。

最後就是該女玩家曬出的另一隻寵物，全紅十二技能的凈瓶女媧，防禦資質1404，體力資質5527，速度資質1279，寵物成長1.282，果然不愧是入駐武神壇的大佬。再來看看技能，高級反震、高級魔之心、高級強壯、高級神佑復生，該有的技能全部都有。僅從裝備、寵物來看，這上武神壇沒有什麼問題了。

對於該女玩家曬出這樣的裝備和寵物，評論區玩家直接炸了。確認過眼神，是說書人高攀不起的大佬，並且還是那種能打蚩尤，並且還能贏的大佬。雖然該女玩家的裝備、寵物非常強強力，但也有玩家表示，這樣的裝備寵物雖然可以上武神壇，但能取得什麼樣的成績就不好說了，畢竟武神壇上多的是大佬！
