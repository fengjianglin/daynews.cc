冠軍教練kkoma，“雙蕉”組合Forge加iBoy，VG新賽季能否起飛？

2019-12-18 573條評論 1,753次閱讀 456人點贊

大家好，我是愛玩遊戲的小蘿醬，每天分享最新的遊戲資訊和有趣的遊戲內容，嗯哼~

雖然轉會期結束了，但戰隊宣布新選手的加入還沒停止。今天IG發布了關於二隊三名選手的調整公告，IGY輔助選手Fate上調至一隊，上單選手705轉會至JDG戰隊，Forge加入了VG戰隊。再加上先前樂言上調至一隊，看來昔日的IGY五人也即將散成滿天星，這些小夥子在自己的追夢道路上又邁出了一大步，剩下的AD選手huanfeng在LDL的表現也是極為亮眼的，應該也會出現在LPL的戰隊。

同一時間VG也對Forge表示了歡迎，自己曾經的下路組合去了IG，IG二隊的寶貝中單來到了VG，兩支戰隊也算是因此「聯姻」了。其實關於Forge去到VG，IG的粉絲們是非常開心的，畢竟這樣一名有實力的選手，應該在更大的賽場上發揮。不過還有一絲捨不得，在之前肉雞離隊的那段時間，Forge臨危受命打了幾場比賽，出色的發揮就已經被大家肯定他可以做肉雞的接班人，所以一部分粉絲們更希望他也可以升入一隊。

說實話肉雞的年齡已經不小了，也遲早會有一天退役，不過現在的狀態還能支撐他繼續扛起IG的中路大旗。Forge雖然強，但是現在來到IG的作用也不能和肉雞相比，所以與其讓他來IG當替補，不如先去VG磨練一番，還有機會與自己的偶像肉雞對線，觀眾們也能一飽眼福，所以說Forge加入VG是兩全其美的一件好事。

VG在之前公布了新的AD選手，正是來自EDG的iBoy，再加上Forge，這對雙C看起來還是非常強的，還被大家成為絕代雙「蕉」組合。Forge在LPL短暫的登場就給人留下深刻的印象，除了實力，另一方面就是長得太像蕉太郎了，而iBoy因為喜歡吃香蕉，也有個外號為「蕉蕉」，之前雙方的粉絲還因此產生過矛盾，現在兩人是隊友了，大家也就化干戈為玉帛，一起為他們加油。

VG在今年的成績並不好，屬於LPL墊底的水平，並且縱觀歷史，VG也沒有什麼拿得出手的好成績。但其實這支戰隊在人員配置上一直不錯，先後擁有過很多世界知名選手，比如Dandy，Mata，Easyhoon，Bengi等等，或許正是VG過於追求選手的質量，其他方面沒能做好，導致陣容起飛但沒有成績。

也可以看出VG是捨得花大價錢的，能得到冠軍教練kkoma的青睞，VG一定使用了大量的人力物力以及財力。無數的例子已經證明了，擁有出色的教練團隊才是重中之重，而kkoma作為全世界最頂尖的教練之一，能得到他的幫助無疑對VG的提升是最大的。

kkoma在訓練隊員上是一把好手，同時Forge和iBoy也是難得的好選手，應該能成為kkoma手中的一把利刃，單從「雙蕉」組合和kkoma教練這一配置上來看，VG就大有起飛之勢，相信新的賽季他們能取得理想的成績。
