風暴英雄不少角色源自魔獸，但設計更加洗鍊

天天要聞
2019-12-21

風暴英雄作為一款匯聚了大量暴雪旗下知名遊戲要素的MOBA，英雄皆來自諸如星際爭霸、守望先鋒、魔獸世界等知名系列。島田一族的源氏，蟲群之母扎加拉，暴風城國王安度因等等……只要對暴雪遊戲足夠了解，你可以在裡頭看到各種熟悉的面孔。

​

或許是由於最近魔獸世界懷舊服的火爆，近期風暴英雄的主題也比較偏重於魔獸主題。首先新版本的標題就是魔獸玩家耳熟能詳的大災變，並且新英雄也同樣是來自魔獸世界的滅世者死亡之翼。先前還舉辦了使用魔獸英雄就會送稀有戰利品箱的活動。雖說該活動已經結束了，但錯過的玩家也不用慌，風暴英雄作為一款良心遊戲，福利活動一直舉辦得很頻繁，近期正在舉辦的2019冬幕節活動，只要完成一些簡單的任務就可以獲得海量頭像、皮膚獎勵。

​

而風暴英雄的魔獸元素，除了擁有目前最具話題的奧山戰場地圖外，還將各種魔獸世界的經典角色以英雄的形式進行了神還原。例如早在魔獸爭霸3便結下樑子的伊利丹和守望者瑪維。這對仇家，一個為了力量不擇手段，一個為了復仇苦苦追殺。雖說魔獸爭霸3中就已經作為可選英雄登場，但風暴英雄選擇將他們重製，將他們的人物特質更巧妙的融入了技能機制當中，呈現出兩名既神還原，又有趣的英雄。

首先身為惡魔獵手鼻祖的伊利丹，雙刀在手，不但擁有著雙持所帶來的極高攻速，身形也十分靈巧。他不但可以在人群中自由穿梭，必要時還能翻滾越過對方的肩頭，一邊閃躲敵人的圍追堵截，一邊揮舞雙刃，把敵人玩弄在鼓掌之間。而大招惡魔變形與惡魔追擊，則很好的還原了伊利丹身上的惡魔之力。

​

至於身為追捕者的瑪維，則突出了抓捕敵人這一主題。她可以將敵人用鎖鏈捆綁起來，並朝著自身牽引，還可以運用空翻閃避敵人的反擊。大招守望者牢籠更是可以利用重重化身形成的牢籠將敵人圍困起來，使其無法逃出半步。

除此之外，諸如巫妖王等魔獸世界的「大腕」也全都有在風暴英雄中登場。想要親身操作艾澤拉斯各界大佬的願望，唯有在風暴英雄中才能得以實現。
