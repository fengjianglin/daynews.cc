《軒轅劍龍舞雲山》評測——新老交替中的遊戲發展方向

2019-12-22 87條評論 1,299次閱讀 245人點贊

歷史的發展總有令人瞠目結舌的情形，一紙遊戲禁令下發，讓中國的遊戲市場徹底陷入了昏暗的境地，所有的工作都轉入地下進行。但偏偏就是因此，催生了中國的網遊市場，中國的網遊市場在數年來不斷成長，現在已經形成全球最大且最具發展潛力的遊戲市場。而隨著智能手機的發展，手游成了遊戲行業的主陣地之一，很多老牌IP開始轉向手游，《軒轅劍龍舞雲山》就是其中之一，他是取材於著名單機遊戲《軒轅劍》系列的改編手游，也是目前唯一獲得正版授權的《軒轅劍》系列手游。

【天之痕劇情藍本，重溫上古神器世界】

國內的玩家，很多人是從《軒轅劍3：天之痕》接觸到這一系列的，這一版也是系列受歡迎程度最高的一部作品。這一次的軒轅劍正版手游，同樣以《軒轅劍3：天之痕》的劇情為藍本，結合全新劇情改編而成。陳靖仇、於小雪、宇文拓這些經典角色和伏魔山之類的經典場景，將又一次出現在熒屏上，於小雪的兩難抉擇、陳靖仇的忍痛弒師，將又一次勾起我們曾經的感動。這一次在手游版中所有的劇情都將被收錄，並進行高清重製，所有劇情均以動畫CG形式進行呈現，能夠直觀的感受每個人的愛恨情仇，比老版本的端游和IOS移植版更為震撼，代入感更強。

此外，開發組獲得了《軒轅劍叄：雲和山的彼端》和《軒轅劍叄外傳：天之痕》正版授權，玩家可以在遊戲中完整體驗老版《軒轅劍》的精彩劇情。值得一提的是，老版本的《軒轅劍》系列有一個問題，那就是說明過少，稍有偏差玩家就記不得往哪裡走了。這次的《軒轅劍龍舞雲山》中，絕對不會有類似的問題，玩家隨時可以查詢正在進行的故事劇情和其他情況，人物的資料和故事也可以進行查詢，即使是新粉絲或者已經不記得劇情的老粉絲，也可以迅速了解天之痕的故事，並快速上手，體會曾經的感動，這算是針對老端游所做的小修小補。

【畫質高清改良，情懷卻稍有缺失】

遊戲既然是全新移植，一番高清重製必不可少，《軒轅劍龍舞雲山》的新場景為全3D開放世界，畫風為唐代風格，場景設計方面，開發人員下了很大的功夫，不但還原了唐代風格的建築和場景人物，更是將一些盛世景象融入遊戲當中，如夢似幻的桃園盛景、空靈聖潔的天山草甸、恢弘壯闊的敦煌大漠、氣吞山河的鐵馬雄關，這些具有唐代標誌性風格的內容都被融入到遊戲當中，即使不做任務，在遊戲的開放性大世界中遊走，感受唐代的唯美和壯麗，也是一件很舒服的事情。

遊戲畫面以水墨畫為主，在具體的細節方面，將虛實、肌理、暈染等東方傳統繪畫技藝加強運用，遊戲中色彩分明，對比強烈，連縫隙和痕迹都可以看得很清晰，效果展示十分豐滿，給人以極強的視覺衝擊力，其特效甚至不亞於很多高品質端游水準。

經過了高清重製的《軒轅劍龍舞雲山》畫面效果驚艷，不過也帶來了新的問題。由於老端游採用的是2D畫面，所以在具體的製作上，情懷顯得略微不足，其華麗的唐朝風格完全佔據了遊戲主流，一些懷舊的場地因為全新重製而顯得有些「面目全非」。雖然看著這樣的盛世唐鳳是一種視覺盛宴的享受，但對於情懷玩家來說，多少還是有點不完美。

【花樣玩法，整體改良】

卡牌戰鬥自《陰陽師》之後就完全大火，不過很多遊戲都是照搬照抄，卻沒有進行革新和研發。《軒轅劍龍舞雲山》對現有卡牌戰鬥的弊端進行了改良，在遊戲中適當的縮小了人物的模型，這樣可以使得人物形象突出，但又不至於太突出導致戰鬥場面不夠壯觀，人物和場景的大小處在一個差不多的火候，視覺效果也有了相應的提升。

玩法上，除了傳統的陣容搭配和技能相輔，還引入了門派技能搭配和護駕、煉妖等系統，這使得在一般的陣容搭配和技能使用基礎上，還需要考慮到額外的門派技能和護駕等因素，這些技能往往是一些可以影響陣容架構的技能，可以對整體局勢產生很大的影響，例如影響屬性，有時候可以快人一步出手，讓一些薄積厚發的陣容直接失去效力，從根本上上改變戰局，甚至有可能影響到整個陣容的優勢發揮！這些系統給卡牌戰鬥帶來了新的挑戰，變的更為刺激，更為過癮。

如果只有卡牌戰鬥，每天只有打來打去，難免會出現視覺疲勞情形。《軒轅劍龍舞雲山》針對這一點，做出了多種不同的花樣性玩法，遊戲中不僅僅是卡牌戰鬥，還有一些娛樂性玩法，隨機取名的說文解字、古代行當系列玩法、海上絲綢之路行商等內容，戰鬥之餘，晚一些趣味性玩法，看一些經典的彩蛋（這也是軒轅劍系列的一個經典設定），可以稍微放鬆一下，也是一個不錯的地方。

【符鬼煉化，毀經典還是新戰力？】

符鬼是《軒轅劍3：天之痕》中的一個特色系統，《軒轅劍龍舞雲山》中，符鬼的收集和培養也是一個重點玩法。玩家可以收集不同的符鬼進行煉化、培養，最終成為自己的戰力。關於符鬼，它們有情感，知忠貞，在培養後同樣可以學習法術，提升戰力，是玩家戰鬥力的重要一環。這一充滿情懷的設定，在內測時期就獲得了玩家好評。不過符鬼煉化那些事帶來的一系列連鎖反應，大家想必都懂。

【全新社交，演繹全新人生】

記得《頭號玩家》中的設定嗎？每個人都以自己的真實形象走入遊戲當中，在遊戲世界中的形象就是你本人。

《軒轅劍龍舞雲山》中也有類似的設定，遊戲中擁有一套完整的社交系統，擁有社交類APP所擁有的一套社交系統，你可以像打造自己的博客，微信一樣來打造自己在遊戲中的形象，上傳自己的照片、書講述己的故事、書寫自己的遊戲傳奇，都可以做，和真正的社交軟體幾乎沒有差別。在這裡，可以玩遊戲、互動、交友，社交功能非常完善。此外你還可以自己規劃角色的人生體驗，你可以體驗中國古代的各個行當樂趣，樂師、收藏家、大文豪，自己的人生隨時可以自己來演繹！

通過聲音找到有緣人，是《軒轅劍龍舞雲山》的一大亮點，在「聲音社交」系統中，玩家可以留下自己的聲音，等待有緣之人的傾聽。遊戲中為不同類型的玩家提供了不同類型的相關設定，無論你是萌系少女還是翩翩公子，都有適合你的台詞可以選擇。展現真實的自己，你就是頭號玩家！

【總結】

新的時代到來了，老端游自然需要有自己的新活力，這樣才能長久不衰。《軒轅劍龍舞雲山》在《軒轅劍》系列的基本藍圖上進行更新，並針對現有同類遊戲的特點進行改良，的確做出了一定的成績。不但收錄有老版本故事，全新的劇情同樣是充滿了江湖俠義和濃濃感情，高清的重製和大量的高清CG，邀請到多位知名聲優進行配音，玩法上也做出了不少創新，這樣的架勢，也足夠證明網易的確是野心勃勃，想要重新打響《軒轅劍》這一經典IP。但這樣的改良真的」過於全新」，一定程度上消除了過去的情懷印記，在移植手游「情懷為重」的浪潮中到底能不能贏得玩家的厚愛，還需要時間來證明。
