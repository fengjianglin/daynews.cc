Theshy帶水友玩騷套路，獅子狗和機器人走下，對面直言被噁心壞了

2019-12-24 36條評論 1,121次閱讀 50人點贊

英雄聯盟這款遊戲已經快要來到了它的第十個年頭S10，在這十年間，可以說英雄聯盟探索出了一套較為成熟的職業聯賽模式，而且英雄聯盟職業賽事每年都會出一些新秀，可謂是人才輩出。如果說天賦和努力是職業選手的兩個重要屬性，那麼兩者兼備的玩家便可以成就一番事業，而Theshy便是天賦和努力兼備的超級明星選手！身為韓國人的Theshy最初來到中國賽區尋找機會，那時的他只是一個身處異鄉的無名之輩，對未來有著無盡的茫然，但他最終熬了過來。而這次Theshy帶水友玩騷套路上分，獅子狗和機器人走下，對面直言被噁心壞了！

在這場韓服高端排位局中，TheShy玩的是下路獅子狗，跟他一起走下路的是輔助機器人。從機器人的ID是「Theshy3」可以看出他是Shy哥的水友，兩人經常一起雙排玩騷套路上分。目前英雄聯盟版本中的ADC英雄實在是太弱勢了，所以這次Theshy帶著他的水友玩了一次獅子狗和機器人的下路騷套路組合，就是瞄著對面ADC伊澤瑞爾殺！在1級時，Theshy的水友所操刀的機器人成功地用Q技能鉤中對面伊澤瑞爾，隨後對面伊澤瑞爾瞬間被掛上了虛弱和點燃，最後Theshy所使用的獅子狗在草叢裡很快樂地跳來跳去並拿下一血！

在Theshy和水友使用的獅子狗配機器人的騷套路拿到優勢後，竟然直接蹲守在下路一二塔之間的那個靠近石頭怪的草叢裡，靜候著獵物的落網。對面伊澤瑞爾明顯感覺到了Theshy所操刀的獅子狗的強大殺意，只得貼著牆走。Theshy發現抓不到對面伊澤瑞爾後，就直接開始斷兵線不讓對面下路雙人組發育。Theshy水友所使用的機器人果斷閃現跟E技能擊飛對面奶媽，隨後Theshy所操刀的獅子狗跟上輸出，直接秒殺了對面輔助奶媽，氣得對面伊澤瑞爾連忙敗退到二塔處。此時Theshy和水友一看殺不了對面伊澤瑞爾，便去下路控線吃塔皮。

隨後Theshy帶著水友就卡在之前的草叢裡，只要對面伊澤瑞爾敢露面，就優先擊殺他，簡直就是行走的「AD去質器」。Theshy所操刀的獅子狗僅在遊戲開局的九分鐘內便拿下了十個人頭。而當Theshy和水友玩的騷套路在下路打出了優勢，對面下路雙人組伊澤瑞爾和奶媽根本沒辦法發育，只得叫上隊友打野盲僧來幫忙抓一波。Theshy所操刀的獅子狗見自己是大優勢，看到對面打野盲僧的到來也不畏懼，直接帶著水友來了個2V3。畢竟Theshy的以少打多經驗很豐富，而且以少勝多是IG戰隊的傳統藝能。但這次Theshy卻被制裁了，送出了1000賞金！

當Theshy看到自己被對面打野盲僧盯上後，便主動開始換對線思路，由無腦壓制對面下路雙人組轉換到控兵線壓縮對面下路雙人組的發育空間，這是間接的讓對面下路雙人組慢性死亡，真是殺人誅心！遊戲進行到中後期，只要Theshy所操刀的獅子狗看見對面的下路雙人組，便果斷開啟大招撲過去，一套技能連招一氣呵成地帶走了對面，沒有一點多餘的花里胡哨操作！Theshy所操刀獅子狗憑藉W技能的回血效果，配合隊友無腦越對面的高地塔，氣得對面下路雙人組伊澤瑞爾和奶媽直言這獅子狗騷套路真噁心人！

Theshy，他是矜琴心君子如玉，謙謙風度以及溫潤有禮的代表。他可以做到凝劍魄英雄無雙，單殺冠絕且壓制兇狠，而縱使上路有英傑百千，他儼然是那上單的審判法則。他的峽谷沒有止戰之地，唯有隊友羈絆在牽動著他的心頭，他就是世界所公認的頂尖上單Theshy！

不知道大家被英雄聯盟的騷套路所噁心過沒？最後不知道大家怎麼看待這次Theshy帶水友玩騷套路上分，獅子狗和機器人走下，對面直言被噁心壞了！歡迎大家的評論和關注喲。
