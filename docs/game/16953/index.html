就在今晚，LPL最有潛力的隊伍組建完成，御三家饞哭了

2019-12-15 717條評論 2,471次閱讀 1421人點贊

就在剛剛，TES俱樂部再度送出兩雙鞋子，宣布將青訓的下路組合提到一隊，也就意味著滔搏新賽季的大名單基本確定了，可以說非常的有潛力，御三家都饞哭了。要說轉會期初期，幾乎一有選手的轉會消息大家基本都說去了滔搏，不過事實證明滔搏只買了Karsa，雖然有錢也不能亂花，而且就目前滔搏的陣容情況，只能說有無限的可能，上限非常的高。

其實今年滔搏的表現大家也看到了，非常的有衝勁，只可惜最終差一步，未能進入世界賽。其中隊伍中最大的問題應該就是下路了，Loken的表現壞多好少成為了隱患，再加上上路369的不穩定性確實讓滔搏未能走出哪那一步，而如今新陣容組建完畢，汪先生覺得真的很可能要預定S10的席位了。

首先我們先從留隊的兩位老成員說起，雖然是老成員，但是其實也是非常年輕的小將，分別是剛剛提到的上單369以及中單Knight9,369的問題就是剛剛提到的，不穩定，上限很高下限同樣低得離譜，不過這終歸還是年輕時候的事，經過一年的歷練想必369肯定能更加的成熟，新賽季的表現汪先生覺得可以期待一手。

然後就是中單左手了，最近左手的狀態依舊的火熱，目前已經是韓服第三，很可能再度登頂韓服，而作為最受期待的Knight9唯一缺的應該就只有世界賽的歷練了。

然後說說大家非常熟悉的Karsa，就汪先生個人認為比之原打野Xx，Karsa應該是更強的。當然了不然滔搏也不會做此選擇，而最近大家也是都知道Karsa和左手的韓服雙排戰績非常的恐怖，其他家的粉絲都饞哭了。

最後就是今天官宣的青訓小將了，這兩位小將汪先生還是比較陌生的，不知道有沒有小夥伴可以科普一下他們的表現。當然了，汪先生也是查看了他們的韓服戰績，兩人都是韓服宗師段位，ADCPhotic最近使用最多的英雄是霞和女槍，表現得都非常的不錯。而複雜yuyanjia則是各類輔助英雄都有見上手，其中錘石應該是比較拿手的英雄。總之新選手的表現還是值得期待的。

而就目前的情況看，滔搏應該是最先能確定陣容的戰隊了，而且這個陣容組合御三家真的都饞哭了。不過也難怪御三家在今年的轉會期相當不好過，說是最慘的也不為過，RNG還好，EDG也還過得去，WE隊員走了也就算了，教練都走了，看了昨天閃電狼的新聞，汪先生都有點擔心WE也突然宣布不參與S10，又或者說我們可以等一手WEweixiao？大家覺得呢
