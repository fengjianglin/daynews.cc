超豐厚福利助力總決賽！海量點券、A車、套裝給你驚喜

2019-12-14 378條評論 1,417次閱讀 433人點贊

2019S聯賽·秋季賽總決賽即將打響，QG與RSG兩支俱樂部將於12月21日決戰成都。在巔峰對決來臨之前，橘妹可是準備了一大波福利給車手們哦~

冠軍勢力榜 預測得1000點券

王者之師與虎狼之師的巔峰對決，車手們更看好誰呢？12月9日-12月21日，車手們登錄遊戲即可於福利中心參與冠軍猜想活動，成功猜到QG和RSG到底誰是冠軍，就可以獲得點券*1000！

車手們每天完成每日任務獲取投票券，為喜愛的隊伍投票打Call，還能贏取橙光能源、許願金幣、改裝科技箱[中]等獎勵喲~

紫電能源

橙光能源

許願金幣

累計投票2次——初級能量體*1

累計投票4次——紫晶能量

累計投票6次——紫電能源+點券*50

累計投票8次——許願金幣+點券*100

累計投票10次——橙光能源、改裝科技箱[中]*1

榮耀挑戰-QG篇

除了競猜冠軍隊伍外，車手們也別忘了正在火熱進行中的榮耀挑戰-QG篇。12月10日-12月16日期間，參與限時開啟的秋決戰隊·QG篇-榮耀挑戰，即可贏A車晶耀之星吧~12月17日-12月21日期間，RSG戰隊的榮耀挑戰也將會開啟哦！

秋決戰隊·QG篇-榮耀挑戰活動期間，完成相關挑戰可獲得科技點、點券、紫電能源等獎勵，並收集聯賽之心。集齊聯賽之心*21即可兌換晶耀之星（10天）！

▲聯賽之心

限時的晶耀眼之星不夠香？還想獲得永久晶耀之星？那就在秋決戰隊·QG篇-榮耀挑戰活動期間，前往QQ飛車手游S聯賽官方微博，在相關微博下回復挑戰關卡名稱對應所指的QG戰隊選手名字，橘妹將從回答正確的車手中，抽取3名送晶耀之星（永久）哦！

猜決賽比分，贏1000點券

決賽預言帝在哪？快來拿點券！12月13日-12月20日，車手們登錄遊戲即可領取冠軍徽章，使用徽章押注總決賽比分，押中的玩家就可以獲得點券*1000+橙光能源*3的獎勵喲~

▲冠軍徽章

組團觀賽點券領不停

精彩刺激的比賽，當然要約上小夥伴一起呀~12月13日-12月21日，車手們可以在賽事中心參與活動，與好友們一同組團觀賽，為喜愛的隊伍打Call。完成組團以及總決賽當天觀賽均可領取點券好禮。

注意：一旦創建隊伍成為隊長就無法再加入其他隊伍， 若想加入好友隊伍，請勿創建，等待好友邀請~

隊伍中有2名成員——點券*50

隊伍中有3名成員——點券*100

3人組隊後，12月21日前往遊戲內電視台觀看決賽3分鐘即可領取點券*300，如果所在隊伍的助威俱樂部獲得冠軍更可領取點券*100！

集選手卡片 贏永久極品手杖

以為福利就這些？NONONO~12月13日-12月25日，車手們在賽事中心參與活動，還可收集決賽選手卡片。累計收集一定數量的卡片，即可領取點券、科技點、永久極品手杖等好禮喲~

▲極速冰晶手杖

收集3張卡片——科技點*88

收集6張卡片——點券*128

收集9張卡片——極速冰晶手杖（永久）

購買S券送賽事大禮包

有手杖當然也少不了好看的套裝！12月22日前，車手們可以在遊戲內賽事中心或是電競公告參與活動。充值滿一定金額的S券即可領取賽事大禮包。S聯賽專屬特效頭像框、炫影服飾、橙光能源等海量獎品等你來拿喲~

應援得S-烈魂者及專屬套裝

S聯賽秋季賽應援計劃也正在火熱進行中，車手們參與應援活動，為喜愛的戰隊助威打call，即可獲得海量獎勵，更有S-烈魂者（限時的，共計70天）和永久專屬套裝——夜遊俠客等你來拿！

▲夜遊俠客套裝

12月21日，QQ飛車手游2019S聯賽秋季賽總決賽準時打響。兩個賽季之後，RSG重返總決賽舞台，與QG一同決戰成都，爭奪冠軍獎盃。車手們要記得觀看比賽，一起見證這一極速時刻、拿豐厚極品獎勵！
