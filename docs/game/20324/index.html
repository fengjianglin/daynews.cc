對話網易&心動：末日等細分品類2019大火 不是追風口、而是等風口

2019-12-17 1,337條評論 3,257次閱讀 1994人點贊

2019 年手游出海出現了幾個小風口，吃雞手游的全球爆火、放置 RPG 在歐美市場風靡，以及末日、黑幫等細分題材的遊戲繼 2018 年興起之後，依然火爆。

受訪人分別為網易《明日之後》海外版發行負責人袁澤輝、心動網路創始人黃希威、金山辦公COO章慶元、寶寶巴士聯合創始人&APP事業部負責人盧學明

在 2019 年 Google Playtime 的開發者對話環節，心動、網易作為遊戲開發者代表接受採訪，就行業最關心的一些問題，例如怎樣抓住市場機會進行了解答。以下為關鍵信息整理：

記者：末日題材很火，但具有全球普適性嗎？

網易袁澤輝：就我理解，全球用戶對末日元素在審美上會有微妙差異。舉個例子，日本和美國的用戶對於『喪屍』的理解是不同的，北美用戶喜歡的喪屍外形更具刺激性、日本用戶不喜歡太噁心的形象、最好搞笑一點。但對於用戶差異，由於海外版是全球同包，調整在發行端，不會在產品端做很多的差異化處理。

末日題材遊戲是表層，惡劣生存下的用戶互動是底層。

因為在生存類遊戲當中，最吸量的元素就是那些惡劣生存條件的元素，就像天氣、喪屍、飢餓。但深層次地去看用戶需求，我覺得生存題材的粉絲，除了感興趣之外，他們更感興趣的是再往下一層的東西，比如說在遇到惡劣生存環境的情況下人跟人是怎麼在一起熬過這個難關的。

我們自己的製作人認為，生存遊戲就是給用戶製造一些生存困難，一群人在困難環境當中活下去或活不下去，其實是比一個人在困難環境當中活下去或活不下去的遊戲體驗要好的。所以我們在全球化過程當中，營銷內容也好，產品內容也好，除了抓生存要素之外，主要在抓的一個東西是我們怎麼讓用戶抱團取暖，就像這個遊戲的國服slogan說的，「活下去，一起」，關注的不單是「活下去」，關注的是「一起」，所以我們在營銷端、產品端更多地去討論一些人性相關的東西，這樣避免去特別細緻地區分印尼用戶喜歡這個喪屍長什麼樣，美國的用戶喜歡長什麼樣，這個是沒有辦法做到一個形象全球用戶都去喜歡的。

記者：《明日之後》這個遊戲剛好踩到了生存題材比較熱的節點，那麼最初立項的時候是怎麼去考慮這個事情的？

網易袁澤輝：我們關注生存這個品類的時間點要比這個品類開始有大家肉眼可見的發展要早得多，大概在三年多之前。

當時決定做這個產品其實更多是一個環境因素，之前我們對這個品類的關注已經有大約兩年時間，發現連續兩年這個品類的用戶都在增長，而且增長倍率是非常高的，可能每半年或者每一年都有一個成倍的增長速度。這個契機讓我們兩年多以前立項去做了這個項目。這是第一點，數據方面的考慮。

第二個考慮，就是開發團隊本身，包括製作人本身自己對這個品類的一些作品，特別是過去的一些主機遊戲玩得比較深。網易本身也是做MMORPG經驗比較豐富的公司，製作人之前也是出身於公司另外一款MMORPG產品，所以他就想自己喜歡的生存遊戲要素跟過往的遊戲開發經歷能不能結合起來做一款產品。剛好在兩年多前的這個節點，想到了這樣的結合方式，把沙盒的元素、營地的元素，加上一些生存遊戲裡面大家都耳熟能詳的元素，去結合做成產品，所以就有了這樣的一款產品。

記者：那問下心動網路的黃總，怎麼看待放置類遊戲這個風口？

心動網路黃希威：放置類這個領域，其實不能說是風口，只是最近大家做爆了而已。

我們第一款做的放置類遊戲是《天天掛機》，大概也在三四年前，這是第一款真正的掛機手游，我們當時是代理，覺得這個產品很好。當時，心動網路剛好又拿了《仙境傳說RO》的IP，就根據我們自己對放置類遊戲的理解做了《天天打波利》，跟《天天掛機》看起來是蠻像的，但它是一個進化的版本，核心部分改的非常多。

《不休的烏拉拉》，也是一款很有特色的產品。所以我們覺得放置類遊戲應該還會有更多的產品，甚至可能成為一個單獨的品類，會有各種不同的玩家進去，但如果做的都是完全一樣的東西，我覺得成功機會很小。

記者：《不休的烏拉拉》目前在全球的表現符合你的預期嗎？

心動網路黃希威：《不休的烏拉拉》這個產品，在前期其實是超過我們對這個產品的定位的，在後期確實因為內容跟不上，沒達到我們自己的預期。

我們第一站選的是港台市場，但是很快就發現總要填一輪坑，創新的產品風險在於很多東西不能借鑒，所以你在補後面內容的時候發現不一定你做的就是對的，所以這個坑要一步步地去踩、去填。

記者：除了從TapTap吸引一些手游開發者之外，像網易這樣的大廠也會把產品交給心動去發行，你覺得心動網路在這方面的核心優勢到底在哪裡？

心動網路黃希威：首先我們會去了解產品，會跟開發者溝通。因為你想拿到產品，一定要得到它的製作人認可。我們會去講我們對產品的理解，如果製作人覺得你懂他的產品，他才會把產品給你。另外有一些之前的案例，比如說我們確實在一些地區做得比較好、比較細，深耕了很久，他們對於我們在這個地區的表現認可。

所以，一般來說我們靠兩端去打動開發者，一是市場，你做得夠深，投入夠大，投入夠大不僅是錢的問題，還有一個精力的問題，特別是做ACG產品，是一個需要大量投入人力的地方，因為要去和玩家溝通，而不只是買量。

另外，就是細節溝通的時候，開發者需要你對他的產品的有一定了解，儘管不需要達到和他一樣的程度，但最少你不能亂講，你要理解這個產品。所以我們做這個的優勢，不僅是說和廠商有合作關係，而是因為我們有TapTap這個平台，去觀察上線以前的產品是什麼樣的，有機會進一步去了解它，這可能是我們一個綜合性優勢。

記者：心動今年在國內外代理了很多熱門的產品，其實你們給自己的定位是什麼？

心動網路黃希威：我們給自己的定位是發行有特色的好遊戲。我們喜歡發行特別有特色的遊戲，不僅限於某一個領域。我們認為有差異化的產品更適合我們來做，如果差異化不夠，可能無法發揮出公司所有的競爭力。如果說定位，就是希望心動網路發出的產品跟市面上的產品不大一樣。
