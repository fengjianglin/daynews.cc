酒館戰棋：補丁之後誰主沉浮？機械惡魔各憑聖盾與大身材統治酒館

天天要聞
2019-12-23

大家好！我是爐石閑說，為您分享爐石魔獸趣聞趣事！

隨著前幾天16.0.5的補丁的發布，酒館戰棋中的魚人流的2張重要卡牌夢魘融合怪遭到刪除，退出了酒館戰棋，另一張魚人前期的血量保證的寒光先知，重新從2星被調回了3星。加之之前的蠻魚斥候從4星被調回5星，魚人流派除了獲得一張魚人人國王之外，再次兜兜轉轉回到了最初的狀態。

魚人流的削弱，以及野獸流強勢點熊媽媽依舊處於削弱狀態，導致其對手機械流再次憑藉強悍的前期優勢，再次在酒館中崛起（酒館戰棋剛開服的時候，機械流就統治過環境）。同時，在魚人、機械和野獸三家之外的，被大家認為是「臭弟弟」的惡魔流派，憑藉著設計師的虛空領主的減星（6→5）,以及新卡牌漂浮觀察者的補充，加上知名主播瓦娘在補丁之前就對惡魔流的不斷「安利」，惡魔流憑藉賣血，在順的情況下，能夠用高攻血的小粉和小綠迅速抬走未成型的其他對手。又因為魚人流派的削弱，惡魔流標誌性的高攻血，少了劇毒魚人和融合怪的天敵，在酒館戰棋中和機械流一樣，混得風生水起。接下來給大家分享下補丁之後各流派的具體情況。

首先是被削弱的魚人流，魚人流從酒館戰棋之初，就不被看好。因為不同於天梯的魚人快攻的印象，在酒館戰棋中，魚人流的成型速度可以說是最慢的。核心牌銅須和溫順的巨殼龍，都屬於高費的5星和6星。所以在加血魚、蠻魚斥候和毒鰭魚人紛紛被降低星級之後，魚人流派憑藉前期能與其他流派抗衡，後期幾乎無敵的優勢就統治了過去一大段時間的酒館環境，加之5星的新卡拜戈爾格國王的加入，魚人流迎來「錦上添花」，更加強勢。在補丁之後，加血魚、融合怪和蠻魚斥候的回調，魚人流再次顯露出前期弱勢的缺點，掉下了統治酒館的地位。

其次是機械流，機械流的核心卡牌是3星的鈷制衛士和曾經的4星的機械回收者。因為比較低的核心卡星級，機械流成型速度最快，能夠在前中期就能擊敗對手，在早期就統治了酒館環境。隨著核心棋子之一的機械回收者被增加到了5星之後，機械流就紛紛被後來的野獸流及魚人流代替，一直處於二流地位。在如今野獸流依舊一蹶不振，魚人流跌下神壇的情況下，機械流再次崛起，也在情理之中。

野獸流就不說了，除了瘟疫鼠群、土狼和「小媽」族群領袖用於前中期的「打工」

之外，幾乎難以成為一個成型的吃雞流派。同時，隨著光牙執行者的削弱，承載光牙buff的融合怪一同被踢出酒館，曾經「滿地找牙」的混合流，也銷聲匿跡。

最後的惡魔流，除了個別主播的「強勢推廣」之外，實際的賣血漲身材，鎖血迅速抬走對手成為了目前酒館戰棋中惡魔流的主流打法。還有其分支打法——噴子流，在前中期同樣是淘汰對手的好手，就是後期因為對手棋子血量上去了，需要轉型才能吃雞。

如今的酒館，復興的機械流和新貴惡魔流成為了2大吃雞爭鋒的流派，同時輔以偏門的刮痧流和奧秘哥的卡德加裂魂打法，可以說和補丁之前，環境算是大變樣了。以上就是對酒館戰棋補丁之後的環境分享，有其他觀點的，歡迎在下方評論指出！
