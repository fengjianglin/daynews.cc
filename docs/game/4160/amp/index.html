DOTA2：新英雄真有這麼厲害？在他兄弟面前，紫貓根本不夠看

天天要聞
2019-12-09

摘要

大家好，這裡是包雞包眼包開心的A電競飛機。說起DOTA2新版本最熱門的英雄，相信不少玩家都會想到紫貓，這個英雄能打能秀，強度極高，給大家留下深刻印象。但仔細觀察想想，各位會發現紫貓在他三個兄弟面前根本就不夠看。紫貓的三個兄弟便是鼎鼎有名的三貓組合，分別是藍貓、火貓、土貓。

大家好，這裡是包雞包眼包開心的A電競飛機。

說起DOTA2新版本最熱門的英雄，相信不少玩家都會想到紫貓，這個英雄能打能秀，強度極高，給大家留下深刻印象。

但仔細觀察想想，各位會發現紫貓在他三個兄弟面前根本就不夠看。

紫貓的三個兄弟便是鼎鼎有名的三貓組合，分別是藍貓、火貓、土貓。

這三兄弟目前雖然強度有所下降，可老玩家都清楚，他們在剛剛登場的那個版本究竟有多麼恐怖。

別看紫貓在天梯混得風生水起，跟另外三貓相比他只能算是一個弟弟。

首先我們來說說火貓吧，他在剛登場的那個版本堪稱無解，那時候火貓的傷害極高，罩子無法驅散還能吸收接近600左右的魔法傷害。

而1技能捆傷害跟時間遠不是現在可以比，最關鍵的是火貓的大招耗藍極低傷害卻相當恐怖，剛到6級火貓就能打出一套技能。

對面來幾個人殺幾個人，根本不和你講道理。

其次便是土貓，這英雄當初登場的時候因為操作複雜導致很少有玩家會使用，但只要是稍微會玩一點的人都明白土貓的可怕之處。

土貓才出來那時候還不是輔助而是核心中單，跟他對線的人千萬要小心，因為土貓不需要到6便能單殺對手，而且還是不講道理的強殺。

而高等級的土貓，簡直就是節奏之王，超遠距離的滾讓他成為許多英雄的噩夢。

最後再來看看藍貓，其實這個英雄根本就不用多說，前有世界第一3號位yyf使用藍貓七進七出怒守高地，後有DOTA2最強天才少年Sumail冠軍藍貓。

這個英雄天生便是創作奇蹟的英雄，雖然目前這幾位英雄都被削弱，但他們都曾創造過輝煌的歷史，與他們相比起來紫貓目前情況確實有些不夠看。

從冰蛙接連削弱紫貓的動作上看，這英雄想要達到他幾位兄弟曾經的高度或許有些困難，希望在接下來的版本中，紫貓可以順利崛起吧。
