魔獸世界懷舊服火法不管副本還是練級，都比不上冰法嗎？那沒事了

2019-12-15 917條評論 1,886次閱讀 525人點贊

魔獸世界無論什麼練級天賦不考慮PVP就是純粹空想，所以自用的天賦多少都有能打下PVP的考慮傾向在。霜寒刺骨固然是好，但一個完全顧及PVP和PVE的天賦是不存在的，這個天賦僅僅是我個人那麼多年FS野外生存，能做到的顧及PVP和練級所能做到的最好的天賦。如果是副本A怪的練級方式，當然就是AOE流天賦、可以solo任務、可以PVP、可以A一陣的天賦。

魔獸世界沒有完全完美兼顧PVP和PVE的天賦，這天賦是我用著最順手的。在升級過程中哪怕要打個戰場，這套天賦也不用洗。要奔單一目標去，肯定有更優的處理天賦。我也想過把冰錐的移到暴風雪上，但這樣PVP就缺了冰錐了。純冰錐練級是不合理的，因為練級壓根沒有命中裝可言，難道A怪純拉綠色等級的？這經驗都變少了，黃色等級的就有幾率未命中，暴風雪A怪是無視命中的，兩種A法誰更靠譜老玩家都清楚。

不管什麼痛苦術士，在60年代就沒有不點腐蝕天賦的，然後說說詛咒增幅。在我的SS生涯里，詛咒增幅絕大部分的使用是給疲勞詛咒的。第一種情況，對方在馬上，或者盜賊開加速，普通疲勞詛咒就行。兩點原因讓我放棄了這個方向，第一，連續性的問題。同時3-4怪後哪怕3-4怪時，由於他是個回藍BUFF，不是像70級的直接+的BUFF，所以有點雞肋了。不分流吧，藍緊張，分流吧，它就沒意義了。

第二點，如果分流抽血的體系的話，它對於控藍的循環體系變得大了，不如換藍抽血那麼直觀。當然，用吸靈魂石+抽藍的方式也可以永動，但就像我上面說的，體系變大，且照樣是抽血。那麼完全可以省略掉吸靈魂石這一步了。簡單些，就是加減法，血少了抽，藍少了換，很簡單。而且藍胖可以上去拉怪，獻祭都省了，3DOT上去，換藍，抽血，就行了。

首先火法練級續航成問題，比冰法坐地板的時候多得多，一下效率就下來。其次自保問題，打怪多引兩隻都吃不消，更何況pvp的存在，實際上如果沒有命中和暴擊的支撐，火法不管副本還是練級都是比不過冰法的，火法是後期裝備支撐下的升華天賦，並不是隨隨便便一想就能實現的。拋開A怪，其實冰法都不慢，何況還有這麼強力的控制A怪能力。
