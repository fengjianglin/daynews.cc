DOTA2：CN DOTA Never Die！VG3：0碾壓EG一穿五奪冠

天天要聞
2019-12-22

12月22日，新加披世界賽進入了最後一天決賽日的比賽，在敗者組決賽中VG2：0戰勝了紅星老師，之後在最後的決賽VG遇到了小組賽曾經2：0自己的EG戰隊，最終VG3：0碾壓EG，成功拿下了本賽季的第一個冠軍。本文素材來源競技寶，感謝jjbone的數據支持。

第一局比賽VG給Ori繼續選擇毒龍，而給拒絕者玩起了比賽勝率不高的猛獁。EG這邊也選擇了版本冷門醬油土貓以及弟弟卡。

對線期Crit遊走效率很高，但是VG這邊對線經濟並沒有落後太多，隨著猛獁和毒龍做出基本裝，VG連續打贏幾波小團戰將局勢翻了回來。之後拒絕者開啟了狂暴刷模式，VG四人抱團推塔，很快就取得了不錯的經濟優勢。後期Abed卡爾淪為天火怪，打團並沒有太多作用，最終VG穩紮穩打依靠薩爾大+猛獁大秒殺RTZ猴子先拿到1分。

第二局比賽EG選了一套小黑體系，而EG則選了飛機電棍雙核。

對線期Ori在大優的情況下被Abed電棍一換一，VG前期唯一的戰鬥力毒龍並沒有把電棍打爆。過了對線期EG開始加強自己的推進節奏，VG開始分散打錢，但是經濟最多的時候被拉開到了1W+。這時候拒絕者站了出來，一直在打錢發育經濟始終沒有落後。找到機會的VG在上路前哨開霧團滅EG，一波打回了5K經濟。後麵糰戰Pyw拉比克穩定偷到666全能大招，VG帶盾成功翻盤EG，在打比方取得了2：0的優勢。

第三局比賽，VG點出了一套絕活陣容，拒絕者的小魚，皮鞋的全能以及Ori的船長。而EG這邊則也亮了很招牌的三號位英雄謎團。

對線期VG上路下路都不佔優勢，只有Ori的船長依靠薩爾的遊走和電棍打了個五五開。比賽進入中期EG團控陣容打出優勢，VG經濟逐漸落後達到5K。但是中路一塔團戰，VG依靠小魚和船長的拉扯打出了不錯的效果，Dy大牛拿到三殺，VG順勢控盾打回了前期的局面劣勢。雖然拒絕者小魚帶盾，但是面對EG的陣容並不好推塔，VG盾的時間被EG托掉。30分鐘EG開霧交出全部大招擊殺了皮鞋的全能和Dy大牛，二人果斷買活反打了EG一波小團滅，VG順勢控掉二代盾。VG拿盾後並沒有選擇上高而是不斷壓制EG野區。46分鐘EG謎團刷新拉大，VG依靠船油以及小魚人的買活成功團滅EG，並且收下第三代盾，最終拒絕者小魚人在上高時盯死謎團，EG敲出GG，VG成功1穿5奪冠！

這次比賽拒絕者的發揮真的太硬了，最後決賽三局比賽真的是把自己的個人能力打出來了。第一局領先其他人1W經濟的猛獁，第二局經濟一直沒被甩開的小黑以及第三局Carry全場的小魚人。現在拒絕者這個狀態真的好，可能這就是天道酬勤吧。
