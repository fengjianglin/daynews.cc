魔獸世界：祖傳肩甲幻化，8.0的還沒出現，9.0的已饞出口水

2019-12-25 58條評論 1,004次閱讀 76人點贊

近期，《魔獸世界》官方發布消息，確定「爭霸艾澤拉斯（BFA）」的最終版本8.3將於2020年1月16日正式上線。同時新團本「尼奧羅薩，覺醒之城」將於新版本上線的次周開放，而縱觀整個BFA，似乎少了一件東西。

無論是8.0初期團本奧迪爾里的戈霍恩，或是達薩羅之戰的吉安娜，又或者永恆王宮裡的艾薩拉，亦或是即將開放的尼奧羅薩里的恩佐斯，都缺少了一個非常關鍵的掉落，那便是BOSS同款肩膀！因此這也讓大量玩家覺得這是一個不完整的資料片。

5.0熊貓人之謎（MOP）的最終團本「決戰奧格瑞瑪」開啟了掉落BOSS同款肩膀的先河，當時的最終BOSS腦殘吼雖然性格魯莽激進，但是身上帶的東西可不含糊，不僅有坐騎還有板甲幻化肩膀「瑪諾洛斯的獠牙肩甲」，至今仍然是無數獸人玩家每周必刷的理由！

6.0德拉諾之王（WOD）的最終團本「地獄火堡壘」中，阿克蒙德也掉落BOSS同款裝備。而它掉落的則更多，不僅有同款肩膀「艾瑞達領主護肩」還有同款頭「惡魔王子的冠冕」與腰帶「阿古斯魔扣腰帶」，使得無數布甲玩家入了魔，畢竟能感受到BOSS的威儀。

7.0軍團再臨（LEG）則出現了BOSS同款皮甲肩膀「薩特先祖的肩甲」，該肩膀並非來自最終團本，而是產自初期團本「翡翠夢魘」的最終BOSS薩維斯。雖然當時屬性非常極品，但是以現在眼光來看已然過時，不過每周仍有不少玩家打著做「能量的平衡」名號來刷這個造型炫酷的肩膀。

BFA好看的裝備都在戰爭前線，副本裝獲得後立馬幻化掉，同時即將開放的團本則是觸手、眼球、牙齒等元素的組合，則讓人更沒有慾望。而暴雪嘉年華上爆出的9.0BOSS噬淵典獄長則被無數玩家定為新目標，從最近曝光的圖片來看，典獄長的肩膀是著實華麗，讓它還沒登場就讓玩家留下了口水。

小夥伴們認為暴雪會把這款肩膀加入BOSS掉落嗎？會是什麼種類的護甲呢？
