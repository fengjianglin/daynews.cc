DNF：搬磚裝備的製作講解，新人必看的攻略大全

2019-12-21 1,253條評論 2,819次閱讀 276人點贊

在上一期文章中，我們講解了搬磚裝備的搭配從入門到精通的攻略，個人認為美中不足的部分就是沒有涉及到裝備的製作問題，為了填補上一期文章的空缺，那麼本期文章抱著「指哪打哪」的戰略策略，重點講解一下關於搬磚裝備的製作問題。

搬磚裝備的製作與其搭配如出一轍，沿著裝備搭配的前-中-後期路線制定一整套相應的裝備製作方法以及對裝備製作的點評。

前期相關：聖物裝備

在通常情況下，聖物裝備是可通過拍賣行直接購買的，但是由於可交易的聖物裝備數量過少，通常能看到的物價都是比較偏高的，直接購買，性價比是不高的，所以，關於聖物裝備我們就要說到其第二種獲取方法，那麼就是自己製作。

聖物裝備是通過對應的假紫裝備，經過相關的設計圖可製作出來。比如說，95級的聖物裝備就是通過95級的假紫裝備進行設計出來的，具體方法如下：

還是以製作95級的聖物為例，首先要到拍賣行購買95級假紫裝備，具體以製作武器舉例：在拍賣行買到95級假紫武器之後，然後找到月光酒館的NPC卡拉卡斯，購買對應的95級聖物設計圖（一定要是對應的等級）以及95級聖物碎片10個

如上圖所示，製作一件95聖物除了要10個聖物碎片，還需12個稀有靈魂，24個高級靈魂以及36個普通靈魂，對於經常分解裝備的玩家來說，這些基本靈魂材料是很容易獲得的，換句話說，製作聖物無金幣成本代價。

雖然說製作成本不大，但是想要製作聖物裝備其必關聯到材料紅玉髓。不管是購買聖物設計圖還是購買聖物碎片，這裡都需要用到紅玉髓這個材料，而這個材料的產出是由相關使命任務獲得的。角色根據等級在升級路上，會對應每個使命任務獲取相對數量的紅玉髓材料，當角色滿級（95級）時，這個材料的獲取途徑變更為，對應相應等級的使命任務獲取，而這個使命任務的刷新頻率為每天，對應產出20個紅玉髓。

角色前期製作聖物裝備，當從1級升滿至95級時，所送的紅玉髓材料，足夠製作一整套聖物裝備了，在升級的路上只是要注意，在對應區域的使命任務需及時領取使命任務所得的材料，以不至於在製作時才想到要去完成前面的使命任務。

中期相關：艾肯系列裝備與哈林史詩裝備

（1）首先說一下艾肯系列裝備。艾肯副本經過改版之後，裝備大致分為3大系列，分別為，守護系列、勇氣系列以及意志系列裝備。這其中的守護系列就是搬磚有關的加速度類型的裝備，由於篇幅原因，勇氣與意志系列裝備就點到為止，而守護系列裝備則作為重點講解（3大系列裝備製作或者說是獲取方法是一樣的）

艾肯裝備大致分為3種獲取方法，第一種就是通關材料換取：

如上圖所示，在魔界集結地的NPC寒冰之休亞的左手邊鬥氣改造機出用材料[塔爾瑪的象徵]進行兌換守護系列自選禮盒，根據角色自身精通的甲種進行選取即可；

第二種就是刷副本[艾肯-遺落的神殿]，在副本中直接掉落守護系列裝備；由於有3類，具體能爆出守護系列裝備還得看臉。

第三種，也是不常用的一種方法，就是利用傳說裝備轉換書轉化以前遺留下來的斗、戰、狂、滅裝備。

以上3種獲取途徑都只是獲得守護系列裝備的原版，意思也就是說，這種原版裝備在沒經過處理之前是不提供速度加成的，所以，想要有速度加成，就必然經歷以下處理方法：

如上圖所示，在鬥氣改造機點擊改造裝備屬性，進入以上頁面，守護系列裝備可供改造的屬性分為四維和速度類，這裡我們選擇速度類的改造，一共可改造兩處地方，每處地方都需材料40個。改造完成後，就是名副其實的加速度搬磚裝備了。

艾肯系列裝備的製作，一件製作周期大概需要花4天時間，根據角色進入副本不需消耗疲勞以及改版後的副本容易刷的特性，在製作裝備過程中就當是搬磚也是可以的，副本每次通關翻牌所得金幣都是1W以上的，4次刷完，加上爆出的藍、白、假紫裝備收益穩定在5W金幣以上，完全可以做到製作裝備與搬磚兩不誤，非常划算。

（2）哈林史詩裝備。通常也有3種獲取途徑，第一種就是深淵掉落，由於深淵掉落隨機性很強，我們想要用於搬磚的哈林裝備為皮甲類的神龍之子套裝，如果想要準確的裝備獲取途徑，那麼只能通過第二種途徑，用材料[異行結晶]進行兌換或者是第三種途徑，對應95傳說裝備升級成哈林皮甲裝備。

除了深淵途徑，其餘兩種是在NPC紅鼻子德拉處實現操作的：

①材料兌換途徑。如上圖所示，兌換皮甲神龍之子套裝禮盒需要750個[異形結晶]以及40W金幣，禮盒為自選類的，缺哪件就選哪件。

②傳說升級途徑。如上圖所示，用對應的傳說防具加上所需的[金色小晶塊]100個、[上等皮革]20個以及[特級硬化劑]10個也是能準確的把神龍之子套裝做出來。

材料[異形結晶]產出副本大致有，普通95深淵、星空裂縫、黎明裂縫、災難領域等，其中大部分材料由黎明裂縫產出為主。

而對應的傳說防具則由材料[黑夜穿行之證]到NPC蒂莫西處兌換即可，[黑夜穿行之證]則是通過刷哈林-強襲模式獲得，以及冒險團的榮耀結晶兌換，有時候還有一些限時活動也能獲得這種材料，但是長期獲取途徑基本就是這兩種了。

綜合上述，哈林裝備的成型難度相對較大，製作一件大概需要花5天時間，其實5天並不長，配合冒險團優勢，前幾件一次性就能完成也不是不可能。

結合艾肯裝備與哈林裝備的製作過程得出，兩者本就可以同時進行，推薦左邊防具做哈林，右邊的首飾以及特殊裝做艾肯，這樣一整套搬磚裝備很快就能出爐，以完成角色中期所需的搬磚裝備需求。

後期相關：泰波爾斯史詩裝備

搬磚所用的就是泰波爾斯5甲中的重甲精靈的榮耀套裝，被評定是5甲中速度最快且手感最好的一套裝備。其獲取的方法有很多，作為史詩品質的裝備，長期獲取途徑以深淵和鳥背為主，而在當前的限時活動中，史詩之路也是獲取的重要途徑之一，以及包括師徒活動，除此之外，通過長期的集對應的碎片也可合成出來，正可謂是獲取途徑因有盡有。也正因如此，作為重點講解的永久途徑就以鳥背說一下吧！

鳥背是玩家的俗稱，它的學名為泰波爾斯-崩壞的樂園。此副本主要產出的材料名為[蒼穹碎片]，用此材料到NPC寒冰之休亞處可兌換泰波爾斯裝備，其中的精靈的榮耀套裝可直接兌換

如上圖所示，15個精鍊的時空石以及600個蒼穹碎片即可兌換一件精靈的榮耀；

除此之外，也可通過設計圖製作出來：

如上圖所示，只需滿足設計圖所需的材料即可製作對應的精靈的榮耀套裝。具體以90B娜迦王或者95哈林重甲失落的宏圖套裝對應部位結合材料[靈魂之源]3個以及精鍊的時空石5個也可設計出來。用設計圖設計的好處就是兌換所需的蒼穹碎片要少一些。

蒼穹碎片基本產於鳥背，也就是說，玩家想要通過此材料製作重甲，那麼就要去刷鳥背，而鳥背的入場門檻相對較高，但是玩家也別太擔心，實在刷不過，不是還有一個鳥背引導模式嗎？利用這個模式，來獲取蒼穹碎片也是不錯的選擇，雖然獲得的材料不及單人模式和組隊模式，但是也很滿足了。

精靈的榮耀套裝作為後期搬磚裝備的選擇，其成型難度是最大的，但是，在國服當前面臨著100級的沖刷影響，活動送出的裝備實在是太多了，想要湊齊一整套也可以說是不怎麼難的。推薦刷完鳥背後，直接去史詩之路肝這套裝備，在目前為止，應該是獲取的最快途徑之一。搬磚的勇士們，請把握住這次史詩之路的機會！
