絕地求生：IFTY舊劇重演？龍四哥直播痛罵薄涼："你是不是飄了"

2019-12-24 2,648條評論 4,945次閱讀 3166人點贊

2019年12月23日下午，IFTY戰隊的日常訓練賽開打，選手也是各自開播。在簽約虎牙之後，IFTY四名主力隊員也是擁有了各自的直播間，雖然按照A+之前的說法，4個隊員不拿世界冠軍直播間不分開，但最終還是各立門戶。而在一局訓練賽中，longskr痛罵薄涼的場景被觀眾盡收眼底，一時間論壇超話議論紛紛。這裡感謝競技寶JJBONE的諮詢支持。

我們在討論longskr與薄涼的爭吵之前，先來看看在爭吵之前IFTY到底發生了什麼。其實很簡單，就是IMPALA的一次落地CSGO，IFTY和天霸跳點衝突自然是要干一架的。在IFTY先手清掉天霸3人後（薄涼打倒兩人），IFTY的開開、龍四哥和老菊準備去高樓端掉最後一個獨狼，薄涼在打葯所以沒有跟。但longskr三人卻被1V3，事情自此開始。

Longskr的脾氣也算是比較爆的那一類。在被一串三之後，三人開始看回放找問題。而問題也是很快被找到，在三人沖房的路上，longskr先手被打時，開開似乎卡住了入口，後續的老菊和開開都沒有補上槍反而是被逐個擊破。對於這個問題，Longskr自然是非常難受，所以一直在teamspeak（職業選手的YY）里跟隊友溝通總結。

但此時薄涼還存活，就訓練賽的規矩來說自然是要玩下去的。這裡矛盾點就出現了，因為隊友一直在復盤所以薄涼選擇了暫時不聽團隊麥，但沒有告知隊友。而後續IFTY隊內怎麼吵的我們自然也是不得而知。不過可以預見的是，吵得很激烈。而關麥這一動作可能也是埋下了伏筆。

距離關麥可能有個5/6分鐘的樣子，在別人提醒下，薄涼打開了TS（TEAM SPEAK）詢問隊友怎麼了，並直言剛才關麥是因為他們在復盤，但薄涼仍存活所以關掉了，怕吵。而就是這個”吵”字，讓longskr可能感覺有些難受，緊接著就是一串髒話從耳機中噴涌而出，最後龍四哥的一句：”你*現在是不是飄了啊nmlg**！”（打星號大家懂的都懂）

在教練的調停之下，雙方也是暫時停止了爭吵，後續俱樂部可能也會有調整。小編還是希望這四個小伙好好打，畢竟從青訓一起打上來，感情也有，多溝通的話矛盾也是很好解決的。但這次直播事故也是給隊伍帶來了很多節奏，後續如何發展我們會及時關注。不過聽說A+已經出面了，我們靜待事情發展好吧。

想聽錄音的小夥伴可以去霹靂霹靂直接搜，小編的快訊分享也就到這裡了。有什麼意見大家可以直接評論區留言哦。
