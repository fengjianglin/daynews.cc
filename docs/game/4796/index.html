打到大師才發現126才是王道，一招空城計賺到70塊！

前言：一鍵仙人跪，兩袖青蛇舞，鍵來！又到了棋友們最喜歡的《雲頂之弈小課堂》環節了，本期有哪些精彩的對弈內容呢？咱們一睹為快。

還是5金幣的英雄夠頂

一位玩家在雲頂之弈論壇上說「還是5金幣的英雄厲害，對面根本沒有招架之力，就跟推土機似的，直接一路平推過去了」。

其實五金幣陣容有一個專門的名字叫大哥流，大哥流雖然猛，但是有一個非常大的缺點-不容易湊。不管是2星天使，2星死歌還是2星亞索，2星烏鴉，都能當大哥；平時玩家一局遊戲中能找到一個2星烏鴉就不錯了，同時找到這麼多2星大哥，只能說{小夥子，你的運氣真的很棒}。

126+空城計=完美開局

剛開始玩雲頂之弈的時候我們見到126開局就會說「哎呀，這局被天譴了，涼涼了」，126開局真的是天譴嗎？

如上圖所示，有24W人閱讀了這篇攻略，800多人評論，將近4000人收藏，將近2000人轉發。在評論裡面有的玩家說空城計就是自取滅亡，鐵憨憨的做法。而有的玩家帶著疑問實踐了一番，發現這個玩法「真香」，這就叫實踐出真知。

近日一位玩家說打到大師才發現126才是王道，配合一招空城計打完石頭人直接賺到70塊，之後想玩什麼就D什麼，基本上可以保二爭一。其實在9.16b版本中空城計等於加強了，因為玩家對裝備的需求越來越高了，下面再簡單說一下空城計的步驟：

1，如果126開局，直接把五張牌賣掉，打完野怪可以吃10元利息；然後繼續賣牌，保證第一輪真人PK結束後自己可以吃到20元利息。

2，在不影響利息的情況下可以留自己想要的牌，什麼是想要的牌呢？比如你玩賭刺，就可以留下三虛和刺客；比如你想玩動物園，就可以留下豹子，蜘蛛。

3，如果覺得自己的牌打不過石頭人，就補強一下。打完石頭人金幣一般可以存到70塊，有這70快，你想D什麼就可以D什麼。

這一套打法還有一個非常核心的優點：可以首先拿裝備，如果玩賭刺就可以拿金鏟鏟加大劍。如果玩動物園就可以拿電刀和離子燙的散件。

惡魔不缺輸出，搭配上護衛更容易上分

今天推薦一套雲頂之弈惡魔護衛陣容，惡魔其實是不缺輸出的，火男在後排只要能放出兩個大招，敵人基本上非死即傷。而護衛在前排可以幫助惡魔抗傷害，打控制；在實戰中來看，惡魔護衛確實很頂，比純惡魔更容易上分。

平時我們玩惡魔元素或者惡魔變形，喜歡把凱南或者納爾變成惡魔。其實還有一個牌非常適合變惡魔，他就是豬妹。在中期如果給豬妹一個暗裔，一局遊戲能放出兩個大招，搭配上護衛惡魔非常的好用。最後友情提示，惡魔陣容雖然基本沒什麼天敵，但是最怕雙龍和龍牙，尤其是帶龍牙的德萊文，惡魔見到基本上等於完蛋。

好了，以上就是全部內容，之後我會繼續分享一些對弈小技巧，喜歡的玩家記得點關注哦。
