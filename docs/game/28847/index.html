絕地求生：外國人眼中誰是最強中國選手，這三位讓世界聞風喪膽

2019-12-21 1,236條評論 4,146次閱讀 2726人點贊

絕地求生是一項非常受歡迎的電競運動，很多職業選手也是參加其中，目前也已經舉辦過各種世界級別的電競賽事，那麼中國選手在國外選手中是什麼樣子呢？讓我們來看看最讓外國選手害怕的中國選手。

4AM戰隊王哥

王哥的水平大家都清楚，是目前4am穩定的核心，今年無論是FGS倫敦賽，還是PGC全球總決賽，王哥都有非常嗨的表現，而在總決賽第一局，王哥一波一穿三的操作更是讓對面的解說感嘆ZGG is god.

​

4AM醒目

如果說王哥是4am的穩定核心，那麼醒目就是4am最讓人害怕的對手，因為醒目上限非常高，很多時候能夠完成別人無法完成的操作，在今年的FGS上，醒目靠著一雷四響以及獨狼吃雞讓那個無數國外的玩家記住了他，這個操作也是無數次登上各個節目的高光。

​

4AM永遠

4am可以說人人都是大腿，今年PGC上面的殺人王，只要有永遠在，就能保證4AM的殺人書，無論是小組賽還是總決賽，今年的永遠表現都讓人驚嘆。

​

OMG小獅子

今年表現最讓人留下印象的是小獅子，但是有一人讓外國人記了一年之久，這個人就是lionkk，作為去年PGI的總冠軍，小獅子靠著34個總擊殺冠絕群雄，直接把OMG拉上巔峰，雖然今年OMG未能參加總決賽，但是小獅子的實力沒有人會之弈。

​

當然國內還有很多實力強大的選手，但是想要讓外國人認識，只有在世界賽上打出成績才可以，很多隊伍也是到了國際賽場就不行了，不知道大家最期待誰的世界賽表現呢？
