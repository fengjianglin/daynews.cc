這些國產手游出海的套路，給國內玩家看笑了，老外卻愛吃這一套？

天天要聞
2019-12-07

摘要

說起中國遊戲在全球遊戲圈的地位，其實是一個極具爭議性的話題，在主機遊戲方面我們拿不出值得讓全球玩家看好的3A大作，而端游部分幾乎也是靠著大量的代理遊戲來填充國內市場，但是手游在這幾年的發展卻與它們有著很大的區別。

說起中國遊戲在全球遊戲圈的地位，其實是一個極具爭議性的話題，在主機遊戲方面我們拿不出值得讓全球玩家看好的3A大作，而端游部分幾乎也是靠著大量的代理遊戲來填充國內市場，但是手游在這幾年的發展卻與它們有著很大的區別。作為一種全新的遊戲形式，國內外的任何一家遊戲廠商都是同在一條起跑線上，但是國外遊戲排行榜從去年開始卻被大量”中國造”的手游佔據了。

在權威的數據分析機構SensorTower發布的一則關於國產手游收入的榜單中，小編髮現了一些有趣的現象。這則榜單當中排名前十的遊戲有超過一半是國產，而排名第一和第八位的兩款遊戲竟然出自國內非一線遊戲廠商。

就拿這款《萬國覺醒》來說可能在國內很多人都不知道它是一款什麼樣的遊戲，但是在國外卻能以收入第一的成績力壓了吸金能力爆炸的吃雞遊戲，那麼這些國產手游到底使出了哪些騷套路讓這些外國玩家瘋狂沉迷的呢？

首先，收入第一的《萬國覺醒》在設定與玩法上的變革絕對是”最騷”的操作，它作為一款SLG手游，卻與傳統的SLG遊戲有很大的區別，以往的遊戲當中對於勢力的歸屬感建立於你對這個遊戲的了解，但是在《萬國覺醒》當中將每個勢力的設定演變成了選擇國家，遊戲里有像是中國、韓國、法蘭西、羅馬等等國家，所以這就讓大量的外國玩家甚至都不需要去了解每個”勢力”的優勢是什麼，為自己國家而戰總是沒錯的。

同時在玩法上也做了很大的創新，光是地圖的一鍵縮放以及自由行軍就給《萬國覺醒》增加了類似於RTS遊戲的遊玩體驗，更不要說SLG本身就具有的發展勢力、攻打敵國了，而在《萬國覺醒》當中，往往一場戰役的參與人數都是以伺服器為單位的。

排名第九的《守望黎明》也非常有意思，相信即使你沒有玩過這款手游，但你一定看到它的短視頻廣告，而”日常追劇””我愛黎明”這些梗更是頻繁的出現在這個遊戲的視頻評論當中。

雖然遊戲是以末日為題材，但是它廣告創意的更新速度比遊戲版本更新速度還快，大量的廣告內容講述了在遭遇喪屍之後如何建立自己的家園，這些洗腦的廣告內容別說是國內玩家頂不住，”涉世未深”的國外玩家必然也是逃不開真香定律，所以國外視頻平台上更是有人製作了這些魔性廣告的合集，而這也絕對是這款遊戲能夠成功的關鍵”操作”。

最後，就是排名第八的《劍與遠征》，之所以要將這款遊戲放在最後說，那完全是因為作為放置卡牌類的遊戲老外一向都不買賬，但是這次《劍與遠征》的大獲成功絕非偶然。

要知道莉莉絲這家遊戲廠商，可能在國內名氣不如騰訊網易，但是他們出海手游的成績卻是十分搶眼的，光是出海手游收入排行榜就已經不是第一次上榜了，除了有《萬國覺醒》這樣吸金的大作，同時還敢將外國人”不感冒”的遊戲題材變得真香，這沒點硬實力還真不能拿出來隨便吹。

《劍與遠征》在開發班底上沿用了刀塔傳奇的原班人馬，而在遊戲畫風上採用了歐式手繪的卡通風格，不論是英雄角色的畫風，還是戰鬥場景的表現都帶有外國人最為喜歡的美術風格，而莉莉絲更將這樣一種美術風格作為文化進行輸出。

在國外不僅舉辦了各種COS大賽，就連玩家自己也自發產出了大量的同人的作品，而諸多的外國coser則更喜歡cos遊戲中諸多偏向暗黑系風格的英雄。

此外，最重要的放置玩法上讓眾多的玩家得到了充分的”護肝”時間，遊戲雖然以推圖為主，但是上陣的英雄只有五名，即使你肝錯了英雄也不需要從頭培養，只需要花很少的鑽石就可以重置英雄，而鑽石的獲取方式更是在遊戲的每個玩法當中都有，舉個最簡單的例子在”馬車”當中重置英雄需要花費20鑽石，但是你完成一個日常就可能會給你100鑽石，所以根本不用擔心肝錯英雄。

不過，雖然英雄不用太肝，但是每個英雄會有各自所屬的種族，而每個種族之間相互克制，所以這就考驗你在推圖過程中的陣容搭配以及站位設置。

總的來說，這些遊戲在國外大火沒有什麼固定的模式可以複製，每一款遊戲都有自己的套路，而這個套路與玩家興趣相碰撞之後，奇妙的”化學反應”就顯現出來了，不過讓小編感到高興的是這些遊戲在外國成功之後也都回國試水，不知道你有沒有玩過這當中的哪一款呢？我們一起來聊聊。
