這些老皮膚都有限定背景，還算不上強行限定

2019-12-25 1,385條評論 3,853次閱讀 1853人點贊

今天來說一些老限定，也是之前向小夥伴們徵集意見的時候，問的較多的一些老皮膚，這些老皮膚很多是限定，但很多小夥伴不知道為什麼這些皮膚是限定，今天給各位說一下這些限定的背景，像至死不渝這種皮膚就不用說了吧？帶字皮膚老皮專門做過一篇帖子，如果你想了解的話，可以去看看。

傳送門：聯盟中非常稀有的9款帶字皮膚，多數絕版，大佬的象徵

奧斯曼大帝泰達米爾

奧斯曼大帝普蘭克很有名吧？（強行限定搞事情的皮膚當然有名）但是蠻子也有一款奧斯曼大帝，這款皮膚標記TGA限定的狗牌，這是什麼鬼？是這樣的，在2013年11月份的時候，由TGA和LPL共同舉辦的英雄聯盟夏季賽季後賽，而購買了LPL職業聯賽總決賽門票內場票的，將會獲贈永久奧斯曼大帝泰達米爾這款皮膚，當時內場票為199元。而購買外場票的觀眾將獲得30天奧斯曼大帝泰達米爾這款皮膚，外場票當時是99元。這就是為什麼蠻子這款皮膚打上了TGA限定狗牌的原因。

舞會公主安妮，密林獵手提莫

這兩款都是TGC限定，那麼TGC是什麼呢？TGC就是騰訊遊戲嘉年華，安妮這款在往年的嘉年華活動中都能買到，在2011~2014年的TGC活動中，基本上都是購票送安妮的舞會公主，提莫的密林獵手也是同期開賣，價格比較低，舞會公主當時價格售賣是3900點券，而密林獵手提莫則是1900點券，兩款皮膚都有TGC的狗牌。

諸神的黃昏泰達米爾，外星裝甲庫奇

2011年有一個遊戲風雲榜，每位玩家每天可以投4票，當時英雄聯盟參與了三項大獎的角逐，如果獲得兩項大獎，那麼參與投票的玩家將獲贈「外星裝甲庫奇」，如果獲得三項大獎，那麼參與投票的玩家將獲贈「諸神的黃昏泰達米爾」，而在2011年，英雄聯盟成功獲得「十大最受歡迎網路遊戲」，「年度十大新銳網路遊戲」，「年度最佳休閑競技類網遊」三項大獎。

叢林俠盜 艾希

這款是英偉達顯卡專屬皮膚。2012年7月份的時候，LOL和英偉達公司合作推出了一款GeForceGT630影馳虎將D5顯卡。購買這款顯卡就會獲得一張特權卡，特權卡上有相應的CDK可以兌換這款皮膚。所以至今為止，這款皮膚的原畫上還帶著英偉達專屬的字樣

萬夫莫開 賈克斯

這是當時網吧比賽的獎勵，獲得省級以上冠軍的才會有這款皮膚，其餘的是期限皮膚，這款皮膚並非一些小夥伴所說的網吧特許可權定，這個和網吧特許可權定不同，網吧特許可權定當前國服也有幾款，比如野獸獵人德萊文，瑟庄妮，泰達米爾這三款還有網吧特許可權定狗牌。

以上是一些小夥伴問的關於一些老皮膚限定背景的問題，如果還有的話，你可以告訴老皮，老皮統計之後再給各位說說。
