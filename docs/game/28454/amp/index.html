王者榮耀：冬冠選拔賽第2日結束，TS，eStar和XQ的戰績都是一負

天天要聞
2019-12-23

王者榮耀冬季冠軍杯的選拔賽繼續進行中，之前也給大家發了第2天的前幾局戰報，這裡我們再來看看後幾局的吧，其中有幾把都是KPL賽區的內戰，可以說在很大程度上決定了出線權的歸屬。

先是Hero久競對戰RW俠，雖然RW俠拿到了第一條暴君，不過久酷這邊也拿到雙殺，還破掉了RW俠下路又拿先知主宰，RW俠則是在邊路gank抓人後推進，中期塵夏的豬八戒被抓了2次有點傷，RW俠在拉扯中拿下黑暗暴君，之後又0換3拿下主宰，慢慢有了優勢，最後一波，趙雲的大招完美，扎中Hero久競核心，打出0換3後1波推掉水晶獲勝。

然後是TS和XQ的頭名之爭，雙方之前都是沒輸過，XQ前期打得不錯，馬超拿到雙殺，並在之後的團戰中連連獲勝，中期TS的團戰更強，張飛上路一波開團打出0換2推掉上塔，之後的幾波團戰也都是TS獲勝，最後一波，暖陽蘭陵王勾引成功，空血跑掉的同時也給關羽發揮的空間，神人拿下三殺後團滅對手，TS笑到了最後。

Hero久競和Nova的一局懸念不大，雖然Nova反藍成功，但Hero久競一波反打拿到4個人頭，之後各種入侵拿暴君和人頭，久誠幹將的輸出非常高，中路連推兩塔，之後孫尚香發育起來，拿下主宰暴君後推掉上路高地，最後一波團戰，又是孫尚香打出極高輸出，拿到三殺後團滅對手，直接點爆水晶獲勝。

TS和eStar這局也非常關鍵，如果TS獲勝，他們就基本鎖定了晉級名額，開局eStarPro的花海切入完美，打出1換3後又繼續推進，TS反打一波1換2扳回一些，中期雙方各種拉扯，但基本都是均勢，最後的一波團戰，諾言的關羽直接1推3，打亂TS陣型，隊友配合打出連環控制效果，eStarPro0換5團滅對手，成功拿下了這局的勝利。

第2天的比賽也是全部打完了，從積分以及隊伍之後要對陣的情況來看，TS晉級的希望是最大的，因為他們戰勝了XQ，之後對戰的2支隊伍分別是Hero和已經被淘汰的NOVA，都很有可能獲勝，而XQ和eStar一戰很可能決定另外一支晉級的隊伍，你們看好誰呢？歡迎大家關注和分享，遊人將繼續給大家帶來王者榮耀冬季冠軍杯比賽的最新消息。
