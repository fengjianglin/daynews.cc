三國殺：被兵了也不用怕，他們沒有摸牌階段都能打

2019-12-25 1,545條評論 3,667次閱讀 1841人點贊

在三國殺中，摸牌階段無疑是一個很重要的階段，對於大多數武將來說沒有摸牌階段給予的牌就沒有其他的牌源，也就成為了一個廢物，可想而知摸牌階段是多麼重要。在遊戲中我們最痛恨的兩張錦囊牌應該就是兵糧寸斷和樂不思蜀吧，那麼今天小編就要給大家帶來幾位不依靠摸牌階段來獲取手牌的幾個武將，讓你再也不用怕兵糧寸斷！

他們沒有摸牌階段都能打!

TOP1:曹植

那麼曹植為什麼可以不依靠摸牌階段就能夠得到牌呢？這一切都要歸功於曹植的技能—-落英：當其他角色的梅花牌因棄置或判定而置入棄牌堆後，你可以獲得其中的任意張牌。這個技能就厲害了，不管是裝備還是錦囊還是基本牌曹植都是可以很輕鬆的收入囊中的。

這個技能使得曹植能夠搭配的武將可就太多了，經常需要判定的張角夏侯，愛制衡的孫十萬等等都可以為曹植帶來大量的回合外摸牌，玩曹植的話最好還是刷一下AK比較穩，大多的梅花還是殺比較多一點，落英唯一的缺點就是穩定性不足。

TOP2:張星彩

星彩是小編很喜歡的一個武將，因為她不僅有著極高的過排量，更是有著回合內強大的爆發能力，直接秒人的快感是很多武將所不能比擬的。星彩的甚賢效果是回合外每次有角色棄置基本牌那麼你就可以摸一張牌。首先裝備和錦囊沒多少人捨得去棄牌，那麼棄掉的都是什麼牌呢？基本牌，這就奠定了星彩強大的過牌能力，可以說只要有人棄牌星彩就有的摸。

這比起曹植的落英可以說是穩定了無數倍的，另外別人被拆或者在別人的回合換掉基本牌你也是可以摸牌的，可以理解為只要在回合外有基本牌進入棄牌堆你就可以摸一牌，星彩和曹植都有著兩個非常適合搭配的武將，他們就是:孫十萬&王基，孫十萬每輪穩定1-2牌，王基更是沒上限，手氣好的王基可以讓星彩一回合摸不少牌。

TOP3:張昭張紘

二張作為三國殺的元老級武將更是有著反賊頭子的美譽，其強度無疑是可圈可點的，固政不但能夠為隊友提供一個手牌上限+1的效果，更是能在敵方被樂大棄牌的時候拿到很多牌。摸到的裝備牌還能夠給隊友裝上順帶給自己過牌，可以說是一個集爆發、輔助、過牌這三點於一起的武將，這也是他為什麼能夠從三國殺到現在還是一個炙手可熱的武將的重要原因。

TOP4:徐氏

相信每個徐氏玩家在遊戲中都曾經說過這樣一句話：”算卦嗎？”或者是在隊友忘記的情況下提醒隊友一定要算卦，問卦作為徐氏的核心技能能夠在回合外給徐氏提供很大的過牌，徐氏更是東吳繼二張之後新一代的反賊頭子，並且問卦這個技能對於玩家的要求也是遊戲中難度排在前3的技能之一。

原因就在徐氏的問卦不但能夠給自己和隊友提供過牌，更是一個可以操控牌堆的強力技能，你可以用它控你對於下家的一半牌，也可以把一些關鍵牌藏起來交給隊友，還有什麼藏牌方式是比藏在牌堆里更為穩妥的呢？更可以把白銀獅子變成每輪穩定所有友方回1血作用，一個識別敵我的桃園結義有多厲害就不用多說了吧！
