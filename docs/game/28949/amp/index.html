暴雪2019年有多慘？一個操作少賺81億元，魔獸懷舊服都救不回來

天天要聞
2019-12-20

大家好，這裡是正驚遊戲，我是你們的正驚小弟。

隨著2020年腳步的接近，各大遊戲廠商也到了年底「沖業績」分紅的關鍵性時刻，在2019年中即將被迪斯尼收購的全球知名CG動畫製作大廠暴雪，也終於從2018年股價暴跌的困境中走了出來，憑藉著魔獸世界懷舊服、酒館戰棋、暗黑破壞神4等一些系列新作，又一次成為了玩家心目中最愛的「爸爸」。

還債的一年：三季度凈營收47.33億美元

對於暴雪來說2019年無疑用來還債的一年，由於2018年暴雪嘉年華畫餅大會的失敗，加上動視方面《使命召喚15：黑色行動4》的熱度下降、《命運2》的分家，使得動視暴雪凈營收的情況一直處於一個低谷期。

第一財季實現凈營收18.25億美元，同比下降了11%；第二財季實現凈營收13.96億美元，同比下降了15%；第三財季實現凈營收為12.82億美元，同比下降了22%；第四季度預期凈營收18.12億美元。

也就是說暴雪這一年預估凈營收將會達到65.45億美元，同比下降15%左右。比去年下降了大約11.55億美元（約合81.25億人民幣），成為了業內名副其實的「撒幣」公司。

新里程碑：牌面！中國女學霸豪取世界冠軍

要說2019年最有牌面的女選手，無疑是咱們人美才華高的Liooon獅醬小姐姐了。直接在暴雪嘉年華《爐石傳說》特級大師賽全球總決賽的舞台上，以三比零的絕對優勢贏下對手Bloodyface，拿下爐石個人賽的總冠軍，成為了暴雪史上爐石項目的首個女性冠軍。

之後獅醬就讀於西南政法大學的學霸身份也被網友們扒出，而中國女學霸豪取世界冠軍這件事也受到了多家官媒的點贊，成為電競界的顏值與牌面擔當！

新風潮：魔獸世界懷舊服開啟！

在《魔獸世界懷舊服》開啟前，「暴黑」們都在調侃著迪斯尼要抄底暴雪股價收購公司的事。但在《魔獸世界懷舊服》開啟後，暴雪就成為了老玩家口中的世界主宰，甚至要反收購迪斯尼了！暴雪在第三季財報中指出，第三季度暴雪娛樂旗下遊戲每月有3300萬活躍用戶，其中《魔獸世界：經典懷舊服》的發布推動了《魔獸世界》刷新史上季度訂閱量的最大增長紀錄。

而隨著《魔獸世界懷舊服》的興起，各大遊戲廠商也將目光聚集到自己旗下經典的IP作品上，紛紛打起了懷舊炒冷飯。客觀來說，《魔獸世界懷舊服》正式開啟了遊戲界懷舊的新風潮（卡普空哭了）。

新辭彙：毛人風

隨著《魔獸世界懷舊服》的興起，這群不安分的老玩家們也創造了一個讓聯盟與部落攜手合作的新辭彙——毛人風。

該事件起源於魔獸世界老牌主播「狂人與風」，當著直播間一眾水友的面，開著武器戰的角色毛了屬於MT的半張風臉、薩鐵以及火炕鞋，直接導致MT須彌坨退會，並引起了眾多魔獸玩家的憤怒。部落玩家大開奧格之門，只為提供聯盟玩家擊殺「狂人與風」的機會。之後魔獸玩家也將遊戲中所發生的毛人毛事，統稱為「毛人風」。

最勵志的戰隊：上海龍之隊

一支42連敗的戰隊在新賽季中連斬豪門強隊，並在聯賽第三階段中登頂奪冠，這是連電競網文小說都不敢亂編的「中二開掛」劇情，但卻真實地發生在暴雪所舉辦的《守望先鋒》全球聯賽的舞台上！

作為《守望先鋒》聯賽歷史上首支建隊的職業戰隊，上海龍之隊從一開始的中國牌面，直接跌落了兩個賽季42連敗的低谷，淪為全世界玩家的笑柄。好在俱樂部老闆在最艱難的時刻並沒有選擇撤資，反而加大了投資力度來引進更出色的教練與職業選手，最終沖雲破霧登上了世界之巔。

連「上海蟲」都能變成上海龍，各位小夥伴你們在新的2020年實現小目標的旅途中，又有什麼可猶豫與害怕的呢？

最成功的虛假營銷：War3重製版

當然暴雪在2019年的黑歷史還是存在了一些，其中備受玩家爭議的就是《魔獸爭霸3：高清重製版》的虛假營銷問題了。在2018年的暴雪嘉年華上官方所承諾的五大全新內容：全新劇情、電影視角的鏡頭呈現戰役過場新動畫、新配音、新UI以及老角色的新故事，在今年開啟B測《魔獸爭霸3：高清重製版》中統統都不見了蹤影。

小弟曾一度懷疑是黃旭東這個「老賊」奶翻了《魔獸爭霸3：高清重製版》，當然也許在最終品質的遊戲公測時，這些內容會被暴雪呈現在玩家的面前。

最香的暴雪嘉年華：2020年的暴雪未來可期

最後小弟也來帶大家回顧下今年暴雪嘉年華中畫出的大餅，展望下2020年的暴雪。《魔獸世界9.0》的希女王手撕了巫妖王的統御之盔，艾澤拉斯與暗影界逐步融合，腳男們將會解鎖更為強大的盟約技能。

《爐石傳說》的酒館戰棋模式已經在自走棋市場收穫了大量的用戶，而新版本「巨龍降臨」更是一口氣發布了N多張超模神卡，爐石棄坑用戶大量迴流。《暗黑破壞神4》與《守望先鋒2》兩款新游即將面世，下一年的暴雪崛起在望！

小弟有話說：縱觀2019年，暴雪公司的財政營收狀況真可謂是經歷了大起大落，而他們對於移動端遊戲的嘗試也隨著暴跌的股價而暫時擱置。在找准自己PC端的用戶定位後，暴雪用實際的用戶增長數據與口碑，向全世界證明了自己寶刀不老的實力：你大爺依舊是你大爺。不知道各位小夥伴，你們在2019年內是否也像暴雪一樣找准自己的人生定位了嗎？

一個正驚的問題：暴雪會在2020年崛起嗎？
