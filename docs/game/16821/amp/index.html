新英雄呂蒙技能曝光！體服嬴政四款皮膚翻新特效出爐

天天要聞
2 天 ago

2019年KPL秋季賽已經正式結束，首先讓我們恭喜AG超玩會拿下2019年KPL秋季賽冠軍，正式服也將在下周開啟冠軍陣容折扣活動，活動內容就是大概就是冠軍陣容和皮膚八折上架。

下面是一些皮膚的近期爆料相關整理，跟大家彙報一下！

貂蟬的FMVP皮膚

有關貂蟬的FMVP皮膚新的爆料，這款皮膚應該要到月底甚至1月中旬的左右了，官方貌似又在重新調整，同時有消息說首周將最低962點券，喜歡貂蟬的朋友記得準備好錢包！

榮耀典藏

昨天曝料10個英雄中，可能包含一個榮耀典藏。現在來看大概率是李白，不過李白那個之前很早加入文件了，所以可能是有個初步計劃做完了，具體如何還得再觀察！

SNK英雄新皮膚

預計將在春節後陸續推出，其中娜可露露新皮膚可能會先上線，預計將在3月份推出。

春節皮膚

除了五嶽系列那些，春節差不多就是S18新賽季了，所以在春節期間還有S18兩款戰令皮膚，而且那時候大概率會公告榮耀典藏新皮膚和情人節限定新皮膚。

皮膚返場

在1月24日會有一款限定皮膚返場，李白鳳求凰的返場概率極高，同時該皮膚的優化也將上線。

有關呂蒙的爆料

之前比較早的三國版本時就已經爆料過呂蒙這個英雄，不過後來一點消息都沒了，但是昨天網上突然就曝出呂蒙相關的技能表，都是比較詳細的技能，下面一起來看看吧！

按照這份技能表效果：

呂蒙被動，疊加五層印記後可以對敵方造成斬殺傷害，同時可以對非英雄單位攻擊產生恢復效果。

呂蒙一技能遁入地面，二段釋放可以造成位移，下潛狀態在野區和河道能獲得加速。

呂蒙二技能當前未知，看起來應該是身周範圍攻擊，需要在地面狀態釋放。

大招參考夏侯惇大招，丟出匕首後需要靠近一定距離後觸發二段的位移攻擊。

繼之前原皮膚全面翻新後，在體驗服嬴政四款皮膚技能優化特效也已出爐了，整體效果如下（以下圖源博主：@濃度）：

  • 暗夜貴公子
  • 搖滾巨星
  • 優雅戀人
  • 白晝王子
