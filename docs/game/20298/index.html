教練Clearlove上線，這或許就是廠長最溫柔的“告別方式”

2019-12-17 1,798條評論 3,305次閱讀 668人點贊

對於LPL觀眾而言，2019是告別的一年。

香鍋、Letme、Imp、Mata、Pawn、姿態，以及那個LPL觀眾最最最不想聽到的名字，Clearlove。

廠長正式轉型教練

關於廠長以夢為馬，砥礪前行的故事，各位屏幕前的觀眾已經很清楚了。身為LPL 001號選手，本以為廠長的退役即使不是空前盛大，至少也應該是滿含淚點。然而作為曾經世上最好的打野，霸氣了整個職業生涯的廠長，卻選擇了最溫柔的「告別方式」。

眾所周知，廠長職業生涯的個人最高光時刻是2015年MSI，在終結Faker不敗妖姬的同時，也成為了MSI史上第一位MVP。雖然S5充滿遺憾，可從後續的故事來看，如果廠長在2015年選擇退役，那其本人必將避免許多黑點。4396、反向突圍八強、評價有無Pawn都一樣等等。

廠長背影

同樣，如果廠長在S5就選擇退隱江湖，那其ID的耀眼程度，也將少了些許時間的鍍金。

如果一名選手，明明沒有取得比巔峰時期更優異的成績，但還是能越來越受玩家追捧，除了該選手上輩子為玩家擋過子彈以外，最合理的解釋或許就是這個人的身上，代表了一個時代。

老WE

自姿態宣布退役後，S2紅黑時期的選手就只剩下廠長一人，廠長的轉型自然也宣告了那個時代的徹底結束。對於新玩家而言可能無關痛癢，但凡是經歷過S2的玩家，不管是老WE的粉絲還是老IG的粉絲，哪怕是之前廠長的理智黑粉，都難免會對這個人的退役離去，感到捨不得。

這就是電競賽事與傳統體育相通的地方，比如籃球領域的「96黃金一代」，無論球迷之前有多痛恨科比，但當這位96黃金一代的最後成員宣布退役時，球迷還是有些難以釋懷。殊不知球迷對於科比表達的敬意，實則也是對那個時代，自己青春的一次告別與和解。

科比退役現場

當然科比與廠長身上的相似度也比並不止這一點。

從性格上來說，科比偏執，廠長也十分堅韌。最大的區別在於前者通常不會在大賽上掉鏈子，後者在這點上不夠穩定。在各自領域中，二人都曾有過絕對的統治，且面前始終都有一座大山橫亘在那裡，科比面對的是喬丹，廠長想要翻越的則是Faker。

Faker

科比也好，廠長也罷，在職業生涯的後期，除了某幾場的高光表現，人們很難再從他們的身上發掘出亮眼的成績。甚至連正常的上場比賽，都成為了一種奢侈。但即便如此，只要登場，就是一偏歡呼。因為對於S2走過來的玩家而言，廠長身上所承載的，並不只是個人光環，而是所有那個時代玩家的情感寄託。

廠長「本命」英雄豬妹

幸運的是，與香鍋等人不同，廠長選擇的是一種最溫柔，也是最能為玩家所接受的「告別方式」。即便這樣的結果依舊苦澀，但有望看到其以教練的身份繼續征戰賽場，甚至完成夢想，已算是「豬崽兒們」的幸運。

鏡頭前的廠長已經即將27歲，而我們眼中卻滿是他17歲的樣子。
