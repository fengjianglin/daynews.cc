蝗家鑒賞團《貓貓突擊隊》評測：化身可愛貓貓，打倒敵人救貓村

天天要聞
2019-12-18

文丨被卡住的方臉貓呀@bigfun社區

在一顆星球上有座溫馨的貓村，貓貓們在這裡快樂的生活著，但是突然有一天，出現了一股邪惡的勢力，使貓村陷於水生火熱當中，你是否願意化身為一隻勇敢可愛的貓貓，加入貓貓突擊隊，與敵人來一場廝殺。

《貓貓突擊隊》是一款休閑向的冒險類遊戲，有著像素風格的畫面、清新的配色和輕快的BGM，給玩家一种放松的感覺。在遊戲內容中有著大量不同的可愛的貓貓可供玩家收集，每一隻貓貓都有著自己的風格，比如騎士、巫師和盜賊等。有一些貓貓的裝扮也致敬了很多經典，比如喵里奧、喵易基和有頭騎士等，每一隻都會讓我們很自然的想到一些「大人物」。

貓貓們可以通過做任務和召喚等方式獲取。在遊戲的開局會提供「刺客」「戰士」和「術士」三隻貓貓拱玩家選擇，玩家可以隨意憑著自己的喜好去挑選，因為遊戲機制的設計問題，並不會對遊戲進程造成大的影響。

遊戲中的有一個專門貓貓準備的地方——貓窩，收集到的貓貓都可以在這裡看到，玩家可以用手指點擊屏幕去擼他們，看看他們會說出什麼有趣的話，或是有沒有感染疾病。貓窩每隔一段時間會刷出獎勵，記得定時領取，當然別忘了鏟貓屎。

貓貓的會感染的疾病，豐富多樣，比如痔瘡、腳氣和感冒等。疾病有好處也有壞處，比如提高速度，降低暴擊。每一隻貓貓的身體都攜帶著多種疾病，感覺作者好變態，搞得和貓貓不生病就不會有攻擊力樣。

遊戲內的玩法很多，其中主要的玩法就是將自己收集的貓貓們組成四隻一支的隊伍，去村外冒險，打倒怪物們，獲得可以給貓貓們升級的素材。每一種隊伍的組合因為貓貓們不同的技能，都會打出不同的效果。一些貓貓們也有著羈絆，將他們放在隊伍中，隊伍會更加的強大。

貓貓們的戰鬥不需要的大量的操作，只需要點點屏幕下方的技能，或是開啟自動戰鬥模式，看著貓貓們與怪物或是被迷惑的貓貓們搏殺，從激烈的「喵喵」聲，感受戰鬥的激烈。這樣的操作設計減少了玩家的遊玩負擔，使遊戲更加的簡便，但同時也減少遊戲與玩家之間的了互動性。

貓最喜歡的食物是什麼，答案肯定是魚沒錯了， 作為一個充滿貓的遊戲，怎麼能沒有魚的存在呢？遊戲內有一個深受大家喜歡的玩法，為自己的貓貓下河捕魚，在擁有魚餌道具後，就可以前往魚塘捕魚了，在規定時間內通過水麵漣漪的大小推斷出魚的大小，再以迅雷不及眼耳之勢點擊屏幕，將魚撈出。

不同的魚給貓貓吃下去後有不同的屬性加成效果，魚的大小也決定著加成的多少。因為魚餌的數量需要靠時間恢復，所以抓魚非常有考驗，靠有限的魚餌，獲得最好的效果。

黑市是遊戲內玩家重要的收入來源之一，打敗敵人會有幾率掉落寶物，這些寶物可以拿去鑒定，如果鑒定成功，可以換取大量的寶石。如果鑒定失敗，則會被打入監地牢。

被打入地牢的玩家，出獄只有三種方式，第一種是等待刑滿釋放，往往要等上五分鐘左右，非常浪費時間。第二種是花上幾千金幣，賄賂獄霸，浪費錢。第三種是打倒獄霸越獄，這一種對於處在前中期的玩家很不友好，因為所謂的獄霸也是玩家，並且是這個遊戲伺服器中戰力頂尖的玩家，也就是服霸，很難被打敗，為了節省時間，只能選擇賄賂。

第三種設定，是我認為遊戲中最爛的地方，在這種沒有什麼操作，僅靠戰力的遊戲內，這樣做對於新手極不友好，服霸也會越來越強，簡直是勸退。

黑心旅社是遊戲中最設計最有意思的玩法，村長的侄子靠關係開的旅社，1000金幣一張的車票，還不定有座位，想要座位就要把其他玩家從車上踢下去。10分鐘的車程中沒有美景，只有每2分鐘一次原價100寶石現價10寶石的大促銷。買到啥全靠運氣，可能是一隻貓貓的碎片，也可能只是條小魚。

玩過一段時間後，感覺上遊戲的機制設計雖然有些問題，但其內容豐富，玩法眾多，貓貓的設計也很可愛，又是偏休閑向的冒險遊戲，每一個戰鬥回合都不會太長，遊戲界面的設計可以讓玩家將手機放在一邊，單手操作，而剛好你又喜歡擼貓，這會是一款很好的消磨碎片時間的遊戲。
