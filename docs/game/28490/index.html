DNF：挑戰書漲了！12月商人積極接盤，拍賣行少了200頁

2019-12-23 1,549條評論 3,303次閱讀 1508人點贊

100級的彩虹深淵才曝光不久，不少玩家覺得彷彿回到了恍惚套的時期，但深淵的確比團本有趣，這已經是沒有爭議的了，所以玩家們的目光開始聚焦門票，本來大家攢史詩靈魂和深淵邀請函是一樣的，但國服減負過！團本攻堅隊的比例不一樣，所以不渝之詠換邀請函最划算，這也讓不少商人激動，什麼？以為100級要囤深淵票！於是，過氣了好幾年的挑戰書竟然漲了。

挑戰書漲了！商人積極接盤，12月拍賣行少了兩百頁

他們以為深淵票和挑戰書一樣的，因為拍賣行的挑戰書漲了不少，就拿跨3B這個跨區來說，在100級彩虹深淵更新前，一張也就2000多金幣，但不到一周的時間裡，已經漲到了3700！有一定道理，因為100級深淵出的史詩屬性很好，玩家們就算有95B也升級不到。

於是商人們積極接盤！畢竟一年了都沒啥材料可以囤，跨3B跨區雖然不大，但因為GZS的存在，挑戰書常駐頁數基本也有200~300頁，可如今再看，僅有31頁，而且要想看到數量上千的挑戰書，還得再往後數10多頁，明顯，這是商人在大量掃貨了。

在100級，挑戰書真的有潛力嗎？

如果是90版本，肯定有！但100級完全不看好，如今多數魔盒產物、裝扮、卡片、換裝武器等太多內容都已經實裝1次交易了，出增幅活動還專門出10.5W的矛盾，大家覺得商人還有機會嗎？這是大勢所趨！

按照挑戰書3700來算，100級一次彩虹深淵成本是130W，按照星空原石3.6W，也是130W（商人還挺會囤），這個價格對於土豪來說，並不高！但商人卻沒考慮到，平民玩家手中海量的史詩靈魂。

如果100級彩虹深淵需要130W刷一次（18個引導石），那麼按照1個SS靈魂換個4個時間的結晶，時間的結晶=可交易的引導石，這就意味著1個SS靈魂可以值30W，相比90級直接漲了4倍！別說了，我這上千個SS靈魂就等著換春節禮包了，更給力的是時間結晶沒有兌換上限，1天就能換出幾千個！

注意，100級普通玩家對於SS靈魂基本無需求！無論是升級還是刻印和繼承都不消耗，跨界從來不敢想，如果1個值30W，平民捨不得自己刷的，全拿去換金幣了，所以說商人囤挑戰書，是哪個師傅讓接的盤啊！

更何況，史派克還有一招，100級魔盒加入彩虹深淵通行證，小禮包也是如此，商人們，時代變了……

不過，如果彩虹深淵真的火，對於平民來說的確不錯，起碼SS靈魂能換更多的金幣，至於自己刷深淵，100級幾乎所有副本都掉無法交易的引導石。小夥伴們，覺得商人這波囤挑戰書有機會嗎？
