夢三國於禁大招完美卡點，塔下成功絲血三殺

2019-12-23 1,102條評論 4,214次閱讀 2316人點贊

說起于禁，這個英雄大家絕對不陌生，這個英雄也算是老牌英雄了，技能方面，相較於過去有了一些小小的改動，但是整體都是差不多的，沒有什麼大範圍的改動，技能也是原來那樣，沒有進行根本上的重做，不過，這個還是很強勢的，身為力量型的法師，除了前期藍不夠以外，其他方面，還真找不出什麼比較明顯的弱點！最後不得不說，珍愛生命，遠離於禁！

于禁！

于禁在劣勢路單人一打二，還是很吃力的，基本都被壓在塔下了，這種情況很是危險，兵線稍微清的慢一點，對方就有機會進行強打，于禁被逼到極限，只能放手一搏，立刻反跳成功大陸一波糜竺，將糜竺扔到了塔下！

于禁！

糜竺躲到樹林里，放了大招，周倉也過來進行圍剿，此時的于禁是朝不保夕。就在這個關鍵時刻，于禁一個大招擊殺了劉表，這個大招也正好擋住了糜竺大招的傷害，以及周倉的技能效果和傷害，這波操作實在是亮眼，故事到這裡，其實于禁已經不虧了，劣勢路，一打三，能換走一個，你還要什麼自行車？然而故事沒有那麼簡單！

三殺！

之後的周倉被于禁成功的留在了塔下，于禁利用防禦塔的傷害，成功打了一波雙殺，而此時的糜竺已經是強弩之末，沒有技能，又被堵在樹林里，于禁的隊友也傳送下來，肯定是跑不掉的，于禁最後一個技能，成功收割，拿下了一波三殺！
