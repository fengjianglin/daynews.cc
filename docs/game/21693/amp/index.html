茶隊1-5，LGD 0-4，中國隊戰績慘不忍睹

天天要聞
2019-12-18

新加坡邀請賽第一天比賽已經全部結束了，如果你是西恩刀塔的粉絲，今天又看了一整天比賽的話，那血壓拉滿就不只是說說而已，大家趕緊躺床上緩緩休息一下，免得明天看比賽看的健康出問題了。

這次新加坡邀請賽很多強隊參加我們都知道了，中國隊TI結束到過年這段時間無心戀戰我們也知道了，但大家仍然對這次比賽有所期待，期待著能有哪支中國隊脫穎而出，將這個分量並不輕的冠軍獎盃給帶回國來。

然而比賽僅打了小組賽一天，粉絲們的心就涼了一半。

來看看這一天下來中國隊的戰績吧，A組裡茶隊0勝1平2負拿到1分，排在小組倒數第一；

B組這邊也沒好到哪去，VG 1勝1負拿到2分，和NaVi戰隊並列第三，而LGD 目前為止一場沒贏，和液體並列B組倒數第一。

因為這個比賽是BO2，如果把每個小分都算入戰績的話，那麼這三支中國隊的戰況分別是：

VG 2勝2負

Aster 1勝5負

LGD 0勝4負

除了VG戰隊還算差強人意外，茶隊和LGD這打的都是啥啊，尤其是賽前被寄予厚望的LGD被打了個0-4，面對NaVi和EG都毫無還手之力，這可太讓粉絲們失望了。

LGD這次比賽中展現出來的狀態就不對，除了Ame一如既往的穩定外，May皇好像迷失了自己。對陣NaVi兩場比賽中Maybe選擇了一把大屁股一把PUCK，兩把用一個字就可以解釋他的表現——劃。

當雙子星中其中一個沒了聲音後，LGD想贏比賽真的太難了。現在LGD的比賽就是全場看著Ame在地圖上一個打五個，其他人全都在跑，這還打個鎚子。

希望LGD和Maybe能趕緊調整好狀態，別的不說，起碼從小組賽出線吧，這要淹死了也太丟人了。

從這次新加坡參賽隊伍來看，趕緊中國隊不管和誰打都有一戰之力，然而真等打起來卻發現一個都打不過。現在LGD和茶隊已經處在被淘汰的邊緣，VG也是掙扎在被淘汰賽和進敗者組之間，希望今晚睡一覺後，明天三支隊伍都能找回點狀態，畢竟我們不想還沒進淘汰賽所有中國隊就全回家了。。

明天小組賽還將繼續，茶隊還有兩個對手分別是J.Storm、Secret，VG還有三個對手分別是Gambit、LGD和NaVi，LGD同樣還有三輪要打，對手分別是液體、VG和紅星。
