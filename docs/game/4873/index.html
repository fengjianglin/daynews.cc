索尼PS系列獲認吉尼斯最暢銷家用遊戲機 網友：該打折慶祝

2019-12-04 0條評論 18次閱讀 0人點贊

摘要

來源：海外網圖源/共同社海外網12月4日電索尼互動娛樂公司（SIE）3日發布消息稱，PlayStation（PS）被吉尼斯世界紀錄認證為“史上最暢銷放置型家用電視遊戲機”。PS從1994年12月第一代開始發售，至今已有25年，截至11月7日，4代PS共累計銷售4.5019億台。

來源：海外網

圖源/共同社

海外網12月4日電索尼互動娛樂公司（SIE）3日發布消息稱，PlayStation（PS）被吉尼斯世界紀錄認證為「史上最暢銷放置型家用電視遊戲機」。PS從1994年12月第一代開始發售，至今已有25年，截至11月7日，4代PS共累計銷售4.5019億台。

據日本共同社消息，第一代PS發售時，正值日本任天堂公司「Super Famicom（超級任天堂，任天堂開發的一種家用遊戲機）」全盛期，但PS支持3D影像，還不斷推出人氣遊戲，一躍成為遊戲機界主角。第一代PS開售當天在日本賣了10萬台，創下歷史記錄。第二代PS於2000年3月投放市場，銷售了1.5億多台，創歷代銷售最多記錄。

截至今年9月，按銷售台數看，賣得最好的是PS2和PS4，分別為1.5億台和1.028億台。

PS3於2006年11月發售，支持任何形式的藍光碟。而PS4能夠提供以固定金額玩在線遊戲的會員服務，目前已有3690萬人註冊，這也開啟了家用遊戲機銷售後也能賺取穩定收益的商業模式。2020年年底PS5將面世。

圖源/朝日電視台

3日，PS之父、索尼前副社長久多良木健在東京舉行的認證儀式上說：「25年前我們曾提出『爭取銷售100萬台』的長遠目標。那種振奮感，現在令人懷念。今後也希望見證PS的巨大飛躍。」

PlayStation是日本索尼公司的著名遊戲機系列，誕生於1994年。根據第三方統計數據，截止到2019年11月7日，PS、PS2、PS3和PS4的合計銷售量為4億5019萬台。

微博網友留言截圖

消息一經報道，引起中日兩國的「遊戲發燒友」熱評。有中國網友留言稱：「很高興成為4.5億中的一員，讓我們共同見證PS的發展壯大」、「索尼大法好」、「一晃就是25年，我貢獻過5台」。日本網友也留言感慨道：「出PS1時我還是中學生，很高興體驗了遊戲機的黃金時代」、「還記得PS廣告，感受到了遊戲機的變革」、「是不是該打折慶祝一下？」（海外網 王珊寧）
