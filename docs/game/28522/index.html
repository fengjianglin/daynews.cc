魔獸世界：飛行坐騎和職業平衡，哪一項設計導致了遊戲的沒落？

2019-12-23 1,499條評論 4,158次閱讀 2851人點贊

魔獸世界時代、環境還有風氣都不一樣了，早些年那會沒有其它遊戲的衝擊，也沒有快餐文化的盛行，大家都能耐下心的去完成任務去下副本。但在現在這種大環境下，懷舊服去體驗去回憶的人應該不少，但是能真正堅持下去的人有多少就不好說了。或許這就是回憶吧，其實遊戲好不好玩已經不重要了，關鍵還是能勾起以前的回憶，就像很多00後不懂一些80為啥喜歡玩傳奇一樣，大家都擁有這美好的回憶。

​

當年西瘟疫之地偶遇一個小母牛戰士在和精英苦戰，作為盜賊的我明明可以熟視無睹，但心中總有一個聲音在說”盜賊要去扶一把”，後來我們成為了遊戲里很好的朋友。經歷了很多個版本，有過很多美好的回憶。飛行坐騎一出野外互動直接歸零了，其實之所以叫世界就是當時生機勃勃的野外，總能遇到各種玩家，練級的跑路的釣魚的挖礦的打劫的，一路走來和玩家之間的故事，要比任務列表裡豐富上千倍，這是遊戲再豐富也做不到的事呢。

​

現在魔獸都是臉滾鍵盤流了，有什麼好玩的。是個怪就能單挑，一挑N不成問題，一點難度都沒有。打副本要按照固定的套路，外加隨機應變。現在打副本一律都是a到底，恨不得連boss一起a。要知道在60年代，操作好壞可是直接區分了高玩和萌新，更不用提新版本有很多致命bug。老版雖然有bug，但是不會那麼致命。不會出現坐飛機飛著飛著就摔死的事情，還摔在某出不來的山溝溝里。

​

我覺得wow後期的平衡有點鑽牛角尖了，為了平衡把所有職業所有種族都搞成一樣了，給人的感覺就像是大家都玩的一個角色不過是換個皮膚罷了。本來是個奇幻背景的角色扮演遊戲，不是即時戰略也不是MOBA，職業平衡是重要的一點，但絕對不是核心，角色扮演遊戲的平衡怎麼能只是考慮PvP呢，練級難度，團隊地位，生活職業，操作難度不是都要考慮的么。

​

競技場固然重要，但是如果只考慮競技場，那還不如直接出個對戰遊戲了。經典不是玩的時間長短決定的，即使沒時間，偶爾上去玩十幾二十分鐘還是很開心，這就是經典。但80後的版本連上線的慾望都沒有了，當初萬年TBC的時候，就爽CD打完了也會每天上線和朋友聊天看風景。
