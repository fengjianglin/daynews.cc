《東方紅輝心》這次輪到紅魔館解決異變了

2019-12-10 901條評論 1,889次閱讀 938人點贊

前言

漢化語錄鑒賞家，NoraNeko評測

遊戲介紹

《東方紅輝心》是一款東方同人ARPG遊戲，講述紅魔館勢力追尋傳聞中的怪獸的歷程。本作可選用大小姐或Pad長作為自機，在動作元素中也加入了東方傳統彈幕要素，在高練度和裝備的加持下可盡享爽快割草體驗，Bullet Hell模式也給了玩家挑戰自我的機會。遊戲的音畫質量出色，角色間互動有愛，對於圈內圈外玩家都相當合適。

啊！這光！啊！這水！

東方同人遊戲中普遍畫面質量不算拔尖，3D同人尤其是這樣，永幻的角色建模整體簡陋，零夢魂的格局略小，相對同類作品本作的畫面水準可謂出色，建模水準不低，動作流暢自然，招式豐富，而場景表現更是出色，在濾鏡效果下光影、水面表現極為優秀，畫面整體舒緩自然，觀感極佳。

夭壽辣彈幕可以跳過去啊

眾所周知，東方本體是彈幕遊戲，眾多同人作也會或多或少加入彈幕元素進去，本作也不例外，雖然pad長和大小姐兩位自機都是偏近戰，只有少量技能算彈幕，但敵人多數仍是妖精等傳統東方彈幕角色，彈幕形式多樣，隨著難度增大密度也會隨之提高。但與同類型的永幻不同，本作我們可以跳，哎，可以跳！很多難以躲避的彈幕都可以靠跳躍進行迴避，甚至大小姐還有滑翔可以提高滯空時間，跳A衝刺瞬間安全走位。

說到兩位自機，本質兩人的劇情基本是一個流程，打的敵人BOSS也是一致的，但由於兩人性能不同體驗也有巨大差異。大小姐純近戰，而pad長稍微手長一丟丟，玩家在平A外還可攜帶自由搭配的倆技能一符卡。個人體驗上大小姐的手感更加無腦莽夫，因為這遊戲有一個根據連擊提升傷害和經驗加成的機制，而大小姐的平A，可以消彈幕，彈幕遊戲平A消彈幕哦！導致大小姐基本各種無腦莽連擊還不斷，弱一點的BOSS追著砍。

總結

優秀的3D東方同人，高質量的畫面給人良好的第一印象，動作和彈幕的結合使得遊戲性極佳，雖然略有瑕疵但不影響本作在眾多東方同人中的優秀表現。
