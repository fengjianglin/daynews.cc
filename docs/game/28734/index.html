漫威宇宙中最能作死的神，最快一次死亡時間到底有多快

2019-12-22 186條評論 1,106次閱讀 142人點贊

作為巨型首領大戰出場的第二個「沙包」，吞叔即將迎來自己第五個月的誕辰。要問為什麼在這樣一個好日子裡卻要出一個「傷害」他的攻略（吞叔哭唧唧），當然是想以這種方式表達對他的愛啦。

自打大女兒莎倫羅傑斯開啟三階以來，就以「哨兵殺手」之稱而得到廣大玩家極高的評價。一炮秒殺第一階段，已經難以用肉眼分辨哨兵血條顏色的場面，更是讓眾多新手玩家趨之若鶩。不過在吞星出現之後，雖然大女兒仍不減當年威勇，但在吞星以哨兵幾何倍數的防禦，讓傳奇守護者失去了本該屬於她的浪漫。所以希望能通過本篇攻略（適用於後期玩家），來為大家展現重回巨型首領戰T0梯隊的大丫。

配置思路及詳細操作解釋

以普遍的玩家思維，悸動是大女兒的首選。不過憤怒對於大女兒的大部分小技能，其實是有著更高的契合度。所以為了充分發揮針對吞叔的威力，選擇了憤怒流派。

在隊伍上，我們也不需要吝嗇。尼克弗瑞隊長白狐右翼，這裡建議需要在寇森和白狐之間取捨一下。寇森是因為二階被動的一個額外作用，適用於隊內全體英雄陣營20%的絕對暴擊率。國服詞條為百分百暴擊率提升20%。此類暴擊率是不受面板限制的，對憤怒的觸發幫助很大。雖然用白狐的話也能保證一定的觸發。不怕一萬隻怕萬一，從數百場的吞叔經驗來看，用白狐在絕大多數三階技的時候還是能保證及時觸發的。

好了話不多說，以上就是個人使用次數最多的一個號大女兒的配置。開始遊戲之前，給大家的建議是：不要讓其他玩家用輸出干涉自己的節奏。純躺，沒錯就是純躺！在進入遊戲的一開始，我們是可以先用福瑞的5技能biu一下的。不過剛開始吞星的拳頭或者巴掌下來會很慢，吼的話最好走上去先等個一兩秒，緊接切換大女兒上前用龍炮接吞星的普攻，達到初期最佳的耗血效果。整個小技能釋放順序：23154（可以按個人手感自由發揮，反正無腦憤怒不用卡，龍炮吃到傷害就可以），所有小技能在兩輪之後，三階技能充能95%左右，遊戲時間大概在9分42秒左右，此時正常情況下吞星的血量不會少於70切回弗瑞。在等待大女兒的切換的冷卻期間，我們可以去四處跑跑，看看吞叔有沒有掉錢包什麼的。當大女兒頭像亮燈的時候，注意時間，這時候吞星在9分30秒前後，是一段比較長的獃滯期。如果在這之前不小心讓血量掉到了68以下。那麼恭喜你，一大波隕石啊、地爆啊，定向全圖炮等著你。不是說這些技能惹不起，因為我們本篇注重的是效率，在吞星放跨階段技能的時候，我們基本是沒有還手之力的，只能走位…走位…走位…所以我們需要找時機，在吞星9分30之前最後一次普攻看到他 要收手了 。此時我們在場的人依然是老尼克，收手之後等3秒，就可以切到大女兒龍炮去接吞叔的下一次攻擊，然後釋放我們的—傳奇守護者。

上圖這一炮或許太狠了，不過由於憤怒的緣故，6技能轟完我們其實還有剩餘兩三秒的6技能龍炮buff，用以追加54技能二連。而且此時的54技能傷害是相當爆炸的，基本相當於3套全部小技能。目前能做到的只有三炮或者兩炮，能攻烏魯全神話，卡片總共79%的能攻。兩炮的運氣成分還是較大，而且35血這種情況依然會被吞星的清圖攻擊噁心到。所以我們把第一炮之後的剩餘血量基準定在51-38之間浮動，對於大多數中高配大女兒玩家還是能接受的。所以在這個血量區間，我們面對的就只是人畜無害的吃球球環節了。在第一炮吞星鑽洞剛出來之後，我們要做的還是小技能全放，血能磨掉一點是一點，但這都不重要。吞星開始吃球前我們只要站在下圖這個位置，嗯，就是這個可愛的小機堡，我們就不會面臨被彈開受到傷害的風險，從而恣意妄為了。但需要的時間也不多，大概第一顆球快碰到吞叔的時候。

我們的下一發傳奇守護者也充能就緒了，所以接下來要做的當然是切到我們敬愛的老局長。給他老人家孝敬點吞蜀黍做的「大湯圓」，也算彌補作為工具人的缺憾了。在吃球結束後，要確保卷福的法陣一定要碰到吞叔，給到自己身上還好，碰到之後反正要切的。如果是路人，那就只能祈禱他有沒有這個腦子了。

當看到吞星無力趴在我面前苦苦哀求的時候，我的腦海中不禁回想起那遙遠的過去：和吞叔長達3分鐘、5分鐘、7分鐘甚至9分鐘「打情罵俏」的美妙時光。而現在這一切都將煙消雲散，一股強烈的酸意湧上鼻頭。所以當吞星趴下的時候，當然是讓老尼克再biu一下，切到大丫。這次就啥都不用管了，直接36技能招呼然後追加54技能。等吞叔在一臉懵中回過神的時候，他的血大概僅剩21條以下了。而且負點責任的說，可以是21以內的任何數字，甚至0。所以這第三炮，如果是10以下，就不用再麻煩老尼克出來了，直接36技能即可，10以上就需要讓老尼克出來，然後重複第二炮的準備工作。接下來就是欣賞自己號上的成果啦。遊戲結束之後可能有特工要問：「一天就只能爽這一次嗎？」答案當然是，你想的太少了。接下來才是整篇攻略真正的靈魂。

如果說平日里要好一點的遊戲好友或者聯盟成員互帶，那首選方式非快打完斷網莫屬了。當然弊端顯而易見，別人「白嫖」你白給。我帶完你你再帶我，總共兩次獎勵兩人平攤。而且再加上技藝不精導致頻繁切人躲技能甚至翻車，浪費的時間那是難以想像的。所以，如果說要犧牲一個人給大家帶來好處，就得是一個信賴你的人把他的號託付給你，然後化作春泥更護花。這樣在持有兩個號的情況下，雖然犧牲了一個乘客位，但至少能為自己帶來理想的收益（由於帶顏色框的消耗都很大，所以只能通過一星體現了）。

攻略plus小技巧

下面來說如何操作兩個號的問題。首先模擬器是必須的，人人都裝。但一雙手操縱兩個號，就算我能做到也很容易首尾難顧。如果直接拍下來的話或許就好懂的多。先開模擬器和手機上號，模擬器登自己的，手機登憤怒大女兒號，然後開好吞星房間後準備一摞書一類的東西放電腦桌左下，靠近鍵盤就行。然後弄個小豬佩奇，呸吹風機，出風口和手柄都對著桌面，在進入遊戲後把出風口抵在鍵盤W鍵，後端靠在書的電腦一側，在倒計時結束之前確定模擬器上的虛擬鍵盤可以移動後，就可以把注意力全放在手機上了。要說為什麼是W鍵，這一點也有一定的依據，按W開始後人物是直接朝吞星下方跑去的，由於吞星正下方有個小凸起無法越過，所以人物一般會靠著按死鍵盤的慣性溜到凸起的左側。中間被左勾拳打死一兩個也無關緊要，打得彈到一邊去的話，吃球階段也會被吸回原位。所以人物停留的這個地方就會在博士的法陣出現後發揮重要作用。因為從這種操作幾百場的經驗來看，絕大多數的法陣都會出現在模擬器的掛機人物上，至於原理無從考證。除了這些，要注意的就是斷網時機了。斷網的話需要三炮的時候最好斷，看到第三炮的憤怒觸發斷開即可。兩炮的情況下由於憤怒加龍炮爆發的不穩定導致最後的血條判定極難預測。如果這時候不小心在在線的時候血條變0，唯一的補救就是在手機屏幕中間轉圈前，直接把遊戲後台清掉。不然等著你的就是人物一灰，今日告吹（下圖證明出過一次憤怒，雖然只有一次）。

總結

總的來說，這是遊戲理解與現實靈感的一次完美碰撞。對此希望這個攻略能夠為大家帶來真正意義上的便利。在新版本已經到來的情況下，大家可以以這攻略為基準，搞到更高的配置，一定能創造更快的成績，掉出更好的獎勵。
