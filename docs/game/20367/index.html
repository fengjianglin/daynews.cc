在《部落衝突》中，亡靈真的是一個“冷板凳”，沒有作用么

2019-12-16 144條評論 1,128次閱讀 224人點贊

亡靈作為黑水兵種的一個分支，它又有著怎麼樣的用途呢？

在《部落衝突》中，許多的首領並不看好「亡靈」這一個兵種，因為玩家實在想不出，除了清邊外，它還可以有什麼作用。

然而，事實真是如此。

我們先來探討一下，如果刪除了「亡靈」，COC會變成什麼樣子。

首先，弓箭手的等級升級，不得不優先提高。因為亡靈的佔位空間是2格，經常用來補全兵種總數空間的尾數，如果沒有亡靈，我們就不得帶上幾個弓箭手，或者法師，用作後期的收尾、清邊。

而弓箭手的傷害，遠不及亡靈，2.5個弓箭手才等於1個亡靈，所以佔用的空間又不得不增大一些。

龍寶寶是大家都比較熟悉的清邊兵種，單獨的時候容易狂暴，適合拆除陣型外，有防禦塔保護的建築，但是它卻沒有亡靈的清邊速度快。

亡靈適合拆除角落建築比較多的區塊，而2個亡靈的拆除建築速度，比龍寶寶快一些，又比龍寶寶少6格的佔用空間。

我們知道「蝙蝠法術」召喚出來的蝙蝠，具有強大的清邊能力，前提條件是需要保留許多的蝙蝠，一起拆除建築才可以，但是3個一起的亡靈，它的拆毀建築的速度，幾乎可以趕得上一群蝙蝠的速度了。

曾經有首領說礦工、亡靈是上杯的好方法，事實上也確實如此。9本實力的村莊，120個弓箭手，在陣型外圍上兩圈，可以拆除40%的防禦建築，而亡靈只需要50個。如果10本以上的村莊沖杯，氣球加亡靈，也是一個不錯的選擇。

並且亡靈的黑水廉價，出營時間快，加速藥水下1小時的出營速度，可以達到無間斷。也是目前版本，偷取陣型外資源，最實用的兵種。

所以亡靈作為《部落衝突》非常冷門的兵種，它的主要用途是在前期的清邊、與後期的收尾工作。據玩家計算，保留4至5個亡靈收尾，收尾的時間可以提速25%，而弓箭手、法師的收尾提速只有18%左右。
