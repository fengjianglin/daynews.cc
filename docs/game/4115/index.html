《莎木3》Huber給出測評！冷飯作品希望哪個重製？

2019-12-09 0條評論 16次閱讀 0人點贊

摘要

還記那個在《莎木3》公布時表現激動的主持人嗎？《莎木3》作為一個經典系列一直以來都是玩家心中的白月光，當《莎木3》正式公布時GameTrailers的主持人Huber展現出了非常激動的表情，這也是大部分期待《莎木3》玩家的反應。

還記那個在《莎木3》公布時表現激動的主持人嗎？《莎木3》作為一個經典系列一直以來都是玩家心中的白月光，當《莎木3》正式公布時GameTrailers的主持人Huber展現出了非常激動的表情，這也是大部分期待《莎木3》玩家的反應。並且很多國外的玩家都非常期待Huber能夠給出《莎木3》的遊戲測評。

遊戲發售後Huber對於《莎木3》的動態近乎是沒有的，只有偶爾兩張遊戲截圖，但是近日Huber發文代表Easy Allies給出了對《莎木3》的測評，並且給予了一個中等偏上的7分評分。從Huber的評測中我們可以看出真的是很公平公正了，沒有一味的吹捧《莎木3》而是很中肯進行對比。《莎木3》忠實地延續了芭月涼的旅程，在經歷了18年的考驗之後，結果基本是讓玩家滿意的。並且遊戲中一大一小地圖的改變展現了平衡，同時內置的小遊戲讓人上癮，賭博也變得驚人地重要，有助於遊戲本體的推進。

但是本該吸引人的劇情在表現上有所膈應，遊戲失去了過去展現嚴肅和憂鬱氣氛的電影美學，在諸多的劇情過場中又缺失了藝術性。讓玩家很難跟遊戲支撐的角色產生共鳴，儘管與莎花產生的羈絆仍然有著動情的時刻，但是新角色在印象上是失敗的，而某些回歸的老角色則讓人感到格格不入，至於《莎木3》的戰鬥系統則是根本沒做到位。

不過不管怎麼說《莎木3》依然是很多人心中的懷舊回歸，這對於那些曾在藍帝復仇之旅上投入過的人來說，本身就是一種無價的回饋了。其實在幾年前甚至十幾年前，有很多好玩的遊戲玩家們都因為平台、語言等等原因沒有體驗過或者短暫體驗過，那麼在如今的條件下這些「冷飯作品」玩家是否期待他們重製回歸到PC端呢？

有不少玩家表示只要這些遊戲敢重製我就敢買！比如說「GTA4」、「荒野大鏢客救贖1」、「超級馬里奧正統歷代作品」、「塞爾達傳說天空之劍」、「巫師1、2」、「大航海時代4」、「軒轅劍」等等，那麼你們心中最希望哪部「冷飯作品」能夠重製歸來呢？歡迎在評論區討論，想要了解更多的遊戲資訊那就關注小編吧！
