見過很多英雄聯盟高端玩家在談到打野英雄的時候，都會格外注意豹女，因為豹女是一個能夠突破認知上限的英雄，大家都說十個豹女九個代，還有一代中代，可為啥豹女能夠受到高玩們如此多的青睞？

首先在打野怪方面，會玩的豹女是可以無傷刷野的，上來三級滿血去對面反野，直接在血量上佔到了一定的優勢，利用被動和超遠距離的技能拉扯，殺人書稍微不注意就能拉滿，這裡可以看MLXG直播就能知道，豹女的節奏想要加快是非常容易的。

由於豹女核心技能的Q是預判性的技能，導致了上下限的問題，高手愛不釋手，新手碰都不敢碰一下，乾脆就望而卻步了，個人認為這也是豹女為啥沒有久居職業賽場的原因，畢竟職業聯賽要的就是一手穩定，人盲僧再拉跨至少還能踢一個提供一點控制，然而豹女前期沒有節奏的話直接就成透明選手了，在職業賽場上沒有虐菜這樣的說法，秒鎖豹女的選手得要多大的自信啊。

但在黃金左右的對局中，豹女卻成了代練、小號玩家們最喜歡的英雄，這是一個能依靠操作拉開優勢的英雄，就算你一條送出來一個爹裝備好的嚇人，但豹女可以依靠操作將這些多出來的裝備屬性化為烏有。

豹女豹形態的W技能和VN的Q技能有異曲同工之妙，甚至還可以多VN一個翻牆，只要走位好排除一下指定性的技能幾乎是可以規避大多數的傷害。可偏偏豹女在後期的能力表現非常差，要是一個不小心沒能拉快隊伍的節奏，打到了後期就會顯得有些被動了，裝備大家都會發育的差不多，豹女的位置變得相當尷尬，Q中了掉血看起來很多，但是要不要突進去呢？突進去了自己很可能會被傷害堆死不說，還極有可能會被對面神裝情況下的後手技能逃生甚至反殺。

其實豹女這個英雄和蜘蛛是類似的，屬於個人能力和前期C向英雄，野爹的稱號大多也是給蜘蛛和豹女起的，一個擁有良好控圖觀的打野，對豹女熟悉的話，很多時候都能打出打魚塘局的效果，建立巨大優勢迅速拿下比賽，這也證明了為啥代練都喜歡用豹女，因為它贏的快還輕鬆啊，只要自己能力強，勝率不知道多穩定。

其實很多遊戲中都有這樣的”爹”存在，一旦上手之後會有一種藐視眾生的感覺，最近前英格蘭國家足球隊隊長歐文，就是被這樣的遊戲吸引來到中國，就為了玩貪玩藍月，沒錯就是那個”屠龍寶刀砍起來的遊戲”，這樣的遊戲到底是啥特色這麼有趣？讓人前國腳都不遠萬里漂洋過海來嘗試？
