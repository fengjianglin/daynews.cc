法海山脈守護神陣容介紹

陣容組成

腥紅收割者-弗拉基米爾、岩雀-塔莉埡、復仇焰魂-布蘭德、深海泰坦-諾提勒斯、魂鎖典獄長-錘石、熔岩巨獸-墨菲特、殤之木乃伊-阿木木、喚潮鮫姬-娜美

羈絆效果

4海洋：全體友軍每4秒恢復30法力值

3魔法師：魔法師有50%幾率雙重施法

4守護神：守護神增加300%護甲

2山脈：隨機一名友軍獲得1500生命值的護盾

解析：如果說之前的法海陣容是爆發秒人，這套新版法海就主打坦度和控制，前排4守護神抗傷害，泰坦，石頭人，阿木木和娜美控制不斷，火男可以放心打輸出，如果一個大招打不死人，那就再來一個。

核心英雄及裝備推薦

復仇焰魂-布蘭德：

鬼書（大棒+腰帶）、

熾天使（眼淚+眼淚）、

帽子（大棒+大棒）/

法爆（拳套+大棒）

熔岩巨獸-墨菲特：

復活甲（大劍+鎖子甲）

深海泰坦-諾提勒斯：

青龍刀（大劍+眼淚）等任意回藍裝

運營思路

開局拿大棒，優先做一個鬼書出來，無論是打奧拉夫還是夢魘都非常有用。

前期依然是來什麼用什麼，中期五人口可以扭曲樹精、詭術妖姬、翠神、腥紅收割者、辛德拉湊到3森林3魔法師2海洋，又或者是魂鎖典獄長、深海泰坦、腥紅收割者、岩雀、暗黑元首湊到4海洋3魔法師2守護神。前期可以合個鬼書給岩雀，或者是有海洋格的話可以給暗黑元首踩格子帶鬼書（暗黑元首放技能很慢但AOE很強）。

6人口時熔岩巨獸來了最好，這時候用熔岩巨獸、深海泰坦、魂鎖典獄長、暗黑元首、岩雀、腥紅收割者湊到4海洋3魔法師3守護神2山脈，在這個階段有山脈提供的1500護盾是很頂的，如果是山脈圖的話熔岩巨獸沒來時可以先用山脈奇亞娜（奇亞娜元素隨地圖元素變化）。

7人口時就要開始搜一下卡了，每波搜到30左右就好，娜美沒來之前依然用著暗黑元首。復仇焰魂來了就直接上，4海洋在這個時候還是非常關鍵的，裝備上齊開始發力，復仇焰魂現在不算熱門也是很好到2星的。有著4海洋法力值恢復的熔岩巨獸和深海泰坦非常好用，控制打的滿滿的。魂鎖典獄長提供的護盾也能為隊伍帶來很大的幫助。復仇焰魂在有裝備的情況下傷害依然是很高的。能穩住血後開始存錢上8人口。

8人口時需要大量花錢了，補上4守護神，沒有殤之木乃伊但拿到寶石也是非常不錯的，娜美來了換下暗黑元首，再給隊伍增加一個控制。後期越多靈風越好，看看剩下的幾個玩家的陣容，找好奧拉夫和永恆夢魘的位置，吹起來後勝率起碼翻一倍！夢魘被吹起來後只能走過來先打前排。奧拉夫被吹起來下來後還需要時間才能開技能，可以趁這個時候將他秒了。

結語

這個陣容在目前的環境中很少人玩，每局大概有一個，多的話就兩個，或者直接沒有人玩，深海泰坦和魂鎖典獄長比復仇焰魂還要難拿都有可能噢。
