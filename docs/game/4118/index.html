在這局遊戲中，卡爾在一級的時候和鐵男在上路野區相遇，但是因為卡爾沒有卡Q技能的傷害，所以第一波並沒有造成什麼傷亡，只是一波換血讓自己非常傷。對線上，對於鐵男這種半坦半AP的英雄也是很吃力的，在前期技能CD沒有起來的時候，也是很難對線鐵男，卡爾吸血鬼也是一直被壓制，還被單殺了一次。在之後的團戰中，吸血鬼也是屢屢被鐵男找機會單殺，導致己方一直處於弱勢，甚至還發起了投降。

即便是如此逆風的局面，卡爾仍然沒有放棄，雖然前兩波團戰打下來我方虧了，但是問題不大，團戰依舊還能打！補充完狀態的卡爾重新回到線上，在小龍坑處爆發了一波團戰，吸血鬼一套技能直接將風女、女槍、趙信三人同時秒掉拿到三殺！卡爾此時沉著冷靜，等技能CD重新加入戰鬥，一套連招收掉妖姬鐵男完成五殺！這波團戰能拿到五殺卡爾也挺吃驚的，隨後靠著這波完美五殺將局勢扳了回來，帶領隊友成功逆風翻盤取得了比賽的勝利！

吸血鬼是一個能在團戰中不斷輸出的英雄，而他依靠的不是刷新CD，而是本身技能的CD就很短，卡爾在將CD疊到45%以後，吸血鬼的Q技能和E技能幾乎都是兩三秒一個，配合上W血池和金身的保命能力，大招加上E技能蓄滿的傷害能很輕鬆的打出成噸的傷害.
