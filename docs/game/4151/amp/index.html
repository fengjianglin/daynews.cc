你我裝備差不多，我沒拿到T1胸，但是我有狗牙和射速 。我覺得換墓碑是最好的選擇 蒼白的攻強只對戰鬥有最大提升 戰匕要的是背刺和白字的暴擊 我暴擊是33.8% 你應該少了個墓碑是最好。我公會KBZ 太猛 常年霸佔前三 我只能跟著第四吸尾氣 全程吃奶的勁 都打不過。

狂暴戰傷害不該這麼低

法師喝水只喝射程未到的那幾秒，打小怪全程不用喝滿，回藍就足夠了，一直喝水就是不會玩或者裝備太差的，傷害組成不對，我們一小時20分鐘團，前10名大多100w左右高不了這麼多，法師很難有機會進前十。

王者祝福對法師的提升非常巨大，能多10%的藍量就是直接轉換成dps了，沒有王者很容易空藍，力量祝福和風怒哪個對近戰提升大看看聯盟部落菜刀隊的速度就明白了，風怒吊打力量祝福啊。
