所有對白加起來不超過20句，這部14年前的神作依然感動了無數人

2019-12-07 0條評論 1次閱讀 0人點贊

摘要

遊戲玩多了，你總能找出一些套路。比如主角往往都從一個小角色開始，然後一路上不斷擊敗一些雜兵來累積經驗與金錢，路上結識更牛的隊友，買到更好的裝備，學會更強的技能，最後成功擊敗強大的BOSS，成為拯救世界的英雄。但！是！

遊戲玩多了，你總能找出一些套路。

比如主角往往都從一個小角色開始，

然後一路上不斷擊敗一些雜兵來累積經驗與金錢，

路上結識更牛的隊友，買到更好的裝備，學會更強的技能，

最後成功擊敗強大的BOSS，成為拯救世界的英雄。

但！是！

阿正最近玩了一款遊戲，它幾乎顛覆了上述所有理念，

主角沒有技能，沒有裝備，身上除了基本的一弓、一劍、一馬就再無其他。

地圖龐大，卻沒啥特殊要素，唯一能做的就是策馬奔騰。

整個遊戲里只有BOSS戰，也沒有什麼雜兵供你練手。

甚至，連裡面所有人的對白加起來都不超過20句。

但就是這樣一個遊戲，卻讓阿正在通關後久久難以忘懷，

它就是——

《旺達與巨像》

它是05年出品的PS2遊戲。

在推出的當年，它迎來了眾人的一片驚嘆，怎麼還會有這樣的作品？

就像開頭說的那樣，它的設計理念完全顛覆了一般人對遊戲的認知。

可問題是，這樣的顛覆配合遊戲的調性卻又是如此的合適，它的顛覆反而讓它成了一代傳說，就算在十幾年後的今天依然粉絲無數。

而在去年，官方更是為了回應粉絲們的期待，專門出了PS4的高清重製版，讓一代傳說得以重生。

這次，阿正帶你來看看它吧。

故事一開場，一股莫名的孤獨感就撲面而來。

這片土地美麗、遼闊，但空無一人，只有一名騎著馬的青年抱著一個妹子，

一路穿過荒野和叢林，走過望不見盡頭的長橋，進入一座不知名的神殿。

青年名叫「旺達」，他為了復活失去靈魂的摯愛，偷走了村裡供奉的神劍「往昔之劍」來到了這片禁忌之地。

傳說，這裡有神明能喚回逝去的靈魂。

旺達抽出神劍，引出了一個自稱「多爾暝」的神秘聲音。

多爾暝告訴旺達，想實現願望，可以。

只是在此之前，必須要旺達用手中的往昔之劍殺死16尊徘徊於這片荒野上的巨像。

不過，多爾暝也警告旺達：「你可能會因此付出沉重的代價。」

旺達的眼神中沒有迷茫——

「我早有覺悟。」

於是旺達，也就是玩家你的旅程正式開始。

遊戲的框架非常簡單，就是不斷在「尋找巨像——殺死巨像——返回神殿——再次出發」的輪迴里往複循環。

旅程單調，甚至可以說是有些枯燥，尤其是在這片土地上賓士的時候，你會感覺到一種幾乎極致的孤獨。

整個遊戲里的敵人只有那16尊巨像，除此之外，沒有雜兵，沒有小怪，甚至連動物都十分罕見。

馳騁在這片大地上的，只有你和你的愛馬「亞格羅」，無論走到哪裡都只有風聲、水聲、樹葉聲。

走得久了，你都感覺不到天氣與時間的變化，

整個世界好像都陷入停滯，而這個世界裡只剩你一人。

可這又如何？

在決心的驅使下，好不容易找到巨像，擺脫了孤獨，但隨之而來的，又是深深的無力感。

那究竟是，怎樣的對手啊？

巨像的身軀有如大廈一般龐大，

比如2號的巨牛，單單一隻蹄子都比自己還高，它的身軀本身就是一座龐大的迷宮，你甚至都不知道如何才能爬上去——

還有13號的飛龍，它在天際翱翔的時候，你在下方看起來是如此的渺小——

你都難以想像，這樣的對手該如何應對？

況且自己只是個普通人，沒有魔法，沒有科技，連在它們身上爬久了都會因為體力不支摔下來。

雖然世上早有大衛戰勝歌利亞的傳說，但這種體型百倍懸殊於自己的對手怎能是凡人能贏得了的？

可即便如此，「我早有覺悟」。

沒有猶豫，也無需猶豫，你用箭射進它們的要害，將巨人的身軀牽引至大地，再攀爬上他們的身體，用劍插入它們的核心。

在經歷了無數的掙扎與咆哮之後，巨像在最後一擊中轟然倒地。

巨像死後化作無數的黑氣，併入侵到你的身體里，你在黑氣的侵襲中暈倒，醒來後，發現自己又回到了神殿。

一切如常，只是祭壇上女孩的臉色似乎紅潤了那麼幾分。

你靜靜地看著她，這一刻不需要任何言語，而下一個巨像又在召喚自己。

可隨著劇情一步一步推進，你覺得情況似乎有點不對勁。

為什麼每次擊敗巨像之後，自己都會被黑氣入侵？

為什麼在神殿醒來時自己身體都會出現一些莫名的變化？

還有遊盪在荒野上的那些巨像，

等你玩久了之後就會發現，其實有很多巨像對你並沒有敵意，

只要你不去主動打擾它們，它們都非常和平，好像它們自古以來就屬於這裡。

但事已至此，你已無法回頭。

「我早有覺悟」。

唯一慶幸的是，自己在旅途上還有好夥伴「亞格羅」一路陪著自己。

在這個荒無人煙的世界盡頭，它是你唯一的慰藉，緊張時，害怕時，孤單時，只要你一聲口哨，它就會向你奔來。

可忠誠如它在也最後一次戰鬥中離你而去。

在通往最後一隻巨像的路上，石橋斷裂，它用盡最後的力氣把你推上邊緣，自己卻墜落懸崖。

「我早有覺悟」。

你咽下心中的苦悶，帶著失去摯友的傷痛，爬上最後的巨像。

隨著巨像發出最後的一聲咆哮，你再次暈倒。

這次醒來或許就能見到她了吧。

可惜，事實沒如你所願，

原來多爾暝只是在利用旺達而已，它是自古就被封印於此的邪神。

它的力量被分成16份，分別封印在禁忌之地的16尊巨像體內。

現在16尊巨像已死，多爾暝即將在旺達體內重生。

而此刻，發覺旺達偷走神劍的祭司和士兵也來到了神殿，他們為阻止邪神的復活而來。

祭司看到被邪神侵襲的旺達，痛心不已，卻也只能讓士兵痛下殺手。

面對士兵將劍刺入自己的心臟，你並沒有太多怨言，畢竟這一切….早有覺悟。

只是，只是，只是，能不能讓我再看她一眼？

可惜，長老趁著邪神還未徹底蘇醒，用神劍的力量啟動了終極封印術。

封印之光將邪神的力量一點點撕碎，露出旺達的身影。

這是遊戲中最後一次可以操控主角的時刻，在封印的拉扯下，你可以操控旺達向著少女靠近，

我只想再看她一次啊。

可無論你多努力，都無法靠近她一點點，只能眼睜睜地看到自己被拉扯進封印之地。

終於，一切都歸於平靜。

邪神歸於封印，青年也灰飛煙滅，只剩下祭壇上的少女逐漸蘇醒——

「旺達，你在哪裡？」

玩到最後，阿正覺得這真是一個悲傷的故事，但也是一個很有力量的故事。

正如最初多爾暝對旺達的警告：你要付出的代價會是慘痛的。

而汪達在一開始就給出了答案——

我早有覺悟。
