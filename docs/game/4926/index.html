酒館戰棋即將改動！龍屬性新卡加入是否比光牙更強？

2019-12-04 0條評論 14次閱讀 0人點贊

摘要

爐石傳說酒館戰棋模式自公測以來人氣較高，在新版本還未到來之際，玩家們等待新卡逐一公布之時在酒館中下棋消磨時光，玩家們也不禁會想到，最近公布的新卡是否會在酒館戰棋不久後的新版本中出現呢？

爐石傳說酒館戰棋模式自公測以來人氣較高，在新版本還未到來之際，玩家們等待新卡逐一公布之時在酒館中下棋消磨時光，玩家們也不禁會想到，最近公布的新卡是否會在酒館戰棋不久後的新版本中出現呢？

在即將到來的新補丁中就增加了3個新英雄包括拉法姆，也能看出在酒館戰棋初期改動和版本更替的頻繁，所以一切皆有可能，爐石中的詞條目前在酒館戰棋中只有野獸、機械、魚人、惡魔，其餘的海盜、龍、元素還未登場，不過在酒館戰棋中已經有棋子隨從效果出現了”龍”這一詞條。

下一個新加入的種族也已經確定為龍，而新版本”巨龍降臨”正好以龍為主題，說不定會有很多新的龍卡或者機制加入戰棋！

各位不妨和小編一起大膽猜測一波那些有趣的新卡如果加入了酒館戰棋會如何吧！首先是瘋狂巨龍死亡之翼，作為戰士的新橙卡在爐石中的用途咱們暫且不說，如果此卡進入酒館戰棋一定也非常有趣，效果進行更改，擁有類似風怒一樣的效果，隨機攻擊所有敵方隨從一直到死亡為止，相當於超級風怒，這個身材，在5費當一個頂級打工仔還是很有搞頭的。

還有騎士新橙卡龍騎士塔瑞薩，亡語效果可以更改為使場上的一張龍牌獲得+3+3以及此亡語，比較合理，而且配合瑞文還能誕生專屬於龍的套娃體系，非常有趣。

龍卡特色往往和手牌中是否有龍相關，也許酒館戰棋中也能加入這一新機制，玩家們需要在手牌中積攢至少一張龍牌才能最大化發揮龍卡的效果，也許這樣的新機制能增加可玩性和複雜性呢，對於玩家而言也更具有挑戰，畢竟要花3個鑄幣購買一個用不了的資源，而且還要合理調控場面，不過以後版本中確定了手牌資源可直接賣出，也許這一機制將來真的會出現？

各位覺得有哪些新卡或者是新的機制可能會加入酒館戰棋呢？

新版本的預購也如期而至，100個新版本卡包、全新戰士皮膚”死亡之翼”和卡背，還有額外一張新卡金橙！酒館戰棋也能獲得三選一英雄和全套表情的特權，新版本即將上線，要抓緊時間了！
