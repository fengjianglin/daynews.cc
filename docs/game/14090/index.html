KPL最後一個席位誕生，Hope戰勝大熱門KSSC，戰術體系值得kpl借鑒

2019-12-11 1,123條評論 2,769次閱讀 270人點贊

KPL秋季賽已經進行到總決賽階段了，而與此同時KPL席位賽也落下帷幕。經過了7局的鏖戰，Hope戰隊在1-3落後的情況下連贏三局，通過巔峰對決贏下了大熱門KSSC，成功拿到了2020年KPL春季賽的資格。

這場比賽雙方打得非常激烈，雙方都是會打運營的隊伍，尤其是KSSC的體系某種程度上有點像QGhappy，而Hope則是有點像沒進KPL之前的TS。前面幾局KSSC拿到領先之後，Hope並沒有慌亂，第五局扳回一局之後，第六局竟然拿出了明世隱加成吉思汗的路人局體系，效果竟然相當好，傾城的成吉思汗拿到一波四殺拯救了隊伍。而巔峰對決Hope的陣容也和其他隊伍不太一樣，輔助位並沒有選擇最熱門的盾山，而是為了搭配公孫離選擇了大喬，邊路拿了坦度比較足的豬八戒，最終也正是憑藉著大喬和公孫離的配合拿到勝利，傾城拿到了偽五殺，成功幫助隊伍打進KPL。

Hope這支戰隊和其他隊伍都不一樣，他們的打野晨陽是絕對核心，不管是刺客野核還是戰士打野他基本上都要吃比較多的經濟，包括楊戩，盤古這些其他隊伍拿來打藍領的英雄到了晨陽手裡都是核心。而還有一個比較有意思的點就是Hope的雙C是換英雄玩的，中單一夢並不是一直打中單，有的時候會換到射手位。射手傾城也不一定一直打射手，可能也會去玩不知火舞，上官婉兒這樣的英雄。總結一下基本上是比較秀的英雄會給到傾城，而相對穩一點的英雄給到一夢，也算是最大化發揮選手的特點。

上面說的是針對英雄方面，還有就是打法方面，Hope也是獨樹一幟的。雖然一夢是名義上的中路，但是這支隊伍實際上可以說並沒有中單，中路的線永遠是幾個人一起清，然後幾個人抱團去邊路壓制，這樣的打法既能保證邊路總是處於優勢，也能保證打野晨陽能夠蹭線，促進他的經濟發育。毫無疑問的是Hope的這種思路值得KPL隊伍借鑒，提供了一種新的戰術體系，如果說他們打進KPL之後繼續保持這種打法的話，上限將會比較高，可能會引領一種新的潮流。

大家覺得Hope戰隊的打法是否讓人眼前一亮呢？歡迎在評論區留言交流。我是天黑君，期待下次再見！
