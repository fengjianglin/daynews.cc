雲頂王者紅蓮，上分陣容推薦：影遊俠和六狂戰

2019-12-19 360條評論 1,860次閱讀 249人點贊

雲頂之弈主播大神，S1賽季國服首位王者紅蓮為你帶來9.24版本的最佳上分攻略，快快碼起來悄悄上分吧。

6人口陣容：

7人口補安妮或者蠍子，8人口上寶石和阿木木。

裝備推薦：

也可以眼淚、法爆、無盡，眼淚給一個是配合第一輪啟動輸出，不要給多，給一個團戰最舒服。

陣容理解：

四遊俠羈絆強度極高，有三影的賽娜強度也是極高，給賽娜裝備所有遊俠輸出提升，效率最高，賽娜帶鬼書全員紅buff，賽娜帶法強裝給到的輸出加成很不錯。所以賽娜裝備到位，加上遊俠羈絆高攻速，輸出已經不差，剩下的關鍵是前排的質量和裝備。其次就是特效裝，寒冰帶颶風沉默控制效率就很不錯。

陣容優劣勢：
打刺客中期優勢，但是決賽圈劍刺的三星太猛也沒辦法。站位只有給後排靈風才能針對。打影召喚只要前排不被吹，是優勢對局。打法師石頭人到兩星帶裝備就挺難打，守護比較克遊俠，不過也有機會。打雷霆劫注意劫的位置，不要跳到遊俠中間就好。打掠食者小劣，打鋼鐵吃力，後排可能會被大嘴濺射死。
過渡跟影召喚一樣，很平滑，吃分不難，後期有水晶強勢點，賽娜到3也很強，有版本t1的實力。

陣容流程：

類似影召喚
二階段正常
3-2啟動，搜出6人口成型陣容，三影四遊俠。重要的是三張影，特別是塞恩和賽娜。
4-1或者4-2升人口，7人口重要的是搜出寒冰二星，寒冰單卡質量比其他遊俠都要高一個檔次。如果前中期特別順，也可以到8人口才花很多錢，再找二星寒冰順便搜兩張守護橙卡。搜到大成，這個陣容吃雞不難。追三的卡建議是塞恩、賽娜、千珏、EZ，塞恩、賽娜必追，EZ、千珏看情況。

站位：

賽娜站邊角位，然後只要讓離賽娜最遠的單位中間放遊俠即可，也就是要一條線的站位，不管是斜著還是直著放。塞恩放第一排，防止狂戰士跳進來。

6人口陣容：

7人口陣容：六狂戰+螳螂補沙漠，8人口可用三雷霆或者三毒，9人口再補回沙漠。

裝備推薦：

​

以上裝備五選三即可。

其他所有人都可帶：竊賊手套

陣容理解：

六狂戰羈絆增強之後在六人口羈絆裡面可以說是最實用的。只需要一張紫卡就成型，成型之後就很不錯。奧拉夫還是絕對主力，裝備要滿，等級盡量在四階段早起提到2，這時候在四階段就有統治力。靈風的理解很關鍵，前排不一定多，吹到對面前排，方便狂戰士們跳進人群打後排輸出一堆。後期不管三毒還是三雷霆都有顯著提升，一般還是三毒比較舒服。

陣容優劣勢：
打劍刺非常好打，可以用來上分。打影看時間點，三階段影強，四階段六狂整體質量夠，有帶裝備的兩級拉夫，六狂戰強，後期再看雙方質量總之沒有很劣。打守護法師相對難打，所以沙漠和毒都很重要。打雷霆劫，劫沒到兩星都很好錘，打掠食者看質量，55開。

陣容流程：

狂戰都得留，盡量在二階段把奧拉夫裝備到位給打工仔打連勝。前期三狂戰兩雷霆配合打工仔過渡是比較舒服的選擇。3-2升6人口，這時候根據情況，如果對子多，或者奧拉夫嫖到了，就可以搜多點，讓陣容成型或者質量提高，如果都不是，建議先用打工仔混到4階段再升人口去沖六狂戰和二級奧奧拉夫。如果連敗走得多，血量少經濟好，也可以三階段選完秀直接拉7人口搜卡。總之四階段奧拉夫的強勢期要把握好。
一般順的話，有兩級奧拉夫的六狂戰中期順利過渡，5階段到8，可以不搜，奔著九人口，更好搜二星煉金，同時保留沙漠的羈絆，方便打守護神，奧拉夫就是打法師和影比較困難。

站位：

全部沉底（站在最後一排）即可，奧拉夫不要被吹，塞恩站的位置吹別人前排，奧拉夫跳別人人多那邊好，但是沒有復活甲盡量不要被核心集火。
