夢幻模擬戰：又見玄學玩法，論反向附魔重要性，或將縮短畢業時間

2019-12-09 0條評論 18次閱讀 0人點贊

摘要

說起裝備附魔，相信很多人心裡都有一肚子的話想要說。作為提升英雄戰力的重要手段之一，每次附魔的心情都是忐忑不已的，如果是一發入魂、直接畢業自然是好事，但是一念天堂一念地獄的吐槽我們也經常看到。

說起裝備附魔，相信很多人心裡都有一肚子的話想要說。作為提升英雄戰力的重要手段之一，每次附魔的心情都是忐忑不已的，如果是一發入魂、直接畢業自然是好事，但是一念天堂一念地獄的吐槽我們也經常看到。比如明明是想要給拉格上一個攻擊加成的清風附魔，結果洗出來一個”智力+15%，智力+20″的屬性，畢業倒是畢業了，但是心情相比不是那麼美好。

但是就和抽卡、賭裝備這類需要考慮到膚色的玩法一樣，麵包總是會有的，只不過是時間早晚的問題。為了能縮短這個等待的時間，傳說君也在之前的文章中提到過，像是午夜零時一刻、吃10顆瓜子諸如此類的玄學形式，用以提升我們當天的運氣。不過這今天在看玩家玩家討論的過程中，似乎又有一種更”靠譜”的形式，可以讓我們快速獲得想要的畢業屬性。

這種新的玄學方式，就是反向附魔。不過具體要怎麼操作呢？且聽傳說君慢慢道來。

像我們平時給裝備附魔的時候，這件裝備大概率是已經穿在適合的英雄身上了，我們一般也是從人物角色那個板塊點進去，然後選擇相對應的附魔券。但反向附魔卻不是，它的關鍵點就在於，要把需要附魔的裝備放在不適合的英雄身上，然後需要附魔的屬性也是和該英雄需求相反的。

就像這位大佬總結得，奶錘放水兵身上，容易出智力；法杖放到雷丁這類坦克英雄身上，也容易出智力；而斧子和劍這種物攻住裝備，則需要放在靠智力生存的法師身上，就比較好出我們需要的攻擊屬性加成。

如此一來，將大概率提升我們希望出現那個屬性的概率，傳說君也是第一次聽說這種反向附魔的玩法，不過就下方評論來看，似乎還算是行之有效？不少同學紛紛表示學到了，等下就去試試，也有已經成功嘗試的同學前來還願。

傳說君這就準備去嘗試一下，這個方式是否可行，如果概率真的較以往有所提升，那可就太棒了！如果有小夥伴已經成功的，歡迎來評論和大家分享一下歐氣哦~
