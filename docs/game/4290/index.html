全明星solo賽：Theshy和Uzi晉級四強 Bwipo眼神是真正亮點

2019-12-08 0條評論 19次閱讀 0人點贊

摘要

在這次拉斯維加斯的LOL全明星比賽中，solo賽和雲頂之弈可能算相對不那麼娛樂的比賽類型，而solo也是全明星比賽的最大看點。在之前的solo賽中，Doinb和Tian雙雙出局，兩人之前猜到了自己的結局，也不算太遺憾吧。

在這次拉斯維加斯的LOL全明星比賽中，solo賽和雲頂之弈可能算相對不那麼娛樂的比賽類型，而solo也是全明星比賽的最大看點。在之前的solo賽中，Doinb和Tian雙雙出局，兩人之前猜到了自己的結局，也不算太遺憾吧。不過Theshy和Uzi晉級了下一輪，LPL選手在solo賽中的火種還是有的，而且四人之中佔了兩個名額，想必LPL的觀眾還是比較放心的。

在這次全明星solo賽的八強賽中，Theshy遇到了G2輔助Mykix。Theshy選出了揚名天下的阿卡麗，這是準備拼操作了，畢竟這是Theshy最拿手的東西。而Mykix選擇了在solo賽中非常少見的龍女，準備在六級拼大招的傷害打一波。因為英雄屬性和操作的原因，Theshy在前期狠狠地壓制了Mykix，而且在7分鐘之前造成了一次擊殺。又因為龍女被壓制的實在太厲害，所以沒等到第二次擊殺到來，Theshy就已經靠推塔輕鬆拿下了勝利。

Uzi在solo賽八強時遇到了Caps，兩人都是solo賽高手。Caps是去年的solo冠軍，而Uzi是上上屆的solo冠軍，所以這場對局非常精彩。Uzi最終選擇了自己最拿手的AD英雄薇恩，而Caps選擇了刺客琪亞娜。儘管Caps在7分鐘之前造成了一次擊殺，並且先做出幕刃，不過Uzi的薇恩在等級起來之後傷害也有明顯提升。因為高超的走位和卡距離操作，Uzi始終沒有被Caps的琪亞娜推到過牆上，並且硬點死了琪亞娜。

最終，LPL賽區的Theshy和Uzi一起晉級了solo四強，其他兩名選手是Bwipo和Levi。今天的solo賽中有一個非常有趣的部分，可謂真正的亮點。Theshy和Bwipo的座位緊挨著，而當Theshy贏下Mykix且握手結束之後，Bwipo很「深情」地看向了Theshy。可以看得出，Bwipo的眼神里滿是羨慕和崇拜，畢竟這是當時暴打FNC的隊伍，而且打穿了上路的Bwipo。據歐洲解說透露，Bwipo多次提到過S8時候被Theshy打爆的經歷，想必是留下了陰影。

接下來的solo賽將是BO3賽制，相信LPL的兩位選手可以走的更遠。
