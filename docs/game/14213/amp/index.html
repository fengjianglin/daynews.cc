歐洲博客疑似泄露《命運石之門比翼戀理愛人》英文版發售日

天天要聞
2019-12-10

2019年8月29日，我們報道了「《命運石之門比翼戀理的愛人》通過澳洲分級」的相關消息（點此獲得新聞原文）。近日，一些玩家在歐洲playstation博客上發現了這款遊戲的蛛絲馬跡：或將於2019年12月10日（歐洲當地時間）推出英文版。

據悉，《命運石之門：比翼戀理的愛人》日文版於2011年6月登陸了Xbox 360，2012年4月移植到了PSP端，2012年5月登陸PS3，2013年3月登陸PS Vita，並於2013年8月登陸iOS平台。目前，該遊戲也已經在2019年3月登陸了NS平台，但是在如此之多的版本中，從來都沒有推出過英文版。

《命運石之門：比翼戀理的愛人》背景設定為「Delta」世界線，岡部倫太郎在這部作品中將與各種各樣的女性角色展開一段青春戀愛物語，這部作品可以讓玩家放鬆心態，同時玩家也可以按照自己的喜好攻略女性角色，體驗有些不同的《命運石之門》作品。

從先前的澳洲分級情況來看，該遊戲包含輕度性暗示內容和輕度粗俗語言，整體評級為「PG」（即：適宜在家長指導下遊玩）。不過，感興趣的玩家可以先嘗試《命運石之門》正篇，而在玩家們遊玩了正篇內容後，此時再食用《命運石之門：比翼戀理的愛人》，會有更加良好的體驗（很甜）。

《命運石之門比翼的愛人》或將於2019年 12月10日推出英文版（歐洲當地時間），至於此次泄露的消息是否真實，還有待觀察。
