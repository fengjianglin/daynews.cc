模擬戰：新天賦“變化萬千”，湊齊5人方可激活能否強過魔種？

2019-12-24 2,917條評論 3,825次閱讀 651人點贊

在王者模擬戰的所有天賦中，A級天賦「魔種血統」可以說是最難達成的CP天賦，需要同時有6個英雄在場才能夠激活，當然成型的魔種陣容也是很強大的，尤其是老虎跟玄策達到3星之後。一般拿到魔種，要麼前三要麼倒數前三，也經常有因為魔種第八出門的。而在體驗服中，又出現了一個新的CP天賦「變化萬千」，同樣是多人羈絆CP天賦。

體驗服新天賦「變化萬千」

魔種需要6個英雄，這個新天賦則是需要5個英雄同時在場，跟魔種增加屬性不同的是，「變化萬千」天賦中的英雄同時上場後，會獲得「第二職業」。先來看下天賦中的5個英雄：上官婉兒、鍾無艷、公孫離、楊玉環和嫦娥。楊玉環定位堯天軟輔，嫦娥定位封神坦克，還沒有上線到正式服中。

下面一一分析這些棋子，婉兒、公孫離跟鍾無艷大家已經都很熟悉了，她們的職業分別是刺客、射手和坦克，CP天賦激活後，婉兒第二職業是法師，阿離第二職業是刺客，鍾無艷第二職業是戰士。

楊玉環則是堯天陣營的「法輔」（體驗服將輔助分為「法輔」和「肉輔」），第二職業法師；嫦娥是封神陣營的坦克，第二職業也是法師。封神陣營的羈絆效果是有幾率增加範圍魔法傷害，羈絆等級為2封神/4封神。

結合這些棋子的陣營以及第一、第二職業來看，她們5個已經湊成的羈絆有2堯天、3法師。在有這5個棋子的情況下，剩下4個棋子怎麼選擇，才能產生比較強力的搭配呢？考慮到阿離第二職業是刺客，加上婉兒是個刺客，可以再上一個刺客湊3刺客羈絆；鍾無艷是稷下陣營，那麼可以上一個魯班七號（稷下射手，2金幣），再加上封神陣營里的后羿（4金幣），這樣就有3射手、2稷下、2封神。

已經有2堯天的情況下，刺客選擇堯天陣營的老虎，再加上一個弈星打控制湊4堯天，一套陣容就成了。

最後這套陣型激活天賦後的羈絆就是4堯天、3射手、3刺客、3法師、2封神、2稷下、2坦克。如果拿到一個法師令旗，那就可以湊到5法。當然弈星也可換成明世隱，根據陣容選擇雙肉輔或雙法輔。雖然看上去花里胡哨，不知道實戰效果會是如何，但是在堯天全員攻速的加持下，3射手+3刺客的輸出還是有保證，如果湊到5法羈絆，婉兒也會猛，其實這套陣容的輸出點還是很多的。

陣容上棋子價值也並不高，堯天中的楊玉環是3金幣，魯班七號2金幣，后羿4金幣，嫦娥1金幣，其餘與正式服一致。那麼在後期追一個3星魯班或者3星阿離，后羿2星，婉兒或者老虎追3，應該是一套不錯的陣容。其實這個A級的CP天賦稍微有那麼一點尷尬，5人中3人的第二職業都是法師，如果再上4法師，可以湊7法羈絆，並不是很合適。

那麼大家覺得同樣是多人羈絆A級天賦，「變化萬千」與「魔種血統」相比較，誰更勝一籌呢？歡迎大家在評論區交流。
