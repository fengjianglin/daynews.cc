當年為何移除藍藥水，兩邊藥罐子出門，水晶瓶紅藍能打10分鐘！

最新入坑的不少玩家可能都不知道一件事情，那就是在最開始的LOL的時候，其實是有藍藥水這個東西的，但是在這個版本中，偷錢的移除，等於說徹底斷了藍藥水的這個東西，如今已經是沒有辦法在獲得了，回藍的時候，只能通過腐敗藥水這種來。

但是大家知道嗎？為何在當時的時候，拳頭決定了要移除藍藥水這個物品呢？

其實關於藍藥水這個東西，拳頭之所以要移除的理由也很簡單，那就是它的存在，讓遊戲的進程慢了太多。還記得最開始的時候，小魚人的出門裝都是水晶瓶，2紅1藍，對線直接吃著葯去補刀，完全不怕你消耗。

藥水吃完了，回家繼續補幾瓶葯，很快到了6級就知道上來殺你，同理潘森也是這樣，當年的無限Q潘森，就是這個原因！

所以說，官方移除了這個讓整體的遊戲進程變慢的道具，不然大家都在無腦的Poke消耗，很多的法師都不在考慮自己的藍量，讓遊戲變得過於簡單了。尤其是中路的很多傳統清線的法師，這種無限藍的話，肯定不可能造成擊殺，也就導致了遊戲一點都不好看！

所以說，這個藍藥水的移除，目的就是讓玩家更注重操作的對拼和打架，別整天想著刷刷刷。觀眾買票進來看比賽，不是看你清兵線的，而當時藍藥水的移除，也確實讓刺客小火熱了一把，然後沒多久又被按死了。原因就是ADC太沒有遊戲體驗！

這個移除的改動，和當時砍13瓶紅葯出門一樣，續航太強了，導致不可能出現對線擊殺，我半血就吃藥，吃完就回家，你能拿我怎麼樣。而且還有一點就是，法師的藍本身就是一種限制要素，有藍條就說明你不能無限用技能。

藍葯加藍buff，導致了基本上都是無限了，不可能有真空期，那誰還玩能量英雄！

所以說乾脆拳頭一狠心，直接取消了藍葯，取而代之的是腐敗藥水這種攻擊型的藥水，不得不說，一個小小的改動，確實讓遊戲變得有意思多了，而近幾個版本的改動來看，拳頭越來越想著是你們前期就打起來，鍍層，小龍，都是前期的巨大優勢，你打後期？別人5層塔皮一吃，直接就上你高地了！
