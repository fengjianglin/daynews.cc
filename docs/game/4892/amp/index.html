摘要

甄姬作為王者中的老牌法師，清兵控線，打團輸出限制走位，都是相當優秀的法師，而且比起婉兒，貂蟬這類型需要秀操作的英雄，甄姬的上手難度最低，移動的過程也能釋放技能，其他法師需要釋放技能後才能移動，就避免容易被刺客瞄準擊殺的風險。

​

一 裝備推薦

老王裝備推薦：CD鞋+聖杯+面具+迴響+帽子+輝月

​

甄姬這個英雄清兵控線相當優秀，然而使用技能的頻率相當高，特別是打消耗的情況下，CD鞋和聖杯能最大限度保持甄姬的回藍和釋放技能CD，面具和帽子則保證甄姬的法傷能對坦克和戰士造成傷害而不是撓痒痒，迴響配合甄姬的被動可以打出爆炸的效果，輝月在對面刺客強行切後的時候使用金身反殺。

這套裝備主打消耗，不過這個賽季老王推薦的另外一套非常規裝備

老王推薦裝備：CD鞋+迴響+吸血書+冰心+面具+賢者書

​

著重來看吸血書，冰心和賢者書的改變。吸血書對於甄姬打消耗戰和團戰來說是最好的裝備，甄姬的技能範圍和幅度非常廣，哪怕團戰中被刺客近身一套殘血，只要大到對手，吸血書回血可以瞬間拉滿2/3，只要操作夠迅速，反殺不是問題，比起靠金身躲傷害，無疑這個更好。而賢者書增加的生命，讓生存能力提高一個檔次，冰心能減少CD還可以減低移速和攻速造成法術傷害，可以說生存能力MAX+

二 銘文推薦

老王推薦銘文：10夢魘+10調和+10心眼

​

這套銘文中規中矩，其他法師也可以用到，移速和法穿是最重要的兩點，畢竟射手法師人均腿短還脆皮，回血速度和最大生命的增加也讓甄姬生存能力大大增強。

​

前期就靠二技能清兵，不出塔，保證不給對面刺客打野機會，等到鞋子和聖杯到手可以線野雙收，等到四級後去遊走支援，躲在草叢看準時機一技能擊飛配合大招直接控住，在卡時間放出二技能打出被動冰住效果，對方有兵線的情況下控到死不是一個夢想。

​

後期裝備成型，有肉有輸出，根本不怕刺客近身，極限的減CD下打團，消耗起來完全不怕跟不上回復，見面就二技能彈彈樂，卡地形放一技能。或者見面就放大，反正CD短，多轉幾圈大招CD又好了，可以說完克對面的坦克刺客。

像貂蟬這麼秀的英雄，只要在兵線附近根本不是對手，彈彈樂一中，預判一下走位，貂蟬只能被罰站等死，婉兒也是同等道理，婉兒想要飛天得打出五段走位，只要在必經之路上一個擊飛，用不上二技能就直接打斷了，根本不怕飛天無敵。
