雲頂之弈在上周進行了版本更新，9.23版本的改動上線後，掠食者成為了相當強力的新流派。美服王者玩家kjnkkevin，在個人主頁上分享了自己的掠食者陣容上分心得，讓我們一起來看看他的打法。掠食者陣容自然是掠食者作為核心，三星的掠食者坦度很高，還能依靠斬殺效果迅速擊殺敵人，搭配劇毒羈絆有更強的作戰能力。

遊戲前期

何時選擇開始構築掠食者？

如果你在遊戲前期(第一或者第二階段)找到了一些掠食者的英雄，那就可以嘗試組建掠食者陣容。一般強勢的開局就是在第一階段就合成了2星克格莫，然後有可能找到沃里克、雷克塞或者斯卡納等英雄。

在你進入第二階段時，請觀察其他玩家的陣容，確保沒有太多選手和你爭奪掠食者英雄。如果超過2個競爭對手，那就可以考慮換陣容了。在第二階段時，觀察其他對手的陣容非常重要，如果有3個競爭對手，那就直接放棄掠食者了。

應該選擇哪個變體陣容？

如果你已經決定玩掠食者陣容，那在遊戲前期有兩種打法。你可以存錢到3-1階段時，瘋狂搜卡嘗試合成三星的克格莫、沃里克、斯卡納或者雷克塞。或者選擇不瘋狂搜卡，而是存錢用來升級，來嘗試找到劇毒協同英雄、奧拉夫、卡茲克或者艾希。

如果你選擇了搜卡打法，那就盡量不要主動升級，而是盡量保存更多的金幣賺利息，前提是不要輸給野怪。然後在3-1階段開始搜卡，因為你在3-2階段時會自然升級。較低的等級會提高你找到克格莫的機會。在搜卡時，除非是有絕對的必要，否則不要搜到0金，盡量保留至少還有10金。

如果你的金幣不是很多，覺得自己很難合成三星英雄，或者有其他的掠食者競爭對手，那就不要去搜卡。存錢升級，儘快拿到其他劇毒配合英雄，或者將4費英雄合成兩星，也能夠加強你的陣容戰鬥力。

除此之外，在遊戲前期時，如果你找到的斯卡納比雷克塞多的話，那就應該考慮水晶變體陣容，否則的話，還是選擇狂戰士或者刺客的變體陣容。

早期陣容搭配範例

和大部分陣容打法一樣，你在遊戲前期的目標就是不要掉太多的血量。你希望儘快獲得強力的羈絆，來保持健康的血量。對於掠食者陣容而言，你可以先用德魯伊林地這個前期強力羈絆過渡，或者利用其他英雄來搭配掠食者英雄湊羈絆。比如，你可以使用沃利貝爾和奧恩來湊成雷霆羈絆。

​

遊戲中期的打法

尋找合適的英雄單位

在遊戲中期時，你可以慢慢嘗試將掠食者英雄升到三星，但不要貪心地想將三個英雄都升到三星，嘗試讓克格莫、沃里克或者其他2費的掠食者升到三星。狂戰士變體陣容的話，則考慮以下的英雄。

​

如果你使用的是雷克塞，可以尋找魔騰來構成鋼鐵羈絆，加強前排的坦度。或者嘗試找到艾希湊水晶陣容，如果你決定要玩水晶陣容變體的話。當你決定要玩水晶陣容時，嘗試將斯卡納升到三星，然後將一些關鍵裝備放到它的身上，比如荊棘之甲、離子火花和一件防禦裝備，如狂徒鎧甲、巨龍之爪等。一個坦度十足的斯卡納，是讓水晶陣容變得強大的關鍵之一。

​

你還可以考慮添加劇毒英雄，保留圖奇和蒙多並且嘗試使用它們，但如果你不使用水晶陣容，那就不要合成兩星圖奇。沒有找到辛吉德時，那就使用圖奇和蒙多吧。當你找到辛吉德時，如果你是非水晶變體的陣容，就可以賣掉圖奇將其換上，但如果蒙多一直沒有兩星的話，也可以考慮賣掉蒙多。辛吉德絕對是一個非常瘋狂的單位，只要有合適的機會，就果斷入手並且加入到你的陣容中。

​

如果你決定要玩刺客變體陣容，那就要在比賽中期就找到奇亞娜，奇亞娜是一個很強的英雄，能夠幫助你度過遊戲中期。刺客陣容的話，則考慮以上的英雄。

一般遊戲打法

對於掠食者陣容而言，你的中期打法和傳統陣容差不多，但如果你在戰鬥中覺得有些吃力的話，那就要在打完野怪後搜卡強化陣容了。

而掠食者陣容的其中一個好處是，你對4費英雄的需求沒有那麼大，奧拉夫、卡茲克、艾希或者圖奇都可以。因此，你可以比起其他的對手更遲進行搜卡，而當對手精簡了4費的英雄池後，你就有更大機會找到自己想要的4費卡了。因此，你無需在4-1或者5-1階段進行搜卡，而是在4-2或者5-2階段才開始搜。

遊戲後期打法

當你進入遊戲後期時，你需要在七級停留搜卡或者升到八級之間進行抉擇。大部分的掠食者陣容，在七級時都已經有不錯的羈絆。升到八級確實會增加你找到辛吉德的機會，但不要太過冒險，它本身是一個5費的英雄數量就很少，加上很多玩家都在搶它。如果你有機會獲得兩星辛吉德，那陣容的後期能力就會達到峰值，兩星辛吉德只能用瘋狂來形容。如果對手的魔法傷害很高的話，可以考慮添加秘術英雄，通常來說，你只能在9人口時才能湊到秘術羈絆。而在狂戰士變體陣容中，你在8人口時也能夠湊到秘術羈絆。而在水晶變體陣容中，你可以將魔騰換成秘術英雄，這樣一來的話，8人口也可以湊成秘術羈絆了。

裝備的選擇搭配

掠食者陣容並非十分依賴裝備，一般來說，你是根據早期獲得的英雄來決定是否玩掠食者陣容，而不是根據已有的裝備來下決定。當然，在這個陣容中，有一個裝備的表現確實會更加出色。

合適的裝備：

-盧登的回聲：對克格莫而言十分有效，提高它的法力值，而且濺射傷害也能夠觸發掠食者的斬殺效果。

-斯塔緹克電刃：對克格莫非常有效，額外的法力值和攻擊速度收益很高，濺射傷害會觸發掠食者羈絆，類似於盧登的回聲。

-離子火花：在任何掠食者身上都很有用，這裝備造成的傷害可以觸發掠食者斬殺效果。一般放在比較坦的掠食者身上，可以最大化收益，比如水晶陣容的斯卡納。

-肅靜：非常不錯的裝備，可以剋制法師英雄。

-莫雷洛秘典：辛吉德或者蒙多使用時能夠發揮驚人效果，增加傷害並且減治療效果實用。

防禦的裝備：

-救贖：盡量在遊戲前期合成，放在前排身上加強回復效果。

-守護天使：放在辛吉德或者雷克塞身上很好，守護天使能夠讓雷克塞有機會放出第二次技能，那次技能很容易打出斬殺效果。

-冰脈護手：在雲端地圖中十分適合前排的裝備。

-龍之爪：如果對手的法術傷害很高，那就可以放在C位身上加強生存。

-幻影之舞：如果對手有很強的刺客陣容，那就考慮合成。

-狂徒鎧甲：適合前排的不錯裝備，尤其是水晶陣容的斯卡納身上。

-荊棘之甲：水晶陣容中斯卡納非常合適的裝備，荊棘之甲的傷害會觸發掠食者的效果。
