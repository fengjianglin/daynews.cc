國產RTS手游新星！《全球行動》讓你回味90年代的經典RTS玩法

天天要聞
2019-12-19

每個時代總有那麼一種類型的遊戲制霸整個遊戲界，現在是MOBA，10幾年前則是RTS。那個時候，網吧里的人不是在打傳奇就是在打《星際爭霸》，更不用說大多數90後接觸的第一款單機《紅警》，而隨著時間的流逝，曾經那些經典的RTS遊戲已經漸漸退出人們的視線，成為一代人回憶。甚至有人說，RTS已死，當然，我覺得這個說法太悲觀了，雖然RTS已經不是主流玩法，但市面上是不乏優秀的RTS新游的，比如今天說的這款國產RTS手游《全球行動》。

復古的場景風格

RTS是什麼？是即時戰略遊戲，同時強調即時與戰略。傳統RTS通常採用上帝視角，3D地圖，單位的每一步移動，每一個攻擊都會清楚地展現在屏幕上。《全球行動》的畫面第一眼就給人很熟悉的感覺。當然依託現在的技術，遊戲的場景表現力相比以前要震撼不少，煙火的特效、炮火在地上留下的燒痕，都顯得非常生動，甚至伴隨每一次轟擊，鏡頭還會有輕微的抖動效果，頗有戰場的感覺。唯一的缺點就是人物模型相對較小，有時會看不太清楚。

同時，遊戲還加入了”戰爭迷霧”這一經典系統，不僅敵方區域會被黑暗覆蓋，未探索區域也會被覆蓋，只能隨著玩家的一步步探索撥開迷霧。俗話說知己知彼才能百戰百勝，戰爭迷霧恰巧將對手的信息全部隱藏了起來，你必須主動出擊，一步步偵查過後，才能實現有效的出擊策略。比如以前玩紅警的時候，我都是先派出幾個小兵出門偵查，每次發現資源或者對手的時候，都十分驚喜，現在一些RTS為了省事去掉了戰爭迷霧，但我覺得戰爭迷霧正是RTS的精髓之一，不能丟失。這點《全球行動》做得非常好，確實給我一種懷舊的感覺。

傳統與創新兼具的戰略玩法

遊戲一上來，會讓你選擇陣營。每個陣營的單位功能大致相同，但屬性會有細微的差別，比如反抗軍速度快、攻擊力強，而聯盟軍防禦高。兩者各有優劣，不過不用猶豫，後面還可以切換過來玩另一個陣營。

遊戲的單位數量不算多，不過卻涵蓋了海陸空三樣，加上單位克制系統，還是將每個單位的特色發揮了出來，比如重型突擊兵可以剋制重型裝甲車和建築，但造價較高，而突擊兵攻擊力雖低，但造價低廉、實用性強，每個單位互相合作，才能達到輸出的最大化。

《全球行動》的基本玩法也遵循經典RTS的步驟，即建造基地、收集資源、培育單位一氣呵成，每項之間聯繫緊密，提升單位數量需要升級指揮中心，指揮中心升級又需要更多的資源型建築，士兵出門偵查又可以獲得額外的資源……以此循環，少一種建築就會制約基地的發展。不過遊戲並沒有固定的資源採集點，只有資源寶箱這一設定，減少了搶佔資源點的樂趣，有點可惜。

在繼承傳統玩法的基礎上，遊戲還加入了”戰術”系統，可以對單位造成額外加成，比如普通攻擊可以在一定時間內加強各攻擊單位的傷害。隨著流程的進行，一共可以解鎖20個戰術。戰術系統算是《全球行動》的一個創新點，不僅加強了遊戲的策略性，更可以拉開與對手之間的差距，讓戰鬥變得更加緊張。

精準到每一個兵的操作體驗

RTS中，面對數十種單位，需要非常精準的操作，因此傳統的鍵鼠是最為合適的設備，轉移到手機端，首先遇到的問題，也必然是操作。這一點，《全球行動》給我的感覺是十分舒服的。遊戲提供了框選和圈選兩種模式，框選就不用說了，就是在屏幕上拉方框選中單位，適用於單位密集的時候；而圈選就不需要有固定的形狀，隨意畫圈就可以選中單位，適合單位零散的時候；另外，雙擊屏幕就可以選中屏幕中的所有單位，非常快捷。

當然，如果只想操作某一個兵，也是可以的，直接點擊屏幕就行。而當你選中單位的時候，巡邏、守衛、停止這些常用的指令便會出現在屏幕下方，便於操作，而不會手忙腳亂，不知道點哪裡。

為了方便管理單位，遊戲還加入了編隊系統，可以將軍隊分成不同的小隊進行多線操作，這個也是傳統RTS具有的。在手機端上，你只需要選中單位，在點擊右上角的編隊，就可以組成一個小隊，同樣指揮小隊也只需點擊編隊就可以一鍵選中，進行操作。可見，《全球行動》並沒有因為是手游而喪失傳統RTS的樂趣，反而能夠根據手機操作的特點進行優化，這點是十分難得的。

總結

《全球行動》目前有3種玩法，單人的戰役模式、多人對戰和排位賽，可惜由於是測試版本，我並沒有體驗到多人合作的玩法和排位賽，蠻可惜的，期待正式上線後，能與其他玩家共同體驗這款遊戲。非常欣喜在RTS不被看好的今天，我們依然能看到有《全球行動》這種玩法經典的RTS遊戲出現，向所有玩家證明了RTS依舊具有魅力！相信它能刺激國內RTS遊戲界，讓未來能有更多優質的RTS遊戲出現在玩家的眼前！
