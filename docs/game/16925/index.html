DNF：“真稀有天空”出現，淺藍色天4外觀，屬性或超越普通天空

2019-12-15 225條評論 2,606次閱讀 1048人點贊

在韓服冬季發布會，很多玩家比較關注100級版本，角色三次覺醒，希洛克團本等等。然而，卻忽視了一套天空，它就是「真稀有天空」，蒼穹之翼的翻版，顏色為淺藍色。不過，玩家更在意的是，在前面加了一個「真」字，那麼，屬性也會上升，跟女鬼劍三覺一樣！而且，值得一提的是，這次「真稀有天空」，韓服發布會免費送在場的玩家！

蒼穹之翼稀有天空，玩家應該都比較熟悉，除了黑色和金色兩種外觀以外，還有其他的顏色，只不過以皮膚裝扮形式存在。之前的時候，魔盒加入了白色天4蒼穹之翼，吸引了一大批玩家，畢竟外觀蠻帥氣的。而韓服這次發布會，卻首次以「真」的形式，出現淺藍色的蒼穹之翼天空套！

我們知道，此次韓服發布會，女鬼劍士三覺空降，技能燃爆了全場，無與倫比的帥氣。但這四位女鬼劍士三覺，前面都加一個「真」字，代表著變強。而淺藍色的天4蒼穹之翼，也帶了一個「真」字，不免讓玩家猜測，屬性會得到提升，甚至將加入屬性強化！不過，一切還只是猜測，這套「真稀有天空」，玩家還沒拿到手。

據悉，韓服開的這場冬季發布會，只要到場的玩家，都能得到一個「真稀有天空」。也就是說，整個韓服伺服器，並不是很多。比較舒服的是，韓服天空套，在拍賣行可以交易。一陣雨在直播時表示，韓服一套克隆天空，差不多120塊就能買到。國服遊戲中，這都不敢想像，後面加一個「0」，都不一定出來！

天空套屬性，已經「按部就班」11年之久，這次的「真稀有天空」，也許是一個突破口，檔次在稍微的提升一下！當然，或許這次「真稀有天空」，就純粹的加一個「真」字，來彰顯其珍稀度，屬性還是跟原來的沒差別。總之，關於「真稀有天空」，玩家們眾說紛紜，各有各的道理，等這套天空發到玩家手中，就能知曉其屬性！
