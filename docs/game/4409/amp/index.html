昨天的1V1 solo賽用德萊文小試牛刀的The shy今天繼續征戰單挑賽，作為目前公認的世界第一上單選手，一向以操作著稱的The shy在本次Solo賽上的表現備受期待！

今年的Solo賽改變規則之後，沒有了補刀和一血機制，而且當前版本由於征服者這個符文的存在對刺客和戰士非常友好，對於The shy來說非常有利！從第一輪的表現來看，The shy在「主場作戰」（大亂斗地圖）顯得也是信心十足！

今天的第一輪16進8面對的是LJL賽區DFM的上單選手EVI，同樣都是上單選手，這次The shy沒有玩花活，直接本命皮膚的劍姬整起，全程壓制了對面的塔姆，最後在縮圈時成功斬殺對方輕鬆晉級！

而賽後採訪和希然的一段「耳語交流」也是引發了大量觀眾的熱議，這段採訪本來看起來沒什麼異常，但The shy這段悄悄話突然氣氛就曖昧了起來，而採訪兼翻譯的希然小姐姐看上去和The shy的氣質很搭！（手裡的雞腿突然不香了）

而8強產生後，緊接著又進行了8進4的對決，這次The shy的對手是G2的輔助選手Mikyx，上一輪掏出一手蠍子成功晉級，這次面對一個輔助選手，shy哥繼續重拳出擊，直接拿出了一手阿卡麗，但是今天的阿卡麗似乎是一個有毒的英雄，Tian和Faker此前在16進8的比賽中都用了阿卡麗這個英雄，但最終都輸掉了！

這場對面竟然選擇了一個龍女，幾乎沒法對線，The shy在一次漂亮的單殺之後，在縮圈前成功推掉了對面的一塔取勝，這也是今天唯一的一場通過推塔取勝的對決，但實際上，兩人的補刀差距實在太大了，即便不拆塔後面龍女也是被單殺！

賽後，The shy再度接受希然的採訪，這一次希然原本左手食指上的戒指竟然不見了，看來這兩人還真有點意思！

「奇怪」的是，這一次採訪雖然時間沒有第一次長，但兩人在交流過程中都臉紅了，莫不是隔著屏幕看到了大家刷的「般配」？

大家覺得，這兩人般配嗎？
