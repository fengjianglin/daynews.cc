黑死病帶給人們的恐慌，死亡和親情#這個遊戲好玩嗎#

2019-12-10 1,698條評論 2,868次閱讀 755人點贊

《瘟疫傳說：無罪》是一款Asobo Studio 開發、Focus Home Interactive發行的動作冒險遊戲。玩家以第三人稱視角控制女主角阿米西亞·德盧恩，遊戲以14世紀黑死病盛行的法國阿基坦為舞台，是一個講述親情的故事。

公元1349年，瘟疫在法蘭西王國肆虐，這場指示歐洲三分之一人死亡的鼠疫傳染病，就是本作的舞台。故事最初，阿米西亞和父親在林子里打獵，陽光從黃色的樹葉中穿出，一切都是那麼祥和美好。

然而好景不長，阿米西亞所在的營地突然遭受了來自宗教審判所的攻擊，不明原因的，這些人想要殺掉阿米西亞的弟弟雨果。母親為了姐弟倆出逃最終慘遭殺害，從此兩個人過上了不斷逃亡、顛沛流離的生活。關於疾病，遊戲雖然有誇大的成分，但整體氛圍營造的極好。被畫上大紅叉的房門、被隔離的區域、被直接處死的人群，帶給歐洲的恐慌彷彿也來到了你的心裡。

遊戲在人物塑造方面非常到位，姐姐阿米西亞智慧、勇敢，雖然在遭受家族巨變時有過震驚和傷心，但為了照顧弟弟而選擇了堅強、冷靜，有時候會讓人忘掉這個女孩也才15歲。弟弟雨果自小患病，和姐姐很少交流，一直和媽媽生活在一起。在逃跑的過程中，雨果經常有不受控制的胡鬧情況，給整個遊戲帶來了很多麻煩，但對於一個只有5歲的孩子來說，一切又都在情理之中，讓人在忍不住破口大罵”熊孩子”的時候，又挑不出一點劇情方面的毛病。

遊戲的畫面細緻，尤其是開場的光影效果，看著讓人非常舒心。而後面場景轉換，破敗、衰退的森林也是那麼觸目驚心。除此之外，人物的建模也做的非常精細，特別是主角阿米西亞跑動中飛揚的辮子，讓人感受到艱難歲月中的一絲青春活力。電影化的敘事鏡頭也給遊戲加分不少，遊戲開頭就可以看出這是想要”講故事給你聽”的。

跟現在流行的開放式地圖不同，《瘟疫傳說》採用的是線性流程模式。這種玩法一定程度上限制了遊戲的可玩性，但對於一部劇本優秀的作品來說，這種模式加深了劇本的流暢感，和遊戲本身完美契合。由於主角是兩個”手無縛雞之力”的孩子，作品主要以解謎和潛入為主，玩家通過道具轉移敵人的注意力得以通過。

除此之外，遊戲中還伴有追逐戰、BOSS戰。阿米西亞手中有一個投石索，在BOSS戰中可以用此作為武器進行攻擊，武器和道具可以進行簡單的改造升級，但由於遊戲的線性流程限制，改造部分的內容不是很多，基本根據遊戲流程就可以完成。

值得一提的是，由於背景設定，遊戲中有一個你避之不及的東西——老鼠群，大群老鼠讓人看得心裡發毛，但你需要運用火把等道具驅除它們，不然不管是因為傳染病，還是因為飢餓，老鼠都會成為你最大的死敵。

遊戲總體節奏快、劇情引人、畫面精美，非常值得一玩，唯一美中不足的就是，線性流程的玩法讓遊戲少了太多的可能性，除非要拿一些成就，不然基本激不起二刷的興趣。另一方面，雖然不能說是短板，但是遊戲的配樂沒有什麼突出之處，雖然在場景運用的時候恰到好處，但沒有給人留下深刻印象的佳作。

總而言之，作為一款動作類遊戲，遊戲難度不大，除了有些壓抑的成分之外，非常適合劇情黨、手殘黨遊玩，可以帶你身臨其境的了解一段歐洲的歷史，親情的部分也壓抑並且感人。

如果不喜歡老鼠的話，那當我前面的話都沒說，趕緊忘了這個遊戲！
