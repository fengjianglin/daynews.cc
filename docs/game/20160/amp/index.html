廠長退役排名反超，LPL顏值圈粉無數，票數碾壓小鮮肉

天天要聞
2019-12-18

在12月7日晚間，各個直播間都被7刷滿了，陷入了7的漩渦中。誰都沒有想到廠長終究是退役了，在S9賽季中一場比賽都沒有上場，卻在賽季末表示仍會堅持下去。可是廠長終究是食言了，EDG官方正式宣布退役，轉型為教練帶隊征戰S10賽季。廠長的粉絲們為了表達內心對於廠長的不舍，也紛紛發力投票，將廠長捧到微博之夜的前5，想要讓更多的人去了解這位老將的存在。

這次微博之夜的投票的確有些許奇怪，誰都沒有想到榜單前五竟然有三個都是電競選手。按照微博之夜的投票規則，基本上都是如肖戰、王一博這類具有顏值的男明星才會佔據榜首。可是這一次的榜首卻是LPL賽區的狂小狗Uzi，雖然這個小胖子從顏值、身材各個方面都是難以與肖戰匹敵，但正是由於LPL粉絲的力量，才會讓小狗霸佔榜首的位置。

尤其是肖戰和王一博部分不理智的粉絲看到這種現象怒斥不公平，認為整個LPL玩家男性居多，只要男性同仇敵愾投一個人，那麼很容易佔據榜首的。但是從不少粉絲的留言來看，Theshy和廠長的排名上升，除去男性玩家居多的事實，更多的因素是因為這些職業選手被了解之後，吸引了更多的粉絲。倘若這些選手沒有一些特殊的原因，怎麼會隨隨便便超過擁有上百萬粉絲的明星呢？

不少路人在看了廠長的照片之後，都認為廠長有些許小帥。職業選手稍微有些名氣之後，便會有造型師、服裝設計師一類對其打扮，本身就長得不錯的職業選手在沒有任何添加劑的情況下肯定是帥過一些其他明星。

更為重要的一點便是這些職業選手背後的故事讓人感動，明星對於粉絲而言更多的是帥氣。職業選手的堅持、為國爭光會在無形之中吸引更多的粉絲。老將廠長的歷史恐怕現在都人盡皆知，Theshy的謙遜態度也同樣讓人喜愛。這或許才是票數碾壓的真正原因。
