我命由我不由天！4AM小鹿實力演繹"鹿頂記"，把4號位打成戰神位

2019-12-20 94條評論 1,325次閱讀 95人點贊

4AM戰隊能夠成為目前PUBG人氣獨一檔的戰隊，跟他們戰隊的成績息息相關，今年一整年來看，4AM無疑是國內成績最好的戰隊，當然，最本質的來源還是隊內選手個人實力都足夠優秀，在今年的比賽中，4AM戰隊一共收穫了3個殺人王！

黃金大獎賽：韋栗子上線斬獲殺人王

今年七月份的黃金大獎賽被4AM粉絲稱為”最難受的一次比賽”，因為這次比賽中，4AM獲得了第2名，跟總冠軍一步之遙，在沒有獲得PCL夏季賽總冠軍之前，不管是粉絲還是選手，對這個首冠都極為渴望，最關鍵是，這次比賽，被認為”4AM最菜”的韋神Carry起來了，幾個打手反而低迷了，比賽結束後，韋神飽含淚水對著鏡頭打招呼的鏡頭也讓觀眾淚目，韋栗子個人拿下了淘汰和總傷害的雙榜第一！

PCM大師賽：王老二終於殺身成神

王總在粉絲里有個綽號，叫”王老二”，因為很多次比賽，他的個人數據總是排在第二，但觀眾們從來只記得第一，FGS倫敦巔峰賽，王總個人淘汰排第二；PCL夏季賽季後賽，王總又排在第二，兩次與殺人王擦肩而過後，粉絲都叫他王老二；不過在10月份的大師賽中，王總終於拿到了第一，拿到了殺人王稱號！

PGC全球總決賽：永遠曇花盛開

PGC全球總決賽，又是一個讓4AM夢碎的比賽，在比賽最後一局之前，他們甚至排在第一，非常有機會奪冠，儘管錯失冠軍，但4AM已經在賽場完全證明自己，永遠的獨狼吃雞，讓全球觀眾為之歡呼，曇花在那一剎那盛開，永遠，世界總決賽殺人王！

而在4AM現在的主力陣容中，僅有小鹿還沒有殺人王榮譽，小鹿的實力固然很強，剛剛加入4AM的首秀（PCL夏季賽）就打出了單局11殺，力挽狂瀾的比賽，鹿頂記一戰成名，現在職業圈關於大E北部的山都以他命名，人稱”鹿頂山”。

小鹿接替的醒目，打隊伍的4號位，斷後、架槍等臟活累活都有他來干，一般來講，4號位就是一個犧牲位，這個位置就註定了選手沒有太多Carry的機會。正常情況下，多數戰隊中，1號位和4號位的選手數據都是最差的（1號位探點，4號位斷後）。

但是在最近的黃金大獎賽中，小鹿的表現卻異常兇狠，連續兩天都是隊內殺敵最多的，在C組的比賽中，更是連續兩局完成一穿三，個人實力演繹得淋漓盡致，第一局，隊友全部都倒完了，小鹿一穿三，第二局同樣是隊友倒完了，小鹿一穿三，4號位選手就是這樣，在打團的時候，隊友都倒完了，最終被迫穿起來。

在隊伍中，1號位白給的可能性會大一點，4號位輸出的機會少一點，但是，也並不是完全沒有機會，機會給到了，把握住了，4號位選手也能打出高光，這才是真的我命由我不由天，不管幾號位，實力足夠強大，都可以打成戰神位，不知道4AM今年，能不能達成”全員殺人王”的成就呢？
