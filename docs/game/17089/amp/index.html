今日，LMS七冠王閃電狼戰隊官宣：不參與明年的PCS賽事。

公告原文如下：

閃電狼《英雄聯盟》團隊自2013年創立以來，在這2,000個日子裡，每次的比賽我們總是戰戰競競，帶著必勝的決心，以最艱難困的意志想擊破無數次的關卡，始得於LMS獲得七連霸戰績。

然而，面對大環境的改變與LMS吹熄燈號，在這樣衝擊下，戰隊歷經數次的天人交戰，在此必須沉重地宣布，閃電狼《英雄聯盟》團隊將不參與2020年PCS事件。

此刻充滿不舍舍與感謝，一切的美好歸功於多年粉絲，贊助廠商，賽事方對閃電狼的支持與愛護，以及在創團隊初期所有曾經​​付出貢獻的管理後勤，成就我們培育出多名優秀的 明星選手；天下無不散的宴席，《英雄聯盟》暫時畫下休止符，我們相信未來將會有相遇的機會。

目前，《英雄聯盟》選手將進行轉隊，《傳說對決》、《爐石戰記》仍會以取得佳績為目標維持運作，未來戰隊也不排除組建其他遊戲項目；再次感謝所有粉絲多年的喜愛，閃電狼依舊會堅持信念！

Flash Wolves 閃電狼職業電競隊

閃電狼戰隊成立於2013年，曾經參加過15-18年的世界賽，七次拿下LMS賽區冠軍。

閃電狼被人熟知的是他們「抗韓奇俠」的名號，自2015年開始，他們就在世界賽場上多次贏下韓國隊伍。2015年世界賽小組賽，他們雙殺KOO；2016年MSI，他們又雙殺去年的世界冠軍SKT；2016年世界賽，他們又在SKT身上拿下自己的世界賽首勝；2017年MSI，對陣SKT 1勝1負；2018年MSI，再次雙殺LCK第一KZ戰隊。

而從閃電狼出來的選手們，很多都在LPL繼續了自己的職業生涯。Maple、SwordArt、Karsa依然活躍在LPL的賽場上，Karsa隨著RNG征戰，拿下兩個LPL冠軍和一個MSI冠軍，而Maple和SwordArt則是參加了今年的LPL全明星賽。

戰馬、牛排兩位教練也都出自曾經的閃電狼，戰馬更幫助FPX戰隊拿下了S9世界賽的冠軍。
