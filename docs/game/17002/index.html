你的CPU夠用么？《實況足球手游-2020》啟發機制下的中場球員推薦

2019-12-14 1,395條評論 3,662次閱讀 1921人點贊

《實況足球手游-2020》本周已經正式公測了！自從新增了啟發機制後，玩家們在中場人員的挑選上就開始犯迷糊了，從前咱們選中場無非就兩種標準，要麼進攻出色的，能夠增加禁區火力點，要麼防守積極的，能夠扮演中場鐵閘的角色。但是現在，中場還要綜合啟發機制來權衡，挑選中場可就沒那麼容易了，不過小編這裡為大家推薦了幾位在新版本中會迎來巔峰的中場（前腰）球員。

盧卡·莫德里奇（盤球突破、地面傳球+、空中傳球+）

魔笛坐擁三項啟發，加上出色的盤帶數值與傳球數值，無疑是天作之合。加上魔笛自己初始攜帶的節能有十個之多，風格也屬於做球的的那一類，所以這一代實況手游里的魔笛還是很強的。

凱文·德布勞內（地面傳球+、空中傳球+）

丁丁雖然比魔笛少了一項啟發，但是傳球屬性與腳下力量都要優於魔笛，並且與魔笛相比，丁丁的進攻屬性更明顯，所以兩人各有千秋，只是在單幹方面丁丁可能不如魔笛來得支援多。

克里斯蒂安·埃里克森（地面傳球+、空中傳球+）

埃神與上述二位相比，最大的優勢在於體力，雖然綜合數據不如魔笛與丁丁，但是體力這一點就可以省去一個替補名額，並且埃神的傳球技能居多，可以彌補他在傳球數值上的不足。

保羅·博格巴（地面傳球+、空中傳球+）

博格巴與上面三位比起來，進攻數據肯定算不上有競爭力，但是他190+的個頭加上93的身體碰撞，在新版本里會是一個很吃香的數據。同時博格巴攜帶的進攻技能也不少，可以彌補進攻數據上的不足。

米林科維奇（地面傳球+、空中傳球+）

與博格巴類似，米林也是勝在模型上，同時全能中場的比賽風格也比較討喜，而其他數據與一流中場也沒有較大差距，加上自身技能能夠彌補進攻數據上的不足，接近完全全能。

總而言之，《實況足球手游-2020》的資料庫在中場方面還算是人才濟濟，而新作更貼近真實賽場的比賽環境（需要調整正確姿勢出球），更適合中場球員們調整後再出球，所以啟發機制也可以看作是應運而生。不管怎麼樣，就算現役不夠我們用，還有傳奇，傳奇不夠用還有精選，精選不夠用還有沒出的新傳奇。

目前，《實況足球手游-2020》已經全平台上線，小夥伴們不妨親自去體驗一下，在啟發機制下與人機隊友打出行雲流水配合的暢快體驗，12月13日~12月15日每晚20：20在線還能領取包含傳奇、精選在內的經紀人喔，精彩不容錯過！
