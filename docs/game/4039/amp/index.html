  • 大心臟

如果是新手，人格夠的話，一定要點上絕處逢生，它會在最後一台電機打開以後回半血，你本來倒地回半血，半血回滿血。

  • 學會壓機

這個點和上方的大心臟相輔相成的，如果隊友在溜鬼，要等到隊友信號再開最後一台機。這個地方我要說一下，很多人一定要等到倒地再開機，其實沒有必要，雙彈或者屠夫被砸之類的都可以，當然具體的還是要等隊友信號。

還有就是隊友在椅子上，不要輕易開機，確定要不要去救，不要等到隊友跑到椅子面前，突然開機，壓好一刀斬的這種事，真的很難受。

  • 常發信號

很多人玩遊戲，不發信號，不知道他在哪裡？在幹嘛？怎麼想的？這個遊戲怎麼玩？就是你受傷了，想去摸人都沒辦法，好多人半血以後也不走，就在屠夫面前晃蕩，然後就倒了，你以為你是前鋒傭兵還是有槍的空軍？

一說起有槍這事我又又想起一事，在氣球上隊友會被打下來的，不要拿著槍一直在旁邊晃悠，等到隊友剛剛被放在椅子上，屠夫動作還沒結束呢，就開槍救人。

後期屠夫如果不追你，你要發信號告訴大家，隊友才好開機或者開大門，第五人格是個一秒鐘就能改變一切的遊戲。

  • 卡血線，爭取時間

現在第五已經改了，之前還看不到血線。隊友上椅，如果機子沒有修夠，一定要卡血線，能給隊友爭取一秒就一秒，如果隊友上掛就沒，盡量換掛，一換一。

  • 各司其職

記得你的天賦，你的人格，你的buff，盡量不要做那種滿血傭兵被其他角色搶救人的事情，有某些低階求生者，也不發信號，等救人位跑到面前就發現她也來了，救完人還不扛刀。

前期盡量讓機子給機械師和盲女，她們去跑圖的話浪費修機進度，不要覺得她們搶你的機子，贏才是最終目的。

一想起來還一堆，入殮師開局和別人一起修機是什麼操作，空軍不救人，修的動機子？前鋒專註修機，那開始為啥不選修機位？
