DNF：10號才開服！狂閃95B史詩，遇到紅腿BOSS平均雙黃起步

2019-12-10 639條評論 1,658次閱讀 550人點贊

地下城與勇士的全新起航版本如遇而至，作為12月的第一次更新，肩負著吸引玩家回坑的重擔，而SPK也是誠意滿滿，各種禮包很划算，當然這是相比之前上架的各種禮盒，以後會更划算的，但玩家們上線第一件事卻不是普雷，而是先去測試「史詩之路」，可沒想到，真的如體驗服掉率一般，高得有點離譜了。

10號才開服！95B史詩隨便閃，遇到紅腿BOSS平均雙黃起步

根據10號開服後的測試，2019的史詩之路最大變化就是掉落的B套史詩比例太高了，無論是刷哪個副本，閃出來的史詩多為95B史詩，也就是超界SS防具，回歸玩家都感嘆，這深淵也太友好了……

開服後玩家們刷SS之路有點兩極分化，問題主要出在隱藏BOSS上，或許不少玩家都不知道有紅腿BOSS，所以沒感受到掉落高！因為這才是會出三黃、四黃的地方。

這一次SPK並未暗改史詩的掉率，活動公告寫明了遇到隱藏BOSS會出1~5件超界SS，事實也如此，玩家們只要遇到紅腿BOSS了，保底1件95B，要這是保底的情況，要是10號更新後只出1件的玩家，那已經算是黑咯。

遇到紅腿BOSS但只閃了一件史詩的玩家，相當於普雷翻出11個黑瞳石！5件是超級金牌界·武器，但2~4件卻並不難，要知道10號才開服，已經有玩家的小號5黃輕鬆畢業！這並不是體驗服，而是玩家們一直玩的正式服，實測爆率就是如此。

這時候再回頭看格蘭之森的測試，小夥伴們還覺得史詩四黃是P的么！這僅是SS之路的第一天，狂閃95B史詩很正常，等刷個一周後，玩家們就會知道背包中的一堆史詩不是獎勵而是累贅了……

因為史派克10號調整不是SS掉率而是材料掉率，本來固定掉2個的權能碎片變成了1~2個，金色氣息也是如此，無形中就讓玩家多花精鍊來刷深淵了。

總結，10號開服後！遇到紅腿BOSS閃出的95B史詩可是平均雙黃起步，但問題來了，小夥伴們一管疲勞能遇到幾次隱藏BOSS呢？
