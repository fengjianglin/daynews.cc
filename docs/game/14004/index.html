承認別人優秀有多難？XDD、韋神再多榮譽，也換不來網友認可

2019-12-11 75條評論 1,829次閱讀 551人點贊

北京時間12月10日晚，第三屆WEGL微博杯落下帷幕，RNG戰隊高開高走奪得總冠軍，「幻神」iFTY拿下亞軍，QM戰隊拿下季軍，大換血的Lstats戰隊拿下殿軍。

在本次微博杯比賽中，RNG「新人」XDD無疑驚爆了大家的眼球，成年復出後的XDD憑藉73殺的成績成功當選本屆WEGL的殺人王，被官方冊封「戰神」，可謂「出道即巔峰」。

眾所周知，自XDD當年直播開始，因其個人技術高超，再加上他本人也愛演外掛，直播間節奏就一直沒斷過，各種「實錘」視頻也讓當時還未成年的XDD倍感壓力，一度在直播間崩潰大哭。

實際上，由於絕地求生外掛橫行的原因，許多職業選手都曾經被質疑過開掛，XDD、韋神以及17shou等選手都有過被藍洞誤封調查的經歷，而這也就成為了許多噴子口中的「實錘」。

老話說說清者自清，雖然官方有時會分不清選手有沒有開掛，但選手在線下比賽中的成績卻能說明一切，對於大部分觀眾來說，不管是韋神帶隊4AM遠征PCG，拿下世界第三的成績，這次微博杯XDD大開殺戒，帶隊奪冠，都算得上是對選手個人實力最好的證明。

但落在鍵盤俠眼裡，這一切也只不過是說明這些選手現在「從良」的現狀，並不能默契其之前「開掛的事實」，甚至有人還因此質疑比賽的公正性，認為XDD奪冠只不過是又一次利益驅使下的商業運作。

其實這種事情也不僅僅在遊戲界出現，前一陣泳壇名將孫楊不也因興奮劑事件飽受困擾？除了當年孫楊在世錦賽失冠之後鍵盤俠們評論「你忘了嗑藥」，而在孫楊被國際仲裁法庭以莫須有的證據污衊時，噴子們又開始「微博斷案」、「知乎尿檢」。

始終有些人寧願聽信道聽途說無中生有，也不願意相信別人的努力和天賦，而那些被鍵盤俠口中「我做不到」、「一般人不會這麼做」等「實錘」所帶偏了的觀眾們，實際上也僅僅是不願意承認這些人比自己優秀罷了。
