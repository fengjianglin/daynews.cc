大家好，歡迎來到《刺激實戰教室》，我是你們的老朋友「奶爸」刺激哥。唉，抱著孩子，就沒辦法寫文章了，真的是太難受了……

這種感覺，真的是有點幸福，幸福的又有些難受。奶粉錢，太不好掙了！廢話不多少了，為大家簡單的介紹一下《和平精英》體驗服的更新內容。

話題：「吃雞」12.5更新，極寒模式正式回歸，遊戲玩法有一些「小改變」

如圖所示，《和平精英》體驗服12.5的更新內容，大致為三點。簡單的敘述一下，就是海島地圖與雪地地圖的聖誕模式、創意玩法極寒模式，以及新武器DBS霰彈槍。

其次，就是經典模式戰鬥方面的一些優化了。經典模式的優化，一共包含了6個內容，其中包含了信號圈顯示效果的優化等一些列的內容。

由於《和平精英》的更新公告，已經羅列出了優化的內容，刺激哥就不再敘述了。

另外需要提到的是，我們盼望已久的「極寒模式」終於回來了。只不過，《和平精英》的極寒模式，與《刺激戰場》的極寒模式，在遊戲玩法上有一些「小改變」。

至於改變的內容是什麼，請允許刺激哥先賣個關子，先介紹一下遊戲內大格局的變化。

首先要講到的是，《和平精英》的12月新版本，為了減少對手機內存的佔用。除了海島以外的3張地圖，都是需要重新下載才可以使用的。需要重新下載的，同樣還包含團隊競技的一些玩法。

另外，這次更新的聖誕模式僅包含海島與雪地這兩張地圖。雨林地圖與沙漠地圖，並未包含在內。

創意工坊新增加的創意玩法，自然是我們所提到的極寒模式了。但除了極寒模式以外，又新增了雪地地圖與沙漠地圖的隊友召回玩法。

個人感覺，創意模式增加這麼多的隊友召回模式，其實是比較多餘的。這等於，間接地把經典模式的玩家，分散到了創意工坊里。

唯一值得一提的是，光子暫定把隊友召回玩法，計入了排位分。至於是否真的計入排位分，我們就需要等到正式服更新以後才知道了。

接下來，我們需要提到的，就是《和平精英》12月新版本，極寒模式的一個變化了。

新版本的「極寒模式」，對凍傷的程度進行了進一步的優化。如：正常體溫、輕微凍傷、中度凍傷、嚴重凍傷四個階段。

在取暖方面，《和平精英》的設定與《刺激戰場》相差誤解，同樣是需要進行烤火、使用暖寶寶進行取暖的。

只不過，《和平精英》在原有的基礎上，又新增加了新道具煤油燈。

在玩法方面，遊戲中又增加了「無人機」，以及雪地滑板的功能。其他的遊戲規則，基本上都是一樣的，只是在原有的玩法上，讓極寒模式變得更加的豐富、以及飽滿了。

另外需要提到的是，我們喜歡的「小動物們」又回來了。只不過它們並沒有擺脫，成為烤肉的命運。

最後，需要提及的是，因為需要帶孩子的原因，外加上家中的老人住院了。

所以，刺激哥在體驗遊戲內容的時候，可能有點片面，沒有太多深入，希望大家可以理解。如果有不當的地方，歡迎批評指正。

好啦，以上就是《「吃雞」12.5更新，極寒模式回歸了，遊戲玩法有一些「小改變」！》的遊戲內容了，更多吃雞遊戲話題，還請繼續關註：刺激實戰教室。
