OMG直接用新陣容打德杯，Smlz首秀就對RNG，能否翻身就看一人

天天要聞
2019-12-19

今年LPL的最後一項官方賽事德瑪西亞杯，將在後天正式拉開大幕。經歷過今年的轉會期腥風血雨後，不少戰隊和一個月之前相比，變化已經非常大了。而德杯將是新賽季開始前，很好的練兵平台，不少戰隊這次都派出了主力陣容。而轉會期陣容變化最大的隊伍之一OMG，今天下午正式宣布參加德杯陣容：上路Curae、打野H4cker、中路Icon、下路Smlz、輔助小五。他們的首戰對手，就是RNG戰隊。

OMG是幾支將以新賽季的全主力陣容打德杯的隊伍之一，這個休賽期OMG幾乎是陣容大洗牌。算上退役的Xiyang，OMG一口氣走了七名上賽季的選手，就連「替老闆擋過槍」的車皇都離隊了。而下的幾人中，Curse是今年新人上單中的佼佼者，Icon則是戰隊核心，而小五、可能就是為了等待Smlz的回歸。

在轉會期成為自由人的馬哥，幾乎沒有和其他隊伍傳出流言，就直接鎖定了老東家OMG。而且還為OMG帶來了SN的打野——H4cker。就這樣，OMG的新賽季陣容便湊齊了，並且相比S9來說，這支隊伍的紙面實力瞬間上去了。

以這套陣容出戰新賽季，OMG或許終於能擺脫自己連續幾個賽季聯盟的墊底的局面。不過這套陣容究竟能不能打出來，OMG能夠翻身，可能還要看一個人的發揮，這就是馬哥帶回來的打野H4cker。

H4cker雖然說在LPL並不算出眾，而且在SN也是背鍋的那位。但是相比OMG這兩年的其他打野卻是比上不足比下有餘，他在年初時表現曾一度還不錯，甚至也和Smlz一起進入過周最佳。從目前這支新OMG的陣容上來看，他可能是最短板的那位。他如果能保持穩定為三線提供幫助，那麼OMG才可能在新賽季實現翻身。而這次的德杯，就將是這支新OMG的首次試煉。
