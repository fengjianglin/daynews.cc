絕地求生黃金大獎賽分組公布！A組真諸神對決 4AM和RNG都在這兒

天天要聞
2019-12-12

隨著RNG的奪冠，絕地求生微博杯也是圓滿落幕，在微博杯結束後下一個杯賽也火熱來襲，12月15日，黃金大獎賽的正賽也將火熱開啟。

此次黃金大獎賽的賽事規則也有了調整，採用了瑞士輪賽事，32支隊伍分為了AB兩組，各組比賽打完後，每組前八名，共16支隊伍分為C組，後八名的隊伍分為D組，兩組再次進行對決，比賽打完後，CD兩組的前八名和後八名再次分在一組，為E組和F組，EF組打完後，最終總積分前十六的隊伍就能夠晉級決賽，就是每支隊伍都要打三天比賽，打完後總積分前16的隊伍晉級！

而在比賽開始之前，詳細的隊伍分組也正式出爐，這次的隊伍分組相當給力，一句話總結就是，A組諸神對決，B組ARK為王！下面就讓飯堂君帶大家看看詳細的分組吧。

首先是A組，這一組的隊伍就全是打架的，微博杯剛奪冠的RNG，還有4AM，QM，iFTY，TSG，SMG還有Lstars都在A組，這真就是死亡之組好吧，跳點重複的隊伍也很多，真就是落地鋼槍打CS了，Weibo心裡苦呀，分在了A組，估計要被阿韋和小海先後軍訓了（狗頭）。

相比A組的諸神對決，B組就相對輕鬆很多了，ARK，VC，FPX，Tianba都在B組，從微博杯的表現來看，北境之王ARK在B組就很舒服了，而且VC，Tianba和FPX還有VG，壓力也不會太大，晉級還是很穩的。

而正賽首日開打的就是A組的對決，第一天就是重頭戲，而且還是周日下午五點開始，各位網友可千萬別錯過呀！
