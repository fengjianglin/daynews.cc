魔獸世界懷舊服：1.13補丁登場，戰場來了，付費轉服好像是場布局

2019-12-11 427條評論 2,867次閱讀 591人點贊

玩遊戲找熱點，點擊關注不迷路

魔獸世界懷舊服1.13版本已經在美服正式上線了，而國服也即將在12月12日更新這個版本。通過暴雪放出的藍貼可以看到這次更新魔獸世界懷舊服1.13版本主要包括以下幾個方面的內容。

戰場

戰場首先開放的是戰歌峽谷和奧特蘭克大戰場。

戰歌峽谷是一個10v10奪旗模式的戰場，一共分為6個等級區間10-19級、20-29級、30-39級、40-49級、50-59級、60級。

奧特蘭克是一個40v40的大戰場，所有51級以上的玩家都可以進入，並且有許多的任務、物品以及有意思的區域。並提供聲望和任務獎勵。部分獎勵物品還是非常不錯的，比如胡里奧指環、生命花環、冰線箭矢以及冰線槍彈等等。

最重要的是這些戰場都是跨服的，備受單陣營困擾的玩家也終於可以體會到真正的PVP了。而在魔獸世界懷舊服黑翼之巢正式開放之前的這段時間，應該是屬於全民戰場的歡樂時光。

元素入侵

熾熱入侵者、雷霆入侵者、水流入侵者和旋風入侵者出現在卡利姆多的各個區域，安戈洛環形山、艾薩拉、冬泉谷、希利蘇斯等地都能找到他們。目前魔獸世界懷舊服中，除了火焰精華之外，水之精華、大地精華以及空氣精華的價格都比較高，因為其他精華的產出比較低。這次元素入侵雖然不能夠徹底解決奇貨可居的局面，但是普通玩家們還是有機會去刷一些材料的。

鑰匙鏈

千呼萬喚的鑰匙鏈功能也終於在1.13版本加入到了魔獸世界懷舊服當中，最多可以存放12把鑰匙，這也緩解了各位小夥伴背包緊張的問題。

插件限制

這次更新過後暴雪調整了部分LUA API介面，阻止某些插件濫用社交頻道的問題。在此之前玩家們已經見識到了在懷舊服中用新版本插件導致聊天延時，比如榮譽統計、團隊活動日曆等等都會被限制，更新過後很多發送到聊天頻道的消息會被阻止。

付費轉服

在這次1.13版本補丁更新中，暴雪提到了未來會開放付費轉服功能，對於很多飽受單陣營困擾的玩家來說肯定是個好消息。但是小編覺得這好像是暴雪下的一盤大棋，那就是把營收利潤做大化的一個操作。在當初懷舊服上線的時候由於玩家非常多排隊現象嚴重，暴雪開啟了定向轉服，定向轉服過後就出現了大量的單陣營鬼服，而且暴雪各種造勢陣營不平衡的聲音也是不絕於耳。

而此時隨著1.13版本的上線也非常「巧妙」的宣布即將開放付費轉服功能，讓很多玩家心甘情願的掏這份錢，小編只能說暴雪的營銷做得不錯，付費轉服開放之後暴雪又能大賺一筆。

這次更新中暴雪限制插件的思路是對的，畢竟很多玩家還是想玩原滋原味的懷舊服，而不是像PVP玩家一旦開了插件200碼之內就能發現敵情這種破壞了原滋原味的東西，玩家們想要的是純正的野外PVP。
