《英雄聯盟》奧恩六級傷害究竟有多高？

天天要聞
2019-12-11

作者：NGA-tylonl

最近幾個月來一直有壇友力薦奧恩上單 逆版本錘戰士，用過的都說好

而在9.23征服者改版之後，奧恩作為傳統坦克英雄地位再度提升，在韓服已經達到t1級別

相信近期玩過或者對線過奧恩的選手都能切身感受到這個英雄傷害極其離譜

那麼常規符文下奧恩的傷害能力究竟在什麼水平呢

這裡就簡單計算一下奧恩在6級情況下幾種基礎連招的傷害

​

靶子依然選擇德萊厄斯和弗拉基米爾作為戰士和脆皮的代表，符文出裝都選opgg上最流行的一套，因為是對抗奧恩所以第三個符文碎片選擇+8魔法抗性。

奧恩也選用最流行的不滅之握+副系啟迪配置，出於保守考慮，我們假設奧恩除了出門的多蘭盾外沒有打造任何裝備，並且沒有從不滅之握中獲取任何生命值。

對兩位靶子選手的技能傷害如下：

​

​

可以看到，奧恩在有不滅的情況下一套WA接Q減速後撤的換血對一般英雄的傷害就達到了35%~40%，QRERAWA的理論完美連招斬殺線約為75%~80%最大生命，如果對方用閃現或者位移躲避了第一段R和E，導致只能打出兩次易碎的話，斬殺線則降低到了55%~60%，並且這一情況下並不能做到在全程擊飛狀態下打完傷害，考慮到對方可能擁有的征服者和恢復技能等，實際斬殺線還要下降一些，此外在出手之前最好想辦法破掉骸骨鍍層，前期3層骸骨鍍層都被觸發的話差不多等於一個10%生命的護盾了。

除了不滅之握外，在一些不容易蹭不滅的場合奧恩也可以考慮攜帶引燃+啟封的秘籍，這樣連招的斬殺線會大幅提高，完美連段的理論斬殺線會提高到87%~95%左右，第一段R和E無法命中時則為67%~73%左右，同時引燃對回復也可以起到一定克製作用。

奧恩在6級時只打出兩次易碎的情況下總傷害接近沒有打出強化E的9.20巔峰蓋倫，打出三次易碎滿傷害的爆發能力則比打出強化E的蓋倫高100~170點左右(但相比蓋倫，奧恩受到骸骨鍍層克制更加嚴重，並且普攻能力和征服者英雄還有較大差距)。
