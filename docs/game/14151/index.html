DNF：生命力最強的三把武器，號稱永不過時，帶到兩百級都沒問題

2019-12-11 22條評論 1,015次閱讀 89人點贊

聖耀強嗎？不能升級還不是淘汰了；蒼穹武器強嗎？比起夜雨黑瞳怎麼樣？現在夜雨黑瞳強啊，可是你敢說未來一百級的畢業武器會比夜雨武器弱嗎？這肯定是不可能的。任何版本畢業的武器，無論你多麼強，都躲不過時代大潮的沖刷，強如星際毀滅又怎麼樣。但是，有這麼一些武器，它們就很難過時，永遠地活在版本的最前沿，傷害不高，但是大家都說好！

一、魔劍·阿波菲斯。

魔劍必須強十三！六十版本的魔劍是在無影劍之前最強的一把鬼劍士武器，魔劍厲害不僅是傷害，還有它拉風的外表，還有它稀有的程度，在零九年，魔劍剛剛問世的價格大概在三千塊錢左右。這玩意本身就是個粉武器爆的很少，還只在悲鳴三圖的魔劍處掉落，堪稱稀有中的稀有。現在很多玩家都喜歡幻化一把魔劍，情懷是一方面，好看那也是真的。

二、流星落。

流星落原名流光星隕刀，叫流星落大概是武器特效能召喚隕石吧。這把五十級的粉太刀當年問世的價格也是幾千塊錢，現在名氣不如魔劍，外觀一點不輸。很多人想幻化血舞旋月刀，其實流星落和血舞長得就差不多，區別一點都不大。拍賣行魔劍的價格都是幾千萬的，而流星落只有一百多萬，對於喜歡長太刀的玩家來說簡直就是福音。記住不要強化哦，流星落和黑刀一樣，強化了就不好看了。

三、光炎劍·烈日裁決。

光炎劍好看嗎？從外觀來說，並不好看，太短了！光炎劍的厲害之處是，它只裝老玩家的B，只有在六十版本PK場經歷過光炎劍制裁的老玩家才對這把武器有非常深的印象，光炎劍早已絕版，現在可以在靈魂之冢的罐子里開出來，當然了那幾率也是非常感人的，作為一把會發射光炎的武器，它一直就是一個傳說，無可替代，無可比擬！

所以說呀，武器的生命力和它的傷害並不很有關係，只有好看才是永恆的，就像那些把地下城當奇蹟暖暖玩的玩家一樣，只要看著順眼，什麼傷害，什麼打團，那都無所謂了。

如果今後策劃一直做不出特別好看，特別驚艷的武器，那麼像魔劍，流星落這些武器，就會一直活躍在遊戲中，永不掉色。
