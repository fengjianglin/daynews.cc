海外媒體評各位置世界前五：世界第一全是LPL選手，doinb沒上榜

天天要聞
2019-12-18

距離S9世界賽結束已經有一段時間了，不少LPL戰隊為了馬上要開始的德瑪西亞杯，已經開始在努力備戰，準備新的賽季了，那麼在快要結束的S9賽季，五個位置中世界前五的選手分別是誰呢？就在12月18日，海外媒體在推特上給出了自己的觀點，引發了不小的熱議。

海外媒體評選的每個位置前五如下圖所示，其中25個選手中，LPL選手佔了13席，而且每個位置的世界第一都給到了LPL選手，世界第一上單theshy，世界第一打野tian，世界第一中單Knight，世界第一AD是UZI，世界第一輔助劉青松。

​首先上單這邊Theshy是沒啥爭議的，爭議的點在於TES的上單369不僅進榜了，還排在了僅次於theshy的位置。主要原因也是369在幾次關鍵比賽中的發揮都有點掉鏈子，比如LPL的季後賽和冒泡賽，又比如說之前的洲際賽，目前來看369還沒有證明自己具有穩定carry的能力，需要下個賽季新TES陣容出來後，再來看看他的表現是否能夠打服大家的心吧。

​打野這邊LPL三位選手入選，分別是tian、Karsa和國豪，這三位目前的確是LPL最強打野，不過馬上開始的S10賽季中，又會有不少新人打野亮相登場，不知道S10過後，是否會有新人取代他們呢？

中單這邊也有爭議的，首先世界第一中單給到了knight，這如果是S9世界賽之前的排名，可能還沒有那麼多質疑聲，但是S9世界賽上TES沒能拿到名額，而且knight在冒泡賽最後一輪面對Rookie的時候啞火了，也是事實，雖然knight的Rank很給力，但是目前也是缺乏在關鍵比賽中證明自己的能力，似乎一到生死局就容易隱身。

​還有就是S9世界冠軍中單doinb居然沒有入榜？doinb的打法在S9世界賽上得到了淋漓盡致的展現，也許他不是對線能力最強的中單，但一定是給團隊最多支持和幫助的中單，doinb的打法可以說讓中單有了更多的可能性，不再是單純的carry。沒想到拿到了世界冠軍也不夠，仍然得不到海外媒體的認可。

​ADC位置沒啥爭議的，UZI、JKL和LWX的確是目前LPL最強的三個AD，唯一爭議的就是UZI到底是不是世界第一AD，不過這個話題仁者見仁智者見智，已經被說爛了，就不多廢話了。

最後輔助這邊劉青松和ming入選是意料之中，沒想到Lvmao也入選了，綠毛的優勢在於有很多騷英雄可以拿出來，劣勢在於要麼節奏爆炸好，要麼節奏帶到隊友爆炸，穩定性不夠。不過作為LPL的輔助，也可以排在前幾位了。

​那麼，對於這份海外媒體的榜單，你們有什麼想說的嗎？
