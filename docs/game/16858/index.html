暴走大俠：大佬紛紛退游，iOS玩家還有望等到更新嗎？

2019-12-16 384條評論 1,485次閱讀 335人點贊

氪金不暴走，大俠全靠肝，大家好，我是「爆肝大俠」小魚乾，歡迎來到由小魚乾開講的《暴走大俠新鮮事》。本期小魚乾來跟大家說一說今天的吐故納新活動，加上閑聊一下。

今天萬眾期待的神技「吐故納新」終於是上架限購了，很多玩家也是毫不猶豫地入手了，而已經入手過的玩家也是購買了殘卷進行升級，雖然秘籍的價格還是1888翡翠葉，但不得不說這一波的殘卷還是很划算的。

小魚乾看到50殘卷的價格是658翡翠葉，很多玩家看到的也是這個價格，那麼算一算，平均就是每個殘卷13.16翡翠葉，而之前活動的殘卷大概是17翡翠葉一個，比現在貴了不少，今天入手的確是至今最划算的一次了。

但有玩家反映，看到的殘卷價格，居然是788翡翠葉！這名玩家看到價格就很生氣了，為何區別對待玩家呢？難道每個玩家的價格都是不一樣的嗎？甚至還懷疑暴走大俠官方暗改價格！

但事實上，這名玩家犯了兩個錯誤，第一個就是，確實會出現不同玩家價格不一樣的情況，但不一樣的原因是因為玩家的等級不同。

第二個錯誤，就是這名玩家沒有算清楚價格就來質疑官方，788翡翠葉可以獲得的是60殘卷，平均每個殘卷就是13.13翡翠葉，如果非要較真，這比658翡翠葉50個殘卷還要划算呢！

所以很多玩家都選擇在今天，把存了許久的翡翠葉，全部用來購買和升級吐故納新了，畢竟這個招式應該算是目前的闖關神技，升到滿級之後，真的是誰用誰知道，實在是太好用了。

有升到滿級的玩家也發出了吐故納新滿級的屬性圖，恢復105%攻擊力的血量，還能附加40000血量，小魚乾攻擊力約為3W，氣血9W，如果吐故納新升到滿級，大約就能恢復12W血量，已經超出了最大氣血，這還是沒算裝備、招式、技能加成的情況下。冷卻時間也快，打一次BOSS可以用兩三次，打世界BOSS更是無敵！

但也有玩家覺得，今天沒有累計消費的活動，感覺還是有些不值，但小魚乾也想說，暴走大俠官方可精著呢，能讓你又拿吐故納新又拿消費獎勵嗎？不存在！

根據之前的預告，17日~19日會有累計消費活動，而17日的限購禮包，不出意外就是紀元武器和寶石箱子，所以這個空子是鑽不了的！

另外小魚乾也說一說今天一些玩家遇到的一個問題：就是組隊的時候，有時會出現「連接戰鬥超時」的提示，一點確定，就直接彈出去退出了，這可怎麼解決呢？

雖然小魚乾已經很久不組隊玩了，但詢問遇到這個問題的玩家之後，還是得出了解決辦法，就是不要點確定，等待一會，可以隨意切一下屏，或者等隊友進入下一關，過一會就好了，反正不要點確定就對了，否則會退出去。

另外也有玩家詢問小魚乾，iOS版本到底什麼時候更新，其實小魚乾也不清楚，但小魚乾也要給iOS的玩家打個預防針，現在暴走大俠已經下架了，很有可能是無法再進行大更新，這個小魚乾在之前的文章已經給大家詳細解釋過了。

小魚乾也聽說很多iOS的玩家退遊了，還是那句話，勸iOS的玩家暫時不要氪金，當然土豪隨意啦！

那麼以上就是本期《暴走大俠新鮮事》的全部內容啦！最後，感謝您的閱讀，各位路過的大俠給小魚乾點個關注吧~
