魔獸世界懷舊服：術士如何在MC打DPS？想打就這麼做 但最好別實踐

天天要聞
2019-12-21

魔獸世界懷舊服MC被大家征服之後，每天無數公會都在組織人手去MC旅遊，但是去到MC的團並不是都能全身而退，甚至滅團都是家常便飯。下面984G十方（筆者）要說的是前者，全身而退之後，有些職業就想玩一玩「花活」了，比如術士如何在MC打出DPS？術士在MC中如何搶DPS？下面我們就來聊聊這件事。

術士在MC的作用

術士想要打出DPS第一名，就意味著犧牲很多對於團隊的貢獻，簡單說來就是不能出小鬼，不能一直吸碎片，不能上元素詛咒，最主要的是容易ot。所以就一般團隊打MC來說，團長對術士的要求就是放逐好怪並且不要ot就可以。曾經遇到過一個公會打MC的時就一個術士，定小鬼、發糖、放逐、元素宇宙都做到了，並且術士的傷害常常可以排在第四第五，而且還是不吃藥情況。別著急，984G十方還有後半句沒說，他們打一次要四個多小時，正因為打的慢，所以術士偷傷害才偷那麼多。

術士要打DPS的關鍵點在哪？

目前魔獸世界懷舊服平心而論，真的沒什麼輸出手法和操作可言，最大的區別也就是裝備和天賦上的差距。如果真的想要偷傷害，準備兩個SS，一個只放暗影ZZ，另一個專心放痛苦和腐蝕，此時如果能再找到一個治療奶術士保證全程分流。這樣的結果就是哪怕術士只有一百法傷，最終效果也會與300法傷的法師打個平手，這正是有些無良G團用所謂「不到第一60%不分G」的規矩黑G的玄機所在。

如果真的想打，那就這樣做好準備

在此借用術士大佬自由之狼的術士配置，衣服冬夜，手惡魔布，披風護腕暗影懲戒，腿天空-黑下5號，戒指C女，之所以用C女不用地獄指環是因為術士要自焚偷傷害，魔杖蒼白-STSM斯庫爾，飾品用黑下將軍信換的視頻以及厄運書換的飾品。此外合劑需要暗影之力藥劑，龍頭buff，強效奧法藥劑，厄運buff，費伍德輕歌花。大藍20組，大紅1組，惡魔符文（暗黑符文）1組，靈魂碎片70+（3惡魔布包），進本每個治療給兩組大藍，螺絲當前分組治療給3組。

接下來再這樣操作

單目標時起手腐蝕，接下來接暗箭、暗影詛咒、暗箭、暗灼、暗箭數個，再補腐蝕。暗箭如果出暴擊就立即暗灼+死纏，暗灼用來收小怪，boss戰的時候卡好CD再扔。小於3個目標的時候，對非集火目標用痛苦腐蝕，集火的目標都按照單目標處理。大於3個目標就召喚胖子，在跑位時上腐蝕，到達自焚範圍後吃胖子自焚，至於加爾下火雨就可以，如果直接跑上去自焚容易倒地。術士想打DPS就這麼做，但最好別實踐，因為需要的配置需要很豪華。

寫在最後

984G十方聽過魔獸世界懷舊服這樣一個說法，那就是在不倒地的情況下，術士法傷高於一切。因為法傷可以增加你血換藍的數值，為什麼看別人換一次就可以打出3發暗影箭，而你卻只有2發？這個差距就是在於法傷上。
