陰陽師：為何玩得好的玩家自稱萌新，“潛規則”還不懂就太遲了

2019-12-11 3,570條評論 4,725次閱讀 3347人點贊

在遊戲中我們經常會看到一些滿級（60級）的玩家在公屏上自稱萌新，但一看其空間卻掛滿了金燦燦的稀有式神，而鬥技分也在2400以上。對於初入遊戲的玩家來言，下面這條潛規則一定要懂：越NB的玩家就越會稱自己為萌新。

萌新大佬傻傻分不清楚，有時候上自稱大佬的黑車時，其車極不穩。而又有時候看到萌新求師傅，收完徒後立即收到大舅手辦皮。很多剛進遊戲的玩家，在萌新和大佬之中很難區別。大家就好奇，為何陰陽師中有這樣一種奇怪的風氣呢？下面就來簡單分析一下。

原因一：陰陽師培養系統健全，式神陣容龐大。

不少玩家在肝出第一個6星後才知道，刷狗糧多麼痛苦，但其實將式神培養到6星只是第一步，還需要將式神的技能升滿，將式神的御魂配好，不然只是一個6星式神根本沒用。但陰陽師中有多少式神呢？除掉沒用的N卡和呱，聯動式神、SP、SSR、SR、R加起來近130隻，其中強勢式神就達到了40隻左右。

因為該遊戲並不看單個式神的強度，而是看一個陣容的強度，尤其是在PVP中。因此培養式神一般都是以陣容為核心來培養。當然，如果為了快速養成陣容而瘋狂刷狗糧，可能式神是練出來了，但還要面臨黑蛋短缺和御魂不足的情況。

目前一般玩家的練度是擁有20隻左右6星，10隻6星以下者則是「真萌新」。

原因二：氪帝、歐皇、肝帝大有人在

氪、歐、肝，如果想要在遊戲中有所造詣，則必須具有其中的一項。氪帝在活動時瞬間買幾十個648禮包可能只是人家的一頓早飯。而肝帝則可能以一天睡兩小時的姿態刷副本。歐皇就不用說了。

更多的玩家其實這三類中什麼都不佔，只是普普通通的遊戲，每日簽到後做做陰陽寮的任務。因為這個遊戲也已經持續了3年，即使保持這樣的狀態，也將會有四五十個6星式神。對於一般的副本都有對應的陣容。在鬥技方面，因為式神總類很多，戰績也不會差到哪去。

當玩家練度上去後，遇到的大佬也越來越多，越發覺得自己的練度就是渣渣，有些自愧不如，就將自己稱為了萌新。更多的是對自己「非洲」血統的無奈。
