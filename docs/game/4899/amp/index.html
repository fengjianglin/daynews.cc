王者：誰能線上穩壓米萊狄？甄姬難打，秒回1000血的站擼法師完勝

天天要聞
2 週 ago

摘要

在王者榮耀中，我們經常會遇到一些不強但是卻非常噁心的英雄，要麼是線上壓制力強，像是楊玉環，要麼是清線快推塔快的，像是米萊狄等等。

在王者榮耀中，我們經常會遇到一些不強但是卻非常噁心的英雄，要麼是線上壓制力強，像是楊玉環，要麼是清線快推塔快的，像是米萊狄等等。其中，米萊狄就是典型的不強但是卻非常噁心的英雄，對線壓制力非常強悍，原因就在於他的小機器人，一般的法師前期根本治不了，本來法師前期就缺藍，然後在加上米萊狄的小機器人，你不打吧，太煩人，你打吧，太耗藍，所以遇到米萊狄我們就很難處理。而且米萊狄還對中單的限制非常大，一般中單都是需要遊走的，但是有米萊狄在，想遊走？那要先問問你的防禦塔還想不想要。所以，在對面選了米萊狄之後，我們要怎麼治她呢？

有人會說，法師中甄姬挺好打米萊狄的，但是這也只是存在於理論上，我們都知道，甄姬的二技能清線能力比較好，但是前期甄姬在米萊狄的手上卻討不了好處。一個是頻繁的用技能清線，會極度缺藍，而且一旦回家，就有掉塔的危險，非常難受；另一個就是甄姬的二技能雖然強，但是它彈射的單位有上限，最多是6次，而米萊狄在清線的時候，放的機器人估計都有至少4個，所以甄姬的一個二技能就算是打到了小兵，很難打掉機器人，打到機器人，很難再打到小兵，最終還是要等小兵進塔，讓防禦塔被消耗血量。所以甄姬對線米萊狄其實並不好打。

要說法師裡面真正吊著米萊狄打的就只有周瑜一個英雄，無論是理論上的壓制，還是實際的對戰，周瑜都是完爆米萊狄的。我們都知道米萊狄最大的優點就是清線推塔，而這兩個有點周瑜都具備，米萊狄放多少機器人，周瑜就能清多少，因為周瑜的火是能夠蔓延的，一旦蔓延開來，不管是兵線還是米萊狄的機器人，都會在周瑜的火圈之內，這樣，米萊狄最大的優勢也就蕩然無存。

同時，周瑜在S17賽季還是可以玩的，屬於一個隱形的強勢英雄，但是熱度並不高，主要是因為這個法師沒有位移，前期很容易被針對，很難發育。不過，我們玩周瑜的時候，完全可以選擇半肉裝備，這樣刺客就秒不掉他，而戰士是不可能打得過半肉的周瑜的，在周瑜的火圈裡面，他就是主宰。而且現在是射手的版本，恰巧，周瑜又很容易針對射手，團戰的時候，就算是不能直接摸到射手，放個火圈，然後風一吹，火圈的範圍就非常大，等於說直接限制了射手的走位和輸出。

而且周瑜站在自己的火圈上面是能夠回血的，只要不被秒殺，周瑜瞬間就能夠將自己的血量給回復起來。而周瑜的變態回血能力是要依靠兩件裝備的，那就是吸血書和不死鳥之眼，出了這兩件裝備之後，周瑜只要站在自己的火圈上面，秒回1000血都不成問題。同時周瑜也是一個比較依賴技能CD的法師，最契合的物抗裝就是極寒風暴了，同時可以增加自保能力和生存能力，所以我推薦周瑜的裝備為：法穿鞋+面具+吸血書+不死鳥之眼+極寒風暴+法穿杖。這套出裝的周瑜的血量輕鬆超過一些戰士，特別是打一些爆發不是很高的陣容，可以說是站那打都行。
