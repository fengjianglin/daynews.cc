王者小課堂（榮耀戰力規則與榮耀排行榜介紹

天天要聞
2019-12-23

榮耀戰力，一個無論是什麼段位的玩家都十分在乎的東西，他也象徵著除了段位以外的第二個榮耀，一個英雄榮耀戰力的高低代表了玩家對這個英雄的了解和技術的多與少，而讓自己的本命英雄衝上國服榜單，那想必是每個玩家心中都有的一份期待與夢想。

當然了，既然說了是夢想，自然是很遙遠很難實現的，不過我相信只要努力遲早有一天會在國服榜單看見你們自己的名字，那麼為了讓大家更早的實現這個夢想，我就來科普一下榮耀戰力以及榮耀榜單的一些規則和技巧吧，等你們把這篇文章理解透徹再摸索一下技術，想必沒有國服也能個市級省級。

再說榮耀榜單之前，我們先來了解一下榮耀戰力的獲得方式，它分為三個部分，（歷史戰績+歷史表現）*常用係數，我們先來介紹一下歷史戰績。

歷史戰績（戰績分）: 戰績分就是上圖的近期戰績，它可以說是榮耀戰力中最好打的分了，上限是3200分，到了3000分加的就很少了，基本只要正常打打，都可以打到3000以上，不用刻意去管。

歷史表現（表現分）: 歷史表現就是大家常說的表現分，也就是上圖的本局表現，表現分的上限是按照你排位的段位來決定的，而表現分加分的情況要按照三種評級來決定:評級預期、評級實際和平均評價，說簡單點就是把你之前遊戲的平均分和這一把遊戲的評分做比較，評分比平均分越高你加的表現分就越多。（另外表現分只有打排位才能加）

常用係數（榮耀係數） : 榮耀係數對榮耀戰力並沒有什麼太大的影響，只要你不是長期不玩這個英雄那榮耀係數基本就不會變，而且即使榮耀係數掉了等你哪天再打依然會漲回來，所以不用太在意這個。

巔峰分（巔峰戰力）: 巔峰戰力是巔峰賽加榮耀戰力最主要的方法，可以看到每100巔峰賽分獲得的榮耀戰力上限為400，也就是說如果你在1200分巔峰賽中把一個英雄的榮耀戰力刷到了400，那麼你在突破1200分之前這個英雄就無法再加榮耀戰力了，按上圖我的加分來看我在1600分段還有109的榮耀戰力可拿，109分刷滿後我再打小喬就是贏了不加分輸了扣分的情況。

巔峰係數 : 巔峰係數與巔峰分一樣是只能靠打巔峰賽獲得的分數，不過它與巔峰分不一樣的地方在於他是全英雄的榮耀戰力加成，也就是說你贏一把巔峰係數加了你每個英雄都可以加榮耀戰力，輸了同理，另外你的英雄榮耀戰力越高巔峰係數扣的與加的也就越多，我上圖小喬是8000的榮耀戰力，巔峰係數贏了1%就是加80榮耀戰力，甚至比巔峰戰力都加的多，由此可見它的重要性，而他的獲得方式是按照你的巔峰分數來看的，巔峰戰力每20分加1%的榮耀戰力，如果你的分數沒達到加成的分，那你一把也就加個幾十的榮耀戰力，所以有很多人都以為是自己戰績打的太差了，導致一把加的分數很少，其實不然，只是因為你的分數沒到戰力加成線而已。

榮耀戰力的獲得方式大概就是如此了，接下來我們來了解一下榮耀榜單，也就是大家熟知的區，市，省以及國服榜單，我們先來了解一下除了國服以外的其他三個榜單。

三個等級的榮耀排行榜是按照你手機設置里的王者榮耀位置從而進行定位的，每周一早上五點刷榜，下圖右上角的設置點開就會顯示你所在的位置，只有每周一可以定位，定位完後就是下圖的樣子，橙色的戰區就代表你定位的所在地，以後你的榮耀排行都是以此戰區為準。

想上榮耀排行榜說難也不難，說簡單也不是很簡單，除去小號以外的玩家基本都得到星耀左右才可以擁有一個區級牌子，到了王者低星才勉強能拼個市級牌子，至於省級不打巔峰賽基本上是拿不到的，而除了提高自己技術之外想要獲得牌子就只能靠改戰區來實現了，我的號就是之前改的戰區，調到了大興安嶺導致市級特別好上，不過黑龍江省也比江蘇簡單不到哪去，所以改戰區要慎重考慮改哪兒，如果只是想沖個區級市級那隨便改個人比較少的地方即可，如果想沖省榜甚至省一那就得著重考慮了，你得先確保你要改的那個省市沒有你沖省一的英雄上國服。

（我的號小喬目前是省一，主要原因是黑龍江暫時沒有國服小喬，而江蘇省一是國服，所以改回江蘇我基本只能做個省2，如果有志想沖省一的玩家你得先在你要衝的英雄國服榜上找一下，確保那個省市沒有國服，否則你最高只能當省2，如果是想沖國服那你在哪個省都一樣，國服還是那個分，不會因為省市不一樣而改變）

（類似台灣這種地方改的人可不止你一個，最好選沒人知道的省市改戰區）

（改戰區的方法網上一堆我不介紹了，不過有些應用登錄王者會顯示第三方平台登錄導致封號，改之前先拿小號試一下）

國服榜與其他三個榮耀榜不一樣，為一個月一評，每月一號早上6點結算，8點發標，在那一天你看國服榜會發現很多都是隱藏玩家，就是很多國服怕被查分導致偷榜，除此之外，國服分為國1到國10，左上角第一個為國一，左下角為國5，以此類推，而且國服並不是只有10個，而是40個，微信，QQ，安卓，IOS，4個大區每個大區都有10個，這也是為什麼很多玩家疑惑國榜上沒有一些主播名字的原因，只是因為他們和我們不在一個大區而已。
