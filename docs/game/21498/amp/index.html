共由說《逃離方塊：生日》銹湖系列入門佳作 巧設懸念且劇情緊湊

天天要聞
2019-12-19

逃離方塊（Cube Escape）系列簡介

Cube Escape，也就是俗稱的逃離方塊，也有玩家把這一系列遊戲稱為方塊逃脫。這一系列由荷蘭RustyLake所開發的密室逃脫類遊戲一經上線之後就引發了不少玩家的討論，不管是湖中逃脫還是四季亦或是悖論，這些版本在一時間內都被玩家津津樂道。銹湖系列遊戲用它獨特的畫風以及新穎但略顯獵奇的世界觀給了一眾玩家全新的密室遊戲體驗。

但在優質密室逃脫類遊戲收穫好評的同時，有一個問題也是它們避無可避的，那就是”難度”。比如此前比較火的迷失島，部分玩家反映不看攻略根本就玩不下去；而從另一個角度來說，遊戲太簡單也會讓人索然無味。所以如何去平衡這兩者之間的關係毫無疑問成了不少策劃的難題，不同的密室類遊戲之間也被這個因素或多或少的影響。

而我們今天要講的《逃離方塊：生日》就是一款在這兩者之間平衡還算不錯的遊戲，本文部分內容涉及劇透，如果還沒有玩過小夥伴可以先去下載嘗試一番。OK閑話不多說，下面我將從這款遊戲的難度、觀感、優點這三個方面為大家帶來一個簡單的測評。

劇情緊湊，不管是萌新還是老玩家都能有一定的解謎思路

對於一個帶有恐怖元素的密室逃脫類遊戲來說，劇情毫無疑問是核心。一方面玩家在不斷解謎的過程中將直面遊戲劇情，劇情的好壞以及是否連貫將直接影響玩家的遊玩體驗；另一方面，優秀的劇情設定可以推動玩家去思考，對整個遊戲的節奏起到正面作用。所以在聊《逃離方塊：生日》的其他優點之前，我們首先來說說劇情。

那麼它到底講了一個什麼樣的故事呢？簡單來說就是小男孩回到了自己小時候的一次生日派對，父母爺爺都在場。但隨著生日派對的進行，一個兔頭匪徒手持湯姆遜，也就是吃雞玩家比較熟悉的芝加哥打字機，強行闖入了他們家中將所有大人全部KO。而小男孩，也就是玩家需要尋找線索一步步修復儀器，最終得到一個方塊讓時間倒流並改變命運。

那麼我們把這個故事按照遊戲章節來梳理一下，就能發現其劇情設計的精妙之處了。

小男孩生日→一步步觸發劇情→兔子闖入滅門→小男孩尋找修復儀器的線索→修復儀器得到小方塊→將方塊放入時鐘內，時光倒流→找到解決辦法，聯合爺爺→先手放倒兔子，拯救家人。

這個劇情精妙在哪裡呢？第一點自然是完整，邏輯線清晰；第二則是矛盾非常明確，玩家要努力的方向也非常清晰；後半部分的懸念好評，反轉結局令玩家舒適。所以不管是已經是密室類遊戲老手的玩家還是小萌新，都能根據劇情去猜到自己下一步的動作，不至於對著屏幕一通亂點最後氣急敗壞查攻略，一頭霧水玩遊戲的感覺確實不好受。

節奏安排合理，分層難度引導讓玩家沉浸體驗倍增

那麼在劇情設計不錯的情況下，如何在劇情的各個時間點給玩家一定的”考驗”也是一門學問。RUSTYLAKE系列的遊戲在這一點上也是跟大部分密室類遊戲趨同，將遊戲的難點分成了幾個層次分散到了不同的關卡中。比如”收集”是這類遊戲中最簡單的一個環節，其次則是”物與物”之間的聯想互動，第三層則就是抽象理解。

當然上面說的三層也只是我為寫文章方便而臨時建立的名詞，大家有自己的理解和取名方式也可以在下方留言。言歸正傳，在《逃離方塊：生日》中這幾個環節自然也是跑不了的，如何根據劇情將這些難點一步步插入到關卡中毫無疑問是個難啃的硬骨頭。

最簡單的環節自然就是收集，在千禧年左右的密室類小遊戲中，收集幾乎貫穿了遊戲全局。可能是受當時技術條件影響？不過到了今天，收集已經變成了最基礎的環節。一般來說就是考驗玩家的細心和聯想能力。在銹湖生日中，收集的環節並不算多，但卻夠用，同時與劇情的聯繫非常緊密。

比如通過”蛋糕”引導玩家去找刀子，通過儀器引導玩家去找螺絲刀。在最簡單的收集環節，RustyLake也算是玩出了花。

第二個環節則是聯想。在玩家完成物資收集後，如何將這些物品根據劇情合理地使用將是一個考驗，這個環節將考察玩家的聯想能力。在這一遊戲中，有一個劇情是老爺子想在酒里加冰塊，這時候玩家毫無疑問是被使喚的對象。但冰箱中根本沒有冰塊，怎麼辦呢？根據水龍頭、冰箱、氣球，玩家可以想出”氣球灌滿水結冰，最後取出冰塊”的方法。

第三個環節則比較抽象一些，一般來說就是真的驗智商環節了。一般來說會有：猜密碼、做數獨、高難度拼圖等任務在等待著玩家。在《逃離方塊：生日》中，切蛋糕這一關算是比較磨人的。沒有提示，只有刀和蛋糕，如何用4刀讓所有蠟燭點燃則是問題核心。

那麼在簡單思考後大家會發現，要得到蛋糕里的”隱藏信息”自然就需要將蛋糕面上的所有蠟燭點燃。按著邏輯推，玩家需要知道如何才能讓蠟燭點燃，答案則是切出來的每一個區域只能有一根蠟燭存在。但要做到這一點，想必也是需要幾次的實驗。玩家通過層層引導成功解決問題後，成就感自然是不少的。

最後在這層層的引導和分級的難度中，玩家的思緒幾乎全部沉浸在了遊戲的劇情和解密之中。毫無疑問合理的難度設置和節奏把控能讓遊戲更加具備吸引力，《逃離方塊：生日》在這一方面就做得相當不錯。（上圖的玩法則是需要玩家找規律排列旗子，最終得到道具，其實也就是考驗玩家的邏輯思考能力）

懸念巧設，類”大圓滿”式反轉結局好評

說完劇情和難度，最後自然就是要說說結局的設置了。這款遊戲的劇情我們之前也基本介紹了個七七八八，其實簡單來說就是分上下兩部分。第一部分是玩家解謎推動劇情發展，兔子闖入滅門；下半部分則是玩家修復時光機，回到過去並改寫歷史。

那麼這個結局的懸念毫無疑問就放在了下半部分中，玩家要如何努力才能改變一家人的命運成為了這款遊戲的趣味之一。最終當玩家得到道具”火槍”時，懸念也瞬間解開。玩家將火槍這一道具交給家裡的爺爺，雖然時間軸向後推，兔子進門的一瞬間被爺爺幹掉，全家人大圓滿結局，生日沒有變成血海。（老爺子：砰！哈哈。）

這種帶有一定懸念的”大圓滿”式結局也是讓這部作品收到了一定的好評。整體劇情和遊戲雖然顯得有些致郁，但在玩家努力後遊戲的結局似乎也稍微溫暖了一些。

總結

遊戲整體觀感不錯，致郁色調與遊戲主題緊密貼合後讓遊戲整體的氣質上了一層樓。劇情設置合理，節奏緊湊且遊戲目的明確，玩家在遊玩過程中空白期極少，時刻基本都在思考。總的來說值得一玩。（暫時沒有中文版，但遊戲中用的英語都不難。）
