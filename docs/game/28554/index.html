TES中野完美聯動，小花生表現平平，369全家桶扭轉局勢

2019-12-23 248條評論 2,234次閱讀 1062人點贊

這裡有最新最全的英雄聯盟資訊，菌菌帶你看不一樣的召喚師峽谷~

12月22號冬至這一天，德瑪西亞杯第二天的賽程持續進行著，今天是BD兩組的對決，TES對陣LGD的比賽備受關注，因為LCK的Peanut加盟了LGD，然後Karsa加盟了TES，中韓兩個野王在德杯提前較量了，LCK冠軍打野Peanut能否帶領LPL保級隊LGD贏下這場比賽呢？

LGD禁：奧巴馬、流浪、蜘蛛、刀妹、妖姬，選：鐵男、酒桶、亞索、女槍、泰坦

TES禁：夢魘、挖掘機、奧拉夫、奇亞娜、阿卡麗，選：船長、瞎子、冰女、賽娜、蕾歐娜

我們可以看到LGD這邊ban了四個中單，可能是比較忌憚Knight，而TES這邊的ban位也是比較有針對性，ban了四個打野，就表明了針對小花生，而LGD選擇了比較坦一點的陣容，而TES這邊的陣容比較有進攻性，Karsa也是拿出了自己拿手以及小花生也拿手的瞎子。

Karsa開局就直指中單，幫定了Knight，想抓亞索，但是Q空了於是繼續刷野。四分鐘Karsa繼續抓亞索，這次沒有Q空，亞索交閃，瞎子冰女繼續追擊，瞎子再次Q中把亞索擊殺，拿到一血。

7分鐘，瞎子聯動冰女想下路4包2，日女先手開被LGD三人化解反殺，然後TES三人撤退，原本4包2的TES被反4包3，瞎子Q中了殘血的泰坦並擊殺，但是自己也是交代了，TES其餘兩人撤退。

9分鐘，酒桶拿了峽谷先鋒，而瞎子選擇拿土龍交換。13分半，LGD三人抓到了回城的蕾歐娜，配合趕來的亞索把日女擊殺，亞索拿到人頭。15分半，亞索塔下守塔，隊友不在自己先手開船長，被TES三人反殺，船長拿到人頭，最後船長也是被泰坦Q中配合塔擊殺，雙方1換1。

20分鐘，TES率先開水龍，酒桶下去把蕾歐娜炸了回來，亞索接R想擊殺，但是蕾歐娜閃現逃走，但是瞎子血量被龍打得太低，被泰坦R+閃擊殺，女槍把殘血的水龍拿了，並且擊殺了TES四人，這波1換4結束團戰。

25分半，TES眾人藉助賽娜的隱身，冰女率先大招開酒桶，但是隊友趕到，泰坦Q中冰女，亞索和女槍接R把冰女秒了，然後LGD選擇去拿水龍，TES趕過來想打架，LGD放棄龍選擇打架，但是被369船長一波二連桶炸到四個人，三人直接殘血，TES乘勝追擊，最後TES1換4把劣勢都給打回來了。

27分半，TES拿了大龍。35分鐘，大龍刷新，雙方再次再大龍交戰，半血泰坦先手Q中人，但是被集火擊殺，冰女也從側面進場，亞索鐵男泰坦三人被擊殺，TES繼續拿大龍。

36分鐘，TES拿下第四條龍，得到水龍魂，這時候TES的勝算已經很大了，帶著大龍buff和龍魂的TES拆掉了中下水晶。40分鐘，最後一波團戰，Karsa瞎子率先被秒，蕾歐娜和冰女把LGD三人秒掉，只剩兩人的LGD無力回天，TES拿下德杯首勝。

賽後點評：這把Karsa的動作比小花生要多，支援也是比較迅速，前中期LGD下路女槍一度優勢，但是中期369一波全家桶，把LGD眾人打殘，在追擊中，TES成功把劣勢扳回，在之後也是拿到了兩條大龍和龍魂，也是基本奠定了勝勢，恭喜TES！
