貓咪即將遭到史詩削弱？藍耗大幅提升，玩家：快樂不起來了

2019-12-03 0條評論 18次閱讀 0人點贊

摘要

眾所周知，在低分段有一個非常熱門的輔助，她就是前一段時間上線的悠米。她的出現給不少女玩家帶來了福音，畢竟本身沒有太大的操作難度，很多時間都附身在隊友身上，可以算是真正意義上的躺贏。近日拳頭設計師透露了新版本的英雄調整內容，其中就包括魔法貓咪。

眾所周知，在低分段有一個非常熱門的輔助，她就是前一段時間上線的悠米。她的出現給不少女玩家帶來了福音，畢竟本身沒有太大的操作難度，很多時間都附身在隊友身上，可以算是真正意義上的躺贏。近日拳頭設計師透露了新版本的英雄調整內容，其中就包括魔法貓咪。

首先被動獲得加強，基礎護盾值由50-300改為50-390，並且還能回復50-160法力值，為接下來的削弱做鋪墊。

Q技能的法力消耗增加，由75/80/85/90/95/100改為85/90/95/100/105/110，同時AP加成由0.45降至0.3。強化基礎傷害由50/85/120/155/190/225 改為50/95/140/185/230/275，強化AP加成由0.65降至0.4。

W技能的固定自適應之力增加，由5/7/9/11/13改為12/14/16/18/20，同時基於彼此的自適應之力也獲得增加，由4/7/10/13/16%改為12/14/16/18/20%。主動效果也發生了變化，附身隊友需要0.2秒引導，期間可以被打斷。如果貓咪被定身，該技能將有5秒CD。同時，她不再能附身使用TP的隊友，從而實現快速返線的效果。

E技能的法力消耗增加，由30/40/50/60/70改為 100/115/130/145/160，同時取消充能機制，改為12/11/10/9/8的冷卻時間。治療量與易損生命值掛鉤的被動被移除，改為基礎治療量提升，由30/40/50/60/70改為70/110/150/190/230。同時治療量的AP加成提升，由0.1增至0.3。基礎加速效果降低10%，但新增攻速提升機制，效果為25/30/35/40/45%，持續時間為三秒。

目前沒有收到大招改動的消息，所以主要還是對Q、E兩個技能進行削弱，來遏制貓咪在順風局的壓制力。相信大家也知道，本次世界賽就出現了貓咪和蓋倫的組合，但貓咪卻不是傳統意義上的輔助，做出盧登帽子的傷害甚至要高於中單。或許這次改動後，貓咪就無緣職業賽場了。不過對於路人玩家來說，影響就比較大了。

目前這次改動也是設計師的初稿，一切內容還是以官宣為準。如果貓咪真的改成上述內容那樣，大家還會如此中意她嗎？
