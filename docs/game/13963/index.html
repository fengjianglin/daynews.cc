八戒，八戒，心腸不壞；八戒，八戒，你快崛起了

2019-12-12 459條評論 3,517次閱讀 1343人點贊

豬八戒，一個從出現在王者榮耀那天開始就是在給賽事準備的英雄。雖然豬八戒的路人排位出場率低到不如一些版本弱勢英雄，但是在kpl上幾乎是每一場比賽，每個隊伍都要用上那麼一場。下面就讓我們進入豬八戒的英雄使用技巧。

一、邊路對線技巧

1、善用卡兵線的對線技巧

豬八戒雖然有一套接近無解的連控，但是其傷害不是很高，而且手短。所以豬八戒在邊路上長時間沒有支援的時候，他需要把兵線卡在自己的塔下面。方法是卡著一波兵線的最後個遠程兵不打，直到下一波兵到時讓他與自家兵線在塔下相遇。這樣既能限制對面的清線速度，又能保證自己的安全。

2、注意中路與河道視野

雖然豬八戒擁有技能霸體機制，但是如果被多人gank也是很難存活的。所以作為一個邊路戰士，需要很強的視野把控能力。一旦出現了對面三、四個人同時在視野里消失，或者注意到對面中單往你這邊移動，就需要自己往塔里縮一些或者叫上隊友支援。

3、使用技能磨血消耗

因為一技能和二技能的連招除了凈化類的技能之後，基本沒有什麼辦法能躲掉。所以豬八戒在跟戰士對線的時候可以多次找機會控對面一波，然後靠著自己的回血複製一波換血之後不用回家。玩豬八戒其實就一個原則，要充分利用自己的回血機制去消耗敵人。

二、豬八戒打野思路分析

豬八戒因為技能控制多，二技能和大招配合野區複雜的地形可以親身控制住多人，所以許多kpl戰隊和玩家常常把豬八戒放在野區，當一個藍領野。因為豬八戒守野區能力極強，所以前期基本沒有多少英雄能跟他搶野區。也使得豬八戒打野變得簡單了起來。但還是需要在團隊節奏上注意以下幾個問題。

1、前期配合打團，豬八戒打野在四級之後，第一波完整的控制鏈就形成了，這個時候可以試著和邊路射手或者中路法師進行簡單的技能配合。

2、中期豬八戒轉向邊路，回歸到跟邊路戰士對線中來，因為豬八戒在中期的時候因為自己的傷害不高，所以儘管有打野刀，對於野怪的傷害還是沒有想像中那麼高。所以乾脆進入邊路發育，等到做出火甲＋不死鳥＋極寒風暴的時候再打中後期團戰。

3、靠著打野懲戒控龍，雖然傷害沒有那麼高到能自己一個人拿龍，但是一般情況下，豬八戒是通過干擾對面拿龍來拖對面的節奏。這也是某種意義上的野區保護吧。

三、團戰的兩個主要細節

1、控制自身血量卡回血機制，豬八戒不是一純抗傷坦克，更多的作用是進行控制，然後讓隊友跟上輸出。但是團戰不可能一直這樣完美，所以一定要習慣自己計算血量，及時退出團戰危險地帶。

2、控制連招容錯率

一般的豬八戒在隊友在附近的時候就開始抓人gank之類的。但其實豬八戒的技能前搖太久了,一技能不接閃現的很難開到人。所以基本做到可以把人留住或拉回來的地步就基本合格了。

四、出裝與銘文推薦

豬八戒面板傷害不高，所以不是很需要銘文上的輸出補充，一般推薦純肉銘文或者調和＋鷹眼＋宿命的帶64穿透的半肉銘文。

出裝上，豬八戒需要比較有點輸出的肉裝在提高自己坦度的同時也能不至於單方面挨打。這個版本血魔之怒改成了主動釋放之後，可以與豬八戒的回血被動配合，讓自己在團戰中偽裝自己的血量。
