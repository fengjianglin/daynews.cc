騰訊雲遊戲，巫師3正版白給，最近的遊戲看點都在這裡

天天要聞
2019-12-19

騰訊和Nvidia的雲遊戲平台：

騰訊和NVIDIA一起正式發布了雲遊戲服務Start，當然目前來看其實只有《QQ炫舞》《劍靈》《流放之路》《NBA2K Online 2》《FIFA》《炫舞時代》等遊戲，這些遊戲很多不用雲都可以玩，而像QQ炫舞這些遊戲不上雲還能好好玩，上了雲延遲相比本地有延遲的情況下怎麼玩呢？

個人推測騰訊搞這一套東西可能更多的是為了加深和Nvidia的合作關係和試驗一些新技術的可行性，雲遊戲本身倒是其次。

PS4手柄背鍵擴展裝置：

一般來說我們用手柄的時候，用舒服的姿勢你只能用到4個手指，相比於鍵盤滑鼠這樣至少能用到6個手指，加幾個功能鍵能用7-8個手指的情況來說會差不少。

微軟精英手柄的解決辦法是直接加了4個可拆卸背鍵，你直接用食指中指就能很方便的控制，精英手柄雖好，但得另買並且十分昂貴，所以只有極度發燒友才會去購買，而索尼提供了新的解決方案——在手柄上外接一套組件提供兩個可以自定義的背鍵。

這一套東西賣249，不是特別便宜，但有PS4的玩家升級成本也就這個背鍵了，再加上按鍵本身可以自定義，所以總體而言如果有需要還是可以購買的。

生化危機3重製版的漲價：

生化危機3重製版的預購已經開放了，Steam上的價格高達414元，基本已經和全價的家用機版本差不多，結合之前育碧漲價，正版遊戲在主流平台上基本都已經恢復到全價，而低價區很有可能將成為歷史。

雖然打折是一定會打折的，但玩正版太貴對很多人也算是個問題吧。

巫師3正版XGP白給：

巫師3，永恆之柱完全版，奇異人生2 第五章和無名大鵝加入XGP訂閱服務，

作為一個正版訂閱服務，XBOX GAME PASS雖然看上去定價挺高，但實際上經常有很便宜的折扣活動，例如前幾個月還有3個月10塊錢的訂閱活動，而當時Game Pass里不但有微軟第一方的作品例如極限競速地平線和光環，也有一些值得遊玩的作品，例如地鐵離去，鬼泣5等新作。

如果能便宜的玩到正版，那自然就沒必要玩盜版了，還是歡迎大家加入正版的行列中來。

新的軌跡

軌跡系列也算是常年的老飯了，本作叫《創之軌跡》日文名是はじまりのきさき，寫成日語漢字的話就是：始まりの軌跡，很明顯有開始的軌跡的意思在裡面，而本作直接使用了新的三條交叉故事線體系，將會講述三個陣營但基本平行的故事。

與此同時，法老空也發布了2個奶粉的冷飯作品：

零之軌跡改和碧之軌跡改，主要還是把原作以高清化60幀處理，純粹奶粉作品。

不知道有多少人在PSP上玩過這兩個作品，不過說實話就這麼直接搬上來了頗有一些誠意不足的意味。
