DNF：春節版本來了！劍魂再次變強，兩大新副本同時更新

2019-12-14 1,530條評論 4,282次閱讀 865人點贊

體驗服在12.13更名為魔界後，老玩家們都知道DNF要有大版本上線了，因為2020的春節來的比較早，而12月的活動全是預熱，吸引平民玩家回坑，95B史詩滿地掉，增幅書輕鬆得，既然預熱已經更新完畢，那麼也要上主菜了。這次體驗服一樣是提前一個月更新，大家期待已久的「春節版本」來了！（SPK這時間把握非常到位，12.14是發布會，兩大爆料齊上線）

體驗服12.13更新，春節版本上線！

在12.13的拆包中，CP副本魔界大戰來了，更名「決戰！魔界風雲」，就是同一個副本，不出意外的話，SPK這次沒重做了，上一次魔獸變超時空，造成漩渦副本沒新意，而這一次史派克應該是照搬所有系統，比如CP技能護符，全職業都是5個技能效果挑選2個。包括「東哈林」、「螺旋王座」等副本也一起實裝，這些內容將於1.14更新。

劍魂再次變強，兩大新副本同時更新

而魔界風雲副本的上線，代表的就是全職業改版！比如劍魂再次變強，提升的傷害比紅眼、鬼泣等職業更多，60多個職業，自然會有一些職業的CP護符屬性更差，除了護符這個新裝備之外，耳環部位終於淘汰盧克光卡了，150點四維的屬性還行，但春節版本還有一個副本亂入，那就是「天界戰記」。因為拆包出來2個小地圖，比如「哈特爾山中腰。」

這天界副本應該是100級的內容了，只不過劇情先行！畢竟100級之後，魔界和泰波爾斯全都炸了，天界陷入大戰也很正常，但BOSS和獎勵均為確定，至於春節套的具體屬性，需要等1月初才會爆料。

不過玩家們卻認為體驗服12.13更新沒有誠意……單憑一個技能改版拖幾個月？沒有新裝備和玩法上的轉變，春節禮包就算送一把自選的界·武器都沒用。

總結，春節版本的內容全在意料中，要是SPK沒有藏一些重磅改版的話，1.14的春節套未上架就已經先涼了，要知道國慶禮包一起追憶四套天空，銷量也不如去年！
