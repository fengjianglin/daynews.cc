黑月Extend：12月19日開測，只要一雙手，連擊可以無限

2019-12-18 3,508條評論 4,684次閱讀 416人點贊

如果說什麼遊戲最能激發身體里的那份熱血，我想除了格鬥遊戲以外，別無他選！回歸本質，感受操作帶來的刺激！在12月19日，一款玩家期待已久的格鬥遊戲《黑月Extend》就要開啟測試啦！遊戲採用2D橫板闖關！每一幀每一秒都是技能！每一下操作都能玩出大片的感覺！

【劇情】

在遙遠的未來，一場大災變改變了人類世界，地球和平行時空在陰差陽錯下產生同步，構建出了一個類似「黑月」的通道！以為是外星人的友好聯盟，誰知卻是災難的開始！異界怪物入侵，少數地球人覺醒了異能，成為「神選之子」保護人類。在遊戲中玩家需要扮演一名「神選之子」，與外星生物展開對決，探索這個廣闊的世界，究竟是人類取得勝利，還是外來生物毀滅地球？

【遊戲亮點】

操作感：擁有無限連招的操作遊戲還剩多少呢？滿打滿算早期的街機，如今的DNF，再來就是PS4上的3A大作！而手機遊戲里能玩出流暢操作加打擊感的遊戲還有嗎？有，這點絕對有，但我只能說是少數。

相比上述說的這些高端產品，《黑月Extend》作為小廠手游來說實為不易，遊戲以技巧打擊為主，削弱範圍技能，這樣看起來不像某些遊戲那樣，特別無腦，而是注重起了技巧操作。當然…遊戲難度相對提升了，沒有三兩三還真玩不動《黑月Extend》。

另一方面，2D橫版操作大幅度簡化了不必要的麻煩，比如玩家的3D眩暈，又或者3D操作劣勢，玩家通常沒辦法對3D遊戲進行一個很完美的操作，一個手機本身的局限性，還一個就是端游和主機設備完善，操作起來也比手遊方便太多！所以，小師弟認為手游設計成2D橫版是最明智的選擇！

肝度：到了各位玩家最關注的點，氪金與肝度，氪金目前無法確認，畢竟遊戲還在測試階段，不過關於肝度情況咱可以和大家好好探討下，早期的測試爆肝情況嚴重，到最近的一個版本肝用情況明顯下降，基本可以控制在1小時刷完每日任務！

目前遊戲將肝與不肝的玩家分得特別清楚，肝的玩家刷裝備，可以上架交易獲得大量金幣！而不肝的玩家每天打打日常組隊，做做組隊本裝備就行。

【玩法】

我把玩法放到最後是覺得，玩法其實沒必要多介紹，像這種打鬥遊戲市面上見過太多太多了，往早期的格鬥獵人，現在崩壞三，這些都是業界較為優秀的產品，如果把它們拿來做對比，其實並沒有什麼優勢。不過或多或少讓大家知道一個大概，這款遊戲到底是在玩什麼！

遊戲本身有優秀的劇本線，而推進遊戲劇本線就在於玩家刷圖，不斷的刷圖解鎖更多劇情，直至完整看完所有劇本。是不是挺簡單的？

沿途階段玩家會面臨職業的轉職以及覺醒，選擇合適自己的職業，看你想往那些方向發展，是要手短還是手長，又或者說想玩玩體術還是法傷。

另外，值得一提的是，遊戲擁有組隊功能，玩家們可以一同征戰副本，相互扶持。刷圖之中還能看起語音巴拉巴拉，瞎嘮嗑，不至於讓你覺得是一個人在玩遊戲！

【總結】

整體看下來就是這些內容了，也不知道各位對此是否還滿意，不過該遊戲在論壇上的評價特別高，12月19日會開啟限量測試，先到先得，想體驗的小夥伴可以快點去關注一波了！PS：該款遊戲之前的名字叫做《合金都市》現改名《黑月Extend》！
