英雄聯盟：轉會期各隊操作評級——VG拿到唯一S+，WE與EDG並列C級

2019-12-21 926條評論 1,904次閱讀 154人點贊

伴隨著LPL各支隊伍陸續官宣德杯陣容或者是新賽季大名單，為時一個月左右的轉會期基本落下帷幕，對於粉絲們來說自然是有歡喜也有愁。

玩加根據各支隊伍轉會期的操作情況做出評價分級，具體排名及原因如下（只是轉會期操作排名，非下賽季實力排名）

S+：VG

作為春夏季賽戰績皆為2-13的聯賽倒數，這一次轉會期VG可謂是重拳出擊：Forge、iBoy、Maestro、Chieftain、zeko多位選手加入，kkOma+PoohManDu+Maokai的教練組成立，不得不讓人感慨VG怎麼突然變得這麼有錢。

現在這支由kkOma領軍的青年軍將要在新賽季的LPL掀起一股年輕風暴，而他們現在需要解決的只是團隊磨合以及選手經驗問題。或許新賽季的VG只不過是一支季後賽一輪游隊伍，但和上個賽季相比這已經是極大的進步。

S：TES、FPX、OMG

FPX在2019年已經確立了自己強隊的地位，更是奪得了S9世界冠軍。所以本次轉會期的FPX也沒有太大動作，只是宣布了上單Khan的加入，能夠維持冠軍陣容已經殊為不易，引入一名世界級上單更是神來之筆。但新賽季的FPX比起2019年來說，提升空間也只有聯賽春夏雙冠以及前無古人的同年大滿貫。

而TES本賽季引入了Karsa，以及用青訓下路組取代了LokeN+Ben的組合，整年來看LokeN+Ben是沒有達到我們年度的期盼，TES距離世界賽也只是一步之遙。新賽季的TES在溝通方面會更為順暢，Karsa的控圖能力也可以幫助隊伍打得更加舒服，唯一的變數就是下路的兩個年輕小夥子能否承擔起在遊戲後期Carry隊伍的能力。

OMG本賽季的補強可謂是恰到好處，Smlz的回歸以及H4cker的加入都是針對隊伍本賽季弱點做出的加強。誠然，2019年的Smlz、H4cker較之前都有不同程度的下滑，但馬哥、冷少和小五也是搭檔多年的好友，如果新賽季的OMG磨合做好還是有衝擊季後賽的很大可能性。不要忘記，上賽季的他們同樣也是2-13位居聯賽倒數第一。

A：LGD、BLG、SN、DMO

LGD新賽季引入的主力選手有Peanut、Killua、Fenfen，這個陣容對於上賽季來說紙面實力是絕對的提升，新賽季成績會有較為明顯的進步。但需要優先解決的是隊伍的交流和磨合，下賽季的樂觀家族還是值得期待的。

2019年的BLG可以說是夏季賽比春季賽進步最大的戰隊，這一次的轉會期他們基本維持了上賽季的陣容，只是中單位置用FoFo替換了Kuro，毫無疑問這在個人實力方面是一個提升，但FoFo又能否適應BLG的戰隊體系還需賽場檢驗。

SN的新陣容里雖然沒有那個備受大家期待的男人，但引入的SofM也還算是一個不錯的加強，現在他們需要向我們印證的就是Bin能否將他Rank中的實力帶到LPL賽場，而WeiWei的轉型又能否和蛇蛇一起在下路抗住壓力。

DMO應該是除了FPX之外，本次轉會期變動最小的俱樂部。在中路位置引援了xiye之後，隊伍也可以有更多的戰術儲備和風格。他們現在需要的就是找回2019春季賽隊伍的狀態，然後打出符合隊伍選手們段位的水平，如果能夠做到，那麼季後賽還是有希望的。

B：RNG、iG、JDG、LNG、V5、RW

就RNG本賽季轉會期的操作來看，Karsa的離隊我可以給到-5分，那麼Xiaolongbao和Betty的加入則是只能+1、+2，畢竟次級聯賽的打野和LMS的選手都還需要在LPL來真正證明自己。而Mata和Tabe的回歸又能否抵消牛排及Dandy的離開，還是需要打一個問號，畢竟這只是Mata成為主教練的第一年。

iG轉會期的情況整體看下來還算不錯，TheShy、Rookie全部留了下來，但是Puff+Southwind又能否扛過下路的大旗還尚未可知。可以說JackeyLove的去向成謎是這次轉會期的最大變數，如果iG能夠將其續約留隊，那麼我想他們應該可以升至S級。

如果說2019年的BLG向我們詮釋了何為進步，那麼今年的JDG就是退步的代名詞，直接從春季賽亞軍降到夏季賽第10名。這次轉會期在imp宣布退役之後，JDG下路又重新引入了LokeN，這個陣容彷彿又回到了2018年，下個賽季LokeN能否重新證明自己，這將會是JDG能走多遠的保障。

LNG這個轉會期將中野進行了更換，SofM+FenFen變為了Xx+Maple。2019年的Maple沒能夠在LPL賽區證明自己作為LMS頂級選手的實力，那麼這一次的加盟LNG又能否煥發職業生涯第二春？而Xx現在沒有了像Knight一樣強勢的中路隊友之後，比賽時的Gank選擇又要重新思考。但或許正像左霧說的那樣，他們就是最適合LNG的吧。

V5能排在B級的原因很大一部分是他們留下了Mole，以及他們實在是沒有什麼退步空間了。Ben4的離隊對於V5來說沒有太大影響，而997、Xiaohan、clx三位新人選手也還是需要時間檢驗。

RW與V5同理，留下了隊內核心Zwuji，而且不再搞輔助輪換，這也算是一種另類的提升吧，但其實最為主要的是作為春季賽和夏季賽的倒數，留給他們退步的地方不多了。

C：WE、EDG

雖然，WE本賽季雙C全部出走，但是引進的Morgan、Plex兩位選手也能夠做到即插即用。可以說新賽季的WE應該會沒有最開始預想中的那麼差勁，但比起上個賽季來看退步是在預料之中。不過，這些選手也都還年輕，在經歷了一個賽季的磨鍊之後，他們的希望還是在未來。

而EDG的這一次轉會期就很離譜，真的可以用「只出不進」來形容。從最開始的Tarzan、JackeyLove、Chovy……各種流言，再到最終確定的大名單，EDG可以說是這個轉會期最讓人感到失望的戰隊。不過還好新賽季的教練組裡起碼加了一個Clearlove，這也是新賽季唯一可以期盼的吧。

新賽季變數

1、JackeyLove到底去哪裡？

2、Clearlove和Mata能否出色完成從選手到教練的轉變？

3、變動較大的幾支隊伍內部之間的磨合
