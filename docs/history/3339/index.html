《軍閥風雲》第76回袁世凱剪辮任臨時大總統 張勳誓死不從斷髮令

1912年2月13日，清帝退位。

當天晚上，袁世凱叫來親信蔡廷干前來幫他剪辮子。

袁世凱道：

「辮子已經沿襲了近三百多年，留頭不留髮，留髮不留頭，這一條『豚尾』的辮子，被許多漢人視為漢人的恥辱，因此我剪掉辮子便是與清廷決裂，也是我維新的開始，擁護共和的象徵。」

蔡廷幹道：

「張謇一直傾向共和，武昌起義後他即剪掉辮子寄回了老家，作為他一生最大的紀念。」

袁世凱道：

「這辮子我保留了五十多年了，一旦剪去我還真有點不習慣呢，我當年練新軍的時候，就覺得這辮子實在礙事—-」

沒等袁世凱說完，蔡廷幹道：

「身體髮膚，受之父母，滿人可特別看重這辮子呀，所以滿人要求留長發，剪掉就是不孝呀，袁大人你認為呢？」

袁世凱道：

「這根本與孝不孝是無關的，你們不知道吧，滿族人為何看重這辮子呢？」

蔡廷干問道：

「我們不清楚，那滿人為何要留辮子呢？」

袁世凱道：

「因為滿族人的先祖發源於白山黑水，主要以打獵捕魚為生，前頭剃髮腦後結辮是為了穿山越嶺枝藤不刮發，射箭瞄準眼前無發遮擋，白天盤在頭上便於生產勞動，晚上解下來可當枕頭。」

蔡廷干聽的津津有味。並道：

「對於辮子還有這麼一些故事，我們一直沒有聽說。」

2月15日，孫中山向南京臨時參議院提出辭職咨文，推薦袁世凱繼任中華民國臨時大總統，黎元洪為副總統。袁世凱儼然成為共和的締造者，參義院甚至稱其為「世界之第二華盛頓，我中華民國之第一華盛頓」。為體現新風尚，袁世凱發出了《斷髮令》。

一時間，北京街頭巷尾到處都在剪辮子，被剪的辮子丟棄一地。

幾天後，徐世昌來到袁世凱府上。

袁世凱摸著自己圓圓的頭道：

「剪掉這麻花辮，讓人感覺更休閑、更年輕，去掉了正式的繁瑣。」

徐世昌道：

「是啊，但也有人明目張胆地不肯剪辮子呢！」

袁世凱問道：

「誰呀？」

徐世昌道：

「張勳就不肯接受斷髮令。仍留著一條辮子。」

袁世凱生氣地用河南話道：

「這不中！象什麼話，我這個總統臉上一點光彩都沒有，你明天去勸勸他，說是我說的。」

徐世昌去後，張勳道：

「我在前清受恩深重，君恩難忘，我誓死不剪辮子！我還要對部屬幾萬人要求，不可剪辮，我還要將前清會客的跪拜禮也有保留下來。」

徐世昌回來複命。袁世凱聽了道：

「這個先鋒真是個怪物！」

徐世昌道：

「人們都叫他張大辮子呢！」

倆人正在議論辮子的話題。

袁克定走進來道：

「孫中山雖然向南京臨時參議院提出辭職，推薦父親繼任中華民國臨時大總統，不過—」

話未說完，袁世凱收斂笑容問道：

「不過什麼？」
