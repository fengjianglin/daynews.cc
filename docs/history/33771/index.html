兩晉奇事：是他拯救了中原漢人，將屠殺漢人的胡人趕回了老家

一千六百多年前，在中國北方，一場空前的災難降落在漢人頭上。在這次大規模的人口滅殺中，漢人瀕臨亡族滅種。正是一個帝王的出現，頒布了一篇不足百字的「殺胡令」，並把數以百萬的入侵者趕出了中原，才讓中原漢人逃出生天。他足智多謀，多次以少勝多，堪稱領兵奇才；他驍勇善戰，每次都衝鋒在前，令敵人聞風喪膽：他讓對手恨得要死，卻在死後又被對手追封為「武悼天王」；他在位雖只有二年，卻使漢人躲過了一場亡族滅種的浩劫，他堪稱千古一帝。他就是十六國時期，冉魏的唯一皇帝冉閔。

公元316年，西晉王朝被匈奴人建立的前趙攻滅，於是，北方異族蜂擁而入，紛紛問鼎中原。其中以匈奴、鮮卑、羯、氐、羌等五族為最大，另有十多個小族，他們兇猛殘忍，大肆屠殺漢人，先後建立了數十個強弱不等、大小各異的政權，自此北方大地陷入一片混亂，這一時期，史稱「五胡亂華」。當時，遷居中原的胡人高達五六百萬之多，很多地方甚至超過了當地漢人人口。在這些胡人中，羯、白匈奴、丁零、鐵弗、鮮卑等部落，主體都是金髮碧眼的白種人，《晉書》上說其「高鼻多須」，其中，又以羯族，白種匈奴，鮮卑族三族最為兇惡。

據《晉書》記載：後趙皇帝羯人石勒，一次就屠殺百姓數十萬；石虎更是嗜殺成性，他在城牆上掛滿了漢人的人頭，將漢人的屍骨做成「屍觀」，恐嚇路人。這些胡人沒有自己的文字，甚至還保留著食人的獸性。石虎的兒子石邃，就把比丘尼身上的肉割下來，和牛羊肉混著煮，然後，把這種食品賞賜給部將吃，讓他們猜測是什麼原料做的，「合牛羊肉煮而食之，亦賜左右，欲以識其味也」；羯族人把漢族女子當作「雙腳羊」來飼養，隨時姦淫，隨時宰殺烹食；鮮卑慕容氏行徑更是怵目驚心，據《晉陽秋》記載，他們次曾把五萬名漢族少女充作軍糧，一個冬天就吃了個乾淨，碎骨殘骸堆成了小山。經過胡人的血腥屠殺，北方漢人銳減六七百萬，造成赤地千里的景象，「北地滄涼，衣冠南遷，胡狄遍地，漢家子弟幾欲被數屠殆盡」，「中原士族十不存一」，可以想像，當時的慘烈之狀，也可以說，漢人處在了亡族滅種的邊緣，這絕不是聳人聽聞，試想，如果北方漢人被屠殺殆盡，胡人又怎能不渡江南下，盡戮江南呢！？

冉閔的出現，拯救了瀕臨滅族的漢人。據考證，冉閔是孔子的弟子冉有所在的冉氏家族後裔，生年不祥，卒於公元352年。冉閔生逢亂世，卻是亂世中的大英雄。他先是在羯趙軍中任將，幫助羯趙滅了匈奴前趙，而後，又殺死羯趙皇帝石鑒，推翻了羯趙政權，於公元350年稱帝自立，國號大魏。冉閔稱帝後，隨即頒布「殺胡令」。全文如下：「諸胡逆亂中原已數十年，今我誅之。若能共討者，可遣軍來也。暴胡欺辱漢家數十載，殺我百姓，奪我祖廟，今特此討伐。犯我大漢者死，殺我大漢子民者死。殺盡天下諸胡，匡複漢家基業，屠戮胡狗為天下漢人義之所在。冉閔不才受命於天道，特以此兆告天下。」

號令一出，民皆響應。河北、山西、山東、河南、安徽、江蘇、湖北北部、陝西的漢人都紛紛起兵，關中八十萬漢族百姓冒著生命危險，千里跋涉投奔冉閔，冉閔一時軍威大振。冉閔滅後趙政權時，在其都鄴城（今河北臨漳漳河岸畔），一次就解放被擄掠的漢族女子達二十萬，殲滅了三十多萬以羯族與匈奴為主的胡兵，又在屠城中，斬殺羯胡二十幾萬。加上全國各省各地的復仇屠殺，羯胡與匈奴在血腥的民族報復中，被基本殺絕。迫於冉閔的武力威脅，氐、羌、匈奴、鮮卑等族數百萬人開始退出中土，各自返回舊地，有的甚至遠走中亞。在返遷的路上，這些人又不斷相互攻擊掠殺，能成功回去的人，十個人中僅有二三人。《晉書》記載：「（冉閔）與羌胡相攻，無月不戰，諸氐、羌、胡、蠻數百餘萬，各還本土，道路交錯，互相殺掠，且飢疫死亡，其能達者十有二三」。

公元352年。冉閔率領的一萬人馬，遭到前燕十四萬大軍伏擊被俘，前燕國主慕容儁嘲笑冉閔：你只有奴僕下人的才能，憑什麼敢妄自稱天子？「汝奴僕下才，何自妄稱天子」？冉閔冷笑道：「爾曹夷狄，人面獸心，尚欲篡逆。我一時英雄，何為不可作帝王邪」。意思是，你們這些人面獸心的傢伙都能當，我是漢人英雄為什麼不能當？慕容儁大怒，「鞭之三百」，然後送至龍城（今遼寧朝陽），斬殺於遏陘山（朝陽境內）。

冉閔死後，山上的草木全部枯死，蝗蟲大起，從五月到十二月，天上滴雨未降。慕容俊害怕了，就派人去祭祀冉閔，並追封冉閔為武悼天王。《晉書》記載：「俊送閔既至龍城，斬於遏陘山。山左右七里草木悉枯，蝗蟲大起，五月不雨，至於十二月。俊遣使者祀之，謚曰武悼天王，其日大雪。是歲永和八年也」。冉閔死後，冉魏國的臣子絕望至極，悲天呼地，紛紛自殺為國守節，場面悲壯異常，慟撼天地。只一小部分逃往東晉，沒有一個投降的。

（全篇完）
