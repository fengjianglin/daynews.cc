劉備臨終託孤諸葛亮，為何要在如今的重慶白帝城，而不在成都？

天天要聞
2019-12-19

在《三國演義》中有個橋段，劉備臨終前在白帝城將阿斗劉禪和蜀漢基業託付給諸葛亮，說出讓諸葛亮淚流滿面的話：「君才十倍曹丕，必能安國，終定大事。若嗣子可輔，輔之；如其不才，君可自取。」這就是著名的白帝城託孤，也叫永安託孤。劉備為何要在永安託孤，而不是在蜀漢都城成都呢？

​公元219年，劉備在漢中之戰中打敗曹操，拿下漢中盆地，將益州（四川）的北部門戶控制在手中，同時佔領上庸等地。關羽在中線襄陽戰場也取得不錯的成績，水淹七軍，擒于禁、斬龐德、威震華夏、圍曹仁於樊城。一時間，劉備蜀漢集團的聲勢日隆。這引起了東吳和曹魏的不安，結果東吳呂蒙白衣渡江，襲取荊州，關羽敗走麥城，而後被擒殺，劉備遭到前所未有的打擊，蜀漢面臨的形勢也急轉直下。

​兩年後，221年，劉備經過一番準備，親率大軍出川，以為關羽報仇為名，向東吳發起進攻，史稱夷陵之戰。居於長江中游的蜀漢相比東吳來說，具有居高臨下的上水優勢，沿著長江順流而下進攻的優勢比較明顯，劉備也是胸有成竹。劉備之所以親征，是因為此戰對他對蜀漢來說非常重要。不單單是為關羽報仇，更重要的是要奪回荊州，挽回失去荊州後的戰略被動局面。雖然戰爭初期劉備進展比較順利，但是結果卻出人意料，被東吳名將陸遜打了個漂亮的防守反擊。夷陵之戰劉備大敗，差點全軍覆沒，二弟張飛也在戰前被手下殺死。夷陵之戰劉備不但沒能取得預想的戰果，反而蜀漢的實力再次受損，只能戰略收縮，重新尋找突破口。

​222年，劉備率領殘兵敗將，逃至白帝城，幸虧趙雲率軍來援算是沒把自己也搭進去。夷陵之戰的慘敗，讓劉備羞愧難當，就此生了大病，最終在白帝城走完了一生。白帝城在哪裡呢？在今重慶市奉節縣城以東約10公里的長江北岸，不過，這裡需要特別說明的是，如今白帝城並非在長江北岸的岸邊，而是在江中的一個小島上，四面環水。你若看地形圖，就會發現。這是因為三峽大壩建成以後，這裡的水位上漲，白帝城所在的白帝山下面已經沒入江水中，成為江中一個小島。白帝城和長江北岸通過廊橋相連。

​白帝城南邊就是長江三峽的瞿塘峽口，大名鼎鼎的長江三峽從此開始。也就是說白帝城扼守三峽的西邊入口，地理位置極其重要。之所以叫白帝城，是因為王莽篡權建立新朝，公孫述割據四川盆地，在此發現「白氣」，當作「白帝復興」的祥瑞，並自稱「白帝子」，在此築城，名白帝城。劉備兵敗撤退至此後，不願再回成都，居住於此。畢竟這裡只是一座城，連縣治都不是，劉備怎麼說也是蜀漢開國皇帝，與其身份不符，遂將此改稱為「永安宮」，這也是為何白帝城也稱永安，白帝城託孤也叫永安託孤的原因。

​奉節縣，有很多古稱，如魚復、永安、夔州、人復等，控帶二川，限隔五溪，據荊楚之上游，自古被稱為「巴蜀之喉吭」、「吳楚萬里之襟帶」，其戰略位置極其重要。劉備堅持不回成都而居於此，除了身體方面的原因外，還有軍事方面的考慮。在白帝城南邊，長江橫切七曜山形成瞿塘峽，瞿塘峽口設有瞿塘關（亦稱捍關、江關、夔門），兩山對峙，中貫大江，素有「夔門天下雄」之稱，為兵家必爭之地。之所以叫夔門，因為奉節古稱夔州，這裡是夔州的東大門，可見其重要性，杜甫也曾有詩這樣形容瞿塘關「白帝高為三峽鎮，瞿塘險過百牢關」。

​劉備當初入川就是先攻取瞿塘關，才有後來的益州基業，自然知道此地的重要性。蜀漢重兵據此進可攻、退可守，進可順江而下對東吳造成壓力，退則可守住奉節，擋住東吳溯江而上進入四川盆地的通道，守住四川盆地的東大門，保住蜀漢的基本盤。劉備在夷陵之戰兵敗後，駐守永安是正確的。即使東吳不來進攻，也最起碼能引起蜀漢上下對永安戰略地位的足夠重視。蜀漢也確實做到了，史載「終蜀漢之世，恆以白帝為重鎮。」

也正是劉備的這次永安託孤，讓諸葛亮忠心耿耿地輔佐後主劉禪，其「託孤寄命，臨大節而不可奪」的高尚品質感動了唐太宗，讓奉節有了如今的名字。
