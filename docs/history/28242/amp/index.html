　　【秦俑奶粉】

　　誕生：從80年代初開始，秦俑奶粉出現在千家萬戶的奶杯里，幾乎伴隨了陝西一代人的成長，至今依然暢銷不衰。




　　【紅星軟香酥】

　　誕生：創建於1998年的陝西紅星軟香酥食品企業集團，憑藉實惠美味、適合秦人口味的紅星軟香酥打開了古都咸陽等二線市場。

黃河電視是國營黃河機器製造廠（786廠）的驕子，國家領導人也曾數次來廠視察。

　　【寶雞啤酒】

　　誕生：寶雞啤酒公司創建於1956年，寶雞啤酒以其獨特的口味沿渭水而下，逐漸征服西安人，佔領了大半個西安市場。

蝴蝶牌手錶

1968年，一批從上海手錶廠來的骨幹到了西安，受命組建國營紅旗手錶廠，也是西北第一家機械手錶廠。

國營紅旗手錶廠後來改名叫西安蝴蝶手錶廠，在那個「三轉一響」（自行車、手錶、縫紉機、錄音機）的年代，蝴蝶手錶和家喻戶曉的大IP梁祝聯繫在一起，因為浪漫的寓意，被年輕人拿來當定情之物，很是搶手。

「太陽鍋巴」一問世，就受到廣大消費者的喜愛，產品供不應求。後來，產量以每月增加100噸的速度發展，可怎麼增加也供應不上，總是缺貨、缺貨。

上世紀九十年代，一聲「秀蘭，額把洗衣機給你買回來咧」，使雙鷗牌洗衣機紅遍大江南北，陝西機床廠、陝西洗衣機廠也因此名冠天下。

「三個小菜，一瓶普太」，是陝西的地方俗話，也記錄了太白酒曾經的輝煌。

　　【鐘樓奶糕】

　　誕生：20世紀70年代，在西安鐘樓附近有一家小鋪面的鐘樓食品店，最早的奶糕從這裡誕生，因此得名「鐘樓奶糕」。

　　【德懋恭水晶餅】

　　誕生：坐落在西安市西大街廣濟街口的中華老字號「德懋恭」，始建於清朝光緒年間(1872年)，問世至今已有144年的歷史。清末慈禧太后來西安避難，經過廣濟街口聞香止輦，品嘗了「德懋恭」水晶餅後大加讚賞，遂將其欽點為「貢品」，「德懋恭」水晶餅自此身價倍增，聞名遐邇。

　　【冰峰汽水】

　　誕生：「冰峰汽水」飲料誕生於1953年。冰峰汽水的生產商為西安市食品廠，冰峰是該廠的其中一個生產車間，廠里人稱為「汽水車間」。最早的冰峰汽水2毛一瓶，瓶子押金8分，200ml的玻璃瓶內裝甜甜的桔子味汽水讓西安人簡直欲罷不能。
