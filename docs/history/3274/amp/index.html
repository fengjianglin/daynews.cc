大汗的底氣，準噶爾汗國的軍工業

天天要聞
2019-12-07

摘要

明末清初之際的百年時間裡，源自瓦剌的準噶爾部逐漸興盛起來。準噶爾雖然是游牧經濟，但幾任統治者都具備了卓越的戰略眼光，居然在西域草原中發展出了初具規模的軍事工業，令人嘖嘖稱奇，讓我們一探究竟。

明末清初之際的百年時間裡，源自瓦剌的準噶爾部逐漸興盛起來。準噶爾雖然是游牧經濟，但幾任統治者都具備了卓越的戰略眼光，居然在西域草原中發展出了初具規模的軍事工業，令人嘖嘖稱奇，讓我們一探究竟。

冶礦業

準噶爾統治範圍涵蓋今天的新疆、青海、蒙古高原西部、薩克斯坦、烏茲別克、阿富汗等廣大地區，接近今日中國版圖的四成。

如此廣袤的地區蘊藏著豐富的礦產資源，有金、銀、銅、鐵、錫、鉛等。俄國人索洛金記載準噶爾領地內：「湖四周的鐵礦相當多，它自古以來就由衛拉特人自己開採。用煉出的鐵製造刀矛、盔甲等。他們那裡這類匠人現在已近千人。」

古人云，家裡有礦，心中不慌！這些資源為準噶爾的軍工計劃提供了充分的物質保障。

其實，準噶爾人的先祖很早就懂得了原始的開採和冶煉技術，是一種代代相傳的老式冶煉法。隨著俄國人索洛金、瑞典人列納特的被俘，為準噶爾軍工業的騰飛帶來了契機，歐洲人被俘後隨即效力於準噶爾，積極幫助改進技藝，準噶爾冶礦業乃至整個軍工業水平都獲得了質的飛躍。

武器鑄造業

資源開採冶煉的目的自然要把它變成武器，準噶爾的武器鑄造業令人印象深刻。他們很早即可鑄造刀、矛、箭、盾、盔甲等傳統武器，在被俘的歐洲人的幫助下開始自生產火槍、火炮。

俄俘索治金描述，他「製成並交給噶爾丹策零使用的武器有四磅炮十五尊，小炮五尊和十磅臼炮二十尊。」準噶爾人製造的炮有鐵質、銅質兩種，根據軍事行動的需要而大小不一，有的可駕於駝背施放，有的可置於木架上射擊。

乾隆二十七年（1762年），清軍在伊塞克湖畔掘出準噶爾軍隊埋藏的自製大銅炮四座、衝天炮筒八個，大小炮彈萬餘發，這都表明準噶爾部的武器鑄造業已初具規模。

在準噶爾軍對清、對哈薩克、對東蒙古的重要軍事行動中，每每都會出現火槍、火炮的身影，而且部隊火器配備率奇高，在戰爭中發揮了重要作用，可見準噶爾的武器製造技藝已達到了滿足實戰的水準，規模甚至已可支撐長期、大規模作戰需要。

傳統手工業

準噶爾民眾為滿足自身需要也運營一些傳統手工業，所生產的主要為生活用品。如製革、制衣、制氈、金屬加工、釀酒等。

當然在發展手工業方面歐洲俘虜也不能閑著，準噶爾人從他們那裡學會了製作呢絨的技術，準噶爾人還從維吾爾人處學會了紡織布匹的技術，這些技術的傳入，對提升準噶爾汗國的手工業技術有很大的幫助，製革、制衣等手工業的發展也為軍工業提供了必要的輔助和支撐。

以冶礦業、武器鑄造業為核心，輔以傳統的製革、制衣業，準噶爾人在草原上建立了一整套軍工體系，這套軍工體系在近百年的時間裡確保準噶爾人能夠建立汗國、西定中亞、北阻沙俄、東拒滿清、雄霸天山南北。以康熙、雍正的雄才大略，付出十餘萬清軍將士的代價，亦奈何不得準噶爾。

堡壘往往是從內部被攻破的，隨著準噶爾最後一任雄主噶爾丹策零的去世，準噶爾陷入內亂，乾隆趁勢平定西域，最終完成中華大一統，準噶爾汗國獨具特色的「軍事工業化」進程也自此戛然而止。
