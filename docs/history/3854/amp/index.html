史上第一位和親公主劉細君：居常土思兮心內傷，願為黃鵠兮歸故鄉

天天要聞
2019-11-29

摘要

美麗的揚州才女，遠嫁到遙遠的西域，她成為史書記載的第一位和親公主，也是史上第一位留下姓名的女詩人。才藝雙全的江都公主劉細君出生在揚州，她是漢武帝兄弟劉非的孫女，被稱為“江都公主”。其父因謀反被處死，家族受牽連，細君因年幼得以倖免。

美麗的揚州才女，遠嫁到遙遠的西域，她成為史書記載的第一位和親公主，也是史上第一位留下姓名的女詩人。

才藝雙全的江都公主

劉細君出生在揚州，她是漢武帝兄弟劉非的孫女，被稱為「江都公主」。其父因謀反被處死，家族受牽連，細君因年幼得以倖免。

公元前117年，劉胥被封為廣陵王，派人找到了侄女劉細君。美貌的細君已出落得高雅清麗，廣陵王非常喜愛她，安排專人教細君學習禮儀、典章、歌舞和音樂，使其成為一位才藝雙全的才女。

烏孫與漢結為兄弟

兩千多年前，烏孫在伊犁河和伊塞克湖一帶（今天新疆溫宿以北伊寧以南，西至蘇聯境內），距離長安8900里。西漢武帝時期，國力強盛，經濟繁榮，希望聯合西域各國，共同抗擊匈奴。張騫建議送烏孫大量的金銀，招引他們逐漸東遷，與漢朝結為兄弟，這樣就斬斷了匈奴的右臂膀。只要聯合烏孫，那麼，它以西的大夏國都可以招來作為蕃臣。漢武帝認為有理，任張騫為中郎將，率領三百人、每人兩匹馬，帶著幾萬隻牛羊及價值幾千萬的錢財、布帛出使烏孫。

張騫帶著禮物來到烏孫，對烏孫昆莫（烏孫首領）說：「如果烏孫能夠東遷，那麼就將一位諸侯的女兒嫁給昆莫做夫人。」

後烏孫派出使者向漢朝進獻馬匹，希望能夠與大漢結成兄弟之國，並為昆莫娶一位漢朝諸侯之女。

漢武帝打開《易》進行占卜，卜辭上說：「神馬當從西北來。」漢朝得到了烏孫的好馬，將其命名為「天馬」。後將烏孫馬改名為「西極」 。

烏孫用一千匹馬當作聘禮來娶漢朝的女子，漢朝派出宗室女子細君下嫁給烏孫昆莫，昆莫立她為右夫人。匈奴也送來一個女子嫁給昆莫，昆莫立她為左夫人。

萬里窮愁天一方 石痕不爛胭脂節

公元前105年，漢武帝親自為細君公主送行。龐大的送親隊伍前呼後擁十幾里，旌旗逶迤婉轉，隨風飄揚。隨嫁宮娥彩女、樂工、裁縫、技藝工匠、護衛武士、宦官等多達數百人，皇室用的車馬、日常器物、食物等一應俱全，陪嫁首飾、絲質錦緞、帷帳等更是令人眼花繚亂。

細君公主臨行前，漢武帝命令懂得音樂的工匠參考古琴、古箏、擊弦樂器、撥弦樂器等創製一種能夠在馬上彈奏的樂器。讓細君公主把漢音帶入烏孫。工匠們研製出一種圓形音箱、直柄、十二柱、四弦、以石為槽、鵾雞筋作弦的樂器，用鐵撥彈之，當時稱秦琵琶，便是後世的阮。細君精通音律樂理，為琵琶設定了音階等。大唐段安節在《樂府雜錄》中講到：「琵琶，始自烏孫公主造，馬上彈之。」

車隊行經安徽靈璧時，細君公主在一處山崖前黯然佇立，手撫巨石，不忍離去，以至在巨石上留下了一枚清晰的手印。據說，這枚手印經匠人鐫石為模，能清晰地看到手指關節。因此，此地成為一處景觀，名為「靈璧手印」。元朝詩人錢惟善在《靈璧手印篇》中記載了此事，以「萬里窮愁天一方，曾駐鳴鑣倚靈璧，靈璧亭亭立空雪，石痕不爛胭脂節」的詩句，感嘆細君公主離鄉背井的心情。

願為黃鵠兮歸故鄉

烏孫是中國古族之一，聚居的地區天氣寒冷，常年多雨。他們以游牧為生，馬匹較多，牧民隨著牛馬游牧住在氈房裡，多食牛羊肉，喜歡打架、野蠻強悍。

當時烏孫昆莫獵驕靡年紀已老，而細君與才二十歲出頭。兩人語言不通、習俗不同，夫妻之間一年會面僅一兩次。細君來自十里荷花、魚肥水清的江南揚州，自幼受過良好的教育，又是宗室之女，面對這樣的生活，心中的孤苦與對家鄉及親人的思念無處傾訴，於是，寫下了悲愁歌：

吾家嫁我兮天一方，遠托異國兮烏孫王。

穹廬為室兮旃為牆，以肉為食兮酪為漿。

居常土思兮心內傷，願為黃鵠兮歸故鄉。

細君常常彈著琵琶，邊彈邊吟唱，聲音忽高忽低變換曲調，身邊的宮女聞聲垂淚，淚水從橫交錯在臉上。嘆息一聲來排解憂鬱，卻更加哽咽，只好拖著鞋起身四處徘徊，舉起衣袖遮住滿臉的淚珠，放棄思慮回房就寢。枕著用散發著幽香的香草做成的枕頭，慢慢入睡。

此詩傳到長安，漢武帝讀後很感動，不禁心生憐憫。於是下達詔書，每隔一年就派使者送去漢朝的食品、書籍、絲綢、錦緞、棉被等用品去看望細君。

在烏孫國，細君命工匠按大漢宮殿的樣式建造房屋，請木匠製作室內傢具，請泥工製作陶瓷、餐具，請縫紉工製作絲綢、錦緞，還賞賜王孫貴族及大臣們金錢財物及絲織品、綢緞、棉被、陶瓷等用品，逐一教會他們使用及保養的方法，因此，備受烏孫百姓的愛戴，百姓尊她為「柯木孜公主」，意思是潔白美麗、如同馬奶一樣純美的公主。

使於西域 不辱君命

獵驕靡年事已高，自知來日無多，欲把細君嫁給孫子軍須靡為夫人。深受漢文化熏陶的細君，尤重倫理，「忠臣不事二君，貞女不更二夫」，所以執意不從。她上書向漢武帝說明情況，天子報曰：「從其國俗，欲與烏孫共滅胡。」

細君遵旨，後來嫁與了軍須靡。獵驕靡死後，軍須靡繼立為王。劉細君與軍須靡生下一女，名少夫。後來細君公主思鄉心切，悲愁鬱結於心中，加上寒冷潮濕的生活環境，不久便憂傷而亡。

細君公主將漢文化帶入烏孫，使得烏孫人民由爭強好勝、打鬥漸漸變得謙讓和睦、民風好轉，生活安定。
