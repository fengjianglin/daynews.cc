古人為什麼崇尚“五”？“天打五雷轟”的“五雷”指哪五雷？

天天要聞
3 天 ago

在中國傳統文化中，「五」是一個很重要的數字。中國古人對「五」有著深刻的偏好和崇拜，這大概和古人最早用手指來計數有關。在廣泛的生活實踐中，出現了大量和「五」有關的詞語。

《尚書》中由「五」構成的名詞將近三十個，《逸周書》中也有三十多個用「五」字組成的名詞。比如人們常常所說的「五刑」、「五行」、「五服」、「五典」、「五服」、「五罰」等詞語，古人喜歡把數字分成五類，五行學說的產生正是基於古人對數字「五」的崇拜。

五行學說是中國古代哲學的重要內容之一，對包含中醫學在內的中國文化影響甚深。正如歷史學家齊思和先生所說「吾國學術思想，受五行說之支配最深，大而政治、宗教、天文、輿地，細爾堪輿、占卜，以至醫藥、戰爭，莫不以五行說為之骨幹。士大夫之所思維，常人之所信仰，莫能出乎五行說範圍之外。」

史料記載，「五行」一詞最早出現在《尚書·甘誓》中，書中記載「有扈氏威侮五行，怠棄三正。」但這裡所說的五行指的該是五種應行之道，和後來金木水火土的五行概念不同。真正把五行確定為金木水火土，是在商末周初。《尚書·洪範》中記載「箕子乃言曰：我聞在昔，鯀堙洪水，汩陳其五行。帝乃震怒，不畀洪範九疇，彝倫攸斁。……五行：一曰水，二曰火，三曰木，四曰金，五曰土。水曰潤下，火曰炎上，木曰曲直，金曰從革，土爰稼穡。潤下作咸，炎上作苦，曲直作酸，從革作辛，稼穡作甘。」其中提出的五行實際上指的是生活中的5種物質，這是早期樸素的五行概念，和西方的地、水、火、風四元素學說一樣，都是早期的古代樸素唯物主義。

早期的五行概念和五服、五典一樣，都是古人以數字「五」劃分事物的方式，但因為金木水火土五種物質和人們的生活息息相關，其地位也就顯得尤為重要，受到了古人的重視，有了更加廣泛的發展。

在生活勞作中，古人慢慢發現了五行相剋的關係。所謂「夫火熱而水滅之，今剛而火銷之，木強而斧伐之，水流而土遏之，為造化者，物莫能勝也。」五行相剋最早也稱為「相勝」，在春秋時期被人們廣泛運用，史官們用它用來預測國家大事，軍事家將它運用到排兵布陣、攻伐防守的細節之中。在制定對敵策略中，戰國的陰陽五行家鄒衍提出了五德始終說，認為朝代的更迭因為無德相互轉移，對中國古代政治產生了深遠影響。

隨著五行概念在實際生活中的廣泛運用，春秋時期，人們逐漸發現五行相勝並不是必然的，也存在「無常勝」的情況。只有在數量對等的情況下，五行相勝的概念才成立，若數量不對等，五行相勝便不成立。例如，金能勝木，若木比金多，相勝的概念就不成立。所謂「金之勢勝木，一刃不能殘一林之木；土之勢勝水，一掬不能塞江河；水之勢火，一酌不能救一車之薪。」

到戰國時期，五行學說逐漸走向成熟，五行相生的概念開始出現。《管子》中最早記錄了五行相生的排列，《五行》《四時》《幼官》《輕重己》等都是有關五行學說的重要文獻。四時更替和五行學說的融合，逐漸形成了一個系統的五行學說體系。《管子》中雖然最早把五行和四時相融合，形成了五行相生的概念，但還比較混亂。

直到漢代，董仲舒才明確提出五行相生的思想，五行學說也在漢代達到鼎盛，不但補充完善了前人的解讀，形成了系統的理論體系，得到了廣泛的應用和發展。五行乘侮的概念便是對五行學說的補充和完善，從此五行學說滲透進中國文化的各個領域，深刻地影響了中國人的生活和思維方式。

「天打五雷轟」中的「五」便是五行。所謂的「五雷」指的就是金雷、木雷、水雷、火雷、土雷。金雷指的是刀劍、鐵器、金屬、車禍等。木雷指的是樹木壓住、棍棒打擊、高處摔落等。水雷指的是溺水、水淹、生病等。火雷指的是火燒、電擊、雷擊等。土雷指的是土埋、房屋倒塌、高處掉物等。古人認為，「天打五雷轟」是對罪孽深重者的一種懲罰，也有「五雷轟頂」的說法，形容人們受到了巨大的打擊，同時也告誡人們莫做傷天害理之事，否則便會受到上天的懲罰。

古人從大自然的各種表象出發，總結事物發展的規律，充滿了令人讚歎的大智慧。
