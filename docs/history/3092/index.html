諸葛亮潛伏隆中10年，卻為何能名聲在外？

2019-12-09 0條評論 21次閱讀 0人點贊

摘要

說到諸葛亮，最令人稱道的就是三顧茅廬了。當時的諸葛亮已經在隆中潛伏了10年多，那麼，為什麼人在隆中卻名聲在外，以至於劉備特地慕名前來請他呢？諸葛亮從17歲開始就一直居住在隆中，直到27歲出山，這期間的十年時光是諸葛亮積累知識修行自我並且讓自己出名的關鍵時刻。

說到諸葛亮，最令人稱道的就是三顧茅廬了。當時的諸葛亮已經在隆中潛伏了10年多，那麼，為什麼人在隆中卻名聲在外，以至於劉備特地慕名前來請他呢？

諸葛亮從17歲開始就一直居住在隆中，直到27歲出山，這期間的十年時光是諸葛亮積累知識修行自我並且讓自己出名的關鍵時刻。諸葛亮喜歡結交比自己年長的朋友，從他們身上能學到很多東西，他們的想法也更能代表最基層老百姓的想法。他有兩個姐姐，她們都嫁得很好，按理說諸葛亮完全可以請求姐夫的幫助，讓他更上一層樓，但是骨子裡的驕傲不允許他這麼做，也不想給姐姐們徒增煩惱，他想通過自己的力量為自己鋪路，這樣得到的一切才更加踏實。

首先要想成功，第一步是走出這個地方，自己走出去很簡單，但是沒有什麼效果，必須得有人來請他，這樣才有意義。那別人為什麼要來請他呢？肯定是仰慕他的名聲，那就需要營銷自我，把自己的名氣紅到隆中以外。古時候人們其實特別會營銷，有很多前人的經驗，比如孟子，所以首先要做出一些不同凡響與眾不同的事情，有一些有爭議的言論，其實想想跟現在娛樂圈明星們的營銷玩人設有點類似，看誰的營銷手段好，誰的名氣就比較大，只是我們這裡的諸葛亮先生是有真才實學的。

諸葛亮很懂得營銷，知道首先要把自己的學識紮實一些，然後再想盡辦法把自己的名氣打出去，要不然自己只能一輩子待在這樣的小地方，自己的學識和雄偉抱負永遠實現不了，也見識不到更多優秀的人，自己也成長不起來。

萬事開頭難，首先不知道第一步應該幹嘛，然後就是有沒有勇氣跨出第一步。很慶幸，諸葛亮找到了他的開頭，就是從學業堂里退學，他的理由是不屑於文憑，只想學到更多更有用的學識。這在當時是很轟動的一件事，居然有人從大名鼎鼎的學業堂里辭學，說他想學更多的知識，那他的學識得有多高啊，諸葛亮算是打響了第一炮，從此小有名氣。

從學業堂回來以後，他就到了隆中，在這裡，一邊耕田，一邊讀書，一個很勵志的讀書人形象彷彿就在眼前，這一舉動讓人們更加感受他的自立自強，更加欣賞這個有志之士，加上他的抱膝長嘯的行為藝術，他的名聲已經水漲船高，越來越高了。在閑暇時光，他還會教父老鄉親們唱《梁甫吟》，要想名聲在外，還要通過周圍的老百姓給你傳播，那就要跟他們搞好關係，而且，教唱曲子不僅能跟他們搞好關係，而且可以將這首曲子傳播的更遠更廣。這對以後他治理國家也有很大的幫助，跟老百姓真正的接觸過，知道他們心裡想要什麼。

諸葛亮也不忘結交好友，一定要有自己的朋友圈，大家在一起喝酒暢談，高談闊論，表達自己的雄心壯志，這時，諸葛亮總喜歡將自己比作管仲、樂毅等名人志士，仰天長嘆，唏噓自己的才華無處施展。這讓他的朋友們也是感覺很驚訝，這傢伙居然把自己比作這麼才華橫溢的人，難不成他也有這樣的才識。久而久之，他這番言論變廣為流傳，大家都知道了這個自比管仲、樂毅的少年不簡單，以至於有人問起他，別人就會介紹，諸葛亮，卧龍先生，自比管仲、樂毅，特別有才華。讓別人成為自己的代言人，這也是營銷的高級之處。

光有普通民眾的讚揚還不夠，還要有一些名人志士對他的推薦，這樣才能證明他是真的學識過人。於是他找到了龐德公這個公認的推薦大師，經過檢驗，他確實配得上他所有的營銷，大師對他的評價是，孔明，卧龍是也。這樣的評價可以說很高了，這讓諸葛亮就好像拿到了走向上流社會的通行證，從此也名氣大漲，俗話說多多益善，諸葛亮還獲得了著名的水鏡先生的高度好評：卧龍、鳳雛，得一可安天下也！這個評價更好，在當時的社會背景下也更加有用，對他以後的發展起到了推動作用，加入了他的出山。讓他從一個名聲在外的文人墨客一下子搖身變為炙手可熱的政治家，謀略家，成為當時統治者們爭相搶奪的軍師。

營銷方法用對了，就成了自己挑選別人，而不是別人挑選自己，這才有了後來的三顧茅廬，正式打開他出山的大門。所以人一定要懂得營銷自我，把自己推薦出去，這樣才能有更好的機會。
