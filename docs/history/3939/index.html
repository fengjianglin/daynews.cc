坐擁百萬大軍的明朝，最後為何慘敗？因為培養了“兩個”敵人

2019-11-28 0條評論 18次閱讀 0人點贊

摘要

坐擁百萬大軍的明朝，最後為何慘敗？因為培養了“兩個”敵人！1644年對於中原大地來說確實是一個充滿奇蹟的年代。坐擁百萬大軍的大明帝國竟然被前兩年還到處跑的李自成給滅了，而估計李自成當年帶著十幾人架著尾巴沒命逃的時候，也不會想到自己會有今天這麼輝煌的成就！

坐擁百萬大軍的明朝，最後為何慘敗？因為培養了「兩個」敵人！

1644年對於中原大地來說確實是一個充滿奇蹟的年代。坐擁百萬大軍的大明帝國竟然被前兩年還到處跑的李自成給滅了，而估計李自成當年帶著十幾人架著尾巴沒命逃的時候，也不會想到自己會有今天這麼輝煌的成就！

而遠在關外，只能隔牆輕嘆的清軍也萬萬沒想到，自己祖輩幾代人都啃不動的山海關，竟然大開城門樓子讓自己入關，最後還一舉擊敗所有敵人最終建立了清朝，並打下了清朝近三百年的國祚。這一年，說是充滿夢幻的一年，估計沒人會反對的。

不過當塵埃落定之後，後人就都開始反思，為何看著十分強大，坐擁著百萬大軍的明帝國，什麼一夜之間就徹底就垮到底了呢？到底是什麼原因導致慘敗的呢？對於這個問題的討論，從明崇禎皇帝上吊之後就一直沒有停下過。而之所以會是這樣，主要還是人類本身的同情弱者基因作怪導致的，畢竟在1644年里，李自成高興過，多爾袞也高興過，就唯獨明帝國一直哭喪著個臉被人打，確實是個可憐的弱者。但是如果我們深深細究的話，明朝在1644年真的弱嗎？這裡我看就未必了，如果明朝作為一個人存在的話，說他弱他肯定不願意。

而明朝之所以這樣反應劇烈，大家其實也能夠理解，畢竟你什麼時候見過坐擁百萬大軍的弱者呢？但如果說明朝不是弱者，那為何明帝國最終還是慘敗呢？關於明朝的最終倒台，很多人都有不同的說法，有說明朝後幾個皇帝瞎折騰的，有說天災的，有說人禍的，更有說沒錢的。但如果細細品味起來的話，明朝的最終滅亡雖然有以上的因素在裡面，但卻都不是主要的原因，事實上，明朝的最終滅亡是明朝的軍隊出了大問題！嚴格意義上的說來，那就是明朝為自己的敵人培養出了最為精銳的部隊，然後再把這些精銳送給對手打自己，這就是大家經常說的那句話：”亂拳打死老師傅”。

首先，明帝國最先培養的第一個敵人叫做努爾哈赤。

努爾哈赤這個人的”舔功”要說第二估計沒人能排第一，面對自己祖宗被李成梁殺害，如此深仇大恨努爾哈赤卻能像個沒事人一樣服侍跟隨李成梁，並學得了所有李成梁帶兵打仗的精髓要點，最後等李成梁和李如柏死去，他利用明朝李成梁部的騎兵技術對付明軍，那是一打一個準，直接就把明軍打成了瘸子。

而第二個明帝國培養的敵人，那叫明末將領們。

而這幫人的危害，要比所有的人都可怕。而這裡面最具有代表的人物就數左良玉和吳三桂。這兩貨吃明朝的，喝明朝的。但乾的那也都是損害明朝的事。左良玉用明朝的錢糧來收買明朝軍隊的人心。但到了最後，他卻靠著明朝培育出來的軍隊當成自己的家資，成了自己的私軍。而為了能夠達成這個目的，他甚至還不惜養寇自重，本來他能一舉蕩平李自成的，可他就是不打人家就是要留著當成給自己撈好處的資本。結果到了，把”蜘蛛”養的比主人還大。

其次就是吳三桂了，吳三桂可以說在發展的過程中大多數時候是靠關係來一步步實現自己目標的，真正的軍事才能幾乎沒有，但卻掌控了明朝花重金打造出來的關寧鐵騎，這幫人在明末可以說是為數不多能和清軍野戰的軍隊之一，結果最後，這幫人卻成了打擊明朝最為強大的武裝之一，真是搬起石頭砸自己的腳了！
