“仁宣之治”的明宣宗真夠有膽魄，寬容造反皇叔，有此圖為證

2019-12-04 0條評論 21次閱讀 0人點贊

摘要

明朝第五位皇帝明宣宗朱瞻基（1399～1435年）在位時開創了盛世“仁宣之治”。並且是一位精於繪畫的皇帝，在藝術造詣上，朱瞻基還是略遜於宋徽宗趙佶，然而在運用繪畫的政教功能方面，賢明的朱瞻基則遠勝於昏庸的趙佶，今藏故宮的《松蔭蓮蒲圖》即為朱瞻基這類作品中的代表。

明朝第五位皇帝明宣宗朱瞻基（1399～1435年）在位時開創了盛世「仁宣之治」。並且是一位精於繪畫的皇帝，在藝術造詣上，朱瞻基還是略遜於宋徽宗趙佶，然而在運用繪畫的政教功能方面，賢明的朱瞻基則遠勝於昏庸的趙佶，今藏故宮的《松蔭蓮蒲圖》即為朱瞻基這類作品中的代表。

對於此圖，學者高雲萍在紫禁城雜誌《畫寓仁孝 巧待皇叔——朱瞻基《松蔭蓮蒲圖》的政治寓意》一文中提出了自己的見解。

明宣宗 網路圖片

據清安歧《墨緣匯觀》記載，《松蔭蓮蒲圖》第一段右首原有明宣宗朱瞻基正書款云：「宣德二年五月，御筆賜趙王。」鈐「皇帝尊親之寶」（朱文大璽），可知《松蔭蓮蒲圖》系宣德二年（1427年）五月，時年28歲的朱瞻基為趙王所作。趙王即朱高燧（1382—1431年），為成祖朱棣第三子，仁宗朱高熾之弟，宣宗朱瞻基之皇叔。

明 朱瞻基 連浦松蔭卷 1

朱高燧永樂二年（1404年）被封為趙王。他深受成祖喜愛，受封后並未到藩，而是留居北平。永樂三年（1405年），他獲得全權處理西北軍務之權。

從此畫畫面內容聯繫到這一時期明宣宗朱瞻基與皇叔朱高燧頗為微妙的關係。關於朱瞻基與朱高燧之間的關係，則須從永樂朝那場歷時二十多年的皇儲之爭說起。成祖朱棣是以藩王名分起兵「靖難」而從皇侄建文帝朱允炆手中奪得帝位的。

明 朱瞻基 連浦松蔭卷2

除第三子趙王朱高燧外，成祖朱棣還有三個兒子。長子朱高熾（1378～1425年），洪武二十八年（1395年）被封為燕王世子，「靖難之役」起，他統軍萬餘，挫敗了李景隆五十萬大軍的進攻，守住北平城。永樂二年（1404年）被正式立為皇太子。他為人寬厚，喜好儒學，大異於乃父的尚武性格，故「失愛於成祖」，儲君之位很不穩固。次子朱高煦（1380～1429年），「靖難之役」中戰功顯赫，幾次扭轉了不利的戰局。永樂二年（1404年）被封為漢王，他武藝高強，成祖曾輕許將來改立他為儲君，故其不甘居於長兄朱高熾之下。四子朱高爔早逝。

明宣宗三羊開泰

成祖朱棣易儲之意時有萌發，加劇了兒子之間的互相爭鬥。朝野間普遍認為皇位繼承問題仍有很大的變數，於是文武群臣大體也分成了兩派：一派是文官集團，擁立皇長子朱高熾為皇儲；而原靖難軍諸將則力主改立次子漢王朱高煦為皇儲。兩派之間的明爭暗鬥，構成了永樂時期政治舞台上的一大主要矛盾。

朱高煦 網路圖片

在這場奪儲之爭中，朱高燧「恃寵，多行不法，與漢王高煦謀奪嫡，時時譖太子」（《明史》卷一百十八），對朱高熾造成了嚴重的危脅。永樂七年（1409年）朱高燧第一次謀奪儲位的陰謀暴露，其王府官員被斬。

後來朱高熾依靠長子朱瞻基與文官集團強有力的支持，多次挫敗了兩個皇弟的奪儲陰謀，保住了儲君之位。明永樂二十二年（1424年）八月，朱高熾登基為帝，是為明仁宗。

洪熙元年（1425年）六月，在位僅十個月的仁宗朱高熾突然病重辭世，使一直覬覦皇位的漢王朱高煦、趙王朱高燧又看到了新的機會。朱高煦甚至謀劃在朱瞻基回京路上劫殺於他。

明宣宗畫作 網路圖片

宣德元年（1426年）八月，漢王朱高煦在樂安起兵造反。朱瞻基將朱高煦父子擒拿回京。班師途中，群臣多贊同戶部尚書陳山乘勢移師解決彰德趙王朱高燧的建議，唯大學士楊士奇堅決反對發兵擒執趙王。朱高煦起兵前曾與朱高燧有過聯絡，而朱高燧未曾告發，有知情不舉之大罪。觀歷朝歷代，遇此事關宗室謀逆之案，多以殺戳或囚禁手段解決。

朱瞻基本可從群臣議，擒拿趙王朱高燧以求萬全。而明宣宗卻沒有治罪趙王朱高燧，給了他改過的機會，還在宣德二年，朱瞻基繪製《松蔭蓮蒲圖》贈送給皇叔朱高燧，讓他安心。

《松蔭蓮蒲圖》卷折射出明宣宗朱瞻基對其政敵皇叔朱高燧採取了妥善的處置，成為歷史上的一段佳話。
