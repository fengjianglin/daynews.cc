在鄰國恐嚇下丟棄的土地，官員慌忙撤退，留下一群棄民獨自奮戰

2019-12-16 1,447條評論 3,198次閱讀 258人點贊

俄國在我國外東北的擴張，是一部當地中國居民的血淚史。除了自古就生活在這裡的各個民族外，內地來的漢人移民和商人同樣也是這片土地的主人。

清政府在俄國人的威逼下，步步後退，放棄了對治下臣民的責任，讓大量中國百姓變成「棄民」，使大片國土淪入敵手。更可氣的是，外東北的丟失，不是因為戰爭失敗，而是被恐嚇出來。

第二次鴉片戰爭，原本與俄國人一點關係也沒有，可俄國人見縫插針，趁著滿清前線吃緊，強行要求籤訂《璦琿條約》。清政府一開始拒絕簽字，直到戰爭失敗，俄國人又貼上來，認為自己調停有空，不但把《璦琿條約》落到實處，還逼迫清朝簽下《北京條約》，攫取了更多的領土。

這真是外交史上的咄咄怪事，中立調停一方取得的權益居然遠遠超過了戰勝的一方，俄國人臉皮之厚、野心之大，實在讓人震驚。俄軍早在滿清承認條約之前，就派兵進駐到自己想要的地方，條約簽下後，再趕走中國官府和軍隊，完成佔領。

而滿清軍隊毫無抵抗力，既沒有足夠的兵力，又不認真的備戰。俄軍一來，便倉皇撤退，即使有少數敢戰之士，也不是俄軍的對手。唯一敢豁出命去保護百姓的程德全，還是官府里的一個幕僚，連個官都算不上。

滿清撤退時，居住在外東北的中國人紛紛前來討說法，怎麼把這片自古以來的中國土地給了俄國人。慌忙撤退的滿清官員就安撫他們，俄國人來了以後，對他們的生活不會有影響，原先的營生俄國人會予以保護。大家還能怎麼辦？只好希望這是真的。這群善良的老百姓就這樣被遺棄了。

一開始，俄國人只派軍隊負責這裡的秩序，確實沒有干涉中國人的生活。但是，隨著官員和移民的到來，中國人的平靜生活被打破了。

俄國人熱衷於搶地盤的根本目的，在於搶資源和搶錢，除了收稅外，他們更喜歡把賺錢的生意搶到自己手裡。而海參崴的金礦是俄國人垂涎已久的目標。

1865年，俄國政府通過法令，禁止中國人從事淘金和伐木。這是生活在此地的中國人無法接受的，不能淘金，怎麼養活一家老小？在海參崴東北方向的蘇河谷地，中國礦工開始了武裝反抗。

被尊稱為太爺的于海，是中國礦工的領導人，他聚集起300多礦工，買來武器，驅逐當地的俄國駐軍。俄軍人數只有50多，完全鎮壓不住礦工，還有士兵被俘虜。

礦工很快得到了土匪的支援，這些中國土匪雖然平時打家劫舍，欺負老百姓，但面對洋人時，和礦工同仇敵愾，一起爭取本地區的自治。俄國政府直到兩年後，1868年元旦才調來一百多正規軍，並配備了兩門火炮。

礦工武裝無法抵擋正規軍的進攻，于海及主要的骨幹被俘，礦工鬥爭徹底失敗。俄國人對統治異族有著豐富經驗，他們扶植聽話的李貴出任新一任太爺，使礦工不再反抗，聽命於俄國政府。

就在蘇河谷地礦工鬥爭趨於沉寂的同時，海上青島的金礦工人又與俄軍發生了衝突。這個青島可不是山東的青島，而是海參崴外海的一座小島。中國人早就在這座島上淘金，俄國人起先並不知道。

後來一艘俄國軍艦路過時，登島才發現中國工人。雙方很快從口角、推搡，發展到武力對抗，中國工人有300多，但只有幾把土槍。俄軍有十幾個人，還有一門炮。俄軍死了三個，其餘人逃回艦上，中國人暫時取得了勝利。

為了摸清島上的虛實，俄方派自己扶植的太爺李貴前去打探。沒想到，李貴並不是真心為俄國人服務，他回來後，向同胞散布假消息，說清政府將要派來2000人和一些大炮，準備支援中國百姓。

這個消息傳遍了整個中國人聚居地，倍受俄國人欺壓的同胞們摩拳擦掌，準備起身配合清軍收復失地。4月26日，1000多名中國人突襲一個俄軍據點，抓到了負責指揮的軍官。俄軍調來增援部隊，不得不放鬆對青島的警備，島上的幾百名礦工趁機逃離了該島。

許多中國人向俄國人的聚居地發動襲擊，要求他們離開這裡，並與俄軍展開戰鬥。整個起義持續了一個多月，大批中國人被捕，還有很多人逃回了國內。

民眾自發的鬥爭給俄國人造成的麻煩，遠遠超過曾經駐守在這裡的清軍。無奈的是，國家的懦弱使這些普通小民毫無尊嚴可言，更別說一群被「遺棄」的小民！
