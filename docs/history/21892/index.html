誰是最摳門的？盤點中國古代10個吝嗇鬼

2019-12-19 1,409條評論 3,294次閱讀 580人點贊

吝嗇，方言「摳門」，主要有兩種說法：一是，古時候有一位財主，視財如命，去寺廟進香的時候看到廟門上刷的金粉，他就想摳下來帶走，就有了摳門這個詞。二是，說有個人就連門把手都捨不得裝，開門的時候不得不用手去摳門縫，於是就有了摳門一詞。今天就來盤點一下中國古代的10個吝嗇鬼：

一、王戎

王戎，魏晉時期的名士，「竹林七賢」之一。

王戎的家裡面有長得很好的李子樹，他想要拿李子去賣，但又怕別人得到種子，就事先將核鑽壞了再賣。

王戎的侄子要結婚，作為伯父的王戎只是送了一件單薄的衣服作為賀禮，等到結完婚之後，居然又給要了回來！

王戎就是對待自己的親生女兒也是如此，在女兒出嫁的時候，借給親家幾萬錢，很久沒有還，等到女兒回來省親的時候，王戎就很不高興，直到還了錢之後才高興起來。

王戎喜歡在夜裡和自己的夫人在床上手執象牙籌計算財產，每天都是如此。

這四件事都讓當時的人認為王戎的吝嗇之症已經無藥可救了。​

二、曹洪

曹洪，曹操的堂弟，曹魏名將。在曹操征伐天下過程中屢立戰功，被封為都護將軍。曹洪憑藉著宗親的身份，聚斂了大筆財富而且極其吝嗇，曹操都曾說過，我的財富都比不上曹洪的。

曹丕還是太子的時候，曾經向曹洪借一百匹絹，但是卻被吝嗇的曹洪百般推辭了，所以曹丕一直懷恨在心。等到曹丕登基，正好趕上曹洪的門客犯了事，曹丕就藉機將曹洪下獄，想要殺了他，最後要不是太后求情，曹洪就要為他的吝嗇拿性命來買單了。

三、元宗逵

元宗逵，唐朝人，官至果州司馬。

元宗逵家中有一個婢女死了，於是吩咐管家說：「我家的這位婢女已經在家裡做很長時間了，應該為她找一口棺材入殮出殯才對。但是我剛剛來到這裡不久，家裡還很窮，買不起新的棺材，你去買一口能用的就行。你也沒必要說是給我家買的，只說是你們家裡要買的就是了。」

後來人們都在笑話這個司馬也太摳了。

四、鄭仁凱

鄭仁凱，鄭州滎陽人，唐朝大臣，官至密、毫二州刺史。

鄭仁凱在做刺史的時候，有一天家中的小廝央求他給買一雙鞋，鄭仁凱答應了。不一會兒，就看見門房穿著一雙新鞋走了進來。鄭仁凱故意讓門房脫鞋上樹去把鳥窩掏下來。等到門房上樹後，鄭仁凱悄悄讓小廝偷穿走了門房的鞋。等到門房下了樹，卻四處找不到自己的鞋，最後只好光著腳走了。

事後，鄭仁凱居然還引以為豪，這吝嗇的程度也是沒誰了。

五、周札

周札，東晉將領，平西將軍周處的兒子。周氏是江東最顯赫的武力宗族，這使得當時權傾天下的王敦開始猜忌，於是悄悄聯合江南的其他士族共同討伐周札。

周札性格吝嗇，就是在王敦已經打到家門口的時候，周札仍然捨不得將庫房裡存著的精良的武器裝備給將士，而是分發給將士們劣質的兵器。將士們見周札在這種生死攸關的時候還這麼吝嗇，都不願意再為他戰鬥了。所以剛交戰就潰不成軍，周札也戰敗被殺了。

​六、蕭紀

蕭紀，南朝梁武帝蕭衍的第八個兒子。蕭紀是個極其吝嗇的人，一點點小錢都要算計。

蕭紀奉命經略蜀地，與吐谷渾通商，勸課農桑，積累了大量錢財，本來可以成就霸業的，就是因為吝嗇的小毛病，最後功敗垂成。

梁武帝死後，蕭紀想問鼎天下，於是將全部的錢財熔鑄成一斤重的金餅1萬個，銀餅5萬個，懸掛在軍營上方，激勵將士。重賞之下，確有成效，蕭紀的大軍取得了數次大勝，可蕭紀又捨不得將這些金銀賞給將士們了，這樣反覆幾次，誰都不傻，知道蕭紀只是畫個大餅罷了，最終導致軍心大亂，蕭紀兵敗如山倒，父子二人皆死於亂軍之中。

七、李越

李越，明朝人，曾任蔡州上蔡縣令。

李越這個人吝嗇、小氣是出了名的，很多事情都讓人十分費解。李越他們家一年到頭也不買一回肉，只有在每年的臘月初八祭祀祖先的時候，才會到賣肉的地方去借一塊熟肉用來祭祀，再拿幾個小碟子放上幾枚銅錢，就這樣祭祀祖先。等到祭祀完了後，還得把熟肉趕緊還回店裡去呢，要是晚了，壞了可是要賠的。被世人笑話也無所謂了。

八、韋莊

韋莊，晚唐著名詩人、詞人，五代時期前蜀宰相。

韋莊的吝嗇是有名的。例如，韋莊每次做飯的時候，下多少米都是定量的；炒多少菜都是稱量好的；就連炒肉，少了一片都知道！

相傳，韋莊有一個兒子，在八歲的時候夭折了，入殮的時候，妻子想要給孩子穿上生前最喜歡的衣服，但卻被韋莊扒了下來。只是用孩子睡覺時候墊的舊草席包裹著埋了，而在埋完之後，居然還把草席拿回來了！這種吝嗇已經近乎病態了。

九、張允

張允，五代後漢時期的官員，歷任推官、監察御史、吏部侍郎等職。

張允自己掌握著財政大權，不會多給妻子一分錢。甚至害怕妻子偷偷拿錢出來用，就把所有的鑰匙都掛在自己身上，走起路來叮噹亂響，像個女人似的。

等到後周太祖郭威兵變後，張允被士兵追趕，隨身的鑰匙都被搶了。等到張允回到家後，發現家裡早就被搶空了，結果心如刀絞，口吐鮮血，一命嗚呼了。​

十、湯斌

湯斌，清朝的政治家、理學家，官至工部尚書。

湯斌自從踏入仕途之日起，一直克己省身，就算是做了江蘇巡撫也是過著清貧的生活，每一餐都有一道菜「豆腐湯」。

湯斌不僅對待自己吝嗇，對待自己的家人也是如此，就連他的妻子身上穿的棉襖都是破洞透絨的。有一天湯斌在檢查家中賬目的時候，發現上面記了一隻雞蛋，頓時大怒：「我來到蘇州還從來沒有吃過雞蛋，到底是誰買的？」下人答說是公子。他便把兒子招來，罰跪在庭下，數落道：「你以為蘇州的雞蛋與河南是一樣的價錢？你想吃雞蛋，就回河南老家去……」
