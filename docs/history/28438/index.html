為什麼古代打仗時，總是雙方主將先單挑，而士兵卻站兩邊觀戰呢？

2019-12-20 1,430條評論 3,898次閱讀 2522人點贊

「天下大勢，分久必合，合久必分」，而分裂的過程之中，總是少不了戰爭的發生。戰場之上，刀劍無眼，人們用自己生命來創造勝利的奇蹟，當然，這裡面還靠戰略戰術以及人的戰鬥能力。倘若看過古裝劇的人或者一些古典小說的人，或許會有這樣的疑惑，為何古代打仗的時候，總是雙方主將先單挑，而士兵卻站兩邊觀戰呢？

首先，在某種意義上，這樣的戰鬥局勢是劇情渲染的需要。由於電視劇拍攝過程中，對經費的要求比較高，因此選擇將領單挑的模式可以極大的節省時間。同時，利用單挑的模式作戰，可以更好的凸顯人物的形象，從而達到突出主角光環的作用，並也因此表現出了非常好的戲劇效果。

其次，在現實生活中，其實也有主帥單挑的局面，不過大多都出現在敵我雙方勢均力敵的情況下。

在這種時候，倘若兩軍直接開打，誰也不知道哪一方會勝利，這樣一來的話，傷亡人數是無法避免的，可能會造成巨大的災難。倘若是敵我雙方的實力相差巨大，亦或者是兵數相差巨大的話，則不會出現這樣的情況，畢竟主帥也是很惜命的。

當然，單挑的人選一般都不會是兩軍的主帥，不然的話，倘若主帥死了，亦或者是被突襲了，軍隊將陷入混亂之中。而上場的人，大多都是雙方軍隊中武功比較高的人，比如岳飛、關羽、太史慈與孫策便是處於這樣的角色立場。比如《三國志》中便記載了一段太史慈與孫策單挑的故事：

（劉繇）但使（太史）慈偵視輕重。時獨與一騎卒遇（孫）策。策從騎十三，皆韓當、宋謙、黃蓋輩也。慈便前斗，正與策對。策刺慈馬，而攬得慈項上手戟，慈亦得策兜鍪。會兩家兵騎並各來赴，於是解散。

最後，雙方主將先單挑的話，其實對士氣的提升也有很大的促進作用。

雖然軍隊中的士兵都是為了保家衛國亦或者是為了開疆闢土而奮鬥的。但是他們大多數的人都是被突然徵召的，所以賣命的人其實並不多。而這種時候，士氣就顯得十分關鍵了。

倘若我方主帥殺了對方主帥的話，那麼我方的士兵就會很有勝利的希望，打仗的時候就會熱血沸騰。而敵方的士兵見到主帥出了問題，本就害怕，再看到對面的士兵如此勇猛，精神抖抖，還沒有打仗的時候就已經輸了。

當然，勝敗的雙方反過來也是成立的。也就是說，主帥的交戰對戰局的結果起到了一定的影響。比如關羽斬殺袁超主將顏良，從而解了「白馬之圍」便是如此：

紹遣大將(軍)顏良攻東郡太守劉延於白馬，曹公使張遼及羽為先鋒擊之。羽望見良麾蓋，策馬刺良於萬眾之中，斬其首還，紹諸將莫能當者，遂解白馬圍。

由此不難看出，在戰場之上，主帥進行交戰是戰局的需要，對戰局的勝敗也有一定的影響。每一種現象的存在，都有其一定的意義，我們可以好奇和疑惑，但是倘若放在當時的歷史環境之中，便會發現其不一樣的含義。

參考資料：《三國志》、《資治通鑒》
