童真無邪，趣味深長，透過兒童繪畫作品了解宋代紛繁的社會生活

2019-12-18 129條評論 2,626次閱讀 684人點贊

宋代是中國發展史上的一個特殊時期。一方面，這個時期的經濟、文化水平發展很高，另一方面，這個時期的軍事力量很薄弱。對外戰事中，總是以失敗告終，並且創歷史的先河，先後建立了兩個不平等條約，「澶淵之盟」和「紹興和議」。當然，從中國歷史發展的長河來看，宋代對古代文化發展的貢獻有過之而無不及，繼承了唐朝的文化政策，宋代的治階級飽覽詩書，更是實行了重文輕武的政策，這就為文人階層的發展創造了有利的政治條件。

「市」的發展促使市民文化的崛起，從國家統治階級到社會底層，人們更加關注「人性」的特徵。所以，在這個時期，社會上掀起了一股新的熱潮，從詩詞到繪畫，無不體現人道主義的內涵。

在繪畫領域，宋代的山水畫享譽中外。作為珍貴的物質遺產，張澤瑞的《清明上河圖》細緻地描繪了宋代人們的生活狀態，這幅繪畫作品涉及內容廣泛，細節處理極致入微，堪稱享譽中外的佳作。這一時期，以兒童為題材的繪畫作品也有很多，大大超越了前代後世。兒童畫的內容展示了兒童天真無邪的本性，內容也多是描寫兒童的遊戲生活。觀察兒童的衣著服飾和遊戲方式，我們就可以大致地了解到古代兒童的生活狀態。繪畫作品傳遞了古代人們對兒童的教育理念，這些繪畫作品也是研究古代兒童觀的經典素材。那麼今天，我們就通過宋代的兒童畫來看看古代兒童的生活狀態，了解傳統的兒童觀具體有哪些內容，進一步了解宋代特殊的文化生活。

一、首先，簡單了解一下古代以兒童為題材的繪畫作品的發展狀況

在封建社會早期，人們對社會的關注就主要是成人的精神境界。雖然，春秋戰國時期，諸子百家的思想中，有一些說到了教育，但是並沒有說到兒童藝術創造。所以在這一時期，以兒童為題材的繪畫作品基本沒有，到目前為止，還沒有考察到那個時期關於兒童的專屬繪畫作品。據記載，中國歷史中最早涉及兒童的繪畫作品是在西漢時期，在一些帛畫、壁畫上有看到孩童的蹤跡，但是這個時期繪畫的重心並沒有流轉到兒童身上。因為這些畫作中的兒童不是單獨或者是被專門畫出來的，而是作為一個整體的一部分呈現。到了西漢的時候，繪畫作品中已經有涉及以兒童為題材的繪畫作品，就比如武梁祠的兩端壁畫上就有兒童的專有形象。魏晉時期的繪畫作品受到了社會環境的影響，也主要是以山水、植物、動物等為題材的繪畫作品，以兒童為題材的繪畫作品仍然很少。

隋唐時期，是中國文化發展史上的關鍵時期。得益於國家長期穩定，經濟繁榮，對外交往密切，思想領域也異常活躍。尤其是世俗文化的發展，促進了文人大家們將關注的重心從外在的事物轉移到了人的身上，男女老少都是社會關注的對象，所以以兒童為題材的繪畫作品在唐朝取得了初步發展。宋代更是繼承了唐朝的文化精髓，將古代的繪畫水平發展到了一個新的階段。所以在宋代，「兒童」成為了繪畫中一個重要的話題，先後出現了多位擅長兒童畫的畫家。就比如擅長民間孩童畫的杜孩兒和楊威，是當時出名的孩童畫師。這個時期出現的以兒童為題材的繪畫作品大大超越前代，論起兒童畫的專業性程度要比之前好很多。後代對兒童畫的繼承就沒有突破性的進展了，但是兒童依然是古代繪畫中必不可少的一個話題。

二、為何宋代兒童畫如此豐富呢？

要說宋代兒童畫取得如此高的成就主要得益於三個方面的原因。第一就是宋代特殊的文化政策和經濟發展水平。第二，隨著社會的發展，人們的思想發生了轉變。第三，對前人繪畫藝術的繼承促進了宋代繪畫藝術的發展，主要就是繼承了唐代的繪畫藝術。那麼我們就來詳細了解一下這三個方面的原因。

在前面也提到過，宋代統治階級實行了重文輕武的文化政策，使得文人階層的力量龐大，並創造出了無數的文化財富。文人的力量龐大，遍布國家的各個領域，他們各司其職，在每個領域力圖做到專業化。就比如，宋徽宗辦畫學，分為佛道、人物、山水、鳥獸、花竹、屋木6科。再來，宋代的商品經濟發展水平大大超越了前代，畫作也是進行交換的商品之一。比如有錢人家用的屏風、摺扇、掛作都是需要好的繪畫作品的，高的市場需求促使不同風格的繪畫作品應運而生。

魏晉南北朝時期，佛教思想傳入，在結合先前的儒家、道家思想，文人士大夫掀起了一股關於人性的新思朝，即玄學。尤其是宋代社會文化繁榮，人們的生活也是千姿百態，孩童的世界最為天真無邪。魏晉時期，人們的畫作主要是描寫山水，涉及到其它領域的內容較少，特別是以孩童為題材的繪畫作品更少，並且專業化程度也不高。所以在繪畫領域，宋代就出現了一批專門描繪社會生活的畫師們，他們是民間畫家的代表，與宮廷畫師涉及的領域有很大的區別。對畫作的專業性要求的提高就催生了一些專門描述兒童世界的畫師。

唐朝的藝術文化發展水平已經很高了，宋代的文化發展成就之高的一個主要原因是對以往朝代科學、藝術等文化的繼承。正如牛頓說的「站在巨人的肩膀上看世界。」這樣就會看得更高、更遠。總的來說，中國的歷史文化淵源流長，離不開各個朝代對前人文化的繼承。所以，宋代的文化成就也是中國長期歷史發展的必然結果。

三、品一品宋代著名的兒童繪畫作品

宋代兒童繪畫作品有很多，其中較為出名的就包括《秋庭戲嬰圖》、《百子嬉春圖》、《觀燈圖》等，我們就來看一看這幾幅作品具體描繪的場景，品讀作品所體現的文化信息。

《秋庭戲嬰圖》是蘇漢臣的作品，畫中涉及到的人物主要是兩個孩童，從性別上不好判斷。但是可以看出來，兩個孩子的穿著乾淨、整潔，應該是中等階層家庭的兒童。兩個孩子正在庭院玩推棗磨的遊戲，旁邊有一顆大樹，樹下長了很多的牡丹花。樹前還有一個圓柱形空椅子，上面放了很多物件，就包括紅色佛塔、人馬轉輪、旗盒、八寶紋紙格、玳瑁盤、小陀螺等。這就說明在宋代的富貴人家，孩子的玩具很多，遊戲生活很豐富，這些玩具多是木製品。庭內的古樹與牡丹花一般象徵著和諧美滿之意，尤其是在古代社會中的富貴人家，這兩種植物很常見。

《百子嬉春圖》也是宋代的名畫之一，也是畫家蘇漢臣的代表作。現在這幅畫被北京故宮博物院收藏。畫面的內容是描繪了一百多名兒童，他們三五成群，一起玩耍。有的兒童在一起說話、有的在一起玩木偶、有的靜坐著、有的在跑、有的在走、每個孩子都有自己的事情做。仔細看，還發現有的小孩在扮演獅子舞，整個畫面十分精彩。這個畫面就反映了古代社會的孩子們玩耍的對象很多，他們可以選擇玩耍的方式也很多。他們的穿戴不一樣，服飾多是灰白顏色的布料。他們的頭型很單一，分為頭頂留髮和耳塞兩邊留髮兩種。孩子們的年齡相仿，個頭也差不多，整個畫面看起來十分和諧。

《觀燈圖》是南宋時期李嵩的代表作品。李嵩少年時本是木工，後來成為宮廷畫家李從訓的養子。所以得到師傅的真傳，李嵩在繪畫領域做出了一定的成就。他的《觀燈圖》描繪的是宋代的人們在元宵時節賞燈的畫面。畫面中有五位美麗的女子或是婦人，女子們有的彈琴、有的吹簫、有的低頭不語或是欣賞音樂、有的站立著。元宵節的燈光異常美妙，照耀在人們身上，泛出了微微的黃色光芒。畫面的右上方有兩個小孩打著燈籠，竊竊私語。左下角兩個孩子一邊往前走，一邊看著身後的小象，顯示出了強烈的好奇心。雖然畫面中的人物不多，但是小孩在這個時間出現就證明，古代的小孩生活很豐富，佳節之期，他們也可以參與到節日中去，創造屬於自己的快樂。繁榮熱鬧的場面也說明了宋代的文化氛圍很濃烈，婦女們不再局限於家庭之中，他們也可以走出閨閣，參與到外面精彩的活動中。

四、品讀宋代兒童畫的特點

宋代的兒童畫取得了很大的成就，但是被保留下來的作品很少。宋代的兒童畫家主要有四個著名的人物，北宋末期，劉宗道和杜孩兒的嬰兒畫作很多，因此享有一定盛名，但是他們流傳下來的作品很少。到了南宋，蘇漢臣與李嵩的嬰兒畫作也響徹一時，到現在為止，蘇漢成的畫作被保留了很多，目前他的多幅畫作被北京故宮博物館收藏。古代以兒童為題材的繪畫作品很多，但是這類作品在當時沒有得到很高的重視，所以，相應的畫作也沒有很好的保留下來。到了蘇漢臣之後，人們對於兒童題材繪畫作品的珍視度提高，所以從這以後，相應的作品也被保留了下來。

從兒童畫的內容來看，古代的兒童繪畫作品多是反應兒童遊戲的內容，接著就是他們的生活，學習次之。特別是從蘇漢臣的嬰戲圖來看，兒童通常是在不同的場景下進行不同的遊戲活動，他們的生活狀態很豐富，精神飽滿，智趣的孩子們總是能在不同的環境中找到自己的樂趣。單單從畫面中看，孩子們的的壓力很小，相比於現代，他們自己探索樂趣的時間更多，因為在很多的貧困人家，孩子上不起學。孩子的生活狀態和父母的生活狀態是不一樣的，所以古代的兒童繪畫作品都是描繪較小年級的孩童，這個時候的他們輕鬆愉快，不用承擔過多的家庭責任和社會壓力。

從古代兒童繪畫作品的精神層面來講，多是反應孩童天真爛漫、純潔無邪的精神狀態。我們會發現，嬰戲圖中描繪的兒童多是積極正面的一面，很少有畫家來描繪貧困人家孩子的狀態。這種情況，主要受到了世俗文化的影響。在唐宋時期，國家總體實力強大，所以，人們更多讚揚的是社會積極的一面，這也是統治階級想想看到的。隨著商品經濟的發展，只有一些有錢人家才能出錢請畫師給自己畫畫。為了生存，畫師們也要配合這種形式。所以，在社會上出名的兒童畫作多是描繪有錢人家的孩子遊戲、玩耍的場面。

五、那麼宋代的兒童畫作有什麼歷史意義呢？

首先，宋代的兒童繪畫的發展豐富了古代藝術內容與形式。在宋代之後，人們對兒童的描繪增加了。兒童繪畫作品多反映了孩子天真無邪的本性，這就為後面發展起來的現代兒童觀提供了歷史依據。特別是清朝理學思想發展成熟，兒童觀在這個時期也得到了一定發展，多少受到了宋代的兒童繪畫的影響。

從這個層面來講，宋代的兒童繪畫促進了新的兒童思想萌生，在某些方面推動了歷史的進步。古代遺留下來的兒童繪畫作品是我們歷史文化的一部分，這些作品是一定時期文化積澱的結果，也反映了那個時期人們的社會生活狀態，所以對於現在研究古代文化有重大意義。比如通過這些繪畫作品可以大致了解宋代人們的服飾特點和宋代孩童的生活樂趣。最後，不得不說，這些作品能夠經歷長久的歷史並且被保留下來，說明我們一直具有民族文化共識，這些作品也是我們珍貴的歷史文化遺產。
