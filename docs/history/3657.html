甘肅清末翰林，在陝西當知縣，公開受賄賂，卻被老百姓大加讚賞

2019-12-02 0條評論 1次閱讀 0人點贊

摘要

甘肅清末翰林，在陝西當知縣，公開受賄賂，卻被老百姓大加讚賞 “齊家治國平天下”是文人的傳統理想，也是


甘肅清末翰林，

在陝西當知縣，公開受賄賂，

卻被老百姓大加讚賞

「齊家治國平天下」是文人的傳統理想，也是建功立業的三個階段或步驟！然而，齊家容易，治國難，平天下就更難了。尤其在亂世之中，一個文人要恪守自己的理想，為老百姓做點實事，自然就非常難。

1.考中了舉人，卻無錢考進士

清末，甘肅甘谷，出了一位文人。他由舉人而進士，由進士而翰林。最後，有外放為一人地方官，成為七品芝麻官，當了陝西一個知縣。縣官，雖然權不大，但只要為老百姓著想，總能幹出不少實事。這位甘肅才子，就是王海涵。

王海涵，甘肅甘谷縣人，生於1858年，字鏡潭，號南坡。1890年考中進士，1899年王海涵外放陝西高陵知縣。在知縣任上，辦案遇到富豪時，必大肆索賄。然而，他卻是貪贓不枉法，雖收禮了，卻仍舊以理斷案。當地人稱「硬贓官」。

清末，陝甘各地戰亂不斷，民不聊生。大批民眾流離失所。儘管，經過左宗棠等人大力經營，但陝甘各地元氣，恢復的非常慢。王海涵家境非常貧窮，家裡連房子都沒有。他父親在通渭榜羅鎮做小買賣，母親在家為富家小姐織裹腳布為生。寒門出英才，他讀書非常用功，先到縣上宿儒王端明門下求學。後來，他又到天水的隴南書院任其昌門下就讀。可惜的是，後因父親去世而中斷學業。

不過，有才華的人無論在哪裡都無法被阻擋。他在考秀才時，就受到了當時陝甘學政吳大徵的賞識。吳大徵是著名的金石學家，也是著名書畫家吳湖帆的爺爺。1888年，王海涵考中舉人。這時，他家境依然不好，連進京趕考的費用都沒有。

天水到北京數千里路，沒有錢怎麼去？就在他發愁的時候，武山的武舉人康國棟幫助了他。康國棟是光緒十五年(1889年)乙丑科武舉人，為人豪爽，熱情資助了王海涵進京趕考。王海涵考中光緒十六年(1890年)庚寅恩科進士，為二甲第109名，選為翰林院庶吉士。

2.十年京官，要求降級外放

翰林院是明清兩代一個比較特殊的機構，是為國家儲備人才的地方，素有「非翰林不得為相」的說法。從翰林院出來後，王海涵任刑部山東司主事，品秩為正六品。此後，他在京城差不多待了十年。

實際上這是苦熬的十年。翰林出身做京官本來是科甲出身的人夢寐以求的路，因為有機會接近皇帝，能與朝廷重臣朝夕相處，且主考、學政、御史，也多從翰林里考選，應該說前途是很廣闊的，但這需要「熬」和「忍」。

「京城居大不易」，房價高，物價貴，還要維持體面的生活。更不堪忍受的是，當時官場充斥著兩大派系之爭，一個是清流派與洋務派的黨爭傾軋；另一個是以籍貫、裙帶結為幫派的相互援引。這都讓他深感失望，也對京官的前途失去了耐心和興趣。在北京苦熬十年後，終於熬不住了，主動申請外放。

光緒二十五年也就是1899年，王海涵外放為陝西高陵知縣。儘管品級由原來的正六品變成了正七品，但陝西高陵距離長安不遠，在西北也算是比較好的縣，他還是比較滿意的。誰知，到高陵任職沒有多長時間，一個空前的考驗就擺在了他面前。

3.招待慈禧，題字換來招待費

啥難題？要接待西行的慈禧太后。1900年八國聯軍入侵，史稱「庚子事變」。看著「八國聯軍」逼近北京，慈禧太后匆忙帶著光緒皇帝，逃出京城。一路忍飢挨餓，輾轉西行，最後直奔西安。這些「難民」浩浩蕩蕩路過高陵，如何招待呢？

接待需要大筆錢的。錢從哪兒出呢？多年戰亂，才安定不久，老百姓真無能為力,肯定是指望不上了。咋辦，只有找大戶人家想辦法了。

好歹，王海涵在北京當過京官。如何運作，還是有些想法。這不，他想出一個辦法，找當地的土豪們，發動他們向慈禧太后報效，以獲得慈禧太后的墨寶賞賜。其中有一個叫安武氏的年輕寡婦，家資巨萬，拿出了大量白銀用於接待「兩宮」，使得這次接待得以順利敷衍過去。安武氏也被慈禧太后認作了「乾女兒」。

實話實話，這次接待也是王海涵飛黃騰達的一個好機會。只要他給太監王公們送錢，再在慈禧面前多露臉，完全可以把接待轉換成生產力，讓自己加官進爵。但他卻拿不出錢，也不願意去行賄太監、王公們，儘管付出了很大禮器，只能算是「及格」。好歹沒有遭貶。而當時的陝西巡撫岑春煊，卻因為接駕有功，很快被提升為一方大吏。

甘肅省甘谷縣人，高級經濟師詩詞家甘肅省楹聯學會副會長廖海洋對王翰林的事迹進行了長時間的搜羅整理。到現在，基本上能勾勒出個大概了。

王海涵想盡辦法，招待慈禧一行，度過了外放的第一道難題，也在高陵紮下了根。

不過，當縣官主政一方，可不容易，雖然掌管著縣內財政人事司法等多項大權，但國家任命的官員，就那麼幾個人，大量的行政事務要靠胥吏來完成。這些人，往往是盤剝老百姓的主力。他們最喜歡老百姓打官司，不論輸贏，他們都能乘機敲詐錢財。這就是縣令的難題了，既要依靠胥吏來維持縣衙運行，又要減輕民眾負擔，同時還要為貧苦老百姓說話。縣官就要在三者之間找一個平衡點。

出身貧寒的王海涵，想為窮苦老百姓做點事。可是，沒有錢，怎麼辦呢？思前想後，他想了一個辦法。凡是富豪來打官司，必定大索其銀兩。這樣一來，豪紳以為縣官要為他們說話。可是，王海涵卻是以理斷案，絕不枉法。有的豪紳，準備狀告王海涵，最後也不了了之。因為，王海涵收來的錢，並沒有據為私有，而是一部分用來增加胥吏收入，以免他們盤剝百姓，一部分用來縣裡公益事業，比如修橋，救災，修文廟，旌表節烈等等。這下，豪紳們有苦難言。

王海涵在高陵縣擔任了兩屆知縣，前後長達十年之久。其間，高陵縣的戶籍、財政考核多次獲得全陝第一。《高陵縣誌》記載他被高陵民間稱為「硬贓官」。「硬贓官」是對王翰林的最大褒獎，也是那個吏治混亂腐敗年代中，在現實和理想之間找到的一條妥協之路，也算是一種生存智慧吧。

4.隱居故里，成一介布衣


宣統二年，也就是1910年5月，王海涵調任涇陽知縣，剛上任三個月，他母親在甘谷老家去世，他辭官回家守孝。離職時,高陵、涇陽兩縣百姓自發送行，分別贈送一千兩紋銀，王海涵對此非常感激，但謝絕了百姓的好意。高陵縣還專門為他立碑以紀念其政績。

不久，武昌起義爆發，次年清帝退位。他從此隱居故里，成為一介布衣，與甘肅名流劉爾炘、張世英、任承允、哈銳等人過從甚密。

王海涵雖然當過知縣，但過的依舊清貧。「我不辦年貨，也不會把我留在年這邊。」這句甘谷民間俗語，說的正是甘谷清末翰林王海涵。

王海涵家中很窮，沒錢置辦年貨，就用自嘲和曠達的口氣說出了這句話。一百多年來，每當年前置辦年貨時，父老鄉親們總不由自主地說起這句話，也是對這位先賢的紀念。

以前，甘谷縣城東關街上有一個很氣派的院落，人們俗稱「王家廳」。這就是王翰林家當年的宅院和祠堂。大門非常氣派，磚雕精美，還懸掛著劉爾炘題寫的對聯，帶著厚重的人文氣韻。這是個有錢人家啊！怎麼說，王翰林還有置辦不起年貨的時候？


5.守護桑梓，亂世文人的擔當

民國年間，甘肅都督趙維熙是與他同榜的進士，禮聘他重新出山到省垣蘭州做事，他婉言謝絕。在甘谷他以保境安民、造福桑梓為己任，擔任了縣內鄉民自治機構的縣總紳士，調解糾紛，應付兵匪。還在甘谷成立了城防局，他任總辦，主持整修城牆。甘谷的第一所小學「伏羌縣高等小學校」也是王海涵捐資成立的，他擔任校董，聘任本縣舉人李蔚起為首任校長。

民國十一年(1922年)王海涵在甘谷家中病逝，享年64歲。

作為翰林，王海涵的文才自是一流的。只是他們那一代讀書人，總是秉承一個思想，居廟堂之高則憂其民，處江湖之遠則憂其君。視書畫為小道，安邦濟世才是他們的目標和大道。王海涵的書畫文章多不留底稿。經過近百年的水火兵匪，他的作品已經非常少見了。

​

他在甘谷大象山太昊宮留有一副對聯：紹荒開泰運，厥後有石子訪道，伯約懷忠，問化育根源，發矇在炎黃顓嚳而上；望古動幽情，其下則煙火成鄰，桑榆布蔭，幸邃初風景，復睹於金戈瓦礫之餘。他送給某醫生的對聯：「守虎蟠龍傳蘇董；出蛇走獺尊華王。」從這些僅存的作品，依然能夠很清晰地看到他的翰林才情。

在科舉時代，隴上走出過多位翰林。比如蘭州的劉爾炘，他以主持修建五泉山而聞名隴上，也早已為人們所熟悉。但也有一些翰林，不為人所知,漸漸地被歲月的塵埃所遮擋，留下一個個或熟悉，或陌生的名字，令人懷念。
