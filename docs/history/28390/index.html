秦穆公之所以能稱霸春秋一時，少不了一對父子的相助！

2019-12-20 547條評論 1,926次閱讀 570人點贊

《生於憂患死於安樂》中記載「舜發於畎畝之中，傅說舉於版築之間，膠鬲舉於魚鹽之中，管夷吾舉於士，孫叔敖舉于海，百里奚舉於市」。在這段記載中著重點明了六個人的發跡之所。今天給大家介紹的這對父子就是百里奚和他的兒子孟明視！

百里奚這個人想必大家都耳熟能詳了。百里奚本身為虞國人，但自幼家庭貧寒，所以百里奚一直遊學在外以乞討為生。後來學得一身本事之後返回虞國效力。百里奚憑藉自己一身本事當上了虞國的大夫，但百里奚好日子沒過多久，晉國晉獻公以假道伐虢之策滅了虞國。百里奚因為虞國國君沒有重視他的意見，所以也當上了亡國之臣。之後百里奚更是被晉獻公抓住以陪嫁奴隸的身份送到了秦國。因為當時秦穆公正好想和晉國聯姻。所以在秦晉結好之時，百里奚就成為了女方的陪嫁奴隸。百里奚不堪受辱所以就想逃回故鄉。但半路上又讓楚國人給抓住當了牧奴。可以說百里奚運氣簡直差到了極致。但正好在這個時期秦穆公聽說了百里奚很有本事，所以用五張黑羊皮從楚人手中贖回了百里奚。百里奚這才得逃大難，並且以才華顯於秦國。

百里奚被秦穆公贖回之時已經七十多歲了。但秦穆公絲毫沒有輕視百里奚。在百里奚主持秦國朝政之時，百里奚主張教化百姓，用我們現在的話來說，百里奚就是想要秦國百姓過上小康生活，並且精神文明建設還要進一步提高。秦國百姓自然十分的愛戴百里奚。據史料記載，百里奚去世後，秦國上下不論男女老幼盡皆悲痛欲絕。以此哀悼百里奚。足見百里奚的富民政策做的多麼到位。除了富國強民之外，百里奚還協助秦穆公對外拓展領土上千里。使得秦國勢力迅速崛起，為秦國拉開了崛起的序幕。也是由於百里奚的努力才使得秦國稱霸於諸國之中，成為了春秋五霸之一。而秦穆公五張羊皮換名相也成為了一樁美談。

俗話說的好「虎父無犬子」。百里奚這種人傑的後代自然也是人中之翹楚。孟明視，也就是百里視。其具體生卒年不詳，但從歷史史料記載，孟明視應該出生於百里奚貧寒時期。史書記載，孟明視年幼時期，百里奚就為了謀求一官半職，常見在各國奔波。孟明視是他母親一手拉扯大的。或許是從小缺乏父愛的關係。孟明視從小就喜歡舞槍弄棒，喜好和人好勇鬥狠。但也因為孟明視喜歡舞槍弄棒的原因，也真讓他琢磨出一點門道來。後來孟明視和母親因為貧寒等原因流落到秦國，這才找到了數十年沒見到的父親百里奚。而此時的百里奚已經貴為秦國相國了。秦穆公聽說百里奚兒子的事情也十分高興的接見了孟明視。因為看到孟明視孔武有力，武藝不凡，所以拜孟明視為大夫。孟明視也正式出仕於秦國。和父親一同效力於秦穆公。

和其他名將所不同的是，歷史上絕大多數的名將都是閃閃亮亮的登場。體體面面的落幕。孟明視則截然相反。孟明視剛效力秦穆公之時，可以稱得上「屢戰屢敗」。但這一點也可以理解，畢竟孟明視經驗缺乏，需要積累經驗。而秦穆公也非常開明，甚至大度的難以置信。孟明視在連戰連敗之後秦穆公還是對孟明視重用有加。當然了，這也和百里奚有一定的關係。

公元628年，秦穆公以鄭國不修德行為由準備進攻鄭國。百里奚怎麼勸秦穆公都毫無用處。秦穆公決定以孟明視為將攻打鄭國。孟明視倒是興奮不已，甚至他還覺的自己的父親老了，開始瞻前顧後了。孟明視滿腦子想的都是打仗，他對於大局、戰略實際上並不完全明白。果然孟明視攻打鄭國的大軍在路上碰上了鄭國的牛販子弦高。弦高一方面用計拖住了孟明視大軍，另一方面派人通知了鄭國秦軍要來的事情。結果孟明視看到鄭國早已有了準備，所以認為再繼續攻打鄭國肯定也占不到什麼便宜。但孟明視又怕回去被秦穆公責備。所以回軍路上順便接了滑國，然後拉著戰利品大搖大擺的準備回國。

但孟明視忽略了滑國是晉國的邊境城邑了。當時晉國晉文公剛死不久，晉國人理所應當的認為孟明視這是在挑釁晉國。所以晉國大軍就埋伏在孟明視必經之地崤山地帶，然後偷襲一舉擊敗了孟明視。孟明視也被俘獲，晉襄公十分氣憤想要殺掉孟明視以他的人頭來祭祀。但由於秦國和晉國之前一直有著姻親關係。而死去晉文公的妻子就是秦穆公的女兒。所以她給孟明視求情，希望晉襄公釋放孟明視，理由就是孟明視兵敗必定會遭受秦穆公的責罰。何必冒著得罪百里奚的風險殺掉孟明視呢？結果晉襄公被一頓忽悠就相信了她的話，因此孟明視算是逃過了一劫。

回到秦國的孟明視並沒有受到秦穆公的懲罰。從這裡也可以看出秦穆公的擔當。秦穆公認為是自己的戰略錯誤才導致了這次的戰敗。所以他並沒有責怪孟明視，反而還好生安撫了孟明視。孟明視深感秦穆公的仁慈寬厚，所以對秦穆公也是感激涕零。孟明視休整兩年之後，再次請求對晉國進行雪恥作戰。秦穆公答應了孟明視的請求。隨即秦晉之間爆發了「彭衙之戰」。彭衙之戰中孟明視再次敗給了晉軍。孟明視再次失敗後羞愧難當。他把自己關在囚車裡請求秦穆公處罰他戰敗的罪責。但秦穆公再一次原諒了孟明視，並且勸慰他打起精神奮勇直上。

經歷了兩次失敗的孟明視這次終於開始潛心研究自己兩次戰爭的得失。他開始注重軍隊的訓練，士兵戰鬥意志的提升。並且開始改正自己急功好利的性子。力爭讓自己變得更加強大。公元624年，孟明視再次向秦穆公請戰，孟明視表示此次戰役再不勝利就提頭來見。秦穆公應允了孟明視的請戰要求。孟明視在出征前為了激發士兵鬥志，提前把士兵的撫恤金都給發了。後來在孟明視大軍渡河之後把所乘坐的戰船也都燒了。孟明視斷絕了自己的所有退路。就是要告訴士兵們，他們已經毫無退路了，要麼戰勝晉國，要麼全軍而亡。孟明視所做的一切還是有效果的。這一次孟明視和晉國大戰，孟明視連下晉國數座城池，晉國被孟明視打的根本不敢出城。晉國無奈屈服於秦國。此次戰役被稱為「王官之戰」。

「王官之戰」後，原來的霸主國晉國由於被秦國所擊敗。秦國也取代了晉國的地位成為了新的霸主。之後在百里奚和孟明視父子的努力下，秦國征服了西戎，成為了西戎的霸主。這一切都是百里奚和孟明視的功勞。《左傳·文公三年》記載「秦伯伐晉，濟河焚舟，取王官，及郊。晉人不出，遂自茅津濟，封餚屍而還。遂霸西戎，用孟明也。」由此可見，百里奚父子對於秦國的稱霸做出了多大的貢獻。
