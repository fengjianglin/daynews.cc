李世民欲殺外甥，姐姐求情，李世民只回了8個字，姐姐無言！說起唐王朝的建立，大家就會想起李世民，雖然唐朝是在李淵的時候建立的，但是最有功勞的還是李世民。之前在太原的時候，李世民就是率先起頭的一個。所以唐朝之所以能成功，就是因為李世民的存在。

李世民影視劇照

有人說李世民是一個非常善良的人，可是李世民並不是一個善類，他還有鐵血和無情的一面。如果他沒有自己的手段，他怎麼會成功？後來又怎麼能坐上皇上的寶座？所以說李世民還是有狠的時候，面對自己兄弟的無情，自己父王的昏庸。於是李世民就決定逼自己父皇禪位，可是自己的兄長和弟弟是攔路石，就狠心殺死了自己的哥哥。

影視劇照

在李世民繼位以後，因為他善於聽取大臣的意見，還重視農業，一度開創出盛世的局面。所以在大家的心裡就覺得他非常的好，是一個好皇上。可是他的外甥卻謀反。於是李世民就要殺掉他，面對自己的姐姐求情，李世民只回了8個字，姐姐無話可說。

李世民影視劇照

大家都知道，李世民的父親李淵有很多的孩子。李世民在家中男孩里排第二，他上面有一個哥哥、五個姐姐，他的弟弟妹妹們更別說有多少了。李世民的的姐姐嫁的都是非常厲害的人物，所以當時李淵準備起義的時候，就通知自己的兒子、女婿們做準備，回應自己。當時李世民的五姐嫁的是趙慈景，在李淵讓他們秘密去太原的時候。因為趙慈景有私人事情，就沒立刻趕去，於是就被抓了。

秦叔寶，程咬金，李世民劇照

可是在李淵成功拿下長安以後，就救下了自己的女婿。可是雖然救下了他，在後來的時候，他還是戰死了。李世民就帶著他們的兩個兒子守寡，後來李淵覺得自己的女兒可憐，就封她為長廣公主，又為她找了一個丈夫。於是桂陽公主就嫁給了楊師道，還把自己的兩個兒子帶了過去。雖然嫁到了楊家，但是因為兩個兒子都是趙慈景的，所以大兒子就叫趙節。楊師道對趙節也非常好，所以在楊師道的教育下，趙節在文武上都非常優秀。

而且這個趙節和李世民的大兒子李承乾關係很好，跟李世民的兒子李泰關係不太好。在李世民後期的時候，李承乾雖然是長子，但是不論在哪方面都比不過李泰。兩方也一直處於爭奪帝位的局勢，但是李承乾覺得自己身為太子，但是受到了李泰的威脅。他不想自己的位置被李泰奪走，於是就準備先聲奪人。但是他的身邊有三位非常不錯的人物，其中一個就是趙節。

在身邊人的幫助下，他們訂下來一條計策，就是先讓李承乾裝病，然後引誘李世民來。李世民一來，太子府中埋伏的兵馬就會抓了李世民，然後逼他退位。第一是名將侯君集，第二是杜如晦的兒子、駙馬杜荷，第三就是表弟趙節。可是身為皇上的李世民就是這樣走過來的，他怎麼會不知道這些小技。於是李世民就派人去抓了這些反賊，其中就有趙節。

可是趙節畢竟是自己的外甥，他父親畢竟是為國家戰死的，他的母親又是自己的姐姐，李世民非常的為難。他就想如果殺掉趙節先跟自己的姐姐說一下，可是他的姐姐卻跪下求情，不想讓他殺了趙節。面對姐姐的求情，李世民只說了8個字：天下之道，不敢違也！這簡單的8個字，讓姐姐無言以對。
