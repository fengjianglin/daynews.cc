史筆記 |《史記》為什麼不從三皇寫起？（下）

2019-11-30 0條評論 21次閱讀 0人點贊

摘要

太史公又接著說：我曾經向西到過空桐山，向北經過涿鹿，向東到達大海，往南在長江、淮河駕舟，也到過一些老者們口中常常提起的是黃帝、堯、舜遺迹的地方，那裡的風俗教化當然是有所不同的，但總的來說沒有違背古籍記載的說法，比較接近孔子的說法。

太史公又接著說：我曾經向西到過空桐山，向北經過涿鹿，向東到達大海，往南在長江、淮河駕舟，也到過一些老者們口中常常提起的是黃帝、堯、舜遺迹的地方，那裡的風俗教化當然是有所不同的，但總的來說沒有違背古籍記載的說法，比較接近孔子的說法。我閱讀了《春秋》、《國語》，能發現其中引用《五帝德》、《帝系姓》的地方很多，只是他們沒有深入考察，其實他們的記述都不虛妄。《尚書》記載的缺失已經很久遠了，它所散失的一些內容就常常可以在其他記述中看到。如果你是見解膚淺見聞寡陋的人，而不是喜好學習，深加思考，從而心知其中的用意的人，我就很難給你講清這其中道理。我把堯以來的史實按照次序一併論說，選擇那些記述的言辭特別規範和可靠的，編成《五帝本紀》作為全書的首篇。

我們可以看到，那麼三皇對於我們，甚至對於當時的司馬遷來說，只能認為其是傳說，其中的原因，一則可能是三皇時代根本沒有文字甚至語言來記錄這些，或者當時可能連記錄的想法都還沒有產生，二則如果認為有，可能因為保存條件差，遺失了。但是無論是什麼原因，總之，在司馬遷能看到的值得相信的史料里都是沒有關於三皇的完整記載。在這些傳說當中，有些稀奇古怪，有些未經考證，其實這些所謂的傳說呢，就是口口相傳，就是你告訴我，我再轉給他，依靠講故事的方式。傳遞的精確性是不大一樣的，有人添油，有人加醋，所以幾經輾轉之後呢，跟原本的樣子可能就不太一樣了。如果把它寫進這個歷史當中就不太靠譜了。那麼，他寫史記是寫歷史書，寫歷史書就不能太多地引用傳說。

如果三皇只是傳說，那麼說五帝就很靠譜了？其實五帝也不那麼特別靠譜，比如說，我們舉個例子，五帝當中的帝堯，《尚書》也是從堯開始記錄的。帝堯是在位70年之後才發現了帝舜，然後他又讓帝舜的攝政20年，等帝堯退休了之後，又過了28年才崩，所以加起來前後這麼一算，即便帝堯生下來就繼位了，前後也是118歲啊。而且我們發現上古的這些大神普遍很長壽。雖然說並非絕無可能，畢竟有點超乎我們的想像。總而言之的這些事兒並不是那麼靠譜，但是由於這個傳說很久遠了，至少有文字這麼記錄，走訪之後大家也都這麼說，那司馬遷也就把這事兒擱在史書裡邊了。但是，這一取一舍之間，舍掉三皇而取五帝，恰恰說明了作者一種儘力而為的一種態度，把明顯的不靠譜去掉，但對有爭議的部分保留，以為後世。

那麼，傳說中「三皇」到底是哪三皇？有的說是女媧，伏羲，神農；有的說是燧人，伏羲，神農；有的說是伏羲，神農，黃帝。相對來說比較確定的是倆人，一個是伏羲氏，第二個是神農氏。基本上不管哪種說法，裡邊肯定會有這倆人。但第三個人呢，就莫衷一是了。

那就先說說這倆人。伏羲氏，是中華民族的人文始祖，《山海經》里講：「雷澤中有雷神，龍身而人頭」，伏羲就是龍身人頭的樣子。他的主要功績，第一是教會人們做網打魚。第二是變革婚姻習俗，使不同氏族之間才能結婚，同一氏族之間男女不能婚配。這就結束了長期以來子女只知其母不知其父的狀態。也就是說，伏羲實際上是意味著中國從母系氏族向父系氏族開始轉變的人物。第三創造書契，用於記事。使所記錄的內容具有可讀性和可傳遞性。那時候估計也就是結繩記事的時代，任何一個民族的早期的形態都是漁獵，就地取材，能會一項簡單技能，必將被同類視作天神。他還發明了八卦，這東西其實很了不起，它應該是上古人類對自然界規律的一種質樸的認識和總結，在當時來說，這種抽象的類似哲學的認識是很超越的思想創造。後來周文王把伏羲八卦推演成了文王八卦，變成了八八六十四卦，還寫了卦辭和爻辭，這個這就是易經，易經加上易傳就是周易。周易這麼學問很深，孔夫子也是晚年的時候才覺得有能力開始學。易中天先生有本書《中國智慧》，裡面講八卦，簡單易懂大家可以看看。

第二位是神農氏，我們都知道神農氏嘗百草，據說神農氏是透明的，五臟六腑都能看見，這吃點什麼東西，有毒的，一下肚就看出來了，通過自身的努力，弄出來好多草藥，能給人治病，一副老中醫的架勢。他為了找到更多的草藥，就伐樹搭梯子，不斷的向山的更高處攀爬，這就是今天神農架的來歷。再者，你聽他這名兒可以知道，他對於種莊稼應該挺在行的，他發明了耒耜，一種像犁一樣的農具，教授人們進行農業生產，神農氏是代表著中國從漁獵採集社會向農業社會轉變的人物。對於種植業技術的掌握，才有能力使上古的原始人類慢慢的從採摘游牧的狀態逐漸過渡到了定居的生活，這對人類文明的發展有著至關重要的作用。神農氏也很可能因為會種莊稼的緣故，這個家族就被推為這個天子的家族了。你想想啊，在別的部落還在游牧生活的時候，他已經可以種植了，說明他已經有能力將生命的延續和自然的賜予分開，不在靠天吃飯，這就是更先進的生產力的代表，必然導致了其部落的人口增長快於其他部落，實力也必然凌駕於別人之上。這也是後世稱之為神農的緣故。

那麼第三位，我更相信是女媧氏，也就是伏羲的妹妹或者兄弟（因為有人認為他是個男的），所以他也是蛇身人頭。當然，我們更傾向於她是一位女性，因為她最著名的事迹就是造人。她照著自個兒的樣子捏小人，就是我們現在的人類，後來不耐心了，弄根繩子沾著泥甩，甩成什麼樣兒是什麼樣兒，泥點子就變成人了。那些照著樣用手捏的人，長的就順溜，就享福，拿泥點子甩的人，那長得就比較謙虛，就得吃苦受累。

還有傳說，這另一個是炎帝的，有說燧人氏的，就是那個搞鑽木取火的。還有說是共工氏，發洪水的那個，還有人說把盤古也算進去。雖然這個意思很好，但是我們說，盤古這個傳說呢，是隋唐的時候才出現的。總而言之，各種說法莫衷一是，相信中國人都有所了解這麼一些傳說。

總之，我們可以看到的是中國人對自己祖先的認識過程，會認為他們是一些掌握了領先於那個時代的特殊技能的人，首先他們是人，因為掌握的某種技能，帶領了部落發展，在後世的傳頌中才把他們敬為神。這些技能本身表達了一種人們對自己起源的美好認識，和對美好生活孜孜不倦的追求。我們不能把上古神話完全看作是先人的杜撰，這只是他們一種有限的表達方式。所以，中國人從來就是無神論者，只相信實實在在的人。誰能帶領人們奔向幸福，人們就會擁護誰。
