匈奴人為何抓到漢人俘虜後不殺死，反而優待他們在本族娶妻生子？

2019-12-08 0條評論 20次閱讀 0人點贊

文/炒米視角

匈奴人不只是襲擾漢朝邊境，除了財物和牛羊牲口外，有抓漢人俘虜帶回去的習慣；而且攻打東胡以及其他西域國家，也會擄掠除了牛羊和財寶以外，大規模掠奪人口。掠奪俘虜回去的自然會慎殺，因為這裡涉及到一個古代戰爭中的重要戰略資源問題。

匈奴人和很多後來的游牧民族不一樣的地方在於，他不是一個單純種族的民族，他的族群里既有白種人，也有黃種人，更多的是混血。造成這個現象的主要因素就在於匈奴對戰爭中，人口資源的重視。

古代中原戰爭中，土地和人口是兩個重要戰略資源。戰爭加劇了土地的荒蕪和人口的喪失，因此有人有地，往往是決定某個集團是否持久的重要因素。

在三國時期，劉備被打得東躲西藏，丟失地盤，但是他每次都會不忍丟棄百姓，扶老攜幼逃離到另一個地方去。不是因為真的百姓覺得只能跟著劉備才能活下去，而是那些人口是劉備東山再起的唯一資本。

劉備取漢中，曹操知道守不住，於是把漢中的百姓全部強行遷走，就一個空的漢中給劉備。因此曹操得漢中之人口，劉備得漢中之土地。但是後來如何守漢中，兵力不夠一直困擾蜀漢，魏延數學好，懂得利用地形合理配置微不足道的兵力守了10幾年，到了魏延死後，蜀漢不得不收縮防線。

蜀漢最終打不下去，實際上根本原因在於人口不過90餘萬戶，民力已竭，實在支撐不起了。為了增加人口，三國時，曹操帶頭娶寡婦，女人的婚嫁時間甚至被強行提至12、3歲。目的就是增加人口。

中原尚且如此，匈奴作為一個馬背上的游牧民族，她最大、最重要的戰略資源，那就是人口。

匈奴鼎盛時期人口不過百萬，在漢朝以前，長時期只保持幾十萬規模。面對當時漢朝千萬級人口規模，匈奴再厲害，如果過分重視血統純正，沒有人口增長和補充，其實根本不會壯大。

匈奴襲擾中原邊郡，通常會分為幾部傾巢出動，因為只有這種戰術，才能保障每次發兵損失最少，收穫最大。如果分兵被殲，元氣通常都會很長一段時間難以恢復。

匈奴在襲擾漢朝的時候，實際就是拿匈奴成年男性的生命去換取漢朝邊郡上的生存物資、牛羊、財富、以及大量人口。所以不管匈奴獲得的戰利品有多豐厚，但是代價就是匈奴成年男性人口的消耗。

這就使得匈奴不得不形成了父親死了，兒子繼承父親的妻子，哥哥死了，弟弟繼承哥哥的妻子這種風俗。因為他們需要匈奴的寡婦們生出更多的孩子，養育出更多的強壯的成年男性。不然想出去打劫，連士兵都沒有。

所以漢朝的張騫、李陵、蘇武、李廣利等，這些不管是出使被扣留的，還是打敗被俘的，只要在匈奴那邊，都會給他們每人配發至少一個匈奴老婆。娶了匈奴老婆就會擁有一定家產，然後就會要求生孩子，不然什麼都得不到。當他們要離開的時候，甚至老婆都可以帶走，但是孩子要留下。
