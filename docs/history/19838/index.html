明治時期日本飛速發展的深層思考與原因分析，值得我們學習和思考

2019-12-18 754條評論 1,725次閱讀 208人點贊

進入明治維新以後，明治政府大力普及教育，培養人才。明治政府的領導者們認為要想真正實現國家富強，民族振興，就必須要推動有利於本國發展的人才。此外，還要求現代軍隊中的軍官和士兵都具備較高的知識水平。為了實現這些目標，政府開始在全國興辦學校，普及教育，想要徹底改變日本。但是，明治政府面臨一個嚴重的問題。那就是資金不足，光靠政府捐助各地建小學是遠遠不夠的。所以，興辦教育的費用必須由國民承擔。而苦於生計的民眾並不想把孩子送進學校，但政府為提高就學率不遺餘力，地方的大地主們也意識到國民教育的必要性，紛紛捐出巨額資金來協助建立學校。在這裡不得不提到，正是德川時代積累的財富和素養，致使地方的大地主們對文化的責任感促使他們這麼做。另外，從德川時代起，日本就已經養成了良好的尊重教育的風氣。就這樣，國民教育飛速普及，到了明治末年，就學率超過了95%。在高等教育方面，政府尤其下了一番大功夫。政府不惜花費重金聘請外國教師，或者送日本學生出國留學等方式培養學者。正是由於明治時代對教育和人才的高度重視才使得日本高速發展起來現代化經濟。

推動明治維新改革的並不只是政府官員，那些後來的知識分子也發揮了重要作用。雖然他們沒有參加明治維新的政治活動，但他們當中有的擔任官職，有的創辦實業，有的從事教育事業，培養人才。無論他們當中的哪一類人，都主張大膽引進西方的技術和制度。這些人都是武士出身，他們從來沒有過參與經濟活動的經驗。而也正是因為這一點，他們具備常人身上所不具備的冒險精神和探索精神，這些這是新型經濟活動所必須的精神財富。當然僅靠有冒險的精神來實現現代化是遠遠不夠的，日本國民的大眾力量是不可或缺的，尤其需要那些為現代化建設吃苦耐勞的人們。日本國民在政府的倡導下，吸收了「文明開化」的思想。逐漸接受西方事物，辛勤勞動，積極儲蓄，積累現代化所需的資本。日本國民同中國一樣，深受儒家思想影響。一方面他們認為獲利是不道德的，這一觀點影響了實業的發展。但另一方面，崇尚勤儉節約，鄙視浪費的觀點，又有益於資本積累和實業發展。在打開國門之後，大量西方文明和新鮮事物湧入日本，甚至有一段時間，日本人普遍輕視原有的文化遺產，認為新事物什麼都好。

明治時期的日本急切的想改變一切。森有禮甚至提出廢除漢字，使用羅馬文字。還有人主張改善日本人的傳統性格。他們認為日本人缺乏毅力，是因為不吃肉的緣故。因此，他們提出要給兒童喂牛奶，再讓他們吃牛肉長大，日本人就一定可以變成有毅力的人。還有人提倡和外國女性通婚以改良人種。如此急切的想改變，難免會造成一些負面影響，但也無疑讓日本更容易吸納西方文明。日本自古以來就養成了主動吸收外國文明的行為，這也正是日本人傳統性格的一種體現。從古至今，日本人都沉迷於模仿外國文明，對其他民族、其他國家的文明十分包容。很久之前，日本人就醉心於中國文化，從不遺餘力地將中國文明引入日本。還引進漢服，收集中國物品，試圖將日本人的生活整體漢化。同樣，在西歐各國進行工業革命之後，日本人也積極地吸收西方文化。日本人之所以有這樣一種傳統的習慣，是有一定歷史原因的。在古代社會，日本是一個小島國。在他的附近沒有大國威脅，只有一個遙遠的文明的中國。古代中國文明領先世界，因此日本人能夠向中國學習，引進中國文化，對他們來說是一種莫大的恩惠。而且，古代沒有現在這樣發達的交通手段，中日兩國雖然相隔不遠，但中國卻無法統治日本。正是在這樣一種歷史背景下，養成了日本人對外國文明毫無戒心，主動吸納的性格。

直到19世紀中葉，日本國門被西方各國用武力打開。剛開始日本被迫接受西洋文明，到後來，日本人開始欣賞西洋文化，並且對西洋文明大家讚賞，開始主動吸納外國文明。明治時期的日本人既希望獨立，又認可外國文明的優越性。在這一時期，日本對西洋各國也是卑躬屈膝。明治24年，俄國皇太子訪日時，不幸在日本遇刺。日本舉國上下紛紛向皇太子送上慰問信和慰問品，還發了許多表示遺憾之意的電報給俄羅斯皇帝。日本政府擔心俄羅斯因此事而藉機挑釁日本，於是主張對加害俄羅斯皇太子的人處以死刑。但法院的法官，頂住了來自政府的壓力。維護了司法權獨立和法律公正的原則，他們認為判處死刑是不合法的，最終判處被告為無期徒刑。從這一點上表現出了明治時期日本司法部門積極進步的一面，也體現出了日本想保持自身獨立性不受侵犯的特點。

另外日本的領導人出於對外國資本不放心，決定通過聘請外籍人員引進外國技術來發展本國經濟是最安全的。在法律，軍事，經濟等各個領域日本都從國外高薪聘請了許多專業人才，這些人在日本踏實工作，不僅為日本提供了知識，還起到了參謀的作用。例如費諾羅薩，不僅在日本從事教育工作，還將日本的美術介紹到了歐洲。還有外務省的丹尼森，在日清戰爭和日俄戰爭的外交活動中都提出了非常重要的建議。這種大量聘用外籍人員熱潮的背後，隱藏著日本和歐洲都沒有排他的民族主義這一特點。明治時期日本的飛速發展，是綜合上述各因素的結果。
