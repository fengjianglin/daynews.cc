拿破崙年輕時照片

槍林彈雨的戰場，是廝殺聲、呼喊聲、槍炮聲，血雨腥飛的慘烈；

硝煙瀰漫的情場，是薄如紙、輕如風、空如影，妙手摘花的悲戚。

《拿破崙情史》這部影片，展示了一個政治男人在戰場和情場之間既血雨腥風又妙手摘花的桀驁不馴和威武霸氣。

1

情緣

拿破崙因政變被囚。

拿破崙的未婚妻黛絲蕾寢食難安。

這個晚上，外面打著雷，下著雨，黛絲蕾忽然聽到了口哨聲。她先是從床上一躍而起，跑到窗口，看到拿破崙在樓下雨中向她吹著口哨。黛絲蕾飛奔下樓，且奪門而出徑直撲向拿破崙的懷抱。兩人緊緊的擁抱在一起，然後就依偎在門口，說著動人的情話。他們如所有熱戀的情人一樣，如膠似漆，幸福甜蜜。即便外面很冷，可是黛絲蕾卻說，外面很美。愛情的甜蜜，讓兩人覺得無比幸福。

拿破崙動情的對黛絲蕾說：我直接從監獄來，就是要你嫁給我。你願意嫁給我嗎？

黛絲蕾一臉幸福的憧憬：願意，願意。

然後就有了98法郎的故事。

拿破崙要去巴黎找行政人員，找黛絲蕾借錢。

熱戀中的黛絲蕾二話不說，把僅有的98法郎借給了拿破崙，資助拿破崙去巴黎。

拿破崙離別時，對黛絲蕾說：回來後，我們就結婚，請你記得，我愛你！

這像極了美好愛情的完美：有甜蜜，有期待，有承諾，有誓言，有兩顆在一起的心。

約瑟芬

2.

情謀

黛絲蕾一直守著拿破崙離別時的誓言，等他回來後和她結婚。

直到她聽到了拿破崙在巴黎的消息，直到她親眼看到拿破崙和約瑟芬在眾人面前談笑風生，

約瑟芬說，我愛拿破崙，他也愛我。

拿破崙在眾人嘲笑拿破崙愛的是約瑟芬的影響力，拿破崙追求榮耀只會創造偉大的英雄，但是不會創造體貼的男人時，拿破崙卻很榮耀的的說，追求約瑟芬這樣特殊的女性，也許能創造兩者。

站在一旁被情灼傷的黛絲蕾，憤怒的把手中的香檳酒直接往約瑟芬身上潑去，然後狂奔逃離了現場。

黛絲蕾傷心欲絕，她信任拿破崙超過任何人。可是，她最信任的人卻背叛了她。她感覺到世界末日的到來。

但拿破崙是一個軍事家，政治家，相比愛情，他更愛權利。他無情的拋棄了這個曾經借他98法郎資助他去巴黎，讓他以後有發展平台的有情有義的女人。他拋棄了對黛絲蕾的所有誓言。

拿破崙拋棄黛絲蕾，因為黛絲蕾不能幫助自己實現在政治上的野心。他追求名媛約瑟芬，因為約瑟芬擁有大量財富，約瑟芬是社交名媛，可以幫他拓廣人脈，可以幫助拿破崙度過人生低谷期。

拿破崙選擇與約瑟芬結婚，有自己的政治需求。

拿破崙是個政治家 ，他的愛情帶著謀略。他的這種愛情觀，註定他對愛情不會專一和持久，因為他追求的女人是要嫁妝豐富，關係良好，有利於他將來政治發展的女人。

拿破崙加冕典禮

3.

情跡

許多年過去，我摯愛的太太約瑟芬未產一子，因政治需要，我要為國家找接班人，為法國著想，只好結束我倆的婚姻關係。

與約瑟芬婚後幾年，拿破崙在眾目睽睽下面無表情的在婚姻解除儀式上宣讀著。

但約瑟芬卻只宣讀了開頭“我想留下，但我的婚姻妨礙國家的福祉。因我不能，履行我的義務—-“隨即，約瑟芬，掩面大哭，情不能自控，後面的由人代替宣讀完。

約瑟芬被離婚，她傷心欲絕。她讓黛絲蕾去陪她。

約瑟芬邊哭邊哽咽，她說：

拿破崙，一直知道我不能生育，他一直知道。

約瑟芬曾經無比幸福，拿破崙在加冕典禮上親手為她戴上皇冠，親手拉起她的手，給了她殿下無比的榮耀，那一切，剛記憶猶新，現在卻心碎一地。

如當初拋棄未婚妻黛絲蕾一樣，約瑟芬也被無情拋棄，哪怕約瑟芬曾經幫拿破崙走上神壇。

拿破崙是一位政治家，他的婚姻都是與政治聯姻的，她需要一個更好的女人來幫他完成他的政治理想，而此時的約瑟芬已經對他無用了，所以，他拋棄了約瑟芬。

約瑟芬被廢，拿破崙馬上娶了18歲的奧地利公主，這個家族的政治色彩很濃，在政治上，拿破崙需要她。

拿破崙與奧地利公主的婚姻，是一場徹頭徹底的政治婚姻，各方都有自己的政治目的。奧地利希望通過政治婚姻改善和拿破崙之間的關係，拿破崙想通過與哈布斯堡家族聯姻，鞏固拿破崙家族在歐洲的地位。

拿破崙迎娶奧地利公主

4.

情滅

你記得我們首次見面的夜晚？我想知道，我若娶你，我的命運將如何？

黛絲蕾接受任務去勸降拿破崙時，拿破崙這樣說。

黛絲蕾回答說：我也在想會怎樣。

拿破崙對黛絲蕾的勸降，由剛開始的憤怒，拒絕，然後，慢慢的軟化。

到了最後，拿破崙把投降的箭交出給黛絲蕾。

黛絲蕾雙手接過投降之箭，說：再見，拿破崙。

拿破崙問：黛絲蕾，你何時開始不愛我？

黛絲蕾說：我不知道，在這條經歷之路的某一處吧。

當黛絲蕾拿著拿破崙的投降之箭離開，拿破崙轉身，一個人面對灰色的天空，從此

拿破崙被放逐到聖塔里那島，抑鬱而終。

他的政治生涯結束了，他因政治而聯姻的女人也都離開了他。

拿破崙曾經說過：一個人要堅持”為自己而活”的生命理念，而絕不是成為”為他人而活”的機器，只有這樣才能找出人生的目標與願望，然後努力去實現它，才不至於辜負上天所賜予的生命全力。

拿破崙一生是為自己活的，無論是他在政治上，軍事上，還是他為政治而選擇的婚姻。

“英雄的心中豪情萬丈，向著王座毅然起航。艱險滿途，無畏無憾。”

做為政治家，軍事家，拿破崙是英雄。

做為婚姻，拿破崙無疑是失敗的，因為政治需求，他未能娶自己愛的女人。當他失勢時，他所有因政治而聯姻的女人都拋棄了他，就當初他無情的拋棄她們。

《拿破崙情史》這部影片，無疑很好的闡釋了英雄的政治與政治婚姻的因果。
