司馬徽舉薦諸葛亮，他自己為什麼不出山呢？

2019-12-22 1,909條評論 2,821次閱讀 1059人點贊

劉備兵敗，投奔荊州刺史劉表，駐紮在新野城的時候，徐庶因為曹操以其母親要挾，離劉備而去之時，向劉備舉薦了諸葛亮，後來司馬徽也向劉備舉薦了諸葛亮，這才有了三顧茅廬的典範，司馬徽自己也很有才幹，為什麼他自己不出山輔助劉備呢？

放在三國亂世之中，那是各方勢力都渴求的人才啊，但是司馬徽卻沒有在任何一方勢力任職，而是開辦學堂，教書育人。當時劉備檀溪躍馬，到了司馬徽家中，請司馬徽出山輔助，司馬徽卻是推辭「山野閑散之人，不堪世用」，後來司馬徽再薦諸葛亮之時，劉備想留司馬徽，司馬徽也是飄然而去，就連劉備這樣仁義之名在外的人，他為什麼也不願輔助呢？

當年司馬徽舉薦諸葛亮的時候，說了一句孔明雖得其主，而不得其時也，是因為司馬徽早就已經分析清楚天下大勢，知道什麼可為，什麼不可為，但是的曹操經過官渡之戰打敗袁紹，統治者北方大片疆土，猶如龐然大物，而東吳歷經三世治理，也是不可小覷。諸葛亮錯過了最好的出山時機，縱使諸葛亮有天大的本事幫助劉備出謀劃策，佔領了大片的疆土，最終形成三國鼎立的局面，但是卻不能憑藉蜀國一國之力來平定天下，諸葛亮晚年的六次北伐，皆以失敗告終，司馬徽知道這是天命，天命不可違，所以他不如不出山，而是過著閑雲野鶴的生活更為自在。

良禽擇木而棲，良臣擇主而事。司馬徽在荊州建了水鏡山莊，教書育人，以司馬徽的才幹，請他出山相助的諸侯霸主，不在少數，荊州刺史劉表也曾請過他，卻被他拒絕了。司馬徽素有慧眼，善於識人，他看出劉表是個心胸狹隘，平庸無能的人，不是一個明主，所以沒有在年輕的時候出仕。後來遇到的劉備，不失為是個明主，但是因為自己已然是個耄耋老人，且也已經看淡名利，固然他有心力輔佐劉備，匡扶漢室，但他的身體也不允許他來完成橫掃群雄，一掃三國的事情了。

在亂世之中，有才幹，又能看淡名利，與世無爭，過著自己的閑雲野鶴的日子，也不失為是亂世之中的人生美事。
