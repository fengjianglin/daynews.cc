西施最後怎麼樣了？最終和范蠡在一起了嗎？

2019-11-28 0條評論 20次閱讀 0人點贊

摘要

吳越爭霸，不可不提西施、范蠡；勾踐從奴僕到霸主的曲折奮鬥史，不得不提西施、范蠡；范蠡為勾踐獻上錦囊妙計，而西施則是文種亡吳七計中最重要的一環。西施和范蠡最終的結局千年來牽動著我們的心，對於他們最後怎麼樣了？

西施和范蠡最終的結局千年來牽動著我們的心，對於他們最後怎麼樣了？

鄰家有女初長成—施夷光

西施本是本名叫施夷光，出身貧寒，她的父親平時賣柴，母親為人浣紗。傳說西施本是天上嫦娥的掌上明珠，因為非常喜愛，命金雞看守，金雞趁著嫦娥不備，把玩時不小心將明珠遺落凡間，金雞隨即下凡追查。

正在這一天蒲陽江邊山下的一戶農家婦剛好在西邊浣紗，看到天上掉下金光閃閃的東西，想用手去抓住，而明珠卻像有思想一般鑽進婦人口中不見了，婦人從此便懷孕了。

十六個月過去了，施妻產下一個光華美麗的姑娘，她便是施夷光，於是便有了：「嘗母浴帛於溪，明珠射體而孕」。

年輕的西施長得非常美麗，她在溪邊浣紗，魚兒被美得忘記了遊動，她的一顰一笑都是旁人喜愛模仿的對象。

范少伯

范蠡是宛地人字少伯，和宛地令文種是好朋友，因為不滿當時楚國政治黑暗，非貴族不得入仕的規定，一起投奔越國。在吳國闔閭攻打楚國之際，勾踐派人偷襲了吳國，雖然范蠡勸諫勾踐不要攻打但是勾踐還是做了。於是吳越之間結下怨仇，後來闔閭回國帶人攻打越國，被箭射傷而亡。闔閭之子夫差登基後，勵精圖治，任用伍子胥、伯嚭等人每日訓練軍士，時機得當後，攻擊越國，將勾踐圍困與會稽山。

大仇得報的夫差，因為文種派人賄賂了伯嚭，赦免了勾踐，但是要求勾踐和范蠡一起到吳國做奴隸，為了能夠報會稽山之仇，勾踐卧薪嘗膽，在吳國兩年後回到越國，勾踐用文種亡吳七計。

「一曰捐貨幣以悅其君臣；二曰貴糴粟囊，以虛其積聚；三曰遺美女，以惑其心志；四曰遺之巧工良材，使作宮室以罄其財；五曰遺之諛臣以亂其謀；六曰疆其諫臣使自殺以弱其輔；七曰積財練兵，以承其弊。」

於是范蠡帶著軍士滿世界找美女，在苧蘿山得西施、鄭旦二人，范蠡見西施天生麗質、秀媚出眾，一下子愛上了西施，而西施見他儀錶堂堂，氣度不凡也愛上了這位范郎。

後來文種得知在苧蘿山得美女，便親自去接，文種的到來結束了這本只屬於范蠡和西施纏綿悱惻的愛情，他將西施帶回會稽，教她舞蹈、體態，三年間范蠡和西施只能發乎情，止乎禮，相對無言，唯有淚千行。

剩下的大家都知道了，勾踐將西施進獻給夫差，夫差馬上被西施的美貌吸引，為她在姑蘇建造春宵宮，築大池，池中設青龍舟，日與西施為水戲，又為西施建造了表演歌舞和歡宴的館娃閣、靈館，從此不理朝政，只願與西施朝夕相處，這樣吳國便開啟了滅國模式。

最後越國趁吳國兵力被齊國滅之際，攻擊吳國，夫差自刎而亡，而西施命運從此的便為人所嘆。

那麼最後西施去哪了呢？

1.被范蠡帶走

蘇東坡曾經寫道：「五湖問道，扁舟歸去，仍攜西子。」在山東肥城陶山，據說有范蠡和西施墓。

吳國滅亡的當天，范蠡做了兩件事，一是寫信勸他的好朋友、一同共患難的文種，趁早離開勾踐。二是在姑蘇台下花蔭深處找到了萎頓不堪的舊日情人西施，倉皇逃到太湖，雙雙駕一葉扁舟，消失在煙波浩渺之中。

2.自縊而亡

吳國滅亡後，西施的使命已經完成，高興之餘，便覺得自己對不起疼愛她的夫差，於是便自縊而亡。

3. 被勾踐沉江

在西施被文種帶回越國都城時，勾踐便對這位美人兒動了春心，奈何報仇勢大，只能將色心埋藏，後來得以報仇，勾踐覺得西施已經不是以前的她了，便將她沉江而亡。

4. 被越後所殺

勾踐報仇後，趁機迎回了最大的功臣西施，想納她為妃，而越後怕西施會影響到自己的地位，便對大臣們說：「西施乃是禍水，不可留。」勾踐沒辦法只能讓越後處理。

5. 從史記記載來看，沒有任何關於西施的記載，范蠡在越國滅吳之後便辭別了勾踐，泛舟而去，他到了齊國，改名換姓，在海邊耕種，他有一位妻子為他生了三個兒子，但沒有記載是否是西施。
