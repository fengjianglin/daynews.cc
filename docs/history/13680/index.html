雍正登基後，老八下場很慘，為何老十四能躲過一劫？

2019-12-11 2,015條評論 3,639次閱讀 1689人點贊

「生亦何歡，死亦何苦」。雍正繼位後，作為「九子奪嫡」中的失敗者老八與老十四，成王敗寇，他們註定沒有完美的結局。老十四在雍正元年被罰去守皇陵了，老八呢，雍正四年，被雍正改名「阿其那」，削籍奪爵，當年便撒手歸天了。其實在我看來，老八與老十四的結局，只有先後之分，而沒有痛苦之別。老十四是康熙朝的大將軍王，是一隻雄鷹，將一隻雄鷹一直困在籠子里達14年之久，可想而知，胤禵或許比八哥胤禩更痛苦。當然，這些只是個人拙見。仁者見仁，智者見智，一百個人關於歷史或許就有一百種思考。有人會問，為何老八慘死禁所，老十四卻沒那麼慘呢，老十四活到了乾隆朝，壽終正寢。原因大概有這麼幾點。

　　

　　其一、情節輕重不同。

　　對於雍正皇帝而言，老八和老十四都是自己多年的政敵。老八，老九是八爺黨的「領袖人物」，老十，老十四最多只能算「從犯」。對於首腦，八阿哥胤禩，雍正絕不輕饒，不能讓他們繼續結黨，繼續擾亂朝政。所以，老八老九慘之又慘，改名，抄家，削籍，奪爵，圈禁，然後，他們在獄中受盡凌辱而死。老十四呢，他只是從犯，開始也只是守皇陵，後來才圈禁起來，這就是他們的區別。

　　

　　其二，親疏不同。

　　老十四畢竟和雍正是一母同胞，他們都是德妃烏雅氏所生。這就是老十四的先天優勢，老八再怎麼比，也比不上人家一母同胞之情。雖然老十四不喜與四哥雍正往來，但親兄弟之間，血濃於水，老十四註定不會比老八慘。

　　

　　其三，老十四身強體健，而老八經不起折騰。

　　老十四文韜武略，有大將之才，是一名合格的將軍。軍旅生活鍛煉了他的意志，強健了他的體魄，儘管被圈禁，他還是能堅定的熬死了四哥雍正，等到了侄兒弘曆上台，也等到了自由。而老八胤禩，人稱八賢王，他一生喜歡結交朋友，頗有賢名，八爺黨最喜歡的恐怕是和京城朋友胡吃海喝，交流感情吧。老八是交際狂人，身體必定不及老十四萬一，四十六歲的老八，怎麼也抗不住圈禁與折磨，老八慘死禁所，肯定也有身體的原因。

　　

　　結論：老八與老十四的人生軌跡，告訴我們，權利可以將你推上天堂，也可以將你打入地獄。在失敗面前，能扛住煎熬的人，才有希望看到勝利和光明。要想活的好，鍛煉得趁早啊。
