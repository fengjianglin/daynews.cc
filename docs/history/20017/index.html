13歲被退婚，她餘生不言再嫁，獨自叱吒商界，享盡奢華後皈依佛門

2019-12-16 903條評論 2,220次閱讀 262人點贊

要堅強，要勇敢，不要讓絕望和庸俗的憂愁壓倒你，要保持偉大的靈魂在經受苦難時的豁達與平靜 ——拿破崙

動蕩中，她以筆為劍，面對困境，奔走疾呼：亂世中，她以身為盾，面對不公，聲嘶力竭；工作中，她兢兢業業，甚得喜歡；商場上，她叱吒風雲，鐵血手腕；思想碰撞中，她鍾愛文言文，華麗辭藻直擊心靈。她是傳說中的千面女生，一個人，多種身份，多重面孔，多樣性格，她是呂碧城，一個一生光輝、一生璀璨的傳奇之人。

年少的歡喜

1883年，呂碧城出生於山西太原，她的父親是光緒年間進士，深受儒家思想的影響，自然，在呂碧城的生活中也少不了傳統文化與思想的熏陶。從小生活在書香門第的她耳濡目染，也對傳統文化頗有研究。

5歲時，呂碧城就顯現出了異於常人的文學天賦。父親在遊園時隨口吟詩：春風吹楊柳，呂碧城便脫口而出秋雨打梧桐，父親大驚，感嘆於呂碧城過人的文學素養。在成長的過程中，呂碧城學習了書法、繪畫並對詩詞情有獨鍾，且尤其擅長，形成了很高的文學素養。當時的人都驚嘆於她的文學功底。

童年時期的呂碧城無疑是幸福的，在這段時光里，她無憂無慮，有著良好的家庭環境，接受著良好的教育，學習著自己熱愛的東西，年少時候的她，擁有著眾人羨慕的童年與天賦。

成長的磨難

呂碧城如何也想不到，一場突如其來的變故，讓她的人生軌跡偏離了原來的方向。呂碧城的父親在一次爬山的過程中因為下雨過之後路面濕滑，失足滑倒，離開了人世。此後，呂碧城的生活發生了翻天覆地的變化。

按照規定，女兒身的呂碧城是無法繼承父親的遺產的。沒有了父親，呂碧城與母親在家中的處境很是艱難。為了擺脫困境，13歲的呂碧城給父親昔日的好友寫信請求幫助，在當時的社會，作為一個女子四處請求幫助，呂碧城也鼓起了足夠大的勇氣。功夫不負有心人，呂碧城終於找到了兩江總督樊增祥，樊增祥感嘆於呂碧城的勇氣與魄力，決定幫助她渡過難關。在樊增祥的幫助下，世人沒有敢怠慢呂碧城的，呂碧城也暫時擺脫了困境。

年少的時候，呂碧城曾與汪家定下娃娃親，呂碧城四處求救的事迹傳遍了大街小巷，當然，也傳進了汪家的耳朵里。汪家人覺得呂碧城的父親去世呂家失勢，並認為呂碧城小小年紀便能扭轉局勢，翻雲覆雨，如果以後娶進家門更是難以管理，於是解除了與呂碧城的婚姻。

那個年代對女子的名節要求極高，被退婚對一個女子來說更為致命，無數人對呂碧城指指點點，呂碧城承受了極大的精神壓力。十幾歲的呂碧城擁有同齡孩子遠不能及的勇氣與魄力，在如此孤立無援的境況下，呂碧城敢於在逆境之中打開一個新的局面，作為一個小女孩，呂碧城知道如何自救並且敢於將自己的想法付諸於實踐，在被別人拒絕之後依舊毫不灰心，接著尋找新的出路。這樣的膽識，這樣的謀略，不禁讓人敬佩。

逆潮流為先

1904年，呂碧城與好友一起聽了關於女子求學的講座，一個遠赴天津的小小種子在呂碧城的心中萌芽。無奈，呂碧城的舅舅受傳統封建觀念的影響，只希望呂碧城恪守婦道，不想讓她拋頭露面，寄人籬下，呂碧城只能暫時聽從。

然而，在呂碧城心中，那團紅色的火焰其實從沒有熄滅。於是，在一個夜晚，在好心人的資助下，呂碧城買了一張去天津的火車票，獨自踏上旅程。遠走家鄉的呂碧城什麼都沒有，只有一顆嚮往教育的心靈。

接受新式教育的呂碧城確實獲得了很多。21歲時，呂碧城靠一首《滿江紅》贏得了眾多喝彩，艷驚四座。《大公報》的負責人被她的才氣深深吸引，留下她做編輯。華麗的文章，高傲的氣節，呂碧城的文章廣受歡迎，在當時，《大公報》的銷量遠遠超過其他報紙。

為了讓更多的女子接受到新式教育，呂碧城萌生了興辦女子學校的想法。於是，她建立了天津公立女學堂，她也成為中國歷史上第一位女校長。女子學堂的建立讓更多的女學子接受到了更多的新式教育，極大的促進了婦女解放和社會的思想解放。

呂碧城是新時代的新女青年。她不顧世俗的眼光，也不受舊的封建思想的影響，逆著當時思想界的主流而為，獨自去異鄉求學、興辦女子學堂，呂碧城的魄力甚至比其他男子還要強悍，永遠不生活在體制的禁錮之中，這是呂碧城的人生信條。

叱吒商場，翻手為雲

袁世凱曾經十分欣賞呂碧城，在機緣巧合之下，他幫助呂碧城擺脫了清廷的抓捕，呂碧城為了報答袁世凱，決定為袁世凱效力。後來，袁世凱極力想稱帝，呂碧城曾經勸說過袁世凱，只是袁世凱心意已決，不聽呂碧城的勸告。呂碧城在此時保持著清醒的頭腦，辭去職務，轉戰商場。

憑藉著美麗的容貌、過人的膽識、超凡的能力以及早年積累下來的豐富人脈，呂碧城在商場翻手為雲、覆手為雨，僅僅三年，呂碧城就在商場嶄露頭角，擁有眾多的財富。

呂碧城十分喜歡華麗富貴的生活，她的家裡富麗堂皇，她出門時乘坐的豪車以及租住的豪華酒店數不勝數，赴不同的宴會時，她的衣服從來沒有重樣，每一件都是花大價錢精心定製的。呂碧城從來沒有掩飾自己對奢華生活的喜愛，相反，她樂在其中。

身歸混沌，佛門圓寂

呂碧城從童年時期便遭遇了各種各樣的變故：幼年喪父、無故退婚、姐姐妹妹相繼去世，呂碧城經歷了很多的打擊，漸漸地，呂碧城加深了對玄學的思考。

1930年，呂碧城選擇在日內瓦皈依佛門，一心向佛。她翻譯了眾多的國外的佛學著作，成為中國翻譯佛經第一人。戰爭爆發，呂碧城輾轉世界各地，宣揚佛法，希望減少人們內心的傷痛，也希望人們放下殺戮，珍惜和平。1943年的一天，她將佛堂擦拭的一塵不染，在頌經中圓寂，身歸混沌，她的靈魂走向了自己嚮往的佛法。

生如夏花之絢爛

呂碧城的一生，經歷了太多，也成就了太多。13歲時被退婚，她一生沒有出嫁；在那個封建思想殘餘的年代，他宣揚女權與女學，敢於立潮頭為先，她開創了太多的新局面：第一次興辦女子學校，翻譯外國佛經，憑藉自己的手段與頭腦在商界叱吒風雲，她的一生過於絢爛。死如秋葉之靜美，最終，呂碧城於平靜中離世，離開了人世間的苦難，去尋找她心中的那一方凈土。
