她是梅蘭芳最愛的女人，卻嫁給青幫老大杜月笙，晚年寫下怎樣感悟

孟小冬一生中最重要的兩個男人，一個讓前半生痛苦凄涼一個讓她後半生安穩無憂。

無論是英雄還是美女大都生於亂世的比較多， 孟小冬是著名的京劇女老生演員，在中國京劇歷史上，孟小冬幾可以說是人盡皆知的，有非常高的地位，在女性反串老生是沒有人可以和她比肩的，在梨園界「冬皇」，一個女子，以「皇」加冕，可見其地位有多高。當時孟小冬在戲曲界的地位和京劇泰斗梅蘭芳可以說都是不可缺少的存在，他們的組合可以說是中國京劇史上最完美的一對反串角色演員，孟小冬反串老生，梅蘭芳反串旦角，兩個人的「反串」的角色都是經典，至今無人可以超越。這樣的組合也讓兩人產生了一段感情糾葛，梅蘭芳也成了孟小冬第一任丈夫。

孟小冬

孟小冬出生在一個梨園世家，父親叔伯都是著名京劇演員。孟小冬從小就是在戲班裡長大，耳濡目染下也展現出了孟小冬的天賦，也許上天早就註定了孟小冬就是吃戲曲這碗飯的，這可能就是俗話說的，祖師爺賞飯吃，9歲的時候孟小冬在後台學別唱老生，被她的姑父仇月祥聽見了十分驚訝，他沒想到孟小冬一個小姑娘唱老生竟然唱的這麼好，不久後孟小冬就拜她姑父仇月祥為師學習老生。

對戲曲天賦極高的孟小冬她14歲登台演出就受到了很多人的好評，十八歲首演《四郎探母》後一炮而紅，名震京城，為了以後更好的發展她們一家搬到北京。十八歲的孟小冬在北京登台演出次數並不很多，因為扮相好，唱的好，沒多久就在北京聲名鵲起。在一次義演上，孟小冬與梅蘭芳因一出《游龍戲鳳》結識並相愛。當時梅蘭芳已經三十多歲了孟小冬還不到二十，而且梅蘭芳已經有兩位妻子（王明華和福芝芳）。

孟小冬與梅蘭芳

兩人感情日漸升溫，已經到了難分難捨的程度，梅蘭芳想要娶孟小冬過門，當時梅蘭芳的妻子王明華不在國內，家裡的事都由小妾福芝芳做主，福芝芳強烈的反對孟小冬過門，而孟小冬是個非常注重名分的人，她是死活都不會做小妾的，梅蘭芳整天為這事愁眉不展不知道怎麼辦好，這時候朋友給他想了個辦法，讓他對孟小冬說他是家裡的長子，長子是可以娶兩位正妻的，這樣一來孟小冬也就不好再說什麼了，還有就是福芝芳不讓孟小冬進門的事，不讓進就不進唄，在別處買個房子不可以了，不進們還能省去很多矛盾，梅蘭芳一聽覺得這個辦法很好，兩全其美，所有煩惱的問題一下都解決了，梅蘭芳迫不及待的去找了孟小冬，告訴孟小冬他可以娶兩位正妻，又說在外面住，是為了孟小冬好，怕她受委屈，又可以過二人世界多好，在愛情里女人都是傻子，就這話外人一聽就知道是騙人的，可愛梅蘭芳愛到無法自拔的孟小冬就是相信了梅蘭芳所說的話，就這樣答應嫁給了梅蘭芳，雖然兩個人舉行了婚禮。

孟小冬1938年拜余叔岩為師時所攝照片（後排左為李少春）

可是婚禮辦的非常簡單，也沒有請多少客人，只有熟悉的朋友，這樣孟小冬心裡起了疑惑，畢竟是新婚燕爾，兩個人過的又很幸福，孟小冬的疑惑也就被幸福淹沒了，可惜的是兩個人的幸福很短暫，婚後的孟小冬不上台了，整天在家裡，與朋友聚聚會，聊聊天，過著普通女人婚後的生活，也許這樣的生活，是很多女人嚮往的，可對於孟小冬來說，這樣的生活實在太單調了，她是為舞台而生的，只有舞台才能讓她那樣的耀眼，而現在她只是一隻被關在籠子里的小鳥。梅蘭芳的演出也越來越多，和她在一起的時間也越來越少，她整天閑著沒事可做，就更渴望回去那個可以讓她展現才華的舞台。她和梅蘭芳說自己想重新回到舞台，可梅蘭芳不同意，勸她說女人就應該安心在家，相夫教子，這才是一個女人應有的本分，還說她要是出去唱戲，別人會怎麼說，會說他梅蘭芳連自己太太都養不起，這樣他的臉往哪裡放，孟小冬也不想讓梅蘭芳丟臉，只要在家安心的做梅太太。

事情往往不按你希望的那樣發展，即使孟小冬為了梅蘭芳放棄舞台，可該發生的還是會發生，有一個孟小冬的戲迷，因為知道了，孟小冬是因為梅蘭芳才不在上台的，心裡十分氣憤，他帶槍潛入梅蘭芳和孟小冬的家裡，想刺殺梅蘭芳，正巧的是他們家裡有朋友來訪，再過亂中被孟小冬的戲迷給槍殺，這件事發生後，梅蘭芳的朋友和家人都反對梅蘭芳和孟小冬在一起，梅蘭芳對孟小冬的感情也起了很大的變化，梅蘭芳感到和孟小冬在一起自己生命遭到了威脅，對孟小冬的感情越來越淡，男人都是自私的，所謂的愛情都是建立在不傷害到自己的基礎上，要是愛情和自己相比大多數男人都會毫不猶豫的放棄愛情。

梅蘭芳與原配妻子福芝芳及子女合影

後來梅蘭芳的伯母又去世，梅蘭芳從小過繼給伯父，伯母和母親沒有差別，孟小冬得知後，前去為婆婆戴孝，可卻被梅府的下人攔住不讓進，說夫人吩咐不讓孟大小姐進府，這一句孟大小姐叫的孟小冬一陣恍惚，這算什麼，幸好這時梅蘭芳及時趕到了，可梅蘭芳說的卻是讓她先回去，孟小冬獃獃的看著梅蘭芳幾秒鐘，她現在算是都明白了，什麼正妻，什麼不進府是怕自己受委屈，都是騙人的，她轉身離去，梅蘭芳看見離去的孟小冬心慌了，因為他知道孟小冬有多倔強，有多驕傲，她對梅蘭芳已經心灰意冷，她覺得梅蘭芳根本就沒把她當成妻子，在梅蘭芳心裡，她只不過是一個小妾而已，孟小冬是一個多麼驕傲的人，她怎麼允許別人對自己這樣的羞辱，她決定和梅蘭芳分手，她在報紙上聲明與梅蘭芳再無瓜葛，孟小冬把自己最好的青春給了梅蘭芳，為了梅蘭芳放棄了自己最愛的舞台，可最後換來的只有一身傷痛。

杜月笙得知了孟小冬與梅蘭芳分手的消息，馬上就趕過來，他勸孟小冬不要衝動要想清楚，不要因為衝動做出讓自己後悔事，當她得知孟小冬受到的委屈和梅蘭芳分手的決心後，他不在勸孟小冬了，他找了梅蘭芳，不知道和梅蘭芳說了什麼，最後梅蘭芳給了孟小冬一筆巨款的分手費兩人算是和平分手。孟小冬一個人四處漂泊，因為和梅蘭芳分手讓她受到了很大的打擊，她也沒有在登台演出，後來杜月笙覺得孟小冬這樣孤單一個人沒人照顧也不是辦法，就讓姚玉蘭邀請她來他家裡住，還有姚玉蘭可以照顧她，因為杜月笙不想孟小冬受一點苦，他是心疼孟小冬的。

杜月笙對孟小冬的感情也不知道是從什麼時候開始的，因為們認識的時候孟小冬才十二歲，那時候的孟小冬就是個天真爛漫的小女孩，而杜月笙已經是三十多歲的大叔了，一個三十多歲的大叔，可以說是在上海說一不二的男人，會對十二歲的孟小冬產生感情嗎？我想是不會的，最多就是欣賞，如果說孟小冬是杜月笙最珍惜的女也不過分，杜月笙對姚玉蘭是男人對女人最原始的衝動是喜歡，看到就想要佔有，而且不擇手段。而杜月笙對孟小冬就不一樣了，是「愛」愛和喜歡是不一樣的，我們都知道，如果你喜歡一朵花你會把它摘下來，可如果要是愛你不會捨得把它摘下來，只會細心呵護生怕它收到一點傷害，對人亦是如此。

孟小冬與杜月笙

杜月笙珍惜她欣賞她，不忍心傷害她一點點，看著她從一個小姑娘成長成出色的京劇大師，他是多麼的高興，他珍惜她成長的點點滴滴，杜月笙喜歡看孟小冬的戲，每次看戲都要送花籃，每次都要去後來和孟小冬說話，和孟小冬說話每次都叫能大小姐十分有禮。

杜月笙一直是孟小冬的忠實戲迷，也一直都在捧孟小冬，雖說孟小冬確實是因為唱的好才有了「冬皇」之稱，但是如果沒有杜月笙捧她，她也不會這麼快的紅遍上海吧？

不知道從什麼時候開始杜月笙對孟小冬的感情產生了變化，長大後的孟小冬選擇嫁給梅蘭芳，杜月笙也沒說什麼，我想他是希望孟小冬可以幸福，因為他的年紀足夠做孟小冬的父親了，只要孟小冬幸福就好。可當他得知孟小冬和梅蘭芳分手的時候，他就趕去給孟小冬做依靠，怕孟小冬一個人無依無靠受委屈，後來又讓姚玉蘭把孟小冬接到身邊照顧，杜月笙從來沒強迫孟小冬做任何她不願意做的事，這才是愛吧，

後來孟小冬和杜月笙在一起了，杜月笙極盡所能把最好的東西都給了孟小冬，哪怕是孟小冬想要天上的月亮，杜月笙也會想辦法給她摘下來。

對一個女人來說，什麼是最重要的，有一個真心愛你寵你的男人才是最重要的，更何況是一個有那麼多傳奇經歷的男人，是每個女人都想要的吧。後來抗戰爆發了，杜月笙拒絕與日本人合作，帶著全家去了香港生活，1949年，61歲的杜月笙身患重病，孟小冬直至在身邊照顧，盡心儘力的伺候杜月笙沒有一句怨言。後來杜月笙計劃搬往法國。孟小冬自言自語的說了一句：「我也去嗎？那算什麼人呢？是丫環呢？還是女朋友呢？

杜月笙看著孟小冬，突然間想到了，這麼多年了，竟然沒有給自己心愛的女人一個名分。一想到這裡，他的心像刀割一樣疼，他怎麼可以這麼糊塗，他只想把最好的都給孟小冬，卻忘了給她最重要的名分，讓她這麼多年就這樣名不正言不順的跟著自己這麼多年，杜月笙覺得自己太對不起孟小冬了，62歲的杜月笙不顧所有人的反對，和孟小冬舉行了婚禮，一代名伶孟小冬終於有了屬於她的名分。

婚後一年，杜月笙因病去世。家中僅剩下11萬美金，孟小冬分得了兩萬美元，和那些杜月笙之前為她買的首飾。

在杜月笙去世後的十幾年裡，孟小冬一心研究戲劇，教授弟子，在也沒有結婚。愛情已經永久的留在了她的記憶里，她也在沒唱過戲，沒有了杜月笙她唱給誰聽呢？

1977年5月27日，70歲的孟小冬去世，她的一生兩場婚姻，一個讓她變心碎，一個寵的至極讓。

愛情是個不可控的東西，如果你真的遇到了，那你就無法控制自己，不管得到的還是失去都會讓你成長。
