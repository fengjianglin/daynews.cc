孔令侃為什麼娶近四十歲的二婚女為妻？

2019-12-18 83條評論 1,107次閱讀 118人點贊

孔令侃1916年生，他是宋靄齡和孔祥熙的長子，1933年畢業於上海聖約翰大學。宋氏三姐妹中只有宋靄齡生有兩男兩女，孔令侃又是第一位男丁，因此，特別受到蔣介石、宋美齡夫婦、孔祥熙、宋靄齡夫婦的重視。所以孔令侃大學一畢業，孔祥熙就有意識培養孔令侃，給他在政界、商界謀職。這使得孔令侃短時間內積累一定經驗。孔令侃繼承了孔祥熙的商業頭腦，表現在經營方面擁有一定的天賦，深得宋靄齡、孔祥熙的賞識。

然而，孔令侃的婚事卻讓父母大感頭疼。起初，他看好了舅舅宋子文的妻子張樂怡的妹妹張貽如，但遭到宋靄齡和孔祥熙夫婦的強烈反對，理由是舅舅和外甥成連襟成何體統，亂了輩分。這事最後不了了之。接下來發生的事更是讓宋靄齡、孔祥熙夫婦大跌眼鏡，如骨鯁在喉，有一種說不出的的難堪及尷尬。
按理說，憑孔大公子的個人條件和孔家顯赫的家庭背景，找個豪門千金、名門閨秀，應該不是問題。可是孔令侃偏偏喜歡上了比他大十幾歲的、清末官僚資本家盛宣懷兒子盛升頤的妻子，這個女人膚白貌美，人稱『白蘭花』。她性格外向，有著豐富的生活閱歷和交際能力，她能洞察孔令侃心理，善於察言觀色，她和孔令侃有共同語言，經常為他出謀劃策，彼此間相互欣賞，孔令侃對她漸漸產生依賴，並深深愛上了她。

宋靄齡聽說此事後根本就沒當真，根本就沒在乎，心想覺得大公子只不過閑得無聊，瞎扯而已。
1937年因為日本軍佔領上海，孔令侃所在的中央信託局遷往香港，孔令侃任常務理事，主持業務。1939年港英當局查獲其秘密電台，將孔令侃逐出香港。宋靄齡便聯繫宋美齡讓孔令侃去美國哈佛大學學習。令其沒想到的是，孔令侃給白蘭花發電報，讓她迅速趕往香港。1939年秋天，一艘美國公司的客輪載著孔令侃和白蘭花在香港起錨了。船靠岸馬尼拉兩天，孔令侃便電告母親宋靄齡，他要與白蘭花在馬尼拉結婚。宋靄齡、孔祥熙聞訊震怒。宋靄齡立即複電孔令侃，令其取消婚禮，並告訴他一定慎重考慮，好自為之。並明確表示不同意婚事。無奈孔令侃我行我素。宋靄齡感到鞭長莫及，亦無可奈何。她常常暗自嘆息，感到心力交瘁。

婚後的孔令侃對白蘭花真心對待，情深意濃，白頭偕老。但令人遺憾的是，兩人沒有孩子。
1992年，76歲的孔令侃在美國紐約去世。
