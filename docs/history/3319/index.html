匈奴人是現在的什麼人，以前的匈奴是現在的什麼地方？

2019-12-06 0條評論 21次閱讀 0人點贊

摘要

匈奴是中國古代北方的游牧民族，興起於現在的內蒙古陰山一帶，在公元前215年的時候，匈奴被蒙恬逐出河套及河西走廊地區，他在西漢前期開始強大起來，並控制了西域，曾多次進犯中原地區，後被漢武帝戰敗。

匈奴是中國古代北方的游牧民族，興起於現在的內蒙古陰山一帶，在公元前215年的時候，匈奴被蒙恬逐出河套及河西走廊地區，他在西漢前期開始強大起來，並控制了西域，曾多次進犯中原地區，後被漢武帝戰敗。

在東漢初年的時候，匈奴就開始進入到塞內，直到46年左右的時候，匈奴國內由於發生了嚴重的自然災害，因而分裂為北匈奴和南匈奴。而南匈奴附漢稱臣，之後在南匈奴的支持下，東漢開始征戰北匈奴，在91年時，北匈奴戰敗被迫西逃。

匈奴人是現在的什麼人

眾所周知，匈奴人驍勇善戰，曾對大漢王朝造成過很大的打擊，雖最終是不敵大漢王朝被消滅，而匈奴這個詞也在歷史的長河中被慢慢淡忘，但當時的匈奴人也同其他名族進行過融合，那現在的匈奴人是什麼人呢？大家都知道，匈奴人擅長騎馬射箭，民風十分彪悍，跟現在的蒙古族十分相似，所以說現在的歐洲民族應該擁有匈奴的血統，還有一部分匈奴人融入了匈牙利，成為了現在的匈牙利人。但很多有關於匈奴的問題都還是未知的，所以具體說匈奴人是現在的什麼人也並不能夠完全肯定。

以前的匈奴是現在的什麼地方

那以前的匈奴是現在的哪裡呢？古代匈奴的疆域是以蒙古高原為中心的，東到內蒙古東部，南沿長城與秦漢相鄰，西到費爾干納盆地，北到貝加爾湖，曾被稱之為 「百蠻大國」。在匈奴分裂以後，匈奴人逐漸遷移，於是古代的匈奴便漸漸的不存在了。

在匈奴分裂為北匈奴和南匈奴以後，歸降漢朝的南匈奴人被安置於今內蒙古、山西和陝西北部以及甘肅東部，但隨著時間的推移，他們進一步遷徙到了山西中部一帶並與漢人雜居，而部分北匈奴人則西遷到了東歐，並在多瑙河中游建立了匈奴王國，隨後又進軍羅馬，也就是現在的匈牙利一帶。
