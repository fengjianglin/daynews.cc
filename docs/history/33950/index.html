齊桓公病榻論相，管仲否定三人，齊桓公為何不聽

2019-12-23 1,068條評論 1,972次閱讀 990人點贊

這是史說星語的第265篇原創文章

話說，春秋初期，齊桓公九合諸侯，一匡天下。然而齊桓公能夠稱霸中原，離不開他的首席謀臣管仲，自從管仲接任齊相之後，進行了一系列經濟、政治和軍事方面的改革，才讓齊國躍居於其他諸侯國之上。由此可以看出，齊桓公在用人方面可圈可點，但是，當管仲病重，卧床不起的時候，齊桓公與其病榻論相，管仲一一否定了齊桓公舉薦的三個人選，而齊桓公竟然還將這三人加以重用，最終造成齊國混亂的局面。這前後矛盾的齊桓公用人之道，便值得筆者與你探討一番。

一、先來看這三人分別造成了齊國怎樣的動亂

齊桓公晚年病重，卧床不起的時候，宮中竟然被這三人重重封鎖，任何人不得入內，就連送飯的人都進不去。而齊桓公最終餓死在病榻之上，並且停屍67天無人受理，這兩個多月的時間裡，屍蟲都爬滿了窗戶，可見齊桓公死後慘狀。

再者，齊國的動亂也是因這三人而起，本來齊桓公已經定好了繼承人為太子昭，但是這三人憑藉手握大權，其中二人支持公子無虧，另一人支持公子潘，逼得太子昭遠逃齊國，向宋襄公求助，同時也引發了齊國六子爭位的亂象。這還不算完，其中一人10年之後，又殺死了齊孝公的太子，另立新君，再次造成齊國動亂。

二、這三人到底是誰，他們憑什麼能犯上作亂

這三人既不是貴族，也不是將軍，在齊國只能算是小人物。一個叫易牙，負責齊桓公的飲食與宴會餐食，通俗來說，就是齊桓公的宮廷御廚。第二個叫豎刁，他地位更低，是負責齊桓公日常起居的宦官，通俗來說就是個太監。第三個叫衛開方，他沒有具體的職務，而且是衛國的貴族，在齊國根本沒有什麼根基勢力。

但就這不起眼的三個人居然造成了齊國長達40多年的動亂，這就要看他們三個職務的重要之處了。

易牙是掌管齊桓公的宮廷御廚，他在食物中動動手腳那是相當方便的，而且齊桓公最終被餓死，易牙便是直接兇手。但易牙要想做到停供食物，還需要另一個人的配合，那就是掌管宮廷內務的太監豎刁，他們二人聯合起來，阻止別人接近齊桓公，那是易如反掌。一方面齊桓公得不到外面的消息，另一方面外界也無從得知齊桓公的近況。就這樣，齊桓公就被他們二人操縱了，他們可以「假傳聖旨」，以達到他們趕走太子昭，扶持公子無虧的目的。

衛開方則支持了另一個齊桓公的兒子公子潘，沒有他的鼓動，齊國也不會一下子那麼亂。況且太子昭回國奪位成功稱齊孝公之後，易牙遠逃、豎刁被殺，衛開方則躲過一劫，並且在齊孝公死後，還殺了他的兒子，另立新君公子潘，也就是成功即位後的齊昭公。

三、齊桓公為何不聽管仲的勸告，依然重用他們三人

先說說管仲是如何評價他們三人的，易牙為了滿足齊桓公想吃人肉的口腹之慾，居然殺掉了自己4歲的兒子，做成菜獻給齊桓公。俗話說虎毒不食子，這麼一個狠人，怎麼能擔當重任呢！

豎刁為了討好齊桓公，居然自行閹割，進宮服侍齊桓公。俗話說身體髮膚受之父母，不懂自愛的人，如何懂得愛人，更別提愛齊桓公了。

衛開方貴為衛國貴族，他捨棄國內榮華富貴，跑到齊國投奔齊桓公，而且為了給齊桓公作樂，自己父母去世都沒回家奔喪，十幾年了就呆在齊桓公身邊。俗話說，非我族類，其心必異，衛開方留在齊桓公身邊到底有什麼企圖，誰人知曉！

再來看齊桓公的態度，管仲雖然痛批了這三人，但齊桓公根本沒放在心上。這就要說到他們三人在齊桓公心中的分量了，易牙對他來說就是個廚子；豎刁對他來說也只是個服侍起居的僕人，況且此前根本沒有太監作亂的先例；衛開方在齊國毫無根基，無法威脅他的霸業。說到底他們三人在齊國都算是小人物，齊桓公根本不會覺得他們能翻起什麼大浪來。

但往往歷史就是被小人物改寫的，齊桓公沒認識到這一點，他的後人同樣也沒有！
