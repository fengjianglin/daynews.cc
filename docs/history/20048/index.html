歷史上著名的4大謊言沒有1個是真的，至今仍有人相信，你被騙過嗎

2019-12-16 1,152條評論 4,102次閱讀 1136人點贊

古希臘關於特洛伊木馬的傳說，那時候希臘聯軍雖然包圍了特洛伊很長時間，但是他們卻沒有辦法拿下這座城池。於是他們假意退軍，卻留下一個巨大的中空木馬，但是特洛伊的士兵並不知道這是希臘軍隊的陰謀，於是非常開心的就把木馬作為戰利品運回了城中。而藏身於特洛伊木馬中的希臘士兵在深夜打開城門，特洛伊這才被淪陷。這只是個傳說，其實在歷史上沒有任何證據證明這個故事是真的，只是後來人們用這個典故比喻在敵方埋下伏兵進行裡應外合的軍事活動。

斯巴達的三百勇士。歷史上謠傳的斯巴達在攻打波斯的軍隊，是以300人對戰200萬人，並且竟然還抵抗了三天，不要在傻了吧唧的相信了，用你最常規的大腦思考這也不可能好不！斯巴達可以是300人，但是波斯軍隊是不可能有百萬人的，多說也就幾萬，還有可能就幾千，歷史上這麼說，只不過想誇張的表達斯巴達人民很勇敢。

埃及豔后。埃及豔后是古埃及的最後一位法老，非常有名，地位非常高。亞歷山大去世後，他的一個手下建立了托勒密王朝，這個人是托勒密，當時統治著埃及的大片地區。因此，埃及豔后不是埃及人，而是古希臘人，這些都是由根據的。此外，埃及豔后並不以其美貌而聞名。很多時候在大家印象中她都非常漂亮，其實不是她反而有些醜陋，就像三國時期的黃月英一樣，因為有才華和智慧被凱撒深愛。

小矮子拿破崙。在許多人的印象中，拿破崙是一個矮冬瓜。事實上，拿破崙身高五英尺7英寸，也就是1.70米，這是十九世紀的正常高度。就算是現在，這個身高也算是中等的。據說，相傳拿破崙矮小的原因是拿破崙曾經說過：「我的個子是比你矮，但是如果你因此鄙視我，我會砍下你的頭，消滅它的差別。」可能也是因為當時軍隊里的平均身高都較高吧
