秦始皇的幾項偉大工程，禍及當時，卻功在千秋

2019-12-24 3,783條評論 4,856次閱讀 3434人點贊

秦始皇時期，建造了很多個堪稱偉大的工程，這些工程禍及當時。不過，從工程本身及其產生的重大作用來看，能看到有的工程可以說是「功在千秋」，接下來就一起來看看有哪些吧。

靈渠。公元前214年，靈渠鑿成通航。靈渠連接了長江和珠江兩大水系，構成了遍布華東華南的水運網。靈渠雖然已經有上千年的歷史，但到現在依然發揮著重要作用。

鄭國渠。鄭國渠是用含泥沙量較大的涇水進行灌溉，增加土質肥力，於是農業迅速發達起來，雨量稀少，土地貧瘠的關中，變得富庶甲天下 。鄭國渠首開了引涇灌溉之先河，對後世引涇灌溉發生著深遠的影響。

秦直道。秦帝國在建立之後，為了應對北方的游牧民族入侵，在自己的國都咸陽修建了一天能夠直接抵達長城腳下的通道。這條道路也為後來漢武帝北擊匈奴提供了方便，直到現在，人們都能夠看到這條道路。

五尺道。你可能只聽過秦直道，那個是高速公路的原型，而五尺道則是打通了雲南和內地最古老的官道，它讓雲南和蜀地的交流變得頻繁起來，可惜比較遺憾的是五尺道如今已經只殘存了350米左右，已經不復當年之壯觀。
