宋代最懶詩人，一生只寫半首詩，卻成為千古名句。你肯定聽過！

天天要聞
2019-12-18

在宋朝時出現了一位懶詩人，這個詩人懶到了何種程度呢？他一生只寫了半首詩，然而這半首詩卻流傳至今，頗為經典。這個詩人是誰？所作的這半首詩又是什麼內容呢？

我們先來看一下作者的出身，這個人叫做蘇麟，是杭州屬縣巡檢。他所作的這半首詩與著名文學家，政治家范仲淹可謂密不可分。范仲淹是一位開明大度的領導，他經常舉薦部下，破格錄用人才。蘇麟運氣不好，總是和范仲淹舉薦人才的時間巧妙錯過。看著別人平步青雲，前途無量。再看看自己總是原地踏步，不盡人意。苦悶之際，他心生一計！

蘇麟巡檢歸來後，去拜見范仲淹。他借請教詩詞的名義，向范仲淹暗暗表達自己意願！蘇麟深知說的過於直接，會引發范仲淹反感。當他們二人走到一處亭台時，蘇麟脫口而出：「近水樓台先得月、向陽花木易為春」。這句詩讀起來朗朗上口、借物抒情，范仲淹文學功底深厚，自然明白其中的意思！

范仲淹仔細考量了一下蘇麟的工作態度，工作業績。反省了一下自己對經常外出巡檢之人的忽略，然後給蘇麟寫了一封舉薦信，蘇麟不久後升職！很多人說蘇麟巧用「樓台、花木」表達內心不得志想法實乃高手；也有人說他的升遷和他的努力、爭取密不可分！

他一生只作了半首詩，你覺得蘇麟懶嗎？
