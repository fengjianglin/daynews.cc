辱母殺人案塵埃落定，德治與法治有待權衡

天天要聞
2019-12-19

古人自以為妙的德治

東周的混亂局面，你征我伐，造就了中國思想史無僅有的大爆發，各種思想層出不窮，諸子百家百花齊放！隨著秦朝帝制的建立，中國至此走向漫長的兩千多年帝制時代。及至漢武，中國的上層社會似乎找到了統治偌大的疆域最好的辦法—那便是德治。孔子被尊為致聖先師，永想祭祀。對於一個由小自耕農組成的國家，同時由於疆域龐大，自然災害不斷，周邊游牧部落始終存在威脅，管理手段有限，這一套方法在漫長的帝制時代確為最行之有效的統治方法。哪怕朝代更替，而統治方法一脈相承！

變強圖新之路

帝制中國的拐點在哪裡，一直以來都是史學界爭論不休的話題。看待歷史的角度不同，找到拐點額大相徑庭。美籍華人黃仁宇，一個隊歷史頗有造詣的史學家，出過幾本中國史的書，銷量也很不錯，他看歷史的角度基本都從經濟角度作經線。畢竟我們從小接受的理念也是經濟基礎決定上層建築，不妨借鑒黃老先生的視角來看待。

元朝是一個隔斷的朝代，雖然不足百年，但它的影響導致了往後五百年中國由盛轉衰直至被蹂躪的悲慘境遇。明代後期思想史的發展可被看作自由主義在中國的發端，清代末期一系列的運動則直接可以當作中國尋求法制的艱苦行程。帝制終結後的百年，更是中國人追求法制離開德教桎梏的漫長摸索。時至今日，我們依然和我們國家的富強相輔相成，實現了中華民族的偉大復興。

幾個簡單的例子

在古代中國，為了便於統治，長幼尊卑分的很清楚，在帝國最小的統治地位—自然村，會有族長，他們依據所謂的族規懲善罰惡，維持秩序。所以有句古話叫「皇權不下縣」！

通姦在古代是非常嚴重的罪名，女浸豬籠，男的亂棍打死。在現在社會看來不可理喻的事情，在古代卻被人拍手稱快，為了德治，統治者無所不用其極。

明太祖朱元璋，一位真正的布衣天子，此生極惡貪腐，剝皮實草就是他的傑作。德行也是老朱極力標榜的，只是他自己的德行又好到哪裡呢？從靖難之變就可以看出老朱連家人的德行都沒有教育好。

早在隋唐就開啟了科舉取士的制度，打破了士族門閥政治，可這並不意味著教育的普及。何為德治，君臣父子，各安其命是德治。

辱母殺人案

「殺人償命，欠債還錢」這便是天經地義，只因前面多了一個詞，才使這個案子不斷發酵，牽連出越來越多的內情。網上對這件事的討論異乎尋常的激烈，忽然想起寶強離婚案，案件不同，但很多人的出發點卻是如出一轍，那便是道德的審判。法制如此深化的21世紀，德治卻甚囂塵上，這其中的原因當然深邃，如此反覆，真怕重新回到失控的境遇，正所謂重口鑠金、積毀銷骨！相信法律最終給出的一定是最公正的審判！

子曰：「以德報怨，何以報德。以直報怨，以德報德」。

違反了法律不代表做錯了，明知不可為而為之是源於我們自己的判斷。法無情，人有義，我們可以有情有義，但是法律同樣應該無情無義。無數革命先烈拋頭顱灑熱血為的不就是用法制管理這個國家。立法的缺失，司法的推諉，執法的不負責任，林林總總造成了諸多讓國人難以忍受的案件。

德治當然有它的合理性，他在帝制時代保證了中國國家統一，同時它也是維持帝制最有效的手段。只是溥儀退位已經105載，短暫的袁大頭也距今百年，中國在民主法治的百年征程中重新找回了自信，屹立於世界民族之林， 因此，不要因為自己的喜惡去干擾法律的公正。

——巫見
