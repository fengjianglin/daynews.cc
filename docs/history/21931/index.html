北美印第安人來自中國但非殷商，南美印第安人源自非洲或古埃及

2019-12-19 1,123條評論 3,007次閱讀 1331人點贊

印第安人稱呼的來歷

公元15世紀是葡萄牙和西班牙的航海時代，由於葡萄牙探險家迪亞士Dias於1487年開闢並控制了非洲航線，西班牙人需要探索一條新航路通往東印度群島（印度和馬來群島）撈金，根據當時的「地圓說」，義大利探險家哥倫布Columbus腦洞大開：葡萄牙人向南航行繞過非洲能去印度，地球是圓球形，那麼向西航行也照樣可以到達印度，這個偉大的探索計劃得到了西班牙王室的支持。

於是1492年8月3日，他們從西班牙的內陸港塞維利亞出發（古稱巴羅斯港），向西橫跨大西洋抵達中美洲加勒比海的巴哈馬群島，哥倫布以為他已經到達了印度，就稱這裡的原住民為「印度人Indian」，中文為了與印度人區別，翻譯成「印第安人」。此時整個美洲大陸約有8000萬印第安人，他們大部分居住在中美洲和南美洲，而北美有將近2000萬。

北美的印第安人要比拉丁美洲的印第安人落後

印第安人看上去長相類似，膚色接近，但文化和服飾差別很大，他們並不是同一個民族，印第安人只是歐洲人對美洲大陸原住民的一個籠統叫法。如果稍一對比南、北、中美洲的印第安人，你會發現：北美洲的印第安人文化遠遠落後於拉丁美洲的印第安文明。中美洲墨西哥附近的印第安人創造了神秘燦爛的瑪雅文明，南美洲安第斯山脈的印第安人創造了不可思議的印加文明。但北美洲的印第安人卻一直處於原始狀態，他們既沒有創造類似瑪雅和印加的文明，也沒有建造類似瑪雅的奇琴伊察、烏斯馬爾金字塔，更沒有建造印加的馬丘比丘、普馬彭谷這些神秘的巨石建築。

北美洲幅員遼闊，既有足以孕育超級文明的6000公里長的兩河流域（密蘇里河和密西西比河），又有廣袤無垠的150多萬平方公里的美國大平原（而中國的東北、華北、長江中下游三大平原的面積總和才85萬平方公里），如此得天獨厚的自然條件，遠比尼羅河流域和中東兩河流域優越的多。但是北美的印第安人卻一直處於史前蒙昧狀態，即使與中南美洲的瑪雅文明和印加文明相比也不可同日而語，這很不正常，不符合人類發展的規律。

北美的印第安人真的來自殷商嗎？

印第安人是歐洲人對南、中、北美洲所有土著居民的統稱，並非單指某一個民族或種族，他們有幾百種不同的語言，並不屬於同一民族。拉丁美洲的印第安人的文明如此發達，而北美洲的印第安人又如此原始落後，說明其來源也必然不同。那麼他們分別來自哪裡？難道北美的印第安人真的來自中國的殷商，我們的殷商文明有這麼落後嗎？

北美印第安人居住的傳統帳篷

印第安人的來源撲朔迷離，眾說紛紜：1、土生土長論。人類起源於地球的各個地區，而不是非洲，但美洲至今沒有發現類人猿或智人遺骸，而且整個美洲只有猴而無猿類，所以美洲並不是人類的起源地。2、印第安人來自歐洲。有的學者認為，最早的美洲人是由歐洲穿過烏拉爾山脈後，從西伯利亞經白令海峽進入美洲，這種說法證據並不充足。3、有些國人認為，印第安人是殷商後裔。他們為了躲避周人追殺，一路向東逃難，箕子立足朝鮮，其他人則繼續向東穿過白令海峽進入美洲。這種說法純屬臆造，我們的商朝已經進入青銅器社會，文明發達，英國人到達北美時，東部印第安人還處於母系氏族社會，西部雖處於父系氏族但尚未開化，這種說法及其荒謬，我們的殷商文明有那麼愚昧落後嗎？

北美的印第安人來自2萬年前的中國華北？

北美的印第安人的祖先可能來源於亞洲。目前，一般認為亞洲人是北美印第安人最早的祖先，他們從亞洲跨越白令海峽到美洲。1972年，中國科學院考古隊在河北省張家口市陽原縣東城鎮的虎頭梁遺址中發掘出200多件楔形石核，而在阿拉斯加的阿克馬克遺址也出土了楔形石核，這兩個遺址的石核形狀和加工方法完全相同！而虎頭梁遺址又顯然早於阿克馬克，這說明美洲的石核技術源自於中國華北。

這一推論被俄羅斯考古學家所證實，他們在臨近北美的堪察加半島楚克奇地區考察時，發掘出了大量的石器(石鏃、石刀等)，證明舊石器時代前，就有狩獵者在此活動。約2萬多年前，會製造楔形石核的華北古獵人，在追獵野獸的過程中，由中國東北進入堪察加半島的楚克奇地區，又穿越白令海峽進入北美洲，把製造石核的技術帶入北美。中國學者的這種說法有待於更充分有力的證據加以佐證。

拉丁美洲的印第安人來自哪裡？

拉丁美洲的印第安人創造了三大文明：瑪雅文明Maya Civilization、阿茲特克文明Aztec Civilization、印加文明Inca Civilization，這三大文明似乎從天而降，各自創造了各自的神奇。他們雖然都沒有青銅器和鐵器技術，都處於新石器時代，卻在建築、天文、數學、農業等方面都有極高成就。他們卻建造了讓人瞠目結舌的各種巨石建築，這與古埃及建造了金字塔和巨石神像如出一轍，讓人百思不得其解。

印加文明的馬丘比丘遺址

他們一定和非洲的古埃及人有某種聯繫，都屬於以現代概念劃分的黃種人，他們都從某個更高等的外星文明那裡學到了巨石加工工具和切割雕刻技術，而且學會了如何建造形狀類似的金字塔和巨石建築。他們的祖先來自非洲，或者他們可能就是古埃及人，和哥倫布一樣，他們從北非橫跨大西洋來到中美洲的墨西哥，創造了瑪雅文明和阿茲特克文明，然後逐漸深入到南美洲的安第斯山脈，又創造了印加文明。
