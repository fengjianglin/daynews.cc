康熙曾孫永晧陵墓變成儲水井，考古人員進地宮都怕被水淹

2019-12-22 292條評論 1,284次閱讀 308人點贊

位於天津薊縣境內的恆敬郡王永晧之墓，早年多次被盜隨葬物品被洗劫一空，只剩地宮輪廓，未列入文物保護單位，後被村民用作灌溉儲水井。一般人不知其為何人墓之遺存，清陵專家徐廣源先生親探了此地宮。

並於紫禁城雜誌《恆敬郡王永晧園寢地宮親探記》一文中對此過程進行了記錄。

清代王爺 網路圖片

從直井式的地宮口委身進入地宮後，我們在地宮門券進行了觀察，我們把門券各部位都拍照了，測量過了之後，下一步就是進地宮金券了。

因為用作儲水井，地宮積水，對於進地宮的方案，我想到的第一個方案就是買一個大笸籮或一個大洗衣盆，人坐在上面漂進去。

恆敬郡王園寢地宮口已砌成了井  2009年8月第一次探訪時拍攝

我產生這個念頭也是有根據的。在1928年東陵被盜後，溥儀派幾個宗室遺臣到東陵重殮乾隆帝的裕陵和慈禧陵的遺骨時，發現裕陵地宮裡有四、五尺深的水，隨員徐榕生就是坐著笸籮漂進地宮的。這個想法剛產生沒幾分鐘，我就自我否定了。

恆敬郡王園寢的一孔拱橋  2009年8月第一次探訪時拍攝

我當即決定讓馮建明開車去村裡買高筒水靴子，馮建明欣然同意。他立刻又用繩子把腰拴好，我們把留在地面地宮口的李宏傑喊應，讓他在上面拽，馮建明邊用雙手往上拔，邊用雙腳蹬南北井壁。好不容易才出了井口。可把他倆累壞了，看來出井口比入井口更費勁。

我一個人留在地宮井口底，李宏傑怕我一個人在井下孤單，就坐在井口和我說話聊天。我想這裡距村裡不到二里地，開車到村裡小商店買雙水靴子，來回也用不了半小時。可是我們盼星星盼月亮眼巴巴地竟等了一個半小時才把他盼回來。一問才知道果香峪村裡沒有賣水靴子的，他是到10多里外的集市上轉了好一會兒才買到。一來現在穿高筒水靴子的少了，二來現在也不是賣水靴子的季節，所以賣的自然也就少了。

恆敬郡王園寢的朝山及一孔拱橋橋面  2009年8月第一次探訪時拍攝

幸虧馮建明聰明能幹，要是我去還真的買不回來。李宏傑用繩子把水靴子繫到井裡。然後又把馮建明繫到了井下。因為有上次的經驗，這次下井比第一次下井熟練多了。

有了水靴子，我們似乎能穩操勝券，勝利在望了。事實證明並不那麼簡單。因為只買了一雙靴子，只能一個人一個人地輪流進地宮。

被破壞的左扇石門

我第一個進去。當我穿著水靴子試著往水裡邁時，還沒有踩到底，水面就與靴筒上口沿平了。看來水太深了，能把靴子沒過來。怎麼辦？我真想一狠心，一咬牙，光著腳蹚進去。可又一想，地宮裡的水長年累月不見陽光，屬於陰水，一定涼得很，如果不顧後果，蹚水進去，一旦得了陰寒，落下疾病，後悔就晚了。

金券的南面

所以我暗自囑咐自己：千萬不要冒傻氣！難道就因為這點困難就半途而廢，打道回府嗎？實在不心甘。我通過手電筒光的照射，發現水裡有幾塊石頭。我試著踩石頭，水沒有沒過水靴子。於是我就踩著這幾塊石頭，居然走到了金券的棺床上。
