淺議唐太宗金口評三大名將：自古知兵者，無出李世民之右耳

2019-12-13 990條評論 2,720次閱讀 614人點贊

《資治通鑒》和《新唐書》記載，李世民晚年，曾經說道：「當今名將，唯李勣、江夏王道宗、萬徹而已。勣、道宗雖不能大勝，亦未嘗大敗；至萬徹，非大勝即大敗矣。」

李道宗是李世民的堂弟，是初唐一位難得的虎將。李道宗為大唐開疆拓土，「斥地千餘里」的他，卻被隋唐演義類小說黑得慘不忍睹。

正史上的李道宗，與凌煙閣第二功臣李孝恭並稱「初唐宗室二賢」。李道宗在正史上幾乎沒有打過敗仗，是當之無愧的名將。李道宗的英雄事迹很多，由於今天故事的主人公不是李道宗，就不繼續說他了。

李道宗劇照，圖片來自網路，感謝原作者。

今天故事的主人公，是李績和薛萬徹。李績，原名徐世績，字懋功，隋唐演義類小說裡面的軍師徐茂公就是他。李淵賜他國姓李氏，所以改名李世績。後來避李世民諱，又改名李績。績和勣通用，所以他又叫李勣。

薛萬徹，原本是李建成的黨羽。玄武門之變的時候，薛萬徹領兵攻打李世民，差點要了李世民的命。李世民不計前嫌，重用薛萬徹，還把親妹妹丹陽公主嫁給薛萬徹，讓薛萬徹成了大唐駙馬。

正史上的李績，可不是什麼軍師，而是一個十幾歲就殺人的虎將。李績不光武功高強，而且善於臨陣指揮，是個智勇雙全的帥才。《新唐書.李績傳》評價說：「其用兵多籌算，料敵應變，皆契事機。」

李淵劇照，圖片來自網路，感謝原作者。

薛萬徹是正史上的敢死隊長專業戶，此君每次打仗，都是帶著敢死隊發起衝鋒。舉個例子，《新唐書》記載，唐軍攻打薛延陀的時候，薛萬徹「與虜戰磧南，率數百騎為先鋒，繞擊陣後。虜顧見，遂潰，斬首三千級，獲馬萬五千。」

薛萬徹每次打仗，都是帶著兄弟們衝鋒陷陣。這樣的將才，雖然可以鼓舞士氣，但是風險極大。一旦主將被殺或者被擒，部隊就無人指揮。部隊群龍無首，就有大敗的風險。

李績勇則勇矣，但是絕不蠻幹。李績這類指揮型帥才，打仗的時候絕不帶頭衝鋒。衝鋒陷陣的幹活，自然有部下兵將去干。李績等人，經常是「於後方瞭陣，謂之持重」。指揮官在後方，即便是前方部隊失敗了，也能組織剩餘的部隊有序撤退，不至於遭遇大敗。

李建成劇照，圖片來自網路，感謝原作者。

武德四年，李績和薛萬徹都遭遇了一場大敗仗。劉黑闥舉兵反唐，攻佔了河北的大片土地。李績和薛萬徹，作為大唐的部將，都參加了平定劉黑闥的戰爭。李績負責鎮守黎州，薛萬徹追隨燕郡王李藝，配合淮安王李神通，與劉黑闥主力決戰。

《資治通鑒.卷第一百九十》記載，李神通率領五萬大軍，擺開了長蛇陣，與劉黑闥決戰。劉黑闥以少勝多，擊潰了唐軍主力，消滅了唐軍三分之二的人馬。

李藝和薛萬徹本來取得了勝利，看到唐軍主力被擊潰了，只能退守藁城。劉黑闥追擊到藁城，李藝只能跟劉黑闥大戰一場。戰鬥一開始，薛萬徹和薛萬均哥倆，就發揮敢死隊長專業戶的特點，率領敢死隊向劉黑闥的部隊發起衝鋒。

李績劇照，圖片來自網路，感謝原作者。

劉黑闥是瓦崗軍名將出身，他能自立為漢東王，戰鬥力也是十分強悍。薛萬徹和薛萬均兄弟的不要命打法，讓他們成為當世名將。只可惜，薛萬徹兄弟這次碰到了強悍的劉黑闥，敢死隊打法不靈了。

薛萬徹和薛萬均衝鋒在前，這次沒有回來，被劉黑闥活捉了。主將被活捉之後，唐軍兵敗如山倒，吃了一場大敗仗。在後方瞭陣的李藝看到形勢不對，只能撒丫子就逃跑了。

劉黑闥活捉名將薛萬徹和薛萬均，可謂是志得意滿。劉黑闥為了羞辱薛萬徹和薛萬均，把他們的頭髮剃光，在軍中當奴僕使用。薛家兄弟畢竟是虎將，武藝十分了得，他們找准機會，奪了漢軍戰馬，得以逃出生天。

薛萬徹被剃光頭的事情，不是小說段子，而是出自《資治通鑒.卷第一百八十九》的記載，原文是：「黑闥就擊之，藝亦敗，薛萬均、萬徹皆為所虜，截髮驅之。萬均兄弟亡歸，藝引兵歸幽州。」

薛萬徹劇照，圖片來自網路，感謝原作者。

劉黑闥活捉薛萬徹之後，向李績所部發起猛攻。李績派五千兵馬迎戰劉黑闥，自己在後方瞭陣。唐軍打不過劉黑闥，李績一看形勢不對，撒丫子就跑了。

此事典出《資治通鑒.卷第一百八十九》的記載，原文是：「甲子，黑闥追擊世績等，破之，殺步卒五千人，世績僅以身免。」

李世績派出去的五千兵馬雖然全軍覆沒了，但是他逃了回來。李世績逃回來之後，組織剩餘的人馬，向李世民所部方向靠攏。李世績與李世民匯合之後，馬上組織反擊，打死了劉黑闥任命的宰相高雅賢。

此事典出《資治通鑒.卷第一百九十》的記載，原文是：「壬辰，黑闥以高雅賢為左僕射，軍中高會。李世績引兵逼其營，雅賢乘醉，單騎逐之，世績部將潘毛刺之墜馬，左右繼至，扶歸，未至營而卒。」

李世民劇照，圖片來自網路，感謝原作者。

李績持重，打仗留有後手，就算是打了敗仗，也能穩得住局面。薛萬徹打進衝鋒在前，贏的時候就是大勝，輸的時候就是大敗。李世民打薛延陀的時候，讓二人取長補短。李世民讓李績擔任主帥，薛萬徹擔任副帥。

李績在後方指揮，薛萬徹帶領敢死隊在前面衝鋒。二位名將相得益彰，取得了平定薛延陀的大捷。李世民知人善任，發揮部下的特長，避免部下的特短，讓部下發揮最大的威力。

自古知兵者，無出李世民之右耳！李世民不僅善於用兵，而且善於用將，難怪史書會給予李世民如此高的評價。
