張飛在長坂橋“橫矛立馬”，許褚夏侯惇等一流猛將為何不敢應戰？

天天要聞
2019-12-22

長坂坡之戰，趙雲七進七出，救出甘夫人及小阿斗。他闖出重圍後，來到長坂橋，大呼：「翼德援我！」張飛於是讓趙雲先走，他斷後，阻擋追殺而來的曹軍。後來，曹軍手下大將文聘、夏侯惇、張遼、許褚等曹營眾將相繼追殺而來。面對長坂橋「橫矛立馬」的張飛，文聘、夏侯惇等諸多曹將為何不敢應戰呢？具體分析如下。（本文純屬三國演義，與正史無關）

文聘為何不敢應戰

文聘是第一個追殺趙雲來到長坂坡的曹將。他來到長坂坡後，只見張飛「倒豎虎鬚，圓睜環眼，手綽蛇矛，立馬橋上」，他從心裡膽寒，又看見長坂橋的東邊樹林中，塵土飛揚，懷疑有伏兵。於是他便勒住馬，等待後續大部隊到來，沒有上前大戰張飛。文聘為何沒有第一時間大戰張飛呢？他曾是荊州牧劉表手下大將，後來跟隨新即位的劉琮歸順曹操。

曹操來到荊州時，蔡瑁、張允等人對曹操卑躬屈膝，而文聘卻沒有第一時間拜見曹操，拜見新主公。他還在為自己沒有守住荊州而自責，感覺對不起老主公劉表，身為大將，卻沒有守住自己的一畝三分地。因此，他從內心上上，對曹操還是抵觸的。

《三國演義》：隨命於襄陽諸將中選一員引軍開道，諸將中卻獨不見文聘。操使人尋問，方才來見。操曰：「汝來何遲？」對曰：「為人臣而不能使其主保全境土，心實悲慚，無顏早見耳。」

在長坂坡，文聘也是第一個追上劉備的，結果被劉備罵了一頓，說他是「背主之賊」，怎麼還敢來追殺我？文聘羞愧而走。這一次，熟悉地形的文聘又是第一個追趕趙雲而來，遇到張飛。他內心有愧疚，同時看到張飛「橫矛立馬」，因此沒有第一時間大戰張飛，而是等待曹營大部隊來到，再做打算。他真的沒必要為了曹操的事情，那麼拚命。

夏侯惇為何不敢應戰

夏侯惇是曹魏宗親大將第一人，可以說是曹操的左膀右臂。遇到難以匹敵的敵將，夏侯惇常常不顧自己生命危險，衝鋒陷陣。在和呂布的對戰中，夏侯惇身先士卒，和呂布手下大將高順進行巔峰對決。兩人大戰四五十回合，高順敗走，夏侯惇縱馬追趕，眼看快要追上時，被高順手下部將曹性射了一箭，正中他的左眼。夏侯惇情急之下，用手拔箭，用的力氣大了，將眼珠帶出，他大呼：」父精母血，不可棄也！」於是「拔矢啖睛」。

夏侯惇忍者傷痛繼續提槍縱馬，一槍挑了曹性。夏侯惇失去一隻眼睛後，武功大減，後來就不再狂了，遇到敵將量力而行。他逐漸轉向決策層，轉向後勤，衝鋒陷陣的時候少了。如這次長坂坡，他就沒有逞能，大戰趙雲。因此，在長坂橋，夏侯惇自知不敵燕人張飛，也就沒有貿然衝上去。

許褚、張遼、曹仁等其餘曹將為何不敢應戰

除了新歸附的文聘、宗親大將第一人夏侯惇外，當時在長坂橋的還有曹仁、李典、夏侯淵、樂進、張遼、張郃、許褚等大將。這些大將之所以沒有大戰張飛，主要有這三個原因。

第一，曹營眾將怕再一次中了諸葛亮的計策。

在諸葛亮初出茅廬後，曹營眾將就多次中計。如諸葛亮火燒博望坡，將夏侯惇、于禁、李典等人燒得不知道東南西北，大敗逃回許都，諸葛亮火燒新野，使曹仁、許褚等將損失慘重。總之，曹營諸多戰將，被諸葛亮燒怕了，怕在長坂橋再一次中了諸葛亮的計策，被再燒一遍，於是在長坂橋等待曹操到來，再決定是戰是守。

第二，曹操沒有明確下達攻打長坂橋「橫矛立馬」的張飛。

曹操來到長坂橋後，看到只有張飛一人，在長坂坡上「橫矛立馬」的張飛，疑心病又犯了，怕偷雞不成蝕把米，於是猶豫不決。他想起關羽曾說過，他的三弟張飛才是真正的「萬人敵」，於千軍萬馬之中取上將首級，如探囊取物。曹操於是告誡自己的手下大將們，遇到張飛，要小心應對，不可輕敵。因此，曹操沒有下達進攻的命令，眾多曹將們也就沒必要逞能了。

第三，曹營眾將和曹操都被張飛的「三聲大喝」嚇著了。

這時，張飛精神倍漲，連續大吼了三聲，一聲高過一聲，第一聲為「我乃燕人張翼德也！誰敢與我決一死戰？」他先自報家門，而「曹軍聞之，盡皆股慄」，曹操也拿掉了傘蓋。緊接著張飛第二聲大喝：「燕人張翼德在此！誰敢來決死戰？」曹操嚇得有了逃跑的心思了。張飛看見曹軍膽怯，發出第三聲大喝：「戰又不戰，退又不退，卻是何故？」

這三聲大喝，一浪高過一浪，將曹將夏侯傑嚇得肝膽碎裂，當場死亡。曹操和諸軍眾將向西逃走。

結語：張飛在長坂坡「橫矛立馬」，嚇得曹營眾多猛將不敢應戰。大概就是以上幾個原因。張飛有勇有謀，在長坂橋故布疑陣，三聲大喝鎮住了曹將，成功拖住了曹軍追兵，掩護劉備等人順利逃走。
