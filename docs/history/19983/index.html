「歷代王朝皇帝世系」春秋戰國（陳國）國君即位順序和世系圖

2019-12-16 431條評論 1,517次閱讀 485人點贊

陳國開國國君媯滿是舜帝後裔，被周武王賜爵封地，建立陳國。陳國在株野（今河南柘城）建都，後遷都宛丘（今河南淮陽）。鼎盛時期，國土大致是河南東部和安徽西北部一部分。

（一）舜裔立國

周武王滅商後，訪求前代帝王的後裔，找到了舜帝的嫡裔媯滿，把長女太姬嫁給他，賜公爵位，封地建國，國號為陳。陳國奉守舜帝宗祀，被周天子尊為「三恪」之一。

（二）依附強鄰

陳國是處於中原地帶的小國，靠依附大國生存。

春秋早期，鄭國對陳國的影響很大。公子佗、厲公、庄公和宣公的繼立都有鄭國的影響。

楚國從楚武王時期逐漸強大起來。陳開始親楚。

晉國在城濮之戰中打敗楚國後，晉文公成為霸主，陳國自然也要結交晉國。陳國親晉的行為遭到楚國的報復，親楚的行為又遭受晉國的打擊，陳國不得不搖擺於兩強之間。

（三）陳國滅亡

春秋後期，東方吳國崛起，與楚國競爭。陳國不得不遊走於吳楚之間。

後來，吳國被興起的越國壓制，與吳交往的陳國被楚國吞併。

（四）陳裔代齊

田氏齊國的先祖陳完是陳厲公的公子，被捲入陳國宮廷鬥爭中，為避禍逃到齊國，並改名為田完。田完受到齊桓公的賞識，在齊國謀一官職，田氏家族逐漸興盛起來。田完後人田和取代姜姓當上了齊國的國君。
