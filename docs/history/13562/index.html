此人精通相面術，他為趙匡胤相面之後，再也不敢相面了

2019-12-12 93條評論 1,452次閱讀 65人點贊

相面術是古代就存在的，最早的相面術出現在了春秋戰國時期。《荀子·非相》中對相面有這樣的解釋：「形相雖惡而心術善，無害為君子也。」，古代的時候，相面的人通常被稱之為「相士」。古代的時候，相士一般都是給人看相算命的。

相士在古代的時候是非常厲害的，首先來說，能夠通過相貌進行算命，本身就是很難想像的。古人經常說一句話：「人的命天註定」，在這樣的情況下，算命之後，無疑是對自己的未來想要有所了解。可是根據人生經驗來看，知道自己的未來並沒有什麼好處。

人的命運是由天定的，想要通過面貌看出一個人的命運，這沒有一定的人生經驗肯定是不行的。相士都是可以通過相貌進行算命的，經傳中記載，黃帝威嚴像龍，顓頊額闊如盾牌等等。總的來說，相士看相是有一定道理的。這是古人研究上千年的東西，肯定是有所作用的。

古代的史書中，即使是正史中，也是存在關於相面的記載。比如說《漢書·藝文志》中就有《相人》二十四卷。民間更是流傳了《麻衣相法》和《水鏡神相》等。隨著科學的發展，相面在科學領域也有所研究，就是在心理學領域有所研究。

心理學家開始重新探索麵相與人格之間的關係，從心理學角度的研究來說，一些研究成果讓人們得以從新的角度重新審視古老的相面之術。五代十國時期，就出現了一位非常厲害的相士，只不過在他的一生中，最後一次相面，就是給宋太祖趙匡胤。

王彥超是就是五代十國的相士，相比較來說，王彥超更是一位武將。從五代十國開始，一直到宋朝建立後，王彥超的官職沒有什麼大的提升。從後晉至後漢，王彥超與當時的武將一同任職，在這樣的情況下，其他武將都開始升職。

只有王彥超自己原地不動，當時的王彥超是復州﹙今湖北沔陽縣﹚防禦使。與其他武將相比，王彥超升遷無望，這時候的王彥超開始研究相面術，於是王彥超對相面術的提升是很大的。在後來的時候，王彥超的相面術已經達到了精通的地步。

值得一提的是，王彥超的一生中，對相面術可以說是達到了痴迷的境地。王彥超學會了面相術之後，開始學會給別人相面，在這樣的情況下，王彥超更是相信自己能夠遇到貴人。甚至是王彥超竟然給其他人不斷的相面。五代的後漢時期，趙匡胤來到了復州。

王彥超與趙匡胤的父親趙弘殷是同僚好友，並且兩人都是一起做官的，由此說來，王彥超與趙匡胤是有某種聯繫的。當時王彥超在復州任職，趙匡胤想通過這層關係，得到王彥超的重用和提拔。只可惜當時的王彥超沒有同意這樣的做法。

王彥超給趙匡胤相面後，給了他十貫錢，於是讓趙匡胤離開了。當時的王彥超對趙匡胤是有相面的，之所以讓趙匡胤離開，王彥超肯定是有想法的。趙匡胤建立北宋後，對這段往事是非常在意的，當時的趙匡胤找到了王彥超。

當時的趙匡胤宴請了王彥超，在宴請的宴會上，趙匡胤說到：「卿昔在復州，朕往依卿，何不納我？」，這句話的意思就是說，「朕昔日來複州投靠你，你為什麼不接納我？」，當時的王彥超回答說：「勺水豈能止神龍耶！當日陛下不留滯於小郡者，蓋天使然爾。」

這句話的意思就是說淺水怎能藏住神龍？當日陛下沒有留滯在小郡，這實在是天意啊！趙匡胤聽完之後，覺得不應繼續追究此事。至於王彥超到底有沒有給趙匡胤看相，至少在史書中沒有記載，但是從這之後，史書沒有了關於任何王彥超相面的記載。
