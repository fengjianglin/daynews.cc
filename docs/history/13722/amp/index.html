他是關羽之子，共斬殺5員敵將，最終為父報仇

天天要聞
2019-12-10

他是三國時期蜀國的著名將領，武聖關羽的兒子，在三國演義中特別是蜀漢後期，十分勇猛，記載中他殺了5名武將，另生擒一名讓張苞殺死。關興弱冠之年就隨父上陣，在關羽生前一直在軍中效力，後來關羽水淹七軍奪下荊州，關興前往漢中報告喜訊，也因此在後來關羽大意失荊州的時候，逃得一命，在關羽死後繼承了關羽漢壽亭侯的爵位，成為了諸葛亮北伐大軍中不可忽視的重要力量。

他就是關興，無論武藝和戰績，在蜀漢陣營的二代年輕人中都是較為出眾的，比張苞要略勝一籌，他的斬獲率還是相當不錯的。

快刀斬吳將李異

李異是三國時期吳國的將領，在軍中有萬夫不當之勇，是屬於武衛都督孫桓帳下的大將，在蜀國皇帝劉備攻打東吳的時候，李異跟隨孫桓前往迎敵，蜀國將領張苞（張飛的兒子）和孫桓帳下的另一名將領謝旌對陣三十多個回合，之後謝旌不敵張苞落敗。

李異眼見謝旌要被殺死，連忙拍馬迎戰張苞，對敵之間，張苞的戰馬被敵軍射冷箭射倒，張苞墜馬，眼見要被李異的大斧子砍到，關興急忙趕來，大喝一聲嚇得李異一哆嗦，然後關興一刀將李異斬於馬下。

秒殺潘璋，為父報仇

關興在關羽去世後跟隨劉備一起參與討伐吳國的戰役，並且拜為了龍驤將軍。在這場戰役中，關興殺入敵方陣仗中，剛巧遇到殺父仇人潘璋。關興此刻是熱血沸騰，遇見仇人簡直分外眼紅，關興立馬緊追正在逃跑的潘璋。

可能是關興的怒火讓自己失了神，竟然在這麼關鍵的時候迷路了。於是關興便在谷中找了一間民宿，屋裡的老人見其是武聖之子而收留了關興住下。事情總是這麼巧合，三更天，逃跑的潘璋也來到這家民房住宿，想要強進住宿一宿，卻碰見了從裡面走出來的關興。

此刻的潘璋還以為是關羽顯靈，愣是被這一幕嚇得呆住了，於是關興不費吹灰之力，輕輕鬆鬆地便把潘璋斬殺了，並且奪回了父親關羽的青龍偃月刀。至此，關興殺潘璋為父報仇，解決了心中的一件大事。

之後，關興成為了諸葛亮北伐的主要將領之一，在種種戰役中表現的十分突出。幾年後，在第六次北伐征討前因病逝世。

關興前後共斬殺敵將五員，雖然其中殺潘璋有取巧的因素，其他四人則沒有什麼名氣，不過關興出手快，都是一刀殺死，所以功夫還是不錯了，頗得關羽快刀的真傳。
