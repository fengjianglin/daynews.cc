原來古代人這樣洗衣服，沒有洗滌劑，一樣洗得很乾凈

2019-12-13 1,508條評論 3,262次閱讀 2072人點贊

原來古代人這樣洗衣服，沒有洗滌劑，一樣洗得很乾凈

嗨嘍大家好，大家平時洗衣服應該都會使用洗衣粉等專用的洗滌劑來清洗，效果好還快速，去污也很好，這主要還是因為這些洗滌劑的成功發明，讓洗衣服變成一件簡單的事情。

但是古代的時候人們一樣需要穿衣服洗衣服，又沒有這些豐富的洗滌用品，到底是怎樣來請洗衣服的呢，會不會很費力，衣服可以洗乾淨么，今天就給大家解決這個問題。

在古代沒有專業的洗滌劑，但是當時的人卻沒有直接用清水洗衣服，很早之前就有人研究出了類似於洗衣粉的清潔工具，就是用貝殼磨成粉末，再加上木灰攪拌均勻後洗衣服，可以去掉衣服上那些清水洗不掉的東西，就連油污都可以去除。

為了能夠更方便，後來也出現了類似於洗衣液的產品，就是將草木灰放入水中浸泡一段時間，然後用這個水來洗衣服，就不用擔心有顆粒難以清洗了，將這些本來也沒有什麼用處的草木灰變成了能夠充分利用的東西。

隨著不斷地發現，還有了一些我們都聽過名字的皂角，皂角的使用已經非常接近我們使用的洗滌劑，因為皂角豆放入水中就會有泡沫，這個泡沫就可以起到去污的作用，除了去污，皂角豆因為泡沫豐富，容易採摘，還應用到了洗澡洗頭方面，加上一些香料之後，衣服不僅會很乾凈，還會留下淡淡的香味，可見古人在服飾方面也是非常講究的。

光是這些用品還不夠，因為這些用品並沒有經過很好的處理，雖然可以去污，但是終究不算太有效率，古人為了能夠讓清洗速度更快，效果更好，就製造了一些用於洗衣的工具。比如洗衣盆可以用於浸泡衣服，省下搓洗的時間，還有專門用於捶打衣服的柜子，代替雙手卻比手洗還有效果，其實現在的很多洗衣機工作模式，也是效仿的古代的木棍捶打模式。

這也就是我們在一些影視劇中可以看到的搗衣杵，當然還有一些搓衣板等，也被沿用了很長的時間，起到了增加衣服摩擦力，讓衣服更快潔凈的效果，因為這些發明，讓智慧的古人在那個時代也可以較為省事省力的洗衣服。
