晉國殺死太子，逼迫重耳流亡的可能另有其人，驪姬也許是無辜的

2019-12-16 292條評論 2,321次閱讀 186人點贊

晉文公在外流亡19年，六十多了才回國繼位成就霸業。大家都知道這都是因為驪姬之亂。但真的都應該怪驪姬這樣一個女人嗎?

驪姬之亂的主要過程是驪姬通過陰謀陷害，讓晉獻公殺死了太子申生，驅逐了公子夷吾和公子重耳。導致晉國統治階級陷入混亂。這整個過程記錄在《左傳》里。但是這件事並沒有這麼簡單。

其實驪姬和妲己褒姒差不多，都是從敵對的小國通過戰爭弄來的美女，而他們的結局也很相似，都是被戴上了霍亂國家，擾亂統治的帽子。但是如今對於妲己褒姒的重新解讀也多了起來。說他們只是歷史的替罪羊。小編也在以前的文章解讀過這兩個人物。而驪姬其實也差不多。以下只從歷史的角度提出另一種視角，僅供參考。

首先看看晉獻公的成長軌跡，這一點很重要。在晉獻公繼位之前，他都只是屬於一個不被承認的親族叛亂集團，也就是在曲沃的小宗叛軍之中。在與翼城的大宗作戰中成長起來。親身經歷了曲沃代翼的歷史事件，所以沒有人比他更理解一個諸侯國內部的兄弟奪權是有多麼的殘酷。

後來晉獻公的父親晉武公終於再一次殺掉了翼城的正牌晉君，轉過一年才通過賄賂周王得到了諸侯的地位，正式的統治晉國。再過一年，晉武公就死去了。晉獻公繼位，這一年重耳21歲。

晉獻公即位後第一件事就是殺光了翼城原來的那些親戚們，並且遷都。他太恐懼他們找自己爭奪君主的位置。接著他憑藉晉武公留給他的高度集權的軍隊，那個因參與叛亂而空前團結的軍隊，南征北戰，中間他消滅了曾經的頂級軍事強國，虢國。這一功績可能僅次於直接滅了周王朝或禮樂之都的魯國。

晉獻公用26年「並國十七，服國三十八」可以說英雄了一輩子，也是他在近70年內戰後迅速的把晉國變成數一數二的強國。但是到了晚年他卻因為聽信一個俘虜來的寵妾就殺了太子，流放兩個得力的兒子，這難免讓人產生疑問。

其實在驪姬之亂中，驪姬的陰謀陷害，且不說這是否屬實，但就算都是真的，那這樣的晉獻公是那麼好欺騙的嗎？恐怕沒那麼簡單吧！驪姬不論在整個事件中扮演怎樣的角色，而其實最終決定權始終在英明的晉獻公手裡。而事實很可能不是驪姬的陰謀陷害，而是晉獻公自己要太子死，要讓夷吾和重耳流亡。

其實問題很簡單，這都是源於晉獻公成長時代就留下的心理陰影，那就是兄弟奪權，自相殘殺。我們都知道重耳好結交賢士，17歲時身邊就有五個賢人相助，等被驅逐的時候，身邊更是跟著幾十號能人異士。而夷吾的情況也差不多，這意味著什麼？這是拉幫結派啊，他們都在籠絡自己的班底，那麼接下來就可能發展成奪權內鬥了。而晉獻公對這種事是最敏感的。但是比起他們二人，太子申生就更加過分。

太子申生很早就 掌管兵權，這可能會追溯到晉武公時期，他手下有很多心腹，其中有一個叫里克就是他的鐵杆。里克在一開始就很維護太子，等晉獻公死後，他直接就在其葬禮上殺死了驪姬的兒子，後來驪姬的侄子也被他殺死，當然也包括驪姬本人。這就很可怕了，老君主剛死就殺掉兩任新君主，這裡面除了報仇外，更多的也是政治鬥爭。反觀驪姬，如果她真有本身逼死太子，也不會有這一天了。

所以當時的情況應該是晉獻公發現太子，夷吾和重耳都有培植班底的行為或趨勢，這讓他反應過度了。這其實和《雍正王朝》里雍正殺掉弘時很像，就是為繼任者掃清障礙，杜絕新的兄弟奪權發生。他以為殺了太子，那麼軍隊就能牢牢的掌握在新君手裡，但是他沒想到里克會做到這一步。

晉國在晉獻公以後都有一個傳統，那就是確定繼承人之後要把所有的公子都驅逐出國。這隻從驪姬之亂這件事來看實在沒有必要，甚至不應該這麼做。但是從晉獻公這裡就是必須要做的一件事了。但是人算不如天算。他沒想秦國有個秦孝公百般干預晉國內政，也沒想到驅逐出去的公子也可以回國繼位，更沒想到他選定的繼承人在他剛死就被大臣給殺了。而最沒有想到的是，重耳竟然那麼長壽，六十多了還能回來繼位進而稱霸。

驪姬可能扮演了很不光彩的角色，但是也有可能只是一個受害者。而真正的幕後黑手其實是從晉文侯時，近100年前就開始的國君位置的爭奪史，晉國為了這個付出了太多了。最後晉獻公防備兒子們互相奪權的手段徹底失敗，但是在秦穆公幾次三番的干預下沒有發展成嚴重的內鬥。而這個困擾了晉國近100年的問題的最終解決，還要交給後人。
