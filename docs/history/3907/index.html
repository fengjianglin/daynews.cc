李悝開創戰國改革的先河，是改革家中的倖存者，給我們什麼啟示？

2019-11-28 0條評論 21次閱讀 0人點贊

摘要

作者：喬永勝

一提起戰國時的變法圖強，大家腦海里首先湧現的就是商鞅變法。因為商鞅變法，使秦國實現了富國強兵，最終完成了中國一統。

那麼，這場變法運動，始於什麼時候，是誰最先開始的自省呢？盪去歷史的塵煙，我們清晰地看到，始於魏國的李悝變法，開啟了戰國時期各國變法的先河，並且取得了相當的成效，只是由於後來被秦國的勢頭蓋過，才歸於默默無聞。

（一）機遇垂青有準備者

這場變法，始於魏文侯和相國李悝的一次推心置腹的談話。

魏文侯說自己一心想儘快使國家富強起來，也採取了一些措施，但總覺得達不到理想的目標，問李悝怎麼辦。李悝深思熟慮後，提出了「盡地力」的農業政策。

一是鼓勵農民勤勞耕作，提高產量，適當減輕徭役和賦稅，給他們一定的利益。同時，鼓勵他們開墾荒地，把田頭道邊的土地充分利用起來，擴大耕種面積，實現糧食增產。

二是搞綜合開發。針對當時魏國糧食品種單一的狀況，鼓勵農民根據地勢和土壤的不同，選種不同的作物，發展多種作物兼種和套種，能種瓜的種瓜、會養桑的種桑、善植麻的種麻，實現多種經營，做到人盡其才，地盡其利​。

李悝的話切中時弊，也切實可行。要改革就得動根本。在私有制時代，土地就是最大的資源。把土地資源盤活了，就能解決好百姓的吃飯問題。

魏文侯一聽這個策略有理有據，就讓李悝全權負責國內的農業經濟改革。李悝責無旁貸，立馬發布改革文告，讓魏國百姓先行先試。老百姓們立即動手，打破舊有的阡陌封疆，不斷開墾土地。隨著土地面積的擴大，糧食產量大增，農民過上了安穩的生活，國庫收入也隨之得到大大提高。

農業豐收了，種糧難問題解決了，賣糧難問題就接踵而至。為了讓農民持續增收，李悝奏請魏文侯，又在經濟上實行「平糴法」。

他把豐年和飢年份​成三等，豐年分大熟、中熟、小熟；飢年分大飢、中飢、小飢。按照年成的好壞確定應納稅額和農民自留糧的數量，由國家平價收購，使糧價不至於暴跌；荒年時再平價出售，保證糧價不暴漲。這樣，豐年不會出現因糧價低而傷農的現象；遇飢年也不會出現因糧價高而百姓買不起糧食。

這個政策執行下來，市場上的糧價一直很穩定。百姓安居樂業，生活安定，國家收入穩中有升，魏國日益強盛達十年之久。

（二）經濟政治改革同步進行

經濟改革要往前推進，就得著手政治改革，破除「腸梗阻」。為此，李悝提出了「選賢任能，賞罰分明」的國策，廣泛招攬天下人才。

他主張不論貴族還是平民，只要有治國安邦的才能，都可以在朝廷做官，領取相應的俸祿。官員要各司其職，有功必賞，有罪必罰，不徇私情。凡無功而作威作福的，即使是貴族也必須除官消爵，取消俸祿。

這種按功勞大小和對國家貢獻多少授予官職和爵位的政策，打破了舊的世卿世祿制，在人才的使用上做到了公允平等、獎懲分明，很快吸納了天下人才湧向魏國，提高了國家機構的辦事效率和官員的素質，魏國上下形成了勵精圖治的良好局面。

李悝的政治改革也獲得了巨大成功。

為了持續推進改革，保護改革來之不易的成果，就需要把改革中成功的經驗、做法形成制度、固化下來，李悝又開始了依法治魏的嘗試。

他在參照以往的法令、吸納各國法令精華的基礎上，根據魏國的實際，編製了中國歷史上第一部比較系統完整的封建法典——《法經》。《法經》分為盜、賊、囚、捕、雜、具六篇，在保護統治階級利益，維護社會秩序、穩定政局等方面起到了重要的支撐作用。

（三）富國強軍齊頭並進

富國必須有強軍作保障。在走富國之路的同時，李悝也重視強軍為本。他自己雖然不懂軍事，但是通過奏請啟用將軍吳起，一起實行軍改。

李悝軍改的目的就是建立一支能征善戰的常備軍。

他選軍士的標準是：軍士需身穿三甲（脛甲、上身甲、股甲），肩負二十石之弓，帶五十支箭。扛長矛、頭帶盔甲，佩劍，備三天糧食，半天行走一百里。

一旦選中，享受優厚的待遇，免除全家徭役，並獎給田宅。

這樣精選出來的將士，個人素質都很過硬，又無後顧之憂。於是，軍隊迅速煥發出向心力，不斷發展壯大起來。短短几年間，魏國這支強大的軍隊的戰鬥力震懾了其他各國，使得「秦兵不敢東向」，各國不敢與之爭鋒。

（四）李悝改革的啟示

改革是一場牽一髮而動全身的事。這樣一場成功的改革，最主要的還是得益於君賢臣能，互相信任。魏文候、魏武侯兩位君主放手讓李悝大膽改革，才使得李悝的政治抱負得以施展。這個有效的領導集體，在那個時代至少從四個方面起到了「破冰」作用。

第一，這是第一個打破傳統井田制度的國家。

第二，這是第一個設立平糴制度，通過國家權力干預市場秩序的國家。

第三，這是第一次在奴隸制的政治體制下進行人事改革，把世襲制改成按能與功受職受祿。

第四，這是第一個明確把法律制度作為治國準則的國家，開啟了依法治國的先河。

這樣一場涵蓋了政治、經濟、社會的全方位的改革，直接衝破了舊制度的束縛，激發出人民無限的改革熱情，使魏國經濟迅速發展，國力日益強大，成為戰國初期一個強盛的國家。變法同時開啟了戰國大變法運動的序幕，各國紛紛變法圖強，最終匯成了一股時代潮流，形成中國古代規模最大、歷時最長、成效最顯著的一場變法運動。這場變法的許多做法，對後世產生了積極影響。

因此，許多學者把這場改革的主要倡導者李悝稱作法家鼻祖是實至名歸​、當之無愧的。

今天，我們之所以緬懷這場改革和李悝，是因為這是場沒有流血和硝煙的改革。歷史上大多數改革家的結局都是很悲壯的。比這稍晚的商鞅在秦國被五馬分屍，宋朝的王安石在變法失敗後鬱鬱而終，明朝的張居正更是在死後把家族帶入了萬劫不復。

那麼，李悝的改革，為什麼沒有出現這種絕地呢？改革作為推動歷史前進的動力，不可避免會在現實中遇到一些阻力，但怎麼才能做到改與造、人與事、進與讓的和諧，才是考驗政治家們政治智慧的根本。

李悝改革的現實意義就在於把人心思動和統治階級利益有機結合，在穩中求變，用法規範，使的一切活動是在一個穩定的獲利階層的主導下來進行的。這樣一步步的穩紮穩打，才避免了後世許多改革有聲望沒民心支撐，有熱情沒法制保護的慘痛結局。

讀史明智，我們通過全面審視李悝改革，就是為了在改革的進程中獲取最大量的支持，減少不必要的損失，把改革不斷推向前進。
