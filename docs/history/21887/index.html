真情易逝流年，人生若只初見，這位奇男子教你讀懂閨怨詩

2019-12-19 403條評論 4,640次閱讀 1445人點贊

他是滿腹詩書的一代詞人，他是風月情長的翩翩公子，他是人間惆悵客。

他是清朝才子納蘭性德，字容若。我說他奇男子，奇在三點：

他出身皇室，奇在淡泊。納蘭容若是位高權重的大學士納蘭明珠的長子，康熙帝身邊的一等侍衛。他的一生註定富貴榮華，鐘鳴鼎食，平步宦海，前途無量。但是納蘭容若卻無意功名，不屑仕途。

他能武能文，奇在兩全。納蘭容若自幼聰慧好學，文武兼修。學識廣博，熟讀經史，並把自己的感悟編纂成冊，天文地理，歷算音樂等等，無所不包。成為進士時年僅二十二歲，又以英俊威武的武官身份，經常隨駕出巡，深受皇帝賞識。

他瀟洒不羈，奇在真情。納蘭容若才華出眾，功名輕取，但是讀他的詩詞，讀出的都是難以掩飾的憂傷。他也並不是胸無大志，他滿腹經綸，本該安邦濟世，然而，官場的庸俗、職業的束縛讓他厭倦，甚至產生了至深的幻滅感。這位敏感的詞人寧願做江南布衣，在閨閣中尋覓真情。對，我說的不是賈寶玉。

納蘭容若因寒疾一病不起，溘然而逝，年僅三十一歲。所謂慧極必傷，情深不壽！

在中國文學史上，納蘭詞光彩奪目。現在我選他的一首經典詞作：

《木蘭詞·擬古決絕詞柬友》

人生若只如初見，何事秋風悲畫扇。

等閑變卻故人心，卻道故人心易變。

驪山語罷清宵半，淚雨零鈴終不怨。

何如薄倖錦衣郎，比翼連枝當日願。

這首詞的主人公是一位為情所傷的女子，她被變心的男人拋棄了，表示要跟他徹底分手。這是一首擬古之作，也就是仿照古代的《決絕詞》而寫的，以一個失戀女子的口吻譴責負心男。

最攝人心魄的就是起句了，「人生若只如初見」，這句話將人生當中那些不可言說的複雜滋味表達得哀怨凄婉：兩個人如果總像剛剛認識的時候那樣甜蜜，那麼深情，該有多好！這是美好的初戀。這也代表了詞人的夢想，然而夢想往往會變成遺憾，美好總是容易逝去。

初相遇時的四目相對、怦然心動，最終泯滅在倉皇的時光里，或許敗給了背叛，或許敗給了現實。然而煙花、流星和愛情，這些易逝的東西，卻有著讓人驚艷的美麗。

這首詞就屬於閨怨詩範疇。閨怨詩分為宮怨和閨怨兩類。宮怨主要寫古代宮女的孤獨或者失寵之後的幽怨，閨怨詩主要寫民間的棄婦的憂傷和思婦的思念、少女傷春懷春等。

閨怨詩寫女子的情感，那麼這樣的詩出自何人的筆下？一個有趣的現象是，大多數閨怨詩是男人寫的。因為過去的女子會寫詩的不多，又或許果真寫了卻被埋沒於閨閣之中，沒有流傳於世。

一類是女人寫的，表達自己的憂傷思念之情。比如卓文君的《怨郎詩》《訣別書》、李清照的《一剪梅》、朱淑貞的《秋葉》等。

一類是男人寫的，借女子的口吻表達對對愛情的追求，或對女子的同情。如：「寂寞空庭春欲晚，梨花滿地不開門。」「天階夜色涼如水，坐看牽牛織女星。」「忽見陌頭楊柳色，悔教夫婿覓封侯。」

再一類還是男人寫的，要表達的是委婉含蓄的懷才不遇的自傷。這是古代文人的「怨婦」情結。如朱慶餘的《近試上張籍水部》：「洞房昨夜停紅燭，待曉堂前拜舅姑。妝罷低聲問夫婿，畫眉深淺入時無。」

不一而足。
