陳平：能用錢擺平的事兒都不是事兒，且看劉邦謀臣陳平的花錢藝術

天天要聞
2019-12-23

一個好漢三個幫，歷史上的帝王在開創他們的帝王霸業時，都有著眾多的幫手，有些善於戰略謀劃，有些善於打仗，有些善於出奇謀。劉邦就是一個善於用人的帝王，而陳平則是劉邦手下最能幹的謀士之一。

▲陳平劇照

說起來，劉邦和陳平實際上還挺相似的，不拘泥於小節，善於使用手上的資源，看人看得都很准。就拿「花錢」這件事情來說吧，陳平花錢的功夫，可以說堪比劉邦用人的功夫。陳平花錢，每次都是花在了刀刃上，往往能出奇效，陳平一輩子為劉邦六齣奇謀，其中就有兩次是和錢有關係的，而且每次都幫了劉幫大忙。不得不說陳平真是把「花錢」這件事情玩兒成了「藝術」。

今天我們就來領略一下，陳平的花錢藝術。

在劉邦被困滎陽時施反間計：用錢砸出來的眾口鑠金

要說項羽為什麼能夠一直順風順水，那麼就不得不提起項羽的亞父，同時也是他最重要的謀臣范增了。范增這個老爺子那是相當的精明，早就看劉邦不順眼了，認為劉邦是扮豬吃老虎的高手，如果不小心防範，以後一定會成為大禍患，所以范增已經好幾次想要置劉邦於死地了。項羽攻入關中的時候，他勸過項羽殺劉邦，沒成功；鴻門宴的時候勸過項羽殺劉邦，沒成功；終於等到劉邦被困滎陽，這次他絕對不會放棄

▲范增劇照

劉邦被困於滎陽一年之久，糧草也已經快沒了，如果沒有出逃的辦法，估計就真的要交代在這兒了。陳平通過分析敵我雙方的主要矛盾，敏銳的發覺了楚軍的一個大弱點，那就是項羽剛愎自用，而項羽的主要智囊卻只有范增一人，猛將就只有鍾離昧、龍且等人。這是一個非常簡單，但又十分有效的分析，於是他果斷向劉邦進言：

項王為人，恭敬愛人，士之廉節好禮者多歸之。至於行功爵邑，重之，士亦以此不附。今大王慢而少禮，士廉節者不來；然大王能饒人以爵邑，士之頑鈍嗜利無恥者亦多歸漢。

顧楚有可亂者，彼項王骨鯁之臣亞父、鍾離眛、龍且、周殷之屬，不過數人耳。大王誠能出捐數萬斤金，行反間，間其君臣，以疑其心，項王為人意忌信讒，必內相誅。漢因舉兵而攻之，破楚必矣。

▲項羽劇照

其實陳平就是說，項羽雖然是貴族，謙恭有禮，很多士人願意歸附於他，但項羽在論功行賞的時候又顯得小氣，士人們心有怨言；大王你雖然痞里痞氣的，但是你捨得封賞，無論爵位也好，錢財也好，這樣的話，那些沽名釣譽的人一定會來歸附的。現在正好用錢來收買那些心有怨恨、沽名釣譽的人，用他們來離間項羽和賢臣們的關係，藉此製造機會讓你逃跑。

▲劉邦劇照

劉邦同意了，陳平拿出了滎陽城中的四萬斤黃金，買通了楚軍的許多將領，通過謠言讓項羽懷疑大將鍾離昧通敵，進而懷疑范增，陳平又通過小事添油加醋，讓項羽徹底不再相信范增，范增氣死。如此一來，直接斷了項羽左膀右臂，為後來劉邦出逃滎陽做好了鋪墊。

用滎陽城一部分的錢財：四萬兩黃金。買來了地方將領的眾口鑠金，換來了劉邦的一線生機，怎麼算都是很合算的生意。可以說，陳平真的是舉重若輕，對人性觀察的非常深入。

白登之圍：用金錢珠寶賄賂單于的女人

西漢建立之後，匈奴南下入侵邊疆，劉邦親率32萬大軍出擊，結果因為冒進，被匈奴40萬大軍圍困在白登山，天寒地凍，糧食告急，而四周的路口都被匈奴軍隊堵住，劉邦不得出。情況幾乎和楚漢相爭的時候差不多，劉邦又被圍困了，只不過這次換了圍的人。

▲匈奴軍隊畫像

被圍困七天後，不知道陳平是不是頭腦風暴，居然想到了行賄！行賄的對象是跟著匈奴冒頓單于隨軍而來的摯愛妻子，她的名字叫閼氏。不得不說陳平真的是觀察入微，他通過冒頓單于每天和妻子閼氏散步說笑形影不離，看出兩人很是恩愛。所以就向劉邦提出了這個「枕邊風」的計劃。《史記》記載：

高帝乃使使間厚遺閼氏，閼氏乃謂冒頓曰：「兩主不相困。今得漢地，而單于終非能居之也。且漢王亦有神，單于察之。」冒頓與韓王信之將王黃、趙利期，而黃、利兵又不來，疑其與漢有謀，亦取閼氏之言，乃解圍之一角。

▲單于和他的妻子

漢高祖劉邦依計行事，送了許多珠寶首飾給單于的妻子閼氏，閼氏於是吹了這個價值連城的枕邊風，說兩虎相爭必有一傷。正好單于又在等待漢朝叛軍會師一起攻打白登，但是漢朝叛軍遲遲不來，這讓冒頓單于生了疑心，最後採納了妻子的建議撤退了。

▲西漢金餅

最終劉邦再次脫困，陳平這次又用錢解決了大問題，直接救了劉邦一命，而且花費的代價不大，可以說這錢實在花得太值了。

總結

陳平這花錢的水平真可以稱得上是一個投資大師，真正做到了用錢買來了帝王的命，而且還是兩次！更用錢買出了自己的前途！花錢花到這個地步，如此高的回報率，稱之為「藝術」，毫不過分。

雖然老話說得好，「能用錢擺平的事兒都不是事兒」，但錢只有用在刀刃上，才能發揮它的魔力，陳平能做到力挽狂瀾，不僅僅在於他敢於花錢，同時，也在於他花了心思去洞察人心，這才是根本。
