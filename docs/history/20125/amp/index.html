為什麼40幾個倭寇能對戰10幾萬明軍，還砍死3千多軍民？

天天要聞
3 天 ago

如果熟悉明史的朋友，肯定會知道有這麼一段歷史。嘉靖三十四年，40幾個倭寇從浙江的平湖上岸搶劫，向著杭州方向逼近，搶掠之後逃向淳安。按說在那時候，倭寇猖獗，經常上岸搶劫，這事也見怪不怪了，可是這群鬼子不知道是路痴還是腦袋短路了，既然一路搶到江蘇境內直到南京城下，明軍調動十幾萬駐軍圍捕，既然還被他們砍殺3千多軍民。這事情可就大了，南京可是之前明朝的京城，這就演變成了重大政治事件，朝野震驚。那麼為什麼40幾個倭寇為什麼會有如此恐怖的戰鬥力呢？

40幾個人，就敢如此放肆，搶劫都搶到了南京城下，在大明帝國十幾萬人的追捕之下，一個多月來，想搶就搶，噬無忌憚。這要是有4千人，那還不得搶到北京城下了。在這次看似平常搶劫，卻造成非同尋常影響的事件，很多人並不清楚，在它的背後，隱藏著讓人驚心動魄的真相。

這是一次非同尋常的搶掠，因為參與這次搶劫的四十多個倭寇並不是一般人，他們是浪人。

什麼是浪人呢？簡單明了就是日本流浪的武士，沒有了家園和土地的日本武士，武士在日本國內也是珍惜動物，數量十分稀少，哪怕是日本國內的諸侯王手底下的武士也有多少。

這種武士他們從小的時候就開始受到嚴格的訓練，不合格者將會被淘汰，他們其中還有些人練過日本的一種刀法絕技—「陰流」，這種刀術傳自日本的絕頂高手，「劍聖」上泉信綱。

而根據史料記載和數據證實，在這次搶劫中的40幾個浪人之中，他們全都是戰敗和在日本國內混不下去的日本武士。而這些人冒著被魚吃掉的生命危險，飄洋過海的來到大明國，為了生存，他們可是來玩命的，所以呢這些人的戰鬥力十分強大。另外一點就是他們所用的特殊武器——武士刀。

而這種武士刀基本上只有這些浪人才配帶，主要原因就是這種刀的製作成本太高，工藝十分複雜，保養又及其麻煩，所以也只有這些專門的武士才會配帶。武士刀製作的原料使用和配比非常講究，要用多種不同的鐵料和鋼料加之在一起再由工匠反覆多次捶打，所謂的千錘百鍊，製作這種刀所需時間周期很長，而且配方又嚴格保密，又只能純手工製作，所以數量其少。

能得到一把武士刀也別高興得太早，因為他比豪車保養起來還要耗錢，買得起，也不見得養的起，養得起也不一定你有那個耐心。這種刀因為是多種鐵和鋼夾雜一起煉製，其容易生鏽。所以必須天天得保養，時不時要拿去磨，而且還是特製的磨刀石，費用很高，每天都得上油，還必須要用植油。比上機油還麻煩。天天都要保養，比大爺還難服侍，除了那幫死心眼的日本武士，誰都不願意折騰這東西。非但如此，這幫孫子把刀看得比命還重，1945 年日本戰敗後，侵華日軍中許多有武士背景的軍官還曾向中國方面提出申請，希望帶走他們的家傳寶刀，表示如不允許，就切腹自盡。

而中國高層給到的答覆就是：切腹自便，把刀留下。

所以戰後就會有很多日本人帶著大批的鈔票到中國買刀的緣故。

而這40幾個倭寇用得正是這種特製的武士刀，他們武藝高強，又不要命，搶一票換一個地方，靈活多變，而且基本上都是在民住區域，大部隊不宜展開，小的部隊不是他們的對手，大部隊又不宜展開作戰，只能跟在他們的屁股後面追。使用前堵後追的戰術，偏偏這群鬼子是路盲，人家跑到哪裡都不知道，明軍又怎麼摸得透他們的行軍路線，幾經周折，既然跑到了南京城下，事情鬧大了，明軍將領才不昔血本的將他們殲滅。而這時統計下來，死在他們刀下的軍民既然多達到3000人以上。

其實造成這樣結果有以下幾點：

1、這群鬼子是武林高手，武藝高強，不要命，而且攜帶特殊武器—武士刀

2、當地駐軍久疏戰陣，他們主要出身於江浙一帶富饒地區，這些大爺當兵是混口飯吃，讓他們去拚命，想都別想，而且十幾萬人多為民兵，戰鬥力甚至還不如一些老百姓。

3、這些鬼子路況不熟，讓追兵摸不透行軍路線，多為搶一個地方就換一個地方，靈活多變，軍隊無法展開圍捕。

堂堂中華豈無才乎？中華武術博大精深，還有比倭寇更加厲害的，那就是少林僧人，他們才是真正的武林高手。京中軍都督府都督同知萬表找到少林寺的方丈，組織了上百名的和尚抗擊倭寇，自此之後，但凡遇到這些和尚的倭寇，沒有一次不敗的。

而大明帝國的軍隊也不是好欺負的，戚繼光領導著主要出自浙江義烏的戚家軍每逢倭寇就大勝，往往是己方傷亡幾個，幾十個，對方就被團滅幾千上萬人。
