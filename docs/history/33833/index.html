有四位帝王被活活餓死你知道是誰嗎

2019-12-24 853條評論 1,755次閱讀 207人點贊

在三百六十行中，沒有「帝王」這個最舒服的職業。這個職業不僅富有四海，而且擁有生殺予奪大權，錢隨便花，嬪妃隨便挑。到死了，兒子還可以接班，永遠世襲。

　　可是皇上也是高危職業，因為帝王太舒服了，人人都想做。江湖有雲，皇帝輪流做，明年到我家。誰都想做皇帝，所以一不小心，就會被趕下皇帝寶座，還有可能死的很慘。歷史上有幾位帝王就是因為種種原因，丟不但掉了皇位，還被活活餓死，

　　一、齊桓公。

　　說起齊桓公的大名，可以說是如雷貫耳，他早年因為重用了管仲，實行了全民皆兵的國防政策，並且大力度減輕農民負擔，很快把齊國治理的井井有條，國富民強。齊國因此崛起，成為五霸的老大。

　　不過在管仲死後，齊恆公開始迷失方向，被奸臣易牙、豎刁所迷惑，不理朝政，疏遠忠臣。最後連接班人問題都沒有搞定。致使五個兒子個個手握重兵，誰都想篡國奪權。齊恆公病重的時候，大家都不想管，都恨他沒有把王位給自己。而且大家忙著爭權奪利。而奸臣易牙和豎刁把宮門關起來，自己去享樂了，把齊恆公一個人丟在病床上不管不問。

　　齊恆公就這樣被活活餓死，死後停放了兩個多月，把蒼蠅都喂得飽飽的，蛆蟲都爬到了窗台上，直到新領導上任後，齊恆公才被安葬。

　　嚴格來說齊桓公應該不屬於帝王之列，那時候的諸侯國君還沒敢稱王，但就其江湖地位來說，作為春秋五霸的第一霸，不比帝王少排場和面子。

　　二、趙武靈王。

　　此君也不是泛泛之輩，少年有為，15歲就登上了大位。趙武靈王登基的時候內外交困，趙國周圍強敵環伺，政權岌岌可危。

　　趙武靈王上任後重用賢臣，搞改革開放，向少數民族學習，建立起一支非常有戰鬥力的隊伍。趙武靈王的做法可以說是一個創舉，史上稱之為胡服騎射。

　　趙武靈王勵精圖治、兢兢業業，很快就把趙國打造成了一個中原地區首屈一指的強國，但此後趙武靈王開始貪圖享樂，陶醉女色，把皇位交給了小兒子趙何，但同時給了長子趙章很大權力，來撫慰趙章因為沒有繼承皇位而受傷的心靈。

　　趙武靈王的舉棋不定，讓兩個兒子開始爭鬥，自己也捲入其中，最後自己跟長子被小兒子趙何打敗，被包圍在自己宮殿里，整整圍困了九十多天。趙武靈王就是天大的能耐也施展不開，也像齊恆公一樣當了餓死鬼。

　　而他的小兒子趙何也就是趙惠文王，在確定老爸被餓死後，才叫人打開宮門，象徵性地哭了幾聲，把他安葬了。

　　三、齊廢王田建。

　　齊王建是齊國的最後一任國君，戰國時期齊國最後一位國君，也被稱為齊廢王。

　　齊王建即位的時候，形勢已經非常危急，秦王朝就像一頭惡狼，到處張牙舞爪，各國先後被秦國征服。

　　但齊王建被秦國的花言巧語迷惑，得過且過，沒有居安思危，陶醉在一片國泰民安的假象中。

　　不久，秦國收拾完了其它刺頭，終於把魔爪伸向齊國，齊王建毫無準備，被秦國的大軍嚇破了膽，不知所措，急得像熱鍋上的螞蟻，滿朝文武也面面相覷，嚇得渾身篩糠，大家的意志也早已崩潰。

　　這時候秦國使者來了，告訴齊王建只要投降，就可以免去一死，而且還能享受優待。齊王建一看大勢已去，心想乾脆就投降吧，於是就打著白旗出城投降。

　　秦王對齊王建不放心，但又不能殺俘虜，於是就採取了部下的建議，把齊王建囚禁起來，不給吃喝，把齊王建活活餓死。

　　四、梁武帝蕭衍。

　　梁武帝蕭衍出南北朝時期南梁的開國皇帝，也是個雄才大略的政治家。

　　蕭衍在早年的時候，為了治理國家殫精竭慮，嘔心瀝血。還專門設立意見箱讓百姓為國家建設獻計獻策。蕭衍還非常關心弱勢群體，多次減輕賦稅，推出不少政策救濟孤寡老人，幫助農民脫貧致富。

　　蕭衍自己以身作則，勤儉節約，剎住大吃大喝風，停建樓堂館所。

　　蕭衍在抓物質文明的同時，還不忘精神文明建設，自己帶頭皈依佛教，號召全國人民信奉佛教，一心向善。用佛經來指導、規範自己的行為。

　　在梁武帝的精心治理下，國家繁榮，人民富足，安定團結、路不拾遺。

　　但梁武帝跟所有偉人一樣，到晚年都避免不了會犯錯。梁武帝到了晚年變得剛愎自用，開始獨斷專行，聽不進不同意見，接納了腦後長有反骨的野心家侯景。

　　這侯景跟呂布一樣是個反覆無常的小人，早年跟著北齊的高歡當馬仔，後來又投奔了西魏的創始人宇文泰，最後混不下去了就投靠了梁武帝。

　　梁武帝的手下都知道侯景不是善類，竭力勸阻，但梁武帝年邁昏庸，一意孤行，執意收留了侯景。

　　後來侯景果然反叛，把梁武帝關了起來，不給吃喝。可嘆梁武帝蕭衍英明一世，臨老了卻被囚禁餓死，叫人唏噓。

　　這四位皇帝的遭遇告訴人們，再英明的人都有犯錯的時候。所以不但要遠離小人，聽得進不同聲音，而且要妥善解決接班人問題，儘早實現權力平穩過渡。
