一位外族貢女，嫁給中國皇帝後，竟要求滅了自己的祖國

2019-12-23 900條評論 3,149次閱讀 1348人點贊

皇后作為一國之母，其可以說是皇帝後宮的主宰，在我國歷史上，曾經出現過許多皇后，而其中僅有一人來自於國外，她就是元順帝妥歡帖睦爾的皇后奇承娘。

奇皇后並非中國人，而是高麗朝鮮人。蒙古帝國崛起的時候，奇皇后的母國高麗曾被元朝兼并。高麗國王為了自己國家的安全，遂臣服於蒙古帝國，並向其每年供奉宮女，奇承娘正是作為貢女的身份才來到了元朝。

奇承娘是高麗有名的美女，其在國內時，曾被高麗許多富家子弟所追求，不過，由於元朝的鐵騎不斷踐踏著高麗的國土，高麗國王遂打算借用奇承娘來消除元朝的野心。元文宗在位時，年紀輕輕的奇承娘被高麗國王作為貢女送給了元庭，當時，貢女在元朝宮女中的地位很低，其一度也被作為低等人來看待。不過，奇承娘卻有自己的晉陞方式，她依靠著自己的美貌和計謀，終於獲取了元順帝妥歡帖木兒的歡心。

元順帝妥歡帖木兒是元朝最後一任皇帝，此人在位時，元朝已經走向了衰亡。順帝好色，奇承娘則善於逢迎，這樣一來二去，順帝遂與奇承娘發生了曖昧。

奇承娘並非是元順帝第一任皇后，在此之前，他還有兩任皇后。順帝的第一任皇后是答納失里，答納失里為元朝權臣燕帖木兒的女兒，文宗、順宗在位時，燕帖木兒也是長期掌理國政，雖然順帝有心扶立奇承娘為皇后，但他攝於燕帖木兒的餘威，也只好暫時放棄廢后的想法。

公元1333年，權臣燕帖木兒因縱慾過度猝死，順帝大喜，開始打擊燕帖木兒的勢力。順帝長期受制於燕帖木兒，所以他對燕帖木兒一派也是非常痛恨，這一年，燕帖木兒一黨紛紛落馬，而那位可憐的皇后答納失里也被元順帝廢黜賜死。

燕帖木兒雖然死了，但元庭內部守舊派勢力依然強大，他們也時刻影響著元順帝的決策。當元順帝妥歡帖木兒提出扶立奇承娘為皇后時，其就立刻遭到了朝廷新貴伯顏的反對。伯顏是元順帝誅除燕帖木兒一黨的功臣，順帝攝於伯顏的權威，只好暫立蒙古貴族之女伯顏忽都為皇后。

奇承娘的皇后美夢再次落空，她自然對元庭新貴痛恨不已，為了誅除伯顏，她便不斷在順帝耳邊煽風點火。後來，奇承娘發現了元庭另一名臣脫脫的野心，她遂利用脫脫的勢力不斷向伯顏發難。終於，伯顏便被擠出了朝政，而奇承娘也成功坐上了皇后的寶座。

美貌與智慧，是奇承娘上位的方式，不過，這位奇皇后在關鍵時候卻並沒有對伯顏處以極刑，為了表現自己的大度，獲取人心，奇承娘遂說服順帝將伯顏貶謫。

一人得道，雞犬升天，奇皇后上位成功，其家族也無限風光，在奇承娘繼為皇后的同時，其家族也在高麗崛起。奇氏宗族在高麗蠢蠢欲動，竟野心勃勃想要奪取高麗國王的王位，高麗王震怒，遂將奇氏一門滿門抄斬。奇皇后對於高麗國王這一做法，自然是深惡痛疾，為了給家族報仇，奇皇后遂慫恿元順帝發兵攻打高麗。

奇皇后不計後果攻打自己的母國，這一做法也引起了高麗人民的共憤，就在蒙古大軍進入高麗國疆域的時候，高麗國民也幾乎是全民皆兵，蒙古大軍被高麗軍隊擊敗，奇皇后復仇的願望也終究破滅。

妖后奇皇后最後的結局也很悲涼，元朝滅亡後，奇皇后隨從元順帝北逃荒漠，不久，她就孤獨地死在了北漠戈壁。
