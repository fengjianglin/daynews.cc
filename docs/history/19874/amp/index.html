明朝最特殊的皇族：貴為皇帝後裔，卻沒有名字，世代為囚

天天要聞
2019-12-17

明朝時期，有一群特殊人群，他們既不用從事生產勞動，也不需要繳納賦稅，卻過著衣食無憂的生活。因為他們是開國皇帝朱元璋的後裔，也就是所謂的宗室。朱元璋稱帝後，就制定了嚴格的封藩制度，他的子孫「世世皆食歲祿，不授職任事」。

然而，在這些天潢貴胄中，有一批人命運卻很悲慘，他們也是朱元璋的後裔，卻有姓沒名，而且世世代代都是囚徒，受到官府的嚴格監控，甚至連婚姻嫁娶，也受到一定程度的限制。那麼他們到底是誰呢？

​這些人在明朝有一個特殊的稱呼，就是「庶人」。「庶人」原意就是沒有官爵的普通百姓，但在明朝卻不一樣，他們是朱元璋的後裔，本來是有爵位的，可是由於各種原因被廢為「庶人」，不過他們並沒有自由，而是被囚禁到鳳陽等地，受到官府的監管。

按照明朝的規定，宗室成員生下兒子，不能自己隨意取名，要通過複雜的程序，向朝廷申請，由皇帝親自賜名。賜名之後，就會被寫入玉牒，才能享受各種宗室所享有權力。由於「庶人」是罪宗，皇帝不可能賜名，所以他們有姓氏，卻沒有名字。

​按照成為被廢為「庶人」的原因，這些人大體上可以分為三類。第一類，因政治原因被廢，典型代表就是「建庶人」、「吳庶人」。靖難之役後，皇位由朱標一脈轉到朱棣一脈，朱棣在成為皇帝後，將建文帝的弟弟朱允熥、兒子朱文圭囚禁到到鳳陽高牆。

朱允熥原先的封爵是吳王，所以他本人以及他的子孫後裔被統稱為「吳庶人」。建文帝的兒子朱文圭則被稱為「建庶人」。這些人被關押了五十五年，直到天順年間，明英宗才下詔將兩家人釋放「聽於軍民之家，自擇婚配」。不過，「吳庶人」、「建庶人」的行動依然受到官府的嚴密監控。

​第二類，因圖謀不軌被廢，典型人物是「齊庶人」、「谷庶人」。朱榑是朱元璋的第七子，被封為齊王；谷王朱橞是朱元璋的第十九子，當年就是他打開金川門，迎接朱棣進入南京的。兩人在永樂年間，因被告發圖謀不軌，被廢為庶人。

宣德三年，朱榑、朱橞暴死於南京。雖然朱橞絕後，但朱榑子孫後代眾多，到明朝後期，達到上千人之多。「齊庶人」（都沒名字）人數眾多，他們在南京為非作歹，欺男霸女，名聲很差，「齊宗人多凶狡」，連官府都不敢招惹。

​第三類，因犯罪被廢，這類人數最多。明朝宗室仗著自己是皇帝的後裔，經常做出許多不法行為，雖然朝廷對他們比較縱容，但也會做出懲罰。比如之前說的伊王朱典楧，公開霸佔官府土地，強搶民女多達上千人，引起了洛陽全城的恐慌。嘉靖帝三番五次做出指示，但朱典楧不予理睬。

最後，嘉靖帝削去朱典楧的爵位，將其廢為庶人，關押到了開封。還比如唐王朱聿鍵，他是朱元璋第二十三子唐王朱桱的後裔，在崇禎年間，因私自率軍離開封地，前去勤王，被崇禎帝廢掉。不過他因禍得福，後來成為南明的皇帝，即隆武帝。

總而言之，明朝時期，被廢為庶人的「罪宗」始終是一批比較特殊的人群，他們之中既有非常可憐的，也有罪有應得的。
