一個臭名昭著的海盜，竟被女王封為海軍上將，憑什麼？

天天要聞
2019-12-04

摘要

提起日不落帝國，就不得不聯想到英國了，作為當時的歐洲霸主，英國的崛起，離不開一名叫做弗朗西斯·德雷克的人。

提起日不落帝國，就不得不聯想到英國了，作為當時的歐洲霸主，英國的崛起，離不開一名叫做弗朗西斯·德雷克的人。他本來是一個臭名昭著的海盜頭子，後來卻被當時的英國人稱為「民族英雄」，更是搖身一變，成為了當時日不落帝國的海軍大將，這個過程可以說是非常戲劇性的，弗朗西斯·德雷克是如何完成從海盜到海軍大將的蛻變？

16世紀的英國海盜

西班牙人的欺騙，埋下了仇恨

弗朗西斯·德雷克，於1540年誕生於英國德文郡一個貧苦農民的家中，因為英國是一個島國，四面臨海。所以出海漁民非常多，由於家庭生活拮据，德雷克從小就到海上謀生，還未成年就在一條往來於北海各港口的近海小船上當學徒，老船長死後，德雷克成為船主。

德雷克開始海盜生涯是與其表兄約翰霍金斯分不開的，霍金斯是英國最早從事奴隸貿易的海盜，他在幾內亞一帶抓獲黑人，然後送到拉丁美洲高價出售，獲利甚豐。德雷克便賣掉了自己的小船，與霍金斯一起從事奴隸貿易和掠奪西班牙在加勒比海地區的殖民地。

1568年，德雷克和他的表兄約翰·霍金斯帶領五艘販奴船前往墨西哥，由於受到風暴襲擊而向西班牙港口需求援助。但是西班牙人對他們的欺騙險些讓他丟了性命。從此後他發誓在有生之年一定要向西班牙復仇。

歷史上的德雷克

英國的危機，讓德雷克得到了伊麗莎白一世的支持

在16世紀的歐洲，西班牙是當仁不讓的霸主，而英國在歐洲政治版圖上就有些無足輕重。並且從亨利八世起，都鐸王朝就沉迷於奢侈品中。各種奢侈品，如華麗的服裝、珠寶、食物或者娛樂都是亨利權力中的一部分。在伊麗莎白時期，它們又成了她統治的主要象徵，宮廷和倫敦就成了財富和權力競爭的中心。

而英國當時國力空虛，根本沒有多餘的錢財去擴充軍隊，迫於西班牙和鄰國的壓力之下，伊麗莎白一世做了一個讓人意外的選擇——海盜！

這個人就是弗朗西斯·德雷克！

作為英國女王，伊麗莎白一世公開向海盜們提供船隻、資金和武器；給海盜船長發「私掠許可證」，英國皇家海盜因此可以公開奉旨打劫。

由於伊麗莎白一世的支持，十六世紀六七十年代，伊麗莎白甚至連自己的船隻都交給了德雷克。

比較古老的海盜船樣式

龐大的財富，成就了英國海上霸主的基礎

德雷克最著名的一次劫掠當屬西班牙寶船「卡卡弗戈號」。1577年，德雷克海盜船隊再次出擊，而他此次出航得到了伊麗莎白女王的資助，西班牙一艘滿載金銀財寶的運輸船卡卡弗戈號正從秘魯駛向巴拿馬城，德雷克於1579年3月3日將遠道而來的卡卡弗戈號候個正著。經過短暫的炮戰以後，卡卡弗戈號投降，德雷克擄獲黃金80磅，白銀20噸，銀幣13箱，以及數箱珍珠寶石。德雷克和同夥花了整整四天才把所有的戰利品裝上船。

德雷克打劫卡卡弗戈號得手以後繼續西行，利用繳獲的西班牙海圖穿過太平洋和印度洋，一年多以後才回到英國，成為第一個環繞地球航行的英國人。1580 年9月26日，德雷克船隊滿載財寶駛進普利茅斯港，受到隆重歡迎。伊麗莎白女王登上德雷克的旗艦「金牝號」，在甲板上授予德雷克騎士爵位，並任命他為普利茅斯市長。伊麗莎白女王對德雷克的表彰理所應當，因為她在德雷克身上每1英鎊的投資，獲得了47 英鎊的回報。

從1585年到1604年，英國每年至少有一百多艘武裝商船出海，專門在大西洋和加勒比海劫掠西班牙運輸船隊，每年的擄獲平均可達20萬英鎊。

西班牙人有一句名言：「錢財搞活戰爭」。有了錢，英國就有了底氣。

伊麗莎白又聽從建議，以快艦的形式來武裝海軍。

英國有了大量的錢財來武裝自己

戰爭開端，西班牙無敵艦隊大敗

英國海盜的劫掠行徑讓西班牙大傷腦筋，西班牙駐英國大使向伊麗莎白女王提出無數次抗議，但都無濟於事。但德雷克率領艦隊到加勒比海大肆劫掠，讓西班牙找到了入侵的借口，而英國的皇家艦隊此時只有區區23艘戰艦，而且長久以來一直奉行近海防禦戰略，從來沒有遠離英國海岸作戰。

德雷克堅決反對這種消極防禦的戰略，他豪氣干雲地宣稱，進攻是最有效的防守，應該主動出擊。於1587年4月19日率領他的私掠艦隊突襲西班牙卡迪茲港。德雷克艦隊在卡迪茲港激戰了12個小時，擊沉西班牙艦船24艘，並焚燒了大批軍用物資。

歷史是上的海戰

自此，德雷克成為了女王的親信，因為他證明了西班牙的無敵艦隊，並不是無敵的。

1588年5月30日，西班牙再次出擊，派出134艘艦船、8000名船員和水手、2000多搖槳奴隸、2.1萬名步兵，浩浩蕩蕩從里斯本揚帆出航，踏上了遠征英國的航程。

而英國沒有像樣的艦隊，所擁有的就是「皇家海盜」100多艘戰艦和9000多名海盜水手。8月6日，無敵艦隊停泊在法國加萊灣。德雷克再次主動出擊，他們把6艘舊船裝滿易燃物品，船身塗滿柏油，火攻西班牙艦隊。無敵艦隊立即陷入一片混亂，各條戰艦爭相斷纜開航，大火和相互的撞擊導致許多戰艦沉沒。

8月8日，兩軍在加萊東北海上進行會戰。西班牙戰艦雖然高大威猛，但是火炮射程沒有英國海盜船遠，並且英國人的射擊精度較高。「無敵戰艦」基本上就成了「皇家海盜」的活靶子。

此戰之後，「無敵艦隊」僅剩43艘戰艦返回西班牙，而英國艦隊則損失輕微。

歷史上的加萊海戰

加萊海戰的勝利，使英國從西班牙手中奪得了海上霸權。英國的觸手逐漸從非洲、美洲伸到了亞洲，在印度成立了東印度公司。

從海盜到海軍大將的最終蛻變。

加萊海戰之後，歐洲大西洋沿岸形成了西班牙、英國、荷蘭三足鼎立的局面。

1639年，荷蘭與西班牙在英吉利海峽唐斯灣爆發了一次大的海戰，史稱唐斯海戰。荷蘭人出人意料地再次創造一個「加萊海戰」，從此之後西班牙再也沒有能力與英國和荷蘭並世稱雄。

一山不容二虎，英國與荷蘭的衝突是不可避免的。

從1652年開始，英國為了打敗日益發展的商業競爭對手荷蘭，并力求保住開始建立的海上優勢和爭奪殖民地，三次挑起對荷蘭的戰爭，前後達20多年，並最終獲勝。此後，荷蘭喪失海上霸主地位。

繼葡萄牙、西班牙、荷蘭之後，英國成為第四個海洋霸主。

現今海軍軍艦

自此，德雷克完成了一個海盜到海軍大將的蛻變，縱觀德雷克的一生波瀾起伏，也是一部傳奇了。在英國，他是名留千古的英雄，而在西班牙卻是惡名昭著的海盜。

所以至今來說，德雷克的定位到非常具有爭議性，反推一下，只要我們認為他是英雄他就是英雄，認為他是海盜，那他就是那個令人聞風喪膽的海盜大魔王。
