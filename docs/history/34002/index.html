敢叫劉備“賢弟”的人，除了曹操，沒一個善終

2019-12-23 2,440條評論 4,491次閱讀 3441人點贊

​【本文由賨人客獨家原創，未經允許，不得轉載，抄襲必究！】

三國裡面有哪些叫劉備「賢弟」的？公孫瓚、陶謙、呂布、曹操、袁紹、劉表、劉璋。

這個「賢弟」可不好叫，因為，這些人除了曹操，其餘沒一個善終。這話從何說起呢？請往下看：

1、公孫瓚

公孫瓚是劉備從小的同學，劉備在官場上混不下去了，就跑去投靠當時勢力還很強的公孫瓚。公孫瓚親切地稱劉備為「賢弟」，可是這個賢弟後來裹著人馬跑去投靠了徐州陶謙，還間接帶走了趙雲。公孫瓚的結局也很悲慘，家破人亡，被袁紹消滅。

2、陶謙

陶謙找劉備來幫他守徐州，也親切地稱他為「賢弟」，可這個賢弟裹走了徐州大戶糜家（糜竺）和陳家（陳珪），在陶謙死後成功地擠走了陶謙的兒子，成為徐州新的老大。

3、呂布

呂布借宿徐州的時候，剛開始也稱劉備為「賢弟」，他以為這個賢弟好欺負，喧賓奪主奪了人家的地盤，又讓人家來給自己看家門。

可這個賢弟並不好欺，他引來曹操直接將呂布給滅了，我得不到你也別想得到，大不了來個魚死網破。於是，劉備又成了曹操的賢弟。

4、曹操

曹操對這個賢弟可不錯，好吃好喝地招待著，還說天下英雄就他們兩個。可惜這個賢弟劇情再演，裹著他的人馬又溜去徐州了，還跟他對著干。還好曹操命大，不然結局和前3位估計不相上下。

5、袁紹

袁紹實力算強大的了，在董卓死後可謂首屈一指，可是自從認劉備做賢弟以後，先後損失了顏良和文丑兩員大將，其中一個還是劉備的賢弟關羽乾的。

袁紹正想拿劉備問罪的時候，這個賢弟又溜了。溜去哪兒了？劉表那裡。

6、劉表

劉表也把這個賢弟當個寶，讓他經常出入荊州的一些重要場合，劉備因此在荊州上層圈兒混了個臉熟，這為劉備後期的人氣聚集起到了重要作用。

劉表一死，荊州就不再是他們家的了，要麼歸曹操，要麼歸劉備，最終，各得一半。

7、劉璋

劉璋邀請劉備幫他打張魯，也把劉備看作好賢弟，可是這個好賢弟翻起臉來比翻書還快，前腳還在打張魯，後腳就開始攻打成都了。劉璋的地盤最終也全部落入劉備手中。

看來，劉備這個賢弟真不好叫啊，除非，你能鎮得住他，不然，千萬別叫。以上幾位就是例子。

再來說孫權，由於年齡實在懸殊太大，孫權無法把劉備叫賢弟，但他卻讓劉備做了自己的小舅子（孫劉聯姻），結果孫權還是搭了半個荊州。

總結：劉備真非池中物也！
