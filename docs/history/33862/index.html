皇帝去官員家裡討水喝，想不到官員放3惡狗迎客，乾隆：給他陞官

2019-12-24 2,232條評論 3,465次閱讀 2239人點贊

有句話叫做「在位時門庭若市，退位後門可羅雀」，說的是官員們在位的時候，由於手中有權，所以各種人都前去他家裡作客，而且都是非常熱情，帶著各種禮品的那種，只是對於官員來說，對於見到這種人是心懷戒懼之心的，因為他們給自己送了禮以後，就往往要求動用自己手中的權。

清朝乾隆時期的和珅，原本也是一個貧寒子弟出身，在成為官員的時候也一心想要當一個清官，但就是因為接待這種人太多了，最終還是沒有忍住誘惑，收了一次禮就有第二次乃至於無數次，到最後竟然自己主動去索禮，讓自己成為世界首富的同時，也讓自己成為了頭號貪官。

與之相比，和珅在官場上的老前輩鈕祜祿·訥親就要聰明很多。訥親是滿清開國元勛額亦都的後人，其家中一門顯貴，祖父是康熙是輔政大臣，父親是正一品的領侍衛內大臣，而訥親自己在雍正五年（1727年）的時候，就已經因為祖宗的緣故，承襲為二等公，又被升為御前大臣。

等到雍正晚期的時候，訥親已經成為了都統和領侍衛內大臣，又進入了軍機處，成為朝野公認的大紅人，幾乎所有人都認定他以後的仕途，還會越來越寬廣，直到位極人臣。而且事實也確實如此，在乾隆登基以後，訥親這一班受到雍正重用的「老臣」，也受到了乾隆皇帝的加倍重用。

訥親那原本看似已經到頂的仕途，在乾隆的大力提拔下，硬是頂破了天花板，他首先成為了兵部尚書兼議政大臣，又在乾隆十年成為協辦大學士、保和殿大學士、軍機處領班大臣，這個時候的訥親可謂是真正的權傾朝野。

很多「有油水」的重要職位，都由訥親一身兼任，無論是幹部人事任免這樣事關人們前途的工作，還是財務預算、工程建設、行政審批、物資採購這樣事關人們錢途的工作，都由訥親個人直接決定，所以每天都有各種各樣的人，以各種各樣的名義前去訥親家中拜訪送禮，他家門前可謂是真正的門庭若市。

可訥親本人是一個極為清廉的大臣，他從小就不缺錢，對於這些「客人們」所送的各種禮物也不感興趣，但哪怕是不收禮，單單是應付這些上門有所求的人就已經夠煩的了，為了能夠盡量減少麻煩，同時也是為了能夠讓自己減少誘惑，從而減少自己被腐蝕的風險。

訥親在家中養了3條惡狗，好將那一些前來登門的人都給嚇走，而且這個方法也確實是很有效，自從養狗以後，來家裡的人確實少了很多。

不過有一次乾隆皇帝微服私訪，信步走到訥親家門前，想要進去他家裡討口水喝的時候，卻意外發現首先迎客的竟是3條惡狗，當時也是被嚇了一大跳，但當他知曉其中的內情以後，卻對訥親表示大大的讚賞，隨後又升了他的官。
