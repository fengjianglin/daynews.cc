日本3大正宮娘娘的結婚照，都儀態萬千，只有她後來居上母儀天下

天天要聞
3 天 ago

日本皇室現在共有3位美麗的妃子，一個是美智子皇后，她是明仁天皇的妻子。一個是德仁皇太子的妃子雅子。另外一個是二皇子秋筱宮文仁親王的紀子妃，3個人都來自民間。她們的長相都非常美麗，都是拔尖的美人，又有氣質，所以受到了日本媒體的追捧，因為她們都是亞洲人的面相，與凱特梅根那些歐洲皇室相比，又有了不一樣的氣質。

我們中國已經取消皇室100多年了，如果對皇室還有那麼一丟丟浪漫情結的人，就可以拿這些黃皮膚的人腦補一下假設我們還有的場景了。

3大正宮娘娘的結婚照，都是儀態萬千，我們來圍觀一下誰更具有母儀風範呢？

雅子和紀子的婆婆美智子皇后，出身商賈大家，是個富商家的大小姐，受過良好的教育。在打網球的時候結識了明仁天皇，明仁天皇對她非常痴情，展開了2年多的瘋狂追求。日本皇室八卦雜誌曾曝出，當年的美智子家並不同意這樁婚事，為此接到了很多皇室的恐嚇電話，導致他們家非常煩惱。

母親富美子更是堅決不同意自己女兒嫁入皇家，生怕將來的女兒會受氣，結果她媽媽預料的非常對。美智子一直受婆婆良子皇后59年的氣！她嫁入皇宮的時候，結婚照上的她的臉龐還是非常圓潤，很有母儀範的，後來一年之內因受氣瘦了20斤，被良子皇后欺負成了啞巴，非常的可憐。

第二個嫁入日本皇室的是美智子皇后的二兒子文仁家的紀子妃。紀子妃的父親是大學教授，她和婆婆一樣也是平民出身，她的笑容非常具有感染力。

因為她是心理學專業畢業，所以很善於察言觀色，在皇室里混得如魚得水，還因為她生了兒子給皇室立了大功，現在也是母憑子貴，她後來居上頂替了自己的皇嫂雅子，將來她是日本皇室的太后。

最後一個嫁入皇室的是德仁太子的雅子妃，她是哈佛畢業，是皇室中學歷最高的人。當年她是個女外交官，被霸道總裁德仁皇太子一見鍾情，德仁的性情和他爹明仁一模一樣，苦苦追求雅子7年之久，兩人到了30多歲才結婚。

因為大齡青年生孩子比較費勁，一直生不齣兒子來，只通過試管的幫助生出了一個獨女愛子來，德仁的皇位將來是要傳給文仁親王家的，雅子明年當皇后，因為沒生齣兒子，未來她不是太后。這些年的雅子從外交官淪為生育工具，一直在抑鬱中度過，弟媳紀子幫忙生出繼承人，才讓她得以喘息。

要說這些妃子里最有福的就數紀子了，紀子妃因為年輕時候很親和甜美，被製成了日本動漫，在日本坊間廣為流傳。他老公秋筱宮文仁親王的「秋筱宮」來自日本秋筱宮寺，其中寺內的伎藝天立像酷似紀子。紀子妃不僅成為親切的動漫人物，而且酷似佛像，想想人還真得貌相啊，世事無常，皇位本來沒有紀子家什麼事的，卻因為自己40多歲拚死生出了皇子，其命運得到了轉機。可能冥冥中的佛像都在保佑她呢。

其實，日本皇室的皇位幾乎沒有任何的實權，只是國家的一個代表，放眼全球的皇室也只有日本國還如此執著於男丁。他們把一個妃子逼成抑鬱，還讓一個妃子冒死產子，整天吆喝男女平等，結果代表國家形象的皇室機構實行起來也是毫不含糊，上行下效，難怪日本職業歧視越來越嚴重，從這就能找到根源。

誰也不能保證下面的王子能繼續生下兒子啊，哪家的女孩敢接這樣的大單呢？其實最後還是要改變觀念，實行男女平等，才是從根本上解決這一問題。只是觀念的更新任重道遠。
