古代的大官若犯了死罪，往往不殺他而是讓他自盡，這是為什麼呢？

2019-12-10 1,333條評論 3,721次閱讀 2636人點贊

在中國，「面子」是根植於文化的社會心理建構，面子在人際交往中形成與表現，面子具有情境性和可變性，面子是一個人自尊與尊嚴的體現，面子是一個人的自我心像，面子是重道義輕功利倫理情趣的表徵… …在日常生活中，我們也常常聽到別人對一些性格倔強的人評價為「死要面子」。

而這種情況，在親戚之間尤為常見，但是，你可知道，在古代「死要面子」卻有別的含義。古代人所說的「死要面子」不同於現在的臉兒薄，指的是：在一些官員犯下死罪之後，雖然無法饒恕，可面子上的功夫，卻不能少了半分。即使是臨死，他們也希望能夠保全自己的尊嚴，更何況，這面子還是皇上給的。

然而，這樣的傳統並非一開始就有，說起來，這一切還要從漢朝初期說起。

漢高祖劉邦斬白蛇起義，在諸位能臣輔佐之下一統江山，可是，當他登上皇位之後，卻一直心中不安，生怕自己死後異姓王起來造反，到時候，自己的兒子們玩不過他們，豈不是要步入秦王朝不過二世的後塵？因此，劉邦索性心一橫，把當時一起打天下的功臣們殘酷殺害。

後來，到了呂后執政，其殘暴程度更尤勝之，一度製造了著名的「人彘」事件。她把劉邦的愛妃戚夫人砍掉雙手雙腳，挖去眼睛，割掉鼻子，弄成一個人棍，然後，再將其放到大缸里，任其在痛苦和絕望中等待死亡。以至於，呂后的兒子惠帝見到這場面後，感慨其母親的狠毒，連皇帝都不想做了。

這樣的情況，一直到漢文帝時代，才有了一些轉變，並且，他定下了一個不成文的規矩：「凡是犯了死罪的大臣，一律不得嚴刑逼供，要讓他們自行了斷，即使是死，也要讓他們死得有尊嚴。」這樣的傳統，到後代一直被沿用，很少有人破壞這一規矩。

其實，說起來，真正起到這麼大作用的，還應該歸功於一位漢朝的元老級功勛人物：絳侯周勃。

周勃本是劉邦的武將，跟隨劉邦一路腥風血雨，終於是打下了江山。在呂后執政之時，他擔任了宰相之職，呂后死後，他更是掌握兵權，與陳平一起平定了「諸呂之亂」。之後，將呂后封的呂姓王全部誅殺，並決策迎立代王劉恆做皇帝，即：大名鼎鼎的漢文帝。

漢文帝登基之後，顧念周勃有功，便任其為第一宰相。而周勃一時風光無限，連走路都趾高氣揚，漢文帝每次都要起身相送，還要目送其離去，可以說，是給了他極高的殊榮。但是，俗話也說過：「全則必缺，極則必反。」周勃顯然也意識到了這一點。

正巧，有一天身邊有人向他進言，說：「您乃開國功臣，而且，手握重兵，不僅誅殺呂后，還新立國君。可如今你位極人臣，表面上風光無限，實際上卻正處在風口浪尖，時刻都可能招來殺生之禍。」這樣的說法，與周勃的想法當真是不謀而合，於是，周勃便上表請辭，漢武帝也很痛快地答應了。

然而，事情到這裡還未結束，周勃雖然放棄了權力，但是，考慮到他的影響力，漢文帝仍然對他不放心。當時，不少諸侯按照規定，應該離開京城到封地去，可是，他們大多貪慕京城繁華，不願離開權力中心。漢文帝便趁此向周勃提議：「您是開國功勛，德高望重，就請帶個好頭吧。」

周勃到了封地之後，漢文帝總算是鬆了一口氣，可是，周勃心裡卻不踏實。

他心想：「漢文帝這麼幾次三番的折騰自己，心機之深實在超出了自己的預料，恐怕自己仍不能使漢文帝放心。」於是，他越想越怕，最後，竟是得了心病。以至於，每次有地方官來看望他，他就以為是漢文帝派人來取他性命，每次接見官員，總是安排家丁全副武裝，以防不測。

這真是，怕什麼來什麼，有一天，周勃真讓人給告了，而罪名正是謀反！當時，漢文帝心中對周勃已經很放心了，並對周勃的實力也很清楚，深知他早已無力謀反。可是，既然有人告了，何不趁機再做做文章，徹底了卻自己的一塊心病呢？

這麼一來，周勃在牢獄中的日子倒是不好過了，整天嚴刑拷打，把他折騰得完全變了一副模樣。周勃乃是武將出生，不善與人交際，再加上，他本身心裡就擔心是漢文帝故意而為之，於是，整日憂心忡忡。雖然，自知清白，卻是百口莫辯，眼看著再這麼折磨下去，自己恐怕就要頂不住，屈打成招了。

正在這時，周勃的兒子給審訊的獄卒送去了一千兩金子，獄卒不動聲色，只是在記錄周勃口供的竹筒背面填上一行小字：「公主可證明無罪。」周勃看到之後大喜，忙將這道消息傳給了兒子，之前，他的兒子娶了當朝公主為妻，眼下這個公主正是他的救命稻草。

在這關鍵時刻，周勃的另一位救命恩人也出面了，她正是漢文帝的母親薄太后。

薄太后聽聞這件事之後，一直忍而不發，待到漢文帝給母親請安時，才大發雷霆，摔冠怒罵。說：「這周勃倘若真要造反，為什麼要等到現在？當初，他誅殺呂后，威震三軍，又手握重兵，若是真有異心，你豈能當上皇帝？」這一頓怒罵直將漢文帝罵得不敢作聲，連連應下，將周勃無罪釋放。

從鬼門關走了一遭的周勃，終於明白了這個「伴君如伴虎」的道理，不禁仰天長嘆：「我周勃曾經統率千軍萬馬，威震八方，如今，竟不如一個小小的獄卒厲害！」周勃雖是被無罪釋放了，可是，當時的官員，卻仍然對此事憤憤不平。

這其中，有一位名叫賈誼的青年才俊，他知道周勃無故受虐之後，便拿著這件事含沙射影地批評漢文帝。他在著名的《治安策》里，講了一大堆儒學的大道理，云云禮義廉恥。說那些有功之臣畢竟是有身份的人，即使是犯了錯誤，也不該辱沒他們的尊嚴，要讓他們認識到自己的羞恥之心，知恥而後改。

賈誼雖然沒敢明提周勃的名字，但是，漢文帝卻是心裡清楚。

於是，面對這樣一位在當時極富影響力的人物，漢文帝終於是立下了一個：「忠臣死罪，責令自殺」的規矩。可是，這規矩剛下來不久，就碰上了一個「硬釘子」。

漢文帝的親舅舅薄昭殺了使臣，犯下死罪，按理當斬。

可是，薄昭乃是漢文帝的有功大臣，又是皇親國戚，加之規矩新立，漢文帝便勸他自行了斷。可惜，這薄昭實在惜命，整日躲在府中，任憑怎麼逼迫，就是不肯赴死。漢文帝便想了個法子，他派上滿朝大臣都去薄昭府上，披麻戴孝，為薄昭哭喪。這國葬的待遇，實在是給足了面子，薄昭於是便咬舌自盡了。

自此之後，這個規矩便一直流傳下來。

漢文帝新立規矩，最終，還是受了賈誼那篇關於儒家倫理文章的影響：秦朝注重法制，以吏為師。漢朝初期實行黃老之學，修養生息，但是，在法制上卻是仍然沿用了秦朝那一套。後來，為了穩定民心，逐步引進儒家思想，到了漢武帝時，更是大舉引進儒學，逐漸發展到「外儒內法」… …

可以說，漢文帝處理大臣死罪方式的改變，也從側面反映了一個由「法」到「儒」的漸進過程。
