雍正王朝：隆科多當過逃兵，還賣掉寶刀，為何還能得到康熙重用？

天天要聞
2019-12-21

隆科多是康熙特意壓制、歷練，留給下任皇帝的保駕重臣。

下面先介紹一下隆科多的家世，

隆科多，清朝滿洲鑲黃旗人，佟佳氏，康熙第二任皇后孝懿仁(雍正養母)之弟，一等公佟國維次子，康熙末年到雍正初年重臣。按照輩分來說，隆科多是康熙的小舅子，是雍正皇帝的舅舅。

這裡要說明一下，歷史中的隆科多，是佟國維的二兒子，但是在電視劇《雍正王朝》中，卻做了改動，讓隆科多喊佟國維六叔，我們今天以電視劇為主談下：

一、隆科多犯錯，受罰被冷落

隆科多年輕的時候就跟隨康熙做一等侍衛，在官場上起點很高，在打仗的時候，還救過康熙，所以康熙賞賜他寶刀。

本來他和康熙就是姐夫小舅子的關係，又立下救駕的大功，所以被派到了關外帶兵。按康熙的想法，要好好鍛煉他，將來好提拔。

從佟國維的意思，難得家族中出了這麼一個好苗子，歷練一下可以興旺家族。

沒想到這小子吃不了在關外的苦，竟然放棄了關外的官職，跑回北京來了。我想了一下，隆科多在關外，已經是都統副都統的職位，已經是高級將領，就是生活苦點，比一般老百姓，士兵強得多，所以佟國維說隆科多吃不了苦，其實是替隆科多遮蓋，真實原因是隆科多想走捷徑，覺得在京城陪著皇帝，依靠家族力量陞官快，不願意在關外苦巴巴的熬資歷，這應該是隆科多跑回來的真實原因。

隆科多放棄官職，跑回京城，康熙和佟國維都是生氣和失望的。沒想到這小子這麼急功近利，既然拿著朝廷的官職這麼不當回事，那麼你就在家老實待著吧，就把隆科多掛起來了，一掛就是二十年。

就是讓隆科多知道官職的重要性，和忠於皇上、忠於職守的重要性。

二、有意歷練，保護隆科多，讓隆科多不要趨炎附勢，站錯隊，留給繼任皇帝用

隆科多跑回京城，本來以為混過一陣子，然後憑著家族的力量，憑著姐姐和姐夫康熙的恩寵，就能在京城的官場上重新起步，沒想到康熙和隆科多形成默契，要好好歷練一下這小子，讓他心如死灰，再讓他死灰復燃，才能對皇帝，特別是下任皇帝忠心耿耿。

同時，也是為了保護隆科多，不讓隆科多亂站隊，站錯了隊。從隆科多私自跑回來，就看出來這小子喜歡走捷徑，但是康熙知道，自己兒子文武全才的很多，自己都拿不定主意傳位給誰？隆科多這小子哪裡能猜得到，所以，如果他胡亂押寶站隊，站錯了，將來新君要拿他開刀，所以冷落他，還真是要保護他。

現在讓隆科多賦閑在家，就是要降大任給隆科多，就要先苦其心志，磨鍊他。將來磨鍊出來，好留給繼任皇帝。

三、恩威並施，留下生死兩詔書，還寶刀給隆科多既是示恩，也是示警

康熙皇帝在人生暮年，自知來日無多，已經選定了繼承人，把傳位詔書已經寫好，現在就需要一個信得過、靠得住的滿洲親貴來做這個傳詔大臣和保駕大臣。

於是，就在密室宣召了隆科多，先讓張廷玉念了一份遺詔，說隆科多結交阿哥，罪該處死，把個隆科多嚇得靈魂出竅。

康熙端坐聽張廷玉念遺詔給隆科多

再念第二封詔書，說隆科多忠心，讓他當顧命大臣。

然後康熙對隆科多明言，聽我的話，好好宣讀詔書，將來榮華富貴一輩子。如果違背我的遺願，想從中取巧，另立新君，我讓你分分鐘去死。

一軟一硬，一生一死，把個隆科多拿捏的死死的，老老實實的決心要終於皇上，完成宣讀詔書的任務，做忠臣。

隆科多被生死兩詔書收拾的服服帖帖

康熙一看效果不錯，就把隆科多的寶刀還給了他。這把寶刀是當年康熙獎勵給隆科多的，本來是隆科多最珍貴的物品，但是因為必須還國庫欠款，而六叔又不肯借錢給他，所以不得不賣掉。

現在寶刀竟然出現在康熙這裡，說明皇帝一直派人在盯著自己，想到這裡，恐怕隆科多又是一身冷汗。

他參透了康熙隱含的深意，既是示恩，又是示警。

示恩，就是你要忠心耿耿，做朕的寶刀，朕一定提拔重用你。

示警，你小子做的啥事朕都知道，你賣了寶刀，我都能買回來。你要是敢不忠心，就用寶刀殺你。

康熙的帝王心術實在是了不得，這就是他恩威並施，控制隆科多的手段。

這就是隆科多為什麼當過逃兵，賣過寶刀，還能被再次啟用的原因。
