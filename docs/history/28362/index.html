關羽死後，諸葛亮為何連忙勸說劉備賜死劉封，他到底發現了什麼？

2019-12-21 1,992條評論 3,812次閱讀 2020人點贊

導語：三國時期的諸葛亮不僅智謀高超，用了短短10年時間幫助劉備從一個落魄半生的小諸侯，鯨吞荊州和益州後成功三分天下，自己也成了蜀漢的丞相。而且他還非常仁義，早年蟄伏隆中的時候經常幫鄉里人改良農用產品，大大提高了當地老百姓的生產效率，掌權蜀漢以後當有將士被罰20軍棍以上都會親自過問，蜀漢老百姓有什麼冤情他也極力解決。但為何關羽死後諸葛亮連忙勸說劉備賜死劉封？要知道劉封可是劉備的義子，同樣也是蜀漢的猛將，他到底發現了什麼？

事情還得從關羽兵敗陣亡說起，之前諸葛亮和關羽一起鎮守荊州，當時曹操不敢輕舉妄動，劉孫聯盟也非常鞏固，三家人維持一個很微妙的平衡關係。後來諸葛亮被劉備調入益州處理政務，臨行前和關羽提出了交好東吳抵禦曹操的政策，關羽採納之後諸葛亮安心入蜀。但讓人意外的是，關羽因為拒絕孫權的聯姻，孫權惱羞成怒下聯合曹操瓜分荊州並斬殺關羽，由此蜀漢失去了荊州這個富裕之地。

劉備得到關羽陣亡的消息後大怒，恨不得馬上發兵攻打東吳，但這時候諸葛亮卻攔了下來，而且還勸說劉備賜死劉封。原來當年關羽戰敗的時候曾經帶著殘兵朝上庸方向突圍，當時的上庸守將就是劉封，關羽逃到麥城的時候已經離上庸很近，寫信向劉封求援但卻遭到拒絕。關羽收到回信之後錯愕的同時也不知道該往哪裡突圍，因為後退的路全部被魏吳聯軍堵死，不久後就在麥城被擒殺。

劉備知道劉封的情況，上庸離宛城很近，曹操在宛城駐紮有5萬大軍，而上庸僅僅只有3萬大軍，假如抽調過去解救關羽的話，說不定會被曹操拿下上庸，儘管劉封有過失，但罪不至死。諸葛亮卻是從蜀漢穩定的角度來考慮，原來當年劉備在荊州的時候為了拉攏當地的豪強，於是把劉封這個士族出身的人收做乾兒子，而且還承諾把他培養成為自己的繼承人，也正因為這樣劉備才得到了很多物質支持，但後來劉禪卻出生了。

劉禪出生之後劉封的地位就尷尬了，因為劉封始終只是一個外人，他和劉備沒有血緣關係，儘管之前做的承諾也只是權衡之計，日後劉備還是會確立劉禪為繼承人。劉封生性剛烈而且只服劉備，劉備在世時他不會如何，但假若劉備以後不在了，朝中又有什麼人來壓制他呢？屆時說不定會引發叛亂，嚴重的話會導致蜀漢滅亡。為了不讓這樣的事情發生，只能追究劉封的責任。劉備權衡利弊之後採納諸葛亮的建議，下詔賜死了劉封。

結語：自古無情帝王家，在那個三國亂世裡面只有權力才是所有人的追求，道義什麼的都可以丟掉一邊，所以馬超為了拿到西涼的統治權不管父親和幾位兄弟的性命，曹丕上位以後也把對自己構成威脅的兄弟和武將賜死。劉備征戰幾十年早已成為一個梟雄，他不可能為了對劉封僅存的感情而留下禍根，為此劉封的結局也只有死亡，沒有關羽兵敗的借口也會有別的借口。
