俗話說窮不習武，古代那些出身底層的武將是如何練出一身武藝的？

2019-12-24 2,978條評論 4,529次閱讀 662人點贊

在古代門閥制度森嚴，即便是科舉制度出現之後，許多寒門子弟想要出人頭地也是千難萬難，因為良好的教育資源都被富家權貴所把持。寒門學子還可以通過科舉制度而一飛衝天，但是，那些底層出身的武將，是如何練出一身武藝做到以戰功封侯的呢？

自古有句老話叫做「窮文富武」，也就是說在古代能夠練武的人家裡多少都會有點積蓄，至少能夠吃穿不愁，這樣他們才能有精力去學習武藝，再通過軍旅生涯一步步升遷，最終得以光宗耀祖。

那麼那些貧苦人家出身的人，難道就沒有機會在戰場上一刀一槍拼出自己的未來嗎？不能說沒有，但是這樣的機會很少。

自古以來，能夠做到將軍位置的人都不是只憑藉武藝出眾的，他們自身還要精通兵法，善於用兵打仗。因為在古代戰爭中，將軍是不必衝鋒陷陣的，像我們在電視劇裡面所看到的將軍身先士卒的情況是很少見的。因此，歷史上真正有萬夫不擋之勇的將軍數量也不多。

項羽是歷史上真正勇武過人的一大猛將，堪稱歷史上武將之最，但是項羽是楚國貴族出身，從小跟隨叔父學習劍法兵法，而且項羽從小不必為生計發愁，他的人生目標就是推翻秦始皇取而代之，所以有大把的時間精力學習領兵作戰的方法和鍛煉強健的體魄，所以才有了勇猛無比的楚霸王。

根據《隋唐演義》描述，大唐名將秦叔寶，他的祖父是北齊領軍大將秦旭，父親是北齊武衛大將軍秦彝，而秦叔寶的真實家境歷史文獻並沒有記載，但是在1995年，山東濟南出土的一塊墓志銘中記載秦瓊的曾祖、祖父和父親均是魏齊兩朝的官員，雖然不是什麼顯赫的權貴，但也是官員之家，秦瓊能有這一身武藝也就說的過去了。

底層出身的武將最有代表性的就是關羽關雲長，有人說張飛難道不算嗎？雖然影視劇中的張飛是豹頭環眼，聲如奔雷的莽漢形象，但歷史上真實的張飛卻是一個英俊的大帥哥，還寫的一手好字，而且張飛家境很好，是涿郡有名的富戶。

而關羽就不一樣了，根據記載，關羽在家鄉時因為看不慣豪強欺凌弱小而出手誤傷了人，因此亡命江湖，遇到劉備時，只是一個賣棗的小販，如果說關羽家中有錢有勢，那為什麼還要亡命江湖呢？所以據此推測，關羽應該出身貧寒。

但是，跟隨了劉皇叔之後，關羽也沒有結束漂泊的人生，因為劉備勢力弱小，所以處處依附於人，而且處處有人想要加害於他，正是因為關羽張飛的保護，劉備才得以在亂世中生存。

以此可知，關羽有著過人的武藝，《三國志》中稱讚關羽「關羽，張飛皆稱萬人之敵，為世虎臣」，《三國志》成書於晉，所以這個評價還是比較中肯的。總而言之，關羽武藝超群那是不必說的。

關羽一個賣棗的小販，是如何學得如此厲害的武功呢？首先我們來看一下關羽的外形，關羽身長九尺，髯長二尺，面如重棗，唇若塗朱，「身長九尺」是個什麼概念，如果換算成米制的話，應該在兩米一二左右，試想一下這樣的身材，別說打仗了，就是往戰場上一站，那也是不可撼動的存在啊。

第二，關羽亡命江湖五六年，又跟隨劉備漂泊了十幾年，大大小小經歷了無數戰爭，自然從中摸索出了一套適合自己的作戰方法。

所以古代底層出身的武將，要想做到將軍，靠的就是出眾的體格以及在戰場上摸索出的經驗，根本不會像武俠小說裡面的大俠那樣習得什麼絕世武功。
