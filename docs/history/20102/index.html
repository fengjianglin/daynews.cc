多爾袞進入北京後，要把真太子當假太子殺，是他年老糊塗了嗎？

2019-12-16 1,493條評論 2,894次閱讀 343人點贊

崇禎自殺後，三個皇子就被李自成捕獲了。後來，在討伐吳三桂期間，李自成害怕他們圖謀不軌，就把他們帶到了山海關。結果，李自成山海關兵敗後，他的部隊一片混亂，在兵荒馬亂之際，三個皇子逃跑了，也就此失散。

三個皇子失散後，皇太子朱慈烺在亂世中活了下來，他一路乞討，成功回到了京城。

回京後，朱慈烺無處可去。後來，他想到有一個叫常公公的宦官，是一個退休的官員，曾經照顧過他。當太子時，朱慈烺經常去常家玩。於是，朱慈烺憑藉自己的記憶，找到了常公公，準備投奔他。

要知道，常公公不過是一個退休的宦官，焉敢收留前朝太子。於是，常公公委婉地告訴他，說自己這裡廟小，供不起您這尊大佛。如今，您可以去投奔外戚周奎，他是您的姥爺，他肯定會照顧您。而且，您的妹妹長平公主，就是被崇禎砍傷的那位公主也在周奎家，你們可以兄妹相認，一起生活。

在常公公的勸說下，太子同意了此事。於是，他們一起來到了周奎府，投奔周奎。

周奎看見太子後，大吃一驚，隨後馬上全力接待。接待期間，周奎叫來只剩下一隻胳膊的長平公主，讓他們兄妹相認。這對兄妹見面後，相持痛哭，觀者皆感動不已。

認完親後，周奎做東，請大家吃飯。宴席期間，周奎一直行君臣之禮，卻根本不提太子居住一事。

宴會結束後，長平公主單獨來找朱慈烺，給了他一些銀兩，還有一些衣服，告訴了他周家人的心思，她讓朱慈烺以後不要再來了，他們以後也不要再見面了。

見此情景，太子雖然大怒，但他也知道周家的苦衷。於是，朱慈烺拿著這些東西，拜別了妹妹。

太子拿著這些錢後，在外面獨自生活了幾天，養尊處優的他哪裡受得了民間疾苦?他忍耐不住再次造訪，懇請周奎收留自己。

得知太子去而復返後，周奎讓自己的兒子(一說是周奎的侄子)周繹出馬，去回絕太子，不讓他再來。

周繹出門後，給了太子一些銀兩，委婉地告訴他:「您的身份太特殊了，我們不敢收留您。您要想在這個世界存活，就得韜光養晦、低調行事。如今，您千萬別說自己是太子，就說自己姓劉，是一個教書的先生，這樣就能避禍了。」

對於周繹善意的勸告，太子不僅不聽，還跟他吵了起來。

原來，太子認為周家人瞧不起自己，認為自己已經無權無勢，就不認這個親戚了，他們這是欺人太甚。太子玩命地嚷嚷，說自己是大明王朝的太子，是周家人的主子，一日為主子，就永遠是主子。

見此情景，周繹大怒，就跟太子吵了起來，他們還動了手。周繹打了太子一頓，讓他閉嘴。結果，太子的秉性跟他父親一樣，寧死不屈，即使被打了，也絕不閉嘴，依舊痛罵周繹不已。

最終，雙方的動靜越來越大，就被人告官了。清朝馬上派人來到周府，把太子和周家人都緝捕歸案。

太子之所以這樣有恃無恐地亮出身份，也是深思熟慮的結果。

原來，在太子眼中，清朝是一個不錯的王朝，多爾袞善待了自己的父親和母親，不僅厚葬了他們，還頒布詔書，要善待前朝遺孤，以彰顯自己的仁義。因此，在太子眼中，大清王朝應該會善待他，給他封王封爵。最不濟，也能讓他榮華富貴一生。

在太子眼中，對比大順王朝，大清王朝再差，也不應該比大順王朝差。畢竟，李自成承諾保他一世富貴，還冊封他為「宋王」。號稱仁義的清朝，應該不會比李自成差。

然而，可惜的是，太子還是太年輕了，他怎麼可能明白世間的兇狠，也根本不明白大清王朝的「決心」。

就這樣，在黃泉路上，又多了一個英年早逝的糊塗鬼。

太子被抓到刑堂後，漢官刑部主事錢鳳覽負責審理此案。他問清事情的緣由後，竟然斯文掃地，直接衝到大堂上，狠狠地揍了周繹一頓。錢鳳覽告訴周繹一個道理，何為君，何為臣，不能這樣背主負恩!

雖然從事實上看，錢鳳覽投降了清朝，也是一個背主負恩的貨，他跟周繹比，不過是五十步笑百步罷了。然而，跟周繹不同的是，錢鳳覽飽讀詩書，深知孔孟之道，還算是保留了最後一點人性。

在錢鳳覽的毆打下，太子出了一口惡氣，算是心滿意足了。隨後，他被送進了紫禁城，迎來了最後的審判。

負責審判他的人，就是大清最高的統帥——多爾袞。

實話實說，在多爾袞眼中，這個太子就是一個燙手的山芋，非常麻煩。雖然多爾袞昭告了天下，要「善待」明朝的遺孤，以「告慰」崇禎皇帝的在天之靈。他可以這麼說，但真的這麼做，就不行了。

這個原因很簡單，太子是什麼人?太子是明朝的法定繼承人，是明朝遺孤們心中的太陽，只要他還活著，就能讓那些人永遠看見希望，進而讓他們反清復明，顛覆自己的萬里江山。因此，這個人不能活，必須將其斬草除根，才能永絕後患。

然而，這個矛盾點就在於此。一方面，你昭告天下，要善待崇禎皇帝的遺孤;另一方面，你卻言不由衷，斬殺了崇禎的後代。這種出爾反爾的事情，多爾袞怎麼能去干呢?如此行事，豈不是失信於天下?

然而，身為大清王朝的睿親王，多爾袞有的是辦法，他略施小計，就解決了這個難題。這個辦法就是，多爾袞不承認這個太子是真的，他是「假冒」的。既然是一個「假貨」，就可以將其名正言順地斬首示眾了。主意已定，多爾袞不動聲色地靜觀其變，準備看眾人的反應。當時，有很多大臣明白了多爾袞的心思，比如那個精通溜須拍馬的周奎。他就第一時間上書，痛罵朱慈烺是「假太子」。

要知道，李自成發動比餉運動時，不是要求「功勛和外戚之家，無定數，人財兩盡而後矣」嗎?結果，這個周奎卻啥事沒有，足見其左右逢源、巴結權貴的本事。清朝入關後，周奎不知道用了什麼招數，竟然結交了滿洲貴族，從此高枕無憂。

可見，周奎就是一個為了保護財富，可以無所不用其極的小人。這種小人要是肯為太子說話，那才叫見了鬼。即使這個太子是他的親外孫，周奎也會毫不猶疑地反戈一擊。

得知朱慈烺被捕後，周奎二話不說，立刻上疏朝廷，聲稱朱慈烺是「假太子」，他來自己家，就是假冒太子敲詐勒索、騙吃騙喝的。望朝廷明察，還自己一個公道。

對於這種小人，真是無話可說。

當然，有背主負恩，坑害太子的人，同樣也會有精忠報國，袒護太子的人。那個為太子出頭的刑部主事錢鳳覽，就是一個誓死效忠太子的人，一直承認太子是真的。還有十個誓死效忠太子的侍衛，他們也堅稱太子是真的，希望清朝善待朱慈烺，給天下一個表率。

然而，雖然這些人堅稱太子是真的，又管什麼用呢?畢竟，在多爾袞眼中，為了自己的目的，這個太子必須是假的。即使他是真的，也必須是假的。

於是，歷史上的一場鬧劇，就此展開。

多爾袞下令召開群體會議，命令明朝宗室晉王、崇禎遺孀袁妃、前朝大學士謝升出庭，辨別這個太子真偽。

當時，晉王和袁妃先來辨別，他們看了太子後一口咬定太子是「假」的。大學士謝升辨認時，面露愧色，欲言又止，但最後也一口咬定這個太子是「假」的。

一聽謝升說自己是「假」的，太子當時就急了，他哭著對謝升道:「先生為何如此說?您還記得嗎?城陷之前，先生在東宮教課，當時教我『臨危受命』一題。這些事情您怎能忘記?」

在太子的質問下，謝升無言以對，他只能面紅耳赤地逃跑了。但是，這個人還是不改初衷，一口咬定太子是「假」的。

這時，一旁聽審的錢鳳覽坐不住了，他站起身來，痛罵謝升和明朝宗室不仁不義、背主忘恩。同時，錢鳳覽痛罵周奎就是一個小人，是一個衣冠禽獸，不殺不足以平民憤。

最後，錢鳳覽要求多爾袞認真審理朱慈烺的身份，還他一個清白，並恩養這位前朝皇子，給天下一個表率。

雖然錢鳳覽為太子鳴冤不平，但多爾袞已經認定了「事實」，他又能怎麼辦呢?何況，這個所謂的審訊，不過就是多爾袞演的一場戲罷了。他的目的，就是要名正言順地斬殺這個「假皇子」，任何敢質疑他指鹿為馬的人，也只能跟朱慈烺一個下場了。

在一番走走形式的審理後，多爾袞下達了判決書——真假無須爭辯，此人就是一個假太子。現將其萬剮凌遲，以儆效尤。錢鳳覽等人不明是非，皆斬首示眾，立刻執行!

有人假惺惺地求情，多爾袞於是改判——念上天有好生之德，將假太子斬首示眾。念錢鳳覽忠於本朝，賞其全屍，改絞刑。其餘涉案人員(那十個忠於太子的侍衛和給太子引路的常公公)，全部斬首示眾，以儆效尤。

在「證人」周奎、晉王、袁妃、謝升的指控下，朱慈烺就這樣奔赴了刑場，成了一個冤死鬼。太子死時，年齡不詳，但史學家普遍認為，他不會超過二十歲。

多爾袞一口咬定這個太子是假的，這個太子到底是真還是假，其實無須爭辯。

從周奎曾隆重接待太子、謝升被朱慈烺問得啞口無言上看，這個太子就是真的。而晉王和袁妃的供詞，根本不值一提，他們根本無法指證朱慈烺是「假」的。

原因很簡單，明朝有規定，藩王不得離開領地，晉王一直老老實實待在山西，自山西被清軍俘虜後，才來的北京。他根本沒有見過太子，焉能知道太子是真還是假?

同樣，袁妃的供詞也不值一提。因為，史學家們一致認為，這個袁妃就是一個假貨。

原因很簡單，多爾袞可能忘記了，在給崇禎皇帝辦理喪禮時，他挨個冊封被崇禎砍死的妃嬪，其中袁妃就赫然在列。一個被崇禎砍死的人，怎能起死回生，來指證太子呢?

後來，史學家們經過分析，認為此袁妃，非彼袁妃。這個袁妃很可能是天啟皇帝的任妃，只是史料記錯了而已。

此女是魏忠賢的義女，是魏忠賢安插在皇帝身邊的眼線。在天啟朝，她可幹了不少壞事。崇禎登基後，念其孤寡無依，沒有將其斬殺，而是關入了冷宮。為巴結新朝，自願做偽證。

伴隨著太子的慘死，崇禎的後代暫時銷聲匿跡，他剩下的兩個兒子也不知所蹤。雖然在不久後，南明出現了一個「太子」，但史料已經證明，那個「太子」根本不是崇禎的兒子。

直到康熙年間，人們才知道了崇禎一個兒子的下落。但是，清朝的政策決定了，這個皇子出現之日，就是他喪命之時。這個皇子，也不會得到一個好的結局。
