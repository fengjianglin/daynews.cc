摘要

袁世凱一生有十個老婆，為他生下了17個兒子，15個女兒，家族成員眾多，家庭龐大。而他眾多的子女中，要數七女兒袁復禎最引人關注了，這個七女兒天生麗質，笑容迷人。袁復禎是袁世凱和他第四個老婆吳氏生的，在他所有女兒中排行老七。

袁世凱一生有十個老婆，為他生下了17個兒子，15個女兒，家族成員眾多，家庭龐大。而他眾多的子女中，要數七女兒袁復禎最引人關注了，這個七女兒天生麗質，笑容迷人。

袁復禎是袁世凱和他第四個老婆吳氏生的，在他所有女兒中排行老七。吳氏是朝鮮王族金氏嫁人時的陪嫁丫頭，得益於強大的基因，袁復禎亭亭玉立，個子高挑，面容姣好，非常漂亮，笑起來更是迷人。

袁復禎成年後，袁世凱將她嫁給了滿洲權貴蔭昌的兒子蔭鐵閣，袁家和蔭家結為親家。後來，醇親王載灃感覺袁世凱意圖謀反，篡奪政權，要殺掉袁世凱。蔭昌說他是個人才，留下必有大用為由，多次替袁世凱求情。

最後，載灃也放棄了殺袁世凱的念頭。為此，袁家一家把蔭昌看做他們家的大恩人，非常尊敬。再後來，武昌起義爆發了，政府命令蔭昌率領北洋新軍前去打壓起義。但是，部隊里誰也不聽蔭昌的命令，根本沒法指揮。

原來，這支部隊是袁世凱自己訓練出來的，只聽袁世凱的話。實在沒有辦法，朝廷只能再次任命袁世凱，這也讓袁世凱再次有機會進入官場，為他以後繼續混跡官場打開大門。

而袁復禎的丈夫蔭鐵閣，雖然出身權貴家庭，但是他沒有子承父業，並沒有進入官場。而是在《良友畫報》當攝影記者。但是蔭鐵閣是個花心的人，後來他在外邊搞了外遇，並且和他的新歡在外邊同居了起來。袁復禎就和他離婚了。離婚後，袁復禎獨自去了香港，在香港她嘗試著拍電影，但是最終沒有能夠成功。
