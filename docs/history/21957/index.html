大軍出征，皇帝：快殺豬犒勞，士兵得知卻怒而造反，直接攻入皇宮

2019-12-19 608條評論 4,966次閱讀 3408人點贊

古話說「重賞之下，必有勇夫」，這點在軍事作戰中更加明顯，如劉邦為了能夠戰勝項羽，下令以「得項王者，賞千金，封萬戶」，於是人們爭相搶奪項羽的屍體，哪怕只是搶到了一部分也得以封侯，可見不管在什麼時候，這種現實的激勵都能夠極大地激發士氣，然而王莽這麼做的時候，卻玩砸了。

王莽這個人在中國歷史上是一個很奇特的存在，唐朝詩人白居易有首很有名的詩，便是專門來說他的：「周公恐懼流言日，王莽謙恭下士時。向使當初身便死，一生真偽復誰知」。王莽在還沒有功成名就的時候，表現得極為謙卑，因此而被天下的人們視為道德楷模，所以得到了迅速地升遷。

在陽朔三年（前22年）的時候，24歲的王莽才剛剛開始步入仕途，當一個品級很低的官員，但是因為他總表現得恭敬有禮，在遇到災情的時候，甚至將自己的馬車給賣掉賑濟災民，於是迅速積累起了自己的口碑，又表現出了卓越的才幹，同時還用種種的手段打擊異黨。

於是在前8年的時候，38歲的王莽就已經成為了位在三公之上的大司馬，成為大漢朝的權臣，開始執掌實際權柄，不過此時尚未佔據名分上的大義，在經過一些列的操作以後，王莽在公元9年正式篡位，並且改國號為「新」，西漢就此滅亡，新朝誕生。

王莽在建立新朝以後頒布了很多的「便民」措施，但是卻都嚴重脫離實際，反而讓百姓們深受其苦，同時又頒布各種法令，引起已經臣服的匈奴、高句麗等等外邦反抗，於是征伐不休，天下百姓深受其苦，米價從漢文帝時期的每石數文錢漲到了兩千錢，後來竟與黃金等價。

老百姓們在實在活不下去了的情況下，只能夠選擇反抗，於是天下各地起義軍蜂湧而起，最初的時候官軍尚且能夠鎮壓，但是卻無法完全地平定，於是王莽開始了他神奇的操作，他一口氣分封了九位將軍，號稱為「九虎」，讓他們率領軍隊去平定天下。

在大軍出征之前，王莽表示要殺豬給大軍犒勞，但是士兵們得知這個消息以後，卻紛紛發怒直接造反，還帶著起義軍們攻入了皇宮，讓王莽就此死於亂軍之中。原來王莽所說的「殺豬犒勞」，是殺了一頭豬讓幾萬士兵一起吃，讓人喝一口豬血之後就去戰場廝殺，還威脅士兵們說不賣命廝殺，哪怕是神靈也不會放過你，結果士兵們首先就沒有放過他。
