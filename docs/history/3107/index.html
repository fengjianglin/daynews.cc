三國故事：馬岱早有機會斬殺魏延，但被張飛以雷霆萬鈞之神威殺退

三國名將中，最有人氣，最受民間認可的當然是蜀漢的五虎大將，關張趙馬黃，同時，因為蜀漢集團在民間的影響力和傾向性，甚至幾位其他蜀漢名將，也被看成實力高人一等。特別是魏延，作為正史中，他被劉備重點提拔的新秀，在小說後期也成為了跟趙雲並肩的最可靠猛將。再加上二十多年前日本遊戲中，魏延跟關羽、張飛、趙雲和黃忠一起縱橫沙場，進一步提高了魏文長在三國迷心中的地位。導致絕大多數人都認定，魏延的勇武不在五虎大將之下，或者略有差距也不多。而無論是小說還是正史里，魏延也的確是諸葛亮北伐期間的重要戰鬥輸出，只不過他在小說中被刻畫成了有反骨之人，始終得不到諸葛亮的信任，所以諸葛亮死後，密授馬岱錦囊遺策，斬殺了魏延。

而馬岱也是三國迷心中比較認可的蜀漢上中地位的名將，畢竟他是錦馬超的族弟，也算是將門虎子，雖然不如馬超勇武，但陳倉侯平北將軍的名號也不是浪得虛名，體內流淌著猛將家族的血液，再加上斬殺魏延的「功勞」，使得他也堪稱是馬家軍的二號人物。但馬岱在殺掉魏延之後，無論是正史還是小說中，都很難再找到他的身影了，另外，他的實力雖然不錯，但總體而言，明顯要比五虎將差一大截的，所以很多人認為，馬岱殺掉魏延，不過是靠著偷襲而已。但是，在小說的設定中，其實馬岱一直就不比魏延差，甚至殺魏延的機會，也不止得手的那一次，換句話說，如果認為馬岱比五虎將差很多的話，那麼魏延的能力，也就遜色不少了，大家印象中魏延不弱於五虎將的設定，其實也只是個誤會。

這麼說當然是有「證據」的，關於這個證據，就發生在小說中，張飛馬超大戰葭萌關之前。由於馬超當時效力於張魯，異常驍勇，無人可敵，諸葛亮便率領張飛和魏延一起趕赴前線，會斗馬超。以張三爺的脾氣，根本不在乎一個年輕的「馬兒」，於是早就主動請戰，要殺敗馬超。但魏延同樣爭強好勝，因此他也率軍搶先一步，殺奔馬超的大營。

等到魏延與馬超的部隊遭遇後，馬岱作為先鋒，搶先出戰了，魏延一看對面小將英武異常，便以為他是馬超，立刻搶攻上前，二將大戰了十幾個回合，不分勝負，這就能看出馬岱的實力了。更厲害的在後面，馬岱一看魏延也不是自己可以輕易擊敗的大將，便決定詐敗，魏延著急立功，不想放過擊敗「馬超」的機會，便緊追不捨，結果被馬岱回身一箭，射中臂膀。魏延受傷，當然無法再戰，立刻撥馬回撤，這時馬岱緊追過去，眼見得可以結果魏延的性命，不料張飛猶如天神下凡一般，以雷霆萬鈞之勢，截殺住了馬岱，救下了魏延。而經過這樣一次對陣，就能看出，魏延在馬岱面前，武力是不佔什麼優勢的，即使是馬岱暗箭傷人，可是魏延也沒能在正面單挑中動馬岱分毫。

再對比馬岱和張飛的差距，間接就能看出魏延與五虎大將的距離了。因為張飛不像魏延那樣，著急搶功，反而是很有大將風度地問來將性命，當得知對方不是馬超後，根本不屑於動手，只叫馬超出來。可是馬岱也不是白給的，當然要跟張飛較量一番，結果兩人打了不到十個回合，張飛就打跑了馬岱，但這裡有一個細節是，馬岱跟魏延已經大戰十幾回合了，體力上有些不足。假如在雙方實力均等的情況下，馬岱未必會輸的那麼「脆」，但最多也就堅持十幾二十幾個回合罷了。

因此，通過上述對比我們可以確定兩點，馬岱斬殺魏延不光是靠偷襲，他的實力是被很多人低估的，魏延的武力也的確在五虎大將之下，跟馬岱實力相仿，與關張趙馬黃等人相比，差距還是比較明顯的。不知道各位是否認同老樂的說法，歡迎大家留言，咱們一起交流探討。
