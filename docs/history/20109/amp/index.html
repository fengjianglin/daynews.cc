三國一對異性兄弟，一人力敵趙雲，一人讓許褚不敢吭聲，結局悲慘

天天要聞
2019-12-16

品讀三國故事，我們不難發現，當時有很多厲害的兄弟組合，例如，江東有孫策孫權兄弟，兩人皆年少成名，一個二十餘歲平定八十餘縣，一個未及弱冠坐鎮江東；再如魏晉有司馬師司馬昭兄弟，把持曹魏大權，為司馬家族以晉代魏奠定了基礎；再如分別效力於蜀漢和江東的諸葛亮兄弟，一人成為蜀漢丞相，把持朝政十一年，一人成為孫權心腹，得孫權倚重。而《三國演義》中，則重點描繪了幾對異姓兄弟。

大圖模式

其中最厲害的一對異姓兄姓，非關羽張飛莫屬——在演義中，關羽溫酒斬華雄，一刀誅顏良，三合敗文丑，威震華夏；張飛虎嘯當陽橋，令曹軍喪膽。只不過，他們二人，一人中敵人奸計，終被擒殺，一人則死於宵小之手，非常惹人唏噓。然而，除了他們之外，還有一對異姓兄弟，雖然武藝比起他們有些差距，卻比他們更令人惋惜。這一對異姓兄弟分別是誰呢？

大圖模式

他們不是別人，正是袁紹麾下的上將顏良和文丑——小說第二十六回，《袁本初敗兵折將 關雲長掛印封金》中提到，顏良戰死沙場的消息傳至後方之後，袁紹要為其報仇，文丑主動請纓道：「顏良與我如兄弟，今被曹賊所殺，我安得不雪其恨？」由此可見，這二人的確算是一對異姓兄弟。那麼他們為什麼比關羽和張飛更令人惋惜呢？

大圖模式

眾所周知，關羽和張飛，雖然死因比較憋屈，也令人唏噓，但是作為一個武將，他們早已揚名立萬，在小說中也有不少精彩的表現，可謂出夠了風頭。可是顏良和文丑，卻死得非常憋屈，甚至看起來有些徒有虛名，這是怎麼回事呢？我們一起來了解一下。

一、顏良的悲劇

大圖模式

顏良是袁紹麾下的大將，他第一次被提及，是諸侯聚義，討伐董卓的時候。當時華雄連斬四將，諸侯人心惶惶，袁紹提到，自己麾下的上將顏良和文丑，可以輕鬆斬殺華雄。不過當時他並沒有出場，直到白馬之戰，他才正式露面，並展現了自己的能力——呂布麾下的八健將宋憲和魏續，直接被他秒殺，名將徐晃上陣，也僅二十合，便敗歸本陣。於是曹操「諸將栗然」，就連後來敢挑釁馬超的許褚，也沒吭聲。

大圖模式

可是顏良此戰，不僅要與曹操對陣，還肩負著替劉備打聽關羽消息的任務。於是，當見到關羽的時候，他並沒有著急出手。可他沒料到的是，關羽不按常理出牌，根本沒有叫陣，直接「倒提青龍刀，跑下山來」，於是顏良「方欲問時」，便被取了首級。以顏良之前表現出的能力，如果公平對戰的話，他就算難以取勝，大約也不會敗得太慘，可惜他剛剛出了風頭，就莫名其妙的被關羽斬殺了，相信他死的時候，驚愕大於恐懼吧？

大圖模式

二、文丑的悲劇

大圖模式

作為與顏良齊名的存在，文丑的表現機會，要比顏良多得多，例如界橋之戰的時候，他不到十合大敗公孫瓚，之後五六十回合，戰平趙雲。要知道，趙雲是一個，能於萬軍之中，七進七出，還能斬殺五十餘員敵將的武將，文丑能戰平他，其能力可見一斑。而且在為顏良報仇的時候，文丑雙戰張遼、徐晃的表現，也令人驚艷。可是當時他的兵馬，早已因中計而潰敗，他無心戀戰，力量又不及關羽，所以「三合而心怯」，選擇了撤退。

大圖模式

可是他沒有注意到，關羽跨下騎的是赤兔馬，速度非普通馬匹可比，於是他被關羽追上，一刀斬下馬來，也丟了性命。

大圖模式

通篇分析之後，我們不得不承認，顏良和文丑這對異姓兄弟，實力出眾，如果公平對決，大約不會這麼輕易的戰死沙場。可是，他們都未能逃脫死於關羽之手的命運，而且死得非常憋屈，實在令人惋惜。
