古代防作弊手段多恐怖？電子科技只是小兒科，看了你還想活在現代

大家對科舉應該都很熟悉吧？這不僅是中國古代通過考試選拔官吏的制度，還是古代貧寒家庭光宗耀祖唯一的出路，所以古代人對於考試非常重視，並且科舉考試原則上允許自己報名參加，不必非得由公卿大臣或州郡長官特別推薦，這一點是科舉制最主要的特點，得到了當地古人的一致推崇。

不過既然有考試，那就肯定就有作弊，那麼你知道古人是如何防止考試作弊的嗎？讓我們一起來看看吧，或許你能夠看到他們最神奇的一面，而這些卻與我們在影視劇和電影中看到的完全不同，甚至顛覆了我們對科舉考試的認知。

首先，考生進入考場前都需要進行嚴格搜查，搜查考生帶著的所有物件，甚至還要脫衣服，而進入考場後，每位考生都會領到一個單間，不要說考試交頭接耳了，就連看都看不到身旁的考生，更別說作弊了。

還有考試後，這一點與現代考試有異曲同工之妙，也就是在考官閱卷的時候根本不會看到名字，也根本不知道是誰，等到所有試卷全部評完，才會將試卷進行拆分、登記，這就是所謂的彌封制，一直沿用至今。

甚至，還有一種更恐怖的防作弊手段，那就是在封卷之後，會有專人將所有試卷抄寫一遍再進行閱卷，而這樣做也是為了防止考生在試卷上作標記、暗號，這就是恐怖的謄抄制。

而如果被抓到作弊，後果也是恐怖的，比如唐伯虎，他就是因為被牽連進作弊案，而害得他凄慘一生。

那麼，你對此怎麼看？如果是你，還想生活在古代嗎？
