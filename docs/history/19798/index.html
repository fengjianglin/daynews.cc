莫扎特、貝多芬、齊白石、豐子愷這些名人也有怪癖？

2019-12-18 289條評論 1,251次閱讀 208人點贊

歡迎關注印客美學

懂美學的人，連顏值都會變高

每個人都或多或少會有一些小癖好，比如去超市偷偷把巧克力捏斷，放個屁躲在被窩裡偷偷聞，拔掉分叉的頭髮在一根一根把叉分開……

藝術家文學家們就更不例外了，或者他們在我們心目中就是「奇葩行為」的代名詞。

據說盧梭喜歡被人打屁屁，溫斯頓丘吉爾喜歡裸體上班，克里斯蒂安七世喜歡打手沖……誰還沒點小怪癖呢。

但他們的這些怪癖可能有點……令人驚嘆！

莫扎特


——穢物癖：對shi的熱愛

在很多書本中，穢物癖又稱戀shi癖或嗜糞癖。

穢物癖指的就是喜歡各類排泄物和髒東西的奇怪癖好。 還有的人覺得東西越臟越好，也接近穢物癖了。

據說，莫扎特不僅自己喜歡，還呼籲讓身邊的親朋好友去吃屎……那麼重口味的嗎？

《開卷八分鐘》是鳳凰衛視出品的一個日播的讀書節目，裡面曾經推薦了一本書叫做《噓噓嗯嗯屁屁》。小印慕名去膜拜了一下，其中有這麼一段內容。

節選： 「我會尿在你的鼻子上，一直流到你的下巴。我祝你有個好夢，祝你上床放個響屁，祝你一夜安眠。明天我理智點我再跟你說，我會親口告訴你屁股猶如火燒，這意味著什麼？哎呀，也許有大便快要出來了。我又想起了你的大便，我曉得，我看過的，我的老天啊，太美妙了。」

這是莫扎特曾經給他寫信給他小表妹寫的信中的話語，虎狼之詞啊。

傳說莫扎特在很多信里的話都很污，大概就是舔菊花之類。

莫扎特對shi的熱愛，並且認為太過於美妙……

倪瓚

——潔癖：把樹洗死的怪咖

看到莫扎特的癖好後，小印腦海里跳出的只有倪瓚了，倪瓚潔癖的知名度可能要不他的藝術成就知名度更高？

倪瓚專門雇兩個人幫他擦文房四寶，因為一會不擦，他就覺得臟。

不僅如此，室外的梧桐樹他也沒有放過，結果好好一棵樹，硬是被他給洗死了。

這只是愛乾淨罷了，更玄妙的是他們家廁所。

倪瓚廁所的下面鋪著潔白的鵝毛，當便便落下，鵝毛就會飄起來掩蓋住便便，能發明這樣的裝置，也是有才之人了。

更有趣的是，潔癖還會「誤事」。據說有一次倪瓚看上了一個趙姓歌姬，但是他有潔癖，不停的看看她身上臟不臟，覺得臟給她洗一洗……

最後折騰了一晚上，乾脆不睡趙姬了，嘖嘖，這個潔癖啊。

貝多芬

——沐浴癖：為找靈感洗禿嚕皮

一聽好像與倪瓚有些類似，但貝多芬倒不是因為潔癖，而是泡在水裡找靈感。

曾有一份《全球沐浴調查白皮書》的調查，結果顯示72%的人在沐浴時產生過新的思路。從心理學來分析，洗澡時思維處於放鬆狀態，這種情況下比較容易激發靈感。

貝多芬喜歡站在盥洗盆前，將大杯大杯的水倒在手上，與此同時抑揚頓挫地唱出旋律，然後在屋裡大步踱步，記下靈感。

據說他的作曲中，有許多樂章的結尾就是在這樣玩水的動作中完成。

但他的這個怪癖讓房東和鄰居很不開心，因為水經常濺落在地板上，然後又滲透到樓下。

根據記載，貝多芬一生中搬家搬的次數很多，可以理解，誰家裡天天滴水也會受不了的，這是一個給別人造成麻煩不受待見的癖好呀。

建議：住一樓。

米開朗基羅

——有味道：不愛洗澡的騙子？

有人愛沐浴成癖好，自然有人不愛。

他沒有辜負他生命的每一秒鐘，每時每刻他都在創作，以至於他沒有時間去洗澡，鞋子都長在了他的腳上拔不下來。——劉晉鋒

你以為鞋子長在他腳上拔不下來只是個單純的在說明米開朗基羅對於時間的珍惜嗎？

不，米開朗基羅是出了名的邋遢，到底有多邋遢？

據說他的住所骯髒不堪，而且他不僅不愛換衣服更不愛洗澡。

在他去世的時候，衣服已經臟到貼在了身上，料理後事的人不得不用剪刀把衣服剪破，再一片片剝掉。

彷彿有了畫面，腳上的鞋子拔不動，身上的衣服脫不下，以後再看米開朗基羅的作品可能都會有別樣風味。

徐渭

——自殺：九番自殺的大畫家

徐渭的人生可用一句話來總結：一生坎坷，二兄早亡，三次結婚，四處幫閑，五車學富，六親皆散，七年冤獄，八試不售，九番自殺，十（實）堪嗟嘆！

鄭板橋一方印自稱為「青藤門下走狗」。

齊白石曾雲「恨不生三百年前，為青藤磨墨理紙。」

如此一位才子卻在中年自殺九次。到底是經歷了什麼？

徐渭出生未過百日，父親就病死了。10歲時生母被趕出徐家大門，骨肉分離。26歲時，妻子得病去世，家破人亡。

家世慘淡，仕途也是不得志，徐渭連著考了8屆到41歲也未能中舉。到44歲時，領導胡宗憲被捕入獄，作為幕僚的徐渭也受到牽連。

當年，出謀掃滅倭寇，如今反受罪牽連，徐渭徹底瘋了。

據《明史·文苑傳》記載，他曾九次自殺：「血流被面，頭骨皆折」，幸而不死；又一次似鬼神附體，他以三寸長的柱釘刺入左耳數寸，然後用頭撞地，把鐵釘撞入耳內，絲毫不覺得痛苦，又不死；後又用椎子擊碎自己的腎囊，仍不死。

可他最後自殺不成卻殺了他的第二任妻子。

到了晚年，徐渭自作《畸譜》，提及殺妻之事，簡略記載說：「病易，殺張下獄。」

「易」即「癔」，意思是在精神失常的情況下殺死了妻子。

真是令人唏噓的一生。

齊白石

——摳：「摳界」之王？

齊白石作為一代藝術大師，他的摳門的程度，在「摳界」也是沒誰可比了。

齊白石賣畫有「三不」：不賒賬，不還價，不以物易畫。

他家的牆上除了價碼錶，還貼著各種「友情提示」：送禮物不畫，請客不畫，為外國人翻譯者無報酬；君子有恥，請照潤格出錢；減價者，虧人利己，余不樂見。

齊老的摳門也鬧出很多趣事。

有一次，一位點明要畫蝦的買客只付了一半的銀兩，老先生心裡不悅，但口頭上未計較，在下筆時留了分寸，一隻蝦只露出半個身子，另半身隱在石頭後面。

若隱若現的蝦也是別有一番風味的，著實是個有缺的老人。其實想想，那遮住一半蝦身的石頭，不也是用筆用墨用功力畫出來的么？

黃永玉描寫了他第一次隨李可染大師去白石老人家拜訪場景。

老人見到生客，照例親自開了櫃門的鎖，取出兩碟待客的點心。……我遠遠注視這久已聞名的點心，發現剖開的月餅內，有細微的小蟲子在活動。剝開的花生也隱約見到閃動著的蛛網。——《比我老的老頭》

齊老真是摳到令人哭笑不得啊。

達利

——自戀癖：逆天自戀狂

大家有沒有看過達利的照片？他誇張上翹的鬍子，似乎是他的個人標籤。

他是一位具有非凡才能和想像力的藝術家，因為他觸目驚心、與夢相關的超現實主義畫面被大眾所熟悉。《記憶的永恆》是達利代表作品。

為什麼說他自戀？我們看一下達利的語錄。

「我並不擅長謙虛。畢加索是西班牙人，我也是；畢加索是天才，我也是；畢加索舉世聞名，我也是。」

「由於我是天才，我沒有死亡的權利，天才會死，天才的作品不朽。」

品品這些話，說他是史上最逆天自戀癖的藝術家沒有冤枉他吧。

但他的確有自戀的資本。

此外，他還有句名言：「每天早晨醒來，我都在體驗一次極度的快樂，那就是成為達利的快樂。」如果你能理解他的快樂，恭喜你加入達利的小組了！

喬伊斯

——猥褻戀物癖：收集女人用過的胸罩和內褲

詹姆斯·喬伊斯是二十世紀最偉大的作家之一。他早先的生活很艱苦，《尤利西斯》出版後，他一舉成名，過起了舒適的生活。

但在私下裡，他卻有個很猥瑣的戀物癖——喜歡女人用過的胸罩和內褲。

據說他每次嫖妓，都要把妓女的胸罩和內褲帶回家玩弄好一陣子，以此激發無窮的聯想。他甚至覺得，一個人偷偷地玩弄妓女的腦罩和內褲，比赤身露體的妓女在他面前更加刺激。

除了戀物癖，喬伊斯還有一個怪癖，那就是一有空就泡在咖啡館和酒巴里，整天和別人閑聊。

據他說，他這樣做是為了避免和妻子接觸太多，以至多生孩子。他還開玩笑地將此稱為「最佳避孕法」，稱他不喜歡有一大群孩子來煩他。

話雖這麼說，婚後的喬伊斯還時常拈花惹草。譬如他在巴黎教英語時，就曾和一個年輕漂亮的猶太女學生有過一段戀情。

著名精神分析學家和醫生卡爾·古斯塔夫·榮格得知他的病情後,主動要求為喬伊斯進行治療，但喬伊斯一再推託拒絕治療。他認為一旦他的病治好了，他創作的激情和慾望也就隨之消逝。他寧願與疾病為伴而持有創造的慾望，也不願身心康復而失去寫作的激情。

真真是藝術家的怪癖了。

米芾

——戀物癖：為石頭而「顛」

米芾也「戀物」，但和喬伊斯截然不同，米芾痴迷異石硯台，曾因石頭丟官，也曾千里獨行只為要回一硯台 。

米芾有個外號「米顛」。何為「顛」？

古代的文人離不開文房四寶這是自然，而米芾對於硯台的研究尤其深，他將硯台的色澤工藝，甚至產地都研究得非常清楚，米芾還有一本著作《觀硯》，這本著作之中記載的便是關於硯台的各種講解。

宋徽宗是個藝術修養很高的皇帝，他聽聞米芾的書法很好，便請他到瑤林殿題詩。哪知米芾一進瑤林殿便被一尊硯台給迷住了，他的目光如何也無法從那尊硯台上挪開。

米芾趁宋徽宗的注意力全放到了書法上之時，將皇上的硯台拿起來往自己的袖子裡邊塞，結果裡面的墨汁將他的衣服給弄髒了。

於是米芾便詢問宋徽宗，說這個硯台已經被自己給弄髒了，能否將它賞給自己呢。宋徽宗便揮手示意將硯台給了米芾。等到米芾離開之後，宋徽宗才感慨「米顛」之名。

豐子愷

——rua貓：吸貓大師

你以為我們當代人才會吸貓，擼貓嗎？豐老才是真正的「貓奴」。

豐子愷一生與貓有緣，他養貓、畫貓、寫貓，乃至有讀者「你一隻我一隻」地給他送貓。

豐老不僅愛貓，還為它們寫過幾篇散文，如《白象》、《阿咪》。

「阿咪者，小白貓也。十五年前我曾為大白貓『白象』寫文。白象死後又曾養一黃貓，並未為它寫文。最近來了這阿咪，似覺非寫不可了。」

——豐子愷筆下的「阿咪」

愛貓就要寫貓！豐老嘆道沒為黃貓寫文章，只有小白喵不夠，要雨露均沾。他寫《阿咪》寫著寫著，就寫起黃貓來了：

「寫到這裡，我回想起已故的黃貓來了。這貓名叫『貓伯伯』。……」

在那個的時代，許多人養貓並非當寵物，而是為了捕鼠，但豐老早看出貓「有功於人生」，他寫道：

「可知貓是男女老幼一切人民大家喜愛的動物。貓的可愛，可說是群眾意見。而實際上，如上所述，貓的確能化岑寂為熱鬧，變枯燥為生趣，轉懊惱為歡笑；能助人親善，教人團結。即使不捕老鼠，也有功於人生。」

「貓奴」本人了。

藝術家需要更加充沛的情感， 誰還沒點怪癖了，不過爾爾……
