藍玉被稱戰神戰功赫赫，與朱元璋親家相稱，為何死後皮骨分離

天天要聞
2019-12-19

『君權至上，真龍天子』，帝王掌權，主管生殺；『君讓臣死，臣不得不死』，帝王之命，不敢違抗。封建主義，君主集權，遇事不公，奈何手中無權，『功高蓋主』就得趁早下手，這些例子比比皆是，明宋打擊魏忠賢的勢力、清朝嘉慶打到和珅的勢力等等。然而今天想要給大家介紹的並不是前面說兩種，現在講講生前戰功赫赫，死後皮骨分離的藍玉吧！

​

開創大明王朝的朱元璋，本是一介乞丐，元朝末期，順勢起義，登上帝位，一路走來不僅因為其能力，還有身邊的得力幹將們，才換得大好江山。就在朱元璋成為明朝皇帝不久，在世人眼中，彷彿變得六親不認，對待曾經的同胞狠下毒手。歷史上這樣的事情不在少數，帝王登位，必殺忠誠謀士，正所謂『一朝天子一朝臣』，不少忠誠謀士在幫助帝王登位後就告老還鄉，就是避免像藍玉一樣被帝王誅殺，藍玉就是在朱元璋登位後忌憚功績而被朱元璋誅殺的，上下兩萬餘人，無一倖免。

​

藍玉被稱為戰神，乃朱元璋身邊大將，史上有名的『藍玉案』講的就是他，受到牽連的人數不勝數。殺了大將不怕老將心寒嗎？少了一員大將不會可惜嗎？朱元璋為什麼要這樣做呢？其原因就是以藍玉為首的一群人嚴重威脅到了朱元璋的地位和權力，而到現在，也有學者不斷研究發生這件事情的具體經過。

​

藍玉是位非常優秀的將軍，被稱戰神，在戰場上無人可擋，戰功赫赫，也就是因為這樣，他的權力便越來越大，得到「常勝將軍」的稱號無可厚非。也就是因為其存在，朱元璋才能夠順利的當王，不然何時才能稱帝？藍玉戰功赫赫，便洋洋自得，無論何時都不經朱元璋就自作主張，愈演愈烈，藍玉覺得朱元璋不能沒有他，逐漸失去本心，變得殘暴不仁，因為守城池的軍人們沒有在合適的時機開門歡迎他們，於是他便要把這些人殺個精光，這件事已經惹怒了朱元璋，但是他還是依舊看在眼裡不言語，殺生搶奪在民間不斷發生，曾經的百姓不再擁戴他，關愛百姓的藍玉將軍，在這個時候已經不見了。

1393年，朱元璋無可忍耐，以『謀反罪』了結了他，有關『是否謀反』這件事情未得到確認，一種說法是藍玉為了讓女兒不受委屈，以下犯上，奪取帝位；另一種說法是朱棣害死了藍玉，因他發現朱棣的狼子野心，勸告太子小心謹慎，太子不以為然，而這事被朱棣知道了，朱棣懷恨在心，遂在朱元璋面前詆毀藍玉，朱元璋心想之後肯定是要出事。這事之後朱元璋開始戒備，他決定還是先下手為強。總之不論哪種說法，藍玉必死無疑。『藍玉案』中不僅藍玉必死無疑，連其親信也被一舉剷除，親信也是砍頭處死，藍玉則被剝皮剔骨，想想都毛骨悚然， 他本以朱元璋以親家相稱，由於這個原因，便被留了全屍，朱元璋派人把他的皮從上到下完整的剝下來，並且給了她的女兒做個念想，想必這是古往今來就慘的刑法了吧。

​

藍玉集團多達兩萬人

《明史》記載：藍玉一案，族誅一公、十三侯、二伯，牽連被殺一萬五千多人

如此大義滅親的舉動，可謂鳳毛麟角，但無人敢持有反對意見，誰讓朱元璋掌握生殺大權呢？落得如此下場，藍玉自己也有責任，憑藉自己的功勞不可一世、目中無人，卻不知『飛鳥盡，良弓藏，敵國破，謀臣亡』的道理，在皇帝面前耀武揚威，可謂愚蠢。此事過後，朝野上下再無反抗之心。想必各位大臣們面對這樣的皇帝，心中也有些許無奈吧，聽到朱元璋的名字是不是也會抖三抖呢？古代帝王皆是如此，只要看見或者聽到了能夠威脅到自己地位的苗頭，把一些人甚至是一群人千刀萬剮也是值得的，天下之人的生死皆在帝王的一念之間。

『藍玉案』讓無數大臣感到心寒，畢竟藍玉鞠躬盡瘁，立下過汗馬功勞，朱元璋本來很欣賞他，想必朱和藍二人未曾想過二人會以如此方式說再見吧。藍玉為功臣，受帝王關照，本應此生順利，奈何迷失本心，落此結局，亦是必然。自古帝王多無情，沒有任何帝王能夠一再縱容大臣挑戰底線，擾亂朝綱。歷史上沒有一個帝王能夠一再縱容大臣一次又一次挑戰自己的底線，留著這樣的人在身邊，遲早是個禍害。雖然這樣的事情在古代歷史上發生過不少，即使殘忍，還算是合理的。

​

不論文臣還是武將，看清定位，好好做事，為帝王分憂就夠了，越界並非良策。藍玉驍勇善戰，但自視甚高，變得飛揚跋扈，成為眼中釘。歷史上被帝王忌憚的良將好官都因『功高蓋主』而被處死，俗話說樹大招風，懂得功成身退，做人不能過於貪婪，不能驕傲自大，藍玉案就是這些人即將面臨的後果，其實有時候並不是因為權力大的人做了什麼過分的事情必須得處死，到底曾經發生過什麼事情，都是憑藉猜測罷了。有時候因為一個小細節，讓帝王們看到了自己的二心，就不會有好下場。我們熟悉的韓信，年羹堯等人都是這樣，對於藍玉案，大家還有什麼別的看法嗎？
