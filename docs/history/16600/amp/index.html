樊噲一屠戶安能早就大事？漫談與劉邦打江山的“鄉巴佬”

天天要聞
3 天 ago

劉邦是中國歷史上，第一個從一個草根逆襲當皇帝的人。當然，之所以能夠當上皇帝，和他的眾多老鄉支持是離不開的。那麼，作為劉邦的老鄉，原本他們都屬於社會最為底層的人物，為何他們跟隨劉邦起義之後，都成了各領域的頂級人才？

《史記·卷九十三·韓信盧綰列傳第三十三》：盧綰親與高祖相愛，及生男，高祖、盧綰同日生，里中持羊酒賀兩家。

對於劉邦的手下個個都是人才這件事，再次證明一句話：時勢造英雄！人，永遠是歷史環境的產物，不同的歷史環境，會讓不同的人，展現出與它相適應的才能，對某些人來說，可能是生不逢時，對另外一些人來說就是生當其時！假如不是亂世，殺狗的樊噲、趕車的夏侯嬰、吹喪的周勃，一輩子都不會有翻身的機會（除了蕭何有機會升遷）。我個人認為原因大致有如下幾點：

一、亂世給了武將展示才華的機會

軍事人才在什麼時候都不缺，缺的是展現的機會，和歷練的環境。秦末亂世，正是軍事人才展現才華的好機會，中原大地處處都是軍事家的實習場所。在這個大環境下，軍事才具卓越的人才迅速嶄露頭角，比如韓信、項羽、彭越、李左車等。才具一般的人也會在戰火中迅速成長，比如樊噲、周勃等人（夏侯嬰算不上軍事大才）。

一場戰鬥下來，單憑一個「勇」字，就可能讓一名士兵名聲鵲起，比如當年的狄青。戰火不斷，在倒下的枯骨中，自然會站立起功成的將軍。就像今天，商業競爭日益加劇，不誕生出一批商業奇才，那才叫怪事。在那個環境下，文人就不行了，拿不動刀、跑不動路、下不去手、見不了血，不被大兵們當靶子砍了就算阿彌陀佛了。

大儒叔孫通因為不舉薦弟子，被這些學生背後咒罵時，就曾經說過：皇帝現在要的是能攻城拔寨，天天把命掛在刀尖耍的人，你們行嗎？等到天下統一後，叔孫通的弟子們一個個魚貫入朝，侃侃而談，把武將們看得目瞪口呆，生氣也沒用，該到文人出場的時候了！

二、冒尖的永遠是人中龍鳳

於是有人感慨：生不逢時！打住，即便生當其時，也只對人尖適用，跟隨劉邦從家鄉一路打出來的，絕不是「十八候」，更不止樊噲周勃這幾個人，為什麼留下赫赫大名的，就只有這些少數人？歸根結底還是人家確實過得硬！

亂世創造了一個英雄不問出處的好機會，請記住，是給英雄的，不是給庸人的。樊噲、周勃等人，也是實戰中一步步殺出來的，絕對是萬里挑一的人才，絕不是幸運大轉盤轉出來的！我們只看到最後站在塔尖的周勃等，沒看到早早躺在亂墳崗里的士兵甲、士兵乙……我們不能選擇歷史環境，但是可以做特定環境下，一個最好的自己。唯有如此，我們才有可能成為被時勢造出來的英雄！

三、閉塞的環境成了劉邦的首選

《史記卷五十七·絳侯周勃世家第二十七》下下邑，先登。賜爵五大夫。擊秦軍碭東。還軍留及蕭。復攻碭，破之。

當然，周勃等人還佔了歷史條件的光，古代信息閉塞的環境，讓人們的交際圈狹窄，劉邦成就大業，鄉親必然是他的首要選擇。最典型的就是盧綰，倆人是發小，同年同月同日生，還住在一個巷子里，交往自然比較深。

今天我們要幸運得多，信息技術讓我們同居「地球村」，你隨時可以結交地球另一邊的朋友，如果願意，幾內天你可以見到世界任何一個角落裡的朋友。也許長江入海口還有一個盧綰，可惜他一輩子無緣碰到劉邦。也許洛陽也有個周勃，可惜當他加盟劉邦時，吹喪的周勃已經是大將軍。

四、利益集團的排他性，營造了「老鄉會」

有人的地方就有鬥爭，原因無非是利益分配的問題，打天下更是如此，拿命搏出來的東西，比什麼都珍貴，形成利益集團是必然。比如朱元璋手下的淮西集團和浙東集團，比如劉邦手下，有功臣集團、外戚和異姓諸侯王幾個利益集團，其中功臣集團就是以蕭何、曹參、周勃等人為首的豐沛家鄉人為主，所以後世乾脆叫他們「豐沛集團」。

豐沛集團都是抬頭不見低頭見的家鄉人，張家姑娘嫁給李家兒子，趙家老二和錢家老大是同學。各種複雜的關係，構成了一張龐大的利益交換網，外人想插進一腳，沒有幾代人的滲透，根本做不到。還是以盧綰為例，如果他不是劉邦的發小，根本沒機會建功立業，樊噲若不是劉邦的連襟，其地位也不會那麼重要。李左車如果是劉邦的老鄉，也許就沒韓信的機會。

五、侃完亂世出英雄的大環境，再細細聊一聊劉邦手下得力的幾位人才！

不可否認，劉邦的老鄉大部分都是來自社會底層，但是不是絕對，也有一些人是人上人，比如說蕭何。

1.蕭何

《史記·卷五十四·蕭相國世家第二十四》高祖為布衣時，何數以吏事護高祖。高祖為亭長，常左右之。高祖以吏繇咸陽，吏皆送奉錢三，何獨以五。

蕭何是出自世家，在沛縣都是世世代代的名門望族，到了蕭何這裡，他們蕭家是更進一步。要知道蕭何是縣長秘書，縣裡的所有大小事情，都是人家蕭何在主持，縣長也僅僅是頂著一個頭銜而已，真正的權利沒有蕭何大。關鍵是蕭何的政績，在當時可以說是天下聞名的。《史記》上記載，每年年末秦朝對公務員考核的時候，蕭何都能夠排到泗水郡的第一名。

這個名次，是能夠入選到秦朝中央當官的，而且蕭何也接受到了中央的聘用書。但是，被他給拒絕了。他覺得秦二世管理的秦朝，是一個烏煙瘴氣的地方，若是自己去了這樣的地方，恐怕是凶多吉少，還是保命要緊。很快到了秦末的時候天下大亂，劉邦起義了，蕭何就跟了劉邦。劉邦有了蕭何這樣的政治能手，他治理一個國家，就如同烹蝦一般容易了。這就是劉邦的老鄉蕭何。

2. 樊噲

《史記·卷九十五·樊酈滕灌列傳第三十五》舞陽侯樊噲者，沛人也。以屠狗為事，與高祖俱隱。

說完了家世比較好，成分比較高的蕭何，我們再說說其它的，也就是社會底層的人。先說樊噲，大家都知道樊噲是劉邦的老鄉，他還有另一個身份，是劉邦的連襟，因為劉邦和樊噲共同娶了呂公的兩個女兒。樊噲也是劉邦起義過程中的一個重要角色，熟知史記鴻門宴的人，都知道樊噲是一個猛士。

在項羽想要殺劉邦的時候，樊噲不顧自己的安危，開始去頂撞項羽去了。經過樊噲的辯論一番，說的項羽都不好意思了。項羽直接說樊噲是一個英雄，還賞給他酒水和肉吃。看劉邦的老鄉，這些社會底層的人，他們都很有凝聚力，關鍵的時刻不怕頂雷，縱使死無葬身之地，也在所不辭。劉邦擁有了這樣的手下，真的是想不勝利也很難。

3. 夏侯嬰

《史記·卷九十五·樊酈滕灌列傳第三十五》嬰已而試補縣吏，與高祖相愛。

再說夏侯嬰，夏侯嬰是劉邦的車夫，在劉邦起義的過程中，他一直都是幫助劉邦趕車的。要知道能夠當領導的司機的人，都是不簡單的人。最起碼領導對你信任，才會把自己的性命交給你。那麼，為什麼劉邦如此信任夏侯嬰呢？

在劉邦當泗水亭長的時候，夏侯嬰有一次和劉邦在一起玩耍，但是，劉邦不小心把夏侯嬰給弄傷了。這件事最後被官府知道了，官府就把劉邦和夏侯嬰給抓起來了，因為秦朝的時候，是不允許別人私藏武器的，夏侯嬰是被劉邦用刀劍弄傷的。

官府就是想知道有沒有這件事，面對拷打的時候，夏侯嬰始終都是說，這個傷口是自己弄破的，和劉邦沒有關係。最後，因為官府沒有證據，只好把劉邦給放了，而夏侯嬰不肯說出實情來，他被關了幾年。看看劉邦的老鄉，都是這種講義氣的人。也只有這種人，才能幫助劉邦奪取天下了。到了劉邦當了皇帝，所謂的這些「狐朋狗友」，肯定也是最會被重用的人了。

分析：

正所謂一人得道雞犬升天，劉邦的老鄉每一個人都有自己獨特的優點，當他們聚集在一起的時候，這些平日看不出的優點會無限的放大。當他們真正的彙集到一起的時候，任何看起來不可能的事情，只要給他們機會，就沒有做不成的事情。

歷史小結：

總之，一個人本領再大，永遠擺脫不了歷史環境的約束。歷史給了豐沛集團一個好的環境，創造了一個英雄不問出處的平台，這是樊噲周勃等人的幸運。周勃們的自身才能，則是他們脫穎而出的必備條件。一句話概括：機遇加能力，缺一不可！
