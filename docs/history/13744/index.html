歷史打假第4集：秦始皇沒有統一中國，唐朝巔峰比不上隋朝巔峰

2019-12-10 95條評論 1,289次閱讀 215人點贊

大家好，我是昊童，一個專註中國古代史的90後小編

今天為大家打假的歷史謠言發生在秦朝與隋唐時期，傳播率也比較高。老規矩，先跟大傢伙聊個3分鐘。

秦始皇沒有統一中國

謠言：秦始皇沒有做到真正的一統，他雖然統一了六國，統一了南越，但沒有統一當時的屬國——衛國，而且這個衛國一直秦二世時期才真正被一統。

正史：很多人都說衛國並沒有滅亡，理由是，根據史料記載：

君角（始皇帝滅魏後，將姬姓後人君角立為衛君）九年，秦並天下，立為始皇帝；君角二十一年，秦二世廢君角為庶人。

咱們今天先不論始皇帝為什麼沒有滅衛，就說這個衛君，到底算不算衛國（前方高能，比較拗口，還請注意）。

三家分晉相信大傢伙都知道，衛國當時就位於這三家之一魏國的邊上。在韓趙魏強盛的時候衛國只能選擇附庸於魏國，衛成王時期，因為國力愈發弱小，為了不引起魏國的忌憚，衛成王廢棄王號改稱衛成候。再往後到了嗣君時期，連諸侯都沒得做，直接被魏王貶成了君，也就是降候貶君的由來，這段史料上記載

懷君三十一年，朝魏，魏囚殺懷君。魏更立嗣君弟，是為元君。元君為魏婿，故魏立之。

從這兒我們就能看出來，此時的衛國已經成為了魏國的真正附屬，相當於臣子，因為只有諸侯王的臣子才稱君，所以衛國已經在懷君31年就已經滅亡了，根本就用不上始皇帝動手。

至於後面始皇帝將衛君從濮陽遷到野望，那也是遷的衛君，不是衛王！

結論：衛國在始皇帝統一前已經被滅了，留下的只有衛君，君是不算國的，所以這個是謠言，大家不要相信。

巔峰的唐朝比不上隋朝

謠言：隋朝「開皇之治」時全國戶口有890萬戶，唐朝巔峰「貞觀之治」時全國戶口只有820戶，而且整個唐朝的各項指標都不如隋朝，比如農業。

正史：咱一個個來說，隋朝巔峰時應該是在隋煬帝大業5年，並不是所謂的「開皇之治」而且有史料為證，根據《隋書，地理志》記載

隋朝的巔峰人口出現在公元609年也就是大業5年，全國戶口數為8907546（890萬），人口為46,019,956（4600萬）。

而唐朝的巔峰也不是所謂的「貞觀之治」而是唐玄宗天寶14年即公元755年，根據《舊唐書.總本紀》以及《通典，食貨七》記載

天寶14年，全國戶口數為8,914,709（890萬），人口為52,919,309（5200萬），比隋朝多了將近700萬人，由此可見在人數上唐朝完勝隋朝。

另外，根據《通典，食貨七》記載

隋氏西京太倉，東京含嘉倉，洛口倉，華州倉，永豐倉，太原倉，儲米栗者多達千萬石，少者不減數百萬石。

算起來這隋朝的5（含嘉，洛口是一起的）大倉加起來也就不到6000萬石，而且後面還記載了僅僅這些就搞得百姓怨聲載道。在看唐朝，根據《中國古代經濟簡史》記載

由於農業的發展，公元749年（天寶8年），封建政府向百姓搜刮的官倉儲糧竟然多達96612220（9600萬）石。

從這兒看，是不是在糧食上也碾壓了隋朝，9600萬和勉強6000萬，相信大家都知道那個強那個弱吧！

我們都知道，在古代，衡量一個王朝強弱必然離不開人口，產糧，有這兩個，其他的才能發展起來。說到這兒昊童多嘴一句，唐朝作為我國封建王朝的巔峰，他的商業也是十分發達的，尤其是在唐高宗和唐玄宗時期，「能和宋朝比經濟」這句話可不是鬧著玩的。

結論：唐朝無論是人口，糧食，商業都是完勝隋朝，所以那句話是一個徹徹底底的謠言（在這兒昊童向大家道歉，之前我曾經引用過隋朝比唐朝強這句話，是我的錯，我道歉，對不起）。
