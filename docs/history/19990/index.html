巨毋霸身高2米3，東方朔身高2米1，但在支提國人跟前他們都是侏儒

2019-12-16 151條評論 4,231次閱讀 740人點贊

誰是巨毋霸？

在《漢書·王莽傳下》中有這樣一段記載：

「 韓博上言：『有奇士，長丈，大十圍，來至臣府，曰欲奮擊胡虜。自謂巨毋霸 ……』 博意欲以風莽。莽聞惡之，留霸在所新豐，更其姓為巨毋氏，謂因文母太后而霸王符也。」

這位巨毋霸是新莽時人。王莽天鳳六年，太守韓博上言有一位奇士，高大異常，身高一丈，腰有十圍，自稱「巨毋霸」，請王莽收留此人為用。

巨毋霸身高2.31米，雖然高但並不離譜

漢時已有七尺男兒之說，（見荀子「七尺軀」之說：「口耳之間，則四寸耳，曷足以美七尺之軀哉？」），漢代一尺約為23.1厘米，也就是說漢代成年男子的普遍身高也就在1.60米左右。

而巨毋霸的身高一丈，相當於十尺，約為2.31米，這身高比姚明還高（姚明身高2.26米）。

在今天這個身高都令人驚訝，只是因為我們看慣了NBA，表示還能接受。而在漢代那個身高普遍偏矮的時代，巨無霸的出現，絕對會被驚為天人。

再看巨毋霸的腰圍，十圍並不是指十個人合抱，那就真成橫寬的怪物了，這裡圍是指兩手十指相對時的周長，十圍，說明此人的腰圍大概在1.5米左右。

總之，無論在漢代還是現代，巨毋霸都是當之無愧的「小巨人」，但是並沒有巨大到不可想像的地步。

而巨毋霸在人們心目中之所以顯得超級「霸氣」，只因為他還有一個特殊身份：馴獸師。

王莽末年，眼看皇位不保，便想到靠這個巨毋霸來力挽狂瀾，就像把寶押在一件大規模殺傷武器上一樣。王莽命巨毋霸為壘尉，驅猛獸助威，隨軍鎮壓綠林軍。一位巨人率領著一群猛獸衝殺而來，這在氣勢上便不只是靠身高唬人了，足以令人見之膽寒。

然而面對濤濤洪流的天下大勢，這也並無什麼意義。

漢代還有一位被忽視的小巨人：東方朔

其實，就在巨毋霸之前，漢朝也出過一位巨人，那就是東方朔。

據《史記》記載，東方朔身高「九尺三寸」，和巨毋霸也就差了7寸，東方朔身高約相當於今天2.1米。但是同樣身為小巨人的東方朔，平時以插科打諢為生涯，後人們也就忽略了他身形巨大這個事。

真正的巨人族——支提國人，巨毋霸和東方朔在他們面前就是侏儒

很多人借著「巨毋霸」這個事引到了另一個話題：古代巨人族的傳說。首先可以肯定，巨毋霸算不上巨人族，但在漢代真的有巨人族的傳說。這見於東漢時代的一本志怪之書《漢武洞冥記》，其中有這樣一段描述：

太初四年，東方朔從支提國來。國人長三丈二尺，三手三足，各三指，多力，善走，國內小山能移之，有澗泉，飲能盡。結海苔為衣，其戲笑，取犀象相投擲為樂。

看到了吧，這才是真正的巨人，身高三丈以上，手足跟咱普通人也不一樣，三隻手三隻腳，每個手掌（腳掌）上有三個指頭（趾頭）。身高三丈二尺，也就是7.39米——這才叫給力。

正因為這給力的身高，這些人見了小山直接給搬開，遇到小溪水，一飲而盡。平常做什麼遊戲呢，拿著大象犀牛互相丟著玩。

這些巨人生長的國度叫做支提國，東方朔曾經去那裡訪問。不用問，身高2米1的東方朔到那邊肯定是被當做小侏儒看待了。

畢竟《洞冥記》是一本荒誕不經的神仙志怪之書，巨人國「支提國」是否真的存在，也只有我們自行想像了。

神魔中的巨人：鄔文化

另外，在著名的神魔小說《封神演義》中也出現過一位巨人：鄔文化。

高身數丈體榔頭，口似窯門兩眼摳。丈二蒼須如散線，尺三草履似行舟。生成大力排山嶽，食盡全牛賽虎彪。陸地行舟人罕見，蟠龍嶺上火光愁。

他身高數丈，嘴巴像山洞大門，鬍鬚一丈二，草鞋一尺三，有排山之力，吃飯一頭牛直接下肚，能拉著大船在陸地上行走。可以了吧，顯然鄔文化很可能和支提國的巨人是一脈的。但小說並沒有描寫他有三隻手三隻腳，所以還不完全是同源。

巨毋霸的出現，開始是一次「行為藝術」，而今成為一個成語

最後要說一點，巨毋霸出現之時，還有另一個原因。推薦他給王莽的韓博，其實是要藉機諷刺王莽。因為巨毋霸身高巨大，在一般人眼中只能算個怪物。而王莽篡奪漢室天下，改朝換代，其自高自大的行為就像是怪物巨毋霸一樣，不過是個醜類一個，並無什麼大用。

原來這是一次以推薦人才為名，以諷刺挖苦為實的「行為藝術」。

而王莽居然看懂了這層意思，所以乾脆把「糖衣」留下，「炮彈」送回。把巨毋霸收用，把韓博殺了。

「巨毋霸」最初只是這個巨人的外號，王莽乾脆就此賜給他一個新的姓氏——複姓「巨毋」。

意思是說，因為文母太后的厚德，老天才生出這樣一個異人來輔佐自己，這是他把「毋」有意諧音為「母」而討個口彩。

而今天，巨毋霸（巨無霸）已成為一個成語，專指碩大無朋之物，甚至還被成了某漢堡包的專用名。
