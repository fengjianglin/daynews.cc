俄羅斯人繪畫清俄雅克薩之城的全過程

2019-12-16 790條評論 1,758次閱讀 309人點贊

康熙二十四年（1685年）和康熙二十五年（1686年），清俄進行了兩次雅克薩戰役。

大清國和當時俄羅斯，都算是大帝國。他們對外主要戰場上，肯定有上萬軍隊，甚至十幾萬人的軍隊投入。而雅克薩之戰對於這兩大帝國來說，只是小的不能再小的戰役，或者可以說是邊境上的武裝衝突。

俄羅斯第一次只有450人，第二次也不過826人。大清軍同樣人數不多，第一次出動了3000人，因為大清國發現對方人數不多，第二次派人也減少了，只有2100人。

看似大清軍隊要多一些，但要了解當時的攻守關係，俄羅斯有城堡可守，大清國是進攻一方，要包圍雅克薩城並發動攻城，那必然要派出多一些的人數。2100人對826人，通過此戰，大清軍隊幾乎零傷亡，最後俄羅斯軍隊投降，喘著氣活著出去的，只有150人，可見俄羅斯是損失大的一方。

我們看看當時的歷史背景。

在老罕王還沒將女真各部沒有完成統一的時候，俄羅斯已經先後佔領西伯利亞汗國，準備入侵布里亞特汗國，西線正在攻打准格爾汗國，佔領現在的新疆，清軍入關的時候，俄羅斯已經佔領現在呼倫貝爾，黑龍江等地。貝加爾湖的一些布里亞特已經臣服俄羅斯。

大清國崛起後，進關消滅了中原內地各大割據政權，此時，東亞大陸上還有準噶爾汗國（西域）和和碩特汗國（青藏高原），大清國的面積並不大。在大清國與準噶爾汗國爭霸中，喀爾喀蒙古（外蒙古）才加入了大清的聯盟。當時俄羅斯的勢力也在四處擴張，他們不斷向準噶爾方向用兵，如果俄羅斯人搶先滅亡了準噶爾，那麼今天的新疆就是俄羅斯佔據的了。就算今天的人喊「自古以來」口號，那也不會有任何作用。如果被俄羅斯吞併，其後就會是蘇聯的一個加盟共和國，就會是和現在不同歷史結果。

康熙年打的雅克薩之戰是一場爭奪戰，是從俄羅斯佔領下奪取黑龍江西部，把外滿洲納入版圖。俄羅斯認為是大清侵略了俄羅斯。雙方同時簽訂一系列條約，確定了清朝佔領喀爾喀蒙古，俄羅斯佔領布里亞特蒙古。

大清國通過雅克薩之戰，遏制住俄羅斯向滿洲的擴張。清俄分割了蒙古，大清控制了喀爾喀蒙古（外蒙古），俄羅斯則控制蒙古的布里亞特。清俄瓜分了北部的蒙古。

俄羅斯其後也停止向滿洲滲透，而繞了一個彎，從海島南下，佔據千島群島，與日本發生碰撞。

到乾隆皇帝時期，經過三代人努力，搶在俄羅斯人之前消滅了準噶爾汗國，乾隆又將西域改名為「新疆」，設置伊犁將軍府管轄。俄羅斯看到準噶爾汗國滅亡了，也趕快出兵佔領了哈薩克等其他的土地。如果俄羅斯先滅準噶爾汗國，那該地區就類似哈薩克的命運了。

在台的滿族族胞JONES在俄羅斯網上看到一組大清國與俄羅斯的雅克薩之戰中滿洲八旗雄姿的繪畫。下邊是我的原創解說詞。

這是一位滿洲人在深山中行走，意外發現一隻滿洲虎。滿洲人和滿洲虎誰也不服誰，都沒有給對方讓步的意思。滿洲人持槍而上，滿洲虎也是一步步地逼進，人虎之戰一觸即發，看的人都不由把心提起來，滿洲人和滿洲虎要爭一下，到底是誰這山裡的老大？

這是一組滿洲八旗軍隊，正在山上眺望山下的俄羅斯人佔據的雅克薩城。

一隻老虎也爬到高處，也許就是上圖那隻老虎，已經屈服於滿洲人武力。老虎似乎是要當個吃瓜老虎，看看即將爆發的戰爭。當然也可能是被滿洲人馴服，要接收滿洲軍隊指揮，去幫助進攻俄羅斯人佔據的雅克薩城。

滿洲人吹起了進攻的號角，一之飛鳥已經被亂箭射死，一隻大鳥還是登在樹枝上觀看。

俄羅斯軍隊趴在草叢中，端著槍迎戰騎馬進攻的滿洲八旗軍隊。黑色大鳥被驚的在天空中盤旋。

雅克薩城是用木材築城，戰爭正在激烈進行中，有的人已經被射中倒地。

城的一邊還沿著河，河中的大魚異常擁擠，說明當時的自然生態非常的好。當然現在俄羅斯佔據的一邊，因為人少，還保持著完好的自然生態。

最後，俄羅斯戰敗，俄羅斯軍隊投降後，被許可乘船撤出了雅克薩城。

嚴格說，黑龍江流域長期是各部自立，大清國最北駐軍在寧古塔，距離雅克薩3000多里，鞭長莫及，寧古塔以北，按當時資料說：「中國人亦無至其地者」。就是說根本沒有控制。而俄羅斯先佔據並實際控制了這裡，最後硬是被康熙爺派軍隊從俄羅斯手中奪回。

《尼布楚條約》簽訂後多年，康熙帝自己就承認過，在尼布楚條約簽訂之前，當地已被俄羅斯控制了五六十年。

作者富察春兵
