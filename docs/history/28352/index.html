世界史：100年前的它們差點侵佔了全世界

2019-12-21 1,648條評論 3,039次閱讀 1207人點贊

隨著1500年新航路的開闢，歐洲的航海事業開始不斷發展，以至於各國都開始利用這個優勢條件進行著血腥的殖民掠奪，他們不僅用全世界人民的血汗完成了第一次工業革命，更是竭澤而漁的拓展屬於自己的海外殖民地。接著，在經過了第一次工業革命之後，這些歐洲列強開始變得更加暴虐，更加兇殘，除了向全世界大肆搜刮原始資本以外，更是險些統治了全世界。

隨後，經濟的發展促進了政治的變革，而政治的變革又反作用於經濟的崛起，就這樣，在經過了大約400年的時間裡，歐洲列強們再一次完成了加強版的「進化」——第二次工業革命，因此，歐洲列強們便順理成章的進入到了壟斷資本主義階段。隨後，它們奪取殖民地的行徑也達到了高潮。

據數據顯示，從18世紀80年代至19世紀20年代，英國奪得了370萬平方英里的殖民地；法國奪得了360萬平方英里的殖民地；德國奪得了100萬平方英里的殖民地；比利時奪得了90萬平方英里的殖民地；葡萄牙奪得了80萬平方英里的殖民地。【註：1平方英里≈2.6平方千米】

可到了20世紀初的時候，英、俄、法、德這四個大國的殖民地領土面積便已經由原來的4040萬平方公里增至到了6440萬平方公里，以至於整個非洲大陸，除賴比瑞亞和衣索比亞外，都已變成歐洲列強的殖民地，然而，整個歐洲的面積才1016平方公里，相當於100年間掠奪了6個歐洲。可更驚人的是，這些歐洲列強所統治的人口也由原來的27380萬增為49450萬。這是一個什麼概念？據世界人口史的數據，1900年的世界人口為16.5億，而這些列強所控制的人口在此時已經快接近世界人口的三分之一了。

當然，除了非洲不幸蒙難以外，就連擁有4400萬平方公里的亞洲也至少有2500萬平方英里的土地由歐洲列強統治著。哪怕是地理位置遙遠的澳洲和拉丁美洲也沒能幸免於難，相反，都被萬惡的壟斷資本主義所蠶食和破壞，以至於當地土著居民凋零銳減。

這個時候有人說，歐洲列強一家獨大，那美國和日本呢？其實，日本雖然經過明治維新擺脫了被殖民的危機，但直到1905年戰勝沙俄以後國際地位才得以提升，不要看如今的日本的經濟如此發達，在100年前，日本仍被歐洲看作世東方的一個落後國家，不僅如此，就算現在，他們不也是一心想躋身政治大國的行列嗎？

我們再來看此時的美國。據歷史記載，1892年，歐洲諸大國把它們駐華盛頓外交代表的級別從公使升至為大使，這才間接的承認了美國是一個大國，雖然在世界面前，美國的級別上升了，但直到1919年凡爾賽會議舉行期間，美國才在世界性的國際會議上佔有重要地位。

總之，到1914年第一次世界大戰爆發前，歐洲已然稱霸全球，如果沒有他們之間的利益糾葛所引發的世界爭奪戰，可能現在的世界又是另一番景象了。
