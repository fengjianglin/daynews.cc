清朝奇案：少女問少年，糕甜不甜？少年搖頭，幾天後兩人雙雙自縊

2019-12-20 2,929條評論 4,443次閱讀 1485人點贊

清朝道光皇帝在位時，在寶山（今上海市寶山區）的鄉下生活著一個農家後生，名叫鄒玉寶。鄒玉寶雖然還不到二十歲，卻生得高大結實，一表人才。不過鄒玉寶命苦，打小就沒了爹娘，是一個孤苦伶仃的孤兒。

鄒玉寶父母在世的時候，二人為鄒玉寶說了一門娃娃親。親家姓王，王家有一女兒，叫王芳。王芳比鄒玉寶小一歲，兩人是對門鄰居，也是青梅竹馬的玩伴，當時鄒家和王家關係很好，又見兩小孩玩得很開心，就給倆人定了娃娃親，約定成年之後，擇個黃道吉日，就給兩人辦婚事。

​鄒玉寶父母去世之後，王芳的父母並沒有因為鄒家敗落而悔婚，而是把年幼的鄒玉寶接到了自己家裡，讓鄒玉寶在家中吃住，鄒玉寶也因此得以生存了下來。時光荏苒，一轉眼十多年過去了，鄒玉寶和王姑娘都長大成人了，鄒玉寶相貌堂堂，王姑娘也出落得水靈苗條。

兩人彼此都知道有婚約在身，是未來的配偶，又在一起生活了十多年，因此感情很好。只是男女有別，又怕王姑娘的父母批評，不敢走得太近，在沒有結婚之前，只好暫時以兄妹相稱。

有一天，鄒玉寶從外面幹活回來，來到了廚房。他發現王姑娘正在做，用紅糖和糯米粉製成的甜糕，就笑著問：「妹妹，這粉糕甜不甜啊？」王姑娘聽了回頭甜甜一笑說：「當然甜了！」鄒玉寶說：「好妹妹，等糕蒸好了，一定要給我一塊，讓我嘗嘗甜頭啊！」王姑娘笑著說：「看把你饞的，我記著呢，等糕蒸好了先給你吃一塊！」

​王姑娘和鄒玉寶兩個小情人說說笑笑，很是開心，忘記了旁邊還有一個燒火的老媽子李媽呢。說完，鄒玉寶就到一邊幹活去了。等甜糕蒸好之後，王姑娘的父母和王姑娘一起吃了起來，但是王姑娘的父母卻沒有喊在幹活的鄒玉寶一起來吃甜糕。

王姑娘見狀，心裡很不是滋味兒，她趁著父母不備，悄悄來到廚房，拿了一塊甜糕遞給家裡的下人李媽，囑咐李媽一定要把甜糕送給鄒玉寶吃。老媽子點頭答應，說：「好的，姑娘你就放心吧。」王姑娘這才高興地離開了。

​幾天之後，兩人又有了單獨相處的機會，王姑娘問鄒玉寶說：「哥哥，我親手蒸的糕甜不甜？」鄒玉寶邊搖頭，邊生氣地說道：「我根本就沒有吃過，哪裡知道甜不甜！」王姑娘驚訝地說：「我讓李媽給你送了一塊啊 ，她沒給你嗎！」鄒玉寶說：「沒有啊，也許是她自己吃了吧！」王姑娘見心上人沒吃到甜糕，知道自己是被李媽給騙了，她氣得直掉眼淚，鄒玉寶說：「算了，不是什麼大事，不就是一塊糕嘛，我知道你對我好就行了！」

​王姑娘氣沖沖地說：「不行，我得去罵那個老太婆！」說完她氣沖沖地找到李媽，對李媽一通臭罵：「你都這麼大歲數的老太婆了，還這麼饞嘴，丟不丟人！」李媽冷不丁地被王姑娘罵，覺得一時下不來台，也對王姑娘反唇相譏：「我老太婆饞嘴是不假，但是比起那些私下給男人送甜糕的，不知羞恥的丫頭片子，可是好太多了！」

王姑娘見李媽做錯了事不但不道歉，反而變本加厲地譏諷自己，更加生氣，就和李媽在廚房吵了起來！這時王姑娘的父母也聽到了爭吵聲，一問之下知道了事情的原委，他們不怪罪李媽，居然喊了王姑娘幾聲：「你都是大姑娘了，還不知羞！」，王姑娘當時就紅了臉，羞得無地自容，氣得眼淚流個不停，哭著回了閨房。

​鄒玉寶見狀很是心疼王姑娘，但是他在人屋檐下，又不敢為王姑娘爭辯，只好也回了自己的房間。當天晚上，王姑娘越想越氣，一時想不開，居然懸樑自盡了！第二天王姑娘的母親才發現女兒自殺了，後悔得捶胸頓足，馬上讓丈夫打了李媽一頓，將李媽趕出了家門。

鄒玉寶得知王姑娘自盡之後，痛不欲生，他認為是自己貪吃甜糕才害死了王姑娘，如今佳人已逝，自己也不想活了！於是當天中午，鄒玉寶也在房中自縊而死，為王姑娘殉情了！就因為李媽這個老太婆貪吃嘴欠，兩條如花的生命就這樣隨風消逝，真是令人扼腕嘆息！
