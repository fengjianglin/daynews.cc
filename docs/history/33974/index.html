曾是目不識丁的窮小子，因何逆襲成封疆大吏？皆因討了個好老婆

2019-12-23 348條評論 3,769次閱讀 1764人點贊

曾經有那麼一段時間，怕老婆的男人常常會被人譏笑。其實，怕老婆這事，並不是現代男人的專利，而是古已有之。可以這樣自豪地說，怕老婆，是中國男人的優良傳統之一。

今天給你講的是，怕老婆可以提高男人的人生境界與顏值。那誰誰，是不是不信啊？那就讓本故事的主人公——張曜來告訴你吧！

張曜

1）戲說或前言

張曜是誰？

他是清朝光緒年間的封疆大吏——曾主掌山東一切軍政、爵封勤國公的山東巡撫。

為什麼想到要講張曜的故事？

因為，張曜早年曾在我的家鄉河南信陽工作過，他人生的第一桶金就是那兒挖的。並且，他曾率兵解過我們老家光山縣之圍，算是有恩於我的先輩。

他能混到如此高位，是官二代嗎？

不是，他早年只是江浙間的一個街頭小混混。

從一個街頭小混混成長為主政一方的封疆大吏，肯定是別人無法趨及的傳奇。那麼，他的成功秘訣是什麼？

怕老婆！

靠，這也算秘訣。何以為證呢？

有史為證。

史載：張曜就任山東巡撫後，有一天宴請山東各有關部門的領導。那時還沒有禁酒令，同事相聚，當然是往痛快了喝。結果，張大人喝高了，問了部下一個私人問題：「你們怕老婆嗎？」

那些酒客們雖然也喝到王三認不出王二的程度，但領導提問了，酒一下子被驚醒了，這問題可得想好了再答。他們就想啊，想，眼前的這張大人，軍人出身，大半輩子過的都是刀頭舔血的硬漢生活，在家裡肯定也是個不怕老婆的主。既然領導都不怕了，我們還還裝什麼孫子啊，都是大老爺們的，誰怕誰啊！於是，異口同聲地說：「回撫台大人，俺們不怕。」

張大人一聽這話，臉立馬就翻了：「媽拉個巴子，你們好大的膽子，連老婆都不怕，你們出來混，能有什麼出息！」說罷，拂袖而去，臉上是一副恨鐵不成鋼的失落。

犯了錯誤的大小領導們，趕緊去向張大人討教出人頭地的處世之道。怒氣已消的張大人這才津津有味地給同事們講起了他的成功經驗及為人處世的道理。

最後，張大人還意猶未盡地說：「我戰場上什麼沒見過，從來不知道什麼叫怕字。但見了夫人，我的膽子就是壯不起來……」

此時，坐在下面聽講的同學，沒有一個人敢笑話這位威風八而又畏妻如虎的張大人了。

（張曜）

2）英雄的早年都很平凡

張曜，生於1832年，祖籍浙江紹興。因為家境貧困，他一出生，過的就是到處投親靠友的流浪生活。

他的少年時代，是在江蘇蘇州的黎里鎮度過的。因為那裡有他一門生活還算小康的親戚。黎里在他的心裡就是故鄉，功成名就後，他還在那裡置辦了房產，取名「退一步處」，希望退休後能回到那裡頤養天年。但是，天不遂人願，還沒等到退休，他就倒在了工作崗位上了。故鄉路遠，魂兮難歸。他留下的那套宅院「退一步處」，現在已經成為當地一景。

少年的張曜，身體發育很快，十來歲就長得像成年漢子。張曜的小名叫阿牛，五大三粗的阿牛不愛讀書，卻對刀槍棍棒、斧鋮鉤叉、拐子流星之類的情有獨衷。好身手的阿牛身後，很快招來一大幫遊手好閒的小兄弟。阿牛哥帶著那幫兄弟，混跡賭場，到處惹是生非……乾的都是平凡的壞事，父母急在心裡，也想不出什麼好的調教辦法。

終於，有一天出大事了——

21歲那年，張曜因與人結怨，棒傷了一個賭徒，以傷害罪被人告到官府。害怕吃官司的阿牛父母，給他準備了一點路費，連路讓他潛逃了。

天地寬寬，霧氣茫茫，阿牛哥的下一站是哪呢？

你猜對了，他被父母一竿子支到遠離蘇州千里之遙的河南光州（今河南省潢川縣一帶），去投靠在那做官的表姑父。

3）從野小子到父母官

張曜的姑父叫蒯賀蓀，開始在光州治下的息縣工作，只是個普通公務員。等張曜直到息縣時，才得知姑父的工作調動了——已調任固始縣的縣令。

背靠的這棵樹更大了，張曜於心甚慰，便興沖沖地跑到固始縣去找蒯姑父。家裡突然跑來這麼個窮親戚，姑父的臉色肯定是不好看的，只是隨手施捨給他幾個銅板，讓他去街上自主創業。

（張曜的家書手札）

懶散慣了的張曜那是做生意的料啊，不幾天，手上的那點小錢就花光了。正當他流落街頭、食衣無著時，他的姑父又及時伸出援手，給他找了份臨時工的差事。

原來，當時的時局動蕩，太平軍在南方各省橫衝直撞，身處北方的捻軍也不甘寂寞，到處攻城拔寨。捻軍的襲擾行動，也直接威脅到固始縣的安危。作為一縣之長的蒯賀蓀，為了保境安民，他想到了要組建自己的地方武裝——固始縣團練。此時，身體健壯、武功超群的張曜在他心裡的份量突然重了起來。他任命張曜擔任固始縣團練的武術教官，訓練三百名團丁。

張曜不負眾望，在短短的時間內，就將那一幫曾經的社會閑雜人員，訓練成一支戰鬥力極強的戰鬥預備隊。我先想，他們的威武指數，絕不亞於當今的城管。我又想，想起了網友曾經的一句話：「給我三千城管，保准踏平日本三島。」張曜雖沒有踏平日本的豪言，但他率領的三百團丁確實守士有功，張曜率領的三百團丁，兩次擊退捻軍對固始的侵犯。張曜本人也在血與火的戰鬥中茁壯成長。

他的英勇事迹，很快傳到奉旨到河南視察、督戰的欽差大臣僧格林沁的耳朵里，張曜的機智與勇敢，讓僧格林沁對他刮目相看。在晚清，僧格林沁可是大名鼎鼎的人物，歷任御前大臣、都統，爵封親王，人稱僧王。能得到僧王的賞識，其前途無可限量也。

此時，姑父也看出張曜是潛力股，便做主把自己的女兒許配給他。蒯小姐不僅漂亮，而且還博學。正是這位亦師亦友亦妻的小表妹，在他人生最晦暗的時刻，給他點亮了一盞燈，指引他一路走向人生的輝煌。

隨後，張曜被僧王招到麾下，跟隨僧王與捻軍作戰，獲取了不少戰功。先是收復光山（就是我的老家），再戰收復息縣。僧王看好張曜，覺得他是可以重用的人才。收復息縣後，僧王指令張曜任息縣的臨時負責人。不久，蒯賀蓀調走了，僧王又把他調回固始縣，正式委任他為固始知縣。

時年23歲的張曜，成為主政一方的正處級幹部。

（張曜「目不識丁」印章）

4）目不識丁，知恥後勇

咸豐七年，十萬捻軍圍攻固始縣城，25歲的張曜臨危不亂，率眾苦守76天，終於等來援軍，保全的一城百姓的生命也財產安全。為此，僧王特意上奏朝廷為其請功。咸豐帝看完張曜的先進材料後，特封張曜「霍欽巴圖魯」勇士稱號，並被擢升為知府。此後，因基戰功卓著，於咸豐十一年，29歲的張曜擢升為河南布政使（布政使俗稱藩台，相當於負責一省人事與財政工作的副省長），一躍而躋身於朝廷二品大員的高級幹部行列。

對於張曜的這一任命，迅速遭到一些人的反對，理由是，張曜只是一介武夫，胸無點墨，不能任文職官員。意見最大的人叫劉毓楠，他是御史，御史就是專門提意見的官。他說張曜「目不識丁」，不能擔當藩台重任。皇上也覺得老劉的話有理，又改授張曜為總兵。雖說總兵也是二品，屬平調，但在那個重文輕武的時代，這樣的安排，張曜的名譽是受到不少影響的，這讓張曜心裡很不高興。也有後悔，恨年少輕狂時，不愛學習，以至受到如此羞辱。

知恥而後勇，才是漢子。張曜就是漢子，遂決心發憤讀書。

都官至二品了，沒什麼人敢當他老師。這時，張曜想到自己家裡那位博學多才的老婆蒯妹妹，回家後就要老婆教他學文化。

面對老公的求知要求，蒯老師發話了：「要我教你可以，不過，我有一個條件，你答應後才行。」張曜同學趕緊恭恭敬敬地說：「只要你教我學習文化知識，什麼條件我都答應。」

蒯老師的條件，就是要老張對她行拜師禮。這個條件看似很簡單，但在那個男尊女卑的封建時代，讓老公給老婆下跪，那可是件驚天動地的大事！沒想到，張曜同學毫不猶豫地就答應了。張曜同學馬上整理好衣服，讓老婆坐在孔子牌位前，對她行三拜九叩之禮。

此後，只要一下班，張曜同學就會按時回到家裡，接受蒯老師的批評再教育。每當他上課時玩個微信、開個小差啥的，蒯老師一發怒，他就趕緊躬身肅立接受批評，一點也不敢回嘴。

不僅學文史經哲，還學琴棋書畫，張曜同學要爭做德智體全面發展的「三好學生」。

與此同時，他還請人刻了一方「目不識丁」的印章，經常佩在身上，時刻提醒自己——要雪前恥，必須要發憤讀書。

（張曜的書法作品）

功夫不負有心人。在老婆的耳提面命下，張曜同學的學習成績進步神速，個人素質也得到全面提高。他不僅能親自草擬奏疏，而且文筆雅馴，頗有才氣，被稱為「淹通圖史，詩文皆有古法」。另外，他的書、畫也別有一格。

（漁樂圖[清]張曜）

5）人有一怕，品格自高

做人的品味格調提高了，未來的天空也會越來越高遠。

張曜的人生進取之路，是在一場又一場的戰鬥中不斷升遷的：

原來，拜師學藝的同時，張曜同學也沒有敷衍本職工作。作為一名軍事指揮員，他一直積極訓練隊伍，處處身先士卒。期間，他率部轉戰河南、河北、山東諸省，在戰爭中造就出一支王牌部隊——人稱「嵩武軍」。

光緒元年，張曜得到左宗棠的舉薦，隨他出征、幫辦新疆軍務。37歲的張曜，又重聚舊部進軍西北。

光緒二年，張曜以提督名義再次隨左宗棠出征，平定阿古柏入侵，迫使沙俄歸還伊犁。

光緒十年，張曜奉旨入關移防北路，特賞巡撫銜。

光緒十二年，54歲的張曜調升山東巡撫。

（張公祠）

在他擬升任山東巡撫的公示期間，又有人寫奏章參他「目不識丁」。這次，他不再選擇沉默，而是上書請皇上，要求面試，結果，從皇上到大臣都被他的學問所折服。

主政山東期間，張曜同學通過自身的經歷，明白了文化的重要性。為此，他舉資興建海岱書院，修葺洙泗書院。很快，他的身邊就吸引了一大批追求進步的知識分子。在山東的那些日子裡，他率眾治理黃河、興辦水利，贏得了百姓的愛戴。他在黃河大壩所植的柳樹，被稱為「張公柳」。另外，他自身為官清正，處事清廉，而且還經常把工資拿出來賑濟災民，這在晚清官場是不多見的。

就任山東巡撫五年後，張曜病逝在工作崗位上，享年59歲。

張曜去世後，被追贈為太子太保，謚勤果，入祀賢良祠。

（光緒皇帝追褒張曜的聖旨）

從目不識丁的一介武夫，到文武雙全的封疆大吏，張曜同學知道，最應感激的人，就是他老婆。

怕老婆本是事關隱私的個人小節，但，能將怕老婆怕出人生高境界的張曜，足以在怕老婆界流芳千古。可愛的張曜同學，該是我們這些心裡怕、嘴上卻死不承認怕的現代男人們學習的好榜樣。
