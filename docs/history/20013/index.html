李世民為何，要把自己的女兒，嫁給花甲之年的尉遲敬德？

2019-12-16 1,038條評論 3,114次閱讀 715人點贊

大家都知道尉遲敬德就是尉遲恭，他是李世民手下第一戰將。但是，早期的尉遲敬德，不但不是李世民的部下，反倒是李世民的仇敵。

那時候尉遲敬德是跟著軍閥宋金剛和劉武周混的，他們的對手正好是李世民集團。

後來在角逐當中，李世民勝出，宋金剛和劉武周選擇投靠了突厥，而尉遲敬德就歸順了李世民。

為了報李世民的不殺之恩，尉遲敬德曾多次為李世民建功立業，並且還在關鍵的時候，救過李世民的命。

當然，對於李世民來說，尉遲敬德立的最大的功勞，當屬玄武門政變的時候，他的表現。

玄武門政變的時候，李世民一箭把太子李建成射於馬下，死於非命。

李元吉一看大事不好，他就調轉馬頭逃跑了，李世民在追敢李元吉的時候，不小心被樹枝掛倒下馬，李元吉跳下馬來，就想奪李世民手裡的弓箭，把李世民勒死。

正是這個關鍵時候，尉遲敬德趕上來，一劍要了李元吉的命。算是救了李世民一命，後來尉遲敬德在逼宮李淵退位的時候，也立有大功。

正是尉遲敬德為李世民做了這些事，讓李世民對他格外的敬重。尉遲敬德仗著自己的功勞大，也是當仁不讓，除了李世民之外，他誰都不放在眼裡。

比如，當時的房玄齡，杜如晦等人，都被尉遲敬德敲打過。

沒辦法尉遲敬德就是一個武夫，不懂那麼多道理，不行直接就上拳頭，沒有幾個人能打得過他。因此，他的人緣也不好。

唐太宗也是看在他之前的功勞上，對他是一忍再忍。

但是，一件事的發生，讓李世民對尉遲敬德忍無可忍了。

這件事就是李世民在大宴群臣的時候，尉遲敬德看到了李道宗坐在了比較靠前的位置上，他覺得以自己的功勞，應該坐在李道宗的前面。

所以，他就罵了李道宗一番，兩個人還大打出手了。最後，李道宗被打的鼻青臉腫，而這一切事情，正是發生在李世民的眼前。

李世民忍無可忍了，要知道李道宗那可是李唐皇室，屬於皇親國戚，論關係人家比你尉遲敬德要近的多。

再說了人家李道宗也是戰功赫赫的老將，無論是資歷，還是功績都不比你尉遲敬德差。

李世民對尉遲敬德說，都說漢高祖劉邦，殺了很多開國功臣，我想他之所以殺功臣，恐怕也不都是他一方面的原因吧。

我可不希望，你們要逼我對你們下手。

李世民說完這一番話，尉遲敬德意識到了事情的嚴重性。這件事之後，他就收斂了很多，他以後做事，就再也不敢囂張了，都是夾起尾巴來做人。

正所謂打一棍子給一個棗吃，李世民找到尉遲敬德，想要把自己的女兒許配給尉遲敬德。要知道尉遲敬德比李世民大十三歲。

若是尉遲敬德接受了這門婚事，那就是典型的老牛吃嫩草。

當然了，尉遲敬德也深知，李世民這樣說，只是為了試探試探自己。畢竟自己是不是真心悔過了沒有。這個李世民還真不大知道。

所以，尉遲敬德就對李世民說了糟糠之妻不下堂的話，然後委婉的拒絕了李世民。

李世民感覺尉遲敬德這個人還真不錯，也是一個戀舊的人，也就再也沒有找過他的事。

最後，尉遲敬德也是得到了善終的，他活了73歲，在唐朝的時候，能活到這個年齡。絕對是算高壽的了。
