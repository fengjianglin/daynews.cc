鮮為人知的古埃及趣聞：幾千年前就有男士化妝和啤酒釀造了

2019-12-22 1,150條評論 2,864次閱讀 1331人點贊

綜合外媒報道，古埃及可能是人類歷史上最有趣的一個時期。它就像是一個蒙著面紗的神秘女子，讓人忍不住一窺其真容。無數的考古學家已經為我們揭示了這個王朝的歷史，但還有很多事實是我們所不知道的，現在就來讓我們揭開這個王朝的神秘面紗，了解那些還不太為人知的事實吧！

1．男女平權。說來諷刺，在現代社會，男女平等一直是女性同胞努力爭取的政治權利，但其實早在古埃及，女性的地位就已經和男性相同了。

2. 男人和女人一樣都會化妝。在古埃及，男人也會化妝，這也許是男女平等帶來的結果吧。那個時期的男人也會像女人一樣，畫著濃濃的眼線和彎彎的細眉再出門。

3. 象形文字。一說到埃及，大家就會聯想到象形文字，因為這是埃及獨有的文化特徵，也是人類歷史上最古老的字體。但考古學家只在金字塔和神廟等一些重要的建築里發現了象形文字，原因是書寫這種文字需要耗費大量的時間，所以埃及人只會在一些重要的場合和場所使用它，在日常生活中卻很少使用。

4.古埃及人發明了牙膏。每天早上醒來刷牙時，你可以提醒自己，這白白的膏狀體是古埃及人發明的，古人的智慧和科學技術的發展同樣功不可沒，為我們現代人提供了便利和健康。

5.除了人體木乃伊，埃及人還製作動物木乃伊。在埃及歷史上，一些位高權重的大人物死後會被製作成木乃伊，以便讓靈魂得到永生。有時候，這些大人物飼養的寵物也會被製作成木乃伊，來陪伴自己的主人。

6.埃及豔后克利奧帕特拉七世（Cleopatra）並非埃及人。托勒密王朝最後一位統治者——埃及豔后克利奧帕特拉七世可以算是埃及歷史上最著名的人物，但事實上，她身上流的是希臘北馬其頓名族的血液。

7. 金字塔內的溫度常年維持在20°C。埃及大部分地區屬熱帶氣候，但幾千年來，無論周圍有多炎熱，金字塔內的溫度常年維持在20°C左右， 令人驚奇，我們很好奇在古埃及，人們是怎麼做到的？

8. 那時就有啤酒了。古埃及人是一群充滿智慧、有科學頭腦的名族，但他們也和我們一樣，偶爾會喝自己釀造的啤酒。埃及壁畫中就記錄了埃及人釀製葡萄酒的過程。本文選題：金戈帝企鵝。編輯：racheldd
