蜀漢滅亡後，司馬家如何取得巴蜀人心

2019-12-19 1,128條評論 2,528次閱讀 722人點贊

公元263年八月，魏國權臣司馬昭開展滅蜀戰略，魏軍自洛陽出發分兵三路大規模伐蜀。

征西將軍鄧艾率兵3萬餘人，由狄道出發進攻姜維駐守沓中的蜀軍主力，雍州刺史諸葛緒攻武都攔截姜維退路，鍾會率領十多萬魏軍主力攻打漢中。

鍾會率軍取得漢中後，南下進攻蜀漢劍閣天險，姜維集中各路兵馬拚死抵抗，鍾會久不能克，戰爭一時陷入僵局。

鄧艾以萬餘精兵偷渡陰平，進入川蜀平原地區，擊敗諸葛亮之子衛將軍諸葛瞻，兵臨成都。劉禪震恐，再三考慮下，出城投降，立國四十二年的蜀漢政權滅亡。

司馬昭自信滿滿，策劃滅蜀三年之後，以巴蜀之地長江上游的有利地勢，水陸並進而滅吳。不過滅吳計劃還沒來得及實施，司馬昭就於兩年後病死，同年其長子司馬炎篡位稱帝，建立西晉。

司馬炎皇帝當的很容易，祖父、伯父、父親為司馬炎做足了鋪墊，滿朝都是司馬家的人，魏帝曹奐在朝廷和軍隊中也沒有任何力量支持，完全是司馬氏的傀儡。

在內部新興政權沒有完全穩定的情況下，是不會輕易揮師伐吳的，這個道理就看後世的前秦大帝符堅在淝水之戰戰敗後是什麼下場就明白了。

晉武帝需要鞏固帝位，司馬家的政權需要時間平穩過渡，制定禮儀、法律和官制等，新取得的蜀地需要消化吸收，自此滅吳戰略被一直耽擱下來。

安撫蜀國故土

攻滅蜀國後，原本魏軍在成都緊守軍法，鍾會意圖發動叛亂被將士誅殺後，軍隊無人約束。魏軍開始在成都燒殺搶掠，甚至導致大批蜀國投降的皇族和官員被殺。原蜀漢太子劉璿、左車騎將軍張翼、漢城護軍蔣斌及關羽家族等。

身為監軍的衛瓘先前和鍾會一起陷害鄧艾，現在怕牽連自己乾脆直接滅口，將運往洛陽路上囚車上的鄧艾一併處死，兩個滅蜀元勛結局如此，不僅令人唏噓。數日後，衛瓘回到成都約束諸將，成都之亂方平。

  • 優待上層階級

面對原蜀國各方勢力，掌權者司馬昭採取懷柔政策。將後主劉禪等宗室遷移到洛陽，斷絕與蜀國故土的聯繫，加以優待重賞，封為安樂縣公。

《三國志•後主傳》：食邑萬戶，賜絹萬匹，奴婢百人，他物稱是。子孫為三都尉封侯者五十餘人。

另外將蜀國舊臣尚書令樊建、侍中張紹、光祿大夫譙周、秘書令郤正等人通通封為列侯。

諸葛亮、蔣琬、費禕這些聲望顯赫的元老重臣後人給予豐厚的獎賞並酌情錄用。

傅彤之子傅僉戰死陽安關城，諸葛亮之子諸葛瞻戰死綿竹，司馬炎都下詔褒美，讚揚其忠義。

  • 削弱巴蜀軍事基礎

蜀國投降軍隊數量高達十萬，西晉朝廷將其全部遣散，當然給了不少遣散經費，下崗回家當農民。

對於益州的重要文武官吏和世家大族勢力，為防作亂將其全部強制遷走，並給予免除二十年田租的特別優惠。蜀國總人口不過九十四萬，這三萬家從益州遷到河東和關中的人口是蜀地的中心力量，其中也包括家眷、奴婢、部曲，可以說是連根拔起。

《華陽國志》：後主既東遷，內移蜀大臣宗預、廖化及諸葛顯等並三萬家於河東及關中，復二十年田租。

通過這些基本措施來降低將來川蜀發生兵變民變的可能性，另外減免賦稅來取得民心，加強對川蜀的控制。「特赦益州士民，復除租賦之半五年」。
