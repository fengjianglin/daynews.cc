孫中山先生：赫赫威名揚天下，誰知他的兒時是這樣度過的？

2019-12-16 95條評論 3,970次閱讀 408人點贊

武昌起義爆發，中華民國臨時政府成立後，總統一席虛位以待。原定由「反正」歸來的袁世凱出任。為何當孫中山海歸後，絕大多數反過頭支持孫中山？

我們共同從孫中山早年生活中尋找答案。

孫中山早年生活

1866年，11月12日。孫中山出生於廣東香山縣翠亨村。其父名為孫達成，以務農為生，兼做更夫。孫中山姐妹共6人，其中二弟、三妹不幸早夭，除去孫中山外，還有兄長孫眉，姐孫妙茜，小妹孫秋綺。

孫中山，幼名為帝象，稍長後，取名為文，字德明，號日新。1886年改號為逸仙。（方言發音與日新諧音）1897年，孫中山在日本進行革命活動。有一次，途經一個名為「中山」的旅館，於是化名為中山樵。在辛亥革命後，人人皆稱他為「中山先生」。

自幼出生於農村，生活於農村，家庭條件相對於貧窮，乃至於10歲時，才進入私塾學習，受到了一些基本的傳統教育。

廣東地處南國，那裡一直流行著一種出樣謀生的風氣。孫中山的大哥孫眉不堪忍受家裡貧困的家境，獨身前往檀香山，幾經艱辛，最終成為當地赫赫有名的華僑資本家。

1878年5月，當時的孫中山先生未滿12歲，母親被迫送孫中山前往檀香山就養於大哥孫眉。這件事在孫中山此後一生影響深遠。孫中山先生曾回憶道：」到此時，『始見輪舟之奇，滄海之闊，自是有慕西學之心，窮天地之想』「。

孫中山最初到檀香山後，被安排於大哥店中當一名店員，在閑暇之際，他總是翻閱各類書籍，並且在短短數月內，學會了當地語言。

大哥孫眉發現孫中山的好學，而且對事物有著強烈的求知慾；以為他是孫家不可多得的人才，於是，次年，將孫中山送往當地最好的學校火奴魯魯一所男子中學就讀。孫中山從此接受了系統的西方資本主義文化教育。所學的教材不僅有西方政治學說、自然科學的基礎知識，還有《聖經》等科目。

孫中山在學校內勤奮好學、不懂就問，並且攻讀中外書籍，很快成為學校內優等生。在課下之餘，孫中山除了自修中文外，尤其喜愛讀華盛頓、林肯等資產階級革命先驅傳記，對歐洲及美國的民族民主革命領袖極為推崇，產生要以之為師的「大丈夫”志向。

當孫中山求學時，夏威夷人民正在進行反對美國殖民地鬥爭，人們在「夏威夷是夏威夷人民的夏威夷」的鬥爭口號鼓勵下，為反對美國吞併夏威夷而奮勇鬥爭。正處于思想形成期的孫中山，由此聯想到中國所遭受到的帝國主義侵犯，漸漸萌生反對殖民統治，要求民族獨立的思想。

在夏威夷讀了四年書後，孫中山成為了一名基督教信徒。孫中山的大哥孫眉聽說後氣急敗壞，擔心孫中山成為一個數典忘祖的「假洋鬼子」，於是將孫中山送回家鄉。不過已經接受四年西方思想的孫中山，回到故鄉後對於故步自封、落後的封閉生活無法接受。

孫中山17歲那年，趕赴香港學習，起初進入拔萃學院讀書，同年底與陸皓東等人一起進入基督教，成為基督教徒。次年便轉入中央學院。（香港殖民當局辦的中等學校）

大哥孫眉聽聞孫中山受拜於基督教，氣急痛心，認為孫中山的行為屬於年輕人「精力過剩」的胡鬧。遂採用中國傳統「早婚」法，設置一個「老婆孩子熱炕頭」的局面，妄想用來拴住孫中山的「野心」。孫眉彙集數百元回家，並且囑咐家中父母，為孫中山選擇一位妻子。

1885年5月，19歲的孫中山奉命回鄉，父母選擇的妻子是同鄉一位商人之女，名盧慕貞。雖然孫中山在家停留數月，但心仍留在香港，同妻子講述後，妻子支持孫中山的選擇。8月，孫中山在赴香港，回到學校，86年修完所有課程。

1886年~1892年，孫中山學醫5年，並且在高人幫助下研讀了中國部分古典書籍。

27歲時，孫中山已受良好的中國教育和豐富龐雜的西方教育，並掌握了一門現代化的職業技能。實話實說，當時的孫中山已經具備西裝革履般生活的條件。但是，在香港時，孫中山對政治的興趣卻不斷地發展，革命意識也在日益增強。

他既從家鄉的鄉俗文化中繼承了太平天國的革命傳統，又從師友處感染了改良主義的熱情。雖然在改良與革命中游移，但對清政府腐敗統治的不滿卻是一致的。

1894年夏，孫中山和同伴不遠千里求見北洋大臣李鴻章，結果卻被拒之門外。他從此走上以武裝推翻清王朝統治為目標的革命道路。

孫中山被李鴻章拒絕後，開始從上層鬥爭轉移到下層鬥爭，華僑、會黨，成為孫中山的第一選擇。於是，他去往了夏威夷，因為那裡不僅僅是他青年時期學習、生活的地方，有一定的人際關係；同時更依賴於大哥孫眉在夏威夷的影響力和金錢。

1894年10月，孫中山胸懷長遠抱負，從上海輪渡夏威夷。但有一個殘酷的現實擺在孫中山面前：雖然華僑關心祖國，但不關心政治。他們安於現狀，唯恐家鄉親屬受到牽連。所有人對他都是敬而遠之，甚至有一些人對他冷嘲熱諷，嗤之以鼻。

但孫中山是何許人也？「咬定青山不放鬆」，打定主意，不畏艱難。經過數月百折不回的宣傳，最終吸引20多位會員加入革命黨，組成興中會。其成員多數為華僑、商人、工人。核心成員有：大哥孫眉、商人鄧蔭南、夏威夷政府譯員李昌、銀行經理何寬等人。

直到1895年10月，廣州起義時，以擴大至100多人。孫中山整個革命活動，這是他重要的一個據點。夏威夷興中會是中國近代史是第一個資產階級革命小團體，中國資產階級民主革命從此進入新的時期。

次年1月，孫中山回到香港，創建香港興中會總機關；對外採用「乾亨行」一名為掩護。香港興中會早期領導人楊衢雲在香港擁有比較優越的社會關係，不僅方便集資、徵集軍火，而且領導著香港一個愛國團體——香港輔仁文社。所以，初期，楊衢云為會長，孫中山則實際發揮領導作用。

1900年1月，孫中山正式成為香港興中會會長。

興中會成立後，孫中山開始組織暴力革命工作。孫中山作為中國民主革命的一位先行者和傑出領袖，從此赫然出現在中國以至世界的政治舞台。
