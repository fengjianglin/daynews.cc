記者問林洙：如何比較你自己跟林徽因？林洙說：跟死人吃什麼醋

2019-11-30 0條評論 17次閱讀 0人點贊

摘要

近年來，大陸不但掀起張愛玲熱，也掀起林徽因熱，林徽因這個民國美貌才女廣受世人關注，而她丈夫梁思成的後妻也寫了一本書《梁思成、林徽因和我》，此書沾了林徽因的光，大受歡迎。在林徽因病故七年後，梁思成娶清華大學女職工林洙為後妻。林洙出生於1928年，她並非是未婚女人，而是離異女人。

近年來，大陸不但掀起張愛玲熱，也掀起林徽因熱，林徽因這個民國美貌才女廣受世人關注，而她丈夫梁思成的後妻也寫了一本書《梁思成、林徽因和我》，此書沾了林徽因的光，大受歡迎。

在林徽因病故七年後，梁思成娶清華大學女職工林洙為後妻。林洙出生於1928年，她並非是未婚女人，而是離異女人。她和前夫程應銓育一兒一女。1958年，林洙與程應銓離婚。1962年，林洙與梁思成結婚。

林洙比梁思成小27歲，比梁思成和林徽因的女兒梁再冰才大一歲。林洙和林徽因同是福建人，林徽因給林洙補習過英語，還在林洙第一次結婚時送了她一份厚禮，並且借錢給她結婚，還不用她還。

2011年左右，記者採訪林洙，提出了一個問題：「如何比較你自己跟林徽因？」

這個問題真是不好回答，憑心而論，無論從出身、容貌、才學、成就、名氣，林洙哪一點都比不上林徽因，可是林洙一點都不自卑，她不屑地說：「「跟死人吃什麼醋，夠不著她！」

這個回答很巧妙，一口把記者的問題給堵了回來，等於是根本不拿自己跟林徽因比較，可是她稱林徽因為「死人」，這實在是太不尊重林徽因了。

雖然林徽因已經病故多年，但是不能稱她為「死人」吧。就如誰也不會稱自己或朋友的已故親人為「死人」，你聽有人說過「我爺爺這個死人如何如何嗎？」也不會有人對別人說：「你奶奶這個死人如何如何？」

中國人為尊者諱，不會自己或朋友的逝世親人為「死人」，而稱為「過世的人」、「不在的人」、「逝世的人」，甚至有人會用「老了」這個詞來代替「逝世」這個詞。

只有在罵人或貶低別人時才會「死人」這個詞，而這樣說，一定和對方吵翻臉了，所以不顧禮貌。

林洙稱林徽因為「死人」，實在是太不尊重林徽因了。如果她為了自尊不想說自己比林徽因差，也可以這樣說「林徽因是我所尊敬的長者，她已經病逝這麼多年了，何必比較我和她呢？」或者說「我跟她各有長處，沒法比較！」

作為後妻，對前妻就算不尊重，也不應該貶低，何況林徽因曾經是林洙的長輩，竭盡全力幫助過林洙。而林洙又利用林徽因的名氣寫書賺了錢，無論如何，她都應該尊重林徽因，不應該在回答別人問題時稱林徽因為「死人」！
