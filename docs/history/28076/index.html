秦始皇的偉大發明，上比兵馬俑下比萬里長城，又一驚世駭俗之作

2019-12-23 1,896條評論 4,914次閱讀 1587人點贊

　　古往今來，我們所熟知的秦始皇時期的「車同軌」指的就是車輪之間的距離是統一的。近期，考古研究者在河南南陽的山區地帶發現有秦朝遺留的鐵路軌跡。經考證發現鐵路運行原理與現代鐵路相似，而且是複線，需要牲畜拉車。這一偉大發現堪比兵馬俑的問世。

　　

　　我國地大物博，物產豐富，尤其樹木森林遍地都是，這為秦朝鐵路的修建提供了物質條件。鐵路的材質，古今各有不同。現代多為鋼軌，古時為木材。故而，秦始皇修的鐵路，採用優質的木材加之防腐的加工，保存完好。不過枕木的保存就不容樂觀了，歷經千年僅可看出大概模樣。

　　

　　路基也是鐵路一大亮點，這條鐵路的路基格外厚實，經考察枕木的木製較軟，為了更好地與路基吻合，古代工人有意而為之，為了行駛過程中的安全平穩、減少波動。另一個驚人的發現就是兩個枕木之間的距離正好是馬步子的合拍，馬匹在軌道上只得飛快奔跑，不能停歇。只有到了專門的車站才可以停下來休息和進食。這樣的設計大大提高了馬兒勞動的效率，節省了時間，無疑是一偉大壯舉啊。

　　

　　那秦始皇為何要修鐵路呢？有人認為是為了滅掉楚國，六十萬大軍的糧草補給需要鐵路運輸。也有人認為這不單是滅楚的原因，而是早有計劃的目標。這樣來看的話，秦始皇修的弛道應該就是鐵路。由於馬兒在上面飛馳，故而稱為弛道。那麼可想而知秦始皇在兩千多年以前就修建了一個「鐵路網」。

　　

　　秦時的車輪寬度一直就是人們探討的問題，到現在可以迎刃而解了。車輪的設計就是為了行駛中的車子在普通的道路和鐵路上都可以跑，而現在的車子則不具備這一條件。

　　

　　但是秦時的鐵路網為何沒有延續下去呢？大多數的觀點認為是由於秦末戰亂，馬匹減少、鐵路廢弛，戰亂中的破壞，久而久之變成了普通的道路。到漢朝時政治、經濟各方面的原因也使重修鐵路無意義，慢慢地秦始皇的鐵路就消失了。

　　

　　到現在為止，人們只有在河南南陽山區發現了秦朝的鐵路遺址，而秦朝的「鐵路網」是人們完美的構想。按常理推斷在荒無人煙的交通之地也可能找到鐵路遺址。如若「鐵路網」被證實它的存在，那將是人類歷史上的又一重.
