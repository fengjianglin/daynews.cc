18歲嫁35歲丈夫，與丈夫恩愛44年，離世僅4個月，丈夫另娶

2019-12-14 861條評論 2,531次閱讀 104人點贊

亂世中，紛飛不定的除了風沙以外，還有數不清的感情。有人居無定所，在世間搖擺不定，也有人孜孜不倦一生，只為尋找一處避風港。人人自危的年代，想要找到一個心貼心的愛人實屬不易，也彌足珍貴。這樣的感情值得每個人去等，更值得每個人用心守護。

民國時期，攜手白頭的名人其實不多，能夠在亂世中從一而終，要克服的困難實在數之不盡。所以也更加的讓人羨慕稱讚。今天我們要講的就是一對伉儷情深的夫妻，他們風雨兼程，度過一生中最好的年華。這對夫妻正是大名在外的將軍李宗仁和他的妻子郭德潔。

郭德潔的生平

能夠比肩李宗仁，郭德潔自然也非池中之物。她出生於1906年，原名叫做儒仙。郭德潔的家境還算不錯，父親郭六雖然不是什麼文人雅士，可是有一技之長，他是當地出名的泥瓦匠，做工精細，很得客人的稱讚好評。而且為人忠厚老實，不貪圖便宜，許多年輕的小夥子都樂意在他那裡學習手藝。所以郭家的經濟條件很不錯。

郭六是生活中的小人物，勝在有一顆事事做好的心。郭德潔很大程度上就遺傳了父親這樣的特點。郭德潔心高氣傲，任何一件事都爭當第一。並且當第一也不是說說而已，她從小就聰明伶俐，八面玲瓏，十里八鄉見到了，誰不誇一句這個女娃是個乖巧的？

郭德潔另一方面是個執著的人，在讀書風氣開放的時候，她說什麼都要去學校念書。可是雖然開放了女子讀書這樣的先河，老一輩的思想可不是說改就能改的。郭德潔的想法遭到了父親的堅決反對，這樣的說法傳到鄰居那裡，甚至還遭到了嘲笑！

可郭德潔哪裡是被這點挫折就打敗的人？不顧世俗的輕蔑，她還是毅然決然的奔赴了學校念書，她要爭做新時代的女青年。

郭德潔巾幗不讓鬚眉，成了鄉鎮周圍的一段故事，有人議論有人羨慕，郭德潔在給自己爭取光明的未來啊！

和李宗仁的相愛

李宗仁35歲那一年，是馳騁沙場，戎征天下的大將軍。而緣分就是那樣的巧妙，兩個人看起來絲毫沒有聯繫的人卻可以相遇相知。

那一年是1921年，李宗仁帶領了十多個連隊在玉林地區休整身心。準備靜待時機給予敵人致命一擊。可他沒有想到，會有一個女子給他的心上來一個「致命一擊」。

李宗仁有一天正在城樓上視察敵情，郭德潔恰巧騎著自行車施施然的經過街頭。李宗仁被眼前的少女抓住了眼球，如果這種感覺沒錯，他想他對郭德潔是一見鍾情了。

李宗仁拜託了許多友人引見才終於和郭德潔正式認識。而郭德潔就彷彿是李宗仁心頭上的白月光，美好得不切實際。在知道了郭德潔巾幗不讓鬚眉的事迹後，更是對她生出一抹欣賞之情。當然李宗仁這位威猛大將軍，也是極具吸引力的。18歲的郭德潔到底還是個小姑娘，對這樣的任務，敬仰之情莫不足以。

他們互生愛慕情愫，幾次約會下來，就確認了戀愛關係。

他們是志同道合的戀人和友人，自然不會顧忌年齡的差距，而且兩個人的思想都是先進並且重合的。一切看起來水到渠成，兩個人的婚姻也是錦上添花。可惜浪漫的愛情背後卻還有一點美中不足，那就是李宗仁家中其實還有一位大夫人，如果要結婚，那麼郭德潔就只能做妾！

郭德潔對第一名的名號尚且執著，會答應在婚姻上做第二人嗎？出乎意料的是，郭德潔還當真願意嫁給李宗仁做妾！為了愛情，她犧牲了自己的驕傲。

郭德潔的婚後生活

嫁給李宗仁之後，婚姻生活也是和和美美的，大夫人是一名裹了小腳的女性，雖然被舊社會影響過，但是不妨礙大夫人做事雷厲風行。李家的大小事務基本上都有大夫人一手操持，和郭德潔一對比，似乎郭德潔在家中的地位不是很高。

沒有夫人的頭銜和權力，可是郭德潔有李宗仁的寵愛啊。18歲和李宗仁步入婚姻的殿堂後，郭德潔就決定要一直守護在這個男人的左右。所以在郭德潔完成了自己的學業後，她的生活就是以李宗仁為中心的。

李宗仁要去前線打仗，那郭德潔就跟著去照顧他，做他的賢內助。李宗仁只管在戰場上浴血奮戰，身後的大小事宜都讓郭德潔處理好了。不僅如此，郭德潔還細心的照顧軍旅中的同伴，貼標語、救治傷員、慰問軍隊，這些看起來微不足道的小事都是郭德潔生活中的主要事情。

恰恰也是因為這些小事，為李宗仁贏得一大片的軍心、民心。郭德潔就像是他的幸運女神，帶著郭德潔在外征戰的幾年，李宗仁的事業蒸蒸日上。

同時，郭德潔還大力開展自己的慈善事業，親眼見過戰爭現場的她更加明白戰爭的殘酷。所以她願意花時間和金錢投入慈善的行業。她出資修建了許多的學校和兒童福利院，收留無家可歸的孩子們。

生命的沒落

新中國成立，李宗仁的勢力褪去，他不得不開始流亡的生活，在美國和香港兩個地方來回奔波。此間十幾年，郭德潔都一直陪伴在他的身邊。也正是有了郭德潔的堅守，他們的生活才不算太苦。女人有深入骨血的浪漫情愫，她陪著李宗仁賞花逗魚，偶爾還邀請朋友打一會兒麻將，生活雖然不如從前，但是還算悠然自得。

只是可惜了，天妒紅顏，郭德潔的美好沒有能夠持續到老。1964年，郭德潔被查出患有乳腺癌。因為郭德潔擔心做手術影響美觀，並且還可能讓丈夫受到周遭人的議論，她的病情一拖再拖。病魔無時無刻在折磨著她，她明白自己的時日不多，唯一的心愿就是能夠落葉歸根，回到祖國的懷抱。

李宗仁雖說對回國還有一些顧慮，但是為了滿足妻子的願望，還是帶著她回到了家鄉。他們的回歸受到了人民的熱烈歡迎，看著祖國一片欣欣向榮的狀態，郭德潔是心生驕傲。祖國在一天天變好，她的身體卻在一天天變差。

在故土上支撐了6個月，她還是與世長辭了。

李宗仁的續弦

郭德潔的離世，對李宗仁來說是個致命的打擊。郭德潔來到他的生命和離開他的生命，都是沒有辦法忘卻的一瞬間。她去世的幾個月，李宗仁整日鬱鬱寡歡，對周遭事物完全提不起興趣。是身邊的朋友不願他繼續這樣的狀態的生活，才張羅著給李宗仁另外介紹一位妻子。李宗仁左挑右挑都沒有心宜之人，直到胡蝶的私生女胡友松出現在眼前。

胡友松倒不是貪圖名號，她是真的仰慕這位抗日英雄才答應和李宗仁結婚的。李宗仁二話不說，就同意和胡友松結婚。

他們結婚三年後，李宗仁便去世了。

和郭德潔相守44年，愛情早就變成了親情。妻子離世四個月後，李宗仁或許是再次遇見了自己的愛情吧！

郭德潔嫁給李宗仁，和他並肩度過繁華的歲月，也一起熬過了寂寞的歲月。每一幀的畫面都是兩個人幸福的面孔。我想，李宗仁的續弦或許還是郭德潔最想看到的，她希望丈夫沒有她依舊能夠快樂的生活。感情往往就是不計較得失，不計較背叛。感情最在乎的是對方是否後悔，是否永遠幸福！
