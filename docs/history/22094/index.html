從門閥鬥爭的角度，看桓溫如何通過“庚戌土斷”來鞏固自身權勢？

2019-12-18 1,665條評論 3,180次閱讀 2102人點贊

東晉中葉，權臣桓溫在破除中樞司馬昱等勢力對自身的限制後，終於得以出兵北伐建功，讓自身在東晉朝中的權勢進一步加強。而桓溫除了靠軍事上建功鞏固聲望之外，在兩次北伐之後還主持了著名的「庚戌土斷」，更加促進了自己在東晉門閥政治格局穩定發展時期的地位鞏固。那麼，桓溫是如何通過「土斷」改革，來實現自己鞏固權勢的目的呢？

何為「土斷」，從東晉的「僑置郡縣」說起

東晉時期的「土斷」改革是為解決自「永嘉之亂」後，北方人口南遷後「僑人」和「僑置郡縣」等帶來的各種弊端，那麼先來看東晉「僑置郡縣」所帶來的影響。

  • 僑人、僑置郡縣

晉自中原喪亂，元帝寓居江左，百姓之自拔南奔者，並謂之僑人。皆取舊壤之名，僑州郡縣，往往散居，無有土著，准其暫挾注本郡，免除賦役。——《隋書•食貨志》

西晉末期，五胡之亂導致大量北方人口南遷，南遷人口到達南方後成為「僑人」。東晉朝廷為了安置這些人口，按照北方舊地名稱重建州郡縣，這便是所謂的僑置州郡縣。一州所涉角度較大，我們只探討東晉乃至南朝頗具影響的僑置郡縣部分。因為是在南方「僑置」舊郡縣，這部分僑置人口並無真實土地，東晉給予免除「賦役」的優撫。而這部分僑人隨著在江東定居多年，部分已經擁有土地，卻依舊享受免除賦役的優撫。並且，僑置郡縣僑人當然並不能與原本北方郡縣人口相當，會存在「十家五落，各自星處，一縣之民，散在州境，西至淮畔，東屆海隅」的現象，不利於轄區郡縣管理。僑人、僑置郡縣所產生的弊端逐漸開始顯現。

不光如此，最初南遷人口因無土地，最好的辦法就是去依附豪強士族。並且南來人口本來就是軍事、生產等方面的重要優勢，因此南方豪強士族也會利用權勢逼迫流民依附。《隋書•食貨志》有記「都下人多為諸王公貴人左右……皆無課役。客皆注家籍」，可見當時豪強士族對於南來人口的強佔現象較多。南方士族豪強借僑置之機強迫僑人依附，還有強佔流民不予登記戶籍的現象。東晉給予僑人定為白籍戶口，南方原來人口為黃籍。可以南方士族豪強利用權勢，逼迫、強佔僑人為「家籍」，或者故意隱瞞人口不予上報戶籍，也就既擁有了大量人口，又不用替所佔人口承擔賦役。而且值得思考的是，東晉僑姓士族為鞏固自身在江左的物質基礎，當然也會存在圈佔土地現象，江東本土士族更不必說，世代經營早就擁有大量土地。而經營土地農莊，需要大量人口，面對南來僑人，強佔人口、隱瞞戶籍現象自然更多。

  • 土斷：以土著為斷

時王公庶人，多自北來，僑寓江左，今皆以土著為斷，著之白籍。——《資治通鑒•成帝紀》

自爾漸久，人安其業，丘隴墳柏，皆已成行，雖無本邦之名，而有安土之實。——《范寧傳》

白籍僑人依靠東晉免其賦役的優撫，逐漸在江東穩定下來，並且北方失地難收復，南來人口也早就沒了北返之心，因此僑人也逐漸開墾土地定居下來，這樣就出現白籍之人擁有土地卻不承擔賦役的問題。也就是這種情況，范寧才有「無本邦之名，有安土之實」的說法。既然有了安土之實際情況，那麼東晉朝廷自然不會放棄這部分有地人口的賦役，因此才推出「土斷」改革，改原有白籍為黃籍，也就是取消了定居僑人的免除賦役的優撫。這種多出的賦役當然對朝廷有利，以此統治者也樂見「土斷」實施。也因此晉成帝咸康年間推行土斷之策，實為推出增長國力考量。

永嘉之後，北人南渡

土斷歸根結底是解決南來僑人戶籍問題，讓原本逃脫東晉賦役的人口承擔賦役。但是「土斷」之策固然能夠解決庶人白籍僑人的問題，但卻面對一個棘手的問題，那便是東晉士族豪強所佔「家籍」或無籍人口難以按照庶人白籍土斷之法解決。山遐曾經擔任餘姚令，為解決士族侵佔人口問題，「繩以峻法，到縣八旬，出口萬餘」。因此山遐遭到了當地士族豪強的反抗，甚至誣陷山遐，最終山遐被免官。由此可見當時東晉士族「挾藏戶口」問題的棘手性，結合東晉當時門閥政治看，僑姓士族和江東本土士族都是強佔人口的參與者，而在東晉朝堂佔據主要地位的門閥士族，自然不願意「土斷」改革奪走門下所佔人口。如山遐雖為高門之後，但山氏並未在江左顯赫，因此也難以撼動東晉門閥政治下各門閥士族的利益所在。

而士族門閥所強佔人口、隱瞞戶籍情況太過嚴重，已經嚴重影響到東晉國力。從山遐施行土斷「八旬」幾個月時間便「出口萬餘」，足以看出當時東晉門閥士族侵佔人口的嚴重程度。可是這種由僑人和僑置郡縣導致的東晉內部弊端，並非那麼容易解決，所涉及到各東晉門閥政治格局下的門閥士族，山遐解決不了，需要一個具備強勢手段的人物才行。東晉中葉，這個人出現了，便是桓溫。桓溫破除司馬昱對自己的限制，得以執掌東晉朝政。並且桓溫依靠荊州之兵兩度北伐，在東晉朝中威望日盛，而且時機正好處於桓溫壓制東晉內部其他反抗力量的關鍵節點，因此桓溫在這個時機也就是興寧二年三月初一推出「庚戌土斷」。庚戌土斷得以推出，雖然結果並非多麼成功，但卻在門閥政治鬥爭的背景之下，幫助桓溫更加鞏固其自身地位。這個過程中桓溫自身的實力是基礎，利用土斷打擊抵抗自己的東晉內部門閥士族力量，也依舊是桓溫推行土斷的重要目的。

「庚戌土斷」所處的東晉門閥政治鬥爭背景

三月庚戌朔，大閱戶人，嚴法禁，稱為庚戌制。——《晉書》

庚戌土斷是在東晉哀帝興寧二年三月初一，因是庚戌日才有庚戌土斷之稱。這段時間內桓溫把持東晉朝政，處於桓溫前兩次北伐與第三次北伐之間。桓溫通過討伐前秦和大破姚襄兩次北伐，權勢進一步得到確立。在時間節點上，第二次北伐在永和十二年即公元352年，第三次北伐在晉廢帝太和四年即公元369年。桓溫主持庚戌土斷是在興寧二年即364年，處於桓溫正依仗其北伐之功運營在朝中勢力，奪取其他各門閥所佔據中樞、方鎮權力的關鍵時期。

當時雖然桓溫正處得勢之時，但卻並非如之後徹底掌控東晉內部勢力。在桓溫第三次北伐之前桓溫通過帳下郗超的郗氏關係得以控制京口勢力，在第三次北伐失敗後，桓溫逼迫豫州袁真叛變得以控制豫州勢力。由此可知，庚戌土斷推出之時，東晉方鎮勢力中有著非常重要影響的豫州和京口，尚未被桓溫控制。再結合後期第三次北伐失敗，桓溫覺得聲望受損，使用郗超之謀行皇權廢立之事，以圖來彌補自身聲望，可以推知在庚戌土斷推行時，桓溫所面對的東晉內部反抗勢力依舊存在。在奪權關鍵時期，推行土斷政策，應當並不僅僅是為了提升東晉國力的考慮。

桓溫主政、北伐期間，正是東晉門閥政治經過王導、庾亮等人發展，步入穩定發展之時，東晉內部經過永和政局門閥勢力能夠得以穩定發展。桓溫通過兩次北伐建功，已經佔據東晉中樞和除豫州、京口之外的荊州、江州等方鎮力量。未能佔據的豫州和京口，雖然明面上遵從當時執掌朝政的桓溫，但卻是東晉內部其他門閥士族的勢力所在。

豫州經過陳郡謝氏謝奕、謝萬、謝尚等人的多年經營，讓陳郡謝氏得以位列門閥士族之列，可見陳郡謝氏在豫州影響頗深。之後庾亮、庾翼等人佔據豫州，也讓豫州頗具庾氏影響。而當時豫州刺史袁真雖然表面上以桓溫為尊參與北伐，但豫州真正勢力卻並未完全由桓溫掌握。也正因如此，在第三次北伐失敗後，桓溫將北伐失敗歸於袁真，逼迫袁真叛亂，最終才得以讓譙國桓氏子弟桓伊任豫州刺史，豫州方鎮才真正落入桓溫之手。京口之地，經由東晉初期郗鑒及後世郗氏子弟經營多年，郗氏在京口影響依舊很深。而京口作為東晉內部能夠威脅上游荊州的下游軍事力量，當時高平郗氏郗愔為徐、兗二州刺史掌控京口勢力。郗愔當時聽從了在桓溫帳中任事的兒子郗超建議，不予反抗當時已經成勢的桓溫，自請出任閑職。桓溫遷郗愔為會稽內史，自己領徐、兗二州刺史，掌控了京口之兵。

從桓溫在後期費盡心機謀取方鎮勢力的情況看，在興寧二年期間主持的「庚戌土斷」就並非那麼簡單了。當時依舊存在桓溫並未掌控的方鎮勢力的東晉內部局勢下，可想而知對抗其掌權的門閥力量也依舊存在。桓溫執政末期王、謝等門閥力量的抵制，更能佐證這種判斷。所以在此種時機之下，桓溫不求軍事建功，反而推行政事，主持庚戌土斷之策，除增加朝廷賦役之外，應當還包含壓制抵抗門閥力量、鞏固自身地位之目的。

桓溫推行「庚戌土斷」的謀算：震懾門閥力量

桓溫推行庚戌土斷，其內容當然也主要集中在對士族豪強所強佔的僑人戶口問題上。土斷政策的實施，在桓溫當時有兩次北伐之功，並且掌控中樞和東晉大半方鎮權柄的基礎上，當然不會遇到跟山遐所面臨的問題。桓溫推行土斷之策，極大地增加了東晉財力，為桓溫第三次北伐提供了有力的國力保障。後世劉裕在推行「義熙土斷」時，評價桓溫「庚戌土斷，以一其業，於時財阜國豐，實由於此」，可見當時桓溫土斷政策取得了一定成效。但是，結合東晉末期以及南四朝時僑人、僑置郡縣問題依舊嚴重的情況看，桓溫乃至劉裕的土斷改革，應當並未徹底斷絕因僑人僑置問題導致的東晉內部弊端。

桓溫的「庚戌土斷」雖取得一定成績，卻並未徹底解決弊端，但從當時東晉門閥政治穩定期的背景看，「庚戌土斷」應當是桓溫篡權之路中的其中一環，也可以說是眾多謀算中的一項。結合桓溫在司馬昱之後得以掌權後的崛起之路，以及其三次北伐其間種種謀算看，桓溫推行「庚戌土斷」的作用，應當與興皇權廢立之事作用相似。只是，皇權廢立之事發生在第三次北伐失敗後，枋頭之敗打破了桓溫先北伐建功，提升朝中聲望，再圖謀權篡位的謀算，因此皇權廢立之事當是不再隱忍的震懾之舉。而相應的「庚戌土斷」發生在桓溫第三次北伐之前，桓溫借前兩次北伐功績，還當是志得意滿之際，施行土斷之策，當是提升聲望之舉。而從土斷所打擊的門閥士族利益角度看，庚戌土斷應當也同樣是對東晉內部主要勢力——門閥士族的震懾。

其實桓溫的謀算並不僅限於此，也並非始於「庚戌」。僅從桓溫謀算豫州、京口方鎮的角度去看，桓溫第二次北伐，曾經上疏請求徐、豫二州兵力出兵，「請徐豫兵乘淮泗入河」，應當是想要借北伐統帥之便，謀算豫州、京口掌控權。桓溫還兩次建議遷都洛陽，應當是有打壓東晉朝中反抗勢力之意。而且哀帝隆和元年，桓溫第二次建議遷都時，有想要遷永嘉之後南來僑人全部到中原的諫言，「自永嘉之亂，播流江表者，請一切北徙，以實河南」。這種建議當然不會被允許，但所提之事比後面所推「庚戌土斷」更為極致，應當依舊是桓溫與東晉朝堂中反抗勢力的角力。

興寧二年五月，桓溫原本打算從合肥出兵北伐，但卻遭到朝中召其入朝參政的詔令，「加揚州牧、錄尚書事，使侍中顏旄宣旨，召溫入參朝政」。此事雖未成行，但值得桓溫警惕，在自己已經有兩次北伐之功的情況下，仍然有想要阻止桓溫繼續北伐建功，讓其入朝參政以削弱其兵權的情況出現。桓溫應當很清楚的知道，此時距離自己謀權篡位的最終目的，還有許多阻礙。這些阻礙包括豫州、京口等方鎮威脅，僑姓士族、江東士族等門閥士族力量。這些阻礙，也正好印證了當時東晉局勢中，門閥政治依然是決定王朝走向的關鍵因素。而召自己入朝一事發生之前，興寧二年三月，桓溫沒有推動軍事行為，而是自己主持「庚戌土斷」。庚戌土斷僅僅兩個月之後，桓溫便被詔令入朝參政，可見桓溫所面臨的的阻礙依舊存在。「庚戌土斷」所增加的東晉賦役，並非桓溫第三次北伐必要條件，從桓溫興寧二年準備從合肥出發北伐一事便知。既然如此，便可推斷桓溫選擇在興寧二年著眼政事推行土斷，應當是對朝中反抗勢力的震懾，結合土斷所針對的門閥士族，當知道反抗勢力正是門閥政治當中的門閥士族力量。

桓溫崛起之路，除了在北伐之前遭遇司馬昱、殷浩等人的限制之外，在桓溫掌控朝政後，依然面對不少力量的壓制。召其入朝參政削弱其軍權一事，正是壓制桓溫之表現；面對東晉朝中的壓制，桓溫兩次建議遷都洛陽、北遷全部僑人以及推行「庚戌土斷」應當都是對朝中對抗勢力的反擊。結合「庚戌土斷」所主要涉及的士族豪強，應當知道桓溫利用土斷所反制的力量正是這些土斷之策能夠影響到的士族門閥。綜上所述，桓溫所推行「庚戌土斷」的目的就顯而易見了，在自己通過北伐建功逐漸崛起的過程中，遭到了門閥政治格局內其他門閥士族力量的反抗，因此桓溫推行「庚戌土斷」正是跟建議遷都、北遷僑人以及後期的廢立皇權的目的相同，都是為了震懾和對抗士族門閥力量，以圖奠定自己在東晉朝中的地位。

總結

東晉中葉，門閥政治正處於穩定發展時期，而愈發穩固的門閥政治局勢，並不樂見某一門閥力量迅速強勢。桓溫崛起於此時，並且有覆滅成漢和兩次北伐之顯赫功勛，譙國桓氏鼎盛、掌控東晉權柄的徵兆顯而易見。這對門閥政治格局中，無法與桓溫功績相比的其他門閥士族來說，極力限制桓溫的崛起，通過徵兆桓溫入朝、削弱其兵權、否決其謀算豫、徐的打算，都是符合當時門閥政治穩定格局的需要，門閥政治中門閥之間的爭鬥也體現於此。

還值得一提的是，結合「庚戌土斷」雖然有效但並未根除弊症的情況，以及土斷推行中僅見桓溫因土斷受到士族豪強阻礙懲戒彭城王司馬玄一例，便能窺見在東晉門閥政治格局之中，即便桓溫有心想要用土斷之法對抗壓制自己崛起的門閥士族力量，也還是將門閥之間的鬥爭控制在合理範圍內。也正是因此，桓溫雖主持土斷，但政治目的更大，實際效用並無深遠影響，等到桓溫死後，土斷之策斷絕。

簡而言之，桓溫推行「庚戌土斷」的目的，在於震懾壓制自己崛起的門閥士族。桓溫掌權後依舊面對門閥政治內部的不少抵抗，否決其用徐、豫之兵協助北伐、徵召其入朝參政以虛弱兵權，正是其他門閥士族力量限制桓溫崛起之力證。而桓溫建議遷都洛陽、遷僑人回中原以及後期興皇權廢立，都是桓溫對朝中壓制自己崛起的力量的反制。同樣的，庚戌土斷主要針對士族門閥所強佔人口，但卻其結果卻並未根除僑人、僑置郡縣之弊端，便可知土斷之法是桓溫想要震懾士族門閥以鞏固自身在朝中地位的謀算。
