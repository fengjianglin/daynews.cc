韓信之所以被騙殺，並非韓信多麼信任蕭何，而是韓信低估了呂后

天天要聞
2019-11-28

摘要

淮陰侯韓信之所以會被殺，最直接的原因是他謀反於關中，最根本原因則是知己不知彼。謀反是不是一定會被殺？若是謀反成功，則必然不會被殺，如漢高祖劉邦；若是謀反失敗，大概率會被殺，如燕王臧荼、淮南王英布以及後來是吳王劉濞。但在漢初，有謀反行為，最後又失敗了的，也有活了下來的，如燕王盧綰。

淮陰侯韓信之所以會被殺，最直接的原因是他謀反於關中，最根本原因則是知己不知彼。

謀反是不是一定會被殺？

若是謀反成功，則必然不會被殺，如漢高祖劉邦；若是謀反失敗，大概率會被殺，如燕王臧荼、淮南王英布以及後來是吳王劉濞。

但在漢初，有謀反行為，最後又失敗了的，也有活了下來的，如燕王盧綰。

故而，我只能說，謀反關中是韓信的直接死因，但不是根本原因。

其實，對於淮陰侯韓信來說，謀反和上戰場沒任何分別，都是和對手打一場仗而已，若是大獲全勝，則可置酒高會，若是慘敗，或是戰死沙場，或是倉皇逃竄。

燕王臧荼就是謀反失敗，最終戰死沙場的，淮南王英布就是謀反失敗，倉皇逃竄，最後為大舅哥所騙殺。

而淮陰侯韓信就比較慘了，謀反行動部署妥當，只待與陳豨接洽完畢便可行動。在這節骨眼上，韓信被皇后呂雉和丞相蕭何騙至長樂宮斬殺。

信乃謀與家臣夜詐詔赦諸官徒奴，欲發以襲呂后、太子。部署已定，待豨報。其舍人得罪於信，信囚，欲殺之。舍人弟上變，告信欲反狀於呂后。呂后欲召，恐其黨不就，乃與蕭相國謀，詐令人從上所來，言豨已得死，列侯群眾皆賀。——《史記·淮陰侯列傳》

所謂功敗垂成，大概就是淮陰侯韓信的遭遇了。

我剛剛說了，對於淮陰侯韓信來說，謀反無異於上戰場，當年在戰場上的韓信之所以能夠戰必勝攻必取，一個很重要的原因在於，他知己知彼，他可以根據對手的弱點來設計打法。

如韓信獨自領兵後的第一個對手魏王豹。

魏豹亡走楚。楚懷王予魏豹數千人，復徇魏地。項羽已破秦，降章邯。豹下魏二十餘城，立豹為魏王。豹引精兵從項羽入關。漢元年，項羽封諸侯，欲有梁地，乃徙魏王豹於河東，都平陽，為西魏王。——《史記·魏豹彭越列傳》

魏豹之崛起，一賴其魏國宗室的身份，二賴楚懷王贈予他的幾千精兵，三賴項羽逼降章邯，所以魏豹才能在沒什麼阻力的情況下連下魏地二十餘城。

換言之，魏豹的真實本事，怕是不咋的。

漢王還定三秦，渡臨晉，魏王豹以國屬焉，遂從擊楚於彭城。漢敗，還至滎陽，豹請歸視親病，至國，即絕河津畔漢。——《史記·魏豹彭越列傳》

漢王劉邦還定三秦，繼續東略地的時候，魏王豹果斷地選擇了向漢王劉邦稱臣，使魏國成為漢的屬國；接著，漢王劉邦在彭城遭遇兵敗，魏王豹又以為漢不行了，迅速地背叛了漢王劉邦。

典型欺軟怕硬之主，既無強大的戰鬥力，也無遠見卓識。

面對這樣一個近似草包的魏王，漢王劉邦還是先禮後兵，派謀士酈食其去勸魏豹迷途知返，但魏豹以漢王劉邦沒禮貌為由，拒絕了。

酈食其歸來，漢王劉邦詳細詢問了魏王豹的戰將安排。

食其還，漢王問：「魏大將誰也？」對曰：「柏直。」王曰：「是口尚乳臭，不能當韓信。騎將誰也？」曰：「馮敬。」曰：「是秦將馮無擇子也，雖賢，不能當灌嬰。步卒將誰也？」曰：「項它。」曰：「不能當曹參。吾無患矣。」——《漢書·高帝紀》

也就是說，酈食其的出訪，雖失敗，但卻帶來了很有用的信息，這些信息再加上漢對魏王豹的了解，漢以對魏達到了知己知彼的程度。

所以，淮陰侯韓信在攻魏王豹的魏國時，直接為這位「老同事」設計了一個明修棧道暗渡陳倉的作戰方案。

其八月，以信為左丞相，擊魏。魏王盛兵蒲坂，塞臨晉，信乃益為疑兵，陳船欲渡臨晉，而伏兵從夏陽以木罌缻渡軍，襲安邑。魏王豹驚，引兵迎信，信遂虜豹，定魏為河東郡。——《史記·淮陰侯列傳》

「魏王豹驚，引兵迎信，信遂虜豹」，讀起來讓人莫名地想笑，魏豹草包的形象一覽無餘，實際上，當時被俘虜的，可不止魏王豹一個人。

月余，魏王豹反，以假左丞相別與韓信東攻魏將軍孫遫軍東張，大破之。因攻安邑，得魏將王襄。擊魏王於曲陽，追至武垣，生得魏王豹。取平陽，得魏王母妻子，盡定魏地，凡五十二城。——《史記·曹相國世家》

魏王豹的母親、妻妾和子女，盡數被俘虜。其中一個姓薄的姬妾還被送到了漢王后宮織室中為奴，想必魏豹其他親人的結局也不會好到哪去。

漢使曹參等擊虜魏王豹，以其國為郡，而薄姬輸織室。豹已死，漢王入織室，見薄姬有色，詔內後宮，歲余不得幸。——《史記·外戚世家》

魏豹之外，其他被韓信擊敗的對手，如陳餘、龍且、項羽，韓信也是非常了解的。

陳餘是張耳的刎頸之交，韓信擊陳餘時，張耳同行，張耳不但可以給韓信講解陳餘的為人，還能利用他在趙地的關係網來刺探陳餘的部署。

龍且是項羽麾下將軍，韓信曾在項羽麾下做過執戟郎中，定然研究過龍且的為人和打仗的策略。

至於項羽，更是韓信的研究對象。早在韓信被登壇拜將之時，韓信就用「匹夫之勇」和「婦人之仁」來精準地為項羽畫像。

如此，韓信怎麼能不在戰場上戰必勝，攻必取呢？

但是，韓信謀反時，卻沒有全面研究即將面對的對手皇后呂雉。這一點，從韓信的遺言中得到了充分的證明。

信方斬，曰：「吾悔不用蒯通之計，乃為兒女子所詐，豈非天哉！」——《史記·淮陰侯列傳》

對於這位日後將稱制達七年之久的漢高後，韓信竟然輕蔑地稱其為「兒女子」，這與龍且面對韓信時的言辭多麼相似啊。

龍且曰：「吾平生知韓信為人，易與耳。……」遂戰，與信夾濰水陳。韓信乃夜令人為萬餘囊，滿盛沙，壅水上流，引軍半渡，擊龍且。詳不勝，還走。龍且果喜曰：「固知信怯也。」——《史記·淮陰侯列傳》

都是自以為了解對手，實際上並不了解對手，都是犯了輕敵的大錯。

如果龍且不輕敵，豈能被韓信用水給淹了？如果韓信不輕敵，豈能輕易地被呂后所誆騙？

​國相紿信曰：「雖疾，強入賀。」信入，呂后使武士縛信，斬之長樂鍾室。——《史記·淮陰侯列傳》

你看看，淮陰侯韓信大搖大擺地去了長樂宮，然後就被呂后砍了腦袋。

要怪只能怪他淮陰侯韓信太輕敵了，所以才在知己不知彼的情況下被斬殺。
