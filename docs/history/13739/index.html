大明二品是多大的官？家宴的主菜竟是燒餅

2019-12-10 553條評論 3,210次閱讀 1578人點贊

二品官，貌似也不是太大，畢竟前頭還有若干個一品大員？可若說起這個人的名字，那定是如雷貫耳的。劉基（1311年7月1日－1375年5月16日）字伯溫，謚曰文成，元末明初傑出的軍事謀略家、政治家、文學家和思想家，明朝開國元勛，漢族，浙江文成南田（原屬青田）人，故時人稱他劉青田，明洪武三年（1370）封誠意伯，人們又稱他劉誠意。武宗正德九年追贈太師，謚號文成，後人又稱他劉文成、文成公。

古語云：「三年清知府，十萬雪花銀。」曹雪芹《紅樓夢》描寫了榮國府的極致奢華的生活，比如幾十兩白銀做出來的茄子等等，那身為堂堂大明朝開國元勛的劉伯溫該用什麼山珍海味來宴請賓客呢？

劉基官至廟堂，卻無忘故土美食，其在《萬里封侯》一詩中，即寫道「八珍鼎食，何如故鄉」。表達了遊子在外，對故鄉的眷戀。

而故鄉的宗親，也念念不忘先祖恩澤。每至春秋，劉氏後裔即聚集「欽建誠意伯廟」（國家文保單位）例行「太公祭」（第三批國家級非物質文化遺產），緬懷先祖，春秋兩祭後舉行的宗族筵席，即為「伯溫家宴」的緣起。而劉氏後人根據伯溫先生《多能鄙事》和民間流傳，選取一些有紀念意義的菜肴，結合南田本地特產，逐漸形成了今日之伯溫家宴。那就一起去一探究竟吧。

《文成米塑》傳統宗族祭祀所用米塑造型，由米粉與傳統柒料製作而成，一般以桃形、蔬果、動物、戲曲人物等為主，舊時作為祭品。

南田劉氏祭祖，傳統有三牲五畜，現伯溫家宴取芋、薯、粟、花生、土豆等五種雜糧穀物蒸熟替代牲體，象徵豐收，即保留了儀式感，又有感恩自然造化，慶祝大地豐收的含義，也符合了現代人的素食風尚。

我已經覺得是不是走錯地方了，美味是美味，健康自然是很健康的，可和想像中的大餐有些差距。台上在演繹著《燒餅歌》，「燒餅歌」是一首甚為準確的預言。它的名為何叫做「燒餅歌」呢？在其中也有提及，話說於公元 1368年某一日的早上，明太祖在內殿里吃燒餅，只咬了一口，便聽到內監會報劉伯溫覲見。太祖心想測試劉基一下，於是便以碗蓋著只咬了一口的燒餅，再召劉 基入殿晉見。

帝問曰：「先生深明數理，可知碗中是何物件？」

基乃捏指輪算，對曰：「半似日兮半似月，曾被金龍咬一缺，此乃餅也。」

開視果然。帝即問以天下後世之事若何。

基曰：「茫茫天數，我主萬子萬孫。何必問哉。」

帝曰：「雖然自古興亡原有一定，況天下非一人之天下，惟有德者能享之。言之何妨？試略言之。」

基曰：「泄漏天機，臣罪非輕！陛下恕臣萬死，才敢冒奏。」

帝即賜以免死金牌，基謝恩畢。

著名的文成燒餅，據說劉伯溫死前都還想再吃一口的。

白落地，本地俗稱，學名銅錘玉帶草，可入葯。與蛋同煎，鮮香可口，又有祛風除濕，活血解毒的功效。

《清清白白》

蘿蔔，文成俗稱「菜頭」，蘿蔔纓，稱「菜頭菜」。取本地鮮嫩高山蘿蔔連纓同燒，其色清白，其味清甜鮮香，是時令鄉野蔬菜的經典。

是不是想起了當年那個春晚上的小品呢？其實真的是白蘿蔔，青菜。

劉伯溫當年隨朱元璋征戰南北，經常攜家鄉熏臘兔肉隨食，朱元璋偶得嘗之，便喜歡上了兔肉，常令劉伯溫從南田給他捎帶。這便是伯溫貢兔的由來。

劉伯溫的家鄉在溫州文成縣，南田山在青田縣城南一百五十里處。南田山的形勢，乃是「萬山之巔，獨開平壤數十里，號南田福地」。《洞天福地記》中記載說：「古稱七十二福地，南田居其一」。如果你來到文成，可想品嘗這樣的伯溫家宴呢？
