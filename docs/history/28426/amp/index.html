姨媽去世前給他一張藏寶圖，他挖到5萬件寶物，卻全部捐給了國家

天天要聞
2019-12-20

在武俠小說裡面，藏寶圖是一種非常常見的東西。許多人靠此獲得了武功秘籍，成為了武林高手，也有人靠此獲得了大量的財富，成為了一代巨富。

不過在現實生活中，也有一個人獲得了一張藏寶圖。這個人就是趙泰來。趙泰來是我國有名的慈善家跟收藏家。趙泰來的姨媽曾經留了一張藏寶圖給他，後來他靠此挖到了5萬件寶物，不過出人意料的是，趙泰來全部捐給了國家。按照當時估計這批寶物的價值高達8億元。要知道這可是在21年之前，如果放在今天至少價值百億。

趙泰來出生在一個名門望族，他的曾外祖父是中國著名的外交家伍廷芳。伍廷芳是中國近代第一個法學博士，後來進入了李鴻章的幕府工作。在洋務運動中，伍廷芳是李鴻章的得力助手。後來伍廷芳還成為了司法總長跟外交總長。伍廷芳是近代的一個偉大人物。

趙泰來的叔公也曾經是香港的富豪。趙泰來的父親早年曾經在香港行醫，後來回到東莞發展。不過後來趙泰來的父母早早去世，趙泰來只能去香港投奔姨媽。當時趙泰來只有15歲。

趙泰來的姨媽也是一個女富豪，在香港跟英國都有不少資產。趙泰來的姨媽膝下無子，於是趙泰來後來就過繼給了姨媽當兒子。1977年，趙泰來的姨媽告訴他，在香港有一棟豪宅，還有堆滿了房子的藏品。另外在英國還有不少資產。姨媽還交給了趙泰來一張藏寶圖，讓他以後去英國繼承家產。

1979年，趙泰來回到香港接收家產，當時房產的價值已經超過了千億，最重要還有滿屋的藏品。趙泰來發現這些藏品都價值不菲。2年之後，姨媽去世，趙泰根據藏寶圖去英國尋找遺產。

原來趙家在80多年之前曾經在英國購置了一個龐大的莊園，並且在莊園裡面建了一個地窖，把許多珍貴的藏品都收藏在裡面。因為這個莊園非常大，趙泰來跟妻子找了很久才找到了這個地窖。

打開地窖之後，趙泰來發現裡面都是珍貴的寶物，包括了字畫、瓷器、銅器跟玉器等等。這些東西都裝在木箱裡面。這些木箱已經塵封多年。原來這些寶物都是當年趙泰來的曾外祖父伍廷芳購買的。伍廷芳曾經在美國、秘魯等國家當公使，所以在海外購置大量的流失文物。

趙泰來發現了這批寶物之後，跟妻子都非常高興。不過他們擔心被別人發現，不敢請人打理，只能兩個人自己清理。他們足足用了10年的時間，才把這些寶物從地窖清理出來。

這10年為了保護這些藏品，趙泰來不敢請保安跟傭人，全部靠自己跟妻子工作。同時趙泰來還養了好幾條獵犬來看家。

等到全部藏品都清理好之後，已經是10年之後的事情，趙泰來總計發現了5萬多件藏品，可以說是規模龐大。這個時候趙泰來卻決定把全部藏品帶回中國，並且捐給國家。當時趙泰來的妻子卻不願意，她認為趙泰來太傻了。這些藏品可以說是價值連城，足夠自己跟後世子孫過上富貴的生活了。

趙泰來卻堅持把文物都捐掉。單單為了把這些文物運回來，趙泰來就賣掉了英國4套別墅，才湊夠了運費。趙泰來把這些文物運回來之後，分別捐給了國家博物館跟廣州博物館，還有廣州的寶墨園。

當時專家鑒定，這些文物按照當時的價格，高達8億元。趙泰來後來在中央台接受採訪的時候，自己把這些消息說出來，震驚了不少人。在這5萬件藏品中，單單是三級以上藏品就達到150件。這些藏品如果放在現在，那麼價值至少百億。或者趙泰來當時把這些藏品賣掉，拿著8億去投資房地產，現在也是百億富豪了。

姨媽去世前給他一張藏寶圖，他挖到5萬件寶物，卻全部捐給了國家，趙泰來確實有祖上遺風，是一個值得銘記的人物。
