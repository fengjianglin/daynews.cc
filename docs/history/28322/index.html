令妃的女兒七公主和九公主，為何品級不一樣呢

2019-12-21 954條評論 2,516次閱讀 618人點贊

通過追劇，大家都知道，她的出身比較低微，為此進了宮裡，一開始也並沒有太高的地位。然而她還是憑藉自己的本事(主要就是美貌)，迅速贏得了皇帝的恩寵，這從在偌大的後宮裡，她還能夠為皇帝生下4子2女一共6個孩子，就完全可以看得出來。

　　說到這裡可能很多人就注意到了，雖然她先後生下了七公主和九公主，但這兩個女兒在宮裡的地位，卻並不是完全相同。這究竟是怎麼回事呢?其實與後宮妃嬪們擁有不同的等級一樣，她們所生下來的孩子，地位也並不是完全平等的。當然地位最高的，還是皇后所生的女兒。按照皇太極當時規定的祖制，這種級別可以稱為固倫公主。千萬不要小看這個名號，因為擁有這個稱號的公主，基本上享有的待遇，和那些親王是完全相同的。至於其它的嬪妃，她們所生的女兒，就只能叫做和碩公主了。其實從公主的稱號，就不難看出她的出身高低。

　　那大家就感到有些納悶了，在不同的嬪妃身上有所差別可以理解，這都是令妃所生，為何兩個公主的地位卻不盡相同呢?其實說到這裡，就不得不提另外一種情況，就是皇帝本人的喜好。如果皇帝當時對公主格外恩寵的話，就會想方法提高她的品級。雖然令妃當上了皇貴妃，但畢竟不是皇后，所以她的兩個女兒，按理說都是和碩公主。但皇帝對7公主卻格外寵愛，卻又不能為了提高公主的品級，就把她的母親封為皇后。其實皇帝雖然把皇后幽禁了起來，總歸沒有廢除她的封號，這裡面有很大的困難。於是皇帝就從其它方面想辦法，把7公主許配給了一個品級比較高的男人拉旺多爾濟。這個人不僅是世襲親王，而且後來還救過嘉慶皇帝的性命。按照宮裡的規矩，公主所嫁的男人是超勇親王，為此公主的等級也應該提高，這裡面體現的是皇家的尊貴。也正是通過這個辦法，乾隆給自己寵愛的7公主提升了等級，她也成為了固倫和靜公主，就比妹妹9公主高了一個品級。
