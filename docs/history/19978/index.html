拿破崙的王朝為何一夜破滅？軍事天才為何失敗？原因只為一人

2019-12-16 2,490條評論 4,691次閱讀 901人點贊

#新作者扶植計劃開啟# 波拿巴.拿破崙一世原是柯西嘉島上一個破落貴族的兒子。1989年法國大革命爆發，二十歲的拿破崙加入法國革命軍，乘著法國大革命的多變的局勢平步青雲。人家運氣你沒法比。一七九九年十月九日，拿破崙發動政變，自任第一執政。一八零四年，元老院授予拿破崙皇帝

稱號，怎樣，很了不起吧。一生征戰無數，軍功顯赫，一八一五年六月十八日歐洲各位君主建立反方聯盟，幾位大佬一商議，不行，要給他點顏色瞧瞧，於是，幾個大佬帶著自己的小弟，抄起傢伙，揚言要揍他，拿破崙一聽這可了不得，於是親自上陣

給自己軍隊加油，決定在他們來之前，將其逐個擊破。維也納會議正在舉行，歐洲大佬還沉迷在勝利的狂歡之中，在會上互相爭吵，如何拿回損失，突然有一枚像嗖嗖的炮彈飛來的消息，拿破崙，這頭被困的雄獅闖出來了，剛剛還在為如何瓜分勝利的果實而爭吵的他們一下安靜了，似乎有一隻利爪攫住了他們的頭，他們急急忙忙找來自己的部隊，決定再次聯

合起來，收拾拿破崙，這一次要徹底的消滅拿破崙。英國率先向法國進攻，普魯士作為援助。拿破崙一下看到了致命的危機，他知道自己必須在他們集合之前幹掉他們，拿破崙帶著自己的全部兵力向英國部隊出發。一場大戰就這樣開始了。

拿破崙不允許自己喘息，也不允許別人喘息，因為每拖延一天，都在為對方增加力量，而勝利的捷報將會向烈性的白酒一樣，是不安的法國民眾如痴如狂，是自己身後的祖國繼續流血，拿破崙帶著自己弟兄到敵人的高地前，而頭腦冷靜，意志堅強的

敵人早已築好了工事，拿破崙的一切部署都是空前細緻的。他的軍令也是空前明白的。他不斷斟酌進攻方案，而且考慮到了自己的處境。他擊敗普魯士軍隊，但沒有徹底消滅，僅僅是擊退，他們隨時可能與英軍匯合，為了防止這種可能性，他調出一部分的軍隊去跟蹤追擊普魯士軍隊。

拿破崙把指揮權交給了格魯希元帥。格魯希，一個氣度中庸的男子，老實可靠，兢兢業業。當他在任騎兵隊長的時候，是很棒的，他沒有超人的膽識魄力，也不是足智多謀的將領，也缺乏英雄氣概。關於格魯希，沒有神話般的傳說，也沒有誰把他描寫成威風凜凜的勇士。在拿破崙的傳奇中，格魯希沒

有顯著的成績，讓他聞名於是的只是他的不幸和倒霉，他跟著拿破崙混了二十多年，參加過多次打架，一級一級升級，直到元帥，他的大哥都掛了，拿破崙自己知道這位小弟不是氣吞山河的英雄，只是因為聽話而重用，自己的幾位得力小弟不幹了，回家陪老婆了。

就在此時，格魯希元帥帶著自己三分之一的力量去追之前潰逃的那撥人，自己現在與英軍打的不可開交，誰都知道，哪方面先獲得支援，就會先干倒對方，他不停的給格魯希發去命令，速來支援。然而他兩距離很近，但騎兵還是要走幾個小時，可是格魯希元帥自己並未意識到自己掌握拿破崙的命運，格魯希元帥的手下熱拉爾請求自己帶一師部隊去那邊戰場看一下，格魯希只考慮了一分鐘，然而這一分鐘就決定了，自己及自己大哥，與整個世界的命運。

由於自己沒有及時得到支援，拿破崙這位偉大的皇帝一夜之間回到了平凡之中。格魯希這一人，他的決斷，使得自己大哥失去了最後的希望，拿破崙這位英勇的戰士，就這樣長眠於地下了。
