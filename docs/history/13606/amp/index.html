天天要聞

“凄涼蜀故伎，來舞魏宮前”——談談魏滅蜀之戰的軍事戰略

天天要聞
7 天 ago

高平陵之變後，曹爽集團被一網打盡，司馬家族掌握了朝中大權。為了安定內部，鞏固統治，司馬懿父子三次平定淮南叛亂，並藉機打壓朝中異己，統治地位日益穩固。等到司馬昭為大將軍時，魏帝曹髦被殺，曹氏宗親基本被剷除，司馬昭集大權於一身，為了維護自己的權威，意欲滅蜀，繼而滅吳，一統天下。

反觀蜀國，夷陵之敗後，國力衰弱，仰仗諸葛亮的賢能理政治軍，勉強依靠支撐危局，即便有恢復中原的鴻鵠之志，但卻因國小力弱，五出祁山而無功。諸葛亮死後，後主劉禪昏庸無能，宦官黃皓擅權亂政，蜀國內政衰弱，民生困頓，漸露亡國之兆。

一、魏、蜀、吳三國並立的動態平衡被打破

魏國自曹芳以後，帝王大多都是孩童，生長深宮，不懂政治，更不知民間疾苦，一切軍政大權落於司馬懿之手。魏明帝曹叡時，司馬懿南拒巴蜀，北平遼東，在軍中和朝內樹立了他的勢力和威望。他在輔政期間，為了緩和國內的矛盾，採取一些休養生息的政策，大辦屯田，廣修水利，發展生產，使魏國的國力大大超過了蜀漢。

司馬懿死後，其子司馬師、司馬昭相繼執政，繼續屠戮曹魏忠臣以及曹氏宗親，經過三次平定叛亂和兩次廢立，司馬家族的政敵幾乎清除殆盡，統治地位日趨鞏固，曹氏成了名義上的皇帝。

景元三年八月，司馬昭認為滅蜀的時機已經成熟，就準備對蜀漢用兵，其主要目的在於立功於國外，以提高國內的威望，方便取魏而代之。

諸葛亮死後，蜀漢由蔣琬、費禕相繼執政。蔣琬執政時，認為魏國跨有九州，根深蒂固，平除不易，應聯盟東吳，逐步吞食。費禕接替蔣琬理政時，認為自己遠不如諸葛亮，提出了「保國治民，謹守社稷」的穩重治國方針，因此姜維每次想大舉出兵，費禕經常限制他，給他的兵力不過萬人。

費禕死後，姜維任大將軍，掌握了軍隊。他主張西連羌胡，奪取隴右，相機進圖關中。在他主持蜀漢軍事的十年中，曾多次出兵伐魏，但取得的戰績不大，反而損兵折將，削弱了蜀漢的國力，史載：「維數出兵，蜀人愁苦」。蜀軍將領張翼和廖化等都一再反對用兵，認為國小民勞，不宜整武。

「兵不戢，必自焚，智不出敵而力小於寇，用之無厭，將何以存。射幸數跌，不如審發，極武黷征，土崩勢生。」

在蜀國朝堂之上，後主劉禪昏庸無能，吃喝玩樂，不理政事。親信宦官黃皓參與朝政，擅權亂政，胡作非為，政治日趨腐敗，邊備不修。東吳使臣薛翔對蜀漢的國情這樣評論：

「宦官專權，國無政令，窮兵黷武，民勞卒疲。競於外利，不修邊備。主間而不知其過，臣下容身以求免罪，入其朝不聞直言，經其野民皆菜色。」

從蜀漢當時的國力來看，一州之地，人口不過百萬，士兵最多十萬，還有四萬官吏。成年的男丁不是被迫當兵就是苦於徭役，生產受到嚴重破壞。蜀漢人民平均每九人就要供養一個士兵，七家民戶，供養一個官吏，負擔十分沉重。蜀漢政權已已經走到苟延殘喘，山窮水盡的境地。姜維請殺黃皓未遂後，怕危及自己，又率軍遠駐沓中避禍屯田，使漢中正面防守兵力空虛，釀成亡國之禍。

二、蜀漢在漢中方向上的邊防部署和戰役指導

漢中北瞰關中，南蔽巴蜀，東達襄鄧，西控秦隴，地形重要，為魏與蜀必爭之地，亦為蜀漢北方的邊防要地。蜀將黃權認為若失漢中，則三巴不保。蜀漢太守楊洪亦說：「漢中則益州咽喉，存亡之機會，若無漢中則無蜀矣」。

蜀漢在漢中方面的邊防部署，在劉備留魏延鎮守漢中之後，便是充實兵力堅守漢中外圍秦嶺各要隘，以御外敵。敵人前來進攻，憑藉山地之險，就能拒敵於外。

姜維擔任大將軍後，認為防守重迭交錯的外圍各要點，只可以抗拒敵人，不能獲得大勝。他向後主劉禪建議，待敵人進攻時，漢中外圍各要點都收兵聚谷，退守漢、樂二城，使敵人進入漢中平地，並以重兵防守各關隘，抗拒魏軍，另派游軍側擊敵人的空虛。

敵人攻關不克，野無散谷，千里運糧，自然疲乏。待敵人退卻時，防守要點的蜀軍一齊出擊協同游軍，并力搏戰，便可消滅敵人的有生力量，取得大勝。後主劉禪採納了他的建議，命漢中都督胡濟退駐漢壽，監軍王含防守樂城，護軍蔣斌防守漢城。

姜維的誘敵深入擊其惰歸的邊防作戰方案，如果兵力部署適當，能發揮效果。在魏軍兵力不強，小規模進攻的情況下，也能大量殲敵；即使在魏國大舉進攻的情況下，亦可能收到敵疲而後反擊的好戰績。但是部署邊防戰略，利用地形設置袋形陣地，袋底必須堅牢。如果袋底很容易地被敵人的矛頭戳穿，袋形合圍的部署就會破產。

因此，姜維的誘敵深入擊其惰歸戰略的核心在於：漢中這個主要防禦方向上必須配置蜀軍的主力，作為強大的機動部隊，先頂住敵人攻勢的矛頭，才能在敵疲之後，配合外圍友軍，并力反擊，殲滅敵人。

但是事與願違，姜維的兵力部署與他的戰略意圖不協調。姜維得罪宦官黃皓後，為了遠離禍患，率領蜀軍的主力屯田沓中，在漢中這個主要防禦方向上，沒有蜀軍強大的機動部隊。這樣，放棄漢中外圍各要點，讓敵進入平地，如此誘敵深入，便等於開門揖盜，這是他籌劃蜀漢邊防的極大失誤。

後來姜維探知「鍾會治兵關中欲窺進取」的消息後，感到漢中兵力薄弱，就應該立即變更兵力部署，把在沓中的蜀軍主力轉移到漢中這個主要防禦方向，拒守諸隘路，以抗擊魏軍的大舉進攻。可是姜維卻並沒有率部轉移，只上表劉禪派遣張翼、廖化率軍分護陽安關口（陝西陽平關），田平橋頭（今甘肅文縣東南），防患未然。

劉禪為宦官黃皓所誤，並沒有及時採納姜維請求增調兵力的建議。直到魏軍大舉進攻時，劉禪才急忙調廖化率軍去沓中支援姜維，調張翼和董厥去防守陽安關口，支援外圍要點。但廖化到陰平時，沒有及時佔領橋頭孔道，反而魏軍諸葛緒部奪取，阻斷了在沓中的姜維的歸路。張翼、董厥部行動遲緩，也未能及時趕到陽平關。

這時，姜維的蜀漢主力還遠在沓中，這樣一來，外圍的部隊已撤下來，後面的援軍還上不去，姜維的主力又過不來，漢中的蜀軍防務部署基本陷於崩潰。如果姜維不為避禍移駐沓中，重視漢中正面的邊防籌劃，在部署山地防禦時，險路設防，堵塞虛隙，不會有鄧艾冒險成功。

三、司馬昭的謀議和決策

魏帝曹奐景元三年，司馬昭準備出兵滅蜀，召集群臣傳達他對吳蜀兩國的情況判斷和攻取蜀漢的戰略：

「自定壽春以來，息役六年。治兵繕甲以擬二虜。今吳地廣大而下濕，攻之用功差難，不如先定巴蜀。計蜀戰士九萬，居守成都及備他境不下四萬。然則餘眾不過五萬。今絆姜維於沓中，使不得東顧，直指駱谷，出其空虛地以襲漢中。以劉禪之閽，而邊城外破，士女內震，其亡可知也。」

魏國的群臣大多不贊成對蜀用兵，只有司隸校尉鍾會贊同。於是司馬昭任命鍾會為鎮西將軍，先行關中作進攻蜀漢的準備工作，同時命令徐、兗、豫、荊、揚各州準備船隻，揚言攻吳，以便聲東擊西。

司馬昭自從平定淮南叛亂、穩定內部後，就整軍經武，準備消滅吳蜀兩國。但是一石不能二鳥，司馬昭為何會做出「先蜀後吳」的戰略決策來呢？

1.從地理角度來說，攻吳要實施江河進攻，攻蜀要進行山地進攻，都是在特種地形地帶作戰。在當時作戰工具條件的限制下，吳國據江守險，魏國無大量的舟楫，要實施渡江作戰，尚有一定的困難。對蜀漢用兵雖在山地作戰，但可以乘虛蹈隙，只要一點突破，沖入腹心，就會使蜀漢的防禦體系解體。

2.從吳蜀兩國的國力來說，吳國據有三州之地，而蜀漢只佔有一州。三國之中，蜀漢最為弱小，加上姜維多次用兵，蜀軍將領都認為「國小民勞不宜黷武，用之無厭，將何以存」。頻年用兵的結果，使得益州兵疲民困，邊備不修。按先弱後強、各個擊破的用兵原則，先向弱者開刀，滅蜀之後，「勝敵而益強」，再圖消滅吳國，更容易成功。

3.蜀漢的邊防部署出現漏洞。姜維收縮漢中外圍各要點的兵力退保漢、樂二城，而他又親率蜀軍的主力遠屯沓中，這就使蜀漢的邊防要地漢中兵力空虛。司馬昭發現了這個空隙，制定了制姜維於沓中，主力出其空虛、直搗漢中的戰略方針，可以說是擊中要害。

四、魏軍滅蜀的戰略部署和戰役經過

景元四年十二月，司馬昭根據制姜維於沓中、大軍出其空虛以直搗漢中的戰略方針，決定進攻蜀漢，各路大軍的戰略部署和任務如下。

西路軍：征西將軍假節都督隴右諸軍事、鄧侯鄧艾統率司馬師篡、將軍惠唐亭侯鄧忠、天水太守王頎、隴西太守牽弘和金城太守楊欣共事軍三萬人自狄道向甘松、沓中進軍，以牽制姜維。

中路軍：雍州刺史諸葛緒統率將軍龐會和田續以及南安、廣魏、安定、武都和陰平各太守共率軍三萬人自祁山向武街（今甘肅成縣）、橋頭（陰平橋頭）進軍，以遮斷姜維歸路井阻止蜀軍進援沓中。

東路軍：鎮西將軍假節都督關中諸軍事關內鍾會統率長史杜預、護軍胡烈、荀愷、前將軍李輔、參軍皇甫闓、司馬夏侯等共率十二萬人，分由斜谷、駱谷、子年穀三道並進，分頭趁虛直取漢中然後南下攻蜀。

另令魏興太守劉欽先期率軍萬人自魏興向漢中進攻，先敵佔領子午谷以及駱谷的南口，保障鍾會的大軍進出容易。

監軍：侍中衛瓘率軍千人持節臨三路軍

司馬昭滅蜀的戰略部署是派鄧艾與諸葛緒牽制姜維於橋頭以西至沓中地區，以使姜維軍無用武之地，方便鍾會的主力越過漢中的陽安關口而直趨劍閣如入無人之境。為了牽制蜀國在漢中的守軍，以保障鍾會的大軍能迅速越過漢中而南下，另派魏興太守劉欽預先攻取子午谷與駱谷的南口，襲擾蜀國漢中守軍的側背。

姜維在沓中探知鍾會的主力已入漢中，恐怕陽安關有失，危及劍閣，成都兵少力弱，難以抗擊，就決心擺脫鄧艾部，立即向東進行戰略機動，退回劍閣，憑藉天險，設防固守。魏將諸葛緒因姜維實施戰略機動造成戰略誤判，自動後退三十里，使姜維大軍幸運地通過陰平橋頭，會合廖化、張翼、董厥諸軍退守劍閣，堵住虛隙，方使局勢暫時轉危為安。

鍾會所部，屢攻劍閣雄關不克，十餘萬魏軍陷入進退維谷的境地，加之糧草不繼，鍾會準備退軍，此時，鄧艾在陰平探知魏軍即將退軍，向鍾會提出出奇制勝、偷渡陰平的襲擊方案。他在建議上說：

「今賊摧折，宜遂乘之。從陰平由邪徑經漢德陽亭趣涪出劍閣西百里，去城都三百餘里。奇兵沖其腹心，劍閣之守必還赴涪，則會方軌而進；劍閣之軍不還，則應涪之兵寡矣。今掩其空虛，破之必矣。」

鍾會採納了鄧艾的方案，鄧艾選精兵一萬人，輕裝從陰平出發，沿山谷小路行軍七百里，鑿山通道，越過馬閣山天險，直奔江油，守將馬邈不戰而降。劉禪急派諸葛瞻率軍迎擊。諸葛瞻到涪城後，其部將黃崇再三勸諫：「宜速行據險，無令敵得入平地」，他遲疑不決。鄧艾乘機長驅直入，擊殺諸葛瞻，佔領了綿竹。劉禪得知綿竹兵敗，成都難守，逃走無路，遂自縛請降，並敕令姜維投降。至此建國四十三年的蜀漢政權被魏滅亡。

小結

魏滅蜀之戰，是三國後期的吞併戰，也是華夏再次統一的前奏。從軍事戰略上來看，姜維率蜀軍主力長期偏處沓中，使蜀漢北方邊防要地漢中防務空虛，給了魏軍可乘之機，因此司馬昭制定了出其空虛，直搗漢中的戰略方針。在劍閣相持的不利情形下，魏軍出敵不意地偷渡陰平，沖入蜀漢腹心，使扼守劍閣正面的蜀軍，陷入腹背受敵的困境，最終贏得了戰爭的勝利。魏滅蜀之後，佔領長江上游，在益州大制舟楫，訓練水師，「順流之勢，水陸並進，乘勢滅吳」，也就易如反掌了。
