#泛文化寫作營#

1880年香港沿水街道

1949年8月1日，對粵海關稅務司經蔚斐來說是個重要的日子，因為與港督有親戚關係，他被總稅務司署廣州辦事處派往總部設在香港的九龍關任稅務司。經蔚斐被擺上了歷史的風口浪尖，身不由己地成了歷史人物。這時離共和國在北京宣告成立剛好兩個月，四野的部隊已經大兵壓境。總稅務司署這個時候把經蔚斐派到香港，只能說是困獸猶鬥。其實對時局的估計，無論是當時的國民政府財政部長徐堪還是總稅務司李度，都瞭然於心。9月2日，李度特地從廣州趕到香港，跟經蔚斐商討時局和對策，實際上是傳達徐堪的指示。相關內容後來由經蔚斐整理後發布在《九龍關政策》上，主要有以下七點：

一是只要情況許可，九龍關繼續行使職能；二是採取適當步驟，確保對九龍關緝私艦艇的控制，並做好撤往台灣的準備；三是共產黨佔領邊界後，與共產黨當局取得聯繫，竭力與他們達成協議，使九龍關儘可能按現行方針工作；四是海關今後的工作，不僅要為共產黨所接受，也要為香港當局所接受。為此，要與香港當局保持密切接觸，達成一項可以接受的通用辦法，在共產黨和香港當局面前，盡量提高海關的身價；五是要向共產黨闡明海關繼續在香港工作，稅務司應由英籍洋員擔任；六是經蔚斐和所有關員，不會自動放棄取得國民政府發給養老金和其他應付款的要求；七是海關被共產黨控制後，關員不再聽從命令時，應作為喪失享受九龍關一切權利和要求論處，九龍關不再對他們承擔責任。

經蔚斐這時的處境跟民國初年江蘇都督程德全有些類似。程德全名為總督，身處國民黨的勢力範圍，不得不在袁世凱和孫中山兩人中間周旋，要想兩不得罪，兩邊討好，那是要些本事的。經蔚斐能夠做末代稅務司也不是浪得虛名的，看他整理的這份九龍關政策就圓滑得很，不光兩不得罪，還三方討好，把港英政府的臉面也顧及上了。不過這時新生的共和國和港英當局都顧不上九龍關，只有已經跑到台灣的李度還惦記著大陸上碩果僅存的這點家當。所以他一接到經蔚斐的密呈後，立即以22號密函作了答覆，再次強調徐堪的指令：A、Y兩級緝私艇必須駛往台灣或海南，絕不能落入共產黨之手。李度擔心這些海上的巨無霸一旦落入共軍手裡，就會變成攻擊台灣的利器，他就無顏面對蔣委員長這個老東家了。同樣是國民政府的僱員，經蔚斐卻是另一種心態，對不起東家不打緊，不能對不起自己的祖國——大英帝國的代表港英當局，還要為將來跟共產黨新政權談判留點籌碼，他要確保對緝私艇的絕對控制，至於是撤往台灣還是廣州則要視局勢的變化來定。

1879年香港的菜市場

共和國在北京宣告成立這天，李度在台灣坐不住了，再次急電經蔚斐把艦艇調往台灣。龍龍關的地下組織得知消息，組織船員聯名上書，拒絕赴台，甚至派人破壞艦艇的動力設備，企圖阻止艦艇啟航。對此，經蔚斐採取了斷然措施，指示艦隊司令、同樣是英國人的馬伯勞，聯繫香港水警，強行將船員驅趕上岸，然後租用拖輪，僱用船員，將4艘排水量1000噸的A型艦和2艘排水量275噸的Y型艦調往台灣。

這裡有必要說說九龍關的地下組織了。民國海關沿襲清末的慣例，實行的是稅務司制度，海關基本上還是由洋人把持，所以別說共產黨，連軍統這樣的特務組織也很少滲透，主要原因是沒有太多的必要。但我黨的地下活動是無所不在的。早在1948年，地下黨員林大琪、李國安就根據黨組織指示，在九龍關成立「進修會」，發動關員為保護關產、迎接解放做準備，並向港澳工委提出了一個具體的行動方案，包括物色積極分子，爭取一個華人副稅務司支持，逼迫稅務司交出設在香港的關產、稅款和私貨倉庫，等等。地下黨員在海關里級別太低，在海關這種體制里，依靠他們自己的力量護關護產已經很難，更別說起義了。所以直到1949年2月，九龍關積極分子已經發展到數十人，成立了護產小組，但也只能達到宣傳發動這個層次，真正要採取行動還得依靠黃昌燮這個骨幹成員以二等監察長的身份去執行。縱是如此，或許是因為對九龍關起義不抱大的希望，或許是因為內地還有更重要的工作，上級竟然把地下組織的負責人林大琪調到北方工作，李國安接替林大琪。

這時九龍關發生了一件大事，促使經蔚斐一百八十度轉向。九龍關緝私稅務司史鐸史和二等監察長楊俊虯到白石洲支關視察邊境工作，被關警扣留。關警提了兩個要求：一是發放三個月的薪津作為應變費，二是把一等監察長杜熾芬和楊俊虯交給關警處理。白石洲支關關警膽敢起事，跟地下組織的宣傳策劃分不開，更主要的原因是形勢的變化，粵港邊境國共雙方勢力倒轉，九龍關風雨飄搖。可是經蔚斐認為，縱使時局如何混亂，海關這種體制下關警是不可能鬧事的，一定是中共游擊隊滲透的結果。這就把黃昌燮推上了前台。九龍關三個監察長，兩個被關警要求清算，有人就向經蔚斐打報告，說黃昌燮是共產黨。經蔚斐就把黃昌燮找來，倒沒追究他是不是共產黨，而是問他能不能聯繫共產黨，把史鐸士放回來。經蔚斐這麼在乎史鐸士，一是因為老史的老婆賴在他辦公室里，活要見人，死要見屍，二是因為老史是美國人，弄不好是國際影響。老經急得亂了方寸，黃昌燮就趁機講條件，問經蔚斐，如果共產黨出面把史鐸士放回來，能不能斷絕和台灣總稅務司署的聯繫，向北京海關總署投誠？經蔚斐倒也爽快，表示如果深圳解放，他願率眾起義。

1865年香港跑馬地

至此九龍關起義基本上已成定局。黃昌燮得到經蔚斐的口頭許諾後，立即去找李國安，請他向上級黨組織彙報，迅速派人來和經蔚斐談判。上級組織並沒有派人來，而是指示黃昌燮向經蔚斐傳達了幾點意見：一是歡迎經蔚斐投誠；二是可以協助營救史鐸士，但要立即停止調緝私艇去台灣；三是要妥善保管九龍關在香港的全部關產，將來移交給接管的解放軍；四是應同意關警發放應變費的要求，由關警派代表來直接談判；五是清算杜熾芬和楊應虯的事，經蔚斐可以不去過問。經蔚斐倒是聽話，立即照辦，三名關警代表到九龍關總部談判，關警的要求基本上得到滿足，史鐸士獲得了人身自由。黃昌燮趁熱打鐵，問經蔚斐：共產黨說到做到，史鐸士回來了，咱們是不是把起義的事給落實了？經蔚斐這時又覺得這事太大，有必要找港督商量一下。大英帝國的這位代表倒是個明白人，知道台灣遠而大陸近，以後要學著與中共處關係了，不得不同意經蔚斐的義舉。老經還是個急性子，立即就要談投誠的條件，黃昌燮只好去找李國安。10月14日，黃昌燮陪同港澳工委派來的代表到稅務司公館同經蔚斐談判，達成了8條協議，主要內容：一是保護關產；二是換旗，升掛海關關旗；三是深圳解放、緝私總部被接管之日，經蔚斐通電北京海關總署投誠；四是派黃昌燮為稅務司代表去緝私總部安定邊境職工情緒，控制局面；五是把緝私艦隊調往寶安縣以南的深圳灣，改掛關旗；六是接管工作由緝私總部開始，其他支關憑黃昌燮簽名的介紹信接管；七是通電北京前雙方保守秘密；八是本協定為君子協定，當晚生效。黃昌燮為本協定的見證人和履行本協定的保證人。

10月19日，深圳解放，寶深軍管會成立。經蔚斐得到消息，給黃昌燮打電話，要求解放軍馬上接管。黃昌燮找軍管會主任劉汝琛請示，劉汝琛說，要接管的單位太多，海關就遲些日子吧。經蔚斐怕夜長夢多，這個講原則的人擔心萬一台灣將他革職了，他如何履行協議？20日一早，再次急電黃昌燮要求立即接管。黃昌燮只好再去找劉汝琛彙報。劉汝琛明白海關的事也怠慢不得，當即決定先接管九龍關。次日上午，軍管會接管了九龍關緝私總部，成立「九龍關臨時接管委員會」，劉汝琛為主任。11月15日，九龍關接管委員會正式成立，九龍關在香港的機構以九龍關駐香港辦事處的名義保留。這裡簡單講講劉汝琛這位新中國九龍關的首任「關長」。他由寶深軍管會而海關軍管會，大主任變成了小主任，後來調廣州，曾任廣州市荔灣區區長、廣東省中醫學院黨委書記，在該院黨委副書記任上離休，官是越做越小。由此可見，我黨當年的幹部確實能上能下。

再說經蔚斐。12月25日，海關總署署長孔原、副署長丁貴堂複電經蔚斐，肯定他保存關產有功。不久經蔚斐向廣州軍管會提出辭職，軍管會海關處根據孔原署長意見，代表海關總署挽留。經蔚斐是個聰明人，知道自己沒這個福份了，表示去意已決。海關總署於1950年2月25日批准經蔚斐辭職，發給6個月的薪津和全家返英的旅費以示獎勵。由《九龍關政策》七條意見中經蔚斐企圖保存九龍關現狀和職位，到起義協定八條措施中全面妥協，到最後打道回府，經蔚斐的心路歷程很耐人尋味。今天我們回顧歷史，知道九龍關和經蔚斐的命運都是註定了的，不可能有第二條路，這是歷史轉型期的必然結果。但如果歷史可以重演，咱們設想一下，如果經蔚斐的企圖得逞，九龍關維持原狀，總部繼續設在香港，他繼續做他的稅務司（或者改稱關長），海關繼續行使原來的職責，那麼一國兩制早就在海關成為一種現實，如今的香港海關和深圳海關可能就不存在，海關的歷史和制度就是另外一種面目。可惜的是，歷史沒有假設。

九龍關被接管後，原華南緝私艦隊還有大、小艦艇27艘留在香港，海關總署決定將這些艦艇駛回廣州。經過一番精心準備和小小的波折，1950年6月16日凌晨，艦隊駛回廣州沙面附近的白鵝潭。1950年10月，九龍海關[1]撤銷了駐香港辦事處屬下所有機構，除幾棟房子外，所有動產陸續撤回深圳。至此，九龍關在香港的「勢力」蕩然無存。

1987年，廣東省人民政府授予九龍關為起義單位。很多當年參與起義的老同志沒有等到這個日子。
