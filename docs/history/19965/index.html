駙馬征戰在外，公主卻懷孕，皇帝下令處死公主，駙馬：孩子是我的

2019-12-17 2,457條評論 3,582次閱讀 2361人點贊

在古代之時，女性的社會地位是非常低下的，受到諸多方面的限制。不過到了唐朝時期，由於社會的開放，女性的社會地位得到一定程度的提高。

到了宋朝之時，女性更是可以自由改嫁，不用受到流言蜚語的毒害。不過到了明清，統治者卻加強了思想束縛，女性們所受到的限制開始多起來，對她們的要求也更加嚴格。

就拿「貞潔」這一塊來說吧，明清時期的婦女如果不守貞潔，便猶如「過街老鼠」一般，要受到社會的歧視。這一時期，很多女性因為失去貞潔而自殺身亡。

​

圖片：宋朝時期的女性劇照

要是你以為「貞潔」只針對尋常老百姓，那就錯了。不僅平民百姓要嚴格遵守「貞潔」，帝王之家也是如此。皇室內的女性更是起到表率作用，比旁人更應該堅守貞潔。

在清朝道光皇帝統治年間，四公主懷了身孕。對皇室來說這本是喜事一件。但是道光帝卻大發雷霆，不顧眾人求情，堅決要處死四公主。這是為什麼呢？公主懷孕就要遭來殺身之罪？

當然不是，因為四公主懷孕的時機不對。當時駙馬外出征戰，獨守空房的四公主又怎能懷上孩子呢？只有一種可能，就是四公主與他人私通。這可是有失貞潔之事，所以道光帝才如此震怒。

不過四公主卻堅稱這就是駙馬的孩子，就連在外征戰的駙馬爺也快馬趕回，稱這就是他的孩子。這到底是怎麼一回事呢？咋一看還挺亂呀。

圖片：清朝道光皇帝劇照

這位公主便是道光皇帝的四女兒，壽安公主，出生於道光六年四月間，生母則是孝全成皇后。壽安公主自幼便聰明伶俐，很討人喜歡。

壽安公主的皮膚非常白嫩，猶如白雪一般。另外，其人還長得非常漂亮，幾乎繼承了父母雙方的所有優點。就這樣，壽安公主集萬眾矚目於一身，過著眾星捧月的生活。

壽安公主不僅人長得美，而且對於讀書寫字特別上心。雖說封建社會提倡「女子無德便是才」，但是疼惜女兒的道光皇帝卻不加阻止，反而還專門找老師教授四公主。

圖片：壽安公主劇照

在道光十九年時，道光皇帝特意頒布了一條命令：「嗣後無論官私大小事務，有應啟知皇后者，除本宮、四阿哥、四公主事務外。」從這一條上諭中可知，四公主是被賦予了特權的，足以見得道光皇帝對她的疼愛。

慢慢的，壽安公主出落得越發水靈，同時也到了談婚論嫁的年紀。對於疼愛四公主的道光皇帝來說，自然要深思熟慮，給她找一位最般配的夫君。

精挑細選之下，道光皇帝有了人選，這位駙馬便是蒙古奈曼部的一個王子，名為德穆楚克扎布。之所以看中他，不僅因他是青年才俊，還出於政治上的考慮，利於雙方修好。

這樁婚姻也可以說成是一樁政治聯姻，但是當事人卻非常恩愛。

圖片：壽安公主與德穆楚克扎布聯姻劇照

正常來說，壽安公主應該遠嫁到蒙古去。但是道光帝卻很疼愛四公主，不願讓她嫁那麼遠，便在京城修建了一座公主府。讓公主和駙馬兩人住在這裡。也就是說，德穆楚克扎布成為了上門女婿。

不過，德穆楚克扎布卻並不介意，他也被四公主的美貌迷住了，只要能與她住在一起，成為上門女婿又如何？不過沒多久，德穆楚克扎布便被道光帝賦予了一項任務：率軍外出征戰。

因此，壽安公主的肚子一直都沒有動靜，事情就這麼相安無事的過去，不過有一天服侍壽安公主的嬤嬤發現了四公主的重要秘密，並私下報告給了道光帝。

圖片：被四公主的美貌迷住劇照

這個消息就是「四公主懷上了孩子」。這一消息如一聲驚雷，險些讓道光帝崩潰。駙馬征戰在外已經一年有餘，壽安公主獨自一人，現在懷了孩子，這孩子自然不是駙馬的。

道光帝最疼愛的四公主，竟然做出這麼出格的事情來，不但讓皇室蒙羞，而且很可能會得罪聯姻的一方，事態進一步擴大，引起戰爭都有可能。

對此，道光帝憤怒之餘，立即下令要處死壽安公主，也算給駙馬一個交代。但是壽安公主卻拒不承認，堅持稱肚裡的孩子就是駙馬的，並且還說如果不相信就召回駙馬，一問便知。

圖片：壽安公主懷上孩子劇照

道光帝對壽安公主的辯解更加惱怒，認為她不但不守貞潔，而且還謊話連篇，駙馬難道還能飛回來與你同房不成？很快，征戰在外的駙馬收到了四公主要被處死的消息，於是快馬加鞭連夜趕回。

經過一路狂奔，還好及時趕到，急匆匆的向道光帝喊了一句話，更是讓道光帝摸不著頭腦。這句話就是：「陛下，刀下留人！」公主背著你出了軌，你居然還要為她求情？在駙馬的解釋下，道光帝才恍然大悟。

圖片：駙馬快馬加鞭劇照

原來，壽安公主肚子中的孩子就是德穆楚克扎布。雖然駙馬征戰在外，但是卻太思念公主，經常偷偷的溜回京城，只為見公主一面。

但是按照清朝的規定，外出征戰的駙馬要回京的話是需要向皇帝報告的，徵得同意之後方能回京。而駙馬擔心向皇帝請示回京探親，會遭到他人笑話，所以這才沒有請示皇帝，選擇偷偷地溜回來。

聽了駙馬的解釋，道光帝的氣也消了。原來四公主並沒有做出讓皇室蒙羞的事，險些錯怪了她。自責之下，道光帝也給了他們一個台階下，對外宣稱駙馬曾中途回京一次，這個孩子應該就是那時候懷上的。

圖片：駙馬偷偷溜回來只為見公主一面劇照

誤會解除之後，道光皇帝也賜給了德穆楚克扎布一個特權，以後回京都不用再向自己請示了，想回就直接回來。通過這件事我們可以看出，古時人們對於女子的「貞潔」十分重視。如果女子失去貞潔，那麼整個家族的人都會蒙羞，終生都要生活在陰影之下。

很多承受能力不強的女性便選擇自殺來結束性命。所幸，現代社會的女性再也不用遵循古時的「三從四德」了，也不用再受到精神的壓迫。
