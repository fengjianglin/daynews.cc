春秋霸主吳王闔閭，“魚藏劍”暗害兄弟，臨終死因竟是馬虎大意？

2019-12-21 677條評論 2,191次閱讀 446人點贊

《吳越春秋》記載：吳王闔閭在位期間「立城郭，設守備，實倉凜，治兵庫」，對其評價：「任賢使能，施恩行惠，以仁義聞於諸侯」，吳王功績可謂名垂青史，但要論一生最後悔的事情，就是容易滋生驕傲輕視的情緒。

​

闔閭是春秋五霸之一，響噹噹的諸侯霸主，但闔閭的早期人生坎坷不平，在成為吳王之前，他有一個鮮為人知的名字，公子光。公元前525年，公子光就能夠率軍和楚國來一次猛烈的交鋒，起初因為軍力數量的對比差異明顯，公子光率領的吳軍屢戰屢敗，一度士氣低落。

可是公子光就像一隻打不死的小強，不僅沒有因此嚇得丟盔棄甲，逃竄回國，反而毫不氣餒地訓練士兵，吸取失敗的教訓。趁著對方洋洋得意的時候，公子光在深夜整頓部隊，對敵人進行了一次出其不意的襲擊，前一秒還準備慶功的楚軍瞬間潰敗，公子光由此取得了一次酣暢淋漓的大捷。《史記》記載：王僚二年，公子光伐楚，敗而亡王舟。光懼，襲楚，復得王舟而還。自從打了這場勝仗，軍事奇才就成了公子光的固定人設。連父親夷昧都覺得他是能夠領軍打仗的將才，但不適合做守成之君，從而王位落到了公子僚的手裡。對此，公子光一直深有怨言，吳王公子僚也對這位同父異母的兄弟心存忌憚，繼位後的第一件事，就是打著攘外戍邊的旗號偷偷收繳了公子光的兵權。

​

公子光一早就意識到吳王僚的行為，讓他心懷不滿、大發牢騷，但是擺在眼前的事實告訴他，暫時沒有反抗的資本，必須要養精蓄銳、等待時機。公元前519年，公子光終於等到了一個千載難逢的機會，這次機會是上次的楚國帶來的，當南方的楚國聽到吳國內部出現分裂的局勢時，敲鑼打鼓地聯合七個小國家來攻打吳國。等到吳國的其他將帥意氣風發地率軍迎戰時，卻發現都不是人家的對手，萬般無奈的情況下，王僚急忙下令請公子光出山，將他視作最後一根救命稻草。可是他萬萬沒有想到的是，這根救命稻草是把雙刃劍，公子光在重新掌握兵權後就變成了吳王僚的催命符。

《史記》記載：十三年春，吳欲因楚喪而伐之，使公子蓋余、燭庸以兵圍楚之六、灊。使季札於晉，以觀諸侯之變。楚發兵絕吳兵後，吳兵不得還。

​

時機漸漸成熟後，公子光就在民間尋覓一個名叫專諸的職業殺手，特意鑄造了一把鋒利的匕首，短得足以藏在魚肚當中。趁吳王僚擺慶功宴席的時候，專諸上前就是一刀，結結實實地奪取了王僚的性命，公子光就這樣順理成章代立為王。

《史記》記載：公子光詳為足疾，入於窟室，使專諸置匕首於炙魚之中以進食。手匕首刺王僚，鈹交於匈，遂弒王僚。

登上國王寶座的公子光，不再過著整天提心弔膽、擔驚受怕的日子，而是泰然自若地將自己的才華盡情揮灑。吳王闔閭的能力出眾，識人辨人的眼光也是一絕，獨到犀利的眼神一掃，就能看到兵家始祖孫武。吳王闔閭還找到了天天想著報殺父之仇的伍子胥，刺殺吳王僚的主意就是他提出的，況且伍子胥的軍事才能的確不容小覷。經過吳王闔閭的不懈努力，吳國的國力日益強盛，百姓的生活也逐漸殷實富裕，吳國就此成了春秋末期最厲害的國家之一。既然有了想要的資本和實力，公子光就琢磨起鄰近楚國的主意，他要按照自己的見解和規劃，把春秋局勢打造成自己想要的樣子。

懷揣著改變世界的夢想，吳王闔閭興沖沖對楚國進行了征討，這一次他得到了一個意料之外的進軍結果，就是一路勢如破竹、節節高歌，打得楚軍毫無還手之力，楚王也灰溜溜地遷都，以此躲避鋒芒畢露的吳軍。吳王闔閭的主動出擊，讓吳國看到了稱王稱霸的希望，連地大物博的楚國都不是對手，又有什麼能夠阻礙他們的呢？雖然楚國和吳國是一對不打不舒服的冤家，但是對於吳國來說，最痛恨的國家卻是大發戰爭財的鄰國越國。縱觀歷史可以看到，春秋時期的諸侯國思想戰略，傾向於遠交而近攻，因為去攻擊遠方的國家，得到的回報遠遠低於付出的代價，回報率低的不是一點半點。況且兩國交戰，難免會被鄰國鑽個小空子，鷸蚌相爭，漁翁得利，於是吳王闔閭開始專心致志地攻打越國。討伐越國的同時，吳王闔閭也要給敗軍之師楚軍敲敲邊鼓，他下令吳軍一路南下，震懾一下蠢蠢欲動的楚軍，這一次闔閭將楚國逼迫到了遷都的地步。

《史記》記載：三年，吳王闔廬與子胥、伯嚭將兵伐楚，拔舒，殺吳亡將二公子。接二連三的勝利將吳王闔閭的名聲進一步打響，無論是聲望還是實力，一時之間，還真沒有能和吳王闔閭一較高下的君主。

​

所向披靡的吳軍勢如破竹，連附近的鄰國軍隊見到都要繞路走，這下可算實打實地滿足了吳王的虛榮心，儼然享受到周王室的待遇。吳國的下一個目標就是打敗越國，挫挫它的銳氣。聽到吳軍到來的消息，越國百姓表現出空前一致的團結，對於屢戰屢勝的吳軍，他們自然不敢小覷。更何況這次伐越之戰由吳王闔閭親自乘著戰車指揮，生死存亡的重要關頭，越國上下定要嚴陣以待。這時的吳國內部卻出現了不一致的聲音，兵家始祖孫武和復仇狂人伍子胥堅決反對吳王闔閭親自督戰，國軍是國之根本，一旦國君有什麼不測，動搖的可是整個吳國。任爾東西南北風，我自嵬然不動，這是吳王闔閭的真實態度，他渴望能夠成就霸業，好好地做一回春秋霸主，這場戰場和吳國的利益相關，必須要親自指揮才能凸顯霸氣。闔閭的一意孤行，讓伍子胥和孫武傷透腦筋，但畢竟吳國是闔閭說了算，伍子胥和孫武只能最大限度地保證闔閭的生命安全。

​

吳軍的精銳戰士時刻把守在吳王闔閭的周邊，戰場局勢瞬息萬變，這一點孫武比誰都懂，雖然吳軍目前處於優勢，但不代表會一直佔領上風。眼見形勢一片大好，吳王闔閭再也坐不住了，他興高采烈地下令全軍繼續前行，打到紹興一帶，好好地敲打一下越國君主。恰好這一時機，越國有個叫做靈姑浮的大夫，他深知越軍遠不是吳軍的對手，從而冥思苦想，設計出一招「臨陣倒戈」的計謀。遠在戰場指揮的吳王闔閭，收到了出自靈姑浮之手的投降書，他想當然地以為是吳軍的銳意進取嚇倒了靈姑浮。欣喜若狂的闔閭讓左右帶靈姑浮覲見，說時遲那時快，靈姑浮猛然掏出藏匿的戈，瞬間割掉了闔閭的腳拇指。場面一度失去控制，群龍無首的吳軍潰散而逃，失血過多的闔閭又痛又氣，躺在馬車上永遠地闔上了雙眼。可憐一代春秋霸主，竟因輕敵大意枉送性命，令人扼腕嘆息。
