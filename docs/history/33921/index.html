有人稱李淵本無心皇位，為何最後還是謀反稱帝？真是李世民逼的？

2019-12-24 1,118條評論 3,553次閱讀 685人點贊

作為大唐的開國皇帝，李淵完全可以說是相對結局最悲慘的一位帝王了，才剛剛建立大唐不久，還沒來得及享受一下舒服日子，就因為自己兒子們的內鬥而不得不最後讓位給李世民了。

說來也巧，有人曾為李世民倒是找了不少借口，稱是李淵自己本身不想當皇帝了，李世民這才不得不接手。甚至還有人扒出了李淵太原起兵之前的黑歷史，說李淵本身就不想起兵，還是李世民強力要求的，所以說李世民稱帝是相對更符合情理的。

那麼李淵當初到底是怎麼走上這條道路的呢？這裡我們先從李淵自己本身的身份開始說起。李淵有六個身份，一是西魏名將李虎之孫；二是唐國公李昞之子；三，八歲時就獲封唐國公；四，文獻皇后（隋文帝的妻子）是他姨母；五，隋文帝是他親姨夫；六，楊廣是其表兄弟。

就這麼四個字來形容：皇親國戚！這是絲毫都不過分的。但是卻也正是這麼一位楊廣最最信任的表兄弟，最終給了他迎頭痛擊，窩裡反了。

李淵作為隋朝外戚，不光深受隋文帝楊堅夫婦的喜愛，在隋文帝時期是楊堅非常看好的一位後生，而在楊廣在位的時候更是非常看重自己的這位表兄，當時楊廣親征高句麗，李淵便是負責督運糧草的。楊廣將如此重任如此放心地交給了李淵，可見李淵的地位和受寵程度。（三軍作戰，糧草可是命脈咽喉）

當然，年輕時候的李淵可也是才能斐然，令人欽佩。大業十一年，李淵率兵擊破毋端兒起義軍，大業十二年，李淵率兵抵禦突厥，屢傳大捷，大業十三年，特賜太原留守，他開始拾掇其了祖上的榮光，成為一位權傾一方的諸侯。

這個時候的李淵可謂是春風得意，仕途輝煌！這個時候的他完全就是沒有謀反的心思的。本身就官居顯赫，諸侯一方，又何必再要求什麼呢？更何況楊家對其還非常照顧，這個時候謀反可就真的是不忠不義之人了。

​

但是此後不久，隋朝江山被楊廣給「糟蹋」完了，自己也被宇文化及給逼殺了，天下這個時候是群雄並起，李淵本身便坐鎮一方，手下有兵有將，幾個兒子又非常的爭氣。這個時候的李淵也還正值壯年，在這麼個群雄逐鹿的時代里，要不你就選擇一人投奔，要麼便自立為王，征戰群雄！

李淵會選擇偏安一方，與世無爭？或者投奔他人，得一安穩？根本不可能！且不說他有著渾厚的資本，更何況他也算得上「根正苗紅」，先確立一傀儡，後期廢之再自己「不得不」接受禪讓，這不是挺好的嗎？

能成為一方諸侯的李淵自然是不傻，這道理他都懂，但是他卻不能如此說和做！只有自己是不得已才起兵爭雄，這才能堵住有心之人的嘴！正如同宋太祖趙匡胤一樣，「黃袍加身」的天賜之子才能更顯得出其正統和天意！趙匡胤有自己的弟弟和手下們幫忙，李淵也是一樣，有自己的兒子李世民以及謀士和將軍！

最後李淵這才「不得已」而起兵，不過確實以「勤王楊侑」的名頭來昭告天下的。李世民或許從旁協助，但是絕對不是主力助手。畢竟要是李淵真的不想，誰拉他都沒用，正如宋太祖趙匡胤一樣，他要是真的不想當皇帝，什麼所謂的「黃袍加身」都沒用，甚至可以說這場「黃袍加身」的鬧劇說不定正是他授意的，李淵也是一樣！

後世說是李世民的功勞，其實很可能是為了從某種程度上來美化李世民的，畢竟玄武門之變這種事情還是有些不好的影響的，而李世民又是千古一帝、一代明君！這些其實都是完全可以理解的~大家覺得呢？
