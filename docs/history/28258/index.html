庄蹻入滇——雲南入中國版圖起步於兩千多年前一個楚人的神秘之旅

2019-12-21 163條評論 3,210次閱讀 1243人點贊

雲貴高原被稱為「西南夷」地區，遍布著上百個大大小小的由少數民族建立的部落、酋邦或方國。據彝文文獻記載推算，古夜郎立國在春秋中葉，至西漢成帝河平二年（前27年）國滅，大約有600多年的歷史；滇立國及滅國時間稍晚一些。夜郎國以今貴州為中心，佔有貴州大部、雲南東部、四川西南部和廣西西北部的廣大區域；滇國在夜郎以西，如《史記》所說：「其西，靡莫之屬以十數，滇最大。」滇國以今昆明、呈貢為中心，佔有環滇池周圍數百公里土地。

西漢時西南夷分布圖

夜郎國及滇國人祖祖輩輩在雲貴高原上生活，日出而作，日落而息。特別是夜郎地區的居民，高山深塹的阻隔，他們很少受到外界驚擾。
構成夜郎國及滇國主體民族的夷、濮、越三大族系及稍後進入夜郎境內的苗瑤族系，在封閉式的生活圈子裡繁衍了一代又一代人。他們很少有人想到過天外有天，山外有山。「夜郎自大」的成語雖是歷史的誤會，卻也表明了夜郎國及滇國人囿於地理環境的影響，信息極為閉塞的事實。如司馬遷所說：「以道不通故，各自以為一州主，不知漢廣大。」夜郎國及滇國人處在自我滿足的狀態中。因此在數百年間，當地經濟社會的發展十分緩慢。終於有一天，這一切都為來自楚國的庄蹻的隊伍所打破。

夜郎國地圖

楚懷王二十八年（前301年），楚國受到秦國軍隊的大舉進攻，楚將唐昧戰死，楚國滅亡在即。楚將庄蹻受命西進，取道雲貴，北入蜀境，包抄秦軍，切斷其後路，以實現夾擊秦軍之目的。
庄蹻佔領且蘭（今貴州福泉）、夜郎後，並未停留，而是繼續往西攻下滇國，佔領了大片土地。適逢秦國出兵攻陷巴（今重慶市及四川省東部一帶）及黔中一帶，庄蹻回楚的路已斷絕，不得不留下來。庄蹻入鄉隨俗，他不僅當上了滇國國王，也統治了包括夜郎、且蘭在內的西南各方國。

庄蹻入滇

這次「庄蹻王滇」傳說最早見於《史記·西南夷列傳》，是考察西南邊疆族群起源的必征史料之一。
史籍上關於庄蹻伐滇的文獻不多。
《史記.西南夷列傳》曰:「始楚威王時，使將軍庄蹻將兵循江上，略巴、黔中以西。庄蹻者，故楚莊王苗裔也。路至滇池，地方三百里，旁平地，肥饒數千里，以兵威定滇屬楚。欲歸報，會秦擊奪楚巴、黔中郡，道塞不通，因還，以其眾王滇，變服從其俗，以長之。」
《華陽國志.南中志》曰:「楚頃襄王遣將軍庄蹻溯沅水，出且蘭，以伐夜郎，牂柯系 舡於且蘭。既滅夜郎。而秦奪楚黔中地，無路得歸，遂留王之，號為庄王。以且蘭有椓舡牂柯處，乃改其名為牂柯。分侯支黨，傳數百年」。
《後漢書.南蠻西南夷列傳》曰:「初，楚頃襄王時，遣將軍庄豪（蹻）從沅水伐夜郎，軍至且蘭，標船於岸而步戰，即滅夜郎，因留滇池。以且蘭有椓舡牂柯處，乃改其名為牂柯」
這些史料的始作俑者是司馬遷，他得到的材料是第一手的。

漢武帝元狩四年(公元前119年)，漢王朝在反擊匈奴的戰爭中取得輝煌戰果，自此「匈奴遠遁，而幕南無王庭」。北方邊患的平息使漢武帝得以將目光投向遼闊帝國的西南一隅。元鼎六年(公元前111年)，漢朝的南征大軍四路並進，席捲雲貴高原東部直至川西山地，滅且蘭，降夜郎，誅莋侯，斬邛君，「西南夷」諸部聞訊無不震恐，紛紛內附朝廷，請置官吏。元封二年(公元前109年)，武帝遣將軍郭昌、中郎將衛廣徵發巴蜀士卒，深入雲貴高原腹地，征討無意內附的西南夷殘餘部國。最先淪於漢軍鐵蹄之下的，是曾經屢犯漢朝使節的勞浸、靡莫二部(今雲南東北部)，與之「同姓相扶」的滇國(今雲南中部及東部地區)獨木難支，於大兵壓境之際舉國歸降。武帝在滇國故地設益州郡，並賜滇王金印，令他「復長其民」。元封六年，漢軍又擊敗西南夷各部中最為架鶩不馴的昆明(今雲南西部)，將其地併入益州郡。持續六年之久的平定西南夷戰役，最終以漢軍的全線告捷落下帷幕。
在這次戰役中，西南夷各部君長大多身死國滅，唯獨滇王盡享殊榮，得以善終(夜郎國君也被賜印封王，但後遭漢廷誅殺。)
漢武帝何以對這個僻遠小邦的君主恩寵若是?據說就是滇國與華夏的關係非比尋常，根據司馬遷在《史記·西南夷列傳》中的記載，滇國王室來自華夏，是戰國時期楚將庄蹻的後裔:「 始楚威王時，使將軍庄蹻將兵循江上，略巴、(蜀)黔中以西。庄玲者，故楚莊王苗裔也。蹻至滇池，(地)方三百里，旁平地，肥饒數千里，以兵威定屬楚。欲歸報，會秦擊奪楚巴、黔中郡，道塞不通，因還，以其眾王滇，變服，從其俗，以長之。」
作為這則史料的始作俑者，司馬遷曾參加元鼎六年遠征西南夷的軍事行動，先後到達邛、莋、昆明等地。如所周知，司馬遷寫作《史記》除參考大量歷史文獻外，還網羅天下「舊俗風謠 ,「傳聞異說」以拾遺補闕，關於秦漢間人物事件的記述多源於第一手材料，極富史學價值。有鑒於此，加之司馬遷「奉使西征」的特殊經歷，後世一些史家便臆測「庄蹻王滇」之說必采自當地，故而深信不疑，班固在《漢書》中予以原文照錄即是著例。

司馬遷（約公元前145年至約公元前90年）字子長，史學家、文學家、思想家、經濟學家，漢武帝時任郎中、太史

《史記》所載的滇國始祖庄蹻到底是何許人也?是草莽裊雄，抑或宗室勛舊?按照司馬遷在《史記·西南夷列傳》中的說法，庄蹻是「楚莊王苗裔」，奉命拓邊的楚國將領。但他在《史記·遊俠列傳》中卻說:「跖、蹻暴決，其徒誦義無窮」，將庄蹻與曠世大盜柳下跖並稱對舉。　庄蹻是楚國人，他的生卒年月不太清楚，其活動年代大約是公元前四世紀末到三世紀初。歷史文獻上經常把他和時代較早的著名的起義領袖柳下跖並提，說他們「無猗頓之富」，又誣衊他們為「盜賊」，為小偷，會用飴糖黏住門楗，使開門偷竊時不發聲音。他們都出身子受壓迫的勞苦人民家庭；敢於奪取剝削者的不義之財。過去《史記·西南夷列傳》說庄蹻是出身於「楚莊王苗裔」的「將軍」；這一點始終不能從其他記載中得到證實。大概因為庄蹻姓庄，後來他到雲南曾被各族人民尊稱「庄王」，司馬遷遂附會以為他是楚國貴族了。

古滇王莊蹻像(雲南省昆明市)

關於庄蹻伐滇的線路，《華陽國志·南中志》:「溯沅水，出且蘭，以伐夜郎」。而范嘩《後漢書·南蠻西南夷列傳》關於「庄蹻王滇」的敘述也是如此:『初，楚頃襄王時，遣將庄豪(按:即庄蹻)從沅水伐夜郎，軍至且蘭，標船於岸而步戰。既滅夜郎，因留王滇池。」
庄蹻溯沅江而上到達牂牁國境的清水江，又由其支流重安江舟揖黃平重安鎮，由西至福泉，完成了他們的水路行程。再由夜郎及旁小邑陸路直奔滇池。
後來的研究者認為，庄蹻入滇選擇沅水西進，主要是由於沅水流域為巴、越、楚錯居雜處之地，民族關係相當複雜，有利於庄蹻軍隊乘隙通過。

庄蹻選擇這條通道，有著深遠的戰略意義;楚威王試圖將西南夷地區佔為己有，使將軍庄蹻由此而過，從中可了解到地理環境，自然資源，以及民族發展等實況，給征服這一地區作好思想準備。牂牁國小，勢單力薄，夜郎還處於民族部落階段,雖旁有邑君，但居住分散，不會給庄蹻構成軍事大對抗。
二十世紀八十年代至今，居住在清水江兩岸的村民在清水江淘金時，陸續在河床下幾米深處發現戰國青銅兵器，數量多達幾百件。這批青銅兵器有劍、鋪、戈、矛、刀、杴、斧。 這批青銅器一方面包含著楚、巴、越文化，另一方面又保持著自身的一些特點。
這批青銅器全是兵器，不見生產工具和生活用具;出土地點是在清水江河床之下，肯定不是墓葬和遺址，也不是由其它地方水沖而來，顯然是水上交戰所為。
庄蹻入滇，拉開了外來強勢文化開發雲貴高原的序幕，西南地區的封閉狀態被打破，雲南和內地建立直接的政治聯繫是從庄蹻入滇開始的。《史記·西南夷列傳》中的「索隱述贊」曰：「西南外徼，庄蹻首通。」庄蹻的開拓之功，不可磨滅。
而庄蹻入滇溯沅水而上的路線選擇，在其後的兩千多年顯示了極其重要的意義。可以說中原王朝兩千年對西南的開發就是沿著這條路線來進行的：宋朝以前，朝廷想的是怎樣打通沅水下游到中上游的通道，以及怎樣在沅水中上游的五溪紮下根來；而在元明清的年代，沅水上游到雲南的陸上驛道已經開通，沅水水道交通也已經穩定下來。倒是水陸交通轉換的那個節點——湘黔交界的臘爾山苗疆麻煩不斷。
