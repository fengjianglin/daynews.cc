隋文帝沾沾自喜，五個兒子出自一母，其實越是親兄弟搶皇位越殘忍

2019-12-14 2,317條評論 3,512次閱讀 1595人點贊

列位，歡迎來到照理讀史。咱們中國古代講，無情最是帝王家，為了爭奪皇位，親兄弟之間互相殘殺的事兒太多了。

當然也有人說，你看雖然名曰親兄弟，但是咱們不要忘了，皇帝一個人，三宮六院七十二嬪妃，他的那些妃子們平時就經常互相爭鬥。

這些年來咱們經常看各種各樣的宮斗劇，那是勢同水火，所以你看既然妃子們都已經爭鬥了，那麼妃子們生的兒子不把對方看成自己親兄弟，互相喊打喊殺，置對方於死地的那也很正常。

這似乎是這麼個理兒，可是反過來想一想，那麼同一個妃子生的孩子那是絕對的實打實的親兄弟了，他們就沒有爭鬥嗎？不是，有時候爭鬥的也是很狠的！

最知名的一對純親兄弟——康熙皇帝的倆兒子，老四胤禛和十四胤禵，這可是一母所生，絕絕對對的親兄弟。

按道理講應該打仗親兄弟上陣父子兵，可是不是，當九子奪嫡的時候，老四和老十四是倆陣營的。甚至現在都還傳說，是老四搶了老十四的皇帝位子的。

也就是說同一個母親的倆兒子，爭鬥得更加厲害。講到這兒，咱們就不得不提另一個歷史人物了——隋文帝楊堅。

楊堅結束了幾百年以來的戰亂，重新統一了全國，也是不世之功。楊堅沾沾自喜的，除了自己能統一全國，還有一個事。

楊堅有一次跟大臣們聊天說，你們看看，咱們古代歷史上各個帝王將相，經常是三妻四妾很多子女，但是他們都比不過我。

為什麼呢？我有5個兒子5個女兒，我這10個子女都是獨孤皇后一個人所生，你們看到了吧？我跟獨孤皇后多恩愛！

古代所有帝王甭管有什麼豐功偉績，有一個在這個事上能超過我的嗎？我這5個兒子全都是親兄弟，所以我大隋朝的江山一定是鐵桶一般！

當然，楊堅不跟別的女人生孩子，不是他不願意、不是他不想，是因為他家裡那個母老虎看得太嚴了。

而楊堅的5個兒子雖然是真真切切的親兄弟，但是他們爭鬥起來也是不遑多讓的。太子楊勇的位置怎麼被人弄下來的？不就是被老二耍盡了陰謀手段給撬下來的嗎！而且楊堅賴以自豪這5個兒子，沒有一個有好下場的，更不要說李唐王室了。

李唐王室跟楊家還有親戚關係，李淵的三個兒子——太子建成、秦王世民、齊王元吉，這仨人也是一母所生，結果怎麼樣？李世民可是親手用箭射死了自己的大哥！

無情最是帝王家，這和是不是一個媽生的完全沒關係，因為在權力的巨大誘惑下，人是什麼都能幹出來的！
