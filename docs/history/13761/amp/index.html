史上驚人的巧合，把明朝皇子名字拼一起，能組一張化學元素周期表

天天要聞
2019-12-10

歷史上有很多驚人的巧合，比如說名為「麗華」的女子，無論是陰麗華還是張麗華，那都是很知名的大美人，當然這很容易理解，畢竟「麗華」本身就有美人的意思，可能古人們看這個女孩子很美麗，就會因此而給她取這樣的名字，但是還有另一種巧合，卻很讓人感到困惑了。

那就是明朝皇子的名字，竟然與化學元素周期表也就是「鈹鎂鈣鍶鋇鐳……硼鋁鎵銦鉈」中間的字眼高度重合，拼在一起就能夠組成一張化學元素周期表。要知道元素周期表是俄國化學家門捷列夫在19世紀中期制定的，而明朝皇子們的生存時間則是從1368年到1644年，這兩者之間不僅差了空間而且還差了時間，為何會有這種巧合現象呢？

一切還得從明朝的開國皇帝朱元璋開始說起，生於1328年的朱元璋，小時候的名字叫做朱重八，他的父親朱五四，祖父朱初一，曾祖父朱四九，這些名字一看就非常簡單，簡單到相當於沒有名字的那種，這主要是因為元朝的統治者排斥漢文化，不讓普通老百姓取正式的名字。

這種家族背景很顯然會讓朱元璋名字非常看重，所以他長大以後就把名字給改了，朱元璋便是「誅元璋」，寓意著一種滅亡元朝的鋒利兵器。朱元璋對自己的名字極為講究，對於子孫後代的名字也有著嚴格的規定，他二十六個兒子的名字都為單字，且都為「木」字偏旁的漢字。

而從朱家第三代也就是朱元璋的孫子輩開始，朱元璋便嚴格規定所有人的名字都得是雙字，第一個字為他所定下的輩分表中的字，第二個字則是含有五行偏旁的字，並且規定以「火土金水木」為順序依次循環，這樣一來，朱家後人的命名就有了極其嚴格的規矩。

朱元璋之所以要制定後人們的命名規矩，一方面是因為他自己吃了名字的虧，所以對此極為講究，另一方面也是因為他存有像秦始皇那樣的「萬世永昌」的嚮往，秦始皇當年給自己的後人規定到一萬代以後的名字：秦二世、秦三世……秦萬世，但是後來卻二世而亡。

於是朱元璋就吸取了這方面的經驗教訓，並不直接取到萬世以後的名字，而是制定了「五行相生」的命名法則，想要因此而讓自己的大明王朝江山永固，讓自己的子子孫孫世代不絕，但是到明朝末年的時候，明朝幾百萬的宗室子弟基本上也就剩下個十分之一。

朱五太子朱慈煥本來已經跑掉了，也因為已經堅持這種命名方式，而被人舉報，最終全家被抓。可見這種命名方式，在一定程度上是有弊端的，而朱慈煥的被抓尚且不是最大的弊端，它最大的問題在於朱元璋強行規定，名字中最後一個字得有五行偏旁。

然而漢字就那麼多啊，想要不跟祖先重名，朱家的皇子皇孫們那是想盡了各種辦法，什麼生僻字都用光了，到最後只能夠自己造出一些對應五行的字，什麼鋅、鑭之類的，這些漢字也就通過朱家的家譜給保存了下來。

晚清時期的化學家徐壽在翻譯元素周期表的時候，對於如何將那一些外文字母，翻譯成對應的漢字極為頭疼，偶然之間看到了明朝皇子的這個家譜，極其驚喜地發現，原來漢字中還有這麼多與金屬礦物相關的字眼，便直接借用明朝皇子們的名字，來翻譯化學元素周期表，這也造成了元素周期表與明朝皇子們的名字，看上去很「巧合」的情況。
