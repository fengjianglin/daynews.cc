美食者不必是饕餮客，而製造美食者，必定閱過人間百味，又能返璞歸真，回到食物本身。

哈嘍，大家好，我是愛啰嗦的厲大叔，歡迎大家來和大叔一起交流美食，如果想了解更多的美食製作，請大家關注大叔，大叔每天都會和大家分享一些美食製作的方法的。今天要跟大家分享的美食是：南京的經典名小吃—-美齡粥。

說到這個美齡粥，這裡面還有一個小故事。相傳在民國時期，國民女神宋美齡有一段時間胃口不太好，整天茶飯不思。這可急壞府上的廚師們，於是這些頂級廚師們試著用香米和豆漿等等一些食材熬制了一碗粥，宋美齡喝了以後胃口大開，從此最鍾愛這一碗粥，後來經過流傳，慢慢的這一碗粥的做法就流傳到了民間，起名叫做「美齡粥」。

大叔前幾年去南京，有幸在南京大排檔吃了兩碗美齡粥，此粥外表看起來細如白雪，上面點綴了幾粒泡發開的枸杞和玫瑰乾花的花瓣跳色，一看就非常的有食慾。

美齡粥是採用現磨豆漿熬制，加入老冰糖，使其有著香甜濃郁的口感，除了香米之外，還加入了山藥泥，使其溫潤細膩，非常的軟糯，喝上一碗，清爽細膩，回味香甜。

美齡粥介紹完了，下面給大家分享簡單的做法：

= 食材 =

主料：黃豆100克，糯米80克，梗米40克，鐵棍山藥200克

調味料：冰糖適量，枸杞適量，玫瑰乾花瓣適量

= 做法 =

第一步：將黃豆洗凈放入碗中，加入清水浸泡1夜，使用豆漿機或者破壁機將浸泡好的黃豆打成豆漿，過濾去雜質備用。

第二步：糯米和梗米混合到一起，淘洗兩遍後用清水浸泡1-2個小時（夏季1小時，冬季2小時）備用。

第三步：將山藥洗凈，切成小段，放入蒸籠中蒸熟，蒸熟後的山藥去皮，裝入保鮮袋中，用擀麵杖擀成山藥泥備用。

第四步：取一個湯鍋，將打好的豆漿倒入鍋中，加入一碗清水稀釋一下，大火煮沸，然後將浸泡好的糯米和梗米放入鍋中，轉小火，加蓋燉煮30分鐘左右。這期間可以將枸杞用溫水泡發一下。

第五步：待米煮的軟爛以後，將山藥泥和冰糖放入鍋內，不停的攪拌防止粘鍋，繼續燉煮至粘稠即可出鍋。

第六步：將泡發好的枸杞和玫瑰花瓣撒在粥上點綴一下，至此美齡粥就製作完成了，是不是非常簡單呢？

小貼士：

如果嫌自己現磨豆漿太麻煩，可以直接買來豆漿做。

山藥蒸熟以後削皮不會使皮膚髮癢哦。

糯米和梗米煮的時候只需要煮到米開花就可以了。

自民國此粥問世以來，美齡先生一直對其鍾愛有加，她一生注重養生及保養，一直到106歲高齡時依然容顏常駐，這與她自身的健康飲食也是有很大關係的，您也想像她一樣嗎？那就從試試這一碗粥開始吧。
