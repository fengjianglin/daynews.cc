一位讓中國王朝穩定發展四十年的女子，無心皇權，卻被迫走到台前

2019-12-21 1,635條評論 2,678次閱讀 1229人點贊

西晉王朝的混亂和無能，導致五胡亂華，中原淪喪。司馬睿逃到建康，建立東晉後，皇帝們的表現也很不讓人滿意。幸好，東晉有士家大族的支持，勉強能夠維持下去。

在東晉建立過程中，王氏家族起了很大的作用，因此有「司馬與王共天下」的說法。皇權鞏固後，司馬皇帝卻一個連著一個短命，龍椅上走馬燈似地換著主人。真正支撐著皇權的是褚太后，一生經歷六帝，保持了東晉王朝的穩定。

褚太后名褚蒜子，是東晉第四個皇帝晉康帝的皇后。她出身當時的名門褚家，父親是衛將軍褚裒。褚蒜子一成年便嫁給琅琊王司馬岳，公元342年，晉成帝駕崩，司馬岳成為晉康帝，褚蒜子被立為皇后。

僅僅兩年後，晉康帝病逝，褚蒜子又變成了皇太后。東晉皇帝的壽命普遍很短，活過三十歲的都不多，不知道是不是上天對他們的懲罰。褚太后兩歲的兒子司馬聃即位，太后開始臨朝稱制。此後四十年里，褚太后多次還政給皇帝，又不得不多次臨朝，也算是創下了一個記錄吧。

褚太后臨朝時也就20歲左右，對國政顯然是沒什麼經驗的。不過，東晉王朝的皇權本身就不強，朝中有士族高官，郡縣政務更是控制在地方豪族手中。這樣一來，需要皇帝處理的公務就不多了。

褚太后最為人稱道的是，她沒有像漢朝太后那樣大封娘家人，造成龐大的外戚集團。不僅如此，她回到娘家，堅持以原有的禮儀對待長輩。從而贏得了朝野上下的一片稱頌。

隨著小皇帝司馬聃的長大，褚太后臨朝十三年後，主動還政，退居後宮。對於沒有政治野心的褚太后來說，這樣的結局是令人滿意的，終於可以擺脫朝政，安靜地休養身心了。哪知，她的臨朝稱臣其實才剛剛開始。

司馬聃只當了三年多皇帝就死了，比他父親活得更短，而且還沒來得及留下子嗣，於是立司馬丕為皇帝，他是晉成帝的長子，晉康帝的侄子。

不過，司馬丕對政事不感興趣，喜歡像魏晉名士那樣，吃藥食散，迷信方士，沒多久就病倒了。大臣們見皇帝病重不能理政，只能把褚太后又請出來，重新臨朝。

司馬丕的病拖了三年，終於熬不下去，死時也只有24歲。褚太后再立他的弟弟司馬奕為帝。這個司馬奕此時已經23歲，看上去能夠有所作為。但是，東晉的朝政已經被桓溫所控制，桓溫已心生自立之意，對司馬奕很是不滿。

桓溫借口司馬奕喜好男寵，所生三子可能都不是親生，逼褚太后下詔廢了司馬奕的帝位，重新做回東海王。沒想到的是，這個司馬奕皇帝沒做成，壽命反而長了許多，一直活到四十五歲，比其他東晉皇帝都好得多。

由此可見，褚太后雖然臨朝稱制，但實權並不完全在她手上，桓溫憑藉北伐之功，掌握了朝政大權，距離謀朝篡位只剩一步之遙。褚太后是個平和的人，威望極高，卻沒有自己的私人勢力，很難與權臣對抗。

在桓溫的堅持下，原先的丞相司馬昱被立為皇帝。司馬昱當時已有52歲，是司馬睿的兒子，輩分極高，並且與桓溫是有很深的矛盾，桓溫自然不會把大權交還給他。司馬昱多次與其抗爭，都沒有結果，登基僅八個月，便憂憤而死。

桓溫也未能圓了自己的皇帝夢，他膽子不夠大，在謝安等士家的抵制和拖延下，還沒堅持到篡位程序的走完，便一命呼嗚了。司馬昱的兒子司馬曜稱帝，年僅十歲。大臣們又把褚太后請了出來，第三次臨朝稱制。

褚太后早就無欲無求，只願朝政平穩，便重新走前台，主持大局。其實司馬曜和褚太后是平輩的叔嫂關係，嚴格地說，不能稱太后。不過，東晉的皇帝位子混亂慣了，也就沒人在乎這一點。

四​年後，司馬曜成年，褚太后馬上還政給他，這次她終於可以安享晚年了。公元384年，褚太后去世，享年六十一歲。司馬曜以太后禮儀將她安葬於崇平陵。

褚太后一生輔佐五位皇帝，臨朝稱制約四十年。她沒有政治野心，每次臨朝都是出於無奈，卻保持了東晉王朝的穩定，使江南地區得到休養生息和開發。這與東晉士族勢力的強盛，皇權受到相當程度的限制，也有很大的關係。
