曹操要禁酒，孔融立馬就開噴：“你咋個不把女人也禁了！”

2019-12-14 1,911條評論 4,624次閱讀 673人點贊

讀一曲三國，可以說就是上了一次酒席，七個碟子八個碗，再加上四五個酒盞，「古今多少事，都付笑談中」。說到三國，就不得不說到曹操，既是梟雄，也是文豪，更是權臣。在他的短短的一生中，無論是哪種成就，都與酒有著不解之緣。無論是「何以解憂，唯有杜康」膾炙人口的名句，還是禁酒令下對百姓勞苦的憐憫，都與酒文化息息相關。

在三國中，就像是和東北人相處一樣，沒有什麼事是一碗酒解決不了的。如果說是有，那麼就得兩碗酒。

這麼說，是因為三國中有著太多和酒相關的故事了，比如說煮酒弄青梅、溫酒斬華雄、周瑜醉酒騙蔣干、張飛醉酒打督郵、張飛酒後丟姓名，等等太多了，哪怕是一開始的桃園三結義，兄弟三人也是一碗酒，直到最後的三家歸晉，慶祝也離不開殺青酒吧，更別提裡面的宴會了，哪一次能少的了酒。

曹操禁酒

就著這麼多文臣武將、士子百姓都偏愛酒的年代，卻有一個人提出來要禁酒。

你敢信？這個人就是那個一直用杜康愛解憂的曹操。

記住了，幹了這杯，大家就不許再喝了！

話說東漢末年，三國之間戰爭不斷，百姓流離失所，各村各家，青壯十不存一。曹操在看到這種情況後，覺得百姓生活不容易，糧食都不夠吃的，一通憂國憂民的思想浮上心頭。

正悲戚間，低頭看了一眼酒杯，想到釀酒要用到很多很多的糧食，頓時便覺得酒的罪孽太多太大了，於是就頒布了一個禁酒令，理由是：飲酒喪德，為正世風。

孔融一次怒噴

本來一心做好事的曹操，本想著大權在握，又是利國利民的好事，肯定能夠得到支持，但沒想到的是，這個命令一經頒布，就迎來了眾多臣民的反對，其中孔融蹦躂的最歡。

你聽的沒錯，就是孔融，那個孔融讓梨的孔融。

他也是非常的愛酒，就開始寫了一篇《難曹公表制酒禁書》反駁曹操了，反駁的是有理有據：

「公當初來，邦人咸扦舞踴躍，以望我後。亦既至止，酒禁施行。酒之為德久矣！古先哲王，類帝桎宗，和神定人，以濟萬國，非酒莫以也。故天垂酒星之曜，地列酒泉之郡，人著旨酒之德。堯非千鍾，無以建太平；孔非百觚，無以堪上聖。樊噲解厄鴻門，非彘肩卮酒，無以奮其怒；趙之廝養，東迎其王，非引卮酒，無以激其氣；高祖非醉斬白蛇，無以暢其靈：景帝非醉幸唐姬，無以開中興；袁盎非醇醪之力，無以脫其命；定國非酣飲一斛，無以決其法。故酈生以高陽酒徒，著功於漢；屈原不鋪糟歃酶，取困於楚。由是觀之，酒何負於治者哉？」

大概就是說：「你剛開始起來的時候，我們都非常的支持你，後來你開始禁酒了，斷了我們的希望。酒的功勞可大了，歷史可長了，上古先哲建立宗室、祭祀上天、安撫國民，用的都是酒。所以天上才有了一個酒旗星，人間也有了一個酒泉郡，人們都在說酒的好處。堯如果不是千鐘不醉，哪能建太平天下；孔子要不是百杯不倒，怎麼能成聖人。樊噲解了鴻門之難，還不是喝了很多酒，要不然怎麼會奮發怒氣；趙國的小廝，東去迎接大王，如果不是喝了杯酒，不然怎麼會有這麼大的志氣；高祖劉邦要不是喝醉了斬了白蛇，沒有什麼能通了他的靈性；景帝要不是喝多寵幸了唐姬，怎麼會有漢代的中興；袁盎入股不是酒的力量，怎麼會逃脫性命；定國要不是美美的喝了一壺酒，就沒有解決他的辦法。所以酈生靠著高陽酒徒的力量，給漢代建立了功勛；屈原沒有鋪酒糟釀酒的話，就會一直被困在楚國。從這些歷史典故來看，酒哪點虧欠政治了？」

曹操做回復

曹操聽了後，回復了孔融，原文現在找不到了，但根據孔融的回答我們可以推測出來一部分。

大概就是：「孔融啊，你的意思我明白了。歷史上是有很多人的成功喝酒脫不了干係，但是呢，還是有很多人，比如紂王啊，就因為建了個酒池就亡國了。酒這東西，喝多了就會誤事，如果是耽誤了重要的事，對社稷的治理壞處太大了。你理解一下，這個酒我是一定要禁的。」

孔融二次回噴

針對曹操的回復，孔融整理了一下文筆。回了篇《又書》：

「昨承訓答，陳二代之禍，及眾人之敗，以酒亡者，實如來誨。雖然，徐偃王行仁義而亡，今令不絕仁義；燕噲以讓失社稷，今令不禁謙退；魯因儒而損，今令不棄文學；夏、商亦以婦人失天下，今令不斷婚姻。而將酒獨急者，疑但惜谷耳，非以亡王為戒也！」

你曹操咋個不把女人也禁了！

翻譯過來就是：昨天承蒙您的訓斥，陳述了夏商兩代的禍亂，以及很多人失敗的案例，都是因為酒的原因，實際就像是來教育我一樣的。即使是這樣，那麼徐偃王推行仁義而導致國家滅亡，您應該下令不準人們仁義！燕噲因為禮讓丟失了社稷，您應該下令不準人們謙卑禮讓！魯國因為一些儒生而滅國了，您應該下令不準人們搞文學！夏代商代也因為女人的緣故滅國了，您應該下令不準人們結婚！現在您只把酒禁了，我懷疑你是吝惜你那幾顆糧食罷了，還非要用亡國的皇帝來告誡我！」

小結

還是孔融看清了曹操的真實想法。前端時間，看到了網上的一些噴子，可如今和孔融一相比，頓時就覺得「噴子不可怕，就怕噴子有文化」。

親愛的讀者們，你們怎麼看？歷史上或者身邊上還有哪些和孔融一樣的噴子呢？毒舌有沒有孔融厲害？歡迎在評論區留言分享！
