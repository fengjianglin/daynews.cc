宋代是歷代王朝的經濟鼎盛時期，為何還有“租房族”的社會現象？

2019-12-24 966條評論 2,464次閱讀 1019人點贊

近幾十年來，中國城市的房價瘋漲，買不起房的只好租房，於是衍生出了「租房族」，大規模的租房現象不止出現於現代社會，早在宋代就已經出現了，「房屋租賃業成為東京最賺錢的一種行業」。當時社會的上層階級中，官員、外戚、舉人、軍人等住宿都以租房為主，宋代房屋租賃業呈現出一片興盛的趨勢，而這種租房現象是如何出現？又對宋代社會產生了什麼影響？需要我們進一步研究。

一、宋代租房現象的成因

雖然宋代實施的重文抑武的基本國策使得宋代「積弱積貧」，但是在中國歷代王朝，其經濟、文化、政治確實屬於鼎盛時期，英國著名經濟史學家麥迪森曾說過:「早在公元10世紀時，中國人均收入上就已經是世界經濟中的領先國家，而且這個地位一直持續到15世紀。」然而這個人均收入處於世界領先地位的國家卻出現了大規模的租房現象。

探究其深層次的原因，我認為有以下幾點：

1、宋代流動人口的增多

據史書記載，雖然宋朝軍隊兵員眾多，但是由於鑒於唐朝末年的藩鎮之亂以及五代的權勢過大危及皇權的教訓，宋代的軍隊制度實行將兵分離的制度，造成了宋朝軍隊實力衰弱，靖康之難後，皇室南遷，定都臨安即如今的杭州，大量宋朝百姓也隨皇室往南方遷移；同時由於種種原因，官員去往別地就職；以及商業貿易的逐漸增多，以上這些原因，都造成了流動人口的增多，也造就了宋代租賃業的興盛。

首先是百姓的南遷，以靖康之亂為例。靖康二年四月，金軍攻破東京(今開封)，俘虜徽、欽二帝及宗室子弟、妃嬪以及朝臣等三千餘人，押解北上。城破之後，金人燒殺搶掠，搶奪財物，連婦人頭上的金銀首飾也不放過，除有金人搶掠之外，而開封府為贖回欽帝，曾派官吏直接闖入百姓家中搜括金銀，橫行無忌，如捕叛逆，當時百姓五家為保，互相監督，如有隱匿，即可告發。

就連當時的賤籍之人如工伎、倡優等，也在搜刮之列。百姓苦不堪言，紛紛隨著皇室南遷，南方城市如臨安等湧入了大量的流動人口，隨皇室在南方定居，但是由於戰亂之時，財物被搶奪，無力購房，只能租房而住。

其次是官員的流動。宋代科舉制度的完善，世家實力衰微，打破了門閥士族壟斷仕途的局面，宋朝有相當一部分官員是寒門出身，他們家底微薄，根本無力支付購房費用，只好租房。

而宋代的官員選任制度沿襲前朝，是為三年任期制，官員不能回故地任職，亦不能長時間的在一個地方任職，所以宋朝官員的流動性大為增加，同時由於宋朝朝廷的財政緊張，絕大部分官員尤其是低品級官員的俸祿並不高，對於他們而言，租房是無奈之舉也是必要之舉，無怪乎曾有宋朝官員感慨:「雖室有十錢之物，亦盡為薪米，屋直之費。」而靖康之難後，舉國南遷，北方士子在南方為官，使得租房需求大量增加。官員的流動性是他們租房的最主要原因。

北宋著名政治家韓琦就曾說過:「自來政府臣僚，在京徽官私舍宇居止，比比皆是，茲乃常事。」而理學家朱熹也言道:「祖宗朝，百官都無屋住，雖宰執亦是賃屋。」可見宋朝官員賃屋之普遍。

2、外交政策上的支出大

宋代經濟發達，但是軍事實力弱於當時的金、遼等少數民族政權，就如同小兒抱金於市，引人覬覦。為保邊疆安穩，也為了安撫遼金等國，簽訂了不少和約，如《檀淵之盟》、《海上之盟》等，和約中約定遼金等國向宋朝稱臣，結為兄弟之國，雙方互不侵犯，宋朝就每年給予遼金等國大量的歲幣、絹帛等物。宋朝的財政收入多是稅收為主，外交上的巨額支出，使得朝廷財政緊張，在官署、軍隊營房以及薪俸上就有些捉襟見肘了。

如宋真宗景德二年，宋朝與遼國簽訂《檀淵之盟》合約:遼宋兩國約為兄弟之國，宋每年送給遼歲幣銀10萬兩、絹20萬匹，宋遼邊界以白溝河為界。雖然據史料記載，真宗景德年間，商稅的收入就已達450萬貫，對朝廷來說負擔不算太重，但是此後，遼金等國越來越囂張，索取之金額也越來越大。

北宋末年，宋、金兩國結成海上之盟，合力滅遼，然而宋軍攻遼失敗，金軍卻勢如破竹，一舉攻下遼中京等，遼國滅亡，原本的燕雲十六州也盡入金國手中，為了贖回燕雲十六州，宋朝承諾以二十萬兩銀、三十萬匹絹給金，並納燕京代租錢一百萬貫，金才歸還燕雲六州及燕京，撤軍之前，還將城中財物搶劫一空，僅留空城。

檀淵之盟

從上可知，宋朝因為勢弱，每年在外交上的支出越來越大，而這些支出，最終還是要平攤至百姓身上，苛捐雜稅過多，加重了百姓的負擔，使得百姓無力建房、購房，只能以租房為主。同時外交上越來越大的支出也使得國家財政緊張，官署破敗、營房破舊無力維修，低廉的薪俸也迫使兵吏外出租房以安身，促進了宋朝租賃業的繁榮。

3、宋代的城市化水平高

宋代的城市化水平之高，在我我國封建歷史中還是首屈一指的。宋朝疆域較前朝小，但是人口卻比前朝多，農村容納不下那麼多人口，只能往城市發展。這也促進了宋代城市的發展，並打破了前朝的坊市制度。

據梁庚堯先生推測，南宋「大部分的城市戶口比率可能在百分之三至百分之十四之間，但也有少數城市戶口比率高達百分之三十幾至四十幾」。這樣高的城市化水平。使得城市人口開始向城外溢出，打破了傳統的城牆限制，如宋真宗年間，由於「都門之外居民頗多」，所以「置京新城外八廂」。城牆的打破，使得城市逐漸向外擴張。

除此之外，宋朝的城市都得到了極大的開發，如錢塘縣:「昔歲風物，與今不同，四隅皆空迥，人跡不到。寶蓮山、吳山、萬松嶺，林木茂密，何嘗有人居!」大量人口的湧入，城市空間變小，道路也擁擠不堪，據《夢梁錄》記載來看:「臨安城郭廣闊，戶口繁黔，民居屋宇高森，接棟連檐，寸尺無空，巷陌奎塞，街道狹小，不堪其行。」這些都是城市化水平高的具體表現。

宋朝疆域小，人口卻在逐年增多，人地矛盾不僅體現在農村，城市也是如此。在這樣的背景下，如同今天的北京城，原有居民尚且擁有安身之地，但是新增的城市人口卻無力購房，租房才是最佳選擇。

二、宋代租房現象的影響

租房現象對宋朝產生了深遠的影響。租賃業作為社會經濟發展到一定水平的產物，它的出現繁榮，是宋朝商業興盛的代表之一，管中窺豹，從租賃業的繁榮可以窺見宋朝商業經濟的發達；與此同時，大量流動人口湧入城市，也帶來了社會治安隱患。

1、宋代商業的興盛發達

租房現象作為社會經濟水平發展較好的產物，它的出現，代表著宋朝商業貿易的繁榮興盛，以商稅為例，太祖年間，商稅收入為400萬貫，之宋真宗年間，商稅已增加至450萬貫，到了天禧末年，商稅已激增之1204萬貫，可見商業貿易之興盛。

而宋朝商業之興盛，離不開統治者的倡導。宋太祖趙匡胤就曾鼓勵百姓經商：「多積金,市田宅以遺子孫,歌兒舞女以享天年」，而他的繼任者也積極繼承了這一思想，神宗甚至提出了「政事之先,理財為急」的口號，可見宋代經商之寬容。宋朝商業發達的證據之一，就是出現了中國乃至世界上最早的紙幣—-交子，還有了經濟法，如鹽法、茶法等。

種種跡象可以說明，宋朝的商貿水平已經達到了相當高的地步，王曾瑜先生甚至對此發出感慨：「商業、租屋業和質庫業，似可作為宋代城市經濟的三大支柱。」可見宋朝商業之繁榮，在以小農經濟為主的古代封建社會中，不依靠農業稅收，改為依靠商稅。

2、社會治安的隱患不斷

租房現象雖然代表著宋朝商業的繁榮，但是也帶來了相當多的社會隱患。租房的存在，降低了城市的進入門檻，大量流動人口的湧入，城市就業崗位的缺失，使得城市出現了相當多的無業游民。而宋朝坊市制度的打破，原本「閭巷皆中繩墨，坊有墉，墉有門，逋亡奸偽無所容足。」

然而坊市制度被打破之後，一些廉價的出租房甚至成為了藏污納垢之所，罪犯、三教九流之人藏身其中，給城市治安帶來了相當大的麻煩。據記載，北宋時「京師承平之久，無知之民，游手浮浪最多，平居除旅店外皆在火房、浴堂、櫃房，雜居里巷。」這些無業游民聚居在此，終日遊手好閒，吃喝嫖賭、坑蒙拐騙無所不作，南宋臨安城中「人物盛黔，游手奸黯，實繁有徒。有所謂『美人局』，『櫃坊賭局』，『水功德局』，不一而足。」

從上可知，租房現象的出現促進了宋代租賃業的繁榮，但是社會治安的隱患卻也變多了。

三、小結

綜上所述，從宋朝的租房現象中，我們可以窺見當時社會的經濟發展水平已經相當之高。從記載中可以知曉，宋代人口的流動、外交政策下的壓力以及當時較高的城市化水平都推動了宋朝租房現象的產生，而與此同時，租房的大量出現，推動了宋朝的經濟發展、商業繁榮，但是也帶來了一定的社會治安隱患。但是具體來看，租房現象的產生，對宋朝社會來說是利大於弊。
