慈禧生前最喜歡的翡翠西瓜，價值6億，如今在美國想要都要不回

2019-12-21 223條評論 1,549次閱讀 219人點贊

中國古代誕生過許多價值連城的珍寶，但被皇族權貴選中下葬的一般都是非同小可的珍貴寶物。

據說晚清時期的獨裁者慈禧太后，就收藏了一批罕世珍寶，等她死後一同陪葬埋入了定東陵。

在這批昂貴天價寶物，當中有一對翡翠西瓜，其價值非常昂貴且擁有厚重的歷史意義。

故此常被世間人津津樂道，可惜當年定東陵被軍閥孫殿英盜墓之後，翡翠西瓜就從此流入了凡間，那麼如今這對翡翠西瓜到底在哪裡呢？

01

人稱老佛爺慈溪的陪葬品乃二十世紀中國地下最昂貴的寶藏，而墓中的那隻由高品質翡翠雕琢而成的翡翠西瓜，更是令無數古董愛好者想親眼目睹一番。

不過「翡翠西瓜」在行內叫法卻不是西瓜，叫碧璽，是由一塊復三方柱子狀的結晶電氣石組成，因為歷史原因跟蘊含了文化底蘊，所以異常的珍貴。

當年慈禧太后在下令重修東陵的時候，其規模奢華程度不僅超過了東太后慈安，而且還把自己的陵墓從原來的西陵搬到了東陵，而當時由於慈禧太后獨攬大權，所以滿清無人敢吭半聲。

對於陪葬品，慈禧更是窮極奢華，欲把全天下的珍寶都拖進去陪葬，可惜陵墓跟棺材空間有限，所以她才挑選最珍貴最稀少的珍寶陪葬。

根據古籍《愛月軒筆記》記載，慈禧老太后的棺材裡擺置有10000件珍寶，真乃世俗罕見。

而當年，慈禧在生時，最喜愛的就是這一隻「翡翠西瓜」，平常無事的時候，總叫李蓮英搬出來仔細欣賞摸弄一番。

為了保養好這枚玉石西瓜，慈禧還專門叫工匠鍛造一隻精美華麗的木盒用來承裝「翡翠西瓜」，因為慈禧認為翡翠具有美容養顏的效果，為此特別喜歡翡翠相關的東西。

關於翡翠西瓜的來源，慈禧身邊的大紅人李蓮英公公曾表示，這一對翡翠西瓜在之前購買的時候，總共耗費了500萬兩白銀，為了博得老佛爺歡心，再貴也是值得的。

注意李公公說的是一對，證明當初的翡翠西瓜是兩隻，而不是一隻。

當年清朝的500萬兩白銀，換算到如今的白銀市場價的話，約為6億RMB，因此老佛爺當年可是財大氣粗，隨便一出手就花了6億買下兩大塊「綠石頭」。

自從慈禧駕崩後，這兩隻翡翠西瓜就跟著她被長期埋入了地下。

02

轉眼來到民國初期，清廷在溥儀宣讀退位詔書之後正式滅亡，一向受到清朝皇家庇護的定東陵此時就成為了貪財之人的「獵物」。

因為當年慈禧太后的葬禮太過高調，很多百姓都知道慈禧的定東陵埋葬著不少稀世珍寶，而那對聞名的翡翠西瓜就在慈禧棺材裡面。

不過定東陵由於特殊原因，一般人依舊無法靠近。

直到1928年，各地的軍閥林立，各方勢力為了謀發展都不顧一切想方設法的謀取巨資用來發展自己的軍隊。

此時，身為國民黨革命家第12軍軍長的孫殿英就打起了老佛爺陵墓的主意。

1986年拍攝的電影《東陵大盜》取景用的就是慈禧太后的真正陵墓，可謂是一部難得的歷史價值電影。

片中講述孫殿英起初先把盜墓的想法彙報給自己的上峰領導，而此領導擔憂影響太大，輿論會壓垮自己，於是不贊同孫殿英去盜慈禧墓。

然而孫殿英狼子野心，不甘受人牽制，為了自己儘快發展軍隊，就趁著自己的上峰出外地辦公的時候，假借軍事演習為名，開著自己的軍隊把定東陵被盜了。

接著把裡面的珍寶搶掠一空，當中的翡翠西瓜也一同淪入了孫殿英之手。

據說孫殿英看到翡翠西瓜的時候，觀其外形擁有青綠色的表皮，瓜肉有紅瓤，裡面還摻雜著白色的籽核，雖然全部是翡翠雕造，但看來十分的靈活靈現使人心愛不已。

孫殿英本想把此等人間寶物佔為己有，無奈他盜東陵的消息很快就被傳得天下皆知，不久後各路軍閥就向孫殿英興師問罪，就連國民總統老蔣也親自下通緝令要抓拿孫殿英。

這時候孫殿英就慌了，他雖盜得慈禧的大批陪葬珍寶，但始終不是天下群雄的對手，為了活命孫殿英就拿著陪葬珍寶到處去賄賂各路的軍閥將領。

過程中孫殿英把慈禧的夜明珠送給了老蔣的夫人宋美齡，把乾隆墓里的九龍寶劍托關係送給了老蔣，還把其餘的東陵珍寶悉知贈給其他軍閥人物，而其中的一隻翡翠西瓜就被孫殿英送給了老蔣小舅子宋子文。

03

至此之後，翡翠西瓜經由宋子文之手流連人間，四處易主，由於當時國家局勢正處風雨飄搖異常動蕩的時期，大量國寶因各種因素被流失海外，不久後翡翠西瓜就被人偷運出了中國。

經歷多年後，國人才在美國的一所博物館裡，見到了久違的翡翠西瓜。

所以當年陪著慈禧太后下葬的那一對翡翠西瓜，有一隻被扣留在美國的博物館裡，想要都要不回！另一隻卻一直沒有音訊。

多年以後，北京有一位居住熙和園周邊的張老伯回憶稱，他在很多年以前，到頤和園附近耕地，意外的挖出一堆異常殘物。

此物體因在糞坑旁邊，散發了濃郁的臭味，伴隨著好奇心張老伯還是用工具揭開了這對殘物，結果在裡面發現了一顆類似西瓜的綠石頭。

接著張老伯就把這塊綠石頭洗乾淨，發現原來正是翡翠西瓜，最終張老伯秉持著愛國無私的精神，逐把翡翠西瓜上交了國家。

其實國家現如今一直都非常注重文物的保護，這主要是文物自身所擁有的含義，文物代表了一個國家的文化底蘊。

如今許多人都喜歡參觀文物，有家長帶孩子去博物館的，也有學校組織的，這樣不僅可以增強人們對文物的認知，還可以提升民族自豪感。

在觀看文物的時候，總會去想像古人的智慧以及精湛的技藝。

只是因為觀賞文物的局限性，許多人並沒有時間去了解。

正因如此，四川人民出版社在今年出版了一本書籍，一本國家寶藏日曆，別出心裁。

以傳統日曆的方式，讓人們了解文物，每一件文物都和古人的衣食住行息息相關，用最通俗的文字描述，每天都能了解新的歷史文物。

2020年共366天，這本日曆包含了366件國寶，完美展現了中華文明的傳承。

《國家寶藏日曆》出版後，好評如潮，更有學者稱「把國寶擺在自己的桌面一整年！」
