中國歷史上武功最高的刺客，不是最有名的荊軻，而是另有其人

天天要聞
3 天 ago

荊軻刺秦的孤勇，千年里一直被人傳頌著，刺客作為古代的一個特殊群體，因為司馬遷的列傳，而流傳於世。但是歷史上武功最高的刺客，卻不是最有名的荊軻，而是另有其人，此人還是蓋聶的偶像，他是誰呢？其實他最後取得了刺殺的成功，可他為何會死呢？

聶政拒絕刺殺

他就是用自己的生命詮釋了”士為知己者死”的第一人——聶政。他因為殺人避禍，帶著自己的媽媽和姐姐到了齊國，在齊國的一個市井做起了以宰殺為生的屠夫。後來，濮陽的依噶叫做嚴仲子在人在給韓哀侯做事的時候，因為得罪了韓國的丞相俠，兩人產生了很大的矛盾。戰國時期的人，單純也粗暴，一個看不爽就要拔刀，嚴仲子就害怕韓相報復自己，逃了。

可是天下就那麼大，當時又是一個可以買兇殺人的時代，嚴仲子就是跑他也覺得不安心，覺得不行，自己得先下手為強，先找人把韓相干掉，就開始到處便走邊打聽哪兒有武功高強的人，可以幫自己一把。有一天當他走到齊國的時候，齊國很多人都在說聶政真實個厲害的人啊，他的武功是多麼的高強，他心中一動，覺得聶政就是他要找的人。

於是就上門拜訪聶政了，還帶著黃金百溢，他到了之後先是和人家聶政的母親喝上了酒，酒過三巡，突然拿出黃金百溢獻給了聶政的母親，說要給她拜壽，母子倆都嚇了一跳，這是怎麼回事？於是嚴仲子撇開眾人，說出了自己登門拜訪的原因。

可是聶政拒絕了，說自己不敢接受這麼多的錢，他說自己高堂尚在，家裡也窮，如果自己答應了去替嚴仲子殺人，就是置自己的母親和姐姐於不顧，在此亂世，兩個女子怕是不好過活。嚴仲子一聽就說是我唐突了，那這錢就留給閣下，你用此奉養老母，在下不敢再提出過分的要求了。此話一出，可是把聶政給感動了，但是他沒接受嚴仲子的錢，盡完賓主之誼之後，就把嚴仲子給送走了。

聶政行刺的理由

後來，過了很久，聶政的母親去世了，在把自己的母親下葬以後，服完孝之後，除了孝服，就想起了當初嚴仲子的請求。感嘆著：想我聶政一介市井小民，靠屠宰謀生，何等微賤？但是人家嚴仲子卻是諸侯之卿相，卻不遠萬里來拜訪我，而且還要和我相交。我之所以接待他，不過是感到奇怪，但是我什麼都沒做呢，人家就願意拿出百金來給我母親過壽，哪怕我沒有接受，也該知道人家是看得起我，了解我！

人家對我這個窮鄉僻壤之人這麼好，我怎麼能自己獨自快活呢？之前我不能幫他解決麻煩是因為要養老母，如今母親已經去世，我應該為了自己的知己而出力。於是他就去找嚴仲子，說：之前不答應你，是因為要養母親，現在母親去世了，可以替你報仇了，你的仇人是誰？

嚴仲子就說自己的仇人是韓國的丞相，但是他常年做韓國國君的季父，家族勢力很大，住的地方還很多衛兵，我一直想阿皮仁去刺殺他，但是一直沒能成功！今天蒙您不棄，我願意給您備車，準備人馬，送您前去。但是聶政拒絕了，他說人多不好辦事，也容易泄露消息，就自己一個人去了。

可見他完全是因為嚴仲子在他微末之時，不嫌棄他屠夫之身，願意來拜訪他，還給自己的母親準備千金拜壽，視他為知己，這讓聶政非常的感動。也因此而引嚴仲子為知己，願意為他去刺殺仇人。

聶政行刺

聶政到了韓國以後，打聽到韓國丞相的住處之後，直接提著劍上門了，直接單挑拿下守門的侍衛，直接登堂入室，走上台階就把韓國的丞相給殺了，丞相的其他手下發現以後，大亂也大叫，於是聶政沒辦法，只好把這些人也給殺了。但是殺完這些人之後，他用刀劃破了自己的臉，自己把襲擊的腸子都給挖了出來，直接自盡了。韓國人就把他暴屍於市，詢問他的身份，說指認的人給千金的賞賜，但是一直沒人認領。

從聶政的刺殺成功可以看出他的武力值，注意他是直接拿著劍從大門堂而皇之的殺了守衛進去的，韓國的丞相很順利地被他殺了，後來的侍衛也被他殺了，人家刺殺都是暗殺，這大哥是從大門硬闖，還是一個人單挑，最後還成功了，如果不是為了不暴露嚴仲子的身份，他才毀容自盡，成功跑到安完全是沒有問題的，歷史上有哪個刺客可以像他一樣武力值如此爆棚？

聶政屍體被認領

而聶勝之所以得以流傳千古 ，還是因為他的姐姐。他的姐姐在知道韓國的丞相被殺，但是賊人的名字卻沒人知道，還暴屍來找人，想到自己很久沒見到自己的弟弟了，以及她弟弟的絕世武功，就覺得是她弟弟。於是她不遠萬里趕到韓國，到了一看發現果然是自己的弟弟，她竟然認領了。周圍的人都覺得她奇怪，因為聶政殺得是人家韓國的丞相，普通家屬恐怕牽連，避不承認都來不及，她怎麼來認屍？

聶勝的姐姐說：我知道我弟弟肯定是感念嚴仲子當年在他危難之際不嫌棄他，還用千金給自己的母親祝壽，視其為知己，過於感動，才毀容，就是為了不牽連嚴仲子以及自己。然後大聲喊了聶政的名字三聲後就自殺在了自己弟弟的身邊。

晉、楚、齊、衛的人知道了，都說：並非聶政是個英雄，他的姐姐也是個烈女，跑了千里來給自己的弟弟留名，嚴仲子實在是得到了一個真正的知己啊!如此姐弟，嚴仲子算是佔了個大便宜？
