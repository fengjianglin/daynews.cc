世界三大軍事天才，拿破崙勉強上榜，而中國有兩位

2019-12-16 2,365條評論 3,949次閱讀 1247人點贊

人類生存至今歷經無數戰爭，更有許多軍事天才湧現帶領我們走出戰爭、走向新的生活，現在我們就來盤點一下世界幾位著名的軍事天才。

第三名拿破崙，拿破崙是一位頻繁出現在我國歷史教材中的人物，大家對其一定有不少了解。他是法蘭西帝國的創造者，也是世界上絕無僅有的軍事家。雖然他身材矮小、其貌不揚，但拿破崙用卓越的軍事才能證明了自己的能力。他是炮兵戰術的創造者，密集的火炮戰術使他滅國無數，他領導軍隊多次對外擴張，將荷蘭、西班牙變成他的殖民地，從而締造了法蘭西第一帝國。他的一生指揮六十多場戰役，只有為數幾場是失敗的，並且執政期間幾乎一統整個歐洲。即使滑鐵盧戰役成為最後的敗筆，但他依舊是一位不可多得的軍事天才。

第二名成吉思汗，他是世界史上最傑出的軍事家和政治家。十二世紀成吉思汗一統漠北，大蒙古國疆域遼闊直達東歐黑海區域。雖然在歐洲人看來成吉思汗是位十分殘暴的君主，他的鐵騎踏遍歐洲，造成東歐人們生活困頓，但成吉思汗的功績是不可置否的。十三世紀初幾乎整個歐洲都被其所統治，但他把亞洲先進文化帶到了歐洲，給黑暗的中世紀歐洲帶來了光明。成吉思汗剛剛建國之時便著手創建文字、制定青冊，頒布了蒙古族第一部成文法，裡面記載國家律令和成吉思汗的言論訓誡。毋庸置疑的是，元朝的蒙古是世界上最強盛的國家。

第一名便是我國春秋時期著名軍事政治家孫武，他是兵法鼻祖，更是中國的「兵聖」。孫武所著的《孫子兵法》曾多次被譯為英語、日語、德語、法語等版本被世界所熟知，也成為了中國甚至世界軍事領域內政治家們必學的兵學典範。他率領的軍隊在公元前506年進駐楚國，僅以三萬兵力幾乎把整個楚國滅亡。他的思想不僅在軍事方面，在哲學、政治、文化領域都發揮了重要作用。他被東西方世界共同視為軍界天才，即使後世傳奇之人接連湧出，孫武在軍事領域的地位都絲毫不會動搖，他是當之無愧的世界第一軍事奇才。
