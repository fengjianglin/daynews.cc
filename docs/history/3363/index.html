這兩個人為劉備三兄弟提供了第一桶金，怎料卻自此消失不見

《三國演義》第一回里，劉關張桃園結義後，迅速拉起了隊伍，但是馬上面臨一個問題：沒有馬。

秦漢時期馬匹是重要戰略物資，尋常百姓是養不起馬的，就算張飛家境殷實，一時半會兒也無從獲得這麼多馬匹。如何快速籌措馬匹器械，是關係到新成立的討賊武裝能否形成戰鬥力的重要問題。雖說廣東話講「馬死落地行」，可帶兵打仗的事兒，溜地跑可不是個事兒。

配圖來自網路

就在這個時候，命運第一次垂青了劉關張三人，有人主動上門學雷鋒了。

配圖來自網路

來的人是誰呢？兩個中山的販馬商人，一個叫張世平，一個叫蘇雙。

這兩個人每年都要往北地販馬，但是今年因為黃巾戰亂，走半道回來了。劉備請二人到張飛的莊子喝酒吃飯，席間說了自己眼下的困難，二人很豪爽的給與了幫助：良馬五十匹，金銀五百兩，鑌鐵一千斤。劉備開心無比，利用這批鐵製作軍械。三人的兵刃也是這個時候打造的。有了馬匹器械錢糧，一切都好辦的多：武裝力量成型了，成功引起官方的認可，順利的加入了討伐黃巾軍的官方戰鬥序列。

配圖來自網路

故事可以這麼講，如仔細分析下，卻有一些不合理之處：

第一點，張世平與蘇雙究竟是誰的客人？如果是劉備的，這邊桃園結義才發生，他們如何知道劉備在張飛這裡？如果是張飛的客人，為何從頭到尾沒有提到張飛？簡單的說因為劉備是大哥所以全部由他來主持全局，似乎不符合常理，畢竟按照小說描述，張飛可是事業有成的成功人士，他的朋友如何一開始就認可劉備？

第二，這兩個中山商人是販馬的，隨行帶著金銀可以理解，帶著一千斤鑌鐵就有點非同尋常了。要知道鐵不是鐵礦石，它是需要專門冶煉的高級生產資料，在亂世它又是重要的戰略物資。這就像現代社會一個搞牲畜貿易的公司在自己的運輸車隊里拉著幾百噸特種鋼材一樣奇怪。

配圖來自網路

整部《三國演義》，這兩個人就出現了這麼一次，宛如驚鴻一瞥，隨即杳然無蹤。相比糜竺資助劉備、魯肅資助周瑜都有後續故事，這次資助顯得很另類。

當然，生活中存在這樣的貴人，人家幫你未必想求你回報什麼。我們不排除張世平與蘇雙家大業大，又憎恨黃巾作亂打斷了他們掙錢的節奏，索性把這些東西資助看起來還不錯的潛力股劉備兄弟，權且當做自己也參與討賊了。

可是，翻遍史書，絕大多數商人的行為都有背後的商業邏輯可尋，比如我們熟知的弦高犒師、白圭猗頓周旋列國、呂不韋奇貨可居、寡婦清資助秦國，包括後來的糜竺資助劉備等，要麼尋求晉身之路，要麼維護/打造商業通路。張蘇二人，所為何來？

配圖來自網路

《三國志》里找到一段文字，大略能給這件事一個較為合理的解釋。《三國志先主傳》中說：「中山大商張世平、蘇雙等貲累千金，販馬周旋於涿郡，見而異之，乃多與之金財。先主由是得用合徒重。靈帝末，黃巾起，州郡各舉義兵，先主率其從屬從校尉鄒靖討黃巾賊有功，除安喜尉。」

原來劉備根本不是二十八歲了還在編草鞋草席虛度年華的沒落王族，而是一個很早就在當地有一定排面、黑白兩道都吃得開的江湖人物了。

配圖來自網路

《三國志關羽傳》里這段記載更有意思： 「先主於鄉里合眾徒，而羽與張飛為之禦侮。」

什麼意思？劉備是涿郡的扛把子，關羽和張飛是他的小弟。

配圖來自網路

這樣一來，一些事情大致能做如此推斷：

張世平、蘇雙是販馬的販子，少不得要打點各地黑白兩道上的各路碼頭、山頭。劉備由於自身機緣，在當地頗有些名氣，二人與之熟絡後出錢請他拉起一支隊伍，一方面保障販馬隊伍過境安全，另一方面也算一種投資，劉備的隊伍保護費收的多了分些利潤給這二人。後來黃巾起事，天下大亂，劉備覺得機會來了，不甘心再做涿郡古惑仔，要拉起一支正規軍討逆謀個功名，可能請二人幫忙籌措馬匹、鐵器、錢糧。背後有什麼約定不得而知，生逢亂世，商人總是很精明的，多一重保護總是好的，於是慨然應允。

配圖來自網路

當然了，無論是僱傭還是風投，我們都得感謝這兩位有錢人，如果沒有他們，可能就不會有後來的蜀漢核心，漢末後的歷史也可能會是另外一種樣子。
