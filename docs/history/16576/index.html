盤點中國歷史上被豬隊友坑慘的名將：這三位最憋屈

2019-12-16 562條評論 1,588次閱讀 428人點贊

一將功成萬骨枯，中國歷史上名將如雲，但是戰爭殘酷，不僅要有良將，更重要的還需要一個精誠團結的團隊，以及精幹的隊友。

而被豬隊友坑死的名將很多，咱們今天講三位。

第一位，只差一步滅強齊的樂毅：作為戰國四大名將的樂毅中國戰爭史上的表現可謂是非常驚艷，率著在春秋戰國歷史舞台上一直默默無聞打醬油的燕國瘋狂了一把，率燕軍和一幫看熱鬧不嫌四大的聯軍，硬生生打下了齊國70餘城，上演一出螞蟻吞大象的故事。

離滅強齊，燕國只差最後兩座城。

但樂毅的理想卻被自己身後豬一樣的隊友，燕惠王活生生給毀了。老子燕昭王給燕惠王留下了這樣大好的一把牌，燕惠王這個年輕的地主只剩下兩張牌關門的情況，臨陣換帥，硬生生馬上完蛋的齊國給活過來。

樂毅被坑死，不過還算聰明，再不和燕惠王這位豬一樣的隊友合夥鬥地主，跑到了趙國，留下一句「善始者未必善終，善作者未必善成」，來自我安慰一下。

第二位：大非川之戰，被大唐將二代坑死的薛仁貴：大唐名將薛仁貴不僅是在說書演義里生猛，在歷史上的成績單也很耀眼，三箭定天山，神勇復遼東等等，一生戰功赫赫。

但薛仁貴也打過大敗仗，大非川被吐蕃名將論欽陵打得幾近全軍覆沒，讓大唐直接喪失在中亞的霸權。

此仗薛仁貴打得窩囊，但並非仁貴大哥無能，實在是李唐皇帝給仁貴兄配的隊友太豬頭。

副將郭侍封、阿史道那真都是大唐將二代出身，瞧不上草根泥腿子出身的薛仁貴，薛仁貴指揮不動二位爺。仗自然打得一塌糊塗。

前面這二位雖然被豬隊友坑了，但只是打了敗仗損了一世英名，但接下來的這位，卻活活被豬隊友活活坑死。那就是北宋名將，說書中的楊家將的楊老令公楊業。

要說這楊業，還真是一個倒霉孩子，他遇到的豬隊友不止一個，都說兩顆炮彈不可能掉在同一個坑裡，但這楊業，卻被兩個豬隊友不同的坑了兩次。

第一次的時候，楊業還在北宋的敵對陣營北漢里呆著，這北漢雖不大，可有太原堅城為據點，又有楊業等實力大將，老趙（趙匡胤）連打了三次都沒打下來，後來趙三當皇帝再伐北漢，北漢方面由丞相郭無為為主將率領著楊業出征，卻不曾想這郭無為卻是那北宋的地下潛伏者，早就想投奔組織，楊業那裡知曉，跟著郭無為被宋軍包了餃子當了俘虜。

雖然被豬隊長郭無為坑了，但楊業也算因禍得福，趙光義不僅沒殺他，還讓他再就業，崗位待遇讓原來跟著趙二趙三那幫老兄弟都有些羨慕妒忌恨，這不後來趙光義在雍熙三年三路大軍北伐契丹，楊業又撈上了西路軍副帥的「美差」，楊業所率部隊一開始打得非常驚艷，寰州、朔州、應州、雲州等地，但因為主力東路軍曹彬在岐溝關被耶律休哥打得屁滾尿流大敗，趙三看勢不對，鳴金搬師，原來西路軍沒什麼損失，但主帥潘美卻想完成趙二交付的一項政治任務：咱大宋立國不久，差啥，就差勞動力，寰州、朔州、應州、雲州這些打下來的地方，地盤是守不住了，可老百姓，能帶多少帶多少回來。大軍一帶著百姓撤退，自然就慢了，當然要有人墊後，誰來當這個冤大頭，楊業。

楊業是個老實人，組織上交給的任務也沒推，只是對主帥潘美說，讓我墊後可以，不過我頂多能撐到陳家谷，隊長你得在那裡留兵接應！

老楊你開口，沒問題？潘美答應得挺乾脆。

事關上萬士卒生死，可千萬不能忽悠？

我是誰，北宋開國四大名將，皇帝他老丈人，吐出的口水說出的話，還能不作數。

楊業率著「楊家軍」與那大遼追兵邊戰邊退，陷入苦戰有效的保障了整個西路軍的安全撤退，但到了與潘美約定的陳家谷，哪有援軍接應，陳家谷連只鳥都沒有。

在陳家谷楊業部被遼軍重重包圍，全軍覆滅。本人被俘。

楊業氣憤難當，不想再大遼三度就業，絕食三日而死。

而豬隊友潘美，事後僅僅被小降了一級，什麼包龍圖的龍頭鋤侍候，那都是傳說，死後潘豬一樣被列為北宋四大名將，配享太廟。（現實好殘酷）
