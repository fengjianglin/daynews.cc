吐蕃用數百人守一小城，卻殺死數萬唐軍，如今古城下仍然屍骨累累

2019-12-23 1,711條評論 3,653次閱讀 914人點贊

石堡城之戰，乃是唐玄宗時期一場相當重要、且傷亡十分慘烈的戰役。自吐蕃松贊干布統一青藏高原以來，吐蕃和唐朝兩大強國便在青海湖一代展開了激烈的爭奪。到了唐玄宗年間，唐朝在唐蕃戰爭中逐漸佔據了主動權。在大唐戰神王忠嗣的主持下，唐朝屢次進攻吐蕃，攻下了大片土地。

唐玄宗十分重視西北地區的形勢,把重兵銳卒集中在河隴,以抵禦、抗擊吐蕃的入侵。洮河流域尤其是今臨潭、卓尼一帶成為雙方反覆爭奪的目標。開元( 713— 742年)初年,吐蕃佔據了河湟門戶的戰略要地建置石堡城，作為侵唐據點。玄宗也看到了石堡城極高的軍事價值,幾次下令攻拔之。

從石堡城的地理狀況看，可以用一夫當關，萬夫莫敵來形容。此地「西控諸蕃,東屏兩郡(天水、隴西) ,南俯松疊,北蔽河湟,西南之要害也。… … 洮陽奠磐石之安,秦隴保金湯之固。是唐王朝西北、西南邊境的屏障,猶如河湟門戶,而石堡城又是其咽喉之所在。

為了攻取此城，唐玄宗曾於公元729年徵發朔方雄兵，一舉而克之。公元742年，吐蕃大舉反撲，重新攻佔了石堡城，並大量增兵防守。此後,皇甫惟明、董延光先後奉命攻奪,都未能成功。玄宗對此一直耿耿於懷,常引以為憾。

其後，在王忠嗣的策劃，唐朝大將哥舒翰屢次擊潰吐蕃，而攻取石堡城的機會便再次顯現。749 年(天寶八年)六月,玄宗命隴右節度使哥舒翰率隴右、河西及突厥阿布思部加上朔方、河東兵六萬三千餘人向石堡城發動進攻。

為了保衛石堡城，吐蕃可謂是「傾國來援」。在石堡城上，吐蕃聚集了數百守軍。在城外，還有大量吐蕃的援軍。從地勢來看，石堡城險要至極，它「半在山上,下臨洮水,三面險絕,惟西南一路可通。」吐蕃人只在西南方向不斷投石、射箭，便可給唐軍帶來難以想像的巨大損傷。

在數次進攻未能得逞，吐蕃援軍不斷逼近的情況下，召來部將高秀岩、張守瑜問罪。二將請求寬限三天。將士奮不顧身,英勇作戰,終於如期攻下了這一軍事要衝。

此戰之中，唐軍為攻陷石堡城付出了可怕的代價。根據史料記載，唐軍在此戰中犧牲了數萬人。由於代價過於昂貴，以致時人和後人多認為得不償失。正所謂「邊庭流血成海水，武皇開邊意未已」。

那麼石堡城的地形，是否正如史書中敘述的那麼可怕？為此，考古學家王昱等人對石堡城遺址進行了實地考察。2009年7月13日，考古學家們在在湟源縣日月鄉莫多吉村附近下車。下車後在青藏公路邊向東眺望, 高聳的石堡城便展現在眼前, 該山紅沙岩體, 東、南、西三面絕壁, 外形酷似一個堅固的城堡。

山體底部和溝壑處有植被覆蓋, 東、南、西三面陡峭處紅沙岩裸露, 山體頂部為近似九十度的紅沙岩懸崖。尕溝山體坡度較陡。考古學家曾試著攀爬岩壁，卻均在爬到一半時，從山上滑下。可以想像在1000多年前，全副武裝的唐軍士兵面對吐蕃人的箭雨和滾木礌石，將遭遇多麼大的困難。

廢了九牛二虎之力，學者們爬到了石堡城遺址之上。當年吐蕃人建設的防禦工事早已蕩然無存，但是在石堡城頂，卻有一個足以容納數百人的高台。在石堡城周邊的山崖上，到處都是可以作為滾木礌石的石塊以及樹木。對於防守一方來說，石堡城的地利優勢實在太過明顯。因此學者們感嘆，要攻陷這樣的城池，不犧牲數萬人根本不可能。

據當地人說，直到今天，石堡城附近仍能挖出人骨, 一般人骨之上都壓有石塊。腿骨比現代人要長一些。還有人在石堡城附近發現了開元通寶以及鐵質箭頭等。在石堡城西側的日月鄉莫多吉村, 村民中傳言山下有死人溝、萬人坑等地名。由此可見，當年的石堡城之戰有多麼的慘烈。

不過另外也有學者認為，當年犧牲的唐軍並非都死於石堡城，有不少其實是被吐蕃的援軍所殺。石堡城的戰役, 有大範圍的戰場和小範圍的戰場之別。大範圍的戰場是指日月山分界線以東藥水河流域的唐軍整個防禦陣地, 唐在這裡設了諸多軍、城、軍事據點。既然吐蕃傾國來援，此地必然經歷了慘烈的拉鋸戰。根據測算，這個大戰場的範圍縱深有80餘里, 容納幾萬軍隊作戰是沒有問題的。

為攻取石堡城，唐軍付出了極大的犧牲。但可惜的是，由於安史之亂的爆發，唐朝實力大減，吐蕃人捲土重來，不費吹灰之力便攻佔了石堡城。此後，吐蕃大舉東進，侵佔了唐朝的隴右河西以及西域。從此以後，石堡城之戰成為帝王盲目開邊的典型，為後人所詬病。
