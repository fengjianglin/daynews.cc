“富不過三代”，必有原因，看漢代列侯們是如何失去爵位的

天天要聞
2019-12-08

摘要

歷史總是在王朝興衰迭代中，不斷地螺旋式前進，有人憂愁，必然有人歡喜。每一個王朝的滅亡，代之而起的則是另一個王朝的勃興，而勃興的王朝背後，始終會活躍著功臣集團。他們懷揣著“建功立業，封妻蔭子”的美好夢想，尋找到可靠的僱主，開啟奮鬥拼搏的創業史。

歷史總是在王朝興衰迭代中，不斷地螺旋式前進，有人憂愁，必然有人歡喜。

每一個王朝的滅亡，代之而起的則是另一個王朝的勃興，而勃興的王朝背後，始終會活躍著功臣集團。

他們懷揣著「建功立業，封妻蔭子」的美好夢想，尋找到可靠的僱主，開啟奮鬥拼搏的創業史。出生入死、櫛風沐雨，這些追夢的人有的看到了王朝的建立，許多人則倒在了征途之中。

新王朝開基立廟，「大封功臣」是免不了的環節，一批新貴成功爬到社會金字塔的上層，封官拜相，賞爵封侯。

貴族們都希望自己的財富、地位，擁有的一切能夠傳至子孫萬代。

想法與現實總是存在著差距，大多數勛臣之家，「富不過三代」，祖父或父親榮光，自己卻落得布衣的下場，而有些列侯們，在自己這一代便被褫奪了爵位。

（功臣宴樂圖）

可憐之人皆有故事，可憐之人必有痛點。

我們以漢朝為例，具體分析一下，列侯們失去爵位的緣由。

《漢唐封爵制度》一書，認為漢共封列侯471人（兩漢），而另一本研究侯國制度的書籍《漢王國與侯國制度的演變》中，則認為共封列侯898人，包括功臣侯314人，外戚恩澤侯154人，王子侯430人。

兩者角度不同，所統計的列侯數字有所差異。無論是哪個數據，都只是封侯的數量，而我們在《史記》、《漢書》、《後漢書》三部史籍中，大量列侯爵位得而復失的記錄，則更為直觀：有的一世而亡爵，有的二世、三世而亡爵，原因可以說是五花八門，不一而足。

  • 列侯無後代襲爵，而國除

古人常說「不孝有三，無後為大」，在今人看來，古人太小題大做了，不就是沒有孩子嘛，有什麼了不得的？沒有孩子的羈拌，或許生活更加精彩。

但古人的社會意識與今天完全不同，他們敬畏祖先，傳承祖業，延續香火是頭等大事，即使是普通百姓也要重視後代的生養，貴族更是如此。如果沒有後代（指兒子），「父死子繼」就無法延續下去，列侯們的財富、爵位自然會被朝廷收回。

（首任西域都護鄭吉）

西漢第一代西域都護鄭吉，破車師，降匈奴日逐王，威震西域，為漢朝鞏固西北邊防，作出過重大貢獻，漢宣帝封其為安遠侯。其子鄭光承襲爵位，卻沒有兒子，他去世之後，「安遠侯」的爵位被朝廷取消。

鄭吉，會稽人也，上嘉其功，其吉為安遠侯，吉薨，子光嗣，薨，無子，國除。
 ——《漢書.鄭吉列傳》
  • 列侯違法犯罪，而被免

張良，是「漢初三傑」之一，劉邦創建西漢王朝，張良是其智囊，劉邦曾說「運籌帷幄之中，決勝千里之我，吾不如子房」，可見張良在劉邦心中的地位。天下安定後，張良識時務，害怕功高震主，只接受了「留侯」的爵位及封邑，並未在朝為高官。張良去世後，自然而然，兒子張不疑承襲了「留侯」爵位。但是，好景不長，孝文帝三年，張不疑因「不敬」罪，爵位被取消。

良多病，未嘗特將兵，常為畫策臣，時時從。漢六年，封功臣，封良為留侯。後六歲薨，子不疑嗣，孝文三年坐不敬，國除。
 ——《漢書.張良列傳》
  • 列侯捲入政治鬥爭，而被殺

漢武帝英明一世，晚年漸漸昏聵，特別崇尚神仙方士，「上有所好，下必進焉」，一批奸佞小人趁機進入朝堂，掀起一股「巫蠱之禍」，在這場持續數年的人禍中，大量的列侯被捲入其中而遭受殺戮，葛繹侯公孫賀、按道侯韓說、重合侯莽通、德侯景建、題侯張富昌等均被誅。

「巫蠱之禍」前期，漢武帝受到江充等人的蒙蔽，支持他們查辦巫蠱罪人，但這些人打著「忠君」之號，行害人之事，膽大包天，把手伸向了太子劉據。劉據想面見漢武帝，又被江充等人隔絕，欲見不得，只好起兵除奸。漢武帝再次聽信讒言，認為太子是在謀反，遂起兵抓捕太子。太子劉據只好逃出長安，在湖州地區被發現後，被迫自殺。

支持太子，或消極應對太子起兵的官員列侯，全部被漢武帝誅殺，如北軍中尉、司馬遷的好友任安。而積极參与對抗、抓捕太子的人，一律受到提拔，封侯的也不在少數。數月之後，風向突然改變，漢武帝反思「太子之死」，知道自己上當受騙，便將案件翻轉過來，受封賞的官員列侯又全被治罪，反覆兩次人禍，受刑失爵者眾多。

（巫蠱之亂）

東漢明帝時，楚王劉英謀反，明帝以此為借口，大張羅網，給列侯治罪。

「楚獄遂至累年，其辭語相連，自京師親戚諸侯，阿附相陷，坐徙者以千數」。
 ——《後漢書.顯宗孝明帝紀》

在「楚王之禍」中，開國功臣之後劉光侯耿阜、楊虛侯馬檀、參鄉侯杜元、山桑侯王廣、白牛侯劉嵩、汝陰侯劉信等均獲罪，全部被褫奪爵位。

東漢的「外戚專權」、「宦官專權」兩重特點，導致朝政異常複雜，明爭暗鬥，「按下葫蘆瓢又起」，不管是外戚當政，還是宦官得勢，一波爭鬥之後，總有一批列侯獲罪失爵。

縱觀兩漢四百年歷史，人數眾多的列侯，不乏那種驕奢淫逸之輩，他們生於富貴窩，忘乎所以，多行不法，完全不知祖先當年創業之艱難，為非作歹，「子孫驕逸，忘其先祖之艱難，多陷法禁，隕命亡國，或雲子孫」。他們或喪葬規格越制，或私自鑄幣，或殺傷人命，或藏匿罪犯，或違法佔地等，觸犯法條，淪為罪人。

而「無罪獲刑」者則更多。此處所說的「無罪」，就是王朝的法律條文上本來沒有列的罪名，只是出於皇帝的意願，他們便被定「罪」。

漢武帝在征伐匈奴的戰爭陷入膠著狀態後，為了破解國庫空虛，民力虛耗的難題，想方設法摟錢，籌措軍費，鹽鐵專賣還說得過去，他還發行了一種「白幣」：用上林苑的白鹿皮製作，等值四十萬錢。實質上，「白幣」跟空頭支票沒什麼區別，值不相稱，但漢武帝還是強制向王侯宗室推行，藉此掠奪他們的財富，不少貴族因「白幣」而獲罪。

漢朝還有一種更為奇葩的罪名——腹誹罪，即通過臣子的表情，判斷他對皇帝的不滿，憑此定罪。這條罪名也是從漢武帝時期確立下來的，兩漢時期都在延續使用。

封建君主制度之下，漢代皇帝不好當，臣子的命運更苦，他們戰戰兢兢，如履薄冰，稍不注意，有可能招禍獲罪。被剝奪爵位的列侯，罪有應得者有之，無故受罰者則更多，以漢朝為代表的封建王朝的「法」，只不過是維護皇帝統治的工具，整個王朝，只有一個人不會獲罪，那便是皇帝：他們常常翻手為雲，覆手為雨，隨意羅織罪名，就可以處理一個「逆臣」，全無「法治精神」，以「人治」來代替「法治」，任意而為，以這樣的手法來治理天下，還妄想「江山千秋萬代」，完全是痴人說夢。
