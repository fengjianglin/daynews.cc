“罷黜百家，獨尊儒術”，漢代董仲舒“天人感應”論的演變過程

2019-12-15 1,017條評論 3,604次閱讀 1757人點贊

提到董仲舒，大部分人會想到漢代由他所提出的「罷黜百家，獨尊儒術」的主張，使儒家思想深深影響了後面的幾朝幾代，甚至今世，何不曾想，作為一名唯心主義哲學家，他所做出的成就遠不止這些，他根據《春秋》所延續出的「天人感應」論更是為漢代的帝王統治提供了重要的依據，也是獨尊儒術的基礎。

董仲舒

一、天與人的涵義

「天」的涵義，據多家學說來看，最完整且最具有代表性的是由勞思光等為代表提出的的五義說：他們認為「天」可以有五種意義，依次為「物質之天」，「主宰之天」，「自然之天」，「義理之天」和「道德之天」，這就像「天」是這個世界廣義的一部分，也包含了我們作為人所需要的一切，生存所需的食物，容納我們所需的天地，主宰著天覆地載的力量，指導著人們言語行為的道德準則。

而「人」的涵義，則有王明的觀點，為兩重，一是人的生命，包括生命的來源、成長和死亡；二是人的生活和行為，包括生活的窮、通、苦、樂，行為的吉、凶、善、惡等。在我看來，這個是最通俗簡單的解釋，人可不就是由生至死的生命個體，其間穿插著我們的喜怒哀樂、行為決定、悲歡苦痛、，從而形成了每個人獨特的生命軌跡，進一步構成了整個人類的大命運。

多重的涵義，豐富的義韻，讓天人問題成為了中國古代哲學的主要圍繞對象。北宋邵雁認為：「學不際天人，不足以謂之學」，《史記》的作者司馬遷更是要「究天人之際」而撰史，可見「天人問題」貫穿了古代整個的學術界，甚至是科學界，為古代哲學的研究與發展的基本問題。

司馬遷與史記

二、董仲舒「天人感應」論前身的發展

董仲舒的「天人感應」論無前人之奠基而無以追溯，而這源頭則可以尋到原始社會時期，那時的人們順應、依賴自然而得以生存，對自然的畏懼使他們具體化自然的種種現象，或天神，或土地神，將他們製成圖騰來進行崇拜，這便是天、人之間最初的溝通形式——祭祀，但這並非之前所提的唯心主義思維即感性思維，而只是一種單純的乞求與敬畏。

真正要有精神上的聯繫那是在後來人們進入農耕社會後，前人留下來的寶貴的生存經驗使得人們將他們供奉，有了寄託性的精神因素在裡面，也有了通靈的鬼神因素在裡面，人們將祭天與祭人相結合，就更有了與「天」溝通的感覺，隨之而來的巫術等都是意圖用意志來求得平安、獲得溫飽，而這些為神的先人就是人格神。

而隨著社會的發展，皇權治國，天子出現了。天子，天之嫡長子，權利神授，秉承天意而治理天下。這當然是統治者為鞏固地位而向人們傳播的控制意識，但當時所形成的「天人宗教」提示著天子極大的能力——上可控天雨，下可豐莊稼，甚至是戰爭勝敗都由天子所掌控，雖然不一定都能做到，但只要豐年雨順，勝戰連連，就是天子行使天的意志的功勞。由此可推出當時統治者的治理離不開神乎其神的占卜，「汝則有大疑，謀及乃心，謀及卿士，謀及庶人，謀及卜筮。」此時殷商的人們從信奉多個神演變為了信奉一個「至上神」。

天子：人民心中的至上神

綜合以上幾個轉變時期，可知，「天」的發展由最初的自然之神發展為了具有倫理性質的天倫之神。天論的出現使得天子不僅要了解天文地理，而且還要為自己的統治尋求合理的途徑，是古代文明發展的必然結果。

春秋時期的「百家爭鳴」使「天人合一」學說得到了系統的討論。多家學派的研究方向各不相同，而董仲舒主要遵循的是儒家的學派。孔子所探討的先秦時期殷商「至上神」所用的「仁」以救人，是使人們能擺脫無法控制的命運的理論，即對「天」束縛的人類的反抗。而後的孟子則認為「天」是道德超越的」，是可以給人們提供人倫與生存上正確方向與指導的至高道德。因為他們所追求的禮與天相同，是順應天道的，所以儒家的禮法沿用至後世。

三、董仲舒的「天人感應」論

而由前人觀點發展而來的董仲舒的「天人感應」論是一個以天為本、運用陰陽五行為思想框架的理論體系，這裡「天」的存在由陰陽五行共同解釋。萬物以氣而生，以元為本，而後才有陰陽五行之運轉生息。董仲舒的「天人感應」論包含以下幾個內容。

陰陽五行學說簡圖

1、天為萬物之中心

天者，萬物之祖，萬物非天不生。」 物質上，天為氣，為元，世間萬物，包括春夏秋冬，寒暑節變，山川河流，化形之人，都為其所產之物，四時萬物節律亦然。精神上他認為「天」的道德以「仁」為本，取仁於天，授命於天，則下可致父慈子孝，朝安民順，上可致君威臣忠，風調雨順。天是指導一切人倫道德的開端，是人行為準則的參照物。順天而生，德之本也，逆天而行，天理不容。

2、天與人相似且相通

「人之形體，化天數而成」，意為天人同類，是二者相通相應的前提。人與天相似之地諸多，相對應的地方也有很多「天以終歲之數，成人之身，故小節三百六十六，副日數也；大節十二分，副月數也；內有五臟，副五行數也；外有四肢，副四時數也。」他四肢對應四季，認為人有十二大節，包括腕關節、肘關節、肩關節、胯關節、膝關節和踝關節，各一對，就一共十二節；由大關節與月份的對應，就認為天數應與人的小節相對應，三百六十六節；《黃帝內經》中已有五臟與五行相對之說。

3、陰陽、五行學說

陰陽與五行相互獨立，合起來形成了陰陽五行學說，而此學說可以解釋天地萬物，解釋所有的消長生息。陰陽，天化氣為陽，地成形為陰，升發者為陽，沉降者為陰，「天地者，萬物之上下也；陰陽者，血氣之男女也」。順陰陽者，和諧自然，養生之道也；陽盛於陰或陰盛於陽，都將因為打破穩態而遭至災禍。五行，即木、火、土、金、水五種元素，在商代被系統整合後，有了五行、五方、四時、五帝等系統模式，再進一步轉化為了仁、義、禮、智、信五種人類德性。

商代畫作：財神比干

由上可見，不論「天」是什麼樣子、什麼形式，古人總能將「天」的運行規律等與人的一切生長作息相對應起來，所形成的可以牽涉到各個領域的完整的理論體系是我們今天中華子孫所要好好保護並加以利用的。

四、董仲舒「天人感應」論與漢代天人問題

漢代，是一個脫離秦朝統治而建立起來的朝代，人心惶惶，生產力嚴重不足，社會也不穩定，漢高祖劉邦便需要有效的指導政策來處理種種的社會矛盾，於是有了堪比百家爭鳴的各種思潮的大發展。當時的人們依然局限在自然之力中，對如何才能減小自然災害對民生的影響，使得當時形成了反「天人感應」與順「天人感應」的兩派。

反「天人感應」就如現在的自然科學，認為大自然是一種客觀存在，人們不應依靠占卜祭祀來尋求平安，這絕對是我國古代科學思想發展的一大進步，但在這種社會情況下，統治者需要更強大的信仰力量來平定人心，重新恢復人們的日常生產作息，並取得更好的發展，董仲舒「人天感應」的思想就很有存在的必要。

古代各種各樣的占卜方法

與其相信不能左右的科學現象，統治者們更願意用精神上的力量來領導平均文化水平不高人們。所以漢代即使有反與不反「天人問題」的兩派學說，但以董仲舒的「天人感應」論為中心的學說發展得更好。回到天人感應學說，就又形成了以儒家、道家、法家的多派學說，雖說但從天人合一這個理論來看，道家清凈無為、融於自然的思想向來是最合適的，但因為它不求物質發展的「軟弱性」並不能適應社會提倡發展的需要，所以還是沒有董仲舒的合適。

由此，董仲舒的「天人感應」論成為了漢代天人問題發展的最高峰，其理論為漢代的政治方法探討提供了重要的依據，並推儒家之道於最高地位。其「仁」理，在立法方面，使君王從「仁」而治，無為而治就取其義，「與天同者大治，與天異者大亂」，順其自然，則能更長久地生存下去，畢竟他認為人天是一體的：「天亦有喜怒之氣，哀樂之心，與人相副，以類合之，天人一也。」所以人必當從天而生死，從然而治之。從而有了文景之治這適應社會、精神需求的統治的到來。

漢高祖：劉邦（劇照）

五、結語

董仲舒的「天人感應論」是結合古人的智慧、天道自然與當下社會的需要所發展出來的有效的精神理論產物，不僅影響了整個漢代社會的制度的發展，更是對後世的哲學研究有深遠的意義。漢代人們對「天人問題」的積極的探討也推動了後來的科學與哲思理論的發展，使人們能進一步認識這個世界。古人一次次的思維大變革，一步步形成了我們今世相對完整的哲學生活思維方式，讓中國古代科學、哲學文化得以丰韻、長久。
