相信大家都在書籍中或者網路上看到有關於鬼谷子的介紹和傳言，其中，對於鬼谷子的說法，大多都是說鬼谷子是道家、思想家、謀略家、法家、兵家、教育家、陰陽家、名家於一身，類似於一本百科全書式的人才，更有甚者，把鬼谷子加以神化，說整個戰國都在鬼谷子的運籌帷幄之中，整個戰國都是鬼谷子的一個陰謀。也有人認為，鬼谷子好像是一個從現代穿越到古代的人，因為鬼谷子自身作為一名隱士，卻能培養出許多優秀的弟子，並且影響著整個戰國時期的戰況。

那麼，關於鬼谷子布局天下，究竟是怎麼回事呢？難道說鬼谷子真的被後人神化了嗎？下面就讓我們一起來看看是怎樣吧！

春秋戰國版圖

一、鬼谷子的來歷以及培養出的能人異士

1.鬼谷子的出身

在傳說中所說的鬼谷子，是被世人所神化的形象，但是，既然能夠被流傳下來，肯定是會有一些現實依據的，並且鬼谷子被後人傳的神乎其神，定是有過人之處。相傳，鬼谷子本是軒轅氏的後人，本姓姓王，名禪，別名詡或利，道號玄微子，鬼谷子的額前有四顆肉痣，成鬼宿之象，所以被後人稱為鬼谷子。

對於鬼谷子是哪國人，一共有三種說法，一種說法說鬼谷子是衛國人，一種說法說鬼谷子是戰國鄴國人，還有一種說法說鬼谷子是漢水之濱關內雲陽人，但是鬼谷子究竟是哪國人，沒有哪一本歷史文獻中詳細說明，所以也不得而知。但是相傳鬼谷子隱匿與鬼谷中修鍊，自稱鬼谷先生，鬼谷子也就是以這樣的一個身份後來被更多的人知曉。

鬼谷子假想畫

2.鬼谷子培養的能人異士

另外，鬼谷子的出名，其實和他的弟子們也有著很大關係，鬼谷子出名的弟子有張儀、蘇秦、孫臏、龐涓、商鞅、白起、毛遂、茅蒙、徐福等等，他的這些弟子在戰國時期可是有著一定的名聲和威望，所以鬼谷子也隨著弟子們的出名，名聲自是傳到各國各地。

就從鬼谷子的這些徒弟中，就能看出鬼谷子的教育能力，比如說張儀和蘇秦兩人，一人習連貫之術，一人習合縱之術，分明是兩種對立的觀念，但是在戰國時期的記載之中，兩人還能互幫互助，在張儀列傳中就有記載，蘇秦明面上打擊張儀，但是暗地裡還是幫助張儀，這也是非常神奇的現象。也正因為張儀蘇秦這兩個人，有人甚至認為整個戰國都是鬼谷子的一個陰謀，而張儀和蘇秦都是奉鬼谷子的命令，但由於沒有確切的歷史文獻證書，所以真相究竟是怎樣也沒有具體的參考。

蘇秦張儀劇照

3.鬼谷子的著名作品

鬼谷子還有著著名的作品傍身，其中有《鬼谷子》、《本經陰符七術》和《關令尹喜內傳》，但是，鬼谷子的書卻被列為禁書好多年，這其實與當時的政治背景有關，鬼谷子的書寫人性本惡，所以書中的很多思想都是關於揣摩人心，用計謀來達到目的的，而封建社會大多都是實行儒家學派的思想，儒家學派主張人性善論，所以鬼谷子的學說和當時的政治背景難免會有衝突，相比焚書坑儒來說，鬼谷子的書只是被作為禁書已經好了很多了，然而現如今，在這個思想高度自由的時代，我們還是有很多人從鬼谷子的書籍中學到了不少為人處世知識，給予人們以重大影響。

二、歷史中對鬼谷子的記載

1、《史記》中沒有關於鬼谷子的記載

鬼谷子，這麼一個一直流傳下來並且擁有著人們較高評價的人物，竟然在《史記》這本書中沒有一句提及到，這令人匪夷所思，因此，就有人開始產生了懷疑，歷史上真的有鬼谷子這個人的存在嗎？如果有的話，為什麼《史記》這本記錄了眾多有名的歷史人物的書籍中，沒有為鬼谷子列一個傳記呢？只是在《史記·張儀列傳》和《史記·蘇秦列傳》中提到過鬼谷子一句而已。

司馬遷與其史記

對於《史記》中沒有關於鬼谷子的介紹，這點不得而知，不知道是不是鬼谷子哪點惹到了作者，或者作者把鬼谷子一時忘記了。這都不是我們能想到的了，但是可以確定的是，歷史中是有鬼谷子這個人物的存在的，並且在一定程度上影響了整個戰國時期的戰況和結局。

2、徐福到日本為秦始皇求長生藥

茅蒙是鬼谷子的一個弟子，傳言中說他白日升仙（《洞仙傳》中有關於茅蒙白日升仙的記載），這讓每天輾轉反側想要得到長生不老的秦始皇得知後，肯定是要找人來問問清楚的，而這個人非茅蒙的師傅鬼谷子莫屬，在《金簍子》中有寫道：「秦始皇聞鬼谷先生言，因遣徐福入海求玉蔬金菜，並一寸椹。」

在日本的《日本國史略》中也有記載：「孝敬天皇十二年春，秦人徐福來。」這也證實了這個徐福蹭到日本的存在。徐福此後就在日本生活，他把他的七個孩子分別派送到日本的七個地方生活，從此徐福的子孫在日本繁衍生息，甚至有人認為徐福是日本國父。後人只知道徐福到達日本求長生藥，卻不知道徐福到達日本還有鬼谷子的手筆。

徐福東渡圖

3、鬼谷子在道家的記載

在《正統道藏》中有記載著鬼谷子的神通法術，精確的記載著鬼谷子的修鍊道場，和修行各種的法術，法術的修行方法等，並且還傳出了《鬼谷子天髓靈文》這樣的法術秘籍，這樣的記載，使鬼谷子從擁有神通本領轉身變成了修道真人，成為了道家學派的鼻祖人物，此後，道家稱鬼谷子為王禪老祖，尊為玄微真人。在唐朝的《錄異志仙》中說：「鬼谷先生者，古之真仙也。」這直接將鬼谷子成為長命百歲的仙人了。

然而，鬼谷子有著這麼大的神通本領，為什麼依然留存在那個戰亂的世界呢？對於這點，道家也給了一種說法：鬼谷子在凡間還有使命沒有完成，所以就收留弟子，交給弟子一些本領，為的就是幫助華夏完成統一大業。

王禪老祖像（八卦城）

三、鬼谷子為什麼會被世人傳成神話一樣的存在？

1、官方的歷史記載

在一些歷史材料中記載著，鬼谷子存活於世一百三十多年，在古代，這個食物匱乏的年代，能夠活到七十歲，已經算是高齡了，鬼谷子存活在世比別人多了一半，這就夠讓人多想了，並且，鬼谷子還是修道之人，所以就被世人認為鬼谷子是活神仙了。

在《戰國策·東周》中，有形容鬼谷子的一句話：「一人之辨，重於九鼎之寶，三寸之舌勝於百萬雄兵。」還在《孟子·騰文下》中的一句話，也是形容鬼谷子的，「一怒則諸侯懼，一息則天下安」。這兩句話可以說明了鬼谷子的能力之大。

戰國策典藏版

2、民間的小道消息

還因為鬼谷子對兵法、教育、道家學說、法家學說、陰謀論還有陰陽五行等方面都有所了解，所以後人學習這些方面的規矩時，都會把鬼谷子當做鼻祖，甚至把鬼谷子神化。

就比如說在唐宋時期，曾出現過一種算命書——《命書》。《命書》這本書所注是：「鬼谷子傳，李虛中注」。李虛中是唐代的一個大臣，也是一位精通算命學說。跟據有關歷史研究專家表示，《命書》並不是鬼谷子的著作，李虛中的註解也是後人假照出來的。在古代的道士中，有不少人為人們進行占卜算命。鬼谷子被後人當做神人一樣的存在，所以就會有一些人用他的名義來寫書，這也是合乎情理的事情。就像是在現在的這個社會，也免不了有人會用名人來博取眼球，就是這個道理，才使鬼谷子越來越神化。

古代道士算命

3、道家對鬼谷子的神化

鬼谷子被世人傳的越來越神，兵法家尊他為聖人，縱橫家尊他為始祖，算命占卜尊他為祖師爺，謀略家尊他為謀聖，名家尊他為師祖，道教尊其為王禪老祖，在文化史上，他是與孔子、孟子、莊子、荀子、墨子、韓非子齊名的學術大家。

神仙一般都是擁有一個坐騎的，比如說老子的坐騎就是青牛，然而，民間對於鬼谷子的坐騎的流傳，卻是兩個動物，一個是獅子，一個是老虎，這可是兩個十分的威武的動物，反應了民間對鬼谷子人、神、鬼三位一體的假想。給鬼谷子的形象增添了一份神仙色彩。

鬼谷子虎豹坐騎

四、總結

不管怎麼說，鬼谷子的自身肯定是非常優秀的，只有這樣，才能被世人所廣泛流傳，這一點是毋庸置疑的。但是鬼谷子肯定也是和我們一樣的肉體之身，並不會像小說中所描述的那樣神奇，所以鬼谷子肯定中間又被後人神化。被神化的原因有三，一是在戰國時期，鬼谷子就有著很大的作用，二是民間流傳著關於鬼谷子各種各樣、天花亂墜的傳說，三是道教對鬼谷子的神化。正是這三點，才成了我們現在了解的鬼谷子的形象特點。
