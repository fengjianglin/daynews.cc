行而世為天下法：古代官員模範，“百僚之師”朱軾的為官之道

2019-12-05 0條評論 17次閱讀 0人點贊

摘要

朱軾，字若瞻，又字伯蘇。號可亭。清中期名臣，著名文學家、史學家、經學家、政治家。當世以及後世評價頗高。被稱為“朝堂良佐”（雍正）“帝師元老”（乾隆）“百僚之師”。那麼，朱軾為什麼會被稱為“百僚之師”呢？這和其為官之道有關。

朱軾，字若瞻，又字伯蘇。號可亭。清中期名臣，著名文學家、史學家、經學家、政治家。當世以及後世評價頗高。被稱為「朝堂良佐」（雍正）「帝師元老」（乾隆）「百僚之師」。那麼，朱軾為什麼會被稱為「百僚之師」呢？這和其為官之道有關。

一、政治能力突出

中國曆來的賢臣、能臣，都有一個共同的特性，那就是個人能力突出。沒有哪個君主會任用一個不學無術，能力低下的臣子（昏君除外）。無論是重任在肩，託孤輔政的霍光、張居正、還是直言敢諫、衷心體國的魏徵、宋景。當然，朱軾也不例外。

朱軾畫像

1、為官清廉，秉公執法

康熙後期，官場貪污腐敗早已成風。官吏中能夠潔身自好的寥若晨星。而朱軾就是其中之一。朱軾奉康熙命令查收漁利，秉公辦事，絕不徇私，舉報了向商人行賄的巡鹽御史哈爾金，深得康熙帝的賞識（授左都御史）。是金子總會發光。到了雍正朝，由於雍正決意整頓吏治，朱軾很快就得到了雍正帝的器重（授文華殿大學士）。進入中央決策高層。

2、勤政惠政，造福一方

自古以來，君主欲治理好國家，維護自己統治，那麼必須要明白「水能載舟亦能覆舟」的道理。可是君主「居北辰」，很難洞悉民情。那麼，他們需要一批可靠的官員來傳達他們的意思，從而安撫民心，維護專制統治。從小洞悉歷史和儒學的朱軾就很合這一點。

雍正畫像

潛江風俗鄙陋，多苛捐雜賦。朱軾知潛江，立即減免賦稅，減少人民負擔。並大力提倡勤儉，這使得勤儉蔚然成風；海寧，上虞自古多水患，人民飽受其害，而朱軾經過研究之後，首創「水櫃法」，從而沿海百姓免除水患。康熙朝由於官員貪污，向國庫借款導致了國庫虧空，甚至河道決堤都沒錢賑災。可見，「水櫃法」的創建意義極大。

3、潔身自好，不搞幫派

康熙後期，由於太子被廢，導致了「九子奪嫡」。中央甚至地方臣子分為數種勢力，彼此攻詰。這個時候，那些潔身自好的人就顯得格外可貴。朱軾也是。賑災的他見到皇四子胤禛，並不阿諛奉承，而是一如平常，為民賑災。這引起了雍正的注意。和鄂爾泰一樣，朱軾在雍正即位以後，立即得到了皇帝的重用。（這是一種「框架轉換」思維。作為皇子，自然希望臣子支持。但作為皇帝，是不允許臣子拉幫結派的）

康熙畫像

二、個人修養較高

1、忠心體國，兢兢業業

中國古代帝王，特別是進入封建社會以後，對於臣子最大的要求，就是要忠心於帝王。肯為帝王辦實事，肯為君主專制服務。朱軾自從入仕後，兢兢業業，從地方知縣（類於現代的縣長）一直升到中央的刑，吏，兵三部尚書（司法部長，組織部長兼人力資源與社會保障部長，國防部長），文華殿大學士（名譽宰相），足可以看到皇帝對他的信任與器重。甚至一向以嚴苛著稱的雍正皇帝都將御筆題寫的「朝堂良佐」匾額賜予朱軾。

2、生活簡樸，為人低調

中國自古提倡樸素美。一些明君賢臣都是簡樸的榜樣。比如漢文帝、唐太宗、比如曹操、馮道。在清代官場上，能夠保持生活樸素的官員很少。何況是貪污成風的環境呢？康熙四十二年，皇四子胤禛（後來的雍正帝）前往浙江與朱軾配合救災，看見朱軾奮力救災，吃粗茶淡飯，沒有僕人，雍正大受感動。從而朱軾給雍正帝留下了很好的印像。

朱軾立功以後，並不像年羹堯一樣居功自傲、沾沾自喜，而是一直低調謙遜。這有效的避免了君主的猜忌。也使他避免了白起、韓信、岳飛的悲慘下場。

三、通曉程朱理學，維護君主專制

程朱理學，是對傳統儒學（孔孟荀董儒家學說）的繼承與發展。在傳統儒學原有的基礎上提倡「存天理，滅人慾」「格物致知」「三從四德」等觀點。從南宋以後，程朱理學成為了官方正統哲學。其本質之目的，乃是維護君主專制制度，從而統一，固化人民思想。最重要的是什麼？是思想。眾所周知，滿清是外族入關統治中國（現在我們都是中華民族），尤其怕漢人萌生「反清復明」的思想，所以，滿清的思想禁錮是歷代最厲害的（比如文字獄）。

拿什麼來禁錮人民思想呢？，那就是程朱理學（有一定優點，但弊端不容忽視。）朱軾從小聰明好學、對於經學、史學、文學無所不通。他是朝廷御用程朱學派的代表，對於維護清廷封建集權，君主專制統治起到了重大的作用。不僅是理論上，在實踐上，朱軾也將程朱理學思想貫徹到其為官治理上去。

知潛江期間，他向潛江百姓提倡程朱理學、推崇儒學、興辦教育，主張「讀對賢書而能發明其義蘊」並在當地刊發《大戴禮記》、《顏氏家訓》等書籍，使得浙江民風為之一變。使得人民向學，安分守己，對於維護社會穩定，維護君主專制起到了很大的效果。

《大戴禮記》

四、調和「民本」與「專制」的關係

民本與專制自古就是一個嚴肅且矛盾的話題。要民本，那麼就會損害君主專制；要專制，那麼就會損害民本。明太祖曾經就對孟子的「民本」思想提出了嚴重的反駁與批評。可是，像朱軾這樣的臣子明白，損害民本，有利於皇帝集權與君主專制，但是不利於國家的長期穩定；損害專制，那麼遲早會有人在思想上「覺悟」，進一步危害君主專制（畢竟他們是封建官僚）。

但朱軾就很能調節兩者的關係，筆者認為朱軾調和二者的方法是「在君主專制的框架下實行民本」。這個思想是非常智慧與理性的。為什麼呢？因為他看到了中國封建社會歷史前進的趨勢，那就是中央集權不斷加深，君主專制不斷強化。從皇帝制創造伊始（秦始皇），漢武帝設置內朝，隋唐創建完善三省六部制，宋實行二府三司制，明太祖廢丞相，明太宗（朱棣）設內閣，康熙帝設南書房，雍正帝設軍機處。

軍機處

中央內部尚且如此，更何況地方和中央呢？所謂「世界潮流浩浩蕩蕩，順之則昌逆之則亡」（孫中山語）潮流是不可逆的，而朱軾的思想，就是在中央集權的基礎上，在維護君主專制的基礎上，將專制與民本進行一個內部的調節，使得二者都能兼顧。所以，朱軾上可以得到皇帝的賞識與器重，下可以得到百姓的尊重與愛戴，所以，他能成為「百僚之師」不是沒有道理的。

五、選拔人才，一箭雙鵰

上文提到，朱軾不僅學識淵博，而且秉公持正。他一生三次主持會試，為朝廷選拔了很多人才。不僅是在考試上，在為官實踐上，他也發現並培養了雍正朝名臣李衛。這是一件於國家於自己都有利的事情。首先，對於國家來說，和現在的公司一樣，人才流失是國家的一個致命損失，為什麼說它致命呢？原因有三。人才流失導致國家缺失了一批為國奉獻的傑出人才，會使國家在智力決策方面遭受一定的影響（特別是封建社會），此一也。

在封建社會，人們常把「野無遺賢」作為一個國家政治清明的標準。因為每逢亂世，亦或是政治黑暗，總是有一批優秀人才隱居不仕。比如陶淵明，劉基。此二也；由於優秀人才不被賞識，難免心生怨恨從而做出敵視中央政權的舉動。在北宋時，由於科舉制度不完善，很多優秀人才被埋沒，他們懷恨在心，投奔西夏，為西夏進攻北宋出謀劃策。更甚者，有黃巢起義差點滅亡李唐王朝。此三也。所以，人才對於國家非常重要。

其次，對於個人來說，也有兩點好處。一、在古代，應試士子對主考官非常尊重與敬仰。而主考官也會把考生當做門生相待。如果這些考生以後成為顯赫官員，在官場上的照顧是不可少的。這是利。二、古代能夠作為主考官是一件極其體面，極其榮耀的事情，而古代的文官也以桃李滿天下為榮（現代也是）。畢竟孔子說「得天下英才而教育之，三樂也」。這是名，名利兼收，也是為官之道啊。

六、總結

作為封建官僚，朱軾的為官之道最根本的一點就是維護君主專制，其為官之道，無論是政治上，還是思想上，都是在為君主專制服務。而封建帝王最需要朱軾這樣的封建官僚來維護自己的統治，所以他可以成為古代官僚的模範，被稱作「百僚之師」也是不難理解的。
