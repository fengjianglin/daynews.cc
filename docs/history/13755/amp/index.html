張耳陳餘的關係真的有那麼好嗎？一些細節實則看出關係糟糕透頂

天天要聞
2019-12-10

古往今來，成王敗寇，誰都不想低人一等，誰都想做世界的主人，《資治通鑒》說，張耳、陳餘原本是「同生死、共患難」只差同床共枕的好基友。關係如此之好那有怎樣，他們二人各懷鬼胎，野心勃勃，都想稱王。擁立武臣為王只不過是自己稱王路上的一個過客而已。為何這麼說？

當初大澤鄉起義，陳勝吳廣作為首領，一路大捷，隊伍不斷壯大，張耳陳餘主動請纓攻打趙國，陳勝於是派武臣攻打趙地，張耳陳餘作為謀士輔助，可是勝利後，張耳陳餘陳述天下群起，慫恿武臣自立為王，面對強大的權利，武臣妥協了，自立為王，而此時正是中了二人的反間計。為何這麼說呢？

當時二人地位不穩，而陳勝特別高傲，殺友人，脾氣暴躁，伴隨在這樣的人身邊如同是在老虎身邊謀生，危機四伏，一不小心就可能被殺，突然想到趙地是個好地方，趁機佔領地位。面對利欲熏心的武臣來說，當個王總比當個將軍好吧，他痛痛快快答應。為了鞏固地位，認真輔佐武臣建功立業，趙國建立的也越來越好。

趙國政權建立不久，武臣便命部將李良攻常山。功克常山後，又被派遣攻略太原，由於久攻受挫和秦軍的誘降，李良開始動搖，恰在這時他又受到武臣姐姐的羞辱，於是就決意叛變，「遂將其兵襲邯鄲。邯鄲不知，竟殺武臣、邵騷」 。在這次變亂中張耳、陳余因事先有人通風報信，得以逃脫。

試想一下，為何他們二人得以逃脫，武臣及右丞相都難逃一死，痛風報信之人為何如此及時，不會是二人的計謀嗎？

他們二人逃脫之後立刻集結兵馬，反立趙國後裔趙歇為王，國已破的趙歇根本沒有任何實權，當時全國上下全由張耳一人說了算，由此實權落到了張耳的手裡，正是張耳所要達到的目的。此時陳餘當然不滿，為何張耳高於自己，後來以至於秦軍兵臨城下，張耳派人冒著性命請求陳餘相救，但陳餘手握重兵卻不相救，最終由項羽相救。二人從此鬧翻，張耳憎恨陳餘，陳餘卻憎恨他居高臨下，總使喚他，慢慢走向決裂。

項羽攻破函谷關進屯新豐、鴻門。自立為西楚霸王，同時封王十幾位，項羽也素聞張耳之才名，又念其扶趙抗秦之功，乃分趙地北部，立張耳為常山王。此為常山王之始。張耳立國，陳餘甚怒，對人道：「張耳和餘功相等。今張耳為王，我卻稱候，項羽對我不公。」於是二人怨念賊深。見不得對方好。

劉邦向東進擊楚國，派使者通知趙國，要和趙國共同伐楚。陳餘說：「只要漢王殺掉張耳，趙國就從命。」可見對張耳恨之入骨。

後來陳餘向田榮借兵攻打常山王張耳，張耳敗走，投靠劉邦，假以時日，張耳與韓信統兵數萬東出井陘擊趙，使萬人過河列營，成背水陣。

此陣一成，只許勝，不許敗。故而漢軍將士皆作殊死戰，勇猛異常。陳餘沒有聽從李左車的建議輕視韓信的背水列陣法，敗後被斬殺於泜水。結束了二人的鬥爭。
