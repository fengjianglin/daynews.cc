梅塞納斯，奧古斯都的貼心助手，也是羅馬文學家的資助人

2019-12-22 175條評論 3,172次閱讀 703人點贊

梅塞納斯是羅馬由共和轉向帝制的肇始者，也是羅馬文藝界最大的贊助人之一。他出身於貴族，卻沒有紈絝子弟的囂張。他生性聰慧，卻沒有篡奪權位的野心。這樣一個偉大的人怎能被歷史所掩埋？今日瓷兒就帶大家一起來認識他。

一、梅塞納斯簡介

提起梅塞納斯（Maecenas）這個名字，相必熟悉文學或藝術史的同學都不會陌生。他的大名至今被英、法、荷、意等西方國家當作是「贊助商」的詞根。比如英文中的Maecenas，荷蘭語中的mecenaaat和法文里的mécénat。知名的奢侈品商萬寶龍就推出過以「梅塞納斯」命名的紀念款鋼筆。

萬寶龍同款紀念鋼筆

梅塞納斯的全名為蓋烏斯·克利尼烏斯·梅塞納斯（Gaius Cilnius Maecenas），是羅馬國家從共和制走向帝制的重要奠基人。梅塞納斯出生於阿雷提烏姆，即今日義大利托斯卡納區的阿雷佐。他的父親大梅塞納斯是一位富有的大商人，主要在共和國內部負責監督官營手工業，並和周邊的民族開展貿易互動。

梅塞納斯從小就是一名勤奮好學的孩子，貴族的出身能為他免除生活上的所有負擔。同樣，從小培養出來的獨特氣質，幫助了他更好地理解當時的羅馬文化。

梅塞納斯一開始便接受了正統的希臘文學教育、拉丁文教育和演說術教育。他從母親那裡繼承下來的伊特魯里亞血統，幫助他延續了注重實際的優良傳統。梅塞納斯並不將學問看為空洞脫俗的東西，他那孜孜不倦致力於投身政治的欲求，不單單是為了滿足家族對他的青睞，更是為了實現他自己的人生理想。

伊特魯里亞雕刻藝術

二、梅塞納斯對屋大維的巨大幫助

關於梅塞納斯的故事，人們提到最多的還是他同羅馬元首屋大維的親密關係。這種關係究竟親密到什麼地步，是我們現代社會的人無論如何都難以想像的。

史學家們考證了一眾羅馬史家的著作，最終認為梅塞納斯同屋大維有過長期的「共妻」關係。即梅塞納斯的妻子同屋大維有過私通，但梅氏卻仍然保持著同屋大維的友好。這種關係一直維持到到逝世，也沒有顯示出要同屋大維決裂的徵兆。更令人不解的是，梅塞納斯立下的遺囑明確規定，他要將他今生的財富全部贈予這個對他有「知遇之恩」的夥伴。

現存英國的梅塞納斯雕像

無論梅塞納斯如何「大度」，我想一個正常的男性都不會允許他的配偶被他者長期侵佔。不過在盛行「同性相悅」的古代希臘羅馬地方，這種不堪入耳的醜事說不定還真能發生。

關於梅塞納斯本領的第一次記述，是在腓利比戰役之後的政治僵局中出現的。在戰爭結束之後，屋大維面對的最大困境便是對退伍士兵的處置問題。按照往常的習慣，征戰回國的將士務必會得到一筆使人滿意的財富。但屋大維卻錯失了良機，他沒有聽從梅塞納斯的建議，反而要用抽抵公民財產的辦法來安置退伍士兵。這個決定引發了軍隊的普遍不滿。

古代羅馬士兵

在這個危機關頭，梅塞納斯趕忙駛往西西里。他此行是為了幫助屋大維獲取來自塞克斯圖·龐培的政治資源，也是為了臨時借龐培之聲望來幫助屋大維抵消國內的反對呼聲。

一波未平，一波又起。安東尼在隨後聽從了她妻子的建議，抽調了7支軍團向屋大維發難。要知道，屋大維在此刻根本沒有拿得出手的士兵們能同安東尼相抗。又是在此萬分緊要的關頭，梅塞納斯活生生地為我們上演了一出驚現的「空城計」。

梅塞納斯利用他沉著的外交手腕，使安東尼一行人認定屋大維在蒲隆地西烏姆城內藏有大量的伏軍。安東尼也因為東方的反叛而顯得焦頭爛額，因此雙方選擇了議和。於是，兩強在公元前40年最終達成協議。決定繼續延續同盟的關係。

在此之後，梅塞納斯便作為屋大維的心腹而活躍在政壇。雖然當時的屋大維剛剛打敗了大敵，但安東尼在東方的用心經略和元老們對於他執政合理性的懷疑卻讓屋大維一直沒有辦法放鬆警惕。

號稱「凱撒右手」的馬克·安東尼

公元前31年，屋大維在亞克興戰役中擊敗了安東尼和埃及女王的同盟。梅塞納斯在這場戰役中雖未登場，但是他的作用仍然是不可忽視的。屋大維在出征之前，曾經授予了梅氏十分廣泛的決策權。在他代理屋大維管理羅馬和義大利的同時，他成功鎮壓了雷必達兒子的謀反。

要知道，尚不到而立之年的屋大維在執政初期是被所有人輕蔑的，包括他最大的敵人安東尼，也嘲笑他是「仍在哺乳的嬰兒」。屋大維除了擁有凱撒繼承人的頭銜外，他的最大優勢便是有一批忠心耿耿的老兵作為他的軍事支撐。除此以外，他所依賴的能夠為他擴充政治優勢的助手便是梅塞納斯了。

美貌絕倫的埃及豔后克婁奧帕特拉七世

三、梅塞納斯對羅馬國家的貢獻

要說梅塞納斯作為屋大維最貼心的心腹，其執政能力之強，是完全能夠與屋大維分庭抗禮的。可令人意想不到的是，梅氏對權力的淡泊和對屋大維的忠誠卻讓所有人大惑不解。

首先，梅塞納斯窮其一生也沒有要求過任何政治頭銜。其所充任的高官職，也不過是屬於「最下層長官」的騎士而已。其次，在屋大維成功征服了所有反叛者之後，梅塞納斯又用他獨到的眼光幫助他的朋友決定採取君主制的統治模式。

在屋大維完成了征服返回羅馬之後，自身其實是並未決定究竟採取何種形式來進行統治的。屋大維最得力的部將阿格里帕比較傾向於維護共和，他曾言：「可以肯定，沒人能斷言我們必須選擇君主制。因為它並無利益可言。」

可梅塞納斯卻在分析了羅馬政局之後，對屋大維進行了一次「洋溢著崇拜之情」的吹捧。梅塞納斯好像十分懂得屋大維心理上的需要。經過了這次勸說，屋大維愈發堅定了他的帝製圖謀。

奧古斯都造像

四、梅塞納斯對文學藝術的貢獻

最後我們來談一談梅氏對羅馬文學和藝術的關照。梅氏由於出身高貴，因此打小就被熏陶出了對文學藝術的鑒賞能力。終其一生，他為自己積累了不少財富。

他曾在羅馬七丘之一的艾斯奎里諾山上建造了一尊屬於自己的花園豪宅。據說這座豪宅是他親自設計和監督施工的。這尊極富有創造性的豪宅還搭建了當時鮮有的溫泉浴池，這種富有藝術情調的奢華住宅到後來招致了包括羅馬哲學家塞內卡的批評。

雖然梅塞納斯也曾經嘗試過自己寫作散文，但他的作品卻被一眾好友嘲笑為「擠膏藥的藝術」。可梅塞納斯卻仍舊樂得保持他對文學藝術的執著。他曾經將自己的農場贈送給大詩人賀拉斯，亦常常在自己的宅邸中舉辦文學沙龍。他還向屋大維推薦了不少善於營造建築的工匠，從這點來看，梅塞納斯對於羅馬文化的發展的確有自己獨特的貢獻。

極具氣派的艾斯奎里諾山

文史君說：

可惜的是，梅塞納斯晚年因受讒言所絆而失寵於奧古斯都，遂逐漸在政治活動中銷聲匿跡。梅塞納斯的冷遇可能與他的妻子特倫提婭最為相關。無論是在蘇維托尼烏斯或是迪奧的著作中，都將特氏描繪成了一位生性放蕩的淫婦。她同奧古斯都的幽會糾纏和枕邊蜜語，可能引起了皇帝對梅氏的警惕和不滿。可是，梅氏仍舊選擇將自己的忠誠和熱情完完全全地奉獻給他的朋友。這種對友情的絕對珍重，讓我們今天讀來，也不無慨嘆。
