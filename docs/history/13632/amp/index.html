上世紀，在漢墓發現一堆竹簡，翻譯成書後，西方專家大呼不可能

天天要聞
7 天 ago

我國有著幾千年的歷史，隨著時代的發展和朝代的更替，歷史不斷在演變著。其中漢朝就是一個強盛富足的朝代。想必其它朝代，漢朝對我們的意義特殊，因為如今我們孤家的主要民族就是漢族，而漢族這一名稱就是源於漢朝。

說起漢朝，我們就會想到強盛和繁榮二詞，漢朝一共存在了400多年，29位皇帝中，有很多都留名於青史。當時的漢朝在整個世界是有著很大的威望的，國土遼闊，國力強大，科技和醫學等也是處在世界前茅。

而在近代的考古研究中，很多面世的大墓都死漢朝的，通過挖掘研究古墓，專家學者們發現了大量珍貴的歷史文物，而這些文物則默默記錄著數千年前人們的智慧和文化。大家應該都聽過金縷玉衣，而這種文物的存在就說明墓主的身份很高貴。下面我們要講的是兩本漢代的書籍。

對考古有研究的人應該比較清楚，湖北江陵地區是我國出名的古代墓葬之地，這個地區的古墓由周朝至魏晉不等，而這些古墓葬中，最主要還是漢墓。張家山古墓就是規模較大的一個漢墓，起初剛被發現時，出於對歷史的尊重，沒有立即挖掘考古，而是一直等到了十多年後的1963年，考古專家們才對其進行搶救性挖掘。

因為這個墓地的地理位置，如果再不進行挖掘，那麼在不久之後，它可能會受到地下水的侵襲。專家們進行了利益權衡，不忍心看著文物被毀，於是開始挖掘考古，然而，當挖掘工作進行了2個多月後，並沒有出現預料當中的青銅器和玉衣等珍貴的文物。

在古墓中，專家們只發現了大量的竹簡，而這些竹簡經過專家們的研究解讀，才知道這竹簡上記載的是八本古典的書籍內容，當中的兩本甚至是很早就失傳的書籍《引書》與《算數經》的佚本。

這兩本本於戰國時期就已經失傳，面世後，徹底顛覆了如今世界上固有的醫學史和算數史，而西方的學者和專家們看到了我國專家還原後的內容，徹底傻了，完全不相信，大呼：不可能！

究竟這兩本書上面寫了什麼呢？西方學者反應竟然如此之大。第一本的《引書》，早在1990年的時候，就有報刊專門講述了它，此書寫於戰國時期，內容主要是道家思想引導的養生手法。

早在秦朝，時就有”黃老之術與中醫、道家不分家”這一說法了，此書更證明了。書中也記載了下頜關節脫位整復術和叩齒術，這兩種手法更是改寫了如今世界的醫學史，許很多人都不敢相信我國會寫出如此記載。

而另一書《算數書》更有名，都知道我國的數學發現得很早，最早的算數的書籍是三國時期的九章算術。而《算數書》比九章算術還早幾百年，所以說此本一面世就改寫了世界的數學史。

其實，此書或許還不算是我國最早的數學方面的書籍，後來在湖南的一古墓內發現的竹簡中，又知道我國早在戰國時就有了乘法口訣。

不過，雖然事實擺在眼前，很多西方學者卻還是持懷疑態度。但事實終究是事實，我們堂堂一個大國，確實不屑於捏造事實來改世界的歷史，相信公道自在人心。
