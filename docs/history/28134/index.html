納蘭容若離世年僅三十，天之驕子為情所困？與納蘭性德又是何關係

2019-12-22 1,308條評論 4,653次閱讀 3623人點贊

想到「納蘭」這個有名的姓氏，不少人腦袋裡會蹦出納蘭容若和納蘭性德這兩個名字。可是你知道嗎，這兩個名字原是同一人！葉赫那拉氏·納蘭性德，字容若。一個幾歲出口成章、十七歲便進國子監、十八考中舉人，卻是康熙貼身侍衛的這樣一個文武雙全的天之驕子。

唐詩、宋詞、元曲，已經固化了人們的認識。宋朝的確有非常多驚艷的詞，但清朝，擅長寫詞的納蘭容若也是驚為天人啊！這個一出生就在皇室受盡恩寵的公子，就像是自帶光環的電視劇男主一般——家世顯赫、樣貌非凡、才華橫溢。無數褒義的形容詞就這樣毫無瑕疵地鑲嵌在他身上。為何這樣一個天選之子，就單單只存於人世三十年?

朦朧初戀，卻棒打鴛鴦

自古才子多情種，納蘭容若也不例外。據說納蘭一周歲抓周之時便已預兆了一生——毫不猶豫的抓住了毛筆與釵子，果然是個會寫詩的情種！

可惜偏偏就是如此命運弄人，鍾情於父母雙亡的表妹。身為皇家大族，母親可是榮親王之女，正宗愛新覺羅家族血脈！怎能允許這樣門不當戶不對的親事存在！「棒打鴛鴦」毋庸置疑，然而納蘭豈是這樣就放棄的？親求、絕食，無所不用其極，但命運就是這般愛捉弄人——表妹被選為嬪妃，這下可變成了皇帝的女人。「一入宮門深似海，從此兩人是路人」！

怎麼辦？回憶湧上心頭，越發心痛難過。文人的憂愁思緒往往能催生出不朽的作品。就這樣，「一生一世一雙人」、「相思相望不相親」的絕美詞句《畫堂春》誕世了。一日一日的思戀憂愁不僅留下了曠世詞篇，也讓他日漸憔悴消瘦。

重燃希望，卻不隨人願

畢竟是皇室，婚姻便不由自己做主。很快父母便給他物色了一個門當戶對的婚姻，他以為一生便將這麼平淡的過去，卻沒想到，盧氏的善解人意、賢惠細膩，竟慢慢的走進他的心裡。融化了他因表妹而築成的城牆。

從此，煮酒聽雨、賞花遊船、無邊風月，他逐漸明白，初戀是朦朧的，但盧氏才是他一生摯愛。可是美好的生活總是那麼短暫，僅僅只有三年時光。盧氏因難產而死，這無疑是給了納蘭晴天霹靂！從此鬱鬱寡歡，盧氏名字中含有「若」字，他便餘生說盡「若」字。《木蘭詞》為她所作，「人生若只如初見」為她所感。不能說自此頹廢，因為他的才氣，以任何形式放在任何地方都不可磨滅。但的的確確是對生活失去了希望。

抗爭、叛逆、隕落

他很命運多舛，他恨父母偏執再次給他擇了正室，他恨自己不能留住愛人，他決定抗爭，而就在這時，不知是命運安排，還是緣分註定，他遇見了江南的沈宛，一個擁有曠世才情又一樣多愁善感的藝妓！她的一句：「原是你」打開了他的心門，說不清是愛還是知己，亦或是在向命運抗爭，他將他帶回府邸。然而青樓女子又怎能進皇室門檻？他乞求著，那怕只是妾的名分，他只想留下她。然而不能，她走了，也同時帶走了他最後的希望。

寒疾複發，這一次，他沒有抗爭。「當時只道是尋常」，是啊，從前的一切歡樂已經逝去，而當時卻沒有好好珍惜。在這短短三十年的前半生多麼瀟洒光彩，後半生就有多麼的思愁萬千，鬱鬱寡歡。他從始至終都只是想和心愛的人共度此生，然而命運卻讓她們一個個離去。他像流星般隕落，但卻留給了後人無數不朽的絕句！

納蘭容若，短短三十年，絢爛過、低迷過、愛過、恨過、欣喜過、最終帶著深深的遺憾離開。也許他不該生在顯赫的家族；也許他不該學會痴情；也許，沒有也許了，他真真實實的離開了。
