豐臣秀吉為何要離開蜂須賀小六？原因發自秀吉內心對腐朽的痛恨

少年的豐臣秀吉叫日吉，17歲那年秀吉遇到了蜂須賀小六。

相遇的時候秀吉的處境十分慘淡，吃不飽穿不暖，連住宿的地方都沒有，只能睡在一個被人丟棄的破船里。

小六答應讓秀吉有安穩的工作環境，但是前提是秀吉要服務於蜂須賀小六，意思很明確就是要讓秀吉給自己當手下。那時秀吉還不知道小六的為人，但是聽說小六為了伸張正義而緝拿自己的親侄子渡邊天藏，因此在秀吉眼中小六樹立了一個正義的形象。

蜂須賀小六確實不同於其他大名，他的內心有很大的抱負，也很有正義心，因此秀吉也覺得沒有認錯主。不過秀吉也沒有正式宣布自己要侍奉小六，而僅僅保持上下級的關係。

秀吉對蜂須賀小六忠心耿耿，小六讓秀吉幹嘛他就去幹嘛，從來不違抗命令，也不會擅自主張。

後來蜂須賀小六的人生中發生了一件重要的任務要做，美濃國國主齋藤道三秀龍要聯手蜂須賀小六攻打義龍。義龍是齋藤道三秀龍的親兒子，父子相殺違背天道，這怎麼會發生在齋藤道三秀龍和義龍父子之間呢？

在日本戰國時期，父子相互攻擊是一件很正常的事情，由於父子可能侍奉不同的大名，而大名之間矛盾不斷，兵戎相見是常事，因而父子之間也會出現思想不和，所以造成了父子成為敵人的局面。而齋藤道三秀龍就對兒子義龍十分仇恨，甚至懷疑義龍不是自己親生的，所以下定決心將親兒子義龍除掉。但憑一己之力難以除掉義龍的兵力，而蜂須賀小六和齋藤道三秀龍是戰略夥伴關係，因此發出聯合作戰的邀請，蜂須賀小六爽快地答應了。

這時的秀吉只是蜂須賀小六下的侍從，在這次作戰任務中秀吉只負責傳話，負責打聽城內的消息。蜂須賀小六和齋藤道三秀龍要攻打的城池是稻葉城，稻葉城就是義龍的根據地，並且要火燒稻葉城。

秀吉接受命令到了稻葉城，站在稻葉城下秀吉獃獃地站著，若有所思。

他知道這次蜂須賀小六這次任務的計劃，也打聽到了齋藤道三秀龍家族內部的矛盾。然而無論多麼大的矛盾，父子相殺這是違背天倫的，蜂須賀小六這樣一個正義的人怎麼會支持齋藤道三秀龍做出這樣的不義之舉呢？難道蜂須賀小六的正義是虛偽的么？秀吉思索著這些，然而始終找不到答案。

望著稻葉城的城牆，上面旌旗招展，隨風飄揚，武士威嚴地把手著這稻葉城。然而稻葉城中的人還不知道即將會有一場災難要發生，稻葉城很快就會淹沒在火海之中。這樣想著秀吉也陷入了深深的嘆息之中。他在想，即便這稻葉城被打了下來，稻葉城也不會遇到明主吧，齋藤道三秀龍這樣一個對親骨肉都敢殘忍殺害的人怎麼會溫柔地對待稻葉城的老百姓呢？齋藤道三秀龍的唯一目的只是想把兒子除殺並且奪取兒子佔領的地盤而已，根本不會考慮到老百姓的感受。

而蜂須賀小六呢？肯定不會成為稻葉城的城主，因為齋藤道三秀龍不會讓給他的，不過肯定會獎賞小六很多寶物錢財。然而一旦齋藤道三秀龍的名聲破壞了，百姓們批判齋藤對兒子的不義之舉，那麼蜂須賀小六自然就成了幫凶，小六的一世英名就全毀了，在小六心中以正義樹立起來的偉大抱負也就灰飛煙滅了。

稻葉城十分秀麗，也十分壯觀，交通非常發達，物產也十分豐富，農業與紡織業也處於鼎盛水平，然而這一切在秀吉眼裡都是虛幻的，這些的一切都會化為烏有。秀吉此時也開始逐漸懷疑蜂須賀小六的為人了，真的像傳說中那麼正義么？齋藤道三秀龍是蜂須賀小六的恩人，兩人交往甚密，因此小六肯定知道齋藤的作為是不義之舉，但是面對朋友的請求怎能不出手相助呢？所以蜂須賀小六在朋友面前無法推脫掉不義之事，所以蜂須賀小六的正義不是絕對的正義，是關乎個人利益的正義。

秀吉也這樣想著小六的為人，突然覺得虛偽的正義也有點噁心，於是他決定日後一定要離開蜂須賀小六，離開這個是非之地，去尋找真正的明主。
