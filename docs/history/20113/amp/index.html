廣東南拳歷史｜清末年間嶺南武術社館的崛起

天天要聞
2019-12-16

嶺南武術的形成和完善，源於嶺南下層民眾的求生慾望和抗暴意識。清末年間，清廷懾外壓內，國內階級矛盾日益尖銳，農民革命運動風起雲湧。

特別是太平軍高舉義旗，建都南京，清政府自顧不暇，對民間練武已疏於防範，而民眾練武之後，進能參加義師，退可強身自衛，故對練武有迫切的要求。

辛亥革命之後，嶺南成為了中國革命活動的中心，孫中山再造中華於粵地，提倡「尚武精神「，主張「強國強種」，一時間全國武林精英彙集嶺南，以振興中華民族為己任的尚武精神勃然興起。

嶺南武術隨之迎來了自己的發展黃金時期，武術社團、館校（包括蔡李佛館、佛山鴻勝館、惠東林家武館等歷史著名拳館）相繼出現。

蔡李佛館的發軔

蔡李佛館成立於清朝道光年二十九年（公元1849年），創建者為蔡李佛拳始祖陳亨。他除了在自己家鄉新會京梅鄉設立祖師堂外，還在廣東各地設立了分館。一年內，分館廣泛散佈於兩廣十縣、三市、二十六鄉，發展神速。

蔡李佛館以繼承和發揚中國傳統武術為宗旨，以「洪英至聖，英雄永勝」為守則——即「奉朱洪武（朱元璋）為聖上，少林英雄的拳棍永遠獲得勝利」之意。它和洪門會的反清復明是一脈相承的，對反清的農民起義持同情態度。

在祖師堂門前，掛有一副對聯「蔡李佛門源自此，少林嫡派是真傳」；各地分館的門聯是：「英棍飛騰龍擺尾，雄拳放出虎昂頭」。

100多年來，蔡李佛拳流行於廣州、佛山、肇慶、番禺、南海各地，風行各省，並遠播香港、澳門、南洋、菲律賓、美國和加拿大一帶，歷久不衰。蔡李佛館源遠流長、根深葉茂，為繼承、發展中國武術作出了卓越貢獻。

佛山鴻勝館的盛衰興替

佛山鴻勝館創建於清咸豐元年（公元1851年），創始人張炎，廣東新會石咀人，為蔡李佛始祖陳亨的及門弟子。

鴻勝館自建立起至1949年結束館務為止，長達98年，是近代廣東較具規模和較有影響力的武館。鴻勝館鼎盛時期（公元1921年左右）設有13間分館，會員達3000多人，招收門徒非常嚴謹。會章規定不收官吏、土豪惡霸和流氓地痞，有正當職業者才能入館。

鴻勝館訂有嚴格的館規：得師（尊師重道）；苦練；得當頭手不善（不得先動手打人）；節色；補食（不暴飲暴食）；雄捱標打力（要學好扎馬和跳躍本領）；敵棋逢來不讓（遇敵堅拼到底）；氣長（練耐力，有耐性）；發響（打拳時呼喝出不同的聲音）；日久多磨推不動（勤學苦練，所向無敵）。

由於館規嚴明，言出必行，所以鴻勝館門徒一般都正直尚義，行為正派，絕少招惹是非。

鴻勝館的不少成員積極投身於廣東近代的革命風暴中，在辛亥革命、歷次國內革命戰爭和抗日戰爭各個時期都作出過貢獻。

由於鴻勝館成員參加了大革命時期革命的工人活動，反動派恨之入骨。1927年4月，當白色恐怖籠罩全國時，鴻勝館也無法倖免，主要成員有的出走，有的被殺。當局嚴禁鴻勝館設館授徒，從1927年起，銷聲匿跡達10年之久。直至1936年，鴻勝館才以鴻勝體育會的身份再度恢復籌建。

惠東林家武館的崛起

林家武館創設於清同治元年（公元1862年），館址在惠東梁化墟。創始人林合，字瑞初，廣東惠陽人，少年時酷愛武術，17歲跟隨福建少林寺避難還俗和尚黃連嬌學拳10年，後在梁化設館授徒。

由於武藝高強，弟子眾多，據傳林合曾任惠州府千總，因不滿清廷的腐敗，毅然棄官辭職，上羅浮山拜廣進禪師為師，苦練3年，技藝精湛。下山後繼續傳藝。為紀念兩位先師，他在堂屋上貼一副對聯云：「先承少林遺妙業，後得華首吐真傳」。

林家武館開創至今，已有120餘年，期間高手輩出，林合之弟林慶元，侄林耀佳、張禮泉、林煥光、林燦光等均為時下名師。

由古至今，武館的開設依然是一代代師祖對武術的執著傳承。武館之所以能發揚光大，除了獨特的技藝傳授外，離不開嚴謹的館規把尊師重道的寶貴精神根植於練武之人心中。師父言傳身教、以身作則，幫助徒弟培養正直、獨立的人格，也一直是武館提倡的師徒相處之道。
