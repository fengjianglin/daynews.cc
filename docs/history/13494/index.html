西晉張華：推動滅吳，國難忠臣，最後為何落了族誅的下場？

2019-12-13 371條評論 2,967次閱讀 851人點贊

自古以來，「好船者必溺，好戰者必亡」，幾乎已成鐵律。凡是鼓吹戰爭者，往往會遭致愛好和平、厭倦戰爭人士的反感。西晉初，朝廷中就有這樣一位「鷹派」，他因極力支持滅吳統一戰略，而遭到包括權臣賈充在內的大臣們的反對。賈充甚至用死亡，對其進行威脅。只是由於晉武帝的極力支持，才幸免於難。滅吳統一大業，才得以付諸實施。晉惠帝時，主庸臣橫，外戚、諸侯，爭權奪利，內亂不已。他雖臨危受命，勉強支撐朝政不紊；但最終還是被捲入權爭的漩渦，並死於權爭。他就是張華。

張華雕像

一、出身孤寒，博聞強識

張華（232年—300年），字茂先。范陽方城（今河北固安）人。他的父親張平曾任漁陽郡太守，也算高門大戶。可惜英年早逝，又因清廉，素無積蓄，家境很快衰敗下去。張華少年時，孤苦貧寒，只好靠幫別人牧羊來謀生，因而對民間疾苦，有所了解。

阮籍畫像

張華雖家境敗落，但他自強不息，博覽群書，勤奮學習。即便圖卦、讖緯、方技之類書籍，也多所涉獵。因而，自少年時代起，張華就以學識淵博、聰敏多才而聞名於世。他尤其擅長撰文，文章辭藻和順華麗。張華還注意修養身心，言行謹慎，舉止合禮。更兼心胸寬闊，膽識弘遠，而備受時人稱譽。「竹林七賢」之一的阮籍在讀到他的文章後，讚不絕口道：「此人真乃王佐之才啊！」由此聲譽日隆。

司馬昭劇照

後來，漁陽太守鮮於嗣推薦他任太常博士。名士盧欽又把他推薦給執掌朝政的司馬昭，轉任河南尹丞。張華後又歷任著作郎、長史、中書郎等職。由於張華對諸多政務的見解，皆卓犖不群，且多有創見，切有成效，大多被採納施行，因此頗受司馬昭賞識。

司馬炎劇照

泰始元年（265年），晉武帝司馬炎以「禪讓」方式，改朝換代，改國號為晉，史稱西晉。張華授任黃門侍郎，隨從左右。以其博聞強識，逐漸受到司馬炎的器重。一次，晉武帝向張華詢問漢朝宮室制度，張華應對如流，且繪圖畫，以佐講解。晉武帝及旁聽者聽得津津有味，都忘了時間，不覺得疲倦。數年後，張華被任命為中書令，職掌中樞決策，頗受晉武帝信任。

二、力主伐吳，統一全國

公元190年，關東眾多州郡長官聯盟，推舉袁紹為盟主，討伐專擅朝政的軍閥董卓。這就是《三國演義》上所說的「十八路諸侯」。董卓敗滅後，各州郡長官，紛紛割據自立，為爭權奪利，混戰不已，東漢帝國已名存實亡。

經過混戰、兼并，至公元208年赤壁之戰後，逐漸形成曹操、劉備、孫權鼎立的局面。後來，隨著曹丕、劉備、孫權相繼稱帝，魏、漢（又稱蜀）、吳三國鼎立局面，正式形成。

公元263年，魏滅蜀漢。公元265年，晉滅魏。至此，三國中，唯有孫吳尚存，形成晉、吳對峙的局面。西晉統治集團中，羊祜最早公開提出滅吳統一的戰略，並為實現這一宏偉戰略，積聚力量。晉武帝雖才能中上，但也有混一寰宇的雄心抱負，對羊祜的策略，頗加讚賞。不過，臣僚中，支持晉武帝、羊祜發動統一戰爭的，可謂寥寥，僅有張華、杜預少數臣僚。大多數臣僚，包括權臣賈充等，都持懷疑，甚至是極力反對的態度。

咸寧四年（278年），羊祜病重。晉武帝派張華去向他徵詢伐吳之計。羊祜對張華分析了迅速伐吳的利弊，張華大加讚賞。因此，羊祜對張華寄予厚望，對他說：「能實現我這個願望的，只有你了啊！」

遊戲中的杜預形象

羊祜去世後，接替他職務的杜預於咸寧五年（279年）上表，請求伐吳。杜預入宮上奏時，晉武帝正與張華下棋。杜預陳奏後，晉武帝似置若罔聞，仍埋頭棋局，不發一言。張華毅然推開棋盤，向晉武帝拱手行禮，勸說道：「陛下聖明英武，國富兵強；吳主孫皓，邪惡兇殘，誅殺賢能。現在伐吳，可不勞而定。希望您不要再猶豫了！」晉武帝這才下定決心，調遣兵將，分路出擊；他命張華為度支尚書，負責策劃及運輸糧草。賈充等雖極力反對，但晉武帝不為所動，還命賈充總統諸軍。

賈充劇照

伐吳諸軍最初進展不利，賈充等趁機大肆鼓噪，建議晉武帝撤兵休戰，以待有利戰機。賈充更奏請武帝，腰斬張華，以謝天下。晉武帝淡淡地說道：「伐吳是朕的主意，張華只是與朕的意見相同罷了。」他下令諸軍，繼續奮進。終於太康元年（280年）滅吳，重新實現了國家的統一。張華因定策功，被封為廣武縣侯。

三、揣錯聖意，遭讒外放

滅吳一統後，張華聲名顯赫。因他博學多識，晉武帝還把撰訂晉史、修改禮制等事務，交付給他。甚至當時晉武帝許多詔誥，都出自張華之手。張華的聲望，中樞一時無人能與之相比，他也因此而遭致許多大臣的嫉恨。

恰在此時，西晉統治集團內部爆發了儲嗣之爭。競爭者有二人：一是司馬炎的胞弟，因司馬師無子，過繼給司馬師為子的齊王司馬攸；一是晉武帝的兒子司馬衷（即後來的晉惠帝）。司馬攸賢明有才，司馬衷白痴之名，朝野皆知。二人並非不分軒輊的敵手，選誰為皇位繼承人，本非難事。但晉武帝私念方熾，致使一干擅長揣摩聖意的小人，紛紛鼓噪選立司馬衷。

遊戲中的晉惠帝形象

表面上雖無所適莫，內心已有定意的晉武帝徵詢張華的意見。坦蕩為國為公的張華，誤以為晉武帝還是當年支持伐吳統一時的善於納諫之君，因而直言不諱地指出：才德兼備，且與陛下至親者，齊王司馬攸也。此論大違晉武帝初衷，對張華的寵信，一落千丈。荀勖等嫉恨張華的權臣窺伺已久，趁機進讒構隙，最終將張華趕出朝廷中樞，外任持節、都督幽州諸軍事，兼任護烏桓校尉。

張華無端被黜，但他為國為民的公心，不曾有絲毫衰減。他到幽州就任後，兢兢業業，恩威並施。深得雜居幽州的各族百姓擁戴、感念。以往歷代都不曾歸附的20多個地方部族政權，都紛紛遣使到朝廷朝貢。邊疆安寧，經濟繁庶。

此時，京中朝臣議論紛紛，大多認為皇帝會讓張華返京重用。甚至有人推測，張華極有可能會被任命為宰相。深得晉武帝寵信，常隨侍左右的馮紞，惱恨張華當年曾不留情面的批評哥哥馮恢，趁與晉武帝議論魏晉軼事時，將張華比擬為當世的鐘會。

鍾會劇照

在選立太子事上對張華仍耿耿不滿的晉武帝，雖詔張華返京，但只是任命他為負責禮儀事務的太常。不久，又以太廟屋樑折斷為由，將其免職。自此，終晉武帝一朝，張華再未被授予任何官職，自然無緣置喙國家事務。

四、力撐危局，盡忠匡輔

永熙元年（290年），晉武帝去世，晉惠帝即位。他雖為皇帝，但朝政實際掌握在外戚楊駿手中。楊駿雖迫於張華威望素著，而任命他為太子少傅，但張華與王戎、和嶠等前朝耆宿，都因德望卓著而遭楊駿猜忌，不得參與朝政。

賈南風劇照

元康元年（291年），皇后賈南風（賈充之女）勾結部分臣僚，發動政變，殺死楊駿，剷除楊氏，控制了朝政。賈南風又唆使楚王司馬瑋，矯詔殺太宰汝南王司馬亮、太保衛瓘等人。叛軍四處擄掠，京師震動。八王之亂由此開始，朝臣驚懼，手足無措。張華向晉惠帝說建議，向眾軍宣布司馬瑋矯詔，解除戒嚴，諸軍自然不會再追隨司馬瑋，叛亂自可平息。晉惠帝採納了張華的建議，司馬瑋所部將士，果然大都離散，司馬瑋兵敗被殺。張華因獻謀之功，被拜為右光祿大夫、侍中、中書監。

賈南風等雖掌控朝政，卻也需要象張華這樣的耆宿名臣裝飾門面。張華雖不齒於惠帝、賈后為人，但出於為國為民的忠心，他毅然接受了任命。在職期間，他盡忠國事，輔佐朝政。對賈后等種種倒行逆施，儘力糾偏彌補。賈后雖凶暴嫉妒，但對張華還有幾分敬畏，也不敢隨心所欲，肆意妄為。因而，儘管惠帝昏弱、賈后殘虐，但在張華、裴頠等人匡扶下，天下仍然基本安定。

司馬懿之子、趙王司馬倫先是諂媚賈后，希冀高位，因遭張華、裴頠等重臣的反對而未果。司馬倫、孫秀將張華視若仇讎。後司馬倫於永康元年（300年）率兵發動政變，廢黜賈后，矯詔殺張華，夷滅三族，終年六十九歲。

司馬倫敗滅後，是否給張華等平反，在朝廷上一直存在爭議和反覆。直至太安二年（303年），晉惠帝才下詔，為張華平反，恢復張華侍中、中書監、司空的官爵，歸還被沒收的財產、印綬、符策，派使者弔祭。

文史君說

張華雖出身孤寒，但他通過勤奮努力，憑藉不凡見識和博學強識，得以躋身高位。使他功成名就的是，他大力支持羊祜首倡的滅吳統一戰略，即便遭到眾臣僚的極力反對，甚至面對權臣賈充等人的死亡威脅，也毫不妥協；當然，他也並非阿諛皇帝之輩，在晉武帝選立繼嗣問題上，他秉持公心，極力主張選立賢明有才的司馬攸，而遭致晉武帝的不滿和疏遠。因若干臣僚讒構，他一度被排擠出中樞，外放地方任職。他並沒自怨自艾，自暴自棄，而是踏踏實實地為國盡責，政績卓越，異族歸心。晉惠帝時，主庸臣橫，朝綱紊亂，張華受命於危難之際，力撐危局，保證了惠帝、賈后時，西晉政局相對穩定。充分展現了一位不計較個人名利得失，一心為國、為公、為民大臣的胸襟和氣魄。張華還愛惜人才，即便出身窮賤者，張華也大力勸勉、薦拔，從不厭倦。

張華在文史領域也造詣深厚。他嫻熟《史記》、《漢書》，與人談論《史》、《漢》，娓娓不倦；他還擅長詩賦，詞藻華麗，備受後人讚譽。他還編纂了中國第一部博物學著作《博物志》，填補了中國無博物類書籍的空白，在中國小說史上，具有極其重要的地位和影響。他雅愛書籍，曾與荀勖等人依西漢劉向《別錄》體例，整理典籍；家中藏書甚多，搬家時，除少數衣物和傢具外，書籍倒裝了30多車；被害時，家無餘財，只有文史書籍滿架盈箱。他還擅長草書、行書，他的草書作品《得書帖》、行書作品《聞時帖》，直至北宋末，仍被收錄、摹寫。
