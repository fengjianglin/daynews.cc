曹操明明很惜才，為什麼要殺死楊修？

2019-12-19 277條評論 1,456次閱讀 477人點贊

三國時期，那是真正的亂世，人們說”亂世出英雄”，確實，在三國時期，有很多比較出名的人，都在自己的平台之上大放異彩，實現自己的宏圖。

其中，楊修，是一個才華出眾的人物，最後的結局確實被曹操殺死。很多人都會開始疑惑，曹操不是很惜才嗎？為什麼要殺死楊修呢？其實，楊修之死還真的是”自找的”。

首先，楊修的家世顯赫，十分愛炫耀智商。三國志記載，楊修生於建安年間，家裡從祖輩開始就是名望之輩。，可以與袁紹的家族相提並論。而楊修自幼基因就好，非常聰慧敏捷，長大後的楊修更是因為卓越的才識，屢次獲得周邊的人推崇，所以才跟隨了當時視為豪傑的曹操。

但是，楊修追隨曹操以後，絲毫不知道收斂，依舊過於驕傲，甚至是經常炫耀自己的智商。

楊修最大的過錯，不是驕傲，而是因為他還曾經擾亂過軍心。事情的起因是這樣的，在曹操與馬超對戰的時候，雙方陷入到了拉鋸戰，攻不進城門，這時候如果撤退，又怕被四川的本地士兵恥笑無能。

曹操正是處於進退兩難的地步。恰逢廚師當天做了一碗雞湯進賬，曹操看到這碗雞湯摻雜一塊雞肋。而夏侯惇前來詢問晚上作戰行動該採用什麼口號比較好的時候，曹操順勢就說了句”雞肋”。隨後，夏侯惇命令大家都把行動稱為”雞肋”。

誰曾想，”自作聰明”的楊修一聽到這個詞，就開始打包收拾行李。夏侯惇就跑過來問楊修，你這要幹嘛啊，作戰前收拾行李？楊修說，曹操之所以會說”雞肋”，全因為現在在這裡作戰就如同雞湯里的雞肋，吃了也沒有味道，丟棄也很可惜，明日很有可能因為現在局勢進退兩難，所以撤回老巢。”

隨後，全軍都知道了這個詞的意圖，引起了一陣騷動，都在打包行李。這話傳到曹操耳朵里的時候，可謂是讓曹操怒氣沖沖，隨即就下令殺死了楊修。

所以說，我們可以毫不客氣的說，楊修之死就是自己”作”的，自作聰明，”禍從口出”，不懂得審時度勢，恃才放曠，不懂得收斂自己的鋒芒，更為重要的是，他不懂得”在適合的時間做適合的事情”。

你認為呢？
