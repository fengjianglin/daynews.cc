明末長達二十年的官印之爭，引發震動西南的大叛亂

2019-11-28 0條評論 17次閱讀 0人點贊

摘要

楊鶴，字修齡，湖廣武陵（今湖南省常德市境內）人.楊鶴這個人雖然不聰明伶俐，但是好歹也是個勤奮努力、刻苦學習的讀書人。在府縣考試中考中秀才，在省市考試中考中舉人，真可謂是一帆風順，暢通無阻，看來在科舉考試，作八股文章上，楊鶴還是很有天賦的。

楊鶴，字修齡，湖廣武陵（今湖南省常德市境內）人.

楊鶴這個人雖然不聰明伶俐，但是好歹也是個勤奮努力、刻苦學習的讀書人。在府縣考試中考中秀才，在省市考試中考中舉人，真可謂是一帆風順，暢通無阻，看來在科舉考試，作八股文章上，楊鶴還是很有天賦的。只不過能夠通過科舉考試的人，也不見得就是什麼年輕才俊、國之精英！

萬曆三十二年（公元1604年），楊鶴在大明王朝中央政府舉辦的全國統一考試中獲得了巨大的成功，一舉考中進士。

萬曆皇帝朱翊鈞對考中進士的讀書人還是很不錯的，二話不說，就封楊鶴為雒南知縣。

朱翊鈞

看來楊鶴的運氣確實還不錯，其他人考中進士，最好也不過是給個編修之類的閑職，而楊鶴一下子就被封為知縣，楊鶴真是「人不可貌相，海水不可斗量」！

此時的楊鶴，前途是美好的，未來是光明的！

萬曆四十年（公元1612年），楊鶴被朱翊鈞提拔為都察院御史，這對楊鶴來說，也算不錯的了，一下子就由地方上的小官變成了中央大員。這對大明的許多官員來說，都是可遇而不可求的事情，然而楊鶴並沒有感到滿足。

楊鶴認為，一個小小的都察院御史不過是自己為官生涯的起點，而內閣、六部、都察院這些部門的最高長官才是自己最後的歸宿。可是楊鶴也知道，要實現自己的理想是需要條件的，有條件的時候，楊鶴自然會好好表現。可是沒有條件的時候，楊鶴也會創造條件，好好表現！

楊鶴居然主動上疏朱翊鈞，請求朱翊鈞讓自己擔任東宮太子朱常洛的老師，給朱常洛講課。

從這一點上看，楊鶴的確是眼光獨到，見解非凡，知道朱常洛現在是沒有什麼權力，也沒有什麼地位，更不用說他身邊的人，可是一旦朱常洛當上了大明最高統治者，那就能夠翻雲覆雨、為所欲為了，

朱常洛身邊的人自然也能夠受到賞識和重用，跟著他吃香的，喝辣的了。這就是所謂的「曲線救國」路線，大明著名的內閣首輔高拱、張居正就是通過這種方式登上大明權利巔峰的。

張居正

大路不平，眾人踩

楊鶴不愧是楊鶴，不但眼光好，而且膽子大，居然在奏疏中直言不諱地說道：不久前，聖上的女兒被皇宮中的太監宮女蹂躪欺負，而公主的夫君也在皇宮中被太監宮女當眾辱罵，公主駙馬前去找聖上告狀，可是卻連聖上的面都見不到，難道這就是明君聖主所為嗎？

楊鶴這番話其實並不是胡說八道、信口開河，而是確有其事。就在楊鶴被朱翊鈞提拔為都察院御史的時候，壽寧公主的夫君冉興讓被掌管宮女的總管梁盈女，太監總管彭進朝等人打得鼻青臉腫、遍體鱗傷。

梁盈女、彭進朝這些宮女太監也並非天不怕地不怕，而是他們知道朱翊鈞整天躲在內宮，過著花天酒地、醉生夢死的生活，根本就沒有時間也沒有精力來管這些雞毛蒜皮的小事情，所以他們才會這麼肆無忌憚、有恃無恐地欺負駙馬，否則就是給他們十個膽子，他們也不敢啊，那可是滿門抄斬、誅滅九族的大罪。

可是壽寧公主也不是打掉牙齒，往肚子里咽，吃了虧就當沒有發生過的女人，她一看自己的丈夫被幾個宮女太監打得鼻青臉腫、遍體鱗傷，當時就像被逼急了的兔子，呲牙瞪眼、翻山躍澗，馬上去找她的父皇萬朱翊鈞為她主持公道。

讓壽寧公主想不到的是，她讓太監通報了三次，朱翊鈞就是不願意見壽寧公主。

壽寧公主

真不知道這是因為那個通報的太監與梁盈女、彭進朝那些宮女太監根本就是蛇鼠一窩、沆瀣一氣，他為了維護梁盈女和彭進朝這些人，並沒有通報朱翊鈞，還是因為朱翊鈞正忙著花天酒地、醉生夢死，根本就沒有閑工夫去理會女兒的那檔子事。

據說駙馬冉興讓聽說此事後，心裡那個不好受，就像是正面挨了泰森一拳似的，當時就甩掉了頭冠，拂袖而去，看來在朱翊鈞當政期間，不管是宮女還是太監，不論是駙馬還是官員，脾氣都是比較大的，性子都是比較倔的。

奇怪的是，朱翊鈞看了楊鶴的奏疏以後，不但沒有恨得牙根兒痒痒，讓楊鶴變成徹底的下崗無業人員，去自食其力，反而提拔楊鶴為欽差大臣，去監督兩淮地區的鹽稅，巡視貴州的少數民族聚居區。本人真搞不明白，朱翊鈞心裡到底是怎麼想的，說他是明君，他又不是明君；說他是昏君，可是他有時候又不像昏君。

楊鶴在兩淮地區監督鹽稅的經歷真的稱得上是平淡無奇，普普通通，本人沒有什麼好描述的，讀者諸君也沒有什麼好看的，可是讀者諸君也不用唉聲嘆氣，畢竟楊鶴巡視貴州的少數民族聚居區的經歷可真的稱得上是驚天泣地、波瀾壯闊。

官印之爭

早在我們的楊鶴到貴州幾十年之前，貴州地區就已經是連年征戰、戰火紛飛了。

我們都知道，雖然大明在雲南、貴州、廣西、四川、湖廣這些地區實行的是土司制度，由少數民族首領管理當地的政治、經濟、軍事、文化事宜，但是這些少數民族首領也必須得到大明的認可以後，才能夠管理，不然就是違規操作。而得到大明認可最好的憑證就是大明皇帝頒發給少數民族首領的官印。

官印

這種事情在我國奴隸社會和封建社會時期簡直是司空見慣、習以為常，比如說漢武帝劉徹元封二年（公元前109年），漢武帝劉徹就賞賜給雲南國王一枚金印，這枚金印就是中國歷史上著名的「滇王之印」。

當然，並不是每一任少數民族首領都能夠得到大明皇帝賞賜的官印，畢竟少數民族首領並不像漢族官員，他們實行的是繼承製度，父親的官職，可以由兒子來繼承。所以大明皇帝只需要給每個地區的少數民族首領賞賜一顆官印就夠了，這樣既可以保證少數民族聚居地區的穩定，又可以省下不少金銀，節省成本，兩全其美，何樂而不為呢？

大明皇帝頒發給少數民族首領的官印，就成為了少數民族聚居區內王公貴族、文武大臣競相追逐的對象。畢竟誰擁有了大明皇帝頒發的官印，誰就是當地實際上的統治者和管理者。高高在上、作威作福的感覺，普天之下，又有幾個人不想體驗一下呢？

烏撒土司安雲龍死後，也不知道是安雲龍沒有子孫後代，還是安雲龍的子孫後代個個都軟弱無能、膽小如鼠，安雲龍的官印居然落到安雲龍族人的手中了。

可是安雲龍族人也不怎麼團結，為了能夠成為烏撒地區的最高統治者，為了能夠過上高高在上、作威作福的富貴生活，他們你爭我斗，血腥廝殺，可是由於他們實力相當，打來打去都沒有打出一個結果。今天安雲龍的官印在這個人手中，明天又到了另一個人手中，就這樣反反覆復，永無止盡。

中國有句古話，叫做「鷸蚌相爭，漁翁得利」，烏撒地區的混亂局面引起了沾益土司安效良的廣泛關注。安效良也是一個吃著碗里的，望著鍋里的，還想著田裡的傢伙，他認為，如果自己能夠將安雲龍那塊官印也搶到手，那麼自己就是烏撒和沾益兩個地區的最高統治者，立萬世之基業，建不世之奇功就不再是夢想。

安效良也是個想得到，就做得出的人，他一想到這裡，就迫不及待地率領大軍闖入了烏撒地區，加入了與烏撒族人爭搶安雲龍官印的隊伍。

安效良把問題想得太簡單了，安雲龍的族人雖然口是心非、貌合神離，為了顆官印就能夠拼得頭破血流、遍體鱗傷，但是在遇到外敵入侵的時候，他們還是能夠放下個人恩怨，重新聚集在一起，團結一致、齊心協力地去對付外敵的。

當安效良剛剛率領軍隊進入烏撒境內的時候，就遭到了烏撒軍隊的頑強抵抗。雙方你爭我斗，你來我往，一打就是三十幾年，烏撒和沾益這兩個地區真的稱得上是屍橫遍野、血流成河、生靈塗炭、民不聊生。

可是最終還是安效良取得了勝利，搶到了安雲龍的官印，當上了烏撒土司，而安效良的父親則佔據沾益地區，扼守住了進入四川、貴州的咽喉之地。如果安效良發動叛亂的話，後果真是不堪設想。

楊鶴的未雨綢繆

幸運的是，楊鶴認識到了這一點，他上疏朱翊鈞，建議朱翊鈞取消烏撒地區的土司制度，將烏撒地區併入貴州，由貴州布政使司直接管理，這樣就可以防止叛亂的發生。

朱翊鈞雖然貴為大明最高統治者，翻雲覆雨，叱吒風雲，但是他除了對花天酒地、醉生夢死這類荒淫的事情比較擅長外，對處理軍政大事根本就是九竅通了八竅——一竅不通。他哪裡知道楊鶴的建議合不合理，可不可用啊，朱翊鈞只好將滿朝文武大臣召集起來，商議對策。

只不過朱翊鈞找錯了人，大明絕大部分官員都是只領工資不幹實事的主，你讓他們鬥鬥嘴，告告人，砸砸黑磚，放放冷箭還可以，讓他們商議國家大事，制定重大決策，討論大政方針，就是給他們幾個月時間，他們也弄不出個結果來。

朱翊鈞一引出議題，滿朝文武大臣馬上分成了兩派，在金殿上大吵大鬧，互相對噴，由於是針尖對麥芒，公說公有理，婆說婆有理，誰也不服誰，幾個月下來，也沒有爭出個結果。

安效良既不是大明文武大臣的朋友，也不是朱翊鈞的兄弟，自然沒有道理等到大明文武大臣討論出結果以後才發動叛亂吧。這就跟我們今天公司的談判差不多，就是要趁對方準備不足，心裡沒底的時候，發動猛烈的攻勢，這樣才能夠取得談判的勝利。

再告訴大家一個秘密，其實追女孩子也是大同小異，萬變不離其宗。要想追到女孩子，就是要等女孩子失戀或者是遇到困難危險的時候，付出自己真實的情感，堅持不懈地追求，一心一意地付出，全力以赴地投入，關懷備至的呵護，這樣才能夠打動女孩子的芳心，贏得美人歸嘛。

所以當滿朝文武大臣正在唇槍舌劍、唾沫橫飛的時候，安效良就發動了叛亂，看來楊鶴還真是個猛人，居然還有未卜先知的本領，真是「人不可貌相，海水不可斗量」！

讓人意想不到的是，楊鶴並沒有因此而受到朱翊鈞的賞識和重用，朱翊鈞對待楊鶴的態度，仍然是不冷不熱、不理不睬，用今天比較流行的話來說，就是朱翊鈞對楊鶴這個傢伙壓根就不感冒。
