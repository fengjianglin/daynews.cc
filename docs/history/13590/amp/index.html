李世民和李建成誰更狠毒，差距太大，玄武門之變後，才看出了答案

天天要聞
2019-12-12

自古帝王都是無情帝，為了自己能夠繼承皇權，不惜手足相殘。我們在歷史書和電視劇中都能看見，比如《琅琊榜》的梁王，為得權力不惜殘害所有人。其實在歷史上這種事情很多，玄武門之變就是其中的一個典型。

玄武門之變大家都很了解，公元626年6月4日，李世民帶著他的將領朝著玄武門走去。李世民這個時候還是秦王，當時玄武門的守衛總領常河在這時候被李世民策反加入到了李世民的隊伍參加這次密謀的行動。

唐朝建立的時候李建成被封為太子，李世民則被封為秦王，因為兩人的職務分配不一樣，太子主政、李世民主戰，所以在功勞上面李世民可能比太子高一點。但是兩人早已明爭暗鬥多年，但是李淵是偏向太子這一邊的，所以在這一點上李建成是處於上風的，因此李建成就成為了李世民的暗殺對象，在玄武門之變結束後，李元吉在混亂中被殺害，李建成則被割下頭顱當場示眾，兄弟三人就在這一天的時間內陰陽相隔。

李建成被斬頭示眾之後，那他的子女又是什麼樣的下場呢？首先我們要知道歷史是勝利者書寫的，唐朝的史書上就說要不是李世民成功奪取玄武門以防李建成謀反，不然哪裡會有盛唐出現，其功勛堪比無法比擬。這樣的說法完全就把李建成給抹黑了，真正的李建成是一個忠厚而且是十分合格的太子，只可惜在這場權位爭奪戰中失敗了。

李世民在這場權位戰中獲勝後做了一件相當無恥的事，因為李世民對李建成的子女和家族做法太過，當時的史官不敢正面記載下來，所以在唐朝的史書上對於李建成子女基本上沒有什麼記載，其實都是無一倖免。

後來唐朝滅亡之後，這件事才被後來的史官記錄下來，因為史書都是勝利者書寫的，在這場仗打響之前其實是李建成的支持者多，這也是李世民為什麼要謀劃這場密謀的關鍵一點。

李世民繼位之後，部分史官故意抹黑李建成讚美李世民，將玄武門斬殺兄弟的事美化，將兄弟子嗣也被殺害的事情掩埋，而且他們死後，家族、子嗣都不能善終，所以他們兩個根本就無力給自己正名。帝王的道路註定是一條孤獨的路，權力的誘惑會讓人喪失理性變得六親不認，所以說一將功成萬古枯，在到達權力的頂端，你會發現在這條路上只有你一人，不會再有親情可言，不然自古的帝王幹嘛都有稱自己為寡人或者是朕呢？帝王家的無情之處就體現於此。
