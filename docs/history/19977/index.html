為何楊秀清死後，石達開、陳玉成、李秀成等驍將就玩不轉了？

2019-12-17 698條評論 1,639次閱讀 665人點贊

1847年，馮雲山被逮捕入獄，洪秀全逃往廣東，紫荊山區拜上帝教徒群龍無首，惶惶不可終日。此時，名不見經傳的燒炭佬楊秀清假託「天父」下凡，向教眾傳達天父皇上帝的旨意，說此事乃天父對拜上帝教的考驗，讓大夥相信只要渡過這一劫難，拜上帝教就可以邁向勝利的天堂。在今天看來，楊秀清這種迷信的說教壓根就是子虛烏有，純屬愚弄大眾感情，只要是明理人肯定不相信。不過，晚清桂平地區的群眾卻相信了，因為這裡一直都有「降童」之習俗，於是教眾便安定下來，表示要堅決擁戴「上帝次子」洪秀全，共創人間天國。

從利用「天父下凡」穩定教眾情緒這件事可看出，楊秀清有著比較強的政治能力，他可以根據地方群眾的實際情況採用相應的宣傳策略。金田起義後，太平軍在楊秀清的指揮下，一路勢如破竹，不到三年時間便打到南京，佔據大清半壁江山。隨後，楊秀清組織北伐、西征、東征，除北伐失敗，其餘兩路連連獲勝。1856年6月，太平軍摧毀清軍江南大營，解除了天京之圍，太平天國也達到了軍事上的全盛時期。很遺憾，由於太平天國內部權力之爭太激烈，楊秀清因人緣關係太差以致在1856年9月的一場政變中最先被幹掉。

楊秀清死於「天京事變」後，洪秀全將石達開從安慶召回天京，讓其主持全國軍政事務，但卻處處排擠，於是石達開憤然外出搞單幹。翼王單飛後，洪秀全只能重新組建「五軍主將」制，提拔陳玉成、李秀成、楊輔清、李世賢等年輕將帥，讓他們承擔起挽救太平天國的重任。在這四位小將中，陳玉成和李秀成的能力最強，在楊秀清時期，他們屢立戰功，如陳玉成克武昌、下鎮江，李秀成則在第一次摧毀江南大營中立下汗馬功勞。不過，自從沒了楊秀清在中央統籌，石達開屢戰屢敗，很少取得像樣的勝利。陳玉成、李秀成雖然能夠第二次摧毀江北大營和江南大營，並且取得「三河鎮大捷」全殲湘軍6000精銳之巨大勝利，但在1861年之後就再也沒有在關鍵性戰役中取得勝利，如安慶保衛戰失敗，雨花台決戰失敗，「進北攻南」計劃流產等等。可以說，沒了楊秀清，石達開、陳玉成、李秀成等驍勇善戰之將領就玩不轉了，其中之原因何在？就「縱橫國史」看來其中之關鍵就在沒了「統一指揮」，沒了可以協調諸將一起行動的具有足夠威望的最高軍事領導人。

說到統一指揮的好處，我們不妨來看下東王楊秀清統一指揮下的兩場影響國運的大規模戰役，即是湖口大戰、天京解圍戰。1854年8月，由於林紹璋指揮不力導致太平軍在湘潭全軍覆沒，湘軍趁勢攻下武昌、田家鎮、半壁山，而後直接順著長江而下，兵臨九江，太平天國西征一階段的成果幾乎喪失殆盡。為此，楊秀清派遣石達開率翼殿將士西征，前往九江布防，阻止湘軍繼續東進，並趁機收復失地。結果大家都知道，石達開在湖口擊敗湘軍，逼得曾國藩兩次投水自殺而未遂，而後太平軍反攻，再次佔領武昌。不過，這主要功勞應該是楊秀清而不是後人所說的石達開，為何呢？道理很簡單，若沒有九江守將林啟榮，梅家洲守將羅大綱，江北秦日綱、陳玉成等人之密切配合，牽制大量湘軍，石達開又怎能取得湖口大捷呢？這些手握重兵的將帥之所願意配合，就在於他們畏懼東王，所以堪願服從楊秀清的調度，若是抗命不遵或者陽奉陰違，必定是軍法處置。

1853年3月，太平軍攻佔南京，並改名為天京。不過，清軍很快也尾隨而至，並在南京孝陵衛組建江南大營，在揚州組建江北大營，從南北兩面將太平天國包圍在中間，意圖困死洪秀全。1856年4月，江南大營清軍攻擊鎮江，守將吳如孝連連告急，楊秀清派燕王秦日綱、檢點陳玉成率四萬精兵前去解圍，並擊敗江蘇巡撫吉爾杭阿。不過，由於秦日綱能力太差，隨後便在金壇吃了大敗仗，清軍趁機反撲，向榮、張國梁等清將連克太平軍營壘和要塞，天京形勢更加不妙。為此，楊秀清只能將遠在江西戰場的翼王石達開調回江南戰場，讓其配合秦日綱作戰，一起進攻江南大營。當時，石達開已經幾乎將江西全境拿下，曾國藩也被自己逼到南昌孤城，只要再加把勁就可以活捉這位湘軍首領。不過，懾於東王的威望，石達開還是乖乖回來，而且是星夜兼程，絲毫不敢怠慢。有了翼殿三萬多精銳的加入，天京太平軍士氣高漲，再次主動進攻江南大營，並將其攻破，天京三年之圍就此解除。

楊秀清死後，由於缺少具有足夠威望和實力的中央領導，太平軍諸王開始離心離德，為了保存實力，他們相互不救援，各自在自己的根據地里自娛自樂。這種脫離集體之做法，對太平天國的影響是致命的，它最終葬送了太平天國的基業。石達開、陳玉成、李秀成在沒有隊友配合之情況下，自然就玩不轉了。例如，李秀成為了銳意經營江浙，對「西征武昌」之計劃絲毫不上心，以致安慶失守，陳玉成被迫退守皖北，最後因叛徒出賣而死。當然，不救援安慶的不只是李秀成，楊輔清、李世賢、黃文金、劉官芳等手握重兵的將領也並未出死力。陳玉成名義上是救援安慶的最高主帥，但卻無法直接指揮這些人的軍隊，所以只能自己孤軍奮戰，叫天天不應，叫地地不靈。後來的雨花台之戰依然如此，若是李秀成具有楊秀清一樣的威望，各路諸侯王怎敢不聽指揮，又怎敢有意識的保存實力，曾國荃區區一萬多疲憊之師又怎能阻止得了幾十萬太平軍。至於石達開，沒了隊友的配合，沒了後方基地的支援，想不敗都難。

總而言之，東王楊秀清不但軍事能力出眾，而且威望高，實力強，可以壓制各路將帥，也可以統一協調軍事行動。沒了楊秀清，石達開、陳玉成、李秀成雖說打仗還可以，但卻因威望和實力有限，根本就無法統一指揮各路大軍，所以他們自然就玩不轉了。所以，統一指揮很重要，強大的領導核心很重要，集體精神更加重要，單幹是萬萬不行的。
