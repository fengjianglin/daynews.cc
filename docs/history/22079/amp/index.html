維也納體系得以維持的根本原因：歐洲協調機制

天天要聞
2019-12-18

按照歷史發展的「慣例」，一般在一場大範圍的戰爭後都會產生一種新的國際體系，這是因為主要力量結構發生了變化，國際體系也就需要適應這種變化。如1618-1648年三十年戰爭後確立的維也納體系、1914-1918年一戰後確立的凡爾賽-華盛頓體系等等。而一種國際體系要能維持下去關鍵在於體系下的主要力量有基本協調機制存在，否則就容易產生戰略誤判，尤其是在大國之間發生衝突時更需要這種機制的存在。為了說明這個問題，小編就以19世紀時期的重要國際體系——維也納體系的運作為例來說明這個論斷。

維也納體系下的歐洲

一、《最後議定書》達成，維也納體系確立的基本標誌

1814年9月以英國為核心的反法同盟國家在經歷6次反法戰爭後於奧地利首都維也納召開戰後歐洲和平會議。這是歐洲歷史上規模空前的「大聚會」，除奧斯曼帝國以外的所有歐洲國家均派代表出席，但這次會議起主要作用的是英、俄、普、奧四大國，四大國都派出了以君主或者重要政治家為核心的代表團，如俄國沙皇亞歷山大、奧地利首相梅特涅。他們各懷詭計，都想爭取拿破崙戰爭后豐富的「利益」。這場會議上大國之間爭論不斷，無法達成共識，比如在波蘭-薩克森問題上四大國爭吵不休，甚至有威脅用戰爭解決的聲音出現。然而突如其來的一場事件打破了這種局面。1815年3月拿破崙再次返回法國掌控大權，大國不得不再次組織第7次反法同盟共同應對。最終徹底擊垮了拿破崙，但是為了防止再次出現危機，大國之間必須儘快結束爭論，達成最終協議。正是在這樣的背景下，各國經過一定的妥協，達成了《維也納會議最後議定書》。

1814-1815維也納會議

這個協議基本上達成了三個大國需要達到的目標：即均勢格局的確立、正統原則的維持和一定的補償。英國需要的是歐洲大陸的均勢格局，這是它的根本戰略目標，也是它參戰的原因；沙俄、普魯士、奧地利等君主國家需要的是消除法國大變革以來的激情浪潮，維護封建正統統治。這兩個目標互相交叉，各國有側重。剩下的就是如何解決具體的「戰爭補償」問題了。按照這種計劃，最後達成了以下基本共識：

法國、西班牙、那不勒斯等國恢復就有的王朝統治模式；成立德意志邦聯，梅特涅出任邦聯議會議長；義大利維持各邦國林立的局面；比利時與荷蘭合併為尼德蘭王國，充當遏製法國的堡壘；瑞典合併挪威；瑞士成為永久中立國；取締黑奴貿易等等。

大國們根據這三個基本原則和具體的安排，構建了歐洲的均勢體系和恢復了受拿破崙戰爭衝擊的歐洲大陸封建正統統治。由此也在歐洲重建了所謂的「和平」，整個歐洲確實進入了相對穩定的階段，維也納體系初步建立起來。但實際上拿破崙戰爭後的歐洲已經不是原來的歐洲了，為了更好的維持這個體系，歐洲大佬們需要進一步合作，共同維護他們辛苦建立的體系。

拿破崙戰爭期間的歐洲

二、大國同盟的形成，維也納體系穩定的重要的基石

維也納體系確立後，有兩件事情是最重要的，也是在會議上大國最關心的兩個大問題。一個是維護歐洲封建君主統治的正統問題。法國的戰爭極大的刺激了歐洲的變革運動，君主統治面臨嚴重的危機。以普魯士，俄國等為首的君主國家需要更加團結的鞏固「正統地位」 這也是維也納體系的一個穩定因素。另外一個問題就是歐洲均勢問題，核心是安全問題。如何才能保證體系下大國主導的和平是一個主要的問題。關鍵就在於如何對待戰後的法國，如何維持均勢。針對這兩個大問題，大國領導們結成了兩個同盟，由此給維也納體系的穩定打下了基石。

神聖同盟，維也納體系下歐洲封建君主正統統治的同盟集團。這個同盟體系成立的根本目的就在於維護傳統的基督教教義和歐洲的封建君主統治秩序。畢竟當時歐洲大陸上大多數國家都是君主制國家，這些正統領袖們是絕對不允許拿破崙戰爭後的歐洲浪潮摧毀了這種封建君主制，無論如何都要保證傳統秩序下的「和平」以及維護維也納體系的穩定和長久。基於此道德和現實利益目的，1815年9月俄國沙皇亞歷山大一世、奧地利皇帝佛蘭茨一世以及普魯士國王威廉三世在法國首都巴黎簽訂《神聖同盟條約》。三國君主表示他們根據基督教教義構建「真正的」友誼關係。這是一個為了維護正統統治和帶有宗教意味的同盟，因此被稱為了「神聖同盟」。後來，除了英國、教皇國和奧斯曼帝國以外歐洲國家均加入其中，一定程度上維護了封建君主制在歐洲的統治，也維護了維也納體系。

1815年神聖同盟

四國同盟，維護歐洲均勢體系的大國軍事同盟。為了維護維也納體系，防止法國的再次崛起破壞歐洲的均勢體系。大國之間達成了一項最初是針對法國，實際上維護均勢的條約，並形成了所謂的「四國同盟」。1815年11月英國、沙皇俄國、奧地利與普魯士歐洲四大國締結了《四國同盟條約》，條約規定了：如若任何一方締約國遭受法國攻擊，其餘各盟國將出兵援助；締約國為了維持安定和繁榮，為了維持歐洲和平（實際上就是維也納體系），定期召開國際會議。通過這兩個同盟，尤其是後面這個四國同盟，歐洲大國全部被納入了同一個體系之下（後面我們會談到法國加入，變成五國同盟）。這樣歐洲歷史上第一次出現了大國定期協商以解決問題的機制，即所謂的「歐洲協調」就正式出現了。它對於維護維也納體系，保障歐洲大國之間的均勢有著極其重要的意義。這是近代以來歐洲歷史的一大進步，也是國際體系一大創新。這種模式影響深遠，不僅是當時，也不僅僅局限於歐洲。

三、「歐洲協調」的運行，維也納體系維持的保障

說到這裡我們已經大概清楚了維也納體系的基本內涵，就是由歐洲大國在戰後通過條約確立，然後組建同盟以此作為維護體系的力量基礎，接著找到一種機制來維護體系下各大力量的均勢。前面兩步我們大概介紹清楚了，接下來就是最關鍵的一步了。前面我們說過，根據《四國同盟條約》，各大國是通過定期舉行會議來調節糾紛和衝突，採取一致行動維護體系穩定的。而這種機制就是所謂的「歐洲協調」。

「歐洲協調」在維也納體系下總共出現了四次，即從1818年到1822年這四年間當時歐洲五大國（英、俄、普、法、奧）舉行四次國際會議來協商各自立場、採取一致行動以此維護歐洲的秩序。這四次會議分別是。

未來的世界五強上演新的歐洲協調，你看出來是那些力量嗎？

1818年亞琛會議。亞琛會議是四國同盟後召開的歐洲協調第一次、也是最重要的一次大國之間的國際多邊會議。會議於9月在德意志城市亞琛舉行，英、法、俄、普、奧五大國參加。主要為解決戰後法國的問題而由法國提議召開的。除法國外的四大國經過協商一致決定就從法國撤軍和賠款問題上達成協定，簽訂了《亞琛條約》解決法國問題。會議後不久四大國共同宣布邀請法國加入「四國同盟」，以共同維護維也納體系。這樣「四國同盟」就變成了「五國同盟」，至此當時歐洲所有大國都納入了歐洲協調機制下共同承擔維護體系的責任，維也納體系也達到了一種表面上看起來最穩定的巔峰狀態。但實際上並沒有這麼簡單，四大國背地裡還是擔心法國會出問題（畢竟法國自從1789年事件後就再也沒有消停過，那個都不敢擔保會不會出問題，心裡都是很虛的），因此秘密的簽訂了對法國的條約，這也反映出實際上的維也納體系和歐洲大國協調機制不穩。

1820年特羅波會議。1820年奧地利的後院（義大利那不勒斯）突然之間就著火了（那不勒斯爆發燒炭黨事件），首相梅特涅非常的驚慌，十分害怕會波及自家的君主制度，由此他召集五大國會議，共同商討對策。在會議上分歧很明顯的出來了，俄普奧三個君主制國家達成協議，主張要把那些通過法國式道路改變政權的國家排除在歐洲協調之外，大國應該干預甚至使用武力來解決這類事件。但是英法這邊卻持保守意見，不同意他們哥仨這種暴力的方式，也不認為共同採取措施干涉是恰當的，只想讓奧地利單獨行動。五大國之間產生了嚴重的分歧，會議沒有太多成果。

奧地利首相梅特涅

1821年萊巴赫會議。這次會議是上一次歐洲大國協調會議的繼續。同樣英國與俄普奧三國分歧巨大，堅決反對他們集體干涉那不勒斯事件。最終奧地利在俄普的支持下出兵解決了。但幾大國之間的矛盾和分歧也更大了。

1822年維羅納會議。就在義大利和希臘等地發生法國1789年式事件時，法國的後院西班牙也同樣起火了。為此五大國在義大利維羅納召開第四次歐洲協調會議，商討解決問題。會上英國和其他大國站在了對立面（英國一直以來都反對干涉其他國家事務，奉行孤立主義，認為這樣會導致其他國家干涉自己事務和破壞歐洲均勢）反對法國等要求出兵干涉西班牙事務的決定。而且在其他大國中俄國和奧地利也有自己的打算，都想乘機擴大自己想要的勢力範圍。這些問題的出現使得歐洲協調非常不「協調」幾乎到了崩潰的邊緣。特別是在此過程中美國插了進來。由於西班牙在拉丁美洲有殖民地，美國擔心歐洲大國的干涉會觸及自己在拉丁美洲的利益，由此1823年美國總統門羅提出了「門羅主義」，認為「美洲是美洲人的美洲，歐洲大國無權干涉」。在這種情況下事情變得更加複雜了。就這個問題，四大國與英國造成了不可彌補的分裂，英國在這場國際會議上外交失敗了（法國最終出兵干涉了西班牙問題），由此歐洲協調的大國會議有名無實了。維也納體系的根基被動搖了，變得及其不穩定，一旦出現重大問題，大國無法及時協調，危機隨時出現。

美國總統詹姆斯.門羅

四、大國衝突的發生，維也納體系崩潰的最後標誌

歐洲協調機制不存，體系不穩定，問題越來越多，局勢越來越複雜。雖然協調機制不在了，但是各大國還是基本按照了維也納體系的基本原則行事，不過穩定的機制不在，未知因素就更多了。1830年法國出現大變，波旁王朝被推翻。緊接著在1848年一系列席捲全歐的法國式事件接連出現。1848年1月的義大利一月事件，法國接著爆發二月事件，接下來維也納三月事件，柏林三月事件相繼發生，整個歐洲不得安寧。尤其是1852年法國又恢復了帝制（法蘭西第二帝國）。維也納體系在這一系列大事件的衝擊下變得風雨飄渺。更嚴重的是體系下的各大國沒有了協調機制，要麼各自張望，要麼各干各的，根本無法解決這系列的問題，維也納體系實際上已經名存實亡了。

最終結束這個體系的是大國之間的衝突—克里米亞戰爭的爆發。大國之間本來就存在著矛盾，以前有歐洲協調機制的時候還能坐下來聊聊，消消氣，有個對話的平台和機制。但是當這種機制不在，大國利益無法得到協調時，只要發生直接的利益衝突，很快就可以摧毀這個體系。而這個事件就是克里米亞戰爭。

克里米亞戰爭，維也納體系的崩潰

1853—1856年的克里米亞戰爭是英國和俄國在「東方問題」上利益不可調和而引發大國之間的戰爭。（具體的原因，過程結果和我們今天要說明的主題不大相關，由於篇幅有限就不過多論述了，想要了解的可以在頭條上查看相關作者的文章）。這場戰爭包括了維也納體系下的三大國英法與俄國，這一事件表明大國們不僅沒有了協調機制，還由盟友直接轉變為敵人，還由此引發戰爭。由於英俄都是維也納體系下的主要力量支撐，還是歐洲大陸的兩大側翼穩定大國，他們由盟友轉變為敵人最終表明，維也納體系徹底灰飛煙滅了，再也不復存在，歐洲又將進入新的「戰爭時代」。

最後做一點總結。維也納體系從確立到徹底崩潰僅僅經歷了不到四十年，成為迄今為止第二「短命」的國際體系。其根本原因在於歐洲幾大國沒能將原來確立的同盟體系和歐洲協調機制維持下去，最終在利益發生衝突時失去了協調和妥協的機會，接著就是戰略誤判，最終導致大國之間的戰爭以至於體系終結。歷史的教訓深刻！
