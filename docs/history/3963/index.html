商紂王作為一代人王，實力到底有多強？自殺之前仍能震散索命鬼魂

其實在上古時期，人族是有人王一說的，人王其實算是凡界之主，受天道所庇護；人族最鼎盛時期的三位帝王（軒轅、伏羲、神農）因對凡人界有功而永受天道氣運護佑，甚至在每年三月十五日時，作為天道聖人的女媧娘娘都要去火雲洞朝賀；這或許是因為天皇伏羲乃是女媧娘娘的哥哥的緣故，但這同樣也表明人王的強大。

而商紂王是最後一代人王，為何這麼說呢？因為在商朝滅亡之後，周朝及之後的帝王都是自稱「天子」，在與「天」的輩分上就差了一輩；也就是自此之後，人王氣運就此消失。那人王氣運到底有多強呢？這個就要看看商紂王對人王氣運的運用了，這也是周武王及之後的帝王們羨慕不來的。

商紂王個人的勇武那是不必多說，一個人單挑多方諸侯猛將仍能不落下風；但這並非商紂王對人王能力的展示，其實商紂王對人王之力運用最徹底的一次是在商紂王敗亡之前；當時商紂王見周營大軍攻入了朝歌城，商紂王心中湧起無力悲涼之感，於是就向摘星樓走去。

就在快到摘星樓的時候，忽然颳起了一陣怪風就地滾來，將商紂王罩住，一股透膽的寒意襲來。當初在摘星樓下，無數被商紂王所殘殺的冤魂惡鬼們一起向商紂王撲來，扯住商紂王大呼：「還吾命來！」又見趙啟、梅伯赤身大叫：「昏君！你一般也有 今日敗亡之時！」

但商紂王忽的把雙目一睜，陽氣沖頂而出，將一眾陰魂撲散，那些屈魂怨鬼隱然而退。商紂王把袍袖一抖，上了頭一層樓，又見姜娘娘（姜皇后）一把扯住商紂王，大罵道：「無道昏君，誅妻殺子，絕滅彝倫，今日你將社稷斷送，將何面目見先王於泉壤也！」

就在姜娘娘正扯住商紂王不放，又見黃娘娘一身血污，腥氣逼人，也上前扯住，大呼道：「昏君摔我下樓，跌吾粉骨碎身，此心何忍！真殘忍刻薄之徒！今日罪盈惡滿，天地必誅！」商紂王被這兩個冤魂纏得如痴如醉一般，又見賈夫人也上前大罵道：「昏君受辛！你君欺臣妻，吾為守貞立節，墜樓而死，沉冤莫白。今日方能泄我恨也！」

說完，賈夫人對著商紂王一掌劈面打來，就在這時，商紂王忽然一點真靈驚醒，把雙目一睜，衝出陽神，那陰魂不敢再靠近，於是便隱隱消散了。其實這便是商紂王對人王氣運的運用的一種方式，他只要雙目一睜，衝出陽神，諸邪便不敢再靠近，可以說，這人王氣運是對邪物是有鎮壓作用的。

從這裡看來，這人王氣運可不是一般的有作用了；從這裡聯想下，當初女媧娘娘想除掉商紂王時，為何在見到商紂王時，轉身就離開了，因為她知道商紂王有人王氣運的庇護，如果她強行下手的話，受到傷害的必然是她，雖然她是天道聖人，但也無法避免天道的反噬。可惜在商紂王被封為「天喜星」後，人王氣運從此也被「封印」了。
