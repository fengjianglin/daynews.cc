清朝八旗恥辱之戰：18位戰區司令陣亡14人，主帥一路狂奔

2019-12-17 880條評論 3,495次閱讀 1116人點贊

自從努爾哈赤在東北深山老林里宣布「七大恨」反對明朝以來，八旗軍隊可謂是一路高歌猛進，攻明朝、降蒙古、征朝鮮，無往不勝。1618年，努爾哈赤在赫圖阿拉附近的薩爾滸擊敗明、蒙、朝聯軍四萬餘人，奪取對明朝戰略主動權；1626年，努爾哈赤相繼佔領遼陽、瀋陽、廣寧，雄視山海關；1642年，皇太極在松錦會戰中取得決定性勝利，為入關奠定基礎。入關之後，滿洲八旗日行千里，李自成闖軍、原明朝軍隊無不望風而降，幾乎毫無招架之力。正因為如此，時人都說滿洲八旗驍勇善戰，是東亞第一雄師，有「滿萬不可戰」之說。

不過，這一支驍勇善戰之部隊在雍正九年卻遭遇了一場大潰敗，一萬多清軍被蒙古騎兵追著打；戰區司令員（都統級別）18人，陣亡、自殺14人，清軍幾乎全軍覆沒；號稱華夏第一名將的傅爾丹一路狂奔，僅帶3將逃回科布多城。這便是讓滿洲八旗倍感恥辱的「和通泊之戰」，「滿萬不可戰」之說成為歷史；雍正懾於蒙古之威力，只好暫時罷兵言和，將解決准格爾汗國的重任交給兒子乾隆。

1644年5月，多爾袞率10滿洲、蒙古、漢八旗共10餘萬大軍從山海關一路南下，將北京據為己有。同年9月，順治從盛京（瀋陽）移駕北京，祭祀天壇，宣布對全國進行統治。與此同時，遠在西北的維特拉蒙古准格爾部、青海和碩特部迅速崛起，並成為西域、中亞一帶新霸主。在半人半仙噶爾丹領導下，准爾汗國實力達到歷史之最，西攻吉爾吉斯、哈薩克，北御俄羅斯，東攻喀爾喀蒙古追擊土謝圖汗和哲布尊丹巴，前鋒距離北京只有700里，康熙下令戒嚴。為了打擊桀驁不馴的噶爾丹，康熙三次御駕親征，在烏蘭布通、昭莫多之戰中大獲全勝，但並沒有將准格爾打殘。四爺雍正即位後，趁著准爾格內部分裂之際，大舉出兵討伐，西路巴里坤岳鍾琪兵團、北路科布多傅爾丹兵團，總計7萬餘人，聲勢十分浩大。不過，在隨後的大戰中，清軍卻輸得一塌糊塗，差點連主帥都掛掉。

噶爾丹被康熙擊敗後，准格爾汗國發生分裂，其侄兒策妄阿拉布坦在激烈的競爭中最終勝出，成為新一代可汗。策妄阿拉布坦死後，兒子噶爾丹策零繼位，繼續奉行對清抵抗政策。准爾格面積雖大，但人口卻很少，估計70萬，精銳騎兵六萬多人，這是極盛時期，到策零即位時，總兵力4萬。相對於擁有近兩億人口的大清帝國而言，准格爾的人口、經濟、軍事總體量未免太過弱小。不過，准爾格汗國地處西域、中亞這戰爭頻仍之地，軍事技術、武器水平多效仿土耳其、俄羅斯以及歐洲。其裝備土耳其重型火炮、中亞重型火繩槍，無論質量、射程、射速，還是精確度，要遠遠高於同時期的大清八旗、綠營列裝的鳥槍、明末火銃、威遠炮、子母炮等早期熱兵器。此外，准格爾勇士騎射水平高，這也是常年駐守在大都市的八旗所無法比擬的，尤其是墮落異常嚴重的京師八旗，騎射水平已經不復當年入關之前的滿洲兵。

對於清軍的劣勢，雍正皇帝自然是知道，若是直接搞野戰，八旗會吃大虧，連招架機會都沒有。為此，雍正決定以國力優勢，從西路、北路出兵，採取穩紮穩打之戰術，一邊築城，一邊推進，慢慢蠶食准爾格汗國。當時，岳飛後裔岳鍾琪擔任西路軍統帥，屯駐巴里坤，目標是破壞烏魯木齊農田、牧場，摧毀准爾格汗國賴以作戰的軍備物資，這是清軍主要之戰略目標。華夏第一名將，曾經突破阿爾泰山口，一直打到准格爾腹地然後全軍而還的傅爾丹為北路統帥，麾下有滿洲八旗、蒙古八旗、科爾沁蒙古兵、東北索倫、達斡爾騎兵，總計20000人，屯駐在新修築的科布多城。按照計劃，岳鍾琪西路軍為主力，直搗烏魯木齊；傅爾丹北路為牽制，吸引敵軍注意力達到分兵效果，為岳鍾琪在夏季突襲烏魯木齊做準備。

理想很美好，但現實很殘酷，本該作為牽制部隊的北路軍卻活生生成了當之無愧的主力，以致釀成災難。1730年冬，准格爾名將小策凌敦多布採取聲東擊西戰術，突襲西路軍糧草、馬匹、物資供應基地科舍圖牧場，幾千清軍全部戰死，馬匹12萬餘匹被掠奪，清軍元氣大傷。此戰後，大將岳鍾琪便對準爾格產生畏懼心理，當2000老弱准格爾兵圍攻吐魯番時，他沒有救援；後出兵時，又被對方牽著鼻子走，士兵、馬匹疲憊不堪，已經不具備在夏季進攻烏魯木齊的能力。此時，主帥傅爾丹正在科布多築城，接到朝廷命令後，他留下6000千綠營兵繼續築城，1300名滿洲兵則承擔起保衛築城部隊、保障糧道暢通之任務；自己則親率定壽、岱豪、查弼納、巴賽、達福、馬爾薩、舒楞額等17位大將，13000餘精銳之師出征，再次突襲阿爾泰。

1731年6月9日，傅爾丹誓師出征，素圖、岱豪為前鋒，定壽等領第一隊，馬爾薩等領第二隊，自己則親率主力在後。大軍出發後，傅爾丹前鋒部隊抓獲幾名准格爾傳令兵，從中得知策零正在阿爾泰集結大軍，共三萬，但大軍還沒有完全集結完畢。此外，俘虜還告知，此時的准格爾正在爆發內訌，策零的女婿帶著一萬多人叛逃；哈薩克、吉爾吉斯等又出兵進犯，策零很是被動。此時，傅爾丹覺得這是一次千載難逢之機會，於是便讓大軍加快速度，直接殺向阿爾泰，消滅還未集結完畢的准格爾兵。一切都是那麼順利，路上所遭遇的蒙古兵均不堪一擊，清軍所向披靡，前鋒素圖、岱豪3000人精銳抵達博克托嶺，不久，傅爾丹率大軍趕到，全師合兵一處。不過，傅爾丹此時覺察到了異樣氣息，自己的大軍貌似已經陷入敵人重兵包圍之中，自己中了對手「誘敵深入」之計。

為了避免陷入重圍以致全軍潰敗，傅爾丹下令撤軍，全軍往和通泊方向回營。這是一條死亡路線，因為蒙古騎兵的速度實在太快了，小策零敦多布的虎狼之師很快就追上了傅爾丹，並利用火炮、重型火槍轟擊清軍陣地，索倫、達斡爾、科爾沁蒙古兵等先頭部隊先後被全殲，所部主帥自殺或戰死。不久，准爾格蒙古兵攻擊傅爾丹大營，4000千滿洲八旗損傷過半，清軍即刻潰敗，傅爾丹一路向科布多方向狂奔。准格爾兵緊追不捨，清軍連戰連敗，最後僅有2000殘兵以及包括傅爾丹在內的4位將領僥倖逃回城內。其餘14位副都統（戰區司令），以及七千多精銳滿洲八旗、蒙古八旗、索倫、達斡爾騎兵全部戰死或自殺，清軍付出了慘重代價。此為史上著名的「和通泊之戰」，八旗最大恥辱，雍正欲哭無淚。

此次戰敗後，雍正無力再次全面發動對準爾格戰爭，而是將這一歷史重任交給兒子乾隆，最終成就了乾隆「十全武功」之首的「平定準格爾之戰」。
