嘉慶帝在家門口遇刺，上百侍衛冷眼旁觀，竟無一人上前阻攔

天天要聞
2019-12-04

摘要

有一種說法，雍正皇帝是被呂四娘刺殺身亡的。原因是呂四娘要為因文字獄被害的父親報仇，所以她假冒宮女侍寢，趁機砍下了雍正的腦袋。以至於雍正入殮時，只得用黃金打造了一個腦袋裝上。這個故事看似有板有眼，但其實是小說的杜撰。

有一種說法，雍正皇帝是被呂四娘刺殺身亡的。原因是呂四娘要為因文字獄被害的父親報仇，所以她假冒宮女侍寢，趁機砍下了雍正的腦袋。以至於雍正入殮時，只得用黃金打造了一個腦袋裝上。這個故事看似有板有眼，但其實是小說的杜撰。清朝確實有一位皇帝，他一生勤勤懇懇，但碌碌無為，經常遭人暗殺，十分倒霉——他就是嘉慶。

（呂四娘刺殺雍正，圖片來源於網路）

提到嘉慶帝，大家幾乎都沒什麼印象，「平庸」就是歷史給他的評價。在他之前的康雍乾盛世太輝煌了，到了他這裡，只知道在乾隆駕崩之後，他除掉了和珅。其實，到乾隆晚年，社會各種問題日益突出。清朝社會的固有矛盾已經積累了180年，帝國由盛轉衰，民怨四起，農民起義在全國範圍內爆發。嘉慶從父親手裡接過來的，不是一盤好棋，其實，從那以後，清朝這盤棋怎麼也下不好了。

嘉慶八年（1803年）的一天，看似一個平常的日子，嘉慶皇帝從圓明園回宮，到紫禁城北神武門的時候，一個手持利刃的漢子突然衝出來，向御輦衝過去。神武門左右上百名侍衛一時間呆若木雞，沒有一個人上前制止。關鍵時刻還得靠親戚，這時幸好有嘉慶皇帝的姐夫固倫額駙拉旺多爾濟上前將刺客抱住，侄子定親王綿恩和其他幾名回過神來的侍衛撲向刺客，將其擒獲。

（神武門，圖片來源於網路）

經審問得知，刺客名叫陳德，45歲，鑲黃旗人。31歲時到北京來謀生，之前在內務府當過五年廚子，所以對出入皇宮的路線比較熟悉。幾年前他妻子病故，現在岳母癱瘓，還有兩個幼子要撫養，生活的變故使他經常借酒澆愁。因不久前因經常醉酒鬧事被原來的主家辭退，因此窮困潦倒，生活難以為繼，便想到要自殺。他又覺得不能默默的自殺，應該在死前做一件轟轟烈烈、天下盡知的事情，那就是刺殺皇帝。

確實，在那個年代，能刺殺皇帝真的是一件天大的事情，對於一個底層的百姓來說，想都不敢想。這個陳德不但想了，而且還幹了，那天他與皇帝近在咫尺。陳德知道自己的力量太小，他本來的計劃是手持小刀沖向皇帝，驚駕之時就會被侍衛制住亂刀砍死，圖個痛快，也死個明白。沒想到他出現後，所有護軍侍衛竟沒有一人阻攔，讓他能夠與皇親國戚近身搏鬥，還砍傷了其中一個。他也沒想到傳說中的皇家護衛竟然這樣不堪一擊，真是太諷刺了。

（圖片來源於網路）

表面上陳德只是一個窮困潦倒的廚子，一個走投無路的普通人，但他的供詞太荒唐，整件事情也很不正常，疑點太多。

  1. 陳德說，刺殺成功後就能做官。陳德在審問過程中說過：「若事成，則公等所坐之處，即我坐處也。」刺殺皇帝可是大罪，如果他成功了，其他的人也會將他擒住，他哪裡會有命活，怎麼還會當官呢？誰許他：「若事成」，就能做官的？
  2. 一個廚子竟能探得皇帝行蹤。無論哪朝哪代，皇帝的行蹤都是機密，一個普通人是怎麼得到皇帝回宮的消息的？況且神武門是皇帝回宮的必經之路，戒備極其森嚴，他一個普通人是怎麼攜帶武器埋伏在附近的？
  3. 皇帝遇刺，侍衛竟無動於衷。陳德動手之時，為什麼上百名護衛都面無表情，眼睜睜看著皇帝身處危險之中卻不行動？可能有人說，這些侍衛大多是八旗子弟，平時養尊處優，誰也不想為皇帝拚命。但是這可是個立功的好機會，大家動作這麼整齊劃一的背後，是不是有人操縱呢？

（圖片來源於網路）

經過數次審問，用盡了各種嚴酷刑法，陳德依舊重複那些沒人相信的口供。究竟是紫禁城的安保工作真的一無是處，還是有其他隱情呢？

皇帝看似高高在上，其實也不好當，雖然說君心難測，但是皇帝也猜不到官員在想什麼。就如這次嘉慶在家門口被人刺殺，有一種說法就是，當時嘉慶皇帝在整頓吏治，有一些官員心懷不滿設計了這次刺殺行動。

按照電視劇的套路，皇帝如果被刺殺，最終會牽連數千人，無論是兇手還是相關人員都會獲罪被殺，像這次事件中的侍衛，就應該嚴厲懲處，免得以後有類似事情發生。明朝嘉靖皇帝因為對宮女太過嚴苛，被宮女刺殺，事後宮女誅殺，滅族。與此事有關聯的宮妃也被誅，之後嘉靖朝再也沒出現過類似事件。

嘉慶皇帝是怎麼處理的呢？兩個月後嘉慶下旨，將陳德凌遲處死，為斬草除根，其二子被絞死。而那些不盡責的護軍統領只是被發配，這樣寬鬆處罰並不能起到作用，只能顯示出皇帝的軟弱，後來嘉慶朝又出現了幾次刺殺皇帝事件，甚至有天理教徒攻進皇宮的時間，令人咋舌。

（嘉慶皇帝，圖片來源於網路）

從這件事也可以看出當時的社會積弊。

  1. 社會經濟狀況惡化，普通百姓生活艱難。無論陳德是否被人指使，他冒著全家被殺的風險去刺殺皇帝，很大部分原因是當時的普通百姓生活困頓，朝不保夕，生活得沒有尊嚴。
  2. 階級矛盾嚴重。當時各級官吏腐敗嚴重，大肆魚肉百姓，不顧百姓死活，貧富兩極化嚴重，激化了社會矛盾。
  3. 官府作風懶散，社會治安惡劣。皇帝的安全都不能保證，在家門口被人持刀行刺，侍衛袖手旁觀，對皇帝的生命漠不關心，還怎麼指望他們對其他的事情盡心儘力呢？

嘉慶雖然知道當時社會弊病嚴重，決心改革，卻沒能從制度上尋找問題的根源。平庸不是他個人問題，從乾隆末年開始，社會問題已經開始顯現，嘉慶的子孫也沒能為困頓的國家找到一條出路。有一天，市井小民走投無路持刀殺向皇帝，清帝國也就只能沿著衰敗的道路走下去。
