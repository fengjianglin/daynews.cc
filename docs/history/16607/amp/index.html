甲午戰爭中葉志超真的是不戰而逃？國內貶損國外卻讚歎有加

天天要聞
2019-12-15

甲午戰爭中葉志超的表現，在國內歷史書上的定位就是：怯戰、貪生怕死，被貼上了不戰而逃的標籤。

但被李鴻章認為是「優智略」的葉志超，真的是怯戰、貪生怕死？如何客觀、理性看待葉志超在甲午戰爭中的表現呢？

又或者，葉志超在甲午戰爭中表現出戰爭策略了嗎？

平壤之戰時日軍進攻路線

葉志超在甲午戰爭中的表現

1、 豐島海戰後，從牙山移軍到公州；

2、 成歡之戰後，放棄公州繞道集中守平壤；

3、 以成歡之戰不勝不敗為由，邀功嘉獎，獲得2萬白銀犒軍；

4、 到平壤後，葉志超天天擺宴席，修復平壤城牆，增加火炮、壁壘等防禦工事；

5、 平壤之戰初階以擊退日軍奏捷報；

6、 在東、南二路軍失利下，集中兵力守平壤；

7、 在日軍主力到達時，意圖棄守平壤但被左寶貴阻止；

8、 左寶貴戰死後，提出投降以率領清軍完整回國；

9、 在日軍尚未答覆下，暗中率軍出城回國，但因日軍的攻擊而兵潰，最終清軍無法成建制回國，丟棄軍械；

10、 棄守安州、定州，直接回國防守鴨綠江。

平壤之戰中的情節

解讀葉志超的行為

從牙山轉移到公州，是葉志超接受聶士成的建議。在當時葉志超有兩個選擇：

1、 袁世凱建議增兵仁川——但仁川無險可守，北洋水師對日本聯合艦隊並無優勢。在北洋水師無法確保遏制日本聯合艦隊的情況下，仁川雖然位置重要，但無力據守也是無用。

2、 聶士成建議駐守公州。公州在牙山的東南方向，雖然有河流直通黃海，且比牙山有險可守，但水師援助路線較遠。

最後葉志超選擇平壤。平壤的優勢有：大同江出入黃海，有利於北洋水師援助；平壤是朝鮮舊都城，人口、經濟和糧草儲備等資源比公州好太多；平壤東、南兩面被大同江阻隔，北面是山脈，只需要防守地勢平緩的西南面，也是易守難攻的地方。

一馬平川的仁川，城小人少路遠的公州，都比不上葉志超選擇的平壤。

葉志超放棄公州集中兵力防守平壤，實現了：

1、 集中兵力，改變敵眾我寡的戰爭局面；

2、 有時間部署平壤城防。

平壤之戰清軍之所以只夠在初期擊敗日軍，原因就在於清軍具有人數優勢和相對完備的防禦工事。

在東、南二路戰事不利時，葉志超撤軍回平壤，是為了集中兵力，意圖繼續保持清軍在人數上優勢；在日軍主力到達平壤，清軍無法保持人數優勢，葉志超因此提出要撤軍回國，不打沒有勝算的戰爭。

即葉志超在甲午戰爭中的數次棄守，根本核心是意圖保持清軍對日軍有數量優勢：

集中兵力守平壤，撤東、南二軍回平壤，日軍佔據人數優勢時意圖主動戰略性撤退。

在左寶貴戰死，清軍丟失牡丹台和玄武門時，葉志超主動提出投降，意圖讓出平壤，將清軍完整帶回鴨綠江回防、保存實力。

葉志超集中兵力、保持人員優勢的策略是成功的，因為事實是——平壤之戰是甲午戰爭陸戰中清軍擊殺日軍人數最多的戰役。

成歡之戰日軍死34人、傷54人；平壤之戰日軍死102人、傷433人；九連城之戰日軍死4人、傷140人；旅順口之戰日軍死29人、傷233人。

戰果是實實在在的事實。相比九連城之戰、旅順口之戰、田台庄之戰等，葉志超時刻實施集中兵力保持對敵人員優勢的策略，才取得甲午戰爭陸戰最大的戰果。

至於葉志超謊稱捷報問題，需要從兩個方面去看：

1、 將平手說成勝利、將小勝說成大捷，是清朝當時的特色和慣例，並非只有葉志超一個人這樣做；太平天國運動、陝甘回亂、捻亂等的平叛，清軍將領誇大戰功的事例並不少見。

2、 奏捷報、獲取獎勵，有利於提升士卒的作戰士氣，尤其是在國外作戰的時期。沒有獎勵犒軍，士兵更加不會積極作戰——國外作戰，並非保家衛國。

葉志超天天擺酒問題——目的很可能就是增加和左寶貴、衛汝貴、馬玉昆、豐升阿、聶士成等來自各地將領和平壤守軍的人際關係。

葉志超是臨時成為朝鮮軍統帥，未必能讓左寶貴、衛汝貴、馬玉昆、豐升阿、聶士成心服口服的聽從他的命令。

想要改善人際關係、有利於指揮，葉志超擺酒請客也是合理的。就是現在中國社會有很流行請客吃飯，工作都是在酒桌上談的。大家應當了解到葉志超擺酒請客，未必只是為了自己享受。

放棄安州、定州，直接回國防守鴨綠江也很好理解。平壤都守不住，清軍有什麼能力守安州、定州？與其將精力放在這些註定失敗的防守，不如集中精力防禦鴨綠江，多建防禦工事，增派援兵和軍需。

平壤之戰日軍木版畫

總結

1、 甲午戰爭中葉志超實行保持優勢兵力對抗日軍策略。葉志超放棄公州守平壤，在東南二路軍不利時撤軍守平壤，在日本主力到達時意圖棄守平壤，左寶貴戰死後意圖通過投降讓出平壤城完整率軍回國等，都是葉志超保持優勢兵力、保存實力的體現。

2、 平壤之戰是一場優秀的防禦戰。葉志超集中清軍，保持優勢兵力，積極建設炮台、堡壘等防禦工事，是清軍能夠戰爭初期擊敗日軍的重要原因。

3、 謊稱捷報是當時清朝將領慣例和提升軍隊士氣的需要。

4、 葉志超擺酒只是為了改善人際關係，有利於指揮。

簡單說，如果我們要求葉志超在甲午戰爭中擊敗日軍，也是巧婦難為無米之炊；或者葉志超和左寶貴一樣戰死在平壤，對甲午戰爭中清軍的敗局又能有什麼影響呢？

對於葉志超而言，這是一道選擇題：

是保存實力的以退為進，還是毫無意義的白白犧牲？
