為什麼太子捨不得盧尚書走？他對太子有何影響

2019-12-11 899條評論 3,326次閱讀 144人點贊

電視的第二個故事就是盧尚書作為主考官監考，險些被害的事情，太子為了防止老師和顧逢恩被害，提前布局收買了許昌平，結果本以為大獲全勝了，但是沒想到許昌平最後反口，在皇上面前供出了太子，老師在教育他的時候說：「你仿臣筆記，幾能亂真，除了殿下沒有別人，你隨臣學習書道15年，一切文藝，不可為陰謀所用，一旦沾染，精神全無，骨氣全無，人亦如此啊！」最後老師問他，太子你許他事什麼了？太子說：「如果李柏洲不滋事，他就可以先知道試題登科，如果李柏洲滋事，就讓他做東宮僚屬」。太子認為那個人是個聰明人，老師卻說：「恫之以權勢，誘之以名利，這是儲君應該做的事情嗎？君王的德行，就是邦國的基石，儲君品行端正，國家就會安定儲君內心動搖，國祚就會傾斜，一個國家，連儲君都要使詐偽之術了，這個國家就會傾覆，子民就會流血呀」！太子說：「我也想當一個純潔無瑕的赤子， 我也想安心當這個國家的禮器，我也想像書上說的一樣，臣行君義，子孝父慈，兄友弟恭，可是，實在是太難了！」尤其是這句太難了，真的是演技炸裂，就像我們這些社畜也經常說：我太難了！

太子剛坐下，老師又說：「是誰告訴你，這世上有簡單易行的道路的？」太子立馬又站起來，老師說：「君子行路，不但要提防小人對自己的傷害，更要提防和小人對抗時，自己對自己的傷害，所以，道比術要難得多，殿下你為了避難，是選擇做那種無能怯懦之人還是選擇做那種你最不想成為的人，你鄙視，不屑成為的那種人哪？陰險詭詐之主，只能招來陰險詭詐之臣，陰險詭詐之主，也只會被陰險詭詐之臣所利用，你許他那些榮華富貴，因為這些背叛了道義的東西，你可以貴他，可以賤他，反過來也一樣，他因為這些東西，他可以臣你，他也可以判你，」說完，戒尺又在手上狠狠的打了一下，太子連忙說：「老師別打，疼！」老師說：「在你的心裡可能還不完全接受這些道理，但是我說的這些道理，殿下總有一天會明白，道理全聽，舉步維艱，道理不聽，死路一條，等你以後可以掌握這個度了，也就是你可以執掌天下之時，在這一點上，殿下要向陛下學習，你偽造試卷的筆跡，臣能看出來，陛下也能看出來，等我走了之後，殿下一定要好自為之。」太子哪能捨得讓老師走啊？「公主不在，母后也不在了，武德侯也離開了，老師不能走」我就是怕李柏洲會迫害老師，所以我才這麼做的，我就是不想讓老師離開啊，老師你要等到我娶了王妃，有了世子，老師，我錯了！這一段羅晉一把鼻涕一把淚啊，看的很多觀眾也跟著捨不得，而王勁松老師的這段教導，演技也很炸裂，不愧是老戲骨啊！人的一生遇到 一個好老師至關重要，在太子的心中，他的老師簡直就是他的精神支柱啊！

但是皇上究竟還是皇上啊，太子的這點小心機被皇上完全知曉了，將太子狠狠的申斥了一頓
