九子奪嫡，雍正能登上皇位，這三個女人為他奠定了紮實的基礎

皇宮是一個明爭暗鬥的地方，處處都隱藏著殺機，一不小心，可能就永墜深淵。尤其是各個皇子之間的爭鬥，那可叫一個激烈，只要能坐上那把「龍椅」，他們可以不擇手段，但往往笑到最後的人，卻是曾經那個不起眼的小人物，比如說我們的雍正皇帝，也就是胤禛，當初誰能想到是他坐上皇位呢？

但從歷史來看，雍正能走到最後，其實是必然的。別看他的出生不好，但是他在年紀很小的時候，就展現出了過人的機智與能力，才剛剛六歲，就進入了尚書房學習，像咱們這麼大的年紀，還不知道在哪兒瘋玩呢？但他是如何做到自律的呢？就是因為他深知自己的身世在各個皇子當中，並沒有任何的優勢，想要在其中爭的一席之地，首先還得自身功夫硬才行。不過，光是提高自己能力是遠遠不夠的，畢竟「千里馬」還需要「伯樂」來賞識呢，沒有足夠的機會，就算你學富五車，你也沒有展現的地兒啊。

所以說，他人生中第一個重要的女人來了，也就是他的養母。為啥是他的養母呢，因為他的生母地位卑微，按照當時的規矩，那是不能帶孩子的，所以只能給孝懿仁皇后撫養，而孝懿仁皇后佟佳氏是滿洲鑲黃旗，更是康熙的嫡親表妹，剛剛進宮就冊封了貴妃，而且還是康熙的第一位貴妃，如此可見，她的地位是很高的，用來當靠山，確實是不錯的選擇。有了中間這根引線後，雍正不僅有了強大的後盾，而且也可以時常的活躍在皇帝的眼前，比那些生下來連爹長啥樣兒都不知道的好太多了，畢竟習慣是一種很可怕的東西，經常活躍在一個人的眼前，想忘記都難。

從表面來看，雍正和養母佟佳氏確實是非常和諧的場面，母慈兒孝誰不稱讚？但說白了，或許只是互相利用的關係罷了，雍正需要一個靠山，而佟佳氏只有一個女兒還早早過世，在那個母憑子貴的年代，沒有後的妃子，始終位置都是不牢固的。

不扯其他，咱再來看雍正是如何步步為營的，有了一個背景強大的養母，只是代表了他擁有了爭儲的資格而已，但想要獲勝，還沒那麼簡單。康熙有那麼多個兒子，哪個的背景不強大？所以，雍正想要進一步擴大勢力，只能通過古人常用的「娶妻」來實現。經過強強聯合之後，雍正手中的籌碼就更大了。於是乎，作為內大臣步軍統領雲騎尉費揚古之女烏拉那拉氏便順理成章的成為了雍正的結髮妻子，之後的孝敬憲皇后。而烏拉那拉氏過門之後，也盡心輔佐雍正，幫他料理家務，讓他可以安心在外面打拚「事業」，再加上自己的父親位高權重，雍正這一步棋可以說走的很妙，家事外事都兼顧到了。

但這個階段，皇宮裡面暗潮洶湧，外面也是戰亂四起，尤其是青海西藏一帶，更是民不聊生。所以，咱們的第三位女主出場了，那就是年貴妃。那她一介女流之輩，又如何能得雍正的青睞呢？這還要歸功於他的哥哥年羹堯，年羹堯是怎樣一個人物呢？他曾配合各軍平定了西藏的戰亂，並且還率領清軍平息了青海羅卜藏丹津，又被加封為太保、一等公，可以說風光無限，如果有了他的助力，那勝率又高了幾分。所以，雍正還是用了老方法，乾脆直接娶了他的妹妹，成為了一家人，現在已經是一條繩上的螞蚱了，你不幫也得幫。

當然，雍正能登上皇位，靠的不僅僅是這些因素，他自己足智多謀，一邊提升自己政務能力，一邊又表現出無爭鬥意思，其他人沒有將注意力放在他身上，他便可以暗中培養自己的勢力，在關鍵時刻，一擊制勝。但也不得不否認，他選擇的靠山都是很強大的，小編估計從他小時候，就在開始籌劃這件事了，只不過當時羽翼未豐，只能猥瑣發育罷了。
