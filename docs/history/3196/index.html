明朝後裔異國生活122年，說漢語用人民幣，心中始終盼望回歸祖國

2019-12-08 0條評論 20次閱讀 0人點贊

摘要

明朝後裔流落海外122年，用人民幣讀中國教材，如今盼望回歸祖國隨著國事衰微，官僚腐敗橫行，明末烽火氣死，農民起義層數不窮。作為明朝的統治者——崇禎帝，雖然滿懷光復大明的熱血，但奈何明朝積重難返，自己又多疑不信忠臣良將，最終始終無法挽回明朝垂敗的命運。

隨著國事衰微，官僚腐敗橫行，明末烽火氣死，農民起義層數不窮。作為明朝的統治者——崇禎帝，雖然滿懷光復大明的熱血，但奈何明朝積重難返，自己又多疑不信忠臣良將，最終始終無法挽回明朝垂敗的命運。公元1644年，李自成攻陷明朝首都北京，崇禎帝自縊於煤山之上，大明朝覆滅在內憂外患之中。

這一年同樣標誌著，中國兩千多年封建統治中，最後以一個漢人王朝至此終結。雖然農民起義軍攻下北京城，但作為統帥的李自成，並未高興多久，其建立的大順政權此時，離覆滅也越來越近了。當李自成坐在明朝皇帝的龍椅還不足3個月時，鎮守山海關的吳三桂，便打開關門迎清軍入關，大順軍在滿清八旗的攻勢下，被打的潰不成軍，最終李自成不得不退出北京。

隨著李自成戰死，整個起義軍頓時群龍無首，在清軍的圍剿下損失慘重。最終在無可奈何之下，只好投靠雄踞南方的南明朝廷。與之相同的是，此時的另一隻起義軍——大西軍，在張獻忠被清軍射殺之後，整個軍隊也選擇了和南明合作。頓時間，中華大地上掀起了反清復明的浪潮，此時的南明政權實力大增，且掌控著南方大片領土，作為清朝皇帝的順治，已經開始萌生劃江而治的想法，可這種相互對質的平衡時期，並未維持多久便被打破了。

南明共經歷了永曆、隆武等政權更替，並在此期間，清朝一直沒放棄對南明的圍剿，而在清軍的窮追猛打之下，明朝宗室便削減的越來越小，整個南明逐漸失去統治根基，最終整個南明政權，被清朝壓縮在東南沿海地區。在明朝最後一位皇帝——朱由榔時期，整個南明政權不得不逃往西南，前去投靠緬甸，整個南明殘餘自此落戶在緬甸北部，並在此休養生息，最終在緬甸形成”果敢地區”。

而在清朝統治時期，果敢地區始終處於統治範圍之內，知道1897年之後，才被分離出去。在1897年至1949年期間，果敢一直都是英國的殖民地，是英屬緬甸人。二戰之後，隨著獨立浪潮的襲來，緬甸最終脫離英國殖民統治，成為一個完全獨立自主的國家，而生活在次的南明後裔，也就成為了緬甸的一部分。雖然已經在緬甸生活300多年，但他們的風俗習慣、宗教禮節等，都保持著原本的模樣，並且在果敢人心中，始終認為自己的是中國人。

現在果敢地區的通用語言依舊是漢語，流通貨幣也是人民幣，當地學校所使用的教材，也是雲南教材，整個地區的電力供應都是由雲南電網輸送，所以，果敢人對於中國有很大的認同感，這個脫離中國122年的明朝後裔，始終期盼著能夠回歸祖國。
