從《慶余年》中的北齊，看歷史上真正的北齊

2019-12-23 491條評論 4,631次閱讀 2662人點贊

《慶余年》中，存在三股勢力，南慶，北齊，東夷城，在中國歷史上，這三股勢力，只有北齊在歷史上有同名的朝代存在，那麼歷史上的北齊是個什麼樣的朝代呢？

在東晉十六國之後，中國歷史再一次出現了分裂，南北朝（公元420年—公元589年）對立，南朝有：宋、齊、梁、陳，北朝有北魏、東魏、西魏、北齊和北周。

國家建立

東魏的有位權臣名叫高歡，他有兩個兒子，長子高澄，次子高洋。在東魏武定五年(547年)正月時，高歡在家中病逝，他在位時權傾朝野，死後他的長子高澄繼承了他的位置，但是後來高澄因為膳奴刺殺而死，高洋就接著繼承了父兄的基業，高家在父兄的經營之下，已經如日中天了，這時候已經是可以左右朝政的龐然大物了，所以在公元550年，高洋廢掉東魏的傀儡皇帝孝靜帝，自己坐上了皇帝，國號齊，建元天保，建都鄴，歷史上成為北齊

北齊地圖

家族遺傳，瘋子皇帝扎堆

北齊（550年-577年），一共存在了二十八年，一共有文宣帝高洋、廢帝高殷、孝昭帝高演、武成帝高湛、後主高緯、幼主高恆六帝。

北齊皇帝列表

北齊雖說只有短短的二十八年歷史，但是卻被稱為荒淫的禽獸王朝，只有孝昭帝高演的評價好點，只可惜高演即位才2年就墜馬重傷而死。國家治理沒有法度，全靠皇帝自己的喜好，人想殺就殺，昏庸無道，沉迷酒色，據說高洋在位期間，喜歡喝酒殺人，當眾淫樂，三綱五常，人倫道德已經不存在，高湛，高瑋等等都是有過之而無不及，真可謂高家之人骨子裡都有「瘋狂」的基因。

究其原因，一是權利的掌控讓人迷失自己，二是高氏原本就是鮮卑族人，就與胡人，保持蠻性，三是家族影響，一個皇帝這樣干，就會是其他人趨之若鶩。

北齊在公元577年被北周所滅，這個王朝存在雖說短暫，但是留給人民的傷痛確實永恆的。

政治貢獻

高洋在位期間，雖說這個人驕奢淫逸，但是他卻喜歡法律，他在位期間，北齊修例了《北齊律》。大理寺這個名字在這時候出現，這部法律對後世法律的形成齊了很大的作用，

北齊律

經濟貢獻

北齊推行均田制，這種是一種按照人口分配土地的制度，對發展農業起到了積極的作用，但是北齊依舊是在南北朝時期，士族門閥大行其道，他們佔據了很多勞動力和土地，壟斷了經濟和政治，影響了農業社會的發展，後期隨著隋唐實行科舉制度，士族開始走向消亡。

北齊的農業、鹽鐵業、瓷器製造業都相當發達，雖說為國家製造積累了不少財富，但是北齊後期，皇帝長久以往的驕奢淫逸，苛捐雜稅繁多，人民壓力繁重，國庫虧空，已經無法維持這個帝國的正常形態，階級矛盾，內部矛盾加劇，滅亡已經不可避免。

總的來說，北齊在歷史上存在時間雖然短，但是他能被世人記住的原因，就是這是一個「瘋子」王朝，在中國歷史進程中，以其獨特的表現，留下了濃厚的一筆色彩。
