一代女皇武則天在歷史上是個傳奇人物，因為有著足夠聰明的頭腦和才華，慢慢掌握了大權，最終走到了皇帝的寶座上，但是對於她這個人，評價只能說褒貶不一，雖然有功，但是也心狠手辣。

其實李治之所以能夠鞏固自己的地位，也是因為武則天在背後默默出謀劃策，當時唐太宗去世之前，知道自己的兒子李治性格柔弱，所以就為李治物色可以輔佐他的大臣，後來，李治上位了，在很多軍國大事的商討上確實都要和那些大臣們商量一下，在諸多事情中，立後當然也要商討。

唐高宗要立武則天為皇后，首先就要取得舅舅長孫無忌的支持。長孫無忌在凌煙閣二十四功臣榜中名列第一。朝廷中，他是文官之首。唐高宗與武則天攜帶厚禮，親自登門拜訪長孫無忌。這長孫無忌固執己見，油鹽不進，堅決不同意立武則天為皇后，導致君臣關係出現間隙。

此時，唐高宗與武則天由於受到了挫折，有了共同的反對派，夫妻關係又變成了戰友關係。武則天是想從昭儀晉陞為皇后，而唐高宗也想利用廢立皇后這件事，打壓朝廷的元老重臣的勢力，以奪回皇權。

鑒此，唐高宗把極力反對立武則天為後的元老褚遂良貶到了潭州，此舉有利於武則天的上位，更能取到殺一儆百的效果，對鞏固皇權也是有利的。

武則天的對手是王皇后，地位非常尊貴了，當時，唐高宗企圖打壓以長孫無忌為首的宰相勢力，就必須扶持另一個勢力與之抗衡，這時，寒門力量開始蠢蠢欲動了，於是這些中堅力量就紛紛支持武則天，唐高宗看得明白，他將首先上疏支持武則天為後的李義府，從中書舍人升到了中書侍郎，不久後又提升為宰相。

不過武則天還是非常有優勢的，因為武則天為李治生了兒子，畢竟母憑子貴，而且武則天在太宗朝的時候就對政治有一定的了解，積累了不少經驗，所以在給李治的諫言中，提到了重農減稅，這中高宗下懷，使高宗死心塌地要扶她上位。

於是李治就說王皇后沒有子嗣，但是大臣們依舊反對，而李治這個時候又想得到開國功臣李勣的支持，但是李勣聰明啊，開會的時候請假不來了，李治又拿不定主意，只能上門去詢問了，這個時候李勣無奈，就說，此陛下家事也，何須問外人？” 果然，高手不愧是高手，李勣沒有得罪其他大臣，同時也討好了李治，自然堅定信心，立武則天為後了。
