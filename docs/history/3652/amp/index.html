著名宦官魏忠賢，權勢滔天，一個合格的壞人

2019-12-02

眾所周知，明朝大宦官是非常了不起的，在明朝許多宦官的權利比一些大臣都要大，有的甚至可以輕言皇帝廢立，宦官能有這麼大權利，還不是一個兩個，而是很多，能做到這樣的恐怕只有明朝了，可能皇帝覺得這些沒有依靠的閹人比較可靠吧，在這些宦官中，一個叫魏忠賢的宦官脫穎而出，成為明朝歷史上不可忽視的一筆。

1. 對家人狠

魏忠賢都出身貧苦，因此沒有接受過教育，大字不識一個，一直混跡於市井之中，熱愛賭博，身無長處。在入宮之前，魏忠賢是有家室的人，但是由於他熱愛賭博，家中一貧如洗，妻子也改嫁走了，走投無路的魏忠賢會怎麼做呢？這時候，他看見了自己的女兒，你以為他會良心發現，改掉壞習慣和女兒好好過日子？不，他賣掉女兒，自己決定進宮討生活。這就是魏忠賢，即使對待自己的至親骨肉，也毫不心軟。

2. 對自己狠

有人覺得，當太監是窮苦人家沒辦法的事，但是在明朝，進宮當太監卻是一個極其熱門的工作，因為明朝太監很吃香，權力很大，魏忠賢沒有錢，就自己把自己閹割了，由此可見，魏忠賢對自己也挺狠，但是，戲劇性的一幕出現了，切完自己的魏忠賢因為過了年齡沒有進去宮，這下子，魏忠賢傻了，自己切了卻進不去，然後魏忠賢只能靠乞討打雜為生。但是天無絕人之路，魏忠賢找到一份工作，這家主人也是一個太監，由於魏忠賢在這裡勤勤懇懇，這家主人就保舉魏忠賢進宮了，魏忠賢就開始了自己的宦官生涯。

3. 對外人更狠

魏忠賢進宮後就開始了他如魚得水的生活，對待有價值的，他阿諛奉承，對待擋他路的人，做法狠絕，為了成為皇帝的心腹，他扳倒皇帝身邊的大太監王安，在王安決定告老還鄉後，魏忠賢還是沒有放過他，最後暗殺了王安，王安死後，魏忠賢理所當然的成為皇帝的心腹。還有當時有名的東林黨，魏忠賢陷害忠良，無數東林黨人死於他的手裡。還有對待皇帝和後宮妃子，他把控皇帝，殘害皇帝後宮，後宮凡事有孩子的嬪妃不是流產就是母子具亡，魏忠賢不是不想讓皇帝有孩子，只是希望晚一點，因為萬一皇帝的孩子不好控制怎麼辦，那他的權勢榮華就全都葬送了。

4. 迎來死亡

魏忠賢混的風生水起，但是這時皇帝病危，魏忠賢一下子就慌了，後來皇帝的弟弟即位，魏忠賢的好日子也就到頭了，最後魏忠賢在絕望中自殺，剩下的人要不斬首，要不流放，魏忠賢一生就此完結。對於魏忠賢，沒有什麼評價，就一句，他是一個壞人，徹徹底底的壞人，純粹的壞人，僅此而已。
