四百年強漢，始於高祖！談談劉邦殘害忠良背後為漢帝國所做的貢獻

2019-12-16 2,801條評論 4,881次閱讀 2305人點贊

公元前209年，秦朝天數已盡。陳勝、吳廣順天時，隨人意，高舉叛旗，討伐秦二世。一時之內，四海群雄皆振臂高呼，紛紛起兵。項羽戰河北，劉邦戰河南。兩人席捲八荒，橫掃宇內。

天下已定，秦朝已亡。然劉邦、項羽二人皆有帝王之野心。兵戎相見，在所難免。由此，便拉開了楚漢相爭的序幕。

烽火連年，兵戈不息，爭鬥之中，劉邦略勝一籌，與亂世之中奪取皇位，開創大漢王朝。王朝初建，命運多舛。劉邦草草北伐，結果白登山被圍，七日之內，提心弔膽。陳平獻計，方得解圍。然兵敗之事，仍被天下人嘲笑。外患未平，內憂又起。天下平定後，劉邦屢殺功臣。韓信、彭越、英布等功臣名將相繼被害。活者亦惶惶不可終日，張良隱居，蕭何意冷，前朝元老，日漸離心離德。

劉邦斬白蛇

劉邦所作所為，引起後世許多人不滿！認為劉邦僥倖擊敗項羽，奪得天下，卻無治國之才。白登被圍，簡直一國之恥，殘害忠良，豈是明君所為?其實真相併非完全如此！劉邦白登被圍、殘害忠良固然可惡可恨，但其所作所為亦有自己的隱情與無奈。其濫殺功臣的背後，亦是為了漢帝國的利益。

一切還要從秦朝的滅亡說起

「天下苦秦久矣！」司馬遷在《陳涉世家》中如此總結秦朝滅亡的原因。固然秦朝大興土木、勞民傷財，天下之人早已精疲力盡。人民對秦朝的怨恨日益增長，最終形成了毀滅秦朝的龐大洪流。但是秦朝的滅亡還有諸多方面的原因。其中極為重要的一點便是秦始皇決定推翻分封制而以郡縣制替代。

「百代皆行秦政法」，毛主席在七律.讀《封建論》中寫道。從後世觀之，秦朝的許多制度都是十分先進高效的。其中郡縣制尤為重要。從頒布郡縣制起，中國才真正成為了一個完整統一的中央集權國家！然而在秦朝，郡縣制卻並未得到大多數人的認可，並且觸及到六國貴族的根本利益！即便秦始皇為推行郡縣制向六國貴族作出不少妥協，然而六國貴族皆利益熏心之小人，又豈能了解秦始皇的宏圖大志？六國貴族從心底痛恨奪取他們權力與領地的秦始皇，並時刻期待著秦朝的滅亡。

秦始皇雕畫

當陳勝、吳廣點燃推翻秦朝的第一把火時，六國貴族皆群起響應。紛紛打著復國的名號企圖奪回自己的利益與領土。而叛軍之中，劉邦卻與六國貴族不同！

與痛恨秦始皇的六國貴族不同，劉邦打心底是崇拜秦始皇的。此番定論有諸多依據。其一：相比於項羽見到秦始皇時發出的「吾可取而代之」的豪言之語，劉邦則表示「大丈夫當如此」。短短六字之中，我們很容易看出劉邦對秦始皇是認可並且崇拜的。其二：劉邦進入秦首都咸陽對其秋毫無犯，儘可能維護了秦朝最後的尊嚴。其三：劉邦的大漢建都於咸陽旁的長安，也是劉邦對秦始皇的尊敬。

所以，當劉邦擊敗項羽建立漢朝時，劉邦也渴望建立起一個像秦始皇的秦朝一般中央集權的、大一統的王朝。然而秦朝二代而亡的慘象尤歷歷在目。如果冒險像秦始皇一般推行郡縣制，自己剛剛建立的王朝是否也會在自己死後被頃刻推翻？另外跟著自己篳路藍縷、屢建功勛的將軍們也都兵權在握，如果推行郡縣制觸動了他們的權力，免不了又是一陣血雨腥風！

劉邦故居沛縣

最終劉邦還是選擇了妥協，僅在自己國家的西部推行中央集權的郡縣制；而將自己東部的領土分與韓信等戰功卓著的諸侯王。可是，建立這樣半郡縣制、半分國制的漢朝並非劉邦的本願，大漢離成為中央集權的統一國家還相去甚遠！為了進一步實現自己的宏偉理想，劉邦必須要有新的作為！而這便為他白登被圍、殘害忠良埋下了伏筆。

欲學秦始皇卻匈奴七百餘里，威震諸侯王！卻不料被圍白登，前功盡棄

「大風起兮雲飛揚，威加海內兮歸故鄉，安得猛士兮守四方。」劉邦是一位志向遠大的開國皇帝。其希望自己的帝國能夠強大統一。然而帝國的現狀卻令其大失所望。於內，自己帝國的一半領土依舊在諸位諸侯王手中，未能真正統一；於外，北方的匈奴在中原內亂的情況下重新侵入中原，而帶領他們的更是匈奴歷史上最有名的領主之一-冒頓單于。

公元前201年，劉邦已年過半百。留給他治理帝國的時間已經不多了。但是劉邦迫切地希望改變帝國的現狀，早日實現漢朝富強而統一的目標。於是劉邦選擇了一條最為危險的道路-御駕親征匈奴。

御駕親征雖然看上去十分魯莽，但是劉邦也有著自己的打算。劉邦知道，自己的王朝剛剛建立，無論是匈奴還是諸侯王之間都還有企圖奪取自己疆土之人。於是劉邦希望借這次北伐，展示自己的國力。從而威懾匈奴與諸侯王，讓他們不敢輕易造次。

匈奴弓騎，僅憑步兵根本無法與之對抗

然而從後世觀之，這場戰註定是打不贏的！

近300年春秋，又200餘年戰國。秦朝建立後又大興土木，進一步勞民傷財。再之後又是滅秦之戰和四年的楚漢爭霸。整整五百餘年間，中原大地都在戰火之中苟存。漢朝建立初期可能是中國歷史上最為衰落的時期之一！反觀匈奴，情況卻恰恰相反。匈奴最偉大的領導人之一-冒頓單于不僅以武力統一了北方各部落；同時銳利改革，發展經濟與軍事。在楚漢爭霸時期，冒頓單于領導下的匈奴已經成為北方最為強大的帝國。另外，劉邦所率大多為步兵，而冒頓單于手下皆為精騎。以步兵對抗騎兵，諸位皆知其難也。

但是劉邦還是太需要這場勝利了。如果這場北伐獲勝，劉邦不僅可以掃除外部的侵略；還可以向各諸侯王展示自身權威。從而用權威逼迫各諸侯王交出權力。可惜，對勝利的渴望迷失了劉邦的雙眼，本就處於弱勢的漢朝軍隊又在劉邦的指揮之下中了匈奴的誘兵之計，被圍白登山。最後陳平獻計，劉邦才僥倖得以脫身。

匈奴人

白登之敗後，劉邦更感到漢朝的弱小，不得已走上了殘害忠良的道路

劉邦雖僥倖從白登逃脫，但是危機卻並沒有因此消除。劉邦深深感到自己現在的漢帝國實在太過弱小。僅憑漢朝現在的國力絕對無法與匈奴對抗。要打敗匈奴，就必須建立中央集權的統一國家。可是帝國的整個東方都被劉邦封於韓信、彭越等功臣名將。要建立中央集權的國家，就必須從他們手中收回權力。

劉邦或許本希望通過一場規模宏大的北伐建立自己的威信，從而逼迫各諸侯王交出權力。然而草草北伐反被匈奴圍困七日，自己更是威信掃地。一時之間竟找不到其他辦法收回那些本該屬於漢王朝的權力。

長安夜景

「卧榻之上，又豈容他人酣睡！」在劉邦去世的前一年，或許是感受到自己大限將至，而自己的繼承人太子劉盈又並非才能卓越之人。劉邦終於決定用最血腥、暴力的方式收回自己分散到諸侯中的權力。一年之內，劉邦接連以謀反之罪殺害韓信、彭越、英布三位功臣。之後更是通過殘忍手段廢除所有異姓王。並號召群臣立下白馬之盟：「凡是非劉姓稱王者，天下人共擊之！”至此，劉邦才了結大願，將天下權力都收歸到劉氏手中。不久，劉邦便與世長辭。

劉邦殘害忠良雖然成為其一生污點。但是不得不說劉邦將權利盡收劉氏手中，是邁向了漢朝實現中央集權統一國家的第一步。劉邦率先剷除韓信、彭越等異姓王，之後景帝通過平定七國之亂進一步削弱了各諸侯王的實力。最後武帝通過施行「推恩令」進一步分散諸侯王權力，最終再也沒有任何諸侯王勢力能夠威脅到漢王朝的中央集權統治。而武帝更是在國家權力統一的基礎下，完成了漢高祖北擊匈奴的理想，並進一步將漢朝建設成為當時世界上最強的帝國！

漢武帝雕像

「明犯強漢者，雖遠必誅」，「國恆以弱滅，為漢以強亡”即便是到公元220年曹丕篡位，東漢滅亡為止。漢朝都是蠻夷部落不敢觸碰的強大帝國。然400年強漢，正起於劉邦！

也許有人會反駁，正是因為劉邦殘害忠良、剷除異姓王才導致劉氏諸王權力坐大，最終爆發七國之亂。可是同姓之間尚且爭權奪位，誰又敢保證異姓諸王能永世效忠？再從歷史的進程來看，七國之亂最終被平息，中央權力進一步得到加強。一切都朝著劉邦所期望建立的漢朝發展！

當然，即便是出於建立中央集權的統一帝國這般宏偉的目的，劉邦殘害忠良的殘忍事實也不會有任何改變。劉邦的是非對錯，恐怕依然會爭論不休！
