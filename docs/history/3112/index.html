雍正登基為何迅速追查虧空？真實情況比《雍正王朝》懲治更為嚴厲

2019-12-09 0條評論 21次閱讀 0人點贊

摘要

撰文/曾國安雍正登基之初就掀起了清查虧空，電視劇《雍正王朝》高度還原了這一歷史畫面。雍正即位前夕，內閣官員就起草了一份意見，打算將豁免官員虧空作為新皇帝登基的恩典。看到這份草擬意見，頓時引起雍親王的極度不滿，由於還未舉行登基大典，並未大發雷霆，只是拒絕施行。

撰文/曾國安

雍正登基之初就掀起了清查虧空，電視劇《雍正王朝》高度還原了這一歷史畫面。

雍正即位前夕，內閣官員就起草了一份意見，打算將豁免官員虧空作為新皇帝登基的恩典。看到這份草擬意見，頓時引起雍親王的極度不滿，由於還未舉行登基大典，並未大發雷霆，只是拒絕施行。這個草擬意見不僅沒能幫到虧空官員，相反加速了雍正的處理速度。登基的一個月後，他就迅速向戶部下發旨意，要求全面清查官員虧空，並由怡親王允祥負責這些專務進行推進。

影視畫面

是什麼讓雍正剛登基就做出如此急不可耐的動作呢？

從根本上來說，是康熙晚期吏治鬆懈導致的官員整體腐化從而導致的國庫空虛，以及這位新皇帝對帝國的責任感。

本質上來講這都是老皇帝康熙晚年推行「寬仁」的措施所導致的併發症。彼時的康熙明顯精力不足，甚至與早年的勵精圖治判若兩人，加之「九子奪嫡」的親情因皇位而出現的焦灼狀態讓這位老人陷入巨大凄涼與無奈，甚至喪失了對帝國治理的熱心。正是在老皇帝「寬仁」的政治氛圍下，清朝各地官員開始大作虧空，無疑這些錢都進入了私囊。對於這些現象，雍正即位之前便瞭然於心，他說:「歷年戶部庫銀虧空數百萬兩，朕在藩邸，知之甚悉。」甚至雍正對於如何抓住帝國矛盾癥結所在是充滿自信的「試思今日之皇帝即飽嘗辛酸苦辣之四阿哥也。」

雍正畫像

可知在雍正心裡，對於帝國所有問題早已瞭然於胸，也深知如何抓住矛盾所在的緊迫性。對於虧空的根源他說，各地虧空錢糧，不是受上司勒索，就是自身侵漁，都是非法的。此前，大行皇帝寬仁，未對贓官明正法典，所謂勒限追捕，也不過虛應敷衍，虧欠並未得到任何解決，庫藏因此空虛，一旦急需緊急開支，必然引起重大混亂。因此在將大清帝國航船的航線從寬到嚴時，雍正從追查虧空入手了。他正式將此下達整個大清官場，如同一股強大旋風瞬時掀起巨大波瀾，在通告中說:「各省督府將所屬錢糧嚴行稽查，凡有虧空，無論已經參出及未經參出者，三年之內務期如數補足，毋得苟派民間，毋得借端遮掩飾，如限滿不完，定行從嚴治罪。三年補完之後，若再有虧空者，決不寬待貸。」《上諭內閣》

可以說，追繳虧空成了雍正繼位後最為重大的事項，並且雍正認為康熙晚年缺乏對虧空的本質的判斷能力。因為這種錢糧奏銷的弊病很大，裡面主要看是否有「部費」，所謂部費指清時官員任實缺時，向吏部人員賄賂的運動款項。也就是拿公家錢施行賄賂。此外當時更為重要的弊端是明明是正當開支報銷，但是戶部卻不給報，必須要給賄賂好處才可以，因此雍正認為「大行皇帝也知道這種弊端，不過不欲深究，從寬處理，然而朕今不能如皇考寬容。」

雍正批示奏摺

在下達對雍正極為器重的「十三弟」允祥的任命後，嚴肅地對他說:「爾若不能清查，朕必另遣大臣；若大臣再不能清查，朕必親自查出。」這種語氣非同尋常，將雍正鐵心清查虧空的底線原則交託出來，不僅是告誡允祥關於此事的重大性和各級大臣對此絕不能抱有僥倖心理，否則雍正就要親自出馬了。雍正選擇允祥追查虧空不久便顯現出效果，可知這個人是選對了。光是戶部庫銀，允祥就查出虧空二百五十五萬兩，占所辦事件的百分之十七。雍正接報，責令戶部歷年來所擔任的主管官員賠償一百五十萬兩，另幾百萬兩由戶部逐年彌補。《雍正朝起居注》

電視劇《雍正王朝》，有關各位康熙阿哥也牽涉虧空，這件事基本屬於史實的還原。如康熙的第十二子履郡王允裪曾主管過內務府，待追索虧空時，竟然將家中的器皿叫人抱到大街上出賣。第十阿哥也虧空甚多，賠了數萬金，還沒賠夠，最後雍正叫人查抄了他的家。因此允祥因此承受巨大輿論壓力紛紛指責他「過於苛刻」，雍正關鍵時刻表示說此事是自己叫他這樣做「清弊竇，飭令著追。」

影視畫面

與此同時，雍正在地方掀起了更為廣泛的追查虧空，涉案官員級別和數量均創有清以來最高。如山西巡撫蘇克濟任職於康熙時期，後來退休在家被告發他曾勒令各府州縣銀四百五十萬兩，雍正下令籍沒其家產，以補償虧空，同時責令其家人賠償二十萬兩。原河道總督剋扣治河工料，貪污錢糧，被抓入大獄，家財充公。蘇州織造虧空三十八萬兩，抄家賠補。

此外與抄家同樣重要的手段是罷官，凡是貪官，一經告發就要革職，不許再入以往那樣留任以彌補虧空，因為按照雍正的理解是「虧空錢糧各官，若革職留任催追，必致貽累百姓。」因為貪官補償過去虧空「不取於民，將從何出？」一時之間在雍正的嚴厲打擊下，清朝出現大量被抓和罷官人員，甚至湖南因為「參劾已大半」無法正常處理公務。直隸總督李衛向雍正奏報，「通省府廳州縣官，在任三年以上的寥寥無幾，官員頻繁更換，原因是被撤職的人過多。」《上諭內閣》

影視畫面

​面對官員自殺想一了百了，雍正發狠說：「畢竟叫他子孫做個窮人。」

雍正對於處理涉案官員也並非一律，一般都懲治，先抓貪污，然後再追查挪移，因此有的官員便將侵佔上報成挪移，以此避重就輕，希望逃避懲罰。雍正此後「將挪移輕罪之項令其完成，侵欺重罪之項令其後完。」這樣就讓這些「希避重罪之人無所施其伎倆。」的官員走投無路，因此開始出現大量官員自殺，以此打算一了百了。廣東巡撫和閩浙總督等高官未等審理就自殺，雍正說:「料必以官職家財既不能保，不若以一死抵賴，留資財為子孫之計。」因此雍正見招拆招，令將各地犯罪官員「嫡親子弟並家人等嚴加審訊」，「所有贓款著落追賠」，並發狠說:「畢竟叫他子孫做個窮人。」因此雍正也多了「抄家皇帝」的外號。

經過雍正幾年的不斷追繳，大清的虧空有了明顯改善，此後雍正所提升的大臣多以清查虧空作為重點。如田文鏡在擢升為河南巡撫後一到任就累雷厲風行清查虧空，在履新的幾個月後就向雍正彙報說:「臣不遺餘力繳委各府州互相覺察，設法嚴查，總期徹底澄清，不容纖毫短少。」浙江巡撫李衛參原淮徐道，雍正命將潘的家產查出，充作浙江海塘工程費用。至此曾經愈演愈烈的虧空在雍正強力懲治下得到有效遏制。

大清最終敗在了虧空上，被虧空的「回頭浪」猛烈擊倒。

大清後期的虧空再度被「回頭浪」​所擊倒，成了清朝腐敗的重災區。從乾隆和嘉慶時期，虧空開始逐漸再度嚴重，單從地方來講，虧空無外乎以下三點:這種因公務導致虧空的情況在所有虧空中所佔比例並不高，嘉慶帝親政之初，御史張鵬展估計因公虧空,不超過百分之三五，他還把數十年來州縣虧空的主要原因歸納為四種情形。

一是州縣官素性豪奢,平時講排場擺闊氣大手大腳慣了。在京里得缺赴任，就大舉借錢，縣官的太師椅還沒坐熱，討京債的已經尾隨而至,沒別的辦法，只好打開庫門，挪用公款償還，日後老毛病變本加厲,鮮衣美食,燈紅酒綠,挾妓挽娼，哪一樣也少不得，每年千把兩的養廉俸祿哪裡夠用?心想已經伸了手，現在不拿白不拿，以致虧空一天比一天嚴重。

二是心地糊塗，茫無主見。隨官上任而來的上百口子所謂「官親」、「長隨”在下面可勁兒地糟蹋，自己還蒙在鼓裡，偶爾遇到上司稍變臉子，就嚇得茫無所措，趕緊琢磨哪裡沒有打點到，哪裡沒有彌縫好，想方設法補救，日久天長，更加麻木不仁，也就不在乎庫帑虧短不虧短了。

三是專意逢迎，千方百計討得上司喜歡。知府、道台過境，正是盡孝心的好機會，督撫兩司大人蒞臨，，則極盡鋪張之能事臨行饋贈，不惜巨金，底子打得差不多了，遇有肥缺，昏夜苞直，乞來保題舉薦，自己劣連被人告發，更是車金單銀尋求龐護，這就是所謂「浮費無度。」

標題:值得一提的是，曾國藩在成功鎮壓太平軍後，向朝廷報銷開支不得不向戶部「行賄」。

與雍正時期一樣，清查後期戶部收受「部費」較之前更是有過之而無不及。曾國藩向朝廷報銷軍費首先要提交戶部，如果戶部高抬貴手什麼不合規定的費用都能報銷，如果他們故意雞蛋里找骨頭，再合理的費用也難以通過。那麼這其中的「部費」就顯得十分關鍵。曾國藩叫李鴻章向戶部打聽報銷需要多少部費時，李鴻章回信是書吏要一厘三號的回扣，也就是報銷一百兩給一兩三錢，而曾國藩報銷的軍費總額是三千多萬兩銀子，那麼光給戶部「部費」就要四十萬兩。

曾國藩感覺數額實在巨大，值得託人做戶部工作討價還價，最終給八萬兩才算通過。在得知朝廷出於他平定太平軍和捻軍作戰功績後，同意他們免於報銷，為此曾國藩在家信中還在感恩戴德慈禧的大氣他對此「感激次骨，較之得高爵穹官，其感百倍過之」。按理說慈禧發了話，這八萬兩就可以省下了。不過，曾國藩卻說，這說好的八萬兩銀子」部費「還是照給，因為閻王好見小鬼難纏，畢竟以後他還需要和戶部打交道。
