看過這100座沾滿血淚的貞節牌坊，你才會明白，男人活著就是擔當

天天要聞
2019-12-17

在看古裝劇的時候，我們經常會聽到妻子稱呼丈夫為「夫君」。很多人覺得這個稱呼很有文藝氣息，十分的喜歡。

實際上，在中國古代，妻子對於丈夫的稱呼幾經更迭，有一開始的「良人」，到後來「先生」、又有「郎君」、「官人」、「卿」、「相公」等等。

但夫君一詞似乎流傳的更加廣泛，細細考究起來，這與男權社會分不開。大家都知道，受儒家道德約束，人的等級是很分明的，故而有「天地君親師」的排列。

君就是君王，排在天地之下、萬眾之上。古代男子為一家之主，有管理之責，是家裡的頂樑柱，可以為家庭撐起一方天地，為表尊重所以稱為「夫君」。

當然，現在我們在家裡夫妻恩愛，妻子稱呼丈夫為夫君，一來是作為調節男女情感的昵稱，另外也有把另一半引為知己的意思。

大家都熟知宋代李之儀寫的《卜運算元》，「我住長江頭，君住長江尾；日日思君不見君，共飲長江水。此水幾時休？此恨何時已？只願君心似我心，定不負相思意。」

這裡的君字，就頗富有「願得一人心，白首不分離」的詩情畫意在裡面了。

因為古時男人是一家之主，是家裡的頂樑柱，那一旦家裡的男人倒下，一般的家庭面臨的境遇肯定是「天塌了」。

夫君一亡故，家裡的妻小沒了主心骨，這還在其次，對於尚在妙齡的妻子來說，面對的將是數十年漫漫的孤寂長夜。

在古代，人們將婦女守節看得相當重。丈夫死了之後，妻子要堅守貞操，撫育子女，直到老死，這樣才會受到社會的尊重，甚至得到皇帝的表彰。

雖然並不是說古代不允許喪偶婦女改嫁，只是改嫁的行為需要一顆很強悍的心，因為寡婦一旦改嫁，就會被社會認定為蕩婦。儒家禮教要求婦女「從一而終」，不但在丈夫生前要貞節，死後更要守節。

對貞婦進行表彰的行為始於漢宣帝，但真正成為習俗，是理學提倡和官方表彰的結果。理學家認為「餓死事小，失節事大」，寡婦改嫁就是失節。到了元、明兩朝，社會特別鼓勵寡婦為丈夫殉情，也就是丈夫死後妻子自殺陪葬的行為，稱之為「殉烈」。

到了清代，「殉烈」不再提倡，但更加重視寡婦守節撫養幼孤、侍奉公婆的行為。

因為守節思想已經成為整個社會認同的行為規範，所以在古代，也上演了許多令現代人看起來荒誕不經的故事。

西晉有一本《列女傳》，裡面記載：三國時期，夏侯令的女兒嫁給了曹文叔，可是曹文叔年紀輕輕就死了，連一個子嗣都沒留下。

夏侯令勸說女兒改嫁，「你有沒有孩子，還這麼年輕，怎麼守得住啊？」她當時就割掉了兩隻耳朵以示抗議。又過了幾年，夏侯令看女兒孤苦伶仃實在可憐，又苦口婆心勸她改嫁。這次，她割掉了自己的鼻子，以表明自己終生守節的決心。

又說，安定有個女子叫張芝，年紀輕輕守了寡。生逢亂世，她和兩個嫂子一起被亂軍給抓了去。為守住貞潔，她先後殺了兩個嫂子然後自盡。

在封建禮教地束縛下，許多女子在丈夫死後，要麼被迫或自願殉情，要麼孤老一生，更有女子效仿《列女傳》裡面的例子，有的自殘、有的毀容，以示守節的決心。

所以當我走進棠樾牌坊群，看著眼前的貞節牌坊時，腦海里無法抹去那些凄苦女子的身影，彷彿那些荒誕和凄涼的事情，在自己的前世曾經親眼目睹。

貞節牌坊，是為了表彰節婦，由皇帝御賜並由朝庭認證，修建的一座類似門樓的東西。

但貞節牌坊並非所有節婦都能獲得，也是需要競爭的，畢竟在古代，甘願守節或被迫守節的婦女實在太多。

獲得牌坊不但要看機遇，比如正巧被地方官當做先進事迹報告給了皇帝；但更要比誰更能熬，熬的時間越長，就越有機會勝出。

棠樾牌坊群，位於安徽省歙縣鄭村鎮棠樾村，屬黃山市管轄。

黃山歷代名人輩出，歷朝皇帝賜給的各種牌坊多到數不過來。而作為程朱理學的故鄉，當地婦女受封建「節烈」思想禍害更深，出現過一大批以《儒林外傳》中王三姑娘為代表的節婦烈女。至今仍有100多座完好地保存著。

棠樾牌坊群就保留了兩座貞節牌坊——清乾隆32年所建的鮑文淵妻坊、清乾隆49年所建鮑文齡妻坊。

這些用「歙縣青」石料豎起的牌坊，上面大都刻著「矢貞全孝」、「立節完孤」字樣，每座牌坊的背後故事，被一隻掛在導遊的嘴邊，「縣誌記載，江氏為棠樾人，26歲守寡後，『立節完孤』，把兒子集成培養成歙縣的名醫，在她80歲高齡時，族人為她請旌，建起了這座宛如其化身的牌坊。」

想著一個個鮮活的生命被壓在這沉重的石塊下面，令人不禁感念現世的美好。

但要說的是，雖然現在已經沒有了要求婦女「守節」的封建思想，但哪個女人不想與自己愛的男人白頭到老、相伴一生呢。

在這個時代，男人和女人同樣分擔著社會和家庭的責任，誰也離不開誰，但作為男人還是更應該多些擔當。

各種不節制的生活習慣正在不斷摧毀很多男人的身體，這才會有了央視那個「他好我也好」的噁心廣告。我要說的是，為了這個家「他好我也好」，男人更要珍惜自己的身體。

除非你願意看著自己的嬌妻為你「守節」孤苦一身，或者改嫁後，讓別的男人抱著曾在你懷裡千嬌百媚的妻子，還不對你孩子好。
