聖人孔子的遠祖，春秋時期宋國的始祖微子啟的曲折、矛盾的人生

天天要聞
2019-12-18

微子，名啟，商紂王的哥哥，因紂王「無道」，他逃離了朝歌，史稱他「乃辭官而隱去」。儒家一直稱讚他為「邦無道則隱」的典型。周成王時，周公平定了武庚祿父之亂後，以成王的名義封微子於宋（今河南商丘），是宋國的始祖。據稱，微子啟的弟弟微仲衍，宋國的第二位國君，就是孔子的遠祖。

《呂氏春秋》載：「紂之同母三人，其長曰微子啟，其次曰中衍，其次曰受德，受德乃紂也，甚少矣。紂母之生微子啟與中衍也，尚為妾，已而為妻而生紂。紂之父、紂之母欲置微子啟以為太子，太史據法而爭之曰：有妻之子而不可置妾之子。」

上述記載說微子啟的母親在生老大微子啟和老二微中衍（即微仲衍）的時候還是小老婆，但是後來被帝乙封為正妻時又生了一個兒子老三名叫受德。當確定誰該當太子時，本來帝乙和妻子都想讓老大微子啟來當太子。結果太史不同意。他認為老大微子啟是在他母親為妾時出生的，儘管後來封為正妻，但仍應當是庶出，雖然是老大，仍然不成當太子。最終帝乙將商朝的王位傳給了受，又名受德，即後來亡了國的商紂王。這個說法也太奇葩了，一是理由說不通，怎麼母親轉正了，兒子反而不能由庶出轉正成嫡子？二是那個太史居然有那麼大的權力？

當然，《史記》中只說微子是紂王的庶兄，並沒有肯定他倆是同母兄弟或者異母兄弟。現在基本可以確定微子與紂王是異母兄弟，而不是同母親兄弟。《呂氏春秋》之記載是依據《尚書》中記載微子為「殷王元子」，誤以為元子就是元妃之子。

沒有當上商王，微子的命運就有些凄慘了。他雖然貴為王子，也是卿大夫的職位，可是面對那個不按常規出牌的帝辛，他這個哥哥可以憋屈得很，應該是受到了死亡的威脅。畢竟紂王連他的叔叔比干都敢殺。

《尚書》和《史記》都記載了微子啟因為紂王無道而去詢問「父師（史記稱為太師）、少師」他該怎麼辦的事。許多人以為父師是紂王的叔叔箕子，少師是紂王的叔叔比干。其實不是的，父師叫疵，少師叫彊（音qiǎng），他們倆都是商朝的樂師，後來他們也都抱著樂器逃到了周國。

微子問父師、少師說：「殷朝不行了，不能統治四方百姓了。看看我的祖先成湯該是多牛，成就那麼多偉大的事業。可是現在的這個紂王，天天酗酒，只聽女人的話不聽我們的話，毫無德性。殷朝的臣民都成了強盜了，卿大臣也不守法，老百姓也相互仇視。殷朝快要滅亡了，沒有半點希望了。」就這麼一直嘮嘮叨叨，彷徨無計又惶惶不可終日。他不知道是該留下還是該逃走。他又問父師、少師：「我是逃離呢？還是留下來呢？這很重要呀，我一旦選擇錯了那可就完了。」這兩個樂師估計自己也是吃了紂王的癟，而且他們也知道憑微子的影響力留在殷朝也起不了什麼作用，就勸微子逃跑。微子一開始並沒有逃到周國，而是離開了朝歌，至於逃到哪裡去了還真的不知道。但是，父師和少師倒是先逃去了周國。甚至有人說他們倆人是微子派到周國請兵來伐商朝，自己在商朝作內應。

要說紂王也確實猛，他的兩個叔叔一個箕子，一個比干，就因為天天在他面前嘮叨，一個給關起來了，一個給殺了，而且比乾死後還被挖出了心肝，箕子最後也剝奪了貴族的待遇，降為了奴。我也不明白為什麼紂王對他的這些個叔叔、哥哥就那麼恨？

當然，最後，商朝真的在紂王手上滅亡了，強大的商朝被弱小的周國給突襲了。

現在問題是，微子啟當時的逃跑從道義上講到底是對還是不對呢？是不是值得稱讚呢？

孔子是極力讚揚他和這個祖先的。《論語》中將他作為殷商「三仁」之一（比干、箕子、微子）。

大多數儒家代表也認為，對於商紂的殘暴統治，微子堅持批判的立場，在批判不起作用的時候，選擇逃亡，所謂「邦有道則仕，邦無道則隱」，逃跑是對的。

可是後來商朝被周國給滅了，作為勝利者、征服者的周武王駕著馬車進了商朝的首都朝歌時，微子啟的表現就讓人大跌眼鏡。

《史記·宋微子世家》記載：「周武王伐紂克殷，微子乃持其祭器造于軍門，肉袒面縛，左牽羊，右把茅，膝行而前以告。於是武王乃釋微子，復其位如故。」

這個微子不僅犒勞獲勝的周師，還要跪著將代表權力的祭器交給武王。上述記載就是說他就拿著宗廟裡的祭祀的器具來到武王軍門前，袒露上身，雙手反綁，讓人在左邊牽著羊，在右邊拿著茅，跪著而行，來到武王面前，表明自己的心意。這形象成了後世投降派投降的典型儀式。

孔子他老人家的這個祖先在當時的表現今天看來怎麼也高大不起來。

當然也有人給他臉上貼金說，認為微子是順應歷史潮流。

其實，孔家知道這事有點說不過去。所以在編寫孔子的家譜時，微子啟並不是孔子的直接的祖先，他還有個弟弟叫微仲衍，孔子是微仲衍傳下來的後人，與那個名聲有點不太好的微子啟只是有那麼一層關係。這似乎又是一個「羅生門」。筆者寫微仲衍這個人時，後文再說這個問題。

毛主席就曾評論說：「他這個國家為什麼分裂？就是因為這三個人（筆者：微子、箕子、比干）都是反對派，而微子最壞，是個漢奸（筆者：應該說是殷奸或商奸），他派兩個人作代表到周朝請兵。武王頭一次到孟津觀兵回去了，然後又搞了兩年，他說可以打了，因為有內應了。紂王把比干殺了，把箕子關起來，但是對微子沒有提防，只曉得他是個反對派，不曉得他裡通外國。給紂王翻案的就講這個道理。紂王那個時候很有名聲，商朝的百姓很擁護他。紂王自殺了，他不投降。微子是漢奸，周應該封他，但是不敢封，而封了紂王的兒子武庚。後來武庚造反了，才封微子，把微子封為宋，就是商丘。」（《黨的文獻》1995年第4期）。

儘管微子在商朝末年的一些表現確實讓人生疑，不管怎麼說，微子後來被周公以成王的名義封於宋，是宋國的始祖，在歷史上還是一個比較重要的人物。這是不是印證了那句「留得青山在不怕沒柴燒」？

從周公封微子於宋可以看出幾個問題：商朝對於公族管理是比較嚴的，他們只有封地沒有兵權。要不然推翻紂王就不應當是周武王，而是自己的同姓王。微子、箕子、比干如果有自己的武裝力量，就會聯合起來對付紂王，不會輕易被殺、被囚以及被逼逃跑。微子也不會投靠周國來自保。微子投降之後，武王並沒有給他加封，只是恢復了他的舊封地。直到成王時，因為蔡叔度、管叔鮮與武庚碌父造反，周公才想到讓微子去管理投降的商朝人，這是因為周公知道微子不會造反，而且他是商朝的貴族，商人比較認可他。讓他管理商朝人是最佳人選。

宋國因為是商朝貴族後代的封國，所以是春秋以來唯一保留商朝貴族習慣的國家。後來被世人嘲笑的微子的後代宋襄公的「仁義之師」，其實就是這種現狀的反映。也就是說宋襄公犯的是貴族病。
