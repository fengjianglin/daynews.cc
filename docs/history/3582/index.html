歷史上的才女不在少數，也有很多流傳到今天的人物，但又是才女又是美女的卻不多，但是卻有這樣一位女子，不僅有才華，長的十分美麗，就連兩個皇帝，甚至是兄弟關係，都能為了她互相殘殺，可見她本人是多麼有魅力，她就是著名的花蕊夫人。

其實歷史上，在五代十國里有幾位共同被稱為花蕊夫人的女性，他們不僅美麗，而且才學出眾，都是因為容貌出色，才有了這樣的稱號，最著名的就是後蜀國的皇帝的妃子，也有人說姓徐，後來被封為慧妃，因為長相貌美如花蕊，還是蜀地四大才女之一，寫文章，作畫都很有天賦，後來因為國家敗落，就被宋太祖俘虜了。

宋朝太祖，也就是我們熟知的趙匡胤，在打敗蜀國之後，看到了這位夫人，覺得她很漂亮，就想將她帶回自己的皇宮，佔為己有，對於一個俘虜，又是一個女子來說，她沒有自主選擇的權力也沒有反抗的能力，自然也只能認命，就這樣跟了他，成了他的妃子。

其實皇帝的弟弟也很傾慕她的美貌，在攻打蜀國勝利之後，送她們回皇宮的時候就想動手，不過花蕊夫人不願自甘墮落，用計巧妙躲開了，也許是因為這樣才讓他們兄弟二人更加有隔閡，也才出了後面的事，弟弟對這個皇帝哥哥很是不滿。

後來到了傳皇位的時候，皇帝按照常理是要把它交給自己的兒子的，自己有兒子，皇位肯定是不管弟弟的事了，但是弟弟偏偏就盯上了這個位子，就請求十分受寵的花蕊夫人替他說幾句好話，在皇帝耳邊說說，希望能把皇位傳給他。

花蕊夫人本來就是俘虜，是迫於無奈才進了宋宮，對於這種事自然不在意，而且她深知弟弟的野心，她是個爭議的人，便拒絕了他的要求，還和皇帝說最好把皇位傳給兒子，後來弟弟知道這件事便對她懷恨在心，就找機會殺掉了她，還編出了借口說是對皇帝好。

其實花蕊夫人死因一直是個謎，有人說是因為弟弟因為得不到她，後來又被拒絕要求才懷恨在心殺掉她，又有人說是因為弟弟不滿她一人專寵，所以才找機會射殺她，但還有人說，當時被殺掉的其實不是她，是另外一位夫人，還有人說她是病死的，真真假假總有太多說法。

她本身的存在也很有爭議，有人說道她其實一直記著家國讎恨，是故意用自己的美貌來挑起皇帝兄弟二人的吃醋，讓他們反目成仇互相殘殺，來達到報殺父和滅國之仇的目的，不管怎樣，她確實是一個美麗又有才華的女子，留下了優美的詩文令後人難忘。
