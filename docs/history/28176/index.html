最富戲劇性的戰爭，戰局在最後一刻徹底逆轉，竟只因一群“野人”

2019-12-22 3,063條評論 4,910次閱讀 1440人點贊

原標題——

韓原之戰：「野人」打仗就是野

（2）最富戲劇性的戰爭

公元前645年，應該已經度過饑荒災難的秦國開始大舉進攻晉國，秦穆公任命邳豹為將軍，他本人也親自隨軍出征。秦軍初期進展順利，在河西地區連續三戰擊敗晉軍，接著渡過黃河，打到了晉國的韓原地區（即今天的陝西韓城）。

晉國在應戰前發生了幾件極為有趣的事情，實在值得一說。晉惠公見姐夫來勢洶洶，感覺不妙，連忙召開軍事會議。會上他問大夫慶鄭：「秦軍已經深入我國境地，該怎麼辦呢？」這個慶鄭極有性格，他早已看不慣晉惠公的所作所為，這個時候竟然公開回懟：「您問誰呢？您讓秦軍護送您繼位，然後就背棄了割地的承諾；晉國饑荒的時候秦國大力救援，去年秦國饑荒了您卻背棄恩義，不僅不幫反而要趁機討伐人家；現在人家打來了，難道不是很應該的事情么？」這個事還有個背景，那就是去年討論晉國該不該支援秦國糧食的時候，慶鄭認為當然是義不容辭，但晉惠公卻否決他的意見，反而聽從了舅舅虢射的建議，所以才導致當前的困局。因此慶鄭在會議上又將了晉惠公一軍：「我不知道該怎麼辦，您還是去問您的舅舅虢射吧，您去年不就聽他的么？」晉惠公對慶鄭當場撂挑子很不滿，但慶鄭又確實占理，因此他只說：「你這不是責怪我舅舅么？」但他恐怕想不到，這絕不是慶鄭最後一次撂挑子。

戰前占卜的時候，對了，那時候打仗還流行占卜，而且占卜的內容十分詳細，甚至細緻到後人都難以想像的地步。這也是西周、春秋時代和後世的巨大不同，光看軍事戰爭就能看出其中的巨大差異來，所以在這個系列初期的經典戰例中，我們必須要把這些涉及到的不同詳細的說清楚，尤其不能盲目地用後世的觀點和認知來套用、推測這個時代的戰爭，不然很容易會得出離譜的結論。

據占卜的結果，最適合做晉惠公「車右」的人竟是慶鄭。之前《長勺之戰》一文中我們就已經提到過這個詞，現在具體說說，當時的車戰，每一乘戰車上一般有「甲士」三個人（有時也有四人，但較少見），左邊的人叫「車左」，主要負責射箭即遠距離攻擊；中間的人叫「御者」或「御手」，主要負責駕馭車輛；右邊的人便叫「車右」，主要負責拿著長兵器進行近距離格鬥。孔子所謂的「六藝」——禮、樂、射、御、書、數，其中射和御便是車戰中車左和御者的必備技能。而車上三人中，「車左」地位比較尊貴（戰爭中以左為尊，不同於其它領域的以右為尊），又稱為「甲首」，還要負責全車及該車附屬步兵的指揮。尤其是國君所乘的戰車，國君一般就自動成為「車左」。這個時候就要給他配備御手和車右了，而這個配備不是光看誰的能力強，也不是一直固定人選，而是每次戰爭前都需要占卜的。

因此，這次占卜的結果就比較打臉了，竟然只有慶鄭做車右最或者才吉祥。晉惠公聽說後認為慶鄭在會上「出言不遜」且對自己不恭順，所以取消了慶鄭作為「車右」的資格，取而代之以自家的仆徒。這種不遵守占卜結果的事情歷史上肯定發生過，但也可以斷定是小概率事件，因此這對慶鄭來說無疑是一個侮辱，很容易傷害慶鄭作為貴族的自尊心。畢竟能給國君做車右，和國君一同拼殺，顯然是一個不小的榮譽。何況，巫師的占卜也不會是空穴來風，不負責任的估計，慶鄭應該多次做過國君的車右，戰鬥能力突出，所以才會被卜中。

同時，晉惠公的戰車所用的戰馬，不是晉國本地產的，而是來自鄭國。在慶鄭看來，這也是大問題，他勸諫晉惠公說：「自古以來，打仗都是用本國的馬駕車。這是因為本國的馬熟悉水土道路，又被調教馴服，無論怎麼指揮都很如意；現在您居然用鄭國的馬，這些馬只是看起來強壯而已，其實外強中乾的。一旦打起仗來，這些馬可就不會那麼容易聽指揮了。到時一旦出問題，您恐怕就要後悔了！」但這個時候晉惠公已經對慶鄭心存排斥了，怎麼可能聽從他的建議？而且這個記載說實話，雖然不能隨便懷疑其真實性，但總覺得有些太過湊巧，因為後面晉惠公確實是栽在了他的戰馬上。

公元前645年九月，秦晉雙方在韓原列陣，準備開戰。戰前晉惠公派出正卿韓簡試探秦國的虛實，韓簡觀察後回報說：「秦軍人少，但鬥志高昂，我們人多，但士氣低落。」晉惠公故意裝作不懂：「哎呀，這是因為什麼呢？」韓簡估計心裡都在想：「這還用說么？」但他可比慶鄭「講政治」多了，一本正經地回答：「您早先被驪姬陷害，出逃的時候秦國幫助過您；後來您繼位的時候秦國又幫助過您；尤其是前年，咱們鬧饑荒還吃過秦國的救援糧。秦國三次施恩都沒有得到報答，所以秦國才打來。現在咱們居然還敢迎戰，秦國軍士因此更加義憤填膺，所以士氣旺盛。咱們不佔理，士氣自然低下了。」

這裡要重點強調一下，當時的戰爭還是普遍遵循「軍禮」傳統的。而按照「軍禮」，戰爭必須要師出有名，就晉惠公的「極品」表現，秦國以此來討伐晉國是完全正義的行為，甚至都不能說是侵略，而晉國按照傳統，是該老實認輸受罰的，它不如此反而敢於迎戰，實際上已經是輿論上的錯上加錯。這一點後世可能會覺得不可思議，但這確實是當時世界的通行規則。所以韓簡才說，晉國的迎戰行為導致秦國軍士更加氣憤，原因就在此。接下來我們就會寫到宋襄公的「泓水之戰」，到時會結合這個最具代表性的戰例對「軍禮」展開詳細分析，這裡且按下。

晉惠公豈會真不了解當前的局勢和輿論？但指望他「幡然悔悟」顯然是奢望，相反，晉惠公此時更是將厚臉皮發揮到極致：「哎呀，我也知道是如此，但是既然秦國都打來了，咱們連迎戰都不做，豈不是被秦國看輕了？連一個普通人都不能忍受輕慢，何況是一個國家呢！」那意思就是厚著臉皮打唄。

因此晉惠公派出使者向秦穆公下戰書：「姐夫，您的恩德我可從來沒忘啊。我現在帶著比您軍隊還多的多的士兵來，這不是要迎戰啊，你可不要誤會。這只是他們要聚集在一起，我想解散他們，但他們都不同意。所以呢，我希望你能退兵，如果你不退，那我估計這些士兵不會逃避你的攻擊。」這番說辭大有之前道歉說辭的無賴風采，說不定都出自一個筆杆子，而且這番說辭軟中帶硬，大有炫耀優勢兵力、讓秦穆公知難而退的意思。

然而秦穆公已然恨透了這個小舅子，他此次揮師攻伐，正是要徹底教訓他一下，豈能就此半途而廢？因此即使總兵力處於弱勢，秦穆公也不顧了，他拿著鏤花的兵戈，向使者表現出決戰的架勢：「你小子現在翅膀硬了，厲害了，好啊，那你們就擺好陣勢，我一定前來領教！」

韓原之戰由此爆發。關於這場戰爭，歷史記載的較為粗疏，有些地方甚至並不能完全合榫。我們結合各種資料以及實際推演，來大致還原一下。首先，雖然秦軍方面佔據正義和輿論優勢，然而這種優勢能否轉化為戰場上的勝勢，那恐怕是沒有什麼必然之理。戰鬥發起後，秦軍憑藉著高昂的士氣，晉軍憑藉著多數的兵力，雙方應該打的勢均力敵。

但很快，正如慶鄭戰前指出或者「烏鴉嘴詛咒」的那樣，晉惠公的戰馬果然出問題了。估計確實是被戰場驚嚇到了，那四匹鄭國戰馬不僅不聽御手指揮，反而混亂間將晉惠公的戰車陷在了泥濘爛泥里，無論進退還是盤旋，都開不出來。御手估計也拿那四匹「洋馬」沒辦法，怎麼控制操作都沒用。周邊的秦軍見勢自然上來圍攻。也是巧合，慶鄭的戰車正好從旁邊經過，晉惠公估計也是著急了，早忘了兩人之前的矛盾，招呼慶鄭前來相救，甚至有讓慶鄭當御手的打算。看來慶鄭不僅當車右很優秀，當御手也很在行。然而晉惠公忘了，慶鄭卻沒忘，雖然戰場危急，但慶鄭竟然選擇了看戲，甚至當場奚落晉惠公：「你違背占卜，剝奪了我車右的資格；又不聽勸諫，固執使用鄭馬。現在你打敗仗是咎由自取，還有臉向我求救？」然後慶鄭就施施然地離開了這個局部戰場。晉惠公臉都綠了，這回算是極品遇上極品了。

那邊秦穆公聽說晉惠公陷在了泥潭裡，已經被秦軍包圍，就立即驅車向這邊戰場趕來，想生擒晉惠公。然而秦軍車寡兵少，秦穆公這一貪功冒進，很快就脫離了大部隊，反而也陷入了周邊晉軍的包圍。看來論起送人頭，小舅子和姐夫都不遑多讓，誰也別笑話誰。

此時俯觀整個戰場的態勢就很有趣了，分別以晉惠公和秦穆公的戰車為圓心，戰場上出現有兩個大的圓圈，而中間還有兩國戰車在混戰，夾雜著慶鄭在兩個圓圈中間的戰場上穿梭。至於慶鄭到底在幹啥，我們待會再說。顯然，這個戰場形勢很明顯，就看誰能先擒住對方的國君了。

此時包圍秦穆公的晉軍局部戰場指揮者是正卿韓簡，估計他的戰馬是晉國馬，因此不僅發揮正常，還抓住戰機圍住了秦穆公。據載他的御手叫梁由靡，車右為虢射。後者正是晉惠公的那個一直主張攻打秦國的舅舅，這時終於逮到秦穆公，那哪還會客氣？奮勇猛攻之下，秦穆公都被攻擊負傷，眼見著就要被晉軍俘虜了。而一旦他被擒住，秦軍士氣必然低落，沒有了士氣依憑，兵力又不如對方，秦軍戰敗勢成定局。

然而就在韓簡、虢射猛攻即將得手的時候，慶鄭吭哧拉歪地趕了過來（估計他駕車的技術確實厲害吧，所以能輕鬆越過兩個圓圈中間的眾多戰車），向他們講：「你們別忙著打了，還是先去救救你們的國君吧！」原來他雖然選擇不救晉惠公，但終究心裡過意不去，因此前來告知韓簡和虢射，希望他們去救晉惠公。

然而他這點未泯僅存的「良心」，竟然反而壞了事，耽擱了晉軍生擒秦穆公的時機。正在韓簡猶豫之時，也就是秦穆公和整個秦軍最危險的時刻，戰場上突然衝進一支部隊，彷彿從天而降一般，剎那間改變了整個戰局！

只見那支部隊不過三百餘人，或是騎馬，或是步行，雖然沒有戰車，但來勢洶洶，勢不可擋，一頭就扎進了包圍秦穆公的晉軍包圍圈，很快就將包圍圈沖開，化解了秦穆公迫在眉睫的危機。其後那三百人又奮勇拼殺，英勇無比，他們賓士所到之處，晉軍難逆其鋒，秦軍也備受鼓舞，聯合攻殺，晉軍大敗而走。晉惠公最終也沒把戰車從泥里開出來，只能淪為俘虜。

原本該以秦穆公被俘、秦國戰敗結束的韓原之戰，就這樣轉眼間以晉惠公被俘、晉國大敗而告終，真是讓人哭笑不得，實在是想說不戲劇性都不行！

而製造這一戲劇性戰例的三百勇士，便是當時秦國的「岐下野人」！這也是「野人」第一次登上中國軍事戰爭史的舞台，以如此戲劇性的方式！
