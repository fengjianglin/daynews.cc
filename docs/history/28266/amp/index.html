古代盜墓賊們都有哪些神奇的盜墓方法？

天天要聞
2019-12-21

盜墓也是一種學問，古代盜墓賊在這方面的經驗很多。傳說，過去經驗老道的骨灰級盜墓賊，根本不像現代盜墓賊這樣用洛陽鏟、探針來探測地下墓葬，僅通過占卜，就能知道地下是否有寶藏，都有哪些寶物。

真的假的？這種盜墓手法，說來太神了其實，與」風水術「一樣，「占卜術」也是中國古代影響深遠的方術之一。

據古人筆記、官野史料記載，在古代確實有盜墓賊使用過這種特殊而又神奇的手段盜墓。清代文人袁枚輯錄的《子不語》中「掘冢奇報」故事主角朱某，就是靠占卜盜墓而發家致富的：「杭州朱某，以發冢起家，聚其徒六七人，每深夜昏黑，便持鋤四齣。嫌所掘老多枯骨，少金銀，乃設乩盤，預卜其藏。」

盜墓者為什麼會想到使用這種特殊手段來指導盜墓？也許是因盜墓有時也得靠碰運氣。

《子不語》所記雖然是類似現代的玄幻小說，但用占卜手段來盜墓，其實並非無稽之談。因為，「占卜」與「風水」一樣，在過去一直是相提並論的，源出一處，師出同門。可以用這些手段來擇墓址、定下葬，當然也可以用之盜墓，通過如何葬明白如何盜。

占卜這種手段的出現，也相當傳奇，歷史上有不少這方面的師爺級人物。這裡，多聊幾句。

古代風水先生在選擇墓穴時叫「卜擇」。《葬經》稱，「葬者，藏也，欲人之弗得見也。」又有「卜其宅兆，而安厝之」之語。

這種把占卜與風水結合起的「點穴」手段，為歷朝皇家使用。《隋書》(卷78)便有這樣的文字，「及獻皇后崩，上令吉卜擇葬所」。

《葬經》的作者據說是東晉時的郭璞。郭璞就是位既擅風水，又懂占卜堪輿高人，史稱，「璞好經術，博學有高才，而訥於言論，詞賦為中興之冠。好古文奇字，妙於陰陽算曆。有郭公者，客居河東，精於卜筮，璞從之受業。」

《晉書·郭璞傳》(卷72)記載，郭璞母親去世後，將其「卜葬地於暨陽」，這個地方緊靠水邊，不過百步之遙。

當時人看了都說離水太近了。郭璞回答，「當即為陸矣。」果然不久以後，此地被河水帶來的泥沙淤積成沙灘，郭母墓的方園十來里都變成了桑田。

南北朝時間宋會稽太守張裕家的祖墳就是郭璞看的。《南史”張裕傳》(卷31)記載，郭璞當時看了兩處，一處是「年過百歲，位至三司，而子孫不蕃」，還有一處是「年幾減半，位裁卿校，而累世貴顯」。

祖上選擇了後者，張裕因祖墳風水好，當上了大官。從郭璞的選擇風水寶地的記錄可知，占卜被用來盜墓，在古人看來是頗有道理的。

郭璞不只會相墓地，還會相墓測病。當時揚州有一個叫顧球的高官，其姐姐一病不起，長達40年。顧請郭璞到家占卜，卜得「大過」卦，稱「大過卦者義不嘉，冢墓枯楊無英華。

振動遊魂見龍車，身被重累嬰妖邪。法由斬祀殺靈蛇，非己之咎先人瑕。」顧家的祖墳不好，連墓樹都枯黃了，姐姐還有病。原來是顧家祖上殺過一條大蛇。後來，顧家依言給蛇燒了紙，祭祀一番後，顧姐姐的病真好了。

更有意思的事還有——郭璞當年曾利用其所擅的占卜方術搞到了一個女人。

郭璞這個人確實很有水平，才高八斗，但他有一個缺點，就是特別好色。有個叫桓彝的朋友每次到他家拜訪時，都能看到他在女人的房間內親熱。因為好這一口，郭璞對女人特別在心，用今天的話來說，就是很會泡女人。

有一次郭璞路過廬江太守胡孟康家，看中了胡家的婢女，卻又不好意思開口討要，便弄了一出小把戲，「乃取小豆三斗，繞主人宅散之」。

據說第二天早上胡家四周出現了數千個穿著紅衣的人，但一走近就消失了。胡孟康請郭璞占卜，郭璞稱是那名婢女的問題，「君家不宜畜此婢，可於東南二十里賣之，慎勿爭價，則此妖可除也。」郭璞便暗暗派人去買了下來，「攜婢去」。這樣的記載顯然並不可信，有神化郭璞之嫌，但透露出占卜術的神奇。

占卜術很神奇，但清代杭州盜墓者朱某能卜得墓中藏物一說，還是值得懷疑的。不過，事雖可疑，通過占卜能測得墓中情況的傳說並不少見。

在郭璞之前的三國時期，曾出現過一個堪輿高人——管輅。管輅是三國時山東人，他長得很醜，而且嗜酒如命，舉止隨便，言談無常。當時有一個女人得了一種怪病，胸疼頭痛，於是請管輅去家裡占卜。

管輅占卜後說，你家屋子西邊房間的地下，埋有兩個死人，一個人手裡拿著箭，主射胸腹，導致你胸口疼另一男持矛，主刺頭，所以你頭痛。那家人刨開後一開，情況果真如管輅所說。

可見，在古代占卜可以被用來亂搞女人，可以測疾病，用於盜墓自然也不會讓人意外了，至於具體怎麼用的，我在《民國盜墓史·秘術卷》中有所交代。
