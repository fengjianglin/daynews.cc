戚繼光的老婆太厲害，都成一個老太太了，還不忘與戚繼光爭鬥一番

2019-12-24 164條評論 1,443次閱讀 322人點贊

中國古代婚姻存在五大弊端：無自主性，承嗣性，抑女性，買賣性，繁縟性。無自主性指的是毫無自主選擇權；承嗣性指的就是男女雙方結為婚姻關係的目的就是繁衍後代；抑女性指的是古代婚姻制度對於女性的束縛；買賣性指的是父母把兒女的婚姻當成買賣來做；繁縟性指的是婚前婚時婚後繁重的禮節。

以至於，在古代的時候，離婚似乎只是男人的專利，女子是沒有權利提出離婚的。

但是，中國歷史上卻有兩個奇特的女子，她們並沒有遵守當時的社會道德觀，而是做了一些「出格」的事情。其中一個是文綉，她主動離婚，將自己的丈夫溥儀給休掉了，她的故事可謂是家喻戶曉。還有一個也是主動離婚休掉丈夫，而這個被休掉的男子則是民族英雄戚繼光。

那麼，這位女子為何要跟名垂千古的丈夫戚繼光脫離婚姻關係呢？

戚繼光的婚姻非常有意思，他十三歲就定親了，二十一歲成家，娶的是萬戶南溪王將軍的女兒王氏。這位王氏非常賢良淑德，對丈夫體貼入微，很認真得照顧著戚繼光的生活起居。那時候，戚繼光還沒有做官，只是一個窮小子。有一次，王氏燒了一條魚，把魚肉都給丈夫吃，自己只吃魚頭和魚尾。戚繼光知道這個情況，抱著王氏，臉上早已熱淚直流。

王氏不僅僅賢良淑德，還非常的有勇有謀，當時，戚繼光帶著部將抗擊日本倭寇，會戰地點在台州，而王氏跟其他軍隊的家眷一起暫時居住在新河城。城內的守軍很少，根本抵擋不住倭寇的進攻。此時，正好有一支日軍向新河城趕來，將這座小城團團圍住。城外的倭寇太多，城內的家眷都驚慌失措了，唯有王氏是鎮定的，她知道現在這個情況唯有自救才能脫離困境。

之後，她說服了城內的官兵，讓城內的家眷都穿上戚家軍的衣服，站到城頭上面去。倭寇們一看，非常驚訝，好奇這些戚家軍是怎麼進到城內的，還是說城內本來就有很多守軍？看到這麼多軍人，倭寇們害怕了掉頭就走。王氏上演了一版「空城計」，將兇惡的倭寇給嚇退了。後來，戚繼光帶著部隊千里奔襲、前來增援，這才發現倭寇早已退去。戚繼光有點不相信自己的眼睛，這些人竟被一群女子給嚇退了！

由此，他的夫人王氏名聲大震，整個戚家軍都知道元帥的夫人有勇有謀，就連北京城也知道了這個事情。這位智勇雙全的夫人，連戚繼光自己也是非常服氣的。

王氏在生活起居上照顧戚繼光，在事業上幫助戚繼光打敗日本人，保護了我們國家的東南沿海。後來，戚繼光被調到北方，負責鎮守長城要塞，成為了萬人敬仰的大將軍。此時的戚繼光被世人稱為國家的「萬里長城」。可是，他和王氏的婚姻也因為戚繼光的北調而走向衰亡。

那麼，這是怎麼回事呢？

王氏有一個問題，就是無法生育，不能為戚家留下後代。不得已，三十六歲的戚繼光納了一房小妾。王氏雖然跟傷心，卻不得不接受這個事實，畢竟，那時候還是「無後為大」的年代。只是，戚繼光並沒有停止納妾之事，他娶了第一房小妾後，第二年又娶了第二房；等到他四十八歲時，他還娶了第三房小妾。

這完全不顧夫人王氏的感受呀，肯定傷害了這個倔強的女人。戚繼光雖然身為大將軍、民族英雄，但是，他的生活習慣與心裡狀態已經大不如以前默默無聞的時候了。王氏終於忍無可忍了，有一天，她拿著一把刀，打算去殺了自己的丈夫解恨。

戚繼光知道妻子王氏的來意，也知道是自己理虧，不敢反抗，只能東躲西藏，避免被她找到。只是，戚繼光經歷過這個事情後，還是不知悔改，繼續著自己的荒唐生活。晚年的王氏也就打包好自己的行李，回了娘家，正式跟戚繼光斷絕了關係。王氏大義凜然得休了自己的丈夫，這在古代是非常少見的。

王氏不願意就這樣將就下去，毅然決然得選擇了離開，我們不得不佩服她的勇氣和對生活追求。我們知道，她並不是一個強勢的人，她能夠將魚身給丈夫吃，已經是體貼入微了；她也不是一個弱女子，能夠給敵人上演一出「空城計」，足見她是有勇有謀之人；她更不是一個小女子，她跟隨丈夫東征西討，平定南方的倭寇，抗拒北方的騎兵，這都不是一個小女人能夠吃得消的。

王氏雖然賢惠、剛毅，但是，她也有自己的自尊心，不會隨意讓別人糟蹋。既然戚繼光不懂的珍惜她，那麼，她就離開，即使對方是民族英雄又怎樣！這就是一位「敢愛敢恨」的奇女子！

明朝有一首曲子，叫《雁兒落帶過得勝令》：

雲來山更佳，雲去山如畫，山因雲晦明，雲共山高下。

倚杖立去沙，回首見山家。

野鹿眠山草，山猿戲野花。

雲霞，我愛山無價。

看時行踏，雲山也愛咱。

它的內容寫的就是一位敢愛敢恨的女子。曲詞剛開始是從一個妻子的角度出發，描述一段年輕夫妻之間的甜蜜生活。可惜好景不長丈夫有了外遇，而妻子只能夠獨守空房。男子在外面魚水歡樂，女子蒙受恥辱，她沒有選擇沉默，而且毅然反抗，只為維護自己的尊嚴和權利。這個曲子，正是王氏真實的寫照。

她的愛情是轟轟烈烈的，對丈夫的恨意也是痛快淋漓的。王氏把自己的一切都給了戚繼光，可是，換回來的確是戚繼光的無情無義。王氏對丈夫很是失望，她整天生不如死，直熬到白了頭。可是，就算是身體不行了、需要別人照顧的時候，她還是會離開戚家，回到自己原來的那個家，以維護自己的尊嚴。

因為，在她看來，尊嚴比任何都重要，她不能夠任人擺布。

我們可以想像，一個風燭殘年的老太太，因為飽受婚姻的煎熬而變得紅顏不再、疾病纏身。她一直在掙扎，希望能夠衝破這個死亡的婚姻，為此她還不惜以離婚的代價來挽回自己的尊嚴。當初，她面對成千上萬的倭寇時她需要保衛自己的尊嚴，後來，她面對名垂千古的民族英雄時她還需要維護自己的尊嚴！

戚繼光直到現在，都是歷史上有名的民族英雄。可是，他的妻子王氏呢？我們肯定很少聽到她的名字了，她給歷史留下了「敢愛敢恨」四個字，她才不管對方是誰呢。
