張作霖的9句經典語錄，展現老帥愛國心切

2019-12-17 3,632條評論 4,633次閱讀 3639人點贊

張作霖的出身算是貧苦農家，自幼父親就離開了他們。張作霖小的時候迫於生計學起了獸醫，也許是他特殊的經歷成就了他後來的東北王。早年張作霖投身軍隊參加過中日甲午戰爭，後來他協助清廷剿滅了土匪杜立三的勢力。由於他的精明能幹職位步步高升，先後擔任奉天督軍、東三省巡閱使，最後成為北洋軍奉系首領。第二次直奉戰爭勝利後，張作霖率部進入北京，任陸海軍大元帥，實際上成為國家最高統治者 。張作霖曾多次抵制日本人的拉攏，被日本關東軍製造的皇姑屯事件炸成重傷不幸逝世。他留給後世的九句經典語錄，也是值得我們深刻學習的。

1、我張作霖沒有別的能耐，但替國家守護這點土地，還敢自信。

2、我唯一的志向，就是把國家治理好，使國家能屹立於世界國家之林。

3、你糊弄老百姓，老百姓就糊弄你。到頭來，老百姓反了，咱也就下台了。

4、大丈夫生在天地之間，凡事恩怨分明。我是有恩必報，有仇不饒。

5、做馬賊、土匪都無關緊要，成則為王敗則賊。混出了名堂就一切好說，但千萬不能做漢奸，那是死後留罵名的。

6、世界的潮流也不能不看一看，現在世界政治的趨勢，還允許由共和變為君主嗎？說我要復辟，那是報紙放屁。

7、土匪做大了，就是皇帝。皇帝做壞了，還不如土匪。關鍵看他會不會混。

8、人家都說我張作霖有錢，其實我哪裡有多少錢呢？你們大家打聽去，哪個外國銀行有我張作霖存的錢？哪個外國租界里有我張作霖蓋的樓房？便宜也要便宜中國人，我不能便宜外國人。

9、他媽拉巴子的，你們好好乾！咱們奉天什麼都有，干好了，我除了老婆不能給你們，什麼都可以給你們！
