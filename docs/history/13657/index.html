年入8000萬白銀，北洋水師軍費卻不敵日本，多十年還少花1000萬

2019-12-11 1,536條評論 4,469次閱讀 1155人點贊

甲午海戰曾經是中華民族之殤，強大的艦隊輸給旁邊一直被認為是蕞爾小國的日本。其實，有個誤區就是，一般人會認為清朝體量巨大，國家收入會遠高於日本。所以，對於北洋海軍的軍費也高於日本。實際情況不是這樣，到底有多大差距呢？

我們先了解一個事實，清朝海軍不只是有北洋，但是與日本作戰的只有北洋。而說北洋的軍費，不代表海軍的軍費。不過，由於南洋海軍太過弱小，所以軍費大頭還是在北洋一塊。梁啟超《李鴻章傳》曾經說過：「以一人敵一國。」李鴻章是淮軍領袖，整個北洋水師以及附屬的遼東和山東海防都是李鴻章淮軍管轄。清朝末期的派系林立使得與日本戰爭中，只有淮軍參戰。淮軍是鎮壓太平天國時期崛起的地方武裝，這支武裝在太平天國平定後就被大規模裁軍了。到了後來，清廷發現雖然太平天國鎮壓了，國家還是不太平。比如淮軍，為了保證大清海疆安全又不得已啟用淮軍。而北洋水師很多人就有著淮軍背景，比如作為水師提督的丁汝昌就是原太平天國將領投降了淮軍。那麼，說到北洋海軍軍費，大清有多麼吝嗇呢？

自1875年北洋水師謀劃開始創建，到1894年也就是甲午海戰開戰以來，在大清朝廷這20年的日子裡，包括海防經費和海防經費之外的各類開支。這裡面還是要算入海防基地的建設這類開銷，北洋水師總共花了約3796萬兩白銀，平均一年差不多是190萬兩。這筆錢看起來很多了，但是要知道當時清政府一年財政開支約8000萬兩算，這才占每年財政開支比例約為2.375%。這樣對比是不是覺得就很少了。這筆賬這麼算可能還不太直觀，因為大清是個大國。這麼大的體量，需要用到錢的地方多了去了。肯定會有人說，這也不少了。那麼，看看日本對於海軍投入有多少呢？

日本從1886年至1893年這個時間，和清朝建海軍的時間少，但是一直都很大的投入。海軍軍費每年的比例一直可都是要超過10%的。在這8年時間裡共開支海軍軍費達到了7600萬日圓，約合5066萬兩白銀，是北洋水師20年總投入的1.3倍，這筆賬如果平均到年的話，可就是3.3倍。很多人覺得平均到年不太公平，因為日本建設時間短，中國建設時間長。而海軍的艦艇由於更新換代比較慢，後期長期是維護，維護費用較少，所以平均不公平。但是，就是比總量，日本投資也是遠高於北洋海軍。北洋海軍在1888年正式成立，根據美國人的《美國海軍年鑒》艦隊實力曾是亞洲第一，世界第九。我們看看前八名都是誰？分別為：英國、法國、俄羅斯、德國、西班牙、奧斯曼土耳其、義大利、美國。可以說，當年建成的北洋海軍在亞洲範圍內基本還是很強大的。

除了海軍軍費投入不夠，內部的爭鬥也在持續升級。在清朝帝黨和後黨的爭鬥也在持續升級，最有意思的是翁同龢的帝黨是主戰的，可是並沒有掌握軍隊。所以，甲午海戰中還是要靠主張和談的李鴻章這些後黨出戰。正是這些種種的問題，才致使北洋水師的覆滅。
