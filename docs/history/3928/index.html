三國的神人之一，就連曹操孫策都害怕，更別說諸葛亮了

2019-11-28 0條評論 18次閱讀 0人點贊

摘要

在三國時期雖是戰亂紛紛，但是這一時期也是人才輩出。說到三國的人才，一定都會想到諸葛亮，從歷史上來看，三國時期的諸葛亮已經算是少見的謀士了，不僅足智多謀，還通天文，曉地理，是傑出政治家，軍事家，文學家，書法家，發明家。最著名的發明有木牛流馬、孔明燈，並且改造連弩，叫做諸葛連弩，可一弩十矢俱發。而且在諸葛亮隱居的時候，劉備的三次請諸葛亮出山，可見諸葛亮在三國時期的重要，不愧是三國中的奇才。但是除了諸葛亮之外，三國中還有一位可與諸葛亮媲美。雖然不及諸葛亮有名氣，但是在奇才方面一點也不比諸葛亮差。這個人就是左慈。

話說在三國中曾有三位神人，一個是給張角《太平要術》的南華仙人，第二個是擁有《太平經》的于吉，最後一位就是擁有《遁甲天書》的左慈。其中最具獨特的就是左慈，尤其是他的《遁甲天書》，此書歲看起來不起眼，但此書的上卷名『天遁』，中卷名『地遁』，下卷名『人遁』。天遁能騰雲跨風，飛升太虛；地遁能穿山透石；人遁能雲遊四海，藏形變身，飛劍擲刀，取人首級。左慈，字元放，是廬江人，道號烏角先生，少居天柱山，研習煉丹之術。明五經，兼通星緯，學道術，明六甲，據傳說差使鬼神，坐致行廚。

據記載左慈年輕的時候就很有神通，曾經出息曹操的宴會，曹操說：到場的都是貴客，雖然山珍海味不少，卻少了吳國松江中的鱸魚做的魚末子，這時左慈要了一個裝滿水的銅盤，用竹竿安上魚餌在銅盤中垂釣。一會兒的功夫，便釣上了一條鱸魚，曹操和在場的眾人都驚訝不已，紛紛鼓掌。可是一條魚怎麼夠宴會上這麼多人吃呢，於是左慈又下餌釣魚，過了一會兒，就又釣上一條和上一條一模一樣的，都是三尺多長，模樣很是找人喜愛。曹操便上前親自將它做成魚末子，賜給所有在場的賓客吃。曹操此時又說了：有了鱸魚，但是沒有蜀國的生薑作為佐料。左慈道：這容易。可是曹操怕是他從近處買，便故意刁難說：我曾經派人去蜀國買錦緞，你可以讓人去告訴我派去的人，讓他多買二端。左慈走了一會兒便帶著生薑回來，又對曹操說：在賣織錦的店鋪里見到了您所派遣的人，我已經告訴他多買二端。一年多之後，曹操委派的人果然多帶了二端織錦。此人告訴曹操，曾經在店鋪遇見一個人，他把您的命令傳達給了我。

還有一次曹操到郊外遊玩，陪同他的人有一百多，可是左慈就只帶了一瓶酒和一片干肉。左慈親自給大家倒酒，並向大家敬酒，官員們無一不吃飽喝醉的。曹操就覺得奇怪，派人查明此事，並走訪了賣酒的商鋪，原來他們的酒和干肉昨天都丟失了，曹操因此大怒，打算暗中殺掉左慈。正在曹操追捕左慈的過程中，他竟直接退到了牆壁里不見了。曹操決定懸重金搜捕左慈，據說有人在集市上看見他，正要逮捕的時候，集市上的人都和左慈一模一樣，根本在分不清哪一個才是真正的左慈。在後來的追捕中又有人在河南城山頂遇見他，並進行追趕，情急之下左慈就逃進了羊群。曹操自知抓不到左慈，就命令手下到羊群中去告訴左慈，已經不再追殺他，只是想試試他的道術罷了，如今已經得以證實，所以只想和他見上一面。這時，忽然有一隻了；老公羊屈起前腿，像人一樣站了起來說道：何必忙亂成這樣。正當他們準備要去捉這隻羊的時候，這幾百隻羊都變成了公羊，和之前那隻做起了一樣的動作，說一樣的話。慌亂中他們又不知該捉哪一隻好。從此之後曹操知道的左慈的本事，便不敢載惹左慈。

左慈不被曹操追殺之後，就開始雲遊四方。當他走到孫策地盤的時候，聽說孫策殺了一位術士，左慈對孫策很是失望。於是左慈就到孫策的身邊去勸說，可是當時的孫策是十分討厭術士的。聽了左慈的勸說孫策更是惱火，左慈一定不會有好下場的。孫策又開始在追殺左慈，可是不管孫策怎麼追捕就是捉不到左慈，這讓孫策漢室震驚，從此孫策對左慈也十分的敬重，不再反感術士。從此之後，沒有人知道左慈到底去了哪裡，有人說左慈活到了130歲，還有人說左慈成了神仙。
