漢朝外戚哪家強？有的廢立皇帝，有的當皇帝，最厲害的還是殺皇帝

2019-12-16 387條評論 2,772次閱讀 655人點贊

眾所周知，從漢朝的第二任皇帝——漢惠帝劉盈開始，我們幾乎處處都能看到外戚的影子，即便漢武帝「殺母立子」，後來的漢昭帝也還是活在「外戚」霍光的陰影里，外戚專權可以說是漢朝歷史上的一個怪胎。

漢朝的外戚從西漢的呂氏家族、衛青家族、霍光家族、王莽家族，再到東漢的竇憲家族、梁冀家族、竇武家族、何進家族，幾乎每一個都曾經擁有炙手可熱的權勢。這些家族在巔峰時可以隨意擁立一個皇帝，也可以隨時廢掉一個皇帝，實力不可謂不恐怖。

如果要說漢朝外戚哪家實力最強，也許有人會說是呂氏家族，他們一共擁立了三位皇帝（漢惠帝、西漢前後少帝），其中西漢前少帝還被呂后毒殺；還有人會說是霍光家族，他們也一共擁立了三位皇帝（漢昭帝、漢廢帝、漢宣帝），而且還廢掉了一個皇帝；也有人會說是王莽家族，他本人甚至還當過幾年皇帝。

然而小編認為，漢朝實力最強的外戚應該是梁冀家族，因為梁冀一生不但擁立了三位皇帝（漢沖帝、漢質帝、漢桓帝），而且還毒殺了其中一位。

下面我們就來介紹一下這位歷史上被稱為「跋扈將軍」的梁冀。

梁冀是東漢漢順帝時期大將軍梁商的兒子，他的妹妹梁氏是漢順帝的皇后。漢順帝去世後，其年僅一歲獨子劉炳繼位，是為漢沖帝，由皇太后梁氏執掌朝政，梁冀則擔任大將軍（位在「三公」之上）一職。

不幸的是，年僅一歲的漢沖帝在位幾個月就去世了，於是梁冀決定擁立旁支里年僅八歲的劉纘（zuǎn）為新皇帝，是為漢質帝。由於梁冀此人性格一向飛揚跋扈，經常在朝堂上頤指氣使，引起了小皇帝劉纘的不滿，於是他當著眾多朝臣的面說了一句：「此跋扈將軍也！」

梁冀聽到這句話後很不高興，畢竟他當初擁立這位小皇帝的目的是想讓其充當一個聽話的傀儡，沒想到小小年紀的漢質帝竟然如此聰慧早熟，日後恐怕難以控制，不如趁早想辦法除掉。於是他讓皇帝身邊的侍從把毒藥悄悄地放在湯餅里，漢質帝吃完後毒性發作非常難受，趕緊派人傳召自己最信任的大臣——李固。

李固進宮後，詢問質帝難受原因，意識到自己已經中毒的小皇帝說：「朕吃過湯餅，現在覺得腹中堵悶，給朕水喝，朕還能活。」站在一旁的梁冀立刻阻止道：「恐怕嘔吐，不能喝水。」意思就是說不讓皇帝喝水，話剛說完，這位年僅九歲的小皇帝就駕崩了。

國不可一日無君，梁冀很快又擁立了自己的妹夫——十五歲的「蠡吾侯」劉志為新皇帝，是為漢桓帝。大臣之中，凡是對梁冀稍有不滿的，都會被其找個借口排擠至死，比如說當時富有聲望的名臣李固、杜喬等人，所以大家對梁冀的專權只能是「敢怒不敢言」。

同樣，漢桓帝也在前任皇帝被毒殺的陰影里，整整當了13年的傀儡皇帝。算起來梁氏家族一共有九人被封侯，三人做了皇后，六人做了貴人，另外還出了兩個大將軍，夫人、女兒中有七人享有食邑，三人娶了公主，其餘官至卿、將、尹、校的有五十七人，是當時名副其實的「第一家族」。

直到有一天，漢桓帝在上廁所的時候和幾位宦官密謀發動政變（所以也被後人稱為「廁所政變」），並最終在他們的幫助下，用計謀誅殺了梁冀，還一舉除掉了梁氏家族。

對於梁冀的一生，漢桓帝給了八個字的評價：「梁冀奸暴，濁亂王室」。梁冀作為一個大臣，竟然可以「濁亂王室」，還能明目張胆地毒殺皇帝，絕對可以說是漢朝最有實力的外戚之一。

不過話又說回來，關於漢朝外戚究竟是哪家最強這個問題，每個人的想法可能都會不一樣，歡迎大家在下面留言討論。
