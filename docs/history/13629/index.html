楊麗華的這個皇后為何在歷史上與眾不同？揭秘楊麗華的傳奇一生

2019-12-11 569條評論 2,811次閱讀 1375人點贊

說起隋朝公主，不得不說的就是隋文帝的長女楊麗華，她是隋朝的樂平公主但也曾是北周宣帝的天元皇后，看過一些關於北周歷史的讀者或許會知道，在北周宣帝宇文贇傳位給宇文闡之後出現了一個非常特別的現象，那就是五後並立。我想，她的與眾不同可能就是在這裡，周宣帝為了展現自己的權力，不惜冊封了五位皇后，楊麗華在這之中又是如何表現的呢?

眾所周知，楊麗華性格潤順，不嫉妒，即使有其他四位皇后和她平起平坐，她也絲毫沒有吐露過自己的嫉妒之心，正是因為如此，大到後宮中的嬪妃小到各院的僕人都發自內心的尊重她、喜愛她。但自古帝王大多多疑，在前朝，周宣帝聽聞民間有對自己不利的傳言，在後宮，人們也各個對楊麗華尊敬有加，這讓他不禁懷疑是否是楊麗華與其父從中作祟。為此，周宣帝喜怒無常，每每無故責備楊麗華更想強加一些罪名在楊麗華身上，但楊麗華並沒有與周宣帝計較，她還是保持著自我本心，從不屈服也不做過多解釋，這樣的反應惹得周宣帝更加憤怒，便準備賜楊麗華死罪，辛虧楊麗華的母親獨孤伽羅的趕來求情，這才免於一死。

造化弄人，周宣帝這個糊塗皇帝當了不多年便染上惡疾，便讓楊麗華的父親楊堅到宮中陪伴，等到周宣帝病危，身旁的劉昉、鄭譯等人乘機假傳聖旨讓楊堅輔佐年幼的宇文闡，其實，這是對楊麗華有利的，畢竟是自己的父親嘛，與其讓其他人得手不如提拔的是自己的父親。但微妙的關係是如何一步步變化的呢，不久，楊麗華便發現自己父親的野心，這樣楊麗華心生不滿，可無奈，即使表面上自己已經表現出來不同意，但她阻止不了父親的步步緊逼，就這樣，楊堅成功的拿下皇帝之位，改朝換代為隋朝。

如我們所看到的，楊麗華的處境是尷尬的，她這樣算什麼身份呢，也正因為楊麗華身份的特殊，一直到開皇六年，他的父親隋文帝才將她封為樂平公主。其實後來隋文帝也曾提議讓楊麗華改嫁，楊麗華堅決不同意之後這才罷休。

其實，故事的前半部分是讓我們羨慕的，楊麗華她出身並不貧賤，有幸得到了周宣帝的喜愛成為了周宣帝的太子妃，後來一步一步的成為皇后，雖然五後並立，她也生活的很好，並沒有被後宮的一貫風氣而改變本心，她踏實沉穩，又由皇后成為了皇太后，這是多少女子夢寐以求的一生啊，但是，隨著她的身份驟然變化，也讓她苦不堪言。她的拒絕沒有得到任何改觀，她本可以做這個皇太后好好安度晚年，怎料，被他人奪了政權，這個人還是自己的父親，這麼說來，又不如安心做個普通人，遠離朝堂的愛情或許會比這樣幸福的多。
