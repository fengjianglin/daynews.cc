慈禧太后應憎恨英國人才對，與巴恪思相戀四年，她有什麼目的嗎

2019-12-24 1,275條評論 4,851次閱讀 2828人點贊

慈禧太后是清朝晚年至高無上，掌握清廷實權的女人。由於她的軟弱無能，不懂得變通，讓百姓遭受西方列強的欺辱，肆無忌憚的踐踏國人的尊嚴。而這位不知恬恥的慈禧，為了自己的私慾，竟然與英國男人相戀四年。

一.鴉片戰爭挑戰慈禧的權威，踐踏她的尊嚴，中國進入半封建半殖民

中國鴉片戰爭主要罪魁禍首是英國人，在1840年發生的鴉片戰爭，打開了清廷閉關鎖國的門戶。《南京條約》的簽訂，宣告中國進入了半封建半殖民地國家。對清廷和百姓的尊嚴任意踐踏。辱國喪權的清廷，實際掌權人的慈禧，沒有感到英國列強對清廷危機感。

在1860年英法分別以「亞羅號」事件與「神甫」事件為借號，發動了第二鴉片戰爭，攻入北京圓明園，大肆的燒殺掠奪，國家珍寶，文物，古文獻書籍等大量流失和破壞，慈禧攜帶清帝逃往承德。《北京條約》的簽訂後，進一步對清廷的蠶食，擴大在華的侵略利益。可恨的是俄羅斯，侵火打劫，以「調停有功」自居，脅威清廷割地100多萬平方公里。

兩次鴉片戰爭後，慈禧太后應該非常憎恨英國人才對。讓她過著逃亡的生活，英國人進入北京後，在清廷的皇宮內，燒殺搶奪，並在慈禧的床邊寫一些污言穢語「皇太后尊前，請恕我佔了您的鳳床，只可惜不能與您共寢。」

對西方列強的忍讓，而國內百姓處在水深火熱之中，外國的欺辱，清廷官吏的橫徵暴斂，苛捐雜稅的繁重，百姓苦不堪言。而躲在深宮的慈禧，享受著奢侈的生活。

二.為了自己的私慾，與英國人巴格思相戀四年

巴格思是一位英國人，懂得俄語，中文，日文等幾國語言，也是帶著侵機掠奪中國財物而來的。因為他懂幾國語言，經常遊走在清廷王公貴族之間。

巴格思是1898年來到了中國，也就是八國聯軍入侵北京的前兩年。在1900年八國聯軍入侵北京，再次清洗了圓明園，將圓明園的財物，清洗一空。慈禧與光緒帝過著逃亡的生活。

在1902年，慈禧再次回到北京。巴格思侵火打劫，得到了600件物品，其中有青銅器，景泰藍，書法作品，玉器等，還有25000卷古籍。把這些物品歸還給慈禧。慈禧對這位巴格思產生了好感，認為他與其他的英國人不一樣。

此時的慈禧已是68歲的老太婆了，而巴格思只有32歲。巴格思經常與清廷的王公貴族來往，對中國文化有一定的了解。熟悉宮廷的規矩。每次與慈禧聊天，深得慈禧的喜歡。一來二去，巴格思就了慈禧眾多男寵之一。慈禧與巴格思相處的時間長達四年之久。這只是根據巴格思《我與太后》書錄，沒有正史來考證的，他的真實值得懷疑。

三.在戰爭中無法戰敗英國人，石榴裙下征服英國人

慈禧太后，在西方列強對清廷肆意的掠奪，簽訂不平條約後。富饒的大清國，曾經以央央大國自居的慈禧，獨攬大權，至高無上的她，也被逼無奈。與大臣曾國藩，李鴻章等商談如何救國，以夷制夷的辦法，實行洋務運動。

另一方面，用石榴裙來的征服痛恨的英國人，來滿足心裡的寂寞和私慾，巴格思引以為豪的戀情，只不過是慈禧太后，對男寵征服的一種手段。

在戰爭中，你們英國人用武器攻打大清國的土地；我用肉體驅使你們英國人，還不是被我的石榴裙征服。這種阿Q的征服方式，真是很獨特。

總結：慈禧作為大清國的實際掌權人，但在西方列強的逼迫下，心中的無奈與屈辱，只能在肉體上對巴格思征服，得到一點慰藉。巴格思所寫的《太后與我》，有些地方有悖歷史事實，他與太后戀情的真與假沒有正史資料可以考究。只是巴格思的一廂情願罷了。

至於慈禧與英國人巴格思的戀情，沒有任何歷史資料來考證，只是從巴格思的書籍里得到他的片面之詞，他們的戀情是不是真實，有待以後的考證。
