髮妻死因成謎，李嘉誠大興土木，表面建樓紀念亡妻，實則另有用途

天天要聞
2019-12-19

張愛玲在她的《紅玫瑰與白玫瑰》中曾經這樣寫道，「每個男人的生命里都有兩個女人,一個是白玫瑰,一個是紅玫瑰。也許每一個男子全都有過這樣的兩個女人。娶了紅玫瑰,久而久之,紅的變成牆上的一抹蚊子血,白的還是『床前明月光』;娶了白玫瑰,白的便是衣服上沾的一粒飯粘子,紅的卻成了心口中的一顆硃砂痣。」

張愛玲看透了愛情的真相，忠貞不渝的愛情寥寥無幾，一生一世一雙人是少數，這世間多的是”吃著碗里的，望著鍋里的“的男人，多的是”有事鍾無艷，無事夏迎春”的男人，很少有人能做到”貧賤之交不能忘，糟糠之妻不下堂”，首富李嘉誠也難逃離這個魔咒。

李嘉誠第一任妻子是自己的表妹，兩人之間的關係如同紅樓夢裡薛寶釵和賈寶玉的近親關係，而關係卻像是西廂記里張生與崔鶯鶯的翻版。年輕時的李嘉誠窮困潦倒，投靠自己有錢的舅舅，富裕的鐘錶商人庄靜庵。

與表妹庄月明相見之時自然是驚為天人，心生愛意。試想一下，一個女人，出身名門，家境殷實，多才多藝，美麗大方，溫柔多情，善解人意，哪個男的會不動心？並且這個女人不嫌棄自己窮，竭盡全力幫助自己，而且背後的財力還能為自己的事業添磚加瓦，娶了這樣的男人，怕是做夢都要笑醒了。

於是，兩人互相愛慕，私定終生。然而，也像說書人的故事裡唱的那樣，兩人是不門當戶對的，尤其是在當時的年代。即使後來李嘉誠的事業慢慢建立發展起來，取得很多成就，但還是配不上自己的表妹大家閨秀庄月明。

也許是好事需要多磨，直到兩個人都30多歲時，李嘉誠的事業才開始蓬勃發展，兩個人也終於說服和感動了父母，於是終於守得雲開見月明，兩人如願以償的結婚了。當時李嘉誠一擲千金討得佳人歡心，他花費了63萬港元為妻子購買了花園洋房，這是他與她的婚姻中送給庄月明最貴重最用心的禮物。

一個成功男人的背後總有一個默默付出的女人。這句話來形容庄月明最貼切不過了，從小就受過良好教育的她才貌雙全，待人謙遜，加入李嘉誠的事業後深得人心。自從生下兩個孩子後，更是甘願退居二線，全力撫養和教育孩子李澤鉅和李澤楷。

低調又謙虛如她，很少接受採訪，很多人甚至都不知道庄月明是誰。但是鮮為人知的是在李嘉誠的巔峰事業「長江實業」中，庄月明出任執行董事，不少看起來高瞻遠矚石的決策，都來源於她這樣一個女子的智慧。她用自己的一生全部的心血換自己丈夫如日中天的事業，不管是在什麼方面，她都鼎力支持自己的丈夫，李嘉誠獲得成功就是她最大的心愿。

可是，庄月明可能忘了，也可能當時沒有想到，商人重利輕別離，男人有錢就變壞。根據坊間傳聞，李嘉誠後來出軌被庄月明發現，庄月明吞服大量藥物自殺。

雖然對外一直宣稱庄月明是死於心臟病突發，但是從兒子李澤楷搬離家庭，還有一些文學作品有意無意的指代，以及坊間捕風捉影的傳聞，還有庄月明死後李嘉誠心虛大興土木，修建風水意義十足的困魂式建築樓來紀念亡妻，讓真相更加撲朔迷離，大家對李嘉誠也是有些懷疑。

李嘉誠倒也真是個骨骼清奇的男子，按道理說不管是有意無意，反正髮妻成功退位，他終於得到了和自己情人在一起的機會，但是他並沒有再婚娶。傳聞里跟他一起氣死老婆的出軌對象是周凱旋，這個女人也是庄月明死後，唯一一個他接觸的女人。

令人迷惑的是，相伴26年，他也從未給這位周凱旋女士任何名分，即使明眼人都能看出來兩人關係匪淺，李嘉誠去哪兒也會帶著這位女人。但是被問及兩人關係時，李嘉誠總是說是”紅顏知己”。

好一個紅顏知己，男人果然都是一樣的，跟現任在一起的時候，憧憬別人的好，而狠心放棄現任之後，得償所願跟另一個在一起的時候，又開始回過頭懷緬前任的好。男人就像貪得無厭、永不知足的小孩一樣，他不管得到什麼都不能完全滿意，即使他什麼都有了卻還像一無所有一樣。

李嘉誠的這位小他33歲的紅顏知己可不是一般人，這位女性也是商界的一朵奇葩，20世紀末，互聯網行業出現發展苗頭。好風憑藉力，周凱旋借著政策和好時機打造了TOM公司，後來成功上市，她一夜暴富，成為了億萬富翁。

隨後，她又看準時機，做出將TOM私有化的重大決定，成功套現了6億。這一波操作，令人瞠目結舌。她不僅自己厲害，還幫助李嘉誠的事業添磚加瓦，給了李嘉誠很多投資方面的建議，例如建議他投資不被人看好的facebook，後來果然李嘉誠賺了個盆滿缽滿，投資6000萬美元，獲得了10億美元的回報收入，真的是翻了一翻。

像張愛玲在紅玫瑰與白玫瑰中的另外一段話所說的一樣，“最美的，都是愛情未明的時候。等到愛情盛放的時候，那極濃烈的香氣中，隱隱就聞到腐敗的味道了。女人愛到深處，便無比勇敢，一心要跟這個男人在一起，而男人面對現實卻退縮了，他想到了自己的事業、名譽，以及她的過去。」

周凱旋是一個勇敢的女人，她不圖錢，因為她已經足夠有錢，她也不圖名分，多年任勞任怨事無巨細的照顧陪伴李嘉誠卻什麼名分也沒得到。

大抵是真愛吧，她愛李嘉誠的態度卑微到了塵埃里，可是一個連髮妻死後都還要利用她的旺夫命的男人，是無情的，這是商人的本質。李嘉誠是愛惜自己的名聲才沒有給她名分，聰明如她不會不明白。但她還是甘之如飴，願意為愛付出自己的真心和芳華。

一入豪門深似海，這箇中滋味，如魚飲水，冷暖自知。誰是窗前明月光，誰又是胸口的硃砂痣，可能只有當事人知道吧！
