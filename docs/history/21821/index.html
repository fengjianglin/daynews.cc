中國最後的官窯瓷，由袁世凱一手打造，價值能比1000套北京四合院

很多人都聽說過宋朝最富影響力的六大名窯——官窯、汝窯、哥窯、定窯、鈞窯，這些瓷器精美絕倫，聲名遠播，甚至在拍賣會上賣出天價，但是很多人卻不知道中國最後的官窯瓷是什麼，是為什麼人而燒制而成的。

今天我們就來介紹一下，中國最後的官窯瓷。中國最後的官窯瓷，以袁世凱復辟的國號為名，叫做——洪憲瓷。袁世凱竊取革命果實，1914年2月，袁世凱擬《中華民國約法》來替代《臨時約法》，自己獨攬大權，就連自己位置的繼承人，也要由自己欽定。袁世凱在1915年12月12日發表接受帝位，宣布1916年為「洪憲」元年，並在元旦正式登基。袁世凱終於戴上了「皇冠」，雖然這位短命的皇帝只做了83天，但是稱帝的「排面」還是有的！

袁世凱的性格一向野心勃勃、自私專斷，他稱帝時，想要拿國之寶物來宴客，於是命令當時燒瓷手藝一絕大大師郭葆昌來制瓷，郭葆昌不敢怠慢，夜以繼日、嘔心瀝血的想盡千方百計模仿古代粉彩瓷和琺琅彩，所花費燒瓷金額高達140多萬大洋，堪稱花費天價制瓷！也許你不知道140萬大洋是什麼概念，這麼多錢能夠在當時買1000套北京四合院！

洪憲瓷的特點：胎白釉凈、無橘皮紋、彩色明艷豐滿，紋路纖細清晰。俗話說：「物以稀為貴」，由於袁世凱只當了短短八十多日的皇帝，所以洪憲瓷的數量十分稀少，並且流傳至今的更是少之又少，於是洪憲瓷在拍賣市場上也是十分的珍貴，它見證了一個重要的歷史性事件，其背後的歷史意義不容小覷。
