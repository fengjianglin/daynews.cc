24歲留學，32歲搞原子彈48年，她為何晚年執意回歸祖國？

天天要聞
2019-12-15

相比於我身邊的各種光環，我更想做我自己，我所熱愛著的是我的事業，而不是它能帶給我個光環，我努力從事著我的事業，不是為了獲獎，而是將它當成了我的生命。

在美國的曼哈頓計劃中，有這樣的一位女性華裔科學家，她為美國的原子彈的開發貢獻了巨大力量，被人稱之為「原子彈之母」，同時，她是袁世凱的孫媳婦，師從胡適，被胡適視為「滿分學生」身份地位，在那個時代堪比頂尖，可她卻用自身的實力，證明了自己，創出了自己的一片天，她就是吳健雄。

書香門第，不讓鬚眉

吳健雄出生於1912年的蘇州太倉瀏河鎮，她的家庭可謂是書香門第，父親吳仲裔在年輕時曾留過洋，思想比較開明，倡導男女平等的父親，對出生的女兒同樣寄予了厚望，按照家譜，吳健雄屬於「健」，遂以「英雄豪傑」為家中兄弟姐妹起名，吳健雄排名老二，至此，這個男性化的名字就誕生了。

因為家庭教育的環境，吳健雄得以上學識字，並且自小聰明，比起其他兄弟要優秀許多，深的父親喜愛，1927年，吳健雄畢業於一家師範學校，成為了一名教師，此時，面對待遇豐厚的教師工作。

父親卻鼓勵她繼續深造，兩年之後，吳健雄成功進入到中央大學的數學系，進行深造，多年之後的吳健雄回憶道：「如果沒有父親的鼓勵，我可能在中國某地的小學教書，這樣的生活或許更加愜意，可現在我懂了，父親要教我的，是做人做大我，而不是小我」，可以說，吳健雄的成功，與父親出色的家庭教育是分不開的。

亦師亦友，滿分學生

而相比與父親，她人生中的另外一個難忘的男人，就是胡適了，吳健雄自從在11歲那年聽到胡適那場名叫《摩登的婦女》的演講之後，就對這個男人心生敬意，當做偶像，隨著時間的推移，這份感情越來越難以自拔。

胡適和吳健雄

這裡要提一句，吳健雄考入中央大學的目的，就是因為胡適此時正在擔任這所大學的校長，而本該沒有任何交集的兩條平行線，也因為一次歷史考試，而慢慢的有了交集，身為理工科專業的吳健雄，以三分之二的考試時間答出了一百分的好成績，引起了胡適的關注。

漸漸地，兩人的交往越來越密切，雖然名義上還是師生，可其實在彼此的內心裡早已生出了不一樣的情愫，1936年，24歲的吳健雄前往美國加州大學伯克利分校留學，胡適也因為一些事前往了美國，兩人的感情更加深厚。

可胡適此時已有家室，理智的二人最終還是恪守住了底線，或許他們彼此都清楚，對方都不會是自己最好的歸宿，所以心照不宣地將這份感情埋在了心底。

1962年，在台北的院士會上，胡適心臟病發作，現場去世，目睹了一切的吳健雄久久不能平靜，直到第二天，在殯儀館望著胡適的儀容時，身體還忍不住的瑟瑟發抖，一顆心，就這樣，緩緩地墮了下去。

偶然機會，迎來配偶

吳健雄主攻數學系，後來為什麼將目光轉向了物理學呢？這其實是一個突發事件，在一次數學系的專業課上，吳健雄打開了一本有關於相對論和放射性等方面的書，面對那些玄之又玄的物理定論，吳健雄非但沒有感到晦澀，反而興起了濃濃的求知慾。

袁家騮

第二年，吳健雄申請轉到了物理學系，1936年，在吳健雄出國留學後，他人生中的配偶也來到了身邊，那就是袁家騮，兩人在異國他鄉成為了同班同學，袁家騮雖然出身名門，相比於爺爺袁世凱的野心，他更喜歡學術研究，相比於父親袁克文的拈花惹草，他更喜歡擇一人白首。

日久生情的兩人很快步入了婚姻的殿堂，1942年5月，兩人步入了婚姻的殿堂，袁家騮聽說過妻子和胡適事，可相比於萬變的風言風語，他更相信妻子，兩人的婚姻生活十分的幸福，1944年，32歲的吳健雄在袁家騮的支持下，成功參加了美國的曼哈頓計劃，參與原子彈的研究，有了「原子彈之母的稱號」。

吳健雄，這個本該獲得諾貝爾獎的人，連續將兩人送上了諾貝爾獎的領獎台，自己卻與之無緣，可吳健雄卻不這麼想，對於她而言，獲不獲獎什麼的，他根本就不在乎，她愛的不是什麼獎項，而是她的事業，值得一提的是，被她送上諾貝爾領獎台的那兩個人，就是著名的美籍華裔物理學家楊振寧和李政道。

袁家騮和吳健雄

連獲殊榮，果斷回國

隨後的幾年裡，吳健雄打破了一項又一項紀錄，1958年成為了百年來普林斯頓大學第一個獲得榮譽博士的女性，1975年，成為了美國物理學會第一位女性會長，1984年，眼看祖國的發展，吳健雄放棄了一切，踏上了回國的路程。

她將自己的積蓄共計25萬美金，捐贈給了自己的母校，1994年，當選中科院院士，在那一天，吳健雄激動地說：諾貝爾獎辜負了我，可國家沒有，相比於諾貝爾獎，我更喜歡中科院院士的身份。

吳健雄，是新時代女性的代表，她用自身的經歷，用自己的能力打破了世俗的傳統，證明了女性的光輝。1997年，八十四歲的吳健雄去世於蘇州，直到現在，每當有拜祭吳健雄得人，依然能夠看見墓碑上的那句話，「他是卓越的世界公民，和一個永遠的中國人」。

王力宏的一首歌中寫道，「遠離家鄉，不勝唏噓，幻化成秋葉」，這首歌的名字就叫做《落葉歸根》，吳健雄的一生，都在努力地追求著她自己所追求的東西，功成名就之後，她選擇了落葉歸根，在祖國最需要的時候回來，並將自己所有的積蓄捐給了祖國。

她是新時代女性的代表，是中國女性的代表，是世界華人的楷模，是世界頂尖的女性物理學家，是不惜功名、熱愛事業的雅士、是一個偉大又平凡的中國人！
