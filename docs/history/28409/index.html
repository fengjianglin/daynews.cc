害親兄，殺小侄，史上著名的美男皇帝居然殘暴至此？

2019-12-20 1,672條評論 4,641次閱讀 224人點贊

害親兄，殺小侄，史上著名的美男皇帝居然殘暴至此？

北齊，中國南北朝時代的的北朝割據政權。由東魏權臣高歡的次子高洋所建。北齊的國壽不算長，前前後後一共只延續了26年，皇帝也只出了6位，而變態的基因似乎是刻在了北齊皇室的基因里。其中出的很多變態皇帝其獸行都足以讓惡名流傳千古，甚至還成就了很多今天依舊廣泛使用的成語。

高歡的兒子眾多，其中高湛最以美艷而出名，然而長得好看卻，這人的內心卻十分醜陋，今天就讓我們來繼續給大家介紹一下高湛做下的諸多壞事。

造化弄皇

違背先帝的囑託殺害侄子之後，孝昭帝高演的內心遭受到了良心的譴責，終於鬱郁而死。良心如此，人之常情。

然而高湛就是那個例外，從他的表現看來，他是幾乎沒有良心的。

高洋死後，曾經信誓旦旦說要效忠幼主的高演，很快就和親弟弟高湛聯合，發動了政變。

而早在高演發動政變之前，他就向弟弟高湛允諾，一旦自己篡位成功，就立刻被高湛冊封為皇太弟，待到自己死後，讓高湛有機會做皇帝。也正因為如此，高湛才能冒著巨大的風險，和高演一起造反，同生死，共進退，可是等到了高演真的成功登基之後，卻違背了當初的誓言。

高演登基後，立刻違背了諾言，將自己的兒子高百年冊立為了太子，死後可以繼承自己的皇位，違背了自己的諾言，讓高湛十分不滿。然而原來高湛和高演集結兩人的力量，才能夠扳動皇權，將年幼的高殷取而代之。而如今高湛勢單力薄，只憑自己的力量，他又如何挑戰坐上皇位的高演呢？沒有辦法，高湛最終只能選擇暫時忍氣吞聲。

微妙的平衡

本來如果故事正常發展，皇子逐年長大，高演父子羽翼豐滿，權柄在握，皇位可以安然的傳到高百年手中，高湛雖然必定會有所不滿，但是也並不能做什麼。

然而誰也沒能想到，造化弄人，高演當皇帝的職業生涯居然如此短暫，上位僅僅一年多，他就被馬踢成了重傷，生命垂危。病痛中的高演心中再清楚不過了，自己如果此時撒手人寰，五歲的高百年必然會在高湛的怒火和恨意下成為下一個廢帝高殷。為了保住自己孩子的性命，高演決定，將皇位直接傳給弟弟高湛。

高演臨死之前，曾經把高湛叫道自己床前囑託到：「百年無罪，汝可以樂處置之，勿學前人。」（高百年沒有什麼罪過，你當了皇帝，希望能好好對他，不要學前人屠戮親人的行為。「高演自以為他已經將皇位給了高湛，他不會在有什麼殘害高洋的理由，可以放下對高百年的殺心。

然而就像他自己即使坐上了皇位，還是要餵給高殷一杯毒酒一樣。一個沒那麼名正言順的皇帝總是對一切可能威脅自己位置的人異常敏感。皇權的瘋狂鬥爭之下，殺紅了心的萬歲哪裡還顧得上什麼親情，何況這個萬歲還是有些變態的高湛呢？

背義殺侄

而高湛大概是從上位開始就想要找茬除了高百年的性命了，高演死後三年，他終於等到了這個期待已久的時機。

河清三年五月，出現了諸多異常的天相，朝野上下一時間人心惶惶，皇帝高湛也投入了惶惶大軍的一員，不過不太一樣的是，他八成是裝的。

對別人來說，異常的天相是災難的象徵；而對找了三年理由早就找的不耐煩的高湛來說，這可能就是天賜的良機。他當時就打算借口要進行「厭勝「的儀式，要殺死高百年來鎮服天災，聽起來略顯牽強，但也不無道理。

可能是老天爺真的想要了高百年的小命，也可能是高湛在暗中做局設套，恰巧此時，高百年的老師就捧了幾張寫著斗大字的紙，前來給高百年參了一本。他說高百年近日總愛寫「敕」字。敕字並不是一個不同的漢字，這個字和皇家的命令時時捆綁，意思是帝王的詔書。

雖然有著這層意思，但這也不過只是一個漢字而已，沒有任何實際的效用，高百年不過是個九歲的孩童，就算是無事多寫了幾遍這個字，也構不成什麼大罪。可高湛好不容易能抓到機會除掉這個「前太子」，又哪能放過這個事情呢？

十分出名的「敕造榮國府」

高湛當即暴怒，命人將高百年拿下，押到殿上來。高湛扔給他一支毛筆，讓他書寫「敕」字，果然和他的老師呈上來的證據別無二致。

可憐年僅九歲就被指造反的高百年，不由分說，當場就被高湛命令的侍衛抓了起來，在大殿上拖行。高洋曾經用劍柄抽打高百年的父親，而高湛此刻更是命令人跟在身後，用劍柄狠狠的捶打高百年，年幼的高百年被打的鮮血直流，所到之處拖得遍地鮮紅。

堅硬的劍柄

劍柄抽一下，高百年就嘶號一聲，而在挨打的間隙，他大聲哭喊著求饒道：「乞命，願與阿叔作奴！」（饒命啊叔叔，我可以給您當奴隸！）高百年可能對自己歷史也不太熟悉，北齊開國皇帝的哥哥就是被廚奴一刀刺死的，狠心的高湛又怎麼可能就此收手？他反而命令左右的人更加用力的毆打他，高百年喊得再凄慘，飛舞的劍柄也沒有停下一刻。沒多久，可憐的高百年就被活活的打死在了大殿之上。打死了還不算完，高湛還命人將侄子的屍體直接扔到了宮內的池塘里，高百年死狀凄慘，流出的血水染紅了整個池塘。

沒當上皇帝就用言語害死了兩個兄弟，當了皇帝之後又把自己的侄子活活打死，寫到這裡，高湛殘暴嗜殺的性格已經顯現了出來，然而他所做的荒唐事又絕對不止於此，具體都有什麼？下期文章，我們再來詳細給您說說。
