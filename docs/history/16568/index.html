狀元第一省 狀元第一市 狀元第一家

2019-12-16 290條評論 2,121次閱讀 619人點贊

狀元文化是中華民族文化體系中一道亮麗的風景線。在南京的尋常巷陌中，有五條「狀元巷」。自孔夫子始，「學而優則仕」是古代知識分子的奮鬥之路，「金榜題名」亦是天下讀書人的畢生追求，「狀元及第」更是「天上一輪才捧出，人間萬姓仰頭看」的巨大殊榮。江蘇歷史源遠流長，經濟富庶、文化發達，歷來重視教育，濃厚的文化氛圍也使江蘇成為「天下狀元第一省」。

天下狀元第一省—-江蘇省

自隋大業三年(607年)至光緒三十一年(1905年)廢止科舉約1300年間，全省共出狀元112名(其中武狀元19名)，約佔全國641名狀元的17.47%，為全國最多的省份。尤其是明清時期，江蘇籍狀元更是超過全國狀元總數的三分之一，多達70名，是名副其實的狀元第一省。

蘇州狀元博物館

江蘇籍狀元不僅人數眾多，許多聞名遐邇的傑出人物也出自其中。著名詞人張孝祥，詞作追蹤蘇軾，氣概凌雲，被譽為「詞作成就最高的狀元」。錢棨先以縣、府、院試三個第一考上秀才，人稱「小三元」，後在鄉試、會試、殿試中又連中三元，奪得「大三元」，共計6個「第一」，合稱「六元」，是中國歷史上唯一「六元」精英。翁同龢一生兩為帝師，也是狀元出身，在晚清帝、後兩黨之爭中為帝黨首領。甲午中日戰爭後，他積極支持康有為變法主張，並將其密薦之於光緒皇帝，最終促成「戊戌變法」。

洪鈞先後出使俄、德、荷、奧諸國，任外交使臣三年。曾利用西方資料撰成《元史譯文證補》，開創利用西方著作、資料研究元史的新紀元，被稱為「大使狀元」。張謇一生先後創辦數十個企業及大批社會事業，影響遍及全國，被譽為「中國之大教育家、大實業家」，有民族工業先驅者、狀元實業家之稱。傅善祥是中國歷史上唯一的女狀元，奪得太平天國三年狀元，被封為「東殿尚書」、「恩賞丞相」，成為天王洪秀全和東王楊秀清在政治上、經濟上的得力助手。

在江蘇大地上，還湧現出諸多科舉世家。以清代為例，先後出現祖孫狀元彭定求與彭啟豐，叔侄狀元汪繹與汪應銓、翁同和與翁曾源、于振和于敏中、吳廷琛與吳鍾駿（堂侄），翁婿狀元彭啟豐與庄培因、陸潤庠與洪鈞。與此類似的，還有父子三鼎甲繆彤（清康熙六年狀元）與繆曰藻（清康熙五十四年榜眼），兄弟三鼎甲潘世恩（清乾隆五十八年狀元）與潘世璜（清乾隆六十年探花），父子榜眼王錫爵（明嘉靖四十一年榜眼）與王衡（明萬曆二十九年科榜眼），父子探花汪廷嶼（清乾隆十三年探花）和汪學金（清乾隆四十六年探花）。即使在武舉科舉考試中，也有清劉榮慶、劉國慶這樣的兄弟狀元。至於同門進士，更是數不勝數、比比皆是。

天下狀元第一市—-蘇州市

據統計，自唐代至清末，蘇州地區共產生狀元（包括祖籍在外地，生長在蘇州）49名，這不僅在全國各市中排列第一，即便是絕大部分的省也難以與之相比。蘇州歷史上經濟富庶，藏書豐富，為文化和教育的發展奠定了堅實的物質基礎。早在北宋景佑二年（1035），時任蘇州郡守的范仲淹，應地方人士朱公綽之請，奏請朝廷，獲准給田5頃，並捐出擬作私宅的用地，創建蘇州府學。後幾經修茸和擴建，規模不斷擴大，名聲也越來越響，被譽為「東南學宮之首」。

蘇州府學

蘇州府學不僅向朝廷輸送了大量的科舉人才，也帶動了州學、縣學乃至社學、義塾、私塾和書院的發展，眾多本土的科舉世家紛紛湧現。長洲彭家（彭啟豐家族）先後出過兩名會元，兩名狀元，14名進士。常熟翁同龢家族、鎮洋（今屬太倉）汪廷嶼家族、吳縣繆彤家族和潘世恩家族、太倉王錫爵家族等，都是蘇州科舉世家的代表，也是蘇州狀元文化的縮影。

此外，吳縣的狀元多達33位。有唐代的沈傳師、陸扆、蘇檢，宋代的莫儔、黃由、阮登炳，明代的施盤、吳寬、申時行、文震孟，清代的繆彤、韓菼、洪鈞和陸潤庠等。與此同時，吳縣還分別在宗乾道年間、明萬曆年間和清順治年間出了朱起宗、陳大猷、於國柱三位武狀元，因此說吳縣是天下第一的「狀元之鄉」並不為過。

不僅如此，蘇州自古享有「上有天堂、下有蘇杭」的美譽，吸引全國許多士子落籍於此，進一步壯大了蘇州狀元的隊伍。如唐代的楊憑、楊凝兄弟，明代的朱希周，清代的徐元文、汪繹、汪應銓、潘世恩、吳信中和洪鈞等。他們能成為狀元，與在蘇州接受的教育分不開。

天下狀元第一家—-吳縣歸家

同門狀元是我國狀元文化的一大特點，祖孫狀元、父子狀元、叔侄狀元、兄弟狀元不在少數，但最為耀眼當數吳縣歸家。唐代吳郡長洲的歸氏家族乃吳地的名門望族，也是書香門第。在唐代中晚期，曾出現過「五子登科」和「祖孫、父子、兄弟六狀元」的盛事，不僅是江蘇狀元第一家，也被譽為「天下狀元第一家」。歸崇敬（712─799），唐吳郡長洲人，曾任戶部、工部、兵部三部尚書，封餘姚郡公。歸崇敬之子歸登（754─820），官至工部尚書，封長洲縣男。歸登之子歸融（？─853），唐憲宗元和時進士及第，也官至兵部尚書，封晉陵郡公。

五子登科圖

歸融生有五子，個個中進士，其中3個中了狀元，開創了「五子登科」的先河。長子歸仁晦，唐開成三年（838）進士；二子歸仁翰，唐大中十一年（857）狀元；三子歸仁憲，唐大中年間進士；四子歸仁紹，唐咸通十年（869）狀元；五子歸仁澤，唐咸通十五年（874）狀元。短短的17年間，同門出了三位狀元。歸仁澤之子歸黯，唐景福元年（892）狀元。歸黯之子歸佾，唐昭宗光化四年（901）狀元；歸佾弟歸系，在唐哀帝天佑二年（905）也高中狀元。到此，前後49年間，長洲歸氏家族就出了6位狀元，成為中國科舉史上的佳話。
