時代荒誕所帶來的副效應，腐朽統治下唐末的一場無厘頭起義！

2019-12-16 2,190條評論 4,823次閱讀 3101人點贊

正所謂「王侯將相寧有種乎」，中國歷史上不乏民變造反之人，有些故事聽起來很慘烈，而有些，則因為時代的荒誕如今聽起來帶有一點無厘頭色彩。

歷朝歷代都不乏各形各色起義之人，西周末年時，「國人暴動」——「九傳厲王，暴虐無道，為國人所殺，此千古民變之始」。此後，自陳勝吳廣於大澤鄉起義，兩千餘年來，中國歷史上的反抗鬥爭從未停止，黃巾起義、隋末瓦崗軍起義、元末紅巾軍起義、李自成起義等等，到了晚清時期，局勢動蕩，民變更是頻頻爆發，白蓮教起義、天理教起義、小刀會起義以及太平天國起義，不僅給腐朽的清末王朝以打擊，同時也讓階級矛盾更為激化。歷來起義都是殘酷的，一旦起義就會有無數人為之喪命的，但放在整個歷史潮流中，有些起義現在看的話充滿了荒誕與無厘頭的色彩！

何為民變，通俗解釋即鄉民所進行各種形式的聚眾反抗鬥爭。讀二十四史，民變一次較早地出自元朝脫脫主編的《金史》

《金史》一書中記「(宣宗至寧三年)西京軍民變,遣官撫諭之」。

此後,「民變」一語開始在正史記載中不斷出現。雖然民變一次出現的時間並不是很早，但民變之事可謂自古皆有。中國歷史上的民變造反，要麼是因為上層統治者殘暴統治導致，要麼就是當地官員嚴政苛稅，要麼就是出於野心而奪取政權。除此之外，還有一些如今看起來沒有頭緒的民變，帶著荒誕而又諷刺的意味。

唐朝是歷史長河中不可避免的一個朝代，但儘管是漢唐盛世，依舊不乏各種各樣的起義，比如中國史上女性自稱皇帝的第一人陳碩真，更是其中的代表人物，甚至影響了之後的中國歷史上第一位真正的女皇武則天。縱觀歷史上的各個王朝，一般一個朝代的初期，統治者大都兢兢業業，朝堂上一切都看起來充滿了勃勃生機；待到了朝代的中期，王朝可能會出現系列問題，如果有明君賢臣，則可能會出現一點中興之景；但到了一個朝代的末期，縱使有天縱奇才，也難以挽救王朝的頹勢，民變不斷，民不聊生，王朝難免被各形各色的勢力所泯滅。

唐朝末年，民變蜂起。民變基本上是有目的的，要麼是世事艱辛，被逼得沒有活路，有的目的則是奪取政權，稱王稱霸。但唐末發生的一起小型的民變，卻充滿了荒誕色彩，率人進入了皇宮，卻什麼都不為，讓後世看到此事的很多學者或者對歷史感興趣研究來研究去，都無法解釋這件事的來龍去脈究竟是怎麼回事 。或許，歷史的真相早已經消逝在煙塵之中，也許正是那個時代的荒誕才造就了荒誕的一群人。

《新唐書》中記載:穆、敬昏童失德，以其立位不久，故天下未至於敗亂，而敬宗卒及其身，是豈有討賊之志哉！

唐穆宗即位時已經26歲了，這個年齡在古代已經並不小了，但如果一個統治者想要有一番作為，對於正值壯年的統治者來說，還是一個綽綽有餘，令人羨慕的年齡。縱觀唐朝歷史，唐太宗李世民隨父起兵，經歷「玄武門之變」，等到自己登基之時已經是28歲了；等到他的曾孫唐玄宗李隆基登基之時，也是28歲的年齡。當然了，太宗與玄宗二人閱歷豐富，是經歷一番政治鬥爭之後才得以繼位的，對於治理國家可以說相對成熟。

如果一個皇帝想做一個明君，聽起來容易，做起來卻很難，因為他需要時時刻刻地保持理智，提醒自己，反省自己；但如果一個皇帝想要飽食終日、遊樂享受，則很容易就能達到這個目標！穆宗的母親是郭貴妃是於大唐有再造之功的郭子儀的孫女，所以唐穆宗李恆可以說有一個比較強大的外戚實力了，也正因為如此，唐穆宗得以在憲宗去世後順利繼位。儘管李恆在儲君期間，惶恐不安，但唐穆宗繼位之後，卻沒有仿效其先祖太宗的勵精圖治，而是縱情享樂，毫無節制。唐穆宗在位時期，任用宰相蕭俛、段文昌，兩人毫無遠見，認為藩鎮已平，應當弭兵。結果馬放南山，軍士無處可去，只好藏於山林。不久河朔三鎮復叛，軍士紛紛歸附三鎮。正因為穆宗對於軍隊的不重視，又沒有足夠賢明的大臣，唐穆宗、敬宗時期才會出現一個荒誕而短暫的民變故事。

古代最不缺的一種人之一就是算命先生，比如漢朝的于吉、左慈，再比如唐朝的袁天罡、李淳風，他們或多或少，對於歷史的發展產生了一定的影響。唐敬宗時期，在偌大的長安城裡有一個叫做蘇玄明的算命先生，算命先生一般都比較善於琢磨人心，與各形各色的人物交流，可以說和三教九流之人都脫不了干係。開元一枝柳，長慶二年春，作為一生經歷了九代大唐皇帝的詩人白居易，他大致經歷了這個長慶年間的荒誕事件，如果他在現場，又會做出何等的評價。

話說唐穆宗剛剛去世沒四五個月，唐敬宗剛剛繼位不久的長慶四年五月的一天晚上，久居長安的算命先生蘇玄明也不知是不是喝多了還是故意的，就跟他的好朋友嘮起了嗑，吹起了牛皮，說著說著，他為跟自己關係挺好的朋友，當時是染坊工人的張韶算了一卦，接著對自己的朋友張韶說：「我嘗為子卜，子當御殿食，我與焉。吾聞上晝夜獵，出入無度，可圖也。」大致意思就是說，兄弟，我給你算了一卦。我算中你會坐在如今皇帝的寶座上，和我一起共進晚餐。現在，咱們的皇上日夜不停地打獵玩球，很少在皇宮，這正是兄弟你造反的機會。聽到老蘇說的這麼信誓旦旦的，張韶於是忽悠了一群染坊工人和街頭的潑皮無賴，接著把自己準備的兵器藏在製作染料用的紫草里，裝上車，氣勢洶洶地朝皇宮進發。

結果，這一干來人還沒到達目的地，就被人家在宮裡巡邏的禁衛軍發現了破綻。有個士兵發現車子有異，就攔下來詢問。張韶一心虛，一不做二不休，抽出自己的刀殺了那個士兵，然後讓自己的手下部眾拿起自己武器，自己帶頭大聲喊叫著衝進了皇宮。 也不知道是禁衛軍太廢了還是他們被張韶的氣勢給驚住了。張韶竟然真就衝進了皇宮，不僅如此，還把剛剛繼位的年輕皇帝李湛給嚇得魂不附體，被一個太監背到神策軍大營里躲了起來。

其實唐敬宗這個人本質不壞，就是沒啥能耐，人家只是一個單純喜歡打個馬球什麼的體育愛好者，就連後來通俗史學家蔡國藩都評價道:「敬宗在位二年，未嘗行一虐政，且於裴度、李絳、韋處厚諸臣，亦知其忠直可用，非直淫昏無道者比，而卒為逆閹所弒者，好游宴，暱佞幸故也。」

皇帝沒啥能耐被嚇跑了，張韶一群人就興緻勃勃地進了大殿，人家張韶坐上皇帝的寶座，馬上想到了老朋友，於是請蘇玄明一起吃東西，對他興高采烈地說：「兄弟，你小子算得真准！」然後繼續開始吃，蘇玄明沒想到張韶就這樣結束了，大失所望，問道：「難道你率人進宮就只為了在這吃頓飯？」 或許，蘇玄明自己都沒想到，他被張韶給坑了，他沒想到自己兄弟竟然只是如此鼠目寸光，人家老蘇本來是鼓勵想鼓勵張韶造反成就一番事業的，這樣自己好歹也是開國元勛啊，弄不好就是張良一樣的人物，結果沒想到自己的主公如此不堪，人家只是打算胡吃海塞一頓。咋辦，造反了就抓緊跑唄，一群人剛沒跑多遠，就被趕過來的將軍和手下的將士們給全都砍死了。

當然了，或許張韶和蘇玄明他們抵抗了，但是這些行為在厚重的歷史中，最終也只能是史書里的一點點描寫了。

「韶惡之，悉以寶器賜其徒，攻弓箭庫，仗士拒之，不勝。存亮遣左神策大將軍康藝全、將軍何文哲宋叔夜孟文亮，右神策大將軍康志睦、將軍李泳尚國忠，率騎兵討賊，日暮，射韶及玄明皆死。」

這就是唐朝有名的蘇玄明張韶謀反事件了，一個荒誕不己，在整個歷史長河裡連一朵浪花都看不到的一個事件，他在後來的史書里也只是如插曲一樣記載了一段話而已。或許，他們的真實經歷可能遠遠不止如此，但在這悠久而長遠的歷史中，卻也只能是一點點波瀾。張韶為了驗證自己朋友的蘇玄明的卦術，蘇玄明為了想要幫他人成就一番事業，兩個人最終都賠上自己的身家性命。這是一個血腥的結局，他本來應該是殘忍的、殘酷的，但用文字表達出來卻是如此荒誕的，帶著濃濃的無厘頭色彩的。

一段文字的背後，隱藏的是無數人的鮮血，之所以發生這樣的事件，恰恰是對於當時時代的最大諷刺。時代的荒誕，才會發生如此荒誕的造反。那個時代，統治者的昏庸，統治階級的腐朽以及百姓的麻木，這一系列內容，不僅使張韶蘇玄明他們付出了生命，同樣的，還有那個時代許許多多的平凡人！
