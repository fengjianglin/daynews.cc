皇帝猜疑將軍讓其妻子入宮，將軍堅決阻攔遭拒，妻子堅持非去不可

大家都知道完顏阿骨打最開始只是女真部族奴隸主的總首領，後來通過自己的努力完成了建國、破遼兩件大事，推動女真族的歷史進入一個新時期。從而建立一個新的朝代—金國。金國建立後，完顏阿骨打就廢除原來部落聯盟長的制度，自己自稱皇帝，確立了皇權的統治。阿骨打沒有象阿保機建立遼國時那樣，模仿漢制立太子，皇位的繼承仍然暫時保留著推選的痕迹，但實際上權力已完全掌握在阿骨打家族手中。今天我們來說一位他的後代子孫。

大家都知道自古以來帝王之爭就是殘酷而艱辛的，金國帝王之爭自然也不例外。完顏阿骨打是一位很出色的皇帝，在他的後代子孫中也有一位同樣出色的帝王，他就是完顏雍。完顏雍的帝王之路自然也是一樣，他甚至還差點被人帶綠帽子，最終他的妻子以死換來他的忍辱負重，他藉助這個千載難逢的機會起兵奪位。最終成就一代帝王，也算是對的起他的妻子。

據記載完顏雍長得很魁偉，性格沉靜明達，又善於騎射。年輕時，他每次出獵，很多老年人都跟了去看，讚賞他的騎射技術，”國人推為第一”。他為人寬厚，常隨叔伯們四處征戰，將士都很推崇他。金熙宗皇統年間完顏雍以宗室子授光祿大夫，封葛王，為兵部尚書。

金熙宗是金國的第三位皇帝，他是完顏阿骨打的嫡長孫完顏亶，完顏亶在位後期，時常酗酒，殺戮親貴大臣，唯有對完顏雍比較好。他特別欣賞完顏雍的才華，就將他擔任統兵在外的大將，並讓他鎮守西涼，然此後的金國的都城東京卻發生了一場驚天大陰謀。海陵王完顏亮起兵誅殺了金熙宗。自己登基為帝，做了皇帝寶座的海陵王就開始排除異己，首先剷除的就是先皇當年的親信，於是在都城盛京凡是和金太祖有血緣關係或者交往過密的人都被直接斬殺。遠在西涼的完顏雍聽到這一噩耗之後，整天擔心受怕，精神恍惚。他明白自己受先皇之命在西涼鎮守。肯定是逃脫不了完顏亮的迫害。果然完顏亮對他的猜疑有加。

完顏雍的妻子是個賢內助，她看出了自己丈夫的心思，就為他出謀劃策。讓他將宋朝進貢的金銀珠寶拿去賄賂完顏亮。結果暫時讓其打消了殺害完顏雍的打算。但是完顏雍作為先皇的部下。又擁兵十萬，這對於任何一個帝王來說始終都是一個禍患。再說完顏雍能文能武，在女真貴族中威望又高，海陵帝對他很不放心，雖說完顏雍送來很對的金銀財寶。又表現出很怕他，對他也恭順。但是海陵帝還是不放心，於是他打算先下手為強。他以聽聞完顏雍的妻子是天下第一美女，就應當進宮侍候皇帝為由。逼迫完顏雍出兵反叛。

開始完顏雍聽到這個消息之後，確實是怒氣沖沖，但卻讓完顏亮失望了，原來完顏雍的妻子聽到這個消息之後。儘管丈夫堅決不讓自己進宮，但她卻執意要去。因為她很明白，這只是完顏亮的一個對付自己丈夫的借口。如果不去，自己丈夫一家就會被全部誅殺。只有自己去了才能讓皇帝徹底打消對丈夫的戒心。但是他的夫人沒有讓完顏亮如意。因為烏林答氏深知完顏亮好色成性，所以在離中都70里附近時，自殺身亡以保全自己的名聲。

烏林答氏臨死前給完顏雍留下了絕命遺書，史稱《上世宗書》。深情地懇求丈夫不要因為她的死而悲傷”作兒女之態”，要”卧薪嘗膽”，居官”修德政，肅綱紀，延攬英雄，務悅民心”，等待時機”奪帝位，一怒而安天下。”完顏雍對愛妻為他而死，忍辱負重，割愛隱藏悲痛，毫不顯露出對完顏亮怨恨之情。也沒有前去烏林答氏的死身地，親自操辦後事，僅是要下人就地草草埋葬了事，而將奪妻逼死的深仇大恨暗藏於內心，致使完顏亮放過完顏雍一馬，但還是派心腹高存福任東京副留守去監視他。
