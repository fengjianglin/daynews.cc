唐代官員蒙冤入獄 結果見證奇蹟發生

2019-12-24 586條評論 3,137次閱讀 2229人點贊

唐朝貞觀十四年春天，董雄擔任大理寺的官員，因受到季仙僮謀反事件的牽連，被上了枷鎖，囚禁在御史台。當時，一同受牽連被扣押的十多人。其中大理丞李忻玄、司直王忻與董雄關在同一牢房，而且每個人都被鐵索牢牢地鎖著。

董雄自年少起就誠信神佛，此案審查期間，他雖身陷囹圄，還是專心誦念經卷，一篇二千多字的經文，短短几天內就背誦了三千遍。

一天夜裡，董雄坐在一旁靜心誦經，忽然手上的鐵枷鎖自動開了，掉在地上。董雄驚訝地告訴李忻玄，兩人還一起查看鐵鎖，發現鐵鎖還是牢牢地鎖著，而且鎖鉤也沒有任何損壞的情況，可是鎖和鉤卻鬆開了數尺。他們都覺得很奇異。因擔心受到責罰，於是董雄告知獄卒，請獄卒重新為他鎖上鐵鎖。

當天夜裡是監察御史張敬值班，張敬命獄吏再給董雄鎖上。獄吏拿著燭火仔細地查看，看見鎖與鉤確實沒有打開的痕迹，卻自動分開了，也覺得很怪異。因此再上鎖時，特意貼上封條纏住了鐵鎖，並在封條上籤上了名字。

待獄吏走後，董雄又重新坐下來專心誦經。到了五更天，鐵鎖好像被人打開了一樣，又掉在地上，發出響聲。董雄有些擔心，又告訴李忻玄。李忻玄說：「現在天色還早，不方便叫獄吏來。」

天亮後獄吏來了，眾人發現，雖然鎖與鉤各自掉在地上，然而鎖依然是鎖著的，並且鎖上的封條也原封未動，鉤也十分地固定密合。不可能有打開的道理。

一同被關押的大理丞李忻玄從來都不信佛法。以前，他看到妻子閱讀經書，常常對她說：「你為什麼一味地崇拜外來的神明，去讀這種書呢？」但當他看見董雄身上的鎖鉤自動開解兩次的事情後，心中非常震驚，霎時醒悟，深深地感嘆道：「我到今天才知，佛法實在是不可思議的，佛是偉大的聖者，無人可與祂相比。」

從此，李忻玄也開始在獄中誦念菩薩名號。當他虔心地念滿三萬遍時，有個白天，鐵鎖也自動解開掉落地上。仔細一看，與先前董雄的情形幾乎沒有差別。因這場牢獄之災，李忻玄為自己以前對神佛無知的言行，深深悔過自責，從此深信佛法。

不久之後，董雄、李忻玄、王忻三人均得以昭雪，無罪釋放。
