擁有2000萬人口的帝國，被170人入侵者毀滅，不敢想像

2019-12-19 53條評論 2,690次閱讀 1266人點贊

印加帝國曾經是一個輝煌的存在，被人們稱為印加文明正是印加帝國所締造。但又有多少人知道，這麼一個有著幾千萬人口的帝國，卻被一個只有170人的西班牙入侵者毀滅。如果這不是真正發生的事，誰會相信呢。簡直和中國的武俠劇有的一拼，一人就可戰勝千軍萬馬。

印加帝國皇帝瓦伊納·卡帕克

1526年，印加帝國皇帝瓦伊納·卡帕克被天花奪去了生命，又奪去了大臣和王位繼承人尼南·庫尤奇的生命。為了王位，印加帝國發生了另一場內戰。內戰爆發在皇帝的另外兩個兒子，華斯卡和阿塔瓦爾帕之間。阿塔瓦爾帕稱父親已經決定把帝國重新劃分，在阿塔瓦爾帕統治的基多建立了一個全新的首都。一場內戰終於爆發了。

戰爭使印加軍隊四分五裂，所以西班牙人選擇了這個機會進入印加帝國。

弗朗西斯科·皮薩羅畫像

170人的隊伍

1532年9月，皮薩羅率領他的170名手下開始向深山挺進。他們所經之處，到處都是戰爭和瘟疫留下的廢墟。皮薩羅一路捕獵逃散的印加人。他命人嚴刑拷打他們，並強迫他們做嚮導。從印加人口中皮薩羅得知，印加人的內戰剛結束，他們的新皇帝阿塔瓦爾帕正率領部下趕往帝國首都庫斯科，眼下正在卡哈馬卡城紮營。

戰爭就是這樣，為獲得敵方的信息可以不折手段，嚴刑拷打是正常事，並不過分。

皮薩羅讓印加人給阿塔瓦爾帕帶去口信：「我，弗朗西斯科·皮薩羅，是西班牙國王的僕人，將引導你們服從基督教和國王的支配。如若不從，我將毀滅你們的一切。」 這群西班牙征服者是虔誠的天主教徒和保皇黨人。

他們所做的一切都打著上帝和國王的旗號。他們認為自己要能改變世界。阿塔瓦爾帕當時30歲，正是火氣極盛的年紀，剛在印加內戰中獲勝，他聽了皮薩羅的口信肯定氣急敗壞。

阿塔瓦爾帕的畫像

阿塔瓦爾帕將西班牙人放在心上，他戰勝華斯卡後控制了整個印加帝國。他關心的是進軍首都，而不是這一小撮入侵者。皮薩羅沒有等到臣服的消息，卻得知阿塔瓦爾帕帶5萬人駐紮在卡哈馬卡城外。他們想就憑170人是戰勝不了5萬人的大軍。皮薩羅和屬下商量，最後皮薩裝出友好的姿態，讓印加人轉告阿塔瓦爾帕：「我將是他的朋友和兄弟。」

皮薩羅雖然無恥但也算聰明，170人怎麼可能和人家5萬人打嘛，只能用這樣的方法讓他們放鬆警惕。

印加的卡哈馬卡城

西班牙人仍不死心，向卡哈馬卡進發。皮薩羅對他的士兵們說：「既然我們來了，就要去試試，也許我們會有意外的收穫。」1532年11月15日，他們來到了印加的卡哈馬卡城，他們想不到印加人居然有這樣雄偉的城市。城市從山腳延伸到半山，彷彿一個巨大的迷宮。城市的中心是由青石板鋪就的，寬廣平坦的巨型廣場，廣場正中立有一個高大的祭壇。

西班牙人當時的選擇很難，他們要麼向前，要麼向後，結果他們選擇了向前。在沒有知識的時代，真不知道印加人是如何將這一切建成的，就算現在的我們也無法解釋當時印加是如何做到的。

印加帝國廣場遺址

而在廣場之下的平原上，阿塔瓦爾帕身邊的5萬名戰士駐地。並且秩序井然，。但是印加戰士的裝備不如西班牙人，他們沒有劍和戰馬。西班牙人小心地進入城市。那些半裸著身子的印加居民遠遠地望著他們。西班牙人生怕那些野人一樣的居民從背後突然殺出，那樣的話，他們無論如何也難以抵擋。

而且他們低估了入侵者，這是一個致命的錯誤。更致命的錯誤是，他們輕信了入侵者的謊言。

戰爭描繪圖

皮薩羅派自己的兄弟和使者去見阿塔瓦爾帕，阿塔瓦爾帕用玉米啤酒款待他們。但他們喝了一口玉米啤酒，覺得味道實在不好，就吐掉了。然後說道：「我是萬能的主和強大的西班牙國王的僕人，」結果他不小心又把杯子碰到了地上。阿塔瓦爾帕聽不懂他在說什麼，不等翻譯出來便將他趕了出去。談判破裂的消息讓每個人都嚇壞了。他們的人數實在太少了，而且根本不可能得到任何支援。

印加人失去了最好的機會將這些入侵者消滅，之後大家都知道，沒過多久印加人就被這些西班牙人設計毀滅了。一個經歷了幾個世紀的印加帝國，就因為小瞧侵略者而被毀滅。

總結

如果西班牙當時提高警惕，將這170人消滅，就不會有毀滅的事了。不過在一個擁有幾千萬人的帝國面前，170人的一個及其小的隊伍中，實在是不起眼，誰又能相信他們成功了，如果皮薩羅的那一小隊人馬被印加帝國消滅，也許後人就不會為印加人的悲劇嘆息了。所以這件事業告訴我們，不管是多麼強大的敵人，還是及其弱小的敵人，都必須提高警惕，以防他們趁虛而入。
