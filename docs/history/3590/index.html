大半生為中國醫學走南闖北，他晚年病重，卻遭自己的醫院見死不救

2019-12-03 0條評論 20次閱讀 0人點贊

摘要

“學醫的目的，有很多人以為會多賺錢，我想他們走錯路了。我認定做醫師的人，必須有犧牲個人、服務社會的精神。”——顏福慶1966年，一波又一波的人被打上反動派的標籤，其中就包括一位七十幾高齡的醫學泰斗。

「學醫的目的，有很多人以為會多賺錢，我想他們走錯路了。我認定做醫師的人，必須有犧牲個人、服務社會的精神。」——顏福慶

1966年，一波又一波的人被打上反動派的標籤，其中就包括一位七十幾高齡的醫學泰斗。在那個特殊的年代，年事已高的他被病魔折磨的奄奄一息，被送到自己一手創建的醫院卻被拒收，只用一點點葯就要將他打發走，理由是荒誕卻又理直氣壯的：反動派不配接受救助。這位老人就是中國醫療事業的奠基人——顏福慶。

種子——生根萌芽

1882年7月28號，在這個平凡的日子裡，一個清貧卻充滿溫情的家庭里迎來了一位新生命，那就是顏福慶。

顏福慶此生可謂是命運多舛，在他6歲那年，父親因感染傷寒而不幸離世，給這個普通的基督教牧師家庭帶來了沉重的打擊。隨後，他被過繼給了他的伯父，也是當時上海聖約翰大學的校長，顏永京。

在小小的顏福慶眼中，病魔是這個世界上最可惡的東西，它像一個詛咒，讓母親常年卧榻在床，又像一個劊子手，殘忍地奪去了父親的生命。於是一顆渴望擊敗病痛的種子在顏福慶的心中開始生根發芽。

樹苗——茁壯成長

顏福慶從小就一直是學習楷模，長大後通過自身的不懈努力和伯父錦上添花的資助下，他被上海聖約翰大學醫學院錄取了。當他邁進醫學院的大門時，也意味著他朝他的夢想更近了一步。1904年，顏福慶畢業了。四年的孜孜不倦使他渴望伸展一番拳腳，於是他隻身一人前往南非擔任礦醫，為在南非的同胞礦工驅趕病痛。

而當他到達南非之後，他才知道事情遠沒有他想的那麼簡單。南非的一切大程度地顛覆了他按照傳統所遵循的一系列思維方式和種種接受習慣，此時此刻，他只感到茫然和無所適從。心有餘而力不足的顏福慶恍然明白了自身的知識還遠不足以解救這些飽受病痛折磨的同胞，於是他在心中做了一個重要的決定，那就是進修深造。

顏福慶遠渡重洋，在那個年代，華人在異國他鄉的不易可想而知，況且顏福慶還是當時被耶魯大學醫學院招收的唯一亞洲人。他在新環境中彷彿一塊缺水的海綿，瘋狂地汲取在國內學不到的知識，懸樑刺股，徹夜苦讀。

對知識的渴求和迫切想學醫救人的心情幫助他抵抗住了來自生活的重重壓力，雖然條件艱苦，但他並不氣餒，堅決不半途而廢，依然堅持夢想。皇天不負有心人，顏福慶在耶魯的學習生涯表現可圈可點，憑藉他的努力，他脫穎而出成為了耶魯優秀博士畢業生，也是亞洲第一位獲得此學位的人。

參天大樹——灑下陰涼

當時西方的醫療技術還並沒有普及，人們對於西洋的技術和事物仍抱有懷疑的態度，甚至是懼怕西醫西藥。顏福慶回國後加入了湖南雅禮醫院時就面臨著這樣一個難題。儘管當時雅禮醫院的頂樑柱胡美博士興高采烈地認為顏福慶的到來簡直如雪中送炭，但害怕受到國人的排斥一直是顏福慶的心頭大患。

但顏福慶也是中國人，同胞的身份打消了人們的防備，且他為人和藹可親又醫術精湛，很快就深得大家信賴，越來越多的人願意接受他的治療，漸漸地，越來越多的人開始接受在某方面見效更快又安全的西醫。

顏福慶後來還被邀請到長沙的各個學校里講授醫學知識，此後，西醫逐漸被整個湖南接納，但洋人創辦的醫院收費也隨著水漲船高。顏福慶經歷了這些，就越發想自己創建一個屬於中國人的醫院。他迫切的想改變中國醫療衛生現狀。

創立一個醫院並不是件易事，顏福慶抓住了政府想發展醫學的絕佳時機。他輾轉幾年，經歷了重重困難才拿到了成功的一紙合約。顏福慶終於在1914年創辦了長沙湘雅醫學專門學校，並出任校長。

在那個戰火紛飛的年代，湘雅醫學院在之後培養出了為我國醫療事業做出重大貢獻的各類人才們。但1926年，北伐的硝煙終於瀰漫到了湘江大地，顏福慶身不由己離開了奉獻了將近20年的土地，將工作重心轉移到了北京協和醫院。第二年他放棄了高薪職位，投入到了國聯衛生組織向他發出的橄欖枝。但出乎意料的是這卻令顏福慶遭到官場的排斥。

之後的幾年顏福慶又轉輾到上海，他親身體驗了上海的醫療水準，於是決定申請當地政府的幫助和支持，創立了第四中山大學醫學院。不僅如此，他還尋求了他的堂兄和上海商界名人葉子衡的理解和幫助。

前前後後花費了巨大的心力，嘔心瀝血，苦口婆心說服各界人士來幫忙籌建這個醫院，使之設備更加完善。英國大陸報曾對這個醫院做出了高度評價：「這所具有重大歷史意義的中國醫學院，它將成為中國的新驕傲。」

枯木——被迫折斷

然而就是這麼一個令國人驕傲的，注滿了顏福慶半輩子心血的醫院，在顏福慶呼吸困難想借用氧氣瓶時「義正言辭」地拒絕，在顏福慶特殊期間被扣上反動派的帽子時，選擇了見死不救。１９７０年１１月２９號上午，顏福慶流下了一行濁淚，終於閉上了眼睛，與人世長辭了．．．．．．

顏福慶一生都在為醫學事業兢兢業業，奉獻自己的一生，最後卻被冠上莫須有的罪名，並陷入了一場轟轟烈烈的鬥爭，他一身錚錚鐵骨，在極端的環境中也保留了身為醫學家的尊嚴。一代醫學泰斗以這樣的方式離開人世令人唏噓不已，特殊時期結束後的平冤為顏福慶正了名，不知道能否以此慰他在天之靈。

回顧顏福慶的一生，矢志不渝地為現代醫學的發展獻身，經歷重重坎坷也仍不放棄，年事已高也堅持在醫學事業的第一線「戰鬥」。他嚴謹治學的精神使人敬佩，也值得我們所有後輩銘記、傳揚。
