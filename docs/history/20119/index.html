古人的名字都會加輩分，為何現在沒了？網友怎麼說

2019-12-16 1,263條評論 3,156次閱讀 1480人點贊

不知道大家有沒有見過這種場景——白髮老人對著小孩子叫「叔叔」，這場面光是想想就有點「滑稽」，不懂輩分關係的年輕人可能得哈哈大笑。其實，以前的人取名都得加上輩分，現在人已經逐漸不加了，這是為什麼呢？難道是太難聽了？

我們先簡單了解一下「輩分」，它是指在家族、親友的長幼先後中一個人所居的位置，也可以指一個家族、親友之間的世系次第，主要包括長輩（高祖輩、曾祖輩、祖輩、父母輩等）、平輩和晚輩。

在古代，輩分決定了一個人日常生活中的行為規範，比如在長輩面前，你必須請安、讓座、讓路，就連吃飯坐什麼位置，都要按照輩分坐，不可亂了禮法。特別是儒家，對論資排輩是最為強調的，在儒家幾千年的熏陶下，老一輩講究輩分是很正常的事情。

林黛玉第一次進賈府，簡直是「步步留心，時時在意」，生怕自己說錯一句話，走錯一步路，當時她剛拜見完外祖母，王熙鳳就來了，林黛玉不知道如何稱呼她，賈母說叫她「鳳辣子」就行，林黛玉敢叫嗎？她當然不敢！幸好其他姐妹幫忙解圍，說「這是璉二嫂子」，林黛玉趕緊叫一聲「嫂子」。

然而隨著時代的發展，這種宗族文化漸漸弱化了，別說「宗族」了，就連一家人天天住一起都不太講輩分、不太講細節，更別說相隔「十萬八千里」的親戚們了，這就是輩分取名消失的第1個原因。

第2個原因，人們審美的變化。隨著越來越多80、90後成為孩子家長，取名字都追求一個獨特、好聽。你要是讓現在的家長看看70年代孩子的名字，肯定會覺得「土氣」，然而再過30年，新的家長們難道不會覺得「子涵」土氣嗎？所以用「難聽」定義輩分取名是不應該的。

其實中國宗譜的取名主要包括3類，一類是「美德、吉祥寓意」，比如：德、仁、明、孝、福；二類是「希望宗族延續和昌盛」：永、傳、昌、盛、興、延；三類是「懷念先祖和歌頌皇天恩德」：澤、祖、顯、榮、恩、錫。

更講究的家族還會請名人、文人幫忙制定宗譜，比如寫一首詩，最有代表性的就是朱元璋一家子，朱元璋沒讀過啥書大家肯定都知道，他原名叫「朱重八」，可人家當上皇帝了，後代總不能叫「朱大三」、「朱小九」吧？於是他請人幫他寫了「姓名詩」。

比如燕王（明成祖朱棣）後代的詩為：「高瞻祁見祐，厚載翊常由。慈和怡伯仲，簡靖迪先猷。」所以朱棣的後代分別叫：朱高熾、朱瞻基、朱祁鎮、朱祁鈺、朱見深、朱祐樘。他們6個人名字里第2個字元合了「高瞻祁見祐」的順序。

要是對朱家取名有了解的人會知道，裡面還有「五行」上的講究，感興趣的朋友可以自行查閱。由此可見，按照輩分取名字絕對不簡單也不土，甚至內涵遠遠超過了查字典找「漂亮」字的取名方式。

也許再過幾十年，就再也見不到大人管小孩叫「大哥」的場面了吧，越來越多傳統文化也在漸漸消失，真是令人感慨。
