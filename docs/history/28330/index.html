隋煬帝是強迫佔有宣華夫人的嗎？宣華夫人該怎麼看？

2019-12-21 400條評論 3,935次閱讀 1949人點贊

宣華夫人姓陳，她是南北朝時期南朝陳宣帝的女兒、是陳後主同父異母的妹妹，封寧遠公主。

宣華夫人三歲時，與陳朝隔長江對峙的北朝發生了政變：周朝權臣楊堅奪取皇位，代周自立，建立隋朝。

不久，陳宣帝出兵攻打隋朝失敗，急火攻心下病發去世，太子陳叔寶即位，即是陳後主。

陳叔寶自恃有長江天險，篤信無亡國之危，便荒廢朝政，沉溺於縱情享樂，其代表作《玉樹後庭花》也成為了著名的亡國之音。

隋開皇九年，隋朝攻滅陳國，完成南北統一，陳氏與其他陳國皇室成員一起被俘虜到隋朝的都城長安。

之後，陳國皇室男性成員被隋文帝分散到隋朝偏遠地區，在隋政府的監視下種田幹活自力更生，陳宮佳麗包括部分公主被賞賜給平陳有功的大臣和隋朝親貴，剛剛十來歲的陳氏則配入掖庭成為了宮女，由於她長得很漂亮，有著閉月羞花之貌，不久被隋文帝楊堅選為嬪妃 。

隋文帝的皇后獨孤氏在世時，她控制隋文帝的後宮，不讓隋文帝親近宮內女人，但是唯有陳氏能夠得到隋文帝的寵幸 。可見陳氏在隋文帝眼中的分量。史書記載陳氏”性聰慧，姿貌無雙” 。

陳氏當時的生活很是艱難，她既想博得寵愛獲取榮華，但是又害怕獨孤皇后的兇狠，還要日日夜夜提防被獨孤皇后抓到把柄處死，算得上是「夾縫中求生存」。她知道隋文帝年紀大了，跟著他以後沒什麼好的結局，於是她便開始攀附獨孤皇后，對獨孤皇后像是女兒伺候媽一樣殷勤侍奉。

此外，陳氏背地裡又給自己謀出路，搭上了隋文帝的二兒子、晉王楊廣。此時的她知道自己是個亡國的公主，她做的一切都是為了自己的生活。

獨孤皇后去世後，陳氏進封為貴人，專房擅寵，掌管後宮，六宮之內無人能與之相比。隋文帝又封其為宣華夫人 。

晉王楊廣想爭奪太子之位，於是經常送禮給宮人們以籠絡人心，為己所用。楊廣把金蛇、金駝等物送給陳氏，而陳氏確在皇太子廢立之際為楊廣出過不少力 。

楊廣從小就長得帥氣，史書上面寫道「美姿儀，少聰慧」，所以得到了父親和母親很多的寵愛，年僅十三歲就被封為晉王，並且官拜柱國、并州總管；此後他為隋朝平定四方的割據勢力，隋文帝授予他全軍統帥的稱號，楊廣也不辜負父親的期望，東征西殺，奪得了許多勝利。

楊廣平時表現出為人謙虛有禮、作風簡樸、不好美色、禮賢下士，贏得了滿朝文武和父母的信任和歡心。開皇末年，隋文帝和大兒子、太子楊勇的矛盾進一步激化，楊廣乘機爭奪儲位，最終被立為太子。

仁壽四年七月，隋文帝駕崩，楊廣繼承帝位。宣華夫人按制，出居文帝別廟仙都宮。

楊廣是一個極其好色的人，在他登基之前就已經是十分風流了，只是礙於母親的面子，他沒有顯露出來而已。楊廣登基後，很快就把就宣華夫人接入皇宮，霸佔了他父親的兩個寵妃宣華夫人和容華夫人，而且開始在全國範圍內搜尋美女。

史書記載，宣華夫人是隋煬帝殺死文帝，殺父篡位的原因所在。不得不說宣華夫人有多麼的美艷動人。然而，宣華夫人陳氏在被楊廣霸佔一年後就去世了，年約二十九 。在歷史上，宣華夫人作為”仁壽宮變”的女主角而出名。

隋煬帝在宣華夫人死後十分傷感，為其寫下《傷神賦》(今不傳) 。

近代歷史小說家蔡東藩曾經對宣華夫人作出如下評價：

宣華夫人為楊堅寵妾，復為逆子廣所烝。如宣華之貪生怕死，貽丑中冓，固不得為無咎，然誰縱逆子，以至於此?

意思是說：宣華夫人是隋文帝的寵妃，被楊廣違背人倫佔有。宣華夫人本人也有責任，因為她貪生怕死。

各位朋友，你對宣華夫人的遭遇有何看法呢？
