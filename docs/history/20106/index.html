同樣受巫蠱之禍影響，為何衛子夫自盡，陳阿嬌卻受到優待

2019-12-16 1,197條評論 2,839次閱讀 55人點贊

漢朝對於巫蠱之事極為忌諱，偏偏在漢武帝時期，兩代皇后都是受巫蠱之禍所累，導致一個被廢，一個自盡而亡。但兩件事有所不同的是，陳皇后是的確做了此事，而衛皇后則是被誣陷的。然而衛皇后最後卻死了，陳皇后反而受到優待。

漢武帝對於巫蠱之術，採取的是零容忍的態度，那麼同樣受巫蠱之禍影響，為何衛子夫自盡，陳阿嬌卻受到優待呢？難道，漢武帝也是個雙標的人嗎？

漢武帝小時候名叫劉彘，原本不受漢景帝重視。後來漢景帝的姐姐館陶公主，找到劉彘的生母王夫人，希望能夠促成自己女兒，與劉彘的一段姻緣。館陶公主在朝中影響很大，王夫人自是欣然答應下來。

館陶公主見到劉彘的時候，還特意問他：「將阿嬌指給你做妻子好不好？」劉彘答道：

「好，若得阿嬌作婦，當作金屋貯之。」

這就是典故「金屋藏嬌」的由來，而這一段政治婚姻，也由此拉開序幕。

在館陶公主的幫助下，太子劉榮被廢。同年王夫人被冊立為皇后，劉彘被立為太子，並改名為劉徹。陳阿嬌也就成為太子妃。

漢景帝去世後，劉徹順利繼位，是為漢武帝。太子妃陳阿嬌，便順理成章當上皇后。陳阿嬌出身高貴，從小就受到父母溺愛，長大之後，驕縱的個性愈發明顯。她常常因漢武帝寵幸別的妃子，而大發雷霆，甚至與漢武帝當面爭執。

漢武帝越來越無法忍受，如此驕橫無禮的皇后，加上衛子夫進宮，得到了漢武帝的寵愛，又為其接連生下三個公主，無子嗣的陳阿嬌心中更是不平。於是在元光五年，陳阿嬌施以婦人媚道，想藉此挽回漢武帝的心，結果事情敗露。

漢武帝下令徹查皇后，結果發現，陳皇后找人施行巫蠱之術，禍害他人。漢武帝一聽龍顏大怒，隨即下發詔書：

「皇后不守禮法，祈禱鬼神，降禍於他人，無法承受天命。應當交回皇后的璽綬，離開皇后之位，退居長門宮。」

陳阿嬌的皇后之位雖然被廢，但漢武帝對她仁至義盡。她搬到長門宮以後，一切吃穿用度都沒有遭到削減，直到她去世，一共又享了二十多年的清福。為何漢武帝要對她這麼好呢？我認為出於三個原因。

第一個原因，看在陳阿嬌長輩的份上。陳阿嬌的母親是館陶公主，漢景帝的同母姐姐，也就是漢武帝的姑母。陳阿嬌的外祖母是竇太后，同時也是漢武帝的祖母，因此看在長輩的份上，漢武帝不好嚴懲陳阿嬌。

漢武帝登基後，館陶公主被尊為大長公主，也稱竇太主。她知道女兒被廢以後，急忙進宮求情，漢武帝則對她說：「我之所以廢掉皇后，是因為她不符合大義，但你不必擔心，她今後的生活與從前不會有差別。」竇太主這才放心。

第二個原因，一日夫妻百日恩。漢武帝雖然無法容忍陳阿嬌，但此人好歹是自己結髮妻子，又有多年的夫妻情分，即使對方再過分，漢武帝也不忍過分處置。再說陳阿嬌最大的願望，就是當上皇后，如今廢黜了她的皇后之位，便是對她最嚴厲的懲罰了。

第三個原因，漢武帝當時還年輕，沒那麼極端。漢武帝無法容忍巫蠱之術，但陳阿嬌此舉，並未產生嚴重的後果。眾所周知，漢武帝晚年猜疑心重，且容易暴怒，但年輕時還是理智的。因此廢黜皇后也就夠了，沒必要趕盡殺絕。

這就是陳阿嬌能夠受到優待的幾大原因。那麼衛子夫呢？

眾所周知，衛子夫根本沒有施行巫蠱之術，她是被人誣陷的。然而當時漢武帝處於晚年時期，疑心變得越來越重，對巫蠱之術也愈發痛恨，加上衛子夫一直是他最寵愛的人，突然曝出她施巫蠱之術，漢武帝一下接受不了。

就這樣，衛氏一族不少人受到牽連，從而丟掉性命，冤死者多達上萬人。這也成為漢武帝人生的一大污點，遭到後世詬病。太子劉據無法述說冤情，情急之下聽從他人之言，起兵去誅殺江充，結果反被人誣告謀反。

後來劉據出逃，漢武帝派人去衛皇后宮中，收回皇后璽綬。衛子夫心中冤屈無法述說，在萬念俱灰的狀態下，只能以死明志，自盡而亡。

雖說衛子夫不是漢武帝直接處死的，但也是間接被他逼死的。而衛子夫死後，漢武帝在短時期內並沒有後悔，因為他對皇后施巫蠱之術，太子謀反一事深信不疑。此時衛子夫的處境，與當年的陳阿嬌已是截然不同，所以她的結局註定是悲慘的。

照二人的結局比較起來，反倒是陳阿嬌要幸福得多。

（參考文獻《史記》《漢書》）
