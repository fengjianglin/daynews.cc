探索：陳阿嬌的悲劇是不是劉徹一手造成的？

2019-12-05 0條評論 17次閱讀 0人點贊

摘要

大家好，我是聞道，今天跟大家聊一聊陳阿嬌

陳阿嬌留給世人的，最著名的莫過於那個有關金屋的荒謬傳說。後世很多人將阿嬌的悲劇歸因於劉徹的不守承諾，。這件事要怪也怪館陶公主和王美人，是她們從一開始就讓年幼的阿嬌沉溺於愛情天長地久的童話里，從沒有一天活在後宮婚姻的殘酷現實中，認為劉徹此生此世就該愛她一人，愛上別人便是違背了當初的誓言。

當時的劉徹哪裡知道這其中的利害，只是蹦蹦跳跳玩著自己的遊戲，就被母親拉過來認親。那時，劉徹還不叫劉徹，叫劉彘。王美人問他喜不喜歡阿嬌、想不想娶她，劉彘於是很爽快地答應，還添油加醋地說：”如果娶了阿嬌姐姐，我就要蓋一座大金屋子給她住。”於是館陶公主和王美人心領神會，心照不宣，喜笑顏開。劉徹的政治生涯也從此埋下了一顆飛黃騰達的種子。

那時的劉徹哪裡會明白，他所允諾的是對一個女子一生託付的誓言。從此這女子一生便隨了他，榮辱與共，同貴同賤。可惜，他心中所想，只是不異於尋常遊戲的角色扮演罷了。可阿嬌不同，此時的她已日漸成熟，雖然年齒尚幼，卻也明白這其中的道理及微妙的含義。她以為，她收到了平生第一份約定，並執著地將它深深埋在心底，讓它錯誤地生根發芽，還開出錯誤的花。

及至劉徹年長成為皇帝，阿嬌成為皇后，他兌現了自己的諾言，金屋藏嬌，專寵她一人。而她卻覺得不夠，離當初那誓言還差得遠，他早說過，此生此世便只愛自己一人。於是阿嬌因愛生妒，終日妒火中燒、蠻橫無理。

而古時男子最厭惡女子的，莫過於一個”妒”字，卻不知這一個”妒”字也是女子之所以成為女子的一個要素。只是阿嬌愛得太過真切，太過蠻橫，自小嬌生慣養的她不明白，她無論如何都不該把這份妒意在丈夫面前表現得如此淋漓盡致。

一切都是那幼時的約誓惹的禍，讓阿嬌先入為主地以為這男人應該愛她一生一世，卻不知愛情是最要不得先入為主的，沒有人能在一切未知時許下對你今生今世的約，她的一生也本是不可承諾、庇佑在兒時的一句戲言上。

更何況，那時候承諾她的是劉彘，不是劉徹。劉彘可以保證愛她一生一世，而劉徹不能。你不是要金屋藏嬌嗎，我履行了當初的誓約，給你一座金屋子。只是那屋子名叫長門，將你此生年華深藏，我卻永遠不踏進一步，可好？

阿嬌卻不明白，儘管她貴為皇后，出身名門，與劉徹的皇位有著千絲萬縷的聯繫。可是隨著時間的流逝，在劉徹眼裡，她也只是幽居深宮的一株花草，再提不起其他的興趣。
