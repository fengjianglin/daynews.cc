《慶余年》范建為什麼不是戶部尚書，而是戶部侍郎？

2019-12-16 630條評論 1,842次閱讀 165人點贊

戶部侍郎范建和監察院院長陳萍萍，一個是監察院院長，一人之下萬人之上，手中握著天下最恐怖的一股勢力；另一個卻是小小的戶部侍郎，在他之上還有尚書、宰相等多個職位，就連與他身份平起平坐的官員也能一抓一大把。

按理說，從小追隨慶帝的范建怎麼說也應該得到個一品大臣的職位吧，可慶帝的安排卻顯得有些不厚道。

其實不然，仔細看劇就能發現，雖說范建只是戶部侍郎，可是就連一品大臣看到他也要恭恭敬敬的稱呼一聲「范大人」，宰相林若甫也要對他收起小覷之心。因為他們都知道，這位戶部侍郎可是在慶帝面前都敢直言不諱的狠人。范建的影響力可超出了他老婆口中所說的「小小戶部侍郎」。

那麼為何慶帝給了范建權利，卻又只給他侍郎呢？

眾所周知，慶帝是個心思深沉的主，擅長制衡之術的同時，也對自己的實權抓的非常到位，而范建就是慶帝安排到戶部的「眼睛」。

戶部的重要性高於任何一個部門，它掌管國庫，只有派范建這麼有能力的人去監督，慶帝才能放心籌謀其他布局。到這裡也有觀眾提出疑問了，既然是監督，那為何不能直接接管戶部，給他個尚書又何妨？

慶帝心有忌憚，陳萍萍已經手握監察院，如果范建再一手掌管戶部，那慶帝的處境就有了一定的危險係數。與《琅琊榜》中梁帝的心中的顧慮一樣，慶帝從來不會允許兩個關係莫逆的好友同時掌管朝廷兩大擁有實權的官職，更何況一個掌兵，一個掌財呢？況且，從慶帝知道範建打牌九輸給范思轍就能看出，他也並不信任自己的這個發小。
