“建築女神”林徽因，一生的追逐，內涵女人的幸福良方

2019-12-25 1,841條評論 2,921次閱讀 2012人點贊

林徽因，絕對是近代中國思想解放開始後的一個新時代的新思想女性，她的才華可以說是現在很多女性沒法比較的，她的高情商是現在人不論是男性還是女性都少有的，智商更是超高的，她是令人欣賞的文學家，也是讓理工學子感嘆的建築學女神。

林徽因出名的而不光是她的文學才情和成功事業，還有她那被眾人追捧的愛情。

愛情，林徽因的愛情可是值得說一說的，她的丈夫是梁思成，公公是梁啟超，並且在新思想開始的初期，她就前往國外留學，並且也是思想已經部分西化。

她的愛情故事更是讓很多人侃侃而談，一直到現在有一半人在稱讚她，不過也有一半人對她的的愛情觀質疑。

她這輩子與三個男人都有不一樣的感情，不過只有她的正牌丈夫梁思成，是她一生的摯愛。

在與梁思成結婚之前，林徽因一直和徐志摩有著聯繫，並且兩個人還有書信往來，而初次見到林徽因的梁思成，已經對她動了心，不過這時的徐志摩已經有了妻子和兩歲的孩子。不過在兩人再次相見的時候，徐志摩與妻子張幼儀離婚，而林徽因卻與梁思成已經結婚，所以一切的相遇都是太晚了，時機總是這樣的錯過。最可惜的就是在1931年11月19日，徐志摩在前往準備參加林徽因的演講會，沒想到卻在半路墜機身亡。

而另一個男人就是金岳霖，他可以說是林徽因一生的精神伴侶，也可以說是最至高無上的愛。梁思成林徽因夫妻每個周末都有沙龍聚會，而金岳霖就是他們沙龍的最常客。因為他們文化背景相同，愛好相通，並且兩家人又是鄰居。金岳霖當年對林徽因的評價非常的高，讚美更是無法比喻。而每當林徽因與梁思成吵架的時候，金岳霖總是去調解勸說，最讓人稱讚的是，金岳霖從來沒有想過真正的擁有林徽因，只是在精神上幫助她，作為愛情而言，金岳霖這樣的愛情是最至高無上的，沒有人的愛情可以超過他。

可是回頭想想，為什麼這些博學才子偏偏只愛上林徽因這個女人，就連徐志摩為了她連婚也離了，她到底有什麼魔力呢？

其實林徽因的被人吸引不是因為她自身有魔力，細看林徽因的一生成就，我們就能明白。

林徽因，她的祖父是進士出身，並且歷官浙江、金華、孝豐等地，父親更是畢業於日本早稻田大學，擅長詩文、工書法，曾經任於北洋政府的司法總長等職。而林徽因本人因為家族的影響，也是從小受到熏陶，在成年後更是與梁思成出國留學交流，並且修完建築學，林徽因這一生在建築學上創造的成就很多，而且在文學上的成就，更是令人驚嘆，就連金岳霖曾經就用「人間四月天」來稱讚她。

可是林徽因不是特別美麗，不能用外貌吸引異性，可是她自身的才學獨特的魅力就已經將這些男人的心打動。作為女性她不美麗，但她絕對是有魅力的那一個。

所以作為現在的當代新女性，我們不光要在美貌上注重自己，更要在氣質內涵上補充自己。男人的心會因為美貌而變化，這個姑娘美麗他就會喜歡這個姑娘，那個姑娘美麗，同樣也是如此喜歡。可是自身魅力就不同，這個東西不是一天兩天就能有的，所以女人內涵很重要，如何做內涵且高情商的女人還需我們細細研究！
