大唐開國皇帝孫女墓被意外挖出，兩古盜洞直通墓室，考古發現不解

2019-12-22 238條評論 4,881次閱讀 3647人點贊

金鄉縣主墓的發現

本文作者 倪方六

在上篇的「梧桐樹下戲鳳凰 」頭條號中，我聊到了唐太宗李世民愛妃韋珪與前夫生的定襄縣主墓（詳見《考古發現「大唐女婿」墓，妻子為李世民愛妃與前夫所生》一文。這篇頭條號，再來聊一位「縣主」——金鄉縣主墓。

侍女，定襄縣主墓壁畫

「縣主」並不是現在認為的「縣領導人」，而是一種封號。據《唐六典》，唐朝皇帝之女封為「公主」，太子之女封為「郡主」，王之女封為「縣主」。金鄉縣主是唐朝開國皇帝李淵的孫女、滕王李元嬰之第三女。

她的墓於1991年在西安被意外發現，是她與丈夫、蜀州司法參軍於隱的合葬墓，具體位置在西安東郊潮橋鎮呂家堡村附近——灞河東岸、驪山西麓的銅人原畔。

金鄉縣主墓的發現比現，因為當地磚瓦廠取土。

1991年8月5日，西安市文物園林局文物處接到陝西省武警第五支隊打來的電話，說潮橋鎮呂家堡村西的於家磚廠，推士機取土時發現一座壁畫墓。

磚廠（無關本文）

考古人員即時趕到了現場。就此，金鄉縣主墓被發現了。

據參與考古清理的王自力後來撰文回憶，當時他們到現場後看到，此墓在西安至高陵的公路東側，附近有一個村辦磚廠，墓就在磚廠的西南角，緊鄰公路。

磚廠承包人稱，墓在8月4日已被發現，推土機推出一些陶俑，露出一段帶壁畫的墓道。不少村民聽說後，都趕來圍觀，看到挖出了好東西，趁機哄搶。巧的是，磚廠南面不遠處是陝西省武警第五支隊，武警主動出警，封鎖並保護現場，制止哄搶。

古墓發現現場（無關本文）

追繳回了50餘件陶俑。考古人員看到陶俑後，就判斷應是唐朝貴族大墓，所出土俑人有演奏樂器的騎馬俑、各種形態的男女立俑等，色彩鮮艷、造型生動，不是一般墓內能隨葬的。

墓坐北朝南。幸運的是，推土機推掉的僅是墓道南部的少部分，墓道東壁小龕被破壞，置於其中的陶俑暴露後被搶。墓道西壁小龕、甬道和墓室均未遭到破壞。

從8月6日開始，考古人員正式清理墓葬，到8月14日結束，前後8天時間，完成了對這座墓的清理髮掘。西安的8月酷暑難耐，考古人員在烈日下緊張清理，其辛苦程度是可以想像的。

古墓發現現場（無關本文）

8天，這個速度是很快的。考慮到墓葬在磚廠的取土範圍內，已無法保護，考古人員採取大揭頂的發掘方法，將墓室上封土全部挖掉，直接露出墓室。有武警保衛，也不擔心文物被盜搶。

在發掘時發現，金鄉主墓已被盜墓者光顧。

挖開表土層後，便發現了兩個大盜洞，直徑在60公分左右，足夠一人上下。

兩個盜洞從不同方向挖掘，但目標一致，直指墓室。一個通向墓室的面北角，一個通向墓室的東北角。

盜洞中已填滿了淤土，在東北角的盜洞中，考古人員發現了被打壞的三彩盤殘片。

唐壁畫墓室（非本文涉及墓室）

這是什麼時候的盜洞？從盜洞已被淤土填滿的情況分析，不會是近現代盜墓者乾的事，是古代盜墓者挖的。而且，這是兩個不同的盜墓者（團伙）所為。

他們光顧金鄉縣主墓的時間間隔不應該太遠，距金鄉縣主下葬的年代，也不應該太久。很可能就是盜墓之風正盛的唐末五代時期，因為盜墓時封土、墓頂都相當完好。

最先光顧金鄉縣主墓的盜墓者，應該是挖西北角盜洞的那伙人，而東北角盜洞那伙人應該是「濾坑」，或者說不知道前面已被人盜了。

考古中發現的古代盜洞（無關本文）

這兩次盜墓給金鄉縣主墓帶來了「滅頂之災」。盜洞破壞了原本較為穩固的墓室結構，雨水、地面水從盜洞倒灌入室內，長期的浸泡、滲漏，讓封土和墓頂鬆動，最終完全塌陷，室內落滿大塊的生土。這也是考古人員在清理時，採用大揭頂方法的又一原因。

清理掉室上部的泥土後，發現有仿屋檐的石頂部殘塊，推測有石槨。

如果真是石槨，墓主就不一般了。

石槨是石質外棺，狀如等比例縮小的屋子，有坡面屋頂、房脊、四周有圍板，其上設窗欞，高級貴族死後才會使用。在已考古發掘的隋唐貴族墓中，已發現了不少石槨。這些石槨本身就是一件精美的藝術品，上有線刻或淺浮雕，如唐武惠妃石槨、太平公主石槨，現在都是重要的館藏品。

唐代武惠妃石槨

因為奢侈，朝廷一度禁用。據《通典·禮四十五·喪制之三》「棺槨制」，「大唐制：諸葬不得以石為棺槨及石室，其棺槨皆不得雕鏤彩畫，施戶度牖欄檻。」能用石槨的，都是極少數的皇親國戚、重臣功勛，由皇帝特賜，否則都是越禮行為。

所以，有石槨的墓葬，墓主絕非一般。

待泥土全部清除後，果然是石槨，但已遭到嚴重破壞。石槨有底、壁、頂三部分，此槨頂已打壞，大部分不存，所餘數塊堆棄於墓室的四南角。對殘塊拼合後發現，當為高級的撫殿式石槨，為三開間結構，雕出了屋脊、瓦壟、滴水等，非常逼真，上檔次。

金鄉縣主石槨復原圖

但石槨頂部不全，四周的槨壁一塊也沒看到，留下的只有一個完整的底座。槨壁哪去了？當是被盜墓者弄走了，因其應該雕有精美的線刻圖畫，盜墓者看上了。

石槨內原有木棺，清理髮現早已腐朽，不見痕迹。人骨架還能看出來，基本上已朽成灰，一碰即散。僅在西南角處有零星的幾塊人骨。

從墓室的破壞情況來看，盜墓者進來很野蠻的，盜擾非常嚴重。石槨周圍的墓室中只找出已碎鎮墓獸、陶馬、駱駝等陶俑和陶器殘片。在石槨底座上的朽骨周邊，還發現了金筐寶鈿飾品、鎏金銅花飾及三彩盅等小件物品。

金鄉縣主墓修復後的鎮墓獸

從這些遺落之物分析，應該是是金鄉縣主收殮時的飾物。可以想見，當年金鄉縣主收殮入棺時，身上應該是珠光寶氣，打扮得雍榮華貴；石槨內放滿了金銀玉器珠等隨葬品……可惜，都讓盜墓者盜走了，破壞了。

使用如此豪華葬具，墓主會是什麼人？在清理了石槨後，考古人員尚不知曉。令人興奮的是，在清理甬道和墓室時，發現了兩合青石墓誌。

「大周故朝散大夫於君志銘」拓片

其中較大的一合墓誌，邊長75厘米，厚155厘米，平放在墓道北部正中。志蓋扣於志石上人，但發現時志蓋已錯位，中間還有黑色灰痕。

怎麼回事情？原來，在放置墓誌石時，上面應該覆蓋了絲織物，然後再合上志蓋，磨爛後便成了黑色。而志蓋錯位，則顯示盜墓者進入墓室後，曾撬開墓誌。

擦拭看到，志蓋上有:「唐故金鄉縣主志銘」9個篆字。至此，墓主的身份揭開——原來是女貴州，金鄉縣主。

唐朝墓誌

而另一方較小的墓誌，是在墓室內發現的，志蓋已被起開，靠在墓室東壁上。志蓋陰刻「大周故朝散大夫於君志銘」12個篆字。不用說，是女墓主的丈夫。

隨後經對志文的釋讀、整理，弄清了金鄉縣主的生世。原來，金鄉縣主是大唐王朝開國皇帝李淵的孫女，父親是滕王李元嬰。

該墓，是金鄉縣主與其丈夫蜀州司法參軍於隱的合葬墓

李元嬰這個人不少網友可能不知道，但提起江西的「滕王閣」大家應該知道的，唐詩人王勃《滕王閣序》，讓此建築名傳千古，至今此文還被選入學生的課本——滕王閣就是李元嬰建的。

因為李元嬰初封地在今山東滕州，他在那築了滕王閣。後駐洪州（今江西南昌）、隆州（今四川閬中），又先後築了另兩處滕王閣。

長大後，金鄉縣主嫁給了在武則天當皇帝時任朝散任大夫，行蜀州司法參軍的於隱。

於隱死於武則天永昌元年（公元689年），次年下葬。金鄉縣主死於唐玄宗開元十年（公元722），兩年後和丈夫於隱合葬，夫妻倆下葬時間相距34年。

唐墓發掘現場

但考古清理時發現了不解的地方。

此墓為單室方形土洞墓，墓室3.4×3.5米大小，只有3天井、2小龕，與出土石槨的情況不相匹配。怎麼回事情？越禮下葬？

後來分析認為，此墓是當年為埋葬於隱而建的，他是從五品下級別，只能這麼大。而金鄉縣主級別是從二品，高很多，可建豪華墓的，但她並沒有另建，而是葬到丈夫墓一起。

但這不合禮制、降級了啊，怎麼處理？提高陪葬品等級。

金鄉縣主下葬時，將以前丈夫的低級隨葬品全部清出，換上高級的隨葬品，如此而已。

彩繪騎馬帶獵豹狩獵男俑

彩繪帶猞猁狩獵女俑

所以，能從金鄉縣主墓中出土塑造藝術高超、燒遣技術精的彩繪騎馬狩獵俑、伎樂俑、鼓吹儀仗俑等精美隨葬品。其中的戴孔雀冠的騎馬女樂俑、裸袒相博的角抵相撲俑、胡姬俑，在當年都屬首次發現，極其珍貴。

這些隨葬品製作水平，是普通民間藝人無法達到的，應是皇家專門用來給貴族隨葬的「東園秘器」，為朝廷所賜。
