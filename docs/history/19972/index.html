五位名人在世時意氣風發，逝世前會留下些什麼遺言呢？

2019-12-17 224條評論 2,054次閱讀 1076人點贊

人的一生不過短短不過數十載而已。在經歷了完自己漫漫人生路的時候，回首自己的一生。想必會有很多所思所想吧！又會留下一些怎麼樣的感慨呢？是對自己碌碌無為的一生感到遺憾？還是為自己的兒女未來擔憂，還是……？讓我們來看看這五位名人。在逝世前留下了什麼遺言？

5. 唐太宗

唐太宗，締造了貞觀之治的明君。他的一生可謂意氣風發。至玄武門之變後，他開疆擴土，勤政愛民，積極聽取群臣的意見，以文治天下。拓塞北，戰西域，征高句麗。被西域諸國國君稱為「天可汗」！。在臨終前他說了三句話。第一句是對太子說的：「汝能孝愛如此，吾死何恨？」 ，第二句是對中書令長孫無忌和褚遂良（感興趣是可以在上面搜索框搜下試試）說的：「朕今悉以後事付公輩。太子仁孝，公輩所知，善輔導之！」，第三句是單獨對褚遂良說的，而這一句，後世也被稱為神預言，「無忌盡忠於我，我有天下，多其力也。我死，勿令讒人間之」。他擔心太子會聽信讒言，離間太子與長孫無忌的關係。不得不說他真是神預言呀！在他死後，顯慶四年，也就是公元659年。許敬宗指使人向唐高宗呈奏密折，稱監察御史勾結長孫無忌，企圖謀反。唐高宗也不與長孫無忌對質審問。便下詔削去他的官職和封邑，流徙黔州。

4. 岳飛

靖康恥猶未雪，臣子恨何時滅？ 北宋名將岳飛，曾率領岳家軍揮師北伐，英勇抗金，收復失地。然而在進軍朱仙鎮時，宋高宗卻一味求和，以12道金令下令退兵。無奈岳飛只好被迫班師。其後被奸相秦檜以「莫須有」罪名所害，命喪風波亭。岳飛臨時臨死前說只說了一句話：「悔當初不聽金山道悅禪師之言！」。這是怎麼回事呢？原來岳飛被12道金牌從朱仙鎮召回時，途經金山江天寺，道月禪師勸他出家，不要回京。岳飛忠心耿耿，他沒有聽道悅禪師的話，堅持南歸。臨走時道悅禪師贈了他一句話：

「歲底不足，謹防天哭；

奉下兩點，將人荼毒。」

大意就是年底天上下雨的時候，奉下兩點既為『秦』字，一個姓秦的人會害你。

在風波亭的除夕夜，當時天下大雨，岳飛在風波亭被秦檜謀害，那一天剛好是十二月二十九日，也應了道悅禪師的話。岳飛當時不知其意，直至含冤被害，方才頓悟，悔之晚矣！

3. 王陽明

我國著名的思想家，哲學家王陽明。57歲的他在兩廣監察軍務的時候， 舊疾複發，他有很不好的預感，於是便上疏請辭。 還未等到朝廷的通告，便自行離去。在途徑梅嶺到江西安南的船上。病情加重，又在安南停留了五日便病危了。臨終前，徒弟哭著問他：「先生有何遺言？」，楊明微微笑道：「此心光明，亦復何言？」便閉目而逝了。陽明先生在仕途上忍辱負重、遭遇不公。但仍不斷追尋自我，追求光明。他的理論建樹，來自於自我修鍊和實踐鍛煉，他為了實踐朱熹的「格物致知」中「物有表裡精粗，一草一木皆具至理」的學說，下決心窮竹之理，「格」了七天七夜的柱子，什麼都沒有發現，人卻因此病倒，從此他便對「格物」學說產生了極大的懷疑，更是促成了心學的誕生，提倡從自己內心中去尋找「理」。他找到了「理」，亦找到了內心的光明。

2. 張之洞

1909年10月4日，晚清重臣張之洞病入膏肓，這年他73歲。彌留之際，他將幾個兒子叫到床前，告誡他們「我病知不能好矣。我一生做人，志在正字、忠字，公忠體國，廉正無私，我可自信。此多之心術也。學術僅行十之五六，治術僅行十之六七。平生不樹黨，不殖產，自幼不爭財產。指公子輩雲，汝等須記得此諭：兄弟不可爭產，志須在報國，勤學立品；君子小人，要看得清楚，不可自居下流。」眼看著老父親邊喘息邊叮囑「遺言」，幾個兒子也不由得一邊哽咽，一邊答應遵命行事。這時倒是張之洞開始安慰起兒子們，說他並不感到自己有什麼痛苦……不一會，這位在中國近代史上深具影響的著名人物的生命走到了終點。

1. 曾國藩

清朝同治十二年，曾紀澤（兒子）攙扶62歲的曾國藩在花園散步，曾國藩告誡兒子，曾家後代不要再去帶兵打仗，因為打仗是最造孽的事了（傳說同治三年七月，曾國藩帶領湘軍攻破天京，對無辜平民展開屠殺與搶掠。當時的南京城被燒毀，平民死傷無數，南京人稱曾國藩、曾國荃兄弟為「曾剃頭」、「曾屠戶」）。此時的曾國藩已經油盡燈枯，一陣風吹過，他直呼腳麻，便倒在兒子身上，再也說不出話了。曾紀澤慌忙把曾國藩扶進書房，躺在太師椅上，一眾曾氏後人也慌張的趕到了書房，曾國藩艱難的抬了抬手，指了指桌子上早已寫好的遺囑，大致內容歸納為四則：

慎獨則心安。（能自律則心裡平和。）

主敬則身強。（精神狀態好則身體強健。）

求仁則人悅。（追求仁愛則人也會高興的依附。）

習勞則神欽。（ 自食其力則鬼神也會稱許。）

之後，一代名臣就此溘然長逝！曾氏後人恪守先祖遺言，後代子孫沒有一人蔘軍，也極少人出仕做官。而曾氏後人也潔身自好，曾國藩去世一百多年間，家裡也沒出過敗家子。
