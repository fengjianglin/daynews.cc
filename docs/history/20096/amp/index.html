“三顧茅廬”的真相，淺析諸葛亮出道計劃的重要性，職場秘籍

天天要聞
2019-12-16

有關諸葛亮和劉備的故事，最有名的應該是「三顧茅廬」了。「三顧茅廬」說的是劉備為了請諸葛亮出山相助自己，三次去到諸葛亮住的茅廬請諸葛亮，在這種誠意的感動下，諸葛亮終於同意出山相助劉備，而劉備這種求賢若渴的行為也為後世所稱讚。「三顧茅廬」也成了求才的代名詞了。

然而，在局勢動蕩，群雄輩出的三國，諸葛亮為什麼會投靠勢單力薄、缺人少物的劉備呢？他沒有更好的選擇嗎？我認為諸葛亮正是運用了一些方法，讓劉備來「三顧茅廬」，讓自己能以高姿態進入劉備集團的核心決策層的。

諸葛亮滿腹經綸待價而沽

諸葛亮，字孔明，號卧龍，當時有傳言「卧龍鳳雛得一可得天下」，可想而知諸葛亮的學識非同一般，可以說是獨步天下，名聲在外，待價而沽。

諸葛亮作為一個傳統的知識分子，他滿腹經綸，學富五車，當然想報效國家，爭個封妻蔭子、光宗耀祖。

沒有金剛鑽不攬瓷器活，才能諸葛亮已經具備，就看誰來「沽」了。

環視群雄勢力比拼

1、曹魏集團

曹操給人的印象是雄才大略，雖求賢若渴但疑心很重，最重要的是曹操沒有提出「匡扶漢室」的口號。

196年，曹操接迴流浪在外的漢獻帝並遷都許昌，隨後，漢獻帝封曹操為司空，行使車騎將軍權，百官都以曹操唯首是瞻，曹操權頃一時。

鐵三角謀士團隊的形成：

191年，被稱「第一謀士」的荀彧投奔了曹操。

196年，被稱「謀主」的荀攸經荀彧推薦投奔了曹操。

196年，郭嘉經荀彧推薦投奔了曹操。

可以說荀彧組團承包了曹操的謀士策劃工作，而且每個人的能力都是堪當大任的，這樣的團隊氛圍讓諸葛亮知道難以插針。

2、孫吳集團

孫權從小就隨父兄孫堅孫策征戰，在江東打下了基業，在父兄去世後繼任並在19歲被朝廷任命為討虜將軍，可謂年少有為。人才的早期儲備，由於有父兄打下的江東基業，孫權直接繼承了地盤和人脈（有周瑜、魯肅、呂蒙等人才），都是沾親帶故的，大家相處融洽配合默契，估計諸葛亮評估過也是不適合投靠的。

3、袁氏集團

198年，袁紹親率大軍攻打併佔領幽州，兼并了公孫瓚的軍隊。自從佔據冀、青、幽、並四州，袁紹的實力更加雄厚了，不過他的野心也日益膨脹起來，而他給漢獻帝的進貢是日漸稀少了。

由於異母弟袁術曾在197年在淮南稱帝，因眾叛親離走投無路來投奔袁紹，慷慨地將「帝號」讓給袁紹，袁紹躍躍欲試，讓親信耿苞尋找稱帝依據並散布言論，結果被大家認為是妖言惑眾，應該殺頭，袁紹才知道時機未到而作罷。

由此可見，袁紹也沒有正統的「匡扶漢室」理念，這個離諸葛亮的理想相差甚遠，諸葛亮也不會去投靠的。

4、蜀漢集團

劉備，人稱「劉皇叔」，西漢中山靖王劉勝之後，正統的漢室之後，名正言順，而且性格溫厚，禮賢下士，名聲在外。劉備個人最大的特點是不管自己的實力是多少的弱小，遇到怎樣的困難都是百折不撓在向前努力，正是這種個人的魅力吸引著諸葛亮。

當時劉備身邊得力的謀士不多，信任的只有關羽和張飛，但他們只是猛將，不會運籌為握，所以急需能謀劃千里的人才，諸葛亮肯定知道劉備的需求，如果自己投奔劉備，肯定很快得到重用的，可以近距離實現自己的理想。

更重要一點的是，劉備是以「匡扶漢室」為己任的，這符合傳統的知識分子思想。

多方思慮，巧施三顧茅廬之計

1、讓自己名聲在外，吸引注意力

在隆中隱居的時候，常常以自己和管仲、樂毅相提並論，用「華夏第一相」（管仲）的光環籠罩自己，也就是現在所說的要包裝自己，讓自己身價百倍。

2、欲迎還拒，自顯身價

當劉備希望徐庶引見諸葛亮的時候，徐庶卻告訴劉備人才都有傲氣，需要親自去請的。結果劉備「三顧茅廬」去請諸葛亮了。

小結：

諸葛亮作為傳統的知識分子精英，對當時的形勢肯定有精準的分析，他運用自已的聰明才智，讓自己以高姿態進入劉備的視線，讓劉備自動來「三顧茅廬」，從使自己一舉成名天下知，站在了人生的制高點。

諸葛亮雖然用計讓劉備「三顧茅廬」，但是他確定做到了「鞠躬盡瘁，死而後已」，用自己的一生承擔起振興蜀漢的重任，不負劉備的重託，也讓諸葛亮這個名字在史書上留下閃光的一頁。
