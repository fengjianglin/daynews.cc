隋唐霸王銀槍彎刀的燕王羅藝，李世民為什麼要置他於死地

2019-12-21 1,640條評論 4,299次閱讀 572人點贊

羅藝是武藝高強是隋末年間赫赫有名的大人物，他鎮守幽州，號稱幽州王。羅藝是名副其實的官二代，他的父親是隋朝的將軍，自幼就跟隨父親學習兵法和武功。常年的接觸戰場讓他擁有一身帶兵打仗的本領，於是他決定要擁兵自重，在天高皇帝遠的地方雄霸一方。

羅藝

在《隋唐演義》中我們知道羅藝的兒子是號稱天下第七的銀槍少保羅成，他的孫子是戰功赫赫的羅通，他的外甥是兵馬大元帥秦瓊。他還有兩個必殺武器，一個是銀槍彎刀，另一個是號稱魔鬼部隊的燕雲十八騎，可以說羅藝在當時是個響噹噹的人物。

秦瓊、羅藝

演義畢竟是演義，那麼真實歷史上羅藝是怎樣的呢？他又是怎麼死的呢？歷史上真實的羅藝其實鎮守在涿郡，在當地自立稱王。後來隋朝被滅後，羅藝就投靠了唐軍李淵。

羅藝、羅成

歸唐後，羅藝被封為大將軍，奉命前去圍剿竇建德和劉黑闥一夥叛軍，在羅藝的領導下很快就將叛軍一舉剿滅，立下了大功，李淵就封他為左翊衛大將軍。此時的羅藝有點剛愎自用、目中無人。當時身為秦王的李世民想要去拉攏羅藝，被劈頭蓋臉的一番羞辱從而得罪了李世民。

李世民

羅藝投靠唐軍是最先認識的是太子李建成，於是想抱這棵大樹，就投靠了太子這邊，稱為太子一黨。後來發生了玄武門事變，秦王李世民當時了皇帝，這樣羅藝開始擔心會不會受到報復，於是每天寢食不安，擔心受怕，再三決定還是先下手為強。

羅藝

羅藝以前就是個山大王，據說當時有個有名的巫師給羅藝夫妻看過像，說：「羅藝有君王之相，羅夫人有貴氣能夠母儀天下。」這讓羅藝有了叛心。

羅藝

羅藝最後糾集兵力，開始造反，他謊稱接到皇帝密詔，帶兵準備圍住皇宮。唐太宗李世民得知此事後大怒，派長孫無忌和尉遲敬德大將軍前去平叛，最後羅藝走投無路死在叛軍之中。羅藝他高傲自大，起兵謀反，曾經還得罪過李世民，所以李世民萬萬留不得他，才下令將他誅殺。
