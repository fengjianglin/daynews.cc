五代有許多低調的王們，他們低眉垂手，卻早已有了鼎立稱雄的能力

眾所周知，五代十國是一個混戰的年代，各種國家更替出現。中原五個朝代（後梁、後唐、後漢、後晉、後周）前後相繼，最短的後漢只存在了四年。中原以外還有十國。十國中有九個在秦嶺淮河以南。沿長江由西而東，分成巴蜀、兩湖、江淮、兩浙四個地區，在加上福建、兩廣，一共六個地區。南方九國先後分別在這些地區中活動。巴蜀先有前蜀，後有後蜀。兩湖則荊南佔有以湖北江陵為中心的一小片地方，楚佔有幅員廣大的湖南，這個地方始終存在著這一大一小的國家。江淮先有吳，後有南唐。他們的版圖西到鄂東，全有江西，是南方最大的割據勢力。其餘兩浙的吳越、福建的閩、廣東和廣西一部分的南漢，各自佔有一個地區。十國中只有一個在北方，即北漢，佔有者今山西省的大部分荷陝西的東北角。此外，契丹正式在北方崛起。當時的中國，怎一個亂字了得。

然而話有說回來了，有敵手的電視劇才好看，有對手的競爭比賽才更加精彩。我一直堅信，一枝獨秀的歷史不好看。在這亂世的年代裡，他們誰都有可能統一天下，又誰都有可能兵敗廬山，一切都充滿了未知數。

細讀這段歷史，你會發現，真實的五代絕非僅此熱鬧而已，有許多低調的王們，他們低眉垂手，卻早已有了鼎立稱雄的能力。他們雖然並沒有建立國家，但實際上已經割據一方。今天，就讓我們談談這些低調的王們。

1. 曹義

唐代後期，沙洲人張議潮擺脫吐蕃統治，以河西歸唐，任歸義軍節度使。以後唐無力西陲，張氏成為一個地方政權。五代時，張家的親戚曹義成為這個政權的首腦。曹氏政權在五代期間並沒有稱帝，直到宋仁宗景祐二年（1035）才被西夏所滅。

2. 劉仁恭

公元894年，李克用攻下幽州，但戍兵橫暴，失盡人心。降將劉仁恭利用這個機會，脫離河東的管轄。從此，劉仁恭在亂世中獨處一偶，卻雄霸一方。朱溫勢氣最盛時，曾挾天子以令諸侯。稱帝之後，舊藩鎮也都被之吞併。原來能與朱溫想抗衡的李克用父子也以苟延殘喘，李存勖嘆息道：「現在能與朱溫抗衡的，也只有幽滄的劉仁恭了。」由此可見劉仁恭政權的實力有多麼雄厚。其實他的實力遠比那些南方某些國家強得多，卻一直沒有稱王，所以並沒有算入十國之列。

3. 李茂貞

唐昭宗大順元年（890年），李茂貞受封為隴西郡王。李茂貞這些很高的官爵使他有了割據的資本。 李茂貞從此便將鳳翔當成了他的根據地，全力經營，他也尋找機會擴充地盤。宦官楊復恭在朝廷無法立足，跑到養子興元（今陝西漢中）節度使楊守亮那裡，一起反叛。景福元年（892年），李茂貞攻克興元府，斬殺楊復恭父子。遂兼并山南西道，成為關中一帶最具實力的藩鎮。李茂貞沒有出兵討伐朱溫，也沒有稱帝，但他卻做起了事實上的皇帝：開設岐王府，設置百官，以其所居為宮殿，其妻稱皇后，各種儀仗都跟皇帝一樣。

4. 周行逢

公元952年，湖南將領劉言趕走唐軍。不久，眾將爭奪權力，連續演出幾幕互相殘殺的慘劇。劉言、王逵相繼被殺後，周行逢掌握大權，殺戮異己，才恢復了平靜的狀態。周行逢出身低微，注意選擇廉潔的官吏，執行於民休息的政策。他的女婿想做官，行逢不肯給他做，說：「你的才幹不能稱職，還是回家種田的好。」他自奉很樸素，有人批評他過於簡省，他說：「馬家窮奢極欲，得到滅亡的下場，難道可以效法嗎？」湖南在他的統治下回復了元氣。周行逢在公元956年取得湖南，名義是受後周任命的武平節度使，實際上卻割據一方，但歷史上並不算入十國之內。

這些實際的割據者都有稱帝的實力，卻都沒有這樣做。他們隔絕與十五國紛爭之外，修養生息、保存實力，並且大部分得以善終。他們是聰明的人，不圖一個空空的虛名。他們不像有些人，為圖一個帝號空名而身死國滅，此乃真勇士也！
