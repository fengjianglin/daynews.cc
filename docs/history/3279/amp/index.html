現存10個歷史名人身體部位：從伽利略的中指到愛迪生的最後一口氣

2019-12-07

摘要

1.貝多芬頭骨的一部分

當著名作曲家和鋼琴家路德維希范貝多芬於1827年去世時，醫生在屍檢中取出了他的耳骨，試圖研究這位部分失聰的作曲家耳朵的奇怪情況。但據推測，這些骨頭是一名醫務人員偷來的，他將這些骨頭賣給了一名「外國醫生」

耳骨再也看不到了，但貝多芬的頭骨碎片在2005年再次出現。其中一部分是在加利福尼亞發現的。據說，它們是貝多芬家族從維也納傳下來的。今天，這些頭骨碎片存放在加州聖何塞州立大學貝多芬研究中心。

2.愛因斯坦的大腦

阿爾伯特·愛因斯坦是20世紀最偉大的科學家之一。1955年他去世後，為了學習和研究的目的，他的大腦被切除了。第一項研究完成後，大腦就消失了。沒人知道發生了什麼事。

1978年，愛因斯坦的大腦被托馬斯·斯托爾茨·哈維博士「重新發現」，他是第一個切除並研究大腦的醫生。2010年，大腦被捐贈給國家健康和醫學博物館。

3.伽利略的中指

義大利科學家、數學家伽利略·伽利略的中指在去世95年後被折斷。這根手指被傳了幾百年，直到它被送到佛羅倫薩科學史博物館，直到今天還在那裡。

4. Rasputin的生殖器

沙皇及其家族的著名神秘顧問格雷戈里·拉斯普京（Gregori Rasptin）也因多次性侵而臭名昭著。他聲稱性冒險和性狂歡是宗教經歷的一部分。1916年，拉斯普京遭到中毒、棍棒、槍擊，最後溺水身亡後，他的陰莖被神秘地從身體上割下。

不知怎的，在20世紀20年代，拉斯普京被割斷的陰莖來到了巴黎，據說是一群俄羅斯女性崇拜的對象。拉斯普京的女兒瑪麗亞發現後，強烈反對這一邪教。她奪回了他父親的身體的一部分。

今天，在聖彼得堡的俄羅斯情色博物館裡，拉斯普京臭名昭著的巨大、畸形的陰莖正在展出。

5.拿破崙的生殖器

據說，在1821年拿破崙的屍檢中，一位憤憤不平的醫生切除了他的陰莖。直到1916才被再次見到，當時它出現在倫敦的「Vigali收藏」中。後來，它四處遊盪了一段時間，直到20世紀70年代中期，一位美國泌尿科醫生約翰·金斯利·拉蒂默（D.r John Kingsely latimer）以3000美元左右的價格買下了它。拿破崙的陰莖至今仍留在美國。

6.佛牙

佛教的創始人高達摩佛於公元前6世紀在印度去世。佛陀被火化了，但他的追隨者保留了一顆牙齒作為神聖的遺物。今天，這顆牙齒在斯里蘭卡坎迪的一個大型黃金聖骨匣里展出。

7.亞伯拉罕·林肯的骨頭碎片

林肯總統中槍後，醫生試圖取出他頭骨內的子彈，但這樣做，他們帶出了血液凝塊和幾個頭骨碎片。

林肯總統頭骨的碎片今天可以在國家健康和醫學博物館的展覽上看到，就在殺死他的子彈旁邊。

8.切·格瓦拉（Che Guevara）的頭髮

1967年切格瓦拉在玻利維亞叢林中被軍隊殺死後，一名中情局特工剪掉了他3英寸長的頭髮。在一次拍賣會上，格瓦拉的頭髮以及他的指紋和照片以11.95萬美元的價格賣給了一位德州書商。這位德州的書商自豪地在休斯頓的書店裡展示了這兩件東西。

9.貝尼托·墨索里尼的大腦

1945年義大利獨裁者貝尼托·墨索里尼被處死時，美國政府要求對他的大腦樣本進行研究，同時也是重要的戰利品。1966年，墨索里尼去世21年後，美國政府把他的部分大腦還給了他的遺孀拉切爾·墨索里尼。2007年，墨索里尼的大腦碎片在eBay上突然出現，當時的售價為22000美元。

10.希特勒的陰莖

顯然，一名俄羅斯士兵割下希特勒的生殖器，並將其作為紀念品。在本世紀初，希特勒的私人會員被出售。說真的，歷史人物的陰莖怎麼了？

11.托馬斯·愛迪生的最後一口氣

從科學的角度講，這可能不是身體的一部分。亨利福特博物館展出了一個試管，據說裡面裝著美國發明家愛迪生的最後一口氣。

亨利·福特和愛迪生是好朋友。在愛迪生生命的最後幾年裡，福特在佛羅里達州冬季莊園旁買了一套房子。

福特1947年去世時，在他的東西中發現了一個裝有愛迪生最後一口氣的試管。據說福特請愛迪生的兒子查爾斯，在愛迪生奄奄一息的時候，把他呼出的氣封起來。
