康熙親自探望50歲親哥哥，他離開之後，哥哥說：我終於可以死了

2019-12-10 345條評論 1,757次閱讀 779人點贊

康熙皇帝常常對臣下說：「無祖母無以至今日」。由此可見，康熙和祖母二人的感情甚篤。這是為何？

康熙皇帝的父親順治失去愛妃之後十分消沉，因而更加信佛以求解脫，甚至還有了出家的念頭。順治的母親孝庄皇太后對此十分擔心，堅決地阻止了順治削髮為僧的行動。

孝庄劇照

順治皈依佛門未成，對愛妃的思念與日俱增。他身心俱疲，因此再也不提立太子的事情了。在孝庄皇太后的再三催促下，順治終於同意在年幼的皇子中選一個人，讓他成為嗣君。

康熙何以在嗣君競選中取勝？

皇長子二歲夭折，皇五子、七子年紀太小，只有皇次子福全和皇三子玄燁是被立為嗣君的最佳人選。

按理來說，應該選年長一些的福全為太子，那為何清朝皇室最終選擇了玄燁？

孝庄劇照

孝庄皇太后喜愛玄燁實為原因之一，但順治帝有些不情不願，因為玄燁的臉上因得過天花而留下了一些痘痕，而且順治害怕天花會複發，以致疾病會隨時奪去玄燁的生命。

皇帝和皇太后意見相左，因此當二人在事關清王朝命運的繼承人問題上僵持不下的時候，便去請教了西方傳教士——湯若望。

湯若望與皇太后、順治皇帝的私人關係非常不錯

早前順治的皇后博爾濟吉特氏患病，孝庄皇太后就曾派遣侍女向湯若望求醫。湯若望根據來人所說的癥狀推測，皇后的病情並不嚴重，於是將一面聖牌交給侍女，囑咐她將此物掛在皇后胸前，即可除病消災。據說事後果然應驗，孝庄皇太后十分感激，還命皇后認湯若望為義父。

順治劇照

順治皇帝對湯若望也有好感。他經常請湯若望到宮中敘談，而這位大鬍子西洋長者，以其淵博的學識和高尚的品德贏得了皇帝的心。順治稱呼老神父為「瑪法」，滿語即「尊敬的老爺爺」。此稱謂里包含著晚輩對長輩、弟子對老師的雙重感情。他甚至詔賜湯若望為「通玄教師」。

面對皇帝和皇太后的詢問，湯若望給出了答案：玄燁雖然年齡比較小，但他已經出過天花，對此有了免疫力，不會再得天花了；而福全雖然年齡較長，但他尚未出過天花，時時都要小心這種可怕的、可以致死的疾病。

孝庄劇照

這便是福全錯過皇位、玄燁登上寶座的理由。玄燁八歲登基，是為康熙帝；而九歲的福全，被封為裕親王。

成年之後的兄弟二人境遇並不相同

康熙成年以後，出過天花的麻臉並未影響他的威儀，一位康熙身邊的傳教士曾記錄下康熙的外貌：「他雄壯威武……五官端正……雖然臉有天花留下的痘痕，但並不影響他的英俊的外表」。

成年後的福全，亦是十分英勇善戰。康熙曾命福全征討噶爾丹，福全接到命令後便馬上出兵。他在戰鬥中的表現十分出色，打得敵軍狼狽逃竄。得勝後，他便下令班師回朝。

康熙劇照

福全自行撤軍的行為，引起了康熙皇帝的不滿。待福全回朝之後，康熙決定議罪。福全沒有爭辯，他流著淚說：”我復何言！”最終，康熙取消了福全的議政權。

福全從此不問朝政。十五年後，五十歲的福全生病，康熙親臨其府內看望自己的哥哥。福全感受到康熙仍十分重視兄弟情誼，不會不善待他的後代們。因此康熙剛走，福全便安心地對自己的兒子說道：「我終於可以死了」。幾天後福全便去世了。
