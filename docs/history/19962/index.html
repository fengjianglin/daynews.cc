老照片，圖二于鳳至年輕時的端莊美照，圖七婚宴上郭沫若容光煥發

2019-12-17 578條評論 1,667次閱讀 336人點贊

老照片，宋美齡和蔣介石的結婚照。1927年12月1日，上海大華飯店的一場婚禮格外引人矚目，這就是30歲的宋美齡和40歲的蔣介石的婚禮。只見宋美齡頭戴一個用花蕾編成的花冠，手捧一束紅白相間的玫瑰花。蔣介石更是西裝革履，神采奕奕。之前蔣介石為了打消宋家顧慮，連續三天登報發表離婚聲明。這場蔣宋聯姻被視為政治婚姻，卻持續了半個世紀。

1916年的于鳳至和張學良，青澀的面孔上是飛揚的青春。19歲于鳳至端莊秀麗，一看就是出自大家閨秀，難怪連公公張作霖都對她讚譽有加。一旁站立的張學良消瘦青澀，二人度過了幾年恩愛時光。于鳳至愛了張學良一輩子，1990年，已至93歲高齡的于鳳至在病重之際，遺言將名下所有財產留給50年未見的張學良，並留下「雖不同生，死要同穴」八字遺囑。不過最後的遺願也未實現，張學良墓穴里是趙四小姐的陪伴。

嫁為人妻的川島芳子，她一生閱男無數，可丈夫只有一個就是蒙王甘珠爾扎布。據說當年為了追求川島芳子，甘珠爾扎布竟然一度得了相思病。二人婚宴非常隆重，可婚後的川島芳子就不是做家庭主婦的料，更無法忍受甘珠爾扎布為她安排的安逸生活。一年多後，川島芳子就不辭而別，跑回了日本。逃婚之後的川島芳子如魚得水，開始在自己的間諜生涯中大展拳腳，成為日本人倚重的女間諜。

民國女神林徽因和梁思成的結婚照，二人郎才女貌，天造地設的一對。1928年3月，在加拿大溫哥華，林徽因與梁思成結束了6年的愛情長跑而喜結良緣。林徽因不喜歡外國的白色婚紗，她身穿自己設計的中國式婚紗，意境唯美，而梁思成買了一身禮服，也是英俊瀟洒。在他們結婚的當天，公公梁啟超在老家發請柬，宴請賓客好友，不過新娘和新郎卻不在場。

西方打扮的婉容和溥儀，看起來也很般配。穿旗袍的婉容溫潤如玉，不愧是王府出身，一身西裝打扮的溥儀頗有書生氣。二人曾在紫禁城舉行婚姻大典，當時的大清已經滅亡十多年了。新婚之夜二人卻各處一室，溥儀在養心殿睡了一夜。婉容在天津的時候很時髦，生活的卻很壓抑。溥儀的淑妃忍無可忍與溥儀離婚，婉容也跟著遭了殃。貪慕皇后虛名的婉容與溥儀侍衛偷情，據資料記載，她的孩子出生沒多久，就被溥儀弄死了。

民國名媛陸小曼和徐志摩的結婚照，1926年10月，北京北海公園，徐志摩和陸小曼的婚禮上演尷尬一幕。徐志摩的老師梁啟超是證婚人，希望二人痛徹前非，但願這是他們最後一次婚姻。「新人及滿堂賓客無一不失色，此恐是中外古今所未聞之婚禮。」婚後他們移居上海，陸小曼一個月500大洋的花費讓徐志摩疲於應付。來往於上海和北京之間，最後飛機失事化為一團煙火。陸小曼以後沒有再結婚，卻和翁瑞午同居了三十年。

在婚禮上朗誦詩歌的郭沫若和於立群。1939年元旦，在重慶禮堂，47歲的郭沫若與23歲的於立群結婚，作為郭沫若的第三位妻子，於立群是一名典型的大家閨秀，年輕貌美。此時的郭沫若是抗戰文化戰線上的旗幟，更是容光煥發，萬眾矚目。而當時他的原配張瓊華正在四川老家侍奉公婆，日本妻子郭安娜正在日本含辛茹苦的為他撫養五個孩子。晚年兒子郭博評價父親：「對於家庭，郭沫若是個罪人。」可謂一言既出，四座皆驚。
