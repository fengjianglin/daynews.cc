曠世才女張愛玲，沒有死於人情冷暖，而亡於與虱子殊死搏鬥之中

2019-12-19 1,198條評論 2,217次閱讀 90人點贊

是蚊子血還是硃砂痣，是飯米粒還是白月光，張愛玲一聲嘆息，道盡情愛間的人情冷暖。她的一生，創作了許多關於愛情的小說，而她也因早年的晦澀生活對愛情過於執念，最終釀成愛的悲劇。她的才華，可能是上天為了彌補她的苦難而賜予她的，最終，她懷揣著這份才華和苦難，孤身離世。

張愛玲的童年並不幸福，無能的父親，拋夫棄女遠渡重洋的母親，惡毒的繼母。她的不幸，一開始便已註定。缺愛的童年令她對待親情過分的淡漠，卻轉而對愛情十分執拗。她為了愛情，曾經「低到塵埃里」。對胡蘭成，她愛的毫無保留。而胡蘭成回饋的，確是招蜂引蝶的浪蕩行徑。

胡蘭成是張愛玲此生的摯愛，雖然她得償所願成為胡妻，卻因他而聲名受累。如此飛蛾撲火的愛情，最終止於胡蘭成的始亂終棄。明明如此渴望愛情的才女，卻偏生情路坎坷。胡蘭成欽慕的是張愛玲的才情，但張愛玲錯把這份惺惺相惜當成了愛情。這段婚姻的最初，便只是一場誤會。此等情殤，令張愛玲在很長時間內對愛情退避三舍，無法忘懷的胡蘭成，已成心病。

張愛玲沒有死於人情冷暖，卻後半生都在和虱子殊死搏鬥。她愛穿旗袍，總將自己打扮的光鮮亮麗，可她卻說生命這件美麗的旗袍，長滿了虱子。由此可見，她深受虱子之苦，頻繁搬家的目的也只是為了躲避虱子的煩擾。張愛玲用了無數辦法去消滅這些虱子，卻全無所用。

而觀張愛玲的行為習慣，可知其熱愛整潔，善於打理，對衛生的要求嚴格，何來的虱子？想來應是她的心病。不得善終的愛情，那段抑鬱的歲月，還有一生的痛苦以及不如意，已經烙印在張愛玲的心中，無法磨滅。如同那小小的虱子，四處跳竄，終究無法全然消滅。

張愛玲臆想出了這些虱子，後一直困擾於這些虱子的存在。這本質，就是她的矛盾點。她不願如此，卻無力改變生活的局面，想要徹底消滅這些煩惱，最終卻死於這場與苦難的搏鬥。

可悲可嘆的是，胡蘭成汲取了張愛玲對愛的能力，後全身而退，徒留張愛玲畫地為牢，將自己的內心鎖死。童年的悲慘和青年的不幸，令她一生沉浸於傷痛之中，不可自拔。她想要逃離這種宿命，卻無法解開自己的心結，只能看著這份心結越積越深，直至死亡。歲月倏忽,往事已矣，留在原地的只有重情重義卻一生被苦難包裹的張愛玲。

與虱子的殊死搏鬥，張愛玲究竟還是輸了。張愛玲的一生，浪漫和悲慘相互疊加。她是毋庸置疑的才女，但逃脫不了情深不壽，慧極必傷的結局。張愛玲之後，再無人能用最溫婉的語言描繪最尖銳的話語，她至情至性，卻無法求得一片真心。對於她的一生，嘆息之餘，皆為敬佩。
