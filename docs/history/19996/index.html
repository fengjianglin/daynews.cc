歷史上有一位皇帝，殺害了他所有的侄子，來看看他是誰？

2019-12-16 1,493條評論 4,181次閱讀 1134人點贊

說到十六國的歷史，北燕應該是重點。慕容氏家族統治北燕，直到慕容雲時代發生變化。慕容雲原來姓高。雖然他成為了皇帝，但他認為自己不是鮮卑人，感到很自卑。為了收買民心，讓大臣們同情他，他每天都開一個大庫房，獎勵文武官員。

其中，馮跋部長是慕榮雲的支持者，但慕榮雲對他持謹慎態度。為了防止自己被暗殺，慕榮雲身邊有著一群武士，與這些武士吃飯睡覺都在一起。沒想到，武士中有兩個叫離班和陶仁的人把慕容雲殺後，馮跋帶領衛兵在殿前斬首了離班和陶仁。而馮跋則被眾人推舉去接替慕容雲。從他開始，北燕改姓馮，開始了一段由馮家族統治的時期。馮跋登上王位後，還是沒有得到安寧。他的堂兄弟馮萬尼和馮乳陳叔叔和侄子，拒絕接受並想奪取王位。幸運的是，馮跋的二哥馮弘率領軍隊鎮壓了它。

看到馮弘的巨大貢獻，馮跋任命馮弘為將軍，並授予中山公爵稱號。從那以後，馮弘的力量一直在增長。在馮跋統治的22年里，他與人民和平共處。他為國家做了一些好事，使動蕩的時代顯露出難得的和平。

但在公元430年秋天，當馮跋病危時，王位之戰再次上演。馮跋最喜歡的宋王妃想要造反，以便讓她的兒子成為君主。然而，馮弘得知了這一事件。他帶領衛兵進入宮殿，控制了整個局勢，因為宋氏母子被禁。重病的馮跋驚慌失措，嚇得要命。

馮跋死後，馮弘成為天王之王，廢殺了馮翼王子。他不顧自己的舊情，為了避免以後的麻煩，把馮巴跋一百多個兒子都殺了。無論是青少年還是新生兒，馮弘都不會丟下一個。自古以來，就有不少君主沉迷於殺害前輩。然而，馮弘割了草，連地拔起了根。這的確是歷史上罕見的血腥皇帝。在殺死了馮跋所有的子孫之後，馮弘接手了馮跋的美麗的妃子。這使他在登上王位時失去了他的心。在叛亂的情況下，馮弘的兒子們為了不被父親殺死，逃到了北魏。當時北魏的年歷史的跋燾趁機出兵，一舉殲滅北燕。

戰爭開始後，馮弘一再敗退。為了保護自己，他不得不請求南宋文帝的幫助，並派了一位使者去要求稱號為弗朗西斯科。有可用的福利。當然，宋文迪願意幫忙。他將馮弘命名為「黃龍國主」，並許諾派遣軍隊幫助北燕。然而，由於南朝離北燕很遠，不能解渴，北魏很快攻破了北燕的城市，馮弘被流放到高句麗。

當馮弘從一個高級君主淪為寄人籬下的時候，他的生活並不輕鬆。後來馮弘轉向宋文迪，懇求宋文迪帶他走。得知這件事的高句麗王勃然大怒，立即下令殺害馮弘的家人。 因此馮弘要為這一命運負責。
