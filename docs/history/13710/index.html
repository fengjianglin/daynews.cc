為何清朝前中期200年沒有列強來

2019-12-10 1,107條評論 3,344次閱讀 1371人點贊

近代以來，尤其是1840年中英第一次鴉片戰爭打開了中國的國門，簽訂了大量喪權辱國的條約，讓中國進入了半封建半殖民地社會。但是明朝中後期就已經有列強來到中國，還多次發生戰爭，其中和葡萄牙大戰三次，和荷蘭大戰兩次，那麼為何清朝入關後到1840年近200年的時間裡，列強再未攻打中國呢？要知道那個時候中國正陷於內戰，應該更好攻擊才對。其實列強之所以在清朝初期200年沒有攻打中國，主要是被打怕了。

根據史書記載，明朝從嘉靖年間就開始和葡萄牙人交手，期間發生過三次海戰，可以看出葡萄牙對於中國也是不死心的，而我們知道明朝的佛郎機就是從葡萄牙購買的，可見在武器裝備上明朝是和葡萄牙差了一大截的，明朝在敗了兩次之後，依然沒有求和而是繼續戰鬥，最終利用中國人常用的水戰戰術火攻戰勝了葡萄牙人。

而在明朝內有外患快要滅亡的1633年，明朝和荷蘭發生了一場大戰，當時荷蘭派出了13艘艦船裝備有加農炮，而且率先摧毀了明朝的福建艦隊，明朝在第一次交鋒中戰敗，崇禎皇帝在內憂外患的情況下依然作出要將荷蘭人趕出去的決定，派出福建巡撫和鄭芝龍一起招兵買馬，最終明朝再次採取火攻的方式將荷蘭打敗，荷蘭艦隊近乎全軍覆沒，此後荷蘭向明朝繳納保護費，已確保在此地區能夠進行貿易。

當時的海上霸主西班牙駐菲律賓總督原本也準備攻打明朝的，但是在看到葡萄牙和荷蘭均敗下陣來，一直沒敢出兵。而之後1661年鄭成功驅逐荷蘭人，收復寶島台灣之後，西方列強都是規規矩矩的和中國做生意，近200年未和中國交戰，主要就是被打怕了。
