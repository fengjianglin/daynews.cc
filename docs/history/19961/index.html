李隆基：一出生被幽禁13年的小兒子，靠什麼在28歲搶來皇位？

2019-12-17 756條評論 2,166次閱讀 474人點贊

762年，長安城太極宮。

一位年近八旬的老人孤零零遊盪著，諾大的神龍殿連個說話的人都沒有。只有清寒皎潔的月光，在地上拖出一道死寂的身影。

他曾是大唐王朝的掌控者，如今卻被親兒子軟禁2年。

當年靠著一次次豪賭，他坐上原本輪不到自己的皇位。締造出輝煌無比的大唐榮耀後，卻又親手將它摧毀殆盡。

皇圖霸業終成空，不勝人間一場醉。

老人神情凄楚的望向夜空，那輪圓月中好像浮現出一張絕美的臉龐，正朝著他嬌笑：三郎，你是想我了嗎…

李隆基出生那年，老爹剛剛當上皇帝。

李旦之所以轉正，因為三哥李顯被廢。
李顯之所以被廢，因為不聽媽媽的話。

這哥倆擁有一位千年虎媽：武則天。眼看著老大老二非正常死亡、老三被趕去湖北，李旦上班頭一天就發通告：我是掛職幹部。

權力更迭對李隆基的吃奶任務沒啥影響，他只是個庶出的三兒子。就算李唐血脈死絕了，也威脅不到祖母的宏圖偉業。

但誰也沒料到，武則天真打算把李唐皇族往死里干。天下姓李的都忙著改戶口本，輪到姓武的吃香喝辣了。

史載酷吏先誅唐宗室貴戚數百人，次及大臣數百家，其刺史、郎將以下，不可勝數。

688年，李隆基剛滿三歲。

皇家存活的王爺們坐不住了，他們聯合反抗卻被武則天藉機一網打盡，連帶著將兩個親孫子都弄死了。

李世民揮汗如雨播下的種子，只剩李旦李顯這對難兄難弟。武則天能留下他們的原因：

1.終歸是親兒子。
2.哥倆性格懦弱。

都說兒女是母親身上掉的肉，但武則天真正做到了權力面前眾生平等。

或許夜深人靜時，她偶爾也會傷懷大兒子被搞到斷子絕孫。於是封李隆基為楚王，名義上過繼給李弘（制令楚王諱繼其後）。

李旦當然不敢有意見，而李隆基還不會提意見。

690年，28歲的李旦光榮退休。67歲的武則天正式上崗，改國號為周。

李隆基跟著父親搬進東宮，家裡的攝像頭比蚊子都多。武家的舅爺們經常過來找茬，想將老李家斬草除根。

李旦啥也不敢幹，整天和樂工們在家裡吹拉彈唱。日漸懵懂的李隆基，就是在曲藝排練廳里長大的。

後人仰慕唐玄宗音律雙絕，還將其供奉為「梨園鼻祖」，卻很少知道緣由竟是這般兇險。

人在幼稚，精神專一通利，蓋幼兒好學，如日出之光。無論天生所好還是環境所迫，自幼用心學到的知識，終究會綻放溢彩。

處於權利旋渦中心，血光之災只是時間問題。

692年，李隆基的王府完工大吉。7歲的小王爺正式出閣，每逢初一、十五都要去朝堂打卡簽到。

有一次，李隆基的儀仗隊遇上了武懿宗。武則天的這個大侄子，看見李家小王爺的氣勢排場很不爽。

武懿宗：瞎啊，你的馬車壓黃線了。
李隆基：小爺分多著呢，隨便扣。
武懿宗：靠，你敢這麼跟金吾大將軍說話？
李隆基：吾家朝堂，干汝何事？敢迫吾騎從！

武懿宗連忙跑去向姑媽彙報，武則天聽完呵呵一笑：我這孫子有志氣啊，讓他搬回宮住吧。

不久之後，李隆基的母親被祖母請去喝茶，卻再也沒有出來…

李隆基追問母親去哪兒了，李旦卻囑咐他千萬不敢聲張。

不光母親被殺，連帶舅舅全家都被流放。官方解釋說他們埋木頭人詛咒武皇帝，其實因為他們都是關隴豪族。

從南北朝末期開始，雄霸中國歷史200多年的關隴集團，到武則天時代徹底關門大吉。

年僅8歲的李隆基，再也不用為過年走親戚發愁了。因為祖母只給他留下兩位血親：姑媽太平公主和小姨媽竇氏。

緊接著，終於有人投訴李旦謀反了。

酷吏來俊臣很興奮，連夜在東宮院里擺滿各種刑具。李旦摟著兒女們瑟瑟發抖，他太了解母親的鐵腕了。

出乎意料的一幕發生了，樂工安金藏為證明李旦無罪，當眾拿刀拋開自己的胸膛，腸子都流了一地。

還沒正式審問就搞出人命，老來都不會按《羅織經手冊》的套路出牌了。

武則天知曉後解散了調查小組。一開場就上演如此慘烈的劇情，不光達到震懾李家黨羽的效果，還省下很多盒飯錢。

權力，真是很容易讓人泯滅情感的東西。母親本該是最疼愛兒子的人，卻還不如一個外人的信任和維護。

安金藏的血水被擦洗乾淨後，好像什麼都沒發生過一樣。但是李隆基猛然間長大了，還第一次感受到血脈之外的溫情。

他和父親更加小心翼翼的活著，只有絲竹之音響起時，才會短暫忘卻生命在權斗面前的卑微和脆弱。

這種忐忑不安的日子，一過又是5年。

698年，武則天放棄了立武家侄子當儲君的想法，因為幹活的大臣全票反對。

流放14年的李顯被接回來當太子，李旦全家搬出東宮騰床位。這哥倆被老娘坑了半輩子，還要被後人戲稱「六味帝皇丸」。

13歲的李隆基雖然被降為臨淄王，卻徹底恢復自由之身。因為大唐江山將要回歸李氏，他終於感受到皇子的尊貴。

自由真好，吃喝玩樂全然由心的舒暢。
權力真好，走哪都有恭敬熱情的笑臉。

李隆基在洛陽城四處撒歡，驚恐灰暗的童年讓他嚴重缺失家庭情愛，反倒更容易對外人產生信任。

最重要的是：權斗和文藝早已融進他的血液。這就像老話說的失之東隅收之桑榆，只是誰也不知道最終能發揮多少。

照目前情形來看，李隆基將來也就是個尊貴無憂的公子哥。

705年，81歲的武則天躺在病床上奄奄一息。

這位傳奇女皇臨終時，沒能看到兒孫繞床的溫暖，只有80歲的張柬之拿著遺囑逼她簽字（見秦嶺一白.張柬之篇）。

神龍政變後李顯登基稱帝，李旦封為安國相王，太平公主添加鎮國前綴。那些被武則天壓制40多年的人物，好像一夜間全站起來了。

就算女人是水做的，男人是泥做的。武則天也絕不是水泥做的，她是碳化鎢做的。

看著武則天被抬進上陽宮療養院，李隆基很震驚：三伯那麼挫的人居然能扳倒祖母，看來多認識些神隊友很重要。

但是，李顯更喜歡結交豬隊友。

唐中宗不光和武三思做了兒女親家，還任由媳婦折騰國事。

韋皇后被婆婆迫害這麼多年，卻依然是她的鐵杆腦殘粉，就連女兒安樂公主都挖空心思地想做皇太女。

太子李重俊很憋屈，親媽死的早就算了，親爹也不替他說句話。這個沒人管的孩子索性造反，想幹掉韋武兩家外戚。

景龍政變，最終以李重俊被殺收場。

唐中宗的暴脾氣上來了，他也開始整治起自家兄弟。很多大臣跑來勸架：你家裡都沒幾口人了，還容不得弟弟妹妹嗎？

於是，原本就存活不多的皇家子弟被趕去外地上班。23歲的李隆基前往山西長治，級別還沒縣長高（潞州別駕）。

從京城高幹變成底層幹部，李隆基的天上人間VIP失效了，卻換來一大批中底層有志難伸的奇人異士。

相對熱衷於搞自家人皇族，李隆基更喜歡真情朋友之間的義氣洒脫。而且無數歷史經驗證明：酒肉朋友永遠沒吊用。

張韋、李宜德這些牛人，都將成為他日後登上帝位的中堅力量。李隆基當然不可能預知未來，只是覺得他們個個挺有本事。

天將降大任於斯人也，必先苦其心志，勞其筋骨，餓其體膚，空乏其身，行拂亂其所為也，所以動心忍性，增益其所不能。

有時候，心理上的磨鍊往往比肉體更痛苦。李隆基沒有像父輩那般被生活磨報廢，接下來將會是無盡的釋放。

老天的對福禍的安排，凡人永遠都猜不透。

710年，李隆基被召回京城。

這位器宇軒昂的年輕皇子（儀範偉麗，有非常之表）告別城鄉結合部的廣闊天地，又在長安城裡結交各路神隊友。

秦嶺一白覺得這還算不上圖謀權力，李隆基的名份和實力都離皇位太遠。只是自幼缺失情感的經歷，讓他很享受朋友之間的真性情。

半年後，李顯很神秘的掛了。

韋皇后安排15歲的小兒子接班，並且光明正大的把持朝政。此時距離武則天去世僅僅5年，各方勢力紛紛心思涌動。

韋家很興奮，再加把勁升級當老闆。
李家很惶恐，再刷一遍就得銷戶了。
其他：啤酒飲料礦泉水，看戲馬扎要不要？

重要崗位都換成韋家心腹，韋皇后準備拿威脅最大的李旦開刀了。李隆基曾親眼目睹過兩次政變，沒想到這次會成為主角。

老爹是指望不上了，李隆基跑去找姑媽太平公主。

李隆基：咥他不？
太平：咥誰？
李隆基：你懂的！
太平：咱人手太少，咥不動啊！
李隆基：不咥那也是死！
太平：咥！帶著你表弟去咥他！

朋友們全都派上了用場，宮裡還有高力士做策應（見秦嶺一白.高力士篇）。造反方案出爐後，李隆基唯獨不給父親抄送。

我拯社稷之危，赴君父之急，事成福歸於宗社，不成身死於忠孝，安可先請，憂怖大王乎！若請而從，是王與危事；請而不從，則吾計失矣。

這是李隆基第一次豪賭，他看看身後興奮而忐忑的部下，又抬頭望向天上的明月。時間真是個神奇的東西，看似同樣的夜晚：

有人呼呼大睡
有人無家可歸
有人通宵達旦的玩樂
有人病痛呻吟的煎熬
…
而他，將要面對一場生死未卜的血戰。

那一夜，先是李隆基的心腹保鏢開溜，繼而鍾紹京又拒絕提供作戰指揮所…最後靠著一次次好運眷顧，唐隆政變才僥倖成功。

第二天，睡醒的李旦當了皇帝。

25歲的李隆基是政變第一功臣，老爹激動萬分地摟著他說：宗社禍難，由汝安定，神祇萬姓，賴汝之力也。

看著老爹在一厚摞文件上蓋個章，近百位有功之人立馬就能升職加薪。李隆基不禁扭頭瞧向旁邊的大哥，對方卻刻意迴避他的眼神。

是啊！李成器才是嫡長子，自己不過是個庶出的老三。李隆基豁然自傲的豪情消散了，連周圍人看他的眼神都覺得很複雜。

李旦、李成器、太平公主，很忌憚這位英明強勢的皇子。
劉幽求、葛福順、崔日用，卻眼巴巴的盼他再上個台階。
李隆基的心裡也凌亂了：失落？衝動？不甘？順其自然？

一腳踏入權力旋渦中心，從此便再無退路。雖然已經離皇位很近了，但是李隆基現在一點發言權都沒有。

唐睿宗開會商議立太子，大兒子李成器當然是第一繼承人。

估計是厭惡朝局爭鬥，再加上太爺爺李世民的經典案例。李成器死活都不願當太子，還當眾推辭：國安則先嫡長，國危則先有功…

那好，皇太子就是李隆基！

李隆基沒想到這麼容易就做了太子，更想不到後來當皇帝比這還省事。老天好像要把多年來的虧欠，一股腦全還給他。

李隆基最開心，他是祖國唯一的接班人。
朋友們很開心，總算跟對了一支潛力股。
大臣比較開心，未來會跟隨一個好老闆。
李旦勉強開心，精明強幹的一個分權人。
太平公主不爽，我特么該咋辦？

可能是繼承母親的基因過多，太平公主的權力慾望比李家男兒們都旺盛。

唐睿宗夾在這姑侄倆中間努力和稀泥，每當宰相來做工作彙報，他都要問：嘗與太平議否？嘗與三郎議否？

結果很明顯，這兩人瞧對方越來越不順眼了。

太平公主明顯占著上風，她從武后、韋後時期積攢的朝中勢力，是散戶大王李隆基根本難以匹敵的。

然而在權斗陰影里長大的李隆基，也擁有太平公主永遠不懂的生存智慧，那就是：低調做人裝孫子。

太平公主大肆叫囂著換太子，反倒是姚崇、宋璟等老臣紛紛站出來替李隆基說話，最後再由唐睿宗出面和稀泥。

《道德經》云：多言數窮，不如守中。太平公主就是輸在嘴太碎、話太多。

712年，天上出現彗星。

太平公主為扳倒李隆基，上趟洗手間的功夫都能想出三個新點子。這次百年一遇的奇特天象，必須要好好利用做文章。

她花十塊錢買通一堆專家，讓他們組團忽悠唐睿宗：據分析帝座及前星有災，皇太子合作天子，不合更居東宮矣。

誰知道唐睿宗一聽有災就炸毛了，連夜派人將李隆基喊過來：傳德避災，吾意決矣。

太平公主傻眼了，她原以為老哥聽說要退位會勃然大怒，按計劃還能搞死李隆基。沒想到人家直接撂挑子不幹了。

龍炕啊，說不睡就不睡了？

爾視之若珍寶，我棄之如敝履。崇尚道家的李旦早就煩透了，既不想幹掉兒子又不想幹掉妹妹，那就幹掉自己好了。

只要保證大唐能姓李，秦嶺一白每天能照常供應土蜂蜜，皇位交給三郎就放心了。

712年，李隆基正式稱帝，氣的太平公主把專家都剁了。

李隆基雖然轉正了，但實力還和姑姑差老大一截。他只有三品官銜以下的任免權，而朝中五個宰相都是太平公主的人。

李旦退休後專業和稀泥，他實在不願見到親人互殺再次上演。但是對於國之重器，從來沒有和諧共享的方法。

第二年，先天政變爆發，以太平公主自盡收場。

舊唐書上說是太平公主先造反，由於消息泄露被李隆基將計就計給滅了。但是，也有人說這是李隆基設的局。

不管真相緣由如何，最終結果都是註定的：不是你死就是我亡。因為權力場上，從來就沒有乾乾淨淨的人。

那一年，李隆基才28歲，距他回到京城才短短3年。

公元713年，李隆基改國號為開元，取自「開元布澤，含和尚仁」之意。

大唐即將迎來「九天閶闔開宮殿，萬國衣冠拜冕旒」的輝煌時代，這也是華夏文明史上最璀璨浪漫的時代。

但是誰也沒有想到，親手締造出開元盛世的大唐玄宗，卻要眼睜睜地看著她分崩離析，最後徹底淪落到黑暗血火之中。

成於斯敗於斯，這一切都和李隆基的性格息息相關…
