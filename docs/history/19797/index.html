名將周亞夫之死：得罪人事小，自己作死才活該？恐怕皇帝才是真兇

2019-12-18 1,260條評論 2,644次閱讀 993人點贊

狡兔死，走狗烹的戲碼，在歷朝歷代上演的不計其數，而周亞夫就是其中被殺的一個。但在漢文帝時期赫赫有名的將軍，為何到了景帝時期才有了被殺的苗頭，得罪梁王不假，自己因一雙筷子作死也不假，但說到底，恐怕都是皇帝想要除之而後快的借口，才是最大的真兇。

周亞夫並不像美人心計裡面演的，他是承襲了名將周勃的次子，也是鎮守一方赫赫有名的大將軍，尤其是在平定七王之亂這場爭端，簡直就是上演了「一夫可當萬夫之勇」的好戲，拯救了漢室江山，這之後他的仕途也升至了人生的巔峰，可隨之而來的就是一場關於皇室和權臣的陰謀，正是這場叛亂改變他的人生走向。

當時為了平定這場叛亂，於是就制定了一條計策，目的其實就是「楚兵剽輕，難與爭鋒。願以梁委之，絕其糧道，乃可制」。因為恰逢叛軍正在圍攻梁王，梁王連著上奏了很多請求增援的奏疏，但周將軍就是請求皇帝不要救援，其目的就是想讓梁王將叛軍的正面火力吸引過去，自己以便從背後攻擊，斷了他們的糧草，這樣來一個前後夾擊，就可以直接擊破。

周將軍並沒有想要害梁王，也沒有什麼對他不滿的心理，但是此舉卻遭到梁王的不滿，他覺得這樣是陷於了危險的境地根本沒有考慮到自己的安全問題，於是就經常和竇太后告他的黑狀，而且專門挑撥他們的關係。在竇太后的心裡當然是他這個小兒子重要了，還曾經一度想要讓皇帝立他為儲君，兩個人加起來在景帝面前挑撥，怎麼可能不動搖。

但是旁人的挑唆只起一個推波助瀾的功效，還不足以致命，最致命的是皇帝察覺到他對皇權有威脅。周亞夫將軍一直給人以一種不好親近，堪比黑臉王，總是表現出一副誰的賬都不買的模樣，對待皇帝都是如此。曾經文帝到細柳營去閱兵，但是進營之前，就發生了很不愉快的事情，眾將士只識將軍，不認皇帝，這要是換做是朱元璋的話，早將這個不識抬舉的將軍送上砍頭台了。

可是漢文帝寬宏大量的原諒他，而且皇帝來到竟然也沒有出門迎接，這是對皇帝的不敬，往好了說是軍紀嚴明、一絲不苟，但是往壞了說，就是擁兵自用、藐視皇權。細柳營一事漢文帝可以忍得過去，可之後景帝的暗中窺探是他容忍不了的。剛剛經歷了七王之亂之後，漢景帝對於這些擁兵自重的人確實非常忌憚，雖然周亞夫的忠誠無話可說，作戰能力也是一級棒，但是沒有哪個皇帝就這樣心大的放任不管，萬一什麼時候出現權臣叛亂，這可是一場非常大的損失。

於是考驗他就就在一場飯局之上，故意在他的碗里放了一大塊肉，但是並沒有筷子，整盤菜就這麼一整塊沒有切碎的肉，而且沒有刀，他的脾氣確實耿直，還讓左右服侍之人給他拿筷子，但卻經過這一舉動之後，他確實莽撞了，皇帝此舉就是為試探他，看看他的脾氣秉性，自始至終沒有工具的情況下就是不動那塊肉，因為本著士可殺不可辱的性子，不能被皇帝這麼侮辱，可景帝這場飯局看到了周亞夫是個難以馴服之人，不得為他所用，而周將軍自己也是對此頗感到侮辱。

這場飯局算是給他最後在景帝心裡判了死刑，一代名將就這樣侮辱了人格，而且還丟了性命，劉啟這種試探真的有些過分，對於這麼一個有功之人，卻用這樣一種剛直之人接受不了的方式羞辱，最後還找了個連三歲小孩不相信的理由，將他下獄，絕食5天而亡。司馬遷評價他的話為「亞夫之用兵，持威重，執堅刃，穰苴曷有加焉！足己而不學，守節不遜，終以窮困。悲夫！」

他在用兵打仗上，對於漢室江山做出了不朽的功績，但是最後在名節這一塊被皇帝這麼陷害栽了跟頭，確實可悲。

雖然他在性格上是不適合走丞相路線，領兵打仗才是他的擅長，但是這並不是他這樣慘死的理由，劉啟有心害之，卻偏偏要抓住他的最在意、最耿直的點肆意攻擊，擺明了就是不給他留活路，最後的死也算是自己給自己掙了一個相對體面的死法，但這場兔死狗烹的戲碼，卻實在是不好看，而且劉啟的吃像也未免有些太難看，可憐一代名將就此隕落。
