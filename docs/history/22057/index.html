崇禎在位時，明朝雖問題很多，但與大多封建朝代相比是好很多了

2019-12-18 414條評論 4,871次閱讀 3568人點贊

明末年間，大局已經難以收拾，但是崇幀又作死加速了滅亡。明朝亡於吏治腐敗，稅收收不上來，邊疆的通敵資敵，部隊腐敗沒有戰鬥力，官員腐敗導致土地兼并嚴重，農民起義，軍屯制度名存實亡，明朝最優越的制度應該是內閣，然而崇幀的獨斷專行和刻薄寡恩破壞了內閣制度，而他個人又無力對抗東林黨，起義軍打到都城要遷都沒人理他。相比之下，他的前幾任都懂得啟用宦官對抗黨爭，比他高明多了。

其實，他接管權力是沒有什麼問題的，魏忠賢一上手就命令崔呈秀放棄了兵部。換言之人家就沒準備抵抗，把肚皮露出來給他看，結果他一刀就捅過去了，就是個憨貨。明末的風氣不一樣，崇禎一沒受良好帝王教育，二沒從底層摸爬滾打的經驗，加上明末風氣特別爛，根本就沒有你說的那麼容易，為什麼南宋諸帝看起來都很弱？

北宋徽宋朝看起來很弱，東漢中後期，清末看起來很弱？風氣不一樣，不光是帝王不行，群臣也不行，天啟朝，崇禎朝，一直到南明，群臣的素質是一個比一個低，明中期皇帝被俘還有大臣力挽狂瀾，明末天啟朝及之後大臣的表現遠比不上之前，崇禎中期用的大臣並不差，楊嗣昌，盧象升，孫傳庭，洪承疇都是人才，差不多也是最頂尖的人才，但這些人才的實力也不夠同時應對農民起義和滿清，對上農民軍還可以打有來有回，對上滿清沒哪個打得了，其它人更不中用。

崇禎打小就沒想過能當皇帝，好比一家企業，本來是個主管，突然天上掉餡餅，一下成董事長了，他壓根就不會幹，更何況崇禎疑心病很強，又碰上東林黨清談誤國。還有毛文龍沒罪，即使崇禎擦屁股也是模稜兩可的罪名「通敵有跡」，犯罪嫌疑人不等於罪犯，所以罪名沒有平反正常，崇禎長編里大臣的反應看也是說毛文龍無罪就戮，事後東江鎮部下為老領導請求恤典崇禎給的理由是：牽制無功和糜餉，都不是死罪。

另外，歷代亡國都是缺錢，但是大多是因為人事上失去控制造成稅收不上來，崇禎面臨的是歷史上前所未有的官商一體，他想不明白為啥遼餉三百萬就天下大亂了，實際上官吏從民間搜颳了五千萬。表面上人事權還在手，他能一茬茬的換內閣，然而你能換了下面幾十萬官吏嗎？滿清入關以後，高舉屠刀，所以順治初年僅僅佔據北方數省滿清的財政就遠遠勝過崇禎時期了。和崇禎最像的實際上是小蔣去上海整頓金融時期的蔣介石面臨的局面，只不過蔣介石認識的更清楚一些，最後，是誰收拾了這樣的局面的？

另起爐灶絕對是成本最低廉的方案。而且哪個朝代末期都是非常缺錢的，明朝這種也算正常的了，在當時雖然有天災，但是還是人禍很厲害一些，錢和糧食都進入了權貴地主階級人手上了，貪污腐敗嚴重的一批，官員直接直接相互庇護，自成一體，官商勾結，而崇禎沒有看清這些事情，本來還可以堅持很久的，因為他的各種操作直接被滅國了，只能說活該。

明亡的直接原因是政府破產，破產的原因說是文官誤國也沒錯，但文官誤國的鍋崇禎是要背的，他真的不如他哥。魏忠賢雖然被文官罵得慘，但起碼知道不從吃不上飯的農民身上增稅，缺錢了打得是富戶的主意，只是手段奇葩了點，所以被罵得很慘。崇禎要是有他爺爺一般撈錢的本事，明是亡不了的，只要能和滿清對峙幾十年，滿清就和金一樣了。
