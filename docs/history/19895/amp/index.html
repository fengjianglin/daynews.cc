據說宋朝的士兵人人一米八，為什麼之後人均身高變矮了？

天天要聞
23 小時 ago

古人的身高一直是現代人探討的問題，一方面大家對於古籍裡面的記載「七尺男兒」「身長八尺」有著尺度上面的想像和推測，而另一方面也是好奇為什麼古人的生活水平沒有現代人高但是身高為什麼就能這麼高？現代人比起古人生活水平更是日漸發達，按歷史進程的邏輯來講應該是現代人比古代人身高要高才對。

但是在宋朝當兵的他們普遍的身高都比較高，宋朝的徵兵制度也是在身高方面做了比較大的限制，要想成為宋朝的士兵首先會被要求應徵兵者要在五尺八寸到五尺五寸。將這個標準換算成現代的計算方式也就是宋朝士兵基本在一米八以上，而且除了身高達標體型也要達標才可以。不過我們在宋朝的靖康之難就發現曾經有矮個子的金兵居然戰勝了身高強壯的宋兵的案例。不免讓人懷疑，宋朝的士兵真的有那麼高嗎，為什麼連一個矮個子都打不過啊？相信不少人都有這樣的疑問：會不會有虛報身高的嫌疑？

表面看是有一定依據的：曾經在朝天宮舉辦的「宋明服飾」展裡面就有幾件來自宋朝的服飾，這些衣服普遍都比較寬大，比一個大男人還要大。除了衣服以外，我們通過《宋史》裡面得知天武、捧日、神衛、龍衛四軍士兵的身高比例和一個月的糧餉（也就是工資）呈正比例關係，也就四如果你的身高越高每個月拿到的錢就越多。

其實不僅是宋朝的士兵身高高，秦始皇的兵馬俑出土的兵俑身高也都是在一米八米左右。大多數都是依照真人大小製造，所以由此估計其實秦朝時期的人均身高也是有一米八的大個子的。

那為什麼宋朝以後人均身高就變矮了呢？其實主要是兩個方面的原因：第一個原因就是戰亂，時逢亂世這百年之間發生了太多的衝突。一直到五代十國戰爭都從來沒有停過，不斷打仗兵力不足朝廷在民間徵兵無數，能逃過徵兵的都是老弱病殘。根據人類遺傳學說推斷父母的身高影響著下一代的身高發育，這條件好的都被拉去當兵了，剩下的都是矮的。他們的下一代，自然都是矮的。還有一點就是宋代時期，大家對於女性腳部審美觀發生了變化，大家都以「三寸金蓮」為美。女人被迫裹腳，這直接影響了女性的下肢骨骼發育，腿部發育就會退化變短，同時也影響了後代的發育。
