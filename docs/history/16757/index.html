沒有蕭規曹隨，就沒有文景之治，也就沒有漢武盛世

2019-12-13 1,978條評論 3,945次閱讀 1084人點贊

提起西漢的建立，不得不提到蕭何，這位西漢初期最為有名的政治家，為西漢的建立立下了汗馬功勞。可以這樣說，若是沒有蕭何，或許秦朝末年的歷史將會是另一個版本。蕭何與劉邦是老鄉，他原是沛縣的縣吏，後來跟隨劉邦打天下，成了西漢的開國功臣，之後更是擔任宰相一職多年。

西漢的開國功臣當中，出身於沛縣的人不在少數，在蕭何去世之後，繼任宰相位的曹參也是沛縣人。只不過他們的經歷雖說相似，但是名氣卻差很多，很多對歷史不熟悉的人都聽說過蕭何的典故，但是很少有人提起曹參。不過史書上卻對兩位宰相在位時期的功績給出了極高的評價，西漢有「文景之治」的繁華，也有「蕭規曹隨」的美談。

漢高祖去世之後，呂后的長子繼位，史稱漢惠帝。漢惠帝性格仁弱，漢高祖在去世之前讓蕭何繼續為相，以輔佐漢惠帝。可惜的是當時的蕭何也並非壯年，在漢惠帝繼位的第二年，蕭何就生了一場重病。從此之後他精力大不如前，身體也十分的虛弱，對丞相一職也已經無能為力了。

之後漢惠帝親自去蕭何府上探望病情，並向他請教朝中哪位官員，在他之後可以擔任宰相一職。只是蕭何並沒有給他一個意見，推脫說一切以漢惠帝的決定為主。無奈之下，漢惠帝只好自己思考哪位官員合適，劉邦在去世之前曾對呂后和漢惠帝提起過曹參。於是漢惠帝就問蕭何，曹參這個人接任宰相合適嗎？

漢惠帝是個實誠人，說話也是直截了當，並不懂得迂迴，若是了解此時蕭何曹參二人關係的人，估計都不會這樣直接詢問。他們二人雖說同時跟隨高祖起兵，最初的地位是相等的，關係也很好，但是後來曹參立功頗多，不過在名氣上始終比不上蕭何。兩個人經常被拿來對比，曹參的付出與收穫不相等，名聲又時常被蕭何壓制著，心裡就發生了變化，二人的關係也就不復從前了，說不上交惡，但是也不親密。

好在蕭何在國家大事之上還是比較有分寸的，即使他與曹參關係尷尬，在漢惠帝詢問的時候，他也沒有因為私心而惡意打壓。反而對漢惠帝說，如果是曹參接任為相，那麼自己就沒有什麼遺憾的了。

只是這個時候曹參並不在都城，而是在劉邦長子齊王所在的封地當宰相。曹參在當齊國宰相之前一直是一位武將，從武職轉為文職，曹參其實並沒有把握做好。於是到了齊國之後，他很虛心地請教了當地有名望之人，共同商討該如何治理齊國。

隱士蓋公向他提議，為官者應該以讓百姓安定生活為本，所以身為宰相曹參最好信奉黃老學說，採用清靜無為的治理方式。曹參聽從了蓋公的建議，在他任齊國宰相的這些年裡，齊國一直很安定。

蕭何的這場大病最終沒有熬過去，在他去世之後，漢惠帝就把曹參從齊國召回了都城。從藩國來到都城，曹參也並沒有改變自己的為官之道，他認為治理大國和治理藩國是一樣的，還是按照黃老學說的思想來處理政務。國家政務都有蕭何設定的一些章程，曹參在就職之後並沒有做出任何改變，所有政務都是遵循蕭何所制定的制度來施行。

俗話說新官上任三把火，但是曹參一直都是無所作為，漢惠帝難免對曹參的表現失望。朝中激進的官員，頭腦里有些新奇的想法，想要向曹參請教，曹參也顧左右而言它，從不正面回應。通過一段時間的觀察，漢惠帝忍不住對曹參的行為產生了懷疑，曹參的所作所為是不是欺自己年少？不然一國宰相何至於此。

恰好曹參的兒子在宮中當值，於是漢惠帝就托他回家問問自己父親到底是什麼意思。但是讓人想不到的是，當兒子質問曹參為何無所作為的時候，曹參竟讓僕人打了他一頓。漢惠帝知道此事之後，更加的不高興了，對曹參的行為也更加的迷惑，於是在朝堂上忍不住問起了此事。

曹參向漢惠帝請罪說，陛下之於漢高祖孰強孰弱，自己之於蕭何誰聰明誰平庸？漢惠帝說他和曹參二人皆比不過漢高祖和蕭何。曹參說，既然我們比不過前人，那麼前人制定的規則都是有一定道理的，我們既然沒有能力做到更好，又為何要改變這一現狀呢？漢惠帝一想確實如此，於是就理解了曹參的做法。

後來歷史也證明了曹參做法的正確性，曹參任職宰相期間，漢朝百姓生活得很安定，後人也很稱讚二位宰相對百姓的付出。
