朱元璋：朕雖然殺功臣，但是朕沒殺沈萬三

2019-12-15 489條評論 2,243次閱讀 561人點贊

眾位愛卿和大明的子民們平身，朕有話要說。

朕本是淮右布衣，當年還是個放牛娃，徐達、湯和這些人都是朕的發小，朕真懷念和他們在鳳陽一起玩耍的日子啊，那時沒有這多勾心鬥角，世間險惡啊。

當年咱老朱窮啊，只讀了幾年私塾，肚中也有些墨水。可是好景不長啊，老家發生了旱災鬧了瘟疫，朝廷不救災，官員腐敗還要壓榨百姓。爹娘和大哥都去世了，只有二哥和我活了下來。

當時沒錢也沒有地，只好將父母草草安葬在鄰居劉繼祖給的一塊墳地上。為了活命，朕和二哥、大嫂和侄兒被迫分開，各自逃生。

朱元璋放牛

朕當時讀過幾年私塾，識得幾個字，便投奔了皇覺寺的高彬和尚，剃度為僧做了小行童。可是災荒不斷，寺廟裡也揭不開鍋了。當時寺里的多數僧人都要外出化緣，當然貧僧也是其中一個。

貧僧從濠州向南到了合肥，然後折向西進入河南，到了固始、信陽，又往北走到汝州、陳州等地，東經鹿邑、亳州，又回到皇覺寺。貧僧雲遊四海整整三年啊。

朝廷腐敗不堪，百姓處於水深火熱中。不久，貧僧聽說紅巾軍起義了。至正十一年（1351年）五月，韓山童、劉福通在潁州揭竿而起。接著，徐壽輝起於蘄州，李二、彭大、趙均用起於徐州。幾個月之間，各地紛紛響應。

當時朕的發小湯和就在郭子興的義軍中，他邀請朕參加義軍，朕當時還有些猶豫。不料這個事被寺里的和尚發現了，要去告發朕。朕走投無路之下，投奔了郭子興紅巾軍。

朕在紅巾軍中作戰勇敢，粗通文墨，很快得到岳父郭子興的賞識。他將義女馬秀英嫁給了朕，還給朕改了名字叫朱元璋。自此朕算是龍入大海，虎歸山林，英雄有了用武之地。

這天下大亂真是個英雄得勢之時，風雲際會之日。朕得了文臣李善長朱升等輩，武將得了徐達常遇春等輩。這朱升出了「高築牆，廣積糧，緩稱王」的策略。朕迅速擴大了實力，和陳友諒張士誠都算是割據一方的土皇帝了，就差個皇帝的稱號而已。

朕夾在陳友諒和張士誠中間，他們都有吞併朕的意思。朕得了劉伯溫，這劉伯溫當初朕請他，他看不上朕，後來他又主動出山輔佐朕，真是個奇人。劉伯溫給朕出了主意，提出朕要避免兩線作戰、各個擊破建策，被採納。

朱元璋

他認為如今最危險的敵人莫過於陳友諒，必須集中力量消滅他。雖然陳友諒勢力強大，但他殺君自立，部眾離心，人民疲敝，故而不難戰勝，只要等他們深入，再以伏兵擊之，不難取勝。

朕採納了他的意見，果然在鄱陽湖水戰中，朕採用火攻，大破陳友諒，陳友諒被亂箭射死。朕一鼓作氣攻下了陳友諒的所有地盤。

緊接著朕又大敗了張士誠，一舉攻下通州、興化、鹽城、泰州、高郵、淮安、徐州、宿州、安豐諸州縣，將張士誠的勢力趕出江北地區。後來攻入平江城，活捉張士誠。張士誠也算是個豪傑，朕不想殺他，結果他竟然不識抬舉，朕只好殺了他。

北面的元朝也是朕的威脅，朕發動了北伐，以徐達和常遇春為大將，發布《諭中原檄》，文告中提出「驅逐胡虜，恢復中華，立綱陳紀，救濟斯民」的口號。朕剿滅了元朝，將他趕回了老家。

洪武元年（1368年），朕於南京稱帝，國號大明，年號洪武。大封諸將為公侯，韓國公李善長、魏國公徐達、鄭國公常遇春、曹國公李文忠、宋國公馮勝、衛國公鄧愈、劉伯溫誠意伯。

朕從一淮右布衣當了皇帝，這是朕從來都不敢想的。朕當了皇帝天下至尊啊，可是朕高興不起來了。朕夜夜失眠，睡不好覺，吃不好飯。

為啥啊，朕也怕失去朕的皇位和江山啊，這是朕浴血拚殺，將腦袋捆在褲腰帶上得來的。可是「打天下難，坐天下更難」，朕怕啊，朕本來是個布衣，卻做了天子，天下有多少個布衣。如果他們都起來要做天子，那樣天下就會大亂，朕的皇位保不住啊。

朕覺得這個丞相這個官職權利太大，一人之下，萬人之上，總領百官，輔佐皇帝治理國政，無所不統。相權太大有點威脅皇權，朱元璋心裡不爽啊。

《史記·陳平世家》「宰相者，上佐天子理陰陽，順四時，下育萬物之宜，外鎮撫四夷諸侯，內親附百姓，使卿大夫各得任其職焉。」

朕決定要廢除丞相這個職務，然後廢了中書省。朕想了想，就先罷了開國元勛李善長的丞相官職，然後用無才無德的胡惟庸為丞相，使了一招欲擒故縱的帝王權謀。

朱元璋

那胡惟庸自以為當了丞相，可以為所欲為，結果洪武十二年，被朕所誅殺，廢了丞相的官職。朕還下令，後世子孫官員不得設立丞相，官員要是說這件事，就凌遲處死。

自古三公論道，六卿分職，並不曾設立丞相。自秦始置丞相，不旋踵而亡。漢、唐、宋因之，雖有賢相，然其間所用者多有小人，專權亂政。今我朝罷丞相，設五府、六部、都察院、通政司、大理寺等衙門，分理天下庶務，彼此頡頏，不敢相壓，事皆朝廷總之，所以穩當。以後子孫做皇帝時，並不許立丞相。臣下敢有奏請設立者，文武群臣即時劾奏，將犯人淩遲，全家處死。

朕覺得這樣做還不夠，將大都督府改成了五軍都督府，使兵部有調兵之權，而無統兵之權：將領有統兵之權而無調兵之權。將御史台還做了都察院。

這樣一來，朕將權力都集中到了朕的手裡，誰也奪不走了。哈哈，朕有辦法吧。本來朕是不想殺功臣的，朕的兒子朱標是可以鎮住這些淮右從龍的功臣的。

可是朱標去世了，皇太孫朱允炆太懦弱了，而且勢單力薄。朕通過胡惟庸案和藍玉案殺了不少功臣，李善長、唐勝宗、陸仲亨、鄭遇春、費聚、趙庸、黃彬、陸聚、馮勝、周德興、傅友德等等功臣都被朕所殺，牽連了十幾萬人。

功臣藍玉

你們也不要說朕無情，政治本來就不講情面。李善長是朕的老部下，從滁州跟著朕一起打天下，忠心耿耿。可是他是朝中淮右集團的頭。他雖然是朕的兒女親家，但是朕還是要殺他。朕做的對，朕這樣做了才有利於大明的江山穩固，這樣老朱家的皇位才能萬世一系。

朕從布衣稱帝，知道民間疾苦，黎民百姓之苦難皆來自於官員不法，貪污腐敗，只有爭取民心才能江山長久，詔令天下：「為惜民命，犯官吏貪贓滿六十兩者，一律處死，決不寬貸。」

由於朕反腐敗決心之大、力度之強、措施之精確，收到了強烈震懾作用。朕當政31年，殺掉貪官污吏15萬人。但貪官還在，貪污腐敗根除不覺。朕無奈啊，朕的做法如此嚴厲乃至於殘酷。朕只能發出「朝治而暮犯，暮治而晨亦如之；屍未移而人為繼踵，治癒重而犯愈多。」這樣的感嘆了。

至於說朕殺了沈萬三這是子虛烏有的，朕定都南京，準備修建南京城牆。可是天下初定，國庫空虛，朕就讓富商出錢效力。

這沈萬三也在其中，沈萬三自己出錢修築了三分之一的城牆，修建的十分氣派，工程質量也好，比官方提前三天就完工了。

沈萬三的做法朕很不高興，這樣做朝廷的臉面何在，體統何在？一個富商竟然不給朕面子，這是在打朕的臉。

沈萬三

這沈萬三覺得做的還不夠好，想自己出錢，替皇帝犒勞三軍。這一下就惹怒了朕，朕非常不高興：「一個賤民奸商犒勞三軍，是個犯上作亂的草民，應該殺了他。」

還是朕的髮妻通情達理，馬皇后覺得這樣做有點過分，就勸朕：「臣妾聽說國家的法律，是用來誅殺懲戒違法亂紀的。並不是用來誅殺不祥之人的。老百姓富可敵國，是百姓自己不祥，不祥的老百姓，老天爺將會懲罰他，陛下何苦去殺他呢。」

朕也不是沒有氣量的人，跟一個富商較勁也有損皇家的臉面。於是朕將沈萬三從寬發落，死罪可免，活罪難逃，沈萬三被貶到了雲南，最後得到了善終。

吳興富民沈秀者，助築都城三之一，又請犒軍。帝怒曰：「匹夫犒天子軍，亂民也，宜誅。」後諫曰：「妾聞法者，誅不法也，非以誅不祥。民富敵國，民自不祥。不祥之民，天將災之，陛下何誅焉!」乃釋秀，戍雲南。見《明史·卷一百十三·后妃傳》。

至於那些說朕殘暴的人，說朕殺了沈萬三的人。那些都是什麼人？那些都是清朝人編排朕的。清朝人為了詆毀朕，將朕畫成鞋拔子臉，滿臉黑子那種樣子，不就是為了摧毀大明在民間的影響。

當然朕相信諸位愛卿的眼睛都是雪亮的，都能分清事情黑白，不信謠不傳謠。

好了，今天朕就說這麼多了。
