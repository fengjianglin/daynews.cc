奇葩祖孫三代，把剛崛起的國家搞滅，卻成了活得最久的諸侯國

2019-12-15 58條評論 2,152次閱讀 122人點贊

這是史說星語的第257篇原創文章

話說，中國古代不乏奇葩國君，今天筆者就來跟你聊聊其中的三位。他們生活在中國的春秋時期，並且是祖孫三代，分別為衛宣公、衛惠公和衛懿公。衛宣公與父親的小妾私通，還搶了兒子的新娘子；衛惠公把母親嫁給了堂兄，還曾逃亡8年；衛懿公跟鶴玩過家家，結果玩到君死國滅。

下面分別來聊聊他們三位：先說衛宣公

他是衛庄公的兒子，卻跟老爹的小妾夷姜有不倫的關係，並且還秘密生下了三個兒子，太子伋、公子黔牟、衛昭伯。等到他繼位之後，便把長子伋立為了太子。等到太子伋長大成人之後，便借著為兒子迎娶齊女與齊國聯盟，由此引發了衛宣公的第二件荒唐事。當他聽說齊女宣姜貌美如花的時候，便動了歪心眼，於是他便將太子伋支到了宋國出差，自己居然跑到迎親途中建了宮殿搶婚。直到宣姜入了洞房才發現，英俊的小鮮肉沒來，進來的卻是個糟老頭子。由此，宣姜便成了未婚夫太子伋的繼母。

而關於宣姜的風流韻事，筆者之前也聊過，她嫁給衛宣公三年之後還生下了兩個兒子，一個叫公子壽，另一個叫公子朔。而公子朔就是今天筆者要聊的第二位主人公衛惠公。

公子朔與哥哥公子壽不同，他從小就表現出強烈的權力欲，但他上有親哥哥，還有太子伋。為了除掉太子，他便和母親宣姜聯起手來在衛宣公耳邊說太子伋的壞話。時間久了，衛宣公也就逐漸對太子伋失去了信任，直到有一天派太子伋到齊國出差，衛宣公便橫下心來決定除掉太子伋，於是他派出一伙人假扮成盜賊，埋伏在途中擊殺舉外交旗幟的人。

不料此事被公子壽知道了，他力勸太子伋逃命未果的情況下，於是灌醉了太子伋，自己帶著外交旗幟趕赴齊國，其實也就是一命抵一命。果然，早已埋伏好的「盜賊」看見衛國的外交旗幟，二話不說就殺掉了公子壽，當太子伋趕到現場的時候，已經為時已晚了。悲痛之餘，他居然請求盜賊一併將他殺掉。

就此，衛宣公接連失去兩個兒子，悲痛萬分，但也無可奈何，只好將「罪魁禍首」公子朔立為了太子。並且在其死後，公子朔繼承君位，稱魯惠公。但他的君位來得可不光彩，剛上任4年就遭到了左右公子的暴力反抗，一舉將衛惠公趕出了衛國，並且立太子伋的親弟弟公子黔牟為君。由此衛惠公逃往齊國，在舅舅齊襄公家裡一躲就躲了8年。

終於在齊襄公的幫助下，衛惠公得以回到衛國複位。但為了平息眾怒，他居然與舅舅商量出了一個極其陰損的辦法，那就是將母親宣姜再次改嫁，嫁給了太子伋和衛君黔牟的親弟弟衛昭伯。

經過衛宣公和衛惠公這麼一鬧，可算徹底將春秋初期衛武公打下的江山快敗乾淨了。想當年，衛武公出兵勤王，輔助周平王東遷洛邑，那是多麼威風，當時也算是諸侯之主，要不是鄭武公搶先崛起成為春秋小霸，衛國前途不可限量。但剛崛起的衛國就這麼一步步衰落，直到君死國滅。不過任誰也沒想到，會滅得這麼快。因為衛惠公剛去世八年之後，衛國就毀在了他兒子衛懿公的手上，這便是今天筆者要聊的第三位主人公。

衛懿公從小養尊處優，難以理解百姓疾苦，而他又有個特殊的嗜好——養鶴。養鶴倒沒什麼，可他是一國之君啊，他那這個興趣愛好就不會那麼簡單了。全國上下為了他到處抓捕仙鶴送到宮中供他玩樂。

鶴多了之後，衛懿公便開始跟它們玩起了過家家的遊戲。他根據這些鶴的品質、羽毛、體態、身姿等的不同，劃分了優劣等級。給它們安上了官銜，這還不算完，他還按照官銜派發相應的薪水，也就是發放俸祿。

這一下可就玩大了，不光百姓苦不堪言，還驚動了一直對衛國虎視眈眈的赤狄。赤狄可不是中原講禮數的諸侯，他聽說衛懿公的腐敗行為之後，立馬率領數萬鐵騎直奔衛國而來。而衛懿公倉促應對，緊急向國內徵兵卻被打了臉，因為百姓對其怨恨，根本無人應徵入伍。無奈之下，衛懿公強制徵兵，並且親自挂帥，前去迎戰狄兵。

畢竟是一個養尊處優的君王，況且這些兵根本無心應戰，敗亡就成了必然，而衛懿公也死在了戰場上。狄人一路長驅直入，趁勢殺入城中，滅了衛國。至此衛懿公因為玩鶴導致了君死國滅的下場。

但巧的是，那時候齊桓公正在稱霸，有了齊桓公的幫助，衛國遺民得以重新在楚丘復國，並且一直活到了秦二世時期。總計907年，成了存世時間最久的周代諸侯國！
