一個農業國如何轉變為世界強國？

至16世紀後，英國經濟表現的十分活躍，位於世界前列，在18世紀時候，許多發明改變了英國的棉紡織業的面貌，出現了工廠制度生產等等，同時，在18世紀中葉，七年戰爭也使得英國在資本主義發展上佔據了重要地位，也為了英國對外擴大貿易需求，發展國民經濟奠定了基礎，七年戰爭使得英國對外擴大了殖民地，掠奪大量資源，為其工業革命提供了大量資金。

英國在資產階級革命前，還是一個封建的農業國家，那麼英國為什麼會首先掀起工業革命呢？可以從以下幾點來解釋：

1，資本原始累積完成，

2，對外貿易的需求擴大，

3，英國資產階級革命勝利里後，實現了君主立憲制。

4，生產技術的成熟。

5，自然科學的進步 。

6，指導變革的新經濟學理論。

7，新興產業的出現。

那麼這次工業革命有啥特點呢？從兩方面來解釋：第一方面：社會特點

1. 它是從棉紡織輕工業開始的。

2. 它是以解決動力問題作為關鍵和重要標誌的。

3. 它是以機器的發明、使用和製造為中心。

4. 它是以本國的發明創造為主，同時也盡量吸收了前人和別國的先進技術。

第二方面：技術特點

1. 在工業革命初期，技術發明還是工人們經驗的結果，科學只起了其次作用，慢慢的，到了後期，科學與技術相結合，使得科學在技術上實現了完美的融合。

2. 制度化的誕生，實現了由傳統的人工慢慢轉變為以機器代替部分人工的過程。

3. 機器時代重要的特徵是生產機器，機器在各式各樣的環境下慢慢應用，結合工人的經驗，提高了生產效率。

這次英國工業革命的成功，有很大的歷史意義，可以從以下幾點來概括：

1. 工業革命使得英國從一個落後的農業國發展成為世界最先進的資本國家，被曾為『世界工廠』，稱霸了世界半個世紀之久。

2. 工業革命的成功使得英國國內的社會關係發生了很大的變化，進一步加深了工業資產階級的統治地位。

3. 工業革命使得城市人口分布及城市分布發生了根本變化，出現了許多新的工業城市和工業區。還推動了農業的發展，形成了農業工人和農業資本家。

4. 工業革命使國內資本主義社會生產力和生產關係的矛盾、無產階級和資產階級的矛盾開始明顯地暴露出來，並逐漸激化。

5. 工業革命對其它國家工業革命的發展，也產生了重大的影響。對以後的技術改革技術創新也有著重要的影響。

對世界也是有很大的影響，對中國的影響也是很深的，2次鴉片戰爭，使得慢慢淪為半殖民半封建社會，但是從客觀上來看，也使中國資本主義的發展了，洋務運動的興起，表示中國近代化的開始。看到這裡，不知道小夥伴們有啥想法呢？歡迎留言評論~想獲取更多資訊，記得關注我哦~
