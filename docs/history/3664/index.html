杜月笙到底何德何能，能讓政要吳開先成為自己的知己好友？

2019-12-02 0條評論 17次閱讀 0人點贊

摘要

杜月笙一生中結交了政界的重要官員數不勝數，除了上海市長吳鐵成之外，還有一直身居要職的吳開先也是他杜公館的座上客，他也沒想到他和吳開先的交情持續了幾十年，一直到杜月笙在上海病逝，吳開先還是為杜月笙辦理身後事的主要負責人之一。

杜月笙一生中結交了政界的重要官員數不勝數，除了上海市長吳鐵成之外，還有一直身居要職的吳開先也是他杜公館的座上客，他也沒想到他和吳開先的交情持續了幾十年，一直到杜月笙在上海病逝，吳開先還是為杜月笙辦理身後事的主要負責人之一。

吳開先，他是上海法政大學經濟系的第一期畢業生，還在江蘇省松江中學任教。吳開先從1922年加入國民黨，他一直是思想進步的積極分子，也曾擔任國民黨在上海市設立的黨部執行委員。吳開先在上海的時間比較長，並且一直身居要職，權勢威望不比上海市長吳鐵成低，不過他交往的大多是政界要人，所以他與杜月笙這樣的上海名人是沒有什麼交情，不過像吳開先這棵大樹，杜月笙可是想辦法在樹下乘涼呢！

杜月笙身為黑幫老大，但他的徒弟眾多，吳開先任國名黨上海市黨部主任，杜月笙的徒弟就在市黨部工作，杜月笙親自打電話，將徒弟陳君毅召來，要他為自己和吳開先牽線搭橋，結交一下。

一次，陳君毅向吳開先彙報工作。彙報結東後，陳君毅就向吳開先傳達了杜月笙的結交之意。吳開先在上海呆了這麼久，對杜月笙自然很了解，不過他一向自視甚高，並不把杜月笙這種上海土皇帝放在眼裡，不過道上的人不敢得罪，就勉強答應了，並交代陳君毅來辦這個事。陳君毅得到了吳開先的首肯，就馬上跑去向杜月笙彙報。杜月笙自然興奮不已，他大大的誇讚了陳君毅一番，然後就立刻安排宴請吳開先。

吳開先本來只是敷衍一下，不料杜月笙的手腳這麼快，請東都送上門了。他是國民黨政府的要員，特有自尊，要放下自己的身段去拜訪一個流氓頭子，還真有些抹不開面子。於是，吳開先就面有難色地告訴陳君毅，說自己最近公務繁忙，陳君毅一聽，馬上就明白了吳開先的意思，他立刻表示沒關係，杜先生一片誠心，肯定願意改期再邀。

杜月笙做事一向很有耐性，過了幾天，他就叫陳君毅再次邀請吳開先，不料吳開先還是找借口，不見面。杜月笙接連碰了兩次壁，心裡有些惱火，要知道在上海灘上，這樣不買杜先生賬的人可不多。

好在杜笙不是一介莽夫，他效仿劉備「三顧茅廬」的做法，又ー次誠心邀請吳開先。這下好了，吳開先的面子賺足了，又見杜月笙這麼有誠意，也不好再拒絕了。再說，杜月笙也是個響噹噹的人物，他吳開先也得罪不起。於是乎，杜月笙終於成功邀請到了吳開先。

到了他倆約定的這一天，杜月笙在華格桌路杜公館設宴。為了迎接吳開先，杜月笙做了精心的準備。他派心腹總管萬墨林專門負責宴會事宜，並親自囑咐杜公館裡的僕從及杜氏門徒等要恪盡職守、服從安排。除此之外，杜月笙還專門邀請了上海各界的名流來作陪。杜公館裡里外外，也全部打掃了一遍，又被精心布置了一番，以全新的面貌來迎接貴客吳開先。

這次，吳開先也拿出了自己的誠意。他沒有帶很多人來為自己造勢，而是獨自一人驅車前往杜公宴。吳開先的車剛停穩，就有訓練有素的僕人跑上來為他開門。在門前恭候多時的管家萬墨林立即恭恭數敬地引導他向府里走去。

華格桌路杜公館是杜月笙發跡後斥巨資修建的，整個布局及裝飾既奢華又精美，沒有絲毫的土氣，它也絕不遜色於任何一位上海名流的公館或別墅。所以，像吳開先這樣長期出入上流社會的人，到了杜公館，也不禁暗贊它的雅緻，同時也誇讚它的主人有品味。

吳開先穿過曲徑通幽的芳草地，走進綠樹掩映的小洋樓，他驚訝地發現：杜月笙實在不簡單！因站在小洋樓門口迎接他的，居然是聞名上海灘的大律師秦聯奎。在上海的上流社會，沒有誰不知道大律師的，他居然心甘情願地替杜月笙迎客，可見杜老闆的面子真不小啊！

吳開先與秦聯奎也是老朋友了，二人客套幾句後，他就被秦律師恭敬熱情地迎入杜公館的客廳。吳開先一眼就看見身著長衫的杜月笙正與滿座賓客談笑風生，而在座的許多人，都是滬上各界的知名人士，杜月笙見吳開先已到，就丟下眾位客人，滿面春風地迎上來。熱情地與吳開先寒暄，並帶頭鼓掌，歡迎他的到來，這又給吳開先撐足了面子。

吳開先原本還有一些矜持，如今見杜月笙如此熱情，也徹底放下了身價，與眾人熱絡起來。如此來，吳開先對社月笙也由不屑與變得真心佩服了。

當然，見了杜老闆那些精明能幹的手下，那些聲勢顯赫的朋友，再加上杜月笙熱情相待，還不願意與之結交的人，世間少有。

這次宴會，賓主盡歡。吳開先與杜月笙也徹底敞開心扉，成了惺惺相惜的好友。以後，兩人相交十年，相互庇佑，互相幫了不少忙。
