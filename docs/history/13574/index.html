小混混贏了錢被懷疑出老千，杜月笙看了他一眼：送給他500大洋

2019-12-12 180條評論 3,619次閱讀 739人點贊

杜月笙從一個削水果的小販到後來成為「老上海灘的地下皇帝」，他的一生充滿了傳奇色彩。

當時上海灘流傳著三句話：黃金榮愛錢，杜月笙會做人，張嘯林喜歡鬥狠。

杜月笙有多會做人？

他人生的第一桶金是依靠黃金榮的夫人林桂生得到的，一共3000大洋。

如何使用這筆錢，杜月笙花得充滿了智慧。3000大洋，在當時的上海灘能夠娶老婆，買大房，衣食無憂。

但杜月笙當時並沒有考慮這些，他把這些錢拿去分給了自己的兄弟朋友，雙倍奉還了自己之前欠下的債，幫了當時急需要用錢的故人。

當時黃金榮一聽到杜月笙這樣使用這一筆巨款，心中不禁對杜月笙有了不一樣的看法。他認為杜月笙和他之前的門徒有很大的不同。至少從他不僅僅貪圖3000大洋上看，杜月笙這個人格局非常大，眼界非常遠。

年紀輕輕就懂得利用金錢來籠絡人心。

於是此後便把煙土生意交給了杜月笙，杜月笙從此開始發跡。

杜月笙會做人是一方面，而他的眼光也超乎尋常的毒辣。

比如他與戴笠的第一次相遇。戴笠早年是個小混混，以賭騙為生，有一次在上海杜月笙的賭場贏了錢，看場的懷疑他出老千，於是請他進了後堂。

看場的問戴笠：你不知道這是杜先生的賭場？

戴笠回答：我當然知道！

看場的眼睛一瞪：那你還敢在這裡出老千？

戴笠面帶笑意：你們有證據證明我出老千？你們這樣對我，杜先生會同意嗎？要不你把杜先生請過來，我想和他聊聊。

當時杜月笙正好在賭場辦事，看場的聽戴笠這樣一說，以為戴笠認識杜月笙。

但杜月笙過來之後，發現並不認識戴笠，但看到戴笠站在他面前非常鎮定，並不是一個小混混的神色。

杜月笙斷定：這個人絕非等閑之輩。於是立即招人拿來500大洋送給了戴笠。並邀請他到杜氏公館做客。在聊天中他越發覺得戴笠這人不簡單，於是便給了他一封推薦信，讓他到蔣先生的黃埔軍校學習。

而後果不其然，戴笠在6年後成為了軍統最高負責人，而回到上海第一件事便是找到杜月笙，並和杜月笙結為了異性兄弟。

杜月笙雖然是青幫出生，但骨子裡卻非常愛。淞滬會戰期間，為了阻止日本海軍，杜月笙率先將自家公司的貨輪鑿沉，也就是在杜月笙的帶領下，其他輪船公司紛紛效仿，有效的阻礙了日本海軍對淞滬的進攻。

而在上海淪陷之後，杜月笙還指揮門徒搶救傷員，暗殺漢奸，當時很多當了漢奸的人死於杜月笙的門徒之手。

當時日本人找杜月笙合作，杜月笙說了一句非常著名的話：我是一個中國老百姓，礙於兩國交戰，難以從命。

這也是為什麼很不近人情暗殺之王王亞樵會單單對杜月笙充滿了尊敬。杜月笙不止在其落難時幫助了他，還和他一樣是個愛國之人，從而此後青幫與之有很大的過節，但王亞樵對於杜月笙卻始終留有人情。

很多人對於杜月笙到底留下了多少財產非常好奇，要知道當時杜月笙所開的三鑫公司，巔峰時期一年的收入相當於當時民國政府一年財政收入的三分之一。杜月笙儘管只佔股三分之一，那也是一筆不小的分紅。

但在臨去世之時，杜月笙給自己家人留下的遺產卻並不多，一共只剩下11萬美元，其他的錢不是借出去了就是捐給了軍隊或者窮人。

當時每一個太太分到了一萬美元，每一個兒子分到了一萬美元，女兒沒出嫁的分到了六千美元，出嫁了的分到了四千美元。

分完錢後，杜月笙還讓家人把欠條箱拿了出來，讓家人把所有欠條都給燒掉。

他的家人都知道，這些年杜月笙花錢如流水，沒有過億也有上千萬，很多人找杜月笙借錢，往往打一張欠條就可以直接領錢走人。

光是王新衡一人便欠了杜月笙500根金條。

杜月笙看出了家人的心思，於是說到：

「借我錢的人，有的流落天涯，有的生死未知，我不想在我死後，你們一個個都成為追債的人，將我活著的這些交情都弄不好看了，能記住我的好的人，在你們困難的時候，會伸手幫助的，記不住的人，你們去追債，也可能招來殺身之禍，大恩如大仇，你們要記住這個道理！」

大恩如大仇，很多人都看不透，但杜月笙看得透。

我們想想，如果某一天杜月笙的子女拿著這些欠條去找人還錢，有錢的自然能還上，那沒有錢的，難免被逼急了會起殺心。因為當年和杜月笙打交道的人，大多魚龍混雜。

可見杜月笙的成功不是沒有道理的，直到現在他的一些話依舊適用於這個社會。

1：不要怕被別人利用，人家利用你說明你還有用。

2：花一文錢要收到十文錢的效果，這才是花錢能手。

3：做事要做到刀切豆腐兩面光。

4：做人有三碗面最難吃：人面場面情面。

5：頭等人，有本事，沒脾氣；二等人，有本事，有脾氣；末等人，沒本事，大脾氣。

9：人活在世上要靠兩樣東西，膽識和智慧。

10：錦上添花的事情讓別人去做。我只做雪中送炭的事情。

11：錢財用的完，交情吃不光。所以別人存錢，我存交情。存錢再多不過金山銀海，交情用起來好比天地難量！

12：每月存款折上多幾個零不算你有多少錢，花出去多少錢才算你有多少錢。

14：做人不能當刺蝟，能不與人結仇就不結，誰都不跟誰一輩子。
