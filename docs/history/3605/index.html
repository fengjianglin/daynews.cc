皇上請宰相吃飯，因為一雙筷子就把德高望重的宰相殺了！

2019-12-02 0條評論 21次閱讀 0人點贊

摘要

帝王掌握著朝中大臣的榮譽與生死，即使深諳帝王之術之人都未必可以一直得到寵信，更不要提那些不會斡旋於朝廷鬥爭中的人了。人際關係是一門值得探究的學問，在古代，似乎顯得尤為重要，當年漢景帝請周亞夫吃飯，為何因為一雙“筷子”就要除掉這位德高望重的宰相呢？

帝王掌握著朝中大臣的榮譽與生死，即使深諳帝王之術之人都未必可以一直得到寵信，更不要提那些不會斡旋於朝廷鬥爭中的人了。人際關係是一門值得探究的學問，在古代，似乎顯得尤為重要，當年漢景帝請周亞夫吃飯，為何因為一雙「筷子」就要除掉這位德高望重的宰相呢？

周亞夫是漢景帝時期一位有名的將軍，他之所以能夠躍居宰相之位，全然憑藉的是自己的努力，他曾短時間內平定七國之亂，又擊敗匈奴。周亞夫可以稱得上是一位真正的將領，對軍隊紀律十分看重，嚴格執行作戰計劃，為人有條不紊。即使是在面對漢景帝時，也要求他嚴格遵守軍營的規定，這其實是一個有利又有弊的性格特點。

利的是他在軍隊樹立了無人能及的威望，大家都臣服於他，弊的是漢景帝擔心他此後可以隨時不把自己放在眼裡，從而產生叛亂之心。所以漢景帝封他為宰相，一部分原因在於有意牽制他的兵權，但是對周亞夫這個在沙場上征戰多年的大將來說，官場上的那一套他實在是吃不消，無意之間就得罪了很多人。

慢慢的，周亞夫的政敵越來越多，如此一來就很難在朝中真正安身立命下去。再者，不管漢景帝做什麼決定，他總是持反對意見，每每都把堂堂一國之君弄得很沒面子。所以漢景帝冷淡他也是情理之中了，以至於最後直接讓周亞夫去戍守邊疆了，眼不見為凈。但沒幾年，這位皇帝就開始想起周亞夫的好了，於是又讓他回到宮中輔佐太子。

本著想要試探周亞夫的原則，一次，漢景帝特意大設宴席並邀請眾多群臣，周亞夫就是其中一個。《史記》中對此有過一段詳細記載：「頃之，景帝居禁中，召條侯，賜食。獨置大胾，無切肉，又不置櫡。條侯心不平，顧謂尚席取櫡。景帝視而笑曰：「此不足君所乎？」條侯免冠謝。上起，條侯因趨出。景帝以目送之，曰：「此怏怏者非少主臣也！」

漢景帝給了周亞夫一塊肉，但是卻未給他筷子，如此一來，周亞夫便認為皇上這是在故意羞辱他，於是找太監要了餐具吃完招呼不打就直接離開了。所以就有了漢景帝因為一雙筷子處死周亞夫之說，周亞夫乃一介武夫，哪裡會懂得漢景帝這般謎語，最後漢景帝只淡淡說了一句「這樣的人如何能夠輔佐少主呢？」

回到這句「此不足君所乎？」我們揣摩漢景帝這句話的言外之意是「你今日既得這番榮耀，還有什麼不滿足的呢？」之後周亞夫又不合時宜的「免冠謝」，只是摘下帽子稱罪就直接離開了，在漢景帝看來這就是大不敬的表現。也發現他還是老樣子，沒任何錶現，在面對自己時都已經這般了，日後誰能料到他會在年幼的新君面前會做出什麼更不合禮法之事呢？

周亞夫死的冤，也是在意料之中。碰上漢景帝這樣一位腦迴路無限長的帝王，不會隨機變通，實在不行啊。
