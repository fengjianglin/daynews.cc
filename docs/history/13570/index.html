杜月笙與王亞樵的交鋒，為何主動退讓？他的回答彰顯處世之道

2019-12-12 47條評論 2,787次閱讀 831人點贊

黃金榮曾經對其門徒說：

王老九這個人很討厭，素來吃軟不吃硬，碰上了給個方便大家相安無事，切莫在他面前惹是生非。咱們在明處，他在暗處，君子咱不怕，毛賊咱不惹。」

黃金榮口中的王老九就是王亞樵，黃金榮也是威震上海灘的大人物，可是面對王亞樵這樣的人物還是選擇退讓。但是今天我說的不是黃金榮，而是杜月笙與王亞樵之間的往事。

杜月笙、黃金榮等人合影

杜月笙和王亞樵都曾是上海灘的風雲人物，杜月笙屬於青幫，王亞樵創立了斧頭幫，兩個幫派平時都攝於對方的強大勢力，不撕破臉，但俗話說一山不容二虎，兩個幫派在發展的過程中還是產生了一些摩擦。

杜月笙我就不介紹了。簡單介紹一下王亞樵，王亞樵，安徽合肥人，家世貧寒，自幼讀書，聰穎過人，後來來到上海，看到碼頭上的工人經常受到剝削和欺負，於是他便把這些來自安徽的老鄉組織起來，每人還發了一把斧頭，專門對付那些奸商地痞流氓和一些大漢奸，當時的人都稱他們為「斧頭幫」。

王亞樵為人非常仗義，嫉惡如仇，非常具有愛國情懷。他曾經成立了一個鐵血鋤奸團，專殺漢奸走狗，他本人也是俠肝義膽之輩，但凡是他認可的朋友，都鼎力相助，王亞樵有點像金庸小說里的俠客，快意恩仇，肝膽相照。

他與杜月笙的一次恩怨來源於一條船。

當時李鴻章的孫子李國傑托關係找到王亞樵，想讓他出手殺掉趙鐵橋，並答應王亞樵，事成之後將招商局最大的輪船「江安」號轉交給王亞樵打理。

李國傑

可是當王亞樵完成這項任務來收船時，得知李國傑卻早已將江安號給了張嘯林，張嘯林也是上海灘有頭有臉的人物，拒絕將江安號交出，為此雙方的人還打了起來。

這一下子就惹惱了王亞樵，當天夜裡王亞樵就帶人將張嘯林家的後院炸出了一個大洞。

軟的怕硬的，硬的怕不要命了，張嘯林雖然也是青幫頭子，但面對不要命的王亞樵時心裡還是有些打怵，便急忙找杜月笙商量對策。

杜月笙權衡利弊，認為與王亞樵拚命並不值得，而且此時自己家大業大，打打殺殺並非良策，杜月笙選擇退讓，並與李國傑商議拿出另一艘同等噸位的輪船交給王亞樵。

王亞樵斷然拒絕了這個條件，他認為「輪船事小，面子事大，」一定要接管江安號。

王亞樵的考慮也是有道理的，這件事明白著是李國傑將這船一女二嫁，挑起了青幫和斧頭幫的紛爭，兩個幫派誰都認為自己是上海灘老大，王亞樵就是靠著義氣來籠絡人心，一旦在青幫面前退讓，自己的威信將大打折扣。

王亞樵

杜月笙給了王亞樵台階，可是王亞樵卻不下，沒有給杜月笙和張嘯林面子，看似一場大規模械鬥不可避免。

實際上王亞樵也確實做好了打仗的準備。

王亞樵帶領五百多人，一路浩浩蕩蕩殺向黃浦碼頭。我查閱了很多資料，都不約而同的記載了這個細節，當天王亞樵站在高處對眾人說：

「想當年皖人抱成一團，提斧頭殺得上海灘的癟三們屁都夾著放。幾年來，我們東奔西走，卻讓烏合之眾尋釁鬥毆傷我兄弟，真是可忍孰不可忍。借今日之機，我皖人要重新聯盟，我王亞樵要與同鄉兄弟同生同死，絕不讓滬土癟三稱王稱霸。今天武力接管『江安』號，權作斧頭黨復興誓師，有膽量的跟我去見見世面去。」

王亞樵本來是抱著大幹一場的決心來碼頭的，但是讓他沒想到，在這卻碰上了滿臉笑容的杜月笙，據現有的資料看，當天杜月笙身著長袍馬褂從汽車走出，滿臉含笑，從容不迫地走向王亞樵的隊伍。面對殺氣騰騰的斧頭幫，杜月笙大有單刀赴會的架勢。

杜月笙

大人物相見，當然不是上來就開罵和動手的，而是客氣得不得了，再說，王亞樵搶船有決心，但是依仗人多暴揍杜月笙，他還沒這個膽量，更何況是笑臉相迎的杜月笙。

王亞樵將矛頭直指李國傑，大罵李國傑忘恩負義，不守信用，而杜月笙也順著王亞樵話直指此次事情的起因就是李國傑將船一女二嫁，害的兩個幫派反目，同時表達和好之意。

杜月笙不僅當場同意將江安號交給王亞樵，而且還當場拿過一個裝有十幾根金條的箱子交給王亞樵以示友好。

整個過程杜月笙表現的和藹友善、不卑不亢、而且給足了王亞樵面子，上文也提了王亞樵這個人吃軟不吃硬，更何況杜月笙也是個有頭有臉的人物。

孟小冬與杜月笙

雙方最終化干戈為玉帛，一笑泯恩仇，此後兩個幫派再也沒有發生大的衝突。其實這個結局對雙方來說是最好的結局，杜月笙為何會忍下求和，用杜月笙的話來說就是：

王亞樵是窮光蛋，惹了事一拍屁股拔腿就走，咱們可是有家有當，惹了他，今日放你一把火，明天殺你一個人，是很划不來的。

杜月笙看似退讓了，但實際上利益得到了最大化。

但是這次事件無論怎麼看，王亞樵都成為了勝利者，三大亨不僅完全退讓，而且還拿錢免災。從中可見王亞樵其人的厲害。

杜月笙與王亞樵是兩種人，也是兩種人生，杜月笙屬於八面玲瓏，而王亞樵是寧折不彎，雖然杜月笙善於與人打交道，但是王亞樵的俠義氣概更讓人佩服，對於王亞樵，咱們偉大領袖曾有著這樣的評價：「殺敵無罪，抗日有功，小事不檢點，大事不糊塗。」
