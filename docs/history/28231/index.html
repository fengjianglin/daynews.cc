張飛的兒子張苞，戰力屬於什麼水平，為何他的死讓諸葛亮吐血暈倒

2019-12-22 293條評論 2,345次閱讀 1077人點贊

三國時期，雖然說英雄猛將不少，但是《三國演義》當中，卻是以關羽之死為分水嶺。我們可以明顯感覺得到，往後的故事，就沒有之前那麼精彩了。可俗話說得好：「龍生龍，鳳生鳳」，雖然這些英雄或戰死或老去。但是他們的子孫，卻繼承了他們的意志和英勇，依然在戰場上或者政治上活躍著。本文要講一下，張飛的兒子張苞。在蜀漢後期的時局當中，張苞的實力到底如何呢？為何他的死會讓諸葛亮吐血暈倒？

首先討伐東吳張苞捨生忘死

在演義當中，張苞雖然武勇，但是比不上他老爹張飛那樣，被三國公認是萬人敵。可張苞在同輩裡面，可以說是中上水平了。甚至在諸葛亮穩定蜀國的時候，和關羽的兒子關興一起，被稱為「小關張」，張苞的實力和魅力，還是可圈可點的。

在《三國演義》第八十二回書中描述道：

桓亦罵曰：「汝父已作無頭之鬼；今汝又來討死，好生不智！」張苞大怒，挺槍直取孫桓。桓背後謝旌，驟馬來迎。

兩將戰有三十餘合，謝旌敗走，張苞乘勝趕來。李異見謝旌敗了，慌忙拍馬輪蘸金斧接戰。張苞和李異戰二十餘合，不分勝負。吳軍陣營中裨將譚雄，見張苞英勇，李異不能勝，卻放一冷箭，正射中張苞所騎之馬。

想要判斷武力，最好的辦法，就是看他本人的戰績，在為父報仇征伐吳國的時候，此時是張苞的第一戰。在演義當中，張苞在這一戰里，先是和謝旌鬥了三十回合，然後贏了，接著再和李異戰鬥，不分勝負。

當然在這裡，我們看出，其實張苞是穩操勝券的。就連敵人，都認為自己隊友沒辦法戰勝張苞，還要在外面放冷箭，才能讓張苞吃個虧。更何況在此之前，張苞還是遭受過車輪戰的，可以說這一戰，頗有他父親當年在戰場上奮勇的風骨。

但是他的對手，在歷史上也不是很有名氣，可能證據力度不足。這也不打緊，因為他後面還出場了很多次。

在夷陵之戰之後，張苞再次出場就是在諸葛亮的北伐時期了。

上文中也曾提到過，張苞和關興，作為蜀國二代人物當中的重要人物，在軍中也是相當受到諸葛亮的器重的。

自然在諸葛亮北伐過程中，諸葛亮也會時常讓這些小將出來磨練。遺憾的是，在北伐當中，張苞雖然很積極，可很多時候，都是領兵受計去配合趙雲或者黃忠這些老將，他是作為副將上陣的，自然和曹魏有名的將領交手很少。根據《三國演義》第九十四回中說到：

關興、張苞兩路兵殺出，圍了曹真、郭淮，痛殺一陣。曹、郭二人，引敗兵沖路走脫。蜀兵全勝。

而這個戰績，可以說是張苞其最好的戰績了。他與關興一起，擊敗了曹真和郭淮，然後追著衝殺一陣，奪走了曹魏的營地。

為什麼說是最好的呢？因為在這場戰中，雖然說沒有拿到有名武將的人頭，但是擊敗了曹真和郭淮啊，這兩人是何許人也？曹真是曹操的養子，不僅在戰場上頗有名氣，在曹丕死後，甚至還作為輔佐大臣，負責對抗蜀國。可以說他的武力檔次，雖然說比不上五虎上將這種，但是也絕不是平庸之輩。

至於郭淮更不用說，在張郃死後，是作為對抗蜀軍的大將軍上任的，在軍事上，甚至能夠將胡人治得服服帖帖地，他和姜維對峙的時候，更是識破了姜維的計謀，讓姜維連續吃了兩次敗仗。

張苞在三國當中武力值還是不錯的，雖然有著關興的幫助，但是戰勝郭淮和曹真，的確也是有著真材實料的，就憑這一點，就奠定了他武藝 的水平。可惜的事情就在於，張苞雖然英雄，卻天妒英才短命啊。還沒在戰場上面發揮餘熱，就已經悄然退出了歷史的舞台。

為何張苞之死讓諸葛亮吐血暈倒？

《三國演義》第九十九迴文中描述道：

忽報有人自成都來，說張苞身死。孔明聞知，放聲大哭，口中吐血，昏絕於地。

張苞在北伐之戰中，雖然英勇，但是最後卻意外跌落山澗受傷而死，令人痛心。從諸葛亮的反應，我們可以看出，丞相對於張苞的死是十分惋惜的，原本想著培養成自己未來的北伐主力的。從側面上，張苞的武勇程度，確實是深受丞相的信賴，不然也不至於讓諸葛亮吐血。由此可見， 如果張飛的武力水平，在三國里屬於第一梯隊。那麼他的兒子張苞，雖然因為英年早逝，算不上第一流，也是可以在第二梯隊里，穩佔上風的。

關於張苞的實力，您是怎麼看的呢？
