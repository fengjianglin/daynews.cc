雍正登位，為何善待廢太子胤礽，而不寬恕大阿哥胤褆？

天天要聞
2019-12-20

雍正登基後，幹了幾件大事，其中就有涉及他那幾個兄弟。比如釋放胤祥並賜封怡親王，賜封老八為廉親王，賜封廢太子為理親王。唯獨可謂眾兄弟該升的升，該放的放，唯獨對大哥，堅持老爹，康熙爺的政策繼續關著。這讓後人很奇怪為什麼，他把曾經的對頭老八都封了親王了，而且同樣被軟禁的太子也封了王爺，為什麼偏偏對老大卻關著不放呢？

一，廢太子是雍正的舊主！雍正登上皇位，還得感謝這個不爭氣的二阿哥

在廢太子沒廢之前，康熙的皇子中就分了兩大派！即太子黨和阿哥黨！雍正那時叫胤禛，跟十三阿哥是太子的左膀右臂！天地良心！那時候雍正跟十三阿哥死心塌地輔佐太子！就是太子借庫銀，太子調戲鄭春華，都是雍正和老十三幫他擦屁股！

後來，廢太子徹徹底底被廢！雍正如願登基，念及廢太子是自己曾經的舊主！也就讓廢:太子善終！也說明了雍正皇帝也不是一個刻薄寡恩的人！同時，雍正內心裡還是非常感謝這個廢太子的不爭氣！不然雍正哪有機會登基？

二，雍正為了彰顯自己的仁德

雍正封太子是為了彰顯自己的仁德，而且可以收服太子的故舊，特別是雍正剛剛登基，朝廷內外都是老八的人，他急需擴大自己的勢力。要知道胤礽可是當了三十多年的太子，雖然被廢，但他的政治影響多多少少還在，用一個沒有實權的親王就能收買這些人，再划算不過了，就算不能收買也能防止他能倒向老八。

三，為什麼不寬恕大阿哥胤褆呢？

因為胤褆不但是魘鎮太子胤礽的事情被曝光後，遭到父皇以及所有兄弟指責之外，還曾憑藉皇長子的身份先後算計過三阿哥胤祉，四阿哥胤禛，可以說人品完全敗壞，失道寡助！最後被康熙皇帝囚禁時，竟然沒有一位皇子為他求情！

大阿哥胤褆早年間曾經跟隨康熙皇帝遠征葛爾丹，並斬獲很多軍功，年紀輕輕就被康熙封王重用！到了後來由於受到舅姥爺明珠的唆使，大阿哥利欲熏心，感覺自己可以和皇太子胤礽爭奪起皇位了！當時大阿哥生母惠妃是康熙皇帝後宮四妃之首，舅老爺明珠權傾朝野，身邊聚集了很多人支持他爭奪皇位！於是大阿哥變得飛揚跋扈起來，除了平日不斷給其他一些皇子下絆子之外，甚至偷偷買通太子身邊的人，企圖用邪術魘鎮太子！

雍正對胤禔還是比較仁慈的，其他幾位被雍正圈禁的兄弟，下場就非常的慘。雖然胤禔失去了身份自由，但是基本的生活還是有保障，沒有遭到任何人的干擾，在他被囚禁的26年里，他與他的11位妻妾，生了29個孩子，其中14個女兒，15個兒子，雍正對他也算是盡了兄弟情誼了。

綜上所述，小編認為大皇子是康熙親自下令圈禁的人。他的「不忠不孝不仁不義」在朝中和其他皇子心中已經定型了，所以雍正永遠不會放大阿哥胤褆。並且，清朝的祖制不允許統治者寬恕不忠、不孝、不義的人。對此大家有什麼想說的呢？歡迎留言分享。
