西晉滅吳之戰沒有步曹操和劉備的後塵，實現快速取勝的戰略分析

天天要聞
2019-12-20

《三國演義》中描述的戰爭很多，但對於西晉滅吳一戰卻語之簡單，遠沒有滅蜀之戰渲染得跌蕩起伏。殊不知，在此之前，曹操與劉備均集聚了數倍於孫吳的兵力攻吳，先後被以弱勝強，落得慘敗而回。西晉滅吳為什麼會用幾乎對等的兵力，卻取得對擁有地利優勢孫吳的最終勝利呢？西晉吸取了曹操和劉備什麼失敗教訓，又採取了什麼戰略而一舉獲勝呢？我們來研究一下《三國演義》沒有告訴我們的這場戰爭。

一、赤壁之戰與夷陵之戰的失敗原因分析

赤壁之戰與夷陵之戰都是三國時期最著名的以弱勝強經典案例。在赤壁之戰中，曹操親率80萬大軍征吳（實際參戰兵力約22萬左右），而孫權方面總兵力約為8萬左右（派給周瑜直接作戰的約3萬人），劉備方面僅1萬左右兵力於夏口。兵力實際對比為6:1多一些。曹操失敗的主要原因是：無強大水軍；北方兵士不習水戰；戰船相連。

至劉備伐吳時，劉備舉傾國之力，親率了70萬大軍（實際參戰兵力約10萬左右），而孫權方面派出迎敵的為5萬左右。兵力實際對比為2:1多一些。吸取了曹操的失敗教訓，劉備在攻吳中採用了陸戰為主，也在戰役的前期雖然取得了一定的勝利，但還是沒有避免慘敗的結果。劉備失敗的主要原因是：戰線拉長，需分兵把守；久攻不下，兵士疲憊；沒有強大水軍。

二、西晉伐吳的戰略分析

西晉滅吳戰略的總設計師是司馬炎，即”先定巴蜀，三年之後，因順流之勢，水陸並進”。這一戰略既吸取了曹操和劉備沒有強大水軍而失敗的教訓，也延承了北方軍隊對孫吳軍隊在陸戰上的優勢。有了高瞻遠矚的戰略還需要堅定的實施者，幸運的是，西晉有羊祜和王濬。羊祜是極力主張並參與密謀滅吳的主要大臣之一，他在戰役發起之前主要對東吳方面實施了”緩兵”的策略。雖在襄陽與東吳名將陸抗對峙，但善施恩惠，主動送還吳軍俘虜、吳國禾麥、吳人射傷的禽獸等，使”吳人翕然悅服”，以為雙方就此會長久平安無戰事。”緩兵”策略帶來的另一個後果是給東吳內亂創造了條件。由於邊境無戰事，繼位之初，表現出了明君之相的孫皓開始飄飄然了，”皓既得志，粗暴驕盈，多忌諱，好酒色，大小失望。” 面對意欲吞併孫吳的西晉，國內混亂的吳國已經沒有了一致對敵的政治基礎。王濬是由羊祜推舉擔任益州刺史的，建造了數量眾多的戰船，為西晉打造兵力7萬多人的強大水軍。至此，西晉伐吳已是箭在弦上了。

三、西晉伐吳的戰術分析

西晉伐吳出動的總兵力為20萬人，兵分六路伐吳。是的，西晉採取了完全違背兵法的戰術，不是集中優勢兵力遷滅有敵有生力量，而是採用了兵家大忌的分兵戰術。要知道在當時孫吳國內兵力為23萬人左右，每路進攻都有可能面臨著同等兵力或是數倍兵力的防守。但如果我們仔細分析曹操與劉備失敗的原因就會發現西晉這一反兵法戰術的高明之處。

曹操與劉備攻吳失敗都有一場戰鬥失敗，導致全局慘敗的共同原因。即攻方雖然取得了兵力優勢，但也有利於守方集中精兵處用地利優勢防守於一點，只要把攻方拖入持久戰，總會找到攻方的弱點而一役敗之。東吳方面看似兵力較多，但分布在廣闊的長江防線上，就顯得捉襟見肘了。況且當時東吳地區山嶺眾多，雖然對守方是有利的地利條件，但同樣也會給守方帶來調動兵力和後勤保障的困難。劉備在攻吳初期勢如破竹，卻在夷陵被迫屯兵於樹林就是上述原因。

所以西晉攻吳雖然兵分六路，但每一路對敵都可以有兵力上的優勢，而且任何一路的進攻，都會讓分散於各處的吳軍無法集結而被各個擊破。特別是來自蜀地的水軍船隊是西晉本次伐吳快速作戰的主力。它沿江順流而下，象一把尖刀，從長江西面直插東吳沿江防禦的側翼，不給孫吳方面持久戰的機會。為了達到良好的的效果，晉武帝還特意要求此路艦隊可以節制沿路陸軍部隊，沿路的陸軍要配合支援水軍的進攻。水陸聯合使得東吳本就捉襟見肘的防備力量，在面對北面和西面的晉軍時，顯得更加渺小。

四、西晉伐吳的戰爭過程

《晉書.武帝紀》（咸寧五年）十一月，大舉伐吳，遣鎮軍將軍、琅邪王伷出塗中，安東將軍王渾出江西，建威將軍王戎出武昌，平南將軍胡奮出夏口，鎮南大將軍杜預出江陵，龍驤將軍王浚、廣武將軍唐彬率巴蜀之卒浮江而下，東西凡二十餘萬（註：王浚即前文王濬）。由司馬伷和王渾組成的東線軍團，死死地將孫吳精銳吸引在江淮一線，畢竟他們兵鋒直至建康，孫吳的重兵精銳集團深受牽制，無法支援長江中上游的吳軍。而其他幾路陸路大軍的目的一個，就是吸引東吳兵力，打通長江之上的各個戰略要點。為王濬所率主力水師順江而下，直抵建康城下創造條件。

而孫吳方面呢，完全沒有了孫權時期眾志成城的決心和清晰有效的對敵策略。在陸戰上，面對司馬伷和王渾的南下，孫皓派遣丞相張悌率主力吳軍北上抵禦。”晉來伐吳，皓使悌督沈瑩、諸葛靚，率眾三萬渡江逆之”。這三萬吳軍應該是精銳的陸軍了，畢竟到了那個時候，吳國已經沒有什麼可以保留的了，但是這三萬主力吳軍卻遭遇了大敗。失敗的原因就是”渡江逆之”, 如果死守南岸，以逸待勞應當有擊潰來犯之晉軍的機會。可惜張悌不聽沈瑩的正確建議，妄想守住寸土，結果兵敗身亡。

而在水戰上，王濬採用了曹操鐵索連船的策略。是的，西晉戰船使用了曹操兵敗赤壁的辦法。因為王濬認為其水軍佔據了順風順水位置的上游，且士卒訓練充分，大船邊以小艇進行游擊保護，根本不懼怕吳軍可能的火攻與襲擊。反而由於大船相連，士兵活動自由，對於吳國船隊有壓制性的優勢。吳軍雖然採取了在險要之地設置鐵鏈、鐵錐阻擋的被動防禦措施。但由於陸戰失利，戰略要點失守，西晉水軍得以時機可以命善水者乘竹筏拔掉江中暗藏的鐵錐，又用火烤熱鐵鏈使其斷裂。輕鬆突破阻礙的西晉水軍，順利攻克了西陵、荊門，並斬殺吳軍水軍都督陸景。當然，孫吳也有忠軍能戰之士，如孫韻所率的吳軍駐守江陵。他並未被巴蜀艦隊的攻勢嚇倒，主動出擊迎戰西晉水軍。曾經號稱三國中最為強大的吳國水軍，無法破解鐵索連船，兵敗而退守江陵。結果，被王濬提前安排的伏兵趁著吳軍撤退進城的時候，尾隨而入失去江陵。在王濬水軍進攻沿途的吳軍守將張象、陶濬，曾率水軍想要阻擊，但都因西晉水軍艦隊的攻勢凌利而迅速失敗。如此聲勢浩大的船隊，在長江地區的戰場上，確實無人能敵。很快，王濬所部艦隊經過三山，順利抵達吳國首都建康附近。

困守建康的吳主孫皓，見西晉的兵馬正在逼近首都，知道自己大勢已去。於是，聽從薛瑩、胡沖等人的建議，派遣使者分別向三位西晉將領投降，並將印璽交與琅琊王司馬伷。隨後王濬率軍攻入南京，孫皓反綁著手把棺材裝在車上向王濬投降，東吳宣告滅亡。

縱觀此次氣吞山河一邊倒的西晉滅吳之戰，西晉的戰略戰術可以說是針對性完美，吸取了曹操和劉備伐吳之戰的經驗教訓，利用兵力分散和吳國只能集中力量抵抗一個方向的特點，上演了一場反兵法完勝的好戲。

五、西晉伐吳的題外思考

曹氏宗親羊祜為什麼沒有被牽連仍獲司馬氏信任？羊祜是是夏侯霸的女婿，算是曹氏的宗親。在高平陵事變後，司馬懿一舉鎮壓曹氏勢力，夏侯霸被迫投降了蜀漢，與夏侯氏有牽連的一干人等均受連累。而羊祜不僅安然無恙，被授予了重兵之權，還依然可以公開照顧夏侯氏家屬，其提出的伐吳戰略也得到了晉帝的支持。羊祜被信任的原因是什麼呢？不難判斷，在高平陵事變之前，羊祜就已經投靠了司馬懿。另外羊氏是當時的名門世族，羊祜的姐姐嫁給了司馬師，性格溫和，為人正直，不拉黨結幫，在當時的名望非常高也是原因之一。在這裡我們要佩服的是羊祜的遠見。

功高震主的王濬怎樣得到善終的呢？功高震主，是自古良將的致命傷，王濬卻逃脫了這一致命法則。他是怎麼做到的呢？”藏精明而露拙笨”，從還沒打下建業起，他就爭功，甚至謊報軍功和與其他將領爭頭功打口頭官司。除此之外，王濬看起來”胸無大志”，平吳後一改過去的樸素作風，十足一個暴發戶嘴臉。這樣的王濬很讓司馬炎放心，越多的人彈劾王濬，皇帝越有安全感。

假如西晉晚幾年再攻吳，吳國還會滅亡嗎？答案會是肯定不會了。因為機會是稍縱即逝的。滅吳之戰，西晉內部爭議不斷，一拖再拖，幾乎錯過了最好時機。如果再拖上兩年，暴君孫皓被推下台，換成一位有為之君，孫吳政權得以穩定，國內官民怨念平息。考慮到司馬炎死後，指定的是一位白痴兒子接的班，又有”八王之亂”諸多政治隱患，攻吳不一定會如此順利，甚至發起戰爭都存在疑問。
