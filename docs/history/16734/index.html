梁啟超與陳寅恪家族三代人的交往

2019-12-14 93條評論 3,551次閱讀 1047人點贊

梁啟超是中國近現代少有的思想家，更是近代中國偉大的啟蒙者，其思想和人格魅力影響了一代又一代中國人，從沒有在國人的視野中淡出或消失，至今仍是現代中國的熱門詞和流行語，其《少年中國說》被多少莘莘學子吟詠，而其教子成才的故事，也是常常被人提及，而撰寫梁啟超其人的著述和文章更是何其多也，本文僅選取和記述梁啟超與陳寅恪家族三代人的交往歷史，藉以緬懷梁啟超先生，因為即將過去的2019年，也是梁啟超逝世整整九十周年的忌年。

梁啟超

1.梁啟超與陳寅恪祖父陳寶箴

梁啟超與陳寅恪祖父陳寶箴的交往始於1897年，時任湖南巡撫的陳寶箴邀請梁啟超出任長沙「時務學堂」中文總教習，這一年梁啟超才只有24歲，就已經是聲名顯赫了，而梁啟超之所以能夠贏得大名，最初是依靠其老師康有為的名聲，並曾於1895年與康有為一起集結600餘名舉人，聯名上書光緒皇帝，反對簽訂中日《馬關條約》，史稱「公車上書」。

梁啟超

陳寶箴是1895年出任湖南巡撫的，而「時務學堂」創建於1897年10月，地址在長沙小東街，是當時湖南第一所新式學堂，也是湖南現代教育的肇始，當年8月發布的《湖南時務學堂緣起》就闡明了設置學堂和培養人才的宗旨，陳寶箴委派黃遵憲、熊希齡具體負責學堂籌備事宜，並任命熊希齡為提調（校長），主持一切行政事務。

陳寶箴與他的孫子們合影

陳寶箴是在黃遵憲的建議下創辦「時務學堂」的，當時黃遵憲剛到湖南出任長寶鹽法道，負責管理一省食鹽的生產和運銷，他到任之後不久，正值湖南按察使李經義因事進京，由黃遵憲暫時代理按察使，而按察使是掌管一省刑獄和官吏考核的官員，黃遵憲上任伊始，就積極協助陳寶箴進行政治、經濟、文化、教育等各方面的改革。

黃遵憲

而梁啟超之所以能夠於1897年10月擔任「時務學堂」中文總教習，也是黃遵憲向陳寶箴舉薦的結果，他提出要邀請梁啟超擔任中文總教習，因為在此之前，黃遵憲在上海辦《時務報》時，曾邀請梁啟超擔任主筆，期間梁啟超以新穎且犀利的議論和通俗流暢的文字，撰寫了《變法通義》、《論中國積弱由於防弊》等一系列政論文章，大力介紹西學，明確提出要改革中國的羸弱和落後，就必須學習西方國家的政治、經濟和文化教育制度。

當時出任「時務學堂」西文總教習的是李維格，他當時是上海《時務報》的西文翻譯，熊希齡對李維格的印象極佳，稱其「西學英文極通，品行也可敬。」李維格比梁啟超年長6歲，後來接受晚清重臣盛宣懷的邀請，出任上海南洋公學的監院（類似於教務長）。

李維格與其侄子李復幾合影

「時務學堂」作為一個中西兼習的新式學堂，梁啟超和李維格可謂是雙星閃耀且珠聯璧合，對西學的傳播、民智的開啟和思想的啟蒙都做出了不可磨滅的歷史貢獻，並且為湖南的文化教育事業開創了新的局面，而這一切都與陳寶箴的開明和開放是密不可分的，當時湖南省在陳寶箴的主持下，也是唯一實施「戊戌變法」的省份。

「時務學堂」存在時間不足一年，期間只招考了三批考生，錄取學生總數在200人左右，卻培養出了許多傑出的人才，其中蔡鍔、范源濂、楊樹達、方鼎英、李復幾、李炳寰、范旭東等人的名字為公眾所熟知，而「戊戌六君子」之首的譚嗣同，當時也是「時務學堂」的中文教員之一。

時務學堂舊址

「時務學堂」後來隨著「戊戌變法」的失敗而停辦，梁啟超遭到清政府的通緝而被迫流亡日本，而作為封疆大吏的陳寶箴也遭到清廷的整肅，來自清廷的聖旨曰：「湖南巡撫陳寶箴，以封疆大吏濫保匪人，實屬有負委託。陳寶箴著即行革職，永不敘用。」削職為民的陳寶箴遂攜全家返回故里定居。

2.梁啟超與陳寅格父親陳三立

陳三立比梁啟超年長20歲，是光緒十五年（1889年）進士，而梁啟超曾在光緒十六年（1890年）春進京參加會試時名落孫山，當時梁啟超尚不滿18歲，這在科舉考試時也屬正常，畢竟還是太年輕了，儘管梁啟超才華橫溢。

陳三立曾出任清廷吏部主事，為正六品官員，相當於現代的司局級幹部，而吏部是清政府掌管文職官員的任免、考核、調動等事宜，是一個頗具實權的政府部門，類似於現代的人事部或組織部的職能。

陳三立

陳三立在吏部任職不久，就以六品官員之身份，回到了陳寶箴身邊侍父，當時陳寶箴正在湖北布政使的任上，期間陳三立曾應湖廣總督張之洞之邀，為兩湖書院校閱試卷，可見張之洞對陳氏父子的青睞與器重。

陳三立與梁啟超能夠產生交集，就是在協助父親陳寶箴推行新政上，陳寶箴出任湖南巡撫之後不久，陳三立就開始服侍在陳寶箴左右，尤其是在羅致人才、革新教育等方面貢獻頗多，當時陳三立與譚嗣同、徐仁鑄、陶菊存等四人被稱為「維新四公子」。

梁啟超到「時務學堂」擔任中文總教習時，與陳寶箴和陳三立父子多有交往，共同為推動百日維新而吶喊助威，由於「戊戌變法」觸動了守舊派的既得利益，以慈禧太后為首的守舊派，於當年9月21日發動了「戊戌政變」，將光緒皇帝囚禁，梁啟超遭通緝被迫逃亡日本，譚嗣同等「戊戌六君子」在京城菜市口被砍頭，而陳三立與其父親陳寶箴被一併革職。

梁啟超

1912年10月，梁啟超從日本回到北京，翌年7月在熊希齡內閣出任司法總長，時隔15年再次與熊希齡共事，此後長期在天津、北京定居，直至1929年1月19日在北京協和醫院病逝，享年56歲。

陳三立曾有機會重返北京任職，袁世凱曾於1907年遊說陳三立進京出任參政議員，但陳三立從政之心已死，不肯再出山為清政府服務，自「戊戌變法」被革職之後，他一直過著閑雲野鶴一般的生活，寫詩和辦教育成為他生活的主旋律，並於1934年寓居北平姚家衚衕3號，與梁啟超錯過了在北平再聚首的機會。

陳三立全家郊遊時合影

3.梁啟超與陳寅恪的交往

梁啟超比陳寅恪年長17歲，當年在「時務學堂」任中文總教習時，陳寅恪還只是個8歲的兒童，梁啟超與之也僅是熟悉而已，不可能有太深的交往，但印象應該還是有的。

凡讀書之人對陳寅恪的人生經歷大都略知一二，陳寅恪是於1925年回國的，此前一直在歐美遊學，之所以稱其遊學而不是留學，完全是因為陳寅恪不以攻取學位為目的，這也是陳寅恪不同於常人之處，當時清華學校校長曹雲祥正在籌備國學研究院和大學部，委託哈佛大學畢業的吳宓代為尋找稱職的國學導師。

陳寅恪

章太炎、王國維、梁啟超等三人首先被納入清華人的視野，除了章太炎之外，王國維和梁啟超都接受了曹雲祥的聘書，梁啟超還向曹雲祥舉薦了陳寅恪，但陳寅恪卻拿不出任何大學的博士學位，而曹雲祥也有礙於梁啟超的極力舉薦，才接納和聘用了陳寅恪，自1926年6月開始進入清華園與梁啟超一起共事，梁啟超在各種場合力薦陳寅恪，為陳寅恪站台，經常稱「陳先生的學問勝過我。」

而主持清華國學研究院的吳宓，所起的作用也不容小覷，當年他們一起在哈佛大學留學時，曾與湯用彤一起被稱為「哈佛三傑」，這次聚首清華園，再次與吳宓一起共事，自然也是一件樂事，陳寅恪是國學研究院四大導師中最後一個到位清華園的，而著名語言學家趙元任比陳寅恪早到清華園，這樣清華國學研究院的四大導師皆已全部到齊。

左起：趙元任、梁啟超、王國維、陳寅恪、吳宓

吳宓推薦剛從國立東南大學畢業的浦江清擔任陳寅恪的助教，而陳寅恪所指導的學科主要有以下五個方面：1.年曆學；2.古代碑誌與外族有關係者之比較研究；3.摩尼教經典與回紇文譯本之研究；4.佛教經典各種文字譯本之比較研究；5.蒙古滿洲之書籍及碑誌與歷史有關係者之研究。

當時清華的教授和學生早就耳聞了陳寅恪其人學問之大，等到陳寅恪開始授課之時，學生們還是被其博學驚呆了，甚至很多教授也都爭先恐後地前去聽課，並以此為幸事，成為當時清華園的一景，因此陳寅恪也素有「教授中的教授」之稱譽，傳說吳宓、朱自清、馮友蘭等教授就經常去課堂上旁聽，可見陳寅恪學問和授課的魅力了。

陳寅恪

儘管清華國學研究院鼎盛一時，但隨著王國維的投湖自盡和梁啟超因醫療事故病逝，清華國學研究院不得不停辦，而陳寅恪、趙元任、李濟等大腕，盡數被傅斯年延聘到國立中央研究院歷史語言研究所擔任專任研究員，再創學術上的輝煌。

這也再次印證了大師的重要性，無論是大學還是科研院所，都完全取決於教授和研究員的學術水平和學術聲望，同時也印證了清華大學老校長梅貽琦的名言：「所謂大學者，非有大樓之謂也，有大師之謂也。」
