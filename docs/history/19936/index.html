清朝一位漂亮格格，年過50歲還想嫁給溥儀！溥儀卻瞧不上她

2019-12-17 961條評論 2,104次閱讀 308人點贊

清朝一位漂亮格格，年過50歲還想嫁給溥儀！溥儀卻瞧不上她

按照清朝的傳統，很多格格或者公主們都要背負政治使命去跟各部落聯姻，但是到了清朝末年，這些鼎盛時期延續的傳統也分崩瓦解，不用去聯姻了，她們有了自己的想法。那時有一位格格生得格外標緻，她是婉容的表妹，家世顯赫，本來可以許一戶好人家，安穩一生，可她卻愛上了一個不該愛的人。

大家都知道，很多親王的格格、貝勒經常在宮裡走動，這位格格跑去宮裡找表姐婉容玩耍時，遇上了一位帥氣的男子，從此對他念念不忘，可惜妾有情郎無意……男子竟對她的美貌毫無波瀾，一場錯伏讓這個女子一生悲劇，這個女子就是多羅格格王敏彤，婉容皇后的表妹，而她心儀的男子便是當時的皇帝溥儀。

到了溥儀這一代，也是大清要滅亡的一代，清朝滅亡後，王敏彤在深宅大院之中，雖也遇過好姻緣，但硬生生地錯過了。解放後，也沒有結婚。她的一生算是很孤獨。再看看這位她鍾情了一生的男人，他當過皇帝，坐過勞獄，可謂起起伏伏。他的皇后被拋棄，年僅四十便離開人世，另一位妻子與他離婚，他自己在勞獄中呆了數年，直至1959年才見到高牆外的世界。

在出獄後，溥儀也是引起了一陣風波，他想組建家庭的消息被放出後，有很多人想要將自己的女兒嫁給他，雖然已經不是皇帝，但是很多人還是覺得他還在廟堂之上，但是溥儀卻都拒絕了這些好意。王敏彤此時已經50多歲，相貌與數年前相比，自然有很大不同，但她還愛著這位男人，多次請求自己的母親邀請溥儀到家裡做客，溥儀飯吃得十分順口，但唯獨對這個女子並沒有什麼感覺。

王敏彤看著這位心上人，卻始終不能如願在一起，於是到醫院裡開了證明，證明自己是清白之身，這讓溥儀苦笑不得，這會讓溥儀愛上她嗎？答案自然是不能，甚至是反感！要不然，溥儀也不會和一位護士結婚。溥儀結婚了，王敏彤只能大哭一場……

那麼為何溥儀當時那麼瞧不上她？其實這跟整個時代背景也有關係！那時宣揚自由平等的思想，而王敏彤作為婉容的表妹，若與她結合，在倫理道德上是越界，也會被外界誤認為溥儀還企圖復辟封建制度，一個清朝格格，一位大清末代皇帝，若是成親，必定佔據新聞的頭條版面，所以說溥儀不會喜歡，也不想喜歡她！

古人常說情深不壽，可王敏彤可以算是一位長壽的人，活到了2003年，但她自始至終都是孤身一人，去世的時候也沒有親屬送她。細看她的一生，就是因為愛上了一個不該愛的男人，其實如果王敏彤能擁有一個比較好的人生，有丈夫、孩子陪伴在側，去世的時候，也不至於沒有一個人送別。

大家都聽說過文綉與溥儀離婚，但婉容被溥儀逼瘋的但這段奇聞很少有人知道，而且溥儀本身身體就不好，又加上遭受了那麼多年的牢獄之災，身體只會雪上加霜。王敏彤跟著溥儀也不會有什麼很好的人生，倒不如選一個自己中意的男人嫁了，這樣也能享受天倫之樂。

王敏彤的這一生，是悲催的，在錯誤的時間遇到了錯誤的人，一生就這樣浪費了。也許，她對溥儀是真愛，能夠為溥儀奉上她的所有，可溥儀對她始終沒有感覺，最能記住的，也就是她家裡的那幾餐飯了。
