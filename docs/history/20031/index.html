秦瓊陪李世民秘密出宮辦事，見一民間奇人使出絕技，背後直冒冷汗

2019-12-16 224條評論 3,935次閱讀 2237人點贊

在唐朝的民間有這樣一位奇人，本是一名剃頭匠，他的命運卻跟李世民聯繫在了一起，在唐史上很少有他的影子出現，但他卻是玄武門之變的最關鍵的因素。

事情的緣由還得從李世民一次秘密出宮說起，大家都知道當時李世民還是秦王，李淵要傳位子給自己兒子，那也是大兒子李建成為先，雖然李世民有經天緯地之才，但也無法破壞祖制，因此原本皇位是跟他無關的，但是他這一次出宮卻發生了一件讓他命運逆轉的事情。

這件事情有點有趣，當時李世民還是一個長滿青春痘的少年，不過鬍子到不少長，有一次他聽說長安城裡面出了一個非常有名的剃頭匠。於是他就秘密帶著保鏢秦瓊，一起出了皇宮來到了這位剃頭匠店面，這位剃頭匠名叫楊三城，他的手藝是祖輩傳下來的，剃頭手藝那是出神入化。李世民走到店面前就被楊三城看到了。

這天楊三城，剛好忙完，只見門口進來兩個人人，一人年輕，20幾歲卻滿臉都是青春痘，上下巴長著密密麻麻的鬍子，另外一個，30多歲九尺有餘，一臉絡腮鬍子的彪形大漢。李世民進來以後，一屁股坐在椅子上說道：「聽說楊師傅的手藝全長安城天下第一」，不知道，我這張臉可以剃否？

要是碰到一般的剃頭匠，那誰敢接這單生意，滿臉的青春痘怎麼下手對吧，但是楊師傅不是一般人呢，既然可以被稱為長安城裡有名的師傅，那肯定有，他的絕技。張師傅先取來一塊熱毛巾敷在了李世民的臉上，沒過一會兒，取下毛巾，說是遲那時快，提刀就在李世民的臉上龍飛鳳舞起來，搞得站在旁邊的秦瓊一臉驚愕，生怕此人，有歹意謀殺了秦王，這要回去，那是死罪一條。站在李世民旁邊臉一下紅一下白又一黑。

而此時的李世民正享受著快感，誰承想，刮個鬍子，還能如此般享受，很快鬍子就刮完了，李世民看著自己的臉，一顆青春痘都沒破，而鬍子一根不剩，大讚道：「楊師傅的手藝真是太絕了」，隨後拿出10銀子正要付錢，卻被楊三城拒絕了，說道：「普通老百姓，剃個頭只需要幾文錢」，可是對於天子之家，10銀子太少。

李世民很驚訝，說道：「楊師傅不但手藝超絕，眼光也是毒辣」，沒錯我就是秦王李世民，其實這時候的楊師傅早就猜到此，二人身份。李世民本想讓其進宮，為其服務，但卻被楊師傅拒絕了，說：「我給人理髮也不是為了賺錢，只是這樣的生活，比較自由，有可以幫助人，而宮裡的生活跟關在籠子里一樣，不適合我」，李世民見楊師傅，心意已決也不好強迫他做此事，於是兩人商定，凡是我若有需要，那就請楊師傅跑工地一趟。

就這樣他們達成了秘密的協議，此後的一段時間，楊師傅經常被李世民請到宮中幫他，刮鬍子。直到有一天，一群人來到了楊師傅的店面，說齊王在此，還不快快下跪，原來李元吉，聽到李世民經常秘密請楊師傅幫他去刮鬍子，並且對楊師傅說：「你知道當今太子是我大哥，李建成，但是我二哥李世民對這個位置虎視眈眈，手握重兵又，如此，勢必要天下大亂」，楊三城道：「我只是一平民百姓，做不了什麼」，李元吉隨後掏出一包毒藥,說道：「這是一包毒藥，見血封侯，到時候你再給，我二哥刮鬍子的時候，只要輕輕的劃破皮膚，毒藥滲進他的皮膚，不出一小時他就斃命」，楊師傅隨即跪倒在地哭求道，我只是平民百姓，這是要誅九族的事情，而李元吉笑罵道：「你要是不照做，我現在就誅你九族」，並且還抓了他的，四個兒子和媳婦，無奈之下，楊師傅只能乖乖照做。

第二天李世民又要請楊師傅進宮去幫他刮鬍子，這一天楊師傅心事重重，因此跟往常不一樣，刮鬍子的時候，拿手一抖，把李世民的青春痘給弄破了，秦瓊在旁邊大罵道，你是想謀害秦王嗎？李世民搖搖手說道：「我看今天秦師傅臉色有點差，是不是生病了，我請宮裡的太醫幫楊師傅看一下」，相信我知道李世民深明大義，對自己如同己出，如果讓李世民坐上了皇帝，那肯定是天下百姓之福，隨後跪倒在地哭訴道：「把李元吉威脅他，讓他來刺殺李世民的事情，一五一十的，說的清清楚楚」，李世民聽後大驚。

隨後秦瓊便說道：「公子，不如先下手為強，既然太子和齊王都想謀害公子秦王您」，李世民思量後說道，明天我請父王當面對質，第二天帶著楊三城來到了李淵前，把李元吉的陰謀詭計都說的清清楚楚，李淵大怒要想把楊三城給殺了，理由是挑撥兄弟之間的關係，最後被李世民攔下，說道：「父王不如明天請元吉和大哥當面對質」，李淵知道李世民的秉性，隨即就答應了，第二天，李元吉李建成走到玄武門之時，被李世民布下的人，射殺。也就是有名的玄武門之變。

之後李淵退位，李世民登基成就了一番帝業，而楊三城，雖在此局中幫助了李世民成功奪得了皇位，但他的家人全都被殺了，他心灰意冷之下，入了大白馬寺坐著一位和尚，李世民因其護國有功，賞他二兩塊匾額，一塊是護國禪師，一塊是天下第一剃頭匠。
