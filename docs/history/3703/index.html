明朝的朱元璋，都是怎麼治理貪污腐敗的？

2019-12-01 0條評論 19次閱讀 0人點贊

摘要

腐敗現象伴隨著人類社會的發展如影相隨。《七同考· 魏職官注》：“六圍已遣御史監郡。”說明春秋戰國時期

腐敗現象伴隨著人類社會的發展如影相隨。

《七同考· 魏職官注》：「六圍已遣御史監郡。」

說明春秋戰國時期諸侯各國就已經開始派遣 「御史」 監督地方官吏。朱元璋當上皇帝之後， 同樣也碰上了反腐敗這樣的歷史性難題。一些開同元勛不但倚老賣老貪污受賄，甚至公開奢靡腐敗，這讓朱元璋非常不爽。

為此，他使出渾身解數， 重拳出擊。

窮苦出身的朱元璋深知一粒一粟來之不易，於是帶頭節儉過緊日子。開國不久，大臣們建議裝飾一下皇宮，他不但拒絕，而且還在宮牆畫上節儉的圖畫用以警醒。對於不差錢的皇帝來說，其車輿、器具等按歷朝成制應該用黃金裝飾，可朱元璋卻下令以銅代替，還當著大臣們的面說，我如果用金，你們可能會用銀，再下面的官員就會用銅，全國有多少官吏，那要花多少錢啊？

他在減少身邊嬪妃數量的同時，還經常帶領工作人員吃野菜和窩窩頭，以此告誠身邊的人要體恤百姓之苦。馬皇后不但帶頭穿破衣服，而且縫縫補補再穿。因而，明初宮裡勤儉節約蔚然成風。朱玩璋深知，僅僅靠以身作則反腐敗是不夠的，應該用制度來制約官吏貪污受賄。 他規定，在京三品以下可以坐轎，四品以下和在外官員可以使用馬車，七品以下的官員只能騎驢。 有位元朝過渡的高官以為朱元璋是開玩笑， 離京辦案偷偷坐了回轎子，結果丟了腦袋 。 明洪武十八年 （公元 1385 年）． 戶部侍郎郭恆等人通過塗改財會憑證上的數字 「一二三四五六七八九十百千」 的手段，大肆侵吞、貪污國家錢糧，案子牽涉朝廷部委辦局和地方大小官員、豪紳數萬人，最後全部斬首示眾。

此案給朱元璋以啟示，光是殺人不能從根本上解決問題，要從制度上反腐敗。尤其是在財務管理方面採取一系列行之有效的措施， 其中最重要的做法是將記載錢糧的數字一律改為漢字大寫； 「壹貳叄肆伍陸柒捌玖拾佰仟」，這一創舉，直到今天還在財務往來中普遍使用 。

幾年後，朱元璋發現要想實現國家的長治久安，單靠以身作則和制度建設還小夠，需要重典治吏，依法治國。 於是，他在總結前人經驗教訓的基礎上制定和頒布實施 《大明律》， 這是一部比歷朝歷代都要廣泛而嚴酷得多的法律，官吏如果貪污60兩銀子，就要處以極刑—— 「 剝皮揎草」 ，被 「 剝皮揎草」 者，先由行刑的人用刀子從其後脖頸開刀，順著脊背割一條縫，然後把皮膚向兩側撕開，皮膚被剝下後，再填上草，縫起來，這個 「皮草人」 就放在衙門裡的官座旁，以此警示。

60 兩銀子是個多人數日呢？ 相當於其時縣長的一年半工資。可見，這種懲罰是嚴重的。為了使法律意識深入人心，朱元璋還把其巾親手編撰針對全體臣民的 《大誥》發布天下， 每戶一冊。並在宮裡舉辦講座親自給大臣講解其精神實質，檢查學法情況。將《大誥》列為高考的必考課日。 在京城舉辦各地活學活用 《大誥》演講比賽，評比發獎，特別優秀的還提拔當官。 如果犯了罪，要是抄出家藏 《大誥》 一冊，可以從輕處罰；反之，罪加一等．
