從北宋到明朝末年，古代的火器一直領先於世界

2019-12-12 344條評論 3,672次閱讀 1390人點贊

說到古代軍隊的武器裝備，我們往往會想到刀、劍、弓箭等冷兵器。其實早在宋朝時，就已經有了類似於槍炮的火藥武器。火藥是我國古代的四大發明之一，起源於古代的煉丹術。古代的帝王追求長生不老之葯，在煉製這種丹藥時，雖然沒有成功。但是卻煉出了後來的火藥。

火藥發明以後，逐漸被用于軍事。根據宋朝的《九國志》記載：大約公元十世紀的唐哀帝時，鄭王番率軍攻打豫章，「發機飛火」，燒毀該城的龍沙門。這很有可能是火藥用于軍事作戰的最早記載。

宋朝時火藥的發展與火器裝備

宋朝時期，火藥的發展非常迅速，開始應用到實戰武器中。由於宋朝的戰爭持續不斷，軍隊對火器的需求越來越大。火器也越來越受到重視。宋神宗時，專門成立了管理各種兵器製造的軍器監。軍器監下轄十大兵工廠，其中的兩個，一個專門生產火藥，一個生產火藥武器。

1、古代的「火箭」是世界上最早的噴射武器

根據《宋史》中的記載。當年宋太祖時的兵部令史馮繼升發明了「火箭」，把火藥筒綁在箭桿上。使用時，點燃箭桿上的火藥筒，火藥在筒里燃燒時，會產生大量的氣體高速的向後噴射，然後利用產生向前的推力把箭射出去。

這是世界上最早的噴射火器，現代的運載火箭就淵源於古代的火箭。這個時期的火器主要是利用火藥的燃燒功能。隨著火藥武器的發展，漸漸的開始利用火藥的爆炸功能。比如北宋年間爆炸威力比較大的震天雷等。

2、突火槍、震天雷、毒藥煙球

  • 突火槍是第一種能發射子彈的步槍

根據《宋史》記載：開慶元年「又造突火槍，以巨竹為筒，內安子窠，如燒放焰絕，然後子窠發出如炮聲，遠聞百五十餘步。

公元1259年，也就是南宋理宗開慶元年。聰明的宋朝人發明了突火槍。這可是世界上第一種能夠發射子彈的步槍。是現代槍炮武器當之無愧的老祖宗。它採用粗竹筒作為槍的身軀。在竹筒的中間設置了一個火藥室，裡面裝有火藥和子窠 ，外壁的上面有一個點火小孔。後面是方便手持的木棍。

發射的時候，把木棍拄地。左手扶住槍管，右手點火，然後就會發出一聲巨響，射出石塊或者彈丸。由於當時的火藥推力很有限，射程只有不到一百米，加上射擊的技巧太過於僵硬，所以只能對敵起到威懾的作用。

  • 能炸穿鐵甲的震天雷

根據《金史》的記載，在金朝天興元年，赤盞合喜守汴京時：「其守城之具有火炮名『震天雷』者，鐵罐盛葯，以火點之，炮起火發，其聲如雷，聞百里外，所爇圍半畝之上，火點著甲鐵皆透」。

除了突火槍，還有一種類似於現代手雷的火藥武器，名叫震天雷。它的外表採用生鐵包裹，裡面裝的是火藥，安裝有引信。使用的時侯根據目標的遠近來決定引線的長短。

震天雷爆炸後不但能夠將生鐵包裹的外殼炸成碎片，還能擊穿鐵甲。可見其威力。它的使用方法主要有兩種。一種使用火點，就地爆炸。另一種使用投機石，拋向遠處爆炸。

  • 毒藥煙球是一種能產生毒氣的炮彈

《武經總要》中記載了宋朝軍隊裝備的火藥武器。其中有一種叫做毒藥煙球的炮彈。它是用紙和麻皮包裹而成的圓球形狀的炮彈。外邊上塗有瀝青和黃蠟，當這種炮彈落入敵方陣地時，會發生轟然的巨響，隨後一道火光騰空而起，噴發出一股嗆人的煙霧。

這也許就是最早的煙霧彈或者毒氣彈了吧。因為這種炮彈爆炸後噴發出來的煙霧是有毒的，所以只要聞到這煙霧的敵人，就會口鼻流血，暈倒在地。能夠使敵人的軍隊產生混亂。

新式火器火銃

1、金屬管狀火銃的出現

隨著火藥的性能逐漸的提高。火藥武器也有了新的發展。到了元朝時，出現了一種新型的火器裝備，這種新型的火器叫做火銃。與突火槍不同，火銃採用的是銅鐵鑄造而成的金屬管狀火器。它的殺傷威力要比之前的兵器大很多。

元代火銃是在元朝建立後，獲得了金和南宋關於火器的製造技術，然後把各地的工匠集中在元大都研製出來的新式火器。由於火銃的殺傷威力大，使用壽命長，還能反覆的裝填發射，所以成為了軍隊的重要武器裝備。

2、明朝開始製造大量的火銃

明朝的開國皇帝朱元璋在統一天下的戰爭時，運用了火銃作戰。通過實戰發現，它不僅可以在陸地上使用，也可以運用於水戰。由於對火銃有了新的認識，明朝開始製造大量的火銃。在設計方面也有了新的改進。

從出土的洪武年間的火銃來看，當時的火銃分為手銃和碗口銃兩類，手銃是單兵使用的輕型火器。比如河北省赤城縣發現的洪武五年時的火銃，長44.2厘米，口內徑2.2厘米，外徑2厘米。銃身刻有銘文「驍騎右衛，勝字肆佰壹號長銃，簡重貳斤拾貳兩。洪武五年八月吉日寶源造。」

碗口銃則是口徑和體積較大的火銃，比如現藏在軍事博物館的一件洪武五年鑄造的火銃。全長三十六點五厘米，口徑十一厘米，重十五點七五千克，銃身銘文「水軍左衛，進字四十二號，大碗口筒，重二十六斤，洪武五年十二月吉日，寶源局造。由於口徑大，可以裝填更多的火藥。所以它的威力也是非常大的。

明朝的火器大規模裝備部隊

明朝的軍隊普便都裝備有火銃，根椐歷史資料的記載，洪武十三年規定，在各地衛所駐軍中，按編製總數的百分之十裝備火銃。二十六年規定，在水軍每艘海運船上裝備碗口銃四門、火槍二十支、火攻箭和神機箭二十支。可見火器在明朝時已經大規模的裝備了部隊。

1、神機營是世界上最早的槍炮部隊

永樂八年，明成祖朱棣得神機槍炮法，成立了專門掌管火器的特殊部隊「神機營」，在親征的漠北之戰中，提出了「神機銃居前，馬隊居後」的作戰原則。神機營配合步兵、騎兵作戰，發揮了重要作用。這支精銳中的精銳，裝備著當時世界上最先進的火器。

神機營作為明朝軍隊的重要軍事力量，擔負著「內衛京師，外備征戰」的重要任務。它是京城的三大禁衛軍之一，是由皇帝親自指揮的機動作戰部隊。裝備有火槍、火銃等先進的火器裝備，後期又添置了火繩槍。是當時世界上最專業的槍炮部隊。比西班牙的火槍兵還要早一百年左右。

2、神機營的編製

神機營的最高編製為營。營編提督內臣2人、武官2人、掌號頭官2人；營下編中軍、左掖、右掖、左哨、右哨五軍，各設坐營內臣1人、武臣1人，除中軍下領四司外，其餘各領三司；每司設監槍內臣1人、把司官1人、把牌官2人。營專習神槍、神炮。

根據《皇明經世文編》的記載。神機營全營共5000人，其中步兵3600人，騎兵1000人，炮兵400人。步兵全部裝備有火銃，合用藥有9000斤。炮兵裝備有大連珠炮和野戰重炮，而且每個炮兵還裝備有防身用的手銃。可見神機營真是裝備精良。

結語

火器的發展在明朝達到鼎盛時期。明末的兵書《兵錄》、《神器譜》、《武備志》、《西法神機》、《火攻摯要》、《籌海圖編》、《軍器圖說》、《火龍神器陣法》等詳細地記錄了明軍裝備火器的製法、圖譜以及火器部隊的編製和戰法。非常重視火器的將領有戚繼光、孫承宗等。可見明朝對火器的使用相當的成熟。

同時標誌著其威力和性能已經發展到較高的水平。但是由於裝填費力，發射速度較慢，射擊的精準度不高等缺點。只能取代部份的冷兵器。大部分的軍隊還是以刀、劍等冷兵器為主。從北宋到明朝末年，中國一直引領著世界火器的發展潮流。直到擅長騎射的清王朝統治中國，為了維護戰馬和弓箭的地位，把火器打入了冷宮。使得中國原本領先世界的火器水平停滯不前，與西方國家的距離逐漸拉大。才引發了後來一系列的悲劇。
