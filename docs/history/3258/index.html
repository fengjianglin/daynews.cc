絕妙諷刺！她為兒子稱帝用盡毒計，卻因一言不慎讓兒子動殺心

2019-12-07 0條評論 19次閱讀 0人點贊

摘要

羅馬帝國從建立以後，有很多位皇帝，但是除了開國皇帝奧古斯都之外，在大眾之中很有名氣的皇帝卻不多，甚至連羅馬帝國鼎盛時期的“羅馬五賢帝”都沒有太多人知道。不過，有一位皇帝，雖然他不算公認的好皇帝，但是，很多人一提到“暴君”一詞，都會想到他，這個皇帝當然就是尼祿了。

羅馬帝國從建立以後，有很多位皇帝，但是除了開國皇帝奧古斯都之外，在大眾之中很有名氣的皇帝卻不多，甚至連羅馬帝國鼎盛時期的「羅馬五賢帝」都沒有太多人知道。不過，有一位皇帝，雖然他不算公認的好皇帝，但是，很多人一提到「暴君」一詞，都會想到他，這個皇帝當然就是尼祿了。

尼祿應該算得上是羅馬皇帝中相當年輕的一位，因此，他的登基，基本跟他自己的努力沒太大關係，主要功勞，應該歸功於他有個非常厲害的媽媽-小阿格里皮娜。

小阿格里皮娜的雕像

小阿格里皮娜是羅馬帝國開國皇帝奧古斯都、也就是屋大維的重外孫女，還是羅馬帝國初期頂級名將日耳曼尼庫斯的親生女兒，身份自然是屬於當時羅馬帝國頂級貴族，不過，尼祿登基時期的羅馬皇帝，卻不是奧古斯都的尤利烏斯家族的人。長話短說，這裡簡單說一下，小阿格里皮娜的哥哥卡利古拉是前任皇帝，但是他非常鋪張浪費，而且行事多半由著自己性子，把帝國上下搞得一團糟，國家財政基本被他玩崩潰了，因此最終在公元41年被人所刺殺，年僅28歲，隨後，被擁立為皇帝的人，則是卡利古拉的叔叔克勞狄烏斯。

雖然是前皇帝的叔叔，但是克勞狄烏斯卻不是尤利烏斯家族的，而是克勞狄烏斯家族的人，而卡利古拉則隨同他母親、也就是小阿格里皮娜的母親大阿格里皮娜被納入尤利烏斯家族中，這也是兩個家族之間的一種平衡。

第一位被刺殺的羅馬皇帝卡利古拉

顯然，卡利古拉被刺殺而克勞狄烏斯即位這件事，尤利烏斯家族裡面只要有點權力欲的人，都會感到不滿，畢竟這天下是尤利烏斯家族的凱撒和奧古斯都打下來的，克勞狄烏斯家族不過是搭了順風車而已，而這些人中，小阿格里皮娜最為不滿，從她後來的所作所為來看，這很大程度是因為她權力欲非常大。

克勞狄烏斯統治前7年，小阿格里皮娜儘管內心忿忿不平，但是真沒辦法對付她眼中的「篡位者」，因為克勞狄烏斯執政能力很強，很受人們愛戴，僅僅依靠奧古斯都血脈，小阿格里皮娜手裡根本沒有對付皇帝的手段。但是，就在公元49年時，機會來了，這一年裡，原來的年輕皇后梅薩麗娜，自殺了。

梅薩麗娜的自殺其實非常有趣，她平日里驕橫跋扈，仗著克勞狄烏斯對她的忍讓非常囂張，還給已經近60的皇帝戴綠帽，但就在公元48年，她趁著克勞狄烏斯在外的時候，居然公然在首都和別人舉行婚禮！這可真是千古奇觀，皇后和別人舉行婚禮，這已經不算是皇家醜聞，而成羅馬帝國醜聞了，於是平日里對皇后睜一隻眼閉一隻眼的皇宮眾人也趕緊統治皇帝，結果克勞狄烏斯居然沒有下令當場誅殺皇后，而是隔天再與皇后談談，這可嚇呆了皇宮眾人，萬一克勞狄烏斯老實人性格發作，皇后沒事，那死的就是他們了，於是，這批人乾脆讓近衛隊長假裝奉皇帝命令，跑去囚禁皇后的地方，勒令其自盡，梅薩麗娜就這樣死了。

克勞狄烏斯的雕像，實際上他身患殘疾，外表並不討喜

這件事傳遍羅馬，小阿格里皮娜於是發現了機會，克勞狄烏斯看來是個對於妻子很沒有控制力的男人，與其在政治這個皇帝長處上爭鋒，不如在他的短處上下手，小阿格里皮娜就是這樣打算的。而皇后死了，位置可不能這樣空著，得找，但誰是合適的人選呢？小阿格里皮娜立刻重金收買了克勞狄烏斯的一個親信帕拉斯，他是克勞狄烏斯手下第三號紅人，帕拉斯將小阿格里皮娜列入到了皇后候選人的名單了。

有意思的是，雖然當時皇后位置空缺，但是除了小阿格里皮娜之外，另外兩個人一個是克勞狄烏斯登基前的前妻，另一位是羅馬第一美人，不過她們都沒有小阿格里皮娜的這樣渴望權力的心，而且也缺乏後者的政治才能，因此，在小阿格里皮娜的收買元老院和利用奧古斯都血統鼓動群眾等一系列操作下，她成了克勞狄烏斯的皇后。

說老實話，為了權力，小阿格里皮娜的行為還是挺具備風險精神的，因為克勞狄烏斯不僅年老醜陋，還有腿部殘疾和口吃，並且還是小阿格里皮娜的親叔叔，這一系列的因素加起來，正常的羅馬貴婦根本不可能打算當皇后。克勞狄烏斯多半也清楚小阿格里皮娜是另有所圖，但是，他實在難以想像這個侄女兼現任媳婦，居然有著極度可怕的蛇蠍之心。

尼祿的老師塞內加

當上皇后的小阿格里皮娜，首先弄死了之前的皇后位競爭者，然後開始認真幫助克勞狄烏斯應對政事，期間偷偷加強自己的威望，用自己的名字為城市命名，要知道，這一般是皇帝的特權。另外，她還鞏固與克勞狄烏斯心腹帕拉斯的關係，幫助後者的弟弟成為高官，這一系列操作下來，小阿格里皮娜穩固了皇后的位置。此外，她還利用皇后的權力，赦免了被流放的哲學家塞內加，調來了普通士兵出身的優秀軍人布魯斯，讓兩個人負責親生兒子尼祿的文武兩方面教育。

隨後，尼祿登場了。小阿格里皮娜在當皇后期間，發現了克勞狄烏斯的一樁心事，那就是皇帝位置的傳承。克勞狄烏斯年老了才有兒子，因此他的兒子基本沒積攢什麼功績，即使克勞狄烏斯把征服不列顛的稱號「不列塔尼庫斯」給了兒子，也不過是一個自欺欺人的行為而已，而自己又找不到適合託付兒子的人。於是有一天，小阿格里皮娜索性建議克勞狄烏斯，將他的女兒、不列塔尼庫斯的姐姐屋大維婭嫁給自己的兒子尼祿，尼祿比不列塔尼庫斯大4歲，這樣以後不列塔尼庫斯就能有一個親上加親的姐夫兼義兄，尼祿完全可以作為他的心腹幹將。克勞狄烏斯聽完，覺得很有道理，就同意了，可是他卻沒料到，這一行為確實可能讓自己兒子有一個得力幹將，可是，這也非常方便尼祿奪取皇位。

年輕的尼祿很受元老院歡迎

尼祿被母親安排了娃娃親，等到他到了16歲，按羅馬法律已經成年，這才終於落實了與屋大維婭的婚禮，同時，根據皇帝和皇后的安排，他還了元老議員，進行了人生中第一場演講。尼祿在演講中圍繞四個議題，用了四種語言演講，表現出了極為優秀的個人才能和領袖氣質，最終，他的提案全都得以通過，第一次登場圓滿成功。

在克勞狄烏斯看來，這是兒子心腹的登場秀，但對於皇后小阿格里皮娜看來，這卻是判斷尼祿是否有稱帝基本素質的嘗試，而克勞狄烏斯，可能是年老的原因，對這一極度危險的信號，卻沒有察覺，甚至讓尼祿的老師布魯斯擔任禁衛軍長官。

現在，所有準備基本就位，小阿格里皮娜終於可以放心大膽地對付老皇帝了，可憐克勞狄烏斯與小阿格里皮娜同床異夢5年，期間完全想不到自己的妻子居然一開始就想殺了自己。就在公元54年10月，克勞狄烏斯在晚宴後吃下了小阿格里皮娜準備已久的毒藥，在深夜毒發身亡。得知老皇帝已死，小阿格里皮娜秘不發喪，而是讓尼祿的老師布魯斯帶著1000士兵進入皇宮控制局勢，等到第二天正午，才帶著準備齊全的尼祿直奔城外的禁衛軍軍營，用重金收買了剩下的士兵，等到元老院反應過來時，尼祿已經帶著軍隊進了羅馬城，雖然克勞狄烏斯有讓兒子不列塔尼庫斯繼承皇位的遺囑，但是元老院根本不敢公開，畢竟禁衛軍的刀都快架到脖子上了。

禁衛軍被小阿格里皮娜收買

至於羅馬民眾，他們知道尼祿是克勞狄烏斯的養子兼女婿，根本不認為尼祿登基有什麼問題，反倒認為是合情合理的，就這樣，克勞狄烏斯搬起石頭砸自己的腳。

小阿格里皮娜籌備多年的陰謀得逞了，按理來說，這時候讓兒子去執政，自己當皇太后享清福就可以了，實在不行，那就大事把把關也成，但是前面說過，她權力欲大呀，於是，她就做了很多不利於尼祿皇權的事情。

皇太后在所有政務場合，都和皇帝尼祿在一起，甚至為了方便自己行動，她還專門把元老院的議事場合從元老院會場遷到了皇宮裡。而在羅馬帝國的貨幣上，她也有小心思，一般來說，貨幣上都會有在位皇帝的肖像，但是小阿格里皮娜把自己的肖像也給加了上去，而且，在皇帝發布的命令中，都會加上她的名義。總之，小阿格里皮娜儼然是與皇帝尼祿並列的羅馬「女皇帝」。

對於母親的這些行為，尼祿卻絲毫沒有什麼反對的跡象，這也能理解，在短短數年時間就具備超乎常人的學識和素養，尼祿勢必在母親還是皇后時，就接受了極度辛苦的教育，由此看來，尼祿和母親的關係，應該很像現代的強勢媽媽與弱勢兒子的關係，如果真是現代，恐怕尼祿還會被視為「媽寶男」吧。

有著皇室血統的名將-日爾曼尼庫斯

不過，理性地思考，其實小阿格里皮娜的行為是有利於尼祿日後權力的穩固的，畢竟尼祿此時年紀尚小，又沒什麼功績，很難服眾，而小阿格里皮娜將自己的存在突出，與兒子尼祿的存在結合起來，可以在尼祿建立起足以服眾的威望前起到過渡作用，畢竟小阿格里皮娜是奧古斯都血脈，而她的父親日耳曼尼庫斯又曾是皇位繼承人兼羅馬帝國名將。

但是有一次，尼祿開始反抗母親了，起因是一名被贖身的奴隸。尼祿登基後，年紀輕輕情竇初開，喜歡上了奴隸阿可特，但他知道母親肯定不同意他和阿可特交往，畢竟堂堂帝國皇帝與奴隸有感情，這簡直是政治災難，於是他選擇拜託老師塞內加。塞內加最終耐不住尼祿央求，想了個辦法，讓阿可特偽裝成宮廷近衛隊長的妹妹，然後常常和尼祿私會。

小阿格里皮娜縱橫皇宮多年，這些小伎倆逃不過她的法眼，很快她就發現兒子乾的蠢事，不過她的行為倒也挺像現在的虎媽，她沒有曉之以理，而是用盡惡毒的辭彙對兒子破口大罵。尼祿當時是老實了，可在事後，他開始怨恨自己的母親，特別是身為皇帝的驕傲刺激著他，終於，他開始剪除母親的「羽翼」了。

皇帝尼祿削弱母親的勢力

塞內加和布魯斯兩位皇帝老師對於尼祿的行為十分興奮，他們有才能，也有實現理想的熱誠，但是小阿格里皮娜掌權一天，他們就無法利用皇帝老師的身份干大事。有了機會的塞內加找到此時皇太后的心腹帕拉斯，勸告後者趕緊離開羅馬這個是非之地，帕拉斯畢竟是先帝克勞狄烏斯的心腹，政治嗅覺很敏銳，在塞內加的暗示下，他知道羅馬即將迎來政治風波，母親和兒子爭鋒，兩人不會有性命之爭，但作為皇太后心腹的他卻可能斃命，因此同意了回家鄉做富家翁，小阿格里皮娜的頭號心腹就這樣沒了。

另外，尼祿又強硬地將母親轉移出皇宮居住，同時憑藉塞內加和布魯斯的手段，成功將母親從各個社交場合的邀請名單上抹去，這樣過了1年時間，人們就開始淡忘皇太后小阿格里皮娜了。面對這樣的情況，居然失去了理智，估計是只在意尼祿是自己兒子的身份了吧，她居然趕到皇宮，對尼祿威脅，如果不是她的努力，尼祿根本當不上皇帝，只要她願意，她隨時可以利用自己的影響力讓不列塔尼庫斯成為皇帝，畢竟他才是先帝克勞狄烏斯選定的繼承人。

小阿格里皮娜的愚蠢在於，因為憤怒，她忘了最高統治者一旦下台，多半會面臨死亡的結局，同時也忘了皇帝尼祿同樣有除掉她的能力，這一番話非但沒有讓尼祿轉變態度，反而讓他發現了母親對自己的威脅之大，已經有除掉母親的必要了，此時此刻，尼祿的對母親的感情破裂了，作為回應，就在小阿格里皮娜說出這番話後不久，不列塔尼庫斯蹊蹺地暴病而亡，尼祿這是在斷母親的退路。

小阿格里皮娜收買了遠在邊境的萊茵河軍團

然而，這之後皇太后與皇帝卻維持了一個微妙的關係，小阿格里皮娜再次重金收買了遠在邊境的萊茵河軍團，同時似乎也接受了目前的處境，與兒子尼祿保持平衡的關係。但是惡語已出，既然於情於理不可能動手除掉自己親兒子，那麼當初為什麼又要威脅兒子呢？或許是小阿格里皮娜強勢慣了，認為尼祿再怎麼大膽，也不會對母親下毒手，另外，從政治身份來說，小阿格里皮娜的身份是尼祿稱帝的道義資源，一個成熟的政治家是斷無可能殺死這樣的母親的。

可悲可嘆的是，尼祿或許看起來聰慧過人，但此時卻缺乏政治家的素養，他再度因為感情與母親爆發激烈的衝突。繼阿可特之後，尼祿喜歡上了一個叫做薩賓娜的美女，還打算立她為皇后，因此準備和先皇克勞狄烏斯的女兒屋大維婭離婚，此時，有一段時間沉寂的小阿格里皮娜卻態度強硬地反對尼祿的決定。從尼祿的視角看來，這是母親再次粗暴干涉自己，是皇太后對皇帝權威的挑釁。不過，排除這一方面的問題，小阿格里皮娜的行為確實很有道理的，屋大維婭是尼祿繼承皇位的法統所在，要保留繼承克勞狄烏斯皇位的正當性，無論如何也不應與屋大維婭離婚，但似乎尼祿看不到、或者說不想理會這一點。

後世描繪小阿格里皮娜乘船沉沒的畫作

尼祿對母親的怨恨已經到了可怕的地步，他有了一個大逆不道的想法，就是一勞永逸地解決母親，為此，他連老師塞內加和布魯斯都沒有告知，而是和宮廷里的其他心腹商量。公元59年3月，尼祿假意與母親和好，小阿格里皮娜不假思索地相信了兒子，畢竟血濃於水嘛。尼祿邀請母親在3月20日參加自己籌辦的皇家宴會，由於小阿格里皮娜這段時間居住在海邊的豪宅之中，因此出行大多乘船，而尼祿著手的地方正是船隻，他特意命令能工巧匠偷偷打造一艘有問題的船，保證能在母親返回時沉海。

於是，就在盛宴結束後，尼祿將許久沒有享受到天倫之樂的小阿格里皮娜送上了這艘船，臨行前還特意熱情擁抱了自己的母親。事畢之後，尼祿回到皇宮沒有休息，而是徹夜未眠，緊張地等待著母親已死的消息，但是，他最後接到的卻是母親的來信，小阿格里皮娜居然還活著！

尼祿這個人和人世間很多兒子女兒一樣，對自己的父母缺乏了解，事實上，想讓母親溺死完全是一個可笑的做法，因為尼祿不知道，母親早年曾因皇室糾紛被流放到小島上居住過，早就練成了一身卓越的泳技，船沉以後，居然游到了途徑的漁船邊，命令漁夫將自己送回了豪宅。

小阿格里皮娜想不到的結局

事到如今，尼祿也不知道該怎麼辦了？只好把兩位老師請來，一五一十交代自己的愚蠢行為，塞內加和布魯斯聽完大驚失色，這麼愚蠢的事情，皇帝怎麼能幹呢？但是事到如今，母親和兒子已經是生死相爭，不如索性惡事干到底。於是，一批全副武裝的衛士趕到皇太后的豪宅，最終結束了小阿格里皮娜的性命。

就我個人而言，我認為，小阿格里皮娜最終還是母性壓倒了政治家的性格，兒子殺自己的企圖已經暴露無遺，僥倖生還後，不管是逃出羅馬還是立刻安排政變，都比給兒子送信好，這或許是因為她還對兒子尼祿有所希冀，認為母子能夠保證最後的和平。當然，也有可能是她根本沒有力量出逃或政變，但以她的能力，如果真沒有這種力量，那多半是因為過去沒有安排。但是不管是哪種情況，至少有一點我們可以知道，那就是小阿格里皮娜當時不敢也不願意相信兒子尼祿會要殺自己。

據說尼祿日後常噩夢纏繞

小阿格里皮娜死了，尼祿的陰謀實現了，他可以立自己喜歡的皇后了，但尼祿勝利了嗎？我認為尼祿沒有勝利，反而失敗了，而且敗得很徹底。從政治角度考慮，他的母親小阿格里皮娜是他最大的政治資本，也是他統治帝國的一大助力，無論如何，小阿格里皮娜都不應該死。而從親情角度來看，尼祿個人其實也有悔恨，小阿格里皮娜雖然權力欲極大，但是在尼祿之父早逝後，她也是一個人撫養尼祿長大，並且最終通過自己的努力讓兒子成了最有權勢的人。據說尼祿後來常常噩夢纏身，每晚都不能安穩入眠。

小阿格里皮娜的死去，以及尼祿與屋大維婭婚姻的破滅，導致尼祿的統治喪失了道義上的合法性，此後，他的統治只能仰賴於羅馬帝國人民本身的支持，而得不到尤利烏斯和克勞狄烏斯兩大家族的鼎力相助，同時也喪失了對於日爾曼尼庫斯血統狂熱推崇的萊茵河軍團的支持，因此，當尼祿統治終結時，居然沒有一支軍隊挺身而出來保護尼祿，如果是小阿格里皮娜還在，情況或許不會這麼令人絕望，不知道尼祿自盡前，有沒有想到這一點呢？
