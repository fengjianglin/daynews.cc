曾國藩、左宗棠與李鴻章，更有才，誰功績最大

天天要聞
2019-12-21

曾左李張四人論學問才華肯定是曾國藩、張之洞，論貢獻非左宗棠莫屬。

（1）許多人都很看重「虛」名，而忘記實際貢獻

比如四人的謚號、官職，曾國藩是「文正」、左宗棠和張之洞是「文襄」、李鴻章是「文忠」，並且說，這代表了對「貢獻」的認定。哪那麼簡單，而且「一個沒落無能的政府」的評價有什麼用。真能代表貢獻嗎？這裡面更多的是「死後」其生前的貢獻，與皇帝的關係、與同僚的關係，下屬們是否掌權等等「活著的人」有關。

比如有人說李鴻章蓋了中國的第一條鐵路、創建了第一支近代海軍、創建了第一家現代化銀行等等，這些有什麼用嗎？

大圖模式

第一隻是說你在時間上佔了便宜，正如近現代許多科技成果我們都知道一些案例，比如因為某人早申請了專利結果這項發明就成他的了，但實際上有人比他之前就研製出了該科研的成果。「第一」什麼也說明不了。你得發揚光大，而且志同道合者甚寡，這時候的「第一」才有價值。

再比如「第一支近代海軍」說的是北洋海軍，但北洋海軍如何了？成了廢銅爛鐵，對當時的戰爭、對後來保家衛國起到了什麼作用？

再比如，輪船招商局和一大批煤礦。李鴻章所謂的「第一」名副其實嗎？以輪船招商局為例，靠的是把「漕運權」交給了它；晚清史上最大的一次併購「旗昌輪船」的300萬兩白銀，也是南洋借的錢而不是自己賺的錢。

大圖模式

圖釋：記住了，這個是李鴻章不是左宗棠

輪船招商局在晚清根本不是一家近代公司、也不是按照近代公司的管理進行運行，只不過是一家打著「近代公司旗號的傳統官辦機構」——清政府（無論清廷還是地方）是他的主要客戶和生存基礎。

（2）晚清時期最主要的主題是三個：改革、自強、民生

許多人都關注前兩點，總覺得沒有前兩點好，第三點民生好不到哪裡去。但是，我們看到從1860年代到1900年代的半個世紀里，改革的成果如何？自強的成果如何？如果民生不好，我的天得死多少百姓！

因此，評價貢獻就要看改革貢獻、自強貢獻、民生貢獻。

在後兩點曾國藩、李鴻章哪項比左宗棠強？張之洞因為差著輩分，我覺得跟著三位比不上，他應該去和他那個時代的人比。

大圖模式

左宗棠收復新疆、打敗了侵略者。雖然這個侵略者相對弱小很多，但也是侵略者，晚清這樣的「戰績」誰有？李鴻章面對阿古柏都嚇得「放棄新疆」，並對慈禧說：阿古柏虎狼之師，左宗棠人老兵疲。

當然，他在左宗棠勝利後，揶揄阿古柏是不過土匪而已，但那是別人打贏了之後才這麼說的。整個新疆、陝西、甘肅、青海、寧夏的大西北地區，民生搞得多麼好絕對不敢說，但保持了基本的穩定，在那一時期就很了不起了。

再看民生和改革。左宗棠做浙江巡撫的時候，就是「戰後重建」搞得最好的官員。眾所周知，他的綠色建設思維、愛民思維、注重吏治反貪腐行為，都是四人中最好的。改革方面：

大圖模式

圖釋：福州船政學堂奠定了從晚清到新中國建立初，幾十年的海軍人才基礎。在這點上，左宗棠才是海軍、海防第一人

左宗棠建立了中國第一家海軍人才培養學校，從北洋水師到整個中國海軍，從晚清到新中國建立初，左宗棠建立的這家馬尾學堂培養的人才都是中國海軍的中堅力量、重要力量。

左宗棠建立的福州船政局擁有中國第一家海軍專業造船廠、第一家電報人才培養學堂。

左宗棠在蘭州開辦的洋務企業也是西北之最，學過歷史的人都知道，當時中國的洋務企業都是在沿海地區。可見，左宗棠對於改革的重視。

說句比較氣人的話，光說沒用，把貢獻拿出來說說，僅平定陝甘民亂、收復新疆、大兵震懾沙俄要回伊犁這一串「軍事成果」，其他的功績還用說嗎？除非你是個「子賣爺田」之徒。
