如果曹操敗走華容道被關羽所殺，三國局勢會偏向劉備？只會更糟

2019-12-16 333條評論 4,805次閱讀 270人點贊

赤壁之戰曹軍大敗，曹操和剩餘這些大將們只能敗逃，走到華容道遇見了關羽的截殺，本來以為自己必死無疑，結果關二爺為了還他當年的恩情，寧願違反軍令狀。但假設他這次沒逃過去，被關羽殺了，三國局勢會不會對劉備一方產生好的影響，其實按照三國時局來看，曹操死了，對劉備更糟。

首先，要是真的殺了曹操，於情於理是說不過去的，因為當初曹丞相為了挽留關羽，那是花費了不少的心思，甚至在他過五關斬六將的條件之下，還能差遣荀彧為他送來曹操手諭，以便他能順利出了曹魏地界，這算是非常大的恩賞了。以曹丞相的性格弱點，猜忌多疑，再加上對手下之人其實是極其不信任的，在他明知道關二爺有異心的情況之下，還能在投奔之時，給予無限的優待，這可是一個非常了不起的舉動。

三國時期，對於關羽這個人來說，其實就是義薄雲天的代表，要是他真的殺了曹丞相，就很不符合這個人設的限定，而且還是關於忠誠和義氣的限定，因此作者肯定會這樣塑造一個這樣的細節，要真這樣設定，那關老爺的人設就沒有這麼完美，成「忘恩負義」了。

其次，曹操的生死確定了三國局勢的穩定，三足鼎立局面得形成，很大程度上都是靠著這三個人帝王之間的相互制衡。孫劉一直處於聯盟狀態，而曹魏一方卻始終滅不了其他兩方，這完全就是一個平衡的局面。但你看後來走向，曹操一死，其實這種局面就被打破了。

而此時要是真的換做是關二爺將他殺了，那麼反而會對劉備更加不利。第一，曹操一死，那麼曹魏的勢力勢必會被削弱，這個時候壯大的不是蜀漢，而是東吳。孫權坐收漁翁之利，此時的蜀漢就會陷入最危險的境地當中，東吳肯定會撕毀聯盟，而第一個被開刀的就是蜀漢。因為它的實力最弱，最好下手。

第二，曹魏的勢力被削弱之後，曹魏一方很有可能因為爭儲之事，鬧得不可開交，此時很有可能又會回到之前東漢末期的場景，多地割據勢力興起，蜀漢作為最弱的一方，也會陷入這種被動的局面當中，好不容易鞏固的基礎，會硬生生的因為時局而變得更加的複雜。

這其實也真是為何讓關二爺作為最後一道截殺曹操的屏障，因為此時的曹孟德不能死，將他換做是張飛、黃蓋還是趙雲，那曹操肯定必死無疑，而唯獨關羽不會。諸葛亮這麼聰明的呢，不會沒有算到這步棋，此處也正體現了關老爺太過講義氣，最後的死也是因為義氣。

曹孟德作為三國時期一個非常重要的人物，按照局勢分析，他要是真的死了，那麼三國就會提早進去魏晉南北朝這種亂世時期，說他是梟雄，其實這是在小說當中為了突出劉玄德的皇叔形象，刻意的美化或者醜化一個人的形象。而說起來這個劉皇叔，實在是有點弱了，不是蜀漢勝在了人和，可能三國早就沒有他的戲份，只能說帶著大男主光環的他，有諸葛亮護身，關張開路，一直順風順水，直到後來白帝城託孤，才領盒飯了。
