去和親的公主，中途發現懷孕了，索性半路自己建立了城堡當王

2019-12-22 361條評論 4,430次閱讀 1680人點贊

古時候的帝王，為了拓展疆土，促進民族融合，又不想發生戰亂，就把和親當成了一種重要的外交手段。歷史上比較有名的昭君出塞和文成公主入藏都是來源於此。只是有時候皇帝們捨不得自己的愛女遠嫁就會另外找人來代替。所以很多派去和親的公主，實際上很多都是皇親國戚或者王公大臣的女兒，還有一部分是普通的宮女，在大唐西域記中就記載了這樣一件事，有一個去波斯和親的公主在路上遇到了戰爭和賊寇，

送親隊伍只好將公主和陪嫁全都安置在一個山峰上，還在山下布滿了侍衛日夜保護，三個月後繼續趕路，可公主卻在路上嘔吐不止，隨行的太醫。經過檢查後發現她懷孕了，在場的所有人都很吃驚，照顧公主不當每個人都有責任，原路返回和繼續前進，都不是好的出路

。於是他們一致決定，就在山峰上建起一座公主堡不再趕赴波斯。第二年，公主生下一個男孩，她宣布建立朅盤陀國，人們在那裡就地發展農耕經濟，放牧牛羊，護衛和侍女通婚繁衍子嗣。歷史上的朅盤陀國經歷了無數次的戰爭，唐朝時歸在了安西都護府的控制之下，宋朝時被西遼侵佔，元朝時被當地的和卓部代替，清朝的乾隆帝平定大小和卓叛亂，才將它收回到統治之下，新中國成立後，又在那裡設立了塔吉克自治縣

，成為了我國五十六個民族之一的塔吉克族，那裡的公主堡遺址也保存至今。
