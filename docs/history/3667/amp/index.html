大家好，我是虎哥。說起海盜，可能我們大家第一個會想到的，是好萊塢《加勒比海盜》系列電影，虎哥也非常喜歡。在影片中，有一個非常著名的場景，那就是「世界海盜王聚會」。虎哥赫然發現，參加這次聚會的，竟然還有一位亞洲面孔的老太太。

經過虎哥調查，這個老太太其實就是中國明末清初著名的女海盜，鄭一嫂。

中國的鄭一嫂，被許多歷史學家稱為世界第一海盜，她才是真正的「海賊王」。今天的迷彩虎歷史，虎哥就帶大家認識一下這個已經成為海賊王的女人。

明朝中後期，中國執行了嚴格的海禁政策。跨洋經商的商人們沒有了合法省份，索性在海上搞起了自己的武裝，走私貨物，捕撈海產，還向其他海盜收取保護費。在當時，走私貨物可以獲得暴利，這些海盜勢力一天天的壯大起來，不但可以跟明朝官兵較量，還跟洋人開戰。比如我們熟悉的鄭成功，其父親鄭芝龍就是當時最大的海盜勢力之一。

明朝滅亡後，鄭成功趕跑了荷蘭人，收復台灣作為自己的根據地。鄭成功的部將鄭建沒有跟上鄭成功，清軍封鎖的又緊，鄭建就帶領部下南下廣東，重新成為了海盜。鄭建的兒子鄭一，就是鄭一嫂的丈夫。

鄭一嫂原名石香姑，嫁給鄭一以後就被大家叫做鄭一嫂。鄭一嫂非常聰明，幫助鄭一打理海盜內部的事務，維護團隊的穩定，而且還頗有膽識，曾獨自率領手下搶劫商船。1805年，鄭一嫂所在的「紅旗會」統一了廣東海盜聯盟，建立起當時世界上最大的海盜組織。

後來，鄭一在颱風中身亡，鄭一嫂扶持義子張保仔上位，迅速穩定了局面，二人緊密配合，將海盜組織發展到了巔峰。他們以大嶼山為基地，統轄10萬部隊，擁有上千艘戰船，控制了整個廣東沿海。

後來，在與清軍和英國、葡萄牙等國軍隊的交戰中，鄭一嫂率領部下取得了多次重大勝利，成為了大英帝國談之色變的存在。

虎哥不得不說，這才是真正的「海賊王」，大家認為呢？
