看一看歷史上最殘暴的5個皇帝，最小的才是十幾歲

2019-12-19 1,174條評論 3,635次閱讀 2055人點贊

第五名，明太祖朱元璋，其實看畫像也知道是朱元璋，歷史上恐怕也只有他敢長成這樣，挺隨便的長！現代男人如果長成這樣，估計這輩子也只能打光棍了，但朱元璋不一樣，人家不僅有老婆，而且還很多，生育能力也不是一般的強，生了26個兒子16個女兒！朱元璋一生的經歷就像過山車一樣，所以他當了皇帝後可以說四個字「懷疑一切」，極其的殘酷無情，別的皇帝殺人是一個個殺，朱元璋是成批的殺，人家零售他專門搞批發，手段極其毒辣！

第四名，漢武帝劉徹，西漢在漢武帝時期國力達到頂峰，一改往日對匈奴的和親政策改為進攻！漢武帝在位54年時間裡，有40多年在對外用兵，漢武帝窮兵黷武，殺人不眨眼，十足的暴君形象！這都是為了建功立業，但是他晚年發生的「巫蠱之禍」純屬自己咎由自取！漢武帝還是歷史上最愚蠢的皇帝，他自己下手夷滅了自己的三族，太子劉據，太子妃，皇后衛子夫，衛長公主全部被殺，出生幾個月的皇曾孫劉病已在秘密保護下存活了下來！

第三名，前秦厲帝苻生！看著造型像不像索馬利亞海盜？這哥們天生就是一個獨眼龍，也許正是因為生理缺陷，導致心理扭曲，看到兩隻眼睛的人就想殺掉，宮殿上根本不是早朝而是屠宰場，稍微不順心他就要殺人，而且還不是痛快的死去，極其變態！苻生雖獨眼龍，但武藝高強，力大無窮，一般人根本靠不近！當初爺爺苻洪活著時就預言這個孩子不是好東西，弄死最好，但是他爹苻健不願意，最終釀成大禍！最後苻生想殺死堂弟苻堅，被苻堅殺死！

第二名，北齊文宣帝高洋！高洋小時候就是個傻蛋，裝瘋賣傻，流著鼻涕也不捨得擦！長得極其醜陋，一點不像大哥高澄（蘭陵王的爸爸）那樣瀟洒！以至於母親「九龍之母」想要扔掉他，但是老爹高歡覺得這兒子不一般！事實證明高洋確實牛叉，高澄準備篡位時得罪了廚子被殺，20歲的高洋迅速搞定局勢，自己當了皇帝，這時候精神了，也不傻了，前期勵精圖治，後來不知道被誰附身，成了精神病和變態狂，喜歡殺人取樂，母親和岳母挨個揍！

高洋的罪行讓人髮指，曾經把自己的老婆殺了，把她的屍體做成琵琶，哭著唱「佳人難再得！」第一名，劉昱！南朝劉宋皇帝，一輩子只活了15歲就被殺！但是他這短短的15年里，乾的畜生不如的事比活了幾十歲的皇帝都多！他喜歡殺人玩，喜歡出宮偷雞摸狗，護衛不能關門，否則腦袋搬家，叔叔劉義宣起兵討伐，結果自己和兒子全被被殺，劉昱還把自己的十多個堂姐全部搶進宮裡做老婆，這還不算，和自己有血緣關係只要長的漂亮的一個不落！
