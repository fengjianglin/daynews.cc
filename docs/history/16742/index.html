明朝皇帝用18.6公斤黃金，打造鎮國之寶，藏在寺廟500年從未被盜

2019-12-14 3,135條評論 4,154次閱讀 761人點贊

明朝皇帝用18.6公斤黃金，打造的鎮國之寶，藏在寺廟500年從未被盜

在中國數千年的封建王朝通知中，由於特殊的時代背景，導致民眾大多都比較迷信。其中，作為王朝的統治者——皇帝，為了更好的統治子民，便開始想到解決其他神秘的力量，來更好的鞏固自己的統治地位，而祭祀便成為彰顯天子權威的方式之一，為了顯示祭祀的隆重性，皇帝還會專門花重金打造鎮國之寶，而一般這樣的寶物，其歷史研究價值都普遍較高。

而今天我們要說的這件寶物，據說是明朝時期打造的鎮國之寶，是一口體型很大的鐘並且這口鐘被刻意隱藏了起來，500的時間裡從未被世人發現，這究竟是怎麼回事呢？

由於古時候特殊的時代背景，使得人們普遍相信”神仙”之能的存在，其中，皇帝最為明顯。而隨著在靖難之役的大獲全勝，朱棣成功從侄子手中奪得了皇位，朱棣登基之後，便將明朝都城前往北京。但由於古代人的迷信觀念，任何大事都會邀請高人來勘驗一番。最終，皇帝讓高人給北京城的風水格局把脈，而這位高人則按照五行之說，在北京的五個方位埋下了相對應的寶貝，以此來鎮壓北京的風水。

而這口明朝皇帝特意打造的國之重寶，就是一件位於東方的文物，全名叫做”永樂大鐘”，這口大鐘可以代表著當時最為先進的工業技術，即便以現在的眼光來看，依舊能發現許多令人驚奇的地方。

目前這口大鐘被懸掛在寺廟中，遠處看可能看不出來什麼，但走進之後才會發現，上面刻滿了奇怪的銘文，有人專門去統計過，這口大鐘的銘文，共計有23萬多字。而且這口大鐘的工藝水平很高，僅明朝掌握了這一工藝技術，這口大鐘的用料除了普通金屬之外，裡面還有18.6公斤重的黃金，以及36公斤的白銀，而也正是因為這些特殊原因，使得這口永樂大鐘能夠千百年不腐不銹。

而且這口永樂大鐘，其體積也十分的龐大，整口大鐘高約6.75米，其總重量高達40多噸，人要想敲響這口大鐘，會是件十分吃力的事情。雖然這口大鐘體型龐大，但懸掛這口大鐘的方法卻十分精妙，整口大鐘完全依靠一根小小的穿針，勾懸在在支撐木上，而且這根穿針跟大鐘的體積比起來，完全可以忽略不計。但就是這麼精妙的設計，使得這口大鐘懸掛數百年，始終未發生任何事故，可也說是非常牢固。

如今這口大鐘，已經在寺廟中懸掛的500多年，至今都未被盜過，既便是戰亂年代它依舊穩固的懸掛在那裡，雖然外人看來，除了驚訝它的提及之外，其他的也就再無興趣，但經過專家的詳細研究之後，這才發現了這口鐘的不同之處，從歷史的研究來看在，這口永樂大鐘，無論從哪個角度來看，都看成是國之重寶。
