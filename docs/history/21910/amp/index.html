慈禧太后最後一天是如何度過的？有老臣大哭，更多人罵她無恥

天天要聞
2019-12-19

慈禧太后作為晚清的統治者，此人在歷史上的爭議頗大，總的說起來，無論是過去還是現在，罵慈禧的人相對更多一點。

慈禧身為女人，卻將朝廷緊緊的握在手中，其實無論她做什麼，肯定都會被人口誅筆伐，其實如果有細心的人就會發現，如今罵慈禧的大多數人都會從她的生活奢靡入手。

大家都在說她如何奢侈，似乎從不會因為其他事情罵她，那麼假如說是皇帝這樣做呢？一個男性皇帝這樣做，相信並不會有太多人抓住這一點不放。

慈禧的一生其實是有不少功績的，而且從統治者的角度出發，她是很出色的，慈禧的能力非常強，假如說她能力不夠，她也沒實力將實權掌握在自己的手中。

慈禧並不是突然死亡的，在她死之前已經有了預兆，大家都知道慈禧命不久矣，包括慈禧自己，宮裡的人每天都陪在她的身邊，生怕慈禧突然沒了。

大臣們也都隨時準備著，等著慈禧交代東西，那麼慈禧最後一天都做了些什麼呢？根據史書記載的內容來看，那一天和平常沒有什麼不同。

慈禧太后在天剛剛亮的時候便起床，太監和宮女們伺候她梳洗，慈禧似乎預感到了什麼，她在吃飯的時候還將大臣們都叫過來，邊吃邊討論一些國家大事。

這次討論的內容大家都心知肚明，那就是討論立誰為帝，因為這會宮裡還有不少人在忙光緒的後事，其實這事也早已定下，只不過在這次討論中最終確定為溥儀新帝。

光緒過世，雖說死因和慈禧有關的可能性更大一點，但是慈禧肯定還是非常傷心的，只不過這個時候的慈禧已經顧不上這些，她要趕緊處理好身後事。

緊接著沒多久，溥儀被緊急招入宮內，慈禧勉強抱著溥儀，和他說了會話，當然這會的溥儀可什麼都不懂，慈禧能和他說些什麼呢？想必也就是讓他做個好皇帝。

緊接著第二頓飯開始，吃過飯之後，據慈禧身邊的人回憶，老佛爺好似泄了氣一樣，吃過飯還沒走到床邊便暈了過去，這可嚇壞了眾人。

人們都以為老佛爺這次暈倒便不會再醒了，就在宮女們大哭的時候，老佛爺醒了，並且非常的精神，了解的人都知道，這是迴光返照，慈禧立即召集近臣前來，最後交代了一番事情。

慈禧交代完重要的事情之後，開始對大臣們說一些體己的話，比如說要照顧好自己的身體，要多多輔佐年幼的溥儀，畢竟他歲數太小了。

慈禧還專門說了自己死後，大清朝的女人不得干政，她也說了自己垂簾聽政是情非得已，其實拋開慈禧個人的權利慾來說，按照清朝的規矩，她垂簾聽政並無不妥，完全符合規矩。

假如說光緒有能力，能夠強勢一點，相信他能奪回實權。慈禧交代完這些，大臣們潸然淚下，慈禧也不再說話，大臣們相繼退了出去。

一直到下午五時，慈禧突然過世，在這個期間，慈禧一直沒有說話，不知道這個時候的慈禧心中想著的是什麼呢？
