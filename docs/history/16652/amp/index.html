狄青故事：惡面狂刀追風雪，力奪戰甲懲奸徒，老少名將合力保江山

天天要聞
3 天 ago

北宋時代有一個非常特殊的情況，就是由於宋太祖趙匡胤為了守住江山，避免其他人模仿自己，對武將進行了嚴格的控制與高度的抑制，所以，北宋一朝可以算是歷史重要大朝代中，軍事實力最弱，行動力最為僵化的時代。因此，能在北宋成為名將的人不多，也許狄青算是其中成就最高，也最有影響力的一個。然而，北宋時期，又因為周邊各政權的先後崛起，使得北宋的戰事不少，這樣一來，北宋時期的名將演義小說故事，又是非常受歡迎的，其中最著名的當然是楊家將。而傳統的話本評書又經常喜歡將各朝代的虛構名人，統一在一套系統中，所以真實的名將狄青，也與楊家英雄中不少虛構出來的人物，不可避免地產生交集。

之所以說是虛構人物，是因為時間的關係，效力於宋仁宗朝的狄青，只能跟楊家第三代「掌門人」楊宗保接觸上，但根據目前的文獻，已經證明出楊宗保是個虛構的人物，反而是小說中設計的楊宗保之子楊文廣，才真正是楊六郎的兒子。但是，按照小說的邏輯，系出名門的楊宗保，是不大認可狄青這個行伍出身的後起之秀的，因此，當西夏來犯之時，楊宗保和狄青這個大宋江山的一老一少兩大名將，為正當主帥鬧得很不愉快。最終，宋仁宗選擇了經驗更豐富的楊宗保挂帥出征，狄青只能鬱悶地回家等候下一步安排了。

然而，西夏的這次進犯，其實是與朝中的胡天化勾結有關，而每一部袍帶書中，都會有胡天化這樣的角色，他想要取代大宋皇帝，必須就先要除掉楊家將以及狄青這些忠臣良將，所以楊宗保出征西夏，本來也是胡天化計劃中的一步，因為他算準了當時的氣候，楊宗保在八月出征，不可能很快打敗西夏，一旦戰事進行超過兩個月以上，冬季到來，身穿單衣的北宋軍士，就必須要面臨保暖的問題，而到了那時，胡天化就可以繼續操作了。果然不出他所料，到了十一月份，宋軍根本無法禦寒，因此在前線的楊宗保，只能向朝廷發出救急文書，請求宋仁宗，快速運送冬衣戰甲過來，以保證前線的戰鬥力。這樣一來，狄青就有了用武之地，他負責作為輸送補給的主帥，押送十萬套冬衣戰甲，趕赴西夏前線。

這樣一道命令，也在胡天化的意料之中，他接下來要做的就是再次勾結西夏，讓他們搶走戰甲，使得狄青背負重罪，就可以順利除掉狄青。而前方的楊宗保，也會因為無法抵禦寒冷，或者凍死在西夏，或者大敗而歸，被朝廷問罪，總之，無論是那種結果，楊宗保和狄青兩大英雄都是死路一條。因此，在狄青出發之前，胡天化主動請纓，作狄青的副手，一起趕赴前線，理由是，因為狄青和楊宗保在三個月前爭奪帥位，矛盾比較深，怕二人在前線激化矛盾，所以他胡天化承擔下了調停的責任。宋仁宗當然對胡天化的「忠心」感到滿意，可是在補給隊伍趕赴西夏的路上，胡天化就開始做起手腳來，他先是讓人騙狄青，說自己被當地的土匪綁架了，這樣一來，狄青必然要區救援胡天化，但實際上卻是調虎離山之計。果然，狄青上當，等他在荒郊野外「救下」胡天化之後，回到大營中，傻眼了，原來戰甲已被「匪盜」搶走。

這時，胡天化勸狄青趕快逃命，否則的話，追究下來，他也必然是死罪，但狄青是什麼人，當然不可能接受這個提議，反而怒氣沖沖，立下誓言，一定要找到盜匪，奪回戰甲物資，然後就離開了。胡天化當然不相信狄青會識破自己的陰謀，便任由他去了，自己卻感到了楊宗保身邊，添油加醋地誣告狄青玩忽職守，整體喝酒，結果弄丟了戰甲，如今畏罪潛逃。雖然楊宗保不太相信這些，可是終歸沒有戰甲，也沒看到狄青，前線又吃緊，所以也只能將信將疑。且說狄青，單刀匹馬開始尋找戰甲的下落，按照小說主角光環的設定，狄青當然會輕鬆查出戰甲的隱藏地點，而且他趕到現場非常及時，如果晚一步，十萬冬衣戰甲，就被西夏軍隊燒掉了。

這時的狄青鬥氣爆棚，帶上他成名的惡面金縷鬼面具，揮舞狂刀，如同凶神惡煞般，衝進敵營，在一個風雪交加的夜晚，狄青殺得西夏軍四散奔逃。接下來的事兒就很簡單了，狄青成功搶回了戰甲，並且抓住了一個西夏軍官，由此真相大白，最後押送著物資來到前線，成功交到楊宗保手中，並且跟胡天化當堂對質，徹底粉碎了這個小人的陰謀。經過這麼一系列的風波，楊宗保也終於認可了這個神勇的後起之秀，狄青也更加佩服這位老前輩，從此，這對大宋最為倚重的老少名將，合力戰勝了敵人，為保國安民貢獻了全部的力量。故事就是這麼簡單，而且就像老樂在開篇提到的一樣，楊宗保作為虛構的人物，就註定此事也不可能是歷史真實事件，但是，無論真偽，狄青和楊宗保的愛國情懷，為了國家不計個人榮辱的精神，才是值得我們去推崇與學習的，這也是眾多虛構的評書小說，可以在民間深受歡迎的一個重要原因。
