失控的王安石變法，宋朝危局的一大推手

天天要聞
2019-12-16

導語

王安石，一個全民皆知的詩人，他有”不畏浮雲遮望眼，自緣生在最高層”的人生決心。做為宋朝時期著名的文學家，他在文學上建樹極高，同時他也是一位改革家，出色的政治家。宋朝的新舊黨政可以說是改變宋朝命運的事情，而王安石做為新黨的代表人物，在掌權成為宰相後展開了變法。而這次變法卻失敗了，同時還使得北宋陷入了一個進退兩難的地步。王安石變法究竟給北宋帶來了什麼？

對王安石變法的評價

提及王安石變法就不得不提及王安石本人，以及對於他的評價。做為詩人而言，王安石的出色有目共睹，他在文學史上留下了自己無可取代的一筆。但是從一個政治家的角度而言，人們對王安石的評價就褒貶不一。

近代愛國志士就梁啟超評價王安石是當時的仁傑，他的變法如果成功必定能改變宋朝的命運。偉大的馬克思主義踐行者列寧為稱讚王安石為古代中國的改革家，是改革的先驅。但這其實是從現在的角度而言，王安石變法在當時是必要的。

圖一，王安石畫像

在王安石之後的許多文人，對於王安石的評價，大多認為他是擾亂宋朝的最魁禍首。北宋理學大家朱光直言不諱的表示王安石做為丞相，破壞了宋朝的法度，因為他主政，宋庭忙於派系之爭，為後來留下了許多弊端。而朱熹更是在他所著寫的書中說王安石以道義之名，行禍國殃民之事，是個十足的道貌岸然之人。

可以說在兩宋文人心中，王安石臭名昭著，他們認為宋朝之所以積弱成為這種樣子，飽受屈辱和王安石有著極大的關係。而對於王安石這種負面的評價，其實一直持續到了明清時期都還存在。對於王安石而言，他究竟是個什麼樣的人，他推行的變法到底為了什麼，只有深入了解才能對於這樣一個時代催生的悲情人物，有一個中肯的評價。

王安石變法打壓政見不合之人

據《宋史·王安石傳》記載: “性強忮，遇事無可否，信所見，執意不回。司馬光謂其泥古，所為迂闊。”從這段話其實就可以窺探出王安石在為人上過於強勢，根據這段記載王安石對於同自己政見不和之人，都採取了打壓排擠的做法，不是利用自己的職權將其外調，就是貶官。

圖二，王安石與司馬光新舊兩黨代表人物

北宋之時賢能宰相眾多韓琦就是其中一位，當時王安石變法中有一項，在全國範圍內推行青苗法。王安石推行青苗法其實是為了改善百姓的生活狀況，使得他們在饑荒之時可以通過向朝廷以極低的利息貸取糧食，錢財。可惜初衷雖然好，但是施行起來，青苗法成為了貪官污吏魚肉百姓的一大助力。韓琦知道後就向當時的皇帝趙頊陳述實情，希望這種做法得到停止。

圖三，新舊黨之間爭論場面

可惜當時王安石一直以他推行此法的初衷為由，覺得民間實情與自己預料的相同。所以還在皇帝面前污衊韓琦，說韓琦是擔心推行此法會損壞自己的利益，並且誣告韓琦中飽私囊，阻礙新法施行。到時朝廷之中權力被新派掌控，所以韓琦被罷相，遭到貶謫。

宋朝當時為了監察官員，設置了台諫官員，而王安石對於這些官員，依舊採用打壓排擠的方式，當時御史台的官員批判新法，強勢的王安石直接將御史台官員罷免，或者貶官外調。使得王安石當時在朝堂之上幾乎推行一言之堂。

變法初衷為善，卻遭奸人利用

王安石變法失敗其實與他個人的性格有著極大的關係。王安石為人過於強勢，難以聽講別人的勸誡，這使得他認為只要支持自己變法之人，都是忠臣義士，而為也正是如此他變法的隊伍之中充斥著一些投機倒把，意圖藉機魚肉百姓之人。

在這場變法之中，王安石在識人上面就出現了極大的失誤。做為他變法的一大助力之人，呂慧卿就是王安石在任用人才上最大的失誤。不可否認，呂慧卿本人才能確實出眾，王安石許多變法內容都有他的參與，包括在推行實施上。但是不是每一個人都能經受得住權力的誘惑，呂慧卿就陷入了權力的漩渦之中無法自拔。

圖四，關於王安石與司馬光爭論的插畫

王安石被貶後，呂慧卿被提拔為宰相，他當權下的北宋政局更為混亂。不分忠奸，為了擴大自己的勢力，呂慧卿將反對自己的人統統清掃乾淨，重用自己的人。而且他打壓的對象不分新派舊派，就包括提拔自己的王安石都難逃他的打壓。為了徹底排除王安石的威脅，呂慧卿可謂絞盡腦汁，為王安石安上了許多罪名，使得王安石徹底失去了翻身的機會。

王安石當初之所以重用於他，一方面是覺得呂慧卿確實有才能，另一方面是因為呂慧卿經常能投中王安石喜好的方面，這使得王安石對於這位下屬十分喜愛。這也是為什麼王安石掌權之時，呂慧卿可以在短短兩年之內從一個名不經傳的人，成為朝廷大員。變法之中這樣的人還不在少數，所以對於變法結果，其實也可想而知了。

王安石變法對北宋影響極大

可以說王安石變法對於北宋政局的影響是極大的，特別是新舊黨爭。新舊兩黨之間不斷更迭，就王安石一方，許多官員就是被貶之後，又再度起用，就算身為宰相的王安石都被罷免過兩次，更別說舊黨了。政見不合，一黨掌權之後，立即推翻之前的制度，使得國家極度混亂。

黨爭到了最後演變成為了派出異己的肅清行動，以及人身攻擊。宋哲宗繼位時，司馬光再度執政，就徹底廢除新政，恢復原本的制度，對於新派全部貶官，無一例外。後來王安石上台，又變成了角色互換的打壓。這使得宋朝官員大幅度被貶南方偏僻地域的局面開啟。

圖五，王安石與呂慧卿

最為嚴重的是，到了宋徽宗時期，新派舊派之間對於相互否定對方功績，黨爭到了不可調和的地步。許多對於北宋王朝發展做出過極大貢獻的人的功績被抹殺，一些已經去世的功臣，連被追贈的謚號都被奪回。黨爭就這樣從持續到了後來北宋王朝被滅，靖康之恥的發生才得以消停。

結束語

有人說，如果沒有王安石變法，沒有新舊黨爭，北宋王朝或許不會積弱到最後皇帝被擄走，在帝國遭到羞辱的屈辱事件發生。王安石變法的失敗不僅僅是因為他意圖動搖當時貴族的利益，更多的方面其實還是在於個人的失誤。這正如”可謂有專門之學者矣。而卒以敗者，無通識，並不周知社會之故，而行不適之策也”一言。
