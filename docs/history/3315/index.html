此人和宰相同時落水，卻大喊：先救宰相，結果宰相淹死，他高升

2019-12-06 0條評論 21次閱讀 0人點贊

摘要

公元224年，曹丕準備第二次伐吳，建造“御舟”的任務，落到了尚書僕射（相當於宰相）杜畿的頭上。隨著“曹丕御舟”的完工，杜畿依然不敢掉以輕心，決定親自登上御舟出去溜一圈。隨行的還有幾人，其中最引人注目的，便是蜀漢丞相諸葛亮的族弟：諸葛誕。

公元224年，曹丕準備第二次伐吳，建造「御舟」的任務，落到了尚書僕射（相當於宰相）杜畿的頭上。隨著「曹丕御舟」的完工，杜畿依然不敢掉以輕心，決定親自登上御舟出去溜一圈。

隨行的還有幾人，其中最引人注目的，便是蜀漢丞相諸葛亮的族弟：諸葛誕。此刻的諸葛誕比起威震三國的諸葛亮來，過得很不得志，若非因諸葛亮的緣故，估計也不會被注目。

如此，曹丕御舟駛入了水面漸行漸遠，哪料禍事來了，突然江面上起了風浪，曹丕御舟翻了船，杜畿、諸葛誕等一起落水。負責救援的虎賁將士，慌忙駕駛小船來救，首先來到了諸葛誕身邊。

哪料諸葛誕卻在水中掙扎著高喊：「別管我，先救僕射！」虎賁將士們也真實在，真就駕船離開去救杜畿。諸葛誕繼續在水中掙扎，終於沒了力氣，暈死了過去。當諸葛誕醒來後，才得知自己被江水衝到了岸邊，而宰相杜畿已經被淹死了！

此事傳到了曹丕耳朵里，曹丕大哭杜畿為朕而死，但也記住了這位「懂事且會做人」的諸葛誕，沒多久諸葛誕便高升，被破格提拔為：御史中丞！諸葛誕終於步入了曹魏高官行列，跟當時的皇親國戚夏侯玄等搭上了關係，混得非常開心！

但世事難料，誰也沒想到曹丕竟然年僅40歲時便死掉了，曹叡登基，諸葛誕的好日子到頭了。不是因為諸葛亮的北伐，而是因為曹叡不喜歡夏侯玄、諸葛誕這類所謂名士們，故而一頂 「沽名釣譽」的罪名壓來，夏侯玄、諸葛誕等齊齊被罷官了！

但需要說明的是，「沽名釣譽」的主犯是夏侯玄，諸葛誕主要是因他跟夏侯玄走得太近。諸葛誕又成了「草根」，只能看著堂哥諸葛亮替他報仇，一次又一次的北伐。真不知此刻諸葛誕該怎麼想。為何他非效力曹魏，而不去投奔諸葛亮？哪怕諸葛瑾呢？

如今只能說，諸葛誕是位想有所作為的人，要用自己的才能開闢一片天地，這也符合三國時期諸葛家族的性格。終於曹叡年紀輕輕的也去世了，曹芳登基！所謂一朝天子一朝臣，夏侯玄復出，曹爽執政，諸葛誕也迎來了春天，官復原職！

此刻的諸葛誕儼然成了曹魏重臣，一面跟夏侯玄關係很鐵，一面還跟司馬懿家是親家，他的女兒嫁給了司馬懿的三子：司馬伷！所以諸葛誕既非曹爽黨羽，也非司馬心腹，他是腳踩兩條船！

諸葛誕的這種模樣，其實依然是延續了他當年落水後，高叫「先救僕射」的精髓，所謂的「懂事」，換個詞就是：功利心太重！因為諸葛誕最重要的事迹，就是「淮南三叛」。

第一次是王凌之亂。王凌是曹魏忠臣，他之所以反叛，是因高平陵之變後，司馬懿控制了朝政，王凌企圖另立新君，以此使曹魏擺脫司馬懿的控制。

第二次是毌丘儉和文欽之亂，由於文欽是曹爽的鐵杆，司馬懿又死了，所以這兩人的反叛，大有當第二個曹爽的意思。

在這兩次反叛中，諸葛誕都出了大力，須知他跟司馬家沾親，而又遭到過曹叡的打壓，自然對曹魏沒啥感情，必然要拱衛司馬家的地位，以至於諸葛誕平叛時，比誰都積極的現象，尤其是在第二次淮南之亂中，諸葛誕是頭一個領兵攻入壽春的。

但哪料，隨著前兩次反叛被挫敗後，諸葛誕愕然發現：壞了，我咋成了司馬家的頭號敵人了？因為此刻的諸葛誕擁兵十萬，坐鎮淮南。所以《三國志》中才寫道：諸葛誕「懼不自安，傾帑藏振施以結眾心，厚養親附及揚州輕俠者數千人為死士」。

也就是說，諸葛誕開始花費重金收買了數千死士，只聽命於他自己——這點後來在諸葛誕死後，成了諸葛誕的一個所謂的亮點，因為這些死士都慷慨為其赴死！諸葛誕的這些舉動，更讓司馬昭坐卧不寧了。故而司馬昭升任諸葛誕為司空（三公之一），調虎離山，想把諸葛誕置於自己的掌控之中。

諸葛誕識破了司馬昭之心，掉頭就投降了東吳反了曹魏，拉開了諸葛誕第三次汝南之亂！可惜，這次叛亂，再次被司馬昭給平叛了，諸葛誕也被殺！

分析諸葛誕反叛，顯然不是因他忠心曹魏。若他真忠心曹魏，為何那麼賣力的平叛前兩次的汝南之亂？所以諸葛誕其實就是想獲得更高的權力罷了！

再說明白些，諸葛誕由於一門心思的想有所作為，成為了一位嚴重的現實主義者。由此來看做人，還真是不能太過現實，若沒有了理想之光引路，很容易迷失自我，哪怕再懂事，會做人地喊出了「先救僕射」也是白搭！
