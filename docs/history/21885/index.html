​「歷史」武則天的父親竟然是一條“野龍”

2019-12-19 3,933條評論 4,858次閱讀 116人點贊

​今天我來揭開女皇帝武則天的身世之謎，話說唐朝有個女的當了皇帝，自然就是武則天，曾經在一部電視劇里被剪了胸的武媚娘。話說武則天這位天女先後嫁過兩位皇帝，這倒是沒啥，關鍵是它嫁在兩個皇上，是一對父子，分別是唐太宗李世民和他的兒子唐高宗李治，在唐太宗李世民死後，武則天就剃了禿瓢去當了尼姑，然後又一還俗就嫁給了自己前夫的兒子李治了，而更有能耐的是武則天還生了兩位後來的皇帝，分別是唐中宗李顯和、唐睿宗李旦，不過最後他也不管自己的兒子，什麼李顯、李旦，直接自己當了皇上，也就成為了這麼一個獨一無二的女皇帝，在死後還為自己留下來一座，但是武則天的出身究竟在哪？卻一直都像這無字碑一樣是一個謎。

話說唐朝那會兒的史觀和記者實在是缺乏歷史的責任感，加上古時候並沒有狗仔隊，就是有狗仔隊也不敢去調查武則天。 所以光武則天老家的說法就有三個，其中頭一個比較靠譜，說武則天是天源籍并州文水徐村，也就是今天山西省的文水縣，跟女英雄劉胡蘭是同鄉。

而第2種說法是武則天生在長安，也就是現在的陝西省西安市。還有一種說法是武則天生在利州，就是現在的四川省廣元市。而解密武則天的出生地到底是山西、陝西還是四川？可以說這三個地方跟武則天都有關係，武則天祖籍并州，生在長安，長在利州，因為武則天的父親武士彟早年跟唐高宗李淵南征北戰打天下，等到天下安定之後，留在原籍的夫人鄉里是已經去世，武士彟於是就續娶武則天的母親楊氏。

當時在長安當官，所以媚娘生在首都長安的消息應該是可信的。而在武媚娘的童年時代，武士彟曾到四川的利州為官，保不齊也就帶著自己的家眷上任，所以武媚娘很有可能也在四川的利州，也就在現在的廣元市​生活。不過不管武媚娘有沒有城居利州，利州人民都是非常在乎武則天的，因為武則天後來當了女皇帝，利州人民便將利州當成了武則天的故里，還編出了一則武則天出生的荒誕故事。

在這個故事裡面是這麼說的，傳說利州有一個水塘，江湖人稱龍潭，龍潭的潭水清澈見底深不可測。 當地於是就有個說法說，有那麼一個月武則天的母親沒來大姨媽，正好天氣炎熱，那會兒也沒有空調，於是武則天她媽就想到了跳到龍潭裡去涼快涼快。於是他就脫得一絲不掛，撲通一聲跳到龍潭裡，不管是狗刨還是蛙泳，反正就是游上了而武則天的漂亮隨她娘，所以武則天的母親也是個大美人，凹凸有致，要哪誘​哪格外有人，他在龍潭裡遊了不到2000米，就驚動了潭底的一條龍。

那條龍本來是潛在潭底睡懶覺，結果武媚娘的母親游泳把這條龍給驚醒之後，這條龍被武媚娘的母親凹凸有致的身材迷得神魂顛倒，睡意全無，於是這龍猛然間鑽出水面，一陣長吟在空中盤旋幾圈，不由分說地潛回水裡，死死纏住武媚娘的母親性感的身體，此處省略12,000字，而最後的結果就是武媚娘的母親懷上了這一條龍的龍種，於康伍德七年龍女武則天橫空出世​，有時候真的不得不佩服我國廣大人民群眾豐富多彩的、超乎尋常的想像力。
