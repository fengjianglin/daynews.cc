古代官員退休，為何不選擇留在京城？而一定要回老家

2019-12-15 1,421條評論 3,770次閱讀 1425人點贊

古代官員的退休制度不同於今天。當時，它被描述為「致仕」。古代官員致仕(退休)後，大都會選擇回老家，即便應皇帝要求留京，但是過段時間也總會不斷找出些理由，要求回老家。所以我們在影視劇中，經常會看到，有退休的官員會向皇帝要求「告老還鄉」。那麼，為何這些官員都不願意留在更為繁榮的京城，而非得回到偏僻的家鄉呢?小編將原因歸結為以下三點!

第一是由於古代人更重視「落葉歸根」的文化情節。古代仕子從小苦讀聖賢書，一心求功名。真可謂「少小離家老大回，鄉音無改鬢毛衰」。且古代交通不便，很多官員在外為官多年，很少返鄉，除了父母死後，守孝三年，基本上可以說是沒有機會回家的。所以，大多數官員致仕後，都會考慮死後進祖墳、入宗祠。所以得趁著自己身體還健康的情況下，趕緊動車回老家。畢竟古代車馬趕路比較慢，距離遠的，甚至得需要一年半載的時間才能回去。一旦客死他鄉，就可能成為孤魂野鬼了，這在古人的眼裡是不能承受之重啊。

第二是由於回老家更能安穩享福。中國古代以農耕文明為主，財富除金銀財寶等外，大都以土地的形式存在。在古人眼裡，土地是最穩定的硬通貨，可以福澤子孫後代。古代出名的大貪官，如和珅、奕劻、劉瑾、嚴嵩、石崇、蔡京等等，哪個不是在老家購置良田無數?古代官員沒有退休金，但在退休前，大都會安排好後路，在老家購置足夠的田地，由族人代管，以便將來退休返鄉，可以繼續過榮華富貴的好日子。

第三則是由於保全自己。我們都知道，古代官場非常殘酷，在朝為官本身就是一件很冒險的事情，稍不注意就可能引來殺身之禍，很多官員可能因為說錯一句話，就會被天子斬首，可謂是凄慘之極。所以，在致仕後，大多數官員當然會想遠離官場，就算呆在京城，也是不保險的事情，倒不如遠離皇宮，回家安心休養，何樂而不為呢?

說到底，古代做官確實是挺不容易的。雖然拿著很多俸祿，卻也承受著巨大的壓力。或許在致仕回家的那一刻，才是他們內心真正開心的時刻吧，你認為呢?
