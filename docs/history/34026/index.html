劉備的天下是哭來的，真的有此事嗎？劉備真的這麼軟弱嗎？

2019-12-23 1,999條評論 4,685次閱讀 1242人點贊

說起劉備啊，我就想說一句，劉備的天下是哭出來的。身為《三國演義》的男主角，劉備的形象那簡直就是絕了啊，不僅愛逃跑，還愛哭，時不時就哭鼻子，一點男子氣概都沒有，真的是讓人唏噓！那真實歷史上的劉備，真的是如《三國演義》演的那樣嗎？

歷史上的劉備給人留下的印象很奇怪：他沒有地盤，可是有人奉送給他（徐州牧陶謙、益州張松、法正）；他常被打敗，單獨一人逃走，可是群雄都願意收留他（呂布、袁紹、曹操、劉表）。換句話說，劉備這人二十四歲左右出道，混了將近二十多年，只有聲望，沒有戰績，可是卻沒有人敢小看他，還把他當梟雄看待。這就奇怪了，就這樣還能人人追捧他？

然而劉備得到眾多人的追捧是有原因的。劉備有一種特性啊，那就是人緣好啊！我們先從劉備的性格和出身說起。劉備出身平民，雖然他一直自稱是漢朝宗室的後代，但是從小過的可是凄苦生活啊。小時候和母親以編織、販賣涼席為生。少年時不喜歡讀書，每天就愛穿得很”潮”，往酒店跑，賭馬、賽狗。長大後的劉備話很少，對人謙虛，很少表露自己心裡在想什麼，並不愛哭，但卻很有義氣，講信用，樂意認識三教九流的朋友。

然而，他結交的朋友還是有用的，他很快拉起一支隊伍，參加鎮壓黃巾之亂。咱們熟悉的關羽、張飛這兩員忠心耿耿的大將，也是這時候加入的。陳壽說劉備很像他的祖先漢高祖劉邦，其實劉邦粗俗愛罵人，劉備卻心胸開闊，舉止大氣，在這點上比起劉邦更有魅力。

更令群雄不敢小看的是，劉備不但政治嗅覺敏銳，而且啊，人家可是有遠大的志向的。經過多年的征戰，周旋在中原各大小軍閥之間，劉備逐漸領悟到：他是不可能和曹操和平共處的，身為漢室後裔，他應該號召天下，恢復漢朝，建立一個以他為中心的新政權。劉備的心可真大啊。劉備雖然東奔西走，常被打得落荒而逃，但卻激發了他爭奪天下的雄心壯志，而且即使窮途潦倒，也不放棄認輸。正是這樣的屢敗屢戰，堅持奮鬥，他才能在年近半百的時候，請到諸葛亮相助，英雄終於有了用武之地。劉備三請諸葛亮，可能就是因為自己經常吃敗戰，需要一個有謀略的軍師幫助自己實現夢想。

電視上演的劉備哭個不停，那都是不太可信的。劉備的江山不是靠眼淚哭來的，而是靠他堅韌不拔的鬥志和一群長期追隨他的忠實部下的打拚，再加上諸葛亮的正確戰略謀劃，共同奮鬥拼出來的！但是說真的，劉備好在他人緣好，而且為人比較正直，很多英雄豪傑願意拚死效勞，他才能有如此的大成就。再說了，劉備自己也是有遠大抱負的，他可是一心想建立新政權，這對他也是有影響的。

俗話說得好，得民心者得天下！很大程度上說，劉備才是最後的贏家，有那麼多忠心耿耿跟著自己的將士，實現夢想自然更容易了。
