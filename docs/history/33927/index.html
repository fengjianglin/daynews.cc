曾國藩去世前，李鴻章問他哪些人能堪大用，他說出五個頂尖的人才

2019-12-24 616條評論 1,803次閱讀 287人點贊

​咸豐初年，轟轟烈烈的太平天國農民起義在廣西爆發了，很快就席捲了大半個中國。太平天國的起義，農民和地主階級的戰爭，直接威脅到封建統治階級的存亡。

起義雖然啟自廣西，湖南卻是鬥爭最激烈、最殘酷的戰場。以曾國藩為代表的理學人士紛紛投筆從戎，湖南地主階級亦紛紛出山，他們組織武裝對抗太平軍。自此，湖南成了清政府鎮壓太平軍的重要基地。

曾國藩畫像

湘軍是太平軍最兇狠、最頑強的敵人之一

1854年，曾國藩練成湘軍、率師出征，並作檄文號召天下讀書人起來「衛道」。他打著封建禮教的旗幟，號召地主階級特別是知識階級投入到這場運動中。

曾國藩率領湘軍水、陸兩師一萬七千餘人從衡陽出發，開始了與太平天國的長達十多年的戰爭。從此，湘軍成為太平軍最兇狠、最頑強的敵人之一。也就是說，無論在省內還是省外，湘軍都是太平軍的主要對手。

曾國藩的人才們

那麼，為何曾國藩領導的湘軍，最終能夠戰勝太平天國的反清起義軍？

一方面，太平天國領導人洪秀全雖然在起義初期能不拘一格、唯才是舉，但在定都南京後，他便逐漸開始任用親信，排斥賢能，造成人心渙散、內訌不斷的局面。太平軍元氣大傷，終於走上無法回天的慘敗之路。

另一方面，在湘軍與太平軍你死我活的戰爭中，曾國藩的身邊湧現和成長起來一大批有勇有謀的人才，而曾國藩知人善用的名聲也廣播在外，連翼王石達開都稱讚曾國藩「能識別賢將」。正是這些人才的助力，逐步為湘軍取得了戰爭中的優勢。

太平天國將領劇照

事實上，早在創立湘軍之前，曾國藩對人才的重要性就有清醒而深刻的認識。他認為「國家之強，以得人為強」。針對滿朝大臣動不動就說國家無人才可用的空談，曾國藩十分反感。在曾國藩看來，天下並非無人才，只是掌握權柄的人不善於發現和培養人才罷了。

曾國藩認為，天下並非無人才

事實可以證明一切。在血與火的戰爭中，曾國藩身邊湧現出了許多「用劍」鬥爭的軍事人才。《清史稿》曾為湖南一百零二人作傳，其中屬於湘軍系統的軍事人物多達八十六人。

曾國藩劇照

曾國藩以湘軍為基地，大量搜羅湖南人才。他利用朝廷任命自己為湖南團練大臣的機會，「別樹一格」，一步一個腳印地組建起了忠於自己的湘軍。

曾國藩十分重視選用知識分子為將，如他的三個統帥（胡林翼、左宗棠、江忠源）都是當時著名的知識分子。

湖南大批知識分子進入湘軍，分別成為湘軍的高級將領或幕僚，以曾國藩為統帥，追隨其左右，為湘軍的建立和發展貢獻了畢生的精力。

左宗棠

不僅如此，在曾國藩的主導下，湘軍中的將領、士兵均可因指揮得力、作戰勇猛得到軍功並不斷陞官。

僅十多年時間，湘軍將領中升至總督的湖南人就有十人，包括左宗棠、彭玉麟、曾國荃、楊岳斌、楊昌浚等；升至巡撫的湖南人有十三人，包括胡林翼、郭嵩燾、江忠源、劉蓉、李續宜等；至於升至總兵、提督的，多達一百多人。

李鴻章

曾國藩最得意的弟子，便是李鴻章。曾國藩在去世前的一段時間，曾與李鴻章見面，並且叮囑他，一定要注意培養後人，承先啟後。

李鴻章詢問何人可堪大用，曾國藩說出了五個名字——左宗棠（文襄）、彭玉麟（剛直）、郭嵩燾（無謚號，可謂「遠識」）、劉長佑（武慎）、沈葆楨（文肅）。以現在的眼光看來，他們都是頂尖人才，亦是國家棟樑。
