中國女皇武則天以外，看看世界史上還有哪些著名女王與王后

天天要聞
2019-12-19

歷史上成功的女性不少，但富有傳奇色彩的女王卻不多。除了大家熟知的武則天，我們來跟隨外國的brainberries.co網，盤點一下世界歷史上還有哪些「活成傳奇」的女王。圖為電視劇中的武則天，其實女王跟女皇還有區別。

當然，目前最著名的就是英國現任女王，伊麗莎白二世。 現年93歲的英國女王伊麗莎白二世已在位66年，堅守自己登基時「窮一生為國民服務」的承諾，深受國民的尊敬愛戴。

維多利亞女王。 維多利亞女王是伊麗莎白二世的曾曾祖母，也是英國歷史上在位時間第二長的君主。她18歲就即位，用64年一手打造了「維多利亞時代」和大英「日不落帝國」的輝煌。

3. 葉卡捷琳娜大帝。 葉卡捷琳娜大帝是俄羅斯歷史上唯一一位有「大帝」之名的女皇,是俄羅斯皇儲彼得的妻子。她喜愛政治哲學，主張法治，政績顯赫，是俄羅斯人心中的一代英主。

4. 埃及豔后。埃及豔后是古埃及歷史上一位傳奇的女法老。據傳她長相美艷，相繼迷住了羅馬帝國的君王凱撒和將軍安東尼，保全了王朝，後戰敗自盡結束了富有戲劇性的一生。

5. 章西女王。章西女王原是印度貧民，生性勇敢，嫁入皇室後又遭英國殖民者入侵。 她為保衛章西披甲上陣，引領了印度民族運動，最後戰敗，英年早逝，她的勇敢和智慧流傳後世。

6. 瑪麗·安托瓦內特。瑪麗·安托瓦內特是法國國王路易十六的妻子。她沒有政治天賦，喜歡舞會，被人民貼上了「放蕩」的標籤，激起了動蕩的法國大革命，最後被處決。她詮釋了法國版的紅顏禍水，但在英文中王后與女王都是queen，因此她也算做一位。

7. 納芙蒂蒂。納芙蒂蒂是埃及歷史上一位重要的王后，嫁給埃及法老阿肯那頓後與他一同執政，自創太陽教，推崇一神論，在30歲左右便在歷史中再無記載，為後世留下了一個謎團。

以上七位女王或是王位繼承人，或是嫁給王位繼承人的普通人，卻同樣深刻影響了一個時代，為歷史增添了濃墨重彩的一筆。本文選題：金戈帝企鵝。編輯：白米飯。
