天道TD:陝西、河南自古淵源深厚，一文說清兩省關係

2019-12-09 0條評論 19次閱讀 0人點贊

摘要

陝西、河南自古淵源深厚陝西、河南自古淵源深厚，交往密切。陝西，河南是什麼關係呢？我們還是從頭說起吧。過遠的歷史難以考證，就從史學界公認的我們的人文始祖黃帝老人家說起吧。當然，這一切並不是天道的研究成果，天道今天只是一個搬運工而已。

陝西、河南自古淵源深厚陝西、河南自古淵源深厚，交往密切。陝西，河南是什麼關係呢？我們還是從頭說起吧。

過遠的歷史難以考證，就從史學界公認的我們的人文始祖黃帝老人家說起吧。當然，這一切並不是天道的研究成果，天道今天只是一個搬運工而已。

目前公認有上古兩個較大的文明，一個是以陝西為代表的華夏文明，一個是以山東泰山為代表的東夷文明（荊蠻文明主體在南方與本題無關略過）。兩大文明從黃帝期間到西周的相爭，我們可以用史學家傅斯年先生都論述一語帶過。傅斯年名著《夷夏東西說》中論述：「三代及三代前，古族有東西二系，夏與周屬西系，夷與商屬東系。殷周間的劇烈變革「蓋民族代興之故」，是兩個民族，兩種文明之間的交替。源於陝西的華夏文明最終取代夷商文化，重振夏文化風氣，從而成為中華文明的奠基者！」從此源於陝西的華夏族走出關中，走向了神州大地。我們下來看看具體的陝西河南的關係。

西周初期，建都豐鎬的西周王朝為了控制廣袤的東部地區。除了將其姬姓宗族大量分封之外，首先派遣周人周公姬旦，營造了雒邑也就是今天洛陽，並且駐紮周人的軍隊既殷八師。從而開啟了這座古城的歷史！

西周滅亡後，周平王東遷，又將部分陝西關中父老遷至洛陽，使洛陽成為春秋戰國時期東周的國都。陝西人所建！

我們在看春秋戰國時期河南境內主要諸侯國的來源。

鄭國:鄭國（公元前806年—公元前375年），國君姬姓。主要的版圖位於今天的河南省中部，其為陝西老鄭人遷徙而至！陝西人所建！

衛國:衛國，周朝的諸侯國，第一代國君為周文王嫡九子康叔封。。衛國疆域大致位於黃河以北的河南鶴壁、安陽、濮陽，河北邯鄲和邢台一部分，山東聊城西部、菏澤北部一帶。為陝西人所建！

曹國:曹國，周代諸侯國，國君姬姓、伯爵。周文王嫡六子曹叔振鐸分封於曹國，建都陶丘（今山東省菏澤市定陶區），疆域大致轄今山東省菏澤、聊城市，河南濮陽市南部地區。陝西人所建！

宋國:西周初期，周公旦輔佐周成王平定三監之亂，遵循「興滅繼絕」的傳統，於公元前1114年封商紂王的兄長微子啟於商朝的舊都商丘，建立宋國。東夷族，陝西人幫其立國！

蔡國（公元前1046~公元前447年），周朝諸侯國，建都蔡（今河南上蔡縣），始封之君為周武王之弟叔度，因叔度跟隨武庚反叛，被周公放逐於郭鄰。蔡叔度死後，周公旦封其子蔡仲（名胡）於蔡（今河南上蔡縣），重建蔡國。陝西人所建！

陳國:陳國（公元前1046年[1]—公元前478年），出土金文資料作敶。是西周至春秋時期的周朝諸侯國，國君媯姓，是虞舜後裔。周武王將長女太姬（大姬）嫁給了從商投降後擔任周文王陶正一職的遏父之子媯滿，備以三恪，奉祀虞舜。山西人所建（媯姓出於山西）！與陝西人姻親。

戰國時期河南境內兩大諸侯國為魏國與韓國。

魏國:魏國（公元前403年—公元前225），周朝諸侯國之一，也是戰國七雄之一。姬姓，魏氏。姬姓宗族源於陝西！

韓國:韓國（公元前403-公元前230）周朝的諸侯國之一，是戰國七雄之一，與魏國、趙國合稱三晉，國君為姬姓韓氏。姬姓宗族源於陝西！

韓、魏兩國，創始國君祖上世代均為晉國公卿，說是山西人所創也可為一說。

春秋戰國之後，隨著秦漢兩代幾百年的大統一。原諸侯國各民眾基本各安其所，雖然歷代不乏遷徙之事，但比起西周分封之華夏族大遷徙來講，基本格局是沒有變的。

陝西，河南的關係基本如此吧。各位說是什麼關係呢？
