當年《唐太宗李世民》片頭曲一響起來，就開始糾結李世民愛哪個？

天天要聞
2019-12-21

李世明（599-649年）唐高祖李淵次子，是中國歷史上少有的一位文治武功兼備的封建帝王。

現代有很多關於李世民的影視作品，其中最出名就是1994年播出的《唐太宗李世民》。

「沉魚落雁，閉月羞花，美的無處藏！…待我拱手河山討你歡，萬眾齊聲高歌千古傳…」這首片頭曲一想起來，很多人都會回憶起電視劇中的情節，不記得是多少年前一直印在大家記憶深處的歌詞，想起來後就搜索《唐太宗李世民》這部劇，聽到再次想起的音樂，很多人都對這部劇——《愛不釋手》（《唐太宗李世民》片頭）

這部劇是1993年拍攝的古裝歷史劇，主要講述了大唐開國的歷史，時間跨度由隋朝末年至貞觀二十三年，該劇前半程的主線是隋唐更迭，群雄割據，後半部就變成了李氏三兄弟爭權奪位，李世民開大唐盛世，其中距離圍繞李世民的感情線就有三條！

1994年這部劇從台灣引進大陸，不知道擊中了多少人的少女心。

看似是權謀劇，愣是讓人看成了偶像劇。

畢竟那個時候主演李世民的林俊賢意氣風發，劍眉星目，滿足了很多人對花美男的幻想，記得那個時候還有人在談論李世民到底愛誰？有說是吉兒的，有說是長孫皇后，也有說是阿史那燕的，更有甚者說是武才人的，爭論的熱火朝天。

其實劇中最難忘的還是吉兒和李世民之間的點點滴滴，第一集里，吉兒與李世民初見：李世民一箭穿喉射鳥，鳥落到吉兒公主的馬車前，李世民尋箭而來，吉兒諷他年少氣盛，刻意炫耀，李世民絲毫不惱，反而戲言她若美如天仙，必娶她做第二個老婆！

吉兒大怒，臨別時，還是忍不住掀起車簾回望，這一望，恰與李世民的目光交接。

這應該就是傳說中的「一見鍾情」吧。

這個回眸讓李世民驚為天人，從此也讓電視劇前的我們念念不忘，正是那「回眸一笑百媚生」的一見鍾情，不經意間的邂逅，卻註定了一世的傾心相愛；一個，英俊瀟洒，優雅復古，溫暖踏實，當之無愧的風度翩翩李世民；一個，國色天香，貌美如花，嬌俏動人，當之無愧的沉魚落雁楊吉兒。

但是作為皇帝，李世民一生不可能只有一個女人，說吉兒是他一生中唯愛的女人，確實有點不客觀，但是從始至終確實沒有人能取代吉兒在他心中的地位，李世民和吉兒的愛是情真意切，但是與長孫無垢又是另一種相愛相惜的感覺。

因為在感情上長孫無垢已經找到了自己的位置，她也知道吉兒先於自己，知道作為一個女人，有些摯愛深情只能由自己來承擔，聰明如她，所以她不會去強求。

當然了，正是長孫無垢這樣的包容和大度，換來了李世民最真切的敬重，所以她才是李世民唯一的皇后。

另一位是性子最烈的，阿史那燕在戰場上是李世民最得力的助手，在情場上她也不想認輸，總要和吉兒一爭高下，她的愛單純熱烈，但這種橫插一腳的任性，得不到觀眾的喜愛，不過對她，李世民也有情。

對於吉兒和李世民的感情，是人們最羨慕的，一個是明艷動人、明眸皓齒的隋朝出雲公主，一個是英武不凡、風度翩翩的平原李家二公子。
一箭定情，他狂妄的說：「如果你的聲音和你的人一樣美，我就娶你做我的第二個老婆。」
她隔著轎簾說：「就算天下男人都死光了，我也不會嫁給你。」

其實這部劇出彩的不只有演員和劇情，還有音樂。

台灣歌手李麗芬有兩首K歌金曲，除了大家循環的《愛不釋手》，還有一首就是《愛江山更愛美人》，那首隨著馬景濤版本的《倚天屠龍記》成名，兩首歌都是1994年問世，據說《愛江山更愛美人》這首歌，原本是為了《唐太宗李世民》而寫，後來被《倚天屠龍記》劇組橫道奪愛，編曲才臨時補寫了《愛不釋手》。

只能說，這兩首歌，現在聽來詞意、曲風都頗為相似。

而且《愛不釋手》這個片頭很明顯表達的就是吉兒的愛

片尾《愛上你給的痛》就是在映射三個女人痴戀的狀態。

真·歷史

李世民是一代有為之君，從公元627年開始的持續時間達23年之久的貞觀之治是中國歷史上最輝煌也是歷史學家最為稱道的盛世、清明之治。在初唐的政治、文化、美學環境中，李世民的音樂思想表現出特有的審美內涵，並以其封建帝王之尊自上而下地影響著這一時期的音樂發展。

李世民本人非常喜愛欣賞琵琶曲，還能熟練地演奏琵琶。

拋開電視劇，李世民一生共娶妻十五位，一位長孫皇后，十四位後宮妃嬪。李世民最愛的女人,當屬正妻長孫皇后。在現實中，長孫皇后才算是李世民的摯愛，畢竟當時長孫無垢才13歲就嫁給了16歲的李世民。

長孫無垢（600-636）河南洛陽人，生於隋文帝開皇20年，出身優秀，先祖為後魏獻文帝第三兄，姓拓跋式，後改姓長孫氏。

公元627年，李世民即位，長孫無污成了母儀天下的大唐皇后，但她仍與尋常百姓家的兒媳一樣，一如既往地孝敬公婆，對太上皇李淵照顧有加，每天早晚都會去請安，時時刻刻提醒宮女注意李淵的生活起居，作為一位掌握大權的正宮娘娘，長孫皇后不僅不爭寵，反而經常規勸皇帝要公正地對待每一位嬪妃。

唐太宗曾跟一位宮女生了孩子，宮女難產而死，長孫皇后就把這個孩子養在自己身邊，別的妃子生了病，她就把自己的藥品拿去給人家用，下面的人對長孫皇后的評價都很不錯，整個後宮都呈現出一種溫馨祥和的大家風範，這是在歷史上極為罕見的。

長孫皇后是中國歷史上一位傑出的女性，作為唐太宗的賢內助，不僅參與了唐初一系列的重要事件，還扮演了非同尋常的角色，發揮了無可替代的作用，而且對於成就唐太宗的人生功業做出了無與倫比的貢獻。

結語

歷史和電視劇總有些不同，其中我國自古以男權為主，以前的電視劇大多以男性的視角貫穿整部劇，雖然有不符合現實的情況，但是還是有真實性的。我國上下五千年的歷史，能拍的人物和故事太多了，五代十國、魏晉南北朝、秦漢、唐宋……
