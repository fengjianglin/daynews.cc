孝庄太后容貌復原圖出來後，網友紛紛感慨：難怪多爾袞願放棄江山

2019-12-14 481條評論 2,832次閱讀 117人點贊

孝庄太后大家都知道，這是一個有故事的女人，出身於蒙古博爾濟吉特氏，在清王朝剛剛建立起來的時候，滿清的時候和漢人比起來差別是很大的，僅僅是漢人的四分之一，所以想要把中原管理好，就必須和蒙古其他的部落成為盟友，因此清朝後宮有不少妃子都來自蒙古，甚至位分還很高。

孝庄太后本來教大玉兒，後來嫁給了皇太極，而她的姑姑就是皇太極的正室，後來的皇后，雖然說看起來比較亂，可事實就是如此，在少數民族這種關係也是常見的，不足為奇。本來大玉兒長的就很好看，加上十分的聰明，一開始嫁給皇太極也備受寵愛，不過海蘭珠出現之後，她的地位迅速下降，自己的親姐姐失去了丈夫，被皇太極所看上，而且還成為了皇太極心尖上的女人，到死都不能忘懷。

所以大玉兒也就不指望皇太極心能在自己這裡，她開始操心政事，而且在這方面很有天賦，海蘭珠去世沒多久，皇太極就跟著去了，而皇后之前也沒留下子嗣，生了三個女兒，只有大玉兒運氣好生了福臨，福臨也就順理成章成了皇帝，就傳說連多爾袞都對她十分傾心，也能看得出來大玉兒的魅力有多大了，在福臨當皇帝期間，也是多爾袞和孝庄兩個人掌握朝廷權利，軍權也都在多爾袞手裡。

所以很多人都好奇，孝庄太后究竟有多好看？要知道在沒嫁給皇太極的時候，她可是蒙古的第一美人，我們不清楚皇太極為什麼會看上守寡的海蘭珠，但肯定和大玉兒的容貌沒有關係，後來專家也用孝庄太后的頭蓋骨做了復原圖，盡量把孝庄的模樣還原到最真實，圖片出來後專家都驚訝了。

我們知道蒙古人給人的印象就是彪悍，瀟洒，因常年在外游牧，皮膚也不是很好，五官顯得粗獷，但孝庄太后完全不是這類型，反而十分的好看，臉上皮膚也是如此好，五官也很精緻，雖然眼睛看上去像上挑的，但仍然給人一種憐香惜玉的感覺，因此能被選到皇宮，還能讓多爾袞為了她放棄江山，果然是絕色。

當然，美貌並非是她最出色的地方，她和普通女人比起來最厲害的就是有心機，當初兒子為了一個妃子鬧得要出家，連皇帝都不想當的時候，她開始培養自己的孫子，把孫子教育好，然後扶持孫子上位，孫子後來也成為了千古明君——康熙。

從這裡就看得出來孝庄太后不是一般的女人，她的心胸和遠見甚至高於男人，所以才能在不被寵愛的情況下，帶著自己的兒子站在高高的位置上。
