有趣的歷史：戰國七雄！縱橫馳騁的各方霸主都是從哪來的？

2019-12-09 0條評論 22次閱讀 0人點贊

摘要

戰國七雄的名字大家都略有耳聞，他們處在一個非常有趣的中國歷史時期，時間在公元前475年—公元前221年，共計254年。這短短的200多年是在春秋百家爭鳴之後又一個歷史大變革時期，這時候的7大諸侯國也彷彿花叢中的花朵般爭奇鬥豔，各出奇謀。

戰國七雄的名字大家都略有耳聞，他們處在一個非常有趣的中國歷史時期，時間在公元前475年—公元前221年，共計254年。這短短的200多年是在春秋百家爭鳴之後又一個歷史大變革時期，這時候的7大諸侯國也彷彿花叢中的花朵般爭奇鬥豔，各出奇謀。既相互征戰，又相互利用，既縱橫制約，又聯盟抗敵，其中精彩絕倫的故事數不勝數，有妻離子散、悲歡離合，有殺伐果斷，攻城掠地，有忍辱偷生，反間離心，各個謀士雄才大展身手，熱鬧了200多年後才被秦朝一統。

戰國七雄

其實戰國七雄只是戰國時期比較大的7個諸侯國，當時還有福建附近的越國、四川附近的巴國、蜀國，以及夾雜7國中的宋國、中山、魯、鄭、衛等。七雄指的的是燕、楚、齊、趙、魏、韓、秦。當然經過數年征戰，哪些小國都被7雄所吞併佔領。

燕

周武王伐紂成功，把自己的弟弟分封到燕地成侯，也就是現在的遼寧地區。他弟弟姓姬叫姬奭（音同是），統治燕國是比較賢明，但身處北地雖然遠離中原戰場，但是發展有限。直到燕國派出荊軻刺秦王，燕國才出現在諸侯紛爭的舞台上，隨後被秦國報復滅國。

齊

齊國的來歷一說就比較傳奇，他的開國者叫做姜子牙。姜子牙輔佐周武王討伐商紂成功後，武王直接封了他為侯爵，賜封地為齊，也就是現在的山東附近。已姜子牙的智慧，將齊國治理的雄霸一方，兵強錢多，之後到齊桓公時代，更是達到富甲天下的程度。後來周朝封田和為侯，賜給齊地，姜姓才開始衰落。田姓掌管齊國後，不斷擴張，十分囂張的後果是被5國聯攻，隨沒滅國，但也大不如從前。

趙

趙國祖先也姓贏，與秦國貴族同根而生，本是商朝的大臣之後，商朝時就被封王封地，後來武王伐紂，這支贏姓因為助紂為虐被周武王嫌棄，逐漸衰落。誰知趙侯的後人抓住時機幫助周王平定了一次叛亂，周王才開始注重趙侯贏氏後裔，又分封趙地給他們，趙國的領土在山西附近，國力直到趙武靈王時期才走上巔峰，但趙武靈王玩了一票大的，把自己稀里糊塗玩死，隨後就被秦國滅掉了。

韓

韓國最早來自晉國的大夫，晉國發生內亂後，三大夫把原來的晉國瓜分了。韓家佔據不少土地，並且內外兼修，不斷發展。周王沒辦法，只能承認韓大夫為王，封河南附近的土地為韓國。韓國是七國里土地最少的，但是其出產非常厲害的弩，甚至傳聞其弩可以射出800步，遠處中箭都能射透皮甲致死。韓國還出產優質寶劍，傳聞韓劍可以輕易砍斷皮甲鐵盾，鋒利至極。但是再優秀的兵器也沒保住韓國，因為地小人少，又處在各大國包圍之中，早早就被秦國所滅。

魏

魏國是在周朝周威烈王時被封為侯，但是其與韓國一樣，同樣被其他諸侯包圍，土地為河南、江蘇、山東、山西部分地區。魏王深知自己國家的觸地，勵精圖變，推行明智的改革政策，雖然被環衛在中，但確逐漸強大，其他國家不敢與之為敵，一直持續了170多年才被秦國所滅。

秦

秦國最早的開端者叫做非子，他姓贏，也就是嬴政的贏。非子是上古部族的後裔，他們族人一直都善於養馬，對於駕馭鳥獸有非常高超的本領，尤其是非子養馬的手藝更是高超。周孝王時期，為了振興朝政，抵禦北方外敵，所以極為注重飼養戰馬，在甘肅地區還建有皇家馬場。有一次，周孝王巡視馬場，聽說一個叫非子的人養馬非常厲害，他養出的馬健壯善跑，還不生病。周孝王招他來見，一問確實對養馬知識了如指掌，立刻讓他負責整個牧場的養馬工作。這非子果然沒有令人失望，幾年後牧場出產的馬匹個個健壯，周孝王大喜。為了表彰非子，周王將兩片連綿的牧場（甘肅附近）分封給了非子，讓他成為周的附屬國，世代養馬供給周朝。自此秦國有了土地和基礎，更有了兵強馬壯的實力。

楚

楚國地處長江流域，在春秋時期就以成為諸侯國，因其地理優越，國土面積逐年擴張，國勢壯大。在楚莊王時期，楚國進入鼎盛，後在戰國時期更是吞併越國，國土面積激增。整個楚國歷史長達7-800年，可謂歷史悠久。但再強大、底子再厚也沒打過秦國，最後被嬴政指引的黑色鐵騎踏穿數百年基業。縱觀戰國七雄，都是始於春秋、周朝，但最後誰也沒有逃脫被秦國統一的命運。不管是獨善其身，還是兼攻天下，秦國黑色的鐵騎、鐵盔上紅色的長纓走遍了他們的領土，從此揮之不去，戰國時代存世200餘年，終歸統一。
