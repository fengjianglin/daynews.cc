廬陵王劉義真之死真有那麼簡單？側面解讀當時士族與寒門生存狀態

2019-12-19 1,375條評論 3,436次閱讀 1876人點贊

南朝第一雄主宋武帝劉裕在位不到兩年便去世，之後又不到兩年的時間就發生了輔政大臣廢廬陵王劉義真為庶人的政治事件，旋即又「使使誅之」。在看似平常的藩王被誅背後，卻隱藏著從東晉末年到劉宋初年的兩股政治勢力，士族與寒門，為爭奪政治權力而開展的政治鬥爭，也從一個側面揭示了當時士族與寒門的生存狀態。

一、劉義真之死事件

劉裕在去世（公元422年）前為穩固政權、輔佐少帝劉義符，效仿前朝的託孤故事，委任司空徐羨之、中書監傅亮及領軍將軍謝晦輔政。景平二年（公元424年）徐羨之、傅亮等以「陷關中害眾將」、「荒法悖禮肆口縱言」及「潛懷異圖、希幸非冀」意欲謀反等罪名以少帝名義下旨廢劉裕第二子廬陵王劉義真為庶人，發配新安郡。隨即劉義真被徐羨之等人下令處死。

劉義真之死並不是孤立發生的，在劉義真死後緊接著發生了少帝被廢、旋即少帝被殺、迎立劉義隆為帝等諸多事件。劉義真之死可以看做是後來一系列政治事件的導火索或者敲門磚，為其後的諸事件埋下了伏筆。

二、劉義真被殺的真實原因

廬陵王劉義真年少得志，作為次子，因「儀貌俊美、神情秀徹」深受宋武帝劉裕喜愛。但隨著劉裕去世，劉義真被外放並架空實權。在劉義真被殺後，少帝劉義符也緊跟著被廢及被殺。徐羨之等朝臣迎立了劉裕第三子劉義隆為帝，即宋文帝。就這一系列事件來說劉義真被殺有其深刻的內在原因和標誌作用。

1、內在原因：劉義真的被殺是為廢除少帝另立文帝做準備

當時少帝劉義符因先皇駕崩時居喪無禮、好為輕浮游狎之事、不聽群臣諫言以及視軍國大事如無物等原因引起群臣不滿，在輔臣徐羨之、謝晦、傅亮以及地方軍事大員檀道濟、王弘等人的策划下準備廢黜少帝另立新君。

而對於新君的人選在大臣之間產生了嚴重的分歧。徐羨之、傅亮等人矚意先帝第三子荊州刺史劉義隆，而謝晦等人以「兄終弟及、長幼有序」的理由推薦第二子廬陵王劉義真繼承帝位。於是徐羨之等人搶先一步，矯詔廢黜了劉義真的王位又將其殺害，斷絕了謝晦等人的企圖。可見劉義真被殺就是廢黜少帝、另立劉義隆等政治事件的一個關鍵環節。

2、標誌作用：劉義真之死標誌著士族和寒門兩股政治勢力較量的勝負

謝晦等大臣在廢黜少帝之前提出擁立劉義真並非是因為他們的理由「長幼有序」那麼簡單。其後背也隱藏著謝晦等人深刻的政治目的，那就是利用劉義真與士族集團的關係來扭轉士族階級自東晉末年的頹勢，以圖在政治中心重新確立士族的地位。而劉義真的被殺就是士族在政治較量中失敗的標誌。

三、劉宋初期的士族與寒門兩大政治勢力

1、劉宋初期的政局

東晉末年，一直在政治、軍事中佔主導地位的士族階級因內部消耗、腐敗蛻化等原因失去了實際權力，只有少數較有才能者還保持較高官階和部分權力；而新晉崛起的寒門庶族則通過軍事才能迅速掌握軍權，繼而佔據了政權的主導。

劉裕就是寒門庶族的典型代表。他通過長期的軍事鬥爭控制了東晉的軍權，又利用軍權控制了政權。在與新舊各方政治勢力達成了平衡之後，最終終結了東晉，建立了劉宋王朝，開啟了南朝的歷史。

劉宋的建立可以說是士族與寒門之間鬥爭、妥協的結果，而劉裕為了進一步壓制士族勢力而採取了一系列措施：政治上加強中央集權，重用寒門；經濟上採取「土斷」抑制豪強兼并；教育上扶持各地方秀才。通過諸多手段終結了門閥專政的局面，形成了門閥士族與寒士庶族相互牽制、保持平衡的政治格局。

2、士族與寒門

從劉裕去世前安排的輔政大臣里就可以看出當時劉裕的煞費苦心：謝晦是名臣謝玄之孫，王弘是王導曾孫。這兩個人一個在中央內部，一個外放地方大員，都是士族階級的首領人物。

「昔日王謝堂前燕」的兩大家族，此時仍然代表著士族勢力；徐羨之、傅亮和檀道濟則或是士族卻「世無顯宦」、或是「先輩無聞」的小戶出身，全部是憑藉著參與了劉裕的軍事鬥爭及政治活動而獲得了劉裕的信任才身居高位的。他們都是寒門的代表人物。

劉裕安排了由士族和寒門共同參與、組成的輔臣集團是經過深思熟慮的：在中央的政治勢力上，士族的謝晦和寒門徐羨之、傅亮保持了平衡；地方的軍政大員上士族的王弘位於蜀中，寒門的檀道濟則在淮南，都是軍事要地，確保了雙方的穩定。在這種平衡而穩定的政局下，通過雙方的牽制、拉引也確實可以保證其子少帝劉義符的帝位安穩。至少可以保證劉氏的傳承。

3、廢黜少帝：士族與寒門的「同」與「異」

在少帝繼位不到兩年的時候，這位少年天子就引起了整個朝臣的不滿。如果說只是因為「居喪無禮」和「好為輕浮游狎之事」，大臣們是會安心繼續輔佐的。因為一個沒有德行服眾且貪圖安逸、不務正業的皇帝正是大臣們趁機把持朝政的最佳人選。但少帝因為對軍國大事的漠不關心達到了北魏大兵壓境屢失要地也不管不問的地步就引發了全體大臣的一致不滿了。

寒門和士族都曾經在軍事上直接與北方的「胡虜夷狄」長期鬥爭，雙方可謂是血海深仇。無論是為了在文化上的「保華夏衣冠之正統」還是出於對於覆滅後被清算的懼怕，他們都必須和北魏鬥爭到底。

於是當寒門和士族在對少帝規勸諫言之後發現少帝並沒有絲毫改悔的時候，廢黜少帝另立新君就成為雙方一致的政治目的和保存自身的手段了。但是在被擁立新君的人選上士族和寒門產生了分歧，士族希望能夠擁立廬陵王劉義真為帝。這也為劉義真的被殺埋下了隱患。

四、劉義真被士族擁護的原因和士族的失敗

1、劉義真的士族政治屬性

劉義真能夠得到士族的支持是有其政治原因的。首先劉義真身邊圍繞著大量門閥士族的幕僚，使其深受門閥士族的影響。《宋史》記載：「義真聰明愛文義，而輕動無德業。與陳郡謝靈運、琅邪顏延之、慧琳道人並周旋異常，雲得志之日，以靈運、延之為宰相，慧琳為西豫州都督」；「義真意色不悅，侍讀博士蔡茂之問其故」。

無論是謝靈運、顏延之還是慧琳、蔡茂之都是門閥士族出身，謝靈運甚至就是謝氏家族的成員。劉裕曾經有過立劉義真的想法，而劉義真也確實表示過繼位之後將重用士族的想法。

其次劉義真得罪過寒門軍功集團。劉義真年少鎮守關中時曾經利用過寒門軍功集團內部的鬥爭而獨攬大權，先後有沈田子、王修、王鎮惡等人被誅殺。雖然其背後有劉裕為平衡政治勢力而挑起多方矛盾的心意和手段在其中，但劉義真也因此被寒門集團所仇視。後來劉義真的罪名中也有「陷關中害眾將」一條。可見劉義真如果繼承帝位將對寒門起到極大的打擊。

第三劉義真受士族文化影響崇尚清談。圍繞在劉義真身旁的謝靈運、顏延之、慧琳等人都是當時的「名士風采」，以談玄而聞名於世。劉義真受他們影響也對談玄十分喜愛，經常與身邊人清談。

可見劉義真在當年引起劉裕注意的「今日之事，誠無算略。然丈夫不經此，何以知艱難」、「安不忘危，休泰何可恃」即是清談的表現形式，而對謝靈運等人「為宰相」的許諾就是清談的習慣，成熟的政治人物絕不會在當時的情況下於大庭廣眾時輕易表露這種想法。

可以說劉義真的種種言行和早年事迹決定了他的政治立場和政治基礎都來自門閥士族，而劉義真在少帝即將被廢黜時被士族看做擁立的最佳人選也是理所當然的。士族以「長幼有序」的禮法作為理由推舉劉義真也是要從道德和宗法的制高點上壓制住寒門，以達到他們擁立劉義真打壓寒門重新奪回政權主導權的目的。

2、士族與寒門政治鬥爭失敗的原因

第一，無論是劉義真還是圍繞在其身邊的士族成員都在政治上極不成熟。劉義真多次對謝靈運等人提到「得志之日，以靈運、延之為宰相，慧琳為西豫州都督」等，可以視為對士族的拉攏、利用，但如此機密要事竟被劉義真、謝靈運等人弄得「眾聞之俱驚」，其後還「義真與靈運、延之、慧琳等共視部伍」，暴露了野心，引起了政敵徐、傅等人的警覺。

第二，劉義真與士族過早地爆發了與少帝的矛盾而被寒門利用。廢黜少帝的計劃在寒門與士族的高層之間早已籌備，而謝靈運等人此時就按捺不住慫恿劉義真「表求還都」，企圖在政治鬥爭中達到「近水樓台先得月」的效果。卻被徐、傅等人利用，「因其與少帝不協，乃奏廢之」。輕易的破解了劉義真及士族的計劃，並藉此「廢義真為庶人，徙新安郡」，徹底剝奪了劉義真的政治資源。

第三，士族手中沒有可以利用的軍權。劉義真在被徐、傅等寒門攻擊的時候士族只有吉陽令張約之這種官微權小的成員「上疏諫曰」，無疑是極其失敗的。隨後張約之被貶又被殺完全是起到了給殺劉義真做演習的效果。因為士族手中沒有軍權導致他們不能像寒門後來廢黜少帝時「徐羨之、檀道濟等人帶兵殺入宮闈」，也不能有效的保護劉義真的安全致使其被「使使殺之」。劉義真和士族當時最正確的方法就是召集蜀中的王弘帶領軍隊進京，至少可以保證雙方的大致平衡。

結語

劉義真的被殺背後隱藏著士族與寒門政治鬥爭。門閥士族企圖利用少帝即將被廢黜的計劃擁立劉義真，以達到一改「晉末之頹勢」、重新掌握政權主導的效果，卻因為種種原因而失敗。

劉義真的被殺也代表著士族已經完全失去掌握政權的基礎和沒有可以任用的人才了，面對著寒門崛起的歷史浪潮只能「無可奈何花落去」，漸漸迎來被淘汰出歷史舞台的命運。
