古代想打光棍太難了-國家發對象

中國古代有光棍嗎？當然有！比如花木蘭，據說她單身的原因是被皇帝看上，她自殺拒絕了，皇帝只好追封她將軍，謚號「孝烈」；又比如寫《滕王閣序》的王勃恃才傲物，認為沒女人配得上他。問題來了，為啥古人圈子那麼小，光棍人數卻很少？說出來你得相信，那時候，國家「包分配」對象！

如今打光棍父母急，古代是父母急，朝廷更急！周朝開始按戶收稅、服徭役，不結婚影響國庫收入、軍隊和民工人數；而且古代戰亂頻繁、生育率低、死亡率高……政府幫光棍娶親，都有「催婚一條龍服務」。地方設「媒氏」，相當於「國營婚姻介紹所」；先秦時，政府每年召開大型相親會，搞點祭祀、畔浴等活動，吸引單身青年參加，當天不限制私奔、幽會，開放程度超乎想像。

這麼幾板斧下來，還有一些光棍，那是爛泥扶不上牆：此時，政府「貼心」給警告，罰款！漢高祖劉邦收「單身稅」，他兒子劉盈上任後，又把賦稅提高好幾倍，打光棍，日子更難過！

南北朝時，女子15歲不嫁，一人單身全家坐牢。所以在剩女眼裡，同齡單身青年，等於「香餑餑」。古代也規定結婚年齡，「男三十而娶，女二十而嫁」，注意！這不是法定結婚年齡，而是最晚結婚年齡，假如超過了…… 我是個女人 但可能這輩子都嫁不出去了

你想孤獨終老？沒門！晉朝司馬炎規定，女子17歲不嫁，官方給你配老光棍；管仲發明「合獨」，「取鰥寡而合和之，予田宅而家室之」，就是朝廷出面，硬湊孤男寡女，還給田地宅子讓你成家，國家都給你臉面了，說，你還敢單身嗎？

有人就開心了，咱們穿越回古代，國家分配對象，多省事！我勸大家別想了，第一，不是所有光棍，國家都關照，你穿越過去，肯定沒戶口吧，等於流民、野人，一不小心被抓了要當奴隸，這些人，國家統統不管；第二，包分配是包分配，可是國家不考慮你對象的長相、年齡、人品、學識、愛好，更不會考慮是不是真愛。你一掀蓋頭再後悔，遲了！怎麼樣，是不是頓時覺得，單身也很幸福呢？
