曹操為何放棄有“山河之險”的洛陽，而選擇定都小小的許昌？

天天要聞
2019-12-04

摘要

公元25年6月，劉秀稱帝，出於鉗制其部下河北諸雄的考慮，定都洛陽而非西漢舊都長安。隨後的兼并戰爭中，劉秀在群雄爭霸中拔得頭籌，再次一統天下，恢復漢家江山，因洛陽在長安之東，被後世稱為東漢。從此，洛陽作為東漢的政治、經濟、文化中心，在華夏歷史上發揮了重要的作用。

公元25年6月，劉秀稱帝，出於鉗制其部下河北諸雄的考慮，定都洛陽而非西漢舊都長安。隨後的兼并戰爭中，劉秀在群雄爭霸中拔得頭籌，再次一統天下，恢復漢家江山，因洛陽在長安之東，被後世稱為東漢。從此，洛陽作為東漢的政治、經濟、文化中心，在華夏歷史上發揮了重要的作用。但是在東漢末年，曹操派人從長安迎回漢獻帝，卻沒有選擇順勢再次定都洛陽，而是將國都遷至名不見經傳的許縣，這是為什麼呢？

東漢末年洛陽與許昌建都比較

建安元年八月，曹操在謀士荀彧的建議下，決定迎接漢獻帝，「奉天子以令不臣」。不久後，漢獻帝封曹操為司隸校尉，錄尚書事，遷都許昌。從此之後，曹操利用天子的政治優勢，逐漸剪滅北方群雄，統一了北方。但是，令人不解的是，曹操為何不將天子迎接回東漢舊都洛陽，而是彈丸之地許昌呢？事實上，這和洛陽以及許昌的建都條件有著密切的關係。

一、洛陽雖有「山河之險」，但建都優勢消失

洛陽早在秦朝時期就是三川郡的治所，秦帝國的東西運輸幹線，秦直道之一的三川東海道便經洛陽而達關中，洛陽成為連接關中和關東的陸路樞紐。不僅如此，黃河北過洛陽，給洛陽帶來了水路運輸優勢，秦和西漢時期，全國各地的賦稅及朝貢物品經黃河水運網路或陸運到達洛陽，給養關中，使得洛陽具有天然的交通優勢。

此外，從地理位置來看，洛陽還具有優越的自然條件。洛陽地處伊洛谷地，三面環山，北倚邙山，東望嵩岳，西有秦嶺，伊、洛、澗、瀍四水環繞，再加上黃河天險，素有「河山拱戴，形勝甲於天下」的美譽。佔據洛陽，西可取崤函定關中，東可出虎牢爭奪天下，因此洛陽可以說是持險防禦、虎踞龍盤的龍興之地。

但是在東漢末年，對於曹操來說，此時的洛陽卻不適合定都。黃巾起義以來，天下大亂，群雄割據並起，洛陽飽受戰火摧殘。

靈帝之時，國家紛擾，各地軍閥混戰，國家風雨飄搖，東漢政權瀕臨瓦解。漢獻帝時，董卓攻佔洛陽，火燒洛陽城，致使百萬洛陽人民更相蹈藉，飢餓寇掠，積屍盈路。此後漢獻帝從董卓手中逃回洛陽後，洛陽宮室殘破，街陌荒蕪，再也沒有昔日的繁榮景象，東漢帝業毀於一旦，建都的優勢早已消失。

二、定都許昌的地理、交通優勢

在中國古代交通通信落後的情況下，定都中原腹地可以起到「臨制四海」的作用。天下以河南為中，而許縣又是河南腹地，北限大河曾無潰溢之患，西控虎牢不乏山豁之阻，南通蔡、鄧實包淮、漢，如此看來，許縣其實也是天下形勝之地。

「州西控汝、洛，東引淮、泅，舟車輻集，轉輸易通，原野寬平，耕屯有賴。曹操挾天子於此，北並幽、冀，南抗吳、蜀。」

許昌地處汝州、洛陽、淮、泅的中心地帶，交通極為便利，陸地交通發達方便車馬轉運，而輻射四周的水運網路更是方便軍用物資的運輸。此外，許昌地勢平坦便於農業生產，定都於此為曹操北定中原，南抗吳蜀提供了重要的戰略支點。

在地理位置上，許昌位於豫西高原，同樣是三面環山，西有伏牛山和白雲山，西北有嵩山山脈，南有大別山為屏障，易守難攻。更為重要的是，在各地郡縣飽受戰亂的情況下，許昌卻少有戰火波及，人口和資源保存的較為完備，方便曹操進一步發展壯大。

曹操定都許昌，除了歷史地理背景，還有著更為深刻的現實因素

一、洛陽為漢室根基，不便曹操發展自身勢力

洛陽從東漢立國以後，當了一百多年的首都，所以一直以來都是漢室重臣的老巢，在洛陽，漢室雖然由於戰亂被削弱，但仍有一定的根基。如果定都洛陽，漢獻帝就是堂堂正正的君主，那麼對曹操來說則難以控制，不利於自身的發展。

此外，當時的曹操雖然在山東平定了黃巾軍，勢力開始壯大，有了一定的軍事基礎，但是在名義上仍然只是漢獻帝的鎮東將軍。這樣的官職在高官雲集的洛陽根本不算是大官，再加上洛陽本不是自己的勢力範圍，曹操如果「挾天子以令諸侯」，很可能會引來一心向漢的勢力的反撲很可能會得不償失。

最後，在當時的人們眼中，漢室還是正統，洛陽在天下士人以及忠於漢室官員的眼中，地位依然崇高。如果曹操定都洛陽，不僅遠離自己的勢力中心，還處於天下人的眼皮之下，稍有不慎便會招致讀書人的口誅筆伐，不僅不能起到「奉天子以不臣」的效果，說不定還會被群起而攻之，帶來意想不到的反作用。

退而求其次，如果定都許昌，這樣既靠近洛陽，又處於曹操勢力的控制範圍，不僅不會遭到天下人的非議，還能夠吸引人才。東漢末年，潁川群為名士之鄉，潁川謀士是曹操謀士群體的重要組成部分，在曹操的決策核心層中居於主導地位。

漢獻帝到許昌後，由於許昌既是新都，又是故鄉，因此大批在外避亂的穎川名士紛紛返回故鄉，「是時許都新建，賢大夫四方來集」。良好的政治環境以及曹操不拘一格的用人方式，吸引大批名士為曹操出謀劃策，為其統一北方聚集了大批的人才基礎。

二、許昌地處中原腹地，方便屯田，增強經濟實力

東漢末年，由於各地軍閥連年混戰，天下大亂，民不聊生。許多軍閥因為缺糧而被消滅，官渡之戰，袁紹百萬軍隊因為糧草被燒，很快就被曹操消滅，袁術更是因為糧草缺乏而最終命喪黃泉。所以，糧食問題便成為制約軍隊勝負的關鍵因素。

糧食生產是保證戰爭的重要因素，曹操深知這一點，因為他曾因軍隊糧食短缺而被打的節節敗退，「自遭荒亂，率乏糧谷，諸軍並起，無終歲之計，飢則寇略，飽則棄余，瓦解流離，無敵自破者無可勝數。」迎漢獻帝到許之後，曹操便充分利用許都周圍的有利的地理條件，大興屯田，發展糧食生產。

「是歲乃募民屯田許下，得谷百萬斛。於是州郡例置田官，所在積穀。征伐四方，無運糧之勞，遂兼滅群賊，克平天下。」

許下屯田是東漢末年曹操在統一北方過程中採取的一項重要措施，為其軍隊提供了充足的糧草，它成功地解決了曹操集團糧食供應的問題，也為曹操集團在其它的地區的屯田打下了基礎，為曹操「摧滅群逆，克定天下」奠定了堅實的物質基礎。

三、許昌在曹操的控制之下，與四方諸侯有足夠的緩衝空間

洛陽居天下之中，地勢雖然險要，但卻為兵家必爭之地，在群雄割據的背景下，洛陽在當時已經是四戰之地。此時的洛陽強敵環伺，四面八方都是前線，洛陽以西，李傕、郭汜控制關中，虎視眈眈，張楊佔據河內，對漢獻帝也心懷鬼胎，張綉盤踞在南陽，也打著自己的小算盤。

洛陽是各路諸侯眼中的香餑餑，誰都想佔據這個龍興之地，但都怕成為眾矢之的，被群起而攻之。曹操如果定都洛陽，不僅需要調動大量的人力物力進行防守，還要防備有人奇襲洛陽，如果漢獻帝被劫走，對於曹操來說豈不是功虧一簣。

此外，曹操在迎漢獻帝時困難重重，多方周旋才得以解決，現實因素使他必須選擇其統治穩固的地方來安置漢帝。許昌向南汝南儘管受到袁術勢力的影響，但卻是曹袁兩股勢力的緩衝帶。向北則只有遙遙相望的河內太守張楊，而黃河天險的阻隔是一個有利的天然屏障。所以，許昌是曹操勢力所能控制的最有利的地方，並且穎川的地理環境，使這裡在諸侯紛爭的過程中，成為無主之地，成為其定都的最佳位置。

經過數年的發展，許昌逐漸在曹操的控制之下，並且他的大本營兗州還可以充當許昌的保護傘。更為重要的是，曹操擊敗了中原地區的黃巾軍後，許昌的周邊地區都被曹操佔領，威脅基本掃除。就算四周強敵想要奇襲許昌，也有足夠的緩存空間，為曹操調兵遣將積極布防爭取到一定的時機。在這樣的情況下，曹操可以放心的對外征戰。

定都許昌之後，為了便於對外征伐，東漢的政治中心北移到鄴城，但許昌仍發揮著重要作用

曹操迎漢獻帝遷都許昌後，在許都屯田，成功解決了軍隊物資供給問題。但是許都所在的潁川群處於中原核心，屬於四戰之地，周圍都是敵對勢力，如何選擇用兵對象，選擇主攻方向成為了一個棘手的問題。即使在消滅呂布、袁術和張綉等軍閥，並於官渡之戰獲得大勝之後，曹操在北進還是南征的戰略抉擇上仍然屢次發生動搖。

後來荀彧進言，袁紹戰敗，士卒離心，應該趁機進攻河北，機不可失。曹操接受了荀彧的主張，回師河北並最終平定袁紹，攻佔了袁紹的都城鄴城。此時，曹操的目的是清除袁紹在幽、青、並的殘餘勢力，統一北方作為根據地，繼而平定全國，鄴城的重要性便開始體現出來。

鄴城位於我國北方的道路要衝，背靠山西高原，面對經濟發達的華北大平原，地處太行山東麓南北交通大道，又是從滏口陘進入華北平原的要隘，是東西南北陸路交通的樞紐，戰略地位十分重要。

從軍事地理的角度來看，它的西邊有巍巍太行山脈，守住滏口陘即能阻擋山西高原的來寇；其南、東兩面有黃河天塹，鄴城南北又有漳、洹、滏水流過，起到阻礙敵兵行進作用。因此，曹操如果希望控制整個黃河流域，平定北方亂局，就得不依靠鄴城。

鄴城處在北方的中心，再加上物產豐饒，人口密集，所以適於充當黃河下游地域作戰的基地和指揮中樞。與之相比，許都距離戰場較遠，大軍在幽、並、青州戰鬥，若是仍以河南的潁川郡為根據地，部隊的來往休整、給養運輸和信息傳遞都需要長途跋涉，相當困難，所以不適應新的軍事形勢。

也就是說，曹操放棄南征劉表，確定繼續在黃河以北地區作戰以消滅袁氏殘餘勢力的用兵計劃，是促使軍事、政治重心轉移到鄴城的決定性因素。

為了平定北方，曹操將自己的統治機構「霸府」由許都遷移到鄴城，作為自己的軍政指揮機構，漢獻帝被架空後，實際權力基本掌握在曹操的霸府手中。隨後，曹操的宗親戚屬與重要將領的家小基本都遷移到鄴，軍隊主力及其家屬也安置在鄴城附近，鄴城由此成為名副其實的軍事、政治中心。

但是，在這一階段，許昌仍舊發揮了重要的作用。曹操迎漢獻帝遷都許昌後，借漢室威名，以正統之師征討天下，才能先後拿下徐州、淮南、向西攻下長安，向北擊敗袁紹，降服烏桓以及匈奴。可以說，如果沒有許都屯田所帶來的軍糧，沒有潁川謀士集團，曹操難以「摧滅群逆，克定天下」。

曹丕代漢後，曹魏政治、軍事中心南移，許昌的地位日益上升

曹丕代漢自立後，許昌的政治軍事地位開始上升。此時曹魏的對外作戰區域從華北變為江淮地區，戰線的南移和西移，使得許昌無論是在中軍遠征還是糧餉轉運方面都具有優勢，可以節省大量的人力物力。

曹丕和曹叡在位期間，明顯加強了對於孫吳的軍事進攻。從地理位置上來看，許昌地處洛陽東南，潁水可經過水網流入淮河，並通過淮河連接江南水系，南抵揚州，甚至能到達長江，這樣一條天然的水路通道具有天然的行軍優勢，許昌成為南下進攻孫吳行軍路線上無法繞開的樞紐。

正是因為許昌優越的地理位置，成就卓著的許下屯田，便利的交通水路，在曹魏定都洛陽後，它在軍事上的優勢依然無可代替，政治地位也因此而上升。魏文帝與魏明帝頻繁駐蹕許昌，就是因為該地是對吳征討的重要兵站。曹魏的中軍主力除了洛陽眾多的「河南之眾」外，還有許昌附近的眾多土家。

由此可見，許昌不僅在曹操取得政治優勢，統一北方方面發揮了重要作用，還為後來曹魏政權征伐東吳提供了重要的戰略支點，正所謂「魏受漢禪，以許昌為別宮，屯重兵，以為東、南二方根本」，許昌的重要性不言自明。

小結

我們在了解歷史時，往往只能管中窺豹，而不能洞察全局。曹操定都許昌，有著深刻的歷史地理以及現實因素，而許昌也在曹魏集團的崛起過程中發揮了不可替代的作用。曹操利用許昌屯田獲取軍需物資，以漢室之名吸引士人紛紛依附，一步步發展壯大，而曹丕以及曹叡將許昌作為對吳作戰的關鍵支點，也為後來魏晉滅蜀、平吳的獲勝奠定了堅實的基礎。但是後來，隨著經濟重心的轉移，帝都近畿政治優勢的喪失，漕運的荒廢，許昌也在歷史的長河中逐漸沒落。
