長知識！古代太子、世子、嫡子、庶子都有什麼含義

2019-12-25 121條評論 2,587次閱讀 622人點贊

中國幾千年宗法制度的核心就是世襲，始於夏朝。世襲制到周朝遂成定製，歷代沿襲，講究的就是子承父業、兄終弟及，嫡貴庶賤、男尊女卑，直至唐朝就成了定律。

元子

元子就是嫡長子，又叫宗子。「元」的意思是第一、開始，如元配夫人。朱元璋的「元」就是第一，璋是美玉，常代指男子。所以，朱元璋的名字就是天下第一美男子。「宗」指宗族，宗族的長子就是宗子。宗法制度規定，嫡長子乃第一繼承人，繼承其父的身份和財產。

太子

嫡長子雖是法定繼承人，但仍需在他頭上戴個光環，這就是太子，真正的皇位繼承人。太子不是尊稱，是繼承人，只能有一個。太子又叫儲君。當然，即便當了太子，也有可能給你取消。漢武帝長子劉據，唐太宗長子李承乾，最後都被削為平民，死的死、關的關，所以，嫡長子絕不能與太子劃等號，有時是皇帝的旨意，有時則緣於宮廷的博弈。

嫡長孫子

嫡長孫是第二繼承人。朱元璋的長子朱標死得早，卻生了五個兒子，後來登基的建文帝朱允炆，就是朱標的二兒子（長子朱雄英早死），他就是按序繼位的嫡長孫。據說有一次，朱元璋帶孫兒朱允炆去檢閱這些叔叔的部隊。朱元璋說，今後哪兒造反，就叫你這些叔叔去打。當時朱允炆就問了一句，要是叔叔們造反呢？朱元璋當即覺得腦殼被敲了一悶棒。後來果然是燕叔叔朱棣造反，即朱元璋第四子發動「靖難之役」打到南京，奪了侄兒的權。

嫡子

嫡子，即那些沒當上太子的同母親兄弟，趙匡義就是趙匡胤的同母的親弟弟。唐玄宗同樣不是長子，他上面有個哥哥，叫李成器。唐玄宗因平亂有功，李成器就主動讓賢，還說：「國家安則先嫡長，國家危則先有功」。好在唐玄宗後來很待見他，得以善終，死後還贈謚號「讓皇帝」。

世子

世子就是親王的嫡長子。皇帝的兄弟封王稱爵，稱作親王。親王的長子可繼承王位、爵位，這個就叫做世子。世子與太子有些相似，世子是王位、爵位的法定繼承人，而太子則是帝位的法定繼承人。不過，他們之間經常轉換，親王當了皇帝，他的世子就轉正為太子；皇帝被廢晾在一邊，太子就變回世子。李建成是唐高祖李淵的長子，李淵在隋朝封唐公，李建成當時是世子。李淵建立唐朝當了皇帝，李建成就變成太子。只不過李建成沒福氣，在玄武門被弟弟李世民幹掉了，否則，他就應該是唐什麼宗。

庶子

嫡一支立完，就輪到庶子。庶子又叫支子、介子，民間稱作孽子。介，副也，非嫡長子之妻為介婦，介婦之子便是介子。對於普通百姓而言，你沒明媒正娶就生子，這叫「造孽」，所以叫孽子。「嫡」對「庶」，「宗」對「支」，庶、支皆系家庭旁支，通指妾，她生的兒子就是庶子、支子、介子。

從子

從子又稱猶子，即兄弟姊妹之子，咱們現在叫侄兒。庶子這一輪之後，就輪到其他血緣相近的，從子便是其中之一。猶子之意就是「猶如兒子」，《千字文》里就有「猶子比兒」一說，足見其親。歷史上從子繼位的頗多，前秦有苻健侄兒苻堅繼位的先例，後晉有石敬瑭的侄兒石重貴，後周郭威的侄兒郭榮，清朝最後一任皇帝溥儀，就是光緒帝之侄。

嗣子

有血緣關係的後代通常說是後嗣。西漢末，漢家皇嗣都被王莽殺得差不多了，綠林軍推出的更始帝劉玄，赤眉軍推出的建世帝劉盆子，皆系西漢皇族遠支，說起來血緣最近，其實已稀釋得只剩米湯了。比如劉玄，漢景帝劉啟有六、七個兒子，長沙王劉發是第六子，劉發生劉買，劉買生劉熊渠，劉熊渠生劉仁，劉仁生劉敞，劉敞生劉玄，你數數是多少代？相當於是漢景帝的六世孫，老長了。歷史上有些朝代往後走不通了，乾脆就往前走，找叔叔輩的，唐武宗李炎死後，他的叔叔唐宣宗李忱承繼，不意成就了唐朝最後的輝煌——大中之治。

養子、假子、義子

上述一大堆「子」，儘管同室操戈六親不認，但有一點卻不得不承認，這就是血緣關係，關起門來都是一家人。但另有些「子」就不同了，比如養子，我養了你，有感情，我死了就由你來當；比如假子，假借之義，前妻、前夫之子，就是我們說的拖油瓶；還有義子，喝酒衝動時整出來的：來，這杯乾了，你這個兒子我認定了！歷史上，正是這些似親非親之子的攪和，才使得承繼問題變得複雜多變，以至生出許多事來。

總而言之，子承父業、兄終弟及，嫡貴庶賤、男尊女卑，是中國宗法制度的精髓，也是古人遵制守規的法則。然而，這些規定並非一成不變，在許多時候會生出意外，由此才使得我們的歷史百變生奇、多姿多彩。
