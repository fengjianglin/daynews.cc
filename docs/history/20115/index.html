按照《拾遺記》的記載，秦始皇其實“修仙”成功了

2019-12-16 2,809條評論 3,875次閱讀 1812人點贊

不知從何時開始，網路上流傳一個段子，說秦始皇見過外星人，還進行過親切友好的交流。這件事記載於東晉王嘉所著《拾遺記》，是有出處的。

好萊塢電影中的秦始皇

雖然這個段子流傳已久，《拾遺記》也確實有相關的記載。但是，秦始皇沒見過外星人，不信咱們看看王嘉是怎麼寫的。

《拾遺記》的「秦始皇四事」中，第二件事開頭是這樣的：「始皇好神仙之事，有宛渠之民，乘螺舟而至。舟形似螺，沉行海底，而水不浸入，一名『淪波舟』。」

螺舟難道是外星人飛船？

單看這一句的話，確實有些詭異。來自宛渠國的人，交通工具是「螺舟」。這東西看起來就像是「螺」，可以在海底航行，不會進水——這不就是著名的USO（不明潛水物體）嗎？

更關鍵的是，宛渠國人身高十丈，和秦始皇聊起了天地初開時候的事情，就像親眼所見一樣。

要是真這麼看的話，確實很像是外星人。

可惜的是，這一段還有很多內容，告訴我們這事兒和外星人沒啥關係。

神仙都是躡虛而行

下面一大段的內容，就是宛渠國人所說的內容。原文中寫道：「臣少時躡虛卻行，日游萬里。及其老朽也，坐見天地之外事。臣國在咸池日沒之所九萬里，以萬歲為一日。俗多陰霧，遇其晴日，則天豁然雲裂，耿若江漢。則有玄龍黑鳳，翻翔而下。及夜，燃石以繼日光……」

這一段很明顯是宛渠國人在「炫耀」自己的生活，你要是仔細一看，就明白了他是一位得道成仙者。第一句話就說自己「少時躡虛卻行，日游萬里」。「躡虛」的意思就是得道成仙之後，可以騰空而行，「卻行」就是可以倒退著行進——所以這個「躡虛而行」，指的就是這位仙人年輕的時候就可以騰雲駕霧，飛來飛去了。

下一句說他老了之後，「坐見天地之外事」。也就是說現在修仙等級更高，自己根本不用動，坐在家裡就能看到天地萬物，這似乎比元神出竅檔次還要高。

之後就明確了他神仙的身份，「臣國在咸池日沒之所九萬里，以萬歲為一日」。宛渠人居住的地方距離咸池有九萬里的距離，那麼咸池是什麼地方呢？《淮南子》有「日出扶桑，入於咸池」的說法，就是說咸池是日入之地。當然，傳說中咸池也是仙女洗澡的地方，所以也被稱為桃花。

《西遊記》的天宮就是雲霧繚繞

而「以萬歲為一日」，通俗的翻譯應該是「地上萬年，天上一日」。所以可以明確的說，宛渠國就是神仙居住的地方，宛渠人就是神仙。

所以才有了下面的內容，他們居住的地方平時霧氣蒙蒙，遇到「晴天」就會有玄龍黑鳳四處翻騰，和我們後世文藝作品描述的天宮極其相似。

一個這麼牛的神仙，為什麼要來找秦始皇呢？

原因很簡單，他是來點化秦始皇，讓他學道修仙的。這一段內容的最後一句是這樣的：「始皇曰『此神人也』，彌信仙術焉。」也就是說，秦始皇認定這是一位大仙，所以更加相信仙術，當然也更加期望成仙，長生不老。

《拾遺記》，其實是一本「修仙宣傳冊」，因為它明確無誤的告訴我們，秦始皇修仙成功了。

《拾遺記》

緊跟秦始皇會見宛渠國人之後，另一件事情就是造「子午台」。書中說了，秦始皇搜羅天下的能工巧匠，寶石珍木，為自己的修仙做準備。除了材料珍惜難得之外，「施工人員」似乎也是神仙派來幫忙的，《拾遺記》記載：「二人騰虛緣木，揮斤斧於空中，子時起工，午時已畢。」從子時到午時，12個小時的時間，兩個人就修建好了子午台，可見不是凡人。秦始皇的修仙之路，專門有人幫忙的。

《拾遺記》關於秦始皇的最後一件事情，就是「子嬰殺趙高」。

書中描述，子嬰在宮中休息，忽然夢到一個身長十丈的人，穿玉鞋坐丹車來找他。這人告訴子嬰「余是天使也，從沙丘來。天下將亂，當有同姓名欲相誅暴。」第二天醒來，子嬰就知道了趙高有問題，於是殺了他。

這段內容最後說「子嬰所夢，即始皇之靈」。嗯，不是「始皇之魂」，《拾遺記》的意思就很明顯了：秦始皇不是死了，而是修仙成功了。不信的話看看前面的描述，子嬰夢中的秦始皇也是「十丈」，和宛渠國人一樣高了……

所以，東晉王嘉的《拾遺記》，通過這幾件事告訴我們：秦始皇修仙成功了。

當然，我們知道這是不可能的，王嘉只不過用這個神奇的故事，「推銷」學道修仙而已。

東晉著名道教學者葛洪

王嘉是東晉時期的方士，初期非常熱心於傳道。傳說他鑿山崖居住，有數百名徒弟跟隨他。之後因為戰亂，王嘉多次遷居，始終有徒弟尋訪他。這本《拾遺記》，就是王嘉隱居期間所著的。

東晉時期，清談之風盛行，像王嘉這種隱居山野，養生授徒的人非常多。也正是這一時期，道教的整體架構慢慢形成，玄學思想大範圍傳播，煉丹修仙的理論也開始慢慢豐富。此時佛教也不斷的傳播，受此刺激，道教結合傳統神仙學說，加入煉丹理論，試圖更廣泛的進行推廣。

於是，一批關於養生、煉丹和修仙的書籍出現了。比如東晉著名的道教學者葛洪，就著有丹書《抱朴子》，流傳甚廣。而《拾遺記》，則是採取講「歷史故事」的方式，宣傳學道修仙。

這本書另外一個有意思的地方是，秦漢之前的記載，偏向於怪誕神奇的上古神話，甚至頗有些《山海經》的感覺。但是從秦漢開始，就刻意加上一些神奇的物品，出現在真實歷史人物和事件中。

比如被追封為漢武帝皇后的李夫人，《拾遺記》中就記載一件奇事。

電視劇中的李夫人

說是李夫人死後，漢武帝非常想念，以至於常常面容憔悴，心神恍惚。於是就召見了神秘道士李少君，希望能再見到李夫人一面。於是，李少君要了百艘大船，上千名船員，還有精通道術的人，去暗海尋找「潛英之石」。

這塊神奇的石頭，刻成人像的話，就像真人一樣，甚至能說話！但是因為有毒，所以只能遠遠的看著，不能靠近。等到找回這塊石頭之後，確實猶如李夫人復活，漢武帝非常高興。

後來漢武帝也覺得這樣不好，於是聽從李少君的話，把石頭搗碎製成藥丸，吃了之後再也沒有夢見李夫人。

道士煉丹

你看，這東西神奇吧。李少君這個人是《史記》和《漢書》中都有記載的道士，本身就非常有名氣。《拾遺記》用他和漢武帝之間的故事，強調道法的高明，宣傳效果確實非常好。尤其是將神石搗碎製成藥丸，則明顯是「煉丹修仙」的套路。

《拾遺記》中此類神人和珍品還有很多，真真假假的穿插於歷史故事之中，生動有趣，讓人心馳神往。

所以，從秦始皇修仙到漢武帝吃丹藥，王嘉通過《拾遺記》這本書，潛移默化的宣傳著煉丹修仙理論，正表明了東晉那個特殊的歷史時期，道家思想的傳播力度。
