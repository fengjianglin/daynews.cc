生母和姐姐都沒有把遺產留給他，只有繼母的遺產給了一生未婚的他

天天要聞
7 天 ago

張愛玲在散文里誇過她弟弟張子靜從小相貌漂亮，可是張子靜一生沒有人愛，父母不喜歡他，姑姑不喜歡他，繼母也不喜歡他，但是最後給他遺產的卻是繼母。

雖然身為宰相後代，可是張愛玲家一點都沒有半點溫情。張愛玲出生於1920年，張子靜出生於1921年，張子靜比張愛玲小一歲，他們姐弟倆個是由保姆帶大的，生母在他們的生活里一直是缺失狀態。

張愛玲父母不和，她母親黃逸梵在她4歲時就跟著小姑子出國留學，此時張子靜三歲。三年後，黃逸梵學成歸國，張愛玲和張子靜過了幾天有母親的日子後，黃逸梵又和丈夫張志沂爭吵，就毅然離婚了。

離婚時，黃逸梵不要兒女，一雙子女由張志沂撫養，黃逸梵出國定居，環遊世界，談戀愛，日子過得十分瀟洒，還有一個非常英俊的外國男友作伴。

偶爾黃逸梵回國探親，來看望女兒張愛玲，關心女兒張愛玲的學業，而對兒子張子靜不聞不問。在繼母進門後，張子靜經常被父親張志沂打罵，也沒人關心他。

在張愛玲長成如花少女時，她因探望生母和繼母發生爭執，父親偏護繼母，毒打張愛玲後又軟禁張愛玲半年，張愛玲半夜逃出投奔母親黃逸梵，黃逸梵收留了張愛玲，並且出錢供張愛玲上學讀書。

而張子靜就可憐了，他也拿著一雙球鞋跟著跑來了，說他不想回父親的家，求母親收留。可是黃逸梵卻很冷靜，她拒絕了張子靜，對張子靜說，她的經濟能力只能供養一個孩子，這個名額被他姐姐用了，所以養不起他，所以他還是回他父親家吧。

張子靜傷心地哭了，張愛玲也傷心地哭了。只有黃逸梵沒有哭，她很鎮靜。但是她的話好像挑拔了張愛玲姐弟間的關係，好像是因為張愛玲，她才不收留張子靜。

生母不喜歡張子靜，而父親和繼母也喜歡張子靜。自從繼母進門後，父親張志沂經常打張子靜。張愛玲在散文里寫道，她平時住校，有一天放假回家，吃飯時，父親突然給了張子靜一個耳光，張愛玲十分害怕，放下碗哭了，而張子靜卻一點反應都沒有。繼母笑話張愛玲：「挨打的是他，怎麼你哭了？」

雖然生母遠離，父親和繼母對兒女不好，但是張愛玲還有一個疼愛她的姑姑，而張子靜沒有一個人疼愛，姑姑不但不疼愛他，對他的態度還不如普通人。

因戰爭原因，張愛玲從香港回到上海，和姑姑一起居住。張子靜經常看望姑姑和姐姐，到了吃飯時間，張子靜還沒有走，姑姑就對張子靜說：「你如果要在這裡吃飯，一定要和我們提前講好，吃多少米的飯、吃哪些菜，我們才能準備好。像現在這樣沒有準備就不能留你吃飯。」

張子靜一下子窘迫了，只能慌忙告辭。這就是姑姑的不對了，別說是侄子，就是普通朋友，到你家作客，能好意思對你說，我今天要在你家吃飯，你要給我準備什麼菜，我要吃多少米。這種事，只能主人讓一下，客人才能留下吃飯。

1952年，張愛玲離開上海去香港。不知什麼原因，她沒有告訴弟弟她要走的事。而張子靜卻像以前一樣來探望姐姐，姑姑開門，冷冷地對他說，「你姐姐已經走了。」然後就把門關上了。

姐姐走了，姑姑連門都不讓他進，姑姑對他，連普通朋友都不如。張子靜曾經寫過一本書，說過姑姑和他父親矛盾極大，因為他和父親和繼母生活在一起，姑姑覺得他受父親的影響，所以對他很反感。

在張子靜的家庭里，沒人喜歡他，父親不喜歡他，母親不喜歡他，繼母不喜歡他，姑姑不喜歡他，姐姐開始喜歡他，後來也不喜歡他。組合家庭矛盾多，大致是如此。

原生家庭如此糟糕，很多離異家庭的孩子都是如此。但是他們會早早戀愛，很早結婚，組織新家庭，擁有溫暖和愛，可是張子靜卻沒有這機會，因為他一生未婚，無妻無子。

1957年8月，黃梵逸在英國倫敦病故，享年61歲。她把遺產她和弟弟分家得到的一箱子古董）留給了張愛玲，託人由英國寄到美國。而無夫無子的張愛玲病故時，把遺產留給了自己的好友宋琪夫妻。

張子靜只得到一個人的遺產，就是繼母的遺產，繼母把一間十平方的亭子間留給了張子靜。這麼小一點的亭子間十分簡陋，沒有廚房和衛生間，只能和同樓十幾戶人家共同。

可是張子靜一生未婚，沒有分配房子，就是繼母這間小屋給了他晚年一個棲身之地。繼母沒有子女，張子靜就成為繼母遺產的繼承人，幸虧繼母沒有用一紙遺囑把遺產留給娘家人。

張子靜一生沒有人愛，原生家庭不幸，長大後也沒有結婚生子，母親的遺產給姐姐，姐姐的遺產給好友，只有那個在張愛玲小說里的兇惡繼母把遺產留給了張子靜。
