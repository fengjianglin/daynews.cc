生活中朋友太少，該怎麼交到新朋友，這些方法一定要學會

2019-12-18 211條評論 3,008次閱讀 514人點贊

畢業了就要工作了，進入職場後，每天除了日復一日的工作之外，如果不去參加活動、社團或是課程，就很難再交到新朋友了。

但是美國耶魯大學的研究者克里斯塔基斯表示，建立關係與拓展人際網路，其實對於一個人的快樂程度而言具有極大影響。

美國楊百翰大學的研究甚至提到，社交圈太小對個人的健康也有危害，風險甚至和每天抽十五根煙一樣嚴重。

因此雜誌《快速公司》的撰稿人斯蒂芬妮-沃扎提出了四個交到新朋友的好方法，而這些科學研究將會另一種角度說服你跨出舒適圈，主動和人接觸！

1. 聯繫老朋友

哈佛商學院的教授弗朗西絲卡-吉諾指出，從「老朋友」開始著手或許是交到新朋友的不錯方式。首先，你已經對於這些舊識有所了解，所以再次嘗試聯繫並非太過困難，而且他們也有機會為你介紹或引薦新朋友。

因此，你可以從在社群軟體上多些互動開始，例如點贊、留言，或是發個消息聊聊彼此近況，再約出來見面吃個飯。在和老朋友互動的過程中，說不定就能獲得認識新朋友的機會了！

2. 在工作中認識朋友

其實，同事之間的革命情感最能轉化成私底下的友誼關係，除了每天花上許多時間相處與合作，麻省理工學院的訪問學者本瓦伯提到，你也可以和同事們一起共享午餐時光，或是在飲水間聊聊瑣事以建立友誼。

甚至有一項研究還提到，在辦公室里的流言蜚語其實有高達70-90%是真實的，而知道但不隨便亂傳這些消息，則能讓你在辦公室這個小型社會中，和其他人更有聯繫。此外，一個成功的團隊其實高度仰賴團隊成員之間彼此的關係，而這也會影響個人的工作表現與成就感，因此試著和同事們融洽相處對於你的職涯發展，也將大有助益。

3. 試著當個好的傾聽者，並尋找共通點

「我也是！」這句話其實是能建立連結、鞏固關係的。研究顯示，人們會喜歡和自己名字相近的人、喜歡和自己名字開頭相同字母的品牌，也會傾向於對有著共通點的另一個人較有好感，但這並不是要你人云亦云，而是藉由傾聽對方所說的話，試著找出彼此相像之處，以真誠的態度聆聽並分享彼此的相似經驗。

普林斯頓大學的神經科學家塔米爾提到，人們在談論自己時能夠得到愉悅感，而試著詢問對方有關他們自己的問題則能增進彼此的親密感，因此讓對方多說話，其實是有助於增加他/她對你的好感的。

4. 參與或成立一個團體

你知道丹麥人為什麼最快樂嗎？其中一個原因是有高達92%的丹麥人都隸屬於一個社會團體之中，從體育活動到各式各樣的愛好都有。因此，你可以試著成立或參與一個讀書會、電影會或是品酒會，讓共同興趣成為你認識朋友的新管道，畢竟這也是你們之間的「共通點」。

而且這些新認識的朋友，也能讓你接觸到舒適圈以外全新的人事物！
