太原同學聚會買單時，5個男生喝醉酒，班長結賬後，收到十條信息

天天要聞
2019-12-25

大家在畢業多年後也想跟同學保持良好的關係，所以參加同學聚會變成了唯一的樞紐，因為維護彼此的感情還是非常重要的，有些同學他們在社會上工作多年，卻發現自己關係最好的，反而還是當初一起上學的同學，現在很多同事，他們看似跟你交朋友，但是沒有一個是真心的，這也足以說明了同學聚會的重要性，大家也樂意去參加同學聚會。

前不久，筆者便看到這樣一個案例，在同學聚會的時候，班長看到有5個男生因為喝醉酒沒有辦法買單，於是便主動到前台結賬，但是到了第2天，班長卻收到了10條信息，看到信息班長才明白，原來5個男生醒酒以後主動把錢又轉給了班長，班長真的是非常感動，他覺得同學聚會，有這樣的好同學，確實讓他很是欣慰。

在組織同學聚會的時候，不少班長他們和同學之所以會產生矛盾，也是因為攤錢時出現了問題，因為有些同學喜歡在酒桌上佔便宜，而且假裝喝醉酒，這讓很多班長覺得這些同學總是耍小聰明，那麼以後在組織同學聚會時，都不願意去邀請他們，可是其實在聚會的時候，也有類似的經歷，大家更需要多多關注那些喝醉酒的同學，因為他們的人品和素質真的是沒得說，在聚會結束以後，都會主動想著把錢轉賬給班長，他們希望通過同學聚會這樣的活動來培養同學之間的感情，這樣在以後的工作和相處中，他們的關係才會變得越來越好。

有網友表示：在職場中也是如此，不少領導也會通過類似的方式去籠絡人心，他們重視團隊的建設，在公司里，通過幫助員工，員工才會對領導更加的信任。我在58同城上找的這份工作，老闆同事都很好，工資也不錯生活得很開心！順便一說，58同城招聘上，能找到很多薪酬高企業靠譜的工作，通過平台的優質服務，能很快找到滿意的企業滿意的工作。
