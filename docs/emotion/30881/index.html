如果父母連皮亞傑都不知道，就不要說自己懂得如何教育好孩子

2019-12-24 2,860條評論 4,546次閱讀 3227人點贊

皮亞傑是誰，很多人可能知道，但我相信90%的父母可能都不知道，尤其是那些沒有接觸過心理學內容的父母。

而今天我為什麼要強調父母必須要了解皮亞傑呢？

是因為我感覺現在很多父母都關注孩子，也研究怎麼教育好孩子，但是大部分都在瞎忙。自己沒有基本的明辨是非能力，被人一忽悠就追著各種牛鬼蛇神膜拜，為人搖旗吶喊，被割韭菜而不自知。你要去點醒她，她還覺得你還是在忽悠她。

所以我覺得非常有必要告訴大家什麼是大師，什麼才是父母必須要了解掌握的育兒知識。作為父母你如果連皮亞傑的理論都不知道，學再多的育兒知識，聽再多的育兒專家講座，都白搭。

避免大家覺得我是在忽悠，先上幾張圖。

《教育心理學》目錄

《兒童心理學》目錄

《發展心理學》目錄

就是下面圈出來的這3本書的目錄。

這三本書的權威性專業性，毋庸置疑，只要是想從事教育行業的，特別是教師，都必須要學的。如果你不信，你可以跟周圍當老師的打聽下，他們考教師資格證的時候，有沒有學習皮亞傑的知識。

好了，現在我們知道皮亞傑的重要性，那麼我們就來了解下皮亞傑，他有哪些重要的理論讓他名垂青史。

皮亞傑是誰

皮亞傑，全名讓·皮亞傑（1896年8月9日——1980年9月16日），瑞士人心理學家，也是近代最有名的兒童心理學家，發生認識論的開創者。他跟愛因斯坦都入選了美國《時代》雜誌評選20世紀最具影響力的百人榜。

皮亞傑出生於瑞士的納沙特爾，他在10歲時就寫了一篇白化麻雀的文章發表在當地的自然科學雜誌《冷杉樹》上。14歲時，皮亞傑又發表了一系列和軟體動物有關的論文，並對正統門德爾的進化論提出質疑。可以說皮亞傑的不凡，從他小時候就已經展現出來了。

1918年，他獲得生物學和哲學雙博士學位。隨後他又了解到心理學，覺得心理學是連接生物學和認知論的紐帶，因此又轉而研究心理學。在此後的20多年中，他先後發表了40多部著作，200多篇論文，被人們譽為繼弗洛伊德之後最偉大的心理學家。

皮亞傑的成就

皮亞傑提出的理論觀點，很多，但是最主要的，與兒童教育有關的主要2個。這兩個理論也基本可以解釋兒童教育的大部分問題。

一、認知發展理論

在皮亞傑之前，人們認為兒童就是縮小化的成人，除了體力、經驗等不足外，在思維方式方法上跟成人並沒有什麼不同（其實現在還有很多父母這麼認為）。直到皮亞傑的出現，經過他的研究發現，兒童在認知思維發展上與成人是截然不同的，存在很多不足。

他把人的認知發展分為4個階段：感知運動階段、前運算階段、具體運算階段和形式運算階段。並指出了每一個階段的特點和不足。看下圖。

光這樣說大家可不能清楚這個理論的意義，我舉2個例子，大家就明白這理論對於人們了解兒童起到的重要作用。

1、客體永久性

對於成人來說，我們即使我們看不見了，我們也並清楚，這並不代表那個物品消失了，我們只是被遮擋了視線而已，這是因為成人掌握了客體永久性概念。但是孩子也一樣嗎？

經過皮亞傑的觀察研究，人們並不是一開始就具有客體永久性認知的，而是隨著孩子大腦的發育，孩子大約18個月時才逐漸掌握這種認知的。

相信很多人都跟孩子玩過藏臉的遊戲，用兩隻手遮住臉，然後突然再拿開，孩子就會咯咯笑個不停，其實這就是孩子的客體永久性認知在起作用。對他們來說，消失的東西突然又出現，他們會感到很驚奇，因此會笑。

2、三山實驗

「三山實驗」是皮亞傑為了驗證兒童「以自我為中心」理論而設計的。

在孩子面前的桌子上放三個形狀和顏色不同的山模型，桌子的另一端放一個娃娃。然後讓孩子從幾張不同角度拍攝的照片中選出玩具娃娃看到的樣子，大部分處於前運算階段的孩子並不能正確回答。因此皮亞傑判斷兒童是「以自我為中心」的。

正是兒童是「以自我為中心」的特點，如果父母這時候教孩子換位思考、共情、分享等理念時，顯然是違背孩子的發育規律，因此父母會發現怎麼教他們也學不會。殊不知這是他們的生理特點就決定了。

也正是我們現在了解了皮亞傑的理論，我們現在才能做到因材施教。

二、道德認知理論

要不我們說皮亞傑偉大，皮亞傑不僅研究了人的認知特點，總結出了具體規律，還研究了人的道德認知發展，並且也提出了相關的道德認知理論。

他把人的道德認知也分為了4個階段：前道德階段（0～3歲）、他律道德階段（4～7歲）、自律道德階段（7～10歲）公正階段（10—12歲）。不同階段的特點大家可以看下圖。

皮亞傑的道德認知理論向大家解釋了人們的道德規則認知是怎麼形成的，這有什麼意義呢？

現在很多家長都喜歡跟孩子講道理，結果他們發現講道理不管用，孩子根本就不聽。如果家長了解皮亞傑的道德認知理論的話就明白，兒童的道德認知是從無道德到有道德，從他律到自律的過程。孩子沒有道德觀念的時候你跟他講道理不是對牛彈琴嗎？況且當孩子處於他律道德階段時，他們的特點就是盲從，只要父母堅持，哪怕是錯的孩子，孩子也會從遵從，並形成相關的道德認知。

可以說懂得皮亞傑的道德認知理論，父母就懂得應該用什麼態度教育孩子。

在他律道德階段，父母不要說跟孩子做什麼朋友什麼的，在孩子看來，父母就是權威，父母說的就是對的，因此父母就應該指導孩子對錯，命令他們，指揮他們，不用跟他們多解釋。

但是當孩子進入了自律階段後，父母就應該學著尊重孩子，要跟孩子講道理。因為這時候孩子已經內化了一些道德規則，形成了自己的道德判斷標準。父母只有從理論上說服他們，他們才能真正的接受父母的建議。

總結

為人不識皮亞傑，閱盡書籍也枉然，做不了一個合格的父母。

可以這麼說，如果父母了解了皮亞傑的理論，從本質上就基本把握了教育孩子的正確方法。其它的像《正面管教》、《愛與自由》等，只是一些技巧方法的問題，哪怕你完全不看這些書籍，教育孩子也不會出現大的問題。

相反，你看了再多的育兒文章、育兒書籍，沒看過皮亞傑的理論，對於兒童的基本特點、規律都不知道，那最終收穫的只有一個結果——稀里糊塗，看誰的都覺得對。
