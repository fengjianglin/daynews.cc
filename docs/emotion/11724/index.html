六旬老人背一麻袋饅頭，歷盡千辛探望外孫，隔輩間有多親？

2019-12-11 43條評論 3,978次閱讀 910人點贊

六旬老人背一麻袋饅頭，歷盡千辛探望外孫，隔輩間有多親？

在中國的傳統當中，「隔輩親」永遠是一個繞不開的話題，尤其是現在社會快速發展的同時，老人也成為了照顧孩子的主要力量。家中有老人，身體還算可以的情況下都會選擇幫助自己孩子照顧下一代，而關於隔輩照顧孩子向來就熱議不斷，有人認為隔輩照顧孩子比孩子父母照顧得好，因為有經驗，但是不少人說老人照顧孩子容易形成代溝。

不管老人照顧孩子是好是壞，都不能否認的是老人對於孩子的愛。年近六旬的劉奶奶為了看一眼自己的外孫，竟然背著一大袋的手工饅頭，歷經九個多小時的路途顛簸，終於見到了自己的外孫。對於這一幕相信很多人都會好奇，為何一個老人會這麼做，尤其是在這樣便利的社會？原因竟然是自己的外孫患了白血病，為了給外孫省錢治療，這位老人才這樣做的。

這位老人談起自己的女兒和外送就非常的心痛不已，尤其是抱怨自己女兒命運的不公。自己的女兒是經人介紹結婚的，由於婚後兩人感情不好，又加上自己的丈夫脾氣暴躁，婚後生活也並不開心。婚後沒多久老人的女兒便宮外孕懷上了孩子，但在醫生的勸說還是冒著風險堅持生下來，並想著孩子的出生全家人會幸福，但是其丈夫脾氣更加變本加厲，無奈之下只好選擇了離婚，離婚後沒多久自己的外孫又查出白血病，現在三個人相依為命。

比起治療，如何預防才更重要！

1.注意個人衛生

孩子蹦蹦跳跳到處跑、趴在地上玩玩具、從外面回家不洗手等行為都會沾染一大堆細菌病毒，如果不及時清洗，很容易就進入體內導致生病。作為家長，要做到引導孩子勤洗手，從小養成勤洗手的習慣，不僅有益於孩子健康，而且對孩子良好衛生習慣的培養也能起到積極的作用。

2.遠離疾病源頭

有的孩子生病，也有可能是接觸到了疾病源頭。很多疾病的發生都和生活習慣有很大的關係，尤其是對於尚未成年的孩子來說，幫助孩子養成健康的生活習慣是作為父母義不容辭的責任。比如：很多父母為了早上起來節省時間，前一天晚上就把第二天的飯菜做好，雖然是節省了時間，但是卻傷害了健康。

3. 讓孩子進行適當的運動

想要孩子平常少生病，遠離一些重大疾病，從根本上來說還需要一個強健的體魄，所以讓孩子進行適當的運動顯得非常有必要。

孩子的健康是父母最大的關注，作為父母都希望自己的孩子能夠健康地長大，因此在關注疾病的同時，孩子的健康也應重視起來。
