練就波瀾不驚的忍耐，再艱難的歲月，也只不過是浮雲，早安

2019-12-20 2,007條評論 4,045次閱讀 1110人點贊

1、人生軌跡不一定是按你喜歡的方式運行的。有些事你可以不喜歡，但不得不做；有些人你可以不喜歡，但不得不交往。當遇到那些自己不喜歡卻又無力改變的事時，我們唯一能做的，就是忍耐。忍過寂寞的黑夜，天就亮了；耐過寒冷的冬天，春天就到了。練就波瀾不驚的忍耐，再艱難的歲月，也只不過是浮雲。早安！

2、如果時間能夠倒退，一切也就沒那麼珍貴了。人生的快樂，走自己的路，看自己的景，超越他人不得意，他人超越不失志。心裡放不下，自然成了負擔，負擔越多，人生越不快樂。計較的心如同口袋，寬容的心猶如漏斗。複雜的心愛計較，簡單的心易快樂。

3、生活告訴我，幻想終究只是幻想，幸福是靠我們的雙手一點點創造的，我們的心靈一天天沉澱的，不管生活多麼不容易，我們都要用最完美的心態來面對它。你對生活笑，生活就給予你快樂；你對生活哭，生活就回饋你雨季。早安~

4、假若心存一個夢想，那麼你就只有一個方向，你也就不會迷茫，就算跌倒你仍然知道爬起來以後該往哪裡走去。困境可以磨滅一個人對生的嚮往，但是它絕對不會打碎堅持夢想的心，因為執著一件事的力量是無可抵擋的。早安！

5、有一點壓力，就感覺天塌下來了；有一點不開心，就覺得自己是世上最可憐的人。你如此脆弱，都是曾經有人寵溺，有人替你撐著，有人替你堅強。累了，蹲下來抱抱自己，告訴自己：也不過如此，我們還可繼續前進。早安！

6、沒有誰能左右你的情緒，只有你自己不放過自己，誰心裡沒有故事，只是學會了控制。得不到的就不要，不是好的就扔掉，我們應該在薄情的世界裡，深情地活著。早晨好！

7、簡訊到，早上好！知道你美夢還沒做完，把你吵醒也別抱怨。揉揉迷茫的小眼，洗洗睏倦的小臉，再吃頓美美的早飯，開始自己美好的一天。親愛的朋友，早安！

8、周末到，拋下辛勞的工作，與忙碌說”拜”；呼喚休閑的腳步，和輕鬆說”嗨”；快樂的大門已敞開，燦爛的笑容綻開來，祝福的話語暖心懷，早安！

9、昨天已過去，新的一天已來到，簡訊祝福傳問候，送你快樂心情好：煩惱憂愁全丟掉，鬱悶煩躁全忘掉，壓力疲倦統統繞，唯有幸福樂逍遙。早安~

10、一聲鈴音打斷了你的美夢，看看我的簡訊不會要你的命，頭不用梳臉不用洗，快把手機拿起，把溫馨的祝福送給你：天天快樂，歡歡喜喜！

11、輕輕劃開窗布，一片陽光披滿全身，伸個懶腰，好舒服！美妙的一天就要開始！拿過手機，發個早安給你：你準備好了嗎？

12、有些人看起來整天面帶笑容，並不是因為他們事事順利，只是他們比你敢於面對問題、善於遺忘不幸、勇於擁抱未來。早安！更多的精彩情感，哲理，美文；敬請關注微信公眾號：心成一片
