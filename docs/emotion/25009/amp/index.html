# 人性 真愛 實踐

為什麼「知道了那麼多道理，也過不好這一生？」

相識是緣，表達為話；你好，我是吳吞：

今天繼續和你聊《靈魂擺渡》中的故事。

很多朋友認為，人性本賤，尤其是男人。因為男人們總是相信：「妻不如妾，妾不如妓，妓不如偷，偷不如——偷不著。」

家花是否永遠不如野花香？

未必！

靈魂擺渡之旅行中的男人

我是一個插畫師，專門畫各種花卉。

也許你會覺得，我一個大男人，專門畫各種花卉，很奇怪吧。但是我知道，這就是我想要的——

「花，是很奇妙的，它們那麼美麗，生命卻又那麼短促。」

為了留住它們的美，我拿起了畫筆，我想把它們的生命，用畫筆延續下來。
我可不是什麼二代，為此我開了一家花店，這樣我就可以一邊照看生意，一邊拿起我的畫筆。

有一天，一個女孩來到了花店。

她像我一樣熱愛著鮮花。

我愛上了她，她也愛上了我。

於是，她成了我的妻子。

我送給了她一盆花，妻子每天精心照料著。她們是那麼的美麗——那一幕，我甚至都嫉妒了。

也許是受到了愛情的滋潤，我的繪畫技術越來越成熟。但是不知道從什麼時候開始，我發現了筆下的花卉，都失去了生命！

我當時很焦躁，我不知道是哪兒出了問題，我描繪不出我的生命，我畫出來的花都是死的，沒有靈魂。

我的編輯看不出來，讀者也看不出來；但是我知道，這對我來說太可怕了——不安，焦躁，迷茫，不知所措——巨大的陰影吞噬了我的心扉，對未來的恐懼佔領了我的精神，甚至我的妻子也安撫不了我。

真正的花，應該開在山野間，開在陽光春風之地，而不是剪下來，放在花瓶里或是栽在花盆中。

我想，我必須去尋找真正的花。我要通過畫筆，把它們給帶回來。

關了花店，背上包，我就那麼放縱的追尋我的夢想去了。

在山野中，我見到了各種各樣的花。這些花，和我在城市裡看見的完全不一樣。

我的畫越積累越多，我的行囊也越來越沉重，但是我完全沒有滿足。我簡直想，就這麼一輩子走下去。

直到有一天，我遇到了山崖上的它。

那株花，和我送給妻子的那盆是一樣的，但是它的顏色更鮮艷，風姿更搖曳，氣味兒更鮮美。

那花兒，似乎有著神奇的魔力。

我第一次沒有立刻攤開畫紙，它讓我想起了家，被我拋在腦後的妻子。

此刻，我的心裡只有一個念頭——我要把它帶回去，我要把它帶給我的妻子。

後來的事兒，我想不起來了。我只記得我又上路了，心裡只有一個念頭，就是回家。

我突然覺得我出來太久了，我想家，我想我的妻子。

當我回到家，奇怪的事情發生了——廚房裡有新切的西瓜，妻子的鞋都擺在鞋柜上，外衣也都掛在衣架子上，只是我送給妻子的那盆花已經快要枯死了。

「這不正常，我妻子絕對不會任由這花這樣枯死的。」

卧室里，妻子平日里常帶的首飾，也都在梳妝柜上，她卻不在那裡。

我突然明白了哪裡不對，是寂靜——是死亡一樣的寂靜。

我的心裡很不安，我覺得發生了什麼不好的事情。

我覺得妻子一定就在家附近，我得找到她！

但是，哪裡也沒看到她。

不！不光是沒有看到她，街道是熟悉的，店鋪也是熟悉的，可是街上，沒有人——整個城市，死一般的寂靜。

只有一間店鋪亮著燈，就是這裡——444號便利店。

我握住店員小哥的手，像是抓住了一根救命的稻草：「你知道這座城市，究竟怎麼了嗎！」

那個清瘦的大眼睛小哥，沉默了一會兒，沒有說話。

這時，一個深沉、嘶啞、令人恐懼的男聲，突然在我的背後炸響：「那是因為你已經死了。」

原來，我在攀爬山崖的過程中，出現了意外，摔死了。但是我心中仍有執念，就是回到我妻子的身邊。

我再次抓住那個清瘦小哥的手，祈求他：「幫幫我，我對不起她！」

可那個黑面鬼說了，我看不見她，她看不見我，我們再也無法溝通——聽到這個消息，我崩潰地大哭起來，哭暈了過去。

醒來後，我來到了一個陌生的地方。

那個小哥遞給我一張古拙的看不出什麼動物做的黃皮紙，說是這張紙可以通陰陽。

他讓我用這張紙寫點東西給我的妻子。

我抓住最後的機會，用心、用畫筆去描繪我記憶中的風景，我想讓我的妻子也看到的風景。

完成之後，我突然明白，這不是我看到的最美的花，最美的花一直都在我的家裡。

什麼是真愛？

最近PUA鬧得很火，戀愛達人/情感騙子們的操作準則是：「如果她涉世未深，就帶她看盡世間浮華；當她飽經風霜，就帶她去坐旋轉木馬。」

沒得到的，才是好的——這就是人性。

「經驗少還是經驗多」這類問題的底層邏輯，其實也是這樣。如果是老司機，自然希望經驗少的，這樣帶起來才有趣；如果小白，自然希望經驗多的，這樣才能順利地體驗到世間的美好。

但有個前提，是為了彼此的幸福，而不是為了炫耀什麼。相愛沒有什麼標準，但要長久相處下去，最低標準就是「三觀要和」。

如果對方總是拿「處不處」這個問題嚼舌根，早點分了得了。

蘇格拉底說：「未經檢視審視的人生，不值得活。」

同樣的，不經歷風雨，怎能體現愛情的可貴。

插畫師走過千山萬水，尋遍了山花野草。最後發現，自己早已見過最美的風景，而且這個風景一直就在他的背後，默默地注視著他。

看盡人世間浮華，依舊愛著家裡的她——得到了依舊愛，這不是是真愛，什麼是真愛？

我被這一幕感動到了。可惜的是，死到臨頭，他才明白這一點——這也許就是人性的悲哀與原罪吧。

為什麼我們知道了那麼多道理，也過不好這一生？

西方有句箴言：

願上帝賜予我平靜，

讓我去接受，我無法改變的。

願上帝賜予我我勇氣，

讓我去改變，我能改變的。

最後，請上帝賜予我智慧，

讓我能夠分辨，這兩者的區別。

人們常常感嘆：「知道了那麼多道理，也過不好這一生。」

為什麼會這樣？

原因之一就是缺乏智慧。更悲哀的是，智慧往往只能從實踐中獲取。

我們渴求智慧，聆聽道理，就是希望少走彎路。但看前人們看到的風景，和我們要走的，並不完全相相同。

他們的經驗之談，我們只能作為參考。一昧的按照前人的路走=缺少了辨證過程的堅信，那不是相信智慧，而是迷信。

我聽過一個小故事，媽媽在愛情的十字路口告誡女兒，要選擇愛你的，千萬不要選擇你愛的。你所愛的雖然看上去很美好，卻會為你帶來無盡的傷痛。

女兒懵懂的看了看媽媽，一不留神，去了自己所愛的那個方向。

許多年以後，當初那個懵懂女孩，已經成了另個女孩的母親。她也像母親當年一樣，告誡自己的女兒。

當然，女兒同樣沒有聽……

「不聽老人言，吃虧在眼前。」這個道理我們不是不懂，但有些事兒：

不親自去體驗一遍，我們心有不甘；

不親自去驗證一番，我們怎麼知道，老人們的經驗之談，是不是沒用處的說教；

不親自去實踐一次，我們不可能找到獨屬於自己的那份幸福、快樂。

「很久很久以前，有一種生物嘗試著邁開腿，學會了用兩腳站立。自此，他們就踏上了旅途，那是被時間驅趕著，永無止境的旅途。」

智慧來自於實踐，智慧不能脫離實踐，智慧也並非捷徑，它的作用，只是能夠讓我們的路途好走一些。

相識是緣，表達為話，我是吳吞，感謝您的降臨！

吳吞，二手的神話學家，故事愛好者，甩不脫胖字的大俗人。
