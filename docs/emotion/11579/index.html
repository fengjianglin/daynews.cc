深夜來電，兒子回一句就掛斷，睡醒後，才知那是母親最後的遺言

2019-12-13 243條評論 3,230次閱讀 1306人點贊

都說世間母愛最為偉大，父母對子女的愛是無私且偉大的，父母不會做出對孩子不利的事情，他們操心的都是那些對子女好的事情，也許這份操心在子女看來成為了嘮叨，也很多人嫌棄這樣的父母。

父母給予子女的關愛往往多於子女回饋給父母的感恩！所謂樹欲靜而風不止，子欲養而親不待，有時候我們不珍惜的東西往往就是我們生命中最為重要且不可逆轉的東西。

下面我們先來看一則家庭情感故事！一位來自東北朋友的自述！內容以編輯！

大家好，我叫徐子良來自東北的一個小夥子，家庭和其他人的有點不一樣，自我記事起就沒有見過我的父親，是母親一手把我拉扯大，在我印象中她一直都是在小學門口擺攤子做早餐的普通人，平時都會笑著面對來來往往的人，在她臉上看不到抱怨的神情。

而我也是在母親的關懷下成長，母親也許是覺得我沒有父親的緣故，所以對我的關心是其他孩子母親的雙倍，而我也習慣了被母親疼愛的感覺。小時候，其他孩子有的我也有，我記得家裡的玩具都是母親主動給我買的，中學那會我喜歡上了彈吉他，但母親說會影響學習，但最後還是在我生日那天給我買了。

我中學的時候也不算叛逆，母親一向很尊重我的意願，就連高考報考的專業都是隨我自己決定的。記得母親送我去上大學那會，那是我第一次出遠門，母親給我大大小小裝了幾個包，吃的穿的用的都給裹上了，送我到火車站去，直到車開走她才離去。

母親很慈祥，但同時也很嘮叨，平時對我都是非常的關心，但是自從我成年之後，覺得很多事情都可以自己做主了，於是母親的關懷成為了一種嘮叨的存在，大學的時候，母親還問我有沒有在學校里談戀愛、學習成績怎麼樣，久而久之我嫌母親問的多，後來沒什麼重要事情都沒有回她信息了，要麼就簡單回兩句。

而大學畢業後，我去了一家新企業，在那裡實習，做的是編程，公司就幾十人，老闆也經常加班到很晚，但有不少員工受不了這工作強度就離開了，在這裡兩年我也成為了老員工，也帶起了自己的小團隊，加上公司現在處於融資狀態，大家都很忙，而此時的我完全沒有精力想其他方面的事情，而母親經常還會問我有沒有找到對象了。

而那天加班很晚，公司的新案子交到我手上，連續高強度上了7天的班，回到出租屋洗完澡就倒頭睡下了，原本想泡的面都忘記吃了。到半夜兩點的時候，手機突然響了，因為老闆叫我們24小時保持暢通，所以一直都處於待機狀態，但這次打電話來的不是公司的人，而是家裡的母親。

我很憤怒接聽了，母親在那頭對我說：「兒子，媽想你了」我聽了之後直接回了她：「大半夜的你打什麼電話，掛了，我睡了」後來又打了兩次，我都沒有接聽，最後埋頭睡覺了。第二天早上，家裡的二伯打來電話說家裡出事了，老家那老宅區昨夜凌晨2點左右起火了，有十幾人沒逃出來，裡面有你母親…那一刻我才知道發生了什麼。

原來，母親當時最想和我多說幾句話，而那些話還沒表達出來就已經成為遺憾，我才知道母親這輩子把我當成了她最自豪的人。而我卻辜負了她的愛，也許人生就是這樣不可逆轉，且行且珍惜，別像我一樣！

以上的家庭故事讓我們明白了有時候我們最不經意的東西，往往是世界上最重要的，但也在失去後才懂得珍惜。人生路上，真正關心你的人不多，至親之人自然在裡面，而多少人又把這份關心當成了理所當然呢！

不管是家人還是朋友，有時候別人的嘮叨並不是什麼埋怨，而是出於對你的關心，而人往往不喜歡別人約束他，自然就會有些厭煩。可當一個人孤獨太久後，才知道有關心的人才是最大的寶藏！

我們要懂得珍惜身邊人，不要失去了，才去想去過去的點點滴滴，因為那時已經來不及了！
