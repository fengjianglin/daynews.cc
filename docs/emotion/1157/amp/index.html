摘要

接受寶貝的性情

無論任何年齡的寶貝，他們如何管理與處理不適，他們失望或者發生情感混亂的部分原因都歸咎於其自身的性情與氣質：他們有多執著，有多敏感，他們的適應力有多強，對事物的反應有多激烈，他們的睡眠和飲食習慣是否有規律，他們的情緒是更樂觀還是趨向悲觀——這都是他們與生俱來的人格特徵。

有些寶貝可能很容易就發起牛脾氣來， 「對於一些寶貝來說，從一個活動切換到另一個活動可能會遇到困難，」邁克·波依說，「所以他們很可能在父母要求他們離開玩具店時發起脾氣。」同時，對於一些寶貝來說，突然之間去改變原有的計劃或者原有的例行事情，也可以引起類似的反應。

還有一些寶貝有很強的耐心和對事物的持久性興趣，這當然是很好的特質，但這也同時意味著你不能強迫他遠離一項活動，否則就可能帶來大發脾氣的結果。

「如果這個寶貝很敏感，那麼他可能很容易覺得自己受到了攻擊和冒犯；如果一個寶貝平時就有些緊張，那麼在發脾氣的時候就會變得有些瘋狂——父母們應該熟知和理解自己孩子的脾氣，並接受。」邁克·波依說，「當然，儘管如此，無論怎樣，每個寶貝都必須學會在發脾氣的時候克制踢人和尖叫的衝動。」

當他們還是小寶貝的時候

愛他，就愛他原本的樣子。「氣質和性情是我們與生俱來的東西，」溫哥華養育輔導師芭芭拉·德馬雷說，「接受比試圖改變它更容易些。」

當一個小寶貝發脾氣的時候，請首先去衡量你自己的回應方式。「靠近他，讓他感到安全，因為此時，他的憤怒其實也正在讓他自己感到不舒服和混亂。」與此同時，也要留心寶貝憤怒情緒的堆積，不要讓他傷害到自己。

對於小寶貝來說，做父母的一般能夠知道什麼事情會引發寶貝的不安，他們的發脾氣是能夠預測的。「了解寶貝的自然節奏，」戴馬雷敦促父母們，「什麼時候寶貝有可能會過度疲勞？他每隔多久需要吃些東西？」另外，挫折和無奈往往會導致發脾氣，所以父母應該確保你的寶貝在合理的界限內做出一些他自己的決定。

當寶貝漸漸長大的時候

當寶貝發脾氣的時候，不要把它當做是在針對你而感到傷心憤怒，而是應該認識到寶貝只是在用某種幼稚的方式表達自己的不滿。

保持冷靜：這可能會很困難，但家長良好的自我控制，是在教導寶貝遇事應該是什麼樣子的，喊叫或是威脅只會讓發脾氣的寶貝更加激動，一點幫助都沒有。

擁抱他：沒錯，要擁抱你那發脾氣的孩子。他可能會把你推開，那樣的你就自然放手，但是你已經提供一個將寶貝的憤怒分流的機會，提供了一個舒適的懷抱，很可能他就此安靜下來。如果寶貝不接受，那麼你就走開，讓他的脾氣發出來。

用「後果」教育孩子：如果一個寶貝發脾氣把自己的房間搞成一團糟，那麼在大家都平靜下來後，他就必須把房子再打掃乾淨。父母不要替他做這件事，我們必須讓寶貝明白自己的行動會帶來什麼樣的後果。

總結：找一個大家都心平氣和的時刻，坐下來與他們平等地討論一下這個問題。注意談論時將重點放在解決問題上，而不是過多要求寶貝的表達方式。

後續的接觸：如果你的寶貝沖你大喊大叫，甚至口出惡言，那你就對他說「我不喜歡用這種講話方式」，然後走開。讓他自己一個人憤怒去。但當寶貝安靜下來後，父母要去接近他，你們要一起做一些事情，比如開車帶他出去轉轉，或者到外面走一走，並談談當他發脾氣時你的感受。

學會平靜：幫助你的寶貝學習如何從瘋狂的齒輪中切換到平靜健康的頻道。可以嘗試深呼吸，散步或其他運動，或者聽聽音樂，去找鄰居家的小朋友玩等等。當然這些技巧都需要時間來重複練習。

表揚他：當寶貝處在緊張情緒，並自己試圖克制的時候，家長一定要第一時間發現，並立刻表揚他：「我知道很困難，但你處理得很好，你能保持冷靜，真棒！」

發脾氣的正面：愛發脾氣的寶貝其實也有其正面的優點呢。一個情緒緊張，容易激動的寶貝也同時會是一個特別愛你的孩子。這樣的寶貝會經常擁抱你，表達他們對你的依戀。這些寶貝到學齡期和青少年期時，都能夠很開放地愛朋友，關心朋友。

太過分了：儘管發脾氣是寶貝成長過程中正常的現象，不過，戴馬雷還是建議：如果你與寶貝的發脾氣鬥爭得太辛苦，那麼尋找一位養育輔導員或許可以提供一個全新的視角和新的嘗試方法。

如果你感覺寶貝脾氣有些失控了，例如在一天內多次發脾氣，那麼應該去和醫生談談，醫生可能會提供一個評估的測驗，以確認他的行為是否屬於正常。
