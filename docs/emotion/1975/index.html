被富二官二短擇過的普通小美白，到底還是意難平？

圖片來自網路

案例背景：詩姐，被官富二代短擇的，未必是小白美，也有普女。身邊有例子，女方出身南方三線城市縣城偏農村，長相4分，身高160，也被小官二短擇過。

女方積極主動，加上工作關係，順利攀上了，之後無法維持，也算是短擇的一種。男方一線城市副部級二代公子，身高188。

男方說， 有一種獵奇心理，因為是完全不同的兩類人，再加上工作的壓力，是特定情況下才開始的短擇。否則就算女方再努力，可能也不會開始。但女方似乎很膨脹，相處中也越來越看不清自己。

男方由於工作和身份背景的壓力，跟她交往了一年，其實大半年都是冷暴力以求女方自己知難而退。分手後女方依舊看不清自己，據說現在想嫁外國人。

女方普通本科，來一線城市的一家中型企業工作，她特別喜歡和外國人交流，運氣又好，所以在企業里也擔任了一些涉外交流工作，好的時候月薪2-3w。

這也是她認不清自己的原因之一吧，行業和運氣的加持，使她比有些碩士剛畢業的工資都多。加上被小官二短擇過，更加看不上她的平配。

為了愛情，故事裡的男方才會這麼做，沒有特定的環境，怎麼會糾纏那麼久？小官二尚且如此，更別說中心圈的公子了。

女方除了一股拼勁和運氣，顏值，身材，背景，學歷沒有一樣拿得出手的，真的沒有可以交換的籌碼。

回復：以上案例貌似在一線城市不少，另外一個網友說她有個朋友是這樣，5分顏值，被2個過億的富二代各短擇，當長期炮友過幾個月，出入高級餐廳酒店和帶她去澳門過一次，沒得到過什麼禮物 。

在這兩個之前找的都是平配相處幾年，現在她同級別男生已經看不上了，經常抱怨自己為什麼沒有靠譜對象，而且很厭惡之前的前男友們覺得他們low窮之類的。

類似這種普通家境的五六分女被二代公子哥短擇過後，已經接受不了平配了。要知道古代有錢有勢的公子哥們娶的正妻都是聯姻制，出身一般的漂亮女子只能當妾。

現代一夫一妻制了，不能納妾，只好採取短擇策略了。問題是被短擇過的普通人家姑娘，很容易意難平。

公子哥們為了愛情和強勢的父母對抗，這些劇情都在電視劇裡面。向上的婚姻真得有實打實的交換籌碼。 ​​​​

這些被官二代富二代短擇過的有些文化普通小美白，會很不容易看不清楚自己。她們吃了年齡紅利，五分以上小美白前男友們是官富二代都不奇怪。但是有門第觀念的世家，即使生了孩子，還是嫁不進去。

有個已婚網友出身一線城市的上流圈子，她叔叔找嫩模談戀愛生兒子，結婚照樣娶門當戶對的白富美。就像梁洛施生了三個兒子照樣嫁不進去李家，李家要是娶她進門，會被他們圈子裡的人取笑。

普通家境的美女裡面能混到給富豪生孩子還是少數的勝利者，其他的都是炮灰罷了，這無形中抬高了她們的擇偶標準，以後高不成低不就。

所以定位特別重要，找到合適的目標人群很重要，不行就得換個池子。很多人不撞南牆不回頭，無法理解正確定位是啥意思？

比如出身一般的窮美女，拚命擠著嫁在意家世，學歷，工作的中產或者有門第觀念的世家貴族(現代人可以理解為豪門二代們)，

你們的優勢無法利益最大化，準確的目標人群應該是出身一般的創一代，土豪哥，暴發戶等等。這類男人自己有賺錢能力，看中軟體遠遠大於硬體。
