退休後和朋友相處，這“十忌”真得放在心上，學習了

老了，身邊總要有幾個知心朋友，有幾個興趣相投的朋友，生活才能充實快樂。而且，老了還得多交朋友，不僅能擴大自己的交友面，還能讓晚年生活更加豐富。

但是，老了後和朋友相處，也不能掉以輕心，千萬別犯這「十忌」！可真得放在心上！

畢竟現在我們結交新朋友的成本太高了。

1、好事也別多說，忌炫耀

你買了輛新車，你買了個新房，你兒子升職了，這種事盡量避免在朋友面前炫耀。你自己都知道不要在網上炫富，怎麼在朋友面前就可以炫呢？誰看到心理都會不舒服，何況是你的朋友

2、尊重他人，忌問私事

不管你倆多好，每個人心中都有不願被別人涉足的角落，尊重他就是尊重你們的感情。

3、睜隻眼閉隻眼，忌多管

跟朋友相處有一個訣竅，就是對有些事情要睜一隻眼閉一隻眼。不要凡事都以自己為中心，也要給朋友一定的生活空間，朋友可不是隨時給你待命的。

如果跟朋友在某一問題上產生分歧，也不要太過執著，睜隻眼閉隻眼就過了，不是什麼大事，就沒必要堅持到底。

4、打牌玩樂，忌賴賬

如果開始玩之前，就已經商量好是打錢的，那就不要賴賬，否則就說好了是打台費或者用其他方式，要不就別玩。

5、吃飯最好AA制，忌讓別人買單

朋友就是朋友，吃個便飯，但也要盡量AA制。一件事兒是一件事兒，雖然不花幾個錢，但也要盡量避免讓人家心理不舒服。

6、金錢往來要清晰，忌含糊

友情一涉及到金錢，往往會變得比較難辦。如果跟朋友借錢，說好的期限就要儘早還，如果是兩人涉及買賣，就更要事先都講清楚，以免日後產生分歧，朋友都做不成。

7、有事當面講，忌猜疑

不要在親戚、朋友間議論誰，更不要在朋友間傳話，以免誤傳，引起矛盾。

我們不要捕風捉影，有事最好當面講，不要背後嘀咕；有疑問當面問，不亂猜測。以免造成互相猜疑，影響和諧。

8、體諒朋友的習慣，忌惡習

你抽煙，你的父母，愛人可以忍受是因為他們跟你是一家人。但是朋友就是朋友，永遠不可能像家人一樣忍受你的惡習，如果他已經表示不喜歡你在他面前抽煙，那就忍忍吧，死不了。

9、不要說別人的壞話，忌多嘴

無論是朋友的老伴，還是朋友的朋友，就算再不好，也輪不到你來說，你的朋友不開口，永遠別多嘴。

10、說話要溫柔，忌刻薄

朋友就是朋友，不要以為那是你的寵物或者你是人家的父母。太刻薄的語言只會傷了和氣，沒幾個人會當那是忠言逆耳。互損為樂，一時覺得親密無間，久之細想，那些刻薄必是捅人的刀子，無形間將你們間的感情一刀刀抹去。

朋友之間總免不了有些小摩擦，小矛盾，只要好好處理，好好談一談，一切誤會都能解開，畢竟這麼多年的朋友可不容易。

朋友間的相處，除了這「十忌」，同時我們也要自己照顧好自己，自己好了，和朋友相處起來也就更加的輕鬆。

1. 安排好自己的時間。

人老了，時間雖多，但很珍貴，有規劃、有安排地生活，報名參加老年大學、跳廣場舞、健身、養花，都能豐富我們的生活。

2. 要有捨得花錢的心。

我們後半生的消費是有限的，沒有必要對自己吝嗇，偶爾買點好看的新衣服，下下飯館，和朋友旅旅遊，也能提升生活品質。

3. 別悶在家裡，常出門走走。

天天悶在家裡，心中一定會煩惱叢生。想要過得開心，不如常出門走走透透氣。

退休了可以參加社區活動，和大家一起打麻將，做義工，去認識更多的人，也可以和三兩好友們去公園逛逛，聽聽清晨的鳥叫。

4. 注重日常儀錶。

當你穿著得體，整個人都顯得精神飽滿，也更自信，看起來會比實際年齡年輕十多歲呢！

5. 堅持運動。

身體健康才是保持年輕的基礎，不顯老的人一定都堅持運動，讓自己由內而外散發著活力。

6. 保持年輕的心態。

心態年輕的人，在他們身上你也許能看到眼角眉梢的皺紋，但更讓你目不轉睛的是他那神采奕奕的面容和熱愛生活的精神狀態。

7. 身體是最大的本錢。

年紀大了，有點小毛病很正常，可別諱疾忌醫。

每年定期體檢，小毛病拖大了，不僅給自己找罪受，更是給子女添麻煩。

記得給自己辦一份醫療保險，才能更好地未雨綢繆。

和朋友相處好，把自己照顧好，我們的退休生活有什麼理由不快樂呢？

不妨把這篇文章送給朋友們看看吧，祝大家友誼地久天長！
