1、在這個浮躁的社會裡，你的好對別人來說就像一顆糖，吃了就沒了。而你的壞就像一道疤痕，留下了就會永遠存在。這就是人的劣根性。

2、心，應該是一顆樹，在緘默中佇立，既能接受陽光，也能包容風雨。青春如酒，讓人沉醉。人生匆匆，轉瞬就是幾十年，因為有夢，有追求，我們總是嚮往著完美，但完美其實就是一種心態，懂得知足完美自然就來了。早安！

3、有些東西，並不是越濃越好，要恰到好處。深深的話我們淺淺地說，長長的路我們慢慢地走。遇見你之後，我的偉大抱負和一腔熱血，都變成黃昏與你歸家走在小路上的簡單願望。

4、尊嚴這個東西，其實是和你的慾望成反比的，你想得到一個東西，就會變得低三下四，死皮賴臉，而當你對眼前這個人，這件事無動於衷的時候，尊嚴就會在你心中拔地而起。

5、你說煙雨微芒，蘭亭遠望；後來輕攬婆娑，深遮霓裳。你說春光爛漫，綠袖紅香；後來內掩西樓，靜立卿旁。你說軟風輕拂，醉卧思量；後來緊掩門窗，漫帳成殤。你說情絲柔腸，如何相忘；我卻眼波微轉，兀自成霜。

6、若，人走茶涼，就守心自暖；若，聚散難免，便獨享浮世清歡。以最真的情懷，站成最美的守候；以最淡的心，守一生最愜意的情。

7、歲月，兩個字，有點纏綿，有點靜好，有點奼紫嫣紅，又有點簡單素白。歲月較比光陰，有了更長更遠的意味。也就可以擁有生活的五味雜陳。那一點酸，一點甜，一點苦，一點辣，一點咸，都成了歲月的味道，生活的調料。

8、一個真正好的人生，是每天都能做自己喜歡的有趣之事，不必去聽太多外界的聲音，內心已然喜悅且充盈。

9、在你想要放棄的那一刻，想想為什麼當初堅持走到了這裡，生命，就是一場單程旅行，人來人往，光陰似箭；很多時候還來不及看清楚，更別說用心去好好把握，那一段路程早已匆匆而過；浮華之世，越來越多的慾望膨脹帶來眾生的浮躁紛雜。

10、你獲得了別人成功的經驗，但同時要避免狹隘了自己。因為，路有很多條，都想走又都不想走？那麼不如自己另闢一條蹊徑吧。成功真的可以如法炮製么？我們需要的是自己的成功。

11、人來到世界上，沒錢並不可怕，怕的是一直等人來救濟；孤單並不可怕，怕的是一直孤單；失業並不可怕，怕的是一直不去找工作；輸了並不可怕，怕的就是一敗塗地。

12、所有困惑的人，最終都得面對自我的內心，如果不能從心上產生改變，那麼困惑依舊是困惑，問題仍然是問題。當人們明白自己能夠改變，也唯有自己可以改變的時候，我們對人生的疑問，才能給出完滿的解答。

13、歲月匆匆，人生短暫，美麗的東西留不住，快樂的時刻也不會太多，懂得珍惜，活得充實，活得快樂，才是最明智的選擇。

14、人生在世，總是有些空城舊事，年華未央；總是有些季節，一季花涼，滿地憂傷。許多事，看開了，便會峰迴路轉；許多夢，看淡了，便會雲開日出。學會思索，學會珍藏，微笑領悟，默默堅強。

15、要知道昨天的太陽，曬不幹今天的衣裳，世上有不絕的風景，人更要有前行的心情。當你迷茫，躊躇不前時，請抬頭向前看，明天的太陽還會升起，說不定更多的驚喜在前行的路上等你。早安！
