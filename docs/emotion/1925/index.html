參加重要飯局，掌握這三個不成文的規矩，讓領導對你高看一眼

    • 在我們國家，酒桌文化已經發展了兩千多年，無論談心交友，還是談事簽訂單，都離不開酒桌；大家在職場中打拚，即使不喜歡這種酒局，也是大家無法避免的，很多時候不得不參加這些酒局；

    因此，大家有必要了解我們國家的酒文化，在酒桌上才能更好地與朋友聯絡感情，和發展自己的人脈。

    在參加飯局的時候，也有很多年輕人他們不懂得規矩，如果年輕人能夠弄明白以下這三個原則，相信在酒桌上你也能夠被領導刮目相看。

    1.互相尊重，互相理解

    在進入職場以後，其實很多年輕人即便是20多歲，他們也已經算作是成年人，所以在跟他人相處時，一定要學會互相尊重，互相理解。

    儘管你在公司是領導是管理者，也不要總是在公司給自己找麻煩，因為你的所作所為已經證明了你想跟同事或者其他員工有矛盾，那麼公司里所有員工都會想出很多方法來收拾你，

    甚至會從公司辭職，那麼在管理員工或者經營公司的時候，你遇到麻煩，那麼你的決策權會大大降低，公司的上級也不會對你認可。

    所以無論是勸酒還是敬酒，都應該適可而止，如果你不尊重他人，那麼他人更不會尊重你。

    2.學會察言觀色

    大家都應該明白，職場是個非常現實的地方，很多職場人之所以能夠在公司里混得風生水起，正是因為他們能夠給公司帶來非常多的效益。

    公司的領導並不是糊塗人，他們再帶員工去參加飯局的時候，如果看到員工碌碌無為，在酒桌上不願意和客戶有說有笑，甚至陪客戶喝酒。

    那麼領導會覺得這類員工很沒有眼力見兒，根本就不怎麼懂規矩，其實作為員工大家在跟領導參加飯局的時候，應該學會察言觀色，知道領導和客戶心裡在想什麼，你才能夠跟他們聊出感興趣的話題。

    3.點菜要學會照顧到別人

    大家要明白，在酒桌上除了簡單的吃飯和談事外，領導還可以在你點菜的時候，看清楚你的人品，如果自己能夠做到會點菜，自然就會讓領導刮目相看，這也是大家皆大歡喜的結果；

    大家要明白，一個不會點菜的人，還會直接影響同事對你的評價，相信大家身邊都有這樣的同時，在參加部門聚會時，通常都會點自己喜歡吃的菜，不顧及別人的口味，這種下屬會給領導留下自私的印象；

    而那些習慣點便宜菜的人，也會讓別人覺得小氣；大家在點菜時，可以參考參加就餐的人數，有多少同事參加，就點多少餐，通常一個人點一道菜就可以了，如果參加的男生較多的話，還可以多點幾道菜；

    只要多為身邊的著想就可以了，這樣不僅會給領導留下好的印象，還能提高自己的好人緣；所以說，大家參加職場飯局時，要學會這些點餐的方式。

    這些事情有些領導會認為你是應該知道的不會告訴你，也有些領導不好直接和你說需要你自己去體會。所以，飯局上如果能做好這些東西，無異是在給你自己的職場加分，同時也給領導漲了面子。
