完善自己，才能抵達幸福；善待別人，才能淡化痛苦，早安

天天要聞
2019-12-21

1、為人處世，讓一步是心，退一步是情，謙讓是靈魂，忍讓是心胸。好多的時候，我們之所以贏得進步，就在於我們謙讓的心靈。許多的時候，我們處世不驚，就在於我們忍讓的心胸。謙讓似水，看似無形，卻時時佔據著有利地形；忍讓似草，看似渺小，卻處處點綴了大地的風景。盛氣凌人不算能人，凌強欺弱不是才能，待人最宜謙遜，處事最應寬鬆。謙讓，忍讓，是人至純、至美的心靈。

寧靜是裝不出來的優雅，心事是裝滿桶的水，有一點縫隙都會外溢。人生是一方田，種啥得啥，因果循環，誰也逃不離。痛就痛了，痛讓你認識自己，也看清了別人。敗就敗了，站起來拍拍灰塵，多少人在乎你摔過跤。人間多少事，看清了想開了，淡了忘了，就靜了，優雅是心底的寧靜開出了美麗的花。早安！

2、人生如戲，或長或短，或悲或喜，都有我們的影子。世事如劇，或難或易，或起或伏，都是我們的鏡子。那些得失，那些起伏，沉澱了我們的劇情，讓我們看到曾經走過的路徑。告訴我們，曾經我們是那樣認真，走的又是那樣匆匆。於影中我們看到了曾經，於鏡中我們理解了曾經。記住也好，忘卻也罷，一幕總是接著一幕，走好現今，才是最主要的劇情。

努力了便明白，追求了便知道，人生就沒有完美。完美只是一種渴望、一種想像、一種心態。現實的生活中，根本就沒有完美，也不可能完美。曾經的歲月里，我們對人生有好多的渴望，期盼完美。經歷了才知道，才明白，原來努力不一定會得到，付出不一定能回報，完美，只是驅使著我們努力的情態。誰也會怨恨，誰也會生氣，但我們總不該因別人的錯誤，老是責怪自己。早安！

3、生活中錯誤難免，失誤常見，我們總不能讓自己常常生氣，就算錯了也是正常，也是在所難免。不犯錯誤的人是沒有的，關鍵是下次注意，今後在意。人生著眼的應該是現在，不要老是盯著過去，給別人一個機會，就是給自己一種歡喜。聰明不一定就有智慧，但智慧一定需要聰明。

生活中有時我們痛苦，並不是源於貧窮，也不是源於認真，而是在於我們過於聰明。許多時候聰明成了我們的弊病，好多地方聰明成了我們的短處，水至清則無魚，人至察則無徒。生活好多的時候，需要的恰恰不是聰明，而是糊塗。為人不能太過聰明，處事不能太過精明，一如酒，不能太烈，就像茶，不能過濃。一半聰明一半糊塗，說烈不烈說濃不濃，最是醉人。人生聰明難，糊塗更難，難在適宜，適度，適當。天堂就在心中，地獄也在心中。早安！

4、如果你的心中一片陽光，待人接物，充滿著真誠，不管身處順境，還是逆境，都能保持善良的品行。不驕不躁，不亢不卑，寬容對人，樂於助人，天堂就在你心中。如果你的心中一片陰冷，為人處世，奸詐陰險；不論是得意，還是失意，都不能善良待人，事事算計別人，處處損害他人，地獄就在你心中。任何時候，天堂都是留給善良之人，任何地方，地獄都是預備給姦邪之人。努力就要從心開始，才能擁有美好的心靈。完善自己，才能抵達幸福；善待別人，才能淡化痛苦，心態安好，則幸福常存。

樹欲靜而風不止，能真正救贖自己的，不是流淌的時間，也不是他人的幫助，而是自我良好的心態。時間的變換，去了舊傷或許又來了新疤，別人幫得了你一時，幫不了你一世。為人處世，把心放平，把心放寬，輕鬆自在。時刻記住：完善自己，才能抵達幸福。善待別人，才能淡化痛苦。早安！
