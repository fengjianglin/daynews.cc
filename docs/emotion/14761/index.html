想別人不敢想，做別人未曾做，創新才能出人頭地，早安

1、也許有人生活窘迫，也許有人工作失意，也許有人學業失敗，也許有人愛情落寞。每個人都有每個人艱辛難熬的時候，熬過來了，一切都會豁然開朗，所以你不必害怕，生活並不只是針對你。如果生活一開始就是簡單的，那還要那麼長的人生幹什麼？

2、人生不可能完美，做人不能太較真，不能自己跟自己過不去。畢竟人的一生中，期望與現實常常會發生衝突。生活本身不可能事事遂人所願，人生也不是理想的化身，雖然我們也辛勤耕耘，但總有一些東西我們一生都不可能得到。早安！

3、如果你仍習慣於常規思維，墨守成規，那你註定地要落在人後，毫無優勢可言，機會也就與你無緣。所以要出人頭地，就必須具備創新才能：想別人不敢想，做別人未曾做。早安！

4、固然繁忙將問候落下，雖然歲月將舊事推遠，但每一次回憶，每一次立足，每一次想起你，心便溫馨怡然，願我的祝願伴你每時每刻，願你今日有個好意情!

5、早上醒來笑一笑，心情快樂精神好；活絡脛骨扭扭腰，生活健康每一秒；事業芝麻節節高，好運連連天天妙！願清晨好心情，早安！

6、一個人的自愈的能力越強，才越有可能接近幸福。做一個寡言，卻心有一片海的人，不傷人害己，於淡泊中，平和自在。

7、沒有一顆心會因為追求夢想而受傷，當你真心想要某樣東西時，整個宇宙都會聯合起來幫你完成。早安!

8、那些嘗試去做某事卻失敗的人，比那些什麼也不嘗試做卻成功的人不知要好上多少。早安，加油！

9、抽出時間去學習，凡事從小做起，不怕單調和重複，長期的積累堅持，想不成功，也難。早安！

10、太陽落了還會升起，不幸的日子總是會有盡頭的，過去一直都是這樣，將來也會是這樣。早安！

11、努力從來不會白費，今日撒下種子，正在你看不見、想不到的某處，悄悄地生根發芽。早安!
