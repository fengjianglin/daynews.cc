“父母生我養我，我卻不想給他們養老”

2019-12-19 2,135條評論 5,000次閱讀 4001人點贊

孔子他老人家在很早前便教導我們「孝悌忠信，禮儀廉恥”，而且」孝「放在首位，可見他老人家對孝道文化的重視程度。

【孝：是孝順。孝順父母和長輩，這是為人子女的本份，孝順是報答父母養育之恩。

孝敬是有感恩之心的基礎，是慈悲的基礎，是一切美德的基礎。所以，有「百善孝為先」之說。】

父母養我小，我養父母老，這不是天經地義嘛，我從來不覺得有任何問題。

但是今天越來越多的年輕人覺得自己生活已經那麼艱難了，還要給父母養老，甚至覺得這是社會給他們的道德綁架，可是真的是這樣嗎？

假如你覺得給父母養老是道德綁架，那麼只能說明你父母生你生的冤枉。

說實話，我媽生了我和兩個弟弟，且我和我兩個弟弟都分別隔一歲，所以我媽把我們帶大真的很不容易 。

長大後，特別是我和弟弟們也分別都有了自己的孩子，更加理解父母的不易，我們都心照不宣的想要讓媽媽晚年過好，只想把自己最好的給到父母，以報答他們對我們的養育之恩。

我們從來不覺得父母以後依靠我們是種負擔，相反我們覺得能在父母有生之年讓他們幸福安康是我們的最大幸福。

可能是整個家族的氛圍也比較好，所以我所有堂兄弟姐妹都對自己父母不錯，並且就覺得父母生了我養了我，我們就應該養父母老，這是一種再正常不過的想法了，並且我們一直在為之付諸行動。

我真的見不得那些不贍養老人對老人不好的人和事，想想每個人從來到世上那一刻起，我們的父母便要付出自己的畢生精力來撫養我們，其中的心酸苦楚只有他們自己心裡清楚 ，但他們卻從來不會在我們面前表露，這份愛真的是沉重而無私，我們憑什麼不在他們的遲暮之年還他們的哺育之恩？

烏鴉還知反哺之恩，那麼我們作為高級物種–人類，為什麼就不如一隻禽獸？

我只知道，在父母有生之年 ，我要好好保重自己的身體，然後努力賺錢，帶著他們去他們沒去過的地方走走看看，讓他們的皺紋可以慢點爬上來，讓他們的笑容再綻放久一點，我想再多點時間盡我為人子女的責任。

所以父母生了我養育了我，我就應該為父母養老，這不是道德綁架，而是生而為人最基本的倫理道德。

在物慾橫流的今天，老子他老人家所傳承的孝道文化越來越不被當代年輕人重視，我們都活的越來越自私，只能看到眼前的所謂困難，卻從沒感受到父母對我們的付出，我們已經完全迷失了自己。

當我們消耗掉父母的青春，便想著把他們棄之不顧，這種行為真的說明我們雖然科技在進步，但是我們的思想文化實在退步的可怕！

每個人都有老去的那 一天，如果今天我們背棄了父母，覺得養老是對我們的道德綁架，那麼以後我們的後人也會同樣的效仿我們，如此下去，我真的不敢想像我們的社會將來變成什麼樣？我們傳承幾千年的人類文化將會走向何方？

從小的方面講子慈母孝，家庭才能幸福安康；往大的方面講，中華孝道文化能夠代代傳承並發揚光大，我們的中華民族才能一直屹立於世界巔峰。

願我們每個人都能尊師重道，願我們每個家庭都能充滿歡聲笑語，願我們的社會一派詳和。
