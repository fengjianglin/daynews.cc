身處社會難免要說話，人際交往做到5點，讓溝通事半功倍

2019-12-17 46條評論 4,381次閱讀 1935人點贊

在這個發展迅速的社會，每個人都希望能夠提升自己的各項能力，讓自己在未來的職場生涯中能夠有更強的競爭力，也可以給自己一個更好的生活保障。

但是如果問起哪一項能力是每個人都需要，但是卻只有極少數人能夠做好這個問題時，大家給出的答案都是五花八門。

如果要我說的話，必然是溝通能力。

我們生活在這個世界上，每天要和那麼多的人打交道，無論是早上賣早餐的阿姨還是公司里的同事，在和別人交流的時候，就能很好的體現我們的人際溝通能力。

如果缺乏一個良好的人際溝通能力，不僅可能讓別人覺得相處不舒服，而且還容易讓自己陷入到一個尷尬的境地中。

我有這樣一個朋友，在短短的一年之內已經相了十幾場親，但是最後結果都是以失敗告終。

並不是說這個朋友的條件有多差，這相反他屬於中上條件的人，但是就是因為他缺乏人際溝通的能力，在和別人聊天的時候，總是只會用一問一答查戶口式的聊天方式去溝通。

這樣的聊天方式，讓別人覺得十分不舒服，所以和他見過面的人基本都沒了下文。

那麼想要改變現狀，提高我們的溝通能力，最重要的就是做好這5點，讓溝通這件事變得更加輕鬆容易。

1)做好溝通前的準備

或許有人會感到不理解，只不過是簡單的聊個天，為什麼還要做那麼多的準備。正所謂準備的越充分，就能讓我們自己更加自信。

例如說今天有個重要的會議要和對方的負責人洽談，那我們在此之前就應該做好相關的準備工作，圍繞著他們的工作內容以及我們需要了解的內容，提出人家感興趣的話題，這樣也能讓別人有一個良好的印象。

而且最重要的是我們要根據對方的身份和生活環境來提出問題，並且不要一直掌握著話語的主動權，適當的將發問權交給對方，在一問一答中才能更好地加深對方對你的好印象。

2)根據對方的個人背景，調整溝通策略

對於不同的人，我們應該採用不同的溝通策略，例如對方的年內，性別，職業，興趣愛好等等，在這樣的背景作用下，我們能夠判斷出對方是屬於什麼類型的人，選擇最適合他的溝通方式。

例如說和剛畢業的大學生，我們就可以聊聊工作選擇人生觀價值觀等理想性的問題。

而對於那些人到中年的職工，我們更多的可以聊一聊他的孩子房價物價的實際性的問題。根據對方身份角色的不同來變換聊天的策略，這才是最為明智的選擇。

3)多聊對方的興趣愛好或專業知識

對於任何人來說，都希望自己能夠得到別人的關注和欣賞，如果我們能夠多聊別人感興趣的話題，或者多說一些他們專業上的知識，讓他感覺到自己有了表達權，這樣才能促使他有聊天的慾望。

如果我們一直掌握主動權，將話題掌握在自己感興趣的範圍內，而不關注別人的感受的話，那麼這樣的話題必然會草草結束，並且會引起對方的不適。

4)給對方製造表達的機會

有些人之所以會覺得聊天冷場，是因為總是想通過自己的力量去打破冷場的尷尬而忽略了調動對方聊天積極性的關鍵問題。

我們應該注重去給對方製造表達的機會，讓對方能夠充分展示自己的才能，這樣他們才能從內心感受到你對他們的尊重，從而也會願意和你做朋友，迅速的拉近彼此之間的距離。

5)善用讚美和批評

每個人都喜歡聽好話，但是一定要注意好畫的標準是真誠且不浮誇，在給對方真誠讚美的同時，能夠讓對方更有動力去奔向目標，發自內心有理有據的讚美才是有效讚美，反之只是一味的拍馬屁會讓對方覺得你是在敷衍，甚至是讓別人厭惡。

同樣在別人犯錯的時候，我們也可以適當的進行批評，最好的方式就是採用「三明治式」的批評法，先提出他做的好的地方之後再指出他做的欠缺的地方，最後在附上對他的期望和讚許，這樣的批評方法能夠讓對方更易於接受，同時也不容易，對你產生其他誤解。

無論是怎樣的溝通方式，我們都應該根據對方的性格來不斷的調整和改變，並且在之後的實踐中不斷的完善自我，這樣才能讓自己變成名副其實的人氣王，受到大家喜愛的同時，建立更多穩固的人脈關係。
