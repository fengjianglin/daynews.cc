當代社會幾大狠人：母胎單身強勢上榜

2019-12-21 362條評論 1,432次閱讀 381人點贊

30歲，好像是女人的分水嶺。如果30歲沒有結婚，沒有生孩子，你一定會被冠上，大齡剩女，性格缺陷……的標籤。

你以為這是現在身處城市女人最大的悲哀嗎？她們可能面臨青春流逝的焦慮，被一群人挑挑揀揀，被阿姨媽媽，甚至被陌生人指指點點。但30歲單身，都不是最慘的。

最近，一個婚戀網站發布一個調查，其中有一個非常戲劇化的故事：男生覺得跟一個沒有戀愛經歷的女生在一起，很累。

「如果女生年過30還沒有戀愛過，第一反應不是女生單純我喜歡，而是她會不會有問題。」

當我問到那些母胎單身的人，為什麼單身至今，其中有個想了很久回答我：

「我覺得我不需要愛情啊。我父母很愛我，朋友也很愛我，我又不缺那個人愛我，所以我也不急。」

她看到身邊，幾乎所有的愛情和婚姻，都以慘烈，不幸的結果收場，她並沒有看出來戀愛有什麼快樂。

而另一波人，她們因為擔心戀愛後，可能需要面對分手的悲痛，決定還是不要開始比較穩妥。

女 神 還 單 身

之前有一條熱搜，#劉雯稱沒有談過戀愛#。乍一看很奇怪，世界頂級模特，形象氣質過人，談吐自信大方，說劉雯是女神，恐怕沒有異議。結果女神是個母胎單身？

但劉雯就是這麼公開表達的。

在《奇遇人生》里，劉雯和節目組去北極。大家聊到人生中最糟糕的約會。劉雯說：「我覺得自己不用回答。」

結果大家逼問，她來了一句：因為我沒有男朋友，我從來沒談過戀愛。

一個女神，30多了還沒談過戀愛，這可能嗎？太可能了。甚至可以說，這都快成為常態了。

現實就是，在一線城市，只要女生咬定不唯愛，不空虛，不將就，單身的概率就會高居不下。有人會問，總是單身行動獨來獨往，不需要和人連接嗎？

當然需要。問題是生活中的男生，基本連接不上，硬聊強聊，有什麼意義呢？

比起一次失敗的相親，和閨蜜一起打卡網紅餐廳不好嗎？比起一個話不投機的對象，養個寵物懶在家裡晒晒太陽不舒服嗎？

《奇葩說》里馬薇薇說，你沒有愛了，你需要陪伴，養條狗啊！

母 胎 單 身 表 現

男生會願意追求母胎單身的女孩子嗎？很多人都表示年輕時候的我願意，現在我不。為什麼？因為年輕時候願意的那幾次，太痛。

人生漫漫長路，誰都渴望著被愛。而女人在這個世間更渴望得到一份至死不渝的愛情，特別是那些情竇初開，戀愛經歷少的女人。

不僅在感情里會傾盡所有，還會為愛瘋狂。而傾盡所有的瘋狂，太可怕了。

喜歡、迷戀某個人，期待愛情的到來，這是人的天性，沒啥好羞恥也沒啥好高興的。關鍵在於是否能「盡在掌握中」。

簡單的說能否控制住自己「期待」的時間和強度。這一點對於處在戀愛關係中的人尤為重要。

可母胎單身的女孩子，傾向於對戀愛抱有不切實際的期待。因為她沒有體驗過，不知道實際的愛情是什麼樣子的。

就像沒吃過豬肉，即使看過再多豬跑，等吃到豬肉的時候，你也無法讓你腦海里豬肉的味道完全符合自己當下嘗到的實際味道。

而一旦她不能或者不夠能滿足自己的這種期待，那麼就容易陷入患得患失、孤獨、憤怒、氣狠等狀態當中。

與其說，這些女孩是被剩下的，那我更加覺得，這些30歲依然母胎單身的人，是自己選擇單身的。她們雖然生理年齡到達30歲，但她們的心理活在18歲，幻想著理想化的愛情。

那份愛情，或許需要財務和社會地位，但更需要一份純粹的，充滿幸福，沒有生活糟粕的感情。

她們在戀愛這件事，或許比所有人都執拗，但她們對於它的期待和要求卻也更高。
