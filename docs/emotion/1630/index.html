跌倒了，爬起來重新來過。不經風雨怎能見彩虹，相信下次會走更穩

人生不易，生活很難，家家都有難念的經，人人都有辛酸的淚，沒人知道，痛苦會在哪個時段出現，也沒人清楚，煩惱會在哪個地方出現。我們能做的，就是做好自己，珍惜擁有。

遇任何事都不要急躁，等一等，緩一緩，不要急於下結論，尤其生氣時不要做決斷，要學會換位思考，大事化小、小事化了，把複雜的事情盡量簡單處理，千萬不要把簡單的事情複雜化。

人活著就是為了解決困難。這才是生命的意義，也是生命的內容。逃避不是辦法，知難而上往往是解決問題的最好手段。

人多的地方，盡量少說話，話多言必失，良言一句三冬暖，傷人一語六月寒，管好自的嘴，講話不要圖一時痛快、信口開河，說話要過腦子，私下裡不說別人的壞話，因為沒有不透風的牆，背地裡多說別人好話，別擔心傳不到當事人耳朵里。

生活不可能一帆風順，總會有波折，總會有險阻。生活是個愛開玩笑的孩子，也許今天給你所有，明天又會讓你一無所有，無需煩惱，該來的總會來，再黑的夜晚也會有黎明到來的那一刻。不管生活有多麼曲折，只要擁有幸福的態度就能挺過漫漫長夜，就能迎來美好的明天。
