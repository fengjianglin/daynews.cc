人世喧囂，名來利往；放下浮躁，心靜自安，早安

2019-12-19 319條評論 1,499次閱讀 554人點贊

1、 總有起風的清晨，總有暖和的午後，總有絢爛的黃昏，總有流星的夜晚，所以不如保持順其自然的心境，把握每一個瞬間，試著去做，去面對每一個昨天、今天和明天。人生中的成敗得失，全憑把握，縱使歷經所有的艱辛苦難，始終要保持一種心境——-順其自然。

2、 堅持是聰明的，堅持是愚蠢的，聰明與愚蠢之間，差異的原因其實只有結果，這是唯結果論的慣用伎倆——因此這主要適用於旁觀者而言。於自身而言，是否堅持，始終是你當時所認為最明智的選擇，因此，聰明還是愚蠢，都不是重點，重點是你做出了選擇。早安朋友們！

3、 生活中總有黯淡無光、咬牙硬撐的時刻，也許奮鬥有時會很累，但別忘了還有未實現的夢想。最艱難的成功，不是超越別人，而是戰勝自己；最可貴的堅持，不是歷經磨難，而是保持初心。放下內心的陰霾，放下無謂的負擔，努力向前吧！追光的人，自己也會身披萬丈光芒。

4、 我認為世界上最大的謊言有兩個：一個是我沒時間，一個是我沒錢。這一切都要看你，把自己的事業看的有多重，如果你真的想靠創業改變命運，那你必定可以把午休睡覺等碎片的時間充分利用起來，如果你只是玩玩的心態，那自然沒有多餘的時間。有夢想，永遠有干不完的事！

5、 只有經歷最痛苦的堅持，才配得上擁有最長久的幸福。真正能讓你倒下的，不是對手，而是你絕望的內心。人的一切痛苦，本質上都是，對自己無能的憤怒。活著就意味著，必須要做點什麼，請好好努力。任何值得去的地方，都沒有捷徑。記住一句話：「越努力，越幸運」。早安，！

6、 很喜歡這句話：不是井裡沒水，是挖的不夠深，不是成功來的慢，是放棄的快。所以成功不是靠奇蹟，而是靠軌跡。美好生活要有四度空間：寬度，深度，熱度，速度。成功者的工作狀態需具備五動：主動，行動，生動，帶動，感動。送給在奮鬥在路上的同路人，失敗的人習慣了放棄，而成功的人永遠選擇堅持！

7、 人世喧囂，名來利往；放下浮躁，心靜自安。人生難得一心靜，只有「寵辱不驚」，方能心態平和、怡然自得；只有「去留無意」，方能豁達樂觀、淡看風雲。所謂「心靜」，就是要非絲非竹而自恬適，非煙非茗而自清芬，不眼熱權勢顯赫，不奢望金銀成堆，不祈求聲名鵲起，水流任急境常靜，花落雖頻意自閑。早安！

8、 奮鬥，如果你不握緊拳頭，你就不知道你力量到底有多大；如果你不咬緊牙關，你就不知道你的堅持到底有多狠；如果你不跺腳而起，你就不知道你的果決到底有多正確；如果你不瞪眼，你就不知道你的內心到底多強大。最強悍的競爭力只有兩個字：行動奮鬥。只有當你足夠努力，你才會足夠幸運。這世界不會辜負每一份努力和堅持。

9、 又到一年將盡時，歲月輪迴多少事！彷彿是轉眼間，一年只剩下最後幾天，回想這一年，你還記得過去的每個月都經歷了什麼嗎？無論是經歷了怎樣的掙扎與挑戰，無論流過多少汗水與淚水，這些都已是過去，新的一年馬上就要來了，讓我們帶著善意、感恩和赤誠，更好地生活；願我們在美好的冬季里，感受溫暖，繼續前行，喜樂平安！

10、 人活這一世，走自己的路，修自己的心，人走這一程，用自己的心，交知心的友，用心辨人心，真心換真心。人人有雙眼，個個有顆心。用眼看世界，清晰而美麗，看過品過，世間冷暖早入心。有時候你再落魄，也有人真心的拉你一把，慢慢看，還是好人多，我們學會感恩。有的事不容易掌控，該放的放，該忘的忘，該守的守，該惜的惜。學著堅強，走穩自己的路，遵循自己的原則底線！早安！
