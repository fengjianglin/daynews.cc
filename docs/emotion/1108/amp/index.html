摘要

朋友說他前段時間，替一個生病的同事大山做了一項工作，沒想到他的項目設計靈感，正好被領導誇讚了。於是，大山就以為，朋友時故意為了搶他的風頭，正好趁他不在，就在領導面前表現，可朋友真的沒有這樣的想法。

於是，大山就以為，朋友時故意為了搶他的風頭，正好趁他不在，就在領導面前表現，可朋友真的沒有這樣的想法。

於是，大山就開始在公司內部散播朋友不好的言論，這讓其他同事都對朋友議論紛紛，朋友受不了壓力，就被迫辭職了。

朋友說：其實這項工作是領導讓他做的，他真的不是故意要搶大山的風頭，早知道是這樣的結果，當時就拒絕了。

其實，職場上每天都有很多這樣的事情發生。因為不懂的拒絕別人，最後卻落得個不好的下場。

作為一名有7年資深經驗的職場達人，今天就來和你探討一下，職場上，什麼樣的事情一定要拒絕。

01替同事傳話

很多人看到這的時候會想，同事之間傳話，是一件很正常的事情啊，根本不是什麼大不了的。

但是在職場中，替別人傳話卻沒有這麼簡單。在幫助同事傳話時，一旦最後出現了問題，那麼就會由你來承擔責任，因為是你在傳達信息時出現了錯誤。

前段時間，朋友就因為傳話，造成了一場同事誤會。

事情的經過是這樣的，朋友的同事小吳因為家裡有事要請假，就讓朋友幫他給另一個同事小劉交代一下工作。

等小吳請假回來後發現，小劉的工作和小吳的要求很多地方都不一樣。他就認定是朋友故意傳錯話，想讓他們再重新做一遍。

還再公司向其他同事說朋友的壞話，導致朋友被孤立，最後受不了，就辭職了。

所以，當有同事讓你幫忙傳話時，能拒絕則拒絕，不要因為一時的好意，造成嚴重的後果。

02搭你的順風車

為了緩解交通壓力，各大城市都開始施行了車牌限號政策。如果你恰好有車，礙於面子，且順路的話，會讓其他同事搭你的車回家。

時間一久，這個同事就會形成習慣，天天搭你的車，你要是某一天拒絕了，他就會認為你是個惡人。

所以，盡量在剛開始的時候，就避免他搭你車的事情發生，那樣就不會再發生後面的事情了。

03替同事背黑鍋

都說職場如戰場，有時候看似關係親密的同事，也可能再背後捅你一刀。

最近，領導讓我和小王兩個負責一個項目。我們兩個做好了分工，我負責對接客戶，他負責數據報告的分析。

在我給客戶看文件的時候，客戶直接就指出了文件中的錯誤，並且說：你們公司也太沒誠意了。

於是，就拒絕了這次的合作關係。回到公司以後，領導把我倆交到了辦公室，問是誰做的文件。

還沒等我開口，小王就說是我倆一起做的，這讓我百口莫辯。最後就受到了領導的處罰。

所以當你和其他同事共事時，一定要提前和領導報備好你們的職責分布，這樣才能避免提同時背鍋。

最後

在工作中，同事之間利益至上，不要因為一時的心軟，就不拒絕同時提出的要求。有些事情，越早拒絕越好。
