常找不到人！最不愛報備行蹤星座Top5　射手厭被管：沒意義的蠢事

2019-12-10 715條評論 1,653次閱讀 724人點贊

你的另一半也是一出門像走丟，回來像撿到嗎？（示意圖／記者李佳蓉攝，下同）

文／

不是在討論要不要報備、或是和情人報備好不好。而是有些人在談戀愛交往後也不會習慣和自己的情人報備、告知自己的行蹤，有人能接受這樣的狀況，有的人則完全不能接受，報備行蹤對於某些人來說是一種安全敢的來源，但是不愛報備的人也覺得這是沒有意義的事，愛不愛與報不報備不見得有關係，哪些星座的情人談戀愛後還是不愛報備行蹤或不習慣被掌握行蹤呢？跟著亞提米斯一起來看看吧！

請參考：太陽、金星星座

★雙子座：

報備是很麻煩的事情，有時候想法一變就什麼行程都變了，哪那麼多功夫一直報備？

雙子座有時候是一種很懶的生物，他們本來就不喜歡被綁住、被拘束著的感覺，談了戀愛當然也不會改變雙子座喜歡凡事有變化的態度和想法，報備就是一種會讓雙子座覺得很無趣、又有一種被綁住的感覺，因此不少雙子座是不愛報備的情人。

很多人可能不曉得，雙子座其實是很極端化的一個星座，他們的想法有時候是會完全翻轉、而且就在他們的一念之間，就像是一個雙子座可能會告訴你，他覺得某部電影很糟糕、但是沒多久後你可能會發現雙子座跑去看這部電影，就算你問雙子座為什麼會突然改變主意？雙子座也會有很多聽起來很合理的理由回答你，為什麼他們會做出這樣的選擇。

總之是念頭一變，雙子座就可能轉個方向去做那樣的事，他們的思考非常的複雜，很常會發生的情況是原本和朋友A約了要喝咖啡，卻在約定的時間前突然想改期，然後約了朋友B去逛街之類的，雙子座還滿習慣自己這樣的態度變動，也不見得一定要什麼理由。

談起戀愛後，如果樣樣事情、去哪裡都被要求要報備，沒報備情人就會抓狂的話，那麼雙子座往往在這段戀愛當中也撐不了多久，因為雙子座本身的抗壓性其實沒有很好，他們習慣用不斷的變化去消解掉壓力，可是如果感情上的另一半就是非要雙子座坦誠報備行蹤的話，有些雙子座會以分手來相逼，理由當然是「我們不適合」。

比較常見的就是開始說謊，反正雙子座也不覺得那是說謊，只是一個理由、一個藉口、一個安撫，說穿了就是混過去，下次還是不會想要報備。

很多雙子座的名言就是，「不報備又不是不愛你，一邊報備還邊偷吃的還不是大有人在？」

嚮往自由的射手座，最討厭被人控制和束縛的感覺。

★射手座：

想去哪就去哪，為什麼一定要報備？我都不見得向我爹娘彙報行蹤了。

射手座不是不看重情人，而是很多射手座就是我行我素慣了，他們有時候就是脫疆野馬、愛怎麼做就怎麼做，不喜歡有被管束或被束縛的感覺，因為他們很重視個人的自由度，不論是在心靈或是在身體方面，射手座都想要保有自己絕對的自由。

說坦白一點就是連射手座的爸媽都不見得管得住射手座，這是一個天生就很愛跑來跑去的野孩子（笑），他們很有勇氣、很敢到處闖、很敢向前沖，有時候不報備要去哪是怕被阻擋，很多射手座總是喜歡去自己想去的地方，問題是這些地點可能在別人耳里聽來就是個危險的地方，談起戀愛之後很多射手座還是一樣上夜店喝酒小酌，聽在有些另一半的耳里真的是相當的刺耳，當然不會願意射手座去這樣的場合，免得又喝出問題來。

「才不會發生什麼事」，很多射手座心裡頭就是打定了這樣的主意，他們劍及履及、想到了就想要去，所以最好的方法當然就是不告訴感情上的另一半，自己等等要去哪裡、做什麼事，免得講了還要朋友幫忙說謊，根本就是一整個麻煩。

是的、麻煩。射手座也是很懶的生物，或該說他們不喜歡去做這些瑣碎的小事情，完全沒有什麼幫助也沒有什麼益處，也許在談戀愛剛開始的時候還會因為情人的要求就把自己的行蹤報備一下，但是久了之後就會越來越懶、越來越嫌麻煩、越來越不想說。

很多射手座是很有勇氣去挑戰某些要求或某些規範的、算是一種爭取自由的行動，因此談戀愛久了就會反過頭來要求對方不要查勤、不要要求報備，如果另一半不接受，這也就容易成為兩人之間感情不睦、起磨擦的原因。

他們就是喜歡自由自在的感覺，「報備行蹤」對射手座來說是件沒有任何意義的蠢事。

★獅子座：

理論上來說，應該是你向我報備，怎麼會是我一定要跟你報備？

獅子座在談戀愛的時候真的不愛屈居下風，不喜歡被當成下人來使喚，當一個獅子座認為「報備」是一種下對上才有的行為時，就會打從心裡頭不想要去做這件事，不是為了想要去玩還是做了什麼不可告人的事情，就只是覺得這樣會低人一截，好像自己被另一半管得死死的，這樣傳出去實在是太沒有面子。

也有些獅子座其實本身就很懶，這種慵懶就是一種絕對的懶惰，能不要做就不要做，能不用講就不用講，什麼事都懶得多說，報備更是一點都不想做，因為覺得這樣很多餘，而且也不覺得做了這種事就真的有什麼用，所以如果另一半沒問、獅子座有時也懶得說、根本不會主動想報備；偶爾聽到另一半抱怨或要求，才會敷衍式的報備一下，否則很多時候獅子座是會因為懶而一拖再拖，拖到最後就乾脆不報備。

在獅子座的心裡頭，他們是很「喜歡」被尊重的，「報備」就是一種尊重，所以這樣的尊重往往都需要是對方先給，獅子座是會看今天有沒有出太陽、心情好不好才會想想要不要向另一半報備自己的去處，這種帶點小任性的大爺心態，往往也有時候會把另一半氣個半死。

對於有些已經「看很多」，也就是大風大浪都見過的獅子座，他們會覺得報備是沒有意義的，因為看過太多「一邊報備一邊偷吃」的狀況，獅子座其實很清楚，這種報備根本就只是一種更沒意義的敷衍，講難聽點根本就是報備心酸的，也不見得能夠理解，為什麼某些人就是非聽到這些報備的言語才會覺得安心？難道聽謊話會比較開心？

在不喜歡被管太多、不喜歡在朋友面前沒面子、個性又還滿懶的種種原因加成之下，獅子座就成了典型不怎麼喜歡報備的情人。

水瓶沒報備的原因有很多，但和愛不愛另一半沒有必然關聯。

★水瓶座：

有時候是忘了、有時候是懶惰、有時候是故意，就是想實驗不報備會怎樣？

水瓶座是向來很隨性的星座，對許多的事情往往都不喜歡有個固定的標準，太有規則性的事物會讓水瓶座生厭，因為不少水瓶座很喜歡挑戰規則，所以不少跟水瓶座交往的人往往也都會覺得好像跟一個很愛唱反調的人交往，你越是要這個水瓶座做什麼、他們就越是不那麼做、或是總有一些非常怪的理由跟你辯。

報備不見得是壞事，水瓶座的抵抗也並不是他們做了不能說出來的事，很多水瓶座是很肯定「報備」會帶來安全感的，問題是「肯定」不代表他們就「願意做」，水瓶座做人處事的標準有時候是沒有辦法用正常邏輯去理解的。

不喜歡被訂一些規矩，也不見得要照別人認定的標準走，因此水瓶座並不覺得報備是很必要的事情，往往水瓶座只會覺得這樣是很無聊的事，講或不講和愛或不愛根本沒有必然的關聯性，也不是在乎對方就一定會報備，難道不小心忘記一次就是什麼濤天大罪嗎？

不少水瓶座是有想到才會說一下，沒有想到就不會說，通常不會想到的機率是比較高的，而如果情人要求越多、水瓶座就越不見得會想報備，有的水瓶座還會上演失蹤記，他們有時候會是想證明「不報備我還是很愛你」，但有時候根本不是想證明些什麼，只是不想說自己去哪裡、不想告訴你而已。

有些比較直接點的水瓶座就會說：「我就是不想跟你講我去哪。」

這句話不見得有什麼內藏的意義，不見得是那麼戲劇化的、好像裡頭藏著很多不可告人的秘密，很多時候就是這句話字面上的意思，「不想說」，人都會有不想說話、不想講的時候，這不見得要有什麼理由。

就是不想說，不可以嗎？

★牡羊座：

總是專心忙著眼前的那件事，就也忘了要把報備的事情放在心上。

牡羊座是只要一忙，或是眼前有什麼事情吸引住他們的注意力，牡羊座就很容易忘了別的事情，在愛情裡頭、談了戀愛的牡羊座也是如此，在一開始談戀愛、把心思都放在對方身上的時候，牡羊座是會記得要報備的，問題在於談戀愛之後報備次數通常會越來越少，甚至於一段時間之後牡羊座是會完全忘了要報備的這件事。

不是感情變淡了，這是個一次只能處理一件事的星座，他們個性還滿急的、心裡頭想到的就想要馬上去做完，有時候根本也管不了與這件事無關的其他事情，而且不少牡羊座會很自然的覺得「另一半應該知道我現在在忙」，這種順理成章的態度更會讓牡羊座壓根不會想起要報備的事。

有些牡羊座則是會很自我感覺「已經講了」，但事實上是根本沒有講過，這種情況常常是出自於牡羊座會有要報備的「念頭」，但是馬上又被眼前的事情給拉過去，因此就「覺得」好像有報備過的樣子，也只能說是牡羊座粗心，有的牡羊座還會發生「簡訊已經寫好、但沒有發出去」，卻以為自己已經把報備簡訊寄出的情況。

粗心大意、加上有時候根本就忘記要報備的事，牡羊座的神經本來就不細，也很習慣一股惱的全然投入他們想做的事裡頭(當然也包括很努力去玩)，大部份的情況下、牡羊座都不會是故意不報備，只是他們沒有辦法記住這麼細的事情，甚至於也沒辦法養成報備的習慣，因為太容易忘記了。

倒也不能怪牡羊座記不住要報備的事，往往另一半如果因此生氣，牡羊座是會記住幾回，然後不久後又沒記性，他們的不愛報備往往出自於是粗心，或是手邊的事情太忙、時間太緊，也就不想多花時間做向情人報備的動作，有時候當你問一個牡羊情人為什麼不報備行蹤，他們很常有的回答是：「就忘了啊！」
