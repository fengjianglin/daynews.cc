那些沉睡的舊物，是時候跟你們告別了

天天要聞
2019-12-22

晚上，老婆要我和她一起找她的一個證書，我們翻箱倒櫃，找了好久才找到她需要的那個證書。

我翻出了她的一個大學時期的未綉完的十字綉。那是她在大二時花了好幾個月一針一針繡的，雖然沒綉完，但是也承載著她那時的回憶。

「留著吧，這個氣質與咱們家不搭，又不會裱掛起來，頂多是留個回憶；扔了吧，又覺得怪可惜的。」

老婆開始糾結。

其實類似這樣的舊物，還有很多，都佔據著家裡有限的空間：

她有寫滿筆記的大學課本，我有備戰各種考試的筆記本；

她留著小時候愛不釋手的洋娃娃，我還留著兒時酷愛的各種四驅車、玩具。

我是個囤積狂，遠比老婆多：

多少次都找不到另一隻的單個襪子，是我最喜歡的那雙，但是讓它「形單影隻」這麼久了。

那支陪我走過備考面試最艱難日子的只有一面有聲音的耳麥。

那幾本早己經過時、已經基本上不會再看了的雜誌。

衛生間門口已經起毛球用了很久的腳墊。

你呢？是不也有類似這些雜物？

你長年保存的一隻耳環，你是不是還一直期望另一隻的出現，但它就是不會出現了；

你那支已經過時好久沒用過的口紅；。

你的廚房的某個抽屜里各種各樣雜貨店、超市的收據啦、票根啦，舊發票；

你平時收集的各種曲別針、螺絲釘、橡皮筋之類的小物件，想在用的時候能夠派上用場，但從來沒用過；

你從超市購物回來的購物袋，你原本想利用當做垃圾袋，但是你的洗碗槽底下已經攢了這麼多袋子；

你那些一大串已經好幾年沒有用過、沒有開過任何鎖的鑰匙，然而你並不知道這些鑰匙開哪些鎖。

…………

時過境遷，我們再也不會用到它們了，只是把它放在角落裡，佔據著空間，但是我們對它們充滿了情感和回憶。

它們對我們倆來說，是人生紀念，這些物品承載著回憶。

我對老婆說：「要麼我們舉行個儀式吧，一個和這些物品告別的儀式。」

老婆說：「你有病呀？」

接下來，我這個「病人」向她解釋如何進行「告別儀式」。

重新歸類

當一個物品承載著人生某個階段的回憶，它就不再是單純的物品了，也不再具有它本身的功能了。

中學的校服，現在肯定穿不了了，那它就不再是普通的衣服；

用過的課本，可能不會再發揮它的功能；

未完成的十字綉，也不會成為家中被裝裱起來的藝術品。

這些東西，就需要放在人生物品的環節里。

正式告別

拿起一件東西在手上，閉上眼睛，安安靜靜的回憶當年的使用時間和場景，然後去判斷這個物品在我們人生時間軸上的重要性，有些東西真的很重要並且會一直很重要，而有些東西可能只是當時覺得很重要。

分清了哪個重要，然後輕輕的放下那些要告別的，揮手並鄭重的和它告別，比如，可以在心裡默念：

謝謝你在在我最孤獨的時候陪伴我。

謝謝你陪伴那個不成熟的我。

謝謝你來到我的生命里，曾經帶給我那麼多的美好。

正是因為有了你，才有了現在的我，謝謝你的見證。

再見，我會永遠記得你。

感謝你來，感謝你走。

對於那些需要留存的，可以通過拍照片、改裝等方式留存，未來可以拿出來自己回憶或與家人孩子分享。

鄭重其事、充滿尊重，真誠告別，認真放手，回憶才會翻篇，新的生活才會繼續。

放下過去，才能輕鬆面對未來。

一晚上，我和老婆告別了一大箱子的東西，雖有不舍，但之後覺得空間大了、心也敞亮了。

從經濟角度，高速增長的房價背景下，騰出騰出一平米的空間，相當於多買了一平米的樓房呀。

從效率角度，留下最有用的，未來的工作、生活也不會顯得那麼雜亂，井井有條、整潔乾淨，找什麼都不費勁。

從人生角度，清理物品的過程也是復盤生活的過程。回顧過去，認清現在，看向未來，有掌控感的人生一定是幸福的。

記得一場舞台劇的最後一幕，舞台上出現了很多張椅子，但是只有椅子沒有人，這是告訴我們：我們要小心，否則我們會淪為物質的奴隸，讓生活中的物質擁有我們，而不是我們擁有物質。

是時候跟你們告別了。

舒適代表著有所節制，當你開始過你真正要的生活，生活就再也無法擺布你。

再見，舊回憶。你好，新生活。

2019年即將過去，願你的2020年，有個整潔舒適的開始。
