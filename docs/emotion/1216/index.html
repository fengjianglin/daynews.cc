【1】認真聽。

高情商的人，他們最大的特質，就是對人的一種發自內心的尊重，尤其是在與別人溝通的時候，他們會用非常好的態度，用眼睛看著你認真，聽你講話認真用心聽你表達，所以他們在態度上是非常誠懇的，這種認真的態度，就會讓你發自內心的喜歡這種人，喜歡和他們傾訴自己內心的想法，這樣他就能更清晰和全面的了解你，從而按照你心理上的需求，給你帶來滿足感，別看他們認真聽力，講話這一個簡單的動作，就能夠把你看得更透徹。

【2】控底線。

高井上的人做事都有自己的底線，而且這個底線不會輕易的打破，比如說他們會堅持控制自己的情緒，絕對不允許自己在別人面前發脾氣，更不會因為別人的一些不當行為，讓自己情緒失控，他們這些底線的設定，都是從別人那兒著眼而對自己的一種要求，這一點，我們也一定要認認真真的反思自己，看看是不是做到位了，如果自己經常對別人發火，激化各種各樣的矛盾就說明你對這樣的一個原則始終沒有堅持。

【3】慎考慮。

高牆上的人考慮問題，通常情況下都是非常慎重的，這種慎重體現在不會輕易的答應別人，一旦答應了別人他們就會用盡自己的權力使盡自己的資源，來兌現自己對別人的答應，大家不要把高情商的人與圓滑事故的人畫等號，高牆上的人是站到人性需求的角度上去與人打交道的，而圓滑世故的人，是為了逃避自己應該承擔的責任，兩者的性質完全不同。

【4】無空炮。

高情商的人從來不放空炮，他們非常懂得建立自己信任和口碑的重要性，所以寧可讓自己吃很大的虧，寧肯停止自己手裡面所做的事情，也會兌現向別人的承諾，更不會在事後去埋怨和怪罪別人，在這一點上，他們做的非常到位，總能夠讓別人感動的掉下眼淚來，你說一下，他們這種高情商的行為是不是給自己贏得了良好的口碑，他們的人際關係絕對差不了。

【5】一個樣。

高情商的人對人的尊重，是從骨子裡面迸發出來的，你根本感覺不到他對你的應付，所以他對身邊的任何人都充滿著敬畏感，對誰都是一個樣的發自內心的尊重，這種尊重與人的高低貴賤，沒有關係，這一點真的值得我們敬重他們。
