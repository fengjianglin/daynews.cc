父母在孩子面前爭寵，結果孩子最愛的是奶奶，為啥父母比不上奶奶

文|福林媽咪

最近看了一個笑話，但是卻十分的戳人心：

爸媽爭寵：

媽媽：「我懷胎十個月，憑什麼寶寶先會叫「爸爸」，只是發出個「ba」的音而已，不生氣。忍不住不生氣，憑什麼！！」

爸爸：「憑什麼我一抱寶寶就哭，回來看見我就是「我媽呢」，我這個當爸的就這麼討人嫌？」

媽媽：「憑什麼我陪著孩子，教孩子規矩，對孩子嚴厲，壞人都讓我做了，孩子卻更信任爸爸，更願意跟爸爸親近！！」

結果，有一次老師布置作文寫《我和我最愛的人》，孩子居然寫的是《我和我的奶奶》!

沒想到，爸爸媽媽爭寵半天，在孩子眼裡，奶奶才是最愛的人。

為啥爸爸媽媽在這兒爭寵半天，結果孩子更喜歡的卻是奶奶呢？

奶奶帶大的娃

很多寶寶都是爺爺奶奶帶大的，等要上學了，才會接到父母身邊來樣。

孩子從小到大都是奶奶管吃管住管安慰的，自然就跟奶奶更親近了，對父母反而比較生分。

就算是後來跟父母關係緩和了，其實也沒有跟奶奶那麼深的感情。

奶奶比父母更有耐心

還有就是老人們在養娃的時候比父母更有耐心。

孩子做錯了什麼事兒，奶奶能調侃幾句或者慢慢說，畢竟年紀大了，也快不起來。

就這份跟孩子溝通的耐心，很多父母就沒有。

這會讓孩子覺得，奶奶跟自己是一夥兒的，父母總是急脾氣、著急的站在孩子的對立面，來指責孩子。

奶奶比父母更寬容

奶奶比父母寬容多了。

現在流行說「隔輩親」，確實也是，父母小時候不寫作業試試！哭鬧要玩具試試！不聽話、不睡覺、非要看電視試試！

但是奶奶對孫子、孫女就寬容多了：不寫作業？沒事兒，這麼多作業，老師也不考慮孩子的睡眠；想看電視？好咧，奶奶給你開開《小豬佩奇》；想吃糖？你嘗嘗這個好吃不，好吃下次奶奶還給你買！

父母對孩子自然就要比奶奶嚴厲多了，這個不行，那個不許，孩子就單純的覺得順著自己想法的奶奶跟自己更親近了。

奶奶對孩子好也是一件好事，有利於家庭和諧。

比那些總是把精力放在寶爸身上，連自己孫子孫女都覺得是外人生的，更別提對「外人寶媽」的態度了。

可能就是因為這個，很多寶媽都說：「羨慕那些有奶奶疼愛的孩子，我家寶寶出生，她奶奶就在滿月的時候來瞅了一眼」。

不過呢，就算孩子有老人寵著、愛著，不管在家庭中孩子更喜歡誰，誰更疼愛孩子，都要做好這幾點：

家庭要統一教育觀念

家庭統一教育觀念非常重要。

從一件小事說起，比如說吃飯。

寶媽想要培養孩子自己吃飯，寶爸覺得孩子太小自己沒辦法做到，奶奶又覺得孩子就要喂飯才能吃得飽。

如果家裡面沒有統一過觀念，那麼孩子會非常迷茫：自己到底要怎麼做。

寶媽、寶爸、奶奶三面又會爭吵不休，引起大大小小的爭吵，吵過完事兒也就算了，就怕遺失未平一事又起，這個家庭很容易不穩定。

家庭成員要統一教育底線

還有就是要統一教育底線。

再來說一件小事，就是孩子寫作業的事情。

讓孩子寫作業、輔導孩子寫作業時一件大事——能把父母們氣出心臟病的大事。

家裡的人自然是希望孩子能好好學習，天天向上。

但是如果孩子不寫作業，首先媽媽會開始嘮叨，爸爸眼瞅著就要上手了，這時候奶奶過來把爸爸媽媽罵一頓，然後跟孩子說：「別搭理你爹媽，咱不寫作業了」。

這不僅會讓孩子學會「欺軟怕硬」，還會讓孩子養成偷懶逃避的心理，這對孩子未來成長可沒有好處。

所以家庭中，對孩子教育的底線要守好：孩子不能做的事情堅決不能做，就算是哭也不行。

還有父母在教育孩子的時候，老人千萬別不清楚前因後果的就上去幫孫子「逃離五指山」，當孩子處於完全的自由時，就容易成為「放縱」。

家裡的爸爸媽媽、爺爺奶奶都希望孩子變的更好，在未來更加優秀。

所以啊，不管在家裡誰跟孩子更親近、孩子跟誰更好，都沒有關係，平時爭爭寵什麼的，也能調節一下家庭氛圍。

但是在面對孩子的原則性問題時，家庭成員最好保持一致的態度。
