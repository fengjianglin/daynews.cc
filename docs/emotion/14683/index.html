人到中年，請逼自己離開這三種群，說的真好

2019-12-16 1,530條評論 2,906次閱讀 698人點贊

在這個社會上，中年人擔負著重要的角色和使命，即便是混的不是太好的，在單位里也算是稍微有一點影響力，畢竟中年人還是有一些積累的，所以大家記住中年人，要想讓自己的工作事業與生活達到平衡的狀態，要想讓自己的事業更上一層樓，必須從那些對自己沒有多少實際意義的社群當中脫離出來了，給自己騰出來更多的時間和精力，才能夠保證自己生活與事業的平衡性，否則的話，很容易陷入到無法兩邊都顧的悲觀情況。

【1】頭銜與務虛群。

稍微在社會上混出個名堂來的中年人，他們有一定的影響力，有一定的社會地位，在單位裡面也算是中間力量，是這個組織的骨幹，所以社會往往會給他們很多虛擬的職務，這些職務不能夠給他們帶來實質性的利益，也不會給他們產生實質性的影響，為了保證自己充沛的時間和精力，建議各位中年朋友，從這些社群裡面從這些虛擬的植物裡面解脫出來，不要要所謂的名聲，因為這些名聲對你來說越多實際上負面影響越大，從很多中年人的名片上就能看出來，這些虛擬的植物對自己有多大的影響，有的一個人上面印了不低於5個以上的名頭，實際上大家都知道名頭越多的人越不務實，所以大家工作和生活不要受這些影響，才是明智的選擇。

【2】兼職與操心群。

在一般的單位裡面，會身兼數職，其實這些兼職自己壓根就沒有時間去過問，但是操心操不到位的時候，責任還是要有自己來承擔的，單位里的領導還沒起名曰，叫做願意付出的人 實際上，中年人的經驗比較豐富，過去經歷過失敗和成功，最具備開拓精神，最具備完成特殊使命的能力，所以要多去做創新的事情，不要把精力耗在一些沒有實質性意義的兼職上 所以建議各位中年朋友從這些群裡面退出來，讓自己專註於自己領域的事情，專註於特殊的使命，專註於自己既定的目標，而不是分散自己的時間和精力，你的時間精力越分散，最後你的成就就會越小，畢竟專一專註才是成就大業的前提條件和根本。

【3】活動與組織群。

中年人面臨一個比較嚴峻的問題，就是各種各樣的活動都會邀請你參加，有些時候你不參加好像是不給別人面子一樣，參加了這些活動之後，你就成為這個特殊組織里的一員，很多的組織者會把中年人列為這個組織小團隊的管理者，比如說，組建一個班級，讓中年人做班長，自己必須要擔負起班長的職責，實際上這對很多中年人來講是一個非常大的考驗，大家積極的參與，本身已經是很重視這些事情了，如果後續再消耗自己的時間和精力，這對中年人來講是非常不划算的事，建議也要從這樣的組織裡面推出來，要不然的話，你就成了別人利用的工具和犧牲品，關鍵是被別人利用之後，不會給你帶來任何的價值，只會消耗你的精力。
