艾小羊：少管閑事，更容易活出氣質

2019-12-19 142條評論 1,402次閱讀 488人點贊

讀者問我，在國外撞見閨密老公跟別的女孩約會，不知道該告訴她還是不該告訴她。閨蜜如今已經懷孕34周，跟她老公關係很好，她也試探過閨蜜，閨蜜表示自己跟老公情比金堅。

如果說了，她的擔心有幾個方面，

第一，如果閨蜜老公死不承認，閨蜜會信她，還是會信老公。

第二，如果閨蜜根本沒有離婚的能力，知道真相，是不是比稀里糊塗被騙更殘酷？

第三，如果閨蜜有離婚能力，肚子里的孩子到底生還是不生，閨蜜萬一選擇引產，她是不是相當於成了間接的兇手。

她問我怎麼辦。

在讀者眼裡，我是萬能的，但我知道自己幾斤幾兩，對於直接告訴讀者怎麼辦這件事，很多時候老中醫也無能為力。

因為生活不是考試，生活里的很多事情是沒有標準答案的，也沒有所謂正確的選擇，只有你認為正確的選擇，而在每個人心裡，對於正確的理解其實千差萬別。

我只能先說說我的選擇。

我會選擇打死也不說，權當看走了眼，做了個夢。

這個消息，如果從我這兒說出去，會改變兩個人，甚至兩家人、包括他們後代的生活，甚至人生軌跡，這麼大的責任，我負不起，也不應該由我負。

說出去的話，潑出去的水，既然我不能保證自己好心辦好事，最好的選擇就是閉嘴，避免好心辦壞事。

所謂打死也不說，就是永遠不說，即使閨蜜有一天自己發現，也假裝第一次聽說。「我早知道」這種話，相當於給閨蜜又插了一刀，把自己也帶進了不仁不義的坑裡。

但如果閨蜜生完孩子準備全職，我會拼上老命去反對。

如果閨蜜把工資交給老公，我會強烈建議她做家庭的財務主管，順便存點私房錢。

我會支持閨蜜永不停步地學習，永不停步地成長，永不停步地愛自己。

我能做的是，盡自己最大可能，幫她成為一個獨立、自信、優秀的女性。

如果有一天，她自己揭開了謎底，希望她有能力承擔自己的任何選擇。

以上，是我所理解的一個對雙方有利的選擇，不接受任何反駁。

最開始已經說了，每個人對於正確有不同的理解，你走你的路，我過我的橋，大家相親相愛，不需要彼此說服。

其實我今天特別想說的並不是閨蜜老公出軌的問題，而是選擇的邊界問題。

我們每天都面對很多的選擇，有些相對容易，有些很難，但很多人，不管是容易的，還是難的，都不會選。

一個選擇，應該遵循什麼樣的標準？我覺得首先是不傷害自己，其次才是對他人有利。

把不傷害自己放在第一位，可以避免很多糊塗的選擇。因為你對自己的了解，一定勝過對別人的了解。

很多時候，你所認為的不傷害別人的選擇，有可能對別人造成致命的傷害。但不傷害自己，這個評判標準，容易掌握，也容易正確。

所以，做任何一個選擇，先問問會不會把自己給套進去。

如果它有極大的可能讓你處於尷尬、為難的境地，你最好不選，除非你選了，能拯救地球。

好心辦壞事，基本上就是不太考慮自己，只考慮別人，拯救情結過剩，刷存在感的慾望太強，結果費力不討好，惹了一身騷。

都說女人愛管閑事。其實我們也可以活得淡散一點、孤清一點、自我一點，把熊熊燃燒的八卦慾望放在不相干的名人、明星、網路事件上，對於身邊人的私事兒，關心的越少越好，畢竟你自己的私事兒都忙不完呢。

你活得快樂、健康、不多事兒，自然會吸引想跟你在一起的人。

淡散孤清的人，未必比心直口快的人朋友少，但一定比心直口快的人，麻煩少。

不管閑事的女人，會氣質更出眾，舉止更優雅，皮膚更年輕，心理更輕盈，不信你試試。如果不靈，來找我，姐就是這麼霸氣。

今天的「在看」很甜，也必須試試〜
