人生路上最真實的現狀：你其實一個朋友都沒有

2019-12-16 300條評論 3,430次閱讀 1127人點贊

一個人在年輕的時候，常常認為只要朋友多，自己的路就好走，因此在生活中就注重多交朋友，也真心希望朋友間彼此相伴，心靈相通，在這個紛繁複雜的世界，走好自己的人生路。

因此年輕的時候，身邊總是圍繞著很多朋友，互相之間來往密切，喝酒吃飯很是熱鬧。

但是，作為過來人的有經驗者都說朋友太多，那都是假的，人這一輩子有一個真心朋友就好。

慢慢長大後，過著過著就會發現我們這一路走來，認識了很多人，也結交了不少人，可是能夠真正留在自己身邊的，的確沒有幾個，許多交往了很久的朋友，也一個個逐漸的散去，因此我們的生活也就變得越來越安靜了，常常是一個人在家裡享受孤獨和寂寞。

一個人只有在低谷的時候才能真正看清自己身邊究竟有幾個真心的朋友。

人的一生，朋友會有很多，可是跟你交心的朋友又有幾個呢？

在這個物慾橫流，薄情寡義的世界上，大家都渴望著真正的友情，需要的是能夠交心，能夠說心裡話，在最困難的時候能夠來到你身邊，給你幫助，有快樂也可以分享的朋友。

可是現實生活中常常是在你有困難的時候，你那些所謂的真心朋友卻不會伸出手來拉你一把，曾經那些交談闊論無話不說的朋友，在你遇到困難的時候，他躲得遠遠的，對你一點不也不理會。

在你有了快樂，要和人分享的時候也沒有朋友會真心替你高興，有的甚至是嫉妒和憤恨。

在過往的人生道路上，大家都有可能有很多的朋友，也不管是男是女都聊的開開心心，也很坦誠，沒有隱私，也沒有私心，過著過著，後來就會發現，實際上大家都是帶著面具的人，幾乎所有的關係網都滲透著一個”錢”字，以前的誓言也只能變成了美好的回憶。

古人說的好，”窮在街頭無人問，富在深山有遠親”、”雪中送炭君子少，錦上添花小人多”。

其實人生只有在不得意的時候，才可能真正看清誰是真正的朋友。因此，當一個人處在人生低谷時，還是不可對人生失去信心，因為只有在這個時候你才可能認清誰是你真正值得去交往一生的人，也往往在人生低谷時才可能交到知心朋友。

一個人，越長大越孤單。

人心難測，沒有永遠不變的心，經歷了事，才會認識人。有的人走著走著就離開了自己的視線，成了陌路人，有些人走著走著就很難說再見了，成了知心朋友。

慢慢人生路上，有朋友，也有所謂的兄弟，事實上，他們不一定是朋友，也不一定是兄弟，只不過是自己一個認識的人而已。

對於一個成年人來說，真的是沒有一個朋友，這可能是我們都經歷過的事情。

常年說的好，朋友不在多，有一個就好，可是人有時候卻連一個真心朋友都沒有。

我們大家的通訊錄都有很多的好友，但其實能找到幾個能說上心裡話的人呢？又有幾個是能真正幫助自己的人呢？在這世間，真心最是難求，在我們的朋友圈裡，朋友增增減減，無需去在乎誰走了，你只要去珍惜真正留下的人就好。

人的一生，每個人都要去面對孤獨和寂寞。

魯迅先生認為，一個人一生真正是沒有一個知心朋友的，也許有人會問，沒有知心朋友的人，難道不會覺得孤單嗎？

事實上，生活中只需要保持平常心，朋友是可遇不可求的，有了緣分就會在一起，沒有緣分就隨意而安吧！

在自己有限的生命里，不要去羨慕別人的朋友多，也不要因為孤獨，而去結交那些沒有真心實意的酒肉朋友，終究是人生苦短，不管自己的人生怎樣，在以後的人生路上，不去結交虛情假意的人，也不去參加沒有真正價值的飯局。

生命原本就是孤獨的，不管你身邊有多少人陪伴，總是有一些人，一些路要一個人走，許多黑夜也要一個人去度過。

有時間去感嘆孤獨難熬，你還不如去學會享受孤獨，用孤獨來凈化自己，提升自我，去做一些自己覺得開心的事，人生才會美好

對於人的一生到底有沒有知心朋友，有多少朋友等類似的話題，你有什麼想法泥?歡迎留言討論。
