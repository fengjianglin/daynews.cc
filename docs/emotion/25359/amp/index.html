不愛發朋友圈的女人，一般都是這4個原因，真的很准

天天要聞
2019-12-21

有時候你點開一個女人的朋友圈，發現她的朋友圈空蕩蕩的，根本就沒有生活記錄，微信對她來說只是聊天工具，朋友圈對她來說可有可無，她從來不會在朋友圈上發很多自己的動態。有些女人特別不愛發朋友圈，不愛發朋友圈的一般是這4類人，錯不了。

(1)不喜歡記錄生活，情緒特別少的女人。

朱朱是那種情緒特別少的女孩子。她平常很安靜，她也從來不發朋友圈，不喜歡記錄生活，她覺得生活是用來感受的，沒必要特地記錄在朋友圈。

她很討厭有些人根本都不用心去感受生活，只光顧著擺拍發朋友圈，她覺得那樣子是辜負生活的一種方式，所以她從來不為了發朋友圈而發朋友圈。

(2)性格沉默內斂，一點都不咋咋呼呼。

有些女人特別咋咋呼呼，什麼事情都往朋友圈說，完全不在乎外人看到朋友圈會怎麼想。其實這樣頻繁發朋友圈的行為會讓別人覺得你太幼稚。

做人應該沉默內斂一點比較好，尤其在朋友圈應該沉默一點比較好，如果你在朋友圈過於咋咋呼呼，開心的事要往朋友圈說，不開心的事也要往朋友圈說。那樣在朋友圈過於外露自己的情緒，其實並不太好，有些人甚至不高興時還會在朋友圈罵一堆特別難聽的話，這樣其實是整體降低你自己的個人修養。

(3)不願意成為朋友圈裡的假人。

陳知麗說她不喜歡發朋友圈，因為她覺得很多女生朋友圈裡面的動態很假，很多女生喜歡在朋友圈裡製造自己是白富美的假象，或者製造自己是文藝女青年的假象，其實她們的生活根本就沒有那麼美好，全部都是為了給別人營造一種自己特別上進，特別美好的現象才這樣的。有些女生生活中非常蓬頭垢面，邋裡邋遢。但朋友圈裡的她非常精緻。有些女生生活中臉上有很多痘痘和雀斑。但朋友圈裡的她就常常美顏過度，看起來就像個實打實的假人。

(4)不發朋友圈純粹就是因為懶。

愛莉從來不發朋友圈，無論是高興的事，不高興的事，難過傷心的事，還是其他什麼事情，她都不發朋友圈，她不發的原因很簡單，她很懶，她懶得發，所以幾乎從來都不發。光在在日常好好活著就已經花光力氣了，哪裡有那個閑工夫整天在朋友圈裡浪費時間。

結束語：不喜歡發朋友圈的，一般都是性格比較內斂或者為人比較懶散，情緒特別少，一點都不咋咋呼呼的女人。不喜歡發朋友圈挺好的，沒必要在一個虛擬的空間浪費太多的情緒，過分情緒化的成年人一般都不會有什麼大作為，不發朋友圈至少說明懂得剋制情緒，真正的生活在生活本身，在日常里，永遠不在朋友圈。
