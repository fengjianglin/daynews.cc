誰說除卻巫山不是雲，終有弱水替滄海

除卻巫山不是雲，終有弱水替滄海。愛情這個話題永遠是最有爭議性的話題。自古至今，由內到外，愛情在世人的描述中是那麼神聖那麼純潔那麼偉大，不求天長地久，但求朝朝暮暮你都陪我度過。一份完美的愛情成了大多數人的無上目標。但人生那麼長愛情真的能夠抗衡一切嗎？

很多雞湯中都會寫道上了大學一定要談一場轟轟烈烈的戀愛，彷彿一段驚天動地、痛徹心扉的戀愛就是你的情感證書，是你不再是小孩子的通行證，證明你終於從小孩子成長為一個大人了。

愛情更需要的是陪伴，高中時代看過很多言情小說，那時候最羨慕有人能拿著一大束的玫瑰花等我下課。幻想過無數浪漫的場景，上了大學之後依舊也沒有實現。我改變了策略，既然沒人給我送，我就自己化身成白衣少年，給自己的朋友送去滿天星。這種事情用她的話說，純粹是自己感動了自己。在別人和她的舍友看來都是浪漫到爆炸，可她自己卻覺得麻煩到爆炸，花沒有地方放，很多人借去拍照，煩惱比驚喜的感覺要更多。我自己事後回憶起來也是很平淡，倒不如晚上我去和她一塊喝奶茶這種小事來的更加深刻。

過於濃烈的開場往往是慘敗的下場，白娘子與許仙斷橋相遇，一場大雨落下一把雨傘成了紅媒，一借二還，千古佳話就此而成；劉徹四歲時見了阿嬌，便許下「若得阿嬌，當以金屋貯之」的誓言；王寶釧與薛平貴一見鍾情，衝破了門第觀念下嫁給薛仁貴。諸如此類數不勝數，開端留給我們杜撰的空間與甜蜜的幻想，卻獨獨結局總是不得善終。

過與濃烈的開場不過是為了讓這出愛情早早收場，故事的最後白娘子永鎮雷峰塔底；陳阿嬌的《長門賦》也不能挽回劉徹的心；王寶釧寒窯苦等薛平貴十八年只等來了西涼公主。

愛情越是浪漫，結局反而越是鮮紅，錯過反而是上天最好的安排。初見你春光明媚，一顆心全被壓抑已久感情蒙蔽了雙眼，你以為這個人萬中無一，只可惜妾心似水千萬種柔情終是敵不過郎心似鐵。

百般柔情似水流，平平淡淡你以為是食之無味，只是生活本就是平淡，愛情始於假象死於幻滅那一刻。

好似《桃花扇》中李香君，一頭撞上，用筆和著血寫下，不知是後悔自己沒能早早參透情深不壽，還是在嘆息自己這般命運。

​在最後的結局中二人雙雙修道而去，沒有歷經千帆磨難後的在修就好，二人終究是看透了，這世間紅塵種種不過是心中自己的魔障。

可恨月是天上月，清風拂面過轟轟烈烈的背後是無休止的煩惱，時間的滾輪一點點的向前行駛。

痴情人皆是可憐人，落得個問世間情為何物，直教人生死相許。夜半時獨自抽泣，一遍遍問自己的心，自己這番究竟是值不值得，那一瞬間智商丟到一邊，只剩下對明天的幻想。
