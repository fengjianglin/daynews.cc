我準備結婚，去給老闆送大米，飯桌上沒提借錢的事，老闆：有事吧

2019-12-23 1,524條評論 3,368次閱讀 1057人點贊

我準備結婚, 去給老闆送大米, 飯桌上沒提借錢的事, 老闆: 有事吧

我是農村的，老家還有十幾畝地，我上班的城市離老家有三個小時的車程，農忙的季節忙不開，我也會去請假回家幫忙。

老闆體諒我，也不扣我的工資。

老闆對我照顧，我嘴笨，感謝的話卻不知道怎麼說。我只能以實實在在的工作來回報。

我是做技術的。我可以說我是技術部最忙的一個。

客戶會「挑剔」來實施的技術人員，公司派去的人員沒幾天，客戶會說，偷懶了，拖進程了，要吃喝了，非要「退」回去。客戶點名要我去。

我做的工作比較「細」，公司發有「項目進程表」，別的同事是實施完，草草應付下填填條目就行了。我不是，我會認真的核實項目，一項一項的比對。所以，我經手的項目沒有出現紕漏的。

客戶沒少在老闆跟前表揚過我。

老闆雖然沒有特別地誇獎過我，卻把我的工資漲了上去，見我連著出差了半個月，衣服都髒了，他把我領到商場直接買了一身衣服。

老闆邀請員工去家裡吃飯，我是第一個。

老闆拿出珍藏的酒，讓我「放開」的喝。

喝了點酒，我的心裡話就多了。跟老闆說，父母辛苦種地把我養大，上學，很不容易。又跟老闆說，我得好好工作，掙錢，讓父母享福……

我說著，就哭了。我還記得對老闆說，「您是個好老闆，我家裡有自己產的米，今年不說了，怕您『嫌棄』，明年我給您送一袋新鮮的。」

是老闆把我送回去的，再見他有點不好意思。老闆倒是隔一段時間，喊我去他家裡吃飯。

工作的第二年秋天，我準備和女朋友到年底結婚。

女朋友家提了個條件，結過婚就想買房。

我一聽，心就「提」了下。這首付、裝修不少花錢。

九月底又到了農忙，割水稻的日子。

忙過，我惦記給老闆說過送新鮮大米的事。把水稻碾成大米，裝成袋讓家人幫忙，跟我送到了客車上。

坐在車上就給老闆發微信，「給您送大米」。老闆就說，那晚上在家吃飯吧。

看著車窗外路邊飛馳過的樹木，我的心也來來回回的拿不定主意。

我算了算，結婚買房的事，怎麼著也得再借4萬元，我有心向老闆張嘴去借，可我又擔心老闆給我掉地上。

我想來想去，決定還是去跟老闆說。

我這幾天在家裡忙農活，人也晒黑了，頭髮亂蓬蓬的，老闆見我抱著一袋大米站在門口，就笑了。

老闆準備了幾個菜，喊我坐下來吃。

邊吃著，邊聊著，問我結婚的事怎麼樣了？

我說到年前了。我幾次到嘴邊說借錢的事就沒說出來。

決定好的事又打了退堂鼓，我一想，乾脆不說了。到事情跟前再商量，走一步算一步吧。

吃過飯，老闆見我心事重重的樣子，問：「你有事吧？」

我一時結結巴巴的，說，「也，也沒什麼事。」

老闆一笑，你說吧。

我這才說了結婚想買房的事，我對他說：「您看能不能借我4萬元錢？」

老闆說那你等一下。他去拿了一張卡過來。

老闆把卡遞給我說：「4萬夠嗎？這張卡里有5萬元錢，你拿著先用吧。」

我拿著卡，眼圈紅了。
