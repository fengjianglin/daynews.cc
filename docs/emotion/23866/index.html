家庭教育著重於良好的氛圍，家長們掌握這些技巧，輕鬆營造

2019-12-18 1,130條評論 3,810次閱讀 998人點贊

文丨艾媽育兒日記（文章原創 ，版權歸本人所有，歡迎媽媽們轉發分享）

家庭氛圍對於孩子的成長，孩子性格的養成來說都是至關重要的，一般來說，在民主型家庭環境下所成長的孩子，他們的性格會更加的活潑，也會更加的自信，但是如果是在權威型壓抑的家庭氛圍之下所長大的孩子，那麼他們性格會更加的孤僻或者是自私。

每個孩子從出生你來都會有屬於自己獨特的風格，而其中性格氣質這一方面大多數都是後天所培養的，而性格氣質的培養與家庭氛圍是有著密切關係的。如果家庭的氛圍是比較的輕鬆和諧融洽的話，那麼也可以給孩子帶來一個更加輕鬆愉悅的感覺，他們和父母之間的親子關係也會更加的融洽。

案例：

朋友平常忙於工作並沒有過多的時間把心思放在孩子的身上，而朋友的丈夫也是如此，所以關於孩子的教育問題，兩個人之間一直都有著很深的矛盾。再加上兩個人相處的時間不多，所以經常會出現各種各樣的問題和爭吵，是非常典型的，話不投機半句多。

分析：

在這樣的環境下所長大的朋友的孩子性格一直都非常的孤僻。在學校里也不願意和別的人過多的說話，回到家裡也不願意和父母說話，都是自己一個人躲在房間，很多自己的情緒都是自己消化的。而孩子之所以會有這樣的一個性格，和他所身處的這樣一個有矛盾的家庭氛圍是有著直接聯繫的。

對於孩子的成長性格塑造來說，家庭氛圍是至關重要的，那麼家庭氛圍良好的話到底有著怎樣具體的重要性呢？

1.良好的家庭氛圍是良好心理素質形成的關鍵

孩子來到這個世界上最經常接觸的人群就是父母，所以說父母是孩子第一任老師，是孩子每天行為學習的榜樣。所以家庭的教育氛圍以及家長的教育態度都會直接的影響孩子的行為以及性格心理的塑造。

一般來說，在家庭環境當中，如果父母對於孩子過分的驕縱有求必應的話，這樣子給孩子包辦代替的教育方式會讓孩子對父母產生依賴性，那麼孩子就會出現任性自私這樣子的性格特徵。

但是如果用平等民主和諧的氛圍去教育孩子的話，這類型的父母能夠給孩子更多的引導關心和愛護，那麼孩子也就能夠有著坦誠自立，大方熱情關心他人的能力。

2.良好的家庭氛圍是良好心理素質形成的前提

所謂的家庭情緒氛圍它其實指的是家庭當中的一種態度和感受。它主要是通過人際關係的處理以及語言交流的方式來構成的，這樣的一種家庭氛圍，他是會直接地去影響著家庭成員當中每一個人的心理，每一個人的狀態。

家庭成員之間如果是相處得很融洽，即便是有一些矛盾都會當面直接的去解決的話，那麼在這樣的氛圍之下所成長的孩子，他們就會更加懂得如何去互幫互助，如何去諒解別人，如何去站在別人的角度去考慮問題。但是如果相反的話，他們就會爭吵不休，處事自私。

這種環境下所成長的孩子，他們的心理一般都是不健全的，所以說家庭氛圍是孩子培養良好心理素質的基本前提。

3.家庭氛圍不好，會成為影響良好心理素質的嚴重障礙

隨著現在獨生子女不斷的普遍，許多的家庭都是只有一個孩子的情況。再加上人的價值觀念，人的三觀不斷地發生改變，所以單親家庭也就變得越來越多。

這樣的一種家庭結構的變化會讓很多的孩子從一個溫暖的家庭當中突然間脫離就會讓他的心理受到傷害。不僅會讓孩子變得沒有安全感，也很容易使他們產生一些變態心理，走上犯罪的道路。

既然我們說良好的家庭氛圍對於孩子的成長來說是至關重要的，那麼又應該如何去營造好的家庭氛圍呢？

方法1：多花時間陪伴孩子

很多的家長在教育孩子這一個角度上面，總是覺得要給孩子足夠的物質支撐，讓孩子有一個好的物質生活條件，這樣就是對孩子好，這樣就是愛孩子的表現。

但是其實在很多孩子的角度上看來他們並不是這樣認為的，比起那些昂貴的衣服玩具，他們會更加希望父母能夠有更多的時間去陪伴自己。

所以良好的家庭氛圍的營造就必須要求家長花更多的時間去陪伴孩子成長，多花時間就有孩子進行溝通，拉近和孩子之間的距離。

方法2：可以寵孩子，但是不可以溺愛孩子

每一個孩子都是上天送給每一對父母的天使，所以父母想要寵愛自己的孩子是都是可以理解。但是很多的家長在表現對孩子愛的時候都會出現一個誤區，他們覺得對於孩子的要求有求必應，在生活上儘可能去滿足。

儘可能地去給孩子更好的物質享受就是對孩子的愛，但是其實真正的愛孩子並不是這種表面上的理解。

放手讓孩子獨立的成長，這反而是一種更好的愛孩子的表現，因為在這個過程當中孩子能夠從實踐當中收穫經驗，從實踐當中擁有更多獨立自主解決問題的能力，這對於他們未來的成長未來的生活來說都是很有幫助的。

方法3：家長要學會控制自我的情緒

想必很多的家長都會出現這樣的一個問題，他們雖然年紀比較大，但是對於自我情緒的控制能力還是比較薄弱的，尤其是他們在工作當中受到了委屈之後，回到家就很容易擺臉色，會把這個氣直接的發泄在孩子的身上。

或者是當孩子犯錯誤的時候，家長都會直接地去批評和指責吼孩子這樣的一種方式，對於孩子的成長來說都是會產生很大的心理陰影的，所以良好家庭氛圍的營造就必須要求家長學會控制自己的情緒。

遇到問題的時候先內化，而不是把他直接的轉移到孩子身上，不要讓孩子在這個過程當中學會了你的壞情緒。

孩子的成長過程當中有許多的因素都是會產生影響的，而且這種家庭氛圍是大家往往會忽略的一個重要因素。

​我們時常在羨慕別人家的孩子，為什麼能夠那麼活潑可愛懂事，落落大方，但是其實和他們所成長的家庭環境也是有著一定關係的。那麼對於這種家庭氛圍對於孩子的影響你是怎麼看的呢？歡迎留下你的看法。
