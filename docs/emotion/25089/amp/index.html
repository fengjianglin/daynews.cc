做一個比昨天更好的人，是給自己最好的十句話

天天要聞
2019-12-22

01、做一個更好的人，確信在遇見一個新的人之前自己是誰，也希望那個人知道你是誰。

02、無論你是誰，無論在經歷什麼，堅持住，你定會看到堅強的自己。有時候，沒有選擇，只有堅持，才是唯一；有些東西，只有一次，放棄了，就不會再有；有些事，其實只要堅持些，再堅持些，才最精彩。每個人，其實都有另一個自己，如影隨形，它是自己的支撐、靈魂和精神，它就是堅強的自己。

03、生活中總會有傷害你的人，所以你仍然需要繼續相信別人，只是小心些而已。

04、做一個比昨天更好的人，是給自己最好的禮物。

05、當你對對別人的不合理的期待、請求和要求說不感到輕鬆時，你就會發現你有了更多的同情心。當你的確對他人說是的時候，你會對你自己和那個人感覺更好。

06、世界這麼大，有人在奔跑，有人怕摔倒，路上坎坷一樣，不一樣的是，奔跑的人心裡只有目標，怕摔倒的人總是盯著路上的荊棘。最終的結果，奔跑的人到達成功終點，怕摔倒的人，糾結一生，枉費才華，遺憾終生！磨難就能獲得成功堅持，奔跑，相信，結果就是成功！

07、有一些事情你不希望它們發生，但你一定要接受事情的發生；有一些事情你不希望知道，但你一定要去了解；有的人你不希望失去，但你一定要放下他。走進你生活的一些事和一些人是為了磨練你，這樣你不需要他們也可以繼續你的生活。

08、世界很大、風景很美、機會很多、人生很短，不要蜷縮在一小塊陰影里。因為世上找不出兩個完全相同的人。生得再平凡，也是限量版。只要內心不亂，外界就很難改變你什麼。不要艷羨他人，不要輸掉自己。

09、你尋找什麼，什麼就會在現實中出現。你的想法和感受將你帶進今天的生活；它們也將會將你帶入明天的生活。如果你真的想要改變你的生活，你一定要首先改變你的想法。當你發生改變，你身邊的世界也會隨之改變。

10、爬山時若在氣喘吁吁時選擇下山，就永遠沒有機會看到山頂的風景；奮鬥的年紀遇到挫折就選擇放棄，就永遠不知道自己有多大的潛力。
