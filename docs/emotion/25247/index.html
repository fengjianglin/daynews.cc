當別人瞧不起你時，不要把別人事情當作一回事

2019-12-21 119條評論 1,298次閱讀 64人點贊

一個人過於在乎別人的看法，往往是自己活得很累。人這一輩子不是活給別人看得，最重要是活出自己。

當別人看不起你的時候，我們最重要是學會忽略別人的眼光。我們需要是把重心回歸到自己身上，我們無需高估自己，但是也沒有必要貶低自己。

不在乎別人的眼光，你才能坦蕩過好自己的生活。因為你的人生是活出屬於自己想要的一生，面對別人冷眼的目光，你最重要是保持淡定平和的心態。

你要相信那些嘲笑別人的人終究會被生活所嘲弄，面對他們不成熟的舉動，我們不用過於計較，生活自然會給她們最真實生動的一課。

而你最重要是依然相信自己，學會尊重自己。請記住，不要因為別人看不起你，你反而去討好他，這隻會自輕自賤，你最需要是沉澱下來， 讓自己擁有平和的好心情，更重要是平和下來，你才會體會到生命曠遠的味道。

面對別人冷眼的目光， 我們還要相信，你是自己生活的主人。這個世界是多元的世界，每個人因為自己的見識亦或視野才會形成自己的陋見，面對別人不成熟的舉動，我們最重要是學會過好自己的生活。

人人都有自己的節奏，人人都有自己成長的規律，有些人看不起我們，是不懂得我們成長規律與節奏，有些人發跡早，有些人成熟晚，我們無需過度羨慕別人。

過好自己的生活，是讓我們珍惜屬於自己的生命，不斷豐富自我的人生之旅，盡量增加生命的豐富之旅，才會讓我們的人生過得幸福自然。

專註過好自己的生活，意味著找到屬於自己人生節奏，而不是盲目去攀比別人，更不是被別人瞧不起就妄自菲薄。

無論我們做得如何，我們都需要明白，人生的路上，最重要是過好自己的生活，當你過得足夠好的時候，那些瞧不起你的人才會知道自己看走了眼。

被別人瞧不起的時候，我們還需要練就自己內心強大的方法，或許我們需要知道，別人看不起我們，那是別人的事情，不把別人事情當作一回事說明我們是智者。

人生的路上，持續精進自己才是我們人生中最重要的任務，當你精進自己的時候，你才會看到自己的成長。

而你也會知道比起他人的眼光，自己的成長才最重要。我們要相信，有時慢點沒有關係，最重要的是讓自己擁有足夠的定力看到自己未來的方向。

人生中，每個人的起點低點都沒有關係，最重要的是我們要有自己的方向，當方向對了，精進自己就是學會和時間做朋友， 慢慢日益變得愈發的厲害。

如何找到自己的方向，最重要是學會從生命中找到讓自己感動的事情，讓自己想要利他的事情。誠品書店吳清友曾說，他從前生病的時候，才慢慢找到自己最想做的事情，那就是一輩子與書為友，照亮別人的生命。

我們的方向最好是能夠利他，你服務的人越多，你的成長就越給力，生命中最值得驕傲不是活給別人看，而是讓自己是光，讓自己的人生能夠點亮自己和別人的生命。

當別人瞧不起你的時候，我們依然要學會忠於自己。忠於自己是讓自己在人生的路上不迷失自我的關鍵。

忠於自己是忠於自己內心的聲音，是選擇自己想要做的事情。每個人都只有一生，我們要學會忠於自己，你才會發出不同的聲音，更能喚醒你內在的生命。

每個人或許要知道的是，人的一生最關鍵是活出自己想要的人生，而忠於自己就是對自己負責。許多人之所以很累，她是對別人負責，而不是對自己負責。

有些人為了父母的願望活著，有些人是為了別人的期待活著，這些都會讓我們精疲力盡，只有為你自己活著的時候，你才會理解到忠於自己，你的人生才會問心無愧，而你臉上閃耀自信的光彩是所有人內心最美麗的畫面。
