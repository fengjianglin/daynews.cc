在社會上，人一定要對自己有信心，才會有希望

2019-12-19 742條評論 1,962次閱讀 339人點贊

導語：社會是什麼，是人與人之間的關係，是人與人之間的比較，是人與人之間的競爭，是人與人之間的價值交換。如果一個人對自己沒有信心，那麼不管你有多少「上游優勢」，都不會勇敢的衝浪在世界的「波濤」中。如果你對自己信心十足，那麼你就可以爆發出驚人的能量，做出「奇蹟」般的事情。

01，信心是自我意識的強烈認可。

一個人沒有自我意識是根本不知道自己的存在的。自我意識太弱也會讓自己的自我存在感很低，同時也會讓自己對自己的認可度大大降低，其結果就是渾渾噩噩，喜歡依賴，膽小怕事，懦弱無能。這樣的人幹什麼都沒有希望。

當信心加註到人的認識之中的時候，人的自我意識就會大大加強，也因此自己對自己開始負責甚至非常負責。只有一個人開始為自己負責的時候，你的人生才逐漸有意義。自我意識強烈會讓人重視你的存在，你的自我認可會讓人得以信賴。

02，信心會增大精神能量。

信心源自適應環境的本能，也是征服外界的姿態。信心會增加你前行的勇氣，會安定紮實那躁動的情緒。信心可以讓你儲備更多的能量，包括行動的力量和精神的能量。然而人只要先擁有了強大的精神能量，才會逐漸具備完善的行動力量。

精神能量主要由認知，情感和意志三個方面匯聚而成。而信心恰恰可以拓寬五感對外界的認知，不會輕易否定自己的情感，堅定自己在意念上的方向，因此能量變大了。這樣人們對你的影響就會變小，卻讓自己精力更加集中。

03，信心是堅韌和耐力的源泉。

有了對自己的信心，就會充分的調動你自身的積極性和能動性。當面對困難的時候，你就不會著急放棄，也不會馬上踏步不前，而是為了克服困難變得很能忍受。即使困難再大，都不會輕易動搖你的決心，反而變得堅韌。

當你遇到問題的時候，並不是都可以輕易解決的，總有難以解決的，可是信心給你帶來的心態就是很有耐心，也很有動力。耐心在於你爭取成功的渴望和等待，動力在於你相信自己解決問題以後的完整感和成就感。

04，信心是點燃潛意識的火焰。

每個人的自我意識都是由正在表現的現有意識和沒有表現出來的潛在意識所組成的。可能很多人不知道我們自己的潛意識蘊藏著巨大的能量，這都是祖先給我們留下來的，封存在了潛意識裡而已。如果可以調取，必將強大無比。

然而潛意識是需要喚醒的，不是隨隨便便就可以開發那裡的，所以就需要渠道去打通它。然而很多人依然不能打通潛在意識里的能量，就是因為你自信的「小宇宙」沒有爆發。而點燃潛意識火焰的正是你對自己充滿信心。

05，信心是對接希望和現實的橋樑。

一個人想要有希望，無非是依靠自己可以擁有幸福，可以獲取成功等。就這麼簡單的兩個願望在現實中操作起來，也不是每一個人都可以順順利利做到的。而信心就是你最需要的一種媒介，它可以讓你有能量，也可以讓別人信任你，因此它可以很好的對接現實和希望。

沒有信心的人，希望都是渺茫的，甚至不會產生任何希望。沒有信心的人，現實對於他們來說就是囚籠，就是地獄。信心少的人會把希望看作比登天都難。然而對自己非常有信心的人就會自動架起通往希望的橋樑，因為有能量，有方向，有技法。

06，信心是逆境中的真理。

當你在逆境中前進的時候，一定要記住一點，那就是別人永遠不會對你絕對負責，所以別人的話僅僅就是旁觀。而真正前行的那個人是你自己，最了解自身處境的也是你自己，所以真理一定發生在你的經歷中。而你只有對自己充滿信心才會證實真理，轉逆為順。

任何真理都會出現在完善成熟的思考中，嚴謹重複的實踐中，極少數人的信念中。而信心恰恰就存在它們每一種情況之中，為什麼會是這樣，那就是因為信心的每一個結果都是成功，而成功的每一次累積就是真理。

結語：如果歷經世事的你不曾對自己有信心，那麼你的自我能量遠遠沒有達到最大。而已經小有成功的你一定會發現，信心就是你那源源不斷的能量供給。如果你沒有多少經歷，也沒有成功過，請相信我說的話，信心會喚醒你那深藏的真我靈魂，點燃你那頑強不屈的鬥志，創造出你自己都不敢相信的奇蹟。
