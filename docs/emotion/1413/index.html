老二是個男孩，媽媽想讓老二跟自己姓，這位爸爸的反應讓人佩服

表哥和表嫂都是獨生子女，兩人沒結婚的時候，舅舅就說到時生兩個，不管男女，大的跟表哥姓，小的跟表嫂姓。後來兩人結婚後，頭胎生了個女兒，就跟了表哥姓，兩人本來不打算生二胎。但後來表嫂意外懷孕，二胎生了個男孩。我去看錶嫂的時候，正好碰到一家人再說老二起名字的事情，我聽出舅舅反悔的意思了，舅舅說，男孩跟著媽媽姓，親戚朋友總免不了說閑話等等，表嫂也有點生氣，直接說婚前不是說好了，兩個不管男女，大的跟爸爸姓，小的跟媽媽姓，怎麼現在因為老二是個男孩就不願意了。剛巧，表哥買東西回來了，聽到表嫂和舅舅的爭執，表哥直接說，說好了跟著小X（表嫂的名字）姓就跟著小X的姓，小X也是獨生女，生養孩子那麼不容易，該跟著小X的姓，不管跟著誰的姓，孩子都是我們的寶貝，我們都一樣愛他們。

聽到表哥這番話，我心裡覺得很暖心。雖說如今隨著思想觀念的開放，獨生子女家庭的增多，兩個孩子，一個隨父姓，一個隨母姓是很常見的事，不僅是對女方家庭的一種安慰，也代表著男人對自己老婆的感激和尊重，夫妻感情好的象徵。但還有一些男人根深蒂固的大男子主義思想，覺得兩個孩子，女孩隨媽媽姓沒有多大問題，但如果男孩隨媽媽姓，很多男人家庭都接受不了，像表哥這樣真的挺難得的。

當然後來老二還是隨表哥姓了，表嫂說：「雖然生孩子這件事女方付出的比較多，但在國內的大環境下，男權觀念還是深入人心，老公能這麼說這麼做，我心裡很佩服也很感激。後來是我堅持讓老二隨老公姓的，本身我對孩子跟誰姓沒有多大意見，我爸媽也沒有讓我們為難，我要的只是老公和婆家人的一種態度，是我為了避免以後的很多麻煩才堅持讓老二也隨老公姓的。」

雖然兩個孩子一個隨母性，一個父姓，是女性家庭社會提升，夫妻感情好的一種見證。但從長遠看，潛藏在裡面的危害也不能忽視：

首先會讓兩個孩子遭到兩邊長輩不公平的對待

雖說很多父母都說，不管孩子跟誰姓，都是自己的孩子，都會是一視同仁，父母儘力都能做到公平公正。但兩邊的長輩們就不一定了，人都是有自己的感情傾向的。一個隨父姓，一個隨母性，爺爺奶奶這邊難免會覺得隨自家姓的孫子才更親，日常生活中難免偏向隨自家姓的孩子。反之姥爺姥姥這邊也是，生活中也會更待見隨自家姓的外孫子外孫女。甚至涉及到分家產的時候，同樣是自己的親孫子或者親外孫，但因為不是自家姓，財產也沒份。長輩們的差別對待和各種不公平，對孩子的心裡和成長也是很大的傷害。

導致親兄弟姐妹之間有隔閡，關係越來越不親

一個隨父姓，一個隨母姓，最直接的影響就是讓親兄弟姐妹之間有隔閡。明明是同一個爸媽，但兩人連姓都不一樣，很容易讓孩子從姓上就自動和自己兄弟姐妹劃分開了，我們不是一個姓，我們不是親的，會導致孩子之間生分，隔閡，沒有認同感。孩子小時候不親，長大更容易因為一些利益而鬧得不可開交，父母也跟著遭罪。這都與父母多生孩子，希望手足情互相陪伴，老了有保證的初衷相違背。
