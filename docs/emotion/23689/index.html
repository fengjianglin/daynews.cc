心理學：一個越是“不合群”的人，內心越是強大，原因主要有3點

2019-12-19 68條評論 2,248次閱讀 510人點贊

導語：心理學：一個越是「不合群」的人，內心越是強大，原因主要有3點

生活中不合群者，獨來獨往的人，一定有他的過人之處。

在這個世界上，大多數人都會選擇隨波逐流，隨遇而安，這類人註定是平凡生活中的一份子，而與這類人相反的是，有些人雖然與眾人不同，被人貼上了不合群的標籤，但他們活得卻很有價值，值得身邊每個人去學習。生活中不合群的人還被分為兩種，一種是被人厭惡的不合群，一種是被人嫉妒的不合群，前者一般都是被人鄙夷的，而後者大多數都很有頭腦。

「合群」這個詞在大多數人眼中會被定義為，大家在幹什麼，那我就跟著干好了，就算有不認同的想法，也只會深埋在心底，去做一個隨波逐流的人，生怕自己被大家不接納。而「不合群」的人從心理學上來解釋，雖然缺乏社交能力，但他們會把社交能力放在提升自身的上面，對於將來的成長，有著很大的益處，根據心理學家的觀察，會發現一個越是「不合群」的人，內心越是強大，原因主要有3點。

圖片來源於網路

1.「不合群」的人能認清方向

俗話說：君子之交淡如水，距離拉近，隔閡加深。不知道大家有沒有看過《烏合之眾》這本書，書中說到：人一到群體中，智商就嚴重降低，為了獲得認同，個體會拋棄是非，用智商來換取那份讓人倍感安全的歸屬感。顧名思義，生活中大多數人為了那份安全的歸屬感，拼了命的去合群，甚至不惜降低自己的進取心，還耽誤了不少正事。

舉個生活中的例子，我們在上大學時，班裡的同學會很積極的參加各種社團活動，為的就是讓自己前途光明，也為了顯示自己的合群，因為大家都參加了，如果自己不走群眾路線就會顯得有些孤僻，但有些人的合群只是盲目跟隨，這類人心中並不知道自己的目標是什麼，而是想著既然別人幹了，那我也一起去，然而這樣的想法只會讓你的生活亂七八糟，並且還沒有收穫。而那些「不合群」的人， 不僅能認清自己的方向，還能將知識深深的扎在血和肉中，比起盲目的合群有用太多，他們的知識儲備和技能相比其他人來說可謂是持之以恆。

2.「不合群」的人敢於追求夢想

其實生活中那些不合群的人，並非是遠離了這個世界，而是擁有了一個屬於自己獨立的空間，他們敢於面對自己的內心，敢於質疑這個世界的法則，不會像其他人一樣，盲目的去追求，他們不僅知道自己想要什麼，還會為此付出努力勇敢去追求。這類人不在意別人眼中的自己是怎樣的，也不會去思考自己合不合群，而是會在有限的時間內做著自己想做的事。

生活中有些人就像是漂亮的花孔雀，除了好看幾乎一無是處，雖然想討好別人，但最後在別人眼中卻沒有了位置可言。要知道這個世界是殘酷的，只要你有能力，即使你再怎麼「不合群」，也一定會有人欣賞你，但如果你沒能力，你再怎麼圓滑，也一定會被人嫌棄，有些不合群的人，喜歡過著獨來獨往的生活，而有些人整天混在朋友之間，反而沒什麼能力，所以我們在生活中，一定不要小看那些不合群的人。

3.「不合群」的人敢於拒絕他人

「不合群」的人往往是走向優秀的開始，他們從不盲目的去合群，就好比奇葩說中的黃執中，黃執中就是一個很不合群的人，他在錄影結束後從來不會參加一些其他聚會，雖然這種行為看起來非常的不合群，但其實黃執中只是不想去合那些他認為毫無意義的群。黃執中是一個能力者，他的圈子本身就要比大多數人高，所以他可以說不，也敢於拒絕他人，因為他本身就很優秀。

黃執中的不合群建立在能力的基礎上，他不僅是華語辯論圈中的高手，還是芸芸眾生中的星光，即使夜空中只有他一顆星，卻也能照亮自己，可以說黃執中的不合群，其實是因為知道自己內心想要什麼。生活中你見過哪個沉迷於學習的人，抱怨過自己不合群？如果有，只能說明他們的壓根不知道自己想要什麼，這類人更像是牆頭上的草，別人說什麼他們就會做什麼，而他們的目的，可能只是為了想要滿足群體的要求而已。

小編總結：一個越是「不合群」的人，內心越是強大，原因主要有以上3點。其實不合群只是表面的孤獨，而合群了才是內心的孤獨，在現在這個浮躁的社會中，成功對於每個人來說都非常的公平，如果你努力了，那你將會越來越幸運，相反，如果你不努力，整天只知道遊手好閒，不會為了理想的生活而付出努力，這樣你就很難得到自己想要的東西。雖然在現實社會上很難餓死一個人，但如果你沒有能力，就會受到社會上很多不公平的待遇，其實命運是很公平的，就看你是否能掌握到它，生活中一個人內心的強大並不是先天而來的，而是需要通過後期不斷的磨練，每個人都是孤獨的個體，隨著時間的推移，人一輩子總有一些坎坷需要自己去闖，而一個真正有能力的人，不僅敢於接納自己，同時也會有很多時間去反省自己，他們會練就一顆強大的心，換句話來說，一個不合群的人往往會比合群的人心智更堅強，正是因為他們懂得利用時間來審視自己，所以才能在成長的道路上不斷去反思，從而讓自己的內心更加成熟，也不容易被一些小事撂倒。
