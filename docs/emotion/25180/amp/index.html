愛情是建立在物質基礎上的奢侈品，當邁向婚姻的那一刻，就不得不談錢了。

好的愛情當然是脫離金錢束縛的，但前提是夫妻雙方都擁有絕對的財富自由，可以對婚後的經濟花銷等忽略不計，也就更大程度的接近愛情本身。

事實上，現實生活中，更多的婚姻是在柴米油鹽中度過的，更多的生活現狀是在金錢的算計中向前的，我們沒有辦法做到不談錢。

每當談婚論嫁的時候，女方都會向男方索要彩禮，男方就會覺得女方拜金、物質，金錢就不得不站在了敏感的中心。

張靜今年37歲了，在談到自己至今為止感觸最大的事情的時候，張靜表現出了自己的憤恨、無措和豁達。

張靜在自己最美的年華里遇到了自己的真愛，跟那個男人相守了6年，從開時候沒多久，兩個人就開始同居了，之後的事呢過活除了沒有領結婚證和舉辦婚禮以外，早已經跟正常夫妻過得沒什麼兩樣了，眼看在一起就快七年了，雙方開始討論結婚事宜了。

張靜這個男朋友家裡面雖然不是什麼大富大貴，但過得還算不錯，張靜媽媽覺得20萬也不多，就當是自己撫養女兒長大成人呢？最主要的是，讓他們家清楚，自己家養的女兒不是那呢輕易就能嫁給他們家的。

但就在彩禮問題上，男方家長卡住了，覺得張靜和自己家兒子在一起這麼久了，就算不給彩禮，還能不嫁過來不成？

這麼一來，張靜就十分在意男人的舉動了，令她失望和痛恨的是，跟她在一起這麼久的男人，此時此刻認為，他父母賺錢也不容易，站在了他父母的一方。

張靜著實被氣到了，於是直接甩臉不嫁了，不僅不嫁，還放出去話，沒35萬別想娶到她！因為張靜覺得，她值得更好的，這個男人負她，她就要讓他看看，她張靜是有能力也更值得更好的。

好又來張靜一連相親了很多個對象，那些男人一聽說張靜就是那個彩禮要價35萬的那個，便紛紛找借口離開了，張靜雖然有些動搖，但是話放出去，就沒有收回的道理。

後來張靜遇到一個茶商，茶商和張靜似乎有一點一拍即合的感覺，一點不拖泥帶水，說願意給張靜35萬，但前提是先領證，就像在商場上買賣物件一樣。張靜覺得這個茶商不拐彎抹角的直接，很對自己胃口。

但就當給彩禮之前，茶商生意賠了，遲遲不給彩禮，而張靜也遲遲未鬆口，這下茶商對於這場「生意」直接不談了，走之前還暗示張靜，也不看看自己多大了，自己的條件配嗎。

張靜開始後悔自己當初做出的決定，覺得自己不應該因為生前男友的氣，就做出後來的決定，現在是真的「搬起石頭砸自己的腳」。

自己年紀越來越大，說實話還真有點慌了，尤其是張靜的父母，天天催著張靜趕緊找，不要管什麼彩禮了，最後還找不到，張靜父母乾脆放出去消息，誰願意娶張靜，她們家願意出房子首付，於是一窩蜂的人開始主動跟張靜相親。

相親的過程中，很多人很不避諱的表示了自己的疑慮，問張靜還生得出孩子嗎之類的，張靜一下子覺得自己可悲到了極致。而她的那些相親對象們也很令她難過，都是些歪瓜裂棗，還有的有不良惡習，經不起一點打聽。

張靜開始由衷後悔，要是自己年輕時那會遇到一個好男人就好了，要是前男友負自己時自己不衝動就好了，現在這個歲數了，還被人挑選來挑選去的盡被人挑剔。

想來想去，她今天的狀況都是自己作的。識人不清，又不知道及時止損，和前男友在一起五年，卻沒有看清他不愛她。

為了報復前男友，又把自己推到困境之中，用天價彩禮來證明自己有人愛，而事實上卻沒有，人們都很現實。結局只能說明自己當初的決定有多蠢，只可惜但是的自己不懂，但現在後悔卻也來不及了。

好的感情當中，男人會主動問彩禮的，而如果是女人先問出了扣，男人的回答也尤其重要，這不僅是金錢的問題，還是男人態度的問題，可以窺探到婚後這個男人對你好不好。

彩禮錢，可以要，但要根據男方家庭的實際情況而定，不能獅子大開口，要懂得有商有量，不然同樣會讓男人看出你自身的毛病。
