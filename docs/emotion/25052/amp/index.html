夢語分享：只要你的心足夠完美，這個世界就會完美！

夢語分享：人生有緣彌可貴，歲月無期當自珍。只要你的心足夠完美，這個世界就會完美。生命的樂趣和幸福藏在平凡中，人生的真實和自然也體現在平凡中。

學會拒絕那些華而不實的事物，慢慢明白生活有陽光有風雨。明白有些貪念才是痛苦的根源，生活並不是完美的，有些東西看開看淡了，煩惱自然少了。人生無常，不強求不將就，淡泊歲月，人間三千事，淡然一笑間。

早上好朋友們，祝福一天好心情。關注我們，一起成長。

想和你坐一列綠皮老火車

文：付志勇

想和你坐一列綠皮老火車

很慢很慢的那種

它在冬天出發 它在春天才能到達

我們選擇臨窗的座位坐下

看田野 山川 河流 城市 鄉村依次出現

它們像我們的兄弟姐妹一樣親切

要有書 也要有茶

我們隔著氤氳說話

陽光就落在杯子里

列車會在每一個車站停下來

有人上車 有人下車

這些站名我們或許記住或許忘掉

夜裡 我們就躺在鋪上

我看書 你聽音樂

窗外夜色濃厚 星星為我們挑落燈花

不問來路 不問終點

我們坐在綠皮老火車上把時光蹉跎也卸掉年輕時一身的勞累和困頓

列車又啟動 我們相視一笑

十指緊緊扣住 就是這樣

你說走天涯我絕不說回家。

夢語分享：今天是冬至，是二十四節氣中的第22個節氣，也是中國民間的傳統節日。冬至被視為冬季的大節日，在民間有「冬至大如年」的講法，所以古人稱冬至為「亞歲」或「小年」。「氣始於冬至」，故冬至被認為是養生的大好時機。因為從冬季開始，生命活動開始由衰轉盛，由靜轉動，此時科學養生有助於保證旺盛的精力而防早衰，達到延年益壽的目的。

冬至，早安。記得吃湯圓水餃哦！關注我們，一起成長。
