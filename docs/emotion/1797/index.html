高僧說：心軟不是善，少管四件事，才不會毀了自己

2019-11-27 0條評論 24次閱讀 0人點贊

摘要

人們常說，做人要心存善良，要成為一個心軟的人。當遇到他人有困難的時候，自己一定要挺身而出，無論他人有沒有求助於自己，都要力所能及的去幫忙，從而讓這個世界更加的溫暖。有些人覺得，自己這樣做，是在為自己積累福報，自己在慢慢的成為一個更加善良的人。

人們常說，做人要心存善良，要成為一個心軟的人。當遇到他人有困難的時候，自己一定要挺身而出，無論他人有沒有求助於自己，都要力所能及的去幫忙，從而讓這個世界更加的溫暖。

有些人覺得，自己這樣做，是在為自己積累福報，自己在慢慢的成為一個更加善良的人。可你不知道的是，心軟並不是善良，幫助別人固然是值得讚揚的，但沒有思考的幫助別人，很可能對自己造成傷害。

心軟與善良從來不是兩回事，善良的人應當是愛憎分明的，而心軟的人卻不懂這一些。因為高僧曾說，心軟不是善良，少管四件事，才不會毀了你自己。不知你有沒有管過這四件事呢？

不要管違背良心的事，做人準則不能忘

個人無論到什麼時候，都不能拋棄自己的做人準則，都不能違背自己的良心。我們願意樂善好施，願意為他人提供幫助，但我們終究不能忘了自己的本分，不忘初心方得始終，如果自己都丟了，又拿什麼去幫助別人？

不要做那些違背良心的事，哪怕是自己關係很好的人求助，也要懂得學會拒絕。中國有句古話，一失足成千古恨，你永遠不知道，你的一時失誤，到底會為自己帶來多少的災難。

你還有自己的家庭，你還有自己所見所愛之人，若你違背了自己的良心，再見到他們時，你不怕心存愧疚嗎？

哪怕你是一個極其善良之人，你也一定要懂得，守住自己的本心才是正經。不要因為一時的心軟拋棄本心，否則你終將被生活所拋棄。

不要管強求你做的事，我們有基本的自由

法律賦予我們最基本的人身自由，我們有自己的義務，有自己的責任，但我們同樣享有自己的權利。人人都是平等的，沒有人有權利要挾你做任何事情，只要你不願意做，那麼你就有足夠的理由去拒絕。

不要因一時的心軟，去答應別人，不願意做的事情，終歸做不好，不願意做的事情，終歸會影響自己的生活。我們是一個善良的人，但我們拒絕做一個心軟的人，我們願意為他人排憂解難，但我們更願意讓自己的生活，幸福安樂。

當別人強求你做事情時，學會拒絕是最重要的。不要害怕別人會對你說三道四，我們不虧理，又有何懼？我們堂堂正正，光明正大的做人，想做什麼就去做，不想做的，誰逼迫都不行。

不要管別人的家務事，家家有本難念的經

俗話說，家家有本難念的經。一個家庭若想關係和睦，沒有丁點煩惱，實在是太難了。關係很好的兩個人，必定會經常扯到對方的家庭，甚至會吐槽家裡的各種不順利。

若你是一個善良的人，此時此刻，你應該做到的就是陪伴與安慰，而不是過多的干涉。不要因為一時的心軟，就把手伸得太長，當你管了別人的家務事，別人不會感激你，反而會排斥你。

沒有誰願意把家裡的各種苦難，都向他人完全傾訴，你所謂的管別人家務事，只會惹人生厭。我們做人時，更要做一個聰明的人，該管的事情不要少，不該管的事情不要碰。

不要管別人的決定，你的決定不一定合適

當兩個人關係要好時，往往會互相的分享生活。每個人在人生的路上，總會遇到一兩個非常重要的決定，哪怕你們關係再好，哪怕你再心軟，你也要明白，不要管別人做的決定。

每個人都有自己的想法，每個人也都有自己的遭遇，沒有任何人能夠感同身受，你做的決定不一定適合他人。若你是一個真誠善良的人，那你便支持你的好友，讓他在前行的路上，知道有人祝福便好。

若你擅自替他人做決定，做的好了，別人只會心生感激，可如果做得不好，哪怕別人不埋怨你，你自己也會心生愧疚。

我們願意做一個善良的人，但我們拒絕做一個心軟的人。善良的人必定是真誠和陽光，他們在與人接觸時不卑不亢，也能受到他人的尊重，而心軟的人看似忙碌，看似好心，總會被人們嫌棄。

我們誰都沒有三頭六臂，不能把所有的事情都做得完美，該做的事情要做好，不該做的事情不要管。

就像是高僧說的那樣，心軟並不是善良，只有少管這4件事，才不會毀了自己，給自己帶來莫大的麻煩。
