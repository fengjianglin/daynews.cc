母親臨終對我說的一段夢，想起來就心疼、掉眼淚，那時我才十三歲

2019-12-18 267條評論 1,274次閱讀 151人點贊

母親臨終對我說的一段夢那時我才十三歲

一九六六年我上初一，放暑假的時候，媽媽的病已經很重了。暑假裡，我就一直在家，寸步不離地陪著母親，幫她做些家務，或者幫她帶帶孩子。當時三弟才兩歲多，四弟才剛剛出生兩個月左右。

那段時間，母親會經常感到肚子不舒服，每當這時就會找人按摩。當時我們村有非常好的按摩醫生，他們家有三個盲人，其中有弟兄兩個都是盲人，五十多歲，都會按摩，並且技術很高，遠近都很聞名。不但經常在本村給人治病，還經常到其他村子去，他們一家很受鄉親們的尊敬，其中一個還會治病，相面算卦，被某一個地區的人們譽為「活神仙」。我們家離他們很近，兩家關係也很好，只要母親不舒服了，我就會跑去找到那個伯伯，用手牽住他的盲杖引到我家中，給我母親按摩，我就在一邊陪著，完了我會拉住盲杖再把他送回去。

母親那一段可能由於病情嚴重，心情很不好，在和人聊天時，會經常談到病情，顯得很悲觀。也可能是冥冥之中感覺到什麼，總是對人說起對家庭和孩子的牽掛。出事的那一個晚上，正處於是三伏天，天氣很熱很熱。由於屋裡太悶，母親想到院子里躺著，我就和父親用一張木板，在院子當中的棗樹底下擺了一張床，鋪好扶母親躺在上面休息。夏天院里的蚊子多，天又悶熱，我就拿了一把扇子坐在母親的身邊，由於怕二弟三弟打擾，特意叫爺爺帶上他們兩個。爺爺正好晚上在外邊看莊稼，就把他們兩個帶到莊稼地里去住，四弟則是父親帶著，吃奶了才抱過去。我不停地用力搖著扇子，給母親扇風，一隻手累就用兩隻手。

開始就我一個人陪著母親，不一會我的一個姨奶奶去了。也就是我媽媽的姨媽，她和我們是一個村子的，離得有二里地的樣子，那天晚上她是特地去看望母親的。我姨奶奶去以後就坐下來和母親聊天，我則坐在那裡一邊仍然搖著扇子，一邊聽她們說話。其中就有一段母親說夢的話，我記得很清楚。母親對姨奶奶說，她晚上老做夢，夢裡老是夢見我們家的院子里長著棵構樹，（構樹是野生的落葉喬木，生命力很強，適應性強，繁殖生長的快，不用人種植也會在野地里發的很多。開始易生為灌木叢，人工修剪以後也會長得很高，現在農村的地里到處都是）。媽媽說夢見院子里的構樹長得很高很茂盛，是不是我的壽限到了，老天爺在託夢告訴我，我活的夠數（構樹）了。姨奶奶就在那裡不停地勸她，說不要胡思亂想，那都是夢不能信。母親嘆了一口氣，看了看坐在旁邊的我，說我如果不在了，這麼一群孩子可怎麼辦呢？我可不能死去，為了孩子謝天謝地也要讓我活下去，把孩子們養大成人。母親好像有什麼預感似的，說這話的時候，好像心裡充滿了恐懼，也充滿了對家庭的眷戀，更多的是對孩子們的不舍和疼愛。她不願撇開她幾個幼小的孩子，一個人離開這個世界，她有太多太多的牽掛。歷來有很多關於夢的說法，有的還賦予它迷信的色彩，我不相信迷信，當時我根本就不會想到那會是真的。但母親確實說了這個夢，並且在那個晚上真的就離我們而去。為什麼會做這樣的夢，如此的巧合，我思考了好多年，現在我終於明白了。那是人的天性，對生命的嚮往，對生活的熱愛。在自己病重之中，首先想到的是自己的責任，想到這個家庭和自己的親人，感到還有很多事情要做，不願意也不能年紀輕輕就離開這個世界；再一個就是母親的天性，自己在病重里最牽掛的就是孩子，孩子們都很小，出於母愛的本能，她不放心，可能想到她若不在了，孩子們可能受到的苦難和委屈，做母親的心裡怎麼能不牽腸掛肚。夢是心裡想，我想母親當時就是這麼想的，由於日思夜想，心性所向，所以就形成了那樣的夢，就是很自然的了。

其實我老家的院子里，在西邊圍牆的邊上，也確確實實的就長著一顆構樹。那不是專門種的，而是不知在什麼時候它自己發出來的。由於緊靠圍牆邊，也沒有管它，就慢慢的長大了，並且長得很高，甚至都超過了牆頭。也可能天意就是如此，那一晚上母親就真的離開了我們，永遠的離開了我們。而且母親走的那一刻所在的地方，就真的離那顆構樹很近很近。

這是我的媽媽在離開我的時候，最後一次也是我一輩子都不會忘記的她說的一段話。後來母親走了以後，我看著院子里那顆構樹，心裡就不時地會想起媽媽臨走的時候說的那段夢，我心頭的怒火頓起，拿一把斧頭就把它給砍掉了。
