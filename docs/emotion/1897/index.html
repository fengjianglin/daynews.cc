每次想買東西的時候，你可能就會想：我不掙錢，還是少花點吧。能不買就不買，省點錢。
你不捨得花錢，不捨得買東西，這樣你的生活質量就會大打折扣。

對自己好一點，就要學會獎勵自己，不開心的時候，獎勵自己一頓大餐，閑暇的周末，獎勵自己一場音樂會，計劃成功達標，獎勵自己一次旅行，保持美麗的秘訣，就是自我獎勵。’

買到了喜歡的衣服，吃到了期盼已久的大餐，你就會開心啊。

對自己好一點，就要充滿自信。

你就是你，你是最完美的你，你是獨一無二的你，人們往往會攻擊比她們強的人，因為沒人會在意弱者。

她們的言語，不過是嫉妒罷了!對於這種言論，報以微笑就好。

你就做你自己，活出最驕傲的自己，用最美的姿態。
