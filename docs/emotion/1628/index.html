3、有種心情，他不屬於記憶，卻如臨窗的心境，能說出來的是一段一段的滄桑，落在眉間的卻是一曲一曲的思念和無悔的歌謠，人生有夢總是難留，似紅塵匆匆，但幸福卻突然而來，不知在哪個拐角處，感覺世界不再有悲傷的侵襲，心牽著所謂的命運，從手心瀉落，卻發現世界變亮。早安！

4、水，越淡越清澈；人，越淡越快樂。淡然，使人簡單；簡單，使人快樂。心善，自然美麗；心直，自然誠摯；心慈，自然柔和；心凈，自然莊嚴。靜靜地感悟，抖落歲月的塵埃，以一顆無塵的心，還原生命的本真，以一顆感恩的心，對待生活的所有。早安！

5、生活告訴我，幻想終究只是幻想，幸福是靠我們的雙手一點點創造的，我們的心靈一天天沉澱的，不管生活多麼不容易，我們都要用最完美的心態來面對它。你對生活笑，生活就給予你快樂；你對生活哭，生活就回饋你雨季。

6、想要成功?那麼就要集中所有力量於已定目標上，絕不動搖是必備條件，同時培養自身破釜沉舟、不成功便成仁的堅定信念，抗擊任何誘惑、懶惰等來自於內外因的所有不良影響，全身心地投入到自己的目標中去。早安~

7、有時候，我們會給自己設置很多想要超越的對手。但我們最強的對手，不一定是別人，而可能是我們自己。當我們忙著超越別人的時候，也要時時提醒和反思自己。當你克服自己一個缺點的時候，你也讓自己前進了一步。

8、有理想，才有追求，才有道路，才有磨練，才有堅強，才有成功，才有告別青年，才是一個健全的人!其實畫中的世界，就在那裡，就在那裡，在你的心裡!早安~

9、我們的倔強，有時候更像一塊牛皮糖，黏在一處就會越來越緊，直至招人厭煩，從而忘記了它最初只是一塊糖，哪個角度都裹著甜蜜。

10、我們的批判念頭就像在做怨氣投資，不斷自動收到「利息」，不幸的是，它們把我們從當下抽離，而當下是唯一有真愛和幸福的地方。

11、在花季中，我希望自己能永遠記住先哲的那句良訓：生活中的船不能沒有理想的帆。生活中的理想就是為了理想的生活。早安!

12、努力也許不等於成功，可是那段追逐夢想的努力，會讓你找到一個更好的自己，一個沉默努力充實安靜的自己。早安！
