【寄語】

99.999%的兄弟幾乎都是在聊天這件事上栽過大跟頭。

【教學乾貨】

一、文藝青年式聊天並不能引起女生的共鳴

舉個栗子：

1、你美的就像一朵玫瑰花，讓人心醉不已啊。

2、你的靈魂和仙子一樣嬌貴，還夾雜一些，青春的嬌氣。

3、今天的天氣格外的讓人舒暢，我們不如出來一同欣賞如何。

說實話，但凡一個女生在看到這樣的聊天的時候，內心都是崩潰的，您說的這麼優雅別緻，難不成還等著女生給你批改一些您這類似作文一樣的文字？

Excuse me? 你是在跟女生閑聊並不是在面試，也不是在考核好不啦！

人家女生明明想的是和你扯幾句淡，但是卻被你書面化的聊天方式打敗了，轉眼就變成尬聊。

女人是感性的動物，跟女生聊天，其實和給女生播放一首歌、一部電影是一樣一樣的，你必須把有畫面感的內容，有效地傳遞給她。

白開水一樣的語言，就像一首無病呻吟的網路歌曲、一部豆瓣評價3分的爛片，會讓女生覺得和你聊天是在浪費時間。

這就叫「言之有物」，怎麼去做呢？

簡單來說，就是多聊「細節」。

每天都有幾百號人誇讚某個女生美麗、可愛、善良、聰明……然並卵。

舉個栗子：

「你的穿衣品味不錯，洋氣又不浮誇」

「你特別能去替別人考慮，和你相處很舒服」

「你身材很贊，一看就健身房的汗水沒白流」

這才是讚美女生的正確打開方式，撓癢要撓到最癢的地方，馬屁也要拍到心窩裡才到位。

約女生去玩，不要說：「我們去看電影吧」，而要說：「我知道家環境特別好的私人影院，我們過去一邊喝雞尾酒吃零食，一邊看《遇見你以前》」

你看，不光地點不用女生操心，看什麼，吃什麼，喝什麼都一條龍搞定了，好浪漫的感覺。連我都想參加了呢！

陳述對某一件事情的看法，不要說：「66666潘曉婷牛逼」

而要說：「潘曉婷打球特別專註，一直挺欣賞她，當然，她身材也很棒~」

好吧，雖然我主要是喜歡她的身材……略略略～

三、唯有框架，方能成功

在聊天的過程中 女孩會自覺的給你測試 而面對女孩的測試 你該如何面對？正確的方法是，給她一個圈套 讓她先跳進去 你再跳進去 她不跳 你不跳最簡單的例子 。

我將引用我的內部學員聊天記錄為大家講解這個方法。
