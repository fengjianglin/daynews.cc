這兩個“很糊塗”的人，眼前過得不如人，但大器晚成，必有後福

2019-12-16 429條評論 2,305次閱讀 674人點贊

原創 ┃ 萬紅雲

原創作品，抄襲必究

我們有時候，總是瞧不起身邊一些看起來「很糊塗」的人，反應也慢，做事也慢，不論做什麼都有點兒跟不上節拍。然而，別瞧不起一些糊塗人，也別看不起別人眼下過得不如人，他們往往是一些不可貌相的，大器晚成的人！

看一個人未來如何，千萬不要僅從一件事情，或者很片面的觀點來給一個人下結論。有些人哪怕你看著根本就不出色，也許多年以後，他是混得最好的一個。 這樣的案例在生活中是不乏素材的。

我以前一個同學就屬於上面的類型，上課回答問題從不舉手，一旦老師點他起來回答，也總是回答得不夠好。然而最後同學們混得最好的一個就是他，不但辦了工廠，還擁有著多家線上及線下店鋪，生意旺得不得了。以前那些壓根看不起他，眼裡只有班長與學霸的班花們，現在都開始圍著他轉。然而，名草有主，只能圍著看看而已。

而下面這兩個看起來「很糊塗」的人，就是這種比較典型的例子。別瞧不起他們眼前過得不如人，但大器晚成，必有後福！

一、被人看貶，瞧不起，卻從不爭執的糊塗人。

有些人明明被人看貶、瞧不起，然而卻從來不去爭執，也總不為自己辯解，看似就是那種糊裡糊塗的人。有些人，一旦被人瞧不起或者是看貶的話，就會顯得情緒特別激動，想盡辦法證明自己的聰明與正確，以此來博得一個公平。

其實，真沒有必要向別人證明自己。如果別人愛自己，就不會傷害自己；如果一個人珍惜自己，就不會輕視自己。所以，不用去證明，堅持做好自己。而一個看似什麼都不明白，什麼都糊裡糊塗的人，其實更是一種大智若愚的智慧。

這種不在乎別人眼中、口中自己的人，他們的內心很堅定，有著遠大的目標及生活理想。只是，礙於眼前的局面限制，不得不委以人下。但當他們積累到了一定的資源，就會厚積薄發，直上九霄雲。這種人就屬於大器晚成，必有後福者。

二、被人利用，被人套路，總吃虧，卻從不討較的糊塗人。

有些人，非常喜歡利用人。他們在生活中，處處顯示自己高超的情商及智商，喜歡設下圈套讓別人往裡面鑽，自己從中得利。這種人完全把別人當傻子來看待。一般的人一旦發現人為的圈套讓自己上當，就會特別生氣，從而生出許多報復的想法並付諸於行動，以此行為，為自己討回公道。

而還有一些給人印象「很糊塗」的人，他們卻在這方面反應的比較遲鈍與麻木，好像被人利用了也不知道，被人套路了還蒙在鼓裡。這種人，吃虧了，也不說。從來不去計較，從來不去爭論個明白。

其實，這又哪裡是糊塗人呢？不過是看著糊塗罷了。往往這樣的人，心胸特別開闊，凡事想得開。總是往好處想。他們認為自己被人利用，因為自己還有所長啊，所以才被人利用。在被別人利用的同時，何嘗不是給自己一個學習與實踐的成長機會呢。

生活中，我們有時候，特別得意自己的聰明與心計。其實，一些看似糊塗的人，他們往往才是最後的成功者。不過，因為他們受環境等各方面因素的影響與控制，而不得不暫居人下，不露聲色罷了。這些人，一旦機遇來臨，就會厚積薄發，一鳴驚人，大器晚成。

今天的話題是，這兩個「很糊塗」的人，別瞧不起他們眼前過得不如人，但大器晚成，必有後福！您認為呢，歡迎留言，發表看法。
