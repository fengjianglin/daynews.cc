人無常勢，水無常形，繁華過後皆雲煙，早安

2019-12-23 142條評論 1,950次閱讀 509人點贊

為什麼我們總是覺得痛苦大於快樂；憂傷大於歡喜；悲哀大於幸福。那是因為我們總是把不屬於痛苦的東西當作痛苦；把不屬於憂傷的東西當作憂傷；把不屬於悲哀的東西當作悲哀。一個人得意時候，戒得了驕縱，失意時候，便不易沮喪。打拚，上進，不是為了做給別人看，是為了不辜負此生。

世界上沒有兩片完全相同的葉子，也沒有兩個完全相同的人！有些事，經歷過，有些人，相遇過，那些深深淺淺的人生痕迹，是殘留在滄桑歲月里的精美詩句，成為我們生命中的經典。早安！

懂得保持距離，適當地離開，到懷念時再相見，這樣的關係，遠比永恆地黏在一起更長久新鮮。很多女人自尋煩惱，就是太執著，總想把愛人留在自己的視線內，令對方窒息，自己難受。遠一點的風景，多一點想像和希冀。這是我們在枯死的生活中渴望旅行的最大意義。

人生，就是一場與生活的較量。再長的路，堅持走下去，一步一步終會走完。再短的路，邁不開雙腳，也無法抵達彼岸。不要在乎一路奔波，會被腳下的石頭絆倒多少次，要知道，凡是走在這條路上的人，他都有摔倒的時候。只不過有些人選擇了忍著疼，站起來，拍拍身上的土，續向前。而有的人，則怨天尤人，就此沉淪，停滯不前。早安！加油！

你的人生不會辜負你的。那轉錯的彎，流下的淚水，滴下的汗水，全都讓你成為獨一無二的自己。只是努力了一陣子，但一遇到困難就各種憂傷，好像自己努力了很久一樣。早安！

安靜時想想過去一年，會有開心，會有難過。某某離開，一切的一切都變了。我變得不哭了，該流的眼淚也流幹了。只是還會想，如果我們現在還在一起會是怎樣，是不是還是深愛著對方。人生就像一場戲，因為有緣才相聚。相扶到老不容易，是否更該去珍惜。為了小事發脾氣，回頭想想又何必。

淡然於心，從容於表，優雅自在地生活。追求，就會有失望；活著，就會有煩惱。不要把什麼都看得那麼重。人生最怕什麼都想計較，卻又什麼都抓不牢。人生如夢，暮然回首萬事空。嬉笑怒罵即是生活，花開花落便是人生。人無常勢，水無常形，繁華過後皆雲煙。早安！

慾望是一片海，人生是一葉舟。常懷感恩之心的人，愉己也愉人；常懷欣賞之心的人，悅己也悅人。真正的瀟洒，是積極進取，樂觀向上。無憂無慮、無恐無懼、輕鬆愉快，就是人生的瀟洒與幸福。

不要太依賴一個人，因為依賴，所以期望，因為期望，所以失望。心有多寬，人生的路就有多寬。人生苦短，沒必要和生活過於計較，也沒必要過於看重利弊得失。你越在乎，就會在得失之間徘徊越久；如果你看淡了、心寬了，那些所謂的得失也不過如此。

生活不是風景畫，不會總是陽光燦爛。人生的修行，就是用智慧看透世間黑白，並用一顆善良的心成就一切美好，用一顆寬容的心，坦然地接納無法改變的不好。修行，就是寬容。容得下別人的中傷，忍得住困苦的折磨，放得下挽留不了的美好。早安！更多的精彩情感，哲理，美文；敬請關注微信公眾號：心成一片
