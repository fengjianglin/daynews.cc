借錢給你是讓你翻身的，不是為了讓我們翻臉的，錢到底什麼時候還

2019-12-24 1,179條評論 3,023次閱讀 1451人點贊

導讀：我們一定都有過這樣的經歷，把錢借給身邊最親密的人，到頭來錢沒有了情也沒有了。

主動伸出援手

我們在把錢借給身邊人的時候，為的是讓對方能夠儘快渡過難關。但有些人，借錢的時候千恩萬謝。錢一旦到了手，就再也要不回來了。

說好的還錢日期一拖再拖。不光錢要不回來，連原本要好的感情也沒有了。如果說對方真的是經濟困難，倒也是可以理解的。可偏偏有些人，日子一天天的好起來。買了車買了房，買高級服裝，就是不還你錢。

你問他要，每次都說沒錢。你不問他要，他絕口不提，要的次數多了，可能還嫌你煩：「不就是這點錢嗎？又不是不還你，天天催的有意思嗎？」

遇上這樣的人就太讓人心寒了，可是面對這樣的人，我們大多時候又真是沒有辦法。因為他們臉皮厚，又不怕難為情。我們要錢的人都要得不好意思了，人家倒是一副無所謂的表情。

小林姑娘就遇上過這樣的事，她說那是15年前的事了。15年前她和阿梅在一起打工，工作是在一家茶樓做服務員，一個月的工資是600塊。

她和阿梅就是在那家茶樓打工時認識的，兩個人關係非常好，晚上睡同一個被窩，白天形影不離。阿梅的父親因為賭博欠了很多錢，人也被關進去了。她的母親一個人在家，就做點手工活維持家用，還有一個弟弟在上學。

家裡為了給弟弟上學也欠了不少錢，所以18歲的阿梅就出來打工了。那年的年底時，她們在聊天時說起各自的願望。小林說希望過完年後，來年能找個好一點的工作，收入能夠高一點。因為一個月600塊，除去開支剩不了幾個錢。

阿梅卻滿眼淚光地說：「我的願望是，現在能有1000塊錢就好了，那樣這個年就能過好了」

小林的心被觸動了，當時1000塊的存款她還是有的，她覺得自己暫時也沒有什麼急用，不如借給阿梅讓她過個好年。就主動把這個錢借給了阿梅，並告訴她不用急著還，等她有錢了再還。

事與願違

阿梅當時很感動，說自己有錢了馬上就還給她。小林當時也沒想著阿梅什麼時候能夠還錢，只是覺得自己能夠幫助到阿梅，她覺得心裡很開心。

她們的關係也由此更親密了，兩個人幾乎無話不談。小林平時不善言詞，而阿梅是個開朗的姑娘。她希望和阿梅能做一輩子的好朋友。

後來她們沒有在一起工作，但也一直是很好的朋友。期間阿梅有了男朋友，之後結了婚，有了孩子，甚至又離了婚。已經10多年過去了，阿梅的父親也已經出來了，家裡也已經買了房子，買了車子。

可欠小林的1000塊錢，卻始終沒有還。

小林每次都很理解她，覺得她可能真的困難。但後來有幾次她真的生氣了，因為每次問她要，她都說：「你怎麼不早幾天問我，剛剛前幾天發了工資，都被我用完了呀」，意思就是現在沒有了。

之後小林每次都踩著點，等她發工資的日子去問她，她依然是說沒有了花完了。而且還要被她奚落說：「你怎麼又沒錢了，每天上班賺的錢還不夠自己花嗎？

一轉頭，又能看見她在朋友圈，曬自己花了多少錢的買買買，一件衣服七八百，一條裙子好幾百。

小林很窩火又無可奈何，每次都被她弄得心情很不好，她本來就不善言辭。後來乾脆也就不去找她了，想著這個錢她不要了，這個朋友也不要了。

反目成仇

小林不去找她了，她倒是又經常來找小林。那天阿梅說她在網上團購了2張美容券，可以做一次臉部護理，約小林一起去。小林本是不想去的，想著見到阿梅可以問她要錢，就去了。

在做面部護理時，美容師一直推薦她們辦會員，阿梅是想辦的。但是小林並不想辦，就告訴美容師說自己沒錢。美容師為了想儘快促成交易，就提議讓她跟朋友阿梅借一點。

沒想到小林還沒開口，阿梅先發話了說：「不借，她借錢不還的！」

這簡直是胡說八道，小林差點被她氣到吐血，怎麼會有這麼顛倒黑白的人？明明是她自己欠我錢不還，怎麼倒成了我欠她錢不還了？

由於平時小林經常問她要錢，她每次都說沒有錢，倒是偶爾有給過幾次幾十塊的。加起來肯定也沒超過100塊，還有就是這次做臉部護理的美容券9塊9，難道她是把這個錢當成是她借給我的了？

氣得不行的小林瞬間從美容床上爬起來，問她：「到底誰借錢不還了？你欠我的1000塊錢準備什麼時候還？」

阿梅可能也沒想到她會突然發難，當著兩個美容師的面，有點難為情。然後說：「不就是1000塊錢嗎？你當時借給我時就說了，什麼時候還都可以。怎麼現在說得這麼難聽幹嘛？」

說完拿出手機就給小林轉了1000塊。雖然錢終於拿回來了，但是小林心裡很不是滋味。她說我當初借錢給她，是為了幫她度過難關，也是為了增進友誼的，不是為了像今天這樣用來翻臉的。

早知是這樣的結果，不如當初不借呢。

要心懷感恩

不說這1000塊錢，過了十幾年早就超過了當年的價值，關鍵是小林當初借出去這個錢，沒有得到對方的感謝，倒變成了現在友誼破裂的關鍵點了。

她實在想不明白這是什麼道理，她說那1000塊錢對現在來說不算什麼，但那是她當年省吃儉用，辛辛苦苦存了半年多才存下來的錢。那是一份心血。

這也許就是人性的「貪念」在作怪吧。因為小林當初借給她時說過，什麼時候還都可以，有錢了再還。所以阿梅每次都說沒錢，反正我也沒說不還，我只是沒錢。

在阿梅心裡，第一是覺得小林好說話，她也不會跟自己撕破臉。第二是自己也沒說不還錢，所以這不算賴賬。

另外當時是小林主動借給她，借得乾脆利落。也許給了阿梅錯誤的信號，以為這是小林贊助給她的，在她心裡就認定了這是不用還的了。

所以我們在借錢給人的時候，還是得醜話說在前面，說好還錢日期，寫好借條。而且要債的時候不能問人家有沒有錢，而是要問欠我的錢什麼時候還。以免發生誤會。

再者1000塊並不多，阿梅可能覺得拖一拖就過去了，小林不至於為這點錢跟她翻臉。有時候人不要臉起來也是很無敵的，而且阿梅似乎是吃定了小林的性格，覺得她不會在大庭廣眾之下不給面子，才會在美容師面前反說小林借錢不還。

大千世界無奇不有，我們在生活中，或許都有遇上過這種愛貪「便宜」的無賴。有很多人借出去的錢，基本上也是要不回來的。如果不是金額特別巨大，又沒有欠條的情況下。

也只能是花錢買教訓了，這也是人生道路上的一種磨練吧。只是在經過這種事情之後，希望以後的人生路上，交友和借錢都要謹慎對待。

我們自己在接受他人幫助的時候，也要時刻警惕。不要放鬆了這顆感恩的心，更不能恩將仇報，或是以怨報德。

雖然我們都是無神論者，但我還是想說，一個人的福報是有定數的。不要覺得沒有人發現自己做了什麼，或者別人發現了也不能把我怎麼樣？就可以隨心所欲，糟踐了人心。

一切上天自有安排。
