通常用鄰里能人勵志身邊人，說的是人，其實是事，說對了嗎？

有人說「人，只能改變自己，不要試著去改變別人」，你，怎麼看？

東施效顰，西施全然不知有鄰里人在仿效自己。

通常用鄰里能人勵志身邊人，鄰里能人也全然不知，更不用說試著改變別人。

通常用鄰里能人勵志身邊人，說的是人，其實是事。

自己以為自己可以改變自己。無論長身體，爾後長知識，之後長本領，都要做個自己對自己認同的自己。長身體，先天性身體是基礎，各有差異。家庭護理成長之後，後天維護則靠自己。有，營養丶睡眠丶鍛煉等。特別對身體健康有影響方面有否克制，煙丶酒丶夜宵。長知識，就讀是否勤學努力。長本領，興家創業，撫育子女，人際關係。

常有拿身邊人比較試圖改變自家人，說的是人，其實是事。天下父毌多數為「普通人者」但都在期待子女「望子成龍」丶「榮宗耀祖」，如，小時候就有聽到鄰里長輩說，「你看，某人家孩子那麼會讀書，你都學不來」；如，高考之後，就有聽到鄰里人說，「你看，某人家孩子真有出息，考上某名校啦」；如，談婚論嫁時段就有聽說，「你看，某家孩子找到美女為媳婦啦」，如，成家之後，夫與婦，孩子與父母之間，都會以某人會做生意賺錢，某人單位好薪酬高，某人買別墅啦，如，老年之後，某人身體真好，某家孩子或媳婦真孝順等比較勵志身邊人。
