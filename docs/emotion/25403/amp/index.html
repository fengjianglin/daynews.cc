女人四十，不是日暮西山，是到了人生最美的華年。

女人四十，應該知道，這個世界上你活的不僅僅是老公和孩子，還有自己，還有自己的父母和兄妹。

女人四十，一定要有閨蜜，不要多，一兩個足夠，可以使你有個肆無忌憚訴說秘密的人，可以有個嘴饞了陪你暢快淋漓吃火鍋的人。

女人四十，一定要有藍顏知己，不能和老公說的話，可以有個傾訴的地方，實在受傷的厲害了，還有個可以讓你靠靠腦袋的肩膀，擦擦你眼淚的大手。

女人四十，一定要有私房錢，不要多，一兩萬足夠，你要知道，你的父母都已老去，隨時都有可能用錢，不要在急需用錢的時候，非問你老公要。

女人四十，不要逛街的時候，給老公買，給孩子買，要記得也給自己買。

女人四十，孩子處在叛逆期，老公處在更年前期，用成熟女性的那份愛去接納和包容他們。

女人四十，不要再買便宜的衣服。你不再是十八，二十，劣質的剪裁，會使你形象大打折扣。品牌的衣服，有實力，可以多買，沒實力，少買。要找到適合自己的穿衣風格，再漂亮的衣裳，不適合自己，堅決不穿。

女人四十，一定要記得保養了，每天稱體重。給自己一個標準，過了這個線，記得減肥，不僅僅是為了好看，還要擔心肥胖引起很多病。要記得每年體檢，要記得有不舒服就看醫生，要把所以的毛病都扼殺在搖籃當中。

女人四十，這時候你的老公功成名就，有房有車，一定會有人投懷送抱，不理她，把自己打理的清清楚楚，漂漂亮亮，風箏的線一定要攥緊，不要輕易說離婚，咱打下的江山，咱生下的孩子，咱餵了十幾年的老公，幹嘛送人？

女人四十，不要管老公的手機里有沒有曖昧簡訊了，不要管他晚上幾點回家，這個世界，沒有不偷腥的貓。他只要心還在家裡，就可以了。大鬧，兩敗俱傷，最後，準是留下你自己在家裡黯然神傷。趕跑一個，還會殺進來另一個，你沒那麼大精力。有那時間，咱拿老公的錢，健身，美容去。

女人四十，有男人喜歡你，開心吧，說明你的魅力依然。不要矯情，不要懷疑人家是不是要佔你便宜，沒準是你在占人家便宜。

女人四十，依然記得要多看書，年輕時晦澀難懂的四書五經，中外名著，這時候讀，你發現，會有豁然開朗的感覺。也可以看看時尚雜誌，休閑娛樂，增加點知識，提升點品位。咱不指著讀書再拿文憑，但是不能讓人當文盲看。

女人四十，沒事一定要上網，網路浩瀚，知識繁雜，快餐文化，QQ聊天，可以解悶，可以消遣，可以增長見識，不至於被孩子們說落伍，何樂不為。

女人四十，很多場合男人說黃段子的時候，千萬別像未婚少女一樣罵人流氓。願意，你可以含笑去聽，不願意聽，也可以含笑假裝去聽，至於進不進你耳朵是咱自己的事，沒人能管住你脫韁的思維不是？再不行，還可以肚子疼，一臉歉意地去趟衛生間。

女人四十，如有可能，談場戀愛，不一定轟轟烈烈，一定是自己真心所愛。這場戀愛，不是因為寂寞，不是因為報復老公，與所有的一切無關，與你的愛有關。但是，記住，這場戀愛，有它的規則，這個規則，怎樣制定，你心知肚明。

女人四十，不要濃妝艷抹，不要搔首弄姿，不要穿太前衛的衣服，不要在人群中太過聒噪，不要太張揚，那不適合你的年齡。一定要大方，成熟，穩重，有氣質。沒有氣質，那就在人前沉默，這比膚淺好一千倍。

女人四十，不要再攀比。有沒有豪宅，有沒有名車，所謂的身份地位，這些不用比，你的神態，你的穿著，你的臉色，你的雙手，會早早將你出賣。要比的，是在5年，10年後，你的孩子活的是不是比別人的孩子強。

女人四十，一定要記得常去看看父母，父母老了，時日無多，多陪陪老人，不留遺憾。
