越南女子與大她25歲的美籍男子通7個月電話，之後見面9天就辦婚禮

2019-12-18 1,165條評論 2,407次閱讀 1262人點贊

一般情況下，人們在離婚後，可能會對婚姻產生懷疑，即使對其他人產生感情，也難以投入到一段新的婚姻生活中。在越南，一名女子離婚後與美籍越裔男子通過電話聯繫7個月，見面不到10天就舉辦了婚禮，他們現在的生活是什麼樣的呢？

據越南媒體12月17日報道，My Ngoc今年31歲，她離婚後通過熟人介紹，認識了比她大25歲的美籍離異男子Anthony Thuan。他們用電話聯繫了7個月，見面9天就結婚。如今倆人感情幸福，還被邀請上電視節目，講述他們戀愛的故事，引發了網友們的廣泛議論。

近日，31歲的My Ngoc與56歲的美籍丈夫Anthony Thuan一起參加越南的一檔電視節目，與主持人聊他們過去的故事，讓現場觀眾都笑得很開心。My Ngoc有過一段失敗的婚姻，她曾經非常沮喪，對婚姻關係失去信心，認為自己再也遇不到共度餘生的人。

那段時間，有朋友看到My Ngoc每天的心情都很低落，擔心她壓力太大患上抑鬱，就給她介紹了遠在美國的朋友Anthony Thuan。理由是Anthony Thuan離婚後的心態很樂觀，正常工作、旅行，做自己感興趣的事情。

朋友認為，如果有Anthony Thuan開導，My Ngoc能慢慢從離婚的陰影中走出來，沒想到他們兩個人年齡相差25歲，竟然會發展出愛情。當時Anthony Thuan在美國，My Ngoc在越南，他們通過電話和網路聯繫，經常分享各自的經歷和想法，發現彼此的觀念十分一致。

7個月之後的春節，Anthony Thuan提出去越南看My Ngoc，然後果斷買了機票。見面之後，他們對彼此都很喜歡，除夕當晚，My Ngoc見Anthony Thuan獨自住在旅店於心不忍，打車過去陪他，沒想到他怎麼都不同意，堅持讓My Ngoc回家，說沒結婚不能住一起。

這個事情之後，My Ngoc和家人對Anthony Thuan的印象都很好，他們一起相處了幾天，去很多景點玩耍，與一些親戚和朋友見面，最後在兩人見面的第9天，舉辦了一場簡單而溫馨的婚禮。2周過後，Anthony Thuan帶My Ngoc回美國又辦一次婚禮，正式生活在一起了。

如今My Ngoc和Anthony Thuan過著幸福的生活，他們都經歷過失敗的婚姻，再加上三觀相似，所以在很多地方都能互相理解。近日，My Ngoc在電視上講述這段經歷，很多網友都為他們感到開心，也有網友認為，見面9天就辦婚禮，太倉促了。這樣的閃婚行為確實有一定風險，並不適合所有人，萬一遇到錯的人，後悔都來不及。那麼，您對這樣的婚姻有什麼看法呢？
