妻子失蹤1年後，女兒也離奇失蹤！是什麼讓這個家支離破碎？央視《等著我》今晚播出！

2019-12-17 944條評論 2,073次閱讀 590人點贊

　小周8歲時，她的媽媽告訴她：「我們都是『神』造的，如果你有什麼困難和問題，都可以向『神』禱告。」

　　從那時開始，小周的媽媽開始帶著小周一起信「全能神」邪教。那是小周第一次接觸「全能神」邪教，一個懵懵懂懂的小姑娘，就這樣走上了邪教之路。

　　那時候的小周和媽媽還不知道，自己認定的這個「神」，其實是早就被國家依法取締了的邪教組織。

　　13歲時，小周第一次考了年級第一名，那時媽媽也開始帶著她參加「全能神」信徒的「聚會」，在「聚會」時，別的信徒對她說：「你知道你為什麼能考第一名嗎？這不是你自己的能力，不是你努力得來的，是『神』在祝福你。」

　　在每天這樣的洗腦之下，小周認為，如果離開了「全能神」邪教，自己將一事無成。

　　後來小周考入了一所重點大學，她仍然相信，這些都是靠「神」的幫助，不是自己的努力。在畢業前的3個月，「全能神」邪教人員告訴她：「你不能再上學了，『神』現在讓你出去，為『神』做事。」

　　結果，受邪教蠱惑，小周瞞著家人，在臨近畢業的時候偷偷辦了退學，放棄了學業，放棄了夢想，成為一個為邪教工作的「機器」。　

　　在邪教組織里，她不能與家人和外界有任何聯繫，不允許使用手機等任何通訊工具，也不能隨便離開邪教組織給他們安排的住處，整個人就像人間蒸發了一樣。每天，邪教組織都會給她安排大量的工作，為了完成任務，她常常只能睡三四個小時。

　　在邪教組織里的每一天，小周也都很想家，想疼她愛她的爸爸，想從小把她帶大的爺爺奶奶。但邪教組織告訴她，人類都是自私的，等時間久了，她的家人就會把她忘了，只有「神」才是可以永遠依賴的。還告訴她，為了「神」，必須放下親情等一切牽絆，全心全意為「神」做貢獻。

　　小周不敢背叛「神」，因為邪教組織一遍遍讓他們發毒誓，如果違背「神」的意願，就會受到最惡毒的懲罰。於是，她拚命壓抑著自己對家人的思念，用沒日沒夜的工作麻痹自己。

　　那時，小周的母親也早已經到邪教組織中工作，家人與她也失去聯繫多年。但即便後來小周也進入了邪教組織，母女倆仍然沒有見過一面。邪教組織內部管理極嚴，只允許上線聯繫下線，離家後在哪裡生活、幹些什麼，都要聽從安排，不能有自己的想法，更不能去隨便打聽。

　　最可憐的是周父，妻子女兒接連失聯後，這個男人幾近崩潰。那個曾經幸福的一家三口，就此分隔三地。

　　所幸，3年後，2019年6月，警方將小周解救出來，通過一個多月的教育轉化，她終於幡然醒悟。

　　「原來，我爸爸一直都在等著我。有那麼多愛我和我愛的人，我不能再傷害他們，辜負他們。」　

　　時隔三年，小周終於與父親相見，父女倆相擁而泣，那一刻，她宛若新生。

　　可是，她的媽媽仍深陷「全能神」邪教組織，至今下落不明。小周現在最大的願望，就是找到媽媽，一家三口坐在一起吃頓飯。

　　來到《等著我》欄目，小周能否找到媽媽，實現她的心愿呢？
