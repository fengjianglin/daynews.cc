心理學：悄悄告訴你，自信的秘密是什麼

2019-12-03 0條評論 21次閱讀 0人點贊

摘要

當面臨一個巨大的挑戰時，失敗的可能貌似在各個角落潛伏著。也許你曾聽過這樣的建議：更自信一點。在你聽到這樣的建議後你可能會怎樣想&quot;如果真有那麼簡單就好了&quot;。那麼，什麼是自信呢？相信自己是正確的，值得被看重的，並且有能力的，加入肯定自己的樂觀態度。

當面臨一個巨大的挑戰時，失敗的可能貌似在各個角落潛伏著。

也許你曾聽過這樣的建議：更自信一點。在你聽到這樣的建議後你可能會怎樣想”如果真有那麼簡單就好了”。

那麼，什麼是自信呢？

相信自己是正確的，值得被看重的，並且有能力的，加入肯定自己的樂觀態度。

這些會讓你變得力量，勇敢無畏地直面挑戰，這就是自信。它可以把想法變成實際行動。

那麼，自信到底從何而來？

影響自信的因素很多：

一：先天的因素，像是基因會影響大腦神經化學物質的平衡。

二：別人對待你的方式，這包括你所處環境的社會壓力。

三：你自己控制的部分，你的選擇，你要冒的險，加上你面對挑戰和困難時的想法，這三個因素密不可分。

但是我們自己做出的選擇，其實在樹立自信的過程中扮演著更重要的角色。

所以，通過牢記一些實用技巧，我們可以真正的擁有培養自信的能力。

技巧一：快速調整心態，善用一些小技巧讓自己在短時間內快速提升自信。

當你面對一項艱巨的任務，試著想像你成功時的畫面，你也可以擺出一個出色的姿勢或對自己說鼓勵的話。

技巧二：相信自己會進步。

如果你追求的是長期的改變，不妨想一想你對自己的能力與天賦的看法。

你認為能力是先天的嗎？還是像肌肉一樣可以鍛煉？

這些想法很重要，因為當你面對挫折時，他們會影響你的造成所為。

如果你有自我設限的想法，那就意味著你對於自身的天資沒信心，認為自己就是不擅長這件事，你也許就會放棄。

如果你擁有”成長性思維”，並認為自己有能力可以變得更好，那麼挑戰於你而言將會成為一次學習與提升的機會。

神經科學證明了”成長型思維”的人轉變學習與實踐，它們的腦突觸間的鏈接會得到不斷的強化。

研究表明，平均而言具有成長型思維的人會更容易成功，獲得更好的成就，並且在面對挑戰之時會表現得更精彩

技巧三：練習失敗，面對現實吧！

你總有失敗的時候，每個人都會有。JK羅琳曾被十二個不同的所出版商拒絕，直至一個出版商決定出版”哈利波特”，數據統計，世界上最成功的人往往也是失敗次數最多的人。

研究表明，那些經常失敗卻還堅持不斷嘗試的人，自身從能力上講，比較有能力面對挑戰與挫折。

他們學會了怎樣去嘗試不同的策略，尋求他人的意見，並且堅持不懈。

所以，考慮好你想要承受的挑戰，明白那不會是件容易的事，坦然接受自己將要犯的錯，並在此過程中善待自己，對自己突破鼓勵，站起來，並引發奮鬥。

真正令人興奮的是，不管你遇到怎樣的結果你都將收穫到更好的知識與領悟。

這，才是真正的自信。
