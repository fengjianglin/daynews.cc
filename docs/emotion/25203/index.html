有些人很幸運，手一牽，也就是一起走過了百年

2019-12-21 1,241條評論 2,951次閱讀 169人點贊

如果想過得快樂，那麼就應把生活跟目標聯繫在一起，而不是糾結在某個人或某些事上，學會在每一天里找到快樂，去感知微小的愉悅。我們常常是：看不到希望，放棄；對自己沒信心，放棄；……放棄是如此容易，可是有一個堅持的理由——我想要。

生活里，有很多轉瞬即逝，像在車站的告別，剛剛還相互擁抱，轉眼已各自天涯。很多時候，你不懂，

我也不懂，就這樣，說著說著就變了，聽著聽著就倦了，看著看著就厭了，跟著跟著就慢了，走著走著就散了，愛著愛著就淡了，想著想著就算了。

不要寄希望於事情變得簡單，還是期待自己能做得更好吧。

我不會去害人。即使我不夠好，任性，打鬧，發脾氣，也許我在外人眼裡是個壞人，但你請相信我一定會對你好，請你不要拿我和別人對比，也許我也並不是漂亮的那一個，但是我喜歡你，我就一定脫穎而出。

我常常是在幻想，有一天能在異鄉的街頭與你相遇，所以，我就愛上了旅行。

有些人不愛了就是不愛了，再勉強那麼也只是徒然。有些人很是幸福，一眨眼，就在一起過了一整個永遠。

有些人很幸運，手一牽，也就是一起走過了百年。不是一輩子的人，那麼也就是不去說一輩子的話，不去勉強，要能放下。
