百歲老人為妻子寫詩65年：擁有愛情是天底下最幸福的事

2019-12-24 1,400條評論 3,685次閱讀 1935人點贊

​用美貌去征服，用智慧橫行霸道

我是喵姬

文/雲水禪心

“六十春秋恩愛篤，雙雙執手難關渡。而今白髮同偕老，朝朝暮暮永相濡。”

從青絲到白頭的愛情到底什麼樣子？

97歲著名物理學家中科院院士、南大退休教授馮老為妻子寫情詩感動了所有人，65年里他將對妻子的深情都寫在了詩里，用這種平凡的浪漫寵溺了妻子一輩子。

有網友留言：此生只為你寫詩，這才是愛情最好的模樣。

但在社會浮躁，誘惑增多的今天，我們往往看到不是某某明星離婚上了熱搜，就是好不容易走到了一起的情侶，熬過了婚前的大風大雨，卻敗在婚後的雞毛蒜皮。

據民政局統計，我國離婚率連續14年持續增高，目前已經高達418.5萬對！

身邊人彷彿都是離婚狂，不是離婚了，就是在準備離婚的路上，原因很簡單就是不願意湊合過日子，一有矛盾就離唄。

看盡了分分合合滄桑繁華，到頭來老一輩人的愛情故事最讓我們動容，他們沒有甜言蜜語，更不會把愛掛在嘴邊，而是牽起那個人的手，一起相約到白頭。

1.生活中的儀式感是愛情保鮮的法寶

王小波曾說：好的婚姻，都需要一點儀式感，儀式感就是婚姻里的詩意。

婚姻就像一朵盛放的玫瑰，雖然嬌艷動人，卻時時都有枯萎的危險，枯燥的婚姻模式，會漸漸褪去愛情最初的熱情與溫暖。

當我們的生活被日復一日的平淡，給慢慢磨掉光芒的時候，儀式感卻能提醒你，在茶米油鹽的一片寒意之中，你們的愛情仍在發光發熱。

馮夫人有一個特別寶貝的紅匣子，裡面裝的不是什麼昂貴的珍寶首飾，而是她和馮老六十五年往來的信件與一截年輕時馮先生為妻子剪下的青絲。

在櫻花四月的時節他倆結為了夫妻，那一年他們許下了一個浪漫的終生約定：

以後每年的四月，一定要一起去看一場櫻花，因為櫻花樹下有他們感情最好的見證。

就像馮老經常說的，平凡的日常生活，離不開柴米油鹽，但偶爾也需要一些詩情畫意進行點綴。

叔叔嬸嬸結婚已經有40餘年，雖然他們都是廠里普通工人，但他們的相處模式卻讓我們羨慕。

她老說回憶都是自己製造出來的，她和叔叔會認真過每一個傳統節日，也會為了春節去集市置辦年貨，在七夕節叔叔總會帶一束花回家。

在沒有手機的年代，他倆還會互贈卡片，卡片上雖然都是一些生活瑣事，但滿滿都是對彼此的愛意。

在快節奏的當今社會，我們總抱怨曾經那個大小節日都會製造浪漫老公再也想不起結婚紀念日、情人節甚至我的生日。

因為現實永遠不是童話，夫妻之間不可避會有磕碰，齟齬，摩擦，可是透過儀式感，你能看到對方那被雞毛蒜皮掩蓋住的愛。

就像樹上春樹所說：儀式感是一件很重要的事，注重儀式感的人，即使生活一地雞毛，也能把日子過成漫天雪花。

生活需要儀式感，它會讓你在平凡又瑣碎的日子裡，重拾對愛情的憧憬，對生活的激情。

2.維持婚姻的辦法

其實就是過好點滴生活

愛情縱使在浪漫，但也抵不過生活的平凡瑣碎。

年輕的時候，我們總以為有愛情就夠了，可結婚以後才發現，再美好的愛情，也要落實到雞毛蒜皮的小事上。

當初說好了為愛情赴湯滔火，最後的我們卻都輸給了柴米油鹽。

當激情已被生活磨的狼狽不堪，婚姻好像就是找一個人湊合過日子時，其實除了儀式感更重要的是過好平時的點滴生活。

這也是一個關於91歲老人的愛情故事。

91歲的饒平如老先生與妻子的婚姻持續了六十年，妻子先他而去，在對亡妻思念中他把與妻子在過去六十多年裡的點點滴滴生活逐一畫下。

三百多張畫的回憶里，幾乎看不到「愛情」「想你」這樣的字眼。

都是在講孩子，講工作，講肉粽子，講砂鍋……這些是他們婚姻的全部。

當妻子在去世前清醒的短暫一刻，也是對女兒說：「你要好好照顧你爸爸啊……」

兩人一生牽絆、一生坎坷，他們像所有普通夫妻一樣被柴米油鹽所困擾，替孩子的前程擔憂，為夫妻二人的聚散糾結，但卻想愛一生相伴一生。

正如平如老先生說兩個人生活一輩子難免爭吵，但維持婚姻的辦法，其實也沒什麼秘訣，就是過好點滴生活。

999朵玫瑰，閃亮亮的大鑽戒，在人群中大聲的說我愛你，曾經滿足了少女時代的我們對愛情所有的幻想。

但結婚後慢慢發現，平凡生活里的愛情才最動人。

奶奶是一名護士，每周都會輪到值夜班至凌晨兩點，可奶奶一點也不害怕，因為她知道，不管多晚，不管颳風下雨，都有一個人在醫院樓下等他，那個人就爺爺。

爺爺是個東北男人，但他每天下班會去菜場買菜，回家束著圍裙圍著灶台給奶奶燒好吃的。

偶爾我們也會聽見爺爺奶奶拌嘴，奶奶說「死老頭子不讓我省心」，可下一秒又罵「還不趕緊多穿點衣服」我們才知道他們的愛早已揉進歲月。

有過激情時刻，熬過柴米油鹽看盡人生風雨，嘗遍歲月冷暖，大概就是愛情的全部真諦所在。

莎士比亞的譯者朱生豪寫給太太有句話特別打動我：「我一天一天明白你的平凡，同時卻一天一天愈更深切地愛你。」

這才是愛情真正動人的樣子，要和你一起過柴米油鹽的平淡生活，也不曾減少一分甜言蜜語的浪漫。

3.幸福的婚姻

就是你在鬧，我在笑

約翰·戈特曼在《幸福的婚姻》一書中寫道：「婚姻成功的秘訣根本不在解決衝突上，而在於平時的相處和魔鬼般的細節中。」

生活中的每一個點滴都是堅守婚姻的理由，但每一個小細節都可能成為壓垮婚姻的一根稻草。

不少夫妻離婚沒有大是大非的理由，而是覺得生活了無生趣，如死水一般，可人最怕的，偏偏就是無趣。

在平凡的婚姻生活中，有趣才是最好的春藥，德卡先生的信箱中有一句話：日常都是很無趣的，有趣的是有你。

之前台灣有位「最可愛的爺爺」老夏，雖然得了阿爾茲海默症，每天都是「我是誰，我在哪兒」，唯一沒有忘記的卻是自己的妻子翠娥。

聽說老伴出去溜達了，要假裝生氣吃醋；

但看見天要下雨了，便急著給出門買菜的老伴送傘；

一不小心還會說兩句肉麻話，對著孫女說：「我怎麼不愛你奶奶，我不愛她我愛誰啊！」

老家鄰居郭爺爺特別寵郭奶奶，有一次他們一起去鎮上看電影，郭爺爺一邊買票一邊裝作嫌棄的樣子說：

「其實我壓根不愛看這玩意兒，都是這小妹妹要看。」說罷指了指身邊的郭奶奶，接著兩個人都笑了。

人這一生其實很短，彼此相遇、相知、相愛、相守，直到頭髮花白時，回想起來也就一瞬間。

所謂幸福的婚姻，也許就是你在鬧，我在笑，在笑和鬧的歲月中，時光也被漸漸拉長，形成美麗的烙印，兩個有趣的靈魂在一起，註定要過有趣的一生。

4.

當高離婚率、家暴、出軌等明星八卦佔據熱搜，我們一度懷疑愛情和婚姻。

但看了這麼多老人的愛情故事後發現，能走到最後的從不是情話多動人，誓言多美好，而是在平淡中相濡以沫，在磨難中相互支撐。

人生漫長，平凡的生活中愛是最好的點綴，願得一心人白首不相離，持子之手與子偕老，在夕陽里挽著手慢慢走的佝僂身影往往最讓我們動容。
