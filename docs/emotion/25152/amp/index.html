做個會花錢的老年人，這樣才能過好幸福晚年

天天要聞
2019-12-21

聰明的老人，先把錢花在自己身上，再把錢花在兒女身上。捨得為自己花錢的人，才是聰明人，懂得花錢的人，是懂生活的人。人生到了這個階段，如何把自己的晚年生活過好，才是我們老年朋友該去想的。

聰明的老人不僅要捨得花錢，更要會花錢！怎麼花錢才對得起自己？我總結了5點，老年朋友來看看對不對？

1、先保健康再談享受

退休後想再就業、繼續學習、出去玩、幫帶孫等都有一個必要前提，那就是保證自己的身體健康。身體健康就好比財富數字里的那個”1″，沒有了”1″，後面再多”0″都只是”0″。

所以，為了我們的身體健康，那些舍大錢省小錢的事兒就別幹了：菜要買新鮮的、隔夜青菜不要吃、水果堅果適當吃、保健品要找正規渠道買，有病要及時去醫院看……當然也要勤鍛煉。如此一來，身體不健康都難。

2、旅途愉快心情好

所謂一切煩惱皆由心生，心不”健康”，吃再貴再好，跑再多步也是瞎忙活。

中醫認為：心主神，心為五臟六腑之大主。心藏君火，君火以明，主不明則十二官危。要養病，先養心。心情舒暢了，愉快了，病痛自然好得快。想要散心舒心，旅遊是不錯的選擇。

去桂林看看山水，去海南看看碧海，去大連吃吃海鮮，去北京故宮賞賞古建築之美……每天與美好相伴，煩惱全消失，順便還鍛煉了身體。

朋友李姐因為給兒子帶孩子，花了心力，結果還被說沒帶好，心情鬱悶，所幸不受這罪了，拉上老伴到處玩。自己跟老伴每天吃吃喝喝逛逛開心得不得了，哪還有心情想那些受氣事兒。還在旅途中交了好些個朋友，拓寬了自己的朋友圈，每天別提多快活。

3、養老事宜安排上

如今養老方式多樣，居家養老、養老院養老、抱團養老、旅居養老……這些都可以先考慮上了。如今年輕人壓力大，我們也容易心疼他們，能自己養自己的，就盡量靠自己吧。玩樂之餘，都可以先做嘗試，為自己選好後路。

比如旅居。一般人都是先玩上個幾年，旅遊經驗多了後，開始嘗試更多方式，旅居就是其中一種。

冬天時，北方、中部等地氣候寒冷，這時候可以選擇去南方避寒，去海南、去廣東，住上幾個月，可以慢慢感受當地生活、文化，深度旅遊，享受慢節奏的與前半生不同的生活方式。

4、老了也不忘打扮

網上總說出門旅遊到處都能看的戴帽子墨鏡披著花絲巾的大媽，他們那是不知道我們中老年女人的喜好，只以自己的眼光看世界。圈子不同不必強融，不用在意別人怎麼看待你，自己開心就可以了。

為了靚麗的外表，為了更年輕瀟洒，多花點錢在打扮上，無可厚非。裝扮了自己，何不是也裝扮了這世界？

5、趕趕潮流也挺好

世界在發展，科技在進步，生活更美好，趕潮流這件事，我們得跟上，可不能被時代拋棄。想要旅遊時多拍點好看的照片又圖方便的買部好手機，想學攝影的買相機，喜歡跳舞的漂流舞蹈服不能少，護膚美妝有興趣的也能嘗試，再是微博微信多玩玩，能學到不少新型辭彙……

奮鬥了一輩子，該有的已經有了，得不到的也不必再強求。在人生最後一個黃金時期，我們唯一要做的就是不辜負自己，不虛度餘生，用儘力氣，為自己而活。去充實自己，去旅遊，去追趕潮流，去做任何自己想做的事兒，為自己的喜好花錢，捨得為自己花錢，活出自己的風采。

一輩子很短，快樂就好！
