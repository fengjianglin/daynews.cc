隔輩親有多親？男孩放學放學等人接，看到爺爺後激動的哽咽，感動

2019-12-15 129條評論 2,473次閱讀 1455人點贊

隔代親的現象越來越嚴重，很多孩子和家裡老人的關係甚至比父母還親。這種情況其實不難理解，老人寵愛孩子，而家長又沒有時間陪他們，所以孩子們通常會對爺爺奶奶或姥姥姥爺更親。那麼隔輩親到底有多親呢？看了下面這一幕，不禁感動。

幼兒園放學，一個小男孩乖乖的站在學校門口等人接，看到一個緩緩走來的老先生後，激動地哽咽，用力的揮手並大喊「爺爺」，接著就跑過去撲進了爺爺懷裡。

小孫子和爺爺僅分開了一天，見到爺爺後竟忍不住落淚，可見爺孫的感情有多深。艷羨爺孫感情的同時，不禁深思：讓老一輩陪伴孩子有什麼好處？

一、老一輩的耐心更足

與年輕的寶爸寶媽相比，老一輩的耐心更足。或許經歷過人生的風風雨雨，老人對什麼事情都看淡了，也看開了，所以面對各種事情也更加有耐心。他們照顧孫子時，不會隨隨便便的亂髮脾氣；孩子問問題也會很耐心地給其解答。所以，孩子們也很喜歡跟老人相處。

二、老一輩人能給孩子更多的陪伴

對孩子來說，陪伴尤為重要。但現在很多寶爸寶媽都有工作，而且一旦忙起來很難抽出時間陪孩子。而爺爺奶奶，大部分都已經退休了，有充足的時間陪孩子。

三、老一輩人更能理解孩子

爺爺奶奶們活了大半輩子，該經歷的都經歷了。所以，對於孩子出現的各種狀況都能夠遊刃有餘、妥善處理。孩子的各種小心思爺爺奶奶也能夠猜出來，對於小寶寶的那些小算盤爺爺奶奶可比寶爸寶媽們懂得多了，也更加清楚地知道孩子喜歡什麼，想要什麼。

雖然爺爺奶奶帶孩子好處多多，但是也有一些小建議需要爺爺奶奶採納。

一、不要一味地寵著孩子

爺爺奶奶在帶孩子的時候，很容易把生活的重心都放到孩子身上。孩子的成長需要愛，但不需要溺愛。一味地寵著孩子，很容易讓孩子迷失自我，變得自私自利。等孩子大了會很難在社會上立足。

二、教會孩子獨立

老人帶孩子，有足夠的時間是好事，但是也很容易造成孩子的依賴心理。所以，老人們在帶孩子的時候，一定要適時的放手，教會孩子獨立。讓孩子多跟其他的小朋友在玩，讓孩子多出去走走，在日常生活中要注重培養孩子的獨立精神。

三、緊跟時代

老人年紀大了，很容易跟時代脫軌。但是孩子是祖國的未來，要走在時代的前面。所以，老人在帶孩子的時候，一定要關注時代的變化，讓自己的思想緊跟時代，這樣才能讓孩子不輸在起跑線上。

隔輩親雖然讓人羨慕，但也有一定的弊端。因此，家長不能一味地把孩子交給老人看管，要承擔起自己的責任。相信在爺爺奶奶和爸爸媽媽的共同守護下，每一個孩子都能夠健康快樂地長大。
