孩子是父母愛情的結晶，每一對父母都希望能夠從孩子的眉眼當中，看到自己和對方的影子，但是細胞在做減數分裂之後再進行重組，會打亂基因編碼上的順序。

因此，有一些不屬於父母雙方的特徵就會遺傳到孩子身上，但是這些特徵不是憑空出現的，仔細觀察之後，就能夠在家族其他人身上找到「同款」。

孩子是親生的，卻被隔代遺傳截了胡？這組圖體現的淋漓盡致

閨蜜瑤瑤最近剛剛升級成了媽媽，在朋友圈裡經常會看到她曬孩子的照片和小視頻，在下方的評論中，我們共同的朋友還會調侃，孩子一點也沒有遺傳了媽媽的大眼睛。瑤瑤也會跟我抱怨，不知道孩子的長相到底隨的誰，既不像他也不像她老公。

前兩天瑤瑤在微信上面神神秘秘地跟我說，她發現了一件特別神奇的事情，隨後給我發來了兩張她公公年輕時候的照片，還一併拍了一張孩子現在的照片。照片中瑤瑤的公公看上去意氣風發，沒有現在的滄桑，眼睛雖然比較小，但是看上去特別有神。

對比過孩子和爺爺的照片之後，我發現孩子眉眼之間的確更像爺爺一些，瑤瑤也覺得特別新鮮，還將爺孫倆的照片都發到了朋友圈裡，在下面的評論里，瑤瑤才知道，自己辛苦生的孩子，長得卻更像家裡的老人，這種被隔代遺傳「截胡「的情況，並不是自己一家個例。

其實這種現象在明星群體中也非常的常見，之前大S的婆婆張蘭在自己的微博里曬出了她和孫女小玥兒的合照，小玥兒的眉眼之間絲毫不見大S的神態，反而和自己的奶奶幾乎是一個模子里刻出來的。

不僅僅是長相，兩個人在吃麵包的時候，表情和動作完全像是在照鏡子，難怪張蘭對孫女異常的疼愛，因為這哪裡是孫女，明明是縮小版的自己啊。

從這些例子中我們可以看出，隔代遺傳的基因有多強大，所以千萬別小瞧了ta

1、孩子的一些細微特徵，最容易遺傳老人

有些父母會發現，明明自己和老公都是單眼皮，為啥到孩子這裡會變成雙眼皮，其實這就要從基因的顯性和隱性上說起了。

在人類的基因中，雙眼皮屬於顯性基因，如果孩子的爺爺奶奶有一方是雙眼皮，那麼雙眼皮基因就會被編入遺傳鏈中，即便是爸爸沒有雙眼皮，可是雙眼皮基因仍然在孩子的基因中佔據一定的地位，父母的基因分裂重組後，孩子出現雙眼皮的機率就會有50%。

2、 一些遺傳疾病隔代遺傳的概率也很高

有一些會被編入基因鏈中的疾病，即便是父母沒有，也有可能會遺傳給孩子，因為如果家裡的長輩有一方基因攜帶者有缺陷的基因，那麼這個基因就會遺傳給孩子的父母，因為這個缺陷基因很有可能是獨立存在的，所以在孩子父母身上不會顯現出來。

3、 孩子的身高，也會受到隔代遺傳的影響

孩子的身高並不是單個細胞決定的，而是多個有效細胞通力合作的結果，所以孩子是不是大高個，主要看從基因鏈中獲得的有效細胞有多少。

如果家裡的長輩是高個子，那麼即便父母的身高相對來說不佔優勢，身體中也是攜帶著一定數量的有效基因的，那麼孩子隔代遺傳高個子的概率就會有35%。
