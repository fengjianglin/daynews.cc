結婚5天，丈夫丟棄她去日本，苦等68年，再歸來時丈夫已有11子女

天天要聞
2019-12-18

民國時期正是新舊文化交替的時代，那時候湧現出一大批炙熱的新青年，他們渴望自由的生活，尤其是在戀愛方面，他們希望有一段美好的自由戀愛，但在古時候男女婚姻不由自己做主，都是父母給選擇婚配的對象，但隨著時代的發展，婚姻選擇權逐漸自由，自行婚配，可也有不少家庭還在尊崇著包辦婚姻，我們今天說的就是一對由父母安排的姻緣，在結婚5天後，丈夫就拋棄了妻子遠赴日本，在妻子苦等68年後，丈夫再回來時已經有了11個子女。

這位妻子叫張瓊華，雖然從小也接受了教育，但始終受封建文化的影響，骨子裡仍然尊崇著那些封建的東西，20歲那年，在父母的安排下，被指婚給郭沫若，那時她就已經知道他有些小有名氣，所以對他很滿意，對她的婚姻充滿了期待。而那時雙方結婚只能到了新婚揭蓋頭那天才能看到對方的長相，因為之前郭沫若的母親對他說對方長得還不錯，所以他就抱著很大的希望，想要見到這個漂亮的妻子，但在揭開蓋頭的一瞬間，他的心就涼了，連話也不想和她說。

用他日後的話形容就是，隔著布袋買白貓，結果買完才知道是黑貓，可以見得他對他的妻子是有多麼的不滿意，以至於在結婚5天後，就拋下新娘子遠去了日本。張瓊華知道丈夫不喜歡她，可她骨子就是一封建女子，尊崇三從四德，認為即使丈夫離她而去，她也應該堅守在家，認定丈夫總有一天還能回來。

她就這樣一直抱著希望，並且將家中事務打理得很好，對待自己的公婆就和親生父母一樣，將他們照顧得很好，贏得了公婆的滿意，但丈夫一直沒有回來，想要得到他的消息就是靠寄回來的一些信件，而這些信中始終沒有提及妻子一個字。她很是失望，但也將所有的信件和丈夫用過的東西珍藏起來。

她就這樣一直苦等著，日復一日，年復一年，而在日本的郭沫若卻過得十分瀟洒，並且另外找了一個老婆還生下了5個孩子，但隨後他就再次不告而別，回國後又娶了於立群，並且又生下了6個孩子，而他對家中等待他的妻子不聞不問。

直到1939年，郭沫若回到了家鄉，他只是簡單地對苦等了他68年妻子深深鞠了一躬以表道歉，妻子見到這個情景，也沒有再說什麼，毫無怨言。小編認為，郭沫若負了她一生，而她這一生都在等待，什麼也沒有得到，沒有兒女的她終在1980年病逝，結束了她孤寂的一生。
