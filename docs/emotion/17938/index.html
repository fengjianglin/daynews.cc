孩子“沒骨氣”膽小怕事？多半是這些原因，父母要多給予關愛

2019-12-16 290條評論 1,848次閱讀 673人點贊

文丨貞媽（文章原創 ，版權歸本人所有，歡迎寶爸寶媽們轉發分享）

前幾天，帶著姐姐家孩子樂樂，在小區公園散步，剛好遇到樂樂的朋友小美，剛開始兩人玩的特別好，後來因為一點小事，小美就特別生氣，樂樂也在一旁一直給小美道歉，但是小美一直充耳不聞，最後生氣地跑走了。我十分困惑為什麼樂樂明明沒有錯，還要一味地給別人道歉？為什麼樂樂不敢反抗，表達自己的想法，而且只知道用自己的妥協換來別人的原諒？

這也讓我想起來自己經歷的另一件事情：在幼兒園帶著孩子做課餘活動時，小剛把小松從滑滑梯上直接推了下來，小松忍著疼痛，在小剛的威脅下，不敢告訴老師，也不敢還手，只能自己吃”啞巴虧”，後來在我的調節引導下，小剛不服氣地道歉，而小松眼裡卻是充滿恐懼。這讓我十分困惑，為什麼面對欺壓，小松不敢反抗，而是選擇忍受？

生活中，很多家長都會遇到孩子”沒骨氣”的難題，自己的心肝寶貝在外面受到欺負，而不敢還手，只會忍氣吞聲，默默地忍受別人對自己的不公平。為什麼孩子會表現的”沒有骨氣”？現在的教育，不僅僅需要教導孩子有良好的品行，還要教會孩子有堅強的性格。父母如何幫助的孩子改掉這一缺陷呢？

一、孩子「沒骨氣」的表現

1.沒有自信心，不敢表達自己的想法

自信心不論是對大人還是對小孩都十分重要，有自信心的孩子敢於表現自我，敢於表達自己。很多孩子，因為缺乏自信心，就在各方面表現得十分懶惰，不想動手，長此以往，孩子就會缺少意志力，缺少骨氣，表現出一副畏首畏尾的樣子，於是就會被貼上”沒骨氣”的標籤。

2.不會拒絕他人，事事順從妥協

有骨氣的孩子，敢於拒絕他人，會對發生在自己身上的不平等待遇說”不”，也會在面臨外界的欺壓時，會主動尋求他人的幫助，討回公道。那麼”沒骨氣”的孩子呢？他們不會拒絕他人，不敢尋求幫助，更不要說反抗了。他們習慣了事事順從，用自己的妥協換得別人的原諒。

二、「沒骨氣」將造成的危害

1.孩子做事畏手畏腦，難成大事

有言道：志不立，天下無可成之事。

沒有骨氣的孩子，會容易產生自卑心理，他們喜歡自我否定，做事習慣敷衍，或者畏手畏腦。沒有骨氣的孩子，在面對困難挫折時，會更加容易產生退卻心理，輕言放棄。在孩子一生的成長道路上，怎麼會一帆風順，沒有一點坎坷？「沒骨氣」的孩子，會不堪一擊，難以成大事。

2.孩子心靈受創，內心十分壓抑

「沒骨氣」的孩子，在面對外界的欺負時，只會忍氣吞聲，而不敢正常的發泄自己的脾氣，長期負能量的堆積，會讓孩子的內心十分壓抑，也不利於孩子的身心健康。沒有骨氣的孩子，會表現得十分孤僻，戴上一個不屬於他的「面具」，把最真實的自己掩藏在「面具」之下，這樣孩子難以找到自己的真心朋友，長此以往孩子的內心會十分壓抑。

還記得《悲傷逆流成河》中易瑤的悲劇嗎？不正是因為長期負能量的堆積，才讓她的悲劇發生。沒有一個人願意把她當朋友，只把她當作欺壓的對象，導致內心十分壓抑，不堪重負，最終選擇了以極端的方式來解脫自己。

三、導致孩子「沒骨氣」的原因

1.父母脾氣陰晴不定，孩子習慣性”討好”父母

父母對孩子的影響是最直接的，因此父母的性格對孩子也會產生很大的影響。現在很多父母脾氣陰晴不定，十分古怪，有時候對孩子是笑臉，說不定下一秒就是一頓痛斥，因為父母的心性不定，孩子為了尋求自己的「救生圈」，於是就習慣性的「討好」父母，習慣性地對父母道歉，長期以往，孩子就會習慣對一切人妥協，這樣孩子就會變得「沒骨氣」，懦弱。

2.父母過分強調與人為善，讓孩子不敢拒絕他人

與人為善是我們中華民族的優良傳統，有的家長為了讓孩子從小養成對人友善的脾性，就給孩子灌輸友好的思想，過分強調與人為善。於是，孩子就會給自己下定義：自己要和別人友好相處，為了他人的開心，可以犧牲自己的利益。與其說孩子「沒骨氣」，不如說孩子是不知如何拒絕，只知道妥協，友善，委曲求全。在這種觀念下，孩子漸漸沒有了自己的脾氣，變得「沒骨氣」。

四、父母該如何幫助孩子重拾「骨氣」？

1.父母給予關愛，不要老是吼罵孩子

調皮和探索未知是孩子的天性，他們喜歡挑戰自我，探索自己未知的世界，而在這個探索過程中，孩子不可避免地會犯錯誤。這個時候家長是會選擇武力威脅、語言恐嚇，還是低聲教育、耐心引導？很多父母都選擇了前者，所以孩子慢慢地”沒骨氣”。在孩子犯錯誤時，家長不要大吼大叫，語言暴力，你和孩子心平氣和的交談，更能讓孩子改正過來，而且會慢慢地養成好的性格。

前幾天刷微博，看到了一個母親的分享，她後悔自己的行為，對孩子從小缺乏關愛，高標準要求，孩子一旦沒有達到要求就吼孩子，終於把孩子吼成了壞孩子。可見，父母的訓斥對孩子的傷害如此之大，孩子的骨氣就是在父母的吼罵中慢慢泯滅的。

2.教會孩子獨立，讓孩子有自己的思想

現在的父母，控制欲很強，他們喜歡控制孩子的行為，包攬孩子的一切事物，為孩子做好一切事情，慢慢地孩子就會變得喜歡依靠他人，思想愚鈍，他們沒有自己的思想，只是一具「傀儡」。他們依賴成性，沒有獨立的機會，所以孩子慢慢地就喜歡聽之任之，漸漸的沒了骨氣。因此，家長要教會孩子獨立，讓孩子有自己的思想，這樣孩子才會慢慢重拾「骨氣」。

3.帶孩子走出家門，多參加實踐活動，給孩子自主權

孩子「沒骨氣」，就會自卑，而喜歡待在家裡，不敢走出家門。因此，父母要多帶孩子走出家門，帶孩子去參加一些實踐活動，養成大膽的天性。在實踐活動中，孩子不僅會學到新的技能，而且可以鍛煉自己的膽量，敢去表達自己，重拾”骨氣”的好性格。

世界上沒有天生有骨氣的人，也沒有天生「沒骨氣」的人，有骨氣與否是後天習得的，受到很多因素的影響，而對孩子的性格產生最大影響力的就是父母。因此，作為父母用積極的方法教育孩子，引導孩子敢於表達自己，做個有骨氣的人。
