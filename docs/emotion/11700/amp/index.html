感恩的三種方法——回憶過去、展望未來，還有傳遞幫助

天天要聞
2019-12-11

一個人要時常有著一顆感恩的心，當我們學會感恩時，才會感受到其中那永久的喜樂。人常常會因為每天要面對忙碌的生活而把目光變得狹窄。然而，我們若是只注重眼前的成敗，就很容易忽略掉本來更加寬廣的一生。但擁有一顆感恩的心，就能讓我們的心胸變得更加寬廣。那麼如何懷有感恩的心呢？

我們要數數自己曾經所受過的幫助。人在世上，離不開互幫互助。你經常思想過去受過的幫助，就是開始邁出感恩的一步了。生活中存在這樣的現象，就是在一群人有難的時候，有一個陌生人伸出援手幫助了他們。但事後，大多數人是會直接走掉的，只有小部分人可能會說聲謝謝。很明顯，大部分受助者缺少感恩的心。但是，這不是他們忘說一句感謝的事情，要知道，缺乏感恩，很可能會讓下一個人失去幫助！雖然，好心的人不少，無私的幫助也挺多的，但正因如此，更要格外珍惜啊！

有人會這樣想，我自己已經很強了，我不需要別人的幫助，而我自己所擁有的一切，也都是憑著我自己的本事賺來的，所以，我不需要感恩。這樣的想法是不對的，靜下心來想想，再強的自己，仍需要感恩。很簡單，我們能活著，就要感恩！如果不是上帝賜予我們新鮮的空氣與燦爛的陽光，我們又怎麼能用自己的本事去做這做那呢？就說地球上吧，有多少國家是多災多難的。然而在我們的國家，大多數地方都還是比較穩定的。想到這裡，誰能說自己是自給自足？所以，身在平安的地方，就要向平安之地發出感恩！

當然，感恩不僅僅是為我們自己，也要為家人，為朋友們感恩，為我們的國家感恩，為我們的一切，向上帝感恩。美好的生活不是從哪個人生紀念日開始的，而是從現在開始！當我們懂得用感恩的心面對生活，我們自然就會由心發出微笑。

除了回味過去的所得的幫助，我們也要對未來有著憧憬。特別是在處於不順的時候，更要有一雙勇於期盼的眼睛。人活在世上，需要有期待的智慧，這樣在面對前方道路時，才不至於抱怨行路的苦累。有一樣東西是越來越少的，但又是極其重要的，就是我們的時間。明白時間的珍貴，我們就會安靜下來，重新思考自己的人生。其實，你對生活有著盼望，你會珍惜時間，也是一種感恩。

不要再浪費寶貴的時間去吵架，發牢騷了，不如做好今天，也為明天做好更充分的準備吧。我們的時間有限，不要虛度任何一刻。與人三觀不同時，不要立刻反駁，思想一下再慢慢討論。也許你認為你的面子對你很重要，但在時間面前，我相信，你會知道，很多你認為重要的東西，原來並沒有那麼重要。懂得感恩，你會謝謝許多的意見不同者。

人生中難免會有苦難，面對困境不能闖過時，人總是會怨天尤人。但是眼睛不能只看環境呀！上天在給你設立關卡的同時，也會讓你擁有闖出關卡的能力，只不過那些經驗，只有你敢闖，才有可能得到的。所以即使處於不順，也多想想走出困境之後的好處，低谷，也同樣會是你進入美好的轉折點。挑戰總會讓一個人成長，也會讓一個人明白自己的不足，從而使他學會謙卑。謙卑了，就會開始感恩嘛。

回憶幫助，展望幫助，我們還要分享幫助的能力。我們不能總是求自己的幫助，卻在生活中從不幫助別人。一個人若只是想索取，那不管多少都不會滿足的，只會使他變得越來越貪得無厭。舉個例子，死海的裡面的海水並不少，但因為它是封閉的，所以裡面的海水從來就沒有出去過，而海水不出去的結果就是讓死海變得又咸又臭，毫無生機。人也是一樣，你擁有的東西再多，如果只是不斷想拿進來，那總有一天慾望會把心塞住的。當心被堵死的那一刻，善良就進不來了，你的善良也別想出去，也出不去了，最後，心也就像死海的味道一樣，又咸又臭。所以一定要在有機會的時候學會施捨，學會分享，這樣，愛和喜樂就會常常在你的裡面。

有人從不相信別人，因為他覺得沒有人會施恩與他。這其實是一種自我封閉，也是讓人經常悶悶不樂的原因。因為你越不肯與人交通，越不想與人分傳遞自己所得的幫助，就越不會快樂。事實上，分享的人比接受幫助的人更得福，因為我們付出去的東西，有時甚至可以給人的生命帶來巨大的改變。當我們看見別人因我們而快樂時，我們的快樂也會是不言而喻的。朋友，不管心中多麼愁煩，不妨走出自我封閉的世界，去向那些需要幫助且你有能力幫助的人伸出一次援手。

人生的意義不在於得到更多，而是在於如何傳遞幫助給那需要的人。懂得付出的人，才能享受到真實的喜樂。懂得付出的人，也一定是感恩的人。
