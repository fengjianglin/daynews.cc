那個為了愛情，一畢業從上下鋪換成雙人床的女孩，如今後悔了

天天要聞
2019-12-25

最近有一個熱播的電視劇【第二次也很美】故事裡的女主角安安，大學一畢業就踏入了婚姻。

嫁給了一個很優秀的老公，事業有成，風度翩翩，可以說是每個女人心目中的理想型了。

安安做了五年的全職太太，優越的生活條件，使她不用體會人間疾苦。

不用像別人一樣，擔心畢業找不到工作，操心每個月幾百一千的房租。

結婚無年完全與社會沒有任何交集，時間長了，一個人在家覺得無聊，希望老公能夠多陪伴自己

結婚之後基本百分之95的家庭，都是女人辭掉工作在家照顧孩子，負責早中晚的飯菜。

收拾家務，男人在外工作，賺錢，養家，一個主內一個主外，看上去好像分工很合理。

往往這樣看上去合情合理的分配，最後居然成了女人威脅女人的一把劍。

從一開始的我養你，變成後來的『連你也是我養的』完全否認了女人在婚姻，家庭里的付出。

女人一旦愛上一個人，就一定會在愛情里迷茫，甚至犯下令人自己今後後悔的錯誤，從此以後女人把家庭，和老公便當作是全世界。

一天不是圍著孩子老公，轉就是圍著廚房轉，忽略了自己的成長，甚至原來的專業都慢慢淡忘了。

對時代的認知還停留在剛結婚時，一心歡喜的覺得，自己身後有一個強大的靠山，為了愛情，丟掉鎧甲，扔掉理想。

你忘了，人是善變的，愛情也是會隨著時間一點點模糊的，有一個人願意愛你，寵你，固然很好。

可是沒有誰能夠成為永遠的依靠，別人給一些東西，但很有可能給到一定程度。

就不想再給了，最終能做自己靠山的還是自己。

女人還是要獨立一些，經濟，思想的獨立，才是一個人能夠立於不敗之地的底牌。

再忙也要記得學習，記得提升自我，只有具體一定的眼界和格局才能選擇生活的方式，

做自己的靠山，才能擁有想要的幸福。

就像安安，本來她可以選擇一種不同的婚姻和生活，完全可以體現自我的價值，有著交大美院漫畫的專業。

而且還得過獎，這樣優秀的女孩子，如果不是選擇一畢業就結婚，她的人生應該是另一番風景。

正因為她選擇了結婚，選擇了一個願意養她的男人，她便喪失了一個成年人應該具體的成熟心態，讓自己活成了巨嬰的狀態。

披著成年人的外衣，但內心和行為是幼稚且缺乏理性的，

面對愛情或者人際關係都有一種不成熟的樣子，只能通過自己的方法，去博得對方的關注，撒嬌，鬧脾氣。

戀愛和婚姻是不一樣的，戀愛時候撒嬌，男生會覺得是可愛，會開心。

如果結婚許多年，相反，男生會覺得反感，甚至是厭惡，所以把握分寸就是一件很重要的事情。

婚姻中兩個人一旦，思維，三觀分道揚鑣，結果往往都不會太好，一個人正在山頂，一個人還在山下，跟不上對方的腳步。

一條平行線越拉越長，最後男方嫌女方什麼都不會，什麼都不懂，女方越來越心寒，彼此失望。

在很多的社交平台上都看到這樣一句話；如果一個男人很愛你，一定會養你。

一些女孩子沒有自我主見，不懂得區分，很大程度地影響了對愛情的價值觀。

當然了，女人都希望找一個能夠讓自己衣食無憂的男人，但是如果女人認為真正的愛。

就是把你圈養起來，讓你的眼界，知識，思維，人生從此定格，那你就大錯特錯。

現實就是現實，還是要保持基本的理智和清醒，如果真的有一天。

你也活成了十指不沾陽春水，兩耳不聞窗事的愛情巨嬰，不要覺得這是什麼好事。

什麼都不做便是做了最大的事，連小朋友到了年紀，都要去上課，學習，培養興趣愛好，什麼琴棋書畫。

反過來，你作為一個成年人，什麼都懶得的去看，去研究，捫心自問，這樣真的不慌嗎？

無論任何時候，任何條件下都千萬別讓自己活成『廢物點心』不要做生活里無厘頭。

這是一種潛移默化的病毒，會導致你失去生存的能力。

不要做金絲鳥，被人關在輪子里，當成一個寵物，如果有更好的寵物，或許結局就是被扔掉。

女人還是要懂得豐富自己，儘管有人養你，但也別讓自己無一技之長，你要有自己的閃光點。

只有你內心世界豐富，有強大的能量，才能不懼任何人的拋棄和離開，才能活得隨心所欲。
