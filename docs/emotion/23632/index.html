一個人靜靜地讀這些句子，便覺得心情慢慢平和！

2019-12-20 396條評論 1,388次閱讀 72人點贊

1、斯人如彩虹，遇上方知有。有趣的靈魂終會相遇，在此之前請先努力成為更好的自己。

2、生活如果不寵你，更要自己善待自己。這一生，風雨兼程，就是為了遇見最好的自己，如此而已。

3、漸漸的知道了，很多東西可遇而不可求，不屬於自己的，何必拼了命去在乎。你在意什麼，什麼就會折磨你。期待是所有心痛的根源。

4、人的一生可能會幹很多蠢事，但最蠢的兩件事就是：拒絕讀書，忽視靈魂；拒絕運動，忽視健康。

5、這世界哪怕再叫人失望，也有一種叫美好的東西在暗地裡生長。

6、「虛驚一場」這四個字是人世間最好的成語，比起什麼興高采烈，五彩繽紛，一帆風順都要美好百倍。

7、很想回到當初聊天你秒回我的時候，每天早上晚上互道安好的時候，你現在回復的特別慢，回復都是幾個字，可能是我說得有點多，又或者我有點煩，又再或者我高估了在你心裡的位置，愛是不是不開口才珍貴。

心語美文集

8、頂著星光，敲打著手中的鍵盤，只想寫下一些與你有關的文字，好緩解我的無限相思。曾經貪戀的舊時光，驀然回首，走過的路一片荒涼。別老想著取悅別人，你越在乎別人，就越卑微。只有取悅自己，並讓別人來取悅你，才會令你更有價值。一輩子不長，記住：對自己好點。

9、後來我不再逢人就掏心窩，我開始計較起付出和回報是不是成正比，在意那個人值不值得，在很多人眼裡我變得很成熟也很自私，好消息是我學會了愛自己，壞消息是我很難再去愛別人。

10、只有等價的交換，才能得到合理的幫助——雖然聽起來很冷，但這是事實。最實在的話：一個男人拚命掙錢卻不注意健康，等於是在給另一個男人打工！一個女人拚命省錢卻不注意修養，是在給另一個女人騰地方。——多麼痛的領悟世上除了生死，其他都是小事。

心語美文集

11、不管遇到了什麼煩心事，都不要自己為難自己；無論今天發生多麼糟糕的事，都不要對生活失望，因為，還有明天。生活就是一場修行，修自己的耐心、恆心、平常心。人生，就是一邊擁有，一邊失去；一邊選擇，一邊放棄。今天陌生的，是昨天熟悉的；現在記住的，是以後淡忘的。

12、我漸漸明白，不是每一個人都像表面上的那麼快樂，有時候一個健談並把你逗得哈哈大笑的人，內心也有傷口和苦痛，我開始知曉，生活是個不管你有多難過它還是會日復一日不斷向前的殘忍機器。我感覺自己還是太幼稚，小小的困難總會影響情緒，一點點挫折也感覺可怕至極，我該學會勇敢和堅強，因為生活如此，弱小就該死。

13、也許現在的你很累，但未來的路還很長，不要忘了當初為何而出發，是什麼讓你堅持到現在，勿忘初心。丟失的自己，只能一點一點撿回來。現在的我不配喊累，因為我一無所有。把委屈和淚水都咽下去，輸不起就不要輸，死不了就站起來，告訴所有看不起你的人：我很好。

14、人生，有很多條路可以選擇。選中一條走下去，便會碰見這條路上的人和風景；選擇另一條，則是完全不同的風景與人。有的人可能與你一起走，有的人留在原地；一起走的人，可能在下個路口與你分開。無需哀嘆，因為都是必然，珍惜願意與你一起看風景的人，下個路口，記得揮揮手。

15、一個人負心，或許是因為他的記憶力不好。他忘記了，所以他能夠負心;不是因為他負心，所以他忘記了。以前種種，他並非完全忘記，但他記憶力太差了，往事已經不再深刻，很快就被新的記憶取代，只記得新人的歡笑，忘記舊人的笑臉。
