我只是在乎你，你卻當我是傻瓜！

2019-11-29 0條評論 21次閱讀 0人點贊

摘要

我是傻瓜嗎？或許我是真的傻，要不是我傻，又怎麼會被你欺騙，一直都在騙我。如果我不傻，又怎麼會一次次的相信你，卻又一次次的失望。如果我不傻，又怎麼會讓你有機會來傷害我。有些事情我不說，不代表我不知道；有些話放在心裡，是害怕你會離去。

我不是真的傻，只是因為在乎你的感受，所以寧願自己傻瓜，當成什麼都不知道，只想好好的與你相愛，一起到老。最後，我發現自己錯了，你卻當我是傻瓜。

我一次次的原諒，你卻一次次的來傷害我。

我一次次的說服自己，你有理由，最後我發現你心裡根本就沒有我。寵著你，愛著你，在乎你，最後變成肆無忌憚。你可知道我心裡也會難受，也會失望，也會心冷的。當我的心冷了，一切都結束了。

有些事情不是不介意，而是更在乎你，所以我選擇沉默，不管不問。但是不要真的把我當成傻瓜，隨便的一個理由就可以糊弄我。

當你一次次的哄弄之後，那也是你失去我的時候；或許你會遇到更好的，但我卻是獨一無二！你若不珍惜，我便轉身離去！
