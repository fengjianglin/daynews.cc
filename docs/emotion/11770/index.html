養娃費用高，姥姥掏錢交培訓班費用，"新型啃老"掏空養老金

2019-12-10 1,862條評論 4,202次閱讀 2344人點贊

我媽最近跟我聊天，提起了她的老閨蜜張阿姨的近況。張阿姨退休前是單位里的副科級幹部，退休工資也不少，按理說退休應該也過得很舒坦才對。可這幾年張阿姨實在開心不起來。

原來張阿姨女兒前年離了婚，於是就帶著孩子搬回了娘家住。張阿姨不僅要日夜照顧女兒和外孫，竟然還要掏錢給外孫報培訓班。這個”新型啃老”真是讓張阿姨又出錢又出力。

幾萬元培訓費，都是姥姥掏錢

孩子讀小學二年級，一門課外培訓費一個學期就要一萬多，語數外三門主課，少了哪一門都不行。

張阿姨的女兒每月只有幾千塊工資，年輕人花錢又不像老一輩那麼節省，隨便給自己和孩子花花，一個月根本攢不下幾百塊。而孩子爸爸那邊，除了按照離婚時的協議，不同意孩子報課外輔導班，也就不肯再多掏一份錢。

可孩子補習班不上，成績跟不上去啊。張阿姨夫妻倆退休能拿一萬多，一對比女兒的情況，只好一咬牙，從自己身上掏錢出來，給外孫報了三門培訓課程。

自此以後，每次要報培訓班出錢時，張阿姨的女兒也不再找前夫開口要錢了，反正也是要不來的，這個開支就直接落到了張阿姨身上。

張阿姨掏錢時，想著是為了孩子好，二話沒說就掏了。可事後一想，不對啊，這養孩子的主要責任，怎麼都落到自己身上了？這女兒和前女婿就像沒事人兒一樣的。雖說自己養老金不少，可女兒和外孫日常吃用都在這邊，現在培訓班的費用也是自己出，再多養老金也吃不消啊。

時間久了，張阿姨心裡難受極了，到處找老姐妹訴苦，可又心疼女兒，弄得自己左右為難。

“新型啃老”讓老人出錢出力

生活里我們對許多”巨嬰””媽寶”都非常不屑一顧，認為這麼大人了，還不能擔負起自己的人生，實在是一件很丟人的事情。

可隨著下一代的出現，很多年輕人沒時間、精力自己帶孩子，就請老人過來幫忙。許多老人心疼子女，看著子女賺錢不容易，不僅出力氣幫忙帶孩子，還會出錢倒貼，這就慢慢演變成了一種”新型啃老”的方式。

“新型啃老”方式，不太容易被人察覺，總覺得似乎是天經地義的。然而，正是因為年輕父母的忽視，導致許多老人很傷心。

  • “都是你們當初讓生的”

有個朋友剛結婚時很年輕，夫妻倆還沒玩夠，一直不想生孩子，可架不住家裡老人催得緊。

朋友的親媽和婆婆都說”趁我們年輕，你們趕緊生了我們好幫忙帶”。

沒想到這句話真成了朋友的”尚方寶劍”，等孩子生出來以後，就覺得帶孩子的任務是老人的，自己只是幫老人”實現願望”。

實際上，不管老人催得多厲害，最終決定生育孩子的始終是自己，養育孩子也是父母天經地義的責任，而老人幫忙帶孩子，始終是情分，絕不是本分之事。

  • “賺錢那麼難，退休金拿來用一用也是沒辦法的事”

說起來很慚愧，我自己的父母在養孩子這件事上，也是貼了不少。用我爸的話說，”現在年輕人賺錢很難，該花錢的地方又很多，靠你們自己想養好一個孩子？做夢吧。”

雖然我知道親爸說這番話是好心，不想自己女兒辛苦，甘願分擔壓力。但許多年輕人，也同樣是抱著這樣的認知，認為老人用退休金幫自己養孩子，是天經地義的事情。

面對”新型啃老”，我們該何去何從？

其實大多數老人幫年輕人帶孩子，無論是出了錢還是出了力，就代表了他們內心還是願意幫忙的。可問題就在於，這個忙幫得是不是夠舒心。

最讓人頭大的情況，就是又出錢又出力，還換不來兒女們一句好話。

現在許多年輕人會埋怨老一輩帶娃方式落伍，認為過去的育兒方式不科學。有的態度還尚屬良好，能夠耐心和老人溝通，有的乾脆就甩臉色給老人看，這叫老人心裡怎麼好想呢？

“新型啃老”里有許多無奈，有時代節奏太快，年輕人必須快馬加鞭去追趕，有經濟負擔確實大，需要老年人伸出一點援手。

可養孩子必定不再是上一輩人的責任和義務，老人完全有權利說”不”。

之所以他們沒有把這個”不”字說出口，無非仍然是對兒女子孫的愛。

  • 丨年輕人要學會將心比心

老人如果願意幫忙，我們當然樂得接受，但要懂得感恩，不要把老年人的付出當做理所當然。

  • 丨接受老人不幫忙

我身邊就有同事，對婆婆帶了三年孩子之後想回老家頗有微詞，為此還和老公大吵了一家。

有的老人實在是累，操勞了一輩子想有自己的晚年生活，年輕人也不應該強求老人去付出。

老人的退休金和時間，原本屬於老人自己的，並不是屬於子女的。老人願意掏錢出力幫忙帶孩子，年輕人也要尊重老人的無私奉獻。

相互理解，才會讓老人的付出有所值，一家人也才其樂融融，孩子才會更健康的成長。

您身邊還有哪些”啃老”形式？
