冬至，我說服了六年沒見的好友，去他那裡吃了頓餃子

2019-12-23 854條評論 3,276次閱讀 1412人點贊

2014年，我們同樣都是因為想要討個媳婦，而相識於服裝廠裡面。

時光匆匆而過，自從那一別。六年沒見過面的好兄弟。通過電話，微信，我不斷的開導，終於鼓足勇氣，肯讓我一起來過這個冬至。

每每想起，我們在一起的快樂時光，就好像昨天發生過得一樣。說話還是那個調調，沒有什麼陌生的距離感。跟我想的一樣，畢竟我們彼此互相了解很深。

說起來也很奇妙，由於家庭的原因所造就，所以他有些內向，更很自卑。而當時的我卻是一個相當輕狂，屬於目空一切的那種人，我們能成為朋友，也是一種很奇妙的緣分。

就如同他姓田，我姓申似的，從進入一個寢室做了簡單的介紹開始，我就發覺，這小子特別對我胃口。而一個大男孩進服裝廠，除了那些技術流的兩口子一起的以為，像我們這樣的大小夥子，其中的目的，當然是不言而喻。

我們一起，上班，下班。趁上廁所的功夫，都會去車間溜達著尋摸美女，搭訕。當然，由於性格的原因都是我拉著他。很多時候也是我對一些女孩品頭論足，而他只會，在那呵呵，傻笑。

不多久，我就找到了一個女朋友。

而他呢，當然還是沒有。不過我還是喜歡拉著他去找美女搭訕，當然帶著我女朋友一起。當初，這樣車間了，每天都會多了一道，兩男一女，三人行的泡妞身影。他當時在乎不在乎我不知道，而我跟女朋友兩個人反正是一點不在乎，還覺得這是一種樂趣呢。

小田呢，也很樂意陪我們一起玩鬧，從他臉上我能看出來，他也是那種很開心的笑。

日復一日，就這樣過了兩三個月。突然有那麼一天，在吃飯的時候，小田滿臉落寞的跟我們說，他準備要離開了。他說，自己從開始進廠就喜歡上了組裡的一個很樸素的女孩。但由於自卑，從沒有表達過心意。可是，那個女孩，在最近幾天跟組長的弟弟確定了男女朋友關係。

聽到這裡，當時我也沒說什麼，那時候我狂到沒邊的性格，只是再想肯定不能這麼算了。至於安慰，壓根都不會安慰人好不好。

沒有過多的話語，我隨便打聽了一下。由於是關係戶，那傢伙性格跟我也差不多。一個字，也是挺狂的。不行，這說什麼也得碰一下子。

沒有跟小田，打招呼，我直接找上他了，明說，我朋友也喜歡這女孩，要麼公平競爭，要麼你直接退出。

當時我廠里廠外的朋友很多，反正也是屬於那種不好惹的吧。他覺得自己優勢明顯，又有自己姐姐再中間撮合，也沒把我話放在心上。抱著玩我們的心態，就同意了。

這才是第一步，關鍵還是小田的心灰意懶。我們都知道，越是自卑的人，越不敢觸碰自己的情感，尤其是跟比自己優秀的男人爭女孩，其中需要提起多大的勇氣，可想而知。

當時的我先讓自己女朋友去開導他。開導來，開導去，小田還是那句話，結了工資，就走了。這給我氣的，我那時候脾氣也爆，聲音都比平時大了不少，就說一句，是兄弟，我絕對不會甘心你什麼都沒做，就這樣離去，哪怕表白，被拒絕了也好，至少你也努力過。至於結果怎樣，不讓以後的自己後悔，就是最好的結果。

離發工資還有一個多星期，小田終於也答應了按我說的辦。當時的我太了解像自己這種年少輕狂的人了，但是改變談何容易。女孩會跟你談戀愛，真要到女生去去考慮託付終生的話，她們肯定會認真再而認真的考慮，再考慮，最後還是不會願意結婚的。我跟當時的女朋友也是沒走到最後，當然，這是後話啦！

在這一個星期里，首先我安排小田，借了個音響，直接去宿舍表白。再我強硬的堅持下，小田，終於到那女孩宿舍里門前，說出來了自己內心的話。

第二天，開始試著約出來，就只是聊聊。小女生還是容易感動的，這個簡單的要求，根本不會考慮太多。

就這樣，託人找人，在女孩耳根里，拿小田雖然窮，不帥，但是他的心真，讓女孩想想，現在組長的弟弟是什麼樣的人，走到最後還不是玩玩算了。

所以說，人都會有自己的考慮，會有所嚮往的愛情。她同意了，真的同意了。下定決心無論如何也要跟小田在一起。

寫到這裡基本概括了屬於他們的故事。其中女孩前男友，不甘心的找我打了一架，就不提了。

如今他們在女孩父母當初不同意的情況下，一起在外。男人養家，女人帶孩子，雖然生活艱辛，一兒一女，也有著不為人所知的幸福。也一起度過了六年的溫馨時光，以後還會繼續下去。

人的情感就是這樣，總想把自己最好的一面展現給朋友親人，不想暴露自己的懦弱。而我想說的是這有什麼，我走進了，你的生活。冬至一起吃頓餃子，不也是很歡樂。

就算我笑笑你，那又怎麼了。我們還年輕，生活總會好起來的。

願世間一切的美好，都不會被世俗所打擾！
