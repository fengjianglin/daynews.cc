沒有人天生就懂得控制情緒，智者時刻留意，不要讓我栽在壞情緒中

天天要聞
2019-12-15

別以為，你的全心幫助，你的傾盡全部，你的真誠以待，就能換來等同的回饋。這世上，總有些不知好歹的人，傷了你的情，忘了你的恩，負了你的願，寒了你的心。永遠記住，要把諾言，留給誠信的人。要把在乎，留給重情的人。要把坦誠，留給忠厚的人。要把忠義，留給交心的人。要把善良，留給懂事的人。要把真心，留給珍惜的人。別慣壞了，不知領情的人；別餵飽了，不懂感恩的心。

雞叫了天會亮，雞不叫天也會亮，天亮不亮，不是雞說了算，關鍵是誰醒了。醒來的，過了一天，沒醒的，過了一生。

有個人買了一箱梨，天氣熱怕梨壞了可惜，每天挑幾個最差的吃掉，最後卻吃了一箱爛梨。總結了一下，作了幅對聯：

上聯：放著好的吃爛的。下聯：吃了爛的爛好的。橫批：永遠吃爛的。

得到一個感悟：人生亦如吃梨，因為在意每天不開心的事，一輩子都得糟心下去；把糟心的事放下扔掉，每天陽光一點，你就燦爛一輩子！ 珍惜當下，積極快樂！沒有人天生，就懂得控制情緒。真正有智慧的人，是時刻留意，不要讓自己栽在壞情緒中！

家長會上，老師在黑板上做了這四道題：2+2=4； 4+4=8；8+8=16；9+9=20。

家長們紛紛說道：「你算錯了一道。」

老師轉過身來，慢慢地說道：「是的，大家看得很清楚，這道題是算錯了。可是前面我算對了三道題，為什麼沒有人誇獎我，而只是看到我算錯的一道呢！」

老師接著意味深長地說：「家長們，教育的真諦，不在於發現孩子錯誤之處，而是賞識他們，做得對的地方！」

孩子如此，成人亦是如此，共勉之！

做人也是這樣，你對他百次好，也許他忘記了；一次不順心，也許會抹殺所有！朋友之間，此類事最多！這就是100－1＝0的人性道理 。其實，家裡親人之間，又何嘗不是常常犯同樣的錯呢？

提此話題，就是為了讓友誼長存，親情永在！朋友之間退一步為上，親人之間讓一步為高。受過人滴水之恩，理應湧泉回報。這才是做人的道理！

可在這裡，有一個問題，還是困擾著我。為什麼這個例子是對三錯一？如果是對一錯三呢？我們來看看，這會怎麼樣？反例：

家長會上，老師在黑板上做了這四道題：1+1=3；2+2=5；3+3=7；4+4=8。

家長們會怎樣？難道是說：「你只算對了一道題」？

那麼老實接下來該怎麼回答？老師轉過身來，慢慢地說道：「是的，大家看得很清楚，這道題是算對了。可是前面我算錯了三道題，為什麼沒有人批評我，而只是看到我算對的一道呢？」

老師接著還能意味深長地說：「家長們，教育的真諦，不在於發現孩子正確之處，而是指正他們，做得錯的地方」？

孩子如此，成人亦是如此，共勉之！

我想一定不是這樣的，問題不在於對錯，而在於多寡。所以很多人，拿這個例子教育人，我覺得很是偏頗，有太大的漏洞，經不住推敲。那麼，用這個道理來表達這個例子，我想有些話就應該換個角度了。我們繼續來看。

家長會上，老師在黑板上做了這四道題：2+2=4； 4+4=8；8+8=16；9+9=20。或者，家長會上，老師在黑板上做了這四道題：1+1=3；2+2=5；3+3=7；4+4=8。

而家長關注的點是什麼呢？「你算錯了一道」，或者「你只算對了一道題」。為什麼？因為人的慣性是關注少的。

此時老師的回答應該是：「是的，大家看得很清楚，這道題是算對（錯）了。可是前面我算對（錯）了三道題，為什麼沒有人誇獎（批評）我，而只是看到我算對（錯）的一道呢？」

而接下來，老師意味深長的說：「家長們，教育的真諦，不是只關注少的，而是多關注他們的優點，放大他們的優點。」

一群人能走到一起不容易，有的強勢，有的隨和，有的厲害，有的溫順，有的計較，有的大度，有的伶俐，有的深沉，有的鋒露，有的狡黠，有的憨厚⋯⋯

人有一百，形形色色。人的相處，沒有天生合適做朋友，需要的是，彼此包容，理解，改變。風風雨雨的磨合，改變著不合適的彼此。尊重他人，莊嚴自己，幫助他人，成就自己！

這是孔子很好的一段話。
