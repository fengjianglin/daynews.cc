家有女兒，爸媽必須要讓她明白這5個道理！（能保命）

天天要聞
2019-12-06

摘要

女孩兒，相比男孩兒而言，總是顯得比較弱勢。尤其是在面臨與男孩兒的紛爭時，最終被打、吃虧的還是女孩兒居多。對此，很多父親都表示：“自從女兒出生後，我的半隻腳就已經跨進了監獄。”是的，面對女兒的柔弱，爸爸作為一個男人只好時刻準備著好好保護她。

女孩兒，相比男孩兒而言，總是顯得比較弱勢。尤其是在面臨與男孩兒的紛爭時，最終被打、吃虧的還是女孩兒居多。對此，很多父親都表示：「自從女兒出生後，我的半隻腳就已經跨進了監獄。」是的，面對女兒的柔弱，爸爸作為一個男人只好時刻準備著好好保護她。

這些日子以來，曦曦媽持續關注著「李心草」事件，看著她跳江之前的視頻，讓我內心極度憤怒。她被兩個成年男子擁抱親吻，還因為不從被扇耳光十多次，面對兩個男子的非禮，她的大學室友不但沒有制止，反而把李心草死死抱住，不讓她反抗。一貫乖巧的她，哪裡承受得了這樣的打擊，在極度崩潰之下選擇了跳江，留下單親媽媽痛徹心扉。

除了持續關注該事件以外，曦曦媽也一直在反思，作為女孩兒的父母，我們究竟應該怎麼做才能最大限度的避免女兒遭遇這樣的不測。我想，以下這五點，是每個家有女兒的父母，都必須牢牢記住的！

（1）在交朋友的問題上面，要具備一定的辨識能力。

俗話說「近朱者赤，近墨者黑」，這句話已經完美的詮釋了學會交朋友是多麼重要的一件事兒。尤其是在孩子青春期之前，父母更要給孩子做好這方面的溝通和引導。

我有一個親戚的孩子，在初中階段學習成績不斷在進步，也正是因為這樣家長便忽略了她的交友圈子等問題。中考時孩子的確如願以償考上了非常不錯的高中，但是由於自己初中階段結識了一些社會不良少年，導致進入高中後老師管理稍微一鬆懈，她便和不良少年廝混在一起。一個月的時間，一個學習成績極好的孩子，就成了眾人唾棄的敗類，從此一蹶不振。

交友是孩子人生中非常重要的一部分，也是非常重要的一種能力。我們應該引導孩子遠離那些品行不端的人，不要把「另類」看成是獨特，從而出現扭曲的崇拜。

當然，在孩子的交友問題上面，家長除了引導以外，也可以做一些適當的干涉。比如，要想孩子遠離一些品行不端的人，我們可以自己多抽時間陪陪孩子，或者製造一些機會，讓孩子與品行端正的孩子一起玩耍，只要逐漸減少了孩子與品行不端的人見面玩耍的機會，孩子才會逐漸遠離他們。

（2）知人知面不知心，女孩兒一定要有警惕心。

雖然我們應該盡量讓女兒感受這個世界的美好，讓女兒熱愛生活、熱愛學習、熱愛一切，但我們也不能在女兒面前否認這個世界的黑暗一面，這樣才能避免女兒在遭遇不測時承受不了。

家有女兒，我們一定要告訴她，我們應該保持一顆警惕的心，這樣才能保護好自己。這個世界，不完全是美好，而人心也有善良和邪惡之分，有些人肉眼可見的都是善良，但其實他就是披著羊皮的狼。

我們可以對這個世界充滿期待，但請在底線的位置放上一句「知人知面不知心」。如果有人約你去酒吧，去KTV，或者一些比較封閉的私人空間，自己一定要多一個心眼，先看看都有哪些人要去，是不是都是自己認識的、熟知的朋友。廣交朋友可以，但請把自己的生命安全等問題放在首位，不盲目地交朋友，不把自己置身於危險的境地之中。

（3）機智一點，發現不對勁趕緊想辦法求助。

如果遇到危險怎麼辦？從容一點，機智一點，一定要想辦法求助，而不是陷入一種崩潰的境地久久不能自拔。

最近幾天，蛇蠍美人勞榮枝被抓的新聞鋪天蓋地的傳來。他與法子英的過往，又一次被翻出來。人們開始總結，如果殷建華的妻子劉敏不找借口去樓下拿錢，恐怕她也會被法子英滅口。機智地她早已讓朋友報警，在自己借口到樓下拿錢之後，法子英被警方圍攻，最終被抓獲。

當自己遇到危險怎麼辦？一定不要靠自己一個人的力量和對方進行抵抗，尤其是在公眾場所，我們要懂得合理利用資源，機智的向別人求助或者找到機會報警，這樣才能讓自己儘快脫離危險。

（4）學會拒絕，不要做一個違背自己良心的老好人。

很多家長，從小就給孩子灌輸，你要懂得分享，自己的東西只要別人需要你就要分享給別人，不然你就不是一個好孩子。父母請記住，教會孩子分享固然重要，但請別逼著孩子分享，我們應該讓孩子體味到分享的快樂，而不是在苛刻中讓孩子領悟分享的痛苦。

如果長期被家長逼著去分享自己的某些東西，久而久之，孩子就會認為自己的東西不給別人分享，自己就有一種對不起所有人的罪惡感；面對別人的邀約，如果自己委婉拒絕了，這樣就顯得自己很不友好；面對別人的灌酒，如果自己明著說「不」，這樣就會打破和諧的氣氛……

教會孩子拒絕，不要總是讓孩子做一個違背自己良心的老好人，這樣孩子才會活得更加精彩，才不至於每天被一些細小瑣碎的事情而折磨到不得安寧。

（5）除了生死，其他都是小事，一定要珍惜生命。

生命只有一次，這是每個家長都必須要時時刻刻警惕孩子的事情。男孩天生好動、頑皮，神經也比較大條；而女孩兒就不一樣，她們天生就比較敏感、細膩，所以更容易在一些小事情上面鑽牛角尖。面對這樣的情況，父母一定要給女兒灌輸——除了生死，其他都是小事。

一方面，作為父母我們要小心翼翼的呵護女兒的自尊心，但另一方面，我們也應該通過一些途徑來增強女兒的抗挫能力，這樣才能把女兒培養成為一個越挫越勇的孩子，才不至於因為一點點小事兒就自殘、輕生。

珍愛生命，這是每個家長都必須給孩子上好的一門課程。我們既要珍惜自己的生命，也不要做剝奪別人生命和危害別人生命的罪魁禍首。即便發生了什麼不好的事情，我們也要理智一點，一定要用正規途徑將壞人懲治。
