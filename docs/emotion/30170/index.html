這17個問題，從沒有人思考，你懂了幾條？

2019-12-25 214條評論 1,214次閱讀 250人點贊

​【原創 丫丫/文】

最近在反覆觀看一個靈性的片子，思考了很多，但是還沒有想好要如何給大家分享。

但這部片子里有很多話我深思了很久，今天把它們分享出來，希望我們可以一起思考和感受。

今天我們不用理性去分析對錯、方法好壞，我們就用最真實的內心，說說你看完這些文字後的真實感受，然後留言給我。

1、作為人類，你最聰明的時期便是當你是小孩的時候。

他們是如此的無限制，享受。他們生活在當下，充滿喜悅。他們可不會去想十年後發生什麼，甚至十分鐘之後會發生什麼。

他們專註於當下，充滿喜悅。而這就是關鍵，你們都可以一直是「小孩」。

2、我們知道，金錢是你們星球上富足的象徵，這都很好，你可以想擁有多少就有多少，但是重點在於，你需要開放它的來源的觀念。

因為當你不再局限地認為富足只能以某些特定的方式到來後，富足便能從四面八方而來，並在你需要的時候去滿足你的需要。

有時候它們可能來自贈送，當有人給予你時，這也是富足到來的一種方式，因為你在創造一個實相，你以開放的態度認可這樣一個從未有過的觀念：即使不經過某些特殊的努力，你也可以被給予某種東西。

3、大眾會告訴你：如果你想得到某種成功的話，你需要不斷地實踐和總結，我們告訴你這並非真相，你需要做的是調整你的振動頻率。

比如，你們生活的星球上對於錢的許多觀念相互矛盾，如，擁有很多錢是不好的；除非你有很多錢，否則你便沒有成功。為何你們會同時持有這兩種觀念，卻又不覺得困惑呢？！

4、因為限制本身決定了信仰體系和潛意識，如果存在限制，你就可以去改變它。

因為每個人與生俱來就都值得（體驗富足），遲早會體驗到。這只是覺察、理解、信仰的問題。

5、很多人認為：除非我擁有很多錢，否則我不能叫成功。但同時堅信：如果我有很多錢，我會變成一個貪婪的人、壞蛋。

這些人不可能讓自己在物質方面變得富裕。因為對他們來說，做一個好人比做一個壞人更重要。他們將擁有很多錢同壞人等同在一起。

因此這是你們需要審視的觀念。因為劃這樣的等號是沒必要的。（作用的）想法不是「你需要變得越來越富足」，而是你認為自己在哪方面是富足的。如果你（認為）自己富有運氣，你就能體驗好運氣。

6、局限認識的人和「靈魂」之間存在著分歧。

你的靈魂並不關心任何輸贏，這毫無關係，你的靈魂不關心你是否付了房租，也不關心你是否是百萬富翁，這都無關緊要，這些只是人的顯意識所關心的。

你的靈魂所在意的是意識演化，即覺察力。你的靈魂關心的是你存在的喜悅，去體驗你的當下。

7、我們會告訴你一些創新性的想法，你是情緒性振動體。

因此不管你的情緒狀態如何，（宇宙）都會回應你對應的能量。因為物質現實是你情緒能量的映射，是你的振動的反映。

8、不管你給出的是什麼能量，不管你給自己定義的是什麼形象，不管你如何定義你自己，它們都會反映到你的物質現實中。

因此你可以去決定是保留這些觀念或頻率，還是改變它們。

9、絕大多數的人們，都將自己寶貴的創造能量專註於不重要的東西上。

10、從整體的視角來看，最有效地轉變你的現實的方法，是有意識地覺察、創造你到目前為止體驗的現實的觀念。

去找出這個觀念是什麼，認清它，會讓你有意識地覺察是什麼在創造你的「實相」。

一旦你找到了最初的觀念或信仰，你便可以了解到它的作用的機制，即它是如何創造你的頻率，進而反映到你體驗的現實之中的。

而當你了解了運作機制，你便可以去選擇你想要的最初觀念，你也可以去期待新的信仰或觀念決定的你的現實，就如舊的觀念所作的一樣。

一旦你替換了新的觀念，接著會產生新的情緒體驗、新的觀念和新的行為，之後你就會得到它們反映出來的新的現實。

11、你是充滿力量的創造者。

你創造了你所體驗的一切，但你可以選擇認為自己只是外在情形的受害者，命運不公什麼的，如是自己父母虐待的受害者之類。

我們知道有些人不想死於疾病或者車禍什麼的，我們說過，每個「死亡」都是一種「自殺」，任何發生在你身上的事都源自振動。

除非你的參與，事情不會發生在你身上。

換句話說，如果你持續恐懼自己會死於車禍，那你可能就會死於車禍。如果你持續地認為你的身體會得病或保持健康（你就會得到對應的狀況）。正如研究員發現問題、解決問題，你也可以保持完美的健康。

12、你存在的本性是自由，你們都有自由意志，因此你是自由的。

我們發現很多人並不同意這個觀點，因為他們覺得情況不是這樣。但你們都是自由的，可以去做任何你想做的，可以去成為你想成為的人，自由去擁有你想要的。

13、儘管你們很多人選擇給自己加枷鎖、限制自己的自由，這也是自由選擇，你就是自由本身。

14、記住這一點：成長來自於你所有面向的集合，而非對任何面向的排斥，它是關於對一切的接納、對平衡的接納。

這也就是為何當你越是提升，你會對更多的黑暗變得更加覺察，因為已選擇接納所有，並在內在取得平衡。

15、你所有的力量都在當下。

所以過去你做過什麼都不重要，重要的是從此刻你將走向哪裡。而現在你就有去往任何方向的力量，就在此時此刻。

16、不必去擔心和恐懼周圍發生了什麼，只需專註於你自己的世界。

17、盡情玩，去愛，去生活，對新想法和靈感保持開放。

​

我知道我分享了這些句子是有些冒險的，但每當看到這些句子，總會讓我想起很多東西，很多活法上的、很多選擇上的。

我想知道你看到這些內容時，是一種怎樣的感受呢？
