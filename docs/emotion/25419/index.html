70老母總擔心40歲兒子以後沒人照顧

2019-12-20 313條評論 1,393次閱讀 58人點贊

#情感##心理那點事兒#

案例：貴芳（化名）,女，69歲,理髮店老闆。主要問題：總擔憂40歲的兒子以後沒人照顧，憂鬱、不放心。

貴芳幼年就失去父母雙親,留下她孤苦伶仃。長大後跟丈夫結婚不久懷孕,可丈夫到外地就一去不返，從此沒有消息。

貴芳靠理髮手藝養活自己和兒子容生。貴芳很喜愛自己唯一的親生骨肉，但為了謀生,白天只能將容生托放在託兒所,自己早出晚歸地工作。到了夜晚,就把孩子接回家好吃好喝照顧，周末還帶他去買玩具,彌補對兒子的歉意。儘管貴芳待兒子如此好,容生還常不滿意,說母親沒有時間照顧他。

容生慢慢長大，從不幫母親做家務，還常要錢,到外面去玩、揮霍。只要容生一鬧要錢，貴芳就趕快從辛苦存的一點儲蓄里取錢給他用。總覺得孩子沒有爸,自己又沒有時間照顧,稍微寵點惟一的寶貝也無仿。

容生高中畢業後沒考上大學，也沒有想法去找事情做，這樣一年又一年靠母親供養他。不知不覺容生已是近40歲的中年人,貴芳也是快七十的老人。貴芳幾年前做了理髮店老闆,還雇請了兩三個人,她依然起早貪黑地辛苦工作。年紀大了有些吃不消，但還堅持著。只是總遺憾兒子變得沒有出息,擔心將來怎麼辦。兒子從前年得肺病,常看醫生吃藥,貴芳更是擔心。她想法設法給兒子滋補身體，兒子不但不領情,還責怪她從小沒好好照顧他身體才會這樣。貴芳聽了很難過。

幾個月前,容生交了個女友,30多歲,離異有小孩。貴芳不太贊成，勸告容生。容生不但不聽,還罵母親吃醋,反對他交女友。容生向母親發脾氣,還乘機向母親威脅,要一大筆錢，準備搬出去跟女友一起生活。貴芳解釋自己的一點儲蓄只夠自己養老，容生一氣之下跑出家好幾天沒回來。

為這貴芳很傷心,夜裡睡不好,白天沒心情做事。一想到兒子這樣沒出息,對母親這樣不好,就流淚哭泣,被店裡的人發覺情況不好,極力勸導來看心理醫生。

分析：貴芳從小喪失父母,後又沒有了丈夫,只有一個兒子,就把整個精神都投注在兒子容生身上,再加上沒時間照顧兒子而來的歉意,於是更加想辦法寵愛,結果孩子需要養成獨立負責的機會都減少了。

兒子容生早已不是需要照顧的「小」孩子,是已經中年的成人,自己可以照顧自己,對自己的生活負責。可是容生利用母親心理上的弱點,把住母親的心,還繼續利用和虐待母親。

經過一段時間的諮詢，貴芳的情緒慢慢好起來。但是一旦容生來找母親要錢什麼的，貴芳的情緒就又會波動，她數十年來繼續的對兒子虧欠的情結，確實難於一時更改。
