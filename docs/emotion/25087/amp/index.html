二十歲和三十歲最大的區別是什麼？

天天要聞
2019-12-22

青春不是年華，而是心境。

二十弱冠，很純粹，愛做夢，感覺自己是特殊的，可以拯救世界。感情上覺得自己不需要愛情，因為這個世界上沒有人能配得上自己。

三十而立，很現實，愛上火，感覺自己太難了，拯救世界這件事，還是交給年輕人吧。感情上覺得自己要孤獨終老，就指望著哪天國家能分配對象了。

我們今天來討論的話題是：二十歲和三十歲最大的區別是什麼？

有人說，女人比男人早熟，會隨著時間推移越發的展現出知性美，而男人也會在社會的歷練下逐步走向成熟。

要問這十年的光陰，究竟是男人變化大還是女人變化大。我想，變化不論大小，在對的時間，遇到對的人，是一生的幸福；錯的時間，遇見對的人，是一生的嘆息。因為時間不僅讓你看透別人，也讓你看清自己。

20歲，一般還在讀大學，校園的愛情是很單純的，沒有什麼壓力，就算交過幾個異性朋友，但對愛情懵懵懂懂，不知道自己想要的是什麼，只需要管家裡要錢，每天開心就行，不需要考慮經濟問題。

30歲，大多已經在社會上磨礪了好些年，知道自己想要什麼樣的生活，什麼樣的愛情。這個年紀的愛情是理性的，多了很多顧慮，會將家庭因素考慮其中，明白有些人適合戀愛但是不適合結婚。

20歲，心思單純，沒有工作經驗，卻總是一副天不怕地不怕的樣子，在溝通上不夠圓滑，容易和身邊人發生口角，遇到困難就想著找人傾訴，尋求安慰，害怕一個人承擔。

30歲，內心成熟，工作經驗豐富，有了屬於自己的生存方式，職場上雖然表面上畢恭畢敬，但內心仍是當初的桀驁不馴。已經能夠自己一個人獨當一面，遇事果斷處理。即便遇到困難，也會選擇獨自面對黑夜，內心會因此越加強大。

20歲，年輕活力，對未來充滿期待，理想在內心沸騰，總是抱著闖一闖的心態，對萬事萬物都懷著滿滿的好奇心，很容易被誘惑和欺騙，也從來沒有畏懼之心，活在自己的世界中。

30歲，閱歷豐富，也會對未來充滿期待，但這種期待是現實的，不像以前那樣天馬行空，不切實際。十年間經歷了社會紛繁複雜的洗禮，做什麼都謹小慎微，對世界也時刻保持警惕。

20歲，會將自己想像成白雪公主或白馬王子，男孩子把汽水罐的拉環當作戒指，套在女孩子的無名指上，女孩子會覺得非常浪漫，感動良久。看到小三上位、出軌、背叛的事情都義憤填膺，恨不得上去打別人兩巴掌，世界非黑即白。

30歲，人們會專註實際，更喜歡收到的禮物是紅包，關注更多的，也是子女教育和實際工作。對於出軌、偷情等多元化的世界，理解每一種現象背後都有原因，懂得理解和寬容每一個人的同時堅持自己的原則不動搖。

20歲的時候，一般會看不起30歲的人，認為30歲的人太老；而到了30歲，又會看不起20歲的人，認為20歲的人沒有大腦。

人生苦短，韶華易逝，往昔有太多的感動，有太多的追憶。經歷了風雨的洗禮，歲月的磨礪，即便不再有年少時的青春爛漫，也不再有年輕時的蓬勃朝氣，但卻貯就了一副成熟和穩重，多了一份經世的閱歷。

年輕的時候，我們嚮往流浪，以為到一個地方漂泊，便是旅行的意義。後來才明白，原來流浪不是身體沒有目的的行走。而是心裡，沒有一個讓你停留下來的人。
