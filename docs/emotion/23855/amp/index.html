人，切莫妄自菲薄，狂妄自大，早安

天天要聞
2019-12-18

天上人間，人人都是天地的過客，生命無論長短，都不過是一場相聚別離。在光陰的更鼓聲中，我們都各自擁有著自己的月影雲煙，過度的清醒會令人感覺生命無趣，過度的沉醉會令人變得迷失心智。若能把凡塵的煙火解讀成一半真實一半夢幻，把人間的流嵐寫意成一半詩意一半璀璨，那樣的人生方可謂快活愜意的人生。早安！


試問，人生在世，有多少男人富甲一方？有多少女人傾國傾城？芸芸眾生，還不是尋常平凡者居多？每個出塵入世的人，不管他是光芒萬丈，還是黯淡無光，不管他是逍遙一生，還是負累一生，最後還不都是歸於塵，歸於土，歸於空無？在光影交錯里，你能挽得住眼前，又怎能挽得住永遠？在時光荏苒中，你能留得住雲的身體，又怎能留得住風的腳步？ 早安！


人生來去，有人喜歡閑雲野鶴，行蹤不定；有人喜歡乘風破浪，拼搏一生；有人喜歡觥籌交錯，沉浮迷離；有人喜歡虛度年華，苟且偷生。其實，人生是一首詩，也是一闕詞，唯有讀懂者才能算真正的活過，唯有領悟者方能真正過得有意義。不管你選擇過怎樣的人生，一切冷暖都只是自知，一切幸或不幸都只在一念。 早安！


紅塵里，多少等待終無期，多少花開終成泥。人與人，一生來去無非是或相互欣賞，或相互扶持，或冤冤相報，或相忘於江湖。緣來緣去，若得他人無情無義，那麼你的人生便會綠肥紅瘦，從青綠到枯黃，生命的原風景將不再有溫暖的底色，你的生活將永遠有陰霾籠罩，痛苦相隨；若得他人真心真情，那麼你的人生便能變成你喜歡的模樣，從此，你心有桃源，四季花開，你的生活將永遠有幸福縈繞，花香滿衣。 早安！


人生修行，難免有名利困擾，有風雨來襲，挺住了，你便能經歷柳暗花明，走向風和日麗，鶯歌燕舞；看開了，你便能落座人間四月天，擁抱歲月蔥蘢，季節芳菲；放下了，你便能臨一方旖旎雲水，守得清幽凈土，如畫江山。一生修行，你若留得一半清醒一半醉，尋常光陰也能被你過得活色生香，平淡人生也能被你過得精彩絕倫。 早安！


風雨人生，山一程，水一程，有些苦我們必須要嘗；有些痛我們必須要藏，有些怨我們必須要忘。也許走進孤獨，我們方能體念安靜處的非凡絕響；也許走進黑夜，我們方能感受闌珊處的絕佳光芒。紅塵有夢，歲月有愛，行走天地間，我們當心有明鏡，心有琴弦，看一路風景，留一路歌聲。
早安！


不管你和他是否有魚和飛鳥間的距離，不管你和他是否有冰與火的兩重天，不管你和他是否有白天與黑夜的不相見，一切都應緣來相惜，緣去相送。緣來時，你與他可來一場春風十里的邀約，緣去時，你與他可行一場萬水千山的歡送。相聚時，共一份美好，好好走一程，讓一路笑語串訂成冊；離別後，共一段回憶，淡淡念一生，讓曾經的留香摺疊入夢。 早安！


紅塵相遇，莫問是劫是緣；紅塵別離，莫問是禍是福。焦躁者定會負累一生，淡定者必能優雅老去，願只願，咫尺相對時，彼此永遠有相視一笑，心有懂得的靈犀；天各一方時，彼此永遠有你若安好，便是晴天的祝福。早安！


人生如戲，生旦凈末丑各有所長，各有所短。人，切莫妄自菲薄，狂妄自大，我們要懂得，生命不是讓我們辜負的，時間不是讓我們虛度的。當你選擇遠航，也許你就能領略大海；當你選擇飛翔，也許你就能擁抱藍天；當你選擇仗劍，也許你就能笑傲江湖；當你選擇挑戰，也許你就能演繹傳奇。早安！


山有所高，才令人敬畏；水有所韻，才令人流連；人有所美，才令人欣賞。我們要明白，最美的風景其實不是在路上，而是在我們心中。若我們一生能以慈悲為念，以懂得為安，看清是非曲直，看淡貧富榮辱，永遠記得快樂和憂傷都是生命的匆匆過場，不忘初心，方得始終，那就不枉人世走一回了。「青山不墨千年畫，流水無弦萬古琴」，紅塵，當留一半清醒留一半醉。早安！
