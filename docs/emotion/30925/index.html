“不會讀書的孩子是在給父母報恩”，校長的一句話，引發父母深思

2019-12-24 1,332條評論 4,107次閱讀 1488人點贊

「本文由雪莉媽媽原創，歡迎大家分享」

前幾日朋友圈火了這樣一段話，是一位校長略帶幽默的40分鐘講話，講話內容的中心是「不會讀書的孩子是來報恩的」。

這樣的中心內容是不是讓很多父母都摸不著頭腦，父母們都希望孩子能夠好好讀書，這樣長大才會有出息，為什麼不會讀書的孩子卻是來給父母報恩的呢？

在校長的話語中進行了解釋，原來讀書好的孩子長大後的確會有好的發展，可能會出國留學，和父母見面的機會少之又少，每天只能通過視頻見面。

而讀書略差的孩子，卻可以在父母老之後常陪左右，帶父母去吃各種好吃的，去各地玩兒。

很多父母都羨慕別人家的孩子學習好，成績優異，殊不知自己才是最幸福的人。

自己在年老之後，會有孩子常伴左右，能夠享受天倫之樂，不必通過冷冰冰的屏幕，才能看到自己的孩子。這樣看來，不會讀書的孩子的確是來給父母報恩。

父母與其羨慕別人家的孩子學習好成績優異，不如多花時間對自己的孩子用心教育，多陪伴孩子。

不要強求孩子做任何他不喜歡的事情，和孩子共同的成長，便會讓孩子更加優秀。

作為父母的你，是否早已忘記養孩子的初衷呢？

從孩子降臨在家庭中的那一刻，每一位父母都希望把自己最好的愛給予孩子，陪著孩子呀呀學語，看著孩子從爬到走到跑的過程，那時候父母們的初衷是孩子快樂長大就好。

但隨著時間的推移父母對於孩子的要求慢慢的變多了，因為在父母的心中不希望孩子輸在起跑線。

從孩子上幼兒園開始，就讓孩子學習各種才藝，上了學之後，不僅讓孩子在學校學知識，放學以及周末又會帶孩子去補課。

此時父母只希望孩子變得優秀，認為對孩子的這些要求，也是對孩子最好的愛。

也許父母已經忘記自己的初衷，讓孩子快樂長大。

四年級孩子的作文，不禁讓人深思。

之前有一位網友分享了四年級小侄子的一篇作文，作文中有這樣一句話，觸動人心。「其實我的夢想很簡單，就是有一個自由自在的夜晚」。

一個四年級的小孩，為什麼會將「自由自在」當成自己的夢想呢？

原來男孩的父母，總是要求孩子做這個做那個，不僅是要求孩子學習，也要求孩子按時上床睡覺，按時洗澡等等。

父母的要求已經成了孩子的束縛，孩子只希望放鬆一下，只希望有一個自由自在的夜晚。

不得不說現在的孩子真的太累了，他們已經失去了自我，在父母設定好的路程上拚命的奔跑，因為一旦停歇便會換來父母的失望。

父母打著為孩子好的旗號，給孩子制定好了各種各樣的目標，強迫孩子做各種各樣的事情，這種為孩子好真的會換來孩子的開心嗎？

也許每一位父母都曾經歷被自己的父母強迫學習，那時候內心的想法同樣是抵觸吧。

為什麼現在又要將這種強迫給予自己的孩子呢？還是需要變得優秀。

但這種優秀並不應該「揠苗助長」，強迫使孩子優秀，換來的很有可能是逆反。

在孩子的成長過程中，父母需要做的是順其自然，而不是強求強迫。

父母不應該給孩子制定好成長的路線，這樣的成長不會有任何的收穫不會有任何的驚喜，讓孩子在未知的路程上進行探索，成長，才能夠讓孩子變得更優秀。

各位父母請不要催著孩子長大，每一個孩子都是不一樣的煙火，讓他們盡情綻放自己的美麗吧。
