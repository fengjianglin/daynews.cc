中國式父母表達愛意難上加難：愛的表達方式很重要

2019-12-14 2,115條評論 3,484次閱讀 632人點贊

都說母愛如水，父愛如山，但是在家庭里如果真正把愛提到檯面上來說，卻又會變得很尷尬。不善言辭的中國式父母，他們不懂得如何去愛。

有的父母認為打罵、縱容、滿足孩子的物質需求是愛孩子，也有的為了能夠管住孩子時刻保持著非常嚴厲的狀態，這種環境生活的孩子有時候會懷疑自己的父母是否真的愛自己。

不善表達愛的父母，養育出的孩子同樣也不知道如何表達自己對父母的愛，他們都習慣性的把愛隱藏起來。

兒童心理學家說：懂得表達愛的孩子陽光自信，充滿感恩。他們心中充滿著愛與被愛的體驗，不管在什麼情況下，都會清楚地表達自己內心的需要，這是成長道理上最重要的能力之一。

要讓孩子懂得表達愛，父母就要毫不掩藏的把你對孩子的愛大膽的表達出來。

1

在《放學後》的一期節目中，家長代表是一位不愛笑的運動員爸爸。爸爸李亞明曾是國家級摔跤運動員，他把對摔跤的熱愛寄托在兒子糖豆身上。

在教練和爸爸雙重角色切換下，糖豆並不是那麼適應。

在熒幕前，李亞明給觀察員展現的一直是高大威猛、嚴厲的教練形象，而很少有溫柔的畫面。即使糖豆某個動作訓練的很好，父親都是一副不帶笑容的嚴肅面孔，更像一座高大的冰山。

看到嚴厲的爸爸，糖豆也會格外緊張。有一次，爸爸給糖豆偷偷準備了一個生日蛋糕，在給糖寶拍照時爸爸終於笑了，糖豆這樣形容爸爸的笑：”很嚇人”。或許爸爸很少笑的原因。

那麼，爸爸過於嚴厲對孩子會有什麼影響呢？成長教育專家蘭海說：孩子聽到的都是命令語氣，會變得沒有主見，長此以往，孩子的安全也會被剝奪。

節目中，當爸爸問糖豆為什麼不愛自己時，糖豆說：”因為你只給我買過禮物。”隨後，爸爸列舉了買給糖豆的禮物清單，以證明自己時愛孩子的。

爸爸的回答引起了觀察員的好奇，他們一致認為如果要舉例子表明自己愛孩子，不應該是給孩子買了多少東西，而是時間的陪伴。

有些父母認為我給孩子買了那麼東西就是愛孩子，其實這種思維會誤導孩子認為愛就是買東西。

當你想要回答愛是什麼時，你的回答會直接影響孩子對愛的理解。蘭海老師說：”爸爸在像孩子表達愛意時，表情一定要豐富，面對孩子不能一直板著臉，並且與孩子要有身體上的接觸。”

對於父母，愛的表達對孩子的人生非常重要，愛的打開方式有上百種，選擇能夠讓孩子感受到你的愛的方式就是最適合的。

2

《悲傷逆流成河》是一部讓人深思的電影，除了校園暴力問題，還有家庭教育的問題，這些情節讓眾多網友感同身受。

易遙的媽媽是一個不懂得表達自己的人，習慣性喜歡用語言暴力和自己的女兒說話，”三字經”罵人常掛嘴邊，從不考慮女兒的感受。

媽媽不知道易遙在她的長期語言暴力攻擊下，早已遍體鱗傷，甚至根本就不願意和自己的媽媽說話，因為她知道只要一開口就會無端的受到責罵。

影片的開頭易遙身體不舒服，想要得到媽媽的關心和幫助，可是媽媽一副極不耐煩的樣子讓易遙把原本想訴說的事硬生生的給憋了回去。

在自己獨自去醫院檢查身體，拿到診斷結果，想要和媽媽訴說病情時，又被媽媽語言暴力給打敗了。

易遙一度覺得自己是一個不被愛的孩子，她能感受到的家庭溫暖僅僅存在於童年時爸爸媽媽帶去遊樂園的場景。

事實上，從影片里可以看得出媽媽是很愛易遙，只是她把那份愛藏了起來，用冰冷的外表偽裝成一個不愛孩子的媽媽，讓易遙感覺不到母親的愛。

媽媽偷偷為易遙攢學費，易遙提前回家媽媽本能的反應是保護她，在知道易遙是因為自己而生病後抽了自己一巴掌並拉著她去醫院看病。

“走，媽帶你去看病”這一句無數網友感動，這哪裡是一個不愛自己孩子的媽媽呢？

當易遙感覺媽媽是愛自己後，那種掩飾不住的喜悅和自信洋溢在臉上，讓人覺得被愛真好，縱使這樣，也挽救不了易遙悲劇的結局。

每個孩子都是父母捧在手裡的心頭肉，愛孩子就應該讓孩子感受得到，如果孩子感受不到，那就是愛的方式出了問題。

阿德勒說：幸運的人，一生都被童年治癒；不幸的人，一生都在治癒童年。良言一句三冬暖，惡語傷人六月寒，愛孩子，從好好說話開始，每個孩子都應該成為那個”幸運的人”。

3

綜藝節目《一路成年》里有一個很感人的片段。

妹妹梁靜曦有嚴重的恐高症，因為擔女兒，梁家輝原本不打算讓妹妹坐峽谷鞦韆，可妹妹堅持想和姐姐一起去體驗一下，並給了爸爸一個大大的擁抱，讓爸爸放心。

勸不過兩姐妹就只好妥協，不放心的爸爸又仔細檢查了設備，在鞦韆盪出去前親吻了兩姐妹，交代要注意安全，但在鞦韆盪出去後，梁家輝還是忍不住抱著李劍哭得像個孩子。

很多網友表示羨慕這樣的父愛，並有網友評論”這是現實中父愛的樣子”。

鞦韆盪出去後妹妹姐姐妹妹先後表白爸爸。

從節目中，我們可以看出梁家輝父女都特別善於表達愛意，會互相說我愛你，毫不掩藏，而這樣一個愛的明顯的家庭看上去溫馨得讓人羨慕。

而同在節目中的李汶翰家就顯得格外含蓄，兩父子都不善於表達，鞦韆盪出去後老李只對兒子說了一句”加油”。

現實生活中，很多家庭都像李汶翰家一樣不會把”愛”掛在嘴邊，甚至做一些親昵動作都覺得尷尬、不好意思。

但是又有很多人羨慕梁家輝這種有愛、有溫度的家庭。

在蜜罐子里長大的孩子，將來他們的下一代也會生活在蜜罐里；生活方式是會傳染的，有一個會表達情感的父母，就會有一個健康陽光、知冷知熱、善於表達的孩子。

蔡元培先生在《中國人的修養》里說：決定孩子一生的不是學習成績，而是健全的人格修養。想要培養孩子健全的人格，家長首先要做的就是改變說話的語氣和方式。

孩子感受不到你的愛，多半是父母說話語氣和方式中傷了孩子，一個兇巴巴的父母很難養育出會表達自己的孩子。

愛的表達方式很重要，父母要學會正確的表達對孩子的愛，一小時的高質量陪伴、一個擁抱、一個擊掌、一個親昵動作或者一句好聽的話……
