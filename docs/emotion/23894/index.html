今天給我可給我嚇的不輕，打110來開鎖，人家跟沒事人一樣

2019-12-18 1,168條評論 2,475次閱讀 901人點贊

今天大連的天可冷了，因為昨天下了一天的雨，早上8點半多接到婆婆電話，她說病了，要去醫院，我說好的，我一會去接你，沒吃早飯，沒洗臉，穿上衣服就出去了，太冷了，又回來換衣服，就去坐車了。

好婆婆家大約是9點多一點，我有鑰匙，但怎麼都開不開門，按門鈴，打電話咋也打不開，就給她兒我老公打電話，也打不通，就給二姑姐(我老公他二姐)打電話，也打不通，你們說，這急人不，我就想著這人在屋裡咋了呢？嚇人不嚇人，擔心不？這時二姐接通了電話，經過溝通，我打110找開鎖的來的

和人家商量幫助外面殼就這樣打開了門

馬上進屋看看，叫了聲媽，推了一下，好嗎？人家睡著了，我都把門鈴按爆了，把電話打爆，把門踢得那麼想，天啊

因為是反鎖，鎖芯廢了，還得換鎖芯，好嗎，四五百沒了，好吧，老人沒事吃啥都強，是不是應該去醫院了啊，人家不去了，咋說也不去，就是翻翻兒女不好，嗨，這人不是老了才這樣的啊，

好頓安慰，好頓聽，好多了，都過了中午，整點飯吃吧

吃飽了，接著說

一直說到下午3點半多了，心情也好多了，我才離開

感覺人都有老的時候，老了真的不容易，但作為一個母親是不是應該對自己兒女多考慮啊，一個女兒不好，兩個女兒不好，一個兒也不好，三個孩子都不好，難道真是孩子的事嗎？我覺得她的這三個兒女都非常孝順。我這個兒媳婦也就更不用說了吧，怎麼做都不滿意，還…嗨，我不知道我老了會什麼樣

你們家裡的老人也會這樣嗎，真是每一分安寧啊，心累
