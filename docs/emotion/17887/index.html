心理學：無情的母親和強大的女兒

2019-12-17 2,044條評論 2,978次閱讀 1964人點贊

希拉里有很多個身份，美國第67任國務卿，柯林頓總統的妻子，特朗普競選總統時的最大競爭對手。儘管後來她敗給了川普，但也仍不能忽視她是一個強大的女人，是美國女性的典範。

希拉里之所以強大，與小時候她母親的教育脫不了干係。她的母親對她的愛，是一種無情但理性的愛。她透過一個看似無情狠毒的方法來訓練希拉里成才，那是一般母親都不敢用的。

故事從她們搬到新環境說起。某天，希拉里到外面玩耍，她很想結識鄰居的小孩。可是，因為她是新來的陌生人，鄰居小孩不但不接受她，還狠狠的欺負她。希拉莉當時受到很大的打擊，覺得很委屈，便傷心的哭著跑回家找她的母親訴苦。

當她告訴母親時，母親只是點了一下頭，完全沒有給她任何的反應。第二次，她又被欺負。希拉莉再一次哭著跑回家，這一次，她跑到家門前，見到她的母親用整個身體擋著門口，不讓她進去。當希拉莉想從左邊躦進去時，她母親用手擋著左邊，當她想從右邊躦進去時，她母親又用手擋著右邊。

希拉莉感到很奇怪，就問母親：「媽，為什麼不讓我進門？」

她的母親說：「我不要看到我的女兒因為被鄰居欺負就跑回家，以為躲在溫暖的家庭，就可以受到保護。在這個世界上，沒有人可以保護你。假如這些人不接受你，不接受你成為他們的朋友，你就要想盡辦法去使他們接受你；假如他們欺負你，只要不是你的錯，哪怕你是女生，也要和他們打架，而且一定要打贏，直至他們不敢再欺負你為止，這樣才是我的好女兒！」

你可能會想，為什麼女兒在外面受了傷害，她的母親不但不安慰女兒，還把她擋在門外？為什麼會有教唆孩子跟人打架的母親？

天下間哪有這麼無情的母親？

其實這種看似殘酷的愛並不是無情的，她不會同情女兒，因為同情心只會滋生弱者，而父母是沒辦法庇護子女一輩子的。

如果希拉裡面對問題時，覺得很悲哀、傷心、難過，然後就跑回家去找母親，母親又給她關懷的話，她就會覺得很溫暖，不會思考怎樣去獨自面對問題。當她踏入社會時，她的競爭力就會變得越來越弱。這也就好像心地善良的人不斷給錢乞丐，無形中也把他們行乞的習氣強化，使他們有更多的理由可以繼續行乞。

父母的愛有兩種：一種是發自內心、形諸於外的愛；這種偉大的愛，千古傳頌，因為他們可以為愛而犧牲，同時內心充滿了激情熱血，與被愛的人連成一體，無法分割。

第二種愛是無情的愛：這一種父母，對子女的愛不比前者遜色，但是他們將愛的見地和眼光建立在高高的山頂上，將感情隱藏在深深的海底下。他們有時候的行為表現，與內心情感相反，這種愛會導致父母與子女漸行漸遠，被愛的人會因此而完成獨立自主的人格。

這種父母也會像第一種父母那樣和孩子同歡樂、共悲傷，但是當孩子面對痛苦的時候，他的同理心會隱藏起來。他們更希望孩子有獨立處理自己遇到的事情的能力。

注重感受者，會用同性心來愛；有遠見的人，放下同情心來愛。

點個關注，跟我一起，每日趣看心理看八卦，學習心理乾貨。
