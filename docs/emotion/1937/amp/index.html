做好這三個溝通場景的應對，你也可以成為職場溝通高手

摘要

優秀的人，溝通能力都不錯，這7個溝通建議，助你成為溝通高手做好這三個溝通場景的應對，你也可以成為職場溝通高手

我相信在很多人的心理，都會有這樣的困惑：什麼樣的人在職場中會更加出色？

我相信一千個人會有一千種答案：

有的人說，專業能力強的人會更出色；

有的人說，學歷高的人會更出色；

有的人說，有項目經驗的人會更出色。

其實，不管是哪個答案，都對。因為只要能夠幫助到你的職業發展，那你擁有這項能力或者知識，都會讓你變得更加出色。但是，我想再問一個問題：在某個領域非常出色的人，要具備什麼能力，才能成為職場贏家呢？

答案可能只有一個，那就是要具備優秀的溝通表達能力。

也許你的專業能力很強，學歷很高，實戰經驗很豐富，但是你情商很低，溝通表達能力一般，得不到領導、同事、下屬的認可和支持，那你所擁有的一切優勢，可能都將會無用武之地。

我見過太多這樣的場景：

做出一份專業的方案，可是卻遭到各個部門的強力反對；

工作能力很強，可是卻始終無法得到用人部門的認可；

管理者專業素養很高，可是卻無法管理好自己的下屬。

以上種種問題，可能都源於你與別人的溝通表達出了問題。

對於你來說，在職場中，無非就是要面臨三種溝通表達的場景：一種是向上溝通，一種是水平溝通，一種是向下溝通。所以，你的職場溝通表達出現問題，也就是這三種溝通場景出了問題。今天，我給你介紹這三種溝通場景的溝通表達技巧，助你更好地應對這三種溝通場景，讓你成為溝通高手，成為職場明星。

場景一：向上溝通

很多人的直接上級就是老闆。由於老闆是公司一把手，我相信很多人跟老闆溝通都會很謹慎，可是還是有一些人在跟老闆溝通的過程中，犯了一些錯誤。

比如，有一天，公司有一些員工因為工資低離職了。你覺得很有必要向老闆彙報一下。於是，你走進老闆的辦公室，你對老闆說：老闆，最近公司有很多員工離職了。老闆抬頭看看你：哦，那怎麼辦呢？老闆一問怎麼辦，你就傻眼了！你可能會反應很快，說：我覺得可以考慮給他們加點工資。老闆一聽，繼續問：那要怎麼加呢？具體加多少？由於你事前沒有做準備，你只好說：我先回去了解一下情況再說。於是，你就退出了老闆的辦公室。

各位，這是一個失敗的溝通案例。因為老闆的時間很忙的，你找他溝通一次不容易。失敗的人與老闆溝通，是說：那我回去再看看！而成功的人與老闆溝通，則是老闆說了最後一句話：那就按照你說的來做吧！

在上面的這個案例中，這個人與老闆的溝通存在兩個問題：

第一，只帶問題，沒有帶方案

老闆在一些領域，專業知識沒有你多，所以很多老闆是無法給你解決方案的，相反，他需要你給你解決方案。如果你每次溝通，都只是給他彙報問題，那你們之間的溝通肯定是會出問題的。因為你如果只是帶問題進去，那你百分之百不會有結果，你不僅浪費了自己的時間，還浪費了老闆的時間。

第二，沒有想清楚問題就去彙報

有些人，只要出了問題就跟領導彙報，而他自己沒有認真思考。所以領導一問三不知。如果是這種情況，那到時就會很尷尬。你可能不得不說，你再回去想想。

如何克服當眾講話緊張？如何讓你有話可講，面對多人侃侃而談？如何讓你的講話有條理，層次分明？想解決以上問題，可以點擊下方鏈接了解《30天當眾講話高手速成》：

基於以上兩個問題，如果你要做好向上溝通，那我建議你做好以下三點：

第一，把問題先捋清楚

比如，公司的員工最近離職率高，你要先做好分析，員工離職率高都是什麼原因。這個離職原因當然是要真實的，所以你最好對這些員工進行逐一的離職訪談，盡量了解真實的原因。然後針對這些原因，你要先提出可行的解決方案。如果你覺得寫下來麻煩，那你在自己的心理也應該要先把這些想清楚，以防老闆問你的時候，你能夠答出來。

第二，帶著方案去找老闆

在你想好你的解決方法之後，你可以去找老闆了。找到老闆之後，你可以先跟他提出問題。這時，他一定會問你怎麼辦？就算他不問你，你也應該找機會把你的想法告訴他，讓他知道你是認真考慮過這些問題，你是有準備的，讓你相信你是有能力解決這些問題的。

在這裡，我想跟大家強調的一點是，你的解決方案，一定不能只有一個，而是要有兩三個。因為老闆喜歡做選擇題，而不是判斷題。你把這些方案的優點和缺點告訴他，讓他選擇就好了，這樣，他省事，你也省事。

第三，你要了解老闆的喜好、看重點

跟老闆彙報工作，你要了解他的喜好和看重點，否則，你很難和他溝通成功。有些人喜歡看你方案的可行性，有些人喜歡看方案的創新性，有些人喜歡你的解決方案少花錢，這些喜好，都是需要你提前去了解，才能提高你和老闆溝通的成功率。

場景二：水平溝通

不管是什麼崗位，只要你在公司上班，那水平溝通就很多。比如，如果你需要推行一個考核方案，那你需要和各個部門的負責人溝通，以讓他們了解你的方案，讓方案更好地落地；如果推薦一個求職者過去面試，那你要和用人部門的面試官溝通，你要說服他錄用這個人；如果你要舉辦一次年會，那要和各個部門的同事溝通，以發動他們積极參加年會。

以上溝通場景，其實點出了水平溝通很重要的一個目的：說服。做好跨部門溝通，其實更多的是說服他們接受你的觀點。

在職場，你經常需要說服一個人去做一件事。可是，如果你說服的方法不對的時候，那你可能就很難拿到你想要的結果。

朋友T的同事I是一個女孩子，是公司銷售部門的助理，上級是公司營銷副總經理。在朋友T的眼裡，她非常不好溝通。

每次朋友T過去找她辦事，都差點跟她吵起來。上周，朋友T想組織銷售部門的員工去外面拓展，所以他想請銷售助理I幫忙做個調查，調查銷售員的意願，以確定人數。

「I，有事想讓你幫個忙！」朋友T走到銷售助理I的座位前說。

「什麼事，說！」銷售助理I語氣生硬地蹦出了一句，聲音很大，周圍的同事都被這句話吸引了，目光齊刷刷地向朋友T這邊看過來。

「下個月我想組織你們部門的員工去拓展，想請你這一兩天幫忙做個意願的調查，願意參加的報名到你這裡！，然後你把名單給我！」朋友T很謹慎地說。

「我很忙，事情很多，沒那麼快！你先把活動方案給我！」銷售助理I說。

「我這個事情很急，能不能加快點？盡量這兩天給我？」朋友T像在哀求她一樣。

「你剛才聽不懂我說的話嗎？我現在很忙！」銷售助理I大聲而且語氣堅定地說。

朋友T一聽這話，似乎火馬上就來了，說：「你這人怎麼這樣，我好好跟你溝通，你什麼態度！」

銷售助理I一聽，就火了，躥地一下站起來，說：「我跟你就事論事溝通，我有說過不幫助你嗎？你發什麼火？」

就這樣，兩個人一言我一語地吵起來了，最終在領導的勸阻下才停下來。

在這個案例里，朋友T跟銷售助理I的溝通最大的問題是朋友T沒有認同銷售助理I，導致他們的溝通失敗！在溝通中，沒有認同，那雙方就很難達成一致。那誰應該先主動認同呢？最大的原則是，誰是被動方，誰就應該主動去認同主動方。朋友T是求別人辦事，自然是被動方。

因此，針對上述溝通案例，對於水平溝通，我有三個建議：

第一，當出現不同意見的時候，要先認同對方

比如，當銷售助理I說「我很忙，事情很多，沒那麼快！你先把活動方案給我！」的時候，朋友T並沒有認同她，而是說出了自己的需求，導致銷售助理I情緒開始出現激動。這是典型的溝通對立面。朋友T正確的說法應該是：「是啊，你確實挺忙的，我看你每天跑來跑去，確實很辛苦！」當朋友T能夠真誠地認同銷售助理I的時候，銷售助理I內心一定會很感動，接下來，你再跟她說你的需求，她肯定不會再情緒激動。

第二，尊重對方的忙，提出你的需求

在認同對方後，你可以提出你的需求。同事之間的溝通，是要相互尊重和體諒的。你認同了她，我相信她也會反過來認同你。所以這個時候，朋友T可以接著說：「這個活動後期不會佔用你的時間，但前期還是需要你幫忙一下。你看看三天後能否給我答案？」正常來說，當你這樣說的時候，她應該會答應。

第三，最好的說服，是一起完成

你可以給對方時間去完成，但如果你確實趕時間，那你可以這樣說：「那你先忙，你看看你什麼時候不忙，我來找你一起商量一下，看下怎麼把這些事情做好！」也許你叫她自己一個人去做，她會抵觸，但是，如果你說跟她一起完成，那她就沒有借口抵觸了！這不就達成了你想要的目的了嗎？

場景三：向下溝通

作為管理者，難免不會碰到難管的下屬。比如，如果公司的招聘效率不佳，招聘及時率低下，你作為人力資源經理，你想叫招聘專員去多開發招聘渠道。比招聘專員叫小李，那你可以用命令的方式給他下任務：小李，公司的招聘渠道只有網路招聘，要不你去開拓一下校招渠道？

小李一聽，馬上說：好好，我馬上執行。結果一個月之後，一無所獲。回來總結，你批評他：怎麼回事？一點效果都沒有！他肯定會說：領導，這是你叫我去做的！

命令式的向下溝通，會存在兩個問題：

第一，下屬容易抵觸

有些向下溝通場合，命令式布置任務會遭到下屬的抵觸，特別是對知識型員工來說，因為它忽略了新時代員工對自主選擇的需求。

第二，下屬容易推脫責任

因為命令式的向下溝通，完全是領導的意思，下屬執行不好，他就容易找借口，從而推脫責任。

第三，做事容易偏離你想要的軌道

有些難管的下屬，都有自己的主見，你越命令他這樣做，他可能就不這樣做，這樣他做出來的東西可能會偏離你的軌道。

因此，如果你要成為一個有管理水平，會說話的管理者，我建議你給下屬布置任務的時候，學會給下屬選擇，而無論他怎麼選擇，都是你想要的選擇。

比如，上面的例子，你叫下屬去開發招聘渠道。你可以對他說：「你覺得除了網路招聘渠道之外，還有哪些招聘渠道適合我們公司？」他可能會說：「校招、熟人推薦、內部推薦等！」那你接下來可以說：「現在公司要拓展招聘渠道，你剛才說的這三個渠道，如果要挑一個去主攻，你想去做哪個？」他一聽，就會從他剛才說的三個里選一個，他也不會跟你再計較這個不合適，那個不好，因為都是他自己的選擇！

做好向下溝通，給選擇比命令更有影響力！不信你試試！
