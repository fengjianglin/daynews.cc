為什麼女生冬天顏值沒有夏天高？

2019-12-16 195條評論 2,157次閱讀 814人點贊

你們身邊有沒有那種一意孤行，根本不聽你勸告的姐妹呢？Tina身邊就有一個，因為她愛上了渣男，並且知道他渣得無可救藥，還是不死心。

今天我們打電話，她又講了和渣男發生的事，聽得我快要氣瘋了，更氣她還心存幻想。都說愛情里的傻女人，得讓她跌個跟頭才知道痛，要回頭，可她跟頭跌了一次接一次，一幫人攔著不讓她跳的火坑還是要往下跳……

⇧日常傾訴，日常被我「懟」，

寧願讓她覺得我講話難聽，

也不想她以後受傷更深。

為了讓她醒悟，這一年我和她的家人一直扮演惡人的角色，面對她的痴心幻想都是無情戳破，但收效甚微。

沒轍了，你們有沒有什麼讓姐妹迷途知返的好方法，不如留言告訴我吧。

一時忍不住和你們傾訴了一下心事，但是說出來感覺好多了。為了不讓你們心情受到Tina的心事影響，今天就分享一些輕鬆的內容！

你們知道嗎？女孩子的顏值是很不穩定的，每當冬天一到，就……沒有夏天那麼好看了！Tina仔細觀察了一下，發現了原因。來，進入今天的小課堂，重回顏值巔峰好嗎？

⊙ 妝容

先說說決定著「門面效果」的妝容，都說夏天是「化妝兩小時，出門5分鐘」，但你好歹願意花兩個小時化妝啊！

在冬天，為了多爭取一分鐘的睡眠時間，兩小時的化妝時間可以壓縮到5分鐘，簡單化化就迅速出門。

⇩夏天化妝2小時

⇧冬天多睡每一分

而且夏天的妝容好看清透，像是甜甜的水果，甜到人心裡去。

再加上blingbling的配飾，和膚色相互襯托，整個人都會明艷動人。

⊙ 穿著

有了好看的妝容還不夠，在夏天，漂亮的小裙子多到讓人瘋狂，穿上更是仙女本仙。

冬天的衣服相反，會遮住身材的曲線，變成「無腰星人」，大長腿也沒機會展露。

夏天穿衣搭配特別簡單，一個T恤+短褲就休閑簡約，在家穿舒適愜意，出門穿清涼有活力。冬天不僅要搭配，還會一不小心變成「企鵝」。

不過不要擔心，可以媲美夏天的穿搭，Tina已經幫你們選好了。

冬天為了避免變成「無腰」和「無腿」星人，就一定要get露出腰線的穿搭。

簡單的方式就是選擇有腰帶或者收腰的外套，有這一點的加持，冬天穿個浴袍都好看。

尤其是這種比較寬鬆的長外套，沒有腰線就會顯得整個人很「龐大」，而一旦收腰就氣質蹭蹭的，搭配闊腿褲，簡約幹練。

長款羽絨服一穿，整個人就變得和衣服一樣蓬鬆了起來，改變這種臃腫的感覺，可以選擇有收腰設計的羽絨服，保暖洋氣，身材比例也能秀出來了。

不喜歡腰部被束縛的感覺，可以系松一些，但是不要太隨便，保證系帶平整整潔，再加上小高領和直筒褲，休閑隨意，但品味不減。

當然，還有一種方式就是通過內搭和下裝的組合，打造出高腰線，在冬天依然能秀出大長腿，而且對外套的要求可以適當放寬。

黑色內搭和亮色的高腰半身裙，時尚亮眼，還有一種酷酷的女人味，外搭中長款的羽絨服，風度溫度都有。

如果不知道選什麼做外套裡面的搭配，那毛衣+九分褲+短靴的組合就不會出錯，無敵顯高，加上皮大衣，走在哪裡都slay全場。

雖說夏天的漂亮裙子很多，但冬天也有巨好看的裙子，再融入疊加技巧，別有另一番味道。

Tina尤其喜歡純色毛衣+各種裙子的上下搭配，從上衣到裙擺，帶著弧度的線條，柔和優雅，讓穿著它們的人能夠氣質大增。

垂感很好的高領針織連衣裙，女人味十足，搭一件有V領效果的毛衣外套，顯瘦又有層次感。

還有可以和黑色小高領任意組合的連衣裙，真的怎麼穿都氣質。尤其聖誕紅，濃烈不高調，很有冬天的氛圍。

小香風毛衣+黑色百褶裙，穿上你就是女神，還可以搭配米白色的大衣，整體優雅有層次感。

昨天還有小可愛問我，沒穿過裙子但是想穿怎麼辦。完全可以從連衣裙開始穿起，不容易踩雷，還顯瘦，擔心裙子花哨，一件純色大衣就能搞定。

大衣+裙子，敞開穿有腰線有氣場，繫上扣子和腰帶，露出裙擺一角，層次感和女人味也絲毫不減，腰間的黑色包包還會注入一絲少女活力。

冬天才能穿的毛毛外套，可愛又暖和，搭配連衣裙，簡單好看。

夏天有好看的配飾和清透的妝容，冬天也有適合的佩戴和化妝方式。

比較突出的就是秋冬口紅了，那一抹啞光土橘，塗上就是知性本人。

冬天的妝容，還很立體，顯得人整個五官深邃迷人。

保暖又甜美的貝雷帽，特別適合搭配毛絨絨或者偏溫柔范兒的外套、上衣，甜蜜度爆表。

冬天可以圍的大大的圍巾，喜歡那種把脖子圍住的感覺，暖暖的，像是自己被呵護著。

還有聖誕節特定的毛衣，穿上感覺自己是生活在童話里的女孩子。

誰說動漫風的麻花辮是夏天的專屬？冬天搭配條紋小高領，清新甜美，含糖量都要超標了。

再說說飾品，冬天必須戴！想像一下，和別人握手或者抬手的時候，露出好看的手鏈，那種不經意間流露的精緻，真是太吸引人了。

毛線帽子+絨絨外套+黑色小高領+秋冬妝容，啊，這種屬於冬季的顏值加成，拜託你們立刻擁有！

好了，今天的內容大家都學到了嗎？趕快活學活用起來，冬天也做精緻小仙女。到時候，別忘了留言告訴Tina效果如何！
