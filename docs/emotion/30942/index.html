同學聚會，土豪：這吃的都是什麼東西，班長：你可以先走

2019-12-24 374條評論 1,569次閱讀 188人點贊

畢業之後，大家為了自己的前途，各奔東西，大部分人都在職場上打拚，為了工作有時候要放棄一部分的生活，有時候要離開自己，親愛的同學，摯愛的親人，在外面漂泊時間長了，見識過職場中的利益，有時候就經常想念這些純潔的感情。

我在高中的時候，是班裡面的班長，其實我並不想當班長，但是老師非要讓我當，於是我就當了班長，作為班長，有責任也有權利，輔助班主任管好這個班級，而且自己要搞好自己的學習，以身作則，在高中的時候我是儘力做好每一件事情。

不知不覺已經高中畢業十幾年了，看著別人搞同學聚會，我也想搞一個同學聚會，每次聯繫大家的時候，這個說他在出國，那個說他沒時間，要不就是說他工作太忙了，各種理由，我想來一個一個人都不能少的同學聚會，可能是我的強迫症吧，總是沒去辦成功。

我也有自己的工作，如果不能來一個完整的同學聚會，我是會有遺憾的，現在十幾年過去了，我覺得是應該辦一場同學聚會了，每一個人都要來，一個人都不能少，我不相信大家都這麼忙呢，於是我就聯繫大家，挨個兒通知，我聯繫了當時的學習委員和其他班幹部，大家一起來舉辦這場同學聚會。

還是有很多同學有很多的理由，他給出一個理由，我就給他一個解決方案，我的目的就很明確，一定要舉辦一場同學聚會，同學聚會AA制，每人200塊錢，，每個人坐車過來，或者我們安排車接車送，不準有炫富行為，大家都是平等的，照顧每個人的感受。

終於舉辦了一個人都不缺的聚會，我很開心，差點流淚了，聚會剛開始的時候還是挺開心的，菜上的差不多的時候，土豪同學說了一句話：這吃的都是什麼東西？然後他就是各種挑剔，各種炫耀，沒有遵守我的規定，於是我就說了一句：你可以先走。

結果他先走了，聚會有一點點遺憾，土豪同學走了之後，大家給我鼓掌，現場的氛圍也熱鬧了起來，大家紛紛談論同學時代的時光，我雖然得罪了一個人，但是給大家帶來了一個值得回憶的同學聚會，我覺得值得了，同學聚會中，每個人都是平等的，尊重每一個人，別人才會尊重你。
