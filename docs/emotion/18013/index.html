和同事發生衝突，記住3大原則，不僅不吃虧，人緣還會越來越好！

2019-12-16 1,231條評論 2,614次閱讀 639人點贊

在職場中，其實跟同事之間相處難免的會有一些衝突摩擦，有些人不會處理，自己吃虧不說，而且人員也會越來越差，那麼在職場中，如果和同事發生了衝突，我們應該怎麼解決呢？其實記住下面這三個原則，不僅不會吃虧，人緣還會越來越好！

第一個：你干你的活，我做我的事，最小阻力原則

其實職場中發生矛盾衝突其最本質的原因無外乎就是利益的衝突，你在意的是過程，而我只看重結果；你在意成本，我著眼收益……，因此其實我們不能直接否定對方的核心訴求，用觸及對方利益的方式來解決衝突，這種做法其實就像是在打對方的臉，你得到的只會是對方更為強烈的反抗，衝突不僅不會消弭，而且還會愈演愈烈，衝突只會是越來越大的。

因此，我們說用最小阻力原則，其實就是要用一種可以讓對方接受，自己也損失比較小的方法來處理，比如「被人搶功勞」這件事情，其衝突點就在於，你自己辛辛苦苦做出的成果，被人輕易的拿走了，這時候一味的反駁其實並無什麼實際用處，這時候阻力最小的方向就是各說各的，既是給了對方一定的壓力，又沒有撕破臉，可是卻明確的給對方了一個信號：這件事並不是完全掌握在他手上的。

第二個：共同的利益和共同的敵人，最大公約數原則

其實職場中發生衝突的原因，大概有下面兩種，一種是性格衝突，另一種就是利益衝突，前者其實不必多說，無非就是性格不合，彼此看不上，這樣的衝突其實一般並不會出現太過不可調和的矛盾，可是後者就會隱藏較深，表面上可能還是你好我好一家親的樣子，實際上卻是背後放冷箭，使絆子，雖說這樣說是誇張了一些，可是涉及利益，關係就會變得很脆弱了。

這時候，如果想要化解衝突，那麼就需要從兩個方面入手，一個是找到彼此共同的利益，另一個是發現彼此共同的敵人，這兩種情況都最容易讓兩個人同仇敵愾，暫時結盟，這其實就是「利益的最大公約數」，在這個過程中，彼此也可以慢慢化解性格上的衝突，所以我們不能只看到利益的衝突，也要看到共同的利益。

第三個：會吵架但是不樹敵，局外人原則

不要以為會吵架就是厲害的，其實不然，真正的高手是吵完架，脾氣發泄完了之後，還不會四面樹敵。為什麼會這樣呢？其實也很簡單，就是他們很會說道理，不管是自己有理沒理，反正他們都能言之有物，讓人聽了之後覺得是不是真的是自己的錯，開始自我反省起來。

其實吵架也是有套路的，簡而言之就是：你跟我說人情，我就跟你講道理，你講道理的時候，我就開始說人情，不必夸夸其談，點到為止就好樂。
