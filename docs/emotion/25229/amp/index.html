與往事乾杯，再見了：你的，我的2019

天天要聞
2019-12-21

白駒過隙。一年又接近尾聲了。

「怎麼感覺還沒有怎麼過，今年就過去了呢？」身邊的很多人都在發出這樣的感慨，說是感慨，其中飽含著辛酸，也飽含著追逐。

每個在生命中踽踽獨行的人，都能夠深刻理解自己內心的苦楚，只是我們把那份生活中的苦楚埋在了自己的心底，不想讓它大白於光天化日之下。每個這樣的人，都是堅強的，也是強大的。我們埋藏自己的苦楚，只是想著把自己最溫暖的愛留給身邊值得去愛的人。

我想把「追逐」這個詞當做一個褒義詞，因為我們每一個心中有夢想的人都在追逐：追逐自己的夢想，追逐自己所愛的人們，追逐自己渴望成長的腳步，甚至是追逐父母不再黑的頭髮和曾經有力的大手……

（一） 親情

今年的1月19日，貴陽火車站，東北的一名范姓大叔在車站自己補損壞的行李袋，他說自己有一個在讀研的孩子，所以生活要從簡。

網友評論說：「我們每個人都有這樣愛我們的父母。」簡單的生活，沉重的父母之愛。他們自己簡單的生活成就了我們的成長，我們成長之路也是由父母的汗水鋪成的。

今年的1月22日，四川成都的朱大爺說自己的兒子在河南安家，讓他去河南團聚。朱大爺挑了六七十斤橘子，還特意買了30棵橘子樹一起帶著。因為行李很重，有些東西還掉在了汽車上。朱大爺說：「自己就是再累也要把家鄉味帶給兒子。」

對於子女，父母從來都是無怨無悔。或許他們不修邊幅，或許他們沒有高深文化，或許他們總是默默無言，但是他們內心始終把兒女當做自己的心頭肉。無論何時，無論何地，身為父母，就操著一輩子的心。

史鐵生曾在文中寫到：「多年來我頭一次意識到，這園中不單是處處都有過我的車轍，有過我的車轍的地方也都有過母親的腳印。」

是的，每一位父母都是在追逐孩子的腳步，而我們卻總也追不上父母老去的速度。 蔣方舟也曾說：「母親是一寸寸變老的，而父親是突然老的。」當我們發現父母已經兩鬢斑白的時候，我們再也追逐不到父母曾經的面容。

（二）友情

一起出演《七月與安生》的馬思純和周冬雨，是娛樂圈裡眾所周知的好閨蜜。前段時間，馬思純主演的新片《大約在冬季》在即將上演之前，周冬雨幫好姐妹宣傳，一下子轉發了四條微博。微博裡面還發起了周冬雨為馬思純連發微博的話題，粉絲們紛紛羨慕她們之間的友情。

在《小小的追求》中，在和尹正聊天的時候周冬雨說到，自己特別討厭和別人一起睡覺，除非是相處了六七年的朋友。然而之後尹正提到馬思純，周冬雨毫不猶豫地回應可以，甚至表示可以兩人一起睡覺，還能蓋同一張被子，這樣的友情真是令人好生羨慕。

魯迅曾說過：「友誼是兩顆心真誠相待，而不是一顆心對另一顆心的敲打。」而周冬雨和馬思純的友誼就是如此：簡單，真誠。

友情是我們生活中不可或缺的，它就是一劑良藥，時刻撫慰我們的心，時刻給我們力量。所以，匆忙的日子裡，一定要記得和朋友真誠相待。

（三） 愛情

馬伊琍曾經說：「戀愛雖易，婚姻不易，且行且珍惜。」就當所有人都以為她和文章的愛情會繼續的時候，她們在2019年宣布離婚了。

「你我深愛過，努力過，彼此成就過。此情有憾，然無對錯。往後，各生歡喜。」這就是生活，明星大腕們的生活也是如此，誰也想著和自己另一半攜手白頭，但是生活中的種種會成為我們的絆腳石。

然而我們大多數不是明星，我們都有自己普通的生活。不要羨慕嫉妒恨，也不要抱怨攻擊打。最近大火的80年代的電影《牧馬人》中男主人公許靈均說了這樣一句話：「感情是信念的基礎。」

其實兩者是相輔相成的關係，有感情才會信念堅定，信念堅定才會感情牢固。每一個珍惜彼此的人，都不要對這兩者產生褻瀆，那樣就是踐踏了自己的感情。

我們每一個人都是圍繞在親情、友情和愛情當中，我們彼此愛著，暖著，所以我們才會產生自己的生活圈子，這個圈子當中的每個人都在成就著彼此。

親情，是上天賜的最大的緣分。友情，是你想要回去的那一段共同時光。愛情，是一輩子的約定。—— 顧西爵

對於大多數的人，人生本就是一堆責任而已。參透此諦，愛情是緣，友情是緣，親情尤其是緣，無論怎樣，皆當潤礫成珠。—— 梁曉聲

而在這年復一年，日復一日當中，我們需要學會的就是去享受這份情，去經營這份愛。關於往事，在自己的心中編織一張網，漏掉那些壞的情緒和記憶，留住那些最美好的畫面。

與往事乾杯，再見了，你的，我的2019。2020年，你我繼續追逐！
