“我媽偏心，但她給了我一個最好的弟弟”手足親情，有溫暖有心酸

2019-12-18 3,675條評論 4,659次閱讀 2251人點贊

都說「扶弟魔」很可怕，其實生活中更多的是，前幾年姐姐幫扶弟弟，弟弟長大成人後轉身再拉姐姐一把，人世間這種美好的親情，才是真正的「手足情深」。

【口述實錄情感系列】

文|關心於

01.

@關關

嗯，我有個弟弟，比我小5歲，小時候嘛，他是混世大魔王，我是霸道小魔女，所以我們經常打得雞飛狗跳的，我現在都記得我們倆曾經打架把冰箱都打倒了，為了搶一根雪糕……

長大後我也不得不讓著點他，學習上、工作上也沒少指點他，包括談戀愛、結婚，我都是幕後總指導，哈哈。當然，他不服的時候也有，但他不服我就必然要拿出大姐大的絕對權威收拾他。

但現在弟弟反而更成熟更體貼，和做醫生的弟媳婦一起照顧我爸媽的身體，把我爸媽的房子換到和他們小區一起，走路送一碗湯的距離，買米買油買菜送給我爸媽，不讓我爸媽幫帶他們孩子，叫我爸媽出去旅遊……

我這個當姐姐的，現在很輕鬆很輕鬆。我弟的理由是，當初他還是混世大魔王的時候，沒少給我惹事，讓我鬧心，現在是時候讓我靜靜心了。

其實我更感謝我爸媽，他們都是很寬容很民主的人，我們家的家庭氣氛一直很好，我爸媽很照顧兄弟姐妹，對舅舅小姑都很好，同時也很孝敬上一輩，我外公腦溢血時，我爸自己洗外公沾了屎尿的褲子。父母是孩子最好的榜樣，他們做出榜樣了，我們姐弟自然也會這樣。

我愛我弟，我為有這樣可親的弟弟而驕傲。

02.

@小板栗

我弟比我小兩歲，是我媽盼來的寶貝，從小我媽就重男輕女，好吃的好玩的好用的，都會首先想著我弟。我弟從小就知道我媽偏心，但那時他也無能為力，只能偷偷地幫我。

重男輕女家庭的女兒註定了要多犧牲，細枝末節的事太多，總之我事事都要給弟弟讓路，比如大學畢業後，我媽就要求我工作，將讀研究生的機會讓給我弟。於是我只好先工作了。

我工作兩年後，我弟堅決支持我去讀研，最後我如願以償，比弟弟低一屆。弟弟是工科，我讀研時他已經存了好幾萬，我讀研後，他給我買i8，給我配電腦，前前後後還給我一萬多。他自己有壓力有迷茫也會和我講，在這個舉目無親的大城市裡，他像哥哥一樣照顧我，又像弟弟一樣依賴我信任我，我們真正就像手與足一樣。

再後來，我結婚了，弟弟一股腦將他賺的錢一分不剩地全給了我，婚禮上，他說，姐，這麼多年你受委屈了。說完一轉身就哭了，一個快一米八的男生，就那麼哭了……

我不氣我媽的偏心，因為她給了我一個最好的弟弟。

03.

@鹹魚少女

我弟比我小7歲，他是世界上最可愛的弟弟了。

我弟從小就是個吃貨，所以做好吃的給我吃是他的一大樂趣；他愛乾淨，小時候給我收拾房間的總是他；我說我是大公主他是小奴隸，所以他就事事處處照顧我，哈哈。印象最深的一次是，工作第一年我生病住院，手術後都是弟弟給我洗頭洗臉洗手，給我講故事，唱歌給我聽，還給我買好吃的好玩的。以至同病房的人鬧了個天大的笑話——他們問我，那個又高大又英俊的小夥子是你男朋友哇？真不錯！

現在我們都成家了，而弟弟更是義無反顧地成了頂樑柱，比如我爸生病，每天都是弟弟陪夜，出院後也是弟弟安頓在他身邊照顧。我弟經常給我老公說的口頭禪是：「對我姐好點啊，我姐可是我們家的公主！」

其實這主要歸功於我的父母。我的父母是那種男女一視同仁的父母，對女兒和兒子一樣地愛、一樣地付出，甚至很多時候他們會教育我弟弟：「你姐是女生，你一個男子漢要對你姐好點！」

弟弟對姐姐怎麼樣，其實是受父母的態度影響的。
