媽媽從老家來呆了兩天，女兒回到住處卻差點哭了......

天天要聞
2019-12-10

12月8日一早，杭州婁先生在30路公交車偶遇了一位講紹興話的扁擔阿姨。

她50來歲，穿著打扮蠻鮮亮洋氣，挑著一根扁擔，前頭一袋子番薯，後頭一編織袋裡後來大家都知道裝了雞蛋。

一上車，司機還以為阿姨是要做生意的，一聊才知道，阿姨來自紹興，扁擔里挑著的東西都是送給在杭州的兒子的。

阿姨說兒子就喜歡土雞蛋打打（做雞蛋羹），她在家裡養了40隻雞、20隻羊、2畝菜地還有番薯，每個禮拜都要挑來杭州給兒子吃。

婁先生也是上虞人，這個遇見，讓周末加班的他一整天心情都很好。

前天，我們對此進行了報道，→《養了40隻雞、20隻羊！杭州公交車上來一位「女商販」，整車人都在羨慕她兒子！》，希望能找到扁擔阿姨，不過截至昨天小時新聞發稿，阿姨還沒有下落，但是我們在讀者網友的評論和留言里，卻找到了很多同款姆媽，找到了一樣的愛。

鄉下有老家的網友都來分享了自己的同款老母親，說著說著就淚目了——

「我也是上虞人。我媽媽養了一百多隻雞鴨鵝，兩三畝菜園。」

「幾崚番薯太熟悉了。我是浦江的，我姆媽70多歲了，每次我回家都給我變著花樣做好吃的，浦江大餛飩，浦江麵餅，還會準備青菜、蘿蔔、番薯，當然也少不了土雞蛋。」

「每次回老家都是愛的後備廂，我媽最厲害一回，連自己腌的腌菜都給我帶上了。」

「我老公老家浙江衢州，我媽家在湖北，每次回婆婆家帶蔬菜雞和肉回來，媽媽家帶芝麻油菜籽油花生油和雞蛋回來，倆媽媽後備箱從來都是要塞滿才讓回來。」

「我公公婆婆每次來杭州，恨不得把老家的東西都帶上，牛肉，分熟的生的兩種，土雞蛋，土鴨蛋，腌好的魚，肉，就連被子也帶來，說是來杭州就住一兩個晚上，省得我們換被褥。」

「我媽呆了兩天回去，我沒怎麼陪，下班回家，發現她把家裡給我拾掇得乾乾淨淨，就連進出門的腳墊都給我洗了。我差點眼淚掉下來。我忘了告訴她，明天阿姨就來搞衛生了。想著她膝蓋不好，還幫我擦地的樣子，真是想哭……」

不少城市小夥伴則表示羨慕，畢竟在鋼筋水泥森林裡，這樣充滿泥土味的愛不是想要就有的。

網友「楊林月」說：在杭州工作的兒子有父母照看的『家庭農場』，家裡需吃的不用愁了，而且安全又環保，吃的很放心！真羨慕這位兒子有這麼好的父母，家裡有這麼多雞，羊……」

網友「檸檬樹下我和你」：評論區里還有跟阿姨一樣「家底殷實」的，看著我也是滿滿的羨慕嫉妒，你們真的不是拉仇恨的嗎？

一根扁擔，肩挑兩頭，滿滿都是對孩子的愛，這份愛其實無分東西，你是不是也有這樣的同款母愛？
