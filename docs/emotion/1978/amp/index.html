校園戀情不靠譜？小伙被女友管東管西甚至覺得自己找了個媽

江源（化名）和女友焦雪（化名）是在重慶三峽學院讀書時認識的，如今兩人已經交往了快五個月時間，但是最近，讓江源十分反感的是焦雪一直管著自己，去哪裡都要向焦雪報備，感覺自己的私人空間都沒有了。

起初兩人交往的時候，因為跟班上的同學還不是很熟悉，所以兩人都是結伴著一起吃飯，有時候兩人一起排隊打飯，但是江源又比較照顧焦雪，更多的時間是江源去打飯。

如果遇到周末休息的時間，江源好不容易想睡個懶覺，卻要被焦雪叫起來陪她吃午飯，還要逛街。這些事情在兩人才交往的時候，都是經常做的事情，在旁人看來，都讓室友們十分羨慕，覺得男友能在周末的時間陪自己算是很大的付出了。聽起室友說起她男友，除了睡懶覺就是打遊戲，完全不會找自己。

江源覺得自己每次都和焦雪黏在一起，像室友的聚餐，有時候焦雪都會要求江源帶著自己去，目的也就是為了防止江源出去鬼混，和室友喝得爛醉之後回來。但這樣的行為不僅讓江源很反感，也讓其他室友覺得很不好意思，經常飯吃到一半，也就才九點左右的時間，焦雪就要求江源回去了，幾乎每次吃飯都沒有盡興，還經常掃興。

江源覺得自己都是成年人了，父母都不怎麼管自己了，焦雪倒像是個媽媽一樣，一天到晚管著自己。每天被要求按時吃三餐，如果沒有吃的話，焦雪就一個電話打來，直接被叫出去，焦雪看著他吃。和室友打球回去晚了，直接去操場上找人，揪衣服扯耳朵都要把江源帶回去。

江源幾乎是每天活在焦雪的監控之下，渾身不自在，也經常被室友嘲笑說，「你真的是給自己找了個’媽媽’一樣，看著真難受。」江源也不想被焦雪一天天地管束著，跟室友一起去吃飯，打球，很晚才回來。焦雪手機都快打爆了，江源也不回電話。

直到第二天兩人見面之後，焦雪一上來就給了江源一巴掌，問到之前為什麼不接她電話，還當她是女朋友嗎。江源也怒氣沖沖地回答道，「你是我女朋友，不是我媽，不要一天管東管西的，我也需要自己的私人空間，不想你每天跟在我屁股後面。」江源覺得兩人並不合適，想要分手，說完之後轉頭就走了。

事後焦雪也想了想自己的所作所為，也確實管得太多，焦雪也想過要挽留這份感情， 但是也確實對江源造成了很大的傷害，所以兩人也算是和平分手。分手之後，江源也直接刪除了她的聯繫方式，就算是在校園裡碰到，也當做不認識了。
