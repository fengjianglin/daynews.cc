孩子賴床不起，家長大喊大叫？遠離“殺傷性”叫醒方式，益處多多

2019-12-10 78條評論 1,321次閱讀 151人點贊

文|羊駝媽媽（本文原創，版權歸作者所有）

科學研究表明，人睡覺的時候大腦實際上會進行「睡眠周期」的循環往複，根據腦電儀的數據分析，人類睡眠周期有兩個部分，一是NREM，也就是非快速眼動睡眠周期，另一個是REM，是快速眼動睡眠期，有研究人員發現，有些人在REM階段被叫醒，就會有起床氣，屬於一種焦慮反應。——知乎日報

孩子在成長的階段中需要經歷最重要的學生階段，而無論是學校還是幼兒園，早上上學都是有規定的時間的，那麼孩子就需要在規定的時間內起床，否則就會上學遲到。

小區里趙女士有一個四歲的孩子鑫鑫，最近鑫鑫和媽媽生氣了，原來是鑫鑫到了上幼兒園的年紀，需要定時起床。

趙女士早上叫鑫鑫起床，鑫鑫受不了媽媽的大嗓門，於是便拒絕和趙女士溝通。原來趙女士叫醒鑫鑫的方法是最原始的大喊大叫方式，鑫鑫之前從沒有過這樣的情況，突然間被媽媽叫醒，嚇了一跳。

孩子起床氣大有哪些原因？

  • 熬夜，睡眠時間不夠

孩子起床氣大、不想起床，說明孩子沒有睡夠，還想繼續睡覺。家長如果發現孩子起床氣很大，要查看一下孩子前一天晚上是幾點睡覺的，是不是比平時睡覺晚，導致孩子的睡眠時間不夠，從而不想起床。

家長需要讓孩子養成良好的作息規律，到了規定的時間就要睡覺，不要熬夜。熬夜對孩子的身體發育是非常不利的，孩子處於成長階段，良好的睡眠質量，孩子才會更好的發育。

  • 家長過度溺愛，孩子沒有規矩

有些家長在教育孩子的時候對孩子很溺愛，孩子在小時候還沒有到上幼兒園的年紀時，孩子想睡到幾點，家長就讓他們睡到幾點，孩子想幾點睡覺，家長就讓他們幾點睡。

在孩子的思想里沒有規矩，孩子從來沒有被約束過，導致孩子行為囂張，不想被管束。所以等到孩子上幼兒園的年紀，孩子不能適應在規定的時間內起床，也不聽從家長的叫醒。

  • 孩子內心缺乏責任感

大人們也會有賴床不想起床的情況，但是大人卻很少會出現不起床的情況，因為大人需要上班，需要掙錢，這是他們的責任。

而孩子不想起床，到了固定的時間也不起來，即使知道上學會遲到，還是賴在床上，這說明孩子心中沒有責任感。孩子沒有將定時起床、到點上學當成自己的責任，孩子對事情的重要性不以為然，認為即使沒有完成也可以。

家長有哪些錯誤叫醒孩子的方式？

  • 突然闖進孩子房間，大吼大叫

孩子要上幼兒園每天需要定時起床，雖然家長給孩子訂了鬧鐘，但孩子的自制力很差，聽到腦中響也不肯起床。

於是很多家長便會發脾氣，看到孩子已經過了很長時間還不起床，於是便闖進孩子的房間，對孩子大吼大叫。家長的這個行為，會嚇到孩子，孩子從睡夢中被驚醒。久而久之，孩子的安全感會越來越少。

  • 拉開被子，將孩子凍醒

有些孩子在睡覺時即使有噪音孩子也可以安然入睡，所以家長在試過鬧鐘、大喊大叫無果時，便會手動將孩子的被子拉開，將孩子凍醒。

孩子在早上的時候免疫力很低，身體各項機能孩子沒有蘇醒。這個時候孩子的被子被拉開，被子中的熱氣被散開，孩子會感覺到非常冷，很容易出現感冒的情況。

  • 用肢體或物體拍打孩子

有些行為粗魯的家長會直接用手或物體去拍打孩子，讓孩子不得不起床。孩子在睡夢中，身體被擊打，無論是心理還是身體上都是退縮的。

家長可以試想一下，如果自己在睡夢中被人突然拍醒，自己是什麼反應。而孩子會變得敏感，性格膽小。有的孩子則會相反，孩子醒來後會非常憤怒，和家長吵架，孩子的性格也會越來越暴躁。

家長該如何正確的叫醒孩子？

  • 提前告訴孩子，自己會去叫醒他

如果孩子對鬧鐘免疫，鬧鐘響了孩子也不醒，家長可以在前一天晚上告訴孩子，自己會在第二天早上叫醒他，讓他有心理準備。

孩子內心有了準備，在面對家長的叫醒時，也不會反應過於激烈。家長每天叫醒孩子的時間也最好相同，控制在固定的時間段內，讓孩子形成生物鐘。久而久之，孩子不用家長的叫醒，自己也可以起床了。

  • 用舒緩的音樂喚醒孩子

人們在早上的時候普遍都會有起床氣，但每個人的情況不同，有的人不明顯，有的人則會反應很大。

如果這個時候用激烈的方式去叫醒孩子，孩子的內心一定是抗拒的，情緒也會非常憤怒。家長可以用舒緩的音樂去叫醒孩子，選擇一些大自然的音樂，有溪流、鳥叫的聲音，這對孩子來說更容易接受。

  • 早飯的香氣，喚醒孩子的食慾

孩子經過一夜的沉澱，胃裡的食物已經消化完畢了，所以早上是一定會感到飢餓的。家長可以將做好的早餐端進孩子的房間中，讓孩子聞到事物的香氣，孩子的胃部會感覺飢餓，大腦會發出起床吃飯的信號。

這種方式，家長不需要通過語言和肢體的方式來叫醒孩子，孩子也會自覺的起床。

家長如何培養孩子早睡早起的習慣？

  • 讓孩子發自內心想要早睡

讓孩子主動早睡早起，最根本上就是要讓孩子從內心裡想要早睡。家長要告訴孩子早睡的好處，與晚睡的壞處，如果孩子晚睡，那麼便會長不高。

每個孩子都在乎自己的身高，害怕自己比同齡人矮，家長可以從這一點讓孩子發自內心的去早睡。

  • 循序漸進，提前孩子起床時間

孩子原來每天九點起床，但是到了上幼兒園的時候卻需要八點就起床，這個時候如果突然讓孩子早起，孩子一定會不適宜。

家長可以採用循序漸進的方式，一點一點的提前孩子的起床時間。第一天讓孩子在八點五十五起床，第二天讓孩子在八點五十起床，以此類推。

羊駝媽媽寄語：

孩子定時起床是人生的必經事，家長也不要每天都去主動叫孩子起床，也要讓孩子養成自己起床的習慣。家長要讓孩子適應用鬧鐘，定時起床也是孩子的任務，自己的事情需要自己去完成，不可以總是依賴家長。
