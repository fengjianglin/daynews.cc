世間總有一些事，是我們永遠無法解釋也無法說清

2019-11-28 0條評論 24次閱讀 0人點贊

摘要

一、無論你走得多遠，平凡人生就在那裡。每個人面前都有一條未知的路，不管你走上了哪條岔路，你一定要相信，態度和性格最終還是會把你引到那條成功的路上，從現在就開始努力吧!!

一、無論你走得多遠，平凡人生就在那裡。每個人面前都有一條未知的路，不管你走上了哪條岔路，你一定要相信，態度和性格最終還是會把你引到那條成功的路上，從現在就開始努力吧!!

二、如果你不握緊拳頭，你就不知道你力量到底有多大；如果你不咬緊牙關，你就不知道你的堅持到底有多狠；如果你不跺腳而起，你就不知道你的果決到底有多正確；如果你不瞪眼，你就不知道你的內心到底多強大。最強悍的競爭力只有兩個字：行動。只有當你足夠努力，你才會足夠幸運。這世界不會辜負每一份努力和堅持。

三、一個人只要足夠自信，積極思考，能夠為了自己的理想堅持不懈地奮鬥，就一定能夠等到雲開霧散、柳暗花明的一天。

四、在人生的大海中，我們雖然不能把握風的大小，卻可以調整帆的方向。不怕重頭再來，就怕沒有未來！成功是送給有準備的人，成功是送給有付出的人，相信是成功的開始，讓我們攜手共進，創造人生新的輝煌。

五、一個人熬過了所有的苦難，也就不期待一定要和誰在一起了；我曾誠心努力過，但結局我不想說。

六、所謂成熟就是你要習慣任何人的忽冷忽熱也要看淡任何人的漸行漸遠。

七、每一天都會告訴自己要好好控制情緒，不抱怨，謹言慎行，這不是將自己變的懦弱和沒有性格，而是在慢慢的提升自己。凡事不以惡意揣度別人，不以私利給他人添堵，不妄自菲薄，也不詆毀他人，這是對自己最基本的要求。

八、一生平淡少曲折的人，難以成「大器」。期盼著「運氣」的施捨、垂青的人，更是命運的奴隸。只有勇踏坎坷的人，才能真正駑駕人生。

九、夢想不拋棄苦心追求的人，只要不停止追求，你們會沐浴在夢想的光輝之中。

十、跟最愛的人在一起談的是感情，跟最合適的人在一起談的是生活，我們最初要的是感情，最後卻都選擇了生活。面具戴久了，就成了皮膚。
