20歲就生娃當媽，對女孩來說真的好嗎？看看過來人的現狀就有數了

天天要聞
2019-12-18

如今在我們身邊經常能夠看到，很多二十五六歲的女青年，她們一個個活得非常瀟洒，也不願談及結婚這個問題。有時候把家裡父母都急得不得了，而她們卻認為這是一個正美好的年紀，自己何不肆意的去享受呢。這一類人群也在慢慢地擴大著，在她們的觀念中提倡晚婚晚育，也成為如今年輕群體中非常時尚的走向。

與此相反的另一個現象也不容我們忽視，那就是在農村或者是一些偏遠的地方，女性生育孩子的年齡越來越早。

有一些女孩子甚至在十八九歲就當上了媽媽，這樣早早的懷孕，對女孩子來說真的好嗎？

之前在網上看到過這樣一個故事：

一對年輕人在高中時期就互相愛慕，並且陷入熱戀中。由於沒有什麼經驗，而且性知識教育也缺乏，因此在戀愛的第二年，男孩驚慌地發現女方竟然懷孕了。這瞞也瞞不住，很快被雙方的父母知道了，雖然父母們非常的憤怒，但是畢竟是一條小生命，最終只能讓兩人放棄了學業，回到家中結婚生子。

20歲的年紀，同齡的孩子正在大學中求學，在積累自己的社會經驗，而他們卻已經成了寶爸寶媽。寶媽從此之後就被孩子所累著，而寶爸因為沒有學歷，在社會上也無法立足，經濟來源非常匱乏，在孩子生下來之後，一家人苦巴巴的過著日子，這種沉重的壓力和負擔，讓兩個年輕人臉上失去了往日的歡笑。

大家看一看這麼早的年齡結婚生子，不但影響了自己，而且給家人也帶來了很多麻煩。

第一、經濟負擔的增長是最大的一個方面

小生命一聲啼哭來到這個世界上，從他到來的那一天，就直接給一個家庭帶來了經濟負擔。不論是成長還受接受教育，我們投入的資金那可是大把大把的。

而過早當了爸爸媽媽的年輕人，他們沒有穩定的收入，此時自己還需要父母不斷的接濟，那麼這個孩子該怎麼辦呢？怎麼去撫養他長大呢？所以勢必會對整個家庭造成一種沉重的負擔。

第二、影響女性身體的健康

這一點大家都是有目共睹的，我們從生理衛生知識中就知道，女性最佳的生育年齡是在25~30歲之間，在這個時期女性的身體才能發育成熟，各個機能也達到了一生中最高的巔峰。

在這個年齡段去生孩子，無論對於自己還是對於孩子，那都是非常好的，而且學業有成，工作也穩定，在經濟做後盾的情況下，也能安然的在家生孩子坐月子。

第三，間接地影響到孩子的未來

父母是孩子的第一任老師，他們在降臨到這個世界上之後，也是從父母那裡先獲取知識。

而此時寶媽都沒有做好當媽媽的準備，懵懵懂懂的，尚且需要去接受教育，在撫養孩子的過程中，怎麼能夠把良好的教養和知識傳遞給孩子呢？在家庭教育方面如果非常匱乏的話，在起跑線上你家的孩子已經遠遠落後了。沒有堅實的基礎，他們的成長之路又怎麼能穩固呢？

所以在這裡提醒那些年輕的女孩們，為了自己，也為了孩子的未來，一定不要過早的去做媽媽。
