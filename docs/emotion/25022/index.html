經常把這五句話掛在嘴上的人，鐵定是在給你洗腦，交往要謹慎

2019-12-23 95條評論 2,616次閱讀 872人點贊

如果一個人經常把下面這5句話掛在嘴上，一定是在想辦法給你洗腦，大家一定不要跳進被別人洗腦的陷阱裡面去，很多人不由自主地掏腰包，就是因為被洗腦之後，自己變成了一個單純而又弱智的人，跟隨著人家的思維往前走，他把你的口袋給掏空了之後，幡然醒悟，但是為時已晚。

【1】人的潛力都是被逼出來的。

這句話告訴你，不是說自己掏不出錢來，而是沒有逼自己一把，而是沒有想更多的辦法，經常把這句話掛在嘴上的人，實際上就是8年腰包里的錢給你逼迫出來，他們把這當作潛力，實際上就是給你洗腦用的，甚至還會用生活的一些現象去解釋，解釋的越多，他的心思就越明顯。

【2】如果你簡單，這個世界就簡單。

這句話也不是知道坑了多少人，如果你變得簡單，這個事件就變得簡單了，大家記住，這個世界絕對不會因為你的認識改變而改變它的自然規律，這個社會很複雜，不會因為你把它想的簡單，它就真的簡單了 這句話也是極具殺傷力的，很多人受不了，最後被洗腦。

【3】懷才就象懷孕，早晚會被發現。

這句話也有很大的問題，是奉勸那些有才華而無法發揮的人，實際上，懷才和懷孕真的區別很大，懷才一輩子沒有被發現的大有人在，而懷孕是不可能不被發現的，大家千萬不要被這樣的言語給左右了自己的思維方式，那是一個很恐怖的事。

【4】有情人終成眷屬。

如果有情人終成眷屬，這個社會就不存在那麼悲歡離合了，也就沒有七夕相會這樣的一個傳說了，而事實恰恰相反，有情人難成眷屬，你深愛過的人，取到手了吧，深愛你的人，你願意嫁過去嗎？活生生的現實，讓我們不得不去面對這個琢磨不透的世界。

【5】有志者事竟成。

這句話在很多人的心裏面奉若神靈，只要自己有自己事情一定能夠做成，一定能夠獲得自己想要的成功，實際上這也是一個偽命題，有志氣的人多了，天天發毒誓的人也多了，天天詛咒的人更不少，成功一定是極少數人的遊戲，大多數人僅僅是參與成為犧牲品而已，這才是事實的真相，如果你的才能和自己的志氣不匹配，志氣再高，也沒有任何意義。
