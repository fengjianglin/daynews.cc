隨著二胎的開放，很多中年人加入了生育二胎的行列，在這些人中，也不乏一些家庭條件困難，手中沒錢的人。

鄰居張姐就是這樣一個二胎媽媽，她的家境比較困難，老人身患疾病，一家人的開銷都靠老公一人上班維持。

今年，張姐懷上了二胎。知道張姐懷二胎後，小區很多人都說她重男輕女，大女兒5歲了，又想要個兒子。

對於張姐要二胎的行為我也很是不理解，有一天我問她，為什麼家裡條件不是很好，還要生二胎呢，張姐的答案是「不希望孩子太孤單，希望兩個孩子能在一起作伴」。

其實很多寶媽生育二胎都不是因為重男輕女，也是不想不開，她們有著自己特殊的原因。

沒錢還要生二胎，往往是出於這幾個原因

01.減輕孩子養老負擔

對於獨生子女來說，他們長大後要面臨的挑戰是非常多的，尤其是生兒育女之後，既要養育子女，又要照顧雙方老人。

而對於二胎子女來說，贍養老人方面的負擔就會減少一些，在老人生病時，也能多出一個幫手。

很多家長出於這方面的原因，在家庭條件不寬裕的條件下，又生了二胎。

02.喜歡熱鬧的氛圍

有的家長從小就有兄弟姐妹陪伴，習慣了熱鬧的家庭氛圍，覺得家庭不應該冷冷清清，多一個孩子就多一份熱鬧。

而且這樣的家長也會覺得，自己年老之後，如果只有一個孩子，恐怕不能長時間的陪伴在自己身邊，自己的生活會非常冷清。

如果有兩個孩子，那麼自己見到孩子的機會就會多一些，未來還會有孫子孫女能陪伴著自己，自己也不至於變成一個空巢老人。

03.給孩子一個夥伴

有的家長堅信，一個孩子始終是孤獨的，自己又不能時時陪伴孩子玩耍，孩子的夥伴就只有那些玩具。如果生育了二胎，孩子就有了玩伴，不至於孤獨。而且等到自己百年之後，孩子身邊也不至於連一個可以商議事情的人都沒有。

出於給孩子尋找一個夥伴的想法，家長會生育二胎。這樣的家長往往不會在意孩子的性別，他們認為性別並不是很重要，兩個孩子能在一起作伴才是最重要的事情。

04.奮鬥動力更強

有的家長雖然金錢上並不充裕，但是在得知自己懷上二胎後，便充滿了動力和對未來生活的期許。

他們堅信憑藉自己的努力，一定能給兩個孩子更好的生活。帶著這樣的信心，家長也會選擇將二寶生下來。

對於生二胎，家長們都有著各自的原因，有的人是為了讓自己的生活更有奔頭，有的人是為了給孩子尋找一個夥伴，還有的人是為了讓孩子減輕負擔。

但是不論是什麼樣的原因，既然選擇讓孩子來到這個世界上，就要給他愛和呵護，給兩個孩子公平，讓孩子在和諧的家庭氛圍中健康的長大。
