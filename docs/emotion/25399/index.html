組圖呈現一個女人成為母親後經歷的變化，張張戳心，是母親的意義

2019-12-20 2,626條評論 3,606次閱讀 1722人點贊

每一位女性在當母親之後都會出現很多的變化，無論是身體上還是生理上又或是心理上都會出現相應的改變，有一些是好的變化，也會有一些是負面的影響，但無論是哪種變化，都會促進母親的成長，並且更加能夠體現出做母親的意義。

一個女性從開始懷孕再到之後的分娩、哺乳、生育、照顧孩子以及教育孩子等等，每一個過程都體現著母愛的偉大以及做母親的意義。

最近微博上就有一位紐約的畫家將做母親的每個階段的意義都用自己的藝術觀念以及藝術作品體現出來，他通過調查一位女性從分娩到作為一位母親之後所經歷的每一個變化，第一幅圖是母親分娩時候的樣子，緊接著是哺乳、養育、擁抱、給予關愛等等過程的一些圖片，這幾幅圖片的配色都非常的具有藝術意義，通過他的畫作，也使每一位觀賞者都深深地感受到母親在這個社會中的作用以及所存在的意義。

母親自己的成長也有兩個層面。

第一層的成長是外在的成長：

如果寶媽把自己所有的心思都花在了孩子身上，那麼寶媽們一定會非常累。每一個女人都愛美，可當他們作了母親後，他們因為對孩子的負責沒有時間去打扮自己，整天灰頭土臉沒日沒夜地為孩子操心，寸步不離孩子生怕孩子在自己不留神的情況下磕到碰到。

孩子有時候會需要家長的陪伴，但過多的關心和溺愛就會讓孩子只會依賴父母，依賴他人。這如今這個競爭激烈的時代，只會一味地依賴他人孩子是沒有辦法成才的。

隨著孩子的長大，媽媽的社交圈大多都變成了孩子幼兒園的同班同學的父母，小學同班同學的父母，初中同班同學的父母，這樣也不過是為了多了解孩子的成長和情況。

媽媽們都失去了自己曾經的朋友圈，不是那麼經常聯繫，也不會經常在一起聚聚，久而久之媽媽的朋友也都會變得少之又少，或許媽媽們還沒有發現，但當他們在勞累的某一天里突然想要找一個曾經經常在一起玩耍的閨蜜聊聊天，這時才會發現，自己已經不再是以前的自己了。

何為家庭婦女，就是完全的把身心投入到家庭中，家庭婦女也會很累，他們每天去做那些無聊的家務，要照顧孩子和丈夫女兒也需要照顧婆婆和公公，就好像一個保姆一般，可能他們也會讓自己的孩子覺得自己的媽媽是一個只會幹家務的家庭主婦。

家務怎樣快捷怎樣去來，能不動手就不要去浪費時間，留出更多的時間來照顧孩子，或者善待自己。所以對於母親來說，在照顧孩子的同時更應該提升自己的外在，注意自己的形象。

第二層是心靈和人格的成長：

一般在一個人成長的過程中與父母接觸的時間最多，所以父母對孩子的成長有著很大的影響。在孩子眼中父母便是自己的榜樣，同樣孩子的性格也會隨著父母的教育方式而發生改變。

如果孩子喜歡譴責別人，很有可能是因為父母平時對他的批評過多，所以他就會養成去譴責他人的習慣，這也是推卸責任的一種表現。

沒有一個人生來便是懶惰的，也沒有一個人生來便會喜歡去依賴他人。現在很多孩子都有懶惰和依賴他人的習慣，這並不僅僅是孩子的問題，很多母親在教育孩子的時候，都會有很強的控制欲，他們會提孩子做出很多的決定，也可能出去溺愛不忍心讓孩子獨立去完成自己的事情，久而久之孩子也就會養成依賴他人的習慣，還有很多孩子都會存在逆反心理，而這種心理在孩子青春期時最為常見，也可以稱作是叛逆期。

但母親們有沒有想過孩子為什麼會有這麼嚴重的逆反心理，很多母親都會用對孩子好的借口去強制性的去讓孩子做自己不喜歡做的事情，然而每個人的心理都會存在對抗，父母們對孩子太過強制就會激發孩子對抗的一面，所以孩子才會處處與你們反著來。

很多母親發現孩子變得越來越自私，不為別人考慮只在意自己的想法。孩子之所以會變得自私很大一部分原因其實在於家長，因為母親的溺愛導致孩子無所顧忌，覺得自己才是最重要的，有了這樣的心理，他們就會變得越來越自私。

所以各位母親教育孩子的時候一定不能對他們太過溺愛，該嚴厲的時候嚴厲，該慈愛的時候慈愛。這也就是母親該提升自己的內在，讓自己變得更加優秀，孩子也會受到她們的影響。

以上就是關於有了孩子之後母親的成長與變化的兩個層面的大致內容的總結了，看到這裡的寶媽們，你們都是如何看待這件事情的呢？在你們的身上有沒有哪些變化呢？
