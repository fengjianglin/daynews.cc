父親的教導：說話要懂得有分寸感，否則人生會毀在一張“破嘴”上

天天要聞
2019-12-23

找我分享你的故事，點擊右上角關注，你就是我的人了！

相信在現實生活中，我們都能夠遇到這樣子的一些人，他們本身的人品是沒有什麼問題，但是就是因為情商太低，往往說話沒有分寸感，和人聊天交流的時候很容易出嘴傷人，因為自己說話沒有分寸感，容易得罪他人。往往不懂得什麼該說，什麼不該說的人，即使工作能力再強，也很難受到重視。

​每個人生活在這個社會上都是非常不容易的，很多時候總會遇到一些失敗和挫折。有的人在遇到失敗和挫折的時候，會懂得反思自己，讓自己下次不要犯同樣的錯誤。但是卻有的人，總是喜歡把自己的苦惱掛在嘴邊，逢人就說，說話完全沒有分寸感，口無遮攔，這樣會讓人對你產生厭煩感。

最近有和父親通過電話，也有和他聊起自己身邊有的人因為不懂得掌握分寸，說話總是口無遮攔，得罪了很多人。聽完我說的話之後，父親便開始教導我：做人做事都要掌握一定的分寸感。尤其是說話，很多時候是禍從口出，即使能夠辦成的事情，也會毀在一張不會說話的嘴上。如果想要自己的人生，自己的生活能夠一帆風順，就要懂得什麼該說什麼不該說。

說出去的話就像潑出去的水很難再收回，雖然很多時候我們想要向他人傾訴自己的煩惱苦惱，但是下面這些事情，卻只適合藏在心裡，不能說出去。

​自己家庭中的矛盾

俗話說得好，家家都有一本難念的經，每個家庭都會有每一個家庭的難處。很多時候，每一個家庭多多少少都會存在一些矛盾，一些紛爭，但是這一些矛盾和紛爭都應該只出現在家庭內部，而不應該拿到外面向外人隨意地訴說。畢竟自己的家裡有什麼事情可以關上門解決，沒有必要讓他人知道的過多。如果一個人不懂得掌握分寸，自己在家裡面受到了委屈，就拿到外面到處說事兒，如果哪一天，這些事情又傳回到了自己家裡人耳朵裡面，就會非常的傷害家人和自己的感情，破壞家庭的和諧相處。

家庭裡面的一些矛盾即使讓自己心裡感到委屈，讓自己覺得難受，也不要輕易的告訴外人。畢竟家醜不可外揚，別讓自己沒有分寸的嘴，去損壞了家庭的名譽，破壞家庭的和諧，最後只會是得不償失。

​對同事或領導的不滿

不要覺得只是簡單的幾句話，不會對對方造成什麼樣的影響。要知道語言的傷害或許比身體上的傷害更加讓人難受，讓人留下更深的陰影。所以我們在面對外人的時候，說話做事都應該做到三思而後行，尤其是在工作上。即使自己對某一些同事或某一位領導的一些行為做派看不過去，也千萬不要向外人對此訴說自己的偏見和不滿，因為一旦你說的這些話，由外人傳到了當事人的耳朵裡面，就會嚴重影響到你和你的同事領導之間的關係，讓你自己的工作很難進行下去。

說話應該要懂得掌握分寸，否則自己什麼時候得罪了他人都不知道，因為自己說了不該說的話，平白無故惹來一些麻煩，就很不應該了。

​自己的軟肋弱點

這個社會是非常複雜的，各種各樣的人魚龍混雜，所以在與人交往的時候應該要謹慎，不要把自己的所有事情都告訴他人，尤其是自己的軟肋，自己的缺點。如果一個朋友是真心對你的，你和他說這些，他自然不會往外說。但是如果你們的關係一般，你向他人說出自己的軟肋和缺點，這兩樣都是屬於自己非常隱私的事情，萬一哪一天你們友誼的小船鬧翻了，指不定，他人會利用你的缺點和軟肋來傷害你。

為了保護好自己，讓自己免受傷害，就要將自己的軟肋弱點藏在自己的肚子裡面，不要告訴其他人，以免招致禍端，自己還傻傻的不知道怎麼回事。

​情感結語：

說話是一門藝術，掌握好這一門藝術，能夠讓你在生活上在事業上事半功倍。但是如果說話做事總是口無遮攔，沒有分寸感，就非常容易得罪人。在當今的社會上，想要有更好的人生，想要自己的人生，沒有太多的阻礙，就要懂得管住自己的嘴巴，懂得什麼該說什麼不該說，做事一定要三思而後行。
