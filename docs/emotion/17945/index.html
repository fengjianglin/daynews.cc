74歲大爺用粉筆在地上寫詩懷念亡妻，每句都有你字，感動不少網友

2019-12-16 502條評論 2,003次閱讀 314人點贊

愛情是什麼？2000多年前，古人用一句「執子之手，與子偕老」，道出了我們對愛情的終極追求。此後，時光荏苒，歲月變遷，很多東西變了，但世人對於愛情的追求卻從未改變過。

唐人李商隱的愛情是「春蠶到死絲方盡，蠟炬成灰淚始干」，宋人的柳永愛情是「衣帶漸寬終不悔，為伊消得人憔悴」，金人元好問的愛情是「問世間，情為何物，直教生死相許」。到了現代，愛情則成了徐志摩筆下「最是那一低頭的溫柔」，席慕容也筆下「會開花的樹」，舒婷筆下的《致橡樹》。

這是詩人筆下的愛情，風花雪月，美不勝收。但對於平凡的我們來說，愛情是柴米油鹽，是回到家後沒見到對方的一絲失落，是在人群中一眼就能找出那熟悉的身影。這種平凡人的愛情之美，在本期的這位大爺身上，被體現得淋漓盡致。

這位農村大爺名叫倪小洪，是河南偃師的一位普通農民，今年74歲。他38歲與愛妻結婚，相伴32年後，妻子於4年前離世。在這4年里，他寫了80多首情詩懷念髮妻，隨便讀一首，都令人淚目。今年1月份，他又用粉筆在地上寫下了這首《思念》，全詩句句發自肺腑，短短几句每句都有「你」字，感動了不少網友。讓我們一起來讀一讀：

《思念》

作者：倪小洪

孤獨思念你，夢中見到你。

失去知音你，無人替代你。

她人不如你，心中只有你。

好女就是你，下輩還娶你。

生前我愛你，和睦我和你。

勤勞就是你，今生不忘你。

從專業的角度來說，這首詩寫得並不算高明，它更像是一段順口溜，但讀來卻朗朗上口，一氣呵成。全詩6行12句，每句以「你」字結尾，共有12個「你」，他是老人對亡妻在訴說著衷腸。

詩的前兩句，寫的是對髮妻的思念。日日思君不見君，幸得夢中能相隨，對老人來說餘生最大的幸福或許就是夢中與愛人相見了。接下來的4句寫的是妻子在自己心中無可替代的位置，兩位老人共同撫養了4個女兒，期間吃過的苦頭、為女兒操過的心，都是他們愛情的見證。

女孩舉行狗剪影

老人對妻子的評價寫在了「好女就是你，下輩還娶你」10個字當中。一個女人終其一生，能得到心愛的男人口中、心裡的一個「好」字，她就是一個成功的妻子。對方若能再說一句下輩子還願再結緣，作為妻子又以是何等幸福。這種幸福與卑微無關，甚至與奉獻無關，只與愛情有關。

最後老爺子用了一個「勤勞」來評價妻子。這兩個字里，到底包含了哪些內容只有他自己知道。是幹完農活回到家桌子上熱騰騰的飯菜，是鞋襪破掉後妻子總能第一時間發現並補上，是冬日裡她在河邊洗衣服，還是她在屋前屋後教女兒走路？這是屬於老爺子一個人的回憶。

板凳在公園在秋天

愛情會讓一個普通詩人變成頂級的詩人，也會讓一個普通人變成詩人，這首詩就是最好的證明。倪大爺和愛妻的故事並不偉大，他們只是萬千戀人中的一對兒，但細細品來卻足以令人動容。「執子之手，與子偕老」兩千年前這10個字，是我聽過最美的情話。大家覺得大爺的詩寫得怎麼樣？歡迎討論。
