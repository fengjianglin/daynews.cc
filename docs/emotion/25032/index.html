據說吃到一起的人，才能睡到一起

2019-12-22 223條評論 3,403次閱讀 2461人點贊

前一陣子朋友聚餐，我心血來潮問身邊的朋友：「你的老公是怎麼追到你的？」

「也沒怎麼追，我們是相親認識的，初次見面我們一起約在餐廳吃午飯，出於禮貌我讓他點的菜，結果他點的每一道都是我的心頭好，瞬間好感倍增。

然後，我們就有了第二次約會，第三次…再後來，他開始每天給我做飯，我就每一頓都吃的精光。」

這回答聽起來很矯情，但是吃到一起才能愛到一起的道理並不假。

孔子在《禮記》里也講：「飲食男女，人之大欲存焉。」也就是說，凡是人的生命，離不開兩件大事：飲食、男女。而當這兩件事碰撞在一起，一不小心就擦出了愛情的荷爾蒙。

電影《喜歡你》中周冬雨飾演的女主，是個趣稚搞怪、靈氣滿滿的創意廚師，而金城武則是一位掌管跨國經濟體的高富帥，刻薄挑剔，什麼都不愛，就愛吃。

一道女巫湯讓金城武喜歡上了周冬雨做的菜，於是紙條開始在空中亂飛，他們在隔空斗菜。金城武的菜單只有周冬雨懂，周冬雨想說的話藉由菜式都被金城武悉數解讀。這一對一點都不搭調的男女，就這樣因為美食而產生了情愫。

最喜歡的片段是金城武為了把周冬雨找出來，讓一個小正太去挨家敲門問：「龍蝦三吃哪三吃？」周冬雨想都不想就回答了。於是金城武成功找到她家並吃到了周冬雨做的海鮮飯。再後來從不跟別人一起吃飯的金城武，最後愛上了跟她一起吃飯，愛上她這個人。美食之於他們是愛、是救贖、是一切美好及溫暖的集合。

然而生活千姿百態，美食的種類也是千千萬萬，要一起生活一輩子的人，口味也要不相上下才對。

我見過一對情侶，吃飯時他們坐在隔壁，兩個人是從網戀奔現，那天是他們第一次見面，但是結局卻是不歡而散。

服務員剛把菜單拿上來，男方就不管不顧地點了幾個菜便將菜單還給服務員。沒有詢問女生任何的意見。

女生起初只是皺了皺眉頭，等飯菜上來的時候面對一桌子都是重口味的菜時，女生輕輕問了一句「沒有清淡一些的菜嗎？」

男生卻說自己不喜歡吃清淡的，所以沒有點。這頓飯看出來男生吃的很快樂，而女生吃的很痛苦。

吃完飯後，男生提議去看電影，女生借口身體不舒服，提早結束了這場約會。

我想他們怕是過不了幾天就會分道揚鑣了吧。畢竟一個偏愛白米飯，一個只吃大饅頭，一個愛火鍋的麻辣酸爽，一個鍾情養生清湯鍋，一個迷戀海鮮披薩，另一個信仰驢肉火燒的兩個人，很難生活在一起吧。

三毛說：「愛情，如果不落實到穿衣、吃飯、數錢、睡覺這些實實在在的生活里，是不容易天長地久的。」

生活不過就是衣食住行，一日三餐日復一日。後來，我們有了喜歡的人，約吃飯成了最不容易被拒絕的借口。再後來，一起去菜市場挑選又肥又大的魚頭，一起在廚房裡研究黑暗料理，享受著魚頭湯的美味，討論著明天吃什麼。

這時候，我們好像也懂得了那句歌詞：

「是誰來自山川湖海，卻囿於晝夜廚房與愛。」

廚房與愛是聯繫在一起的，在油煙和炒鍋前氤氳過，於是懂得生活是一件艱難卻又溫馨甜蜜的事。

而我們懂你的浪漫，更懂你的辛苦，華日家居定製櫥櫃系列，用最專業貼心的設計，給您帶來最舒適的廚房體驗。
