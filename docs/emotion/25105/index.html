女兒找對象結婚，這位爸爸只有兩個要求，真的很有遠見

2019-12-22 3,369條評論 4,705次閱讀 908人點贊

文|文兒

很多人都說，女孩子找對人談戀愛和結婚是很重要的，一個好的對象會讓一個女孩越來越優秀，而一個糟糕的對象可能會讓女孩子陷入萬劫不復之地。所以每個女孩都需要一雙慧眼和玲瓏心，看清男人的面目，不要讓自己在愛情和婚姻中栽太大跟頭。而女孩們在找對象上的聰明和睿智，並不是天生就有的能力，與父母的教育息息相關。我一直覺得我的戀愛和婚姻都挺順利的，在戀愛和婚姻上幾乎沒走過什麼彎路，這多虧了爸爸和家裡的長輩們的一些人生經驗。我把這一切都要歸功於爸爸，關於找對象，很早他就給了我一些囑託和要求，所以才有了我後來的順遂。女兒找對象結婚，這位爸爸只有兩個要求，真的很有遠見。

一、找對象的時候，對方有一定的經濟實力很重要，但重要的要求是對方品性要好，心術要正

爸爸很早就告訴我，找對象應該看對方的經濟條件，不要妄想著高攀，也不要太過低就。因為經濟基礎決定上層建築，經濟差距很大的家庭，生活環境和思想觀念都會有很大的差異，有些骨子裡的東西是很難改變的。除非你能保證你能和他達到一定的高度，或者願意讓自己落到和他一樣的水平線。不然這些差距，會讓你們相處起來矛盾重重。

當然最重要的是一個人品性要好，心性要正，一個男人真正重要的是腦子裡的智慧，肩膀上的責任和心裡的擔當，最後是骨子裡的教養，那些外在的都是附加值。一個人愛你或不愛你、是會變的，人和人的感情很容易變，找一個正直善良有責任心的，即使以後不愛了，也會對家庭負責，也不會半路拋棄你，也會給你體面。心術正、正直有擔當，愛你的時候會珍惜你、不愛你了也會給你體面。同樣的，一個善良正直的人不會讓任何一個人狼狽不堪。選朋友，選愛人，人品都應該是首要考量。人品決定了各種關係的下限。

​二、結婚前一定要先能養活自己，不管什麼時候結婚，跟誰結婚，要求女兒要能養活自己

後來和對象打算走上結婚日程的時候，爸爸就說，你的工作沒有穩定下來我是不會同意的，你應該把工作穩定下來，這是底氣。也算是在爸爸的要求下，在未來老公的監督下，考上我理想中的單位。後來懷孕生子，才知道爸爸當初是多麼明智和有遠見，有了穩定的工作，是給我最大的保障。因為有自己的工作和收入，我在婆家有一定的家庭地位，懷孕產假期間工資照拿，產後有生育津貼，真的是避免了很多的一地雞毛。婚後日子平穩而有序地前進著。

女孩子任何時候都要有養活自己的能力，尤其是結婚後，只有人格獨立才能談平等，而經濟獨立是人格獨立的保證。女孩子什麼時候都要保證能養活自己，不管什麼時候結婚，跟誰結婚，首先要能養活自己。
