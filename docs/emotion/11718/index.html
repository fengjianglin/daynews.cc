一中年網友哭訴：老房子拆遷賠款，全家人卻因此反目成仇

2019-12-11 1,125條評論 2,344次閱讀 1233人點贊

大家好，這裡是《婚姻百態》

「看婚姻百態，品百味人生」

前段時間從老家得到消息，我們家裡的老房子因被徵用要拆遷，我把這個消息告訴家人，大家都十分開心。拆遷老房子，自然免不了一筆豐厚的拆遷款，這筆錢不少，分下來都是一筆不小的數額。

我們家裡人口多，那時候政策不一樣，家裡我排行老三，還有一個大哥，一個二哥和一個妹妹。老爹很早就走了，老媽還在大哥家裡住著，我們三兄弟會每年輪著接老媽到自己家裡贍養；妹妹也會出些心意。平日里逢年過節的，也都會相互串門，看看後輩，可以說，如果沒有這次拆遷老房子的事情，我們整個大家庭都是一片其樂融融的景象。

問題就出在這個拆遷賠償款上，確定這個事情以後我們一大家子人聚在一起商議如何分配。我的建議是我們三兄弟和老媽平分90%，小妹拿剩下的10%。

大哥第一個跳出來反對，他的理由是從小到大對爸媽的付出最多，甚至小妹念書，房子翻修都是自己出的錢，付出得這麼多，現在分錢了就說平均分，這不合適。

二哥也反對，爸媽沒有和自己脫戶，還在自己的戶口下，老房子也是自己一家人在打理，爸媽還在的時候，全是自己在照顧他們，多少無形的支出？現在分錢就說平均分，這不合適。

小妹也很反對，雖然是嫁出去的女兒，但是這些年出的錢不比你們這些當哥哥的少，帶老媽出去旅遊一趟就花了好幾萬，出的不比你們少，現在分錢就只有10%，這不合適。

可以預見，一大家人各說各的，吵得很激烈，拍桌子摔凳子也沒有吵出個所以然來。當時老媽也在，她說自己半截身子都在土地裡面了，錢跟紙沒有區別，她那一份不要，全部分給我們。但是我們的焦點根本就不在老媽身上，她的錢那是大家一致認為是她應該拿的，幾個兄弟姐妹之間，這才是誰都不服誰。

大哥今年55歲，年輕的時候就爭強好勝，逢年過節給爸媽封的紅包總要比我們多一點。二哥53歲，為人實誠勤奮，但是精明，吃不得虧。我49 歲，小妹46歲，我們四兄妹加到一起都250歲了，還差點動起手打人，也不怕其他人笑掉大牙。

隔天我接到了好多個侄子侄女的電話，語氣還算尊敬，但說話都是一個樣，充滿了硬直。拆個老房子，全家搞得烏煙瘴氣。

最後我的那一份和小妹的對換了，所有人也都不在說什麼，我兒子不服氣，非要去爭。我制止了他，你們有手有腳的，凡事要靠自己去拼，爭這些東西有什麼意思，爭贏了沒意思，爭輸了更沒意思。
