導讀：職場上，很多深刻道理都是在吃過虧以後，才徹底明白。在過去，我在同事眼裡，是個老好人，對任何人都挺信任。當然，老好人並不代表就沒有能力，相反，我在市場部因為業績突出，還多次得到老闆的表揚。

因為我對同事沒有戒心，一次在競爭主管位置時，無意中對同事表達了對自己領導的不滿情緒，結果第2天就被領導找我談話，當然，那一次主管也沒有當上。但這件事讓我學會了，和同事相處，要留個心眼。

我有個同事叫謝小梅，市場唯一的女生，來公司時間不長。不得不說，女生在職場也還是有些優勢，因為長的還算不錯，又愛撒嬌，有時候工作完不成，都會有男同事主動上去幫忙。即使工作中犯錯，跟其他員工求求情，總能讓其他人願意擔責，而外出聚餐時，她從來沒有掏過錢。可以說謝小梅把自己作為女性的優勢發揮到了極致。
這種人，情商其實是挺高的，也善於利用別人。因為我太好說話，有一次我為公司想出了一個點子，為公司節約了10幾萬，老闆為了鼓勵大家，單獨給我獎勵了1萬塊錢現金。

那時候謝小梅來公司5個月，看到我拿了獎金後，一個勁的誇我能力。我作為一個男性，當然也很享受別人的誇獎。到最後，謝小梅突然跟我說，她父親身體身體不好，這段時間在家住院，問我是否能借五千元給她，語氣很誠懇，也讓人同情。

我相信在這樣才情況下，很少會有男生會拒絕，更何況自己手裡確實有錢，她拿到錢後，承諾30天之內還。在前2個月，我都沒有想太多，直到第3個月的時候，剛好弟弟上大學需要學費，才想起來她還有5000塊錢沒還。找她要的時候，她又很可憐的樣子，說自己沒錢。我雖然不高興，但又不能逼她還錢。

有一次我休年假，回來的時候發現謝小梅已經辭職了，我感覺有點不妙，還有5千塊錢呢？我打電話給她，她也確實接了電話，但就是一句話：沒錢。我不甘心，每過幾天就會給她打電話，最後謝小梅乾脆不認賬了，說自己沒借過錢。當初借錢給謝小梅的時候，根本就沒有寫過借條，現在可真是鬱悶了。後來朋友教我一招：打電話時，語氣好一點，然後錄音，我也照做了，果然她在電話里也承認了。

因為都在一個行業，我通過朋友打聽到她新入職的公司，這時候已經過去300天了，最後我給她發了9個字簡訊，她主動把錢還給我了。簡訊內容是：通話錄音已發你郵箱。

現在想想，如果不是朋友教我一招，我還真要不回這錢。
