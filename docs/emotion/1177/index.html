兒子瘋了，相繼離世，不是機緣巧合，而是上天註定。

2019-12-07 0條評論 23次閱讀 0人點贊

摘要

接著上一篇文章說吧，父母因意外去世以後，安葬完他自己的爸爸媽媽，成績優秀的他，最終選擇了離開校園。

接著上一篇文章說吧，父母因意外去世以後，安葬完他自己的爸爸媽媽，成績優秀的他，最終選擇了離開校園。

爺爺奶奶早就不在了，接著父母也過世了，這對一個高中生來說是一個天大的打擊。換做任何人都承受不住的，現在和他有點血脈關係的親人，就只有自己的大伯和幺叔，其他再也沒有任何親人了。當然他自己還有一個年邁的外婆，但是外婆人老了，很多時候出行都不方便，也不能照顧他。

父母喪事結束之後，所有親人，認識的人，都相互勸他一定要完成學業，就算你的父母不在了，你還有我們。我們能幫助的都會幫助，你上大學的費用一切都不用擔心。但是他最終都沒有聽取任何人的意見，自己孤身一人呆在屋子裡三天兩夜，肚子也沒有進入一顆米。

大家都擔心他，後面村裡面的人給上面反應此事之後，安排了幾個人上門勸說。經過他們幾個小時的安撫和勸說，他答應返回學校讀書，至於讀書費用上面說了，都給他全部補貼和報銷。說實話，有這樣的政策我也是被感動的。

近看只有十五天就得高考了，還在痛苦之中沒有走出來的他，是否能戰勝最後的十五天，這十五天對於任何人也就是一瞬間就過去的，對他來說，可能就像度日如年。放不下心態的他，去上第一天課，第一節課都還沒有下課，就跑出來了了。此時此刻，或許他真的對讀書毫無意義了。

他回到家裡面，就是禁閉自己，不想見任何人，一個人在屋裡，誰說都不管用。每天就喝一點兒水，大伯和幺叔叫他吃飯從來都不理。可想而知，他要想從傷心中走出來，真的很困難。現在已經沒人勸他去讀書了，只想讓他吃點飯，再這樣可能人就沒命了。

到了高考了，那天你總算出來了，終於見到他的人影，不然已經快半個月沒見到他出門過。但是誰也沒想到，這時候他居然神志不清了，遇到誰都叫自己的爸爸媽媽，還不停的說著「我要好好讀書，我要去高考，考一個好學校」，說完以後，還喜歡大喊大叫，這時候所有人都知道了，他已經由於傷心，痛苦導致自己精神分裂，才出現的癲瘋，想想真的好可憐。

瘋了之後，從此沒有人再見到過他了，直到去年開年的時候在村裡面都沒聽到過他的消息。真的，這麼可憐的人，走到哪裡又有誰會放心啊，他自己也沒有電話，。他大伯們都聯繫了警察幫忙尋找，也是經過個一個多少都沒有一點消息。過了幾天，那邊打了電話過來，說人已經不在，餓死在山坡上了，離安葬他父母那裡不遠處。我也不知道這個是不是機緣巧合，還是上天註定，我自己承受不到這種家庭的結局，真的寫完了我自己已經流淚了。

父母離開了世界，他也相繼離開了世界，這樣的命運，你我他任何人都想不通，我只想說一句：安好，願天堂沒有意外。
