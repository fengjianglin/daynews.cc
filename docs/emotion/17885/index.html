越是沒本事的子女，越喜歡在這些事情上斤斤計較，讓父母很為難

2019-12-17 772條評論 2,085次閱讀 984人點贊

文|文兒

家有子女，每個父母都希望他們手足互助、相親相愛。尤其是現在鼓勵生二胎，很多父母都是想讓倆孩子作伴，以後長大互相幫襯、互成親人。如果是這樣，那麼父母養再多孩子，受再多累都是值得的。

不過，多子女家庭也有發愁的時候，那就是子女之間可能相互爭吵和埋怨父母偏心。尤其是長大後，越是沒本事的子女，越喜歡在以下這些事情上斤斤計較，會讓父母很為難。

財產分配

就像我老家鄰居，有兩個兒子，哥哥有出息在外地打拚，弟弟沒本事在家裡混日子。這個沒本事的兒子，不是說父母天天跟著他吃飯把他吃窮了，就是說父母給哥哥家的地分多了，甚至連哥哥的宅基地都想霸佔了。總之，就是凡事都斤斤計較，想盡一切辦法多分父母一點財產。

或許，就是因為自己沒本事，才會惦記父母的財產，甚至還會霸佔兄弟姐妹的財產。相反，有本事的子女，自食其力，眼光長遠，根本就不會惦記家裡的財產分配。可是，作為父母卻很為難了，因為都是自己的孩子，父母也想盡量公平。可一個孩子咄咄逼人斤斤計較，一個孩子凡事忍讓有時也會抱怨，父母自然很為難。

幫襯帶娃

家有多子女，大概父母都要幫襯帶娃，但越是沒本事的人越喜歡在這件事上斤斤計較。比如，經常抱怨父母給另一家帶孩子帶多了，或者說「你們給他家帶娃，也要給我帶」、「你們偏心，捨得給他家孩子花錢，不給我們花錢」、「不給我帶娃，就不給你們養老」……看這種計較，多麼理直氣壯，但其實父母有義務幫子女帶娃嗎？只有沒本事的子女，才會在這件事上斤斤計較。

因為自己沒本事，處理不好帶娃的事，才會把娃丟給老人，更會在帶娃問題上跟其他兄弟姐妹斤斤計較。如果自己有本事，有錢，生完孩子完全可以請人帶娃或者自己帶娃，根本就不需要麻煩老人，也不需要在帶娃的事上跟其他兄弟姐妹鬧不愉快。

​給父母養老

說起養老，這幾乎是多子女家庭普遍存在的難題。因為一個和尚有水喝，再添一個兩個甚至更多和尚，就可能抬水、沒水喝了。子女給父母養老也一樣，只有一個孩子，雖然養老辛苦，但不得不擔負責任。如果多子女家庭，大家可能你推我我推你，養老相互推諉。而往往沒本事的那個孩子，是最斤斤計較的。

沒本事的子女，會在出錢上斤斤計較，生怕自己吃虧。沒本事的子女，也會在出力伺候父母上斤斤計較，不想多耽誤一天功夫。這些，都讓父母很為難，很無奈。

所以，多子女家庭的父母，一定要注意在這幾件事上開導孩子，讓他們學會自食其力。然後，就是盡量做到公平公正，不給子女們斤斤計較的機會。網友們，你家存在這種斤斤計較的人嗎？
