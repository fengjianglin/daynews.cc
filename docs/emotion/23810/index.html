我為了5塊錢，害得班花狼狽不堪，同學聚會後，我這樣彌補她

2019-12-18 584條評論 1,625次閱讀 502人點贊

我打工多年，混得還不錯。（別聽我瞎吹，我就是那種一頓操作猛如虎，工資還沒2500的人，哈哈！）以前總是工作忙，沒有時間聯繫老同學。（混得慘哪，不敢聯繫。）

我沒有賣電動車之前，日子過得還不錯。

後來表姐拉我進了一個老鄉群，我看見了幾個熟悉的名字，是我的高中同學。一番聊天后，高中同學拉我進了一個同學群。（我求著他拉我的，呵呵。）

剛進群還默默無聞的我，後來成了群里的活躍份子，還組織了一場同學聚會。（哈哈，是班長組織的，我只是隨聲附和了一下。）看著群里從不發言的班花，我總有種莫名的惆悵。（哎呀，一不小心就暴露了本性。）

聚會的日子，定在一個冬天的周末。那幾天，群里異常活躍，隨時都有幾百條聊天記錄。

當年腦袋被驢踢了，不知道要好好讀書。

到了那天，在老婆和藹（兇殘）的微笑中，我請了兩天假去參加聚會。

（喂！提醒一下屏幕前的朋友們啊，我可不是視屏段子里，說的那種怕老婆的男人。我在家可是一點不慫啊，我在家那是……那是真慫啊！我們家大事我做主，小事老婆做主，可我們家的大事就是洗衣做飯，WC無情啊！）

到了聚會的地方，我懷著激動的心情和老同學們拉家常，心裡卻在盼望班花趕緊來呀！（朋友，記得看完一定不能點贊。我老婆也喜歡看今日頭條，要是讓她知道我寫的這些，她會去買甘蔗和榴槤，甘蔗外用，榴槤皮內服。一定要記住啊。）

終於，班花婀娜多姿（長胖了）的身影出現了，還是那麼美。班花只是冷冷地看了我一眼，我猜她一定是害羞吧。

同學來聚會，能拆一對是一對！哈哈！

酒過三巡，菜過五味，我一直沒有找到和班花單獨說幾句話的機會。好在我們還有下半場，到了歌廳後，我勇敢地（酒壯慫人膽）走到班花面前。我說：「好久不見。」她說：「還記得你當年寫的情書嗎？」

這件事她不說，我還真忘了。當年，鐵哥們大王給我5塊錢報酬，讓我幫忙寫一封情書。當時，我心裡想著班花，下筆如有神助，洋洋洒洒幾千字（我吹牛的，只有200多字），說盡了人間悲喜，感嘆了愛情冷暖，簡直是前無古人後無來者。寫完後我一再囑咐大王，不能用原稿，一定要自己抄一遍。難道，當年，那個土賊原封不動地送出去了。

我慌張地說：「你怎麼知道的？」她說：「除了末尾名字不是你寫的，其他全是你寫的。」我心中一萬隻草泥馬奔騰而過，還好大王今天沒來，不然我要親切地問候一下他的母親。

然後，就沒有然後了，班花提前回家了。

為你寫詩，為你靜止，為你做不可能的事。

從一位老同學那裡，我知道了班花對我冷漠的原因。當年我代筆的一封情書，雖然被她一眼識破，可她還是和大王好上了，最後還結了婚有了孩子，不過現在離婚了還欠著債。

看到這裡，你一定隔著屏幕在說：你也太扯了吧！

然而事實確實如此，那個時期的女孩，不會像現在視頻段子里的女孩,動不動就「有車嗎？有房嗎？有存款嗎？」

男生只要長得帥，或者球打得好，要麼看著有點壞，一般就能俘獲女孩兒的芳心。巧的是，上述三點，大王全部具備。

相遇，你亂了陣腳，我亂了心跳。

這下我總算明白，因為我的錯，害得班花現在活得狼狽不堪。每每想到這裡，心裡就十分難受。

走出歌廳的時候下起了雨，和老同學們告別後，我孤獨地走在大街上，冷冷的冰雨在臉上胡亂的拍。（千萬別信，我瞎編的。真實情況是，天沒有下雨，我走到一家小超市買了一瓶礦泉水，倒在頭上。剛倒了小半瓶，打了一陣哆嗦。哦！太冷了，受不了，還是算了吧。）

回到家後，心情久久不能平復。恰巧同事接了幾單私活，一個人做不完，就讓我幫忙。就這樣，我辛辛苦苦（偷偷摸摸）地幹了一年，攢夠了三萬塊錢。多方打聽後，了解到班花欠著一個老同學三萬塊。我找到那個老同學，替班花把錢還了。

總覺得青春欠我一個滿意，卻不知，我欠青春一個努力。

這是我唯一能幫她做的，真的沒有什麼非分之想，只是心裡有個坎，不跨過去憋著難受。我很珍惜我的家庭，縱然我說一輩子假話，這一句一定是真的。

這之後的一個月，班花給我打來了電話，我也不知道該不該接，也不知道接起來要說點什麼？

各位老鐵，大哥大姐，帥哥靚女，給我出個主意唄！在線等！哈哈！
