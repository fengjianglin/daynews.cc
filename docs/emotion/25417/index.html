人生苦短，眼光一定長遠；生活平淡，不必留下遺憾，早安

2019-12-20 2,997條評論 4,887次閱讀 2116人點贊

1、這個世上有多少種心情，就有多少種開心，有多少種不幸，就有多少種酸痛。大有大的歡欣，小有小的酸心，沒有誰能避開傷痛，獨享歡欣，也沒有誰一世悲痛，沒有一點歡欣。人生在世，沒有一種快樂，是專門為我們設計的，沒有一種痛苦，是單為我們預留的。人的一生，沒有誰事事得意，處處風光。告訴自己，淡然一些，看開一點，生活的過程中保持一

種平常的心態，淡然而簡單，你會愉快。沒人知道，痛苦會在哪個時段出現，也沒人清楚，煩惱會在哪個地方出現。我們能做的，就是做好自己，我們應做的，就是珍惜擁有。不管今後遭遇多麼大的困難，不論以後碰到多少的不快，我們都應真誠相處，坦誠相待。指責要留有餘地，批評要和風細雨，學會控制自己的情緒，相信你的生命之路會行走的更加順利。早安！

2、 人生的過程中，我們難免意見相左，不論意見多麼不同，切記不要傷人。有形的傷口易於醫治，無形的傷疤最不好治癒，一旦心靈受到傷害，癒合常常是困難的。打人不能打臉，傷人不能傷心，身上的傷易治，心上的痛難醫。生活的過程中，我們難免沒有爭吵，不論吵的多厲害，切記不要傷臉。雜草多了，水流就緩慢，雜念多，腳步就慢了。

人是心的口袋，什麼都不裝時叫心靈，裝一點時叫心眼，多裝時叫心計，裝更多時叫心機，裝得太多就叫心事。寬容是愛的基礎，理解是愛的根本，愛說白了是喜歡，更是理解。生命中那麼多擦肩，相守的能有幾人，歲月里那麼多人並肩，相知的又有多少。生命匆匆，誰能讀懂誰的心靈。在歲月的長河中，誰又能解開誰的心音。這個世界少的就是彼此了解，缺的就是相互理解。早安！

3、 原來人生難的是理解，痛的是不解與誤解 。愛的世界裡，誰都不是誰的唯一，情的海洋里，誰都不是誰的附庸。愛他就給他一些自由空間，讓他展翅飛翔。想他就與他好好地交流，親密無間。愛的天平上，最怕綁架，最怕設防，不要架著喜歡的名義，處處防備對方，不要說著愛的甜蜜，事事芥蒂。限制不是喜歡，圈定不是愛意，愛情最大的喜歡就是尊重。尊重彼此的感情，尊重相互的喜好，有樂分享有愁分擔。

人的一生，你好不好只有你知道，你樂不樂只有你明白。人生苦短，好不好都不必遺憾，樂不樂都不要失望，過得好是精彩，過不好是經歷。一輩子不長，只要你盡過心用過情，也就不必遺憾，更不必悔恨。當你開心時不要得意，傷心時不要失意，人世間悲傷的不只是你我，走過坎坷，迎面的也許就是歡樂。早安！

4、 為人處事適宜最好，千萬把握好一個度，才不能彼此產生誤會。做人不能太過精明，做事不能太過認真，水至清則無魚，人至察則無徒。任何事情，都要適宜，做人需要精明，但不能太過精明，太過精明，處處不會讓人，時時都要搶先。做事需要認真，但不能太過認真，太過認真，樣樣不能容人，事事不能包容。大家整天不能笑容面對，有的只是相互埋怨。嚴於律己寬以待人，容人之時就是寬人之處，讓人之情就是容人之心。

真正的美麗，不是青春的容顏，而是綻放的心靈。在人生旅途上保持一份童趣和閑心, 珍惜當下已有的，把所有的美好輕輕呈現。在來得及的時候去努力，去付出去在乎去愛，不謀結果，只為這過程。因為你不知道明天是否還有機會去做這一切事情。為了美好的明天，努力用心去做不讓自己後悔。早安！
