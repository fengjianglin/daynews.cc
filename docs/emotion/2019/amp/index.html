自從放棄做好人，我的抑鬱症不治而愈了

天天要聞
2019-11-24

摘要

一個大姐，40多歲了，對她大半生的了解，那是一個特別好的人，婚前對父母對姐妹，婚後對公婆，對妯娌，對孩子，誰提起來她，那都會豎起大拇指，又孝順，又會照顧丈夫和孩子。和公婆家的親戚朋友處的都很好。

一個大姐，40多歲了，對她大半生的了解，那是一個特別好的人，婚前對父母對姐妹，婚後對公婆，對妯娌，對孩子，誰提起來她，那都會豎起大拇指，又孝順，又會照顧丈夫和孩子。和公婆家的親戚朋友處的都很好。可就是這樣一位滿身挑不出一點毛病的人最近忽然和婆家娘家都鬧翻了，自己跑出去租房住，自己掙錢，誰勸也不回來。外人也不好問。只知道她不回家，孩子老人都吃不上可口的飯菜。眼看著都顯瘦了。這事吧，外人也不好問。

那天我在路上遇到她。就說她，40多歲了，也不是小年輕的了，出口氣就回家得了，孩子老人丈夫都離不開你。她滿眼沁淚的說:他們現在知道都離不開我了？以前呢，你知道嗎？以前我回娘家，洗衣做飯的永遠是我，我回去一進門父母就會告訴我哪裡哪裡都有什麼活，我就去干，而兄弟姐妹們留只管在那裡喝茶聊天，我還得候著端茶倒水的侍候。做好飯，都是她們先吃我做，等我上桌吃飯基本上都是殘羹剩飯，全家沒有一個人想起來給我留點熱飯，重要的是，我剛坐下，他們就說你先把碗筷拾掇一下，我們得喝茶打牌呢。奧，你們打牌要緊還是我吃飯要緊啊？這會，父母說話了:你怎麼當姐姐的？你不會一邊吃去，就不會讓我省心點。就這麼一句話我稱全家討伐的對象了，都說我不該惹父母生氣，這是我的錯嗎？你知道當時我多麼心寒嗎？對父母，多對兄弟姐妹，父母如果看到我的付出，就不會那麼說我，兄弟姐妹還不是看父母不替我考慮才那麼肆無忌憚的嗎？

再說婆家，當時結婚，我理解他們難，沒要房子，沒要車的。婚後還和老人一起住，侍候一家老小，才開始還覺得我是好人，理解她們老兩口。以後有錢了一定給我買大房子。妯娌呢。結婚鬧著給買了房子，人家一直單獨住著，過星期天就來吃飯，美其名曰看望老人。還不是我去買菜做飯，人家也是坐那裡哄老人說幾句漂亮，哄的老人還覺得我這個當牛做馬的不順眼，憑什麼啊？就因為我性子軟，好說話，就這麼欺負嗎？還說什麼我是好人，我是好人就該誰不得勁就踩一下啊？重要的是孩子和老公，他們被侍候慣了，也把我付出當成理所當然，從來沒想過理解我的累，那天在婆家和別人吵了兩句嘴，他倆還說我忍忍不就行了。沒一個替我說話的你知道嗎？這些年。我忍忍讓讓的，都憋出毛病了，經常半夜不睡覺滿屋子裡走，醫生說我得了抑鬱症，老公和孩子還說我得抑鬱症就是閑的，閑的？我也上班，下班的家務活都是我乾的，我有那功夫閑嗎？

現在我自己搬出來住，自己上班，下班自己做點自己喜歡吃的東西。再有空就交上幾個姐妹出去轉轉，每天吃的香睡的好的。抑鬱症真不大用吃藥了。我回去？我才不回去呢，他們說離不開我，還不是找不著出力不要報酬的好人了嗎？這好人我當夠了。

聽她說完，我也無話可說。是啊，在父母面前不得寵，婆家不重視，老公和孩子都不理解了，不得抑鬱症才怪呢，我突然也覺得她真不該回去住了，你覺得呢？？
