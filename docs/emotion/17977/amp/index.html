老公嫌棄全職媽媽，育兒師用“六桶水”懲罰，太解氣了

天天要聞
2019-12-16

小韓是個全職媽媽，可是最近發的狀態都是工作，朋友們紛紛給她發消息，怎麼不在家帶孩子了？這麼問其實是有原因的，小韓在懷孕前，是單位的骨幹，但老公掙得多，家裡也不缺她那份工資，所以家人就勸她辭職，她也想把全部的精力都用在孩子身上，於是就毅然決然地辭職了。

全職媽媽當了兩年，小韓不怕苦不怕累，可是卻被老公深深地寒了心，原來，她每個月跟老公要家用的時候，老公開始還挺痛快，對她和顏悅色，但是後來就開始盤問她的賬單，怎麼這個月花了這麼多，都花在哪裡了？最後發展到讓她買什麼要小票，最後集中起來給他過目，小韓徹底受不了了，對身邊的閨蜜說，本以為全職媽媽沒有壓力，沒想到手心向上的日子這麼難熬，她一定要出去找工作，哪怕掙得比保姆還少，至少不用看別人的臉色過日子。

都說全職媽媽是風險最高的職業之一，此話不假，在《我的前半生》中，羅子君被老公一句「我養你」感動，心甘情願地在家帶孩子，可是最後，老公卻告訴她，他愛上了別人，無藥可救地愛上了另外一個女人，因為這個女人讓他感覺到被理解，有共同語言。全職媽媽跟社會脫節，是誰的錯？難道老公就沒有一點責任嗎？最後卻變成了被離婚的理由，真是無奈又可笑。

現實中，也有如此不對等的夫妻，在一期《超級育兒師》中，一個全職媽媽帶著三個孩子，他的老公是單位的領導層，媽媽在跟育兒師交流的時候，曾經一度哽咽，「我真的希望他能幫我分擔一些，有時候孩子生病了或者打架了，老公就把全部責任都推給我，說我照顧得不好」，而面對老婆的痛苦，旁邊的老公卻全程冷漠臉，甚至對老婆說，「你控制點好吧？」，給人一種「她配不上我這麼優秀的老公，真是身在福中不知福」的感覺。

育兒師面對這位寶爸時，他卻開始抱怨妻子的不懂事，我剛開始認識她的時候，是一個乾淨利索的職場女性，但是有了第一個孩子後，她發生了很大的變化。當育兒師指出，寶媽的變化都來自於老公時，這位寶爸開始不屑地笑，育兒師用了「趾高氣昂，自以為是」八個字來評價他，也決定放大招來懲罰他。

六桶水的懲罰，老公被狠狠打臉

在一個操場上，育兒師準備了六桶水，從1-6分別編了號，然後讓老公以最快的方式，把水桶從這一頭搬到另一頭，老公非常不屑，不就是搬水桶嗎，誰不會呀，他一下子拿起一號和二號水桶，飛快地朝目的地跑去。

當搬第三桶水的時候，他發現水桶里的水多起來，明顯有點吃力，育兒師在一邊催促他，為什麼只拿一個水桶？你應該拿兩個，快，趕緊的。讓人幹活還這麼理直氣壯，寶爸心裡不情願，但也不好發作。

當搬到第五個水桶的時候，裡面的水越來越多，寶爸非常吃力，開始氣喘吁吁，當搬到第五桶和第六桶時，裡面幾乎加了滿滿的水，寶爸明顯吃不消，累得直喘粗氣，育兒師的態度也從剛開始的指責變成了鼓勵，最終，寶爸完成了任務。

六桶水，其實代表了帶孩子的勞累

六桶水，前兩個裝的水非常少，寶爸搬起來非常輕鬆，這其實代表了第一個孩子的出生，媽媽帶孩子還是吃得消的。可是到了第三桶和第四桶水，這代表了第二個孩子的出生，會感覺到有些吃力，有時候會非常勞累。接著，第五桶水和第六桶水，代表了第三個孩子的出生，此時的媽媽已經完全吃不消了，她幾乎沒有自己的時間和空間，每天都圍繞著三個孩子，而老公就相當於那個育兒師，一點也不理解，還在一旁指責，為什麼只搬一個水桶，為什麼這麼慢？

聽完這個育兒師的話，老公理解了讓他搬水桶的用意，也開始正視妻子的付出和一切的變化，再沒有高高在上的指責和挑剔，原來這麼多年，妻子已經在三個孩子和家庭上耗費了太多心血，老公只看到了她不堪的一面，忘記了，正是自己的種種，才讓她變成了現在的樣子。

全職媽媽這位職業，創造的價值並不亞於男人

東京大學經濟學教授柳傳范之，將全職媽媽每天的勞作，按照勞動市場的價格進行了換算，得出了下面的結論：

一個日本家庭主婦每年的工作價值為720萬日元，摺合人民幣為42萬元，這個薪資已經達到了公司中層管理者的水平。在英美兩國，因為人工更貴，所以家庭主婦創造的價值更高。那麼在我國呢？按照保姆4000元月薪，專職司機3000元月薪摺合，一位照顧學齡前兒童的家庭主婦，每年所創造的價值，應該在15-20萬之間，而且，她對工作付出了前所未有的熱情，耗盡了大部分的體力，犧牲了社交和個人喜好，讓自己變得跟社會脫節，還要面臨老公和周圍人的不屑，這些損失，又有誰來承擔？

全職媽媽的崩潰，都是從老公的態度開始的

記得在廣州的地鐵上，一位媽媽蹲在地上放聲大哭，我生孩子有什麼用？我生了兒子生了女兒有什麼用？她拒絕了旁邊人遞上的紙巾，到站後擦乾眼淚下車，全職媽媽的崩潰，是從自我價值感喪失開始的，而這一切的始作俑者，就是自己的枕邊人，當初滿心歡喜想要廝守一輩子的人。

你不就是在家帶個孩子嗎？我在外面如何如何。是啊，一個工作滿8小時就可以想做啥就做啥的人，可以對著一個工作24小時制，工作強度無以復加的人指手畫腳，這是多麼可笑？而後者竟然可以傻到，只要前者給一點語言上的安慰，來一個理解的擁抱，就可以繼續連軸轉，不要加班費，不要年終獎。但最終，老公們連這個都吝嗇，最終換來了媽媽們的崩潰和覺醒，這裡面何來的公平和尊重？

所以更多的全職媽媽開始出去工作，哪怕掙得還不如保姆多，但至少可以理直氣壯地生活在同一個屋檐下，六桶水解決了一個家庭的困境，或許可以讓這個老公覺醒，但是更多的家庭呢？要靠什麼來讓老公理解和肯定另一半的付出？

真的希望越來越多的寶爸能夠看到媽媽們的付出，幫著分擔一下，哪怕只是言語上的肯定，全職媽媽也會甘之如飴，努力堅持下去，你說對嗎？
