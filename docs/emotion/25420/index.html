想讓每個人都喜歡我，讓每個人都滿意，自己活的很累

2019-12-20 561條評論 3,363次閱讀 1948人點贊

我的同學，熟悉我的都會說，我是所謂的成功人士，如此優秀，我在工作上可以象一個戰士一樣披荊斬棘。可是，我對待自己是怎樣的？事業成功後的多年，我仍然活的比以前還謹小慎微，唯唯諾諾，還不止，我一直在懲罰自己。我沒有愛人，沒有朋友，沒有親密關係，我一直很孤獨的生活著。我事業成功，應該是成就感，很光榮啊？可我為什麼會這樣懲罰自己，虐待自己？ 一直不知道為什麼這樣的？現在才知道，原來我如此怕背叛原生家庭，可以想像到，我的母親是個極度控制的人，我們一直被她用PUA的方式對待，背叛她有多難，多可怕。這也是近期才發現。她表面好可憐的，實際的，常常暴怒，暴打我們，貶低？那是天天發生的事，象吃飯一樣。親人，有時比敵人還可怕。 我事業成功，應該是成就感啊！我應該恭喜自己啊。

因為我就是這樣一個極度討好的人，害怕看到別人失望的眼神害怕別人不開心，害怕自己不收到肯定…感謝這一篇讓我放過我自己，覺得討好也有了新的價值，讓我不用那麼糾結可以更多一點開心，也希望同樣有「討好型人格」的我們都能接納這個糾結善良不容易的自己，更勇敢真實地說出自己想法。

絕大多數的痛苦來自於自我攻擊，排斥自己，不接受自己。接納真實的自己。 我們才會真正看清自己擁抱自己愛自己。

想讓每個人都喜歡我，讓每個人都滿意，自己活的很累。我很羨慕那些很自我的朋友，不用去考慮別人的感受，敢說敢做。

與其說我們擁有的是「討好型人格」，還不如說，我們擁有的是一種「按需討好」的能力。
