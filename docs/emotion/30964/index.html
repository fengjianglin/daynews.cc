深夜正能量雞湯:有的人笑在開始，有的人贏在最後

2019-12-23 384條評論 1,513次閱讀 255人點贊

1.縱使世界對你以荒漠相欺，也不要輕言放棄，因為在世界的某個角落，總有一個忠誠信奉你的人想要對你溫柔以待。

2.跌倒了就不要傻坐在地上，要知道傻坐著是沒用的，你頭頂的皇冠也不許你就此屈服。

3.兩個小時的電影，將最後的片尾時間留給自己，你可以沒有他，但至少你還擁有尊嚴。

4.要想別人愛你，你首先得學會自愛。

5.一朵花凋零荒蕪不了整個春天，一次跌倒也荒廢不了整段旅程。同樣，只要一步步靠近成功，失敗再多次也總會有盡頭。

6.勇敢地迎接逆境，即使不能實現最初的夢想，也會打開另一扇夢想的大門。任何時候都不要懼怕前行道路上的任何坎坷，兵來將擋水來土掩，總會柳暗花明又一村的。

7.人生是個很奇妙的東西，你永遠不知道，下一步迎接你的將會是什麼，但是夢想永遠是個不會改變的東西，只要你認定他，向著他努力，就會讓你感覺離他越來越近。

8.自我打敗自我是最可悲的失敗，自我戰勝自我是最可貴的勝利。

9.沒有一顆珍珠的閃光，是靠別人塗抹上去的。

10.就算跑最後一名又怎樣，至少我又運動精神。而且我一定會跑完全程，只要跑到終點我就成功了！

11.再苦再累，只要堅持往前走，屬於你的風景終會出現。人生就是一場漫長對抗，有些人笑在開始，有些人卻贏在最終。

12.沒有哪件事，不動手就可以實現。只要你願意走，哪裡都會有路。看不到美好，是因為你沒有堅持走下去，人生貴在行動，遲疑不決時，不妨先邁出一小步。每天都要做最棒的自己。

祝各位冬至快樂！
