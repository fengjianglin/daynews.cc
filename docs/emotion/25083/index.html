讓下屬死心塌地跟你乾的6個訣竅，學會你也能上位當領導

2019-12-22 632條評論 3,815次閱讀 1655人點贊
  • 點擊右上角【關注】霸王課頭條號，收穫更多加薪秘籍。
  • 本文共1068字，閱讀全文約3分鐘

    • 在職場上，我相信沒有哪一個管理者不想讓自己的下屬死心塌地的跟著自己干，如果沒有一些套路和方法，贏得下屬的心也並非易事。下屬也不是傻子，沒有會人很犯傻的跟著一個不懂下屬心思的領導。

    所以要想讓下屬死心塌地的跟著你干，光靠手中的權力和行政命令，讓大家做表面文章，是絕對不行的，你必須做到下面這六點，那樣個個都會死心塌地的跟隨著你哦。

    【1】有好的氛圍的營造

    作為一個管理者，兼具領導的角色，一定不要崇尚老闆文化，因為老闆文化直接禁鉬了一個良性氛圍的營造和形成，要根據自己下屬的個性特徵，設計一種合理的範圍，比如說好漢氛圍，江湖義氣氛圍，團隊協作氛圍等等。

    這個要根據你下屬的綜合特點而設計，作為管理者，不要過多的干預下屬的事情，在你的團隊之中找一個意見領袖，按照團隊整體的氛圍和風格來開展工作，在這種氛圍的帶動下，各個下屬幹起來才會有勁。

    【2】在這個平台自己不可以有任何污點

    作為一個領導者，必須潔身自好，如果自己犯了一些錯誤，有一些做法上的問題，有一些手不幹凈的問題，和下屬關係不正常的問題，等等。

    只要染上哪一條，都會讓自己陷入到被動之中，會讓下屬看不起自己，就不要去談下屬信服你的問題了，所以不要讓自己粘上任何污染，哪怕是一點點都不可以有。

    【3】未來有好的規劃

    你的下屬不可能長時間原地踏步的跟隨你，如果讓你在現在的職位上做一輩子，可能你也不願意，所以作為一個管理者，必須要多替下屬的未來著想，對下屬的未來要盡量進行詳實的規劃。

    只有讓他們充滿了想像，經過他們的努力和你對平台的搭建，他們能夠在您的手下一天一天的取得進步，不僅在工資上得到增長，而且在職位上也一步一步的往上升，誰又不喜歡呢？

    【4】良好的分錢機制

    再好的激勵措施，沒有金錢作為基礎也是白搭，所以作為一個精明的管理者，必須要懂得把錢分好，才能把下屬管好的這麼一個基本的理念。

    分錢不是一個公平的事兒，更不是一個平衡的事，而是造成一個平衡協調競爭的局面的事兒，也就是我們在第一條上面說的營造一種競爭的氛圍，才能更好的讓他們有動力去工作。

    【5】不擺架子大眾化的溝通，越土越好

    各位管理者，千萬不要把自己看得太重要，一定要放下自己的架子，走到你的下屬中間去，否則的話，你很難了解他們真實的想法，也很難感覺到他們真實的做法。

    你只有讓自己大眾化了，下屬才敢跟你說話了，只有下屬敢跟你說話才會說實話，你才能了解更多的情況，才能有針對性的調整自己的策略和方法。那樣工作效率和質量一定會更好，要有一種朋友間的關係。

    【6】清除垃圾保一致

    在自己的隊伍裡面，必須徹底的把那些垃圾給清除出去，那些思想不統一的，對自己意見一大堆的，在下面起鬨鬧事兒的，和自己的價值觀有很大出入的，等等。

    這些垃圾不清出去的話，你很難在你的下屬當中樹立起真正的威信，因為負面的情緒也是一種氛圍，如果這個範圍不能夠引導為正面，它的危害還是相當大的。

    管理是一門藝術，也是一門學問，做到領導崗位的人我相信也對這方面有一定的研究，但請你也對員工管理問題上思考多一些，可能會有事半功倍的效果哦。好的領導一定會有好的管理方式，我相信你也可以的，加油哦~
