人生在世要想要不被別人傷害、請你牢牢地記住以下7點

天天要聞
2019-12-19

明天和意外是一對冤家兄弟，沒人知道哪一個會先到來！我們要做到的就是盡量讓意外遠離我們，讓幸福包圍在我們的左右；那麼我們就要明確的知道該怎麼樣去做，才能讓自己避免不必要的麻煩發生。

一、學會靈活變通
有時候你不受待見、並不是因為你的工作能力不行，而是你不懂得圓滑之道！做人有的時候要學會像泥鰍一樣、渾身滑滑的，不讓別人輕易地抓住自己、控制住自己。若是為人處世不懂圓滑之道，必處處招人非議！圓滑是一種哲理，更是一種人生的態度！

舉個例子
當一個酒杯子保持住它圓滑的外表時，人們都喜歡把他舉得高高的，一旦它被打碎了、鋒芒畢露，人們都會對它敬而遠之，怕被他傷及無辜！

二、大智若愚
高智商的人都是裝糊塗的高手，如果你表現的太精明能幹了，那麼的光芒太過於耀眼，蓋住了周圍人的光芒、別人就不願意和你打交道！因為每一次和你走在一起，你都是主角、而他似乎沒有了自己的角色可以扮演，所以會選擇退出！
那麼何謂大智若愚？有時候我們適當的裝作自己不懂，給身邊的朋友留一點表現的機會、讓他們也可以找到存在感！或者我們可以把一些事情留給別人去做，讓他們取得一些業績、覺得和你在一起其實挺開心的，願意和你做朋友！

三、多說好話
這裡所指的多說好話，並非是讓你去溜須拍馬，而是學會在別人取得成績的時候多美言幾句、讓別人心裡美滋滋的！當別人失落的時候、多多鼓勵別人幾句，讓他們能夠找回自我、從而對你充滿感激之情！
舉個例子
很多人去應聘一件工作，或者求人辦事情的時候，因為去的人太多了，讓別人產生了選擇困難、這個時候嘴巴甜一點的就可以脫穎而出！對於當事人來說，機會給誰都是給、你把話說的好聽了，他心裡舒服了、為什麼不把名額給你呢？而那些不願意說好話的人，往往就是出口傷人、還沒有開始辦事情，別人就對他避而遠之、所以這類人經常感到生活中沒有一件事情是順心的！

四、善意的謊言
有多時候情況比較特殊，遇到了突發事件。當事人心裡一時半會無法承受，需要一定的時間來緩衝。而這個時候、如果誰心直口快的、把實話給說了出來、對方一時間無法承受這個巨大的打擊，瞬時間崩潰了。那麼在場的所有人都會覺得你這個人實在不會做人，誰都不願意和你再交往！因為他們覺得你比較冷血，沒有感情、不懂得體諒別人的難處、做什麼事情都不會顧忌身邊朋友的感受，太過於自私、只考慮到自己。

五、懂得為自己爭取
有的人覺得自己只想做一個老好人，把機會讓給別人、就會贏得別人的尊重。可是現實中往往不是這樣的，特別是在職場上你不懂得為自己爭取、不單單只去了升職加薪的機會，而且給老闆的第一感覺就是你的能力不行，不敢提出要求。所以領導認為你所從事的這份工作，一直都是在混日子，任何企業單位都不願意去，招一些混日子的人來為自己工作！所以有的時候不懂得為自己爭取，換回來的是別人看不起你、對你能力方面的懷疑。

六、做人不能太軟弱
有的人為了避免不必要的麻煩，經常告訴自己忍一時風平浪靜，退一步海闊天空。但是有個前提條件，你要看對方做的是什麼樣的事情？如果是傷害你人格或者自尊的事情，那麼這個時候並不建議你後退，因為會去傷害別人人格和自尊的人，他必將不會只傷害你一回！你忍他一時，他有可能回過頭、再來咬你幾次。我們所要容忍的都是一些雞毛蒜皮的小事、或者一些突發的事件，離開了之後就不會再有下一次！對於無理取鬧的人，無可避免的時候，適當的給他們一些顏色看看、讓他們學會規規矩矩的做人，不再敢去欺負別人。

七、學會累積財富
雖然不贊成追求榮華富貴，但是一個人不能活得太窮了、經常找親戚朋友借錢，時時刻刻為了柴米油鹽而煩惱！當一個人過的實在太窮的時候，身邊的人就會遠離他。但是這個世界上很多事情都需要團隊合作，當身邊可以合作的人越來越少時，想要翻身就變得無比困難！所以一開始我們就要學會理財，不要太過於盲目的投資，學會做自己力所能及的事情。做任何事情都切忌急功近利，因為一旦失誤讓自己輸的太徹底了，可能一輩子都沒有翻身的機會，但是世人都是嚮往美好的生活，所以沒有人願意幫助一個看不到希望的人！他們更願意去投資已經成功的人，因為風險小、回報大！

總結
做人學會保護自己，不去惹事、但也不要怕事！為人處世要學會圓滑一點，不要吝嗇自己免費的微笑，給別人帶來快樂的同時、也給自己帶來了便利。
感悟
生活就是個大雜燴，誰也不知道下一道菜是什麼？所以你要保住自己的利益，就要學會圓滑處事之道！
