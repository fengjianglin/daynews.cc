據說很多爸爸陪娃睡覺，都有這3個特徵，好笑又暖心

不過隨著爸爸和孩子接觸時間的增多，爸爸對孩子的愛也是顯而易見的，例如爸爸陪娃睡覺時，這幾個小動作就很暖心啊：

1、睡床邊

很多家庭在有了寶寶後，夫妻並沒有分開睡，而是一起照顧寶寶，中國很多家庭都是孩子和媽媽一起睡的，一旁的孩子爸怕壓著孩子，總是不敢離孩子太近，經常只是把著床邊睡覺，有時是蜷縮在床的一角，只為了寶寶和妻子有更大的空間。

2、總愛探寶寶呼吸

晚上媽媽要給寶寶餵奶，一旦睡著會睡得比較沉，而一旁的爸爸半夜醒來經常少不了這個動作：發現寶寶睡著不動，忍不住探寶寶的呼吸，再仔細觀察胸口，看有沒有起伏，確定沒事後，再看看旁邊的老婆，睡覺怎麼也一動不動？又手放在鼻孔下面，發現還有氣，這才放心，哈哈哈哈！

3、給娃蓋被子

隨著孩子一天天長大，踢被子的功力也日漸深厚，孩子爸夜裡醒來第一件事就是給娃蓋被子，也許是不放心，一晚上要醒來給娃蓋好幾次，第二天孩子爸說，晚上給娃蓋了七八回被子，妻子還不以為然呢，以為他在吹牛，其實人家真的為了孩子操碎了心！

哈哈，你家孩子爸中槍了嗎？如果全部都中，說明他是一個細心體貼的好爸爸哦！
