「血淋淋」人性圖：你是否從創可貼女孩身上，看到了自己的影子？

什麼東西都是處於一種不但流動，不斷進步的狀態，當然對於自己的朋友來說，也是需要經常走動的呢。如果長時間的沒有來往，就算是再好的朋友，彼此之間也是會出現問題的呢。所以對於自己的那些朋友，沒事的時候還是需要經常交流，就算是出去吃個飯也是可以的呢

目光所及只有自己，有時候真正困住我們的是我們自己狹隘的眼光，看任何事物都只能看到自己。這樣又怎麼會懂得別人所在想的是什麼？又怎麼會做到為別人著想？因為他眼中只有自己。

看上去這個女人手機拿著的就是這個孩子身上鎖子的鑰匙，可是她卻不能夠親自去打開這把枷鎖，女人們在有了孩子之後就相當於是將自己困在了孩子的身邊，不管怎麼說都是沒有辦法狠心的讓自己更加的自由，這也正是母愛的偉大！

每個人的一生都在追求好的東西，但是那些被我們視為珍寶的往往都被埋藏在深深的「泥土」當中，只有肯下功夫努力的人才有機會實現夢想。但這裡說的努力可不是簡單的放手去干。所有沒堅持到最後的努力都等於失敗，在通往成功的路上困境跟突然地危險肯定會有的。我們要做的就是排除萬難堅持別人所不能堅持的，最後有所收穫的才能是你而不是別人。所以如果你還沒達到目標請不要放棄，看到別人的成功也不要妒忌，畢竟人家靠的不只是運氣。

男人正拿著一塊和高級的布把自己原本破破爛爛的小車偽裝成了高級的跑車，其實這樣人真的挺多的，自己買不起高級的東西，就用一些的虛偽的手段來，滿足自己的虛榮心。
