那個陪向佐五年的女孩最後沒能走在一起

天天要聞
2019-12-21

秋來也無信，秋去亦無蹤，我們都在青春里成長，也在青春里老去。但青春是一盞燃在心裡的燈。心不死它就不會滅。

在青春里深愛著的那個人，愛對了是一生；愛錯了終究不過淪為一名過客，再無交集。

抖音里無意間看到了一段關於向佐的視頻，他說:曾經有一個女孩和他在一起五年，但是由於那個女孩想要一個小孩，而他還沒有準備好，所以後來分開了，他心有愧疚……

別人的感情我們無從插嘴，故事的始末細節也無從知曉。只是他簡短的幾句話，卻將我帶入了深深的沉思。

有多少愛情，沒有第三者，沒有異地距離，卻僅僅是敗給了錯誤的時間。

你在尋求那份安定的時候，他的心卻還在流浪。這樣的愛情怎麼可能開花結果。

思思曾經有過一段長達七年的愛情長跑，她說起那個人的時候，目光總是很深邃，好像眼裡映射出無限的憂傷，關於他們的故事，容我娓娓道來。

她們相識在大二那年，畢業時雙方家長覺得兩個人情投意合，希望他們早日成家。

那個人說:再等等吧。我們還年輕，先把工作做出點成績，我不想太早陷入家庭的漩渦。

思思同意了，雙方父母簡單的吃了飯他們便算是訂婚同居了，而婚期則是遙遙無期。

之後的四年，他們像所有待在大城市裡的人一樣努力拚搏的工作，雙方的生活偶爾也會爭執，但始終算是不離不棄。只是四年之間那個人口中從未提過結婚二字。

身邊的朋友們，都開始結婚生子，父母與好友也時常問起思思:怎麼還不結婚，別人像你們這樣戀愛的，孩子都會打醬油了。

思思總是無奈的推脫著說:還年輕，不急。

只是她的心已開始一點一點冷了下去。

在七周年時，她笑著試探的問:「我們都在一起七年了，工作都穩定了，爸爸媽媽都盼著我們結婚抱孫子呢。你該不是到了七年之癢，不想娶我了吧?」

「當然不是，我就是不想那麼早結婚」

「可是，你都要當爸爸了！哈哈，開心嗎？」

「真的？假的？思思，別鬧了，你知道的，我愛你，可是我們不應該那麼早要孩子，現在的情況你也知道，爸媽都沒退休……」

思思的心像萬蟲啃噬般被撕裂，她繼續扯著謊尋找最後的答案:「七年了，你只用告訴我一句話，你要不要這個孩子」

那個人沉默了良久之後……只是搖了搖頭……

思思想了一夜，他的反應演變成了一根細長的魚刺卡在了思思喉嚨里。她咽不下去，也吐不出來。她努力地往下咽，可是刺卻梗在喉嚨里擠不出來，她想吐出來，可是怎麼也辦不到。淚水沖刷了七年的甜蜜與溫存。

愛情里容不得一粒沙子，更何況這是一根魚刺。

思思第二天一大早便離開了這個她曾以為共度一生的男人。

七年的感情，她用一封信做了了結:

我可以在沒有你陪伴的夜晚里開著燈睡覺；

我也可以在39.3的高燒下穿梭在人潮擁擠的醫院，一個人排隊拿葯；

我甚至可以忍受別人說我像等待你迎娶的可憐新娘；

但是我不能忍受你撕裂了我最後的防線，卻說是我在無理取鬧。

只是因為我愛你，

所以我把你的夢想當做我的夢想。

只是因為我愛你，

所以我等了七年，只是我還有多少個七年可以等。

我卑賤的猶如一顆塵埃，

只是因為我是愛著你的那個女人。

也不是沒有想過要放手，只是仍然願意相信你總會回頭看我，陪我塵埃落定。

那些為你卑微到骨子裡的日子。

真的像極了一場夢。

若我再見到你，我還是那個我，你依舊是最初的模樣，時光不染滄桑，你我皆在心上，未來各自安好。

思思切斷了所有與他的聯繫，半年後，她結婚了。

她說:現在的我，很幸福。從前總覺得愛一個人要耗費所有的力氣，現在才明白，原來真正的愛是兩個人可以輕鬆自在的一起生活陪伴，那些張牙舞爪的愛永遠抵不上細水長流的這份柔情。
