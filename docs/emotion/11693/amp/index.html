天天要聞

不要人到中年，才明白孟子的“行有不得，反求諸己”

天天要聞
7 天 ago

面對人生苦海，求內勿求外，方得解脫。

《孟子》曰：「行有不得，反求諸己。」

我們做事如果沒有達到自己想要的結果，應該從自己的身上找問題。

壹

印度瑜伽修行大師薩古魯講過這樣一則笑話：

有一天，一個人偷偷的和朋友去喝酒，本來打算8點就回家，但一杯接著一杯，早就忘記了時間的流逝，當他低下頭，盯著手錶瞅了半天，才發現已經凌晨2點半了，太晚了。想起妻子憤怒的吼叫，抄起外套就往外趕，為了早點能回到家，他選了一條最近的路。

他搖搖晃晃的走著，一腳滑進了一片荊棘叢中，整張臉全被刮花了，爬起來就這樣回了家。在家門口費力的對準鑰匙，又花了半個多小時。當他終於打開門，進到浴室里，看著鏡子發現自己滿臉都流著血，翻出醫藥箱，費力的給自己上了葯，然後悄悄的爬到床上去睡覺。

早上，妻子一杯冷水澆到他臉上，他一下子坐了起來，妻子喊道：「你這個白痴又喝酒了？！ 」他使勁揮手說「沒有啊，親愛的，我一滴都沒有喝啊！」。

妻子拽著他的衣領扯到了浴室里，指著鏡子給他看，上面全是創可貼……

當你受傷了，如果你把創可貼貼在鏡子上，並不會治癒你的創傷。

《中庸》里曾寫道：「射有似乎君子，失諸正鵠，反求諸其身」。

君子立身處世，就像射箭一樣，把箭射歪了，不要去歸咎於今天的風向不對，箭有毛病，而要從自身找問題。

貳

  • 少年理想，中年危機

真正經歷過了，才知道，世界不是自己想像的那麼簡單。

年少的時候，看著外面世界的波起雲涌，總盼著快快長大，出去成就一番功名事業，自己一定獲得怎樣的成功。

但大部分人真正的在社會上歷練時，才發現生存的艱難。沒有背景，資金，人脈和優秀的思維很難獲取到想要的成功，甚至保證基礎的生存資源都很難。

只有少部分人能夠脫穎而出，其他人只能默默無聞。

房，車，家庭這三座大山壓的人喘不過氣來，好不容易生活穩定下來，卻還有可能面臨公司突如其來的一紙裁員通知……

抬頭環視辦公室里埋頭苦幹的年輕人，摸摸自己稀疏的鬢角，發現中年危機悄然來臨。

想起下個月的房貸和車貸不禁仰天長嘆：生活為何如此艱難。

我們抱怨這個世界的不公，眼睛盯著老闆的無情，同事的冷漠，物價的飛漲，家人的不理解。

開始批判周圍大大小小的事件，將自己的苦難歸咎於外面這個世界，通過宣洩著內心的憤懣來給自己療傷，換取內心片刻的滿足。

但事情不會有任何好轉，依然要面對生活的現實。

有時候我們就像那個喝醉酒的人，把創可貼貼在了對面的鏡子上卻不自知，自己的傷口並沒有得到任何治癒。

叄

  • 不念過去，不懼將來

《長安十二辰》里有一句台詞：「人活的不是一個點，人活起伏」。

不管做什麼事情，總有遇到困難的地方；不管和誰談戀愛，總有分歧的時候；不管對未來有什麼樣的期望，總有失望的時候，凡事都有周期，有波峰，有低谷。

改變世界很難，但是改變我們的內心卻只是一念之差，瞬息之間。

有一天，大偵探福爾摩斯和助手華生一起到郊外旅遊。

凌晨3點多，福爾摩斯拍醒了華生，華生半睜著眼睛說：「怎麼啦？」

福爾摩斯問：「你看到了什麼？」

華生說：「滿天的繁星呀」

福爾摩斯問：「這對你意味著什麼呢？」

華生翻身側過去說：「明天應該是個大晴天，可以曬到太陽」，並且反問福爾摩斯：「這對你意味著什麼？」

福爾摩斯緊握著拳頭咬牙吼道：「我們的帳篷被人偷走了！」

詩人葉千華說過：」人生想開點，身心自當隨遇而安。」

我們不能總希望事情按照自己的預期去發展，如果事情能夠按照每個人的期望去發展，那這個世界就亂套了，天地有造化，世界自有它的運行規律。

就像玩電子遊戲一樣，如果這個遊戲，你隨時知道對方下一步會出什麼招式，你會知道自己即將遭遇什麼，早知道自己的結局會怎麼樣，那玩不了幾遍就膩了。

所謂人生無常，福禍相依，你不知道接下來會發生什麼。

正是因為存在種種的不確定性，世界才能夠呈現出它的真實模樣。我們才能夠真正的去體驗到什麼是生活。

但，面對世界的不確定性，難道我們就只能從積極的角度去看，然後讓命運隨波逐流，飄到哪算哪嗎？

不是的。那是盲目樂觀，鴕鳥心態，對人生的不負責任。

我們無法改變過去，無法預測未來，生活唯一能確定的便是當下，要學會覺察自己的內心，做好當下的事情。

肆

  • 置心一處，止於至善

一直很敬佩譽滿全球的漫畫大師蔡志忠先生，看他的採訪，讓人知道什麼才是當下的力量。

「雖然世間萬物，我只取孤瓢一飲，一個人只要尋找他最拿手，最喜歡的，然後把它做到極致，就沒有什麼不能成功的啦。」

老先生為了作畫，每一刻都心無旁騖，保證做到最好。

曾經坐在椅子上五十八個鐘頭，90度角坐著，沒有離開過，導致屁股上留下了一道紅紅的疤痕；

曾經四十二天沒有打開過門，關在屋子裡面不出去；

曾經孤身一個人去日本四年，畫《諸子百家》。

就這樣，創造出《莊子說》，《老子說》等100多部作品，全球銷量超過4000萬冊，譽滿全球。獲得了琳琅滿目的國際大獎，並表彰他「通過漫畫將中國傳統哲學與文學作出了史無前例的再創造」。

每個時刻都做到最好的自己，做到止於至善，一個人如果有蔡老先生這般專註力，我想沒有什麼不能做成了吧。

盡人事，聽天命。即使事情結果沒有自己期望的那樣，但再也不會後悔，更不會再去歸咎其他人了。

伍

生活，從來沒有容易二字，每個人都步履維艱。

早日明白「行有不得，反求諸己」，進而做到「置心一處，至於至善」，生活便能少一份抱怨，多一份淡定與從容！

當別人向你投石頭時，不需要去抱怨世界的不公，怒火併不能改變現實。

撿起這些石頭建立一座堅實的堡壘，便可以保護自己。
