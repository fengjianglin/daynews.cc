力量法則：每個人的內心世界都蘊藏著巨大的、無窮的潛能

天天要聞
2019-12-22

讓我們開始這節課的學習吧,經過這堂課,你將了解自身所擁有的巨大潛能,通過開發這些潛能,你的生活會更加美滿,你會體驗到更多的幸福.

你想獲得更多財富嗎?那就開發你的歌富潛能;想要更健康嗎?那開發你的健康潛能;想要更快樂嗎?那就開發你的快樂潛能。但是,你需要深入探求這些潛能的內在實質,只有這樣,你才能隨意施展它們。

你不需要再去尋找這些潛能,你已經擁有它們了,只是你可能還不了解它們,不會利用它們,不能控制它們,本卷的目標就是帶你去認知這些潛能,掌控這些潛能,讓它們成為你生命中的一部分,這樣你就能夠征服所有的困難。

很多的實踐證明,付出越多,收穫越多,付出越少,收穫越少,對任何人而言,他的成功都是來自一點一滴的積累,而非偶然。

人類與其他生物最大的不同,就是人類有思想,有創造性,人類的一切都是自己創造性活動的結果。所以,一個人今天所處的環境和人生境遇定都是他過去創造性活動的結果。

人的思維方式決定人的創造性活動,我們「想什麼」才能得到什麼,不「想」就不可能得到。因此,所有能力、成就和財富的秘密就在於我們的思維方式。

我們想要做成一件事情,必須先具備相關能力,只有能力具備,才能成功。但是,我們無法運用自身不具備的能力,所以獲得成功的唯一方式就是認識並發揮自身的能力。

人們總是忽略自身的潛能,一個人要想重新認識自己,就要意識到自身潛能的存在,而要意識到自身潛能的存在,就要知道,一切能力都源於自己的內心世界。

年輕人適合黑髮打坐

雖然每個人的內心世界都不可觸摸,但都的確存在,而且蘊藏著強大的力量.這是一個由思維、感覺與力量等元素構成的能動的世界。

思維統治著人的內心世界。如果我們能深入探索這個世界,就能發現使我們困惑的所有問題的答案,也可以找到產生這些問題的原因。一旦我們掌握了這個內在世界,所有能力、成就與財富的終極法則就掌控在我們手中了。

外部世界與內心世界相對應,外部世界的一切都曾存在於內心。內心世界中蘊含著無窮的智慧、無限的力量與無盡的資源,這些都是人類生命中不可缺少的一部分,並等待著我們去展示、發展和表達。內心世界中的這些潛能一旦被認知和釋放,結果就必然會如實地反映在外部世界。

一個人內心世界的和諧反映到外部世界。就是擁有良好的人際關係,舒適的生存環境,處理問題的高效,最佳的精神狀態和健康的身體。這是所有偉大,力量、造詣、成就和成功的必要條件。

一旦我們能夠控制自己的思維,並確立正確的生活態度,那麼我們內心世界就達到了和諧。

內心世界達到和諧,我們就會感到樂觀和滿足,感到滿足,我們就會對外部世界感到滿意。

外部世界也同樣能夠反映出內心世界的變化和發展。

如果我們能夠正視內心世界所蘊含的潛能,就可以開啟和釋放內心世界的潛能,並擁有將這些潛能運用於外部世界的能力。

如果我們相信自己的內心世界蘊含著無窮的潛能,就意味著我們相信自己具有這些潛能。接下來,我們就要學習如何使用這些潛能一這些潛能是我們自身和諧發展的必要條件,不能正確使用它們,我們的生命價值就得不到完全的彰顯。

我們的生活其實就是自己內心世界的投影,並無外界雜質混入其中我們在生活中所認識追求的一切必須先在我們的內心世界中體現出來。

人類只有與自然和社會和諧相處,才能獲得巨大的精神能量,否則就會處於困擾之中,也就是說,人類的生存和發展必須要以服從自然法則為前提。

我們通過意識與外界聯繫。大腦是意識的載體,腦脊髓神經系統使身體的各個器官和組織緊密配合,使我們對每一種光、熱、氣味、聲音和味道產生反應。每當我們運用正確的思維方式認識到世間真理,此時腦脊髓神經系統便會向全身傳遞出具有價值的信息,我們自然就會感到愉悅、和諧,從而更加強壯、生機勃勃和富有創造性。反之,意識也能將悲傷、疾病、短缺局限及所有不和請因素帶入我們的生活。錯誤的思維方式會使我們的生活出現混亂、不和諧。

與意識不同,潛意識負責我們的內心,是意識與內心世界建立聯繫的橋樑,腹腔神經叢是產生潛意識的器官,交感神經系統負責各種主觀感覺,例如高興、恐懼、愛情感、呼吸、想像和其他所有潛意識現象。

人類對這兩種精神活動的協調以及對它們功能的認識是生命的一個重要秘密,一旦掌握這個秘密,我們就能讓客觀(意識行為)和主觀(潛意識行為)協調一致,從而使自己和諧地發展,這樣一來,我們就可以將看似捉摸不定的未來把握在自己手中。

眾所周知,總有某種法則遍布整個宇宙,充斥著一切空間,在任何地方都以同一方式展示威力。它豐富、強大而充滿智慧,無所不在,最重要的是,所有思想與事物都符合這一規律。

普遍適用的理念指導人們的實踐。在這種理念的指導下有助於我們把想像轉變成現實。儘管個人對這個理念的認識可能不一樣,但它發揮的作用是一樣的。不同的認識只是它不同的表現形式。

普通適用的理念在本質上都是相同的,所以,所有理念歸根到底就是條理念。人們要認真體悟和領會事物的規律才能找到這條理念。

一般來說,你大腦中的意識和其他人大腦中的意識沒什麼不同,因為每個人都是上述理念的個性化體現。

普遍適用的理念是一種靜態的或潛在的能量;它的簡單之處在於:只有通過個體,它才能夠體現,或者說只有通過它才能證明個體,它們是一回事。個人思維特點和思考能力的不同是人與人之間的主要區別,也是人的內在外化後的主要表現。

潛意識從本質上來說,是靜態能量的一種微妙形式。意識是精神活動的一種,是精神的動態階段。潛意識是靜態能量,意識是動態能量,它們是同一事物的兩個階段。

世界萬物的內在屬性都包含在普遍適用的法則中,這些屬性必然以它們的潛在形式一直存在於每個個體之中,因此,當一個人思考時,他的屬性使他的內心活動體現在外部世界中,外部世界的體現形式與他的內心活動互為呼應。

如上所述,一個行為所產生的後果說到底都是其精神活動的產物。因此,對精神活動的控制至關重要。

所有力量都來自內心,認識它,並實踐它,你就能完全掌握它。當你徹底理解這個法則,能夠隨心所欲地控制精神活動時,你就能將它應用在任何環境中。換句話說,你將開始有意識地與自然界的普遍法則合作,普遍法則是萬物的基礎。

普遍存在的法則也同樣是每個原子的生命原則,原子又產生各種變化,創造更多生命。所有生命都具有智慧,努力完成自身使命。大多數人只關注外部世界,很少有人發現內心世界。其實,人的內心世界是如此的豐富並具有創造力。

精神活動是你自身天生的一種能力,當你領悟了外部世界與內心世界的關係,你就能釋放這種潛能。人們不應只是單純從外部世界去尋找答案,而真正的答案往往要到內心中去尋找,這也是更為根本的解決之道。大多數人試圖通過對「果」施加影響來改變結果。他們不知道,這麼做只不過是將一種不幸轉變成另外一種不幸。為了改變這種局面,我們必須改變事態發展的「因」,而這個「因」就存在於人們的內心世界。

內心世界是力量的「源泉」,外部世界是它的「出口」。我們接受的能力取決於我們對這個力量之泉的認識,這一點不僅對你適用,對其他所有人都適用。

精神潛能訓練

現在你要做的是:選擇一個你能獨處、不受干擾的房間,找個讓自已舒適的椅子坐下。讓你的思緒自由地在內心世界徜徉,每次持續15分鐘到半個小時。用三四天或一周的時間持續做這個練習,直到你能確保完全控制你的思維。

對於有些人,這樣操作的難度很大,而有些人卻會容易一些。不過切記:只有當你能完全控制自己的思維活動的時候,才能進行其他練習。
