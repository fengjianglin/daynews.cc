遠離那些愛吐槽的人，因為他們可能處於”確認偏誤的陷阱“

天天要聞
2019-12-19

作者：千帆落

​1、

都說女人的第六感非常準確，可是倘若總是憑感覺評價別人，不僅可能會給別人帶來不必要的傷害，也會讓自己疲累不堪。

我在瑜伽班認識一個女孩小美，因為我們的位置挨著，所以小美在休息的片刻總是過來找我聊幾句。

一開始我們聊天氣，聊最新的電影，聊各自的愛好，漸漸的熟悉之後，我們開始偶爾聊瑜伽班的同學和老師以及各自的朋友。

小美總是跟我傾訴她的不愉快。

比如瑜伽班的樂樂今天上課前在電梯里碰到她卻沒有打招呼，小美就說，電梯里那麼小，她肯定看到我了，所以她肯定是故意不跟我打招呼，她一定是瞧不上我這種還沒工作的。

比如前幾天小美逛街的時候看到自己的同學和導師在吃飯。

小美就說，怪不得導師總是在開會的時候誇她，原來她私底下總是請老師吃飯啊，還不知道她背著我們給導師送了多少禮物呢。

本來就是互相吐槽，無傷大雅，所以小美絮絮叨叨地同我講這些的時候，我只當她是在發泄，通常會耐心地聽完並禮貌性地笑笑。

2、

可是有件事改變了我的看法。

那天瑜伽老師跟大家聊天的時候聊起自己旅行的經歷，大家紛紛分享自己旅行中的趣事以及被宰的慘痛經歷。

小美一臉不屑地跟我說，這有什麼啊，不過是在互相炫耀吧了，我要是個富二代，我現在早就把五大洲走遍了。

我突然意識到，小美總是帶有偏見的闡述事情。

心理學上有個詞叫確認偏誤，是指一個人不顧事實與否，總是偏好支持自己的成見和猜想的情節。

這會使我們有選擇性地闡述事情，搜集那些支持我們結論的有利細節，忽略矛盾的信息，對一個事情進行片面闡釋。

小美看似是在毫無遮攔的說出自己最真實的想法，其實是在表達她對所有事情的偏見。

樂樂可能那一天心情很不好，所以不想跟任何人打招呼。但小美主觀臆斷樂樂看不起自己。

小美的同學可能是個懂事，優秀且努力的女孩，那天恰巧因為導師幫助自己解決了難題而請客表示感謝。但小美卻認為她是個擅長討好老師的心機女孩。

我們那次聊旅行，只是大家都樂意分享旅行的樂趣和不愉快的遭遇。但在小美眼裡，成了所有人的互相炫耀。

3、

女人的心思細膩，總是憑感覺判斷很多事情，但是倘若在評判別人或某件事的時候，一直把自己的偏見帶進去，而不去客觀，公正的對待人和事，這不僅會讓自己時時不快樂，也會給別人造成很多麻煩。

良言一句三冬暖，惡語傷人六月寒。

情商高的女人，絕不會掉入確認偏誤的陷阱，不會妄議他人。

她們會把自己的確認偏誤盡量矯正，站在一個客觀公正的角度，用一顆寬容理智的心去待人處事，這樣自己能活得更輕鬆，身邊的人也能避免不必要的傷害。

END
