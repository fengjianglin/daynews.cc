領導打壓你除了默默忍受和離開，還有這三招，聰明人都這麼做

2019-12-11 275條評論 1,856次閱讀 368人點贊

在職場生活里，莫名其妙被領導打壓可謂是最令人頭疼的問題了。因為你往往不知道領導是因為什麼原因突然對你這樣，又不能直接去問，要是領導表現的明顯一點還好，自己能根據蛛絲馬跡推斷領導生氣的原因，然後加以改正。

但若是領導一直默默憋著，也不告訴你原因，就是莫名其妙對你冷臉、否定你的工作，不處處打壓卻在某些方面挑你的刺，在小事上讓你十分難受，又在大事上挑不出錯處，讓你難以知曉他的真實心意，也就無從談起改正了。

小王就是這樣一個例子。她初到公司，一直小心翼翼，剛開始的時候還好，工作上不出什麼錯，領導也比較照顧她。她和領導一直相處的很融洽，對領導也心存感激。但日子久了，領導就慢慢變了，處處打壓她。有時候說她的文案寫得不好，有時候又說她的工作能力沒有別人強，總之處處挑刺，讓小王非常難受。

本來就是新換的工作，剛站穩腳跟，小王不想輕易放棄，但領導處處打壓她，又讓她在工作時完全失去了熱情和信心。所以小王非常困惑，不知道哪裡惹到了領導，也不知道該怎麼做才好。

其實，很多人都會面臨這樣的問題。剛開始和藹可親的領導，隨著時間的推移也逐漸變得面目可憎起來，而自己恨得牙痒痒，又不知該怎麼辦，只能默默忍著，其實這時候忍是最不可取的。我們應該先分析一下，領導到底是由於什麼原因，才處處打壓自己呢？

1、是否鋒芒畢露

很多新人覺得自己剛入職，應該好好表現一下才能站穩腳跟，所以工作十分努力，極力表現的最好，將自己最好的一面呈現給公司，所以顯得太過鋒芒畢露，搶了領導的風頭。這個時候，領導心裡就會不太滿意，可能就會打壓新人。

2、是否言行恰當

在剛剛進入一家公司的時候，我們都是小心翼翼的，生怕自己做錯了什麼。但一旦日子久了，就會放鬆下來。這個時候就有可能言行不太謹慎了，顯現出自己的許多缺點，惹得領導不太高興。所以我們在這個時候應該反思一下自己最近的言行舉止是否合適，是不是哪裡舉止不恰當惹到領導了。

3、是否他人挑撥

刨去上面兩個原因，我們應該再想一下是不是有同事向領導打小報告了。自己的突兀出現和極力表現可能會給同事危機感，尤其是和自己實力相當的同事。這時就應該考慮一下是否有人專門挑撥自己與領導的關係，不過不可大張旗鼓的懷疑，否則可能寒了同事的心。

那麼分析完原因，得知自己可能出現的問題之後，我們該如何做才能挽回領導的信任，阻止領導對自己再度打壓呢？

1、謹言慎行

在公司里，天大地大，領導最大。他如果有心打壓，你一定要更加謹言慎行，讓他挑不出錯來。工作要更加認真努力，不要太過鋒芒畢露，也不要太過低調內斂。要讓領導知道你的努力，但面對領導時也要更加謹慎，不要說錯話做錯事，這樣他抓不住你的小辮子，自然對你無可奈何。

2、及時交流

如果有可能的話，要與領導多交流。側面打探一下他打壓你的理由，就算打探不到也要讓他知道，你的出現不會威脅他的地位，你對工作也是十分認真努力。若有人挑撥的話更要及時與領導交流，了解清楚領導的心意，順勢而為。

3、不必害怕

其實面對領導打壓時不必太過害怕，因為他如果真的覺得你不適合這個崗位，自然會讓你走人，而不是處處挑刺，讓你難受。所以一定要相信自己，展現出自己的能力，能力才是最重要的。只要你有能力，即便他再打壓你，你也不會失去這份工作，而且你還能夠通過自己的努力來改變他心裡的印象。

人人都有可能面對被領導打壓的情況，在這個時候，不要太過害怕，仔細思考一下領導生氣的原因，再加以改正。即便想不出領導生氣的原因，只要自己小心謹慎，認真工作，努力提升自己的專業能力，你的能力也會幫你穩穩地在這個公司立足。
