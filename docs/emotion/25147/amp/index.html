激勵人的正能量句子：熬得住出眾；熬不住出局；這就是人生

天天要聞
2019-12-21

一、 日子就是這麼過的，不怕苦，就苦不了你；不怕難，就難不倒你。要想實現某些目標，就得準備承擔某些壓力。人生本如此，壓力何所懼！當那些目標實現的時候，你一定會忘記疲憊，感覺非常舒坦。

二、 如果你有大才華，就去追求大夢想；如果你覺得自己的能力有限，才華也不夠支撐起你的野心，那就安靜下來，扎進小的失敗和挫折中，汲取營養。

三、 人生沒有對錯，只有選擇後的堅持，不後悔，走下去，就是對的。走著走著，花就開了。人生，靠的不是時間，靠的是珍惜。抱最大希望，盡最大努力，做最壞打算，持最好心態。記住該記住的，忘記該忘記的，改變能改變的，接受成事實的。太陽總是新的，每天都是美好的日子。

四、 任何事情，都是一個道理，熬得住，出眾；熬不住，出局；這就是人生！只有堅持，才是真正的成功；只有拼搏，才是充實的生活。

五、 在心甘情願的浪費的過程里，青春漸漸逝去，直到消失，而我們卻繼續著在青春上所遺留的習慣卻不自知。不知道我們在重複著浪費的舉動，也不知道自己其實要給青春交出答卷，更不知道要用何種方式去祭奠青春。

六、 你想要揭掉一個好標籤，太容易不過，只要對應著做一件壞事即可；而你想揭掉一個壞標籤，這太難了，千番努力之後，即使能揭掉，也會帶起一層皮，留下一塊難以磨滅的疤痕。

七、 人不會苦一輩子，但總會苦一陣子。許多人未來逃避苦一陣子，卻苦了一輩子。

八、 水再渾濁，只要長久沉澱，依然會分外清澈；人再愚鈍，只要足夠努力，一樣可以改寫命運。不要憤懣起點太低，那只是我們站立的原點。人生是一場漫長的競賽，有些人笑在開始，有些人卻贏在最後！

九、 從不獲勝的人很少失敗，從不攀登的人很少跌跤，要想知道成功的滋味，就得少言多行，決而不斷，斷而不行，必然貽誤時機。

十、 在坎坷的生活道路上，想做的事，只要有能力做，那就不要等，不要害怕失敗；想付出的愛，只要覺得可以，那就大膽些，不要留下遺憾。生活要靠自己經營的；工作要靠自己努力的；伴侶是要用心呵護的；家庭是要彼此珍惜的。

十一、 只要認為是對的就去做，堅持去做，不要在乎別人的看法，哪怕是錯，至少你有去做過證明曾經你努力過。

十二、 如果你在努力奮鬥，你有可能會失敗，但是，如果你不奮鬥你就已經輸了；今天是你往後日子裡最年輕的一天了，因為有明天，今天永遠只是起跑線。

十三、 人生不怕平淡的日子，只怕的感覺不真實。生命有限，所有得到，最後終會失去。只要用心珍惜過，就不必太在意失去。人間恩怨與困惑，由心所生。我們每個人在任何時候都需要一顆清凈之心。

十四、 帶奮鬥一起飛翔，因為有了它，讓我擁有理智之思；我才使過去的失誤不再重演到今天的影片里；我才能使過去的成功在人生中繼續升華；我才能真正收穫金秋豐碩的果實，品味人生的快樂。

十五、 當你成功的時候，人人都會讚美，其實你不一定那麼好；當你失敗的時候，人人都會輕賤，其實你不一定那麼差。

十六、 我們現在經歷的那些，無論好的壞的，全部都是微不足道的小事。度過那些搖搖晃晃的日子，所有艱辛也會內化成為力量，留下閃光又珍貴的回憶和一路上堅韌不拔的勇氣。

十七、 我們會遇上順境，也是會遇上逆境。在所有的成功的路上折磨你的，背後都隱藏著激勵你奮發向上的動機。想要去成功的人，都那也就是必須去懂得知道如何將別人對自己的折磨，去轉化成一種讓我們自己克服挫折的磨練。

十八、 明白自己是誰，適合做什麼，目前該幹什麼，比盲目去努力更重要。

十九、 生活不會向你許諾什麼，尤其不會向你許諾成功。它只會給你掙扎、痛苦和煎熬的過程。所以要給自己一個夢想，之後朝著那個方向前進。

二十、 過去和現在一目了然，而未來只是或許，把握現在，吸取過去的經驗教訓，努力創造未來！

二十一、 當看到他人的病痛和殘疾時，才覺得健康是多麼的幸福和珍貴。我們沒有資格去輕視生命，同樣，我們沒有理由去忽視健康。有了健康，生命才會更加美麗和精彩；有了健康，人生才將更加輝煌和燦爛。

二十二、 懶惰是很奇怪的東西，它使你以為那是安逸，是休息，是福氣。但實際上它給你的是無聊，是倦怠，是消沉。
