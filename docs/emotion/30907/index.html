習慣相互依賴者，如何穿越設立健康個人邊界的壁壘？

2019-12-24 1,793條評論 3,074次閱讀 1336人點贊

序 互相依賴者的典型行為和其成因

在互相依賴的主題文章中有一個非常流行的話題，那就是習慣互相依賴的人在設立健康的邊界方面所遇到的挑戰。

讓我們從典型行為及其成因的概覽角度來近距離看看互相依賴。

1、互相依賴者通常來自功能失調的家庭，在家庭中學會把他人的需求放在更重要位置，時常隱藏自己的需要。

互相依賴並非是一項心理健康診斷，相反，它是習得的行為，通常是在孩童時代的很早期就發展起來了。互相依賴者大多數（並非全部）來自功能失調的家庭，這些家庭通常存在互相依賴史及成癮史，以及自戀或情緒不穩定的伴侶。

互相依賴者的父母以及成癮的人或自戀的人給孩子們樹立了模式去理解關係。從互相依賴的父母那裡，孩子們學會了要在關係中更加努力，全力以赴滿足那些自戀、成癮甚至是虐待他人的父母們的需要，並且他們總是能夠做出必要的改變來修正或調整關係。

在這類功能失調的家庭中，由於父母過於陷入他們自己破壞性關係的動力系統中，孩子們真正承擔起父母角色的情況並非罕見。孩子們學會了「戰戰兢兢」，學會了隱藏自己的需求並將他人的需求放在首位，還認識到關係的外在表現是多麼的至關重要。

2、在關係中，不會形成自主的自我價值，因此很難為自己設定關係的邊界。

習得了這些經驗的孩子們不會學著培養他們的自我價值感，也不會將自己視為一個重要且自主的個體，他們會在未來的關係中掙扎。他們總在尋找某個人來調整、修復或照看關係，而不是找一個安全、獨立且有愛心的伴侶。

如果這聽起來讓你覺得熟悉，並且照看你父母的需求比你自己的幸福更重要的話，那你可能是互相依賴的。

在關係中設立邊界或許會讓你感到極其困難，如果伴侶變得憤怒、不高興或是不安，你也許會發現甚至是對伴侶說「不」的念頭也可能會導致焦慮。同時，你在這段關係中也感到不快樂。大部分互相依賴者感到被一段關係所困，相較於獨處的需要，需要和某人在一起的念頭顯得更為可怕。

3、由於投入大量精力但沒有回報，長期積累的沮喪感，會導致對自己和他人的強烈憤怒。

與此同時，對一名互相依賴者而言，在一段關係中投入了大量精力和時間但卻又毫無回報，他們通常都會感到沮喪甚至憤怒。

久而久之，這種憤怒和沮喪會堆積起來，但通常都會朝向自己，因為向伴侶表達憤怒和怨恨只會增加衝突和對抗的風險，這正是互相依賴者在奮力創建「完美」關係的過程中想要避免的。

硬幣的另一面也能在互相依賴者身上發生。隨著時間的推移，由於不斷的要求而產生的沮喪感以及時間和精力上的消耗感會積聚起來，結果就是對伴侶大發雷霆。這也可能導致互相依賴者選擇一位需要被照顧的伴侶，從而允許互相依賴者有某種程度的控制。同時，這位需要照顧的伴侶會抗拒這種控制，也會大發雷霆，並且在整個關係中製造衝突和失控。

所有這些衝突和爭鬥的根源，無論是內心的還是關係中的，往往都是由於缺乏界邊界造成的。互相依賴者願意在其個人幸福和需求上讓步，以企圖和一個永遠無法被滿足的人建立一種積極的關係。

某種程度上，互相依賴者承認這種不平等，但也害怕被拒絕、害怕對抗。請記住，互相依賴者的目標在於讓一切都符合他們心目中完美夫妻的概念，以及在於周圍的人會怎麼看他們。說出「不」或者選擇去做一些對他們重要而非是對伴侶重要的事情，這些都是與目標的實現所背道而馳的。

一 邊界對相互依賴者的意義

邊界是規則，或者是隱形的牆或柵欄，用來把人和人分開，以及把你生命中的生理、情感和精神元素同你周圍的人分離開。

我們也可以藉助邊界來讓人們知道哪些是我們接受的，哪些不是。當我們不能為我們自己設立邊界，我們就自動默認允許他人來設置這些邊界。我們也就把自己敞開了，必須不斷去應對來自那些想要得寸進尺者們的荒謬且不合理的期待。

二 相互依賴者設立邊界面臨的挑戰

設立邊界的關鍵在於讓對方知道邊界是存在的。

簡單舉例，想像你有個孩子並且允許她在廚房裡拿任何她想吃的東西，並且你對於飲食也沒給出任何限制。孩子很自然地會選擇他們喜歡的食物，並且會沉迷於冰淇淋、餅乾、薯條和其他類型的食物，會不出所料地避開蔬菜和水果等健康食品。

爸媽通常都會在飲食方面給孩子設立邊界，讓孩子們知曉規則和期望。這可能包括飯後來一小份冰淇淋，或偶爾來點數量有限的餅乾。

當把規則向孩子們解釋了，他們也就學到了這些邊界。那並不代表他們不會試探這些邊界，父母必須得保持堅決和一致。通過明確邊界，然後保持一致，規則就被創立起來了，這既能夠允許行為的積極方面，也提供了關心和保護。

一段關係也需要相同類型的邊界。然而正如我們剛剛看到的那些互相依賴者的行為，就很容易理解為什麼設立並堅守這些規則或柵欄會是一項艱巨任務了。

互相依賴者也許能理解他們需要的邊界，但是強迫他們去做，就會出問題。自戀或成癮的伴侶非常熟悉這些動力，也會非常樂意去施壓和操控這些邊界，或者是直接突破邊界，這些都會讓事態變得更加困難。

互相依賴者會將邊界的存在視為挑戰，常見的原因包括：

1、害怕衝突。對互相依賴者而言，僅僅是勇敢地面對伴侶，這一行為也會是淹沒性的，尤其是如果這類問題曾經有導致伴侶的離開、威脅要離開或演變成情緒或身體虐待。

2、不了解需求。包括和父母的關係在內，如果你從來不知道人際關係的邊界，那也就很難了解你自己的需求。

3、認為對方更重要。在建立和維護邊界所面臨的挑戰中存在一個壓倒性的因素，也是一個根深蒂固的信念：對方更重要，需居於首位。

4、害怕失去認可。當伴侶在關係中總是我行我素，尤其是和自戀者在一起時，設立邊界會導致失去認可，這會讓互相依賴者的恐懼感爆棚。在此期間，去找治療師或者生命教練開展工作，這會有助於發展出有效的應對策略，而不僅僅是屈服並回到過去缺乏邊界的關係中。

5、採用責怪的套路。當試著設立邊界時，有些互相依賴者會變得不斷抱怨或咄咄逼人。他們不會去設立並堅守邊界，而是不斷重複信息，批評或責怪對方改變了關係的「規則」。這可能導致互相依賴者威脅要離開，或是威脅要干一些她不會幹的事，當伴侶挑戰了這一新的需要或價值，邊界的缺失就會加劇。

6、缺少支持。在一段互相依賴的關係中，一個人通常都會在情感或社交上被孤立。你畢竟把時間拿去取悅你的伴侶了，也就經常會忽視或冷落朋友和家人。這種孤立和缺少支持會讓自戀者、成癮者或情感虐待者受益，因為他意識到他是你唯一的聯繫。

7、低自尊和自我價值感低。假如你是互相依賴者，你的自我價值感和自尊都很低。這讓你難以看到自己值得他人尊重，也使得你很難設立這些邊界，然後難以堅守由遵循邊界而來的結果。

好消息是，這類的信念、恐懼和議題是可以被改變的。重要的是學習設立邊界的新方法，從而提高自主權、給予自我照顧，並且幫助你對自己和周圍的人都感覺舒適。

三 如何穿越邊界的壁壘

穿越設有恐懼和障礙的邊界並非易事。這需要時間、支持以及願意學會去相信你自己的價值。在任何關係中，你都有可能學會設立並且清晰表達邊界，然後也要準備一些結果，用來讓自己對這些邊界負責。

設立邊界的關鍵包括以下方面：

1、疼愛你自己。首要步驟就是開啟一段旅程，來學習愛自己並且發展出自我價值和自我重視感。這是一個學習的過程，包括學著忘記所有那些負面信息並且開放地從一個全新的角度看待自己。

2、明確你重視的東西。邊界是籠統的；它們並非適用於所有人。在關係中，一個人可能願意接受的，對另外一個人而言可能就不行。然而，被接受的東西必須是健康的、積極的和有益的，否則它不是關係中的有效邊界。審視一下你所重視的和你的需求，這是設立適當和有意義的邊界的起點。

3、審視一下你想要幹什麼。時間和優先順序通常是互相依賴者的問題。他們害怕失去友誼或地位，於是他們對任何事情都來者不拒，即使他們沒有時間或精力去完成他們承諾過的事。通過學會花時間考慮你想幹什麼，並優先安排你的時間，你就更容易找到說出「不」或「我那時會很忙」的方法，而不會因為沒有站出來滿足你的需要而被內疚、恐懼或自我憎恨所壓倒。

4、談論邊界。與諮詢師、生命教練和家庭成員交談，可以非常深刻地了解他們是如何設立邊界的。了解如何在所有類型的關係中運用邊界，而不僅僅是在親密的夥伴關係中，這樣既有教育意義又增加見識。

5、練習信息溝通——學習如何表達和傳遞你的需求、願望和價值，這始終是必不可少的。這也許像是在學習一門新語言，尤其如果你是一個從來沒有談論過需求的人。練習如何交流是你人際關係中需要掌握的核心技能。

6、向外求助同樣也是必不可少的。學會設立邊界並且讓自己守住這些邊界，不再屈服或者是回到以前不良的行為方式，這既複雜又困難。有時候，屈服只是似乎更容易些，但每次你屈服，你都會給自己和對方一個信息。這個信息就是你在這段關係中沒有價值，但這絕非如此。

假以時日，設立和維護邊界會變得更加舒適。可能有時你會開始消極思考，有時只是想避免衝突或是情緒感到疲勞，但即使在這些情境下，你也可以變得堅強、自信，並照顧好你自己。
