20句晚安心語，快拿筆抄下來，選一句發朋友圈

2019-12-21 435條評論 3,844次閱讀 1221人點贊

1、人生，有多少計較，就有多少痛苦；人生的堅持，就是學習的堅持，儘管是遭遇困苦，但是我們卻是能在一次次的苦難中去成長，重新的去爬起來。這就是奮鬥的人生。晚安。

2、有誰的幸運，憑空而來，只有當你足夠努力，你才會足夠幸運。這世界不會辜負每一份努力和堅持。時光不會怠慢執著而勇敢的每一個人！晚安！

3、沒有第一次的成功，就沒有後面的更大的成功，這個過程是漫長的，也是艱難的。但是世界上聰明的人很多，有知識的人遍地都是，但真正能發大財的卻少，要把知識變為知本，只有和資本聯姻才行。

4、幸福很簡單，就是每天過得平淡美好。每天花點時間，滿足自己的饑渴。因為我們追求的是自己的幸福，而不是比別人幸福，晚安

5、不管你用什麼方式活著，我們只有一個目的，別違心，以及別後悔，還有，去他媽的人言可畏。

6、和你在一起的時候，覺得時間過得很快，現在又到了晚安的時候了，人卻特別想你。不能在你身邊陪著你安睡，希望你也能有個好夢，晚安。

7、一直在想，很多年以後，如果我和你，就這樣再也不聯繫，可突然有一天，就這麼站在喧囂的人群里，相互注視著對方，第一句話需要多大的勇氣才說的出。

8、趁你現在還有時間，盡你自己最大的努力。努力做成你最想做的那件事，成為你最想成為的那種人，過著你最想過的那種生活。

9、心如止水，亂則不明。很多事，你越是想去弄個清楚，反而越是困惑，就像線團，只會越扯越亂，凡事順其自然就好。既來之，則安之，這才是生存之道。

10、人生在世都不容易，懂得珍惜該珍惜的一切，懂得放棄一切應該放棄的，懂得把握一切應該把握的，懂得享受該享受的一切。讓自己這輩子，過得充實、快樂、健康。晚安

11、我會好好的生活，不為別的，就為這些年我虧欠自己的。有時候可以看得很淡然，有時候卻又執著得有些不堪，不管發生什麼，都不要放棄，堅持走下去，肯定會有意想不到的風景。晚安。

12、猶豫越久，就越容易喪失成功的時機。從最糟的結果來說，如果一個較差的決定是比較早做出的，出了問題你還有補救的時間；而一個決策的作出如果既差又遲，那麼你連補救的餘地都沒有了。

13、處在的無邊的夜晚面前，有時才突然明白：活著，就是一種簡單的幸福。人活著，是一抹心情，在孤寂中沉默，在沉默中成長！寂靜的夜晚，其實也就是心在行走，心在一路低吟，一路歌唱。

14、失去的風景，走散的人，等不來的渴望，全都住在緣分的盡頭。何必太執著，該來的自然來，會走的留不住。放開執念，隨緣是最好的生活。

15、人活著，不是所有的委屈都需要告訴所有人，沒有誰能對你的痛苦感同身受，所以別總是用自己的遭遇去博得別人的同情，你該學著用自己的經歷去贏得別人的尊敬。

16、從今天起，努力去做一個可愛的人，不羨慕誰，也不埋怨誰，在自己的道路上，欣賞自己的風景，遇見自己的幸福。

17、經歷越多就越不想說話，環境的不同，想說的話別人未必能懂，也就慢慢學會了自己默默承受。

18、如果找不到堅持下去的理由，那就找一個重新開始的理由。生命太短，沒有時間留給遺憾。若不是終點，請一直微笑向前。

19、你或許很辛苦，跌跌撞撞一身傷，也許會有人安慰你，但不會感同身受。這個世界就是這樣，只能看到勝利者的汗，卻看不到失敗者的淚。你要明白，不是只有你一人在忙碌，大家都各懷心事默默付出。這點傷真的不算什麼，苦當做禮物，永遠在路上，才是一生不改的風景。

20、強大就是遭受過人生的不幸，但仍期待幸福；受到過別人的背叛，但仍勇敢去愛；看見過世間的醜惡，但仍付出善意。
