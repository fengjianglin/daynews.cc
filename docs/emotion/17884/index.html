研究表明：與爺爺奶奶親近的孩子 成年後會更加快樂

2019-12-17 867條評論 1,904次閱讀 899人點贊

有那麼一些研究表明，那些能夠與孫子孫女在一起的爺爺奶奶往往擁有更長的壽命，而反過來，又有一項研究發現，這對孩子們同樣有益。與爺爺奶奶關係密切的孩子更快樂，將來也不太可能出現抑鬱的情緒。這次，V哥就將與你探討關於這一類研究的詳情。

1、來自牛津大學的研究

隨著家庭格局的變化，預期壽命的增加，雙工家庭的數量增加以及家庭破裂的增加，祖父母現在在孫子的生活中發揮著越來越重要的作用。牛津大學的研究人員安·布坎南教授和他的團隊決定更深入地挖掘祖父母與孫子孫女之間的緊密聯繫，以了解更多的信息。他們收集了英國1500多名兒童的數據，並對他們進行了檢查。通過統計發現，與祖父母住在一起或與祖父母關係密切的孩子情緒問題較少，而且更快樂。

在英國家庭政策中，目前尚不清楚祖父母的角色，並且祖父母沒有合法權利去見孫子。然而，布坎南教授的研究表明，他們的參與與減少所有家庭類型的適應困難密切相關，尤其是在離異或分居家庭的青少年中。

2、帶來更好的生活質量

祖父母，也就是爺爺奶奶的陪伴，還能帶來更好的生活質量，長大後出現抑鬱的機會也更少。事實上，另一項研究聲稱，當祖父母和他們的孫子孫女在一起時，這些有益的癥狀是雙向的。祖父母和孩子在一起時都會更快樂

3、但長遠來看，陪伴也可能增加老年人的抑鬱癥狀

但並不是所有的研究都是積極的。一項發表在牛津大學的《老年學家》上的研究表示，這種關係也可能會增加老年人的抑鬱癥狀，因為老年人只接受護理，無法給予任何回報。也有人觀察到，孩子們一旦到了成年早期，往往會離開祖父母。但當他們長大後又回來了，因為他們開始想念他們的「奶奶」。

可悲的是，在世界的某些地區，祖父母對他們的孫子孫女沒有合法的權利。如果父母決定帶走他們的孩子，他們將無法接近他們。這項研究是為了讓人們認識到孫子孫女和祖父母在一起有多麼重要。

​孫子孫女使他們祖父母的生活更加活躍和快樂。奶奶們要照顧孩子，爺爺們要指導孩子們，給他們講他們經歷的偉大故事。與那些不關心孩子的人相比，花時間陪伴孫子女的祖父母的死亡率要低37%。因此，讓孩子們和他們的祖父母在一起是存在很大幫助的。

你在成長過程中和你的爺爺奶奶很親近嗎？這是否讓你感到更加的快樂和幸福呢？
