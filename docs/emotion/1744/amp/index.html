後來，母親斷了她的學費，小玲只好厚著臉皮去上學，還好學校寬容，沒有硬逼著交學費。後來，她申請了助學貸款。可母親一直不承認助學貸款的事情，稱沒有給她擔保。後來，小玲畢業了，母親悄悄詢問，「助學貸款還清了嗎？」

很多時候，父母感覺到的並非事情的本質。有些家長喜歡自欺欺人，有些不捨得在子女身上付出金錢。這是子女無法改變的客觀事實。

不是每個父母不去學習，就懂得付出和承擔，懂得如何愛人，完全依賴自主學習，主動探索。學習如何跟小孩相處，學習怎樣給小孩適當的愛，讓其愉快長大。如果能夠做到這些，不管家庭經濟條件如何，小孩內心都是富有的。

不是所有的父母都是合格的，完全能夠理解小玲的經歷，上學自己助學貸款，自己掙生活費。希望她前面吃的苦，都能在後面用幸福填滿。如果說做父母按百分制來算，其實很多人的父母連50%都夠不到。你覺得一生該花的錢，該體驗的東西，他們會極力反對。干預你的生活，規劃你的人生，哪怕他們一生都是無所作為的市井小民，還非要你去做明知不會有好結果的事情。適婚年紀逼著你結婚，不管對象如何，可離婚後他們就安靜了，不管你「新品」變「二手」，搭上了青春和金錢。

另一位「懶羊羊」的網友有同樣的經歷，至今都想不通原因，小時候上小學，別的小孩都有一兩塊錢的早餐錢，我沒有，真的不是因為家裡窮。長大一點才明白，那時候家裡有十幾萬存款，有時候要一塊錢被父母打罵。父親是那種極度節省的人，抽煙也撿最便宜的，找他要一塊錢就是找打，而母親經常趕街的，記憶中光是帽子、鞋子都有好幾櫃，而對她你總是連一塊錢都捨不得，後來上了初中要住學校，一個星期給一百塊錢。她的弟弟上小學，每天都是五塊、十塊、二十塊的給。她清楚記得，她弟弟看不上一塊錢，用盒子裝起來，一年下來也能攢三四百塊，後來幾年被一個商販用一百的整錢換走了。她那時候穿的也不好，還拿了學校發的一千塊錢補助金。

養育孩子是一項人生投資，父母不僅要滿足小孩最基礎的生活和開銷，也不要忽略孩子的心理健康。小孩兒時所受的傷害，可能會攜帶一生，很難磨滅。如果家有多子女，父母不要區別對待小孩，讓孩子得到均分的愛，更利於他們的成長。

一個富足的小孩，取悅自己，肯定自己，不炫財富、不去攀比，外界評價不完全依賴，不會成長為人前一套、人後一套的「雙面膠」式的人。

有的小孩花銷大，父母可以讓孩子明白，「我們錢不多，家裡也不是提款機。雖然不能給你那麼多錢，但是我們依然愛你。」
