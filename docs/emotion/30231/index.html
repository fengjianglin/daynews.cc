《奇葩說》：要放棄一切跟伴侶去大城市嗎？這是我聽過最好的答案

2019-12-24 591條評論 2,478次閱讀 631人點贊

​遵從內心的選擇，堅守真正的幸福

前兩天，看了一期《奇葩說》，辯題是：「伴侶在大城市找到很好的工作，我要不要放下一切跟他去？」

當時看到這個辯題時，我的第一反應是，當然會跟著去，因為自己身處小地方，加之尚且年輕，本就沒有一切，又何曾放下呢？

之後在辯論期間，印象最深刻的是詹青雲和熱依扎的論述，讓我恍然明白了一個道理：其實，每個人的年齡階段，心智模式，工作能力和自身閱歷等不同，所處的生活狀態也不一樣，這就使我們在面臨一道人生選擇題時，內心會自然而然地湧現出不同的答案。

譬如這道辯題，有的人願意放下一切奔赴他（她）去，與其共同經歷風雨，彼此支撐和陪伴；而有的人心裡會有所顧慮，他（她）想要守住當下的擁有，或覺得先活好自己，再考慮追求未知，這樣興許會更妥帖。

誠然，這些不同的選擇並沒有對錯，因為凡是有得到，必然會失去。重要的是，我們該如何權衡眼前這個得與失的重要性，該如何做決定，才能更接近，也更清晰自己內心深處的聲音？

思考一：最好的年紀要敢於追求，因為錯過將是人生最大的遺憾

詹青雲是這道辯題的正方，她的那段論述，我反覆聽了好多遍，每次回聽，內心都深受觸動，且總不禁淚流滿面。

她提到，像自己這樣典型的中國式乖孩子，非常善於等待和忍耐。讀高中時，她遇到了喜歡的人，當時心裡明白等高考結束後再談，可上了不同地方的好大學後，心想著等學業有成再談，而到了畢業，各自又開始奔工作……

於是，就這樣一直等下去，過了三年又三年，最後這段還沒真正開始的戀情，便就此擱淺了。

再之後，在香港讀書期間，她曾有一位伴侶，但後來去美國留學時，那個人不願意跟她去，然後這段感情又結束了。

這是她經歷過的幾次擦肩而過的愛情，原本彼此可以更努力地爭取在一起，但正因為當初缺乏那份愛的勇氣和決心，最終留下了深深的遺憾。

當一個人經歷過「在最好的年紀，遇見你」的欣喜，她才深知「在最好的年紀，離開你」，原來是這麼刻骨銘心，這麼悵然失意。

正如她說的「這世界上，到底有什麼東西是我們放不下的，到底有哪條路有這麼難走，要讓我們把所有的青春，秋天，都錯過呢？」

我們每天都在做大大小小的選擇，有些選擇做錯了，我們可能並不那麼在意，恰如往事如煙，心裡了無掛礙。

但也有些選擇，一旦做錯了，便會讓我們不覺然地錯過生命中很重要的人、很重要的事，而且每當回想起來，總會有些時刻，自己的內心會感到格外孤獨，抑或十分遺憾。

因為每個人生階段中的過錯，只是短暫的遺憾，而在最好的年紀里錯過，那將是整個人生路途中最大、甚至永遠的遺憾。

在《大魚海棠》這部電影中，片尾有句話觸動了無數人：「這短短的一生，我們終將都會失去，不妨大膽一些，去愛一個人，去攀一座山，去追一個夢。」

執著的椿會為了鯤違背天規，犧牲掉自己的半條命，深情的楸又會為了椿而奮不顧身，最終捨命成全椿與鯤。他們雖年少輕狂，但卻向我們詮釋了這世間最動人心的愛與感動，也彌補了現實生活中人們愛而未追、愛而未得的遺憾。

其實，在這道辯題面前，我們要遠比楸幸運，至少我們有了一個伴侶，他（她）懂得你的愛，也願意攜手共同承擔這份愛的代價，更何況，他（她）又得到了這座大城市給予的機會，這讓彼此多了一份踏實和確信。

如果兩個人是真心相愛，其實「大城市」和「好工作」這兩個客觀因素，已經不那麼重要了，換言之，它們只是生活贈予的一個獎賞。因為我們更在乎的是，自己能否和心愛的人在一起，即便是異地戀，未來也總有共同生活的經歷。

真正親密的關係，真正平等的愛情，是沒有高貴和卑微之分的，而是一路相伴，一路同行。

可如果錯過了，又怎麼會有後來的我們呢？

年輕時，喜歡一個人，但沒勇氣表達，更不敢去追求，等過了那個年紀，才意識到當時的自己好傻，竟錯過了一個那麼好的人。

年輕時，熱愛一件事，但怕人嘲笑，更害怕失敗，等到了老的時候，才後悔當年沒抓住機會，竟錯過了一個那麼熱愛的事業。

人這輩子，到底有多少個三年、十年，有多少個青春年華可以讓我們錯過呢？

在最好的年紀里，我們最不該怕的便是試錯，因為隨著年齡的增長，生活會教會我們很多事，但給我們試錯的機會也越來越少。

對大多數人而言，很惶恐職場上的35歲定律，擔憂自己過了這個年紀，人生再也不出現重要的轉折點了。其實不然。

記得懷左學長曾在閱讀職業規劃方面的書時，有分享一個真實的數據，指人這輩子大概90%以上的財富，是在自己40歲以後賺到的。

但這裡存在一個重要的因素，即我們今後所擁有怎樣的生活狀態，一定取決於前些年自己的積累資本，而年輕時我們的努力，大概率可以縮短那個到達自我理想狀態的周期。

同樣，在年輕的時候，如果真的能夠和伴侶一起在大城市裡奮鬥，去尋求更好的發展機會，那我們不妨多去試一下，相信未來所接觸到的那個成長環境，會給我們帶來更多期望，甚至想像不到的可能性。

再不濟，我們還有另外一種選擇，就是再回到小城市裡來，重要的是，在此之前已和伴侶共同經歷了不一樣的生活體驗，哪怕那段日子過得很艱難，但至少，讓彼此更堅定了在小城市生活的信念，明白後者才是最終的歸宿，這難道不也是一種選擇嗎？

事實上，在每個選擇前，我們都會猶豫，但也會極為固執；而每個選擇後，我們都曾後悔，但也會更加堅信。

思考二：不管是愛情還是婚姻，只有自己保持了獨立和自信，才是最大的安全感

本期女神熱依扎對這道辯題的看法很獨到，她持有的是正方立場，而確如她說，大概是和自己的年齡有關的緣故吧。

如果把時間軸拉到十年前，那年她23歲，經歷了一段很好的感情，當時對方就問她：「你願不願意放下自己的一切，跟我一起打拚？」

她說絕對不可能。

因為十年前的她，兜里只揣著1000塊錢，父母的工資也不高，而且家裡還有兩個孩子要撫養，生活甚是窘迫。彼時一無所有的她，依然要被某些東西承擔和牽動著所有。

所以她不敢做那個大膽的決定，然心沒著落地跟著對方走，不僅僅是因為對家人、生活和未來等的絲絲顧慮，更重要的原因是，那時的她，沒有任何強大的背景，各方面還無法實現獨立，對自己也沒有信心。

但現在的她，再面對這個問題時，顯然會做出截然不同的選擇，因為她到了33歲這個年紀，人生閱歷更豐富了，工作能力和經濟條也有了一定的基礎。當實現了物質和精神上的獨立後，她對生活更有掌控感了，也有重塑一切新生的勇氣和自信。

其實，仔細想來，有些事，你曾堅決不同意，不是因為不相信他，只是自己完全沒底氣；可後來，之所以會再堅定遵從他選擇，也不是因為他變得如何，而是因為你更相信自己了。

所以，在聽完熱依扎的一番觀點之後，頃刻間，我想起了自己，就像十年前23歲的熱依扎，沒有後盾，沒有依靠（甚至所處的境遇遠不如當年的她），一切只能憑靠自己去探險的時候。如果此時再讓我來做決定，​我依然會選擇跟著另一半去大城市裡奮鬥，只是在做這個選擇前，自己心裡一定要明白，最大的安全感是源於自我的獨立與自信。

其實，這也就提醒了我們，在融入一個新的社會環境中，你要慢慢地去建立和堅守自己那份珍貴的獨立與自信，那麼，不論身處哪裡，內心的強大都會支撐著你一路向前走下去。

我們始終相信，在大城市裡的人有自己的追求，而生活在小城市裡的人，也一樣會有自己心中的夢想。

青年作家李菁有說過一句話：「你想要擁有什麼樣的伴侶，就先成為那樣的伴侶。」

很多人羨慕李菁遇到了靈魂伴侶，羨慕有人願意為了她而放棄繁華的大城市，獨身來到湘西偏遠的小鎮上與她朝夕相處，過上田園生活。這童話般美好的故事，讓多數人見證了他們愛情的偉大和婚姻的幸福。

其實，在他們兩人還未相遇之前，各自已經實現了物質和精神上的獨立，也都成為了更好的人；他們擁有那份自信，對當下的生活有著明晰的方向，也對未來的伴侶充滿美好的期待。

所以，李菁並不在乎對方擁有了什麼，她認準了這個在她心中描摹了無數次的伴侶畫像，從而遇到的對的人；誠然，她的另一半也不在乎自己曾在大城市的優越生活，心甘情願地來到小鎮上與她過簡樸的日子。

我們大多數人會認為，女孩子天生缺乏安全感，談戀愛時，會首要考慮男方有沒有房車存款，值不值得信任、理解和託付終身，之餘，也期待偶爾的小確幸和儀式感。

當然，這種想法很正常，也是女孩子們擇偶時普遍會考慮的範疇，但這只是安全感的附加值。因為真正的安全感，不是來自對方，而是源於自己，即你自己在愛情或婚姻中是否保持獨立與自信。

很久之前，知乎上曾有過類似的問題：「要不要跟伴侶去大城市？」

記得底部留言最高的一位網友，持有感理性並存的觀點，站在理性的角度上，他認同反方，因為他覺得面對這樣一個兩難題，獲得善終的概率著實太低，譬如，帶著犧牲的感情（家人親戚，同學朋友，安全感），會使對方有壓力，或許將來兩人心裡也都不舒服。

但面對愛情，很多人又往往一腔熱血，不曾理性，所以站在感應的立場上，這位網友又非常贊同正方，其中有段話很打動人：

恰恰是因為我們勇敢相信，才讓這段莫測的關係，更像愛情里的樣子，如果一味計較利弊得失，不斷地在想對方的家庭，職位，收入，外貌，性格等，那麼，我們管這叫「強求的合適和匹配」，而不是「真正的喜歡和愛」。

是啊，在這個世界上，世俗的眼光太理性了，所以很多時候，面對愛情，我們寧願去做一個懵懂無知的傻孩子，果敢地與相愛的伴侶擁抱真實的生活，共創美好的未來。

寫到最後

綜上，這道辯題，如李思恆所說的：「它不是在為難我們，不是在考驗我們，它是在提醒我們，在這個年紀，你到底想過一個什麼樣的生活，想要擁有一段什麼樣的愛情？」

我們的選擇，決定了我們擁抱愛情的姿態，也決定了我們未來的生活狀態。

相信讀到這裡，我們每個人的心裡應該都有答案了。你不必循著別人的聲音和腳步走，只需要在別人的故事裡，汲取更適合自己的共行力就好，從而選擇一個更接近自己內心所向的路走。即使走錯了，也不會太留遺憾，更何況人生路，我們走的每一步，其實都算數。
