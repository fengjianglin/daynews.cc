“父愛如山”！父親3句話，讓我心甘情願成了啃老族

天天要聞
2019-12-20

父愛如山：父愛深沉、穩重，如山一般偉大。

長大成人，離開父母。經過十多年的打拚，買房，成家，生子。有了自己的一個小家。工作，回家，看孩子，逢年過節回兩千里外的父母家。日子安靜，祥和，其樂融融。

按部就班的舒適日子被父親的一個電話徹底打亂了。電話里父親說：「你回來一趟，有個事和你商量。」回到父母家，才知道父親要說的事有多大，大的改變了我的生活。

「我們老了，你回來吧。」一句話把我懵圈了。定定神，才明白，原來父親是說他們養老的事。

「我三哥不是和你們住一起嗎？」；「我二哥住的離你們也不遠。」；「我回來幹嘛？」；「再說我在那麼老遠，搬家嗎？」面對我的一番說辭，父親一聲不吭。

「要不去我家住吧，好不好？」「不去，我們老了，不折騰了。」「可我回來也不是一句話的事呀。老婆、孩子、工作。好不容易才穩定下來，這不行。」

「叫我哥他們一起商量、商量？看怎麼辦？」「不用和他們商量。」「雇個保姆吧，我掏錢。」「我們有錢，不雇。」「哪怎們辦？」我急躁地說，「我們回來，住哪？幹啥？吃啥？」「我和你媽老了，腿腳也不靈了。」「你回來就買買東西，跑個腿，做個飯。這又不難？」

我沉默著。這不是難不難的事。我知道「鴉有反哺之義，羊有跪乳之恩。」可這個事好象也不能和那相提並論呀！看著父親我無言以對。

「把你媽的養老金給你。」「兩千多，也夠你們生活了。」「租個房，房租我和你媽掏。」聽到父親的話，無名之火騰地升起。「為了兩千，我們好好的工作不幹，可能嗎？」我們自己的房子不住，回來租房，還你們掏錢？什麼事！」

「我三十大幾的，帶著老婆、孩子回家啃老？」「讓人都笑話！」父親看著怒火衝天的我，淡淡地說：「什麼啃老？一家人那來那麼多說道。錢是不多。可這是我和你媽的心意。」

「不對，我可沒答應回來呢。怎麼就扯什麼租房，給錢的？不行，回來這個事不行。」我毫不猶豫地說。「這不是錢的事，工作、孩子上學，一系列的事？不是那麼簡單。」

「孩子就在這上幼兒園、上學，不行嗎？」「你們倆一個找工作上班，一個在家給跑跑腿啥的。不就行了。」「我和你媽馬上八十了，也沒幾年了。你就回來吧。」

「別說了。」我真不知道該和父親說啥了。是呀，父母都快八十了，我也當爹三年多了。“養兒方知父母恩”，可是搬回來，一切都從新開始，整個大變樣。再說父母百年後呢？哥姐怎麼想？左思右想這個事不行。

「你看著辦吧。」父親的話把我從思緒中拽了回來。「小時候我有事，你讓我自己定。現在，你有事，讓我看著辦？」我看著父親雪白的頭髮，布滿皺紋的臉龐。「看著辦！」就是那壓死駱駝的最後一根稻草。我知道我扛不住父親的要求了。我不再猶豫，「我回去處理一下就搬回來。」

「行，儘快吧。我和你媽還有點存款，你需要的時候吭氣。」「回去好好和老婆說，你媳婦不錯。」「別說了，我知道。」可憐天下父母心。回來啃老，或許這是一個不錯的選擇。

回到自己的小家，賢惠的妻子沒說什麼。「回就回吧，爸媽也老了。你不是答應了嗎？」辭工作，領導吃驚的勸我別衝動；和朋友道別，朋友讓我冷靜冷靜。哥姐七個，兩個和父母在一個城市，父親還讓我回去。這是老父親要成全我。「子欲養而親不待」，該衝動就衝動吧，冷靜就不用了。看著自己的孩子，想著老父親、老母親，和妻子忙著收拾家。僅僅半個月我們就啟程回父母家了。

父親三句話，讓我心甘情願成了啃老族，開始了我的啃老生活。心裡常念著「父母呼，應勿緩。父母命，行勿懶。」

作者簡介：心說20191019，70後，自由人，堅定的伴娃族，愛琢磨學習、教育、健康。
