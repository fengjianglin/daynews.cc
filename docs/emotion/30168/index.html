他們戀愛21年，60多歲才結婚，今被網友稱為神仙伴侶

2019-12-25 1,619條評論 2,604次閱讀 833人點贊

我們常在耳邊聽到別人家的愛情是多麼的神聖，別人家的愛情是多麼的偉大，別人家的愛情是多麼真摯，換言之，你的愛情不好嗎？家家有本難念的經，好的婚姻生活肯定需要夫妻雙方的努力和推進，而不是一方的付出。當然，現在的婚姻家庭一方付出的還有很多，他們結局如何呢？

不用說，在愛情盡頭也是不堪的。家庭地位現在很受人爭議，誰都想要在婚姻中做第一，避免在家中不能擔任領導角色，可婚姻是這樣嗎？不是！別人家的愛情你也可以擁有，兩人換位思考、互相包容，遇到事情為何不席地而坐交談對方的思想呢？

明星們的生活也是這樣，有的命運中婚姻很坎坷，而有的卻是恩愛有加，今天小編說的這一對情侶，他們可以堪稱神仙伴侶，一直攜手走進晚年，真的讓人羨慕不已。

羅家英我們都知道，飾演的角色可是深入人心，而且到現在也是很經典，他的愛人是汪明荃。不少的港劇中都有她的影子，而且她年輕的時候也是男人心中的夢中情人，在五月二日的時候他們在微博上發了一條消息，終於兩人領證了，有的網友會驚奇，這兩個人為什麼到現在才領證結婚呢？

其實他們在一起的時候也是挺坎坷的，兩個人在四十歲的時候互相才遇見彼此，相識之後一直都在維持他們的感情，就這樣過了二十年，這二十年里他們的感情一直像年輕人一樣熱戀，有的時候也會製造一些驚喜，在愛情面前兩個人始終保持新鮮感，就是這樣他們很恩愛。

當時汪明荃在事業上受到了打擊，她在最困難的時候他的出現，讓她感受到了生活的希望，而且羅家英也承諾不會讓她在舞台上再次遭受苦難，兩個人因為這個承諾一直扶持到現在，而這種信念也是兩個人的支撐點，互相尊敬，互相幫扶，互相珍惜。

如今兩個人領證了，也在大眾面前確定了他們的夫妻名分，這就是大家眼中的神仙伴侶吧，唯有別人的感情是美好的，而自己的感情在別人眼裡也是如此，所以每個人都要珍惜當下，珍惜身邊人，一路走來都是花費了很多的感情和心思，而這份美好直到你們晚年也要保持，這份感情才是真愛，願他們兩人白頭到老，也希望事業順順利利。
