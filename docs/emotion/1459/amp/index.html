我曾經不止一次說過，這個世界上有些感情很珍貴，要懂得去珍惜，比如說親情，愛情。

人的一生不可能永遠一帆風順，總有遇到困難的時候。當我們遇到困難了，親人會伸出援助之手，幫助我們渡過難關，那麼做人要懂得感恩，等我們條件好了，要學會報恩。

有些女人出嫁了，可她依舊貼著娘家，畢竟娘家是她生活幾十年的地方，那裡留下了她許多童年的回憶，也有著她最摯愛的親人。

女人補貼娘家，我們要學會理解，包容。但過度的補貼，不僅會傷害夫妻感情，甚至會影響婚姻的穩定，更會激發婆媳矛盾。當你發現你的妻子經常補貼娘家，你會怎麼辦呢？

哥哥做生意，我瞞著老公支援15萬。當兒子滿月了，哥哥送了一份厚禮，哥哥到底送了什麼禮物呢？

我出生在一個偏僻的農村，上面有個哥哥，我和哥哥同時念書，爸媽比以前更加辛苦了。

從小哥哥和我感情比較好，而且在學校哥哥非常照顧我，我和哥哥也發誓將來一定要做個有出息的人，要好好報答父母。

在農村，一個普通家庭同時供兩個孩子念書，壓力之大可想而知。看著父母頭上的白髮漸漸變多，哥哥不顧爸媽的反對，輟學去打工掙錢以減輕爸媽的壓力，供我念書。

後來我考上大學了，畢業後留在了武漢工作。在工作當中，我遇到了現在的老公，他家條件不好，我們在一起戀愛了兩年就結婚了。

我們結婚那會，老公沒錢買房，我們只能租房過日子。直到工作3年後我們才貸款買了房。

我記得我們結婚的第二年，哥哥從廣東辭職回來了，他說他厭倦了打工的生涯，想回鄉做點小生意，可爸媽都反對哥哥做生意，但哥哥不顧爸媽反對，還是出來做生意了。

哥哥的生意剛開始並不是很順，堅持了半年後生意終於有起色，沒想到生意剛有起色，卻遭遇資金周轉不過來的危機，爸媽又沒錢幫他。哥哥沒辦法，只好找我幫忙，我悄悄的給了哥哥15萬。

沒想到哥哥後來生意越做越好，在縣城開了好幾家分店。去年哥哥也娶城裡的嫂子，看著哥嫂一家日子過得紅紅火火，我這個做妹妹的也感到挺高興。

今年7月份我生了二胎，是個兒子，在兒子滿月那天，哥嫂都來了。我沒想到哥嫂帶來了厚重的禮物，嫂子拉著我的手說：妹，這是我和你哥的一點心意，你務必要收下，謝謝你在你哥最困的時候幫助他，沒有你的幫助，也就沒有你哥今天的成就。這是我和你哥給你們買的房子和車子，你們拿著吧。

我和老公老都驚呆了，在哥嫂的強烈要求下，我和老公收下了這份厚禮。

文中的哥哥和和妹妹感情深厚，在農村供兩個孩子念書不易，哥哥輟學打工供妹妹念書，妹妹大學畢業後留在了武漢成家。

哥哥想出來創業，卻遭到父母的反對，還好妹妹支持，後來哥哥做生意遭遇危機，妹妹毫不猶豫拿出15萬，這才讓哥哥度過難關，如今外甥滿月，哥嫂送房送車，這樣厚重的報恩方式，也讓妹妹和妹夫感動不已。

這件事雖然這結局很完美，但我覺得文中的女主做法不妥，因為既然結婚了，夫妻之間就要相互坦誠，哥哥有男應該幫，但不能瞞著丈夫一下子給15萬，畢竟這15萬對於大部分來說是一筆巨款。

娘家有難，我們不能做事不管，可有時候有些事要協商，不能自作主張，因為夫妻最重要的相互信任，大小事相互商量，這才是互相尊重。你們說對嗎？
