原生家庭毀一生？董卿用行動給出答案

2019-12-18 2,520條評論 3,826次閱讀 1675人點贊

最近和身邊幾位朋友聊天，他們都跟我提起「原生家庭」這個詞兒。

其中一位說，自己是個花錢特別大手大腳的人，總是忍不住超額花錢，她分析這和父母小時候比較節儉有關，所以她現在會報復性花錢。

然後轉頭問我說：「哎？你爸媽也很控制你的花銷，你怎麼就沒有我這種報復性花錢的習慣？」

另外一位覺得自己文化水平和層次不太夠，對很多事情的看法也比較淺顯。自從知道了「原生家庭」這個詞兒之後，她恍然大悟，原來自己現在的這些問題都是因為父母不會教育，後來幾次聊起父母的時候，她都是一臉的嫌棄。

我當時心裡默默的想，我的爸媽連初中都沒有畢業，可是這並不妨礙如今我考學讀書，並且成為身邊朋友公認的「有涵養、有文化」的人。

近幾年，原生家庭一詞兒特別火熱。經常看到的文章標題動不動就是「原生家庭有多可怕…」「原生家庭帶給一個人的輪迴傷害…」原生家庭似乎成了一個人出現問題的唯一原因。

原生家庭真的應該背著個鍋嗎？

1、沒有完美的父母，能不能有優秀的小孩？

董卿在一次採訪中表示了父親對其過分的嚴苛，從小不讓董卿照鏡子，也不讓媽媽給她買新衣服，不允許她有任何文體活動。

從高一到高三的寒暑假期間，父親都讓她去打工，到賓館裡當清潔工，每天掙1塊錢，一個月也就掙30塊錢。

這樣的父母在小孩心中是極其不講道理，也十分令人害怕的。年少無知時，董卿也埋怨過父母，討厭過父母，然而沒有父親曾經近乎不近情面的逼迫，可能也不會有如今唐詩宋詞張口就來，在事業上取得了巨大成就的董卿。

如今已經成為央視一姐的董卿終於理解了父親為什麼會那樣對她，董卿的父親出生在小縣城的農村家庭，家境貧困，憑藉自己的一切努力刻苦學習，才最終考取復旦大學，改變了自己的命運。因此他將這一方法用在自己的教育中，要求董卿必須要刻苦，要吃苦。

董卿說：「真正勇敢的人，從來不是沒有眼淚的人，而是含著眼淚繼續奔跑的人。」董卿說「父親已經不可能改變了」，而董卿擅於從中汲取積極的一面，練就自己的本領，最終走上了屬於自己人生的巔峰。

這是董卿對於原生家庭給出的最好答案。

創辦了風靡全球的奢侈品品牌香奈兒的可可·香奈兒，出生於救濟院，父親在生下她之後因為不想承擔責任，就將他們母女拋棄。母親軟弱無助的淚水、極其貧窮的生活，這些就是香奈兒的童年。後來香奈兒的母親在極度奔勞和病痛中患肺炎去世，那時的香奈兒才11歲。母親去世後，父親終於回來了，卻把她安置在孤兒院後從此一去不復返。

不幸的童年沒有磨滅掉香奈兒對美的追求，她一手創辦了香奈兒品牌，締造了偉大的時尚帝國，她本人更是化身獨立優雅的風向標，被一百多年來的女性熱烈追捧。

香奈兒受過家庭的傷，可是如《肖申克的救贖》中那句經典的台詞「有些鳥兒註定是關不住的，他們的羽毛過於閃亮」，最終香奈兒還是成為了萬眾矚目的明星，成為了被記入史冊的時尚女王。

這樣的案例數不勝數，不信你試著去翻一翻那些獲得成就的名人履歷，很少有誰是天生擁有一對完美的父母和一個被人羨慕的童年的。

這世界本就不存在完美，關鍵在於處於其中的人如何應對。

2、 究竟誰在抱怨？

我曾經因工作原因，在畢業時對同校的同學進行過一次採訪。我的母校是北京外國語大學，在外語領域全國排名第一，整體來說，學校為學生提供的平台、教學水平和資源都相當不錯。

儘管如此，當我採訪了多位同學後驚訝地發現，同學們對於母校的態度分為兩種。一類同學對學校充滿了感激，他們一致的表示，大學四年的時光帶給了他們很大的成長，學校教會了他們校訓那八字「兼容並蓄、博學篤行」，並表示將來也會繼續踐行。

而另外的同學則對母校沒什麼感覺，甚至流露出了些許不滿意和抱怨。

這些同學當時的處境分別怎麼樣呢？對學校表示感激的同學前景極佳，考上劍橋的、通過外交部遴選的等等。抱怨學校的同學呢？當時的前景不佳：考研失敗的、還沒找到合適的工作的等等。

如果把學校比作每一位同學的原生家庭，那麼他教養出來的孩子前途本該一致，為什麼大家的差距會如此之大呢？

就我的觀察，前景絕佳的幾位大學時候都極其用功，去外交部的那位男生，幾年如一日6點起床學習背誦外語、長期按時堅持身體鍛煉，不缺席任何課程，連續幾年都是年級第一名。

越是對學校有抱怨情緒的越是此刻狀況不好，也越是大學時候沒有好好努力學習的。但是最終，他們沒有從自己身上去尋找原因，反而很自然的怪起學校來。

那時我一下子意識到，不是學校給那些優秀的人提供了更好的資源，而是那些優秀的人憑藉自己的努力做出成績並對學校充滿感激。也不是學校虧待了那些不夠優秀的同學，而是那些同學沒有做出自己期待的成績才抱怨學校。

3 「認知失調」讓原生家庭成了背鍋俠

心理學中有一個概念叫「認知失調」，說的是「一個人的行為與自己一直以來的對自我的認知（而且通常是正面的、積極的自我）產生分歧。」

這在生活中常有發生，比如我們一旦犯了錯，就會出現我們想像中的自己和現實中實際的自己相矛盾。這時候，我們會產生不愉快、懊惱的負面情緒。

為了避免認知失調帶來的痛苦，人就會找各種借口進行自我保護。比如通過「記憶偏差」的方式，去選擇記憶一些對自己有利的內容，或是運用「自我合理化」的方式，為自己找到各種理由，把問題歸咎於其他人。

一位心理學醫生表示「這是一種本能反應，和道德高低無關。即使是那些修養很高的人，第一反應仍然是「我沒錯，錯不在我」，主動從自身找原因是他們經過學習後的第二反應。」

原生家庭這件事，也是如此。龍生九子各有不同，我家就是兩個孩子，我和妹妹成長在同樣的家庭，我桀驁不馴，但是積極上進。我妹妹溫順柔和，同時也比較安於現狀；我情緒波動大，容易生氣，我妹妹情緒穩定，沉穩淡定。同一對父母經常教育出截然不同的孩子，可想而知，父母只是影響孩子成長的一個因素，並不是唯一的決定性因素。

4、對自己負起責任 是成年人唯一的出路

實際上每個人的生命都有兩次，一次來自於父母，一次來自於18歲後的成人和再生，成年人要學會對自己的生活負責。

父母給了我們生命，把我們帶到這個世界上，供養吃穿，提供港灣已是盡到了為人父母最基本的責任，他們文化水平不高，眼界見識有限，表達愛的方式十分「粗鄙」，這些都不是他們的錯，我們不該在追星時總是說「他已經很努力了」，對於自己的父母卻少不了百般的苛責。

更重要的是，別把自己成為合格成年人的那次機會浪費掉，別拿父母做自己不努力，不對自己負責的擋箭牌。

從小我們就讀書受教育，18歲後更是有非常多的機會重塑自己，選擇什麼樣的價值觀？選擇什麼樣為人處事的方式？選擇什麼樣的工作及環境？選擇和什麼樣的人接觸和交往？選擇把時間花在哪裡，就會成為什麼樣的人。

一個人對現狀的不滿意，終究是對自己的不滿意。解決的方法也沒那麼神秘，如果你對現狀不滿意，想想自己具體出了什麼問題，誰做的比你好，去請教他們是怎麼做到的，然後照做；哪本書講了你身上的這個問題，去閱讀並踐行書中早已為你準備好的方法；主動尋找線上線下的免費或付費課程，聽課並付諸行動；

改變往往會在持之以恆的行動後自然的發生。

托爾斯泰說「一個人若沒有熱情，他將一事無成，而熱情的基點正是責任心。」

抱怨和推脫對我們沒有任何好處，對自己負起責任，是往後餘生的最佳出路，幾乎也是唯一的出路。期待你早日把自己的命運握在自己的手裡。
