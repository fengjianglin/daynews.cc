離婚也能致富，每月生活費達230萬，家裡的愛馬仕包包堆成山

2019-12-16 981條評論 2,791次閱讀 389人點贊

在中國，如果兩夫妻離婚的話，很多結局都是兩敗俱傷，甚至還有不少的女人會錢財兩空，日子會變得異常的拮据！不過現實中就一個非常成功、而還讓很多女人羨慕的成功離婚案例：那就是遠在新加坡的蔡欣穎 Jamie Chua 跟她前夫魯迪安 Nurdian Cuaca 的離婚，不僅僅讓蔡欣穎從此富裕起來，而且日子還過得相當奢華！

原來魯迪安 Nurdian Cuaca 原本就是印尼富商，據說他的財產是富可敵國，跟蔡欣穎 Jamie Chua 協議離婚後，兒子和女兒跟隨蔡欣穎，每月的生活費由魯迪安來全部支付，數目驚人，高達230萬人民幣！

蔡欣穎 Jamie Chua 自從每月有了這個穩定收入後，那恢復單身的日子是過得相當的滋潤，甚至連去超市買東西都連被2個愛馬仕包包；再加上她天生就聰慧、很有經商的頭腦，每月收入都是上千萬，自然日常買奢侈品包包就跟買白菜似的，手到擒來！

對於擁有雄厚經濟實力的蔡欣穎 Jamie Chua 來說，平時除了保養、參加一些派對活動之外，剩下的時間基本上就是買奢侈品包包，當然，目前她到底有多少個愛馬仕包包，應該連她自己都沒有統計，但是小編粗略看了一下，絕對超200個，就如上圖那種場景，首次看到，還以為是HERMES官方的產品展示了。

蔡欣穎 Jamie Chua天生就顏值高（人家本來就是一名空姐），作為一名名副其實的「富婆」，平時裝嫩、炫富，那是必須要做的功課， 比如說上圖一條粉嫩色連衣紗裙，穿在身上就如同20歲出頭的小仙女，當然背景那一面牆的愛馬仕包包也是非常惹眼，所以說她家的愛馬仕包包堆積如山，一點都不誇張！

現在看完了這篇離婚也能致富的成功案例，大家有什麼心裡話要說嗎？還是一起來留言吧，當然如果覺得小編寫得不錯，也麻煩大家分享、收藏，最好關注一下，非常感謝！
