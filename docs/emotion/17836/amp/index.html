個性化哲理短句：發自己的光，但別吹熄別人的燈

天天要聞
2019-12-18

一、果實熟透了才可以任意採摘，思考沉穩了才可以充分表達。

二、對不知道的事，坦率說「不知道」，才是最輕鬆最省心的。

三、沒有親眼所見，就不用急著用你的嘴巴來證明。

四、人紅是非多，如果你鬥不過「是非」，不如暫時放棄「當紅」，以退為進為好。

五、忍得了就忍，忍不了就改變，改變不了就包容。

六、遇事緩一緩，說話停一停，很多東西便會空前清晰。

七、如果別人朝你扔石頭，就不要扔回去了，留著做你建高樓的基石。

八、莫與小人為仇，小人自有對頭。

九、不忘形驕傲，不刻意渺小。

十、每個人總有不願意公開的秘密，千萬不要苦苦相逼。

十一、有些事，很多人都在做，你不做，並不代表你錯了。

十二、求神不如求人，求人不如求己，求己不成還要求人。

十三、發射自己的光，但不要吹熄別人的燈。

十四、簡單事不爭吵，複雜事不煩惱，發火時不講話，生氣時不決策。

十五、當你意識到自己錯了，你的思路還是對的。

十六、寧可保持沉默像傻子，也不要一開口就證明自己是傻瓜。

十七、沉默是反擊無恥的無聲風暴，憤怒是抗擊外辱的有效「自殘」。

十八、別把自己不當人，別把自己太當人。

十九、心中有敵，天下皆為敵；心中無敵，無敵於天下。

二十、欲勝人者先自勝，欲卑人者先自卑。

二十一、許多人在重組自己的偏見時，還以為自己是在思考。

二十二、你無法判斷別人是好人還是壞人，但你自己可以做一個好人。

二十三、忍得一時風亦靜，容得萬事心自清。

二十四、諾不輕信，故人不負我；諾不輕許，故我不負人。

二十五、如不識貨，一時窮；如不識人，一世苦。

二十六、笨人的自知之明在於，他能巧妙地運用其傻勁。

二十七、不要急於表達自己，那不是有見地的體現；也不要急於否定別人，和諧共存才是生存之道。

二十八、傷害你的人，不是比你強大就是比你弱小。如果他比你弱小，寬恕他；如果他比你強大，寬恕自己。

二十九、如果你想走到高處，就要使用自己的兩條腿，不要讓別人把你抬到高處，不要坐在別人背上。

三十、做人要自信，但不能自信得過於狂妄；做人應低調，但不能低調得失去了自信。
