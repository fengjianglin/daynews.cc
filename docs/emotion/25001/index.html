成長到後來都是改變自己，接納，寬恕原諒，感恩，覺察，無我十六

2019-12-23 2,685條評論 3,772次閱讀 1706人點贊

那麼這裡面還有一個部分，大家要記住的，他從終極的力量來講，其實直覺力是我的那種拖延的感覺在哪裡，我找找拿出來，拿出來沒有，忽然之間你觸摸到最深層的力量，忽然之間原來不用靜心，原來這些都不存在的，只是自己創造出來的，你忽然之間感覺很有力量，這時候也是一種直覺，那麼直覺力他有很多的點就是，忽然身邊的人給你一個建議，這個建議很對，這也是直覺，他沒有內和外。

忽然一個人讓你很痛苦，他激怒你啦，你總是痛苦，這也叫直覺，其實直覺在提醒你的是什麼呢，你內在有個痛苦，其實再透過另外一個人來更痛苦，然後你才尋求改變，很多人的成長都是因為太痛苦了而改變，為什麼呢，你內在和外在產生了一種衝突，其實是你的內心沒有改變，發現很多的成長到後來都是改變自己，接納自己，然後寬恕原諒，然後再去祈禱，感恩，到後來再去進入到覺察，覺知，等到後來進入無我，再布施，再慈悲等等，到後來都是一種和解。

問：你現在到什麼狀態了？

答：回歸。

我們的狀態是講不出來的，就像開悟，這是一種體驗，有很多東西是語言講出來的就不是。雖然我們的語言在表達，其實這些語言都是無效的，講的在哪個階段哪個階段，但是你要感覺講是這樣，要感覺的。

例如你閉上眼睛，用心去感受，我和這個世界的關係是什麼。其實古人的智慧是什麼，這個世界這麼好玩，以前沒有科技，這個星星在告訴我，這個太陽是有什麼神，他們在探索，探索的點和我們現在是不一樣的，沒有思考就沒有探索，我們覺得這個世界好玩不好玩是我們自己要好玩，不是這個世界不好玩。

如果我們一直被限制了幾十年，為什麼我們被限制在裡面一直不出來。在不好玩的感覺里，後來發現現實世界真的不好玩，我活在一個不好玩的感覺里，我活在一個拖延的感覺里，就真的在拖延，活在一個感覺當中，沒有懷疑這個感覺。有的人天天憤怒，說要改變這個憤怒，但從來我在憤怒的感覺里，但這個憤怒的感覺在提醒我什麼，可能有的人會說，沒有學會愛，就像她感覺的這個世界不好玩，一直活在不好玩的感覺里還天天在這個世界玩。

一直想從不好玩的感覺跳出來，從哪裡跳出來？首先不知道從哪裡跳出來。因為你創造出來一個不舒服的感覺，然後你還要從這個感覺里跳出來，是從你的世界跳出來，還是從你的感覺跳出來，還是不玩了遠離這個世界，從感覺中跳出來，你可以賦予自己另外一種感覺，這個世界可能也挺好玩的。

探索，首先是我願意去感受這個世界，因為我們從來沒有感受世界，這句話非常重要，雖然這句話非常簡單，我們一直在評判這個世界，沒有感受這個世界，你覺得這個世界不好玩，是你的一個評判。

你可能經歷一個事情，這個人和這個單位，你可能做了一個工作，這個單位都不好玩，後來你把整個世界都否定了。有的人問我，我今年不順，是不是以後永遠不順的啊，我說我不是算命的。就是這裡面我們以偏概全了。

所以呢，這裡面在直覺的探索當中內在智慧的探索，我在評判這個世界還是在感悟這個世界。這句話非常非常重要的，這句話其實你能經常去感受的話，你這個時候，來到一個環境當中啊，然後你發現這些人這個人一看就很惡，低級，再看看這群人又怎麼樣。走了。為什麼？因為你評論了這個，因為你的評論拒絕了體驗，那你到了以後，這些人可能不好，但是我也感悟體驗一下。就當一種學習和成長，然後你去感悟跟他們交流了，感悟每個人的那種東西啊，我透過學習的態度，去工作啊，學習態度去體驗啊，而不是一種高高在上的地方，而不是一種自私評判，其實我們很多人都在評判。

然後呢，自己還不知道自己的問題，總覺得外面的世界不好，但是自己一直活在一個評判里，把自己封住了封在裡面，戴了有色眼鏡看這個世界，其實這個世界沒有那樣。像這個水，我喝好喝，她說不好喝，其實每個人活在了自己的世界裡面，它到底好喝還是不好喝，等到你渴的時候，啥都喝，沙漠里只能喝雪，喝怎麼樣。

很多的時候，我們在評判這個世界，我們的評判太多了，我們限制了自己，不是這個世界限制了我們，有些規則和道要去尊重的，法律要尊重，有些道和規則就要尊重，你不可能天天在大街上開著車，橫七豎八的到處撞，那就要來制約你，法律來制約你，其他來制約你。
