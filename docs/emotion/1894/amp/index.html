“我想成為爺爺，啥不幹還有錢花”孩子理想驚人，媽媽回答太經典

鏡子
悠悠今年六歲了，馬上就要上學的她，突然有了理想：「媽媽，我這輩子就像成為爺爺這樣的人，一天什麼也不用做，但天天有錢花。」小悠悠一面說，一面望著窗外，神情很是陶醉。

面對悠悠這麼「不正經」的理想，一邊的奶奶直抱怨爺爺。可媽媽卻淡定自若，甚至對悠悠說的話還能面對微笑。好半天，悠悠又問媽媽：「媽媽，你覺得我的理想怎麼樣？」

「挺好啊」，媽媽的回答讓奶奶長大了嘴巴。「不過，悠悠，你知道爺爺像你這麼大的時候都在經歷些什麼嗎？」悠悠茫然的搖著頭。

「爺爺小的時候，冬天沒有鞋子穿，一年四季吃不上一頓飽飯，但這樣的艱苦也沒能改變你爺爺的學習興趣，他爬過學堂的牆根，用草棍在地上練字。直到16歲當上兵，才有了筆和紙。不過那個年代當兵相當於出生入死。正是有了這樣的經歷，爺爺才有資格過上老有所養的好生活。」

​悠悠這時候已經聽入了神。媽媽接著說：「悠悠，如想成就爺爺這樣的生活，必須得吃爺爺當年所吃的那些苦，你願意嗎？」悠悠的頭搖得好像撥浪鼓。

「那你將來可怎麼辦呢？」媽媽眨著眼睛問悠悠「可我不想離開媽媽，也不想沒有飯吃」悠悠眼圈紅了，眼淚就在眼淚里滴溜溜的打著轉。

「其實，時代不同時，你不需要離開媽媽，也不需要餓著肚皮，但學習上的苦你是要承受得住的，吃過了這份苦，你才對得起自己未來的生活，才能在年老的時候不遺憾。」媽媽說的悠長而認真，悠悠聽著聽著，使勁了點了點頭。

事實上，在現實生活中，有這樣理想的孩子還真是挺多的。不過並不是所有的父母都能像悠悠媽媽這樣講究方式方法，且沉得住氣。絕大多數的人都會覺得，孩子擁有這樣的理想真是太糟糕了，太沒有出息了，卻忘了。我們小的時候視乎也是做過這樣的夢的。

10歲以前的孩子，所謂理想，多數都還會發生變化，所以，對於孩子的不良理想，當父母的不要生氣，而是要注重引導。就像悠悠這樣，不動聲色，卻有意識的將爺爺的現狀與努力鏈接起來，讓孩子懂得一個道理，這世界上沒有什麼不勞而獲，每一個人的成功背後都有著別人不了解的艱辛和磨難，也只有熬過了這些，人生才能走向成功，才能富有意義。
