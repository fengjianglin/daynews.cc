農村一哥哥火了，13歲就要養5個弟妹，完美詮釋：長兄如父

天天要聞
2019-12-20

文丨洛芸（文章原創 ，版權歸本人所有，歡迎個人轉發分享）

時下家庭經濟條件都比較不錯，可以說孩子們是在蜜罐中長大的。父母們對於孩子的需求更是有求必應，物質上的極度滿足在各個家庭中較為常見。

父母們對孩子的養育從吃飽穿暖轉移到了搶佔教育資源上。經濟條件的滿足使得父母們更加註重孩子的素質培養。不過也有一小部分家庭的經濟條件並不如人意，那麼生活在這樣家庭條件中的孩子將會面臨怎樣的生活窘境呢？

案例：

前幾天在網路上看到一條新聞，一名13歲男孩輟學在家照顧5個弟弟妹妹，最小的弟弟才只有三歲。孩子們的父親入獄，而母親為了養育孩子們選擇外出打工。

照顧弟弟妹妹們的責任就落在了身為老大的男孩兒身上。同齡的孩子還只知道伸手跟父母要錢時，男孩兒便已經知道要節約家庭開支。每每吃飯的時候男孩兒便會借故躲起來，因為他害怕飯菜不夠吃。

男孩覺得無助的時候便會躲起來一個人哭，因為他害怕弟弟妹妹們看到。雖然家庭經濟不好，但是他仍然希望弟弟妹妹們能夠過得快樂。

雖然看著同齡的孩子都在無憂無慮地上學，但是哥哥並不後悔，因為他要照顧弟弟妹妹們，只要弟弟妹妹們能上學，她就很開心。

新聞發布後，網友們紛紛指責男孩兒的父母不負責任，而更多的網友則表示男孩兒是中國好哥哥，即便家庭環境很糟糕，但是有這位哥哥在弟弟妹妹們也生活得很有安全感。

分析：

相信這位哥哥肯定也希望自己可以和同齡的孩子一樣，擁有無憂無慮的童年，擁有可以肆無忌憚撒嬌的懷抱。

但是為了弟弟和妹妹，他只有負重前行，承擔起家庭的責任。可以看出他在小心翼翼地保護著弟弟妹妹們的自尊心，既然自己也有孤獨無助的時候，但是一直表現出勇敢、堅強。

好的哥哥對弟弟妹妹的影響

影響1：有助於形成樂觀的品格

在生活中人們總會遇到各種各樣的挫折，而好哥哥就像一個明燈，指引著弟弟妹妹們樂觀地看待眼前的一切。雖然已經不在身邊，但是有哥哥的存在，弟弟妹妹們並不會感覺到家庭的缺失。

影響2：有助於形成堅毅的品格

在面對生活的窘迫，有哥哥在弟弟妹妹們更加能夠懂得堅持和忍耐。在困苦的環境中養成堅毅的品格，這對於弟弟妹妹的人生成長也是大有裨益的。

影響3：激發對生活的熱愛

弟弟妹妹會被哥哥熱愛生活的態度所感染，即便吃得不好，穿的不好，但是弟弟妹妹們去會覺得很溫暖。如果在弟弟妹妹們並不會覺得自己是留守兒童，而家還是一個完整的家。

如何做一個稱職的哥哥？

（1）有責任感

作為哥哥，在家庭責任感方面要強於弟弟妹妹們。不僅僅因為哥哥的年紀比弟弟妹妹們大，更因為哥哥的身份所在。有責任感的哥哥是弟弟妹妹們的榜樣，更加能夠助力弟弟妹妹們健康成長。

（2）有愛心

作為哥哥，應該懂得關愛弟弟妹妹們。幫助弟弟妹妹們應付生活中的困難，這不僅有助於手足親情的維護也有助於建立弟弟妹妹們對哥哥的信任感。

可以說家有哥哥是弟弟妹妹們的福氣，在多子女家庭中手足親情的維護很大程度上取決於哥哥的榜樣作用。大家對於稱職的哥哥是怎樣看待的呢？對此又有哪些經驗分享呢？
