學會成為一個“不愛做聲”的女孩，評論：這種人真優雅

2019-12-16 923條評論 2,104次閱讀 83人點贊

靜坐常思己過，閑談莫論是非

今早打開微信，刷到了同事小李（簡稱”小李”）的朋友圈。上面寫著：”早上不小心起早了點，多餘的時間就練練字。”並配上一張太陽的圖片。

靜心練字是她最大的興趣愛好。

說起小李，她不圓滑，做事一般，但單位上的人大都比較信任她。只因，她有一個特點就是：不愛做聲。

這個不愛做聲不是不會說話，而是在她懂得在恰當的時候閉嘴。

在大家工作之餘聚在一起聊聊天，如果是聊一些很普通的話題，小李會說一兩句。

但是，如果繼續往下深聊，大家抱怨工作中的瑣事的時候，她從不多說一句話，也不附和。

別人不問她，她也不說。別人問她，她只回答”不知道”。

久而久之，我發現她說話很懂分寸，不該說的或者不想說的，絕對不做聲。她在3種情況下不做聲，不會隨意表達自己的態度:

1.閑話不做聲

"靜坐常思己過，閑談莫論是非。"

這句話用在小李身上再合適不過了。 她不會去主動說閑話，也不會附和著他人。

最多說的一句話就是：”我不知道哦！”，大不了就笑一笑，她也不會去傳閑話。所以在大家心目中，她是一個值得信賴的人。

當一群人圍在一起說閑話的時候，不做聲，不參加話題討論。

在閑談的時候，總會不經意間說出一些是是非非，更何況人多口雜。有人說閑話，就有人會傳閑話，這就是社會的常態。

在聊天過程中，還能夠有趣，有笑，能讓場面上的人有面子，可是需要高情商的支撐。如果做不到，還不如不說。

  • 閑話不做聲其實是一種保護自己。

2.私事不做聲

小李今年結婚了，但是在這之前大家都不知道她有男朋友。

等她發喜帖的時候說了這麼一句話：”在一切未成定局的時候，男朋友對於我來說也只不過是個普通人。當我們結婚了，我才會告訴大家，我的世界從今往後多了一個人。“

有時，我們會不經意間說著說著就給自己的私事說出去，還和人家附和著：”你別告訴別人哦！”。說完又擔心人家會不會說出去，給自己心理增加麻煩。

私事，是自己的事，也是內心裡比較敏感的事情，能藏在肚子里就讓它腐爛在裡面。你也不能保證誰會拿你的私事去開玩笑。

  • 私事不做聲，是對自己的一種尊重。

3.家事不做聲

每個人家裡多多少少總會有一些雞毛蒜皮的事，有夫妻關係，有婆媳關係，有母女關係等。

但從未聽小李提過。而且她主要報喜不報憂。她本人是個很熱愛生活的人。也不會去抱怨什麼。

當其他人在討論婆媳關係的時候，順便問了她一句：”小李，你婆婆對你怎樣？”

她不說怎麼怎麼好，就說了一句：”好，我很喜歡我婆婆。”大家知道聊不下去了，也就沒說什麼。

  • “清官難斷家務事”，家事不做聲，是對家人的一種理解。

不愛做聲的女人不是柔弱，也不是笨的接不上話，而是她內心裡知道不該做聲的時候選擇閉嘴。

不愛做聲的女人，這種女人很優雅，內心裡都藏著一個帥氣的自己。
