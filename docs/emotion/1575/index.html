生活，無需複雜，簡單就足夠了；人生，無需繁華，只需平淡就夠了

2019-11-30 0條評論 20次閱讀 0人點贊

1、生活不可能一帆風順，總會有波折，總會有險阻。生活是個愛開玩笑的孩子，也許今天給你所有，明天又會讓你一無所有，無需煩惱，該來的總會來，再黑的夜晚也會有黎明到來的那一刻。不管生活有多麼曲折，只要擁有幸福的態度就能挺過漫漫長夜，就能迎來美好的明天。

2、痛苦的情緒和不愉快的記憶，如果充斥在心裡，就會使人委靡不振。因此，掃地除塵，能夠使黯然的心變得亮堂。

3、人生路上常有風雨，需要一個好的心態。再難的路，只要不放棄，一直走下去，總會走到終點；再重的擔子，笑著是挑，哭著也是挑，又何必讓自己難堪。

4、即使生活有一千個理由讓你哭，你也要找到一個理由讓自己笑，這就是人生！給自己一些時間，原諒做過很多傻事的自己。接受自己，愛自己；過去的都會過去，該來的都在路上。早安！

5、人總是說人抵擋不過時間，其實是時間抵擋不過我們的善變。

6、優雅的人生，就是用平靜的心，平和的心態，別再為錯過了什麼而懊悔。你錯過的人和事，別人才有機會遇見，別人錯過了，你才有機會擁有。人人都會錯過，人人都曾經錯過，真正屬於你的，永遠不會錯過。

7、生活里不必要渴求別人的理解和認同，因為別人也沒有這些義務。靜靜的過自己的生活，心若不動，風又奈何。你若不傷，歲月無恙。

8、勇敢的人不是不害怕，而是怕也要向前走；閃亮的人生不是未經黑暗，而是在黑暗中也努力燃起一道光。學著在跌倒中忍痛，學會把哭調成靜音。

9、生活，無需複雜，簡單就足夠了；人生，無需繁華，只需平淡就夠了。人生如行路，一路艱辛，一路風景。目光所及之處，就是自己的人生境界。

10、人生首先要是望遠鏡，看遠；再就是顯微鏡，看細；接下來是放大鏡，看透；其次是太陽鏡，看淡；最後是哈哈鏡，笑看生活。給自己一些時間，原諒做過很多傻事的自己，接受自己，愛自己。過去的都會過去，該來的都在上！！

11、不是每個人生來就是長得漂亮，但每個都可以讓自己活得更美好 ，不迷茫，不依附，有自尊，獨立，就是每個人最大的精彩 無論你單身與否！好好努力，你才可能變成自己所想的那種人，走自己想走的那條路!

12、以為蒙上了眼睛，就可以看不見這個世界；以為捂住了耳朵，就可以聽不到所有的煩惱；以為腳步停了下來，心就可以不再遠行；以為我們需要的愛情，只是一個擁抱。
