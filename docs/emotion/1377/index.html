如今有些老人不願意幫忙帶孩子，這是為什麼呢？老人說出3個理由

隨著時代的發展，現在有很多農民工都開始到城市打工了，有的夫妻為了可以賺到很多的錢，於是就將自己的孩子託付給爺爺奶奶帶，對於這種情況，在農村還是非常正常的，但是現在為何有些農村的老人卻不願意幫忙帶孩子了，這究竟是什麼回事呢？下面我們就來看看農村老人是怎麼說的吧。

首先老人說自己的負擔大，而且因為觀念不同，非常容易和子女鬧矛盾。相信很多人都應該知道農村有很多老人在照顧孩子都是沒有什麼講究的，所以農村有很多孩子都是穿著舊衣服，然後孩子的性格也是非常調皮，整天都是爬高上低的，所以看到這種情況後，有些父母就認為是爺爺奶奶沒有將孩子教育好，但是他們在平時的時候又不會給爺爺奶奶孩子的撫養費，這樣就導致老人因為負擔大，容易和孩子之間鬧出矛盾。

其次就是老人的身體不好，經不起孩子的折騰，老人在歲數大了以後，精力就沒有年輕人那麼好了，而且身體的免疫力也是非常的弱，所以如果讓他們照顧孩子，時間斷還可以，但是時間長了以後，他們就會感到十分的累，畢竟有些年輕人都受不了，所以老人當然是更不行了，所以有些老人為了避免麻煩，於是就不願意幫子女帶孩子了。

最後就是現在有些子女不懂得感恩，導致老人出力不討好。對於這種情況在農村還是比較常見的，因為農村有很多子女都認為老人幫忙照顧孩子是應該的，所以他們為了上班就開始將孩子丟給爺爺奶奶，但是在外面，他們心裡又是不放心，擔心孩子被爺爺奶奶教壞了等等，對於這種情況，老人自然是不願意帶了。
