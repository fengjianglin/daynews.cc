情感：殘疾孩子被爺爺撫養大，多年後，親母來尋子，兒子卻無視她

2019-12-18 63條評論 1,092次閱讀 85人點贊

世間最偉大的東西，反過來後也就成為了最為殘酷的東西，感情就是這樣的存在！世間有三大情，親情、愛情、友情，如果非要排個順序，無疑親情就在第一位，因為這是血脈相連的至親之人，也因為是這樣的人，往往才容易傷害的最傷！

朋友斷裂了，還可以尋找新的朋友，但親人離棄了，卻是一輩子無法取代的了，朋友可以有很多，但親人就那幾個。

世間又有多少人真的可以做到拋妻棄子那樣呢，但也有，面對這樣的情況，除了讓人憤慨和唏噓還能怎麼樣呢，下面我們來看看一則這樣的情感案例！

江離明是天生的殘疾人，自出生的那一刻，右手就是畸形的，因為先天的缺陷已經很不公平了，可是禍不單行，江離明還被自己的親生母親給拋棄了，兩歲的時候，母親就把他丟給了他的爺爺，自己離開了這家，再也沒有回來過。

離明家就他爺爺和父親，但是父親因為之前犯了事情一直蹲牢子，好像有二十幾年，至於是什麼事情爺爺也從來沒有向他提起，只是撒謊說父親去了很遠的地方，母親也過去陪他了。

離明小的時候常常被其他小朋友欺負，原因就是他的右手遭人嘲諷，加上他自己沒有父母，常常被人欺負後也不知道找誰。而他爺爺平時也在後院的小作坊里，爺爺年紀大了，做體力活做不了，好在他會一門手藝，就是編織草帽和其他編織品，他每天都會編織一些草帽和其他什麼的挑到街上叫賣。

也因為要掙錢的緣故，離明很少在白天里和爺爺待一起，後來到了上學的年紀又得去上學了，離明因為成長背景的緣故，性格有些孤僻，唯一的笑就是在爺爺的面前，在爺爺面前他才會變得開朗一些。

他讀小學那會，學校要他們買校服，因為校服比外面的衣服還有貴些，所以離明沒有開口問爺爺，直到有一次爺爺見到和他一起回來的同學身上穿的是嶄新的校服，才去找老師給他買了兩套。也許，離明的成長環境讓他更加懂事了吧。

高中後，離明也上了普通的中學，和普通的人一起讀書，他也不再在意別人看他的眼光，因為他習慣了，但儘管如此，他還是有幾位關係不錯的同學。後來分科的時候，離明去報了美術，也就是藝術班，因為他知道自己的原因，這樣的他如果將來能靠一隻手養活自己就得從事文類工作。

也好在他比較喜歡美術和性格比較文靜的緣故，他畫的比班上很多同學都要好，其他科成績也不錯，高考考了一所名牌美術學院，班上的同學都非常替他高興，而他爺爺也是非常的驕傲，在那後上街買東西，熟人都誇他有個有出息的孫子。

離明大學也十分刻苦，後來畢業和大學同學開了工作室，奮鬥了兩年小有成就，現在的生活條件已經非常不錯了，每個月也有差不多一萬塊，對於殘疾人的他來說已經非常的難得了。而爺爺年紀也大了，78歲的爺爺也挑不動擔子了，在老家那邊但也沒有隨孫子去城裡，他覺得生活還能自理，暫時不想打擾孫子的事業和生活。

但是離明也知道爺爺喜歡和老街坊待一起，於是就沒有勉強，他也經常會一個月回來一次陪爺爺。而就在他26歲那年，一個不速之客出現了，就是他的親生母親，他來認自己兒子來了，她穿著打扮十分華麗，像個闊太太一樣，胭脂水粉各種都有。

她來尋找當年拋棄的兒子，原因是她改嫁之後沒有生到孩子，因為自己身體的緣故，只有一位繼子，有錢的丈夫前陣子去世了留下了家產，她就想爭取一些回來，畢竟現在她是那個家的女主人了。就是沒有後感到很無奈，才來尋找當年丟下的殘疾的兒子離明。

離明也知道她的來意，一直不願搭理她，她還各種利誘說:「跟媽回家，那有房子車子」,,可是離明並不屑的回答：「拋棄的那刻起，我就爺爺一個親人」。這位母親並沒有放棄，還說：「都是媽的錯，現在給媽一個彌補的機會」可是離明最後也沒有理她，因為在他最需要的時候，這位母親不在他身邊，恰恰還是拋棄他的人！

以上的家庭案例，相信類似的還有很多，有些人沒有作為父母的擔當和責任，卻要生下孩子來這世界上受苦，而好在家貧出孝子，堅強的意志和逆境讓他們比普通人更加吃苦耐勞，也為自己換來了想要的生活。

如果沒有想盡到父母的責任，那就不要生下孩子，因為那是極度的不負責！你們怎麼看這類父母？

我是莫愚，為平凡的情感護航！
