世上最暖報恩！3年前姑娘救了一隻鳥，患癌後鳥兒竟救活了她

———————————————-

經歷過九死一生，再回味往事，Dina動情地感嘆道：

「當你做善事時，宇宙總有辦法給你同樣的回報。」

益美君今天故事的主人公，是Dina和一隻冠藍鴉。

他們初識還要追溯到3年前一個早上，那天Dina在自家花園的樹下，發現一隻奄奄一息的小鳥。

它弱小又眼中帶淚的模樣，任誰看了都忍不住心酸。

何況Dina還是社區遠近聞名，特別善良溫暖的姑娘。

Dina小心翼翼把它帶回家，並給它取名格雷西。

就這樣格雷西，從一隻被拋棄的小鳥，轉眼間成為Dina家中最小的家庭成員。

Dina的丈夫和女兒，本就特別喜歡小動物，格雷西這個毛茸茸小傢伙的到來，更是把他們高興壞了。

「一定是特別奇妙的緣分，讓我們走到一起成為一家人。」Dina的丈夫，這位前WWE職業摔角的大塊頭動情地說道。

但麻煩也隨之而來，Dina雖救助過許多小動物，卻從未飼養過小鳥，「面對這個楚楚可憐的小傢伙，她首先考慮的是怎麼把它餵飽」。

為此Dina特意跑去鳥類專家那請教，才知道要先給格雷西喂流質食物。

在一家人精心的看護和照料下，不到一年時間，格雷西就長出美麗的羽翼，它會在家周圍自由盤旋，和每個家庭成員都玩得不亦樂乎。

跟爸爸開玩笑

在小姐妹身上嘰嘰喳喳，表達自己喜悅的心情。

雖然格雷西和一家人打成一片，但Dina心裡卻清楚地知道：格雷西有一天終歸要回歸自然，大自然才是鳥兒廣闊的天堂。

因此伴隨格雷西成長的過程中，Dina不斷對它進行野外生存訓練，包括在樹枝間掌握築巢技巧，引導它走向昆蟲學會自主覓食···

而格雷西也格外聰明，什麼技能都是一學就會，它能迅速找到Dina藏在樹洞中的堅果，然後巧妙地啄出大快朵頤。

時不時還跟家人開個玩笑，把小東西藏起來讓家人找。

就連Dina寫字，它都要湊上一眼，有模有樣地效仿。

此情此景，讓Dina和家人甚感欣慰，如果當年沒把它帶回家，就要錯失聰明伶俐的小傢伙了。

可轉念一想，又有些落寞，格雷西掌握的技能越多，也就意味著它離開這個家的日子越近。

心中雖萬般不情願、不捨得，這一天還是到來了。

當格雷西有足夠的信心飛到附近的樹上，Dina和家人試圖將它放歸野外，然而格雷西卻一動不動，

一聲不吭的它用實際行動「抗議」：我不想離開。

Dina看著小傢伙久久不願離去，心軟下來，她告訴自己：

「就兩周，再多兩周，到時一定讓它離開。」

兩周後，格雷西真的飛走了，眼睜睜目睹它飛離，一家人哭成淚人。

「媽媽，格雷西還會回來嗎？」「我想不會了吧，它會在野外組建自己的家庭，日子久了或許就會忘掉我們。」

可讓所有人始料未及的是，第二天一推門，小女兒就發現格雷西又回來了，她高興地在院子里大喊大叫。

Dina聽聞也喜不自禁，她想不到這個小傢伙還能如此眷戀舊情。

從那之後，格雷西每天都在院子里往返，每次待個10分鐘左右就走，每天堅持不懈地能往返15次。

或許它始終記得這一家人的善良、溫暖，因而即便在野外築巢，每天也堅持回家看看，時不時還給家人帶些小禮物。

而Dina一家對此也格外珍惜，它回家的時候家人就陪它玩，即便什麼都不做，靜靜地待一會，一家人也覺得特別滿足。

因為愛，我願意跋山涉水來到你身旁。

這樣溫情而規律的互動，如果沒有意外，或許將一直持續下去。

然而好景不長，2016年1月，在一次身體檢查中，Dina被查出卵巢癌。生活一夜間陷入黑暗。

家裡的歡聲笑語，很快被Dina病痛折磨的聲音取代，而格雷西似乎感受到家中緊張的氛圍，那天飛走後，它很長一段時間沒再回來。

「聰明的它大概感覺到家中的變故，所以這次真的離開了。」

正當所有人以為它不會再歸來的時候，格雷西不僅飛了回來，而且每次停留的時間更長。

「那時我正與癌症抗爭且身體無法從枕頭上挪開，它會飛到我身旁，落到我的肩膀或臂彎上，時不時跟我說會兒話。」

它似乎在提醒我：來到這個世上，有多麼不易，面對困難要充滿勇氣。

神奇的是，有了格雷西長久的陪伴，Dina的精神狀態完全大變樣，漸漸地她沒了淚水和抱怨，笑容越來越多地浮現在臉上。

她也更願意走下病床，感受戶外幸福的陽光。

「它在我最需要的時候來到我身邊，完全改變了我的信念與生活。

如果沒有格雷西，我抵抗癌症的經歷將完全不同。它是世界上最甜美、最善良的小鳥。」

讓人難以置信的是，在積極治療下，Dina兩年後戰勝了癌症，一家人，包括醫護人員，都感到不可思議。

「這絕對算得上奇蹟的病例。」

只有Dina心裡清楚，最黑暗的時刻，是格雷西給她以強大的精神力量。

經歷過化療和手術的折磨，每天有格雷西落在肩上，就是最踏實幸福的時光。

如今雖已戰勝癌症，格雷西似乎還是放心不下，每天依舊雷打不動地飛回來看她，靜靜地陪伴她。

丈夫有感於這個小天使帶給家人奇蹟般的轉變，將格雷西和Dina的故事寫在書中《Gracie’s Wild Adventures》。

結果無數人被這隻知恩圖報的小鳥感動，書籍前所未有的受歡迎，也一定程度上緩解了Dina癌症治療導致家庭經濟窘迫的狀況。

無形中這個小天使，又幫了Dina一家大忙，萬物皆有靈，格雷西雖然跟人類語言不通，但它卻懂得用自己的方式默默陪伴。

陪伴，是最長情的告白。

人如此，動物亦是。

但行好事，莫問前程，願你每一段孤獨煎熬的時光，都有愛你的人寂靜陪伴。
