青春不告而別，歲月不請自來，趁還年輕就趕緊去旅行

2019-12-19 1,103條評論 2,618次閱讀 374人點贊

二十歲的年紀，要做八十歲想起來都會嘴角上揚的事兒。

願你在旅行的路上，把歲月變成最美的詩篇。

時間都去哪兒了還沒好好感受年輕就老了

生兒養女 一輩子

滿腦子都是孩子哭了笑了

時間都去哪兒了

還沒好好看看你眼睛就花了

柴米油鹽 半輩子

轉眼就只剩下滿臉的皺紋了

….

2014年春節，王錚亮的一首《時間都去哪兒了》深深打動了億萬觀眾的心。
我們在感嘆著時光易逝，我們在最美的青春時光許著最美的諾言。我們曾經夢想著仗劍走天涯，我們曾經夢想這看遍世界的每一處風景，我們給年輕寄予了太多的期望。

可是夢想總在現實的沙海里折沉，我們開始為生活讓步，告訴自己年輕時拚命掙錢，等老的時候存夠一定錢，再去環球旅行。
然而事實是，到了那個時候，你可能會因為年紀太大，身體不好，體力不行，心裡膽怯等各種原因，很多東西都沒法體驗了，甚至稍微遠一點的飛行都沒法承受了。

趁著年輕，給自己一次說走就走的旅行吧！別等青春不告而別，歲月不請自來的時候才追悔莫及。
去看大海

只是靜靜的觀望著海面，所有內心的嘈雜都會隨風而走，陽光、鮮花將你包圍，如有什麼生活是令人嚮往的，那一定是「面朝大海，春暖花開」。
遠方的海平面、天空中飄著輕輕的雲、海浪拍打著沙灘，目之所及的一切都在述說著無限美好。

去看沙漠

從未想像過沙漠的蒼茫與遼闊，飄揚的駝鈴悠揚清脆，喚醒了戈壁沙灘的沉默，喚醒了心中的熱情。
大漠、孤煙、流沙、落日，或許再多的語言都顯得蒼白。

去看草原

旅行的意義在於體驗生命中的每一種不可能，畢竟生活中可以沒有榮華富貴，但不可以沒有清風暖陽。

草原的清風溫柔的撩起你的頭髮，輕輕地閉上眼睛，將你的思想放空，細細的感受這一切。

去登高山

海子曾說：「你來人間一趟，你要看看太陽。」爬上高山，守候一場日出，無言的浪漫其實就在身邊。

山坡上翠綠的樹木，小道兒邊的野花，滋養著我們在城市中日漸枯萎的心。

去賞名勝

在歷史的長河裡讓自己更加沉穩，熟懂歷史便能洞悉人生百態。
去看看各地的名勝古迹，去觸摸一下歷史留下的滄桑記憶，去感受一下古人的聰慧，去飽讀一次歷史的經典。
如此，你世界一定會更加精彩！
去逛古城

踏上青石板老街，沐浴在古城的陽光里，彷彿置身與千年輪迴，你會明白平淡的生活才是最有意義的存在。
或者覓一處幽靜的茶肆，品一杯醉人的香茶，坐看往來的遊客，聆聽各樣的故事，做一次超脫世俗的長行。
我們想去的地方有很多，可是我們的人生畢竟有限。在我們有限的人生中，趁年歲還輕，時光未老，眼還未濁，腿還能動。

給自己一次說走就走的旅行吧！別等青春不告而別，歲月不請自來的時候在才追悔莫及。
