父親的出現是一種獨特的存在，對培養孩子有一種特別的力量。

成都某學校在對學生進行了一個調查發現：男孩的中性化趨勢越來越明顯，比如膽小、怯弱、缺乏勇氣等。

實際上，這就是父愛教育缺失的表現，因為”忙”爸爸將教育孩子的重擔交給了媽媽和老人，爸爸對責任的履行越來越弱。

在孩子的成長中，爸爸的地位是毋庸置疑的，爸爸的位置和作用從來都不是媽媽所能替代的。

古今中外，有很多默默無聞的爸爸，用他們的愛培養出了揚名天下的人才。

達·芬奇的父親：充分發展孩子的興趣

達·芬奇的父親彼特羅是一位絕世好爸爸。

他教育孩子的信條, 就是給孩子最大的自由，發展他們的興趣。

達·芬奇6歲上小學、在學校中他不僅學到了很多知識，而且對繪畫也很感興趣。

一天達·芬奇在課上不認真學習，反而還畫了一幅速寫。

回家之後，達·芬奇讓父親看他的速寫，父親不但沒有批評他，反而誇獎他畫得好，並決定培養他在這方面的才華。

正是因為父親的鼓勵和開明，達·芬奇才能全身心地投入到喜愛的繪畫中。

刻苦學習的達·芬奇掌握了很多繪畫的技巧，終於成為一代大畫家。

安徒生的父親：保護孩子的尊嚴

丹麥童話作家安徒生出生在小城鎮中，那裡居住著大量的貴族和地主，而安徒生的父親是個窮鞋匠，母親是個洗衣婦。

貴族們怕降低身份，所以從不讓自己的孩子與安徒生玩。

雖然安徒生的爸爸對此非常氣憤，但他並沒在孩子面前表露出來，反而十分輕鬆地對安徒生說：”孩子，別人不跟你玩，爸爸陪你玩吧!

父親將安徒生的房間布置的像小博物館一樣，牆面掛滿了圖畫和瓷器，櫥柜上擺了一些玩具，書架上放滿了書籍和歌譜，就連門玻璃上都被畫上了風景圖。

父親常常給安徒生講《一千零一夜》等古代阿拉伯的故事，有時還給他讀喜劇作家荷爾堡的劇本，或是英國作家莎士比亞的劇本。

為了豐富安徒生的精神世界，父親還鼓勵他到街頭觀察埋頭苦幹的手工藝人、沿街乞討的流浪者、坐著馬車橫衝直撞的族等。

後來，這些活靈活現的人物縮影出現在《賣火柴的小女孩》《醜小鴨》等童話故事中。

莫扎特的父親：不惜放棄工作培養孩子

奧地利作曲家莫扎特被無數人譽為”神童”,他的許多作品被視為經久不衰的經典曲目。

莫扎特之所以取得如此大的成就，與父親的精心培養是息息相關的。

在莫扎特3歲時，父親就發現他常常靜靜地坐在一旁觀看姐姐練琴，而且還一臉認真的態度。於是，父親就對他進行啟蒙教育。

在父親悉心的指導下，莫扎特在5歲時就開始創作短曲，6歲時就會 創作鋼琴協奏曲。

後來，為了更好地培養莫扎特，父親放棄了在宮廷里當樂師的工作，將全部精力放在了莫扎特身上。

總而言之，若是沒有父親那份執著的精神、嚴格的要求和深厚的關愛，就不會造就出享譽世界的音樂大師。

奧爾森的父親：以身作則的榜樣

奧爾森是美國赫赫有名的企業家，曾被美國《幸福》雜誌評價為”美國最成功的企業家”,他是美國數字設備公司(DEC)的始創者。

在談及自己的成功經驗時，他總是會講到自己的父親，因為父親用行動影響了他的一生。

父親奧斯瓦爾德是沒有大學文憑的工程師，並擁有幾項專利，後來改行成為一名推銷員。

有一次，一位顧客想購買奧斯瓦爾德推銷的機器，但是他發現這位顧客並不需要這部機器，所以就極力遊說這位顧客不要購買，這讓老闆非常氣憤，但奧斯瓦爾德卻因此贏得了好名聲。

奧斯瓦爾德的誠實品德給3個兒子帶來巨大的影響。

孩子們都養成以父親為榜樣，誠信做人，全都成了著名的工程師。奧爾森本人就秉承了父親的優點，講原則、重誠信，因此在員工和商業圈中贏得了良好的口碑。

先不論這些故事是令人感動還是給人以啟發，他們都真實再現了父親對孩子深沉的關愛從而顯現出父親在孩子成長過程中的巨大影響力!

有的爸爸對孩子的成長或是不聞不問，或是態度粗暴少耐心，成為家庭教育中的”甩手掌柜”，對孩子的成長是極其不利的。

父親角色的弱化和缺失，或多或少會給孩子的心理帶去不安，父愛教育和母愛教育的結合才能鑄就完整的家庭教育。

父愛給孩子帶來的是誠實、無畏、敢於冒險等，這些恰恰是母愛缺乏的。

美國耶魯大學的一項研究成果顯示，由父親帶大的孩子往往更加聰明、智商更高，在學習和社會中也更可能取得顯著的成就。

既然父愛如此不可或缺，那麼，廣大父親們該如何表達父愛，最大限度地發揮父愛在家庭教育中的作用呢?爸爸們不妨從以下3點做起：

1.表達愛意

爸爸可以採用握手、擁抱、擊掌等方式向孩子傳遞愛意，當孩子取得成就或是失敗時，可以採用口頭或書面的表彰、安慰形式，激發其上進心。

與此同時，還要關注孩子的思想、學業、心理等方 面，經常與孩子一起談論問題，交流看法，互換心得，滿足孩子的情趣需求，以此增進感情。

2.共度時光

爸爸可以每天抽出一點時間，陪孩子玩遊戲，看動畫片、球賽，聽音樂等，那些為生計奔走的爸爸也應如此。

孩子的性格和氣質並非一朝一夕可以塑造的，唯有與父親相處的時間達到一定的”量”才有可能，否則孩子記憶的深處是不會有父親的印象的。

3.以身作則

孩子模仿能力很強，所以，作為孩子的第一任老師——家長， 尤其是父親，起著不可忽視的作用。

媽媽的性格影響著孩子未來的心理是否健康，生活是否幸福。

而爸爸的性格和態度，直接決定了孩子未來的人生高度。

父親與孩子間良好關係的建立，直接影響著孩子的性格、世界觀、人生觀的建立。

有些事，必須由爸爸來做，媽媽代替不了。

與天下父母共勉！
