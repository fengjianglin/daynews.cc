大人的一句玩笑話，會讓孩子心裡造成怎樣的恐懼感？

天天要聞
2019-12-22

每次帶我們孩子出去碰到街坊鄰居，總是喜歡和我們U小主開玩笑，逗逗她~「不認識我啦？「、「叫我什麼呀？」「你爺爺去哪裡啦？」「你媽媽呢？」這些話去問孩子，U小主每次都是不吭聲不回答，我知道其實她不是不知道，而是覺得對待一個陌生人需要一點時間，分不清楚你到底是善意的還是有什麼目的性的，所以孩子通常會在這種情況下選擇沉默或者會躲到我身邊~像剛才問的這些還算是對孩子的一種「客氣「~不得不說有些大人玩笑可能開的稍微大了一點，比如會對著孩子說」你叫不叫我，不叫我我就不讓你走了「、」你不理我我就打你媽媽「（故意做那種打我的姿勢）還有的對著孩子直接說」你不叫我，你媽媽就不要你咯，我跟你媽媽走了，你就留在這裡吧「~

當聽到這些言語的時候，對於一個2，3歲的孩子來說，她可能真的會當真，因為她很單純，她不懂得什麼叫開玩笑，而且她也分不清楚為什麼別人會說這些話，是不是真的不叫人媽媽就不要我了呢？

在這個時候，孩子其實心裡是受到一定傷害的，她會恐懼，會害怕，因為她在家裡不聽話的時候，媽媽也會生氣，再加上其他人再加強了這種不聽話的孩子就要受到懲罰的言語更會當真；所以在這個時候，家長的做法真的很重要！

1、 碰到問孩子的街坊鄰居，你再次問孩子「李奶奶你還記得嗎？「如果孩子點點頭，那說明她是知道的，她只是對外人的一種抗拒（表明了我跟你不太熟），那這個時候媽媽可以站出來替孩子回答」孩子認識你的，她只是比較害羞而已，熟了就好了！「這個時候鄰居也會明白，本來就是和孩子開玩笑不會太認真的~

2、碰到有些鄰居故意說些「狠「話來刺激孩子的時候，你要立馬抱起孩子，媽媽很愛你怎麼可能不要你呀！」千萬別讓孩子誤以為這是真的，這樣的做法其實也在暗示說「狠」話的鄰居，你這個玩笑開的過了，適可而止了~

千萬不能順著鄰居的話去看孩子的反應，孩子都很聰明，她會記得誰說過什麼話的，雖然她可能一時沒有辦法去表達什麼，但是她心裡會記得很清楚，如果有些家長會礙於面子，不想得罪鄰居，也會硬讓自己孩子叫對方感覺不叫就是不禮貌，那孩子在這種情況下多數會傷心的哭，她會認為：「媽媽為什麼和其他外人一樣，是不是真的不想要我了呢？」

現在很多的長輩都喜歡逗孩子，覺得逗孩子是一種樂趣，但是需要有一個尺度，因為孩子的心裡都是很脆弱的，她沒有大人想的那麼多面性，她只知道好與壞，如果你不想讓孩子對你有個好印象，那你可以肆無忌憚的說任何話，如果你想在孩子面前留下好的印象，千萬不要為了玩笑、好玩去說一些或做一些「傷害」孩子的話和事！

我是悠悠的小野貓，一名2歲寶寶的媽媽，也是一名上班族，是一個典型的「寶辣媽」，歡迎更多的「寶辣媽」們留言分享育兒經歷呀~
