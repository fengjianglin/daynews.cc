大家都是成年人了，有些話聽聽就好，真心很貴，別見人就給

2019-12-21 2,058條評論 4,747次閱讀 3235人點贊

人這一生，會遇見很多人，也會告別很多人，有的人能相伴一生，有的人卻形同陌路。

常言道：人生，怕走錯路，真心，怕給錯人。

人與人交往是一門學問，不管感情是否長久，真心與否，都希望你可以寵辱不驚，去留無意。

別高估你與任何人的關係

聽過這樣一句話：「一個成年人與人交往最好的態度就是，做好自己，然後親疏隨緣。」

深以為然。

看過這樣一新聞：重慶26歲的徐小姐，跟公司同事楊小姐非常要好，兩人還相約要組隊辭職創業。

可是當她辭職之後，楊小姐卻以手頭有項目為理由一拖再拖，甚至還主動申請五一加班。

被放鴿子的徐小姐十分憤怒：「我覺得她根本就不想辭職，是我太天真，竟然相信了她的話。」

而面對徐小姐的指責，楊小姐只是輕描淡寫地回應：「是她太衝動了……」

網友在心疼徐小姐的遭遇同時，卻也紛紛表示：大家都是成年人了，有些話聽聽就好，千萬別當真；不是每個人都值得你交心，你的真心很貴，別見人就給。

生活中，我們經常會對一段關係過於樂觀，以至於產生一些理所當然的想法：

「我們認識這麼久了，這點小忙他肯定會幫的。」

「我們都結婚三年了，讓他替我照顧一下父母肯定沒問題。」

「我當初對他那麼好，這回我遇到困難他一定義不容辭。」

可是我們往往忘了，感情是最複雜最縹緲的東西，不是你付出真心就一定能換來真心相待。

這世間沒有什麼東西是一成不變的，更沒有那麼多的理所當然，春夏秋冬四季交替，感情會淡，人心會變。

對別人期待太高，反而失望越大。

行走在世間，永遠記得一句話：人情似紙張張薄，世事如棋局局新。

別高估任何一段關係，在這個喧囂浮躁的世界保護好自己的內心，才不會輕易受傷，才是最好的處世之道。

2

別試探人心，別驗證感情

聽過這樣一個故事。

老張和老馬是一對友誼十分深厚的老友。

有一次，老張跟朋友聚會喝酒，席間大家打賭，輸了的人要給自己的好友打電話借錢。

老張賭輸了，他信心滿滿地打給了老馬，說最近手頭緊想借十萬塊錢，而老馬最近孩子剛出生，支出大，便婉轉的拒絕了。

這下老張面子上掛不住了，加上周圍人的集體起鬨，忍不住大罵老馬小氣，不夠義氣。

後來這事傳到老馬耳朵里，得知老張竟拿借錢開玩笑，還如此詆毀自己，也是火冒三丈。

兩個曾經的摯友就此撕破臉皮，老死不相往來。

再好的關係，也經不起試探；再深的感情， 也會有所隱瞞。

要知道，人與人之間的感情就像織毛衣，建立的時候一針一線，拆除的時候卻只需要輕輕一拉。

千萬別輕易去驗證一段感情，這隻會讓親密無間的關係產生間隙，引發猜疑、矛盾，最終兩敗俱傷。

別去試探人心，別把事事追問。

做人，有時就該糊塗一點。放得下，才能沒有煩惱，簡單點，才能活的輕鬆。

3

不依賴，不強求，不失望

最近，當紅小生肖戰在採訪中表示：「自從自己的工作有了起色後，曾經的一些朋友就漸漸疏遠了他。雖然知道這是成長的必經之路，可是想想還是很難過啊。」

是啊，我們總有這樣的經歷，明明曾經無話不談的朋友，可走著走著卻只剩下了沉默……感情這東西經過時間的洗禮，有時歷久彌新，有時卻也隨風飄散。

這世間，誰會是誰的一輩子呢？

不走到最後，誰都不知道答案。

《送你一顆子彈》里寫道：那些與你有關的人，就是與你有關的，逃也逃不掉的，就算你們只見過三次，三年才搭理一次，就算是你們隔著十萬八千里。

有些人註定是你生命里的癌症，而有些人只是一個噴嚏而已。

永遠記住：要走的人留不住，裝睡的人叫不醒，不喜歡你的人感動不了。

「你瞧這些白雲，聚了又散，散了又聚，人生離合，亦復如斯。」

以平等的心態去看待周圍的人，不委屈自己，也不虧欠他人，少一些計較與得失，也少一些遺憾與失望。

不刻意去追求某段關係，也不刻意挽留已經流逝的感情。

4

不虧欠別人，別委屈自己

一直覺得，這世上兩種人最容易受傷：念舊的人最容易受傷，喜歡拿餘生來等一句別來無恙；太重感情的人容易受傷，因為看得太重，所以傷得更深。

所以，永遠不要把真心悉數奉獻，愛人愛七分，留三分給自己。

有些話聽聽就好，不必當真；有些事看破就好，不必點破；有些人看清就好，不必深交。

人山人海，別辜負相遇；歲月如梭，別看輕自己。

願我們在漫長的歲月里，將自己活成自己最強大的依靠。

願我們懂得感恩，保持清醒，保持獨立，不虧欠別人、更不委屈自己。

餘生，做好自己就好，就讓親疏隨緣，愛恨隨意。

人生所求，不過雲淡風輕，歲月靜好。任它世事滄桑，內心始終安然無恙。
