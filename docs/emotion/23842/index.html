一個老人的悔悟哭訴：我終於把我小兒子偏愛成了一個“白眼狼”

2019-12-18 1,356條評論 2,708次閱讀 439人點贊

文‖風舞心緣

每個人都希望自己能有個幸福的晚年，在自己晚年的時候，能夠得到子女的悉心照料。很多人總覺得，生的孩子越多將來的保障就越大，覺得這個靠不住還有那一個；還有的覺得，家裡一定要有男孩子才更有面子，甚至有的家庭是生不到男孩誓不罷休，所以，很多家庭就越生越多。然而，事實證明，孩子孝不孝順跟生了幾個完全沒關係。

在眾多子女中，很多父母都很難做到「一碗水端平」，他們或許是「重男輕女」，或許是別的原因，總是會偏愛其中一個。他們以為他們偏心的子女會更可靠，其實，到了他們晚年才明白，往往被他們偏愛的孩子更容易成「白眼狼」。

看看身邊嚴重重男輕女的家庭里，被偏愛的兒子總是有恃無恐，他會覺得父母所做一切都是理所當然的，一旦父母沒有用處了，只會被嫌棄，很少能突然變得明白事理。

昨天碰到一個好久不見的同學，看著她滿臉憔悴的樣子，我又心疼又驚訝，趕緊拉著她去奶茶店坐會，問她最近怎麼了？發生什麼事了嗎？她過了一會，開口就是憤憤不平，「我真替我媽和我小姨感到不值！」

原來，就在前一個月，同學外公病危，小姨打電話說估計撐不了幾天，同學就請假陪著媽媽趕回去，150斤的外公瘦得不到80斤，完全就是皮包骨，六十齣頭的外婆也滿頭銀絲，看著特別讓人心疼。

本來同學是不想回去的，因為從小目睹了外公外婆嚴重的重男輕女，同學對他們並沒有很深的感情，想到那個啃老上癮的舅舅更是一個頭兩個大。可不管怎樣，外公外婆是給了媽媽生命的人，縱然有千般錯萬般錯，也不能真的斷絕往來。

第二天，外公就不行了，臨走前，只是看著媽媽和小姨流淚，嘴巴動了動卻根本發不出聲，只是眼睛瞪得大大的。所有人都知道外公是想要看舅舅最後一眼，那可是他這輩子掏心掏肺去愛的兒子啊！

同學媽媽就趕緊給舅舅打電話，舅舅回了一句：「還沒死啊，真是夠能拖的，你們都在那呢，還要我去幹啥？又不是沒人收拾。」

同學媽媽哭得一塌糊塗，還是好言相勸：「小弟，你別這麼說啊，爸爸從小就寵著你，節衣縮食給你修房買車，把所有的一切都給了你，到死還惦記著你，你就去見他最後一面，讓他放心走吧！」

舅舅非常不耐煩的回應：「他做的一切又不是我逼的，再說他的病醫生都治不好，我去了有什麼用，這麼大冷天的，我們還要睡覺，你們別再吵了。」

終究外公還是沒能最後再看一眼他那心心念念的小兒子一眼，一直到外公下葬，舅舅一家三口就來匆匆燒了一點紙錢，就走了。

大家都想不通，舅舅可是兩個老人心尖尖上的寶貝啊，到了臨了舅舅的做法卻寒了老人的心，活脫脫一個「白眼狼」，這到底是為什麼呢？其實，歸根到底就是

家長偏心

當初老兩口的「重男輕女」是導致孩子不孝的原因之一。被偏愛的兒子有恃無恐，習慣了父母優先為自己考慮，理所當然的認為父母的一切都是自己的，久而久之就被「慣壞」了。只知道索取，不知道珍惜。

在同學媽媽記憶里，從小家裡有點好吃的，父母藏著掖著都留給兒子，打豬草，做家務，從來都是兩個女兒的責任。

長大後，兩個女兒出嫁連套像樣的嫁衣都沒有，而兒子娶妻生子，修房買車，外公外婆砸鍋賣鐵都支持。

過度溺愛

有的孩子一出生就成了全家的寶貝，含在嘴裡怕化了，捧在手裡怕摔了。無論大事小事，家長們都會竭盡全力為孩子包辦。在這樣無止境的溺愛之下，父母的一切付出都成了「理所當然」，那還有什麼「感恩」、「回報」呢？

在外公外婆的觀念里，女兒遲早要嫁人，生來就是被使喚的，兒子才是延續血脈，必須捧在手心裡。舅舅從小就是要什麼，外公外婆傾盡所有也要給辦到。

外公一輩子都在為舅舅當牛做馬，本意應該是為了養兒防老，沒想到卻是給白眼狼兒子提供了最絕佳的土壤，所有的付出都是自討苦吃，理所應當。

不知感恩

從孩子小的時候開始，家長們只曉得一味地付出、奉獻，卻忘了教會孩子「感恩」。總覺得隨著年紀的增長，這些道理孩子應該自然而然的就懂了。實際上，就是這樣的心理，為自己晚年的「不幸」埋下了禍根，使孩子的「不孝」成為了理所當然。

就在半個月前，同學外婆在送孫子坐校車時不慎跌倒，右腿骨折，需要坐輪椅，舅舅買了一箱餅乾和礦泉水放在外婆卧室，就再也沒去管過。

還是隔壁鄰居實在聞不了外婆屋裡的臭味，打電話給了同學媽媽和小姨。

終歸是自己母親，總不能真的不管吧！同學媽媽和小姨就把外婆接到了同學家裡，兩個女兒就輪流照顧著，可是自打外婆住進同學家裡，不管小姨和媽媽怎麼對待外婆，她老人家始終覺得都是理所應當的，嘴裡念的，心裡想的全是白眼狼舅舅，擔心他在家餓著，凍著。可是，在這期間，那個舅舅沒有探望過一次，連電話都沒有打一個，外婆卻從沒有說他半點不對，還總是叮囑兩個女兒不要和弟弟計較，還說當姐姐的就要照顧著弟弟。

最寒心的是，前兩天舅舅是破天荒的來找外婆了，可是沒說兩句話，舅舅就向外婆要錢，開口就是5萬，外婆哪裡有錢，就向媽媽和小姨要。這一次，同學媽媽和小姨當然是拒絕再做「扶弟魔」，不論外婆怎麼說，都沒有給外婆錢。最後，氣急敗壞的外婆，就把同學媽媽和小姨罵了一頓，還說她們兩個沒良心，堅決要去兒子家住。外婆對兩個女兒的付出視而不見，心裡只想著兒子，讓同學媽媽和小姨的心徹底碎了。就依了外婆，把她送去了舅舅家。

可是，到了舅舅家門口，舅舅一看是把外婆送回來了，卻怎麼也不給開門，說，「老太太都這樣了，送他來這裡幹啥？啥活也幹不了，還得伺候她，伺候老太太就是你們當姐姐的事，等她好了再說吧！」

聽了舅舅的話，外婆也是氣的直哭，她不明白為什麼，她那麼疼愛的小兒子為啥會對她這樣？

舅舅那裡不接待，同學媽媽和小姨又只好把外婆拉回來，但是，再也沒有以前那麼上心了。外婆也是一天唉聲嘆氣的。

可事實上，同學媽媽還是時不時會嘆氣流淚，為父母不值得，更為自己和妹妹不值得。

現在，我們在網上經常會看到，兒女不贍養老人，被老人告上法庭的新聞。我們常常都是譴責不孝的兒女們，可是，從來沒有去剖析造成這樣後果的原因是什麼？所謂的「父母不慈，兒女不孝」，也是這個道理吧！

而那些偏心兒子的父母們，永遠也不會承認這種做法不僅會給女兒帶去殺傷力，十有八九還會害了兒子。

世上沒有哪個父母能做到百分百的公平，而子女在乎的也並不是絕對公平，只要能得到足夠的愛和溫暖就好。

父母疼愛子女和子女孝順父母，應該是長期發酵的過程，不過分強調公平，也沒有特別的偏愛，才能讓「養兒防老」順其自然。如果父母把孩子帶來這個世界，只是因為愛，無關其他，那孩子也一定會在父母老去之時，溫柔而耐心的陪伴。

最後，我們都是要為人子女，為人父母的，願我們，為人父母慈愛子女；為人子女孝順父母；往後餘生，幸福美滿！
