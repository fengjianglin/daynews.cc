我的姥爺姥娘，不是親的，卻比親的還要親

2019-12-14 1,003條評論 3,105次閱讀 2124人點贊

作者：黃緒鋒

又是一年陰曆十月一，秋風蕭瑟，葉黃草衰，我和父母來給姥爺姥娘掃墓。點上香。燒了紙錢，娘又跪在墳前，對著墓碑，娓娓訴說這一年家裡發生的事。碑文是我自己寫的，聊以寄託無限的哀思！

裊裊青煙里，姥爺姥娘的音容笑貌浮現在我的腦海里，越來越清晰……

姥爺姥娘與我們並沒有血緣關係，然而，卻比親的還要親。

姥娘是出身自朱家埠南村的大家閨秀，舊社會男婚女嫁講究門當戶對，所以姥娘嫁進了侯家屯的孫姓大戶。

孫家出了好幾個八路，八路軍山東縱隊5支隊65團，是抗日戰爭期間膠東根據地六大主力部隊之一，團宣傳股長孫濟魯（《從即墨走出去的65團》一文中提到過）就是姥爺的親二叔。在孫濟魯的帶動下，家裡好幾個青年子女都參加了革命工作，其中姥爺於1946年秘密加入了中國共產黨。孫家衚衕是有名的「共匪窩子」，時常受到日本鬼子、國民黨、還鄉團等勢力的打擊迫害。由於青壯年都出去當兵打仗了，家裡只剩下老幼婦孺，在動蕩的年代裡，姥娘和幾個妯娌就成了家裡的主要勞動力。

孝敬公婆，團結妯娌，姥娘任勞任怨地打理著這個大家庭。一家四代人三十多口，做飯、洗衣、做針線活，姥娘從不含糊。而且姥娘特別心靈手巧，給婆婆做了漂亮的繡花鞋，婆婆穿上滿大街的炫耀，直誇有個能幹的兒媳婦。

解放後，姥爺擔任了長直鄉鄉長，新中國剛成立，百廢待興，姥爺一心撲在工作上，常年不在家，家裡的重擔還是壓在姥娘肩上。

我娘出生於1950年，本姓胡，在十個月大的時候，父母雙亡，她8歲的哥哥被本家大娘收養，而襁褓中病弱的丫頭片子，沒人願意要。這時，姥娘聽說後，把可憐的孩子抱回了家。親戚鄰居都不理解，很多人勸說姥娘：「非親非故的，抱來這麼小個閨女，不是蒼蠅飛在牛眼裡—-找累（淚）吃嗎？」但是，姥爺姥娘卻非常痛愛這個孩子，對別人的勸說不理不睬，一把屎一把尿的撫養著，視若掌上明珠，對這個女兒百依百順，甚至，姥爺去大井挑水，還得背著她。

那個年代，老百姓積貧積弱，命如螻蟻，姥娘的大哥大嫂雙雙撒手人寰，撇下兩個女兒，大的5歲，小的3歲。姥娘義無反顧的又收養了兩個侄女。然後，又生育了自己的一兒一女，加上我娘，每天，五個孩子如同一窩嗷嗷待哺的雛燕，兩隻老燕子辛辛苦苦的覓食，也難以填飽五張嘴。

天災人禍的饑荒歲月里，全家人吃野菜、啃樹皮，吃得娘小臉浮腫。姥娘的心像被貓抓一樣難受。家裡值錢的東西都賣光了，最後用一條陪嫁的綢緞被換了一包乾地瓜皮，成為孩子們的美餐。迫不得已，姥娘拖著年僅8歲的舅舅，出門60多里外去乞討度日，卻不讓女兒們出門，怕遭受別人的白眼。

每當回憶起那段艱難困苦的日子，娘的眼淚總是像斷了線的珠子。

苦日子慢慢熬過去了。

我記事的時候，總是在姥娘家生活，我家裡窮，爹娘都得下地掙工分，先後把姐姐和我寄養在姥娘家，我的表弟表妹們也都在姥娘家生活過。姥娘門上的童年，天真快樂，無憂無慮，白天我們幾個去水塘洗澡，去溝里摸魚，去林子里逮知了；晚上就跟姥爺鬧哄，姥爺累了一天，見到我們就高興得像個孩子，姥爺愛喝酒，當時村裡有個地瓜酒廠，酒很便宜，都買了用飯盒裝。姥爺一邊喝一邊用筷子蘸著給我們舔，我們幾個像小狗一樣趴在炕上，回味著地瓜酒的甜美，煤油燈下，姥爺清瘦的臉頰紅撲撲的，笑容是那麼燦爛、溫暖。

這場景不知多少回縈繞在我的夢裡。

我們都特別喜歡在姥娘家住，有一次，爹來接我，我就和表弟跑到地瓜壟溝里趴著，用地瓜蔓蓋在身上藏起來，爹找了半天也沒有找到我，只好作罷。還有一次，兩個表弟偷喝姥爺的地瓜酒，爛醉如泥，姨夫用偏簍把他倆裝上，像賣小豬一樣用自行車帶回家。

後來，舅在城裡買了房子，大約是2萬塊錢，當時對一個農民家庭來說已是天文數字，借了一屁股債。為了還饑荒，姥爺姥娘起早貪黑，種了很多地，種過麻桿，種過棉花；養牛，養豬，養羊，養兔子，養蠶。每天忙得腳打屁股，過度操勞，姥娘的腰早早就彎了，站在那裡像一張弓。

有一個冬天的夜裡，滴水成冰，一頭老母豬居然下了16個崽，姥娘在豬圈裡蹲了整整一晚上，給母豬接生，把小豬抱進屋裡，用破棉襖包上，一是防止凍死，二是防止被老母豬壓死，然後把小豬輪流送回豬圈吃奶。姥娘像給閨女伺候月子一樣呵護著老母豬，好讓它多下奶。16隻小豬終於都養活了，姥娘高興壞了，見人就說：「這頭老母豬真『甜烘』人啊！賣了小豬又能打死一筆饑荒了。」

轉眼間，小豬真的出欄了，錢在姥爺懷裡還沒捂熱，對門鄰居來了，她家四個兒子，學習都很好，老大已經讀了中專，這次，老二又考上了大學，全家人高興之餘，為了不菲的學費犯了愁。姥娘知道鄰居是來借錢的，但是難以開口，因為姥爺等著錢還債呢！姥娘果斷地說：「嬸子，別愁！『出山』（出息的意思）個大學生多不容易，無論如何得讓孩子上學，才賣了豬，錢先拿去用！」

姥娘的善良、熱心是骨子裡與生俱來的，儘管生活困難，卻經常接濟鰥寡孤獨。村裡有個姓馬的青年，無依無靠，整個冬天，穿著露著腳後跟的單鞋、露著胳膊肘的破棉襖，蓬頭垢面，雖然心眼不夠頭，但每到吃飯的時候準時就來了，姥娘從不煩氣，說：「怪可憐人的，還能看著他挨餓？」

還有姓李的、姓林的孤寡人，姥娘都經常接濟。怎麼那麼多窮苦人偏偏讓姥娘碰上了呢？姥娘也許是菩薩轉世吧！

到了上學的年齡，我和表弟們都回自己家了，去姥娘家的次數越來越少，「外甥狗、外甥狗，吃飽了就走」，姥娘念叨著，其實是心裡難以割捨！

在我上高中時的一天，噩耗傳來，姥爺突然去世了！娘撕心裂肺地痛哭，幾次暈厥。「子欲養而親不待」的遺憾折磨著我們兩代人。

娘和姊妹們輪流著贍養起姥娘，希望彌補對姥爺的缺憾。姥娘常說「知足了，知足了。」

姥娘活到95歲，安詳地走了。

在我考上大學時，娘帶著我去姥爺墳前磕頭，告慰他的在天之靈。從此，養成了一種默契的習慣，每到清明節、十月一，娘就會跪到姥爺、姥娘墳前，拉家常一樣訴說家裡一年中的大事……

​
