“你們儘管生，不行我來養”姐姐豪言壯語的背後，其實另有目的

2019-12-20 2,058條評論 4,464次閱讀 3025人點贊

文|秘籍君

自從四年前開放二胎政策以後，很多70後獨生子女家庭就蠢蠢欲動，個個都想打著「為孩子找個伴兒」的借口生二胎，卻沒問過家裡老大是否同意。

要知道，我國《婚姻法》第二十九條規定，在父母喪失撫養能力後，作為兄姐必須擔負起養育未成年姐弟的重任，可想而知，不經大寶同意就生二胎會掀起怎樣的家庭戰爭。

於是，是否生二胎就成了當代社會的熱議話題，眾人對此褒貶不一，相關的新聞下面，時不時就有年齡相差大的姐弟現身說法，立場總是勸別生，只因為將來麻煩太多。

在一片反對聲中，我表妹的存在簡直是一股清流。

只因在小姨提出想要生二胎的時候，表妹已經22歲了，本以為當了這麼多年獨生女的她會強烈反對，沒想到她大手一揮，直接說「你們儘管生，不用問我，支持。」聽到這話後，小姨還以為表妹大學法律沒學好，就掏出《婚姻法》讓她看第二十九條。沒想到表妹看完後，立場依舊不變，還主動說：「到時候你們要是不行了，我來養弟弟！」

這大義凜然的態度終於引起了小姨的逆反，覺得自己也沒有把女兒培養成聖母啊，怎麼家裡要多一個孩子跟她搶父母愛了，她的表現如此奇怪？未免有點過度熱情了。

於是，小姨就偷偷找到我，想讓我去打探下表妹的真實想法，畢竟這件事實在不對頭。

作為一名無間道愛好者，我深知「卧底是沒有好下場的」，果斷找到表妹要求對方坦白從寬，不要逼我動手。沒想到她聽完「噗嗤」一笑，直接交代她的確動機不良，另有目的。

「獨生女太辛苦，多個弟妹好分擔養老義務」

去年我媽和我婆婆同時生病，我和老公忙得腳打後腦勺，到最後實在分身乏術，只好把放寒假的表妹叫過來幫忙值夜。而那幾天我和老公的焦慮和崩潰，就被表妹看在了眼裡，享受了二十多年獨生女生活的她，第一次正面認識到以後會面臨的養老壓力。

自此，「我要有個弟弟妹妹就好了」的想法就在她腦海中出現了，以至於當小姨跟她說想生二胎時，她以為是老天爺顯靈了，聽到了她的心聲。

「不想結婚生子，多個弟弟不用被催婚」

表妹說，她是獨身主義者，看透了男權社會下的婚姻制度，因此不想受困於婚姻，可惜她是獨生女，將來必然要被父母逼迫著踏進婚姻圍牆。

但是，若她有個弟弟或妹妹結果就不同了，小姨精力有限，往後二十年肯定要圍著二胎打轉，自然就沒時間管她。就算催婚，父母也不會太過逼迫，壓力減半，她自然對生二胎喜聞樂見。

聽到這，我這才明白她背後的小心機，然而面對小姨我還是隱瞞了第二條，雖然那才是表妹答應的最主要原因。
