當你放下對自己的苛刻，忽視別人對你的成見，自會一身輕鬆

1、每一縷清晨的陽光都是那麼的美好，連急促的呼吸聲中都蘊帶著大自然散發出獨特淡淡甜甜的味道。早安

2、生活或許總會有一點點小小的遺憾，但是未來的日子依然如此美好，加油朋友們。

3、人的煩惱，往往取決於我們個人的心態，也將成為我們現實生活中唯一必不可少的一種考驗。

4、有的東西，當你靜下心來，慢慢的去觀察，感觸，聆聽，行動，你會發現，其實就在你的身旁，無論走到哪裡，都是最美的地方。

5、走過的路，遇見的人，不管是否對與錯，他們都是我們生命中該出現的人，請珍惜友誼，.青春.愛情.恩情。

6、時光在匆匆，一切在無言中，多少的青春，不要浪費在自己迷茫的歲月中，思緒千萬種，凌亂在風雨中，迷茫的前塵，最後心已冰凍，少年不用功，老來後悔有何用。

7、新的一天；新的開始，不管我們現在的生活過得怎麼樣，我們依然堅持自己的選擇和理由去面對不同的人生，加油！

8、人生的棋局，有時一步走錯步步出錯，滿盤皆輸，一步得意，則步步為營。有時眼見的勝負，轉眼就峰迴路轉，可見棋局和人生一樣變幻莫測。妻子總是越來越少，人生總是越來越短，所謂的舉手無回，也如同人生的不可重來。

9、世間本無事，庸人自擾之。當你放下對自己的苛刻，忽視別人對你的成見，告別無效的社交，自會一身輕鬆。

10、人活一世，每個人都有自己的煩惱和無奈，生命本就是一場漂泊的旅程，走過的每一個地方，遇到的每一處風景，都將會成為人生驛站。
