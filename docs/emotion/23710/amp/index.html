為何比你笨的人混的越來越好，而自己混的越來越差？四字真言說透

在我們身邊，總有一些人被我們看不起，論能力在你之下，論資歷與你也沒有辦法比，可過了幾年之後，一個一個的都比你混得好，特別是那些你最看不起的人，甚至你都懶得和他們有交流，也混的比自己強，還有可能成為自己的頂頭上司，讓自己情何以堪，為什麼會出現這種能力與發展倒掛的現象呢？送給大家四字真言，就能明白其中的道理了。

【1】注。

所謂注，就是專註，注意力集中的意思。那些能力不是太強，也是就我們眼中比較笨的人，他介對於事情的專註程度更深，更能夠長期堅持做某一件事，由於牽扯的面比較窄，所以，往往做的比較精，而這恰恰是一個人獲得成功最關鍵的因素，越是能力平平，越是更穩定的一步步的發展，積累下來，就顯得比你更成功的了。

【2】溢。

所謂溢，就是自滿，總覺得自己了不起，當我們這樣做的時候，而別人則一步一個腳印的按照自己的步調在穩定的往前走，所以，我們沒有人家混的好，往往被這一個溢字所害，由於自己覺得比別人優勢更明顯，總會站到功勞薄上停滯不前，而這正是導致你和人家拉開距離的殺手鐧，如果不能認識到自己的不足，你與別人的這種差距還會擴大，所以，自己混得不好，一定是有理由的。

【3】偏。

越是能力平平的人，越是偏執於某些事，這種偏執被我們看成是一種笑話，往往最後笑話變成了現實，而我們所謂的高明倒成了平庸，所以，在我們內心裡，不要有看不起人這一說，要保持著一顆平常心與身邊的人相處，多向身邊的人身上借鑒人家的優點，而不是用看不起的眼光與那些現在不如自己的人比較，那樣的話，往往讓你錯誤的高估了自己的能力，是自己毀了自己的優勢，而不是發揮優勢。

【4】暗。

越是低調的人，越會暗中使勁，我們在明處做事，人家在暗處努力，一不小心，就跑到了你的前面，這也是導致這種發展不均衡的原因之一，所以，一些人平時不愛說話，並不是人家不會說話，而是有意的隱藏自己，人家能力平平，不是沒有能力，而是進一步的積蓄力量，等待著日後的關鍵時候爆發，不要發自內心的看不起任何人，因為，大家的起點本身就差不多，你的懈怠，就是別人超過自己的最好時機。
