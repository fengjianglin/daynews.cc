同事借3萬還5萬，再一次找我借錢正準備轉賬老闆打來電話：不能借

2019-12-01 0條評論 21次閱讀 0人點贊

摘要

同事借3萬還5萬，再一次找我借錢正準備轉賬老闆打來電話：不能借職場和生活，借錢是常有的事。有些人，不借不合適，借了可能更不好。華子，周敦和我是一個村的，三人關係都不錯。周敦比我們大3歲，是我們3個當中的老大，很有當老闆的派頭。

職場和生活，借錢是常有的事。有些人，不借不合適，借了可能更不好。

華子，周敦和我是一個村的，三人關係都不錯。周敦比我們大3歲，是我們3個當中的老大，很有當老闆的派頭。而華子，還算是挺老實的，可能因為家庭不太好吧，話也不多。

周敦畢業做了1年銷售，就在長輩資助下，開始自己辦公司。因為有資源，有頭腦，不到2年，已經是小有成就。而我和華子，晚2年畢業，直接就到他公司上班了。和忙著找工作的人相比，我們算是很幸福了。可周敦給我們安排的工作，卻讓我不太理解。他讓我們2個都做銷售，他說，以後想要創業，必須先做銷售。我在心裡懷疑，華子這麼老實一人，真的能做嗎？

剛開始，華子做的確實挺艱難，不過好在有周敦的親自指點，以及資源的傾斜下，倒也開了一些單子。而我做的雖不是很出色，但整體還行。做銷售經常需要應酬，有時候也要陪客戶打牌之類，這都是常有的事情。就這樣過了2年，華子的銷售水平確實提升很快，也賺了不少錢，還買了個小公寓，看起來都挺好的。

前幾個月，華子變的特別豪氣，還請我去高檔酒店吃飯。我一看他也沒開大單啊，問他，什麼也不說。可是，有段時間卻總聽見老闆和華子在辦公室爭吵，具體什麼，我也不知道。每次出來後，周敦很是生氣，而華子什麼也不說。

最後一次，老闆直接讓華子走人，而華子也真的生氣就辭職了。就我一個人莫名其妙。問老闆，他也不說，只是讓我少學點壞毛病。過了不久，前同事華子找我借錢，說家裡裝修，缺3萬塊，我想也沒想就借給他了。一個月之後，他還了5萬，多餘的兩萬萬，我肯定不會要。

不到一個月，華子又來找我借錢，說在外面做生意，需要十萬，我剛想借給他，可這時，老闆突然給我打電話問我，是不是借錢給華子了，我說是啊。他說：不準借。
然後說華子現在特別愛打牌，房子都輸沒了，欠了很多錢。以前爭吵就是這個原因，可是華子不聽，他現在還拿著客戶的十萬塊沒有給自己。華子父母現在都在怨周敦，為什麼當初讓華子去做銷售，他現在也很後悔，當時只是礙於華子的面子，才沒有跟我講。聽完，我愣了。

你們身邊有沒有華子這樣的人呢？
