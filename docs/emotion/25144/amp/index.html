老鄭的心事 6.莫名的奇遇

天天要聞
2019-12-22

幾十年了，雖然同在一個縣城，老鄭好像幾乎沒有見過老張。雖然高中同學一起的時候總會有人提起老張。她也多少知道一些老張的情況的，只是好像從沒有在任何場合遇到過，最多也就大街上偶遇過而已，那也只是遠遠地看見！

老鄭不想看到老張，大學四年老鄭先後談過兩個對象，花前月下，也度過了美好的大學時光！可是種種原因畢業後還是各奔東西了！參加工作後就和老頭子走到一起！雖說老頭子人不錯，家境也好。可是老頭子是退伍軍人，文化程度不高，和自己一起總是話不投機，磕磕絆絆就這樣下來了。說過的不好吧有吃有喝的，說好卻也談不上，也沒感受到幸福到哪了！

今天咋就廣場準備跳舞的時候碰見老張了呢？

吃過飯老鄭決定約高中同學老李一起打麻將。

微信約了，老李說人約夠了叫她！

過了會老李回過來微信說：速度！

老鄭多問一句：還有那兩個呢？

老李回：老王和老張！

老鄭回：不好意思，我臨時有事走不開了，你再約別人吧！

老鄭決定玩手機看看抖音，點進去在同城一看，竟然又是老張發的早上跳舞的。 不看抖音了，靜下心來寫點東西吧！

老鄭把自己前幾天寫的一篇回憶母親的文章修改好，發給同事推薦的一個平台編輯。心裡如釋重負，總算完成了自己一個心愿，為辛苦了一生的母親寫了一篇題為《我的母親》的文章，也算是給老母親的一份薄禮。

也許是老鄭的文字功底好的緣故，第二天睜開眼，打開微信就看到了平台發的自己的《母親》！ 奇怪的是平台咋同時發了兩篇同題文章呢？那先看看人家咋寫的！

老鄭準備打開另一篇《母親》，可是她卻首先看到了題目下面的作者名字！：老張。

老鄭心事重重，這到底是咋了呢！
