男子為見網戀女友，連發10個1314紅包，42歲大媽：錢收到，已拉黑

2019-12-19 2,543條評論 3,911次閱讀 1318人點贊

戀愛本來是一件幸福的事兒，但如果被感情騙子利用其單純或善良的話，就會將幸福變成災難。近日大連的王先生結交了一名女網友，因為兩人相談甚歡，女方提出見面之後，一旁的王先生欣喜所狂，就連對方提出發10個1314的紅包也答應了，結果欣然轉賬的結果就是被對方拉黑！等到民警發現時，才知道，原來王先生所謂的女友，已經42歲並且已經結婚！不少網友表示：再也不相信網戀了！

​幸福戀愛變災難

據犯罪人高某交代，起初自己只是為了尋找婚姻中，失去的自信，結果發現王先生很聽話，於是就動了貪戀，陸續讓他轉賬，後來還是怕發現，得到15026元之後就將他拉黑。雖然錢已經被追回，但失去的信任卻再難彌補。

近年來，不少人網戀被騙，有人說，這是願者上鉤，畢竟是你主動上門的，所以，網戀還需謹慎，別成為別人砧板上的魚肉，任人宰割而不自知。還有利用別人感情的騙子，收斂一點，別因為騙人一點錢財而難逃牢獄之災。

​網戀，中年人也難逃魔爪

網戀受騙，不僅發生在涉世未深的孩子身上，中年人，也難逃魔爪。在山東臨沂，50出頭的戴先生，事業有成，兒女雙全，覺得生活無聊，於是通過網戀認識一名「女士」，結果付出了200多萬的代價，在意識被騙之後報警才發現，原來他朝思暮想的姑娘時，竟然是一個年齡比自己還大，禿頭大肚子的老頭。

比這個還誇張的，還有一個單身多年的男子，因為屢次被家人催婚，所以想通過網戀泡妞練手，當他通過微信找人，成功和一位主動添加的女子，聊得熱火朝天，日久生情。女友要什麼就給什麼，就連對方父母離婚打官司，兄弟吸毒進監獄需要保釋金等，都問該男子要錢，等拿了8萬多的時候，居然被人拉黑。後來發現，哪裡是什麼女生，明明是隔壁的摳腳大叔嘛！

​網戀變網騙，你還別不信！

有人貪圖女生年輕貌美，有人貪圖男生帥氣多金，正所謂有需求就有市場，於是乎，頭像一換，年齡一變，大媽也變成美女，大叔也帥氣逼人。在廣西南寧就有這麼一起詐騙案，40多歲的女子馬某，就利用閨蜜的照片，偽造身份證和同村的三名已婚男網戀，並且多次借口父母生病，一共騙取110多萬元，之後吃喝賭博，揮霍殆盡！

​不僅如此，網戀，有時候連了解你的閨蜜，也有一份。在重慶江北上班的小麗報警說自己被男朋友騙了60萬，等警方介入調查才發現，背後的幕後主使竟然是自己的閨蜜。因為知道小麗家庭條件不錯，但性格靦腆，所以在網戀的兩年時間裡，小麗也沒提出見面，甚至被騙了60萬元。

更為可怕的是，網戀還有團伙作案，他們通過專業的術語培訓，專門培養出一批在網上調情的精英，專門利用各大網站的App，以露骨的照片，動聽的聲音，吸引廣大男性。不少人紛紛中招，這就是傳說中的婚姻詐騙團伙。

​我們不排除網戀成功的人，但如果對方以得到你的錢財為目的，對你進行各種誘騙，你還願意「願者上鉤」嗎？
