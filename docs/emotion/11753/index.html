熬過去了，你就懂了，誰都不是生來強大

小和尚說

不是幸福的人就沒有痛苦

只是熬過去了，面帶微笑心裡酸楚

笑的最傻最天真的人，內心是最苦的

挑得起擔子的人，受過的傷最多

誰都不是生來強大

只是熬過去了

假裝笑著面對生活

1、留不住的，就放開這是成熟

有一种放手是

叫最愛的人離開

你還在原地守候

當初你笑的多傻多天真

如今你傷的就有多深刻

當初彼此擁抱有多暖和

如今彼此就有多遠離多遠

天黑路滑，心累有屁用

你流的淚，除了懦弱，

啥用沒有

分手時候的一句告別

記得清晰

你們從此江湖再見，再也不見

你關心是多餘的

你孤獨是自己能力太差

有本事找一個新的

舊的不去，新的不來

2、熬過了苦，才佩擁有幸福

有一些人，你費儘力氣也是徒勞

有一些路，你自己不走，誰也不會陪你走

有一種成熟叫改滾的都滾吧

你最終會明白，拚命追求的一點都不值得

喜歡到死，也不要回頭

捨不得的要命，也不要真的不要臉的去蹭熱乎

有緣會相逢，

有愛會遇見

如果沒有人搭理你

你要自己強大

不只是嘴上說說

吹吹牛B，就忘記了

是付諸行動

你不勇敢，別人只會看不起你

你不愛惜自己，吸引來的只是佔便宜的狼

3、你若牛叉，誰都是你的菜

有一天你不在乎任何人

不是你怕了

是你真的懂了

熬過的夜

淋過的雨

吹過的狂風

都是你強大的墊腳石

受多大罪

享多大福氣

願你熬過所有

苦盡甘來

百尺竿頭

柳暗花明

人生大不多

恭喜發財
