測試解析

選擇A：大眾印象

選擇B：高冷印象

選擇C：溫暖印象

在朋友心裡你是一個很熱心腸的人，不管和什麼人在一起你總能找到話題，不會忽略任何一個人，面對朋友的困難時你從不推脫，所以你的人緣特別的好，在朋友和家人眼中你是一個溫暖的人！

選擇D：天真印象

你的性格很活潑，哪裡只要有你的聲影哪裡就不會太安靜，你是大家的開心果，總能讓大家開心大笑！直率又天真的，有話會直接說出來，從不掩飾你的喜惡感，所以在你的朋友中喜歡你的人特別喜歡你，討厭你的人會說你難以相處！你是一個活得比較自我的人，不要太去在意一些世俗的眼光！
