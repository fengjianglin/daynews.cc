測試解析

選擇A：大眾印象

你在別人心裡的第一印象談不上多好，也談不上多壞，跟別人第一次交往的時候很難給別人留下難以忘懷的影響，平常的你就是不善於言辭的人，在陌生人面前就跟難說幾句話了，這樣下去你的人際交往圈可是不太大，以後學著多改變下自己吧！這樣才能更好的在別人心裡留下好印象！

選擇B：高冷印象

在別人心裡雖然你的顏值比較高，但是伴著冰冷的面孔，不愛說話！顯得很生人勿進，但是你很有分寸，而且頗有頭腦思想，讓周圍的朋友都很佩服你的冷傲矜持。你真是朋友圈裡捧在手心的女神。你越是高冷就越是受大家的喜愛！

選擇C：溫暖印象

在朋友心裡你是一個很熱心腸的人，不管和什麼人在一起你總能找到話題，不會忽略任何一個人，面對朋友的困難時你從不推脫，所以你的人緣特別的好，在朋友和家人眼中你是一個溫暖的人！

選擇D：天真印象

你的性格很活潑，哪裡只要有你的聲影哪裡就不會太安靜，你是大家的開心果，總能讓大家開心大笑！直率又天真的，有話會直接說出來，從不掩飾你的喜惡感，所以在你的朋友中喜歡你的人特別喜歡你，討厭你的人會說你難以相處！你是一個活得比較自我的人，不要太去在意一些世俗的眼光！

註：測試僅供參考，三分天註定，七分靠自己，祝你開心好運每一天！
