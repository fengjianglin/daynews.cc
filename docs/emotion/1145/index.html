當生命已經無法挽回，不如好好告別

小燕母親去世了，我們親近的朋友都沒有接到通知，她悄悄地辦完了母親的喪事。

怕她太傷心，我們也沒有主動提起，只是眼看著她日漸憔悴，忍不住想要開導開導她。

周末我約她帶孩子出來玩，把孩子們送進遊樂場，我們在休息區坐下來閑聊。我盡量繞開她母親的話題，沒想到她主動提起了。

「是我放棄了我媽媽！她本來還可以再多活一些日子的。」

她一開口眼淚就順著臉頰滾滾而下，我想好的安慰話一句都說不出來，只能緊緊握住她的手。

小燕是單親家庭，從小和母親相依為命，結婚後就把媽媽接過來一起住，順便幫她帶帶孩子。

幾個月前母親檢查出胃癌，儘管積極的治療，依然沒有控制住癌細胞的擴散速度。

上次我去醫院探望的時候，她母親已經行動不便，必須卧床休息。

那時候老太太就不想活了，一直央求小燕讓她走，不要花冤枉錢了。母親心疼小燕，很愧疚為了給自己治病，讓本就不富裕的女兒債台高築。

其實小燕也知道母親的病是好不了了，她只是想讓母親多活些日子，多陪陪母親。

可是她實在沒有錢了，在走投無路的時候，她和老公商量賣房。老公讓她好好考慮一下，因為房子本就是貸款買的，就算賣了，也拿不到多少錢。

她猶豫了。結果就在那天晚上母親偷偷的在被子里拔掉了注射器的針頭，永遠地走了。

還沒有來得及好好告別，母親就這樣走了。小燕覺得是自害死了媽媽，這些日子她一直在自責，她覺得是自己太沒有用，才讓媽媽走的那麼委屈。

我不知道說什麼好，我給她講了我舅媽的事。

舅媽是去年年底的時候突然病倒，在ICU住了一個多星期。醫生說醒來的希望渺茫，可表姐不想放棄，各種方式都嘗試過了，還是沒有救回舅媽。

舅媽去世的時候，身上插滿了管子，傷痕纍纍。表姐看著心疼不已，突然覺得自己不應該這樣折騰，應該讓舅媽安靜的離去，少受些痛苦。

其實我也不知道自己到底想說什麼，關於死亡的問題，我們總是刻意迴避。也許，不管生命走到哪一個里程，我們都會覺得，我還想要一些時間，我還有一些遺憾。

可是生命是脆弱的，很多時候我們也是無能為力。我有時候想，如果可以，我希望自己能有機會主動選擇和這個世界告別，以一種溫馨的方式。不要那麼多痛苦和遺憾。
