窮有多麼可怕你知道嗎？在醫院 一個人沒錢給孩子治病 拿頭撞牆

2019-12-18 73條評論 1,917次閱讀 409人點贊

窮有多麼可怕？窮的時候會被人看不起，窮的時候沒尊嚴，窮的時候連命都變的很賤。

本期話題：窮有多麼可怕，你真的知道嗎？

@成志：我爺爺就是因為沒錢看病走的79歲，那時真沒錢，爺爺的病有錢就不用死，有錢就可以不用受罪，記的爺爺清醒時說去大醫院，我們的心像刀割一樣。現在我爸又有病了和爺爺一樣的病，可我現在比哪時有錢了，三年住了30回醫院，我就感覺這樣能心安一點，我親愛的的爺爺你在天堂希望沒有病痛。想你了爺爺

@鵬城那些事：12年一個人在福建工地打工，家裡出了變故。不小心把食指手指頭拉了一個口子，骨頭都露著。為了幹活偷偷的自己用白酒洗了洗，然後用創可貼包起來。第二天照常幹活，最後由於天熱化膿。手指頭彎曲都困難，強忍著疼，用玻璃刀把腐爛的肉和膿水全部一刀一刀的劃掉。現在想，人一旦窮到低，剩下的就是賤了。連身體都賤的可以不用藥都不知道疼了。

@HanJun俊 ：10年時候，剛建了房子，結婚有小孩兒，欠了不少的外債，在廣東打工，小腿迎面骨摔倒劃傷露著白色骨頭，去小診所人家幫忙包紮一下讓去醫院，醫院讓住院半個月，問了下大概要花一萬多塊錢，想了想孩子還吃奶粉呢，花錢還誤工，就自己買了碘酒每天擦拭換紗布吃消炎藥，過了五六天還是發炎了，就自己拿美工刀把發炎的一圈肉割下來，折騰了一個月也就慢慢好了，期間沒有休息過滿勤。

@微笑jwl：一個孩子，八九年前，那時候我還是一個路橋公司上班，我是焊工。那個孩子剛來的時候我們發工作服，他好久都捨不得穿，開始以為他嫌棄工作服，後來才知道是捨不得。他是我其中一個徒弟，也是我教的最認真的，他家是真的窮，窮到什麼地步我不知道，我知道他跟我說過，在家一年都不一定能吃一次肉，我教會了他我所有技術，我離開的時候，用了一次我自己的權利，把他從小工，變成大工。哎，想想他說的那些事，真的心酸。

@戀愛先生：在醫院的時候窮。人的命真的會低賤一些。我剛畢業的時候，在腫瘤科實習。有一天，遇到一對中年夫婦。丈夫陪著來檢查，結果查出妻子患有宮頸癌。男人一下子就哭了，妻子反倒在輕生安慰他。他們說沒錢做化療，更沒錢做手術，丈夫打算帶著妻子黯然離開。那時候，我還年輕，以為是這個男人捨不得花錢，大聲地指責他。丈夫哭著說：丫頭，我們窮，命賤，爛命一條，。說實話，後來日子裡我打心底看不起這個狠心的男人。
