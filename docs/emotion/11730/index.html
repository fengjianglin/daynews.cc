25歲小伙道出中國式悲劇：父母無為，望子成龍

2019-12-11 833條評論 2,055次閱讀 290人點贊

大家好，這裡是《婚姻百態》

「看婚姻百態，品百味人生」

我今年25年，是四川大學的一名在讀研究生，我患有嚴重的抑鬱症。

現在，我每周都要到華西醫院去接受心理治療，並在藥物治療的控制下，能保持還算正常的狀態。

之前有段時間，我會整晚整晚的不睡覺，精神壓力過大，感覺對不起父母，對不起自己，對不起這個世界。一度很想死，很想很想。我有次爬到寢室的陽台，看著腳下發神，腦子裡想的全是死了之後我父母應該怎麼辦。後來，被室友發現才給拽下來。

我父母是很普通的小鎮市民，父親是公務員，母親是教師。他們都屬於各自體系里的底層，干最多的活，領緊巴的工資。

他們最愛對我說一句話：「兒子，你要多讀書。將來可不能像爸媽這樣，受人欺負。」

他們對我的管教極其嚴格，我很少有自己外出遊玩的時間，基本都被關在家裡好好學習。望子成龍，這是他們掛在嘴邊的四個字。而他們自己，卻要時不時喝酒打牌來抒發自己平日工作時愁苦的心情。

好在我有個還算聰慧的頭腦，在他們督促下學習不可謂不用功，最後讓我以一個十分優異的成績，考進了四川大學。那一刻，我以為我圓了他們的夢，可他們腦子裡卻只有比較。「隔壁李科長的孩子去了復旦。」「哎呦，辦公室那個教英語的王老師，人家孩子去的哪？清華！」

那個瞬間，十二年的寒窗成了一個笑話。

大四，我早早在實習期就物色好了一個不錯的事業單位，順利考入。我以為我的父母會很高興，興匆匆地給父母打電話。父母很不高興：「現在工作，人均研究生起步，一個本科生進事業單位能有多大優勢？」「兒子啊，能讀書就多讀，不要以為一個四川大學的本科文憑就很好了，真拿到外面和那些研究生比，可差遠了。」

我是帶著十分沮喪的心情入職的。新人入職，工作不大適應，常常需要加班很晚。每每回到員工宿舍，接到母親電話，都是勸說我不要滿足眼前，要放遠未來。

我實在架不住他們的勸說，還是報備了當年的研究生考生。我一邊工作，一邊備考，壓力頗大。考試前3個月，我辭去工作，專心備考。那段時間，我開始失眠，一閉上眼就滿腦子都是父母望子成龍的說教。後來，我養成了個習慣，睡不著，就抱著備考書，進行全文背誦練習。每每要精疲力盡才能入睡。

就這樣，我考進上了四川大學的研究生，我的身心也因此毀了。

如今，我最大的感受是：父母望子成龍的觀點，本來是為了孩子著想。可望子成龍不能只是嘴上說說，是要切身從孩子實際情況來考慮的，不能給予超出他們實際能力的期待。那樣的期待不會成為一種祝福，只會成為一種摧毀他們心靈的壓力。

要知道，一個人是多樣的，但龍卻只有一種。
