黃執中說的真理：最讓孩子害怕的是這種媽媽

天天要聞
2019-12-21

我之所以愛看《奇葩說》，是因為時常能從其中聽到一些振聾發聵的觀點。

這次《奇葩說》里黃執中講了一個觀點：「犧牲感，是親密關係中的大忌」。這點我非常認同，何止是親密關係，「犧牲感」更是親子關係中的大忌。

執中舉了一個例子，說有個媽媽，一輩子都吃魚頭，把魚肚子肉留給兒女。兒子小的時候問：媽媽，你怎麼老是吃魚頭呢？

媽媽就笑笑說：傻孩子，媽媽就愛吃魚頭。於是孩子就心安理得地吃魚肚，每次還吃魚，特地把魚頭留給媽媽。

結果母親臨終前，拉著兒子的手說：孩子，媽媽從來都不喜歡吃魚頭。然後那個兒子就懵了。

這個故事以前是拿來歌頌母愛真偉大的，但是細思恐極，這個兒子好倒霉啊，當媽的跟潛伏四十年的特務一樣，一切地隱忍都為了臨終前那一下暴擊：驚不驚喜？意不意外？最重要的是，內疚不內疚？

不知道為什麼中國人特別喜歡「犧牲」兩個字，文死諫，武死戰，君王死社稷。這個死字就是犧牲的意思。做將軍的不能馬革裹屍，做大臣不能直言上諫，啪啪啪打皇帝的臉，就算不得名臣。

這種不死不開心的文化延續到現在，犧牲仍然是一個好詞。很多人都喜歡拿來用修飾自己，甚至陶醉於其中。

很多媽媽都像那個「魚頭媽媽」一樣，愛玩「默默奉獻」的把戲。這個把戲的精髓在於默默，就是我做這件事的時候，事先不告訴你；我也不管這件事，是不是你希望我做的，總之我做了，我付出了，我犧牲了，我好偉大。

更可怕的是，這種犧牲感是累加的，就像黃執中所說，當一個人有犧牲感的時候，她就築起了道德大壩，然後她所做的一切奉獻都是在大壩後面不斷地加籌碼，總有一天，這個大壩會垮，那時候你就要面對她所有的指控，不得不對她產生愧疚感。可那時候沒人想起來說：那是你自願做的，我又沒逼你。

如果一個媽媽愛玩「犧牲感」的把戲，孩子會很痛苦。

有的媽媽在孩子出生後不久就放棄了不錯的工作，在家全職，把全副精力投放到孩子身上。孩子長大後成績不理想，媽媽說：我為了你，不出去工作，全部心血花在你身上，你怎麼考成這樣？

孩子就很內疚，媽媽為我從早忙到晚，我還考不好，我太糟糕了。

事實是，即便媽媽真的是為了孩子選擇全職，可是事先並沒有徵求過小孩的同意，是她自己放棄了對自身價值的追求，而把全部希望寄託到孩子身上，以犧牲感來要挾孩子獲得好成績，這是一種情感控制和情感勒索。

《小歡喜》里陶虹飾演的宋倩，原來是一個中學老師，因為女兒喬英子即將高考，主動辭職一心一意在家照顧女兒。為了不讓女兒吃油條豆花，不吃火鍋，不碰各種垃圾食品，每天很早就起來做早飯，還燉各種滋補的葯湯，完全無視女兒嫌棄的小眼神。

宋倩看到女兒長青春痘，立即去買了很貴的燕窩，花幾個小時燉成一小盅，自己一口捨不得吃，讓女兒吃，可女兒並不愛吃，還拿去給媽媽最討厭的小夢阿姨吃，那是宋倩前夫的未婚妻。

宋倩逼著女兒放棄自己的夢想，毀了她的航空模型。一定要英子考清華北大，女兒考了年級第二，她都不滿意，甚至偷偷地改了女兒的志願，一直逼得英子陷入抑鬱，要跳海。

宋倩憑什麼有那個底氣，如此干涉女兒的生活和志願呢？就是「犧牲感」啊！你看，我為了你放棄編製，在家照顧你，變著法子給你補營養，我為你犧牲這麼多，要你考年紀第一怎麼了？要你考北大清華怎麼了？

我犧牲了這麼多！

可是想一想，這種犧牲是英子所要的嗎？宋倩的付出成了壓在英子身上的巨石，讓她逐漸喘不過氣。

她不是看不到媽媽的犧牲，可恰恰是因為看到，才讓她沒辦法無視媽媽的心愿去做自己，她會愧疚。媽媽的犧牲太沉重了，這是導致英子陷入抑鬱的元兇。

如果英子可以選，她一定更想要一個好好工作，有自己的事業，沒時間煮飯，可以叫叫外賣，偶爾有空能陪自己搭搭航模的媽媽。

熱衷於玩「犧牲感」的媽媽，從本質上來說，其實是自私的媽媽。她的付出最終目的是為了控制孩子。我付出了那麼多，你竟然不聽話，你應該內疚。這是一種隱蔽的情感操控手段。

來設想這樣一個場景：

媽媽：我要拿走你的自由。

孩子：憑什麼？

媽媽：因為我做了這個、這個還有這個。

孩子：我沒讓你做啊！

媽媽：不夠的話我還可以做那個、那個和那個，我做了這麼多，你還不滿足嗎？你還是個人嗎？總之你要聽我的。

沒見過被強買強賣的人會開心。被動處在這種模式的孩子也不會快樂，甚至很可能扭曲。

昨天爆出一個上海有個17歲女兒家暴父親的新聞。說是女兒想買一雙800元的鞋，父親沒答應，女兒就打他，把他衣服都剪爛了。

父母家暴子女時有耳聞，這麼硬核的子女還第一次聽說。乍一聽這小姑娘簡直不是個人。

警察一到，父親就開始傾訴：送女兒去美國讀書，去了一年就不去。為了讓女兒放鬆心情，給她報了吉他課，七八百元一節課，她也常常不去。

而女兒是這樣說的，拒絕買鞋只是導火索。實際上她覺得父母一直在控制她的生活，一切都是他們說了算，自己覺得很挫敗。而且父母經常對自己挑三揀四，自己早上做早飯，多了父母嫌多，少了又說不夠吃。

這又是一個典型的：我為你犧牲那麼多，對你提點要求怎麼了？控制你又怎麼了？的邏輯。這個小姑娘為啥那麼暴躁呢？因為內疚感到一定程度，會變成憤怒感。

升米恩、斗米仇大家都聽說過，當這個恩無論如何都沒法報得完，欠債的一方就會由感恩轉為憤怒。

既恨自己無能，不能報恩，也恨對方為什麼要付出那麼多。

所以媽媽們不要再沉溺於「犧牲感」中無法自拔，當你付出的太多，孩子無論如何還不清時，會把他推到另一個極端。他可能會怕你，甚至根本不想見你。

拋棄「犧牲感」的正確姿勢

其實不做「討人嫌」的媽媽也很簡單。就想黃執中所說，你要做一個快樂的媽媽，優先照顧到自己的需求，讓自己覺得快樂，再讓那種快樂和幸福「溢出來」感染到自己的孩子。

他舉了一個例子：媽媽很喜歡吃冰激凌，只買了一個，告訴孩子，媽媽很喜歡吃哦，然後讓孩子少嘗一點，自己吃多一點。

當時彈幕中很多人說這不可能。不好意思我家就是這樣。我愛吃楊梅，女兒愛吃草莓。買楊梅的時候，我吃大半，給她吃小半，買草莓的時候她多吃一點，我少吃一點。

我和她都很開心。她有時候給我多留一點，我也會誠心誠意地跟她說謝謝，然後開心地吃掉。

想一想，如果我明明自己很愛吃楊梅，卻忍住不吃，都留給女兒，然後心生怨念，在她寫作業的時候說：媽媽最愛吃的楊梅都給你吃了，你不好好寫作業就是對不起我！

女兒會比較開心嗎？

她可能會在心裡翻個白眼：我又沒不讓你吃！

我可以要求她好好寫作業，但不是用我為她做的犧牲去要挾她。

「母親」不是一個苦大仇深的名詞，我們可以開心一點，輕鬆一點。不必凡事都以孩子的需求為優先。

只有一隻雞腿，媽媽很想吃，那就把它吃掉。如果看著孩子吃比自己吃還要開心，那就留給孩子吃。唯獨不要強忍著不吃雞腿，然後覺得自己真偉大，為孩子付出了那麼多。

在買包和給孩子報班之間猶豫，如果覺得買包會比較開心，那就買包。如果覺得看到孩子用功上進比較開心，那就去報班，但是不要報了班以後跟孩子說：你必須好好讀，媽媽為你放棄了心愛的包包。

最後，把一段5歲小男孩的話送給所有的媽媽們：

你做我媽媽的時間很長，

你一輩子都是我的媽媽，

所以愛我的時候，

別忘了愛自己。

我愛你！
