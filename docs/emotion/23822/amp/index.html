大徐中午做了蛋包飯，成本不到10塊錢，和閨女娘倆吃的一點不剩

天天要聞
2019-12-18

生活不易，必須熱愛！

大徐今天有點渺茫，從心底冒出絲絲涼氣，非常的焦慮。說到焦慮，它來自於生活的各個方面吧，有時候一件棘手的工作，別人的幾句不太好聽的所謂「忠言」，家人對你抱有的「過度」期望，等等很多事情，假如在情緒上處理不好，都會容易造成焦慮。大徐的焦慮來自於想要更好的安身立命吧，怕自己會在安逸的生活里一事無成失去了自我。雖然有焦慮，但大徐也會自我調節，列出焦慮的原因，然後一條條去分析去找解決的辦法，辦法總比困難多，很多事情勇敢的面對了，它便不再是你害怕的問題了。

人活著，沒有誰是更好過的，呵呵一笑，抬頭看看天空，那染滿蔚藍的遼闊和壯美足以讓你忘卻煩惱！朋友們，不開心的日子裡不妨到戶外走走，也一定要記的好好吃飯才是！

今天中午閨女不想在學校里吃食堂，一心想回家吃，剛好我也能拿出時間，就把閨女從學校接回來，我們娘倆一起聯手做了蛋包飯，更確切的說是蛋蓋飯吧，食材簡單，做法家常，我使勁往高處算成本總共不到10塊錢，娘倆吃的飽飽的，很滿足。閨女還動手幫忙用番茄醬做了造型，說蛋包飯是一隻小烏龜。

食材用到了：胡蘿蔔丁、黃瓜丁、火腿丁、米飯、雞蛋液。米飯是早上上班前電壓力鍋預約好的。

鍋里倒入少許油，油熱加入胡蘿蔔丁翻炒幾下。

緊接著加入切好的香腸翻炒幾下即可。

接著倒進去米飯翻炒，翻炒時要盡量讓米粒攤開炒勻。

翻炒兩分鐘後，最後倒入黃瓜丁繼續翻炒一兩分鐘，撒上適量食鹽即可出鍋盛盤。

炒飯做好了。下面開始煎蛋餅，一個雞蛋打散，鍋里滴幾滴油就可以，把雞蛋液倒入鍋中晃動鍋子均勻攤平，小火慢慢烘蛋，煎蛋嫩一些才好吃。可以把蛋餅翻個個，雙面煎好備用。

炒飯裝入盤中，把煎好的蛋餅蓋上凹個造型就可以美美的吃了！

我和閨女一人一盤，閨女狼吞虎咽的吃完自己盤裡的還借了我盤子的糧，看著她吃的香大徐心裡美美的。

大徐最後一啰嗦：蛋包飯做好後可以上面撒點黑白芝麻，既好看又好吃。做米飯一定要做的硬一些，炒的時候勁道好吃，米飯可以是現做的，剩飯也是可以的，就是炒的時候時間久一點。蛋包飯里有肉有蔬菜還有蛋，非常有營養還簡單好做，是比較快手的飯。朋友們無論生活里遇到什麼都要記得好好吃飯呀！

大徐是一枚吃貨，也是一位喜歡在廚房折騰的煮婦，我一直覺得沒有美食的生活是沒有靈魂的，好慶幸我這個人喜歡吃美食也喜歡嘗試烹飪美食！期待朋友們的關注，一起共享美食吧！
