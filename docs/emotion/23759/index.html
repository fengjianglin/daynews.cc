人生似茶，平淡是本色，苦澀是歷程，清香是饋贈，早安

2019-12-19 627條評論 1,961次閱讀 928人點贊

1、看透是一種領悟；看透是一种放下；看透，讓人清醒；看透，讓心無憂。看透財富，追逐就會適度。看透名利，就少了明爭暗鬥。看透離別，就不傷心淚流。看透感情，就不苦苦哀求。看透生死，何懼凄苦暗流？

2、什麼時候，你都要明白，你是活給自己看的，認清最真實的自己，找到屬於自己的位置，然後昂首闊步走自己的路， 你會發現，做自己的這種快樂，遠勝過一切。

3、生活，是一幅豐富多彩的畫。無論是絢麗多姿，還是素色淡雅，都要靠我們自己去描畫。生活，是一首跌落起伏的歌，每一天都有歡聲笑語的飛揚，歌唱多彩的人生，歌唱幸福的生活，歌唱美好的明天！

4、人生似茶，平淡是它的本色，苦澀是它的歷程，清香是它的饋贈。生活就是一個過程，把每一天醒來，當作一種歸零，要以一種獨一無二的姿態，活出自己與眾不同的精彩。早安！

5、生命中沒有永遠不變的風景，只要心永遠朝著陽光，你就會發現，每一束陽光都閃著希望的光芒。只要心中有風景，何處不是花香滿徑。

6、請你，擔負起對自己的責任來，不但是活著就算了，更要活得熱烈而起勁，不要懦弱，更不要別人太多的指引。每一天，活得踏實，將份內的事做好，做得盡自己能力之內的完美，就無愧於天地。

7、生活真諦：一份真心誠意，二分天真純潔，三分健康快樂，四分寬容忍讓，五分和諧鄰里，六分執著堅守，七分孝順父母，八分樂觀激情，九分減壓放鬆，十分愛人愛己。願你擁有生活真諦，享受生活幸福美麗!

8、每一個清晨，都是一個美好的開始，都意味著前進的腳步，不管昨天怎麼樣，今天你只做最好的自己，帶著微笑，帶著善良，帶著夢想起航，你的努力里藏著的是美好生活的明天！

9、每一個清晨，伴著陽光上路，清新的空氣純凈著靈魂，滿天的雲霞變幻著色彩。給自己一個微笑，告訴自己今天會更美好。

10、天會大亮，是因為經過漆黑；春之溫暖，是因為冬的苦寒；相信現在你，不會被疲倦擊倒。希望有一天，你能站在最耀眼的地方，呼吸最新鮮的空氣，活成自己曾經最希望的模樣。早安！更多的精彩情感，哲理，美文；敬請關注微信公眾號：心成一片
