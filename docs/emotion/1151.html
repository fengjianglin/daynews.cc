「我知道他好奇璀璨的宇宙，也知道今後我只會環繞他這一顆星球”，或許這就是父母對孩子的愛吧！！！

「人生，是一場不停輾轉的旅程，家是最好的依靠。”我認為這句話很好的說明了家的重要性，就算是天塌了也有家給你頂著！！！

  • 哪裡有寒冬,哪裡就會有人燃起灶火。
  • 家在哪裡,胃最知道。
  • 味至濃時是家鄉 。

  • 世界很美，而你正好有空。
  • 因為世界很圓，所以終會遇見。
  • 因為看慣離別，所以我們相信遇見。
  • 美好的事不就是在對的時間、對的地點，我剛好遇見你！！！

  • 喜歡這種東西，捂住嘴巴，也會從眼睛裡跑出來。
  • 愛，是陪我們行走一生的行李。
  • 最懂你的人，是陪你一起做夢的人。

我說過最煽情的話是：我們間的距離，最近的時候，是眼裡有你。最遠的時候，是心裡有你。

雖然我不在你身邊，但你一直在我心裏面。

原來愛情是：我正要表白，而你也剛好「正在輸入」。

所有說出來的，未必都能被理解。用力生活的女孩都值得，用理想的方式告別今天。加油，努力向更好的明天邁進吧。

  • 只要步履不停,我們總會遇見。
  • 重要的不是什麼都擁有,而是你想要的恰好在身邊。
  • 不可能永遠收穫所有好評，你要做的，就是做好你自己。
  • 你未必出類拔萃，但肯定與眾不同。畢竟每個人都是不同的，都有自己獨一無二的魅力！！！

童話里的故事都是騙人的，而現實中的麻煩不僅是免費的還包郵！！！

卸下你心裡的圍牆，你會發現生活原本的模樣！！！

就算這力量再微弱,也想要成為自己，成為你的光。

浪漫，就是浪費時間慢慢吃飯，浪費時間慢慢喝茶，浪費時間慢慢走，浪費時間慢慢變老。

因為你要做一朵花，才會覺得春天離開你；如果你是春天，就沒有離開，就永遠有花。

  • 即使身無長物，但也有巨大的生活熱情。
  • 一個人知道自己為什麼而活，就可以忍受任何一種生活。
  • 不管前方的路有多苦，只要走的方向正確，不管多麼崎嶇不平，都比站在原地更接近幸福。

長大，可能很慢。實現長大的心愿，可以很快。

沒有清醒的頭腦，再快的腳步也會走歪;沒有謹慎的步伐，再平的道路也會跌倒。

運氣不可能持續一輩子，能幫助你持續一輩子的東西只有你個人的能力。
