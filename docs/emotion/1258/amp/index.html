“你急啥，跟催命一樣。”媽媽：我真是難上加難

天天要聞
2019-12-06

摘要

當我們還是小孩時，討厭父母對我們嘮叨，等我們成了父母時，也開始對著孩子嘮叨。可不管我們怎麼嘮叨，怎麼催促，孩子們仍然無動於衷。前幾天，我朋友就在跟我瘋狂吐槽他的娃：早上起不來愛賴床，怎麼叫都叫不醒，苦口婆心說一大堆後，祖宗起床了，之後呢，慢慢悠悠穿衣服，磨磨蹭蹭洗臉刷牙。

當我們還是小孩時，討厭父母對我們嘮叨，等我們成了父母時，也開始對著孩子嘮叨。可不管我們怎麼嘮叨，怎麼催促，孩子們仍然無動於衷。

前幾天，我朋友就在跟我瘋狂吐槽他的娃：

早上起不來愛賴床，怎麼叫都叫不醒，苦口婆心說一大堆後，祖宗起床了，之後呢，慢慢悠悠穿衣服，磨磨蹭蹭洗臉刷牙。

晚上回家寫作業也是不慌不忙，先玩夠了才開始寫，有時候作業一多，就得熬夜。好不容易作業寫完了，上床後又開始鬧騰，死活就是不睡覺。我朋友忍無可忍催促了一句，孩子竟然回：「你急啥，跟催命一樣。」

這下可把我朋友氣壞了，口口聲說他心累。

其實，這些都是孩子拖拉的表現，從專業角度看孩子這種行為被稱為「拖延症」。時下最常見的病，估計很多孩子身上都有，很多家長也對此手足無措。其實，只要找對辦法，家長就能輕鬆解決孩子的拖延症。

首先，家長要找到問題的關鍵，要知道孩子們拖延的原因是什麼？是因為沒有時間觀念？還是遇到困惑？又或者是自制力太差，容易受到外界因素的影響？找准原因後，我們才能對症下藥。

1. 培養時間觀念

兒子上學時候總是拖拖拉拉，早上起床都得提前半個小時叫他，賴床、穿衣服、洗漱能磨蹭很久，時間就在他這樣磨嘰中過去了，我和妻子很是頭疼，我們做的最多的就是督促，然而這並沒有什麼用。

這樣的事情總是經常發生，我和妻子開始找尋原因，最後發現：孩子還太小，心智還未成熟，他們不明白什麼是時間觀念，總是想幹什麼就幹什麼。這就需要我們去培養他們對時間的感知能力。

於是我和妻子就開始培他的時間觀念，比如帶他出去旅遊會告訴他火車一般都是定點發車，若是遲到就趕不上火車，沒有辦法玩耍。給他看動畫片會告訴他這個節目也是每天準時播出，錯過這個點就沒有了，就這樣，慢慢的開始培養他的時間觀念，讓孩子知道準時的重要性。

如今，孩子大一些了，時間觀念加強了很多，每天早上他定了鬧鐘，鬧鐘一響就會馬上起來，準點出門，按時上下學，這種作息表已經在他心理紮根了。

作為家長，需要從生活中的小事情來培養孩子的時間觀念，給孩子制定作息計劃，用行動代替語言，和孩子一起執行作息時間。

2. 遲到挨批評刺激法

記得兒子小學要求八點到校，在他的拖延症下，遲到可謂是家常便飯，有一次早上他遲到了，晚上回來後我和妻子拒絕了他看電視的要求，兒子便嚷嚷著明天要早點去學校，在學校被老師罵，回來還看不了電視。

這讓我突然覺得這種方法可行，所謂吃一塹長一智，讓孩子多遲到幾回，多挨幾回批評，讓孩子自己承擔「後果」，這能讓孩子更加懂得準時的和重要性。

3.多鼓勵，少催促

孩子她每天都賴床，你可以對孩子說：「你今天比昨天早起5分鐘哦」。同理，可以用到生活的各個地方，「你今天比昨天做作業快了」「你今天上學比昨天早了」……通過這樣的鼓勵，能讓孩子意識到不拖延的好處，激發他們的動力。

4. 制定目標

在教育孩子時，一定要教孩子制定目標。無論做什麼是都要有目標，根據自身情況而定，不能過於好高騖遠，要容易執行。

有了目標，就要嚴格執行，這點可以和第一點結合使用，但一定情況下也可以靈活改進，比如作業多的時候，就多安排點時間來寫作業。

5. 給孩子自由

說了這麼多方法好像都是條條框框，限制了孩子的自由。其實不是的，在孩子還小的時候，我們需要讓孩子養成良好的習慣，當他達到既定的目標時，就可以允許孩子有一定的自由時間了，做些自己喜歡做的事情。

有這樣的慣例，孩子肯定會抓緊一切時間完成他的目標，然後做自己的事情。

總之，時間是非常寶貴的，家長要培養孩子的時間觀念，而在這個過程中，也要給孩子一定的自由、一定的時間。不要一味的打罵催促，掌握了正確的方法，孩子就不會再有拖延症。
