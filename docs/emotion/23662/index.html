從古至今，三件事決定了一個人成功與否

2019-12-19 462條評論 1,885次閱讀 476人點贊

作者:HI木易子楊

人生在世，無論處於哪個階段，都離不開說話、辦事、做人。

要想立足社會，想在這個社會吃得開，就要掌握三種本領：會說話、會辦事、會做人。

有人說會做人是首要前提，其實，會說話才是。不會說話的人，何談去做人做事。一開口就惹人肯定很難成人成事。

一個人只要會說話，辦事可以辦得圓滿，做人可以順風順水。

會辦事是會做人的前提，想要得到別人的信任和認可，就得會辦事。

以上三個方向，也是生存的三大技巧，一個都不能忽略。誰能用好這三個能力，他一定也就掌握了人生成功的方法，這一生一定過得無比精彩。

一、會說話不僅僅是技術，更是藝術，也是做人的智慧

今天，一個人成功的絕大部分已經不是學歷和專業了，而是踏入社會以後會不會說話。從懂事開始，到求學，交朋友，談婚論嫁，求職升遷，無一例外都會用到這個本事。

會說話，開口就惹人喜歡，小事上能討人歡喜，贏得好感。如果說到大處，能在關鍵時候保全自己。會說話的人，朋友多，敵人少，成功多，失敗少。

說話好不好，也有三條準則：說的好不好，說的精不精，說的巧不巧。

很多人即使天天說話，但不一定就能把話說好。就像很多人每天都在工作辦事，不一定就能把工作辦好一樣。

怎麼就是說得好？

標準就是——把話說的對方心坎上，言者無意聽者有心。最好能彼此共鳴。

怎麼就是說的精？

標準就是——言簡意賅，不侃大山，不說廢話。

怎麼就是說的巧？

標準就是——話要說到點子上。

二、會說話是一種能力，會辦事更是一種能力

如果說話只是第一步，那麼做事就是進階。說話只是對表達的考量，辦事是對更高能力的要求。

人活著每一天都是在解決各種問題和矛盾，心態和方法不同，解決問題的方向就不同。

世上沒有辦不成的事，只有不會辦事的人。

一個人會辦事，真的，感覺人生到達了高峰。駕馭任何人輕車熟路。一個人如果掌握了會辦事的能力，就能把不可能變成可能，最終達到自己的目標。

會說話只是動腦子，動嘴皮子，會辦事可是要有所行為，自己要做一些事情。這就不是說幾句話就能解決的。

凡是不會辦事的人，要麼愛說大話，要麼虎頭蛇尾，要麼生性固執己見，不會變通。

會辦事不是天生的，即使性格是天生的，做事也不是，都是後天養成的。

一件事情辦的好不好，一份事業乾的成不成，並不是衡量這個人講了多少好聽的話，看他的熱情高不高，而是看他在那段時間用什麼方法，做成了什麼事情。

會辦事也有三個衡量維度——辦事辦的到，辦的牢，辦的穩妥好。

什麼是辦的到？

衡量標準就是——說出來就做到了。

什麼是辦的牢？

衡量標準就是——做事之後，領導和同事放心。

什麼是辦的穩妥好？

衡量標準就是——除了領導交代的，其餘的細節都面面俱到，經得住檢驗。

會辦事是獲取財富的第二階段，凡是成功的人，即使不會說話，但是在做事情上一定是面面俱到，把所有跟自己成功有關係的人的利益擺的絕對平衡。

三、會做人是成功的修仙之道

絕大多數的人，活了一輩子都沒想明白如何做人。做人是一輩子的事情，早一點頓悟，就早一點得道。

《紅樓夢》中有句話：「世事洞明皆學問，人情練達即文章。」

人的價值初次看，在於說話，短暫看在於做事，長遠看在於做人。

做人成功，一生成功，做什麼都成功。得人者昌。會做人，人財兩旺是水到渠成的事情。

不會做人的人不會處理做人的問題，事業上一敗塗地，生活也處於一片茫然之中。學會做人就得從我們自身開始，提升我們個人的修養和素質。會做人對於每一個人都是最重要的。「人」是通過「會做」而成，「做」的行為、實踐使得一個人成為「人」。

會做人得處理好三種關係：人自己的心身關係，人與社會的人際關係，人與自然的天人關係。會做人就是要做一個善良的人、樂觀的人、寬容的人、真誠的人、智慧的人、正直的人、謹慎的人、有志向的人、有教養的人。簡單地說，就是要學會讓人、敬人、愛人、寬容別人、善待別人、尊敬別人，不張揚、不狂傲、不顯露、不虛偽，因為這是做人的基本要求。

會做人是大難事，也是高深的境界。從普通平凡到不普通不平凡、從不普通不平凡上升到超凡脫俗，再從超凡脫俗上升到鶴立雞群，這就達到了做人的最高藝術境界。會做人、會說話、會辦事，此為人立身處世之三寶。

誠然，並不是所有的人都能把話說好、把事辦好、把人做好。但是，每個人都可以通過後天學習來獲得理念上的正確認識和行為上的靈活方法。為了幫助讀者儘快成為一個會說話、會辦事、會做人的人，本書以實用、方便為原則，將日常生活中最有效、使用率最高的口才技巧、處事方略、做人哲學介紹給讀者，使讀者在短時間內掌握能言善道、精明處事、完美做人的本領；讓讀者懂得如何在說話、辦事、做人時做到「到什麼山唱什麼歌，見什麼人說什麼話，遇到什麼事辦什麼事」。
