至親與至疏 留守家庭的雙面獨白

天天要聞
2019-11-24

摘要

“所謂的父女母子一場，只不過意味著，你和他的緣分就是今生今世不斷地在目送他的背影漸行漸遠。”而對有些孩子來說，沒能長到父母目送自己的年紀，卻早已習慣了目送父母遠去，在某個荒涼的路口，或在某個悲傷的車站，依依不捨，卻無可奈何。

「所謂的父女母子一場，只不過意味著，

你和他的緣分就是今生今世不斷地在目送他的背影漸行漸遠。」

而對有些孩子來說，沒能長到父母目送自己的年紀，卻早已習慣了目送父母遠去，在某個荒涼的路口，或在某個悲傷的車站，依依不捨，卻無可奈何。這是中國數以百萬計的留守兒童的常態，他們就像「小候鳥」一樣重複著與父母告別與守望的儀式。

父母遠遊，兒女留守，有一種專門拆散親子關係的怪物，叫做距離。陪伴缺失帶來的陌生無言，溝通技巧缺失帶來的詞不達意，讓彼此間生長出一道又一道隔閡的高牆。然而親子關係，始終是每個人生命中極其重要的一部分，站在高牆兩端的他們，內心一定隱藏著對親情的渴望，希望這件事有變好一些的可能。

關愛留守/流動兒童，希望能幫助這些父母和孩子打破距離的界限，找回親情的溫度。為了架起溝通的橋樑，我們走到高牆兩端，傾聽各自的內心獨白——

電話這一頭 是漫長思念後關閉的心房

電話那一頭 是千里之外無力言說的愛

「每隔幾個星期，爸爸媽媽會打電話回家，但我根本不想接。每次我都講個兩三句，就把電話扔給弟弟妹妹，但他們也不想接。電話筒就像個燙手山芋，在我們之前傳來傳去，最後都扔給奶奶。

奶奶說，他們去了很遠很遠的地方，坐火車要很久很久，接到他們電話，就算是見著他們了。我才不同意呢！前兩年，我還坐著電話機前等他們，遇到開心或者不開心的事，我都會攢在心裡，準備打電話時候講給他們聽。可是等啊等啊終於等到他們打來電話，我已經不想說了。每次問他們什麼時候回來，他們的回答卻是問我考幾分。後來我也不等了，反正一年到頭都見不著面，打電話又有什麼可說的呢？！」

——兒子劉方（化名）11歲 貴州

「我其實很關心孩子，但不知道和他們說什麼。每次打電話回家，他們都推來推去，不肯跟我講話，我真是又急又傷心。我在外邊幹活兒，見不著他們，只能抽空打個電話。可他們什麼都不願意和我說，和他們講道理也不肯聽。小孩一轉眼就大了，我也沒法陪他長大，說要他好好學習，其實是盼他以後過得更好一些。

有時我也想通過別的方式多了解一下兒子，朋友圈啊之類的，後來發現他早把我屏蔽了。唉……我真是個失敗的爸爸。」

——父親劉傑（化名）38歲 湖北

相隔千里的他們，電話是惟一的溝通橋樑。然而無法及時分享心事、獲取反饋，總是遲到的關心、無關痛癢的叮囑，卻讓彼此越來越無話可說。據《中國留守兒童心靈狀況白皮書》中調查顯示：全國有1500萬留守兒童3個月只與父母電話聯繫一次，還有7.7％的孩子與父母一整年都沒有見面。常年的分離，消磨了思念，徒增了生疏。電話里大段大段的尷尬沉默，淹沒在沉默里的關心，更日漸拉開了心靈的距離。

保持高頻率溝通，向孩子傳遞「我們雖然無法在一起，但我們的心一直在一起」的信息，是縫合心理裂縫的開始。很多人以為，每周打一次電話是最佳頻率。但有關研究表明，這恰恰是最糟糕的頻率。當孩子遇到心事時，既無法即時找到父母分享，也無法找到其他傾聽者形成穩固的關係。專家建議：父母至少每兩天與孩子打一次電話，在親子關係中形成穩固的依賴感。

他錯過她最愛的故事

也錯過了她成長的故事

分開5年，活成兩道平行線

「去年有一天，爸爸突然回家了！我興匆匆地捧著好不容易借到的書，想要把裡面最喜歡的故事給他聽，誰知道爸爸氣沖沖地催我去寫作業，還指著我的考卷說我不好好學習。可那張考卷明明是上學期的！好不容易見一面，還發脾氣，有時候真懷疑自己是不是親生的！他根本不關心我，不關心我喜歡什麼，開不開心，甚至連我讀幾年級都不知道！

每天放學我都要照顧弟弟，幫爺爺干農活，有時候真覺得生活太苦了。可是看課外書的時候，我好像能飛到故事裡五彩繽紛的世界……可是爸爸根本不會懂。他不讓我看，我還就偏要看，反正他離我那麼遠，也管不著。」

——女兒徐小江（化名）10歲 鐵嶺

「去年換工作時候，回了老家一趟，女兒挺開心的，拉著我講故事。我在煩心下一份工作的事情，根本沒心思聽這些。看到她考卷上的叉，忍不住罵她不好好搞學習，就知道浪費時間看些有的沒的。沒想到她傷心地哭了起來，之後好久都沒跟我講過話。

其實我最大的願望就是她將來能考個好大學，不要像我一樣，只能到處給別人打工。我也知道我們那邊農村條件不太好，連個像樣的圖書館也沒有。後來過年的時候，我特地去書店給她買了幾本故事書，聽店裡人說是和3年級課本配套的，結果回家發現她已經讀4年級了。現在我倆對課外書這事兒都不講了，我每次也都是想問又不敢問，怕一不小心又鬧得不愉快，總覺得心裡隔著什麼，但我也沒什麼辦法。」

——父親徐武（化名）35歲 瀋陽

她在貧窮家鄉苦苦留守，他背井離鄉為愛打拚。分開5年，沒有參與對方的生活，對彼此生活之苦的諒解也實屬苛求。因為不了解，那些包裹在嘮叨里的思念，欲說還休的關心，一不小心卻變成了誤解。那些說錯的話，那些送錯的禮，那些使錯勁的對峙在無形中把彼此推向陌生。徐武父女的故事並非典例，很多家長覺得，自己為孩子辛苦打拚，孩子卻不好好學習，忍不住臭罵一頓。因為在外打拚多有不易，更難免會把消極的情緒發泄在孩子身上，產生負面溝通的效果。

用愛溝通，好好說話，是修復陌生與對峙的溫柔力量。別總說為TA好，卻忽略TA的真正愛好。把「你應該如何」的大道理，換成體量彼此的小關心。如果無法改變地理空間上的距離，持續的言語關心，也是力所能及的重要陪伴，適時的安靜傾聽，也是參與對方生活的另一種方式，正向的誇獎鼓勵，更是親子溝通中，拉近孩子心靈距離的助推器。生活很苦，讓彼此不再對立，一起並肩對抗吧。

她說離開才能給我更好的家

而我們在一起才是最好的家

「我很小的時候起，媽媽就經常不在家，我每年只在過年的時候見到她。奶奶說她去了很遠的地方打工，說她要賺錢養我。媽媽說她出去打工是為了給我更好的家，那什麼是更好的家呢？我很羨慕那些有媽媽陪的小朋友，反正我覺得都比我家好。

今年聽說媽媽要回來上班，我高興得快要跳起來了，覺得像在做夢一樣。剛開始，我還是習慣和奶奶一起，後來媽媽每天都來接我放學，晚上抱著我給我講故事。原來媽媽還是疼我的！周末的時候，她還帶我去她工作的餐廳玩，陪我寫作業，教我做手工。她說以後一直就會這樣陪著我，真是太開心了。世上只有媽媽好，媽媽回家，是我今年最高興的事啦，比考100分還高興！」

——女兒林溪言（化名）9歲 鄭州

「小言出生後不久我就出去打工了，後來她長大了一些，就總問我為什麼要離開家。好難和一個孩子解釋，我出去打工是為了多賺點錢，讓她有更好的生活條件。但每年回家，發現她跟我都不親，也不愛跟我講話，反而跟他奶奶更親。

去年，聽說鄭州要開一家肯德基小候鳥媽媽主題餐廳，我趕緊跑回老家來參加了招聘會。沒想到這裡不僅工作穩定，還可以靈活調班。現在，我每天都有時間接小言放學，邊上班就能邊陪她寫作業，照顧她吃飯。剛開始，孩子雖然挺開心，但還是不太愛和我講話，我心裡有點難過。但幾個月里天天陪她之後，明顯感覺孩子變得活潑了。有時候餐廳還組織活動，讓我們帶著孩子一起做手工，我也趁這個機會多表揚她，讓她感受到我的關注和肯定。現在孩子遇到什麼事都喜歡跟我講，我心裡的愧疚感也少了很多。回家，是我做過的最好的選擇！」

——朱晨（化名）31歲 鄭州

「我雙手搬著磚就無法抱你，雙手放下磚就無法養你。」這是每個小候鳥家庭面臨的兩難。

「家裡都沒有媽媽在，什麼又是更好的家呢？」而小言的不解，也是梗在許多留守兒童心裡最尖銳的那根刺。

成長是不可逆的，每一次陪伴的缺席，或許都是未來回首的遺憾。久別重逢後，心與心的隔閡無法一鍵刪除，而親子關愛的補償，也不能是大水漫灌式的，而應該是滴灌式的。朱晨母女是幸運的，在小候鳥媽媽主題餐廳里，點點滴滴的親密互動，細緻入微的關心陪伴，漸漸修復了母女之間的隔閡。更值得慶幸的是，如今肯德基依託遍布全國近1300個城市的門店系統，至今已開放包括河南、重慶、湖南在內共260餘家餐廳，幫助背井離鄉的媽媽回到孩子身邊，從地理上到心理上拉近媽媽和孩子的距離。

而對於更多的留守家庭而言，不論是返鄉工作的重聚，還是節日里短暫的團圓，如何通過高質量的溝通陪伴填補過往的分別時光，避免兩地分隔導致的隔閡，依然是他們面對的重要課題。

雖然相隔千里，

那些刻在血緣里的愛從未消失，

因為相隔千里，

那些血濃於水的關係也開始不可避免地被稀釋。

三段「小候鳥」與父母的心靈對視，

也是千萬個留守家庭的縮影。

電話里大段大段的尷尬沉默，

相聚時眼神里的陌生和疏離，

面對隔閡的高牆，至親像是遠親。

父母與子女的緣分，只有這一次，

無論是遠隔千里還是近在咫尺，

我們都機會讓彼此的關係變得更近，

用愛溝通，好好說話，打破隔閡，互相靠近，

讓每一次帶著體溫的對話都能成為彼此取暖、

相互慰藉的力量。

我們也將持續努力，

期待能有越來越多的小候鳥媽媽餐廳，

讓媽媽不再遠遊，孩子不再留守，

彼此相伴守護，而不只是目送。
