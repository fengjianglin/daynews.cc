女兒出生在雪天，公公給女兒起了名，孕媽覺得這個名太棒了

天天要聞
2019-12-20

萌萌和老公是大學同學，在大學的時候他們的關係就非常好，畢業之後他們都去了同一座城市，之後就確認了戀愛關係，沒幾年就結婚了。結婚之後他們在婆婆的逼迫下他們就決定要生個孩子。

其實萌萌和老公一直也想要個孩子，主要是他們的工作實在太忙了，又都在事業的起步階段。如果生孩子的話，萌萌的事業還得從頭開始。可是婆婆和公公實在太想要一個孩子了。

萌萌沒多久就順利懷孕了，懷孕3個月之後，萌萌就辭了工作，她在家裡備孕。萌萌懷孕的時候就去做過B超，萌萌的老公通過熟人知道這一胎是個女孩，萌萌非常的擔心婆婆和公公不喜歡女孩，可是令萌萌沒有想到的是，婆婆和公公非常的喜歡女孩，他們說女孩是貼心的小棉襖，比男孩好多了。萌萌聽到後非常的高興。

懷孕四個月的時候，家人就開始著手給寶寶取名字了。萌萌的公公是個大學教授，所以起名字的任務就交給了公公。可是公公一直沒有想到一個合適的名字，公公覺得得根據孩子的出生時間去給孩子起名字。

寶寶出生那天，趕上一場久違的大雪，雖說天氣寒冷，但是家都覺得心情特別美好，全家人都盼望著寶寶的出生，經過剖腹產之後，萌萌成功生下了一個女孩。全家人都非常的開心，都為這個新生命的到來感到高興。

寶寶生下來沒多久，萌萌的公公就想到孩子的名字了，就叫：雪兒！聽到公公給女兒起的名字，萌萌覺得太好聽了，寓意更是深刻，女兒出生時正好趕上一場大學，而且雪象徵著潔白、純凈，女兒起這個名字簡直是絕配。

以後萌萌的女兒就有了一個非常好聽而且寓意深刻的名字了。多虧了公公能夠想起這樣一個名字。

大家覺得萌萌女兒的名字怎麼樣，真的沒有想到萌萌的婆婆和公公這麼喜歡這個女孩，這在許多重男輕女的老人心裡是非常難以接受的，其實小編覺得現在生男生女都是一樣的，女孩和男孩一樣都會成為一個家族的希望，同樣能夠家族帶來榮耀，我們應該摒棄一些不良的封建思想。
