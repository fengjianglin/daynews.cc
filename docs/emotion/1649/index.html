寶爸作為家庭的一員，應體諒寶媽難處，適當對寶媽伸出援助之手

2019-11-29 0條評論 21次閱讀 0人點贊

近期網路上流行了一個新鮮詞兒，名為詐屍式育兒，他是什麼意思呢？他是指那些平時對孩子不管不顧，出點兒事兒就跳起來瞎指揮的爸爸，確實和先前的喪偶式育兒有著異曲同工之妙，與其相同的還有一種育兒模式叫做裝死式育兒，也就是說完全忽略孩子的行為，裝聾作啞，假裝看不見。

一位網友在網路上表示，她老公就是裝死式育兒，在家裡也是屬於醬油瓶倒了也不會扶一扶的那種人，完全忽略孩子的存在，根本不插手孩子的教育問題，美名其曰教育只能一個人來全權交給媽媽做，不過是偷懶的借口。面對這種現象其他網友紛紛表示附和，可見這樣的爸爸並不是少數。

作為家庭的一員，我們應該體諒母親的難處，適當的對媽媽伸出援助之手，一起攜手插入孩子的教育問題，幫助孩子成為更好的自己。那麼在家庭教育中父母應該注意哪些問題呢？
相互配合相互協作
關於孩子的教育問題，父母應該統一戰線站在相同的位置對孩子進行教育，千萬不要公說公有理，婆說婆有理，一方過於強勢，而另一方過於弱勢，會給孩子帶來心靈上的差距，孩子都有著自己的小聰明，他們會傾向於弱勢方，也不利於孩子與強勢方的親子關係，對於孩子的教育問題，父母應該處於打配合的地位，正所謂相互配合相互協作，才能夠促使孩子走向更美好的未來，給孩子制定統一的規則，並且讓孩子按照此種規則實施進行，對於孩子的規則意識有著不可多得的作用。

適當示弱
面對孩子的時候，父母應該採取適當示弱行為，不要把自己拔的過高，凌駕於孩子之上，適當的弱化一下自己的權威，有助於培養親子關係，我們不難發現那些總喜歡把自己置在高處的父母，往往與子女的關係不是那麼和諧，不利於二者之間的共同出力，因此作為孩子的父母，我們應該學會示弱，往後退一小步，邀請孩子共同解決問題，尊重是大前提，尊重孩子的選擇自由權，相信孩子解決問題的能力與實力。

適當裝死
一些網友認為父親的裝死侍育兒並不好，但事實上在很多時候父母採取適當裝死行為反而有助於孩子獨立性發展，那麼在哪些方面裝死教育呢？其實很簡單，在關於孩子的問題選擇以及孩子的個人衛生上，孩子的家長不要過多參與，讓孩子自主完成自己的選擇，並且對自己負責，包括孩子的房間等乾淨處理，也需要讓孩子自己動手，這種過程看似微小，但卻會給孩子內心帶來極強的能量，不僅有助於孩子獨立自主的能力培養，也有助於緩解父母的艱辛。

懶惰是一門學問，適當的懶惰反而有助於孩子的成長，合理看待這個問題，不要包辦孩子的全部生活。

面對這個話題你是如何看待的？你有什麼新的見解，歡迎在留言下方評論。
