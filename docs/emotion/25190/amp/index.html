隨著時間的推移，這幾種份子錢不用隨，一定記住了

天天要聞
2019-12-21

故事 前幾天我的一個朋友和媳婦子打了一架，最後在朋友們的勸說下，才從歸於好。原因很簡單，就為了200塊錢，你們說可笑嗎？事情是這樣的，我們朋友的同事買了一輛車，他問老婆要兩百塊錢，給同事隨禮，老婆不給，然後就大大出手。其實他老婆也不是個吝嗇的女人，對人很熱情也很大方。他老婆是這樣說的，那個同事剛來單位才幾天，話都沒說幾句，沒有必要隨份子，我們的這個朋友不行，說是大家都說要去隨份子，我不去多沒面子。

那麼問題來了，隨禮需不需要誰叫都去，不管什麼事情，面子不能丟。是這樣嗎，大家認真思考一下，想想自己都隨過什麼樣的禮，有沒有和愛人為了隨禮發生過爭吵，我想答案是肯定的。我在這裡總結了幾條沒有必要隨禮的事情，供大家參考。

第一、好久沒有聯繫的朋友不用隨

一個朋友好久都沒有聯繫你，很明顯已經不算是朋友或者同事了，突然打電話叫你（兒子結婚、姑娘出嫁等）過來和喜酒，你覺得去了有意義嗎。無非就是突然想起還有你這麼個人，碰碰運氣看你來不來。

第二、孩子升學宴、買了新房子、提了新車的（關係一般的就不要去了）

我為什麼這麼說呢，向這種酒宴都是關係比較親密的家人或者朋友一起聚聚，開心開心，一個普通朋友去了，我覺的一點話題都沒有，是不是太尷尬了。他叫你去參加只不過是向你炫耀炫耀，看我買房子了、我換好車了、我們娃多有出息。到時間你說你生氣嗎，所以說還不如不去。就像我們那個朋友，為了給朋友賀車，和老婆大打出手，得不償失。

第三、經濟條件不好，不要勉強隨份子

不管是親戚還是朋友，能幫了就幫一把，沒有能力，就不要勉強自己，過好自己的日子最重要。有句俗話「打腫臉充胖子」，沒有一點點必要為了所謂的面子，所有的禮都隨，弄的自己月底緊張的都沒錢生活。我的一個朋友就是那樣的，別人隨一千，他也要隨一千。人家一個月掙一萬多，他三千多，還要和人家一樣。

第四、有去無回的禮不要隨

我們都講究的是禮尚往來，搭出去的禮如果收不回來，就不要去隨。我們也不是說，隨禮就是為了讓給我們還回來，不是這樣的。可有些人就是只收禮，不隨禮。

總結：雖然說，錢乃身外之物，但是也不能到處亂花，我們要花在該用的地方上。不要為了一時的面子和自己的愛人傷了和氣，得不償失。

以上就是我列舉的幾點，估計有點片面，但是都是發生在我身邊的事情，希望對大家有幫助。歡迎留言討論
