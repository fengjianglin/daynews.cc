著名作家喪子後喜結新歡，妻子願意離婚，他和新歡過上幸福生活

2019-12-25 661條評論 1,615次閱讀 240人點贊

婚姻大多數時候和愛情無關。―金庸

大多數人都是從小看金庸武俠劇長大的，都嚮往著俠義江湖。卻不曾想這些大俠背後的創造者金庸，在對待自己妻子時卻絲毫無俠義情懷。他為了娶年輕漂亮的老婆而與陪伴自己艱苦歲月的夫人離婚。在這些情感糾紛的背後，金庸又是為何做出這些決定的呢？

追求兩年卻遭初戀背叛

金庸在後來情感上背叛自己的夫人，在心性的改變上極大可能來源於第一位妻子―杜冶芬。

要說初戀才是最難以忘懷的，杜冶芬對金庸也是如此。當初金庸還只是一家報紙的編輯，負責笑話漫畫的板塊，而杜冶芬的弟弟則被金庸的笑話和漫畫吸引，成了金庸的迷弟。

後來報社舉辦了聚會，杜冶芬的弟弟也帶著杜冶芬一起參加聚會。結果金庸對杜冶芬一見鍾情，於是金庸對杜冶芬展開了猛烈的追求。金庸一閑下來便去杜家找杜爸喝酒聊天，討得杜家人的歡喜，然後每天很驚喜的給杜冶芬準備禮物和浪漫。就這樣金庸就把杜冶芬給成功的追到手。

後來金庸為了自己事業更好的發展便帶著杜冶芬來到了香港，而杜冶芬因為水土不服生活很不習慣，金庸因為工作也不是天天陪著杜冶芬。於是杜冶芬加入了一個同鄉會解悶，在同鄉會裡杜冶芬認識了一個男的，結果越走越近最後出軌了，金庸得知後十分難過但還是選擇了分手。

不得不說杜冶芬的出軌的確在金庸意料之外，自己苦苦追求兩年的妻子竟然會因為一個陌生男子背叛自己。一部分可能因為金庸工作繁忙沒有盡心陪在杜冶芬身邊，另一部分也是因為杜冶芬在這邊生活不如意所以才會選擇出軌。而杜冶芬的出軌和背叛也一定程度上改變了金庸，讓他不再忠誠於婚姻。

與妻子同甘共苦，生活苦盡甘來

金庸後來又找了一個妻子朱玫，朱玫很願意和金庸同甘共苦。在當時金庸曾想出來創業，於是創建了《明報》。結果沒想到工作很久卻一直沒有起色，最後窮的連員工都聘請不起，啥事都自己干。

朱玫看丈夫這麼幸苦於是辭掉了自己的工作，幫助金庸一起來整理《明報》。為了省錢他們上下班寧願走路也不坐公交車，後來為了讓金庸將報社開下去，把自己的首飾都賣了，還低著頭回家找家裡人借錢。即使日子再苦，朱玫也沒有絲毫的怨言，盡心儘力的幫助金庸度過難關，可見朱玫對金庸有多好。

後來金庸的工作有了起色後，他們日子也好了起來，朱玫也給金庸生了好幾個孩子，生活也十分的幸福。

就算朱玫再怎麼盡心儘力的陪伴金庸，金庸還是出軌了，而出軌的起因竟然是金庸的大兒子因為失戀自殺了。

因喪子之痛結識新歡，婚內出軌背叛妻子

朱玫本以為自己和金庸的生活能一直好下去，卻沒想到自己的孩子因為失戀竟然自殺了。金庸在得知噩耗後痛不欲絕天天去酒吧買醉。

金庸常去的那家酒吧有一個年輕美麗的服務員林樂怡，要說有才的人到處都有粉絲，金庸也不例外，林樂怡就是金庸的鐵杆粉絲。

林樂怡每次看到金庸一個人喝悶酒就忍不住問他發生了何事，金庸也想找一個人傾訴就一五一十的告訴了林樂怡，於是林樂怡就開導金庸。

年輕漂亮又有活力，是金庸看到林樂怡的第一印象，而在林樂怡的開導下金庸慢慢走出了陰霾，心情也開朗了起來，聊著聊著兩人就摩擦出了愛情的火花。

金庸也覺得在愛情面前年齡不是阻礙，家庭更不是，所以他也不顧忌自己的年齡向年輕的林樂怡吐露心意，林樂怡竟然也同意和金庸在一起，願意讓這個年齡都可以做自己父親的金庸成為丈夫。而這他們這段跨越時間的愛情也多為後人評價。有人覺得金庸為老不尊，也有人罵林樂怡想貪圖金庸的財產。

金庸覺得自己是真心喜歡林樂怡而不是只是和年輕姑娘玩玩，在得到林樂怡的回應後金庸就想和妻子離婚，讓林樂怡成為自己正式的妻子。金庸對朱玫從一開始的愛情隨著時間流逝已經變成了親情，也沒法從朱玫身上再找到愛情的味道。而且那時候朱玫一直忙著照顧孩子，金庸也忙著工作，兩人交流日益稀少，於是他覺得這可能是自己和妻子沒注意到大兒子自殺的原因，所以希望找一個年輕有活力的可以更好了解自己孩子的心思和想法，不會有代溝，這樣能避免自己大兒子的悲劇再次發生。

最後金庸還是沒有因為妻子和自己曾走過那段艱難歲月而動搖自己的決定，選擇向朱玫提出了離婚。

妻子同意離婚，婚後與新歡幸福生活

朱玫在得知自己丈夫出軌後，十分的傷心和難過，自己還沒從喪子之痛中走出，同甘共苦的丈夫竟然背叛了自己。朱玫還是十分的理智，她明白金庸的心已經不在自己這邊了，強行挽留也沒有用，於是她向金庸提出了兩個要求：

第一自己已經年紀大了很難再繼續工作養活自己，要求先給自己支付一筆養老的錢。

第二必須要林樂怡做結紮，避免她因為金庸的財產而來，等金庸死後搶奪屬於自己孩子的財產。

金庸為了追求自己的愛情竟然同意了朱玫的要求，帶著朱玫絕育了。金庸也如願以償的和朱玫離婚，與林樂怡過上了老夫少妻的生活，而朱玫卻一個人孤獨終老，孩子在她死的時候都不在她的身邊陪伴。

林樂怡作為年輕漂亮的女孩，為何會選擇金庸作為自己的丈夫，是因為崇拜還是因為真的想貪圖金庸的家產也不得而知。但是金庸在婚後喜歡上了林樂怡選擇與朱玫離婚，卻不曾想起朱玫曾陪伴他度過那艱難的時光，拋棄家中糟糠之妻。

這般行為或許在金庸自己看來就像大俠般瀟洒任性不受拘束，但是卻也失去了作為丈夫的責任，愛情上的事還真是說不準。金庸在得知朱玫死訊的時候也曾沉默良久說了句對不起，可能在他心裡對朱玫始終抱有著愧疚吧。
