幸福的人，也有難言的苦；悲傷的人，也有快樂的心，早安

2019-12-20 180條評論 1,321次閱讀 361人點贊

1、人生就像一場戲，每天上演著不同的各種角色，這場戲沒導演、沒有編導、沒有預告，有的人找對了角色，幸福一生，有的人演錯了角色，痛苦一生。有的人累了，懂了自己要什麼，懂得找個地方停靠。有的人累了，還拚命去演這個角色。有的角色，不是所有人都適合去演的，我們主導不了別人，但可以主導得了自己。

一輩子不長，有些精彩只能經歷一次，有些景色只能路過一回，錯過就錯過了。不要等，有時候等著等著，就讓等待成為一種習慣；不要怕，能說的立即說，能做的馬上做，不要瞻前顧後，不要畏首畏尾，你今天不做的，或許就是永久的遺憾；不要悔，路是自己選擇的，走過的錯過的，都是自己的情願。生活不一定要有驚天動地的情節才叫精彩，情感並非要有山盟海誓才算真愛。我們都是自己劇本中的唯一主角，最大悲哀就是不願做自己。早安！

2、 看起來幸福的人，心裡也許有難言的苦；時常掛著笑的人，心裡也許有無聲的淚；炫耀生活的人，可能遠沒表面那麼風光。一個人的幸福，一定是只有自己才懂的快樂。所以，不要跟自己過不去，不要糾結於別人的評說，照著自己舒服的感覺生活。幸福如人飲水，冷暖自知，你的幸福，不在別人眼裡，而在自己的內心深處。

人生是一種平衡，你擁有了這樣，必然會錯過那樣；你什麼都想得到，結果往往會失去更多。不要奢望人生中有絕對的公平，公平就如天平的兩端，一端的付出越多，別一端才能承載更多的希冀。世上的很多事，並沒有是與非的答案。女人的幸福在於：他真的愛我。男人的幸福在於：她值得我愛。女人最重要的，是優雅、有內涵、美而不艷俗。最好的男人，不一定要很有錢，不一定帥氣。男人最重要的是寬容、有承擔、忠誠而顧家。早安！

3、女人再美不能蒼白，男人再富不能俗氣。優秀不是外在條件，而是內心世界。一個女人的成功之處，就是把自己的男人塑造得讓更多女人喜歡。一個男人的成功之處，就是把自己的女人呵護得讓更多異性稱讚。有苦有甜才是味道，有黑有白才是顏色，有山有水才是風景，有起有伏才是道路，有晝有夜才是日子，有冷有熱才是季節，有情有意才是朋友，有愛有恨才是愛情，有聚有散才是生活，有哭有笑才是人生。如果一成不變，人生是多麼的乏味。

願每一位看到本篇文章的朋友，生活多姿多彩，快樂無限。看不開，就背著。放不下，就記著。捨不得，就留著，自己開心就好。等有一天，背不動了，就看開了！記不清了，就放下了！留不住了，就捨得了！所以說有些事情不要太計較，睜一隻眼，閉一隻眼，就會過去的。所以，沒有必要凡事較真，遇事別較勁。早安！

4、珍惜眼前的人，做好眼前的事。一切都是美好的！朋友請記住，陰天，不一定會下雨。人生的渡口，我們皆是過客。不是所有的人，都能知心。不是所有的緣，都能永遠。有時緣分，不在於遇見多少，而在於真正擁有多少。有時朋友，不在於陪伴多少，而在於懂你的人有幾個。沒有過不去的事情，只有過不去的心情。只要把心情變一變，世界就完全不一樣了，一定要活出自己的精彩。

活著，就要善待自己，別跑到別人的生命里當插曲。有時候，這個世界很大很大，大到我們一輩子都沒有機會遇見。有時候，這個世界又很小很小，小到一抬頭就看見了你的笑臉。所以，在遇見時，請一定要感激；相愛時，請一定要珍惜；轉身時，請一定要走的優雅；揮別時，請一定要面帶微笑。生命中，有苦有甜才是味道，有聚有散才是生活，有情有意才是朋友，以上共勉。早安！
