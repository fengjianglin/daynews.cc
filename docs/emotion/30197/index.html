平安夜｜人生路坎坷，做一個精神明亮的人，走過黑暗，一生平安

2019-12-24 2,626條評論 4,748次閱讀 2141人點贊

我們生活在這個世上，並不會事事平安，時時順心，平凡的生活中，總會有這樣那樣的不如意。

生老病死、酸甜苦辣、悲歡離合，各種滋味嘗遍，我們常常是滿腹辛酸，肩頭的壓力沉沉。

錢多錢少，平安就好，人生最大的福氣，就是做到「心安」，「心安」的人，才是真正有福氣的人。

一、只有流過血的手指，才能彈出世間的絕唱。

天有不測風雲，人有旦夕禍福。看開就是福。

生活中，同樣是遇到不幸的人，有的人能化險為夷，可是有的人卻不能躲過災禍，使自己一步一步陷入更深的災難。

同樣是得了癌病，有的人活了下來，有的人可能被死亡奪取生命。一方面因為病情不同，另一方面可能是心態不一樣。

斯坦福大學的一項研究表明，心態決定你的壽命。

斯坦福大學的心理學助理教授Crum博士和商科博士生Octavia Zahrt，通過分析六萬名美國成年人的調查數據，揭示出思維方式和心態直接影響我們的健康。

研究人員強調，該研究雖不足以證明有著不健康的心態的人們會早死。但是，它可以說明心態和健康之間確實存在因果關係。

活在世上，旦夕禍福既然不可避免，那麼，就要學會正確面對災禍，看開了，看淡了，能化險為夷，轉危為安。

大家熟知的內地演員胡歌，2005年因在電視劇《仙劍奇俠傳》中成功塑造了的「李逍遙」而成名，成為不少粉絲心中的男神。

2006年，胡歌與女助手張冕，乘坐的現代旅行車與一輛廂式貨車發生追尾碰撞，助手搶救無效身亡，胡歌右眼重傷，臉和脖子加起來縫了100多針。

一個演員毀容，是難以接受的冷酷的現實。我們無法想像在那些絕望的黑暗的日子裡，胡歌的內心經歷了怎樣的煎熬和悲催。

但是，他走出來了。看開了，眼前一片明亮。

正如泰戈爾說：只有經歷地獄般的磨鍊，才能煉出創造天堂的力量；只有流過血的手指，才能彈出世間的絕唱。

胡歌出了車禍後，他對人生的思考有了質的飛躍，與死亡擦肩而過，劫後餘生，他變得沉穩、大氣。

眼角有點瑕疵，看上去，帥氣中透出一點滄桑，更富有成熟男人的味道。

從《琅琊榜》到《偽裝者》，胡歌逆襲了，慘遭車禍之後的胡歌，不但沒有垮，演技愈加精湛，他演的這兩部劇都成了很多人心目中的經典之作。

正如歌德說：流水在碰到抵觸的地方，才把它的活力解放。

悲觀的人，有可能被苦難壓扁，樂觀的人，在苦難中升華，造就驚人的才華。

人生之路，沒有一馬平川，總有磕著、絆著的時候。過日子，沒有事事順心，總有這個疼、那個痛的。

但是最難的時候，往往是我們最接近成功的時候。歷經世事滄桑，做到安之若素的人，才能內心無恙。

二、人生路坎坷，做一個精神明亮的人。

人生有什麼劫數，不可預知。

我們沒有辦法去避免天災人禍，但是我們有能力去改變自己面對苦難的態度。

當不愉快降臨的時候，我們所能做到就是保持陽光的心態。

我的表姐患了乳腺癌，右邊的乳房被無情切除了。

出院後，她就是一次又一次的返回醫院去做化療。醫生說一般人化療到第八次，頭髮就會掉落到稀疏。

可是堅強的表姐，直到第八次化療結束，一頭漆黑的頭髮依然完好無損。

別人問她是怎樣做到的？

她說，自己就是心態好。該吃吃，該喝喝，該玩玩，從不去胡思亂想。想多了，不就是自己用負面情緒來懲罰自己嗎？

她說：當別人用鄙視的目光冷漠的看著你的時候，你依然要狠狠的熱愛自己。

她說，自從一側乳房被切除之後，遭遇最多的是「各種各樣」的目光，同情的、鄙視的、不屑的、詫異的、幸災樂禍的……

她說，如果都去在意，那麼心上該被插上多少根芒刺啊！

她說，每個人都是過自己的日子，我是生病了，可是我又不礙著誰，我不需要去在意他們的眼光。

「我只要自己開心起來，自己活得好就行。」

岩石縫隙里開出的花朵，總是更能讓人敬佩。表姐是樂觀的，她化療了8次之後，去檢查的時候，醫生說她恢復得異常的好。是病人中少見的堅強的人。

於此，為表姐祝福，願她餘生平安。願心態陽光的我們餘生都平平安安，健健康康。

三、心態決定一切，心安是福。

當今社會複雜多變，人們每時每刻都要承受著或多或少的壓力，這些壓力來自社會、家庭、工作、婚姻等各個方面。

當壓力超出個人所能承受的心理界限時，心理疾病就會產生。所以，做一個精神明亮的人非常必要。不去憤世嫉俗，克服卑微懦弱的心理，減少恐懼和緊張的心理。

經常保持愉快的心情，你要這樣想：人生最難的，也就那麼幾件事，疾病、衰老、失業、意外，來了就坦然面對、承擔、接受、改變不了就想辦法適應。

逃避、抱怨、畏懼只會讓事情變得更糟。當你害怕困難的時候困難就會嚇倒你，當你不怕困難的時候你就能把困難嚇跑。

做人，要禁得住挫折。不管遇到什麼棘手的事情，首先把心態擺正。

心態正了，人生的路也順了。走過黑暗，方能一生平安。
