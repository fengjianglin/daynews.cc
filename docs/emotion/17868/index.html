7個信號告訴你，孩子不是鬧情緒，可能有心理問題

2019-12-17 58條評論 1,654次閱讀 730人點贊

你知道嗎？

5個孩子里，就有1個可能有心理問題。

這是11月5日聯合國兒童基金會和世界衛生組織聯合公布的一組最新數據。

我的一位朋友正在讓6歲的兒子看心理醫生，原因是他的小動作非常多，經常攻擊班裡的同學，還時不時情緒大爆發，場面很難控制……

朋友說，經過幾次沙盤治療，孩子的情況明顯有好轉。

越來越多的媽媽送孩子去做心理諮詢，但更多的父母，可能會覺得孩子有心理問題，是件不光彩的事。

生病很正常，但心理生病，就很難接受：

「再等等看，或許是我過度反應了」

「孩子長大不就是有很多反常行為嗎？」

可是，如果孩子有這些行為表現，還是需要重視起來。

孩子出了問題，而父母不願面對

當孩子發燒或咳嗽時，父母會毫不猶豫地打電話給醫生尋求建議。但是，如果孩子出現兒童精神疾病的癥狀，如抑鬱、焦慮、躁鬱症和破壞性情緒失調障礙（簡寫為DMDD，是一種與持續的易怒和頻繁發脾氣相關的疾病），他們卻多半不願直接面對。

辛迪·弗賴菲爾德居住在美國紐約州的亨廷頓，她的女兒亞歷山德拉一開始喜歡蜷縮著入睡，但是到3歲的時候，她開始堅持讓媽媽用一種奇特的方式說晚安。「我必須對她連續唱三遍『我愛你』，如果我做得不夠好，或者音調不對，她會讓我從頭再來一遍，可以這樣持續差不多1小時。」

這個習慣讓人感到不安，由於是第一次為人母，辛迪向朋友們表達了這種擔憂。「每個人都說，孩子們渴望規律的生活很正常。」於是辛迪試圖對此一笑置之。

但一年後，亞歷山德拉開始把玩具和其他東西——娃娃衣服、舊手機、沒吃過的棒棒糖等，按特定的順序擺放在房間里，還因為擔心朋友們會碰她的東西，拒絕任何人過來。

「我丈夫一直說，『她很好，這只是成長必經的一個特定階段』。」辛迪說，「但我走過她的房間，看到她一個人在擺放那些東西，我就會有一種可怕的直覺：不，我的小女兒一點也不好。」儘管如此，辛迪仍然不知道接下來該怎麼做。

為什麼會有逃避情緒？

美國Parents雜誌與兒童心理研究所（CMI）合作，在全美範圍內調查了大約400位母親和父親。65%的受訪者表示，如果發現自己的孩子有心理健康癥狀，會等上幾周甚至更長時間再尋求幫助。

調查中絕大多數家長說，他們想等等再看醫生，是因為不想反應過度。但其實，為孩子尋求幫助從來都不是過度反應——即使你最終發現他沒問題。

調查中，70%的受訪者認為心理健康診斷帶有恥辱的色彩，只有36%的人說他們會坦然討論對孩子的診斷。Koplewicz博士建議：「需要全社會做出更多努力，消除孩子會被羞辱或被認為是『瘋子』給父母帶來的恐懼，使他們可以毫不猶豫地尋求幫助。」

識別心理健康「預警信號」

最終，讓辛迪真正意識到並承認女兒的狀況真的不好的，是一個艱難而戲劇性的時刻。一天，辛迪讓女兒最喜歡的表弟來到她家門前，而亞歷山德拉在地板上打滾、尖叫，彷彿處於極度痛苦中。「我內心最後一絲印證女兒沒問題的幻想也消失了，」辛迪說，「我和女兒一起哭了起來，很明顯她真的非常痛苦，我們需要幫助。」

如果你擔心孩子的心理健康，那就先和醫生聊一聊。「我們經常把這些擔心說成是『預警信號』，而不是『危險信號』。」Briggs博士解釋說，「雖然它們不一定是臨床疾病的明顯信號，但它們可能是病情發展中體現問題的微妙證據。」

最常見的7種信號如下：

1.睡眠不規律

除了嬰兒時期，兒童應該每晚睡10小時左右。患有抑鬱症的孩子有時似乎過於睏倦，並且睡眠時間不固定。那些患有焦慮症、注意缺陷多動障礙（ADHD）或破壞性情緒失調障礙（DMDD）的人通常需要數小時才能入睡，且每晚醒來多次。

2.胃腸道問題

肚子痛是孩子們經常抱怨的問題，但經常出現的胃痛不能僅僅用便秘或食物不耐受來解釋，要考慮可能跟心理有關。長期以來的研究表明，兒童慢性胃腸道疾病與焦慮和抑鬱確實存在一定聯繫。

3.強迫思維或恐懼感

在那些可能有焦慮的孩子中，某種想法可能變得偏執而強烈，以至影響到日常生活。常見的強迫思維導火索，尤其是對患有強迫症的兒童而言，是「安全」和「細菌」。比如，患有強迫症的孩子即使在不方便的時候，也可能強迫性每天洗無數次手，以減輕他的焦慮感。

4. 凡事都提不起興趣

普通孩子都會有不同的激情，但抑鬱的孩子對任何事情都不會感到興奮。「他們很容易就會發現，即使是那些曾經讓自己興奮的事情，現在也無法從中找到樂趣。」聖路易斯華盛頓大學醫學院早期情緒發展項目負責人Joan Luby博士說。

5.有負罪感

「一個容易抑鬱的孩子可能會因一些小小的過失而感覺非常糟糕，需要異乎尋常的安慰才能感覺稍好一些。」Luby博士說。最近的大腦研究顯示，在有負罪感傾向的兒童和抑鬱的成年人中，其大腦中被稱為「前腦島」的區域低於人類平均水平，暗示這種性格特徵可能是日後抑鬱的一個強有力的預測因素。

6.爆炸性的憤怒

具有攻擊性、破壞性行為，和日常生活中有異常激烈的憤怒跡象，可能是破壞性情緒失調障礙、抑鬱症和其他相關疾病的癥狀。

7.消極的想法或自殘行為

Luby博士在針對抑鬱的學齡前兒童的研究中發現，許多人在想像的遊戲中表現出病態的行為。即使是很小的自殘行為也可能是一個預兆。

專家建議的3種療法

辛迪在網上搜索了有關女兒癥狀的線索，亞歷山德拉的表現完全符合強迫症癥狀的描述。很快，辛迪找到了一位專家，對女兒的行為狀況作出診斷，並開始用認知行為療法進行治療。不到6個月，亞歷山德拉就能更好地放鬆自己，向父母傾訴自己的擔憂了。

專家普遍認為，以下方法對患有精神類疾病的兒童特別有效：

1.認知-行為療法（CBT）

認知-行為療法可以幫助孩子打破扭曲的思維模式。治療師會幫助患有抑鬱症的孩子確認某個消極的想法（如「我沒有朋友」），引導他找出更客觀的現實觀點（如「我有兩個朋友，但我想要更多」），並提出應對策略（邀請新朋友一起玩）。焦慮的孩子受到恰當的鼓勵，可以循序漸進地面對自己的擔憂，並學習在恐慌時找出應對的技巧。

2.親子互動療法（PCIT）

整個遊戲過程由臨床治療師引導。爸爸媽媽要學慣用強烈的讚美來鼓勵孩子，使其往想要達到的行為走向發展，一旦孩子有爆發傾向，也要用溫和而堅定的方式阻止。親子互動療法適用於2歲以上的孩子，對患有破壞性情緒失調障礙（DMDD）、注意力缺陷多動障礙（ADHD）和其他行為障礙的孩子很有幫助。

3.藥物治療

藥物治療並不是兒童的首選治療方法，因為在過去的時間裡，針對學齡前兒童的用藥研究相對較少。儘管如此，仍有一些藥物被證實用於學齡前兒童是安全的。如利他林（鹽酸哌甲酯）被美國食品藥品監督管理局（FDA）批准用於6歲及以上的多動症兒童，而百憂解（鹽酸氟西汀）則被FDA批准用於8歲及以上的抑鬱症兒童。

找對醫生、用對方法，可以產生強有力且持久的治療效果，而且效果來得很快。要想對青少年或成人產生影響，需要3年的干預措施；而對一個年幼的孩子來說，可能只需要4～6個療程。可能孩子在很長一段時間內都要與這類精神疾病作鬥爭，但越早採取措施，就能越早帶給他擺脫的希望。

「太丟人了，我的孩子不可能得這種病」，「其他人都說沒事，也許過段時間他自己會好」……也許對家長來說，只是固有觀念帶來的不自覺遲疑，但卻因此讓孩子錯過了最好的治療時機。在孩子真正的健康和快樂面前，所有的猶豫的想法都應讓位。只有積極尋求幫助，孩子才能在愛和支持中迎來改變。
