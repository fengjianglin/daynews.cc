“捨不得”這些，職場就會捨得淘汰你了

天天要聞
5 天 ago

有沒有很多東西是你捨不得的？一張舊照片、一個軟體、一個朋友、一段回憶、一個小時的時間……

幾年前的一天，我看見合租的鄰居家突然變得乾淨又整潔，又看看自己家裡，東西多，髒亂差，和她家相比，真是天壤之別。

「你是怎麼把屋子收拾得這麼乾淨的？」我問。

「很簡單，就是把你現在不用的東西全部處理掉。」

「什麼？那你以後不用嗎？再說，怎麼捨得？」

「以前我也捨不得，但東西越積越多，都堆在屋子裡也沒什麼用。後來終於下定決心把東西清理了，屋子瞬間整潔了很多。」

不只是整理房間，人的大腦也是一樣。

如果人的大腦是一個有限磁碟，那麼這個磁碟里只能放下有限的東西，和電腦手機一樣，剩餘空間越小，事物處理速度越慢。

這也就是為什麼如果一個人同時做很多事會低效的原因。

所以，如果你想成功，必須清除一些你捨不得卻沒什麼用處的事物，發展才會越來越好。

職場是一個需要你投入大量時間精力的場所，存留太多的「捨不得」，你就無法集中精力應對工作中形形色色的人物事物。你要做的是，將阻礙你的「捨不得」清理掉。

今天，我們就來看看，阻礙你發展的都有哪些「捨不得」？

1. 捨不得的「空閑時間」——刷手機

你捨得拋下刷手機的時間來認認真真看一本書嗎？先別回答得太早，即使你現在捨得，在拿起手機看到各種吸引人標題的那一刻，舍不捨得就不一定了。

根據一家德國數據統計公司的報告，我國每天人均刷手機的時間為3小時，位居世界第二，僅次於巴西。

​也就是說，我們平均一天要看手機約150次。

多麼恐怖的數字，如果用手機是看書、聽課、做些有益的事還好，不過，大部分我們都是在瀏覽雜七雜八的消息，感覺也沒有學到什麼，每天3個小時就憑空「丟了」。

如果用這些平時刷手機的空餘時間騰出來做些對職業發展有益的事，將會是多麼大的財富啊！

假如，你每次看無用文章或視頻的時間為20分鐘，每次點開第一個「想看的東西「時，提示自己這樣刷手機是無用的，趕緊關掉收起。計時20分鐘，用這20分鐘的時間做些有利於你短期目標的事，比如：讀書、運動、喝水、聽課、做題等等。每天這樣做三次就能省出1個小時的時間，一年就是365個小時。每天工作8小時算，你一年約有45天的時間在全日制投資你的未來。比一個在職研究生整年上課的時間都多呢！

如果你捨不得放下手機，那就留心記錄一下你每次、每天玩手機的時間，想一想如果用這些時間來做你喜歡但卻沒時間做的事情，會多麼開心！

2. 捨不得的衣服

俗話說得好，三分的長相，七分的扮相。雖說現在化妝術大行其道，但一個人若只會化妝，不會穿衣搭配的話，看著也很彆扭。但我們常常會存很多衣服，一些保養不及時、年齡過老的舊衣服，或自己不喜歡的衣服呢？

如果這樣的衣服你已經超過2-3年沒穿過了，我還是勸你把它處理掉，你可以給親戚朋友，或是捐給貧困山區，要麼掛到閑魚等二手網站上，要不然就直接扔掉。

因為舊衣服你穿上整個人的精神和品質都會大打折扣，至於你不喜歡的衣服，穿上了還會有美麗心情，還會自信嗎？

當然，如果你將部分還能挽救的衣服拿出來洗乾淨不顯舊，也剛好配得上其它的服飾，也是個不錯的選擇哦。

3. 捨不得的「朋友「

打開你的微信通訊錄，數一數你能找到幾個認識的人，如果滿眼都是不認識的陌生人，那麼恭喜你，你的朋友圈已經被廣告包圍了。

試著將認識的人和不認識的「廣告「分開吧。要麼把他們刪掉，要麼把這些」推銷員「單獨分作一組，這樣你才會擁有真正的朋友圈。

另外，你的微信中有沒有那些常年不聯繫且不喜歡的人，你可以將他們分作一組，如果覺得實在佔用內存，也可以直接刪掉。

損友敬而遠,益友宜相親。

如果有一些人，經常讓你覺得和他一起玩沒有提高，人生境界越來越低，每次聊完或出去玩之後總感覺心裡空落落的，交往過程中感覺有些無聊或浪費時間。這說明，你該離這樣的人遠一些，用跟他在一起的時間做些有意義的事了。

4. 捨不得的公眾號和軟體

同樣，打開你的微信，看看哪些公眾號是你真的有用才添加的，哪些是廣告推銷？第一步，你覺得無用的廣告先刪除。

還有一些公眾號是你當時覺得有意思，之後它發的文章都是廣告或者你不太喜歡的。第二步，內容質量低下的公眾號請刪除。

另外有一些公眾號是你還蠻喜歡，就是沒時間看的。因為這些喜歡但沒時間擺弄的東西使你的手機已經多次提醒「儲存空間所剩不多，請及時清理內存「了。可以先用本本或者百度雲等工具記下來，再刪除，等你手頭上的事情忙完，有空閑時間了再下載也不遲。

軟體也是一樣，先刪除經常推送廣告的垃圾軟體、因為廣告才下載的軟體，再刪除內容質量低下、你根本不喜歡的軟體，最後刪除沒時間看的軟體。

人生是一個無限與有限的交互體，你可以用無限的眼光去看待這個世界，卻敵不過有限的歲月慢慢流逝。

韶華易逝，青春有限，舊的不去，新的不來。在很多時間，我們要找出自己的理性面，將一些無用的「捨不得「在現時的人生中刪除，以更加專註的姿態面對當下的人生，以高度集中的精力處理手頭的事物。

集中精力工作，敞開心扉生活，勤奮、耐心、堅持，成功將在不遠處等著你哦。

你覺得還應該丟下哪些「捨不得「呢？
