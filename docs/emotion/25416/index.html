做好以下三件事，才能撐起家人的幸福

2019-12-20 1,618條評論 2,900次閱讀 945人點贊

點擊上圖關注我哦~

作者：青衣

到2019年，最後一批80後，也已經步入了30歲的行列。

雖然心裡還覺得自己像個小孩子一樣，只是不那麼容易哭出聲了。

到了30歲該懂得的道理,必須要懂得了，就像前不久網上看到的那句話：

你要搞清楚自己人生的劇本——

不是你父母的續集，不是你子女的前傳，更不是你朋友的外篇。

我們都有責任為自己的一生負責，活好這一生，記住以下幾點。

擁有獨立的經濟

有些人，結婚之後就不想出去工作了，就想呆在家裡過衣食無憂的生活。

這樣的生活真的就那麼無憂嗎？

前不久，火爆網路的一段短視頻，擊碎無數人的夢。

視頻以真人為原型，講述他們真實的故事，被採訪者分別從事著不同的行業，有著不同的生活困惑。

然而，他們有一個共同的特點——都是30歲。

其中一位最讓人心酸的講述者，深深打動了無數人。

她是一位全職媽媽，在30歲的時候離婚，獨自一人帶著孩子。

離婚，切斷了她的一切經濟來源。由於很久不工作，她發現自己什麼都不會。

圖片源自網路，侵刪

她必須像個小學生一樣，跟著比自己小十幾歲的年輕人，一切從零開始，做收入低廉的工作，學最基本的生存內容。

她眼裡的膽怯和不自信，讓人看著心疼。她說：

因為之前是全職媽媽，沒有工作經驗，30歲了，離婚後發現很久沒有工作了，只有努力去適應社會的環境。

其他幾位從容淡定，目光如炬的同齡女士，都已步入高管階層，有著不菲的收入。

相比較那些早早步入職場的人，她缺乏的東西太多了。

一個30歲女人外在的從容淡定，以及風度和氣質，其中一個重要的原因就是獨立的經濟能力。

想要獲得好的生活，首先要有一份工作。

具備基本的生存能力，獲得了獨立的經濟能力，贏得獨立的人格，才有資格談論詩和夢想。

當意外來臨的時候，不至於一敗塗地，能給予自己和所愛的人，最基本的生活保障。

獨立的經濟，撐起來的不僅是生活最基本的需求，還是活好這一生的最低保障，同時是誰也拿不走的人生體驗和生存的底氣。

熱愛你的工作

稻盛和夫有一句話說得好：

要想度過一個充實的人生，只有兩種選擇：

一種是「人，自己喜歡上工作」，一種是「讓自己喜歡上目前的工作」。

熱愛你的工作，像戀愛一樣迷戀你的工作。

中央電視台有一期被轉發60多萬次的《心理訪談》節目，其中女攝影師汪瀅瀅的親身經歷，深深打動了無數觀眾。

她36歲才擁有第一台單反攝影機，選擇了攝影師這份工作。

之前她從來沒有學習過任何攝影技巧，然而她為顧客拍出的寫真集，讓每一個人都感動到哭。

在汪瀅瀅的鏡頭下，那些年齡過了30歲的女人，臉上的皺紋和斑點都清晰可見。

她不用過濾鏡頭和修圖，她不提倡虛假的完美，她只展現每一個人最真實的一面。

她不僅僅是用鏡頭拍攝，她跟每一個顧客交換人生故事，了解她們的過往和思想，拍下每一個人最真實的樣子。

圖片源自網路，侵刪

照片不完美，但是真實有厚度、有故事，這是最打動人的原因。

她通過攝影這份工作，讓更多的人看到真實動人的自己，通過照片呈現這個年齡段最自然的狀態。

工作的意義不僅僅是賺錢的工具，還能從工作中獲得成長，也可以通過工作增進人與人的交流。

人生獲得更好的淬鍊，工作是獲得充實和滿足感的一個重要途徑。

前不久看到一則新聞，美國心臟外科醫生邁克爾·馬克做了一件瘋狂的事。

他向自己91歲的病人發起挑戰，兩人一起約跑5公里，以此慶祝手術的成功，結果他們做到了。

邁克爾·馬克大夫說：

「對於老年患者來說，他們的生活質量比他們的壽命更重要。

能夠看到他的生活質量回到從前，那就是手術的意義。」

一位醫生的職責是幫助他人解除身體的病痛，這就是他工作帶來的神聖意義。

每一份工作都有特殊的責任和義務，每一個人都要有一份自己的工作。

無論是男人還是女人，通過工作服務他人，收穫成長，人生得到淬鍊，這就足夠了。

愛惜身體

如果工作能獲得成就感、滿足感和幸福感，那麼承載所有這一切美好的前提是：

必須有一個健康的身體。

30多歲，真的是一個非常尷尬的階段，頂著巨大的壓力，是家裡的頂樑柱，一定要愛惜自己的身體。

近年來，每年我國過勞死的人數多達60萬，是超過日本的第一大國，而且年齡以30歲上下的人居多。

記得一則新聞，成都的楊先生，從事IT工作，大家公認的「工作狂」，熬夜加班是家常便飯，身上兩部手機24小時隨時待命。

有一次，他的朋友圈裡發的動態是：

每天下班，都有種要虛脫的感覺，我需要打雞血。

在一次外出辦公的途中昏倒，再也沒有醒過來，年僅34歲，留下上有老下有小和悲痛欲絕的妻子。

還有，在北京地鐵站台，突發腦溢血去世的副主編金波，經常加班熬夜，工作起來像個拚命三郎。

表面上看起來非常強壯的身體，由於長時間的過度消耗，早已積勞成疾。

還有優秀的潘潔，她在日本和德國拿到雙碩士學位，從事審計工作，經常加班到半夜1、2點才回家，有時候甚至到凌晨3點多。

據她媽媽回憶說，女兒太累了。

她去世前一晚回到家，沒有洗臉，沒有洗腳，倒頭就睡。

第二天早上，潘潔開始發高燒，一度昏迷，被送進醫院，再也沒有睜開眼睛看看年邁的父母和這個美麗的世界。

有人曾經說過：我們這個社會是怎麼了？每年都花很多錢保養車，怎麼就不知道好好保養保養自己的身體呢？

生命是一次沒有回程的旅行，用力過猛註定走不遠，用十倍的精心來保養自己的身體，怎麼愛惜都不過分。

因為我們擁有強健的體魄，才能撐起家人的幸福。

學會認慫

復旦大學教授陳果說得好：

世界上只有兩件有價值的事：

第一、一定要好好活著；

第二、在有可能的情況下，幫助別人好好活著。

工作是為了更好地生活，累了該休息的時候休息。

不要硬撐，累到撐不下去不是「打雞血」，而是休息，失去生命什麼都沒有了。

不是所有的認慫都是孬種，因為還有一種認慫叫「退一步海闊天空」，還有一種認慫叫「留得青山在不愁沒柴燒」。

  • 我們一定要好好活著，單位里少了一個人，馬上有人取代你的位置；
  • 而家裡少了一個人，就像被拔掉的一顆牙齒，是永遠無法取代的思念的痛。

我們十年寒窗苦讀換來的知識，擁有了工作的能力，獲得經濟能力，一切不都是為了生活變得更美好嗎？

怎麼在忙碌和焦灼中，忘記了最初的初心了呢？

如果連愛自己都做不到，何談愛家人、愛工作，愛生活，從現在開始，從30歲開始，一點都不晚。

  • 能不熬的夜，就別再熬了；
  • 能不喝的酒，就別再硬喝了；
  • 能不逞的強，就別再死撐了。

到了30歲的年紀，能對自己好一點，就對自己好一點吧，我們活著就是為了見證生活的美好。

30歲多好的年紀，還有許許多多未來的期許，我們一起好好愛惜。
