那些成績優秀的孩子，有著和他們一樣優秀的父母，你不妨也這樣做

2019-12-19 825條評論 2,576次閱讀 1063人點贊

一個孩子在樹立自信、堅強、樂觀的性格基礎上，需要父母在孩子的成長過程中不斷進行正能量的熏陶。

那麼爸爸媽媽們在孩子成長過程當中，對他們說的哪些話可以更加有益於孩子健康人格與優秀品質樹立起來呢？

父母常常鼓勵孩子。

就像人們在談戀愛的時候，更加渴望另一半會說一些看起來不實用，但是聽起來非常浪漫的話，可以在一定程度上滿足自己正能量的需求一樣。孩子在成長過程當中，對於父母一些鼓勵的話，也十分期待和看重。

當孩子在出門之前，媽媽說一句：「加油，你一定行」。可能在孩子眼中這樣的話顯得肉麻並且毫無意義，但是長此以往的鼓勵語言成為了加強與孩子之間溝通的必須時，孩子心中也自然會產生「我能行」的潛意識，並且這樣的溝通有利於孩子們接收到父母認可自己的態度。

用商量的語氣培養孩子的自信。

每一個生命個體在整個世界當中都應該受到尊重，擁有自尊心的人們，對於一件事物的選擇會有自己的判斷能力，而對於強制性的控制，自然會產生一種心理抗拒，孩子也期待著大人們能夠給予自己一些建議，而並非由父母全權做主，於是這時候家長們用商量的語氣和孩子溝通，會有效增強孩子的自信心，增強他們獨立自主的選擇意識。

比如經常在孩子面前說：「你在某方面的能力十分突出，真棒」。而大多數孩子在成長過程中極度渴望得到家長的認同，為了維持大人眼中的優秀模樣，他們會儘可能的去發展被認可的那一項才能。

在孩子成長過程當中，很多優秀的習慣在父母身上一旦被培養起來，孩子們也會潛移默化的去模仿去學習。例如上述說的兩項，應該按照每個家庭每一位孩子的不同情況，而制定合理的範圍。

部分家長會認為道理都懂，但是做到的沒有幾個。這類父母可以換個角度想想，當自己和孩子們一起成長的時候，很多困難商量著一起去克服時，自然會發現有些教育只是看起來很難而已。畢竟父母們都是在孩子成長過程中付出艱辛的那一位，既然都朝著讓孩子更加優秀的一個目標而努力奮鬥著，家長們也應該明白，哪些語言可以有效促進孩子優秀成長。
