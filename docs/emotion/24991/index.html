人到老年1個變化有3個特點，如果她是婆婆，孝順兒媳要懂得

2019-12-23 1,709條評論 3,809次閱讀 1068人點贊

婆媳關係始終是婚姻中無法繞開的話題，對婆媳關係不理想的，常常用勢如水火來形容，這個比喻沒錯，處理不好的話，婆媳之間還真的是水火難容。

如果知道了一些道理，運用了一些方法，婆婆與兒媳的關係就會是另一種和美的狀態。

水還是原來的水，火還是原來的火，也能和睦共處，相得益彰。火可以發揮出自己的優勢，把水燒開，消滅裡面的細菌病毒，讓水變得純凈清澈。水也可以展現出自己的長處，讓火燃燒的更有價值，而不是燃燒了自己，做的卻是無用功，甚至是燒毀衣物。

當然，這期間少不了一個重要人物的加入，這個人就是媽媽的兒子，妻子的丈夫，他充當的是鍋的角色。

水火各司其職，配合合理，鍋處於中間，感受到的就是家庭的溫暖，愛的快樂。水火不容，爭爭吵吵，各自為政，毫無默契，鍋會很遭罪，不是鍋里沒有水，火把鍋燒壞，就是鍋底沒有火，水無法加熱，鍋很寒冷，慢慢地銹跡斑斑。

都是女人，婆婆也是從兒媳一步一步走過來的，今天的兒媳也會是明天的婆婆，按理說，婆媳之間的相處應該是件很容易的事，可實際情況有時難免讓人失望，不融洽的婆媳關係，會把婚姻家庭帶到彎路上去。

《洞》是一部1997年上映的韓國影片，講述的就是因婆媳關係不合，母親誤殺了兒子，最後自己也含恨離世的故事。

一位含辛茹苦的母親，丈夫早逝，與唯一的兒子相依為命了30年，母親對兒子傾注了全部的愛。

改變是從一個年輕漂亮的女孩，嫁到這個家庭開始。

母親對兒子的婚姻並沒有做好心理準備，她覺得兒媳的到來，就是要把兒子從自己身邊奪走，而兒媳又看不慣婆婆對丈夫過分的寵愛，也要奪回自己的丈夫，矛盾由此開始激烈上演。

母親害怕失去兒子，對兒媳百般刁難，故意給兒子的婚姻施加阻力，甚至要除掉兒媳，兒媳身心憔悴，多次爭鬥後，選擇了退出。

兒子知道母親的想法，但也不想失去妻子，就要離開這個家，與妻子在一起生活。

母親害怕的事情還是發生了，她不會讓兒子離開她，就已死相威脅，兒子上前阻止母親的極端做法，在兩人拉扯的過程中，發生了意外，兒子永遠的離開了人間。

沒有哪一個母親，是不愛護兒子的，影片《洞》里的母親，對兒子的愛更為深切而複雜，丈夫早逝，她把對丈夫的愛也給予了兒子，當兒媳踏入家門，她怕失去的，並非只有兒子，而是怕自己的愛失去託付的對象，潛意識裡，她甚至是認為兒媳是在和她爭奪丈夫。

她對兒子的愛，其實愛的並不純粹，有一部分其實她愛的只是自我感受，兒子能夠給她付出的愛，帶來最好的反饋，她對丈夫的懷念，對坎坷經歷的無奈，她都以愛的形式，表達在了兒子的身上。

對這樣的一位母親，雖然她的有些做法，是讓人無法認同的，但不應該去責怪她，理解才是對她的同情與尊重。

如果她的兒媳，能夠知道婆婆的內心世界，換一個角度看待問題，換一種方法與婆婆相處，結果可能大不相同。

人到老年，隨著心理上的變化，情感會變得自私，但這種自私和思想本質上的自私，是完全不同的，只是一種人到老年期的心理轉變，經歷坎坷的老人，表現的會比較突出。

老年人的自私心理，主要有以下幾種表現：

1：吝嗇。

年輕時很慷慨的人，老了會變的吝嗇起來。

人越老，身體機能下降的越快，身體素質不同以往，老年人就會有這樣一種心理，我啥也干不動了，活的沒有價值了，朋友離我遠了，兒女也開始討厭我了，能指望上的就剩下手裡的積蓄了，一旦有了病，有了花錢的地方，向誰開口討要，也不如自己有。

基於這樣的心理，老年人就開始吝嗇。

2：感情上的自私。

人到老年，兒女們卻正是風華正茂、年富力強、打拚事業的年齡，陪在父母身邊的時間，不像以前多了，而這時的老年人，該退休的退休了，子孫也長大了，有了大部分空閑時間，老年人也知道，自己正在走向人生盡頭，對子女家人的情感依賴，就會變得非常強烈。

希望兒女多陪伴他們，希望兒女多關心他們，希望家人團聚是他們最現實的情感需要。

3：愛以自我為中心。

年輕時不愛計較，很開朗大度的人，老了會變得很自我，埋怨他人對自己不夠熱情，抱怨誰都在疏遠冷落他（她），凡事都愛用你的、我的來形容，你的我不要，我的不給你，防備心理有所加強，氣人又好笑，完全就是一個老小孩。

老年人日常接觸到的，與自己有密切關聯的人中間，丈夫或妻子已經相伴了幾十年，知根知底，兒女是自己生養的，也不算太隔心，孫子孫女也與自己是血緣親情，只有兒媳與自己的關係好像是遠了那麼一點，再加上老年人心理情感上的變化，女人性格上的特點，就導致了婆媳關係比較難相處。

愛自己

現實生活中，也並不是所有的婆媳關係都緊張，有的婆媳關係就很和睦，相親相愛，這樣的家庭，讓其他的婆婆們羨慕，也讓其他的兒媳們嫉妒。

其實每一個婆婆，每一個兒媳都是希望婆媳關係能夠親密友好的，不然，也不會去羨慕嫉妒婆媳關係好的家庭了。一家人和和氣氣、彼此關愛，對哪一方都有好處。

想要婆媳和諧相處，兩不隔心，真的把對方看作是一家人，彼此之間有情有愛，婆婆要知道，你的一些情感心理上的變化，對老年人來說是常見的，這一點可以理解，但你不要把兒媳當成外人，這樣做就是你的失誤了。

你可以想一想，因為你情感心理上的變化，是不是導致過夫妻之間的爭吵，比年輕時有所增多呢？是不是子女也曾提醒過你，你和以往不一樣了？

首先把兒媳和子女同等對待起來，和兒媳之間即使有了紛爭，也是正常的，和自己的親生兒女也是會有意見不統一的時候，兒媳不是外人，對她另眼相待，她心裡一定會很難過。

孝順的好兒媳

老年人心理情感上的變化，是很常見的現象，做為婆婆要清楚這一點，在這樣一個時期，要調整好自己的心態，客觀公平的對待兒媳，不要讓情感心理的變化，影響婆媳關係，你是火，兒媳是水，用火把水溫熱，家庭才會溫暖起來。

做為兒媳，也要理解體諒婆婆的身不由己，老年人情感心理上的變化，不是她能夠掌控的，有時的吝嗇、有時的情感自私、有時的自我中心，並不是她對你有意見，也不是她只針對你，不要首先就把自己想成是個外人，把她看成是自己的親生母親，你會有不一樣的感受，會對老年人情感心理的變化，有更深的了解和體會。

兒媳是水，要去滋潤老年人日漸乾枯的情感世界，她的情感飽滿起來，會對家庭幸福做出她該有的奉獻。
