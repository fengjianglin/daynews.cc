我媽就是被我爸寵壞的，人人討厭、自私自利的老公主，現在作我了

2019-12-20 1,458條評論 2,775次閱讀 1529人點贊

看過網友分享：我也很受寵，但是我做家務也上班。公公寵婆婆幾十年寵得老太太跟個三四歲的孩子一樣，連一點生存技能都不會。結果公公剛過五十歲就沒了，留下這個巨嬰天天鬧啊。我覺得一個人的福報是有限數的，不要早早就預支完了……

今日話題：有一個很作的媽媽是什麼體驗？

看看網友是什麼說的：

網友：我都想被寵啊！關鍵是沒有人寵，有老公也是個沒用的，他朋友或者親戚一叫他根本都去，而我生病，呵呵，從來沒理過，連醫院都不願意陪我去。反正是和他非常的不合了！所以他生病我都懶得理他。

網友：被寵沒有錯，但是持寵而嬌就有問題了，特別是那些有女兒的媽媽，老公寵你你要有個度，不要嬌縱，要不然你女兒以後學你一樣嬌縱可不好找對象，畢竟不是所有女人都可以有個好運氣找個沒地位盲目寵的老公。

網友：我覺得小時父母願意寵，父母也是幸福的；成人時伴侶願意寵，伴侶也是幸福的；有子女時兒女願意寵，兒女也是幸福的；最後還能得到孫輩們的寵，那一大家人都幸福的。不幸的是父母寵大的孩子，成年時得不到伴侶的寵愛；得到伴侶寵愛又被子女嫌棄看不慣的；

網友：老公很寵我，10歲的女兒看不慣，對她奶奶告狀說媽媽在家什麼事都不做，都是爸爸做。孩子不懂這是恩愛的家庭，不爭不鬧，相互寵愛。幸福走完茫茫人生路。孩子等你大了也希望有個寵你的人陪你慢慢變老。

那麼，有一個很作的媽媽是什麼體驗？
