兩張圖片拍攝前後有30年，上面這一張拍攝於1984年，當時男孩只有八歲，女孩只有四歲，30年過去了孩子們長大了，年輕的夫妻也漸漸地老了。如同歌詞里唱的，時間都去哪兒了，還沒好好感受年輕就老了，生兒養女 一輩子，滿腦子都是孩子哭了笑了。
