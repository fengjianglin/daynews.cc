無論好壞、無論悲喜，畢竟都過去了。未來需要我們一步步去開拓

2019-11-28 0條評論 22次閱讀 0人點贊

摘要

1、世間，最說不清楚的是緣分，最值得珍惜的也是緣分，誰會真知緣分？一切可遇不可求，一切隨緣就好。生命，最抵擋不住的是時間，最值得期待的也是時間，誰會真惜時間？一切摸不清猜不透，一切自然就好。人生，最深奧最費心的是懂得，最渴望把握的也是懂得，誰會真的懂你？

1、世間，最說不清楚的是緣分，最值得珍惜的也是緣分，誰會真知緣分？一切可遇不可求，一切隨緣就好。生命，最抵擋不住的是時間，最值得期待的也是時間，誰會真惜時間？一切摸不清猜不透，一切自然就好。人生，最深奧最費心的是懂得，最渴望把握的也是懂得，誰會真的懂你？一切如煙又如霧，一切依命就好。

2、一個人，還要敢於正視自我，準確定位。心理專家認為，能客觀、正確地認識自己，並自覺地用理智控制自己，是一個人心理成熟的最高標誌。「天生我材必有用」。既不要妄自菲薄，也不要貶低自己。君可知，人生兩境況：順境與逆境。順境，可用來發展事業;逆境，可用來磨練意志。早安~

3、青春，就應該像是春天裡的蒲公英，即使力氣單薄、個頭又小、還沒有能力長出飛天的翅膀，藉著風力也要吹向遠方；哪怕是飄落在你所不知道的地方，也要去闖一闖未開墾的處女地。這樣，你才會知道世界不再只是一扇好看的玻璃房，你才會看見眼前不再只是一堵堵心的牆。早安!

4、不要去羨慕別人的人生，你看見的，並不是他們經歷的所有。你只羨慕他們的光鮮外表，卻看不到他們為此付出的代價和努力。生活不是一定要有驚天動地的情節，才叫精彩。每個人都在演以自己為主角的偶像劇了，在這場戲裡，你的角色與戲份沒有人能夠取而代之。早安!

5、人，活著，真是不易，尤其在病痛的折磨下活著。有時，不管你承不承受得了，磨難壓在身上，就得扛。誰也幫不了。同情幫不了，眼淚幫不了，憤怒幫不了，只有自己堅強起來，面對所有苦痛折磨，走過風雨熬煎的日子。

6、放棄是一種美麗，懂得放棄的人更美麗。因為只有放棄過去，我們才能展望於未來。對於我們每個人來說，過去的，無論好壞、無論悲喜，畢竟都過去了。然而，未來的路，需要我們一步步去開拓。

7、人只要不失去方向，就不會迷失自己。有能力就不必去氣餒，有價值就不必炫耀。生命的質量取決於每天的心境，通過改變態度可以使得自己經常處於良好的心境狀態。新的一天，早安！

8、幫助下屬，其實就是幫助自己，因為員工們的積極性發揮得愈好，工作就會完成得愈出色，也讓你自己獲得了更多的尊重，樹立了你開明的形象。早安!

9、酒肉朋友一杯濃酒，酩酊之後是一場虛有，知心朋友是一杯清茶，細品之後清香彌久，澆樹澆根。交友交心，拋開名利的紛擾，交得真正的朋友。

10、最強壯的狼也有它脆弱的地方，要想做到最好，僅僅靠自己是不行的。只有不斷地取長補短，才會不斷地進步。早安!

11、人生像一段旅程，一路走來，我們無法預知遇見的風景，只要真心去欣賞，一切都是美不可言。早安，朋友。

12、人生中最艱難的是選擇；工作中最困難的是創新；生活中最痛苦的是自擾；做人中最苦惱的是委屈。早安!
