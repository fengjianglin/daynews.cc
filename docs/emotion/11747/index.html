別人的嘲笑，比你的夢想還珍貴嗎？

2019-12-10 376條評論 1,416次閱讀 467人點贊

「我長大了要當科學家」「我要當飛行員」「我會上太空」……還記得我們小時候這些偉大的志向嗎？那時我們年紀小，頭腦中對這些行業不一定有多少認識。但正因為如此，那時我們才最敢做夢，從不懼怕別人的嘲笑。

可是現在，隨著年齡的增長，我們越來越在意別人的評價，甚至有可能因為別人的一句嘲笑就放棄自己的夢想。

的確，因為太多天真或者不切實際，我們的夢想很可能受到別人的嘲笑，但如果我們因此就放棄自己的夢想，肯定會讓自己抱憾終身。

如果我們有夢想，千萬不要輕易放棄。就算你的夢想十分遠大，那也沒關係。此時我們沒有能力實現它，但並不代表我們永遠沒有能力實現它。我們是在不斷成長的，只要堅持不懈地努力，總有夢想成真的一天。

不僅如此，我們還必須明白，在看待他人的夢想時，每個人都是站在自己的立場上。對我們來說，這個夢想可能遙不可及，但並不代表他人同樣無法實現。

如果有一天，我們的夢想成為現實，你就會感謝很多東西，例如那是努力的自己，那是支持我們的朋友，甚至是那些對我們夢想的嘲笑聲。因為正是這些嘲笑聲，才讓我們有了堅定前行的勇氣。

所以，就算被嘲笑，也不要輕易放棄你的夢想。你唯一需要做的就是，努力讓這份夢想成為現實，全力展現給他人看。因為總有一天你會慶幸，你沒辜負自己的執著和努力。當你再次面對那些嘲笑你的人時，你可以平靜的說一聲，還好我沒放棄，沒有變成那樣庸碌的人。

推薦大家看一部電影《當幸福來敲門》根據真人改編，威爾斯密斯把這個角色演繹的太勵志了。
