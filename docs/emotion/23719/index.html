中國有5800萬留守兒童！再好的物質，也彌補不了孩子情感上的空缺

2019-12-19 65條評論 2,791次閱讀 1863人點贊

文/媽媽補習班，內容原創，歡迎個人轉發分享！

父母似乎是每個孩子的頂樑柱，在幼時的我們心中，父母好像無所不能，可以像孫悟空一樣「七十二變」變出自己想要的玩具，也可以像唐僧一樣對自己嘮叨個沒完。

但不管怎麼說，我們的心中對父母總是依戀、欽佩的，他們用自己的肩膀撐起一家人的重擔，在外面為我們擋風避雨，他們無所不能卻又有所局限。

中國有特別多這樣的父母，尤其是在農村，父母生了孩子將孩子放在老家給爺爺奶奶帶，就踏上了去往他鄉的征途。漫長的一年中，可能孩子可以見到父母的機會只有過年那幾天，其餘時候只能隔著冰冷的電話線互相慰藉，表達思念。

父母出於謀生養家的目的，讓孩子留在家裡無可厚非，一邊是孩子一邊是生活，父母不得不低下頭顱面對現實，「有情飲水飽」這一辭彙太過空泛，完全沒有考慮到現實。

這也是為什麼鄉村的留守兒童那麼多的原因，要是身在城市還好，不說大富大貴，父母在城市中糊口養家還是沒有問題的，但偏偏農村的父母腳踩著這片土地，一點沒有大城市的喧囂和熱鬧，但也少了很多的謀生機會。

在2005年的人口普查數據顯示中，中國農村的留守兒童大約有5800萬人，相比2000年來說，五年的時間，不是留守兒童逐漸消退，反而是不斷的增長。

更為通俗的說法就是每四個家庭中可能就有一家的孩子是留守兒童。

還記得去年在微博上引發熱議的冰花男孩么？2010年出生的王福滿也是一名留守兒童，他以滿頭的冰霜闖入眾人的視線。

王福滿的母親遠走，父親外出打工，他只有和姐姐、奶奶三人相依為命，冬天的時候需要徒步4.5公里起床去上學，小小年紀的他懂事又善良，會幫著奶奶打豬食、餵豬。

他還說他最想去的地方是遙遠的北京，以後想成為一個警察除暴安良，小小的孩子心中藏著的是一個遠大的夢想。

公司之前的王姐家在遙遠的河南，和老公跑到遙遠的杭州打拚生活，兩個兒子也是丟在家中給老人帶，一旦放暑假王姐就會親自回家接孩子來杭州過暑假。

杭州到河南雖說並不遠，現在交通那麼發達，似乎天南地北都可以去，但是對於一個家庭來說，車費、誤工費都要算在裡面，她們要為孩子考慮，遠遠沒有想像中的那麼簡單。

王姐也是，每次回家接孩子自己都是買的晚上的火車票，困頓疲乏趕到家，第二天就把孩子接往杭州，一般很少在家逗留。王姐說：「這樣就可以早點去公司上班，少扣點工資」

因為住的是公司的宿舍，所以孩子接來一家人也只能蝸居在一個不足三十平米的小屋，白天王姐去上班，提前給孩子準備好吃食，讓孩子在家寫作業，有時也會將孩子帶去公司。

暑假很快過去，孩子們都很高興，也都盼望著下一個假期再來。

在我看來，王姐的孩子已經算是留守兒童中比較幸運的了，至少他們還能在短暫的假期和父母相逢，培養缺失的親情，但是大部分的留守兒童並沒有這個機會，他們見到父母的機會少之又少，每年只能眼巴巴的盼望著春節的來臨。

但是還有很多父母連過年都不回家，前段時間看了一個採訪，一位民工大哥頭年將所有的錢都寄給家裡，忘了給自己留車票錢，無奈只能留在城市，每天在救助中心領免費的泡麵吃。

那些泡麵無色也無味，但他吃的很開心，認為自己又給孩子省了一筆讀書錢。之後的好幾年他都是這麼過來，不是不想家，不是不能回家，只是在經過深思熟慮之後他們放棄了回家的機會。

一年又一年，一月又一月，孩子已經長大，她們從小缺失父母的愛，長大之後也不再期盼父母的愛，甚至很多留守的孩子還與父母有很嚴重的隔閡。父母對於他們來說，或許只是生了他們，每個月固定給錢的人，之間的親情淡薄的依稀看不見。

而且帶來的影響遠不止如此

一：留守兒童很難有所成就

雖說現在隔代親很常見，但是父母給孩子的愛是爺爺奶奶所不能給予的，父母能夠教給孩子的人生道理也是不可替代的。

特別是在鄉村的留守兒童，本來就受到資源和抵禦的限制，教學條件有限，而且父母還不在身邊管教，爺爺奶奶不知道該怎麼管教，其中稍有差錯就會影響孩子的一輩子，孩子會產生厭學的心理，覺得自己根本得不到關注。

二：和父母親情淡薄，心理易產生問題

儘管留守兒童心裡明確的知道父母是為了家、為了自己才遠走他鄉，但理性上能體諒是一回事，感性上又不是這麼一回事。

在他們受到挫折、委屈的時候無法尋求父母的意見，就連想在父母身邊鬧一鬧、得到一點關注的機會都沒有。久而久之，任何委屈他們也只能自己吞下，有苦自己吃，有傷自己舔，對父母有怨懟，也從不抱希望在他們身上。

嚴重一些的，還非常叛逆，對學習完全失去了興趣，對自己的人生也沒有任何興趣。

理智驅使孩子諒解，感性教唆孩子怨恨，雙重摺磨讓孩子心理脆弱不堪。道理似乎都懂，但家家有本難念的經，生活所迫，誰又能輕鬆好過呢？

但是父母還是可以有所作為的

一：多給孩子打視頻電話

父母可以給老人買智能手機，下班的時候打視頻電話和孩子聯繫，可以維持在每天一次或者兩天一次，不必在意時間的長短，但是必須要有這樣的「儀式感」。

哪怕電話打過去什麼都不說，只讓孩子看看父母的樣子也好，至少心中也能有所慰藉，孩子們心中也會想，我的爸爸媽媽是愛我的。

在視頻中父母可以盡情的噓寒問暖，了解孩子的近況，學習有沒有進步、生活中有沒有碰到什麼小問題，或者有沒有什麼煩心事和悄悄話要對自己說。

二：多問問孩子喜歡什麼，投其所好

可以問問爺爺奶奶孩子喜歡什麼，或者在電話中套話，時不時給孩子一點小驚喜。

了解孩子的興趣愛好，家長也有更多的話和孩子聊，現在淘寶購物也很發達，要是孩子喜歡一些小玩意，不妨給她們偷偷買下來，孩子收到會很高興，而且可以減少父母不在身邊的缺失感，一舉兩得。

總結

留守兒童不可避免，也許只有哪一天鄉村也同樣發達，這樣的情況才能有所改變，但是父母還有很多事情可以做，幫助孩子走出沒有父母在身邊的陰霾和困惑。
