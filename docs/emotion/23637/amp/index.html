我終於理解了“久病床前無孝子”背後的心酸和無奈

天天要聞
2019-12-20

大家好，這裡是《婚姻百態》

「看婚姻百態，品百味人生」

我今年30歲，我至今未婚，家裡的情況也不是特別好，家裡只有一個卧床5年的媽媽，和一直照顧她的爸爸。

在我媽沒有生病之前，我會認為我是一個絕對的孝子，會讓父母有個安穩的晚年。但我的想法在5年前戛然而止，因為那年我媽生病了，腦血栓導致了癱瘓，今後估計很難能下床了，一病就病了五年。

剛剛得知媽生病的時候，我有一份十分得心應手的工作，薪資不錯，在單位和同事關係也非常好。聽到媽的病狀，就向單位請了假，就回家看望了。可是媽的病並不是一下子就能好得了的，我只能一次次的向單位再請假延長，終於有一天，公司說如果我再不回去上班，就要考慮換人了。

我陷入了兩難的決定，一方面始終不放心媽媽，她的病症並不安穩，經常半夜都需要幫忙，雖然有爸爸一直在照顧，可是爸爸也已經50歲了，體力和精力跟不上。另一方面我已經有半個月沒有回單位上班了，公司也不可能為我一個人等這麼久。媽媽病狀好了些，在他們的一再催促之下，我在公司的最後期限內回去了。走的時候爸爸說「沒事，你媽有我。」

可是我剛回去工作沒幾天，突然一天在上班的時候接到了爸爸的電話，一接通，那頭就傳來了急促的聲音「快，你媽進ICU了。」我一聽頓時就慌了神，連夜買了車票奔了回去，也不好意思再向單位請假，索性辭了職。

媽看見我第一句話就說「兒子不好意思，又給你添麻煩了。」聽見母親的話，我一時強忍淚水，「媽，你安心養病，我沒事的。」這一次過後，我擔心母親又出現什麼意外，爸爸一個人是肯定照顧不過來，就回到父母的城市，跟他們住在一起，就近找了一份工作。

所幸後來媽媽病症穩定住了，但是爸爸之前做生意賺的那些錢，也用得差不多了，家裡的老底子基本見光了。一家人還要生活，爸爸重新做生意，我也換了一份工資高點，但是辛苦的工作。

但是人總有疲倦時，爸爸經常把家裡整的亂七八糟，臟衣服臟褲子亂扔一地，臭氣熏天。我工作了一天本來就很疲憊，開門一看卻是這個亂糟糟的樣子，心裡也窩火，就這樣也跟爸爸吵過好幾次。有一次吵得厲害了，媽媽就在房間裡面哭，我們兩個大男人只有悔恨無比，強忍眼淚的去安撫她。

這些年裡我也交往過幾個女友，但是她們一聽我家裡有個癱瘓的母親，都沒有再說什麼，直接走掉了。所以到現在，我還單著。

有時候閑來無事，刷刷朋友圈，看著以前的同事，朋友們都生活幸福自在，風生水起。再看看自己當下的處境，也有一些心有不甘。夜深人靜的時候我會想：如果，如果媽沒有生病，我可能已經有了自己的家庭，自己的事業。我精神上遭受的折磨和無奈，不知道該向誰訴說；我犧牲的這一切又該向誰訴說。

都說久病床前無孝子，這幾個字背後是多少心酸和無奈，我現在算是深刻的理解了這句話的含義。這麼多年，我的精神被拖垮了，鬥志被磨滅了，愛情也丟失了；我犧牲了太多，也受了太多委屈，但是看著媽媽日漸康復的身體，我這麼多年受的委屈和辛酸也終於感覺到了一絲安慰。
