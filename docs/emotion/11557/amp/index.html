辦公室勾心鬥角好可怕，不善於“斗”的人該怎麼生存？

天天要聞
2019-12-13

有網友說：辦公室勾心鬥角好可怕，不善於「斗」的人該怎麼生存？

有人的地方就有江湖，勾心鬥角是很難迴避的，千萬不要覺得換一個地方就一定很和諧，但凡跳過幾次槽的人都有體會，你只不過從一個火坑跳到了另一個火坑裡去！

面對複雜的人際關係，你可以不善於」斗「，但是不能失去」生存「的能力！

下面，就這個話題，我個人再說三點看法，僅供大家參考。

一、勾心鬥角哪裡都可能存在，你要做的不是逃避，而是適應

職場上的勾心鬥角在哪裡都可能會存在，除非這公司就你一個人，否則有人的地方就會有利益和矛盾，你想要一個完全理想化、沒有勾心鬥角的辦公室太困難了！或者乾脆直白點說：那不存在。

所以不善」斗「怎麼辦？逃避可以嗎？

並不是完全不行，有人就選擇跳槽，然而發現到了另一個辦公室依然勾心鬥角。

還有人就裸辭在家了，這樣確實也隔離了職場環境，可以不參與勾心鬥角，但是一直在家不工作能生存的又有多少人？

想明白這個點，你就發現原來逃避這條路太難了，還是得面對現實的職場環境，要靠工作賺錢養家糊口的，就得學會適應，想辦法讓自己在勾心鬥角的環境中生存下去。

二、不善於勾心鬥角的人，最好不要直接參与」鬥爭「

那麼對不善於勾心鬥角的人來說，是不是應該學會去如何」鬥爭「呢？

恰恰相反，我倒是建議你不要直接去和人家PK。為啥呢？我說幾點理由你聽聽：

第一，」勾心鬥角「這件事本身就有情商、先天性等因素在裡面，你要從一個不善於的人轉變成很善於的人，難度是非常大的，與其花時間在這上面，不如盡量保持中立，不選邊站參與鬥爭去。

第二，一旦直接參与內部的」派系鬥爭「，你這個本就不擅長之人很可能就成了別人的炮灰、犧牲品。

曾經我有個朋友，就是一入職後就被捲入了辦公室政治鬥爭，然後兩邊鬧得很激烈，領導最後是先拿他來開刀，沒辦法，誰讓你資格最嫩，好欺負呢？

第三，」勾心鬥角「這件事太累了，明明不擅長，卻非要搭進去，對自己的精力消耗太大，一方面影響自己的工作，另一方面還得罪人，心理層面會有很大的負能量，得不償失。

我所見過最聰明的一些人，就是」大智若愚「，玩」平衡「戰術，不選邊站。

三、想要在」勾心鬥角「的環境里生存下去，核心是保持自己的職場競爭力

在任何一個企業，無論勾心鬥角的程度怎麼樣，最不擔心自己安全的還是那些有核心競爭力的人員。

舉個例子，如果你是一家公司的技術骨幹，掌握了關鍵技術能力，那麼其他人想要勾心鬥角地排擠你，也沒那麼容易，因為你被排擠掉了，公司的損失就非常大，一來那些想排擠你的人不敢輕易行動，二來公司有領導肯定也會護著你，畢竟你身上都掌握了公司的」命脈「，你被」斗「倒了，公司的希望都要沒了！

再退一萬步說，如果你真的不幸成為了公司里勾心鬥角的犧牲對象，那麼你也不必有任何擔心，因為其他競爭對手會搶著來挖你，實際上真正損失的並不是你，而是公司，包括那些勾心鬥角之人。

但要是你沒有啥核心競爭力，總有比你更善於玩弄心機或者權術之人，今天可以保住自己的地方，不代表以後就可以一直保得住。
