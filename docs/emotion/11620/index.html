任何事情，你只有用心，才能做到最好，共勉，早安

2019-12-12 172條評論 1,818次閱讀 901人點贊

1、對一個聰明人來說，每一天都是一個新的開始，你當然可以謀劃自己的理想和前程，甚至可以放眼世界尋找更好的機會，但不要忘了我們首先得為「今天的牛奶努力」，在每個「今天」執著、塌實地走好每一步，展現在你眼前的將是滿園春色。

2、浮生若茶。茶葉因為沉浮才釋放了本身深蘊的清香，而生命，也只有遭遇一次次挫折和坎坷，才會激發出人生那脈脈幽香。那些不經風雨的人，就像溫水沏的茶葉，只在生活表面漂浮，根本浸泡不出生命的芳香。生命中，經歷一次磨難，就是鍛煉自己的意志；挺過一次災難，就是增加自己的力量。

3、人就這麼一生，沒有來世。所以讓我們從微笑開始！人活一輩子,開心最重要。擁有健康的體魄，在快樂的心境中做自己喜歡做的事情，安全地實現自身價值，是人生最大的幸福好景不常在,好花不長開;人生短暫,好好地去珍惜它，善待它，把握它吧！珍惜你身邊的每個人。珍惜！珍重！

4、逆境，是上帝幫你淘汰競爭者的地方。要知道，你不好受，別人也不好受，你堅持不下去了，別人也一樣，千萬不要告訴別人你堅持不住了，那隻能讓別人獲得堅持的信心，讓競爭者看著你微笑的面孔，失去信心，退出比賽。勝利，屬於那些有耐心的人。

5、年輕人現在誰都知道錢是有生命的，機會是這麼多，條件是這麼好，你可以拿錢去按揭，做今天的事，花明天的錢；也是可以拿錢去投資，去『充電』。錢只有在它流通的過程中才是錢，否則只是一沓世界上質量最好的廢紙。

6、花是淡的香，心是淡的真。花香太濃易熏人，無味易遭疏遠，淡淡的香味反而最受歡迎，最受歡迎的花香才是真香。人心太真易傷懷，太假易遭人棄，若即若離保持著大家都能認可的距離，才是最長久的相依，最長久的才是最真的。看人看事，太較真必然看破，看破的結局傷己又傷人。

7、天空，不是每天都放晴；生活，不是時時都歡喜。向日葵看不到太陽也會開放，當它經歷陽光下更顯燦爛；生活看不到希望也要堅持，人生嘗盡了百味方覺甜更甜。所以，偶爾隱匿的陽光，再來時更溫暖；撲朔迷離的希望，出現時才令人驚喜。保持樂觀的心態，走著走著幸福就來了。

8、每個人，都是一道獨特的風景。你站在橋上看風景時，看風景的人在樓上看你。不必艷羨他人，你該學會相信自己，再學會欣賞自己，試著把自己最亮麗的一面找出來，並呈現在陽光下。

9、命運的所有饋贈，都暗自標好了價格，路要一步一步的走，苦要一口一口的吃，這個世界上最難走的路就是所謂的捷徑。厚積而薄發，請相信，這才是通往成功最快的路。

10、用心，就是用負責、務實的精神去做好每一件事；就是不放過工作中的任何一個細節，並能看透細節背後潛在的問題；就是要讓自己比別人做得更好。任何事情，你只有用心，才能做到最好。 早安！更多的精彩情感，哲理，美文；敬請關注微信公眾號：心成一片
