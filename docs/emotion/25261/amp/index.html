導語：大家好，現在晚上寵女的小視頻可以說是鋪天蓋地，相比之下兒子的境遇就有點不妙了。為此很多人感嘆「重男輕女」這種現象終於消失了，但是事實真的如此嗎？或許大家經常在生活中聽到這樣的說法：生男生女都一樣！但是真的一樣嗎？最近網上有一組照片引起了大家的熱議。

爺爺喜得雙胞胎孫女，表情卻異常凝重

醫院是一個見證人生百態的地方，不同的家庭，在這裡上演著不同的故事。最近網上就有一組產科病房的照片熱度飆升，有兩位家長的表情呈現在了大家的面前。有位爸爸看著剛出生的兒子滿臉笑容，內心的喜悅與激動完全沒有被掩飾。鏡頭一轉，同病房的一位爺爺進入了大家的視野，不過看這位爺爺的表情完全沒有一絲一毫的喜悅。

原來這位爺爺的兒媳生的是雙胞胎女兒，據說家中的老大也是女兒，所以這位爺爺的表情就有些凝重了。或許在這位爺爺看來自家沒有男孩，這就是要「絕戶」的節奏，不得不說這位爺爺的表情太過真實。這兩人的表情對比，讓大家感觸很深，網友們在留言區里的點評更是發人深思。

網友熱評

深深的藍：說男孩女孩都一樣的一般就只有兩種人，第一種是未經人事，不諳世事的年輕人；另一種就是只有女兒，鴨子死了嘴硬的成年人。

胡蘿蔔小姐：這樣的現象已經比較少見了，大部分人都能平靜看待生男生女了。

大笨笨：傳宗接代依然是很多老人看重的東西，年輕人就相對好一些了。

重男輕女不可取，家長要正確看待生男生女這件事

小編覺得有位網友的點評是非常到位的，重男輕女這樣的現象現在大多出現在老人身上，而年輕人都已經比較看得開了。所以大家在網上見到的寵女視頻，大多是年輕父母拍攝的。還是那句話，重男輕女這樣的行為是不可取的，作為家長更應該正確看待生男生女這件事，不管是生男還是生女，都是值得高興的事情。如果執著於此，往往會引發家庭矛盾，甚至一發不可收拾，後患無窮。
