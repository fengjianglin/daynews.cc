也有人說：「有孩子是個意外。」

無論是什麼原因，是否生下孩子從來不是孩子的選擇，而是成年人的決定。既然選擇了生下孩子就應該善待他，而不是讓孩子小小年紀就被冷血的對待。

11個月嬰兒被遺棄樓下，邊哭邊爬

今天刷到一條舊新聞，看到的畫面讓人觸目驚心，視頻里一個嬰兒在室外的土地上邊哭邊向前爬行，仰著頭不會說話，只能「啊啊啊」的哭嚷。

嬰兒爬行的方向遠遠的是一個坐在摩托車上的男子，面對嬰兒的哭嚷和艱難的爬行，表現得是非冷漠，對嬰兒視而不見，只低頭玩著自己的手機。

當嬰兒爬到跟前男子嫌棄的避開，坐在石階上。嬰兒因為在土地上爬行，滿身泥濘，卻依舊像男子爬去。

看畫面，男子應該是嬰兒熟悉的人，可是面對孩子的哭求卻視若無睹，毫不心疼。

這一幕被附近居民發現，立刻報警。

警方介入了解到，嬰兒只有11個月大，男子是孩子的父親，孩子的父母是未婚生子，兩人因孩子的撫養權未能達成共識，關於孩子的撫養權，雙方都是相互推諉的態度。

男子此前告知女方會將孩子送來，若她不領孩子就 將孩子棄在樓下，女方也是心大，只當男方是在嚇唬人，並未留在家裡等，而是出門了。這才有了這令人心酸的一幕。

網友的故事：

@葉子260：「女兒一歲的時候發現老公外遇，二位對方懷孕，我帶著女兒離婚，後面認識現在的老公，在一起第八年耳朵時候生了老二，老二一歲的時候老公生意失敗，簽下很多錢，被人誣告，坐牢，現在老二三歲，老大十三歲，我一個人打工養他們，也從沒想過放棄他們，老大老二都很懂事，天天會告訴我，他們愛我，其實我也很愛他們。」

人們常說「為母則剛」，其實也是孩子們給個媽媽們勇敢起來的勇氣，這個世界上不負責任的父母還是少數。

但仔細追究還是那些成年人，從小的教育出現了問題，異於常人的成長經歷，讓他們沒有學會責任和品德，才會出現那些「自私」的家長。

@番茄雞蛋：「我是沒有孩子就沒有自己的母親，對不起，可能不是那種辣媽或者成功人士。但孩子就是我得全部，為了孩子也要經營好自己的婚姻。成年人離婚都能過，孩子卻只有一次童年，他需要我們父母的呵護，才能健康長大，心理健康，身體健康。」

這位媽媽的一句話我非常贊同「婚姻失敗可以離婚從來，但孩子的童年卻只有一次」。

但並不是說，在不幸的婚姻里要為了孩子容忍，那樣孩子「只有一次」的童年會不會幸福，要讓孩子童年幸福首先要「婚姻幸福」而不是「婚姻完整」。

@宿命30763：「在此感謝我的父親，在我14歲那年離婚24歲那先才再娶，好多人給我爸做介紹，他都是，等我女兒初中畢業再講，等我女兒高中畢業再講，等我女兒大學畢業再講。而我媽，因為女人帶著孩子不好嫁把我留給了我爸，在離婚第二年又再婚了，想回到我爸身邊，放出風聲，除非我爺爺跪倒她面前她就回我爸身邊，結果就是我爺爺都老了很多年了，我爸也找了人，她還是離婚沒結婚證的跟著那邊的男人過。」

父愛與母愛在孩子成長的天平兩側，都是同等的重要，這位爸爸為了女兒的成長中不再受到自己婚姻的干擾，讓孩子再次受到傷害，選擇默默等待女兒長大成人。網友的成長中並不缺少愛。

賢爸小結：

要孩子從來都應該是一個慎重的決定，孩子出生後是一個鮮活的生命，他能感受到身旁的愛與冷漠，也會感覺到痛。

既然選擇生下他，就要對他負責，為他的生命、成長、教育負責。
