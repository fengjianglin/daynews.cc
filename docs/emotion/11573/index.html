優秀的女兒上了高中，他們一家卻陷入了悲傷……

2019-12-13 716條評論 2,463次閱讀 1276人點贊

在旁人看來，葉子一家可真是「人生贏家」：爸爸媽媽事業有成，女兒懂事孝順，一家人關係融洽。今年，女兒還考進了市裡最好的高中的實驗班。

但是自從孩子上了高中後，葉子明顯憔悴了不少，整天唉聲嘆氣的，一問才知道，原來葉子的女兒得了焦慮症，已經辦了休學手續，準備好好治療了。

葉子紅著眼睛說，女兒在初中時很用功，中考考得不錯，為了讓女兒放鬆一下，暑假沒讓她學習，帶她出國玩了一圈。回來後高中第一次考試家裡人就傻眼了——女兒居然是班上倒數第二！老師上課講題都是按照競賽的難度來講，沒上過銜接班的女兒連最基礎的題都不懂，根本跟不上老師的節奏。

女兒最開始天天哭，整夜失眠，後來到了完全聽不進課的程度，整個人也變得焦躁不安。去醫院看了醫生，醫生診斷她得了中度焦慮症。沒辦法，只能先休學養病……

當代社會壓力大， 家長們不願意讓孩子落後於他人，殊不知這樣會給孩子更大的壓力。近年來，學生患抑鬱症或焦慮症、甚至輕生竟然已經到了「屢見不鮮」的地步。

壓力是一方面，大部分家庭里忽視對孩子的人格培養又是另一方面。許多家長只告訴孩子要努力學習，卻沒有幫他們養成堅強的人格。

如果說人的成長是一棵樹，那麼人格就是樹的根，有紮實的根基，大叔才能枝繁葉茂。人格的培養是一個潛移默化的過程，所以有些家長就認為孩子「長大了就懂了」，不用刻意注意什麼。但就是這份「不用在意」，可能就是悲劇的前兆……

孩子小的時候，性格上難免會有一些缺點，膽小、脆弱、太依賴大人……家長應該怎麼做才能幫助孩子從小培養健全的人格呢？

培養開朗人格

不要給孩子太大的壓力

@小花：我家寶寶依賴性特彆強，特別粘我，出門也總是不敢說話，很膽小，在幼兒園被欺負了也不敢跟老師說，我要怎麼來改變她的這種性格呀？

有些孩子天生比較內向，不如其他孩子開朗，因此也會更加依賴自己熟悉的人。家長可以想辦法調整一下孩子的內向性格，但切忌急於求成給孩子太大壓力，逼迫孩子在許多人面前唱歌跳舞等等行為都會適得其反。

中國兒童中心學前教育部部長楊彩霞博士指出，家長可以先跟老師溝通，讓老師在幼兒園時多給孩子一些在眾人面前的發言機會，如上課回答小問題等等，幫助孩子漸漸地適應，同時增強自信心。

有的時候，孩子並不是內向，而是不知道如何表達自己的情緒。孩子看的東西越多，越能理解到自己腦海里有的情緒是什麼，就越擅長表達。

培養共情能力

學會換位思考

@向日葵：我家孩子有幾個玩得很好的小夥伴，有次有個小朋友跑著跑著摔倒了，他不但沒去扶，反而在旁邊笑人家笨，這簡直是塑料友誼啊。

相對於其它品質來說，共情能力是一個比較少被提及的點。具體來說，共情能力指的是一種能設身處地體驗他人處境，從而達到感受和理解他人情感的能力。

我們有時可以看到小孩會做出一些比較殘忍的舉動：捅螞蟻窩、使勁扯貓尾巴等等，這些其實都是缺少共情能力的表現，他們不知道這樣做會給別人造成傷害。

很多人長大後，共情能力也會漸漸地變強，但有小部分人的共情能力並不會隨著年齡的增長而增長，這樣的人就很有可能做出傷害他人的事情。

如果發現孩子有類似於別人摔倒後嘲笑別人的行為，楊彩霞博士也支了招：家長也不要生氣，可以跟孩子好好地聊一下，問他「如果摔倒的人是你，別人還在笑你，你心裡難不難受」，幫孩子學會換位思考。

培養抗挫能力

參加體育競賽

@青青：我家孩子可嬌氣了，平時在幼兒園小朋友不小心碰一下就哭，搭積木搭兩次沒搭好也會哭，怎麼能讓孩子堅強一點？

小朋友年紀小，遇到的挫折比較少，有時一些小小的挫折就能讓他們情緒變得很不穩定。對付這種情況，楊彩霞博士提到了一個好辦法，那就是讓孩子參加體育競賽，跑步、啦啦操、游泳等等都可以。

對於孩子來說，體育不僅可以強身健體，還能讓他們有初步的規則和競爭意識。體育競賽難免會有失敗，失敗之後，家長一定要及時給孩子鼓勵和支持，讓孩子有「天塌下來爸媽頂著」的踏實感，減少對失敗的恐懼，慢慢地，孩子的抗挫能力就會變得更強。

培養安全感

對症下藥

@小橘子爸爸：我家女兒很怕黑，好像特別缺乏安全感，不管幹什麼都讓人陪著，搞得大人什麼都做不了。

家長可以想想，孩子害怕的到底是什麼？許多孩子都怕黑，怕的可能並不是黑暗本身，而是會在黑暗中出現的「鬼」或者「怪物」。家長們可以放一些布偶在孩子的床前，減緩孩子的焦慮；如果孩子對鬼怪和黑暗的害怕已經很嚴重了，就盡量不要讓孩子閱讀或觀看有鬼怪的故事和視頻。

怕一個人獨處，可能有兩種情況：一，對孩子的陪伴太少了，因此孩子比同齡人更渴望親人在身邊；二，家長包辦了孩子的一切事務，孩子沒有自理能力。孩子小的時候多陪孩子，跟孩子多互動、做親子遊戲，跟孩子建立依戀關係，同時在保證孩子安全的情況下給孩子最大的探索自由，培養孩子的自信，孩子自然就會有安全感。

「幸運的人一生都被童年治癒」，家長們要珍惜孩子的童年時光，幫助孩子養成健康的人格，這對孩子的一生都至關重要！
