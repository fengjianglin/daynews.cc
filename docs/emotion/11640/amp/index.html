客廳內，原本歡聲笑語的一家人，頓時愣住了，兒媳這是怎麼了？

難道…

以上故事情節，來自韓國近期爭議極大的電影：《82年生的金智英》。

韓國，首爾。

金智英在結婚前，原本是個都市白領，自女兒出生後，她就離開公司，在家做起全職寶媽！

不知不覺，女兒一晃2歲，智英雖然很愛女兒，但總覺得人生似乎少了點什麼。

一次寶媽聚會中，幾個人瞎聊，其中一個寶媽戲謔道：想我也是一個堂堂首爾大學的理科生,當初那麼努力上學，不知是為的什麼？所學的知識，難道只是教兒子做3+3=6！

智英聽後，甚有同感，自己當年也是那麼努力的上學，曾經也以為自己會在事業上有所成就，但隨著孩子的出生，當初還真不如不上學！

… …

智英聽後，心理異常的不舒服：

難道在家帶孩子，就是偷懶？

難道不出去工作，就是我願意的？

難道你們以為，我願意呆在家裡無所事事？

這部《82年生的金智英》上映於2019年，改編自同名小說，目前豆瓣評分8.7.

值得一提的是，這部電影從原著小說出版時，就在韓國引起兩性之間的巨大矛盾。

口碑兩極分化，男生覺得這部電影太過於女權，打出的評分僅僅有2.59分；女生們則覺得這部電影拍出了她們的心聲，是一部真實的好作品，紛紛打出9.51的高分。

Red Velvet女團成員Lrene因為在公開場合談及這本書，便遭到男粉焚燒照片抗議；

女演員鄭裕美，因為飾演該片的女主角，也被大量男粉絲攻擊：鄭裕美你現在也可以滾了，拜拜！

一些激進分子，在這部電影開拍時，甚至還跑去了青瓦台，請願政府停拍！

然而，正是這樣一部被腦路閉環的人抵制，所謂「禁書」改編的電影，在韓國上映時，當天票房就超越了正如日中天的《小丑》，拿下了票房第一。

猴哥作為一個男性，這個周末懷揣著好奇觀看了這部電影，想看看它究竟拍得有多麼偏激，才造成如此大的兩性矛盾，還鬧到總統府，驚動文在寅！

然而看完後，我不理解了，不懂得韓國人的包容度什麼時候變得如此不堪一擊？

這裡不單單是說男人的包容度，也包含女人。

因為這部電影，它講述的故事非常客觀，就是現實中的生活：

不理解，男人抵制什麼？

也不理解，女人激動些什麼？

活得異常抑鬱，精神也出現問題！

朋友老婆由衷地說：真的好羨慕金智英！

另外我這個朋友的老婆學歷也不低，湖南大學研究生，在上海活得跟狗似的！

這是我們這一代人，最普遍的現象，羨慕韓國人不用做房奴！

老公不理解，就不是一個懂老婆的好男人，可以選擇離婚！

一個家庭，並不是男尊女卑，也並不是女尊男卑，兩個人既然搭伴過日子，首要的責任，就是要推動這個家庭的良性發展，讓家向好的方向走，要有責任意識，所以說，金智英的老公真的沒有錯！
