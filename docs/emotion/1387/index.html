被PUA操控的女性，原來都有一個相似的童年... ...

2019-12-03 0條評論 24次閱讀 0人點贊

摘要

最近關於PUA的新聞層出不窮，每次看到，都讓人感覺沉重和心痛。

PUA原意是「搭訕藝術家」，在2008年進入中國，後來卻慢慢發展成詐騙形式的不良PUA。這一類人群，經過專業培訓後，專門實施情感詐騙，越來越多的女性成為其「獵物」。

不僅欺騙感情，更有些人直接誘導對方自殺，來獲得變態的「成就感「。

圖片來自節目《和陌生人說話》

曾有調查顯示，那些上當受騙的女性，大都對他人的虛情假意和甜言蜜語沒有分辨力，更無力抵抗精心設置的套路，輕易就被對方操控。

簡單說，就是自我價值感很低。

而穩定的價值感，並非先天擁有，需要父母從小培養，這將會影響孩子的一生。

很多父母一方面想要培養一個價值感穩定的孩子，一方面又不知所措。

價值感並不是單一的某種感覺，有兩點非常重要：配得感和滿足感。

圖片來源於網路

一、「我覺得我配不上很好的樣子」

有一個來訪者的故事，讓我印象深刻。

她在32歲的時候，才第一次發現自己有很深的不配得感。而那個時候，她已經被這個信念控制了太多年。

她本身有些胖，長得不醜。一直致力於減肥，但是往往在小有成效的時候就放棄了。她以前一直覺得奇怪，似乎每到那個狀態的時候，都有一股力量在拉著她，不讓她繼續瘦下去。同時，她會又開始狂吃，再吃回原來的體重，就會有一種莫名其妙的心安。

這個困擾折磨了她十幾年，突破點發生在前不久的一次體驗課上，她看到了兒時的一些情景；

在她八歲的時候，有一次她穿了一件自己很喜歡的衣服，但衣服稍微有點短，只要一彎腰，後背就會暴露出來一點。

她爸爸看見後，嚴厲地訓斥了她：「你穿成這樣，還像一個女孩子嗎？」並且命令她迅速把衣服換掉。

在她進入青春期，身邊的有些朋友開始化妝打扮自己。她有一天也畫了眉毛和口紅，又被爸爸訓斥：「這像什麼樣子？趕快去洗了！」

於是，她就有了一個認知：我不能太漂亮。

這個認知在某種程度上影響甚至塑造了她的生活：

她的身體一直瘦不下來，因為這樣的身材符合了「我不能漂亮」的信念；

和朋友一起去商店買衣服，看到好看的衣服不敢試穿，如果朋友說好看，她就會買；長大後，遇見喜歡的男人，習慣後退，覺得這麼好的人，她配不上；

很多事情就在最後快要大功告成的時候，她卻放棄了最後一步的努力；

… …

圖片來源於網路

這就是深深的不配得感，雖然她的意識上很渴望自己更好，但潛意識裡卻與更好的自己一次次失之交臂。

露易絲.海在《生命的重建》中曾說：

「我們一遍又一遍反覆經歷的那些事，都源於我們自身的某種需求，這總與我們的一些觀念保持一致。一定有某種東西存在於我們身上，使我們需要變胖，需要糟糕的人際關係，以及失敗、煙癮、憤怒、貧窮、虐待，還有我們身上的其他問題。」

而不配得感，就是這種需求的根源之一。

——「我認為自己配不上，我的好壞由別人說了算。」

這樣的孩子長大後，就會創造和內心信念一致的外在現實：差強人意的婚姻，卑微討好的人際關係，內在不斷的壓抑和衝突，以及拚命努力想要得到認可的恐懼。

圖片來源於網路

二、「孩子不停地要東西，難道都要滿足嗎？」

尊重與放縱，似乎很難平衡。

就像前一段時間在課堂上，一個媽媽說：「孩子不停地要玩具，今天買了，明天還要買，那是不是都要滿足，如果不給他買，就會讓孩子感覺到不被愛呢？」

話音剛落，台下就有好幾個媽媽附和：「是啊是啊，我家孩子也是這樣，我不敢不買，但又不知道怎麼拒絕？」

的確，我們對孩子愛的意識提升了，但又很容易滑向另一個極端：放縱。

圖片來源於網路

孩子有各種各樣的需求，都要滿足嗎？關於這一點，惠文老師講過一個很有趣的例子：

她女兒小的時候特別喜歡蛋糕，也知道要過生日才能吃，一年只過一次生日。

惠文老師只要帶著女兒外出，看到喜歡的蛋糕店就一起進去。女兒就站在櫃檯前，滿心歡喜地看著那個蛋糕，告訴服務員：「這個幫我留到七歲生日時候買，那個留到八歲生日再吃。」

就這樣一直排到四十多了歲，「預定」了四十多個蛋糕，而且每個蛋糕都記在了自己的小本本上。

有時候，孩子就站在店裡看著那些蛋糕，一看就是半個多小時。惠文老師就陪著她、等待她，直到孩子把喜歡的蛋糕看夠了，「預定」好了，再帶著她心滿意足地離開。

孩子雖然沒有及時吃到蛋糕，但惠文老師用允許和陪伴告訴孩子：你可以喜歡蛋糕，你可以享受這種喜歡的感覺。孩子從中獲得了極大的滿足感。

現在女兒上中學了，早已經不是那個愛吃蛋糕的小女孩兒了。但她知道，她的喜好是被允許的，她的感覺是被及時滿足的。

圖片來源於網路

所以，真正的滿足，不是我給你買一堆蛋糕，來讓你開心。而是要滿足孩子慾望背後的渴求：

我的想法和喜好是重要的，我是被理解的。

大楊老師曾經說，外在的欲求永遠無法被滿足，孩子真正需要的，是內在感受和精神的滿足。

在日常生活中，有多少父母陷入無法滿足孩子的焦慮中呢？

圖片來源於網路

三、低價值感來源於父母對孩子的催眠

配得感和滿足感，是衡量價值感的兩個很重要的面向。

孩子的低價值感，通常是由於父母把孩子的價值和孩子的外在划上了等號，在養育過程中，不斷催眠。

父母把孩子的評價權抓在了自己手上，慢慢的孩子就會認定，我的價值和我的行為有關，和別人的評價有關，所以就慢慢開始依附於外界的評價。

有多少孩子深陷這種困擾里？

遇到困難，不敢尋求幫助，更不敢享受別人的主動幫忙，認為自己不值得；

有好的展現自己的機會，也沒有力量爭取，覺得和自己無關；

特別在乎他人的評價，自己努力的目的，就是為了得到他人的認可；

如果一旦被別人質疑，就會自我攻擊，自信心就會瞬間崩塌；

甚至有的孩子，明明平時學習成績還好，卻在考試時經常發揮失常。

… …

我們常說的「玻璃心」、「自卑」、「討好」等，都是低價值感的表現：

「我必須通過你的確認，才能感受到自己是好的，是優秀的」

「沒有你，我就是不存在的。」

不穩定的價值感帶來個體的虛無和無意義，讓一個人無法發展出獨立的自信和自我。成年以後，要麼終其一生都在追求他人的認可，要麼自暴自棄隨波逐流。

圖片來源於網路

四、如何培養孩子穩定的價值感？

價值感的來源，簡單說，就是孩子在和父母的互動中，不斷反覆地確認我是被愛的。

這份愛，不是父母要求孩子應該如何，也不是父母為孩子做了什麼，更不是讓孩子好好報答父母。

而是，我享受和你在一起，我理解你所在的狀態，並且願意與你一起在那個狀態。我陪伴你，允許你，接納你。

圖片來源於網路

如何讓孩子感受到呢？可以從這三點入手：

1. 給予孩子感受上的回應，而不是反應

很多時候，我們無法理解孩子，是因為無法理解孩子行為背後的感受。

孩子需要的並不是真的幫助他們解決某個具體問題，而是我們不否定、不代辦、不指責，只做感情回應就足夠。

比如，孩子問媽媽：「我為什麼收到的禮物比哥哥少呢？」

我們習慣性地反應，會很容易對孩子升起評判，覺得孩子是想要更多的禮物。

其實，孩子這句話的背後，只是在擔心，媽媽對她的愛，是不是和對哥哥的愛一樣多？

我們如果能洞悉到孩子這份擔心，就能和孩子共情，孩子就會感受到：我是被理解的，我的擔心是正常的，媽媽是愛我的。

2. 給予孩子成長上的空間，而不是速度

孩子的智慧，只能從自己一次次的體驗慢慢積累，無法從他人的教導中快速獲得。

比如，困擾很多家長的問題，孩子如何和同伴相處。這需要讓孩子在和不同人的接觸中，根據雙方的感受，去調整讓彼此都舒服的相處模式，從中學會人際交往。

否則，如果父母忽視或者強行介入，會限制孩子人際交往能力的發展，同時讓孩子感到：我什麼都做不好，都要聽父母的。

最好的關係是，你慢慢走，我陪著你，不管前方是風雨還是陽光，我都在你身邊。

這份陪伴和等待，讓孩子感受自己的每一步嘗試都是重要的，他才有勇氣和力量，去探索未知的世界。

3. 給予孩子無條件的信任，而不是交換

「信任」這個詞感覺有些虛幻，但在完形，它不僅僅是一個口號，更是特別落地的行動。

在我無法理解你的時候，我相信你一定有自己的理由，我會去詢問和核對，你的想法是什麼，而不是自以為是地貼個標籤；

在你情緒低落的時候，我相信你一定是內心有了衝突，我會陪伴和等待，而不是指責你不堅強。

無條件的信任，並非是孩子做了什麼才可獲得，而是信任他本身的存在，就是有價值的。

圖片來源於網路

一個價值感穩定的孩子，她的內在信念是：

雖然我現在學習還沒有那麼好，但我是價值的；

我雖然沒有及時幫助到同學，我依然是有價值的；

他人對我好與不好，都不影響我對自己的客觀評價；

我的價值和外在的東西，沒有任何關係。

這樣的孩子，在面對未來的不確定性時，才能夠保持對自己的清晰和堅定，做自己生命的主人。
