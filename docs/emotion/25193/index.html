沒有任何人能夠代替你去重新活一次

2019-12-21 107條評論 2,071次閱讀 276人點贊

我獨自站在熟悉的街口、恍惚之間就回到了那些和你一起走過的日子、懷念你的背影、你的聲音、你的笑臉、你的眉頭深鎖、你的侃侃而談、你的……而到如今、似乎一切都那麼遙遠。

不要去拒絕忙碌，因為它是生活的一種充實，不要去拒絕微笑，因為它是你擁有的最大魅力。

一個社會如果都是由對自己的人生負責的成員組成的，那這個社會就必定是高質量的有效率的社會。

我有那麼多的路要走，那麼多地方要去，只有你是我路途的最美麗的風景。

繼續說我不夠好，說我做不到。因為我總會一次次證明給你們看，我行的。 ​​​​

與其兩人不幸福，不如獨自不快樂。沒人會注意你的苦痛，但所有人都會關注你的錯漏。

不要用沉溺在過去來治癒往昔的傷口，而是應該更好地活在當下。

成功學只是對有能力，運氣好的人有用，因為他們激發了精神萎靡的人的精神力量。

一轉眼就已是揮霍了最後的青春。現實在一點一點的去消磨我們的稜角，侵蝕我們的夢想。

每個人都有一個死角，自己走不出來，別人也闖不進去，我把最深沉的秘密放在那裡，你不懂我，我不怪你。

每個人都會累，沒人能為你承擔所有傷悲，人總有一段時間要學會自己長大。 ​​​​

人生無須太多的準備，上帝給了我們腿與腳，就是讓我們不停地前行。

少在乎一點，就不會這麼糾結了。自己擁有的要心懷感恩，命運難測，有時，只在事過境遷後，我們才知道曾經的自己是多麼幸福。

們會放棄，常常不是因為難度問題，往往是因為，不再值得繼續下去。

負責的去生活，我不能想像，一個在人生中隨波逐流的人怎麼會堅定地負起生活中的責任。

重要的是做事，什麼事都是有意義的，而所做之事的成敗則是變得不很重要了。

我們每個人在世上都只有活一次的機會，沒有任何人能夠代替你去重新活一次。

美好的日子給你帶來快樂，陰暗的日子給你帶來經驗。所以，不要對生命中的任何一天懷有遺憾。
