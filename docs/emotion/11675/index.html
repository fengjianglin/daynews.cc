老闆主動談話，一定要注意這5句話，看是關心你，其實是在發警告

2019-12-11 2,031條評論 4,727次閱讀 636人點贊

分享職場故事，交流職場經驗，歡迎關注「閆湘」。

剛入職場的表妹，非常在意領導是不是重視她，尤其是每當領導找到她說話時，就覺得領導是在關心她。有時候領導說了一句關心的話，都可以讓她開心好久。因為領導從來不會疾言厲色的批評，每次都是和顏悅色，善意滿滿。但是聽了表妹的表述後，瞬間發現表妹這是太過天真了。她的領導其實是在敲打她，而她卻渾然不知。

那我們如何辨別領導是真的關心還是敲打呢？有那些話能夠給我們提醒呢？

一是要注意領導說的「你最近很忙吧」。

我們工作任務有多少，時間夠不夠充裕，領導其實心裡一清二楚，為什麼領導要說這句話。主要是因為在領導心中，我們的工作速度和完成率達不到他的期待，所以才會假裝關心我們到底忙不忙。當領導說了這句話，一定要對自己的工作任務再從頭梳理一遍，避免出現任務拖延和忘記的情況。把自己整改的態度讓領導看到，千萬不要懵懂不知的以為領導是關心自己。

二是要注意領導說的「你最近狀態不好，是有什麼困難嗎」。

當領導說這句話時，大多數情況下並不是在意我們是否遇到了困難。畢竟當工作中遇到困難時，都會向同事求助或者領導反饋。領導的重點是在你最近狀態不好，有可能是在指出你工作注意力不集中、完成任務不細心或者是考慮問題不夠全面。立即端正自己的工作態度，不要把任何事當做借口。領導不想聽這樣那樣的借口，而是要你把精力全心全意的放在工作上。

三是要注意領導說的「你辛苦了」。

我們聽到領導這句話，一般都覺得是領導的關心。但很多時候，領導往往是在我們比較閑的時候說這句話比較多。其實，當領導這麼說時，都是在反向提醒我們最近太過清閑，這時候我們就應該要去主動尋找業務或者是向領導彙報是否還有工作需要自己做的。讓你的「辛苦」達到領導要求的標準時，領導才會看到你為了工作付出的努力，千萬不要把領導說的辛苦了當做自己可以偷懶了的信號。

四是要注意領導說的「同事之間要和平共處」。

表妹說因為自己剛進公司，和同事之間難免產生摩擦，鬧了不愉快。結果第二天領導就特意跟她說了這句話。看似是領導關心她和同事之間的相處，其實領導可能已經先入為主的覺得她就是那個挑事的人，是在變相的批評她與他人交流的能力。領導一般都不會管同事之間芝麻蒜皮的小事，當遇到這種情況，一定要敢於說出和同事產生矛盾的原委並反思自己的問題。

五是要注意領導說的「說話要委婉點，不能這麼直」。

畢竟是年輕人，表妹有時遇到事情不會委婉的表達，某天直接懟了領導，雖然領導沒有發火，還是給表妹說了這麼一句話。領導這時對錶妹的表達能力已經產生了不滿，懟領導就是質疑領導的權威，雖然語氣中沒有責怪的字眼，但整句話都是在旁敲側擊。所以，當領導說了這句話，之後就一定要注意和領導說話時的表述，盡量不要太直。

我們在職場中，修鍊情商也是很重要的一個部分。學會聽懂言外之意，學會聽懂弦外之音，就能夠在與領導和同事相處過程中更愉快。領導每說一句話都有自己的含義，尤其是職場新人，在遇到領導突然找到你說一句話時，就要保持警醒。一定要深入思考領導說這句話的原因，不然很有可能會讓領導對你更加失望。

所以，你覺得該怎麼理解領導看似關心的5句話呢？歡迎分享說出你的觀點。
