天地有大美，於簡單處得，人生有大疲憊，在複雜處藏，早安

2019-12-17 1,558條評論 3,382次閱讀 1654人點贊

1、總有挫折打敗我們的心，讓希望背上沉重的包袱；總有困境分散我們的夢，讓未來染上昏暗的色彩；總有悲傷疼痛我們的情，讓生活踏上迷茫的遠方。選擇在風中吶喊在雨中哭泣，並不能改變現狀，只有擦乾眼淚，踏實地走好每一步才能更快的接近希望。迷茫的日子，我們不能被悲傷埋沒，必須學會振作自強。

2、人生之路是由成功與挫敗、得志與失意、輝煌與黯淡、平坦與曲折拼湊而成的，連接著我們的淚與笑，承載著我們的悲與歡。無論置身成功，抑或處於逆境，還是面對敗北，都只是一個過程，路依舊在向前方延伸。要相信，梅雨季節再長，也澆不滅陽光的熾熱；艱難困苦再多，也遏不住我們飛騰的夢想。早安！

3、你過得快不快樂，只有你自己知道。其實誰都有一段不為人知的故事，其實誰都會脆弱的想要一個停靠，其實誰都想和某個人完成曾經諾言，其實誰都能微笑然後轉身流淚，其實誰都的生活都多少有那點苦澀，經歷風雨。我只想要少一點悲傷悲傷，多一點快樂。我只想要少一點孤獨，多一點幸福。

4、生命的天空里，有風和日麗，亦有雲遮霧障。不是每輪艷陽都暖人，不是每片烏雲都下雨。既然決定不了命運的走向，那就踏實走過每一天；既然左右不了天空的變幻，那就悉心裝扮自己的心空。請相信：珍惜了，能擁有；付出了，有回報；走過了，不後悔。

5、生有你，我會珍惜。我的空間有你，我會歡喜。問候有你，我會快樂。分享有你，我會感動。一路相伴，溫暖永久。天涯海角，真誠永遠。輕輕的問候，默默的祝福！一路走來，有你真好！心存感恩，感恩有你，早安吉祥！

6、人生就好比一場特殊的賽跑，對於不同的人來說，即便在同一起跑線，也可以有不同的方向，因為終點並非一個。只要找准自己的目標，腳踏實地的去走每一步，就可以一路欣然向前，我們最終都會找到自己的最佳位置！早安！

7、人生之簡單，如巨畫中的幾筆線條，有著疏疏朗朗的淡泊，有著清清涼涼的寧靜。人生之複雜，若潑灑在宣紙上的墨跡，渲染著城府與世故，揮不去嘈雜與迷惘。天地有大美，於簡單處得。人生有大疲憊，在複雜處藏。早安！

8、小雞「喔喔叫」，早晨又來到，外面的世界真美好，賴床的人兒快起床，呼吸新鮮空氣很重要，老師、老闆等你去報到，晨風載我信息到，一聲「早安」祝你不遲到！

9、惟有身處卑微的人，最有機緣看到世態人情的真相。一個人不想攀高就不怕下跌，也不用傾軋排擠，可以保其天真，成其自然，潛心一志完成自己能做的事。早安！

10、清茶，或淡或濃，清香自留心間；距離，或遠或近，真情永不變淺；祝福，或早或晚，傳遞我的心愿：周末了，願你忘記煩惱，好好休息！

11、早上開手機，串串鈴聲連連響起！旭日伴隨我的信息，給你快樂的開始；白天隨時隨地，祝福簡訊發給你，開心無比。早安，朋友！

12、快就像瓢潑的一樣，看那空中的雨真像一面大瀑布！一陣風吹來，這密如瀑布的雨就被風吹得如煙、如霧、如塵。早安！更多的精彩情感，哲理，美文；敬請關注微信公眾號：心成一片
