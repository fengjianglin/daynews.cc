五世同堂，“隔代親”的跨世紀對視感動眾人，百歲老人也懂“愛”

2019-12-11 244條評論 1,651次閱讀 143人點贊

文丨宋媽說育兒（原創文章，歡迎轉載）

相信很多人都是從小跟著爺爺奶奶或者姥姥姥爺一起長大的，回想起童年的時光，最懷念的還是老人家最拿手的一桌飯菜和他們年輕時的背影。

隨著時代的進步，人們的生活節奏越來越快。更多的人選擇定居他鄉，結婚生子，很多時候，連父母家都很少有時間回去一趟，更不要說和自己的爺爺奶奶、姥姥姥爺見上一面了。

前一陣兒在網上看到一個故事，重孫子都有了寶寶，105歲的老人當上了高祖父，而這一天，他第一次見到了自己的玄孫。

105歲老人和玄孫初次見面，是跨世紀的對視，也是五代人的傳承

小陳（化名）的老家離自己的定居城市很遠，自從有了寶寶還一直沒回去過，這一次，他帶著老婆孩子一起回家看望老人。更讓人期盼的是，此次回鄉也是家中太爺爺第一次看到他的玄孫。

知道要見到自己的玄孫了，老人家非常激動，連著幾天都徹夜難眠。終於，相隔五代的二人就這樣見面了。

都說人老了就像個「老頑童」，老人見到自己的玄孫笑得像一朵花，對於孩子更是越看越喜歡，連睡午覺都離不開小寶寶了。而小寶寶看見自己的高祖父，更是沒有絲毫的生疏感，小眼睛溜溜盯著老人看，還咯咯地笑起來，這份無法割捨的親近感可能就二人血脈傳承的結果吧。

看著一老一小這般可愛，小陳也被這樣溫馨的畫面感動了。想想自己從小就是太爺爺帶大的，現在自己也有了小孩，而太爺爺的臉上卻布滿了歲月的痕迹，不由得心裡一緊，鼻子酸酸的。

有些故事看了感動，有些故事看了感同身受

看到這跨世紀的相望，網友們也動容了，紛紛留下了感動的淚水，也有不少網友覺得透過小陳家的故事，看到了自己的生活，想起了自己的童年。

@千事叔：一直都覺得這世界上最愛我的人就是爺爺奶奶了，總是有很多話要對他們說，卻總是覺得言語太過淺薄……他們是我的軟肋，但也是我的盔甲。

@鍋小瓶：工作後，賺了錢就給爺爺奶奶買東西，但他們總說我破費，買什麼也不捨得吃，這次買的下次我回來還留給我。每次回家，爺爺總愛嘮叨，問我成績怎麼樣，編製考上沒，談朋友了沒，每次都覺得他很啰嗦。但每當他們看我回家沖著我笑嘻嘻，還責怪我回來的次數太少，就覺得心裡很暖很暖。

@劉大志：那年春節全家人圍坐在一起，奶奶突然站起來問我想不想吃面。當時的她已經82歲了，手腳顫顫巍巍，我明明可以說「不用」，但還是說了「好的」。家裡人都覺得我不懂事，我自己也這麼覺得。但當我看到奶奶一個人坐在火爐邊，我知道我曾吃過無數碗奶奶做的面，但這一次恐怕是最後一碗了。

其實，做了爸爸媽媽的人更是對「隔代親」感觸良多，很多網友表示自從有了寶寶，自己已然「失寵」。對於孩子們，都說有一種餓叫姥姥覺得你餓，也有一種時尚叫姥姥覺得你時尚，其實，玩笑歸玩笑，背後的現實值得每個人深思。

為什麼有「隔代更親」的現象，背後原因是什麼

1）新的生命，新的延續

當老人知道自己時日無多，卻看到自己的生命在孩子的身上得以延續，內心有欣慰也有感動，這是一種對生活美好願望的表達。

2）「老頑童」的性格和孩子更「玩」得來

有人說年紀大的老人是「老頑童」，其實這也是有一定依據的。隨著年齡的增長，不管是智力方面還是心理方面，老人會越來越童稚化。

在這個過程中和兒女的共同話題越來越少，對後輩的愛會全部表達在最小一代的孩子身上。而且，孩子反向對自己的親近和喜歡，也在某種程度上彌補了一定的晚年寂寞。所以說，老人和孩子更「玩」得來。

小編的碎碎念

忙忙碌碌，兜兜轉轉，最疼愛自己的永遠是家中的老人和自己的父母。其實他們要的並不多，最盼望的就是孩子們的陪伴。

我們努力工作目的不就是為了更好的生活嗎？不要讓「子欲養而親不待」的無奈發生在自己身上，如果可以還是要多陪陪家人，多照顧照顧家中的老人。

雖說人們普遍認為爺爺奶奶帶娃還是弊大於利的，但既然有「隔代親」這樣的現象出現，就說明有它獨特的意義，這就需要子女們的理解和妥善處理，比如在自己帶娃的同時抽出一些時間帶孩子多回家看看。

臨近年末，今年春節你是否有帶著孩子回家看望老人的想法呢？還是你一直都是這樣做的呢？下方留言我們一起討論。
