“我好像一晚沒合眼”，小女孩半夜11點驚醒，哭著刷牙準備上學

天天要聞
2019-12-21

「焦慮症」是當下十分流行的一個詞，因為近些年社會發展速度太快，人們為防止掉隊脫節，只能逼著自己趕緊跑，生怕稍有懈怠就被其他人甩到身後。然而，這種快節奏的生活，不僅在無形中增大了人們的心理壓力，也催發了各種負面情緒，使人食難咽寢難安。因此不少人都曾感嘆，好想變成懵懂的孩子，重回無憂無慮的童年時光。

但是，小孩子就一定活得輕鬆嗎？恐怕也不見得。在全民加速的大環境下，不僅成年人疲於奔命，孩子們其實也難逃生活的重壓。近日，網上流傳著一段很火的視頻，讓網友們覺得既好笑又心酸。視頻中，一個小女孩滿臉委屈地站在衛生間里刷牙，一邊哭一邊向媽媽大吐苦水，說自己昨晚好像一宿沒合眼。聽到女兒滿腹委屈的話，一旁的媽媽忍不住笑出聲。

原來，這名來自四川宜賓的小女孩在當天晚上9點就上床睡覺了，但在11點多她又忽然驚醒，迷迷糊糊中以為天亮該上學了，便起床開始洗漱。由於沒有睡夠時間，小女孩覺得特別委屈，就撇著嘴哭了起來，但又因為不敢遲到，所以她只好一邊哭一邊刷牙，順便跟媽媽哭訴自己的不容易。看著女兒既可愛又可憐的樣子，媽媽忍俊不禁，趕緊告訴她長夜還沒過完，還可以繼續睡。聽到這話，小女孩才放心地抹掉眼淚，情緒漸漸好轉。

​

小女孩鬧出的這段笑話被媽媽發到網上後，很快引起了網友們的熱評。有人調侃說：小女孩睡懵的模樣，像極了曾經的自己。也有人為小女孩點贊：這麼冷的天氣還能自覺起床上學，是個好孩子。還有人心疼道：現在的孩子確實不容易，學業壓力讓他們也睡不好覺，真是可憐。確實，現在的小孩子也活得不輕鬆，所以家長們在平時一定要多關心他們的精神狀態，以免他們壓力過大而產生心理問題。
