孩子出生後是斷掌，婆婆很高興，去體檢後全家人卻都笑不出來了

2019-11-25 0條評論 24次閱讀 0人點贊

摘要

孩子出生後是斷掌，婆婆很高興，去體檢後全家人卻都笑不出來了孩子出生後是斷掌，婆婆說長大一定有出息，體檢後卻高興不起來了

阿麗和丈夫都是村裡的大齡青年，後經人介紹走在了一起，結婚後不到半年就懷孕了，家裡人都特別高興，阿麗懷孕期間婆婆的照顧也是無微不至。

懷孕後不久阿麗聽別人說最好去醫院做個體檢，然後把這件事件告訴了婆婆，因為離市裡較遠，需要坐很久的汽車，婆婆擔心阿麗動了胎氣，就和阿麗說：「沒必要去體檢，我們那年代生孩子，都沒有做過體檢，生下來的孩子不也都健康么」，阿麗也覺得不會有什麼問題，這件事就這麼過去了。

臨產的日子終於到來，阿麗生下了一個男寶寶，這可把全家人樂壞了，這時候有人說：「你們看，這孩子還是個斷掌」，婆婆樂得更開心了，因為在當地的說法，斷掌長大都會有出息，邊說邊抱著孫子跟其他人炫耀。

這種快樂一直保持到了寶寶第一次體檢前，滿月後按照規定帶著寶寶去做了一次體檢，醫生檢查後發現寶寶雙眼的距離有些遠，後來經過仔細檢查確定了這個一個唐氏綜合症寶寶，這個消息對阿麗來說簡直是晴天霹靂，不敢相信自己的耳朵，丈夫回家後把結果告訴了婆婆，婆婆剛開始還不知道什麼是唐氏兒，丈夫給她解釋後傻眼了。

什麼是唐氏綜合症

唐氏綜合症又叫21-三體綜合征，是由於染色體異常造成，有大概60%左右胎兒在孕婦懷孕時候會流產，如果倖存下來出生後也會有明顯的智力低下、面部特徵特殊、運動發育障礙、生長發育遲緩等等一系列問題。

導致唐氏綜合症的原因

1、遺傳因素

如果夫妻雙方任一家族內，出現過唐氏兒，那麼這對夫妻生出唐氏兒的幾率也會比正常人大。

2、高齡產婦

大齡夫妻，特別是高齡產婦，由於身體內細胞已經不再年輕，畸變的概率也會增加，所以唐氏兒的出生率也會增加。

3、環境因素

在胚胎的發育過程中，輻射、病毒感染、誤食藥物等，都可能造成染色體異常，增加唐氏兒出生風險。

唐氏綜合症如何預防

唐氏綜合症目前還沒有治癒的方法，只能通過孕期時候通過「唐氏篩查」，去判斷是否有患唐氏兒的風險。

任何的夫婦都有可能生出唐氏兒，特別是高齡產婦胎兒的患病率更高，所以懷孕16~18周的時候，一定要去醫院做唐氏篩查，在孕早期進行有效的判斷和預防。

每一個迎接新生命的家庭，都希望有一個健康活潑的寶寶，但是有時候會毫無徵兆的就降臨在了寶寶身上，我們要試著去接受這個只比我們多了一條染色體的寶寶，我們正常人有的感情，他也會有，甚至會更加毫無保留，隨著醫學的不斷進步，希望有一天唐氏綜合症可以被徹底治癒，讓唐寶寶也能像其他小朋友一樣，享受快樂人生。
