每天重複這樣的日子，

他已經過了整整10年。

能幹的妻子10年前病了

裴永康家住尖山新區（黃灣鎮）閘口村41組，這是一幢普通民房。屋內擺設簡陋，基本上沒什麼裝修。

年過八旬的裴永康，年輕時做過10多年的生產隊長。20歲的時候，他娶了19歲的妻子顧四寶。如今，他們已結婚60年。

他們生有3個兒子，一個女兒。如今兒女們都已人到中年，各自有了家庭。他老兩口吃住在小兒子家。

顧四寶年輕時身體很好，力氣蠻大，家裡養了10多隻羊，全靠她一人割草飼養。

意外發生在2010年。顧四寶感覺手腳有些發麻，並不由自主顫抖，便到市二院去看病，被診斷為帕金氏綜合征。

網上權威醫學資料記載：

帕金森氏病又稱震顫麻痹，是中老年人最常見的中樞神經系統變性疾病。震顫是指頭及四肢顫動、振搖，麻痹是指肢體某一部分或全部肢體不能自主運動。其得名是因為一個名為帕金森的英國醫生首先描述了這些癥狀，包括運動障礙、震顫和肌肉僵直。一般在50～65歲開始發病，發病率隨年齡增長而逐漸增加，60歲發病率約為1‰，70歲發病率達3‰～5‰，我國目前大概有170多萬人患有這種疾病。目前資料顯示，帕金森氏病發病人群中男性稍高於女性，迄今為止對本病的治療均為對症治療，尚無根治方法可以使變性的神經細胞恢復。

帕金森氏病的病因現在還不是很清楚。目前公認其病因是神經細胞的退行性變，主要病變部位在黑質和紋狀體。這裡有一種叫黑質細胞的神經細胞，黑質細胞數量的逐漸減少、功能的逐步喪失，致使一種叫多巴胺的物質減少，從而引起上述癥狀。根據動物實驗和流行病學的結果認為帕金森氏病與遺傳也有一定的關係。

本來想著老來兩人可以享享清福了，裴永康沒想到妻子患上了這種病。

顧四寶第一次在市二院住院治療，在醫院一待就是十幾天。住院期間，裴永康不眠不休照顧妻子，給她擦身洗臉按摩喂飯，病情有所好轉。

出院回家後，裴永康每天堅持給妻子按時服藥按摩，本以為妻子的病好了，可是過了沒多久，顧四寶病情又複發了！

隨著時間的推移，顧四寶病情越來越重，再次到二院住了10多天院，就這樣，病情稍有好轉後出院回家。

裴永康沒想到的是，在往後的十年中，這樣的模式幾乎成了他生活的全部。

妻子病痛痛在他心

醫生告訴裴永康：「這種疾病目前世界上沒有什麼特效藥能根治，只能緩解癥狀。」

但裴永康不願意放棄，他想總有一天科學發展了能治好這種病，讓老伴站起來。就這樣，老伴一發病，他就帶著她往返在二院和家之間……

這麼多年來，顧四寶共住了13次院，每次住院，除去報銷的醫藥費外，總要自負三四千元。到如今已花去醫藥費十幾萬。

即便如此，裴永康一聽說二院來了杭州大醫院的專家，就馬上帶著老伴趕到醫院，找杭州專家看病。

可是，杭州專家也告訴他：「目前的科技水平很難治癒這種疾病，但要堅持服藥，不使疾病加重就好。」

裴永康背地裡偷偷的抹淚，更替老伴感到難過……但他不甘心，也不願意放棄，他不希望讓老伴感覺到治癒沒希望了。

後來，他還帶著老伴去海寧市人民醫院，請知名專家給老伴治病，但結果與杭州專家的說法差不多。

日子一天天過去，原本利落、能幹的老伴行動漸漸不便，變得痴呆、麻木，生活不能自理。

裴永康終於接受了這樣的現實，他放棄了到廠打工的機會，用無微不至的照顧來緩解心頭那種絕望和悲傷。

「夏天還行，到了冬天真是苦不堪言，有時老伴半夜疾病發作渾身顫抖，她坐在床里痛苦地掙扎，我也沒辦法，給她按摩，也無濟於事」。

天氣寒冷妻子就容易發病，他總是半夜起來，慢慢扶起妻子，抱住妻子，幫她按摩顫抖的身體……

病痛的是妻子，心裡疼痛的是他。

10年來像照顧寶寶一樣照顧老伴

老伴患病後，老夫妻倆的生活作息就完全變了。

10年來老兩口每天只吃2頓飯。

一是老伴總要晚上起夜幾次，裴永康精疲力盡，作息幾乎沒有日夜之分。二是老伴為了減少上廁所，總是盡量少吃……

每天，這幢普通的民房裡就會上演這樣的場景：

裴永康買菜總是捨近求遠，不到黃灣集鎮買菜，而是早上騎著自行車去袁花買菜，因為那裡菜式更多價格更便宜。

回來做好飯菜，他把一塊毛巾鋪在老伴的膝蓋上，以免飯粒掉下，盛飯、端菜，左手拿著碗，右手拿著調羹，將飯菜拌勻，一勺一勺地送到老伴嘴邊……




他每天把家門前的水泥曬場打掃得乾乾淨淨，連一根稻草都沒有。為的是出門讓妻子看到整潔的環境。

喂完飯，裴永康就推著坐在輪椅上的妻子到室外曬太陽或散心，365天除了下雨天不出去，每天都如此。


一路推著老伴，他們最遠到袁花集鎮、黃灣集鎮，很多時候，裴永康就推著妻子到黃山村的文化禮堂去噶鬧忙，讓老伴露出舒心的笑，也是他一天中最開心的時刻……

在今年農曆二月初八，黃灣噶太平，他還推著老伴去噶太平，到黃灣鎮去軋太平……路人對他投以好奇的目光，可是裴永康感覺很驕傲，老伴還在他身邊，他們正感受著熱鬧的生活……

最困難的是老伴晚上大小便。「畢竟我現在年紀大了，抱不起她了，有時抱得肋骨也痛，但咬著牙堅持著，每晚總要二三次……」裴永康說。

這樣的日子裴永康不知道什麼時候是個頭

他不去想也不敢想

他只想老伴陪在身邊

雖然她很多時候已經話也說不利索了

但至少是個活生生的人

顧四寶不發病的時候，會看著自己的丈夫動情地說：「我的丈夫真好，沒有他，我活不了這麼大歲數。」

「我20歲那年娶了賢惠的她，她把最好的年華留給了這個家。她晚年得病，既然我是她丈夫，就應該好好照顧她，陪她一起走到生命的盡頭。這樣，我也沒有遺憾。」裴永康說。

這世間，你常問愛是什麼模樣？

不羨慕街頭擁吻的情侶，

只羨慕黃昏牽手的老人。

愛，就是在磨難中，

那個人從未離開！
