01.

小時候家裡條件不好，有段時間父母感情又不太穩定，我媽媽帶著我回外婆家住過一段時間。在我們老家那，外嫁的媳婦回娘家，是一件不太光彩的事情。但是出於無奈，我媽媽還是帶著我回去了。

外婆一直住在農村裡，是個很善良、和藹的人。

記得讀小學時，我比較嘴饞，經常把外婆廚房裡的各種盒子翻來翻出找吃的。外婆有個已經生鏽的的鐵盒子，裡面放著大姨買給外婆的餅乾和速食麵。

那個時候，大家生活條件都不太好，外婆每次吃的時候都會給我一點。有一次我太餓了，自己從鐵盒子里拿了一包速食麵泡著吃，外婆看到後順口說我真是一個小饞貓。現在長大了，有時候在超市裡看到鐵罐的餅乾盒，我都會想起外婆家裡那個生鏽的鐵罐盒，那個盒子，封存著美好的回憶。

從我懂事起，我就發現媽媽對外婆很孝順。家裡要是有什麼好吃的，媽媽都會送一點給外婆吃。外婆年紀大了，自己洗衣服不太方便，媽媽也會幫她把臟衣服洗好。

等我慢慢長大後，媽媽因為工作比較忙，有時候沒時間去外婆家，就讓我帶了點水果和牛奶去給外婆。外婆年紀大了，身體不太好，當她的葯吃完後，我趁著周末有空，也會去街上藥房買葯回來。

02.

我曾在一本書上看過這樣一句話：你的父母對他自己的父母好，將來你的孩子也會對你好。

這句話雖然有點繞，但是我覺得還是挺有道理的。當你的父母對他們的親生父母好，你作為晚輩，在這樣的環境長大後，經過耳濡目染後，也會做一個照顧老人，對老人好的人。等到自己父母年紀大了，也會好好的贍養父母。

在這方面上，我的媽媽就做得比較好。我在這樣的環境下長大，對父母也挺尊敬的，不會做什麼讓父母難過的事。大家都是自己人，家和萬事興，和和睦睦比較好。

不過，在孫奶奶的家裡，她的女兒從小和她關係不是很好。現在孫奶奶已經75歲了，女兒卻很少去看她，基本上每月就打一兩個電話，簡單象寒暄下，就匆匆的掛了電話，讓孫奶奶比較傷心。

03.

孫奶奶和老伴兒的感情不是很好，多年來兩個人雖然生活在同一個屋檐下，但是兩個人卻像是最熟悉的陌生人。她們很少吵架，更多的是雙方用隻言片語來交流。

在這樣的家庭環境下，自己的女兒也漸漸變得有點冷漠，以前老師曾這樣形容孫奶奶的女兒：「這位同學，人倒是挺好的，就是不太喜歡說話，同學們都不太和她交朋友，每天一個人上學放學，看著挺不合群。」

孫奶奶心裡清楚，自己女兒之所以比較孤僻，是她和丈夫的原因造成的。當孫奶奶還是一個中年婦女時，她想走進女兒的心扉，和女兒交個朋友，帶女兒出去散散心之類的。可事與願違的是，女兒都以平時在學校比較忙，回來要做功課，要上補習班等理由拒絕了。

就這樣，孫奶奶一家人在一些親戚朋友們看來，都是比較奇怪的人。

直到孫奶奶快過60歲大壽時，她的老伴兒突然說要和她離婚，原因是感情早已破裂，與其將就著再生活個幾十年，還不如各自放過對方，成全對方，去找屬於各自的幸福去。

孫奶奶的老伴兒原以為孫奶奶會拒絕的，沒想到孫奶奶一口答應了下來。現在女兒已長大成人，跟不跟誰過已經不太重要了。後來，孫奶奶和老伴兒順利的辦了離婚手續。

在孫奶奶的女兒看來，自己父親提出了離婚，可能已經相上了心儀的對象。她覺得自己媽媽都一把年紀了，還被離婚了，有一點可憐。

女兒雖然也人到中年了，但是看著自己父母的感情一直不太好，她對於感情也沒什麼信心，一直沒有找對象，用現在一句不太好聽的詞來說就是大齡剩女了。

出於好心，原本女兒自己在外面租房子住的。現在自己父母離婚了，她就搬回家和孫奶奶住在了一起。

04.

孫奶奶原以為離了婚後，自己的日子會變得好起來。但是她有時候回想起自己上一段不太幸福的婚姻，就陷入了自責中。久而久之，一個人經常在家裡對女兒說一些碎碎念的話，有時候一個人也會在客廳里偷偷流淚，有時又好幾天不怎麼說話，對著女兒傻笑。

女兒看到媽媽這樣的情況，以為她得抑鬱症了，就讓孫奶奶去看病。孫奶奶一聽自己女兒竟然說她身體有問題，一個勁的說自己沒有問題，身體很健康。

出於好意，女兒還是擔心自己的媽媽將來會出問題。有一次女兒瞞著媽媽，帶她去醫院看抑鬱症的醫生，被媽媽發現後，就想離開醫院。幸運的是，在醫生的勸說下，孫奶奶開始了治病。

不過，因為這件事，母女間的關係就開始變得僵硬了起來。孫奶奶埋怨說，女兒不該騙自己來醫院看病，這樣是不信任自己的表現。可是，人心是肉長的，女兒看到自己母親都這樣了，哪怕是說了個善意的謊言，只要能把媽媽的病看好，也是可以的。

後來，孫奶奶和女兒的關係越來越僵了。女兒覺得自己母親誤會了她，就乾脆又搬出去住了。剛開始，孫奶奶覺得一個人住更清凈，可是時間一長了，她慢慢發現沒了女兒的照顧，再清凈也沒多少意思。這個時候，就開始念叨著女兒的好了。

孫奶奶開始給女兒打電話：「女兒，媽媽錯了，媽媽現在知道還是我的乖女兒好。你要不回來和我住一起吧，你住在外面還要錢，回來住就省錢了。」

女兒接到電話回了句：「媽，真不是我說你，我以前就說你生病了，要去看醫生。你又比較倔，還不想去。現在你一個人住著覺得沒意思了，又想到了我。」

孫奶奶不好意思的回復道：「我的好女兒，媽媽知錯了。媽媽都75歲了，現在只能和你相依為命了，媽媽見一次就少一次了，你哪怕不回來和我住一起，也應該常回家看看我。」

女兒是個好人，緩了緩神，回了孫奶奶一句：「媽，我沒生你的氣。我收拾收拾行李，過會就搬回家和你一起住，好好孝順你。」
