結了婚的你，對婚姻有怎樣的感受呢？

這個問題是不是讓你的心裡五味陳雜，或許也曾靜下心來想過一會兒，隨後又陷入了柴米油鹽的煙火歲月。

沒結婚的時候，我們都渴望婚姻。看著電視劇里，陽光正好的早晨，夫妻倆人一起刷牙洗臉，在廚房做早餐，丈夫給妻子一個擁抱，妻子回應丈夫一個甜美的微笑，用烤好的蛋糕堵住他的嘴。這邊是我們認為的婚姻。

可結婚後發現，早上起床一個比一個困難，刷牙洗臉根本不在一個時間線上，微波爐和烤箱從沒有在早上的時候工作過，美好恬靜的清晨因為要趕著上班而變得熱鬧忙碌。

人家都說，人間煙火氣，最撫凡人心。可我們還是會因為誰做飯誰刷碗的問題而煩心。

我們想像的婚姻，求而不得。我們現在的婚姻，也會讓你失望。

既然婚姻像個孩子一樣，那為什麼很多人還是如此的渴望它呢？

不妨讓我給你講講小鍾收到妻子的一封信。

小鍾和妻子結婚7年了，每年他過生日，妻子都會陪著他一起過。可今年的生日，只能自己過了，因為在生日前一天，妻子被通知出差。

32歲的人了，收斂了很多孩子氣，變得遇事有些成熟了，他沒有抱怨，而是告訴妻子注意安全，照顧好自己。然後想趕在生日之前加班把手頭工作做好。

生日那天早上，他從公司回家，準備好好休息一天，可原計劃變了，只想著先回家睡一覺再說。

到家後發現卧室的柜子旁邊，有一個扎著彩帶的禮盒，禮盒上面有一張紙。

他知道，這是妻子準備的禮物，每年都不會少。他興奮地拆開禮盒，裡面是一個電動剃鬚刀，旁邊放著一個小紙條，上面寫著：這個好用，以後別再扎我。

一股暖流湧入心底，想拿起電話打過去撒個嬌，還是忍住了沒去打擾。

他拿起了那張紙，原來是妻子寫的信。他心想老夫老妻的這是要弄哪一出，開始認真讀了起來。

02

老張生日快樂：

不覺間，我跟你了7年，也不算吃虧，整體來說，你對我挺好，我也還算滿意。

七年之癢開始了，你準備好了嗎？

很遺憾不能鑒證你越來越像個爺們的時刻，我相信，我們會一起跨過很多的7年，跨多了，也就不癢了。

跟你在一起這些年，從來也沒有想過要矯情的寫點什麼，來表達我對你的愛有多深，一切都交給月亮好了。哈哈，開個玩笑。

從一個人的自由身，變成了兩個人的相互糾纏，婚姻給你帶來的是什麼感受呢？

這個問題，我偶爾會問自己，你也不要緊張，我沒說你不好，我只是想把心裡話用文字的形式跟你表達。因為這些話當面說，說不出口。

給我的感受是，婚姻最真最好的狀態時：痛並快樂著。

還記得每次你和我講道理時，我就會痛一次，你沖我大聲嚷嚷，和我吵架，甚至要伸手時，我的痛就會疊加。

雖然老話常說「家家都有本難念的經」，但是我認為這本經大部分的地方都是雷同的。

比如因為廚房，因為家務，因為個人衛生，因為錢，因為情等。這些是作為已婚人都要經歷的。

幸好你沒有做過對不起我的事情，繼續努力，不要驕傲。

婚姻告訴我：現實的婚姻不會一直讓你快樂，偶爾也會讓你很痛。因為完美的人不存在，缺點裡往往就藏著使你痛苦的原因。

痛過以後，婚姻才能蛻變。痛只是陣痛，痛能讓你清醒，讓你更加珍惜眼前人，讓你看清眼前人。

當然，快樂佔據了大部分的時光，快樂是你想要走進婚姻的決定性原因。快樂就像是婚姻的養料，可以讓婚姻變得更加有愛，更加新鮮。

希望我們餘生可以痛並快樂的糾纏在一起，祝32歲生日快樂。

愛你的老婆

小鍾看後淚崩，原來自己心裡所有的困惑和對以後婚姻的迷茫在那一刻全都煙消雲散了。

其實這種狀態最好，不要在去羨慕別人的婚姻生活了，你眼中別人的婚姻，關上門的時候，也許還不如你。

柴米油鹽也好，吵架冷戰也罷，都是婚姻該有的一部分，失望與美好共生，這便是婚姻該有的樣子。

在痛中珍惜，在樂中領悟，這是我對婚姻的感受，你呢？
