治癒系，10句很簡單的道理，收藏了

1.沒人能預知下一瞬間的事，所以不必堅持更不必那麼固執，順其自然，就像下雨就打傘，冷就蓋被子，開心就笑難過就哭，很簡單的道理 你不愛我我就自己愛自己。

2.和聰明人交流，和靠譜的人戀愛，和積極的人共事，和幽默的人隨行。人生若能如此，足矣。

3.釋迦牟尼的一句話：「伸手需要一瞬間，牽手卻要很多年，無論你遇見誰，他都是你生命該出現的人，絕非偶然」。–––若無相欠，怎會相見。

4.心本身不大，千萬不要背負太多。昨天的糾結，只會囚禁你的今天和明天。人生，一定要學會釋放！

5.沒必要刻意遇見誰，也不急於擁有誰，更不勉強留住誰 。一切順其自然，最好的自己留給最後的人 。

6.小的時候，我們都長身體去了，所以很天真、很快樂。等長大，不長身體，就都長心眼去，就活的很累了。

7.很多人不是孤僻，而是有原則有選擇地社交。和喜歡的人千言萬語，和其他的人一字不提。

8.有事情是要說出來的，不要等著對方去領悟，因為對方不是你，不知道你想要什麼，等到最後只能是傷心和失望，尤其是感情。

9.避免失望的最好辦法，就是不寄希望於任何人、任何事。期待，是所有心痛的根源，心不動，則不痛。
