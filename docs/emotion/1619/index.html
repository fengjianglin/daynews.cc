哪裡來的那麼多人生感悟，不過是碰壁後的自我安慰

2019-11-30 0條評論 24次閱讀 0人點贊

摘要

親愛的陌生人，有幸你來，不負遇見曾經我們都以為自己可以為愛情死，其實愛情死不了人，它只會在最疼的地方紮上一針，然後我們欲哭無淚，我們輾轉反側，我們久病成醫，我們百鍊成鋼。

親愛的陌生人，有幸你來，不負遇見

曾經我們都以為自己可以為愛情死，其實愛情死不了人，它只會在最疼的地方紮上一針，然後我們欲哭無淚，我們輾轉反側，我們久病成醫，我們百鍊成鋼。

你還年輕，並且浪漫多情，可以為錢，為臉，為身材煩惱，但不要為感情折腰

如果你不吃別人家的飯，就別太把別人的話放在心上。

哪裡來的那麼多人生感悟，不過是碰壁後的自我安慰。

聽到別人的惡意中傷，沒必要對此念念不忘，轉頭放下便是。如果一直不停地去想，就像把別人吐出來的東西，自己又撿起來吃一樣愚笨。

不要太過在意別人的目光，你只要記得，你不是為別人而活著，你為的，是你自己。

今天你能承受多大的卑賤，明天你就能承受多大的榮耀。這不是雞湯，這就是現實。

始終記住，別用自己的一往情深綁架著別人，得不到回應的情深，請適可而止，別糾纏了別人，也別廉價了自己。

你臉上雲淡風輕，誰也不知道你的牙咬得有多緊。你笑得沒心沒肺，沒人知道你哭起來只能無聲落淚。

當現實抬手給你一巴掌的時候，你應該和它擊個掌。就算是破罐子，也要摔得比別人響！

融不進去的圈子別硬融，做不了的事別硬做，你也是人，不必討好全世界。

大冬天的時候給你送吃的的是外賣小哥，逗你笑的是喜劇演員，無論天南地北都順路來接你的是滴滴司機。所以餘生，有錢就夠了。

只有不迴避痛苦和迷茫的人，才有資格去談樂觀與堅定。命運不會厚待誰，悲喜也不會單為你準備。

不要做那種人人都喜歡的人了，要做那種有一部分人喜歡，被幾個人超級喜歡，被一個人深深愛著的人。

每次崩潰的理由在別人看來都是小題大做，可是只有自己才知道，這根稻草，壓倒了多少千斤重的難過。

花不會因為你的疏離，來年不再盛開；人卻會因為你的錯過，轉身為陌路
