人生不如意十有八九，切不可自暴自棄，相信自己是最棒的

2019-12-02 0條評論 21次閱讀 0人點贊

摘要

“我是最棒的、“我是最優秀的”這是我的“自我對話”，當我遇到不尊重自己的人，不管他多麼優秀、傑出，我一概列入拒絕往來戶，不再與他有任何的往來，原因是跟這些人交往，只會打擊自己的自信，毫無任何的益處。

「我是最棒的、「我是最優秀的」這是我的「自我對話」，當我遇到不尊重自己的人，不管他多麼優秀、傑出，我一概列入拒絕往來戶，不再與他有任何的往來，原因是跟這些人交往，只會打擊自己的自信，毫無任何的益處。
我日常生活所交往的朋友，都是「相互肯定」的朋友，如果我認為朋友不好、不傑出，我根本就不會與他交往，所以我的朋友，我都認為是世上最棒、最傑出的人，何其榮幸，能夠跟這些好友認識，但同樣的，我的朋友也都肯定我，大家相互肯定，友誼才能長久。

很多人都想要結交優秀、傑出的人，但如果自己不能夠成為優秀、傑出的人，又如何能夠與這些人交往呢？如果這些優秀、傑出的人根本就不想與自己交往，毫不尊重自己，又何必「熱臉貼冷屁股」呢？
過去曾聽過許多的演講，講師在台上演講，如何追求理想的情人，台下有許多的觀眾，問講師追求理想的情人的秘訣是什麼？講師雖然傳授了許多的秘訣，但最後總是強調要先把自己變作別人理想的對象，才能找到理想的情人，畢竟理想對象也在找尋理想情人。
我之所以不太容易被外在事物所干擾及影響，是因為我心中一直認為，別人認為你是什麼樣的人，一點都不重要，而且也改變不了自己，但如果自己也如此認為，就會改變自己，做任何事情如果不能「迎合自己」，而去「遷就別人」，到頭來受苦的還是自己。

「迎合自己」，不是說要表現出高高在上的態度，「貶低別人，成就自己」，而是內心的「自我肯定」人如果肯定自己，別人自然而然就會肯定你，但自己都不能肯定自己，試問又如何要求別人的肯定呢？
「自我洗腦」、「自我想像」、「自我對話」這也是我不太容易被別人影響的主因，畢竟「人是為自己而活，而不是為別人而活」當你不斷的「肯定自己」，想像心中認定的我，與現實中的我，差距還有多少，只要實現心中想像的自己，誰能說自己不是最棒及最優秀的人。
