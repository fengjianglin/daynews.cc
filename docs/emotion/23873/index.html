“男友”借走28萬後失聯，原來是一場“戀愛陷阱”！

2019-12-18 540條評論 2,430次閱讀 217人點贊

原以為是一段美好的良緣，不曾想是一場以「愛」為名的騙局。2019年10月1日，光明公安分局玉塘派出所接到事主文女士報警，稱其被男友騙走285000元，現與男友失去聯絡。

經進一步了解，今年5月，文女士在一休閑會所認識了溫某，在之後的一個多月時間裡，文女士和溫某在微信上展開熱聊。通過一段時間的相互了解，雖然只見過幾次面，但雙方還是確立了男女朋友關係。

「我的銀行卡被凍結了，裡面還有3475600元，最近手頭緊，你能不能先借點錢給我把卡解凍了。」6月1日，微信那頭傳來了本該引起警惕的文字，但是文女士卻不假思索地給自己的「戀人」轉了156600元。在這之後，溫某先後以「店鋪裝修」、「開車出事故」、「修車」等名義，多次向文女士借錢。自認為是「熱戀期」的文女士一一滿足了其要求，又向溫某轉了共128400元。

在文女士開始向溫某索要欠條和還款時，溫某均以各種理由推脫，緊接著將文女士微信、QQ號拉入黑名單，失去了聯絡。此時，文女士才恍然大悟自己被詐騙了，遂向警方報案。

了解情況後，民警斷定這是典型的交友詐騙行為，立即立案進行偵查，並成功鎖定了溫某的住址在寶安區石岩街道附近。掌握線索後，民警多次組織警力前往寶安區石岩街道伏擊蹲守，但溫某一直未出現。但民警始終沒有放棄，經過近兩個月的堅持，最終於12月9日23時許將溫某抓獲。

經審訊，溫某（男，22歲）對其涉嫌詐騙的事實供認不諱。據嫌疑人溫某交代，其卡里的3475600元並非自己所有，是其朋友暫時存放在其卡里用於購買車輛的，當時是為了博取文女士信任才稱這筆錢是自己的。據調查，掉入溫某布下的「甜蜜陷阱」的不止文女士，還有其他六七名女士。溫某詐騙得來的錢，全部用於汽車租賃投資和其他消費開支。

目前，嫌疑人溫某因涉嫌詐騙，已被光明警方依法刑事拘留。

近年來，隨著移動互聯網的大量普及,在線婚戀交友也日漸頻繁。在此，光明警方提醒各位市民，網路聊天要謹慎，不要輕信花言巧語，不泄露個人隱私，慎轉賬，慎見面，切勿掉進騙子為你精心編織的情感陷阱。
