幹掉自己人性的人，人性最後也會幹掉他

天天要聞
2019-12-11

做生意，明碼實價，童叟無欺；做學問，言之有據，持之有故；做官，不奪民財，不傷無辜；做人，不賣朋友，不喪天良。

不損人利己，尤其不能損人不利己，不破壞社會風氣，這是「底線」。

做人要有原則，做事要有底線。每個人活在這個世界上，都有自己為人處世的方式，但無論你選擇怎樣的生活，做人的原則不能丟，做事不能沒有底線。

底線不能旦夕缺失。因為底線是基礎，是根本，是不能再退的最後一道防線。基礎不牢，地動山搖；防線失守，全盤崩潰。

底線是一個人活在這個世界上的骨氣，也是一個人不能喪失的尊嚴。再窮也不能去做傷天害理的事；再苦也不能去做違背良心的事。

你有什麼樣的底線，就會有什麼樣的人生。做人，守住底線，才能活得一世安生。

一，底線不容逾越。

打破原則，最終的結局，終將是失去人性。而幹掉自己人性的人，人性最後也會幹掉他。

放縱自我，就是墮落的開始。稍一失足，便是人生的深淵。

正如卡爾維諾所說：「一旦你放棄了某種你原以為是根本的東西，你就會發現你還可以放棄其他東西，以後又有許多其他東西可以放棄。要守住底線，底線一旦被突破了一次，就很難再有底線了。」

一個沒有底線的人，做起事來會不擇手段，為了達到自己的目標，不惜損害別人的利益，來滿足自己的私慾。

一個沒有底線的人，可以為了金錢出賣自己的朋友，也可以為了利益違背自己的良心，在他的眼裡，金錢最重要，仁義如糞土。

一個有底線的人，無論何時，都會堅守自己的原則，把人品放在第一位，把良心看的最重要，做事光明磊落，做人坦坦蕩蕩。

一個有底線的人，講仁義，重感情，視金錢如糞土，把人品看的比自己的生命還重要。

每個人都要維護好自己的底線。可以忍受貧窮，但不能突破底線。富貴不能淫，威武不能屈，貧賤不能移。唯有此，才能得到別人的尊重，人生才會過得有尊嚴。

人過留名，雁過留聲，到什麼時候都要記得，做人，一定要守住做人的底線，切不可失守。守住底線，方能始終。底線在，人格就在；底線破，人格就滅。一個人有些東西可以捨棄，人的底線千萬不能捨棄。

二，有底線的人，才能走得更遠。

守住底線，便是守住安心，守住事業，守住人生。

很多人覺得，原則問題上偶爾的變通，是一種圓滑的機靈，更是晉陞之路必要的墊腳石。

其實不然，堅守底線的人，才能真正走得長遠。

稻盛和夫是「京瓷」和「KDD」的創始人，被譽為日本「經營之聖」。

在日本有一個時期，許多企業爭先恐後參與不動產投機，想撈一把快錢。

稻盛和夫的很多好友也勸他投資不動產，以期獲得更大的財富。

但是稻盛和夫並沒有這麼做，他秉承著自己的人生原則：天上不會掉餡餅，這錢來的如此輕鬆，定是不義之財，雖然來的容易但去的也必然很快，只有額頭流汗，靠自己努力賺來的錢才是真正的利潤。

於是，但凡有關投機賺錢的方法，稻盛和夫一概回絕，只是兢兢業業，踏踏實實地耕耘著自己的事業。

後來，果然如稻盛和夫所言，泡沫破裂，許多企業都走向了破產。

只有稻盛和夫的企業，直到如今，依舊躋身世界500強，屹立不倒。

人活於世，底線其實像是人生的保護線，它擦亮我們被利益蒙蔽的雙眼，把我們和那些高風險的危險事物分割開來。

底線是一種提醒，底線之上，是艱難卻終有出路的陽關大道，底線之下，是看似容易卻暗藏玄機的危險小道。

守住底線，便是避開了那些潛藏的陷阱，或許走得很慢，卻能走得長遠。

三，堅守道德底線，守住良心，才會活得坦然，活得輕鬆。

《易經》：「善不積不足以成名，惡不積不足以滅身。」

多多做善事，自然有好報，但行好事，莫問前程。

多行不義必自斃，做人必須要有自己的底線。

一個人，沒有道德底線，就什麼壞事都敢幹。壞事做多了，害人害己。

古訓說：「不是不報，時候未到。」

一個人毫無底線，為所欲為，就算能一時呼風喚雨，卻終會隨時受到報應。

人在做，天在看。別看現在鬧得歡，小心將來拉清單，無論出於什麼動機，都不要急功近利、不擇手段。

想人家好，自己才會好。

做人不必有遠大理想，但是一定要有道德底線，做到「心中有畏」和「心中有戒」，就可以守住底線。

堅守底線，有時候更是守住良心。

堅守底線就是要喚醒我們內心深處潛藏著的良心，竭盡所能讓我們自己的人生問心無愧。

人生在世，當常立原則，長思底線，守住底線，也是守住了心靈的一片凈地。

堅守底線，是一種智慧，取捨有道，才是踏實而幸福的人生。

四，做人處世不逾矩，才能活出真正的自由。

不論你有什麼樣的活法，冷暖自知，但是底線這個好東西，不論何時，希望你有。

底線不會讓我們立刻快樂起來，卻會讓我們活得更有尊嚴，而在漫長的生命體驗中，尊嚴是最終極的快樂。當你學會堅守底線，旁人才能學會止步於你的底線之前;當你學會堅守底線，驀然回首之時，你的手裡才不會握著一把爛牌。

堅守底線，你不一定得到了全部你想得到的，但你所得到的，一定不是你不想得到的。

認認真真做事，清清白白做人，對得起天地良心，於人於己問心無愧，對父母盡孝，對朋友盡義，對事業盡忠，就會一輩子活得坦然，活得輕鬆，活得有模有樣。
