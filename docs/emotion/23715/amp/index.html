為尋子，鞍山母親舉著尋人啟事四處奔走；54天後，娘倆終於團圓

天天要聞
2019-12-19

2019年10月18日，鞍山岫岩的東子出門之後一直沒有回來。東子患有精神分裂症，一直和母親相依為命，他走丟了之後，年過六旬的老母親更是日日憂心。徐大娘將兒子的照片印成尋人啟事，四處散發，但卻遲遲沒有兒子的音訊。

東子今年41歲，自從9年前患了精神分裂症之後，他總是不愛在家裡呆。「他以前學了電焊的手藝，還能掙些錢，但是後來患了這病，也沒法工作了。」徐大娘提起兒子總是很心疼，丈夫離世後，兒子也患了病，只剩她一人撐起這個家。除了照顧著兒子的起居生活，她還一直打零工攢錢，只為了給兒子治病。

10月18日，徐大娘忙完回家，發現兒子又走了。「他以前都是出去一兩天，就會回來的，但是這次一直沒回來！」轉眼一個月過去了，徐大娘只能天天拿著兒子的照片到處尋他。

視頻截圖，走失者已被找到

11月19日，鞍山的王大偉警官得知東子走失的情況後，聯繫了頭條尋人，希望通過網路進一步擴散尋人啟事，接力合作，幫徐大娘與兒子早日團圓。

頭條尋人工作人員得知消息後，隨即發布了尋人啟事——《岫岩急尋：41歲精神障礙男子走失，1米82，鬍子長，上穿黃色迷彩服、下穿黑色長褲，綠色解放鞋》，並將信息推薦給鞍山岫岩的部分今日頭條用戶。

與此同時，王大偉警民合成作戰室的熱心民警也在幫忙尋找，根據民警們實地探訪，有群眾反饋，11月24日前後，曾在紅旗大山咀村發現過東子，但他離開後，又不知所蹤了。

「我兒這次出門，身上沒帶一分錢，他一直喜歡低頭走路，可能不會走遠。」徐大娘天天出門尋找，可一直追不上兒子的腳步，她沒想到，東子其實已身在60公里外的庄河。

原來，12月9日，庄河市民政事務服務中心救助服務部聯繫了頭條尋人，希望幫一名剛收助的流浪男子尋找親屬。頭條尋人發布了尋親啟事——《尋親：中年男子在大連莊河被救助，疑似精神異常，穿軍大衣》，恰好被王大偉警官看到，他發現，這名受助者就是從岫岩走丟的東子。

「這一次他整整在外面走了54天！」徐大娘得知的兒子蹤跡後激動不已，她對頭條尋人工作人員說道：「太感謝大家幫忙了！」接回東子後，徐大娘將兒子送往當地的精神病院治療。「人能回來就好！」年過65歲的徐大娘再次看到不禁紅了眼眶。

王大偉警官也第一時間將尋人成功的喜訊告知關注此事的好心人，王警官留言中說道：「感謝關注參與尋找的愛民擁警志願者，感謝今日頭條，感謝庄河警方及庄河民政救助站！」

這是今日頭條尋人公益項目自成立以來，成功尋找到的第12066名走失者；也是頭條尋人與各地救助站合作後，聯手幫助找到家人的第9145名受助者，也是頭條尋人與各地警方合作後，成功找到的第1887位走失者。

頭條尋人是一項面向全國的免費公益項目，致力於幫助各類家庭尋找走失者，其原理為在走失者失蹤地點附近彈窗尋人信息，藉助頭條龐大用戶，極大提升可能目擊者幫助尋人的幾率。過往成功案例已經證實，頭條尋人的精準地域彈窗對短期內走失的老人、精神障礙患者等移動能力較低的人群，有較高的尋人成功率。郵箱：xunren@toutiao.com
