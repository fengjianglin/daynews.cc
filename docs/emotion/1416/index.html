1.人有的時候真是奇怪，選擇了的會後悔，放棄了的會遺憾，但是完美只能是一種理想，而不可能是一種存在。

2.「熬過今天」是解決一切難題的咒語，其實熬著熬著，不知道什麼時候也就熬過來了。後來想想，這就叫「堅持」吧。

3.請不要把陌生人的些許善意，視為珍稀的瑰寶，卻把身邊親近人的全部付出，當做天經地義的事情，對其視而不見。

4.到底有多少人到現在還是不明白，人和人之間想要保持長久舒適的關係，靠的是共性和吸引。而不是壓迫，捆綁，奉承，和一味的付出以及道德式的自我感動。 ​​​

5.喜歡那些對我始終有所反饋的人。說出來的話能得到回應，表達的情緒能被在意，發出的信息可以收到回復，好像再渺小的事也有了意義。 ​​​

6.願時光能緩，願故人不散；願你惦念的人能和你道晚安，願你獨闖的日子裡不覺得孤單。 ​​​

7.真正的修行不是逃離，不是躲避，而是欣然的面對，全然的接受，接受此刻你正在經歷的一切好與壞。 ​​​

8.不要仗著寬容就肆無忌憚 有些人看起來好像是原諒你了但其實是因為你已經不那麼重要了 ​​​​

9.如果你一事無成，再多溫柔也是廉價的！——這話不分男女。人對於另一半的需求，一半來自情感，一半源自虛榮。你之所以被深愛，除了感情上的付出，一定還是值得被仰望的。能力是永遠開不敗的魅力。你的優秀，值得被寵。 ​​​​
