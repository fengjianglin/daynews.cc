你認為這是一個約會的好地方，但是在女孩子眼中，跟你約會的過程就是一種折磨。

跟自己喜歡的女孩子一起漫步在街道上，這個確實非常的浪漫。但是這個真的要分情況了：如果女生穿了高跟鞋，你還帶人家壓馬路的話，可能就是一臉的黑人問號了。表面或許不說，但相信此時女生內心早就已經在心裡「開罵」了吧~
