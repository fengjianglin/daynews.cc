不再去追求那些所謂的空虛享受

現在，我明白了

才是身體最重要的

從前，總喜歡各種熱鬧

喜歡那些華而不實的東西

現在，我明白了

平平淡淡才是真

陪伴是最可貴的

一間書房、一本書

閑庭養花

歲月安好

從前，有過的糾葛慢慢都模糊

那些說不清楚的愛恨情仇

慢慢都隨風去吧

有時候放過別人

也是放過自己

我對你付出的青春

這麼多年

成全了

成全了

成全了

你的今天與明天

成全了

我的下個夏天~
