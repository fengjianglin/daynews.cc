語文閱讀感悟，我們已上路很遠，你還有夢嗎？誰偷走了我們的夢？

之所以會發出這樣的疑問，是因為發現自己這些年一直在路上，好像自己已經丟掉了自己的夢想，自己到底想要活成怎樣的樣子？似乎自己已經不敢奢求什麼了。

人總是矛盾的，或者說就生活在不斷生成的矛盾中，然後再去解決這些矛盾，或者說找到一個可以既可以不委屈自己又可以讓自己舒適的區域。

太過堅持夢想，有時候會碰的頭破血流；不肯堅持自己的夢想，有時候又讓自己有頗多的遺憾。

堅持自己的夢想，可以讓自己走得很遠；能夠接受別人的「好言」，又可以讓自己少走彎路。

堅持自己的夢想，有時候會讓自己四處碰壁，甚至寸步難行；於是我們開始做出改變，讓自己順應生活，適應社會。於是自己就慢慢的放棄了自己的夢想。

堅持夢想有時候會輸得一無所有，浪費了精力，浪費了自己寶貴的青春；放棄夢想，做出改變，也許會讓自己柳暗花明，或者可以生活的更好。

以前同學說，我們改變不了生活，我們就改變自己。這就是適應生活，或者說在慢慢的放棄夢想。當然，我們可以委曲求全，先讓自己活下去，等著合適的機會，合適的土壤，有一天，讓自己內心的種子再次開花，綻放自己的理想。

​堅持夢想沒有錯，適應生活沒有錯。我們要守住自己的夢，但也不要去偷別人的夢。每個人的夢都很寶貴，我們應該讓別人自己去選擇。這似乎有點難，每個家長不是在偷孩子的夢呢？但是我們理解為「這都是為你好」，我們理解為一種愛；或者說「你的想法太幼稚，不夠成熟，我們過的橋比你走得路還多」雖然家長有時候偷走了孩子的夢，但是發現，那些孩子生活的也不差，因為那些是父母的經驗的路。至少不會碰的鼻青臉腫。
