愛情是什麼？

　　有人說愛情是依戀，有人說愛情到最後都是變成親情，也有人說愛情只是一種化學物質，它什麼都不是。

　　在小編看來，愛情是兩個人的相濡以沫，是長久的陪伴。

　　之前在網路上有一名網友寫出了自己跟男朋友之間的愛情故事，感動了許多人。更是被眾多的網友評為最美的愛情。

　　這名網友是一個剛滿20歲的女孩子。

　　她說在20年前，她只是路邊的一個棄嬰而已，來歷不明父母不祥，如果沒有意外的話，也許她就凍死在路邊了。

　　但是意外的是有一名18歲的男子經過了那裡，聽見了她的哭聲，把她抱回了家裡。男子來自偏遠的農村，在一線城市打工。他還沒有結婚，更沒有自己的孩子。

　　然而男子卻不忍心這名棄嬰在路邊就這樣活活凍死，於是他決定撫養這個孩子。為了養活孩子，給孩子買奶粉，他只好放棄了他原來穩定的工作，在路邊擺了個小攤，一邊可以照看孩子，一邊還可以賺錢。

　　消息傳回了老家，他的父母強烈反對他收養這名孩子，原因很簡單，這名男子至今未婚，卻帶著這樣一個孩子，以後想要結婚就很困難。哪家的女孩子願意嫁給這樣未婚卻帶著孩子的男人呢？

　　但是這名男子卻非常倔強，堅持要把孩子養下來。

　　就是這樣，憑藉著這名男子的辛苦撫養， 小孩子漸漸長大了，而這名18歲的男子也一直都沒有結婚，雖然他很渴望擁有自己的一個家庭，但是很多女孩子卻嫌棄他帶著一個拖油瓶。

　　後來男子憑藉自己的生意頭腦，創業成功。在城市裡面立足了腳跟，買房買車。這個時候有很多女孩子願意跟他一起過日子了，但是男子卻擔心這些女人只是看上他的錢，並不會好好的善待他收養的孩子，因此一直都拖著終身大事，並沒有結婚。

　　轉眼之間18年過去了，當年收養的那個丑巴巴的女孩，如今已經長得亭亭玉立，男子一直供她上學，讀書，成績也非常好，非常優秀。

　　然而，兩個人從最初的親情卻漸漸轉變成了愛情。女孩子非常依賴男子，無論在哪裡讀書，她每周都要回家一次。她自己說：只有在他身邊，我才是最有安全感的。他給了我最好的一切，寧願放棄自己的婚姻，也要把我養大。我不知道還有什麼樣的感情可以超越他對我的感情。至少在我眼裡，這就是最好的愛情。

　　而女孩子對於男子來說，從最開始收養只是同情憐憫，到後來漸漸的變成了憐惜疼愛。也讓孤身一人在城市裡面打拚的男子，漸漸感受到了溫暖和依賴。如果沒有女孩，他不會更拚命去賺錢，不會去創業，也不會有今天的他。

對於他們兩個人來說，這是最本真的親情，也是最美好的愛情。
