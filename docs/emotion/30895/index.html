人走到哪裡，都是行走於個人經歷之中

2019-12-24 1,119條評論 2,808次閱讀 323人點贊

一個完整而穩固的自我，能夠幫助你真正走進各種關係，並從關係中療愈自我，不再困於自己的世界，無人回應。一個真正做自己的人，就是在和他人相處時，能夠獨立自主思考的同時，也能和他人保持情感連接的平衡。放飛自我，很想做自己，卻表現得過於激烈，無法考慮到別人的感受。這種人自我是萎縮的，內在缺乏一個清晰穩定的個體。只有自我分化功能良好的人，才能分清情緒和理智，才能夠在他人的處境當中溝通作合時宜的事情，做到讓自己和讓他人都舒服。自我比較穩定的人，自然能夠與別人相處中保持靈活的距離，做到比較清晰和理解分析，能夠調整自己，合理的處理情緒做出合理的判斷。這些只有通過你不斷刻意的練習，才能轉化為技巧，才能對你的內在特質產生掌控和改變，真正激發出你內在競爭力，成獨一無二的充滿魅力的自己。

把最真實的自己袒露給別人多了，慢慢的就開始掩蓋了，久了就會發現很多事情都掩蓋不了。

「依戀外界失敗，除了會收穫一個不夠好的外界外，也會收穫一個不夠好的自己」，這麼強的因果敘述是在強調環境對自我成長的【決定性】影響嗎？那怎麼看待那些克服環境，最後也有很好的自我成長的人呢？還是只不過各人不同的性格問題，讓每個人的選擇不一樣呢？

一個人的成長，和後期接觸的環境和關係、自己的學習和覺醒，都有關係，每個人都有潛力成為自己

我就是這個樣子，別人看我很自嗨，其實只是因為我沒有太多的朋友，這兩天的內容里，都說到了我心坎里，其實我也想改變，我也想擁有一些長久的關係的人，真的，是說實話。

關係之所以能成就自我，也許是因為自我是通常意義上的小我，而關係意味著大我。不同層次的我需要整合，否則就有可能成為孤島，被囚在小我中。
