5歲女孩父母離婚被爸媽拋棄，與年邁外公外婆蝸居2平方米小屋

天天要聞
2019-12-19

父母離婚最受害的就是孩子，許多年幼的孩子父母離異後都被爸媽拋棄，只能跟隨年邁外公外婆生活。

這個越南小孩的名字叫阿娜，今年5歲，當許多小孩在這個年紀都是父母接送著去上幼兒園的時，阿娜卻因她的父母在她小時候離婚，她被拋棄跟隨年邁外公外婆在越南胡志明市第3郡小房子生活，由於家裡貧窮，阿娜無法前往幼兒園讀書，每天都只能蝸居在家。

平時的阿娜都是自己一個人在家，平日里，外公騎著摩托車去載客，而自己的外婆則去家裡附近的餐館打工。

家裡的玩具衣服等都是鄰居或者是相識的人給她的，而阿娜也一直很懂事，她說前段時間自己的母親還經常來看她，而現在都沒有來了，她想肯定是媽媽在努力賺錢養自己。

阿娜的奶奶叫卡秀玉，他表示，阿娜在1歲的時候，自己的女兒和女婿離婚了，一年之後，女兒嫁給了新的老公，然後就把阿娜帶回家給他們。自己女兒剛剛嫁人的時候還經常回來看望她，每次都給10-20萬越南盾，但是現在她生了新的孩子之後就再也沒有來看過阿娜了。

而阿娜的外公外婆每天賺的錢也不多，只能夠吃飯，他們並沒有多餘的錢給孩子去上學。

不過卡秀玉表示阿娜馬上六歲，直接送去附近小學讀書，近期為了使阿娜能進入一年級學習，卡秀玉帶著阿娜常常出去，與小朋友一起玩。

第5郡人民委員會陳姓主席說，卡秀玉一家最早居住在第5郡，之後他們家買了房屋搬走去其他均，不過多年後他們還是搬回第5郡，當時人委會知道她的處境很艱難，因此動員了當地人將空置的房屋給她住。第5郡表示對於目前阿娜這種情況，他們會幫助阿娜，讓孩子能夠早日有學上。

最近在胡志明市舉行的一次兒童保護和照護會議上，胡志明市教育大學講師阮氏玉萍說，當家庭出現暴力或父母離婚後，孩子會沒有愛，沒有家人的支持。從那時起，兒童將有心理障礙，容易成為流浪兒童或離家出走並沉迷於遊戲，很難如正常兒童那樣長大，而阿娜這種情況就是其中之一。
