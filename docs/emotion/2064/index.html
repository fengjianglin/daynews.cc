有酸甜苦辣的滋味，人生才不乏精彩！早安

2019-11-24 0條評論 21次閱讀 0人點贊

摘要

真正重要的不是生命里的歲月，而是歲月中的生活。人遠比自己想像的要堅強，特別是當你回頭看看的時候，你會發現自己走了一段自己都沒想到的路。只要內心不亂，外界就很難改變你什麼。不要艷羨他人，不要輸掉自己。順境善待別人，逆境善待自己。人生路上，要學會善待他人，也要懂得善待自己。

真正重要的不是生命里的歲月，而是歲月中的生活。人遠比自己想像的要堅強，特別是當你回頭看看的時候，你會發現自己走了一段自己都沒想到的路。只要內心不亂，外界就很難改變你什麼。不要艷羨他人，不要輸掉自己。

順境善待別人，逆境善待自己。人生路上，要學會善待他人，也要懂得善待自己。善待他人，可以讓人生走得更遠；善待自己，可以讓生命活得滋潤。無論是善待誰，其實都是溫暖在流轉，都是愛在延宕，最終，施及別人，惠澤自身。

人總要學會看得開，放得下。說起來可能容易一些，其實做起來，也沒有那麼難。很多事情，擱在心裡，你不去釋然，它也未必會有所改變。你若糾結成一個打不開的死結，它也依舊不改已有的模樣，那到頭來豈不是白白折損了自己的心情。很多時候，過不去的，不是事情本身，而是我們自己。早安！

淡然，得與失，成與敗，浮沉之間看紅塵；豁然，人生看的是風景，生活賞的是心情；釋然，歲月是一場跋涉，人生是一盤棋局；安然，情沒了就彆強留，人走了就彆強求；隨然，風雨要自己擋，滋味要自己嘗；欣然，人生有五味，生活有百態。有酸甜苦辣的滋味，人生才不乏精彩！早安！

人最悲哀的，並不是昨天失去得太多，而是沉浸於昨天的悲哀之中；人最愚蠢的，並不是沒有發現眼前的陷阱，而是第二次又掉了進去；人最寂寞的，並不是想等的人還沒有來，而是這個人已從心裡走了出去；人最無聊的，並不是隨便撥通一個電話號碼，而是別人撥錯時，他還一直的說下去。

生活的真理我比誰都還懂，但是有的時候還是會發生那些少不了的衝動，是因為還年輕，還是我真的有點弱智、不夠穩重成熟。只有我們通過努力的爭取才能給那些冷漠的眼神加已反擊。早安。

生命的豁然總會隨著第一縷明媚到來。每一次的失敗，都是成功的伏筆；每一次的考驗，都有一份收穫；每一次的淚水，都有一次醒悟；每一次的磨難，都有生命的財富。

要珍惜自己愛護自己，也是珍惜所有在一起的時光；是要珍惜花開時候的絢麗奪目，也是要珍惜紛紛花落時候的從容和自如。我們的一生，都是經歷了太多的事，總有一些會留下來的，到老的時候，可以慢慢地回憶。早安。

生命是一連串的選擇，我們可以選擇抱怨和指責；也可以選擇努力經營和投資。選擇了什麼，就會經歷什麼。人生必須由自己的選擇而決定。只能祝福，在曦光來臨前，準備好讓自己生命的花朵迎接希望。早安。
