一個家庭2個兒子，媽媽喜歡內向的哥哥，爸爸喜歡霸道的弟弟

自從國家二胎政策放開以後，很多家庭都覺得一個孩子太孤單了，於是二胎打著「陪伴老大」的旗號就出生了，兩個孩子雖然都是一個爸爸媽媽生的，但難免在性格喜好上會有所不同。今天我們說的這個家庭有兩個兒子，哥哥偏向內向，弟弟卻完全相反，他張揚霸道，這種性格反而得到了爸爸的喜愛。

爸爸說：我覺得弟弟這樣的性格在外邊不容易被人欺負，我更喜歡他。

兄弟二人雖然相差兩歲，但是穿的衣服都是一模一樣的，媽媽說：爸爸認為哥哥有的弟弟也應該有，所以為了不讓兩人爭奪，他們兩人的衣服都是買的一樣的。

有人在這個時候問到，萬一其中的一個孩子不喜歡這件衣服怎麼辦？媽媽無言以對。

兩個孩子想吃零食，媽媽在家裡翻找了一遍，只有一塊小麵包，這個時候媽媽說這個麵包只能一個人來吃，要不然你們兩個人猜丁殼吧，誰贏了誰吃，結局是哥哥贏了，拿到了小麵包。

爸爸這個時候回家了，他看到弟弟不高興的趴在沙發上，眼神還巴巴的看著哥哥手裡的麵包，爸爸就說：「麵包分弟弟一點」

哥哥在這個時候表現出了不願意一邊哭鬧一邊廝打著爸爸說：「這是我的，這是我的。」

但是因為力量小，麵包還是被爸爸撕了一半去拿給弟弟，哥哥生氣的把手裡剩下的另半塊麵包給扔掉了，也趴在沙發上委屈的哭著，媽媽從廚房裡走了出來，抱起哥哥說怎麼了？我明天我再給去買一點麵包就好了。

而弟弟因為麵包掉到了地上，爸爸也不讓他吃了，弟弟也不高興，爸爸這個時候摟著弟弟說：「好啦，好啦，沒事，爸爸明天去給你買，你還記得你生日的時候想要什麼嗎？」

很顯然媽媽更喜歡內向的哥哥，爸爸更喜歡弟弟。

媽媽說：他們的爸爸說過，只要是哥哥有的東西，只要弟弟想要，哥哥必須無條件無理由的分享你弟弟，我覺得這樣做對哥哥不公平，所以我對哥哥更好一點。

家裡有兩個孩子，爸爸喜歡弟弟，媽媽喜歡哥哥，打著「陪伴老大」出生的弟弟卻奪走了父愛

不知道你們有沒有想過這會造成什麼樣的局面呢？說到這裡，我想到了我一個同事，她說她的公公婆婆對待兩個兒子就不公平，公公喜歡哥哥，婆婆喜歡弟弟，他們倆在結婚的時候，婆婆把自己偷偷存的1萬多塊錢給了自己的老公，婆婆還囑咐自己的老公，這件事不要告訴你爸。

她說她跟老公結婚以後，老公很少回家，每次都是打電話叫媽媽過來，讓媽媽常住在自己的家裡，而他和哥哥也很少聯繫。

兩個孩子出生在一個家庭，父母卻不能一碗水端平，後果是很嚴重的，既然在孩子還小的時候，媽媽就已經發現了爸爸偏向於弟弟，那麼在這個時候你只是偏向哥哥是不對的，此時你的做法應該是和老公說明情況，手心手背都是肉，不能偏向某一個人。公平的對待兩個孩子，才能促使你們的家庭更加的和諧幸福。
