“單身人士”最多的城市，大部分不想結婚生子，工資水平卻很高

天天要聞
2019-12-20

現如今我國的經濟發展的速度已經越來越快，科技也融入到了人們的生活中，帶來了非常多的便利，而人們生活水平提高的同時，壓力也在不斷增大，這一情況體現在很多方面，就比如說是工作中的不順心以及家裡人的催婚，還有年輕人們經常會面對的結婚買房養孩子問題。

其實這是很多人的生活現狀，大部分都是在大城市工作，卻沒有穩定的住所，只能靠著一步步的打拚積攢更多的積蓄，但對於結婚生孩子之類的問題就是避而不談，反而會比較注重自己的精神生活，平時跟著朋友們一起出去旅遊，或者是參加更多有意義的活動，都比脫單要重要很多。

久而久之的也就形成了人們口中的單身狗群體，那麼你知道我國單身人士最多的城市又是哪裡嗎？它就是魔都上海，很多人剛聽到就會覺得很詫異，但同時也會覺得非常合理，確實上海這座古老而又現代化十足的城市，因為早早的就接觸西方文化，所以人們的思想觀念非常的開放，更注重自己的精神生活。

而且這裡的工資水平也特別高，人們在有了一定的積蓄之後，更願意投資在自己身上，去外面見識一下更廣闊的世界或是用在一些有意義的事情上，即使是月光族也會覺得非常幸福，雖然慢慢的大齡剩男剩女越來越多遍地都是，但基本上也是有實力又有顏值，不會畏懼社會上的條條框框。

可能在老一輩人的印象中，如果年紀大了再不找人嫁出去，就可能再也沒有人要，但既然人們的思想觀念已經發生了變化，就不會再去服從家裡的安排，大部分都不願意結婚生子，尤其對女生們來說，結婚之後可能要付出的時間和精力要比男生多很多，所以寧願單下來讓自己活得更精緻。

所以說環境以及思想的改變，確實會影響人在很多事情上的選擇和觀點，那麼大家對於此事有什麼想要表達的意見嗎？歡迎在下方留言評論
