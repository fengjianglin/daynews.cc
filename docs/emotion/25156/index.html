找不到都別回來過年！兒子走丟，老父母責備女兒女婿，如今終團圓

2019-12-21 604條評論 2,325次閱讀 296人點贊

「本來想把小舅子從重慶老家接來威海照顧，結果他走丟了！」老楊自責不已，2019年春節後他把伍子接來，快半年的時間伍子都平安無恙，但沒想到8月29號他出門後一直沒回來。「我岳母在老家天天哭，還和我們說，找不到他你們都別回來了！」老楊苦苦尋找110天，每日都遺憾而歸。

圖片來源：媒體

直到12月17日，頭條尋人工作人員給老楊帶來一個好消息，使用「識臉尋人」之後，發現伍子在煙台被救助了！

一人走丟，成了全家人的心結

伍子今年36歲，因自幼患智力障礙，他沒讀過書，此前也沒出過遠門。年過七旬的老父老母一直在重慶秀山的老家照顧著伍子，2019年春節後，兩位老人決定讓女兒女婿將伍子帶去威海。老楊考慮到兩位老人照顧兒子辛苦，也想著能帶伍子去城市裡見見世面，就同意將他帶著身邊照顧。

來到威海後，伍子有些不敢出門，平時總喜歡呆在家裡。8月29日這天，老楊回家發現伍子不見了，他和妻子急得四處尋找，但卻尋不到他的蹤跡。

「我小舅子特徵很明顯，他雖然只有36歲，但是滿頭白髮。」老楊以為伍子的特徵明顯，應該很好找，但遺憾的是，這裡不同於農村老家，周圍居民不認識伍子，沒有人留意到他的蹤跡。直到9月20日，老楊得知可以通過頭條尋人發布信息，急忙將伍子的信息提交在今日頭條上。

頭條尋人工作人員得知消息後，隨即發布了尋人啟事——《威海急尋：青年智力障礙男子走失，滿頭白髮，穿粉紅色襯衣短袖》，並將信息推薦給威海的部分今日頭條用戶。

「老人說氣話的時候，給我講如果找不到他，我們也不要回老家過年了。」老楊又自責又焦慮，他聯繫了許多朋友幫忙擴散消息，大家也安慰老楊，耐心等一等，說不定過一陣就能找到了。時間一天一天過去，卻還是沒有伍子的消息，他經常接到岳父、岳母從老家打來的電話，聽到老人家哭得傷心，他卻覺得無能為力：「我們實在不知道該去哪裡找他了！」

離家60公里，識臉尋人發現他

轉眼就到了12月17日，老楊正在忙著幹活，突然接到頭條尋人工作人員小王的回訪電話。「一直沒有找到啊！老人在家裡急得天天哭，請頭條再幫忙找找吧！」老楊得知小王通過人臉比對發現一個匹配度很高的結果，趕快打開微信辨認照片。

「是的是的，就是這個人，麻煩你跟我說一下，你在哪裡找到的！」老楊激動不已，他趕快給岳父打電話告知這個好消息。「我們一家人都太開心了！真的沒想到，他都從威海走到煙台去了！」

原來，9月30日煙台救助站聯繫了頭條尋人，希望幫一名流浪人員尋找親屬。「受助者在萊山的一個小區附近流浪，將他安置在站內之後，我們嘗試了很多辦法想和他溝通，但是他口齒不清，我們看他滿頭白髮，可能是有50多歲了。」

沒想到，這則尋親啟事——《煙台尋親：五旬男子被救助，自述叫吳某，高1米65》的受助者，竟然真的是伍子！

「其實他姓伍，不是姓吳，實際年齡是36歲，可能在外面流浪得比較久，加上頭髮全是白的，大家都以為他50多歲了。」老楊得知情況後，立馬驅車趕往煙台救助站。

老楊對頭條尋人工作人員說道：「我印象里都沒見他哭過，我去接他的時候，他還是記得我，一下就哭了。」老楊心疼極了，看到伍子像個孩子一樣流淚，他也紅了眼眶。

離家110天，相距60多公里，伍子終於和家人團圓了。「我們這邊的活就快忙完了，能趕在春節前帶他一起回老家，我們一家都非常感謝今日頭條！謝謝你們啦！」辦理完接領手續，老楊牽著伍子離開了救助站，將他帶回家中悉心照顧。

頭條上線「識臉尋人」功能

「識臉尋人」背後是成立於2016年的位元組跳動人工智慧實驗室。其原理是通過接入實驗室的面部識別技術，利用AI演算法，將用戶上傳的走失者照片與尋人資料庫的走失人群大數據進行比對，更高效地幫助走失人員回歸家庭。

2016年2月上線至今，「頭條尋人」已經積累了大量走失人員信息，形成7萬多例尋人資料庫。此外，「頭條尋人」已與民政部及其下屬的1976個救助管理機構對接，整理髮布了約4.5萬條受助人員尋親信息；並與公安部及各地警方、醫療機構、新聞媒體、志願者組織合作發布約2.5萬條尋人信息。目前頭條尋人已發展成為國內最大的公益尋人平台之一。開放頭條尋人資料庫，更高效地幫助更多人尋親並圓夢，讓技術和信息不再僅僅是冷冰冰的數據，這正是「識臉尋人」功能上線的初衷。

這是今日頭條尋人公益項目自成立以來，成功尋找到的第12083名走失者；也是頭條尋人與各地救助站合作後，聯手幫助找到家人的第9158名救助站受助者；也是通過人臉識別幫助找到的第60名救助站受助者。

頭條尋人是一項面向全國的免費公益項目，致力於幫助各類家庭尋找走失者，其原理為在走失者失蹤地點附近彈窗尋人信息，藉助頭條的龐大用戶，大大提升可能目擊者幫助尋人的幾率。過往成功案例已經證實，頭條尋人的精準地域彈窗對短期內走失的老人、精神障礙患者等移動能力較差的人群，有較高的尋人成功率。
