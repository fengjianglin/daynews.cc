俗語：“寧願鄰居養條牛，不願鄰居出諸侯”，你會這樣想嗎？

2019-12-23 574條評論 1,817次閱讀 327人點贊

導語：俗語：「寧願鄰居養條牛，不願鄰居出諸侯」，你會這樣想嗎？

人生活在這個世界上，就不可避免的要和人接觸交往，必須將自己融入一個又一個的圈子，這些圈子中除了我們的家人，接觸最多的就是同事圈、親戚圈、朋友圈、鄰居圈。同事是工作選擇的，親戚是血緣關係選擇的，朋友是自己選擇的，鄰居是客觀生活選擇的，這些關係共同組成了我們最重要的生活圈，所以朋友、鄰居、親戚的話題一直以來也是人們討論的熱門話題，關於它們的說法也有很多，就比如小編就常常聽家裡老人說這樣一句話：「寧願鄰居養條牛，不願鄰居出諸侯」，意思是寧願鄰居家裡養了一頭牛，家境一般，也不願鄰居家裡出了大人物，飛黃騰達。為什麼會這樣說呢，接下來我們就一起來看看。

寧願鄰居養條牛

一、鄰居家的牛可以借過來用，如果鄰居家有頭牛，那麼農耕時還可以借過來幹活。之前在農村的時候，不可能每家每戶都有牛的，所以一般都是有一家養牛，然後旁邊的幾家在種地的時候可以用，而且放牛也是全村集中起來，輪流放牛，這樣既方便也可以減少勞動力，要不然每家的牛都得自己放，豈不是每天都要花費時間。

圖片來源於網路

二、交往中讓感情更加深。鄰居借給你牛，在鄰居需要幫忙的時候你也會主動伸出手來，正是在這樣一來一往的幫助中才使得鄰居關係更加的親密。俗話說「遠親不如近鄰」，鄰居的重要性正是在於距離比較近，可以很及時地互相幫助。鄰里之間的生活水平相當，那麼彼此之間就可以你幫幫我，我幫幫你，這樣有來有往，彼此間也不會有什麼虧欠，而且這樣的相處模式更令人放鬆，無所顧忌。

鄰居有牛，牛是實實在在存在的東西，所以所能接觸到的好處也是就在眼前的，但是鄰居富貴了，對於自己來說這是一個比較虛擬的東西，你並不會知道自己會不會從鄰居富有得到好處，所以寧願鄰居養條牛。

不願鄰居出諸侯

一、朋友圈的差異，常言道：「物以類聚，人以群分」，任何人之間的交往是建立在平等的基礎上，這裡的平等不僅僅說的是人格的平等，更重要的是經濟基礎的平等，身份和地位的平等，所以有錢的人很難和貧窮的人交往，當官的也很難和平民百姓交往。殺豬的和買煎餅的能成為好朋友，白領和白領能成為朋友，老闆和老闆會交往，但是殺豬的卻很難和老闆交往，這就是經濟基礎和地位決定的人的朋友圈，所以要是你的鄰居突然有錢了或者是當了大官，一開始或許還會有交往，但是之後這個關係就會迅速瓦解，成為陌生人。

二、自卑心理在作祟，鄰居算是在你周圍距離你最近的距離了，差異總是通過對比才能顯現出來，遠房親戚的富有並不會顯示出你的貧窮，但是鄰居的富有卻能襯托的你更加貧窮，所以就會更加自卑，害怕會被他們輕視，看不起。就算是鄰居還願意和我們交往，每日看著他們的生活，你就會覺得你們完全是兩個世界的人，在與他們相處時，不自覺地將自己放低一個層次，最後我們也會因為自卑心理而自動遠離。

三、嫉妒心裡在作祟，不可否認的是每個人都會有嫉妒的心理，但是這樣心理的程度是非常主觀的，也是通過對比才會更加明顯。就好像世界首富本來就距離你很遠，所以你並不會因為他的更加富有而產生嫉妒心理。但是鄰居不一樣，正是因為曾經有交往，大家的各方面水平也差不多，要是突然之間有一方變得富有，那另一方一定會產生嫉妒的心理，正是因為嫉妒心理才讓我們天然的不希望鄰居富有。

所以說「寧願鄰居養條牛，不願鄰居出諸侯」這句俗語還是有些道理的。

小編有話說：人是社會動物，一個人總是生活在一個相對有限的社會環境之中的。鄰居是你身邊的人，是跟你有比較近的「比較」上的聯繫，所以你就會自然而然的進行對比，如果比自己好了就覺得心裡不快，如果沒有自己好就心裡舒坦，人的快樂都是對比出來的。其實大家遇到「鄰居出諸侯」的情況，完全可以轉變一下自己的心態，為什麼同樣的水平鄰居家裡就能富貴、出能人，為什麼自己家就不行，從自己的身上找找毛病，看看是在哪裡有所欠缺，就在哪裡改過，將出諸侯的鄰居看作自家的目標。也不必有自卑心裡，畢竟人和人的境遇就是不一樣，過好自己的生活，走好自己的路就行。
