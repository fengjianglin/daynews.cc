為什麼有些人越交越厚，有些人分道揚鑣？不相向而行，就漸行漸遠

2019-12-24 3,154條評論 4,232次閱讀 650人點贊

生活中，有些人越交越厚，越交越透，成為一輩子的朋友。有些人越交越薄，漸行漸遠，最後分道揚鑣，成為路人。

說起朋友，每個人都可以列出一長串名字。但如果問起有多少交了一輩子的朋友時，很多人恐怕一隻手都能數的的過來。

我們和哪些人的友誼沒有一直走下去？想一想，大概有三種人吧。

一有錢就變臉的人

人常說，患難見真情。有些人有錢了，會變得更加低調。可有的人一有錢就變臉，翻臉比翻書還快。

我的一個同學，當初停薪留職時，騎個自行車正天奔波在大街小巷，樓上樓下，挨門推銷，看盡了人臉，說盡了好話。我還記得，當初在他貸不到款的時候，是我為他做的擔保，才解了他的燃眉之急。當時他許下願說，以後我有事用錢的時候，就是一句話的事。

其實，我也根本沒把這當回事，想著同學之間，能幫的忙就幫了，沒有那麼多客套。而且，這種話，他給許多幫助過他的人都說過。

可後來，當一些曾經幫助過的人找到他幫忙時，都被以各種理由回絕了。

許多人說，人一有錢，「勢」都大了。人有錢了，說話的態度和方式也變了。他不再對你和顏悅色，而是吆五喝六；他對你不再謙遜，而是一副高高在上的樣子。他可能覺得，錢撐硬了他的腰杆子，全然忘記了別人對他的好。

既然你已不是原來的你，也就沒要維持這份友誼。

把誠信當擦屁股紙的人

缺誠信，不可交。

網上不是常說嘛，要考驗一個人的誠信程度咋樣，借錢一試便知。

的確，現在，往外借錢是一項高風險的事情，借錢不還的人大有人在，稍不留意就可能掉進坑裡，還會結下仇恨。

前幾年，一個曾經的朋友遇到了困難，找到我借錢救急。想一想也是認識多年的了，雖然沒有經濟上的過度來往，但感覺還好吧，於是，就借給了五千塊錢。

說好的半年時間歸還，可沒想到是肉包子打狗一去無回。要了幾回後人家還急了：「不就是幾千塊錢嗎，我承認欠你的，但我沒錢還。」完全耍起了無賴，全然沒有了朋友間的誠信。對於這種耍流氓的做法，我當然是不會縱容的，最後以適當的方式要了回來，但兩人的友誼從此也掰了。

誠信是立世之本，不單單是借錢，生活中的任何事情都要講誠信。但有些人偏偏拋棄了這兩個字，把誠信當成了擦屁股紙。

既然你已丟失了誠信，這份友誼也就失去了價值。

只往裡彌不往外補的人

生活中，每個人可能都有過這樣的經歷，就是有人不論他家的大小事情，都給你通知，讓你參加。而這去了，就要隨份子。可當你有事了，他就像人間蒸發了一樣，裝聾作啞。

我的一個高中同學，他家的事沒有不通知人的。兒子結婚、父母去世都應該去，可他連他弟弟食堂開業、自己買個車也要通知大家。有時候你覺得煩，不想去，他就讓別的同學通知。

可當別人有事通知他時，要麼不接電話，要麼就說人不在本地，基本上是來不了。有時候大家說：「你人來不了，把份子錢讓誰捎來也行。」可他總是說：「我要當面給人家，要不然顯得不尊重。」基本上是屬鐵公雞的，一毛不拔。

相信有過病住院經歷的朋友一定都遇到過這種事情，當你病好以後，碰見一些你曾經幫助的過的人時，總會聽到這樣的話：「我還說去醫院看你去，沒想到你都出院了。」「我一直說去看你，可最近事情特別多。」這其中有多少虛偽，只有你知他知。

這世界上，聰明人不少，但傻子也不多，但自認為聰明的人卻很多。

交朋友，兩顆心走不到一起，兩個人怎麼可能走到一起。

既然友誼不在，不如一拍兩散。
