50後、60後面對的煩惱：活半輩子不能享福，要照顧七八十的公婆

2019-11-26 0條評論 25次閱讀 0人點贊

摘要

作為年輕父母的一員，憧憬自己老了之後可以逍遙自在，孩子們都已經長大成家立業，自己跟老伴就可以遊山玩水，做自己喜歡的事情。每年可以去幾個地方，享受享受當地的美食，在當地住上幾個月。哪怕是這樣想想，也覺得自己的晚年生活是幸福快樂的。可憧憬是憧憬，憧憬往往會被現實所打破。

50後、60後面對的新煩惱則是：活半輩子了卻不能享福，還要照顧七八十的公婆。

自己已經是中老年人，家裡還有著「老小孩」。最近聽說張曉的家庭就是這樣，自己將近花甲了，本想著日後就可以在家裡平平淡淡的過了。

不用擔心兒媳、想帶著孫子孫女就帶幾天，不想帶就過著自己的慢生活。因為張曉的公婆今年是70多歲了，身體還很棒，只是日常的衰老。

因為公婆還要養著他們，如果他們生病了，張曉也要幫忙照顧。當然，我們也可以有其他的選擇，如果你的身體健康情況不太好，在公婆的養老上面，還可以選擇其他的方式。

如常見養老的是三種形式，主要有去養老機構養老、這種養老的方式是指上了年紀的人，可以在養老院養老。還有一種養老方式，則是通過家政等方式，在家裡可以度過老年生活。

最後還有一種養老方式就是社區養老，這種是通過相關的社區服務，從而獲得資源來讓老人養老。

大家可以根據自身的情況去選擇，給父母養老則是一件理所應當的事情。

50後、60後除了要面對公婆的養老以外，還要面對兒子兒媳婦帶孩子的問題。生兒子要娶媳婦，現在結婚的可能比較晚，但大多數人在父母50、60時歲的時候就已經結婚娶兒媳婦了。

對於50後、60後的人還要面對的一個問題就是帶孫子帶孫女，屬於上有老下有小的狀態，在這種狀態下，往往壓力也是非常大的。

所以，如果你是他們的孩子，現在已經有了兒子女兒了，就要多考慮考慮老人的感受，如果覺得父母非常辛苦，就不要讓他們太累了，好好做好兒女的責任。

在帶孩子上，父母是沒有必要責任幫你去帶的，如果父母因為某種原因無法帶孩子，也不要強求他們。再另外想辦法去解決，讓父母放心也是你應當做的。

如果你屬於「夾心」的狀態，你會如何去做呢？
