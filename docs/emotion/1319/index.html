她一生做好事無數，受到總書記親自接見，偉大的母親都是這樣教育孩子的

2019-12-05 0條評論 21次閱讀 0人點贊

摘要

她用30載愛心讓一村之中老有所終，幼有所長，鰥寡孤獨廢疾者皆有所養。富人做這等事是慈善，窮人做這等事是聖賢，官員做這等事是本分，農民做這等事是偉人。那是一個寒冷的冬天，窗外天寒地凍，陰暗破舊的小屋裡生起了暖暖的爐子，熱氣漸漸蔓延到屋子的各個角落。

她用30載愛心讓一村之中老有所終，幼有所長，鰥寡孤獨廢疾者皆有所養。富人做這等事是慈善，窮人做這等事是聖賢，官員做這等事是本分，農民做這等事是偉人。

那是一個寒冷的冬天，窗外天寒地凍，陰暗破舊的小屋裡生起了暖暖的爐子，熱氣漸漸蔓延到屋子的各個角落。林秀貞坐在兩位孤寡老人的床邊，老人的臉上露出了難得的笑容……

從這天起，林秀貞就擔起了兩位老人的衣食起居，時間飛快，一晃四年過去了，這天，劉秀煥老人把林秀貞叫過來，她說：「妮兒啊，不瞞你說，那時候俺就尋思著，有一天活不下去了，俺就和你六爺爺一起把這個一吃，吃完了一睡，死了就算了。」老人擦擦淚繼續道，「後來有了你了，俺們就算有了親閨女了，俺可不犯那個傻了，這安眠藥啊，俺就給你交出來吧」。林秀貞聽了這話，含著眼淚把安眠藥仍到爐子里，兩個人抱頭痛哭。

「人人都幫人，世上沒窮人，人人管閑事，世上沒難事兒。」

母親的這句教誨，一直深深的影響著林秀貞，她也是這樣做的。人們常說「久病床前無孝子」，很多人對待自己的親生父母尚不能做到如此，而林秀貞卻能30年如一日的贍養和自己沒有血緣關係的6位老人。林秀貞說：「母親教育我三條路走中間，不能斜，不能偏，一輩子只能做好事，不能做壞事。我一看見有困難的人，我不幫覺得心裡下不去」。

後來劉秀煥老兩口病重，72天里，林秀貞夫妻倆沒有睡過一個囫圇覺，有一次，林秀貞去廁所許久沒有回來，丈夫擔心她過去看，發現林秀貞已經在廁所里睡著了。癱瘓的老人每天都要換尿介子，林秀貞就把帶著大小便的床單和尿介子裝到簍子里，拿到河邊去洗，當時河裡已經結冰，她就把河面鑿一個窟窿，用刺骨的河水去洗。懷有身孕的林秀貞就這樣落下了中指和食指彎曲變形的毛病，在這72天里，她的體重不增反減。

林秀貞的大女兒回憶說：從她記事起，劉秀煥老兩口就住在她家裡，她始終以為這就是自己的親爺爺奶奶，長大一點後，她們兄妹就學著媽媽的樣子來照顧兩位老人。一天，哥哥在給老人洗腳，她看到了要過去幫忙，可是當她剛把手伸到洗腳盆里時，老人本能的抬了一下腳，洗腳水不小心就灑在了她的臉上，她沖旁邊吐了一口口水。這一幕被剛進門的林秀貞看到了，晚飯後，她悄悄的把兩個孩子叫到屋裡說：「妮兒，你這樣做不對，在伺候老人的時候，你不能吐口水，也不能皺眉頭，老人本來在咱們家裡就不落忍，你再這樣做，他們會感覺自己是個累贅。」聽完母親的話，大女兒感覺非常愧疚，她也深深的感覺到這些年來，母親的不容易。像這樣的教育天天有，時時有，這種教育讓孩子們受益終身。

後來，林秀貞辦起了玻璃鋼廠、橡膠加工廠，又創立了河北省棗強縣宏達防腐工程公司，生意做大了，日子好過了。富起來的林秀貞不忘鄉親，她無償為殘疾人提供玻璃鋼、橡膠技術培訓。自1998年起，林秀貞在自家小廠里安排了8名殘疾農民就業，使他們不靠天、不靠地，用雙手支撐起自己的生活。

為方便這些殘疾職工生活，林秀貞還專門請了一名炊事員，免費為他們提供一日三餐，送開水、搞服務。這些殘疾職工在林秀貞的小廠里工作得舒心，生活也有了勁頭。有位殘疾職工的父母都已過世，林秀貞自己掏了4000多元，為他張羅成親辦喜事，解決了他的終身大事。

也是在這一年，林秀貞到合肥洽談業務，在火車站候車時發現一名棄嬰，嬰兒患有先天性心臟病和小兒痴呆症。林秀貞把嬰兒抱回了家，花了8000餘元給他治病。她把孩子看得跟親生兒子一樣親，正式辦理了領養手續，給了孩子一個溫暖的家。

林秀貞熱心社會公益事業，自1987年以來，先後為鄉、村各項公益事業累計捐資3.91萬元。還出資4萬多元幫助鄉村中小學改善教育條件，資助14名貧困家庭子女步入大中專院校，幫助29名困難群眾走出困境。林秀貞對村裡的事更是熱心，捐款5萬元為村裡安裝自來水，受到群眾的高度讚揚。

今年74歲的林秀貞先後榮獲全國優秀共產黨員、全國「三八」紅旗手、全國「五好家庭」、河北省優秀共產黨員、2005年「感動河北十大人物」、2009年入選「100位新中國成立以來感動中國人物」，2019年9月25日，入選「最美奮鬥者」個人名單等榮譽稱號。

這位普通的農家大媽做出了不平凡的人生選擇，她的事迹讓人從內心深處受到一次洗禮，一種如何做人的洗禮，一種精神上的升華，一種共產黨員給我們的震撼……
