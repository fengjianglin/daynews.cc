“媽，電視音量能小點不？”單親媽媽留一封信，女兒看後泣不成聲

2019-12-03 0條評論 24次閱讀 0人點贊

摘要

小藍回到家中正在忙著公司的工作，媽媽正在看電視，小藍突然抬起頭，「媽，我正在做公司的工作，你能不能把電視的音量調小一點？」看到媽媽沒有說話，小藍撇了一下嘴，悄悄帶上耳機繼續工作。

過去的忘記

在小藍的記憶中，從來都沒有爸爸這個角色，不過，他有一位完美的媽媽，她還有一個任務，充當爸爸的角色，仍然將每件事都處理好。把女兒養育成人，這是她努力工作的唯一驅動力。每當小藍犯錯時，她會很厲害教養她，但是事情過後，又會抱著女兒哄她。小藍明白，她是媽媽唯一的愛，小小年紀就立志將來努力賺錢，照顧和報答媽媽。

大學畢業那一天，她穿著畢業服和媽媽合影，並對媽媽說，「媽媽，我一定不會忘記對你的承諾！」

小藍戴上耳機還是覺得吵鬧，徑直走到電視機前關掉了，「你把電視音量調這麼大，怪不得你聽不見我跟你說話！」。媽媽退休後，變得沉默寡言，經常不聽別人說話，有時答非所問。

公司處於業務發展的關鍵期，工作的壓力和任務讓小藍喘不來氣。在一次重要的簽約會議上，突然接到一個電話，「你媽媽被車撞到了！」她嚇得放下所有的工作，趕去現場。原來媽媽聽不到後面人的提醒，結果被小推車上的貨物砸到了腳踝。小藍要陪媽媽看醫生，被媽媽拒絕了，稱沒什麼大礙，還要回家做飯。看到媽媽的背影，她已經不知不覺間老了，看到她滿頭白髮，步履蹣跚，小藍在想，自己是否有足夠的能力照顧媽媽？

有一次下班，小藍推開門，媽媽很高興地對她說，「生日快樂」，平時的辦公桌上收拾得乾乾淨淨。小藍就詢問媽媽，「你沒有見我的一個文件，它非常非常重要。我早就告訴你了，不要動我的文件，為何你永遠都不聽我說的話呢？」找到文件後，小藍翻看文件，發現其中幾頁合約不見了，不小心把蛋糕碰到地上，「沒有這些文件，我會被公司開除的！我真不懂怎麼跟你交流？」小藍一氣之下離開了，媽媽蹲下來，把打翻的蛋糕翻轉過來，蛋糕已經碎得不成樣子了，媽媽表情凝重，有些哽咽。

小藍本不想對媽媽發脾氣的，不過失去了那份合約，她的工作也就沒了。她拚命工作，只希望給媽媽更好的生活，為何媽媽不懂呢？或許自己沒能力照顧她？

小藍回到家中時，發現媽媽不在家，桌子上留下一封信：藍藍，媽媽已經老了，耳朵已經聽不清楚，帶給你很多麻煩，我從垃圾站里翻出來你的合約，雖然無法給一個完整的家，但你很好，也很懂事，讀書也努力，終於在一家發展不錯的公司時努力掙錢。媽媽知道，你想給我更好的生活，其實，只要你在我身邊，已經是最好的幸福生活了。

小藍看到這裡已經泣不成聲，她衝出家門，找到了在外面獨坐的那個熟悉的身影，「媽媽，我們回家吧！」。

曾經，小藍以為是媽媽不聽她的話，原來聽不到的不止媽媽一個，還有她自己，忽略了老人的聲音。以後的日子，她會好好陪著媽媽，用心聆聽。
