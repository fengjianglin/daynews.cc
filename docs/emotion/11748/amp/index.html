圖像心理測驗：憑直覺選出圖中“最老的人”，看你是否容易被騙？

天天要聞
2019-12-10

點個關注，跟我一起，每日趣看心理看八卦，情感故事，還有各種趣味測試！

俗話說得好，害人之心不可有，防人之心不可無。現代社會是一個信息爆炸泛濫的時代，各種套路和騙紙層出不窮，你有被套路和被騙的經歷嗎？

憑直覺選出圖中「最老的人」

測試結果

選A的人

你是一個基本上沒有戒心的人，始終樂觀開朗，待人真誠。你喜歡交朋友，也喜歡和人分享你的心事，並且善於傾聽。

你是一個善良的人，並且深信人性本善，相信你很喜歡和他人吐露自己的心聲，也善於傾聽他人的心裡話。也許你並不天真，但是你是一個善良的人。

你的內心深處是相信人性本善的，認為這個世界還是美好的事物居多。而你真心對待身邊的每一個人，所以別人也會信任你，真心待你。

注意：太老實容易吃虧哦。

選B的人

你對他人的戒心非常重，並且性格敏感多疑，缺乏安全感，從不會輕易相信他人，不會讓別人走入自己的內心，對他人豎起高高的防禦牆。

你對人性從來都持悲觀態度，認為人性本惡，一切都是利益至上，任何感情都敵不過。你對於陌生人的接近極為警惕，會非常謹慎的處理，這可能源於你曾經受過這方面的傷害，從而導致產生了陰影。

其實有防備心理當然重要，但過度就不好了，而你的狀態已經讓你自己有些疲憊了，或許可以適當放鬆一下。

選C的人

你對他人有一定程度的戒心，你的性格較為內斂，不太擅長人際交往，如果不熟悉的人靠你太近，你可能就會感到不適和難受。

你對於親近的、信任的人會完全卸下防線，而對待不熟悉的人，你就會加以戒備。

並且你也是個很謹慎的人，說話做事都非常注意，不會向別人透露太多個人的信息，也會謹防別人套自己的話，這在如今複雜險惡的社會，是很有必要的。

選D的人

你的戒心不算是非常重，你是一個比較隨和的人，與人相處得都很好，雖然深知在與人相處時必須要有一定的防備心理。

但實際卻是你往往對人沒什麼戒心，或者說是你忘了這一點。迄今為止，你的運氣都還不錯，沒有遇到過對你有惡意的人，因此也就沒有受到過什麼傷害。

也有可能是你的善良讓人不忍傷害你。雖說如此，但我們還是要有基本的防備心理，要學會識別別人是否不懷好意，更不要相信莫名其妙接近你的人，保護好自己才最重要。
