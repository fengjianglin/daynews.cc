    • 01識人不清的根源在於沒有進行人際關係分層

    老丁在公司工作3年，不僅專業能力過硬，還為人和善，被部門裡年輕的同事親切的稱為「丁爸」，大家在工作中遇到什麼問題，也會第一時間找老丁求助，老丁也會予以相應的解答。

    前段時間，部門經理退休，領導決定在現有的優秀員工中進行篩選，綜合能力強的人可以被提拔為部門的負責人，當所有同事都覺得老丁是部門經理的不二人選時，現實卻狠狠的打了所有人的臉，領導選擇了部門裡變現平平的劉濤做了領導。

    後來，有同事說，那天路過總經理辦公室的時候，無意中聽到劉濤在和領導打小報告，說上次的項目之所以失敗，就是因為老丁一直鑽牛角尖，不肯退讓導致的。領導聽後自然面色不虞，然後和劉濤說注意觀察部門人員的一舉一動，有什麼事情隨時向他報告。

    大家聽後都為老丁鳴不平，一方面大家都知道，上次項目失敗是由於老丁的堅持不假，但是如果不堅持的話公司會遭受更大的損失，另一方面，老丁平時對劉濤就像對自己的孩子一樣，沒想到最後給自己捅刀子的確是自己最親近的人。大家忿忿不平，有的說要找劉濤理論，有的說要向領導說明情況，可最後統統被老丁回絕並阻撓了下來。

    後來，老丁和大家說，職場中不公平的現象比比皆是，如果每件事都玻璃心，去據理力爭，分清個所以然，不僅傷了同事關係，自己的升職之路也會越走越窄。遇到問題不要抱怨，改變不了別人，可以從自己入手，如果你是對的，終將會有水落石出的一天。

    老丁的這一席話對我整個2019年的職業生涯產生的非常大的影響。工作中，我們每個人都會遇到真心錯付，被最信任的人背後捅刀子的現象，造成這一現象的本質原因，其實是因為自己沒有對自己的人際關係進行一個很好的分層，所以導致任何人都可以「近身」對我們造成致命的傷害。

    02整理人際關係，有層次的管理人脈

    職場中，我們會遇到很多人，如果人人交心，只會讓自己輸的很慘。每個人都希望自己的同事喜歡自己，我之前有位同事就一直因為人際關係的事情鑽牛角尖，原因就是因為公司來了個實習生，和誰都相處甚好，唯獨就是不喜歡我的同事，即使後來實習生離職，同事還因為這件事情悶悶不樂，把有限的精力浪費在了不值得的人身上，導致自己的社交價值在不斷降低。

    人的精力非常有限，愛打小報告、蠻橫的同事比比皆是，而且一個人的本質是很難改變的，與其糾纏只會兩敗俱傷，不如把時間精力放在有用的人身上，互相成長，互相成就。所以我們一定要學會給同事進行一個分層：

    1. 親密層：親密層的同事一定是可以和你相互成就的一類人，工作上是友好競爭，互相激勵的君子，生活中，是排憂解難，互相幫助的摯友。這樣的人一定是你重點維護和花心思交往的對象，因為他會對你的未來發展起著至關重要的作用。

    2. 人際層：人際層是職場中較為正直的一類人，雖然平時不會有太多太深的交際，但是這類人也不會加害與你，可以真誠對待。

    3. 社交層：社交層的同事不建議投入太多的精力，畢竟身在職場，防人之心不可無，人多嘴雜，難免會有好事之人，對於這類同事不建議深交，只是做到相互尊重即可。

    03同事也有「七年之癢」

    很多時候，同事相處過程中，也會存在情侶間的「七年之癢」，因為一件小事，默契的合作夥伴便互相相看兩生厭，這個時候，就需要我們建立正確的認知，避免矛盾激化導致關係破裂。

    1. 人際容納度

    人際容納度指的是你可以和多少人建立深度的關係，也就是可以容納多少人。很多時候，我們會刻意迴避審視自己，導致也沒辦法接納他人，想要建立良好的人際關係，我們要做的就是要努力提高自己的人際容納度。

    2.人際交往的4個階段

    理想期：看到對方符合預期的一面。自己也努力把優秀的一面展示給對方，會有計劃的隱藏自己的不足。

    衝突期：指責對方和自責轉向逃避。通過一段時間的交往過後，會發現對方不像自己想像中的那麼完美，心裡會有些失落，自己的缺點也會逐漸暴露，開始變得內疚，內心非常衝突。

    整合期：一起評估差距在關係中成長。通過磨合，不斷地成長，找到共情的部分，善於接納對方。

    協同共創期：接受對方的短板，一起面對不確定性。在真正的了解對方的缺點之後，接納對方，一起完成共同的目標。

    04如何多角色處理好職場關係？

    1. 對上司：先尊重後磨合

    每個領導，既然可以干到這個位置上，必定有他的過人之處，無論是為待人接物還是專業技能一定都有值得我們學習的地方。在遇到矛盾時，我們先要用全局的眼光看問題，因為領導考慮的可能是整個戰略的布局，而我們思考的只是當下的結果，面對這種衝突時，要心平氣和的和領導進行溝通，有禮有節有分寸地磨合，然後說服對方。

    2. 對同事：多理解慎支持

    常在一起工作，和同事低頭不見抬頭見，時間久了對彼此的脾氣性格也會有所了解。當與同事發生不必要的爭執時，先要站在對方的角度考慮下問題產生的原因，避免情緒化說出傷人的話。在工作時，無論自己對錯與否，是否占理，在你和其他人發生爭吵的那一刻，你的形象都會受到嚴重的損害。另一方面，對於同事的決定，也需謹慎支持，避免給領導留下拉幫結派之嫌。

    3. 對客戶：善交際勤聯絡

    在現在激烈的競爭環境下，工作隨時可以被替換，但是積累的人脈是不容易很快流失掉的，在工作中，要多結交一些優質的人脈，私下多聯絡增進感情，說不定在你遇到困難之時，就會有貴人出手相救。

    4. 對下屬：多幫助細聆聽

    在工作方面，只有職位上的差異，作為領導也應該謙遜，在下屬面前做好表率，主動的幫助下屬解決困難，善於發現下屬的優勢，多聆聽多指導，讓下屬的價值實現最大化。
