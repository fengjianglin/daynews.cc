人生看似是一輩子，不過為三天，生活有起有落，日子有苦有甜

2019-12-13 223條評論 4,209次閱讀 761人點贊

人生看似是一輩子，其實只不過為三天，昨天、今天和明天。沉溺昨天的人，最糊塗，揮霍現在的人，最愚蠢，畏懼明天的人，最懦弱。

人生，一邊是拿起，一邊是放下，生活，一邊是忘記，一邊是珍惜，要想活好這輩子，必須做到三件事：一、把不好全留在昨天，二、把希望都留給明天，三、把努力都用在今天。任何人，都有遺憾，任何事，都有不順。最重要的其實不是結果，而是參與的過程。忘記悲傷的過往，珍惜眼前的一切，和過去告別，從此以後，珍惜所有，不留下遺憾，活好當下，不辜負人生！

生活有起有落，日子有苦有甜。做人，大度一些，與其憤怒生氣，不如平心靜氣。真的永遠假不了，對的永遠錯不了！不管遇到什麼事，什麼人，都不要生氣。

你生一分鐘的氣，就是作踐自己的福氣；生一小時的氣，就是跟心情過不去；生一星期的氣，是跟自己的身體過不去。學會不生氣，學會放寬心，再大的事，都會變成往事，再痛的事，都會漸漸淡去。別和自己過不去，別和他人生悶氣，要好好愛自己，賭氣絕食，是傻子，慪氣喝酒，是獃子，只有哈哈一笑，不計較，他人生氣我不氣，才是人生的大贏家！

人生如夢不是夢，朋友如酒未必久，一生之中，認識的人數不勝數，有的擦肩而過，有的走過一程，有的臉上帶笑，有的杯中放葯，誰是朋友，誰是知己，患難才能分清，時間才能證明。知己不是天天見面，吃喝玩樂、相互吹捧；而是懂你，在精神上支持你、鼓勵你、幫助你；在你有所不足時，指正你；在你失落時，安慰你。我們都需要一個知己。在滿腹心事堆積如山時，能有個耐心傾聽的伴；在壓力增加無處釋放時，能有人默默陪在身邊。如若人生有幸得一知己，請千萬要珍惜！
