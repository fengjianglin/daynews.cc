你這麼歲月靜好，一定沒有嘗過缺錢的滋味吧？

2019-11-26 0條評論 25次閱讀 0人點贊

摘要

剛工作一年多的同事前段時間和我聊天，她說這一年自己變化特別大。從前的自己會說笑、會打鬧、有心事就一股腦傾吐出來；現在的自己也會說笑、會打鬧，表面什麼事情都沒有，但其實心理已經歇斯底里很多回了。我問她，這些變化是何因呢？她思索了一陣子，淡淡道：&quot;最大的原因是缺錢吧。

剛工作一年多的同事前段時間和我聊天，她說這一年自己變化特別大。從前的自己會說笑、會打鬧、有心事就一股腦傾吐出來；現在的自己也會說笑、會打鬧，表面什麼事情都沒有，但其實心理已經歇斯底里很多回了。我問她，這些變化是何因呢？她思索了一陣子，淡淡道：”最大的原因是缺錢吧。去年我媽住院動手術，一下子要拿出好幾萬，那一刻突然成長了，發現自己也要成為父母的依靠了。” 她笑稱自己是”三無青年”：手裡無錢，身後無人，全年無休。

似乎成年人的崩潰，都是從缺錢開始的。我們不辭辛苦，一腔孤勇，只是希望自己賺錢的速度能超過缺錢的速度。因為只有品嘗過缺錢的滋味，才會知道，在有些關鍵的時刻，缺錢會成為壓死駱駝的最後一根稻草。有錢才有選擇權，它意味著更好的生活、更好的教育、更好的醫療……一個人成熟的標記，不是年齡的增長，而是明白錢的重要性。
