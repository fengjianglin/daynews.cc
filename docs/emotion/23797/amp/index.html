自棄者扶不起，自強者擊不倒，漂亮的20句話，轉給孩子看看

天天要聞
2019-12-19

1、從來只有拼出來的精彩，沒有等出來的輝煌。

2、人不會苦一輩子，但總會苦一陣子；許多人為了逃避苦一陣子，卻苦了一輩子。

3、人往往在沒有退路時，內在的潛能才會發揮出來。你不逼自己一下，永遠不知道自己有多優秀。

4、有人幫你，是你的幸運；無人幫你，是公正的命運。沒有人該為你做什麼，因為生命是你自己的，你得為自己負責。

5、自棄者扶不起，自強者擊不倒。

6、只有經歷最痛苦的堅持，才能配得上擁有最永久的幸福。

7、抱最大希望，盡最大努力，做最壞打算，持最好心態。記住該記住的，忘記該忘記的，改變能改變的，接受成事實的。太陽總是新的，每天都是美好的日子。

8、人生的弓，拉得太滿人會疲憊，拉得不滿人會掉隊。

9、往事不必遺憾，若是美好，叫做精彩。若是糟糕，叫做經歷。

10、不要因為一點瑕疵而放棄一段堅持，沒有一份工作是輕鬆的，沒有人是完美的。耐心點，堅強點；總有一天，你承受過的疼痛會有助於你。生活從來不會刻意虧欠誰，它給了你一塊陰影，會在不遠的地方撒下陽光。

11、不要活在別人的嘴裡，不要活在別人的眼裡，而是把命運把握在自己的手裡。

12、靠山山會倒，靠水水會流，靠自己永遠不倒。

13、沒有口水與汗水，就沒有成功的淚水。

14、有時候，沒有下一次，沒有機會重來，沒有暫停繼續。有時候，錯過了現在，就永遠永遠的沒機會了，所以好好把握現在吧，你的負擔將變成禮物，你受的苦將照亮你的路。

15、現實會告訴你，不努力就會被生活給踩死。無需找什麼借口，一無所有，就是拼的理由。

16、真正的強者，不是沒有眼淚的人，而是含著眼淚奔跑的人。

（美文）

17、比別人多一點努力，你就會多一份成績；比別人多一點志氣，你就會多一份出息；比別人多一點堅持，你就會奪取勝利；比別人多一點執著，你就會創造奇蹟。

18、你平時的堅持，藏著你未來的樣子。

19、天上下雨地上滑，自己滑倒自己爬，每個人的路都得自己走，累不累，只有自己知道。每個人的淚都得自己擦，任何人也無法代替你自己。

20、​​​​​​​​​​​​人生不怕起點低，就怕沒追求；不怕走的慢，就怕走錯路；不怕不如意，就怕想不通。與其羨慕別人，不如做好自己。
