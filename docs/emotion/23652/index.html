“不要二胎，你太自私了！”|生了二胎的媽媽們，後來怎麼樣了？

2019-12-19 1,656條評論 4,186次閱讀 1598人點贊

1.

前幾天和一個老同學在微信上聊天，說起孩子的事情，他問到，”你們什麼時候要二胎？”

我發自肺腑的跟他說，不要了，我一個人帶娃太累了，好幾年的時光，一眨眼就沒了，若再要一個，我一輩子就不用干別的了，我還等著娃大了，去享受自己的人生呢。

何曾想他竟然回到:”你怎麼能這麼自私呢！只想著自己，怕辛苦，也不為孩子著想，只生一個，孩子以後多孤單。”

屏幕這邊的我，實在沒想到，莫名其妙的，就被貼上了自私的標籤。

他家境優渥，爺爺奶奶都在幫忙帶孩子，所以，去年他家就要了二胎。

我們，因為各種原因，老人不能幫忙，我全職在家，老公經常在外地，綜合考慮，我們不打算要二胎。

你無法做到感同身受，但你最起碼要做到尊重。

好在，關於要二胎的事情，一則他對於我來說，無關緊要，二則，我自己意志堅定，所以，任誰勸說，也不會動搖。

但，有很多人，卻沒有這麼幸運了。

2.

我的好友，小北，不止一次堅定的跟我說，不會再要二胎。

她和老公，都在單位上班，收入一般，還經常加班，經濟和時間上，都不富裕。

但婆婆一直催促著要二胎，

總是說，孩子一人，太孤單，沒個兄弟姐妹，長大後有了事，連個幫襯的人都沒有。

加上老公站在婆婆的陣營，

她一人難以抵擋，就要了老二。

本來婆婆說了會幫忙帶，但孩子出生後，卻一再稱病推脫。

原來，家裡老大是個女孩，婆婆本想要個孫子，沒想到老二又是個女孩，不遂心意。

無奈，只好把孩子姥姥從另一個城市接來。

這件事情成了她心裡的一根刺。

既要努力工作掙錢，又要兼顧兩個娃，常常焦頭爛額，不能自持。

疲憊不堪的時候，她跟我說，真不該要老二，現在生活一團糟，控制不住情緒，夫妻經常吵架。

聽了婆婆和老公的，要了二胎，結果最辛苦的還是自己。

我為他們著想，他們可曾為我著想？

有時真想離婚算了。

聽了她的訴說，我也很無奈。

她的婆婆可不在少數。

懷孕之前，大包大攬，生了以後，又覺得太累，能少帶就少帶。

3.

我的同學小雪，也遇到了同樣的問題。

本來不想要二胎，但公公婆婆輪流勸說，再加上老公，於是舉手投降，要了二胎。

婆婆一開始，確實從老家來到城裡，幫忙帶孩子，但過了幾個月，又開始借口家裡有事，頻頻回家。

小雪懷孕之後，因為反應劇烈，是辭了職的。

當時說好，孩子大些，婆婆白天帶，她接著去工作。

但眼瞅著婆婆的意思，是想讓她自己帶。

她考慮再三，狠了狠心，把一周多的孩子送到了婆婆老家。

並告訴婆婆，自己要去上班了，沒時間帶老二，她每周休息都會回來看孩子。

婆婆雖然不情願但也沒辦法，誰讓當初為了讓小雪要二胎，承諾了要幫忙帶孩子。

小雪雖然看起來洒脫，其實也很無奈，孩子那麼小就長期兩地。

4.

前幾天看了一篇《2019年出生人口預計約1100萬，斷崖式下跌》的文章。

文章中稱截至今年11月17日，全國出生人口為1016萬，預計2019年的出生人口將不到1100萬。

而2018年的出生人口為1523萬，相比較，跌了不少。

從圖片上可以看出，相比於1987年的2508萬，2019年若是1100萬，30來年，人口出生，下降了一多半。

不知這篇文章的數據是否準確，但2019年的人口出生，確實顯著少於去年。

4.

雖然數據如此，但我想還是有很多人，贊同多多生孩子的。

每個家庭不一樣，所考慮的也不同。

不可否認，有些家庭，生了二胎，更加幸福和諧。

但彼之蜜糖她之砒霜，各種滋味，只有身處其中的人才真正了解。

我們經常開玩笑說，”真正帶孩子的人是不會想要二胎的。”

“比帶孩子辛苦的，就是帶兩個孩子了。”

雖是開玩笑，但帶孩子的艱辛，卻不是玩笑。

所以，要不要生二胎，乃至要不要生孩子，都要身處其中的人，再三思量。

畢竟多個孩子，並不是多雙筷子那麼簡單。

別人的意見，你若聽進去就是意見，若聽不進去，不過是耳旁風。

至於無關的人，請不要隨意置喙別人的家庭，因為那和你沒有一點關係。

萊布尼茨說，”世上沒有兩片完全相同的樹葉。”

世上也沒有兩個完全相同的家庭。

希望每一雙父母，都切實的站在自己的角度，不受言論的裹挾，做出最適合的選擇。
