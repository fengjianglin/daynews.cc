叔本華：人們最大的弱點，是對他人態度過於重視

叔本華

叔本華是現代悲觀主義哲學和唯意志論的開創者、主要代表人物，從唯意志論的哲學視角出發，叔本華認為人們最大的弱點，往往是對於他人的態度過於重視。如果我們對於人的這一最大的弱點不加以克服、壓制，那麼，我們就會成為他人意見的奴隸，最終活在別人的目光之中，無法真正獲得自身的幸福。

根據叔本華的哲學觀點，世界是我的意志，這是本質的東西，這是作為根本的世界的一面；世界的另一面，就是世界的表象，作為表現的世界僅僅是世界的外在的東西，並不是內在的、本質的東西。因此，一個人的幸福，歸根到底只是和他自己的內在有關，而和別人的評價無關，別人的評價無法改變一個人的本質。叔本華的這些哲學觀點，集中體現在其著名的《作為意志和表象的世界》一書當中，這本書就是叔本華哲學的最為關鍵的著作。

叔本華在《地位，或「人在他人評價中的位置」》中，開篇第一句便說道：「人性有​這樣一個弱點很特別，即通常人們對別人如何看待自己思慮過多，雖然這種看法所造成的反響微乎其微，無論人們的意見如何，它自身與幸福並無本質的聯繫。」

牢記叔本華的這句話極為重要，我認為，這句話就相當於數學當中的一個重要的「公理」，根據此「公理」可以推論出其他的許多真理，從而有益於促進人生的幸福。這句話首先表明，對於他人的意見過於重視、思慮過多，是人性的一個弱點，這就對此現象進行了「定性」。其次，他人對自己的看法，與自己是否幸福並無本質的聯繫。

只要我們謹記這句話，我們就不會在工作、生活當中迷失自己。為何這麼說呢？你看，在人們對你有意見的時候，惹你不高興的時候，你就應該想一想，他們的意見是否客觀公正，如果他們提的意見有道理，盡可合理地採納、加以改進，我們沒有必要為此生出許多煩惱，這是對待他人意見最為重要的處理之道。

另外，面對別人的「捧殺」、獻媚，你大可不必過於甜蜜、過於微笑，他人的恭維並不會使你變得更好，如果你對於他人的恭維之語，對於他人的拍馬屁完全「笑納」，這就會促使你越來越落後、越來越囂張和變得驕傲，從而導致自己逐步走向失敗，即便是他人對你的恭維並無惡意之時也是如此。如果他人對於你的恭維是惡意的，那麼，恭維你的他人就是你的敵人了，他們達到了「捧殺」你的目的，這就會使你離幸福越來越遠。

叔本華更進一步明白地闡釋說：「對於他人所說的話給予過高的估價，這未免太尊敬他們了。……所有人的最有價值的東西以及真正的人生，是掌握在自己手中，而不是別人如何看待。」
