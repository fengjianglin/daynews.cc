同樣是生孩子，一場同學集會看出差距，果然幹得好不如嫁得好

2019-12-21 587條評論 1,930次閱讀 86人點贊

文/小萱媽媽育兒經（本文原創，版權歸作者所有）
中國有句老話叫做：「女人幹得好不如嫁得好」，這句話在過去那個年代可以說是擇偶標準了，女性們的身身份看似高貴了，實則低賤了，現在我們都追求做一個經濟獨立的女性，就是因為不想寄人籬下，獲得要成為伸手向丈夫要錢，或者是自己花一分錢都要報告一下的感覺。

但是不得不說這回對女性的考驗和要求實在太狠毒了，想要站穩腳，你要犧牲的也就很多。於是很多次你就會想起父母曾經說的「女人幹得好不如嫁得好」甚至你會想要妥協和認同。

同樣是生孩子，一場同學集會看出差距，果然幹得好不如嫁得好

曾經聽閨蜜說他這輩子最難堪的一件事和最心痛的一瞬間就是在去年的同學聚會上。身為成年人都懂同學集會究竟意味著什麼，沒錯不是反比就是炫耀，要不就是發展客戶攬收資源。這就是成年人的世界，女人們更是比較自己比較老公比較孩子，他是我們認為生完孩子保養的算是好的

了，但是他去同學就會之後自卑了，因為幾乎都生孩子了，但是人家依舊看起來更加漂亮了，好像多了女人成熟的韻味。而自己呢，肉眼可見的魚尾紋，粉底都遮不住的斑點和暗黃的皮膚。同學們私下裡也都議論紛紛，說他自己乾的再好又能怎麼樣。自己的老公多多麼多麼寵愛自己，心疼自己。

為什麼有的女人生完孩子變漂亮？有的變醜？

1. 沒人幫著帶孩子

很多寶媽會說自己的孩子當然自己帶最放心，剩下的交給誰都不放心，要是這種心理那就別抱怨自己變醜之類的，最主要的是大多數孕婦是不願意自己帶孩子的，因為寶寶剛出生。

睡眠時間短，半夜起夜喂夜奶，幾乎就沒有一次能睡個好覺，這樣折騰會讓寶媽一天之內蒼老很多，後期再怎麼不也沒用，因為孩子只會越長大越麻煩。

2. 老公不理解

男人的思維大概是我自己在外面打拚養你們，你還天天吵吵累？然後就會沒事看你做的不好就會說你，這樣寶馬根本就不能打扮，要是只顧著打扮嗎，他們就會說孩子都沒照顧好打扮給誰看？我辛辛苦苦掙錢是為了養活孩子不是你。這樣你就情緒受到影響慢慢的就會變成一個保姆，黃臉婆，沒有任何地位。

生孩子後這樣做可以拒絕變成黃臉婆

1. 放開手腳

沒有媽媽不心疼孩子我理解，但是你要是不在第一時間放開手腳被孩子舒服，那麼你就會變成一個被習慣利用的傀儡，讓老公知道和婆家知道，你照顧孩子是可以但是一定要有幫手，不然自己帶娃要不就全天不準說我任何不好，給錢不要有任何怨言，要不就我出去上班孩子找人帶。

2. 打扮自己

不要覺得結了婚生了娃你這輩子就差不多了，你要知道人生很長會發生什麼變數都是你無法預料的，所以保持自己的初心，就要做一個辣媽愛美的辣媽。要讓老公有危機感，而不是把危機感都給自己承受，這樣你的婚姻包括你自己都會更加有自信更加美滿。

小萱媽媽暖心寄語：

給各位寶媽們一個忠告：不管生活多忙碌，不管生活多打擊你，千萬不要迷失自我和犧牲自我，人都是自私的，尤其是男人更自私，不要做那個一直妥協的人，你是找了一個能照顧你的能保護你的，能擔起家裡重任的男人，不是找了一個柔弱不堪的兒子。
