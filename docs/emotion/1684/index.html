"我弄死你們！"父親掐住妻子脖子，拿鎚子和鐵栓破門走向女兒

2019-11-28 0條評論 24次閱讀 0人點贊

摘要

&quot;我爸家暴我媽和我到底怎麼辦？&quot;11月27日，網友櫻花鋪子連發5條視頻求救：一個男人拿著鎚子強行破門而入，面目猙獰，叫囂著&quot;我弄死你們&quot;。櫻花鋪子姓趙，今年25歲，正在娘家河北滄州照看孩子。視頻里的男人是她父親，她沒真的被弄死，這讓事情看上去多了一點黑色幽默。

“我爸家暴我媽和我到底怎麼辦？”

11月27日，網友櫻花鋪子連發5條視頻求救：一個男人拿著鎚子強行破門而入，面目猙獰，叫囂著”我弄死你們”。

櫻花鋪子姓趙，今年25歲，正在娘家河北滄州照看孩子。視頻里的男人是她父親，她沒真的被弄死，這讓事情看上去多了一點黑色幽默。

視頻截圖

趙女士表示，這段視頻錄的自己和母親被父親打了，而他是拿著斧子和鐵栓來的，實在不想叫他”爸”。因為爐子滅了他不高興，罵煤球不好。母親在院子里劈木頭點火，木頭很硬，總是劈到磚上，就被他罵了。

趙女士當時在屋裡給孩子餵奶，聽到母親的尖叫趕緊放下孩子出來，就看見趙先生掐著母親的脖子，還說早晚要弄死她們。趙女士很生氣，說：”你再這樣別怪我不顧情分報警了！”

趙先生說”你試試”，就過來推她。母親起身攔住他後關門，趙女士打開手機錄像，之後他就拿著鐵鎚沖了進來，還抄起了一個鐵門栓。

網路配圖

警察接到趙女士的報警後趕到她家調解，而趙先生拒絕道歉。趙女士說，從經驗來看，晚上應該是很平靜的。

她問過母親離不離婚？母親說不想離。年輕時她就想離婚回了娘家，結果趙先生天天半夜去家裡砸門罵街。現在姥爺都好90歲了，冬天身體又不好，丈夫一定會鬧，要是老人出了事擔不起這個責任。

趙女士也表示，母親有很嚴重的心臟病，自己也是因為這點不敢和父親正面剛，”我怕失去我媽媽，她是我生命的支柱！”而母親也不願意讓女兒趟混水，覺得反正自己活不長了。

網路配圖

據紅星新聞報道，吳橋縣公安局報警中心的值班民警表示，確實接到了趙女士的報警，具體情況交由派出所處理了。

11月28日，趙女士表示，自己打算帶母親和孩子回青島，丈夫也在向自己這趕了。截至上午10點，她還沒有等到民警和婦聯的工作人員，決定先離開，有消息再回來。

父母結婚26年，父親的暴力從未停止，最嚴重的一次幾乎就要掐死母親了。叫他出去工作也要大打一場，幹活也總是干幾天就不幹了。

網路配圖

和所有家暴受害者一樣，很多人問為什麼不離婚，為什麼這麼懦弱？趙女士很無奈，母親雖然有很多想的不對做的不對的，但她是一個好人，顧慮太多讓她懦弱了。”壞人只想自己，一定不會懦弱！”

誰會比家暴受害者更清楚，離開施虐者是多麼危險呢？家庭暴力的最後一步，就是殺掉她。因此無論受害者什麼時候打破沉默，都已經做到了最好。
