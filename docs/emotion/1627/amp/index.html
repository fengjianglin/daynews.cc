夢語分享：人生真正的強大，不是征服什麼，而是能夠承受什麼 。

夢語分享：一個人真正的強大，不是去征服什麼，而是能夠承受什麼 。承受生活給予的所有傷痛和苦難，接受自己最真實的面目和現實 。無論命運如何安排，只要堅強了，努力去改變了，你都可以做回最初的自己，這樣才能成就最強大的人 。堅強的品格不是天生的，必定要經歷人生的一番歷練才能形成 。人生需要堅強，堅強是一個人的精神支柱，一個堅強的人，他的內心必定是強大的 。內心的強大才是真正的強大 。關注我們，一起成長。

有一句話說，與人溝通時，70%是情緒，30%是內容。 與人發生矛盾時，先問下自己，你是想傳遞內容還是傳遞情緒，是想解決問題還是擴大麻煩？沒有人應該為你的心情負責 。人活在世，誰都少不了磕磕碰碰，沒有人會永遠不犯錯，誰也不會希望自己一犯錯就有人抓著把柄似的肆意責備 。一個心懷慈悲的人，懂得時常反躬自省，哪怕別人出了錯，也不會隨便給他人難堪 。 一個寬容大度的人，懂得換位思考，就算不理解不贊同，也會包容不同行為方式的存在 。 一個心智真正成熟人，明白相比毫無覺悟之心地怨怪別人，凡事先從自己身上找原因，避免同類問題再次發生，才是有擔當有責任的表現 。所以，遇事不責備，才是一個人最高級的修養。

沒有人活在真空里，人和人之間總要打交道。有了交集，也就有了矛盾。有的人一心報復，最終兩敗俱傷。有的人選擇原諒，最終握手言和。有的人選擇無視，最終笑看人生。這是三種不同的態度，也是三種不同的人生境界。真正的強者，選擇忽略他們，拒絕回應惡意。堅守自己內心的善念，把報復的衝動變成向上的能量和動力。讓自己幸福美滿的時候，一切都是無關緊要且可以坦然處之的事情。

繁華美景像是童話里的紅舞鞋，漂亮妖艷而且充滿誘惑，一旦穿上便再也脫不下來，最終留給舞者的只是無限的疲憊與厭倦。所以，簡單的不一定是最美的，但最美的一定是簡單的。天地有大美而不言。簡單的友情看起來平淡如水，但卻能夠歷久彌堅，經得起災難和誘惑的考驗；簡單的生活看起來波瀾不驚、清淡乏味，卻總在點點滴滴的小事上給人帶來暖心的幸福！

關注我們，一起成長。
