人生“脆弱”期，70後為何成“離婚主力軍”

天天要聞
2019-12-13

70後這代人很悲催，是唱著「學習雷鋒好榜樣」長大的，從小就被灌輸「好人有好報」、「吃虧就是福」……這代人基本上都很「懂事」，屬於「放養」的，兄弟姐妹好幾個，家庭觀念比較強，骨子裡一般比較善良，有正氣，也有責任感。客觀的講，70後相對80後、90後的人來說，在婚戀觀念上比較傳統，然而讓人不解的是，70後卻是離婚率最高的人群。

70後離婚，大多都是「錢」惹的禍

這幾年，70後的離婚率特別高，已成為「離婚主力軍」。按道理講，70後都步入中年，要奔五的人了。這個年齡階段，上有老、下有小，職場沒有了競爭力，而且疾病也容易找上門，正處於人生抗風險能力最弱的時期，更應該夫妻互相扶持，共度中年危機。可現實中，很多70後卻在人生最「脆弱」的時刻，遭遇婚姻危機。

生命已過半，婚姻也早已經過了「七年之癢」，70後為何還要離婚？根據統計，70後離婚的最大殺手是經濟原因。70後大多是「窮二代」，很多人沒有「富爸爸」可以依靠，基本靠自己白手起家。混得好的70後，有錢有閑，有些人「飽暖思淫慾」，想抓住青春的尾巴瀟洒一回，因出軌而婚姻解體。而混得不好的70後，真切感受到了「貧賤夫妻百事哀」的滋味。有些女人在拜金享樂的潮流下，對身邊的男人有了諸多不滿：你也是大學畢業，為什麼人家當上局長了，你連正科都沒混上？為什麼同學買了豪宅寶馬，我卻只能跟著你還房貸坐公交？這種比出來的失望，往往擊碎女人「望夫成龍」的夢想，最終釀成離婚的結局。

78年出生的老李就是這樣被離的。他天生屬於樂天派，沒有什麼野心，只求平安健康就好，一直在工廠里按部就班，每個月四五千工資，不多也不少，日子還過得去。但是，老婆是個要強的人，眼看著身邊的人混得越來越好，她就特不甘心，總是莫名地發脾氣，把怨氣撒到老李身上，埋怨老李不是個男人，不思進取，不求上進，掙這點死工資連家都養不起，還讓自己跟著吃苦受累。吵著吵著，兩人感情越來越淡，心越來越遠，最後女人出軌另攀高枝，揚長而去。

忍了15年，我想為自己活一回

70後那個年代，不像現在這麼開放，大多數結婚都是遵從「父母之命、媒妁之言」。那個年代普遍經濟條件不好，很多是為了生活湊在一起，根本不懂什麼「三觀」，也不懂什麼是愛情，看著順眼差不多就行。有的並不是自己心甘情願的，只是因為年齡大了，不想讓父母為自己的婚事而爭吵、操心，草草地就結婚了。

隨著社會的發展，物質生活越來越好，網路、手機早已經滲透到生活的方方面面，接觸的信息多了，視野也開闊了，自然對婚姻有了更多的思考，開始追求精神上的契合。經過歲月的侵蝕，感情已經褪色，如果不能轉變成親情，「先結婚後戀愛」的婚姻很容易就風雨飄搖。以前為了生活、為了孩子，可以忍著，而今天，在經濟上不需要依賴對方了，孩子也漸漸長大了，就不想再忍了。因為，有的婚姻真的是生不如死。

46歲的王女士，孩子拿到大學錄取通知書的第二天，她就迫不及待地走進民政局辦離婚手續。她是性格比較柔弱的人，沒有什麼主見，是什麼都聽父母話的「乖乖女」。當年，父母做主為她找了個村子裡的「能人」。男人的家庭條件比她家好很多，就是大男子主義比較嚴重，什麼都要聽他的，家務事從來不管不問，回到家就跟「老爺」一樣，沏茶倒水、好酒好菜伺候著。男人腦子活，開始跟別人去北京打工，後來自己干，還成立了公司做了老闆。男人有錢了，接觸的人也多了，思想觀念也變了，兩人的差距越來越大，除了孩子這個話題，再也沒有什麼話可說。後來，男人外邊有了別的女人，他們的婚姻成了喪偶式婚姻。王女士也想過離婚，但是害怕影響孩子，就選擇了忍氣吞聲。可是，「忍」字頭上一把刀，這些年她常常偷著流淚，在孩子面前還要裝作若無其事，日子過得苦不堪言。拿到離婚證的那天，她有種如釋重負的感覺，覺得整個世界都在沖她微笑。

餘生很貴，請對自己好一點

70後，已過了不惑之年，經歷過人生的沉浮,也目睹過生死的更迭,活得明白了，更加懂得珍惜自己。「如果可以湊合，有人可以依靠，誰又願意離婚，選擇獨自堅強？」這恐怕是70後離婚者內心最真實的聲音。

每一個70後離婚的人，都是有故事的人，無不是抱著「必死」的決心和勇氣走出婚姻。這個年紀，比任何時候都更渴望真情，更渴望歸屬，也就對自己的婚姻有了更正確的審視。如果沒有感情可以堅守，婚姻里只剩下錢和性，那麼再走下去會很難。

70後，歷經滄桑，不是感情麻木了，而是懂得把最真的愛留給值得的人。餘生很貴，不要在一個不疼惜自己的人身上浪費時間，對自己好一點，怎麼幸福就怎麼活吧。
