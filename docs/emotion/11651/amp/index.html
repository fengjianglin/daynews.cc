“不顧妻兒三十年，最後懲罰我的是孩子”，男人別等老了，再懺悔

天天要聞
2019-12-12

文|公子逸

女人最不需要的就是，男人犯錯之後的懺悔。

如果這種懺悔，還發生在犯錯很多年之後，那女人就更不需要了。因為，傷害也受夠了，在這種傷害中把孩子也養大了。你的懺悔，可能僅僅出於，你用到女人了，而非真的想要彌補什麼。

其實，有時候，哪怕你想彌補，也沒用了。時光不能倒流，而有些傷，對女人來說，是永恆的。

我老家的一個遠房伯伯，年輕的時候，放蕩不羈。他本來跟妻子有兒有女，但就是不好好過日子。

說起，這個伯伯，最想說的是，家庭教育非常重要。如果你有兩個或者兩個以上的孩子，一定要記住，你可以教育他們要互幫互助，但你更要讓他們知道，一旦各自成了家，各自經營好自己的小家庭才是最重要的。

我這個伯伯的母親，也就是我的三奶奶，封建思想非常嚴重。她屬於那種非常強勢，思想還非常落後守舊的那種老人。她對兒女的教育很嚴格，始終給兒女貫徹這樣的思想：「別的人都是外人，只有我們這一家人是親人，誰也沒有我們這一家人親。」

外人指的是，兒媳婦和女婿，包括了孫子孫女外孫子外孫女。

她這種想法完全不合理，但是她就覺得自己是對的，她要求自己的兒子過得好的就必須管過得不好的，哪怕自己的孩子過得不好，也該幫襯兄弟。

在她的這種教育下，我的這個伯伯在結婚後，始終特別顧家。顧家到什麼程度呢？農村秋收的時候，他要先幫弟弟妹妹去秋收，而把自己家的地完全交給自己的妻子。

她妻子每到了秋收的時候，根本指不上自己的男人，完全靠自己。最怕的是下雨天的時候，就是他自己家的糧食都被雨水沖走了。他想的也是，先幫弟弟妹妹，而完全置自己家的糧食於不顧。

他的這種忘我，得到了很多人的讚頌。村裡很多人，說他是孝子，是好哥哥。他的母親也把他當家裡的頂樑柱，他的弟弟妹妹們也非常尊重他。

外邊所有人都捧著他，他的虛榮心得到了極大的滿足，他覺得自己這樣做就是對的，這樣做才是好兒子，好哥哥，好男人。

但是，他唯獨忘記了自己的妻子和孩子。他不知道的是，她的妻子和孩子，是如何在大雨里落湯雞一樣，只能眼睜睜地看著自己家的糧食都被沖走。

他不知道的是，當他把自己的錢，把妻子種的菜，妻子做的好飯，全部端走之後，他的妻子如何的不滿，他的孩子是如何的憤怒。

我這個伯母說過這樣一件事：「那年，孩子說想吃紅薯。我到我娘家要了好的紅薯秧子，在自己家地里給我兒子種了一個畦子。我兒子很開心，每次我下地的時候，他都要跟著去，要給他的紅薯澆澆水，抓抓蟲。

他們全家都知道，我兒子寶貝那些紅薯。我甚至在紅薯快熟的時候，打聽了很多紅薯的做法。可後來，他自己刨了那些紅薯，給他的父母弟弟妹妹們分完了。他竟然一塊都沒給我兒子留。我兒子就那樣守著那片地，哭了好幾個小時。

他沒有一點愧疚，還罵我兒子跟我一樣小氣，為一點紅薯有什麼可哭的。他說，有了好東西，難道不該分給自己的父母兄弟嗎？

我跟他吵，質問他為什麼不給兒子留幾塊。他竟然怨我。他說，誰讓你種那麼點。你要多種點，哪裡有這種事。你明知道，家裡人都喜歡吃紅薯，就種那麼一點，你是有多自私只想你自己。

他根本不知道，那點紅薯秧子都是我從我娘家拿的。他心裡只有他的父母兄弟，根本就沒有我和孩子。」

我長大後，每次想起這件事，都會覺得我那個伯伯是顧家到了什麼地步。他竟然不允許自己的妻子，只給自己的孩子種點紅薯。他覺得，所有的好東西，都是該分給父母兄弟的。如果妻子沒想過把好東西，都分給他的父母兄弟，那麼他就覺得妻子自私。

他就這樣為了自己的父母兄弟無私付出了30年，他父母去世，他也老了，到了需要兒女照顧的時候。他跟妻子感情始終不好，很早就分開住了。我那個大伯母常年在外地打工，而我這個大伯就全年在家幫幫弟弟妹妹的忙。

他自己能照顧自己，也能顧及弟弟妹妹的時候，他們的關係都不錯。畢竟，一個人的完全付出，另一些人的完全接受，這種關係沒辦法不好。

一個月前，我這個伯伯中風偏癱了。他的弟弟妹妹把他送到了醫院，給他的妻子孩子打了電話。他的妻子表示：「這些年，他都顧了你們了，你們管他好了。」

然後，他的弟弟說：「我們這不是把他送醫院來了。我們又不是夫妻，這本來就該你和孩子照顧他。」

他的妹妹說：「你怎麼這麼無情無義。你不管，孩子難道能不管自己的親爹。你不回來，我們就讓你兒子回來照顧他。」

我這個伯伯親自給在外邊工作，始終沒錢買房子結婚的親兒子打了電話。

他覺得，兒子照顧父親，天經地義。他覺得不管我對你好不好，沒有我就沒有你，你都該回來照顧我。

他兒子也真的回來了，他跟自己的母親說：「你不用回來，回來了，他就賴上你了。你就不要管他，讓他看看，他的弟弟妹妹對他多好。」

他的兒子給他請了一個照顧他的保姆，然後就走了。他會定期給我這個伯伯打錢，但是一次都沒回來看過。

可笑的是，自從我這個伯伯偏癱之後，他的弟弟妹妹就不去看他了。他們怕被自己的哥哥賴上，他們覺得萬一自己的哥哥對他們有錢或者是照顧上的要求，他們該怎麼開口拒絕呢？

我這個伯伯被保姆照顧了一個月，也漸漸明白了自己弟弟妹妹的薄情。然後，他開始懺悔了。

只要家裡其他的親戚去看他，他就懺悔自己沒好好對妻子和孩子，導致孩子對他沒什麼感情。他希望家裡的親戚能當說客，去告訴他的兒子，他已經知道錯了，已經後悔了，趕緊回來照顧他吧。他需要的不是保姆，是自己的妻兒。

我媽也被他要求去當說客，結果我媽冷冷地跟他說了一句：「你們這種男人就是活該。」

我曾經在我大伯母去世後，寫過我們家的家史。我爸也屬於那種不顧妻兒的男人，我媽一輩子深受其苦，又怎麼可能去當說客。

我這個伯伯，還有我爸，為了自己的兄弟姐妹，那是掏心掏肺地付出。可到最後，我爸在我姑姑的傷害下，徹底對親情清醒。我這個伯伯，也在自己的晚年，有了這樣不太好的結局。

反觀那些自私自利只顧自己小家的男人，反而跟妻子孩子的關係很好。當然也有人罵他們忘恩負義，可他們也會回上一句：「那是他願意的。」

到底什麼是親情？

親情該是相互的。你為了他好，願意付出，而他為了你好，不願意讓你付出。就像那點紅薯，你講親情，去送了。他們講親情，就不該一下子分完了。他們該想，這是你兒子最喜歡吃的，要多給他留一點。

如果親情沒有這種相互性，那就不是真正的親情。

到底什麼是善良？

很多人說過我父親善良。他這一輩子行善，不管對父母，對兄弟，還是對外人，都很好。可他唯獨沒善待自己的妻子和孩子。我小時候關於父親的很多記憶片斷，都是他如果偏袒了我姑姑一家，如何傷了我媽的心。

我父親這是善良嗎？

我不知道。我只知道，我這一生都不會嫁給我爸這樣的男人。

我跟我那個遠房伯伯的兒子，也就是我的遠房堂哥，在成年後，有過一次對話。

他說了這樣的一番話：「我叔雖然也顧大家，但是我叔掙的錢多，也算是給你和你弟提供了一定的物質保障。我爸是自己沒能力，還都顧了別人。導致我和我媽，不管精神上，還是物質上都沒過過一天好日子。」

我算是在男人顧家這件事上，看到了另一個側面。如果你有能力，顧大家的時候，也沒有讓小家很艱難。那麼，你跟妻兒的關係，還不至於太僵。但是，情感上，妻子和兒女肯定會有怨懟。

可如果你沒能力，卻依舊只顧大家，完全不顧小家。那麼因為你的妻兒吃了更多的苦，他們就很難做到原諒。

就像我這個堂哥，他找了保姆，出了錢，算是盡了義務。但是卻不肯回家看父親一次。

因為，小時候的苦太多了，即使已經成年也越不過去了。
