“母親越漂亮其子女的智商越高”，這有沒有科學道理？

2019-12-21 1,199條評論 2,226次閱讀 344人點贊

有很多人說「母親越漂亮其子女的智商越高」，這有沒有科學道理？很明顯，這句話沒有任何科學道理！並且，還顯得很缺乏科學素養哈~看似表達了基因遺傳的強大性，實際上卻並不知道遺傳的本質是什麼。

首先，我們可以來看看母親漂亮的好處是什麼？答案很簡單：孩子可能會遺傳到媽媽的嬌俏模樣！但是，這裡是有前提的，那就是母親的基因足夠強大，使得孩子的樣貌完全跟父親沒有明顯的相似度。

其次，漂亮的母親等於智商高嗎？答案：不一定！因為漂亮雖然可能包含了臉型、五官、膚色，甚至是身材，但它並不體現智商這個東西。也就是說，一個漂亮的媽媽可能同時兼具了智商高這個特質，但她也可能只是一個漂亮的普通人而已。

不管孩子的長相，還是孩子的智商，還是孩子的身高等，這通常都是由父母雙方的基因共同決定的，而孩子到底遺傳誰多一點，長得跟誰更像一些，這相當於是一個隨機性的結果罷了。當然了，不管是智商，還是一個人的樣貌，雖然後天可以通過一些方式來進行彌補，但真正的起跑線都是剛剛出生的時候就已經劃好了。

客觀而言，影響智商的因素也不少，而首當其衝的就是遺傳這個關鍵點。通常來說，只要孩子的父母智商較高，他們結合後繁育出的後代智商都不會低。並且，一般父母所在地為異地的孩子，其平均智商也會高於雙方都是本地的父母。而那些父母甚至時表親關係的孩子，出現低智商的概率也往往會更高，這也是不允許近親結婚的主要原因。

而影響智商的第二個因素，其實是母乳。這個答案可能會引起很多人的爭議，尤其是當代社會奶粉的盛行。但就科學研究而言，母乳中的確含有多種能夠促進孩子智力發育的活性物質，這不是其他人工加工產品可以替代的。牛磺酸是一種對智力發育特別重要的物質，而母乳種的含量可以比牛奶高出十倍以上。也有調查數據顯示，母乳餵養長大的孩子平均智商會比吃其他乳製品長大的孩子高出3到10分左右。

而影響孩子智商的第三個重要因素就是環境，這裡包括了家庭環境、學校環境，乃至他所處的小型社會環境。這也是為什麼說父母永遠是孩子最好的老師，家庭對孩子的影響永遠大於一切，甚至大於很多父母千挑萬選的學校。教育，真的不是學校單方面的事情，孩子的智力雖然出生就決定了大部分，但後天環境對智力的開發也是尤為重要的。

當然，孩子的智力開發本就是一個複雜的過程，而影響智力的因素也有很多，我也就不再一一例舉。我回答這個問題的意義就在於讓大家了解，智商和樣貌是兩個完全沒有關聯的東西。雖然，這兩樣東西我們都可以通過遺傳來獲得，但遺傳本身就具有不確定性。

如果希望自己的孩子不僅樣貌出眾，而且還智商過人。那麼，首先孩子的爸爸和媽媽都需要滿足這兩個基本條件，才能將後代的期望值控制在較低風險之內。因為，只要有一方不具備這樣的條件，便有可能會導致出生的孩子竟然跟這一方具有很高的相似度。

簡而言之，智商和美貌你有可能會兩者兼得，但也可能跟大多數人一樣都只是個普通人。如果在尋找配偶的時候真的特別在意這方面的信息，可以根據自己需求進行選擇。當然，前提就是自己也是智商和顏值都在線的喲，不然到頭來可能也只是自己的一廂情願罷了。
