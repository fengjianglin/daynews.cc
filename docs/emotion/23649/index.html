要想成為會說話的人，首先要學會傾聽，比自己夸夸其談高明多了

2019-12-19 1,126條評論 3,572次閱讀 559人點贊

馬歇爾·盧森堡在《非暴力溝通》中指出，在人們清醒的時間裡，有百分之八十的時間在用來進行人際溝通，這其中又有百分之四十五的時間用於傾聽。蘇格拉底也說過：上天賦予我們一個舌頭，卻給了我們一對耳朵，所以我們聽到的話比我們說的話多兩倍。可見傾聽在我們的生活中是多麼的重要，所以，與其先學會說話，不如先學會傾聽。

很多人之所以沒有辦法傾聽他人，就是太專註於傾瀉自己的情感和訴求，在很多場合下，這不僅僅是不體貼，不善解人意的問題，甚至會讓人感覺到自己不被尊重。 其實，只要有意識地為他人考慮，尊重別人表達自我的權利，傾聽真的沒有那麼難。

學會傾聽，要懂得尊重對方。1916年12 月，蔡元培出任北大校長。為了改革北大的舊官僚習氣，蔡元培大膽聘請了一批具有革新意識的知識分子，以此來補充北大的教師隊伍，其中就有留學美國的胡適。初入北大，胡適感到壓力很大，畢竟北大是當時中國的「第一學府」。蔡元培知道胡適會有些緊張，便在胡適登台講課的頭一天晚上來到了胡適家中。 果不其然，蔡老剛一坐下，胡適就一個勁地說東說西，擔心講不好、學生可能會不積極等問題，而蔡老只是耐心地傾聽著他把話說完。只在臨走前誠懇地說了一句：「我聘你到北大來，就是信任你。」胡適每當想起這件事，就很感激蔡元培那天晚上一直傾聽他說話。傾聽，不是一言不發的沉默，而是真切的尊重。對方有話向你傾訴，是對你的信任。你的悉心傾聽，則會讓對方收穫信任的回報、情感的滿足和人格的尊重。

學會傾聽，要給與對方一定的回應。很多人說我聽了，但效果真的不好，那是因為你僅僅是在用耳朵聽而已，要知道，傾聽不是被動的接受，而是一種主動的行為。首先需要注視對方，表示你在聽他講話，然後再通過一些簡單的語氣詞，或者是一些肢體動作和表情以做回應，這樣才能提高訴說者的興緻並且使對話更加的融洽。在生活中最常見的例子就是，你在興緻勃勃的和朋友講一個事情，他卻自顧自的玩手機，對你所說的事情沒有表現出半分的興趣，你就自然而然的沒有再說下去的興緻了。你的心不在焉，敷衍塞責，那隻能令對方失望、失落，感覺冷漠。所以，一個好的傾聽者需要給對方一定的回應。

學會傾聽，要和對方保持同一戰線。在這裡和大家說明下，所謂的傾聽不是光是你聽他說就完了，而是要引導對方去傾訴，適當的和對方去互動，即使你不同意對方的觀點，也不要打斷對方，讓他去說就好了。給大家舉個例子：閨蜜：我失戀了。你：啊？怎麼回事？閨蜜：我看到他和小紅在約會。你：然後呢？……

整個傾聽過程就跟著對方的思維走就行，引導對方去說，適當的加上問句，以及語氣詞和對方去互動，對方說舒服了就好了。這樣對方才會感覺到被尊重，覺得你和他是站在同一戰線的。

《簡讀》中提到，在你未來的人生道路上，你常常會發現不由自主地被當作知己，去傾聽你熟人的隱秘。你的高明之處不在於談論你自己，而在於傾聽別人談論自己。傾聽是一種心與心的對話，是人與人之間的心靈約定。傾聽是訴者的誠心，是聽者的虛心。成為一位好的傾聽者，別人自然而言的就更願意和你交流，所以，想要成為會說話的人，首先需要學會傾聽。
