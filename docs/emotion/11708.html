森田療法：在順其自然中，接納自己，愛上自己

2019-12-11 2,312條評論 4,775次閱讀 2521人點贊

書語人間：每天10分鐘，陪你讀懂1本好書，點擊文章右邊的「關注」，一起成長

大家好呀~

今天，靈遙將為你帶來一本新書，名字叫做《森田療法：醫治心理障礙的良方》。

打從我們小時候開始，我們的父母、老師便一直都在用各種方法，來訓練我們用更加快捷的方式，去解決一個個問題。

尤其是加入了一家公司後，能否有效地解決領導、客戶的問題，更是判斷一個人是否是一個好員工，能否升職加薪的標準。

於是，在遇上了抑鬱、焦慮等心理困境後，不少人便會沿用學生時代的方法，把這些個情況當成問題去解決，試圖讓自己再有活力一點，再淡定一點。

但是，這真的是最好的方法嗎？

在回答這個問題以前，先來和靈遙一起讀讀森田正馬，一位來自日本的，醫學博士的故事吧！

01.

多舛的童年，神經症的常客

1874年，日本高知縣，一個叫做森田正馬的孩子，降生在了一個農戶家庭里。他的父親名叫森田正文，雖說以務農為生，但也是村裡為數不多的文化人。

在兒子出生後，父親便對他寄予了「望子成龍」的厚望，要求之高，竟然達到了苛刻的地步。

記得在是森田正馬年幼時，父親便開始逼著他讀書，寫字，若學不會，便不允許他的睡覺。

結果，在父親的逼迫下，年幼的森田正馬開始對學習產生了嚴重的厭惡情緒，幾乎每天早上都會大哭大鬧，懇求大人允許自己不去上學。

他的做法自然是得到了父親和母親的一致反對，以至於他長期因為壓力大，精神緊張而體弱多病，引發尿床、頭疼等多種疾病。

一開始，森田正馬與大多數人一樣，因為自己的情況憂心忡忡，拚命地想要好起來，甚至不止一次因為自己對生命的無力感，想要偷偷地結束自己的生命。

直到考上大學後，對人生心有不甘的他，因為不想帶著滿腔遺憾的離世，便開始拼了命地去學習，不再去多想那些困擾他的情緒疾病。

結果，奇蹟出現了。

沒過多久，他便從負面情緒中走了出來，病也跟著好了，成了日本首屈一指的精神科醫生，也可謂是「久病成良醫」了。

02.

為什麼說「不抵抗比努力更有效」？

從森田正馬的故事來看，我們發現，有一類神經症的患者常常會有這樣一個共性，那便是，敏感，又好強。

一方面，他們常常對自己有很高很高的要求，不願意輕易承認自己不如其他人。另一方面，這種高要求也會使得他們常常忘記了去照顧自己的感受，以至於活生生地把自己給累病了。

靈遙有一個朋友便是如此。

他畢業於國內一所知名大學，學習成績也很優秀，但是，因為在小時候，常常被父親打壓，被父親嫌棄，因此一直沒有辦法正確地看待自己。

結果，進入了公司後，為了向領導和同事證明自己的能力，他常常會讓自己超負荷工作，把自己累得個半死。

久而久之，抑鬱情緒也就悄悄找上了他，逼著他暫停工作，回家靜養。

由此可見，「上進」雖說是個褒義詞，但是，無法接納自己的「偽上進」卻遲早有一天要把你逼瘋。

03.

應用：森田療法與抑鬱情緒

有別於其它的解決問題式療法，「森田療法」最大的特點，便是讓你學會去接受你的極限，而不是，逼著自己像是一把強弩之末一樣，一味地向前。

尤其是在面對不舒服的情緒時，你更是應該接納這些情緒，與它們友好相處，而不是拚命地想要去消除它們。

因為，你越在意，越是急於求成，越是無法理性思考，越是難以得到一個好的結果。

我給你舉個例子，抑鬱的一個典型癥狀，便是活力缺失。


這種活力缺失，一開始便既有身體上的病變，也有情緒上的無能為力。

比如，你在外面勞心勞神地跑了一天，應付了客戶的諸多刁難，那麼，回到家裡因為疲憊想要一個人待會，其實是一件很正常的事情。

既然如此，你為什麼不先讓自己休息好了以後，再來應付家裡人給你提出各種要求呢？

同樣，活力缺失既然是一種癥狀，就像發燒對於重度風寒感冒一樣，那麼，你不妨索性先接受自己是一個病人，然後安心去「享受」家裡人的照顧，靜待康復。

因為，只有當你不再逼著自己好起來，對自己未來也沒有那麼多要求後，你身上的擔子才會減輕許多，離康復也會更進一步。

以上，便是全書的內容。

雖然，我們從小便被教育著要上進，要努力，遇到困難，咬咬牙撐過去便好了，但是，有時候，我們不妨留出些時間給自己，放任自己去吃一頓好的，小睡一會兒，曬會兒太陽，和朋友聊天，泡個澡，等等，讓自己靜一會兒。

畢竟，你來到這個人世間，不只是來完成任務，達成成就的，感受生活也是很重要的一部分喲~

祝好！
