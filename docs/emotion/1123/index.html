測試解析

A：你未來的生活並沒有什麼區別，你不是一個急性子，你經常會把今天的問題留到明天去解決，所以有時你更像一個頹廢的人，沒有目標，更不要談夢想了，即使有時有自己的想法，但你沒有立馬去實施，所以這就導致幾年後你跟現在還是一樣，每每個月還依然是那個月光族，如果你接受了現在的情況，那麼你很難輕易的改變自己！

B：你未來的生活比現在更好。現在的你做事有點拖拉，這讓你工作起來會比較吃力，你意識到自己的問題後會想著要改變，但你不知道如何改變，如果你真的不知道如何改變它，那麼你應該去學習，經過時間的磨練，你知道你有什麼是你未來的努力方向，所以你熱愛你的工作並且在自己的領域做得很好，所以未來比你現在你過得更好的生活。

C：你未來的生活將是奢侈的。你生在一個好的家庭，但你不是一個啃老族，你是個很聰明的人，你應該知道如何利用身邊的資源，所以當你想做一件事，你會認真的去完成，並在中間會自己去學會利用手中的資源，這是你成功的關鍵，儘管在生活中會受到很多的質疑，但你從來都不會放棄！

D：你未來的生活比現在富裕多了。事實上，你的生活已經算是富裕的，但你追求更美好的生活，你不滿意你目前的狀態你會花很多時間來思考如何讓自己過得更好，如何給下一代創造更好的條件，你很能吃苦，所以你的努力和你的野心讓你在幾年變得更加富裕。

註：測試僅供參考，三分天註定，七分靠自己，祝你開心好運每一天！
