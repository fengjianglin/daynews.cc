一定要讓孩子明白這四字箴言，讓孩子健康成長

2019-12-16 155條評論 2,837次閱讀 1600人點贊

每個父母都會把最好的愛，給自己的孩子都希望在人生的道路上，孩子可以遇到挫折的時候不氣餒，遇到失敗的時候不墮落成為一個心智成熟，人格健康，三觀向上的人。作為父母，我們更知道，孩子終將會長大，父母與兒女註定是漸行漸遠，終於有一天是要分開的，人生的路很漫長，父母無法代替孩子來成長，必須讓孩子自己去體驗，所以我們不可能為孩子安排好一切，更為重要的是要讓孩子學會獨立，擁有面對風霜雪雨的能力，那麼父母一定要讓孩子明白這四字箴言，才能從容應對生活中的紛繁複雜的情況。

第一、自衛。作為家長一定要教會孩子如何去保護自己，尤其是女孩子，這個世界上有太多陰暗醜陋面，在孩子成長的過程中，做父母的不可能時刻陪伴在孩子的身邊，所以我們要教會孩子如何保護自己，是一件非常重要的事情，不要與情緒化，性格偏激的人交朋友，這樣的人容易做出極端的事，也不要做那些不安全的事兒，更不要去那些危險的地方，一定要讓孩子記住人身安全永遠是第一要位與財富地位，這些相比所有的一切都不如人身安全有份量，遇到危及人身安全的緊急關頭，任何事情都可以捨棄，唯有生命才是最重要的。

第二、自理，很多父母習慣像保姆一樣照顧孩子，包辦了孩子的所有事，很多孩子都已經很大了，都不能安排好自己的生活，甚至有很多孩子連洗衣服，疊被子，這樣基本的自理能力都沒有，孩子遲早要走出家門，離開家長的庇護。父母能照顧她，陪伴她一輩子嗎？如果你做不到，那就必須讓孩子學會獨立自主的能力，我見過很多孩子在第一次離開父母進入大學之後，第一次流下眼淚，竟然是因為不會洗衣服，收拾床鋪，被同學所歧視，如果一個孩子連自己都照顧不了，怎麼在社會上立足？成就自己的夢想呢？一無不掃，何以掃天下？這樣的警示，大家都應該明白吧，所以作為父母，應該從孩子小的時候就教會孩子如何去自理，從洗襪子，收拾玩具，整理書包，這些力所能及的事兒開始逐漸去培養孩子的自理能力，不然。孩子就會成為永遠都長不大的巨嬰。

第三、自律。缺乏約束的自由不是真正的自由，不自律的人就像是一匹脫韁的野馬，面臨的可能是萬劫不復的深淵，只有懂得自律的人才能讓自己更優秀。才能有底氣，自由行走在世間成就理想的人生。我們會發現，那些懂得自律的孩子，總是勤於鍛煉，堅持運動，身體更健康，不用任何人催促，自己就會安排好學習計劃，成績也會更出眾，當然，自律不是天生的，是需要父母耐心培養的，早點幫孩子養成良好的生活習慣和品性自律的孩子，醫生就會像開掛一樣精彩。

第四、自信。人能在一生之中取得成就，畢竟只有一個源頭，而這個源頭就是自信，孩子的自信與家長的培養有很大關係，生活中如果父母經常說一些打擊孩子的話，就容易讓孩子產生自卑的心理，對他們的成長。非常不利，而家長如果經常肯定和鼓勵孩子在他的內心就會充滿自信和勇氣，沒有什麼溝溝坎坎，能嚇得住他。自衛，讓人平安，自理，讓人成長，自律，讓人優秀，自信，讓人成功，一定要讓孩子明白這四字箴言，讓孩子健康成長。
