蘋果散發的香氣，彷彿是想你的催化劑。這組作品是米拍攝影師@John小強 拍攝的。

抖音上最近有一段話特別的火，你什麼時候去見那個你超級超級想見的人。

我聽到這段話的時候鼻子已經酸了，那個你愛的或者是曾經愛的人，現在卻連見面都如此困難。

多希望自己像金魚那般健忘，7秒便能將你遺忘腦海里。但自己並不是金魚，而你也永遠的印刻在我腦海中思戀的最深處。

每當走過那條街，看到那個景，下意識的就想對身邊的你說句話，卻發現你早已不在身邊。

房東的貓曾經有一句歌詞，我多想再見你哪怕匆匆一眼就別離，人的一生最難欺騙的或許就只有自己的感情了。

那個你愛過的人，哪怕有恨，卻依舊會在某個特定的時候想起。

在愛情中，我願意永遠是那個彈著吉他對你唱歌的傻瓜，很多事不看、不聞，只要你在我身邊，能感受你掌心的溫度。便是這輩子最大的小幸運。
