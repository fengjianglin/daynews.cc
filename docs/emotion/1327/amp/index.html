奶奶的痛悟：“我為年輕時做錯的事，付出了晚年凄苦的代價”

天天要聞
2019-12-05

摘要

犯了錯，早晚會付出或大或小的代價，根本逃不掉因果循環。及時發現錯誤，儘管改的艱難卻也不至於留後患，察覺不到則會像滾雪球，越拖越嚴重，只有不斷變大的份兒，甚至到了一定的程度，面對暖烘烘的太陽一時半會都融化不了。而心寒，亦是這個理。

犯了錯，早晚會付出或大或小的代價，根本逃不掉因果循環。

及時發現錯誤，儘管改的艱難卻也不至於留後患，察覺不到則會像滾雪球，越拖越嚴重，只有不斷變大的份兒，甚至到了一定的程度，面對暖烘烘的太陽一時半會都融化不了。

而心寒，亦是這個理。

當傷害日積月累，心涼透了，在需要對方的日子時，才驚覺對方的心狠，可對自己一手造成的後果，連哭訴說委屈的地方都沒有。

所以為人處世永遠要留有一線，別把一件事做絕了，把自己的後路堵死了，如此無論什麼時候都能有餘地。

但很多人都不明白這個理，習慣在能得意的年齡不管不顧的隨心所欲，等到要依託的年齡，後悔才席捲而來，白奶奶就數其中典範。

來自一位白奶奶的痛悟：「我為年輕時做錯的事，付出了晚年凄苦的代價」。

我媽每次提及這位白奶奶都是又可憐又生氣，可憐她一手拉扯大三個孩子老了沒人照顧，可氣她年輕做的糊塗事，親手促成了現在的局面。

白爺爺三十多歲的時候在工廠幹活出了事故去世，拿了筆賠償款的白奶奶一個人肩負起來家庭的重擔，拉扯兩兒一女。

由於丈夫的去世，她為了不被輕視欺負漸漸變得十分潑辣，在家對孩子給予的希望也各有不同，而這便是她做錯的第一件事，重男輕女。

雖說賠了錢，但也不夠坐吃山空，所以在兩個兒子上大學，小女兒高考完後，她為了減輕負擔拒絕女兒上學，要求輟學打工替家裡減輕負擔。

哪怕女兒哭著求她，說勤工儉學不拿家裡的錢，她都不答應，因為她覺得自己夠好了，像別人家的姑娘，還沒上過高中呢，於是在她強勢的把控下，女兒早早去了廠子上班，兩個兒子繼續上學。

從這起，母女之間的關係有了難以癒合的隔閡，加上日積月累下的一些不公平待遇，把母女情分一點點消耗殆盡，終止於女兒遠嫁，一年到頭不回一次家中。

即便如此，白奶奶也不認為自己做錯，還一直埋怨女兒心腸毒。錯過了緩解的最佳時期，最親的兩個人徹底疏遠了。

白奶奶說她最後悔的是想著以後靠兒子養老，女兒是別人家的，從而區別待遇寒了她的心。

但這時的她完全沒有意識到錯誤，只顧著給兩個兒子張羅親事，她做錯的第二件事，源於性格強勢的問題，得罪了小兒媳，偏心了大兒媳。

白奶奶的大兒媳嘴甜會說話，加上早幾年嫁進來的緣故，婆媳相處的和諧融洽，而小兒媳嘴笨不會來事，白奶奶有點瞧不上，在大兒媳有意無意的挑撥下，時常刁難一二。

讓兩個人關係徹底水火不容的導火索，是小兒媳嫁過來兩三年沒生孩子，就經常指桑罵槐說對方是不下蛋的老母雞，後來生了個女孩，她心裡介意連月子都沒伺候，從這起，彼此的梁子算是結下了。

在小兒媳的對比下，白奶奶對生了兒子的大兒媳更加偏愛，於是在一起過的那段日子家裡啥苦活累活都派給小兒媳，而大兒媳天天只需帶孩子就行。

別的小事或許能忘，可月子仇讓小兒媳用實際行動表明，她能記一輩子，至於偷奸耍滑，愛倒閑話的大兒媳，她心性的自私涼薄，根本靠不住。

做錯了這件事的白奶奶，讓她到了老年沒有一個人肯管。如果說得罪了兒媳還有兒子，那麼白奶奶做錯的第三件事，把錢私自補貼給了大兒子，讓她失去了最後的倚仗。

大兒媳為什麼對白奶奶那麼傷心？追根究底是她惦記老太太手裡的那點錢，為了給自己爭取利益，踩妯娌捧婆婆。

兩家分開過了之後，白奶奶住在大兒媳家，天天帶孫子，反之小兒媳家幾乎不怎麼去。

於是在這期間，大兒媳天天叫窮抹淚，幾個來回後，她手裡的一點存款全補貼給了大兒子，給自己一分沒留，給小兒子也沒補貼過一分。

當試探到婆婆沒錢了，大兒媳的態度就變了，對白奶奶愈發的隨意起來，沒過多久，就提出生了兩個兒子，天天住在一家算什麼回事，得要兩家輪換著住的意見。

起初小兒子認為理所應當，就答應了，但在後面發現老媽把錢全給了老大家，心裡就不滿了，為此兩家人大吵了一家，然後白奶奶就遭到了嫌棄，被兒媳婦摔盤子砸完，被兒子冷落。

現在白奶奶即使一個人住，也要仰人鼻息，受人臉色的生活，因為吃飯因為生病。

有時我媽去看她，她經常拉著我媽說，「侄女，年輕時做錯的這三件事，代價是老了無所依，我是真的後悔，你可別犯和姑一樣的錯。」

聽到白奶奶的痛悟，只能用一句可憐之人必有可恨之處來概括，心疼她現在的遭遇，又怪她一切咎由自取。

從白奶奶的一生，可以總結出三點，一是永遠別重男輕女，手心手背都是肉，即便做不到公平，也要盡量減少偏心，避免和子女產生隔閡，讓兄弟姐妹之間有了嫌隙。

二是人沒有誰高誰低，給予相對的尊重，為以後積攢福氣，否則傷了兒媳的心，你和兒子的也會疏遠，畢竟枕邊風的威力不容小覷。

三是必須給自己留養老錢，要明白兒孫自有兒孫福的理，別把自己最後的退路補貼給孩子，當你手裡有閑錢，老了沒人依靠也能有尊嚴的活著。

做對了這三件事，為自己留三條後路，要是條條都通，你老有所依，要是前聊條靠不住好歹還能靠自己。

別學白奶奶，在年輕時把這些事搞砸，等到晚年付出凄苦的代價。
