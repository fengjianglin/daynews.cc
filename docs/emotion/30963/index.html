16.不要裝得太堅強，別人會把它當成傷害你的理由.

17.微笑著回憶你對我說的那些話，想想只是我太傻了.

18.有些話不必說出口，有些人不必放在心上.

19.膩了連謊言都不用編製了，剩下的只有冷淡的敷衍.

20.有些人明明說好了不在意，卻還是忍不住去關注.
