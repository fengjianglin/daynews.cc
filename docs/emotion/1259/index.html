媽媽利用孩子去“碰瓷”，原來這個世界上，有些人並不配當父母

2019-12-06 0條評論 20次閱讀 0人點贊

|本文由糰子媽育兒原創，歡迎個人評論、分享

「有媽的孩子像塊寶」在大部分人的認知里，媽媽對孩子來說就是全部，他們無比信任自己的媽媽。

但是這個世界上，有很多人是不配做父母的。孩子在他們身邊，時刻都存在著危險。

媽媽帶著孩子「碰瓷」？

制視頻中的的孩子站在一旁，這時剛巧有一輛車行駛過來，媽媽也「飛奔」過來。

就在我以為她要把孩子拉回去的時候，這位媽媽的舉動讓我很「無語」。

她不但沒有把孩子拉回去，反而將孩子推向了麵包車，好在當時車速不快，及時停了下來。

然而這位媽媽還沒有結束她的表演，她順勢躺在車的前面開始打滾。看起來就像是帶著孩子「碰瓷」一樣。旁邊的男子，把孩子拉了過去。

看到這裡我想說：世界欠你一個「奧斯卡」

後來了解到，視頻中是一家三口。因為夫妻吵架導這名女性情緒失控，做出了這樣的行為。

孩子在這場「碰瓷」中，雖然沒有受傷，但是在他被推出去的那一刻，估計心裡也會有陰影。

而那位媽媽有沒有想過，一旦出現了意外，他是否能承受的住後果？

這個世界上，有些人根本不配當父母

如果說剛才的那位母親是一時衝動，那下面這對夫妻則是「預謀已久」。

「孩子摔倒了才能騙到錢」

這是從孩子親生母親口中說的話，這個十幾歲的男孩叫小金（化名），身上和頭部有很多新傷和舊傷，而這些傷口都是「碰瓷」後留下的。

前後不下20次，小金被父母逼著碰瓷，有一次還被父親直接從三輪車上踹下車。

當兒子慢慢察覺到自己做的事情是不對時，他的父母逼著兒子下跪服軟。這種來錢快的方式，讓這對「惡魔」父母越來越上癮。

甚至在一次碰瓷後，摔成枕骨骨折，他的父母竟然想趁著兒子的傷比較嚴重，多出去碰幾次，能多騙一些錢。

後來小金在家裡偷了一些錢和戶口本，逃離父母回到老家。但無奈被父母哄騙回去，開始了更頻繁碰瓷。

終於在一次碰瓷後被警察發現，這才得以解脫，而他的父母也得到了應有的懲罰。

他說：這輩子再也不會相信父母了，如果我以後有孩子，一定不會讓他做壞事。

父母不慈，別怪兒女不孝

如果說「養兒為防老」，那有些家長想養出孝順的孩子很困難。父母沒有盡責，不能怪兒女不孝。

1）不負責任的家長，憑什麼要求孩子對自己負責

既然把孩子生下來，既要對他負責。但是不負責任的家長，從不關心孩子的成長，使孩子從小沒有感受到愛，他長大了之後自然不懂怎麼回報父母。

就像小金，在心底里已經對父母失望，甚至是絕望。當他慢慢長大回想起父母當年對自己做的這些事，感激之心從何而來？

2）經常打罵孩子的家長，當心「虐待輪迴」

對孩子非打即罵，稍有不順心就那孩子撒氣。這樣長大的孩子，在心裡更多的是對父母的怨恨，想要他孝順很難。

被打罵長大的孩子，承受身體疼痛的同時，內心也很受傷。經常遭受虐待的孩子，隨著年齡的增長會出現一些列的問題。

自卑、暴力、極端性格、而且還很有可能將這種暴力延續下去，變成一發不可收拾的「虐待輪迴」。

3）「上樑不正下樑歪」

如果想讓自己的孩子孝順，首先看看自己平時是如何對待父母的。

我們對長輩的樣子，大概就是我們老時，孩子對待自己的樣子。

如果孩子在父母身上沒有看到過什麼是「孝順」，那他必然不懂得怎樣孝順。

「上樑」歪了，「下樑」也不知道怎樣正了。
