哲理生活情趣短句：過了秋楓冬雪，便會迎來春天的櫻花

2019-12-19 2,194條評論 4,195次閱讀 1194人點贊

一、一輩子不長，要做自己喜歡的事，不要再對誰滿懷期待，愛情算個屁，發財最要緊！

二、原諒所有的人與事情，活在當下，少一些依賴，照顧好自己，相信前方的路上，總會有不期而遇的溫暖。

三、你再優秀也會有人對你不屑一顧，你再不堪也會有人對你視若生命。所以牛逼時不要得瑟，落魄時不要墮落。

四、在一切變好之前，我們總是要經歷一些不開心的日子，這段日子也許會很長，也許只是一覺醒來，所以耐心點，給好運一點時間。

五、永遠不要沉溺在安逸里得過且過，能給你遮風擋雨的，同樣能讓你不見天日。只有讓自己更加強大，才能真正地撐起一片天。

六、不想回的消息就不回，討厭的人就離遠一點，不開心就表現出來，不要總活在看別人臉色的生活中，得早一點變酷，可愛不是長久之計。

七、不論結局，感恩相遇。生3命中遇到的人，你只能盡量去愛他們，即使終有一別，也別辜負了相遇。

八、總會有一件事讓你瞬間長大，也總會有一個人讓你淚如雨下。願你早日領教這個世界的深深惡意，讓自己活得開心得意。

九、生活給你反擊的時候，你要做的就是啃一個甜甜圈，然後把壞情緒全吞下去，化食物為力量，而不是脂肪，哭有屁用，妝花了還得自己補。

十、別在年輕的時候，吃得最胖，活得最懶，愛得最傻，一生中最好的年輕，都是一晃而過的，願你能用心享受它。

十一、錯過了秋楓冬雪，便會迎來春天的櫻花，也就是說：今日遺憾的，來日一定會實現。

十二、沒有任何人會成為你以為的，今生今世的避風港，只有你自己，才是自己最後的庇護所。

十三、把自己抬得過高，別人未必仰視你；把自己擺得過低，別人未必尊重你；沒有人是完美的，無須遮掩自己的缺失。

十四、人的一生，豈能盡如人意，但求無愧我心，人生苦短，好不好都不必遺憾，樂不樂都不要失望，過得好是精彩，過不好是經歷。

十五、不要把人生浪費在討厭折磨你的人這種事情上，一生只有一次的人生，好好過都嫌時間不夠呢。你有你需要去愛護的人，好好去過你該過的人生。

十六、你滿臉笑意，可別人並不知道你牙冠咬的多緊。你笑得合不攏嘴，可沒人知道你哭的時候是有多麼的撕心裂肺。只有你很努力，才會讓人覺得你毫不費力。

十七、給物質生活做減法，給精神做加法，寵愛自己的家人，過力所能及的生活，並且感受到幸福，就是奢侈的人生。

十八、好脾氣的人不輕易發火，不代表不會發火；性格好的人只是裝糊塗，不代表沒有底線。

十九、留不住的人沒必要強留，不合適的人終究會分開，可以不聰明，但別不清醒。

二十、餘生沒那麼長，不用一味的付出，去慣得寸進尺的人，請忠於自己，活得像最初的模樣。

二十一、願以後的你：穿自己喜歡的衣服，和不累的人相處，和所有喜歡的一切在一起，過想像里的一百種生活。

二十二、任何事，任何人，都會成為過去，不要跟它過不去，無論多難，我們都要學會抽身而退。

二十三、有些事，不可避免地發生，陰晴圓缺皆有規律，我們只能坦然地接受；有些事，只要你願意努力，矢志不渝地付出，就能慢慢改變它的軌跡。

二十四、不要以靈魂知己的名義，去等不該等的人，去蹉跎不該蹉跎的青春。這個世界上，有些人，有些事，比愛情這種東西，更值得感動。
