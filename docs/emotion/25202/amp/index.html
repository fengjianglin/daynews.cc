哲理生活潮流短句：曾經的海枯石爛，抵不過你最後一句好聚好散

天天要聞
2019-12-21

一、我從不指望任何人心疼我為我扛下所有。沒關係天塌下來我自己扛。

二、不要為了迎合所有人，把自己過得這麼累，費盡心思讓所有人都開心，你會忘了自己該怎麼笑。

三、我在這個世界上活著，手無寸鐵。沒有人站在我身後，擋住洶湧的風浪。更沒有人能站在我面前，擋住浩瀚的陽光。

四、在一廂情願里只有賤沒有愛，我們都一樣，到最後跪著感動的只是自己。

五、你只是看起來很快樂，那些漆黑的夜晚。那些不敢哭出聲音的日子，眼淚在眼眶裡打轉的感覺。只有你自己知道。

六、遇見了形形色色的人之後，你才知道，原來世界上除了父母不會有人掏心掏肺對你，不會有人無條件完全信任你，也不會有人一直對你好。

七、我知道太敏感會心累，可我就是那種什麼事都往壞處想的人，畢竟提前失望總好過突然失望。

八、我以為只要很認真的喜歡，就可以打動一個人，卻原來，我只打動了我自己。

九、得不到回報的付出，要懂得適可而止。否則，打擾了別人傷了自己。

十、心累到一定的程度，連生氣和計較的力氣都沒有了。在我們說「沒事」的時候，卻往往是最難受的時候。

十一、大概是小學寫作文的時候，把「最開心的一天」和「最難過的一天」份額都用光了，後來的人生才會顯得如此平淡而無趣吧。

十二、感情這個詞，或許真的很吸引人吧，畢竟，我也曾不甘一切為它潦倒過。令人遺憾的是，只要再提起，我能想到的，是疲憊，沮喪，失望，和無能為力。

十三、這世上有很多事情是你無能為力的，好端端的身體突然就生了重病，深信不疑的人突然就背叛了你，剛剛還在微笑的自己突然就哭了，那個說著永遠在一起的人突然就走了。

十四、一個人熬過了所有的苦難，也就不期待一定要和誰在一起了；我曾誠心努力過，但結局我不想說。

十五、有時候，我們想笑，卻必須要裝出悲傷的神情；有時候，我們想哭，卻必須要擠出僵硬的笑容。

十六、這南牆我是撞了，丟了的自己是撿不回來了，我摔得最慘的地方，就是我重新開始的位置，餘生不用你指教了，這是我的感情，我自己收拾。

十七、難過就不要聽擾心緒的歌，餓了的話就吃東西，怕黑就不要待在暗處，你總是一個人，別為難自己。

十八、我們像是表面上的針，不停地轉動，一面轉，一面看著時間匆匆離去，卻無能為力。

十九、人與人關係脆弱又敏感最難受的感覺，莫過於很多關於我的事我自己都不知道，你聽別人說完了我的所有，卻從沒有當面問過我半句。

二十、更多的時候，身邊沒有舒服的沙發，沒有啤酒，沒有朋友，沒有可聊的陌生人，也沒有愛的人，沒信念，沒熱情，一無所有。

二十一、聽說幸福很簡單，簡單到時間一衝就沖淡，曾經的海枯又石爛，抵不過你最後一句好聚好散。

二十二、每次崩潰的理由在別人看來是小題大做，只有自己心裡清楚這根稻草到底壓垮了多少千斤重的難過。

二十三、好像沒有我，你的生活也不會有什麼差錯，你開心的時候照樣開心，難過的時候照樣難過，在你的世界，我與萬物並列，或許，不如萬物。

二十四、感情里最難熬的放不下大概就是，你我皆良人，我未曾做過傷天害理的事，你也未曾跨越鴻溝做對不起我的事，只是緣分這東西偏成全不了你我。
