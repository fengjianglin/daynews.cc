一人一個活法，若你總是羨慕別人的生活，自然無法幸福

2019-12-24 1,341條評論 2,277次閱讀 576人點贊

​你關注我、我陪你愛

文丨萱小蕾、又名漠泱

▼

知足常樂這個詞大家都知道，只是要確切用在自己身上、或時刻記得並提醒自己的人卻很少。

真知足和不知足之間，有一大票的人在掙扎，在徘徊，在自我折磨。

有些人甚至歪曲知足和不知足的意思，將知足說成是一種懈怠，是懶惰不求上進，而將不知足當成是一種積極進取。

不排除有些人總喜歡給自己的碌碌無為找借口，這類人可能懶惰，不求上進，總在原地徘徊，或是等著救世主來救自己，等著他人來拉自己一把，甚至懶到有人拉都懶得伸出手。

但這類人如果也知足，覺得自己有口飯吃就行了，那也無可厚非，是他自己願意過苦日子，並且還開開心心。

就像泥鰍喜歡在爛泥里打滾，你把它放在清水裡，它反倒不高興。

如果他不羨慕那些住在高樓大廈或別墅里的土豪，也不想吃那些美食大餐，每天只要有稀飯榨菜都很開心，那他也算是有境界，窮苦一點也沒關係，可能他自己還覺得挺幸福，無憂無慮不操心沒壓力。

但還有一些人，明明可以生活幸福，卻因為慾望無止境，總想繼續往上攀爬，自己卻又急功近利，力不從心，從而活得很累。

這種攀爬，不是要戰勝自己，不是比昨天的自己更好，而是總拿別人跟自己攀比。

比如鄰居家如何如何，親戚家如何如何，所以一定要比他們強，比他們厲害，比他們有錢。他們買了10萬的車子，自己至少買11萬的、20萬的。他們買了房子，自己就想買比他們更大的房子……

在這樣的對比之下，慾望不斷攀升，或者說對面子的看重不斷擴大，他們就自然而然活得很累。

當然，還有一些人不是為了面子，而是因為羨慕。羨慕別人住高樓，羨慕別人開豪車，羨慕別人的老公帥氣，老婆漂亮，於是別人有的他都想有。

如果自己有這樣的能力，有這樣的鬥志，並且在這個過程中覺得很快樂很有挑戰性，那也沒關係，這些羨慕反倒成了一種動力，一種正能量。

然而很多時候，這種羨慕並沒有讓他們覺得充滿動力，而是覺得頹廢沮喪，覺得技不如人或吃力。

於是容易因為羨慕過頭、轉變成嫉妒，而嫉妒很容易讓人「面目全非」，甚至想要搞一點破壞拉低別人。

這樣自然就不對了，就有點危險了，這對自己不利，對他人不利，完全就是損人利己。

如果有了這種心態，那每天可能都活得比較累，羨慕完別人、又痛恨自己追不上別人，每天都會活在糾結掙扎中。

其實呢，每個人有自己的生活面貌，他人看起來好像都挺好，但是每個人也有自己背後的故事。

有可能別人華麗的袍子後面滿是破洞，有可能光鮮亮麗的人，背後有不為人知的陰暗面和痛苦。

還有那些你看起來笑得很大聲的人，背著人的時候可能會流淚會憂傷。

那些看起來住在高樓大廈住在別墅里開著豪車的人，可能欠了大筆的債，時刻擔心被人追債。

而那些總是在旅遊在拍美景美照的人，有可能囊中空空，而且吃了不少苦。

又或是，單身的時候，羨慕人家有家庭有孩子，一家人其樂融融。可是可能在你看不到的地方，孩子在哭，夫妻在因為家庭瑣事吵架。

你羨慕人家一個人單身很瀟洒，卻不知道夜深人靜的時候、他不舒服的時候，想要有一個人倒杯水給他的時候，身邊空無一人，滿心凄涼。

所以說，你羨慕別人一生瀟洒，無牽無掛，別人可能在羨慕你有家有他有人等你回家。

總之，如果羨慕別人可以成為你的動力，那你儘管去羨慕那些看起來生活過得挺好的人，努力做到跟他們一樣，並享受這個過程。如果羨慕讓你覺得很累很痛苦，那就還是放下吧。

因為每個人都有每個人的活法，選擇最適合你自己的那一種，選擇你當下能選擇的那一種，並且享受你現在可以有的狀態。

不然你一直活在羨慕中，就會忽略你現在當下的生活，總認為幸福在遠處，所以看不見感受不到當下本就有的幸福。

很多時候，幸福只是一種感知能力。有些人遇到一點細微的小事情，都會覺得幸福。有些懂得知足的人，收穫一點點美好的事物，都會覺得心滿意足。

所以不要任由自己的慾望膨脹，不要總看別人擁有什麼，也要看看自己擁有什麼？

哪怕是細微的美好，哪怕是不足以為與外人道的小歡喜，只要能讓自己開心，能讓自己舒服，那就是幸福的源頭。

多珍惜這些細微的幸福，讓自己感受當下的幸福，才不會因為羨慕嫉妒讓自己面目全非。
