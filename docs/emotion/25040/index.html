父母偏心兒女時，不能太過分，容易讓人寒心

2019-12-22 944條評論 3,467次閱讀 2265人點贊

你關注我、我陪你愛

文丨萱小蕾、又名漠泱

▼

一家兒女多的，父母想做到毫不偏心可能比較不容易，因為這些孩子肯定也是個性不同，各有各的討人喜歡之處，各有各的不討人喜歡之處。

而那些懂事的，討人喜歡的，或許就讓父母刮目相看，多半偏向一點。如果這種偏向只是情感上的偏向，有時候不太明顯的話，也不會影響孩子們的關係。

但是如果偏的太明顯，而且又都牽扯利益的那種事情，可能就會讓兄弟姐妹們關係因此破裂，或者互相有隔膜，甚至對父母也有意見。

所以說，想要一碗水完全端平不容易，多少會有一些波瀾，但大方向上還是要公道，才能讓兒女們心裡都平衡舒坦，家庭關係也會因此更穩定更和諧。

很多家庭的偏心問題沒辦法解決，特別是在兒子和女兒之間，這種偏心的問題特別的多。

他們都覺得兒子才是自己家裡的人，而女兒終究會嫁出去，會像潑出去的水，所以偏向兒子好像是理所當然的事情。

然而有這種想法的人，要麼就做到不要佔女兒的便宜，不要想著讓她回報父母，或者為了讓她對這個家庭有利益，才將她出去，比如嫁有錢人家，要一大筆彩禮之類的。

或者說嫁出去了，就不要再繼續讓女兒為娘家付出，那麼他們偏向兒子還情有可原。

如果女兒嫁了人，娘家卻總是希望她再不斷付出，希望她持續為娘家的哥哥或弟弟出錢，出力。從而影響到女兒的生活，那就不對了，那這偏向也就太過分了一點。

阿華說，自己的老婆就遇到了這種偏心的父母。他岳父母就這麼一個女兒，但有兩個兒子。

阿華自己的條件還可以，所以當初岳父母彩禮要了20多萬，他也都給了，而老婆的陪嫁大概幾不超過五萬塊。

阿華沒有計較，因為他對老婆還是有真感情的。但是婚後，岳父母一直陸續讓女兒繼續拿錢補貼家用。

小錢不算，大筆的也有。比如兩個兒子買房子買車子的時候，大小事都不會忘記找女兒幫襯。

阿華說自己老婆也有賺一些錢，自己條件也不差，所以老婆幫助娘家，他也沒說什麼。

然而後來，岳父母那邊修高速路，給家裡賠了不少錢，卻問都沒問這個女兒，就讓兩個兒子均分了。

這一點讓阿華夫妻倆覺得不舒服，女兒雖然嫁了，但是這些年幫家裡也幫的不少。

岳父家二兒子結婚的時候，錢都是阿華出的，接近10來萬，說好結婚份子錢還給他們，結果一分沒有給。

大兒子買房的時候，阿華夫妻也出了10萬塊，之前沒有說還不還，現在賠的錢也沒有說要還他們錢，更別說分給他們一點。

阿華有意見，老婆也覺得寒心，對娘家也有了意見，所以兩個人決定以後盡量少跟娘家打交道。因為看出他們心裡只有那兩個兒子，沒有這個女兒。

阿華這件事情，或許很多人覺得：女兒是潑出去的水，是嫁出去的人，所以家裡面賠償什麼的、分錢什麼的跟她沒關係。按理說，也可以說得過去。

可是重點是這個女兒嫁出去這些年，娘家沒少麻煩她，也沒把她當成毫無關係的人，也總在找她幫忙、找她拿錢。

那麼即使不給女兒分錢，也應該把拿她的錢還一部分，這樣才說得過去。

而且即使女兒嫁了，也是一家人，不說跟兒子一樣分的多，多少給一點也算是心理上的安慰，也算是一點表達親情的形式。

可是這個娘家，完全都沒有想到女兒的處境，或者女兒會不會難過之類的，他們完完全全把她當成了一個外人，並且是一個他們想要索取的時候就可以無條件索取的外人。

這樣的家庭，只有在需要你的時候才會找你。因為他們覺得養女兒養了很多年，他們付出了很多，所以是他們理所應當得到回報。

但是他們自己有利益的時候，卻不會再為女兒付出了，可能覺得養大她就不錯了。

這樣的心理自然是不好的，傷了女兒的心，也傷害了這份感情。作為這個家的女兒，想必付出再多，父母和兄弟都會覺得應該，永遠不會感恩知足。

其實我們見過很多有兒有女的家庭，父母偏向自己的兒子，覺得自己老了要他們養老送終，自己生病了兒子會負責。

然而有很多時候父母生病，兒子兒媳不管，可能工作繁忙，也可能有別的借口。然後把照顧老人的任務交給家裡的女兒，也就是嫁出去的那個。

也就是說，父母有東西可以給兒女的時候，就給了兒子，認為他們才是自己最後的依靠。

然而有事的時候，卻發現兒子並不一定靠得住，反倒是那個女兒內心柔軟，即使娘家人對自己並不是很好，即使知道父母偏向哥哥或弟弟，也依舊會心軟，會覺得那是自己的父母。

所以無要件照顧他們、孝順他們、為他們付出是天經地義的事情。

遇到這樣的女兒，自然是父母的幸事。但是所有父母也應該明白：不管是兒子女兒，都不應該太偏心。

有什麼東西拿出來大家平分，即使做不到平分，也不要太虧待其中某一個人。

父母的表現平等，子女心裡也會舒服很多。

因為這也牽扯到一個愛的問題，給誰多，就表示更愛誰。

所以說，有時候兒女在意的不是那一點利益，而是誰被愛的更多一點，誰被偏愛、誰不被愛才是他們心裡的結。
