這位朋友告訴我

他已經三年沒回家了，自從欠了這百萬負債，從來沒有敢回過家，哪怕是偷偷，也沒有！他不敢回去，欠的債太多了，對於一個農村孩子來說真是天大的數目，他每天都躲在小屋子裡，晚上做夢都是要債的，沒有一次不被嚇醒的！痛不欲生，絕望！

他最挂念的就是他的母親，60多歲了，身體一直不好。常年吃藥維持！還要面對很多要債人的臉色。他好難受，撕心裂肺的痛，不但沒有讓父母安詳晚年，卻讓兩個老人跟著自己被人追債，被人嘲笑，被人看不起，他好想回家，真的好想，做夢都想回去，他覺得沒臉回去，對不起父母，也不敢面對那麼多的債主

從小媽媽對他就很疼愛，都沒捨得打過他，父親是個老實本分的人，平時不愛表達，在家也是想著辦法掙點錢維持家裡的生活，媽媽說，他父親愁的也睡不著，整個人瘦了兩圈，每天唉聲嘆氣。此時他說到這裡，突然大哭起來，我沒有見過一個男人哭的這麼狼狽，我不知道到怎麼勸他，我的眼淚也掉了下來！只是拍了拍他的肩膀，想哭就哭吧，也許哭出來會好一些！

負債很可怕，使人絕望，束手無策！更何況一個深陷巨額負債的80後！人在外面漂泊時間長了，無論你有沒有外債，都會感覺到累！能回去就回去呆一段時間吧！家裡的父母也需要我們去陪伴，雖然他們不說，但是無論你多大年齡，在他們面前都是孩子，他們依然愛你，為你擔心，挂念你，

你有多長時間沒回家了！多久沒給父母打電話了，知道嗎？他們一直在等你回家，哪怕是吃上一頓飯，說上幾句話，都是他們的奢侈！他們的時間不多了，能多陪陪父母，就陪陪他們吧！事忙不完的，錢也賺不完的，平平安安，健健康康，和和睦睦，才是人生的真諦！

再次希望大家有能力的多幫幫身邊那些落魄的朋友，他們的落魄也只是暫時的。也許指不定那一天我們也會落魄，也需要幫助！人生很短，能一起走的，請拉他一把！他會記你一輩子
