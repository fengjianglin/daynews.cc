走過的是人生，看過的是雲煙；嘗過的是苦辣，回味的是甘甜，早安

天天要聞
2019-12-21

1、人的一生,其實就是一次漫長的旅行。有時一人獨行，有時二人結伴，有時三人前往，有時成群結隊，途中走走停停，不斷有新人入伙，也不斷有舊人掉隊。一路走來，風風雨雨，坎坎坷坷。沿途的風光，有的已存為永久的記憶，有的因為走的匆忙而疏於收藏。

人生猶如一首歌，音調高低起伏，旋律抑揚頓挫；人生彷彿一本書，寫滿了酸甜苦辣，記錄著喜怒哀樂；人生又像一局棋，布滿了層層危險，也撒遍了道道機遇；人生恰是一條路，有水窮山盡的坎坷，也有柳暗花明的坦途；人生如同一條河，有時九曲迴腸，有時一瀉千里；仔細一想，漫漫人生路，從小到大、從大到老、從老到終實屬不易。多少回迷茫不知道該去何方，多少次迷失了自我找不到前行的方向，多少回沐浴著春風喜氣洋洋，多少次享受著陽光心情激蕩，細數珍藏在心底的風景，有幸福，有憂傷。早安！

2、 每個人都把這難忘的記憶珍藏在心底，永久收藏。有些人總是覺得自己擁有得太少，總是感覺自己壓抑、失落、不快樂，其實是他追求的太多，想要得到的東西太多。人生在世不可能事事順心、萬事如意，可總是有一些人，明明知道有些事情根本無法實現、有些問題根本沒有荅案、有些故事永遠沒有結局、有些追求永遠都是夢幻。儘管如此，這些人還在苦苦追求，還在黙黙等待，還在空空的幻想，你說他們苦不苦？你說他們累不累？只有順其自然、坦言相對，才能隨遇而安、開心開胃。

人之所以不快樂，就是因為計較的太多，想要得到的東西太多，該爭時必爭，不該爭時應及時退位，要學會把握時機，審時度勢。總想過上高品質生活的人，整天總是忙碌著、奮鬥著，得到了開心一時，得不到痛苦一生。人之所以不知足，就是因為虛榮心太強，太好面子。早安！

3、 俗話說知足者常樂，但世上能有幾人能達到這樣的境界，大千世界無奇不有，人們總是面對太多的誘惑，不可能不動心、不可能不奢望、不可能不幻想，很多人在金錢、美女等等的誘惑面前，沒能好好的把握自己，迷失了自我。人之所以不幸福就是因為人們不會珍惜，人們面對著一天好似一天的生活，面對著越來越好的條件，總是想往更高層次跨越。

其實幸福是一種期盼，是一種美好心靈的感受，只要我們珍惜擁有的一切，用心去發現，用心去感受，你就會發現你就生活在幸福里，幸福就在你身邊。人之所以活得累就是因為想的太多，整天背負著沉重的思想包袱，心累就會危及你的健康，增加你的煩惱，你定會感覺氣喘吁吁，力不從心。只要你放下思想包袱，遇事不專牛角尖，你就能在平靜中享受生活，在平淡中品味生活。早安！

4、人之所以會煩惱，就是因為人們的記性太好，把該記的不該記的裝滿了頭腦。有人說：傻子最可愛，因為他忘記了人世間的恩恩怨怨；忘記了人們對他的冷嘲熱諷；忘記了世俗的功名利祿；忘記了世間的爾虞我詐，他只活在自己的世界裡，為所欲為、隨心所欲。所以，傻一點更可愛。

走過的是人生，看過的是雲煙，嘗過的是苦辣，回味的是甘甜，經歷的是滄桑，成就的是遵嚴。在人生的舞台上，每個人都綻放出不同的光采，你愛虛榮、就會被獻媚所迷惑；你愛錢財、就會被金錢所收買；你愛奢華、就會被糜爛所腐化；你愛美色、就會被色情所迷茫。心中有了空隙，就會被外力所感染。心中裝滿自己想法的人，永遠聽不到別人的心聲。人就是這麼矛盾，顯得格外可憐，經常站在十字路口猶豫不定、止步不前，荒度人生。早安！
