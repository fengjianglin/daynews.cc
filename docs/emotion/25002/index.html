繼母去世，班級里他一滴眼淚沒掉；清明節，扣扣空間他留言5年

2019-12-23 1,245條評論 4,050次閱讀 379人點贊

文/趙主任

一個小學同學做了一個品牌汽車地區的總代。

我一朋友要買車，小學同學給足了面子，便宜了好幾萬。

實話實話，我和小學同學當年的關係，很到位。

到位什麼程度？

他屬於上學時，很另類的那種。

班上每個同學都討厭他，包括老師。

就我願意搭理他。

小學同學是個命苦的人。

他媽媽在他很的小時候，就意外過世了。

爸爸給他找了後媽，後媽還是他的初中班主任。給我們帶班的時候，他後媽就吵吵胃老疼。他爸和他說，醫院檢查了，情況不太妙。

同學貌似也不太關心。

班級同學都說，後媽對他不冷不熱。不過我覺得挺好的，不過他後媽有時候，也挺惡毒。

記得最狠的一次，班級上有三個人在看小說，班主任把就把他揪了出來罰站3小時。

我同學和他後媽強了幾句：那麼多人看，為啥就讓我罰站！

班主任氣得把小說撕得粉碎，扔在他臉上：滾！

還有一次，班級掃雪，班主任給他分的區域最多。

零下30度的大東北，同學氣的直哭：我什麼時候能幹完！

班主任：干不完，別回家吃飯！

那天，同學氣得一夜沒回家，跑我家和我住了一個晚上，我在被窩問他，你媽在家對你好么！

他說，我和她沒話。

後來據說，他後媽冒著大煙炮，找了他半宿兒，後來從班長那得知，他兒子在我家，才回家安心睡覺。

班主任給我們帶了不到一年，就住院了。

新來的老師說，班主任的情況不好，應該不會再給你們接著帶班了。

我偷偷問同學：你媽咋了！

同學一邊翻著小說，面無表情：胃癌！

那個表情，就像在說鄰居家的媽媽。

班主任（同學媽媽）去世那天，班級誰都不知道，就看很多老師往外走。

中午，他和往常一樣，和我在食堂一起吃飯。

我問他媽媽怎們樣了！

他異常平靜：後天出殯。

下午他和往常一樣，班級里曬著太陽，看著小說。

我們全班在傷心。

我有時候挺可憐他，命真苦！

又恨他：為什麼心冷如冰，雖然是個後媽，也算是養你十幾年，就是鄰居家小貓沒了，我們還要心疼一陣子，他卻一滴眼淚沒掉！

事情過去好一陣子，他就當什麼也沒發生。

八月十五，中午，我那天給同學點了蛋炒飯。

同學沒吭聲，一邊吃蛋炒飯，一邊看小說。

突然發現他的書濕了，抬頭一看，他劈里啪啦的掉眼淚。

我問咋了！

他說，她以前每天早上，都給我做蛋炒飯。

熙熙攘攘的同學，誰也沒在意，一個學校沒人搭理的性格古怪之人，在飯堂的角落裡，一邊吃飯，一邊劈里啪啦的掉眼淚。

那天，我媽媽給我帶的鹹菜，我帶到了食堂。

他問這是誰給你的。

我順口來一句，我媽！

他一口沒動。

中午午休也不見他。

最後我在學校旁邊的池塘，找到了他。（他經常一個人去那）

他看我來，一下撲在我懷裡。

那天，至少在我懷裡哭了十分鐘。

他那個碩大的肚子，我瘦小的身軀，也強撐著。

初中畢業，同學成績考的一般般。

進入社會，我們也沒怎麼聯繫。

聽別的同學說，他性格大變樣了，這些年混也不錯，做事情和他媽一樣霸氣！

買車的朋友為了表示感謝，要請我和我這個小學同學吃頓飯。

我說，不用！

小學同學說：必須的！正好今天咱們聚聚。

雖然以前在初中一起搭夥，畢竟這麼多年沒聯繫，我們之間沒有啥共同話題。

席間，就是一些：你家孩子多大了之類的寒暄。

喝著喝著，感覺進入狀態了，也許我倆太長時間沒見面了。

小學同學念起了酒嗑：

班級我最後悔的事情兩件，一件就是畢業沒和咱班級合影，另一件就是。

同學眼圈紅了。

買車的朋友看出了端倪，提前結了賬，找個理由先走了。

我倆出門送送，送走後，同學突然回頭抱著我，然後趴在我的肩膀上，大聲嚎哭起來。

旁邊的保安一定在想：這個傢伙真沒少喝！

其實，那天我倆每人就喝了半瓶子啤酒。

那天，兒子借我的扣扣上他同學群（我兒子沒有手機），我幾年也不用扣扣，順手翻翻我的好友，無意間看到了這個小學同學的動態。

發現每年清明節，他都在空間里，發兩朵鮮花。

還有，那天飯店臨走時，我們去洗手間。

聽見一個酒多了的主兒，在衛生間嗷嗷直吐，邊吐邊哭：我媽咋就沒挺過來呢！馬上就要過80了！

小學同學在洗手池低著頭洗臉，洗了老多遍，我恍惚看見他委屈的眼淚，劈里啪啦不停的掉。

委屈的樣子很可憐，可憐的像一個沒媽的孩子。

回家，媳婦說：我生平最恨這種人，就是在你面前，裝作無事。一旦觸景生情，就哭的不行。

我是趙主任，每天和你一起懷念良心。
