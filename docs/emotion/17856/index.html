他辭去城市高薪，回鄉養花，眾人嘲笑太瘋狂，一年後，羨慕到眼紅

2019-12-18 978條評論 2,357次閱讀 889人點贊

我們常說平平淡淡的生活才是最真實的生活，但是有時候其實生活還是需要一些瘋狂的，如果沒有這些瘋狂的舉動，那麼再好的生活也總感覺缺了一點什麼。而今天故事中的這位主人公，便做了他人生中最瘋狂的一個事，他辭掉了自己在大城市的高薪工作，回鄉做一個養花的花農，相信這在我們很多人看來，都是一個瘋狂到不能再瘋狂的事。

他辭去城市高薪，回鄉養花，眾人嘲笑太瘋狂，一年後，羨慕到眼紅！

起初遭到眾人不理解的嘲笑，後面，他用了一年的時間打造一個鄉野花園，讓當初嘲笑他想法瘋狂的人，羨慕到眼睛都紅了。

其實這應該就是一個在大城市中生活久了的人的正常心理，已經城市奔波20多載，不想再繼續過這樣的生活，相反只想回到自己當初的老家，安安靜靜遠離喧囂。屋後種一片菜園，門前來一個花園，每天看著花起花落，四季流轉，這才是我們該有的生活，也才是所有人心之所向的生活。

在最開始，這位小哥回到家鄉種花的時候，眾人皆不支持他，也不理解他的行為，畢竟一個人放棄城市的高薪回到農村，干一個毫無收入的事情，確實是不那麼容易被接受。但是最後，他卻依靠自己的堅持和毅力讓這一個所有人都認為不能夠成功的事情成為了現實。

他給自己原本不大的小樓房旁邊建了一個小花園，在花園裡面種著各種各樣的花。以前院子的外圍是鐵制的圍欄，為了讓月季更好的生長，它將所有的圍欄全部拆掉了一層，做成磚制的圍牆，然後圍繞著這一堵牆，周圍種滿了月季。不同的品種的月季開出的花色不同，到了月季盛開的時候，我們看到的並不是圍牆，而是一朵朵五顏六色的月季花。當然了，如果說僅僅在院子裡面種上一條樂器，就能夠讓人心服口服的話，這也是說不通的。

所以他肯定也不僅僅只是種了月季，還在自家的後山承包了一大片的農田，在農田中種上了紅杜鵑以及各種各樣的薔薇花。雖然說這些薔薇花在深山中沒有人欣賞，但是它們依然在努力讓自己變大變美。一年後那一片原本普普通通的小農田，變成了一個五顏六色的薔薇花海。這也讓當初那些不理解他行為的人生投射出了羨慕的眼光。

大家是不是也非常羨慕這樣的生活呢？如果喜歡的話，也不妨從家中的陽台開始種上自己喜歡的花草吧！相信很快的一天，大家也能夠擁有一個漂亮的小花園呢！
