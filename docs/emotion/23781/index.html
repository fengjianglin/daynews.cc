職場上，做到這樣別人就會喜歡你

2019-12-19 223條評論 4,507次閱讀 3348人點贊

卡耐基的《人性的弱點》的第二篇就講到如何讓別人喜歡你，總結起來的方法如下：

圖片來自於網路

第一、真誠地關心別人

維也納著名的心理學家阿爾弗雷德·阿德勒在他的《生活的意義》書中說過：「對別人漠不關心的人，他的一生困難最多，對別人傷害也最大。所有人類的失敗，都是由這些人造成的「。

但在現實生活中，我們往往只想引起別人的注意，讓別人對我們留下印象，這樣，我們就不會有許多真摯而誠懇的朋友。朋友，真正的朋友，是我們要主動地注意和關心別人。

如果我們想要交朋友的話，就要用生機和熱情地應對別人。對別人表示你的真誠關心和關注，這樣不但可以讓我們交到許多朋友，還可以為我們自己或公司增加客戶的忠誠度。

對他人表示關心，這與其他人際關係一樣必須真誠。這不僅使得付出關心的人會得到相應的回報，而得到這種關心的人也會同樣有所收穫。如果想讓別人喜歡我們，或者培養真正的友情，或是既幫助別人又幫助自己，那就是要做到：真誠地關心別人。

圖片來自於網路

第二、記住別人的名字

記住一個人的名字，對別人來說是所有語言中最甜蜜、最重要的聲音。我曾經在湖南一個地方做過職業學校的老師和班主任，因為那時才畢業，做班主任和老師沒有什麼經驗，學校校長在一次開會中就說：做班主任最低的要求就是在三天之內記住學生的姓名，及每個學生的家庭成員情況，一個星期之後到什麼地方都能叫出班上學生的姓名，另通過了解家庭情況才能更好做好這個年齡段學生的思想教育工作，這樣做班主任才算合格。我也是按校長這個方法去做，也做到這樣，果然，我班上的學生，無論是調皮還是不調皮，對我都一種敬畏，也把我當作一位可信的老師和兄長來看待。

記住別人的名字，後來我從學校辭職，一直沿用到企業工作和管理中，在一個製造業的企業里，公司有七八百人，幾乎所有的人我都能叫出名字。後來，也當上部門經理，對自己下屬的姓名、家庭情況、生日，那時我必須要了解的，這樣也讓大家對我很信任，覺得自己是一個真誠的人，有的同事，正因為這樣變成了生命中最好的朋友。

圖片來自於網路

第三、學會微笑

整天拉著一個臉，誰都不願意和這樣的人做朋友，好像別人欠他錢一樣的，相反的是微笑是人類最好的語言。一個微笑，無色，卻能讓世界色彩斑斕；無味，卻能芳香滿人間；無形，卻改造著世間萬物，撫慰人間冷暖。

微笑能給家庭帶來幸福，能給人生帶來好運。它會使疲倦者感到愉悅，使失意者感到歡快，使悲哀者感到溫暖。微笑買不著，討不來，借不到，偷不走，微笑是無價之寶，有人過於勞累，發不出微笑，把你的微笑獻給他們，那正是無助疲乏的他們最需要的。

圖片來自於網路

第四、做一個善於傾聽的人，鼓勵別人談論他們自己。

一心只談自己的人只會為自己著想，所以要成為一個善於談話的人，就要做一個善於傾聽的人，要使別人對你感興趣，首先就要對別人感興趣。一個與你談話的人對他自己、他的需要、他的問題比對你及你的問題感興趣超過上百倍，一個工作不順利的人，怎麼改變自己現狀，比馬雲今年賺多少錢的更重要，比哪個明星又出軌更關注。

圖片來自於網路

第五、談論別人感興趣的話題

談論別人感興趣的話題，雙方都不會有損失，而且每次與人談話時，這種收穫從整體上會豐富我們的生活，也會從別人那裡見識到更多的東西，彌補自己短見的不足。

圖片來自於網路

第六、讓別人感到自己重要，並真誠地照此去做。

我們真誠地關心別人、記住別人名字、對人微笑、善於傾聽、談論別人感興趣的話題，在職場或生活中，還是要做到最後一點，就是讓別人感到自己的重要，能幫別人解決問題，實現自己的價值，否則也只是一面之緣而已。
