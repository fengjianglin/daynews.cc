人到中年，品嘗了世事無常，也就看淡了人走茶涼

2019-12-21 636條評論 2,595次閱讀 547人點贊

世間所有的相遇，都是一場久別重逢，所有的經歷，都是世間給予的最好安排，所有的發生，都是人生美好的回憶，所有的失去，都是上天賦予的另一種得到。世事無常，我願自己能夠學會品嘗。

人生路上有兩樣東西最難得，一是知音難遇，二是世事無常。

1、白落梅說，世間所有的相遇都是久別重逢。

人的一生會遇見很多人，擦肩而過的陌生人，相交不久分離的人，還有相愛不能相守的人。也還有更多的是不曾謀面的人，網路時代，追星人很多，看看就罷了，若真的捨去一切去追最後會讓自己失去自我也連累家人。

如果我們去留戀每個人，心裡一直放不下已經錯過的人，那樣現實生活中的人，身邊人的也將錯過，此生，豈不在悲哀和悔恨中度過。

那些為情所困，被情所傷的人，誰不是固守著曾經的諾言，在自己描繪的世界裡獨自暢想未來的美好。

走遠的人，已經走了，不會再回來的，即使回來了也不再是曾經的那個人，心變了，情也變了味道，再相守必定會是彼此折磨。

那些說過的誓言和相擁時的蜜語，在時間長河裡依然走遠，不必糾結，也不必難受，唯有放下，才能夠走出來。

都說，明天和意外誰也無法預測那個先到來，看淡人間的無常是常態，不變的是人生的變化。

再滾燙的水也會有涼的時候，再飽滿的熱情也會有冷淡的時刻，再愛的人也許會因某些原因離開。

那些信誓旦旦的對你說一輩子都不分開的人，在不知不覺中就漸行漸遠。那個曾在月下發誓今生只愛你一個人的人，也總會在半路就走散了。

人生就是這樣，一邊擁有一邊失去。會被人毫無徵兆的愛著，也會被人悄無聲息拋棄。

在人生長河裡，品嘗了塵間的世事無常，也就習慣了人走茶涼，也就學會了獨自成長和孤單寂寞時的勇敢。

2、人到了一定年歲，經歷了一定世事變化後，會說，如果當初怎樣，那麼現在就會是怎樣的……總會對現在的日子不滿，暢想未曾過了日子。

比如當年畢業選擇自費去學習這件事，很多人上班後可以帶薪去深造，也有的人選擇自己放棄工作自費再去深造，而沒有去的人，若干年後，會很羨慕他們自由的生活，也許，當時因某些不得已，也許只是貪戀安逸的工作和生活，就是感覺別人的人生都是精彩絕倫。

二十年前的帶資分流時，每個人都是矛盾體，不帶資吧，看不到未來，拿著現金走的人呢，出去了才發現世界那樣大，就是沒有能夠讓自己歡喜的工作。

二十年後，出去闖的人感嘆自己很累，不如在單位里安逸，就在單位的人看到出去人的自由，還有的人做的風生水起的，也是羨慕不已的。

女人會在歲月里感嘆，若當年選擇了那個誰是不是比現在過的好呢？

男人也會在寂寞獨自抽煙時感慨，若是當年和某某結婚，是不是現在會少聽那樣多的嘮叨呢？

但，如果當初選擇了另一個人，選擇另一份工作，選擇和現在不一樣的生活，今天的我們就一定過的快樂嗎？

世事無常，真的不好說。

現實中那樣多再婚家庭，真心真誠相待的又有幾人，大都還是生活的一地雞毛，還是把曾經的路再走一次。

正如張愛玲說，娶了紅玫瑰，久了就成牆上的一抹蚊子血，白的還是「床前明月光」，娶了白玫瑰，久了就成了衣服上的一粒飯粘子，紅的卻是心口一顆硃砂痣。

其實，無論選擇誰，都會在生活中有這樣那樣的不如意，也不會盡美盡善的。

生活的精彩在於它的不確定性，也在於未知性，我們只要用心有心的為未來努力去做就好。

生活中的不可預知在於不知道下一秒會發生什麼事情，但，壞的事情不會總是這樣壞下去，好的也不可能永遠好下去，都會在時間的長河裡變化，也就是所謂的世事無常，才會演繹著生活五彩斑斕。

我們能夠做的就是靜心品嘗，耐心等待，用心去做，勇敢的去面對，不可一味的賣慘，說喪。

3、在日復一日的生活里，我們會遇見形形色色的人。

有在困難艱苦時扶一把的人，也有在你陷入困境時看著說笑再踩一腳的人，更多的是萍水相逢就各自東西的人。

那些無緣的人，我們不必強留，畢竟人留心不在。那些經過風雨依然相守在身邊的人，是我們今生最要珍惜的人，因一份情緣來之不易。

人生這一路，說長不長，說短不短，有人從起點陪伴你，卻在中途下了車，有人半路遇見，卻陪你一輩子。

有的人相遇了，是為了教會我們成長，然後就走散了。有的人相遇了是為了告訴我們如何去愛別人。還有的人相遇，是為了告訴我們該如何用一顆禪心來看待塵間，面對世間的變化無常，教會我們做一個閑看雲捲雲舒，靜待庭院花開花落。

能夠在塵間相遇，都是一場前世今生的重逢，我們所有的經歷都是塵世間最好安排，所有發生，也都是上天安排的定數，得到是幸運，失去也是一種成長的費用。

過去的已然過去，不必遺憾；未來的還未到來，不必憂慮；現在的正在進行，請活好放下。

世事無常的生活，看淡便放下。

人走茶涼的真實，想開便隨緣。
