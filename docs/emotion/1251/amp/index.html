在職場上，每個人都努力工作，積極表現自己，這不僅是為了能夠有好的業績，同時也希望能夠得到領導的認可，獲得領導的重用。

但有時候，也會適得其反，給領導留下不好的印象，被領導排擠。

琳琳剛進新公司不久，雖然與同事們相處的很好，但是卻始終得不到領導的信任，領導經常讓她負責一些雜七雜八的、很瑣碎的工作。

為了獲得領導的信任，博得領導開心，琳琳很努力的工作，經常加班工作到很晚才下班，終於在月底的時候，業績達到第一名，獲得了領導的肯定。

那之後，琳琳工作開始獨斷專行，做事情經常要同事來配合，有時候甚至幫領導做決定，這樣的行為嚴重威脅到了領導在公司的地位，導致她更加不受領導待見。

領導雖然表現不說什麼，但是有好的項目，領導從來不交給她，甚至聯合同事一起排擠她。

在工作中，業績能力突出並不一定會得到領導重用，想要獲得領導的喜歡，情商也是很重要的。即使不能被領導喜歡，也不能被變成領導討厭的人，否則你在職場中的處境，只會更加艱難。

在職場中，不管是做事情還是與人交往，都要小心謹慎，千萬別讓自己成為領導不喜歡的人。通常這幾種人，領導最不喜歡。

1、無中生有，造謠生事的人

這種人在職場中，應該是很不受領導待見的，他們最喜歡無事生非，喜歡製造麻煩。他們最喜歡挑撥離間，不僅挑撥同事之間關係，還會挑撥同事與領導之間的關係，讓他們之間產生嫌隙，每天勾心鬥角，將辦公室搞得烏煙瘴氣。

雖然工作能力一般，但是惹麻煩的能力卻不一般，經常給領導惹麻煩，散播不好的言論，所以這樣的人，在職場很不受領導的待見。

2、得過且過，不求上進的人

當一天和尚撞一天鐘，說的應該就是這類人，他們在職場中是不會受領導重用的，因為他們沒有上進心，做事情得過且過。

每天只知道完成自己手頭上的工作，從來不去主動思考，更不會考慮自己未來的發展和公司的前景，沒有任何責任感和上進心。

所以，這樣的人在職場中，是永遠不會得到領導重用的。

3、自作主張，愛搶風頭的人

這種人看似很有想法，但是處理事情經常自作主張，從來不顧及別人的感受，有時候甚至會直接幫領導最決定，挑戰領導的權威。

在職場中，最忌諱的就是不尊重領導，處理事情根本不問領導的意見。更有甚者，情商低，喜歡搶風頭，不僅會搶佔同事的功勞，還會搶領導的風頭，到處得罪人。

所以說，這樣的人在職場上很容易受到排擠，被領導討厭的。

4、斤斤計較，經常抱怨的人

有些人經常在職場中抱怨，不是埋怨同事，就是吐槽公司和領導。他們從來不會反思自己，從自己身上找原因，總是喜歡為自己找各種理由和借口，沒有責任感和擔當。

他們經常為了一點蠅頭小利斤斤計較，工作能力不怎麼強，但是從來不會讓自己的吃虧，有時甚至為了自己的利益，而損害公司的利益。

所以，職場中，這樣太計較個人得失的人，是不會得到領導器重的。

在職場中，一個人的能力固然重要，但是比能力更重要的是一個人的品行，如果你的品行不端正，不管你工作的能力多麼出眾，都不會被領導重用。

所以，在職場中想要被領導器重，一定不能的成為領導討厭的人，做事情一定要注意分寸。
