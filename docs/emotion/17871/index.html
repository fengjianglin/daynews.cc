為什麼我們總是容易對自己的親人發火？

2019-12-17 920條評論 2,263次閱讀 534人點贊

中國自古以來就是人情社會的國家，無論是禮儀，還是道德，都是首屈一指的。在這樣一個人情社會的國度里，情商已經成為了我們每個人的標籤，大家都希望自己成為一個高情商的人。

在生活中，那些被我們標榜著高情商的人，他們無論是在工作中，還是生活中，都可以說是如魚得水，和同事、朋友都能和睦相處，就算有什麼不愉快的，也能好好說話。

但是，在這些高情商的人中，有一部分人回到家以後就完全變了一個人，對自己至親至愛的人從來就沒有好好說過話，反而經常對他們發火，甚至大發雷霆。稍微有點什麼不順心的事，他們就能把家裡弄得雞飛狗跳，搞得全家都不高興。

不知道大家是否有過同樣的感受，我們也經常很容易的就會對自己的親人發火了，這到底為什麼呢？

1.自己對親人期望值過高

你認為親人作為最親密的人，不是外人，就應該理解你，對你好，導致你對親人期望值過高了。殊不知，親人也不可能在任何時間、任何事情上都能站在你的角度上理解你，所以就容易產生很多的誤會。

在《紅樓夢》中，有一回賈寶玉在花園裡題詩，得到了他爸爸賈政的誇獎，把沒有得到認可過的賈寶玉給高興壞了。於是，他就爽快的答應並把身上的手鐲、玉佩等飾品都賞賜給了僕人。當林黛玉聽到了這事，再看到賈寶玉身上的飾品已經全都沒了，以為賈寶玉把她送給他作為定情信物的荷包也賞賜給別人了。黛玉二話不說，就直接質問寶玉，「我給你的荷包你也給她們了？」由於寶玉把荷包藏在了衣服裡面，當著別人的面不好意思說出來，就猶豫了一下，沒想到林黛玉直接發火了，「你明兒再想要我的東西可不能了！」說完就氣呼呼的跑回房間，把正在給寶玉繡的香包給剪了。直到寶玉把荷包從衣服里拿出來，並哄了好一會兒她才肯原諒寶玉。

為什麼林黛玉就不能等賈寶玉先解釋完再發火了呢？她難道不怕誤會了寶玉嗎？原來是寶玉喜歡她，她也喜歡寶玉，她已經把寶玉當成自己最親密的愛人了，期望寶玉能把她送的東西當寶一樣看待，所以當她看到寶玉身上什麼東西也沒有的時候，就直接發火了。

2.親人對自己的期望值太高了

有時候家裡的親人對你的期望值很高，你沒辦法達到他們的期望值，又不想讓他們失望、讓他們不開心，這個時候你很容易把自己逼到崩潰了，一崩潰就會發作。

我以前有一個湖南的同事，來我們廣西工作好幾年了，每年春節都不回家過年。我雖然很好奇，但是涉及到別人的隱私，就一直沒開口問他。有一年準備春節的時候，他在跟他媽通電話時突然就大發雷霆，然後直接把手機給摔了。當時我就在旁邊，把我給嚇一跳。他平時跟我們相處的時候，都是文質彬彬的，脾氣也是非常好，那一次真的完全出乎我的意料。直到後來去喝酒的時候，他才告訴我，他家裡人逢年過節就給他打電話催婚，雖然他也很想滿足他們，早點結婚生子，但是他又不想隨隨便便找個人就結婚了。為了這事，他經常跟她媽吵架，所以才一直不願意回家過年的。

我這個同事就是典型的家裡人對他期望值過高，給他造成巨大的壓力，導致他焦慮、憤怒。

3.跟親人發火代價最小、成本最低

有時候，你跟外人發生矛盾，你估計都不敢隨便發火，因為別人有可能會反擊你或者在背後整你。但是，你對自己的親人怎麼發火，他也不會把你怎麼樣，所以你才會經常肆無忌憚對自己的親人發泄自己的不良情緒。

不知道你們是否也曾跟我一樣，有時候心情不好，對家人很是不耐煩，莫名其妙的就發火了，事後也自己覺得太不應該了。但是，下次的情況還是一樣，根本控制不住自己。

歸根結底，就是我們不擔心親人會把我們怎麼樣，沒有壓力才會有恃無恐。

不管是什麼原因，隨便對自己的親人都是不對的。所以，我們一定要學會換位思考，改掉這個壞毛病，畢竟他們都是我們最親愛的家人。
