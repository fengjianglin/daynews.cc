“奶完孩子，我把飯桌掀了”產後冷暴力的坎，若邁不過去就掀了吧

2019-12-01 0條評論 24次閱讀 0人點贊

摘要

文/貝貝豆（原創文章，歡迎個人轉載分享）面對“月子之仇 不共戴天”這個問題的時候，總有人會說寶媽矯情

文/貝貝豆（原創文章，歡迎個人轉載分享）

面對「月子之仇 不共戴天」這個問題的時候，總有人會說寶媽矯情、小肚雞腸，他們認為婆婆和老公都是自己最親的人，應該大度一些，不能斤斤計較，可是換位思考，面對寶媽你做到了「將心比心」了嗎？

「奶完孩子，我把飯桌掀了」產後冷暴力的坎，若邁不過去就掀了吧

說起冷暴力，我真的覺得異常恐怖，它不像吵架那樣轟轟烈烈，卻無情地把人傷害的遍體鱗傷，而我的閨蜜就遭受著冷暴力的摧殘，讓一個原來擁有大好時光的姑娘，變成了終日以淚洗面的林黛玉。

閨蜜在19年初生下了她的女兒，小女孩大眼睛高鼻樑十分的可愛，可是再漂亮有什麼用？婆家只想要男孩，給她的女兒貼上了「丫頭片子都是賠錢貨」標籤。

當時生完孩子出院後，整個月子里婆婆都沒有一個好臉色，整天說閨蜜沒有本事，就會生個閨女，一次次地將她的自尊心放在地上踐踏，而老公呢？典型的媽寶男，什麼都是他媽說的對，在母親的責罵中一唱一和的數落著媳婦的不是。

直到一天，我收到了她的微信：「奶完孩子，我把飯桌掀了，這次邁不過去的坎，我掀了過去」。

之後我了解到，原來那天中午吃飯的時候，孩子正在調皮的哭鬧，於是她放下碗筷，就去給孩子吃奶。結果等她安頓好孩子，出來之後看到餐桌上的盤子早已空空如也，心中積累委屈再也控制不住，於是她掀翻了飯桌。

此時此刻只有我知道她的內心到底有多苦，產後婆婆看她不順眼，丈夫也不待見，為了孩子忍氣吞聲的生活著。桌子上的空盤子也見證了她不被人珍視的心寒，我想是時候發泄下心中的不滿了，要不再這樣下去她會瘋的。

產後寶媽遭受著各式各樣的冷暴力，將女人

1、重男輕女思想

縱使在新社會已經男女平等，但是重男輕女的思想仍然不能抹去，在這樣的家庭中，也讓產婦「啞巴吃黃連，有苦說不出」，生男生女並不是產婦能夠左右的，完全是男性的精子所決定的。但是只要是生的不是男孩，責任就必須要女人來承擔，這未免有點太過分了。

2、對娘家人蔑視

產後坐月子一般娘家人都會來伺候自己的閨女，可總會遭受到女婿和親家蔑視，而娘家人為了女兒卻忍氣吞聲。女兒看在眼裡卻急在心裡，心裡不知道有多難過，奈何自己當初「瞎了眼」。而父母如若不是為了自己的女兒，何苦來這裡受氣？

3、風涼話重傷人心

不管是產前還是產後，總有一些風涼話時不時的刺激一下。「別人都是順產，你就非的剖腹產，誰還沒有生過個孩子，浪費錢」、「我坐月子3天就能幹活，你怎麼就不行呢？」、「月子里自己帶孩子，我可沒有義務幫你」等等，聽著這些刺耳的話，你覺得女人的心裡能好受嗎？

4、家人的漠不關心

在月子里是產婦最需要幫助的時候，面對整日嚎嚎大哭的孩子，初為人母總是會手忙腳亂。可是家裡人呢？「喪偶式」的老公和不願管事的婆婆，總是當作看不見，能躲多遠躲多遠，只有自己默默承受這所有的痛苦，這種傷害遠比拿鞭子抽她更可怕。

5、丈夫的出軌行為

因為需要照顧孩子，夫妻之間難免會分開睡覺，而面對自己的生理需求，丈夫就會有肉體或者精神上的出軌行為，這難免就會對產婦造成中傷。男人和自己的知己談心、聊天、吃飯、唱歌，而把老婆和孩子扔在家裡不聞不問，哪個女人不會寒心呢？

突然間想到2年前，一位孕婦因為陣痛難忍，3次跪求家人要求剖腹產手術，卻遭到拒絕而跳樓的新聞。面對這樣的冷暴力，誰又能理解產婦的尊嚴和感受呢？

產婦在生完孩子後，由於生理和心理的因素，都會產後抑鬱，最初的表現為緊張、恐懼、內疚、疑惑，嚴重的會讓產婦絕望、離家出走甚至走上自殺的極端行為。

當女性面對這樣的冷暴力，千萬不要憋著，要學會這樣做

【該哭哭 該鬧鬧，千萬別憋著】

有些時候憋著會讓怨念越滾越大，每當委屈的時候，其實哭一場也是一種發泄的方式，把壓抑的情緒發泄出來，心理就會好受一點。讓家人也明白原來善良的你也是有「暴脾氣」的，以後就不敢輕易地欺負你了。

【學會和家人溝通】

產婦心中有很多的不愉快，面對這樣的問題，我們要勇敢的說出來，讓家人知道你心中的想法，如果不知道如何開口，那可以利用一封信、一段話來表達你想要說的，讓他們知道自己的難處，遇到問題及時解決。

【放手也是一種選擇】

對於人類來說，最大的恐懼是被拋棄，最糟糕的狀態是產生絕望感，而冷暴力恰恰地將這兩種狀態完美的施壓給一個人。如果在這個家裡真的沒有存在的價值，其實離開何不為一種解脫。

冷暴力就像一把殺人的凌馳刀，當你用這種行為去傷害她人的時候，何曾想過你也會傷到自己，有句話叫「善有善報 惡有惡報 不是不報 時候未到」，我始終相信善良的人理應被溫柔以待。

今日話題：你遭受過「冷暴力」嗎？面對冷暴力你又是如何做的呢？
