安排工作的這三個細節，暴露了領導是提拔你還是排擠你——

我們在職場上打拚，一定要想明白一個最簡單，最基本又最明了的道理，領導如果不培養你如果不給你很多政策的傾斜，如果不照顧你，光靠你自己的本領，靠你自己的實力，可能你這一輩子都不可能混得太好，所以從領導安排的工作裡面要讀懂領導的基本態度，如果領導給你做下面這幾種工作的安排，這裡面蘊藏著領導的各種各樣的套路和心機，大家一定要理解背後的邏輯和領導真正的意圖到底是對你想幹什麼。

【1】是主還是次——只安排你次要的重複性的簡單工作，明顯是在排擠你。

領導安排工作的這樣一個基本的態度，說明領導對你是有一些猶豫不決的，如果只給你安排一些重複性的簡單的一些勞動，誰都能做，言外之意誰都可以替代你，實際上領導是對你不放心和不踏實的，記住在單位裡面要做到無可替代，至少在你這個崗位上你做的比其他的人都突出一些，只有這樣才能找到自己確定性的位置，領導才能把重要的任務交給你，所以從對待這種工作安排上的態度上，就能判斷出領導對你是真心實意的培養，還是虛情假意的利用。

【2】是長還是平——不安排你長本事的工作就是明顯的要放棄你的節奏了。

這一條也非常的關鍵，如果領導安排的工作是讓你長本事的，雖然很累，雖然你的工作非常的繁瑣，涉及到的部門也很多，但是對於自己長期的成長是很有好處的，能夠快速提升你的綜合能力，相反的情況下，如果領導給你安排一些簡單的，幾乎不需要什麼技能的，也就是你今年干這個活不費太大的勁，10年之後干這個活依然不費太大的勁，背後的邏輯和意圖很明顯，就是把你放在當下這個簡單的崗位上，沒有打算讓你長期在這個單位裡面做，也沒有打算培養你的意思。明白了領導的這個意圖，你就明白很多事情沒有你想的那麼簡單，當然你想的也不要過於複雜，改善領導之間的關係，改善你的心態，讓領導重新認可你才是關鍵的關鍵。

【3】是私還是公——有沒有私底下的單獨交待，意味著你有沒有好的機會

我們在判斷領導和你之間關係的親密程度是一定要有一個質的區別，就是私底下你們之間有沒有交往，有交往，說明關係還不錯，沒交往說明關係很一般，所以私底下領導對你有沒有交代，經常不經常找你私下談心，決定著領導基本的態度，如果這些事情通通沒有，說明你在領導心目當中是沒有概念的，說不定領導連你的名字都叫不上來，哪裡可能去培養你呢，我們必須理性的看待你和領導之間關係的性質，不可以任由自己的性子發展，更不可以按照自己的判斷得出結果。
