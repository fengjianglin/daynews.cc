《親密關係》：愛，是種能力，但它也需要技巧

天天要聞
2019-12-21

每個人都是有頻率的，不僅有頻率，而且還是可以調頻的。你要幸福，就要允許自己幸福，幸福是自己可以說了算的事。

而親密關係是生活中的一個重要組成部分，它有助於長期的幸福，所以我們要更關注它。

本文是羅蘭·米勒《親密關係》的第二篇書評，上一篇從吸引力和第一印象兩個方面，來分享如何建立親密關係。本文將分享如何正確面對、解決親密關係之間的矛盾，還有維持關係的技巧。

關係出現了矛盾，應該怎麼辦？

我們都知道，通過相互吸引、認識，建立了親密關係，那麼對於親密關係的依賴程度，會受到什麼因素的影響？也就是什麼情況下，我們會決定離開這個人，結束這段親密關係？

相互依賴理論認為，感情就跟買賣一樣，所有人都在尋找最合他心意的商品。我們都在追求以最小的代價，獲取最大的獎賞。

能夠給我們提供最大價值的人，我們最願意跟他交往，能夠給我們提供足夠利益的人，我們就能夠跟他維持親密關係。這裡的獎賞並不只是物質方面的，它包括「我跟你在一起的時候，我感受到的你給我的接納和支持」，或者「每天能看到你，我就覺得好開心」等等。

人際交往的結果並不是由正負，來評判兩個人之間的依賴程度的，而是由期待收益和替代收益。

期待收益，也就是我們對結果的期待，我們的期望值。

比如，我的朋友王蕾她對現在的感情滿意度很低，是因為她過去經歷了一段困難重重的親密關係，所以對於感情的期望值就變得很低。有一些人，如果他之前交往過一個獎賞價值很高的伴侶，對方對他非常好，讓他感覺特別滿意，那麼他對於接下來的這段感情，就會有較高的期望值。

所以期望值其實是受到很多因素的影響，結果和期望值一對比，就能夠得出你對這段感情的滿意度。也就是現實當中，你這段感情的結果，高於你的期望值，那麼你的滿意度就會很高。

那是不是滿意度高，對這段感情的依賴程度就高，就絕對不會離開這段感情呢？不是。

滿意度並不是唯一，甚至不是決定性的因素，它對於親密關係的持續影響有限。

相互依賴理論另外的一個假設叫替代收益，也就是說如果我離開現在的伴侶，那麼我覺得我接下來會過得怎麼樣。

你對於感情的依賴程度，取決於如果你離開現在這段感情，你有希望得到比目前關係更好的收益，比如你能夠遇到一個更好的對象。

如果你能夠得到比目前更好的收益，即便你對現狀還滿意，你也很有可能離開現在的伴侶，去追求更大的收益。

即便你對目前的親密關係不是很滿意，但是沒有更好的替代選擇出現之前，你也不可能脫離現在的關係。

這個觀點就可以解釋，為什麼有些人即便在親密關係當中很痛苦，但是也不願意離開？因為儘管現實非常痛苦，但他們認為一旦離開，情況會變得更糟糕。

比如我如果對現在的伴侶已經投入很多，花了很多心思和錢，我投入的代價很大。這樣就會影響我再找一個新歡，因為收益太小。這個代價還包括了財產和心理方面的東西，而且我離開現在這段關係的話，周圍的朋友都會看不起我等等。

所以對親密關係的滿意程度，並不是決定我們要不要保持這段關係的主要因素。

如何正確結束一段關係？

現代人對於婚姻，有著更高的期待，所以離婚率不斷在攀升。對於親密關係的解體和消亡，該怎麼去處理呢？

有研究表明，如果兩個人在結婚之前，就發現關係當中有脆弱的地方或有矛盾，那麼離婚的概率就高。還有另外一種情況，就是如果兩個人剛開始滿意度特別高，對婚姻充滿了幻想，那麼離婚率也會高。

所以離婚之前，有很多的信號，但是離婚本身是個非常複雜的事情。

對於結束一段關係這件事，它其實有五個階段：

第一，個人階段。兩口子當中有一個人感覺到不滿意了，經常表現得很沮喪。

第二，雙人階段。不滿意的那個人經常表現他的不滿意，兩個人之間就會出現很長時間的協商對峙，甚至他們之間會互相嘗試再去適應對方。

第三，社交階段。伴侶開始公開與他人分享他們的苦惱，抱怨遭遇，並尋求支持和理解。

第四，善後階段。開始通過努力工作克服自己的失敗，丟棄過去的伴侶關係，重新修正整理自己的記憶。對於離婚合理化，重新評價兩個人之間的關係。

第五，復興階段。已經離婚的伴侶作為一個單身，重新再進入到社交生活中。

一項研究發現，如果兩個人的婚姻持續了12年，那麼離婚需要消耗掉5年的時間，也就是這五年時間，天天考慮分手，最後才能夠離掉。

成熟的人不僅能夠駕馭一段高質量的親密關係，他們也懂得選擇在什麼時候，結束一段親密關係。

愛情和婚姻的保鮮秘訣

不管是愛情還是婚姻，都是需要實施保養與維護的。

心理學家認為，愛情由三個構成成分組合而成：親密、激情、承諾。

愛情的火熱來自於激情，而溫情來自於親密，相比之下，承諾是一個跟情感或者性情沒有關係的決策。這三個成分在愛情關係當中都很重要，每個成分都可以讓人感覺到很滿意很滿足。

在運營愛情上，可以享受激情，但不能把它作為維持愛情關係的基礎。建立和培養跟愛人之間的友誼，彼此互相信任和支持。此外，保持新鮮感很重要，一定要把握每一個跟伴侶共同進行探索的機會，比如一起玩新的遊戲、一起尋找一些好的玩法，前提是一定要一起進行，這樣才會提高對感情的滿意度。

那麼，對於婚姻該如何維護呢？

1.保持忠誠。

在認知上，忠誠的伴侶，他們在考慮問題的時候，不會從個人出發，會從兩個人、整個家庭出發。而且對自己的伴侶評價會很積極，特別欣賞自己的伴侶，自動無視那些替代伴侶。

行為上，他們的自控力特彆強，而且能夠關注和回應對方的需求和願望，能夠幫助伴侶成為他們想要成為的人。除此之外，他們還能夠忍受伴侶的一些不嚴重的苛刻對待，但不會還擊，也就是說，忠誠伴侶他們會做出比較順從的反應，控制自己的衝動，避免激起對方的挑釁而引起不必要的矛盾。

2.保持滿足。

想要提高對親密關係的滿足感，你要學會欣賞伴侶。比如，你可以每周都找出三個你最欣賞TA的地方，不管大事、小事，講出來，一定要表達對他的欣賞。

其次，要表達感激。對你們之間的關係，進行感激，感激會促進親密關係。比如，我和范先生，我們都不會認為自己得到的這一切都是應得的，無論是他對我，還是我對他好，互相之間都會經常去表達這段關係，我們會感激對方，因為有這樣的親密關係，所以才會感覺到幸福感。

然後重複上述兩個步驟，欣賞伴侶，表達感激，這樣就能夠促進兩個人之間的親密關係。

3. 如果兩個人的關係出現了問題，很多人分手或離婚的時候，會上網求助，找一些所謂的專家，嘗試解決問題。比如我曾經跟初戀分手的時候，我就去網上找心理諮詢師解決感情問題。當時諮詢師教了我很多挽回對方的方法，實踐後最終還是將這段感情划上句號。

所以，面對這樣的感情問題，作者羅蘭·米勒在書里特別提到一點：你要自己解決的話，一定要警惕那些不專業的意見。

在愛情跟婚姻方面，大眾媒體給出的意見常常存在問題，有很多不專業的機構，他們非專業性的意見可能是完全錯誤的，做出一個自己的評價和斷言，來指導別人。

因此，在自己解決感情問題的時候，一定要注意非專家的意見，要去找真正的專家諮詢。

最後，來分享幸福婚姻的秘訣。

研究發現，滿意度高的夫妻，他們都非常珍視婚姻，他們認為婚姻是長期的承諾和忠誠。

在親密關係里，細心周到地呵護和保養伴侶之間的關係是非常必要的。它就跟我們的車子一樣，你不能只開不保養，時間長了，車子就很容易出問題。婚姻也同理，一定要呵護與保養。

其次，幽默可以促進感情的發展，讓婚姻保持足夠的新鮮感。所有長久的婚姻關係都需要新鮮感去支撐，給生活添加潤滑劑。

我們還會發現，身邊有很多伴侶都非常的相似，那是因為他們在大部分的事情上都可以達成一致性的意見。此外，打心裡真正喜歡自己的配偶，常與配偶共度美好的時光，可以大大地提高幸福滿意度。

心理學家羅傑斯說：愛是深深的理解與接納。兩個人的關係越來越深，就不容易審美疲勞。

人的內心既有對愛的渴望，也有對愛的絕望。當愛的渴望級別很高時，就很容易建立親密關係，但是如果愛的絕望很深，不再有渴望，就很容易成為橡皮人。

希望每個人都能夠建立穩定的親密關係，一定不要小看親密關係對你的作用，如果能夠擁有一個健康穩定的親密關係，我們這一輩子都會活得心滿意足。
