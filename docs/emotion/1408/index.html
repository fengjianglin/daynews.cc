但是關鍵點不是這句話，而是：每個人的核心利益到底是什麼？並且，每個人的核心利益會隨著男女關係的改變(或者進展)而作出怎樣的改變？

我舉幾個非常小的例子吧。（以下說的男朋友不是同一個人，有可能是前任們，不重要就不加區分了）

我從小就學習鋼琴，學過樂器的人都知道，這個是需要長期堅持的。我從小到大，基本上保持每天練習一個小時。剛談戀愛的時候，男朋友對我這個技能非常的欣賞。（這就是其中的一個利益訴求，男人喜歡會樂器的女生，尤其在剛接觸階段）後來，我們在一起很長一段時間後，感情穩定了，我依舊保持下班後去琴行練一個半小時琴，回到家裡，基本上就是快7點了，所以我從來都沒辦法做晚飯，都是他做。這個時候矛盾就來了，我以前彈鋼琴是佔用自己的時間，與他無關，現在卻需要他負擔這個時間。再者，他擁有一個會鋼琴的女朋友對他已經沒什麼可激動的了（已經習慣了，並且炫耀完畢），此時，對他而言，回家有頓熱飯吃更重要。

第二個例子，我有一個朋友，長得很美，就是皮膚狀態一般，需要每個月去美容院保養，才能保持好相貌。她的每一個男朋友一開始都很寵愛她，後來時間長了，就對她去美容院表示一些不滿，因為美容院開銷不小。這也是核心利益變化的原因，最開始核心利益是女朋友的美貌，且這個美貌不需要他負擔。但是後來呢？核心利益變成金錢。當然了，我的朋友自己也有錢，不需要男朋友支付美容費。但是自己的錢隨便怎麼花，可是組建家庭以後呢？那就是兩個人的錢了，沒有哪個男人會喜歡自己的老婆每個月花幾千塊在美容院里。此時，男人們的核心利益慢慢由美貌變成金錢，而我朋友就觸碰到了她老公金錢的利益。

可能看到這裡，有些女性朋友會覺得男的怎麼這麼討厭之類。但是我們換位思考一下，就知道這個簡直是很常態並且很難避免的。

那家庭里有沒有不被嫌棄的技能？有！做飯！生孩子！掃地洗衣服！給他睡！可是你開心嗎？

因為金錢是雙方的共同利益，你多賺點他開心，他多賺點你開心。並且你們自己也開心。

1、我曾經說，矛盾永遠都是矛盾，沒有解決方法，這句話我收回下半句。

我的解決辦法就是：當你做了有利於自己，而不利於對方的事情的時候，請你心甘情願的道歉，以及作出其他的補償。

所以，當你和對象因為種種問題吵架吵得不可開交的時候，你仔細想想，到底這件事情，對誰有利益，傷害誰的利益。然後，得到利益的那方絕對不可以再得理不饒人、看不清楚事情的本質，導致感情的破裂。

所以我的整篇文章，都不是為了拿感情的陰暗面去嚇你們，而是說，每份感情，絕對離不開每個人的利益，哪怕親情，都是一樣的道理。而我們，理解這一點以後，能更好更寬容的看待生活中的種種爭吵。
