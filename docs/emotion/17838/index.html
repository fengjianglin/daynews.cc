少一些無奈與感慨，多一份從容和淡然，早安

2019-12-18 457條評論 2,790次閱讀 602人點贊

欣賞這樣的人，智慧卻深邃，飽滿卻低調。他的人生，從年少的奔涌沸騰，到後來認清現實後的沉默，逐漸從悲轉喜，樂觀豁達，充滿力量。而這裡所說的豁達，指的是，走過萬水千山，閱盡世事百態，並未消極沉淪，或刻意避世，相反，是釋然與坦蕩，是歷經滄桑後，依然清澈純粹。願我們，都能在渾濁與黯淡里，尋到屬於自己的路。共勉。早安！

每天提醒自己「我用不著為這一點小事而煩惱」，你會發現，你心裡有一種不可思議的力量，內心會豁然強大起來。早安！

每一天都會告訴自己要 好好控制情緒，不抱怨，謹言慎行，這不是將自己變的懦弱和沒有性格，而是在慢慢的 提升自己。凡事不以惡意揣度別人，不以私利給他人添堵，不妄自菲薄，也不詆毀他人，這是對自己最基本的要求。早安！

生活總是這樣磕磕絆絆，有喜有悲，所以你要保持一顆平常心，戒驕戒躁。若成功，則不忘努力，懂得珍惜；若失敗，則當是磨礪，增加經驗。除了生死，都是小事，別為難自己。早安！

人生頓悟貴在有一顆平淡的心，人生路上，可能春風得意，也可能坎坷不平，無論如何，我們都要一直走下去。榮耀也罷，屈辱也罷，都要以平和的心態去面對，少一些無奈與感慨，多一份從容和淡然。早安！

我之所以這麼努力，是不想在年華老去之後鄙視自己。早安！更多的精彩情感，哲理，美文；敬請關注微信公眾號：心成一片
