個性哲理精美短句：活的不盡純粹，至少要過得乾淨利落

天天要聞
7 天 ago

一、保持一份自信，保住一份尊嚴，寧可高傲到發霉，也不要死纏到發瘋。

二、懶於解釋，也無謂能否做個好人，活得不盡純粹，至少要過得乾淨利落。

三、「其實他人不壞。」我們因為這句話，容忍了生活中多少不該容忍的人和事。

四、人最大的修養，是知人不評人。不懂別人就少說話，議論最掉價。

五、放過自己的唯一方式只有以下九個字：該吃吃，該睡睡，愛誰誰。

六、沒有用的東西，再便宜也不要買；不喜歡的人，在寂寞也不要去依賴。

七、人生就像一場舞會，教會你最初舞步的人卻未必能陪你走到散場。

八、海是倒過來的天，換個角度看世界。

九、時間就像一張網，你撒在哪裡，你的收穫就在哪裡。

十、時間的三大殺手：拖延、猶豫不決，目標不明確。

十一、別因為一點點不順心，就把今天輸掉。

十二、用時間和心看人，而不是眼睛和那些閑言碎語。

十三、真正的善良不是軟弱更不是退讓，而是從不去主動傷害別人，不會糾纏不休，懂得適可而止。

十四、選擇比努力更重要，智慧比聰明更重要，眼光比當下更重要！

十五、成年人的世界，就是應該果斷又乾脆。不耽誤任何人，不消耗任何人，不浪費任何人，這是一種善良。
