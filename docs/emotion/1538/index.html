結婚十年仍像熱戀：巧用職場理論，你也能收穫更美好的生活

今年的中秋假期，我和老公一起去了貴州旅行。在攀爬一座海拔1千多米高山的過程中，老公一路上對我呵護有加：往上爬一個近乎90度的階梯時，怕我沒有抓牢滾下去，就一直在我身後保護我；靠近懸崖邊時生怕我一不留神會掉到懸崖下，一直緊張的拉著我走最靠裡邊的道，還不讓我到懸崖邊去拍照；面對陡峭的下山路，主動幫我背包，路面太滑，用腳給我做人肉墊子防止我摔倒。他的種種舉動，讓結婚十年的我心情愉悅，像是回到了熱戀期，被人呵護的感覺，真好。

絕望階梯，四手爬上去的

你可能想問，為什麼我們能夠保持這種感情濃度不減呢？那是因為我用了兩個職場上常用的技巧（用戶訪談和高低峰值）來規劃這次路線，讓我們在整個旅行中心情愉快，感情升溫。作為一個朝九晚五的普通人，我們每天的主要戰場就是職場和生活。你是否有曾想過，用職場上的技巧來解決生活中的難題呢？今天我就來帶你看看，如何巧用職場思維收穫更美好的生活。

1、巧用用戶訪談，挖掘客戶&對象的真實需求。

《洞察人心》這本書中有提出一個概念叫」用戶訪談「，通過這個方法來了解你的研究對象的真實想法和需求，在現代商業活動中則常被用來精準定位客戶。

客戶的秘密

腦白金的創始人史玉柱在早期確定產品定位的時候，常常跑到公園與老爺爺老婆婆們聊天。他問到如果有一樣很好的保健食品，價格也很便宜，你們會不會買？老人們說不買，那再便宜一些呢，老人們還是說不買。追問到最後，他發現這些老人家不是買不起，而是希望把錢留給孫子孫女們，他們對自己是很摳門的。但是如果有人買給他們，尤其是子女買的，他們就會很樂意接受，因為一方面可以感受到子女的孝心，另外一方面呢，也可以向其他老人炫耀。史玉柱對老人們的這個用戶洞察就像是一顆鑽石，照亮了營銷團隊的前路。腦白金團隊花了大量的時間鑽研，如何把送禮這個事情說得好說得明白，才最終出現了那句經典的廣告詞。

今年過節不收禮？

我們的旅行能這麼開心，主要是在旅行前期就對對方的需求做了用戶訪談。我和老公都熟知對方的性格，我是急性子，喜歡有計劃的生活，事情沒合我心意就會發脾氣；老公則是慢性子，講究隨遇而安，開心就好，性格差異導致我們在生活中會有摩擦。於是我們提前說定：有分歧時不抱怨不吵架，心平氣和的溝通，有了這個前提在，我們果真沒有爭吵，情緒平靜心情也沒受到影響還是開開心心的。

實際生活中，你可以用「用戶訪談」這個方法來挖掘你的另一半的真實需求，在制定計劃、日常生活時，讓你們產生更多的共鳴，提升生活感受。

2、巧用峰值尾值理論，讓生活情趣錯落有致

心理學中有一個很簡單的理論，Peak Value and End Value，我們把它稱為峰值尾值理論。它說的是，一個經歷在事後給你帶來的愉悅感並不取決於整個事情持續性的平均體驗，而是它給你帶來的體驗的高峰和收尾的平均值。

最低谷的時候是不是很難受？

前幾年上映的《諜中諜4》就是峰值尾值理論的反例。這部電影里不乏讓人心裡一緊的小場面，但依然只能算是花瓶電影。他們都有兩大致命傷：一是小高潮迭起，但是無論是打鬥戲追車戲激情戲，都沒有非常突出的大高潮（峰值體驗不足）；二是它們都沒有戲劇性的收尾，在你期待還有點什麼的時候，它就平淡的結束了（尾值體驗不足）。它們帶來的體驗還不如一部只有四分鐘的短片《蒼蠅一分鐘的生命》（動感有趣的高潮和讓人意外感動的結尾）。這也是為什麼很多電影的短片版本都比一百多分鐘的長電影版本好看，為什麼很多作家寫的短篇小說比長篇小說更精彩的原因。

這個場景，你還記得嗎？

我們的整個旅程，在出發之前就已經設計好峰值體驗了。那時老公就對我說黃果樹瀑布是這個旅途中最好看的一個景點，而其他的就不要期待太多了。後來的經歷確實也證明了這一點，我們在黃果樹度過了非常高興的一天，還吃到了很美味的菜肴。而後的尾值體驗，完全是老公給我的一個驚喜，以至於我回來後還一直在回味這次旅行，感覺意猶未盡。

好山好水好心情

這個理論可以在生活中的很多場景應用上。比如，另一半過生日，你為他/她設計一個有峰值尾值體驗的約會/party，給他/她留下深刻的印象，提升你們的感情濃度。或者說，你要負責組織一場活動，就可以在活動過程中應用這個理論，讓參與者在活動過後還能記住活動中的高潮和結尾部分，在說起你的組織能力時對你讚不絕口，你也因此信心倍增。

我們每天職場生活連軸轉，很多人不希望把工作和生活混到一起。其實，我們把在職場中常用的理論應用到生活中，常常會收穫意想不到的驚喜，這兩者，本身就是融合在整個社會中的呀。所以，我們要學會巧用職場理論，來為我們的生活增添光彩。
