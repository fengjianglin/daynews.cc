如何跟女領導打交道？要知道除了級別大小，領導也有性別之分！

2019-11-28 0條評論 20次閱讀 0人點贊

摘要

一提起領導，大家最喜歡的，往往是關心領導的級別大小，比如他是正處級，你是正科級等等，但是相比於級別來說，很多人往往忽略了一點，那就是領導其實還有性別之分，比如，很多人對於如何同不同性別的領導交往，就覺得有些琢磨不妥的感覺。

一提起領導，大家最喜歡的，往往是關心領導的級別大小，比如他是正處級，你是正科級等等，但是相比於級別來說，很多人往往忽略了一點，那就是領導其實還有性別之分，比如，很多人對於如何同不同性別的領導交往，就覺得有些琢磨不妥的感覺。

那麼最常見的，就拿人與人相處和打交道來說，你覺得跟男領導更好打交道呢還是跟女領導更好相處呢？其實相比於女領導來說，我覺得大多數情況下，男領導可能更好相處一些，也更好溝通一些，這可能也是為什麼大多數領導都是男性的原因之一吧。

說實話，現在社會男女平等的觀念基本可以說是深入人心了，比如生活當中，大家尊重女性甚至很多時候女性優先基本已經達成了共識，但是，為什麼女性領導的比較少？

這個問題我覺得很有意思，可能大家最熟悉的原因，比如女性更多關注重心在家庭經營方面，以至於在工作中很難干出讓大家心服口服的業績，另外比如很多女性因為生理因素，至少產假哺乳假期間，可能就會被同批次的男性趕超。

但是，除了這些大家熟知的原因以為，為什麼女性當領導的比較少，或者即便當了領導的女性，也總是被大家覺得不好相處呢？

關於這個問題，我覺得也應該一分為二看待。

比如一方面，我覺得男性和女性本身就屬於不同的物種，你要讓兩者之間完全相互認同相互理解甚至思維同步，我覺得這本身就不是一件容易的事情。另一方面，很多女性在職場中給大家的形象，往往是話多嘴雜八卦等等，而且即便當了領導，也讓很多人覺得心思過於細膩不夠大方等等，我想這些都是導致大家覺得跟女性領導不好相處的原因吧。

當然了，話說回來，畢竟沒有哪個規定了領導一定就必須得是男性才能當，而且有些女性領導工作中可能幹的未必就比男性領導差，所以，多一些包容和理解，一切皆有可能的吧。以上就是我對於這個問題的意見，歡迎有不同意見的朋友繼續交流。
