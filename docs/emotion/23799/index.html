“人肉椅子”面面觀：女人是寶是草，懷孕一次就知道了

2019-12-19 59條評論 2,160次閱讀 162人點贊

嫁給愛情，是每個女子的畢生心愿，從懵懂的女孩到白髮蒼蒼的老太婆，誰都想做那個男人掌心裡的寶。

老婆產檢，醫院長廊，老公甘做老婆的「人肉椅子」。

那天上網，無意間看到一個視頻。

視頻里，一對夫妻等候產檢，大概等的久了，孕婦疲憊，於是男子就地而坐，大肚子老婆則坐在他的背上。

很短的一個視頻，暖哭了無數網友。

有網友譴責醫院設施不齊全，有網友痛罵坐著的人沒道德不為孕婦讓座，有網友感慨老公體貼老婆好

福氣嫁給了愛情，也用網友調侃：我是那不疼老婆的人嗎？我是缺個老婆，懷孕的老婆。

想起了很多年前的一首歌，《最浪漫的事 》：

我能想到最浪漫的事

就是和你一起慢慢變老

一路上收藏點點滴滴的歡笑

留到以後 坐著搖椅 慢慢聊

我能想到最浪漫的事

就是和你一起慢慢變老

直到我們老的哪兒也去不了

你還依然 把我當成 手心裡的寶

一首歌唱出了小女子的心聲，嫁給那個恰恰也愛著你的男人，一輩子做他手心裡的寶。

在老公眼裡你是寶是草，懷孕一次就知道了。

老婆之於老公，有四種可能：一直是寶，由寶到草，由草到寶，一直是草。

通常而言，第一次懷孕的准媽媽們，都還是被老公當作公主一樣寵著的吧。

就我個人而言，第一次懷孕的時候，先生那時恰恰工作不是太忙，只要有空，飯是他做，家務是他做。一起出門，只要他跟著，決不容許我手裡提有重物。想吃什麼，想要什麼，他只要能做到的，都會儘力做到。

講真，一直到現在我都記得他叼著煙歪著腦袋幫我擦皮鞋的樣子。感覺一個男人最美的時候就是為工作為家人忙碌的樣子，有同感的嗎？

不過，等到追二胎的時候，我已由寶淪為草了。

雖然先生在家的時候還是會幫忙做些事，但適逢創業正關鍵的時候，他在家的時間屈指可數，所以大多數的時候，我只能依靠自己，挺著大肚子照顧自己與大寶。

很羨慕那些二胎依舊有老公陪在身邊被寵成皇后的姐妹們，羨慕嫉妒恨啊。

香港影帝周潤發與夫人結婚多年來，一直未有孩子，而深為粉絲們遺憾。

但據報道，他們曾經有過一個孩子，不過後來流產了，於是，周潤發決定再不讓夫人受苦受痛，遂一生無子，卻把夫人寵成孩子，惹眾網友稱讚稱羨不已。

這，愛老婆的典範啊！

父母對孩子最好的愛就是相親相愛。

社會的發展推動了人類對於生活品質的要求進一步提高。

相對於上個世紀的生下來活下去，我們這一代現在更注重生活質量，包括情感的需求。

如果說愛情是一見鍾情的激情與衝動，那麼婚姻靠的是實實在的經營與付出，容不得任何偷懶與投機。

醫院裡陪老婆做產檢且願意把背讓給老婆坐的男人，我們不知道他的個人狀況，但是可以肯定，他是一個聰明的男人，聰明的男人都是把老婆放在首位的男人，因為他知道，是老婆在給他生兒育女，給他一個溫暖的家。

曾經有一篇貼子，《我年薪30萬，撐起家的卻是老婆》，看哭了很多媽媽們。

為什麼？

就是因為現實中理解老婆的男人們太少了，他們把老婆對於家庭的付出視為理所當然，而自己高高在上做著甩手掌柜。

在很多女性朋友那裡，我總是聽道一句話：老公對我不好，但確實挺愛孩子的，就這麼湊合著過吧。

我很納悶：一個不愛老婆的男人，真的愛孩子嗎？

這種自相矛盾要怎麼跟孩子解釋？

孩子需要的是什麼？是完整的家沒錯，但更是父母相愛的家。

一個男人，連自己孩子的母親都能肆無忌憚地傷害，難道不是在傷害孩子嗎？

所以，真的愛孩子，如果你是母親就試著去愛孩子的父親吧；如果你是父親就試著去愛孩子的母親吧。

一個和諧家庭的存續絕對靠的不是激情，而是細水長流的溫情。

關愛孕期的老婆，就是關愛自己的未來。

懷孕生子，對於女人而言是一生中不可繞行的經歷，也意味著脫胎換骨的改變。

因為受孕，女人的生理心理都在悄無生息地發生著變化，這種細微的變化只有自己才能感知得到，因而也變得更為敏感。

那麼，作為丈夫，就應該盡量去理解並幫助妻子渡過這種不適期。

近些年來，因為這樣那樣的因素，自殺的媽媽越來越多。究其根源，很多都是在孕期或產後患上了抑鬱症，這種病症不為丈夫重視與理解，反而一味責怪妻子的矯情，無疑就起到推波助瀾的作用，把妻子推到死亡的邊緣。

這個時候，丈夫的一句話，一個眼神，一個行為，都起著決定妻子生死存亡的作用，可以把妻子從絕望的邊緣拉回來，也可以把妻子送上絕路。

在我周圍，把老婆寵成寶的男人，基本上都是家庭和睦，母慈子孝，而那些對老婆冷漠，對外人如親人的男人，晚境一般比較凄涼，被子女所看不起。

我想，這就是蒼天何曾饒過誰，你付出什麼就會得到什麼的最佳解釋吧。

少年夫妻老來伴，經營婚姻當如經營事業，目光放長遠一點。

你說是不是呢？
