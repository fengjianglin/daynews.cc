即興講話容易卡殼？訓練三個連貫法，思維靈活說話更自信：三維樹

2019-11-30 0條評論 21次閱讀 0人點贊

摘要

有朋友問：在即興說話的時候，說話容易卡怎麼解決呢？他講，我是剛入職做財務工作。與別人交談的時候，總是不能快速地回應別人的話，經常是對話結束之後才想出該怎麼回答，並且說話時常有卡殼現象。尤其是遇到跟自己關係不好的人，或者與那些氣場強大的人在一起就感覺到局促不安。

有朋友問：在即興說話的時候，說話容易卡怎麼解決呢？

他講，我是剛入職做財務工作。與別人交談的時候，總是不能快速地回應別人的話，經常是對話結束之後才想出該怎麼回答，並且說話時常有卡殼現象。尤其是遇到跟自己關係不好的人，或者與那些氣場強大的人在一起就感覺到局促不安。最近，我也經常朗讀一些文章或報紙，不知道能不能得到一些改善。

說話為什麼容易卡殼？與不熟悉的人為什麼會有局促不安的感覺呢？

從我的經驗看，這是思維不連貫，思維跟不上說話的節奏。比如，一個人去面試，當面試官問了他一個沒有想到問題時，突然卡殼不知道怎麼去應付，這是因為對有些問題很陌生，一時找不出答案而卡殼。或者遇到不熟悉的人，又覺得對方比自己強，被對方的氣場影響到了。

怎樣訓練思維的連貫性，變得更自信，能夠自如地說話呢？

思維的連貫性，是指你看到某事物或遇到某個問題，立刻能夠聯想到與之相關的事情，再形成自己的觀點。比如，一個老農出門的時候，看到天空烏雲密布，螞蟻搬家等現象，立刻想到可能會下大雨了，之後再返回家去拿把雨傘。這就是思維的連貫性。在職場，要怎樣訓練思維連貫性，使即興說話的反應快呢？

訓練的方法主要有三個：視覺化，鏈接法和目的性。

首先是視覺化：是指在大腦中建立圖像。

將事物在大腦中圖像化，有助於你客觀地去看待事物。

比如，你看到「蘋果」這個詞，習慣于思維是詞語化思考的人，看到的詞語，只是個詞語，與其他詞沒什麼特別的地方，使思維比較模糊。

而思維視覺化的人，見到「蘋果」這兩個字，他的大腦中就「看到」一個水靈靈的大蘋果，它圓圓的，大大的，使你對事物認識的更清晰。

其次是鏈接法：是指將事物與其他事物聯繫起來。

對一個事物或問題的理解，在於你能夠聯想到與之相關的其他事物。

比如，你的大腦中有一個「蘋果」，你就會聯想到蘋果樹，以及果園等，這樣時候，你就對「蘋果」有了更真實的理解。

最後是目的性：是指將事物與你的目標結合起來。

當你訓練了對事物的視覺化和思維的鏈接能力，你就會發現，你在任何場合的說話，遇到任何挫折，你都不會氣餒，而是會儘可能地用你的想法或目標，去影響聽眾的想法或行動。

比如，一個小伙美女表白的時候：

小伙：我很喜歡你！

美女：你長得這麼丑，我怎麼會喜歡你呢？

小伙：這就叫郎才女貌啊。

美女：你很窮，我們在一起不合適的。

小伙：我們剛好可以一起去奮鬥啊……

無論美女怎麼拒絕，小伙都能將對方引到他的目的「兩人的幸福生活」中去，讓美女聽著就開心，當然也增加了表白成功的機會。

總結來講，說話思維的連貫性，能幫助你在社交中建立自信，在人際交往中，變得有話可說，並且說得讓人喜歡聽。
