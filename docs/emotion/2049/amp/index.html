沒事就想想如何發財吧，別總在感情的世界裡傷春悲秋，捏在手心的錢永遠要比那抓不住的心踏實。難過這東西，難是難，終究會過。

人是不能閑的，一閑就會想得太多，一閑就會感情泛濫，所謂矯情屁事多，空虛寂寞冷，都是因為懶散墮落閑。

1、我們經常這樣，為了迎合所有人，把自己過得這麼累，費盡心思讓所有人都開心，卻忘了自己該怎麼笑。小時候、我們詞不達意；長大了、我們言不由衷。

2、什麼都藏不住，什麼事情都忍不住要說出來，後來慢慢的發現，這樣做真的好傻，安全感和安慰都是自己給的，別人只會看熱鬧不嫌事大。

3、生活有一點我不是太喜歡，就是它總讓更懂事的人來承擔糟糕的感受和結果。深情往往都是被辜負，只有薄情才會被反覆思念。

4、不要在乎那些刺耳的聲音，等你真正破繭而出的那天，你會發現，原來那些傷害你的言語，都會變成，慶祝蛻變成功的交響樂。

5、別人怎麼對你，你用相同的態度回應就行了，生而為人，誰都是第一次，沒必要慣著誰。

6、生活有時會逼迫你，不得不交出權力，不得不放走機遇，甚至不得不拋下愛情，你不可能什麼都得到，生活中應該學會放棄，就像清理電腦中的文件一樣。

7、閨密玩好了就是小時代，玩不好就是甄嬛傳，哥們處好了就是水滸傳，處不好就是上海灘。

8、道不同不相為謀，你討厭我，我也未必喜歡你，各走各的人豈不是更瀟洒，何必咄咄逼人費了口舌也討人嫌，你閑得慌但是我沒空陪你。

9、我玩得起，也收得起心，我可以專一到讓你驚訝，也可以花心到讓你害怕，我喝過最烈的酒，也放棄過最愛的人，我可以像個瘋子一樣的玩，也可以像個爺們一樣工作，更可以在家裡做賢妻良母燒一手好菜，一切的一切取決於你是誰，也取決於你如何待我，將心比心，所以請不要站在你的角度看我，我怕你看不懂。

10、不屬於我的雨傘，我寧願淋雨走路；不屬於我的心，我不會挽留，不屬於我的東西，我不會要，不是真心給我的東西，我不稀罕。

11、有一點是你要明白的，感情這東西呢它始終是相互的，只有這樣，才能夠有來日方長。你可以對我忽冷忽熱愛理不理，肆意去揮霍我的熱情，也可以不理會我的失望沮喪難過，但我對你的喜歡就這麼多，你用冷淡磨完了，也就完了。總之，不會再有。

12、很喜歡的一句話：「為什麼有些人明明看起來友善，卻總是獨來獨往？」「待人友善是修養，獨來獨往是性格。」

陪你最多的人還是你自己，所以做自己喜歡的人就好。放過自己的唯一方式只有以下九個字：該吃吃，該睡睡，愛誰誰。
