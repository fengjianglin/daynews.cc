為什麼內心明明反抗，說著說著就變成了順從？

天天要聞
2019-12-19

文章來源於曾奇峰心理工作室（ID：zqfxlgzs) ，作者禾幾汐

我們每天都會用語言表達自己。我們的需要，我們的想法，我們的情緒都是通過溝通的方式傳達出去。可有的時候，我們會心口不一，會言不由衷，這阻礙了我們與世界的交流。
有多少次，我們和顏悅色的答應了別人的要求，背後卻在硬著頭皮，滿心的不舒服；有多少次，我們卑微的在圓場，背後卻在傷心的滴血，或是內心燃燒著憤怒。
我們習慣性地表達「好的」，有的時候甚至忘記了自己的需要。在壓力和分歧之下，我們把習慣性地、自動化地表達出順從他人願望的溝通方式，稱作討好型溝通模式。
這種習慣性的溝通方式從何而來呢？又怎樣堂而皇之地成為我們溝通方式的一部分呢？

討好型溝通：口中的順從，內心的反抗

我接觸過一個具有「討好型溝通模式」的女孩子，她叫丁丁。

有一次，丁丁在驛站寄快遞，本來前一天問好的價錢，第二天想寄出快遞的時候卻被無故加了價。工作人員一臉的不耐煩和含沙射影的腔調，讓丁丁感受到了壓力。雖然丁丁覺得很不公平，可她的感受像微弱的信號，在壓力之下，一閃而過。她感受到對方希望她以更高的價錢寄出快遞，於是她”服從”了，對話竟然進展得十分和諧。

可出了驛站的大門，剛才失聯的感受忽然接通了。她忽然感到十分生氣！回家以後丁丁大發雷霆，嚇壞了她的父母：這孩子為了這麼點小事，怎麼這麼大反應？

她對驛站沒有誠信的行為感到生氣，也恨自己怎麼會把情緒暗暗壓了回去，對不誠信的行為屈服了呢。壓抑的憤怒沒有因為被壓下去而消失，而是在得到釋放的時候，伴隨著更大的殺傷力。

「討好型溝通模式」是一種自動的反應，它常常會將自己「不討好」的情緒或願望壓抑掉，順從地滿足了他人的願望。

丁丁關注到自己的心口不一，希望表達出自己真實的聲音，可她真實的表達和內在的溝通模式起了衝突。

某天，丁丁拼車回家，司機沒有按照約定好的目的地把丁丁送達，而是想偷懶把她放在離目的地還有一段路程的岔路口上。

丁丁有些不安：她的目的地沒有到，怎麼辦？

討好的心理和沒有受到公正對待的情緒開始左右拉扯。最終丁丁放下了討好，與司機據理力爭。她沒有在司機停車的逼迫之下選擇下車，而是言辭激烈的和司機爭辯，要求司機「特意」送達約定地點。

這樣簡單的要求差點演變成一場大鬧劇。對於丁丁來說，沒有順從他人的願望，她很擔憂。她一方面要控制住自己的擔憂情緒，一方面要別人滿足自己的要求，這使得丁丁和司機溝通的時候格外用力，顯得情緒很激動。即使事後，丁丁仍在惴惴不安，因為她沒有滿足別人的願望，是不是自己做錯了？這樣的感受如同烏雲蓋頂，揮之不去。

「不討好」的擔憂從何而來？

「討好型溝通模式」會害怕對別人說「不」，害怕不和諧，害怕惹別人不高興。我們為什麼害怕「不討好」呢？

丁丁兒時的經歷里常常聽到這樣一句話：「如果你不聽話，我就不要你了」。這讓還是小孩子的丁丁，得出這樣一個結論：不順從就等於失去母親的愛，甚至被母親拋棄。

這樣的體驗，讓一個孩子不得不發展出表面順從的方式去應對父母或者養育人，以保護自己能夠生存下去。這種溝通方式會慢慢形成一種習慣，變成一種自動化的反應。成年後，遇到類似的情境時，會自動啟動這種反應模式：習慣性的選擇迴避真實的表達，或按照別人期待的方式去表達。

這次不和諧的體驗讓丁丁難以應對，她打了退堂鼓。她要退回到「討好型溝通模式」，想「捍衛」她賴以生存的方式，覺得那裡更安全更舒服。

如何擺脫「討好型溝通」？

若表達自己的感受就是不討好，這相當於在做自己害怕的事情，我們還要表達嗎？答案是肯定的。

1. 與自己的感受建立連接

我們往往因為自我價值過低而忽略自己的感受，覺得自己不應受到重視，不值得被關注。所以，在與自己的感受建立連接之前，應客觀的看到自我價值，增強自信心。

然後，我們試著捕捉自己與別人溝通時的感受，覺察自己在短暫的互動中的情緒，甚至肢體語言，認清自己真實的心靈體驗，保護它們不受他人願望的干擾，我們就與自己的感受建立了持續的連接。此時，我們只需看到它們的存在，不去認同，也不去排斥。這將為真實的表達打下基礎。

2. 克服真實的表達帶來的擔憂

當自己真實地表達不願意，不高興，不接受時，我們會擔心對方生氣，這讓「討好的」自己心情焦躁，無路可走。我們要怎麼樣看待自己的感受呢？
對方的憤怒會激發我們內在的擔憂，我們擔心：是我們做錯了事情，而不是別人的問題。此時好像出現了一種似曾相識的模擬場景，對方臨時的扮演了我們的父母或者養育人，對我們施加壓力。

我們需要給自己時間，讓自己接受原有的溝通模式。允許自己退回去，繼續啟動「討好型溝通模式」，重新把自己保護起來。告訴自己，我很擔心，我怕自己做錯事情。直到自己有足夠的能量走出來，面對不討好後的不同反應。

我們需要明白的是，擔憂來自於我們內心的小孩，他害怕拒絕和反抗。曾經「討好」的經驗是為了應對父母或養育人，而我們現在已長大成人，我們不再需要以「討好」的方式去求生存了。對方的反應無論是高興的還是生氣的，都是對方的情緒，只要初衷不是故意傷害別人，對方的情緒由他們自己負責。

3. 一致性表達成為可能的溝通方式

慢慢地，原有的反應模式被打破，我們不再依靠自動的反應模式去應對生活。我們不斷的練習，而發展出新的溝通方式。面對壓力我們可以嫻熟的表達自己的真實感受，也可以審時度勢的體諒當下的需要而做出犧牲。

這樣的體驗讓我們獲得能量，這些能量一直存在我們的身上，只是以往用來抗衡自己額外擔憂的情緒，被調動使用了，而現在這份能量回來了，用來自然而然的表達自我。你學會了嗎？
註：本文主要論述的是薩提亞的「求生存溝通模式」,與弗洛伊德提出的「防禦機制」相似。與「討好型人格」、「迎合型人格障礙」不屬同一範疇。
