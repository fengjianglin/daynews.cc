沒有過不去的事情，只有過不去的心情

2019-12-22 401條評論 3,931次閱讀 2274人點贊

很多時候，放不開的是失敗對於自己的打擊；

放不下的是別人對於自己的否定。

沒有過不去的事情，只有過不去的心情。

現在我們還不曾獲得理想當中的那份成功，但我們依然不能放棄自己的理想追求。因為人生還在繼續，一切皆有可能。

人只有放下過去、放鬆心情，才能迎來明天更美的晨曦。

徹底地放空自己吧，留一顆純凈的心，讓它歸於自然恬淡的世界裡去。

儘管生活如此不易，但每個人的人生旅程是不同的，縱然這一路荊棘密布，但最後都會通向成功的終點。昨天已經過去，有夢就要勇敢去實現，只要努力，明天依舊充滿希望。

只要擁有一顆靜謐的心，適時放下，快樂自然會隨時圍繞身邊。

正所謂「魚和熊掌不可兼得也」，世界上許多東西，平心靜氣地捨去，才能真正地得到。

反過來說，有得必有舍，很多人事業有成後，造福桑梓，救濟鄉鄰，回報社會，由此得到了好的名聲，人格與品質也同時達到了提升。

生活中，不僅要學會獲取，還應懂得放棄。

亂花漸欲迷人眼，淺草才能沒馬蹄。不要被為某些時髦的事物所誘惑，網路的精彩，球賽的熱鬧，酣睡的香甜，遊戲的暢快，都應適當地拒絕。

放下浮躁的內心，精神才可自由地馳騁。

人的一生，曲折而漫長，面臨的事很多，不能把一切都背負在身上，適時地放棄，人生之路上才會走得輕鬆、愉快，才能達到終極的目標。

人生之路，何其寬廣！唯有心境渺小之人，才四處覓一些羊腸小道來走。

人生，沒有過不去的事，只有過不去的心情。

放下過去，是一種淡泊的心態。

以一顆平常心，對待名利、地位與物質，採取超然物外的態度，笑對人生的挫折與苦難，掙脫束縛，不為得失所累，才能真正從困境中走出，從容地面對人生。
