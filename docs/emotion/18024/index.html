父母工作忙，那寶寶在家中是由老人帶著到底是好還是不好呢？

2019-12-16 2,478條評論 3,506次閱讀 895人點贊

哈嘍，大家好，歡迎點擊右上角關注，每天閱讀育兒好文哦

隔代教育的話題算是現在很熱門的話題了，也算是現在的家庭教育的現狀吧。會有越來越多的老年人或是主動、自願地想要去幫助兒子來承擔照顧孫輩的責任，或者就是由於子女的需要、特殊情況、再要二胎的問題上就得需要老人出面幫著養育才行。算是都是現實生活中的無可奈何吧。

其實不光是在現實生活中我們感受到來自周圍家庭教育中老人會參與幫著帶子女的孩子的情況比較多，就在官方的調查中也會有所體現，在2017年中國教育學會家庭教育專業委員會「中國城市家庭教養中的祖輩參與問題研究」課題組發布的調查中會顯示，從被受訪的兒童在出生到小學階段里，有接近八成（79.7%）的城市家庭中會存在祖輩（祖輩的父母或者外祖父母）是參與到兒童的教養中來。

看到這樣具體的數字顯示還是說明隔代教育在整個家庭中的參與比重還是蠻重的，也不難看出會有不少的家庭教育問題是出在了隔代教育方式，像有的年輕父母會常常嘮叨的老年人的過舊的養育孩子的方式、觀念是看不慣，沒辦法接受。而自己又礙於時間上、精力上的有限，如果總是在老人帶孩子時表現出不滿意將會造成家庭矛盾。

更為值得做父母的需要注意的是，孩子會被這兩種截然不同的家庭教育方式夾在中間，甚至是摸清了老人和父母的軟肋，在父母批評孩子的時候，孩子就會去找老人當救兵。

這也就引出了即將和大家分享的話題，寶寶在家中是由老人帶著到底是好還是不好呢？影響也許不只是您想像中的那麼一點點，需要引起高度重視，父母的做法會成全三代人。

01、老人帶孩子也會有其優勢

一、讓孩子感受無價的疼愛

您也許會感受到老人對於孫輩的疼愛是遠超過當年自己年輕時帶孩子的那種關愛，也就有了不少年輕父母會感慨的是老人帶孩子就是太過寵溺孩子了。

其實只要老人了解到家庭育兒需要注意的地方，適度把握疼愛孩子的度還是會給到孩子無價的疼愛和指導。也有研究顯示，那些得到過（外）祖父母看好的孩子，在整個嬰兒時期以及以後成長發育得更好。就會有研究昆蟲的姥爺帶著外孫去到室外觀察各式各樣的昆蟲；也有喜愛音樂，與音樂教育相伴一生的姥姥開啟了外孫女的音樂啟蒙。

二、能給到年輕父母適當的幫助

幫著父母減輕家庭的負擔是老人最直接的幫助，這些之外，其實老人的優勢還有生活豐富的閱歷，更有耐心，能夠給到父母很多生活上的建議。

當年輕的父母提出請老人給出一些建議時，作為年長的老人會給到年輕的父母一些有用的幫助，將自己的觀點和方法分享給子女，並且不要強求子女非要贊同、執行。

02、在家庭教育中父母要這樣做

一、提供給老人現代的教育方式

畢竟老人帶孩子的時候是相隔現在蠻久了，很多時候你再讓老人去帶孩子，老人也會有些不適應，更何況是子女的孩子，自己就更不能犯錯了。

鑒於這樣的情況，作為年輕父母的就要給老人提供些現代的教育方式，像讓老人參加孩子的足球賽、鋼琴演奏活動，在孩子參加有意思的創作訓練營時，老人也參與其中感受孩子在新穎的創作活動中是多麼快樂。

二、主動擔起家庭教育中重要責任

老人始終不是孩子的父母，更多是來幫助子女去照顧孩子，因此年輕的父母也要明白自己才是孩子的重要撫養者，不能夠事事依賴著老人來承擔。

想到了這一點，做父母的就該明白，自己在平時有時間的時候就要主動承擔起照顧孩子、陪伴孩子的責任，這樣留給老人一定的空餘時間可以去休閑、放鬆。

只有在家庭教育中，做家長的父母也好還是來幫忙的老人也罷，都能夠互相理解，一心為了家庭好，共同努力將下一代養育的更好，那麼存在於兩代人之間的教育問題都可以迎刃而解。
