痛苦是比較出來的，幸福是珍惜得來的 #人生感悟

2019-12-19 277條評論 2,138次閱讀 532人點贊

1.優美的弧度需要驚艷的角度發現，雨後的彩虹需要藍天白雲來襯托，完整的人生需要坎坷的人生來磨礪。

2.只有不迴避痛苦和迷茫的人，才有資格去談樂觀與堅定。命運不會厚待誰，悲喜也不會單為你準備。

3.成長就‌是一‌個反覆為難自己的過程，如果你始‌終處‌於一‌種很安逸的狀‌態當中，那就說‌明你的生‌活其實在走下坡路。

4.永遠別忘了拉過你一把的人，即使有一天她不再有資格做你的朋友，也要記得她的好，因為他曾經在你舉日四望、茫然不知所措時幫助過你。

5.心有光亮，自予光芒。寒潮湧動的歲末，願你被溫暖的光包圍，或者，去做那束光。

6.那些獨自死撐的時光，不是不想被人打擾，而是不想打擾任何人。

7.如果今生很乏味，我害怕來生；如果今生很精彩，那麼今生就已足夠。

8.餘生很貴，努力活成自己想要的樣子，願你能穿運動鞋擼鐵汗如雨下，也能穿高跟鞋瀟洒貌美如花，不負青春，不負自己。為了未來美一點，現在必須苦一點。

9.常抱怨自己的生活為什麼一如既往的糟糕，後來想想自己只是常常努力一段時間後自我安慰自己已經很辛苦，然後就不了了之。也是，上天哪會辜負有心人。

10.如果驕傲沒被現實大海冷冷拍下，又怎會懂得要多努力才走得到遠方，如果夢想不曾墜落懸崖千鈞一髮，又怎會曉得執著的人擁有隱形翅膀，沮喪時才會明顯感到孤獨的重量。長大後，才體會到這段經典歌詞的分量。

11.世上最愛憎分明的是時間。你蹉跎歲月，它就讓你庸碌一生；你珍惜時光，它就送你前程萬里。用好每分每秒，你就是那個被時間偏愛的人。

12.痛苦是比較出來的，幸福是珍惜得來的。越是計較，心理越不平衡，越不平衡，煩惱越多，我們因此變得煩躁。 ​
