我們總以為電視劇里的劇情都是假的，終有一天，那些故事真的發生在我們身上，我們才明白：那有可能是真的存在。

朋友前幾天跟我分享一個故事，真的難以置信，卻又是不爭的事實。

事情得從三年前說起，高一的時候，她暗戀同班的一個男生。她成績很差，男生成績很好……為了讓自己和男生一樣，她開始發奮學習。當你有了目標，並為之不懈努力，上帝都會為你落淚。最終，她的成績基本穩定在年級前20，當她以為自己終於有能力向男生告白的時候，男生轉走了，去了其他學校。她悵惘了好一段時間，她沒有了目標，也沒有了往日的激情，覺得學習沒有了意義。曾經所作，會在未來予你相應的果。她成績一落千丈，最終只考上了二本。

大學以後，她找了男朋友，男朋友本來是網戀，後來考到了相隔的兩個省，見了面，也給了彼此的第一次。當她以為自己會很平靜的這樣下去的時候。在一次和從前暗戀男生的聊天中，她知道了，原來男生也一直暗戀著她，兩個互相暗戀的人，到高中畢業都沒有在一起。得知消息的那一刻，她整個人都震驚了。男生的話里有透露出希望交往的意願。她陷入了糾結，很糾結。

人生就是這樣，你以為一切都結束了，而它才剛剛開始，想要給你平靜的生活激起千層浪。

她男朋友跟她說：你不喜歡做飯的話，將來我做飯給你吃，但對於那個暗戀三年的男生，朋友說：我想為他做飯。我很驚訝，也不知道該怎麼回答。換作是我，我想也會很糾結吧。那幾天，她徹夜難眠。也許兩個男生在腦子裡不斷浮現。一邊是愛自己的男朋友，自己也愛他；一邊是暗戀了三年的男生，也希望交往。自己該怎麼辦。

我跟她說，無論你做出什麼樣的結果，後果都要自己承擔。如果放手重新來過，會傷害一個無辜愛你的男朋友；如果斷了暗戀的他，你過不了自己心裡那道坎。上帝有時候真的太殘酷，同時拋給我們兩個至關重要的問題，卻只給我們一個答案。或許，這就是人生吧，在面臨很多抉擇的時候，我們都不知所措。又必須做出抉擇，所面臨的，是相應產生的後果。

也許一時疏忽，我們會為自己做的決定後悔。但很多事情，一旦做過，錯過，便再也回不去了，破鏡也許可以重圓，但一定不會如初。

我不知道朋友會如何抉擇，我只希望，她所做的不會讓未來的自己後悔，也希望，她處理好三者之間關係，不要不明不白，不要無辜傷害。

生活中，我們有太多類似如她，怎麼做，真的太難，如果你也是其中一個，希望你可以一直幸運，不為過去後悔，不為後果背負傷罪。

如果你有所想法，希望你可以評論，給同樣境遇的我們，給出你真誠的建議。

如果你喜歡我的文字，可以點擊關注，我有故事你有酒，陪你感知點滴生活。
