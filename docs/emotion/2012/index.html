女人感到孤獨，不要只想著哭泣，嘗試做這3件事

大家好，我是約翰格雷，全球知名情感專家，致力於解決家庭矛盾，提供戀愛技巧。

今天給大家分享一下，一個女人感到孤獨的時候，可以做什麼事情？有些女人喜歡哭，還有些女人想要喝酒，甚至，還有更誇張的方式來逃避。

聽到你們如此對待自己，很心疼你們，我曾經和一位女生做出交流，聽她講了接近一年的孤獨過程，她在這樣描述自己。她叫李女士，29歲。

李女士這麼描述到：我感覺到孤獨是有原因的，22歲大學畢業以後，談過一場戀愛，當時25歲，戀愛整整3年，因為異地分手了。對方想要去外面，追求更好的工作，我難以接受，導致我們之間關係破裂。

整整一年多，我的內心非常痛苦，直到如今，我感覺到我生命還是很孤獨的。在一份關係裡面，沒有感覺到體恤和珍惜，忽然覺得，生命沒有什麼意義，人活著沒有愛，還有什麼意思？

那個時候，我其實已經染上了煙癮，甚至還想過輕生，不知流過多少眼淚。因為不知道怎麼處理，也不懂去跟別人分享，我內心的壓力很大，朋友關係也逐漸遠離。後來想想，她們離開我也有理由，如果是給我，也不知道該怎麼安慰我。

現在已經第4個年頭了，偶爾想起來的時候，心裡還是很痛苦的。我特別想走出來，需要人幫我一把，想著除了每天哭，還能做些什麼呢？

聽到李女士的分享，真心為她感到難受。失戀帶來的痛苦，不是一朝一夕可以得到緩解的。但我想著，可以做這些事情，在這裡要提醒一些女性。不要為了尋得別人的歡心，而出賣自己，不要把自己弄得遍體鱗傷。

想辦法讓自己走出來，我給她3點建議，嘗試去做一下。

01旅行一場

我很喜歡旅行，在全世界各地都想去玩玩。有時候趁著演講，會到各個國家，領略不同的風采，感受不同的文化，被當地民族的傳統感染的時候，我會覺得我人生格局太窄了。

旅行，是打開人眼界的一種方式，不再讓我活在自己的世界裡，使我們可以放開心，去感受世界的美好。探索和發現，總結和記錄，回憶和感受，體會生命的美好。

在一趟旅行當中，盡情的忘掉所有的煩惱，嘗試把心裡所有的話，透過寫字，講給這個大自然，跟以前的自己告別，重新開始新的生活。

02培養一個愛好

讓我們成熟起來，給自己培養一個愛好。記住。讓自己的生活豐富起來，而不是逃避，或者以麻痹的方式來面對。孤獨是有原因的，因為那時一直在逃避。

一個愛好，可以是畫畫，可以是讀書，可以是運動。總之讓自己開始感受世界的美好，排除煩惱，變得成熟。

因為有一個愛好，就會有新的夥伴，我們要有勇氣，開始一場新的友情，走出煩惱的世界，告別孤獨的那一刻，我們就不在孤獨了。

03幫助一個人

想要自己的內心得到醫治，最好的方式是陪伴別人。曾經看到過一些例子，病人對病人的安慰，是最好的。因為感同身受，情緒跟情感，都是相通的，愛的力量，也會發揮到更大。

我一個人煩悶的時候，通常是找同樣有煩悶的朋友，我們一起訴說，講完以後，我們內心就暢快了。

不必在乎別人怎麼看，就算一個乞丐能給到你安慰，你都可以嘗試。放下內心的包袱，走出孤獨的區域，只要有一個同樣經歷的人，站到你面前，你的心已經得到了同理。

三種方式，嘗試去做一下，重要的是，自己的生命要過得豐盛，不要再那麼孤獨，要告訴自己，孤獨是自己的選擇，並非別人的冷落。

當你感到孤獨的時候，都在做什麼事？如果有，可以來分享。
