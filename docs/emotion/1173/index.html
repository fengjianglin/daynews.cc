一位35歲鄉村教師的無奈：省吃儉用不敢花錢，我要存彩禮

文/雪姐說情

原創作品：抄襲必究

不瞞大家說，我小時候的夢想就是當一名教師，因為小時候家裡非常窮，所以想要成為一名老師的原因就是因為可以工資高，至少在我那個年代，教師的工資非常高。

可是呀，後來的我不爭氣，最終也就只能與教師行業無緣了。

但是長大之後，才明白小時候的想法到底有多麼的膚淺。我才明白教育的意義到底有多麼的重大。一位好的教師，可以說他的某一句話，某一個行為，都可能會影響孩子一生。

我很幸運，也有點遺憾，自己沒有成為一名教師，幸運的是我可能沒有能力承擔起那一份教書育人的責任，我怕我教育不出優秀的孩子，辜負了家長的期望，也耽誤了孩子們的未來，遺憾的是，它畢竟是我小時候的夢想。

最近，有一位老家的一位教師朋友和我閑聊，我才知道他這個年紀，難的不是教書育人，而是如何才能解決自己的單身問題。

01.

像許多的普通人一樣，成哥就是一個家境非常普通的人，他的父母和我的父母一樣都是農民，他們沒有很高的文化水平，所以供養自己的孩子讀出大學就已經非常努力了。

所以，他們再也沒有太多的能力為成哥建房、買房，存錢娶老婆了，這所有的一切，都必須要成哥一個人來承擔。

大學畢業之後，成哥就被分配到了我們鄉村當老師，他儘力地做到對每一位同學都呵護有加，對自己講的每一堂課都負責，可是不得不面對的現實就是，他的工資沒有辦法讓他承擔起結婚這件事情的經濟壓力。

現代很多人結婚，男方都要先準備好房子，車子，彩禮。而這卻難住了成哥。

35歲的他每天吃得格外的簡單，為的就是省錢存彩禮，然後娶老婆，過一個有家的日子。

看到此處，我格外的有些心酸，因為在村裡，成哥已經屬於妥妥的大齡剩男了，他說，甚至於有何很多的人都嘲笑他，可能這輩子要打光棍了。

不是所有人都那麼的幸運，很多人生下來什麼都沒有，很多的是事情都是需要他們一點一滴的去打拚的。

02.

寫在此處，我不由得想起了身邊的一個朋友。她出生在一個家庭條件優渥的家庭。所以在很多的言談舉止中，統統透漏出的都是會自我的優越感。

「你朋友沒結婚嗎？需要我幫她介紹男朋友嗎？她什麼家庭條件啊，自己有房有車嗎？」

「哦，什麼都沒有啊，那我的朋友可能看不上她，因為門不當戶不對嘛……」

不得不承認，有一個好的原生家庭真的是一件非常幸運的事情。比如王思聰，他在一出生的時候就已經在我們終點所不及的地方了。

可是怎麼辦呢？我們除了靠自己打拚之外，沒有人會無緣無故的幫助你。

但是我們至少不應還鄙視他們啊，他們正在努力不是嗎？他們已經努力地在改變他們的命運了不是嗎？我們有什麼資格去無視輕蔑別人的每一滴汗水呢？

結語：

我知道，現在很多的人在面對婚姻的時候，都必須要考量對方的物質條件。但是有的人整的在很努力地去湊婚姻的每一筆錢了。

誰不希望自己的人生可以輕鬆一點，但是人生的道路真的太過於短暫了。

仔細的回顧過往，捫心自問，你們多少的快樂是與金錢有關的？曾經記者問王健林：「你這麼有錢了，應該會很快樂了吧！」王健林說道：」我的快樂來自於金錢的僅僅佔5%「。

其實我們很多人忽略了，我們一直努力賺錢，錢帶給我們的只是一種安全感而已，是在面對失業、疾病中的一種無畏而已。

可是我們與之追求的幸福和快樂，卻只能來源於溫情，來源於某個人對你特殊的關愛與疼惜。所以啊，我多麼的希望，如果可以，你們都可以所遇良人，然後餘生都以笑、以幸福來宣告你們過得有多麼的人間值得啊！
