“不結婚，你必定會老無所依”，活到40歲，我徹底相信了母親的話

天天要聞
2019-12-14

我是東林夕亭，你有故事，就來找我。

點擊上面『關注』，你就是我的人了。

所謂「人活著就有希望」，只有經歷過絕處逢生的人，自己懂得建立希望的人，不讓自己心死的人，才能深刻體會到這句話的涵義。

之所以這樣說，是因為很多人雖然活著，但是並沒有希望。其實不是現實中和人生中不存在希望，而是，他們自己否定了所有希望，自己都認為自己沒有希望了，那還怎麼建立希望。

人活著，很多時候得懂得做取捨，知進退，這樣才不會在一條錯的路上越陷越深。

所謂「退一步海闊天空」，並非只指人與人相處時的妥協和讓步，放大到人生和生活，很多時候也得學會退一步。

當你走進死胡同的時候，鑽牛角尖的時候，走不通了，很痛苦了，這時候就需要你退一步。此路不通，換一條路走可能就通了。而如果該退一步的時候你還一意孤行，那你可能一直到老了都還在原地踏步。

下面這個男人的痛苦就來源於「執拗」，來源於他自己不懂得退一步，以至於才活到40歲，就感覺人生走到了盡頭，我們一起來看看他的故事，或許對你有幫助。

東林老師：

我感覺現在的我病得不輕，不知道是不是得了抑鬱症，我總是對人生感到絕望，在生活中看不到希望，雖然還活著，但是跟行屍走肉沒什麼區別。

我現在已經40歲了，以前不覺得年齡有多恐怖，可是現在，一想到自己的年紀，一想到自己的人生，我就感覺很無奈，感覺一片黑暗，感覺自己有點活不下去了，如果不出意外的話，我註定會孤獨終老。

曾經在30歲的時候，我父母就要求我結婚，讓我去相親。可是，那時候的我，覺得自己還年輕，覺得自己還有遠大的抱負沒有施展，人生還有得拼，所以就一直在拒絕。

哪怕後來我父母將催婚變成了逼婚，我依舊沒有妥協。就這樣，我父母漸漸放棄我了，我媽放棄我之前跟我說了這樣一句話：「不結婚，你必定會老無所依，必定會一無所有！」

當時我暗自發誓，一定要讓自己過得好，一定要按照自己的規划去走，只有這樣，我才能讓父母明白，我的選擇和堅持沒有錯。

可是，我空有滿腔熱血，卻沒什麼能力和本事。一個朝九晚五上班，工作沒什麼技術含量，一年到頭拿死工資的人，不跳出這個圈子，是不可能有大的作為的，可能過了十年二十年，依舊是老樣子。

就這樣，不知不覺我就40歲了，就像我上面說的，十年時間裡，我除了年齡增長了，其他方面沒有任何增長，甚至體力和精力方面還有退步，漸漸就感覺自己拼不動了，感覺人生陷入了灰暗。

尤其是身邊人陸陸續續都結婚生子了，沒有人陪我一起玩了，我就更加孤獨了，有時候甚至連個說話的人都沒有。

更可悲的是，我非但沒有婚姻，甚至連房子都沒有，一直到現在還在租房住。

以前想著等賺夠了錢就在自己喜歡的城市買個房子生活，可是現在，我一直賺不到什麼錢，房價又一直在漲，最後真成了我媽說的：我成了個一無所有的人！

活到40歲，我徹底相信了母親的話，我現在雖然想改變，雖然想結婚了，可是沒有人願意嫁給我這個一無所有的人。沒有人願意嫁給我，我就連將就湊合結婚都沒有可能，一想到這些，我就難受得夜夜失眠。

​此路不通，要及時換條路走！

很多人在活著的過程中，都會慢慢滋生出固定思維，都會慢慢活出一種固定模式。

如果你活出的固定模式是你甘願選擇的，不管怎樣都不後悔，是沒問題的。最怕的是你活成固定模式了之後，突然後悔了，這就很可悲了。

上面那個男人就是一種活成固定思維和固定模式的人，人在年輕時有理想有抱負不是什麼錯，關鍵是，你的理想和抱負應該切合實際。

你覺得適合你的路，你可以走走看，能走得通固然好，但是如果走不通，你就得做好兩手準備：

如果你繼續堅持的話，你可能會碰得頭破血流還一無所有，只要你不後悔就行。

如果你不想堅持了，那就及時換條路走。堅持雖然是一種好品質，但錯誤的堅持不是什麼好事。放棄也並非全都是壞事，放棄了一條錯的路，必定會是好事。

希望其他人好自為之，不要在一條錯的路上盲目堅持，你應該關注到現實，關注到年齡，不然這一切都有可能成為你之後痛苦的根源。

我是東林夕亭，你有故事，就來找我。

堅持到現在，你對你當下的生活滿意嗎？
