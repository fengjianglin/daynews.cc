英國青梅竹馬的小情侶被迫分離，40年後各自離婚的他們又走到一起

2019-12-19 286條評論 1,284次閱讀 346人點贊

很多人都有些青梅竹馬的朋友，他們可能從小一起長大現在還在一起，也有可能小時候相識但後來因為一些變動只能各奔東西。就像文中要講到的這對情侶，本是青梅竹馬一起長大的朋友，後來卻不得不分開，但40年後他們又重新相逢。

據外媒12月178日報道，這對戀人來自英國赫爾市，分別是今年50歲的John Burn，和他51歲女友Lisa Reveley。二人從小就是一起讀書的好朋友，對彼此也有好感，但卻因搬家只能被迫分開，如今40年後重新相逢，他們又重新戀愛了。

John與Lisa的第一次見面是在小學，他們很快就成為了好朋友，並從那以後就形影不離。漸漸地，隨著年齡的增長，情竇初開的他們對彼此的感覺從友誼變成了愛慕，並成為了一對別人羨慕的小情侶。

小學畢業後，由於搬家的原因，John與Lisa去到了不同的中學。從那以後，他們便很少再聯繫。長大後，他們也各自結婚，有了自己的家庭。然而，由於婚姻不幸，兩個人都離婚了，直到他們在Facebook上重新相逢，才脫離了多年的單身生活。

John說：「我點開了她網上的主頁，清楚地記得給她發的第一條信息是什麼。我說，你好，Lisa，40年沒見了，你還好嗎？沒想到她也很快地就通過了我的好友請求，從那時起，我們每天都要打一個半小時的電話。」

Lisa也表示：「我們從小一起長大，小時候我總是去他家玩，我們是彼此的第一個異性朋友，沒想到40年後能重新相逢。我記得我們的第一次約會就在一起待了7個小時，雖然我們有40年沒見了，但就好像從沒分開過，他也沒變樣子，只是比以前老了些。」

雖然去年11月份二人就重新聯繫上了，但直到今年才正式確定戀愛關係，Lisa說：「我本以為我今年要一個人過聖誕節了，打算在床上躺一天，但沒想到會遇到他，2020年將是屬於我們的一年。」

目前兩個人還沒有明確表示是否會結婚，但他們的孩子們都很支持他們的感情，他們也經常拿出小時候的照片回憶過去。John說：「如果我當初沒點開她的資料，可能就不會有今天的我們了，我覺得這就是命運的安排。」

網友們看後都發表了不同的看法，有人表示，果然緣分是命中注定的，儘管過了這麼多年，但該在一起的人還是會在一起的。但也有網友表示，如果真的相愛為何當年不去尋找彼此，老了之後都空虛寂寞了，才想起來舊情復燃。

不管怎麼說，愛情都是個你情我願的東西，不管是因為什麼原因，他們現在還能在一起了就是緣分。希望這次他們不會再分開，能夠攜手走進婚姻殿堂。
