李玫瑾教授：嬰兒哭鬧究竟該不該抱他？脾氣暴躁源於什麼？

2019-12-11 1,099條評論 2,983次閱讀 234人點贊

李玫瑾教授

免費領李玫瑾育兒課程全集（52期）

公眾號：李玫瑾育兒之道

李玫瑾教授育兒實錄：

有的人特別激動，你幹嘛那麼激動，幹嘛那麼暴躁，他過後吧他也說，我也控制不住自己，我當時是偏激，

那麼還有一種人你跟他打交道，你會覺得這個人非常的大度平和，就是你跟這個人接觸，多會說他都是那麼和藹，多會說他都是沉穩，他很少跟你急。

那這兩種人我告訴你：

前一種暴躁的人，他早年的撫養一定是虧欠的；

後一種人，早年的撫養一定是非常到位的，為什麼？

因為我剛才講了，人在一歲之內他完全不能自主，他要難受的時候，我們知道人最難受的事兒，其實不是餓、渴、睡覺，最難受的是不能翻身，我們成年人大家睡覺都會翻身，可是小孩是不能翻身的。

三翻六爬，三個月之內他是不能翻身的，那麼他不能翻身的時候，孩子就會大哭大鬧。

我記得我的孩子小時候，老人告訴我說你別老抱他，抱成習慣了趕明兒老鬧，現在我知道什麼問題。

其實孩子鬧他實際是要翻身，以一個姿勢躺在那很難受的。

那麼這個過程當中有的時候，孩子哭，我們要不懂的話，你就不去管他或者你就說，缺乏那種到位的撫養。什麼呢，只要餓不著就行了，也不管他，那麼這孩子會在那哭，

他有時候一哭，給你哭個半個小時、一個小時，主要是因為什麼呢？是我們的內臟。

就是情緒更多的和內臟相關，所以他內臟會出現什麼狀態呢，當哭時間長了我們都會出現抽泣，就是呼吸都會抽泣，然後包括你的胸都會疼，

那麼這孩子也這樣，他哭時間長了，他的內臟都是緊張狀態，因此當他成年他稍有一點不快，他的內臟就會出現相應的反應，

這時候他一定會出現激動，相反這孩子一哭呢，馬上就有人過來了，給換個體位抱一抱，過一會兒再放下，這個孩子他就很快得到緩解，

所以他就不會那麼急，那麼到他成年之後也是這樣，有事刺激他不會很快就急起來，他很平和。

那麼這個兩個人的性情，就可以看出他們早年的撫養背景，所以我講到這個情感發展，然後他決定言語，言語決定社會性，社會性會決定認知方式，包括後邊的性格。
