孩子總是不自信？比起學習成績，更重要的是孩子客觀的自我評價

天天要聞
2019-12-17

李子是一個很容易情緒低落的孩子，同時他也很容易因為外界的一些事情，而否定自己，其實也就是不自信，他不太相信自己能夠做「好」一件事情。
就拿最近發生的一件事情來說，他和他們班裡面的小麗鬧了點矛盾，事情的經過很簡單，就是小麗在玩玩具的時候，自己也想要玩，所以就盯著小麗看。突然小麗鬧了脾氣，一邊把玩具丟給了他，還一邊說：「老是看，煩死了，給你還不行嗎？」李子看著手裡的玩具，一臉茫然，不知道要怎麼做才好。

事後，小麗跟他道歉了說自己不應該那樣子對他發火。可是卻有個聲音，在李子心裏面說，是自己做錯了，忍不住懷疑自己，在想自己是不是太失敗了，什麼都做不好，總是會遭到別人的討厭。所以最後李子自己選擇疏遠了小麗。
其實，上面這種類型的事情，在那些不自信的孩子身上每天都在上演，而他們卻不知道要如何去改變這種狀況。這些會容易對自己有不信任情緒的孩子們，實際上就是由於他們不懂得如何正確的對自己進行評價，這才導致於他們對自己沒有給出過一個積極的評價，同時也沒有給出一個對自我比較客觀的評價。而是一味地去否定自己，習慣性的去責怪自己。所以可以通過去建立正確的自我評價的，以此來達到減少不自信心態的作用。

不自信會給孩子帶來很多傷害。

從上面就可以看出，不自信對孩子的傷害是有多麼的大，不僅會讓讓孩子喪失對自己的信心，同時他也會喪失很多，在日常生活中他所能夠爭取到的機會，特別是那些他本該可以經歷的一些美好。
不自信，不僅會讓孩子「錯過」，同時也有可能會讓孩子傷害自己。

就比如說，不自信的孩子他們容易出現抑鬱情緒。而抑鬱情緒有可能會讓孩子得上「抑鬱症」，可能有的父母會認為，這樣好像有點誇大其詞，但是其實抑鬱症本來形成的原因，很大程度上就是由於對自身的不自信，對自身的不看重，而長期的不自信，就處於這樣子的狀態，小孩子越來越抑鬱可能會出現自殘，甚至於輕生的念頭。
所以，不自信所帶給孩子的，可能不僅是心理上的傷害，很有可能還會有生理上的傷害。所以讓孩子們學會如何去正確的對自我進行評價，這件事情很重要，因為這樣可以在很大程度上避免孩子有由於對自己的判斷失誤而受到傷害的情況出現。

自我評價的重要性

評價在我們的生活中扮演著很重要的角色，不管是物品，還是人都或多或少會受到他人的評價，但是也會因為這些評價而受到影響。評價的力量是強大的，但是很多時候人們會忽略一點，那就是對自我的評價也很重要。
很多對自己不自信的孩子，一般都是在對自我的評價這一個環節出現了問題，他們不能夠比較客觀的去看待自己的優點和缺點，他們會無限的把自己的缺點放大，因此忽略了自己的優點，他們下意識認為自己沒有優點，只有缺點。這種悲觀的自我評價，會讓孩子在看到這個細節的時候，一直籠罩著一層烏雲。這層烏雲裡面滿滿寫的都是「他不夠好」，「他不能夠」「他沒有優點」。

正確的自我評價所能起到的作用。

  • 第一，可以讓孩子們建立自信。

就像前面說的那樣，大多數人會出現不自信單情況，有可能是他們因為失敗，而產生對自己的負面評價，或者是本身根深蒂固的自卑情緒在作怪。正確的自我評價所能起到的作用，第1個，就能夠讓孩子們建立自信。

因為不管是對自己的負面評價還是自卑情緒，影響這兩者的一個很大的因素就是，錯誤的自我評價會導致於這樣子的，所以正確的自我評價可以有效的消除對自己的負面評價同時，也可以減輕孩子的自卑情緒，客觀合理的看待每一件事情，和在每一件事情裡面自己所扮演的角色不用過高的，把太多的責任都交給自己來承擔。

  • 第二，可以幫助到孩子們的學習。

正確的自我評價，能夠帶給孩子的不僅是可以幫助讓孩子去建立他們的自信，同時也所以在學習方面上面去幫助到孩子。因為在這個自我評價裡面會使用到一個思維方式，那就是對於問題進行歸納，然後再進行分級處理，然後再一一給出一個客觀的評價，以此來達到得出一個比較不那麼主觀的對自我的評價。這種思維方式就可以很好的運用在學習方面。所以實際上正確的自我評價可以幫助到孩子們的學習。
既然正確的自我評價可以給孩子的幫助這麼多，那麼父母應該怎麼做才可以讓孩子開始學會正確的對自己進行評價呢？

首先，父母們可以通過使用ANEW模型，來進行對孩子們自我評價的培養。
ANEW模型
這個模型其實就是一種思維方式，父母們可以通過讓孩子進行下面4個層面的思考來，進行一個比較客觀的對自我的評價。
第1層面：儘可能的客觀的說出，概念或者是形成當前結果的原理是什麼？能夠使用的知識有哪些？
第2個層面：能夠說出知識點之間的關聯，並且能夠合理的進行聯想，來使這些知識點之間產生關聯。
第3個層面：可以通過所學的知識，來讓孩子們學會如何去把一個題目進行解構並且來找到正確的答案。
第4個層面：能夠把自己所學的知識應用在不同的地方，其實這也就是常說的舉一反三，就是通過一種類型的題目裡面所學的知識點，擴展到對所有事物的思維方式。
其實這個模型，不僅可以使用在孩子的學習上，同時也可以使用在孩子對自身的分析上。

比如說，學習方面：上面的概念或者是形成原理就是那些公式的背後原理，以此類推。

自身分析方面：而當孩子要進行對自身的評價的時候，上面說的概念其實就是她對自身的認知是什麼，關聯就是自己為什麼要這麼做，以前有這樣做過嗎？解構是對自我進行解構，合理的看待自己的優缺點，最後總結。

其次，父母們要懂得鼓勵孩子。

很多時候孩子會出現不自信的情況，出現對自我的評價有誤差的情況，可能是因為身邊的人對孩子的批評，擾亂了孩子們對自己的評價。

所以父母要懂得去鼓勵孩子，鼓勵能夠帶給孩子的不僅是對自我的認可，同時也會讓孩子是相信自己是一個有價值的人。

就比如說很多父母會在批評這方面有一個誤區，那就是他們認為自己去批評孩子，是一個可以讓孩子改正，並且可以讓孩子變好的方式，而很有可能父母所認為的這種正確的方式只是在反其道而行，對孩子帶來的只有傷害，並沒有起到任何實際的有利的作用。同時很有可能孩子在這種批評里所得到的只是對自己的不自信和對自己的悲觀想法。

最後，父母們要懂得把鼓勵和正確的自我評價培養，兩者結合使用。

所以父母們在孩子的教育中也千萬不要忽視鼓勵的重要性，要懂得合理的對孩子們進行一些鼓勵。就比如說一些言語上的鼓勵，實際上就是一種對孩子的認可在孩子完成事情的時候，父母們不妨送上一句「寶寶，這件事你完成的真好」的誇獎。再加上對孩子正確的自我評價培養，兩者相結合相輔相成，相信孩子就不那麼容易自信了。
