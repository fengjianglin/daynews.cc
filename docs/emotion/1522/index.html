遇見你之後，每個夜晚，我和星星一樣睡不著​

在感情里，懷念不一定就要相見，喜歡不一定就要在一起。要相信，每一種距離都有它存在的意義。

要記住每個對你好的人，因為他們本可以不這麼做的。

不管遇到了什麼煩心事，都不要自己為難自己。

不管前方的路有多苦，只要走的方向正確，不管多麼崎嶇不平，都比站在原地更接近幸福。

生活不可能像你想像得那麼好，但也不會像你想像得那麼糟。我覺得人的脆弱和堅強都超乎自己的想像。有時，我可能脆弱得一句話就淚流滿面；有時，也會發現自己咬著牙走了很長的路。

不要為模糊不清的未來過分擔憂，只為清清楚楚的現在奮發圖強；

「你看我那麼努力，所以請你愛我吧！」得了吧！省省吧！

猶記否？小橋流水人家，姑娘著青衫，芊芊素手弄琴弦，白牆黛瓦，錦瑟流年。

兩個人之間的感情就像織毛衣，建立的時候一針一線，小心而漫長，拆除的時候只要輕輕一拉。

曾經，你是我生命中的過客，一次偶然你成為了我的主角，我願意為你，退守四邊，只為與你一生不離。

所有的道理我都明白，但是理性終究敗給了情緒

說不盡的心酸也只能當笑談，述不盡的往事也只能隨風散
