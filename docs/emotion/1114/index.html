大地和天空從來不撒謊

而人類善於揣摩它們的忠誠

寂靜的山谷，已淺淺入夜

側耳傾聽溪水潺潺

小木屋的和諧交流之聲

昏黃的帳篷燈微弱的照耀

拉開簾幕

你來或者不來

我都是奔跑的羔羊

因為我的愛人阿

他在日出日落時

酒里有春光，開眼見星河，窮極一生在期待。我既要做獨居生活里的英雄，也要在人海茫茫中尋找你。最遠的距離是人心。或許我們有過片刻想要廝守終生的勇氣，或許我們只是一起被動做了情感遊戲中的傀儡。行走于山海間，唯情關難渡。在潮濕的夜裡，誰是最後兵臨城下坐上寶座之人，不得而知。
