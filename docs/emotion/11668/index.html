懶得向別人介紹你，直接說你是我的

2019-12-12 109條評論 1,028次閱讀 85人點贊

我想，我真的很喜歡你，很依賴你，想和你有很長很長的未來，很想把所有好東西都給你，很想肆無忌憚的愛你，很想每天跟你在一起，很想你生病的時候我都在你身邊，很想每天都有你大大的擁抱，很想你不開心的時候我都可以哄你，很想你也同樣依賴我，很想每天可以跟你說晚安。
你有沒有很想，和誰重新認識一次。
關於感情我承認我拿不穩，但我寧願摔碎了也不會拱手讓人。

生活加個你，就甜了。
我總不能跑到你面前說，我還是喜歡你，我很想你，你來陪陪我好不好，算了吧，真的矯情。
未識公子前，只覺得天下萬物風光俱為一色，幸識公子後，眼中哪還有什麼萬物，眼眸中單單公子一人，倘若沒了公子，這日子怕是很是難熬。

我真的是不太會表達的人，如果你感覺到我喜歡你，再乘個二十倍，就是我真正喜歡你的程度。
萬一努力的盡頭，是你呢。

喜歡你，繞八條街都是偶遇。

如果寂寞能下酒，那愛情便是一場宿醉。
段位和你，我都想上。

有人說，一段感情里，我需要你的時候你不在，以後也就不用在了。確實，沒有陪伴的愛情，總是讓人沒有安全感。

我心有顧忌重重，一重是你，千萬重，也是你。
辣條給你，西瓜給你，雞腿給你，芒果給你，統統都給你，能不能不要喜歡其他的豬。
你說過你就是那種沒有結局的人，可是我想像中自己踏上的每一條路的盡頭都必須有一個你。

別再透支我對你的好，哪天收走了你將一無所有。
最近的夢裡，十有八九都是你。
你應該知道我只待你不同。
