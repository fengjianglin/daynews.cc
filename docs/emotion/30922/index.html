彩禮引發血案：女友墮胎分手，男友弒母被捕——彩禮吃人?

2019-12-24 136條評論 1,865次閱讀 668人點贊

慘劇!為三萬彩禮，女子腹中胎兒被墮致死，男子母親被打致死，兩條人命不值三萬元，人心險惡，讓人不能直視!

彩禮三萬元，應該說不算多。男女雙方友好協商，男方家量力而出，女方家有禮有節，不至於導致慘劇發生。

由於男方母親拒付彩禮，女子非三萬元不嫁，甚至打掉了腹中的孩子，孩子無辜受死，毫不心疼，這女子心狠手辣還有人性嗎？

而男人只想到自己結不了婚，心中痛苦憋屈，責問母親付彩禮，母子起爭執，男人砸打家裡物品，最後獸性大發追打母親，木棍擊中母親頭部，母親倒地身亡。他竟進山躲避，由於警方追緝，才出來自首。他視人命如草芥，敢對母親下毒手，實在是禽獸不如。

一起由彩禮引發的人倫慘劇，暴露出目前社會男女婚戀中的頑症——金錢至上。

彩禮，古代也叫聘禮，由媒人交給女方父母。相當於現在的的定金。古代女子地位十分低下，男女沒見過面，女子就要嫁到男家為人妻，行使傳宗接代功能。女方家庭養育女兒，男人家得福利。所以，男方為了婚姻穩固，給女方家庭一定金錢和實物，以補償女家的損失。女方的代價是：女子出嫁永不得返回娘家。這是朝廷有明文規定的，誰違反誰被治罪。

現在的男子結婚，往往都會被彩禮難倒。因為車子房子本已開支巨大，最為令人頭疼的就是女方索要的高額彩禮，男方家庭不勝重負，有的會因此背上高昂的債務，甚至可能發生海南類似慘案。

如何對待和處理男女結婚彩禮問題?

一，男女確認感情，女子是否因彩禮而改變感情方向。

因彩禮，雙方家庭意見不一致。要結婚的男女不要輕易表態，應該兩人互相確認雙方的感情是否堅定，女方是否會因彩禮不滿意而放棄感情。結婚是男女雙方的事，如果男女情比金堅，彩禮就不是什麼問題，即使女方家庭不同意，也不會影響兩人結婚成家的方向。

如果海南男子事先與女子確認，知道了女子會因彩禮而變心，就不至於干下弒母之事。

二，男女統一認識，將心比心，改變觀念

如果男女是城市青年，接受過一定的教育，兩人相愛是一件難得的幸福之事。兩人就應該認真討論商量彩禮的事，統一認識：彩禮只是個形式，不能真正檢驗人心和感情。了解雙方對彩禮認識有多大差異，能夠調解平衡最好，如果女方堅持彩禮不變，男方可以承受，也是可以的；如果超過了男方的承受能力，男方就要考慮自己的感情出路了。

海南男女，肯定沒有統一認識，各行其是，導致事態惡化。

三，尊重雙方父母意見，各自開導勸慰自己父母。

女方家庭索要彩禮，無外乎兩個目的：一是延續舊習，檢驗男方是否真愛女兒；二是借女兒發財，獅子大開口。

如果是前者，雖彩禮過高，可以看出還是可以與女家商量的，只要不是」賣女求財」，大家好好說話，不傷感情。如果女方家為了面子執意而為，男人也需要三思。

如果是後者，男方可以商量是否請對方降低彩禮，如果對方固執不變，那男方就需要考慮，這個婚還是否要結了。

海南男女，女子順從家庭意見，男子自己沒有三萬元錢，太無能了。又不懂表達，逼迫母親，粗魯殘暴，造成弒母悲劇。

四，本著和為貴的原則，少埋怨，多體諒。

男女將要結婚了，男女雙方家庭就將成為親戚，這是多大的緣分!

本來，愛情跟金錢是沒關係的。愛情粘上金錢就會變味，除非是意志十分堅定的人，才能堅持愛情。

正因為如此，一個結婚彩禮，讓我們看到人性的弱點，改變不是一下子的事，有時需要暫時妥協，特別是態度和方法，一定要有「和為貴」的原則，男人要剋制自己的憤怒情緒，女人要剋制自己貪財念頭，雙方盡量不要衝動。

海南男女，完全沒有考慮自己行為的後果，女子為三萬元墮胎要挾，男子為三萬元弒母，人性崩壞，教養太差，距離「和為貴」差之萬里。

五，盡量不要被人言左右。

中國人要面子，「抬頭嫁女兒」，結婚要做給別人看，錢多被別人讚美，自己「面子」光彩。殊不知這種面子，並不能支撐夫妻幸福的「里子」。

虛榮一樣會害人。《項鏈》裡面的女主角用幾十年的人生還債，原來那只是一串假的珠寶。多少虛榮面子被拆穿，自己受罪自己知。

彩禮，好比一樁生意，女方要對方出錢，男方應該有議價的權利。如果都沒有商量，互相不體諒，還叫什麼親家，還有什麼面子!

海南男女為三萬元彩禮，斷送兩條性命，鬧出一樁慘劇。所有的要結婚的男女們，千萬不要重蹈覆轍，要記住他們血的教訓了!
