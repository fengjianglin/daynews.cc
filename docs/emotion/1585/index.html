這些騷氣的句子和背後的故事 溫暖你整個寒冬

2019-11-30 0條評論 25次閱讀 0人點贊

摘要

1.可能你不快樂 ，可我要你快樂 。可以讓你快樂是我的快樂。——來自趙紫驊《可樂》首先，大一喜歡的歌

1.可能你不快樂 ，可我要你快樂 。

可以讓你快樂是我的快樂。

——來自趙紫驊《可樂》

首先，大一喜歡的歌，現在才知道創造背景，有點遺憾。

句子來自一首歌，很多人單曲循環，我也是其中之一。這首歌寫的是愛情，故事的主人公是他的母親和他的繼父，兩個人都是在另一個家庭被傷過的人，走到一起，相濡以沫，彼此照顧，很幸福。忽然有一天，厄運來臨，他的繼父離開了人世，他的母親日夜哭泣，心悲涼，淚也干，無盡的痛苦……這觸人心弦的故事，感人至深的真情，人的一生很短也很長，不要說來日方長，誰也不知道明天和意外哪個先來。

2. 人來人往都不用在意，因為在我眼中只能看見你。

——來自謝春花 《一棵會開花的樹》

其次，高中時同桌喜歡的一首歌，而我就喜歡這一句。

對死後的思考，就甘願做一棵會開好的樹吧，在來年三月開花。一半在土裡安詳，一半沐浴陽光，一半給予陰涼，還有一半化作春泥。句子很短，樹很暖，情很長。有一些沉默，有一些安寧，有一些無聞，不尋找，不依靠，會開花。死後很悲，但不知曲子歡快？

3. 不念過往，不等時光，不懼將來，不負青春，世間美好都會與你環環相扣。

然後，來自即將十二月的原創

這句話送給正值青春正奮鬥的人，溫柔的十二月即將到來，它會善待我們。有為考研備之許久的人，我相信你，祝你好運，歸來仍是少年；有考了兩次英語四級沒過的人，祝你第三次必過；有在英語專業考級，老師和專業都給予你壓力的人，穩住，你能贏。奮鬥的青春是值得懷念的，願你我不負好時光。

4.好久不見，我想穿越人海與你相擁。

再次，有點想家，想過去的朋友。

有的思念無法用語言形容，有的思念電話中寒暄不懂，有的思念必須見面深擁。聽說，有人想你了，但是，你卻忙碌的沒有時間回復，後來，等你有時間了，對方已經下線。當朋友需要你時，無論你有多忙，請停下你忙忙碌碌，去找她。曾經有一段羨慕的友誼，就是我想你了，無論你在多遠，能立刻出現在我身邊。所以，有空記得給你好久不見的人一句問候，或者是一個溫暖的擁抱。

依稀記得，一個朋友說，她大學放假的早，她去看她高四的同學小葉，小葉見到她，什麼也沒說，直接跳到了她身上，含著眼淚抱著她。

5.儘管歲月冗長，我也陪你囂張。

最後，有些想做沒有做的事情。

小時候，快樂是簡單的。長大後，簡單是快樂的。有些人就是想去挑戰不可能，人的一生何其短暫，她需要一句：我陪你啊。 四個字，給了她多大的勇氣。有一些人，端著面子獨闖，我想，她的內心一定很害怕，願有人站在她身後陪她。

還有溫暖你的句子嗎？請你在評論區，燥起來啊。
