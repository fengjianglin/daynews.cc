謝謝你們，帶給了我三十幾年來最大的感動

2019-12-24 274條評論 2,098次閱讀 874人點贊

昨天冬至，離曹先生離開我們有二十多天了。從十五號的那天起到今天就整整刷了八天的頭條，微博和視頻，一邊刷一邊笑再一邊哭。直到昨天，心真的難過的不行了，就隨手寫了一篇感想，我沒讀過太多的書，語文也不好，就純粹的只是自己的一些感受。15號，因為休息，所以除了做飯給孩子們吃，我就手一刻不停的拿著手機，在那裡拚命的刷視頻，在我心裡，曹先生他從來都沒有離開過。就像陳導和焦俊艷所說的，他只是去了另外一個地方，我們只是暫時的分開而已，總有會相遇再見的時候，人嘛，總是會離開的。本來以為，隨著他的離開，熱搜榜會降下來，但是其實沒有，我們喜歡他，不是因為他的瀝川往事，而是因為他的善良，儒雅，紳士等等，他值得世間一切所有的美好名詞。他那麼多人愛，那麼多人喜歡，還有他跟焦俊艷之間說不清的感情，很多人都說他有女朋友，在我看來，婚姻和愛情本來就是兩碼事，他們那時候的情感就是那種純純的不參與任何雜質的感情，就像初戀。而不是像現在的世人那樣，每一次戀愛都以家庭背景，門當戶對，有房有車為前提，每個人的心裡都裝有一個自己內心深處想要靠近的人吧。焦俊艷就是曹先生心裡的那個人。他們以前的點滴能被人今天慢慢的回味，就證明了他們是感動我們的。有人說，他們是演員，專業的，生而為人，你是第一天看電視嗎？即使不看電視劇，就看曹先生最後的告別視頻跟排名跟他的日常，我們也會感受到滿滿的幸福。我們不是要在這裡爭辯對錯，我們也不想傷害任何人，就是想按著自己的內心，甜甜的去回憶他們的故事也就夠了。至於不好的視頻，不要再重複的發上網路，那樣會繼續刺痛我們愛曹先生的每一個人的心，比如他的父母家人還有焦焦的心，這也是不能挽回的事情。如果我們不喜歡看浙江衛視，就選擇不看吧，人生苦短，一切就從心，但不能打著愛曹先生的名譽，來質問他們，真相是什麼已經不重要了，可能這麼說會得罪很多人，但曹爸爸曹媽媽他們會做好的，如果我們真心愛曹先生，我希望在自己的有生之年，能到他的休息地去給他獻上一份花和給曹爸爸曹媽媽他們一個擁抱，難為了他們了。再說一說曹先生的經紀公司吧，你那天的聲明有些過了，我們是因為喜歡他，才這樣來思念他的，你不能用圓滿，這只是結束了一件事情而已，我們也沒有佔用公共資源，國家領導人還需要民意呢，中國的強大靠的都是人民，你不能這樣來傷害我們。我們只是為了曹先生的善良不想與你計較，相信曹先生也是這樣想的。謝謝頭條，我關注了頭條三年了，在這裡刷著我最心愛的心靈雞湯度過了我人生當中最難的三年，也感謝頭條這段時間裡帶給我們的感動。謝謝星空浩瀚888，謝謝美好33，謝謝你們對曹先生所有的美好的東西，很喜歡你們的文章。謝謝焦俊艷，一個可愛的愛笑的懂曹先生的女生，看到你出來的那兩張圖片，眼裡都是淚，而我卻跑到你的微博里來想證明你很愛他很難過，其實我才是最壞的那個人，我往你傷口撒鹽了，對不起‍♀️。謝

謝你們的遇見王瀝川，我只看了花絮，還沒有勇氣看電視劇，總有一天，我會慢慢的回味的，願你的以後，回憶他的過往，滿臉都是幸福的味道。曹先生沒有離開，他一直在我們心裡，謝謝你們，我不會寫文章，昨天到現在多了很多粉絲跟點贊的朋友，也謝謝你們，不用關注我，我是一個很懶的人，我只是因為愛他們，他們觸動了我內心深處的最大的感動，所以才隨口說兩句，願我們善良的這一生平安健康快樂，心情不好的時候，我們再看一看那些曹先生的感動我們的每一個可愛的視頻，讓我們永遠美好的生活，直到離去的那一天。
