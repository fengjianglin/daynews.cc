她賠上了自己的命，卻永遠也不知道她愛的人，真的不值得

2019-11-30 0條評論 25次閱讀 0人點贊

摘要

說起小姑的事，那得回到上個世紀80年代，她是爸爸二叔的女兒，從小身子弱，但聰明乖巧，二爺爺也只有一兒一女，夫妻倆非常寵她，把這個小女兒捧在手心裡。

小姑中專畢業，在廠長里做化驗員，20幾歲的姑娘，長得不錯，工作也好，當然就有許多人追。

可小姑那麼多條件好的沒看上，卻看好了在同一個廠子里，在車間幹活的小伙兒小剛。

小剛家那時剛從農村回城不久，家裡兄弟三人，他是老大，家裡房子很小，兄弟都住上下鋪。而小剛本人也才初中畢業，是車間里一個工人。

二爺爺家裡都不願意，自己的女兒從小嬌貴著長大，嫁給那樣的人家，她也吃不了那樣的苦啊。可跟女兒怎麼也說不通，一向乖巧的小姑認準了小剛，家裡人說什麼都聽不進去。

那時二奶奶一邊勸著閨女，一邊給她介紹對象，為了不讓小姑跟小剛交往，二爺爺像對小孩子一樣，早上上班送晚上下班接，反正是無論小姑怎麼說，家裡就是不同意她和小剛交往。

可一向被寵愛的小姑性子也犟，一天在家她找到了給菜地打葯的一瓶敵敵畏，她想嚇一嚇父母，可剛喝一口，她馬上就後悔了，上屋裡告訴了父母，二爺爺聽了，馬上扛起閨女就往醫院跑，路上因為扛在肩上，小姑把農藥吐了出來。

到了醫院以後，醫生給小姑檢查，告訴二爺爺，小姑不行了，二爺爺不相信，說只喝了一口農藥，而且半路上還吐了出來。

醫生告訴二爺爺，小姑患有肺結核，正值病情開放期，只要一滴敵敵畏，就可以致命。

就這樣，年輕的小姑走了。

大家把小姑自殺的消息告訴了小剛，以為小剛會很傷心，哪成想，小剛聽後，並沒有表現出悲傷，相反說出：「多虧我沒跟她結婚，要不是結了婚，有了孩子，哪天不高興自殺了，剩下個孩子怎麼辦。」

可憐那個為了愛情，想用死逼迫爸媽同意的小姑啊，你所謂的真愛，真的不過如此[流淚][流淚]
