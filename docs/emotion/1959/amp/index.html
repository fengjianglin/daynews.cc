聽說吃素久了的人能聞出吃肉的人身上有臭味，真的嗎？

天天要聞

摘要

我一閨蜜吃素，從小就不吃肉，連蛋奶都不碰，父母和外公外婆都是如此。閨蜜老公追她的一年半一直說自己也是吃素的，在閨蜜家吃了無數次飯也沒露出馬腳。閨蜜跟我說：雖然身高沒達到期望，但碰到個也吃素的不容易啊。不然將來在一起了倆人怎麼吃飯？

我一閨蜜吃素，從小就不吃肉，連蛋奶都不碰，父母和外公外婆都是如此。

閨蜜老公追她的一年半一直說自己也是吃素的，在閨蜜家吃了無數次飯也沒露出馬腳。

閨蜜跟我說：雖然身高沒達到期望，但碰到個也吃素的不容易啊。不然將來在一起了倆人怎麼吃飯？

現在呢，常態是她老公和我老公擼串，我陪她在一邊啃黃瓜吃糖拌西紅柿，權當減肥了。

能聞出吃肉者的臭味……難道我閨蜜和她父母技能點錯了？

=============

以上為原回答。

沒想到隨便舉的例子，竟讓大家如此好奇，統一回復幾個問題吧：

1.閨蜜膚白貌美，個子確實不高但比例很好，我老公說楊超越瘦一點能比我閨蜜強點不多，所以絕對不是身材佝僂面有菜色；

2.閨蜜和她的父母性格絕不怪異，特別隨和，可以說有些大咧咧那種，不然我們也不會初中後一直在一起；

3.營養方面。首先，她家頓頓有豆腐，南北豆腐都至少有十幾種烹飪方法，甚至干豆腐(千張)也有七八種吃法。從初中起就是我見過的最會做豆腐的叔叔阿姨。

其次，據我觀察，雖然不吃肉，但閨蜜一家的熱量攝入並不低(畢竟人家不是為了減肥)，烹飪重油，還愛吃甜食。我就沒見過愛吃水果糖的同齡人，閨蜜是唯一。確實不健康，但也能理解…不然豆腐也不會做得那麼好吃。

再次，各種維生素微量元素，我初中還不怎麼懂的時候，閨蜜就已經認認真真在吃了。蛋白粉她父母吃不吃我不知道，但閨蜜肯定是吃的。

4.健康情況。肯定是比普通人差一些的。閨蜜月經紊亂、皮膚容易過敏、牙齒大學就有鬆動現在已經種了4顆了。以前總是喊累，但健身後好多了。但總的來說，並不是病懨懨的狀態，是很活力四射、很樂觀、很有趣的一個人。

5.信仰這個問題，到我閨蜜這一代應該就截止了。其實她自身也談不上信，只是從小就是那個環境，而且家裡從不吃肉，她長大後也就自然而然地排斥。知不知道不健康？肯定知道啊，但身體就是排斥啊沒辦法。
