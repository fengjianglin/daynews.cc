經歷了什麼事才知道錢的重要性的？網友:因為沒錢只能睡公園

2019-12-18 889條評論 2,122次閱讀 291人點贊

六七年前，一無所有。睡過公園，吃過白開水下麵條，欠一身的帳，也苦過身邊的人。好在一切都過去，可是也物是人非了。現在長沙兩個車一個房。

就現在吧。家裡經濟真的沒得多餘的錢，還房貸，還借了別人的錢，借出去的錢又拿不回來。真的難，問我媽能不能拿出兩萬到四萬去學傳媒，都沒辦法。

女朋友跟我兩年多了，我們是高二在一起的。大學異地戀，那天我坐火車去看她，她陪我看了天氣之子（我是新海誠的死忠粉）。當時在電影院我說買桶爆米花她沒同意，說為我省點。

25歲，負債200多萬（公婆投資失敗沒分家所以一起負擔），家裡農村家庭獨女，親爸生病住院2000塊錢拿不出來，坐大街上抽一整晚煙，不知道該怎麼辦。

17高中畢業出來。一個月包吃包住3000。一個星期花完。幾個月都是這樣。後面幾個星期都是一些兄弟照顧我吃飯，我後來覺得錢太重要了。但是我還是改不掉花錢大手大腳的習慣。然後去了部隊。兩年出來了。

最窮的時候應該就是現在吧，身上剩幾塊錢，並且負債纍纍，做生意失敗欠了幾十萬，但是我相信一切都會過去的。只要我還活著。現在對我好的等以後我一定會還回去，看不起我的以後也一定會讓他刮目相看，加油吧。
