人，三十歲前窮，可以怪父母；三十歲後仍窮，就是怪自己...

天天要聞
2019-12-19

人們常說三十而立，自己到現在還是和幾年前沒有改變！有很多的事，你不努力是沒法成功的！

三十歲是個尷尬的年紀，上有老，下有小。

我初中畢業就出來打工到現在也有十年了，回顧這十年來自己所擁有的，深深地感到有一種無力感。我和我老婆有兩個女兒，跟著爺爺奶奶在老家，我們在外地打工，過著分別的日子，只是為了生活不得已而為之！我想這是普通很多打工者的生活！

三十歲這個年紀，往往孩子剛上學，父母年紀大，家庭的重擔全部壓在自己身上。你能做的只有努力工作，為了孩子不走自己老路，受到更好的教育，為了家庭幸福，為了更好的生活而奮鬥。

我零八年出來打工，記得那時父母常年在外，自己跟著爺爺奶奶生活，得不到家庭的關愛，生活學習都不如意，到了初中成績越來越差，為了減輕家庭的負擔，追隨父輩腳步出去打工的。這也是生活所迫，這種現象好像在我的記憶里成為了一種出路，很多人都是這樣做的，在浙江打工一個電器廠員工上千人大部分都是沒結婚的年輕人，由此可以看出打工潮，在當時很多農村家庭這是一種很好的出路！

打工那麼多年有了家庭，孩子。生活慢慢變好，快三十歲了，我想的是不求大富大貴，只求家庭幸福！我不會怪誰，自己選擇在當時是正確的決定，也沒有什麼好後悔的！

成家立業，是每個普通人最基本的願望！不要求自己要有多大事業，但是起碼不想孩子父母為了錢發愁，只能繼續努力，畢竟你只有靠自己努力才能使家人過上好日子！我覺得我還是很幸運的，有一個愛我的老婆，兩個可愛的女兒，疼我的爸爸媽媽！為了這些我所擁有的努力！

我們需要變相的看待這個問題。不能盲從，也不能人云亦云。

在30歲之前，我們都是已經有了一定的社會經驗了，並且，我們接受了相當長時間的教育，有了明辨是非的能力。

也已經工作了3,4年了。當然因為我們之前沒有什麼積蓄，在結婚，買房，生子中有可能是需要父母的支援的。

但是，我們並不能把我們有可能遭遇的困難和稍微的拮据的生活就怪罪到我們父母的頭上，這是一種不恰當的說法。

我們的父母在我們小的時候，他們也是年輕人，也有自己的追求，只是因為有了我們他們才努力的工作，努力的生活，為了我們的將來。

難道我們長大了，在經歷了一段時間的歷練後，會把我們在現階段所謂的貧窮強加在父母的身上嗎？她們並不是不努力，不是不想要進步，可能是受到了當時的條件所限，或者是我們的爺爺奶奶，因為小孩子必須安穩的生活。

這些在我們工作了一段時間後，一定是有所了解的。我們的父母也是不容易的。

在我們30歲後，我們也很努力的工作，為了自己的家庭和孩子。

但是發財的必經還是一小部分。因為人需要在什麼年齡段就做什麼年齡段的事情，在30歲後，我們也一定結婚生子了，在父母，在丈人和丈母娘那看，我們的生活挺好的，沒有必要因為所謂的發財而去冒險，因為我們已經是一家之主了，一位孩子的父親，一位妻子的丈夫，一對老夫妻的兒子。

安居樂業，這可能也是我們父輩時候的無奈，那時父輩們也和我們現在一樣。因為有了老婆孩子這樣的家庭，所以冒險也就不是很恰當了。

窮和富的界限，我們在30多歲的年紀的時候，我們應當也有了自己的界定，這是完全反義的兩個字。但是我們已經到了不單純的以數字來界定它的年紀了。

在我看來，我的生活很富裕，有愛自己和我也愛的妻子，一個可愛的孩子，一對身體健康的父母，這就是我的幸福了。

而貧窮呢？也是不能以一串數字來界定，它的含義有很多，像是什麼我們精神世界不富裕，需要書籍知識來補充等等，這些也是一種貧窮。

最後，我想說的是，我們在任何的年齡都不能要求我們的父母怎樣怎樣，美好的生活還是要靠我們自己努力的拼搏！
