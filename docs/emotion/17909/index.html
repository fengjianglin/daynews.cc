人只要踏實肯干，總會出人頭地，人只要勤奮努力，終會眼眉吐氣

2019-12-17 323條評論 1,376次閱讀 431人點贊

人這一生，最怕什麼？不怕沒錢，不怕沒權！怕就怕，荊棘的路不敢走，喪失骨氣；怕就怕，為了賺錢昧良心，啥事都干！一個人，想要工作上出眾，就得行動上出力！沒錢沒啥，只要有勤勞的雙手，腳踏實地的雙腳，就可以改變你的生活。

踏實肯干，是一個人做事的通行證。堅守誠信，誰都願意靠近；感恩圖報，誰都願意幫襯。勤奮努力，是一個人成功的關鍵牌。踏實肯干，總會出人頭地；勤奮努力，終會揚眉吐氣。

勤勞的人，命運總會峰迴路轉，人這一輩子，若想出人頭地，就得勤勉肯干！有坎坷，又怎樣？認認真真干就完了，得之我幸，失之我命！勤勉肯乾的人，比一般人豁達，勤勞的人，不懼風雨，樂觀的人，不怕失去！

活這一輩子，若想踏實安心，就得坦坦蕩蕩！人生道路，荊棘遍布，沒人幫你披荊斬棘，只有勤快的雙手可以另闢蹊徑！現實生活，人心莫測，沒人為你遮風擋雨，生活的忙碌，讓你熱情洋溢，人活一輩子，金錢會散，名氣會淡，財富再多，都是雲煙。唯有一個人的勤勉，讓人生充實無比；勤快，能讓你遇見想要的未來；勤快的人，好運會一直追隨你！

踏實肯乾的人，寬容大度。他人有錯，能包容；他人計較，會讓步。懂得容人，會贏得好人緣。勤奮努力的人，善良慈悲。見人落魄，不取笑；看人風光，不毒舌。換位思考，會收穫好稱讚。

人活一世，活到底，講究的就是要靠自己的努力，贏得生存於世的資格。投機取巧，錢再多，也有走霉運的一天。藏奸耍滑，再有人幫，也有機關算盡的時刻。唯有靠自己踏實肯干、勤奮努力，才會被人稱讚，才會被人高看。

人這一生，靠自己，才最有底氣。靠自己的人，行得正，站得穩，背後沒人議論，說話有人贊成。人這一生最寶貴的永遠是踏實肯干、勤奮努力，知輕重，知情義，懂感恩。人這一輩子：人品放歪了，做事就難了；人品擺正了，做事就順了；人品弄丟了，做人就輸了；人品保住了，做人就贏了！

人才是磨練出來的，人的生命具有無限的韌性和耐力。不積硅步，無以至千里；不積小流，無以成江河。凡欲成就一番功業，都需要付出堅強的心力和耐力。只要你始終如一的腳踏實地做下去，無論在怎樣的處境，無論大小事都不會去拖延，你便可以創造出令他人欽佩的成就。
