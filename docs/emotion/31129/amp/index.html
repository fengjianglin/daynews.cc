為人處事：遵守這六條天規，才叫大智慧！

在為人處事的時候，一定要遵循規律，要學會用智慧與人打交道，尤其是下面六個天條，哪一個都不要想著法去破壞，自己才能夠獲得良好的人際關係，為自己的成功插上翅膀。

【1】不狠不服——你不狠點，你不毒點，別人一定認為你好欺負，你的善良讓人貪得無厭，肆無忌憚，得寸進尺，學會攻守，不能以理服人，就以力服人。這就印證了一個道理，很多時候霸道，沒有道理，往往事情進展得更順利，這是更高層次上的思想統一，理解的就理解，不理解，先執行。

【2】不勢不利——絕大部分人都是看重你的勢力才與你交往的，你的勢不在，交情說斷就斷，當你沒錢之時，人情說散就散，別把人間的情看的太美好，大多都有刺。大家都要看淡這一條，事情就是這麼的現實，不要去說什麼風涼話，更不要去埋怨這個社會的沒有人情味，人不為己，就沒有積極性，即便是有積極性，也是一種應付，把這一點看透，才能夠有助於理解人與人之間關係的本質。

【3】越己越人——你最大的競爭對手是自己，不能超越自己，你永遠超越不了別人。很多人並不明白，自己最大的競爭對手到底是誰？每天在苦苦的尋覓自己的競爭對手，當發現沒有競爭對手的時候，不是競爭對手不存在，而是自己眼光太低，其實這個對手就是你自己，克服了自己身上的毛病，戰勝了自己身上的弱點，什麼樣的競爭對手都不是你的對手。

【4】氣怒小和——生氣的時候，不要發怒，小的利益可以實現眾和。和人打交道，控制自己的情緒是有多麼的重要，只有失敗者體會才會更深切，通過這種事情，一定要搞明白，只有給大家一些小的利益，也就是先捨得投入，別人才有可能給你帶來回報，而且這種回報不是必然的，是一個自然的過程，不予強求

【5】體人者贏——能體貼人性人心的人，才是大贏家。一個人最終的輸贏，一定輸贏在人心上，所以誰能夠體貼到人性當中的人心，誰才能夠成為最終的贏家，這就應驗了一個大道至簡的道理，得人心者得天下，在職場上，得人心者得晉陞。

【6】眾智者愚——真正能集眾人之長的人，表面上都顯得非常低調，甚至讓人感覺有點愚笨。對於這樣的人，可能我們都有一點看不起他們總是很低調，總是顯得笨笨的，其實那是他們裝出來的，真正具有大智慧的人，他讓你覺得他很平凡。
