看穿借錢真相，規避"借錢是大爺，要錢是孫子"

2019-12-20 2,025條評論 4,548次閱讀 1707人點贊

跟一個朋友聊天，扯到一個千古難題：幾年前，憑著一股哥們兒義氣，借了一筆上萬的錢給一個老同學。剛借出那陣子，人家對你噓寒問暖，打得火熱。時間稍長，聯繫就少了。中間自己缺錢時，含蓄地討了幾次債，沒要成。結果是聯繫更少了。現在呢，我已經聯繫不上對方了。微信聯繫不上了，電話打不通了，錢打水漂了……

這位朋友道出了被借錢的人的三大痛楚:

第一痛：不借，自己的心靈飽受道德折磨：不夠朋友，不夠義氣，見死不救的愧疚。

第二痛：借出，收不回，自己飽受經濟壓力折磨：自己已經勒緊褲腰帶過日子了，外面的債卻一要一個空。

第三痛：錢要不回，還不敢得罪人。但最終，還是得罪了。落得個人財兩空。

遭遇這種借錢坑的人多是「好好先生」。

好好先生最大的特點就是經不起別人賣慘和裝可憐，不敢對人說不，不敢得罪人。這是好好先生絕對的死穴。特別是遇到那種曾經給過一點小恩小惠的熟人，拿著「你看那會兒我對你如何如何，你看現在我這邊情況緊急……」這句話，對其獅子大開口。此時，好好先生是毫無招架之力的。死穴被點中，內疚之心與道德正義感頓生。手指一抖，錢就划過去了。當你後悔時，要麼人與財杳無音信，要麼人家翻身變大爺，吃定你就是拿我沒辦法。

此時，你或許會在網上、朋友圈當個噴子，發發牢騷，寫寫這個社會人心不古之類的評論。這或許能聊以自慰，但對於擺脫厄運，是毫無助益的。下一次借錢坑，該來的還是會來，無非是換換馬甲，變變套路而已。

除非，你能自省自救。有句話說，「禍福無門，唯人自招」。這句話就是告訴你，你借錢被坑，別人可以同情你，但你自己絕不可以同情和原諒自己。因為，這都是你自己招來的。你唯一要做的就是反觀自省，為什麼中招兒的是我而不是別人？自己到底是哪根神經搭錯？

自己曾經也是一個典型的「好好先生」，只不過現實中經歷了太多”借錢是大爺，要錢是孫子”的慘痛教訓，開始反觀自照，逐一審視自己輕易借錢出去的心理動機中到底藏了什麼梗。最後，發現了一個極其荒謬之真相。

這個真相就是：在借錢上容易遭坑的人，幾乎無一例外地對「借錢」這個行為寄予了太多的價值期待。你期待對方視你為知恩圖報的大好人，你期待維繫或者加深友誼兄弟姐妹情，你期待獲得別人美譽，比如「你是個夠義氣的朋友」，「夠豪爽的哥們兒」，「夠知心的閨蜜」諸如此類。

但你沒意識到的是，你期待「什麼」，這個「什麼」就將成為你的死穴。善良的人，不會抓住你的死穴整你。但對於可以厚顏無恥、可以不折手段的人，有這樣死穴的你則正好是他們到處檢索獵狩的對象。你不入他們眼，誰入？所以，你附加在借錢這件事情上的期待越大，這個期待就越成為那些目光精賊的人控制你的資糧，你遭訛的概率就越大，痛苦糾結自然就是無法避免了。

怎樣解除這個死穴？我後來用的方法很簡單，也很有效。就是一招：無所期待。這招「無所期待」有兩層細微且至關緊要的要訣。

第一，對拿友誼情深來兌換你錢財的人，要對他所謂的友誼情深絕對的無所期待。

借錢出去，有所期待，人之常情。我投之以桃，心裡期待你報之以李，人情往來，我覺得這是普通人進行人際互動再正常不過的事情，並沒有什麼大問題。

但如果你把這些期待寄托在錯誤的對象身上，那就有問題了。而慣拿友情、親情、義氣等來說事的人，就屬於這種錯誤的寄託對象。

為什麼？

先問一個問題：假如有人拿著刀架在你的脖子上，然後說「老兄，兄弟一場，怎麼也得資助一下吧」。這種人，你會指望他以後在你困難時反過來幫你一把嗎？

你可能說，我連還都不指望了，哪裡還指望他以後會幫自己喲！

既然是這樣，那請問，拿著各種情義道德架在你身上，管你要錢的人，和那個拿刀威脅你借錢的人有什麼本質區別嗎？

答案是，他們都是善用威脅手段來借錢的人，只是一個違法，一個不違法。

威脅是一種最划算的交易。公平的交易是你幫我十分，我還你十分。這個雙方都得付出對等的成本。而威脅，如果成功的話，付出的成本為零，但是卻可以獲得十分的收益。

假如，你讓對方的威脅成功一次，就意味著你在暗示他或者跟他一樣精賊的同類，還可以第二次，第三次，以至於無窮次……此時，你還想收回本錢？等他們良心發現，恐怕也是下輩子了。

因此，面對這種威脅式借錢，第一次，你的應對方式至關重要。

怎樣應對？教你一句咒語。

即當你遇到凡是帶有威脅性質的借錢時，甭管他威脅得多麼大義凜然，你就一個勁兒地在心中默念：「嘿嘿，又來了，不稀罕；嘿嘿，又來了，不稀罕……」。然後，堅決、果斷、乾脆、利落地拒絕。不用找搪塞的理由，也不用多費口舌解釋。不借，就是不借，你要絕交，要如何如何，請便。

當然， 這個咒語湊效的前提是你得看穿一個事實真相：他們開給你的友誼票，兄弟姐妹情大概率都是空頭支票。你只有看透這一點，你才會對他們所謂的「真誠、赤熱」的「誓言、承諾」打心底地無所期待。只有你無所期待，你才敢狠狠地得罪他們。因為你知道，即便得罪，你也根本就沒有損失什麼，反倒是保護了自己的資產。得罪這樣的人是要有資本的，對他們的情啊義啥的，不報任何期待，就是你得罪他們的資本——面對對方強勢的道義攻擊和綁架，你可以心安理得地無動於衷。從此，再不會在借錢這事上磨磨唧唧。

第二，假如你一定要借，那就對「收回錢」這件事，保持無所期待的心態。

有朋友可能會說，現實生活中，總有些時候，不管是在生意場上，還是工作場的人情關係互動往來中，免不了會遇到同事、朋友確實需要借錢的時候。人家也不對你進行情感道德的綁架。你不能總抱著「鬧僵就鬧僵」的態度啊。此時，該怎麼辦呢？

還是一樣，無所期待。只不過這裡的無所期待是指，如果不得不借的話，你要對「收回錢」這件事情，保持無所期待的心態。這並不是要你假裝不在乎錢，而是要你懂得策略性地讓借出去的錢對自己來說真的無所謂還不還。

這個策略就是「他借大錢，你給小錢」。比如，別人給你借2萬，你最多給兩千，那即便不還，就當為福利事業做貢獻了。或者學錢鍾書的做法：凡有人借錢，一律打折到自己不在乎的數量範圍，然後直接送給對方。這樣，即便沒完全滿足對方的要求，他也不好找你難堪，或者跟你撕破臉。

有的人建議借錢時，防範風險的最好方式是在你借錢之前，別礙於情面，要對方留下字據等法律保護手段。這樣的建議某些場合有用。但是，對於好好先生來說，即便留下了法律保護手段，往往也無濟於事。因為，借錢的人如果有心要賴你，他一定會比你更了解你，有很大把握吃定你的性格，他才會向你下手。一般，他不會給你借很多。這樣到時賴賬時，有好人心的你，多會因為不大點損失，選擇以和為貴，忍氣吞聲。所以，就算你打了借條，他深諳好人們多半都不會為了一點小錢把事情鬧得很大，給自己找更大的麻煩，他們還是可以賴得心安自在。

對付這樣的爛人，其實只有一個辦法，就是要比爛，你的手段可以比他更爛，更狠。但倘若你真的能做到這種程度的話，你又不會入他們的法眼了，多半也不會找你借錢了。

所以，面對各種借錢的坑，最好就是看透，看破，然後做到兩個「無所期待」:

1.積攢得罪人的資本——對爛人的深情大義無所期待，不怕得罪，從而把他們擋在外面;

2.積攢慷慨解囊的資本（努力多掙錢），遇到確實該幫或者不得不幫的時候，讓散出的錢無法影響到你，因此收不收回，你都無所期待。

如此，錢，借，或不借，你都洒脫自如。
