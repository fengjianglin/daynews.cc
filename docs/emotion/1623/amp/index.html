導語：我是@鄉村匠人勇哥，一直想寫一篇這樣內容的文章，打開電腦就想起了我仍然生活在農村的父母。

那還是兒子出生的時候，我們在城裡買了房，每次商量父母和我們一起住，可是爸媽只是住一晚上就回。這次因為孫子出生了，需要媽媽幫助照顧，於是欣然答應和我們一起生活。起初還好，過了一周時間，媽媽才發現自己生養了3個兒女，到現在才發現自己不會照顧孩子了，抱孫子都是緊繃繃的，媳婦兒總是帶笑容委婉的接過來，我感覺到媳婦兒不放心年邁的媽媽照顧，媽媽也是心有餘而力不足。最後，還是決定顧了個月嫂！

同時，我發現爸爸媽媽從來不在洗手間上廁所，有一次我碰見爸爸媽媽在離家200米外的公廁上廁所，回家爸爸和我說，我和你媽實在用不慣坐便，沒法上廁所，特別晚上憋者真難受”。聽著我很心酸，和媳婦決定把洗手間改造成蹲便，媽媽死活不讓！就這樣過了1個月的時間，我漸漸發現爸爸媽媽不像以前那麼愛說話，也看不到了他們鄉村的那份笑容。有一天爸爸終於忍不住在飯桌上說：”我和你媽商量好了，明天回鄉下，孩子也滿月了，你媽年齡大確實照顧不了什麼，我們在這還得給添麻煩，我們也不自在，享不了這福。你們孝心爸媽都知道，你們好好的我們就放心了，等孫子再大點兒，帶老家來我們哄他玩兒！”看到爸媽決心一定，我也沒再說什麼。

默默的誰也沒吱聲，我一晚上也沒有睡，第二天把爸媽送走了，看到他們背影有心酸又無奈！一直在思所也許我們錯了，不應該把我們認為的享福強加給予他們，也許對他們就是在遭罪！

農村父母在城裡生活的尷尬究其原因

1.生活習慣的衝突，讓父母不自在

農村父母一輩子習慣了農村的田園生活，每天聽到雞犬的聲音覺得踏實，車水馬龍讓他們感覺躁動不安，失去了寧靜的生活狀態，心神不寧哪來的幸福啊？同時與孩子一起生活總感覺不是自己的家，起早了怕打擾孩子們的生活，起晚了又睡不著很難受。在家裡剩菜剩飯不捨得扔可以喂餵雞鴨鵝狗，在樓里只能扔倒垃圾桶看著心疼，就連嗑個瓜子還得將皮扔到垃圾桶，不像在家方便。抽煙的父母更是只能躲到樓道里等等。城裡成為了他們生活的籠子！

2.心靈的孤獨，讓父母心情壓抑

一輩子的鄉里鄉親互相串串門嘮嘮嗑，抬頭不見低頭見都是熟人，碰到就能聊上一會，有什麼事情大家都能說說心裡痛快痛快。可是在樓里就是對面鄰居都說不上一兩句，關門大吉了！出去後更是誰也不認識，不知道說什麼，大不了就是寒暄幾句。回到家和孩子們又沒什麼可說的，談起工作的事兒又插不上嘴。真是”邁開腿確張不開嘴”，讓他們生活的非常壓抑！

3.默認”自己沒用”，母覺得”混吃等死”

來到城裡本來以為與兒女生活在一起享受天倫之樂，慢慢才發現自己真的老了，什麼也做不了，就是在混吃等死。不像是在農村生活還能自食其力有活兒干，很充實感覺自己還沒老。其實，父母最可怕就是心裡默認自己老了，當有一天你發現父母臉上的笑容少了，他們真的老了。對於父母精神生活的幸福才是他們真正的”享福”！他們最不願意成為兒女的”累贅”！

總結：兒女的孝應該是建立在對父母精神生活和生活習慣足夠尊重的基礎上，強加給予只能適得其反。我們有理由相信作為孝敬的兒女，當自己的生活條件好了想給予父母更好的物質生活。因為一想起父母曾經的苦和為了兒女默默付出一生的心血，就恨不得把自己認為的好都讓父母享受的到。

其實，當父母年老了他們真是無力消受山珍海味、高樓大廈。當他們老了，他們就是一個孩子需要只是多一點的陪伴、理解、包容。哪怕是一句問候和短暫相聚也好！只要一個”安心”,在他們心裡只要”孩子們健康、和睦、走正道他們就知足的不得了！

因此，農村父母在城裡生活的尷尬，看似尷尬實則不是享福而是遭罪，別在讓自己認為的”享福”來綁架父母，如果你還不知道父母真的想要的福是什麼？那說明你好久沒有好好的觀察和了解自己的爸媽了，當下能做只有四個字”及時行孝”!
