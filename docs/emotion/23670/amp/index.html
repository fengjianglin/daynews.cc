自信的人，不過就是明白了這5個道理

天天要聞
2019-12-19

自我價值感低是我們生活中非常普遍的一種心理,它對我們的影響就是感到自卑,在與人交往中不夠自信。

那麼,怎麼才能改變這種情況,讓自己變得更自信一點呢?今天小編就和大家分享幾個實用的小方法。

01

找到自己的優勢

很多人以為,讓自己變得更自信的關鍵是怎麼改掉自己的缺點。其實,當我們這樣想的時候,可能離自信反而越來越遠。

為什麼這樣說呢?

原因很簡單,問題是解決不完的,即使你再優秀,也不可避免的會存在這樣或那樣的問題。假如我們一直盯著自己身上的問題和缺點不放,是永遠也不可能獲得自信的。

因此,最重要的不是改了多少缺點,而是能把自己的優點發揮到什麼程度。一個自信的人,一定是在某些領域擁有一般人無法比擬的優勢,這就是一個人的核心價值。

所以,我們一定要清醒地意識到:你能否相信自己,或者讓別人相信你,關鍵在於你有沒有價值,有多大的價值,而不是有沒有問題。

02

培養從容和淡定的習慣

如果我們留心觀察就會發現,不自信的人眼神總是飄忽不定的,因為他們的內心是慌亂的,所以眼神就會顯露出這種不安,表現在行為上則是急躁、毛手毛腳的。

這是因為,他們太想急於表現自己,證明自己的價值,所以總是過度地把注意力放在結果上,而不能夠踏踏實實地把過程做好。這其實就是一種急於求成的表現,結果就是讓自己更焦慮,事情反而更不容易做好。

所以,從容是在說,我們要把注意力更多地放在做事的過程中,把握好節奏,一點一點的來,不拖延但是也不慌亂。當我們能把過程做好的時候,好的結果也就是水到渠成的事了。

而當結果如我們所願的時候,我們的自信心也就很容易爆棚了。

03

不要輕易追求完美 ,追求足夠好

人人都想要完美,完美是一種理想的狀態,能實現固然是一件很值得驕傲的事情,但是我們也要清醒地意識到,完美是可遇而不可求的,它有時候會出現,但並不總是能夠做到。

所以,事事追求完美的結果只能是:對自己越來越不滿意,充滿了挫敗感,甚至變得自卑。

事實上,真正有成就的人不僅追求理想,更懂得接納不完美的現實。他們非常清楚地知道,不完美才是常態,所以不會無謂地自卑。

他們在設定目標的時候,不會去追求不切實際的完美,而是追求一種可實現的足夠好。這次我做到足夠好,下一次在此基礎上讓自己做得更好一點點,這樣一種不斷的進步和自我提升積累的多了,就會讓自己從一個不禁風雨的小樹苗,在時間的沉澱下最終長成讓所有人都仰望的參天大樹。

04

盡量獨立,承擔必要的責任

「一個人的幸福程度取決與他能夠在多大程度上獨立於這個世界」。

很多的時候,獨立意味著在生活中遇到事情和問題時,能夠更多承擔起自己的責任,能夠用自己的能力承擔起責任的人必然會變得更加自信。

自信的人都明白這樣的道理:

1. 自己是自己,別人是別人,自己的事情自己完成,這是對自己的負責。能夠做到這點,就能夠找到自我的安全感。

2. 在別人需要幫助的時候,假如自己有相應的能力,也要積極的幫助別人解決問題,這是對身邊人的擔當、能夠做到這點,就能夠找到自我的價值感。

責任也是一種壓力,而適當的壓力是可以激發讓我們變得更好,更優秀的動力。所以,勇於承擔責任,也是讓自己變得更自信的重要途徑。

05

把錯誤和失敗當成自己必須承受的代價

就像我們剛才說的,問題是解決不了,只要我們的生活在繼續,那麼問題就會像泉水一樣不斷地湧現,所以,如何正確看待問題,看待自己的錯誤和失敗也是一件很重要的事情。

培養自信不但要認識自己的優勢和核心價值,還要理性地看待自己的問題,不誇大它們的嚴重性。錯誤也好,失敗也罷,它只是讓我們吸取教訓,並從這些教訓中汲取經驗的,而不是用來否定自我的。

一個真正自信的人,絕不是只能接受自己的好,而不能面對自己的不好的人,這樣的人即使表現得再自信,內心也是很脆弱的。真正自信的人,是知道自己的優勢,同時承認自己的不足,接納自己不完美的人。

所以,真正的自信,屬於那些看到自己的不足以後依然選擇相信自己的人。
