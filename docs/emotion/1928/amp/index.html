劉強東24歲是在中關村賣盜版光碟。

這就是我們現在的現實，20歲出頭的年紀，肩膀上扛著父母，心裡裝著妹子，眼裡看著的是未來，現實里想著房子車子。

面對這些，我們真的不用太急。我們不用在意他人的眼睛，只要記住我們要對自己負責。

胸懷是被委屈撐大的。

生而為人，誰都是第一次做人，面對委屈我們不用過分計較，一笑了之。我們要化動力為勢能，學會忍受。

沒有一種工作是不委屈的

  • 剛畢業，身邊很多的同學在互相的詢問如何在職場中成長：
  • 我們的主管每天只有一張苦瓜臉沒有笑容難看死了。
  • 我們要如何從學生的狀態轉換成職場人。
  • 我們剛從學出來，沒有那麼多的錢，怎麼在一個陌生的環境裡面生存下來。

在抖音中看到一句話：當你十年後回頭看這些事情時，那都不是事。真的！

評論區的一句話亮了：哪裡需要十年，一年就足夠你能夠感覺到千山萬水物是人非。

一經歷坎坷，記住美好

我們都是乳臭未乾的孩子，初生牛犢不怕虎。可是理想很豐滿現實很殘酷，工作的本質是：委屈撐大了胸懷，建立了事業。

這句話我想了很久，反覆的刪除，反覆的敲打。可是這就是現實吧！

多年前劉若英在接受採訪時，沉默了許久，慢慢說出為什麼這一生總能給人溫和，不急不慢的感覺，並不是她沒有遇到崩潰。

那是因為我知道，沒有一種工作是不委屈的。

她寫出了《我的不完美》，在她還沒有正式出道前，她是著名音樂人陳升的助理。

在助理這崗位上，她什麼都要做，包括刷廁所。每周她要刷三到四次，她和另一位助理排班，另一位助理就是金城武。

二你能展現的只有態度和行動

剛進入職場，我們要學習很多，從與上司的對話，到與同事的交流，與下屬的溝通。我們要儘快學習工作的必備技能，並且轉化成自己的風格，我們所有人都知道在大學裡面學的有八成在工作中用不到。

這個時候一個人的學習能力和勤奮是最大的優勢。我們要向所有人學習，從你的上司，你的前輩。他們的待人接物，他們的說話方式，他們的工作方法。

我們不是富二代，沒有耍個性的資本。這個時候我們就是像被蒙住眼睛的大象，摸著石頭過河。沒有家人，沒有老師，沒有朋友，有的只有自己這顆堅毅的心。周圍的一切快速的從你的眼前經過，就像是快進的影片，身邊走過的同事面無表情。這一切都在飛速的進行著，我們有的就是在深夜獨自哭泣，第二天鼓足勇氣接著前進。

和我一同入職的朋友，本身是個慢熱的性格，有時候上司布置任務當他剛說出「哦」的時候，領導已經走了，只留下恍惚的他。這不是在責怪，這就是職場，而且這就是深圳的職場，快速，簡介，明了。

有時候我們要學會給自己鼓掌。在這裡我們的付出領導都在看著，但是他會很明顯表現出對你的關懷，因為有太多人了。他們也不會用力的批評你，這就是職場的心態：不悲不喜。我們要學會做的職場成熟人。

我們的同事沒有好壞，只是我們是否在同一個層次上，這無關學歷。這個關於的我們的格局，我們的為人處世。

三我們都在度過寒冬，要學會給自己充電

我對面的窗戶做的是我的主管，他每天在外非常的慈祥，但是回到自己的椅子上，馬上面部就變成了嚴肅，冷漠。

他需要面對的是每天90後團隊，面對上司的業績要求。有時候在我們面前的和藹，在上司面前又變成了犯錯的孩子。思考團隊的發展，思考團隊的方向。

每天都在思考，學習!

不求最後，只求每次進步一點

我們無法把一件事情就到完美，但是我們能夠重做一件事每次做到更好。

一位教學總監對我說過：

讀萬卷書不如行萬里路，行萬里路不如閱無數人，閱無數人不如高人指路。

同樣的事情前輩們的交流能夠給我很多的知道，他們犯過錯的能夠給我們很多的指導，能夠減少很多的不必要的彎路。

有時候做到極致不是最好的選擇，每天進步一點才是更好的！
