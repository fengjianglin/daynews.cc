一個真正活明白的人不會讓自己活得太明白

2019-12-24 2,930條評論 3,949次閱讀 1527人點贊

很多東西捨不得丟，很多東西捨不得忘，就是因為這些捨不得，才讓我們變得脆弱。

人性中確定存在醜陋的東西，比如自私、貪婪等，重要的是你看問題的角度，就如從前面看孔雀開屏，十分艷麗，可你非要轉到孔雀屁股後面去看，結果慘不忍睹。所以我們要積極地看待人性中美麗的東西，如正直、善良、誠信、責任、忠誠、友愛等。激發正能量，傳承好品德。

我們的生命，不是因為討別人喜歡而存在的，我們是自在之物，我們不必討任何人的喜歡，就可以歡天喜地地背負大地，面朝青天。只要你認定了這一點，枷鎖就被打開，你就可以自由地呼吸了。

人追求的當然不是財富，但必要有足以維持尊嚴的生活，使自己不受阻擾的工作，能夠慷慨，能夠爽朗，能夠獨立。

我沒有感覺到絕望，也沒有感覺到希望，我告誡自己，不要以現在的心情去規定未來。

所謂的行事體面，情商高，和正義，開朗、溫暖，友善等等這些外在的美好品質，都建立在一個人自身能量充沛的基礎上。這種能量包括健康體魄、穩定的情緒、精神的滿足感，它們是人一切行為活動的底層支持和儲備，就像一棟大樓的根基，一旦某方面能量供給出了問題，人馬上就會變得力不從心，無法得體了。

我比較欣賞為人處事有界限感的人，他們看似對待他人不熱情，甚至有些冷漠，但深入相處後，往往這些人才是真正可以交心和值得信任的人。而缺乏界限感的人，一開始或許對人非常熱情，毫無隔膜，但最後往往是人際關係中最求全責備、忘恩負義、自私自利的一方。因為他們會本能地認為，你的時間就該是我的時間，你的資源就該是我的資源，你的生活就該同我的生活建立聯繫。

人生有度，過則為災，世上沒有百分百的完美，缺陷，本就與完美並存。做人無須刻意，做事也不必苛求，活得豁然一些，人便能快樂一些，花看半開時，酒飲微醉處。

人老去的標誌之一就是，悔恨逐漸代替了夢想。

如果看不到確定的未來，還要不要付出？只能說，並不是每一種付出，都是追尋的結果。在付出的路上，能夠收穫的，是清楚地看到了自己想要的，或者不想要的，這又何嘗不是一種寶貴的結果。

一定要記住，千萬不要用離開去威脅任何人，最後你會發現，原來你真的沒有那麼重要。

曾經以為自己活得很明白，後來才發現：一個真正活明白的人不會忍心讓自己活得太明白。

但凡有了欲蓋彌彰的解釋，都會有一個不可告人的後序。

人生的選擇其實很簡單，就是往自己心裡感到踏實的地方走，不會錯。

你拿詩和遠方和苟且斗，你永遠斗不贏。因為苟且有一萬種需求，一萬種痛苦，而詩和遠方只有一種快樂。
