願有人與你共黃昏，有人問你粥可溫

2019-12-18 332條評論 2,992次閱讀 225人點贊

我喜歡冬日暖陽，喜歡在黃昏時分，看那枚圓圓的太陽，在天邊慢慢地，慢慢地墜落。橙紅色的光線，落在身上，很輕，很暖。那種感覺，讓人會不自覺地想起沈復先生《浮生六記》中的那句：「閑時與你立黃昏，灶前笑問粥可溫。」我覺得，這是沈先生對幸福的最高概括。每每讀到此處，即使窗外冬意料峭，內心依然溢滿溫馨。

　　沈復的幸福，是只要有時間，就會陪著芸娘，游山看水，共黃昏，看夕陽。兩人牽手相伴，吟詩唱和，「獸雲吞落日，弓月彈流星」。他的家，有兩人舉杯對飲的默契，有鍋碗瓢盆的交響，有他笑問芸娘粥是否溫熱的幸福。捧著《浮生六記》，我常常沉浸在他和芸娘的世界，忘記世俗人間。

　　相信，每個人的心中，都渴望擁有一份這樣的溫情，渴望有一個人與你看夕陽，共黃昏。

　　現實總是一地瑣碎。下班回到家，從「啪」地那聲關門聲響起，黃昏被關在了門外，世界也被關進了門外。家，是疲憊一天後可以休憩的港灣。

　　在這個港灣里，有粥可溫，有情可待，有人可守，便是最美。《文學回憶錄》里記載，木心在最後一課里說：一個人到世界上來，來做什麼？

　　「愛最可愛的、最好聽的、最好看的、最好吃的」！

　　生活不易，既然來到這個世界，我們就盡情地去愛可以愛的人，去聽最好聽的歌，去品嘗最美味的生活。生活，不僅僅是柴米油鹽的滋味，也應該有賭書潑茶的風雅。清風明月也好，青梅煮酒也罷，只要快樂，就是幸福。

　　日子一天天，無聲地逝去。過好每一天，把日子過成幸福的模樣，才真正懂得人生滋味。找一個陪自己共黃昏的人，一地雞毛的日子，也會甜蜜而溫暖。與心愛的人廝守一生，歲月靜好。一如林語堂先生所說：「一個謙卑渺小的生命能快樂地過一輩子，是宇宙間之至美。」

　　生活之美，在於你黃昏燈下，拾撿一地瑣碎。生命之美，在於你黃昏燈下，有人陪你一起拾撿一地瑣碎，為你端來可溫的粥飯。

　　所以我願，往後餘生，有人問你粥可溫，有人陪你立黃昏；

　　所以我願，每一個熱愛生活，熱愛黃昏的人，終有一天，你期盼的，時光都會贈你；

　　所以我願，在你的字典里，永遠寫滿笑容，落滿幸福。
