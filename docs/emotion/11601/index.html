A：先成家，後立業，你很相信這句話，很多時候，你在單身之時，因為沒有底氣，根本就無法決定一些重大問題，所以，你的事業一直停滯不前，並沒太大的發展，幸好，到了結婚之後，你總算是有了一個強大的後盾，能夠得到很好的機會，總算是騰飛了，在事業上開始發財，有了 更多的好機會，事業順心如意，一飛衝天，大紅大紫，這只是一個開始，未來的人生將會一帆風順。

B：其實你是容易早早成名的人，相對別人的發達時機，更早一些，比如十幾歲就揚名天下。但是之後你容易遇以一些危機，並且事先根本無法防備，這可能會導致你人生曇花一現，因為基本上不愁錢財的事情，人也比較晚熟一些。或者要中年，48歲之後才又迎來人生巔峰。

C：不是你努力了就可以成功，也不是你努力了就可以有錢，發達需要一定的機遇，如果這二者缺少其中的任意一個條件，你都不可能取得成功。一般情況下，你不願意為了自己的理想而努力，因為你不願面對巨大的壓力，你害怕受到傷害。如果有了機會，你不去努力，那麼你將會失去這個機會，這樣一來，你不會飛黃騰達。所以，趁著年輕，好好努力，說不定，你還有飛黃騰達的希望。

本測試僅供娛樂，不論選到好的還是壞的都不要過於的開心和難受。這些都是外力，更多的是靠您自己，命運只是一部分，需要命好+努力+機會才會成功，如果缺少某個就很難獲得大成就。所以說不論選擇到好的還是壞的，終究還是得看你的努力、勤奮等。留下你的評論吧！如果有趣可以關注我，每天為大家帶來各種有趣、開心的心理測試。
