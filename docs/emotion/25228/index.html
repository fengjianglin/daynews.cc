我不是孫悟空，也不是至尊寶，終究成了城牆下的那群人

2019-12-21 157條評論 1,433次閱讀 498人點贊

有些相遇是完美的，有些事不完美的，從一開始的爭吵慢慢的改變，慢慢的發現已經習慣了這樣的愜意，但是是終究成不了情侶的。但是內心還是放不下這一種感覺，彼此雖然心裡有對方，相互牽絆，那怕很久沒有聯繫，也能被對方的一句話或者一個動態牽動心神，再次撿起來，依舊是無話不聊。

　　情路多坎坷，失望總比喜悅多。

　　彼此都明白的感情比談一場戀愛簡單的多。因為沒有點破的愛可以有很多種，朋友，兄妹，知己，愛人……用各種各樣的方式與方法去關注你，沒有了戀愛的風風雨雨與兩敗俱傷，只剩下珍藏呵護與關心。

　　也在夜裡狠心想過從此不再聯繫，讓自己減輕一些痛苦，讓這種感覺慢慢的褪去。但一次又一次的為自己找到無數種理由與借口，晚上掙扎想著到底要不要繼續愛你，卻永遠沒有答案，當你每天早上太陽升起的時候，有時愛你新的一天。

　　沒有傷害，沒有傷心，只有靜靜的陪伴，哪怕十天半個月不聯繫，不見面，相遇之後，沒有任何的遺憾，責備，有的只是滿心的歡喜與久違。彼此一個微笑，就能感覺到無比的滿足。

　　恰到好處的愛情是太陽，溫暖而熾熱，過度靠近會灼傷，而晚來的感情猶如春風，溫和而和煦沒有那麼多的大起大落。有的只是包容與熟悉。

　　伸手怕錯過很愛不能在一起，有時候很甜蜜，有時候很痛苦。

　　沒有什麼相見恨晚，只是過了深情的年紀罷了。有緣無分，那就沒有糾纏。

　　在喜歡的世界裡，本來就沒有多少的盡興而歸，既然喜歡，知曉結局，為何不享受屬於自己的快樂呢？過了喜歡與愛的年紀，就應該有著屬於自己年齡段的成熟與穩重，要學會自己給自己療傷，不管是黑夜還是在低谷，有傷口都要學會自愈，時間都是平等的，不管在劫難中你失去了什麼，他都會以另外一種方式對你進行補償。所有的傷口在實踐的洗滌下終究會慢慢的變成美好的回憶，不管是曾今深愛的那個人，還是你自己，你會發現經歷過之後，你還是你，只是會慢慢的變得越來越懂得人情世故，越來越成熟。

　　保持一顆熱情洋溢的心，就能發現人間處處皆美好。

　　沒有完美的人，也沒有完美的事，得與失，只有等過去了，才能予以評價，活在當下。命里有時終須有，命里無時莫強求。別活在記憶里，掙扎徘徊。可以懷念，但千萬別影響你的生活。

　　你若安好便是晴。不完美的相遇讓我懂得該如何成長。

　　到頭來，我不是孫悟空，也不是至尊寶，終究成了城牆下的那群人，看著別人的愛情，咀嚼自己的青春。
