盤點50句心理學經典，無論明天怎樣，至少今天沒有蹉跎

2019-11-28 0條評論 21次閱讀 0人點贊

摘要

世上充滿了有趣的事情可做，在這令人興奮的世界中，不要過著乏味的生活。一、凡不關心別人的人，比會在有生之年遭受重大困難，並且大大傷害到別人。也就是這種人，導致了人類的種種錯失。二、傷害人的東西有三種：煩惱、爭吵、空的錢包。

世上充滿了有趣的事情可做，在這令人興奮的世界中，不要過著乏味的生活。

一、凡不關心別人的人，比會在有生之年遭受重大困難，並且大大傷害到別人。也就是這種人，導致了人類的種種錯失。

二、傷害人的東西有三種：煩惱、爭吵、空的錢包。

三、從你來到世界上這一天開始，你所有的每一種行動，出發點都是為了你自己，都是因為你需要些什麼。

四、你希望別人怎麼待你，你就該怎樣的對待別人。

五、知性和年齡有關，和閱歷有關，和心態有關。

六、人生沒有絕對的公平，卻又是相對公平的。在一個天平上，你得到的越多，也必須比別人承受的更多。

七、如果我們有安寧的思想和足夠的勇氣，那麼坐在自己的棺木上，人們也能欣賞無限的風景，在饑寒交迫時也能高聲歌唱。

八、我們之所以感到疲勞，是因為我們的情緒是我們的身體緊張。

九、聽別人數說我們的錯誤很難，但假如對方謙卑地自稱他們也並非完美，我們就比較容易接受了。承認一個人本身的錯誤，就算你還沒有改正過來，也可以幫助改善行為。在指責別人之前，先想想自己的錯誤。

十、在這個世界上，影響他人的唯一方法就是談論他們想要的，並且告訴他們如何得到。

十一、只有愚蠢的人，才會批評人、斥責人和抱怨人，但若要寬恕和了解他人，那就需要在人格及克己上下工夫了。

十二、一個在奮鬥途徑上努力的人，要是不把步驟分清楚，等於你旅行一個地方，不先規定睡眠和行程一般。分清步驟，是十分重要的。

十三、一個人的事業成功與否，完全在他對這項事業是否感興趣，而不是用苦幹去打開的成功大門。

十四、不論推銷什麼，你首先要做的第一件事情就是儘可能地了解它。

十五、盡量在舒適的情況下工作。記住，身體的緊張會製造肩痛和精神疲勞。

十六、為了成功地生活，少年人必須學習自立，剷除埋伏各處的障礙，在家庭要教養他，使他具有為人所認可的獨立人格。

十七、世上人人都在尋求快樂，但只有一個確實有效的方法，那就是控制你的思想。快樂不在乎外界的情況，而是依靠內心的情況。

十八、一個不容否認的事實就是，凡是你遇見的人，都會覺得他們在某方面比你強。巧妙地承認對方的重要性，並由衷地表達出來，就會使你得到他的友誼。

十九、無論明日，有多落魄，至少今天，沒有蹉跎。

二十、許多人無法成為一名良好的與人交流者，主要是他們只會談他們自己感興趣的事情，而這些事情卻令其他人感到無聊透頂。

二十一、做一個有堅定信念的普通人。

二十二、從希望別人所給我的，而去給別人。

二十三、人不是因為沒有信念而失敗，而是因為不能把信念化成行動，並且堅持到底。

二十四、對別人好不是一種責任，它是一種享受，因為它能增進你的健康和快樂。你對別人好的時候，也就是對自己好的時候。

二十五、對別人的恭維，不過是婉轉地表達對自己的讚美。

二十六、我是否曾主張，我們應對向著我們而來的一切災難低頭屈服？絕不！那只是宿命論的主張。只要有讓我們解救情況的一絲機會，我們便要奮鬥。

二十七、一個人如果真誠的關心別人，在兩個月時間裡所交的朋友，要比等著讓別人對你發生興趣，在兩年時間裡能交的朋友還多。

二十八、做一個善於傾聽的人，鼓勵別人談論他們自己，這是讓別人如何喜歡你的方式之

二十九、如果你能確定，在你一天中你有一半的時間是對的。那你可以去華爾街了，日賺百萬美元，買遊艇，娶舞女了。如果你不能確定，你憑什麼要指責人家的錯誤呢？

三十、快樂是有傳染性的，而只有使別人快樂才能讓自己快樂。

三十一、只要相信自己做的對，就不要在意別人怎麼說。

三十二、此心自有道理，是為道理所不自知。

三十三、設法協助別人，恐懼便會消失。

三十四、世俗有「時間是金錢」這句話，所以竊取他人時間的小偷，顯然該加以處罰。

三十五、如果你被人批評，那是因為批評你能給他一種滿足感。這也說明你是有成就的，而且引人注意。小人常為偉人的缺點或過失而得意。不合理的批評往往是一種掩飾了的讚美。

三十六、任何人的信用，如果要把它斷送了都不需要多長時間。就算你是一個極謹慎的人，僅須偶爾忽略，偶爾因循，那麼好的名譽，便可立刻毀損。所以養成小心謹慎的習慣，實在重要極了。

三十七、愛情不是去愛一個完美的人，而是用一段時間去明白，對方是個不完美的人。承認我們都不完美，然後去試著理解對方的那些缺點，並與那個雖然也看到自己的一身缺點，卻依然愛自己的人攜手前行。

三十八、找醫生看病的病人中有70%只要能夠消除他們的恐懼和憂慮，病自然會好起來。

三十九、對於我們大家來說，生活中最重要的事情不是遙望將來，而是動手理清自己手邊實實在在的事。

四十、困難也許是上帝對你的特別饋贈，將開啟另一段幸福生活。

四十一、你所認為的你，並非真正的你，反倒是你怎麼想的，你就會變成什麼樣的人。

四十二、我們的疲勞通常不是由於工作本身，而是由於憂慮、緊張和不快。

四十三、與人相處時，應該記住。我們不是應付理論的人，而是在應付感情的人。

四十四、女人可以不美麗，但不能不智慧，智慧能重透塑美麗，唯有智慧能使美麗常駐，能使美麗有質的內涵。

四十五、比自身生命更高貴的奉獻精神，會帶來真正的成功與快樂，會帶來更多朋友。

四十六、世界上充滿了這些人：自私、樂於擄取，所以少數不存私心為別人提供幫助的人，能夠大獲成功，他們幾乎遇不到競爭對手。

四十七、記住人家的名字，而且很輕易地叫出來，等於給別人一個巧妙而有效的讚美。

四十八、如果你對某項結果足夠關注，你自然一定會達到。

四十九、永遠按照對方的觀點去想，從他人的立場去看待事情，這或許會成為影響你終生事業的一個關鍵因素。

五十、我們雖然是平常的人，但不代表我們就該承受折磨。我們唯一能做的是正視生活本身，在經歷中不斷學習，以確保未來不會發生憾事。
