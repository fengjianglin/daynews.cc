沒有經歷何來成長，經歷了最痛的愛情故事，嘗到了最狠的分手方式，可以流淚，但不可以頹廢。

不計得失，不說愛恨，相逢一笑泯恩仇。

愛情本就是甜蜜伴隨著傷感，又有幾人能逃過這大喜大悲。

人生就是這樣，帶著些許的不舍和惋惜，帶著些許的不甘和遺憾，慢慢的理解生活，慢慢的更新自己。

沒有人能一路單純到底，但別忘了最初的自己。

飯要和你投緣的人吃，日子終要和懂你的人過。

過好自己，別人的故事不需要你……
