中國為什麼會演變成結婚要求男方車，房，彩禮，並且愈演愈烈？

2019-12-21 330條評論 2,619次閱讀 184人點贊

各位男同胞想要杜絕這個問題，我覺得一定要有一以貫之的價值觀，一定要不管面對任何對象都要堅決反對，假以時日，這個現狀肯定可以改變的。

一個男人一生中會面臨這些問題大概有三次是與自己切身相關的。

1、自己找老婆的時候，要堅持不找要房要車要彩禮的女人，你要相信，這種女人有的，而且還不少，除了房車彩禮之外，相信你肯定有其他可以吸引這種女人的獨特魅力。

2、有姐姐妹妹的男性，自己姐姐妹妹結婚時，一定要堅決抵制自己父母問男方要房要車要彩禮，並且在自己能力範圍之內說服父母將自己的姐姐妹妹和自己公平對待，不論什麼要麼都不給，要麼給得差不多，不要因為性別而區別對待。

3、如果自己有兒有女，兒子結婚，拒絕找要房要車要彩禮的女人，女兒結婚堅決不要房不要車不要彩禮，公平對待兒子女兒，不因性別而區別對待。

如果以上這三點可以做到，結婚要求男方房車彩禮什麼的都將成為歷史。

現在你知道為什麼現在是這樣的狀況嗎，因為從上一代開始，他們只想做到1，而不想做到2和3。這在本質上是矛盾的。

如果都想只做到1而不想做到2、3，那麼這樣的現狀大概不會消失。

故事展開話題

村東頭辦廠的老王嫁閨女，閨女從小貴族教育，才貌雙全，出國留學遇到個德才兼備，門當戶對的富二代，富二代給了百萬彩禮還給老王頭的宅基地翻蓋一番，變成了洋氣的帶車庫露台泳池的三層小洋樓，老王頭很滿意，給女兒隨了市內商品房豪華裝修，陪嫁了百萬級豪車，小兩口躺在蜜罐里，甜蜜的經營屬於他們的愛情，包工頭老李是老王的多年好友，一次去和老王把酒言歡時，隨口說到，你真是找了個好女婿，老王自得之餘不忘老祖宗教導的謙虛做人，給老李夾口菜，擺擺手說到：「還算那小子有點良心，這不又在市裡買套房，裝修花了百十來萬，年輕人真是不知道錢難賺，屎難吃，我也老了，由著他們嘚瑟去」，渾然不提裝修錢和陪嫁的車子是自己掏的腰包，也許這樣更能襯托女婿的優秀和女兒的魅力，老李頭聞言，不再啃聲，一句恭喜，一句好命，有一搭無一搭的直到飯局結束。

時隔兩年，老李頭自家的閨女到了談婚論嫁的年紀，談了一個男朋友，雖不算大富大貴，也配得上小康之家，父母都是普通公務員，在市裡有套房，有輛十來萬的車子，工作穩定有發展，老李頭雖感慨沒有老王家的女婿優秀，但自己也算生意人，這些年也攢下來一些錢，索性結婚時多貼補一些，不能以後和老王頭喝酒時弱了面子不是？

老李女兒的婚禮排場很大，各路朋友都來慶賀，酒過三巡，菜過五味，賓客散盡，只剩下幾個老朋友還在談笑風生

老王問到：「老李，咱這姑爺啥來頭，場面整得挺大啊」

老李聽到老王的這句話，頓時感到貼補的錢沒白花，雖然心裡隱隱作痛，但聽到老王的這一句話也值了

隨即答到：「那小子的父母是個局裡領導，和咱這糙人不一樣，具體管啥也不清楚，反正巴結他們人挺多的」

許是想到自己貼補出去的錢，心裡有些隱隱作痛，又惡趣的補充道：「雖然他家庭條件不錯，但追我女兒可是花盡了心思，三番五次上門，各種發誓保證，要不我才不把我女兒嫁給他呢」

說者無心，聽者有意，老丁，許是老哥幾個里混的最一般的，一般的工作，一般的收入，唯一讓他自豪的是自己培養出了一個和老王，老李同樣優秀的女兒，這也是他自豪的一點！然而今天這頓酒吃的有點窩心，因為他的女兒也到了要嫁人的年齡了

女兒的心儀對象，是她大學四年同窗，對她體貼備至，但由於剛畢業，工作不錯，前景可期，但收入一般，男孩家裡也和老丁一樣，是個踏踏實實的本分人家，談婚論嫁時，老兩口在市裡給孩子付了首付，孩子自己還房貸自己出錢裝修，婚禮也打算從簡，畢竟都是場面上的事情，省點錢給小兩口過日子才是真。

然而這一切，在老李女兒婚禮後改變了，老丁生氣，為什麼我的女婿比他們兩家的差？一個給蓋房，一個全款買房？而我的女兒還要跟著還貸？是我女兒的問題嗎？不是！我的女兒不比他們的差，那就是我女婿的問題，老王和老李都說了，他們的姑爺，腰纏萬貫，德才兼備，還上趕著追求他們的女兒！我這未來女婿，連喝酒都不會，傻不拉幾一點眼力見沒有，我要讓他全款買房，不行就給個百八十萬的彩禮，實在不行這門婚事就做吹！我女兒這麼優秀，隨便找個男人都比現在的強！

前面的三個故事，是我感受到的現在婚姻的縮影，人們習慣記住自己在意的事情，而對自己的缺點選擇性忘記，這才是他們痛苦的根源，有些時候作為「上位者」無意透露出的一些優越感，就會給他們無比沉重的心裡負擔，看不清自己的人，往往不具備解決問題的能力，無能卻不能無欲，想得而不可得，事情最後就會變得歇斯底里，無可理喻，巧取豪奪，棒打鴛鴦。

單純的批判並沒有意義，所有的結果都只是一個又一個原因導致的，不去改因，只去批判果，聲音再大也是沒用的。

這世上除了自己孩子不能交易外，其他的都能交易。

隨著我國市場經濟的進一步發展，市場的理念進一步深入人心，婚姻作為影響個人命運的重大決定，一定會被市場經濟重新定義，婚姻一定會被買賣，這其中唯一的區別就是市場化程度的高低而已。所謂的愛情，這不過是買賣的遮羞布，讓婚姻顯得不那麼赤裸裸。

很多時候，當男的結婚時在女方不出錢的情況下，直接讓對方加了名，又沒有共同還貸，即使當初女方沒有騙婚的想法，隨著婚後生活矛盾的進一步的爆發，女方會本能的選擇對自己有利的選擇，跟男方離婚，拿走房產的一半，造成實質上的騙婚……

人生就是如此，要直面現實，而不是自欺欺人，跟鴕鳥一樣，把頭埋在沙子里，寄希望於問題自動消失……
