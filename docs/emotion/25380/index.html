微信心情短語：原本安於位卑言輕，因他卻生窺神之心

2019-12-20 1,285條評論 2,612次閱讀 885人點贊

我的心是綿密的浪，在你的歌中湧向了海岸。
我的心是曠野的鳥，在你的眼裡找到了天空。
好看的模樣，清澈的笑，暖心的嗓音，
動人的旋律，善良的心，落寞的背影，
我總是靜靜地感受著你的一切，
在你的陪伴下接受孤獨、接受喜歡、接受命運。
感謝有你，感謝有你們。我們在各自的命途中，願安好永遠。

關於什麼時候可以結婚，網上有句話可以參考下：「我認為，要達到能夠結婚的狀態，需要兩個人都達到自我內心的安定：對新鮮的異性不再蠢蠢欲動，對生活的前路不迷茫，已經體驗或經歷過的感情抑或其他事達到自己預計的分量，決心靠譜而安心過日子了。這才是可以結婚的時刻。

當時是初中吧，唱歌還挺好聽的，畢業的時候我和一個女生唱了這首歌，然後呢，高中的她和我喜歡的人在一起了，我也談了一場有始無終的愛情，記得那時流過的眼淚，在喜歡的人面前義無反顧的離開原來真是最痛苦的事，現在了，上了大學，再次聽到這首歌，還是，很痛呀…

網上看到一段話「其實我不用看到你的臉才能喜歡你，也不用聽到你的聲音，不用知道你有什麼消息。我所知道的是，在生活中不多見的閃光的時刻我會想到你，煙花升起來，雪落下去，人們圍上來擁抱我，這樣的時刻。我猜你和世界上所有的美好連在一起。所以我想盡量去看一些好的東西，這就是我喜歡你的方式」

看到網易很多歌的評論都是關於愛情的，我想提醒自己，愛情不是生活的全部，我還年輕，還要向前看，要有追求，成為更好的自己，才值得更好的人。

學友唱「就算有天我吻別人亦當親你」 奕迅講「從背後抱你的時候期待的卻是她的笑容」 嘉欣說「和誰在一起都用來共你相比」——每個人經歷過的感情都是不一樣的，卻有著類似的感慨~

復讀那年 錯過了你的告白 才發現自己已經陷進你的溫柔 而你有了另一個她 我們卻還是朋友 下晚自習的路上 我們一起回家 這首歌一直循環著 你問我這首歌叫什麼 我都笑笑不告訴你 在你面前我甚至連一句不快樂都無法說出口

最近一直在追一個女孩，無論怎麼冷淡，我都在找話題，當問到你喜歡馬頔，陳粒這些民謠嗎？當她來了句：不喜歡，哈哈。我立馬接了句：我也是，一點都不喜歡。沉默了數分鐘後，我告訴她：我坦白，我很喜歡這些。然後我就很累了，不想再追了，給她說了句晚安，我說一切就這樣吧，這個還不是你要的
