1、不要忘掉別人生氣時說的話,往往那才是真相!

2、不知道我放在你那裡的心是不是也落滿灰塵無人問津。

3、低吟、淺唱，那一段不為人知的傷，沒落繁華的過往是誰的歌聲在回蕩。

4、擠不進去的世界，就不要擠了，礙了別人的眼，累了自己的心。

5、離去，讓事情變得簡單，人們變得善良，像個孩子一樣，我們重新開始。

6、明明說著看開了，卻總是在微笑沉醉時輸給了現實.

7、與其慢慢讓愛變質，還不如趁愛完好時製作成標本，放在心裡憑弔。

8、一起傻笑，一起開心，那樣的日子雖然不多，但卻讓我很想念。

9、剛開始會難過吧，只不過成年人的世界裡，除了父母，不會特別需要誰，離了誰，生活依舊在繼續。

10、不善表達的人真的很吃虧，付出多，用情深，卻沒一個人說你好。

11、相遇是兩個人的巧合,離開卻是一個人的選擇。

12、多年以後的某個深夜你會不會突然想起我，想起我掏心掏肺付出過那麼多，然後淚如泉湧，才發現你曾虧欠我太多。

13、你不去對一個人死纏爛打的時候你就是最迷人的。

14、遇到你，是我的緣;愛上你，是我的劫。

15、被我喜歡著 你也很痛苦吧

16、熬過這一劫，以後誰給的糖我都不要了。

17、感情都是一點點淡的，人心也都是一點點涼的，不是所有的人都能一直等待，也不是所有的愛都能失而再來。

18、有時候我真想忘了你，只記得這個世界，然而，我常常忘了整個世界，只記得你。

19、今生我什麼都放棄了，唯獨沒有放棄你。今生你什麼都沒放棄，惟獨放棄了我。

20、如果可以，就讓我們只做半生不熟的好友，這樣你便捨得一直溫柔待我，而我也捨得一直對你溫柔。因為我們不熟，所以沒有放肆的理由。
