費了九牛二虎之力，讓女兒考入985高校，畢業之後卻做了全職媽媽

2019-12-18 688條評論 4,134次閱讀 509人點贊

最近姑姑跟我訴苦：姑姑家的堂妹生了二胎，姑姑的婆婆癱了，為了照顧老人，不能幫妹妹帶孩子。妹妹自己的婆婆腰椎盤突出疼得厲害，說是抱不了孩子。妹妹沒有辦法，只能辭職在家全職帶娃。

姑姑很是忿忿不平，當年妹妹小的時候，她可沒少費心。

姑姑算是最早的那批虎媽的擁躉，妹妹從4歲就學彈鋼琴，姑姑總是拿著雞毛撣子坐在一旁，陪著妹妹練琴。她總是能無視妹妹的哭鬧，一直到妹妹考過10級為止。

雖然妹妹自從考過以後，就沒有怎麼彈過琴了。但姑姑總是說，會彈琴的女孩子氣質都不同。

妹妹高二的時候逆反，逃課跟同學逛街，看電影，姑姑得到消息，把妹妹抓回來，一把眼淚一把鼻涕，苦口婆心地勸，母女倆一直說到抱頭痛哭，一直到妹妹答應從今以後發奮讀書，絕不逃課為止。

妹妹高三第一次月考成績不理想，姑姑拿了四個月的工資，一次交了1萬補習費，讓妹妹1對1地補課。妹妹高考前，姑姑焦慮地不行，成晚失眠。還要一大早起來給妹妹做早飯，不敢多說也不敢不說。

妹妹高考完，姑姑大病一場。姑姑說，這輩子沒這麼緊張過，病都不敢病，總算可以鬆口氣了。

好在妹妹考得不錯，考上了武漢大學。姑姑十幾年來吊著的那顆心才算落進了肚子里。妹妹讀大學、工作、談戀愛、結婚一直都挺順利的。姑姑一直十分引以為豪，覺得都是她從小抓得緊的功勞。

可是沒想到現在妹妹快要35了，卻被迫成了全職媽媽，姑姑十分想不通。難道她當年那樣費心費力地教育孩子，就是為了她當全職媽媽？

「當年送你妹去上大學，你不知道我有多開心。我一直就覺得女人應該靠自己。十年前年你嬸嬸不工作在家帶孩子，我還笑話過她呢！」

高學歷女性因為家庭原因全職在家的情況不在少數。我們不能也不應該用過去看待家庭婦女的老眼光來看待她們。我能理解姑姑的失落，可是高學歷女性全職在家帶娃，並不是一件不可接受的事情。

首先，高學歷女性學習能力更強，帶出來的孩子更棒。現在都講究科學育兒，高學歷女性通過學習各種育兒理念，肯定要比老人帶孩子更靠譜。她們往往更注重孩子的智力發育，願意帶孩子探索世界，這關係到下一代的素質提高，所以未嘗不是一件好事。

其次，高學歷女性等孩子上幼兒園後更好找工作。我自己也曾經歷過一段全職帶娃的經歷。當時孩子小也沒急著找工作。後來因為兼職幫一家公司運營公號，對方直接詢問我願不願意去上班。我自然答應了，連求職投簡歷這些都沒經歷。

妹妹以前有過8年外貿經驗，在家安心把二胎帶到3歲，讓婆婆來接送上幼兒園，又是一條職場女漢子。歐美國家的女性生育後都有三年帶薪或留崗假期。就當跟國際接軌了，有何而不可。

反倒是學歷比較低，又沒有過硬工作經驗的女性，如果一旦在家全職，再要出山可能性就很低了。可以選的只有超市收銀員之類辛苦的活。

第三，高學歷女性在家也能賺錢。還是以我自己為例，當時在家時間長了，就開始琢磨事干。靠寫作也賺了一些錢，加加減減也不會比上班少很多。妹妹英語過硬，可以做做翻譯什麼的。等身體恢復了，寶寶大一點，還可以周末給小朋友補習英語，做做家教。

這是一個全民兼職的年代，誰說全職媽媽不可以有兼職收入了？

前三點姑姑都很認可，慢慢地也不那麼焦慮了。最後一點，我沒有跟她講。但是我願意跟手機前的你討論討論。

學歷是什麼？

姑姑那一輩的人，大多覺得學歷就是一塊敲門磚，有了這個磚就能找到好工作。但我覺得學歷對一個人最重要的影響是改變他看事物的方式。一個高學歷的人在任何時候都應該擁有冷靜思考，仔細分析，以及重新學習的能力。

這才應該是學歷留給我們最大的紅利。

說到底，對女性來說，如果有的選，願意全職全職，願意上班上班才是最理想的狀態。但現實往往不允許。那麼只有接納現實，改變自己，才是我們唯一的出路。
