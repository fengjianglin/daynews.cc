“我辛辛苦苦掙錢，就為了讓兒子放心啃老”這位爸爸的話引人深思

天天要聞
2019-12-21

文|文兒

現在啃老這個詞越來越被大眾提起，而且還是帶著貶低和不恥。啃老的概念就是長大不成人，說的是大部分孩子都已經畢業，有了謀生的能力，但仍未斷奶，需要靠爸媽供養。如果說起誰家孩子啃老，大家都會紛紛同情這家爸媽，一輩子不得安生。然後各種吐糟孩子，打心眼裡覺得孩子沒本事，不爭氣，拖累爸媽。養兒育女的爸媽，更擔心自家的孩子以後成為啃老一族。

前幾天和一位爸爸聊天的的時候，說起啃老這個話題，問擔不擔心自己孩子以後會啃老。這位爸爸很爽朗地說：「我辛辛苦苦掙錢，就為了讓兒子放心啃老」然後這位爸爸說了一席話，讓我對啃老有了更多的考慮。

前人栽樹後人乘涼，財富就是一代代積累起來的

往大了說，明明到了我這一代已經通過我的努力，爬到了半山坡。下一代踩著我的肩膀，再加把勁就可以到達山頂。難道我還要讓下一代再從山溝里爬起，重複我那幾十年的生活，那我辛辛苦苦那幾十年又有何意思呢，這樣無意義的重複下去，我們祖祖輩輩永遠在山底下，看不到山頂上的風景。

前人栽樹後人乘涼，家族之間就是個傳承。之所以讓孩子啃老，就是希望不辜負我這打拚的幾十年，我已經建好了地基，我希望我的兒子能站在我建好的地基上，去繼續開拓，整合利用資源，建好完全的房子，實現家族財富的最大化，財富是一代代傳承和積累的。

這個社會上單打獨鬥，白手起家的人又少又難，富二代永遠都比一般人有更好的資源和途徑。現在，之所以不讓啃老，很多時候恰恰是爸媽沒本事的借口，他們沒有條件讓孩子去啃老，所以孩子只能從底層一步步做起，走更多的彎路。很多時候，每一代人都努力一些，讓下一代有老可啃，家族才能長盛不衰。所以我會告訴兒子，你一定要啃我的老，而且要好好啃，我辛苦打拚了幾十年，就是為了給你一個更好的起點。對於有志向，有想法的孩子，能啃老是福氣和資本。

爸媽永遠是孩子的底氣和後盾

往小了說，爸媽也是從孩子走過來的，知道一路走出來有多難，所以才知道打拚的時候有一個堅定不移的後盾有多難得。所以我辛辛苦苦掙錢，就是為了讓兒子放心啃老。就是用實力告訴孩子，當爸媽的永遠是你堅強的後盾，永遠給你托著底，做好你的後勤保障，你可以安心地深造打拚，遇到挫折了，想想還有爸媽站在身後，那麼你一定有信心和容易再度起航，等孩子真正翱翔了，那麼孩子以前的啃老就是家長對孩子的投資，孩子以後一定會回報爸媽更多。

所以如果孩子在進步，在成長，那就不能算是啃老。真正的啃老是自己貪圖享受，停滯不前，消耗著爸媽的血汗錢，揮霍著爸媽辛苦打拚下來的基業。這樣的啃老，不管什麼時候都是讓人所不恥的。

​爸媽讓啃老，但我們要知道什麼時候不啃

爸媽讓我們啃老，是對我們的愛和給的底氣。爸媽會告訴我們，放心啃老，有我們，但我們要知道獨立永遠是我們自己選擇，我們可以選擇什麼時候不啃。任何人初入社會，都有過渡期，這個時候啃老是去投資自己，減少時間和機會成本，然後更早的讓自己獨立起來。
