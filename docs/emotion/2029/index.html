不要高估了人性的善，也不要低估了人性的惡，善良有尺忍讓有度

2019-11-24 0條評論 21次閱讀 0人點贊

摘要

善良是一種美德，忍讓是一種修養；但不是每個人都值得善良對待，也不是所有人都值得被忍讓。做人，一定要善良，有的人雖然能力強但是品行不端，困難時，有幾個真心愿意幫忙的人？做人，要懂得忍讓，忍一時風平浪靜，讓一步海闊天空，忍讓不是懦弱，而是一種大智慧。

善良是一種美德，忍讓是一種修養；但不是每個人都值得善良對待，也不是所有人都值得被忍讓。

做人，一定要善良，有的人雖然能力強但是品行不端，困難時，有幾個真心愿意幫忙的人？做人，要懂得忍讓，忍一時風平浪靜，讓一步海闊天空，忍讓不是懦弱，而是一種大智慧。

有些人，心太善良，見不得誰流眼淚；別人說上幾句好話，就掏心掏肺的對他好，但總是好心沒好報。所以請為你的善良加上一把標尺，忍讓也要懂得節制，千萬別把好心，給錯人。

無原則的善良，會被有心人利用，善良要分人而為，懂你的人知道你是好心，不懂你的人只會覺得你做作。總會有人，利用你的善良傷害你，最終受傷的還是自己，所以，你的善良要經得起人心的複雜。

古時就有「讓他三尺又何妨」的箴言，從古至今，忍讓都是人們推崇的好品質，但並不是每一件事都值得你忍讓。人總是挑軟柿子捏，事事都太大度和寬容，別人不會感激你，反而覺得你太懦弱，人應該適當的有點脾氣，才不會被欺辱。

如今社會，佛口蛇心的人太多，千萬不要高估了人性的善，也不要低估了人性的惡，別總是天真的認為自己身邊全是好人，喝酒不要超過六分醉，說話懂得留三分餘地，別對誰都掏心掏肺，毫無戒備。

對待惡人，如果一味的善良，軟弱的忍讓，只會助紂為虐，自討苦吃；你的善良和忍讓，要給那些懂得感恩，知道進退的人。

善良過了度，容易被欺騙；忍讓沒底線，就會被傷害。對你真心的人，千萬別辜負，那些騙你的人，及時與他分道揚鑣，記得做到善良有尺，忍讓有度。
