爸媽催生的正確打開方式：一對高知夫婦的精彩應答，太值得學習了

2019-12-24 1,576條評論 3,937次閱讀 768人點贊

這是年輪故事計劃的第155篇原創文章

關注私信回復「福利」，領取免費禮包

紀伯倫曾寫過這樣一段話：你的孩子，其實不是你的孩子，他們屬於生命對自身的渴望，而成為你的兒女。

兒女作為獨立的個體，在重要的人生節點上，做為長輩需要為他們做些什麼呢？

在鑼鼓喧天、張燈結綵、熱鬧的婚禮之後，所有的長輩都會長長地鬆一口氣：孩子終於結婚了，再就等著生個大胖孫子，幫他們帶大，我的任務就徹底完成啰！

可是不少走進婚姻里不久的年輕人，有的想趁著沒有孩子牽絆的時候發展事業，也有的覺得先清閑幾年，享受幾年二人世界的美好；還有的覺得自己的心智還需要再成熟一些，才能把孩子培養好；有的甚至奉行丁克主義，不想生孩子。

針對小輩們不想過早要孩子的現狀，不少家長想趁著自己年齡不算太大、身體還硬朗的時候，早點享受天倫之樂，把孫子帶大。於是他們的碎碎念從之前的催婚轉移到了催生，緊盯著新娘的肚子。

特別是家裡有超過二十七八歲的適齡產婦，幾次直接了當地溝通不奏效後，催生的方式也層出不窮：

今天在兒女面前叨咕「誰誰家的媳婦比你還小兩歲，前幾天生了七斤的小閨女」；

明天又說「過節了，誰誰家的帶著孫子回家看奶奶了」；

要不就是來個善意的提醒：「高齡產婦生孩子時可是有危險的哦」；

甚至還有的長輩重男輕女，生了孫女心有遺憾，硬逼著小輩生二胎、三胎，直到生了孫子為止……

直催得小輩們談生色變，令人啼笑皆非，甚至為此產生了感情矛盾，影響到了家庭幸福。

生不生孩子？什麼時候生？生幾個？不少家庭針對這個人生大計，進行積極溝通，隨時調整心態，達成共識，取得了皆大歡喜的效果，值得大家學習和效仿。

62歲的表姐退休前是一名產科醫生，姐夫是檢察院的檢察長，在他們的培養教育下，外甥女胡麗麗聰慧伶俐，知書達禮，到紐西蘭留學、工作，還帶回來一個金髮碧眼的英國帥小伙兒戴緯回來完婚。

他倆工作出色、婚姻美滿，親戚鄰居羨慕地說：「到時候麗麗再生個中英混血兒，那才漂亮聰明呢。」表姐和表姐夫聽了心裡美滋滋的，簡直比吃了蜜還甜。

可是結婚六年了，還是等不來麗麗懷孕的消息，這可急壞了表姐兩口子，特別是姐夫，一下子從繁忙的工作崗位上退下來閑賦下來，還真是難以適應。

他們多想早點兒抱上外孫，享受天倫之樂，填補退休後的失落呀。

可是一提生孩子，麗麗就顧左右而言他，為此，表姐和姐夫曾專門飛往紐西蘭，以不破樓蘭誓不還的決心，前去催生。

見面後的促膝長談，拉近了兩代人之間的親情，麗麗和戴維都是從事人類學和環境保護工程方面的工作，都在努力做一個對人類、對社會有責任心的人。

麗麗和女婿表示，關於生孩子這個事，在財力、精力、還有心理準備好了的情況下，會考慮的，一旦決定要寶寶了，他們盡量自己帶孩子，全力以赴把下一代也培養成一個對社會有用的人。

同時，麗麗也希望自己的父母，也要向國外的父母一樣，改變中國父輩一味地為下一代付出的做法，退休後享受應該屬於自己的美好時光。

推心置腹的話語，使表姐和姐夫也感慨頗多，他們認識到：兩代人之間只有相互尊重對方的選擇，才是處理觀念衝突的最佳方案。

共識達成了，表姐全家恢復了之前的其樂融融的常態，雖然在紐西蘭表姐兩口子也參加當地華人社團的一些活動，並且那裡風景優美、氣候宜人，但終究不如在國內環境熟悉、老朋友多，表姐和姐夫決定還是回國生活。

回國後，他們馬上參加了當地樊登年輪學堂，選擇學習感興趣的課程，還經常參加社會公益活動，到敬老院、社區義務演出，增長了知識，結交了新朋友，臉上成天洋溢著笑容，再也不為麗麗生孩子的事憂慮了。

對了，他們最重要的一項活動是參加了英語班學習，把以前學過、現在忘得差不多的英語拾掇起來，表姐說：「萬一哪天有了外孫，麗麗需要我們去帶孩子，出門買菜至少能進行簡單的交流，時刻準備著呢。」

正說著，姐夫不失時機地唱起了「時刻準備著，時刻準備著….」

那鏗鏘有力的歌聲，配著兒童團特有的動作，引得我們哈哈大笑。

對於婚後年輕人生不生孩子，生幾個，什麼時候生，這些問題我認為，首先我們要尊重年輕人，不過分干涉，只提供中肯的建議。

如果年輕人選擇做「丁克」家庭，建議他們一定從心理上做好準備，思想有了新變化，也可以隨時調整原來計劃。

如果準備要寶寶，做為長輩子我們應該給年輕人三點建議：

1、經濟條件不允許的時候不能要孩子。

工作不穩定，收入不穩定，溫飽解決不了，無法給孩子提供最基本的生活保障，更別提以後的教育投資了。

今年曾上演的黎巴嫩電影《何以為家》，講述了一對夫婦家徒四壁卻生了一堆孩子，飢餓、愚昧使他們12歲的兒子將這對夫婦告上了法庭，理由是：為什麼生我？讓我對童年的回憶只有暴力和辱罵？他的句句控訴引起了全球人的反思，我們從中認識到了什麼？

2、婚姻不穩固的家庭暫時不能要孩子。

排行榜123上一個觸目驚心的數據雷到了我：2019年第一季度，全國登記離婚的人數為104.84萬對，同比增長7.8%。

從這個數據可以看出，近年來，離婚人數逐年攀升，透過冷冰冰的數據，多少個孩子彷徨無措的面龐浮現在我們面前！

現在年輕人閃婚閃離的現象時有發生，即便結了婚，沒有三年四年的性格磨合，也有可能導致婚姻的破裂，孩子就成了最大的受害者。

前幾天，我法院的朋友和我聊天，談起有一對農村夫妻認識不久就結婚了，婚後不斷地爭吵，最後法院判決離婚。

從法庭出來，這對年輕的父母搶著騎著各自的電動車分道揚鑣，生怕四歲的女孩成為他們的累贅，孩子哭著追去，最後摔倒在法院門口，在場的人無不心酸，幾個女同志都流下了眼淚。

3、經濟條件達到了、婚姻也穩固了，心理上沒準備好也不能要孩子。

近幾年的教育環境和早些年有了很大差別，望子成龍，信奉棍棒教育的大有人在，《刑事法律圈》里報道，今年1月3日陝西省咸陽市武功縣一名家長在輔導兒子學習，因為嫌棄兒子不用心而進行毆打，致使孩子頭部受傷，嘔吐不止，經搶救無效死亡。

作為母親雖然後悔不迭，也難逃法律的制裁，最後法院以過失致人死亡罪，判處母親有期徒刑三年，緩期三年執行。

可見，沒有正確教育孩子、引導孩子的能力，同樣沒有資格生孩子。

龍應台曾經在《親愛的安德烈》一書中寫到：「孩子，是一個獨立的個體，他/她不是父母的，是會慢慢成長為別人，跟父母有血緣關係的別人。 」

既然是孩子是「別人」，當孩子已經成熟，做為長輩，我們在祝福他們的同時，就應該尊重「別人」，而不是橫加干涉，在「別人」重要的人生節點上，適時地提提建議，把把關，如此，便好。
