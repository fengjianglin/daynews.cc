“我真的做不到”，孩子遇到困難只會“自我否定”，別再讓他迷路

天天要聞
2019-12-19

本文由Ainee媽咪原創，歡迎個人轉載分享

如今很多成年人都會抱怨現在壓力太大，在這飛速發展的時代，各行各業競爭力都非常激烈。其實不只是成年人，孩子也是一樣。

現在的孩子們之間的競爭十分的激烈。有的孩子在很小的年紀里就已經有了多種才藝，有的孩子卻連一種才藝都沒有，這些沒有才藝的孩子們在很多時候都會陷入深深的自我否定中。

因為孩子的壓力特別大，這種壓力甚至把很多孩子都會壓垮，從而孩子會出現消極心理，產生嚴重的自我否定。

「我真的不行，算了吧」

蘭蘭家的女兒今年已經是上小學的年紀了，但是最近她發現孩子經常進行自我否定。他們家的孩子由於畫畫能力很棒所以進入了一所專攻畫畫這門學科的興趣班。

本來他們以為女兒在那裡學習會很開心，因為這是她所熱愛的事物。沒想到的是，孩子進入繪畫班之後竟然比之前更加的沒有自信了。

因為那裡的孩子的家庭都是很富裕的，所以她的同學們大多數都是多才多藝的。孩子跟他們一比之下就顯得十分的普通，因此孩子總是認為進行自我否定，認為自己是不行的。

這不僅讓孩子的自信心受到了一定程度的打擊，還讓孩子的畫畫能力在逐漸的下降，因為孩子總覺得自己的畫畫能力比起其他人來說很不好，認為自己不行，做什麼事都想要放棄。

當孩子的班主任推薦孩子去參加一個比賽，孩子獲得了很優秀的成績的時候，她不會覺得這是由於自己的努力得來的成功，她覺得這是因為班裡的其他同學都太忙了，沒有空去參加，自己才會獲獎的。

孩子經常在進行自我否定這種環境下生活的話是不利於孩子們的健康成長的。

其實不只是他們家的孩子，現在許多孩子們對自己的要求都十分的高，當他們做不到的時候，他們都會進行自我否定，認為自己是真的不行，想要放棄。孩子們的這種行為被定義為自我否定傾向。

為什麼孩子經常出現「自我否定」

1. 壓力過大

當孩子們遇到全新的挑戰的時候，他們會十分的擔心自己的能力能否很好的完成這項任務，他們在做這件事情的過程中總是會產生很大的壓力。

當他們的壓力積攢到一定的程度的時候，孩子們就會出現拖延行為或者是過度行為，這對於他們任務的完成來說並沒有積極的效果。

雖然當他們成功的完成一項任務的時候周圍人的讚賞會給孩子們帶來短暫的愉悅心情。

但是這種促使他們進行自我肯定的愉悅心情會很快就消失的，孩子們在下一次挑戰來臨的時候，因為壓力過大，又會再一次的陷入自我否定的這個情感漩渦之中。

2. 連續的失敗

孩子容易自我否定，其中有一部分原因就是因為孩子自己連續的失敗。

因為孩子在不斷的嘗試中，結果都不盡人意，所以這時孩子就會產生悲觀的情緒，認為自己做什麼都不行。

外加上有很多家長都忽視了孩子的感受，也沒有對孩子進行一些鼓勵，所以導致孩子內心更產生了自我挫敗感，認為自己幹什麼都不行。

連續的失敗給孩子的打擊是很大的，如果我們家長不重視孩子內心存在問題的話，終將會造成孩子「自我否定」的情況發生。

3. 對未來沒有「憧憬」

還有一部分孩子經常出現自我否定的原因，就是對未來沒有任何「憧憬」。換句話說就是孩子自身沒有希望感，喜歡平平淡淡的生活，喜歡與世無爭的狀態。

孩子自己內心沒有那股衝勁，那麼孩子自然也不會為了一些事情而做出努力，沒有動力的話孩子就會經常的自我否定，會失敗。

所以這就提醒了我們家長一定要給孩子一些讓孩子可以繼續努力的緣由，只有這樣孩子才能擺脫對自己的「自我否定」。

長期的「自我否定」，對孩子會產生哪些消極影響？

1. 對生活的「無力感」

孩子自身長期的自我否定，會讓孩子對生活有一種「無力感」。

因為孩子不想為任何事情而努力，在他們心中他們已經認定自己終究會失敗，所以他們就會認為自己沒有必要做這件事情，從而對這件事力不從心而產生無力感，這是一種消極的情緒。

很多時候在做事情的時候，人終究會有失敗的，但是孩子會把這失敗的一面放大，而導致孩子對之後的事情沒有信心，從而生活中的無力感也隨之而來。

2. 性格日漸孤僻

長期的自我否定也會是孩子的性格日漸孤僻。因為孩子內心是自卑的，由於自己的失敗，和其他人的成功產生了鮮明的對比，從而孩子自己內心形成一種自卑感，逐漸也變得性格慢慢孤僻，認為沒有人能肯定他的努力。

長此以往下去孩子身邊就會沒有朋友也會變得越來越孤僻。

3. 失去更多的朋友和機會

當孩子的性格變得孤僻時，那麼給孩子帶來另一個影響就是失去更多的朋友與機會。

孩子變得自卑，性格孤僻，身邊的朋友也不會像以往對他一樣熱情，而他也不能像以往積極回應，從而導致孩子的身邊朋友一個個減少。

並且孩子如果長期的自我否定，他同樣也會喪失很多機會，比如說學校參加的很多活動，孩子自己本身是十分優秀的，老師也認為他可以，但孩子自己內心會認為自己不行，從而不去參加活動，這樣孩子優秀的表現就不會得不到其他人的發現與肯定。

家長如何做好引路燈，防止孩子迷路？

1. 正能量的傳遞

經常自我否定的孩子們總是會帶有一些負面的情緒，他們長時間把這些壓抑的情緒積壓在內心的話會造成一定程度的身心健康問題。

因此我們家長要幫助孩子通過正確的方式幫助孩子們增強內心的正能量，讓孩子們知道自己的能力是能夠達到這樣的成功的。

當孩子們擁有正能量的時候，他們就不會在自己獲得成功的時候他們就會擁有真實感了，他們的成就感會得到增強也會為自己能夠做到這樣的事情而感到自豪。

2. 越失敗越鼓勵

父母要盡量給與孩子積極鼓勵，這樣孩子才有更大可能不會「自我否定」自己。允許孩子做得不好，降低對孩子的期望值，越失敗越要鼓勵。

其實，孩子的好勝心也是非常強的。但越是好勝心強的孩子，父母越是要關注他們的失敗。有時候，允許孩子做得不好更利於孩子的成長。

比如：孩子學打籃球，總會有進不了球的時候，這時，父母不妨以平常心來對待，告訴孩子，做不好很正常。這樣父母才不會把過多的壓力加重在孩子身上，從而讓孩子因為壓力而變得不堪重負。

允許孩子做得不好，其實也是在降低對孩子的期望值，只有期望值降低了，父母才能以更平和的態度對待孩子，這可以避免孩子對自我產生懷疑和不自信。

3. 陪伴就是溫暖

我們家長要做好引路燈，防止孩子自我否定，我們能做的還有就是陪伴在孩子的身邊，因為陪伴是最長情的告白，我們家長如果能夠陪伴的孩子的身邊就能給孩子帶來溫暖，同時也能鼓勵孩子走出這一段的迷茫期。

家長的話對於孩子來說是最有用的，而且如果我們家長能夠經常鼓勵孩子陪伴孩子的話，孩子也會自己增強自身的自信心。

當孩子有了自信心時，那麼孩子也就不會再自我否定，也會發現自己的努力是能被別人認可，也是可以成功的。

所以這就提醒我們家長一定要多多陪伴孩子，孩子內心是敏感和脆弱的，如果我們能正確的加以引導，那麼對孩子未來的成長也是十分有益的。

對於父母來說，做好後天的引導，對孩子性格中獨立自我的養成是很有必要的。而對於孩子來說，一個好的後天環境，往往能讓他們找到真正的自我，從而變得更加自信。

作為家長要引導孩子跟自己的過去比，跟自己的過去比看進步，不要跟周圍同學比看差距。

因為和別人比，難免有偏見。稍有偏見，仍易回到消極的心理狀態。只跟自己的過去比，經常能看到成績，看到積極的方面，看到前途，有助於更好地發揮自己的長處，開發自己的潛能。
