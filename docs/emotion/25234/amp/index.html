成了啃老族，啃了7年老，才知道啃老有4大好處

天天要聞
2019-12-21

啃老族：已有謀生能力但仍然主要依靠父母養活的成年人。

「父母命，行勿懶。」父母老了。在父親的感召下，毫不猶豫辭掉工作，告別朋友，回到父母跟前，成了啃老族。開始了啃老生活。

開始兩年，每天柴米油鹽，吃喝拉撒。陪母親說說話，和父親溜溜腿。去趟銀行，買個葯。生活平淡如水，單調無味。

兩年後，衰老的原因，父親卧床了，話也不說了。生活起了波瀾。喂水喂飯，收拾屎尿，清洗身體搞衛生。和老母親琢磨父親，互相交流心得，每天忙且累著。不到三年，父親在一個深夜悄悄走了。

父親走後，日子恢復了平靜。除了提醒母親按時吃藥，就是聽母親的絮叨，絮叨和父親七十年的愛恨情愁。為了讓母親少幹活，多休息，偶爾和母親拌上幾句嘴。安靜、悠閑的日子過了兩年多。因為心衰，母親住進了ICU（重症監護室）。短短十天，母親也走了。

料理了後事，七年啃老的日子放電影似的歷歷在目，才知道啃老的好處是時間和金錢無法衡量的。

啃老的日子能聽到時間的腳步，讓你感悟生活。

沒有了朋友之間的交際，沒有了繁忙的工作，電話少了，朋友少了。

快節奏的熱鬧日子不再見，什麼都隨著老人的節奏慢了下來，簡單的只有吃喝拉撒睡。慢和簡單，讓人不得不安靜。數著錢過日子，發現生活也不象以前那樣費錢，以前的物慾好象太奢侈了點。物慾少了，心凈了。

一個安靜，洗盡了鉛華，看見了時間從自己身上流過；

一個乾淨，褪去了浮華，發現生活就是活著如此簡單。

活著就是吃喝拉撒睡。沒有了往日的焦慮，急躁和煩惱，擁有了一份恬靜和祥和。所有看見的都不再象以前那樣掃興，一切都顯得那樣美好。

啃老的日子讓你看見孤獨的美麗，學會孤獨。

開始啃老，左鄰右舍會誇你孝順，好孩子。日子長了，會說你混吃混喝，懶得不工作。舌頭低下壓死人。誇你，你不能飄；慫你，你不要怨。淡然處之，慢慢的所有評價都銷聲匿跡了。你還是你。

親戚、哥姐旁敲側擊的挖苦，含沙射影的埋汰，講理嗎？沒地方講。吵架嗎？耽誤幹活。只能你說你的，我干我的。努力做到，他們會幹的，自己會幹；他們不會幹的，自己也會幹。活計都是你的。

日子一天天過著，朋友少了，圈子沒了。跟著老人過著老年生活，好像沒有了熱血和激情。能抱怨嗎？不能，因為這是自己的選擇。自作自受也是一種擔當。

父母偶爾的嘮叨，愛人忍不住的埋怨。裝傻充愣，自嘲一番，斗個開心，箇中滋味，只有自己知道。

不是自己不會說，是因為自己不願說；不是自己不敢鬧，是因為沒有必要鬧；不是解釋不清，是因為清者自清；不是自己沒怨恨，是因為自己看得開。心開了，心大了，容下了一個世界。

啃老能讓自己預見將來的自己，規劃以後的人生。

啃老讓你天天看得見，摸得著，感覺到衰老；啃老讓你領略生命的堅強和脆弱，堅強的讓你震撼，脆弱的讓你心驚。父母言傳身教，給你講人生最後一課。讓你面對衰老，不會恐慌；面對死亡，不再陌生。

「父母去，人生只剩歸途。」因為啃老，自己對歸途不再迷惘；因為啃老，自己知道歸途自有風景。因為啃老，讓你警醒，珍惜當下，把握當下，防微杜漸，歸途變的漫長又美麗。

啃老讓你感情豐滿，情感圓滿，此生無憾。

啃老能讓你重新認識父母，修正和父母的情感，升華和父母的感情。啃老能讓你感受不一樣的父愛、母愛，你還是孩子，父母還在養你。

啃老能讓你見識別樣的人情世故，領略別樣的世態炎涼。讓自己擁有淡定和從容。

啃老能讓你重塑自我，正視自己，完善自己。

父母給自己做過什麼？自己給父母做過什麼？養育之恩，何以為報？唯有啃老。「子欲養而親不待」人生憾事。唯有啃老讓你此生無憾。

四大好處金不換；啃老啃出新人生。
