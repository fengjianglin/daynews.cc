錯誤，可能只有一次，也是要命的

2019-12-24 731條評論 2,073次閱讀 655人點贊

阿元

先來看個故事。李太太有天晚上和一群人到歌廳唱歌，中間上廁所時路過一個包間，聽到裡面傳拉熟悉的聲音，儘管周圍的聲音很響，可她還是聽出來了，裡邊是她老公的聲音，好像喝了酒，大聲嚷嚷，「怎麼回事，不是說讓你們把你們這最漂亮的都找來嗎？還不快點？」

李女士的腦子嗡的一聲，想起老公說今天要同哥們出去吃飯，要晚點回家，怎麼跑這裡來了？

為了把事情搞清楚，李女士和同事們說家裡有事，先走了。李女士到了走廊上，拉著一個服務生，問他那個包間里的情況，他什麼都不肯說，於是李女士就往他手裡塞了2 0 0 塊錢。他告訴我裡面有5 、6 個男的，找了幾個小姐陪著唱歌。

李女士心情特別的不好，但還想看看接下來還有什麼，於是就在外面一個不顯眼的地方，等她老公出來。

也不知道過了多長時間，他老公和一些人出來了，還有幾個衣著暴露的女人，分別上了幾台計程車。李女士馬上也坐上了一台，讓司機跟著他坐的那台。看著老公在一家賓館門前下了車，扶著個女人就進去了。

李女士感覺自己頭腦從來沒那麼清醒過，她不想去看那些醜陋的場面，也沒有去制止的心情。這時給老公打電話，接不通。

李女士稀里糊塗的回到家裡，躺在床上眼淚控制不住的流下來，她沒有想到，那個口口聲聲說愛她的男人，就是這樣一個齷齪而骯髒的人。

不知道過了多久，聽到用鑰匙開門的聲音，李女士用被子擦擦眼淚裝作睡著了。她老公沒開燈，躡手躡腳的走進來，可能以為李女士睡著了，又走出去，直接去了書房，去上網了。

第二天早上，李女士把家裡存的錢，全拿了出來，存到了自己媽媽的名下。然後又把自己家的房子，轉到了自己的妹妹名下。

接下來，李女士把老公叫了出來，找了一個比較安靜的地方，問起了那天的事。但他就是低頭什麼也不說，李女士急了，把他去了什麼什麼地方，住的什麼什麼房間，都說了一遍。

李女士的丈夫馬上撲通跪下了，抱著她的腿請求原諒，說以後再也不去了。李女士哭著說，你做那樣的事時就沒想到過有今天？我還告訴你，錢已經不在我的帳戶上了，房子也要過戶到別人的名下。你要是覺得自己吃虧了，你可以想盡辦法去告我。他還跪著不起來，說他什麼都可以不要，但是不能失去我。我說你起來吧，沒用的。等房子的事辦好，你在離婚協議上簽字就好了。

李女士現在已經下定了決心，要離婚了。她說，男人都是容易輕易原諒自己的，覺得這樣的事不是什麼大事，但是這事要是發生在女人身上，試問有哪個男人可以原諒？

對於李女士把家裡的財產都給弄走了，是不是合法，這個是法院的事，咱們不是律師，沒法說這事的對錯。所以在這裡，阿元想說說李女士的老公。對於李女士的做法，有很多人不理解，說是不是太狠了啊？說如果男人犯一次錯誤就離婚，那麼天下應該沒有家庭和婚姻了。

還有的人說，李女士還是應該好好考慮一下，除非你不再結婚，不然保證不了下一個男人不犯同樣的錯誤。

對於這種說法，阿元雖然是男的，也不能同意。原因是有的錯誤，你一次也犯不得的。象歷史上最有名的壞男人，陳世美，不也就犯了一個錯誤嗎？結果是怎麼樣呢？他掉了腦袋，叫包公給鍘了。

還有一個英國的國王，領兵去打仗。出發之前，他讓鐵匠給他的釘馬掌。這馬掌上應該釘四個鐵釘的，可那個鐵匠，為了省點時間，只釘上了三個。

這國王到了戰場上，衝鋒陷陣的時候，他的馬掌就掉了，馬也摔倒了。結果被敵軍輕易而舉地俘虜了。當士兵看到國王從馬上摔下來被俘，於是便紛紛逃跑，使本來力量占絕對優勢的國王，不僅打了個大敗仗，還丟了整個國家。

就這件事，有人曾作出了這樣一個精闢的概括性總結：少了一個鐵釘，失了一個馬掌；少了一個馬掌，失了一匹戰馬；少了一匹戰馬，丟了一個國王；丟了一個國王，輸了一場戰爭；輸了一場戰爭，失了一個國家。

這樣的故事告訴我們的，人，有時候可能只犯一個錯誤，造成的可能是不可挽回的損失。

而且話說到李女士的老公身上，這樣的錯誤可能更加的不應該。為什麼這麼說呢？難道她老公是火星人嗎？不知道找小姐是一種錯誤的，讓人噁心的行為嗎？既然不是，那麼他的行為，實際上是故意的。一個故意去犯這樣錯誤的人，你認為他能愛老婆嗎？他能故意犯這麼噁心的錯誤，他還算是個人嗎？

更重要的是，愛情，或者說夫妻間的感情，那是相當的脆弱的東西。你好好愛護，兩口子都想好好過日子，還可能天天吵架呢，它怎麼架得住你這麼樣去禍害呢？

記得有人說過，愛情就象碗一樣。碗打了，再鋦起來，但那條裂痕，是會永遠存在的，它永遠不是一個完整的碗了。愛情，或者夫妻間的感情，也是一樣的。你把它打破了，那是用什麼膠水也粘不起來的。

錯誤，可能只有一次，也是要命的。
