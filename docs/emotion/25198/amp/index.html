二線城市工作的女青年，告訴你，我為什麼拚命掙錢

天天要聞
2019-12-21

前幾天寫過一些女孩子夢想嫁個好老公，就想在家做全職太太，衣食無憂，放棄努力。

可是並非所有的女孩子都有這樣的想法，北上廣深這些一線城市裡，或是我現在生活的這座二線城市中，奮鬥的女青年比比皆是。

她們或為升職加薪而拚命加班，或為自己的夢想風餐露宿，或為了自己的家庭吃苦忍耐……

有人說，女孩生得好，不如嫁得好，可是我想說，女孩嫁得好，不如自己拼的好。

一位二線城市工作的女青年，告訴你，我為什麼拚命掙錢？

努力的追趕父母老去的速度

以前都是被父母保護著，從未想過有一天自己要反過來守護他們。

或許你直到發現爸爸的頭上有了白頭髮，身體總是有一些小病小痛，才有一種父母漸漸老去的感觸。

我們會想著掙錢，能夠為父母做一些事情，帶他們去旅遊，讓他們享受外面的世界；帶他們去吃好吃的，那是他們從未接觸過的美味；想給他們買更好的營養品，好讓他們的身體更康健。

我們努力的去追趕父母老去的速度，不讓自己此生留有遺憾。

自己富足，才能勇敢去追自己想要的

如果我不拚命掙錢，一件漂亮的裙子，看看價格就要默默放下；如果我不拚命掙錢，口紅只能用廉價的，那些時尚品牌只能存在於傳說中；如果我不拚命掙錢，遇上了條件不錯的男孩，因為自卑可能就會錯失緣分。

拚命掙錢的女孩並非拜金，她們只是更知道，金錢能夠給自己底氣，也只有擁有足夠的底氣，才能毫無顧慮的去追求自己想要的。

有一場說走就走的旅行，有一筆可以隨意「揮霍」的金錢，有一份自身努力得到的不錯工作……這些就是那些女孩兒努力的意義。

想過好平凡的一生，真的要好努力

對待女孩子，也許生活中有很多人都會憐香惜玉，可是生活不會。

每個人初入社會，心中都懷揣著或大或小的夢想，可是工作幾年之後才會慢慢明白，一切似乎不是那麼回事兒。

夢想解決不了自己的吃飯問題，也解決不了住宿問題。自己當初心思單純，夢想設定的不切實際，假大空。

而現實是我們真的很平凡，平凡到你走進下班的人群中，瞬間就被淹沒。路上人們行色匆匆，自己是多麼容易被忽略的存在。

有多少人就是這樣放棄了自己的那顆追夢之心，放棄夢想的副作用，就是拚命掙錢。

夢想和金錢，我們總要擁有一個。

當然不能否認，有一部分幸運兒，他們同時擁有夢想和金錢，在逐夢的過程中，也能收穫財富。

可是大部分人即使只是想過好平凡的一生，也需要很努力才行。

既然沒有衣食無憂的命，就拚命掙錢吧。能夠及時孝敬父母，匹配上更好的生活，擁有足夠的底氣去爭取自己想要的一切。

當你努力向前奔跑時，一定能夠時常收穫不期而遇的糖，這是你應得的的獎勵。
