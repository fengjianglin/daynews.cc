她22歲嫁給英雄，卻用一生來等待愛情

天天要聞
3 週 ago

摘要

女人婚後通常有兩種狀態，要麼成為家庭主婦，相夫教子，柴米油鹽，把丈夫、兒女的人生當成自己的人生來活；要麼堅持自己的事業和熱愛，活成瀟洒、張揚、底氣十足的女王，主動掌控生活，而非被動地被生活掌控。在現代社會，很多女性慢慢將自己修鍊成了第二種人。

女人婚後通常有兩種狀態，要麼成為家庭主婦，相夫教子，柴米油鹽，把丈夫、兒女的人生當成自己的人生來活；

要麼堅持自己的事業和熱愛，活成瀟洒、張揚、底氣十足的女王，主動掌控生活，而非被動地被生活掌控。

在現代社會，很多女性慢慢將自己修鍊成了第二種人。她們任性、自信、氣場全開，活出了真實的自己。


但時光倒回去幾十年，中國式的婦女，絕大多數都是第一種人。

今天我們要講的這位中國式的傳統婦女，就是麥家《暗算》一書中的人物， 701破譯專家陳二湖的老婆。

麥家用寥寥數語，將一位女性艱辛不易的一生和盤托出，令人唏噓不已。

時代的厚重背景下，身為英雄背後的女人，她甚至沒有留下自己的名字。為了方便閱讀，我們後文將稱其為陳夫人。

革命時期的婚姻，沒有愛情，沒有被愛，常常是通過組織介紹，男女雙方見一面，覺得可以，就成了；覺得不可以，多見幾次，基本上也就成了。

陳二湖的單位性質特殊，701情報部門，對於找對象這種事兒，篩選極為嚴格。


介紹的女孩子，必須經過各種政治、社會、家庭、現實、歷史等等的審查，像給皇帝選妃一般隆重。

雙方第一次見面時，陳二湖窘迫極了。

「這是一個走出紅牆就不知所措的男人，他不是來自生活、來自人間，而是來自蒸餾器、來自世外、來自隱秘的角落，你把他放在正常的生活里，就如水裡的魚上了岸……」

但才過了一個月，兩人就結婚了。原因很簡單，只有六個字——服從組織分配。

不得不說，那個年代的婚姻，在上層的干涉下，的確成就了不少和諧的家庭，但因為將就，生生把兩個不熟悉的人湊在一起，也釀成了不少悲劇。

但那又怎麼樣呢？那是專屬於一個時代的烙印，身處其中的男男女女，被命運的紅線牽著走到一起，是不會發現有什麼不妥的。

《暗算》一書中阿炳和林小芳的結合，就是一種英雄主義式的悲劇，但雙方並未覺得有何不對。

黃依依倒是冒天下之大不韙，奮起反抗時代，偏要跟心愛的人結婚，奈何螳臂當車，終究是自取滅亡。

陳二湖夫妻的結合也是一樣，在時代的背景下，被安排得順理成章。正因為太順理成章，才造就了一個時代的悲劇。


要把千金嫁給一個大她快十歲的男人，當母親的肯定不願意。但陳夫人的父親卻無比贊同。因為他是個老紅軍，自小在黨的庇佑下長大。

出於一種報恩的心理，黨看中了她的女兒，要把她嫁給英雄，他自然求之不得。

就這樣，30多歲的陳二湖娶了年僅22歲的陳夫人，二人的結合與其說是因為愛情，不如說是革命需要，是作為一項政治任務來完成的。

當英雄的妻子看似光鮮，其實是特別苦的，因為她們不僅要活成外界期望的樣子，甚至不得不為了保持外界期望的樣子，把苦往肚子里咽，打落牙齒活血吞。

而更可怕的是，她們無比樂意地去維持這種期望，飲鴆止渴，甘之如飴。

陳夫人原本在省機關工作，嫁人之後，隨著丈夫調到了現在的單位，離家十幾里路遠。

她騎著自行車，一天來回兩趟，接送兩個孩子上下學，給她們做飯洗衣，十幾年如一日。

她用一個人瘦弱的肩膀撐起了一個家，既當爹又當媽。她是陳二湖最強大的後盾，讓他可以安心工作，其他事一概無需操心。

所以在孩子的眼裡，「父親對家裡的事情從來都是不管不顧的」。

陳二湖對國家、對黨、對組織可以說毫無虧欠，但對妻子、兒女，卻虧欠了太多太多。

他沒有盡到為人夫、為人父的一丁點責任，他工作忙，一個月才回家一次，還總是晚上回來第二天早上就走。

但這並不妨礙妻子對他的了解。某天，多日未歸的陳二湖突然回家，離家還有幾十米遠時，正帶孩子吃飯的妻子就察覺到了。

妻子告訴孩子，你們爸爸回來了，說著放下碗筷，進了廚房。等她端著洗臉水從廚房出來時，果然傳來了陳二湖沉重的腳步聲……

這一幕的描寫充滿了時代性，令人無比動容，相信很多家庭都會感覺似曾相識。

小時候，每次爸爸剛進樓道，媽媽就能聽出他回來了，正在放自行車，正在爬樓梯，馬上到家門口……

不得不說，麥家的文字寫出了時代的氣息，令人不自覺深陷其中，沒有絲毫違和。

電影《一出好戲》中說：等一個人就像買彩票，買一輩子彩票也未必會中獎，但卻能讓你這輩子有個盼頭。

對於陳夫人來說，等待就是她一生的希望。等待丈夫回家，等待兒女長大，而她，則在無窮無盡的等待中，慢慢老去。

陳二湖是個充滿威儀的家長，不苟言笑。他一回家，家裡的氣氛就會變得緊張兮兮，孩子們低聲下氣，生怕惹惱父親。

因為只要惹父親生氣了，不管三七二十一，母親是永遠站在父親那一邊的。

正如麥家所說：「女人和男人是不同的，女人的世界是男人。男人的世界是世界。」陳夫人的整個生命都是丈夫的，就像陳二湖把自己的一生都獻給了紅牆一樣，她也把自己的一生都獻給了迷醉在紅牆裡的丈夫。

在陳二湖眼裡，沒有任何事比得上紅牆裡的事重要。縱使妻子病得很嚴重，就要跟他訣別的時刻，他也從未完整地陪伴過妻子一天。

陳夫人是病死的。從某年春節開始，她便偶爾肚子疼，但她以為只是普通的胃病，一直拖著。疼了就喝碗糖開水，吞兩片鎮靜劑，過後照常騎車十幾公里去上班。

後來，胃痛越來越嚴重，常常疼得她昏迷不醒，虛汗直冒。但丈夫在單位，兒女不在身邊，加上她有心隱瞞，便一直沒人知道。

她把所有的體貼和懂事都給了丈夫、子女，卻從來沒有愛過自己。

她像一個孤軍奮戰的將軍，從頭到尾，都是一個人的兵荒馬亂。又像一個丟失了羊群和草地的牧人，一無所依，比荒原更孤獨，比大地更桀驁。

從這種意義上講，她跟陳二湖是一類人，都是一種不要自己的人。他們沉浸在各自的理想信念中，傾盡心血，縱使犧牲自己，也在所不惜。

陳夫人的病情是被女兒發現的，發現時已是肝癌晚期。發現的那天晚上，她「蹲在地上，頭靠在床沿上，因為痛苦而扭曲的臉上，流著兩串長長的淚水，蓬亂的頭髮像一團亂麻。」

見到女兒的瞬間，她心底的防線徹底崩潰，像個孩子似的哭起來，嗚咽著說自己不行了，要去醫院。

「她佝僂的身體像遭霜打過的菜葉一樣蔫巴巴的，在昏暗的燈光下，就像一團揉皺的衣服。」

這一部分的描寫，讓讀者感受到一種席捲而來的真實感，小到母親當時的神態、話語、動作，事無巨細，一一道來。

但麥家寥寥幾筆，便將那一幕寫得好似發生在眼前，個中功力，可見一斑。

作為英雄背後的女人，陳夫人這一生註定被打上標籤，無法為自己而活。未嫁從父，出嫁從夫，夫死從子，這種古代女子的「三從」，竟在她身上無比貼合的展現，令人唏噓。

印象中，女人應該是明媚的，哪怕嫁了人，也要活得生氣勃勃，會撒嬌，會發火，會大吵大鬧。

但陳夫人一輩子順從，不爭不搶，把現狀看作理所應當，這才是時代造就的最大悲劇。

陳忠實說：成熟了的杏兒，把兒鬆了，風一吹就落地了，風不吹也要落掉了。成熟是勝利，也是悲哀。

她原本也是明媚的少女，但生活的重擔、英雄主義的標籤，讓她成熟，也讓她漸漸失了自己。悲劇悲劇的從來不是悲劇本身，而是身處悲劇中的人，自然而然地承受了這份悲劇。

麥家的文字於無形中將我們帶入了一個時代，一個充滿悲情意味的時代。

他把身處其中的每個人物，用寥寥數語，卻又血肉豐滿地剖白給我們看。那種蒼白感，令人心酸，心疼，乃至心驚。

其實一直很想問，陳夫人究竟愛陳二湖嗎？

或許愛，或許不愛，也或許那份愛，不是當今時代的我們所能理解的愛。

就像一枚泡沫俯身去親吻一艘沉船，這是我愛你；一顆石頭墜入湖心，沒有驚起一陣漣漪，這也是我愛你。

如果說愛的話，那陳夫人一定屬於第二種。
