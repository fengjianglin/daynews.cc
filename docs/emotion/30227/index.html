你們家庭事業兩手抓，我只想好好帶娃

2019-12-24 269條評論 1,235次閱讀 250人點贊

中國人民公安大學教授，研究生導師李玫瑾教授

主要研究犯罪心理學；犯罪心理畫像；公安社會心理調查研究；未成年人犯罪心理及預防；有組織犯罪問題等問題的中國人民公安大學教授，研究生導師李玫瑾教授在接受某記者採訪時這樣說：

我要告訴所有年輕的父母：在準備懷孕、準備養育孩子時，一定要做好思想準備，在孩子出生後的第一年不管如何辛苦、如何艱難，一定要自己帶孩子，不要讓孩子離開你！

在孩子１２歲之前，不要隔絕了孩子與你的依戀關係；孩子年齡越小越需要你；你帶孩子的時間越多，孩子對你的依戀越深。

父母對孩子的心理影響力與心理控制力，不在父母掙錢有多少，也不在父母多有知識，更不在父母的事業有多成功，只在你為他生命的初期付出的辛苦有多少；在你為他依戀時期（指１２歲以內）的相伴時間有多少！

孩子的依戀期就那麼短短的十幾年，機不可失。

李玫瑾教授從多年前就開始呼籲：孩童學前教育非常重要，倡導女性休三年產假。

自從我生完二胎，一個微商同事一直聯繫我，要我加入他們溫暖勵志的大家庭。

並且總會有幾個寶媽，用她們差點被拋棄的慘痛經歷告訴我，女人不能不掙錢，不然會被老公看不起，很快人老珠黃，老公會出軌年輕小姑娘。

也很心動過，看著群里各個精英媽媽羨慕不已。

無奈我雖是個二寶媽媽，就算有孩子姥姥奶奶各種幫忙，還是手忙腳亂，天天只能和洗衣做飯，大寶幼兒園花樣百出的手工，二寶的屎尿屁做鬥爭。

每天還心驚膽戰大寶在幼兒園又感冒了，會不會傳染給二寶，大寶二寶的針還沒有打，我想我可能就是同事說的廢柴媽媽。

真正讓我打消念頭的是小區院里媽媽們的勵志楷模晨晨媽媽：

晨晨四個月的時候，晨晨媽媽開始加入微商大軍，根據孩子需要賣某紙尿褲，開始只是想著賣不了自己的娃用，有收入更好。

後來架不住洗腦般的營銷，開始帶著5個月的晨晨做地推，建了寶媽微信群，分享育兒知識。

在晨晨6個月的時候，晨晨從床上掉下來，頭磕到了桌子角，去醫院縫了兩針。

我們都以為她會收手，專心帶娃，沒想到在偃旗息鼓一周後，繼續帶著受傷的晨晨地推賣貨。

事實上，晨晨成了大家的娃，關係好的寶媽都幫忙帶過。

馬伊琍

2018年，著名演員馬伊琍在《非常道.黃金時代》節目時，主持人提問：成功女性如何平衡家庭和工作？

馬伊琍的回答是：所謂達到一個平衡，一定是多方面一體的，你一個人是平衡不了這些事情的，我絕對不相信誰可以做到如此光鮮靚麗的職業女性和一個優秀的合格媽媽去兩者兼并，這絕對不可能。」

馬伊琍

有多少人在看似帶娃，其實在忙事業，我只是想說，真的很難兩全，希望微信群里的媽媽們不要在鼓吹全職寶媽掙大錢，分分鐘提瑪莎拉蒂，也請有人幫忙帶的寶媽們，不要一副我行你怎麼不行的樣子教育那些真的只有一個人帶娃的寶媽。

或者你一個人真的行，但是我們只想好好帶娃。

刷單兼職被騙的大多數都是寶媽，這些寶媽都是因為沒有生活費嗎，晨晨媽媽家庭優渥，愛人溫和，最重要的還是害怕不能證明自己的價值，害怕不被尊重。

李玫瑾教授在一次演講中說：我們全社會應該形成一個共識，人是國家實力最基本的東西，所以養育人是國家的大事。尤其在最重要的三歲之前這個階段誰來照顧？毫無疑問是母親。呼籲國家立法給母親三年的撫養期，在家養孩子實際上是替國家養，孩子養育在家庭，走出社會之後就是社會人，好的人會造福一方，但如果它變成一個犯罪人，所有人誰遇到他就是的災難。

我相信真的有媽媽可以做到孩子優秀，家庭出色，但是大部分還是帶得了娃，就上不了班，就算娃娃上了幼兒園，也只能找一份契合幼兒園時間的媽媽，不能加班，不能出差，為家庭做出犧牲。

孩子是祖國的花朵，是明天的太陽，請對這些哺育祖國未來的媽媽們不要唯事業論，要讚美她們對家庭的付出，肯定她們的價值。

不要再宣傳全職媽媽就是懶媽、媽蟲這些言論，說得久了，不光女人信了，男人也會理所當然，我們要說，孩子就是我們最重要的事業，好好帶娃不僅僅是對家庭最大的貢獻，更是對社會穩定發展最有力的支持。
