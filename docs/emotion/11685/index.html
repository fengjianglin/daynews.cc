本以為是幸福的開始，沒想到是噩夢的開端，6歲男孩讓人心疼

2019-12-11 2,200條評論 3,687次閱讀 389人點贊

對於孩子來說，爸爸媽媽在身邊，陪伴著自己長大，就是最幸福的事。

但如果爸爸媽媽做的不好，那就是孩子的不幸，若父母若常打罵孩子，那麼孩子就會非常痛苦。

活潑可愛的男孩，有了後媽，對未來充滿期待

亮亮是一個活潑可愛的小男孩，雖然從小沒有媽媽，但是在奶奶的教育下，懂禮貌，又乖巧，非常討別人喜歡。

在亮亮六歲這年，爸爸領回來一個女人，說是自己的媽媽。

雖然亮亮很小，但是因為平時聽過不少故事，再加上網路發達，也知道這應該是自己的繼母。

雖然很多人都說繼母對孩子不好，但是亮亮並不這麼覺得，因為這個媽媽來的時候，都會給自己帶好吃的，還給自己買玩具，買衣服，對自己也特別好。

所以亮亮選擇相信這個媽媽一定會對自己好。

在媽媽搬進來那天，亮亮覺得自己也將會是個幸福的孩子。

本以為是幸福的開始，沒想到是噩夢的開端

可是很快，亮亮就發現，自己錯了。

原本自己以為的幸福的開始，其實是不幸的開端。

因為媽媽來了之後，不僅沒有給自己買衣服買玩具，也不怎麼理自己。

後來，逐漸的讓亮亮干更多的活。

爸爸看妻子每天什麼都不幹，就知道出去打麻將，對此很不滿意，就要求她每天把家裡打掃乾淨，做好飯。

之後，爸爸每天回來，家裡確實被打掃的很乾凈，飯菜也已經做好，但這些工作，全部落在了亮亮的身上。

之後，更是因為一點點小事，新媽媽就打罵亮亮，還威脅他，不許告訴爸爸和奶奶。

孫子看到奶奶就跑，還以為是對自己生疏，直到掀開孩子的衣服

這一天，亮亮外出的時候，在路上遇到了奶奶，眼圈瞬間就紅了。

但想起繼母的話，自己不敢哭，害怕奶奶看出來，亮亮甚至都沒敢靠近奶奶就跑開了。

奶奶也看到了亮亮，以為亮亮這麼幾天就疏遠了自己，有些難過，追了上來。

沒想到在想要拉住亮亮的時候，無意中掀開了亮亮的衣服，才發現，亮亮的身上竟然布滿了傷口，新傷舊傷全都有。

奶奶一想，就明白了，瞬間心疼的落下了眼淚，帶著亮亮去找爸爸。

幸好被奶奶發現，爸爸還是愛亮亮的，他之後才能逃脫後媽的魔爪。

當面對家暴，孩子該怎麼做？

還有很多更加不幸的小朋友，可能正在遭受打罵，就像一位小姑娘，家裡面爸爸媽媽弟弟，經常打她，她已經躲都不敢躲，已經麻木了。

儘管一直在呼籲，但是孩子在家裡面遭受家暴的事，依然頻繁發生。

這些孩子，小小年紀就遭受著這些，該是多麼無助。

當孩子遭受到暴力對待，該怎麼辦呢？

1、像周圍的人求助

在遭受暴力時，不要一味地忍耐，可以像周圍的人求助。

比如亮亮，在受到繼母的暴力對待後，不要因為害怕不敢說，應該早點告訴爸爸和奶奶，向他們尋救幫助，就可以早點脫離苦海。

​2、向學校反映

畢竟孩子的年紀小，接觸的範圍也很狹窄。在遭受到暴力對待的時候，可以向學校反應，在情況不嚴重的時候，學校會找家長約談，如果更加嚴重，學校有義務向相關部門彙報。

3、報警

最基礎的，還是要學會報警。其實，即便是家庭中遭受到暴力，也可以給110打電話報警，警察上門，就可以解決很大一部分問題。

孩子，因為年齡小，見識少，是弱勢群體。

原本，家庭應該是孩子的避風港，但如果因為家人的暴力對待，讓孩子受到傷害，這個情況將會更加嚴重。

如果孩子受到傷害，或者發現身邊有這樣的孩子正被家暴，一定不要袖手旁觀，花幾分鐘，伸出你的援助之手，很可能改變孩子一生。
