我們笑著說再見，卻深知再見遙遙無期

2019-12-23 311條評論 4,491次閱讀 2676人點贊

前段時間，一個好久沒聯繫的朋友，發了朋友圈，配圖是蛋糕。我第一反應是她在過生日，立馬點了個贊，又在評論框里輸入了「美妞，生日快樂哦！」剛想點發送，又覺得不對，到底是不是她過生日啊！如果不是，那多尷尬啊！我又默默刪除了評論……

原來我們一直都沒有給對方慶祝過生日，更不知道彼此生日是哪天。

現在想想，真是覺得有點遺憾。

她其實是我的同事，和我一個學校的，算是我的師姐。那時我也是剛畢業參加工作，自然而然的我倆慢慢處成了好朋友。

一起工作，一起吃飯，一起住宿舍，甚至都是一起辦的辭職手續，又一起租房子。雖然沒有住在一起，但是都在同一個小區里，彼此都能互相照應。

晚上下班後經常一起去爬千佛山。站在千佛山上俯瞰濟南的夜景，真是太美了。我倆都會用羨慕的口氣感嘆，啥時候我們也能在濟南買套房子啊！有個真正的屬於自己的家，有盞燈能照亮我們回家的路。

後來隨著工作的變動，我們住的越來越遠，見一次面得坐好久的公交車。我們一起去大明湖，芙蓉街，曲水亭街……吐槽工作，分享小心事，她也一路見證了我和老公的戀愛史。

再後來，我因為懷孕生產，就在老家住了一年多，我倆聯繫也就少了。等我再回濟南後，也約著見面吃過飯，可再也不是曾經的感覺了，大概還是生疏了吧！

現在的我們，都成家立業了，也都在濟南有了屬於自己的房子，當初的期盼都變成了現實。可我們的友情卻變淡了，淪為了朋友圈點贊之交。

問題到底出在哪了呢？

人與人之間，共同話題少了，交情自然也就變淡了。

從小到大都是如此，一路走來，朋友慢慢都走散了……

小時候，有一個最好的朋友，我倆一起上幼兒園，小學，初中，只是高中她沒有讀，就輟學了，再後來我去外地上學，她去打工，就幾乎沒再見過了。

小學時，我倆一直是同桌，有一次她上課流鼻血了，我陪她出去清理。結果剛回教室，凳子還沒暖熱，我又流鼻血了，她又陪我出去清理。同學們都哄堂大笑，連老師都笑著調侃我們真是患難朋友啊！連流鼻血都湊一塊。

現在想想，那時候多好啊！無憂無慮的童年，只是再也回不去了。

上初中也有一個好朋友，我倆做了兩年同桌。互相督促學習，早起背英語單詞，從家裡給彼此帶好吃的，送生日禮物，畢業留念冊上寫了滿滿的勿忘我，貼滿了大頭貼，離校分別時哭的稀里嘩啦的。

高中不在一個學校，學校又不讓用手機。只能通過寫信聯繫。聊學習，聊喜歡的男生，聊以後考大學，聊千萬不能忘了對方……

只是高考結束後，上了不同的大學，分散在不同的城市，走著走著又散了。

上大學後，第一次離家那麼遠，陌生的環境還是挺擔憂的。

好在遇到一幫溫暖的室友，大家一起軍訓，一起吃飯，一起上實驗課，互相練習扎針，一起熬晚自習，一起去海邊……

那時我是宿舍年齡最大的，都喊我大姐，喊我老公（那時候還是男朋友）大姐夫，故意在我們通電話時搗亂，起鬨。

有一次我和男朋友打了一宿電話，這幫傻乎乎的小妞們，以為我在說夢話呢！現在想想真慶幸她們當初沒有把我踢出宿舍啊！

畢業前夕，我們約好了誰結婚，其他的就都去做伴娘，看著她美美的出嫁。

可最後依然都沒有兌現……

唉！為什麼回憶著，回憶著，眼淚出來了呢……一個一個朋友，一幅一幅畫面，都深深地印在腦海里。

原來自己一直都未曾忘卻，只是不敢去想，不敢去碰觸。

每一個朋友，每一段友誼，都有一個故事，一步一步見證了我的成長，我的青春……

微信圈裡的你，好像瘦了，頭髮也長了，變得漂亮了，笑的還是那麼明媚，那麼溫暖。我瞬間感覺真好，曾經對彼此的祝福，不就是以後要狠狠地幸福嗎？

年齡越大越明白，世間好多事物都是有保質期的。麵包會有，牛奶會有，秋刀魚罐頭會有，愛情會有，友誼也會有，並且在不知不覺中就過期了。

人生命中每個階段都會出現一些人，陪你走過一段路，然後離開。在告別一段關係後，我們又會遇到新的人，再開啟新的生活。

時間久了也會發現，「以後常聯繫」這種話，不是承諾，而是一種禮貌的告別。

每個人都走在人群里，你走得離我遠了，就會離另外一些人更近了，未必不是好事情。

或許又交了新朋友，或許周圍環境發生了變化，又或許是根本不想再袒露心事……就這樣安安靜靜走散了也好。

也沒必要再刻意維持，刻意尬聊，相見不如懷念，就讓我們的友誼活在回憶里吧！

那時那刻的我們的確是真心相對，想做一輩子好朋友。誰也沒有欺騙誰，誰也沒有辜負誰！

走吧！你回頭也好，不回頭也罷！

就像天上下雨，沒必要非說成是天使的眼淚，只會顯得太矯情。

大家都是成年人了，要適應順其自然的法則。

無為而治也挺好，幹嘛要強求呢？

我們現在唯一能做的，就是好好珍惜當下，多製造一些美好的回憶，不給以後留太多遺憾。

過去的終究是過去了！

世間美好，都是恰逢其時。至於那些聚了又散的人或事，淡記就好。或許，終有一天，心歸寧靜，從容樸素。

我什麼都沒有忘，只是有些事只適合收藏，不能說，不能想，卻也不能忘。

我們笑著說再見，卻深知再見遙遙無期。

只能祝福我的老朋友們，所求皆如願，所行化坦途，多喜樂，長安寧。
