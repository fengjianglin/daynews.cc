人生，就是生下來活下去；風吹雨打知生活，苦盡甘來懂人生

天天要聞
3 天 ago

不要艷羨別人。要明白，他的人生和你一樣，像一幅刺繡，在正面呈現那柔亮艷麗的圖景之時，背後卻布滿了蕪雜而痛楚的針腳。失去了的就是失去了，不要期待再次擁有，因為不再完整。失去了就是失去了的，它永遠都不會恢復到完好無損。所以我們要拿得起放得下。

活著是一種心情，窮也好，富也好，得也好，失也好，一切都是過眼雲煙，只要心情好，一切都好。不管你曾經被傷害得有多深，總會有一個人的出現，讓你原諒之前生活對你所有的刁難。我們都在最沒能力給別人的時候，遇見最想承諾的人，也在為了理想不得不前進的時候，遇見最想留住的人。

生命中的許多東西是可遇不可求，刻意強求的得不到，而不曾被期待的往往會不期而至。趁著夢想還有溫度，人生還有轉機，抓緊時間去奮鬥，不要等到中年時，只能嘆息自己的碌碌無為。沒有必要把你懂的所有東西都說出來，人們不喜歡他們身邊有人比他們懂得更多，那會讓他們很惱火。規矩，是一種提煉自己的能力，時間，是一種抹殺未來的刀，如果沒有今天的付出，就沒有明天的感動，什麼天長地久，什麼爭名奪利，都是有準備的。

世上沒有通過想像和推測就可以判定未來的事情。 路都是自己走出來的，絕不是參照別人的樣子推斷出來的， 別人在某件事情上做得再好，用的方法再妙，換成你的時候， 又都將會成為另外一個樣子，等同於又是一條新的路。

什麼樣的選擇，決定什麼樣的生活。 在生活這場考試中，選擇題的選項不能讓別人代選，那樣到最終你很可能不及格。 大多數人認可的價值觀從來就沒有適合過所有人，我們不必為成功放棄親情，我們不必為穩定埋沒才華，在很多問題上，我們並不是沒的選擇。

每個人的心裡，都有一本難忘的書。獨自的時候，總想翻開，因為那些地方，那些人，那些事；疲憊的時候，總想觸摸，因為能給人生活的力量，給人生活下去的勇氣；想起的時候，總是難以釋懷，因為只有自己懂得，自己最明白。

有些路不合腳，卻必須走；有些選擇不合心，卻要知道適應。人生，就是生下來活下去，雖然很難。沒有哪個男人，在細節上都是完美的；也沒有哪個女人，在每個時刻都是美麗的。因為優點而愛，會因為缺點而離開；因為相貌而喜歡，會因為容顏而厭淡。

每個人的路都得自己走，累不累，腳知道。經歷了流年聚散，體會了人情冷暖；經歷了物是人非，學會了自我療傷。有苦，自我釋放；有樂，欣然品嘗。風吹雨打知生活，苦盡甘來懂人生。其實人生，就是一種感受，一場歷練，一次懂得。

珍惜身邊的幸福，欣賞自己的擁有, 背不動的就放下，傷不起的就看淡, 想不通的就丟開，恨不過的就撫平, 人生本來就不易，生命本來就不長, 何必用無謂的煩惱, 作踐自己，傷害歲月。

人生就像一杯白開水清澈透底，也像一杯苦咖啡耐人尋味。看淡了，是是非非也就無所謂了；放下了，成敗得失也就那麼回事了。人間三千事，淡然一笑間。面對人生種種境遇，一笑而過，是一種人生的優雅。

我是胖胖，感恩生活，分享快樂！認真回答每一個問題，用心結交每一位朋友，謝謝閱讀，認同請轉發和點贊， 謝謝關注頭條號：胖胖的愛情！
