同事相親，女孩消費580，他結賬，女孩：剛剛，你老闆發來微信

李亮剛到公司，財務李姐看到他，就讓他到財務辦公室。

「出差回來了？上次給你介紹那個女孩回來說了……你一月掙錢也不少，大方點，你還跟人家AA啊……呵呵，我一個遠房親戚家的女孩也來這城市上班，周六去見見面？前幾天女孩去我家吃飯，你哥還誇你呢……」

正說著，老闆也進了公司，聽見說話，問，「李亮出差回來了嗎？」

「是，是，回來了。」

李亮抬腳準備出來，跟老闆彙報工作，李姐在後邊說，「看你夠辛苦的，這鬍子拉碴的。」

財務李姐是老闆的夫人。

李亮跟著進了老闆進了辦公室，老闆示意他坐，自己拿水杯接上水，又讓前台給他接了杯。

李亮喝了口，剛要說，老闆從辦公卓的抽屜里拿出兩盒中華煙給他前面一放，說：「少抽煙。」

李亮木訥地笑了笑，開始彙報起工作。

聽李亮彙報的老闆先是眉頭緊鎖，臉龐綳著，然後眉頭舒展，嘴角有了笑容。

李亮說：「放心吧，解決問題了。」

老闆沖李亮點點頭，說過「好」後又說，「你已經連續出差三個星期了，一會兒你就直接回家吧，回家休息休息……正好後天周六，你有時間嗎？」

「嗯？去，去相親？」

「你嫂子跟你說過了吧，去吧，見見面。你也大方點，嘴會說點，」老闆笑著說，「你都28了。」

到周六，李亮收拾利落，怕再笑話他「不大方」，兜里塞上了1000元。

相親的地方是一家西餐廳。

女孩比他先到，他坐下後，女孩看了他兩眼，又低頭去看手機。

女孩看來有點「害羞」，長長的頭髮，眉眼也很耐看。

李亮自我介紹下，一會兒，就「結巴」了，不知道說什麼。

就喊服務員過來，讓女孩點菜。

女孩讓了下，點了幾個菜。

菜名說出來，李亮下意識地去摸了摸裝錢的那個兜。

等一會兒菜上來，女孩打破了冷場問：「你是技術部的經理是吧。你，你周末都做什麼呢？」

「我，我，不忙的時候，在家看書。你呢？」

聊來聊去，兩人的興趣接近，聊得越來越「熱」了。

快吃過飯，女孩手機來了個消息，她去看。

李亮喊服務員過來結賬，服務員說消費580，李亮去掏錢。

女孩攔了他下，說，「剛剛，我哥，就是你老闆發來個微信，他來買單。發了個大紅包。」

看來老闆還是擔心他不大方啊。
