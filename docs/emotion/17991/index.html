看得人心酸！小男孩裝好衣服要跟父母一起走，出門時父母已經走了

2019-12-16 1,016條評論 2,240次閱讀 332人點贊

每個家庭都需要經濟的支撐，所以有很多小城市的勞動力會流向大城市，追求更多的機會和收益。可是需要關注的是，這些奔赴其他城市謀生的人，大多數都是青壯年，也都是孩子們的父母。

在外打工的父母們不可能有時間和精力把孩子照顧好。所以就出現了許多身在異鄉思鄉思親的漂泊者還有苦苦思念等候父母歸來的留守兒童。

有一句形容留守兒童的話聽起來就讓人心酸：「父母回來的時候，他高興的就像過年一樣。」而微博上也有同樣的一種辛酸。父母假期結束，又要走了，孩子裝好衣服要跟父母一起走，可等他出門時，父母已經走了。孩子號啕大哭的樣子讓人心疼。圖片中可以看出，孩子還只有四五歲，本該無憂無慮的快樂年紀，他卻被迫和自己的父母分離。

孩子站在門口，兩隻小手中各拎了一個塑料袋。一隻袋子里鼓鼓囊囊的，全部都是孩子自己胡亂塞進去的衣服。另一隻手中的袋子里只有一雙鞋。一雙和孩子另一個袋子里的衣服不同的，嶄新的休閑鞋。猜想這雙鞋可能是父母回來時帶給孩子的禮物，還是新的。而孩子也不捨得把這雙鞋子和其他的東西胡亂塞在一起，而是單獨找一個袋子裝起來。

孩子多麼愛自己的父母，多麼想和父母在一起，從這雙珍之重之的鞋子就可以窺見一二。而且，父母肯定也不是不思念孩子，不是不想和孩子生活在一起的。可是生活有時候就是這樣的無奈。

網友們看了微博也特別有感觸，網友們留言：「太真實了，小時候很多孩子都是這樣被騙開的，但是大人為了生存也是沒辦法，我們父母不在身邊，小倆口自己邊上班邊輪流帶娃，雖然比較辛苦，但是真的不捨得孩子」、「我小時候就是這樣，那種感覺很絕望」、「孩子長大了或許不會記得，但如果記得一定很難過，看到這圖就很心碎，但是他們的父母也一定是迫不得已才這樣，去大城市打工，很多人自己都擠在宿舍，怎麼照顧孩子，孩子怎麼上學，一切都很現實，曾經經歷過跟父母分離的我唯一能做的就是自己更努力，給孩子更好的一切，包括陪伴。」

孩子需要父母的陪伴，可是父母也有父母的無奈。現在很多的留守兒童因為太久沒有見過父母，而和父母變得生疏。哪怕他們的心裡都渴望著親情，渴望著在一起，可是當父母子女真的面對面的時候，更多的卻是相顧無言。

當留守兒童的留守已經成為現狀，那麼我們至少需要付出努力讓現狀變得更好。建立留守兒童與母女之間的親子關係，需要父母格外的努力。

⑴盡量參與孩子的日常生活

雖然父母們外出工作，不能陪在孩子身邊，可是這不代表著父母必須得在孩子們的生活中銷聲匿跡。尤其是在網路和智能手機、電腦發達的今天，視頻通話基本上可以保證實現了。

家長們可以多多通過視頻、打電話聊天來陪伴孩子，和孩子交流。而且也可以選擇經常給孩子寄一些禮物。這禮物不需要多麼貴重，可以是爸爸媽媽自己做的好吃的，也可以是一兩本故事書。但是這樣的方式卻能讓孩子們在生活中也感受到父母的存在。

⑵孩子寒暑假接到工作城市一起生活

父母要工作，回來的機會本該就少，所以更應該創造多一些機會讓父母和孩子團聚。平時孩子也要上課沒有辦法，可是孩子會有寒假和暑假。在這段時間裡，父母可以把孩子接到自己工作的地方，好好地和孩子生活一段時間。

如果爸爸媽媽工作實在不能照顧孩子，也可以讓孩子的爺爺奶奶一起過來，一家人團團圓圓的在一起，解一解大家的思念之情。

⑶多與孩子溝通，將愛說出來

留守兒童的親子關係很難建立，其中一個重要的原因就是家長和孩子們長時間不相見而缺乏情感的交流和溝通。孩子和父母們彼此都滿懷著思念和愛，可是卻通通說不出來。

外出的家長們其實更應該多表達自己的感情，告訴孩子自己愛他，但是爸爸媽媽也是有無奈的。有些東西不說出來，只靠感覺是無法真切地認識到的。

只有父母的良好的溝通，積極地表達自己的感情，才可以讓孩子感覺到被父母愛著，讓孩子理解父母，然後建立起更深刻的親子感情。

⑷參與到孩子的教育中

很多留守兒童和家長之間都有過這樣的非常心酸又有些傷人的時候。比如，家長偶然和聊天，卻無意中說錯了孩子的年齡或者孩子上幾年級。每當這種時候，孩子的心裡一定會感覺到難受。為了少讓這種情況發生，家長們也應該去關心並且儘力參與到孩子的教育中去。

父母不能去孩子的學校，但是至少應該在和孩子日常的相處中知道孩子在學校都發生過什麼事情，知道孩子關於老師的看法，儘力了解孩子每一門學科的成績，還要和孩子的教育者建立一定的聯繫。

這些工作不是無用功，這讓孩子感覺到自己是被父母關注並且在乎的，而且也能夠和孩子有更多的話題。

留守兒童本該就缺少陪伴，缺失了和父母一起生活的幸福和快樂，但是無論如何，父母對孩子的愛絕對不能缺席！因為愛是父母和孩子的心中最牢固，也隱藏最深的紐帶。雖然父母不能陪伴孩子，但是父母不能在孩子的生命中失蹤，父母應該讓孩子知道他們對孩子們的愛。留守兒童和父母建立良好的親子關係要難得多，父母和孩子也要付出更多，才能回報一點點。父母付出是關鍵，生活無奈不是借口。只有讓孩子在生活中能夠感受到父母的痕迹，感受到父母的愛，才能真正地建立起親子關係，而不是成為「最熟悉的陌生人」。
