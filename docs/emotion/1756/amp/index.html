那個愛折騰的同學負債50萬，那個甘於平凡的同學家庭美滿

3 週 ago

摘要

從大學畢業起，我們每個人的人生就像那天空中飛揚的風箏。有的想越飛越高，最後被扯斷摔得粉碎；有的安於空中的高度，迎風招展。我們都是不一樣的個體，每個人都有自己的夢想，自己的選擇。有的選擇平平淡淡過一生，有的不甘於命運的安排，想要努力打拚出一條屬於自己的路。

從大學畢業起，我們每個人的人生就像那天空中飛揚的風箏。有的想越飛越高，最後被扯斷摔得粉碎；有的安於空中的高度，迎風招展。

我們都是不一樣的個體，每個人都有自己的夢想，自己的選擇。有的選擇平平淡淡過一生，有的不甘於命運的安排，想要努力打拚出一條屬於自己的路。我相信你的身邊也絕對有那麼兩類同學，一類愛折騰，總是看他一直在路上。一類容易滿足，在自己的小天地里自得其樂。

小靈是畢業6年，這6年沒有絲毫存款，反而負債50萬。他做過銷售，做過直銷，創過業……這6年，他好像就沒有安定過，換了5個城市，8個行業。

小靈學的是酒店管理專業，大四快畢業的時候被校招到一家五星級酒店實習，做了三個月的餐廳服務員，三個月的前台。他說：「之前選擇這個專業是看電影里酒店管理者都很專業，很紳士，沒想到畢業後卻是給人端盤子。」每天早九晚十，周末也沒有雙休，職能調休。餐廳服務員的工作完全是純體力勞動，沒有絲毫腦力可言，他開始懷疑自己這四年的大學有什麼意義。於是選擇了辭職。

辭職後，他沒有再選擇酒店相關的工作。他大學的時候就有一個夢想，他想去中國最繁華的都市——上海，在上海最繁華地區的寫字樓上班。於是他隻身一人拉了個行李箱來到上海。只是，他沒想到的是，以自己三本院校的學歷，別說大公司，連一家像樣公司都不願意錄取他。為了解決生存問題，他很快入職了一家小公司從事銷售的工作。

一年後他從銷售小白到月入兩萬的銷售精英，憑著他出色的銷售業績，也存了一點錢。他開始不滿足於幫別人打工，於是衝動地辭職了，回到老家，開起了服裝店。沒有任何服裝經驗和開店經驗的他，很快就面臨關店的結局，從原來的存款5萬到負債20萬。

年僅25歲的他就負債20萬，他開始焦慮，拚命想多賺錢，有賺錢的方法他就想去嘗試。他看有朋友做直銷賺錢，又投入2萬塊錢去做直銷，但是直銷豈又是那麼容易做的呢？看到朋友做微商賺錢，又開始折騰去做微商，但是他進入的時候已經過了微商的紅利期，只是成了別人的韭菜，任人宰割……

6年過去了，他的債沒有減少，反而越欠越多。他開始疑惑，自己真的這麼差嗎？其實他的能力在同學中是比較出眾的，也能吃苦，愛拼搏。反觀另一個同學，雖然平平淡淡，卻家庭美滿。

小想和小靈是高中同學，第一年他和小靈一樣，只考了個三本院校，但是他不願意讀三本，於是選擇推遲一年入學，選擇了復讀。雖然復讀一年的成績也沒有太好，但至少是個一本師範院校。畢業後，他沒有選擇去大城市，而是選擇回到了縣城參加了縣城高中老師但招聘，成為了一名高中數學老師。

老師但生活雖然平淡，卻也安穩。他也很懂得享受生活，周末去縣城的好店吃吃美食，和同學們打打籃球，也會為了賺取額外的收入接接私教課。憑著他這幾年的收入和父母資助，也在縣城買了自己的房子和車子。去年也結婚有了自己的家庭和小孩。

如果從傳統意義上看，小靈的人生無疑是失敗的，畢業6年，換了N份工作，N個行業，負債50萬，卻依然沒有自己的競爭力。小想在縣城有份穩定的工作，有房有車有家庭，生活安逸，工作穩定發展。如果是你，你願意選擇誰的人生呢？

難道真的越折騰越窮嗎？如何折騰才能讓你更接近成功呢？

1.有一顆強大的內心，不害怕失敗。

姜子牙我們都知道是輔佐武王消滅商紂，建立周朝的功臣。但是他一生卻並不順遂，姜子牙出世時，家境就已經敗落，所以他年輕的時候干過宰牛賣肉的屠夫，也開過酒店賣過酒， 但是他人窮志不短 ，無論宰牛也好，賣酒也好，從來都沒有放棄努力，不斷學習攻城謀略，可是直到70歲還是一無是處，閑居在家。但是他沒有就此放棄。終於在72歲，在江邊垂釣時遇到了姬昌，成為了一代名臣。

一般愛折騰的人都是內心不安於現狀的人，他對未來充滿了期待，希望自己能建功立業，有所作為。但是，現實卻總是不如你所願。古人有言：「故天將降大任於斯人也，必先苦其心志，勞其筋骨，餓其體膚，空乏其身。」

有折騰就會有失敗，我們需要有一顆強大但內心來面對失敗，才能在下一次更好地迎接成功。

2.不要輕易選擇創業，特別是負債創業。

我知道很多人都有一顆老闆夢，希望有一天可以擺脫上班的生活，可以隨意安排自己的時間。很多年輕人年紀輕輕就辭職創業了，最後一無所獲還負債纍纍。

說說我剛畢業那會兒身邊的例子：鄧玲（化名）剛畢業兩年，借錢在上海某地開了一個咖啡店，不到一年，咖啡店倒閉了，負債50萬；李新（化名）畢業三年，和男朋友一起在家鄉市區某學校旁邊開了一家餐飲店，不到半年，維持不下去放棄，負債10萬；林倩（化名）剛畢業一年，信用卡透支加父母資助，在老家市區開了家服裝店，不到一年，服裝店倒閉，負債20萬……

現在隨便瀏覽一個網頁，一個視頻都會看到某某創業的故事，某某創業過上了富足的生活，月入百萬的生活，首先不談論這個故事的真實性，我們想想就算故事真實，現實生活中創業成功的又有多少呢？特別是那些沒經驗，沒資金，沒有市場分析，只憑著一腔熱血的年輕人。

那些有多年工作經驗和社會閱歷的人創業都有90%的失敗率，更別說年紀輕輕剛畢業的大學生們。所以，如果你不是家裡礦，請不要隨意選擇創業。

3.我不反對你折騰，但請有目標的折騰。

有的人折騰是完全沒有目標的，也不知道自己要幹嘛，成為什麼樣的人，完全就是肆意妄為。如果從職業發展的角度看，那些經常跳槽換行業的人很難有一個好的職業發展道路。但如果你有自己的目標，未來也並不是只有職場一條路可以走。但前提是你有一個自己的定位和目標。

有人說如果不折騰就沒有今天的馬雲。但是馬爸爸創業之前就很清晰地知道要做屬於中國自己的網站。並且他從沒有改變過，即使當時創業路不順，他去擺地攤也只是為了資助他這個互聯網夢想。

結語：這篇文章並不想告訴你應該選擇哪條路，因為哪條路都有各自的精彩。我們要做的就是選擇自己所想，堅定自己所選。年輕人多折騰也並沒有什麼不好，只有折騰才有改變命運的可能。但我相信，那些安於現狀的生活也是另一種人生。
