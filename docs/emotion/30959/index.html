家長總是“奪走”孩子的依戀物，過分控制讓孩子失落了什麼？

2019-12-23 123條評論 1,356次閱讀 254人點贊

01

朋友兩歲大的女兒蓓蓓，擁有一隻玩具熊，朋友常和我說，「蓓蓓對這隻玩具熊有一種很特殊的眷戀。」

特殊到什麼程度呢？

就是任何玩具都不能替代它的地位，無論去到哪裡都要帶著，每晚上床睡覺前還要和它玩上很長時間才會入睡，就連吃飯的時候，也一定要擺在身邊看得到的地方，這樣的情況持續有三四個月了。

朋友很苦惱，問我：我這樣允許的她的行為，是不是錯誤的呢？

聽到朋友這樣問，我想了想自己的孩子，「我覺得不是錯誤，但是你平時可以多發現她其他感興趣的事情。」

我的大女兒沒有表現出有特別喜歡的依戀物，但是有一些觸覺過分敏感導致的不良習慣。

比如：一緊張就扭手、咬指甲；總是忍不住要摸我的耳垂；喜歡用拳頭去頂自己的肚子等等。

小女兒現在也是兩歲多一點，她也很喜歡毛絨玩具，有時候會帶著它們睡覺，這些玩具中她尤其喜歡熊貓，出去玩的時候也經常會帶著，碰到別的小朋友看到熊貓時，還會抱得緊緊的，生怕別人搶走似的。

但是好像也沒有朋友家孩子那種到了「迷戀」的程度。

02

兩歲的孩子，會走路、會說話、還會跑會跳，他們已經學會了一些基本的技能，當他們發現自己會得還不少時，於是就萌發獨立意識，開始探索這個世界。

但是，對他們來說，世界那麼大，又那麼複雜，小心臟好緊張噢。

情緒緊張的時候會有哪些表現呢？咬吸自己的手指頭、說話容易結巴、遇到一點小事就大發雷霆、躲在媽媽懷裡。

或者，尋求安慰物，他們會把當時身邊的一些東西作為安全感來源的依戀物（譬如她心愛的娃娃、身上蓋著的毛毯、隨身攜帶的毛巾，或者自己的手指頭），久而久之，這些依戀物會被當成他們的「幻想夥伴」，幫助他們消除緊張情緒。

心理學家唐納德溫科特對兒童所表現出來的依戀於某一軟質玩具或物品的現象進行了研究，認為這種現象在兒童的心理發展中具有過渡性意義，並命名其為「過渡物」。

他認為隨著孩子慢慢長大，在心理上愈發獨立，就自然而然會把這種情感依託轉移到其他地方。

03

在威斯康星大學兒童心理學者理查德帕斯曼的研究中發現，「軟物依戀」情結在全世界普遍存在，如在美國、荷蘭、紐西蘭等國，「軟物依戀」情結的孩子比例達到了3/5，韓國的比例是1/5。

其實，除了小孩子，很多成年人也有一定的軟物依戀情結。

比如：把毛絨玩具當做禮物、看到娃娃機里的可愛玩偶就毫無抵抗力、偏愛毛絨絨的睡衣、沙發上軟軟的靠墊、硬地板上鋪上毛毯……這些軟質物品通常會讓人感到放鬆和舒服，甚至有減壓的效果。

我自己也是很愛軟質物品的，特別是到了冬天，到處一片冰涼的時候，就特別喜歡周邊充滿著毛絨絨的物品。

04

小樊家的女兒現在快八歲了，還一直有著咬指甲的習慣，每次看著她十隻手指頭光禿禿的樣子，小樊都告訴她不要咬手指頭，知道女兒很愛美，就試著從美的角度出發跟她說，但是一直沒有效果。

後來，她在尋找對症的時候，才發現她是三歲前，大部分時間和爺爺奶奶生活在一起，被過度保護和限制，而導致了觸覺過分敏感。

造成觸覺過分敏感的原因主要就是大人的過分限制所引起的，比如在手口敏感期限制孩子吃手、不允許孩子用手抓飯吃、因為怕臟不讓孩子玩沙玩水、拿走孩子的依戀物等等。

上幼兒園後，女兒來到父母身邊，爸爸每次也很是反感她挑食、咬指甲、愛哭鬧等等一系列行為，每次看到就開始數落和進行嚴厲的制止，有時候可能當時湊效了，女兒被權威嚇到在他面前立即停止了該行為。

實際上，卻是給孩子造成了一種緊張感和壓迫感，加深了大腦中對這些行為的印象，再經過爸爸多次嚴厲的強化記憶之後，大腦對這些行為的記憶就越加深刻，從而演變成越說越做，越做越說的惡性循環。

05

自從有了孩子之後，我也時常提醒孩子爸爸多看一些書籍，和孩子一起成長，奈何孩子爸爸從來都覺得自己已經是成年人了，學習不都是小孩子才做的事情嘛。

我家大女兒三四歲的時候曾發現她一個習慣，就是當我制止她去碰觸某些東西的時候，雖然她收回了手，卻會很用力地扭手指，並且喉嚨里發出很用力的「嗯」的聲音，一開始，我們看到她這種行為，可能會想不讓你碰個東西，還不服氣怎麼地？

後來，我分析她做出這些行為的背後，應該是在努力剋制自己不要去碰，但是因為手的過分敏感又讓她非常的不舒適。

當我把這些觀念和孩子他爸分享的時候，他認為我想多了。我們的觀念經常背道而馳，沒有共同語言，我想這也是造成孩子特別緊張焦慮的原因之一。

當我說你這樣簡單粗暴制止孩子是起不到任何效果時，他卻認為是因為我不管教孩子，他覺得孩子聽我的話，我卻不教他也聽爸爸的話。

孩子兩歲以後，已經有了自己的判斷意識和喜好，教育孩子，並不是讓她們成為一個個被大人控制的「乖娃娃」，而是能夠通過自己的去明辨是非，不成為人云亦云的烏合之眾。

一些不是原則性問題的小習慣，我們採取的措施應該是去弱化和轉移。大人小題大做的結果就是，經過你嚴厲的警告之後，反而把一些小習慣變成了戒不掉的癮。

知道這些行為背後的原因之後，為了緩解孩子的觸覺過分敏感。

我會和孩子一起洗澡，通過戲水讓孩子感受水溫刺激，並幫她擦潤膚霜進行撫觸按摩；我會幫孩子梳頭扎辮子，通過梳子刺激她的頭皮；我會一邊講故事一邊幫她摸背，幫她紓解緊張。

但是我不會嚴正聲色的告訴她，你有觸覺過分敏感，你得改。

05

依戀物對於一個孩子的成長來說，其實是有幫助意義的。

就和我們每個人在成長的過程當中，都是會把情感寄託到我們周邊的各種物品或者人身上一樣，再正常不過，可能有些孩子會顯得更迷戀一些，那也沒有關係，那只是一個過渡，大人不要因為自己看著不舒服就總是試圖去制止孩子，而只需通過在時間中不動聲色的轉移，就可以幫助孩子順利成長。

如果強行拿走了他的依戀物，就相當於突然剝奪了他的情感依託，他總需要再去尋找替代物，當身邊沒有可以寄託的物品時，最順手的就是自己身體的一部分了，一旦成癮，他會剋制不住地去一直做這個動作，哪怕很痛。

孩子之所以來到這個世界，是因為聽到了愛的召喚，作為父母不要強行去剝奪他的寄託，讓他在一個偌大的世界裡茫然失措。

當我們用足夠的耐心，以愛迎接他的到來，也以愛陪伴他長大時，他的心智會越來越適應這個世界。

孩子最怕的就是這種父母，在他還小的時候，著急的想逼他長大；當他長大要獨立了，又緊緊抓著不肯放手。

孩子在學習適應這個世界，而父母要學習的，是適應孩子成長的節奏。
