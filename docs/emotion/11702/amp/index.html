一個不團結的家庭，都是這幾個原因造成的

天天要聞
2019-12-11

文/我在田野中走過

家和萬事興，一個團結的家庭，父慈子孝，一團和氣，每個人都能感受到家庭的溫暖，和彼此之間的關心照顧和愛意；一個不團結的家庭，彼此冷漠，甚至為了個各自的利益，相互爭吵打鬧，甚至變成了仇恨，老死不相往來，無疑是人世間最為心寒的事。

我們的父母和兄弟姐妹，是我們最親的人，因為血肉相連，是曾經的一家人，是彼此間最應該珍惜的緣分，打虎親兄弟，上陣父子兵，打斷骨頭連著筋，你心痛我，我照顧你，才能顯出親情的可貴，你幫我，我幫你，有事兒共同承擔才會分得清人情的遠近，關係隔一層，都不能近一步。

一個不團結的家庭，都是以下幾個原因造成的。

1、不明事理的父母。

我們每一個人都希望父母能公允的善待自己的孩子，俗話說：一碗水要端平。然而有些父母總是有自己的私心，最為常見的就是「重男輕女」，一切事，都優先善待兒子，這就造成了，兒女之間的心理落差，兒女之間在家產和養老問題上就會有矛盾。

有的父母偏心，特別寵愛某一個孩子，袒護某一個孩子，這樣造成了兒女之間的矛盾也是正常的。

有的父母在遇到兒女之間有矛盾的時候，不能公正的處理子女之間的矛盾，甚至在裡面製造了更多更大的矛盾，這樣的父母無疑是讓人失望的。

有的父母不能尊重和善待自己的兒媳或女婿，一味的嫌棄和瞧不起，這也是造成家庭不團結的原因。

2、不孝順的兒子媳婦和女婿。

有的家庭會養出不孝的兒女，不但不對父母感恩，甚至還時不時惹事生非，令父母有打不完的饑荒，還令父母沒有臉面見人，這樣的父母與兒女之間也很難和平相處。

都說兒媳沒有撫養公婆的義務，但是你的老公有，愛屋及烏，選擇對公婆的尊重，才會迎來老公對自己的愛與敬重，都說有個好兒子，不如有個好媳婦，有個好閨女不如有個好女婿，一個家庭團結的關鍵兒媳女婿會佔一大部分責任。

孝敬父母，是兒女天經地義的事，如果子女不孝，也就會有父母的絕情和狠心。

3、子女之間自私自利，矛盾重重。

每個家庭成員之間，相互退步相互忍讓，才會有團結和平安無事，事實上一個兒女多的家庭，沒有人裝聾作傻，是不會有團結的；每個人都精明的要命，斤斤計較，都怕吃了虧，有事了，沒有人願意出錢出力，有好處了，都想多分一杯羹，這樣的家庭是不會有團結的，一旦為了各自的利益，撕破臉，那就叫，仇人見面分外眼紅，打鬧是常事。

有一些家庭，因為兒女多，常有拉幫結派的事發生，甚至幾個子女聯合欺負一個人，一個家庭不團結，多是因為利益的紛爭，和意見的不合，如果鬧到拳頭相見，大打出手的地步，一個家也就完了。

4、親兄弟，明算賬， 子女之間錢財賬目不清。

子女之間的矛盾，往往在父母養老問題上，相互會不滿，有的人以為自己多出了力；有的人以為自己多出了錢，相互不滿的結果，一定會相會指責，相互攀比，產生矛盾和不快。

有時候，為父母花的錢過多這個賬是要算清楚的，比如說父母過生日或者生病了，需要較多的錢，如果只靠一個人出錢，就會感覺負擔太重，有的子女 會自覺的把錢拿出來，有的子女，甚至一毛不拔，這個時候，兄弟之間是需要明算賬的，不要吃虧的總是老實善良的人，每個人都有撫養父母的義務，每個人都出一份力出一份錢才是公允的，把事情說開，把賬算明白，才不會有過多的矛盾。

親兄弟，明算賬，不要為了面子一個人都擔著，過了時候卻有心上的不滿和態度上的不善，給別人甩臉子，說氣話，這樣就不好了。

幸福的家庭都是相似的，不幸的家庭各有各的不幸，一個家庭不團結，就會有矛盾和怨恨，如果親人之間，變得冷漠仇視，無疑是令人遺憾和傷心的，每一個人都對自己的家庭獻出一份愛，我們的家才會變得溫暖美好，有情有義。
