你並不了解男人

你並不了解男人，並不是一個特別通透的女人，你活在自己的世界裡，一直是一個小公主，只關心自己，只關注自己，你內心的壓力並不大，周圍的人比較寵愛你，所以你沒有吃多少苦頭，沒有承受太大的壓力，你也不願意了解男人，他們的心思很難猜，你要讓男人追著你跑，而不是患得患失，那和天生特別幼稚，根本控制不了男人，也不了解男人！

跟其他人相比你太了解男人了，你是一個很聰明的女人，你知道如何控制一個男人，知道如何讓一個男人死心塌地，很了解男人心，知道他要什麼，能夠滿足對方，讓對方一心一意，在感情之中，你沒有吃過苦頭，沒有受過傷害，愛的轟轟烈烈，甜甜蜜蜜，他對你是真的好，也是真的幸福，一生無憂無慮，享受了榮華富貴，享受了美好的一生，男人會對你死心塌地，會愛你一輩子！
