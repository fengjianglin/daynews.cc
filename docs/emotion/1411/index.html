女人心海底針，一個男人也是海底針，根本很難了解，很難了解真正的人心，對大家來說，生活的一切都要靠自己，了解男人，能經營好感情，經營好彼此的愛情，對一個人很重要，尤其是女人，不安感更強，更想了解男人，你是不是真正了解男人呢？做一個心理測試吧，下面有三件古裝，如果想引起皇帝的注意，哪一件古裝最好呢？別猶豫，選出你的答案，測一測你是否真正了解男人？

選擇A

你並不了解男人

你並不了解男人，並不是一個特別通透的女人，你活在自己的世界裡，一直是一個小公主，只關心自己，只關注自己，你內心的壓力並不大，周圍的人比較寵愛你，所以你沒有吃多少苦頭，沒有承受太大的壓力，你也不願意了解男人，他們的心思很難猜，你要讓男人追著你跑，而不是患得患失，那和天生特別幼稚，根本控制不了男人，也不了解男人！

選擇B

你太了解男人了

跟其他人相比你太了解男人了，你是一個很聰明的女人，你知道如何控制一個男人，知道如何讓一個男人死心塌地，很了解男人心，知道他要什麼，能夠滿足對方，讓對方一心一意，在感情之中，你沒有吃過苦頭，沒有受過傷害，愛的轟轟烈烈，甜甜蜜蜜，他對你是真的好，也是真的幸福，一生無憂無慮，享受了榮華富貴，享受了美好的一生，男人會對你死心塌地，會愛你一輩子！

選擇C

你不想接近男人

你是一個很聰明的女人，並不想接近男人，並不想靠近他們，你覺得男人太壞了，會傷害女人，會讓你一個女人變得痛苦，並不能真正的帶給女人幸福，你是一個女強人，是一個大女子主義者，對你來說，生活的一切要靠自己，男人可有可無並沒有多少的存在感，所以你不想理解他們，也不想了解男人，遠離對方，才能真正活得開心，才能真正過得幸福！

此心理測試僅供參考，人的命運要靠自己掌握，這只是幫你更加了解你自己，一切還要靠自己，希望你未來成功，創造屬於自己的富貴，每一天開心、快樂、幸福。
