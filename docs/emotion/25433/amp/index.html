母親的生活能力像個孩子，家裡經常揭不開鍋，家不像家，母親走了

天天要聞
2019-12-20

昨夜，在夢中，媽媽走了，燈火通明，來來往往的人很多，卻不見媽媽，我四處尋找，大聲哭喊……

想想，媽媽這一輩子……

外公為躲避戰爭，逃到一個小山村，獨門獨戶，過著自給自足的生活。媽媽是家裡最小的，有外公外婆的疼愛，哥哥姐姐的庇護。按農村裡的說法，吃糧不管事，無憂無慮。

媽媽在16歲時，嫁給了我爸爸，從獨門獨戶的小山村，突然間到幾百人口的公社所在地的村莊生活，她適應不過來。那個連吃飯都成問題的年代，人人過著節衣縮食的生活，而媽媽，顯然是茫然的，手足無措。

孩兒不嫌母醜，多少年後，想著那個年代的媽媽，她的生活能力，就像個小孩。

近幾年，遠房親戚看見我，問起我媽媽的近況，我說媽媽多年前就去天堂了。親戚一陣唏噓，說我媽媽其實是個好人，沒有心計，要說她算計別人，是不可能的，只有……

媽媽在出嫁時，算是風光的，有一扇大紅油漆漆的大衣櫥，在我記事時，大櫥已是別人家了的。一位堂叔結婚，沒有新大櫥，堂叔的媽媽把她自己的舊大櫥和我家的新大櫥換了，送了一隻大公雞給我家。

小時候，我是媽媽的跟屁蟲。看著家裡沒有米下鍋了，媽媽就去借米，但她又不好意思空手去借，拿了我剛穿過一個星期的新棉襖到了一戶人家，說是那件棉襖就送給她家女兒穿了。我是個小孩，根本沒有能力阻止媽媽的行為，只有看看。以此仿效，媽媽變著法兒，拿著家裡的東西，換回借回能吃的食物，有些究竟是借的換的恐怕媽媽自己都搞不清，拆了東牆補西牆。爸爸經常在外搞副業，也無法知曉家裡的一切。

一位表姐說，小時候到我村裡做客，就喜歡到我家吃飯，因為別的親戚家都是吃蕃薯飯，只有我媽媽給她白米飯。我媽媽拿出家裡僅有的米燒了給她吃。而後，我們只有眼巴巴看著別人家有蕃薯飯吃，我家連蕃薯飯都吃不上了。

一個早晨，媽媽用飯盒裝了一盒白米飯，讓我送給在田畈里耕田的爸爸吃，爸爸問我早飯吃過沒有，我支吾著說不出話來，爸爸放下趕牛的毛竹梨俠，拉著我的手一起走回家。揭開鍋蓋，一看鍋里就是一鍋湯水和蕃薯，沒有幾粒米飯，爸爸很生氣的對媽媽說，知道家裡沒有米了，應該把那一盒飯和蕃薯一起煮，一家人一起吃蕃薯飯，要是他不回家，孩子們還不要都餓肚子？

一次，爸爸出門搞副業之前，到谷櫃里拿稻穀碾米。爸爸剛巴拉了寸把厚的稻穀，發覺不對勁，下面竟然是一件蓑衣，拿起蓑衣，下面是蕃薯絲，蕃薯絲下面是一些石頭。爸爸震驚了，發怒了……

長此以往，家不像個家，媽媽走了，爸爸帶著我兄妹仨，艱難過著日子。當時家裡窮得無法形容，廚房裡只留下一口燒菜的平底鍋，床上只有一床奶奶結婚時的黑不溜秋的棉絮。

後來，農村土地責任田開始包產到戶，爸爸辛勤勞動，勤儉持家，培養我兄妹讀書。大嫂結婚沒多久，鄰居提著一隻茶壺對我嫂子說，那隻茶壺是她和我媽媽一起買的，我媽早把茶壺送給誰都不知道了，她家的茶壺還好好的。我嫂子輕聲說，婆婆雖然不持家，兒女都聰明持家。嫂子沒有一句說我媽媽的不是。

前兩年冬至前幾天，我也夢見過媽媽，就是沒有說上話。我走在老家後山菜地邊，看見送信的郵遞員，問他我媽媽在哪裡，我找不到，他說他會帶我去找。我跟在郵遞員後面，在老家牆角轉彎處，看見媽媽拎著乾糧袋，一閃走進老家大院……
