  • 自幼隨著母親長大，我沒有見過生父一面。隨著逐漸長大懂事，我發現了自己與常人不一樣，很多同齡的孩子都不願和我玩耍。有時候我們幾個人玩的正盡興，別人的母親便呵斥她們回家，後來，也沒有人再願意接近我。
  • 關於生父，聽人說他接受不了一出生便患病的我。承受不了壓力便選擇了不辭而別。母親和我在一起，她親手送走了爺爺奶奶以後，我們兩個人更孤獨了。我也曾不懂事的問過母親。

“爸爸到底去哪了？他以後還會回來嗎？”

母親每次都以沉默不語來回應。父親一直都是我心中的疑惑。他究竟為何離開了家？為什麼一直不願回家？他真的不想我和媽媽嗎？

  • 有一日我突然心慌氣短，母親趕緊和我去了市醫院。因為我的病情特殊，院內開會準備請專家來會診。還沒等到專家來，由於躺在病床上多日，我腰部已經有局部皮膚開始潰爛。母親心急如焚，整日整日的哭。
  • 有一天夜裡，我突然心率異常，人也陷入昏迷。本院要求讓我們轉院治療。幸好有一表姐學醫，得知情況特殊，她聯繫了接手醫院的朋友。盡量爭取去了不耽誤時間。
  • 因為要趕夜路，也為了節約錢。母親找了臨時護送的車。車上也同意用一床他們備好的新鋪蓋，到了以後同車費一同付給他們。
  • 天亮剛好趕到，這個醫院有專門為急診病人開通的綠色通道。小姨交了押金推來床要送我走，可是我的母親卻和司機因為一床被子討價還價！母親意思這被子很新，就蓋了一晚上，我們不買了，讓他們帶回去。但是別人不同意，凡是用過的，就得買。再說，是我們同意買下來，別人才拆的包裝袋。母親覺得他們態度堅決，又說抹去被子零頭，只給一百塊錢。可別人說回去交不了差……
  • 那會的情況，對我來說分秒必爭。一同來的表姐過去給了被子錢，讓我們所有人趕緊推我去看醫生。雖然我人迷迷糊糊，但是這一幕幕感覺就是親眼所見……
  • 去了手術室的我，對外面的情況一概不知。只是後來聽說醫生當天下了三次病危通知書。我的母親一次次在醫院樓道里哭的尋死覓活！
  • 大夫讓繳費，她便退縮，支支吾吾，同去的親戚便都主動墊了很大一部分錢。後來我出院。親戚們又都來看我。可我母親絲毫不提還錢的事情。我們有醫保，手術費基本都報銷了，可她卻不願意還錢了。
  • 有一次我和母親溝通，想讓她主動還了叔叔嬸嬸親戚們墊的錢。她竟對我翻臉破口大罵。罵我吃裡扒外！溝通無果後，她對我經濟也卡的很死。除了交電話費，每月最多給一百塊零花……

很多時候，我都在想那天急診室門口的一幕。如果表姐不曾去付錢，或許母親還會像買菜一樣討價還價，不知道什麼時候才會想起還有個垂死掙扎的我。如果那天因為一床被子而耽誤了救治，多年後我的母親是否會後悔？或許不會，或許我那天在她心裡沒有一床被子值錢！

但造成這一切的根源是什麼？還不是一個單身母親的心酸。她拉扯我長大，又要照顧公婆。沒有什麼穩定的經濟來源，只能面朝黃土背朝天種地！我不曾知道她有多少次是恨不得一分錢掰成兩半花。只是這一生，她過得太辛苦。

後來才知道，她不願意還錢是怕我病情又一個複發，她只想留下這點救命錢……往事一幕幕湧入心頭，這些年我覺得她的摳門小氣，只不過是她對生活的無奈，更多的是心酸。而這一切，竟被我忽視。她有多少愛，藏於心，又有多少愛，在我眼睛看不到的地方熠熠生輝。

內心深處，總有一處溫暖我的小角落。那份愛質樸無華且無私！這一生，我都報答不完母親的恩情！只願時光能夠慢慢流淌，讓我傾我所能，給她一個好一些的日子！
