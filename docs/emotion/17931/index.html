善念從心開始，快樂如影隨行，你知道雪花的重量嗎？

2019-12-17 80條評論 1,664次閱讀 491人點贊

這是我的一次親歷。母親的話語很輕但卻使我幡然領悟，原來那個一直喜歡自我抱怨甚至懷疑的那個人是自己。

那天,我工作很累.回到家裡便開始對父母發牢騷。

本來，我是等他們來寬慰我的，可結果是兩人相互看了對方一眼，表現的出奇平靜：父親繼續聽他的收音機，面前是擺著一杯清清淡淡的白開水。而母親呢!則是在一旁，不急不慢地收拾著桌上的碗筷，叮囑孩子們吃飽了快快寫作業。

晚上，在空閑之餘，母親卻這樣的開導我說：

孩子，對於你今天的抱怨我們也不好說對錯，娘不了解你的工作，只知道做人和做事一樣講求個「本本分分」。母親的目光瞟了一眼窗外。她突然問我：你想過這窗外雪花的重量嗎？

當時，我很好笑：沒有呀，它本來就很輕呀，又哪來重量可言呢！

母親說，自己當年也是這樣回答外婆的。可在後來，她發現自己錯了，錯在忽略了生活那麼多的細節：

於是，在某個雪天，老樹底下多了一個仰望數雪花的小女孩兒，一片、兩片、三片、四片…

母親數呀數，晶瑩的雪花落呀落。直到九千零一十四片的時候，大團大團的雪花終於把樹枝壓得喘不過氣，她無奈的屈服了低下頭，而雪花們依然快樂著從母親的頭頂垂懸而下….

想想吧！恕自己一過，則萬過必從之而生。

更何況牢騷滿腹防腸斷呢？一個人如果從別人身上僅是看到太多缺點而非優點，那他自己一定也快成了一個無藥可救的人了，一朵雪花就好比我們心中的一個善念的想法，棄之一朵瑩白沒關係，但如果你一味地拋棄她太多太多，那你眼裡的世界也就成為黑色了，那裡沒有期待沒有喝彩更不會植下掌聲，不是嗎？

記著：善念從心開始，千萬別忽視一朵雪花的重量！
