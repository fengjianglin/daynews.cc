初中生：情侶頭像難被發現，高中生：我有情調，大學生：我最秀

雖然早戀是被學校禁止的，但是學生們都對校園戀情充滿了嚮往，連秀恩愛的方式也都是別具一格。快來圍觀初中生：情侶頭像不容易被發現，高中生：沒我有情調，大學生：我最秀兒！

01.在上中學的時候，老師和家長對早戀都是抓的比較嚴的。所以就算是一些戀愛的小情侶，秀起恩愛也算比較低調了，那會選擇那些比較不容易被發現的情侶頭像。畢竟自己的好友列表中也有自己的父母，在學校首飾也不能帶，但是也會在放學的時候戴上專屬於他們的情侶手鏈。

02.現在高中生早戀的也是比較多的，但是現在有一些比較開明的父母和老師，只要不影響學習。這方面也管的也不是特別的多，基本都是睜一隻眼閉一隻眼。所以他們在選擇頭像的時候也不會說特別注意，這兩個頭像放一起一看就知道是情侶頭像。他們還會選擇在休息的時候穿情侶服裝。

03.最後兩個的情侶頭像就是大學生們經常用到了到的了。看來大學生還是比較追求潮流個性的。用耳機繞成的人物做頭像。已經比較優秀了。但是用微信和qq的圖標做頭像，這也是沒誰了。估計不放一塊兒誰都想不到，他們用的是情侶頭像。真「蒂花之秀」本人啊！
