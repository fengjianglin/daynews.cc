“將來孩子和你一樣可悲”只因家長忘批改作業，老師群中指名責罵

2019-12-24 1,163條評論 3,334次閱讀 132人點贊

文 | 崛朗育兒（文章原創 ，版權歸本作者所有，歡迎個人轉發分享）

導語

家庭作業、課後作業是我們小時候，每天都要完成的事情，而每天回到家的第一件事情也是寫作業，先寫完作業才能去玩，先寫完作業才能吃飯。

家庭作業成了每個孩子的噩夢，因為完成它需要花費不少我們寶貴的娛樂時間。

孩子對家庭作業的厭惡，讓孩子很多時候想敷衍了事，這時候就需要家長們做好監督工作，監督孩子完成作業。

但是不知道從什麼時候起，監督孩子寫作業，甚至是批改孩子的作業，變成了家長們的工作。

很多老師的工作也從監督孩子的作業，變成監督孩子的家長們有沒有批改孩子的作業。

孩子的學習、教育家長們有必然的責任，但不代表，老師可以做「甩手掌柜」，把事情都丟給家長。

曾在網上看到了一位家長在微博上發的一條動態。動態的內容是幾張孩子家長群里，老師和家長的對話。

老師在群里責怪家長：「那些沒有批改孩子作業的家長，是二年級的口算自己都不會做呢？還是昨天晚上去掙了幾百萬，沒有時間批改孩子的作業」。

有的家長在群里反駁到：「改了呀，孩子說其它的不用了」。

還有位家長說到讓老師多多指點孩子，沒想到一位姓謝的老師卻說到：「你也看不懂嗎，學習沒有時間一一補回她的不懂。」

還有一位老師甚至反駁到：「二年級的題都不會，你現在的做法只能讓孩子走你的老路，將來孩子和你一樣可悲。」

網友們看完後紛紛評論：「想減負就丟給家長？」、「以前是孩子要挨老師罵，現在在家長沒有給孩子改作業也要挨罵」、「難道修改作業是家長的工作嗎？」……網友們紛紛對老師說法提出質疑。

不得不說老師們說的話確實過分了，難道給孩子答疑解惑，補缺補漏不是老師的本職工作嗎？

學校不也是孩子學習的地方嗎？不然幹嘛要把孩子送到學校去？雖然說家長們也有責任，但是老師的態度和說法不禁讓人感到詫異。

誠然，輔導孩子寫作業，監督孩子的學習是家長們的任務，但這並不全是家長們的的責任。

家長們能做的是輔助，傳授孩子知識，教孩子做題寫作的人，還應該是老師。

家長們參與到孩子寫作業中，能更好的監督孩子，讓孩子更好地完成作業，幫助孩子提升學習成績。

但是教育孩子學習的主要任務還是應該學校來負責，這也是家長們為什麼要送孩子去學校上學的原因。

現在越來越多的學校以及老師，喜歡給孩子布置家庭作業，同時也是在給孩子的家長布置任務。

為何愈多學校流行留家庭作業呢？

1. 家庭作業能讓孩子對知識有一個鞏固和複習

單是靠上課聽講，對孩子的學習來說是遠遠不夠的，現在孩子的課程越來越難，很多題目其實拿出來讓大學生做，都不一定會做。

這就需要孩子們花費更多的時間去學習，不僅是在學校的時間要學習，回家也需要一定的作業來鞏固知識。

正所謂「溫故而知新」，不管是對舊知識的複習還是對新知識的鞏固，都能讓孩子有所收穫，這就是家庭作業的作用。

2. 家庭作業也是老師了解孩子情況的途徑

老師講完課，想了解孩子一天下來的學習效果，近期的學習情況，其實靠的就是孩子的家庭作業。

老師們課業從批改檢查孩子作業的過程中，了解到孩子的掌握情況，也能根據情況改變自己的教學方式，給孩子更適合的教學。

3. 家庭作業讓孩子能花更多的時間去學習

孩子放學之後就像逃出圈的羊羔，徹底的放飛自我，想要讓孩子乖乖自己看書複習是很難的，這就需要家庭作業來約束孩子。

家庭作業讓孩子能夠在多花一點時間在學習上，其實這也是為了孩子自己。

4. 家庭作業其實也是老師了解孩子家長情況的方式

從孩子完成作業的好壞，就能看出家長有沒有關心孩子的學習。

老師和家長的聯繫其實並不多，除非是一些家長會或是其他的一些學校活動，不然老師和家長其實是沒什麼交集的。

老師不可能主動去和每個家長聯繫，家長們也忙於工作，而家庭作業就成為了老師和家長的溝通方式。

家庭作業並不是為了給孩子徒增壓力，家庭作業也是為了孩子好，想讓孩子多花點時間在學習上。

家長如何正確對待，學校的家庭作業呢？

1. 配合學校，監督孩子

孩子在學校有老師管，回到家之後，家長們就應該起到監督的作業，接班孩子的學習。

家長們要配合好學校及孩子的老師的教學工作，完成老師布置的任務，監督孩子完成家庭作業，好好複習和鞏固知識。

2. 認識到教育孩子，不是老師一個人的事情

很多家長會有一種錯誤的心理，就是孩子去上學了，那老師、學校就要負責孩子的一切。

不管是孩子的學習，還是人格教育，都是老師們的責任。

但其實學校的工作是讓孩子學習，並不是兒童託管所，更沒有義務去管理孩子的衣食住行。

3. 做好孩子學習的反饋工作

家長們要配合孩子的老師，做好反饋工作，及時的反饋孩子的學習情況。

學校的一些家長會、家訪活動，家長們要積極的參與，讓老師多了解孩子，能讓老師更好地幫助孩子。

4. 家長們對家庭作業要重視

給孩子一個好的榜樣，孩子的學習還是應該放在第一位的。

家長要重視孩子的家庭作業，才能讓孩子也重視自己的學習，家長的以身作則是很重要的。

這裡提醒大家，在孩子們完成作業的時候，家長也要避免在一邊玩手機看電視，要給孩子一個更好的學習環境。
