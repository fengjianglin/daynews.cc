趁著你還在大學裡，別等大學畢業“馬後炮”，人生哪來那麼多如果

2019-12-21 2,785條評論 3,817次閱讀 2161人點贊

今天，學姐的朋友來了，她高中畢業後，學醫，不是本科，是一家專科。

可是畢業後，工作也挺好找的，做醫護，不僅收入高，而且社會地位也不錯。

可惜她做了一個決定，她結婚了，一年後有了孩子，今天看著她喋喋不休和學姐說著孩子的事，婆婆的事，學姐根本插不上話。

頓時覺得非常陌生，她的生活就是做飯買菜照顧孩子做家務，學姐想聊聊工作，她無從談起。

結婚後，她沒多久就懷孕了，所以不得不辭職在家待產，生完孩子就很難再走出去工作了。

提起後，她也後悔當初如何如何，覺得自己現在的生活過於千篇一律。

所以今天還挺啟發的，寫了這個，談談畢業後選擇問題。

(1)無論你讀什麼大學，你都充滿了無數的可能性

之前學姐也寫過，二三流的大學或許會讓你畢業後選擇變得艱難一些，但是不會決定你的未來。

因為我們上了大學都知道，大學裡對於成績的要求不高，只要不掛科就可以了。

除非個別情況，在一中極其自主學習的環境下，個人的選擇變得尤為重要。

所以基本上大學都是靠自己。還是那句話，不要因為自己的大學而放棄自己。

要知道好大學裡，努力的都那麼多，你大學不出名，自己也不考慮未來，不努力，家裡沒礦。

你想做什麼？能做什麼？

可能給初中生，高中生說這些他們會反駁你，但是咱們都成年了，就不用多說了吧。

(2)學習不是唯一的出路但是一定是大趨勢

很多人會覺得，我就算讀的書不算多，學歷不高，我做什麼都行。

360行，行行出狀元，但是，這個社會最基本最底層的要求你得滿足吧。

比如我朋友，有朝一日，她想通了，能夠放下孩子出去工作，最起碼她有醫學專業證明，有最低學歷標準的畢業證。

所以，如果你現在讀專科，學姐希望你們有機會有條件一定要好好學習專升本。

因為現在很多行業入門門檻就是本科，所以你可以不拚命，但你不能因為自己一時思慮不全，把自己未來的後路都斷掉。

你可以因為一些原因，沒有達到某些高度，但最起碼你保證在你想做一件事的時候，你不會被限制。

(3)大學入門就是競爭

不用給自己找太多借口，等到畢業了說後悔。

我大學時如果……就好了，這不就是我們所謂的「馬後炮」嗎？

在大學裡，看別人努力，自己干著急。

定了無數計劃卻沒有行動，等畢業了自己生活過得不好又開始抱怨，就是一個很廢的人。

我不知道你們身邊有沒有這種人，自己不學還想把別人拖下水，自己不努力還酸別人。

所以你身邊有這種人，需要你有強大的自制力，去守住自己的底線，不被同化。

（4)畢業工作需要技能不是理論

入職後，你就會知道，工作里就是少說話多做事，不需要你有很多花里胡哨，你做的多給公司謀的利益多，領導器重你。

這就夠了，沒有人會看不起你。

其實對於關係社會，一些「關係戶」憑藉捷徑也得到不錯的工作。

其實最難受的不是你找不到好工作，而且你父母親人想盡辦法給你找到的好工作，你卻沒能力勝任。

在別人眼裡永遠都是一個「關係戶」，而不是通過你自己的努力，讓別人知道，就算你走捷徑，你也一樣有能力可以做好。

所以趁著在大學，好好找機會鍛煉實戰，不要嘴巴一套一套實際什麼也不會。

（5）趁著在大學正視自己的三觀

走出校門，你很容易被社會洗腦。

比如「學的好不如嫁的好」，其實現在很多人已經醒悟了。

也都知道了，有錢人不傻，娶你一定因為你在某方面突出，或是外貌，或是才華，或是家境。

所以天天做著白日夢而不行動的人，如果畢業後過得不好，你沒資格抱怨，因為是你自己的選擇。

而且一定要正視自己的想法，男女生都一樣。

不論怎麼樣，要有自己的工作，即使你不缺錢，你最起碼保證自己能夠不和社會脫節。

工作不僅為了賺錢，也是為了讓你的大腦每天轉著。腦子不用真的會銹的。

（6）在大學裡還有時間學會提升自己

提升自己除了提高專業能力，也要學會提升氣質。

可以看看推薦的好書好電影開闊自己的眼界，學會打扮和化妝，懂得各個場合的衣著如何得體大方。

試著堅持運動，給自己一個好身體，因為工作後再想堅持運動是很難的。

所以在大學裡有時間就多運動運動，多讀書，相信「腹有詩書氣自華」，你讀書多了談吐和氣質都會不同。

總之呢，大學裡大好年華，遊戲可以玩，旅遊可以逛，可以剁手。

但是一定不要把所有時間都浪費在放鬆娛樂上，因為四年彈指一揮間就過去了，畢業後你面臨的問題太多了。

不要因為自己荒廢了四年，畢業給自己留下滿滿的遺憾。

不做後悔黨，不做「馬後炮」。四年不荒廢，青春無後悔！
