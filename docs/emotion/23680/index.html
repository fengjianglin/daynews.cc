公交車上奶奶抱熟睡孫女半小時，周圍無人一人讓座，網友：好心疼

2019-12-19 1,747條評論 4,310次閱讀 355人點贊

本文由Ainee媽咪原創，歡迎個人轉載分享

中國是禮儀之邦，我們從小就被家長、老師教導要尊老愛幼，內心要善良，但是不知道為什麼隨著年齡的不斷增長，內心卻變得越來越冷漠。

「爺爺奶奶年紀大了，要站起來給他們讓座呀！」類似於這樣的話，大部分人小時候都聽過，還記得那個給爺爺奶奶讓座都能開心好久的自己嗎？

前兩天看到一個視頻，有一位奶奶看起來60歲左右的樣子，要去接自己在上小學的孫女放學，但是學校離家比較遠，需要坐很久的公交車。

往天也就算了，但是今天孫女放學的時候就跟奶奶說自己不舒服，頭有點疼，奶奶本想著帶孫女打車回家，可是這個時間段根本攔不到計程車，沒辦法奶奶只好帶著孫女坐上了回家的這趟公交。

孩子的放學時間很不巧碰上了這座城市的晚高峰，上班族紛紛拖著疲憊不堪的身體擠上了公交車，等著祖孫倆上公交車的時候已經沒有座位了，索性公交車上站著的人還算比較少。

眼見著孫女難受得不行，看著周圍乘客疲憊的臉，深深的嘆了一口氣，就選擇了一個小角落坐了下來，而自己則抱著孫女，讓孫女眯一會。

可能是因為孩子太難受，沒一會孩子就睡著了，奶奶也著實是心疼孩子，壓根不忍心叫醒孩子，就保持著這一個姿勢一動不動，這一待就是半個小時。

公交車裡的乘客上上下下，卻沒有一個人打算起身這對祖孫倆讓個座，周圍人該低頭玩手機的玩手機，該看窗外思考人生的思考人生，而奶奶的動作其實很明顯的說明自己已經堅持不了多久了。

這段視頻上傳到網上，網友們的態度卻不盡相同

西風：好心疼這個奶奶，為什麼周圍人不能讓個座啊！

我是條鹹魚：上班族也很累呀！而且上班族可能離家更遠呢，難道要站一個多小時到家嗎？

今天晚安：說實話在我看來直接坐在地上也挺舒服的，再加上誰也不了解情況，這奶奶又不說，大家都累一天了，周圍人怎麼可能就給你讓座。

我姓弓長：我之前每次都給周圍人讓座，可我懷孕的時候壓根就沒人給我讓座，自那之後我就知道我善待別人，別人不一定善待我，從此就決定再也不給人讓座了！

讓座是一種美德，作為家長我們應該將這種美德傳承下去，在日常生活中我們應該教會孩子哪些品德呢？

1. 教會孩子要感恩

孩子在成長過程中離不開身邊人的幫助，無論是父母、老師、朋友甚至是自己的競爭對手，每一個人對其的幫助都是很重要的。

假如孩子沒有感恩的心，那麼與人相處中就會帶有太多的功利心，認為人與人之間是相互利用的關係，抱著這樣的心情是沒有辦法獲得真正的朋友的，因為任何關係都是相互的，沒有人願意永遠付出而得不到回報，漸漸地孩子就會變得越來越孤獨。

而懂得感恩的孩子，他會記住周圍人對自己的好，會認為自己身邊的人是善良的，對每個人都抱有善意，會覺得自己生活得非常幸福，當付出有了回報，自然而然身邊的人也會很喜歡他。

2. 教會孩子要與人為善，互相幫助

老話說「人之初、性本善」，家長在日常生活中要教會孩子與人為善，如果可以幫助別人的時候最好能夠伸出援手。

生活就像是一面鏡子，如果你看這個世界是好的、陽光的，那麼你的生活也會是陽光的，即使是出現一些小意外，孩子看到的往往也是這件事情積極的好的一面。都說「贈人玫瑰，手有餘香」，當別人需要幫助的時候滿足的不光是對方，也有自己的內心。

之前聽說過一句話「你在凝視深淵的時候，深淵也同樣在凝視著你」，當孩子的內心是陰暗的，那麼他所看到的每件事情都不自覺地會從陰暗的方面想，時間長了孩子會變得冷漠，不相信任何人。

3. 教會孩子尊老愛幼

尊老愛幼其實每個家長都會教給孩子，但是家長作為孩子的榜樣和模仿的對象，在日常的生活中一定要謹言慎行，假如家長本身對於長輩就十分不尊重，那麼即使是跟孩子說再多遍的道理也沒有用。

家長在日常生活中可以給孩子講一講尊老愛幼的小故事，如果情況可以的話也可以鼓勵孩子進行實踐，讓孩子體會這種感覺。

其實關於該不該給人讓座的問題，因為每個人的生活環境和背景不同，其實並沒有一個明確的說法。但最重要的是我們這種傳統的美德並不應該被丟棄。
