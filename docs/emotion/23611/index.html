女子替烈士未婚夫照顧母親40年：“你為國捐軀，我替你盡孝！”

2019-12-20 723條評論 2,447次閱讀 855人點贊

40年前，袁和菊的獨子在戰場上犧牲了，她把兒子的遺像擺在枕頭邊，整日以淚洗面。繼喪女、喪夫後，她不得不接受喪子的殘忍現實。

40年後，90歲的老人面色紅潤，會噘起嘴孩子般地挑剔，然後一笑就露出慈祥。從她身上，似乎已很難窺見從前痛苦和絕望殘留下的影子。

讓老人幸福感洋溢在臉上的功臣，是誰？是兒子的未婚妻，是精心奉養烈士母親、一句「媽媽」一喊就是40年的袁玉蘭。

袁玉蘭（右）悉心照顧袁和菊（左）


假媳婦 真母女

重慶市萬州區武陵鎮，將軍路57號。袁玉蘭前幾天替袁和菊洗的圍巾晾乾了，她正要往衣櫃里收。

圍巾整齊地搭滿了兩個衣架，足足有16條之多。「全都是袁婆婆的嗎？」袁玉蘭笑著點點頭：「老太太臭美著呢，還挑。有些是我給她織的，有些是我買的，還有些孫女給她買的。」

90歲高齡的袁和菊耳背，當年沒日沒夜的哭泣讓她的左眼至今不太能視物。吃飯，袁玉蘭拿著飯碗跑到袁和菊耳邊叮叮咚咚地用筷子敲；這邊袁和菊剛低啞地咳了一聲，那頭袁玉蘭趕緊遞來了垃圾桶。

不知情的人，定會認為這是一對再親不過的母女。當年那一句「你為國捐軀，我替你盡孝」的承諾背後，是跨越血緣的動人大愛。

▲家裡有規矩，老人每頓都吃新鮮飯。

「嫁人可以，我得帶著媽」

上個世紀70年代，忠縣石寶鎮姑娘袁玉蘭和小學同學、同村小伙張宜華確定了戀愛關係。1975年，20歲的張宜華應徵入伍。兩個年輕人許下愛的諾言——退伍之時，便是成家之日。

4年後，張宜華在對越自衛反擊戰壯烈犧牲的消息傳回。兩個女人流不盡眼淚。替未婚夫照顧好母親的承諾猶在耳邊。不顧所有人反對，袁玉蘭頂著閑言碎語的壓力，毅然搬去和袁和菊同住。「他為國家奉獻了生命，是英雄。他的母親，是英雄母親，我也該承擔起我的責任，奉獻我的力量。」

操持家務，忙裡忙外，袁玉蘭沒叫一聲苦。她的付出，最先看不下去的，是袁和菊的親弟弟：「玉蘭才26歲，給她找個好婆家吧，莫耽誤了別個姑娘大好的青春。」

袁玉蘭極力反對。「我嫁人了，媽怎麼辦？」最後袁和菊也加入勸說隊伍，袁玉蘭終於鬆口，提出了唯一的要求：「嫁人可以，我得帶著我媽。」這個條件近乎苛刻，但萬州武陵鎮的彭國政欣然同意。

彭國政老實本分，至今說起和妻子的相知相遇，他憨厚一笑，認為是自己的福氣。「她是個有情有義的好女子，當年就被她的善心和孝心打動。」

「只是多一副筷子。」

袁和菊自己也不知道，究竟是多了女婿，還是多了個兒子。但夫妻二人的付出，她卻看在眼裡。回憶起當年的日子，彭國政只說辛苦。袁玉蘭一旁快言快語地補充，「是你們想像不到的，實在是太苦。」

一家五口擠在20多平方米的小土屋內，老人帶著孫女睡一張床，夫妻倆摟著兒子睡另一張床。上有老下有小的負擔甚至讓一個女人在水碼頭扛起一百多斤重的包袱，做起和丈夫一樣的搬運的工作。

後沒後悔過？這次彭國政答得很快「只是多一副筷子。如果後悔，我當初根本就不會接媽回武陵。」袁玉蘭和丈夫相視一笑，然後袁玉蘭輕輕地說了一句，「所以我說，『軍功章』絕對有他的一份。」

袁和菊像個被寵愛的孩子。記者輕輕地握了握她的手，沒有想像中農村婦女該有的粗糲，掌心手指尚很柔軟。「今年90歲了，牙口不好了。」袁玉蘭笑言雞蛋沒有白吃，「你看我媽這個紅潤的臉色，皮膚上連老年斑都沒長几個！」

久病床前也有孝子

「老人家能這麼長壽，真的多虧兩口子照料得好!」椅城社區書記周青瓊告訴記者，她們印象中，袁和菊唯一一次生大病，是4年前去街上看戲時不小心摔斷了股骨。住院20多天，一直是袁玉蘭陪侍病床前，老人喊痛，她耐心哄，擦拭身體，喂飯復健從不假手他人。

彭國政也沒有閑著，他負責做飯。老母雞、鯽魚換著花樣給老人補身體，連主治醫生鄧紅平都驚訝於老人的恢復速度。

直至出院，同房的病友誰都沒有懷疑過，這對為老人忙到腳不沾地的夫妻，其實和老人沒有任何血緣關係。

▲一家人的幸福洋溢在臉上


唯一心愿：帶母親掃墓

五口之家三次遷移，但不管搬到哪裡，除了始終被帶在身邊的袁和菊，牆上張宜華烈士遺像彷彿也約定俗成，一直被掛在家中進門最顯眼的位置。

平日，老人愛顫顫巍巍地踱步到照片下，仰頭看看兒子的戎裝照，微笑說：「你，你啷個爬到牆上去了啊?」聽到袁和菊說這話，袁玉蘭忍不住眼眶發紅。「以前媽一看到照片就哭，現在，她至少笑得出來了。」

前年，戰友帶來了張宜華的墓碑照片。38年後，烈士母親總算知道兒子的歸處、陵園具體位置：雲南屏邊。「我現在唯一的心愿，是希望能在有生之年，帶我媽去雲南為張宜華掃一次墓。」

▲袁和菊拿著兒子的遺像，仔細地瞧。

「她永遠是我的親人，她需要我，我也需要她。她是英雄的母親，也是我的母親。」袁玉蘭並不善言辭，她就是一個普通的農村婦女。她緊緊地握著袁和菊的手，「照顧她是應該的，我們要為她養老送終，讓她長命百歲。」
