夫妻收留單親媽媽三個孩子，不久單親媽媽去世，夫妻小房變成豪宅

夫妻收留單親媽媽三個孩子，不久單親媽媽去世，夫妻小房變成豪宅

國外有這樣一對夫妻，他們一共有五個孩子，最大的今年十八歲了，而最小的孩子才剛剛上幼兒園。即使是在國外經濟水平都還可以的情況下，照顧這麼一家子人也是很不容易的。

妻子是一個百貨公司的收銀員，丈夫是資產保安員。他們每天都在為生計而煩惱，所以根本沒有時間去關心周圍的人和事物。他們甚至連周圍住的鄰居都不知道是男是女。

直到有一天一個自稱是他們鄰居的女人來敲了他們的門，並問他們能不能收留她的三個孩子時，他們才第一次看到自己周圍的鄰居。

原來鄰居是一個單親媽媽，一個人帶著三個孩子，才搬來這個地方不久，所以對周圍的一切都不太熟悉。可是她知道自己的鄰居家裡面有五個孩子，所以就不得已提出了這個無理的請求。女人說這是她最後的願望，她在前不久被查出來得了癌症，可是她沒有聽醫生的話呆在醫院裡面接受醫療，因為她想要和自己的孩子利用最後的一段時間好好的相處一下。

可是現在她知道她的病已經越來越嚴重了，她感覺自己已經沒有多少時間了，可是她的孩子還是那麼的小，那麼的可愛，即使她不願意離開這個世界，她也不得不離開了，她唯一放心不下的就是自己的三個孩子，於是就想要給他們一個好的歸屬地。

而無意間知道這對夫妻家裡面有五個孩子，肯定是很有照顧孩子的經驗的，所以她才想到了這個主意。妻子雖然是一個被生活折磨得已經面目全非的女人，可是她仍然是一個媽媽，她看著女人蒼白瘦弱的臉，臉上滿滿的都是懇求，她知道一個女人如果遇到這種事情的心情肯定是不好的，所以妻子心軟了，於是說服了自己的丈夫收留了鄰居家的三個孩子。

後來沒有多久女人果然就去世了，而這對夫妻也履行了自己的承諾收留了這三個孩子，可是原本就有五個孩子的家庭因為又多了三個孩子所以顯得更加的擁擠了，而且多了三個人以後經濟壓力也是巨大的。

夫妻兩個人為了養活這個家庭不得不更加的拚命了，可是就是前不久他們發現經常有一些鄰居來到他們家裡給孩子送一些吃的東西，而且最近他們還給這對夫妻送了很大一筆錢，聽說是大家一起湊的，因為他們得知了這對夫妻收養了三個沒有血緣關係的孩子。

他們覺得這對夫妻真的是高尚的人，他們被夫妻倆的做法給打動了，所以就一起湊了一次不小的錢。而拿著這筆錢夫妻倆也從小房子變成了豪宅，所以這就是好人有好報吧，最近政府也知道了他們做的善舉，所以也準備對他們進行獎賞，這個家在社會各界的幫助下已經變得越來越好了。
