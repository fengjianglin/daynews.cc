結婚率持續走低，離婚率繼續上漲，恰恰說明這是愛情最好的時代

2019-12-23 1,217條評論 3,576次閱讀 1993人點贊

據2018中國統計年鑒顯示，我國單身人口規模已達2.4億。近年來我國離結率也已達到37%，某些城市甚至高達50%。而結婚率卻連年下降……

婚戀市場相當蕭條啊，難道是大家都不再相信愛情了嗎？不，恰恰相反，離婚率高、結婚率低，單身人口持續增長，恰恰說明這是愛情最好的時代。

為什麼？因為大家終於可以只為愛情結婚了啊！

以前我們成家，很大部分原因是生產力低下，一個人，特別是弱女子，難以獨活，那麼建立家庭、相互協作，是生存的最優途徑。所以無論有沒有愛情，幾乎都會結婚。

此外，養兒防老也是原因之一。因為在過去，沒有養老院，沒有退休金，人上了年紀喪失勞動力後，幾乎全靠子女給一口飯吃。養兒防老是剛需，結婚生子也是剛需。

最後就是傳宗接代的問題，因為深受「不孝有三，無後為大」思想的影響，很多人結婚真的不是因為愛情，而是為了傳宗接代完成任務，不結婚會在道德上備受譴責。

而現在，作為以前支撐婚姻的必要條件：協作生存、養兒防老、道德譴責，在我們這一代已然難以成立、甚至轟然倒塌。

為什麼？

因為隨著社會發展，養老體系日趨完善，養老金普及，養老院增多，人們思想越來越開明，即使有子女也不想給孩子過重負擔，養老全憑自己的實力……所以，養兒防老已經不再是剛需。

其次現在的年輕人，隨著教育程度的提高，思想開放度的提升，對自我價值實現的追求重於結婚生子的追求，「不孝有三，無後為大」已經很難降服他們了，他們只會呵呵一笑「為什麼一定要生孩子，難道家裡有王位要繼承嗎？」

最後是一個人也可以活得很好，沒必要協作生存。滿足基本的物質需求不難，不過一張床一日三餐的事。而在精神上也不會像過去一樣孤獨，因為網路把所有人連結在了一起，而你總能找到自己的圈子。

那麼我們還有結婚的必要嗎？除了愛情我想不到其他更好的理由了。

在《家族的形式中》，男主對女主求婚的時候說：「我一直覺得結婚這個制度本身就大有問題，共享二人時光什麼的說起來好聽，但本質還是剝奪了一個人獨處的時間吧，所以我也不是那麼想結婚。雖然不是那麼想結婚，但我一生中或許再也遇不到這樣合拍的人了，這或許是一個小小的奇蹟，但是既然我們相遇了，恕我冒昧，與其 一個人過，要不要試著兩個人 一起生活。」

是的，我們都是一個人也能過得很好的生活，但是我們難得相遇，愛情更是一個小小的奇蹟，不結婚就略顯可惜了。

可現在離婚率那麼高，愛情還可信嗎？

2018年我國離婚登記大數據顯示，離婚的最大原因是感情不和，77.51%的夫妻因感情不和向法院申請解除婚姻關係，也有14.86%的夫妻是因為家庭暴力離婚，也有許多現代人婚姻的開始是盲目的、衝動的，出於感情用事或生理衝動，甚至奉子成婚。熱戀時期的隱忍和遷就，終會在平淡的婚姻里爆發。他們最終發現自己真的選錯了，便會及時糾錯，重新選擇。

其實離婚率高，也恰恰說明我們終於不再將就沒有感情的婚姻了，在過去離婚遭受的歧視遠比今天的大，很多人即使活得痛苦也不敢輕易離婚，現在社會對離異者包容度則大了很多。

所以你看，這難道不是愛情最好的時代嗎？

最後，愛情是否始終值得追求？因為愛情結婚是否有必要？

李銀河說：「人生在世，最美好的生存狀態是沉浸在愛之中。因為吃喝拉撒只是簡單的生理活動，毫無美感可言，有些甚至是醜陋的。絕大多數勞作不過是為了謀生，也毫無美感可言。當然，創造性的勞作除外，它可以為人帶來愉悅和美感。這樣找來找去，只剩下愛，只有愛是人最美好、最純凈、最有趣的生存狀態。」

而在《無問東西》中，沈母對兒子說：當初你離家千里，來到這個地方讀書，你父親和我都沒有反對過，因為，是我們想你，能享受到人生的樂趣，比如讀萬卷書行萬里路，比如同你喜歡的女孩子結婚生子。注意不是給我添子孫，而是你自己，能夠享受為人父母的樂趣。你一生所要追求的功名利祿，沒有什麼是你的祖上沒有經歷過的，那些只不過是人生的幻光。」

愛情是人生最好美好的體驗之一，和心愛的人結婚生子也是。我們之所以選擇結婚，是因為遇到了和我們一起墜入愛河的人。

我們之所以遲遲沒有結婚，是因為不能將就，畢竟愛情是我們在這個功利的世間，尚有餘力可以保存好的最美想像之一，不能自毀。

而這個時代，好就好在，我們終於可以光明正大地不必再將就。
