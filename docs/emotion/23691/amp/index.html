尷尬的人情冷暖：鄰里之間關係沒處好，酒席擺好沒人來

現在的農村幾乎成了空殼村，只有少量的老人和孩子留守，大多數人都在城市裡打工，一年裡只有過年過節才會回來一兩次，回來也沒有幾天時間在家裡,就又要回城裡上班，那麼在村裡，鄰里鄉村的有什麼紅白喜事，大多數人都沒時間去參加，不在同一村的親朋好友，那就更顧不上了。因此，不管是與村裡人還是親戚朋友，之間的關係也就慢慢的疏遠了。

其實在農村也比較喜歡攀比，農村人愛熱鬧，每當有喜事的時候，人越多，就說明他越有面子，在村子裡也有實力，農村裡面老人過壽等喜事還是比較多，光每個人做壽這件事，一輩子都可能要三四次，現在都出門在外，如果你有一兩次沒參加，也說得過去，但是有些人在外打工十幾年，甚至二十幾年，村裡的那個紅白喜事，幾乎一場都參加不了，慢慢的村裡人都把他們忘記了。

等到這些在外打工的村民，他們的子女20多歲的時候，到了結婚的年齡了，他們就想在農村為子女辦一場熱熱鬧鬧的婚禮。因為他們常年在外，也沒有去參加人家的喜事，那麼這個時候來給他慶祝的人也就很少了，雖然準備了很多酒席，但來的人少，這樣就很尷尬了，感覺自己也沒面子，還鬧了笑話

中國有句古話叫做”遠親不如近鄰”，如果沒有把鄰里關係搞好，很多事情就不好辦，生活在同一個村子裡面的人，當別人有喜事的時候，如果你人到不了但禮也一定要到。現如今大家都在外面為了生活而奔波，沒時間來，大家也能夠理解，但是你禮都沒到，你們的關係以後就難處了。以前農村只要哪家有什麼紅白喜事，大家都是會一起過來幫忙的，現在的農村，這種氛圍好像很少看到了。
