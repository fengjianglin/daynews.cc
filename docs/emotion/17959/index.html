我一直很納悶，為什麼有的人一個勁傷害別人，自己反倒喊疼

2019-12-16 114條評論 1,240次閱讀 99人點贊

親愛的陌生人，有幸你來，不負遇見

有一天，我會指著心臟，驕傲的告訴你，這裡，拆遷了，沒地方讓你住了。

得不到回應的熱情，要懂得適可而止。不虧待每一份熱情，不討好任何的冷漠。

一顆心要傷多少次，才會被迫選擇放棄；一個人要傻等多少回，才知自己只是多餘。如今冰封的心，曾經是最熱烈；如今無情的人，曾經是最深情。

我以為只要很認真地喜歡，就可以打動一個人。卻原來，我只是打動了我自己。

太多的事，慢慢地就不能做了，太多的人，漸漸地就不見了。原來，成長就註定是一個要丟失的過程。

後來，一杯酒能解決的事，也就不想再浪費眼淚了，喜歡上一個不屬於自己的人真難受，連吃個醋都名不正言不順。

我一直很納悶，為什麼有的人一個勁傷害別人，自己反倒喊疼

世上的女人只有兩種，一種是幸福的，一種是堅強的。幸福的被捧在手心裡無需堅強，而堅強的卻是不得不堅強。
