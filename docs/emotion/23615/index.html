時間，‎讓深的東西越‎深，讓淺‎的東西越淺 #語錄#情

1.胸襟的寬窄，決定命運的格局，你能包容多少，就能擁有多少。凡事看得開、想得透、拿得起、放得下，學會隱忍性情，懂得剋制慾望，退卻時理智，謙讓時大度，你的天地才會壯闊遼遠；唯有萬事先修德，養性必制怒，刪除繁枝末節，除卻雜念私心,多些隨緣自適，少些攀比計較，人生方可行雲流水，成大風景。 ​​​

2.要以微笑面對生活，遇上困難才能從容。只有懂得享受生活和尊重生活的人，生活才會帶給你樂趣和希望！正能量和負能量就像乘法，正正為正，負負得負，正負還是負。是有傳遞性和感染性的！生活亦是如此！

3.心胸開闊，世事看淡，大事聰明，小事糊塗，恩怨不計，舊恨不記。相逢一笑泯恩仇，心中坦蕩蕩，人生最自在。人心簡單就快樂，生活簡單就幸福。人生苦短，別省、別等、別放、別忍、別恨，風雨一輩子、開心最重要。

4.有些人相遇，不是為了結局，而是為了讓你找到最大的障礙，其實是我們自己。此去經年，未能化蝶飛舞，也無憾。靜守流年，將每一天都過成詩。

5.心中有了愛，才能去包容對方的缺點，即使在沮喪失望時，依然深愛對方，這才叫做真愛。

6.‎時間，‎讓深的東西越‎深，讓淺‎的東西越淺。‎守護彼此純真‎的心，‎簡簡單單的去過‎好每一‎天。無懼‎時光時光消逝，‎它只能見證‎我們‎的情，為‎我們相‎依的臉頰刻‎上一道道吻痕。‎讓我們‎記住靜寂時‎光里的‎陪伴，孤‎獨時的守候。生‎活有陽光，‎亦有‎你的陪伴，你的‎情，你的‎溫暖。

7.距離常生美，無間的親密只會令雙方窒息，無論朋友還是愛人，別愛得太近。愛的藝術就像風箏，只有給它風一般的自由，你才會看到它飛舞在藍天的景緻。

8.拚命對你好的人，一瞬間抽離然後冷淡，就算這是你不喜歡的人，你都會犯賤地對他產生了感覺。在一起是一種緣分，不要輕易讓喜歡你的人哭泣傷心，好好珍惜在一起的每一分鐘，美好的回憶應該留給快樂和歡笑。

9.君子圍爐煮雪，三兩趣味，在清談之間，四五閑情，寄裊裊茶煙，六七心結，作庭前吹雪。人生不如意十有八九，且把一世夢幻泡影，換了爐煙碗茗。待光陰散去，風令人閑，茶令人醉，恰是一爐暖意，最人間。 ​

10.能忍著想踹人的衝動,還很淡定的和討厭的人聊天,就是在成長.

11.遇見你，我變得很低很低，一直低到塵埃里去，但是我的心是歡喜的，並且在哪裡開出一朵花來。雨聲潺潺，像住在溪邊。寧願天天下雨，以為你是因為下雨不來。

12.去見想見的人吧，趁陽光正好，趁微風不燥，趁繁花還未開至荼蘼，趁現在還年輕還可以走很長很長的路，還能訴說，很深很深的思念，世界還不那麼擁擠，趁飛機還沒有起飛，趁現在自己的雙手還能擁抱彼此，趁我們還有呼吸。
