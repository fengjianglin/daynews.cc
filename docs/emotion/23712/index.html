解讀上世紀的非法婚姻“拉幫套”：兩男一女的世界，可悲又可憐

2019-12-19 251條評論 3,248次閱讀 2308人點贊

東北地區有句俗語叫，「招拐子養崽子，崽子大了打拐子」，但什麼是拐子呢？

這就要從上世紀六七十年代之前在東北鄉間悄悄遊行的一種非法婚姻「拉幫套」說起。

「拉幫套」的原意：馬拉車時，害怕一匹馬拉不動，就在旁邊套上一匹馬來幫助拉車。

「拉幫套」式家庭

到了上世紀七十年代，「拉幫套」的本意發生了變化，代表了一種非法婚姻形式：

當丈夫患重病不能撫養妻子兒女時，在徵得丈夫同意後，會另招一個體魄健壯的單身男人進門，擔負照料全家生活的重任，同時，也可以與患病男人分享他的婚姻。

而這位被後招進門的男人，就是「拐子」。

因為這種婚姻形式不受法律保護，所以常常會發生「拐子」擔負起家庭負擔，幫男主人養大孩子後，最後被孩子「掃地出門」的凄慘下場，這才有了上文那句「招拐子養崽子，崽子大了打拐子」。

同題材的影視劇

在農村這種閉塞的環境下，「拉幫套」的當事人常常被人說閑話，孩子也受到排擠，時間一長肯定會出現問題。

但同題材的影視劇《三個人的冬天》卻沒有像現實生活中的陰暗面發展，而是留給了觀眾以無限美好的想像空間……

故事要從上世紀四十年代末的長白山林區說起，伐木工人魏大山為了救人自己被砸成重傷而癱瘓，一家人的生活日益艱難。

走投無路的魏大山想起了自己的徒弟黑塔，希望他來家「拉幫套」。心地善良的黑塔沒有趁人之危，沒有答應「拉幫套」，卻仍願意幫助師父一家人。

自此他背著魏大山四處求醫，但他的善良吸引了魏大山妻子云鳳。

這樣魏大山很是氣憤，於是他開始虐待妻子，卻不知此舉反而將雲鳳推到了黑塔那邊。

直到有一次，黑塔看到師父虐待雲鳳，他忍無可忍打了師父一頓決定離開。

誰料，雲鳳已經懷上了黑塔的孩子，黑塔只能繼續留下。

後來，他們聽說熱水泉可以治病，便每天拉著魏大山去洗溫泉，最後魏大山真的痊癒了。

恰好此時，新《婚姻法》開始實施，「拉幫套」這種舊習俗畢竟結束。最後魏大山扛著行李消失在了雪地里……

這個結局是編劇和導演留給觀眾的美好念想，可真實的拐子結局卻大多悲慘……
