昨天和兒子短暫的相處了一晚，然後又急匆匆的分開了。

自從母親生病後了，我和老公就把兒子送到了爺爺奶奶家，此前從未和兒子分開過。然後開始了每天晚上接視頻的畫面。

大伯哥有個女兒6歲了。此前穿過的很多衣服都還很好，所以我公公就把能穿的整理出來給我兒子穿，於是我經常在視頻里看到兒子穿得像女孩的樣子。此次見面，是因為我們從廣州回來，已經沒有到銅仁的票了，於是臨時決定買了貴陽的票。走進屋的那一刻，兒子和姐姐正在玩玩具，我喊他沒有反應，走過去抱他都要哭，快一個月沒見面了，已經有點生疏了，哄了好久才要我抱的，作為一個母親，當時非常到難過。

吃晚飯的時候，才主動走過來讓我抱著喂飯，那麼久沒見面，兒子又重了點，就是沒怎麼長個。

依舊是搗蛋的樣子，沙發上到處都是他的玩具，翻來翻去，爬來爬去，手腳並用，一天忙不得很靜不下來。臉上有小塊淤青，聽他爺爺說是撞到桌子邊上了。右手還被爐子燙傷了一小塊。我心疼得很久說不出話。離開的時候，他在家裡的稱上站著，一直去按那個稱上的按鍵，我喊他都沒反應。

爺爺奶奶年紀大了，可能有時候會有點照顧不那麼周全，我很感激他們在用心的給我帶著。稍微磕著碰著也是很正常的事。但是真正看著還是有點難過的。

因為母親現在正處於化療期，沒每個月都要往返銅仁—廣州一到兩次，所以沒辦法把兒子貼身帶著。

人生總有那麼些不如意。寶寶對不起，等外婆稍微好點了，媽媽就去接你！
