父親賣血養大兒子，不買40萬豪車兒子砸碎電視，父親：掃馬路掙錢

天天要聞
2019-12-20

愛孩子是件好事，但是過度的愛很可能會變成溺愛，溺愛屬於家長對子女一種畸形的愛，也是一種失去理智，直接影響孩子身心健康發展，這不，34歲男子正當而立之年卻毫無上進之心，還在依賴父母，得不到滿足就不起床不出門，這究竟是怎麼回事？我們一起來了解一下。

錢大叔：說起來，我也真是很無奈，我兒子已經好幾年沒上班了，因為每天不務正業，兒媳也和他離了婚。現在他還是整天無所事事，每天就是躺在家裡睡覺。他以前還吸毒，現在只要他一晚上不回來或者晚上回家晚，我就很是擔心，怕他再走老路。

據了解，錢大叔今年已經60多歲了，但是老兩口還是每天都在工作養活兒子，為兒子的未來做打算。即便這樣，兒子一點也沒有感激之心，每天還是頹廢生活。最近，頹廢的兒子更加封閉自己，面對父母的談話，總是將電視音量調到最大來表示抗拒。

錢大叔：這段時間他吵著讓給他買車，非得要一輛40萬的車，之前已經給他買過一次車，那次已經讓家裡負債纍纍了，現在又要買。他也沒有駕照，所以我們就很猶豫，一直沒有答應他，他就開始賭氣。讓我最擔心的是，他這段時間又花了一筆大錢，問他幹什麼了他也不說，我就怕他又開始吸毒，之前戒賭戒了整整兩年。

眾人一次次的勸說換來的卻是兒子一次次的出言不遜，錢大叔面對又將電視音量調大的兒子，只好關了屋裡的電閘，沒想到這一關徹底激怒了兒子，兒子發瘋似的砸了電視。面對兒子這樣的行為態度，錢大叔很是無助。

錢大叔：我們這一生都在為這幾個孩子，我們一共有三個孩子，把他們拉扯大很不容易，以前生活困難的時候甚至靠賣血維持一家的生活。這個小兒子從小到大我們更是操碎了心，不管是工作還是找媳婦，每一步都是我們在操勞，但是兒子卻永遠不滿足。

面對錢大叔的心酸，我們其實都能明白，小兒子之所以這樣，與錢大叔夫妻倆對他的溺愛有很大的關係。

錢大叔：我們倆現在也打算搬出去住了，放手讓他自己去闖，他已經34歲了，應該學會為自己負責。我們現在還每天在掃馬路掙錢，他理直氣壯地衣來伸手飯來張口，以後他還有很長的路要走，我們也不可能養他一輩子。

已經34歲的兒子，經不了風雨，擔不了責任，找不到方向，現在甚至連走出家門的勇氣也沒有。面對父母的放手，我們希望他能慢慢長大成熟，能懂得父母的良苦用心。面對這件事，大家有什麼看法？
