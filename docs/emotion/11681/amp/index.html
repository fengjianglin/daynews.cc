因為喝酒，75歲兒子被105歲父親“狂打”，淚點被飆到了極致

天天要聞
7 天 ago

文：辛晴

還記得第一次被父親打嗎？提到父親，難免會觸動每個人的心弦，不曾想有一天「被打」也會成為父親最奢侈的愛。父愛是深沉而微妙的，看到105歲的父親，狂打75歲兒子的視頻，笑著笑著我就哭了。

父親105歲了，如今身體健壯，兒子已經75歲了，也是當爺爺的人了，可是在老父親的眼裡，兒子永遠是那個長不大的孩子。75歲的兒子外出，天已經很黑了，還沒回到家，105歲的老父親在家裡，著急的團團轉，當看到院子里電動車燈光的那一刻，老父親激動極了，「你怎麼才回來？」蹣跚著自己的身體，朝屋外走去。

兒子晃晃悠悠的撞進了屋裡，「又喝酒了？」父親真是氣不打一出來，「你怎麼又喝酒，你騎個電動車，就不怕摔倒你，我讓你喝酒，」說罷，老父親上去就朝兒子扇過去，兒子用手輕輕抓住了老父親的胳膊，「爹，我下次不喝了，」「給你說過多少次了，騎車不讓你喝，你偏不聽。」說著老父親又想朝兒子身上打去，兒子連忙攥住老父親的手，「爹，這次我錯了，這次我錯了，我再也不敢了。」

說實話，這樣的視頻我是不敢看，又忍不住看的，因為父愛已經不在，藏在內心的最柔弱處，一旦被想起，那是一種撕心裂肺的痛。有人希望自己75歲的時候也能挨老爸的打；有人說75歲能被父親打，也是一種幸福；還有人說，小時候挨爸爸打心裡特恨他，現在卻特想特想天天能挨爸爸的揍。

是啊，小時候的我們總對父親的愛有偏見，長大了，才體會到父愛是一種醍醐灌頂的愛。終於體會到那句話：養兒一百歲，長憂九十九。或許就是這個故事最真是的寫照吧！

作為我們家喻戶曉的董卿，在電視節目中，提到父親，幾度哽咽，含淚說，「我小的時候，我爸爸對我非常嚴厲，如果不小心打碎了一隻碗，他會用瞪大的眼睛，嚴厲地告訴我「你做錯事了」。

我有一度很不能理解為什麼會這樣，可是當我長大一直到現在，我才知道，每一個父母都會傾盡所有去愛自己的孩子。

​正是由於小時候父親要求她背古詩詞，所以今日的她成為了一個功底深厚的主持人，正是由於父親的嚴格，才成就了她如今的嚴謹。有些父愛看似不苟言笑、高高在上，但卻是一種最深沉的愛，這種愛只來自單純了為了孩子好。

父愛有時走著走著就丟了，如鳥雛長大，銜食哺其母，否則留下的只是父愛的回憶。
