“女本柔弱，為母則剛”生活所迫罷了，誰的內心不藏著一顆公主夢

文/奶球媽咪（原創文章，歡迎個人轉載分享）

最近總是看到這句話，在網路上鋪天蓋地地捲來。在我看來，除了歌頌母愛的偉大，更多的是批判一個男人對家庭的不負責，對孩子的撒手不管。為了孩子能夠長大成人，這些「柔弱」的媽媽不得不「為母則剛」。

「女本柔弱，為母則剛」，生活所迫罷了，誰的內心不藏著一顆公主夢

阿琴與丈夫因為生活，不得不分居在兩座城市。8月份，阿琴非常想念丈夫，便帶著1歲的孩子去慰問丈夫，阿琴興高采烈地在電話里告訴自己的丈夫，但是丈夫的回答非常冷淡。

「隨你，但是火車需要11個小時，你還帶個小孩，那你就做好站著過來的準備。」

果不其然，阿琴在火車上把孩子哄入睡以後，她就用趴在靠背的姿勢守護者孩子，站了11個小時。周圍人都在稱讚阿琴，這便是母愛的力量，確實「為母則剛」。

但從另一個方面考慮，阿琴為何會站11個小時呢？有人說是因為貧窮。的確，如今的高鐵四通八達，11小時的火車可以坐飛，但這些都是昂貴的交通方式。

在丈夫知道阿琴要去看他時，沒有說替阿琴找好最佳的出行方式，僅僅告訴他一句，站11小時，作為負責任的丈夫難道不能為妻子買上一張高鐵或飛機票嗎？哪怕是一張卧鋪票，都不會讓妻子如此乏累。

因為男女身體構造的原因，女性往往給人柔弱的感覺，男性給人剛強的感覺，但是當自己的丈夫不愛自己的時，女性也會產生一種剛強，只是生活所迫罷了。

哪個女性的內心不藏著一顆公主夢呢？「為母則剛」其實就是丈夫的不負責任，沒有做好家庭的保護傘所導致的。「為母則剛」的女人肯定是需要稱讚母愛的，但更重要的是譴責丈夫。

無論怎樣，生活仍在繼續，既然不能依賴你，那就靠自己

曾經聽過這麼一句話，一個男性對他的女性朋友說：「我好想我的妻子像你一樣，可以自己擰瓶蓋，自己修燈泡，自己修下水管，那樣的話，我就可以有更多的時間去完成自己的事。」

這時他的朋友說：「我倒是要羨慕你的妻子，可以找到一個幫她擰瓶蓋，幫她修燈泡，幫她修下水管道的男人，如果有人去做，我肯定不會親自下手。」

所以說「女超人」並不是她想如此獨立，而是為了生活，如果自己不去做，那就只能任憑事情壞下去，因為沒有人去幫助她擺平這些事。

面對「女本柔弱」的妻子，丈夫理應如此

1）盡最大的努力去寵她

每一個女人心底都藏著一個公主夢，都渴望被照顧，被疼愛。未出嫁時自己的父親把自己當成小公主，而嫁人之後，丈夫就應該繼續讓她沉浸在公主夢中。「女人是用來寵的」確實沒錯，因為她，你才擁有一個家。

2）成為她的依靠

一個合格的丈夫定是家庭的保護傘，在女人心裡，丈夫是什麼？其實是一種依靠，是一種寄託。無論受了多大委屈，看到丈夫就如同看到了希望。無論女性自己處在多麼危險的困境，只要身邊的依靠——丈夫在，那就可以感受到心安。

TIP：

生而為女，那就讓她柔弱吧，將剛強留給男性。愛到一個人什麼地步，才能願意為他生孩子，受分娩之痛呢？所以，各位寶媽，你如果表現的也是剛強的一面，那就要從丈夫身上找問題。希望各位寶媽在生活面前，仍能如「小公主」一樣悠然自得。
