心理測試：3條道路，你會走哪一條？測是什麼阻礙了你的成功之路。

A

B

C

測試解析

A：阻礙你成功的關鍵是思想過於保守，缺乏開拓精神。

也許你的家教很嚴，使你養成了恪守規則的習慣，做事一根筋，不懂得變通。身處日新月異的大環境中，要想在事業上有所發展，就必須有一個靈活機智的頭腦，善於在紛亂中尋找機遇，在各方面尋求自我的突破。不過因為自己保守的心態，潛意識裡害怕改變，從而產生矛盾的心態。可以慢慢調整自我，不要立即做大幅地轉變。

B：阻礙你成功的關鍵是急功近利的心態。

你頭腦靈活，機智多變，但是做事喜歡走捷徑，總想藉助別人的力量，一步邁入成功的行列。只是成功不是一朝一夕的事情，需要為此付出非常大的努力，選擇捷徑也許能讓你走得更快一些，但終究不耐久，沒有打下一個牢固的基礎，遲早還是得走回頭路的。

C：阻礙你成功的關鍵是做事缺乏信心。

你的內心其實還很不成熟，有依賴別人或逃避責任的心態，遇到事情首先想到的是讓別人幫助，而不是自己面對。這是一種拒絕長大的心態，遇事害怕擔責任，同時對自己也缺乏信心，這種心態使你很難在事業上有所成就。直面自己的內心，不要害怕失敗，努力提高自己獨立做事的信心和能力，漸漸地你會越來越成熟，成功也會離你越來越近。

註：測試僅供參考，三分天註定，七分靠自己，祝你開心好運每一天！
