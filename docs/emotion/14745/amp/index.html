收藏在備忘錄的碎碎念——“遇見，才是一個新的開始”

天天要聞
2019-12-15

  1. 「比起重逢，更喜歡遇見，因為這世上重修舊好少，重蹈覆轍多，而遇見，才是一個新的開始。」
  2. 「遇到步伐一致的人很重要，既有前程可奔赴，也有歲月可回頭。」
  3. 「有些風景只能喜歡卻不能收藏，就像有的人只適合遇見卻不適合久伴一樣。」
  4. 村上春樹說：「如果我愛你，而你也正巧愛我，你頭髮亂了的時候，我會笑笑地替你撥一撥，然後，手還留戀地在你發上多待幾秒，但是，如果我愛你，而你不巧地不愛我，你頭髮亂了，我只會輕輕地告訴你，你頭髮亂了喔。」
  5. 「無非是夜色撩人，皮囊都好看迷人，無非是酒精作祟，深吻也難以抗違。」

  1. 「愛你實在是一件滿心歡喜的事情，忍不住偷偷地回應你，在千千萬萬的溫柔的小事情里，我的心裡確認過千千萬萬次，嗯就是你了。」
  2. 「愛你的人東西南北都順路，想你的人白天黑夜都不忙。」
  3. 「我從來不相信，一個人一輩子只會愛一個人，但我肯定，總有那麼一段歲月，你會碰見一個想用一輩子去愛的那個人。」
  4. 「如果我愛你，我在這裡，等你涉過所有疼痛的河流，向我溫暖的懷抱走來。」
  5. 「我於昨晚去世，走時心如止水。我於今早重生，來時心懷暖陽。」

  1. 「你以為人生最糟的事情是失去最愛的人，其實最糟糕的事情是，你因為太愛一個人而失去了自己。」
  2. 「不管全世界所有人怎麼說，我都認為自己的感受才是正確的，無論別人怎麼看，我絕不打亂自己的節奏，喜歡的事自然可以堅持，不喜歡怎麼也長久不了。」
  3. 「遺憾的是那年夏天遇見的人，沒能熬過冬天一起看雪。」
  4. 「我不希望有人喜歡我是因為我的好，我希望有人看到我所有的不好，依舊喜歡我。」
  5. 「我衷心希望，你的日子裡總有光亮，笑起來的時候可以什麼都不用去想，對的要堅持，錯的要遺忘，世間山河之廣，祝你如願以償。」
