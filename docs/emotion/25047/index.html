在人情與世故中逐漸解瑣自我，體驗生命之美

2019-12-22 816條評論 3,208次閱讀 700人點贊

曾經結識一個女孩，搞藝術的，人很美，氣質不俗，腦子也好使，她的作品風格也是我喜歡的類型之一。初識時，感覺她很美但很冰，即便一起吃飯也無多少話可說。會有一搭沒一搭地說上幾句，便有點聊不動了。我也不想勉強自己，便各自看看手機，或者就那麼沉默地呆著。

再後來，接觸多了，我們有了一些共同的話題，她會和我聊一些家長里短的事了，那時候感覺這位仙女總算有些入世、有了煙火氣了。只是也難免會有冷場交流不下去的時候，但涉及到深入探討人性的東西我們往往有很多共識，我會驚訝於她小小年紀竟然將人性看得這麼透。然，當她有一天將自己的一幅幅作品傳到網上，分享到朋友圈時，我看到的一剎那感覺事複雜的，這裡面有欽佩、喜歡，但還有一種明顯的「酸酸」的感覺，我知道這種「酸酸」的感覺的俗稱是「嫉妒」。

其實原本在我們還是很小的孩子，沒有那麼多對事物的評判，也沒有總被拿來與他人比較的經歷，當我們看到別人擁有的美好的東西而我們沒有的時候，會生起一種叫「羨慕」的情緒。「羨慕」可以推動我們去爭取自己想要的。

我們可能會纏著爸爸媽媽要那個想要的東西，我們或許能爭取來，也或許爭取不來，這都不是最重要的。重要的是，爸爸媽媽對我們的態度。如果媽媽說：「哦，你也想要啊，那個東西確實很好看，很特別，我也很喜歡，只是媽媽手頭的錢不太夠，過段時間買給你好不好？」我們羨慕的感覺得到了肯定，也能夠理解爸爸媽媽的難處，而且相信我想要的只是暫時得不到，我們可以等待，總有一天會來到的。

可如果媽媽說的是：「啥你都想要，那個東西有啥好的，別人有的你就得要啊？真是沒有一點主見！」或者「要要要，就知道要，我們掙點錢容易嗎？買那東西有啥用… …」那個小小的人的感受就被否定了。如果一再遭遇這樣的情況或者某一次的情緒感受太過強烈，這個小小人的內心深處會種下「我不好，我不配或者我想要的不會輕易得到」的種子。以至於長大後也依舊抱持著這樣消極的信念，一再重複遭遇不順遂的境遇。並且遇到別人擁有而自己想要卻沒有的情況時，對他人心生嫉妒心，在羨慕別人的基礎上平添了恨意，對自己生起不滿甚至怨恨。

當我覺察到了它，我承認了它，我便也可以選擇放下它。我由衷地讚美她的創意，並在內心默默地祝福她能創作出更多自己的作品，在自己的領域裡如魚得水，有造詣。

之後我們電話溝通了一番，我也表達了自己想向她學習的意願。昨天，我們約會了，她送了我她的作品，精緻而美好。我看到她的一剎那，感覺她比以往任何時候都更美，我知道其實是我打開了自己的內心，願意讓美好進來，願意看到外面的美好。

這次的見面比以往任何時候都更輕鬆、自在和愉悅，我們之間原有的那層薄薄的阻隔的東西已經消融。我們一起吃飯、逛飾品店，交流心得以及各自的夢想，還一起分享了自己有關嫉妒的體驗… …我們待到了商場打烊，一起走走停停，在寒冷的北方的冬夜裡站在外面又聊了許久才依依惜別。

有她，讓我的世界更添了一份安寧和美好，感謝她在我的生命中出現。
