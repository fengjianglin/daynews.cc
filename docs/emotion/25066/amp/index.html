「你懷沒懷我看不出來？」孕媽坐公交被大媽臭罵，孩子的話太解氣

生活中總會出現一些不文明的現象，針對這些現象，大部分人都不敢開口，無非是擔心惹禍上身。然而每當這時，真正敢站出來的卻是一些年輕人甚至是孩子，因為還未經歷過太多事情的他們，是非常正直善良的，同樣也是具備高情商的。

小麗是一個懷孕3個多月的孕婦，還沒有顯懷。有一天她產檢結束，和往常一樣擠公交車，剛坐下便被身邊的大媽催促讓座。

大媽看了小麗一眼，若無其事的說：這年頭，年輕人是越來越不知道尊老愛幼了呀。小麗知道大媽說的是自己，於是便告訴大媽：阿姨，我懷孕了，不方便給你讓位，你去後面找找座位吧。大媽聽到小麗的解釋之後瞬間怒了，直接對其大聲罵到：不讓座就不讓座吧，竟然還用假懷孕當借口，你懷沒懷我難道看不出來嗎？

小麗聽完之後委屈極了，想要離開座位，卻把產檢報告掉在了地上。這時一個小學生將報告單撿起來給小麗並對她說：阿姨你坐在這裡就行，老師說了要給懷孕的阿姨讓位。小麗對學生的做法深表感謝和感動，而大媽看見產檢單後無言以對。

小麗因沒有讓位，被大媽指責假懷孕，所以這位大媽是毫無素質的做法；而小學生的做法與大媽形成明顯的對比，也能看出他的高情商。那麼，如何培養孩子擁有較高的情商呢？

第一， 讓孩子學會站在他人的角度思考問題

雖然沒有完全的感同身受，但起碼要學會站在對方的角度，考慮對待一件事情的態度，才能明白對方為何要因為一件事，做出和自己完全不同的態度或選擇。在明白這一點之後也，就能迅速的解決問題。往往不會傷害自己與對方的感情，也不會讓對方難堪，更會以正確的方式來對待一件事。能做到這樣的孩子，自然是高情商的表現。

第二， 培養孩子的社交以及互動能力

高情商體現在待人接物方面，因此要讓孩子擴大社交圈，並提高社交能力，多參加一些集體活動，有助於培養孩子的團體作戰精神，也能讓孩子在集體活動中，養成互幫互助和共同分享的優點。從而才能讓孩子將這些優點運用在生活中，和與他人的交往中，這樣的孩子在與他人正常社交時，就會被人認為是高情商的人。

第三， 讓孩子學會控制自己的情緒

想要成為一個高情商的人，首先要做到控制自己的情緒。在面對一件事情時，冷靜的機智化解才是真正高情商人的做法，而不是一味發泄自己的不良情緒來達到泄憤的目的。因此，當孩子有負面情緒時，家長應用正確的方式指導孩子如何發泄以及平定。只有這樣做，才能夠良好的塑造高情商的基礎。

在生活中，高情商的人能受到大眾的歡迎。他們可以撐得起任何場合，也能迅速的脫穎而出，憑藉自己的高情商，被廣大人群認可。
