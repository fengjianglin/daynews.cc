10條走心感悟，句句精闢有溫度，哪句讓你百看不厭

2019-12-16 106條評論 1,075次閱讀 77人點贊

文/初時情感 你這麼帥氣美麗如果喜歡 請不要忘記「點贊，收藏，轉發呦」

1、這世間所有因緣巧合，聚散離合，都在走著既定的路線，沒有多一步，也沒有少一步，有時拚命想改變已經發生的，事實卻總是不盡如人意，我們苦苦糾纏於昨日，不如放下心中那點執念。春去秋來花開花謝，世間萬物繁茂與凋零都有其中道理，人生亦是如此，莫不如讓一切順其自然，好與壞，來與散只是一個過程，若干年再回望今天不過天空中一抹青煙。

2、我們都在憧憬展望著美好的明天，奈何做事情只有三分鐘熱度，喜歡做一件事，做了幾天沒有了新鮮感，就想放棄了，找到一份工作，與同事相處不融洽，又離職了，時間如此往複中丟掉了一大半，回望走過的路迷茫又痛苦，其實只是缺少了一分堅守，對一件事著迷的態度，時間是寶貴的，每個人都經不起蹉跎，找到你想要的，堅持下去，贏得了自己的掌聲就是勝利。

3、正所謂打蛇打七寸，人是經不起誘惑的，每個人或多或少都有軟肋，如果真有人想要誘惑你入深淵又怎麼能抵抗呢？太極的標誌涇渭分明，非黑即白，中間有彎曲的線來隔斷兩點，這兩點能不能逾越由你說了算，這黑白兩點就像兩個人，當我們互換下位置，以對方的角度看對面的自己以及它的目的，會給我們帶來什麼，想明白了，這誘惑就不能叫誘惑，也可能叫毀滅和墮落。

4、大道理聽得太多了，每個人的見解都很獨到，都很有道理，但適不適合自己呢？有些字看看就算了，沒必要太較真，有些話，聽聽就得了，如果真鑽進去，糾結的還是自己，所以取精華，去糟粕才是重要的。

5、一輩子會遇見無數多的人，如果遇見一個就說和我們有緣，那緣分不是泛濫了，就像交朋友一樣，不需要多，有那麼一兩個知心的，沒事吹吹牛皮，閑聊天不摻雜利益就挺好。

6、人的性格會隨著時間推移，價值觀轉變而發生改變，以前張揚的性格逐漸變得內斂，鋒芒也沒了稜角，這並沒什麼不好，只是換了一種方式和自己同外界和平相處，溫柔中以寧靜的姿態看待我所理解的世界，僅此而已。

7、每逢周末，都會靜靜的聽聽歌，看一會書，練練字，不是有多好學，只是想要內心充盈一些，畢竟心靈還是浮躁的，人來人往中夾雜著各種慾望與貪念，為了不被這些所污染，只能學得簡簡單單，讓時間放慢，再慢一點。

8、特別喜歡做的一件事：在安靜的午後頂著陽光，躺在公園的長椅上，吹著清涼的風，看著天上點點白雲，偶爾有飛機飛過，從這邊到那邊，目光靜靜轉動，這一刻我感受到了自在，仔細想想這不就是我們期望的生活嗎，恬靜悠閑，心無雜念。

9、每一次的離別，都伴隨著母親的目光，她不善言辭，就這樣靜靜的看著我離去，母親的愛是等待與離別，回來時也沒有歡呼與吶喊，一切就是那麼尋常，我的肩膀不算寬廣，背影也不算偉岸，但總有一刻明白了母親的不易，讓我勇於挑起生活的扁擔。

10、工作是做不完的，錢也是掙不完的，所以工作開心，錢夠用就挺好，每個人的追求不一樣，但最終的本質是一樣的，至少我們老了之後什麼都帶不走，讓自己身心愉悅，沒必要給自己那麼大壓力，有時風輕雲淡好運氣就來了，你說怪不怪？
