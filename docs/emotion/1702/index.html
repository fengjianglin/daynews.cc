醫院裡看到這樣的一幕，讓人瞬間熱盈滿眶，當媽不易

有很多歌曲、詩詞都是歌頌母愛，表現母愛的偉大，可以說，母親是偉大之類詞語的代名詞，但是，這份偉大的背後，是充滿著多少汗水與時間的付出。

她們所面對的困難和壓力，並不是每一個人都能理解，也不是每一個人都能忍受。孩子與媽媽心連心，一舉一動都牽動著媽媽的心。

媽媽最見不得的便是孩子難過，生病。只要寶寶一有反常，媽媽都會十分緊張，問題得不到解決，媽媽的神經就不會放鬆下來。

當媽不易——鏡頭下醫院裡的媽媽

有個網友在網上上傳了個視頻，拍攝了醫院裡媽媽的身影，鏡頭下的她們或高或矮，或站或坐，但都有一個共同的神情——焦慮。

通道的這位媽媽，是個二胎媽媽，不知是大寶或二寶生病，剛看完醫生，準備回去。只見媽媽單手抱著睡得沉沉的大寶，推著躺著二寶的嬰兒車，肩上背著寶寶們的用品，小小的身子，承載著大大的責任。

或許在寶寶出生前，媽媽在爸爸面前，也是個柔弱的女子，逛街提個包都會喊累，但在寶寶面前，她就是超人，再苦再累，也會咬牙堅持，女子本弱，為母則剛。

隨著鏡頭一轉，視頻里出現的另一位媽媽，抱著寶寶，看著叫號顯示屏，懷裡的寶寶等到已經睡過去了。

醫院人滿為患，椅子都被坐滿了，媽媽便一直抱著生病的寶寶，在等號區站著，神情充滿了焦慮，又無可奈何，只望能趕快輪到自己，看完醫生趕緊可以帶寶寶回家休息。

站在樓梯上，鏡頭往下一轉，便又看見另一位媽媽。媽媽抱著寶寶快速的下樓梯往一樓走去，行色匆匆的，趕著去做檢查。寶寶趴在媽媽的肩頭，悶悶不樂的，沒什麼精神。

媽媽估計心疼壞了，越走越急，邊走還不忘輕聲安慰寶寶:”寶寶乖，很快就不難受了哦。”下了樓梯後，媽媽的步伐也越邁越大，很快就消失在視頻里。

醫院裡的媽媽看起來都很累，她們不是抱著寶寶等待叫號，就是抱著寶寶去拿葯，做檢查，她們的神情都很焦慮，很擔憂，寧願自己生病也不願意寶寶這樣難受著，看得讓人不禁瞬間讓人熱盈滿眶。

但沒有辦法，寶寶抵抗力弱，生病是在所難免的，不管他有多難受，媽媽只能挺住，和孩子一起戰勝病毒，每一個媽媽都是這樣過來的。

媽媽需不斷成長的重要性

在育兒的過程中，不僅是寶寶在長大，對於媽媽來說，也是一種成長。

她們的偉大之處不僅僅在於為孩子的日常瑣事勞累，更在於她們願意為了孩子，去改變自己，不斷地學習，不斷地進步，做孩子的好榜樣

董卿在節目里說，曾有位友人說過一句話，”你希望孩子變成什麼樣的人，很簡單，你去做一個什麼樣的人。”

這句話，讓董卿猶如醍醐灌頂，她覺得”在成長的過程中”是最美好的一種狀態。所以為了讓自己變得更好，她接受了一個全新的挑戰。她開始籌備《朗讀者》，這一次，她不再僅僅只是主持人，還是製片人。

只要用心努力的付出，總會有回報。《朗讀者》的播出，好評如潮，她成功了。在成為孩子尊敬的人的道路上，她又邁出了一大步。

陪孩子一起成長，做令孩子驕傲的媽媽

最好的媽媽，是不斷進步的，永遠不停下成長的腳步。

在孩子成長的過程中，他們慢慢的會有了自己的認知和自豪感，這時，你可以用自己的經驗和成就告訴孩子，沒有儘力而為，只有全力以赴，只有你肯努力，肯付出，未來一切可期。

與其做一個孩子背後偉大的媽媽，還不如做一個讓孩子驕傲的偉大媽媽。

和孩子一起成長，讓自己成為孩子的驕傲，這不僅僅是一件很美好的事情，也是送給孩子最好的禮物。

我是尼莫媽媽育兒師，也是4歲寶寶的媽媽，在養育和教育孩子過程中，遇到任何問題可以來問我，希望我的一點建議能有助你解決帶娃過程中的困惑與煩惱，讓我們的孩子能更好的成長！
