一個成年女人說不出口的願望…...

2019-12-25 632條評論 1,568次閱讀 306人點贊

一女士，經常覺得自己陪伴孩子的時間太少，因此內疚。事實上，她幾乎每天都會陪孩子。我問她什麼時候內疚感最強，她說身體很累又不得不陪孩子的時候。

她想了想，其實內心隱藏著對孩子的強烈憤怒，以下是一些對孩子憋了很久、但說不出口的話：你都9歲了，怎麼天天要我陪著？就不能自己呆一會嗎？你好煩！9年了，我的時間都奉獻給了你，完全沒有自己的時間，你還不滿足！

她的記憶里，似乎沒有父母陪伴的時光，她的內心有個孤獨的小女孩，甚至沒有勇氣說出這個小小的願望：爸爸媽媽，請你們陪陪我！

所以，當她自己有了孩子，便把自己童年沒有實現的願望投射到孩子那裡，她非常篤定，孩子需要非常非常多的陪伴，哪怕自己累得筋疲力盡，還在強撐著陪孩子。其實，這位女士內心深處憤怒的是沒有時間陪自己。她說，也許下一次感到內疚的時候，可以嘗試告訴孩子，她需要跟自己呆一會兒了。

總結：意識上的內疚，其實潛意識裡是憤怒，是不是很有意思？所以，媽媽們首先要陪好自己，照顧好自己的身心，其次才是照顧孩子。無論你的身份是媽媽、妻子、女兒還是領導、員工，你首先是你自己。
