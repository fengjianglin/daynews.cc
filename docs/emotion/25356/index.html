男子的女友比自己高一個頭，看到女友臉後，網友：你拯救了世界

2019-12-21 29條評論 1,021次閱讀 77人點贊

其實感情這個東西確實是比較難抉擇的，很多人說現在如果要談戀愛的話應該考慮東西，尤其是兩個人般配不般配的問題了。

很多人衡量兩個人般配的問題一般都是看的顏值，大家都覺得只有好看的人才會喜歡上好看的，一旦情侶雙方有一方顏值比較低可能就會覺得顏值低的人家裡有礦，這種想法確實是不正確的，並不是所有的真愛都是因為顏值走在了一起。現實中有很多這樣的例子，就比如下面的這對情侶，讓我們一起來看看吧！

小哥哥的女友比自己高一個頭，兩個人因為身高的問題受盡非議，小哥哥的身高放在人群中就是一個平常人，但是小哥哥的女友身高有一米八多，在女生身高中算的上是很高的了，更是比他高一個頭。除了身高之外，看到女友臉後，大家更是懷疑這個世界到底是不是公平的了，因為這位小姐姐長得實在是太好看了啊！簡直就是女神級別了。

網友們看了之後表示這位小哥哥就好像是拯救了世界一樣，或許是這個小哥哥比較優秀吧，用自己的優點吸引了這位小姐姐。實在是太羨慕這位小哥哥啊，女友簡直是太完美了，就像是很多人的夢中女神！不過還有很多女生表示接受不了比自己矮的男生當男友。

大家覺得兩個人是因為什麼原因在一起的呢？你們能接受這樣的男生嗎？
