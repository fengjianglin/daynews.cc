該不該欺負老實人？不欺負你的處境就會艱難

天天要聞
3 天 ago

老話說：老實人不可欺。很多人其實栽在這句老話上。當然我們得承認這些人是善良的，秉承了不欺負老實人的古訓，兢兢業業自己奮鬥。但老話並不都是無懈可擊的，它有各種各樣的漏洞。就拿「老實人不可欺」來講，這裡面漏洞就值得鑽空子。

老實人經常被欺負

我們知道，一個團體空間里，所佔有的資源相對是有限的。如果你是個領導，擁有一定資源，如何利用這些資源做大、做強？就是把業績做好、把自己做得進一步升職？這裡面一個重要辦法就是「欺負老實人」。其他辦法這裡暫且不談，專門談談老實人如何被欺負，或者說為什麼要欺負老實人。

老實人通常不善於人際交往，人脈較少，只顧著自己工作，掙個工資，活個命。當工作出現紕漏或者瑕疵或者漏洞時，把這些「髒水」潑在老實人頭上是最佳選擇。潑在自己身上肯定不行，潑在調皮、無賴、搗蛋鬼身上是有代價的。只有選擇老實人，他們一不會鬧事，二會選擇忍氣吞聲，三會選擇默默承受。他們甚至奉行「吃虧是福」的理念，從這個角度看，不選擇他們倒是對不起他們的這個特性。

被欺負的老實人只有低頭承受的份

如果你晉陞面臨競爭，首先就要欺負老實人，不然淘汰的就是你。職場上混，不能有絲毫同情心，你同情了老實人，那麼最後倒霉的肯定是你。你的同情意味著你在退讓，在晉陞和評先進、拿獎金這些好處面前，同事之間就是「你死我活」的「拼刺刀」，尤其是大家在差不多條件的情況下，你敢於率先「欺負」老實人，那麼你就佔得了先機。至於如何率先欺負？情況千變萬化，小編不能細說，只是提醒一些善良本性的人，如果你同情了誰，最後輸的肯定是你自己。

晴雯也會欺負更弱小的丫鬟

「欺負老實人」有沒有罪過？道德婊們一定會指責小編不教人做好事，專門傳播負能量。其實不然，一定空間裡面能量、資源都是有限的，你不欺負別人，就要淪落到被別人欺負的處境，想想看吧，到底該不該「率先欺負別人」？

我們再想想這個世界，歐美日是如何發達起來的？尤其是英法德日西荷這些老牌資本主義小國家在發展初期，誰沒有欺負過亞非拉貧窮的老實國家？哪個不是欺負得這些國家鮮血淋漓、千瘡百孔，把老實國家的財物搶走，殺掉了很多老實國家的老實人民？馬克思曾經深刻指出：資本來到這個世上，每一個毛孔都流著血和骯髒的東西。

其實很簡單：強者幾乎都是踏著弱者的血肉和眼淚富裕起來、強大起來的。國家之間的深仇大恨歷歷在目，比如中日，日本侵略中國時，他們有沒有管你中國普通人民是不是老實而不殺？

職場如戰場，雖然不存在那些毛骨悚然的鬥爭。但是必要的「欺負」到處存在。要想不受欺負，首先自己得強大起來。至於你強大了，還會不會欺負老實人？那得看你的修鍊成果了，你的位份很高，肯定不屑於欺負普通老實人。你只會在你的層次上繼續「欺負老實人」，只有這樣，才能繼續強大自己，鞏固自己的優勢。

《紅樓夢》里的晴雯死得可謂很慘。但是別小看她只是個丫鬟，她也會欺負人，小丫頭墜兒偷東西被她戳得很可憐。她在欺負比她更弱小的人時，有沒有想到別人可憐？因果報應，她最後也被比她更強大的人欺負得無處可去，死在茅棚草房裡。
