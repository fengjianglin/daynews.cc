逼婚、催生、為女兒打造“圓滿人生”：控制欲太強，是一場災難

2019-12-21 28條評論 1,039次閱讀 68人點贊

心理學大師海靈格說：幸福的家庭都有一個共同點，家裡沒有控制欲很強的人。

然而中國是個人情社會，奉行的就是「我中有你，你中有我，我的就是你的，你的就是我的」，大都患有一種心理疾病，那就是病態共生。

無論是夫妻之間，父母與子女之間，還是兄弟姐妹之間，常常沒有「自我」和「別人」這兩種概念，互相喜歡以愛的名義裹挾對方。

尤其是父母對子女，伴侶對伴侶，不由自主就把自己的價值觀強加於對方身上；隨時隨地窺視對方的各種隱私；把對方當成私有財產對各種事務越俎代庖；甚至把對方當成實現自己理想的工具……

這種裹挾是一種可怕的控制欲。在中國式家庭中，90%以上的矛盾，都源於上述這些強烈的控制欲。

我的老家是個四五線小縣城，對於鄰居李大媽來說，女兒媛媛已經30歲了還沒有結婚，她總覺得這樣的人生太不圓滿了。

其實李大媽這個人的思想平時還算開明，遇到別人家孩子40歲不結婚她都能表示理解，還替人家辯護「結婚不結婚是人家的自由」。但當這件事擱在自己身上，她就做不到這麼超然物外了。

她每時每刻都在想著媛媛還沒結婚這件事，時間一長就成了一塊心病。

其實媛媛上大學時談過一場戀愛，但李大媽認為上學期間就應該全身心學習，何況那個男孩家的條件不太好，她很不滿意。媛媛身高1米65，眉清目秀，學習成績優異，談男朋友可不能隨便。

於是她硬是跑到學校鬧騰了一番，將一對「小鴛鴦」強行分開了。沒想到這才幾年，她又開始為媛媛不談戀愛不結婚而發愁。

雖然媛媛的工作單位很好，也有屬於自己的房子和車，但在李大媽眼裡，這些好條件終究會隨著年齡的增長逐漸貶值，就像錯過了早市的新鮮蔬菜，哪怕還水靈，不便宜賣要的人也不多了。

於是工作單位一般的，沒房沒車的，學歷低個子矮的，只要是男的活的，李大媽都著急上火的催著媛媛去看。

媛媛覺得如果沒有真愛，一輩子一個人過也挺好的，但李大媽卻覺得，一個人頭疼腦熱了誰來照顧？一肚子話要跟誰去說？出了事兒跟誰商量？晚上回到家連口熱水都喝不上……

就在這樣的抑鬱中李大媽越來越消瘦，最後看媛媛實在沒有結婚的打算，就以死相逼，總算逼她嫁給一個40歲離異沒有孩子的博士。

媛媛婚後並沒有過上幸福生活，相反的，她活得更窒息。

博士老公是個十足的小心眼，媛媛去哪裡都要跟他報備，而且報備的形式很奇葩。不是開視頻掃蕩一圈，就是打電話讓旁邊的人接，以證明現場沒有其他異性，也證明媛媛沒有撒謊。

即便如此，每天回到家媛媛的手機還要被老公再檢查一遍。

平時媛媛喜歡喝咖啡，老公卻說咖啡因對身體不好，還是白開水健康。媛媛想吃燒烤，老公又說燒烤不衛生還容易發胖，多吃點水果吧。

媛媛很苦惱，她說婚前受媽媽控制，婚後受老公控制，感覺每天都喘不過氣來。但李大媽卻兩手一攤，男人管得嚴也不是什麼大事，你這個年齡才結婚，還不趕緊要個孩子。

劉若英曾經說過：「真正愛一個人就是讓對方活得像自己。」

不管是作為父母也好，愛人也好，子女也好，我們都要明白，真正的愛不是支配和控制，也不是以愛的名義綁架對方。

控制多了，家庭矛盾糾紛也就多了，悲劇也就由此產生了。

媛媛結婚剛半年，李大媽又進入了瘋狂催生模式，她想外孫想得都要發瘋了，總感覺女兒沒有一個孩子總是一大遺憾。

然而李大媽並沒有得償所願迎來大胖孫子，她得到的是女兒生病住院的消息。可能是長期生活太壓抑太苦悶的緣故，媛媛得了乳腺癌。

在手術室外等候的李大媽哭得死去活來，博士只是來醫院呆過兩次就再也不見人影了，媛媛還沒出院就收到了離婚協議書。

就這樣，從鬼門關走過一遭的媛媛，從原來的大齡剩女，變成失去了一隻乳房的離婚婦女。

她躺在病床上，用微弱的聲音對李大媽說：「我處處按照你的想法去生活，這樣的結局你滿意了？」

李大媽淚流滿面捶胸頓足，恨不得把心掏出來給女兒看：「我管這麼多，還不是因為關心你，怕你吃虧，為你好嗎……」

李大媽大概到死都不會明白，她的女兒不是她的女兒，是一個完全獨立於她的「別人」。

之所以事情發展到今天這個地步，就是因為她沒把女兒當成一個完全獨立在自己之外的「別人」，缺失應該有的界限感。

有一期《奇葩說》辯論話題是：如果一鍵就能為孩子定製完美人生，你會按下按鈕嗎？

所有控制欲強的父母都會，但是，孩子真的願意過家長為他定製的所謂完美人生嗎？家長眼中的完美人生，難道一定是孩子想要的完美人生？

實際上家長們嘔心瀝血費盡心機為孩子定製「完美」未來，孩子們根本不買帳！

完美人生，就應該擁有屬於自己的充分的選擇權。

而李大媽這樣的母親，總是因為兒女的事失眠，因為兒女的事生病，因為兒女的事心情不好，還總是希望通過改變兒女來改變自己的情緒。

當她們在控制兒女的時候，常常是因為自己沒有穩定的自我價值感，所以才需要通過兒女的言語和行為，來收穫肯定以及安慰。

要想改變這種情況，李大媽們除了要明白「愛就是要放手，讓對方活得像自己」，還要建立一種穩定的自我價值感。

當自己不再需要別人的肯定、感激、安慰、褒獎來證明自身價值，才能不去控制兒女。

既相親相愛又擁有個人空間，才能擁有幸福的家庭生活。

這世上最好的愛，是陪伴，而不參與。

與君共勉！
