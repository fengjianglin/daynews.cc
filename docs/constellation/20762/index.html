雙子座總是覺得自己有點缺愛症，越重視越在乎越好，因為剛開始被寵愛的感覺實在太棒。然而現實是根本不能得到這麽多的寵愛，所以有時候覺得是自己給自己挖了個坑，自尋煩惱。雙子座內心其實是喜歡低調的，但是一個忍不住總有很多熱情爆發，然後就開始討厭自己。所以有時候雙子座會討厭自己的星座。

雙子其實也搞不太懂自己有時候到底在堅持啥

雙子座的愛情總是富有戲劇性和傳奇性的。雙子的多面性和善變性，使雙子的愛情總是一波三折。雙子對愛情的控制欲很強烈，翻手為雲覆手為雨。然而反覆無常的變化有可能換來的是愛情幻滅的痛苦。無論多麽深厚的感情也經不住三番五次的遊戲。

雙子座的愛情常常曲折又離奇，雙子其實有時候也搞不懂自己，有一派是專情到誇張，另一派是花心到不行，雙子天性就是比人調皮，敢冒險是他們的天性，不過也因為這樣，有許多的雙子在感情上也常常陷入自己的死胡同，怎麽樣都甩不開，或許雙子的感情就是要多了一點歷練吧！！
