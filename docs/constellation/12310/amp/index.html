塔羅催眠乾貨｜兩張牌將“現實自己”和“完美自己”的合一

天天要聞
2019-12-13

在塔羅方向的工作15年來，這幾年更多的是希望通過塔羅牌來來更好的療愈當下的，而不是把所有的關注點都放在未來。這幾年，我將催眠的技術運用到了塔羅牌里，我想在這頭條號的第一篇我們就來聊聊，塔羅和催眠兩個內容是如何一起工作的。

催眠和塔羅一樣神秘嗎？

催眠在很有人眼裡其實和塔羅一樣很神秘，很多電影里的片段就是這樣：打一個響指，來訪者就昏睡了過去。由於電影的情節讓很多人對催眠產生了好奇，但更多產生的是害怕。大多人是會覺得那是很可怕的事情，生怕自己失去了自我意識而讓催眠師掌控了。我經常和來訪者開玩笑說，放心，你的銀行卡密碼告訴我之前，你肯定會自己醒來。

其實，生活當中很多的事情都是在催眠的過程中，你看小說，不由自主的變成了霸道總裁的女主角的時候；你玩遊戲化身戰隊中最勇猛的將士的時候；你看電影淚流滿面的時候其實都是在催眠的過程中。這樣的過程中你的內心變化是起伏的，有一些認知在逐漸的被改變，這就是催眠需要帶來的。

而塔羅牌的畫面和它背後完整的神秘體系的支撐，可以讓人更好容易促發催眠現象。而通過塔羅牌來促發的催眠之後，我們需要做的就是調整來訪者當下的情緒，處理他的一些認知和行為。

下面我們來通過案例來加深理解

案例圖片

療愈的目的：現實中的自己和最滿意的自己的合一

使用的牌：塔羅牌21張大阿爾卡納

來訪者：35歲女性 已婚 工作動力缺失，迷茫未來的工作。

基礎諮詢分析：來訪者是來看未來工作運勢的，但是她更多的問題是在不知道如何尋找回內心的動力，所以需要幫她調整當下的自己再抽牌預測一下未來的工作狀況。

現實中的自己感受：她感覺自己在牌面中就是右側那個黃色頭髮的女生，我問她什麼感受的時候她覺得自己渾身發冷，並且很羞恥，並且整個身體很僵硬。這時候，引導她進入到催眠的狀態，這個時候她看到的是小時候發生的一些事件，她一直處於很自卑的狀態，所以成長中的自己一直沒有處理好。在過程中處理了她的事件之後，回到現實中來。

最滿意的自己：看到力量牌，她覺得自己像那個獅子而不是那個女神。她覺得自己是一頭溫柔的獅子，但是隨時都可以成和女神一起戰鬥的勇士。讓她去感受獅子內心的想法和力量，並且取出這種力量化身自己身上的一個物件提醒著自己。當時，她取的是獅子的一顆牙齒當作了自己的一條項鏈。

關於塔羅催眠師

作為一個塔羅催眠師內心需要有著愛和仁慈，這樣才可以療愈更多的人，希望通過文章讓更多人能了解塔羅催眠，更好的運用這個技術來幫助自我和他人，隨後我會日常跟新相關的內容，努力寫一些乾貨送給大家。

這是我們在頭條上認識的第一次，初次見面，感謝關注！
