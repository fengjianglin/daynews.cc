鬼谷子的3句識人要訣，學會了，看人不會“走眼”

2019-12-18 2,383條評論 4,355次閱讀 2095人點贊

大圖模式

鬼谷子，是戰國時期的著名隱士。雖然他的一生，都只是局限在一座山上，但他的思想、謀略、學問卻間接影響了那個波雲詭譎的時代。

而他的幾個徒弟，卻都闖出了偌大名聲。如張儀、蘇秦，是當時最有盛名的縱橫家；而孫臏、龐涓，也是首屈一指的軍事家。

不僅如此，鬼谷子一生著作頗豐，即便在識人用人這方面，他也有著高明的領悟。今天筆者就給大家介紹鬼谷子的三句識人要訣，學會了，可以讓我們看人不會「走眼」。

大圖模式

1、若夫見可憐則流涕，將分與則吝嗇，是慈而不仁者。

第一種人，是空有仁義的人。在別人遇難之時，他們會覺得十分可憐，想要生出自己的援手。可一旦真正的去付諸行動後，這些人又會疼惜自己的財產，變得十分的吝嗇。

到最後，這種人儘管掙足了面子，但隨著時間的累積，自身的口碑便會變得非常壞了。鬼谷子的點評很是到位，有些假仁假義之輩，他們看上去慈悲為懷，比和尚還要悲天憫人，可一旦關乎自己的利益，他們寧願一毛不拔，可見這並非是真正的仁義。

我們在為人處世之時，應該注意身邊是否有這樣的人。正如現在不少明星，為了自己的名聲，從而高調宣布自己做慈善事業；但是背地裡，這只是他們轉移財產和博取路人緣的一個手段。近年來，不少明星詐捐的事情被曝光，足以見得這個行業的烏煙瘴氣。

對此，我們的正確做法是，當自己身邊的人遇到危難之時，我們應該理智正視自己的實際情況，從而做出最真誠的援手。而不是雖然面子工程做的足夠，但到了出手之時卻吝嗇無比，以免被人貽笑大方。

大圖模式

2、處虛義則色厲，顧利慾則內荏，是厲而不剛者。

第二種人，在面對大是大非的問題時，總是莊嚴而又肅穆，給人的感覺也是剛直、真誠。但實際上，他們不過是在沽名釣譽，一旦別人給出的砝碼足夠誘人，他們便會暴露出自己內心真正的貪慾。

而說起沽名釣譽，相信不少人都會想到三國時期的袁紹。大家都知道，袁紹的家世非常好，是四世三公的大族，其門生故吏也是遍佈於朝野內外。

這樣高的起點，讓袁紹在諸侯混戰之初，便積累了足夠的資本。鼎盛時期的袁紹集團，佔據整個河北地區，又有一大批能人異士幫助。但袁紹這個人，卻是一個沽名釣譽之輩。

大圖模式

他常常能做到禮賢下士，待人接物都是一副高風亮節的模樣；他也能做到虛心納諫，河北地區的百姓也對他十分愛戴。可這樣的袁紹，只是一個假象。

在官渡之戰發生期間，袁紹偏聽則新，一味相信自己的同鄉謀士，從而對頗有智謀的沮授，表示出了極度不信任。

歸根結底來說，袁紹是一個自私的人，他其實只顧得上自己的利益，從而不斷的猜忌對自己忠心耿耿的下屬。時間一長，難免會出現君臣離心離德的下場，最終讓強盛的袁紹集團在頃刻之間便煙消雲散了。

在我們的日常生活中，如果我們沒有及時與這種人保持一定距離，那麼很有可能會被他們利用，給自己帶來不必要的損失和麻煩。

大圖模式

3、睹危急則惻隱，將赴救則畏患，是仁而不恤者。

除此之外，還有一類人，他們有著一定的同情心，見到不平的事情，也能想到伸出自己的援手。但問題在於，他們儘管有了一定的仁慈之心，卻沒有相應的體恤之心。

一旦將要做的事情可能會出現麻煩或損失，他們將會及時抽身。比如現代社會中的某些慈善家，他們為了達到宣傳自己的目的，便通過一系列的包裝手段去營造出一種感人的假象。

可他們忽略的是，被援助者這一個弱勢群體，其實也需要他人的尊重。倒不如說這種人的心思很壞，只是他們的出發點雖然是有些好，但卻建立在了一定的功利心上。

所以這也在告誡我們，在援助他人的同時，也要考慮到被援助者的感受和尊嚴。畢竟光有一個仁義的出發點，卻不懂得體恤他人，是一點不符合人文關懷的。在這個人人平等的社會，如果大家都相互體恤對方，相互理解各自的難處，或許我們的生活才能共同進步。
