一周星象12.23—12.29分析，太陽進入摩羯座，帶來新年憧憬

2019-12-21 770條評論 1,845次閱讀 596人點贊

時間來到12月的最後一周（還有30和31日）。我們即將迎來聖誕節和元旦佳節。2019年也走到了尾聲。本周將會進入到一個新的星座月，太陽會在22日進入摩羯座，摩羯月即將開啟，在此先祝福摩羯座們生日快樂！同時水星也會在本周進入到摩羯座，至此群星聚集在摩羯座，一輪新的規則將會被建立，對新年的憧憬也會開啟。接下來青峰就為大家帶來一周星象分析，看看2019年的最後一周，會有哪些星象預警和驚喜吧！

太陽進入摩羯座

太陽來到摩羯座，摩羯月即將開啟。在這個充滿嚴肅和現實意義的星座月，群星將會迎來一次新的調整。在事業、家庭、婚姻、情感、財運、健康等各個方面，都會有一個全新的開始。目前群星聚集在摩羯座，太陽的到來徹底激化了群星的力量，土元素成為主導力量，但是又會與多種元素對抗，新的星座月充滿了挑戰，但是也充滿了機遇。隨著上周金星來到摩羯座，我們對新的一年有了更多的期待和憧憬。明年的大環境形勢必然更加嚴峻，我們將面對更加嚴峻的環境和想對壓抑的氛圍。所以，如果當下，你還沒有積極的心態，還沒有想要突破和爆發的決心，那麼本周是時候對明年展開計划了。

太陽三合天王星

這一星象的變化很切合摩羯月的主題。雖然在年底的這段時間，你可能會覺得身心俱疲，而且好多好多事情集中爆發，讓你分身乏術，不知道如何處理各個事情之間的關係。但是在年底的最後一周，你卻有非常不錯的心情，和愉悅的生活氛圍。這段時期，你的工作環境會比較和諧，雖然可能會有人離去，或者你自己正面臨失業或跳槽，但是不需要擔心，這反而是對過往的終結。如果你做的好，做的開心，相信你也不會選擇輕易的離職。所以你可能會有一個更好的發展。在感情方面，同樣是如此。一些朋友可能會面臨斷舍離，但是這也意味著，你將會得到解脫，不必在這段感情里傷感。

太陽木星合相

這是一個很好的相位，這個相位會激發我們內心的一些潛在的渴望。如果這段時期，你對某個人長久以來的好感，已經到了一個臨界點，那麼你會更有勇氣去表白。如果你覺得當下的生活索然無味，你可以選擇去學習一項技能，參加一個培訓班，或是社會的公益活動，會給你的心靈帶來一次新的體驗。這段時期，你也回很想求新求變，在當下的工作中，玩點花樣，給自己帶來一些新奇的過程，和更特別的結果。你也可能會在某個時刻，發現自己的能力，一種你從未被挖掘的能力。

水星進入摩羯座

在摩羯月，各種能量在摩羯座面前可能都要低頭，但是並不是會消失，而是以更趨於摩羯座的屬性的方式，來釋放能量。所以本周，很多事情會更加講究效率、規範、嚴苛。本周，開車的朋友一定要多多注意，不要違規違法，減速慢行，尤其是停車的時候，不要有僥倖心理。在工作中，你要更加謹慎小心，尤其是在處理一些重要的數據時，一定要備份。同時，要注意與人之間的溝通，因為你很可能會跟不上別人的頻率，在與大客戶談判時，腦子一定要保持清醒。不過對於考試的朋友來說，這段時期你的考試運會特別好，好好複習，保持清醒，好成績可以期待。

新月日食上演

重要的時刻來臨了。本年度的最後一次新月日食發生在摩羯座。並且此次新月日食的影響力很大，影響的時間會很深遠。在新月日食的影響下，必然會有一些人生大事要被重點討論，可能會有一些重要的決定。比如，戀愛很久的情侶，可能必須到了要考慮結婚的時候了。所以婚姻大事會被重點討論和提上日程。再比如，你當下的工作中，某個項目已經進展到了一個瓶頸期，是繼續大力投入，還是選擇放棄，總之必須要做出一個決定了。類似這樣的情況還有很多很多，可能表現在生活的各個案例上。也許這些決定並不會立刻做出，但是你可能到了必須要把這些決定提上日程的時候了。

青峰結語

新的一周，摩羯座上又多了很多星象發生。2019年末到2020年初，都會是摩羯座為主題的星象更多，這也預示著明年，會有更多的挑戰，凡事也會更加講究合情合理。也許有很多人說，這帶來了太多的挑戰，但是青峰卻覺得這是個糾正錯誤，幫你更精準的選擇未來的時刻。2019年即將結束，我們是時候好好計劃一下未來，不能光憧憬，某些事情也要提上日程了。周運很快到來，敬請期待。
