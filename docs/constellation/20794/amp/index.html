射手座

在提到花心、不專情的星座時，射手座一般都會榜上有名。之所以會這樣，是因為射手座的人幽默風趣，懂生活，所以很能吸引異性的目光，但這僅僅是他們單身的時候。一旦在戀愛後，射手座的人就會變得很專情，因為他們同時也是一個非常怕麻煩的星座，他們覺得同時顧及兩段感情的話會很複雜，所以，他們會一心一意的對待自己的戀人。

摩羯座

摩羯座的人本就是一個有力量的星座，所以也不會在意感情中的問題，在1月下旬，摩羯座的人開始解決感情困擾，自己的感情中，摩羯座也不會被那些問題繼續困擾，在愛情的路上，也會更加清晰，不放棄自己對愛情的希望。

獅子座

獅子座在愛情中的獨立，會表現得很明顯，越是優秀的獅子座，就越不害怕失去愛情，所以在愛情中的獨立，往往才是讓人欣賞的一點。在愛情中的獅子座不喜歡太過於粘人，也總是分得很清楚，在愛情中的獅子座更喜歡依靠自己達成目的。

天秤座

天秤座的人熱情大方，面對任何事情都表現得從容優雅，透著不凡的氣質，但是天秤座在面對愛情的時候總出現猶豫不決，他們是典型的矛盾體，選擇恐懼症，對愛情並沒有那麼理智，所以在面對喜歡的人沒有勇敢說出來，天秤座錯失過不少脫單的機會，他們自身也對自己行為非常懊惱，事後一直悔恨之中。 經歷過感情挫折，天秤座學會作出正確的選擇，在他們心目中，愛情是一種信仰，再次迎來愛人的時候也會勇敢表白，捲入在愛情的紛擾之中，只想和對方牽手一生。在12月下旬中，喜鵲歡唱，錦上添花，天秤座將擺脫單身，情緣美滿。
