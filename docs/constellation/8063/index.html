通過塔羅牌，可以給你解答多方面的疑惑，煩惱纏身的可以試一下

2019-12-03 2,681條評論 4,718次閱讀 3647人點贊

塔羅牌被廣泛地運用在推算預測、心理分析、精神治療三種層面上：

一、推算預測方面：

預測你的人生、愛情和未來。它能發現你的個性問題，並深入問題的核心，指出你所面對的危險、障礙、機會和最佳的成功方向。它讓你了解人生的機遇和運程，為你指引一條可供參考的光明大道。

二、心理分析方面：

塔羅牌深入一個人的內心，挖掘其心理和精神的隱秘發展方向。它提供整體的人生經驗，配合每個人的心理特徵和成長曆程，提供一個機會讓人能夠進行自我心理分析，正確認識自我，探索人生的神秘歷程。

三、精神治療方面：

塔羅牌揭開神秘的精神層面，透過撲克牌的象徵圖案打開潛意識之門，一般人可以通過它的神秘暗示，找到自己的力量、超越或限制，指引自己的思想、行動朝著更積極的光明一面，使內心和外在世界充滿和諧與平衡。

塔羅可以占卜些什麼：

  • 感情：解答暗戀、熱戀、異地戀、失戀複合等等的感情疑惑
  • 緣分：測你的正緣會是一個怎樣的人、什麼時候在哪裡相遇
  • 桃花運：測你未來一段時間的桃花運勢如何
  • 事業：測你事業的發展情況、工作狀態
  • 生活：生活上各方面的解惑

占卜規則：

  1. 排除雜念，保持內心的平靜
  2. 心裡默念你想要占卜的問題
  3. 點擊這裡即可開始占卜

獅子座

獅子座是一個很容易讓人產生安全感的星座，這也是由於他們自身性格比較強勢，在任何事情面前都有著絕對的把握，所以和獅子座的人相處，人們不由自主就會被他們自身強化的能力所吸引，而且獅子座對於自己身邊的人，也非常有責任感和義務感，他們願意主動關心自己的朋友，當對方遇到困難的時候他們也會伸手幫助，而自己面對人生的時候，永遠都會用一種積極樂觀的心態去面對生活，所以當他們身邊的朋友與他們相處時間變久的時候，自然也會被他們這種正面情緒所感染。

處女座

處女座的人容易有這個極強的洞察力，所以他們很會在生活中，觀察對方的真實想法以及性格，人們覺得處女座是一個善解人意，懂得體貼他人的心中，也是因為處女座會用自己的善良去選擇幫助對方。而且處女座別人也會有這一種同感心理，當對方需要傾訴的時候，處女座絕對會做一個合格的傾聽者，而且他們也會憑藉自己溫柔樂觀的一面去感染對方，讓對方的生活重新開朗起來。

射手座

來自火象星座的射手座，天生性格風風火火，在他們的生活中就很少存在煩惱這一問題，任何事情射手座的人都不會放在自己心中，所謂往事隨風，一些過去的事情，射手座就會讓他過去，也不會過於糾結之前的結果。所以在一群人之中，射手座也會輕易成為別人的開心果，當別人煩惱的時候，射手座也會用自己樂觀看到的一面去感染別人，希望別人能夠忘記煩惱，重新振作起來。
