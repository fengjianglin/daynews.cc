十二星座雙商排名，誰才智雙絕，又是誰雙商墊底？

2019-12-18 331條評論 2,177次閱讀 631人點贊

情商智商一直都是人能否成事，或者是評判一個人的標準。雙商是什麼，相信大家都很清楚，情商和智商。有人說情商更重要，有人說智商才重要。其實在小編看來真的是缺一不可，畢竟這個社會太複雜。

今天我們就來說一說十二星座的雙商綜合排名，誰勇奪桂冠，又是誰慘遭墊底。

​

白羊座

智商：四顆星

情商：一顆星

白羊座是那種比較頑皮的星座，但是智商絕對夠用，他們就是那種玩著學卻也成績不錯的人。因為白羊座很有靈性，所以智商不錯。但是在情商上，白羊為人太耿直，有啥說啥難免會有點沒有顏色，被人詬病情商低。

​金牛座

智商：三顆星

情商：兩顆星

其實金牛座是智商情商比較平均的人，但是相比起情商，金牛的智商反而更高一點。因為金牛是那種邏輯思維很好的星座，很多東西都是一點就通，比起文科可能金牛理科更好一些。情商金牛是有的，不過時好時壞。因為金牛不太擅長奉承和討好，說白了就是知道該幹什麼但是卻身體僵硬，違心的事情做得不漂亮。

​雙子座

​智商：四顆星

情商：四顆星

雙子座的智商真的很不錯，他們悟性十足，靈性十足。就是專一程度不夠，也許成績不夠好，但絕對是三心二意的鍋，絕不是智商的鍋。在情商上雙子更是一騎絕塵，活躍氣氛小能手，行走的段子手，有他在就一定不會冷場。

​巨蟹座

智商：一點五顆星

情商：四顆星

巨蟹座屬於那種笨鳥先飛的類型，他們知道自己沒那麼聰明所以會默默的努力。但是在情商上，巨蟹座真的很不錯，因為巨蟹的共情能力很強，總是能夠感同身受。不會讓別人下不來台，也不會咄咄逼人開一些不合時宜的玩笑，因為他會換位思考。

獅子座

​智商：兩顆星

情商：兩顆星

其實獅子座智商情商都不算是特別高的類型，但好在獅子座有一顆真誠的心，所以有很多人願意和獅子交朋友，但是在交友方面獅子座有的時候太過於耿直，甚至有的時候會因為自尊心太強而覺得對方的玩笑過分，當場翻臉。這一點會讓很多人覺得，獅子座的情商不行。

​處女座

​智商：兩點五顆星

情商：五顆星

處女座是一個在不熟悉人的面前是不太愛說話的類型，但是處女座很會注意細節，總是在細節上給予對方尊重，讓對方覺得舒服。雖說處女座智商沒有那麼出色，但是處女座情商絕對滿分。就是因為處女座注重細節，所以他會注重對方的感受，如果你留心觀察就會發現娛樂圈很多以情商聞名的明星都是處女座。

​天秤座

​智商：四顆星

​情商：五顆星

作為一個風象星座，天秤座的情商絕對是風象星座中最優秀的。在十二星座中天秤也是名列前茅的，因為天秤不爭不搶老好人的形象真的深入人心。他脾氣好，而且不喜歡爭執，擅於當和事佬，還很幽默。智商也是毫不遜色，就是天秤沒那麼認真，否則學業上也一定不差。

天蠍座

​智商：四點五顆星

​情商：四點五顆星

​天蠍大佬智商情商絕對是雙高，因為天蠍的洞察力很強，直覺也很強。所以他知道什麼時候該做什麼事，什麼話最得體，說什麼能不惹對方生氣。因為他能夠感覺到什麼才是對方的痛點和癢點，智商情商絕對的沒話說。

射手座

智商：三點五顆星

情商：二點五顆星

射手座智商不差，但是射手座總是在學習的時候開小差，所以影響了成績。情商射手總是發揮不穩定，有的時候情商超高，有的時候總能開出不合時宜的玩笑，情商忽高忽低只能取個平均值。

​摩羯座

​智商：兩點五顆星

情商：兩點五顆星

摩羯智商情商都是處於中間水平，但是不要忽視摩羯這個中間水平。因為摩羯的潛力無窮，一旦爆發實力不容小覷。

​水瓶座

智商：三點五顆星

情商：五顆星

水瓶座的智商真的不低，但是水瓶座的腦洞太大，總是不和老師同學在一個水平線上，別人理解不了水瓶所以覺得水瓶智商沒那麼高。情商上面水瓶可以說非常棒了，上能活躍氣氛，下能給人台階，看透不說透是水瓶的智慧。

​雙魚座

​智商：一點五顆星

情商：兩點五顆星

雙魚不管在情商還是智商上都敗在了太感性，因為太感性所以藝術才能比較好，但是學習方面就沒有那麼高。情商雙魚也是時高時低，因為有的時候太過感性沒辦法發揮出真實水平。
