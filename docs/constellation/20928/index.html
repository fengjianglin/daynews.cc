你的長相，不僅藏著你的生活，還藏著你正在追求的人生

2019-12-19 1,306條評論 4,421次閱讀 1045人點贊

俗話說：「相由心生」。一個人的相貌會隨著心念的改變而改變。這個世界，沒有無緣無故的美，也沒有莫名其妙的丑。相貌，是歲月在人身上的沉積和凝聚。你的長相，就是你靈魂的模樣。

暴躁的人，眉頭容易緊蹙；溫和的人，嘴角掛著微笑；蠻橫的人，眼中充滿戾氣；友善的人，總是一副安然的模樣。雖說人不可貌相，但在面對不了解的人時，我們往往是從外貌開始打探，從面相決定第一印象。

一個長相精緻的女人，一定過著精緻的生活。再忙也會在出門前化妝，再難也會將嘴角上揚。堅持健康的生活習慣和樂觀的生活態度；秉持獨特的審美眼光和成熟品味；修得豁達的心胸和深厚素養……總之，好看的長相背後，是從不懈怠的愛自己。

英國文學家說：「一個人的臉，就是一個人價值的外觀它不僅藏著你的生活，還藏著你正在追求的人生」。

人生茫茫，我們讀過的書，走過的路，遇見的人，都會在生活中留下印記，在潛移默化中改變我們的氣質。

著名主持人董卿，就是這段話最好的例子。她用行動向我們展現「腹有詩書氣自華」，也用經歷向我們證明「若有詩書藏於心，歲月從不敗美人」！

我們經歷過的美好與苦難，跨越過的平原和山川，包容過的世事和艱難，都會歷練成平和、柔美、睿智和醇厚。外化成你的面相。

所以，若你靈魂挺拔，長相自然不會太差；若你心底明媚，便能散發出曠日持久的魅力。
