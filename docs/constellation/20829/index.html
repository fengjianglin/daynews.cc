12.17星座運勢：你是否還記得自己曾經也是一個瀟洒自由的年輕人

2019-12-17 743條評論 2,585次閱讀 534人點贊

【前言】姑娘們一定要懂得：先愛己，後愛人。這是很多姑娘在感情里做不到的事情。總是愛著愛著，就把天平移向了對方，就開始關注對方的一切，甚至開始控制對方，忘了自己曾經也是一個瀟洒自由，來去如風的存在呢。

【今日運勢】

【白羊】你在今天會收到一些期待已久的好消息，或者是朋友給你送來一些小禮物。但是在感情中可能會出現讓你一些不愉快的事情，或者你可能發現你的另一半有了一些曖昧對象，而你對這一切突如其來的變化而感到惶恐不安以及六神無主，這大概是感情危機。

【金牛】今天不適合做重大的決定，因為今天的你有點猶豫不決，飄搖不定，沒有立場。工作上你要學會怎麼樣利用自己現有的資源去做一個整合以及合理利用，可以嘗試向新的一些領域涉獵，會有貴人來到你身邊幫助你。感情上單身的金牛今天的桃花運勢是不錯的，可以好好把握一下。

【雙子】今天心情有點焦慮不安以及煩躁，要注意控制好自己的脾氣，否則會因為自己暴躁的脾氣而傷害了身邊的人。今天可以嘗試參加一些社交活動或者是一些慈善募捐活動，到街上做做義工，這樣子你反而可能會有意想不到的好運降臨。感情上的雙子今天要避免和另一半發生爭執或者是冷戰。

【巨蟹】今天的你適合自信一點，打破自己現有的局限性，去嘗試一些你以前只能在腦海中設想的想法，今天都很適合付諸實踐哦。感情上的巨蟹座適合自己積極主動地去爭取你想要的東西，而不是被動地等待著對方，感情是雙方向的輸出以及收入。

【獅子】今天要學會過濾一些外界的不利因素或者是不同的聲音，你可以傾聽別人的想法，但是你要有自己的主見。工作上會有一些你期待已久的機會會在今天找上門，一定要抓住機會好好把握。工作上建議你自己結合自己的精力以及時間安排，力所能及的去做一些事情，注意身體健康。

【處女】你要學會真實地去面對自己，以及接受自己的不足。人無完人，這是自古以來的名言。同時在與親密的人相處，你的家人，你的好朋友或者是另一半，都要學會給他人留有空間和餘地，而不是過度的去控制他們。感情上一定要懂得雙方怎麼樣去溝通，互相包容。單身的處女座感情上非常的有利於你們脫單咯。

【天秤】工作上你能夠讓項目的進度或者是工作上的進度牢牢的控制在自己的理想進度之內，大多數結果都是好的，你能夠非常出色的完成你的工作任務，自信滿滿，但是切勿驕傲。感情上的你有點過度於糾結了，比較優柔寡斷，對另一半的一些行為或者是怠慢都會耿耿於懷。

【天蠍】今天的你工作動力十足，精力充沛，甚至能夠在繁忙的一天工作結束後進行一些健身活動。工作上的你比較注重細節，並且對細節把控得十分精準，這讓你在工作上優人一等。感情上的天蠍今天則不好過，總覺得你的另一半不支持你，總是會背叛你，或者是莫名其妙有一種被背叛的感覺，疑心病比較重。

【射手】你非常享受當前的生活，並且樂於去享受當前生活，帶給你的饋贈。工作上以及感情上，你都會非常積極主動地去尋找對你有利的機會，並且將自己的優勢充分地展現出來。單身的射手，在今天比較利於去告白或者去約心儀的異性出來約會，脫單的幾率非常的大。

【摩羯】感情中的你比較傾向於主動的去付出，去為這段兩性關係付出，但是付出也是要有個度的。今天特別要注意一點哦，可能會跟你的家裡人包括你的父母發生一些不愉快的事情或者是爭執，但可能也只是因為一些生活上的瑣事。建議你控制自己的情緒，不要口出惡言傷害你最深愛的人。

【水瓶】今天工作上的執行力度以及判斷力非常的不錯，敏感度也是非常的不錯，能夠輕易的覺察一些別人不能夠覺察的細節上的漏洞，從而讓你的工作得到他人的賞識或者是優人一等。感情上你要學會懂得去感恩，而不是過度的去索取對方的付出，你也要多多關心對方的生活或者是心情。

【雙魚】不好不壞的一天，但是在為人處事上你會比平時更加的圓滑周到，這樣子也有利於你的工作或者人際關係中的經營。感情上的你還是比較堅持自己的原則，寧缺毋濫。近期的生活可能會讓你覺得比較的起伏不定，不算好也不算壞。再堅持一下，好的生活總會來臨。

【結語】我的男孩他不是玩具，也不是長期飯票，他是我唯一的寶貝，我會給他所有的溫柔，在我這他永遠都有台階可以下。
