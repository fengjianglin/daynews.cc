巨蟹座

巨蟹座，去年是本命年，命運不是特別的好，過的非常的辛苦，但是卻沒有賺到錢，生活依舊是簡單的不行，想吃的，想穿的都不能享受。今年他們真的是把不好的運氣全部都守著走了，終於有好日子了，上半年的時候並不明顯，但是12月20日起，運氣爆棚，財運飆升，事業會蒸蒸日上，達到一個峰值，這樣的人生對於巨蟹座的人來說應該算是最好的歸宿了。所以在未來的一段時間巨蟹座的朋友一定要把握機會，錯失了發財的機會，就不知道什麼時候能再次發財了呢。

獅子座

獅子座的人自信又好強，擁有非常強大的氣場，而且他們行事非常有自己的風格。因為前段時間運勢不是很好，讓他們過得不是很順心。不過從12月20日起，他們將會好運不斷，尤其是財運，在投資理財方面會有不小的成就，其他的方面也會有好的結果。從這以後，獅子座將會霉運散盡，生活順風順水。就算是有水逆到來，也擋不住獅子座的好運勢，事業財運雙豐收，喜訊連連。

水瓶座

水瓶座有自己的看法和見解，不在乎別人的想法，不管別人說什麼，他們都會按照自己的想法做事，特立獨行有主見，還有豐富的想像力，別開生面，提出很多有建設性的意見。水瓶平時一副冷淡的態度，其實還是很希望能夠實現價值得到肯定的，水瓶座霉運消散之後，在12月20日起就會迎來好運，事業上得到認可和肯定，接觸到更有趣的事物，努力賺錢，豐富自己的生活，非常快樂。

把握命理大運流年的變化，順勢而為，正所謂，知命而樂，知運而通，招財、轉運、旺姻緣！
