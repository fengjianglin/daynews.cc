金牛座：傷心時會陪著你的5個星座，為你擦去眼角的淚水

2019-12-16 776條評論 1,988次閱讀 362人點贊

每一個人傷心的時候都會想有人來陪，作為大好人的金牛當然也是一個在別人傷心的時候挺身而出的一個，他們會不求回報的陪著他們認為重要的人，讓他們走出傷心。你若是傷心的時候，你身邊會不會出現一個陪伴你的金牛來為你擦乾眼角的淚水。那麼，就讓我們來看看傷心時總會有金牛來陪的5個星座都有誰吧！

第五名：天秤座

別看天秤平時對所有事都一副毫無興趣的模樣，可他們的心思也是極其細膩的，有時在不經意間也會因為身邊的某些人或是某些事而傷心，雖然天秤的傷心並不會表現出來，可我們熱心腸的金牛也是會在與之相處的時候察覺出天秤的不對勁，因而陪在天秤身邊來讓他們在傷心的時候不會過於難熬。

第四名：水瓶座

聰明的瓶子在傷心的時候總是喜歡找一個沒有人的地方，默默的讓眼淚順著眼角滑落，這樣的瓶子往往總會牽動金牛的心，讓金牛們會為之感覺到心疼不已，因而過多關注身邊的瓶子。一旦關注了，就越發的想要在瓶子傷心的時候找到瓶子，拉著他們走出那些讓他們傷心的陰影之中。

第三名：摩羯座

總喜歡小心翼翼的摩羯，也是最缺乏安全感的一個，太過敏感的他們同樣也是極其容易因為身邊的一些人而傷心的，凡事總喜歡放在自己的心中讓自己獨自傷心淚流，這樣的摩羯總能讓金牛發現並且小心去呵護，當然摩羯傷心的時候，也許會主動找上金牛，來讓金牛陪在他們的身邊。

第二名:雙魚座

魚兒們總是最容易受傷的，過於善良的他們在付出很多之後才會知道面前的人並不值得他們掏心掏肺，可這個時候已經晚了，他們總歸是已經被深深的傷害了，默默哭泣的魚兒這時會讓金牛內心深處也跟著難過不已，能夠牽動人心的雙魚又怎麼可能不讓金牛在她們傷心的時候陪伴在身邊呢！

第一名：巨蟹座

愛心滿滿的巨蟹在傷心的時候，那就意味著那些不珍惜他們的人徹底的把他們給傷了個徹底，否則的話巨蟹絕不會露出如此脆弱的傷心面容，這樣的巨蟹讓金牛看到的話怕是會徹底的為之心碎，因為平時的巨蟹實在是太過的有愛心，故此在冥冥之中也會讓金牛對他們多出很多的愛心，心甘情願的陪著巨蟹傷心。

​

當你傷心，你會想起誰呢？你的身邊會不會被那個隱藏在暗處的金牛關注著，時刻怕你獨自傷心難過。
