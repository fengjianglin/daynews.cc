巨蟹座在遇到事情的時候不慌張，總能以平常心去對待。巨蟹座的好運氣多來自於自己的好心態。處事不驚的作風易得到他人的肯定和讚賞，好的心態能讓自己有好的發揮。心態好的人對生活總是抱著希望，相信美好光明的一面，不看重得失，心態好的人心情也大多是好的，有好心情才能有好運氣。
