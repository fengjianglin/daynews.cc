下周過後，回想過往，難以入眠，3大星座輾轉反則，破鏡重圓

2019-12-10 296條評論 2,958次閱讀 1640人點贊

有一種單身，是為了愛可以不顧一切。他們所等候的只是讓自己等待的那個人明白自己的心意，然後選擇自己，接著，牽著手，一起走下去。不怕別人的嘲笑，因為他們始終相信，那個等待的人，會到你身邊，擁抱自己。每一次想你，我都得提醒我自己，如果你想和我聊天的話，你早就開口了。

我曾為你流過淚，為你喝醉過，為你一次次在街頭獨行，為你一次次在深夜傷心歌唱！難到我付出的一切真的無法感動你，也許你給我的笑容，對我的成若都只是為了同情我的多情，安慰我的眼淚！下周過後，回想過往，難以入眠，3大星座輾轉反則，破鏡重圓。

摩羯座

摩羯座的人任何事情都非常的專註，認真，不想被外界打擾。他們從來不在人群中彰顯自己的能力，所以別人從來都不知道摩羯座其實是一個非常有能力的人。他們少言寡語，沒有所謂的交際圈，他們很少真心實意的和一個人交朋友，看似高冷實則低調，往往被大家所忽略。

摩羯座的人最最在乎的是自己的事業，他們對於別人的評價毫不在乎，哪怕有人說他們壞話。就因為是這樣的人，所以對待感情格外的認真，總是心系從前，渴望重歸於好。

巨蟹座

巨蟹座很會照顧人、居家顧家的星座代表，他們很看重自己的家人，也會願意為了家人去好好努力，希望能用自己的努力去給家人們換取更美好的生活。剛開始接觸巨蟹座的時候，總覺得他們會有些高冷，但實際上他們只不過是外冷內熱，當和你熟了之後，就會卸下自己高冷的外殼，變成一個可愛的小話癆。

心軟的巨蟹座不忍心將舊情拋棄，他們只會想辦法將對方找回，在這種情況下，巨蟹座是不會發展新感情的，他們可以結交新的異性朋友，但是談情說愛，他們是很難辦到的，即便有人追求，巨蟹座也會躲避，他們不想花費時間再去了解一個人，相比之下，舊情是重要的。

天秤座

天枰座表面上很浪漫骨子裡卻比較實際，剛與天秤座陷入情網時，覺得一切都太浪漫、太華美！天枰座還有一個性格特點，那就是神經質。心思正是一座靈敏的秤子，只要一陣微風，便足以使之輕顫不已。但儘管這樣，對待愛情還是很認真的，分手後的天秤座表現得很淡定，好像一切都是意料之中的事情，沒有挽留，沒有傷心，給過對方很多遷就。

我們都以為天秤座分手後不再愛對方，實際上天秤座想要成全對方，他們希望對方離開自己能幸福，沒有想過對方還會回來，天秤座只想忘記過往。如果你主動向天枰伸手，他們向來都不會對另一半下狠心，在舊情面前也是如此。
