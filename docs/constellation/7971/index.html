12星座專屬“多肉植物”獅子座逆境不死鳥，天秤座呆萌小白兔

2019-12-06 472條評論 1,496次閱讀 431人點贊

1、白羊座

白羊座的專屬多肉植物是可愛的虹之玉，這是一款生長的很快的植物，特別是在春夏季是它們的主要生長季節。就像是白羊座的人一樣特別的可愛。特別像是提子一樣，忍不住想要咬上一口。

2、獅子座

獅子座的專屬多肉植物是不死鳥，這款像是一隻孔雀一樣，特別的有生命力，是一個生命力旺盛的植物，正如獅子座的人一樣，特別的能在逆境中快速的去成長起來，變得越來越強大，越來越有能力。

3、處女座

處女座的專屬多肉植物是姬朧月，這種植物非常的奇妙，平常看起來是綠色的，在接收到了陽光會變得朱紅待褐色，是一款特別奇妙的植物。

4、巨蟹座

巨蟹座的專屬多肉植物是寶石花，這是如同寶石一般，但是因為長得特別的像是一朵奇妙的花朵，特別的好看。

5、雙魚座

雙魚座的專屬多肉植物是七色堇，這是一款特別好看有著五彩繽紛的多肉，鮮艷的顏色真的是讓人的眼球都離不開。你們有沒有見過這麼漂亮的植物。

6、天秤座

天秤座的專屬多肉植物是碧光環，這款植物特別的萌，就像是一隻小白兔一樣，活蹦亂跳的，特別的呆萌。看到就覺得要被它們的外表萌化了。

7、金牛座

金牛座的專屬多肉植物是姬玉露。這款多肉就像是水晶葡萄一樣，晶瑩剔透的，特別的好看，又像是翡翠一樣，特別的綠，好看級了。

8、雙子座

雙子座的專屬多肉植物是猴面小龍蘭，這款植物特別的有創意，有著小猴子的臉蛋。真的是非常奇特的一款植物，正是適合雙子座這麼古靈精怪的人。

9、水瓶座

水瓶座的專屬多肉植物是嘴唇花。這款植物酷似一張嘴唇，因此才被命名為嘴唇花。紅紅的特別像是人類的嘴巴，是不是很奇特啊！

10、摩羯座

摩羯座的專屬多肉植物是薄雪萬年草。這是一款怕熱耐寒的一種多肉，它們適合生長在暖和的地方，整片的萬年草真的是非常漂亮的。不過對於有密集恐懼症的人來說還是挺害怕的。

11、天蠍座

天蠍座的專屬多肉植物是冰嬌，這款植物看起來冷冰冰的，其實內心是特別的熱情，有著一種出淤泥而不染的感覺，正是和天蠍座的人一樣，外表冰冷，內心卻非常的細膩。

12、射手座

射手座的專屬多肉植物是火祭，這款有著火紅火紅如同焰火一般的植物，和射手座的人一樣充滿著幹勁，特別的有活力的一個星座。也和這款植物一般。
