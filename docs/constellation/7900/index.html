獅子座

本周是獅子座調整身心的一周。心緒不穩，情緒起伏。本周獅子座的心弦好像是被什麼東西牽引著，如若必須在本周作出重大判斷，請慎之再慎，工作有境遇反覆的跡象，所以可能會情緒不穩甚至失控，需增強定力和緊急情況下的判斷力。有小幅破財，但並不是壞事，有破財免災的勢頭。這周身體需要加強鍛煉。

處女座

本周一個存在很久的問題可能會最終達到一個很棒的結果。這個結果可能會在本周達成，要不然就是有關這個結果的某個消息會在本周傳達到。這肯定是值得慶祝與展望的，是期待遠非恐懼。一定要找個時間慶祝一下，因為這可能是你人生很重要的轉折點，也可能對其他什麼人的人生有著同樣重要的影響。本周對你某個沒時間進展的興趣愛好也帶來一個機會，讓你不能再找借口，最終可以真正開始進行下去，而這會給帶來許多快樂與愉悅，會增你的生命的維度。

水瓶座

本周水星會與金星在你家宅宮內牽手，散播暖意。當金星在此地逆行，她將與你關係宮內的火星構成棘手的角度。你可能會與現任或過去的伴侶關係緊張，但也會得到海王星憐憫之力的幫助。記住，金星逆行對所有人都有影響，你也許需要對某些人敬而遠之。
