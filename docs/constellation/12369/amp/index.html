1、【日運的影響時間是怎樣的？】

塔羅提取的是短期能量，這個能量可能會移前或錯後。比如今天的日運，當然可能是你當天的事，但也可能是你昨天發生過的事，或者明後天發生的事。

2、【到底「意向」是什麼？】

「意向」是指任何含有這個元素的事物。比如熊，那就是含有熊的耳釘、衣服、熊餅乾、去畫展看見含有熊的畫，等等等等。你不用真的去戰鬥民族牽一頭熊回來:)
