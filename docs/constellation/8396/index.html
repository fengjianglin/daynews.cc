水瓶座人聰明伶俐，智商極高，在工作上總是佔上風

2019-11-22 196條評論 1,116次閱讀 83人點贊

水瓶座（1月20日——2月18日），守護神是天王星，是風象星座。所以具有創新、多變、倔強、反叛的主要特徵。

這個星座常被稱為「天才星座」或「未來星座」。因為它的守護星是天王星，而希臘神話中上通天文、下知地理，並有預知未來能力的智慧大神——烏拉諾斯，是它的守護神。所以他們具有前瞻性、有獨創性、聰慧、富理性，喜歡追求新的事物及生活方式。他們的心胸寬大、愛好和平，主張人人平等、不分貴賤貧富，不但尊重個人自由，也樂於助人、熱愛生命，是個典型的理想主義和人道主義者；他們深信世上自有公理，所以常有改革的精神。

水瓶座人由於主宰行星是天王星，是精神的喚醒者，是精力和變化的行星，可能會使這些人很早就呈現在大眾面前，並賦予他們有吸引力的靈性。天王星的雙重影響，削弱了這個星座的一些不良影響；水瓶座人的體質比較優秀，環境適應能力強，身材勻稱，渾身散發著個性美，愛好和眼光獨到，尤其在服飾和髮型等方面總能引領潮流。

水瓶座的男性具有敏銳的洞察力和去偽存真的判斷力，正義感極強。通過才智和努力，水瓶座男性可以獲得順利發展；如果你在企業工作，應盡量避免在企業內部，因為聯絡業界同行，與團體或會議進行外聯更適合你發揮才能。

水瓶座女性，同情心豐富，對朋友友情深厚。 在討論會上，總是會率先提出自己的意見；對於朋友的事情會赴湯蹈火全力以赴，對於自己的事情，即便有風言風語，也會默默地承受忍耐；水瓶座人獨創性極強，意見不一致時，總會把自己的想法堅持到底。當最終達到預定目標時，或意見和提議被採納時，會產生幸福感和滿足感。

水瓶座人聰明伶俐，天生就比別人聰明，求知慾強，喜歡探索新的事物。他們思維獨特，腦袋裡裝滿了天馬行空的創意，只有別人做不到的，沒有他們想不到的。他們喜歡與人交流想法，分享知識，並且會吸取別人身上的優點；他們總有稀奇古怪的點子，他們思想異於常人，他們智商極高，總能想到別人想不到的東西。對於新奇的知識和技術，他們往往會表現得很有興趣，可謂是活到老學到老的典範了，他們似乎總是可以把大腦開發到極致，而且很少有人能像水瓶這樣擁有著多方面的知識與跳脫的思維。所以他們在工作當中，總是會佔上風，比別人高出一籌。
