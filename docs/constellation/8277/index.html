難得糊塗，遺忘不快，這些星座珍惜情誼，感恩生命中所有遇見

2019-11-25 165條評論 2,050次閱讀 360人點贊

人的一生中總會有幾次遇見不同尋常，有的人也許會因此而放不開，對於一些不甘心的事情，總是會顯得斤斤計較，那些不美好的回憶也都會保存在他們的心裡，隨著時間這些回憶並沒有因此而消退，而是會更加的讓他們變得凌厲起來。有的人因此可能還會因為這些遇見而改變自己，變得不再像以前的那個單純善良的自己。但是有的人他們卻難得糊塗遺忘那些不快樂的回憶，在十二星座中有三大星座就是這樣的人，他們總會珍惜生命里的美麗風景，感恩一路相陪的人，那是遇見對的人還是錯的人，他們都會抱有一顆感恩的心。

天蠍座

天蠍座可能在一開始的時候還會表現的有一些嚴肅，但是後來隨著年齡增長他們可能會變得越來越溫和，他們很容易看透一些事情，能憑藉自己的第六感很容易就能夠輕易地判斷一個人的本質，但是他們卻看透不說破，他們不是一個熱心腸的人，不會幫別人去拆穿事實和真相，現在的人看來他們有的時候會有點傻裡傻氣，但是他們卻是清醒的，他們倆有著一顆包容的心態，感恩生命中所有的遇見，那是那些開心的事情，還是不開心的事情都會去接受和包容。

摩羯座

摩羯座的人不喜歡就糊弄別人，他們也有沒有精力和時間去搞一些小的心思和花樣，他們把自己所有的精力都放在自己的事業和目標中，在人際交往中摩羯座習慣保持這種安全感，他們的內心非常強大，可以包容那些對自己不太美好的回憶，能屈能伸從來都不會讓自己特別的勞累或者難受，只要能夠達成自己的目標，那麼一切都會讓自己白費，他們在情感方面其實並不會顯得特別的較真，給人一種難得糊塗的感覺，他們永遠都會秉持著一種包容的心態，感恩所有的遇見。

水瓶座

水瓶座的智商很高，因為他們的宮位在11宮福祿宮，所以他們註定會有一段不平凡的人生，他的行為總是特立獨行，有著很強大的創新思維和能力，但是他們的思想就會有一些悲觀，他們的積極樂觀是家裡在看透很多事情的基礎上的，對他們來說每一段感情都不會特別的圓滿，同時每一個人都會特別的完美，對於每一件事情或者每一個人，他們都有著自己判斷，但是他們表面上卻不會輕易點破，仍然會交許多的朋友，畢竟他們喜歡與人打交道，對於那些不愉快的記憶，他們通常很快就會讓他煙消雲散。

總而言之，咱們這三大星座他們並不是那種特別放得開的人，只是對某些事情的理解和判斷有著自己的主見，雖然他們的微商給人一種什麼都不會在意的感覺，但其實他們心裏面 比誰都清楚很明了。
