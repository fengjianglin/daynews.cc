沈嶸2020開運方法公開！　衣服穿對5顏色：招來潮流正向觀感

天天要聞
2019-12-20

記者林彥君／台北報導

「神明代言人」沈嶸今（19日）出席紅心字會老人中心舉辦的公益活動，沈嶸現場捐款10萬元給紅心字會。沈嶸提醒長輩平時也要多穿喜氣的顏色，例如紅、橘紅、桃紅等，有助於心情開朗，可祈求「藥師佛菩薩」或「藥王菩薩」的護佑，幫助維持長輩的身心的健康，切記有病痛就要就醫，切勿延誤醫治。

沈嶸（中）出席公益活動。（圖／紅心字會提供）

此外，沈嶸老師也針對2020年要怎麼樣才能擁有正能量的方法，表示要強化工作知識技術，學習最新運作方式，例如直播銷售、網紅路線、或以社群影片打開知名度，提高能見度，學習各式最新銷售技巧，懂得銷售就是創造財富的關鍵。廣結善緣和人脈，結合四周支持你的人與團隊一起打江山，雖說同溫層看起來像一群人在互相取暖，但也有其功效，團結可以把石頭變黃金。

至於穿著部分，要穿對顏色讓自己運勢更旺，灰紫紅色、哈密瓜的橙色、古典黃色、寧靜藍色、薄荷綠色等，將這些流行色穿在身上，將帶給人走在潮流上的正向觀感，有助於讓自己運氣更好。想要成功、創造更大成就、賺大錢就一定要請掌管事業的武財神「關聖帝君」，可以每日向「關聖帝君」上三炷香，誠心供奉「關聖帝君」可以幫助您在事業上能夠有更順利的發展並興旺家運。

沈嶸出席耶誕公益活動。（圖／紅心字會提供）

而針對上班族而言，最重要就是能夠發揮價值讓人看見，如果空有長才不懂適時表現，反而會被埋沒，擁有良好的溝通表達與說服力很重要，讓自己被看見就能幫助自己越來越旺。可以向「五路財神」和「彌勒佛」是最受上班族喜愛的神明，「五路財神」掌管每日的進帳，「彌勒佛」掌管未來的大財富，與這兩位神明維持良好互動，讓兩位神明的護佑交乘為您帶來持續穩定的大財富。

最後，沈嶸老師說：社會上有很多命理老師是真正敬業地服務人群，我希望能夠成為命理界的榜樣，每個月持續捐贈算命收入做公益，就是對社會的回饋，正因為曾「代父入獄」看到許多弱勢的女性因為沒有錢，生活困苦因此犯下錯事，因此特別想要幫助社會上需要幫助的人，不論是老人小孩或流浪動物，我希望拋磚引玉，讓更多人跟隨魔法事業把愛傳遞出去。

沈嶸在活動中分享開運方法。（圖／紅心字會提供）
