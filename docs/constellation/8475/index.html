天蠍座

天蠍座的聰明從來就不是浪得虛名，能說會到的本事也不是造假的。未來一個月，天蠍座如果參加工作會得到良好的回饋。如果已經工作多年，那未來一個月就是他人生的一個轉折點。曾經天秤座因為自己的失誤而不小心犯的錯誤會在12月初全部挽回。天秤座的學習也會取得一個好的成果，過往的低迷將不復存在。

天秤座

天秤座從來沒有放棄過各種嘗試，他失敗的次數並不在少數，但他沒有哪一次因為失敗而停滯不前的。他相信自己，也相信生活。天秤座會總結每一次失敗的經驗，直到尋找到最適合自己的一種方式為止，他的努力不是盲目的，而是富有節奏和韻律的。12月初，天秤座甩掉了過去種種失敗的陰影，他的人生厚積薄發。

巨蟹座
