【天眼】並不神秘，是為了忽悠人而故意把它吹噓得天花亂墜

2019-12-21 422條評論 1,861次閱讀 518人點贊

人們在生活中往往看不起老實做人的人，認為他們窩囊，實際上，一個人在這個人間所能做的所有事情中，以老實做人為最難，也以老實做人需要的勇氣為最大。

由於我們的祖宗中一些自私無恥的不肖者，一萬年來，一直想把礻申塑造成一個高高在上、超出人的能力、又能審判人、能收拾人的模模糊糊的一個東西，用它來嚇唬人。一直這樣宣傳，時間久了，信的人就以為真有這種東西，不信的就堅決否定有這種東西，認為這是迷信。人類的未來，就是這樣被這些不肖子孫給毀了的。

所有的先知，就是比普通人先懂道理的人，其實都是來告訴大家生命的真相的。

人類的絕大多數不知道生命的真相，太久了。祖宗們一直這樣忽悠、誤導後代，人從出生，從父母長輩和老師那裡學到的關於生命人生的知識多是假知識，在假信息中長大的人類，怎麼可能懂道理呢！

可憐的人啊，走到現在，突然有一天聽到本來是稀鬆平常的一點真相，聽到有人告訴他他自己的真實身份是個什麼東西，他都驚奇，感覺是不可能的，認為是荒唐可笑的。

我們平常老說，人人平等。但是，又不知道為什麼要這樣說。人人憑什麼會人人平等呢？人人平等肯定是有深刻的原因的嘛。人人平等的原因，只能有一個，就是大家的本質、本性、真身都是一模一樣的。只要我們的本質、本性、真身有一丁點不一樣，我們就不會人人平等。這是多麼簡單的一個邏輯思維問題啊！多麼明了的問題啊！

能看到這個簡單的邏輯思維問題的人，叫開了天眼。開天眼，不是那些搞迷信的人宣傳的那麼神神秘秘的。開天眼，其實就是可以從現象中可以找出一些簡單的正確的邏輯。

只能看到這個物質世界的各種現象，只能在表面現象上打轉轉，找不到物質現象背後的正確邏輯關係的，叫肉眼看東西很亮，叫天眼還沒有開。

聽到別人說，要找到物質現象背後的正確自然規律、公理、邏輯推理，可以理解，而聽到要找到天道法則，就理解不了的，就是天眼還沒打開。這種就好比，只能理解爸爸是爸爸，聽到別人喊爹、父親，就不能理解了，不知道爹、父親也是爸爸的意思。

告訴大家人人都有意識，人人都有感覺，基本上人人都能理解。而告訴大家人人都有靈魂時，就有一部分人理解不了了。再告訴大家人人都是一模一樣的礻申時，基本上就很少有人能理解了。而實際上，這三個說法就如同爸爸、爹、父親一樣，是一個東西三個叫法。

中國人創造了一個字，叫，意。意這個字就是專門指我們自己的真身的。如果用另外的字詞來解釋意這個字是什麼意思，就是：意的另外的名字還有很多，意也叫靈魂，也叫魄，也叫礻申，也叫元神，也叫元嬰，也叫如來，也叫阿賴耶。人們雖然使用不同的名稱來稱呼它，但是稱呼的都是同一個東西。

我們有時候說我們有意識，也有時候把意識換成心識，或換成神識，都一樣。心這個字除在表達那個物質的肉形心臟器官以外，在其他的場合都是指意，都是指礻申，所以才有心意、心神的說法。

意識，有兩重含義：一、識作動詞用，意識指意在識，意在認識，意在感知感覺。比如，我意識到胳膊有點疼，我意識到你想幹壞事。二、識作名詞用，意識指意的識，意的感知感覺。比如，我的意識開始模糊了，我的那個意識是錯的。

別管識作名詞還是動詞用，前面的那個意字的含義都是不變的。意，就是我們的真心本性，就是我們的真身。

我們的真身一一意，是躲在肉體背後的，是無形無色的。所以我們往往意識不到自己的真身是意，是礻申，是靈魂。我們開始時只能意識到我們有個物質身體，於是就誤以為這個物質肉身就是自己。其實這個物質肉身只是我們操控的一個肉做的機器而已。我們平時所認為的物質肉身是我們自己的身體，這個是個假相。物質肉身，是我們的假身，不是我們的真身。我們另有真身。我們的真身，就是真正的我們自己，不是物質的，而是非物質的。我們的真身是非物質的。

我們本來就是投胎人世來做找自己的真身的遊戲的。當然開始只能認物質的肉體是自己，而認識不到自己的真面目是無形無色的意。

如果我們投胎人世後，一下子就可以看出來自己的真面目是無形無色的礻申，那這個做人的遊戲就沒法玩了。我曾打過一個比方，大家都花錢買門票去一個迷宮玩，如果一個人一眼就看出來迷宮的路徑了，一開始就知道怎麼找出口，這個人是沒有興趣花錢買門票進去玩的。

玩迷宮找路，與我們做人找人生的路，差不多，要有複雜的未知、複雜的變化，要富有挑戰性，才有趣，才好玩。不然，我們一出生就可以看出來我們的物質肉身不是自己的真身，就可以看出來我們的真身都是無形無色的一模一樣的礻申，就可以直接打開DNA里的天道法則，比著做出來。這樣，只用幾天的時間，我們做人的過程就結束了。誰有興趣玩這種做人的遊戲呢？

一個沒有感覺沒有意識的有形有色的物質肉體，加上一個有感覺有意識的無形無色的我們(靈魂、礻申、意)，共同組成我們這個人形動物。所以說，我們活著時，是由精神和肉體兩部分巧妙地結合在一起共同組成的活體。我們所謂的死了，不是真死，是假死，所謂的死亡是我們不再操控這個物質肉體了，與這個肉體不再結合了。我們不與一個物質肉身結合了，我們還是存在的，我們還是活的，我們的感覺意識仍然在。只不過我們不與物質身體結合後，我們再去認知這個世界就不需要像有物質身體時那樣通過物質身體來感知這個世界了，沒有物質身體的限制後，我們可以直接去感知認識這個世界，所以，人人都是，只要一死，一離開肉體，什麼都知道，一切盡知。
