金牛座分別都有哪些炸裂的人設？

2019-12-16 829條評論 2,081次閱讀 972人點贊

理財小能手

金牛座對金錢的在意程度是一般人完全沒法想像的。他們是當之無愧的理財小能手。他們確實很會處理金錢，很少有缺錢的時候。只要手裡有了一點積蓄，就會想著讓錢生錢，不是買理財產品，就是去炒黃金，總之不會讓自己閑下來，他們也確實非常擅長做這些事情。

省錢小達人

金牛座不止會安排自己的金錢，還會絞盡腦汁地省錢。一分錢不僅能掰成兩半花，簡直能掰成二百瓣花。別人覺得這是摳門的表現，金牛座卻覺得這是自己會省錢，會理財，還會覺得很驕傲。畢竟人就是不應該花冤枉錢。能省錢就是一種很值得驕傲和炫耀的品質。

美食鑒賞家

金牛座還有一個炸裂的人設就是美食鑒賞家。他們對美食特別愛好。其實他們對其它方面可以說都是清心寡欲，沒什麼追求的，唯獨對於吃的，絕對不能湊合。他們會用很多時間和精力去找一家好吃的店，也會因為吃一頓好吃的又從垂頭喪氣到鬥志滿滿，這就是金牛座。

摳門掌門人

金牛座還有一個非常響亮的名號，那就是摳門掌門人。他們簡直是太摳門了。每一筆錢都要算得清清楚楚也就算了，就算戀愛的時候，兩個人一起出去吃頓飯，都要各付各的錢。一般人真的很難受得了這個，但在金牛座眼中，這真的是很正常的事情。
