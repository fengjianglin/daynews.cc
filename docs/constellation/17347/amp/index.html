答案分析：

A、前世你和同桌的血統一樣高貴。

前世你是精靈血統，你是一個無憂無慮的精靈，每天飛舞在花叢與山水之間。你掌控著萬物生長，花開花落都聽從你的命令，你的心靈純潔的就像山間的清泉，透徹而明亮。你同桌前世是一隻白狐，狐狸族天生具有好看的臉蛋，是個非常吸引男生喜歡的狐狸。但你是個具有善良之人的狐狸，你不會傷害任何人的狐狸。因此，前世你和同桌的血統一樣高貴。

B、前世你和同桌的血統都很高貴。

前世你是人族，是皇室血統，藍色孤傲而冷漠，你是一個皇室貴族，擁有數不盡的珍寶與讓世人尊敬的權利。因為有許多的商人會獻出珍貴的寶物給你，而你就會將這些寶物捐獻給貧困的人民，所有大家都非常的敬愛你。而你的同桌前世是一隻鳳凰，他具有高貴的血統，因為前世這種鳳凰的類型很少見，因此前世你和同桌的血統一樣高貴。

C、前世你的血統比同桌高貴。

你前世是妖族血統，你同桌前世是魔族血統。雖然兩個種族的血統是平等的，但是你卻是妖族的族王，是妖族的最高首領；而你同桌是魔族的一般成員，你的性格堅強霸氣，深謀遠慮；而你的同桌卻性格懦弱，是魔族最低等的地位。因此，前世你的血統比同桌高貴。

D、前世你同桌的血統比你高貴。

你同桌前世是皇室的公主，是皇帝的女兒，是美麗而高貴的公主。而你前世是一個脾氣古怪最冷酷的吸血鬼。你脾氣很犟，很固執，自己認定的事那是拋頭顱灑熱血都要堅持啊。所以有時候不聽勸阻，會走彎路。對愛情很執著，很直接。你還是很堅強，經得起打擊。外表感覺很冷漠，其實內心是火熱的，很善良，不容易拒絕別人，尤其是自己看重的人，受傷了就會想要逃避，多數是吃軟不吃硬！
