象徵以及符號

天秤座的符號自然是一座天秤，♎ ，這個還是比較形象和貼切的。這個星座稱為天秤，主要關切的就是平衡性的一個問題。在中國的24節氣上來看，北半球每一年的9月22日進入秋分，也就是進入了天秤座管轄範圍，南半球則是春分進入天秤座，春秋分時節的晝夜是完全均等的。
