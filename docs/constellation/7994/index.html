亦正亦邪可帥可二的男明星陳偉霆塔羅命盤解析

2019-12-06 302條評論 2,025次閱讀 147人點贊

前段時間野狼disco在抖音特別火，陳偉霆出了一個搞笑版本的野狼disco MV，特別好玩。

那我們來了解一下陳偉霆的塔羅命盤是怎樣的？

陳偉霆的塔羅命盤

竟然又是有幸運之輪的命盤，真羨慕陳偉霆，長得帥，還有好命盤。

陳偉霆的塔羅命盤裡的死神牌代表陳偉霆性格非常直男，做事很有決斷力，能量超級強。兩個幸運之輪代表陳偉霆的貴人運和事業運特別多。

陳偉霆2003年，因參加全球華人新秀香港區選拔賽而進入演藝圈。2006年成為Sun Boy』z組合一員。2008年開始獨立發展，第一次推出個人首張專輯《Will Power》，就獲得香港十大勁歌金曲頒獎禮最受歡迎男新人金獎及香港十大中文金曲最有前途新人金獎。

2013年，陳偉霆將工作重心轉移至中國內地，也就是在那一年陳偉霆接拍了《古劍奇譚》 讓他在內地打響知名度的第一步，不愧是塔羅命盤帶著兩個幸運之輪的人，努力就會有機遇，有運氣。陳偉霆出演的古裝仙俠偶像劇《古劍奇譚》在播放期間獲得了全國同時段電視劇收視率第一，網路播放量單集平均點擊量突破1.5億的成績 ，而陳偉霆也因大師兄陵越一角被內地觀眾所熟知，並獲得國劇盛典年度最受歡迎全能藝人及年度電視劇媒體最期待演員 ，同時還提名上海電視節白玉蘭獎最佳男配角。

2015年，陳偉霆主演的民國偶像劇《活色生香》開播全國網收視率達3.43，打破近十年電視劇開播的首播紀錄，首日網路播放量10731萬次，兩集破億，刷新內地電視劇網路播出首日的網路播放量記錄，陳偉霆飾演安逸塵。

2016年陳偉霆主演的民國懸疑劇《老九門》在東方衛視周播劇場播出，陳偉霆飾演有膽有識的張啟山，並為該劇配唱主題曲《我門》，該劇在播出後引發了頗高的網路熱度和話題熱議度，長期居穩同時段收視榜首，其微博衍生話題累計閱讀量超150億次，網路播放量突破百億，成為全網史上首部破百億的自製劇，而陳偉霆亦成為各大熱搜榜的話題人物 ；8月，單曲《W企劃》全網上線，該曲發布首日位居各大音樂排行榜榜單前列，上線僅12小時破千萬次點擊率 。陳偉霆主演的愛情喜劇電影《那件瘋狂的小事叫愛情》於上海某影城點映，單場票房近31萬，刷新全國單場最高票房紀錄，其在劇中飾演剪輯師江洋，該影片於8月12日全國公映 [ ；陳偉霆舉行「Inside Me」巡迴演唱會，這是其首次個人巡演。

2017年，陳偉霆首次登上央視春晚舞台，搭檔鹿晗演唱歌曲《愛你一萬年》 ，陳偉霆以張啟山的形象入駐上海杜莎夫人蠟像館，成為年度首位以影視角色入駐上海杜莎夫人蠟像館的藝人。

又有演技又帥氣的陳偉霆，希望他能持續使用幸運之輪的運氣，星路一直平坦。
