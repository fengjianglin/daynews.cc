「“愛因斯坦”的控訴」雙魚射手的死穴點

天天要聞
3 週 ago

去年冬天那個火遍微博和國外視頻網路的四川妹子，你們還記得嗎？

她叫宇芽。

她的逆天仿妝技術讓所有人嘆為觀止。

最初走紅網路的是她把自己畫成了愛因斯坦吐舌頭的模樣，有很多網友評論她放了原圖，但仔細比對，看看吐出來的舌頭就知道，此愛因斯坦非彼愛因斯坦。

後來翻她的視頻，才發現她真的很牛，除了愛因斯坦，她還能畫出很多名人，蒙娜麗莎、山田直美、戴珍珠耳環的少女、強尼德普…不僅還原名畫，還能易容。

今年冬天她又上熱搜了，但這一次不是因為她高超的易容術，而是「家暴」。前天她在微博上曝光自己的前男友沱沱的風魔教，聲稱自己經歷了半年的煎熬與惡魔期。跟所有遭受家暴的女性一樣，她第一次的原諒只換來了一次又一次的家暴升級。而此刻，她選擇勇敢的站出來講出自己的遭遇。

當「家暴」再一次出現在公眾視野並拿出來討論的時候，又是喚醒一批女性的機會。意識的覺醒不是那麼容易的事情，它需要我們勇敢的站出來，一遍一遍的給予那些心軟原諒渣男的女性勇氣，讓她們離開。

前有百老匯一部膾炙人口的歌舞劇《芝加哥》，後有改版的《天朝渣男圖鑑》，一如原版的配置，改編翻唱版也是六個女罪犯的自訴，其中一人是被丈夫/男朋友陷害，無辜入獄。同樣是殺人，但改編引用的卻是一個個真實鮮活的案例。出軌、家暴、不育男淹死養子、的士司機性騷擾，試圖強姦、被男友誣陷販毒、殺女配陰婚。爆火三天，因為爭議太大就遭到了封殺。可是它難道不是在教我們「及時止損」嗎？

成年人不易，成年人學會及時止損更不易。

哪些星座的姑娘容易被渣男精神控制？

上次我們講PUA情感詐騙的時候，就有聊到：射手座、雙魚座、巨蟹座、金牛座、天秤座。

射手座是因為她們悲憫救世主的化身，雙魚因為自己的心軟和幻想以及無人能及的治癒力，巨蟹是因為前無古人後無來者的包容力，金牛則是因為懶得動彈，金牛不輕易掉進去，但是一旦掉進去形成習慣，你讓她爬出來也不是那麼容易的。天秤是因為自己的猶豫不決。

如果在三駕馬車裡（太陽、月亮、上升）剛好霍霍在這幾個星座之間，特別是射手和雙魚的搭配，乖乖的將拯救天下渣男作為己任啊！

清醒點吧，姑娘們！

那什麼星座配置容易遭受家暴呢？

很明顯是星盤裡有日火、日冥兇相位的人，特別是火冥在7宮，本身在原生家庭里可能就存在暴力行為的情況，而盤主卻無法自查，很容易將這些帶入自己的婚姻，再加上脾氣急躁，嘴上得理不饒人，非要對方臣服於自己的話，有極大的可能帶來矛盾升級，可能對方就是會暴力的人，可能對方不是，但被你刺激得動了手。所以當有這種配置在星盤的時候，請意識到自己言行上的過分以及遠離有暴力傾向的人。
