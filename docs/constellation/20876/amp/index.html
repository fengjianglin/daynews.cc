首先，星座跟生肖只不過是跟出生日期有關。當年年少無知，曾經相信過一些星座性格啊，測試性格啊什麼的。在關注了這些很久之後我腦袋自動彈出一個很大的疑問：既然那麼多人同年同月同日生，那他們性格豈不是一樣？性格測試得出同樣結果的人，莫非他們的性格也是一樣？可問題是：世界上有性格完全一樣的人嗎？那就肯定有人問：為什麼那些星座性格跟我的性格真的很像？其實第一次知道自己是××星座，有這樣那樣的性格，自己心裡肯定會有種潛意識——原來我是這樣的人！

如果你真的相信你是那樣的人，你就會不停地給自己灌輸「我是這種人」的信號，然後不經意地變成那樣的人，從而讓你確信星座跟性格真的是准得太邪乎了！你要真問那他那些所謂的性格描述是怎麼來的，我只能說：是那些傢伙蒙出來的——只是用了一大段籠統，而且很多人都可能有的性格去逐個逐個套而已。那性質跟給你看面相，手相的神棍差不多。要知道，性格是父母，社會，環境，文化，體制等等各種複雜因素形成的，哪是你百度上搜的星座性格這麼幾行字就能概括出來的？

其次，我是巨蟹我真的是討厭別人的接觸，有人碰下我胳膊我汗毛都能站起來，有時候男票拉我的手或是突然碰我我都會反感。不想與人住一起，家裡人都不想，我很愛我媽，我媽是世界上我最不能失去的人，但是我也不想那麼近，有的時候她抱我親我我都會有點抵觸，但得裝作很開心。喜歡人也是一條吧，喜歡你的時候你作天作地都可以，我都能給自己找理由順著你慣著你，但是一個月我就厭了，我習慣扣分制，你要扣的差不多，那你這個人就快over了，我就會盤算什麼時候和你分手。你來求我我會覺得好笑，有人說巨蟹心軟，我覺得不對。

我覺得其實自己很心硬，表面上答應你複合，但其實你早就被判了死刑，複合只是不想你繼續墨跡而已，等哪一天該提照樣提。沒有什麼會一直想著的人，或者很少吧，兩三個吧，一般主動聯繫你就是沒意思沒啥玩的，你感覺跟我玩挺好的挺熱乎，但其實也就是個解悶的人。真正能留心裡的得經歷很多事才會肯定，我考驗朋友考驗了九年+。我不會不合群，超愛玩超能玩超喜歡刺激，但是很難玩熟，也能玩完就翻臉不認人。

我朋友經常會說，有的時候看你或者喊你，你真的就一臉冷漠，轉頭過來的時候那個表情看我們就像看陌生人，就好像這個世界與你無關，你就在一邊兒冷眼旁觀似的。做什麼事情都有目的，完全結果導向，每個選擇都是有原因的。喜歡謀劃自己的未來，按著目標一步步走，我的未來說實話沒有考慮任何人，只是想自己怎麼走會更好，會偶爾思考父母，但不會因為他們改變自己的計劃。

我時間最長的朋友十二三年了，他是唯一一個敢戳我刺的人，可能感情好是因為像吧，他也是一個很冷情的人，朋友少，他說我們兩個是我見過最精緻的利己主義。我的家庭沒有變故，是個大家庭，是很有愛三觀比較正常的那種，家裡最小，從小挺受寵的。不敢想像如果是個缺愛的家庭自己會怎樣。天生性情冷漠，非常敏感，看你一眼你這個人幾乎就能知道個八九不離十。這個真的不誇張奧，你家裡條件怎麼樣你什麼性格值不值得接觸是真性情還是各種婊，給我一張照片我都能大概說說這人性格行事作風什麼的。

表面很愛笑好相處，挺會來事兒的，想哄一個人開心很輕鬆，心裡怎麼想就不一定了。愛三分鐘熱度，但是想要的事情一定能做成，可有可無的事情就隨心情。巨蟹玩得好的，在她心裡那不叫朋友，那叫玩伴。她也會喜歡別人，也待人真誠，也治癒別人，也會陪伴別人，但是她誰也不相信，誰也不依靠。真正的冷漠從來不是愛搭不理，而是她對你露出最溫暖的微笑，卻對你毫無期望，她憐愛你如同憐愛一隻受傷的小貓，喜愛你如同喜愛一片璀璨的星空。她一面深情地注視著你，對你說我永遠愛你，一面透過你的眼睛冷漠望著你靈魂的空虛。她會無微不至的對你好，時刻寵愛你，但你要她的心，那不可能。
