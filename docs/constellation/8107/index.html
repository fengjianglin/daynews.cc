很有氣場自身也很有素質涵養的4大星座，有你的星座嗎？

2019-12-02 96條評論 3,223次閱讀 1218人點贊

巨蟹座一般對自己的身形和身材會有一個很嚴謹的管理方式。一般來說，長得高的人，或者是身材好的人，對自身就會有一種相對自信的感覺，也很容易讓人產生出一種優越感。巨蟹座在別人的面前會比較有架子，有氣質。當然，除了外在的氣場之外，巨蟹座同樣讓人敬仰的還有內在的涵養，他們處事不驚、做事有條不紊，這方面的抗擊打能力也讓人欣羨不已。

射手與生俱來的自信感，是很多人後天學習都無法媲美和相比較的，射手的氣場是從骨子裡透露出來的。射手的氣場就算是穿著樸素簡單也還是無法掩蓋。沒辦法，有些人天生就像是星星一樣閃耀。射手屬於很感性的人，喜歡閱讀很多思想境界很高的書籍，比如哲學，所以射手其實也是很有思想的一類人，自身也很有素質涵養，是令人羨慕不來的，大概也只有敬仰的份了。

摩羯的氣場其實主要就是來源於他們的臉色啦，在陌生人面前，臉上幾乎沒有表情，於是就會給人一種不好接近的感覺，大家就會覺得摩羯的氣場特彆強大。其實摩羯對熟人很好的，而且還是一個很可愛的沙雕選手，這一反差萌常常是迷倒了眾人，致使很多人都會喜歡上這樣萌萌的摩羯。摩羯也是一個對自身要求很高的星座，大部分魔羯都是知識沉澱很豐富的人。

獅子座的氣質和氣場都是由內而外的，他們身上的氣場很強烈。就算沒有讓人眼前一亮的顏值，但是卻有讓人望而卻步甚至是不自主會對對方刮目相看的氣質。他們大多身懷一技之長，並且能夠在自己擅長的領域有一番大的成就。獅子座之所以能夠讓人敬仰，當然還有他們自身良好品質的原因。獅子座不僅優秀，還是一個不斷追求完美的完美主義者。
