天天要聞
2019-12-21

很多人單身久了就會渴望愛情，尤其是看到他人的愛情時就會更加渴望自己也能擁有美好的愛情。但是愛情總有不順利，再美好的愛情也可能會有走散的那一天。每到這時，又會覺得單身的生活更為舒適。不曾擁有，也不必承擔失去時的苦痛。時而想要戀愛，想找個人陪伴；時而又享受單身的自由，覺得單身生活為自己免去了那麼多麻煩。總之就是這樣不斷搖擺，剛跨出去的步子又迅速地收回。

天秤座

紅鸞星耀，良緣駕到！2019年順利脫單的天秤座，愛情有望修成正果。眾所周知，天秤座的情商是超級高的，而且他們特別會交朋友，可以說他們走到哪裡，都會留下人脈資源。而對於天秤座這樣天生條件比較好，有顏值有魅力的人，他們的虛榮心也相對會比較高。所以，他們想要的另一半，至少也是大方得體的，與自己站在一起應當是是非常般配的。一般來說，物以類聚，人以群分，像天秤座這麼優秀的人，他們身邊的朋友都不會差。而在2019年，天秤座的桃花運勢良好，他們將遇到同樣優秀的另一半，兩個人非常般配恩愛，良緣天成，羨煞旁人。

雙子座

雙子座的看似對愛情不上心，可是實際上，面對愛情的期待，雙子座不比別人少。而雙子座也相信是自己的愛情，始終都會到來，也依然希望自己的生活中能夠出現合適的另一半。在2019年，雙子座的人也終於在等待中迎接到了屬於自己的愛情良緣，月老牽線讓雙子座成功遇到愛情，真愛的如影隨形也讓自己的生活越來越甜蜜，越來越幸福。

獅子座

獅子座是性格很豐滿的人，獅子座的人不能單純的說是慢熱還是開朗，獅子其實是百變的性格，當獅子遇到不同的人，獅子展現出來的性格也是不一樣的。當獅子面對的是同事的時候，獅子儘可能展示的是自己的實力，獅子會想要自己在工作上的專業程度讓對方看到。當獅子面對的是朋友的時候，獅子就是隨心所欲的，獅子就是想說什麼就說什麼的，獅子就是開朗又不用遮掩的狀態。當獅子面對的是戀人的時候，當戀人變得強勢的時候，獅子就扮溫柔，當戀人不講理的時候，獅子也會一起跟著耍無賴。所以說獅子的性格是豐滿的，不是單個的幾個辭彙就能描述的。 12月初的時候，這樣的有趣的獅子，怎麼能不被桃花羨慕呢。當獅子良緣自來的時候，獅子沒有絲毫的遲疑，獅子有望被真愛青睞了。獅子希望戀人是和自己一樣多變且有趣的人。

白羊座

白羊座在感情中，會比較容易受傷。因為白羊座一旦愛上了一個人，就會真心實意的對待對方。可如果對方卻並不領情，而是會做出一些事情傷害白羊座，那麼白羊座也會放棄這一段感情，再不回頭。未來11天，白羊座與前任訣別，不再聯繫，忘記過去，擁抱命定之人。雖然曾經也有過幸福的，快樂的時光，但是這些都抵不過白羊座內心的傷痛。與前任訣別之後，白羊座依然會處在情緒低落的狀態。可是即使如此，白羊座依然會勇敢的擁抱生活，在挫折來臨的時候，白羊座也會勇敢的面對。相信在未來，命中注定的人會與白羊座浪漫邂逅，牽手永不分離。
