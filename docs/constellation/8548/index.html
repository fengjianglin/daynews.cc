11月19日

火星進入天蠍座

星象重要程度：★★★★★

（至2020年）

火星進入力量最強的位置，為很多人注入了強大的行動力。這樣的行動力不是魯莽，不是衝動，而是具有決策能力的勇敢果決，將非常有利於很多計劃的貫徹落實。

高效地開展工作學習，積極地備戰團隊比拼或個人競聘，著手解決此前一直困擾你的大麻煩等等，成為這段時間的主題。

不過需要提醒大家的是，天蠍座的火星也給人帶來強烈的佔有慾和嫉妒心，因此不管是在職場還是在感情中，都要注意，不要忽略對方，特別是弱者的感覺。

// 宜：競聘、拼業績、絕地反擊、衝出困局

幸運小星星：天蠍座、白羊座、巨蟹座、雙魚座

11月21日

水星在天蠍座恢復順行

星象重要程度：★★★★

天大的好消息，2019年的最後一次水逆結束了。水逆帶來的種種混亂，都將慢慢恢復到正常的秩序中，不管是大家低迷的工作和學習效率，還是糟糕的交通狀況，溝通交流過程中含糊不清的表達等等，都會有改善。

如果這一次水逆期間的你特別難，那建議你去回顧一下自己的經歷，視角的轉變或許會讓你有新的正確的判斷。

幸運小星星：天蠍座、雙子座、處女座及所有受水逆影響的大家

11月22日

太陽進入射手座

星象重要程度：★★★★

（至12月22日）

告別天蠍月，進入自由隨性的射手月，讓大家緊繃的神經得到放鬆，雖然可能日常依然很忙，但心態卻變得積極了。

你會發現，雖然萬事開頭中間結尾都很難，但樂觀積極的行事風格的確能幫到你，越來越多的人在這個月里揮灑自己的創造力，去獲得滿足感與認同感。

不僅是對於現實的考慮，這也會是大家計劃和預見未來，重拾夢想的一個月，一些人會開始涉足那些從前未曾進入過的領域，或計劃長途旅行，這都有助於眼界和心胸的開闊。

// 宜：旅行、留學、考試、法律事務、國際貿易

幸運小星星：射手座、白羊座、獅子座

11月24日

金星合木星

星象重要程度：★★★

（至12月2日）

金星與木星在射手座的合相，為很多單身狗帶來脫單的可能性，同時也開啟了超虐狗一段時間。更多社交機會的出現，主要是你好像也不太排斥，願意去參與，這讓你有機會遇到讓你心動的人。

而已經有另一半的人就不必多說了，你們將更親密更甜蜜，目測狗狗的早餐午餐晚餐宵夜都有著落了。

// 宜：開party，約會，談合作

幸運小星星：射手座、獅子座、白羊座
