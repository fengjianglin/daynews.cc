圖中哪棵大樹最有靈性？測你命中注定是什麼命樹

2019-12-18 3,633條評論 4,728次閱讀 544人點贊

小時候老師就教過，大樹呼出氧氣供人類呼吸，又把人類的廢氣回收回去凈化，不斷的循環空氣。在人們眼裡，大樹可是很有靈性的。今天小編要和大家所做的測試也是和大樹有關的，圖中哪棵大樹最有靈性？測你命中注定是什麼命樹！

A：看起來是不是也很有靈性呢？不過選擇了第一棵大樹的你，命格當中似乎沒有什麼特殊的。像你的這種命格，隨便丟到人群中根本就找不出來，因為和你相似的命格實在太多了。不過你也並不是相信命格的人，向來都認為只有通過自己奮鬥才會有好日子過，所謂的好運相伴，只是你沒看到人家背後的付出罷了。世界上的人有那麼多，不可能每個人對幸福的定義都完全一樣，可能有很多人會選擇過富貴的生活，可是你卻並不那麼覺得。和金錢相比，你更加看重自己是否過得快樂，凡事你覺得會讓你不快的事情，你都不會去做。哪怕那件事能給你帶來再大的利益也一樣，你最想要的就是遵從自己的想法而活。

B：選擇了第二棵大樹的你，身上有一股自帶的富貴氣息。可以說你很會投胎，在你還沒出生以前，你的上一輩就已經為你打好了物質基礎。所以在你出生以後直接享受他們的勞動成果，天生不用為了物質而發愁。不過你也並沒有因此而荒廢自己，反而把上一輩作為榜樣，既然你的父母可以為你提供良好的資源，你就會回報他們以更好的生活。而且你也會為了你的下一代而奮鬥，讓你的子孫也能夠為你這個長輩自豪。

C：第三棵大樹看起來其實也非常不錯，你的命運卻只是一般。你的命格天生自帶勞碌，原本你的人生過得就不是很順利，再加上你除了操心自己的事，還要操心別人的生活，可以說心中很少見到陽光。你的性格有些隱忍，在家庭生活中，你寧願自己受委屈也不願八家裡鬧得雞飛狗跳。喜歡完美的你，就像是一位潔癖病人，只要是經過你手的事情就不允許有一點瑕疵存在，如果讓你發現了一份稿件當中有一個錯別字，你可能會直接把那份文檔廢掉重新寫一遍。你知道上天是不可能公平的對待每一個人的，所以你也不會去抱怨所謂的命運不公，你比很多人都更加樂觀，就算深陷困境，你也會自己想辦法走出來。

D：選擇了第四棵大樹的你是比較幸運的，說來也是奇怪，很多時候別人所無法避開的危險都讓你給避開了，完全沒有受到任何傷害。可以說你的日子是過的最順利的，雖然沒有很大的成就，但也並沒有經受大的挫折。更奇怪的是，雖然霉運會自動遠離你，可是有什麼好事也並沒有落到你身上。不過你可不要認為人生當中只要運氣好就足夠了，要知道有時候單靠運氣是不夠的，只有努力提高自己的實力，才有資本在任何事情面前都無所畏懼。你所要做的事情是趁著好運還在，學習自己以後需要用到的知識，想辦法把別人錢包裡面的錢放到自己口袋裡面，足夠的實力加足夠的運氣，你才會一直好運下去。
