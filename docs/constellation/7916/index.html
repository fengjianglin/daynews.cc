（月運）波頓2019年12月星座財務運

2019-12-08 502條評論 3,814次閱讀 465人點贊

娜塔莎·波頓，美國著名占星師、占星專欄作家。主要風格為事業與財務占星。

轉載請註明星譯社及譯者爬爬。

★★★白羊座★★★

你代表事業的第十宮本月的行星活動激增，白羊座，這是因為木星會在你的事業宮進行大約一年時間的旅行。木星加入土星和冥王星的行列，進駐摩羯座，這肯定預示著，在未來十二個月甚至更久的時間裡，你在事業和金錢方面會利好！在12月10日到12月30日之間，水星位於同是火象星座的射手座，這會鼓勵你以豪爽的方式交談，而土星和冥王星對你腦中浮現的任何提議，都強調細節和責任。

★★★金牛座★★★

12月開啟了旅行的一年，金牛座！木星進駐你代表遠距離旅行的第九宮，會給你帶來諸多遠距離旅行的機會。這是調研全球性公司和組織的絕佳時機。在木星開拓眼界的時候，你會沉迷於對哲學問題的探討和思辨，而且你精神、心靈層面的視角會受到著重強調。另外，帶來突然改變的天王星，會繼續在你的第一宮旅行。在天王星沉重古怪的步調中，你會改寫自己的個體呈現，改寫的方式如此猛烈，以至於你都無法認出自己。這一切對你而言都是積極的，因為你長久以來所懷抱的，在事業方面改變的渴望，會在今年一飛衝天。享受這段旅程。確保自己的護照在有效期內，以迎接海外的差旅。

★★★雙子座★★★

共同的儲蓄和投資會激增，這是因為木星進入了你代表夥伴關係資源的第八宮。火星位於你代表健康的第六宮，這使你需要在本月減少聚會往來。從一個地方去另一個地方的時候，讓自己放輕鬆慢慢來，早睡的次數多一些，一切就會如常。位於射手座的水星與你的星座構成對沖相位，這可能會在12月10-30日之間，給你的溝通交流帶來一些壓力，並增加一些熱度。最大化利用資源的辦法是，專註於個人和工作夥伴能跟你一起做的事情。

★★★巨蟹座★★★

金星、土星和冥王星本月對沖巨蟹座，這需要你在工作中，以禮貌且高效的方式表達自己的想法。水星在你代表健康的第六宮旅行，這會給你帶來諸多信息，從而讓你以最佳狀態邁入全新的一年！被稱作「火環」的日環食會在12月26日出現。屆時你的選擇會變得格外清晰，這是因為你的強大守護星月亮，會在那天與太陽構成食相！這是把你對未來一年渴望發生的改變，進行可視化的絕佳時機，所以花點時間讓自己專註於此。

★★★獅子座★★★

去奪金吧，獅子座，12月15-17日位於獅子座的月亮，會與位於友好射手座的水星同步。出席會議，或者站在聚光燈下。本月來自土星、進行和冥王星的刑克，會激發你在前進之路上的競爭之心。擁擠在第六宮的行星，帶來了能量的阻滯，這就需要你在本月關照自己的健康和精力。在12月的前三周，慷慨的射手座太陽，會給你投射三分相的光芒，所以行動起來吧，然後在最後一周讓自己放慢腳步。享受假日！

★★★處女座★★★

金星、火星、木星、土星、和冥王星與處女座構成的良好相位，會讓你在本月多數時間平穩航行。等12月22日太陽進入摩羯座的時候，畫面就完整了，所以用時髦的方式結束本月吧！行星活動集中在你代表歡笑和玩樂的第五宮，從個人層面和工作層面而言，這都會為你帶來一個歡樂的假期。本月的最後十天，金星進入你的第六宮，可能會在感情方面給你帶來一點小波折。但總體而言，這是充滿歡聲笑語的一個月，處女座！

★★★天秤座★★★

很多了解天秤座的人，都認為天秤座是藝術與優雅的，但讓人跌破眼鏡的是，其實天秤座跟土星非常和諧。土星在天秤座屬於能量曜升，這對你而言是個好消息，因為本月你的守護星金星加入了冥王星、木星和土星的行列，進駐摩羯座。第一周在工作方面最為高效，尤其是12月11日，因為冥王星、土星和金星會在那天精確合相！這是重大目標成型、構建並得到權力層支持的時候。12月19日的天秤座月亮會完善大畫面，所以回顧一下自己規划過的步驟，並將它們付諸實踐吧。

★★★天蠍座★★★

天蠍座，出去走走，因為你代表本地社群的宮位充滿了能量。你的守護星火星整月都能量十足，這是因為它正在穿越天蠍座。你的第二守護星冥王星，可能會在12月12-14日與土星發生摩擦。金星與天蠍座處於激情的六分相，其中含有隱秘的弦外之音。這個時候你最好留意自己的言辭，尤其是在工作環境中。可能會出現話中有話的局面！總體而言，你會非常享受本月，尤其是等太陽進入對你友好的摩羯座以後。既然你喜歡緊張刺激，你正好能在家享受歡樂和嬉鬧，（越強烈越好），所以除了之前提到會在月中旬發生的合相之外，何不全力以赴呢？

★★★射手座★★★

你的守護星木星，在12月3日離開射手座，去了臨近的的摩羯座。雖說不會像去年木星在第一宮一樣，給你帶來擴張感和積極感，但你要相信，發生的一切都在支撐有益的收穫，這些收穫都源於特定的區域——也就是你如何看待自己，以及如何向世界展現自己的這一區域。你代表有形資產和無形價值的第二宮，會被蒞臨土元素星座的木星大幅強化。你會看到自己的收穫變為現實，而這些收穫源於你過去一年所追逐的目標。土星位於其守護的摩羯座，會讓你得到自己應得的東西，不多不少。所以，在經歷了過去一年的慷慨付出，這是收穫的時候了。太陽會在射手座停留到12月22日，這會令你歡欣愉悅，所以辦個聚會吧，可以比你上個月底舉辦的那個更安靜更私人一些。

★★★摩羯座★★★

領導者的角色令你感到舒適，這很好，摩羯座，因為在你全然支配的區域內，招待著一眾行星，包括金星、木星、冥王星，之後太陽也會在12月22日到來。12月26日落在摩羯座的日食會結束某件事。如果可以的話，讓自己休息一天，從而讓自己能以專註且強大的方式，去謀劃自己在未來一年想呈現的東西。此外，月中旬的時候，小行星刻瑞斯會與金星合相，這會促使你關注假日裝飾，以便能夠呈現出絕美的效果。你可能會因自己的藝術才能而成為街坊鄰里的驕傲！

★★★水瓶座★★★

你不落俗套，水瓶座，這也是為什麼大家都愛你！因為你的異乎尋常、令人驚異的世界觀，以及你與世界交流的方式，讓你成為了各個宴會的寵兒。本月你可能會發現自己矛盾複雜，可能上一秒還大膽歡脫，下一秒就羞怯退守了。12月1日-22日，太陽和水星在對水瓶座友好的的射手座，這是舉辦或出席假日宴會的最佳時機。等太陽加入木星、冥王星和土星的行列，進入你代表隱居的第十二宮時，你會選擇遠離公眾視線而且比較偏好寧靜。不過，12月21日金星會進入水瓶座，這會提升你的自尊，而且能讓你的溝通交流格外美妙，不論你處於何種環境之中。如果你能順應能量生活的話，這對你來說會是著實不錯且多樣化的一個月。

★★★雙魚座★★★

本月機會環繞著你，雙魚座。土星、木星、火星、金星、冥王星都與你的星座相位和諧，在12月22日之後，連太陽都會照耀著你。這會給你很多機會去兌現自己應得的獎。要記住的一點是，由於這些行星大都與你的星座呈六分相，所以需要你的行動來促使它們實現。對於自己在金錢和事業方面的目標，別因假日而完全分心，一切就會有不錯的進展。12月4日在雙魚座的上弦月，會讓你開始思考自己想在新的一年裡想達成何事。
