雙魚座：感情規劃

多情的雙魚座，他們的心裡總是編織著自己美好的童話故事夢，期待著一場浪漫的邂逅。這個星座的人，生性浪漫又感性，很容易會因為外界的風吹草動而牽動自己的心緒。他們對事業沒有遠大的追求，對錢財和物質也並不感冒，因為在雙魚的心裏面，精神上的感受最為重要。要是能夠擁有一份浪漫的愛情，雙魚座大概會願意為它奮不顧身吧！

而來到年末之際，對事業對賺錢不大感冒的雙魚座，自然不會把自己的心思放在這些事情上。不管單身還是已有伴的雙魚座，對他們而言，年底最應該整理和規劃的就是自己的愛情。雙魚通常都會努力把自己的戀情往積極的方面去想，並且會自動過濾很多不開心的因素，然後年底規劃的感情中，他們只會按照一般的程序，去幻想和計劃著兩個人下一步下下一步下下下一步……應該是什麼樣的。

水瓶座：尋找新靈感

水瓶座為人雖然與一般人相比顯得有點奇怪，但是他們確實12星座當中情商和智商最高的一位。這個星座的人，擁有著許多的奇思妙想，在他們的腦袋裡似乎裝著一個與眾不同的世界。而在普通大眾來說，他們前衛的新點子新想法，總是會顯得有點標新立異，所以不一定都會被人接受。但是水瓶座，卻從來都不在乎他人的看法和評論，只是專心做自己。

這個星座的人，在年末到來之際，他們更多的時間都花在了忙著尋找心靈感上了。這一年的餘額不足，意味著新的一年馬上就要到來，而過去的一些老舊的想法和點子眼看著就要過時了。為了新的一年有更多好玩刺激的事情可以做，又或是為了來年工作有更好的開展，水瓶座會在年底的時候就忙著去尋找新構思，開發一些全新的靈感和點子，力求要搶在別人發現之前。

摩羯座：總結與反思

成熟穩重的摩羯座，做事的風格向來都是腳踏實地，一步一腳印去走。這個星座的人，從來都不會幻想一些不切實際的事情，而相比起別人，他們更願意相信和依賴自己，乃至於摩羯座平日做事總是那麼的我行我素。不過基於能力和經驗的不斷提升，摩羯座依靠自己的力量在職場或是生活中獨當一面是沒有問題的。

而之所以能夠走到今天這樣的成績和成就，那便是基於摩羯座善於去總結和反思自我的這個習慣。尤其是在一年即將結束之際，摩羯座更是會騰出私人的時間好好跟自己對話。尤其是過去一年犯下的錯誤或是失誤的地方，以及本應做好但卻沒有做好的地方，摩羯座都會進行一個自我的反思過來，不斷改進和完善自我。這樣的反思過後，也更利於新一年的成長。
