人馬座是神射手，他是半人馬族的智者

天天要聞
2019-11-23

大家好，我是星座萬物，今天來聊一下黃道12星座中的人馬座。說道人馬座，有許多的朋友會稱之為「射手座」，其實「射手座」這個稱呼是從日本流傳出來的。受到日本的一些影響，在中國也存在「射手座」這個非正式名稱，根據希臘神話，這位「射手」是半人馬族的智者「喀戎」，因此中國稱為人馬座。

星座位置

在夏夜時分，從天鷹座的牛郎星沿著銀河向南就可以找到人馬座了。因為銀河中心就在人馬座方向，所以這部分的銀河是最寬最亮的。

人馬座正對著銀河中心的方向，所以人馬座裡面的星團和星雲特別多。在南斗σ和λ兩星連線向西延長一倍的地方，可以看到一小團雲霧樣的東西，在望遠鏡里看上去，它是由三塊紅色的光斑組成的，特別好看，它就是「三葉星雲」。人馬座里的星雲還有不少，比如在南鬥鬥柄μ星的北面，有個星雲很像馬蹄子的形狀，被稱之為「馬蹄星雲」。

研究歷史

古希臘人把人馬座想像為張弓搭箭的馬人喀戎，箭頭正指向西面的天蠍座那隻大蠍子。

我國古代把人馬座這一區域的星分成兩部分，南斗六星是斗宿，喀戎弓箭下邊的星屬箕宿。斗是量米用的，箕是揚糧食用的，它們都是農具，常常並稱。

天體

從地球看來，本銀河系的中心位於人馬座，雖然銀河中心被人馬座的手臂上的星雲和塵埃帶所遮擋，但是中間還有很多明亮的星團和星雲的。人馬座有多達15個梅西耶天體——這是所有星座中最多的。其中包含了：礁湖星雲（M8），奧米加星雲（M17），三裂星雲（M20），小人馬恆星雲（M24），M70 球狀星團等等。

黑洞

位於銀河系中心的「人馬座A」是一個變幻莫測的無線電波源，同時它也是一個強大的紅外線和X射線輻射源。天文學家們猜測這些射線的產生可能與黑洞活動有極大的關係。據德國科學家在2008年最終證實，在與地球相距2.6萬光年的「人馬座A」確實是一個質量超大的黑洞（質量約為太陽的400萬倍）。
