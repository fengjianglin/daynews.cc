和這3位紫微主星吵架，你永遠只有挨罵的份

2019-12-04 3,423條評論 4,388次閱讀 3058人點贊

在和別人爭吵之後，自己常常會想：當時怎麼不說這句話呢！說了這句話就能把對方噎的死死的吧！我這腦子，真的是沒得救了！這種事後才想起來如何懟人的操作，真的是比差一分就及格還要懊惱，最要命的是這種事情發生的幾率還很高！那種吵完架之後，自己默默練習如何在下一次獲得勝利的樣子，真是有點可愛又好笑。但是接下來的這3位紫微主星，從不用擔心這些問題，「你狠任你狠，見招我拆招！」

紫微星（獅子座）

紫微天生就喜歡統領全局的感覺，所以在吵架上，也絕對會不甘示弱！和紫微吵架，他們永遠會拿出一些很小很小的例子來反駁你，並且思路清晰，條理清楚。而且紫微還是屬於那種，只要吵起來了，就一定不會想著自己會輸，這會激起他們的慾望，讓他們在吵架上越挫越勇！同時，和紫微吵架，有些急躁的性格會讓他們無意中把自己的聲調也提高上去，在氣勢上就已經壓制了對方！

破軍星（射手座）

破軍一般都不太願意吵架，能夠息事寧人就不會多浪費口水來和別人爭吵。但是這並不代表他們不會吵架，要是觸及到了破軍的原則問題，那就真無力還嘴了。他們的吵架值能夠像跑道上的賽車一樣，一路飆升，懟到你啞口無言。同時在吵架上的邏輯問題，破軍也能及時地抓住你的漏洞，並且放大，給你整的明明白白，那感覺就像是在進行一場辯論賽，而且對方和自己的差距就像是小蝦米對大鯊魚。

巨門星（水瓶座）

對於巨門，會吵架可能算是一個隱藏技能！天生慢熱的巨門很少會與人起衝突，即使與人有爭執了，也能肯定大部分不是巨門帶來的。但是不要以為能在吵架上欺負巨門，一旦他們開啟連環炮模式，你是真的沒辦法插上一句。甚至他們還會翻起舊賬來增加自己的殺傷力！真的是很讓人有無語的感覺。當然，也千萬別在吵架時，試圖和巨門講道理。因為他們根本就聽不進去！
