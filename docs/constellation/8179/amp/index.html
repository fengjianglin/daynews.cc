假如《山海經》是真的，那麼來看看你姓氏血脈中繼承了怎樣的神力

天天要聞
3 週 ago

我國每一個姓氏的起源背後都有著不一樣的故事，而其中有一些姓氏的祖先更是可以追溯到上古時代，而《山海經》中就記載了很多我國上古時期的歷史，但是因為《山海經》裡面的故事太過離奇，人們對於它的真實性一直都有所懷疑。

但是假如《山海經》是真的，那麼就和小編一起來看看你姓氏血脈中繼承了怎樣的神力吧。

首先如果你的姓氏是是伏、郝、任、包、宿、句、乘中的任何一個，那麼你們也有著共同的祖先，那就是伏羲。

其實伏羲原本的姓氏是”風”，不過在之後漫長的歲月里，他的後人出現了許多不同的分支。

而《山海經》中記載伏羲的父親來自一個叫做雷澤氏的巨人部落，所以你們的血脈中或許有著巨人的血。

而如果你的姓氏是羌、姜、允中的一個，那麼恭喜你，你的擁有著戰神的血脈。

提到戰神我們第一時間想到的或許是和黃帝、炎帝大戰的蚩尤，但是在中國的傳說中最強大的戰神無疑就是九天玄女了，而根據《山海經》的記載，九天玄女是真實存在的，她就是當時弇茲氏的第一代首領。

而她的後人就分化成了前文提到的三個姓氏，所以這三個姓氏的人如果血脈之力覺醒，估計都會成為一個個戰神。

而在《山海經》中還記載有這樣一位名叫顓和的女神，她掌管著美貌，不僅可以讓人顏值上升，更可以青春永駐，而他的後人如果繼承了她的神力，那可真是讓人羨慕。

而顓和是顓頊一脈的後人，顓頊的後人很多，如今曹、潘、李、陳、孫、鄧、黃等姓氏的人都是他的後代。

所以在這些姓氏人的血脈里，說不定就蘊含著讓自己變美的神力，不知你們身邊這些姓氏的人是不是都是帥哥美女呢？
