嘴上說不用…內心「超渴望過節」星座Top5　天蠍只提1次：忘記就死定了

2019-12-22

在你身邊也有沒過到節就會氣噗噗的星座嗎？（示意圖／記者李佳蓉攝）

記者李佳蓉／綜合報導

下周即將迎接聖誕節、跨年的到來，你已經約好另一半、重要家人及朋友一起過節了嗎？無論是小倆口的約會，或是開趴狂歡，一同享受過節氛圍。而在你身邊，是否也有一些人嘴上說不想過，但其實內心很渴望，只是你不知道。Youtuber《》分析一定要過節的星座Top5，一旦忘了小心面臨分手、絕交的命運。

聖誕節、跨年要來了！安德魯分析必過節的星座Top5：

1.獅子座

獅子們很愛過節，尤其是和伴侶和重要朋友，想要營造美好回憶。但大多的獅子嘴巴都很硬，是標準的口是心非，對於過聖誕、跨年老愛裝做不在意，等到日子逼近才開始忐忑不安。若你不打算和他過節，小心他玻璃心碎亂想「在你心中是不是沒地位了」「你是不是不在乎他了」。

愛面子的獅子座想要霸佔你心中最重要的位置，且千萬別忘了萬壽之王喜歡被眾人崇拜、被注目，一旦你今天因工作忙碌或是一時粗心忘了約他，獅子嘴上說著沒關係，但心裡其實很受傷，當他發現你沒那麼需要他，可是會傷心的選擇離開。

2.天蠍座

「節日」對天蠍座來說很重要，且會在一開始就會表明，要你千萬別忘記！冷靜派的天蠍不會把喜怒哀樂表現在臉上，尤其是開心也愛藏在心裡，因此常在你準備大驚喜、細心安排後，又被他臉上尷尬又不失禮貌的微笑給嚇到，擔心是自己做得不夠好，才讓天蠍不開心，事實上只是他們有些奇怪的標準罷了。

雖然天蠍外表冷漠，但仍能透過他「細小動作」知道他開不開心，只要發現他慢慢的對你關心變多、對你形影不離、牽你牽得更牢、抱你抱得緊，透過這些小互動都可以察覺出來他內心的歡愉。但要再三提醒的是，天蠍跟你說「這節必須過」千萬別忘記，因為他只會講一次，你沒記起來後果不堪設想。

3.巨蟹座

巨蟹要的不多，只希望能和你在一起，在特別的節日里寫下最珍貴的回憶，可能只是吃飯看電影，但內心就很滿足。細膩的巨蟹容易想很多，想和你過節卻又擔心「會不會只有他一個人想過節」，無法猜測你的想法。巨蟹的暗示大多很細微，身邊的朋友也常迎來許多小驚喜，所以當你發現節日近了身邊的巨蟹有些不一樣，像是約會時的餐廳氣氛特好、特浪漫等等，就能看出他很在意過節。

4.處女座

對自己要求很高的處女座，想要成為他的及格戀人怎麼可以忘記過節呢？當節日快到時他們會精心計畫很久，把每個小細節都處理得妥當，當節日到來時就是他可以表現的機會，但卻又怕看到另一半或是重要朋友臉上的失望表情，因此總是耗費心思準備。當你發現處女座驚心為你準備節日驚喜，請不要潑他冷水喔。

5.雙魚座

12星座里最愛過節的就是雙魚座！喜歡沉浸在過節的氣氛和浪漫中，懂得經營感情的雙魚座，認為談久了的戀愛會變得無聊又乏味，因此會利用節日製造驚喜讓感情升溫。雙魚寶寶的戀愛很單純，堅信著「我對你好，你一樣也會對我很好。」當他準備過節招數時，也渴望你能有熱情回應，一同享受他準備的約會和驚喜，無論大大小小的節都會想辦法過得很精彩。
