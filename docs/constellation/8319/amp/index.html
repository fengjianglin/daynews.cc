一周11.25—12.1星象分析，火星對分天王星，射手座新月上演

天天要聞
2019-11-24

這是水逆後的第一周，也是11月的最後一周，本周水逆的陰影期也會漸漸退去，但是陰影事件並不會就此沉寂。本周，火星和天王星的對分，將會進入到一個比較比較激烈的階段，很可能會給你帶來一些咋舌的事情和消息。水星恢復順行後與土星會形成一個和諧的角度，這個角度會幫助我們得到一些有用的信息，推進一些事件的進展。金星與木星相遇後，在本周來到土象摩羯座，這會給我們帶來一些比較實際的金錢提升和穩定的情感狀態。射手月已經開啟，本周在射手座會發生一次新月，包括射手座在內的很多星座會迎來嶄新的局面。接下來青峰就為大家帶來一周星象分析。

火星對分天王星

這一星象目前已經走到了比較激烈的對抗程度。這一星象會給我們帶來很多內心層面的衝擊，以及現實層面的對抗。具體的影響如下：

  1. 火天對分，會激起人們內心最深處的渴求。你會變得非常有保護欲，特別想要封閉自己的內心，不讓別人走進來。很多時候，會想得比較極端，凡事更願意往壞的方向去想。
  2. 做生意或是管理團隊的朋友，這段時期要格外注意，非常容易出現生意夥伴突然拆夥、離去。團隊成員離開，或是因為某件事矛盾升級。
  3. 在面對一些機會出現的時候，你會非常猶豫，甚至有些暴躁的情緒，這種機會可能表現在情感、事業、學業等各個方面。
  4. 與家人、同事、情侶、子女的相處中，你會在某個點裡，難以控制情緒，非常容易引起衝突。這段時間甚至比較容易出現暴力事件，要注意控制自己的情緒。

水土相合
水星和土星目前有一個和諧的角度，在過去的一個月里，你是否在水逆中曝出了一些問題，一些事情是否已經進行到收尾的階段？你目前是否某個合作上，即將要經歷終極的談判？本周非常適合去做一些溝通的事情，尤其是在談判方面，會有意想不到的結果。而你之前所暴露的問題，也會得到解決，也許可能是被迫去解決，但是會有一個妥善的結果。同時本周，在這一星象的影響下，也很適合情侶訂婚、見家長等，都會有很舒服，很和諧的過程。

金星進入摩羯座
本周金星會來到土象摩羯座，這一星象會給我們帶來一些比較實際和現實層面的提升。首先就是金錢上的提升。這段時期，大部分星座的偏財運可能不會特別旺，但是正財方面的財運非常值得期待。正值年底，很多星座可能會升職加薪，或是得到年終獎和分紅。做生意的星座，也會有更多的訂單和銷量，即使是不景氣的生意，也會有些起色。同樣這段時期，也會給我們帶來情感上的穩定。大部分的星座雖然受到火天對分的影響，可能情緒不會很穩定，但是在感情上會比較親密，大部分的時候還是比較和諧的。這個時期，你也會更加的關注自己的家庭、房屋、地產、車輛的問題，很多人會在買賣、維修、保養等方面有比較多的支出。

射手座新月上演

太陽目前已經來到射手座，射手月已經正式開啟了。本周會在射手座上演一次新月，包括射手座在內的很多星座都會迎來一次蛻變，和一個新的開始。對12星座的具體影響如下：

  1. 如果你正經歷疾病和傷痛的困擾，此次新月會給你帶來一些治癒。很多人會走出疾病的困擾。
  2. 在事業上，你會有很多新奇的想法，或者對當下的工作重新進行規劃，對未來的發展有個新的認識，你會堅定某些事情，也會放下某些事情。
  3. 身心上都會有一些疲憊，特別想要得到一次休息，但是現實可能並不允許，所以你可以選擇一個適合自己的方式去放鬆自己
  4. 在處理重要文件、合同、收據、欠條等方面，要謹慎一些。
  5. 這段時期會有比較多的維權事件出現，如果可以退一步的，盡量不要引起更大的衝突。

青峰結語

雖然本周的星象變化很多，有好的影響，也有需要注意的地方。但是這是11月的最後一周，無論怎樣，你要學會克制自己，管理好自己的情緒，一是給11月，一個好的結局，二是給12月一個好的開始。明日會為大家帶來12星座一周運勢分析，敬請關注。
