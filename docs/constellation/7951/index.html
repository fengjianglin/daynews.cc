什麼樣的女人容易幸福｜幸福密碼

2019-12-07 1,249條評論 3,359次閱讀 624人點贊

珊珊和老公李昂是高中同學，他們在一起是大一的第一年，珊珊追的李昂，用珊珊的話說，「我對他一見鍾情，從高一第一次軍訓他一身軍訓服出現在我眼裡我就惦記上他了，高中三年我就怕他跟別的女生在一起，還好命運眷顧我」。

圖片來源網路，侵權刪

珊珊是那種相貌平平，個子不高，還有點微胖，唯一的優點是圓圓的臉蛋上輕易就有笑容的女孩子，走到哪裡都能看到她在笑，愛笑的女孩運氣不會差這話形容她最合適不過了。而李昂屬於陽光大男孩，自帶好人緣，班裡第一次競選班委，竟有百分之八十的同學選李昂做班長。李昂愛打籃球，體育課，運動會李昂打籃球的觀眾里珊珊永遠是最激動的那個，她說她最喜歡李昂做事全神貫注的樣子。喜歡一個人一定是有原因的。

圖片來源網路，侵權刪

大一第一年珊珊就主動加了李昂的扣扣號，直接表達了自己的心意，而李昂沒有拒絕，雖然是異地但是他們之間永遠有聊不完的話題。後來珊珊問李昂為何選她，李昂說高中三年里除了高強度的學習外記憶力最深的就是珊珊圓圓的臉上甜甜的笑容，每次看到都感覺很輕鬆很幸福。笑容是會感染的。

大學畢業兩年後，珊珊和李昂就結婚了，現在珊珊是兩個寶寶的媽媽，問起珊珊的幸福密碼「遇到對的人要勇敢去追求，永遠相信最初的選擇，往後餘生知足感恩」珊珊笑著說。我問「你們不吵架嗎」「也吵，但是都是小吵小鬧，我們也經常嫌棄對方脾氣怎麼變大了，對方的生日怎麼忘記了，但是，在最需要對方的時候我們還是一起扛。大的事情面前我們一起商量，小事上要麼我聽他的要麼他聽我的，只要不破壞我們的感情其他都是小事，」。

圖片來源網路，侵權刪

幸福的密碼就是永遠相信最初的選擇，第一眼便是一輩子，我不奢求更多隻希望和你長相守，婚後的柴米油鹽醬醋茶我們一起承擔，你永遠是我眼裡那個最初的大男孩。

圖片來源網路，侵權刪

楊欣是珊珊的發小閨蜜，兩人從小學就認識，雖然上了初中兩人就不在一個學校，但是每次寒暑假兩人見面就有說不完的話。楊欣是家裡的老大，下面還有弟弟妹妹，所以楊欣做任何事都以榜樣的標準來要求自己，學業上、事業上、甚至找男朋友上。

楊欣第一次戀愛是大學時期，初戀男友提出牽手擁抱等楊欣斷然拒絕了，因為她覺得不是很滿意男友，珊珊問「那你們為什麼要戀愛，不滿意不在一起就可以了，為什麼還那麼勉強」，「不知道，只覺得上大學了該談一個男朋友了」楊欣淡淡的說。最後楊欣和初戀就這樣分手了，一直到工作了好幾年楊欣一直單身，家人和朋友介紹男生楊欣總是談幾天就斷了聯繫。遇到的人很多但是都走不進心裡，那麼寧願單身也不想湊合，一輩子只有一次感情可不能勉強。

圖片來源網路，侵權刪

楊欣終於戀愛了，雖然男生看起來黑黑瘦瘦，相貌一般，但是他的眼裡全是楊欣。楊欣上下班男生總是在公司門口等她，楊欣發脾氣男生總是一副笑嘻嘻的樣子。楊欣決定結婚了，她說「我要的不多，恰好你都給，這就足夠了，我脾氣不好但是你都懂，感謝你的包容和理解」。

幸福的密碼就是我遇到的那個人一定是願意包容我理解我的。雖然有時候會意見相左，但是在你面前我不用偽裝，我是最真實的自己。

叔本華說「一個人相信什麼就會成為什麼」

在愛情面前，我們最初想得到怎樣的愛人你的愛情觀便會靠近怎樣的愛人，有什麼樣的信念就有什麼樣的人生就是這個道理。
