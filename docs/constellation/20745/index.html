明朝的欽天監、陰陽學、陰陽戶都是幹什麼的？

2019-12-18 2,040條評論 3,522次閱讀 2046人點贊

明朝的欽天監不是官名，而是一個官署機構（相當於國家天文台），其職掌是觀象、編歷。所謂觀象，就是指按規定日期，觀測日月、星辰、風雲、氣色，如有變異，要即時報告。明朝對天象非常重視，比如出現日食、月食的天象，欽天監要先期測出報告，禮部移文內外諸司，屆時京師及全國各地要進行伐鼓救日（月）的活動。祭祀和各種典禮，皆由欽天監選擇時日。所謂編歷，主要是指編寫《大統歷》、《御覽月令歷》、《七政躔度歷》、《六壬遁甲歷》、《四季天象錄》等資料。

明朝在建國前設太史監，吳元年（1367）改監為院。洪武元年（1368年）改太史院為司天監，洪武三年（1370）改司天監為欽天監，洪武十四年（1381）欽天監定為正五品衙門，設令、丞各1人。洪武二十二年改監令為監正，監丞為監副，總領觀象、編歷之事。

欽天監下設天文科、漏刻科、回回科、歷科。除了監正（正五品）和監副（正六品），還有主薄、五官正、靈台郎、保章正、挈壺正、監侯、司歷、漏刻博士、司晨等若干人員，專業性比較強。根據明朝的規定：本監官不得改遷他官，子孫世業，非特旨不得升調、致仕，如有缺員，由本監逐級遞補。

明朝地方設有陰陽學（有些類似天文學校，但卻是一個多功能的地方性官方機構，具有陰陽學教學和管理之職，兼司欽天監之責，以及組織官方的祭祀儀式）。府級的管理人員稱正術（從九品）；州級的稱典術（品秩未入流）；縣級的稱訓術（品秩未入流））。洪武十七年置，設官不給祿。民戶中還有陰陽戶，其正丁被稱為陰陽生，基本差役為計時、報時、記錄災異祥瑞等，陰陽生另以風水、擇日、占卜等為生計。
