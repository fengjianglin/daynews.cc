職場中最會划水的3個星座，水瓶有能力不表現，雙子只想吃喝玩樂

2019-12-12 692條評論 4,682次閱讀 1514人點贊

或許對於大多數人來說，工作的根本目的就是為了生活，通過工作來獲取經濟回報，從而保證自己的經濟生活穩定。可能每個人都不願意讓自己的工作太辛苦，但有些人為了能讓自己的收穫越來越多，他們願意在工作中付出更多努力，從而讓自己的事業能夠不斷進步。但有些人卻比較懶散，只要工作能夠保證生活穩定就足夠了，沒有必要給自己找麻煩，所以就顯得比較消極。那麼你知道哪些星座的人最沒有上進心，工作態度消極如同混日子一樣嗎？

雙子座：貪玩

雙子座一般都比較貪玩，雖然他們也希望自己的物質生活品質能夠越來越好，而且有時候雙子座也會熱血上涌的想要做出一番成就。但等到真正開始做的時候，雙子座卻會選擇退縮，或者說在工作的過程中雙子座始終難以專註下來，總想著去做些其他事情。雙子座愛玩，對於他們而言生活中最重要的事情就是玩樂，如果在工作與玩之間做選擇的話，雙子座會毫不猶豫的選擇後者。至於工作，雙子座只要夠吃、夠玩就已經足夠了。

雙魚座：懶散

雙魚座一般都沒什麼上進心，或者說他們對自己的物質生活一直不是特別重視，大多數雙魚座人都生活在自己的幻想之中，不願意付出努力。很多時候雙魚座情願坐在辦公室里發獃也不願意動手工作，對於雙魚座來說只要能吃飽穿暖就足夠了，如果要付出更多的努力來換取更好的生活，雙魚座感覺很不值得，所以他們在職場上的表現一直以來都比較懶散，給人一種混日子的感覺，而雙魚座自己則很享受這種悠閑的工作狀態。

水瓶座：內斂

水瓶座一般個性都比較內斂，他們不喜歡出風頭，也不願意太過張揚，甚至很多時候水瓶座都給人一種很不合群的感覺。這主要是因為水瓶座比較個性，他們的想法以及為人處世的方式都和別人存在很大差別。因此對水瓶座而言，工作不需要太過努力，太努力了就太吸引別人眼球了，那樣被關注的感覺讓水瓶座很不舒服，所以很多水瓶座的人即使是有能力也不願意表現，在職場上也多是混日子，盡量的讓自己處於「隱身」狀態，水瓶座覺得不被人關注是最好的。
