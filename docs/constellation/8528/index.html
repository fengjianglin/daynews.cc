11月中旬開始，火星即將在天蠍座--靈魂將像鳳凰一樣轉變

2019-11-16 1,646條評論 4,827次閱讀 3588人點贊

火熱的火星即將在風系星座天秤座中結束，並在幾天之內進入水上星座天蠍座。就在天蠍座的水星還原即將結束之際，火星將在天蠍座完成2019年（至2020年1月3日）。火星是近兩年前（2017年12月-2018年1月）在天蠍座的最後一次。

天蠍座的水星逆行可能給我們所有人帶來了很多挑戰，但是在天蠍座的火星，您將有機會使事情變得正確，並使過去幾周發生的任何事情都回到正軌。如果您被推回，壓低，回溯或完全失敗，那麼此火星在天蠍座的運輸為您提供了像鳳凰一樣升起的機會。

火星

火星是占星中的能量和動力。在您的出生圖中，火星顯示了促使您採取行動的方式以及行為方式。在運輸過程中，火星會顯示您目前的駕駛情況，當前精力的集中點以及您當前打算採取的行動。

火星是一個充滿熱情和令人興奮的星球，它將這種能量帶到任何接觸的地方。這是通過火星巡迴演出的信號引導的，並針對您本人將重點放在火星旅行中相位所統治的生活領域（不知道您的圖表嗎？請查看出生圖表計算器！）。

天蠍座的含義

天蠍座是帶著轉變力量的星座。天蠍座的自然統治行星是冥王星，即變壓器。使用Scorpio，您必須更深入地挖掘並做出深刻的改變。這就是天蠍座和冥王星與變革之星天王星（以及天王星統治的水瓶座）不同的地方。天王星想要做出的改變更膚淺，而冥王星和天蠍座則希望做出深刻而有意義的改變，並且這種改變將持續很長時間。天蠍座是認真對待事情的標誌，並且有著強烈而堅定的專註。無論它想做什麼，它都會完全做到這一點。這個進程沒有中途！這使Scorpio可以進行那些深刻的轉變。

天蠍座比以往任何時候都更強大，更強大，更專註。這個星座與從灰燼中升起的鳳凰相連，天蠍座沒有失敗。天蠍座的火星與天蠍座的其他行星天蠍座實際上曾經是由火星統治的，所以天蠍座的火星比其他行星更容易處理能量。這是一個強烈的信號，要處理其他行星可能會有些棘手。

太陽將這種能量帶入我們的個性中，而天蠍座的月球將這種能量帶入我們的情感表達中，因此我們可以通過這兩種方式真正感受到天蠍的強度。天蠍座的水星可以輕鬆一點，因為如果您可以控制自己的注意力，心中總會有一個出口（因為水星已經逆行，這並不是一件容易的事），而當天蠍座的金星將很多激情帶入關係中（如果您單身或缺乏牢固的聯繫，可能會有些沮喪。）。

近年來，木星和土星也即將進入了天蠍座（我們不必長時間在天蠍座中與天王星，海王星或冥王星打交道。）。木星是有幫助的，但可以增強強度，而土星則使我們吸取了迫使我們轉變的經驗教訓。

風系（雙子座，天秤座，水瓶座）這顆天蠍座火星的過往可以專註於您的工作，無論您是實際的專業工作還是日常生活中的工作。您可以進行一些轉換，以完成更多工作並更加實用。緩慢而穩定地進行工作可以幫助您取得更大的進步，並且可以以實用，穩定的方式著手進行轉型。水星逆行可能使您難以取得進展和完成工作，因此現在進行轉換可以幫助您擺脫困境並專註於正確的道路。

水系（巨蟹座，天蠍座，雙魚座）這場天蠍座火星的過境可以專註於您想要將創造物帶入世界的方式。變革可以幫助您入門，並可以打開樂觀和機會。您可能會覺得自己正以嶄新的眼光看待這個世界，沒有什麼能阻擋您。水星逆行可能是阻礙您前進的原因，但是天蠍座的火星讓您有機會從灰燼中重歸，就像典型的鳳凰！離別的想法…
