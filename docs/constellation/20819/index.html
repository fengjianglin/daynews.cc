土冥的持續合相影響力

一般情況下，土冥合會以約32~38年的循環重複發生。而影響力，則視落座而產生不同的變化。

2019開始的土冥合摩羯，對國家政府、大型組織企業帶來極強的顛覆和重生之力。回顧百年來的土冥合相，從1914年的土冥合巨蟹開始，是從土星從陷落逐漸走向廟旺的過程，可以說，2019的土冥合摩羯，是強強對決，而非世紀初的恃強凌弱，而縱觀冥王星的世紀運轉流程，會發現，冥王星每次落入摩羯的時間是大約15~16年。每次冥王落摩羯，都會帶來一些動蕩，也帶來一些制度的革新。土冥的合相則增強這種力量，2020年，土冥的合相還會加入木星。

2月，6月，11月，是三顆星的交匯期，土冥對於木星的牽制自然會增加，這也會帶來經濟的減緩，和宗教問題的曝光。同時也會帶來之前一些過熱的紅人的真相被揭露，以往那種為所欲為的擴大化的炒作商業行為將不得不被收斂。

但是，2月依然會表現出較好的期望，因為土星很快進入水瓶，帶來一批高科技的紅利期許，6月則會顯得壓力較大，經濟與資源將產生變化，許多投資必須謹慎，11月全球的情勢會變得緊張一些，因為水瓶帶來一種顛覆和反叛色彩，這也讓木星不甘被壓制。

總之，2020年，傳統的經濟模式，會得到不小的顛覆，而新興的經濟模式，也會遭遇擠泡沫，總的來說，謹慎前行，是很必要的。也因此，未來一年，大家在政經、管理、不動產界或批發業容易有不錯的發展。

二大日蝕與三大月蝕

日月蝕是非常重要的星象，它們通常成對出現，為我們的生活領域帶來巨大的改變，喬遷，移民，買賣房屋，辭職，分手，離婚，晉陞，裁員，創業……可以說，是讓當事人感應特別深刻的星象。

但日月蝕的效力通常有一段延後期，所以，在日月蝕發生的三個月內會看到較為明顯的推進。

2020年兩大日蝕分別發生在：2020年6月21日 11:46和2020年12月14日 21:34

6月21日這次日蝕可以說是一場天象的四部曲。在這一天，早晨5點44分太陽剛剛進入巨蟹座，中午11點46分就出現了日蝕，到下午14點02分月亮也進入了巨蟹座，四十分鐘後，新月也就如期而至。

這是2020年最重要的一次日蝕，對我們的家庭、家族、房產等「歸宿」性問題都會產生深遠的影響。

巨蟹座是月亮的守護星座，在日蝕發生時，與太陽合相的月亮能釋放出很強的感性力量，而這種感性，也有可能是消極的。

尤其是在這次日蝕發生時，水星還在逆行，並且對沖木星、冥王星。這就意味著大家感受到很強的壓抑感，甚至是恐懼和困擾。人們在恐懼時，更容易表現出的狀態是憤怒，所以這一時段人們也容易出現衝動和抱怨。

受土星的壓制，我們的表達也可能會不夠清晰，也容易表出現冷漠的一面。

如果你的生日在本次日月蝕前後五天，也就是6月16日到6月26日（生日是日月蝕當天效應更強），未來90天內乃至一年能看到連番變化帶來的巨變。

日月蝕前後容易發生自然災害，所以明年的六月，我們要注意的有地震，海嘯或者山體滑坡，水災等等自然災害的產生。

也因此，在這段時間著重提醒星盤裡有巨蟹0~14度重要星體或者軸點的朋友，需要注意的幾個方面：包括日月蝕前後不草率決策，無論是辭職還是分手，還是搬家，等等，都最好等到一周後再辦理。

而日月蝕前後也要控制情緒不要過分激動，因為這可能只是短期效應，從長期看，往往是好事居多。因為，12月14日射手座的日蝕會給我們之前巨蟹座日全食的回饋，無論正面或者負面，都關乎我們在上一個日蝕時做出的決定。

除了影響巨大的日蝕之外。今年還會出現三次的半影月蝕。

首次半影月蝕，發生在2020年1月11日的巨蟹座，此次月蝕影響的主要面向包括：愛情、家庭關係、同事關係、親子關係，也會影響到你對環境的適應；

第二次則是6月6日在射手座的月蝕，此次的月蝕主要影響：理想主義的盛行、社會價值的重塑、傳統的一些認知被顛覆；

最後一次則是11月30日位於雙子座的半影月蝕，則會影響更多人的愛情走勢。

這次月蝕承接了前幾次的日月蝕走向，由巨蟹-摩羯軸線開始與完結，走向雙子-射手軸的輻射。

可以說，明年家變會增多，而企業里的變化也會增加，對個人來說，不得不更加務實的考慮生計和如何去順應潮流。在感情上，也容易承受變故，而雙子射手軸則預示著未來一年，關於教育方向，娛樂方向，傳媒方向，將出現一些震動，可能會有新的輿論方向，而無論怎樣，更提示了我們需要保持一顆獨立思考的自由之心。
