雙子座：定力不強

雙子座其實想要成功並不是什麼困難的事情，只不過他們沒有繼續堅持下去，而是選擇了放棄。也正是因為這樣的性格，註定了雙子座是無法走下去的。雙子座定力不強，還沒等看到成功的時候就會選擇打退堂鼓，這就會讓人們感覺到很懊惱。為什麼從一開始的時候不直接選擇放棄，而是非要浪費了這麼多時間之後才選擇放棄呢?這樣的話意義在哪?雙子座其實自己也說不明白，只是覺得不能繼續堅持下去了，隨便找個理由退避就好了，雙子座就是這麼簡單的一個人。

巨蟹座：過早認輸

巨蟹座的想法很簡單，不想繼續加油了，那就退縮唄!反正巨蟹座是能夠耗得起的。很多人都沒有這種覺悟，巨蟹座有這樣的覺悟。有的時候巨蟹座會覺得成功了也沒有什麼意義，所以他們會在成功之前選擇放棄，因為他們所追逐的東西並不是成功，成功對於他們來說只是虛名而已，並沒有那麼在意。巨蟹座更喜歡享受中間的過程，雖然聽起來有些裝，但是巨蟹座還真就是這樣的一個人。你如果總是把他們想像成那種激進的人的話，那就是你的錯了。巨蟹座從來都不激進!

獅子座：不敢堅持

獅子座別看平時特別的強大，其實在遇到困難的時候他們也是發矇的。他們也不願意繼續堅持，畢竟他們沒有那個資本。等到窮途末路的時候就算是堅持一分鐘，獅子座都會覺得非常困難的。他們更喜歡半路退縮，這種做法才更像是獅子座的作風。獅子座並不是什麼敢于堅持的人，他們有很多顧慮，哪怕是一些細節問題他們都會考慮進去，所以千萬不要覺得獅子座定力很強。如果你把他們想像的太強大了，那他們就真的只會讓你失望了，分分鐘打臉，想都不敢想。
