大話神話單詞--時間神克洛諾斯篇

2019-12-15 146條評論 2,351次閱讀 578人點贊

克洛諾斯Cronus是時間之神，他的妻子瑞亞生了許多子女，但每個孩子一出生就被Cronus吃掉，就好像時間無情的吞噬一切。

[天文]Cronus在羅馬神話中被稱為Saturn，他還同時擔任著農神一職，經常拿著鐮刀，所以古羅馬天文學家以Saturn命名土星。而為了紀念他，星期六也被命名為Saturnday。

[星座]星相學家認為，由於這一行星離太陽最遠，運動遲緩，因此出生於這一星座下的人具有一種憂鬱的氣質，陰沉而孤僻。英國人在15世紀時創造了saturnine這個詞，意思是「性情陰鬱的,性格乖戾的」。

Chronicles n.(舊約聖經)歷代記

有一種名叫synchronized swimming 的表演。就是配樂水上芭蕾（ballet）表演，即花樣游泳。

synchronous a.同時發生的；同步的[分解]syn- 同一 chron 時間 –ous的；「同時」

【記】syn-(同步的) + chrono-(時間的) + flash(閃光)
