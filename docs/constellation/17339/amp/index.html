常見的12星座你會記憶嗎？一招讓你難以忘記

天天要聞
2019-12-16

記憶十二星座：

水瓶座、雙魚座、白羊座、金牛座、雙子座、巨蟹座、獅子座、處女座、天秤座、天蠍座、射手座、魔羯座。

運用串聯聯想法我們可以這樣想像：

從一個水瓶（水瓶座）里游出了兩條魚（雙魚座），結果它們一游出來就游到了白羊（白羊座）的肚子里了。這時，一頭很大很重的金牛（金牛座），一腳把山羊踩扁了，這兩條魚從白羊的屁股里被擠了出來，變成了兩個小孩子（雙子座）。接著，一隻巨大的蟹子（巨蟹座）用它那兩個大鉗子鉗住這兩個小孩的屁股，鉗得他們哇哇大哭。這時，一隻獅子（獅子座）張開它的血盆大口，一口把巨蟹咬成兩半。再一看，獅子身上坐著一個美麗的少女（處女座），少女的頭上頂著一個天秤（天秤座），天秤上掉下來一隻蠍子（天蠍座），想要去咬少女美麗的臉蛋。說時遲那時快，出現了一個射手（射手座），這個射手坐在一隻山羊（山羊座，即魔羯座），一箭射來，就把這隻蠍子射死，把少女救了。

串聯聯想法適合用於記憶一連串需要按順序來記憶的資料。運用串聯聯想法一個一個地把資料串起來，可以很快記住數十個甚至上百個資料。然而，串聯聯想法也有其不足之處。

串聯聯想法的缺點：

第一，一次的記憶量不夠大。

運用串聯聯想法來記憶，資料的內容不能太多，太多就容易混亂。例如記幾百個數字，如果使用串聯法來記憶的話，需要把數百個圖像一個一個串聯起來。理論上我們可以把無限的信息緊密地串起來，但事實上，串聯聯想法就像串珠子一樣，如果串得太多，線就可能會斷，其中有些圖像的記憶就會模糊，就會容易忘記。

第二，缺乏回憶的線索。

串聯記憶最關鍵的就是要記住第一個圖像。回憶的時候，回想起了第一個圖像，其它的圖像就會一個一個被「鉤」出來。然而，往往最難回憶的就是第一個圖像，因為它缺乏回憶的線索。如果這第一個圖像回憶不出來，那麼，後面的所有圖像都很難回想起來。

第三，不利於快速準確地回憶出其中的特定信息。

有些時候我們需要快速準確地回憶出其中的某個信息，如法律條文，當別人問到第幾條的時候，必須能夠馬上對應地回想出來。如果運用串聯聯想法來記憶，要從前往後或者從後往前地一個一個數，這樣的回憶就顯得很慢了。或者記撲克牌，別人問第25張是什麼的時候，你不可能從第一張到第25張回想一遍才回答，這樣就太慢了。

第四，當記憶信息比較複雜的時候不容易進行串聯。

例如記36計，每一計都由多個複雜的畫面或動作組成，而並不像「太陽」、「森林」這些名詞一樣僅僅是一個簡單的畫面。所以，如果要用串聯法來記36計，由於每一計都由多幅畫面所組成，當要用這一計來與下一計進行聯結的時候，不太容易作出清晰準確的聯結，因此也就不利於記憶。

所以，串聯聯想法一般來說僅適用於一些比較單純的畫面之間進行聯結，而並不太適合用於一些比較複雜信息的記憶。
