星座說：2019年年末各個星座闊太錦鯉屬性大盤點

2019-12-03 565條評論 1,948次閱讀 145人點贊

嗨，大家好我是甜甜的木糖醇。

其實大家對於「闊太」這個詞有著很大的誤解，在眾人眼裡一直是被鄙夷的，通常形容女性出手闊綽，生活質量標準較高，她們人生觀以及消費觀讓很多人不能理解，而且經常出入高端美容院，再加上近年來網路報道媒體黑化導致在眾人眼裡「闊太」成為了一種諷刺，比如說網路上出現的「闊太炒房，闊太炒黃金，購買大量奢侈品等等」。

其實「闊太」也是人家經過多少年打拚有著足夠的資本去享受什麼，只是被這個社會輿論覆蓋，讓人覺得是嫁入豪門成為闊太，其實又有多少人能嫁入豪門呢？大部分人還是靠自己的努力奮鬥，其實你只看到現在的她光鮮亮麗，卻不知她們在打拚時是多麼的拼，現在去享受是理所應當的，接下來盤點一下各個星座的闊太錦鯉屬性，不是以外貌而形容，請不要誤解她們哦。

天秤座，凡事要做到心寬，有時候會因為一些人或者一些事情導致自己一大階段處於不開心狀態，做什麼事心不在焉，嚴重影響心態，所以天秤座錦鯉屬性定位：心寬。

處女座，瓶頸之時如何順利突破，處女座在一定時間不管是工作還是心理上會遇到瓶頸，心思過於較真導致自己煩躁不安，保持隨性狀態也學一瞬間雲開霧散，所以處女座錦鯉屬性定位：佛系。

摩羯座，所熟悉的摩羯座屬於有目標性的一個人，對於自己的目標相當的努力，成熟下的認真超級害怕，有時候過於認真會產生委屈狀態，所以摩羯座錦鯉屬性定位：平衡。

射手座，給人感覺比較舒服的射手座，不管做什麼總是會「我認為」，看人看事總是自己的觀點不會在意別人怎麼想，過於自我導致他人反感，所以射手座錦鯉屬性定位：沉澱。

天蠍座，大多數天蠍有著極強的佔有心理，自己上進心超強可是實際有點無奈，但是不會過於抱怨目前所遇狀況，會很正視自身，所以天蠍座錦鯉屬性定位：勇敢。

獅子座，正面超級剛的獅子座背後也是有著小女人的一面，渴望別人的關心，渴望別人的懷抱，但是表面還要裝作很強硬的一面，所以獅子座錦鯉屬性定位：放鬆。

金牛座，對於美食毫無抵抗力的金牛座其實很反感別人對自己說三道四，表面上裝作無所謂的狀態內心已經把你列入黑名單了，不會給你好的臉色，所以金牛座錦鯉屬性定位：淡然。

雙子座，三心二意狀態下的你真的很差勁，特別是遇到一些煩心事情導致自己變得慵懶，越煩躁越無所謂，不管是你還是別人都會反感這個階段的你，所以雙子座錦鯉屬性定位：減壓。

雙魚座，內心已經拍完了多少部宮廷劇的你，特別愛幻想著總是害怕發生點什麼意外，內心極度缺乏安全感的你整體憂心忡忡的，所以雙魚座錦鯉屬性定位：靜心。

水瓶座，給人感覺安靜又有主見的水瓶座過於封閉自己內心，內心有許多想法不願與人分享，說水瓶悶騷吧有時候也挺逗的，所以水瓶座錦鯉屬性定位：釋放。

白羊座，熱情樂觀派的白羊座會被現實中的點滴束縛著，雖然會很快消化掉，但是下一個問題接踵而來，承受能力差的人會變得壓抑，所以白羊座錦鯉屬性定位：認知。

巨蟹座，過於懂事的巨蟹座總是在意別人的點滴以及對自己的看法，甚至到了會刻意的改變自己，當你為了一句話去改變自己去取悅別人你內心開心嗎？所以巨蟹座錦鯉屬性定位：自我。

其實每個星座的錦鯉屬性我給了兩個字，留給自己去深思你會懂得其中之意，以上僅小編個人觀點，評論區歡迎大家探討，小編會積極回復，喜歡星座題材關注我哦，不同的星座話題來襲。
