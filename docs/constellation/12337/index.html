為人耿直，真誠善良的射手座，毫無心機，值得交往

2019-12-12 408條評論 1,624次閱讀 620人點贊

1點星座

面對這個變幻無窮的社會，為了更好的適者生存，我們都學會了偽裝自己，不輕易把自己的真實內心展示在他人面前，變得小心翼翼。有時候不是人們想要無情冷漠，在面對外界的環境時，選擇自保是人之本性。但是也有一部分人，他們在看透了社會現實後，依舊初心不改，真誠善良的對待他人，即使受到傷害也不計較。那麼十二星座中的每個星座都有著不同的特性，射手座就屬於那種特別耿直，真誠待人的星座，沒有心機，和他們做朋友是極好的選擇，今天我們就一起來看看吧！

為人耿直

火象星座的射手，一直都洒脫自由，他們活在自己的世界中，從來不會在意外界對他們的看法。有人認為他們太過於放縱不羈，屬於不安分的類型，往往讓人對他們產生了偏見，認為他們就不是能夠成事或者值得交朋友的人。追求自由的他們，時常讓人覺得他們心性​不穩，不成熟，或者是說太過自我不會替他人著想。但是真實的他們，在與人相處的過程中，都是有一說一，不會拖泥帶水，總是熱心地幫助身邊的人，也不會要求什麼感謝，為人十分的耿直，不管是作為朋友還是工作夥伴都是很好的。

真誠善良

處於變動宮的射手座，表面上容易看著躁動不安，沒有耐心，但是他們有自己做人做事的原則。他們一旦答應了別人的幫忙或者是下定決心要做什麼事情，一定會信守諾言，不會因為自己的情緒波動就影響到他人的事情。幫助他人的時候，也是拿出了自己的真心實意，把別人的事情當作自己的事情一樣，認真地去做，非常的善良。

毫無心機

每天忙著探索發現新事物的他們，根本沒有空閑的時間停下來想太多。與人相處也不會想得太多，你對他說什麼他就聽什麼，如果你對他拐彎抹角地說話，他可能還會因為聽不懂而一頭霧水。所以，與射手座打交道，不需要遮遮掩掩，大膽地說出自己的想法，對雙方都是事半功倍。毫無心機的他們是不懂的套路的，也沒有心思去想太多。

值得交往

自由瀟洒的他們，每天都那麼快活。不喜歡太在意一些外表的東西，追求自己內心的平衡。往往外界對人的看法，都是看表面，這個人成功不成功取決於做什麼樣的工作，住了什麼樣的房子，開什麼樣的車。他們對於這個物質的東西，並不過多的追求，與人交往也不會在意這些，他們看人都是隨心而行，覺得和這個人聊得來，三觀一致便會深交。為人耿直，真誠待人的他們，是值得交往做朋友的。

在這個社會當中生存，需要我們學會圓滑處世，但我認為圓滑也有一個度，不能因為圓滑而讓自己變得太過於虛偽，失去了生活的意義，變成了一個行走的機器人。在這個浮躁的社會環境下，學會靜下心來，做真實的自己，開心快樂​最重要。​​​​
