十二生肖十二首詩，好美好美，總有一首是你喜歡的！

七律*十二生肖詩之一(子鼠)

深宮小隱冀平安，黑暗偏能學藝專。

論算我當憑耳目，求生誰肯問心肝。

緣梁走索雖名巧，過市何時不膽寒。

獨幸支肖居首位，芳聲竟少有餘歡。

七律*十二生肖之二(丑牛)

勤勞力大善耕田，俯首甘為孺子牽。

獻乳何曾嫌啃草，拉車豈敢怕挨鞭。

傾心吐膽療人病，刻骨雕皮著史篇。

莫道一身多土氣，曾隨老子論虛玄。

七律*十二生肖詩之三（寅虎）

獨嘯山林號大蟲，休憑牙爪亦威風。

一身權勢由狐假，半卷丹青累犬充。

論志厭陪君主座，問心唯仰武松功。

相逢儘是謀皮客，寧守天然不受封。

七律*十二生肖詩之四（卯兔）

身行敏捷善賓士，株守何曾與論痴。

執杵陪娥居月殿，清塵洗鏡上蓮池。

明心可照千秋夜，作筆能書萬古詩。

相見時常咧嘴笑，謀安三窟耐人思。

七律*十二生肖詩之五（辰龍）

興雲播雨隱蒼穹，攪海翻江氣勢雄。

乾卦六爻涵聖德，東方七宿顯威風。

留名應是尊天子，匿跡原為愛葉公。

自古齊民唯信仰，何從首尾論神通。

七律*十二生肖詩之六（巳蛇）

原為大澤一長蟲，能屈能伸號小龍。

以毒損名還治毒，逢凶救主更懲凶。

足無亦可攀高頂，皮蛻尤添不老容。

至爽為人甘吐膽，清肝明目建奇功。

七律*十二生肖詩之七（午馬）

飛龍雅號誌昂揚，衛國隨軍騁戰場。

亂佐英雄爭霸業，安馱雅士賞風光。

名成不是悲鴻畫，命定何求伯樂相。

錦繡金鞍雖富貴，還憑勞苦創輝煌。

七律*十二生肖詩之八（未羊）

親族繁榮秉性敦，常將身命報人恩。

乳干願解群生渴，毛盡為求寒士溫。

跪食懷仁行禮義，捐毫吸墨寫乾坤。

齋餐陋舍心安逸，日日歡歌盪草原。

七律*十二生肖詩之九（申猴）

生身好動性聰明，敢替人間抱不平。

寶殿御前參玉帝，老君爐上煉金精。

閻羅示弱龍王懼，武藝高強魔怪驚。

蓋主功勛知退隱，千秋佛國鑄芳名。

七律*十二生肖詩之十（酉雞）

彩服丹冠似鳳凰，能歌善舞愛風光。

籬邊草地常梳羽，戶外花叢自找糧。

喜啄銅盆傳雅韻，欣書竹葉踩寒霜。

為驅黑暗離天下，夜夜辛勤喚太陽。

七律*十二生肖詩之十一（戌狗）

勇敢忠誠嗅覺靈，搖頭擺尾善逢迎。

巡山捕獵無辭險，守夜防偷最盡情。

護主功高烹豈懼，看人眼惡罵猶榮。

不言衰敗唯言旺，願保黎民享太平。

七律*十二生肖之十二（亥豬）

古已成家發展長，群雄逐鹿盡能詳。

爭權戚氏成人彘，待客伯奢隨鬼王。

酬德無忘身報主，結緣堪使佛跳牆。

生肖十二雖居後，幸得兒孫奕世昌。
