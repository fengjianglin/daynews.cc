近日星象：木星進摩羯座、水冥六分、金火六分

今天是一個重要的日子，木星從射手座進入摩羯座，帶來了踏實、認真與努力。面對現實的問題，這是檢驗過去你究竟是有所積澱，還是無所事事的時期。木星在射手座時可以憑藉著運氣去做事，這樣湊合的方式在木星進入摩羯座時期已經不適用了。摩羯座會用嚴厲的方式告訴你，你究竟在這個社會上擁有什麼價值。

​同時，順行的水星還與冥王星有個和諧的相位，有些人會收到驚人的消息與分離，分別，改變有關。本年度餘額只剩最後一個月！金星六分火星，這組能量會幫助我們去達成一些合作，看到關係的進展。不過一些人在關係中也容易存在權利上的鬥爭，但是也很容易質疑與鑽牛角尖。

金牛座
　　2020年，金牛座得到的福報和桃花有關，在落寞失意的時候，能有異性貴人相助，不僅能從對方身上學到東西，還能得到溫暖的關懷，尤其對於單身金牛座而言，這份福報甚至能給單身金牛帶來一段美好的姻緣，不過金牛座還是應該考慮清楚，究竟是否值得自己付出感情，要考慮的不是對方是否適合自己，而是要考慮兩人能否長期相處，條件是否匹配。

　　天秤座
　　2020年，天秤座比較容易因為做出了正確的選擇，而得到自己想要的結果，無論是在感情上還是在財運上，都能收穫意想不到的驚喜，這就是天秤座2020年最大的福報，其實天秤座最應該感謝的人是自己，如果沒有一顆平靜的心，以及腳踏實地的努力，不會得到這樣的好結果，繼續保持下去，未來更是一片光明。
