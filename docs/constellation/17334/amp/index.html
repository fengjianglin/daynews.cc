測試結果

A 、你自己

你非常逞強，凡事定要爭個輸贏，且很有骨氣，討厭被人看不起。在生活中你幾乎不會太依賴別人，因為你根本無法完全相信別人，控制欲很強，要把一切都拿捏在手裡，只有自己有把握才算是真的可靠。這樣的你在最艱難的時刻，也會殘存著意志要翻越過去，不能被打敗的信念令自己去開創天地，而周圍的人也很了解你的性格，除了家人幾乎不會太插手，你也不會讓他們插手，甚至不會表現出來這種痛苦，而是裝作這些問題很好解決的樣子。

B 、 你所愛之人

你是個防備心很重的人，而且比較敏感，因此在生活中你很難完全相信一個人，只有充分的了解，成為親近之人你才會卸下心房。這裡的所愛之人指的是：家人，朋友或者戀人，只有他們才是你在遇到麻煩，或者最艱難困苦的時候會幫助你的人。他們了解你，並且愛著你，當你痛苦的時候會陪在你的身邊。你也會依賴他們，形成彼此信賴的關係，幫助你渡過難關。

C 、 你結緣之人

你處事知禮得體，謹慎做人，落落大方，心態淡然平和，會給人一定的距離感，但心是善心，人是善人，也頗受好評。你在生活中會無意識的結緣，由於你的妥善應對，這些人並不一定與你親近，但卻結了感情。當你最艱難的時候，自然有貴人相助，這些人可能並不是你平時結交的人，也不是親近之人，卻是你人生的貴人。同樣，你的父母親也是你的結緣人，也是你在遇到挫折時最好的躲風避雨港灣。

D 、 你忽略的人

你做人比較高調，有時也不太拿捏分寸，大喇喇的性格讓你在生活中忽視了很多美好的東西。例如一直守候在你身邊的人，或者無意之間結交的人，或許這些人並不與你推心置腹，但卻能夠在危難之中救你一把。你可能在一瞬之間得罪了什麼人而不自知，也可能在生活中讓什麼人愛上你也不自知，有時甚至並不能分辨待你推心置腹還是只不過酒肉朋友的差別。你忽略的人，不會遺忘你，而當你艱難困苦的時候，那些你記著的，不一定會在你身邊。
