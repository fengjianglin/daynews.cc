招黑體質，得罪不少人的星座，風評欠佳，人緣差

2019-12-18 963條評論 3,097次閱讀 97人點贊

1點星座

有一種人即使傻不拉幾也招人喜歡，因為別人不會覺得他傻，反倒覺得他真實不做作，而有一種人無論他做什麼都會遭到一群人在背後嚼舌根，總要給他挑出點毛病來，老實說有時候他已經做得很完美了，但還是有人看不慣他。他心裡也明白有的人就在坐等看他笑話，期待他在眾目睽睽之下出糗。這種人就是招黑體質，可能是長得沒有親和力，也可能是情商比較低，不懂人情世故，因此得罪了不少人，那麼哪幾個星座中招了呢？

風評欠佳人緣差

代表星座：天蠍座、獅子座

天蠍座就是那種甭管做什麼都會招到別人討厭的人，總會有人在雞蛋裡挑骨頭，對他指指點點。有時候他並不想理會，只想專註於里的事，但那群人就像是趕不走的蒼蠅一樣，持續不斷地在他附近嗡嗡嗡，並且杜撰事實，損壞他在其他人眼裡的美好印象，導致他的風評欠佳，摸著良心說一句，天蠍座除了性格有些孤僻之外還真沒什麼壞心眼。

獅子座性格強勢，凡事都想跟別人爭輸贏，一點不妥協，遇到理解他的人就會欣賞他的能力，但要是遇到跟他氣場不和的人就會難以理解他，還會在背後誹謗他，這讓他很生氣，難免就跟他撕起來，結果就是他的人緣變得越來越差，只能說他性子太急，處理方式不當，坐實了壞人的位置，而讓真正的小人得意忘形。

情商低得罪人

代表星座：天秤座、雙魚座

天秤座智商再高，最終也毀在了自己的低情商上。他說話太直接，不懂得婉轉，心裡想什麼就說什麼，這就是他所謂的「真性情」，可不是每一個人都能理解他這樣的性格，所以就就會遭到部分人的反感，無形中得罪了一些人。雖然他本質是好的，並沒有傷害任何人的意思，但語言表達上欠缺考慮，導致別人對他有所誤會。

雙魚座要麼就不要說話，要麼就想好了再說，如果一直不改變急躁的性格的話，日後還會給自己惹出各種各樣的麻煩。雙魚座有時候太把自己當回事兒，而太看輕別人，所以說話不節制，以至於該說不該說的他都給說了，最後得罪了別人。對方要是沒什麼利益衝突的話還過得去，但有些人偏偏跟他有直接的利益牽扯，所以他有時候如履薄冰。

招黑體質的人一定要謹言慎行，別再問「為什麼有那麼多人不喜歡我」這種問題，多想想自己應該從哪裡改，應該怎樣改，要做一個內方外圓的人，少一些抱怨，多一些坦誠，相信自己的真誠一定可以感動對方，會讓對方明白自己的心意，努力去掉招黑體質，獲取好的人緣，畢竟多個朋友多條路，別老是得罪人，跟人結梁子，這樣對自己肯定是不利的。
