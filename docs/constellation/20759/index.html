情感測試：你更喜歡哪個自行車的款式？測你內心最渴望得到什麼

選擇這款自行車：說明你內心最渴望的是休息。不論是在工作還是生活中，你每天都是在操著各種各樣的心，而且對於父母和愛情中各種各樣的事情，你總是竭盡全力去做好，以此來證明你自己的能力，但是你的能力越大，你肩上的膽子也就會越來越重，因為你不想讓他們失望，所以你只能做到更好，因此這樣的你也會非常的疲累，內心非常渴望的就是能夠讓你放鬆一下，這個時候你可以放下一切東西去外面散散心，這樣會讓你有不一樣的收穫。

選擇這款自行車：說明你內心最渴望的就是自由。現在的生活，讓你感覺到有一些束縛，你想要去重回自己以前自由自在的生活，但是現在的生活讓你沒有辦法去追求自由，因為有著家庭的束縛，你只能腳踏實地，努力工作，去改變你們現在平淡的生活，因為你的內心總是不滿足於現狀的。

選擇這款自行車：說明你內心最渴望的就是改變自己目前的生活。你覺得現在的生活有一點平淡，你內心總是嚮往比較默契很刺激的事情，所以這樣日復一日的平凡生活讓你感到非常的疲累，你不想循規蹈矩，你想去做自己的事情，去追求自己想要的東西。

選擇這款自行車：說明你內心最渴望得到的是愛情。你是一個外表高冷而且比較堅強的人，不論在遇到什麼事情的時候，你總是自己扛，自己去完成，但是生活中你也不善於表達自己的感情，同時你也非常渴望有一個人在你的身邊關心和照顧你，因此這個時候你應該大膽一點，主動一點，這樣才能讓你收穫美好的愛情。
