繼續下拉，查看答案

A

對人對己高要求

你是個關注自己外表的人，在乎自己在別人心中的印象，所以對服裝、外貌都十分講究，希望吸引大家的注意。而且你是一個很有自信的人，對於任何事物要求都比較高。

B

衝動、沒耐性

你個性比較衝動，對事物沒耐心，做事容易三分鐘熱度。但是你也有率真簡單的一面，有一說一，從來不會刻意偽裝。你在遇到問題時會顯得焦躁，特別是遇到棘手的問題時，會不耐煩地走來走去，偶爾停下來用力地撓頭髮。

C

思維活躍，想法獨特

整天嘻嘻哈哈的你，看起來沒個正經，幽默樂觀的你還經常會製造一些笑料，是大家眼中的開心果哦。你的習慣動作就是做事情時會不自覺地抖腿或者轉筆，因為它們能給你帶來靈感。

D

憨厚老實，做事沉穩

你成熟幹練，情緒穩定，大家都認為你是個值得依賴的人，而你也常熱心助人。但與此同時，你還是個安於現狀的人，不願接受太大改變或冒險，比較容易留戀過去。

E

洒脫隨性，天真單純

你是個隨性的人，個性天真而單純，心思簡單。你的想像力也十分豐富，看影視作品或者小說的時候都會不自覺地代入到角色中。而朋友們喜歡的就是你的爽快和簡單。

華利學堂

心理學家榮格提出，內/外向的區別在於心理能量指向的方向。內向者的能量指向內部，他們對內心世界的興趣更大，更喜歡安靜和獨處。

所以，內向不等同於社交恐懼/焦慮。相反，他們還可能非常適合擔任領導者，只是覺得社交對他沒那麼重要。

社交的終極意義，其實是讓我們把時間留給對的人。
