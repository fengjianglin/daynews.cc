測試結果：

A、 你的抬杠指數 ★★★★

生活中，你是典型的抬杠精，每天不懟幾個人就心裡不爽。本身別人好好的跟你聊天，但說著說著你就習慣性的開始抬杠，不管是什麼話題，不管跟誰，你都可以杠上一嘴。你的挑釁會滿足自己的傲嬌的個性，只要抬杠成功，自己就會非常得意。其實，喜歡抬杠的人是缺乏安全感的，這會讓人覺得你沒有修養，也沒有禮貌。因此，還是少抬杠的好，多給自己留點後路，培養一點人際關係！

B、 你的抬杠指數 ★★★

抬杠對於你來說是一件很有趣的事，你也不是要誠心要抬杠，而是為了豐富自己的生活興趣。你並不只是和自己不喜歡的人抬杠，而是為了表達自己的觀點，喜歡說話罷了。你抬杠的時候不會一味的堅持自己，也不會爭的面紅耳赤，如果看到別人不太高興，你也不會堅持下去。你的抬杠還是會適可而止，懂得察言觀色的！

C、 你的抬杠指數 ★★

生活中，你並不會經常抬杠，你知道抬杠是不太禮貌的，是一種唱反調的表現。平時偶爾抬杠，你也只是跟自己不喜歡的人杠幾句，和自己關係很好的人，你一般會順著說，不會抬杠。你並不具備抬杠的本能，也不喜歡抬杠，在以後的日子裡，你會減少懟別人的機會，好好建立良好的人際關係！

D、 你的抬杠指數 ★

生活中你是一個比較乖的人，不會跟人抬杠，甚至爭執臉紅的事都很少發生。對於自己不喜歡的人，你會無視，自己喜歡的人會珍惜。生性善良的你會善待身邊的每個人，不會敵視他們，更不會可以抬杠。你偶爾會為了堅持自己的觀點而爭執幾句，但也會適可而止。人無完人，生活也並不完美，能夠和自己喜歡的人相知相遇不易，你會珍惜自己所擁有的！
