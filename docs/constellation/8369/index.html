周易64卦詳解

第一卦：乾 乾為天 乾上乾下 第一卦 乾 乾為天 乾上乾下 乾：qian ﹝粵﹞虔 凜凜皇者之象 自強不息之意 ( 困龍得水,上上卦 ) 卦辭 ：困龍得水好運交，不由喜氣上眉梢。一切謀望皆如意，向後時運漸 漸高。 推斷 ：詞訟和吉，病人痊癒，功名有成，求名大吉。 大象 ：天行剛健，自強不息

【周易64卦詳解】第二卦：坤 坤為地 坤上坤下 第二卦 坤 坤為地 坤上坤下 坤：kun ﹝粵﹞昆 兼容萬物之象 以柔制剛之意 ( 餓虎得食,上中卦 ) 卦辭 ：餓虎得食喜氣歡，求名應事主高遷。出門吉利行人到，是非口舌不 相干。 推斷 ：找人得見，走失有信，疾病見好，凡事皆順。 大象 ：大地承載萬物，以德服人

【周易64卦詳解】第三卦：屯 水雷屯 坎上震下 第三卦 屯 水雷屯 坎上震下 春木更新之象 艱難險阻之意 ( 亂絲無頭,下下卦 ) 卦辭 ：風刮亂絲不見頭，顛三倒四犯憂愁。慢行緩來頭有緒，急促反惹不自由。 推斷 ：疾病 未好，婚姻不巧，口舌瑣碎，做事顛倒。 大象 ：屯者難也，萬事欲進而不得進。

【周易64卦詳解】第四卦 蒙 山水蒙 艮上坎下 童蒙啟智之象 迷模摸索之意 ( 小鬼偷錢,中下卦 )卦辭 ：卦中氣象犯小耗，謀望求財枉徒勞。婚姻合夥有人破，交易出行犯嘮叨。 推斷 ：為人 仔細，掙錢費力，有心學好，小鬼偷去。 大象 ：蒙者，昏而無所見也，故宜「啟蒙」

【周易64卦詳解】第五卦：需 水天需 坎上乾下 第五卦 需 水天需 坎上乾下 密雲不雨之象 待機而動之意 ( 明珠出土,上上卦 ) 卦辭 ：明珠土埋日久深，無光無毫到如今。忽然大風吹去土，自然顯露又重新。 推斷 ：謀望 有成，婚姻最合，求財如意，也喜動挪。 大象 ：雲登天上而未雨，不能急進，等待時機之…

【周易64卦詳解】第六卦：訟 天水訟 乾上坎下 第六卦 訟 天水訟 乾上坎下 背道而馳之象 無端起訟之意( 二人爭路,平平卦 ) 卦辭 ：二人爭路未肯降，占著此卦費主張。交易出行有阻隔，生意合夥有平常。 推斷 ：月令 不濟，光生閑氣，好事難成，求財費力。 大象 ：乾天升於上，坎水降於下，相背而行而起

【周易64卦詳解】第七卦：師 地水師 坤上坎下 第七卦 師 地水師 坤上坎下 地勢臨淵之象 以寡服眾之意 ( 馬到成功,上平卦 ) 卦辭 ：將帥領旨去出征，騎著烈馬拉硬弓。百步穿楊去的准，箭射金錢喜氣生。 推斷 ：疾病 大好，走失能找，行人有信，百事湊巧。 大象 ：養兵聚眾，出師攻伐之象，彼此有傷

【周易64卦詳解】第八卦：比 水地比 坎上下坤 第八卦 比 水地比 坎上下坤 眾星拱月之象 安逸進取之意 ( 船得順風,上吉卦 ) 卦辭 ：順風行船撒起棚，上天又助一篷風。不用費力逍遙去，任意而行大亨通。 推斷 ：走失 可尋，見官有理，婚姻求名，保管恭喜。 大象 ：一陽統五陰，比鄰相親相輔，和樂之象

【周易64卦詳解】第九卦 小畜 風天小畜 巽上乾下 密雲無雨之象 蓄養實力之意 ( 密雲不雨,下平 卦 ) 卦辭 ：濃雲密排下雨難，盼望行人不見還。交易出行空費力，婚姻求謀是枉然。 推斷 ： 月令中平，忍耐從容，疾病口舌，交節安寧。 大象 ：蓄養之量有限，力不從心，故宜忍耐

【周易64卦詳解】第十卦：履 天澤履 乾上兌下 第十卦 履 天澤履 乾上兌下 如履虎尾之象 險中求勝之意 編碼 ：111011( 俊鳥出籠,中吉卦 ) 卦辭 ：俊鳥幸得出籠中，脫離災難顯威風。一朝得志凌雲去，東南西北任意行。 推斷 ：合夥 如意，遷移如心，買賣興旺，求財十分。 大象 ：柔弱遇剛強，欲行卻難行之象，難且危也
