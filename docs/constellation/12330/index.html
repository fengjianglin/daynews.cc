近日星象影響：

情感關係也追求更深入的互動，靈與肉的協調性，兩人共同的財務問題，與關係的忠貞度是這段時間人們所重視的。我們可能會體會到人性和真諦，和一些不良的情緒情感體驗，比如慾望焦灼或是情感深埋又無法合理表達。我們要選擇為什麼而戰，找到需要幫助的群體，「拯救」受苦的人們和流浪動物；勇敢地暴露自己的脆弱吧，誠實地說出自己的孤獨和不安。那些不畏懼你的陰影面的人，將是值得信賴的朋友和伴侶。有種能力能把想像中的種種轉化為現實，現實與理想絕妙聯結，人情味也在逐漸凸顯，而非拘泥於規則。這段時間你感情充沛，心思澎湃，有「胳膊向內拐「的傾向，雖然你更注重家庭，但容易因為情緒失控導致事態變化，你感到很難清醒地投入某件事，情緒會影響你的立場，讓你變得防禦與以進為退。將會激發更多浪漫的情愫，宿命戀情與美好的未來遐想也將會充斥著這段時間的關係。行事衝動與戾氣也將驟減，禮貌和權衡將會主導這段時間人們的行事風格。雙子座滿月，是我們溝通想法，總結一年計劃和方案的時候。可能你所得到的和學習到的已經達到一個瓶頸，你需要新的、更好的人或環境。之前的「壯士斷腕」已然換來新生，雖然依然路漫漫兮，卻至少已經有清晰的前景可以看見，辛勤勞動也開始逐漸變現。
