這四大星座沒錢的時候，是否會找朋友借錢，誰最要面子？

天天要聞
2019-12-22

俗話說得好：「月有陰晴圓缺，人有悲歡離合」，每個人都會有遇到困難的時候，但是每個人的應對方式都是不一樣的。有的人覺得朋友就是用來解決困難的，如果連錢都不願意借給自己，那麼就不算是朋友。但是還有的人卻會覺得，朋友之間一旦涉及到金錢，就會讓彼此的感情變得奇怪。一起看看這四大星座沒錢的時候，是否會找朋友借錢吧，你是這樣的嗎？

天秤座

天秤座是非常在乎面子的星座，他們即使是吃不起飯了，也不會找朋友借錢。在他們的心目中，現在的生活特別方便，如果沒錢了，還可以用花唄信用卡，沒有必要找朋友借錢。他們如果開口找別人借錢，就會覺得內心極其的不舒服，彷彿有一根刺卡在喉嚨。所以，天秤座無論如何，也不會找朋友借錢。

射手座

射手座從來不會找朋友借錢，除非到了走投無路的時候。他們的心理是非常奇怪的，即使有的時候找朋友借錢，也非常迅速地還錢了。可即便這樣，射手座還是會覺得有壓力，會覺得欠對方人情。所以在往後的日子當中，他們就會對對方更好，無論做什麼，只要對方開口了，射手座肯定會答應。他們是屬於有恩必報的類型，覺得對方既然肯將錢借給自己，那麼自己在對方的心目中，一定也是有所地位，他們會同等的對對方。

獅子座

獅子座的人比較大大咧咧，對於金錢也沒有太大的概念。當真的反應過來自己沒錢的時候，他們會選擇開口借錢，而且不會當一回事。在獅子座的心目中，大家都是朋友，互幫互助是理所當然的。不要覺得獅子座一副義正言辭的樣子，但其實他們確實有這個資本。他們本身就非常的仗義，在朋友需要幫助的時候，就會挺身而出，留下來的都是過命的交情，自然不會在乎這麼一點兒錢。

金牛座

金牛座懂得設身處地替別人考慮，更加知道談錢傷感情。他們很少會有讓自己捉襟見肘的情況，他們有多少貨多少，絕對不會超過自己的能力範圍。而且金牛座對金錢有自己的規劃，不會隨意地打破，即使沒錢了，也可以幾塊錢吃好幾天。

如果可以的話，盡量不要找朋友借錢，畢竟每個人在這個世界上生活都不容易，不要給別人製造負擔。
