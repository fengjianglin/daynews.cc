話癆一個，天天都在朋友圈抒發感慨的星座

2019-11-28 2,311條評論 4,166次閱讀 1555人點贊

有的人不喜歡發朋友圈，你會發現他們千百年都不會冒泡的那種，很少會把自己的心情或是自己的事情展現出來給別人看。而有的人則是相反的，天天都很多的感慨。今天就讓我們一起來看下，十二星座中哪些人話癆一個，天天在發朋友圈的呢？

白羊座：發給在乎的人看

白羊座是那種很聰明很有想法的人，他們平時很少表達自己，個性也比較含蓄內向，更不會到處泄露自己的信息。白羊座也很少在微信上發表看法，但是他們一旦在微信抒發感想，就意味著他們想讓在乎的人看到自己的內心，想讓對方注意到自己，這也是他們的一種方式表達情感。

獅子座：就是太閑太無聊了

獅子座就是太閑太無聊了，每次想要出門玩，但是又找不到可以陪同的人。讓獅子座自己一個人出去的話，他們是十分不樂意的。因此不知道要幹嘛的獅子座就會變得特別的憂愁善感，經常就是會感慨頗多，動不動的就能夠想出很多心靈雞湯出來，因此才會一直在發朋友圈。

天秤座：為了顯擺炫富

天秤座是那種很愛面子，很喜歡被人誇獎的人。他們平時就很喜歡發朋友圈，無論做什麼事情，他們都要在朋友圈炫耀一下。雖然有時候看著像抱怨，但其實更是花式炫耀。所以一旦天秤座在微信抒發感慨，他們其實是換種方式在顯擺，在炫耀自己。

雙魚座：單純做個記錄

雙魚座是那種很注重自己感受的人，特別是現代社會，他們覺得很多事情，很多心得如果不記錄，很快就會忘記。而現在的人記性又往往不是很好。所以雙魚座有時候在微信抒發感慨，更是一種記錄，記錄自己生活的點滴，這樣做十分方便，以後也能回憶。
