獅子座最挺誰的話？我們一起來看看

天天要聞
2019-12-10

獅子座可是個桀驁不馴的人，他們有膽有識，氣度非凡，不管在哪方面都力爭上遊，不敢落後於人。下面就一起來看讓獅子座乖乖聽話的星座都有哪些吧。

NO.1 天秤座

天秤座能夠讓獅子座乖乖聽話，天秤本身就非常會說話，為人圓融通透，擅長維護人際關係，跟不同的人也知道該如何打交道。而且天秤也不會讓人厭煩，能夠條理清晰地講明道理，將事情分辨清楚，能夠公正地處事，提出一些好的建議。儘管獅子座很強勢，但如果天秤座說得在理，又能維護好他的面子的話，也是會欣然接受的。天秤座就是有安撫別人的能力和魅力，獅子座對天秤還是很信服和欣賞的。

NO.2 雙子座

雙子座古靈精怪，天生就是腦筋活絡的人，常常能想出很多鬼點子，雖然獅子座比較霸道，但是跟雙子相比還是稍遜一籌的。雙子座最擅長的就是挖坑，搞一些小動作，他們也沒有什麼壞心思，只是喜歡戲弄別人罷了，也擅長用一些方法哄獅子座開心，不知不覺就會答應雙子的一些條件和想法了。如果平時的話，獅子座可能沒有那麼好說話，但是在雙子面前一切都能得到很好的解釋，雙子座欲擒故縱，立馬把獅子座降服了。

NO.3 天蠍座

天蠍座能夠讓獅子座乖乖聽話，他們同樣都是獨立有能力的人，自然能夠明白獅子內心的想法，即使相處不多，通過認真地觀察和了解也知道獅子座的性格和為人，喜歡什麼，想做什麼。雖然天蠍座平時給人冷冰冰的感覺，其實也是很通透的人，想要做成什麼就會用一定的計謀，獅子沒什麼心眼，在天蠍面前還是乖乖敗下陣來了，天蠍座最懂得利用人心，獅子座爭強好勝，有些時候只要欲擒故縱就行了。
