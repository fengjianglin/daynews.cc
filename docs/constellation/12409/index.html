2020年你將苦盡甘來—木土合相（2019年底序言篇）

2019-12-10 3,044條評論 4,167次閱讀 886人點贊

在剛剛過去的一周裡面，占星學的世界裡被木星進摩羯刷滿了屏，可是這才是剛剛開始，接下來木星很快從摩羯座走近水瓶座♒️，並與土星在水瓶座♒️（0℃29’）精準合相（2020年12月21日），這是從土元素摩羯座♑️轉化至風元素的水瓶座的一年，因為是零度代表著一切的未知的開始，探索的渴望，代表著一個新紀元的開始，而這個時代大概持續220年，這樣周期性的運作所要詮釋的是代表的是社會秩序，結構（土星）和社會價值觀的演化過程。

在占星學裡面，木星代表的是吉星，而土星代表著限制與困難，但也意味著收穫，他們的合相則意味著一種潛在的卓越的可能性。

希臘神話的原型

土星（Saturn）,也就是希臘神話中的克羅諾斯（Kronos），他開創了一個新的王朝，一則神諭告訴他將來他的孩子們會取代他的地位，因此他把孩子一個一個的吃掉，而宙斯（木星）是他唯一一個沒有被吃掉的孩子，而最終也是宙斯開創了新的紀元，取代了父親的地位，成為奧林匹克山新的統治者。

以土元素為主導的木土周期都發生了什麼？

這是資本主義經濟快速發展的周期，自2000年以來，房地產經濟的崛起，高鐵建立的飛躍大發展的時期，物質和實用主義盛行的周期。人民更關注的是自身所積累和具有的財富，車子，房子，票子成了最普世的價值觀，還記得二十年前一部流行的《蝸居》深刻的演繹的淋淋盡致。

借用英國作家查爾斯•狄更斯《雙城記》的原文所詮釋的那樣：

這是一個最好的時代，也是一個最壞的時代；這是一個智慧的年代，這是一個愚蠢的年代；這是一個信任的時期，這是一個懷疑的時期。

這是一個光明的季節，這是一個黑暗的季節；這是希望之春，這是失望之冬；人們面前應有盡有，人們面前一無所有；人們正踏上天堂之路，人們正走向地獄之門。

即將以風元素為主導木土周期可能會發生什麼？

如果土代表把願景變成產物，風元素則負責將其通過聲音，傳播出去，它包括對生產的東西做出意義上的詮釋以及對事情提出計劃與方案。那從合相水瓶座來看，2020年我們國家將全面實現小康生活，人工智慧，5G業務，探索太空，高科技研發都將值得期待。大家將會看到更加自由，平等，獨立，理性的時代。

2020年你準備好了嗎？
