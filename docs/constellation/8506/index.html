臨卦九二，咸臨，吉，無不利，陳平未按劉邦之意而放樊噲

2019-11-17 3,057條評論 4,530次閱讀 3083人點贊

臨卦，九二。咸臨，吉，無不利。《象》曰:咸臨，吉無不利，未順命也。

意思就是，九二，一同到來，吉祥。沒有什麼不適宜的事。《象》:一同到來，吉祥沒有不適宜的，這是因為不是以順從命令而做到的。

臨卦中兩個陽爻就是九二和初九。他們相連，也就是一同到來，是陽氣始生。是為咸臨。

臨卦本來就有到來、壯大之意，此時初九、九二同時到來，他們又是陽爻，是陽氣上升的氣勢。其勢如此，則可吉祥，沒有不適宜的。

九二在下卦的中位，臨之下卦為兌，兌為悅，下卦中位的九二最能代表其兌之意。九二和六五正應，六五在上卦坤中，坤為順。

然九二和六五雖正應，卻都是失位，九二是陽爻居柔位，六五是陰爻居陽位。九二的二爻位置是大夫位，六五的五爻位是天子位。換而言之就是九二為臣，陽爻剛正誠信。六五為君，陰爻柔順。

君臣之間本來應該是君剛臣順，就是因為九二和六五都失位，所以就表現為他們意見不同，在做事的時候九二沒有按著六五的意思去做。不過雖然如此，他們終究是相應的，所以結果並沒有什麼不適宜的。這也就是九二不是以順從命令而做到底的。

故而為「咸臨，吉無不利，未順命也。」

這裡就是說君王教給臣子的事，而臣子並未按照君王的旨意來做，不過雖然如此，這個臣子的結果卻是吉祥的，就如劉邦臨終之前讓陳平斬殺樊噲的事。

劉邦讓陳平斬殺樊噲

樊噲是西漢的開國功臣，其勇武過人，在鴻門宴上幫劉邦解過圍，他的勇武是曾得到西楚霸王項羽讚揚的。他作戰勇敢，為大漢的建立立下汗馬功勞。

樊噲不僅追隨劉邦較早，對劉邦忠心耿耿，而且他還是呂后的妹夫。不過就算如此，劉邦在臨終前還是曾下令讓陳平、周勃去殺了樊噲的。

那時劉邦平定叛亂回來卻得到了重病，而這時卻又接到燕王盧綰叛變。於是就讓樊噲帶兵大軍去征討樊噲。

然而樊噲走後，因為樊噲是呂后的妹夫，呂后又干預朝政讓劉邦很不滿，這時就有人向劉邦進言，說樊噲和呂后串通，希望皇上早做準備。

劉邦聽到這些，知道事情的嚴重的性。因為此時樊噲有兵，又是呂后的妹夫，而若劉邦一旦駕崩，掌權的就是呂后，那時呂后若何樊噲聯手，將會對大漢帶來威脅。

想到這裡，雖然樊噲對他忠心耿耿，但是劉邦還是決定殺了樊噲，於是就和陳平商量後，讓陳平和周勃一起去換掉樊噲然後殺之。

然而陳平心眼多，是個聰明之人，他知道樊噲是劉邦的心腹愛將，又是呂后的妹夫，此時劉邦已經病重，將來掌權的一定是呂后，若真的把樊噲殺了，將來自己肯定吃不了兜著走，自己也不會有好果子。

於是陳平和周勃就先把樊噲抓起來並沒有馬上殺死，而是把他押送京城，讓皇帝自己決定是殺是留。這樣就跟他們沒有關係了。

當押送樊噲進京的路上，劉邦因病去世，呂后則順理成章的掌握了大權。這時陳平慶幸沒有殺了樊噲，不然呂后姐妹是不會放過陳平的。

當呂后姐妹知道樊噲還活著的時候，她們也終於放下心來，而陳平也得以免於災難。這才有了後來陳平封為丞相，而且陳平、周勃後來又得以平定諸呂之亂，推尊漢文帝劉恆為帝。
