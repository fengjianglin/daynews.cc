不要賣弄心機，這四個星座很容易看穿人心

2019-11-17 137條評論 4,351次閱讀 2289人點贊

巨蟹座的人一直都是很善良敏感的，他們很少去表達自己的想法，都是安安靜靜不參與任何的爭論，因為巨蟹座很看重別人對他的評價，總是希望可以留給其他人一個好的印象。而這樣的巨蟹座就會特別留意其他人的舉動了，也就練就了察言觀色的能力，只要對方有任何的行為異常，巨蟹座都可以很快地發現。所以不要看巨蟹座對人很好就以為他們好欺負，你的任何行為舉止巨蟹座都看在眼裡，但巨蟹座不會去拆穿你，善良的他們不想去惹什麼事端，只要不是針對他們的，巨蟹座也不會去出頭指責什麼。

雙子座是特別善於觀察的星座，非常喜歡交際，所以朋友很多，他們很聰明，可以在很短的時間內摸頭對方的習慣，交往的過程中投其所好，不管發生了什麼樣的事情，雙子座都可以很快地想出解決辦法。在日常生活的交流中，他們可以通過表情動作來判斷對方的狀態，然後提前想好對應策略。雙子座心思細膩，很容易就可以洞悉對方的想法，當我們在面對他們的時候，最好不用有什麼壞的心思，因為你的小伎倆，可能已經被雙子座拆穿了。

處女座最擅長的就是細節，做人做事總是深究背後的本質，永遠都是追求事物本來的面目，處女座很容易從一個人的表現判斷出對方的想法。觀察細膩和縝密分析，通過一些語言語氣，處女座就可以看懂對方隱含的意思，以及對方的善惡。當然他們不是那種到處亂說的大舌頭，處女座對事物可以抽絲剝繭，任何一絲表象都要深挖到底，只要發現任何可疑的現象，他們就會順藤摸瓜，直到發現你的真正心思。

天蠍座給人的感覺就是冷淡，不容易接近，總是很神秘很少和人接觸，喜歡自己一個獨處。不要看他們溝通得很少，其實他們大多數時間都是在觀察，所有的事情他們總是作為旁觀者，作為一個局外人，但是對方的一舉一動，任何的小動作小心思天蠍座都看在眼裡。天蠍座很善於揣測人心，對方心裡想的他們一眼就可以識破，當然只要不在他面前耍弄心機，天蠍座會一直作為一個旁觀者，不會參與和干預，就當是看戲一樣，但是如果你敢針對他，那絕對會得到天蠍座更厲害的報復。
