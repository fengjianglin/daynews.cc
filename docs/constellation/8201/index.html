看似堅強的天蠍，一旦掉眼淚了，會比誰都痛苦不堪！天蠍也會崩潰

2019-11-29 64條評論 1,388次閱讀 392人點贊

　　在這個的環境里、這段關係里、這龐大的壓力里，這不公平的世界裡，有一群人，是最不愛服輸、也不肯認輸的，再累也要逞強過來，那就是天蠍座！

原因之一：不喜歡認輸

　　天蠍會掉淚是因為這個環境的冷酷，適者生存，不適者淘汰，好強、好勝的天蠍座，當然不肯接受認輸，所以不管遇到多困難的事，天蠍都會逞強，也要熬過那些日子，即便壓力有多大，天蠍也會拼了命咬著牙，而天蠍的自尊心也不允許自己就這樣自暴自棄的過生活，所以不管天塌下來，天蠍也要硬扛起來，也是天蠍這樣的愛逞強，天蠍才會累到掉眼淚。

原因之二：不被認同

　　天蠍會掉淚，是因為自己的努力，沒有得到好的認同，別人看不見天蠍花了多少心血、多少努力、多少時間、多少睡眠，天蠍才會有這些成果，只要一句不認同，就等於打翻了天蠍所有心血，所以天蠍就會掉淚，氣到讓天蠍，會覺得自己又何必這麼努力呢？沒有好的結果，對天蠍來說，打擊太大了。

原因之三：付出了太多

　　天蠍會掉淚，是因為天蠍不管對待什麼人、什麼事，天蠍都會用盡心力去付出，會不記所有的代價去付出的，尤其是在感情里，往往付出最多的天蠍，那些付出會容易被人踩進在腳底下的，因為現在的人容易習慣，就會變得不珍惜天蠍的付出，再加上現在的人變得很自私，就會覺得天蠍做的都是應該的，拿不出真心回報給天蠍座，天蠍就會愛人愛得很累，漸漸地天蠍也會覺得越來越不愛了，那些付出，就和眼淚一樣難收回來。

原因之四：藏了太久

　　逞強太久、努力太久、付出太久、累積了太久，心裡裝了太多、太多，複雜又負面的情緒，天蠍在怎麼自尊心高，藏了太久、太久，堅強很久的天蠍最後還是會受不了哭出來的。很愛天蠍的人，還請你們多心疼、心疼天蠍座，天蠍只是比人用心努力多了一點，只是比人付出多了一點，為的只是認同跟好的回報而已，所以請好好珍惜天蠍付出的那些！
