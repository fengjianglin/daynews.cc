最不愛欠別人，人情跟錢財都及時歸還的四大星座

2019-11-25 581條評論 1,736次閱讀 397人點贊

最不愛欠別人，人情跟錢財都及時歸還的四大星座

獅子座

天生傲氣，不喜歡被人看不起的獅子座都是很愛面子的，不欠別人才是最好的，欠了什麼東西都是要及時還回去的，不然就沒面子啦。而且不到萬不得已獅子座絕對不會跟別人欠錢，去貸款也好，抵押也好，對他們來說都比欠別人的要舒服多了。獅子座很怕自己出去借錢別人會在背後嘴碎，各種猜測獅子座是不是遇到困難了，沒什麼本事了，然後從心底可憐他們，這是獅子座最害怕也最討厭的事情。

巨蟹座

負擔不起的東西沒有必要去強撐面子，因此巨蟹座一般不會去眼饞自己沒有能力承受的東西，即使是有時候衝動消費了也會自己買單，不會因此就去借錢，一般萬不得已有了要用錢又沒積蓄的時候也會努力掙錢及時歸還。不想讓自己活得很累的他們，有事情會先向親人求助，這樣心理負擔會小一點。一旦借了錢也會向家人先索取還給朋友，欠了人情也是一定緊盯時機能還就還了。

金牛座

如果要有一筆大的花銷的時候，金牛座都會提前規劃好，並不會臨到頭來發現自己沒錢又去借錢，對金錢有一種掌控欲的他們，只會在自己兜里有了足夠支撐很長一段時間的開銷的積蓄之後才考慮額外的支出。並且有了富餘的資金之後會存起來而不是大手大腳。要是他們借錢了也會很快周轉好，能不麻煩別人就不麻煩別人也不會怎麼欠別人熱情。

水瓶座

生來獨立自主的水瓶座，不在意獨來獨往也不在意是否孤獨，喜歡跟大家保持一種君子之交淡如水的氛圍，不想難堪也不想太過於深交。就算是深交了也希望大家還是舒服為主，不想被羈絆。不愛欠人情也不愛欠錢的他們如果真有急事要借錢，那肯定是省吃儉用甚至不吃飯也要先把錢還上，否則水瓶座們真的會心裡很難受，覺得人際交往都不舒服了。
