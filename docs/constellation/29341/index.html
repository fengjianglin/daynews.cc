前言

現在距離2020年已經不到10天的時間，回顧這過去的一年，有多少人經歷了前所未有的壓力與責任。為錢而奮鬥，飽受生活的磨礪，可是即便如此，卻依舊對未來抱有希望。其實，許多人自己都知道，除了努力向前，我們別無選擇。

本周，摩羯座的新月日食，將會敦促我們快點去制定未來一年的計劃與奮鬥的目標。新的方向即將確立，意想不到的驚喜就在本周上演，你是否已經做好準備了呢？

本周星象運勢

必看篇

25日，太陽三合天王星，我們總是被各種突發的意外所包圍。25日前後，這樣的情況更是明顯。不過，往好的方面來看，這也將是一個適合科研突破、考古發現、網路科技獲得進步的時機。我們將會對未來科技抱有更濃郁的興趣。對於希望，我們會都有著不一樣的態度和看法。這是一個適合發布變革、更新、整改的位置。

就企業而言，我們會看到一些網站的更新與改版。曾經的店鋪，如今換了個名字重新開張。各種跨界領域的合作、併購、收購、重組都會成為行業內的新風向。因此，這也將大大影響到投資市場的變動。

這幾天，我們總是會想要嘗試著去突破自己，改變自己。小到更換一個髮型，大到突然決定要自己創業。如果，你正好有這個念頭，不要被自己的突發奇想所驚訝。敢於嘗試，一直是天王星給我們帶來的動力。同時，我們也會在這一天看到許多人會對自己或周圍進行一個清理。天王星的疏離，會讓我們切割或是分離那些無用的東西。洗牙、手術切割、雙眼皮手術、拔牙等都容易出現。

雖然，變革，為我們創造了更多的可能性。但是，在情感方面，我們容易因為突然的情況，而不得不增加異地戀的風險。有伴侶的人，會因為對方的臨時出差、回家探親等原因而造成短暫的分離。不過，好在，這樣的情況都只是暫時的，所以不用太過擔心。

資金方面，25日前後，如果你有收到任何對你工作或是財務方面的信息，都不用感到驚訝。太陽的出現，會讓許多人獲得一筆額外的資金。尤其是對於那些曾經你忽略或是不曾關注的資金來源。或許，這筆資金的金額並不多，但可以確定，這筆錢並不在你的預計範圍內。所以，本周不要錯過任何的資金信息。

26日，摩羯座的日食新月，將會為摩羯座的朋友帶來全新的機遇。如果你正在籌備某個重大的計劃，或與房屋、地產、事業、人生規劃有關，那麼在未來的6個月里，你都有機會獲得成功。許多人會在這天前後，突然下定決心專心做某件事情。

從事件方面來看，這是一個容易出現跳槽、離職、自我尋求創業的機會。許多人會為了創建自己的事業王國而不得不從基礎做起，這段時間，我們將會看到大規模的重組、人員調動等情況。摩羯座，是一個收縮的星座，即便有木星加持，可是，我們仍然會看到裁員、多人競爭一個崗位的情況。

摩羯座同時也代表著地質、土地、房屋。因此，我們也需要注意，在新月前後，各種火山爆發、地震、山體滑坡、架構崩塌、企業倒閉等信息都需要我們注意。這是一個去舊迎新的位置，為此，重新建設、過去的一些事情再次需要重新審視，還一個公道。

28日，太陽合相木星，往日的努力獲得回報。各種年度報表的公布、年會、獎金、頒獎典禮、獎狀的發布都在向他人訴說著你過去的努力。即便你是公司的小小支援，你也有需要發表或是編輯未來工作計劃的情況。

29日，水星正式進入摩羯座，空氣中的溫度將再次創下新低，各種感冒引起的呼吸道慢性疾病複發更是需要注意。這段時間，我們看待問題的方式將會變得更加嚴謹與冷靜。我們善於求證、尋找證據來確定自己看到、聽到、知道的是正確的。不但如此，就連日常看到、聽到的信息和內容都充滿著專業性。我們會被專業的電視、信息、資料所吸引。

這也是一個適合簽訂長期合約的時機，許多人會為了自己的目的而許下承諾。合約的簽訂、求婚、結婚等，都讓我們變得更具有責任感。一些文件的審核、審批、規章制度需要我們去遵守。
