選擇轉運星A的人

2020年你將喜得貴人相助，如果你現在陷入困境，明年會有貴人扶持你，幫助你走出困境。如果你現在事業很順利，明年你會更上一層樓，仕途鴻泰，步步高升，加官進爵，平步青雲。

如果你經商，今年虧了，明年會轉虧為盈，往後生意蒸蒸日上越做越大，一本萬利，財運廣通，財源廣進，財氣廣聚，財力廣大！

選擇轉運星B的人

2020年你將走桃花運，沒結婚的脫單，結婚的更恩愛，千里姻緣一線牽，幸福美滿。2020愛你愛你，你愛的人會更愛你，你懂的人會更懂你。

2020年你將喜得貴子，子來財來家興旺，夫妻恩愛家和悅，兒女孝順都成才，父母安康享天倫。

選擇轉運星C的人

2020年你將會行好運，即使你現在過得再艱難，無論你現在運氣有多差，明年都會時來運轉一切好起來。今年再大的事，到了明年都是小事，都不是事！

2020年你將一順百順，無論愛情、親情還是友情，都會順心順意；無論福運、財運還是官運，都會順風順水！
