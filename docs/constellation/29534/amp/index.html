當汽車也搞起了星座學……（下篇）

天天要聞
2019-12-21

假如車圈的汽車們也搞起了星座學，那麼大家覺得的與每個星座相對應的會是什麼樣的車呢？上期Zoo長和大家分享了水瓶座、雙魚座、白羊座、金牛座、雙子組以及巨蟹座，今天咱們繼續討論討論剩下的六個！

獅子座：路虎攬勝

獅子座最大的特點就是自信強勢、熱情似火。定位於中大型SUV 的路虎攬勝，憑藉大尺寸的車身、硬朗的腰線、霸氣的進氣格柵以及極具科技感的內飾配置和優秀的動力表現，一度受到高端市場的火熱追捧，就如同崖邊睥睨眾生的雄獅一般。

處女座——梅賽德斯賓士C級

處女座最大的特點就是注重細節、完美主義。而在眾多車型中，賓士C級無論是外觀造型還是內飾設計對細節的重視都是毋庸置疑的，乘坐體驗也是極盡舒適感。與處女座自帶的精緻氣息可謂是完美融合。

天秤座——沃爾沃XC60

天秤座最大的特點就是冷靜理性、追求平衡。而以安全性能著稱的沃爾沃XC60絕對是典型的天秤座。而除了堪為標杆的安全性能之外，沃爾沃XC60強勁的動力、理想的油耗、實用的空間以及穩重的外觀，都更為其增添了幾分平衡理性的味道。

天蠍座——道奇挑戰者

天蠍座最大的特點就是桀驁不馴、堅韌執著。說起桀驁這一點，大家是不是立刻就想到了《速度與激情》系列電影中道奇挑戰者，就是男主范·迪塞爾那款祖傳肌肉跑車。無論是滿富攻擊性的外觀造型，還是深不可測的動力儲備，都像極了難以馴服的天蠍座。

射手座——JEEP牧馬人

射手座最大的特點就是熱愛自由、探險獵奇。自然也要一台擁有強大越野性能對的車才更適合這個星座了。而要論越野性能的話，JEEP牧馬人必然是上佳之選，搭配強大的改造潛力以及豐富的改裝件，無論何種複雜的地形都能輕鬆征服，真正的放蕩不羈愛自由！

摩羯座——凱迪拉克ATSL

摩羯座最大的特點就是執著好勝、實力低調。簡單來說，就像俗語中說的”扮豬吃老虎”一般。凱迪拉克ATSL作為一款中級運動轎車，卻有著遠超同級車的強大動力輸出，百公里加速度僅為6.2秒。車身外觀獨特的鑽石切割工藝，也是打造了一個低調卻更具實力的強者形象！

總結一下，每個人對於不同星座的性格解讀都是各有差異的，對每款汽車的理解也是大相徑庭。所以，Zoo長所列舉的十二款車型也只是在Zoo長看來比較貼合對應星座的車型！那麼屏幕面前的你又覺得哪款車更符合這十二個星座的設定呢？
