做事果斷，翻臉無情，這可能也是天蠍座神秘性的來源之一

2019-12-19 215條評論 3,047次閱讀 777人點贊

首先，星座只是一個人以何種方式形象表達自己，和冷漠與否並無直接聯繫。你可以覺得雙子花心，天秤虛偽，水瓶冷漠，也可以覺得白羊無腦，獅子心硬，射手浪。還有巨蟹敏感，天蠍病嬌，雙魚矯情，或是金牛摳門，處女煩人，摩羯心冷為什麼你會看到這樣他們？可能因為你遇到的不是好人，他們發揮出的是自身星座中負面的能量。如果你跟天蠍座是泛泛之交，或者是熟識但不深入的關係，那麼天蠍給人的印象則是非常好的，跟冷漠絲毫不沾邊。她們很優秀、很大方、很勇敢、很真實、生活非常的多姿多彩等等如果跟天蠍座是很熟悉親近的朋友關係，則分兩種情況。

一種是很受天蠍座喜歡，認可的。那麼你會擁有一個特別完美的天蠍座朋友。你會從天蠍座身上學習到特別多優秀的品質；你們相處的又輕鬆又快樂；你們對於彼此而言是給予力量與勇氣的存在；天蠍座甚至會為了你融化自己的稜角、尖銳、固執，天蠍座會跟隨你的腳步、規劃，會很願意聽你的話，並且絲毫不會覺得自己在妥協、在將就。能到這個程度，就足以證明天蠍座是非常喜歡和認可你的了。因為天蠍座內心其實有一部分是很自我、很強勢的。如果這樣的她，會願意完全跟隨你，依靠你，就代表她非常喜歡你，內在堅硬的部分在面對你時，已經完全柔軟化了。

另一種是，你跟天蠍座是很熟悉的朋友，天蠍不能說完全不喜歡你，但是值得肯定的是她內心深處並不接納你，認可你，而你又恰巧是一個能敏銳的感受到的人，那麼恭喜你，你就成為了能見識到天蠍座冷漠的那一部分人了。你會發現，天蠍很理性自我，甚至可以說是自私，如果遇到什麼事情，她們能立即做出有利於自己的選擇，而絲毫不會考慮別人。完全不會心軟、糾結。你會發現，天蠍座很「雙標」，明明你和另一個人都是天蠍的朋友，天蠍對你們倆看起來差不多一樣好，但是天蠍在跟你相處時，你能感受到她身上的尖銳、內在的強勢和固執己見。

然後，你會發現，當你們發生矛盾時，你歇斯底里，沉浸在痛苦裡無法自拔，生活全都被情緒埋沒打亂的時候，天蠍仍然能正常的該怎樣還怎樣。甚至在你痛苦難過的時候，在你身心最脆弱的時候，你會發現，天蠍作為你所謂最好的朋友，她仍然能沉浸在自己的快樂里，絲毫不加掩飾。你就像一個素不相識的陌生人。這個時候，也會是你對天蠍的冷漠絕望，對你們之間的關係絕望的時候，你會發覺你所有的付出，你的感情，在她那裡好像一文不值。

所以說來說去，天蠍不在乎你，你對她再付出真心也沒用。當你感覺到天蠍不怎麼喜歡你時，做事不會考慮到你時，盡量不要讓自己陷入試圖感動她的困局中。這種不對等的關係不會是長久、良性的關係的。把你的真心留給互相喜歡、互相珍惜的朋友吧。傳言外冷內熱無比記仇，實則外冷內冷心冷。做事果斷，翻臉無情。前一秒對你熱情似火，下一秒就被打入冷庫，差別可能就是一句話揭示的三觀不合。

這可能也是天蠍座神秘性的來源之一，因為難以捉摸的性格。對越熟悉的人越放肆，開心就粘著，不開心就冷著；反倒是對不熟悉的人客氣周到小心翼翼的。天蠍座的冷主要來自理性，過於理性的生活和思考，讓他們活得太累也太明白，很多時候不是刻意保持距離而是覺得有距離有秘密對彼此更好一些。

最後，星座這只是一種心理暗示，這將會形成一種悖論：你信則准，它給了你積極的心理暗示，不信也准，它給了你消極的心理暗示！抱任何一種觀點再改變觀點你會陷入這種循環中。每個星座間都有著被文字遊戲包裹的一致。舉個不含任何星座，但關於星座的文字遊戲的例子：並不是你所相信星座決定了你，而是你被相信的星座所左右。請問這句話的意思究竟是星座影響了你，還是由於你相信星座而改變？請你仔細思考一下，再對比一下之前的星座解析，你就會豁然開朗。
