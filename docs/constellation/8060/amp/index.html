今日星象大吉，留下你的願望，七日之內收到好消息

太陽雙子座或土星水瓶座男人——外遇騷包男

他對你的愛，就像是狂亂刮過的一陣颱風，短暫急促。在你驚魂未定時，扔給你一枚激情的炸彈逼你就範。等你暈了後，卻又對公司新來的女同事幻想; 明天對隔壁的鄰居小妹暗送秋波，後天又和女網友神交……根本就不拿老婆當回事。想要下班見到他在家，不是等月圓，就是等過年。他不會戀家，只會戀花。被這 陣大風襲擊過後，苦的只是你這個災民，所以星婆奉勸你若想和此男建立家庭，必須要慎重才行。

上升天秤座或木星射手座男人——歹衰無賴男

他不是一個愛情長跑選手，覺得能給你一個名分，就算很對得起你。想讓他盡丈夫的義務，他也會努力地吃你做的飯，穿你洗的衣裳，用你給的錢花!不管你有任何意見，他也絕不會聽進去的。生氣離不離婚也無關緊要，因為在他眼中，愛上他的你是個笨女人，就該被他吃到死。而他死皮賴臉就想賴死你，你就只能 自認倒霉認命自己有眼無珠怎麽會喜歡這種不要臉的無賴啦。
