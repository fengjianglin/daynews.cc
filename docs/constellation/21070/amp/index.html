15. 君子。外在喜歡錶現的彬彬有禮，即使心裡不耐煩也會客客氣氣，比如接到推銷電話也會耐心找機會掛掉，這點容易被人利用。永遠都是一副一本正經的樣子，被人挑逗或在需要嘻嘻哈哈的場合會裝傻充愣（因為不知道怎麼接），插科打諢之類的事情天秤永遠不會去做（也沒這個天份），臨場應變能力極差，所以天秤不善於討人歡心但是非常值得信賴和託付。對喜歡的人不會計較付出（因為喜歡的人必定也是公平的人），對看不慣的人態度很冷淡，但仍然不會拒絕對方的請求。內在則有很多的準則和底線，觸碰底線的事情獲利再大也不會去做。天秤有時也會覺得自己虛偽，大多是在想法與做法發生矛盾的時候，然而君子論跡不論心，一個人裝了一輩子的君子，那他就是一個君子。

16. 心地善良。極容易被弱者或悲劇打動，即使被騙也會幫助弱者。即便自認為理性客觀，在關鍵時刻即使是對惡人也會手下留情。

17. 孤獨。看過有句話形容天秤很貼切，「繁華的最高境界是孤獨」。天秤深知沒有人可以真正理解自己的內心，即便很喜歡張羅朋友聚會，但是聚到一起後反而變成了最安靜的那個。即便喜歡和別人一起玩，天秤也享受一個人的感覺，在兩性關係里天秤也同樣需要自己的空間。什麼樣的瑣事壓力天秤都可以自己默默承受，很少會跟任何人傾訴，什麼樣的苦惱天秤自己都可以解開，不需要別人的勸慰。自己有什麼高興的事情，天秤也不一定會積極告訴別人，即使告訴別人也多是為了讓對方高興。

18. 有藝術細胞/鑒賞能力。天秤或多或少都會有欣賞藝術的能力，可能和注重細節和善於思考有關係。天秤很會把東西延伸從小延伸到很大，思考其中的聯繫，總結事物規律。對藝術會有自己的理解和感悟。如果天秤座推薦你書，電影，音樂之類的，絕對是值得一看的。

19. 興趣廣泛。天秤對很多東西都有好奇心和求知慾，當然恐怖故事裡常出現的不讓打開的房間除外，在天秤心裡承諾和規則比好奇心重要的多，所以天秤也是很好的保密者。對感興趣的東西會去查資料研究它的由來，喜歡冷知識。所以天秤座很健談（當有談話動力或慾望時，平時還是喜歡安靜和沉默），因為對方感興趣的東西自己多多少少都了解過一些。

20. 低調。天秤座的人不會炫耀自己的東西，即使內心極度渴望得到別人的賞識和認可。比如當一個人在大家面前炫耀自己取得的某方面成就時，即便自己的成就比他高天秤也是不會說出來的，但是心裡會希望有個機會可以讓自己「不得不」說出來。

21. 心態平和。由於喜歡觀察和思考，事情也看得透，對發生的事就會接受度比較高。了解哪些是真正珍貴的東西，內心也就不會被無意義的慾望扭曲。也就會經常覺得人的需求並非一定如馬斯洛金字塔那般，我覺得有些東西是與生俱來的，是會凌駕於現實之上的。
