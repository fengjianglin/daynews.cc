每日星座運勢（12月1日）：摩羯座貴人運很強，去勇敢解決困難吧

2019-12-01 461條評論 2,025次閱讀 560人點贊

摩羯座

今天的幸運色是橙色。只要你今天想找人幫忙，就會有厲害的人出手幫你。你的工作嗅覺很敏銳，知道什麼事情，找什麼人是可以馬上解決難題的。今天你的貴人運很強！

幸運色：橙色。

適合：將難題拋出。

不適合：獨自承擔。

天蠍座

今天的幸運色是藍色。今天你的財運很強。如果你覺得自己符合加薪的資格，今天去找主管討論加薪、申請費用，有機會能夠得到老闆的認可。今天你對風險的判斷能力也很強，可以做自己重大決策上的判斷。

幸運色：藍色。

適合：爭取自己想要的。

不適合：過於謙虛。

處女座

今天的幸運色是白色。今天你的頭腦清晰敏銳，你可以開發和拓寬自己的工作領域，找不同的管理者去聊聊你的理念，尋求更多相同理念的領域，發展你的工作和專業。

幸運色：白色。

適合：找尋機會。

不適合：浪費時間。

雙子座

今天的幸運色是綠色。今天如果你覺得對未來焦慮，有一種不確定感，你需要讓自己停止思考。去感受一下你自己，什麼對你來說才是最舒服、最適合的，而不要去羅列對你最有利的條件。

幸運色：綠色。

適合：正視自己真實想法。

不適合：過於利益取向。

射手座

今天的幸運色是深藍色。對自己沒有信心也是正常的事，不要任何事物都要搭配「是否有意義」這樣的思考，這樣你的心始終沒有真正休息到，怎麼會不累呢？暫時隨遇而安，能安撫你的情緒。

幸運色：深藍色。

適合：隨遇而安。

不適合：想太多。

雙魚座

今天的幸運色是淺綠色。害怕擔心的事情成真嗎？有些事如果你覺得不得不做，那就不要擔心了。你問問自己：如果我擔心，就是就此停止，不去做嗎？如果答案是「不會」的話，就不要自我矛盾了。

幸運色：淺綠色。

適合：放寬心。

不適合：過度擔心。

白羊座

今天的幸運色是淺紫色。今天不要太過心急，很多事情可能峰迴路轉。你可能以為是發生了一件不好的事，譬如說原來說好的合作泡湯了，但其實這件事情的結束是幫助你開啟另外一個新的可能，所以別把事情先想得太糟了。

幸運色：淺紫色。

適合：靜觀其變。

不適合：先入為主。

金牛座

今天的幸運色是金色。今天你的另一半很融入你的朋友圈，他也很支持你去做你想要發展的群體生活，快去謝謝他的用心和為你的付出吧。

幸運色：金色。

適合：多跟情人說心裡話。

不適合：只考慮自己。

巨蟹座

今天的幸運色是鵝黃色。今天你的信念，能幫你帶來不一定的可能，你只需要給自己多點信心，就有機會發展出不一樣的可能。

幸運色：鵝黃色。

適合：堅持信念。

不適合：不相信自己。

獅子座

今天的幸運色是祖母綠。今天你覺得好像很多事情不順利，可能是因為你急著想要達成某件事情，但有可能現在還不是對的時機，多一點耐心去累積吧。

幸運色：祖母綠。

適合：多點耐心。

不適合：心浮氣躁。

天秤座

今天的幸運色是咖啡色。今天你不僅能夠維持高質量的生活狀態，也能有賺錢的機會，好好利用今天的好運氣做想做的事情吧。

幸運色：咖啡色。

適合：放鬆心情。

不適合：精神太緊繃。

水瓶座

今天的幸運色是墨綠色。今天很容易發揮自己的潛能，可能是某些狀況刺激到你，或是某些信念讓你有新想法，總之今天會發現不一樣的自己，這讓你覺得自己的發展來到了一個新的階段。

幸運色：墨綠色。

適合：發揮新潛能。

不適合：原地踏步。

每日一句：有時候，學會思路上的轉彎。
