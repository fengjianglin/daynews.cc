入門瑪雅本命盤kin1 kin2 天賦、卡點、生命狀態高峰低谷……

2019-12-13 2,348條評論 3,673次閱讀 694人點贊

瑪雅文明的代表：瑪雅金字塔

瑪雅曆法，是古老的瑪雅文明給我們留下的重要的文化遺產。

在瑪雅曆法中，一共有260個kin，對應260種能量模式，也可以理解為20種性格的人。

瑪雅曆法，可以用來看本命盤的天賦卡點，可以讀流年，還可以快速看多人關係合盤，非常神奇。

今天，我們就從大家都關心也最容易對照的本命盤開始趣味入門吧！

我將以連載方式，為大家介紹不同kin的天賦、卡點、生命狀態高峰低谷……

如果你還不會查詢自己的瑪雅本命盤，請在小程序13月亮歷中輸入你的出生年月日，頁面右上角的kinXXX，就是它的編號啦！

kin1 磁性的紅龍

所屬波符：紅龍波

代表名人：曾志偉、彼得·傑克遜

看看這些關鍵詞像不像你：

開創、衝勁、創造、創新、篤定、自信、富足、有貴人

綜述：

你攜帶著開創的力量和遠古的智慧，衝破混沌，以創造為使命，來到這個世界。

你的天賦：

你在人群之中，天生閃耀。

當人們遇到困惑與困難之時，總會想到可以向你求助。

因為你不但有破開迷霧的才智與眼光，也有帶領眾人開天闢地、一往無前的勇氣與衝勁。

你，是在世界一片混沌之時，最值得信賴、託付和跟隨的人。

你的卡點：

當你遇到你認同的有使命感的事情，你會將自己全情投入。

以至於傾你所有，把自己耗干。

挫折來時，低迷頹廢是你固有的模式。

學習如何滋養自己，補充能量，保護自己的身體，是你重要的功課。

你自帶的自信與篤定，會讓不了解的人覺得你盲目自大、自以為是。所以，務必要謹記不要固步自封，謙虛低調，才能成為真正的領袖，帶領大家打開更多世界的大門。

你的生命狀態兩極：

生命力低迷時（低頻）：

在你生命力低迷時，你總會感到孤獨與迷茫，感到自己是一個人在孤軍奮戰，周圍沒有人能真的理解你。

生命力旺盛時（高頻）：

作為一個拓荒者和實幹家，你閃爍著太陽般的光芒。

你猶如來自源頭的壹束光，用你極具穿透力的慧眼，透視亂象，劈開道路，照耀前方。

你將召喚所有為了奉獻而來的生命，聚集成光，共同創造。

kin2 月亮的白風

所屬波符：紅龍波

代表名人：梅西、鄭智化、楊瀾、寧澤濤

看看這些關鍵詞像不像你：

溝通、傳播、分享、呼吸、靈動、順其自然、喜悅、平和、自由

綜述：

你是一個領導者、探路者，純真善良，能給身邊的一切人事物帶來春風、春雨一樣的滋養，促進萬物生長。

你的天賦：

你的溝通能力非常強，和諧有愛，八面玲瓏，天生自然的人緣好得令人羨慕。

你是天生的有人格魅力的領導，因為會自動地會接納包容弱者，給予弱者他們最需要的關愛與同情。

你的卡點：

要有智慧，順勢而為，從心出發。與人交往中少用套路，做事時避免用腦算計，就可以得到更多助益，最終實現大成就。

慾望是你最大的敵人，它會將你引入歧途，甚至陷入別人設下的全套。每當你感到慾望的狂熱時，就要立刻提高警惕。

你的生命狀態兩極：

生命力低迷時（低頻）：

你在溝通中，不容易執著己見，很容易被人影響，所以當你生命能量低迷時，容易被別有用心的人利用，被他們精心編製的謊言引誘踏入陷阱。

生命力旺盛時（高頻）：

當你充滿生命力時，你可以凈化萬物。順勢而為。你無拘無束自由順暢。很容易在生活中得到小確幸式的滿足，活得舒服自在。
