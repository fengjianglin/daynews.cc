巨蟹座鄰近12月一周，一點一點去靠近夢想，不抱僥倖的奢望

2019-11-26 2,385條評論 3,816次閱讀 2298人點贊

巨蟹

本周一開始火星和天王有一個負面的角度，工作中你可能需要去負責一些創意、偏娛樂方面的項目，或是有一些工作內容需要去和團隊里的人溝通探討，雖然你們很可能會持不同的意見，或是直接和你對著干，整體氣壓也比較低，但仍然要注意自己的態度和語氣，不然極有可能會與周圍人鬧不愉快。有些人所在的組織團隊可能會迎來新人，或是送走一些人，團隊內部人員會發生一些變化。你也可能會在朋友婚禮、滿月酒上隨份子，或是聽聞這方面的消息，然後給他們送去祝福和禮物。

緊接著射手座新月發生在了你的日常宮，平淡的生活之餘，你可能還會去創造一些新鮮的體驗，讓自己的生活生動起來。你也會逐步去調整自己之前被打亂的生活節奏，規律作息，更注重自己的健康。對於正在找工作的人來說，可能會收到一些面試邀約或有所進展。海王在你的教育宮順行了，對於想要提高自己學歷和技能的人來說，或是在學習中渾噩的人，你可能會重拾自信並且堅定決心，一步步去規劃好自己的未來，並踏實的邁進。你在傳媒、出版或是一些法律事件方面遭遇到的延遲、反覆的情況，會有所好轉。

臨近周末時，金星和天王產生了聯繫，你可能會結識一些新的朋友，或是參加了一些有意思、愉快的約會，讓你的心情往好的方向發生轉變。人際關係有所提升，有些原本聯繫並不緊密的朋友可能會來找你合作。有些人可能會去找比較親密的朋友說出心裡的壓力，尋求安慰，可能通過這次交流讓你們的關係更加穩固。

在感情上，對於單身的人來說，你可能在朋友的介紹下，認識了不錯的人，你也可能會因此對你們的未來有諸多設想，但還是先確立關係並踏實的走好每一步比較好。部分人可能會急切的投身入一段感情，但這股情緒可能來得快去得也快。對於有伴侶的人來說，你們的關係目前來看還是蠻堅實的，保持著穩定、親密的交流和約會。如果你的另一半近期社交比較多，你可以多關注一下。部分人可能會有結婚、生子，正式建立起一個小家庭的計劃。

建議：不要和自己過不去，是擁有快樂的一個小訣竅。
