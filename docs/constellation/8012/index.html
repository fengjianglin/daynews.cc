就算看清世界 也看不透巨蟹的心

2019-12-05 489條評論 3,936次閱讀 1419人點贊

小編曾經有個巨蟹座男朋友，這個男人是小編非常喜歡的類型， 剛開始的時候非常喜歡。但是到後來出現了很多問題，在小編的世界裡面是非常純凈的，不允許有任何瑕疵，這和小編處女座情懷有關，可偏偏這個巨蟹是個榆木疙瘩，做事情慢吞吞而且還低調到讓小編都無法吐槽。

他做事情非常認真，也非常勤奮，他總是默默地做一些看似有野心的東西，總是朝著一個方向努力。不知道是什麼原因，他非常不愛表現自己，小編最頭痛這一點，小編是非常喜歡在別人面前表現自己的，而身邊的這枚巨蟹男友低調到不行，做什麼事情，只是一個人像老黃牛一樣默默的做，從來不計較成果哪怕成果被別人竊取，對於他來說他也會恬靜地回答一句沒關係。

每次遇到這種問題小編就會和他大吵一架，吵著吵著就逆反了，他無法接受這種填鴨式的教育方式，而小編我也沒辦法接受這種低調的生活。或許是這種矛盾的關係，反而讓戀情持續下去，很多人會奇怪，這不是互相虐心嗎？其實不然，在生活中巨蟹還是給人一種富有同情和溫暖的感覺，他們很注重家庭。對家人和朋友非常忠誠，總是能夠在家人或者朋友需要關懷的時候出現，不止如此，巨蟹還喜歡將自己的工作場所或者是家庭收拾的非常乾淨，讓自己可以在充滿愛的小窩種生活。

可是在富裕同情心也阻擋不了他複雜多變的性格，巨蟹男友生性多疑，總是喜歡他就未知的領域。有時候甚至覺得他像一個長不大的男孩，總是和對方一較高下，在他們的世界中或許沒有男女朋友的關係，更多是競爭對手的關係。最讓小編頭痛的就是，敏感多疑的性格，總是暴露無遺。在小編的世界中總是希望兩人可以坦誠相待，但是巨蟹男友總是默默的不吭聲，就算你問道他，他也只是無所謂的回答一句。每次坐下來安靜地聊這件事情的時候，巨蟹男友總是覺得不以為然，甚至覺得對方事情太多，他覺得只要讓他做事情就好，不要讓他去多揣摩別人的內心。

這個永遠長不大的孩子，不僅敏感多疑還喜歡被愛人崇拜仰望，別看這隻低調的巨蟹，可是很強的大男子主義，他們總是喜歡自身條件非常優越或者是非常成功的人士。喜歡浪漫擔憂機器敏銳，總是因為繁忙的瑣事而忽略了生活的樂趣。天生的大男子主義讓他們總是希望自己的另一半可以像仰望男神一樣望著他們。可偏偏自己骨子裡面有比較糾結，畢竟值得被仰望的人都不是低調的人，至少實力不是低調派。

或許很多人認為，這樣的戀情註定沒有辦法幸福。可我們是否想過，真正的生活不都是來自於對對方的吐槽，哪對夫妻不會相互吐槽，哪對情侶不會相互嫌棄。很多人認為小編是個怪咖，小編則認為自己喜歡的動自己一定要努力爭取，至少要比別人先邁出那一步，甘願做更多的犧牲。如果真的感覺不合適，在離開也不遲。

但是如果你能夠走進巨蟹的心裡就會發現其實他們很專情的，只要是他們認定的人就會愛對方一輩子，絕不會變心。
