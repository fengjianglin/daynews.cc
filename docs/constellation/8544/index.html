生性乖張，不講情面，殺氣很重的人

2019-11-16 405條評論 2,713次閱讀 1751人點贊

生活中，總能遇到這樣一種人：面露兇相，眼光犀利，殺氣騰騰，一生氣就臉色煞變，風捲殘雲一般，正如人們通常形容的「翻臉比翻書還快」。

這種人生性乖張，戾氣很重，時常翻臉不認人，不講情面，給人一種極不好接近，接近了又像「伴君伴虎」一般的惶恐不安。無論你傾注了多少真心和心血，都跟他培養不出感情來。

生性乖張的人，為人很偏執、不馴服、很冷僻，與常人思維不同，根本不按套路出牌，做事不講究，不正規，喜歡強勢暴力地按自己的方式行事。稍不如意或不順從，就會採用戾氣很重的方式推動執行。

這種戾氣很重的人，通常手段很殘忍，會走極端，凡事做得狠，做得絕，做得非常詭異，令人費解。對人對事，他都容易不理解、不尊重、不耐煩、不信任、不滿意。要麼很暴躁，要麼很陰沉，幾乎都是不達目的不罷休。

這種人，極不講情面，極少能相處良好，常常是六親不認。表現在工作上，眼裡只有他的目標，所有人在他那裡，都不過是物件，僅僅是手段和力量，做完了就做完了，從來不覺得應感恩、感謝，事情一過，照樣可以對你做出歇斯底里、匪夷所思的事來。

殺氣很重的人，發動做事，從來不講情面，不講感情，不會想到讓人打心裡願意積極幫著去做。但他特別擅長綁架他人。通過設計採用一系列特殊的方式方法，對周圍人群，形成一種莫名強大的威脅、脅迫的氣場，讓在場或有關的人，不得不順從地去做事，還不敢有半點聲張。

總結開來，這種人，大處講有秦始皇、張獻忠、朱元璋、成吉思汗，小處講，各種性格乖張、戾氣很重的人，都算。按成王敗寇的理論來說，成功了如秦始皇、朱元璋、成吉思汗，他們就成為梟雄、皇帝，失敗了的，就都如張獻忠、甚至各種極端罪『犯。

這類人，聲名通常不好，非常具有爭議性。但一旦成功，成名，也便會成為千古一人。

面對秦始皇、朱元璋、成吉思汗，我們極少敢說，這些生性乖張，戾氣很重，不講情面的人是壞人。可生活中，工作上，我們卻十分不喜歡這類人。

你們覺得呢？頭條的網友們。
