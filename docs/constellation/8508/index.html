看完周杰倫這段經歷，你會發現，摩羯座真的不簡單

2019-11-17 1,039條評論 3,024次閱讀 1535人點贊

關於周杰倫的成就和名氣不用多說什麼，畢竟他已經是家喻戶曉的大人物，但在他成為大人物之前，也有過非常辛酸艱苦的小人物經歷，而通過這些周杰倫的奮鬥史，我們很容易在其中找到摩羯座特有的樣子。雖然不是每個摩羯座人都能走到周杰倫的高度，但通過了解你會從中發現，摩羯座真的不簡單。

成名之前，周杰倫在吳宗憲公司默默無聞的寫過許多歌，其中包括給劉德華，張惠妹等當紅的大牌明星寫的歌，雖然大都沒有被錄用，但周杰倫並沒有因此放棄。之後吳宗憲時常談起周杰倫的那段寫歌經歷，說當時周杰倫就是一個只會拚命的怪人，總是把自己關在陰暗的工作室，一個星期都不出門，也不喜歡和別人說話。但他高產的時候，一天能寫10首歌，如此精力，讓人不得不佩服。

我們回到摩羯座的身上來看，這確實非常相符。摩羯座沉默寡言，不好接觸，但卻十分努力，只要有了目標，他們就會拚命的去工作，其韌性和精力，真的不是普通人能夠相提並論的。拋開天生的才華不說，就憑摩羯座的自律和拼搏精神，又有什麼事情是做不到的呢？

早期的周杰倫不管是私下，還是參加綜藝活動，話都非常少，看起來很靦腆，所以常常被人說是「怪咖」。其實這都是典型的摩羯座性格，他們很擅長做事實，但卻不太擅長與人溝通，所以在交際過程中常常會顯得有些木訥。但摩羯座的頭腦非常清晰，內心也非常堅定，他們時刻都知道自己的目標是什麼。

可以說摩羯座的人能夠成功，並不是因為機遇比別人多，而是因為他們時刻都在做準備，讓自己隨時有能力去抓住機遇。周杰倫的一次大機遇，毫無疑問是他的第一張專輯，而這張專輯發行的背後，還有一個有趣的故事。

當時周杰倫總找機會懇求吳宗憲幫他出專輯，無奈之下吳宗憲和周杰倫表示，只要他能在一個月內寫完50首歌，就從中挑選10首給他出專輯。這對於周杰倫而言，自然是一次很難得的機會，但不得不說這也是很大的挑戰。只要從事過創作工作的人應該能體會到，這是何等高強度的工作，畢竟創作可不是單純的體力活。

為了能夠完成這超負荷的工作，周杰倫準備了一箱速食麵，一直呆在工作室里，直到完成任務。而最終他只用了10天的時間，就寫完了這50首歌，其毅力何其驚人，當時憲哥也被驚得目瞪口呆。或許起初他只是給了一個不可能完成的任務給周杰倫，好以此來作為推脫幫出專輯的借口。但當他看到一首首曲譜工整的漂亮歌曲出現在面前時，不得不由衷佩服這個性格怪異的年輕人。於是，周杰倫的第一張專輯《jay》誕生了，而周杰倫的開掛人生，也隨之開始。

通過以上這段周杰倫的經歷，我們就能發現，摩羯座真的是人狠話不多的角色，那種忘我的拚命工作精神，在他身上表現得淋漓盡致。其實這個世上有才華的人真的不少，但為了實現目標能如此拚命的人，那真的不多。另外，如果你仔細品味周杰倫那首《聽媽媽的話》，也能從中發現，他所謂的才華，其實也不過是小時候比別人更多的付出和努力罷了。

事實上，並不是每個摩羯座都能夠達到像周杰倫這樣的高度，但通過對他奮鬥史的了解，我們能夠很形象的找到摩羯這個星座的許多特性。所以，摩羯真的是一個很不簡單的星座，那麼身為摩羯座的你，努力去奮鬥吧！
