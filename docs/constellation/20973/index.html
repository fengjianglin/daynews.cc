水瓶座：人的一生里，應該要有最樸素的生活，與最遙遠的夢想

2019-12-19 1,594條評論 2,588次閱讀 820人點贊

程小媛

很多事情都並非遙不可及，在我們還未來得及去面對一切的時候，有些人就已經悄悄地來到自己的身邊。這便是命運，他從來不會給你任何有所喘息的機會。那些真正令自己刻骨銘心的人或事，都是在猝不及防之間遇見的。

水瓶座的他們，知道自己永遠也不會明白，未來究竟是何模樣。但是我們該有向前奔走的力量，而不是在原地停駐不前。哪怕當下的生活尚未稱得上幸福，哪怕愛情距離我們也遙不可及，還是要有心中最為溫暖的勇氣，猶如一盞明燈指引著水瓶座的他們前行。人的一生里應該要有最樸素的生活，與最遙遠的夢想。

人生總是充滿著苦難的，我們也許會在這個過程中，遇見許許多多的人。就如同在一本書中，閱讀了許許多多的故事。那些不同的情節，那些不同的悲歡離合，都是我們情緒的真實寫照。當你的人生。陷入到從未有過的困境之中。當你眼前所經受的黑暗，是那樣的驚心動魄，是否還會有重拾希望的勇氣，是否還會想要站起身來，去尋找那僅有一絲一毫的光明。

每分每秒在這個世界上，都有許多人不約而同地選擇放棄。生活本就不易，當我們流盡了痛苦的眼淚，當我們嘗盡了心酸與無奈之後，是否還會懷揣著最初的希望，像一開始那樣，滿懷期待地去追逐。

未來總是虛無縹緲，如果從始至終這條道路上，都只有自己一個人的足跡，那麼又該用什麼堅持下來。水瓶座也曾經懷疑過，也曾經想要放棄。他們的靈魂總是自由而不羈的，討厭受到任何的束縛。如果不能夠遇到那個真正想要的人，不能夠遇見那一段真正想要的愛情，那麼水瓶座寧可孤獨終生。即使是一個人的孤獨，也好過兩個人的同床異夢。

水瓶座的他們，總是向來把一切都看得無比透徹。水瓶座始終都是睿智的，對於這個世界上，很多隱藏在暗處的道理，他們都再清楚不過。水瓶座從來不甘於平靜而沉默地縮在角落裡，他們勇於創新，勇於去探索那一片未知的嶄新的世界，勇於用雙手來改變自己餘下的人生，而不是跟在別人的身後，亦步亦趨。他們永遠都有著自己的想法，和自己想要去堅持的原因。不論如何，水瓶座都不會放棄。那是他們對於自己人生的堅定，和對於靈魂自由的堅持。

不論眼下的生活多麼沉寂，水瓶座的內心裡始終都充滿生機。也許在未來的某一天里，他們也會遇見另一個人，和自己有著同樣的想法，在前方的道路上，也甘願牽起水瓶座的手，兩個人一起去面對共同的風雨。水瓶座希望可以擁有這樣一個知己，希望他們所喜歡的人，並不僅僅是因為愛情而存在於水瓶座的身邊，更多的原因是志同道合，是陪伴，是鼓舞。

人生總是太漫長，水瓶座的腳步始終空虛。也許一個人堅強已久，強大到已經不需要任何人的保護。可是那樣一份來自於對方的溫暖，卻仍舊能夠衝破重重阻礙，直達水瓶座的內心。越是強大的人，反而越不需要周遭的掌聲與讚美。他們真正想要的，只是有那麼一個人可以站在和自己同樣的高度上，借給自己一個看似軟弱，實則堅強的肩膀。

沒有誰是真正刀槍不入的，水瓶座只是希望有人能夠懂得他們的夢想，懂得他們不肯停止的腳步，也懂得他們內心些許的脆弱。能夠在水瓶座孤單無助的時候，借給他們一個溫暖的胸懷，讓水瓶座安心地依靠。

我們每個人在這世上，總是要有一個值得自己去為之奮鬥的夢想，和那一個心甘情願為之付出生命的愛人。
