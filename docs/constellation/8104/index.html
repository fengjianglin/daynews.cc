12月1日

星期日 農曆十一月初六

在月亮變得更亮之前（目前是較寬的蛾眉月，位於摩羯座），可以嘗試著用裸眼觀測仙女星系（M31）和英仙座的雙星團，它們的視星等都為4，相距僅22°。天黑後它們在東北方高空，分別位於仙后座的右上方和右下方。在11~12月的全天星座圖中，它們都被繪製在中心區域。

12月2日

星期一 農曆十一月初七

在今晚早一些的時候，可以看到月亮左下方的北落師門（南魚座α）——孤獨的秋季恆星。

如果你位於北半球中緯度地區，在天剛黑的時候，當北落師門位於子午線上（正南方向），畢宿五（金牛座α）和昴星團（M45）將從東方升起，獵戶座的主星會緊跟著在它們下方升起。

這時候，北斗中的指極星（天樞和天璇）位於北極星的下方，並指向正北方。

12月3日

星期二 農曆十一月初八

現在，夜幕降臨時，北落師門幾乎位於月亮正下方。今晚接近上弦月。

當星星出現時，仙后座的W形位於東北方高空，到了前半夜後期，仙后座已升至北邊更高處，形狀已轉變成一個平躺的M形。

12月4日

星期三 農曆十一月初九

上弦月（北京時間14:58）。

天黑後，在月亮上方可以找到橫平豎直、方位端正的飛馬座四邊形。

獵戶座約在晚上八點完全從東方地平線升起，這也取決於你所處的經度。在獵戶座上方閃耀著橙色的畢宿五，畢宿五的上方是小小的昴星團，大小約為你伸直手臂的指尖那麼寬。

畢宿五和昴星團的左側較遠處閃耀著的是明亮的五車二（御夫座α）。

獵戶座下方，明亮的天狼星（大犬座α）約在九點升起。

12月5日

星期四 農曆十一月初十

在晚上早些的時候，月球位於飛馬座四邊形東側的下方。

12月6日

星期五 農曆十一月十一

月亮位於飛馬座四邊形的左下方。

隨著本周接近尾聲，木星逐漸離開我們的視線，金星正逐漸遠離木星，向土星不斷靠近。（藍色10°的距離大約是你伸直手臂一個拳頭的寬度）

12月7日

星期六 農曆十一月十二

晚上早些的時候，織女星仍然是西北偏西天空中最亮的星，位於它上方最亮的星是天津四（天鵝座α），織女星左側稍遠處是牛郎星（天鷹座α），亮度介於織女星和天津四之間。

它們三者組成了夏季大三角，當你看到夏季大三角時，你還可以在南方高空子午線上找到飛馬座四邊形。

如果你位於北緯40°附近（如秦皇島、北京、包頭、大連等），那麼今天是一整年裡日落最早的一天，儘管距離冬至還有兩周。

最早日落時間距離冬至日的偏移時間和最晚日出時間的偏移是相平衡的，也就是說，1月4日是太陽最晚升起的時候。為什麼呢？這是由於地軸的傾斜和地球軌道的離心率決定的。

本周行星動態 ( 未標紅的不適宜觀測 ）

水星

金星

火星

木星

視星等-0.6，觀測較佳，是2019年黎明觀測的最佳時期！在日出前1小時左右，可以在東南偏東方向的低空觀測到它，本周越早觀測，它的位置越高。

大角星位於它的上方較高處，可能有一點點偏左，這取決於你的地理緯度。水星距離較暗的火星和角宿一左下方更近一些。

（視星等-3.9等，位於人馬座）在黃昏時位於西南方低空，它每周的高度將不斷上升，在接下來的整個冬天至明年春天，它將作為一顆昏星閃耀在高空中。

（視星等+1.8等，位於室女座）在黎明早期位於東南偏東低空，水星右上方。更明亮的角宿一位於火星右上方稍遠處。這三者的連線寬度在本周由24°增加到33°，其中水星不斷下移，火星和角宿一不斷上移。

（視星等-1.8），在黃昏時，位於金星右下方較遠處，並且變得更難觀測。本周，二者之間逐漸遠離，距離將由6°增加到12°。

土星

（視星等+0.6，也位於人馬座），是金星左上方一顆穩定的，淺黃色「星星」。每晚，金星和土星距離都靠近1°，本周，它們之間的距離將由13°縮減到5°。

天王星

海王星

天王星（視星等5.7，位於白羊座南部）和海王星（視星等7.9，位於寶瓶座東部），在前半夜後期，分別位於東南方和南方差不多等高處。

半人馬座A

僅僅1千1百萬光年遠的半人馬座A是距離地球最近的活躍星系。這幅清晰的望遠鏡影像呈現了跨度超過6萬光年的奇特橢圓星系，還星系還被稱為NGC5128。
