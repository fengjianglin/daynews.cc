身份證尾數帶"X"的人，是有什麼特殊身份嗎？看完漲知識了

2019-12-10 844條評論 2,052次閱讀 1061人點贊

身份證是我們每個人的身份象徵，是許多人在外旅遊、玩耍的必須品。如果沒有了身份證，那麼他去社會中很多地方都會受到限制，比如不能去網吧、不能乘坐高鐵、飛機等等。總之，沒了身份證，我們的生活就是非常的不方便。

許多人經常會看到有些的身份證尾數是”x”，這就究竟是怎麼回事呢？不少朋友打趣道：”這個人肯定有特殊身份”，那麼真的是這樣子的嗎？其實不然，今天小編就來和大家一起來看看這個身份證是”x”究竟是怎麼回事。

身份證上的18位數字，是你在出生的時候，派出所上戶口的時候就會有的了，前4位數字代表的是你戶口所在的省份和地區；第5、6位數字所代表的是你所處的縣。接著就是你的出生年月日，然後第15和16位代表的是持證人的戶口所在地的派出所的代碼。第17位所代表的就是性別，男生是基數，女生是偶數。

而第18位數字所代表的是一個校驗碼，是系統上隨機出行10以內的一個數字，用來驗證你信息的準確度。在沒有科普這些信息以前，對於最後一個數字的說法有很多，比如x代表的是你出生在什麼顯貴家庭，其實這類說法都是大錯特錯的，x代表的是10,10又能直接寫在號碼後面，不然身份證就是19位數字了，所以就用x來代替。

姓名是有可能一樣的，但是身份證上的數字，有著這麼多的含義，可以更好地表明一個人的身份，讓大家不至於搞混，也間接突顯了每一個獨一無二的特性。

好了，以上便是小編今天所分享的內容，對於身份證上最後一位的含義你們都了解了嗎，最後由衷的感謝各位閱讀小編我的文章。
