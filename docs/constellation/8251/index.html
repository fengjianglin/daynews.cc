選珠寶還要看血型？快看這裡，教你挑選最適合自己的那款

2019-11-26 1,661條評論 3,212次閱讀 746人點贊

我們佩戴珠寶時有時會看生肖，生辰，有時甚至會看星座。那麼你知道我們在挑選珠寶時，還有一種參照依據嗎？那就是我們的血型。血型之說是從西方傳過來的，跟看星座差不多，那麼不同的血型的分別適合佩戴哪種珠寶呢。

A型血（性情較為平和，平時需注重養生）

性格特徵：A型血的人做事專註、穩重，很能忍耐。他們是完美主義者，但是過於的在意別人的看法，會讓自己變得小心翼翼，情緒多變，胡思亂想。是多慮型的人格。另外，重感情也是A型血的特徵，他們重視家庭，渴望得到安穩的生活。

適合佩戴：紅珊瑚、碧璽、琥珀、南紅、等顏色鮮艷的珠寶。

這些顏色艷麗的珠寶有助於讓A型血的人敞開心扉，打破他們保守的性格，彌補他們的缺陷，在平靜的工作生活里給予他們一盞明燈，讓他們可以更好的發展自己的事業，長期佩戴還有一定的養生功效。

B型血（特立獨行，要加強和外界的溝通）

性格特徵：強勢，積極樂觀、活潑外向、鬥志昂揚。活的很自在，不會很在意其他人的看法，但是有時說話做事很容易忽視別人的感受；主觀性太強，經常僅憑直覺就去判斷好壞，想法根深蒂固，不容易改變；特立獨行慣了就討厭規則的束縛，適應能力不足。

適合佩戴：翡翠、純金或者純銀、琥珀飾品。

這些珠寶有很濃的傳統意味，比較容易獲得朋友的認可，就能讓佩戴者更好的去融入周圍人的圈子。而且，文化氣息較重的珠寶可以讓強勢的B血型逐漸變得內斂，更好的讓自己的內心平靜。

AB型血（浮誇樂觀）

性格特徵：AB型血的人都比較理性、頭腦很靈活、直覺很敏銳、擅長觀察、討厭墨守成規，性情比較多變、缺少耐心。在待人接物方面有自身的過人之處，在團體中也經常站在領導的地位上，會尊重認同大家的想法和決定。但是他們虛榮心強，常高看自己，有時就會顯得浮誇。

適合佩戴：低調的木質飾品，如蜜蠟、黑檀，或者小葉紫檀等低調的珠寶。

這些珠寶可以凈化人的內心，讓人情緒穩定，補充這類人的性格弱勢，帶

O型血（為人熱情，但是要注重提升氣質）

性格特徵：O型血的人非常開朗、熱情而且奔放、不拘小節、不服輸，活的很洒脫。做事有很強的目的性，注重人際關係，有領導的風範。但是爭強好勝，急性子，一點小事就經常被惹火。既現實又浪漫，一定程度上屬於理想主義。

適合佩戴：珍珠、琥珀、菩提子、沉香、硨磲等首飾。

這些珠寶深沉又內斂，可以很好的襯托出佩戴者的氣質，提升個人的魅力。而且集好運和收藏價值與一體，兩全其美！
