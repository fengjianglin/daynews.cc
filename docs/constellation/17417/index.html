你能征服這樣的獅子座嗎？讓她做你的貓

2019-12-14 1,305條評論 2,888次閱讀 833人點贊

獅子座（Leo），黃道十二宮之第五宮，出生日期為7月23日~8月22日，位於巨蟹座之東，處女座之西。

才女鄧紫棋是獅子座的哦~

獅子座慷慨大方不愛計較，對朋友很豪爽。面子，真是獅子座不能不維護的大事。他們很驕傲，因為無法接受自己卑賤存貨於世。

獅子座滿身傲骨，可能會給人造成自戀、狂傲的錯覺，但其實是他們太看重自己的自尊心了。

獅子座喜歡佔上風，無論在什麼場合，他們都希望自己是最吃得開的那一位。他們欣賞很有才幹的人，所以要跟獅子座相處，你就得有點兒才能。用結果，證明你的實力給他看。

獅子座雖然擅於處理人際關係，但他們卻不擅於跟陌生人打交道。你如果理解為他們冷漠，那就錯了。只要你願意略微的主動，打開一個話口，獅子座立馬變得熱情洋溢，毫無防備的跟你聊起來。

獅子座是規律性很強的一種人，他們習慣事先做好計劃再行動。所以不要輕易的打亂他的時間表。

昆凌也是獅子座的

獅子座喜歡掌控屬於自己的東西，所以讓他們來掌控局面，是長久相處的秘訣。他們討厭別人不能理解他們的想法，討厭自己的熱情，卻得到別人冷淡的反饋。更討厭別人的不信任。他們也許出於無奈，對人有一些世故的套路，但他們的三觀仍是屬於很正的。他們非常看不慣當面一套，背後一套的人，喜歡簡單的直來直往。

獅子座忽冷忽熱，當他們鬧小情緒時，要安慰他哄他，不要用自己的小情緒，去挑釁，爭吵。他們只是在試探，你能否用愛包容他們。

獅子座的愛，有超乎想像的用心和樸實。所以要懂得陪他們度過情緒上的瓶頸期。
