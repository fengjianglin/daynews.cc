2019年底煩心事都不見，心態變好，幸福不期而遇的星座

2019-12-10 1,444條評論 3,421次閱讀 1078人點贊

2019年已經所剩不多，不管2019年發生過什麼事情，又經歷過什麼樣的坎坷，它都將會化成一份回憶。不管每個人是否如願以償，或是差一步距離、留一份遺憾，都要帶著努力和心中的那份美好，繼續向未來前行，繼續發起衝擊。美好的事情終將會來臨，幸福也會不期而遇。就如這三個星座一樣，2019年底煩心事都不見，心態變好，幸福不期而遇。下面就讓小編帶著大家一起來看看他們都是誰吧！運勢參照上升和太陽星座

雙魚座：內心愉悅，收穫順利

2019年底，摩羯座新月日食，新月日食發生在雙魚座的福德宮。在人際交往的過程中，雙魚座會大膽展示自己，也會接觸到一些新的朋友，更會去拓展人脈資源。但在結交新朋友的同時，也會與一些老朋友的關係疏遠，甚至出現絕交的狀況。受到新月日食能量影響，在工作的過程中，雙魚座會變得更加順利，不僅會引起領導的賞識，還會有著比較罕見的機會。水星與天王星呈現出一個和諧的角度，在談判的過程中，雙魚座變得更加順利。他們的內心會感到愉快，但俗話說，期待越高，失望越多。雙魚座平平淡淡看待每一件事情，生活總是順順利利。

摩羯座：把握機會，大步向未來

12月26日，新月日食發生在摩羯座，這會給摩羯座帶來很大改變，人生也會發生不一樣的轉折。在生活中，他們會對許多事情有著獨到的看法，也對未來有著詳細的計劃。摩羯座會變得更加清晰，也知道自己該去做什麼，該去捨棄什麼。在與人交往的過程中，他們能夠猜測對方的內心，人際關係不會變得那麼複雜。受到海王星能量的影響，在工作上，摩羯座會出現變動，但只要適合，就要大膽去把握。水星進入摩羯座，摩羯座的思維變得更加清晰，也能夠強勢反彈，以一個全新的姿態迎接新的一年。

射手座：升職加薪，注意支出

2019年底，新月日食發生在射手座的金錢宮。對於射手座來說，在財產方面，射手座會變得更加理智、清晰，也會有著獨到的理財觀念。在生活方面需求方面，射手座會參考一些性價比的東西，也會變得十分拮据，更會精打細算。水星與天王星呈現出一個和諧的角度，天王星代表變動。在工作方面，射手座會升職加薪，一些不確定的事情會有好的結果，也會受到領導的誇獎。可射手座要保持小心謹慎的態度，預防突發情況出現，也要做好十足準備。

總之，在2019年底，不管會收穫多少東西，又會失去多少東西，都要保持良好的心態去迎接新的一年。對待未來，我們都要保持一顆清醒的頭腦，也要有著詳細的規劃。不要計較於眼前的得失，更不要與別人進行比較。做到無愧於自己，心安理得便是最重要的事情。未來會有很多精彩的事情發生，或許下一秒就能夠迎來幸運。在月底，摩羯座新月日食以及水星進入摩羯座，但整體來說還是一個不錯地結束。
