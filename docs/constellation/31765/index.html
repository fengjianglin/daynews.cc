齊白石75歲畫十二生肖圖，憧憬未來期許轉運，沾沾大師的福氣吧

2019-12-24 582條評論 3,988次閱讀 2404人點贊

《齊白石十二生肖圖冊》這套圖冊是老先生祈願開運之作，創作於1937年，據說他相信「脫丙運，交辰運」的說法，期許未來小康生活的來臨。這一年齊白石75歲，在他的創作生涯里，這是信手拈來的藝術成熟時期，十二生肖畫冊的藝術價值正在於此！

十二水墨畫，略施粉黛，意趣盎然，齊白石先生對美好生活的憧憬溢於言表，大多數人在如此年齡還有憧憬嗎？我不知道，也不能想像，但齊白石先生寄情與畫，表達了自己對未來的熱情期待，僅憑這一點就勝人無數

十二生肖中子鼠配花生、丑牛配垂柳、兔子配桂花、蛇行與草、羊配柏枝、猴捧蟠桃、狗立門戶、豬與圍欄正是相映成趣的遇見美好，虎龍馬雞等獨身而自在，不與配景確而生動有趣，這是中國人對傳統文化的最美解讀，沒有高大上的理性空中樓閣，只有對生活實實在在的安康訴求，務實精神在這裡具有浪漫化的藝術表達，這是齊白石獨有的「小」境界，是市井裡你我他的共同理想！

從藝術角度而言，齊白石在這套畫冊中技法嫻熟，各種技能混搭操作，沒骨與線描，水墨與焦墨相映成趣，駕輕就熟，靜水流深之中方見幾十年渾厚的功力噴薄而發，在對未來的期於之中，老先生也表達了對時光流逝的眷顧，在亥豬圖上題詞云：「丁丑小年，喜畫生肖冊，有感時光太匆匆矣」
