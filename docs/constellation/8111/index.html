巨蟹座

早些年的時候，巨蟹座的日子一直過得都不是很順心，命格當中總是有凶星作祟，導致運勢一直被壓制，巨蟹座也始終難以賺到錢。而在12月1號，巨蟹座命中將會有吉星「八座」守護，財運節節攀升，讓巨蟹座是順利發大財，過上好日子。

水瓶座

水瓶座的朋友們從12月1號開始，未來半年內財從天降，得佛祖保佑，事業一路飛漲，升職加薪更有額定獎金入賬，多福多金，如果妥善處理好人際關係，可保一生衣食無憂，富貴有餘。

獅子座

獅子座喜歡嘗試各種不同的挑戰，而且他們從不說空話，說了要做的事情，就必須將其辦完，不然他們會非常難受，在事業上發展，他們總能把領導交代的各種難題處理妥當，所以常是領導身邊的紅人，得到重用！獅子座在12月1號開始將碰見許多機遇，撞上許多挑戰，他們將二話不說，說做就做，將各種機遇轉變為成功，將各種難題解開，然後實現升遷，取得極大的成功！

射手座

其實射手座是一個運勢很旺的屬相，只是之前的一段時間命中有凶星作祟，導致射手座的日子過得不是很好，錢總是不夠花，而且還常常被瑣事煩心。而在12月1號開始，射手座命中午火生陽火，陽火旺運，讓射手座錢財大賺，日子越過越順心。
