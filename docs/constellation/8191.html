年底奇觀｜19個必看天象！雙子座流星雨不容錯過

2019-11-29 174條評論 1,740次閱讀 441人點贊

雖然12月是一個極端的月份，北半球寒冷夜長，南半球溫暖夜短，但對於業餘天文學家和觀星者來說，這也是一年中最好的月份之一。2019年，在12月的夜空中可以看到的天文事件和景象太多了。從流星雨到日食，甚至有一位星際遊客……這個月，抽出一些時間去欣賞星星吧！

來看看12月的夜空都會有哪些美景出現吧！

12月整月均可觀測的2I/鮑里索夫星際彗星

圖源：theilr via Flickr

在12月，任何一個擁有較好的雙筒望遠鏡的人都將能夠看到人類有史以來發現的第二個星際天體2I/鮑里索夫。星際彗星2I/鮑里索夫將在12月8號達到近日點並於12月28號離地球最近。因為這個原因，它也被稱為「聖誕彗星」。不過他的正式名稱取名為「2I/鮑里索夫」，是因為它是一位名叫根拿迪·鮑里索夫的業餘天文學家發現的，並且是人類迄今為止知道的第2個來自星際（Interstellar）的彗星。

考慮到12月12號是滿月，所以大家儘可能的月初或者月末去觀測鮑里索夫，這樣能看得更清楚。天文學家說，在更好的設備下，鮑里索夫的觀測時期也許可以延長到2020年9月。想要找到2I/鮑里索夫，可以先確認仙后座的位置。更多信息請上維基百科。

12月2日-鳳凰座流星雨最大值

圖片來源：Manuel Escuder via Flickr

雖然12最令人印象深刻的是另外兩次流星雨——月中活躍的雙子座流星雨和12月下旬的小熊座流星雨-而本月的第一次流星雨實際上是12月2日晚上，在這一晚，鳳凰座流星雨達到高峰，每小時可達100顆流星。

對於北美洲的大部觀星者來說，鳳凰座是位於地平線以下。而赤道起到南半球較近的地方，鳳凰座流星雨的輻射點最明顯。對於中南美洲，南歐，中東，非洲，東南亞和大洋洲的觀星者來說，這是一個很好的機會，可以嘗試在本月發現另一種流星雨。

12月6日 仙后座φ流星雨的高峰

如果你想找到鮑里索夫彗星，可以考慮12月6日晚上去。今天晚上也是仙后座φ流星雨的高峰就在同一天空的一部分彗星鮑里索夫可能被發現。

發現確定仙后座φ流星,要尋找輻射點位於仙后座的天空。從這裡每小時你只能看到幾顆流星。根據地球在12月6日晚上的自轉，大多數可能被短尾作為進入的權利「在」我們。

不幸的是，月球在12月12日繼續走向全相位。從12月6日開始，它很可能會受到星星和流星的干擾。

12月7日——船尾座-船帆座流星雨的高峰期

圖源: Rockwell McGellin

你對流星雨感到厭倦了嗎?12月7日，又有一場流星雨達到了它的頂峰:鮮為人知的船尾座-帆船座流星雨。在夜空中，你可以從小船尾座和船帆星座之間的一個點上輻射出幾顆流星。最好的選擇是在當地時間午夜到凌晨3點之間外出觀星。

12月9日——麒麟座流星群的高峰期

對於南半球的天文學家來說，這又是一場流星雨!12月9日整晚，在麒麟座的輻射點尋找流星。最佳的流星觀測時間是當地時間凌晨1點到3點之間。

12月12日—-長蛇座σ流星雨高峰期以及觀賞大麥哲倫星雲最佳時機

圖片來源: Emilio Küffer via Flickr

12月12日晚上的小樂趣。第一個是北半球觀眾的好選擇;對南半球的人來說是第二個。不幸的是，今晚的月亮是圓的。當有如此多的光污染時，你是否試圖發現這些事件取決於你。

首先,12月12日是12月另一個流星雨的高峰:σ-Hydrid流星雨。在這個夜晚，你可以從九頭蛇星座的輻射點每小時看到幾顆流星。嘗試和觀測流星的最佳時間是當地時間13日凌晨2點到5點的黎明前。然而，在12月12日日落之後，你應該可以看到一些流星。

第二，大麥哲倫星雲(LMC)將是12月12日晚上觀測的最佳地點。它將在當地時間午夜到達南方天空的最高點

12月14日是雙子座流星雨的高峰期

圖片來源: Henry Lee via Flickr

如果你還沒有看到任何流星雨到目前為止，12月14日晚上!在這個夜晚，雙子座流星雨將達到每小時120顆流星的高峰——這是一個偉大的表演!

尋找雙子座的流星。用雙子座的亮星來定位東北天空中的星座(對於大多數觀眾來說)。流星活動預計在當地時間晚上11點左右達到高峰。雖然滿月剛過兩天，而且月光已經照了90%，但這仍然是一個出門享受12月夜空的好時機。

12月15日- NGC1981是月球和M44的最佳位置和接近點

圖源: Thomas Bresson

在15日，天文學家可以在12月的夜空中尋找另外兩個天文景觀。兩者都是發現深空物體的好機會!

位於獵戶座的開放星星團NGC 1981將會是一個很好的觀測地點。它將在當地時間午夜達到最高點。雖然你可以在非常黑暗的天空中用肉眼看到NGC 1981，但最好還是用雙筒望遠鏡或望遠鏡來觀察。

同樣的，M44(蜂群)在12月15日晚上與月亮近距離靠近時比平時更容易被發現。雖然月亮將在85%的照度下明亮，但你可以等到12月15日黎明前它們到達南方天空的最高點時再看最接近的時刻。

12月16日——Comae Berenicid流星雨的高峰期

圖片來源: Tucker Hammerstrom via Flickr

對於大多數北半球的觀測者來說，Comae Berenicid流星雨將出現在地平線以下。南半球的觀測者可以在12月16日晚上觀賞流星雨的高峰。在這個夜晚，你可以從獅子座的輻射點每小時看到最多3顆流星。

12月20日——12月獅子座流星雨的高峰期

流星,流星,無處不在!雖然12月獅子座流星雨是這個月最不為人知、最不活躍的流星雨之一，但如果你發現自己要去慶祝冬至，而且天空很暗，這仍然是一個有趣的天文事件。

12月20日晚上，在小獅子座的輻射點每小時最多可以看到3顆流星;對大多數觀眾來說，它將在西北天空的低處。

12月21日——冬至

圖片來源: Alexandre André via Flickr

21日是冬至。我們列出了一份很棒的冬至慶祝清單，以激勵你度過一年中最長的夜晚……或者你也可以計划去觀星，因為月亮的亮度只有18%。在南半球，夏至是一年中最短的夜晚。如果你對這些也感興趣的話，我們有一份夏至慶祝清單。

12月22日-接近月球和火星

圖解: Auvo Korpi

在經歷了幾個月的有限的火星觀測選擇後，從12月21日到22日，這是一個在夜間觀察這顆紅色星球的好機會

雖然這太遙遠，無法在一個單一的視野內觀看，你可以使用雙筒望遠鏡獲得更近的視野。

月球也只會被10%的光照亮，所以它會幫助你到火星（這是很明顯的，因為它的紅色色調）而不會遮擋視線。

12月23日——小熊座流星雨達最大值

圖片來源: Mike Lewinski

雖然雙子座是12月流星事件的主角，但別忘了小熊座流星雨也會出現，今年23日達到最大值。小熊座流星雨發生在12月17 – 26日。在活動高峰期(今年12月23日)，你每小時最多可以看到10顆流星。在雙子座流星雨之後，這個數值可能看起來很低，但是小熊座流星雨仍然是每年晚些時候令人尊敬的持續的流星雨。

小熊座流星雨從小熊星座輻射出來。尋找北極星，找到小熊星座，然後掃描整個北方的天空，試著發現這些流星。由於小熊星座是一個環極星座，所以整晚都可以觀測到流星。此外，月球將幾乎在其新的階段，天空將是異常黑暗。

12月25日——日環食

為了慶祝聖誕節，地球的某些部分將會收到一份天文禮物:日環食，也被稱為「火環」日食。至少在非洲、大洋洲、亞洲和太平洋上的島嶼上可以看到日偏食。全食路徑將穿越沙烏地阿拉伯、阿聯酋、阿曼、印度、斯里蘭卡、印度尼西亞、馬來西亞和菲律賓的部分地區。

12月28日— NGC2232是接近月亮金星處於（觀測）有利位置

圖片來源: Arvind Govindaraj

12月28日又是一個值得一看的天文景觀之夜。首先，疏散星團NGC 2232將會被很好的觀測到。這個深空天體位於獵戶座附近的麒麟座。雖然肉眼很難看到NGC2232，但你可以用雙筒望遠鏡或望遠鏡在12月的夜空中跟隨獵戶座的腰帶「向左」看。

你也可以在12月28日晚上看到月亮和金星之間的近距離接觸。它們彼此之間的距離在0°58 ‘以內，用肉眼或雙筒望遠鏡很容易就能看到(但對望遠鏡來說距離太遠了)。由於今晚的月亮只會被3%的亮度照亮，你可以用新月的銀色部分來發現天空中明亮的金星。

12月29日- NGC 2244處於有利位置

圖片來源: K Bahr via Flickr

為了圓滿一個月的天文景觀，這裡還有一個。尋找12月29日晚玫瑰星雲中的NGC 2244疏散星團。這個星雲和星系團將會在天空的高處，而且位置適宜觀看。它們將在當地時間午夜到達最高點。玫瑰星雲位於麒麟座。
