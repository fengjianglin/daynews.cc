從甲骨文到易經、道德經，看懂水的陰陽，才會明白上善若水的含義

天天要聞
2019-11-21

說到「上善若水」，幾乎沒人不知道這是讚美水的至善美德的，水善的最高表現就是「柔「，守弱貴柔，以柔克剛，能以「天下之至柔，馳騁天下之至堅」。就這麼簡單嗎？顯然不是。

「水」的前生今世

遠古時期的水崇拜。「水」是生命之源，沒有水就沒生命。人類在原始時期，認為水具有神性，能養育萬物也能毀滅一切，因此在人類最早的自然崇拜中，水就是人類崇拜的對象，人們賦予水以神性，祈禱水給人類帶來安詳、豐收和幸福。中國傳統文化中的龍王就是對水的神格化產物，祭祀龍王，祈雨降吉祥是中國最古老的傳統信仰。

「水」的象形造字法。「水」與天、地、雷、水、火、山、澤一樣，也是最早被賦予「字形」的物質。我們看一下「水」字的演變過程：

圖中，甲骨文的寫法有多種，但不管如何寫法，都有自上而下的流動之意。到了金文和篆文基本定型為

水的形狀。在甲骨文中，動態的流水一般寫作縱向的字形，靜止或漫流的水通常寫作橫向的。

在造字時代，水的源頭叫「泉」；飛濺的山泉為「水」；山泉匯聚處為「澗」；山澗在地面匯成的清流為「溪」 ；眾溪為「川」 ；眾川匯而為「河」，大河為「江」。

先秦時期，易學興起，自然科學和社會倫理都被融入並不斷豐富，因此，《易經》成了涵蓋萬有，綱紀群倫，包羅萬象的哲學思辨之書，被稱為中華文明的源頭。但是無論如何變化，其精髓不變，那就是「陰陽」觀念一以貫之。

易經是陰陽術數之學，天地萬物都涵蓋在最初的「八卦」里，這「八卦」中就有「水」的卦象：

看到「水」了嗎？第一排第三個「坎」卦的卦象就是水，對於這個卦象可以有千種解讀法，但都有「坎」的本身意義，即低陷之處，至於如何發揮，因人而異。我們不做易理探討，我們只看這個卦象的陰陽構成：這個卦象本身就是水的古體象形寫法，它的上下爻均是「陰」，中間為「陽」，這種寫法的寓意在於，水的表象是陰柔的，但是它卻並非純柔不剛，它的內心是陽剛的、強大的。而水的屬性是趨下的，人們常說，人向高處走，水向低處流，說的就是水的趨下「德性」。

之所以給這個卦象取名為「坎」，就是此處低洼，是水的歸宿地，與「坷」恰好相反，一個「欠土」低陷，一個「高土」為埂，坎坷，意為高低不平，事有不順。

想想看，萬水東流，曲折往複，避高就低，一直趨下而行，直到找到安身立命之處。但當河床偏高，前行受阻時，便會在低洼處積蓄力量，強力決口，這就是水性，它很柔也很弱，但若「坎」溢滿、不能滿足生存需要時，便會繼續尋找更低、更大的生存處，於是突破河堤薄弱處，咆哮前行。這時候的水是剛強的。

明白了水的陰陽屬性，再來解讀「上善若水」，就容易多了。

「上善若水」解讀

「上善若水」一詞出自《道德經》第八章：

上善若水。水善利萬物而不爭，處眾人之所惡，故幾於道。居善地，心善淵，與善仁，言善信，正善治，事善能，動善時。

夫唯不爭，故無尤。

第一句是總括，最高境界的善就像水那樣。為何這樣說呢？因為水利益萬物而不與萬物爭，它以謙下為宗，所以說，水是最接近「道」性的。道無所不在，無所不利，而水亦然。但水並不是道，道是無物之象，無狀之狀，而水是看的見的實物，所以只能說「近乎」道。

何謂道？傳曰：「一陰一陽之謂道，繼之者善也」，道尚自然，無慾念。「上善」，是至高境界的善，並非世俗人所謂的善。老子在第二章說過：天下皆知美之為美也，斯惡已；皆知善之為善也，斯不善已。天下人都認為的善是爭強好勝，是矜誇虛偽，是步步登高。而水不是，水是避高趨下的，江海所以能為百穀王者，以其善下之也，所以，水是最接近道的品性的。

總括之後，老子列舉水的「七德」之善：

居善地。水以下為宗，謙下能匯聚涓涓細流而成磅礴之勢，故謂之為善地。人應該像水那樣，學會謙卑，善居人後；

心善淵。內心有如深潭一樣寧靜，不受外界環境的影響和干擾，無論流、止，隨處可為淵、可為「家」。人就應該像水那樣，善於保持寧靜深沉的狀態；

與善仁。水滋養萬物，無所偏私而一視同仁而。人應該像水那樣，善待眾人而不求回報；

言善信。水不因自己歷經曲折而失信於人，水的「信」表現在有如銅鏡般，映照事物本來面目。人應該像水那樣，雖有變故，也要誠實守信；

正善治。水性清靜平正，能過濾污濁，清靜自守。人應該像水那樣廉潔公正；

事善能。水外柔內剛，剛柔兼濟，隨方就圓，天下莫柔弱於水，而攻堅強者莫之能先。人應該像水那樣揚長避短，圓融四通；

動善時。水的流、止，合乎時宜，滿則溢，多則流，無欲無求，完全是順乎自然。人應該像水那樣把握時機，順時而動。

上善者正是因為像水那樣具有水的謙下之德，與人為善，與物不爭，所以便沒有過錯，也不會招來怨恨。

總是，水具剛柔之性，有陰陽屬性，所謂七者，核心只在「不爭」，而以處善為本。故《易》云：謙者，德之柄。水得之矣。
