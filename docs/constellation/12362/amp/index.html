#十二星座指南之射手座使用說明書#

這個世界是豐富的。人群中我們撞上各類不同性格的，有點彼此融合，有的互相不解，這不是他她的錯。只是不同星座各自的反應敏感的方向不同，導致不同的覺察，也就各自表現不同。

這兒談談三個變動星座的不同的天賦。

雙子座♊

#雙子座，你們是風之精靈#

雙子座，風象星座，風象變化就是空氣流動。風象星座的頻率對著感官世界，因此細微的聲音、味道和嗅覺，他們都能覺察而且分辨清楚。

大家有沒有發現身邊的雙子，語言學習能力強，只要與當地人接觸，很快就能模仿他們的發音。而且他們也能敏銳地指出他人發音的部位不對。

而且在噪雜環境中，眾人都被四周喧嘩干攏地不想說話。雙子不僅鬧中話多，竟還問大家是否聽到某個樂曲聲音?多數人都表示哪有什麼樂曲聲，只有雙子的說話聲。

語言變化其實就是嘴及舌的變化，使（空氣）風振動不同產生吧。風象又變動的雙子很容易對風變化把握住也就敏感。

同樣雙子對味 嗅覺的敏感也來自食物的氣味（風）變化的敏感。

射手座♐️

#射手座#

火象射手是個直覺敏感的星座。直覺顯然與感官的敏覺不同。

直覺，指直觀感覺。指沒有經過分析推理的觀點，不以人類意志控制的特殊思維方式。

也就是說，雙子座的聽覺嗅覺是鼻子耳朵等感官感覺到的，而射手座說出的理想或目標，就是隨口就出。你問為什麼？他們說就是這樣的，非常自信的樣子。

因為大家都知道射手座的守護神是宙斯，眾神之王，是射手座智慧的潛能量。由射手座的神話也可以看出射手座是智慧的化身。

因此射手座的朋友常常可以感受射手聚會時說的那些哲理性溫情的言語，好似直中他們的內心，受到鼓舞，這就是射手座的直覺本能，熱受哲學宗教的射手直覺更是超然的。

雙魚座♓

雙#雙魚座#
感覺是什麼呢？可以看看你們周圍的雙魚座們。雙魚座不像雙子，射手嘻嘻哈哈言語多，活潑熱鬧。雙魚走著，坐著都是眼晴在動，左看右著觀察著。

其實雙魚也沒具體看什麼，她們又不是處女盯著小細節，其實就是找找感覺，街道上四周是否與往常一樣？旁邊朋友是怎麼樣性情？

都說雙魚是黃道十二宮最後一宮，集中十二星座的優缺點。那麼射手座的直覺，雙子座的聽覺嗅覺等，雙魚都具有。而且還能通過肢體語言悟出些東西。

水象星座的她們融合了這些，又受海王星影響，雙魚的感覺有些靈性又夢幻，唯美，她們易沉浸在自己感覺的浪漫中。你讓雙魚演（白雪公主）的故事，她們演得很投入，她們能自己製造出這種感覺氛圍，似夢似現實地演著。

射手座的直覺，雙子的聽覺，雙魚的感覺各自不同，彼此互相理解，不能因為對方與自己不同的覺察能力，就懷疑否定。

抱著理解和欣賞，包容看待對方，才能彼此交心，甚至某些時候這些特殊的覺察能力能幫助大家。
