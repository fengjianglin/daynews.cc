Asya一周運勢｜|11月25日-12月1日（射手、雙魚、水瓶運氣上升）

2019-11-24 966條評論 2,097次閱讀 544人點贊

金星進入魔羯座，逐漸遠離木星，與天王星形成和諧相位，同時得到火星的支持，本周金星能量依然強旺，我們對待金錢的概念會相對穩定、保守一些，也想要做出一些改變和調整，更加有動力掙錢呢。金融行業和美容行業等可能會做出更加保守穩固的改變。

太陽進入射手座，本周太陽的能量比較純粹，而且也比較強旺，很多人會考慮做一個長遠的計劃，大家的思想更加open。

火星依然與天王星形成對沖相位，我們執著的事情或者是我們的行動力近期會發生一些改變，好在火星受到水星的不良影響越來越小。

總體來說本周的星象比上周的狀態整體上升。

本周天秤們要注意人際關係方面的問題，可能你的合作夥伴對你的要求增加、對你的態度兇巴巴，或者是由他人來影響你們之間的合作關係，讓你感受到不爽，有些事情需要自己多一些心眼兒，提高警惕，而有些情況需要自己忍一時風平浪靜。好在這一現象不會持續太久，之後的運勢Asya會告知你們什麼時間會更加順暢哦。

本周你們會更加在意與家庭相關的事務，家庭也會給予你一些支持，多餘家人溝通，增進感情，不過家庭方面可能還會遇到一些棘手的問題，或者因為他人而讓家中的事務沒有按原計划進行。

感情：❤

感情狀態不穩定，容易發生改變，戀愛中的天秤座可能會受到一些爛桃花的誘惑。⚠

本周天蠍座的個人能量逐漸上升，但是命主星火星被天王星對沖，可能你們會迎來一些改變，也許是思想上，也許是行動上，可能想要在工作方面做一些調整，跳槽、調崗都是有可能的。

本周財運提升，部分天蠍座會升職得到更高的薪資待遇，本周的財帛宮能量很旺，一定要把握好時機努力賺錢哦，給領導展示更優秀的自己，你們被提拔的機會增加哦。

與人合作方面容易受到他人的干擾，可能會影響你的原計劃，特別是與客戶的關係，一定要提前做一些預防工作，多一個方案會使你更加有底氣。

感情：❤

另一半的爛桃花增加，你自己的態度也許會發生變化而使你們的感情不穩定因素增加。

本周射手座的運氣提升了不止一兩個檔，太陽進入你的命宮，給你們帶來了貴氣和能量，有意義的事情本周放開大膽的去做吧。

前段時間心情不穩定，本周你的脾氣平和一些，他人對你心情影響的程度減弱，可能你們經歷了一些事情選擇了改變自己的處世態度或者遇到事情的處理方式，這對你們的內心有促進作用。部分射手座近期可能突然決定去做整形或者突然放棄去整形，突然決定/放棄參加某個公益團體。

本周你們在財務方面要多注意一些，可能會有些多餘的開銷讓你們財務方面消耗較多。

感情：❤❤❤

度過一個兩人的私密二人時光吧，沉溺在愛情的海洋里，想虐死單身汪嗎。

金星進入你的命宮，會給你帶來事業機會，有子女的魔羯座與子女相聚時間增加，或者娛樂的時間增加，部分魔羯座考慮去做某個投資項目，不過本周能量並不是非常好，在投資或事業方面要注意有公司資金的問題或者是人際關係的問題而使事情進展沒有那麼順利。子女方面要多注意孩子的內心想法，多一些高質量的陪伴。

好在本周你們的心態比較好，能夠承擔很多不愉快的事物，心裡有很多事情容易展現出來，心裡藏不住事情。打算做整形的魔羯座也將會迎來比較好的時間，利於做一些關於美容、信仰、身心靈、公益方面的事務。

感情：❤❤❤

桃花運增強，你想要獲得甜蜜的戀愛，戀愛機緣也增強了。但是本周容易遇到爛桃花，或者戀愛受挫。

前兩三周的時間，水瓶座感受到事業方面帶來的壓力和挫折，本周開始事業方面的事務逐漸好轉，當然不會立馬改變現在的局勢，但正在走出泥潭，本周你們在事業方面還是會有一些新的想法或者想要打破目前的局勢，上周有部分水瓶座反饋失業了，在找工作，在12月份事業運才會走入正軌哦。

本周人氣方面比較給力哦，可以多參加集體活動，與朋友聚會，也許他們能夠給你帶來機會。

感情：❤❤

另一半本周的桃花運勢不錯哦，可能你會吃對方的醋O(∩_∩)O，可以與對方一起參加聚會、出去遊玩哦。

雙魚座本周也是好運榜的頭牌呢，太陽進入你們的事業宮，事業方面本周的運勢非常好哦，努力工作、規劃事業，將精力投入在事業方面是明智的選擇。在對的時間做對的事情，才可以事半功倍。可能同事或者下屬會給你的工作帶來很多的好處，讓你覺得事業發展更加順暢，好隊友的重要性相信你們懂得。

考試運勢上升（前兩周有提到不建議雙魚座在那期間考試），但是在你的學習或者考試中容易受到他人的干擾，還是不可掉以輕心哦。

感情：❤❤

與另一半可以一起出行旅遊，利於你們的感情哦。雙魚座桃花運勢上升。
