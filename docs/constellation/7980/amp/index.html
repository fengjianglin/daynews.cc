我的獅子座朋友，年薪70萬，卻沒有交過女朋友

天天要聞
2019-12-06

他熱情大方，充滿陽光活力，一副鄰家大哥哥形象，這種性格是大獅子們與生俱來的優點，後天無論多麼困苦的生活也無法改變。但他愛面子，做出點成績就容易沾沾自喜，做什麼事情都有自信，甚至變成了自大。對人生美好的憧憬在多年的摸爬滾打之後，才發現原來自己只是浮塵里一粒塵埃，他是我的獅子座朋友。

劉多餘不知道母親的長相是什麼樣子的，可能只有在夢裡見過吧！因為在他才一歲的時候，母親嫌棄家裡貧窮，跟著別的男人跑了，從此以後就再也沒有回來過，再也沒有聯繫過。

他在家排行老三，還有兩個哥哥，父親說他本不該來到這個世界上，因此給他起名為『多餘』！

劉多餘從來不知道母親的懷抱是有多麼的溫暖，也不知道母親的關懷是有多麼的貼心，他只知道在跌倒的時候只能獨自站起來的無助感，只知道被小朋友們罵是個沒媽媽的孩子時的悲傷。

多餘從小就很懂事，在我們幾個小夥伴中顯得像個小大人一般，只要別人有什麼困難，他都會儘力地去幫忙，做什麼事情也是第一個沖在前面。學習也很用心，基本上在我們班上都能考第一名，再加上他做事認真負責，所以從一年級到六年級都當班長，同學和老師們都很喜歡他。

他憑藉優異的成績考上了重點中學、高中、大學，讀書期間每年都能拿到獎學金，還得到社會愛心人士的資助，能讓他順利的完成學業，因此他也擁有一顆感恩的心。

多餘雖然從小就生活在單親家庭中，但並沒有讓他形成孤僻內向的性格，反而獅子座的性格特點能在他身上體現的完美。

他很熱情，心地善良！

那一年冬天，我們在北方的一座沿海城市讀大學，西伯利亞冷空氣帶著鵝毛大雪席捲了整座城市，我們幾個同學瑟瑟發抖的走在街上，路上的行人寥寥無幾，但卻在寒風中有一個身形單薄瘦弱的人顯得格外刺眼，那是一個沿街乞討的流浪漢。

劉多餘走到流浪漢面前，雙手在衣服口袋裡摸索了半天，終於拿出了20元錢，看著那人凍得烏紫的臉色，稍微猶豫了一下，便把身上的大衣脫下來給流浪漢穿上，20元錢也給了他。

一個同學趕緊上前去拉住劉多餘，「你幹嘛？這些要錢的人很多都是騙子啊！」

多餘微笑道：「那又如何，都是窮人，有錢人誰願意冒著刺骨的寒風出來要錢！我也只能給予那麼一點幫助而已。」

我知道送給流浪漢的那件衣服是他最喜歡的一件，也是冬天裡僅有的兩三件可以保暖的大衣，受他的影響，我也拿出5塊錢送給了流浪漢。

獅子座的他對待事業精明能幹且有充滿自信，他從來不會懷疑自己不行，老闆委託的任務什麼都敢接，特別是遇到自己喜歡的事情，更是精力充沛，三天三夜不休息都無所謂，只要干出成績來便會覺得自己無所不能，有點自大狂的感覺。

事業上信心爆棚，感情上卻膽小怕事，多疑善妒！

可能是受到單親家庭的影響，對於愛情他很不自信，不敢輕易的和別的女孩子交往，如今已過而立之年，依舊沒有女朋友。

多餘憑藉自己的聰明能幹在公司當上了區域經理，年收入70多萬，也算得上是個成功人士了吧。以他的為人和條件，自然是吸引了一些女人們的愛慕，向他表白者也不少，但他依然不敢在感情上邁出第一步。

我和他聊天時曾經問多他喜歡什麼樣的女生，他沉思了好一會才喃喃說道：「喜歡什麼樣的女生我也不清楚，應該是一個不會拋棄家庭的吧，只希望我的孩子能有母親的愛！」

「那你恨你母親嗎？恨她拋棄了你們。」

「我連她的樣子都不知道，甚至父親也從來沒有在我面前提到過她的名字，這樣一個似乎從來沒有出現過的女人，我根本不知道如何去恨她。」

兩個人之間最大的悲哀不是多麼的怨恨對方，而是徹底的遺忘了對方，連名字也不記得了！

感情上他是一個很沒有安全感的人，能給予別人幫助卻不能普渡自我，害怕失去卻不敢去擁有。希望他會遇上一個他愛的人，一個也愛他的人。

獅子座有著如同雄獅般兇猛剛強，他們熱情大方、陽光活力、自信還有點大男子主義。也有著如貓咪們溫柔的一面，他們也需要安全感！
