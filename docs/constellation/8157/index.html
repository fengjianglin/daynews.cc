射手座的箭射向箭靶，雙子的風刮向四方，在人群中如何區別出他們

2019-11-30 482條評論 3,767次閱讀 2464人點贊

#射手座# #射手座性格#

#十二星座朋友指南#

射手座與雙子座，有些共同特點，熱情言多，自由洒脫，花心無耐心。他們雖是對宮，卻似乎相似性大過對立性。

然而，既然是對宮，肯定是內在的不同，外在的一些表象相似而己。

從希臘傳說中可知，射手座又叫人馬座。與一個半人半馬的生性兇猛族群有關。

族群中有個叫喀戎(Chiron)，他賢能智慧，多才多藝，精通音律醫術，善長弓箭搏擊，而且還能預測未來。

喀戎金髮披肩，雙目有神，隱居山中，親切和藹， 授徒為業。

其中雙子座傳說中的兩兄弟，卡斯托爾（Castor）及波呂克斯(pollux)就是喀戎的學生。他倆向喀戎學習成為了育馬能手及格鬥家。#雙子座性格#

賢能的喀戎為神人愛戴，可在一次意外中，他被自己學生赫刺克勒斯的箭所傷而死亡。宙斯同情他，就升喀戎到天上，供後人景仰，就是射手座。

因此，射手座本就是個知識卓越，才藝超人德高的智者的化身。而雙子座是兩兄弟一個神一個人的化身，在內在他們是射手座的學生。正巧，射手座是雙子座對宮，內在完全不同，雙子座一直需要學習射手座的大智慧。

也許喀戎教會了雙子兄弟育馬及格鬥，還沒來及教授他們精神層面更深地探索，就升上天，這樣雙子座內在學會了競技技能，這技能就讓他們在人群愛表現自己，更善於爭辯。

而在水星影響下的雙子座，雖能言善辯，卻無心探尋深入本質東西。

卡斯托爾和波呂克斯神廟

水星是「中性」的性質，代表我們一般的直覺意識和效果，代表收集知識的驅力，語言表達的方式。

水星給雙子帶來熱情親切，善於交流，理智冷靜的一面，卻也有自負的、膚淺的、浪費的、感情用事的、俗氣的、無教養等一面。

水星影響下，雙子座愛好廣博，對世間任何事情都有著濃厚的興趣，卻雜而不精， 雖積极參与爭辯卻無明確方向。觀點思維就象風隨意刮向四方，給大地表層一些改變，就又轉向其它方向。

卡斯托爾和波呂克斯神廟

在人群中，射手座象雙子座那樣開朗，喜歡說話，積极參与，但喀戎化身的射手座，坦率寬厚，追求智慧，熱愛宗教，哲學的探討。

他們看起來自由超脫的，追求人生目標，勇往直前，像射手座手握的箭，射向箭靶，箭靶就是他們的理想目標，所思所說都是指向這個明確日標。

因此，人群中，區分射手座與雙子座，可聽聽他們的言論內容，

射手座會聚會中興奮講述自己理想，然後拉著朋友們一齊討論，他們會尊重朋友耐心傾聽，認真回應，讓朋友理解自己的理想目標。

而雙子座更注重個人言論表演，不注意旁人言語，討論的主題無邏輯聯繫，他們只在意自己語調內容能否吸引四周。倘若朋友參與聊天，雙子甚至會粗魯打斷他們，只為眾人只聽他們發言。可惜眾人聽後一頭霧水，因為他們風一樣思維變動地毫無邏輯。

論勇氣，火象射手為了目標實現，敢於冒險，勇住直前，雙子大概說說而已，也缺乏射手獨立上路的精神。

因而，射手座的花心，是為了更好地追求他們堅持的理想目標，雙子座的花心只是追隨新鮮。

射手雙子都追求自由，射手慢慢走向哲學宗教，雙子只是隨著好奇心而變化著。

人群中也就區分出他們了。

這裡談論的星座性格，只是針對比較典型的射手座，雙子座，供大家參考。
