雖然說，當我們碰到喜歡的人的時候，是很難掩飾自己的愉悅與緊張，但是有一些人，越是喜歡對方，就越會隱藏自己的那分心思，然後用一些舉動與關注，來吸引對方，讓我們來看看有哪些星座有這樣的毛病。

第四名：射手座

射手座給人的既定印象就是「洒脫」、「不羈」，但你知道射手座若是喜歡上一個人，那可就沒完沒了了。

射手座想要關心對方，但又怕對方識破自己的伎倆，想要靠近，就在那邊糾結，各種洒脫在這就蕩然無存了。平時跟朋友或同事相處的射手座，總是幽默風趣，討人歡心，一但遇到喜歡的對象，射手座無法掩飾自己的緊張，只能說這樣的射手座更可愛。

第三名：水瓶座

待人一向溫和的水瓶座，喜歡結交各式各樣不同的朋友。但突然有一天，水瓶座發現，他對你不只是單純朋友的喜歡時，他會開始慌張，害怕自己喜歡你被發現，把你嚇跑了。

另一方面害怕自己因為要投入感情而失去了自由，還有害怕各種生活上要共同經營的事情。當他有以上的念頭時，他就會漸漸減少與你的接觸，是因為這個「火星人」，怕自己真的愛了就無法自拔。

第二名：處女座

在處女座的字典里，你找不到「衝動」兩個字，因為處女座真的非常會隱藏自己的情感，主要是靠著他那堅強的意志力，好好控制對於你的喜歡，一直保持著美好的姿態。

當處女座喜歡上一個人的時候，他那無法掩飾的小驕傲，就會出來作祟，這樣想要隱藏的情感，就會不攻自破的讓大家都看得出來。不愛主動聯繫你的處女座，表面上似乎對你很冷淡，直到他有一天受不了了，主動地表達，那才是處女座真正的開始。

第一名：摩羯座

不擅長表達自己的情感，也嘴笨的摩羯座，常常只會默默的付出。在不清楚對方是否喜歡自己的時候，他會施展「敵不動，我不動」的攻勢，主要是觀察對方透徹後再出現，等到你需要摩羯座的時候，他又一副好朋友的姿態，對你義不容辭地出手相助，就是怕你看出摩羯座對你的小心思。

所以摩羯座會一直隱藏自己對你的情感，直到你表達出，其實自己也喜歡摩羯寶寶，他才有可能會主動喔。
