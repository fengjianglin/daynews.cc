十二生肖誰居藍？來看142期雙色球定藍斷語

2019-12-10 393條評論 4,499次閱讀 2850人點贊

一語斷藍：似指一正，出自公益時報的中華彩票。

上期的「共道一聞」本人看5.6.7.8主8，只及雙數，最終是5.6.7.8四數相加等於26，屬狗的歲數，藍球開14。

今期來看一看「似指一正」，首先先對著參照表（在評論區）看圖解字及領會斷語所說的中心意思。

一.似、就相似，相近類似，有類的意思。

二.指、就是手指，特指人的手指。

三.一字、專一，全部，單一，這裡先取全部。

四、正字，當然是正確，不偏不斜，在參照表中正在月份的正月寅月。

似指一正，聯繫在一起理解就是象人的手指是正月虎中歲數的全部總和（一取整體），表中虎的歲是10.22.34.46，其總和是112歲是猴的歲數（112減多幾次12就有猴4.16.28.40）。那十二生肖有象人的手指的就只有猴了，對應藍球碼4.16。

再取其一，會意似指一正中的指字有有蛇的筆畫「匕」，一字取專一之意，一正就是專一月份中蛇屬的數字——四月，那藍碼取4就是。

圍藍以雙數2.4.6.8.10.12.14.16

生肖以鼠虎龍猴定藍肖（12.10.8.4.16）

一肖定藍：猴4.16

一碼定藍：4

取紅球生肖：豬羊兔狗（1.13.25.5.17.29.9.21.33.2.14.26），定膽一碼9。

興趣而發，莫亂投注。
