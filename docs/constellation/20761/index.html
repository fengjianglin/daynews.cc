白羊座：吃喝玩樂

白羊們有著極其旺盛的活力和充沛的體力，為人處事不喜歡掩飾自己，總是表現出自己最真實的一面，並且他們的行動力和決策力都非常果斷快速，常常是想到什麼做什麼，有什麼就說什麼。對於自己喜歡的東西，他們從來都不會吝嗇。如果有想吃的東西就會排除萬難去吃，有想要體驗的活動就會去參加，極少有顧慮，總是勇往直前。所以，他們的錢包總是處於飢餓狀態，畢竟他們吃喝玩樂很隨意啊。

金牛座：吃吃喝喝

大多數金牛們都有著極其旺盛的食慾，喜歡嘗試各種各樣的美食。向來重視金錢的牛兒們對待美食的時候才會表現出大方的模樣，因為在他們看來能夠治癒人們難過的美食是最寶貴的，所以願意在美食方面花費更多的金錢。他們不但喜歡嘗試來自世界不同地方的美食料理，更加喜歡來自不同地方的零食。當他們閑來無事的時候，他們就會買各種各樣的零食來解解自己的嘴饞。由此可見，他們的工資多半都花費在吃吃喝喝方面了。

雙子座：各種興趣

雙子們有著極其旺盛的好奇心，喜歡探索和嘗試各種各樣的事物，喜歡感受和體驗各種人文風情。由於他們不喜歡一成不變的生活環境與氛圍，因此他們有著非常豐富的興趣愛好。無論是哪一種愛好都是需要花費金錢來支撐的，以畫畫作為興趣就需要購買畫紙、鉛筆、畫具等；以音樂作為興趣就需要購買樂器；以遊戲作為興趣就需要購買光碟；即使是以讀書作為興趣也需要購買書籍。為了支撐自己的興趣愛好，他們的工資就這麼花完

了。
