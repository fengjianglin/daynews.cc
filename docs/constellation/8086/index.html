近日星象須知：

新的一年總是會有更多對新年的期待，而對生活的美好渴望，也少不了自己的努力。生活中的心愿很多，能夠心想事成，也會讓自己的生活有更多的期待，努力過後的事業，不僅會給自己帶來回報，也會成為自己今後努力過的成功，在近期，一直嚮往美好的心愿，也終於能夠實現，近期，心愿心想事成，事業得以高升。

雙子座的財運在2020年是屬於波動比較大的那種，經歷了年中的富裕以後，雙子座的性格就是會比較喜歡享受，有點小錢就會不知不覺中花多了，所以到了年末雙子座的手頭又是比較緊了，他們自己都不知道錢都花到哪裡去了。畢竟他們沒有作賬目的習慣，即便每次買的都是小東西，但是買多了也都是很大的一筆開銷，再加上年末一些促銷活動力度比較大，所以雙子座就是各種買了。

木土換座，四餘星也全部換座，2020年註定是運勢起起伏伏的一年，遠不如2019年木星射手土星摩羯雙入廟能量那樣專註純粹，不互相干擾，多方面得到成長，在掌聲中看到自己的獨一無二的價值，在困境中沉澱自己的核心競爭力。
