12星座專屬“刺激戰場武器”獅子座AWM狙擊，雙子座搞怪平底鍋

2019-12-20 971條評論 3,708次閱讀 821人點贊

小編就和大家一起 聊聊我們十二星座的專屬「吃雞武器」是什麼？獅子座是AWM狙擊槍，我怎麼是平底鍋？

1、白羊座

白羊座的專屬吃雞武器是UMP9衝鋒。對於白羊座這樣風風火火的性格，這款武器很是適合白羊座的人，射程遠，還能裝備很多的配件。特別的適合白羊座這樣大大咧咧，直爽，且還是一個急性子的人。

2、雙魚座

雙魚座的專屬吃雞武器是M249輕機。對於雙魚座這種比較較弱的人來說，一款輕機槍很是適合雙魚座的人，雖然輕，但是威力還是很強的，而且還有很大的容量一下能夠填充100發子彈，只是這款武器還是比較少見的，畢竟是空投武器。

3、射手座

射手座的專屬吃雞武器是VSS射手步槍。對於射手座這樣活潑好動的星座，一款簡單的自帶倍鏡的武器還是特別的實用。雖然這款武器單發傷害低，但是卻自帶消音器，能讓人不輕易就發現你。

4、獅子座

獅子座的專屬吃雞武器是AWM狙擊。對於我們霸氣的獅子座來說，他們是特別適合這款空投武器。這款有著最強的單發栓動的狙擊。特別適合獅子座這樣的很有王者風範的人。

5、巨蟹座

巨蟹座的專屬吃雞武器是M416突擊。巨蟹座的人給人一種溫柔體貼的感覺，這款M416是一款外型美觀，穩定性很強，還能裝備很多的配件，這樣一款武器最是適合巨蟹座這樣柔和的人。

6、處女座

處女座的專屬吃雞武器是大砍刀，這款有著防身作用的砍刀最適合處女座這樣的人。如果有人要欺負你，拿上砍刀，揍你。處女座比較是小心翼翼的人，這樣一款防身武器可謂是特別的實用。

7、金牛座

金牛座的專屬吃雞武器是信號槍。金牛座的人老實穩重，給人一種特別成熟的感覺，這時候的他需要一把信號槍的支持，信號槍能夠召喚空投，對於金牛座的人來說那真的是超開心，有著這樣的武器。

8、天蠍座

天蠍座的專屬吃雞武器是98K。天蠍座的人給人一種腹黑的感覺，這個時候就需要一把98K配上8倍鏡，躲在隱蔽的地方可以來個突襲。98K是能夠一擊摧毀二級頭，特別適合遠距離的操作。

9、天秤座

天秤座的專屬吃雞武器是AUG步槍。對於天秤座的人來說，他們是比較有正義感，是喜歡維護世界和平的，這款空投武器，是一款幾乎沒有什麼明顯缺點的武器，這樣一款完美的武器特別的適合天秤座的人。

10、摩羯座

摩羯座的專屬吃雞武器是Mk14。對於摩羯這樣冷靜執著的星座，他們是特別有忍耐性的，而且還是特別有責任感。這款Mk14也是一款威力很大，單發傷害高的空投武器。

11、水瓶座

水瓶座的專屬吃雞武器是S12K霰彈，這款容量比較低的武器，在近距離發射上還是有著很強的威力。這樣一款武器適合水瓶座這樣有著高情商和高智商的星座。

12、雙子座

雙子座的專屬吃雞武器是平底鍋。個性搞怪的雙子座，平底鍋在吃雞中可是任何子彈都無法擊穿他的，真的是比三級甲還厲害，傳說中的四級甲。這樣的武器正是適合機靈古怪的雙子座。
