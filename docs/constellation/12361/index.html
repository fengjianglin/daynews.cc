智商高但很晚熟，後知後覺的星座，一旦醒悟，會有一番作為

2019-12-11 1,742條評論 3,760次閱讀 889人點贊

每個人都有自己的人生軌跡，有些人從小就能敏銳地了解社會規則，他們的學習成績也許不好，但精於世故，在人際關係上混得很開，他們知道自己要的是什麼。而一些人心智晚熟，雖然智商高，但總是後知後覺，容易陷入不切實際的幻想當中。雖然成長的速度較慢，但一旦醒悟，定會有一番作為。今天小編為大家帶來的話題是智商高但很晚熟，後知後覺的星座，一旦醒悟，會有一番作為。

首先我們分析的是射手座。射手座熱衷於自我空間，討厭被管制。但他們說話方式又很耿直，不懂得委婉之道。所以射手座是令人又愛又恨的。雖然嘴毒，但他們的智商很高，能看清事實真相。但問題來了，射手座屬於能讀懂人心，了解規則，但卻不安於現實，喜歡和實際情況反著來。不少射手座的語言天賦很高，而且具備高遠的格局觀。但射手座很晚熟，他們過於理想化，凡事愛隨心所欲。不過射手座一旦醒悟，會激發他們的正面能量，因為射手座的悟性很強，成熟之後，會有一番作為。智商高但很晚熟，後知後覺的射手座，一旦醒悟，會有一番作為。

接著我們分析的是水瓶座。水瓶座很自我，只不過他們的自我意識不容易被人發覺。可是如果你想試圖強迫水瓶座是做一件他們不愛做的事情，或者經常否定水瓶座，那麼你會看到一個叛逆固執的水瓶座，哪怕他們知道自己不對，也會跟你唱反調，但不會過於張揚，這也是水瓶座晚熟的表現之一。論智商，水瓶座是很出色的，他們的奇思妙想很多，而且能夠把兩個不相關的事物結合起來，創意十足。

但水瓶座也由於太關注內心世界，而忽略了外界的變化，也就是說水瓶座是不太喜歡主動去和他人打交道的。水瓶座的後知後覺，不在於他們領悟不了，而是懶得理會。他們會下意識地把自己和其他人隔開，但水瓶座靈氣逼人，一旦醒悟，容易令人刮目相看。所以水瓶座也容易擁有突如其來的好運氣，成就一番作為的概率很大。智商高但很晚熟，後知後覺的水瓶座，一旦醒悟，會有一番作為。

最後我們分析的是雙子座。雙子座狀態很不穩定，也許前一秒對你笑嘻嘻，後一秒就對你冷臉。這和雙子座內心戲豐富，關注力常常隨著情緒所轉的原因有關，所以雙子座會出現晚熟的特點，他們可以是人來瘋，也可以是細膩敏感的「林妹妹」。雙子座的思維敏捷，反應靈活。他們擅長迅速接收和傳播信息。但由於雙子座的變動感太強，他們的沉著度不夠，厭煩一層不變，枯燥乏味的生活，所以雙子座會不停地尋覓適合自己的方向。但他們一旦醒悟，穩重度提升後，往往會有一番作為，因為雙子座學東西很快，而且觀察力和分析力都很出色。智商高但很晚熟，後知後覺的雙子座，一旦醒悟，會有一番作為。
