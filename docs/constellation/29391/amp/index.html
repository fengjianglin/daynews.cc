金牛座

未來兩周，金牛座的朋友衝剋太歲，各方面的運勢也會受到影響，在上周的時候運勢低迷，自己也比較迷茫，幸好衝剋太歲影響較小，只要自己兢兢業業的工作還是會有不少的收入，尤其是在未來兩周，他們的運勢會有所好轉，在年底之前迎來新一波的上漲運勢，不僅能順利擊退頹勢，而且生財有道，能夠在職場正財上獲利巨大。

獅子座

獅子座的人生性成熟，聰明伶俐，並且為人靠譜，行事相當穩健。不過無奈在最近數月里遭受到惡星的破壞，財運下降十分嚴重，明顯阻礙了前景的發展，好生無奈；未來兩周，人緣好了之後，貴人就出現了，不管獅子座們準備做什麼項目都能得到成功，到了年底還會有更大的收穫。

巨蟹座
