樸素實在，愛藏拙的星座，外愚內智，一點都不簡單

天天要聞
2019-12-22

身邊有不少人，外表看起來真的很樸素實在，根本就沒有什麼亮點。但如果相處下來，這類人的優點就會一個個地冒出頭，一點都不簡單。這也說明了有些人屬於外愚內智的類型，不是不聰明，而是愛藏拙，知道木秀於林的隱患。這才是大智慧呀！那麼，像這種類型的星座又有哪些呢？一起來了解了解吧！

對於藏拙這件事，摩羯座一直都很擅長。有很多人也是被他們樸素實在的外表給騙了的。他們生來愛忍耐，如果有賺錢機會，也是悶聲發財的那種。摩羯座只要不說，其他人是很難從他們的言行中看破的。而摩羯座也不在乎別人怎麼看自己。只要能夠得到實際的利益就行。而摩羯座也深知過於張揚的負面效果。所以他們總是會默默地提升自己的實力。

巨蟹座身上一直都帶有摩羯座的影子，也就是說，這兩個星座有很多相似之處。巨蟹座溫柔多情，但在關鍵事情上，他們可一點都不含糊，該精明的時候絕對不會犯傻。而樸素實在的形象也給巨蟹座在人際交往上增加了很多分數。很多人也是被巨蟹座的柔情體貼所打動，但事實上，巨蟹座活得通透，但不愛把話說破。當然也不喜歡把自己暴露在外。

所以，這個清純文雅的星座，並沒有想像中的弱勢和無助，相反巨蟹座常常會偷偷地制定計劃，韜光養晦，在不知不覺中一步步爬上高位。而且，巨蟹座也具有當管理者的潛力，有不少巨蟹座堅韌有毅力，能夠抗住重壓，最終走向成功。在隱忍度方面，很多人會想到摩羯座和天蠍座，但其實巨蟹座也是一個被低估了的高手，一點都不簡單。

金牛座是個很懂得藏拙和偽裝的一個星座，也是最容易讓人意想不到的星座之一。如果你深入了解了金牛座的本心之後，再看看他們樸素實在的外表，會驚嘆於他們把自己藏得異常之深。不是他們刻意為之，而是金牛座本身就喜歡低調，不愛張揚，同時也能忍耐許多負面的情緒。金牛座不發脾氣，不是沒脾氣，而是他們在剋制自己。

他們不喜歡爭辯，不是看不清你的人品，而是懶得去撕破這層紙。外愚內智說的就是金牛座，他們其實很通透，也懂得默默努力去爭取一切，但金牛座就是不願意把真實的自己展現出來。而其他人看到的只有憨厚和遲鈍，殊不知金牛座一點都不簡單。如果一個金牛座突然暴怒，一定是忍耐了太久。金牛座並不像人們眼裡看到的單純呆傻，他們也有靈活的頭腦和計謀。
