最樂觀的星座，積極調控消極情緒，對生活充滿信心，有鬥志求上進

2019-12-22 1,553條評論 4,294次閱讀 1323人點贊

樂觀主義者從每一個災難中看到機遇，而悲觀主義者只會從每一個機遇中看到災難。都說「世上本沒有絕境，只有對絕境產生絕望的心」，有時候心態主導著人的意志，積極的人能絕處逢生，在逆境中看到希望的光芒，而悲觀的人卻順應天命，早早地放棄了最後一絲希望。最樂觀的星座，積極調控消極情緒，對生活充滿信心，有鬥志求上進。哪個星座最積極樂觀有動力，遇事也能自我消化調控，不會輕易退縮。

射手座：陽光洒脫

很多時候，射手座在其他星座的眼中都是自由自在、不受管束、略帶強烈自我風格的星座，他們從不在乎別人的看法，為人處世非常洒脫，喜歡隨性而為。在射手座的觀念里，自己開心就是最重要的事情。遇到有人給自己添堵，射手座從不手軟，愛憎分明的性格讓他們行事簡單，自己喜歡的就寵著，自己不喜歡的就直接忽略，而侵犯自己利益的，則不需要考慮直接用行動讓對方後悔招惹自己。在射手座這裡，如果落入深谷，與其自怨自艾、埋怨消極，不如尋找走出低谷的方法，儘早脫離這一負面情緒。他們擅長用陽光洒脫的性格直面困境，積極消化勇往直前。

獅子座：自信達觀

如果說獅子座的優點是自信的話，那麼他們的缺點有時候也恰好是過分自信，但是不可否認，自信的態度讓獅子座在面對困難時選擇勇敢面對，有初生牛犢不怕虎的衝勁。很多時候，獅子座總是沖在第一線，就像狩獵中的萬獸之王，從不退縮。而消極的情緒、負面的情況，獅子座也早已學會了消化和接受，只能勇敢的直面，哪怕用血和淚換回成果，獅子座也是在所不惜，因為驕傲的自尊心不允許他們「害怕」。面對消極的情緒，獅子座會自我說服，給自己加油鼓勁，增加信心。畢竟在自信心爆表的獅子座看來，這都不是事，是事就一陣，一陣就一會，一會就完事！

白羊座：豁達樂觀

很多人對於白羊座的印象永遠是樂天派，天不怕地不怕地面對所有的事情，在遇到瓶頸和低谷時也能夠微笑面對，而且大大咧咧的性格也能豁達地面對人和事。其實白羊座是非常不耐壓的星座，在單純的他們看來，世界本應該是純粹的，面對生活中的人情往來、職場上的爾虞我詐，白羊座常常不知所措，在高壓和負能量下也會軟弱哭泣。但是白羊座又像小草，有春風吹又生的耐力，即使被壓彎，改天又可以樂觀應對。白羊座的智慧就是這樣，開心也是一天，不開心也是一天，那麼為什麼不選擇開心面對呢，所有的事情都有過去的那一天，積極面對無愧於心就好。
