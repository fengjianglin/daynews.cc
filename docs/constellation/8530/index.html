11月已經過半了，你心中還有什麼煩惱疑慮的，進來給你答案

2019-11-16 951條評論 4,133次閱讀 3041人點贊

塔羅可以占卜些什麼：

  • 感情：解答暗戀、熱戀、異地戀、失戀複合等等的感情疑惑
  • 緣分：測你的正緣會是一個怎樣的人、什麼時候在哪裡相遇
  • 桃花運：測你未來一段時間的桃花運勢如何
  • 事業：測你事業的發展情況、工作狀態
  • 生活：生活上各方面的解惑

占卜規則：

  1. 排除雜念，保持內心的平靜
  2. 心裡默念你想要占卜的問題
  3. 點擊這裡即可開始占卜

塔羅小知識：

塔羅牌占卜不是迷信，它只是一種工具，所謂的迷信只是某些人自己的心。

迷信，是偏執的認為占卜結果不可變，人的主觀能動性對於事件發展沒有影響，被動等待預言的結果實現。

事實上，塔羅不是宿命論，占卜的結果因為我們行為或心態變化而發生改變，所以，並不是你對占卜的結果無能為力。

塔羅占卜的原理是根據目前現狀，推測出的未來的發展情況，也就是說，如果一個人的想法和行動變了，那麼，結果很可能會變化。所以，占卜的作用就是通過預測的結果讓我們對目前現狀中存在的問題有所覺知，然後做出調整，最終目的是趨利避害。

近日星象：

水星火星紛紛換座，使得平和兼聽的交流氛圍改變，天蠍式的洞察與深刻的情感互動將融入人們日常的談話氛圍中。這段時間真相對你來說是極其重要的，擁有確信的信念是你所謀求的，一些暗中的詭計與想法更容易產生，揣測與保留的談話氛圍可能時常出現在生活中。神秘學，研究背後的動機與一些懸疑偵探類題材的作品此時會引發人們的關注。而行事衝動與戾氣也將驟減，禮貌和權衡將會主導這段時間人們的行事風格。這不是一段適合冒險、開創的實際，相反比起勇往直前的個人主義，人與人之間的協作性被提升了。你可能更多考慮自己在社交場合是不是合乎禮儀，是否有冒犯到他人，戀愛與約會是否有毛毛躁躁。理智牽制著你的行為，卻也會帶來猶豫不決，可能導致辦事效率不佳，缺乏果斷。
