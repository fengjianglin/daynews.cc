射手討厭另一半做什麼工作？要是有你的職業，就快換份工作吧

天天要聞
21 小時 ago

射手談戀愛非常需要另一半的呵護，另一半如果一天不寵愛自己，射手一天的心情就會不好。

所以和射手談戀愛，你要花時間陪伴射手。如果你的職業是這三種，就會讓射手感到苦惱，想你的時候卻找不到你。

下面我們就一起來看看是哪些工作吧！

第一：空乘

空乘也是航空乘務員，如果射手另一半是做空乘的工作，就會很少見面。由於工作的原因，可能和射手的相處機會是很少的。

射手是很需要用心陪伴的，有時射手也想撒嬌，但當射手想對你撒嬌的時候，而你卻在飛機忙裡忙外。再加上空乘這個職業在飛行的時候是不能玩手機的，所以射手就會經常想念另一半。

可是光有想念又有什麼用呢？

因此，射手座是很討厭另一半做空乘的，再加上做空乘的很難給射手安全感，一出門就是好幾天才回家。

第二：銷售

銷售這個職業分很多種，但無論哪種射手都不喜歡銷售當男友。先不說銷售工作忙不忙，就說他的工作很多人就不喜歡。

主要是大多數銷售油嘴滑舌，這會讓射手感覺很沒有安全感，再加上交的朋友都是魚龍混雜的。

所以銷售這個職業是很多射手都討厭的，現在到了年底，還經常會需要應酬，自然而然就沒有時間陪伴自己。

因此，射手討厭另一半做銷售的，射手對待感情是很需要呵護的，不然射手談戀愛的意義是什麼。

其實射手也像小孩，你對射手要有耐心，銷售這個職業會容易忽視射手的感受。要是你是這個職業，就快換份工作吧，不然射手寶寶遲早被你氣走。

第三：模特

雖然模特這個職業的人顏值都很高，但是很難給射手安全感。在顏值方面，肯定是符合射手的要求，但是模特接觸的人比較多，而且大多數都是長得很好看的。

而射手會為自己的顏值堪憂，射手不希望自己的愛情出半點差錯。所以當你的職業是模特，射手內心會感到不安。

射手對愛情特別缺乏安全感，再加上射手·內心單純，一旦受到傷害，就很難再癒合傷口，在射手內心深處，就會留下「傷疤」。

總而言之，射手討厭以上這三個職業的人是愛人，如果你是這些職業，就需要抽空來陪伴射手，和他一起周遊四海。

其實射手對待感情非常認真，一旦射手和你在一起，就意味著射手只對你專一，你該好好珍惜射手。

再加上射手又是火象星座，從小就喜歡自由，一旦你辜負了射手，射手就會轉身離你而去。儘管射手很樂觀，但是射手也是有底線的，你不愛射手，還有很多人想追求射手寶寶。
