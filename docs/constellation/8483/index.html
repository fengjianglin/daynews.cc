——雙子座——

11月25日，天蠍座火星對分金牛座天王星，你在工作中可能遇到意外狀況；可能因為朋友或交際關係影響到目前的工作前景，需要注意休息，小心積勞成疾。

11月26日，金星進入摩羯座，接下來你會更加享受兩性關係，享受肉體的歡愉；有可能獲得意外的好處；謹慎的理財投資可能會讓你獲得回報；但戀愛方面如果太過衝動可能會招致麻煩。

11月28日，天蠍座水星三合雙魚座海王星，雖然目前麻煩不斷，但你對工作事業依然充滿嚮往；這期間有利於慈善、藝術行業的工作。

11月29日，摩羯座金星三合金牛座天王星，學業或未來方面不安的積累，可能會給你的戀情帶來壓力，甚至造成兩人疏離；但有機會在學業獲得幫助或意外好處。

11月30日，天蠍座水星六合摩羯座土星，你內心對目前的工作有所不滿或者抗拒，可能在工作方面會遇到阻礙。在此期間可能接觸到投資或者錢財方面的工作；注意身上的老毛病。

——天秤座——

11月25日，天蠍座火星對分金牛座天王星，你在婚姻、合作方面的問題可能導致財務出現狀況甚至損失；你也可能因為錢財問題與戀人發生衝突；這期間不利於投資。

11月26日，金星進入摩羯座，接下來你可能在家中花費更多時間，會更享受家庭時光，享受與家人相處的平淡點滴；你有機會在家中從事投資活動；可能通過家庭或房產獲得好處。

11月28日，天蠍座水星三合雙魚座海王星，你會制定未來財務計劃，可能在健康方面有所花費。需要注意口腔、飲食問題。

11月29日，摩羯座金星三合金牛座天王星，你在投資方面將有所收穫，在遊戲娛樂方面會獲得好處；有利於戀人與家人的相處。但即使在家中也要注意安全。

11月30日，天蠍座水星六合摩羯座土星，你在財務方面可能受到來自家庭的壓力，家人可能監督你的理財計劃。

——水瓶座——

11月25日，天蠍座火星對分金牛座天王星，你與家人在事業上意見不一致，可能會發生衝突；事業上可能出現問題，與上司交流時也可能會出現狀況；小心與人爭執，注意出行安全。

11月26日，隨著金星進入摩羯座，你可能慢慢停下生活腳步，不斷放鬆身心，企圖從內心獲得安寧，達到平和；可能想要通過內心去探索整個世界；你有更多機會遠行，也可能參與更多的家庭活動。

11月28日，天蠍座水星三合雙魚座海王星，你可能會對賺錢更感興趣，有機會從長輩或上級那裡得到財務方面的建議，有利於創意、遊戲行業，但亦有可能在事業上損失錢財。

11月29日，摩羯座金星三合金牛座天王星，家中可能出現變故，學業可能受到影響，你可能需要遠行，也可能會與家人分隔兩地。

11月30日，天蠍座水星六合摩羯座土星，不利於戀愛、玩樂，你需要在投資、事業或與上司溝通方面更加慎重，事業方面容易出現問題。
