A：選擇這個證明你這一生最大的福氣就是你的事業，你其實有一份你很喜歡並且也不錯工作，雖然你對自己平時的生活要求並不是很高，但是你是一個很有責任的人，你會將自己的工作做一個規劃，並且會花費心思去好好完成。這樣的舉動不僅讓那個同事們對你很欽佩，就連你的領導也很看重你。你覺得接下來的每一步對自己來說都是一種經歷，你也會很認真的去對待，你在生活中是一個心態很好的人，而你有所成就的原因也是這個。平時的你就算不專門去打扮，也會給人一種很精緻的感覺，這正是你的魅力。你很在乎自己的生活是不是充實，你的氣質會顯得你很有靈氣，所以你身邊的愛慕者應該也很多。

B：你本身的善良就是你的福氣，在生活中，你是一個很喜歡去幫助別人的人，你或許會為了一個很可憐但是和你有沒有關係的人去流淚，而且如果你的朋友需要你去幫助他，你會很儘力的。當然在，正是因為你這樣的善良讓你的福氣一直都在伴隨著你，你知道現在可以說是很幸運的。在平時的交往中，你很會觀察別人的心思，並且說出的話也很合時宜。你重視和自己的關係很親密的人，因此你的交友圈相對來說比較小一些。

C：選擇這個衣帽間證明你的父母就是你最大的福氣，他們能夠一直幫助你，在你的生活中遇到問題的時候也會想辦法幫你解決。其實不管你現在在哪裡，他們都會很擔心你，當然，你做的決定他們也會默默的支持著你。所以，你更應該去感恩他們，雖然你總是給人一種很高冷的感覺，但是在面對他們的時候，還是要和善一些。你不是一個很在乎功名的人，所以你過得很簡單，你也很享受。

D：你最大的福氣是你的婚姻，你遇到的那個人和你真的很合適，並且你們之間會很幸福。即便是你們會有爭吵和矛盾，但是你們能夠互相的理解，在工作中的事情也能夠向對方詢問一些意見，這是很好的一種狀態。你們兩個都有一種責任感，之後會陪著你們的孩子一起成長。
