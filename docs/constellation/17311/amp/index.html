前言

在過去的一個星期裡面，有多少人因為別人的言語而被傷害過？一句話，可以成就一個人，卻也可以毀掉一個人。我們會因為一句話被治癒，卻也會因為一句話而被深深傷害。傷口，並不會因為時間的流逝而徹底消失，我們總是用麻痹的神經來假裝無視那地方留下抹不去的疤痕。漸漸地，我們變得可以小心翼翼，失去了冒險的勇氣。

16日，是一個容易收穫好消息的位置。木星的幸運加上天王的不確定性，讓我們充滿了期待，有些人會突然接受令你意想不到的好消息。目前，木星正在摩羯座運行，為此，這個信息往往會與職場、薪資、收入有關。有些人之前辛苦努力，終於獲得了屬於他的報酬。有些人則長期無工作，卻突然接到了有人請他做兼職賺錢等等。不要用自己的想像去猜測可能會發生什麼，調皮的天王星是不會給你有猜中的機會。

這是一個適合變革、推動新項目以及規則的位置。木星過於的自信感，讓我們每個人都像個充滿冒險精神的勇者。為此，一些跨界機構的合併、分離、收購等信息的出現，會讓我們看到許多不同的企業方針。但是，由於受到木星摩羯座的影響，我們的工作也會變得更加忙碌起來，做的越多，收入也自然也多，可是我們的消費卻也明顯增加。各種年會、年貨的準備。有些人更是會大批量的購買電子產品 、奢侈用品、遊戲設備等等。都容易成為你們消費的首選。

各種額外的消費將會貫穿整個一周，越來越多的人會花錢如流水，各種非計劃外的消費常常佔據著我們。對於自己創業或是自媒體的人來說，這是一個能者多勞，多勞多得的時機。如果你的工作與傳媒、媒體、出版有關的話，那麼本周會額外忙碌。因此，如何合理的安排時間，避免造成拖延，是大家需要考慮的問題。

人一旦忙碌起來，就想要放假，這是普遍的惰性。本周，這將加大外出與離去的特徵。有些會開始計劃日後的旅遊與出行。機票、車票的購買，制定旅行計劃、想要去海邊走一走，看一看。這樣的情懷在許多人的腦海中都容易出現。說走就走的旅行，廣闊的田野、碧藍色的海洋，每個人的身心都得到放鬆。但是，這也將加大異地戀、冷戰的風險。

19日，火星六合土星，原本停滯不前的項目獲得了突破性的進展，一些新的想法以及創意會獲得來自管理層的認可以及推進。同樣，火星的攻擊性會讓許多的結構、架構、框架崩塌。為此，我們需要小心各地的塌陷、房屋倒塌、腳手架掉落等。

一些與牙齒、骨骼、脊椎、發燒方面的健康問題，需要我們注意。有些人會有拔牙、針灸、推拿、拔火罐的情況。

20日，金星進入水瓶座，情感方面的一見鍾情和令人意想不到的相遇。但，冷戰、異地戀、分分合合的情況糾結著人們的思緒。有人為了追求情感里的自由而選擇分手的人將大有人在。無論我們增加多麼的相愛，都會想要保持一定的距離，避免過於親近。往好的方面來看，這也是一個適合創造浪漫，製造驚喜的一個位置。為此，許多人會在這個時候願意給自己的另一半一個驚喜。求婚、告白、脫單等容易在本周上演。水瓶座的不按常規性，也會讓我們在這段時間裡，看到許多令人意想不到，打破常規的戀愛關係。

從金錢方面來看，這段時間，人們的消費變得更突發奇想，總是更願意去嘗試一些曾經未曾購買過的用品。去參加新品的發布會。買了退，退了又買，反反覆復，讓商家也有些摸不著頭腦。其中化妝品、美妝、護膚、食品都會是許多人的首選。一些曾經並不愛護膚的人，也會變得愛美起來，想要改變一下自己的造型，或是努力護膚、化妝起來。對於有投資計劃的朋友來說，則需要小心大起大落的情況。

同日，水星刑克海王，我們變得更加粗心，忽略細節，也更容易被欺騙。物品丟失、被偷、信息誤刪除等給大家的工作和生活帶來麻煩。對於那些喜歡說謊的人來說，想要圓一個謊就要用另一個謊去圓。這段時間，我們需要小心從一個坑掉入另一坑，連環的套路，總是會讓人防不勝防。為此，各種垃圾簡訊，推銷電話的不斷，讓我們感到心煩。這幾天，請盡量保持警醒。

22日，太陽正式進入摩羯座，整個傳媒、宣傳、導向都會由之前的不負責任的誇大，轉變成了實事求是。講究證據，這也是一個容易有維權、要求給出說法的一個時間。為此，一些管理層不得不需要出面來解決問題。通告、聲明的出現，讓我們看到了事情的進展。不過，往好的地方來看，這也是一個公布成果、容易出領域領袖的位置，所以，以頒獎典禮、獎金髮放等形式來認可大家這一年的努力，是摩羯座的課題。

上周開始，有多少瓶子收到了來自自己前公司領導的信息呢？對方是否想邀請你用業餘時間來幫忙完成某個項目呢？但是，前提是你需要為此進行保密。這裡仍然請瓶子們注意，儘可能的按照實際收費，不然你會發現，這個項目會異常的繁瑣，甚至需要反覆修改。當你答應之後再提加錢，容易造成你們之間關係的問題。

日常生活方面，你們會收到來自與房子有關的事情，其中以合同方式呈現的情況尤為明顯，不用擔心，這是一份對你有利的合約。日常生活中，我們的瓶子會開始為自己的造型而額外上心。只是，你是否真的決定要嘗試那些曾經不曾嘗試的造型嗎？你是否真的可以駕馭呢？小部分人會有女生買男裝穿，男生使用女生的護膚用品等。

感情，彼此扶持，心意相通。

PISCES 雙魚座

2.19~3.20

最近這段時間，你是否明顯感受到來自同事、朋友之間的互動明顯增多。許多不聯繫的人現在也開始活躍在你的聊天群里。你們會開始為了一個共同的話題開始探討。小部分人更是會遠程進行與法務相關的諮詢。 不但與朋友之間的溝通增加，日常工作上的溝通也明顯提升，只是常常找不到對方的你，讓你很是抓狂。

感情，攜手同游風土城市，領略不同文化。
