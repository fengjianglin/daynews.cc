　　導語：年底職場人士總會分外忙碌，如果運勢穩定順暢忙碌一點，辛苦一點也無所謂，但要是運勢不太好，就需要分外小心了，以免雞飛蛋打，白忙一場。今年年底，有些星座確實可能會經歷職場風波，一起來看看哪些星座需要面對諸多的職場風波吧。

年底職場風波多，需小心謹慎的星座

　　TOP4：白羊座

　　這個年底，白羊座會面臨很多職場上的問題，尤其容易遭到小人陷害。表面上看起來像是機會的挑戰，其實並不值得去爭取，更不應該為此浪費太多的時間和精力，即使全心全意去處理其中的負面狀況，也會招來流言蜚語，還會被推卸責任，被迫承擔不應該由自己承擔的事情，最後出力不討好。

　　TOP3：雙魚座

　　這個年底，雙魚座有可能會登上裁員榜，感覺到苗頭不對，雖然雙魚自己也很想通過個人努力來改變狀況，但事實上不管做什麼都是徒勞，外界的環境不斷在發生變化，常常會令雙魚有一種身不由己的危機感，神經脆弱，性格怯懦的雙魚，在這種時候很容易崩潰，既然事情的發展由不得自己調整，心情比較重要。

　　TOP2：天蠍座

　　這個年底，天蠍座可能會遇到對自己的事業產生阻礙的小人，對方會給天蠍座帶來表面看起來像是機會，實際上卻是陷阱的坑，在誘惑面前，天蠍座一定要睜大眼睛看清楚，不要因為貪圖一時的利益，掉入小人的陷阱。在職場上也別傳播流言蜚語，否則最後自己會成為無辜的背鍋俠。

　　TOP1：天秤座

　　這個年底，天秤座需要發揮自己的社交才能，但仍然搞不定職場上的人際關係，並不是因為外界的環境太複雜，而是天秤座缺乏一點技巧，可以做老好人，但在非常時期，和稀泥的做法得不到他人的認同，反而會把事情搞得很亂，如果事情跟自己無關，最好採取高高掛起的態度，不要去參與，這樣對職場人際有好處。

　　結語：上榜的這幾個星座最容易經歷職場風波，別太擔心，只需要小心謹慎就能完美避開各種風波，偶爾給自己放鬆的機會，別太計較職場得失，可能在職場中度過的每一天都會比較舒適，年底的心情自然會好起來。
