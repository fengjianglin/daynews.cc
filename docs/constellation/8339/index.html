天秤座如何克服自己的選擇困難症？6招讓你從此告別猶豫不決

2019-11-24 353條評論 1,291次閱讀 258人點贊

1、減少選擇項。

可供選擇的項目越多，天秤座就越容易陷入迷茫。比如，一個漂亮的天秤座女子，身邊一定也有很多的追求者，自己到底該選哪一個？這可愁壞了我們這位天秤女。相反，一個長得丑的女子，因為沒什麼男人可選，反而會較快做出選擇。

減少可選擇項目，可以採用「評分法」或者「一票否決法」。前者是對可能的選擇項進行打分，從中選出最優。後者，則是去除那些明顯的不符合入選標準的選項，比如，那些吸煙的男人不嫁，喜歡賭博，喜歡喝酒的不嫁。

2、減少評分因素。

採用評分法的時候，切忌給自己太多的評分因素。比如，選對象要求身高多少，體重多少，工作收入如何，有沒有房子，家裡公婆如何，有沒有兄弟姐妹，文憑多高，朋友多少，是否經常出差等等。因為，太多的評分項目，只會讓你無所適從。

不過，追求完美的天秤座，還是喜歡事無巨細地逐一打分，甚至在選擇餐館這種小事上，這就太浪費時間了。建議你只給自己5-7個評分因素，並按照重要性列下來。比如，選擇男朋友，第一重要，就是是否愛我？第二，是有沒有上進心？第三，工作是否穩定或者有發展潛力？評分的因素盡量不超過7個，最後，以綜合總分最高的為準。

3、限定最終截止期限。

不論何種決策，都要給自己一個截止期限。期限之前，你可以各種考慮；期限一到就立馬拍板，不要再猶豫了。

天秤座最大的問題，就是做選擇或決定，永遠不給自己截止日。這樣一來，他們每天都可能推倒之前的認定，當然無法做出決定。所以，你需要給自己一個最終期限，不能一直拖下去了！

4、坦然接受最不利的後果。

可是，假如我做錯了決定，怎麼辦呢？天秤座的你，或許會這麼問。其實，很多決定或者選擇的後果，真沒你想的那麼恐怖。比如，在這個餐館吃飯，和在那個餐館吃飯，會有多大的區別？好，就算你這次吃得不愉快，你大不了下次換一家，不就得了嗎？

問一下自己：如果我做錯了選擇，最壞的後果是什麼？在大多數的情況下，做錯選擇也不是世界末日！結錯了婚，大不了就離婚嘛，有什麼啊？找錯工作，那就換一個啊，又死不了！怕什麼？敢於試錯，坦然接受最壞的結果，你才能變得堅強自信果斷起來！

5、讓他人為自己做決定。

如果你對自己還是沒自信，那就請別人幫你做決定吧！不過，你一定不能後悔啊！

讓他人做決定，這並不是最好的做法。可是，如果天秤座的你已經陷入了嚴重的選擇困難症，那你還是這麼做吧！你應該找一個你信賴的人，讓他幫你做出決定。但是，一旦對方為你做了選擇，你就不要在一旁嘀嘀咕咕的了。學會帶著感恩的心，接受他人為你做的選擇吧，或許，那就是天意！

6、抓鬮。

如果連他人都沒法信賴的話，那好吧，你就抓鬮，聽天由命吧！不過，這其實也沒啥技術含量的，事實上，我們的大多數決定，都沒有絕對的好壞之分，而且都是可以改變的。既然你怎麼也做不出決定，那就只有走這條路了，我也只能祝你好運！
