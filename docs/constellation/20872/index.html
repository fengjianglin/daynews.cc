冬至起，相思煎熬，恰逢桃花，有緣相遇，真情不改，共度餘生

2019-12-16 1,677條評論 3,765次閱讀 1126人點贊

#情感##複合#有一句話說：「人的一生會遇到兩個人，一個驚艷了時光，一個溫柔了歲月。」驚艷了時光的那個人，是青春回憶里最絢爛、最耀眼的存在，不後悔跟他經歷過的快樂與感動，哪怕後來的大風大浪都是他給的，但還是想對他說，有生之年，欣喜相逢。

#射手座#

射手座的人雖然愛好自由，可是他們一旦接觸了愛情，將會很重視，不會輕易放棄這一段感情，同時也會繼續將自己的目標完成。但有時候，愛人與目標很難同時兼得，射手座更多的時候會選擇目標，而忽略愛人。長久以來，愛人因為射手座的忽視而感到心冷，從而離開射手座。

冬至起，相思煎熬，恰逢桃花，射手座與前任再次牽手，永不分離。愛人的離開讓射手座如夢初醒。悲傷了一段日子之後，射手座將決定追回前任，讓愛人再次回到自己身邊。在未來，射手座將不會再讓愛人受到冷落，將會與愛人一同打造幸福的生活。

#水瓶座#

其實在用情至深的時候，水瓶座是很難再控制自己的行為的，因為，他們明白對方就是自己認定的人，所以即便過去犯下了什麼矛盾他們並不在乎，只想與心愛的前任重新在一起。

冬至起，水瓶恰逢桃花，有緣相遇，真情不改。接下來的水瓶座愛情運勢得以逆轉，桃花運也更為旺盛，水瓶很有可能可以順著這一份是與自己心愛的人再度重逢，這個時候如果水瓶能夠好好表現自己的真心，相信他們最終能夠牽手承認再去舊情。

#天蠍座#

天蠍座的人以事業為重，永遠比同齡人更成熟，穿梭於工作和愛情之間，有時候甚至忙得忽略了對方。而天蠍座也是出於善意的他們想要給彼此一個美好的未來，讓自己更有能力去承擔責任，但因為拼搏的腳步太過快，和戀人聚少離多，最終感情出現裂痕。

在舊愛選擇離去的時候，他們會如雷驚醒，意識到自己過往的錯處，一段感情需要雙方相互維繫，想要走到長久，不單單要靠實力陪伴相當重要。冬至起，相思煎熬，恰逢桃花，紅情綠意天蠍座將會摯愛重逢，舊情復燃。
