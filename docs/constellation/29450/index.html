天蠍座

在過去幾個月不順心的天蠍寶寶們，情感和事業出現了一定的波折，但是建議天蠍座的朋友們不要泄氣，到了1月份這一切都將改變，一切都會變得十分順利。在情感上會找到自己喜歡的女生，擁有幸福的生活。工作上也會有重大的突破，會受到領導的重視，升職加薪在這個月都會發生。

摩羯座

1月對於有些星座來可能不太順利，但是對於摩羯座來說越是一個幸運月，他們在1月將會有重大的突破。在情感上很容易擺脫單身，走上幸福的愛情之路。在事業上摩羯朋友們也會順風順水，如果是在創業將會擁有賺錢的好項目，但是千萬不要衝動，要慢慢來，這樣好運氣才會持久。

獅子座

雖說三分天註定，剩下的九十七分靠打拚，但是碰到運氣好的時候，不需要怎麼努力就可以得到意外之財。所以獅子座的朋友在1月份可要好好地想一些做點啥，可能做什麼事情都是比較好運的，這樣可就賺的不止了。
