跟蔡康永學說話之道，12.16-12.22星座運勢

16日，木星三合天王星，許多人會在這天開始為日後的旅行而忙碌。購買機票、車票、制定旅行計劃、想要去海邊走一走，看一看。從事件層面上來說，一些跨界機構的合併、分離、收購等信息的出現，會讓我們看到許多不同的企業方針。

其次，由於目前木星正在摩羯座運行，我們不但工作上的任務與事情會明顯增多之外，我們需要消費的地方也會變得增多起來，各種電子產品、遊戲設備的購入，都容易成為你們消費的首選。對於自己創業或是自媒體的人來說，這是一個能者多勞，多勞多得的時機。如果你的工作與傳媒、媒體、出版有關的話，那麼本周會額外忙碌。

20日，金星進入水瓶座，首當其衝的就是，為了追求情感里的自由而選擇分手的人將大有人在。爭吵、冷戰、分手、異地戀也往往會讓我們彼此的距離越來越遠。往好的方面來看，這也是一個適合創造浪漫，製造驚喜的一個位置。許多人會在這個時候願意給自己的另一半一個驚喜。求婚、告白、脫單等容易在本周上演。水瓶座是一個打破常規、不按套路出牌的星座。因此，我們也會在這段時間裡，看到許多令人意想不到，打破常規的戀愛關係。

從金錢方面來看，這段時間，人們的花銷也會明顯增大，只是，相較於木星的位置來說，這個位置的消費金額往往會是大額的，並且是曾經不想買，而突然決定添置的那些東西。

22日，太陽正式進入摩羯座，整個傳媒、宣傳、導向都會由之前的不負責任的誇大，轉變成了實事求是，講究證據。這也是一個容易有維權、要求給出說法的一個時間。一些管理層不得不需要出面來解決問題。通告、聲明的出現，讓我們看到了事情的進展。摩羯月，是一個公布成果、容易出領域領袖的月份，所以，我們會看到各種以頒獎典禮、獎金髮放等形式來認可大家這一年的努力，是摩羯月的課題。

在過去的一個星期裡面，有多少人因為別人的言語而被傷害過？一句話，可以成就一個人，也可以毀掉一個人。我們會因為一句話被治癒，卻也會因為一句話而被深深傷害。在上周，語言、文字的力量，讓我們不容小覷。如何學會說話之道，或許蔡康永，會給我們一個啟示。

從蔡康永老師的星盤上來看，太陽雙魚座的他，有著別人都難以察覺的細膩。但是，我們都知道，一個人的言語溝通，更多的需要看他的水星星座。如果說，12星座最奇葩的一個是哪個，相信水瓶座一定會當選。被天王星所主宰的行星，天生就待著一股叛逆，但他們卻往往也是社交的高手（風象星座，信息的傳播使者）。水瓶座作為12星座的第11個星座，代表了社交、群體。所以，當水星進入到這個星座的時候，他除了可以快速找到與他人不同的觀點與看法之外，也能找到與他人之間的共性。所以，在娛樂圈，他有著眾多的朋友。

有人說，水瓶座的人是理性的，當水星進入到水瓶座時，他們可以從宏觀的角度去思考，並且用包容的態度去接受各種不同的聲音。他們忠於自己的觀點，卻不會強求你需要與他們站隊。他們理解每個人的背後都有著不同的故事，所以他們更願意傾聽別人的傾訴。這或許，也是為什麼，水星落在水瓶座的人，身邊總是有著各種不同領域的朋友吧。

本周的白羊需要注意了，你們會深刻體會到日有所思夜有所夢這句話的精髓。白天的所見所謂、內心所擔心的事情，總是會通過夢境來顯示與反饋，從而加大了你焦慮的情緒。為此，有些人會聽到周圍的人在告訴你，出去旅行，或者暫時放下工作請假吧。

對於正在參加考試或是學習的朋友來說，則會遇到會的題目不考，考的都是你之前一掃而過未曾精讀的內容，部分人更是需要注意因為粗心造成答錯題目的情況喲。

這段時間，你總是需要處理各種被拖延的文件與信息，過往幾個月的文件你不得不重新找出進行資金的對比與核對。不過好在，你的努力會獲得回報，金星進入水瓶之後，你會獲得更多額外的收入來源。

上周的雙子座滿月，你是否感受到了文字的威力呢？一個用詞不當，一句話就能觸碰你內心的柔軟與委屈。本周，如果你與開發商或供應商之間存在矛盾，需要注意，以防被欺騙或糊弄的情況出現。

當太陽進入摩羯座的時候，獅子座的你們是不是化身了拚命十三郎，想要在最後的一個月里，沖一下自己的業績。本周開始，你們手上的工作將會明顯增多，從而不得不造成你們需要熬夜加班去完成工作項目。

有時候，手上剛剛完成一個項目，新的工作又會無間隙的出現，絲毫沒有給予你喘息的機會。對於從事宣傳、推廣、廣告相關業務的朋友來說，需要避免因為粗心、錯別字引發的商務損失。

本周的處女座們需要注意了，受到海王星與水星的影響，你們在與伴侶或家人溝通的時候，容易因為忽略細節而出現牛頭不對馬嘴，溝通出現拖延等情況。現在你的伴侶可能被某件事情所欺騙，不要一味的指責而忽略了中間的細節。

日常工作中，受到火星的影響，我們容易變得暴躁易怒，尤其是在與他人溝通的時候，常常會被他人的言語冒犯。拒絕的話總是在本周頻繁出現。

工作方面，有人會接收到額外兼職或是工作的情況，多份酬勞的來源，讓你們更有資金去消費。但是，這裡也需要提醒，避免因為商家噱頭而出現被誤導或購買到假貨的情況喲。
