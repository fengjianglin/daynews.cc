你聽說沒有？煙大竟然有“十二生肖”護法

2019-11-24 250條評論 1,374次閱讀 265人點贊

嗨，Ytuers

最近的天氣娃娃也是脾氣十分暴躁呢

你可否招架住了天氣娃娃的古怪性格？

小煙最近迷戀上了性格研究

猛然發現在我們的煙園裡

竟然隱藏著十二生肖的「化身」

什麼？你不信？

那不妨跟小煙一起來看看吧！

1子鼠

生肖性格：做事態度積極、頭腦機智手腳靈巧

煙大代表物 : 大學生活動中心

舞台的特效，歌曲的回蕩

藝術的揮灑，晚會的狂歡

聽！那裡好像又要舉行校級賽事了！

它總想讓每一位煙大學子走進這裡

去敢於展示自己的才華！

2丑牛

生肖性格：勤奮努力、忠厚老實

煙大代表物 : 鐘樓

歐洲風格的建築

散發著古典而又浪漫的氣息！

有時候，感覺它就像一位老者

耐心地聽每一位煙大人

講訴自己的故事，演繹自己的人生

3寅虎

生肖性格：熱情大方、頑強自信

煙大代表物 : 體育場

一圈又一圈的跑道

我們與朋友、戀人同行

一起去追逐人生，追逐夢想

汗水與笑聲，速度與激情

他讓我們的青春在這裡花開綻放

4卯兔

星座特徵：溫柔善良、氣質高雅

煙大代表物 : 三元湖

浮香繞曲岸，圓影覆華池

年年此時，與君相約

清早的讀書聲，晚間的談心話語聲

與最親近的人漫步三元湖泮

幸福的氣息總是瀰漫心中

5辰龍

生肖性格：勇往直前、專心致志

煙大代表物 : 千米文化藝術長廊

現代建築的設計美，光影的變換

連接五學院的文化長廊

雖沒有傳統中式建築的典雅

但他卻讓孤獨、鬱悶的煙大人

留下思想碰撞的永恆懷念

6巳蛇

生肖性格：認真負責、思路敏捷

煙大代表物 : 九隆廣場、升騰廣場

初到煙大時，你會發現這裡

雖沒有樹林的靜謐，花開的清香

但卻能感受到煙大人

「海納百川，敢為人先」拼搏精神

7午馬

生肖性格：剛毅果斷、落落大方

煙大代表物 : 綜合樓

思想火花的碰撞

讓我們一次次接近真理的彼岸；

它記錄了我們的青春時光

讓我們與所愛之人每一次相遇

都見證了我們從青澀、懵懂到成熟

8未羊

生肖性格：富有創造性、有耐心

煙大代表物 : 小樹林

雖不知冷暖，但卻知道

如何給人帶來心靈的慰藉

看過斑駁的餘暉

聽過稀疏的輕啼

或許在緊張的學習之餘

這裡能夠舒緩你那顆急躁的心

9申猴

生肖性格：友善、樂觀、極強適應力

煙大代表物 : 七所學生餐廳

鍋勺噹噹的碰撞聲

掀開厚重的門帘撲鼻的香氣

每天送給你味蕾不同的體驗

還有工作人員熱心的問候

無論何時

這裡永遠有你愛的味道

10酉雞

生肖性格：腳踏實地、穩重、嚴肅

煙大代表物 : 承先圖書館

時光在寬大的玻璃窗外

漸漸消逝不見

一種靜泌怡然氣息不禁襲來！

人生的急躁功利

在這裡卻悄然無影無蹤

它教會了我們腳踏實地，無畏向前！

11戌狗

生肖性格：創意十足、智慧、獨特

煙大代表物 : 實驗室

一器一皿，都彰顯著思維的新奇

一舉一動，都流露出邏輯的嚴密

當活躍的思維碰上已知的奧秘

當清晰的邏輯遇上冰冷的儀器

盡顯青春，盡顯創意

12亥豬

生肖性格：膽大心細、仁慈、天真

煙大代表物 : 煙大東門

海邊有鬆軟的沙灘

海里有無數微小的生靈

海面上撒著明媚陽光和皚皚月色

正如他的內心

仁慈天真

看了那麼多，當然了

這些生肖性格也是小編百度到的哦

如果，你有更多的想法，

歡迎在下方評論哦！

小編邀你一起探討：

「煙大『隱藏』十二生肖」

你還在哪「見過」？
