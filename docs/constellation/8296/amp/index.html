A：

從30歲左右開始你的魅力便逐漸顯現並且會呈現一種井噴式散發，並且這之後還會持續很長很長時間。歷經磨練的你褪去了年少時的狂妄無知，多了一些理性和學識，並且在處理一些突發問題上也足以體現了你的沉穩，並不會出現頭腦發熱的決定，這樣知性理智的你使得魅力大散，追求者也會不計其數。

B：

可能你的魅力高峰來臨的日子會很遲，在60歲左右才會到達你的魅力頂峰，不過請先不要急著灰心，60歲的你並不會平靜的在家裡養老，那個年齡段你將打開你心中遺留很久的童心，老頑童的你便是你的魅力所在，你不服老，你想再年輕一回，所以追求那時的潮流便是你的生活重要部分，你不想落後於時代，可以說到了那個時候，再也沒有比你更加潮氣的同齡人了。

C：

你的魅力高峰是20歲左右。20歲，是一個黃金年齡，因為任何可能性的迸發都聚焦在這一點。處在這個年齡段的你對什麼事情都有著好奇的心，對一切都充滿著激情，想要嘗試、挑戰一切未知事物，雖說這個時候你涉世未深而且還有一點幼稚，不過這並不妨礙你散發的青春的魅力。

D：

你的魅力高峰在45歲左右。此刻的你或許已經有了屬於自己的家庭，和你的另一半幸福的生活著，而且你的孩子也已經穩定下來，不必再令你操心了，所以總體來說你的生活很是悠閑，能夠做很多想要做的事情，所以你的精神頭也會隨之變好，充滿著陽光與正能量，所以，45歲左右的你將是你的魅力巔峰。
