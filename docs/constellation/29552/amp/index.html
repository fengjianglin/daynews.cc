易霖駿 ‖談傳統意義的“數字”能量與“五行”能量

天天要聞
2019-12-20

易霖駿最年輕的易經大師

  • 手機號可以讓您的人生少走彎路？掌握數字，就能掌控自己的財富人生？以手機號的數字推算出對人生運勢的影響，易霖駿認為：這只是吸引著夢想一夜暴富，找到賺錢捷徑的「人」， 數字改運只是遊戲，非影響命運的易學！何為傳統意義上的「數字能量」？

易霖駿認為：傳統意義上的「數字能量」，其實是「五行」能量。

有關五行能量在四書五經的《尚書》有明確記載，而且中國古人根據《尚書》和《洛書》數確立了一套「五行相生相剋」對應的數字也有生克制化的完善理論體系，並應用到醫、人生及住宅等領域，但基礎是每個人的五格數理、生辰八字，喜神用神，三命通會，日干支分析，月日時命理和空間（出生地、現住宅格局）等，是專業的人士詳細分析而得出的結論。易學是於「神」的對話與橋樑，而且須要長期的心無雜念。並非單單的數字遊戲 ，需要專業人士解答易學命格。

而且所謂「幸運數字」也只是影響一個人的因素之一，並非是這些「數字能量就能決定命運、運勢甚至生死攸關。因為事物是矛盾與統一的，人也一樣，會有兩面性。這些數字里真的藏有吉凶信息，真的可以看出一個人的運勢。不用告訴個人的生辰八字，只用你的手機就能看出你的運勢，從手機可以看出你的婚姻、異性緣、情感、財運、能力、投資理財運、職業、疾病、意外、口角、血光、個性、守財密碼等等。從手機號碼就能洞悉你的運勢？

易霖駿認為如果不懂命格八字，單憑手機號碼就能判斷出對人生的利或不利，那都是臆想。

測算數字能量學，是建立在數字上 ，還是建立在本我上？只有命格才是人一生的中心，喜什麼 、忌什麼，反應的內心想法也多有不同，境物知心，從而推知出你內心的想法與秘密才是正確的，而不是玩數字遊戲大概率。同樣是8有些人使用非常好，而有些人使用就非常不利 。因為只有通過本身命格才能由喜、忌決定人生的能量。

易霖駿認為數字能量與命格更不能混為一談，數字源於易經的分支、也是代表九宮，八卦。

如數字6屬水、7屬木、8屬火、9屬金 、10屬土 等，當下很多人會用九宮圖排列，看什麼數多或數少或者缺什麼數，就開始解析人生命格也是一葉概全。

數字個性在易經裡面也多有解析，比如一個人喜歡看一中幅畫也是一樣，憂愁、喜樂，各自喜歡不同。反應的內心想法也多有不同，物也是同理。要想全方位的了解命格必須找出立極點而不是玩數字的概率不是簡單的缺什麼或者少什麼數就補什麼，而是要找出命格喜、忌而不是簡單的補數字。要看命格裡面缺什麼數字補什麼數字。而命格是跟人走的，而不是數字跟人走的。

易霖駿指出沒有明確命格喜、忌何談數字對症下藥。

所以有人問我適合什麼數字。 我會告訴他只有通過命格，找出中心點才根據你的述求找出人生路途中的運勢所在。克我這為官，我克者為財， 生我者為印。

易霖駿做為中國最年輕的易學大師，有義務為東方文化正名，堪輿學亦是中國的學問，但不能用一個簡單的數字遊戲來，來單純的解讀中國傳統文化。包括老祖宗留下來的瑰寶。……人生，應該無懼艱難，不走捷徑。
