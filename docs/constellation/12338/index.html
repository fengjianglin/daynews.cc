塔羅牌測試你目前該不該換新工作？

2019-12-12 2,473條評論 3,471次閱讀 1170人點贊

現在這份工作讓你覺得無聊和厭煩？原來的崗位沒有升遷的希望？遇到了不好相處的上司和同事怎麼辦？很多人面對這些問題的時候可能會考慮跳槽或者換一份新的工作，目前的你該不該換新工作呢？今天為大家帶來這組測試。

塔羅牌占卜規則：

1、先閉上眼睛，靜心30秒鐘。

2、在心中默默念「我目前該不該換新工作？」

3、憑直覺從下面的牌陣中選擇一張牌對應牌看答案。

注意：選好數字後不要更換，第一感覺才是最好的。

牌1：聖杯侍從正位

目前這個工作未來還會有不錯的發展，也還有很大的學習空間，建議你留下來好好磨練，這樣才能增進自己的的競爭力。

牌2：星幣十逆位

這個工作未來狀況穩定，不會有什麼新發展，喜歡穩定的朋友可以繼續待下去。

牌3：星幣六逆位

這份工作看起來有不錯的發展性，同時要評估自己的實力和野心是否配合，才不會百忙一場。

牌4：聖杯王牌正位

現在工作未來還有很大的發展空間，只是剛開始，如果能好好待下去，升遷和加薪都有機會。

牌5：寶劍王后逆位

這個工作接來還是比較多糾紛，除非你覺得自己能夠戰勝他們，否則還是儘快尋找下一份新工作。

牌6：星幣八逆位

這粉工作未來只會越來越辛苦，沒有什麼發展性，所以還不如認清自己實力，建議尋找其他發展機會。
