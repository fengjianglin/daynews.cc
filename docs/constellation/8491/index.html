《周易》分為“經”和“傳”，“經”是什麼？“傳”又是什麼？

2019-11-19 807條評論 3,637次閱讀 2668人點贊

《周易》，又稱《易經》，是我國最為古老且又十分神秘的一部經典，它是「群經之首」，是中華文化的總源頭。其實，《周易》分為《易經》和《易傳》兩個部分，《易經》簡稱為《經》，《易傳》簡稱為《傳》。

​那麼，《經》是什麼？《傳》又是什麼？它們之間有什麼樣的區別和聯繫？今天，就讓我們一起來看一看這個問題吧。

《周易》的《經》，其實指的就是「六十四卦」，即「周易六十四卦」，也叫「易經六十四卦」；當然，除了六十四個「卦象」之外，還包括「卦辭」和「爻辭」。我們知道，「易經六十四卦」的每一卦有六個「爻」，那麼，總共就有六十四乘以六，等於三百八十四個「爻」。

另外，「乾」、「坤」兩卦雖然也都是六個爻，但卻各有七個「爻辭」（乾卦的「用九」和坤卦的「用六」，沒有「爻位」，卻有「爻辭」），所以，《經》的全部陣容就是——六十四個「卦象」，加上六十四個「卦辭」，再加上三百八十六（384+2）個「爻辭」，如此而已。

關於《經》這一部分的著作者，我們普遍都認為是周文王姬昌，其實，除了姬昌之外，還有他的一個兒子——姬旦，也就是大名鼎鼎的「周公」，因為那三百八十六個「爻辭」，正是出於周公之手。當然，我們現在只說《易經》的作者是姬昌，而不提姬旦，其實也是有道理的，因為按照中國傳統的宗法觀念，既然有「老子」在場，那麼「兒子」也就別漏風頭了。

至於《周易》的《易傳》，也就是《傳》，它的出現，就要晚很多了。《傳》由《彖·上》、《彖·下》、《象·上》、《象·下》、《繫辭·上》、《繫辭·下》、《文言》、《說卦》、《卦序》和《雜卦》組成，由於它包含十篇文章，所以，《傳》又被稱為「十翼」。「翼」者，翅膀也，它的意思是說，這十篇文章就像是《易經》的翅膀，能夠讓《易經》飛得更高，飛得更遠。

關於「十翼」，據說是出自孔老夫子之手，不過，經由一些專家學者們考證，「十翼」的作者其實並非孔子，但是到底是誰，目前尚未有定論。儘管《傳》的作者不詳，但是它出現的時間，還是可以確定的，我們普遍的觀點認為，《傳》誕生於戰國時期。

《傳》誕生於戰國，而《經》則是出自周文王姬昌時代，換句話說，《傳》的出現，至少要比《經》晚了五百年以上。所以，有時候我們就會開玩笑地說：「《周易》這本書，足足寫了好幾百年，難怪它會成為經典中的經典！」

好了，以上就是《周易》的《經》和《傳》，關於什麼是《經》，什麼是《傳》，你是不是都已經清楚了呢？你還要什麼不同的看法呢？一起來留言討論吧！
