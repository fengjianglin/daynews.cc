雙子座12月運勢：不要沉浸在幻想中，給生活找一個平衡點

天天要聞
2019-12-03

2019年12月，雙子座的整體星座運勢表現大致平順，本月你在各項運勢的表現不盡相同，你需要給自己的生活找到一個平衡點，這樣才能更加順暢地走下去。

進入十二月，雙子尤其需要注意的就是不要沉浸在幻想中，只想而不去做的話是不會收穫什麼的，要知道專註於當下，才能獲得長足的發展。人際關係方面，要注意自己的真心話不要隨便和別人吐露，因為你身邊某個十分信任的人很可能不是你的真心朋友。

感情指數70%

單身雙子座2019年12月在愛情方面運勢表現大體良好，單身的雙子座願意去學習如何和自己的好感對象拉近距離，彼此互相了解。戀愛有固定另一半的雙子座會變得很依賴對方，希望能夠對方的保護。

本月在感情上要注意的就是要珍惜眼前人，不要盲目地模仿或者是羨慕別人的愛情，這樣很可能會讓你變得敏感，神經質。有伴的雙子要注意耐心地傾聽對方的脆弱和委屈，及時給予安慰和擁抱；單身的雙子桃花雖多，但是有爛桃花復甦的傾向，建議要對爛桃花及時地處理，以免對於日後的生活產生影響。

工作指數70%

在職雙子座2019年12月在工作方面運勢表現大致尚可，有工作的雙子座可能會被安排自己不適應的工作，你需要改變一下自己的工作方法。

求職中的雙子座則可能遇到和自己要求不匹配的求職機會。求學雙子座2019年12月在學習方面運勢表現大致尚佳，本月你能夠通過自己的努力在偏科方面獲得一些改善。

事業上，要注意保持耐心，本月你的工作可能會比較駁雜，需要花時間去處理，耐著性子一件一件的處理完就好；學業上，在這段時間要注意對自己所學知識及時的複習。

財運指數70%

雙子座2019年12月在財利方面財運表現大體平順，本月財利呈現平穩的狀況，無論是正財還是偏財都能夠保持收支平衡的狀態。

本月會有一些衝動消費，購買之前要考慮好物品的實用性再下手，還有要注意理財，有計劃地存錢。

幸運搭配

穿衣風格，毛呢大衣：毛呢大衣又保暖又好穿，款式多樣讓你美不勝收。破洞牛仔褲：破洞牛仔褲利落有型，搭配各類上裝都有良好的效果。

活動，剪頭髮，剪一個新髮型等於換一個造型，心情和運勢也會大不一樣。
