蘇珊米勒每周星座運勢12.23-12.29

2019-12-22 1,157條評論 4,158次閱讀 1488人點贊

白羊座

一位朋友可能會激發你對金錢的擔憂，因為本周位於你的友誼宮的金星與位於你的收入宮的天王星相衝。你可能會發現，一場社交活動所涉及的錢比你以為要花的錢還多，比如一場旅行婚禮。然而，在摩羯座的月食將與天王星相合，所以你將能夠克服這些障礙。隨著摩羯座與你的職業宮相對應，你可能會發現你的職業生活中有一種新的使命感。

金牛座

這次聖誕和新年假期你可以選擇去國外，無論是通過旅行，還是通過接觸新文學和文化。周三，一輪新月將落在摩羯座——與金牛座無縫合作的星座。這次的新月是充滿活力的，播種新鮮的種子，鼓勵鼓舞人心的冒險。摩羯座代表著你的旅行和高等學府，這意味著你可以踏上一次新的旅程，而金牛座的天王星正向這個新月傾斜，意味著你的旅程將是很有趣的。

雙子座

如果你曾尋求外部資金來源，如抵押貸款經紀公司理事會，那麼周三落在你的他人金錢宮的新月可能會激發新的申請和提案。新月將使你把重點放在設定意圖和開始項目上，而不是看到過去努力的回報。無論你現在開始什麼項目，都可能在六個月內以下一系列的新月時達到頂峰。太陽將在周五與幸運的木星相合，對外部資源支持的主題做出進一步的貢獻。

巨蟹座

隨著行星聚集在你的關係宮，意味著你可能在思考如何與配偶、最好的朋友或商業夥伴建立聯繫。本周三，新月將進一步突顯合作關係的主題。新月常常會帶來情節上的曲折，從行動的角度來看，天王星可能會帶來一些意想不到的變化。金星會在這件事上與天王星陷入僵局，可能會出現一些不安全的情況，但最終這次新月和天王星會相合，以確保這些意外的變化是有建設性的。

獅子座

本周所有的工作和娛樂都不會無趣，因為周三的新月會動搖你的工作和日常活動宮。如果你休假去度假，那麼你可能會想去臨時購物或用餐。此外，你的事業上可能有更多的變化，因為這次新月將與天王星(驚喜)在你的事業宮相合。如果你在工作中專註於解決問題的話，那麼當你攪拌蔓越莓醬或烤餅乾時，完美的解決方案可能會閃現在腦海中。

處女座

這段假期將讓你增加創意，因為周三的新月將在你的樂趣、孩子和自我表達宮閃耀。如果你有孩子，你可能會受到鼓舞去和他們一起製作卡片或薑餅屋等手工藝。或者，你可以選擇通過自己的禮物或者把時間花在你的藝術實踐上來表現自己的創意。這次新月將與天王星相合，向處女座發出理性的信號。如果你想要重置你的生活，那麼新的開始很快就會到來。

天秤座

假期可能會讓你感到財務方面很緊張，因為天王星在你的他人金錢宮，與位於你的樂趣宮的金星相衝。比如，一份禮物可能比你預想的要貴，迫使你調整你的其他支出。儘管如此，天王星將對太陽和位於你的家宅宮的新月形成一個美麗的角度。周三的新月恰巧是一次日食，所以能量將是強大的，也會激勵你和家人一起創造一些令人難忘的時刻。

天蠍座

火星通過你的身份宮，意味著你可能並不總是知道如何引導你的能量。金星和天王星之間的相衝則更會放大這種感覺，引發你的家庭和愛情生活之間的緊張。你可能會因為把你的心上人介紹給家人感到緊張，或者如果你和你的另一半住在一起，你可能會因為誰來倒垃圾而爭吵。清晰的溝通將提供解決方案，因為天王星將與周三的太陽系新月相合，以提高你的談話和傾聽技巧。

射手座

假期總是會帶來更多的開銷，而作為射手座，無論是贈送禮物還是分享美味的食物和飲料，你都知道自己是慷慨的。然而，隨著木星通過你的收入宮，意味著你也可能有機會增加你的儲蓄。本周你甚至可能看到意外收穫，因為不可預測的天王星將為周三的新月提供支持，而新月也將落在你的收入宮。隨著天王星在你的工作宮，你可能會簽呈一份新合同或得到一份很豐厚的獎金。

摩羯座

本周，你的身份宮聚集著這麼多的行星，你可能會感覺到自己身上充滿了一種幾乎是電的能量。這種感覺將在周三達到頂峰，因為新月也來到了你的身份宮。這次是非常強大的新月，所以現在是為來年定下目標的好時機。新月將與天王星和諧相處，天王星是一個充滿驚喜的星球，所以在這個節日期間，你可能會感覺到一種提升，鼓勵你去召喚並快樂地生活。

水瓶座

假期通常會讓我們能與家人共度比平常更多的時間。金星在水瓶座，意味著你有潛力與你最親近的人在一起，但她當前與天王星的相衝可能會引發一些爭執或者舊的行為模式。然而，金星是終極外交家，所以你很可能能夠優雅地處理這些問題。此外，天王星將在周三與新月相合，強調休息和自我反省的重要性。

雙魚座

由於你的友誼宮擠滿了行星，因此這個節日期間你可能會在派對上飛來飛去，你會發現社交中會帶來很多快樂——周五，幸運的木星和太陽相合，並且，令人驚訝的是，天王星將與周三的新月相合。這一次的新月會使你的社交生活恢復活力，讓你重新與舊朋友聯繫或者認識新朋友。但還是要確保你能有一些安靜的獨處時間，因為金星在你的隱居宮，可能會讓你渴望獨處。
