下周運勢｜|12.23-12.29周運（白羊/金牛運勢上揚，獅子座加油）

2019-12-21 499條評論 2,892次閱讀 796人點贊

星象概述/對大眾的影響

太陽進入摩羯座，摩羯座生日快樂哦！接下來我們的意志力更加穩定，更加具備責任感，堅持去完成一些任務，我們要的思想、人生方向更加的務實、明確。

水星在本周依然位於射手座，水星與海王星三分相，並且越來越靠近木星，水星受到木星影響很大，我們近期說話、表達可能會有些誇大的成分，思維更加的活躍、多變，我們要穩住自己，盡量客觀的評價事務。交通比較順暢，利於做一些與文書、溝通、學習方面的事情。在29日進入摩羯座，水星的能量發揮受限，到時我會再與大家說星象的意義。

上周說過金星與天王星形成刑克相位，本周在27日金星與天王星的關係解除，近期我們的感情容易遇到分分合合的情況，包括金錢、進賬不太穩定，審美的角度可能也會發生一些改變，在27日後逐漸穩定，但是我們對待感情、金錢的態度會更加理性。

火星終於不受海王星的影響，得到冥王星和土星的支持，我們的行動力與自己的想法能夠保持一致，不容易再產生腦子裡這麼想的，行為確實不能很好的配合腦子，接下里大家要努力做事哦。

運勢必看

堅持每周的運勢分析與分享，希望能夠帶給各位一些方向，預防一些問題。

而任何運勢必須基於本命盤，每個人在出生的時刻就已經有了自己的本命盤（星盤，你可以認為這是宇宙在你出生時刻給予你某種能量）。一個人如果本命盤有某種徵兆（比如容易在交通上出問題、容易得到某種好處、與哪些事務有緣），那麼在行運引動時，正好遇到了引動本命盤某種特徵的星象，那麼這種事情發生的概率就會大大增加。

所以同一時間段，同樣的星象特徵，同樣上升星座的人，有的人感受強烈，有的人不痛不癢，是因為我們每個人的「命」的其他配置不同。大眾運勢更多的是大概率可能性，希望通過對星象的了解給大家帶來一些幫助。但是與個人精確地命盤推運無法相比，客觀看待。

白羊座本周整體運勢：⭐⭐⭐⭐

財運：★★★

感情：❤❤❤

白羊座本周事業、學習方面的能量非常強，你們的關注點也在事業上，近期總在想如何讓事業更加穩定，你的付出都是值得的，而你們自身的能量也比較強旺，所以還是可以去消耗精力做很多事情，你們近期還會關注到如何獲取更多的資源，如何賺取更大的錢財的事情上，可能會為了事業發展而投入資金，同時在偏才上也可能會有一些進賬，近期的星象能量不錯，要加油去工作哦。考試運也不錯，部分白羊會有升職、調崗的情況。與合作夥伴、另一半的關係在經歷了一些波折、變動後慢慢趨於穩定。

感情：在25日之後感情的風波會緩解，近期感情可能會產生距離，距離產生美，可以給你們的感情創造一些另類的浪漫。

金牛座本周整體運勢：⭐⭐⭐

財運：★★★★

感情：❤❤❤❤

金牛座近期可能要在事業方面費心較多，可能要去做一些規劃和調整，部分金牛座近期還有失業、跳槽的情況，星象給我們指引，讓我們發現更多的可能性，在25日之後你們經歷了一些變動後慢慢恢復穩定。金牛座近期會變得更加理性，理性的看待自己、看待事業，客觀的分析自己的計劃，這也是一種成長。本周你們的偏財運勢不錯，可能會有好的進賬，或者有好的資源、投資項目找到你，要好好把握哦。家庭方面的事務可能要你們多去關注一下，家庭運能量較弱，看看家裡人需要你做哪些幫助，給予他們一些支持。

感情：桃花運勢上升，戀愛機緣增加，與另一半的關係不錯。

雙子座本周整體運勢：⭐⭐

財運：★★★

感情：❤❤

雙子座近期受到海王星的負面影響較多，時常感到迷茫，容易對自己產生懷疑，自身的能量也消耗較多，會為他人考慮比較多，特別是對合作夥伴、客戶等人際關係消耗你較多的精力，而近期特別容易在事業方面感到迷茫，不知道在這個公司到底能不能走下去，不知道自己的事業平台可行嗎，等等問題困惑你。本周27日後雙子座可能會做出一些調整，而這個調整更多的是向著好的方向去改進的，你們需要放下自己的疑惑來客觀的看待很多事情去制定計劃。給自己補充一些能量，多吸收一些靈感，可以多出去走走，學習，與他人交流，會給你帶來一些好的想法。

感情：近期你與另一半的關係可能會有一邊倒的傾向，雖然很難有完全平等的愛情，但是兩方的勢力過於懸殊必定會導致弱勢方的爆發。

巨蟹座本周整體運勢：⭐⭐⭐

財運：★★

感情：❤❤❤

巨蟹座近期個人的能量還是蠻強的，你們近期人際關係方面的事務較多，合作宮非常「熱鬧」，可能會有他人找到你想要與你合作做事情，或者你們要處理很多客戶的問題，部分巨蟹座可能會自掏腰包去滿足他人的需求。財運一般，近期巨蟹座有些耗財，最大的感慨就是：花錢的地方可真多！家庭、人際、團體方面的事務近期可能會有些變動，面臨人際關係的分離或重組，你們在這些方面學會了更加冷靜客觀的看待，在27日之後這方面的事務慢慢穩定。

感情：桃花運很旺哦，不過有伴侶的巨蟹座要防止爛桃花來影響你們的感情哦。

獅子座本周整體運勢：⭐

財運：★★★

感情：❤❤❤

本周獅子座的命主星太陽換座至摩羯座，近期太陽的能量消耗較多，獅子也可能會覺得自己近期很多事情力不從心，需要做的事情較多，精力有點跟不上。太陽進入你們的工作宮，接下來會將很多精力投入在工作上，並且工作中的很多事情讓你覺得很費心，可能會突然要出差、家裡臨時有事影響了你的正常計劃。部分獅子座近期可能會考慮投資、理財、保險方面有關的事情，運勢也不錯就 ，只是比較耗費你的精力。事業方面近期可能經歷了一些變動，方向的調整，本周末要相對穩定一些了。

感情：桃花機緣不錯，容易發生辦公室戀情，或者在工作中遇到心儀的異性。

處女座本周整體運勢：⭐⭐⭐

財運：★★★

感情：❤❤

本周你們可能還要處理一些家庭方面的事務，下周開始進入子女宮，你們的關注點隨之轉移，本周與家人多一些相處的時間吧。前段時間處女座可能會陷入渾渾噩噩的狀態中，對自己產生疑惑，在人際關係方面產生一些問題，可能會遇到一些不靠譜的人，現在逐漸清晰，突然發現之前怎麼會相信這個人？本周海王星對你的影響逐漸減弱，能量慢慢恢復正常，也會看清一些事情。對於處女座寶媽、寶爸，你們近期在子女方面可能會發生較多的事情，孩子有很多事情需要你的幫助，你們近期也適合去學習一些教育方法。處女座事業方面的事務經歷晦暗期，本周目標會逐漸清晰。

感情：感情的事務有些複雜哦，近期桃花機會多，但是容易遇到爛桃花哦。
