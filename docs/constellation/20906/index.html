12月20日起，表面開朗，內心憂傷，寂寞的三星座

2019-12-20 1,752條評論 3,722次閱讀 1080人點贊

靈簽測算規則：

1.保持內心平靜。

2.默念自己的姓名。

3.點擊下方開始測算。

@小陳靈簽

–

雙子座

不瞞大家說，雙子幾乎在所有人的眼裡，都是一個活蹦亂跳的模樣。他們的眼裡似乎有著光亮的影子。有人愛用沒心沒肺去形容雙子，而我想說，那隻不是雙子會掩飾自己罷了。20日起，雙子的這一特點尤其表現得明顯。

很難有人，能夠看透雙子的悲傷。善良的雙子，20日後尤其孤獨。他們更易去一個人承受悲傷、委屈和彷徨。雙子是默默無聞的，他不解釋，並不代表他們就是傻傻的樣子。他們只是想要，你們透過眼底，看向他們的心底而已。

天秤座

20日起，被稱為傾聽者的天秤們更是會發揮這一特性。因為，天秤們能夠感同身受，換位思考，所以才會更努力，更用心地去成為一個傾聽者。20日後，天秤們將成為你們最信賴的一個星座。

想要讀得懂天秤們的心，你要做的，就是在他們沉默的時候，選擇靜靜地陪伴，你不需要說話，也不需要肢體語言。就是單獨地陪伴，就會化解天秤們的悲傷和無奈，天秤們就會很快滿血復活。

射手座

20日起，射手們將更加重視情義，更加重視對方的感受，不讓對方感受到一丁點的不舒服，這就是我們的射手們。他們的心直手快，他們的不善偽裝，但卻在照顧人時更加貼心。他們常常忽略自己，也常常不計得失，但他們有時也會無助，有時也會同樣需要關心。

靈魂的射手們是個有點傲嬌的模樣，但他們的傲嬌只有懂得人才能看得透，只有在緊緊擁抱射手後，才能更體會到屬於射手們的倔強。因為，他們更渴望你能讀得懂他。
