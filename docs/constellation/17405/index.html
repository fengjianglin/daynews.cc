A：你是貴婦，過著優越的生活，讓他人仰慕，這不僅僅需求的是命運的眷顧，還需求你本身具有成為貴婦的條件，現在的你就有這個條件，不論是氣質上還是容貌上，情商智商雙高，智慧過人，現在唯一缺少的就是你的白馬王子了，以你的命運信任不久就會遇見你的白馬王子。

B：你是普通一般人，你在往後既成不了貴婦，也不會變成貧婦，在你的日子中就只要一般，這與你的選擇有關，你喜歡安安靜靜的日子，快快樂樂的生活，美好愜意的日子，不會為了一些高目標去儘力，去奮鬥，你只想和另一半過著最一般最甜蜜的小日子，你們會這些美好的過下去。

C：你往後是貧婦，你往後或許成為貧婦，你成為貧婦或許是因為過分在意一些事物，變身成為貧婦，或許是因為你所處的環境問題，不得不變身貧婦，讓自己全是裝滿刺，只為保護自己，這並不是你所希望的，也不是真實的你，你需求一個愛你的人，把你從這種情況中解救出來。

D：你可能是個對立體，你有著貴婦的條件，可能成為貴婦，也可能成為貧婦，成為貴婦的你人見人畏，即使是你的愛人也會對你敬一尺，成為貧婦的你，人見人憐，你的愛人更是對這樣的你愛的無法自拔，決定這一切的取決你的性格，假如你改變你的心態，心氣不要太高，就會成為受人歡迎的貴婦，這要你的愛人才會更體貼你。
