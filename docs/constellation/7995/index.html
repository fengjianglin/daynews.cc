降溫別生病，12星座一起來給自己買一份“安心”吧

2019-12-06 905條評論 3,549次閱讀 1164人點贊

大家好呀，我是不露臉小姐。

強烈推薦大家看《幸福三重奏》，我最近真實的快樂源泉。我要是有一天死了，一定是被郎朗和吉娜兩口子活活笑死的。世界上怎麼會有吉娜這麼可愛的女孩子啊，明明長得像芭比娃娃一樣，還是個德國混血，結果一開口就跑偏到象牙山。甚至有個很有才的網友寫了以他們倆為主角的鄉村愛情劇本，我看的津津有味，每天在論壇撒花求更新。還有一集節目里，吉娜穿了德國的傳統服飾，一條特別好看的花裙子，然後甩開郎朗遞過來的外套說，一點都不冷，漂亮最重要。真的笑死我了，彷彿看到了我好幾個女生朋友的樣子。

但是吧，美麗不扛凍也是真的。最近大降溫，我這幾個愛美的女朋友全都感冒了，無一例外。嚇得我趕緊翻遍了萬事屋，找出了一組流感季大家都能用得到的好物做了一次折扣，包括暢銷日本的殺菌防塵消毒卡，防霾不勒耳朵的PITTA口罩，小孩子也能用的免洗消毒洗手液和提高免疫力、促進消化的泡騰片。

冬天最冷的時候還沒到呢，希望大家都別生病啦。

10月份換季的時候我買了這個消毒卡，也不知道是因為它還是因為我自己爭氣，這兩個月辦公室里的同事這個生病那個感冒，但我目前還安然無恙。它有送一個小袋子，可以像工作證一樣戴在脖子上隨身帶著走，袋子里還能塞一張交通卡，很方便。而且沒有我想像中消毒劑的味道，它是無色無味的，還蠻推薦辦公室或者教室不通風的朋友們買一張備著，就像披了一件哈利波特的隱形斗篷一樣。

喝起來像果醋，我幾乎每天都喝，雖然是沒瘦啦，但是天天大吃大喝倒也沒胖起來。早上到辦公室泡一杯，看到杯子里噗嚕噗嚕冒泡泡，覺得一整天都元氣滿滿的。
