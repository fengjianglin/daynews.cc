　　水瓶座在2020年里的整體事業狀況都不錯，你和他人之間的合作關係能夠為你的事業提供保障，而在很多問題的處理上，你能夠和他人保持一致的態度，對你事業發展有推動作用，水瓶座的人更願意相信自己的能力以及在事業上的堅持，即便是你在早期做出了錯誤的判斷，也能夠在下半年有所改變，可是大半年已經過去，即便是下半年再做改變的，你事業能否追回，也是一個未知數，在10月往後水瓶座的事業狀況會有明顯轉變，然而在此之前你的事業一直以來都處於跌宕起伏的狀態，對你個人的生活也會有不利影響，很多事情如何能夠掌控發展機遇是水瓶座的，你要學會的。

　　射手座

　　射手座在2020年的整體運勢都不是太好。特別是在工作上你非常勞累，對你的生活也會有不利影響，而你對自己的要求過高，也是你在事業中做出錯誤判斷的重要緣故，很多事如何平和的應對是關鍵的，越是給自己太強的壓力越會讓你的工作變得一團糟，小心思太多也難免會讓你與他人的關係出問題，如何能讓生活變得更加愜意，是射手座的你一直在考慮的，可是越讓自己過得舒服，反而會對你的事業有不利，在這一年裡你也有逃避責任的想法，尤其是在事業的關鍵期，你卻給自己太多借口，反而會讓你的事業有不利，要勇於面對，否則你在事業上總是會因小事而有失去，也會讓你後悔不已的。
