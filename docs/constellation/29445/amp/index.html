活字印刷術，源於北宋的畢升，金屬活字出現於十二世紀，自此知識的傳遞，進入了一個嶄新的紀元，訊息的傳播也更加快速與方便。現在的我們，已經習慣在電腦前面，看著由0與1所構成的數碼字，現在的書籍，都已經是大型機器所印刷出來的。而金屬活字印刷，就在時間的洪流里，被淹沒了。

在台北西區的太原路，還有一戶人家，堅持用鉛鑄字，它就是正體字的世界裡，唯一的鑄字行－日星鑄字行

日星鑄字行創辦於1969年，店名是由第一代老闆張錫齡所命名，取名日星意為日日生，希望日日生產，確實當時的日星鑄字行，生意大好，在極盛時期，日鑄十萬字，員工30人，客戶200多家。第二代老闆張介冠接班時，一切都改變了。為了讓這項傳統技藝保存下來，張介冠打開大門，轉型成為文化產業，對喜歡漢字的人開放。

第二代老闆張介冠

日星鑄字行最大的資產就是擁有30多萬字的正體銅鑄字模，鑄字機，以及充滿熱情、希望活字印刷術復興的老闆張介冠。

在這裡，文字不再是冰冷的，平面的，而是可以觸摸的到點橫豎勾，感受的到凹凸起伏的地方。

每個參觀日星鑄字行的人，都需要把包放在置物櫃中，並且遵守規矩。

日星鑄字行目前只生產三種正體字型，分別是楷體、宋體與黑體，每一個字型再分7個尺寸。除了正體中文之外，還有英文與符號。

不大的空間里，鉛字一排一排的放在字架上，等待人們的發掘。

每一個字架上，都寫著字型、尺寸與鉛字名，如果有興趣，可以在櫃檯拿著撿字盒，把您喜歡的字挑出，或者是請工作人員，協助找字。

日星鑄字行除了賣鉛字之外，還有一些文創商品，提供挑選。

櫃檯前上的字，說明的日星鑄字行對於文字的態度與初衷。

因為日星鑄字行的緣故，周圍開了許多跟印刷有關的商店，像是一間印刷行，裡面有更多關於印刷的產品。

如果您有興趣，還可以買一台隨身活字名片印刷機，讓您的名片，每一張都是親手印刷，每一張都可以感受字的凹凸。

2019年7月，日星鑄字行與作家張大春聯手合作，用活字印刷的方式，出版《活葉集》，手工製作100份。
