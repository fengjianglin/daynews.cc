很多時候，無論我們面對的是人還是物亦或是事，我們都不能用「絕對」的認知或者判斷觀去定義它們，但實際上「概率」定義也確實能夠相對準確的定義人或物以及事。這種概念對下面我們要說的這個星座來說是非常實用的，且也相對準確，接下來我們就來揭秘一下這個星座內心深處所藏著的秘密。

說真的，很少有人會把自卑與這個星座聯繫起來，但事實真是那樣兒

處女座—這個星座的自卑也叫自信，自信也叫自卑，它們是婉轉的也是隱晦的

說到自信，火象星座不失為最好的代言人，但是當三個火象星座缺乏自信的時候，他們每一個人個體獲得自信或者掩蓋不自信的方式卻是不同的，白羊搞怪或沉默或貼金，獅子會故意凸顯自我，射手默默的，要麼就是刺眼的。

不過，我們也要以客觀的角度去看待火象星座的自信或不自信，準確的說，火象星座的不自信只要不是從小被「委屈」而成的，那他們基本上是不會不自信的，即便有也只是某一個受挫失敗的階段或者面對某些他企及不到的人或事。

總之他們的自信或者不自信你都可以準確地判斷出來，這不需要什麼概率來定義，只要你細心。

但處女座的自信與不自信你看不出來，因為他們並不會如實表達。不開玩笑的說，除了極個別極度自信的處女座以及極度不自信的處女座之外，你根本不可能從一個正常的處女座身上分辨出他的自信與不自信。

那麼問題來了，處女座到底是自信的還是不自信的？

如實說，他們是不自信的，或者說是自卑的也可以。說到這兒，可能很多朋友都不能理解這個概念，那我們就詳細的來分解一下這個概念，跟年輕的處女座打過交道的朋友應該都知道，年輕的處女座有的很龜毛，有的也很裝X，他們無論從打扮還是狀態上都是這個味。那麼反想一下，什麼樣的人需要這樣去表現自己的個性？

答案毫無疑問，缺乏自信的人，自卑的人，他們很需要得到別人的肯定或者很需要特殊的自己去吸引周圍的眼光跟關注，他們需要自己與眾不同。

但是客觀上來講，自信的人不需要奇葩或者過度招搖的打扮。

但是，如果論內在的話，如此外放的處女座，內在卻不是豪放有個性的，雖然這個星座與極度需要存在感的獅子挨得很近，但他們真的做不到獅子座那樣的豪放。

說起來，有很多比較特殊的處女座的例子很難讓人理解，比如談戀愛之前處女座送給對方一件比較昂貴的禮物，但是兩個人最後分手了，處女座還要算賬，甚至於有些處女座還會想辦法將損失彌補回來，

這個星座也很需要自我的平衡。

當然像這樣的處女座基本上屬於不自信的、自卑的處女座。

我之前很愛跟火象星座的粉絲朋友開玩笑，我總是說他們不知道自己兜里有多少錢，絕大多數火象星座只知道自己大概有多少錢，而不知道具體有多少，且放的少也會往多了想，他有一百能當五百的去花。

但是這個問題在處女座這裡幾乎不存在，你要說一個處女座不知道自己兜里有多少錢，那這就真的是玩笑了。實際上除了處女座自己清楚自己的家底之外，其他人是不知道的，包括他們的父母，而且處女座的家底基本上都會被他們「安排」一遍，合理使用。

相信大家對此也深有感觸，處女座很有建設性，而且相對謹慎，且在加上他們嚴格追求自我標準的完美，這使的處女座成為了一個「最少犯錯」的星座。

老實說，這真的是一個不折不扣的優點，但凡成熟一點的處女座，無論在什麼崗位、什麼樣的環境下，他們總能讓自己很少犯錯，也都能選擇一個有利於自己的方向跟想法。

但是同時：處女座的這個本質也註定了另一個本質，這個本質就是處女座極為缺乏狂熱跟偏執。

說起來這個點很矛盾，由於處女座總是能夠做出對的選擇、想出好的想法，所以很多狂熱的、偏執的想法跟做法他們都不會去選擇，而不選擇是因為他們害怕結果的可怕與不可承擔。

那麼，這是一種自信還是自卑呢？

我們把這個概念再放到感情中去理解，處女座幾乎不會去選擇一個比自己分數低的人，兩人分數差距過低基本上也沒啥可能了，除非處女座是真的發自內心的喜歡一個人或者欣賞仰望一個人。

而在一般情況下，他們的選擇的人要麼與自己分數相當，要麼就得高過自己，但是在這種最最一般的情況下，處女座依然是糾結的、神經質的。

所以就大齡剩男剩女這個問題，處女座免不了會「分一杯羹」。

當然，有的朋友會覺得處女座的想法跟選擇沒錯啊，現在大家都是這樣的。

但是容我說句客觀的話，這樣的人真的只是一部分，別的不說，大家去看看風象星座、火象星座，為什麼他們的愛情能那麼熱烈、氛圍特棒？這是有根本原因的，說白了，他們遵從自己的選擇，也相信感情，更相信自己能夠將感情變得更好，至於做到或者做不到，那得看他們的努力，但他們的心是狂熱的、自信的。

所以，最好不要認為你身邊的處女座是自信的，你要學會適當地給他們一些狂熱，這樣一箭雙鵰。
