千萬粉占星師改造33年老屋，連梁靜茹、林宥嘉都去過她家直播談心

2019-12-19 672條評論 1,676次閱讀 422人點贊

你知道一個擁有微博1217萬粉絲、Facebook臉書123萬粉絲的占星女學者，她的新家不僅有小夾層秘密基地、陽台藏了鞦韆、明星更是每周造訪，即使一個人住，也能這麼精彩？！

台灣知名星座專家唐綺陽的新家，來過的明星藝人數之不盡，包括：哈林庾澄慶、勇氣女王梁靜茹、很少女的林宥嘉、主持人黃子佼、台劇俗女養成記主創演員…等明星都在19年到訪過她的新家。在此之前名模林志玲、主持人蔡康永、小S、歌手蕭敬騰也都曾道過她家參與餐桌直播呢。

身兼藝人、作者與占星學者等多重身份的唐綺陽，在天時地利人和的情況下與33年的老屋結緣，雖然擁有地緣以及價格的絕對優勢，但不管面積、採光、格局以及樓高等等都差強人意。

對於唐綺陽來說待在家的時間很多，工作直播、寫書、休閑生活，都是以家為基地運轉，也因為頻繁的直播型態時常邀請明星名人在家裡作客，讓她的自宅成為全台灣的藝人里最公開、知名度最高的特殊私宅。

為了改造，她找了她– FUGE馥閣設計黃鈴芳設計師來進行改造！

馥閣設計黃鈴芳設計師，是台灣知名的空間設計師，曾榮獲多項設計大獎，包括：義大利 A’Design Award、德國 Red Dot Design Award 紅點設計大獎、德國 iF Design Award、日本 Good Design Award、台灣 TID Award 台灣室內設計大獎、中國 築巢獎、金堂獎…等，設計串連室內與室外，用心為使用者打造自在的生活，讓空間展現感動人心的生命力，娓娓道出不同的人生故事。

究竟，唐綺陽的家，改造後有哪些設計重點呢?!

傾聽屋主需求：安全感、隱私性、互動性

雖然身為星座專家，但占星專家唐綺陽對於新家的要求其實以生活的舒適性與內心的安全感為主。

進行直播工作，需要的是與外界交流時的安全感；專註於個人寫作與閱讀時，需要的是能沈澱情緒的安全感；而當親友來家裡相聚時，需要的是能展現真性情的安全感。

馥閣設計的黃鈴芳總監在與唐綺陽老師多次深聊後，設計上掌握她渴望的安定性與工作的社交性兩大主軸，在私領域的設計上以榻榻米和室、小半坪穴居以及室內陽台建構出能安靜獨處又能熱絡交流的串聯關係。

不同形式的個人空間，提供長時間在家的占星專家能隨居家情境轉換心境。

她在家，也能享有三個秘密基地

1. 書桌上的秘密基地

小半坪復層的下方安置了放鬆的按摩椅，面對陽台的簡易桌板，完成唐綺陽喜愛「窩著」寫作的心情。上層的平台可說是小閣樓的秘密基地，賦予生活不同視角，坐在上方環顧四周，雙腳隨意搖晃，在工作疲憊之虞能換個角度，在室內也能體驗日光陽台的愜意。

2. 陽台藏有大人的鞦韆

將原本卧房全數打掉後改善室內光照，並藉由玻璃拉門打造一座室內的陽台空間，良好採光弱化了梁下僅210公分的壓迫感，加上一張鞦韆椅，搭配端景綠色植栽，充滿童心的設計，滿足占星專家對於陽台的渴望。

3. 獨處的榻榻米區

原本可能閑置的畸零空間透過階差架高後化身小和室，榻榻米區域是熟識友人專屬的談心天地，也是經紀人與親友稍微小憩的地方；搭配升降桌設計後又是一個能安靜寫作思考的小天地。

格局：她用回字型，創造分流動線

玄關、客廳、餐廳、卧房等家中場域的位置安排與動線密不可分。在這次老屋改造首先拆除包覆的天花板，將部分樑柱露出；不能改變樓高，就設法創造空間深度與廣度。

依照佔星專家需要的住宅機能分配場域位置，打破傳統老屋僅一個走道的壅塞動線，將L型集中在房子邊緣，讓剩下的區域能順暢地串聯，塑造室內多向行進路線，讓人能在不同空間相遇。

為了讓占星專家唐綺陽在長桌直播時可以有更多元的背景，特地將玄關進入內部的動線規劃在餐廚中島區域，透過拉門穿越熱炒區也可以抵達私領域，降低在直播或拍攝時有人走動的可能。

占星專家可以選擇坐在電視牆前方或面對室內陽台，重整後也讓窗外的台北地標101大樓納入室內視野，透過空間與動線的布局，賦予觀看直播的觀眾更多新鮮感。

視角：斜角與弧形，改變老屋陰暗廊道

過多封閉牆阻擋光線，連帶讓內部長廊變得昏暗擁擠。

在無法改變開窗方向及數量的前提下，黃鈴芳設計師將傳統空間內部生硬的直角用弧形取代，轉角變得圓滑後，狹長走廊以喇叭狀由外向內延展，縮短走道也讓空間在視覺上更為開闊。

改造格局後，採光重新被串連，形塑出連續的大面積光感，明亮了室內空間。

除了由玄關通往內部的廊道做了弧形設計，直播長桌面對的電視牆、以及更往內部延伸的寢居空間，都帶入了親切的圓弧。

曲線在略為冷調的空間中創造了柔美意象，無形中影響了屋主的情緒，讓身為公眾人物的她能卸下心防，享受家居生活。

安排家，也是工作場域

一進門的客廳空間，於居家生活能作為平日閱讀、休憩使用，於工作社交上提供隨行訪客舒適的等候區；空間規划了獨立客浴將機能升級，讓客廳自成一區。

經過廊道後安排一張長桌是直播的主要場景，餐廚中島平行長桌，將生活機能帶入工作區域，方便準備茶水點心招待來訪客人。

挑選傢具時除了造型搭配外，客廳一張Cassina沙發以能夠好躺好睡為標準，其內部的幾張設計單椅亦是如此。

色彩也要治癒系，白調和黑灰材質

總是以黑色穿著為主的占星專家唐綺陽，黑色對他來說是最習慣也讓她感到自在的色彩，但過多的黑色會讓空間顯得暗沈抑鬱。

因此新家在基調設定上選用灰色系搭配黑色的皮革櫃體，並於動線頭尾用大面積的白色帶入明亮感。內縮陽台的白色塗料將自然採光反射至屋內，弱化灰色調帶來的沉重感。

選擇礦物漆塗料由地面延伸至牆面，玄關牆面、中島上方吊櫃與天花板樑柱亦以清水模塗作溢漿形式塗裝。塗料的自然紋理在平面上更顯立體，也降低灰冷的調性。

需要均勻光照的直播區利用上方雙向的軌道燈，均勻打亮，破除間接照明的缺陷；以放鬆休閑為主的區域則安排局部崁燈，搭配色溫配置，讓以灰階為主的空間能擁有屋主習慣的色彩，達到心理上的安全感。

直播長桌排列的8張Cassina單椅，也選擇了接近礦物的質樸顏色，呼應地板以及牆面的礦物塗料；帶著金屬鏽蝕質感的薄磚餐桌也順應著自然礦質主題，讓植入的色彩顯得自然合理。

植栽：改變灰階空間的表情

空間里雖然以灰階為主，但內部其實悄悄地安排了住宅里唯一的色彩。

墨綠色被小面積點綴於客廳收納櫃以及走道空間，加入顆粒感來提升的色澤變化。

綠色靈感一部分來自於這是自然界最常見的色彩，這樣的特別色也最容易被屋主與空間接納，又能呼應內縮陽台的植物造景。

來自屋主本人的心情反饋

從19年五月入住後，唐綺陽最明顯感受到家人之間的互動感提升了許多。

過去親友來訪時總是來去匆匆，但現今卻能看到侄女們有的在小半坪上、有的坐在陽台的鞦韆椅，長輩安適地坐在和室榻榻米，家人們各據一方但卻能自在的談天交流。

改造後的老屋多了好幾個屬於自己的秘密基地，在新的空間直播心情也大為不同，增加與觀眾的互動感。
