雙子、金牛、白羊：潛藏在深處的“小性格”

天天要聞
3 天 ago

我們願意把好的一面展示出來，會用一些方法將自己一些不為人知的性格隱藏起來，但是紙包不住火的，有時候那些小性格會冒出來，不同星座有不同的小性格，來看看你是否也有這些小性格吧：

邊緣人格嚴重的雙子座。雙子座的能言善辯，是出了名的，這也讓他們有了好人緣，雖然口才好，但是他們內心的孤單與寂寞是很多人不明白的，因此潛在人格會分裂出另一個性格，與自己的開朗性格達成平衡，一個性格在與他人交談甚歡，另一個性格在與自己和解。有時候兩個性格也會發生爭執，在外人看來雙子座就像是神經病一樣，其實這是他們的緩解壓力的方式。這樣的邊緣人格，很多雙子座都有，只是不會輕易被發現。

雙相情緒的金牛座：經常把自己的情緒積壓在心裡的金牛座，不願意與人交談心事，他們與你談話的內容很多也很豐富，但是真正觸及到內心時，他們就不願意再說下去。任何情緒如果長時間憋在心裡，不能得到釋放，就會有壓抑或是悲觀發生，心境一旦產生，會有超長時間無法自拔，這個時候人會變得更敏感，同時會迸發雙相情緒。

偶爾有些暴力的白羊座：我們常常說暴力無法解決問題，同時也會對更多人帶來傷害，因為我們更願意把這種小性格深深隱藏在內心裡，不到萬不得已，不會表露出來。對於白羊座來說隱藏似乎是一件極困難的事情，因為他們天生衝動做事也不會想後果，聽不進別人的勸告，因此有一些不好的情緒出現，暴力傾向就會顯露出來。

有些小性格可能有些人一輩子都不會被外人知道，他們用這樣方式，在與自己和解，與自己交談，舒緩著壓力，這些表現不算是缺陷，但是如果長期壓抑，那麼並不是會好事，有時候放鬆下來，多多與人交談，了解更多事，才會真正讓心情得到緩解。
