八卦是三爻變六爻，而不是三爻變四爻？告訴你，這才是古人的智慧

2019-12-17 726條評論 3,744次閱讀 1728人點贊

八卦是中國最古老的學問，也是迄今為止，最神秘的學問，歷代的大家都能從八卦中獲得無上的智慧。從最初的伏羲八卦，到唐宋之後的六十四卦，八卦的變化時很大的，最重要的就是由原來的三爻卦，變成了現在的六爻卦。那為什麼不是由三爻增到四爻，而是直接增到了六爻呢，知道為什麼嗎？

三爻卦變六爻卦

想知道這個問題，首先要知道三爻卦的來歷。我們知道從八卦初創之時，就是三爻卦，排列組成了「乾、坎、艮、震、巽、離、坤、兌」等八卦卦，也算是八個自然現象，這八個現象就是古人通過觀察「天、地」的變化而來，這其中就包含三個重要的元素，那就是「天、地、人」。

天地人

以人去觀察天地的變化，後形成了八卦的雛形，所以又成「天、地、人」為三才，遂成為三爻。而「三才之道」又分陰陽變化，以橫「—」為陽爻，以「——」為陰爻。沒卦各取「天地人」三陰陽爻中的一個，形成了三爻卦。

陰陽變化

其實這是很有道理的，沒件事物不可能同時有兩種狀態的顯現，任何時候，都是一種現象的展現，所以儘管「三才」有陰陽6種變化，可是只能取一種變化使用。這能看出來，古人的智慧事多麼的先進啊。

IPv4到IIPv6

知道了三爻的來歷，再去看六爻就方便了。其實六爻卦的應用，其實是因為世間事物的變化更加的繁雜了，原來的三爻卦，已經不能滿足我們日常的使用，就要面臨升級，就像我們現在的互聯網，正在從現在的IPv4時代，向著IPv6的時代前進，為什麼呢，就是IPv4已經不夠用了唄。

六爻卦的組成

要升級，就得有一個標準，不能隨便的由原來的三爻升到四爻，就算升級了，不然《易經》也不能算是最神秘的學問了。其實六爻卦，也是從三爻卦而來的，它的原理，簡單的來說就是兩個三爻卦的疊加。六爻卦從下開始數，一至三組成的三爻卦，為內三卦，四至六的三爻卦，為外三卦。這樣內爻和外爻組成的，就是一個新的六爻卦。

某事物生命周期過程

如果從科學的道理去講解，也是很有科學道理的，如果去看化學、生物、物理的理論，就會發現，各種事物的變化，都不會超過六種狀態，超過到了第七種狀態，就成了一個新的事物了。這不就是我們的六爻卦嗎，從內爻的第一爻開始變化，直到最後第六爻的變化，總共六種變化，再去第七種變化，不好意思，已經沒有了，因為再變化，就成其他的卦了。

八卦

這時候也避免了，同樣的「天地人」三才的陰陽變化，在同一個八卦中出現了，因為分開了內外卦，相互之間的關聯就減弱了。所以說，這就是古人的智慧，當然這是我們現代的見解，也許古人的見解更加的獨到，也說不準啊。

這就是三爻卦的升級，為什麼不直接升到四爻卦，而是直接升到六爻卦的原因了。有喜歡的可以關注啊！
