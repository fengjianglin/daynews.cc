標準獅子座：我不是自負，天生就是領導人

2019-11-28 105條評論 1,029次閱讀 57人點贊

獅子座火象星座，7月23到8月22 日。大多印象獅子都是森林的王者，她們霸氣又高調。你給獅子座的標籤是什麼，評論區討論一下吧。

這個姑娘說自己是標準的獅子座哦，大多獅子座該有的性格都和他特別符合，可能偶爾有人會把這種性格說成自負，但是也不介意，因為她喜歡這個標籤，獅子就是如此。

獅子座的交際圈就像她們的秀場，在這個秀場里熱情大方，大散光芒。獅子座有自大的資本，因為她們能力強心思細膩，不論是言談舉止還是行為處事她們都會把自己的優點發揮到極致。獅子座的女性樂觀，喜歡樂於助人幫助弱勢群體，天生的王者風範在人群中特別受歡迎，因為擁有群眾魅力，說話和做事就很有說服力。

她們喜歡自己生活有高品質，如果你有讓他們欽佩的本領那麼他們就會想和你交朋友，畢竟強者的同伴也要是強者。如果你喜歡獅子座的女生，就不要欺騙她們她們喜歡尊重平等的交往方式，對於那些沒有禮貌或者粗魯的就會直接被pass了。

當然天生霸道的性格會讓獅子比較獨斷專橫，所以被人說傲慢自大也是情理之中了。
