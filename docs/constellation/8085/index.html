2020年開始，不再水逆，運勢扶搖直上，大器將成，被上天眷顧的人

2019-12-02 714條評論 4,678次閱讀 1825人點贊

好運靈簽，快來測：

-深呼吸幾次

-在心中默念自己的名字

–點這裡測算

近日星象： 木星在年尾進入摩羯座，給整年虛幻空想的浮誇畫上了句號。木星在摩羯座停留到2020年12月18日。這是貧瘠的位置，和上一年敞開的機遇之門不同，我們可能有經歷寒冬感受，一分耕耘一分收穫將是2020年的主旋律。不過我們會更加了解到做夢是沒有意義的，腳踏實地才會帶來大豐收。這一年我們的經驗得到極大的增長，也變得更接地氣，更成熟。我們可能更加認清現實世界和社會規則。這是權威、權力大力發展的一年，有很多人會攀上政策的順風車，開啟新的發展。這也適合尊重權威，跟隨政策，攀龍附鳳的年份，懂得按照規矩做事的人就會順利發展。老齡化，養老問題也可能再度成為社會熱議焦點。我們可以期待未來一年有利好政策出現，或是原有僵局，困境出現突破。金火六合金讓這段時間感情充滿了熱度，各種浪漫戀情如火如荼展開，容易沉浸在美好融洽的情感中。

1、雙子座

雙子座活潑外向，很容易討人喜歡。當他們擁步入愛情之後，他們會很認真地對待感情。他們願意在感情中付出，也願意為愛情改正自己的種種問題。但是他們不會因為愛情失去自己。他們雖然很愛對方，但是不會一切都依著對方，不會因為生怕對方離開而委屈自己。他們覺得對方若要離開，一定因為兩個人有不合適的地方，若是兩個人不合適還非要在一起，會造成兩個人的痛苦。因此，他們珍惜愛情，但不盲目。

2、水瓶座

水瓶座待人真誠，與他們相處，一定會被他們的魅力所吸引。水瓶座遇到自己愛的人，他們會以真摯的心對待，絕對不會虧欠對方。他們心裡深愛對方，但是也不會過分擔憂對方的離開。他們不害怕孤單，他們本就享受屬於自我的時間，即使單身，他們也能將生活過得很精彩。而且，他們覺得愛情的存在應該是雙方都變得更好，若是愛情使得兩個人不再快樂，那也就沒有繼續的必要了。他們敢愛，也敢放手。

3、獅子座

面對愛情，獅子座敢於主動追求，而且衙門對愛情滿懷熱忱，會用心寵著對方，希望對方能夠與自己此生相伴。獅子座在愛情里付出真心，也想要收穫對方的真心。他們愛著對方，但是他們也從不擔憂愛情的失去，甚至他們會在發現自己的真心沒有被珍惜的時候主動選擇分手。獅子座本就尊嚴感重，他們絕不會在愛情中乞求對方留在自己身邊。獅子座決不會讓自己苦苦挽留對方。
