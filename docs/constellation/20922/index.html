未來17天，壓力得以釋放，生活恢復輕鬆的星座

2019-12-19 2,081條評論 3,959次閱讀 2617人點贊

生活常常會選擇與我們開玩笑，當我面對一件事情，如果看到結果的時候就會選擇鬆一口氣，這個時候生活並不會讓我們去放棄，反而還會在事到臨頭給我們增加一些阻礙，這就會讓我們明明離成功更進一步的關鍵時刻卻與成功失之交臂。一件事情的阻攔，比較會在我們心中產生一種遺憾和悔恨的念頭，但是在這一年即將結束的這段期間，我們也要學會釋然與過去說再見，這個時候生活也會恢復之前的輕鬆。

處女座

12月對於處女座來說，一直是自己運勢穩定回升一個狀態，這段期間他們不無論是在事業或者生活上都能收到一定的好運，同時自己心態方面的弱點也會在平時做事中暴露出來，在未來17天，新月日食會發生在處女座的官祿宮，這就更容易幫助處女座在事業上提升自己的能力，在面對一些事情面前，也會有著不同的看法中得到領導的賞識，而自己的事業也更容易步步高升，但同時新月日食帶來了巨大能量，也會將處女座身上的不足暴露出來。受到自己完美主義的影響，這段期間處女座的心態可能會變得比較煩，希望自己能夠改掉這些缺點，在這段期間也希望處女座的人不要過於焦急，將自己的這種缺點暫時放下，有的時候有舍才有得，當我們能夠放鬆心態的時候，自己未來的生活道路也會變得更加寬闊。

雙魚座

雙魚座是一個很容易在一件事情面前陷入糾結的星座，受到水象星座的影響，他們本身性格會變得更加靦腆和糾結，在未來十幾天之後，新月日食會發生在雙魚座的福德宮，對於他們拓展人脈，或者在新環境認識陌生人有著很大的幫助，屬於變動星座的雙魚座本身就是一個適應性很強的星座，這段期間對於雙魚座來說，但是他們生活更多好人緣和人際關係的一個不錯時機，但同時也希望雙魚座的人能夠做到審時度勢，對於過往的一些遺憾及時拋棄，畢竟未來還有更大的機遇在等著自己。

摩羯座

在12月底這幾天的時間，對於摩羯座的運勢來說會發生翻天覆地的變化，新月日食的到來，會讓摩羯座的人在他們情緒上會受到一定的波動，生活也許會面臨著選擇，這個時候也需要摩羯座的人能夠保持自己清醒的大腦，思考問題的時候，依然可以保持之前的冷靜態度，同時也希望摩羯座的人能夠看透一個人的本質，在與人交往的過程中，能夠仔細分別對方的話語，避免自己受到影響，同時受到日食能量，摩羯座內心深處也就會產生一種壓抑的狀態，這個時候就需要摩羯座在面對事情面前學會主動出擊，盡量適應新生活。

未來永遠都是充滿挑戰，我們必須時刻做好準備，才能面對生活帶給我們的機遇，一個人想要守護自己的夢想，或者在未來的生活中走得更加深，所以保持自己心態平穩是一個很主要的原因，同時學會拋棄過往那些不重要的東西，學會與遺憾說再見才能讓自己全新的眼光去看待這個世界。
