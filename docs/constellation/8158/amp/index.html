2019-11-30

正月之火

古文：正月微陽之火，隱於木中，雖有可親可愛之意，但冰霜之氣未消，遇木生之，則陽氣發揮矣。逢金為財，徐、揚人富而好禮。金木同行，官居鼎鼐。逢金值水，夭折無疑。荊、梁、冀、雍之人尤甚。見土盜泄微陽，浮薄卑賤。

正月的火，古人用兩個字來形容：微陽。有點陽氣，有點溫度，有點暖和，但是火勢不大，溫度不高，故稱為微陽。正月就是寅月，月令寅中藏火，故云：隱於木中。這些都是正月之火的特點。根據這些特點，火的喜忌頭條號周易微學堂說了五點：

1、得木生之，可以發揮陽氣。

2、宜金克木以生火，類似於庚金劈甲引丁之意。如果是生於南方，為人富而好禮。

3、金與木同行，印定義財，或者是以財去獲取印綬，有成朝廷大官的可能。

4、金水相生，金來克木，水來克火，木火皆傷，夭折之命。尤其是西方人、北方人。

5、土來泄火，福薄卑賤。所謂的福薄卑賤，是指這個人的素質、本事。因為無能所以卑賤。

乾造：癸亥 甲寅 丙戌 甲午

思路：

月日時寅午戌三合成了火局，這個火局由日干丙火來代表。由於這個火局成局，那麼月柱甲寅之木在吸收了年柱癸亥水的能量之時，會轉生於火，這個就是所謂的官氣通身。頭條號周易微學堂。幸好有這一個火局，火的能量夠大，否則的話這個甲寅木被癸亥水生了之後會變成濕木，如果不夠分量的火它是點不著的。

丙火是很難在其中得到好處的。由於火局成，火的體已經足夠能量去消化甲寅木的能量，這就是古文所講的「借木而明，陽氣發揮」。頭條號周易微學堂。這個丙火能夠體現出他的溫度，能夠展現出他的光明，說明這個人有本事。他的本事主要體現在哪，他的本事不是體現在管官，也不是體現在抗官，也不是體現在取財，而是體現於吸收木這個印綬的能量。所以這個人讀書一定不錯，成績一定不錯。

當然這個印綬甲寅木與官星癸亥有相合的關係， 其中一部分甲木還是從官星癸水坐支中出來的，甲木本身就帶有權印的意思。那麼丙火能夠借木而明，說明他這個人也會有點貴氣，但是這個貴能有多大呢？我們還得具體去分析。

首先我們知道了他讀書很厲害，學歷肯定很高，這個人又有本事，而且很陽光，很開朗，丙火，很積極，火五行本身非常積極的五行，更何況地支寅午戌三合成了火局，他是積極向上的，也很勤奮的，也很努力的。頭條號周易微學堂。但是我們來這個三合火，中神午火在時支是由甲木所佔領，並不是由丙火所佔領，不是命主坐中神，而是印綬來坐中神。

那麼日柱丙戌代表命主，他僅僅是參與了這個三合局而已，並且只是這三合局當中的一小分子，說明他貴，但不會大貴。他能管一部分人，畢竟丙火坐在比劫庫上邊，表示著他管理著一些人，但是還是以人家為主。

他的能力主要不是體現在管人，他只是這火庫中的代表，正真管理戌的是時柱這個甲午，他的本事是化泄印。所以他讀書厲害，本科畢業，畢業後從政，在市政府工作，最起碼來講目前也沒當什麼大官。雖然說他以後前途無量，但這個量還是有限的。
