兩周過後，運勢得到上升，厄運退去，心愿得以實現，不斷發生好事

2019-12-07 565條評論 3,092次閱讀 302人點贊

新年的出現，會讓我們內心的不安漸漸消散，在生活中，難免會遇到一些挫折和困難，可是在遇到這些麻煩的時候，停止抱怨，有時候會讓生活更接近成功，有些人覺得自己的生活並不如意，自己的努力也沒有得到認可。可是在生活中，有些人的確努力了，收到回報很正常，可是有些人只是看起來很努力，並沒有實質性的進展，而我們要做的，就是要真正的努力，這樣才能在新年中，讓陰霾逐漸散去。新年的到來會帶來驚喜，自然新年的出現，也會讓之前的不愉快都慢慢放下，新年的出現會帶來新的希望，陰霾的散去會讓自己變得更好。來盤點下，兩周過後，運勢得到上升，厄運退去，心愿得以實現，不斷發生好事

摩羯座

今年的摩羯座他們的生活可能並不是很順利，做任何事情摩羯座都會發現有一定的阻礙。雖然摩羯座認定目標之後很難輕易的放棄，但是也正是因為這種堅韌不拔的性格，讓摩羯座在前進的道路當中，遭遇到很多困難。所以在諸多困難降臨的時候，摩羯座人需要堅持下去，但是更加需要仔細的分析。兩周過後，摩羯座對於這些問題，有了大概的解決方向，當這些問題遠離之後，摩羯座想要達到的效果也會出現。

雙子座

雙子座 雙子座的人，頭腦聰明，心地善良，目光獨特，善於發現生活中不一樣的事物，是非常有魄力的有才之人。再過7天有財神眷顧，財運亨通，不但求財有道，連家庭之中也有實實在在幫襯自己的手足親情，可以大刀闊斧把錢賺，一切不順將全部翻篇，變成一個大富翁！兩周過後，喜鵲臨門，臘梅向陽，雙子座有天喜星加持，福運駕到，霉運不再

雙魚座

雙魚座不管與什麼人在一起，都能擦出不一樣的火花來，他們是那麼的善良，總是會為別人著想，而雙魚座也是最好的傾聽者，但是他們卻不懂得如何去拒絕別人，也因此讓雙魚座在愛情里，總是會遇到別的異性的表白，或許也正是因為這樣，雙魚座與愛人之間的誤會便開始了，而雙魚座卻不喜歡去解釋，誤會就這樣慢慢的加大了。兩周過後，好運駕到，愛情大好，前任回眸，雙魚座情未斷，愛相守。分手以後的雙魚座，也終於能夠成長起來了，與對方之間的愛情也終將會冰釋前嫌，再次的走到一起，再度愛河，共建雙魚座的小家園。

處女座

處女座的人心思細膩，但是很會勸解別人，在遇到悲傷的事情時，處女座不希望把難過讓別人知道，自己默默的忍著。不過在2020年初，新的一年總會有新的希望，能夠吹走悲傷的情緒，並且也會有心情舒暢的好到來，好好調整心態迎接即將到來的新年吧。

獅子座

獅子座的人性情豁達，而且自身非常的優秀，一般來說，都會成就一番大事業。但是在過去一段時間，獅子座的人卻被霉運影響，無論是工作、生活還是感情都不是很順利。不過，兩周過後，這樣的情況就發生變化了，此時會時來運轉，各個方面都會非常順利。獅子座也能夠走出困境，名利雙收。
