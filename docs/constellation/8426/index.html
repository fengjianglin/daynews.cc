天蠍座

天蠍座2020年整體運程大旺，但是部分天蠍座運氣落入低谷，做事情難以如意順利，事事多難，容易中途變卦。但是大多數人還是運勢不錯，主要是因為在今年遇到三合太歲，確保運勢的根基，保住了福元宮。整年運勢大吉大利，尤其是2020年天蠍座還是長生之力，生機勃勃，具有無限的潛力。整年運勢的根基非常的穩定，做事情容易成功。

水瓶座

水瓶座的人，精明能幹，天資聰穎，一生運氣頗佳。在2019上半年裡受「劫煞」凶星影響，沒能賺到什麼錢，有時甚至荷包縮水，入不敷出。但是到了2020年，因為「三台」星的駕臨，水瓶座人會在事業上迎來高峰期，能得到領導的器重，同行的認可，升官發財，加官進祿是必須的。

天秤座

天秤座的人，在2019年里，天秤座的人們命格中進入了三台吉星，有了吉星的降臨會令他們全年的運勢都是不錯的。到了上半年的時候，他們或許因為自己思想比較保守，所在在投資行業以及生意場上都沒有多大的舉動，這樣就無法賺很多財富。到了2020年，等到機會成熟之後，他們立刻開始執行，而且行動能力很不錯。到了時候，他們的財富越來越好，成為最有潛力的星座。
