最讓人忌憚的三星座，特立獨行，且智商極高，遇到最好避讓三分

2019-12-15 1,869條評論 3,384次閱讀 1516人點贊

導語：

人生的迷茫不在艱難的小路上而在十字路口，老鷹和蝸牛都都分析要想到達金字塔的頂端，就要認清形勢。老鷹採用了先天的機能，利用螺旋飛行的方式飛到頂端，而蝸牛他分析是即使有翅膀的也不一定和老鷹一樣能飛上去，比如小鳥，蝴蝶，蜻蜓。有腳的不一定都可以堅持下來，比如蜘蛛。所以能爬到金字塔頂端的生物並不多。所以它決定堅持不懈地往上爬，因為成功的路上並不擁擠，堅持的人並不多。

天蠍座

眾所周知，天蠍座的人一直都有一副冷冰冰的外表，讓常人難以靠近，而他們這幅冷冰冰的外表不僅能夠保護自己，而且在外人看起來，他們的這種外表氣場非常的強大，最是讓人畏懼，而且天蠍座的人平常也不怎麼發脾氣，他們懂得控制自己的情緒。但這並不是說天蠍座的人沒有脾氣，一般真的惹怒到了他們，他們就會瞬間爆發，對方也會付出非常慘痛的代價的，而且天蠍座雖然平常不怎麼說話，但是他們的智商也是非常高的，如果你想要欺騙他們，是非常容易被他們發覺的，而且他們也會毫不留情的反擊，他們也總是能夠憑藉自己聰明的頭腦與個性，成功的去解決很多難題，這樣的天蠍座可以說非常讓人忌憚了。

金牛座

很多人都認為金牛座的人比較老實穩重，就覺得他們應該非常的溫柔，基本上沒有什麼脾氣，其實金牛座也是有暴躁的一面的，在他們的溫和的外表之下，隱藏的就是暴躁的情緒，他們非常的固執，自己認定的事情，即便是8頭牛也拉不回來，如果你不懂得順從他們，他們就會爆發自己的脾氣，讓人非常的忌憚。不僅如此，金牛座還有著超高的智商，如果你惹到他們了，他們雖然表面上還是會對你和和氣氣的，但是在暗地裡他們就會爆發出不一樣的殺傷力，即便你被算計的，你也是毫無知覺的，這樣的人可以說是非常讓人忌憚了，遇到最好退避三分。

獅子座

眾所周知，獅子座的人是比較強勢的，而且脾氣非常的暴躁，動不動就會生氣，只要別人不按照他們的意願行事，他們就有可能發脾氣，因為他們喜歡當領導者，不喜歡被別人指揮來指揮去的，如果你的能力高於他們，他們還有可能會接受，如果你的能力沒有他們強，反而來指揮他們的話，他們會非常的抗拒，甚至還會對你爆發脾氣。但在獅子座暴躁的脾氣之下，同樣隱藏著的還有他們的高智商，如果你得罪他們了，他們有可能不會對你發脾氣，但有可能讓你在不知不覺間輸得一塌塗地，甚至你自己都不知道自己輸在哪裡了，這樣的獅子座可以說是非常讓人忌憚了，遇到的話最好退避三分。
