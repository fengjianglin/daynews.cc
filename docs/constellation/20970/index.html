一周星座運勢：12.19-12.25摩羯座打開天窗，走出困境

2019-12-19 1,027條評論 3,872次閱讀 1850人點贊

看著日曆這一年只剩下13天的時間，我們要和2019年正式說再見了，回顧這一年：我們摔倒再站起來，倒下再振作這樣的過程里，我們反反覆復磨練自己的意志力，而在下半月的一周里，運勢已經顯示於眼前。12.19-12.25周運里，摩羯座脫穎而出，倒不是白羊座下周的運勢有多好，只是他們的改動較大，打開天窗說亮話，將自己的煩惱和不悅統統宣洩而出，告訴自己堅持下去才能看到更美的光芒，不再困於混沌當中，一起來看看白羊座的運程如何？（運勢參考太陽星座和上升星座）

木星、土星落入摩羯座，位於10宮11宮

木星和土星相繼落入摩羯座，這對摩羯座而言，他們的生活註定會有些改變，木星落入第十宮，從這個落宮角度來看，摩羯座會通過社會地位和其他人的贊同而獲得認同感，他們將生活上升的程度放在首要位置，在工作和生活里投入更多的精力和心力，把高度熱情放在喜歡的事情之上。木星的照耀，讓事業方面有了明顯的上升，而且佔據了很多主動權，自身的發揮空間變大。再從土星落入十一宮來看，土星進入福德宮，使得摩羯座更加渴望想要認識對自己有幫助的人，在人際關係方面摩羯座會改變自己刻板印象，將自身性格方面的缺陷藏起來，和重要的人保持更加的持久的情意。

火星六合土星，方向為出相

火星六合土星的相位，勢必要讓摩羯座在這一周當中有不平凡的機遇，讓他們性子里增添了一份忍耐謹慎，凸顯在應對別人難題的時候，摩羯座更加誠懇，心中的毅力促使自己全心全力去追尋解決辦法，即便遇見困難摩羯座也不會就此倒下。其實火星注入的能量，讓摩羯座更從容應對未來一些困難。建議摩羯座珍惜這個過程，其實每次自己努力克服困難之後的心智會變得更強，對於摩羯座來說，生活就是一場戰役，但土星又給予了摩羯座一些生活上的阻力，好在被火星消化掉，從中得到了最大的利益就是讓他們不再脆弱，反而越戰越勇。

太陽進入摩羯座

太陽進入摩羯座，使他們的野心越來越強大，不滿足於當下的收穫，他們想要得到更多的機會和幫襯，但也意味著太陽使摩羯座外在的表現更明確。生活不再那麼愜意，但與此同時帶給他們是別人所得不到的出口和機會，目標明確的摩羯座不困於混沌之下。這裡建議摩羯座，雖然內心固然有千百種想法，也要思慮周全，不要野心蒙蔽了雙眼，看不到自己真正的實力而挫敗。太陽的進入，給摩羯座帶來了許多自信，這份自信是發自內心不用別人鼓勵就有的，這就給摩羯座未來巨大上升空間埋下了伏筆，未來一周，摩羯座不再固步自封，想要走出舒適圈，迎接挑戰。

結語

每個人的生活都不會是那麼順暢，有磕磕絆絆存在著，但也因為這樣我們才可以說生活是五彩繽紛的，有絢爛多姿的紅色，也有黯淡的灰色，但我們不想就此失去信心，就好比上面的摩羯座，就算在生活里有安穩也有挑戰，他們依舊保持初心前行。在這裡要提醒摩羯座一句，生活本就「喜怒無常」，先從自身出發，不要因為別人的叨擾而亂了心智。
