獅子座鄰近12月一周，成長道路不會輕鬆，只有熬過去的人才能收穫

3 週 ago

獅子

本周天王和金星、火星產生了聯繫，在工作中，你可能完成了一個階段性的任務，工作中會輕鬆一些。有些人可能會被調到一個新的崗位上，或是突然轉變工作環境，需要重新去適應新的工作和處理人際關係。另外，你的工作和家庭中可能會有變動，兩邊會互相產生影響，雖然你會盡最大能力去應對和處理，但可能還是兩邊都顧及周全。你也可能會有重新裝潢一下家中的衝動，並且可能已經開始實施。部分人可能會離開自己的現居地去追求自己的事業目標，或是在工作中有了新的突破。

緊接著海王星在你的偏財宮順行了，你之前在投資理財、購置保險方面可能會有些不順利，踩了一些坑，或是財務上總是不清不楚也存不住錢，但現在你會擺脫當下的現狀，更有規劃的去進行理財，也可能會拿回之前朋友欠自己卻忘記還的錢。射手座新月發生在了你的喜樂宮，你的生活中可能會有一些愉快的事情發生，比如交到了新朋友或是組織一些朋友聚會。部分人可能發現了新的興趣愛好，有嘗試發展一下的心情。

臨近周末時，水星和土星、海王產生了聯繫，你在工作中可能需要去處理一些文書、證件類的工作，或是經常召開會議，你要認真對待其中細節方面的問題。你可能會遇到親戚、朋友來向你借錢，但對方所訴說的情況不一定屬實，不要因為一時同情心泛濫就把錢借了出去。部分人可能會為了處理一些家務事、置辦家居而有一筆開支，但保持清醒，不該花的錢不要亂花。

在感情上，對於單身對你來說，新月的能量會給你增添更多戀愛的機會，你可以多去參加一些社交活動，說不定你的緣分就在新認識的人里。部分人可能會和自己的上司擦出火花。對於有伴侶的人來說，你們在激情褪去之後，漸漸的回歸到平淡的生活中，更多的考慮到一些現實的問題，有些人可能會有回家見父母的計劃。但也會有人想要尋求一些新鮮、刺激的體驗，因此影響到當下的感情。

建議：成長的道路上從來都不是輕鬆的，但只有熬過去的人才能收穫甜美的果實。
