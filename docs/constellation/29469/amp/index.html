只有在生活中不再討好別人，這3個星座才能真正崛起

天天要聞
2019-12-21

生活中每個人為人處世的態度都是不一樣的，有的人不管什麼時候都能保持自我，不附和別人，也有的人為了利益委曲求全，不惜犧牲自己的尊嚴，什麼時候都把別人看得比自己更重要。十二星座中，這3個星座的人總是在妥協，委屈自己，只有堅持做自己，不討好別人，才能真正地崛起，我們一起看看吧。

1、獅子座

獅子座在和別人相處的過程中，很少會堅持自己的觀點，大多數時候都會為了維護關係而妥協，就算自己受委屈也不說出來，時間久了自然成了被欺負的對象。有時候這樣是會讓自己的路更好走一點，但是並不會改變自己在別人心中的地位，如果要把別人的快樂建立在自己的痛苦之上，這樣是非常不值得的，每個人都是獨立的生命體，都是第一次做人，憑什麼要讓著別人，只有互相尊重才能讓兩個人或者更多的人走得更遠。

2、處女座

處女座一向都很強勢，不管做什麼事情都是自己說了算，只要自己覺得不滿意的地方就得改。但是在感情中他們會委屈自己，來讓對方滿意，這樣既打擊了他們的自尊心，也對感情造成了不可磨滅的影響。每個人生而平等，誰也不比誰高貴，要想贏得別人的尊重，首先自己要尊重別人，但是遇到蠻橫不講理的人，就根本沒有比較繼續相處下去，更不需要去討好對方，做人要有自己的底線，如果沒有了底線，那這個人活著也是沒有任何意義的。

3、雙魚座

雙魚座很重感情，他們把感情看得比什麼都重要，一旦愛上一個人就會把對方當成自己的全部，而且其他的什麼都不在乎，所有的朋友都比不上對方。他們為了這段感情可以放棄很多東西，就算是對方做了很多過分的事情，他們也能原諒對方，只要有機會，他們就會好好把握。雙魚在感情中的地位很低，經常會受委屈，就算是這樣，也難以改變感情破裂的結果。與其讓自己受委屈，還不如一開始就堅持做自己，不去討好別人，不模仿別人，用真心換真心，這樣才不會讓自己後悔。
