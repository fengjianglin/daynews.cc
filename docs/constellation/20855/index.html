射手座不僅擁有有趣的靈魂，更是百年難得一遇的哲學家

2019-12-17 471條評論 4,051次閱讀 2154人點贊

作為【變動星座】四霸主之一的射手座，與生俱來的執行力不容小覷。所以我們常常可以看到，有一個想法很多，並且非常樂觀開朗的人，他們似乎在什麼領域都能如魚得水。這就是射手座。

射手座樂善好施，他們不僅擁有著自己豐富的個人生活，並且在每一領域都有所研究，不會僅僅局限於皮毛。即使他們非常博學多識，他們也不會散發出高人一等的氣息。他們通常非常喜歡分享和安利自己的體驗、資源給周圍的朋友。

射手座的性格，也非常積極正能量，他們對於這個世界的好奇心是無止盡的。與此同時，他們也會因此結交到志同道合的各路朋友。但他們不會奔波於所謂的朋友酒肉之歡中，他們更多的會以興趣為出發點與適合的朋友一起度過每一個有意思的時光。

很多時候，大家認為射手座很像個孩子，所以會認為射手很簡單，其實不然。射手非常善於思考，並且結合自己豐富的歷練和經驗，他們考慮到的遠比一般人更高階，更長遠。只是他們不會擺出一副道行高深莫測的樣子罷了。

善於鼓勵、探索的射手座。始終會散發著有趣的靈魂香氣，他們很多時候會把真摯的提醒化作一句簡單的提醒，告訴別人，但是他們不會一味過度去要求、改變他人。比較典型的就是在《嚮往的生活》中，何炅提到，黃磊曾和謝娜說過，她的人生經歷太淺薄了。謝娜哭了一晚上，只是一味訴說自己艱辛的過往，卻完全沒有參透黃磊的提醒本意。

射手就是這樣的小可愛，如果你的周圍也有這麼一個朋友，那麼請好好珍惜他們。他們不僅會帶來歡樂，更會給你帶來更多的精神財富與支持。

所以你的周圍有這樣一個射手座的朋友嗎？快快留言告訴迷落落吧～
