12星座說謊的原因，巨蟹因為太善良，雙魚心虛才騙你

天天要聞
2019-11-21

　　有些時候不得不說謊，這些說謊的原因，你覺得合理嗎？快跟沐柒一起來看看吧！

　　白羊座

　　白羊座的個性非常直接，而且根本就不太會說謊，所以當白羊座真的騙人了，那絕對是出大事了，只有事情嚴重到不能讓你知道，他才會選擇說謊！

　　金牛座

　　金牛座也是比較少說謊的星座了，因為他們為人正直，說謊對他們來說也是比較不擅長的，金牛座會說謊，通常都是一些善意的謊言。

　　雙子座

　　雙子座會說謊，絕對是想要逃避一些責任，對雙子座來說，他們是非常討厭麻煩的，如果說謊可以逃掉一些麻煩事，他們是很願意說的。

　　巨蟹座

　　巨蟹座實在是太善良了，根本就不懂耍什麼心機，所以當巨蟹座說謊，那一定是為了保護你或是保護別人，總而言之，他們的謊言絕對沒有惡意。

　　獅子座

　　獅子座會說謊，通常都是為了鞏固自己的面子，畢竟獅子座是比較虛榮自大一點的，有時為了營造出自己的氣勢，不得不說點謊呀。

　　處女座

　　處女座也算是直言直語的星座，講話一針見血很真實的。所以他們說謊通常都是不想要讓別人擔心，才會用謊言來營造出自己很好的感覺。

　　天秤座

　　天秤座的謊言通常都是出自於禮貌，像是有些話、有些事實會有點傷人或是不符合禮儀，天秤座為了顧全大局，還是會撒點小謊的。

　　天蠍座

　　天蠍座的謊言大部分都是為了布局，畢竟天蠍座敢愛敢恨，沒有什麼不敢說的，如果他們選擇說謊，那一定是心裡正在做什麼打算。

　　射手座

　　射手座是非常快言快語地，想到什麼就說什麼，這樣的射手會說謊，肯定是有所顧慮的，通常射手座會為了不要破壞氣氛，而說一點小謊的。

　　摩羯座

　　摩羯座還是稍稍自私一點點的，他們會說謊，就代表事實對他們並不利，摩羯座為了維護自己的權利，即便要說謊也在所不惜。

　　水瓶座

　　水瓶座是不太會說謊的，但偏偏他們的話卻很像是「假的」，也許是他們的思維太天馬行空，才會聽起來怪怪的，當水瓶座會說謊，就是想「裝穩重」，希望自己不要再這麼輕浮。

　　雙魚座

　　雙魚座會說謊，那絕對是心虛了！雙魚座其實不太喜歡承認自己的錯誤，也不喜歡被指責，所以必要的時候，雙魚座是會選擇說謊的。
