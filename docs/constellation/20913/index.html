摩羯座，

摩羯座的人沒有定性，他們一會想做這個，一會又想干那個，三心二意，結果最後什麼事情也沒幹成。浪費了大好的時光和金錢最後卻一無所獲，很多人都在背後看摩羯座的笑話。直到受盡了別人的冷眼和嘲笑，摩羯座才有了成功的信念，他們學會了從一而終，在冬月25起，摩羯座的人就會獲得成功。冬月25起揚眉吐氣，摩羯座會時來運轉，大富大貴。

雙魚座

雙魚座的各位大都是溫柔隨和，脾氣溫順的老好人，在工作上踏踏實實，勤奮努力，在家庭方面也能夠耐心細緻，將一應事物都處理的井然有序，這些好的品質經過了歲月積累，自然就會變成上好的氣運。所以雙魚座的朋友們在冬月25起中將會好運多多，並且還有貴人前來加持，更是前路坦蕩，順風順水，各種喜事當然也自己找上門來。

天蠍座

天蠍座的人通情達理，個性沉靜，在思考問題的時候都很全面，不會讓一切問題都浮於表面，只要是領導和朋友交代的事情都能很順利地完成，所以能在工作中得到很多人的幫助。冬月25起，財運爆棚，往後守財為最重要的目標，千萬不要盲目拓展新的業務，只要繼續勤勤懇懇的工作下去，就一定能夠過上幸福的生活。
