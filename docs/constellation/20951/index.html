最容易被騙星座女TOP3　第一名男生花式出招一哄就暈船

2019-12-19 3,191條評論 4,942次閱讀 3805人點贊

記者李玟儀／綜合報導

你是個容易被男人花言巧語迷惑的人嗎？星座專家Amanda老師在《命運好好玩》中分析最容易被哄騙的3種星座女生，分別從太陽星座、上升星座，以及主管思考、判斷的水星，與負責戀愛態度的金星來分析。

第三名 處女座「只見樹木，不見樹林」

最容易被男生騙星座女。（圖／翻攝自YouTube／命運好好玩）

處女座的女生最容易被「銀彈攻勢」收服，一般來說處女座給人的印象是聰明、精明、很會盤算，甚至擅長一眼看出對方的缺點，在談戀愛的時候，務實的處女座會特別考量另一半有沒有一定的經濟能力，就是因為這樣，特別容易陷入「銀彈」陷阱中。

舉例來說，約會時男生開名車接送，處女座女生當下就會認定對方是有一定的經濟條件，卻忘了觀察名車是不是跟朋友借的，處女座的女生特別容易被這種「片面的資訊」給騙了。

第二名 獅子座「太過單純，沒想太多」

最容易被男生騙星座女。（圖／翻攝自YouTube／命運好好玩）

獅子座女生最容易掉進「帥氣外表」的陷阱中。獅子座特質強烈的女生自我要求很高，同樣的也會以高標準來看待另一半，只要對方長得帥，大概就可以拿到80分了，被迷惑後很容易忽略對方的缺點。

第一名 雙魚座「缺乏判斷，認人不清」

最容易被男生騙星座女。（圖／翻攝自YouTube／命運好好玩）

雙魚座最容易被花言巧語迷惑，和獅子座、處女座不同，後兩者可能是真的享受過另一半的花招，但是雙魚座是連看都沒看到，只要對方出一張嘴就會被哄得暈船，要特別留意自己的盲點。
