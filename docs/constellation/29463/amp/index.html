一秒覺得舒服！最能療愈十二星座的那一瞬間

天天要聞
2019-12-21

牡羊座：打完戰的那一瞬間

當那些該討的、該戰的、該拼的，該努力的那些都很用力的事情完成後，鬆懈下來的牡羊座會覺得現在的他就是全世界最快樂的人。其實這種反差很讓牡羊座著迷，因為他們很難閑下來，可是那個忙與忙之間的空檔，就是他們充電的最好的方式了。

金牛座：美食一入口的那一瞬間

吃貨絕對是金牛座當之無愧！他們熱愛各種感官的享受，食物尤其是特別的重要，當一天的工作繁忙卸下後，可以去最愛的店點上一壺酒，吃上他最愛的那些特色小菜，金牛座會覺得這一切都是那麼的值得，那麼樣感覺活著真好啊。

雙子座：與人分享趣事的一瞬間

雙子座是一個很喜歡跟大家分享發現的好東西的星座，如果對方還真的能因為他的分享，而找到興趣或是顯得很愉悅的話，雙子座會覺得特別的開心，他們天生喜歡一群有共同興趣的人可以一起在上面琢磨，這對他們來說是很重要的精神糧食。

巨蟹座：與老友親人膩在一起的一瞬間

巨蟹座喜歡回憶，也喜歡老友相聚在一起的感覺，他們很喜歡那種他覺得那些他最重視的人，現在都在他身邊，大家一起敘敘舊，懷念過去那些有趣的事情，這會讓巨蟹座覺得幸福感倍增，因為這些都是他很熟悉的人事物。

獅子座：站在舞台被鼓掌的那一瞬間

當獅子座完成一件他覺得他很努力的事情，而且被大家看到，被大家當做模範時，獅子座會很驕傲自己是能夠影響他人影響世界的。那如果，大家又能對他報以很肯定的態度，獅子座絕對是會療愈到雲端啦！

處女座：看到缺失的東西被修補的一瞬間

處女座有一種化腐朽為神奇的的能力，什麼亂七八糟的東西，在他們手上，就是能被修復得井然有序！所以當處女座將一些亂糟糟的事物擺平後，那個看起來完整完美的感受，會讓處女座得到全所未有的療愈啊！


天秤座：當都看起來那麼順眼一瞬間

天秤座喜歡看到一切都是那麼的有秩序、舒服、和樂的畫面！所以他們希望一切的狀況都在他們的控制下是那樣的美好，這樣的性格是不管工作、人際關係、家庭都是一樣的狀態，天秤座就會覺得圓滿。

天蠍座：挑戰別人不敢做的事的那一瞬間

天蠍座天生就是一個喜歡有點危險事物的人，如果天蠍座覺得按照社會的框架在過日子，常常會覺得蠻厭世的，所以他們很喜歡挑戰一些別人不敢做、比較暗黑危險的事情，只要不犯法，那又有什麼不可以呢？

射手座：當新鮮的事物來到生命里的一瞬間

射手座是一個不太喜歡日子太過於平庸的一個星座，他們需要去闖、需要各種新鮮的事物進到生命里，所以當他們發展各種新的可能時，都會在那段狀態里感到有活力！這不管是戀愛、生活、工作都是這樣的狀態喔！

摩羯座：當努力的事情終於開花結果的一瞬間

摩羯座的自律性很強，他們常常有一些蠻久遠​​的目標，是需要時間才能看到成果，當摩羯座發現他們努力已久的事物開始長出果實時，他們就會覺得這一切的努力都是非常值得的。

水瓶座：熱愛研究的事物有了回饋的一瞬間

水瓶座是一個富有科學家精神的人，他們喜歡沉溺在某項小眾的事物里，那會讓他們覺得那種與社會上格格不入的感覺被解放，所以當他們喜歡的小眾事物也被大家喜歡的時候，他會覺得這真的是一件很愉悅又得意的事呢！

雙魚座：什麼都不想的那一瞬間

雙魚座就是那種想太多想太雜，然後把自己搞得很煩的人，所以他們最好的療愈狀態就是那個什麼都不想的自己，一個完完全全將腦袋凈空的自己，所以他們最療愈的方式就是讓自己回歸到一個零的狀態，反倒是特別覺得舒服。
