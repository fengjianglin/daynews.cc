01.

愛的能量路線

：坐下來嘗試拍張照？

一樓拍照，六層觀展。

歡迎來到3.3大廈，當你看到「粉色」「圓球」「花園」，說明你距離愛的能量越來越近了，拍張照讓咱們繼續往前走。上扶梯，你開啟了一場通往2020年的旅程，路程稍長請耐心走完，畢竟所有神秘的禮物都值得慢慢揭曉喲。

愛的能量：3.3大廈一層打卡區

02.

愛的指引

：看到星辰怎麼忍得住？撒開了拍吧！

「即刻啟程，探索關於愛的能量宇宙」

這是通往宇宙的門，當有人類造訪，它便開始釋放能量。透過它我們彷彿能看到四元素的能量宇宙。我們曾經虛構過這樣一個故事，或許在很多年前，十二星座一起探訪能量之眼睛，它們驚奇的發現：愛是一切的起源。（溫馨提示：作為展覽第一個打卡點，穿上淺色系服裝，讓它為你印上「愛的痕迹」——星辰打在衣服上的光）

《奇點》，浮思*，單頻有聲彩色影像，能量之眼展覽現場

「愛是你所要的」

——來自巨蟹座的自白

倘若你看過《愛的藝術》第一季，那你一定了解這位被大眾定義為「壞女孩」的藝術家。如果沒有，那我們重新來認識一下這位柔軟的巨蟹座——翠西·艾敏。那位用炙熱的紅心包裹著藍色熒光手寫體文字，給大家傳遞愛的能量的堅強女性。

翠西·艾敏，《愛是你想要的》，限量電子版本，展覽現場

04.

愛的能量之「四元素」

：探索自己的星座，開拍吧！

火元素·信仰

（白羊座、獅子座、射手座）

「信仰最大的敵人是人類自己。」

簾幕的「紅」便為你預先了解火象星座。當你看到沙發，它有著邀請你入內的王者風範，唯有足夠自信才能駕馭它。懸掛的燈籠，會在視聽與氛圍的渲染下，讓你感受到它的搖擺與控制。

安德魯·托馬斯·黃，《集合》，單頻彩色有聲影像，火元素空間展覽現場

安德魯·托馬斯·黃的《集合》，會帶給你一場有儀式感的獨白。這場集合里白羊座是「半裸爬行」的演繹者，他們是拓荒者、戰士、為目標奮鬥的人；獅子座就如同影像里的燈籠，如同「信仰」，他們以忠誠、高尚和重視聲譽的特質來啟發別人；射手座是探索和尋找，他們永遠渴望走的更遠，得到進一步的發展。

風元素·心智

（雙子座、天秤座、水瓶座）

「財富與秩序正在瓦解，一切都將重組。」

掀開風的門帘，踏上這充滿神秘的空間，走進風象星座的心裡。在這裡，戴上耳機沉浸聽「新憂傷」與黑川良一的《海妖5》探索數據的美學，打破、重組，將跨時空元素置於當代語境重新發聲，試圖讓世人大吃一驚，這似乎是黑川良一和「新憂傷」在起擬一個關於經濟的新篇章。

黑川良一&新憂傷，《海妖5》，單頻彩色有聲影像，風元素空間展覽現場

它會讓你體會到風向星座的「智」，畢竟他們負責把這些信息告知所有人。水瓶座所含有的前瞻性，以及抽離感都會在這裡得到體現；作品標題的靈感來自於希臘神話的海妖，她們引誘水手進入危險境地，象徵了導致經濟崩潰的市場狂熱。數字作品渲染的視覺形態和聲音相組合，其強度的漲落與經濟衰退的形式保持一致。當財富與秩序正在瓦解，萬事萬物既將重組，感受「風元素」注重心智的養成，以及探索世界的新鮮視角。

暗藏蘇珊·米勒星座秘密

當你來到這個部分，你一定發現了蘇珊·米勒成長照片間於風元素與水元素之間，讓我們來大膽猜測，她是風象星座or水象星座？

「蘇珊·米勒會客廳」照片牆，展覽現場

水的流動，是能量循環的象徵，而火象和風象更能主動自我表達，因為他們總在「釋放」。有時甚至是全然沒有節制地「傾瀉」能量：其中，火象星座主要通過直接的行動表達，比如行動力爆棚的白羊。而風象星座則是通過社會互動和語言表達，比如口若懸河的雙子。

菲利普·克萊默，《瀑布》，單頻彩色有聲影像，「蘇珊·米勒會客廳」展覽現場

水元素·情感

（巨蟹座、天蠍座、雙魚座）

「看似平靜的水面，實則暗潮湧動。」

水象星座比其他星座更感性、更富有同理心，能夠敏銳捕捉情感彷彿是水元素賦予水象星座天生的超能力，這裡水滴就如同從雨林流出來的眼淚。水象元素代表的是深層次的感情和感覺的領域：從強迫性的狂熱到摧毀一切的恐懼，乃至對世界萬物的大愛。

水的本質是清潔與凈化，同時它又具有神秘感。巨蟹座將愛的能量引向家，它的水如同家裡浴缸的水讓你倍感舒服；天蠍座將愛的能量引向強烈的親密關係，它的水表面看似平靜，實則暗潮湧動；雙魚座將愛的能量引向眾生一體的歸屬感，它的水就如同眼淚，讓你想去呵護。

AES+F，《輪迴》，單頻彩色有聲影像，水元素空間展覽現場

初看AES+F的作品《輪迴》，你大約會被它們弔詭的超現實「未來感」與「古典」融合設計震驚到，那些作品裡看似輕快明亮的綠色，充滿了童真的趣味。但當你細細觀察畫面中人物的面孔、眼神和神態時，卻會發現一個完全不同的世界。藝術家將人物放在一個超現實的未來場景中，讓他們與現代世界產生衝突。伴隨巴洛克式合唱聲，感受「水元素」內心世界故事、秘密的誕生，暗涌情緒的誕生與碰撞。

土元素·物質

（金牛座、處女座、摩羯座）

“他們與生俱來了解物質世界的運作方式」

土象元素代表著謹慎、深思熟慮、保守，他們關切的是真實的世界，不像火元素那麼憧憬未來。在他們看來，能看得見摸得著的東西才是最實在的，說的天花亂墜還不如直接行動得出結果。他們生來就了解現實世界是如何運轉的，這種能力能令他們比其他星座更有耐心，更有自我約束力。

安德魯·托馬斯·黃，《肉體巢穴：第9章》，單頻彩色有聲影像，土元素空間展覽現場

安德魯·托馬斯·黃的作品《肉體巢穴：第九章》創造了一種沉浸式技術-反烏托邦的場景，呈現了大災難之後的數字煉獄，殭屍、生化巨人、空物件佔據了荒蕪的機器世界，在數字瓦礫和虛擬的混亂場景中攀爬。暗中契合了藝術家對未來的數字的一個大膽想像，當這樣奇特、新穎的情境交織，自然會激發不同人對它的理解，同時也用情境的創意不斷突破人們對未來世界的想像。

05.

愛的能量之「探索愛」

：和王嘉爾一起探索「愛」吧！

王嘉爾 x 達彥

「讓我們繼續發現愛、探索愛、喚醒愛。」

用一個點、一條線、一個平面，表達內心的掙扎。打破世俗偏見、規則，堅定自己從「心」出發獲得愛的能量。世間大多數心痛都是自發的結果，若再被歡樂反襯便會更加令人難過。

王嘉爾 x 達彥，《BULLET TO THE HEART：0328》，三屏交互影像裝置空間，特別能量空間

關於蘇珊·米勒

蘇珊·米勒，全球星座女王、美國著名的占星家、《紐約日報》專欄作家、暢銷書作家、實業家。1995年，推出了占星區網站(Astrology Zone)。由於該網站每月發布的星座綜合運勢預測非常準確，使蘇珊·米勒受到了來自全世界範圍的讚譽。唯一經由蘋果公司邀請在手錶上發布星座運勢預測的占星師、擁有650萬獨立讀者以及2000萬的頁面瀏覽量。

作品被眾多重要媒體爭相轉載，並且還被翻譯成中文、土耳其語、西班牙語、日語等多種語言，坐擁全球數百萬讀者。根據自己對星象的研究，準確地預測到了路易斯王子的出生周、碧昂斯的婚禮年、「小甜甜」布蘭妮的復出以及奧巴馬總統的連任。

關於「LOVELOVELOVE愛的藝術」

「LOVELOVELOVE愛的藝術」是雲圖映畫出品的原創展覽品牌。於2018年11月在北京首展的《愛的藝術：親密》，在3個月展期中獲得了十萬觀眾現場觀展，並在新浪微博、小紅書等社交媒體引起數億次傳播討論，成為現象級展覽項目。展覽也將在重慶、瀋陽、上海、深圳等地巡展。而隨著「愛的藝術」第二季展覽「愛的能量：蘇珊·米勒星座藝術體驗展」在北京開幕，「愛的藝術」系列將成為中國第一個全國巡迴、全年持續、全網互動的原創展覽IP，創造大眾藝術傳播新紀錄。

關於雲圖映畫

「雲圖映畫」是創新的藝術內容品牌，也是頂尖藝術IP運營機構。我們的創始團隊融合了藝術、時尚、科技、傳媒基因，通過藝術內容，連接大眾與世界。雲圖映畫成功打造了「大藝術家」、「愛的藝術」的原創IP產品，並通過展覽、視頻、出版等形式，影響年輕一代用戶。雲圖映畫的使命是探索人類最傑出的人文藝術，為每一個人創造一座「美的殿堂」。
