只要一急就會發火的星座，太衝動，自控力差

2019-12-18 33條評論 1,018次閱讀 57人點贊

在朋友圈子裡，我們與每一個人相處時都知道對方的脾氣如何，如此才能更好地打交道而不輕易激怒對方。有些人忍耐性很好，別人一般很難惹火他們；而有些人的脾氣就很大，就算沒有人招惹也因為自己急起來而發火。就如十二星座中的以下三個火象星座成員，他們只要一急就會發火，顯得太衝動且自控力差，下面就一起去看看吧！

白羊座

白羊座可謂十二星座中忍耐性最差的星座，他們做任何事情都是風風火火的，倘若誰的動作稍微慢一些他們就不高興，有時候一急就喜歡發脾氣，看起來太衝動，自控力差。事實上白羊座確實沒有多大耐心，對人對事都是如此，比如當他們對一個問題百思不得其解時就急起來就發火，卻沒有想過是不是因為太難還是人力不能為之。

對人也是如此。當白羊座跟你共事時，他們一開始跟你不太熟悉就不會挑你的毛病，可是相處久了他們就不會對你客氣，而是有什麼話直接說出來。你若是跟不上白羊座的做事節奏，他們往往一急就會對你發火，雖然白羊座只是情急之下對你發脾氣並非出於惡意，卻讓你意識到他們比較衝動，而且自控力差，不會控制自己的脾氣。

獅子座

獅子座雖然做人做事有一定的原則，平時也懂得剋制自己的情緒，但是在情急之下他們還是比較容易發火的，給人留下太衝動，自控力差的印象。實際上獅子座的脾氣就比較急，且容易一急就會發火，讓身邊的人頭疼不已，好在他們平時懂得寬容別人，真正大發雷霆的機會不算多，只是一些小脾氣一直少不了。

有些時候獅子座本可以避免一些無所謂的發火，只是由於他們對待每一件事情都極為認真，容不得別人有任何閃失而影響了進度，才會一急就朝別人發火。雖然獅子座的火氣發泄出來可能傷害到別人的面子，但是他們往往對事不對人，對方覺得自己受到了傷害也是誤傷，不可否認的是獅子座還是有太衝動和自控力差的一面，才會一急就會發火。

射手座

說到射手座的脾氣，大部分人都豎起了大拇指，因為在彼此相處時他們都是一副樂呵呵的樣子，怎麼看都不是那種喜歡發脾氣的人。射手座對人友好且不愛發脾氣不假，可那只是他們在沒有被別人招惹或情急情況下的反應，倘若他們遇到不順心之事或者一急就會發火，還是表現出太衝動和自控力差的一面來。

因為射手座耐心不足，他們只要急起來就暫時不想控制自己的脾氣，想發火就發火，沒有一點偽裝做作的意思，完全是真性情的表現，這一點也讓他們得罪了不少人。不過射手座倒不想改變自己，再說本性難移，他們要想改掉一急就發火的毛病難上加難，只能保持一份好心情才能每天嘻嘻哈哈的。所以說射手座還是太衝動和自控力差，往往一急就會發火。
