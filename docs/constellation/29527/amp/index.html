運勢 | 12星座接下來要如何做，才能開啟“金磚人生”？

天天要聞
2019-12-21

2019年，土星跟冥王星的多次相遇，給我們每個人的人生都帶來了影響——或者準確來說，是一種衝擊，好似讓你離開你原先的習慣，讓你走出舒適區，我們也越來越意識到，只有改變才能真的成長起來。2019年，每個人的工作或者所屬行業都出現了新的變動，這些變動也許是顛覆的大改變，也許是我們需要調整個人的事業規劃，但不管是主觀還是客觀的原因，我們的初衷卻從未改變——去實現個人的追求，讓自己的生活變得更加有品質。

我能感受到2019年，身邊人最大的變化，就是更加野心勃勃了，不知不覺之中，大家談論的工作「搬磚」變成了走向「金磚人生」的「搬金磚」，目標、方向變得更加清晰，所以，希望每個人，可以在接下來的時間裡，調整自己，從辛苦的搬磚，變成有品質，實現個人價值，走向「金磚人生」的「搬金磚」。

那麼，12星座們，你要如何做，才能開啟金磚人生呢？

基本星座

（白羊座、巨蟹座、天秤座、摩羯座）

你的努力：

你真的給自己施加了蠻多壓力的，這個2019年，有數次想要放棄，甚至瀕臨情緒大爆發的時刻，但你真的很不錯，在關鍵時刻忍耐了下來，理智了起來，你的眼界變得更加「大局為重」了。雖然我知道，這個過程讓你覺得孤單，讓你無法完全的去相信他人，只能憑藉自己的實力和才華去攻克難關，但也在這期間，你不斷的提升了自我——熬過了孤獨，才能品嘗到收穫的喜悅。我知道你們在人際關係上經歷了一些不愉快的事情，這些不愉快也影響到你們很難再次付出真心，但相信我，你不會一直如此的。

走向金磚人生：

基本星座的你們，要開始大展拳腳了，是不是心裡也格外的躍躍欲試，充滿著期待。你們在接下來則是要堅定一條路，這條路你知道並不順暢，但是結果卻是你想要獲得的，那麼只能堅持和鼓勵自己，同時延續在2019年你們所做的努力——鋪墊人脈資源，維繫關係，以及讓自己眼界變得更高瞻遠矚。從搬磚走向搬金磚，你們決心、毅力都滿足，唯一需要注意的是，在這段搬金磚的路途上需要找到合適的隊友，你需要被分擔一些擔子，試著去接納和相信他人，你的金磚人生就會開啟了。

固定星座

（金牛座、獅子座、天蠍座、水瓶座）

你的努力：

2019年，真的一直在較勁兒，一直不服輸，倒不是說你們多想要成功，多想要贏，就是有太多「氣不過」了，你們知道嗎，就是因為這股耿直的個性，反而被一些人利用了，你也因此吃了虧，工作也好，生活也好都出現了一些變動，但你們就是有頑強的生命力啊！你們會涅槃重生，你們會轉化壓力變成動力，你們也開始意識到自己技能和經驗的缺乏，在不斷的學習和嘗試陌生領域的事情，我知道你們在等待一個時機，讓自己真正的轉型和蛻變，放心，你不會等很久的，但請你保持學習的狀態，繼續提升自己的核心競爭力吧。

走向金磚人生：

你們真的還是要克服內心的恐懼，我知道你們內心存在許多事業發展上的顧慮，特別是到了一定的位置，又或者到了某個年齡（心態）的緣故，反而更加害怕改變，或者失去什麼，勇往直前的基因也會因為這些影響而漸漸消逝，但你們如果真的想要走進「金磚人生」，真的就要拿出決心和勇氣。克服你內心的恐懼，有非常多的方法，拿出你們的邏輯和分析能力，把機會一分為二，一部分看優勢另一部分看劣勢，去權衡它分析它並去假設一些問題發生之後是不是有能力去承擔它，我相信你就會明白要怎麼做了。你們是最有實力搬金磚的一群人了，也一定會因學習和努力而過上更有品質的金磚人生。

變動星座

（雙子座、處女座、射手座、雙魚座）

你的努力：

2019年對你來說衝擊力比較大，其實就是一個「摧毀重塑」的過程，好比你原先堅信的東西發現有了不為人知的一面，也因為人心、鬥爭的關係，讓你處於「風口浪尖」，也需要你在這個混亂之中做出選擇。你們有在挖掘自己的潛能，在2019年你們不斷的打破習慣，走出舒適區，願意肩負重任，逐漸被人看到了光芒，因實力被不斷認可，外在存在諸多機會可以轉換，你的含金量被提升了，這是你整個2019年辛苦換來的成果。有沒有感受到，自己在今年經歷了迷茫、自我質疑之後，反而找到了一條全新的路，你的心態也會從谷底中重新上揚回到屬於你的位置，你要堅信，努力真的會換來回報。

你們是深愛品質生活的，這就像是你們的個性一樣，追求細節，擁有夢想，內心心懷一個藝術家的夢，這也是為什麼，你們熱愛完美主義，留戀細節上的點睛之筆。在接下來的職業發展上，你們要儘可能讓自己放鬆一點——太過於注重細節，微觀，你們真的很容易錯失很多機會，而且一旦格局變大，勢必容易忽視掉一些微小的問題，但變動星座們，身邊總是會有人來幫助你的，你完全可以放下心防去信任他們，他們帶來的往往是1+1>2的效果。因你們身上含金量提升，在走向「金磚人生」的路途上也會順遂許多，但記得，減少你的抱怨和心態上的糾結，多想一想，你是「搬金磚」的人，努力創造價值，取悅自己，是一件很美好的事情。
