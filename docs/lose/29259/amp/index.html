為什麼業餘乒乓球選手很難達到專業水平

打乒乓球為什麼要規範擊球動作？

規範標準的擊球動作，並非誰規定的。而是前人無數次反覆探索實踐，演化改進，總結提煉出來的。是簡單合理，快速高效，舒展利落，命中率高，便於發力，還原，銜接的最佳擊球方法。使擊出的球，能夠達到快，准，狠，變，轉的效果。所以，規範的擊球動作，是前人經驗的結晶，也是後人進步的捷徑。

為什麼業餘球友動作多數不規範？

成人打球，不象兒童那樣，有教練嚴格要求，有理想目標，初期，都是為了健身出汗找樂趣，多數不請教練，憑兒時水泥台上的「三腳毛」功夫，跟球友們瞎混，業餘球友們多數也不懂規範練球，即便有懂行的，也懶得去說，因為，說也白說。對於成人初學者來說，與其站在牆角枯燥無味地練徒手揮拍，那如上台隨心所欲地瞎打痛快！於是，盲人騎瞎馬，走到那算那。整天和球友相互叫板，沉湎於開場比輸贏之中。以為，這樣打下去就能提高。等到真正喜歡上乒乓，希望再上一個台階的時候，受到錯誤動作的制約，很難提高。此時，才想到規範動作，發現已經很難學好了。多年孤僻動作已經形成，且根深蒂固，要改，如同「浪子回頭」、「癮君戒毒」，談何容易？就象小刀劃木板，前邊開始就劃歪了，後邊再劃，刀子順著原來軌道就走了，越劃越深，再想劃直極難。所以，改動作成功的，屬少數執著追求完美者。最後，多數人選擇既成現實，任其發展的態度。

為什麼業餘球選手很難提高到專業水平？

「天天苦練殺敵絕技，夢想將來成為高手，可是球齡比小隊員年齡還大，卻屢屢敗在人家拍下」。這是我多年苦思不得其解的一個問題。多數球友不明白，動作不規範，是制約技術發展提高的瓶頸；沒有紮實的基本功做基礎，夢寐以求的高水平就成為空中樓閣。業餘再練也達不到專業的高度。就象在平房的基礎上，怎麼也蓋不出摩天大樓一樣。終於弄明白此道理後的業餘球友，就象樓房蓋到半截，發現基礎不牢固。又面臨二難選擇，要不就此封頂；要不推倒從來。多數人選擇了前者。也有象我一樣痴迷者，決心推倒從來，於是停止打比賽，先改動作，後練基本功。當然，從頭再來，比新手初學還難。就像房地產商拆遷舊房蓋新樓一樣麻煩，多年養成根深蒂固地孤僻動作，就像祖宗幾代就在此居住的「釘子戶」一樣，一時半會也很難清除。等到苦苦練習若干年後，終於練得有點樣了，發現，年紀老了，反應慢了，力氣沒了。好不容易學會個拉弧圈，腰卻拉傷了。在醫生警告下，無奈，只能變削球防守打法，腰倒是不痛了，可前後左右，急速奔跑，老腿又跟不上趟，只能練球，無法比賽。沒辦法，前功盡棄，再換成「名聲不太好」的高膠，重新學「滾，刮，磕，撇」，不用退台，站那兒慢慢「瞎督打」吧！五年換了三套打法，何談精雕細琢？憑啥贏球？自此，徹底喪失了打敗專業隊員的想法了。人過五十而知天命！確實如此。

但此時，我仍然無怨無悔，我努力了！我嘗試了！我知道球怎麼打了！我有資格說乒乓了！我不再盲目迷信崇拜高手了！

為什麼業餘球友多數不會轉腰發力？

主要是因為乒乓球反彈力大，速度快，對手感要求高。初學者很難控制球，且還喜歡開場比高低。新手比賽時，無須主動發多大力擊球，將球碰回去不下網出台就可能贏對方。日久，自然形成只用手臂打球的錯誤習慣。

所以，學打乒乓初期，不經高師指導，不按程序練球，所謂的「以賽代練」打著玩，極易養成孤僻動作，步入誤區，歧途難返，封閉上升空間。我見過沒上過軍校當將軍的；也見過沒上經管學院成為企業家的；可沒聽說過未經體校系統訓練打世界冠軍的。

為什麼一個動作進行要分解練習？

一個完整的快攻擊球動作看似簡單，實際是很複雜的，腳，腿，髖，腰，肩，臂，腕，手指等，各部位依次先後發力，合理協調運動，最後將力量傳遞，集中作用到球拍上，就是通常所說的「鞭打效應」，動作做得規範標準，流暢舒展，發力集中是很不容易的。不經高師指導，不反覆練習，絕難掌握。人的注意力有限，動作不熟練時，往往顧頭不顧尾。我們最好把魚分成幾段，一段一段地吃。先分段消化，然後串起來綜合練習。這樣，容易掌握一些。

千萬別小看分解動作這一點，我沒上過體校，可年輕時在部隊當過團示範班長，一個正步，先學原地擺臂，再單練踢腿，做標準後，再進行一步兩動，一步一動，這些全過關了，才能連貫起來走出像樣的正步。打乒乓比走正步難多了，正步只要動作規範，步子標準就行。而打乒乓動作標準規範，只是打好球的基礎，更難的是把變化多端的來球，打回對方球台上。所以，掌握熟練的徒手規範擊球動作更為重要。徒手動作規範，打球時未必能動作規範，徒手動作還做不規範，在應付快速變化來球時，那就更沒了樣了。

為什麼要進行定點定線和多球練習？

我們經常看到乒乓賽場上，左攻右打，精彩激烈，雙方拚命著對方死地而後快。這就是競技運動的殘酷性。這大大誤導了業餘球迷，以為乒乓就是那樣打。俗話說：台上三分鐘，台下十年功。運動員台下苦練基本功的時候，那些單調無味的，千篇一律的，循規蹈矩的，循序漸進的點線多球練習，很少為人所知。造成多數球友不明白，練球和比賽的自然不同。平時賽練不分，甚至只賽不練。還美其名曰「以賽代練」，「練與戰緊密結合」。

定點定線快攻練習是練擊球的穩定性，以提高控球能力和命中率。專業隊員每天規定5000板。據說王楠一個球能打4500板不掉球，所以，實戰中，才能做到想打那打那。不象業餘隊員，打那算那。多球訓練確實是個迅速掌握技術的好方法。比如練拉球，開場比賽，一場球下來也沒幾次機會讓你拉球。一盆球能頂一周打比賽的拉球數量。況且，一門心思地連續拉，能根據球下網出台情況，及時進行用力方向和控制板型的手指微調，便於肌肉形成記憶，動作定型。
