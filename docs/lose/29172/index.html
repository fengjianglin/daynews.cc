髖關節疼痛的朋友，查明病因後，可以嘗試這些鍛煉方式來緩解癥狀

2019-12-23 2,436條評論 3,619次閱讀 2222人點贊

除了膝關節，髖關節疼痛可能是最常見的關節疼痛了，尤其是那些長時間開車的朋友、長時間從事重體力勞動以及經常進行瑜伽鍛煉的女性朋友，另外長時間飲酒以及使用激素等誘因導致的股骨頭壞死也是髖關節疼痛常見的原因，那麼有了髖關節疼痛的問題，如何進行康復鍛煉呢，咱們來聊聊！

對於有髖關節疼痛的朋友，建議不要盲目的進行康復鍛煉，因為不是所有的康復鍛煉都適合所有的患者，進行康復鍛煉之前應該去醫院進行相關的檢查，明確自己存在著咋樣的問題，再進行相關的康復鍛煉，這樣會更安全可靠一些。

做什麼樣的檢查呢？

對於髖關節疼痛的朋友，我們首先應該檢查骨盆的平片或者是痛一側的髖關節正側位片，但可以初步了解患者的關節是否存在著髖關節間隙的狹窄，是否有大量的骨贅形成以及髖關節的整體發育是否正常。如果我們發現患者的關節間隙狹窄，那麼多見於股骨頭的壞死或者是髖關節骨關節炎導致關節間隙軟骨損傷；如果x線發現在股骨頭與頸連接的部位有異常的隆起，那麼很有可能患者存在的髖關節撞擊綜合症。

而且不代表進行了x線檢查，就完全排除了患者的問題，有的時候早期的病變或者是髖關節軟組織的問題，在x線上是不顯影的，所以我們也建議有髖痛的患者要進行髖關節的核磁檢查，如果是單側的髖關節疼痛，甚至應該進行單側的髖關節核磁檢查。因為單髖的核磁檢查可以更好的判斷髖關節盂唇是否撕裂。

如果經過檢查以及醫生的查體，患者的癥狀不是特別的重，沒有特別嚴重的康復鍛煉禁忌，那麼可以進行適當的康復鍛煉，幫助自己維持關節的靈活性增強關節周邊的肌肉來幫助自己緩解癥狀。

要是一些常規的，相對來講比較簡單的髖關節康復鍛煉辦法。

建議朋友們在進行髖關節康復鍛煉之前可以洗個熱水澡，這樣可以使肌肉處於最放鬆的狀態，不容易在康復鍛煉的過程當中造成損傷，或者是因為肌肉的緊張導致疼痛。

●抱膝：做這個動作的時候要躺在地上，做一側膝關節的保膝動作，另一條腿一定要保持伸直，用兩隻手抱住屈膝的膝關節維持10秒鐘左右，之後慢慢的鬆開膝關節，建議做動作的時候要呼氣。

●髖關節旋轉：這個動作需要平躺，屈膝屈髖90度，之後向身體的兩側旋轉髖關節，請大家注意，如果膝關節向左側扭轉那麼頭要向右扭。做這個動作的時候，堅持20~30秒左右之後慢慢的恢復原位，向另一側旋轉。這個動作不僅對髖關節有利，而且對腰椎也有一定的好處。

●臀部以及下背部拉伸：這個動作的時候也需要平躺，用雙手將雙膝向胸部做抱膝動作，要注意膝關節蜷向胸部的時候要做呼氣，建議堅持20秒到30秒左右，如果做不到標準位置也是完全可以的，只要堅持做就可以。

●髖關節屈曲：做這個動作的時候需要我們站立位，一側腿伸直，另一側腿向上抬，建議抬到臀部的高度或者是稍高的高度也是可以的，如果不能抬到臀部，那麼只要抬到舒適的位置也可以。如果單腿獨立做這個動作比較困難，可以用一隻手撐住牆壁，或者說是扶一把椅子，做這個動作也是完全可以的。

●髖關節伸展：這個動作的時候需要我們站立，最好扶住一把椅子，然後將一側腿，盡量向後伸，做動作的時候盡量支撐的腿要伸直，向後伸的腿可以堅持4~5秒鐘，兩側腿分別做10~15次。

●髖關節外展：這個動作需要我們站立，用一條腿支撐身體，另外一條腿盡量向側方外展，而且要求支撐的那條腿要一定伸直，否則容易導致臀部的扭轉，可以保持這個姿勢3~5秒鐘，外展具體的位置以不誘發髖關節疼痛為主，具體能外展到多大角度會隨著康復鍛煉的增加而增大。

●向後踢臀部：站立，儘力用小腿向後去觸碰臀部，建議每側做5~10次。一做這個動作的時候一定要慢，以免摔傷，可以扶住椅子或者是牆壁做這個動作。

●迷你蹲：這個動作可以說是鍛煉髖部周邊肌肉比較好的動作了，做這個動作的時候，一定身體要站直，雙腳要與肩同寬，開始鍛煉的時候可以扶住一把椅子，大家切記做這個動作的時候背部一定要挺直，可以對著鏡子做這個動作，隨著膝關節的屈曲慢慢的下降身體，大家做這個動作不要以為就是簡單的向下蹲，這是錯誤的感受，一定要有一個什麼樣的感受呢？就是用臀部向後方找椅子。在做這個迷你蹲的時候，膝關節才不會超過腳尖，不會增加膝關節的負擔，而是單純的鍛煉髖部的肌肉。大家可以好好的看下面這個動圖，發現其中的細節，一般我們建議練習這個動作4~6次就可以了，慢慢的可以逐漸增加到12次，不需要過多。

如果實在找不好做迷你蹲標準的話，大家可以按下圖，這樣在身體的後方放一個椅子，之後對著鏡子去找感覺。

總結

以上就是對於髖部疼痛的一些簡單的康復鍛煉介紹。大家一定要明白，以上的這些康復動作不是適合所有的髖關節疼痛的病人，最好的康復鍛煉應該在專業的康復師以及康復治療師的指導下，根據每個人各自的片子以及身體狀況去制定每個人的標準計劃，才是最好的、最適合每個人的。這些動作如果大家沒有條件可以自行嘗試，如果哪一個動作做完之後覺得髖關節有明顯的舒適感，那麼就可以堅持進行鍛煉，如果哪一個動作做完誘發明顯的髖部疼痛加重或者是不舒服，那麼就不要做！而且在做鍛煉的過程當中沒必要強求先做的特別標標準，在最開始鍛煉的時候一定要把感覺找到，這才是最關鍵的。如果您還有其他的疑惑，可以在下方留言，大家共同探討。

另外謝醫生再多說一句，看完這些康復鍛煉介紹，很多朋友會覺得這些康復鍛煉太簡單了，但是是這樣簡單的康復鍛煉，也不是每個人都能堅持做好的。所謂康復鍛煉，其實就是要教給病人一種正確的鍛煉辦法，但是執行起來還是要依靠病人的意志力，如果您的髖關節問題不是特別大，那麼完全可以在這些康復鍛煉的過程當中，逐漸因為關節的靈活性增加和肌肉力量的強化得到緩解，問題主要在於能否正確的堅持，您覺得呢？

我是堅持用簡單語言解釋複雜疾病知識的謝新輝，碼字不易，如果您贊同我的觀點，請幫忙點個關注或點個贊吧，如果您或者是您的家人也有髖部疼痛的困擾，請把這篇文章轉發給需要的他們吧，謝謝了！
