老公的減脂餐，解饞又減肥，跟著這麼做晚餐，褲子越來越寬鬆

2019-12-16 1,036條評論 2,437次閱讀 1393人點贊

黑貓的廚房，與你分享每日家常菜和一日三餐的細枝末節，認真生活，好好吃飯！

米飯這樣煮，碳水含量少，不挨餓又能吃飽，有肉有菜，瘦子都在試

聽說，關注「黑貓的廚房」的人，怎麼吃都不會胖！又回到減脂減肥這個話題上，話說，前一陣，老公單位組織一年一度的體檢，體檢報告顯示膽固醇過高，體重要適當控制啦！平時總笑我說那些減脂餐沒什麼用，這回看到體檢報告，可真要老老實實的管住嘴，制定運動計劃，認真看待自己的健康問題了，畢竟，人到中年，新陳代謝各方面指標都開始放慢，如果再不忌口，退休了可不是天天去公園了，那得天天去醫院了。

說減脂減肥，節食不主張，而且主食還是要吃的，只是平日里的精米白面都要用一些粗糧來替代，例如紅薯、玉米、糙米等，這一類的主食，含粗纖維多，而且飽腹感強，也許剛開始許多人都不適應這樣的飲食，但是沒關係，可以循序漸進，只要對身體好的，能改善自己的健康的食物，我們都應該去接受。

在老公最近減脂減肥這段日子裡，我給他制定的減脂餐里都有一定的主食，而且把主食這樣做，增加一些肉和蔬菜，吃起來的口感特別好。如果你不希望冬天囤太多脂肪，還能大口吃肉，今天這食譜就非常合適喲！許多身材出眾的健身黨們都這麼吃。

【糙米藜麥雞肉飯】​

食材：糙米100克，藜麥100克，雞胸肉100克，青豆1把，胡蘿蔔半根，澱粉1茶匙，橄欖油1茶匙，鹽少許，黑胡椒少許。

具體步驟：

1.糙米質地較硬，因為沒有經過太多加工，同時也保留了粗纖維和更多營養維生素，但煮米飯前都需要把它提前浸泡3小時或者提前一晚，把藜麥淘洗一下，和糙米一起加入少許清水煮成熟飯。

2.準備一些雞胸肉，減脂期內，雞胸肉和適量的牛肉都是不錯的肉類首選，把雞胸肉切成丁，加一小勺澱粉和料酒，少許鹽和黑胡椒拌勻腌制。​

3.熱鍋冷油，倒入一小勺橄欖油，把雞肉丁和青豆胡蘿蔔放入炒鍋中，小火翻炒，把雞肉炒熟。​

4.煮好的糙米藜麥飯倒入鍋中個，與雞胸肉一起翻炒，這樣平淡無味的糙米飯也有了肉香味喲！​炒勻出鍋前，加少許鹽調味。

糙米無味道，但口感有一種韌勁，特別好吃，加上雞胸肉的肉味和青豆雜蔬的點綴，讓這一份減脂主食再也不單調啦，吃得好又吃得飽，熱量不高，飽腹感強！

【小提示】：

  1. 糙米經過浸泡，吸足了水分，所以煮飯的時候加入的清水要盡量少一些，大概是平時煮米飯的1/3水量即可。
  2. 分量作為兩頓飯的主食，一次做2份，比較容易烹制。
