肌肉拉傷後的五個辦法，但願用不上，建議收藏

天天要聞
2019-12-21

嗨，大家好，我是你們人見人愛花見花開的小簡，今天和大家聊聊肌肉拉傷後該怎麼做！

常在河邊走哪有不濕鞋，不幸肌肉拉傷後請這樣做！

健身中不小心拉傷了肌肉怎麼辦？

其實這是每個健身者都應該學會的應急技巧，因為常在河邊走，哪有不濕鞋呢？在運動中如果肌肉發生拉傷，多數情況是運動的時候分心走神，以及在使用重量過大和動作沒做到位等情況下，肌肉就有可能發生拉傷。肌肉一旦拉傷的話，會給我們的生活帶來極大的不便，因為輕則疼上一禮拜，多則兩三個月甚至更久，所以運動的時候一定要多加註意。

那麼萬一這種不幸的事情發生了，我們該怎麼做呢？

接下來，我們就來聊一聊關於運動拉傷後該如何處理等問題。

1.如果感覺到自己的某個部位肌肉可能發生拉傷，第一點要注意停止你的健身活動。

很多人會不在意，自我認為拉傷並不是特別嚴重，就接著訓練了，其實這是很錯誤的想法。因為如果置之不理，肌肉的拉傷只會變得越來越嚴重，也給後期的恢復造成更多困難。所以一旦感覺到肌肉被拉傷了，那麼不管你前期準備了多久，請停止你的訓練。

2.認真檢查是否發生肌肉拉傷。

一旦肌肉發生拉傷，通常會第一時間感覺到疼痛，一動就疼。而且肌肉拉傷之後稍微做點日常的活動就會產生疼痛感，比如腘繩肌拉傷，那麼你直腿彎腰時，大腿後側的肌肉就會特別痛。這時，肌肉可能已經拉傷了。

3.如果確認拉傷，那麼第一時間冰敷。

像一般的肌肉拉傷，基本上都是可以進行冰敷的，而且冰敷的作用是非常好。可以把冰袋放在拉傷的部位，如果找不到專業的冰袋，可以買一瓶冰水，雪糕，都是可以的。冰敷的主要功能就是為了收縮血管，減少淤血的發生，在肌肉發生拉傷的1到2天之內都是可以進行冰敷的。

4.三天後選擇適量的熱敷。

這個環節很多朋友會問了，怎麼一會兒冷敷一會熱敷呢？前1到2天之內可以選擇冰敷，然後就可以選擇用熱敷。因為這時候拉傷部位的淤血已經停止了，但是還停留在體內，熱敷的話可以促進血液循環，幫助消除體內的淤血。但熱敷的時候溫度不宜過高，過高的話很容易再發生燙傷，就非常悲催了。

5.恢復期要注意加強傷痛部位的輕微按摩。

適當的按摩可以幫助肌肉放鬆，快速好轉，同時也能夠很好地避免肌肉僵化。如果是自己所能掌控的範圍內，可以自行按摩，如果是在自己不方便的部位，也可以找專業的人士或者朋友幫自己做下按摩。

簡單回顧一下，在運動的過程中如果不幸拉傷了，首先你要做的是停止運動，其次在休息的地方可以用冰來敷傷患的部位，3天後可以進行適量的熱敷，再輔以按摩對拉傷部位會更好。

希望上面說的這些廢話你們都用不到，好好健身，享受健身的快樂就好。

好啦，今天就和大家分享到這裡了，建議收藏，但願用不上。喜歡小簡的話，記得點擊下關注哦，每天更新大量運動乾貨，總有一款能擊中你的心。
