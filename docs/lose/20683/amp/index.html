一、如何發出有威力的球

1、正手奔球

特點：球速急、落點長、衝力大，發至對方右大角或中左位置，對對方威脅較大。

要點：拋球不宜太高；提高擊球瞬間的揮拍速度；第一落點要靠近本方檯面的端線；點與網同高或稍低於網。

2、反手發急球與發急下旋球

特點：球速快、弧線低，前沖大，迫使對方後退接球，有利於搶攻，常與發急下旋球配合使用。

要點：擊球點應在身體的左前側與網同高或比網稍低；注意手腕抖動發力；第一落點在本方台區的端線附近。

3、發短球

特點：擊球動作小，出手快，球落到對方檯面後的第二跳下不出台，使對方不易發力搶拉、沖或搶攻。

要點：拋球不宜太高；擊球時，手腕的力量大於前臂的力量；發球的第一落點在球台，不要離網太近；發球動作盡量與髮長球相似，使對方不易判斷。

4、正手發轉與不轉球

特點：球速較慢，前衝力小，主要用相似發球動作，製造旋轉變化去迷惑對方，造成對方接發球失誤或為自己搶攻創造機會。

要點：拋球不宜太高；發轉球時，拍面稍後抑，切球中下部；越是加轉球，越應注意手臂的前送動作；發不轉球時，擊球瞬間減小拍面後仰角度，增加前推的力量。

二、如何精準判斷來球的旋轉

如何精準判斷對方發球的旋轉，是球員必須掌握的重要技術。通常，可從四個方面作出判斷。

1、板形

一般情況下，發上旋球時，板形都比較豎，下旋球比較平、斜，這種特點與發球時要接觸球的部位直接發生關係，因為發旋轉球和不旋轉球時，接觸點比較靠近球的後中部;而發上旋和下旋時，向中下部和底部摩擦才可能比較轉。

2、動作

發下旋和不轉球時，球與球拍接觸的一瞬間，手腕擺動的幅度一般不是很大，並時常與假動作相配合;在發上旋和下旋時，手腕擺動相對大一點，這樣容易「吃」住球，動作也比較固定，擊球後常有一個停頓，即使加上假動作，也不會像發側上旋和不轉球那樣連貫。

3、弧線

上旋球和不轉球的運行一般比較快，常有往前「竄拱」的感覺，發短球時容易出台，弧線低平;下旋球運行比較平穩，弧線略高，短球不易出台。

4、出手

發上旋球和不轉球一般出手比較快，並且突然，動作模糊;下旋球的出手相對要慢一些，因為要給球以足夠的摩擦時間，才能使球產生強烈的下旋效果。

三、如何在第一時間判斷來球的速度和落點

通常區別兩種情況作出判斷。

一般情況下，發球者如果想把球發得很快、很長，第一落點多在自己本方檯面的端線附近。如果力量差不多，球的運動速度，側上旋和不轉球明顯要快於側下旋和下旋。

對短球的判斷，由於發球者想要把球發短，手上就不能發很大的力量，要收住一點。所以，短球較難發揮速度的優勢，比較多是考慮的落點和旋轉。在發短球時，第一落點一般距球網較近，根據這一特點判斷來球的落點長短。
