娛樂圈型男曾經胖到250斤，現在八塊腹肌

2019-12-10 2,570條評論 4,344次閱讀 944人點贊

微信公眾號：青竹新健康

最近很多明星都在健身，努力的想成為肌肉型男。比如曾經快樂男聲魏晨就是其中一個，在我們印象中娛樂圈的好身材只有彭于晏。

其實近日黑人陳建州的肌肉照又被翻火，其實很多人都知道他是運動員出身所以對他的肌肉照沒那麼驚訝，更覺得非常正常。因為是運動員嘛，不過很多人不知道他曾經的體重到達過250斤。今天就來重點說說這位勵志型男。

在今年的7月9日，黑人陳建州把這個多年的秘密公開了，曝光自己曾經是一個小胖墩，從照片我們就可以看出體重非常驚人，黑人更是放話說最重的時候達到250斤。

而現在他的近照身材非常完美，渾身都是肌肉，特別是非常性感的八塊腹肌，很難鄉下黑人陳建州曾經是個胖墩。

大家也不要羨慕黑人減肥成功，其實他也用了5年時間才做到。最好的減肥方法就是堅持，黑人自曝說是一直泡在健身房減肥的，也是最好的減肥辦法。

他提到減肥成功後的前後區別時，他是在肥胖的期間，因為體重受到的嘲笑很多，可能會陪伴一生，但後來減肥成功後，整個人的感覺都和之前不同，所以他一直在做鼓勵肥胖朋友的組織，希望用自己的故事幫助更多因為體重帶來煩惱的朋友。
