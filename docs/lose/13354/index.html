好穿！領導，這雙評測鞋我扣下了

2019-12-12 156條評論 3,076次閱讀 803人點贊

從2017年底PUMA FUTURE正式亮相以來，短短兩年時間PUMA為這款球鞋推出了5代產品，一代和二代類似，三代和四代類似，兩兩之間算是小改換代。但是到最近的第五代產品，FUTURE有了質的變化，從穿著感到實戰感，給老編帶來了顛覆性的體驗！我們一點一點來說。

FUTURE一代推出時，最主要的宣傳點放在了鞋面的NETFIT SYSTEM上，通過鞋帶的不同穿法，打造最個性化的包裹鎖定效果以及鞋面圖案個性。現在回看FUTURE一代，其實其他方面都很普通，超纖材質鞋面、evoKNIT織物鞋領、定位速度與靈巧、由攻擊球員代言，上腳實戰感受是一雙非常有個性的「塑料」鞋，有特點但並非不可取代。等到2018年世界盃之前，PUMA為FUTURE推出了小改第二代FUTURE 2.1足球鞋，注意，名字有了小變化。鞋面大同小異，只是覆膜面積和NETFIT面積的區別。改變帶來的實戰變化是，腳背比較高的小夥伴可能不壓腳面了，也更容易穿進去了，舒適度有提升。

到今年年初，PUMA FUTURE 19問世，整雙球鞋變化不小，算是一次常規的年度換代。鞋面用到了全針織材質，同時使用3D Havoc製作輔助紋理；標誌性的NETFIT系列變換形式，變得不如前兩代有可調節性和可玩性。從這一代開始，PUMA FUTURE上腳後就有一種偏硬的感覺，因為鞋面織物的特殊處理方式，需要比較久的磨合時間，很多EZer就是都被這種初上腳的硬質感擊敗了。好在，好在從FUTURE 19開始，萬能MG大底出現在高端19.1級別上，吸引了不少人入手。就第三代FUTURE的實戰感來看，屬於老編不太會選擇的鞋款，因為不愛那種硬邦邦的鞋面觸感。再到今年夏天，PUMA又為FUTURE升級鞋面，同樣的「evoKNIT+3D Havoc+NETFIT」配方，換了一種紋路形式，變化不大，腳感相似，略讓人失望。有一說一，其實19年FUTURE的兩代戰靴，是不如18年兩款超纖鞋面的，那時候起碼鞋帶可以想怎麼玩兒就怎麼玩。

但是！最近的一雙PUMA FUTURE脫胎換骨了，相信老編，這是一雙值得擁有實戰的FUTURE！特別是MG版本！

先來說說試穿感受。這次PUMA的升級換代是跨年的大改進，大家可以通過整個鞋面的材質、造型可以看出。雖然鞋楦沒變，大底沒變，但是鞋不是以前的FUTURE了。現在FUTURE 5.1 NETFIT擁有非常柔韌的evoKNIT Pro織物鞋面，軟、有韌性、有厚度，讓整個鞋面在雙腳插入之後非常舒適、貼合，完全沒有不適感，完全不像在試穿一雙全新的球鞋。另一方面這次新鞋的鞋面覆膜也有了大調整，面基比較小，很薄，觸感比較澀，而且鞋頭位置還增加了特殊處理，試穿時雖然感覺不到控球輔助，但是能感受到整個鞋面的包裹性和舒適性，比之前的兩款織物FUTURE真的是好了太多太多！

老編平時尺碼JP270，試穿的尺碼為EU42、JP270，長度、寬度、腳背高度、腳踝壓力都非常合適，完美至極。而且就鞋寬度來說，因為鞋面變軟了，對寬腳的兼容性也更好了。類似老編這樣的常規腳型，是必須要利用鞋帶來鎖定的。說到鞋帶，FUTURE 5.1的NETFIT系統雖然比不上第一代那麼花哨，但可調節性比2019年的兩款要好很多，鎖定效果更好。

新FUTURE依舊沒有換底，使用的依舊是Pebax材質的RAPIDAGILITY FG/AG大底。怎麼說呢？這款大底沒什麼硬傷，但就是鞋釘比較挑場地，如果人工草球場條件一般，大家還是應該選擇MG大底。

試穿體驗可以非常明確的告訴大家，新FUTURE的改進很到位，舒適度、包裹性、穩定性表現突出，就算是第一次穿也不會有一種穿新鞋的感覺，現在主流針織材質足球鞋內，沒有一款能夠做到這點，完全改變了老編對FUTURE的看法。新FUTURE，看來是要成為2019年度最佳戰靴的一匹黑馬了！

關鍵信息：

1. 尺碼準確，腳型兼容性好；

2. 不同於前四款FUTURE的穿著體驗，極為舒適；

3. FG大底未更換，強烈推薦高端MG版本！

在試穿完FUTURE 5.1 NETFIT之後，會非常讓人期待穿著這款球鞋落場實戰，畢竟這樣一雙舒適度很好的球鞋，要是實戰表現也完美，那就太不錯了。所以頂著北京冬天夜場的冷空氣，我們完成了初次落場評測。

磨合

試穿之後大家心裡就會很清楚，FUTURE 5.1應該是一雙磨合非常快的球鞋，等到落場實戰之後才發現，磨合個大頭鬼，這雙鞋老編穿起來完全不需要磨合，即插即用！從系好鞋帶的那一刻到最後踢完全場，中途一次都沒有調整過球鞋，也完全沒有在意這是一雙新鞋。在這一點上，新FUTURE又贏了。

舒適感

整個測試比賽的過程中，FUTURE 5.1是非常舒適、包腳、穩定的，這有什麼直觀的感受嗎？其實會有一些明顯的現象。穿新球鞋如果出現擠腳、壓腳面、勒得太緊而腳酸，這都可以通過慢慢的磨合達到最佳效果，多穿幾次球鞋自然撐開了。但是上腳之後就覺得鞋面、鞋舌、後跟或其他某個部位不舒服、彆扭，跑動時不跟腳，觸球時不自然，這就是舒適度、貼合度不太好了。比如很多EZer穿習慣了高端鞋款，突然換一雙中端，肯定會有不舒服的感覺。

就新FUTURE而言，相信只要不是腳型特殊，人人穿上都會很舒服。因為這款新FUTURE的織物鞋面真的太贊了，帶著一點點延展性的軟，應該有支撐的地方有覆膜加持，完全貼合在腳面上，還有可調節的鞋帶輔助，完全像一雙襪子一樣，這次不吹牛，真的像一雙襪子。

觸球感

老編把新FUTURE排進現在市面上織物球鞋最佳觸感前三。就觸感而且，其實是比較個人化的體驗，不僅僅要看鞋面的材質，還要看整體的包裹貼合度，要看鞋倉空間飽滿度，是一個全方位配合的結果。

新FUTURE在鞋面換代設計上，進步明顯。觸球感真實、細膩，有著織物材質一貫的觸感體驗，給雙腳的感受是直接的。同時鞋面的新款覆膜設計，帶來了不錯的輔助功能，鞋身到鞋頭位置的覆膜都比較很澀，觸、控、停球時有很好的摩擦感。同時，覆膜還提升了織物材質的韌性，讓其保證舒適度的同時還有不錯的支撐性。

在鞋面、鞋帶系統的雙重作用下，FUTURE 5.1還擁有非常不錯的包裹性，讓球鞋的穿著一體感很強。直接反應為踢球時動作更為自如、輕盈，腳下觸控準確，用力也收放自如，真的適合中前場那種能突善帶、能射會傳的攻擊型球員。

問題

這一次的測試老編沒有發現任何問題，新FUTURE從頭到尾都讓人滿意到不行，即便是FG/AG大底的鞋釘偏長，在冬天的人造草球場上也能忍。接下來的時間還會繼續測試，因為它好穿，這個冬天可能就一直穿下去了，發現問題會及時跟大家交流。

關鍵信息：

1. 實戰性能出色，當下織物戰靴前三！

2. 不需磨合；

3. 鞋面舒適、觸球感好；

4. 包裹緊密、支撐出色，穿著一體感強；

5. MG版本買就完了！

總結

新球鞋穿的太多，其實很少有讓人驚喜的鞋款。在2019年里，目前PUMA FUTURE 5.1是最令老編意外和滿意的鞋款，超過夏天的Rebula 3，目前排名心中最佳。當然，對球鞋的喜好人各有異，可能EZer試穿、實戰完之後會覺得新FUTURE不錯，但卻不會喜歡它，這也正常。最後，誠意推薦全新FUTURE第五代，幾何倍數完勝之前的四款，同時強烈推薦MG版本哦！
