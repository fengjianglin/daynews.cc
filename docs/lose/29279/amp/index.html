我們看看這篇文章，“拳在矛盾”，也許你會有所收穫

天天要聞
2019-12-21

時光如河，歲月似魚。雖百般險阻，仍逆流而上。就算一日不成，總會柳暗花明，自然而然，功夫最純。冬天不掛雪，就不是冬天。夏季無風雨，就不是夏天。打拳無功勁，不稱其為拳。養生與拳勁，並不矛盾，好勁養人。練著舒服，運著通透，有強度，有變化，身心皆得康泰，這是好的。夜燈吸蛾，蛾喜撲光。飛燕抄水，一沾即起，漣漪層層。萬物天性使然。雪花是飄落，寒露是滴落。形式不同，動態必不同。有了沉勁，再快，也穩當。善活勁，再大強度，也是不滯。勁一定出於自然，自然培育，養拳如育樹。樹大參天，高不可攀。根深蒂固，茂密龐大。中正擴張，是練拳必修。關節活，竅內空間，含藏開合之力，拳會由慢漸快。有了帶速的拳，是好的。腿軟人不立，拳無支撐力，等於放棄了攻。不蹬不撐。攻擊要蹬，亦要旋撐，身子勁奮出來。奮的意思是，內掙力。內壓整，內腔動，根節出勁，暴起開合。拳若實用，離不開這條路。七八分勁，就合適了，不要上到十成。留餘地，留餘力，方能轉化，或是疊加。綜合全面的，是好拳。

奮起拋鼎，一怒摔碑。拍，為掌力，本是合腰腹之力。好比鼓掌，力道沉實的人，拍出來不是脆聲，而是悶的，甚至嗡嗡的。手臂發漲，這是內壓貫通，好事兒。不使勁，周身也充盈，內勁上身。接骨斗榫，意思是搭建結構。懂得用關節旋轉，校正功架角度，這是聰明人。省勁的東西，往往角度聰明。骨正，架子整，再圖變化，內勁自會通達。大氣盤旋，勢如高山。武人要豪放，捨我其誰，骨子裡自信，意志力強大。士不可不弘毅，內在剛強，心懷堅毅，方好男兒。靜功，似乎戰戰兢兢，不是害怕，並非打哆嗦，是一種局部細微鬆緊。靜心調配神經，時而是，三頭肌抽動。又或是，胸肌鬆緊。還有那，臀肌剛柔，如此等等。靜樁，雖大形未動，不意味著局部無鬆緊。零距離啟動，又稱觸動猶靜。為何說，不動之動厲害？無位移，無徵兆，不知從何而來，瞬間開始，陡然結束，這是靜中求動出來的功夫。深刻體會靜為何物，能培育一種特殊的動。有些諺語，說得是結果，而非練功過程。好比說，沉肩是結果，並非練功過程，硬拉生按肩膀，狠壓鎖骨，人會練出問題。應該松活肩軸，進而沉肘，既舒服，又出功力。肩松活，方能開發肩胛骨。胸大肌，背闊肌，斜方肌，彈韌有力。功深之後，肩部空間已成，沉感自現。拿結果當方法，就錯了。

拳在矛盾，與道俱成。抱得丹田力，鬆緊求矛盾。提神領首，虛襠裹腿，脊柱就合適了。不能光松，要暗含鬆緊，松是對緊而言的。這句話很關鍵，抻裡頭，旋內在，固核心，周身擴。萬物皆相對而生。有個核心力點，在小腹丹田。丹田愈密，周身功勁愈擴，功力由此產生。能量源自一種模式，大無外，小無內。同一空間，最深的塌縮，引發無窮之擴張，這是矛盾的終極。打拳如是，最根本矛盾，即是核心功法。塌縮丹田，結構擴張。之後，運用螺旋，生髮變化。用矛盾練功，用螺旋變術，這就是太極拳理。並非先有螺旋，而是先有矛盾對立。能統一各種矛盾，為我所用的，就是旋轉。打個比方，一人端大水缸，形看似抱合，其實這不是整勁。整勁源自矛盾。結構擴撐與塌縮丹田，引發內壓膨脹，輪胎勁就來了。抱死物憋力，拳學不取。脊柱如是，上有領勁，下有虛襠。先下襠勁，順脊柱，開後腰，再微屈膝即可。目的不在蹲，而在脊柱矛盾貫通，很像掛衣服。手肘關係，也含矛盾。手合中線時，掌必遠推。反過來，掌合中線時，肘必外撐。換髖也是，大腿根里裹外撐，腳底下旋兒就有了。剛時要思柔，慢運必圖疾，也是利用矛盾練功。太極浮屠。揉合相參，太極混成。我裹著你，你含藏我。萬法矛盾，拳亦陰陽。天道如此，我本自然。
