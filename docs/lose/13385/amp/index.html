中老年朋友提高舞蹈水平，這幾個注意事項很關鍵

天天要聞
2019-12-11

舞蹈是一項特別棒的運動，它不僅可以給人帶來精神方面的享受，而且經常跳舞的人體形漂亮，還可以減少身體的疲憊，緩解人的精神壓力，提高身體的各項機能，還能起到減肥的目的。可以說跳舞的運動價值完全不會輸給長跑，而且這是一項合適各個年齡段的運動。

老年人跳舞更為合適，不僅可以打發他們的時間，給枯燥的生活增加情趣，而且還可以預防老年痴呆，達到強健身體、保持大腦清醒的效果，真的是好處多多。不過，老年人因為身體的特殊性，在跳舞的時候也要注意一些問題。

​中老年人怎樣學跳舞

1、首先老師要這樣教學

做為這樣一個群體的教師一定要專業，要針對中老年的生理，心理特點來安排我們的教學計劃，針對班上學員情況由淺入深，循序漸進的完成每個階段的教學計劃和任務！只有通過正確的方法和教學讓舞蹈帶給他們的是愉悅、而不是身體損傷並且逐步提高，,只有按照正確的方法去學習才能有質的飛躍！

​2、簡單基本功和形體訓練很重要

中老年普遍不願意學形體、練習簡單的基本功，錯誤的認為我這把年紀了還學那做什麼？殊不知連基本的直立都沒有改善、站都沒站好、又何談手舞足蹈呢？所以建議新的學員先學形體，有些簡單的基本功的練習，這樣再來配合舞蹈的學習就會好很多。

​3、課後也要練習

當然在教學中，只要到您的極限就可以不要勉強自己，但是一定要去嘗試，不能覺得我做不到就不做！老師要將動作分解的非常細，要反覆教授和練習，強迫學員記憶、只有先記住、才能再跳好，找感覺!當然學員下去還要自己消化，這樣才能逐步提高！現在大多數學員就僅僅靠課堂上的時間一星期一次、下個星期來時又還給老師了，所以只練不習也是不能提高的。

​4、互相學習很重要

有很多學員只是學會了就不跳了，甚至還覺得自己很不錯、心態很重要，所謂，能者為師，有些「小老師」沒有放下自己，這樣是學不好、也學不到的，大家互相學習最重要，學習每個人的長處，這樣才會提高、才會越來越好！

5、要聽老師的話

當大家跳會了以後要多聽老師講解，盡量去做，讓自己更好

6、學習方法很重要

不要一味的跟跳，要強迫自己記動作找方法，找適合自己的方法。動作要反覆學幾遍，要學會自己消化，多琢磨琢磨。

​中老年怎樣提升舞蹈水平

1.努力提高節奏感

舞蹈是音樂的靈魂和身體語言的優美結合。在準確的節奏指引下，舞者在舞池中表現力是最好的，不能夠準確地按照節奏進行舞步，讓人沒有欣賞美的感覺，這樣就傳遞不了舞蹈的動作美和音樂美。

​2.舞姿雅緻

要做到上體保持直立，但不緊張，而是自然直立，像是一棵亭亭玉立的桂花樹。如果不能做到上體保持直立，在遠處的燈光下一看，好似塌陷的雪雕，根本談不上優美。

​3.眼神

所有舞蹈魅力的傳達有兩種形式，一種是舞者的眼神，另一種是舞者的肢體語言。充分領略舞伴的眼神和扶在你腰間的手，眼睛會示意你如何跟隨，而腰間的手則會指引你在舞池中的位置，兩者相輔相成，要點是時刻與對方的眼神保持一致，在一條線上。這也是需要與舞伴存在一定的默契。

​4.自我表現力

當你對於舞蹈比較熟練時，你就可以自我發揮，舞蹈的自我表現力是你個人對舞蹈以及伴奏音樂的理解，是你對舞蹈以外的空間的理解。 與默契成熟的舞伴在一起跳舞時，彼此可以閉上眼睛而不擔心會踩到對方的腳，與此同時，彼此的手中會覺得對方輕飄飄的，像輕紗一樣柔軟，似乎不存在，但確確實實就在你眼前，而且和你同舞一曲，彼此宛若驚鴻，翩似游龍。

5.自信

舞蹈的靈魂是自信，沒有自信，所有的舞蹈都是失敗的。

​另外，對於喜愛跳交誼舞的中老年人，下面四點注意不可不知。

（1）跳舞以早晨八九點或傍晚五六點為宜，此時空氣質量較好，溫度適中。可選擇飯後半小時去跳舞，但酒後絕對不能跳舞，因為酒後可能會誘發心絞痛及腦血管意外。

（2）跳舞時要選擇寬敞，空氣新鮮的場地，人不宜過多。每次跳舞的時間應控制在1~2小時為佳，時間過長會導致身體過於疲勞，健身效果反而下降。

（3）平整光滑的地面有助於表現舞蹈的平衡，穩定和流動感，大理石、細水泥和瓷磚地面都是合適的選擇，跳舞時應注意選擇一雙底軟、輕便，能夠保護腳踝的鞋子。

（4）跳舞者應注意由緩入急，由慢及快，循序漸進。跳前充分做好踝、膝、髖、肩等關節的準備活動，用幾分鐘做做伸展運動，從而避免運動損傷。如不小心扭傷，應先冷敷，然後採取相應的治療措施。
