2年減掉51kg，瘦身後撞臉閔孝琳，這減肥歷程才叫脫胎換骨啊

2019-12-23 917條評論 4,289次閱讀 1829人點贊

最近在韓國，有位爆紅的美女Youtuber Jinny，在開始營運Youtube頻道不到一年後就累積了足足10萬名超過的粉絲數！成功的秘訣是什麼呢？難道僅僅只是因為媲美偶像等級的超高顏值？絕不！

Jinny的減肥記錄，最開始是99kg的胖妹

記錄日常的飲食

日常運動打卡

99kg →76kg→68kg→65kg→60kg

60kg→51kg並保持

瘦了整整半個自己啊

是因為她本人真的很勵志啊，而且推薦的減肥方法也是非常容易學習的！

說出來可能會嚇死你，但這位正妹在三年前還是位「體重高達100公斤的大尺寸女孩」！

如今擁有超凹腹肌的魔鬼身材，Jinny前後總共花了足足兩年！

成功「用體重整型」，秘訣究竟是什麼？Jinny也在個人開設的Youtube頻道分享秘訣給大家！就讓我們一起來看看吧～

減肥前應先想清楚『為什麼』

Jinny：「瘦身是學會如何愛自己，而不只是目標設定！」

對想要減肥的朋友們，Jinny提出了建議。她認為「在減肥之前應該先想好自己『為什麼要減肥』的理由」，如此一來才能更堅定意志，明確的建出目標。

Jinny說，自己會胖到一百公斤，全是因為在高中時期每周的炸雞啤酒狂歡派對。當體重暴增到100公斤時，Jinny甚至被診斷出椎間盤突出，而這也終於讓她下定決心要減肥。

成功減掉半個自己，Jinny說「瘦身其實是學會如何愛自己，而不只是設定每個月瘦5公斤的目標」。

而能擁有魔鬼般的完美體態，Jinny說這一切全靠「習慣養成」，她也為想減肥的大家在網路上分享了自己的方法，就讓我們一起來學學！

一、堅定意志、從生活習慣開始改變！

攝取過多碳水化合物和高鈉食品都是減肥路上最大的敵人！在下定決心要瘦身之後，Jinny便開始改變自己的飲食習慣！

但這並不表示要餓肚子，比起激烈的絕食瘦身法，Jinny選擇健康的少量多餐，並搭配每天飲用3-5公升的水。

因為主要採取的是「運動瘦身法」，Jinny在飲食上也會選擇更高蛋白的食物及蔬菜，若是有時候不小心吃太多，就會增強當日運動量，維持平衡。

二、制訂計劃表，按表操課的運動法！

關於運動，Jinny選擇每天制訂計劃表，紀錄當日的運動量。她推薦減肥初期需要靠「激烈運動」瘦身的人，可以使用「上下半身交替的重量訓練」搭配「有氧運動」制訂每周計劃。

一周計劃表

▲Jinny公開自己的個人計劃表給大家參考。

她在禮拜一會利用重訓鍛煉上肢，在禮拜三集中下半身的重訓，而禮拜五則回歸上半身的肌肉訓練。二、四、六的部分，Jinny排定做「有氧運動」來燃燒脂肪，而禮拜天則放自己一天假，讓身體休息。

▲「333運動法」

Jinny說自己會使用「333運動法」，組合各種運動執行。

舉例來說，以「兩種無氧運動搭配一種有氧運動」為一組，做出三種組合，每種組合執行三次。一般來說，「333運動法」的運動種類，她會在「伸展」、「10分鐘有氧運動」和「循環運動（Circuit Training）*」里挑選執行。

*循環運動（Circuit Training）是指重量訓練加上呼吸循環的耐力訓練。屬於輕度重訓的一種，為「中間不休息的重量訓練」。*

▲在每項組合都各執行過三輪後，再搭配腹肌運動和一小時的有氧運動做收尾。

耗費整整兩年，不追求餓瘦的骨架身材，堅持一周6天的運動和飲食控制，真的不是常人可以堅持的，但Jinny收穫了健康的身材和運動的快樂感，還收穫了一大票粉絲，這就是她的動力所在吧！
