上班族如何擺脫“假裝健身”的困境？3個辦法讓你從假裝變真運動

天天要聞
2019-12-23

最近關於「有1億中國人假裝健身」的話題惹人關注，實際上許多上班族確實也挺無奈。現代人瑣事多、工作壓力大、上下班路途遠，所以許多人感覺沒有時間鍛煉，儘管每個人都知道運動對健康的意義。那麼，如何在這樣的生活方式下，讓自己也能夠從運動健身中獲得益處呢？

有一點是可以肯定的，如果每周運動頻率達不到三次，那麼運動效果就無法累積，也就是無法實質性地改善身體健康狀況。多數人所認為的「只要動了就是對身體有好處的」，其基本前提就是「每周運動至少3次」。

具體該如何做呢？下面三個辦法可以試試。

辦法1：低頻率、低強度運動，每周至少保證1至2次

剛才不是說每周至少3次才管用嗎？是的，但許多平時確實沒時間，那麼可以在周五晚上、周六或周日這三天里安排1至2次運動。雖然這樣的運動頻率，尚不能夠保證運動效果的累積，但是卻有助於培養比較規律的、長期的運動習慣。所以，想開始運動、一時又太忙沒有時間的人，可以先嘗試這個方法，慢慢養成運動的習慣。要求每次運動時長至少達到30分鐘。

對於普通人來說，一個人偶爾30分鐘的運動是挺難長期堅持的。御行君的建議是，儘可能參加集體性的運動項目，比如每周六參加一次乒乓球或羽毛球運動。即使只是在小區里進行快走鍛煉，最好也能和家人或鄰居一起鍛煉，運動的枯燥感和艱苦程度，遠超大多數人的想像。幾個人一起鍛煉，對於新手來說可以增加新鮮感，形成良好的運動小氛圍。

這個方法的主要目的是培養運動習慣，為今後時機適當時提升訓練頻率打基礎。

方法2：短周期健身，追求儘快出效果

有些人其實確實並不喜歡運動，但是健康上已經出了狀況，比如說體重過大、經常腰酸背疼、頭疼腦熱不斷等等，但又沒時間，或無法下定決心要長期堅持鍛煉，怎麼辦？

御行君的建議是：花2~3個月的時間，集中進行一段時間的體育鍛煉。在這個周期內，應堅持每周至少運動3次。每次運動時長達到40至60分鐘。這個方法，不僅可以累積運動效果，而且只要這兩三個月你能堅持下來，就一定能看到顯著的效果。

經驗：如果你覺得2至3個月的時間太長，那就別計時，改為計次。以2個月集中鍛煉為例，假設每周運動4次，則一共需要鍛煉32次。每完成一次，就在家裡的枱曆或日曆上勾銷一次，你會很有成就感。

減肥新手在第1個月就可以看到比較良好的減肥效果。身體的疼痛緊繃、協調感差、心肺能力差等情況，也能夠在1~2個月里得到明顯的改善，多數人的睡眠質量也能夠有改善。如果能夠堅持2~3個月規律鍛煉，各種積極的身體反饋會讓這部分人，直接體會到運動的好處，有些人則將在此基礎上慢慢培養出運動的習慣。

這種方法顯然要比第1種方法更好，因為健身效果出現得更快。和大多數人所認為的「偶爾運動一下」相比，還不如照這個辦法集中兩三個月，打一場「密集的健康保衛戰」，讓自己的身體發生一個比較明顯的變化。哪怕你後續不再堅持鍛煉，你也嘗到了運動的甜頭，會讓人更有動力再次投入健身。

方法3：從短周期過渡到長周期

就御行君的個人觀點，我會認為第1種方法屬於娛樂而非健身，或者說是「假裝健身」。第2種方法，則絕對屬於在健身了，只不過由於只有兩三個月，時間較短，運動習慣還沒有養成，後續很容易放棄。怎麼才能夠從短周期過渡到長周期，成為一種長期的生活習慣呢？

沒有任何捷徑或「一招鮮」的好方法，只有靠鍛煉者自己長期堅持。如果一定要找出一個「好辦法」，那就是：鍛煉者要想辦法，不斷從健身中獲得積極的身體反饋，這對於保持健身動力非常重要，也非常有效。

經驗上，如果能夠規律堅持運動健身超過一年，那麼基本可以算是養成了健身的習慣。事實上，即便是堅持了很來年的健身者，也會有想放棄的時候。堅持運動的時間越久，健身的習慣也就越牢固。

建議：零碎時間健身不可行

比如現在許多人都希望在每日步數榜上名列前茅，所以就在工作時多走幾步，或者回家後去散步。這些零散時間積累而來的步數，並不能形成健身上的意義，對於健康的改善並沒有什麼實質性的幫助。因為，健身效果的達成，至少需要運動強度、每次的運動時長這些要素共同作用才行。零碎時間的「所謂運動」，都達不到要求。

健身效果的形成，並不是零碎的運動時間的相加，它是一項非常系統性的工程。因此「不想假裝健身」的朋友，應該投入專門的時間去健身，而不要在」零碎運動時間」上浪費體力和精力。

什麼是「零碎時間」？就健身效果的達成而言，一次有質量的運動，時長不應低於20分鐘，最好能超過30分鐘。低於20分鐘的，都可以視為是零碎時間。

雖然方法1「低頻率、低強度運動，每周至少保證1至2次」，我認為屬於「假裝健身」。不過，想要最終進階到方法3「從短周期過渡到長周期」，對於平時忙碌、又沒有運動基礎的新手來說，從方法1開始仍不失為一個比較實際的辦法。是不是「假裝健身」不重要，重要的是：立即行動！
