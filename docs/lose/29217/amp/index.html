16歲高中生健身2年練出44厘米的臂圍，學校里沒人敢欺負他

天天要聞
2019-12-22

經常在健身房健身的小夥伴們都知道，在健身房裡臂圍超過45厘米就已經屬於「王者」級別，甚至有些健身房貼出標語：「臂圍不到45，別亂扔啞鈴」，可見臂圍40是一個分水嶺，超過45的沒有幾個人！而接下來我要說的這個小伙是一個00後，今年16歲的他健身2年練出44厘米的臂圍，學校里沒人敢欺負他！

他叫：李蓋次（上圖左），出生於2003年的他今年16歲，目前是一名高二在校生。他身高158厘米，體重86公斤，臂圍44厘米，曾在今年8月份獲得環球國際超級聯賽新秀獎。他叫李蓋次，個子不高，而他的偶像也姓李，個子也不高，他的偶像叫李·普瑞斯特（上圖右，澳大利亞健美巨星）！

據小李透露，他在健身之前身材瘦小（體重只有48公斤），雖然他喜歡上體育課，但是他個子矮身上也沒有什麼肌肉同學都喜歡和他搭夥，為了改變這種狀態不受到同學排斥才決定健身。於是2017年11月他說服父母給他辦了人生中的第一張健身卡，從此開啟了健身之路。

其實，要辦健身卡健身的時候父母並不支持他，他們認為健身會長不高、練壞身體，但是小李健身心切，苦苦哀求數月才贏得父母的同意。在健身剛開始他就和父母又在飲食上產生了分歧，父母認為他不應該吃那麼多雞蛋和肉。但是健身需要這些能量物質，無奈之下小李只能自己在外面買雞蛋吃。

健身數月後小李的身材慢慢的發生了改變，而這種改變也贏得了父母的支持，這是小李健身3個月後的效果：

健身期間也走了不少彎路，也發生了不少趣事。其中最有意思的一個梗就是「開水沖蛋白粉」，小李說剛開始健身的時候知識匱乏不知道蛋白粉要冷水或溫水沖，自己第一次沖蛋白粉是用開水沖的，結果蛋白粉都結塊了，當時他並不知道是怎麼回事，反正就把結塊的蛋白粉那樣喝了下去。

健身2年，小李已經練出44厘米的臂圍成為學校中的小明星，如此強悍的肌肉在學校沒人敢欺負他！臂圍44厘米，顯然已經接近「王者」級別。這時候可能有人說臂圍超過45就是「王者」，臂圍達到50就是「大神」級別，那而鹿晨輝應該屬於什麼級別，鹿晨輝賽季臂圍58厘米，當然屬於「超神」級別，因為世界上在賽季臂圍能達到58厘米的人沒有幾個，要知道眾所周知的健美之王施瓦辛格臂圍也不過是56厘米。

說起健身的過程，小李說健身不僅給他帶來的是身材上的改變，他說自從健身後他就改掉了泡吧打遊戲和吃垃圾食品的習慣，正是這種自律才讓他練出如此身材！

除此之外他還說自己的成功離不開導師木蘭姐姐（上圖右）的指引，他說：「在健身的路上有一位優秀的老師，會讓你往正確的方向努力，少走很多彎路，我很幸運能遇到花木蘭姐姐 」

就是這麼一位00後少年，健身兩年練出了如此身材，他擁有同齡人難以企及的臂圍，同時他的自律、感恩之心也值得我們為他點贊！
