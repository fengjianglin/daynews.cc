細節提升游泳速度：輕鬆移臂，製造“手蹼”，增大打腿面積

在頂級高手的對決中，細節決定成敗。

在業餘愛好者中，不斷完善細節動作同樣可以讓你的游泳速度有質的飛躍。

今天我們來聊聊三個容易被忽略的游泳細節，把它們加入到你的游泳中，一定會有很大提升！

放鬆的移臂，讓你長距離游泳更輕鬆

不管是蝶泳、仰泳還是自由泳，在空中移臂的時候，都不要「拿著勁兒」。

尤其在自由泳中，如果你追求長距離遊行，那麼一定要在移臂的時候，讓手臂「放個假」。

很多人在空中移臂的時候，手臂依然在牟足了勁兒發力，控制手臂從後到前的移動。

不放鬆的移臂很耗費體力

大部分人沒有意識到這是個問題，覺得這樣沒毛病，那你可能短距離游泳比賽看多了。

在短距離比賽中追求快頻率，所以為了更快的完成移臂動作，一方面藉助慣性，另一方面會讓手臂發力完成快速移臂。

如果你要是追求長距離，那麼這個就行不通了。

上面移臂也用力，下面劃手也用力，手臂一直處在發力的狀態中，兩三百米游下來，手臂肌肉就僵硬酸痛了。

長距離游泳，一定要學會在移臂中放鬆手臂，尤其是小臂和手，越放鬆越省力。

因為在移臂過程中，並不能產生任何推進力，也就是說對你的游進沒有任何幫助，所以，在沒有幫助的動作上省力，留下更多的力量放在水下划水這個主要產生推進力的動作上。

這樣，你就省下了一半的力量，至少能多游兩百米！

如何才能在移臂的時候找到放鬆的感覺呢？

來，試試下面這個針對性練習。

手掌微張，製造「手蹼」效果

很多愛好者都熱衷於看游泳比賽，不知道大家有沒有觀察過運動員在劃手時手掌的狀態，他們並不是手掌併攏的在划水。

手掌微微張開能夠增大對水的阻力面積，讓你的划水實效性更高。

很多年前，運動員就是這樣做的，但是近些年來，有專業機構開始研究，究竟手指分開多少才能最大限度的增加划水效率？

專家團隊用計算機模擬了手臂進行試驗，他們把實驗中的手指分開程度分為五級：第一級手指併攏不分開，之後每一級逐漸增加5度的分開角度。

最後研究得出結論：手指之間大約分開10度時，划水效率最高！

手指微張10度最佳

可能很多人會問，手掌分開了，水流不會漏出去嗎？從指縫中流走了，還能增加划水效果？這個阻力面是有漏洞的啊？

在這方面，專家也給了科學的解釋：手指分開後，一些水流會進入手指間，而游泳過程中水產生的湍流會相互作用阻止指間的水流動，因此手指微微分開可以有效的增加手的表面積。

雖然不可否認肯定有漏網之魚的水流透過指縫流走，但是來自荷蘭埃因霍溫科技大學流體動力學家Josje van Houwelingen稱，在某種程度上，增加手部表面積的效果大與指間水流產生的阻力。

所以，這個並不難的小細節也可以幫助你掌握到職業選手的技術點！

腳掌內扣，增大你的打腿面積

在自由泳和蝶泳中，我們一直在強調打腿時，腳掌要內扣，形成「內八」，而不是豪放的「外八」。

腳掌的內扣可以讓你在打腿下壓時，對水面積更大，從而產生更強大的推進力。

我們以自由泳垂直腿的角度可以更明顯的觀察出「內八」與「外八」的區別。

內八垂直腿

外八垂直腿

而兩者之間腳背對水面積的差異是非常巨大的！

不同腳掌狀態對水面積不同

所以，在打腿過程中，讓你的雙腳大腳趾「靠近」，形成腳掌的內扣，打腿能力會得到大幅度的提升！

總結

OK，今天我們分享了三個游泳的小細節動作，這些細節都是職業運動員必須掌握的技術要領，沒難度卻很實用，同樣適合業餘愛好者練習領會，不要覺得專業就一定很難喲！
