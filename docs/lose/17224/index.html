誰說楊丞琳沒有過人之處？她鍛煉身體這麼努力，還不夠優秀嗎？

2019-12-16 49條評論 1,008次閱讀 90人點贊

娛樂圈有那麼多歌手，每個人的唱歌風格都不一樣，所以，經常會有小夥伴出現聽不太懂唱作人是什麼用意，這很正常，小練身邊有小夥伴說聽不懂楊丞琳的歌曲，幹嘛要聽懂？旋律好聽就可以了，你要做的是學習她健身的方式和方法。

楊丞琳每次現身演唱會現場的時候，都身穿的特別有特色，這張現場照中，她身穿著一條青綠色弔帶裙，配合著深紫色染髮，看著很有感染力呢。

楊丞琳平時在健身鍛煉也非常認真，你看照片中她身穿著黑色運動背心，左手叉著腰的樣子，很顯然是剛健身完，這模樣是不是很迷人呢？

除了去舞蹈室做力量拉伸運動之外，楊丞琳還會去鍛煉舞蹈，她穿著一件白色緊身背心，搭配著一件深色背帶褲，顯得很有活力。

頹廢現在已經成了很多小夥伴在減肥瘦身時候都會遇到的狀態，我們變得不喜歡上班、工作、甚至去健身鍛煉，整個人變得很疲憊，這是因為你鍛煉過度而造成的，你可以休息幾天再繼續鍛煉身體，就會感覺好很多呢。

體式一：樹式

身體平衡性不是只有靠鍛煉鶴蟬式才能夠提高，其實鍛煉一個簡單的樹式體式也是可以幫助我們提高一些身體平衡性的。

體式要點：開始鍛煉的時候，我們要把雙腿綳直，並且自然的站立在地面上，而後將其中一腿彎曲抬起，雙臂也要舉過頭頂，才能更好的鍛煉。

體式二：戰士三式

除了樹式之外，戰士三式也是可以幫助我們提高身體平衡性的，但是這個體式較難，鍛煉時要細心一些才行。

體式要點：讓左腿綳直，並且穩定的站立在地面上，然後在上半身向前彎曲的時候，順帶著將其中一條單腿向上抬起。

體式三：女神式

相比上個體式而言，女神式主要鍛煉的是身體的雙腿柔韌性，而且對上半身肌肉要有適當的拉伸效果。

體式要點：不同於戰士三式，把兩條腿向左右兩側分開伸展，隨著上半身下沉的同時，膝關節也要彎曲，最好將雙腿彎曲的尺度大一些。

體式四：單腿下犬式

單腿下犬式相信你很少會鍛煉，因為這個體式並不適合初學者鍛煉，但如果你身體平衡性很好的話，不妨試著鍛煉下哦。

體式要點：與標準版下犬式相比，單腿下犬式需要在鍛煉時，將其中單腿向上伸展，但是不要因此而亂了陣腳、失去平衡。

體式五：戰士三式變式

今天小練主要帶大家鍛煉的是身體平衡性，所以，我們在最後要再重複鍛煉下戰士三式，來讓身體肌肉產生記憶，便於我們再次鍛煉。

體式要點：戰士三式也是需要把上半身向前彎曲的，同時把雙臂和其中單腿向上抬起，盡量保持3-4分鐘，以此來達到更好的鍛煉效果。

鍛煉瑜伽和做減肥運動、做健身運動是一樣的，因為我們不管做哪項運動，我們都是需要堅持做下去的，哪怕身體會感覺勞累，哪怕會感到肌肉很痛，所以，經常健身的人遇到困難也不會輕易言敗的。

小練點評：有人說楊丞琳的發聲一直都不是很好，高音比較牽強，會讓人感到有一種氣息不足的感覺，但最近幾年，她的唱功得到了提升，感覺就要好很多，她對自己聲音的控制越來越好，還是讓很多人看好她的。#清風計劃#
