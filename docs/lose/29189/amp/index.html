不看不知道！青少年打籃球的好處

天天要聞
2019-12-23

籃球，是青少年最愛的運動之一，很多孩子從小開始練習籃球，不但鍛煉了身體，還認識了不少朋友，受益良多。金剛籃球俱樂部通過多年的教學經驗研究發現，打籃球的好處實在是太多了，下面我們一起來看看打籃球到底還有多少好處吧！

1、 身體健康，促進成長

據調查，全國85%以上的青少年處於亞健康狀態，包括疾病多發、骨骼發育緩慢、體能缺乏、過度肥胖瘦弱等。籃球作為一項綜合球類項目，充分鍛煉到孩子的身體各部位，全面有效的提高綜合身體素質，促進骨骼發育成長，長期鍛煉有增高的效果。

2、 訓練神經系統反應

打籃球可訓練腦部思考和判斷能力，及隨思考後身體繼發的反射動作速度。提高孩子的反應能力及身體協調能力。

3、 培養孩子勤奮、執著的精神

籃球看似簡單，其實難度很高，不經歷足夠的訓練磨礪根本打不好球。每個動作都需要不斷的練習，日復一日，需要孩子長期的堅持。所以會打籃球的孩子必然在場下有堅持嚴格的訓練，保持著勤奮、執著的心。

4、 團結意識增強

籃球是五個人的運動項目，一場比賽的勝利離不開賽場上五個人的配合。籃球運動不是一個人在戰鬥，喬丹是很強大，但他的公牛王朝也是五個人在場上一起努力打拚下來的，你永遠不能忽略了其他隊員的功能。

5、 有利於個人發展和人文素質提高

通過籃球訓練和比賽的過程，能使參與者的個性、自信心、情緒控制、意志力、進取心、自我控制與約束等方面都有良好的發展，以及培養團結、努力協作、文明自律、遵紀守法、尊重他人的道德品質和集體主義精神。
