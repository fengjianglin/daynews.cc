假如世界上沒有Nike與adidas...這些冷門好鞋同樣值得入手

2019-12-24 843條評論 3,626次閱讀 270人點贊

假如世界上沒有Nike與adidas，你會選擇上腳哪些球鞋？AJ1、Nike AF1與YEEZY這些熱門球鞋看多了，難免會讓人產生視覺疲勞。今天，不妨跟隨毒App一起「換個口味」，將目光放在這些不輸給Nike與adidas的冷門好鞋上吧。

AMBUSH x CONVERSE Pro Leather

毒App用戶 @KW

（點擊圖片可直接跳轉毒App小程序購買）

今年下半年，我們迎來了人氣街頭品牌AMBUSH與CONVERSE的一系列合作。其中，白色版本的AMBUSH x CONVERSE Chuck 70備受高街愛好者青睞，而另一款AMBUSH x CONVERSE Pro Leather的實力亦不容小覷。

該鞋款以純黑色貫穿皮質鞋身以及硫化中底部分，並在後跟部分加入檸檬黃色「AMBUSH」字樣以及不同的鞋碼數字作為點綴，黑、黃雙色所呈現的搭配辨識度極高。

毒App用戶 @無名之人

同時，該鞋款採用高幫Pro Leather鞋型作為藍本打造，十分適合冬日穿著搭配。該鞋款內里部分亦以麂皮絨材質塑造，隨球鞋附送的黃色鞋帶亦可為整雙球鞋的設計增色不少。

CONVERSE x GOLF le FLEUR* Gianno

毒App用戶 @十五個幽靈

（點擊圖片可直接跳轉毒App小程序購買）

知名說唱歌手GOLF WANG創立的潮流品牌GOLF le FLEUR*與CONVERSE已經有過數年的合作，今年GOLF WANG與CONVERSE亦迎來了一次「劃時代」的改變。

此番CONVERSE與GOLF le FLEUR*並未選擇人氣鞋款Chuck 70以及One Star作為設計藍本，而是另闢蹊徑地選擇Gianno鞋款塑造全新聯乘系列。

毒App用戶 @Potachee

CONVERSE x GOLF le FLEUR* Gianno分為黃/藍以及白/綠雙色，該鞋款在外形上與近年流行的「山系」球鞋有著異曲同工之妙，寬大的廓形搭配厚實、全天候適應性極強的中底部分，對於喜歡日系、工裝穿搭的朋友來說是不二的選擇。

Vans Lampin x CDG黑白

如果你是Comme des Garçons的粉絲，你一定對其副線CDG聯合Vans打造的滿印聯乘鞋款記憶猶新。近日，Vans則再度同CDG展開了新一輪合作。

毒App用戶 @梁學成TUHE

本次，CDG選用名不見經傳的Vans Lampin鞋型作為聯乘鞋款的原型，依舊延續其簡約黑、白雙色主調以及滿印logo設計，為該鞋款帶來十足辨識度。

毒App用戶 @GYOOH

同時，這雙Vans Lampin x CDG更在鞋身處加入黑色麂皮材質予以搭配，與白色鞋身帆布材質形成較大反差，同時亦為該鞋款形成出眾的層次感。

Retrosuperfuture x Vans Vault OG Style 43 LX

毒App用戶 @EasonHe_

（點擊圖片可直接跳轉毒App小程序購買）

球鞋與眼鏡品牌的跨界合作，聽起來是否有些不現實呢？日前，Vans就攜手來自義大利的眼鏡品牌Retrosuperfuture帶來一系列全新聯乘球鞋，其視覺效果大大出乎了眾人的意料。

該鞋款以Vans Vault OG Style 43 LX為藍本，鞋身分別以黑、白雙色作為色彩主調，並採用細膩、質感上乘的翻毛皮材質塑造該部分。至於硫化中底部分，則採用活潑、跳脫的果凍色作為主要設計，黑色鞋身搭配紫色鞋底、白色鞋身搭配熒光綠色鞋底，所形成的視覺吸引力頗為不俗。

Bodega x New Balance 997S「No Bad Days」

毒App用戶 @belanlo

（點擊圖片可直接跳轉毒App小程序購買）

進入2019年後，來自波士頓的球鞋店鋪Bodega攜手New Balance打造了兩雙聯乘997S鞋款，其中一雙是已經炒至天價的「No Days Off」配色，另一雙則是本次毒App要為大家介紹的「No Bad Days」。

毒App用戶 @折序

該款Bodega x New Balance 997S「No Bad Days」採用多重材質搭配塑造鞋身，包括編織紋理、網眼面料以及麂皮材質等，在色彩選擇上亦精彩紛呈，其整體給人的感覺就如同一部集大成的「球鞋教科書」。

毒App用戶 @小連

同時，為更進一步彰顯其聯乘身份，該鞋款在鞋舌部分加入特別版本「NB」字樣logo；該鞋款後跟處亦以紅色刺繡「997S」字樣點綴，在細節方面的用心程度不容小覷。

PUMA x Ader Error RS 9.8灰藍

毒App用戶 @Korina張

（點擊圖片可直接跳轉毒App小程序購買）

今年，知名球鞋品牌PUMA迎來了與韓國年輕人的「最愛」——ADER Error的一系列合作，雙方所帶來的聯乘鞋款除了我們此前介紹過的TrailFox外，亦推出了一款RS 9.8合作跑鞋。

毒App用戶 @Twentyne

該鞋款採用填充棉材質搭配麂皮面料塑造鞋身，米白、淺灰雙色交織賦予該鞋款更為低調、百搭的既視感。ADER Error的設計風格以「多彩」為人所熟知，該鞋款亦在鞋舌、後跟部分加入ADER Error品牌標誌性藍、橙雙色作為點綴。

毒App用戶 @KIMkimmmm

該鞋款另一大設計亮點則在於鞋身部分的「標籤外縫」設計，這也是ADER Error主線單品中常用的設計之一，從這之中我們也能領略如今年輕潮流愛好者們所欣賞的解構美學。

Puma Thunder Spectra White泫雅同款

說到復古跑鞋，可別只想到Nike Air Max 1，來自PUMA的Thunder Spectra white也是一款復古氣質十足的跑鞋，就連國民女神泫雅亦對其青睞有加。

毒App用戶 @Booker女朋友

該鞋款採用帶有做舊效果的米白色作為鞋身設計主調，並以此貫穿中底部分，整體設計簡單卻不失年代感。同時，該鞋款更在鞋側中底處加入紅、綠雙色細節作為點綴，以較為強烈的色彩反差使得該鞋款更具視覺吸引力。

Reebok CLUB C 85 VINTAGE

毒App用戶 @漢尼拔不冷血

（點擊圖片可直接跳轉毒App小程序購買）

最近提到Reebok，你是否會想到其與adidas聯合打造的新科技跑鞋呢？在球鞋黑科技方面Reebok從未落後，而在復古風大熱的當下，Reebok CLUB C 85 VINTAGE更是一款不可多得的OG精選。

該鞋款採用低幫板鞋鞋型作為藍本打造，鞋身輪廓與上世紀80年代所流行的跑鞋外形極為相似。同時，該鞋款採用經典的米白/森林綠配色作為設計主調，經典網球鞋氣質呼之欲出。

毒App用戶 @IMWHY123

該鞋款在鞋側、後跟以及鞋舌部分均應用綠色塑造「Reebok」品牌logo字樣細節，並通過鞋身多重部分的剪裁、拼貼營造更為突出的層次感。如果你想要入手一雙具有高性價比的復古鞋款，這雙Reebok CLUB C 85 VINTAGE大概是眼下不錯的選擇之一。
