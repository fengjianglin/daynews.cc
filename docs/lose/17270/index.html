徒手練胸中縫，用這個動作就可以

2019-12-14 1,435條評論 3,089次閱讀 1979人點贊

今天有朋友問我說,我不去健身房,沒有器械,怎麼練胸中縫?其實在家徒手用鑽石俯卧撐就可以,但是鑽石俯卧撐需要很強大的三頭才可以做,在做不了這個動作的時候你可以用這個動作:斯萬推胸。

圖片來源於網路

動作要領:

1. 挺胸直立,沉肩收緊核心,脊椎保持中立自然,雙腳與肩膀同寬站立。

2. 雙手用力,在身體前方擠壓一片輕量的杠鈴片(如圖),掌心朝內,指尖向前,與胸等高。

3. 在與胸同一水平高度上手掌用力壓實杠鈴片送的同時用力擠壓杠鈴片,送到肩膀前方短暫停留後,將杠鈴片拉回胸前。即為完成一個動作

4. 重複至規定次數即可。

圖片來源於網路

訓練時注意細節:

1.全程需要保持核心穩定,身體不要擺動。

2.全手掌壓實杠鈴片,手掌壓得越緊，胸肌擠壓感會越強烈。

3.向前送出杠鈴片的時候,盡量讓杠鈴片朝肩部的正前方送出。

4.還原的時候將杠鈴片拉回胸前。一定要注意,是拉回胸前,而不是往下放。

5.還原的時候肘關節向兩側分開移動,而不是向下運動,全程保持在一條線上。

圖片來源於網路

當然你也可以選擇躺在凳子上面做空上動作,躺下的好處在於身體的重要由凳子承受,能更好的把注意力集中在胸縫處,肩夾穩定,能更好的保正杠鈴片在動作過程中保持在一條直接上。由於圖片中用的是杠鈴片,你在家中沒有的話拿個本子就可以,要是你想注意力更集中可以用手機,(嘿嘿,此處有笑聲,開個玩笑)其實徒手也可以注意過程雙手向內用力就可以了。如有其它問題可以關注並私信我,你的關注就是我前進的動力。
