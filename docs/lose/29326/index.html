會跳舞&不會跳舞的人，這4個地方差別不是一般大

2019-12-20 187條評論 2,094次閱讀 282人點贊

身體體態

跳舞時最容易犯的毛病就是”塌腰撅屁股”，這說明他們還不善於控制自己的身體，或者說，他們的身體還處於生活中的自然狀態。跳舞時首先應該把腰立起來，在立腰的同時，把小肚子、小屁股都收起來。這樣身體就能保持正直了。當然，身體正直的要求還不僅這些，它還包括，把雙肩打開，後背平直，感覺到往那兒一站，身體好像長高了似的。能做到這些，身體就能克服自然狀態，保持跳舞所需要的那種正直。所謂正直，並不是要使身體緊張，以致從面部肌肉到脖頸部都僵硬起來。用僵直的身體跳舞，不會帶來美感。在跳舞的過程中，不論是站立、下蹲還是跳躍，身體都需要一種正直而自然的美。

伸直手臂

　　當一個不善於跳舞的孩子把兩隻手臂向上舉過頭頂時，別的孩子往往譏笑他”像投降”。細看一下他的手臂，有兩個毛病：一是手臂彎度太大，二是手臂位置過於往後。這個例子說明，在一般情況下，手臂要伸得長些才好看。另外，不管是在頭上邊，還是在身體旁邊，手臂都在身體前邊一點才好看。

　　但手臂只有長，長得沒有一點彎曲，像個棍兒，也不好看。手臂不能只有長，還要有圓。從肩膀開始至臂肘、手腕、手指尖，都要保持一點彎曲、一點圓。但不能彎曲得過大，過大手臂就顯得短了，臂肘、手腕也像要折了。

　　怎樣才能做到手臂長而圓呢?舉例來說，兩隻手臂如果在身體前面或旁邊手心相對端平，從肩膀開始至手指尖，應該一點點低下去，在肩頭上滴水，可以使它沿著大臂、小臂一直順著手指尖漸漸流下去。能夠大體上達到這種要求，手臂也就顯得長而圓了。如果不是這樣，手臂彎曲得過大，滴水就流不到手指了，那樣，將會破壞手臂的美。

　　除此之外，手指也要鬆弛、自然，不要因為故作姿態，而使手指僵直。舞姿的美主要表現在上身。而上身又主要表現在手臂上。如果我們的手臂理順了，也就為人們帶來了美感上的享受。

腿部線條

　常見有人跳舞時，把腿翹得很高，但膝蓋是彎著的，腳是勾著的，這就顯得很不美。腿的線條美與不美，不在於抬得高與不高，而在於有沒有伸直。一個腿抬得很低的舞姿不見得就不好看。當然，也有使腿彎曲的舞姿。比如常見的一條腿直立、另一條腿向前或向後，小腿向上彎曲的舞姿，也是很美的。但這種彎曲是有控制的，有意識使它這樣彎的。敦煌舞蹈中的許多舞姿，不僅膝蓋是彎的，腳也是勾的，但那是一種特殊韻味的彎，一種特殊需要的勾。

　　會跳舞的人，有時也有那種彎腿或是勾腳的動作，不是因為跳舞的人不會伸直腿、不會綳起腳，而是彼時彼地需要他那樣做。但孩子們跳舞時表現出來的，常常是不會伸直腿，不會綳腳。比如孩子們常做的”跳跑步”，一隻腳踏地、一隻腳抬起時的腳是勾著的，無論如何也不美。自由體操或是藝術體操，雖不具備更多藝術內容，但運動員那拱起的腳背，那腿部外開的、清晰的線條，能給人以美的享受。所以，一定要會伸直腿，這樣學會綳腳，跳舞時不僅別人看著美，自己也會覺得舒服。

身體的協調

　　舞蹈與體操不同的地方，就是舞蹈最講究動作感覺的美。如一隻手伸出去了，是高的，還是低的，手心是向上，還是向下，動作是柔和的，還是有力的。視線隨著看到了什麼地方，是柔情地看，還是堅毅地看，等等。這些因素加在一起，就能使人感覺到孩子們是在看花，還是在採花;是在指什麼，還是在向遠處眺望什麼。

　　要注意對孩子們眼神的要求與訓練，要引導他們從手眼配合上做起，去感受動作的美。動作感受的美還表現在上下身的隨和之中。如果一個擺動，身體卻向右傾斜，或是直立不動，上下身就顯出了不和諧。如果一個右手在頭上，左手在胸前的舞姿，不僅手是向左傾的，整個身體的感覺都是向左傾的，就會產生動作和諧的美。
