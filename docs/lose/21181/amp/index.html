人不瘋狂枉少年，球不提升枉白練，每天不打球，白來世上走

天天要聞
2019-12-19

對於球友而言，每位球友在其接觸羽毛球的過程中，總會陷入一種「瘋狂熱愛」的狀態，這些熱愛表現到行動上，真是讓人大開眼界。為了多打羽毛球，簡直憋出了「喪心病狂」的想法和行動。

我們就看看為了打球做過最瘋狂的事！

1. 冬天天冷，普通人取暖有男朋友的抱男朋友，有女朋友的抱女朋友，而我就比較厲害了，我抱著球拍睡，而且晚上還能夢到打球。

2. 在陪好基友打球和陪女朋友逛街這兩件事上，果斷選擇了前者【但一定記得給女朋友發個紅包，不然很容易淪為單身狗】。

3. 你拉著女朋友的手，總是不自覺地體會不同的握拍感覺。

4. 你還沒結婚/沒有女朋友，但此時你已經為自己孩子制定了學習羽毛球的目標。

5. 你為了打球「不擇手段」的討老婆開心，只為留出時間來打球。

6. 曾經以生病為由，從工作崗位上溜號跑出去打羽毛球。

7. 曾經在真正生病的時候，還跑去打球。

8. 曾經為了練羽毛球曠過課，為了打羽毛球比賽，放棄四六級考試。

9. 你買了網球拍，但目的不是為了打網球，而是為了訓練腕力。

10. 你的身上總有一些東西能證明你超愛羽毛球，比如手機殼，衣服的牌子，小飾品等。

11. 一年燒了很多裝備，看到新品總是「控記不住我記己」的手去買買買。

12. 你握拍的那隻手臂看起來明顯要比另一隻大很多，傳說中的麒麟臂。

13. 各種小長假，不出去旅行，天天泡球館。

14. 帶著吃得東西去球場，從上午打到晚上，餓了就當場解決，儘管身體很疲勞，但精神依然很亢奮。

15. 當有人質疑羽毛球是女人運動，噴你喜歡的球星是「千年老二」時，你會和他講解，甚至會和他去撕逼，更甚者道不同不相為謀拉黑朋友圈。

16. 你會為了一個技術動作，在球館枯燥的練上千百遍，直到自己滿意為止。

17. 你在家練揮拍，把球拍漆水磕掉，把天花板打花。

18. 你在電視機前看羽毛球比賽時，總是時不時站起身來，若有所思、煞有介事地手足舞蹈，模仿其中的動作。

19. 在家時，用刀切菜是標準握拍，夾菜的過程，必須按照先內旋後外旋的步驟進行。

20. 走路時，總習慣最後一步跨出去，別人總是用異樣的眼神看自己。

21. 明明已嚴重拉傷大夫警告禁止激烈活動，要多休息，但還是忍不住要提前去球場小打幾下，興趣一來，小打變大打，從而導致傷上加傷。

22. 明明已經感冒了，非要來球場大幹一場，說打球可以使感冒好的更快一點。

23. 你讓人攝下你打羽毛球的鏡頭，發到網上請大神指導。

24. 你收集所有的羽毛球比賽資源，只為留住經典，光硬碟就好幾個，因為一個高清視頻好幾G。

25. 你總是為看不到主流的媒體轉播羽毛球比賽而破口大罵。

26. 無論高溫還是嚴寒，無論狂風還是暴雨，都阻擋不了一顆打球的心。

27. 一個球多級利用，好球時打比賽，打掉毛後留著訓練用，直到訓練把毛打光，這才是真正的物盡其用。

28. 在球館激戰正酣時手機響個不停，沒有重要的聯繫人，絕對不接，要不然肯定打不成球了。

29. 你心目中最理想的伴侶是能找一個能打混雙的老婆（公）。這樣也算走上人生巔峰了。

30. 前些年腳掌心長個雞眼，忍痛打了兩周球，結果自行脫落，硬是被我踩死了！

31. 最瘋狂的是本人去球館出勤率比吧台管理員高，結果管理員把鑰匙給我了，讓我給他看場子。

32. 中午在辦公桌上趴著睡覺時，夢見跳殺了，一使勁差點把桌子掀了，被同事投來鄙視的眼光。

33. 懷孕初期，忍不住天天看，什麼懷孕鬱郁啊，狂躁啊，到球場人就順暢了，到中期還去球場打球，腳不方便動，練手呀前期中期都住院的呀，好點就去了……就去了前期中期都住院的呀，好點就去了……就去了。

34. 有位球友剛和老婆剛在民政局登記完，被問晚上還去打球嗎？當然去啊！老婆說那你應該和羽毛球結婚。

35. 我更瘋狂，為了打球開個羽毛球館！（請接受我的膝蓋）
