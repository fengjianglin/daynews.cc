李若彤自曝健身20年！看到她穿薄紗裙身型，終於明白為啥隱瞞年齡

天天要聞
2019-12-19

95版的《神鵰俠侶》在眾多金庸武俠劇中堪稱經典中的經典了，小龍女的扮演者李若彤也是因這部劇一舉成名了，當年的小龍女算是驚為天人，而如今的李若彤狀態也是絲毫不減當年，最近參加活動的黑色紗裙造型更是令人驚艷。

李若彤本人也是在社交網路上發出這張身穿黑色紗裙的照片並且自曝已經堅持健身和保養20年。可以看出李若彤的身形雖然瘦，但是非常健康，手臂線條優美，細腰沒有一絲贅肉，一看就是長期鍛煉的結果，V領加弔帶更襯出她修長的天鵝頸，紗裙下擺一雙美腿若隱若現，這樣輕盈纖瘦的身形，果然還是當年那個小龍女。

在條裙子的基礎上，李若彤又搭配了一件粉色亮片的西裝外套，既保暖又打破了黑色紗裙的沉悶感，再加上李若彤本來就白，這件外套更是襯得她白到發光。

李若彤一直對外沒有公開自己的年齡，可現在看來她的狀態真的猜不出她到底幾歲了，從近期的照片看，比起當年劇中的形象，李若彤就是臉型比以前少了膠原蛋白，更有線條感了一些，但是那種清冷的氣質仍然在，所以這樣西裝加紗裙「又娘又man」的造型非常適合李若彤。

生命在於運動這句話確實不假，李若彤能保持現在的狀態絕大多數的因素還是科學合理的進行身材管理，再加上香港人的飲食偏清淡滋補，對於皮膚和身材也是非常有好處的。變美還是能依靠自己後天努力的哦！

李若彤現在的造型都是走優雅幹練的女神范，而且她身形又修長纖瘦，非常適合修身的禮服裙。像這套露肩連衣裙，不但展現出她優越的肩頸線條，側邊的開叉設計露出一點紅色的拼接設計，增添了時髦感，腳上的紅色高跟鞋又與之相呼應。在穿搭中，鞋子或配飾可以選和衣服的印花或者拼接設計色系相同的，可以增加整體造型的精緻感哦！

她的私服也都是優雅幹練那一掛的，這件孔雀藍的大衣真的超級顯白，再配上花灰色的貝雷帽更是優雅到骨子裡。貝雷帽也是秋冬天的扮靚利器，如果自己秋冬天的大衣都是深色的話，可以選淺色的貝雷帽來搭配，輕鬆打破沉悶感哦！

像珊瑚橘這樣的亮色李若彤也駕馭的非常好，裙擺的建材很特別，露出腿部側面的優美線條，是一番別有風情的性感，三七分大波浪卷的髮型很有復古港風的味道，彷彿是剛從老港片中走出來的美人。

李若彤的長相也是偏東方古典美的感覺，所以這種帶有中國風元素的衣服非常適合她，這件黑色上衣是模仿旗袍的版型，復古又時髦，下身的半裙也是黑白色系的印花手繪，跟上衣搭配起來更顯飄逸動人。不敢嘗試旗袍或者漢服這類服裝的話，這種帶中國風元素的單品也很值得入手哦，和基礎款搭起來也很好看。

李若彤自曝健身20年！看到她穿薄紗裙身型，終於明白為啥隱瞞年齡！看到這裡是不是覺得手裡的炸雞奶茶曲奇餅乾立馬就不香了呢？好身材好皮膚都是靠著運動和保養日復一日年復一年的努力換來的，想要穿什麼都好看，還是先好好減肥吧！
