球界最可怕的過人技術，羅納爾多鐘擺式只能看不能學。

天天要聞
3 天 ago

足球過人，是一種踢球方法，目的是將足球主動性牢牢把握在腳下。然後尋找有利時機快速傳球或射門的過程。在世界知名球星中，有兩種球星成長最快。第一種是射門技術高超的球星，這種球星很快就能成為世界知名的球員。比如圓月彎刀老祖貝克漢姆，還有球星C羅都屬於第一類。第二種是過人技術高超的球星，比如內馬爾就屬於後者。

而在眾多過人技術高超的球星中，我們找到了幾位最具觀賞性的過人技術。第一位過人技術高超的球員就是我們所熟知的大羅，羅納爾多。羅納爾多經典過人技術為鐘擺式過人。這種過人技術需要一定的身體素質，同時運動幅度很大。因為他是靠著身體的優勢，將身體兩側的球員排開，然後尋找空擋一腳射門。所以看上去簡單，實際上操作難度挺高的。

第二位過人技術高超的球員是小羅。小羅的過人技術其實更具觀賞性，他的過人叫牛尾巴過人。這種過人技術是利用球員的運動靈活性，和反應能力。先將足球踢到對方面前，讓對方一陣慌張，然後再用腳快速將球勾到，接著在往另一個方向突破出去。因為用腳勾球本就是一門技術活，所以小羅的牛尾巴過人是很少有球員能學會，並且這種過人技術是個人身體素質和臨場反應的體現，別人就算學會了，恐怕也不會用。

第三位過人技術高超的球員是齊達內。齊達內的成名過人絕技名為馬賽迴旋過人。這一招聽著古怪，但其實操作難度更高，馬賽迴旋過人同樣分為兩個步驟，第一個步驟是正常帶球過人，正常帶球過人的瞬間，利用身體臨場反應，向後轉體，將足球運動軌跡帶往對手相反的方向。在這一瞬間，對手的腦袋就像被侮辱了一樣，根本摸不到頭腦，所以小羅便藉助這一招，輕鬆過人，輕鬆傳球。

第四位過人技術高超的球員是內馬爾。內馬爾過人技術想必大家已經很熟悉了。他屬於那種天才式的球員，天賦異稟。過人技術分為三步。一般內馬爾這種過人技術，恐怕只有真正的天才才能在球場上用到，並且即便學會，也不一定能用得到，能臨場發揮。第一步為衝刺階段，內馬爾遇到對手的時候，會正常帶球衝刺。當遇到對手瞬間，就會瞬間停下來，用後腳跟將足球帶上一個小弧度。當足球在他腳下有了一個小小的弧度後，他會繼續用後腳跟踢第二腳。第二腳踢出去之後，足球所承受的力更大，旋轉弧度更高一些，足球就從他的肩膀穿過去，再被他一腳定住。整個過程可以用眼花繚亂來形容。所以內馬爾過人技術也被球迷們親切稱為彩虹過人。

不過因為彩虹過人實在太酷炫，所以內馬爾也一直被對方球員們所嫉恨。但凡遇到內馬爾的彩虹過人，對方球員會頓時憤怒，感覺智商被侮辱，二話不說就給內馬爾一個犯規的如來神掌。
