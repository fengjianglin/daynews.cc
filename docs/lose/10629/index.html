散打的學習方法和要領

一、武術散打學習各階段目標和任務

初型概念階段：

這是基本理論知識、基本功、基本技術動作學習和掌握階段。此時應該加強基本功訓練，掌握基本理論知識，動作原理，掌握正確的基本動作，這決定著擊打力量、速度和攻防轉換等致勝關鍵要素。這一階段一定要打好基礎，練好專項素質和基本功。專項包括：力量、速度、靈敏、柔韌等。基本功則包括：基本的站立姿勢、步法、拳法、腿法、摔法、防守技術等。

攻防意識和戰術意識形成階段：

在鞏固基本動作、不斷提高專項素質和熟練掌握多種技術動作組合的前提下，通過進攻、防守、反擊的訓練，逐漸加強攻防意識，此階段還要注重戰術意識的培養與練習，逐漸提高水平。有了一定的基礎後就應該去熟練掌握攻防技術，全面、系統地學習實戰的攻防技術和戰術。掌握招式組合的規律，學習組合的攻防技術動作，形成個人特點，同時也要針對自身的不足進行訓練，彌補不足。

技術特長、技術打法的形成階段：

通過學習，個人的技術和實戰水平有所提高，形成了個人的技術特點，這時應該去培養出自己的絕招，在比賽中我們常常看到一些運動員是有自己的招牌動作的，如邱建良的神龍擺尾，柳海龍的柳腿劈掛等。在全面提高攻防技術的前提下形成自己的技術特長，不斷提高專項能力增強擊打力量，逐步走向成熟，在實戰中找到自己的不足和問題，並加以改進，積累經驗、檢驗自己的訓練成果，同時也可以學習他人的優點，要善於總結的發現，這是這一階段的目標和任務。

自學散打的幾點技巧

1、 掌握看圖或視頻的方法

在自學時，看圖學習是一種很好的方法，看視頻也是一樣，在看圖或視頻時要認真看完講解或文字說明，分清一個動作分為幾個步驟，並按要求先進行分解練習，待到能夠掌握規範的分解動作後，在進行完整的動作練習。同時還要了解動作路線和位置，邊看邊做，逐步分解體會，琢磨練習，一個動作一個動作的練習直至掌握技術要領。

2、 相互交流

在自學時，如果有人結伴效果會好。一人按照說明檢查，一人看教學資料後按要求進行練習，互相觀摩糾正，體會技術要領。這種方法在學習攻防技術更為關鍵。可在所在地觀察是否有散打練習者，或到一些

現在網路科技發達，自學的朋友可以以論、視頻軟體等網路社交的方式進行交流，相互學習，同時也可觀看一些教學視頻、文章、圖文等。
