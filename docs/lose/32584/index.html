「高級進階」 欲揚先抑——從啟動小跳說開來

2019-12-24 1,058條評論 2,788次閱讀 986人點贊

啟動小跳，是每個球友都比較熟悉的腳步技術動作，在中羽論壇里，也一直是個比較有爭議的說法，一般認為，步伐啟動前需要有個快速的小幅度的起跳動作，銜接啟動後的跨步會比較快速有力，在實際應用中，不少球友也會感覺到這個小跳帶來的便利。但同時也有球友指出，專業的運動員或者業餘高手的啟動，是沒有這個小跳再銜接跨步的動作，看起來就是一步跨出去，他們認為這樣的動作會更節省時間，方便你快速到位，額外增加的小跳雖然可能使跨步更迅猛，但本身會浪費一點時間，可能無用甚至得不償失。

啟動小跳銜接跨步能更快的啟動步伐的第一步，在這一點上，大家都沒有異議。其實深究起來，這裡面的道理不那麼簡單。小跳提升跨步的爆發，一般認為是把靜止的重心提前動起來了，減少了從無到有加速的那個相對比較滯後的過程。這個是比較通俗的說法。其實從運動生理的角度來說，這裡牽扯到一個肌肉的超等長收縮（Plyometrics）——將肌肉拉長之後，很快切換過來進行收縮的動作。我們可以形象的把肌肉和附屬的肌腱看做是一根彈簧，把它拉長之後就會儲存一些彈性勢能，從而可以在接下來的收縮過程中增加一些力量。

跳躍和跨步等對爆發力要求比較高的動作，可以通過超等長訓練（plyometrics training）或者稱為「SCC（牽張收縮循環）訓練」來提高爆發力。其代表例子是被稱為「跳箱練習」的方法，從半高的平台或者箱子上跳下，在著地的一瞬間快速地切換過來向上跳躍，並在動作切換時集中性地發揮出肌肉力量。另外如果是卧推，在負荷降低到胸部的瞬間，快速推起；如果是深蹲，在蹲下的瞬間，快速站起。動作切換時，如果能有意識地將肌力最大限度地發揮，就會變成超等長訓練。

那麼，回到這個啟動小跳，羽毛球步伐中的啟動，其實就是一個快速蹬地伸膝伸髖的動作，小跳動作其實主要是一個快速的下蹲再起跳（屈膝屈髖接伸膝伸髖的動作），是會把啟動動作中需要收縮的股四頭肌和臀大肌預先拉伸的過程，其實小跳結束接跨步的瞬間，我們還是會有一個輕微下蹲的動作。這個反覆拉伸的過程可以積蓄更多的能量，提供更大的爆發力實現快速的蹬跨，但也可能花費了較多的時間從而造成啟動滯後。從這個分析我們可以看到，這其實是一個反覆的超等長收縮，通過預先多次拉長肌肉，再很快切換過來進行收縮，可以增加我們蹬地跨步的爆發力，也就變相加快了我們的啟動。

同時對這個小跳的分析我們可以看到，如果是為了實現超等長收縮，只要在蹬跨步之前迅速地下蹲，拉伸相應肌肉再跨步，就可以實現更大的爆發力。專業運動員和業餘高手的啟動蹬跨其實就屬於這種。在跨步之前，他們都會有一個很小幅度的下蹲（重心下降）的動作，拉長肌肉再收縮就可以實現快速的蹬跨。從效率來說，這樣的一個小蹲（重心下降）所花費的時間更少，也就顯得他們的步伐更為快速流暢了。但這種技術，對腿部本身的爆發力會要求更高一點，需具備較好的爆發力，才可以在稍微拉伸蓄能後即可完成快速蹬跨，提升整體的啟動速度。這個力量需求對一般業餘球友來說可能很難實現，可以考慮通過上文提到的SCC進行鍛煉來提高爆發力。

總結一下：腿部爆發力較弱的業餘球友可以通過「小跳」來最大程度的提高爆發力，從而實現啟動時的快速蹬跨，但小跳所花費的時間相對較長，是一個蓄能時間略長的大招。爆發力較好的業餘高手和專業運動員就可以通過「小蹲」來實現最快速度的蹬跨。兩者爆發力上的差異決定了技能選擇的不同。

另外，還要多說一句，前文中我把肌肉比喻成彈簧，但兩者還有一點很大的不同，就是肌肉拉長後的彈性勢能會很快地耗散掉，不會像彈簧那樣拉長後可以保持很長時間。在接下來比較短的時間內如果快速收縮，才可以借用這部分勢能，實現超過百分百的爆發力。

這也是我們強調啟動時機選擇的原因。實戰中你判斷錯了對手出手時間，提前下蹲拉伸了肌肉，卻要等一下才能跨步，這時候肌肉拉長所儲備的能量已經耗散，你的蹬跨步就比不上踩著點的啟動那麼快了。這也是我們看到有時候球明明就在身邊，而你因為啟動這一步慢了而只能眼睜睜看著球落地，腳步就像是被釘在了地上。在面對假動作和動作一致性好的對手時，這種情況就會經常發生，重心被晃是一個原因，無法踩著點實現肌肉超等長收縮的爆發力也是其中一個原因。

其實，羽毛球技術中有很多超等長收縮的技術動作，比如前場撲球，拍子舉起來之後迎著球，在擊球的瞬間會有個手腕回收再快速揮擊的過程，接殺的時候也會在觸球的瞬間手腕快速回縮再彈擊，這樣的技術，都可以最大限度的實現超等長收縮，增加擊球的爆發力。大家想想看，還有哪些需要爆發力的技術動作應用到了這個原理？

再想得遠一點，其實在平常的生活行動中，這個原理的應用也十分普遍，仔細體會一下我們的跳躍，是不是起跳之前都有個不自覺地微蹲，從而重心下沉的過程？其它如揮拳等等，無一不應用了這個超等長收縮從而實現超越百分百的爆發。

大道至簡，百姓日用而不知。來源中羽：球齡七年半
