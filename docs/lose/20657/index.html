因肥胖而自卑，00後選擇擼鐵逆襲！

2019-12-17 443條評論 3,154次閱讀 339人點贊

我叫賴文傑，19周歲，170cm，80kg，廣東人，現在在工作室當著小教練，進過廣州環球新秀健體a組前十讀中職16歲的時候，因為肥胖而自卑……然後突然有一天下定決心要改變自己，一開始是單純的想減肥，但幸運的是學校旁邊健身房有很多大神，看見他們我十分羨慕，於是開始有了想當健美運動員的想法了。現在採用的是單獨的訓練法，一天兩練，加強弱勢肌肉，例如我上胸比較薄弱，所以練胸會分為兩次，第一次只練上胸，四到五個動作，一個動作5～8組，然後第二次正常的胸部訓練，六到八個動作，一個動作4～5組。同理其他肌群也是這樣練飲食部分的話只喝蛋白粉和bcaa，兩次訓練中都會喝bcaa，練完後蛋白粉。主食以燕麥和紫薯為主，蔬菜西藍花為主，蛋白質以雞蛋、雞胸為主，每天四餐。

剛開始健身啥也不懂，看見別人就是照葫蘆畫瓢，在動作不對的情況下加大重量導致現在腰和膝蓋都受過傷，現在訓練懂的發力，知道重量不是唯一，質量才是最重要的
