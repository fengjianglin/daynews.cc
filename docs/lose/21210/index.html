陳其遒談球：聊一聊雙打的攻守練習

2019-12-18 172條評論 2,101次閱讀 87人點贊

上一期講到二對二高吊對控的雙打練習方法（陳其遒談球：二對二全場高吊練習），到此已經基本把雙打的訓練方法介紹給廣大愛好者了。還有一些攻守的練習和強化的練習，現在推薦給大家。

我在文章中反覆強調，單純練習羽毛球技術是一個不斷重複、熟練的過程。做到這一點其實不難，大家只要付出時間和體力，經過大量正確有效的練習，就能具備一定的技術水平。

真正有差別的是運用技術的能力，每個人對球的理解深度不一樣，打出來的球也會有很大區別。因此，打球是需要遵循一定的規律的，尤其初學者要提升到中級水平階段，掌握規律相當重要。

但是，球場上有些特殊情況就需要球員主動去捕捉機會，跳開規律的束縛。前面我說的規律之外有特例，就是這個道理。

比如推壓分邊，在特定的情況下，推壓後可以不分邊，推壓的球員留在網前橫向移動去接球，這是為什麼呢？優秀球員經常能夠捕捉到連續封網連貫的機會，是因為對自身技術質量有充分了解，以及對對手習慣的分析，在快速對抗中做出正確決策。這種靈感超出了普通規律，說一個球員是否有天賦、有才華，就是看其在看似相持甚至被動的局勢下，是否能夠通過高質量的技術變成主動得分的機會。

提前移動跑位，提前判斷，就是抓對手的習慣，敢於放棄一部分範圍，集中去抓對方回球概率最高的範圍，打時間差，最快速度出手，給對手製造困難，贏得進攻得分的機會，這就是我們所說的「變速」。

當然，變速有風險，抓錯了就會被動，所以羽毛球就是控制與反控制的遊戲。

全面的攻守練習

雙打配合練習，還有全面的攻守練習。練習雙方，防守方平行站位，進攻方前後站位。防守方直接發高球給進攻方後場，進攻方開始進攻，手段不限，高球、吊球、殺球、劈球都可以綜合運用。防守方亦如此，綜合運用防守技術，可以通過彈挑後場、擋網、平抽等技術，反被動為主動，形成進攻。這是雙打的綜合表現形式。

二對二的控反控

再就是二對二的控反控練習。從發球接發球開始，和雙打比賽一樣，沒有任何技術上的限制，只是練習過程不計分。如果加上計分，就是正式比賽了。

在強化防守的訓練中，我們可以引入二防三、二防四的練習，加大防守方的練習難度。個人覺得，業餘選手如果能夠比較有質量地練習二防三，水平已經相當不錯了。

關於雙打方面的訓練方法到此已經介紹完畢，希望能夠對大家有所幫助。最後送上湯仙虎教練當年贈予我們的一句話作為結尾：「自信是成功者的生命線」。願大家在球場上面對勝利和失敗，都能始終保持自信，鬥志昂揚，享受羽毛球的快樂！
