說抽繩背包和運動水壺是健身房兩大“法寶”，你同意嗎？

2019-12-20 2,772條評論 4,934次閱讀 1896人點贊

運動水壺和抽繩袋一直是健身房必備的兩個法寶，幾乎是人手一套的狀態，所以今天要測評的就是Nike新款的抽繩袋和運動水壺。

【抽繩袋】

這次春夏新款的抽繩袋，外表看起來給人的第一感覺就好像突然置身普吉島。淺灰和熒光綠的款式，就像烈日下的椰樹林，熱情中帶著一絲清涼。

黑白塗鴉配色的這款又好像坐在沙灘上，沐浴陽光的同時喝著清熱解暑的椰子水。

不過健身房運動的小可愛們會選擇抽繩袋最關鍵的原因，並不是因為它百變的印花造型，而是它輕盈防水易收納，背起來很方便。

新一季的兩款抽繩袋對比原來舊款的也做了很多改進，主袋從原來的無隔層變成了可以乾濕分離的隔層袋。在這個基礎上還做了前副袋，整個包變得的非常有層次感。

【運動水壺】

Nike運動水壺的瓶身是經過人體工程學設計，非常好握好拿。

又因為運動完之後急需補充水分，像這種擠壓式水壺免去了開蓋這個繁瑣的步驟，的確省時省力了不少。但健身房是個人流量非常大的場所，用壺嘴沒有遮蓋物的水壺直接飲水，可能還需要考量一下衛生問題。

並且這次新款的配色其實也沒有特別吸引人，不管壺身底色用的是粉紅色、藍色、黑色還是透明，他們都用了黑色去搭配，難免會讓人覺得這個設計隨便沒有用心。

這次Nike 2020年春季新款抽繩袋，感覺上顏色都比較顯眼亮麗，乍看之下可能沒這麼好看，不過背起來，搭配一身健身裝備，還是挺亮眼的；至於新款的健身水壺，保留了去年的人體工學防滑設計，用色上也變得比較年輕，而非只是單純的黑跟白，對於喜愛健身的小可愛們來說，也就比較多選擇。

那愛健身的小可愛們平時去健身房都會用到哪些裝備或者大家想看哪些健身裝備測評呢？歡迎留言哦~！
