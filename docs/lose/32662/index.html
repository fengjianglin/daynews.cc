17歲少年健身3年，看到這肱三頭肌，真酸了

2019-12-23 488條評論 3,675次閱讀 2435人點贊

雖然健身任何時候都不晚，但是年輕人由於旺盛的荷爾蒙分泌，他們增加肌肉的速度也會更快一些，網路上不乏很多勵志的年輕小伙，通過健身改變自己的身材和命運。這位叫Tristyn Lee的少年，他的確給了網友們很大的震撼，看到他最近的身材狀態，我承認自己真酸了，這差距真的很大。

講真看到這肱三頭肌，真的練得很漂亮，可見想要增加臂圍，更需要將注意力集中在肱三頭肌上面，而非過度追求二頭肌的發展！

這位17歲少年健身3年左右，從14歲開始鍛煉，15歲上了健美舞台，一直練到如今，他給自己今年的任務是杠鈴卧推達到145公斤左右！

這是他最近兩年的身材轉變，一切都來源於他刻苦的健身訓練，這兩年他經常和最優秀的健美、健體運動員學習，並掌握了充分的增肌減脂技術，這幫助他的肌肉完成了很大的改進。據悉Tristyn每天都是生酮飲食，體脂率一直保持在5%以下，這裡並不推薦各位這樣學習，畢竟碳水是不可或缺的營養元素。

當然有很多網友可能對其身高感到好奇，事實上他目前身高152厘米左右，雖然之前醫生曾表示他有希望長到1米7，但似乎一直沒有太大的長進，再加上熱量控制的異常嚴格，1米7的身高看起來是並沒有什麼希望了。

不過可別認為是健身限制了其身高發育，這也是很多網友對他最大的誤解，在安全的前提下，適度健身會增加生長激素的分泌，也會讓身體發育的更好。他的狀況據某些健身達人推測，最有可能的原因是生長激素缺乏，所以推測他可能使用了生長激素，幫助他增加了肌肉，而非類固醇。

Tristyn的目標一直是職業足球運動，並沒有改變過。酸歸酸，對於他的努力真的非常敬佩，一位17歲的少年志向遠大，並一直通過自己的努力朝著目標一步步的靠近，這樣的精神的確是值得去學習的。
