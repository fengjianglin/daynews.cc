瑜伽與慈善，分享我們的聖誕愛心

2019-12-23 736條評論 3,980次閱讀 1631人點贊

瑜伽與慈善，因為瑜伽大家走到一起，又因為瑜伽而一起做一次有意義的事。今天分享一下我們的聖誕愛心。

以前總覺得慈善離我們很遠，但是又很近。 不管是災難還是給捐贈給需要幫助的人群，很多人都在幕後幕前靠自己的努力和熱情把這項活動堅持下去。

各個領域對於慈善活動都有不同的表現和理解。但是都離不開團隊合作和積極的參與者。

在瑜伽的世界，大家找場地、報名費等都用來捐贈給某個慈善機構。 我相信數不勝數的捐贈活動在世界各地設立舉辦著，大家為著明天會更好而努力。

最近我們這邊也有很多類似活動。因為澳洲森林大火連綿不斷，很多瑜伽館舉辦活動然後收入全數捐贈給政府消防部門。 其實這和普通課程沒什麼大區別，唯一區別就是價格高些，然後看你自己對這類活動感興趣是否願意參加等。

我覺得不一定要去參加慈善活動有了捐贈才算做好事。 還有很多方式方法能展現自己對這個社會或者對有需要的群體提供幫助也好、提供信息也好，或者還有其他創新的方法。

參與到這個活動中，既得到了瑜伽練習，又交到新的朋友也未嘗不可。 可以問自己一個問題，如果慈善活動是做瑜伽，沒有練習過的你會因為是捐贈去參加嗎?

這是一家在健身中心裡的瑜伽房

今天是超喜歡的陰瑜伽練習

懶得動不了

沒有正確或者錯誤答案。 但是都可以做的就是，無論參加與否都應該感恩，好的心態面對周遭發生的一切。

我很享受這練習不僅僅是我愛瑜伽，更是因為大家聚在一起那種積極向上的能量。 我們的主題是募集資金給13-17歲女性，幫助那些無家可歸、家庭暴力或者早孕、沒機會接受教育、貧困等等生活有困難的女性。 希望她們也能享受到相對正常的生活，也對生命有信念，知道這個世界很多關心她們的人。

這次活動的主題是籌集聖誕節基金，給她們購買一些小禮物並且舉辦一些外出聚會活動。 讓她們也感受到節日與家人團聚的美好。 在這裡家人的定義真不限於有血緣關係的人。

相信、力量、信念

試想自己在那個花季少年是什麼模樣？我們還會有過叛逆和迷茫,嚮往過自由。 可在那時從來沒想過同齡的她們卻可能連基本的都得不到保障。

這個世界有太多不幸的事發生，我們不可能關注到所有，但是我們有空時可以抽點時間去看看外面，多了解發生了什麼。 感謝自己擁有的一切。

身心放鬆的周末

瑜伽老師分享時說了一句話，感觸很深。 她說，當你們覺得穿和買內褲是很平常事情時，我們還在幫助一些女性朋友購買這類生活用品，對於她們這不是日常。

我們的認知很平凡，但是需不僅限於小小的自我。

將愛傳遞
