太極“千斤墜”真的是“頂門杠”的原理嗎？

天天要聞
3 天 ago

播過一檔關於中國傳統武術的節目。記得其中有一期講的是關於太極拳養生和實戰的專題。

在該期節目中，老師（以下簡稱老師傅）給記者展示了面對20人齊推而巋然不動的「絕活」。主持人給觀眾解釋道：這就是傳說中的千斤墜。隨後用「頂門杠」的道理給觀眾解讀其中的「力學原理」。看後一直覺得用這個「頂門杠」的原理對普通人習武人，可能存在誤導誤讀的作用，故提筆寫了幾句。

首先該節目理解的千斤墜可能就與千斤墜的原意有點分歧。大部分人理解的千斤墜原意為：練功者躺在地面或扎馬步，旁人不能使其離開地面，稱為千斤墜。節目中展示的千斤墜是眾人齊推不能使其離開原地。好吧！姑且算是千斤墜吧！但用頂門杠原理解釋這樣的展示，我還是覺得不能恰如其分。

作為主持人原意是老師傅修鍊多年，身體如鋼鐵或木棍一般，與地面形成強有力的支撐作用，並能輕鬆將眾人的推力傳入地下。就像一根頂門杠一樣，從而輕鬆化解20人推來的巨力。

乍一聽，覺得有點道理。但細想以後，還是發覺有點不通。這樣牽強附會的解讀，千斤墜也就罷了。但用頂門杠原理來佐證千斤墜，怕很多研究力學的老師們取笑我們練武人。

在講問題的是非之前，我們還是先來了解一下頂門杠抵門的工作原理吧。

有點生活知識的人都知道，頂門杠抵門時，一般地下都有一個小坑（從前地面大多是土地面，地下的坑窪隨處都是，也有抵在其他地方的）。有的還在抵門時將門稍稍提起來約3-5厘米，（那時是木門比較鬆動，現在的防盜門是提不動的）然後將門栓處的橫樑壓在頂門杠杠端。由於門的重力下壓作用，對頂門杠下端起到了固定的作用，從而達到更好的頂門防盜作用。

認為老師傅的「千斤墜」用頂門杠的原理解釋，失當之處至少有二：

一：主持人把老師傅的身體神話的比作「如鋼似鐵的頂門杠」，就比較牽強。

人練得再好，全身都是骨頭，他也不能和鋼鐵一樣硬吧？哪怕就是比作木棍的硬度，我看都過分了，更別說是比作鋼鐵了。古語有云「人乃血肉之軀」對吧？主持人說老師傅身體就是如同頂門杠一樣堅硬，我看就是一個不小的錯誤的比喻。

二：眾人在推老師傅的時候，老師傅的腳下無坑無洞無對抗外力的支撐點。可以說在不改變對老師傅推來的受力點情況下，能對抗眾人推力的只有老師傅的兩腳加上自身重力和所站弓箭步與地面形成的摩擦力。以這麼小力，來對抗20人的推力。沒有一個支撐點的話，老師傅即使真的身體堅硬如鐵，我看老師傅也未必能頂得住。

你不妨自己親自做一下實驗——頂門杠上端固定在門栓上（相當於老師傅抓住眾人的第一人），而頂門杠下端杵在光溜溜的地面（就相當於老師傅兩腳與地面的關係），在這種情況下，只要從門外用力推不了幾下子，頂門的杠子就會因為地面沒有支撐點而鬆動滑脫，失去頂門的作用了。若地面沒有支點，頂門的杠子抵門坡度再太大點的話，頂門杠本身能不能妥當的靠在門上都是問題，更別說頂門的作用了。

鑒於此理，我認為用頂門杠的原理解釋老師傅所謂的千斤墜又是一失當之處。

那麼為什麼眾人推老師傅不動呢？主要是老師傅身體本身受到的力，還沒有大到一定程度。若受到的力，超過了老師傅體重加弓箭步和兩個腳與地面的摩擦力，誰都頂不住。

是科學，不是玄學。要符合力學原理，不要自我神化，誤導常人。那樣的話，覺得會被很多明眼人笑話的。還是實事求是一點好。

大家公認的一句話叫：只有民族的才是世界的。鑒於武術實戰的一面，想講一句：征服不了「世界」的，說明你還是「民族」的。
