張鈞甯的“仙女坐”火了！彭于晏唯一公開承認女神，簡直太帥了

天天要聞
2019-12-16

瑜伽有很多流派，但是體式差不多相類似，至於你喜歡什麼樣的節奏和方式都是自己的選擇，我們每天分享的這些瑜伽體式在各個流派里都有，每日一練，你的瑜伽之路會更加順暢，身材也會有驚喜的改變。

一檔《嚮往的生活》的綜藝節目，張鈞甯又火了一把，別的嘉賓都還在睡懶覺，我們的自律女神已經早起盤腿冥想。

張鈞甯的自律，連彭于晏都毫不避諱的在微博大喊女神。

她在節目中更是純素顏出鏡，肌膚狀態彷彿停留在18歲那年，少女感十足。

前段時間51歲的蕭薔上熱搜，也再次向我們證明：與其說歲月易老，不如說投入太少，你對自己的投資，時間會給你反饋。

簡易坐姿式

直角坐姿坐立在墊子上，彎曲左腿，將左腳放在右大腿的下面，盡量將腳心朝上，雙手放在雙膝上，掌心朝下，上身直立，頭頂引領脊柱向天花板的方向伸展。

坐姿側彎式

選擇簡易坐姿坐立在墊子上，雙臂向上抬起，向左彎曲身體，左手放在身體左側的地面上，右手靠近右耳向斜上方伸展，同時轉頭，眼睛透過右大臂看向天空，然後換側同樣練習。

貓牛式

雙膝跪立在墊子上大腿和手臂同時垂直於墊面，尾骨向膝蓋方向沉下去，背部盡量向上拱起，腹部收緊下沉，感受腹部肌肉的拉伸感。

新月式

從站立前屈式開始，左腿的膝蓋、小腿，腳背落地，緊緊壓實墊子。屈右膝蓋，使大小腿90度，右膝蓋不超過腳尖,直立起上半身，延展脊柱向上，呼氣，雙手背後交握，打開雙肩與胸腔。繼續沉髖向下，收腹收肋骨，頭部引領脊柱向後向上伸展，讓腹部遠離大腿，眼睛看向斜上方的天花板，在這個體式保持20秒呼吸。

加強側伸展式

身體前傾向下，將雙手放在右腳兩側，回勾左腳，腳尖蹬地，將髖部抬高，伸直雙腿。將腹胸貼向右大腿。把你的右臀往後拉，左髖往前伸，保持骨盆中正，雙手推地或者抓在右腳腳踝處，穩定臀部和後腳的根基，繼續將你的頭往下，感受它的沉重。

戰士三式

直立起身體，雙手合十於胸前，從髖部開始身體前傾向下摺疊，同時右腳向後向上提起，右腳始終回勾，向外蹬出，右腿向上抬高至與髖部同一個高度。待身體穩定後，延展脊柱向前，收腹收肋，保持平直，保持骨盆中正，眼睛看向正前方。

單腿站立手抱膝式

直立起身體，抬右腿向前，屈右膝，抬右膝向上，交叉抱住右膝，雙手用力將腿部拉向身體，眼睛看向正前方。

戰士二式

曲左膝大小腿呈90度，大腿平行於地面右腳使勁向後蹬，伸直右腿延展脊柱，手臂向兩側平展，在一條直線上，保持髖部中正，尾骨下沉，胸腔打開，上半身垂直於地面。脖子後側放鬆延展，雙手向兩側無限延展，轉頭眼睛看向右手的延長線方向。

兩腳分開約有一條腿的距離，左腳稍微內扣，保持骨盆中正，尾骨下沉，胸腔打開，上半身垂直於地面，脖子後側放鬆延展。以右側腹股溝為折點，身體向右側彎，右手放在右腳的內側或者小腿上，左手向上高舉過頭頂，左手指向天花板的方向，轉頭，眼睛看向左手指尖方向。

每日練瑜伽，感受身體的放鬆與體態的平衡，練出好身材。
