有些時候俯卧撐就是要求數量，放機靈點吧少年

2019-12-10 1,740條評論 4,239次閱讀 1056人點贊

俯卧撐是很多人從小就接觸的訓練動作，俯卧撐既是健身的一個黃金動作，也是很多地方檢測體能的一個標準，比如說某些學校和體育機構會考俯卧撐的次數，有的保鏢公司也會對俯卧撐做訓練，部隊里也會對俯卧撐有嚴格的要求，這些地方不是健身房，針對俯卧撐的要求很簡單，就是數量。

有的人做俯卧撐做幾下就已經練不動了，對普通人沒什麼，因為他們不需要，對健身房裡的人也不重要，他們想要的是肌肉維度的增長，訓練的模式就是肌肉最大限度的增長，但是別的地方就不這樣了，像保鏢公司和部隊，這裡的考核標準就是數量，別管你發力和動作多精準，少做一下試試，整不死你。

想要突破數量的瓶頸，就是要讓你的肌肉最快的適應目前的強度，這樣就會對增加俯卧撐的數量有一定的幫助，也要學會胸肌和肱三頭肌的結合發力，讓你的「續航」能力更強。俯卧撐是用胸肌和肱三頭肌作為主發力區的，健身的時候俯卧撐是練胸的，但是想要提高數量，就把這種想法暫且扔掉，有些時候你需要的是提升俯卧撐的數量，並不是增肌。

想大幅度提高俯卧撐的數量，你可以把做俯卧撐的速度提升上去，俯卧撐這個東西，在一個範圍以內，速度越快越省力，速度越慢越費力，不信你用一分鐘的時間緩慢做一個俯卧撐試試，看看你能做幾個，然後再用你目前最快的速度做俯卧撐，數量肯定會更多。

但是快速做俯卧撐這種情況在部隊里吃不開，喊1下去喊2起來，喊完1以後就沒音了，撐著吧，想快點做也得讓你做啊。想快速提升俯卧撐的數量就不要想著增肌，相反，想增肌就別在乎數量了。
