人體成分/體脂稱相關體態指標意義解讀

2019-12-14 536條評論 3,456次閱讀 1947人點贊

經常在健康管理中心、健身房等健康相關機構會被推薦做一個檢測，叫人體成分或者叫體脂稱、體脂儀等。

為什麼要用到人體成分？

因為兩個同樣年齡、性別、身高、體重的人，可能健康程度、美觀程度相差很多。為什麼呢？

關鍵在體脂肪率/肌肉率

市面上該檢測用到的原理基本上都是生物電阻抗分析技術，簡稱為BIA（BioelectricalImpedanceAnalysic），是一種創新性無創檢測技術。指利用人體內血液、肌肉等組織具有良好的導電性，而脂肪幾乎不導電的原理，以極其微弱、對人體完全無害的微弱電流通過人體，通過精密測量手段得到人體的電阻抗值，從而得到人體內生物組織信息的一種非侵入式生物檢測技術，該技術已在全球臨床醫學領域廣泛應用。

體重Weight

體重衡量人整體的重量（包括表皮、肌肉、脂肪、臟器等），為裸體或穿著已知重量的工作衣稱量得到的身體重量。需關注至少2次以上的數據變化，如體重偏低，需關注肌肉度和骨質量以及肌肉萎縮、骨密度下降；如體重偏高，需關注脂肪和內臟脂肪以及高血脂、高血壓、糖尿病、代謝性疾病等。

體重增加對身體風險：

體重每增加2Kg，收縮壓升高2-3mmHg，舒張壓升高1-3mmHg。

體重過輕會影響成年人體力，與免疫力地下、骨質疏鬆、貧血、抑鬱等病有關，最終影響壽命。

體重過低的原因：

1、營養不良

2、慢性疾病：胃腸道功能差，甲亢，糖尿病，腫瘤，厭食症，長期使用某些藥物。

體重過低對身體的風險：

1、導致人體儲備功能不足。

2、內分泌紊亂，血糖代謝異常。

3、抵抗力下降。

4、易患骨質疏鬆

5、易發老年痴呆

6、年輕女性易導致不孕。

7、青少年影響生長發育。

體格指數BMI

體質指數(BMI)是評估體重與身高比例的指標，反映了人體體重與身高之間的匹配關係。一般來說，最理想的BMI是22（女性21），因為在此狀態下，人體體重與身高是最匹配的，人很少會由於肥胖或消瘦的原因而生病。

體質指數(BMI)=體重(kg)÷身高(m)2。BMI每增加2個單位，患冠心病、腦溢血、腦栓塞的相對風險分別增加15.4%、6.1%、18.8%。當BMI≥28時，伴隨發生疾病的危險因素的概率可達到90%以上。

腰圍Waist

指的是經臍點(om)的腰部水平圍長，是反映脂肪總量和脂肪分布的綜合指標。腰圍是反映脂肪總量和脂肪分布的綜合指標之一。作為計算腰臀比的依據之一。一個人的身高體重指數（BMI）即便處於正常範圍內，但如果腰圍過大，過早死亡的幾率仍會大大高於常人。監測對早期預防肥胖症、糖尿病、心血管等疾病具有積極作用。

檢測方法：

（1）自然站立，雙腳開立25~30cm，雙臂自然下垂

（2）暴露腰部，找准腰圍測量位置（髂前上棘與十二肋骨中點的連線，繞過肚臍眼上一橫指的地方）

（3）將皮尺經臍上0.5-1cm（肥胖者可以選擇腰部最粗處）水平繞一周。皮尺圍繞腰部的鬆緊度應適宜（使皮膚不產生明顯的凹陷）。皮尺上「0」點相交的數值即為測量值。

（4）在自然呼氣末記錄數據，以厘米（cm）為單位，保留小數點後一位。

腰圍增加的風險：

1、腰圍每增加5cm血糖增加0.14mmol/L，相當於70g葡萄糖。

2、腰圍在正常範圍的基礎上每增加1cm，男性患糖尿病的幾率增加6%、女性則增加4%，患高血壓風險增加22.5%。

3、腰圍高於正常範圍者中風後的機體恢復能力明顯降低。腰圍高於正常範圍者心絞痛和猝死的發生率較腰圍正常者增高4倍。

4、腰圍超出正常範圍者腦出血的發生率是正常者的3.6倍，

腦梗塞的發生率比正常高3-5倍。

腰臀比WHR

腰臀比指腰圍與臀圍的比值，是判定中心性肥胖的重要指標。累積在腰部的脂肪，比大腿和臀部脂肪對健康的影響更大。腰部脂肪會破壞胰島素系統，而且腰部脂肪的新陳代謝相當快，還會產生不同的激素，導致糖尿病、高血壓、高血脂等病症。

體脂肪率TBF

體內的脂肪簡稱為體脂肪，體脂肪率是指脂肪含量佔總體重的百分比。體脂肪率不僅關係到體重和體型，而且對人體健康有著重大意義，體脂肪過高或過低都意味著更容易患上慢性疾病。體脂肪過多，可導致體型虛胖、腰圍增大，局部囤積過多脂肪會壓迫神經，引起頸項疼痛、頭暈、心慌；此外，易造成心肌梗塞、糖尿病、高血壓、癌症等諸多病症。

體脂率異常危機:

1、體脂肪過多，可導致體型虛胖、腰圍增大，局部囤積過多脂肪會壓迫神經，引起頸項疼痛、頭暈、心慌；此外，易造成心肌梗塞、糖尿病、高血壓、癌症等諸多病症。

2、體脂肪率低於正常值，則意味著著體型消瘦。內臟器官失去保護、支撐。

3、指數超標，說明機體脂代謝增高（異常），肥胖患者患冠心病的風險是正常人的1-2倍，患高血壓的風險是正常人的1.2-2.5倍。

4、體脂率低於正常值機體營養不良，各內臟器官和肌肉組織得不到充足的營養出現功能低下，長期可能會出現疾病風險。

體脂異常引起的疾病：

1、向心性肥胖，體代謝紊亂的表現，

2、長期內臟脂肪會導致高血脂、脂肪肝、心腦血管疾病、身體器官機能下降等併發症。

內脂指數VFI

內臟脂肪是人體脂肪中的一種，是囤積在身體中斷胸腔腹腔當中，覆蓋在心、肺、肝臟以及大小腸中的脂肪。內臟脂肪率1相當於內臟脂肪面積10平方公分。

內臟脂肪對人的內臟起著支撐、穩定和保護的作用。內臟脂肪過多是身體代謝紊亂的表現，長期內臟脂肪過多會導致高血脂、心腦血管疾病、身體器官機能下降等。內臟脂肪過高，在醫學上稱之為向心性肥胖，極易引起代謝疾病、心腦血管疾病、癌症等。

內臟脂肪率過高的危害：

1、指數每超出正常範圍一個單位，相當於10cm²（手掌的1/5大小）的脂肪附著在內臟表面。相當於心臟被脂肪覆蓋了1/5.

2、內脂在6-9，代謝綜合征（三大營養物質代謝異常引起的一系列疾病，如肥胖、三高）患病率增加5.3%；

3、內脂超過9，並且每增加2單位，代謝綜合征患病率增加32.9%。

內臟脂肪過高引起的疾病：

向心性肥胖，體代謝紊亂的表現，

長期內臟脂肪會導致高血脂、脂肪肝、心腦血管疾病、身體器官機能下降等併發症。

壓迫其他組織器官的正常功能。

體水分率TBW

體水分是指人體體內的細胞內外液的水分，而非代謝水分，包括血液、淋巴液、細胞外液等，亦稱為體液。人體體液總量約為體重的50~70%。

水是生命的源泉，人體內營養成分的運輸、各種代謝的進行以及代謝產物的排泄全都離不開水。體液作為載體，為細胞提供營養和氧氣，並將二氧化碳和體內垃圾溶於水送達人體各器官進行化學處理。

意義：這些體液占體重的50～60%，作為載體它為細胞提供營養和氧氣，並將二氧化碳和體內垃圾溶在水裡送到人體的各器官進行化學處理。體水分和肌肉量有著極其密切的關係，因為肌肉中含大量水分(大概70%），這項指標能夠反應減重的方式是否正確，如果體水分率下降，不但有損健康，更會令體脂肪率上升。

危害：體內水分過少會導致皮膚過度乾燥，體力不支，虛弱。當人體失水占體重10%，就會產生脫水，出現少尿、心跳加快、血壓下降；當失水占體重的15~20%時，就會造成電解質紊亂，危及生命。

骨骼肌率SM

骨骼肌率是骨骼肌（四肢骨骼肌、內臟肌和皮膚肌）占人體重量的百分比。

肌肉量減少，人體基礎代謝降低，熱量消耗隨之降低，攝入的過多熱量便轉化成脂肪堆積於體內，形成肥胖；肌肉還是心臟的可靠助手，肌肉衰弱必然累及心臟，成為誘發心血管病「幫凶「；腿肌力量下降，上下台階就會吃力，走路步幅會變小，而且容易被絆倒，易遭受骨折之苦；肌肉量減少，關節負荷會增加，產生關節痛，進而出現姿勢變形而導致腰痛。

肌肉量減少的因素：

1、運動量減少，沒有達到有氧訓練。

2、蛋白質補充不足。

3、疾病引起

4、遺傳因素

5、藥物因素

肌肉量增加的好處：

肌肉越多，基礎代謝越高。保證一定量的肌肉可以促進代謝，減少與代謝紊亂有關的慢性病，如動脈粥樣硬化、血脂異常、糖尿病等的患病風險。同樣的原因，肌肉含量較充足的人，衰老的速度也較慢，會比同齡的缺乏肌肉的人更年輕一些。

肌肉能強壯骨骼和關節。缺少肌肉力量，會增加關節變形、關節退行性變、腰間盤膨出或脫出等問題的發生風險。關節的慢性病變以及衰老，與肌肉力量不足有直接關。

肌肉量減少危害：

肌肉量減少，人體基礎代謝降低，熱量消耗隨之降低，攝入的過多熱量便轉化成脂肪堆積於體內，形成肥胖；

肌肉還是心臟的可靠助手，肌肉衰弱必然累及心臟，成為誘發心血管病「幫凶「；

腿肌力量下降，上下台階就會吃力，走路步幅會變小，而且容易被絆倒，易遭受骨折之苦；

肌肉量減少，關節負荷會增加，產生關節痛。

低肌肉型的人，不論體重是否超重，大多都患有肥胖症，易引起蛋白質嚴重缺乏，出現營養不良、水分缺失、免疫力低下，機體乏力、運動能力減退、肌肉萎縮。

骨礦含量BMC

骨礦含量是指單位體積內，骨組織骨礦物質（鈣、磷等）和骨基質（骨膠原、蛋白質、無機鹽等）。

骨礦含量是構成人體骨骼的重要因素，是支撐身體的重要支架。其含量高低直接影響到骨骼的強度和密度。骨礦含量過低，導致骨骼脆化、骨質疏鬆、易骨折；骨礦含量過高一般見於大量進行鈣質補充，導致骨骼內礦物質含量增高，超出人體接受範圍，可能會引起骨質異常增生。

基礎代謝BMR

基礎代謝(basal metabolism, BM)是指人體維持生命所需要的最低能量需要量，是在人體在清醒而又極端安靜的狀態下，不受肌肉活動、環境溫度、食物及精神緊張等影響時的能量代謝率。
