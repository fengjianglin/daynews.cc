國內卧推實力標準，看看你能在什麼位置

2019-12-11 774條評論 4,171次閱讀 1062人點贊

提起卧推，喜歡健身的朋友應該都知道，它是訓練胸大肌，胸小肌，肱三頭肌，肘肌等上肢肌肉的一項運動。尤其是針對胸部肌肉群有明顯的效果，是如俯卧撐等運動沒法比的。

對於卧推，很多人都能做幾個，可你的實力，在國內到底在什麼位置呢？

第一梯隊，20KG。空杠卧推，這也是大多數新手的階段。由於新手對杠鈴控制差，加上肌肉感力量不足，很容易出現身體左右擺動，杠鈴歪斜。

第二梯隊，30KG。度過了新手空杠期，可以適當加重了，一邊5KG。來試試吧，注意不要搖晃杠鈴。

第三梯隊，40KG。這時候你已經小有成就了，胸肌和手臂會有小小的變化，繼續加油啊。

第四梯隊，50KG。到了這，恭喜你已經超過國內男性卧推的平均水平了。（有數據，國內男性卧推平均45KG）

第五梯隊，60KG。對於這個重量，一般沒受過訓練的人是很難達到的，你也已傲視普通人了。

第六梯隊，70KG。此時你的胸肌比以前有了更加明顯的變化，在健身房可以自豪的找人幫忙協助了。

第7梯隊，80KG。能推起這個重量的人，即使穿著普通的T恤，走到街上也能一眼看出強壯的上肢肌肉

第8梯隊，90KG。這個重量，即使在健身房也是大神的存在，會不會在訓練時，總有可愛的美女過來請教呢。

第9梯隊，140KG。這個重量，已經是正常男人體重的兩倍，能正常標準做組，就已經超過了國內百分之90以上的男性，這時候頂級大門正向你敞開。

第10梯隊，200KG。這個重量，在普通的健身房找不到了，已經屬於專業運動員的級別，恭喜你進入的國內頂級行列，可以試著參加比賽了。

最後是大神的級別！

國內最紀錄，大力士岳陽-260KG

世界最高紀錄，Kirill Sarychev(俄羅斯）-335KG

​

看完後請大家對號入座，看看自己在什麼位置吧。
