倆娃寶媽不運動僅吃生酮飲食，堅持11個月會有何變化？

2019-12-13 1,294條評論 4,092次閱讀 3126人點贊

何為生酮飲食，在我們的觀念中就是吃肉減肥，事實上這樣的說法並不正確，生酮飲食並不是僅僅吃肉就行了。看起來高大上的減肥吃法，其實核心概念還是在追求飽腹感的同時，減少過多碳水的攝入，來達到熱量的負平衡的目的，當然你必須要做到的一點就是營養均衡。

筆者就是忠實米飯愛好者，曾經嘗試過生酮飲食，最初頭會比較暈，後期適應之後力量下降不少，後面就捨棄了，所以如果你是正在哺乳的媽媽、體質虛弱者，健身者就不必採用了。那生酮飲食到底減肥效果如何？堅持11個月會有何變化？看看這位倆娃寶媽的經驗就知道了。

她的名字叫Mecca，是一名老師，也是倆娃寶媽，最初的體重194斤，這是最初的狀態。她的減肥方案就是生酮，多喝水，不做任何的健身運動。

她這樣的飲食只是減掉了大量碳水，但不得不說營養做的真的很不錯，這的確是值得很多人學習的方式，不管用生酮、間歇禁食或者是其他的，營養需要保證，蛋白質、脂肪都是必須的。

這是生酮第3個月之後狀態，沒有健身運動減肥速度並不快，第一個減掉8斤，第二個月減掉4.5斤，第三個月6.6斤。她認為生酮最大的好處，可能就是享受美食，不必忌諱油水。

第1個月和第6個月的對比圖，她總共減掉了28斤，平均一個減掉不到5斤，講真單靠飲食減肥速度的確不快，但是卻比較輕鬆。

第1個月和第8個月的對比圖，此時她已減掉了35.4斤，肚子上的變化真的很明顯，她自嘲道以前看起來像水桶，如今終於能看到一點腰了。

最終她用11個月減掉了52斤，成功從不是在一夜發生，保持耐心、信心，堅持做下去，你一定會看到結果的。

她的確靠生酮飲食取得了很棒的減肥效果，雖然速度不快。那此時問題來了，生酮減肥健康嗎？

網路上對生酮的爭議真的很多，但有一點可以肯定的是，沒有哪種減肥飲食是適合每個人的，是絕對健康的。不少文獻認為長期生酮可能會增加低血糖、便秘、脫髮甚至是腎結石的風險。畢竟我們吃了那麼久的碳水，不是說丟就丟掉的。

所以生酮飲食在使用前，需要諮詢相關的營養師，不要盲從。另外話說回來了，即使它的確可能增加一些健康問題風險，卻總比那些追求節食，速瘦的減肥餐靠譜多了，那些追求一個月速瘦15/20斤的食譜，每天清湯寡水，青菜黃瓜度日的減肥餐，單營養和生酮比起來就落於下風了。
