寶媽是大胖子，就會生下個已“10歲”的孩子？科技是這樣解釋的

2019-12-15 303條評論 2,007次閱讀 1062人點贊

10個女人10喊著減肥。但8個「越減越肥」，1個維持原狀，1個或許能瘦下來。

我屬於這8個中的一個。

為啥這麼多人減肥失敗？因為你得跟人性、大環境作鬥爭。

真不好減的，好吧。

我們人類進化幾千年，物質豐富才是近200百年的事，在漫長的歲月里，能有口飽飯就不錯了。所以，有好吃的，減肥？不可能。先吃飽！對熱量的渴求已經刻在我們基因里了。

從外在來說，食物豐富，各種美食、各種飯店，你當人家擺設呀！都在想辦法讓你來吃！

減不掉不可恥，是人性。可是，人性歸人性，咱真的不能超重。

之前看過一檔減肥節目，去參加節目的是一群十八九歲的超重少年，醫生對他們檢測發現，他們的身體相當於五十多歲的老年人。肥胖，讓他們身體器官老化了。

我的天吶，真嚇人。

於個人身體而言，我們不能超重。

小編也多次提醒孕婦，要控制。孕期吃太多，容易影響後期生產，而且容易引起孕期妊娠糖尿病，也不利於產後減肥。

然而，還有一項危害，說出來觸目驚心。

肥胖女性生的孩子，生理年齡自出娘胎始，就比其他孩子老。

我來科普一下。

人體細胞里，有「端粒」，這個端粒呢，很神奇。每一次細胞分裂，它就少一段（當然只是非常小的一段）。所以，端粒的長度，也能在一定程度上反映人的衰老程度。

研究人員發現，孕前、孕期超重的媽媽，生出的孩子，平均端粒比正常孩子縮短5.5%，說通俗點兒就是，可能比正常孩子生理年齡多了5-10年。

研究人員沒有確定這一現象的原因，但他們推測可能是多度的脂肪堆積，引起了炎症，炎症傷害了胎兒細胞，從而縮短了端粒。

無論什麼原因，這個研究都給了我們警示：一定要控制體重。

有的人本身體重大，肌肉成分比較多。骨架大。我們說要控制體重，主要是控制自己的體質數。也就是常說的BMI指數。正常人的BMI指數是18.5-25之間。我們要做的就是多鍛煉，把自己的體質數控制在這個範圍內。

怎麼計算體質數呢？這是公式。

體質數=體重kg/（身高的平方）

比如，你是50kg，身高1.6米，那麼，你的體質數就是：50/（1.6^）=19.5

肥胖是非常危險的，不論是孕婦還是普通人群，都應該注意。減肥、鍛煉不易，常駐我心，大家一起努力哦！
