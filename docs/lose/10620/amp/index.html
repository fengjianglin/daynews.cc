170斤的小胖變肌肉男，用了4年時間，以施瓦辛格為偶像

天天要聞
2019-12-07

都說胖子是潛力股，但是真正能夠實現逆襲的胖子並不多，大多數胖子是一直胖下去。胖子減肥不成功，歸結起來主要有兩個方面原因，一個是動力不足，遇到點挫折就想放棄；二是採用的減肥方法不夠科學。印度小伙帕桑覺得，減肥一定要明確目標，找一個偶像，並且為之奮鬥，成功只是時間問題。帕桑用行動證明了自己，他花了4年時間，從一個身材微胖的小伙變成了肌肉型男。

施瓦辛格的身材和電影形象一直激勵著帕桑去行動，雖然他沒有真正變成施瓦辛格，但是他的身材距離施瓦辛格並不遙遠。帕桑把施瓦辛格當成自己的偶像，他覺得自己的成功轉變，是因為心中一直有施瓦辛格這樣一個完美的肌肉男神形象。有目標的行動，會更有力量。

練出完美的倒三角身材，粗壯的手臂，做出施瓦辛格曾經的招牌動作，是帕桑一直想要完成的夢想。他現在的身材距離施瓦辛格還有距離，但是他相信，只要堅持不懈努力，一直會離偶像更近。他目前已經是健美運動員，在印度也算小有名氣，並在網路開設課程。

很多人覺得練身材太苦了，特別是肌肉感十足的身材。一開始，帕桑也是這麼覺得，堅持一年以後，他才發現，曾經的美食對他並沒有那麼大的吸引力，而且他想要享受這些美食也不是不可以。他每周會安排一次欺騙餐，盡情地吃一些大家眼中的高熱量食物。

帕桑的特別肌肉線條特別出眾，經常有網友詢問他如何訓練，他表示健身動作其實就那些，自己並沒有什麼秘訣，但是他表示自己偏愛於頸後下拉。這個動作不適合所有人，盲目借鑒還容易受傷。但是，如果你的關節活動度好，肌肉張力也都比較好，那做頸後下拉能更好的刺激背部，可以讓背部細節更完美。

帕桑經常分享自己的健身日常，但是他不希望網友盲目模仿，因為有些動作不適合新手，需要有一定的身體基礎才能完成，就比如他經常練的頸後下拉這個動作。他希望網友看到的是他對健身的精神和執著的態度，日復一日地堅持才是讓你轉變的關鍵。
