注意了！這些行為會讓你在球場討人厭

2019-12-10 297條評論 3,676次閱讀 1370人點贊

在打羽毛球時，

你的一些無意之舉，

可能會讓人無比生厭，

究竟都是什麼行為呢？

今天來統計一下，

以後打羽毛球的時候，

要注意哦！

NO1.抽煙

公共場合本來就禁止吸煙，

何況是在球館裡，

要抽煙可以出去抽啊，

最煩的是坐在場邊吞雲吐霧的，

運動時人的呼吸頻率加快，

結果人吸的全是你的的二手煙，

爽了你一人，嗆了一球館

還怎麼讓人愉快的打球？

NO2. 隨地吐痰

你吐痰在球場上，

別人很容易踩到，

十分噁心不說，

也很容易滑到受傷。

NO3.霸著場地不下來

大家都是來打球的，

占著場子不下來從頭打到尾，

害的別人只能在場下乾瞪眼，

即使你體力好也不是這樣表現的，

也該發揮下禮讓的精神，

別上了球場就不肯下來了。

NO4.有事沒事打電話

一場球下來接好幾個電話，

極大的影響了球友的興緻，

別人打得正在興頭上，

你電話接個沒完還要等你講完再打，

讓球友沒了心情，

既然出來打球了，

除非真的是十分要緊的事，

其他的打完球再說嘛！

NO5.球場光膀子

雖然大熱天的打球，

很容易汗流浹背，

但是你光著個膀子，

看起來不太雅觀不說，

滿身的汗水灑到場地上，

也會讓人不小心滑倒受傷！

NO6.對場上隊友指指點點

事後諸葛亮，事前豬一樣，

自己打的不怎樣卻總質疑隊友

這種行為，相當的讓人討厭，

畢竟，出來打球都是為了開心，

誰也不願意被人指揮來指揮去，

除非是別人主動向你虛心請教

NO7.隨便找借口

失誤就是失誤，

沒發揮好就是沒發揮好，

只要認真盡全力了，

大家都會看在眼裡，

不會去說你什麼，

但是如果你連這些都不承認，

出現一點問題就開始找借口，

什麼地板太滑，

什麼光線不好，

這會讓球友相當鄙視你。

NO8.打球不認真

既然站在球場上了，

就要集中精神打球，

左顧右盼心不在焉的，

也是對對手的不尊重。

NO9. 場地內隨便走動

這種人真的超級討厭，

人家在場上打球正起勁，

他旁若無人的走來走去，

萬一被打著了人家還得賠醫藥費，

可人家背後也沒長眼睛，

拍子上也沒有後視鏡啊！

NO10.一毛不拔的人

要是俱樂部的活動倒也罷了，

場地和球都有供應，

可要是幾個認識人約在一起打球，

還出現這樣的情況就不太厚道了，

帶個拍子就來打球了，

球快打爛了非等別人換才罷休，

別人出球你總得出點場地費水費吧！

轉給你的球友們，

以後打羽毛球的時候，

一定要注意哦！
