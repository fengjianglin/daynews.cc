太極拳的招熟、懂勁，而階及神明

2019-12-21 672條評論 1,751次閱讀 685人點贊

「招熟」「懂勁」「神明」是在正確方法的基礎上，一步接一步的功夫。

太極拳在練習方式及對敵方法上，與一般拳術不同，要求不恃力迎敵、取敵，而以巧勁力求以弱勝強，因而強調牽動四兩撥千斤。基於此，拳術要求並必須做到「懂勁」。

太極拳在行功期間，講究諸處皆圓，觸處成圓，非圓即弧。勁力的圓轉融和，主要靠行拳盤架來體會，所以必須熟練掌握每招每式勁路，此謂「招熟」。

「招熟」只是「懂勁」的前提。從招熟漸漸懂得自己的勁力走向，與對輕重緩急的把握，如此方能去體悟他人的勁。行家一伸手，便知有沒有，自己的勁走得圓活無礙，則能從對方舉手投足之間，察斷對方的勁力是否圓活松整，搭手間即能洞悉其大小與走向。所以，「招熟」是先懂自己的勁，繼而懂得別人的勁。

「懂勁」後，通過推手來練習走化、擒拿、發放與解脫，在不丟不頂中，逐步走向以弱勝強。在此基礎上，能認識知道所學招式只是「正招」，由此千變萬化而合拳理，則是進一步的招熟。到此境界，則是真懂勁。

此後，通過加深練習，能在任何時候做到不丟不頂；化發同舉，一體同步，隨心所欲地隨勢變化而敵莫能測，此謂因敵變化示神奇。正如《孫子兵法》所云：「故形兵之極，至於無形。無形，則深間不能窺，智者不能謀。」

到此境界，勁力如水，敵方找不到可以著力之處，我方則隨心所欲而不逾矩。兵法形容「夫兵形象水，水行避高而走下，兵勝避實而擊虛，水因地而制行，兵因敵而制勝。兵無成勢，無恆形，能與敵變化而取勝者，謂之神。」於拳之體用上，則是「遭著何處何處擊，我也不知玄又玄」，此即「階及神明」矣。

當代人生活節奏快，面對的誘惑多，很難像前人那樣下功夫，也就很難達到前人的境界。所以，練至初步「招熟」容易，真「懂勁」就很難了，委實無多。這也無妨！倘忘卻了祖論尚有「動急則急應，動緩則緩隨」之要求，只把拳定死為摸魚態，不知發勁如放箭，還口口聲聲攻擊其他太極拳態，這……
