2019.12.07手術後昏迷第102天

2019-12-07 130條評論 1,165次閱讀 136人點贊

今天沒發燒，完全側躺，令左肺不受壓力；去甲腎上腺素1；呼吸機cpap模式。這幾天頭和手有些浮腫，由於側卧臉都變形的很難看！

心裡感受到再也無法挽救爸爸，心情異常的煩躁、難受！還是忍不住罵媽媽。然後大家都更難受！

看著別人或三三兩兩，或成雙入對的吃飯、散步，健身，好羨慕這種我原來也擁有的無憂無慮自由自在哪怕是平凡無奇的生活。我原來也是這樣啊，只是因為一個錯誤的決定完全的打破了。

明天是一年一度的丁蜀跑步節，往年我會約上三兩跑友去感受跑友積極樂觀和廣闊的大自然，可現在我完全沒有勇氣和精神去參與到其中。無法面對正常生活！上周開始也本想去健身房鍛煉，哪怕只是獨自跑跑步，但依舊不能鼓足勇氣和精神，僅僅是每天去健身房洗個澡，算是外出參與了生活。

感受到生活，能夠平淡、無焦慮，無痛苦，安心踏實就是最大的幸福！

烏雲背後有陽光，我真希望我內心能夠有陽光！我只想說：爸爸，對不起！爸爸，堅持，等待烏雲散去！
