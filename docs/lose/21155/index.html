高效多能，趣味性強，而且生活中不可缺少的一個動作“農夫行走”

2019-12-19 1,643條評論 2,697次閱讀 472人點贊

大部分人健身的目的無非是保持健康和打造身材，健身動作雖然成千上萬，但是人們往往忽略了一些功能性極強的訓練動作，而「農夫行走」就是一個漸漸被大家遺忘的經典動作，顧名思義，就是要像農夫一樣，拿著兩個重重的東西行走著，也是人類最原始最簡單的一個動作，看似動作簡單，但卻是一個高效多能，趣味性極強，而且還是生活中不可缺少的一個動作，可是現在也是出場率極地的一個訓練動作，偶爾只能在大力士和專業的運動員訓練上才能看得到。

能夠鍛煉到全身大部分的肌肉

「農夫行走」不受場地的限制，可以拿起身邊任何的重物進行訓練，它可以是啞鈴，杠鈴，鈴壺，水桶等等，通過負重持續的行走在5分鐘以上或是50米左右的距離，所以對肌肉耐力和力量都有較好的提高，看似簡單的動作，它可以帶動全身各部位的肌肉群，包括背部、肩部、手臂、握力、而且它可以鍛煉你的核心力量，甚至是提高身體的穩定性。

加快燃燒脂肪的速度

大重量的農夫行走強度非常的大，短短一段距離，就能夠耗盡我們的體能，而且農夫行走訓練屬於一種複合型的力量訓練，一般拿起的重量都比較重，通過多關節多肌肉的參與，能夠有效地提高基礎代謝，從而提高脂肪燃燒的速度。

糾正不良姿勢

上班族總是會感到腰酸背痛，或者由於長時間的工作導致體態不良，通過農夫行走正確的動作，能夠收緊核心，腰部和肩膀也會始終持續發力，原本姿勢不對的部位就能被很好的拉伸到，所以能夠很好的改善體態。

「農夫行走」正確的訓練動作

保持腰背部的挺直，收緊身體的核心、慢慢屈膝蹲下，雙手牢固地握住啞鈴或負重物品後，上半身要保持背部繃緊挺直，以深蹲起的形式，慢慢先伸膝再伸髖站直身體，將雙臂貼緊身體兩側，不要讓整個手臂懸空，這樣能大大降低阻力臂，保持腰背部的挺直，肩胛骨打開下沉，頭部保持正視前方，核心收緊讓脊椎處於中立位置，控制好平衡後，向前移動，走完預定距離後，屈膝下蹲，放下器械。

為了提高訓練難度，對肌肉有更好的刺激，還有許多變式包括Z字農夫行走和過頂農夫行走。

「農夫行走」的注意事項

1.對重量和時長要循序漸進，在訓練前做好熱身訓練，讓身體有一個適應的過程，避免直接上大重量，造成運動損傷。

2.拉起重物時要像做硬拉訓練一樣拉起，放下重物時，要屈膝，不要弓腰，以降低腰背部承受的壓力。

3.在運動過程中，保持身體的穩定性，勻速且控制均勻的步幅，過大的步幅會增加腰部的受力，容易造成運動損傷。
