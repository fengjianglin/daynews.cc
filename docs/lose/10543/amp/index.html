每一個胖子都是潛力股，400斤的胖女孩5年後，讓眾人都羨慕

天天要聞
2019-12-10

這個胖女孩從小就比非常喜歡吃甜食，在學校讀書的時候，她由於太過於肥胖，常常被班上的學生取笑，她甚至都懼怕去上學，一放假就往家跑。肥胖不僅導致她的學業一瀉千里，而且身邊的好友也少的可憐。可是天上派下一個男孩子，不在乎她的身體，並且向她求婚，最終兩人成婚了。

這位胖姑娘跟她的丈夫在一起生活的非常的幸福，她表示：很幸運可以碰見這個不厭棄她肥胖的男孩。

新娘和她的伴娘們。這位胖姑娘的好友非常少，全因為肥胖導致她非常自卑，老是下意識以為別人會嘲笑她的身體。可是她的老公一點都不在意這些，不在乎這位胖姑娘的身體，也不在乎別人的眼光。

成婚以後，這位胖姑娘不再暴飲暴食，她不再吃那些容易發胖的甜食，並且常常陪著丈夫出去散步。就這麼保持兩年以後，她的身體發生了很大的轉變。後來她開始增加了自己的活動強度，科學的飲食，身體越來越好，身材越來越苗條！

就這樣保持了五年以後，她整個人完全不一樣了，她變得很時髦。很多人都無法相信她之前是一個胖達400斤的女人。那些已經嘲笑她丈夫的人，現在都開始羨慕起他的丈夫來了。
