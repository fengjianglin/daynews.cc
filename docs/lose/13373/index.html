慘劇再發！小伙被120公斤杠鈴砸胸口，一下倒地

2019-12-11 388條評論 1,827次閱讀 725人點贊

更多精彩熱點，點擊關注⬆️

昨天，健身圈被下面1條小視頻刷屏了

小伙獨自做120KG卧推時突發意外

令人心驚不已

從視頻可以看出

小伙當時獨自卧推，在推起一次大重量時

由於脫手，意外瞬間發生，杠鈴重重地砸在他身上

隔著屏幕都令人膽寒….

彼時

他還順勢將杠鈴滾落到下腹部

安全的將杠鈴移走，而後起身準備離開

但沒走兩步，他卻轟的一聲摔落在地

趴在地上昏迷不醒，視頻到這戛然而止

看到這一幕，不少網友都驚出一身汗，大家紛紛留言：

@幾哈跨世：太TM嚇人了，一定要找人保護啊！！！

@你先上了那輛離開世間的車：我初一有個同學就是這樣被杠鈴砸到，很不幸，去世了！

更多的網友則關心這位小哥最後情況怎樣了。

而網上關於這件事也是議論紛紛，有人說小哥被砸身亡。

一時間傳的沸沸揚揚，直到後面有知情人表示：小哥被砸之後，沒出現大問題，下午還在健身房繼續訓練。

並且貼出了群聊記錄。

不得不佩服小哥強壯的身體，慶幸小哥平安無事的同時，小編也想告誡大家要引以為鑒，在獨自訓練時，一定要量力而行，不要盲目自信。

特別是在大重量卧推的時候，一定要找靠譜的肌友保護，一個不慎就可能引發事故。事實上，根據數據統計，因為力量訓練而死亡的事故中60%都是來自卧推。

舉幾個著名的案例：

案例一

34歲的來自俄羅斯的力量舉運動員Igor Golushkin在力量舉比賽中，用危險的「空握法」卧推185公斤時，杠鈴脫手砸中胸部，導致嚴重的心臟出血，損壞了他的橫膈膜，砸斷了他的肋骨，並撕裂了他的腹腔。Igor Golushkin賽後死於心臟震蕩，也就是胸壁損傷引發的猝死。

案例二

2012年，23歲的卧推世界紀錄保持者MikhailAlgash在大重量訓練後感覺不適，後被送往醫院救治數日後死亡，醫生診斷死因為肝破裂和失血過多。儘管醫院為他進行了緊急輸血，還是沒能拯救他的生命。
