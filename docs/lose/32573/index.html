對業餘來說，較慢的球速，其實在更大程度上限制了高手的水平

2019-12-24 268條評論 1,193次閱讀 168人點贊

說一個人羽毛球水平高，其實是包含了很多方面，比如腳步快、動作一致性好等等，其中，出球速度快是其中一個比較顯著的特點。一般來說，高手的出球是會比較快的，快到你來不及反應，不會存在一個高手打出的球慢悠悠還能把你搞定（如果真是這樣，只能說，你的步伐太慢了…..）。

以出球快作為高手的一個指標，我們就可以發現，如果用慢速的球，其實是變相限制了高手的球速，造成了其出球的威脅性下降。但是，也會有人說，球速變慢了，對雙方的影響都一樣啊。乍一聽，好像很有道理，你我雙方的球速都慢了，影響一致啊。仔細分析下，就不是這麼回事了。

羽毛球的球速是所有球類運動中最快的，傅海峰創造的340公里/小時的殺球速度是球速的世界之最。這麼快的球速其實已經超過了人類反應速度的上限，也就是說，球飛過來，你還沒來得及做動作就已經落地了。業餘球友的球速比較慢，有可能高手的球速就會剛剛超過了對手的反應速度，處於反應的臨界點上。表現出來的就是對手邁步出去夠不著球（這個比較少），或者是能夠著球卻無法發力和精確地控制自己的技術動作，從而失誤或者回球不到位。這時候，一旦選用慢速的球，就有可能對手從夠不著球到能夠著球，從無法精確控制技術動作到可以精確控制。這兩者之間的差別是巨大的，決定性的。對高手來說，這可能是非常鬱悶的一個局面。一旦球速下降了一個層次，對手就可能可以接起來很多原來失勢甚至直接丟分的來球，而對手的球速本來就在那個臨界點以下，再慢那麼一點點，對高手而言區別並不大。

這樣比較下來，其實高手在球速上面的損失就遠大於對手，比賽的勝負就要靠其它方面的因素了。如果對手在步速和體能方面佔優勢的話，平時能贏球的高手就只能鬱悶的敗下陣了（嗯，最近天冷，球慢，我就這樣輸給了平時隨便拿捏的對手，哭一個先……）。來源中羽：球齡七年半
