一斤豆芽半根胡蘿蔔，5分鐘就能上桌，美容減肥，鮮嫩營養又入味

2019-12-17 1,074條評論 2,833次閱讀 1179人點贊

一道非常美味的家常食材，維生素C的含量很多，每個家庭的餐桌上，都會看到這菜的身影，它就是又鮮又嫩的綠豆芽，菜市場賣8毛錢一斤，是平衡油膩的最佳首選，常吃豆芽的朋友都知道，這是由綠豆或者黃豆發芽而成的，非常的乾淨衛生，沒有任何添加劑，含多種維生素，營養豐富，清淡爽口。

綠豆在發芽的過程中，維生素類物質的含量會大大增加，其中，維生素B2增加了2～4倍，胡蘿蔔素增加了2～3倍，維生素C增加了6%，這也是我愛做的一道菜，涼拌清爽可口，加入了胡蘿蔔絲、黃瓜絲，色彩繽紛，熱量非常低，適合想減肥的朋友，如果你這幾天的胃口大打折扣，看到飯菜就發愁，那應試試這道清爽的涼拌菜，冬天平衡油膩的一股小清風。

食材：綠豆芽，胡蘿蔔，黃瓜，小蔥，香菜

調味：鹽，雞精，米醋，食用油

1、準備一塊錢的綠豆芽，放入水中浸泡一會，沖洗乾淨，撈出控干水分。

2、準備半根胡蘿蔔，洗凈先切片，再切成細絲，再準備半根黃瓜，用同樣的方式，切成細絲。

3、鍋中把水燒開，下入綠豆芽和胡蘿蔔，焯燙50秒鐘撈出，這裡解釋一下，綠豆芽非常容易熟，時間足夠了，如果是黃豆芽之類的，要延長時間。

4、撈出綠豆芽和胡蘿蔔，過一遍涼白開，下一步控干水分備用。

5、控干水分後，把黃瓜絲，胡蘿蔔和綠豆芽裝在一起，掰幾個干辣椒，大蒜拍扁切末，切一些小蔥、香菜，加入適量的鹽、雞精、米醋。

6、全部添加好，淋入熱油，這樣更能激發出香味，蓋上蓋子再燜2分鐘，時間到了，翻拌均勻，裝盤開吃。

7、一道美味的涼拌豆芽就做好了，清脆爽口，解膩開胃，看著就很有食慾，還能當作一道下酒菜，喜歡的朋友，一定要試一試。

菜譜小技巧

焯水的時候一定要注意，時間別超過1分鐘，會影響口感，調料的用量無法詳細寫出來，別上來就放許多，先品嘗一下，不夠再慢慢增加，不喜歡吃辣的可不放辣椒。
