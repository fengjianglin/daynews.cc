只有現代人才減肥？其實在楚靈王時期，就連男人也在“跟風”

2019-12-11 175條評論 2,275次閱讀 729人點贊

生命不息、運動不止！在當今這個普遍愛好健身和減肥的時代，運動也成了時尚的標誌之一。雖然大多數人對於運動和減肥都抱著支持的態度，但是真要參與到這個活動中來，卻是極其考驗人的意志的。畢竟，愛好吃喝享樂是絕大多數人的本能，逃避流汗和運動才是常態。然而，很多人不知道的是，減肥在中國可謂是一項歷史悠久的活動。

因為早在兩千多年前的春秋戰國時期，減肥就已經風靡楚國。說起這場全國性的減肥風潮，就得從楚國的君主楚靈王身上說起。楚靈王：羋姓，熊氏，初名圍，是楚共王的次子，殺了侄兒楚郟敖後自立，即王位之後改名虔，公元前541年，自立為楚國國君。楚靈王是春秋時代有名的「個性」君王，他的「個性」之一就是「好細腰」。不過，誰讓人家有這樣的資本呢？

根據《墨子•兼愛》記載：昔者楚靈王好士細腰，故靈王之臣皆以一飯為節，脅息然後帶，扶牆然後起。比期年，朝有黧黑之色。這裡的「士」，指的就是朝堂上的大臣。也就是說，楚王的「好細腰」，已經不僅僅是對後宮女子的偏愛，同樣也體現在了對自己的大臣身上。這裡也就不得不為那些為了「細腰」，餓得已經得扶牆走路的大臣們感嘆了，明明是干著靠才華吃飯的工作，卻要過著靠身材吃飯的生活！（不過，這種不健康的減肥方式，卻不值得提倡，畢竟，減肥也得以身體健康為前提條件）。

根據記載，當時楚國朝堂上的大臣為了迎合君主的喜好，大多都餓得面黃肌瘦，因為細腰的臣子更容易得到君主的賞識，為了讓君主高看自己一眼，這些文人雅士也是拼了命的減肥。這裡對於減肥的最直接做法就是少吃，不得不說，楚靈王的這個愛好，可真是給楚國省糧食了。正如《尹文子》和《管子》里所描述的：「楚庄愛細腰，一國皆有飢色」， 「夫楚王好細腰，而美人省食」。由此可見，古代追求身材管理，也是不分男女的！

也許是受到楚靈王的啟發，後世不少的君主也發現了：不管是朝中的文臣雅士、還是後宮的嬌艷美人，都是瘦些更好看。於是，這減肥的風氣就持續了幾千年，只不過，後世君主的愛好更多的體現在了後宮的美人身上，朝中官員算是解放了不少。西漢時期，就有不少君主對瘦美人痴迷不已。比如說漢武帝時期，一個頗受漢武帝寵愛的宮女麗娟，據說這麗娟瘦的得用衣帶把她系在錦被上，要不然就會被風吹走，從這裡足以看出這姑娘得瘦成啥樣！

有了前人的打樣，後面的君主也一個個地跟著上。漢成帝時期的趙飛燕，可謂是歷史上最有名的瘦美人吧！這趙飛燕得多瘦呢？根據記載，趙飛燕身材纖細到可以在人的手心裡起舞。而且，趙飛燕上馬的時候，由於體重太輕，馬都感覺不到自己背上騎人了！到了魏晉南北朝時期，凡是跟名士沾邊的，絕對都是「仙風道骨」的范兒，也就是說，大多數的名士也是瘦骨嶙峋，因為只有這樣，才會出現風吹起衣服時、仙氣飄飄的效果。

這愛瘦的風氣，到了唐朝時期，得到了一定的控制。畢竟作為四大美人之一的楊玉環，就是一個豐盈的美人。而且，唐朝時期，時人大多認為美女必備的條件之一就得身材豐盈，這一點從唐朝的仕女圖上就可以得到印證。不過，減肥運動雖然在唐朝時期斷了一段時間，但是很快就還是得到了認可和追捧。時代一直在變，但減肥的現象卻從來都沒有斷絕。就算是一直到現在，不少人都還是有著減肥的心態。畢竟，合理地減肥不止可以使人身體輕盈，還可以使人保持身體健康！

參考文獻：《墨子•兼愛》、《尹文子》、《管子》、《漢史》
