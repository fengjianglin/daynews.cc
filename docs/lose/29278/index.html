骨科大夫提醒運動愛好者，警惕你的後跟痛

2019-12-21 473條評論 2,405次閱讀 1376人點贊

生活在現代都市的人們，總有被四方湧來的壓力逼得喘不過氣來的時候，有些人選擇奔跑來宣洩壓力。隨著「半馬」、「全馬」、登山等運動的湧入，走走族、跑跑族對運動的「狂熱」讓他們忽略了身體的抗爭。據不完全統計，走走族、跑跑族最常見的抱怨之一就是後跟痛，而跟腱炎就是其罪魁禍首。

行走、跑步或跳躍時，跟腱承受著巨大的負荷，甚至是體重的數倍。跟腱是人體最強壯的肌腱，由小腿後方腓腸肌和比目魚肌共同延續融合而成。在正常健康的跟腱組織中，跟腱在腱鞘內自由的滑動。反覆的、過度的滑動引起炎性反應，損傷的組織細胞釋放出大量的化學物質，導致血管滲透性增加，血細胞滲漏到組織中，肌腱和腱鞘之間摩擦力增加，最終引起疼痛以及肢端的腫脹。

跟腱炎的早期癥狀並不明顯，僅僅感到跟部輕度的疼痛或不適感，常常被人忽略。早晨起床足跟接觸地面時，跟部出現緊張或疼痛，適度活動後即有所減輕，往往不會引起重視。當運動量逐漸增加時，開始感到沿跟腱走行的持續疼痛，呈進行性加重。觀察到癥狀變化與持續性使用或活動有關，這點非常重要;如果沒有得到合適的治療，組織的損壞和肌腱炎加重，最終進展為跟腱撕裂。

臨床中將跟腱炎分為四期

I期：活動後疼痛。功能正常，無步態改變;能夠正常行走和跑步。

II期：活動時和活動後疼痛。仍然可以參與運動，運動前的熱身活動增加了局部的血循環、減輕疼痛。短時間內感覺良好，但肌腱的損傷並未癒合。

III期：活動時和活動後疼痛時間延長。感到持續的疼痛，運動距離縮短或步態改變，有時需要使用止痛藥。

IV期：肌腱出現撕裂，需要外科介入。

跟腱炎的治療

跟腱炎的治療是困難的，早期治療的核心是去除跟腱炎的病因，常見的病因有：

  1. 運動前沒有合理的熱身或牽伸訓練
  2. 奔跑強度過大或頻率太多
  3. 持續性的，重複性的跳躍動作
  4. 不適合的鞋並沒有起到保護的作用

通過限制或去除以上因素從而減緩或反轉早期的炎性反應，並治療跟腱炎。

如果疼痛癥狀較重或無明顯緩解，建議就診聽取醫生的建議和評估，得到專業的治療建議。雖然跟腱炎並不經常需要外科介入，但支具或石膏制動能有效控制疼痛癥狀。如果在受傷早期階段得到重視，則可避免其進展為嚴重的階段。

如果跟腱炎只引起酸脹和疼痛，並沒有影響步態或運動狀態，推薦：

  1. 穿矯形鞋、抬高足跟，以減輕跟腱的張力
  2. 充分熱身活動和常規牽伸訓練
  3. 改善跑步姿勢或訓練方式

在臨床上，採用不同的治療模式或器械來增加跟腱的血供，有助於減輕水腫和炎症。當跟腱得到充分休息、適當活動後，開始進行力量和協調性訓練。

外科的介入僅僅針對於嚴重的跟腱炎，伴有或不伴有跟腱止點鈣化或跟腱斷裂的患者，比如劉翔。2008年，劉翔坐在起跑線上痛苦的表情記憶猶新。手術方式的選擇多種多樣，應得到專業的足踝外科醫生的治療。

跟腱炎的預防

國際著名足踝外科專家，重慶西南醫院骨科運動醫學中心唐康來教授認為：預防大於治療。運動前必須要充分做好活動準備。人體非常奇妙，幾乎可以適應任何環境。從低強度、低頻率的訓練到中強度、中頻率的運動項目，逐漸進展到為一個平台期，繼而過渡到合適的生理適應，最終成為一個自然的過程。根據參與運動的內容和特點，必須花足夠的時間改善運動所需的技巧，逐漸增加訓練負荷和強度，最終允許我們的身體恢復以及適應這些運動。另外，舒適合腳的鞋或護具，運動前合適的熱身運動，以及通過不同的牽伸訓練和深部組織的按摩維持機體的彈性都很重要。

步行、跑步是一種理想的耐力訓練，對心臟循環系統產生有益的影響，並能提高能量基礎代謝。有節律的呼吸、規律的擺臂以及適度的步伐，使得機體新陳代謝得到改善。定期進行長跑的人，身體素質、體態及心理能力都有所提高。為了能夠繼續進行所鍾愛的跑步運動，我們必須採用一些必要的措施，保持我們平時以及在運動過程中的健康。
