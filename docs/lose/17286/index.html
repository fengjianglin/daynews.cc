擊球時你被“擠”到了嗎？留出合適的空間是關鍵

2019-12-14 2,005條評論 4,599次閱讀 1016人點贊

北陲特務 好動網球

滿腦子裡全是網壇男神女神的精湛技藝，這時的你，有沒有把自己的網球技巧提升一級的衝動？好動網球特別推出特別欄目《新春技戰》，將從發球、接發球、正手、反手、截擊、高壓、體能與步伐、熱身與損傷治療、雙打戰術以及網球裝備等方面給大家帶來系統的學習過程。

從高手到菜鳥，都或多或少面臨類似的問題——有時擊球施展空間不足，身體被擠到了，而有時又距離太遠，感覺是夠著球打。尤其在反手擊球時，很多人甚至擠做了一團，只能僵硬地把球擋回去，而不能施以充分的力量。

我們以邊線做為參考坐標，發球機要沿著邊線送球，以此來觀察合適的擊球空間。

正手位擊球時，建議胸部距離擊球點至少有半米距離，把手臂伸直，大約就在邊線上方，留出的空間要足夠手臂伸直。

當身體向右靠近了一些，擊球時身體就會被擠到，擊球無力，勉強輕輕擋了一下的感覺。

這個動作重複練習100下：在擊球點將動作暫停，體會合適的空間，如果感覺站位緊迫，向左調整兩小步。

再來了解一下雙反或單反，無論你使用開放式站位或閉合式站位，對於雙反，要確保擊球瞬間左臂是伸展的，雙手伸出胸前大約30公分，而擊球點同樣是離身體半米遠的位置，擊球之後順勢向前上步。

注意，雙反擊球還有另一個關鍵，雖然擊球的空間把握得很好，但擊球之後卻迅速把球拍縮了回來。理想狀態應該是，反手擊球後也要有隨揮，整個擊球動作把球包裹了一下。

單反擊球也是同理，但站位時留出的空間要大一些，擊球後要徹底打開手臂。

正反手各重複練習100下，在擊球點將動作暫停，體會合適的空間，然後再練習連貫動作100下。之後就可以與發球機對練了，確保自己擁有足夠的擊球空間，一旦發現自己的動作被擠到了，那就是離球太近，腳步要收回一些，通過練習糾正站位，獲得適合的擊球空間。
