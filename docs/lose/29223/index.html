都是健身1小時，人家為什麼練得那麼好？我有什麼沒做對？

2019-12-22 136條評論 2,525次閱讀 463人點贊

每天健身多久才有效？1小時。這答案夠簡單，也夠粗暴，而且眾多的普通健身者，差不多都是以每次1小時的運動時長來健身的，也能取得不錯的效果，是一種共同的經驗。

但管用，並不表示它是真理或唯一的答案。這個世界的複雜性在於，很少有「絕對的答案或標準」。「每次健身多久才能有效」這樣的問題，同樣有其複雜性，涉及每次時長、訓練頻率、堅持規律運動多久、運動強度，以及個體差異、健身目標和飲食等各種影響因素。

和運動相關的哪些因素左右著鍛煉的「有效性」？

  • 因素1：每次時長

減肥者們一定聽過教練經常說的一句話，「有氧運動每次最好超過30分鐘，這樣才能發揮更好的減肥作用」，這就是最典型的「運動時長對效果產生影響」的例子。然而，運動時長並不是一個孤立的影響因素，比如：

用散步或快走這樣的低強度有氧運動想達到較好的減肥目標，就必須投入比中等強度下長得多的運動時間。

用高強度間歇訓練（HIIT）這樣的方式來鍛煉，一次約20分鐘的訓練就可以達到傳統有氧運動可能40至60分鐘才能達到有燃脂效果。

而力量訓練就不能簡單地用每次運動時長來衡量，因為訓練效果如何，是身體對具體被施加的訓練刺激的反應，動作的標準、重量的大小、間歇的長短、動作組合的方式（比如超級組、三聯組）等因素產生的影響要比時長更重要，而且也並非鍛煉時間越長越好。

  • 因素2：運動頻率

通常無論是有氧還是力量訓練，每周至少3次的運動安排是累積鍛煉效果的前提。但不同訓練水平的健身者，由於目標的不同、恢復能力等原因，運動頻率也會各不相同。

一個剛剛開始力量訓練的新手，一周三次，每周40分鐘的力量訓練，可能已經足夠了。而對於一個訓練多年的健身者來說，每周五六次或者有時候「一天兩練」的訓練安排，才能達到他想要的效果。每次的訓練時長則每個人不盡相同，有些人可能只需要45-60分鐘進行某次訓練，而有些人卻可能長達2小時。

  • 因素3：運動強度

在因素1「運動時長」中，實際上已經談到了運動強度對於時長的影響。在力量訓練中，運動強度表明了「鍛煉負荷對於身體刺激的程度」，受到重量、難度、動作的速度等多重因素影響，其中「重量」是最重要指標之一。重量越大時，訓練者需要恢復的時間也需要更久一些。

  • 因素4：訓練質量

一個有訓練經驗、懂訓練技巧、訓練時保持高度專註的資深健身者，當然可以在更短的時間內完成訓練，且取得更好的訓練效果，因為訓練質量高。與此形成鮮明對比的是，剛進入健身房進行力量訓練的新手，往往不知道從何開始、也不知道如何練，最常見的就是什麼器械都練一遍，一次訓練的時間很長，效果幾乎為零。

  • 因素5：堅持多久

「健身是以『年』為單位來衡量效果的」，這句話諸位看官想必和御行君一樣熟悉。新手有氧運動減肥，大多數在最初的兩三個月就能看到不錯的效果。但力量訓練者成長為標準的肌肉男，就非得以「年」為單位來計算了。

上述這些因素將交織在一起，共同發揮作用，影響著每次健身的時長，以及最終的健身效果，而不只是由「運動時長」一個因素來決定是否健身有效。不過，在這些因素中，運動時長、運動頻率和堅持多久是最基礎的三個因素，只要確保它們達到訓練要求，才能談論健身效果好壞的問題。

不可忽視的兩個問題：健身目標與飲食

  • 健身目標和運動階段的影響

假設鍛煉者是新手，目標僅僅是減肥，那麼每周三至五次，每次40至60分鐘的有氧運動，就可以達到有效減肥的目標。

但如果某個男性鍛煉者已經有了一兩年的運動基礎，想將體脂率由16%降低到10%，單純的有氧運動方案就不能滿足他的要求。除了力量訓練，還可能需要隨後進行半小時或更多時間的有氧運動，運動方案也需要更有針對性（譬如採用間歇跑、燃脂跑），或者單獨安排另外的有氧訓練或其他訓練。

增肌者同樣如此，不同階段和不同訓練的目標影響和決定著他們的訓練時長和頻率。例如：一天只安排一次訓練，可能長達1個半小時，但一天兩練，可能一次訓練安排60分鐘。

  • 談健身效果永遠繞不開飲食

主流的減肥理論要求保持「熱量負平衡缺口」，以迫使身體消耗多餘的脂肪供能。在不節食的情況下運動，減肥效果會有，但會被大大削弱。一次訓練的時間越長，燃燒掉的體脂占能量總消耗量的比例也就越大。所以，想不節食、又想要好的運動效果，那麼就需要投入更長時間的運動。

增肌訓練者的情況則更為複雜，在增肌階段需要確保熱量盈餘，在減脂階段則需要更嚴格地控制飲食，確保儘可能地維持住來之不易的肌肉量，卻要儘可能消耗掉多餘的脂肪，以塑造更好的身體線條。這個過程將同時伴隨著運動時長的調整，比如在減脂期增加有氧運動的投入。

健身時長的參考策略

（1）採用大眾經驗，每次訓練1小時左右，適合初中階訓練者。

（2）力量訓練者的關注點是，如何達到提升力量、增加肌肉圍度的效果，這些主要取決於訓練方案和實際的訓練質量。只要不導致過度訓練，每次30分鐘、45分鐘、60分鐘或者更長的訓練時長，都可以。

總結

（1）每次健身多久（訓練時長），只是影響健身效果的一個因素。

（2）運動時長和運動頻率、運動強度、運動質量、堅持多久等因素共同作用，影響最終的健身效果。因此，不能孤立地看待運動時長和健身效果之間的關係。

（3）健身目標、運動者所處的健身階段以及飲食，也是影響健身效果的重要因素，同樣對運動時長產生影響。

（4）大多數人每次健身時長1小時就行，而力量訓練者更應關注訓練方案和實際的訓練質量。
