訓練太極拳的發力，分階段分層次，誰練誰有

力，就其形式而言主要有：線性力(直線力、弧線力、單向力、雙向力)、槓桿力(平衡力、支撐力)、旋轉力(離心力、向心力、螺旋力)。就其做功形式而言，主要有衝擊力和膨脹力。就其釋放能量快慢而言，又有瞬間發力和緩慢發力。就其類別劃分則可分為：簡單力和複雜力，靜力和動力。在拳中又分為整體力和局部力，明勁和暗勁。線性力和衝擊力、槓桿力是簡單力；旋轉力和膨脹力是複雜力。複雜力是兩種以上力的複合。

發力訓練有兩個階段，三個層次：

第一階段 是精神意識的訓練，像人受了驚嚇，渾身一顫，毛髮盡豎。前提仍是松靜，就精神發力的原理，符合生物科學，精神緊是一種應急的本能反映，一緊全身俱緊。腦緊，神經傳導加快；心緊，瞬間收縮壓加大，加快血液(能量之基)對全身的供應；腹腔壓力瞬間增高，對臟腑起穩固作用，同時腎臟在壓力的刺激和中樞神經的傳導下分泌大量的腎上腺素，增強應急反應的敏感，使人處於高度的臨戰狀態，整個身體的功能都被激活，使人精神高度振奮，威武神聖，敵不可欺。神人敵之髓，精神先勝人(同時亦要有膽量的訓練)。

第二階段 是形體發力的訓練，從松靜著手，勿追求力的大小、快慢、輕重，只在鬆緊中體驗便能人道。形體訓練分局部訓練和整體訓練，整體訓練是在局部訓練後的綜合訓練，重點是腰胯發力。

腰胯發力是在轉腰運胯的基礎上，主要訓練瞬間回彈。挫胯訓練好壞是決定發力質量和速度的關鍵。訓練不能強求，重點仍放在體悟上。從感覺上找，感覺對姿式便沒問題，或稱擰腰挫胯，猶如在旋轉一個強勁的彈簧，撒手便回彈。擰一分力回彈三分力，擰三分力便回彈十二分力。擰的越慢蓄力越足，撒的越快，回彈越大，(顫力或震力、振力是在回彈時由阻力而生，有主動和被動之分)由慢入快，悉心體悟。

主要方法是抖翎、開合訓練。開合有前後、左右、上下，開就像拉皮筋，拉彈簧，合就像是回彈，只在一松一緊，瞬間即成。抖翎是在腰胯發力的基礎上，手臂任意開合運轉、抖彈。這種訓練之所以稱之為綜合訓練，腰胯並非在平面擰轉，而形成立體的運動，頭頂、身拔、足蹬、腰伸、胯縮，各種要領協調配合自如，擰拔中又有回彈，反饋，共振。在腰胯擰挫回彈的一瞬，由於阻力的運用，不發而發，由腰部向身體各部順骨節輻射，節節貫穿達於四梢，行於手指，使整個身體產生瞬間的抖顫、驚彈。像動物抖毛，鳥之抖翅，是謂抖翎，這種力是武人苦苦追求的上乘之力，太極拳稱之為「金雞抖翎」。

第三層 精神發力訓練方法亦是第一階段的深化，但仍有不同的要求和必須注意的問題及成熟的表現。初級階段只是一般感覺，是在意識的誘導下完成的。這一層則完全是一種不自主的體驗，意念上似有若無，似是而非，精神超然放鬆，反應極敏，達到一種超脫的狀態，毫無妄想，一切均是條件反射，即所謂煉神還虛，亦即前人所述的「天人合一」的境界。

在訓練中你如能體悟，並進入如此境界，全身極輕鬆，動作極活靈， 自覺已發不出力時，無力而有力，你將成為一名真正的太極高手，一切神功不在話下，僅憑知覺感應的反射所發之力，無堅不摧，你將不懼任何來犯之敵。

最後說說功夫的極限，人的潛力是有限的，把自己的潛能最大限度地挖掘和自如調動即是高層功夫，要兼收並蓄，以養為主，不能過分超負荷超極限發揮和使用能量，否則就等於殺雞取卵，透支存款。前人功夫了得，英年早逝的現象不能不思，今人不能重倒復轍。
