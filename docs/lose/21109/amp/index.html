不過我好喜歡這種天氣，我真的很怕冷嗚嗚嗚

小編今天給大家帶來的是一位帥氣的教練小哥哥，要說真正長得好看的健身教練應該沒有幾個。

男神級別的健身教練更是少之又少，而且還有很多教練的身材居然沒有他的學員身材好。

多數教練都是身材能看，但是臉蛋不能看，而今天這位小哥哥確實是無論是身材還是臉都無法挑剔！

看的出來，沒有一張照片加了美顏，但是張張都很美，沒有沒有和濾鏡的照片小編覺得非常的有質感。

其實現在很少有男生留鬍渣，因為現在都流行那些，就是現在的審美都是流行那些乾乾淨淨的小哥哥。

所以很多男孩子都把鬍渣刮掉留起了厚厚的劉海。

而這種平頭、鬍渣，肌肉小哥哥其實才是真的好看，小哥哥出生於1992年，今年27歲。

這個年齡正是男人最美的年齡呀！！

小編看過的帥哥還是蠻多的，有些人的照片修圖嚴重，也有些人的照片開了很大的美顏和濾鏡。

總之，為了讓自己的照片看起來好看大家都在不擇手段。

而這位小哥哥我估計現實中比照片還要好看！！

小哥哥雖然並不是什麼網紅，也沒有什麼粉絲，但是光芒總是掩蓋不住的。

還有知名攝影師給他拍寫真，還有各種各樣的雜誌。

經過精修後的雜誌封面也是看起來精緻很多。

拋去美顏和濾鏡，我們的顏值還剩多少？但是無論還剩多少，一定要勇敢的做自己。

因為照片再好看，見到真人會嚇跑對方的，你說是吧！
