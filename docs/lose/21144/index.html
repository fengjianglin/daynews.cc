運動補水飲料白水最好？了解什麼是低滲，高滲，等滲

2019-12-19 303條評論 1,352次閱讀 162人點贊

在運動過後，補充水分是非常重要的，水分與人體中礦物質平衡密切相關，在運動過後大量的流汗，必須要及時補充水分，否則會對人體造成非常大的影響。當然在補充水份時我們要選擇適合飲料，那白水是不是最好的選擇呢？下面給大家簡單科普一下。

我們在補充水分時要關注水鹽平衡，人體內被精確調控的水鹽平衡，水鹽失衡時人體會發生各種細胞群的功能紊亂。所以我們在選擇飲料時便有了低滲、高滲、等滲之分。

1.高滲

高滲是一種生理現象，為體內的滲透壓高於體外，水由環境中向體內擴散，體內的鹽分向外擴散。通過排泄作用排出多餘的水，鹽分通過食物和組織攝入。

體內的滲透壓高於體外，水由環境中向體內擴散，體內的鹽分向外擴散。通過排泄作用排出多餘的水，鹽分通過食物和組織攝入。

高滲飲料：高滲飲料，比如水果水和功能飲料中的溶質粒子濃度比血液中的高，體內的物質濃度達到平衡，水會通過腸壁從血液進入腸道，因此運動者一開始會因為缺少體液而感到口渴。只有在一段時間之後，在下端的腸道內各種成分才能再次達到平衡，共同進入血液。由於這種時間上的滯後性，高滲飲料不能還用來快速補水。比較典型的高滲飲料有：各種高濃度的功能性飲料、濃縮高糖果汁等；

2.等滲，指的是滲透量相當於血漿滲透量的溶液。如0.9%NaCl溶液和5%葡萄糖溶液。低於血漿滲透量的溶液稱為低滲溶液，細胞在低滲溶液中可發生水腫，甚至破裂。高於血漿滲透量的溶液稱為高滲液，細胞在高滲溶液可發生脫水而皺縮。

等滲飲料：等滲飲料中的溶質粒子濃度，即進入腸道的鈉離子（Na+）的濃度與血液中的相同。流失的電解質可以通過飲用等滲飲料快速得到補充。等滲飲料中的水（H2O）到達血液的速度只比低滲飲料的稍慢一點點。等滲飲料對長時間的負荷訓練很重要，因為運動者在進行這種訓練時身體內的電解質流失是主要問題。比較典型的等滲飲料有：運動電解質泡騰片、運動恢復沖劑等。

3.低滲，水分子通過半透膜向溶液擴散的現象稱為滲透現象，簡稱滲透；溶液促使膜外水分子向內滲透的力量即為滲透壓或滲透吸水力；以血漿的正常滲透壓(7.6個大氣壓或5776毫米汞柱)為標準，低於血漿正常滲透壓的溶液則稱為低滲。

低滲飲料：低滲飲料中的溶質粒子濃度低於血液中的濃度。如果是飲用極度低滲飲料（其鈉離子濃度遠低於血液中的濃度），在特殊的運轉機制的作用下，血液中的鈉會被轉運到腸道中，於是人體對水分的吸收反而延遲了。但普通的輕度低滲飲料則不同；水分子迅速流入鈉離子濃度略高的血液中。輕度低滲飲料因此成為迅速補充水分的最佳選擇。比較典型的低滲飲料：自來水、純凈水；輕度低滲飲料：礦泉水、礦物質水等，如果將電解質泡騰片或固體飲料加入超量的水溶解，也是很好的輕度低滲飲料。

在一般訓練中，如跑步時間不超過60~90分鐘，出汗量不那麼大，電解質的消耗也沒那麼多，那麼在運動過程中和運動後飲用富含鈉（含量為400~1100毫克/升）的礦泉水就足夠了，當然如果礦泉水配合適量的運動飲料會更科學。

當訓練強度增高或者時間增長時，如在馬拉松比賽中或賽前LSD的高強度訓練中，則必須採用等滲飲料結合輕度低滲飲料來達到補水和電解質的需求。

等滲飲料可以快速補充電解質，例如：採用符合運動營養標準的電解質泡騰片飲品、固體運動飲料沖劑（一般運動飲料也可以），或者食用一支能量膠並伴以礦物質水，也是不錯的選擇。

輕度低滲飲料則能特別快地讓身體重新達到水分平衡的狀態，建議採用固體飲料沖劑或者電解質泡騰片等根據濃度需求來自己製作低滲飲料，或者採用礦泉水配合1~2顆電解質鹽丸也能夠達到同樣良好的效果。
