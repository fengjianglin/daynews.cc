「跑者玄學」：了解自己個性特質，更好地制定跑步比賽訓練計劃

2019-12-18 1,794條評論 3,888次閱讀 661人點贊

大家心裡其實都很清楚，想要成為一個優秀的跑者，制定訓練計劃，嚴格執行訓練要求，控制每天的食物攝入，都是必不可少的。

訓練計劃的制定上是有講究的。一般來說，會根據跑者的體能水平、訓練歷史和過往成績設置不同的強度和訓練內容，訓練計劃的框架基本相似。更常見的情況是，很多跑友會從網上或者別人那裡把訓練計劃照搬過來，大概認為，既然這個計劃適應別人，自己也應該差不多。

雖然說一樣米可以養百樣人，但是很多人會發現自己的訓練計劃執行上並不順利。可能跑的不舒服，或者自己的時間協調不過來，總之，很多訓練計劃剛開了個頭，就無疾而終。這時，大部分人常常責怪自己不夠自律，沒有毅力，但是很少有人會想到，自己的個性特質會影響到跑步訓練計劃的執行。

關於性格學的研究其實不算陌生，有一段時間國內就很流行什麼『九型人格』、『性格顏色說』等等理論。大部分的性格學研究被用在溝通、交流等人際關係方面，但是性格個性實際上會影響人的整體，是在一件事情上，決定實施什麼樣的行為的根源。

呼嚕爸爸今天要介紹的是一個國外研究歷史很長，實施使用範圍很廣的性格學理論，可以簡稱為『DISC性格理論』。

DISC性格理論背景簡介

DISC理論，由哈佛大學的William Moulton Marston博士研究出的，用於找出人類行為中可預測行為以及個人習慣。Marston博士把這些可以預測的人類習慣歸結到DISC的個人特質類型中。

按照DISC性格學理論的研究結果，把人分為四種個性特質：主導型（Dominant）、影響型（Influential）、穩定型（Stable）和服從型（Compliant），這也是理論名稱的由來。

上面的每種類型都關聯著一些特定的性格特徵、思維模式和行為習慣。比如說主導型，他們更傾向『做就好了』的這樣的行為模式，不會關注細節，追求兩點之間最短的距離，更不喜歡研究過程；而影響型人群，會注重人與人之間的關係，關註解決問題的步驟，他們的重點會放在『如何更快樂地做一件事』而不僅僅是把事情搞定。

對於穩定型人群來說，首先不願意與不喜歡的類型共事。行為決策之前非常希望得到明確的保證，確保行動結果不會變得糟糕而無法接受。而且他們更喜歡呆在舒適區里，不太願改變；服從型的人群在細節上與主導型完全相反，細節越多越能使他們放心。所以服從型人群行動之前，一定要徹底了解清楚所有來龍去脈才會走向下一步。

人類行為從來不是簡簡單單，非黑即白的，不同的性格人群當然會有行為模式上的區別和行為慣性。那麼我們如何比較準確的判定自己是哪一種類型呢？

回憶一下自己的行為習慣，當你做決定之前，會不會花很多時間去思考，甚至需要考慮得面面俱到？或者是容易衝動，一時心血來潮就會馬上採取行動？你是喜歡把所有的事情記到本子上，還要用馬克筆標上重點符號？還是只要聽到耳朵里，心裡過一下就完全Okay的？

無論是哪一種類型，只要更了解其背後的含義，就更能預測自己的行為模式，並規避一些不利的習慣。實踐證明依據性格調整行為，對工作、人際關係還有日常生活習慣的培養都會產生很大的影響。跑步也不例外，畢竟跑步也是一種特定的人類行為。

依據自己的個性特質來跑步訓練，不僅可以讓訓練計劃執行更順利，在訓練效果和完成效率上更可能進一步優化。當訓練計劃與個體的特質吻合時，每一次訓練就變成自己最想做的事情，訓練結果也會變成自己主動追求的目標和成功方向。自然，每一次訓練結束，都會讓跑者感覺離成功更近一步。

我們回到如何把DISC理論實施到跑步訓練上。

第一步：確認自己的類型

通常這樣的性格類型測試，有專門的測試程序還要支付一定的費用，能得到很詳細的報告與解讀。但是對於普通跑者而言，我們完全可以利用上面介紹的信息和下面的一些方法，使每一種個性特質具象化，從而參考估斷自己所屬於的基本類型：

主導型：主導型是衝動性的決策者。對所有事情抱有一種『做就好了』的態度。非常富有競爭性，渴望勝利。做事情的動力來自於成功或者特定目標的達成，是典型的只看結果、不管過程的人群。

影響型：注重人際關係，依據各種標準來判斷並做出行為決策。換句話說就是非常守規矩。喜歡與人溝通，嘗試新鮮的事物，但容易在壓力下崩潰。

穩定型：行為決策時非常猶豫不定，需要強有力的保證才願意把事情向前推進。希望呆在自己的舒適區里，做事情時喜歡保持愉悅的心情。

服從型：非常注意細節和具體的證據，在遇到壓力時會埋怨自己，害怕被人批評、評價，做決策時非常慢，需必須要有足夠的數據支撐才敢於做出行為決策。

我們必須要注意到，很多人會發現自己同時有兩種以上的類型特徵。這其實很正常，純粹單一的個性特質其實很少。

很少有人是單一類型的特質，基本上是以某種特質為主要特徵，然後其他特質表現得稍弱一些。

一個人的主要、次要、偶爾甚至缺乏的個人行為習慣混合形成了DISC四種個人特質，這個混合體會影響每個人每天在不同環境之下的行為、表現、與其他人交流的方式、組織能力、面對衝突時的處理方式等等。

理解一個人的DISC個人混合，就能夠採取較為主動的方法把人們放到自我感覺更舒適、更有激勵的環境下，讓他們積極地面對問題，正確理解和預測自己的行為。

當發覺自己是某兩種或者某幾種類型混合時，首先看平時表現最為明顯的特徵，以自己的主要類型為依據來進行下一步的計劃制定。

第二步：根據性格類型選擇適合的訓練方式

了解了自己的主要性格類型，在選擇訓練方法和制定訓練計劃時，就可以把這個因素納入到整體中一起考量，而不單單只看體能和訓練目標。這麼做的好處就是訓練計劃會更有效果，訓練活動也能夠執行得更好。

下面是一些例子可以作為參考，然後舉一反三：

主導型：乳酸閾值跑

主導型可以用清晰的目標和結果來刺激自己的訓練動力，然後在每一次的訓練中挑戰目標。

乳酸跑很適合主導型人群，基本上只要出門開始跑就行了。訓練計劃中只要有距離、有配速要求，不用過多細節。記錄下每次的訓練成績，在下一次訓練時嘗試突破，把每次的跑步訓練變成與自己的比賽。

訓練中，主導型人群不要過多考慮配速、距離以及每一公里的細節。這個類型的人群如果過度分析訓練，會容易有挫敗感並感到疲憊。這些思考同樣會消耗能量，非常容易讓跑者把訓練計劃完全放棄，再起爐灶。

影響型：長距離訓練

影響型可能對長距離訓練更情有獨鍾。這個距離里有很多事情可以做，又有參照的標準，這會讓跑者感覺很開心。

長距離跑訓練給了影響型人群一個很好表現機會，長距離的勻速穩定讓這類型人群可以有堅持的標準。在跑前做好線路的規劃、補給方案甚至返程方案會有加分效果。

但是，影響型人群進行長距離訓練時，要避免不做規劃就跑步。這樣在陌生的線路和路況下，配速的過多變化會讓影響型人群有失去控制的感覺，會跑得很不舒服甚至崩潰。

穩定型：操場跑道上的訓練

穩定型的人群希望在訓練時有一定的保障，所以操場上和朋友一起訓練會感覺舒適。

穩定型人群和跑友一起到操場跑，一起完成預定的訓練，比如三組六乘400米。或者讓其他跑友帶著一起衝刺400米，這樣不用自己決定使用什麼配速，怎麼沖等等，會讓這類型人群感覺訓練更輕鬆一些。

最需要避免一個人在操場上進行速度訓練活動（比如間歇跑）。從心理角度而言，穩定型人群適合群體活動，不適合成為獨狼跑者。在做速度訓練時，沒有其他跑友，就必須自己做很多決策，而且會喪失共同完成的成就感。

服從型：法特雷克跑

服從型人群完成法特雷克跑，能夠滿足更多對於細節的追求。

法特雷克跑法多變，可以在時間基礎上跑不同的距離、配速或者調整恢復時間。所以在訓練開始前，把每一段的細節都寫下來，決定好每段的距離、恢復時間等內容，在執行訓練時會感到很大的成就感。

要避免訓練前不計劃每一段的細節。這樣不得不在完成每一段時再去想下一段如何跑，對於服從型類型來說，這會耗費大量的心理能量再訓練中找尋細節，結果就是訓練效果很差。

第三步：訓練計劃的組合

一個有效的訓練計劃是不同的訓練類型組合而成的。即使每一種人群類型有更適合自己的跑步類型，也不可能放棄其他的訓練。

因此，通過對自己個性的分析，找出每種訓練中需要完成或者需要避免的內容，設計出自己獨有的訓練類型，最後再根據自己的體能水平、訓練歷史和訓練目標，進行組合，就能制定出完成效率更好的訓練計劃。

有一點很重要，前面我們提到了類型的混合。大多數情況下，主要類型會影響到個體基本的行為習慣，但也有次要性格對跑步影響更大的情況。比如影響型人群有時也會覺得自己想的過多，最好像主導型那樣說干就干。而想要做到這一點，就需要對自己的個性特質了解得更清楚才行。

結語

按照個性特質來制定訓練計劃，甚至考慮比賽策略，乍聽之下可能覺得是跑步玄學。但是人類的行為雖然複雜，但並非不可預測，這是心理學、性格學理論很重要的存在依據。

在這裡，最重要的不是哪一種性格學理論更可靠，而是在考慮自己的訓練計劃和比賽策略時應該綜合更多的影響因素，這樣才能夠讓跑者進一步釋放能量，發揮潛力，跑出自己的佳績。
