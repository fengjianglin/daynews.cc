女孩因肥胖被男友分手，一氣之下決定減肥，網友：為未來老公默哀

現在這個看臉的時代，長的好看才能活的漂亮，可是對於長的好看，可是身材不好的人很多人都是覺得天生的，沒辦法改變，可是卻有些人，偏不信這個邪，他覺得只要是自己想改變，就沒有改變不了的。

視頻中的小姐姐就是這樣，她的長相可以說是很漂亮了，不像現在網路里的網紅臉，而是非常自然大方的美，可是小姐姐的長相很好看，身材卻有些微胖。不過對於很多人來說，她這點胖根本不算什麼，而小姐姐卻因為這點肥胖而失去了自己最愛的男友，一氣之下做出這樣的決定，從分手的那天起她就發誓一定要瘦下來，讓那個變心的男人看到自己漂亮的樣子。

於是小姐姐從網上購買了各種健身用品，不管是練拳擊的，還是連引體向上的，都買來了，就是希望自己有天可以穿著漂亮的衣服出現在前男友的面前，而小姐姐平時的練習可以說是接近不要命的狀態了，她不但不斷給自己增加難度，還時常的練習到忘我的狀態。直到她穿上曾經前男友送給她的那件藍色連衣裙才知道自己是真的瘦了。可是小姐姐的心態通過健身也改變了，她覺得不需要為了向誰證明什麼而改變自己，只要是別人不喜歡，自己再改變也沒有用。所以自己現在的樣子，就是最好的樣子，不再為誰而改變自己，看著這麼彪悍的女孩，做引體向上這麼厲害，可見力氣不是一般的大，這樣的女孩一般人駕馭不了，看後網友紛紛評論：誰敢娶

網友看到後紛紛表示：這女孩以後誰敢娶啊，力氣比男人還大。還有網友吐槽道：為她以後的老公默哀。不知道你們對此有什麼看法呢？歡迎大家在下方留言評論哦！
