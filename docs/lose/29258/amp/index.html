倩狐減肥：2020年女生標準體重出爐？少給我來這套

天天要聞
2019-12-21

2019年已經開始進入倒計時啦，大家對今年的身材滿意了嗎？很多妹子都熱衷減肥，對自己的體重秤上的數字特別敏感，恨不得怒減40斤才滿足。但一味追求數字，並不一定能獲得好身材。

最近九收到了姐妹發來的最新版女生體重自查表，如果你之前關注過標準體重表的話，就會發現「標準體重」、「美體體重」和「模特」體重都有所減少。

但這也對女生太苛刻了吧，我的體重我做主，不應該定什麼體重標準。就算胖一點怎麼了？

九早上在微博熱搜也看見有網友說自己因為胖聽過很多最傷人的話，但他們並沒有因為那些惡言而放棄自己，反而給了改變自己的動力。

他們的故事

微博id：CallMeRed豆兒

平時開玩笑，就會說「哎呀，你怎麼這麼胖，你看找出來的照片，太胖了……」。其實我也知道是玩笑，但是有的時候自卑心上來就會很認真。我問男朋友：「胖瘦很的很重要嗎？」他說：「介意。」那一刻，真的挺傷心的。因為他沒有看到之前我為減肥所付出的努力，只看到了那幾天吃太多……

平時無意識的話，其實是最傷人的。
不過他的介意也是為了我好，不想未來，我太胖，身體不健康。好吧，調整自己心態！加油減肥！

微博id：招財的黑貓

每次被人說到胖瘦的問題的時候，我不禁想質問瘦=好看嗎？不過是些帶偏見的眼光，胖瘦都有各自的風韻。

微博id：神也很煩啊

剪完頭髮我媽說我，頭髮紮起來是劉歡，放下來是高曉松

微博id：富婆唯我是也

我覺最傷人的話，男生被罵死胖子，女生被罵肥婆。
其實，每個胖的人都是潛力股，瘦下來分分鐘變男神女神。所以，胖的人記住管住嘴、邁開腿多運動才是正道。 你已經長大了，就不要再長胖了！

微博id：Zh_香菇醬

胖也挺好的，滿員的時候副駕駛永遠是你的；胖也挺好的，吃飯都你一個人坐一邊（四面的方桌，六個人一起吃飯）；你這脂肪冬天還怕冷？多少年了，來自周圍滿滿的「惡意」

微博id：打到小怪受


突然覺得自己還是幸運的，雖然被人說過。被自己暗戀過得人說過傷人的話，但是這麼多年，自己真的很幸運了，周圍的朋友都對我很好，沒有嫌棄我，只是關心我，鼓勵我，這個世界，還是很美好的啊~

微博id：__世事如煙

其實胖的人內心都敏感脆弱，你覺得沒有惡意的開玩笑對胖的人來說都有可能是重傷。

有時候羨慕國外的有些女孩，很多你看著胖的女孩都穿的特別漂亮，言談特別自信，從不為身材自卑。

那種骨子裡散發出的「老娘這樣也是美」的氣場真是我這輩子做不到的…有段時間特別想夢回唐朝

微博id：十安大漂釀


肉肉的女孩子不管看起來有多開朗 心裡還是會壓抑著自卑 你說我胖我覺得你在嘲諷我 你說我不胖我還是覺得你在嘲諷我 但是很幸運 我身邊都是一群溫柔的人 他們從不會因這個話題傷害我 所以 願每個人都活在溫柔中吧

看了那麼多，九在最後還是要給你們喂點雞湯，勇敢面對面對這個世界的滿滿惡意，不管別人怎麼看你，胖還是瘦，都應該由你自己決定！
