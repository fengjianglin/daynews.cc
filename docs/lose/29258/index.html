倩狐減肥：2020年女生標準體重出爐？少給我來這套

2019-12-21 2,039條評論 4,129次閱讀 1653人點贊

2019年已經開始進入倒計時啦，大家對今年的身材滿意了嗎？很多妹子都熱衷減肥，對自己的體重秤上的數字特別敏感，恨不得怒減40斤才滿足。但一味追求數字，並不一定能獲得好身材。

最近九收到了姐妹發來的最新版女生體重自查表，如果你之前關注過標準體重表的話，就會發現「標準體重」、「美體體重」和「模特」體重都有所減少。

但這也對女生太苛刻了吧，我的體重我做主，不應該定什麼體重標準。就算胖一點怎麼了？

九早上在微博熱搜也看見有網友說自己因為胖聽過很多最傷人的話，但他們並沒有因為那些惡言而放棄自己，反而給了改變自己的動力。
