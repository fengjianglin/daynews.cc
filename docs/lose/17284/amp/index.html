從土肥圓變高富帥，是種怎樣的體驗？

天天要聞
2019-12-14


內個，事情想必你還不知道

近日，一個微博名為「西瓜杉Shawn」的小哥可謂爆紅網路，他不僅顏值高，身材好，而且還是位93年出生的小鮮肉，最關鍵的是，人家還擁有讓絕大多數男人都望塵莫及的192cm的身高，且單身單身單身。想必很多妹子們心裡又該默默地提醒自己「他是我老公」

其實，很多小夥伴不知道的是，就在幾年前，西瓜哥還是一副土肥土肥的身材，後來是健身改變了他的生活，不敢說是整容刀吧，起碼跟變魔術是一樣兒一樣兒的！

這是西瓜哥15歲左右的樣子，確實是他，你沒看錯！

最胖的時候是高中階段，體重一度飆到230多斤！

後來他留學美國，在伊利諾伊大學香檳分校UIUC攻讀計算機專業，這幾年過程中，他一直都保持著健身的好習慣，學業和運動雙管齊下！

西瓜哥因為之前體脂太多，所以他遵循的是先減脂後增肌的健身過程，他從230多斤減到170斤，然後再增肌增到如今的190斤左右，體重浮動不太大，但身材卻比以前好太多。下面是前後身材對比，你來感受下！

歸國後，這位93年的西瓜哥如今在自己創業，領域也跟健身有關，是健康食品領域。雖然每天都忙得不可開交，但在固定的時間段去虐肌肉是他雷打不動的日程！

而聽西瓜哥說，他之所以能擁有好身材，除了勤鍛煉之外，高蛋白的健身飲食也是關鍵所在。所以，這位1米92的大小夥子學會了自己做飯，雞胸肉、牛肉、西蘭花、煮雞蛋都是餐餐必備！

別看西瓜哥有一副魁梧的肌肉身材，其實內心裡，他還是一位文藝青年，賣萌神馬的也是相當在行！

別人都說肌肉男一個比一個愛自拍，看來，還真是那麼回事，嘿嘿！

最後我再曝光下大家最關心的西瓜哥的感情問題！

我們常說「帥哥都有男票了」，然而西瓜哥卻是一位筆直筆直的直男，他本人是這樣說的。
