壓縮腿套到底是真有用還是心理作用

天天要聞
2019-12-22

隨著跑步的熱潮，壓縮裝備也隨之興起，各種壓縮裝備在馬拉松比賽中已不鮮見，比如壓縮腿套、壓縮襪，壓縮衣、壓縮褲、壓縮護臂，壓縮裝備對於跑步到底發揮什麼作用呢？是真心有用還是心理安慰作用呢？

壓縮裝備的原理

壓縮裝備能夠起到作用，原理是向單向流動的血管瓣膜提供正壓力。腳踝與膝蓋部位的壓強差別，可以促進血液向心臟迴流，同時在一定程度上對抗地球引力對血流的影響。簡單來說就是用彈性梯度織物產生壓力差，從而更好地促進體內血液和淋巴流動。

在體育運動中，壓縮裝備設計的目的是利用梯度彈性提高運動表現和加快恢復。雖然目前對於壓縮裝備還存有爭議，但是也有相當一部分的研究表明壓縮裝備確實起到一定的效果。

穿壓縮腿套有哪些好處

1.能提高跑步成績

壓縮腿套通過擠壓血管，目的是讓更多的血液和氧氣供應給肌肉，這樣就可以讓氧氣和能量物質更快進入肌肉，同時也幫助代謝廢物和二氧化碳更快從肌肉中排出。理論上說，這樣就有助於提高能量產生效率，讓跑者跑得更快。

值得一提的是，如果你平時就不怎麼跑步，指望穿著一雙壓縮腿套就能跑完馬拉松，這是不太現實的事情。對於能力一般的大眾跑者而言，壓縮襪具有一定緩解疲勞或者說推遲疲勞發生的作用，推遲疲勞發生，相當於就是提高了運動成績。

2.減少肌肉疼痛感

在跑步過程中，腳騰空落地時帶來的衝擊會導致肌肉震顫，這被認為是引發肌肉損傷和引起延遲性肌肉酸痛現象的重要原因。從原理上分析，壓縮腿套通過給予肌肉支撐，在一定程度上可以減少肌肉震顫從來減輕疼痛。

以全程馬拉松為例，每位跑友至少要跑30000步以上才能跑完全程，在這個過程中，每一次著地的地面衝擊力將導致肌肉隨之震動，而這種震動是導致肌肉損傷和肌肉疲勞的重要原因。

壓縮襪的壓縮作用使小腿肌肉在跑步全程中晃動減少，可以在一定程度上降低肌肉疲勞的程度，同時這種外在的環繞式的壓力也使穿戴者對疼痛的本體感覺降低，可以跑得更加順暢自然。

3.加速疲勞恢復

運動後穿著壓縮腿套，通過加速血液流動，有利於清除代謝廢物，並及時引入肌肉修復重建所需要的物質。有一些研究表明，運動員和長時間坐飛機的人在穿了壓縮腿套之後覺得身上的酸痛感減輕了。

跑多長距離需要穿壓縮腿套

1.短距離效果不明顯

較短距離，比如10公里以內跑步使用壓縮腿套似乎用處不是很大，因為短距離跑步疲勞程度較輕。

2.全馬後半程效果明顯

很多穿壓縮裝備參加過多次馬拉松的跑者們表示，大多數人都認為壓縮腿套在前半程感覺效果不是太明顯。不過在後半程會明顯感覺壓縮腿套對小腿肌肉有很棒的固定作用隨著距離的增加和路跑一成不變的姿勢，容易產生疲勞，而這時也明顯體會到壓縮裝備可以提供一定的輔助支撐作用。

3.越野跑效果最明顯

在越野跑賽場上，我們能見到很多跑者都穿上壓縮腿套或壓縮襪，感覺非常正常，可能還是有不少人認為是裝酷。但作為熱愛越野跑選手，認為最主要的原因是越野賽道上會有很多野路，不穿腿套的情況下，非常容易被路邊雜草或樹梢劃傷。

同時在越野跑中，整個腿部在急速和瞬間跳躍的不同變化中，會發現壓縮腿套可以相對減少運動消耗和損傷。如果你去跑山訓練或參加越野賽，會明顯感覺到壓縮裝備對腿部保護有不小幫助。

壓縮腿套對於短距離的跑步比賽成績，並沒有明顯提高及效果。但在長距離路跑和越野跑方面，卻有著不同的保護和恢復作用，比如穩定小腿肌肉、減少晃動，減緩乳酸堆積、給予腿部適當的壓力、緩解疲勞等等。

為什麼有些跑者覺得腿套沒用呢？這主要有兩方面的原因。第一就是腿套本身鬆了或緊了，沒有起到作用；第二個就是跑者本身訓練太少。腿套只是個輔助工具，沒有經過系統科學的訓練是不可能光靠腿套就能全部杜絕後半程抽筋的。
