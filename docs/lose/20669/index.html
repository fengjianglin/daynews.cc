減肥期間如何破解皮膚乾燥和暗淡無光

2019-12-17 850條評論 2,436次閱讀 89人點贊

每日小知識

人瘦了，皮膚會變得鬆弛，乾燥，這些都要在過程中加以預防。

減肥期間，很多小夥伴會出現：整個人就像是被抽幹了水分，看上去就像乾癟的老太婆，沒有一點年輕人的活力光彩！

1、油脂不足，肌膚乾燥。

許多瘋狂的小夥伴減起肥來連一滴油都不碰，最後沒見她瘦多少，肌膚卻由於缺乏了必需的油脂成分，幹得像沙漠。

2、缺乏營養，指甲凹凸不平。

營養不良也會導致指甲凹凸不平哦！這是因為體內缺乏鐵元素、鈣質等蛋白質、維生素及礦物質所造成的。

3、營養不均衡，毛髮缺少製造原料。

製造頭髮的角質蛋白質成分不足，可能會大量掉發哦！另外，由於髮根細胞的代謝也跟著衰退，剪過的頭髮完全沒有變長，發質也跟掃把一樣。

應對方法：蛋白質+維生素A+鋅，恢復美麗！

豬血中既含蛋白質又有鐵，熱量也比較低，另外瘦肉和豆類也富含蛋白質，都可以用於減肥期間補充蛋白質。此外，組成肌膚角質的維生素A和鋅也很重要，可選擇深綠、黃色蔬菜和海鮮湯補充維生素A和鋅。

補充7種營養素，這些營養素可以幫助我們改善膚質。

1.卵磷脂

它是消滅脂肪團的核武器，通過填充和重建細胞壁來修復皮膚組織，促進器官最大程度吸收水分，讓皮膚變得有光澤、年輕、堅韌，使得脂肪團難以凸顯出來。

食物來源：雞蛋、豆製品、花菜、花生及花生醬、橘子、西紅柿、土豆、菠菜、卷心生菜等，每天應至少攝入2000～4000mg卵磷脂。

2.葡萄糖胺

身體狀態不佳，深層皮膚即真皮層會受到損傷。漏了的真皮層當然無法保存住充分的水分，整個皮膚組織就會像堤壩一樣開始漏水，最終破裂。於是，皮膚缺乏彈性，出現皺紋和脂肪團。葡萄糖胺便是修復真皮大堤的磚石，同時還能加固身體的結締組織讓皮膚返老還童。

食物來源：食物中很少含有葡萄糖胺，除非你不剝殼就把龍蝦吃下去，所以需要服用專門的葡萄糖胺補充劑來補充。

3.脂肪酸

不僅能吸引水分滋潤皮膚細胞，還能防止水分流失。

食物來源：橄欖油、紅花油、亞麻油、堅果、種子等，不健康的脂肪多存在於黃油、人造黃油、動物脂肪、油炸食品中。

4.氨基酸

是生命的支柱，也是皮膚生成膠原質、彈性蛋白的必需物質。

食物來源：肉類、家禽、魚肉、乳酪、雞蛋、豆類等，一頓營養均衡、蛋白質來源豐富的餐點能提供皮膚每日所需的9種必需氨基酸。

5.抗氧化劑

能保護表皮和真皮不受外界的損傷，使它們不會變薄，把脂肪鎖在皮膚下面。還能保護皮膚組織不斷裂，以防出現脂肪的小腫塊、小凹陷，讓皮膚表面變得凹凸不平。

食物來源：所有的蔬菜和水果都含有抗氧化劑，但漿果和柑橘類水果是含量最豐富的。一般而言，顏色越深、越鮮艷的蔬果含量越豐富。

6.抗發炎食物

顏色鮮艷、富含抗氧化劑的蔬菜、水果，健康的不飽和脂肪酸，能保護皮膚不因為發炎而受到損傷。

食物來源：豆子、水果、亞麻油中的α亞油酸，魚油中的Ω-3脂肪酸，以及花菜、生菜、菠菜等。

7.維生素B族

能夠有效阻斷脂肪團的進攻，將脂肪團轉化成緊緻光滑的皮膚和結締組織。

維生素B族：包括8種維生素。我們所需的維生素B絕大多數來源於啤酒中的酵母，麥芽、穀物、豆類、深綠色蔬菜、低脂和脫脂奶製品、魚、蛋、家禽等，日常飲食能夠滿足人體所需的絕大多數維生素B，如果能再補充複合維生素B，效果更好。
