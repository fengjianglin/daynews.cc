一雙有5項專利的籃球鞋只要300塊！它會是性價比之王嗎？

2019-12-22 1,169條評論 3,141次閱讀 1454人點贊

Hello大家好，這裡是最懂你的WEN鞋評，我是果子，大家記得點擊關注，不錯過我們每日的更新喲。

好了好了，今天絕對不再拖了，今天給大家帶來的就是Q4的實戰測評，還是先來看看這雙鞋的綜合評價吧。

綜合評價

適合的場地：內外皆可 木地板體驗最佳

適合的人群：體重適中的朋友（60-85KG）

適合的打法：突破型後衛 鋒線

適合的腳型：所有腳型均可選擇 尺碼偏大半碼

產品定位

「PE」版實戰鞋

非性能參考項打分

輕質化：中等偏低

耐磨：高

透氣：較低

性價比：高

舒適度：中等

開箱測評

這雙鞋確實讓大家久等了，其實開箱的時候沒想到大家會對這雙鞋這麼期待。

不過不管是因為這雙鞋所謂「PE」的名號也好，還是大家對新品牌的獵奇的心理，檢驗一雙籃球實戰鞋的好壞，以及適合怎樣的人群的唯一標準，肯定就是實戰。

但是從開箱之後，我也了解了這雙鞋更多的背後的故事，在本期實戰測評中，我會和大家慢慢交流。

廢話不多說，來看看這雙鞋在球場上的表現吧。

因為這雙鞋市場上目前只有43-46碼，所以我們只能選擇最小的43了。

整體來看，這雙鞋還有偏大一點的，前掌的位置有一點空，但是對於整體的包裹影響並不大。因為這雙鞋的鞋楦屬於一個正常偏窄的狀態，所以對於腳面的包裹還是不錯的。

而且開箱的時候我們特意說到過這個鞋帶，這雙鞋的鞋帶是那種比較粗糙的材質，而且幾個鞋帶孔的位置比較集中，所以再把鞋帶拉緊之後，整個鞋面的包裹是相當值得信賴的。

好在這雙鞋的鞋舌比較厚，所以把鞋帶拉的很近也不會覺得讓腳背很難受。同時，這雙鞋完全不需要擔心在打球過程中鞋帶有散掉的風險。這裡插一句，如果匹克的態極-閃現能用這樣的鞋帶，我覺得包裹會好很多。

不過也是因為這樣的原因，這雙鞋在穿脫的過程中會比較麻煩，需要你把鞋帶松的很開才可以。但是這一點我覺得無傷大雅吧。

在鞋面支撐方面，這個織物的硬度還是可以的，球鞋基本不會出現過度的形變，基本處在了一個及格以上，接近優秀的水準。

後跟方面，港寶的硬度以及面積絕對是頂級的，同時外面還有品牌專利的E.G.S。™外置支撐設計，所以後跟的鎖定還是不錯的。不過因為尺碼偏大，所以在運動的過程中，會出現一點點不跟腳的情況。而且因為它的硬度太大了，同時沒有額外的鵝蛋式的填充，所以它的包裹要比鎖定支撐差一點。

接下來我們看看中底。

開箱的時候也說到過，這雙鞋的中底用到的是Q4的專利技術：「KOMpress Technology™」，關於這項科技其實我沒有太多的了解，不過應該就是普通的EVA發泡。

剛上腳的時候這雙鞋是真的磚，特別我們這雙鞋是和庫里6一起錄製的素材，我跟文胖當時就是，今天這是穿著兩塊磚來打球了，回去肯定膝蓋疼。

不過這雙鞋的中底在踩開了之後，還是能稍微感覺到一點腳感的，不過對於庫里6的HOVR只能說好一點有限。後掌的腳感的整體表現像是羅斯9上bounce的表現。

因為文胖在測試緩震的時候，還是表示有一點點震的，所以目測這雙鞋的緩震極限也就是85KG了，而且我們不太推薦有緩震需求，膝蓋有傷的朋友去選擇這雙鞋。

前掌緩震為零，但是由於鞋底比較硬，在彎折的部分會因為結構材質過硬，有明顯反饋，及格偏上。啟動的時候，沒有拖沓的感覺，但是同樣因為前掌的材質過硬，所以場地感相較於我們之前說的庫里6，米切爾這種場地感頂級的球鞋，相對一般。

接下來，來看看它的外底表現。

這雙鞋的外底在內場，尤其木地板的表現，簡直無可挑剔。在做動作的時候，是真正地完全釘死在地板上。

但是，仔細聽這個但是，在前掌帶Q4logo的這個地方，偶爾會出現一些打滑的現象，這個位置也正好是你在做轉身的時候發力的地方，這裡算是一個隱患。

剛剛我在錄製視頻之前，還發現了一個開箱沒注意到的細節，鞋底標註了一個「non-marking」的字樣，字面意思是不留痕迹，但在籃球鞋上，這個特指無痕橡膠鞋底的意思，這樣的鞋底材質就是專業打室內球館的橡膠外底，所以它的外底，在室內的表現會異常出色。但是，因為外底材質偏硬，同時紋路較為稀疏，所以到了外場的塑膠地，尤其是那種比較平滑的塑膠地，估計會很滑。

說到這可能有細心的觀眾會diss我：你開箱的時候不是說，這雙鞋是水泥殺手嗎！這怎麼又只適合內場了。

大家先別急，這雙鞋在內場的表現頂級，但是它也適合水泥地，這樣的橡膠可以說是最適合水泥地的一種材質了。雖然這雙鞋的紋路不深，耐磨性可能相對一般，但是我跟你打個賭，這雙鞋鞋底即使磨平了，去水泥地也一樣好使，你信不？

最後來看看這雙鞋最出色的地方：抗扭。

我覺得這雙鞋的抗扭絕對是頂級了，整體的剛性太足了。這裡要和大家澄清一件事情，這部分並不是碳板，但也不是普通的TPU，這同樣是Q4品牌的一個專項技術：名為T.S.S。/26™的抗扭技術，它是以足部的骨骼數量命名的。而且它的實際面積要比你看到的這部分大得多。它的抗扭絕對是最頂級的水平。

至於鞋墊，中足這部分是一塊硬質橡膠，它提供的抗扭加成可能並不大，但是對於足弓的支撐，簡直太到位了，後面這部分，是PORON®專利發泡材質，是一種可以提供良好彈性並持久保持原形態的材質，俗稱「不變形橡膠」。我敢負責的說，這絕對是超越了一雙500塊以內球鞋價值的。

前掌外側這裡的延伸也是很到位，加上這部分的材質比較硬，所以它的防側翻也不會有翻車的風險。

OK，來看看這雙鞋的打分吧。

整體支撐性：9

總分：81

不知道有沒有細心的朋友注意到，我在這期實戰測評中，有一個詞出現了很多次，就是：專利。

這雙鞋上出現了至少4項專利技術，且不論它的實際效果，這其實是一個讓我感到很驚喜的地方，可以說這是一雙蘊含了大能量的不起眼的籃球鞋。

在這段時間的信息搜集中我也了解到，這雙鞋其實是針對速度型後衛的一款實戰鞋，不過它的中底腳感會讓這一定位稍微有一點點尷尬，或許是我們的腳踝力量以及整體的爆發力還不足以hold它的剛性。

對於Q4這個品牌，我是心懷敬意的，之前也有人科普過，Q4的創始人是前匹克的一位設計師，所以也有朋友親切地稱它為美國匹克。

它真正讓我佩服的點是，這個品牌與球員之間並不僅僅是簡單的僱傭關係，而且合伙人的形式，球員本人會自己參與設計，推廣甚至是售賣。

同樣，在這雙鞋的每個地方都充斥著球員自己的元素。包括我在開箱測評沒有提到過的，鞋盒的正面logo是Em的縮寫，也就是伊托萬-摩爾本人的名字，球鞋上的55號球衣號碼，球鞋上的新奧爾良狂歡節的圖案，以及鞋舌處代表了新奧爾良的「the big easy」字樣等等。

這雙鞋現在好像花300多就可以買到了，從我的角度來看，它的性能，它的意義，它的球星元素，要遠遠超過它的價格。

當然，還是那句話，我們不會盲目地安利任何一款鞋，不過假如你覺得它可以滿足你的需求，它絕對值得你選擇。

本期Q4的實戰測評到這裡就全部結束了，喜歡的觀眾老爺記得點贊投幣收藏支持我們一下，我們下期再見，bye bye。
