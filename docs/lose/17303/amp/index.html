做法：

1、這裡的羊肉呢，需要用到羊身上的羊腰窩肉，這裡的肉，做出來的手抓羊肉最嫩、最好吃。剁成小塊，在水裡浸泡幾分鐘並清洗乾淨，需要用1-2顆香菜，用手掰開就可以了。大蔥也準備好，切成碎末就可以了。

2、需要用到料汁，羊肉做好了以後呢，蘸著料汁吃，這才過癮，蔥姜蒜香菜還有生抽、胡椒粉和芝麻油等，都攪拌在一起，喜歡吃辣椒油的，可以單獨攪拌一碗備用。

3、開始煮羊肉，將肉放入鍋中，加入水，開過之後，撇掉上面的一層髒東西，同時加入料酒、八角大料等，使勁兒燉，轉小火慢燉，直到燉爛，爛到脫骨，或者入口即化，這樣蘸著料汁吃，簡直是一種享受，孩子都可以吃，不用擔心肉塞牙，羊肉是溫補性的食物，冬天建議多吃。

新疆囊包飯

步驟：

1. 首先呢，我們將肋條切成小塊，用菜刀剁開就可以了，然後呢，羊肉需要焯水，去掉表面的泡沫，焯水以後撈出來，並清洗乾淨。

2. 做手抓羊肉呢，新疆那邊都用是適量的麥芽糖，或者用蜂蜜也可以，刷一層在羊肉的表面，放入燒熱的油鍋中，當炸至金黃色的時候，撈出來，瀝幹上面的油分。

3. 接著呢，在炒鍋里留下適量的油，不用太多，加入蔥姜蒜爆香，然後呢加入羊肉和清水，大火燉開轉小火，燉至羊肉軟爛，一般燉2個小時，然後呢，關火燜30分鐘就可以了。

提示：

裡面的花椒大料呢，如果在煮的時候沒有放入料盒裡面，在熟了以後呢，用漏勺撈出來，扔掉就可以了。

3 .烤好的新疆饢切成8瓣，放入一個大盤子中，將羊肉放在饢上面，然後呢，澆上一些羊肉汁放在上面，加入孜然粉和辣椒油或者蔥絲等，就可以吃了。

營養均衡，吃著好吃，沒有比這更好吃的了。

烤包子

做法：

1. 做這道美食需要先和面，準備麵粉加入清水用筷子攪拌均勻，之後呢，分幾次加麵粉和水，攪拌好了以後呢，用手揉搓，直到麵糰光滑，蓋上保鮮盒，恆溫下醒發30分鐘左右。

准好洋蔥丁，切成很碎的碎末，還有薑末，準備適量的食鹽，將他們放在一起並攪拌均勻。羊腿肉切成很小的丁，加入孜然粉和香油以及食鹽等攪拌，腌制10分鐘。然後呢，倒入腌好的羊肉中混合均勻，美味的料就做好了。

2. 像我們平時揉饅頭一樣，分割成均勻的劑子，用擀麵杖壓扁，中間呢，我們放上剛才腌制好的肉餡兒，表層刷上一層蛋液，就像咱平時做千層麵包一樣，烤出來金黃酥脆可口。

3. 在烤盤上鋪上錫紙，一是為了乾淨，二是錫紙導熱性很強，可以節約時間，刷一層食用油，上下溫度200度，放上半成品，在一般15分鐘左右，就可以拿出來了。

提示：

烤箱的溫度可以自己控制，最好不要太高，容易烤糊，也不要很低，出現不熟的現象。比新疆饢還好吃，這裡面還有羊肉呢，真香。

椒麻雞

做法：

一聽這個名字，咱就知道裡面得有花椒和食鹽，塗抹在整隻雞上面，腌制40分鐘左右。

2. 將準備好的八角大料以及蔥姜等放入鍋中，加入清水，大火煮開，煮出香味兒，雞肉放進去一起煮，煮熟以後呢，放入盤中晾涼。

雞湯千萬不要倒掉，倒入切好的大蔥的盆中，這樣的話，大蔥就入味兒了。放入食鹽和花椒油浸泡，調味。

雞肉裡面已經加入了食鹽，建議如果再加，親手嘗一下鹹淡，因為是椒麻雞，吃起來麻麻的感覺，一定要多放一些，越多越好。超市裡有賣的，一瓶可以用好幾次。

洋蔥或者大蔥切絲，雞肉戴上一次性手套撕開，攪拌均勻，就可以吃了。如果不過癮，可以用手抓著吃，不過呢，最好用筷子。

我感覺比口水雞還過癮，遇見這樣的美食，可不要錯過。
