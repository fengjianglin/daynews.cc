二胎媽媽堅持健身肚子竟越減越大，生娃三年後仍像孕婦

2019-12-07 1,748條評論 3,561次閱讀 1351人點贊

據江蘇公共·新聞頻道《新聞360》報道：很多產婦生完孩子後，腹部會鬆弛，也就是俗稱的”大肚子”。很多人認為，這是正常現象。可是，南京的王女士，生娃三年後，肚子依然大如孕婦，即便每天健身、收腹，也始終瘦不回去。最近到醫院檢查發現，原來是得了嚴重的腹直肌分離，而這種病，60%產婦都會得！

【產後”大肚子” 二胎媽媽堅持健身竟越減越大】

南京市民王女士，三年前生二寶時，體重達到170斤，生下寶寶後，體重下降到140斤，但是半年過去了，肚子依然沒有變小。她開始每天健身、收腹，堅持了半年，非但沒有任何效果，肚子反而越來越大，像懷孕六個月左右，有時候上公交車的時候，還會有人給她讓座。

【腹直肌分離在產婦中的發生率高達60%】

王女士被診斷為腹直肌分離。醫生介紹，根據上海華山醫院截止到2019年的數據研究，腹直肌分離在產婦中的發生率高達60%，其中30%的產婦，一年之後無法自己恢復，變成永久的”大肚子”。不僅如此，腹直肌分離還會導致腰酸背疼、內臟脫垂等併發症。

醫生介紹：每個人肚子前面都有兩塊肌肉，叫腹直肌，懷孕的時候，由於腹腔不斷地變大，把腹直肌撐開了，很多人在產後腹直肌不能複位。這種腹直肌分離的情況下是不建議運動的，因為運動時會很用力，腹腔內的壓力會進一步增加，加重腹直肌分離。

【肥胖、多次妊娠為高危因素 半年後是關鍵干預期】

王女士隨後接受了手術治療，將腹直肌複位，去除多餘脂肪，目前，王女士的肚子明顯縮小，身材如願恢復到產前。

不過，醫生介紹，手術治療只針對中重度腹直肌分離、沒有再次生育需求的女性。輕度腹直肌分離，可以在專業醫生指導下，做康復訓練，進行改善。醫生提醒，肥胖、多次妊娠都是腹直肌分離高危因素，產婦產後半年，如果肚子依然無法恢復、甚至鬆弛，就需要就醫。目前已有近一半的省級婦幼保健院、省會城市的婦幼保健院開設醫療美容科，可以到這些機構的醫療美容科尋求幫助。

據了解，江蘇省將在各級婦幼保健機構推進醫療美容科建設，同時成立產後修復專業學組，培養更多專家，重點針對女性產後修復開展服務。

(江蘇廣電融媒體新聞中心記者汪樂萍、王苗軍 報道)
