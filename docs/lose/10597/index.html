楊蓉腿部綳直，雙腿不留縫隙，想要擁有就練習推牆瑜伽

2019-12-08 2,121條評論 3,447次閱讀 423人點贊

我們在站著的時候，往往腿型也能夠看得很清楚，有些人在站起來的時候，小腿就會與小腿之間有空隙，大腿也會有一些空隙，但有的人小腿和大腿之間就沒有空隙，其實這就是腿型的不同。楊蓉在站起來的時候就能夠看到她的大腿是沒有空隙的，這就是我們平常所要擁有的腿型，但是想要擁有總要付出一些什麼，那麼就努力的去練習。

​

練習總要是有一個過程的，它並不是說你想要練習就能夠很快到達想要的效果，如果你想要慢跑練習，那麼首先你需要了解慢跑的技巧，同時應該進行去練習，但是一定不要渴望有飛快的進步。其實想要鍛煉並不是那麼難的，難的是找到合適自己的方式，因為你並不是很清楚自己的身體適合什麼樣的練習，那麼該怎麼辦呢？這時候我們就可以從最簡單的開始學起，最簡單的開始學起能夠讓我們不再盲目練習，同時這個動作還可以適合所有人。

​

下面就和大家介紹一下這個動作，其實這個動作就是推牆瑜伽，因為它的練習並不是那麼的劇烈，如果我們是上班族或者是學生，這個動作對我們來說剛好合適，每天只需要拿出十分鐘的時間就能夠讓我們放鬆身體，也同時還可以緩解工作帶來的腰酸痛，能夠讓我們的消化能力有所增強。

​

這個動作的好處非常的多，能夠讓我們有效的拉伸腿部的肌肉，尤其是膝蓋周圍的肌肉，因為在平常的訓練中我們是無法練習到這些小部位的，而通過這個練習可以有效的拉伸此處的肌肉，同時還可以讓我們的腰部背部不再那麼酸痛，有緩解作用。

​

在生活中非常容易見到的就是便秘，消化不良，而這些通過這個動作可以有效的緩解癥狀，因為在這個動作中能夠讓我們的腹部收緊，同時還能夠讓我們的全身得到鍛煉，這樣就會使人血液循環系統得到優化，那麼我們的內分泌狀態就會逐漸變得健康，那麼我們的新陳代謝速度也會慢慢增快，所以就可以讓便秘消化不良這種問題輕鬆解決。下面就和大家分享一下這個推牆瑜伽動作，想練習的小夥伴們一起練習吧。

動作：推牆瑜伽

​

1.上身保持挺直，雙手自然的放在身體的兩側，身體呈站軍姿式站立，我們可以通過山式站姿為起始姿勢。

2.緩慢的抬起雙臂，使雙臂平行於地面，同時還要與肩膀的高度一致。在伸直後，用力的壓向牆壁。可以彎曲左腳，向前進一步，右腳向後放，這樣可以有效地伸展腿部，同時還可以讓我們施加更大的力量對於牆面。如果可以的話，要將你的腳緊壓地面，進行深呼吸。

3.收緊你的腹部和臀部，慢慢的彎曲手肘，將腿部綳直，嘴部呼氣，有效的練習10到15秒。

4.雙手慢慢的伸直，壓住牆壁，同時右腳也要伸直用力的蹬地面，持續10到15秒，然後換另一邊練習。

​

在這裡也要說明一點，如果你的膝蓋比較疼痛，那麼應該及時停止訓練，不要勉強自己，自己的身體應該有所了解才對。

​

在練習過程中要保持自然的呼吸，不要憋氣或者說深吸一口氣，那樣不太好，尤其這個動作主要練習的就是腿部周圍的小肌肉，所以在剛開始練習的時候可以尋找一位專業的小夥伴來監督你的練習。這裡還有一個小建議，如果你想要練習的話，可以在洗完澡之後再進行練習，因為洗澡後肌肉處於完全放鬆狀態，練習這個動作效果會更加明顯。

很高興能和大家分享健身相關的小知識、小技巧，希望大家關注我的賬號，我將持續發布健身相關內容，希望大家多多點贊，轉發和評論。

你的努力，超乎你的想像。
