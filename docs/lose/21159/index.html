認識冥想（4）：冥想的時候，我們要怎樣做？2

2019-12-19 285條評論 2,338次閱讀 1355人點贊

紫蘿

冥想中我們將意識轉向自我，去觀察、感受和體會自己的身體、呼吸、情緒、思維、感知等。而這些觀察、感受和體會是遵循一定的原則進行的。沒有了這些原則的冥想，也就不能被稱之為冥想了。

原則二：無目的。無論你是帶著什麼目的去做冥想，或者希望通過冥想得到什麼特定的結果。在開始冥想的那一刻，就要把所有的目的擱置一邊，因為冥想本身是沒有目的。冥想關注的是過程，而不是結果。即使是放鬆，也不是它的目的。冥想的核心是每一刻意識與自我的交流，交流本身就是目的。每一次冥想都是與自我間的一次聯繫，一次對自我的認識。從這個角度而言，沒有「好的」或「壞的」或「成功」或「不成功」的冥想體驗。冥想練習之後，通常會有放鬆或清明的感受，但並不是每次的冥想都會有這樣的體驗；沒有出現這樣結果的冥想並不意味者是一次「失敗」的冥想。只要是進行了一次練習，就是與自身多了一次接觸。不要評判你的冥想練習，好或不好，有沒有進步；每次冥想結束後，注意一下你的感受與坐下時有什麼不同就好。

冥想過程中，我們訓練將意識轉向自我，去觀察、感受和體會自己的內在世界，但是意識卻不會總是聽我們的話，它常常會遊離出去。出現意識遊離的時候，常會感到沮喪，這種沮喪其實是一種目標沒有完成時出現的感覺。也就是說，我們下意識的把專註設成了目標。冥想的確會訓練和提高我們的專註力，但提高專註力不是冥想的目標。在冥想中，如果將專註設定為目標，專註力反倒難以得到鍛煉和提高。所有人都會出現意識遊離。覺察到遊離，覺察到沮喪感，輕輕放下「目標」執念，簡單的把意識拉回到我們選擇去關注的事物上來。這一走，一回的過程，就像是一種肌肉的拉伸，才是真正的訓練了專註力。

原則三：是發掘的過程，不是尋找的過程。冥想時，我們會深入的發掘自己的身體感受、情緒、行為和思維模式。但不是去尋找特定的狀態、感受或一個設定好的自我。所以，沒有所謂冥想「應該」有的體驗、狀態或感受。每個人的冥想體驗都是不同的，每個人每一次的冥想體驗也會是不同的。冥想之間不具有任何可比性，沒有哪一次冥想更好或哪個人的冥想更好之說。冥想中的自我發掘和發現是通過體驗、感受和覺察來實現的。這與我們常規的經由分析、評估和學習來認識和理解事物和自我的方式不同。這裡我們需要聚焦的是我們的感官系統，調動的是我們的好奇心。細心的體會每種感受，觀察每段思緒，看看那裡有什麼，這裡有什麼，它們是什麼樣子的。

總結一下，在冥想中，我們練習放下評判、放下介入，放下目標，用好奇心去發掘和發現自己內在的世界。除了在正式的冥想中練習，在日常的生活片段中，我們也可以練習運用這些原則：嘗試暫且把評判放在一邊，去觀察事物的本身面貌。在向一個目標努力的過程中，暫且忘記目標，聚焦到做事的過程，發動好奇心，發掘事情本身的魅力，而不僅僅是它能帶來的結果。看一看在這樣的視角下，自己的生活會有什麼不同。其實，這三項原則與我們日常的思維定式是相悖的，最是難以做到，我們不可能，也不需要完全按照這樣的原則去生活。但是，在一定程度，一些片刻可以超越常規思維定式，進入一個新的視角，會給我們不一樣的自由感。這就是冥想對我的意義。
