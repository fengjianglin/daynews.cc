「功法」意念、感覺、勁的體驗——你不知道的太極門內“立體樁”

2019-12-10 1,456條評論 4,252次閱讀 2918人點贊

太極拳中的樁功訓練，是使身體本來具有的某些能力被改造或消除，同時使太極拳技法所需要的某些新的能力能夠被逐漸地開發、建立起來，並得到強化與固化，形成新的自然之能。太極拳中的站樁訓練應該是勿忘勿助、順其自然、適可而止。

樁功或站樁，一般是指靜功而言，是傳統武術中十分流行而且效果很好的訓練方法。其功用主要是通過比較長時間的訓練，在相對固定的形體動作上，以意念暗示去指導身體內景外情的變化，並逐漸使這些變化固化。太極拳中的樁功訓練，是使身體本來具有的某些能力被改造或消除，同時使太極拳技法所需要的某些新的能力能夠被逐漸地開發、建立起來，並得到強化與固化，形成新的自然之能。樁功在某些方面的作用比較顯著，比如放鬆、穩定性、協調性、全力法（全身無處不彈簧）、陰陽變化等。

北京吳式太極拳門中幾個傳統的、有代表性的樁功練習方法是：立體樁、馬步樁、弓步樁、坐步樁（七星樁）。不同的樁功練習有不同的特定目標，練習時要有針對性。必須明白樁功訓練只是輔助練習，不要過於強調，喧賓奪主。太極拳訓練中，盤架子永遠是第一重要的功夫。初級階段里先介紹意念應用比較簡單的立體樁，其他在後面的訓練階段中介紹。

需要注意的是在樁功訓練中，訓練量的控制十分重要。有人練習站樁時總是先給自己規定時間，也有人認為站得越長越好。事實上，硬性規定練習時間是一個常見的錯誤，因為站樁時最忌諱的是不能保持正確、準確的意念指導。人的意念是很難長時間集中的，越靜越難。過去叫「心猿意馬」，就是說這種心神、意念的不穩定性。如果意念不正確，就收不到正確的結果，甚至可能得到錯誤的結果。因此，當用一個固定姿勢站了一段時間後，意念會開始不能按要求集中、即守不住，俗稱「跑了」，是很正常的。當然，「跑了」可以再拉回來，就是鎖心猿栓意馬。但是如果老是跑，就不好了。因此在訓練中，這個意念不能集中的感覺，就可以作為站樁時間的參照。也就是說，如果站樁時發現意念老是不能集中，則就應該停止了。這時如果因為規定時間未到而繼續，可能就會產生負面效果。在訓練的初期，能夠守住意念的時間可能很短，越練能守住的時間越長。但是站樁也並不是時間越長越好，即便是能夠守住意念，站得時間過長，容易引起身體的局部僵化，進而產生惰性。因此，太極拳中的站樁訓練應該是勿忘勿助、順其自然、適可而止。

立體樁

立體樁，也叫一字立體樁，是強化太極陰陽變化的站樁練習。其基本體態是雙腳分開與肩同寬，全身直立站穩，兩臂抬起在胸前抱成圓圈，稱臂圈，同時身法九要中的各個要點都需做好。按手掌的朝向，分為坎卦樁與離卦樁兩個練習方法。當兩掌均為陰掌，掌心均朝向圈內時，為坎卦樁。當兩掌均為陽掌，掌心均朝向圈外時，為離卦樁。

坎卦樁（正、側面）

坎卦樁取八卦中坎卦之意，坎為水；坎中滿，卦中三爻為陰陽陰，即外陰包著內陽，陰為柔，陽為剛，是外柔內剛。坎卦的屬性如水，外柔是說能夠隨形就勢，內剛是說不可被壓縮。坎卦樁以陰掌為基本手型。練坎卦樁的主要目的，就是要讓身體里產生這種外柔內剛的勁，並通過陰掌體現出來。

站樁時的意念：想像兩臂中圈抱著一棵大樹，在臂圈中，兩手指尖相對，中指之間還有一寸左右的距離相隔。意念放在背後的夾脊穴，再向兩邊分開，串列到兩肩井穴、兩曲池穴、兩勞宮穴、兩中沖穴，似乎兩臂被抻長，使兩中指將將可以合攏。略想膻中穴有微微一空之感，同時感覺兩中指尖微微一碰，有鬆一口氣的感覺。當兩手中指尖一碰上，馬上想所抱的大樹長大變粗了，使兩手中指之間又自然分開了一寸左右的距離。這時將意念返回到夾脊穴，重新再使兩手中指合攏的意念走一遍。反覆重複這個抱樹、樹變粗、再抱、再變粗的過程，直到感覺意念無法集中為止。當這種意念的指導練習達到一定程度後，可以將放在背後的意念從夾脊穴移到命門穴，這時抱樹的感覺從一個平面圈變成一個球。這樣就可以形成在各個方向上的立體效應。

站樁時的體內感覺：當意念放在背後的夾脊穴時，有涵胸拔背之感。當意念移到命門穴時，除了有涵胸拔背之感，還有松腰收腹、拉背弓的感覺。當意念在兩臂上串列時，有沉肩墜肘、兩臂抻長、兩掌微微內旋之感。當意念引導兩中指尖相觸時，有氣貫指尖、使之強壯有力之感。當練習到一定程度後，越抱樹越粗，越粗越抱，這會使雙臂內側的氣感增強，以及由此產生向內的環抱之勁，進而引起向外的微微膨脹之勁。

坎卦樁中勁的體驗方法

勁的體驗：坎卦樁以訓練外柔內剛的勁為主，當練習到一定程度時，可以藉助外力體驗這種勁。體驗的方式有很多種，最基本的方法如圖所示：自己以坎卦樁的方式站好，保持全身放鬆，並以站樁訓練的意念操控。請一訓練夥伴站在對面，將其雙手放在你的右小臂上，這時你的左臂可以保持在原位或自然放下，然後夥伴以不變的直力推你。你必須要保持提頂、涵胸、拔背、沉肩、墜肘、松腰、溜臀、裹襠等基本身體形態，同時保持意念的串列。最重要的是，不能將意念放到與對方雙手接觸的位置上。要做到這點，就必須保持站樁時所練習的意念，強制自己的意念轉移。如果各方面都能做好，則對方就推不動你，而且你越放鬆，對方就越推不動。反之，如果想在與對方的接觸點上與對方直接對抗，則馬上會全身僵硬，特別是兩臂用力，肩、腰上發緊。如果對手比自己強大，則越用力抵抗越被動。在試勁的過程中，如果對方突然將手撤離，從而導致自己身體重心不穩而前傾，則說明自身沒有充分放鬆，而且意念也沒有真的從接觸點上移開，導致在這點上有與對手直接對抗之力。如果所有要點都能夠做得正確，則對手會有一種有點推不上勁的感覺，或者感覺你並沒有與他相爭，但是他的勁有點用不上的意思。這時如果你能夠松到底，腳下輕靈，身體可以有水中浮球之微微浮動的感覺，可以導致對手身體隨之傾斜、失中。這裡最重要的是從放鬆中體驗動態的掤勁。

離卦樁取八卦中離卦之意，離為火；離中虛，卦中三爻為陽陰陽，即外陽包著內陰，陽為剛，陰為柔，是外剛內柔。離卦的屬性為火，外剛是說高溫可傷人，內柔是說火焰其中為虛空。離卦樁以陽掌為基本手型。練離卦樁的主要目的，就是要讓身體里產生這種外剛內柔的勁，並通過陽掌體現出來。

站樁時的意念：想像兩臂圈撐成為一環狀，中間為真空狀，故壓力從四面八方一起向內壓。在臂圈中，兩手指尖相對，中指之間還有一寸左右的距離相隔。意念放在胸前的膻中穴向後推背後的夾脊穴，以抗外壓；再向兩邊分開，串列到兩肩井穴、兩曲池穴、兩勞宮穴、兩中沖穴，似乎兩臂有抻長之感，使兩中指相對微微分張，使整個臂圈有向四周抵住壓力、外撐之意。感覺兩掌心微凸，兩中指尖將將一分，有要吸一口氣以補充外爭之感覺，這時馬上想臂圈中更空了、外壓增加，使兩手中指之間又微微合攏到一寸左右的距離，有點臂圈被微微壓縮之意。這時將意念返回到膻中穴向後推夾脊穴，重新再使兩手中指分張、臂圈外撐的意念走一遍。反覆重複這個外撐、圈內空、再撐、再變空的過程，直到感覺意念無法集中為止。當這種意念的指導練習達到一定程度後，可以將放在身前的意念從膻中穴移到神闕穴，以神闕穴向後推命門穴，這時外撐的感覺從一個平面圈變成一個球。這樣就可以形成在各個方向上的立體效應。

站樁時的體內感覺：當意念放在胸前的膻中穴向後推背後的夾脊穴時，有涵胸拔背之感，同時有虛心實腹之感，氣沉丹田。當意念移到神闕穴時，除了有涵胸拔背之感，還有松腰收腹、拉背弓的感覺。當意念在兩臂上串列時，有沉肩墜肘、兩臂抻長、兩掌外旋推撐之感。當意念引導兩中指尖相分時，有氣貫指尖、使之強壯有力之感。當練習到一定程度後，越撐圈內越空、外壓越大，越空、越大，則越撐；這會使雙臂外側（在圈內）的氣感增強，以及由此產生向外的支撐之勁，同時引起向內的微微環抱之勁。

勁的體驗：離卦樁以訓練外剛內柔的勁為主，當練習到一定程度時，可以藉助外力體驗這種勁。體驗的方式有很多種，最基本的方法如下圖所示：自己以離卦樁的方式站好，保持全身放鬆，並以站樁訓練的意念操控。請一訓練夥伴站在對面，將其雙手放在你的右小臂上，這時你的左臂可以保持在原位或自然放下，然後夥伴以不變的直力推你。你必須要保持提頂、涵胸、拔背、沉肩、墜肘、松腰、溜臀、裹襠等基本身體形態，保持意念的串列。最重要的是，不能將意念放到與對方雙手接觸的位置上。要做到這點，就必須保持站樁時所練習的意念，強制自己的意念轉移。如果各方面都能做好，則對方就推不動你。這時，只要你的手掌有向外翻轉之意，則對方就會感覺抵抗不住；如果你有兩手分張之意，則對方就會感覺以他自己之力將他向其左後上方推起。如果對方想要離開，則你可自然跟進，將其推出。反之，如果想在與對方的接觸點上與對方直接對抗將其推出，則馬上會全身僵硬，特別是兩臂、兩腿用力，肩、腰上挺力。如果對手比自己強大，則很難推動對方，而且越用力推、自己就越被動。在試勁的過程中，如果對方突然將手撤離，從而導致自己身體重心不穩而前傾，則說明自身沒有充分放鬆，而且意念也沒有真的從接觸點上移開，導致在這個點上形成與對手直接對抗之力。如果所有要點都能做得正確，則對手會有一種如果他不動，則無事；只要他一用力推，馬上會感覺一種堅不可摧的力量，使其如同觸電般被反彈回去。

以上是坎卦樁與離卦樁的基本練習方法，在實踐中，為了能夠運用自如，有一些變化也應該經常練習體驗，比如單臂圈、立圈、混合圈等。單臂圈是說只用一條胳膊形成臂圈，從形式上講，只有半個圈，而另一半全憑意念。立圈是說將胳膊立起來，使平的臂圈成為立圈。坎卦樁的立圈練習形式：提頂、松肩、墜肘，兩手舉起、兩掌心向內抱，松腰、溜臀、裹襠，兩膝微曲。意念從百會穴到夾脊穴，經肩井穴、曲池穴、勞宮穴，向後經命門穴到尾閭穴，向前微微托起。離卦樁的立圈練習形式：提頂、松肩、墜肘，兩手舉起、兩掌心向前扶抱，松腰、溜臀、裹襠，兩膝微曲。意念從百會穴到夾脊穴，經肩井穴、曲池穴、勞宮穴向前推撲畫一個圈，經命門穴到神闕穴，小腹微收，身體略沉。

混合圈是說兩條胳膊做不同的圈，一邊為柔中剛的坎卦圈，另一邊為剛中柔的離卦圈。坎卦與離卦樁的混合圈練習形式：兩腿前後分開一小步，提頂、松肩、墜肘，右臂抬起、掌心向內抱，意念想右中指與夾脊穴相連成一個平圈，如坎卦樁；松腰、溜臀、裹襠，兩膝微曲，左臂彎曲下落，掌心向下，意念想左中指向前、向上、再向下與夾脊穴相連成一個立圈，如離卦樁。坎卦之平圈與離卦之立圈相交，形成一個剛柔相濟、鼓盪激勵的大球。事實上，這類變化更接近於實際應用。

從立體樁的訓練中，最重要的是體驗陰陽掌中的勁以及基本的陰陽變化。陰陽掌中的柔中剛、剛中柔的勁，是技擊技術中最基本、最常用的勁。技擊時，陰掌，吸為合，呼為開。陽掌，吸為開，呼為合。基本的陰陽掌變化，即是動靜、虛實、吞吐、開合、剛柔變化。有時可以說，所有技術的基礎就是陰陽掌之間的轉換。這種轉換，開始是通過具體陰陽掌之間的轉換練習來體驗。當能夠真正掌握這種內在的轉換，外形動作就不重要了，勁在實戰應用時，也就無所謂形式上，或者說形體動作上的陰掌、陽掌了，只是心意中的轉換即可。

前面講過，所有訓練中的練習方法都是疊加的，因此，這裡介紹的樁功練習會延續到頂級階段中。需要說明的是，我們這裡介紹的幾種藉助練習同伴進行的勁的體驗練習，都是為了配合能夠更清楚地理解單人站樁練習。雖然我們在這裡介紹了這類練習方法，但事實上，真正的練習應該放到中級階段到高級階段的訓練中去。因為在初級階段中的練習者，還不具備所需的基本能力，如果練習，不會取得好效果，甚至還會產生副作用。
