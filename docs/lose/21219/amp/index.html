馬拉松賽道出神鞋！這些2000元左右的頂級跑鞋，你願意為誰買賬？

天天要聞
2019-12-18

老鐵們的本座又上線啦！

不知道老鐵們發現沒有，現在買雙鞋老費勁了！又是科技又是配置又是用戶體驗啥的，誰家不整點玄乎的東西都不好意思說自己的賣鞋的，這各種騷操作下來感覺自己壓根就不是買鞋，而是在買車啊！

就比如今天要說到的這四雙鞋子，均價2000大洋，玩的就是前衛、玩的就是噱頭，說到這估計老鐵們的腦子裡面已經跳出來那麼幾雙，那麼今天本座就帶老鐵們盤點一下這幾雙兩千大洋的神仙跑鞋。

第一雙：李寧飛電䨻競速跑鞋

李寧這個冬天的日子可是不好過啊，不但寒了自家消費者的心，還差點兒把自己整涼涼。回頭看看李寧的2019年，真的是充滿爭議的一年，這種爭議是把雙刃劍，名聲大噪的代價是消費者信任的缺失。

就拿今年推出的這雙飛電䨻舉例，人們討論最多的不是䨻科技有多牛逼，而是這雙鞋2099元的定價是真牛逼！這波操作直接讓一句吐槽成為了「定律」：以前沒錢買李寧，現在沒錢買李寧。這話沒毛病！

本座先前也做過飛電䨻的開箱評測視頻和圖文，評論區的負面評論還是不少的，䨻科技加碳板加MONO紗就值2099元嗎？當然不值！但是老鐵們有沒有想過，這雙馬拉松跑鞋最值錢的地方到底在哪呢？

說直白點，這雙鞋最值錢的地方就是在於其造成的「轟動效應」！並非本座在洗李寧，Nike的質量和消耗速度是出了名的渣，但是Nike不就是憑著一次次的轟動效應讓自己越做越大嗎？這也是一種手段。

第二雙：Nike Zoomx VaporFly Next%

你看看，這就是李寧的成功之處！說到飛電䨻就必須要說Nike這雙最頂級的馬拉松跑鞋，這不就和匹克態極那套做法一樣一樣的嗎？說到態極就要評價一下Boost，只不過匹克玩的隱晦，李寧過於直白。

Next%的定價也是2099元，不過大家對這個定價貌似並沒有什麼爭議，其中的原因本座就不廢話了，實在憋不住的老鐵可以寫在評論區。這雙鞋在某些第三方平台也有不少的折扣，1700左右就能拿到。

這雙鞋本座也做過開箱視頻和圖文，感興趣的老鐵可以前往主頁查看。總的來說這雙競速鞋確實是相當帶勁，如果說本座穿著䨻是想跑的話，那麼穿上Next%就是想飛！鞋子碳板部位設計真是太厲害了。

鞋子雖好但並不是人人都能穿，這雙鞋的重心比較高、吃地面積比較小，所以養生跑者看看就完了，進階跑者和精英跑者可以小心一試，據說這雙鞋跑全馬的「成本」是50元/公里，所以非土豪用戶就……

第三雙：Adidas Alphaedge 4D

這雖然是一雙概念跑鞋，但是Adidas在去年秋天就開始大規模市售了。你看看Adidas的牛逼之處就在這裡了，嘴上說的我這是概念鞋啊！然後瘋狂的往出掏各種配色，這是欺負本座不懂概念是啥咯？

這雙鞋目前在市面上的標價是2499元，而且極其考驗審美，在本座看來，花兩千五買這麼一雙丑出新天際的跑鞋真的是一件很難理解和想像的事情，但是Adidas卻十分理直氣壯：我這可是概念鞋啊！

這雙鞋子本座瞄了很久了，可是一直都下不去手買來做視頻，因為本座覺得除了那長的像蜂窩煤一樣的中底，這雙鞋是真沒啥好說的。而且對於Adidas倡導這種噱頭本座真的是不感冒，看一看就完了。

但是我們誰都不能否認Adidas通過這雙鞋所表達出的一種態度，那就是現代跑鞋科技的發展遠遠還沒有到達終點，「超前」這種事情也從來不會消失，這對於充滿未來感的老鐵們來講不失為一件好貨。

第四雙：Asics Metaride

做個小科普，在Asics跑鞋家族中，但凡帶上「Metra」這個前綴的話那必然就是高級系列，而且這類跑鞋也會充分體現日本人在創造中的奇葩屬性，在看這雙跑鞋的圖片時就會有一種莫名其妙的感覺。

分離式大底的設計本座沒少見過，但這種分離一般都是橫向上的分離，而這雙鞋子直接來了一個縱向的分離式中底，也就是雙層中底，這造型看上去就想崴腳啊！還有這牛逼上天的鞋頭，左腳指月嗎？

這雙跑鞋目前的標價為1890元，這是啥意思？加層底子就加一千嗎？不過相對於營銷玩的賊溜的前幾家來說，這個定價還是相當良心了，而且遇到活動也會降點價意思意思，所以暫時沒那麼招人恨。

客觀來說，Asics做跑鞋還是比價實在的，當家科技捨得用、好的配置捨得上、遇上活動也捨得降價，性價比方面還是比較有保障的，另外Asics的跑鞋矩陣也很完整，想買什麼樣的跑鞋基本都能買到。

以上就是今天的全部內容。總的來說能把一雙跑鞋的價格干到兩千的絕對不是科技和配置，更多的還是一種營銷策略和噱頭，而往往這些策略和噱頭能打動不少人的心，一傳十、十傳百，效果就達到了。
