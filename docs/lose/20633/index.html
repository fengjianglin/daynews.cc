瑜伽中的唱誦"AUM",意義太深了

2019-12-17 2,134條評論 4,911次閱讀 1028人點贊

在瑜伽課堂上，課前或者結束時，學員們經常在老師的帶領下唱誦”AUM”,我們只是在唱誦時覺得有一股暢意，其實其背後的深義我們還不怎麼明白，今天我們就來講一講這個”AUM”。

《楞嚴經》的《觀音耳根圓通章》中就提到過，觀音菩薩教大家從聲音走進去的方法：”反聞聞自性，性成無上道”。就是專門要找些聲音，進入那個聲音，讓聲音完全穿透自己。

這個聲音我們不要去找，我們血液流動的聲音，我們整個身體聲音的融合和合一的就是這個 “AUM”。

當你用你的雙手捂住雙耳，如果捂得夠死，沒有了任何外界的聲音，並且閉上雙眼，關閉你頭部所有的其他感官，這時候你靜靜地聽，你會突然發現，裡面竟然有一種天然的”AUM”的聲音，它是你所有身體的聲音融合在一起最後展現出來的一個振動。

其實這個聲音也是宇宙的聲音，當火山爆發前或者大地震前，地球也會了出類似的聲音，地球內部中心的聲音也是這個，西藏傳統做法事、佛事時要吹響海螺，那種海螺口你可以聽到來自深海的這種隆隆聲音，海螺吹出的聲音也是這種AUM聲。說白了，這個聲音就是宇宙的聲音，它是高頻高能的，它將啟動你全身的能量，它能自發地激活中脈的能量，當你聽到這個AUM聲，你的整個人都好像激活了。

當你能夠在藏傳佛教念經的會場里聽他們念一個鐘頭的經（他們念經的聲音都趨向於非常深沉的AUM）,你就會發現整個人變得火熱起來，這個AUM能夠激勵能量向上運動，即使你沒有跟著念，只是坐在那裡聽，你整個人都會開始變得振動起來。

《瑜伽經》裡面講這個聲音其實與你的身上的七個脈輪有關，人身上從海底輪到頂輪有七個脈輪，每個脈輪都有屬於自己的聲音，七個脈輪的

聲音正好是七個音階，從最低起，分別是：

AH對應於底輪（不要發出R，而是Ah如spa中a的）；

A對應在萊登中心（發出able 加長a）；

E對應於太陽神經叢（發eve中加長e）；

I是心輪（長i）；

O對應喉輪（如open中的長o）；

U對應冠輪松果腺（發出true中的u音）；

M頂輪

這七個脈輪的合音正好是AUM。

你不斷地在你的內在唱誦AUM，而不是外在，以你的內在唱出這種聲音里，就意味著你的脈輪能量不得不向上升，因為你在震動第七個脈輪，你內在的能量將自動地被吸引到七，自動地升起來。

《瑜伽經禪修要訣》里說:當你的能量達到七，你內在的意識就會覺醒，這是一個必然的過程,而當你從七開始的時候，那麼一到六的障礙就會被消除。這點非常奇怪，如果你從一開始，那麼意味著你將面臨二、三、四、五、六所有的障礙，但是你一開始先從七開始，你會發現所有的力量會被自動地向上吸引，那個障礙會變得不那麼阻礙了。

這個就是《瑜伽經》給我們的一根有力的拐杖。
