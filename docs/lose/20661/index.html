暢通任脈 、督脈的搖櫓式運動健身方法

2019-12-17 166條評論 1,217次閱讀 158人點贊

櫓——使船前進的工具，比槳長而且大，安在船梢或船旁邊，用人來划動。

大家都知道，人體的任、督、沖、帶脈,皆為奇經。

人體奇經有八,即任脈、督脈、沖脈、帶脈、陽蹺脈、陰蹺脈、陽維脈、陰維脈。這八脈的循行沒有共同規律,多為「別道奇行」,又與臟腑沒有直接的絡屬關係,相互間也沒有表裡相合,故與十二正經不同,所以稱「奇經」。因數為八,又稱「奇經八脈」。任、督、沖、帶為其中之四。

搖櫓式任脈鍛煉方法

任脈,起於中極之下,在女子相當於胞宮。任脈為一身「陰脈之海」,足三陰經在小腹與任脈相交,使左右兩側之陰經通過任脈而互相聯繫。因此,任脈對陰經有調節作用,故稱「總任一身之陰經」。另外又有調節月經、妊育胎兒的功能,又說「任主胞胎」。

督脈,起於中極之下,在女子相當於胞宮。督脈為一身「陽脈之會」,手足三陽經都與督脈交會於大椎。督脈對陽經有調節作用,稱「總督一身之陽經」,又督脈起於胞宮,下出會陰,沿背側正中線上行,經頸部進入腦內,並有支脈絡腎,所以說「督脈屬腦絡腎」。腎生髓,腦為髓海。因此,督脈基本上反應腦與脊髓的生理功能和病理變化。

搖櫓式督脈運動方法

沖脈,亦起於少腹以下,在女子相當於胞中。有調節十二經氣血的作用,所以沖脈有「十二經之海」和「血海」之稱,亦主婦女月事。上述任脈、督脈和沖脈,在女子均起於胞中,故有任、督、沖「一源而三歧」的說法。這三條經脈主要調節人體的生殖功能。

鍛煉人體的奇經八脈對健康至關重要，長期伏案或久坐的人很難活動到位。所以，推薦一個鍛煉身體的好方法，搖櫓運動健身方法。海邊長大的人不會陌生，在老電影里也經常看到漁民下海，划動小船，或單槳或雙槳，穩穩地操控船兒前行，任憑風浪起，一路向前進。

那麼人經常習練這一動作，也可以起到穩定身體的平衡作用，還可以增強體質。

運動要領——

1、搖動單櫓，單櫓一般放置於船的尾部，人可以通過左搖或右搖，多是側身用力。當人體面向船體的右側時，當右腿在前，人體的右側身體多用力，當人體面向船體的左側時，當左腿在前，右腿綳直一些，人體的左側身軀多用力。配合呼吸，吸氣是往回收，呼氣時用力搖而向外推。

2、搖動雙櫓，雙櫓一般放置於船體中間的兩側，左右各一。人可以站立於小船的中間位置，兩足與肩膀同寬，平行站直，目視前方，平穩均勻地向前搖或向後搖動雙櫓，左右可以有適度的微調動作。

搖櫓式運動起式

3、當人體用力前搖的時候，身體自然弓背，背部相應拉抻，運動我們的督脈系統，頭部前傾並抬起。背部脊柱有效得到鍛煉。當人體用力後搖的時候，身體自然後仰，胸部挺拔，拉抻腹部，任脈系統得到有效的鍛煉。身體一起一伏，一前一後，運動有序，自然而然起到平衡陰陽的功效。人體八脈是一個整體聯動，間接起到調理效果。

4、平時多習練這一動作，意念配合最重要，想像自已就是一位漁家，有出海的身臨其境之感，這時運動的體驗和功能顯現最佳，鍛煉目的就可以達到，同時可起到事半功倍的效果。
