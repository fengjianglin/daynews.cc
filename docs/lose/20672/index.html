男生打籃球時，大多數人會說的口頭禪”，“我的、我的”上榜

2019-12-17 378條評論 1,317次閱讀 239人點贊

有人說 ，世界上最動聽的聲音就是球過網的唰唰聲。男生之所以喜歡打籃球，並不因為它只是一項運動。它是有生命，有靈魂的~（這個理解不了，不怪你…..）籃球這一項運動，確實在生活中給了我很大的激勵。

如果大家觀察仔細的話，就會發現男生在球場上，是經常會說這3句話的~所以今天小編要跟大家聊的話題是：男生打籃球時常說「這3句口頭禪」，「我的，我的」上榜，最後一個沒人能扛得住！

第一句就是：「我的， 我的」！

在籃球場上出現失誤，這是一件很正常的事情。比如說沒有接住隊友的球，投籃沒有投中等等，這些小失誤都是在所難免的，也是可以理解的。

但是有時候就會心生愧疚，覺得自己對不起隊友的信任什麼的。於是就會經常很抱歉的跟隊友說：「我的，我的」，哈哈，是不是還有一點小可愛呢~

第二句就是：「好球」！

打籃球時的「快樂源泉」，當然就是進球了。只要是能進球，就會得到一片的讚許聲~每一次在進球的時候，男生都會不由自主的喊一聲：「好球」！不得不承認，這一句話在籃球場上，真的是對一個男生最大的贊同了…這也是讓你拚命進球得分的動力！

第三句就是：「你打球好像xx坤」！

哈哈，這一句嘲諷的話，真的是太狠了，這誰能扛得住啊？就算是你關係再好的夥伴，如果在球場上，你對他說出這句話，我敢保證他會暴走的…..畢竟這一句話真的是太傷人了，奉勸大家最好不要輕易說出口，不然一不小心就會挨揍的~

哈哈，小夥伴們，你們對這件事怎麼看呢？不知道你在打籃球時，經常說的口頭禪是什麼呢？快來跟小夥伴們一起分享一下吧~
