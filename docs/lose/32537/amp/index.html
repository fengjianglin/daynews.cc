「高級進階」 一陰一陽之謂道——閑談一下羽毛球的技術和體能

天天要聞
2019-12-25

技術和體能，猶如易經中的陰陽，相互促進、相互制約，共同決定了我們羽毛球的總體水平。在不同的階段，其顯隱不定，分別呈現為當時的主要矛盾，主宰著期間個人的主攻方向，隨著時間的推移，又會由於矛盾的轉化主客易位。一個人的羽毛球水平提升，就是這麼不斷的此起彼伏，螺旋式上升。

在業餘四五級之前，技術是限制的主要矛盾，我們需要投入大量的時間精力掌握和熟練基本技術動作，技術的熟練度基本就決定了整體羽毛球水平。在五級之後，體能的重要性逐漸上升，並隨著本身總體水平的提升不斷提高其重要性。在接近專業水平的層次上，體能和技術互相制約和促進，越是高級的技術需要更強的體能去支持，更強的體能又促使自身進一步提高其技術。兩者就在這不斷地相互作用中此消彼長，一點點的提升我們的羽毛球水平。我們如果能夠認識到現階段的主要矛盾和短板，針對性的投入時間和精力，就可以得到最大化的進步。

一陰一陽之謂道，陰陽非道，其所以陰陽者，道也。在技術和體能之上，還有所謂的意識，就如同「道」一樣，決定著技術和體能的具體表現，它在頂級選手的對決中表現得最為明顯。林丹和李宗偉就是這一層次的代表。

兩個人的對決，就是在最高層次的較量，通過對自己當下技術和體能的清晰把握，通過對對手狀態的解讀，在每一球的回合中不斷調整，一點點的絞殺對手。於無聲處聽驚雷，每一個平平淡淡的回球都有著千錘百鍊的技戰術意圖和強悍的體能基礎。他們的對決，真正是達到了「技進乎藝」的最高層次，是羽毛球之道的具象化。兩人的互相對抗和促進，也是陰陽之道的真實表達。

在另一個角度上來說，林丹在中國，具有良好的訓練環境和對抗體系，本身又獲得榮譽無數，不用在意小陣仗的勝負，就可以看到他很少出戰，只對大比賽的桂冠有所牽掛。李宗偉在大馬，個人訓練環境還不錯，平時訓練缺乏像樣的對抗，個人生涯也缺乏大賽的勝利，就採取以賽代練的方法，積极參与各種比賽，彌補對抗性的缺陷。兩人的職業生涯，讓人有瑜亮之嘆，卻也如同陰陽大道一般讓人著迷。

不斷磨練自己的體能和技術，不停提升自我的羽毛球意識，永不止步，樂在其中。來源中羽：球齡七年半
