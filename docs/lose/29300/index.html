常練這組經典的支撐體式，強化臂力核心鍛煉，輕鬆甩掉蝴蝶臂

2019-12-21 1,640條評論 3,270次閱讀 760人點贊

常練這組經典的支撐體式，強化臂力核心鍛煉，輕鬆甩掉蝴蝶臂！

很多美眉在減肥的時候特別容易忽視手臂上的肉肉，也就是所謂的拜拜肉，不僅會造成一種粗壯魁梧的既視感，特別毀形體，也會引發一些身體疾病，不利於身心健康。

造成拜拜肉的因素有很多，無外乎有以下幾點：1、無法控制飲食者（喜歡胡吃海喝，特別是高蛋白高脂肪食物）2、懶於運動者（愛吃又不愛動，卡路里沒有消耗則轉換成脂肪）3、不良習慣者（長期伏案、不良的站姿及坐姿、經常性彎腰拱背等）

手臂粗則氣質塌，手臂是否纖細也是會影響到氣質的。楊穎，做為《奔跑吧》的常駐女嘉賓，在生完娃後選擇迅速復出賺奶粉錢，顏值在線的她身材仍舊保持苗條勻稱，纖細小腰也尤為搶鏡。

前段時間楊穎的1組美照曝光，身穿一襲珊瑚色的弔帶薄紗長裙，纖細的雙臂令人驚羨，修長筆直的美腿若隱若現，不愧是混血的美人，簡直太迷人了。

這肉可是易長難消，特別是手臂上的肉肉又是最為難減的，想要消除拜拜肉，那得多練練支撐體式，今天跟大家一起來學學經典的支撐體式——鶴蟬式+半蓮花式，想要減手臂的美眉記得收藏起來。

1、鶴禪式+半蓮花式

鶴禪式（Bakasana）+半蓮花式，高階的複合瑜伽體式之一，可在完成半蓮花式後，雙手打開同肩寬，注意手肘微彎，掌心推地支撐起軀幹維持平衡，故也是經典的支撐體式。膝蓋彎曲使得膝關節貼於雙臂後側，腳跟盡量靠於臀部，腳背朝向地面，腳尖朝後綳直，收緊臀部肌肉朝向後方，脊椎向上拱起，始終保持腹部內收緊。

體式功效：

A.加強腰腹部核心力量，減少多餘脂肪，刺激腹部器官，改善內分泌系統，提高代謝能力，緩解便秘及腸胃疾病，強化腰部曲線。

B.增強核心臂力鍛煉，消除手臂上的贅肉，強健肘及腕關節，緊緻肌肉線條，使得雙臂變得纖細。

C.滋養神經系統，促進血液循環，消除疲勞及焦慮等，緩解失眠癥狀，使得其充滿活力。

習練要點：

A.站姿，雙腿打開同肩寬，腳趾外旋45度，雙腳跟相對，雙臂自然放於體側。

B.屈雙膝，坐骨下沉，身體下蹲使得小腿後側貼於大腿內側，雙手貼於體側撐地，完成瑜伽蹲式。

C.吸氣，左腿抬起疊放於右大腿根部，雙臂前舉，收腹，軀幹前屈使得手肘微彎，掌心朝下貼地。

D.掌心推地，收臀，軀幹離地抬起，同時背部向上拱起，頸部上仰，維持體式20秒。

多練支撐瑜伽，加強臂力核心鍛煉，輕鬆甩掉蝴蝶臂，記得每天堅持！
