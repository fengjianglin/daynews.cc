該如何改正這個問題，擁有仙女肩頸呢？

答案就是練背，增強背部核心力量。大多數時候我們體態不對就是因為背部肌肉力量偏弱，不足以支撐起正常的脊柱形態，背部肌肉力量增強，我們自然能保持脊柱挺直，肩膀也會自然打開，從而保持肩頸垂直。

下面推薦幾個有效的肩背練習動作，可以增強背部核心肌肉：

1、划船運動：

划船機絕對是健身房裡被低估的無名英雄，划船半小時能消耗300大卡的熱量，是燃燒卡路里的優質選擇。不僅如此，划船相比跑步對肌肉的刺激更加突出，能夠有效調動人體九個主要的肌肉群和百分之八十五的肌肉組織，使我們的背部更強壯，雙臂也更加緊實。

注意要點：握住手柄，拉伸時將其拉至下腹部，使上背部充分收縮，收回時將手柄儘可能靠近輪軸，使背部充分拉伸，注意保持一定的速度，建議每次20分鐘以上，一周三次。

2.山羊挺身：

山羊挺身是鍛練下背部的一個有效方式，主要是煉下背肌群（豎脊肌），提高腰部力量。

注意要點：調整好羅馬椅之後，雙腳貼在器械下端的位置，使器械上端接近大腿根，雙手抱頭或交叉於胸前，身體上部直立下壓至接近地面，然後回正身體與器材平行，在這個過程中，注意保持上體直立。建議每組12～15個，每次四組。

3.引體向上：

引體向上我們都不陌生，是男孩子們必做的體測項目，其實它對女孩子來說也同樣有效，能夠充分運用自重，鍛煉我們背部的背闊肌和大圓肌，實現緊緻塑形，提升肌肉力量。

注意要點：雙手抓握，挺胸，腰腹、臀部始終繃緊，垂直上下，充分調動背部肌肉，保持肩部背部始終發力，雙腳可以保持懸空，要注意拉起時呼氣，下落時吸氣。建議了2～3天一次，每次按照自己的身體狀況適量來做。
