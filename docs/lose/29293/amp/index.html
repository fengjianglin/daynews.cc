一盤普通的捲心菜，加兩瓣蒜一勺橄欖菜，就會好吃得不得了

天天要聞
2019-12-21

最近有一則新聞很火爆，說是我們中國人有4億人購買了運動裝備，但真正經常運動的不足3億，也就是有1億人買了運動裝備後沒有運動，很多人辦了健身卡最後變成了洗澡卡，也就是買了運動裝備卻很少運動，辦了健身卡卻很少去健身，只是假健身。還有一個現象是很多人買了運動裝備之後馬上又買了自拍設備，打算在朋友圈好好晒晒自己的運動照。不過我可以自豪地說，我應該不是那1億分之一，已經連續辦了15年的健身卡，堅持一周三次去健身房出汗，同時在健身房裡有了一批固定的健身夥伴，還有一個極好的教練。

跟健身一樣，很多人經常說著要減肥，但是真的要管住嘴邁開腿的時候，往往堅持不下來。其實要保持良好而穩定的身材，還有很重要的一方面是良好的精神面貌，除了經常運動和出汗，管住嘴也是非常重要一個環節，管住嘴不一定要長期節食，不吃油膩葷腥，其實含蛋白質和脂肪的食物一定要吃，而且每天要吃，但是一定要適度，每餐吃七、八分飽，長期堅持你就會發現自己的體重始終能保持在一個穩定的範圍內。當然，如果一天一杯奶茶，隔三岔五吃宵夜，每日無肉不歡，不吃得十分飽不肯離開餐桌，那是肯定會長肉的。

說了那麼多，今天來推薦一個我經常吃的蔬菜吧，一家三口四個菜，兩葷兩素比較合適，兩個葷菜中一個魚一個肉比較好，葷菜的量準備得少一點，而蔬菜的份量則要大一點，把素菜做好吃了，家裡人都愛吃，也就不用你天天苦口婆心勸家人多叫蔬菜多健康了。今天這個蒜香橄欖菜炒捲心菜就是最近我家常也非常愛吃的一個菜，只要加一勺橄欖菜，就能讓本來清淡寡味的捲心菜炒出來有一股清甜的味道來。當然炒以前，先加兩三粒壓扁的蒜瓣煸香，這道捲心菜會更香更好吃。有興趣的朋友跟著我一起試試吧。

【橄欖菜炒捲心菜】

主料：捲心菜半顆350克，橄欖菜一大勺30克

輔料：油20克，蒜3瓣，鹽1小勺

做法：步驟1、新鮮的捲心菜半顆就夠了，剛好能炒一大盤。炒這個菜要準備點蒜瓣更香，橄欖菜則是用來入味的。

步驟2、捲心菜清洗乾淨後切絲備用，蒜瓣拍扁，橄欖菜挖一大勺備用。

步驟3、熱鍋冷油，下入蒜瓣，小火扁香，油少一點就夠了，橄欖菜里還含有較多的油。

步驟4、下入一大勺橄欖菜一起炒出香味

步驟5、加入捲心菜絲一起翻炒

步驟6、待略略炒軟

步驟7、再加入半碗水，一起燒開，捲心菜不易軟，要加點水一起煮

步驟8、略略加半勺鹽，因為橄欖菜是鹹的，要注意鹽分的控制。

步驟9、加蓋燜煮一分鐘，開蓋翻炒均勻就可以起鍋了。

雖然只是一盤捲心菜，因為有蒜瓣和橄欖菜的加入，而變得非常好吃。

分類: 減肥
