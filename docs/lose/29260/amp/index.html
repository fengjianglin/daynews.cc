巴菲特成就千億帝國的秘訣是什麼？查理.芒格：重複是絕對核心

天天要聞
2019-12-21

當查理.芒格一家住在明尼蘇達州的一棟房子里時，為了提高孩子們的網球技術，芒格給家裡的網球場添置了一台發球機。

然而，使用發球機最多的並不是孩子們，而是芒格。他一有時間，就跑去網球場，擺好發球機，然後開始無休無止的網前截擊練習——每當遇到離網很近的球時，人們都會本能地想要扣殺，但常常因為站在離網很近的位置，一下自己沒掌握好力度，就可能把球擊到網上，或是打到球場外面去。

很多人覺得這太難了，但芒格卻樂此不疲，因此很快就完全掌握了網前截擊的技術，比如採用大陸式握拍方法，因為大陸式握拍法既可以打正手截擊也可以打反手截擊；身體的中心需要稍置於前，不允許兩腳的腳後跟死死地貼在地面上，一定要處於稍微提起的狀態，等等。從那以後，芒格的孩子們都害怕成為父親網球場上的對手，因為他的網前截擊技術實在是太棒了，沒人能夠抵擋。

芒格很早就知道：要達到人們所希望的那種有深度的「熟練」，重複是教導的核心。有時候，你會遇到一些困難的問題，這些問題的困難程度迫使你變得非常勤奮——進行非常勤奮的練習，直到把問題解決。在這個練習的過程中，你必須讓心完全靜下來。

芒格的搭檔巴菲特也從小就認識到了這點。

巴菲特第一次為威斯徹特社區送報紙時，早上四點就起床了，早飯也沒吃，就匆匆跳上公交車，趕到社區門口。四點半左右，送報紙的車來了，卸下來好多捆報紙，上面有編號，可他根本不懂是什麼意思，也沒人教他。他們只是給了他一個本子，上面是訂戶姓名和地址，就走了。巴菲特忙活了幾個小時，把報紙分好疊好再捆好。最後一數，不夠，原來有人路過順手拿走幾份，他太忙了根本沒注意。

因為是第一次送，路線也不熟，走了很多冤枉路，從早上4點一直忙到上午11點，整整忙了7個小時才送完。這樣的效率肯定是不行的，於是巴菲特啟用了過去一年送報紙的經驗，開始研究一條高效的高效送報路線。

威徹斯特包括了5幢建築，面積達27.5英畝。5幢建築中有4幢相鄰，有1幢獨立成戶。送報區域包括了2棟以上公寓樓，要穿過教堂大街，還有馬林大街和沃里克大街。同時，沃倫還要把報送到威斯康星大街那邊的一小片獨家住宅。

在熟悉了整個社區的樓盤後，他把報紙分為兩半，一半放在最高的8樓，另一半放在中間的4樓。然後，他從8樓開始一層一層地往下送報紙。為了節省時間，他總是站在樓道的一頭，從一大摞報紙抽出一份，對摺一下，然後一卷，在大腿上拍扁，手腕往後一勾，像扔飛鏢一樣，把報紙一個個準確地扔到訂戶的門口。

別看巴菲特年齡小，他最遠能扔20米，而且扔得特別准——正好扔到人家門口一兩尺遠的地方。要知道，很多人家有訂牛奶的習慣，而送奶員總是一大早就送來牛奶，因此巴菲特扔報紙的時候就需要小心了，既要扔得離門口夠近，又不能碰倒牛奶瓶。這可是個相當有難度的技術活，需要練習很多次才能熟練。

就是靠著這種重複的練習，巴菲特達到了那種有深度的熟練，讓他在三年時間裡攢下了2000多美元。1945年的2000美元，相當於現在的6萬多美元。

在芒格看來，天才之所以成為天才，並不是因為他生來就比其他人優秀，而是他一直在進行重複的工作——勤奮地練習。當然，這種重複的練習都是有目的的練習，即有著定義明確的特定目標，而且在練習過程中足夠專註，並能過反饋來準確辨別自己在哪些方面還有不足，以及怎麼會存在這些不足、如何改善這些不足。這也是很多成功人士的不二法門。
