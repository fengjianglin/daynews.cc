大體重胖子不建議跑步減肥？你該淘汰這個教練，他out了

天天要聞
2019-12-18

立筆之初，要先聲明，一點都沒有歧視胖子的意思，相反的，本著希望胖子們生活更美好的出發點，寫的這篇文章。

周邊的朋友很多都是胖胖的姑娘和仔，可能真的是心寬體胖，胖子們多數內心純真、心地善良，喜歡和這樣的人做朋友，而他們當中有很多人想要跑步減肥，可是教練卻不支持，所以表達一下自己的看法。

一、明確下什麼是大體重基數的胖子

國際上常用BMI來衡量人體肥胖程度和是否健康的重要標準。具體來說，BMI=體重（公斤）÷身高（米）的平方。如果BMI介於24-28之間，屬於超重；而如果BMI>28，就屬於肥胖，就是我要說的大體重基數的胖子 。

二、胖，也沒有什麼不好

如果胖沒有給你帶來什麼副作用和不便之處，而你也還算享受自己的樣子沒有很介意，那就胖著好了，沒有必要刻意去減肥。

但是，如果下定決心想要減肥，那就務必認真對待，思考一下整體的計劃，找到最適宜自己也最健康的方式來進行。

大體重基數的胖子們，如果減肥，建議先去醫院做一個專業的身體檢查，排除身體的疾病隱患，聽取醫生的飲食和運動建議。畢竟前不久台灣藝人高以翔錄節目時劇烈運動猝死給所有人都敲響了警鐘。生命，有時候真的很脆弱，沒有什麼比活著更好，胖點、瘦點、懶點，勤快點什麼的，相比起來都顯得不那麼重要了。

三、大體重胖子減肥不難，減到位比較難

作為一個大胖子，減肥的第一要件，應該是學會控制自己的飲食，控制自己少吃一點、控制自己抵制高糖高脂的食物，千萬不能吃垃圾食品。這一步做到了，減肥便成功了70%以上。

而另一方面，大體重基數的胖子，減起來其實更容易獲得成就感，體重基數大，絕對重量掉的自然也會快，但是要瘦身到標準體重範圍，卻需要相當的堅韌和毅力。

四、大體重胖子不能跑步減肥嗎？

再說胖子的運動，隨便搜索一下，全網都會建議大胖子不要跑步減肥，給出的理由基本上都是「傷膝蓋」。因為存在這樣一個說法，跑步每邁出一步，膝蓋便要承受體重×7倍的壓力。所以長期、長距離的跑步，對膝關節傷害很大，而且還是不可逆轉的。

上述論據並沒有問題，只是沒有說完事實的全部。

比起走路，跑步時膝蓋的壓力確實更大，但是人在跑步時步幅更大，腳和地面接觸時間也更短，所以膝蓋承受壓力的時間更短，幾乎只是走路時的一半，所以，跑步時膝蓋承受的總壓力，比起走路時還要小！

除此以外，跑步能夠促進膝蓋的血液流動和細胞再生，從而起到保護膝蓋的作用。

人體，天生就是適合跑步的。

——這句不是我說的，這是跑過18次馬拉松的布魯爾說的。

所以，不管你是不是大體重的胖子，不管你是基於什麼目的想要去跑步，那就去奔跑吧，這是我們與生俱來的權利。

如果你的教練告訴你說，會傷膝蓋，真的，你可以換掉他了，他的理論不夠專業，知識技能可能out了。他應該做的是指導你正確的跑步姿勢和呼吸。

五、關於大體重胖子的運動提醒

運動的目的，是為了健康，但是運動要適度，拒絕一切矯枉過正的行為。莫要急於求成，做劇烈運動，尤其是大體重的胖子，一定注意運動過程中的安全。

因為，胖子運動起來對心血管、血壓都會造成壓力，如果在運動或休息時曾出現過短暫的眩暈、黑朦和心慌, 必須引起足夠的重視；若運動中出現胸痛，應立即停止。
