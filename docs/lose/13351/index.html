「單腿下蹲訓練」跑步者們提高核心力量的制勝關鍵

2019-12-12 687條評論 4,269次閱讀 1283人點贊

跑步愛好者們一定了解，跑步並不是印象中雙腿交替前行的刻板活動，而是一項真真正正的全身運動，只有擁有了強大的核心肌群力量，才可以幫助跑步者們提高訓練效率，避免多處運動損傷。

跑步是一項全身運動

單腿下蹲訓練能夠很好地改善跑步者們的平衡力，增加臀部髖關節穩定，挑戰你的核心穩定性。而由於單腳支撐的原因，你會需要動員更多運動單位來維持軀幹的排列穩定防止肩膀跟髖的旋轉。

即使跑步者們在日常間隙只堅持做這一項訓練，對核心力量的提升也會十分顯著。

核心力量對跑步的重要性

如果你希望跑步之後不會腰酸背痛、疲憊不堪，那你一定要試試看這一項簡單有效的訓練。

訓練步驟

（1）身體站直，將左腿伸直後抬起離開地面。

左腿伸直

（2）緩慢的彎曲右腿膝蓋，將身體蹲下去。

雙手平舉緩慢下蹲

（3）下蹲到最大程度後起身回至原始狀態。

下蹲至最大程度

（4）單邊腿各做15組。

注意點

（1）腳跟需要保持著地狀態，不可提起。

（2）手不可以扶地，需保持平舉狀態。

（3）若剛開始訓練無法十分標準的完成，可以循序漸進，進行多次練習。
