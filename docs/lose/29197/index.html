這幾類人，最容易練出太極好功夫。看看有你嗎？

2019-12-22 615條評論 2,241次閱讀 97人點贊

當今太極熱，「太極」二字一再活躍於各大網路媒體與實體，刺激著人的眼球，挑動人人的神經。

藉助網友的總結，練太極有以下好處：改善神經系統；增強心臟功能，改善循環系統；擴大肺活量、提高人的平衡能力，防止骨質疏鬆；具有健美作用等等。

有些人不禁好奇太極的神奇，尤其嚮往太極功夫的柔中帶剛，招式動作的飄逸自然，所以他們通過電話諮詢，但他們往往也會問到同一個問題：我怎樣能更快的進步？我什麼時候能學會或者學完太極？

太極作為中國傳統文化的重要組成部分，歷史創拳到現在，形成了豐富的太極理論和動作招式。在太極拳的每一招式中都蘊含著不同的意義。

所以練太極不是一朝一夕的事，對於想急於求成的人來說，羨慕別人有一身的功夫，一時興起學太極，確實太極需要更多的人關注和了解，但是缺乏能堅持到底的人。

所以，什麼樣的人，容易練出太極好功夫？

以下幾點，可能是關鍵：

態度

今天的太極拳館的環境很靜，並且每個練功廳也都會相對空曠，高山流水的曲子能讓人靜下心來，聽著音樂，做著動作，真的很愜意。既然來鍛煉身體，那就請放慢節奏，一步一步來認真學習太極，這是前提。

其次，不要自滿。假如一下子學了好多套路，自認為這下把太極全學完了，其實不然，太過於求成，就會產生自滿情緒，練太極也是講究悟性的，初學者只是學動作，基本要求就是動作熟練即可，之後才會一點點的糾動作，你只是囫圇吞棗的學完，根本學不到實質性的東西，產生的自滿更會讓你停滯不前，慢慢的體會一招一式中蘊含的東西，你才能體會到練太極的妙處。

再次，放下功利性的目的。同時，要學會堅持，可能你資質很好，很快就學會了簡化套路，但如果不堅持的話，不用多長時間你就把你辛苦所學的還給了教練。

素 質

熱愛中國傳統文化

太極拳早已是公認的中國武術中的優良拳種。太極拳也可以說是中華傳統文化的精髓，太極拳的拳理拳法、理論體系與中國傳統文化中的儒、道等思想體系關係密切，也可以說太極拳融合了傳統文化中的儒道思想。所以一個人要真正在太極拳方面有所造詣，他在對傳統文化的理解和認識方面也應該有一定深度。當然，作為一個中國人，即使不練拳，也應該對深厚的傳統文化有發自內心的熱愛，以及發自內心地保持對傳統文化的敬畏。

尊師老師，善待同門

每個習練太極拳的人都是跟著老師學的（自學成才的還從來沒有見過），中國素來有尊師重道的優良傳統。拜師學藝，自然也有同門。一個人如果不尊敬老師，不善待同門師兄弟，則說明其道德品質有問題。有很多人為了追求名利，不惜背棄師門，甚至弄虛作假，捏造事實，本來是跟A學的，為了抬高輩分，非要說自己是跟A的老師學的。也有人心胸狹窄，不屑與同門為伍，甚至視同門師兄弟為仇人。這樣的人怎麼能學好拳呢？學好拳又有什麼意義呢？

對太極拳有宗教般的熱情：

對一件事有強烈的興趣，自然會深入去研究它。一個在太極拳方面有深厚造詣的人，一定是對太極拳極其熱愛的。如果對太極拳有宗教般的熱情，把它當成一種信仰，那麼一定能學好。如果一個人為了獲取名利而去學拳練拳，期望練出深厚功夫，幾乎是不可能的。

持之以恆與虛心好學

能夠堅持長久堅持，大概是天底下最難的事情，一百個學習太極拳的人，最後能堅持下來有所成就的，不過三兩個而已。太極拳本身已經比較難學，而堅持不懈地練拳，則更不容易，何況現代人生活節奏快，事情多，很難靜下心來堅持練拳。所以一個人如果有志於在太極拳方面取得成就，首先要努力克服外部困難，克服自身惰性，做到每天練拳。每天練拳的時間不一定很多，但盡量要做到每天都練。只要長期堅持，自然有收穫。

虛心好學，深入研究在任何一個領域要能有所成就都得深入研究，太極拳自然也不例外。

太極拳是一門實踐的藝術，需要多多練套路，套路熟練了，動作用法明白了，可以通過推手來進一步加強實踐，積累經驗。太極拳也是一個文化拳，有著相對完整的理論體系，因此也應該多研究這方面的文字資料，加深理論修養。只是需要注意一點，當今關於太極拳的書籍資料如汗牛充棟，良莠不齊，需要注意甄別。
