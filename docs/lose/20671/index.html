國貨=土？國貨黑科技爾克奇彈勢不可擋

2019-12-17 151條評論 1,424次閱讀 185人點贊

在NIKE與阿迪兩大巨頭鋪天蓋地的推廣下,我們似乎已經習慣了更多的去討論NIKE與阿迪的鞋子如何好,然而我們國產品牌真的有那麼差勁嗎?前有李寧悟道國際時裝周大放異彩,後有國貨黑科技爾克奇彈橫空出世,熱銷海外,極致軟彈的舒適腳感讓外國友人也大呼驚奇。

「彈出新高度」是爾克奇彈跑鞋的霸氣宣言。這樣的霸氣來源於國內知名體育品牌鴻星爾克最新推出的α-flex國貨黑科技材料。眾所周知,「柔軟、回彈、支撐」三大功能點相互牽制,太過柔軟必然要犧牲回彈和支撐。

如何將它們完美的結合於一雙鞋上,一直是跑鞋科技的難題。鴻星爾克研發團隊經過多年嘗試,終於成功研發出一種半透明、柔軟的α-flex智能流體材料,爾克奇彈在傳統的「非牛頓流體緩震材料技術」的基礎之上,將α-flex智能流體材料混合不同化學製劑, 經600次顆粒發泡實驗,60000次耐折測試,最終實現彈適自如的緩震體驗。


如果說α-flex國貨黑科技是爾克奇彈跑鞋的內涵,是有趣的靈魂,那麼獨特的設計就是奇彈跑鞋的外觀,是好看的皮囊。難能可貴的是,有趣的靈魂萬里挑一,好看的皮囊也非千篇一律。在鞋子的設計上,鴻星爾克緊緊把握時代潮流的風向坐標,設計出更符合國人審美和運動需求的產品。

不羈之風,拒絕平庸,爾克奇彈黑騎士創造性地採用了全鞋面反光黑紗設計,將低調與張揚兩種極端的反差完美的融於一雙鞋上。正常環境中,黑騎士低調百搭,一旦到了黑暗中,在低光的照射下,黑騎士鞋面會呈現出燦若繁星的視覺效果,清新靈動的星光斑點,跳脫出彩,極其張揚,自帶焦點,為你的夜跑保駕護航。黑暗中最亮的星不是別的,是你腳下這雙爾克奇彈黑騎士。

如果說爾克奇彈黑騎士的滿天星設計已經讓你讚不絕口,那麼接下來的爾克奇彈二次元的新奇創意,肯定也會讓你大呼驚奇的。動漫世界裡的新奇紀元,黑與白的衝突較量,是鞋身線條的靈感來源,交錯縱橫,熱血正義,或個性或美好或憧憬,肆意飛馳二次元空間,釋放中二少年探索世界的熱情和活力。

從國貨黑科技到鬼馬新設計,我們看到的是鴻星爾克作為大國品牌的底蘊和創新。作為一個深耕運動市場多年的國產品牌,鴻星爾克深知國貨崛起絕不是說說而已,而是要靠實力,要有拿得出手的科技,也要有第一眼就讓人淪陷的顏值。

近年來,隨著國家扶持體育產業政策的不斷深入與落實,國內各大體育品牌迎來了新一輪的高速增長。消費市場升級促使品牌從成本和規模競爭進入到品質和技術的競爭。鴻星爾克憑藉近20年在運動體育領域的科技積澱和對潮流趨勢的敏銳嗅覺,潛心打造國貨黑科技爾克奇彈系列產品,開闢出屬於自己的品牌升級之路,向世界發出中國品牌的澎湃之音。
