「高級進階」 業餘高手的單打策略——快、穩、變、狠

2019-12-24 101條評論 1,626次閱讀 559人點贊

對於業餘中水平較高的球友來說，單打我覺得應該遵循上述的幾個原則，這僅僅是我的一家之言，提出來僅供參考，也歡迎大家拍磚。

快：天下武功，唯快不破。我覺得，任何對抗性的競技運動，最重要的是這個「快」字，在絕對的速度優勢下，其它任何技巧都是蒼白無力、鏡花水月。單打對決中，能形成對抗的前提，就是兩者的速度在一個檔位上。就拿開車來比喻吧，你加速最多能上4檔，碰到了能在5檔情況下打球對手，即便你能勉強增高轉速，抵擋一時，略作抵抗，高轉速下的車子是容易發熱、出狀況最後導致拋錨的。體現在羽毛球場上的快，包括步速和球速，步速慢了，永遠只能勉強去夠球，無法充分發力、變線，只能掙扎著多拖幾個回合；球速慢了，無法造成對方的被動，不能爭取到有利的進攻時機。在我看來，步速快，是首先立於不敗之地，球速快，才是克敵制勝的關鍵。林丹和李宗偉在北京奧運的終極對決，就是林丹通過長期的調養和催眠，達到了個人速度上的巔峰，完全在高於小李子一個檔位的狀態下運轉，才能壓制對方兩局只拿了一局的比分。不過，我要強調一下，這個快，不是一味的快，而是在你能完全發揮自己技術前提下的快，讓一個短跑運動員上場打球，他的速度是夠快了，但在那種速度下，基本的技術動作都無法很好完成，也就談不上克敵制勝了。專業選手中，能發揮技術的前提下能達到的最快速率，林丹可以全程保持5檔，絕殺時甚至能衝上6檔，剩下的小李子，小陶子等等，最多只能保持大半場，速度上的優勢，是林丹能成為絕對No.1的殺手鐧。另外，據說速度超過了一個極限，就可以追溯時空，逆轉因果，這一傳說中的境界曾經有位大神周杰倫（阿傑？）在籃球場上創造（詳情請諮詢電影網…….）。速度上的差異，在籃球場上也有類似的說法，比如說到CBA和NBA的差異，號稱CBA是走著打球，NBA是跑著打球，速度的決定性差距，體現在分數上是兩位數的差別。

穩：對於穩定性，我想就不用多說了，業餘中，能回球比較到位的來個5/6板，基本對手已經失誤，拱手把分送上了。對我們而言，沒有專業的大量重複多拍的訓練，多拍的穩定性永遠是一個不可逾越的致命之處。

變：這個變字，我指的不是技戰術、針對性的球路變化，對業餘來說，這樣的高度一般是無法企及的，我想強調的是手法一致性前提下的球路變化。能夠把高遠球、平高球、殺球、劈吊等手法做到基本一致，在出球的一霎不被對方察覺，基本已經窺見了勝利的大門。單打，要照顧的區域比雙打大了將近一倍，跑到各個邊邊角角的時間也要多得多，只要能讓對手保持預備的姿勢多出0.幾秒，他就可能最後差個半步，無法順暢的發力而導致被動。如果，再能適當增強手腕的爆發力，在小範圍的發力出球也能有吊球和平高球、平快球的變化，基本就具備了專業的能力了。

狠：這個要求，其實對業餘來說是排在最後的，就是在出現中後場高球的時候能發力殺球，一錘定音，如果能有後場跳殺終結對手的能力是最好——不過，貌似只有頂尖的職業選手才具備吧？

這幾個要訣，在我看來，其重要性是依次遞減的，當然是針對水平到了一定層次業餘筒子們而言，那些3拍就主動失誤的暫不考慮。快是對抗的前提，穩是不敗的必備，變是制勝的法寶，狠已經是錦上添花了。如果有一位高手和你PK，笑眯眯地許諾：我不殺球哦，這下公平了吧？小心，那只是怪叔叔的陰暗心理，不要被騙上場被虐得體無完膚，欲哭無淚哦……..

嗯，有些筒子提出，穩 才是業餘單打的第一要素，我來簡單說說自己的看法：
舉個極端一點的例子：兩個高手，一個快卻不穩，三板必失誤，一個穩卻不快，見人就慢半拍。這樣的兩個人，在業餘對決中誰能走得更遠？應該是前者吧，速度上的絕對優勢，基本三拍內就解決了戰鬥，而後者，還沒等到10拍8拍，發揮自己穩定的優勢，已經由於速度慢，步伐不到位導致回球不到位，被人咔嚓了。
其實，穩，其前提也是快，沒有快速的步伐，就無法比較主動的發揮手上良好的感覺，回出比較到位的線路。在這個意義上，我再次強調，快是一切對抗的前提，速度上的差距過大，是沒法用任何其它的因素來補償的。來源中羽：球齡七年半
