入冬易餓怎麼辦？抗餓小能手來襲，號稱中國版的“士力架”，見證老師傅60年功底

2019-12-18 614條評論 3,854次閱讀 2105人點贊

一到秋冬季節，胃就像長了個黑洞似的，不管往肚子里吃多少東西，總感覺很餓！

想著冬天穿太多衣服，人已經顯得笨重、臃腫，為了保持身材，告訴自己要忍住不吃！可是不到1個小時，就會因為餓得頭昏眼花，忍不住狂吃，最後是吃得更多了！！

有沒有一種方法能讓自己既能滿足口欲，又能不長胖呢？當然有，今天要給你們帶來的是這款來自福建漳州的肉豆乾，號稱中國版的「士力架」，幫你趕走飢餓，重點是高蛋白零食，不容易發胖。

一不留神就會吃光！

第一次吃這款肉豆乾時，就先被它的名字勾起了強烈的好奇心，豆乾大家都吃過不少，豆製品大家也吃過不少，特別是大家愛吃的辣條。

同樣作為豆製品，這款肉豆乾堪稱是「豆製品里的佳品」，單從側面，就能看出肉豆乾的用料紮實。

肉的紋理緊密排列。一看就知道用的是好肉。

撕開包裝，先小小咬一口肉豆乾，可以感受到豆乾吸收了滿滿的鹵料汁。

嚼兩口，就有汁水在嘴裡迸發出來的感覺，豆乾吸收了肉汁的味道，在口腔里晃晃悠悠無縫不入，下一口連著肉一起塞進嘴裡，緊實的肉塊搭配著Q彈的豆乾，產生了一種奇妙而均衡的口感。

大片豆乾配著大塊肉乾，搭配出別緻的美味肉質緊實，豆乾Q彈肉香豆香鹵料香顛覆你對豆乾的認知，一口就好吃到想哭。

平時吃香精味的辣條吃多了，這麼真材實料的肉豆乾可是第一次吃到。這等美味，要感謝閩南的客家老師傅。

植物中的牛奶，蛋白質滿滿

想做出Q彈的豆乾，離不開好黃豆。黃豆被稱為「植物中的牛奶」，含有豐富的蛋白質和多種人體必需的氨基酸，能夠很好提高人體免疫力。

《食療本草》也提過黃豆，稱其能：益氣潤肌膚。產自東北肥沃黑土地的飽滿豆子要先泡發8小時後磨漿。

水和豆子的比例控制在5:1，反覆磨3次出漿，先把第二遍漿和第三遍漿混合一起煮，煮到85攝氏度後出鍋，然後把第一遍生漿（生漿比較濃）與出鍋的漿液混合。

濾漿後待漿溫降至80℃左右時加入涼水，降至70℃左右時點腦，做成鮮嫩的豆腐。

下鍋油炸，反覆2次過油，做成金黃的豆乾，剛出過的豆乾纖維組織分明可見，蜂窩狀的氣孔緊密排列，造就了豆乾層次豐富、彈爽有嚼勁的口感，還能讓豆乾吸收更多的滷汁。

金燦燦的豆乾就像夏日裡的陽光一樣明媚，豆乾香氣四溢，看著就讓人歡喜。

炸制後的豆乾還需要在配置好的秘制滷汁里浸泡，這一步用閩南人的說法叫做吊汁。至此，鮮香的豆乾才算做好了。

一方肉豆乾，老師傅60多年的功底

肉乾的製作也不能含糊，畢竟這塊肉是肉豆乾的精妙所在。

為了做好這塊肉乾，精選上好的閩西土豬，取其後腿肉，因為那裡筋少肉多，且肥肉少瘦肉厚，吃起來不費勁。

小孩子在長牙齒的特殊時期也能嚼得動，最適合拿來搭配肉豆乾了。

鮮豬後腿肉切塊，秘料滷製，大火燒20分鐘後吊干湯汁，再進烤箱烘烤15分鐘。

最後和滷製過的豆乾夾在一起，在烤架上慢火烘烤30分鐘。肉豆味道相互交融，豆乾韌性有勁，肉塊大而入味。一塊能勾起無數饞蟲、讓人口水直流。

一泡二吊三燜鹵，一位客家老師傅長達六十餘年的家傳功力，就這樣凝結在一塊小小的肉豆乾之中。

用牙齒輕輕一撕就綻開的美妙紋理，真材實料，嚼在嘴裡充盈著肉、豆、醬的三味妙香，捨不得吞下去。吃完之後，連手指頭都沁著香。

火爆小紅書的抗餓零食， 美食博主都在吃

這不，在國民種草平台小紅書上，不少美食博主也都推薦了這款肉豆乾，稱它是辦公室必備的抗餓零食。

可不是嘛，忙到沒時間下樓吃飯、叫外賣，抽屜里經常備一兩袋這個，隨手抓一個就撕開吃，十分方便。

熬夜的時候，只吃著有肉味的速食麵實在無趣，往面里丟幾個肉豆乾，瞬間化身日式料理店的高級拉麵。

入冬了，肚子越來越容易餓，嘴裡淡得發慌亦或是想吃東西的時候，拿一包噴香的肉豆乾出來慢慢地嚼，多吃也不怕發胖，還能讓自己過一過嘴癮，何樂而不為！
