技術貼：在不吃藥的情況下，怎樣練出鹿晗那樣的身材

2019-12-24 1,053條評論 2,734次閱讀 132人點贊

作為當紅小鮮肉，鹿晗受到一眾少女的追捧，在很多少女的心目中，鹿晗無論是顏值還是身材都是完美男神一樣的存在。之前鹿晗曾經在自己的社交賬號中發過一張自己的肌肉照，雖然多數正經健身的直男對此嗤之以鼻，但這絲毫不妨礙少女們盯著他的一身魔鬼筋肉犯花痴。

就像當年勁舞團那麼弱智的遊戲，因為有大量女玩家的存在，所以照樣可以吸引大量男玩家的湧入，或許本著實用主義的目的，為了把妹，真的會有男同胞想要練成鹿晗那樣的身材。所以，今天小編就跟大家正經的討論一下，作為一個普通人，在不使用藥物的前提下，如何練出鹿晗那樣的身材。

鹿晗的身材給人的感覺第一個字就是瘦。但是如果你想用健身教練給你推薦的減脂方法去減脂，你是很難練的和鹿晗一毛一樣的。因為鹿晗雖然瘦，但是他的皮脂並不是很低，我們可以看一下皮脂率和腹肌的對照圖

根據此圖的對應關係，鹿晗的皮脂率應該在13%-15%。健身教練給會員推薦的減脂方法，都是在盡量不減少肌肉量的情況減少身體的脂肪。而我們要想要練出鹿晗那樣的身材，就需要肌肉脂肪一起減，最好的辦法就是節食。因為節食的情況下，肌肉和脂肪都會不斷代謝燃燒。建議盡量少的攝入一切熱量，每頓飯都不吃主食，只吃蔬菜，除此之外，每天可以吃一顆雞蛋，但不得攝入任何魚蝦肉等富含蛋白質的食物。建議適量的吃一些豆製品，原因有兩點，1、人體激素和免疫系統的維持都需要一定的蛋白質，而豆製品富含蛋白質。2、豆製品含有一定量的雌激素，對於減少自身雄激素水平，增強自身女性化特徵具有一定的好處。同時，雌激素還會幫助我們在持續降低體重的情況下，保持一定量的脂肪。

很多老鐵可能會覺得單純的節食太慢，事實是我們可以加入適量的運動以加快減重的過程。但是運動的選擇上一定要注意。首先，不要選擇戶外運動，因為戶外運動會把我們的皮膚晒黑，這樣就與鹿晗白白嫩嫩的形象不符了。其次，盡量選擇低強度的有氧運動，如慢跑，瑜伽等，盡量避免因為無氧運動造成肌肉增長的情況。

最後，祝願各位想要練出鹿晗那樣身材的老鐵們早日如願，早日抱得美人歸。
