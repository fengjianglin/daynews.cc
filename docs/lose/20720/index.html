減肥欺騙餐：羊肉抓飯！胖子不是一天吃胖，瘦子更不能一頓餓瘦

2019-12-16 2,383條評論 3,785次閱讀 1514人點贊

我在頭條已更新過近200道減脂餐，感興趣的小夥伴點擊頭像，關注後就可以查看具體的做法

冬至將至，天氣進入寒冷深冬，減肥的小夥伴是不是越來越不知道吃什麼了？少油少鹽的飲食確實是健康，但是對於口腹之慾則很難滿足，天冷減肥真的好難啊，今天就安慰下大家，咱們來吃美味的羊肉抓飯！

減肥還能吃羊肉抓飯？還有肉和堅果？堅果可是妥妥的高熱量啊？帶著這個疑問，我們先來了解下減肥的另一種飲食模式：欺騙餐。

欺騙餐是指減肥期間不用關注熱量，要適度安慰自己的身體，吃完再說。這樣的做法是有科學依據的，欺騙身體，能夠讓你進一步瘦下去。

從身體機能講：

人體的適應性很強，當人體吃的食物一直維護在某個狀態，身體的熱量消耗也會慢慢適應這個狀態 ，如果沒有一定運動量的支撐，熱量的消耗會維持均衡狀態，從而會進入減肥的平台期，平台期突破有3種方法：1、輕斷食，改變飲食結構，讓身體重新適應； 2、欺騙餐，改變飲食結構，給身體安慰和獎勵，重新出發； 3、加大運動量或改變運動模式，調整身體的適應能力。

從精神層面講：

一個滿足胃部的機會，這在精神刺激上是一種促進作用，並不影響整體結果。也有小夥伴擔心一次高熱量攝入會讓減脂效果前功盡棄，實際上這是多慮的，沒有一天胖起來的瘦子，也沒有一天瘦下去的胖子。減肥，更重要的是長期的營養均衡，更是一種健康的生活方式！

天氣寒冷，更應該來安慰下自己的身心，今天就和寶寶們分享減肥期騙餐：羊肉抓飯的做法。

食材：帶骨羊腿肉250克、胡蘿蔔1根、洋蔥150克、大米100克、核桃仁、葡萄乾適量、姜適量

調味：料酒、生抽、鹽

做法如下：

1、大米提前泡一晚，泡軟；

2、羊腿肉帶骨剁成小塊（盡量小一些），放在水龍頭下，多次沖洗，洗去血水，然後加料酒和薑片，腌制半小時以上；

3、洋蔥切成細條、胡蘿蔔清洗乾淨切成條，葡萄乾和核桃也清洗乾淨表皮，備用；

4、鍋內倒油燒3成熱，然後放入少許的洋蔥煸炒，洋蔥炒變色後撈出，鍋內留下蔥油；

5、然後倒入腌制過的羊肉煸炒，炒干羊肉的水氣；

6、再加入胡蘿蔔和洋蔥，一起翻炒，加適量的生抽和鹽；

7、加水沒過食材，燒開水；

8、撈出羊肉塊；

9、然後加入浸泡好的大米，把食材倒在電飯鍋里，水量為沒過1個手指關節；

10、然後把之前撈出的羊肉放在米飯上面；

11、用小火煮到米飯表面的水汽 幹了以後，加入葡萄乾和核桃干；

12、然後再燜10分鐘，羊肉抓飯就大功告成了，盛出裝盤。

二喜姐的TIPS：

1、欺騙餐是吃一餐，而不是吃一天，欺騙餐最好放在午餐時吃；

2、吃健康以健康新鮮的食物為主，雪糕、蛋糕等甜食盡量少吃或吃；

3、羊肉要選易爛的腿肉，不要帶筋，在水龍頭下多衝洗幾次，盡量去掉血水；

4、料酒用來去腥，生抽上一點底色，放點鹽做為底味，羊肉抓飯的鮮味是來自於食材的混全味道。
