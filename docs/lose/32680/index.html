英國“牛魔王”，巨石強森的加大版，真的太強了

2019-12-23 436條評論 3,369次閱讀 1612人點贊

英國肌肉猛男，Martyn Ford，馬丁·福特。

如今35歲的Martyn身高2米，體重300斤。

140公斤杠鈴彎舉

260公斤硬拉也不在話下

220公斤坐姿推肩

機械坐姿腿舉750公斤

目前的Martyn可謂是紅遍了整個歐美健身圈

但是好身材也不是從娘胎裡帶出來的，

年輕時候的Martyn比細麻桿還要桿。

年輕時期的Martyn曾夢想做一名板球選手，可天不隨人願。

18歲那年，因為訓練而受傷，不得不讓Martyn放棄板球運動。就是這樣的基礎上，他開始健身並改變自己。

失去夢想後的Martyn化悲痛為動力，全身心的投入在健身中。

每天5點不到就早起健身，高強度的訓練讓他每天至少攝取7000大卡來維持體態，其中蛋白質、蔬菜水果是不可或缺的。

「不在沉默中爆發就在沉默中滅亡」

Martyn的健身事業越來越好

在Martyn變成威猛先生之後

越來越多的人找他健身

並且不少影視公司也紛紛找上門來

即使Martyn大多飾演的都是影視反派和配角

但Martyn心中感到很開心

出演影視的Martyn因為其野獸般的身材迅速火了起來

曾經只是分享教學與健身的ins賬號

如今粉絲也兩百萬了

Martyn如今也結婚生子

有自己幸福美滿的家庭

雖然近期Martyn舊病複發

好在手術很成功

傷口恢復的很好

Martyn也繼續從事健身教練的工作

偶爾拍一些影視

慢慢走上了人生巔峰

一切都歸功於從「140斤」到「300斤」的努力和毅力。

Martyn Ford對自己說，

這不是你開始的地方，

而是你完成的地方！

幾個月前，我在死亡床上，

沒有絕對的痛苦，

巨大的手術和失去7塊石頭，

甚至無法離開病床，

說那是黑暗的時刻，

這是一種輕描淡寫的說法，

但是每天過去了，

我進行那場戰鬥以返回，儘管我離那裡很遠，

但每天我都離它更近了一步。

在這樣的時刻，

關閉所有消極情緒並專註於商品的能力是關鍵，

每天都有機會讓自己變得更好，

成為你想要成為的人。

— END —
