一桿走天下——miliboo 705二代獨腳架使用評測

2019-12-17 187條評論 2,028次閱讀 734人點贊

零、序

近期，miliboo上新了一款獨腳架，miliboo 705二代獨腳架，作為經典的705升級款，主要升級了獨腳架的雲台，變臉王MYT801升級版，相比之前的雲台，重量更輕，操作依舊順滑，手柄更加舒適，按鈕旋鈕更加人性化，本RUA也借著雙十一的機會，購入了miliboo 705二代獨腳架鋁合金版，為何買鋁合金而不買碳纖維呢？還不是因為窮，而且二代鋁合金重量比一代碳纖維的重量還要輕哦

一、品牌介紹

miliboo，中文叫做米泊，廠家為上海科摯實業有限公司，是一家致力於攝影支撐器材產品設計、研發、生產、銷售的企業，而miliboo米泊，其設計理念為追求鐵塔般穩定，將法國埃菲爾鐵塔，百年屹立不倒的精神植入到每一件產品的精髓中

就像miliboo配件包中的贈送的小硬幣，一面為miliboo的logo，另一面就是法國埃菲爾鐵塔，鐵塔精神也是miliboo的精神，也是每一個miliboo攝影支撐器材的精神，不管是三腳架，還是獨腳架，都要向法國埃菲爾鐵塔一般，恆久穩定，屹立不倒

miliboo產品覆蓋獨腳架、三腳架、雲台、搖臂、桌面三腳架、配件等等，幾乎覆蓋全面攝影支撐器材，此次評測的就是miliboo暢銷獨腳架705的二代款——miliboo 705二代獨腳架

二、miliboo 705二代獨腳架開箱

包裝整體採用紅灰撞色配色，正面有一個外國佬坐在地上在使用獨腳架進行拍攝，這正是miliboo獨腳架特有的功能，90°放到拍攝的一個運用

包裝盒內為一個黑色收納包，黑色為主，紅色收線，點綴出整體收納包輪廓，正面為miliboo的logo，收納包採用尼龍材質，結實穩固，防水安全，同時，採用雙拉鏈設計，拉鏈手感順滑，絲毫沒有卡頓的感覺

拉開拉鏈，內裡帶兩個扎帶能夠將獨腳架固定住（首次打開的話，獨腳架會套袋包裹，手桿帶有泡沫保護，這裡已經被本RUA拆了），並且扎帶有大小之分，下方小的用於固定獨腳架底部，上方大的用於固定手柄主桿，保護雲台

而說明書、保修卡、相關配件都收納到側邊的收納袋中，收納袋拉鏈處還有一塊皮革，來防止拉鏈劃傷獨腳架

miliboo 705二代獨腳架貼心的送了收納背包，並且背包內部功能設計處處體現出細節，並不是簡簡單單的一個背包而已，這一點好評！

小配件包內裝著之前提到的小硬幣，三個不同尺寸的內六角扳手，登山桿底座、以及附送了一顆快接板螺絲

收納背包整體背起來比較方便，小巧，內部除了獨腳架外，還有一定的富裕空間，可以裝些別的物件

三、miliboo 705二代獨腳架細節

首先我們先將miliboo 705二代獨腳架整體安裝好

1、MYT變臉王雲台

不得不說，新版MYT801變臉王雲台不管從設計，還是從外觀，都比之前一代要強得多

雲台整體高度12cm，底座直徑75mm，與一代一致，鋁合金材質，自重由一代的1.15kg降到了0.86kg，但承重還是保持在10kg，miliboo的logo相對一代的正中間改到了右側上部，下邊造型好似傳動帶一般，紅色鋁合金圓環更加具有金屬材質，底部面積比一代更加小巧

快裝板鎖鈕採用三角造型，相較於一代旋鈕設計，更加方便解鎖與鎖死

鎖鈕同樣能夠調節，只要拉起，就能夠改變鎖鈕位置，以防止被上方相機擋住

手柄安裝旋鈕跟一代相比，變為鑲銅螺孔，更加的防滑耐磨，同時支持左右安裝，不論你是左撇子，還是右撇子，都可以順手使用

從左邊來看，可以看到雲台更加小巧，鏤空設計使得雲台更加具有設計感（因為本RUA為左撇子，因此手柄安裝到了左邊）

上方前半部3/8螺孔可外接閃光燈等附件

而後方紅色半圓色的按鈕為快裝板卡扣，一代為紅色按鈕，而新一代改為了扁半圓的設計，從設計來看，更加的富有設計感，並且按鈕面積相較於一代來說更加大，更方便使用

底部俯仰鎖緊旋鈕流線造型，更方便受力，相對於一代一字設計，也更具設計感

手柄鎖止旋鈕與快裝板的設計一致，手柄設計更加漂亮，相較於一代來說更加精鍊，更細更短，更加方便操作，扶手處為橡膠材質，帶有菱形凸起，底部紅圈也與整個獨腳架配色呼應

不管從正面看，還是從背面看，能夠看到雲台採用鏤空設計，不光美感升級，最大的作用是減輕了雲台的重量，由之前的1.25kg直降到了0.86kg

背部（靠近操作人那一側）為圓形水平鎖緊旋鈕，上部為輔助水平儀，一代來說，旋鈕由單側三角形旋鈕改為了圓形旋鈕，而水平儀由側面改為正中間了

頂部預裝著快裝板

快裝板兩側帶有菱狀突起的兩條防滑板，起到固定相機並防止劃傷相機底部作用

快裝板背部可以看到，帶有橡膠塞防止螺絲脫落，前方帶有「LEMS→」標識，鏡頭方向

螺絲並沒有使用快裝設計，而是使用硬幣（不管是附送的小硬幣還是一元硬幣）鎖死，相對不是那麼方便，好在後期可以自己更換

卸下快接板後，可以看到雲台表面增加了1/4和3/8備用螺絲收納，而據說面板座相比一代更加長，限位樁也由相應增高，並且增加了防快裝板倒裝功能（不過根據本RUA實驗，倒裝還是能夠裝的）

2、腳管

miliboo 705二代獨腳架的腳管有兩種材質，AL的鋁合金和CA的碳纖維，這裡介紹下本RUA購買的鋁合金款，這裡要注意下，AL和CA兩款並不僅僅是材質不同，材質帶來的重量不同，收納高度和工作最大高度也不同哦。重量方面，AL自重2.1kg，而CA更加輕，重量為1.98kg

頂部雲台承重平台支持1/4、3/8螺絲介面，其中3/8螺絲為空心回彈設計，壓下去就會露出1/4螺絲，適用性很強

平台下方為防滑橡膠保護套，類似泡沫手感，不會凍手，但是會沾手油及皮膚代謝，不好清理，保護套和平台中間帶有一個腕帶，上面帶有紅色miliboo的logo

在下方為露出的管體，帶有品牌logo以及型號，miliboo MT7705，管徑為34.2mm，壁厚1.2mm，給獨腳架帶來十足的穩定性

下方為三段式板扣，採用黑紅配色，鋁合金壓制，加厚設計，防止斷裂，板扣鎖緊力可以通過上方螺口調整

板扣阻尼適中，腳管調節順滑，操作方便，管徑底部最細的為20mm，上方最厚的為34.2mm

miliboo 705二代獨腳架工作高度支持最低（左邊）75.5cm到最高（右邊）181.5cm，可以根據個人身高及需要調整，而CA版調節範圍為76cm-191cm

本RUA跟獨腳架最高高度幾乎一致了，其實獨腳架因為本身的結構原因，高度更高帶來的穩定性就越差，而且使用起來，過高的高度也有一定的雞肋性，這也是本RUA不選擇CA的原因之一（本RUA凈身高180cm，當然AIR MAX 270的增高能力也是很強的）

3、三角爪

miliboo 705二代獨腳架的三角爪十分漂亮，頗具設計感，實際使用中，腳爪的收放也十分方便

直立鎖止旋鈕，為獨腳架帶來了快速的垂直鎖止功能，當將獨腳架調整至水平後，需要將直立鎖止旋鈕鎖緊，如果你的獨腳架安裝好後發現會有一定的晃動，那就要檢查這裡了，往往是因為沒有鎖緊導致的哦

球閥整體阻尼能夠通過側面調整，球閥阻尼盒一處帶有凹陷，滿足90°傾斜拍攝需求

腳爪內部帶有彈簧，保證限位扣能在腳爪放下時拉緊

加長的腳爪長度保證了獨腳架的穩定性，採用柔性防震可踩橡膠腳墊，讓人踩的舒心，踩的放心

除了部分特殊拍攝，普通使用獨腳架時需要使用腳踩住獨腳架的一隻腳爪，確保獨腳架不會意外倒地導致相機受損

腳爪的放下可以直接使用手或腳直接操作，不過收起時，只能上手了，將兩邊的限位閥向外拉，就能將腳爪收起

四、miliboo 705二代獨腳架使用

通過使用小硬幣，將快接板與相機連接，根據相機自重及鏡頭重量調整重心位置

miliboo 705二代獨腳架支持快接板斜壓式快裝，將快裝板一側放入快裝板卡扣那一側的凹槽內，並將另一側壓入，會聽到「ding」的一聲按入的聲音，鎖緊旋鈕就好了

而取下也十分簡單，擰松旋鈕，按下快裝板卡扣，將相機滑出即可

miliboo 705二代獨腳架支持75°仰拍，以及90°俯拍

液壓雲台手感十分順滑，帶有一定阻尼，移動過程中始終保持平穩有序

當調整到適合角度後，鎖死俯仰旋鈕，即可固定

其實，限制雲台仰拍角度的關鍵原因，在於水平鎖止旋鈕，當雲台調整仰拍角度到最高時，雲台底部會與水平鎖止旋鈕相碰撞，導致無法繼續調整

水平方面，miliboo 705二代獨腳架支持360°水平調整，需要鬆開水平鎖止旋鈕，同時一手扶住防滑橡膠套，即可進行水平調整，如果不扶住防滑橡膠套，容易導致底部鎖止或腳管鬆開，這裡需要注意下

同時還有一個操作，90°傾斜拍攝，這裡本RUA只是演示下，畢竟不是專業的，這個推鏡是需要長期練習的，首先底部垂直鎖止旋鈕是放開的，也就是說獨腳架會有一定的晃動，其次在前推的過程中，需要始終保持鏡頭的水平，自己試了試，沒有長期的經驗的話，很難

當拍攝使用時，氣泡水平儀和水平鎖死旋鈕會在正前方，方便確認相機的水平情況以及調整機位，手柄根據個人習慣調整左右，而俯仰鎖死旋鈕會在左側，左手操作，右手拍照

使用時，一踩，一扶，都是必要的，確保鏡頭穩定

2.1kg重的獨腳架，再加上相機，隨心移動（專業動作，請勿模仿！如果你模仿導致相機、鏡頭不慎跌落，損壞，本RUA不承擔一切責任）

五、miliboo 705二代獨腳架成片

獨腳架的使用，其實更加適合拍攝視頻作品，當然也可以當做一個簡易的三腳架，不過畢竟沒有三腳架穩定

使用獨腳架，拍攝寬幅畫面後期合成起來十分方便，並不會出現因手抖導致的畫面畸變

俯仰拍攝可以根據需要從高到低，或者從低到高推動鏡頭，帶來需要的感覺，比如用的比較多的是從下往上推，然後出現一個你想看見的人

平移鏡頭，miliboo 705二代獨腳架帶來是十足的穩定性，鏡頭沒有任何抖動，在展現多個產品或人員的時候，更多可以通過平移鏡頭來實現

俯仰前推，本想拍攝成下沖後平穩落地推進的效果，不過因本RUA技術有限， 拍出了第一人稱視角的摔倒畫面（笑）

六、miliboo 705二代獨腳架其他功能

miliboo 705二代獨腳架除了可以承擔本身獨腳架的功能外，還能夠作為桌面三腳架或者登山桿使用

1、桌面三腳架

將中間的腳管去掉，直接將雲台和三角爪合體，就是一個桌面三腳架了

2、登山桿

將三角爪，去掉，更換配件包中的登山桿底座，miliboo 705二代登山桿就完成了

多段長度可調節，帶橡膠防滑套，腕帶，看著挺完美的，不過雲台承載平台上的3/8螺絲看著有點突兀

七、miliboo 705二代獨腳架總結

總體來說，miliboo 705二代獨腳架滿足了本RUA的期待，不管是拍照還是視頻拍攝，都起到了比較重要的作用，並且顏值在線，功能多樣，也是它的優勢，配有最新的MYT801變臉王液壓雲台

在此，本RUA經過幾天的把玩，對miliboo 705二代獨腳架總結如下：

1、採用變臉王二代液壓雲台，10kg的承重，帶來了更加舒適的操作手感，拍攝過程中穩定平滑，給人足夠的信心

2、機動性強，2kg左右的重量，配合獨腳架特有的操作方式，隨時隨地想拍就拍

3、可用性強，一架三用，靈活變身桌面三腳架與登山桿

當然，獨腳架因為更加追求體積及操作，穩定性一定會不足於傳統三腳架，但是miliboo 705二代獨腳架通過加寬加厚的腳管、垂直鎖止設計，在穩定性上已經算是同品類表現較好的了，如果你更加追求便攜性，以及操作的便利性，或者拍攝視頻、VLOG更多的話，一款稱心的獨腳架也許更加適合

我是RUARUARUA，一個業餘不能在業餘的攝影小白，不追求器材，也不追求模特，只信奉適合自己的就是最好的。此次評測就到這裡，那麼，我們下次見嘍~
