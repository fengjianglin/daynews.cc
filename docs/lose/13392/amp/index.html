秋冬易發胖，教你吃完不長肉的秘決

秋冬是美麗的季節，但更是一個容易發胖的季節！一不留神就讓你胖幾斤

整個夏天的肥都白減了有木有！

在這種喝口秋風都胖的日子裡，到底怎麼做才能美美的瘦下去？

首先你必須尋根問源，弄清楚為什麼秋天這麼容易發胖！

生理原因

其實秋天就算正常飲食，飯量不變，身體依然會在不知不覺間偷偷胖三斤。這是正常的生理現象，隨著秋季到來，身體內的脂肪也感受到了涼涼秋意，於是開始慢慢地聚集在一起，為過冬做準備，這時候人體便開始逐漸變胖。

胃口變好

夏天的時候因為天氣悶熱，人們普遍食欲不振，可是到了秋天就不一樣了，沒有酷暑的煎熬，人們的食慾增強，飲食也會不知不覺地過量，從而導致熱量的攝入大大增加。

運動減少

春困秋乏，很多人寧願在家裡打瞌睡也不願意出門去運動。深秋的時候天氣更冷，更不願意出門運動。這樣運動量逐漸減少，整天宅在家，吃得多、熱量消耗少，長此以往能量攝入大於能量消耗，一不小心就長胖了。

就算秋天的時候衣服越穿越厚，肉肉被擋住了，也不能鬆懈減肥的心，長出來的肉都是要還的，吃的時候有多開心減的時候就有多傷心。所以說，瘦的人防胖，胖的人減肥，秋天才是減肥的好時機！另外，秋天減肥還有一些額外的好處！

秋季減肥好處多

有效改善秋乏

由夏入秋溫差變化大，機體也慢慢進入一個休整階段。換句話說，在整個夏季過度勞累的機體開始鬆懈下來偷偷懶兒，於是大腦就一直感覺昏昏欲睡很睏倦。但是減肥時，適當的運動能夠保證大腦皮層細胞的活躍，從而有效改善精神不振，睡不醒等秋乏狀態，保持精神的飽滿。

改善肺腑功能

由於氣溫溫差大，秋天很容易造成內分泌失調，特別是對於身體胖且患有慢性疾病的人而言，秋天減肥能更好的調節身體各項技能，尤其是能改善身體內臟脂肪的分解代謝，有效改善肺腑功能。身體各項免疫力也能得到增強，這樣冬天天氣冷也不容易感冒啦！

防止身體持續長胖

如果秋季不減肥，到冬天身體只會越長越胖，因為冬天的時候熱量儲存會更多，熱量攝入也會更多，因此，從一開始就將脂肪扼殺在搖籃里，遏制體內更多脂肪的形成，從源頭控制住肥胖，到冬天的時候就不至於越來越胖。對於脂肪高的人而言，秋天減肥是最好的，因為夏天比較熱，冬天又太冷，這些都會影響減肥情緒。

秋季減肥常見誤區

1、飲食毫無節制

天氣變涼，吃點熱乎乎的食物瞬間心裡暖洋洋的，因此很多人在秋天的時候食慾更加旺盛，一不小心就胃口大開，尤其鍾愛火鍋之類的食物，殊不知這種食物重油重辣，可以說是減肥路上最大的攔路虎，這樣毫無節制的吃胖起來真的控制不住。

建議：秋天飲食口味最好相對清淡一點，比如各種山藥粥、玉米粥、小米粥等，這些食物不僅有利於消化，而且還有減肥功效。

2、吃水果減肥

秋天是碩果累累的季節，尤其是蘋果、秋梨、橙子等，各種水果都成熟啦！當季水果鮮嫩多汁、口味脆甜，因此很多人喜歡在秋天吃水果減肥，但是單單吃水果身體營養會跟不上，因此秋天吃水果減肥是不對的。

建議：秋天可以適量的吃水果，愛吃水果不是壞事，但是不能把水果當飯吃，可以適當減少飯量，然後再吃一點水果當做加餐，這樣能有效增加飽腹感，也是很有效的減肥方式。

3、愛叫宵夜

秋天旺盛的食慾肯定讓很多人擱淺了減肥大計，清冷的深夜飢腸轆轆，能吃上一口美味的食物，一定是最幸福的事情！但是寶寶你可要想清楚了，吃進去的是食物長出來的可都是肉哇，晚上攝入過多熱量這是分分鐘想長胖的節奏！

建議：夜間消化水平降低，食物進入腸胃後不易消化，非但容易攝入過多熱量，還會影響睡眠質量，所以睡覺前最好不要吃東西。如果實在覺得餓的睡不著，可以喝一杯牛奶或吃一根香蕉填胃。

4、盲目貼秋膘

民間素來有貼秋膘的傳統，立秋那天以懸秤稱人，如果比立夏瘦就叫「苦夏」，這時候就需要貼秋膘，吃點大魚大肉等美食佳肴來補補身子。今天雖然沒有這個說法，但大部分人還是認為秋天應該進補。飲食的突然變化也很容易引發肥胖。

建議：秋天可以適當進補一些有營養的食物，但不要過度進補，避免高熱量的食物。要講究營養均衡，合理搭配。優先選擇海魚、海蝦、雞胸肉、瘦牛肉等富含蛋白質熱量卻較低的魚類和禽類，能補充膳食纖維的穀類和蔬果，這些都是秋季進補的首選食材。

5、飲水量太少

夏天的時候因為天氣炎熱出汗較多，所以人的飲水量也會比較大，但是秋天氣溫降低出汗較少，很多人就不愛喝水了，其實這種做法是大大的錯了！秋季空氣乾燥，人體反而會損耗比夏天更多的水分，應該喝比其它季節更多的水，這樣才能保持肺臟與呼吸道的正常濕潤度。

建議：秋天臉上容易出油，多喝水有利於排毒去油，美容養顏、還能養肺潤肺。每天最少保證1500毫升的飲水量，從起床後開始，最好每間隔兩個小時喝一杯。

6、喜歡喝奶茶

氣溫下降之後，街上的奶茶店、咖啡店，瞬間生意火爆了起來。天冷的時候一杯香濃溫熱的飲料才是真愛！於是喝著喝著，一不小心就胖了！

建議：其實秋天喝點茶會更好，像綠茶、烏龍茶、苦丁茶、大麥茶、花草茶等，都很適合秋天喝。秋天氣溫乾燥容易上火，喝點茶不僅降火還有排毒養顏功效。也可以自制檸檬茶，喝起來和外面的飲料差不了多少而且熱量低！
