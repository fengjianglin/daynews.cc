做什麼運動有助於膽結石的排出？

天天要聞
2019-12-17

眾所周知，形成膽結石的原因之一，就是因為長期卧床和極少運動，使鈣鹽沉澱於器官管道及空隙之中。如果經常運動、多喝水，肯定能減低鈣鹽的沉澱及濃度，減少結石形成的機會。據統計，經常參加體育運動的人，結石患病率最低。有人證實，按以下運動處方進行鍛煉，防治效果很明顯，有的排出結石，故可命名為排石操。

　　動作一：兩腳「八」字站立，舉起雙手，成弓箭步旋轉向前向後。

　　動作二：雙手抱膝，向前後屈身。

　　動作三：胸膝卧位，持續5～10分鐘。

　　動作四：屈髓向前。

　　治病要有醫生開具處方是一件眾所周知的事，但運動也有運動處方一事就不是每個人都了解的。一張合理的運動處方包括以下幾方面內容：運動的內容和方式；運動應達到及不宜超過的運動強度；每次運動持續時間及每周運動的次數；自我觀察的指標以及規定情況下出現異常反應時應停止運動的指征。

　　膽結石能預防嗎從以下幾方面加以注意，在一定程度上，膽結石是可以預防的：

　　積極治療能引起膽結石的病症，如膽囊炎、肝炎、肝硬變、脂肪肝、溶血性貧血、肥胖症和高脂血症等。

　　注意避免能引起膽汁瘀積的一切因素，改掉不進早餐的不良習慣。

　　長期靜脈內營養患者定期使用促膽囊收縮藥物。

　　妊娠期間儘可能規律進食。

　　定期體檢，一旦發現膽囊功能差、結構異常，膽囊膽汁透聲差，出現膽泥或膽沙等，經過積極、合理的治療，膽囊功能是可以恢復的，膽泥、膽沙或瘀積的濃稠膽汁能被排出，這樣，就可以早期阻斷形成結石的路徑。
