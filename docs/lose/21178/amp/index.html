3337減肥法：3337殘忍減肥法等於是變相絕食！就是忍者都受不了

　　3337減肥法是一種很變態的節食減肥方式，雖然減肥效果見效快，效果好，但是同樣的會給我們的身體帶來不可忽視的危害。那麼，下面就一起來看看3337減肥方法的危害有哪些吧？總之，妹紙們千萬不要為了自己擁有理想中的身材就去做危害身體健康的事情，3337減肥法可以說是一種比較極端的減肥方法，建議妹紙們不要輕易嘗試，以損害身體健康為代價獲得的減肥效果，只會讓你得不償失。

　　3337減肥法易導致精神緊張

　　3337減肥法一個周期就有3天的時間是只喝黑咖啡的，平常適量的喝點咖啡能幫助提神醒腦，但是這樣只喝咖啡會容易導致中樞神經長時間處於興奮狀態，導致出現失眠、精神緊張、甚至出現耳鳴、頭暈的情況。

　　3337減肥法易引起骨質疏鬆

　　咖啡作為3337減肥法中的主要食物，長期飲用的話，黑咖啡中含有的咖啡因容易導致體內鈣質的流失加快，嚴重的話會引起骨質疏鬆。

　　3337減肥法易造成胃部疾病

　　3337減肥法中不管是咖啡還是蘋果，都還能刺激胃酸分泌的，但是由於喝咖啡和吃蘋果的時候都是不能吃其他食物的，沒有足夠的食物提供給胃部進行消化，會使得胃酸和胃蛋白酶過多從而對胃進行攻擊，長時間容易引起胃炎、胃潰瘍等胃部疾病。

　　3337減肥法易引起低血糖

　　3337減肥法可以說是長達半個月的時間不吃主食，甚至是不攝入碳水化合物和糖分，這樣做是可能因主食營養的缺乏，發生體內血糖含量降低，出現頭暈、心悸等低血糖現象。

　　3337減肥法易導致營養不良

　　3337減肥法前幾天光是喝咖啡和吃蘋果，後面幾天也只是簡單的吃點青菜類，完全不能身體對營養物質的需求，不能維持營養的均衡，長久使用這個方法，是會發生營養不良的情況的，甚至導致氣血不足，出現貧血的現象。

　　3337減肥法易致內分泌紊亂

　　雖然3337減肥法可以在短時間內有很好的減肥效果，但是女性在短時間出現體重下降過快的情況，是會出現脂肪代謝不正常、內分泌紊亂的。還會因為內分泌紊亂，出現月經不調、閉經甚至嚴重的還會造成不孕。

　　3337減肥法易導致記憶力下降

　　3337減肥法的兩個周期是攝入碳水化合物是極少的，而人的大腦活動是需要葡萄糖作為能量的，葡萄糖是需要通過血液來持續提供。如果身體的碳水化合物攝入不足，為大腦供應的葡萄糖就會相對減少，對於記憶力會有很大的影響，嚴重的話可能會導致記憶力下降。

　　3337減肥法容易掉頭髮

　　頭髮的生長也是離不開各種營養物質的，但是3337減肥法攝入的能量是有限的，像蛋白質、脂肪酸和碳水化合物等都是3337減肥法攝入比較少而頭髮生長需要的營養物質，長久下來，會導致頭髮乾燥、掉頭髮的情況。

　　3337減肥法易致皮膚乾燥鬆弛

　　3337減肥法實際上是節食減肥的方法，會導致蛋白質的攝入不足，而我們身體活動每個器官的運轉都是需要攝入足夠的蛋白質。3337減肥法時蛋白質攝入不足，會使得整個身體機能受到影響，而膠原蛋白的流失會導致皮膚變得暗淡乾燥無光澤。而且身體的脂肪和肌肉是皮膚的最大支撐力，減重過快會使得皮下脂肪流失，容易使得皮膚變得鬆弛。

　　3337減肥法易導致抵抗力下降

　　3337減肥法中由於攝入食物比較少，各種維生素和身體需要的營養物質的攝入都變少了，長時間使用這個方法，會使得身體的抵抗力下降，從而引發各種疾病的發生。

　　3337減肥法容易反彈

　　因為3337減肥法時間是控制進食熱量，實際上是節食減肥，但是後面一旦恢復正常飲食，就有可能出現體重反彈的情況。
這種殘忍的減肥法，對人身體也是有害的，容易引起維生素缺乏，導致內分泌紊亂，甚至引發身體疾病，一般不提倡使用。相比減肥來說，健康更重要。
