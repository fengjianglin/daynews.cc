健康悅讀｜避開三大誤區，王隴德院士教你這樣減肥

2019-12-06 1,274條評論 2,343次閱讀 706人點贊

《掌握健康鑰匙》連載10

誤區5：減肥避開三大誤區

書名：

掌握健康鑰匙

作者：

王隴德

出版社：

人民衛生出版社

誤區①

集中鍛煉一個部位可以局部減肥

鍛煉是改變整個身體代謝的過程。鍛煉首先消耗內臟脂肪，然後才是皮下脂肪。皮下脂肪是血流量多的地方先消耗，如四肢、臉頰部等。所以，鍛煉減肥要看出身體外形的變化，尤其是腹部脂肪的減少，效果相對滯後。必須堅持鍛煉一段時間後，才可以看出明顯變化。

局部鍛煉可以增強局部肌肉的力量（使肌纖維增粗），但不能局部減肥。如只是過多地做腹部鍛煉，腹部的肌肉增強而皮下脂肪沒有減少的話，反而看上去肚子會更大。

誤區②

節食即可達到減肥目的

過分節食會造成必需營養素缺乏，影響機體功能和免疫力。因為過分節食不可能持久，只是短時間內少吃，而未改變飲食習慣。所以靠節食減輕體重的人，90％以上會反彈。

誤區③

洗桑拿浴等大量出汗即可減肥

人體中的水分主要在體液（如血液、細胞間液）和肌肉細胞中，而不在脂肪細胞中。大量出汗後會出現口渴感，導致頻繁飲水，結果會迅速恢復原體重。排出汗液的好處是可帶走體內的部分代謝產物，但不堅持長期適量運動，只想通過大量出汗實現快速減肥是不可能的。
