小聶：跑動中的腰腹力量如何運用

2019-12-18 1,245條評論 2,363次閱讀 752人點贊

在我研究了很多很多世界頂級的精英跑者的動作視頻，結合很多技術流派的技術文章後，無論是長跑或是短跑，最後結論就是這些跑者的步幅通常都會超過自己的身高。

我困惑了大半年，每次跑步都會想：人家是如何做到的？

對大眾跑者來說，有相當一部分人喜歡沉迷於某些技術數據，每次交流都離不開自己的步幅、步頻、觸地時間這些。但對於如何解決這些問題，他們是茫然不知的。

在我跑步初期，有一個精英跑友建議我大量做高抬腿來提高步頻，因為他自己每天堅持做500+次快速高抬腿練習。

但事實上，他目前的跑力已經遠遠在我之後，以前他引以為傲的10公里39分至今都沒有再突破，而我已經很輕鬆能跑進38分區。他的年紀比我小五歲。

因為，我有自己的想法，我覺得練習方向比盲目的努力更重要。

我曾經報讀過某個技術流派的課程。他們主要是提倡不後蹬和主動提拉，認為提拉小腿是跑動中唯一需要發力的動作。

在練習初期，改變主動跨步的習慣的時期，這樣的動作觀點會很有作用。

但隨著技術動作的運用越來越熟練，跑力越來越強，我發現這樣的觀點已經不能滿足我在跑動中的需求。

然後，我在堅持了8個月的基礎力量練習以後，在某次跑動中，我突然找到了騰空時間的發力感覺，步幅也自然地提高到一個我之前想都不敢想的數據。

腰腹部發力去跑步，我不知道這樣的表達算不算一種革命式的定義。也許有同類的定義，可能是我孤陋寡聞吧，至今沒有獲取到官方結論。

我曾經表達過一種觀點，那就是如果你在跑動中腰腹力足夠控制你的動作的時候，你的下肢動作其實是可以為所欲為的。

和我同時表達的無為意識貌似有點衝突。

但事實上是沒有衝突的。因為我先後經歷了這兩個階段。

在我們跑動和身體素質處於初級階段的時候，我們想的當然就是提拉意識來改善主動跨步的習慣，從而有效利用重力牽引來跑步。這個時候，動作意識是很重要的。

特別是落地點的控制方面。

控制落地點，在練習初期需要無為意識來決定，就是你絕對不能在跑動中去想如何落地。擺動腿在拉起過程中自然前擺，自然落地就好了。這樣的自然感覺會幫助你克服大腿做功，小腿前擺的不良動作，增加跑動效率。

而當你找到腰腹部發力感覺的時候，落地方式在思維意識當中是可以控制的。因為我們的騰空時間會加長，運用動作的時間是足夠的。

在腰腹部發力感覺產生的時候，蹬地這個動作非常重要。

剛才提到的那個技術流派建議不要蹬地，也表達了我們在跑動中蹬地是反作用力的反應。其實，這樣的觀點是有點片面的。

我們可以假設一個情景，當我們在一樓往下丟東西遠還是在三樓往下丟東西比較遠呢？

在跑動中，如果僅僅以反作用力彈起的拉起小腿形成的高度再滑行獲取自然步幅，肯定是不夠的。

我們獲取自然步幅的方嚮應該是這樣的：落地，彈起，再拉起這是一個垂直向上的作用力，也就是我們所說的垂直振幅。然後在騰空一霎那腰腹部刻意往下壓。

記住：這裡是腰腹部刻意主動往下壓。這個感覺會有效控制我們的垂直振幅。而且和前面的動作形成的垂直向上的力融合成往前的一股動力。

只有足夠的騰空高度才能把這股力最大化，然後更有效的在騰空時間裡滑行，產生自身最大的步幅。基普喬格的1.91米的步幅就是這樣得來的。雖然我體能有限，但我也能在這個動作感覺中400-1000米距離跑出1.65-1.80米的步幅，觸地時間維持在180毫秒左右。

因此，後蹬是必須的，但後蹬的時機非常重要。這是後話，我會另文介紹。

主導這個動作有效形成，腰腹部的感覺非常重要。你在垂直向上的騰空感覺中，能否以你的腰腹部力量壓著往前，取決於你的個人能力。

如果你的腰腹力量非常薄弱或沒有這個發力技巧，這個動作意識免談。

因此，我們可以得出這樣的結論：我們的跑動應該是以腰腹力量來主導。

先通過大量的腰腹基礎力量練習，當然是要結合跑動身體適應性的動作來練習。然後從基礎提拉意識建立開始，首先把下肢動作做熟練，形成有效的肌肉記憶。那麼，在中後期腰腹部發力跑的時候就不會顧此失彼了。

值得注意的是：這個時間也許很漫長，也許就在一瞬間。力量基礎和發力技巧缺一不可。

好了，初期感覺就這些，後面再分段給大家分享一下這些練習技巧。
