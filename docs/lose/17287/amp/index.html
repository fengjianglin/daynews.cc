薦讀：格鬥競技項目體能訓練原理和方法

天天要聞
2019-12-14

體能是技術的基礎，不具備基本的體能，再好的技術也無從發揮。同樣有好的體能，則可以放大已有的技術。但是究竟體能訓練的原理如何，及具體訓練方法應該遵從什麼原則？針對格鬥競技項目專項體能訓練的原理和方法進行解析。

競技體能可以比喻為百分之五十蒙娜麗莎，百分之五十愛因斯坦，對於如何幫助運動員提高賽場競技表現是一個一直在爭論的話題。往往許多體能訓練師傳達給運動員的信息過於複雜，也有很多還未認識到體能重要性的運動員過於鑽研專項技術忽視了體能和技能的一體性。

提起體能這兩個字大致可以拆分為身體能力和能量代謝系統，這兩個重要的因素構成了體能的基礎。我們先來說一下身體能力。

身體能力

身體能力包含硬體和軟體兩方面。硬體如骨骼肌肉，穩定的支柱等外在原因。軟體如身體的中樞神經系統，良好的運動模式，高效的肌肉募集速度等內在原因。強大的硬體在高效軟體的支配下才能擁有最好的運動表現。

上述說的身體能力都需要建立在良好的能量代謝系統下。

能量代謝系統

人體生命的維持是因為人體能夠源源不斷地通過代謝獲得能量以維持機體的運轉，包括、肌肉收縮、神經傳導、血液循環、組織合成和腺體分泌等。人體運動時所需要的代謝原料（如氧氣）需要經由血液流至代謝的組織細胞，而代謝後的廢物（如二氧化碳）同樣需要經由血液進行排除，因此血液量是能量代謝程度的一個重要標誌。

安靜狀態下，心臟每分泵出的血液量只有1200ml，其中21%的流往了肌肉；但是最大運動時心臟每分泵出的血液量高達22000ml之多，並且88%的血液流往了肌肉。因此人體運動過程中，尤其是高強度運動過程中，肌肉是能量代謝的主要場所。

肌肉收縮做功的直接能量來源是三磷酸腺苷（ATP），但是ATP在肌肉內的儲量非常有限（約5-7mmol每公斤濕肌），只夠肌肉進行3-4次最大收縮或者1-2秒的最大持續做功。此後，肌肉需要通過其他途徑合成ATP，以繼續收縮做功。ATP的合成包括無氧非乳酸（磷酸原）、無氧乳酸（無氧糖酵解）和有氧（葡萄糖和脂肪的氧化）三種途徑。ATP合成的三條途徑也被稱為人體能量代謝的三大供能系統。

通過上述對身體能力和能量代謝原理的簡單解釋，所以格鬥運動員想通過體能訓練來提升賽場上的競技能力應該考慮以下幾點。

針對性訓練能量系統

縱觀格鬥類型比賽的時間時長一般都在5分鐘一回合進行3-5回合，一般不會有連續高時長的情況出現。很多運動員在進行能量系統訓練時往往會進行長距離奔跑，這樣主要鍛煉的是身體的有氧能力而且強度相對穩定。一場綜合格鬥的比賽在5分鐘內往往是短時間高頻率的做工輸出和一擊就能終結比賽的能力，所以在能量系統訓練的方面要盡量考慮賽場上的因素。

穩定的支柱力量

支柱包含髖、膝、踝、肩、脊柱，構成了身體的支柱，許多人理解為核心力量。良好的支柱力量是良好運動能力的基礎，身體的支柱主要的作用是傳導力和承受力。觀察許多運動項目運動員在高強度身體接觸，高速的奔跑，揮動球拍時無論身體處在何等狀態下都擁有穩定核心支柱。

靈敏的中樞神經系統

在競技體育當中充滿了不確定性尤其在賽場上誰都無法預料下一秒發生的情況，如足籃球中對手突然的變向，格鬥賽場上對手突然的下潛抱摔，都需要有靈敏的中樞系統作為支撐幫助運動員應對突發情況。在美國職業橄欖球聯盟中做過這樣一個調查幾乎最好的四分位身體能力都不是最好的，但是他們的中樞系統反應速度尤為突出，所以常常稱身體能力強，頭腦又好的運動員為頂級運動員。推薦使用敏捷燈，敏捷球等簡易工具就可以訓練。

高效的爆發力

在格鬥運動中幾乎絕大部分的動作都離不開爆發力，格鬥運動員想要擁有高效的爆發力往往都忽視了非常重要的一點就是在做工時運動單元募集肌肉的速度以及數量。研究表明百米飛人博爾特肌肉募集的速度高達90毫秒是一般人的3倍，這也是他擁有超凡速度的關鍵。

這裡推薦大家選擇「快速伸縮複合訓練」也有稱為超等長或者增強式（ssc），快速伸縮複合訓練利用預先拉長肌肉或先反向運動而實現快速、有力的運動效果。這種運動包括了一個拉長——短縮（收縮）周期（SSC,Streth—Shortening Cycle）。拉長收縮（超等長）的目的是利用肌肉和肌腱的彈性成分以及拉伸反射作用，使接著進行的運動更加有力。

合理的恢復再生

肌肉恢復除了營養的補充外還有很多的恢復方法，利用筋膜球、筋膜刀、泡沫軸等再生工具能使肌肉擁有更好的運動表現。我這裡推薦格鬥運動員在訓練前和訓練後都可以使用放鬆工具，例如在訓練前使用泡沫軸可以有效的放鬆肌肉當中的「板結點」。

降低肌肉張力，提高肌肉持久力，也可以激活肌肉當中的本體感受器「肌棱」降低受傷風險，訓練後使用可以快速刺激「副交感神經」降低肌肉興奮程度起到冷身的作用，還可以通過滾動梳理的方式讓肌肉快速補氧加快血液循環起到再生作用。

以上專項體能訓練的原理和方法解析，可謂言簡意賅、充滿乾貨，希望能對大家的格鬥競技體能訓練起到一些指導作用。老規矩，歡迎發表評論回復您的看法。
