1.新月式

2.孔雀式

瑜伽體式最重要的就是耐力，很多人都無法接受瑜伽的強化過程，從而放棄了這個運動方式，其實只要能夠堅持住耐力，就可以很輕鬆的擁有好身材了。

3.輪式

腰部贅肉多，卻始終沒有找到適合自己的動作，那麼就可以嘗試一下這個體式，可以有效的針對腹部兩側贅肉，經常練習很快就可以擁有好身材了，抓緊時間運動吧。

體式要點：腿部自然分開，膝蓋彎曲，身體向後用力，手臂順勢伸出，支撐在地面，調整身體的位置，一條腿朝向空中伸展，腳背緊繃起來。

4.聖瑪麗琪一式

這個體式主要是幫助我們調整身體肌肉狀態，當自己身體感覺到疲憊的時候，就可以鍛煉一下這個動作，快速的調整到身體的最佳狀態，感興趣的朋友們可別錯過啦 。

5.舞蹈式

這個動作有些難度，不適合新手練習，韌帶不夠柔軟的朋友們，可以先進行簡單的拉伸體式，然後在開始正式的體式練習，防止韌帶拉傷的情況。

體式要點：單腿伸直，支撐身體重量，另一條腿後面彎曲，超過頭頂位置，上半身微微前方用力，手臂抓住腳踝骨，另一個手臂搭在牆壁上。
