胯寬腿粗腿短腿不直？那是因為你還沒get到這幾條褲子

2019-12-12 881條評論 2,010次閱讀 1010人點贊

晚上好呀，仙氣幺來了

剛剛和小表妹視頻發現她喪氣滿滿，原來是她試了一下在網上買的新褲子！

發現自己居然有了假胯！！！腿還沒以前直了！簡直要命！

很多人的腿型或多或少都會有點問題，所以之前就有好多小仙女問過小幺，腿不直怎麼辦？

特別是冬天要來了，褲子什麼的肯定要穿起來了，那麼今天就來簡單的給大家說說~

為什麼腿不直？

還不是因為腿型不好了，不懂腿型？今天小幺就來和大家簡單講講！

主要以O型腿和XO型腿為主，而那些還不知道怎麼分辨自己腿型的小可愛，直接見下圖找腿！

理想腿型也就是正常腿型，擁有這種腿er的小夥伴喜歡怎麼穿就怎麼穿！

什麼好看就穿什麼！完全沒在怕的！X型腿一般是先天形成的，嚴格意義上來說不是特別常見，小幺這

里就不過多說明了！

O型腿

不適合：緊身褲/鉛筆褲

說到O型腿，首先想到了是允兒，其實小幺覺得允兒蠻可憐的！

每次說到O型腿都會被拽出來當做教學案例，允兒的腿是真的很O了，可是她的顏也真的很漂亮啊！

O型腿特點就是小腿肌肉外翻，視覺上就會拉寬腿，顯得胖和短，O型腿在穿搭上要避免選擇突顯線條

的下裝，如冬天的襪子和顯腿型的緊身褲等等。

XO型腿

不適合：面料太薄、太緊的褲子

XO型腿是最常見的一種腿型，我也有輕微的這種屬性。

XO型腿其實整體來看還是挺直的，在穿搭上不需要注意的太多。

其實腿形問題真的很常見，很多我們羨慕的明星都有同樣的困擾，比如膚白貌美大冪冪就有XO型腿。

小幺之前以為大冪冪是X型腿，也有很多人難以區分X和XO型腿，其實只要記住X型腿的腳踝是無法並

攏的，XO型腿的小腿間的縫隙比較大，並且兩腿的小腿肚沒法挨著。

除了大冪冪，還有很多明星也都是XO型腿，小腿粗出了名的吳昕也是這種腿型，不過最近的照片中吳

昕的腿型看著有點改善過了，也不知道是修圖效果還是真的有在做訓練。

志玲姐姐一向以好身材著稱，但是她的大長腿也有點彆扭，特別是小腿的部位，外翻得略明顯。

你看無論是大長腿的志玲姐姐，還是有美腿之稱的大冪冪，都或多或少有點腿型問題。

所以你們就不要擔心啦！輕微的XO型腿，也是什麼都可以穿的。

其實腿不直，不管是X/O/XO型都是要避免缺陷的，像是會自爆缺陷的緊身褲、鉛筆褲都是要避開的。

除了這些，小幺想和小夥伴推薦三款，今年最流行，也是最不挑腿型的褲子，輕鬆get又長又直的超級

美腿，一起來看看吧~

煙管褲

煙管褲（Skinny Trousers）

是指有著纖細褲管的褲子，也有窄管褲之稱，有的地方又叫它錐形褲、窄管褲、吸煙褲！

煙管是小幺認為繼闊腿褲之後，任何腿型都能駕馭的褲子，百搭又顯瘦！

錐形褲的褲型從上到下呈現出逐漸收窄的趨勢，這種設計不容易暴露腿型的缺陷！

普通身材當中最讓人煩惱的假胯寬、梨形身材等。都很適合穿煙管褲，而且，它是標準的九分褲長度，

穿的時候露出腳踝會有明顯的顯瘦效果。

煙管褲的寬鬆程度介於直筒褲以及鉛筆褲之間，並且它的設計在貼合了我們臀部的同時，又給了大腿

處一定的空間，所以可以有效地遮住大腿贅肉，並且讓腿型看上去更為筆直。

煙管褲搭配任何上衣都很合適，大家只需要參照普通褲裝的搭配標準就可以了。

煙管褲+T恤，用一件質感良好的純色T恤搭配煙管褲，是再好不過的了，兩件簡單的單品就能輕鬆穿出

大牌質感。如果你腿短、比例不佳，短款的T恤配高腰煙管褲會是一組很顯腿長的搭配。

煙管褲的精緻感和大方感，是其他褲子沒有的，所以穿著它到任何場合都不會出錯也不怕失禮。

作為一個職場萌新，備一條煙管褲是一個正確的選擇，所以本仙推薦襯衫+吸煙褲，不會特別正式也不

會太隨便，通勤日常都可以穿。

除了襯衫以外，搭配biouse也是很好的選擇，blouse屬於襯衫中的一種，指的是擁有特殊設計（例如泡

泡袖、荷葉邊、褶皺、收腰、系帶等）的襯衫。

比常規的襯衫多了一份女人味兒，blouse本身已經夠時髦了，在搭配上盡量要從簡，簡潔的煙管褲是

它最好的CP。

除了闊腿褲外比較舒適的一款就屬它了！版型真的很贊，氣質妥妥的在線，小個子搭配平底鞋也不會

顯矮，看看小個子代表周冬雨和艾瑪就知道了！

小個子星人可以選擇高腰/寬腰帶來提高腰線，上衣+吸煙褲的搭配要點是，上衣要塞進褲子里，更顯

腿長。

在鞋子的選擇上也很百搭，高跟鞋、粗跟鞋等都是可以的，不過要注意鞋子顏色與整體look相呼應。

穿不習慣有跟鞋子的小夥伴，也可以選擇樂福鞋、穆勒鞋、小白鞋搭配。

紙袋褲

顯腰細、拉長腿的紙袋褲，你要不要來一條?

所有的褲子中，矮個子或是腰線低，再或者是上半身和下半身五五比例的，選擇高腰褲，緊急時刻絕對

是救命的穿搭！

so~今天小幺要安利的是，闊腿褲的升級版—「紙袋褲」，港真！為這條超美的褲子瘋狂打call！！

紙袋腰褲（paper bag pants）

和名字一樣， 紙袋褲像闊腿褲一樣寬鬆，而腰間用腰帶扎口袋一樣緊束腰部。

使褲子看上去好像是用繩子扎住的牛皮紙袋，基本上款式都比較寬鬆，不但顯瘦顯高還非常的舒適。

（Stella McCartney 2017 春夏）

紙袋褲有高腰花邊修飾，能幫你把腰線提得更高，什麼胯寬屁股大啊！

都能很有效地幫你收一收！

之前的闊腿褲腰部往往是拉鏈或暗扣的形式，隨便塞個上衣就鼓鼓的不好看。

紙袋褲的腰部則有更多看點，顯得更加有趣，小肚腩什麼的再也不怕藏不住了。

束腰的部分有顯腰細、拉長腿的功能，不用PS都能讓你變長腿，就憑這一點都不得不愛吧！

直筒牛仔褲

牛仔褲是每個女生一年四季少不了的單品，選一件時髦和復古相結合又不挑腿型的，直筒牛仔褲就是最

佳選擇啦！

直筒牛仔褲是中性風的代表，隨著街頭風席捲時尚圈，越來越多的人愛上，高跟鞋和直筒牛仔褲的娘

man組合。

直筒牛仔褲一般沒太大彈性，面料硬挺，能模糊胯部和臀部的界限，掩蓋胯寬的缺陷，而且寬鬆的褲

腿能修飾你的腿部線條，小腿粗、小腿彎什麼的都不是事兒~
