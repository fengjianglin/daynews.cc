拇指的這個動作，能決定你的反手擊球是否高遠

2019-12-23 792條評論 1,917次閱讀 485人點贊

絕大多數業餘球友，打反手球的時候，好像都有這樣的一個概念：必須用大拇指推動拍柄，才能擊球有力。

但是很多球友打完反手之後都會出現大魚際酸痛甚至發青的情況，事實上這就是因為大拇指的錯誤發力拍柄撞擊所導致的。

拇指尖頂在拍柄，擊球時，用拇指發「推力」，把球拍推出去。由於拇指指尖頂球拍時拍柄留有對於手掌的衝擊距離，所以容易對手掌造成傷害。

」

體會一下持拍「推出去」的感覺

球拍拿開，是這種握拍手型，注意拇指

這樣使用拇指將球拍推出去是反手發力擊球的錯誤方法，不僅僅手掌受傷的風險大增，而且球拍的無謂移動會消耗掉一部分能量最終影響反手擊球的力度。

打反手球，拇指發力，「拇指尖」是不應該被用到的，被用到的應該是「拇指中下部」捻動，並且「拇指」只能起到輔助發力的作用，不能主要靠「拇指推力」打出反手球。

反手高遠球，「拇指尖」這個部位，不能被用到。

應該被用到的是「拇指中下部」
