胖就是因為懶惰不運動？請看看這位女士

2019-12-21 1,223條評論 2,499次閱讀 874人點贊

專業瑜伽教練、出版作者、健康生活運動提倡者 Jess Amyn

積極樂觀的Jess Amyn

「每天我都還是感覺我需要重新奪回我自己的身體。以防我忘記我屁股很肥，而這不是件壞事。以防我忘記我背上的肥肉無比誘惑，就像你說你今天要節食但是看到的星巴克那塊巧克力蛋糕，然後你說’去你的節食’ 買下來吃，外加一杯焦糖瑪奇朵。以防我忘記的確有牛仔褲可以裝下我的肚子。以防某個混蛋讓我感覺自己像個肥胖的海怪，直到我想起來小美人魚里的女巫厄蘇拉，真正的肥胖海怪，是迪士尼歷史上最酷的反派之一。以防某個蠢貨穿破我的自信心。中指豎起來，肥身體亮出來，我很自豪。我就是這樣活的。以防我忘記，酷女孩遊行永遠不會停。胖是種強大的感情資本，在他人發射的子彈傷下生存是場值得打的戰鬥。」

「我可能永遠也不會理解網路上那些發惡意評論、脂肪羞辱的人。他們難道不知道這樣看起來有多蠢？難道他們不知道所有人腦子裡早就已經安裝好了深刻的 ‘肥胖=死亡’ 那套說法，我們早就聽過很多遍了，為什麼還要彷彿這是什麼新聞似的播報？你覺得在電腦上發送一些充滿恨意的文字會讓他們產生良好的自我感覺嗎？ 我真的完全不在意。我只是感到很困惑，每次有人評論我說我在宣揚一種不健康的生活方式，因為我告訴人們他們可以做開心、愛運動的胖人。我只覺得：難道你生活里都沒有事要做嗎，這樣浪費時間？ 致準備唇槍舌戰的你們，請不要為我與這些蠢貨費口舌。他們有著脆弱的內心，並且活在一種我實在無法理解的枯燥和不幸福的狀態中，他們不值得任何人的時間和精力。 如果你想證明什麼，請活出自己的真相。以你的行為和生活方式來證明一種健康的新方式，這是所有人能找到的最好的論證方法。」

Jess Amyn出版了「任何體型」通用的瑜伽書籍

其實除了Jess Amyn，我們可以在很多領域看到體型並不是那麼「纖細」的運動員。 例如這張關於各類運動員體型對比的照片。 不同的身體基礎，訓練方式和飲食計劃，雕刻了各式各樣的體型。

當然，這屬於比較極端的對比，然而替換到普通人中卻也是這個道理：

我們先天的基礎決定了我們會有怎樣的一個體型框架。因此有的人生來會比較飽滿結實，有的人卻天生體長纖細。

正是這些先天的不同，使我們可以活躍在各自擅長的領域，甚至成為其中的佼佼者。 媒體要求明星藝人們呈現出纖長贏弱的體態，卻不意味著每一個人都應該是那樣纖瘦。 的確，為了獲得健康，我們應該追求平衡的飲食與運動。

但你應該記住：

永遠不要恨自己的身體。

不要苛求自己變成註定無法達成的體型 無論何時肯定自己，你是最好的。

一種體型並不能代表某些褒義或貶義的辭彙，它僅僅是我們生來的樣子而已。
