減脂過程中，每個月下降1-2斤的速度是正常的，不要焦慮

2019-12-23 71條評論 1,776次閱讀 87人點贊

「3-30日減脂-1222-06」

減脂知識

我曾1個月減7斤，使用的方法是大量運動（每天跑步半小時以上，重度無氧運動半個小時以上）以及健康飲食，每晚11點睡，然後我一個月後就沒堅持了。這次我在飲食結構上又調整了，並且還調整了其他細節的東西以及很多方面的不良的生活習慣，但運動量變少了。我希望不要有那麼多運動量，所以，每個月體重下降1-2斤是標配，其他的看具體的身體數據，很輕鬆，生活習慣越來越好，生活感受也越來越好，我覺得這樣才是長久之計。

早餐

冬至，吃了湯圓，因為覺得是儀式，所以無論如何要吃一點，覺得還不錯~確實吃糖很少了，可能因為牙齒的原因，其實並不太愛吃甜的食物，這個習慣對於減脂的人來說，真是好(⊙o⊙)哇~大家都要吃湯圓或者餃子喲~

主食：地瓜、山藥、湯圓

蛋白質：蛋、豆腐

蔬菜：上海青、菠菜、生菜、西蘭花、胡蘿蔔、白菇、杏鮑菇、洋蔥

午餐

時間過得好塊，KEEP一趟，做個早飯，吃個早飯，寫篇文章，喝一杯水，吃一個蘋果，一早上就過去了，趕緊做飯了陪娃吃飯，下午還有一堆事情要做。今天天氣特別好，陽光從窗戶照進來，我坐在瑜伽墊上看群里活動，暖洋洋的。

主食：地瓜、山藥

蛋白質：豆腐、丸子

蔬菜：生菜、上海青、西蘭花、白菇、香菇、海帶、黑木耳、胡蘿蔔、菠菜

晚餐

帶娃的周末，要想著給她做什麼飯，好在，她乾飯就可以吃了，再燉個湯，把我的青菜分給她，就好了~

主食：地瓜、山藥

蛋白質：豆腐、排骨、丸子

蔬菜：奶白菜、茼蒿、包菜、胡蘿蔔、黑木耳、海帶、香菇、白菇、菠菜、長豆、香菜、蔥

喝水

今日份白開水3000ml

水果

蘋果1個

運動

KEEP運動20分鐘

睡眠

昨晚12點睡，早上5點多起來一次，7點半起來，深睡眠2個多小時，還可以。

結語

今天是第一期訓練營結營前最後一天測體重，不是最佳的，今天體脂還上去的，這一期21天，我的體脂下降才1.1%，體重才降了2.3斤，其實這是一個特別健康的下降過程，但是大多數人一定會非常焦慮，其實我也有一點，但是關注到自己身體的變化後，我好像沒有那麼焦慮了。其一，我想到，人體減脂的周期是90天的周期；其次，按健康角度來說，每個月體重下降1-2斤是正常的速度；最後，人的情緒是減脂過程中最重要的因素，千萬不要受情緒控制。

馬上進入第3個月了，並且，過年也要來了，這最後一個月非常關鍵，你會怎麼做呢？期待啊~
