膝關節不穩容易受到損傷？加強力量訓練很有必要

2019-12-20 362條評論 3,259次閱讀 201人點贊

最近有個膝關節疼痛的客戶找到我們，主訴有膝下的疼痛，不敢跑步。簡單了解後知道該客戶在兩年前有過膝關節的損傷，交叉韌帶Ⅱ級損傷，當時未做處理。評估後發現他的膝關節穩定性很差，韌帶鬆弛，肌力也較差，膝超伸嚴重，髕骨軟骨也有一定的損傷。

對於這種由於韌帶原因導致關節不穩、體態異常的情況，除了做臨床處理修復之外，力量訓練也很重要。也許很多人會覺得在膝關節不穩定的情況下去做力量訓練很不合適，但真的是這樣嗎？膝關節損傷後不能做力量訓練嗎？並且力量訓練對於膝關節康復有多重要？接下來我們來慢慢了解一下。

我們的膝關節，上承骨盆，下接踝關節，在負重及行走中起到很重要的作用。所以膝關節必須要有一定的穩定性，這樣才能保證我們的行走及體態不出現問題。

所以膝關節的結構相對要複雜一些。它是由大腿的股骨，小腿的脛骨、腓骨，以及在膝關節前方的髕骨構成的。而在股骨和脛骨的中間，還有一個負責緩衝震蕩的軟骨：半月板。大家都知道，膝關節可以做屈曲和伸直的動作，所以膝關節在做運動時，不可能左右搖晃和上下移動，所以，就有了韌帶-限制膝關節往其他方向運動和穩定住膝關節的。和膝關節相關的韌帶有很多，其中最主要的有那麼幾條：

①位於關節里的交叉韌帶:前後交叉韌帶，主要是限制膝關節過多的往前和往後移動的。在一些比較激烈的競技運動中，交叉韌帶經常會受到損傷，而導致關節不穩定。

②位於膝關節內側和外側的內側副韌帶和外側副韌帶。這兩個韌帶是限制膝關節往左右移動的，當在內側或外側突然出現一個強力應力時，這兩個韌帶就容易出現損傷。

韌帶的損傷會導致關節產生不穩定，如前面所說的那位客戶，就是因為交叉韌帶的損傷導致的膝關節不穩，從而出現了嚴重的膝超伸，髕骨過度下壓脂肪墊而出現膝下疼痛。

除了韌帶之外，膝關節周圍的肌肉也負責著膝關節穩定性的工作。因為光靠韌帶是不夠的，附著於骨頭上多而體積大的肌肉是維持關節穩定的第二要素。

其中就包括了：

①大腿前方的股四頭肌，主要作用是使我們膝關節伸直，髖關節屈曲。

②大腿後方的腘繩肌，主要作用是使我們膝關節屈曲，髖關節伸直。

③內側的內收肌，穩定膝關節內側。

④外側的髂脛束，穩定膝關節外側。

⑤小腿上的小腿三頭肌，從大腿後下端向下走行於小腿後方，穩定膝關節後側。

⑥臀部上的臀大肌臀中肌等。臀部上的肌肉控制骨盆的穩定性，骨盆穩定性足夠，膝關節才更穩定。

韌帶損傷後很多人趨於停止運動，休息恢復。在急性期這是正確的，我們需要停止繼續讓組織損傷的運動。但長期的不運動會使得肌肉力量下降，甚至於肌萎縮，不能給予關節很好地拉力，就不能很好的穩定好膝關節。

所以，韌帶和肌肉是相輔相成的，韌帶脫離了肌肉難以很好地穩定膝關節，肌肉力量的提升能更好地使膝關節穩定。不止是對於膝關節損傷如此，對於術後人群也是如此。

膝關節力量訓練的步驟

在疼痛緩解以後可以先做一些局部的力量訓練。

①核心激活：仰卧位，雙腿間夾一瑜伽球，雙腿伸直，核心收緊，下肢和上身同時抬起，雙臂在身體兩側做小幅度的擺動，同時保持腹式呼吸。10s/組，3組。

②扶桿靜蹲：站立位，雙手扶著一桿，雙腿打開與肩同寬，膝蓋與腳尖方向一致，手

只是作為一個固定，不需太用力。腰背挺直，慢慢地向後屈髖，然後再慢慢地屈膝，感受到臀部和大腿後側的發力。保持30s。3-5組。

③負重提踵：在雙踝綁沙袋，站在一台階上，前腳掌負重，後腳掌懸空，雙手可扶桿，雙腿伸直，依靠身體的重量使腳跟下沉和抬起，15次/組，3-5組。

④彈力帶訓練：將彈力帶一端繫於桿上，另一端繫於小腿遠端，站立位，挺直身體，伸直雙腿，做腿的外展、內收、前屈和後伸訓練。每個方向10次/組，3組。

④閉眼站立：脫去鞋子站立於一軟半球上（也可站在其他軟面上），雙腳打開於肩同寬，屈膝站立，雙手叉腰，腰背挺直，愛上眼睛保持穩定。

局部力量加強後可進化做些動態訓練及有氧訓練加強耐力。

①橢圓機上熱身訓練，注意膝蓋不要內扣，5分鐘左右。

②扶繩單腿蹲：雙手拉上方的繩，單腿支撐，另一腿伸直膝蓋往前屈，核心收緊，單腿下蹲起身。10次/組，3次。

③仰卧位，雙腿下放一瑜伽球，核心收緊，抬起屁股時雙腿屈膝把瑜伽球往身體方向滾動，腳尖勾起。10次/組，3組。

④原地跳：雙腿打開與肩同寬，核心收緊（前腹和側腹都收緊），屈髖大於屈膝，原地屈髖跳起。10次/組，3組。

膝關節損傷後的康復要循序漸進，不可心急吃熱豆腐。力量訓練也要從局部到全身，從蹲，到跳，再到跑，不能太早進行跑步，避免在膝關節穩定性還不足的情況下再次出現損傷。
