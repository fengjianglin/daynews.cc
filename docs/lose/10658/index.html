冬天怎麼吃才能瘦？取代沙拉的10道暖胃食譜打包送你

2019-12-06 210條評論 1,215次閱讀 276人點贊

上海本周正式入冬，新一輪降溫來勢洶洶。頂著冷風回到家，只想趕緊吃點熱乎乎的東西。靠著吃沙拉減肥，在冬天行不通了。但大魚大肉的吃一冬，這大半年吃草的辛苦就白費了。有沒有兩全的方法，吃得熱乎又不長胖，做起來還不費時間？

⓶ 熱呼呼的主食

低溫會讓我們對主食的需求更大，所以冬天的主食既要滿足我們儲存熱量的本能，還要味道好，吃完之後還要降低我們對其他碳水的渴望，紅薯和玉米是最佳選擇。雖然和米飯麵條的熱量差別不大，但紅薯和玉米在膳食纖維、維生素和其他微量元素上加分不少。當然，再優秀的主食，吃多了還是會胖哦~
