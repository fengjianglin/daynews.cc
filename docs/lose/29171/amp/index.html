小聶：業餘跑者的體能該如何練成

天天要聞
2019-12-23

有部分跑友給我提建議，說我太刻意強調跑步技術的作用，對體能訓練隻字不提。其實不是這樣的.

我也是非常重視體能訓練的撒，要不然，我怎麼會平均每個月保持280+公里的跑量呢，或是每次給學員制定訓練計劃都基本上保持5-7個小時的跑量呢。

再說了，我在文章里也寫過我對體能訓練的理解和具體技巧的。而今天，討論體能訓練的技巧，也只是補充一些以前的觀點。

對業餘愛好者來說，體能訓練其實很簡單。

重點是：耐心和堅持。

體能的積累需要時間，而這個時間是考驗個人對跑步的投入和理解。

有人真心熱愛跑步，那麼，積累的時間再長，他也會感覺很短。

而有的人僅僅是跟風式、社交型的跑步，那麼積累的時間對他來說就是浪費時間。

所以，我們該如何界定這個時間呢。

從人體生物學原理來說，體能積累是一個反反覆復的循環過程，對心肺功能反覆地刺激恢復強化，對肌肉細胞反覆地刺激破壞恢復再生。如果我們真正的按照這個自然規律去循序漸進地提高自己，那麼，積累的過程可以長達10年，20年。

曾經有個學員諮詢我，他問我如何界定配速跑和放鬆跑的訓練感覺。

我說配速跑會讓你跑得很舒暢，而放鬆跑可能會讓你懷疑人生。

他思考了一會，回了一句：懂了。

體能訓練，首先要從慢跑開始。

理論上，我們每個人都有自己能力範圍內額定的放鬆跑配速。但很多人對自己放鬆跑配速把握不好。為什麼呢？性格和理解使然。

舉個栗子

假如一個人的10公里PB是50分，配速是500.那麼他的放鬆跑配速理論上可以是600配速，但實際運用上，需要結合他個人的年齡、跑齡來衡定。那麼，他的放鬆跑配速可以從550-650都是合理的。

當一個人的10公里配速是500的時候，他會發現他在日常訓練中跑520-530配速是最舒服的，感覺很舒暢。事實上，這個感覺其實就是他的全馬配速。

但要求他去跑600-650配速的時候，他會發現跑動很慢，在心理上有點不自然，從而感覺不到愉悅感。這就是我剛才說的懷疑人生的感覺。

而有的人會產生疑問，為什麼在日常訓練中進行配速跑，感覺很舒暢，但比賽中為什麼會感覺很累呢？答案就是我們大部分人在比賽中容易迷失自己，會跑出超出能力範圍之內的配速，當然會感覺累很多，甚至會在後程崩潰或抽筋。

而從人體特性來說，比賽中腎上腺的刺激作用，實際比賽配速自然會比訓練的配速快一點，這些都是合理現象。

因此，我們在進行有效的放鬆跑時，當你感覺你的目標配速很慢，你也是要保持這樣的感覺來跑的，而且，從心理上去理解和接受這個感覺，讓自己適應，才能達到最好的有氧能力的積累效果。現在有很多跑友比較迷戀心率訓練。特別是比較容易接受低心率跑。

事實上，低心率和低配速慢跑意義和作用是一樣的。僅僅是監控手段的不同而已。

有的人，在慢跑中感覺很慢，會懷疑人生，甚至會覺得這樣的慢跑配速不利於發朋友圈。

而低心率跑，就算顯示同樣的配速，也會感覺高大上。

其實就是心理作用。當你老老實實低配速慢跑的時候，你會發現，你的心率同樣很低。

慢跑訓練非常考驗性格。也就是我說的耐心。

在我的學員群裡面，有3名學員短距離能力非常強，但他們耐力很差，就是因為他們過去在跑步訓練的時候過早進行速度訓練導致的。

而我給他們的訓練計劃，會要求他們跑極度懷疑人生的低配速。開始有一名學員非常不理解，感覺不可思議，甚至和我鬧彆扭。

事實上，我這樣的安排不止是針對他們的體能特點，更多的是在磨練他們的性格。

在大量的懷疑人生的低配速中磨練他們的性格，比單純的體能訓練更有利於他們的進步。

特別是對於跑步的理解和健康的運動方式。

後來，當其中一名從最初的五公里19分，但全馬412的學員進步到全馬324的時候，他就明白了我的苦心了，現在讓他繼續慢跑，他也很高興地接受了。

體能積累，堅持很重要。不能三天打魚，兩天晒網。

我們如果真心熱愛跑步，堅持是衡定我們進步的基石。

從人體生物學的原理來說，運動能力，只要中斷一周開始，就逐步下降，中斷超過三個月，基本就回到解放前了，那麼進步就是不可能的事情。

因此，大道理不說太多，大家也應該明白堅持的意義。

說到這裡，不知道我表達清楚沒有？

好了，以後會多一點分享體能訓練方式的理解。
