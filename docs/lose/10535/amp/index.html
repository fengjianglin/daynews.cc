為何肌肉強悍的健身教練，很多業績不如瘦子教練？

天天要聞
2019-12-10

經常聽到一些健身教練感慨，自己肌肉練得好，還打健美比賽，為何業績總是不如健身房的瘦猴教練？當然部分私人教練本身脫衣有肉，穿衣顯瘦，儘管他們的身材符合普通人的審美，卻也被冠以瘦猴的蔑稱，這著實是令人看不懂了！

這裡以一個普通健身會員的角度來說這個問題，深究原因，或許我們就可以發現為何很多練得好，肌肉很大的健身教練，業績不如瘦子教練。

之前在健身房有一位健身接近半年的男生，也是筆者的朋友，最近他買課了。這著實令人好奇，詢問他練得還不錯，乍想去買課了呢？他回答道：「儘管練了這麼久，但是終感覺欠缺了點什麼，或許找個健身教練帶著練下會更好點！」

那問題來了，為何他之前不買呢？練了半年才買？其實原因很簡單，因為他慫，不敢去搭訕健身房的私人教練，所以一直沒有人指導。當然這次為何又買了呢？而且是買的一位新手健身教練的課程（說真的這位私教真的是毫無訓練痕迹的那種，甚至能看到明顯小肚子），教練的身材不如他的會員，奇怪的事情就這樣發生了，當然在健身房這很常見！

一名有小肚子且毫無訓練痕迹的教練，為何能開發到這位會員呢？且不看私人教練專業如何，作為一名新手教練，他急切的渴望業績，沒有業績就會被辭退。所以他在健身房總是積極的搭話，去幫助很多健身新手，儘管10次中會有8次被拒絕；一天會打上百個電話詢問要不要上節體驗課，要不要帶著熟悉健身器械，儘管100個電話有80個是拒絕，有10個考慮，有5個可能會來，有1個才可能會買課。

所以很明顯，健身教練的肌肉雖然的確是名片，但客戶都是被動的，他們不會主動找你，甚至因為你太酷了，肌肉太大了而害怕。而健身教練想賺錢，需要充分發揮主觀能動性去開發客戶，而不是守株待兔，找到一個好平台，每天規定打100或者200個電話，每天規定要指導多少位會員，這期間你可能會面臨太多的拒絕，但能堅持下來的人，才是「剩者」為王，做多少就能收穫多少。

不管是健身教練，或者是各行各業的培訓教學或者銷售，很多人總抱怨大環境不好，抱怨自己肌肉練得這麼好，乍不如瘦猴教練？那沒有意義，這是把鍋推給了環境甚至會員的審美。提高業績的方式很簡單，那就是努力、堅持不懈的去開發客戶，總不能想著靠著一身肌肉，坐在前台就能有人主動買課，這並不現實。就算是強如傑夫教練，也得出視頻打廣告；黃哲勛不也總宣稱自己是自然健美，然後推廣自己的健身課程么。

學會在自己身上找原因，而不是抱怨大環境，甭管客戶喜不喜歡肌肉教練，甭管瘦子教練多麼會說話，甭管自己的肌肉會不會嚇到別人。努力切實的朝著目標努力奮鬥，而不是僅僅舉鐵的時候很勤奮，而打點電話，場開客戶就邁不開腿。所以不管是任何行業，有了這樣的精神，想必都會取得不錯的成績。
