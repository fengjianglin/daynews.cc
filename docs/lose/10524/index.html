水花兄弟一起走，誰有緩震誰是狗？庫里6開箱測評竟令人期待滿滿

2019-12-10 1,536條評論 4,896次閱讀 3537人點贊

Hello大家好，這裡是最懂你的WEN鞋評，我是果子，大家記得點擊關注，不錯過我們每日的更新喲。

今天要給大家帶來的，是讓我和文胖真香的一款鞋：UA的Curry 6代，廢話不多說，先來看看這款鞋的基礎信息吧。

基礎信息

尺碼：US10 EUR44

發售價：1199

市場價：900

單只重量：360G

有無偏碼：無

啊，我們這個「UA黑」終於開始做UA的球鞋了哈，此處彈幕應有：你們恰飯咯，你們放棄原則咯等等一系列彈幕飄過。

這裡先解釋一下，首先，我們之前不做UA的鞋的原因很簡單，腳感太磚，價格太貴。庫里6上市都一年了，不挑配色的情況下，黃金碼還是1000左右的價格，是真的買不起，而且胖子之前穿過庫里345，給他留下的心裡陰影真的是太大了。

因為後台確實每天小夥伴都在說想看一下庫里球鞋的測評，正好偶然看到了UA官網黑色星期五的活動，文胖就搶了一雙。

聽周圍的朋友反應說庫里6有點偏小，所以我們買了44碼，然後我也借了一雙42.5的來，我都試穿了一下，我是覺得42.5的尺碼對我來說，貼合感更高，胖子覺得42.5很擠，然後穿44的很舒服。但是我穿44就大了很多。

所以這雙鞋的尺碼，如果大家穿毛巾襪的話，我覺得按照正常碼數去買就好，如果你是寬腳、高腳背，我覺得買大半碼比較適合。

這雙鞋的鞋盒，整體是藏藍色的，正面有庫里的標誌性logo，一個形似30（庫里的球衣號碼）的「3C」字樣，C就是庫里（Curry）的縮寫。

背面是安德瑪的全稱。

打開鞋盒，裡面有一個UA的籃球袋，還有一個官方的訂單，證明了這雙鞋是從UA的官網入手的。

我們這款鞋是「國際大道」配色的庫里6。它的設計靈感來自於奧克蘭的國際大道，這條路也是通往甲骨文中心的必經之路。所以這樣一個配色還是很有意義的。雖然本賽季的金州勇士已經將主場搬離了甲骨文中心，但是毫無疑問，這座球館還是給庫里以及庫里的粉絲留下了很多幸福美滿的記憶。

來看球鞋吧，庫里6代其實依舊延續了前幾代這種低幫的造型，庫里系列球鞋外觀的一大特點就是，它不是那種流線型，整體輪廓更像是一個圓形的線條，所以看起來就很飽滿。

球鞋整體是一體式的鞋面造型，鞋口處有一個襪套的設計。鞋面的材質採用了不同密度的編織，大家可以看一下，從前掌到後跟位置，它的紋路和密度是在一直變化的。

這是那種相對比較厚實，比較粗糙的織物。強度還是可以的，那把手探進去，可以摸到裡面還有一層類似於硬質橡膠的那種內襯，所以鞋面總體來說強度還是可以的。

在前掌的外側的，有很大一部分的熱壓材質，加強了前掌位置的支撐包裹，同時起到了一個防踢的設計。

鞋舌部分，是這種高彈力的編織材質，而這一部分是沒有內襯的，所以這部分就很透氣，大家可以看到，我把手伸進去在鞋面外是可以看到的。

同時，鞋舌這部分的延展性還是不錯的，降低了一體式鞋面對寬腳、高腳背朋友的不友好，適應性還是比較強的。

鞋舌上有一個皮革的材質，上面用這種鏤空的處理，刻了一個安德瑪的logo，一個庫里的logo，還有一個6，證明這是庫里的第6代簽名鞋。

它的鞋帶也比較有意思，看起來很像是動態的鞋帶系統是吧，而且鞋帶扣這一圈用到了和鞋面色差比較大的這種熒光綠，是不是感覺它很像是，這是一根線，從鞋身裡面貫穿了整個鞋面。

但其實不是，這些都是單一的鞋帶扣，因為我們不管拉哪個扣，都是拉不動的，所以庫里6的鞋帶系統是一個「偽動態系統」。

後跟部分，有一個很明顯的向內收斂，在腳踝處有一個不是很明顯的橫向填充。

球鞋內置的港寶也是很硬的，面積也很大，同時，外面還有一圈硬質塑膠的加固，整個後跟的包裹和鎖定看起來還是很值得信賴的。

接下來我們聊聊它的中底。

本次庫里6終於給庫里的簽名鞋加上了緩震科技，不再是單一的EVA緩震材質。

在庫里6身上，UA搭載了當時最新的緩震科技HOVR。這項科技在UA的跑鞋上，腳感還是不錯的，但是僅從試穿的角度來講，雖然我們現在用手使勁按呀後掌這部分可以看到一些形變，但腳感整體還是偏磚的，這也是UA籃球鞋最大的一個槽點。這項科技的具體體驗肯定還是要等到實戰之後再和大家講述會比較客觀，這裡就先不多說了。

這雙鞋中底整體的厚度還是不錯的，前掌有做很明顯的上翹設計，所以看起來是很適合突破黨的。後跟有做很明顯的上翻設計，提升了穩定性。

同時啊，庫里6中底還有一個亮點就是，在中底科技里埋進去了很多網格狀的線，我記得當時官方說明是為了限制中底的形變，不過我覺得還是裝飾作用大於實際效果吧，畢竟這個中底硬度，應該不會有什麼過分的形變的。

同時在後跟這裡，還有一塊塑料的tpu，上面雕刻著庫里的logo，我覺得算是這款鞋顏值畫龍點睛的一個地方。

鞋底的紋路真的是很分裂，前掌外側是圓形的紋路，內側則是網格狀的，中足位置向內收攏地很明顯，感覺又是一雙對扁平足不太友好的鞋子呢。

鞋底中間的位置就是豎向的紋路，後跟處是圓形的紋路。

同時在鞋底我們還可以看到很多彩蛋：前掌紋路里藏著庫里老婆以及兩個乖巧女兒的名字，中間部分的紋路鏤空著庫里的logo，內側的tpu上，是庫里的手寫簽名，同時還有他自己的名言：I CAN DO ALL THINGS。

最後來看下這款鞋的抗扭，有形變的位置只是前掌的發力位置，整體的剛性是很棒的。

總的來說，不論是顏值還是設計，這雙庫里6我個人是很喜歡的，我還是很期待它在場上的表現的。過幾天我們也會給大家帶來這款鞋的實戰測評，有興趣的各位記得關注我們後續的實戰測評咯。
