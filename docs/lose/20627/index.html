內家拳的秘傳和真傳，不得真傳就難達上乘

2019-12-17 589條評論 1,781次閱讀 331人點贊

內家拳法向來以真傳秘技著稱，不得真傳就難以練至上乘水平。這使得「真傳」問題永遠是武術界最敏感的話題之一，既是傳統武術信徒對廣大武術愛好者缺乏真實水平的借口，又是造成懷疑論者曲解武術的因素之一。那麼，內家拳的秘傳和真傳究竟是怎麼回事呢。

就我的接觸來看，內家拳確實有秘傳，並且是否得到秘傳真傳是能否達到武術真實水平的關鍵因素之一，這一點是由所謂「內家拳」的性質決定的。真正的內家拳講的是用「道理」打人，從最好理解的角度講也就是使自身的運動機制優於對手從而獲得戰鬥的勝利；所以練這種功夫把自身運轉研究的越細緻越合理功夫也就越精純。

內家拳的秘傳就是老師在學生練拳的過程中不斷在適當時機告訴學生適當的運動方式，以至最後達到符合內家拳標準的運動機能和運動方式。比如︰如果一個人練掄胳膊，掄一千下肯定練的比一下不掄的人力氣大，這裡面沒有更多道理可言，相比之下可以叫做「笨功夫」；而合格的老師可能在學生掄到五百下時看出他這一層次的素質和熟練程度都已經達標，而讓學生在後五百下中加上肩胯的折迭運動和肢端的螺旋運動，就可以把練出來的掄勁轉化成清脆的抽勁；如果老師在學生練到後四百下時感到滿意，就會在學生的最後一百下練習中加入胸肋的支撐開合內容。這樣，同是掄了一千下胳膊，前者練出了把力氣，掄一般人一下也受不了，但比後者差了兩個境界。這就表現出了是否得到真傳的區別。許多武林典故中一些習武者痛下苦功也趕不上嫡傳弟子的原因也正在於此。

這種秘傳的難點還不在於知道練拳的要領是什麼，因為內家拳的訣竅現在通過傳媒大家也或多或少知道一些，對其理論都有一些了解。真正的難點是老師知道在學生練到什麼程度的時候加上什麼樣的訓練內容，最終通過一次次糾正達到高水平。這起碼要求第一是老師有豐富的教學經驗，能準確的把自己達到的水平傳遞到學生身上；第二是老師要能定期監控學生的訓練過程，了解學生的狀態，把握加東西的節奏。

這裡順便再談一下內家拳訓練的強度問題，我的體會是其訓練強度體現在要費腦筋保證每一個動作都符合正確要領上，是一種「心路活兒」。這種訓練不體現在時間上，而體現在注意力上；例如我們練拳時要保證半小時內每時每刻全都把拳做對，就會體驗到是一種多麼大的心理和生理強度了。但實際上這是不可能的（即便是喬丹也不可能在整場籃球賽都保持注意力，練拳也同理），所以需要在更多的練習中思考和尋找正確的狀態，以期最後達到不期然而然做到正確的地步。這與單純的體力強度是不同的。
