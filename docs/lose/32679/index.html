BK小知識｜比仰卧起坐更高效的虐腹動作，讓你擁有夢想中的腹肌

2019-12-23 1,981條評論 4,546次閱讀 3624人點贊

大部分人初次步入健身房的目的很簡單，想要好看的腹肌馬甲線。

但常常，腹肌馬甲線還沒看到，腰痛脖子酸就先找上門了。

究竟怎麼做才能安全、高效的訓練腹部呢？往下看⬇️

低效的仰卧起坐

一說到腹部訓練，許多人第一時間想到的就是體育課做過的「仰卧起坐」。但它其實不是一個很好的腹部訓練方式。

因為它只有在動作前半程有練到腹肌，後半程主要是髂腰肌發力

圖片來源網路

髂腰肌由腰大肌和髂肌組成，從上圖中它們的起止點可以看出，髂腰肌的作用是讓骨盆和上半身前屈，在仰卧起坐中的表現就是把上半身拉起來⬇️

而我們常說的腹肌，即腹直肌，它的主要作用是使上半身彎曲⬇️

所以，「仰卧起坐」這個動作，僅一小部分有練到腹肌⬇️

除此之外，「仰卧起坐」還容易導致脖頸、腰背酸痛。

因此，想練腹肌，這個動作可能不太建議，更符合腹直肌發力模式的訓練動作——「卷腹」。

正確「卷」腹

既然腹直肌的生理結構決定了它是作用是彎曲上半身，那要想加強它，乾脆直接把訓練目的改成捲起上半身。

下顎微收雙手交替放在胸口並吸氣，然後腹部收緊帶動肩胛骨離開地面，同時吐氣，保持住1-2秒即可。這樣腹直肌訓練效率更高，刺激更強烈。但有幾個點要格外注意：

第一個就是在動作開始前，一定要刻意讓腰部貼緊地面⬇️

這樣做一方面能夠減少腰部代償，避免腰椎壓力過大；另一方面是讓腹部更易找到發力感。

第二個常見錯誤是脖頸前伸借力，通常是仰卧起坐帶過來的壞習慣⬇️

這個壞習慣帶來的問題就是，練完了脖子比腹部還難受❌

針對這個問題，可以嘗試放鬆脖頸，頭部輕靠在墊子上，雙手抓住墊子，訓練過程中讓墊子始終支撐著頭部⬇️

在逐漸調整之後，你就可以做出一個標準的卷腹了⬇️

建議的訓練量是15-20個，做4-6組。

那如果幾乎沒練過腹部、腹直肌力量薄弱，做5個都非常吃力要怎麼辦呢？

請往下看～

輕鬆實現進退階

卷腹訓練有2種退階方式，先介紹第1種動態助力訓練。

彈力繩不僅可以增加強度，也可以用來減小強度。

將彈力繩固定在門框上或請小夥伴幫忙拿住，就可以開始訓練了⬇️

因為彈力繩在動作啟動階段提供的助力更多，頂峰收縮階段提供助力更少，因此它能夠很好地幫助我們完成訓練動作，十分推薦。

另一個退階訓練為靜態靜力訓練。先保持坐姿，然後上半身慢慢下放的同時感受腹部發力，定住並保持。

對於沒有彈力繩或不方便使用彈力繩的情況，可以使用這個方法來鍛煉腹部。

在順利完成訓練並已經有能力做好標準卷腹之後，可以嘗試將雙腿抬起懸空，進行簡單的進階訓練。

去掉雙腳的支撐後，可以進一步減少髂腰肌的借力，同時也對腹部肌群的控制力要求更高，渴望進階的小夥伴可以嘗試一下

運動是為了身體健康，同時讓身材更加好看。但錯誤動作如果不及時調整，盲目練習，不僅效果差，還可能帶來其他的運動損傷，得不償失❌

所以大家在訓練前，一定要先學習好正確的動作模式，這樣才能安全、有效地朝著理想身材前進，一起加油吧！
