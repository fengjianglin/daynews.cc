誰說奶爸就沒時間健身？這些健身方式不僅能健身還能培養感情

2019-12-06 54條評論 2,630次閱讀 1163人點贊

80後的朋友們大概已經都成了奶爸，有了自己的小公主或者小祖宗。無論是工作還是生活都開始擠壓自己的時間了。基本上就沒有專註健身的時間了。

為了幫助那些喜歡健身並且不想做油膩大叔的男同胞們。特地在網路上搜集了一些奶爸專用健身動作。不但能幫助你健身，還能培養和孩子們的感情。

一起來學習下吧！

抱娃手臂彎舉

抓住孩子的咯吱窩和大腿，猶如做彎舉一般，慢起慢落。

舉娃臂屈伸

雙手握住孩子的咯吱窩，肘部打開，伸直軀幹，猶如臂屈伸般上下移動。

背娃引體向上

把孩子背在身上，正常完成引體向上，注意慢起慢落。

背娃俯卧撐

讓孩子抱住你，腰背保持平直，就像完成負重俯卧撐一樣，最後趴在地上等孩子下來再爬起來。

PS：請一切以安全為上。要時刻注意孩子的安全！

這些動作雖然既能陪孩子玩耍也能起到健身作用。但是終歸和使用健身器械有差異。若是你特別愛健身，還是建議購買專業的健身器械，如啞鈴，跳繩，俯卧撐支架，健腹輪等。有些健身小器械也是可以進行親子互動的。
