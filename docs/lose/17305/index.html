學太極拳如何練到理法合一 | 實修營同學分享

2019-12-13 2,626條評論 4,460次閱讀 2310人點贊

太極觀真蘇州實修營的目標就是理法合一。

第二期太極觀真實修營圓滿結束了，我回到家中開始回味咀嚼消化實修營五天內所學東西。

這是我第二次參加太極觀真實修營訓練班，我慶幸：我來了。

都說太極拳七分知己功夫，三分知彼功夫；那怎麼才能練出知己功夫呢？學習，學習，再學習；除此之外別無溪徑。

最為關鍵的是這七分知己功夫究竟應該如何練？

太極觀真蘇州實修營在康偉老師的言傳身教下由知到覺體驗感悟更深了一步，正如同學所說：同樣在一座山上，你站在10米高的地方和站在100米高的地方看到的風景感受是不一樣的。

拳勢中的機在哪？轉關又在哪？為什麼說得機得勢？看到了，學習了。通過實修營學習進一步理解了磨盤腰與拳論中所說的命意源頭在腰系的關係，為切實做好太極六式基本功法之三.四.五式增加了信心從而明確了練出功能的重要性，不再彷徨不再懈怠！

六式功法之一式結露凝珠動作看似簡單，要求做到位可沒那麼簡單，小臂也是要動的，尤其兩臂下落時小臂不能僵在那裡。進一步領悟明確了如何把沉肩墜肘松而不懈的狀態運用到拳勢中。通過老師的修正認識到了自己的不足之處，印象更深刻，例如不要為了松肩把手臂懈掉。

通過學習對在拳勢中什麼時候把身體拎起來，什麼時候屈伸有了新的認識和概念。通過推手認識到沉肩墜肘的重要性。明白了容和讓的含義，容–我中有你；讓–有你沒我。總之，太極觀真蘇州實修營收穫多多正在消化進行中。按康老師教的方法修練，力求把拳打出太極味真正做到理法合一。

這次太極觀真蘇州實修營來了一位練了50多年太極拳德高望重的81歲朱老拳師，通過和康老師搭手體驗明白了真棚勁兒不丟的道理，深有感觸的說：「我見到了真太極拳，我回去以後按康老師教的方法習練……。」

還有一位軍人出身的老同學夫婦倆也是第二次參加實修營，已經多次參加康老師的培訓班，感受頗深的深情表白：「只要是太極觀真的培訓課，我毫不猶豫都參加」。還有從新疆不辭勞苦跑到蘇州來參加太極觀真實修營學習的同學等等……。

充分說明了大家對真太極拳的追求和熱愛，上善若水見太極，太極觀真——觀傳統文化之真諦。

太極觀真實修營一群善良的人聚集在一起學習迸發出友愛的火花，令其感動的人和事很多。

第二期太極觀真實修營訓練班，我和同學們一致認為時間短，內容多，任重而道遠！

楊俊平
