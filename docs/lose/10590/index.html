瑜伽中常見的幾個梵語，理解背後的意義會深化你的瑜伽練習

2019-12-08 2,093條評論 4,038次閱讀 1174人點贊

瑜伽課上會經常聽到各種梵語或者唱誦，你知道它們的意思嗎？以下是您在課堂上最常聽到的5個梵語。一起來看看它們是什麼意思。

1.Asana

Asana正確的發音是AH’-sah-nah。從字面上看，它的意思是「座位」，但在瑜伽課中，它是「姿勢」的意思，也就是瑜伽體式。例如，Balasana =嬰兒式，Navasana = 船式 ……等等都是Asana。但比起這個字本身，更重要的是，它只是瑜伽八支（即是瑜伽里八大修習領域）裡面的一項。所以現代人大多隻包含身體練習的所謂「瑜伽」，其實只是瑜伽的一小部分而已。

2. Namaste

這是我最喜歡的梵語單詞，Namaste[na-ma-ste]是由幾個印度音所組成：「Nama」代表著鞠躬，「as」意味著我，及「te」代表你。因此，Namaste即意味著 「你我互相鞠躬」 或 「我向你鞠躬」，中文的意義上可以理解為您好、及致意的意思。
Namaste是代表著彼此靈魂的交流，透過Namaste，瑜珈的哲學中說每個人的心輪中心(Anahata ; Heart Chakra)都有著神聖的力量來源。透過Namaste的姿勢，人與人的關係與信賴感可以透過這個精神中心而相互交流。

3.Om.

瑜伽聖音「OM」，是印度教和佛教禱告語，也被印度教徒認為是最神聖的聲音，宇宙原始音。它是一切曼陀羅（咒語）的根源。它由梵語的三個音（a）、（u）、（m）構成，代表創造、維繫和毀滅三神合為一體。據說是在冥想中可以聽到的聲音。

Om的書面版本已成為瑜伽的普遍符號，它經常會掛在瑜伽教室的牆上，有很多伽人還把它紋在身上。從本質上講，我們都是這個宇宙的一部分，一直在移動，總是在變化，總是在呼吸。當您吟誦 Om時，便是在感受這種振動。

4.Shanti

和平。當您高喊「 Om shanti shanti shanti」時，這是對和平的呼喚。Shanti，源自梵文，英文亦寫作Santhi或Shanthi，主要指和平與平安（peace），同時包含著安息（rest）、平靜（calmness）、安寧（tranquility）和幸福（bliss）的意思。在佛教和印度教傳統中，一般念誦三遍，代表身體，言語和思想上的安寧。
瑜伽行者透過動作練習及冥想去培養平靜的心，而念誦或唱誦Shanti是其中最常聽到的

5.yoga

我們都知道瑜伽是身體，思想和精神的結合。Yoga，印地語：）是一個漢語辭彙，最早是從印度梵語「yug」或「yuj」而來，其含意為「一致」、「結合」或「和諧」。這就是瑜伽這個詞的意思。確實，這是連接我們的身體，思想和精神的一種方式。他把肉體結合至最佳狀態，把生命和大自然結合到最完美的境界。
