喜歡運動的人，精神力意志力會比一般人強嗎？答案是肯定的。

2019-12-22 1,809條評論 2,739次閱讀 156人點贊

哈嘍，大家好我是健身AK，今天談的這個問題相信很多人都有疑問， 我也是健身好多年了。我也是健身好長時間了，運動健身的人精神力意志力是要比其他人要強，下面我就具體說一下。

1、因為在我們運動過程中，我們身體會非常的活躍，就我個人而言吧在我以前沒有運動的習慣，就是非常懶的那種，感覺每天無精打採的一點活力都沒有，做什麼事情都沒有精神。自從我回來每天都堅持運動，我一般是每天早上起來跑跑步，下午呢做下俯卧撐，舉下啞鈴，就這樣堅持了一個多月吧！我感覺我整個人和以前不一樣了，就是每天精神都特別好，幹什麼都有活力，不像以前那樣每天都渾渾噩噩的。

2、運動也是可以提高我們的意志力的，為什麼這麼說呢？就拿我自身的經歷來說吧！這樣可能更有說服自己，你說，如果我自己都沒有經歷過說這種話就是沒有任何的說服力的。我發現一個問題就是我現在每天都堅持運動，每天我都會有一點突破，不僅僅我在運動過程中我更能忍受痛苦，就連我在生活中做很多的事情我都能堅持突破，這有可能就是運動給我的改變吧？以前如果遇到稍微大一點的困難我就選擇放棄了，可是現在我都會儘力的去克服。這也許就是我在運動過程中意志力得到磨鍊了吧，使我的意志力變得更強。

3、其實，運動能改變我們的精神和意志力，還可以改變我們的形象和氣質，運動的人和不愛運動的人站在一起是看的出來的，前者更有活力整個人看上去氣質會更好一些，不然的話現在為什麼那麼多人走進了健身房，其實運動的好處還有很多，具體的還是要自己親身去感受才會知道。

健身不僅可以改變自身的身體素質，還可以改變自己形象和氣質，每天健身一小時，快樂生活一輩子。
