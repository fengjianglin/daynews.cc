太極拳拳理與道家思想的理論關係

太極拳,是運用我國道家無極和太極的理論觀、陰陽學說、五行學

說、八卦演變之法,結合人體內外運行之規律與道家內丹養生之法,而

形成的剛柔相濟、動靜相間的優秀武術拳種,是武與道的完美結合。

要想理解太極拳,首先要理解太極之含意。太極即大恆,「大「指空

間,有至廣至大之意,也就是無大無小;「恆」指時間,有至常至恆之意

也就是無古無今。

太極拳動則生陽靜則生陰,這是太極拳理中對陰陽的解釋。「陰

陽」是古代哲學理論中用來說明一切事物內部不同屬性之間的相互關

系的代名詞,陰與陽具有對立統一、相互依存、相互協調、相互轉化的特

點。陰中可有陽,陽中可有陰:陰能生陽,陽能生陰。太極拳就是在這

理論的基礎上,創造出的剛柔相濟、順逆纏絲、內外相合、上下相通

快慢相兼、形意結合的拳種。

道家還講:無極生太極,太極生兩儀,兩儀生四象,四象生八卦。這

種觀念最早使用的,是《莊子》和《易傳》,不過在中國的典籍中,《易》素

來被視為窮盡天地之奧秘的典籍,其對成卦過程的分析,也就是對天地

開闢的概述;其對卦的解釋,就是對各種天地自然現象的描述。太極生

兩儀,便是由太極的分化形成天地的過程,兩儀即是天地,也即是陰陽。

在道家獨特的理論體系中,太極的觀念始終是其宇宙觀、修養理論和法

術理論的重要基礎。道家長期以來在自己的修行實踐和理論探索中

用太極的觀念,並且大大地豐富了太極的內涵,形成了以太極為核心的

系統學說。其中最重要的,便是《太極圖》了

《太極圖》最初由陳摶所傳,原叫《無極圖》。陳摶是五代至宋初的

一位得道高人,是位傳奇性的人物,對內丹術和易學都有很深的造詣。

據史書記載陳摶曾將《先天圖》、《太極圖》以及《河圖》、《洛書》傳給其

學生种放,种放分別傳給穆修、李溉等人,後來穆修將《太極圖》傳給儒

家周敦頤,周敦頤寫了《太極圖說》加以解釋。現在我們看到的太極圖

就是周敦頤所傳的。

《太極圖》最初叫做《無極圖》,叫做《太極圖》時,和原圖的圖形一

樣,但讀法不同。兩者的圖形如下:

煉神還虛復歸無極(脫胎成仙)

無極而太極

取坎填離(得葯)

五氣朝元(和合)

五行各一性

煉精化氣鍊氣化神(煉己)

乾道成男

坤道成女

玄牝之門(得竅)

生化萬物

(此圖從下而上,以明逆則成丹之法)

見《朱氏易卦圖》上卷

太極圖

如上面左面一圖,是涵括內丹修鍊之理的無極圖。讀它的順序是

從下而上。最下一層稱為「玄牝之門」,玄牝是人身之元始真氣所藏之

地,是一切內丹修鍊的根基內丹的修鍊都從這裡開始。在此通過適當

的功法,使我們能夠控制元始真氣從伏藏之地開始運化,打開內修之

門,故又稱為「得竅」。第二層稱為「煉精化氣」、「鍊氣化神」,是運用元

始真氣,加以升降抽添循環往複,在體內不斷進行合聚、化煉,將分散

於人體各處之氣回聚為五行之氣故又叫做「煉己」。第三層為「五氣朝

元」,表示將通過煉己而聚合的五行之氣歸於本元,即將五行之陽、五行

之陰皆還于丹田,也就是五行之氣合聚為陰陽二氣,故又稱為「和合」。

第四層為「取坎填離」,到這一境界,則將伏藏在陰陽二氣之陰中的真陽

填補到陽中,以形成純陽,也可以說是以純陽煉化純陰而現出本色,無

陰而聚陽神,取得了可以回返太虛的基礎,故又叫做「得葯」。在卦象

言,這就是將坎卦中的陽爻抽出填入離卦中,這樣三爻皆成陽爻,成乾

卦,意味著人變成純陽之體。最上一層圓圈,稱「煉神還虛,復歸無極」,

指在得葯的基礎上,陽神內觀,進一步修鍊回返到無極,那便是虛無飄

渺、無有極限的神仙境地了。整個煉內丹的過程便告完成,脫出煉成的

聖胎,成為仙人,所以叫做脫胎成仙。此圖最下一層和最上一層,皆為

虛無,當中一段為有,表示從無到有,又返歸於無。虛無為萬有之根本,

故名之為無極圖。

至於太極圖,是從上往下讀的。為什麼同一圖可以從兩個相反的

方向來讀它呢?原來道家認為,從大道化生萬物的順序看,是從太極分

陰陽開始的,煉內丹則是逆著來路返回到大道,與道同體,自然長生不

死成神仙。所以順化逆化方向相反,但所沿著的路徑則是同一條。

周敦頤《太極圖說》曰:「無極而太極。太極動而生陽,動極而靜;靜

而生陰,靜極復動。一動一靜,互為其根。分陰分陽,兩儀立焉。陽變

陰合,而生水、火、木、金、土。五氣順布,四時行焉。五行,一陰陽也;陰

陽,一太極也;太極,本無極也。五行之生也,各一其性。無極之真,二

五之精,妙合而凝,乾道成男,坤道成女。二氣交感,化生萬物,萬物生

生而變化無窮焉。惟人也,得其秀而最靈。形既生矣,神發知矣,五性

感動而善惡分,萬事出矣。聖人定之以中正仁義而主靜,立人極焉。故

聖人與天地合其德,日月合其明,四時合其序,鬼神合其吉凶。君子修

之吉,小人悖之凶。故曰:立天之道,日陰與陽;立地之道,曰柔與剛;立

人之道,曰仁與義。又曰:原始反終,故知死生之說。大哉易也,斯其至

矣!」

周敦頤的《太極圖說》,以簡潔的語言解釋了太極圖,基本上是符合

道家思想的。雖然後半段比較多地講了儒家的仁義道德思想,此思想

道家也認同。前半段則相當明顯地表述了道家的哲理。

太極圖有另一種更加簡捷的畫法:就是用一個以曲線分隔的圓圈

表示,有如下圖。

在這一圖中,一條曲線將它分為兩半,形成一半白一半黑,白者表

示為陽,黑者表示為陰,白中又有一個黑點,黑中又有一個白點,表示陽

中有陰,陰中有陽。分開的兩半,酷似兩條魚,所以俗稱陰陽魚。此圖

是旋轉著的,表示陰盛則陽陽盛則陰的理論觀。還有一種說法,是說

此圖表示順時針旋轉成人,逆時針旋轉成仙的觀念。這一圖,與前面講

到的周敦頤太極圖有著密切的關聯,可以說是前者的更加簡明的表述

它深刻而又形象地說明了,世界上的一切都是由陰陽這兩種對立的形

勢相互聯結而形成的統一體。這一圖,過其圓心作任何一條直線將之

分成兩半,任何一半中都包含陰陽兩個因素絕不存在孤立的沒有內在

子盾的成分。有時人們又在圖外配以先天六十四卦,表示太極是一切

運動的發動者

魏孟康在《三統曆》中注:「太極元氣,含三為一。」歷代對此有不同

的解釋:(1)就太極圖言,「三」是指陰陽魚結構,即白、黑及白黑的分界

線;「一」是指太極圖的大圓圈,白、黑及分界線包含在同一圓圈之內。

(2)以黑白二色代表陰陽兩方、天地二部;黑白之分界線即劃分天地、陰

陽界限的人部。天、地、人為「三」;陰、陽、分界標準為「三」。太極圖

由正、反、合三者組成的整體,體現了老子「萬物負陰面抱陽,沖氣以為

和」的思想。白方中一點,為天之中點,表示陽中有陰;黑方中一白點

為地之極點,表示陰中有陽。(3)太極圖為相反相成的兩方屬性及其劃

界標準或活動場所三者的統一。其相反指水火對立,從空間言指水星、

火星,從時間言指夏熱與冬寒;其相成指木金生成,從空間言指木星和

金星,從時間言指春始、秋終。「土」即是水火和木金的劃界標準,也是

火水木金的活動場所。(4)「三」指三爻,「一」指一卦喻三爻組成一卦

天地人合為一體之意

《太極圖說》也就是《太極拳論》上說的「靜之則合」,即陰陽統一於

太極。太極拳修鍊的最終目的就是「太極」,即「混元太一」的狀態。太

極拳之所以得名「太極」,其根源就在於此

另有學者研究,太極拳的姿勢與道教科儀踏罡步斗極相似,其間

禹步」之術是我國古老的養生術,即為今天的八卦步、太極圓環步的雛

形。此步法曾被著名的武術家「鐵臂沱南俠」——韓其昌先生風趣地比

喻成為「貓步」,並笑稱「用此步不管白貓黑貓,抓著耗子就是好貓」。此

步又名粘連步,要求「運勁如抽絲,邁步如貓行」,講究在擒敵時「手不驚

風,腳不沾塵」。

太極拳理論直接來源於老莊道家思想,在重視生命、珍惜生命的同

時,並以尊道崇德為宗旨,故此太極拳形成了一系列養生、修身、煉己以

求長生的鍛煉功法。例如,在太極拳中講到的借力打力、四兩撥千

以柔克剛、以靜制動等理論都來源於老莊哲學中「柔弱勝剛強」的思想

理念,而《道德經)中「天下之至柔,馳鴉天下之至堅」和《莊子》中「示之

以虛,開之以利,後之以發,先之以至」的思想在《太極拳經》中則表現為

「極柔軟,然後極堅剛」,而老子「物壯則老」與「無過不及,隨曲就伸」的

思想則成為了太極拳推手的法則……如此等等,太極拳中所講究的中、

正、松、柔、圓、活、平、靈等特色風格,均能在道家義理中找到豐富而恰

當的解釋。

太極拳等同於道家內丹術的動功,與內丹術的修鍊過程有相同的

理論淵源,並有相似的內煉歌訣我認為太極拳內功與內丹術是同宗同

源,只是在目標和層次上略有分歧。作為拳術,防身之用的太極拳重在

技擊,用意調動丹田之力發於四肢,力顯在外;而內丹術作為修道成仙

的長生之術,重在結成金丹,脫胎換骨,迫求長生,體現在更高的內煉和

精神層面上。

太極拳是古代道家養生觀念的產物,雖然不是道家的專利,但是在

道家中一直繼承、發展和傳播著,幾乎是提及太極拳,就會聯想到道家,

道家與太極拳有著不可分割的「血緣”關係,因此可以說太極拳是道家

文化的形象體現

總之,武與道相結合,按上述理論去修鍊,持之以恆,就可以練到周

身上下、內外形氣一體,如太極之象,渾然一圓的境界。至此,太極丹道

即成矣。
