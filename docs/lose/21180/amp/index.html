秘籍一：節食減肥

案例介紹：這是一個最最最最最普遍，最最最最最多人試過的減肥辦法，這個減肥辦法的效果是相當之兇殘！！！許先生在2009年初，使用了這個辦法成功減肥60斤！大概3個多月的時間。當時的飲食方案是：

早飯生黃瓜+生番茄~午飯米飯+炒菜（燙蔬菜燙肉啥的）~晚飯吃空氣

就這樣，三個月的時間，瘦到我媽以為我在吸毒。

這個過程及其痛苦，除了每天挨餓之外，你還要面對來自各種損友的各種誘惑！

在三個月以後開始慢慢的吃晚飯…結果可以想得到，一個月讓你到170，畢竟每天多攝入了三分之一的熱量，不反彈才怪。

結果隨著自己慢慢的加餐加量，直到2012年又開始從事銷售工作，應酬那就是必不可少的，帶來的效果就是，反彈到我媽以為我水腫了！體重也就成功慢慢反彈到了200+！

案例分析：胖，其實就是脂肪多了。當人類還在遠古時期，食物不像現在那麼的多，也沒有現在那麼多保暖的服裝。所以脂肪在那個時候可是非常重要的。脂肪不僅可以讓我們的祖先沒那麼容易餓死，也沒那麼容易凍死。而脂肪形成的基本邏輯就是人類將攝入的多餘能量的一部分轉化為脂肪儲存在體內。

知道了以上兩個基本知識點，反過來想就可以知道，要讓脂肪變少有一個很有效的辦法，就是「減少攝入的能量」，讓身體的能量收支平衡處於赤字狀態，也就是消耗大於攝入。

但是這裡存在兩個問題，一個是被科學考證的一個是我自己領悟（瞎扯）的。

  1. 能量消耗分為基礎代謝與運動代謝。運動代謝很好理解，就是你使用你的身體去完成各種動作。而基礎代謝就是你只要活著，哪怕躺著一動不動，也要消耗的能量，而影響這個指標有一個很重要的參數，就是肌肉含量。問題就在於，節食減肥由於營養攝入的不足，讓你「掉肌肉」，所以你的基礎代謝就會變得很低！這也是身體的一種自我保護，也就導致你沒力氣（身體不讓你做更的運動）。
  2. 我嚴重懷疑人類的大腦里被編寫了什麼程序，一旦你節食減肥，大腦可能會給身體發出這樣的信號「警告！警告！這丫經常挨餓，被餓死的概率提高，請身體將攝入能量轉化為脂肪的比例提高！」也就是說，你只要吃夠，就有更多的能量被儲存起來。

舉個例子：許先生節食減肥前，基礎代謝2000卡路里，運動代謝800卡路里，每天攝入2800卡路里可以保持不胖不瘦（這裡先不計算食物本身的消化代謝）

許先生開始節食減肥，基礎代謝2000卡路里，運動代謝800卡路里，每天攝入2200卡路里，所以每天身體需要用儲存的脂肪提供600卡路里（這裡不考慮別的營養素的能量轉換，理解起來容易點）

許先生節食減肥三個月後，基礎代謝掉到1600，運動代謝被限制到600，每天如果攝入2200卡路里，達到能量收支平衡（也就是瘦不下去了）

許先生結束節食減肥，基礎代謝1600，運動代謝被限制到600，每天攝入2800卡路里，所以每天身體會將多餘的600卡路里轉化為脂肪。

結論：節食減肥後，由於基礎代謝變低，運動代謝受限，導致就算恢復到節食減肥前的能量攝入，可能會讓你更胖！而且，很遺憾的告訴大家，漲脂肪容易，你把掉的肌肉練回來試試？所以，一次次的節食減肥，慢慢的就變成了，傳說中的「易胖體質」！
