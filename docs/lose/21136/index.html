運動傷害關節？那不運動是不是能保護關節？看完你就知道了

2019-12-19 417條評論 1,465次閱讀 144人點贊

關節要省著用，這句話對於大家來說都很熟悉。關節的疾病有很多種，也是比較常見的骨科疾病。最常見的有骨性關節炎，骨質疏鬆，關節積水，風濕性關節炎，腱鞘炎等等。尤其是人一道中老年，關節就會發生退行性病變，病變到一定的程度，就會出現骨性關節炎。所以，就有人說了，人體的關節負荷越大，骨性關節炎的發病率就越高，那個關節活動的多，承受的重量越大，那個關節就容易患病。所以，關節要省著點用，才是延緩和保護關節的方法。

關節要省著用，是不是就不活動？

經過研究：健身跑的人關節炎發生率為3.5%，而選擇靜息生活方式、久坐不動人群的關節炎發生率卻為10.2%，競技體育的賽跑者的關節炎發生率13.3%。所以，可見不動比健身跑步更傷害膝蓋。這是因為人的膝蓋相鄰的骨面上都有一層軟骨，這層軟骨非常重要，能保護膝關節避免磨損，但是關節軟骨本身沒有血管供血，需要從關節液獲取所需的營養。關節活動會對關節軟骨造成擠壓和放鬆，通過這個過程軟骨才可以吸收進營養、代謝出廢物。

缺少運動的人，對關節軟骨的擠壓少，營養和代謝就會相應減少，這容易導致關節軟骨營養不良，使軟骨更容易出現損傷，一旦軟骨破損，膝關節喪失了一道保護屏障，就會出現疼痛等一系列問題。並且軟骨的損傷很難修復。

所以，是很多人都把這個省著用的意思理解錯了，省著用不是說少運動或者不運動，就算是省著用，還是要運動的。因為對於骨關節而言，適量的運動是必須的。我們人體中關節存在的意義，就是方便活動，如果不活動，關節不僅會失去意義，也會逐漸的退化。

運動的時候骨關節會產生適當的衝擊了，肌肉也會收縮和鬆弛，對於軟骨也是有意義的。但是在劇烈的衝擊超過其承受能力之後，軟骨就容易老化。如果不運動的話，軟骨也會老化。所以，對於軟骨來說，最好的辦法就是給與一定的擠壓，也就是進行適度的運動。

多大運動量算是適度運動？

只要你運動完後，膝蓋不疼，說明目前這個運動量是可以的。如果你運動完之後，膝蓋有點不舒服，但是在不用藥的情況下，兩小時以內可以緩解，說明這個運動量也還可以；

如果你運動完之後，膝蓋疼痛或者腫脹不適，超過兩小時不緩解，說明你運動超量了，明天再運動的話需要減量。

如果這種運動形式已經減到極低的運動量了還是疼2小時以上，那說明你的膝蓋現在沒法進行這種運動，需要改變運動形式。

如果日常生活的步行之後膝蓋都得疼2小時以上，並且3個月都沒有改善，吃藥了改善也不明顯，那說明你的關節炎很重了或者關節結構有問題，及時看醫生評估是否需要手術。
