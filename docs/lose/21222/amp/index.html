打乒乓球時如何正確揮小臂？看看便知

天天要聞
2019-12-18

揮小臂擊球的優點在於球的穩定性顯得好控制，可以有效地避免抬肘、漏球等現象，但這僅是權宜之計，並不是正確發力的最高境界。

　　有球友從快攻向弧圈技術過渡時，到達一定程度後再提升，就遇到了瓶頸而無法超越，其主要原因就是在練習快攻起步時，缺失了自發力擊球的這一課，他們只習慣於台內的上升期借力回球，而在高點期自發力擊球這一必要環節上，由於缺少好的陪練或沒有多球練習等條件而誤過了。
　　他們的定點練習中回合能打上幾百個不失手，但實戰中卻不會發力，甚至不知道如何去迎球，有球友說這是節奏掌握上不夠熟練，殊不知他們的自發力能力並沒有過關，不會發力，又如何能掌控節奏？

　　歸根結底，仍然是小臂正確使用的問題。當然，借力為主也是一種風格，但自發力卻是我們技術提升的必要條件，單從台內挑球這一單項技術來說，習慣於揮小臂擊球者必定沒有收小臂發力者的效果好。同時，正手的習慣也必然會帶到反手技術上去，這一點在橫板上的表現尤為突出，凡反手好於正手的橫板選手，其正手大多是揮小臂而不是收小臂。還要說明的是，反手技術的入門階段，揮小臂的發力控球穩定性好的優勢會得到明顯體視，尤其在防守或控制技術上，自發力的所佔比例較小，穩定性尤顯突出，但在反攻、反拉等自發力為主的強攻技術上，因為大、小臂的結合運用不好，旋轉在一定程度上會有、速度及力量卻遠遠不夠，就會顯得心有餘而力不足了。　　

綜上，我不否認揮小臂發力在乒乓技術體系中的作用，但卻反對將揮小臂的發力方式當做是唯一正確發力方式的觀點，我堅持認為，收小臂發力方式是正確發力的根本基礎，揮小臂發力只應是這個基礎上的一種再造形式，在整體技術中，應以打牢基礎為先，再圖提升和豐滿，這才是我們提升技術必由的正確之路。為此，這裡提出兩條建議供參考：
　　1.練習正手快攻階段中，在借力擊球達到一定熟練程度後，一定要把自發力為主的發力方式再加以深造夯實，二者缺一不可，不能偏廢；
　　2.正手攻技術稍有基礎後馬上進入反手攻練習並爭取早日進入正反手結合練習，使正反手的發力感覺得以均衡、同時固定和提升。如果正手熟練後再練反手，在反手的提升過程中會遇到想像不到的麻煩，使整體技術的提升進度大為減緩，甚至會出現偏一手的現象。
