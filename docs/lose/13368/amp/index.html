始祖鳥攀爬運動鞋防滑不悶腳，戶外出行或城市休閑穿著都比較適合

天天要聞
2019-12-11

本次試用拿到的是黑色款。雖然它的定位是攀爬運動鞋，但對於城市生活的人來說，畢竟沒有那麼多需要「攀爬」的地方，所以只能從通勤/徒步鞋的角度來評斷。當然包裝盒上的標籤也是徒步鞋 本人日常運動鞋穿著41碼. 此款鞋7.5碼（255）試用來說還是比較正的，合腳。但這款鞋較合適腳背略窄略瘦的，如果腳偏厚的話，建議可以適當大一些，給腳部留出更多的空間. 鞋面有一定的防水性，鞋底是圓形釘狀設計，對攀爬提供很好的阻力，硬度偏硬耐磨,鞋底內村鞋墊是皮革的，走起路來比較舒適，有一定跑鞋的「踩屎感」，日常徒步很舒服，外觀中規中矩，低調內斂型的，可能藍色款的會比較張揚些

記憶搭扣設計，快速調整系帶，加強鞋子的包裹性，也無需擔心走著走著鞋帶散了，細節處理也是很精良，幾乎看不到任何的溢膠，和阿迪耐克這些大眾運動鞋的做工對比來說是好太多了.

面料柔軟並且抗撕裂效果出眾，折到45度也能完整復原如.鞋跟可摺疊設計，對懶人很方便，可隨時變成拖鞋，又不用擔心鞋跟被踩壞.

出於攀爬鞋的性質，鞋前掌的密集型防滑腳釘和後掌減震設計絕對能勝任幾乎所有場景. 既考慮到了登山/徒步的功能性需要，也滿足日常的穿著環境

鞋面的防水性拿自來水試過了，類似於衝鋒衣的那種效果，滴水不沾鞋. 不知道是不是GTX面料，應該是GTX吧.鞋頭是一體式防滑橡膠設計，硬度夠，也能抗外力保護腳尖.

最近夏天天氣太熱，上海也沒有什麼山可以去體驗的，就先用做徒步鞋來用. 就是平時日常徒步和上下樓梯的感覺來說，鞋子輕盈，不磨腳，沒有累贅感，包裹感雖然沒有專業跑鞋好，但就徒步來說已經很好了，稍微有跑鞋的「踩屎」感. 鞋面的透氣性非常好。親測大夏天的徒步1-2小時也絲毫感覺不到悶熱. 今後等天氣涼快些再去爬個山體驗一下登山的實際感受.

鞋子總體顏值中規中矩，鞋頭的不規則透氣孔和鞋面的單手快速系帶系統設計也很人性化. 鞋子內部的鞋墊是皮革的，透氣又貼膚.穿著和舒適. 鞋頭和鞋後跟處都是一體式防滑橡膠設計，重量300g不到，既輕盈又防滑,能有效的保護整個腳掌的安全. 整體而言，彰顯了始祖鳥的做工精湛和人性化.另外不得不提一下摺疊式鞋跟的設計實在是太方便了.對於我這種懶人而言，回家不想換鞋了，直接摺疊後跟就可以當拖鞋來用了

優點：鞋子包裹性強，防滑不悶腳。款式百搭，鞋面透氣防水性良好，戶外出行或城市休閑穿著都比較適合.

缺點：鞋子總的來說是非常好的，做工精湛，材質精良. 如果價格稍微親民一些的話，就是完美
