羽毛球殺球的重和快到底是怎麼回事？

2019-12-24 523條評論 1,866次閱讀 699人點贊

羽毛球殺球，每個接殺的球友都體會到所謂的殺球很重的感覺，至於這個快字，那就不用提了，都有專門的測速儀器可以測量，據說最快的是傅海峰，可以達到305km/h？可能我記得有點出入。

快，這個很好理解，至於這個重，就讓我有點摸不到頭腦了。羽毛球脫離拍面之後，它本身只受到空氣阻力和重力，球的速度只和它的初速度有關，應該說，初速度越大，它的平均速度就越快，相應這個球應該就讓接殺的人感覺越重。快 = 重？但實際卻是：有的球快&重，有的球快卻不重。這是為什麼呢？
和aifaith、李妙言等人討論，我逐漸形成這麼一個觀點：羽毛球不僅有直線的飛行，其本身還有自轉，也許，這個重，就體現在羽毛球自轉的速度快上？？就如同高速旋轉的籃球可以將碰到它的物體轉飛一樣，高速旋轉的羽毛球就可以給接殺的球友以「重」的感覺，你要付出額外的力量來克服這個旋轉的勁道。
這麼一來，有些一直困擾我的問題就可以得到解決了。重殺，掄動大臂殺球，扣殺的動作充分，力量大，拍面和球接觸的時間相對較長，球獲得的動能多，一部分轉化為球向前的速度，一部分轉化為球的自轉，這樣的球是 重&快；點殺，靠手腕發力，力量集中，拍面和球的接觸時間短，球的動能主要體現在球向前的速度，球的自轉較慢，所以殺球是快卻不重。
同時，根據我多年燒拍的感覺，不同的拍子，殺球的重和快也是不同的。以兩個極端來舉例：老7，個人感覺拍面對球的包裹比較好，可以完全吸收扣殺者的力量，殺出快&重的球；9KX，拍面基本是鏡面反射，球速快，但不利於扣殺人的充分發力，殺球並不重。
個人體會，還請各位多加指點。
PS：
羽毛球脫離拍面時候的動量mv應該等於拍面的衝量Ft，F為拍面的作用力，也就是我們揮拍傳導到拍面的力量，t為拍面對球托的作用時間。忽略一些發力方式、滑板等等複雜因素，殺球應該就和力量和作用時間相關。那麼，重殺F大，t也大（揮拍動作完整，球拍有個跟隨的過 程），羽毛球所獲得的動量肯定大，為什麼球速有時候並不比點殺這種F小，t也小的殺球方式快呢？
個人認為，這個因素要考慮羽毛球本身的構造，球托的後面是16根羽毛，其空氣阻力極大，在重殺的時候，羽毛球獲得的初始動量很大，球剛
剛脫離球拍的一剎那速度非常大，其所面對的空氣阻力也很大（記得空氣阻力是和速度的3次方成正比？可能有點出入）。如此之大的空氣阻力會造成羽毛球的強烈旋轉（就和大風吹過，風車的高速旋轉一樣，羽毛球的構造，和風車有點類似），同時也會造成羽毛球的極快減速，以至於重殺的球速並不比點殺要快，但其自轉的速度絕對比點殺時候的要快得多。由此，造成了重殺的「重」的感覺。
