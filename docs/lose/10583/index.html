一招變美 韓國女星減肥法！IU、秀智和朴信惠瘦身菜單

2019-12-09 572條評論 1,894次閱讀 367人點贊

▲韓國女星身材一個比一個好，幾乎人人都有自己一套瘦身秘訣和減肥菜單。（圖／翻攝IG@ssinz7、@dlwlrma、@skuukzky，以下同）

張毓容／台北報導

韓國演藝圈競爭激烈，每年光是樂壇出道新人、團體就多達幾十組，除了本身專業能力、才華要夠，對於外貌身材的要求標準更是嚴苛，也因此韓國藝人不分男女對於身材空都不遺餘力，甚至可說幾乎人人都有自己的一套瘦身秘訣，像是「國民初戀」秀智、「滿月社長」IU（李知恩）和「韓劇女神」朴信惠這3位女神級女星，都曾有過肉肉臉的嬰兒肥時期，靠著私房減肥菜單成功瘦出小V臉和纖細窈窕身材，來看看她們的瘦身餐都怎麼吃。

▲「國民初戀」秀智的減肥菜單以番薯為主，且晚上6點後就不再吃東西。

裴秀智（Suzy）

16歲就以miss A女團成員出道的「國民初戀」秀智Suzy，一開始還有著可愛的baby fat，身形也較為圓潤，隨著接拍的戲劇越來越多，為了上鏡更好看她也開始努力瘦身，瘦下來被許多人大讚女大18變，更多了女神氣息。

她曾在節目上公開自己的減肥菜單：早餐吃100g番薯＋50g雞胸肉和半杯脫脂牛奶，午餐則是會以一小碗糙米飯搭配蔬菜沙拉，晚餐則是再吃回2條番薯，並且晚上6點後就不再進食。

▲IU靠著自創的「蘋果減肥法」搭配運動，曾一周瘦下5公斤。

IU（李知恩）

因韓劇《德魯納酒店》的火紅，人氣更創高峰的IU（李知恩），劇中各種華麗復古的造型讓她展現出不同以往的成熟小女人魅力，而現今維持在42公斤纖瘦體態的她，剛出道時也曾有過嬰兒肥的肉肉時期，後來因為要拍戲，為了在短時間內瘦下來，靠著自創的「蘋果減肥法」，進行快速瘦身計畫。

IU曾在節目上透露她的一周減肥菜單就是早餐吃一顆蘋果，午餐是兩條番薯，晚餐則是一杯蛋白質飲品，將一天攝取的總熱量控制在大約300卡，同時搭配上健身房或做伏地挺身、上下樓梯1小時，一周內就瘦下5公斤，變成巴掌小臉。

▲韓劇女神朴信惠的黃瓜瘦身餐非常出名，瘦下來後更以健康方式維持好身材。

朴信惠：黃瓜餐搭配紅豆水

童星出身的朴信惠，從小就是美人胚子，但易胖體質可是讓她吃了不少苦頭，像是以《原來是美男》在亞洲爆紅後，就常被揶揄下半身肥胖，甚至還被網友笑說臉比男主角張根碩還大，後來以出名的「黃瓜減肥餐」順利瘦下來。

這套減肥菜單主要以黃瓜和白菜為主，並搭配少許澱粉補充體力：早餐一根黃瓜加一杯低脂牛奶，午餐是白菜加半碗白飯，晚餐則是白菜加一根黃瓜，如果肚子餓了，就再吃一點白菜或小黃瓜充饑，並且搭配紅豆水、南瓜水幫助消水腫。她每天這樣吃，加上一周騎腳踏車運動2～3次，一個月瘦下10公斤，狠甩肉臉和粗腿。成功瘦下來後，她也慢慢恢復正常飲食，但會注意選擇營養又不易胖的食物，且更積極運動，用健康的方式持續維持好身材。
