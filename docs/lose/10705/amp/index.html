這些“怪癖”，只有愛健身的小夥伴才懂

天天要聞
2019-12-04

健身，通常是為了讓自己變得更好。

而無論是健身的習慣、觀念，還是健身的相關知識，或許都會一定程度上影響自己的生活習慣。

那麼，今天我們就盤點一下那些健身「引發」的特殊「癖好」。

喜歡照鏡子擺造型

反正只要是反光的物體，看到了就想欣賞欣賞自己的肌肉、身材，有時還會擺一些自己感覺很帥的造型哈哈。

都說愛健身的人自戀，來之不易的肌肉，當然開心要多「欣賞」啦。

夏天看到鏡子更是不能放過！

有時感覺自己的身材很帥，但有時又感覺自己的肌肉很弱~

平常穿衣

雖然不用緊身、但首選能顯身材（尤其是胸肌）的版型，大多都是修身。

經常感受肌肉發力

平常做體力勞動時都會感受一下相應肌肉發力，趁機練一練~

例如：搬東西時會感受二頭肌發力；逛街掂東西時，如果不是很重就會練練「單手提拉」；甚至於翻炒菜時 都會感受肩部肌肉發力……

摸自己的肌肉

也不知道為什麼，自己沒事時也喜歡摸一下自己的肌肉

有時就是想摸摸肌肉的感覺，尤其是胸肌與腹肌，順手就來~

有時是為了感受一下自己的肌肉有沒有變大、進步，腹肌有沒有更清晰。

有時感覺很不錯，而有時又感覺自己摸起來還是不夠理想~

享受肌肉酸痛的感覺

雖然我們都知道，肌肉酸痛並不是肌肉生長的直接標誌。

但對訓練後幾天的肌肉酸痛總有一種滿足感，覺得自己又要變強啦。

雖然對於健身愛好者而言，這種感覺確實能給心靈上的滿足，但我們還是要正確看待肌肉酸痛這件事

關於肌肉酸痛：「肌肉酸痛」並不是訓練到位的標誌！（點擊閱讀）

有意調整姿勢至中立位

自從了解到硬拉時要腰背挺直，以免傷腰，搬東西的話也都會刻意調整一下姿態，以減少自身的損傷，順帶練一下。

甚至於在蹲廁時，都會刻意調整一下「膝蓋正對腳尖」。

買零食看營養成分

只要帶包裝的食物，都會看一下背後的營養成分。

首先看蛋白質含量，只有蛋白質含量差不多，再看熱量的高低，然後決定是否購買。

如果沒有包裝，也可能會查詢一下相關食物營養成分，大概了解自己吃了多少營養與能量。

平常盡量不會吃很油、營養又不豐富的食物。

觀察他人身材

除了經常照自己的身材，也會不自主的觀察他人的身材如何、肌肉如何。

如果能看到有訓練痕迹的，可謂是眼前一亮。

對於女生，除了看顏值，那就是看臀部啦，反正練身材小夥伴都懂。

或許健身之後，自己對身材的審美也或多或少會有一定改變。

訓練離不開音樂

無訓練，不音樂，音樂就是動力！

瞬間力量加成，提高運動表現。

有時就喜歡聽著激情的音樂練，很有感覺，

但有時卻喜歡聽著抒情的音樂訓練，感覺很舒服。

哈哈，感覺健身也不單單是訓練的那段時間，對生活的點滴也會有潛移默化的影響。

這些「怪癖」，有沒有說中愛健身的你呢？
