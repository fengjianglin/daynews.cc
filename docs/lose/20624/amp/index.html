經常食用堅果可降低健康人群肥胖的風險

天天要聞
2019-12-17

「開心果作為眾多堅果中的優秀代表，它的卡路里很低，一盎司美國開心果的鉀含量相當於半個大香蕉鉀的含量，在運動後食用有助於及時補充這種重要的礦物質，調節體液平衡；更令人驚喜的是，其中的葉黃素、β-胡蘿蔔素和γ-生育酚等天然抗氧化劑，將極大促進肌肉恢復，成為訓練過程中的『秘密武器』」……12月15日，由美國開心果聯合《WOMEN’S HEALTH健康女性》，在成都舉辦了一場SWEET SWEAT 蜜果瘦訓班。此次訓練班並非一般意義上的健身課，而是在汗水訓練中加入健康飲食分享，帶領女性科學健身，重視均衡營養膳食，在這個冬天達成訓練目標，收穫更美好的體型。

健身教練在分享中告訴現場的女性們，「俗話說『三分練，七分吃』。想要早日實現健身目標，在積極鍛煉的同時，還應注重飲食，選擇正確的食物。」她認為，儘管大家每天都會接觸到「吃」這個方面，卻最容易出現差錯。比如，大多數女性都對堅果望而卻步，認為高油高熱量的堅果會不利於打造健康體型，這是大大的誤區。最新研究表明，經常食用堅果可以降低健康人群肥胖的風險。食用堅果的成年人與不吃堅果的成年人相比，體重、體重指數和腰圍都會更小。究其原因，恰好是其能夠增加飽腹感、飽腹信號和降低代謝，從而幫助控制體型。

「在冬天來一場酣暢淋漓的健身體驗也是為了不錯的！」「我的目標是，明年瘦10斤！」在共計一小時的兩節健身訓練課中，女性健身愛好者們不斷提升身體的協調、拉伸和力量水平等，收穫屬於她們的運動派對。訓練之餘，更有健康飲食主題分享，解答極為重要的瘦身營養困惑，提供多種飲食建議。主辦方還帶來了美國開心果和以它為原料的創意美味，作為運動補給邀請大家共同品嘗
