業餘羽毛球雙打前場隊員的無縫“發-接發”環節

天天要聞
2019-12-24

無縫「發-接發」就是：雙打時，前場隊員在發球-接發球的環節防範緊密，能直接得分或為搭檔創造天衣無縫的連貫進攻機會。

業餘雙打多是女選手在前場，所以這個光榮而艱巨的使命通常要由女選手完成。

一、 發球：發球從落點選擇上看有發前場和偷後場之分。偷後場先不提，前場球有左中右三個點可以選擇。我的體會：發到T字線附近的球最多；其次是邊線；然後是中間（追身球）；都可以直接得分。直接得分的關鍵是：發出的球要貼著網帶過去，一過網就下落。

*首先，對手通常是站在前場中間位置等待接發，所以發到T字線的球要貼近T字線的邊角，第一容易造成誤判，第二容易起高球，第三反手位時容易直接得分；

*其次，發到邊線的球。有了高質量的T字線發球做基礎，對手防範邊線時就會分散注意力，這兩個落點混合搭配使用就能迷惑對手，讓對手顧左顧不了右；

*最後是追身球。追身球可以出其不意地使用，得分概率非常大。業餘羽毛球選手常常會忽略第一拍，其實，此時是進攻的絕佳時機。發球，是羽毛球對抗里唯一不受干擾的進攻時機！

二、 接發球：如果遇到發球時「很講究」的對手，接發球就變得有趣了。接發球也是攻防轉換或直接得分的好時機。

*首先，對手發到前場的球，是自己正手位的話（右手持拍），可以直接撥向左邊網柱附近；是自己反手位的話，可以撥向右邊網柱附近。落點的控制對接球選手的要求比較高，要盡量接近網柱而不出界。

*其次，接發球時腳步要移動！

尤其是要搶上一步！不管對手發向前場那個點的球，其實都可以在對方擊球後迅速跨步向前，趁球剛剛過網的那個時機推撲過網：推到底線或中腰均可。這個搶上一步可以迴避追身球和各種刁蠻落點的發球。

網前隊員的推撲還有個最大的好處：即便不能直接得分，也可以讓對手不得不起高球。對方一起高球，自己的搭檔就舒服了，呵呵～或扣殺、或變線、或輕打……諸事隨意。

*最後，有了前面兩種接發球做基礎，對手就會「被迫偷後場」。
