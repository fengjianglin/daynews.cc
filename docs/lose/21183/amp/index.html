關注慧跑，助您輕如羽、跑無傷

《跑者世界》（Runner』s World）雜誌於1966年由鮑勃安德森創辦，是一本關於純粹跑步的專業雜誌；

經過50餘年的發展，已經發展成為全球最具知名度的跑步內容出品方，在很多國家都用當地語言發行，當然也包括中國。

《跑者世界》每年會推出年度推薦裝備，這其中最有名的就是年度跑鞋，《跑者世界》每年審查數以百計的跑鞋，其所推薦的跑鞋基本來自於其編輯或者資深跑者的研究和體驗，具有一定的公信力。

我們快來看看《跑者世界》所推薦的2019年度30款跑鞋，這些跑鞋涵蓋了從路跑到越野，從訓練用穩定型、緩震型跑鞋到馬拉松競速跑鞋。

有些品牌的跑鞋跑者在國內可以買到，有些則尚未進入中國市場，只能海淘，你穿過其中幾雙鞋呢？

為完成2019年年度30款跑鞋，《跑者世界》雜誌一共選取了78雙鞋在實驗室進行了測試，共計300名跑者完成了超過100英里的實際測試。

主要測試了跑鞋的舒適性、穩定性、平順性，最終從79雙鞋中選取了其中30雙值得推薦的跑鞋。
