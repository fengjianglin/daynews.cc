這個冬季因你而溫暖——李寧雲五代SHIELD跑鞋

天天要聞
3 天 ago

冬季戶外跑步是一件挑戰意志力的事，特別對於晨跑族來說，更需要有堅強的意志力。用強大的心理對抗寒冷的同時，合理的跑步裝備也是必不可少的。每到這個季節，李寧都會推出幾款適合冬季使用的跑鞋，比如最近出品的李寧雲五代SHIELD跑鞋，這是一款防風防潑水的冬季防護跑鞋。

外觀細節

從喜慶紅的包裝盒上獲悉這款貨號為ARHP143-6的李寧雲五代SHIELD跑鞋隸屬於李寧雲系列，李寧目前很經典的一個系列，近6年的歷史。從開箱的一剎那，引起我注意的倒不是這款焦綠色/松香綠/亮茄紅的跑鞋，而是貼在盒子上的防霉貼片，這是我第一次在跑鞋包裝中看見以防霉貼片取代乾燥劑。

焦綠色/松香綠/亮茄紅的色彩搭配還是比較簡潔歡快。外形方面，李寧雲五代SHIELD跑鞋顯得較為厚實。

跑鞋的鞋重驗證了李寧雲五代SHIELD跑鞋的真材實料。美碼9碼，單只鞋重350g不到，很重磅的跑鞋，考慮到常規慢跑鞋通常在300g左右，而作為冬季防護跑鞋的雲五代SHIELD，350g的鞋重符合預期。

李寧雲五代SHIELD跑鞋的鞋面內涵豐富。前後緻密織物鞋面通過熱覆膜工藝進行無縫處理。前掌的高周波壓印紋路造型，立體感十足。整個鞋面採用李寧WATER SHELL防潑水技術，鞋面對水有著顯著的隔離作用，當水滴落在鞋面上會自然形成水珠，輕微抖動即能甩離鞋面，這樣可以有效防止打濕鞋面並保持鞋內的乾爽。熱貼合條紋大李寧LOGO標彰顯品牌屬性。

雲五代SHIELD跑鞋後跟除了內置半杯狀硬質TPU外，還在外層設置了”丄”型TPU，外層TPU具有一定的厚度且富有彈性。另外，後跟部分的綠色啫喱大底幾乎包裹了整個後跟，質地堅韌不易變形，這種構造非常新穎和罕見。內外雙加持TPU結構結合堅韌大底包圍設計，更能穩定地支撐雙足，減少跑步過程中足部過度外翻，降低受傷的風險。

雲五代SHIELD跑鞋的鞋舌和後幫的填充非常飽滿，不但有很好的貼合，還具有保暖作用。後跟的提鞋帶便於跑鞋的攜帶和穿脫。

鞋眼使用加厚麂皮布材質，經久耐用。”WATER SHELL”、”SPLASH-RESISTANT”字樣表明這是一款放潑水跑鞋。鞋舌增加了防偏移的定位設計。銀色LOGO標具有反光功能。

雲五代SHIELD跑鞋使用了個人認為李寧跑鞋中腳感最舒適的”Turbo”鞋墊。這款鞋墊柔軟且富有彈性，和足底的契合度非常高，腳感十分舒適，5毫米厚度也確保了緩衝感的真實存在。

中底採用的”LINING CLOUD”材料是目前李寧看家的中底材料，具有良好的緩衝、減震和回彈性能。橫向波紋能夠有效化解中底在長期承壓後產生壓縮細紋而影響美觀度。

後跟易磨損區域採用質地堅韌的啫喱大底之外，中前掌大底採用縱向環繞式設計，構成了S形步態引導線結構，橫向窄邊緊密紋路設計具有更好的抓地力。

實測體驗

李寧雲中底自從2014年面世以來，5代雲系列跑鞋本人都進行了完整而全面的體驗，所以說對整個雲系列跑鞋非常了解。雲系列跑鞋一直以緩震出色，回彈充分而贏得稱道。特別是雲二代開始，雙密度雲中底的使用，讓雲中底的性能更顯均衡。而雲三代實現了單密度雲中底質的飛越，讓整個雲中底趨於完美。從實驗室數據分析，李寧雲材料的性能完全不輸一些國際品牌所使用的中底材料。李寧雲五代SHIELD跑鞋作為雲系列跑鞋中的新貴，繼承了雲跑鞋緩震出色、回彈充分的特點。

李寧雲五代SHIELD跑鞋雲中底材料十分充實，當跑步受到壓力時，衝擊力化解非常及時，並能迅速的轉換到回彈通道。整個中底給人的感覺不是那種綿軟無力、鬆鬆垮垮的減震，而是緊實富有韌性的緩衝，是一種具有反作用力的減震方式。雲五代SHIELD跑鞋緩震的重點在後跟，用前掌跑法去試探前掌性能時，則會發現相對後跟，前掌的緩震回彈是偏弱的。

相比前幾代雲跑鞋，李寧雲五代SHIELD跑鞋最大的提升在於穩定性。內外雙加持TPU結構結合堅韌大底包圍設計，這種為落地穩定性而進行如此豪華的配置實屬罕見，反應到跑步中，會發現後跟落地後左右搖晃的幅度會明顯縮小，而且支撐感比較強烈。在進行稍長距離跑步時，支撐感、穩定性的持續能力都能得到有效的保持，這種支撐和穩定可以讓人專註於跑步。

李寧雲五代SHIELD跑鞋主打的是防潑水冬季用鞋。鞋面進行過防水塗層處理，鞋面和填充物厚實飽滿，不但防潑水，而且保暖。12月，魔都清晨6點不到還是比較陰冷潮濕，四次晨跑的氣溫基本在5℃～9℃，如果穿著傳統跑鞋，跑步時冷風會竄入跑鞋內，足部也會感到涼意，但是這款跑鞋的鞋面不但對晨露有良好的抵禦能力，而且能將寒風完全拒之鞋外，保暖能力超強。跑後襪子的潮濕程度也驗證了這款跑鞋的保暖能力，當然，魚與熊掌不可兼得，保暖性好也意味著透氣性的不足。

綜合評價

李寧雲五代SHIELD跑鞋在防潑水、保暖性、緩震性和穩定性方面是值得稱道的。如果純粹從跑步角度出發，這款跑鞋由於受限於鞋重和體量，更適合進行慢跑運動。如果從使用環境角度出發，那麼更適合北方的天寒地凍。

天氣漸冷，真跑者從來不會畏懼室外的嚴寒，一雙李寧雲五代SHIELD跑鞋或許是這個冬季跑步路上最溫暖的夥伴！
