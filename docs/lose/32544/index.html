《愛的迫降》孫藝真美貌升級的秘密健身方法、漂亮姐姐這樣當

2019-12-24 507條評論 1,520次閱讀 565人點贊

​由韓國高人氣、高顏值、好演技演員玄彬、孫藝真主演的韓劇《愛的迫降》，首播收視就開紅盤，劇情、男女主角的服裝造型都成為網友討論的焦點，而孫藝真的美貌又再度驚艷眾人。孫藝真外貌清新甜美、氣質高貴優雅，一直以來都是韓國演藝圈中的女神代表，即使已經37歲依舊美到發光。

孫藝真曾公開分享，自己的外表和身材都是努力而來的。孫藝真表示，隨著年紀增長，新陳代謝變慢，單靠飲食控管是不夠的，所以她經常健身運動，以提高代謝。只要時間允許孫藝真每天都會花40分鐘至一小時的時間來運動，但如果行程比較忙，每周至少會運動三天。

TRX健身

孫藝真曾在韓國綜藝節目《家師父一體》中分享自己非常熱愛運動，自從開始做TRX到現在已經超過十年，甚至還有TRX講師證照，以下公開她的TRX健身方法。

睡前伸展運動

除了TRX外，孫藝真睡前也會花一點時間來做伸展運動，像是簡單的前肢體前彎及背部肌肉伸展等，都可以舒緩肌肉的壓力、雕塑身材線條。

加入精油泡澡

孫藝真分享自己的腿很容易水腫，因此會定期泡半身浴加速身體的新陳代謝。另外，在泡澡時加入精油，還能幫助舒緩壓力。

同場加映:保養秘訣

熱毛巾敷臉

已經37歲的孫藝珍皮膚仍宛若少女！女神保養最重視「毛孔管理」，在洗完臉後，孫藝珍會用熱毛巾敷臉一分鐘，然後按摩臉部穴道，針對顴骨、眉骨等部位按摩，可以促進臉部肌膚的循環，提升後續保養品的吸收力。

檸檬水洗臉

孫藝真平時有檸檬水洗臉的習慣，只要將檸檬片切放進冰水10分鐘，再用泡過檸檬片的水來洗臉，就可以縮小毛孔。因為冰檸檬水有安定肌膚、收縮毛孔的功效，而且還富含豐富的維他命C，可以補充肌膚營養。

冰鎮化妝棉

將化妝棉沾滿化妝水放進夾鏈袋後冰起來，一小時後再拿出來擦拭肌膚，可以達到去角質與粉刺的效果！簡單來說，就是利用熱漲冷縮的原理，打造零毛孔肌膚。這招除了孫藝真外，擁有世界第一美稱號的韓國演員Nana(林珍兒)也會用。

初學者瑜珈1：貓式

開肩、全身伸展動作

這個貓式不僅可以開肩、也很適合在最後的時候做，可以放鬆全身的肌肉，讓肌肉條更好看！初學者肢體比較僵硬的話，上半身也可以不用全趴下去、可以在胸部位置墊一塊瑜珈磚或是枕頭增加高度，做起來比較輕鬆、但還是可以達到效果！

初學者瑜珈2：上犬式

開肩、背部伸展動作

這個上犬式也是一個可以開肩、伸展背部的動作，對於初學者來說可能會覺得腰部不太舒服、一開始建議可以以手肘撐地、上半身半抬的方式即可。

初學者瑜珈3：駱駝式

大腿、腹肌伸展動作

如果要練腹肌很常做到的駱駝式，對初學者來說要這樣雙手碰到腳踝其實很有難度！建議可以以瑜珈磚或是枕頭輔助，讓雙手可以有支撐點，之後在循序漸進慢慢往下到腳踝，比一開始就硬練、對身體及肌肉造成的傷害更小。

初學者瑜珈4：背後扣手式

開肩、肩膀伸展

初學者瑜珈5：坐立前屈式

腿部、全身伸展動作

這個坐立前屈式除了可以伸展整個大小線條之外、也可以伸展到全身，如果無法整個趴下去的話、也可以用毛巾或是彈力帶輔助、儘力將身體往腿部方向帶即可，一樣可以伸展到腿部肌肉喔！

這幾款其實都是針對常見的瑜珈姿勢的簡易版本，對於初學者來說不僅更好上手、也可以達到原本的伸展、舒緩肌肉的功效！
