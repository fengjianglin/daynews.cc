二十四 （足球）肌肉系統和運動

肌肉力量引發運動，並控制運動。運動是通過身體骨骼各部分不同的活動而實現的，運動實際上是不同身體部分同一時間出現的基本動作的組合。若想了解軀體運動的能力，需要學習肌肉、骨骼和關節的結構與機能。關節可能的運動方式，關節周圍韌帶和肌腱的長度與彈性以及肌肉運動的形式，都會影響動作的幅度和速度。本部分內容主要是幫助教練員了解肌肉的運動、肌肉運動的形式以及與運動有關的基本的力學概念。

肌肉的機能

肌肉和骨骼在神經系統的控制下產生運動。肌肉是由一束束的肌纖維組成的，肌肉之間通過結締組織相連，在肌纖維束之間有神經和血管。毛細血管給肌纖維提供營養物質和氧氣，並運走代謝廢物。

肌肉受神經的支配，來自大腦的神經衝動，通過神經纖維傳遞至脊髓，再傳遞至收縮的肌肉。

在每塊肌肉的兩端，各有一根或多根肌腱，把肌肉固定在骨骼上。當肌肉收縮時，兩肌腱向心移動，肌腱所附著的骨骼圍繞關節旋轉，於是產生運動，例如，股四頭肌起於骨盆，經由股骨和膝關節止於脛骨。當踢球時，股四頭肌收縮，大腿前擺，於是完成踢這個動作。

快肌纖維與慢肌纖維

肌肉由不同類型的肌纖維—慢肌纖維（紅肌）和快肌纖維（白肌）組成。慢肌纖維產生張力的速度慢，主要通過有氧氧化形式供能，具有較高的耐力。快肌纖維產生張力的速度快，但耐力差，易疲勞。

長跑運動員耐力較好，肌肉中通常慢肌纖維多，而短跑運動員需要在最短的時間內產生最大的能量輸出，通常腿部肌肉中快肌纖維百分比較大。

有氣氧化酶

通過系統的耐力訓練可以提高快肌纖維的耐力水平。肌肉中有些酶決定肌肉的有氧工作能力，還有一些酶則調節脂肪的分解，這些酶統稱為有氧氧化酶。研究表明，一名受過良好訓練的足球運動員肌肉中有氧氧化酶活性比未訓練者高，而且毛細血管豐富。如果肌肉長期得不到適宜的運動刺激，肌肉內的有氧氧化酶活性下降，因此，足球運動員必須維持適宜的運動水平以保持有氧氧化酶較高的活性。

訓練

足球運動員若想在比賽中正常發揮自己的水平，除了防止損傷之外，還必須系統地進行肌肉力量和耐力兩方面的訓練。通過正確的訓練使肌肉產生一系列的適應，如肌纖維體積增大、毛細血管數目增多、某些有氧氧化酶活性提高等，這些都有助於運動能力的提高。

除了力量和耐力之外，足球運動員還需要良好的協調性和平衡能力，這對有效地完成技術動作有好處。

運動前準備

為了使足球運動員在運動中發揮適宜水平，運動前進行準備活動和伸展練習是非常重要的，這可以最大程度地使動作放鬆，減少肌肉緊張，而且可以預防運動損傷。

例如，當準備活動和伸展練習做得不充分時，大腿後群肌肉，如股二頭肌、半腱肌、半膜肌等受傷的機會增多。這是由於股四頭肌體積大，收縮力量也大。當股四頭肌收縮時，大腿後群肌肉也必須同時加強收縮力量，但是很多人都忽視這塊肌肉的訓練，於是發生損傷。

肌肉放鬆

劇烈運動後進行肌肉放鬆也是很重要的。因為運動中產生的乳酸會造成肌肉酸疼，影響肌肉運動。運動後恢復性慢跑和伸展練習有助於清除代謝廢物，若同時還進行促進肌肉再生的運動和按摩將會更有幫助。肌肉受傷後不運動會降低肌肉力量，損傷恢復後必須先提高肌肉力量。如果肌肉力量沒有恢復，運動員很容易再次受傷。

青年選手的訓練

運動員學習技術的最佳年齡是8~12歲，這時，身體的協調性和節奏感最強，反覆學習的動作差不多可以無意識完成。一個動作反覆幾次後，動作的形式便貯存在大腦里，需要時可重新記取，如運動技巧、傳球、帶球跑等。開始做這些動作時，運動員需要集中精力，但是隨著自信心的增強，技術不斷鞏固，動作可以無意識完成。對12歲以下的小選手，訓練重點應著重在跑動中和掌握技術時的步法和協調性，以及正確技術的運用上。這個年齡組的孩子靈敏和柔韌素質發展得最快，相應地還應進行靈敏和柔韌性的訓練。

12~16歲的選手應繼續進行上述方面的訓練，只有經過長期的訓練，他們才能保持身體與球之間的節奏感和良好的平衡感。他們還應繼續進行速度與協調性的訓練，但是力量訓練的負荷以不超過他們自己的體重為宜。

對於16歲以上的運動員，可逐步提高在對抗訓練中的力量。這個階段應主要進行力量訓練，這個階段也是進行速度耐力訓練的適宜時間。然而，必須強調的是，在進行上述訓練時應該逐級遞增運動負荷，避免出現過度訓練。
