練蜜桃臀不一定非要深蹲！簡易版，隨時隨地練起來

3 天 ago

運動做得多不如做得精，想練出翹臀，選擇對臀部肌肉有最大刺激的運動、CP值更高！健身教練分享，在我們的運動項目中加入橋式、臀推，練屁股效果事半功倍，不只改善不佳體態，臀線被提高、後腿看起來還會更修長唷～一起往下看介紹吧！

年輕女運動健身上衣和短褲持有測量與磁帶的腹部，她微笑和孤立的白色背景上。4 k.慢動作

想要練出渾圓的蜜桃翹臀，練對部位很重要！對於瘦身練屁股，我們總會直接聯想到「深蹲」，深蹲確實有效，但不如「橋式」來的直接！

橋式對於臀大肌的刺激最多，甚至超越深蹲有更好的練臀效果，因此練習可以明顯提高臀位、變翹屁股，也能改善假跨寬與骨盆前傾的問題。編這邊再補充說明，橋式與臀推的差異：前者動作是人仰卧在地面或墊子上執行；而後者的臀推則要肩膀靠在椅子上。

簡單來說，新手或者熱身時我們可以選擇做橋式，但想要獲得最好的鍛煉效果就要加入臀推（加負重）的訓練。接下來～編為大家帶來三款針對臀部非常有效的運動，讓我們循序漸進、一起練出蜜桃臀吧！

練臀運動1：橋式

首先仰躺在地，並屈膝、將雙腳掌踏穩在地板上，接著，透過屁股與核心出力將身體水平向上抬起。

我們可以參考這個影片來做正確的臀橋運動。幾個特別注意的小技巧，首先身體上升與下降都要維持水平，不要變弓背，其次是下巴要內收（看到雙下巴就對了！）但不是頸部下壓。

練臀運動2：單腳橋式

單腳橋式是橋式的進階版，對於初學者如果不熟悉出力方式，可以在大腿與肋骨之間夾一顆網球或是寶特瓶，這樣子可以更容易地讓軀幹保持水平、不弓背。進階者則可以練習下圖臀橋，透過臀肌出力帶動軀幹，但同樣要讓脊椎保持水平不彎曲唷！

練臀運動3：負重臀推

最後，是我們在健身房中常見的「臀推」訓練，也是所有追求性感飽滿翹臀的人必備的運動姿勢！不同於臀橋，我們將上半身肩胛骨靠著椅子，雙腳自然與肩同寬後，將杠放在大腿根部附近的位置（如果覺得不舒服，可以在杠之下放軟墊保護），而後屁股用力向上夾緊，到達頂點後停頓1-2秒，再慢慢放下。如果覺得杠的重量太重，可以用影片方式改用彈力帶作為負重，會更容易執行唷！
