變老等於衰老嗎？運動讓你逆生長的秘密是什麼？

2019-12-16 2,016條評論 4,825次閱讀 2966人點贊

絕大多數人的傳統認知是：隨著年齡的增長，人就應該患有疾病、衰老；但科學研究結果與眾多的事實告訴我們：疾病和衰老現象並不是年齡增長的必然結果。人們下意識地將變老與衰老無縫連接起來，併當作一個信念，人們認為：當年齡增長，人就會變得虛弱、生病，就會很快死去，生活質量的下降也是理所應當。如果你抱著這樣的想法規劃生活，將是很危險的事。

實際上你會變老，但你可以高質量地活著，憑藉當今的醫療、科技水平，活到80歲、90歲是大概率事件，因此你需要讓自己生命的最後三分之一過的精彩，而不是籠罩在肥胖、關節疼痛、心腦血管疾病和了無生趣的陰影里。我們完全可以逃脫大多數衰老現象，並快樂、優雅地老去，真正實現自我生命的豐盛與滿足。

全球年級最大的模特：87歲的卡門 （Carmen Dell Orefice)

一、變老與衰老的區別是什麼？

變老與衰老有一個至關重要的區分點：

變老是不可改變的生物機制，對此你無法改變：比如頭髮會漸漸變得灰白；地心引力的作用會使皮膚無法像20多歲時那麼富有彈性和緊緻；最大心率會隨著時間而逐漸下降；眼睛會花等；但是，你的行為、精神狀態和心理卻不一定非得變老。這才是最重要的！

雖然目前還沒有找到永葆青春的方法，但變老的過程可以是緩慢的、細微的、優雅的。健康、乾淨的老年人和「順其自然」的老人大不相同。

同樣74歲，一位是醫院常客，一位是健身房常客；一個靠氧氣安享晚年，一個做超級網紅開創新事業。

67歲的同齡人

二、為什麼很多人未老先衰？

幾十萬年前的人類祖先是生活在非洲大草原，他們每天需要通過奔跑去捕獵動物；或者行走幾個小時去採集漿果，這種運動讓身體變得精瘦、有力量；骨骼、關節變得越來越強壯；心肺功能的改善為肌肉提供血液和氧氣，使肌肉變得結實、柔韌、協調；免疫功能也增強。

行走和奔跑產生的神經遞質比如內啡肽讓人感覺快樂、精力旺盛，好奇心增強，更願意探索未知事物，所以，我們的祖先是精瘦、健康、樂觀、肌肉強健、心臟強壯並擁有良好的免疫系統。好消息是，我們每個人都傳承了這些強大基因。

精瘦、靈活、肌肉強健的人類祖先

但是現代的生活方式正在摧毀我們原有的健康基因，比如吃大量垃圾食品、熬夜、長時間刷手機、久坐少動、喜歡獨處，加上工作、婚姻、人造光線和噪音、上下班擁擠等諸多壓力，給身體和大腦產生致命的改變。

當人選擇久坐不動、獨處、情緒抑鬱，身體誤以為冬天到了，於是啟動低熱量消耗模式，儘可能將能量以脂肪的形式儲存起來；通過讓肌肉消失、關節老化、關閉免疫系統等方式降低能量消耗，衰老、疾病就這樣發生了，即使你還很年輕。

新華網發布的《2018年十大不健康生活方式報告》，久坐不動、超長使用電子產品、熬夜位居前三

三、覆蓋衰老信號，讓逆生長成為可能

我們天生就有運動基因，只有運動起來，哪怕僅僅是走路，你的身體也會收到微弱的生長信號；也就是說，你每天都需要動起來，告訴你的身體現在是春天，讓生長的信號覆蓋衰老信號。

請記住：衰老不是年齡增長的必然結果，而是生活方式選擇的結果。如果你養成運動的習慣，無論是有氧運動、力量訓練還是瑜伽，那麼減緩衰老速度，讓自己逆生長是絕對可以實現的，我自己就是成功案例（感興趣的讀者可以看我寫的微頭條：健身讓中年的我找回自信 https://www.toutiao.com/a1652083386076171）

在未來的日子我會繼續分享運動與延緩衰老的科學研究結果和依據。
