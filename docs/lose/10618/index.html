了解SMART原則，制定減肥計劃

2019-12-07 645條評論 2,040次閱讀 448人點贊

圖：來源網路，如有侵權，請聯繫
文：小c

時間：2019-12-6，天氣：晴，溫度：4℃，狀態：平和

根據原計劃是要在5號當天進行訓練的，結果3號訓練帶來的乳酸堆積後遺症導致延期了一天，今晚訓練時雙腿依然傳來陣陣酸痛感，好在反應沒有那麼強烈了。

最近，在排酸這幾天，小c一直在思考一個事情，那就是制定計劃的原則，工作常用的SMART原則，完全可以拿到這裡來使用，SMART原則分五個維度去制定一個計劃：

一、SMART原則一S(Specific)——明確,具體

明確性和具體性主要是指以下兩點:

1、目標設定不能過於寬泛 。

2、 目標設定不能主觀。

二、SMART原則二M(Measurable)——可衡量性

衡量性就是指目標應該有一組明確的數據,而且這組數據必須要具備漸進性,作為定期追蹤和衡量是否達成目標的依據。

​

三、SMART原則三A(Attainable)——可達成

可達成性其實是包括兩個方面的:1、目標要能在目前的狀況下得以實現 2、目標必須要有挑戰性。

四、SMART原則四R(Relevant)——相關性

相關性就是目標要與目的先關聯,如果目標脫離了目的,就會撿了芝麻丟了西瓜,甚至可能會南轅北轍。

五、SMART原則五T(Time-Bound)——時限性

目標特性的時限性就是指目標是有時間限制的。

以小c減肥這件事舉例來說，遵循SMART公式設定的目標是小C通過有氧運動和無氧運動和合理的飲食調整，180天減重24斤，體重降到160斤，體脂降到17%。到這一步還不夠細，還需要進行具體的時間節點來達到目標，例如1個月為時間節點，180天就是6個時間節點，每個時間節點體重掉4斤，體脂掉1.5%（這個1.5%相當難，尤其是後面時間節點的1.5%）。一個月還需要進一步細化，一個月分為四周，一周訓練4—5次，休息兩到三次，無氧訓練為主，冬天盡量少戶外跑步，以免風灌進脖子，身體容易出現傷病，影響減肥計劃。

看似挺簡單的一個目標，但對小c來說一點都不簡單，目前自己的體脂是26%，最近一年習慣了飲食不節制，突然控制飲食量相當難受，腹部隔一段時間傳來飢餓感時很是崩潰，最近除了訓練休息外， 也在嘗試合理安排自己的飲食，晚上開始自己做水果酸奶沙拉，有點喜歡上在家​倒騰點吃的感覺了，那種​斤斤計較自己進食分量的感覺也確實挺有趣的。

今晚的訓練內容挺簡單，俯卧撐訓練為主，說來慚愧，今晚做的俯卧撐姿勢連入門級都不到，純循序漸進姿勢，做起來不難，身體容易承受，不過對目前的小c來說還是有些難度，​身體力量大不如之前了。

訓練姿勢一：上斜俯卧撐

訓練目標：每組40個，共三組

訓練體驗：自己一開始做這個姿勢是為了打算熱身用的，然後挑戰膝蓋俯卧撐，再挑戰半俯卧撐，直到標準俯卧撐。可惜，理想很豐滿，現實很骨感，小c差點倒在上斜俯卧撐的路上。印象中的上斜俯卧撐是信手拈來熱身專用的，結果臉被打的那個疼，第一組下來，雙臂就顫抖的厲害，很想停下這個運動目標，躺床上玩玩手機刷刷劇，讓扯淡的目標滾得遠遠的。第二組訓練明顯感覺力量在流失，身體開始不聽使喚，只能自我激勵，每完成一次起伏就給自己一個贊，三組訓練就在身體傳來的抗拒和自我激勵下狼狽​的完成了。

訓練姿勢二：垂直引體向上

訓練目標：每組50次，共3組

訓練體驗：這個姿勢難度比較低，選擇這個姿勢純粹是為了緩解雙臂的酸痛感，三組上斜俯卧撐下來之後，以自己的身體素質來說，膝蓋俯卧撐鐵定完不成訓練目標次數，便將這個姿勢納入到了當晚的訓練​進程內。訓練完成之後，手臂痛感並沒有好轉，小c忍著身體傳來的酸痛感，進行當晚​牆壁俯卧撐的訓練。

訓練姿勢三：牆壁式俯卧撐

訓練目標：每組50次，共3組

訓練​體驗：得知這個姿勢自己都快駕馭不了的時候，煩躁情緒一閃而逝，顫抖著把手臂放到了牆上，​開始了最後三組的訓練。三組下來什麼感覺呢？感覺手臂不是自己的了，原本當晚寫的訓練日誌活生生被拖到了今天下午，現在手臂依然有酸痛感傳來。身體弱姿勢就難，身體強訓練姿勢就弱，這次的牆壁俯卧撐訓練是截止到目前記憶最深刻的一次。還好這都是可以後期循序漸進​彌補過來的​。

訓練飲食：半個蘋果，半個獼猴桃，一根香蕉，5個聖女果，一片全麥麵包，一包150ml酸奶，少許核桃仁做成的水果沙拉餐，吃著很贊，自己對廚藝都開始有自信了。

小c慢跑，運動路上的追尋者，期待你的一路同行。

只要開始，永遠不晚。

Never too late to begin！
