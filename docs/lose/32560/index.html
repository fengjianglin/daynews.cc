場上的指揮官——淺析前場隊友在雙打跑位中的領導地位

2019-12-24 2,186條評論 3,594次閱讀 1326人點贊

本帖最後由 球齡七年半 於 2015-12-9 12:21 編輯
還是要說下大前提，只針對業餘雙打，尤其是男雙（沒打過女雙，以下內容可能不適用），要求拍檔的雙方都具有比較高的技戰術水平，按照中羽的分級至少要5級以上，沒有明顯的技術弱點，基本兩人都比較全面，可前可後。
好吧，大家退後，我要開始zhuangbility ！！！
業餘雙打中的跑位，一直是個比較頭疼的大問題，跑得好，是1+1>2，跑不好，是<2，甚至是場上內訌以至於<1……一般來說，業餘很少有比較固定的搭檔，可以做到很有默契，相互比較了解各自的習慣和技術特點，有針對性的形成比較好的配合。這時候，一個比較普適性的跑位原則能讓大家合作無間、齊心協力以至於達到琴瑟和鳴、顛鸞倒鳳共赴高潮的效果（好像說錯了什麼…….）。在我看來，原則就是：一切行動聽指揮，一切指揮出前場！細細展開，有如下三點：
一個中心：這個中心指的是場上的中心區域，也就是以左右半場分界線為中心的一個區域，雙打拍檔的兩人位置連線的中心一般要落在這個區域裡面。具體範圍見下圖（大概是斜線陰影部分）。這個比較容易理解，以前的很多帖子也針對這點闡述得很詳細到位。簡單說來，就是兩個人位置互補，覆蓋整個場地，不要在場地上留白。這個中心區域，在進攻的時候會比較偏向後場隊員位置的那半區，畢竟前場隊友要考慮封堵直線回球的話，一般會和後場拍檔站在同一個半區。
一根槓桿：跑位是以位置連線中心為支點的槓桿，撬動兩人位置變化。也就是你上我下，你進我退，類似一個剛性連接體，場上每個人的位置都要隨著拍檔的位置變化而隨之變化。但總的原則是要保證連線中心是落在中心區域里。這樣的隨動時刻保證對場地的最大覆蓋和控制。
一個指揮：前場主導，帶動後場跑位，覆蓋全場。關於前兩點，一般水平高點的球友應該都比較了解，並在實際雙打配合中應用得比較好了。我要強調的是這一點，場上的指揮官一定要是前場隊員，所有的跑位的發起人都應該是前場。絕對一點的來說，凡是前場隊員的選擇都是對的，後場隊員都要無條件的配合。從視野的角度來說，前場一般是無法看到後場拍檔的站位的，他只能根據自己的習慣和特點進行擊球和跑位，後場是可以看到全場站位和隊友跑位情況，就一定要根據前場隊友的跑位來相應補位。而不能按照自己想當然的正確跑位來行動。要知道，雙打是兩個人的事情，後場需要配合前場，前場是沒法去配合後場的。後場不能因為自己「正確」的跑位而前場不配合從而抱怨。再跟著我念一遍：第一條是前場隊友總是對的，如果有錯，請參考第一條。
一般來說，站位有前後進攻站位和左右防守站位，在進攻站位的時候，自然就分出了前後，後場的隊員進攻線路需要參考前場隊員的站位，盡量保證球從前場隊員的頭頂或旁邊下壓，讓隊友可以封堵對方威脅最大的直線回球，這時候，如果後場是殺球，前場應該適當後撤，防備對手的回抽直線或斜線，如果後場吊球，前場應該前壓，防範對手放網並逼迫其繼續起球。如果對手的中後場球前場覺得可以後撤下壓，並且向後場移動，這時候後場隊友一定要相應前壓，一切行動配合前場的跑位。關於這一點，李在福的相關視頻講的很清楚，大家可以去看看。進攻的時候前後場位置清晰，分工明確，比較容易實現配合。
左右防守站位的時候，我建議斜線位置的隊員要適當壓前，一方面可以更好的防範對手的斜線劈殺和劈吊，另一方面可以錯開兩人的位置，形成另一意義上的前後場。此時，前場隊員可以根據自己的能力和技術特點大膽採用各種接殺手段，後場隊友用餘光隨時關注前場的動作和跑位，相應選擇站位和對策。這樣的一個錯落站位，可以很好解決中路球的爭搶問題，站位前的隊友不用顧忌視野外的拍檔，根據自己能力搶中路球或者放過，後場隊友只要瞟到前場有動作，就可以繼續穩紮馬步，如果前場沒動作就放心大膽的出拍。
另外提醒一點，關於中後場隊友放網誰上網的問題，如果按照我上面的三個原則，此時前場的隊友應該在斜線位，最好是側身可以用餘光看到拍檔的身影，此時如果拍檔習慣性上網，前場隊友就可以退下補位，拍檔不動的話，就積極上網封堵。
另外說個配合中的小失誤，接發球我輕推直線腰位，並稍微後撤抓直線回球，對手推直線過我頂，這時我後撤準備下壓，但我的眼睛餘光並沒有看到隊友的身影，擔心隊友也過來搶球就趕緊停下。結果球落在身後，後場隊友嘀咕：你退了兩步我就沒過去接球了，誰知道你又不動了。我只能在心裡吐槽，我既然後退了你居然還不上前補位，我只知道你在我身後哪能知道你會不會搶球？安全第一啊。
好吧，就這麼多了，歡迎探討。來源中羽：球齡七年半
