健康體適能，告訴您如何科學運動？

2019-12-18 363條評論 4,213次閱讀 1254人點贊

健康體適能是對人的素質、體質和健康知識三方面進行全方面測量的綜合衡量系統，通過這一健身監測與指導體系可以全面了解自身的健康水平，主要包括心肺耐力、身體成分、肌肉力量、耐力、柔韌性、反應能力、平衡等與運動機能相關的身體素質。通過對這些身體機能的測試和分析，以及運動前風險的評估，可為健康人群、超重肥胖，以及高血壓和糖尿病等慢病人群制定個性化運動處方，讓大家切身體會到專業的運動管理模式，真正做到安全、科學、有效的運動，提高運動效果，預防不必要的運動損傷。

1.健身先健心，才有百歲身——心肺耐力測試

心肺耐力是評價人體生命狀態的第五大生命體征，決定人的健康水平，心肺耐力對健康的預測完全可以和抽煙、高血壓、高血脂、糖尿病等傳統健康危險因素相媲美。心肺功能與死亡率有關，低心肺耐力是影響全因死亡率的最大風險因素。

2.腰帶扎在肚皮下，體肥血栓危險大——體成分測試

人體內各成分比例合適並保持相對穩定是健康的重要條件之一，如果某種成分過多或過少，身體成分失調，會導致各種疾病的發生風險增高。人體成分分析儀是通過生物電阻測試方法，準確測量人體內脂肪、肌肉水分等基本成分。其中體脂率這一指標可以準確評估一個人的肥胖程度，可以讓測量者清楚了解自己體內成分的現狀。

3.握力出奇蹟，死亡風險低——肌肉力量測試

握力每減少5公斤，因任何疾病而死亡的風險增加16%，心臟病發作和中風的風險分別上升7%和9%。握力每增加1公斤，死亡風險可降低3%。

4.筋長一寸，壽添十年——柔韌性測試

柔韌性是運動能力的基礎，是人們完成日常工作和進行鍛煉的必備能力。柔韌性差與腰肌勞損和椎間盤突出密切相關，增加拉傷、扭傷等日常生活中身體損傷的風險。

5.遠離衰老和跌倒，平衡、反應是關鍵——反應、平衡能力測試

身體反應、平衡能力表現了身體肌肉協調能力、中樞神經系統處理信息的速度、各種感覺器官的功能及靈敏程度，是一個人身體綜合素質的體現。良好的平衡、反應能力，讓您遠離跌倒風險，延緩衰老。
