乒乓球如何練好側身拉斜線？許昕戰例講解拉斜線難點，其實並不難

2019-12-23 608條評論 1,524次閱讀 72人點贊

許昕，中國乒乓球運動員，左手直板反膠，正手弧圈球力量大旋轉強，全台可以正手拉沖。

直板，在反手位側身正手拉沖，是威脅最大的進攻技術，但很多業餘愛好者無法真正學好，我們來討論一下這個問題，難點在哪裡。

上圖為許昕對陣張本智和的側身沖戰例。

許昕發球後，張本智和正手擺短，許昕回擺。張本智和正手加轉弧圈球拉起，許昕側身反手發力沖。

反手位側身，很多業餘愛好者，過於注重手上動作，而忽略了最重要的步法以及站位。

站位，許昕是左手，左腳略後，右腳略前，而右手運動員則相反，左腳略前，右腳略後，略側向。

步法方面，在反手位，需要用並步，有的人也稱為墊步，其實一樣的，稱謂大同小異。從戰例中，可以看到，許昕腳下一直在並步跳，側身時，需要向右方跳一步，讓開來球路線，讓出空間，方便正手發力。注意腳下的位置。

上圖為拉球完畢後的站位。

拉球完畢後，許昕為左手，左腿發力蹬轉。

業餘愛好者常見的錯誤，主要還是腳下步法，以及站位問題。

來球速度過快，很多愛好者並步過慢，來不及讓開來球的線路，被直接頂住。這個問題，主要還是眼睛盯球的線路，對球的線路判斷較慢造成的。日常練球，多體會來球線路，體會節奏以及力量控制。

站位問題，很多愛好者站位兩腳較平行。

過於平行的站位，在側身時，以右手運動員為例，右腳移動的範圍過大，有時可能需要並步跳兩下才側的開。如果這樣，不如站的時候，左腳就略前，右腳略後，這樣方便了正手側身時，腳步撤的開，撤的快。

平行站位，很多時候在橫板反手相持時相對方便，但是側身難度會增加很多。而且反手再好，也不會比正手威脅大。所以站位時，略側向站，對使用正手會很方便。

乒乓球，四條線路。

側身位的難點，主要就是並步撤開，腳下站位，這兩點是側身拉斜線的必須。如果自己出現問題，可以從這兩個方面找一下原因。

手上動作其實沒什麼改變，能在正手位拉穩定了，就可以拉斜線。很多愛好者覺得拉斜線動作要改變，或者不順暢，主要還是站位問題，側開站，右腳略後一些，側開身給正手擊球的空間，也就動作一樣了。

乒乓球四條線路里，反手位側身正手拉斜線最難，需要多練習並步撤開，側向站位，然後會很順利的發力進攻。直板橫板都一樣，站位正確了，都可以拉斜線。

側身拉沖，之後的銜接很可能會撲正手，這個需要使用交叉步，將在其他文章專門講解這個問題，本文僅討論一下側身斜線的難點。

乒乓球單項技術，需要刻苦努力練習，結合一定的理論，提高水平會快很多。

水平有限，敬請指正，共同學習，共同提高。
