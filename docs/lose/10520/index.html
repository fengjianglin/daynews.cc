膝蓋總是向內偏，影響美觀又容易受傷，糾正方法其實就這麼簡單

2019-12-11 107條評論 1,990次閱讀 411人點贊

很多人在日常生活中或做下肢訓練時，尤其是女生特別容易出現膝蓋內扣的問題，出現膝蓋內扣不僅影響腿部美觀，而且也是膝蓋疼痛的主要誘因，很容易導致功能性「X」型腿、膝蓋周圍韌帶拉傷以及膝關節受損等各種問題，給日常生活和訓練帶來很大的影響，膝蓋的損傷往往是不可逆的，所以改善膝蓋內扣刻不容緩！

很多人常常把膝蓋內扣歸結於是個人習慣或先天原因，認為很難糾正，其實主要的原因只是肌肉的失衡和錯誤的發力模式導致的，膝蓋內扣沒你想的那麼難糾正，今天就跟隨健身教練Paul一起聊聊膝蓋內扣這一話題。

什麼是膝蓋內扣

膝內扣是指在日常生活中或一些下肢動作過程中，雙膝向身體中線方向靠攏，出現了內收或內旋的動作狀態。

尤其是在膝蓋做屈曲動作的過程中，更容易出現這種現象，比如日常訓練中女生最喜歡的深蹲系列動作中。

膝內扣是怎麼造成的

一般我們在走路或下蹲時，髖關節重心的前移會使股骨順其自然的產生內收和內旋的力矩。

但是對於很多剛剛接觸健身和未接觸健身的人來說，缺乏調整膝蓋外展的能力，很自然地做出膝關節偏離中線的錯誤動作模式。

而膝內扣問題多數發生在女生身上，這是因為女生骨盆的生理結構與男生有所不同導致的，女生的骨盆較男生更寬一些，所以與大腿股骨所形成的角度也比男生更大，男生的角度為90度，女生為120度，所以在做屈膝動作時，膝蓋向內的壓力也更大，更加容易出現膝內扣。

如何自測是否存在膝蓋內扣

1.閉眼原地踏步三秒，使身體放鬆，還原最初體態，停止後觀察自己的膝蓋是否與第二腳趾方向對位，若向內說明膝蓋內扣或雙膝相觸，若兩腳踝空隙仍然較大說明存在膝蓋內扣問題。

2.對照鏡子，深蹲時，膝蓋應始終與腳尖方向一致，起身時在鏡中觀察，若膝蓋不自覺的向內，說明有膝蓋內扣的情況。

膝內扣的危害

膝蓋是人體最大最複雜的滑車關節。只能做屈伸運動，若膝蓋在屈伸過程中出現內扣會出現：

1.髕骨軟化

膝內扣會造成股骨內旋的體態，也就是我們常說的「X型腿」，這樣的腿型在使用時，會使膝關節始終有向內的擠壓力，擠壓髕骨，最終導致髕骨軟化。

2.半月板和韌帶的損傷

股骨內旋，使膝蓋在屈伸運動中始終受格外擠壓力，擠壓內側半月板，會導致半月板和韌帶的損傷。

3.下背部、髖部、腳踝等部位損傷

膝蓋出現內扣，日常生活或訓練中力量都是跟流暢的從上傳導到足底，若傳導時出現偏差，這時不僅僅對於膝蓋會出現損傷，其他相鄰關節也會出現代償性損傷。

哪些生理構造會導致膝蓋內扣？

1.肌肉方面

臀部肌群無力

臀部肌群是主要的髖外展肌群，臀部肌群的薄弱是膝蓋內扣的主要誘因。

內收肌緊張

內收肌緊張也是導致膝內扣的原因，尤其是在深蹲時，若內收肌緊張，會抑制膝蓋向外發力，從而出現膝蓋內扣。

踝關節的靈活性不足和足弓塌陷，也是引發膝內扣的重要原因。

2.神經方面

很多資深健身人群，可以在常規情況下可以很好的控制膝蓋的運動軌跡，但當沖重量時，也會出現膝內扣的情況，這是因為由於重量過重神經系統更親向於膝蓋內扣來本能的借力來完成。

清楚了引發膝蓋內扣的原因，自然為我們接下來的調整帶來思路。

多數人都清楚膝蓋的很多損傷是不可逆的，但很少關注膝蓋是否處在正確的位置上受力，膝蓋內扣不僅影響腿部美觀，而且也是膝蓋疼痛的主要誘因，膝蓋大部分的疼痛和損傷都與存在膝蓋內扣體態有關。而又有很多人認為體態問題糾正起來很麻煩，其實多數的體態問題是因為生活中不良的習慣導致的肌肉失衡導致的，找到最根本的原因，你會發現問題變得如此簡單！
