“腰帶黨”？不管練啥都帶腰帶，別讓腰帶摧毀你的核心

2019-12-24 940條評論 3,734次閱讀 1605人點贊

本篇文章不是針對任何人

只是站在事實的角度向大家闡述真相

擼鐵、三大項經常練還有腰部問題？

問題到底出在哪？

經常訓練三大項，腰部依舊不太好，然後就走上拒絕練蹲、拉的動作，安慰自己是腰不好，的確要排除其它方式受傷的小夥伴，但別的小夥伴你是生下來就腰不好嗎？

先說我自己，我在訓練的過程中，除了超出我體重自身重量1.5倍的硬拉和深蹲，會用到輔助腰帶，其它的訓練環境都沒有這個習慣，但發現就有一批健身愛好者，一進到健身房就把腰帶帶上了，不懂得人可能還以為是腎不好，什麼保健腰帶的呢，稍微懂點的還認為那是束腰，也只有自己知道他最簡單的目的就是保護腰


腰椎的強大

想了一下這腰椎是泥巴捏的嗎，這是要多脆弱呀，深蹲沒體重重，就連練背，練胸都帶，依舊沒有多大的進步暫且不談，就這還整天安慰自己是自己的腰不好，不能蹲又不能拉的，借著這個話題，今天就分享一下健身護腰真正的使用時機

我們的腰椎的設計是可以進行360度旋轉的，但我們要知道負重情況下不發生任何角度是最安全的，既然能最為上肢與下肢唯一的骨連接，它肯定就沒有想像中的那麼脆弱，它是由你的核心肌群來全面保護你的腰椎，當你沒有受到外界很大量壓力的時候，帶上腰帶的那一刻你的核心或多或少的就偷懶了，久而久之得不到充分發揮的核心，也就退化了


（上圖就是典型的，輕重量就帶上腰帶）


核心肌群

核心肌群：想必大家都知道，不太清楚的小夥伴也知道核心是腹部，當然這是核心肌群的一部分，胸大肌以下，股四頭肌以上包括腹側面和背側面，總共29-31塊大大小小的肌肉就稱為核心肌群

那這些29-31塊肌肉合起來的肌肉群是幹什麼的，我們的腹部除了脊柱沒有一個「剛性」結構，只有內臟、脂肪、中空的器官，通過呼吸腹部收緊，膈肌上抬，腹腔裡邊的額壓力增高，腹腔里的臟器就像流體一樣，密度和水特別接近，這部分流體就會在脊柱兩側施加壓力，使腰骶部張力增大，從而得到穩定腰椎的作用

當你認為你帶上腰帶之後腰部的穩定得到增加，嘗試自己身體承受能力之外的重量，採用硬拉深蹲，深蹲的時候由於股四頭肌離心收縮力量不夠，有保護杠還行沒有保護杠就坐下去，硬拉的時候，猛的發力產生的剪切力，這些都會在最壞的情況下，對腰椎造成一些不可逆轉的後果，椎骨前移後移、椎間盤盆出


那你為什麼還要用腰帶呢

你控制不住，但是你覺得你帶上腰帶後能控制住，事實呢？是你用外力幫助你完成動作了而已，結果可能造成嚴重的後果，就是腰椎間盤壓縮、關節脫位，甚至發生剪切力造成急性肌肉扭傷

那說了這麼多，腰帶到底用不用，當然是用！什麼時候用，挑戰你身體合理承受範圍內的極限重量，還有就是官方定義的深蹲、硬拉超出體重的2.5倍才用，大家既然都是健身愛好者，不是力量舉運動員級別的，建議深蹲硬拉超出體重1.5倍就可使用腰帶的輔助，當然前提是你身體的水平可以承受這個重量，最後就是腰帶不要勒得太近，能放個手掌即可


還有就是現在健身氛圍好像對入門訓練者不太友好，訓練過程中，大重量你就牛、輕重量你就渣，不知道什麼時候定下的這種規矩，就導致很多人跟風盲目的沖一些大重量，動作不標準的情況下真的太可怕了，我勸一部分的小夥伴一定要慎重呀、量力而行

加強核心

有的小夥伴說了，我核心差就必須練啥都要腰帶輔助，那核心差你為什麼不去訓練加強呢，天然強有力的腰帶不用，非要買來帶上，裝酷？核心的30多塊肌肉怎麼練？很簡單上肢訓練結束加上幾組核心、下肢訓練結束加上幾組核心、堅持一段時間核心還差勁，那你真的與大重量的訓練無緣、與變大無緣

看到這裡帶著鍵盤行走江湖的人也有可能發問了，這些知識書上都有，什麼什麼的，你們還在這裡看什麼，哈！這種自作聰明的人也是屌，雖然我不知你健身什麼水平？參加過什麼體育項目？培養出什麼運動員？其實一般大佬也不會有這樣的自作聰明

寫在最後：說一些心裡話，真的沒必要大家去跟隨潮流，上交一部分智商稅，人家用什麼我也用什麼，核心力量不足好好練核心，輔助動作不行好好打基礎，這樣你才能更健康並且走得更遠

總之也就那麼點東西，健身愛好者也沒必要去追求什麼激素、什麼特效補劑、起這個橋好起那個橋好的，盲目追求這些東西就是在「害自己」，你還覺得自己還很happy，你就是你一定要保護好自己身體，才能笑到最後，下期見
