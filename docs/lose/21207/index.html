37歲王鷗身材有多好？4個動作，打造零贅肉身材

2019-12-18 1,809條評論 4,872次閱讀 1877人點贊

著名女演員王鷗相信大家都對她比較熟悉，她出演過很多口碑很好的電視劇，這讓觀眾對她留下了很不錯的印象，很多大獎的提名都有她的身影。王鷗現在已經是37歲了，作為一個選秀出道的演員，她的身材一直是人們注意的焦點。

近幾年來隨著越來越多的人們加入到了健身隊伍之中，明星健身也不再是一件少見的事，越來越多的明星的健身都被曝光出來了，大家看著他們的好身材其實也是羨慕得很。一般我們想起娛樂圈中身材好的女明星，很多人都不會想到王鷗，但是王鷗的身材在娛樂圈之中確實也是非常出色的。

37歲的王鷗身上不僅沒有一絲贅肉而且她的腰肢還是很多女孩子可望不可及的螞蟻腰，穿上一條牛仔褲都可以有幾個皺褶，更別說穿上高腰的裙子了，要是穿上裙子或許就是娛樂圈裡面的霸氣女王了。

看到王鷗微博的健身照片我們就知道王鷗是一位健身達人，不管是負重運動還是瑜伽，她都可以駕馭的遊刃有餘，正是因為長時間的健身訓練所以才可以讓37歲的她有那麼好的身材，如果你也想擁有這樣苗條的身材，接下來就教給你4個妙招，助你打造完美身材。

一、 跑步

跑步是我們生活之中最常見的運動了，跑步其實給我們帶來的好處是有很多的，而且動作要領也是相對簡單，首先保持頭部和肩部的穩定，然後身體軀幹連成一條直線，然後注意身體的擺臂那樣子就可以了。

二、 深蹲

深蹲是一種可以鍛煉到臀部的運動，而且它可以瘦大腿，它的動作要領也很簡單，首先腰背要保持好直線，使得髖關節低於膝關節，然後慢慢的蹲下去，就可以完成深蹲的動作了。不過要記得在做深蹲之前做好身體的熱身，以免拉傷或是關節受損。

三、 仰卧起坐

首先仰卧在墊子上，然後兩腿併攏，兩手上舉，利用腹肌的力量，兩臂慢慢的向前擺動，身體成為坐姿後，上身繼續前屈，直到兩手觸及腳面，然後恢復原來的姿勢就可以完成這個動作了。

四、 俯卧撐

俯身呈一條直線，手臂放在胸前的位置，兩手相距略寬於肩膀。然後在做這個動作的時候，用2到3秒緩慢下降身體，最終胸部距離地面幾厘米，最後用力撐起身體，就完成了這個動作了。
