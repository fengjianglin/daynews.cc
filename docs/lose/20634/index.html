羽毛球對打中高遠球動作變形及解決方案

2019-12-17 1,630條評論 2,600次閱讀 1659人點贊

我的高遠球站在原地打效果還好，但是移動中的就難說了。

而實際比賽中，幾乎沒有站著不動就能接高遠球的機會。對手通常是看到你在前場，才會打高遠球，所以，在後退中回擊高遠球就成了問題。

我主要是動作變形導致回擊效果差，打不遠，造成給對手喂球的情況出現。喂球，就意味著丟分。

動作變形無非以下幾種：

1擊球點變低。這會使回球飛行高度不夠，到對手場地時，球已經是很舒服的位置了，然後，對手直接就扣殺了；

2腳步慢，擊球點後移。後撤時沒用側身後退，導致移動速度慢，羽毛球下落時，自己還沒跑到合適的擊球位置，回球質量差；

3移動不到位，擊球點太靠左或太靠右。這個也是移動速度的問題，快速向左或向右後撤時，重心不穩、跑動不及時造成的，同樣會使回球效果變差，不能打到預期的位置；

所以，在直線後撤或斜線後撤中回擊高遠球成了新的問題。

我跟教練說了這些現象。

教練在訓練中加了這麼一個項目：

我在中場啟動跑向發球線，在A點用球拍觸地，然後迅速後撤到底線，在B點回擊教練的高遠球，然後再跑向A用球拍觸地，再後撤到B擊球,如此循環。教練有時打直線、有時打偏左或偏右的高遠球過來，我就在循環跑動中調整自己的速度、站位並抓住最佳擊球時機。

我得承認，

目前，移動中回擊高遠球的效果還不理想。

所以，這種觸地回球的練習應該還會有很多很多次。
