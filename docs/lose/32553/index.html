盤點了十種減肥水果，熱量第一的居然是香蕉？

2019-12-24 378條評論 1,517次閱讀 612人點贊

臨近過年，家家戶戶都會選購大量的水果過年，加上水果的營養價值也十分高，但是該囤些什麼水果，才能在過年期間不會長胖呢？脂玫樂首席營養師劉潔為大家盤點了十種水果，一起來看看有沒有你愛吃的水果。

第一名：香蕉（115大卡/100g）

香蕉是大家過年首選的水果，因為香蕉比較方便，剝開皮直接吃，不臟手。而且普遍都認為香蕉的通便能力很強，認為吃香蕉可以達到減肥的目的，其實我們購買的香蕉通常採摘下來的時候都是生的，後期通過催熟的香蕉通便能力其實相對比較弱。

第二名：石榴（72大卡/100g）

石榴出了名的能「抗氧的水果」，不但可以清除體內的自由基成分、延緩衰老、它還可以降血脂，保護腸胃等作用。而且石榴含有大量的膳食纖維，可以促進排便和健脾消積，但因為吃起來麻煩，導致失去了很多水果「粉絲」。

第三名：奇異果（61大卡/100g）

獼猴桃富含豐富的碳水化合物、膳食纖維、維生素以及微量元素。因為維生素C含量較高，還有「維C王」的美稱。吃獼猴桃可以可促進消化、美白抗衰老、吸附和清除結腸內毒素的食用纖維，也能起到一定幫助排便的功能。

第四名：蘋果（53大卡/100g）

蘋果是每個家庭首選的水果，因為蘋果不但易保存，營養價值也很高，富含果膠和鉀元素，能夠吸附體內的毒素，還可以起到消腫的作用。蘋果對便秘者也非常友好，其排便效果甚至高於香蕉！而且蘋果還是減肥愛好者的最愛。

第五名：櫻桃（46大卡/100g）

櫻桃可以抗貧血和促進血液生成，而且含鐵量很高，位於各種水果之首。吃櫻桃雖然不像蘋果那麼大眾，但是慢慢也走進了每一戶家庭，吃櫻桃還可以防治麻疹麻疹、降低體內的膽固醇和血糖，櫻桃也是減肥人群很好的選擇。

第六名：菠蘿（44大卡/100g）

菠蘿也叫鳳梨，夏天比較常見的水果，它的用處多的數不清：可以健脾養脾、養胃健胃、補血養血、增強記憶力、祛濕、開胃消食、止瀉和消除水腫。而且菠蘿中的蛋白分解酵素分解蛋白質的能力很好，可以促進胃液的分泌，具有不錯的減肥功效，孕婦謹慎食用。

第七名：葡萄柚（42大卡/100g）

葡萄柚是近幾年飲品店的「寵兒」，富含的維生素和活性成分，不但能加快人體內脂肪的燃燒、消除疲勞、美容養顏、預防高血脂。還能增加腸道的蠕動，促進消化，也能很好的達到減肥效果。

第八名：檸檬（37大卡/100g）

檸檬雖然味道相對較酸，平時很少會有人直接吃，但是可以切片泡水，檸檬的酸可以促進能量的代謝，瞬間化解疲勞感，且富含豐富的維生素C。每天喝一杯檸檬水，具有美容養顏的功效。

第九名：西瓜（31大卡/100g）

西瓜在冬天的市場較小，因為不是當季水果，價格相對較高，果味也沒有那麼甜，但是現在隨著科學的發展，過季水果還是很容易可以購買到，西瓜中含有的氨基酸具有利尿的作用，可以消除身體水腫的狀態，將體內多餘的水分，隨著尿液一起排出體外，幫助減肥。

第十名：西紅柿（20大卡/100g）

最後提到的就是西紅柿，也是我們所說的番茄，既可以作為蔬菜也可以作為水果，富含許多的茄紅素、果膠和膳食纖維，其營養價值很高，有抗氧的作用。而且西紅柿的熱量非常低，可以抑制脂肪的形成，是減肥的最佳單品。

多吃水果有益身心，這十種水果都是對身體十分有益的食物。而且大多數熱量極低，可以起到減肥的作用，但是我所說的減肥並不是說用這些水果代替主食，哪怕是熱量很低的西紅柿，一味的進食，體重也許會得到控制，但是也十分容易反彈。營養不均衡也會直接導致營養不良，提倡科學減肥才是健康向上的！
