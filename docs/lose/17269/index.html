小博士柔性辟穀5天與復食最真實的感受

2019-12-15 2,616條評論 4,676次閱讀 507人點贊

小博同學辟穀5天，想談談其中自己這5天來最真實的感受，小博是一個不折不扣的胖子，體重86公斤，經過5天辟穀總共瘦了11.5斤，下面我談談這5天的心路歷程。

第一天使用菊花一錢，蒲公英一錢，蓮子心一錢泡水，三餐只喝了水，中午與晚上有明顯的飢餓感，想要吃東西的慾望非常強烈，特別是晚餐，最是難受，但是忍過去一段時間，便不會有飢餓感了，原因是因為身體自身機制分解了糖原，胃裡分泌的激素變少了。第二天早上稱重，體重少了3斤，稱重過後，在昨天三種藥材的基礎上往水杯里添加了兩個紅棗，早餐吃了一枚水煮雞蛋清，中午和晚上均有飢餓感，而且比第一天更為強烈。稱重時候體重減少3斤.第三天，感覺與第二天相似，但是精神狀態還不錯，體重下降2斤，第四天，對於飢餓感已經比較輕了，對食物的慾望不太明顯了，上樓梯時候，下肢肌肉有些乏力，由於我當天出差去林場指導造林工作了，走了很多路，出了很多汗，感覺身體有些乏力，稱重少了2.5斤左右。第五日，也就是今天早上稱重比以前共少了11.5斤，因為想到晚上要復食了，所以一直忍著，感覺勝利就在眼前。晚上復食，猶豫腸胃多日未進食，害怕快速進食傷胃，因此選擇了半杯熱牛奶，一包小堅果，和水煮豆子，一枚水煮雞蛋，沒有吃主食，用牛奶打底醒胃，進食沒有不適感，發現舌頭對食物的感覺更為敏感了，從來沒有注意的一些味道突然能夠感受到了，這種感覺很細膩，很舒服。

以上就是我這5天開闢谷的真實感受，後面復食也需要注意清淡，搭配好營養，希望從此擁有養成一個好的飲食習慣。謝謝大家，如果有辟穀這方面的打算可以關注私信我，互相交流。哈哈������
