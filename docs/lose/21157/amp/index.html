黑巧克力被吹成“減肥零食”，裡面有什麼貓膩？吃黑巧前不要犯錯

天天要聞
2019-12-19

說起減肥零食，不少人都會想到黑巧克力。畢竟它是近些年來人氣超高的巧克力品種，經常會帶著「健康」的標籤出現，而且很多廣告里都稱它為「吃不胖的巧克力」。

那麼是不是減肥人群就可以放心吃黑巧克力呢？爆炸營養課堂的營養師和大家好好說一下：

首先要承認，黑巧克力確實是公認的「最健康巧克力」。因為比起其他巧克力，它的健康成分含量較高，可可的含量超過70%，而且能量也比較低。但要說吃它能減肥，那完全就是無稽之談了。

其實從理論上來說，吃任何食物都是會胖的，所以根本就不存在「能減肥的食物」。這是一個能量守恆的道理，和其他任何因素都沒有關係。

當然啦，生活中所說的「減肥食物」，往往指的是相對而言食用後更不容易長胖的食物。把它們吃進肚子里增加的能量較少，當我們通過鍛煉消耗能量時，會更容易達到減重的目的。但很可惜的告訴大家，黑巧克力依然不屬於此類食物。

雖然比起其他巧克力，它的能量的確會更低一些。但巧克力這種食物，一直都是脂肪大戶，通常含量都是百分之50左右，就算有些黑巧克力可以達到較低的30%，也依然屬於高脂肪食物，

此外，它們的碳水化合物均在50%以上，所以一個巧克力的糖和油總量在80%以上，大家覺得吃了它們能減肥嗎？

數據顯示，每100克的巧克力所含的能量均在500千卡以上。也就是說，當你吃了100克的巧克力後，就攝入了一天超過三分之一的能量，多吃必然是很容易長胖的。如果你是巧克力的忠實愛好者，可要注意管住口哦。

最後提醒一下大家，很多人害怕巧克力軟化，買回來就會將其放進冰箱里。但實際上，這種做法是錯誤的，因為巧克力在低溫的條件下，很容易發生脂肪結晶的晶型變化。

雖然說也不會變質，食用之後也對身體無害，但這樣的巧克力口感非常差，不再均勻細膩。萬一把它們放進冷凍室里，這種情況會更加嚴重。

一般來說，巧克力放在十幾度到二十幾度的室溫下最為合適。如果室溫過高，大家就得想想辦法了，比如放進葡萄酒專用的儲酒櫃里。
