掌握好這幾點，讓你成為一名攻守兼備的中場

天天要聞
2019-12-19

大家好，今天我們分享幾個中場球員需要注意的要點，希望大家喜歡!

今天的要點並不都是為進攻隊員，同樣也適用於防守隊員，熟練掌握會讓我們的場上表現提高一個檔次，我們現在開始學習：

防守卡位：

假如有一個50/50的球，我們我們有很多方法可以，試著從對手腳下拿到球權。

圖1-很多方法拿下球權

但這些做法都不適合我們今天講到場景，假設對手人高馬大，又佔據控球的優勢，那麼我們硬上肯定不行。

圖2-不能硬上

他們肯定想要保持控制，因為這是他們所擅長的，而我們的目的是搶斷，我們不需要衝撞他，而是要尋找一個恰當的時機。

圖3-不需要衝撞

此時，我們真正要做的是卡位，尋找機會卡住位置，用肩膀和胳膊抵住對手，我們要善於用自己的肩膀和胳膊。

圖4-用身體卡位

用我們靈活的特點去卡位，然後護球，這樣我們就能奪回球權，繼續保持控球，這一點在比賽中非常重要。

圖6-卡位實例

接球前的準備：

作為中場在接球之前，我們一定要回頭觀察，所以我們必須時刻把頭轉起來。

圖7-將頭轉起來

否則我們就會經常「撞車」，或者被經常不突然出現的對手嚇得不知所措，我們肯定不想成為這樣的球員。

圖8-經常撞車

當球過來，我們觀察到身後有一定的空間，我們就可以不再簡單將球接到身前，而是可以直接轉身將球領到身後。

圖9-不在簡單的停身前

除了會觀察，我們還要有良好的身體姿勢，我們要習慣側身接球，尤其是在狹小的區域內，我們經常沒有時間觀察，半轉身的接球讓我們在場上更有效率，我們可以直接將球接向我們想要去的方向。

圖10-直接將球接到目標方向

當我們可以確保第一腳觸球就接向自己想要的方向，控球對我們來說就是一件簡單的事情，即使對手人高馬大，我們也不必擔心被對手搶斷。

接球訓練：

下面我們進入今天的訓練，這是一個反應速度訓練，我們不需要足球，不需要標誌盤，如果有更好，沒有也沒關係，我們在場地中央保持踮腳做好準備。

圖11-做好裝備

當感覺球要傳來，衝刺跑向右側，觸地，然後滑步返回，衝刺回來也可以，接著再做好準備，準備到下一個方向。

圖12-準備下一個方向

我們要練習向各個方向，這樣更接近實戰，給我們肌肉這種不斷的刺激。

圖13-給肌肉不斷的刺激

如果我們有隊友，我們也可以帶球聽他指令，我們做好準備，隊友喊出方向，我們帶球過去，然後再回來。

圖14-根據隊友指令接球

我們要不斷總結，並要掌握要點，如果有隊友就和隊友一起練，沒有就自己練，我們有責任讓自己變得更好。
