健身房帶毛巾只能用來擦汗？有些膚淺了，深蹲卧推硬拉都離不開它

天天要聞
2019-12-18

去健身房帶條毛巾，似乎是只針對跑步者的忠告。但我要告訴你：力量訓練者更應該帶毛巾。

除了擦汗，毛巾的用處還有很多。就拿深蹲、硬拉和卧推來說，多一條毛巾，說不定訓練效果會更好呢！

深蹲

選擇的深蹲重量較大，下蹲時後背的壓力越大，很多人深蹲感覺不適，並不是因為動作本身有問題，反倒是因為覺得背上「硌得慌」。

有護墊的健身房，深蹲者可以選擇使用護墊，但是護墊本身存在厚度，墊的東西厚會改變杠鈴在背部的位置，將杠位提高，並改變下蹲時重量的力線。

如果自己帶一條毛巾，必要時可以墊在斜方肌處，這樣能夠減弱上背的壓力感，同時不會讓杠鈴位置改變太多。

深蹲護墊

對於新人來說，使用此類護具是更安全的選擇。但是當身體逐漸適應深蹲的強度和重量後，還是應該去掉護墊和毛巾的。

同時還要在深蹲訓練中進一步優化杠鈴位置，從斜方肌慢慢向下移，同時需要調整手腕和肘部的位置，讓杠鈴下到三角後束和斜方肌中部處。

高杠深蹲（綠色）和低杠深蹲（紅色）時杠鈴位置

硬拉

毛巾在硬拉中的作用，和握力有關。握力是影響硬拉的重要因素。如果手掌出汗或者是抓握力變弱，即使你的後背、腿部尚有餘力，也不足以順利完成硬拉。

毛巾的第一個作用是擦拭汗漬，讓你的手掌保持乾燥。第二個作用，可以使用毛巾來改善握力。

握力的強弱和我們的前臂、腕部力量水平有關，經驗豐富的訓練者告訴新人：抓握器械的大重量訓練（如硬拉、俯身杠鈴划船）、自重訓練（如引體向上、懸垂卷腹）能夠改善握力。

這個說法自然不假。但新手往往因為握力的不足反過來制約了這兩類訓練動作表現。因此我們很難說握力訓練和大重量訓練，孰先孰後。

對於握力的改善，最直觀的影響是抓握杠鈴時更加穩定，當然對身體健康的幫助也很大。尤其對於伏案工作者，握力的提升有助於全身肌肉律動的動態平衡，也可以加速訓練者血液循環，緩解工作疲勞。

如果你的健身房有高處的單杠，那麼你可以將毛巾掛在杠鈴的兩端，雙手緊緊握住毛巾下端，讓身體處於懸掛狀態。毛巾的粗細決定你抓握的難度：如果毛巾卷比較細，那麼手掌可以全部攥住毛巾末端，這樣的難度小一些；如果毛巾較粗，手掌可能握的不全，這對握力的挑戰更大。

如果你只有一條毛巾，可以試試單臂懸掛，或是雙臂通過一條毛巾懸掛。這個的難度係數更高一些……

如果你覺得高杠懸掛比較困難，那就用深蹲架做杠鈴反向划船。

正常的杠鈴划船，人處於站立狀態，將杠鈴從底部拉向自己；反向划船時，杠鈴位置高於人體，此時我們的身體保持平直，通過手臂發力將身體拉向杠鈴。

這兩個動作不需要毛巾也可以完成，但還是建議你使用毛巾。毛巾的引入，給手掌抓握力帶來了挑戰，強迫我們在運動過程中，把更多注意力放到握力上。

反過來講，我們在硬拉時使用助力帶，是為了集中注意力在背腿肌群上，避免握力對動作造成不利影響。所以這和毛巾的使用，在訓練目的和動作重點上是截然相反的。

卧推

很多健身房的平板凳由於常年使用，凳面非常光滑。如果你衣服的面料也比較細滑，那麼在推起重量時會比較難固定肩胛骨，影響動作的穩定性。

這個時候在平板凳上放塊毛巾，可以增大摩擦係數，讓卧推更穩。

基於這樣的訓練需求，毛巾不宜選擇光面材質，一般純棉材質即可。同時毛巾要夠長，最短也要滿足「頭頂到肩胛骨」這個長度區間。當然了，肩胛不穩的同學，最好不要在卧推日穿光滑材質的衣服，如速干緊身衣。
