從肥胖熊貓腰到纖細螞蟻腰馬思純經歷了什麼？健身已經成為她日常

2019-12-22 2,470條評論 4,575次閱讀 3442人點贊

從”熊貓腰”到”螞蟻腰”馬思純經歷了什麼？

馬思純，中國新生代內地女演員。2014年，憑藉《左耳》獲得了金馬獎最佳女配角提名，2016年配節電影《七月與安生》讓她和周冬雨斬獲金馬獎最佳女主角獎。

獲獎之後，馬思純在採訪中談及身材對自己的影響 。她認為自己之前多次試戲失敗，都是因為當時自己太胖了，四處碰壁無戲可拍。在演藝圈裡，身材對一個女演員太重要了，上鏡要符合角色設定，更要好看的，身材大多數都是要求很瘦很瘦的。娛樂圈，身材好的人太多，你不優秀，就會被別人替掉。

馬思純認為在這一行里，永遠是別人因為你胖這件事去否定你，然而你在鏡頭上也能看到自己真的很胖，自信心會受到打擊變得不自信。於是乎，馬思純下定決心減肥，減肥期間，看電視的同時蹬自行車運動，每天晚上夜跑6公里，一天只吃一頓飯，多吃水果蔬菜，晚上少喝水，避免水腫，最終20天瘦了15斤，”螞蟻腰”小蠻腰也明顯可見，臉上的輪廓也更加明顯，自己也自信了不少。

減肥讓馬思純嘗到瘦下來的好處，瘦下來的她也是一直在積極健身保持身材，為了保持身材，天天運動，這給多少的胖女孩增添了信心。相信長期堅持運動健身，你就是下一個馬思純！

今天介紹一組體式，幫助大家減掉身上多餘的贅肉，練出美好的身材，一起來看看吧！！

1.臀橋

橋式，也被伽人們稱作小橋式，英文名Bridge Pose，梵文名Setu Bandhasana，Setu是壩、堤、橋，Bandha是鎖意思，Setu Bandha是築起一座堤道或橋樑的意思。

練習方式：

仰卧，雙手放在身體的兩側，雙腳打開與髖部同寬，雙腳腳尖指向正前方，屈雙膝靠近臀部，小腿與地面垂直，呼氣，抬起髖部向上，雙肩雙手臂向下壓向地面，胸腔向上抬起，雙手掌可以合十放在身體的下方，也可雙手握住雙腳腳踝，保持雙腿平行，膝蓋指向正前方，小腿垂直地面。

進階練習：

兩手支撐在腰部，小臂垂直地面，大臂內收，兩腿提起向上，身體以肩部、手臂支撐身體，頭部擺正，維持體式30秒，重複練習。

橋式可以幫助瘦臀又瘦腿。女性經常因為腿粗、屁股大而煩惱。橋式鍛煉橋式鍛煉有助於增強臀肌可以讓臀部和腿部更加健美、緊緻，體形更勻稱。一起加油哦！！
