被譽為“中國第一瑜珈老人”！90歲還堅持練瑜伽，令人佩服

2019-12-18 3,232條評論 4,681次閱讀 278人點贊

他叫沈維德，1928年生，是一位上海退休機械工程師，他從62歲開始自學瑜伽，至今已28年，被譽為「中國第一瑜珈老人」，在中國達人秀的舞台上，他用精彩表演，讓全場起立。

沈維德退休前在上海華東電子儀器廠做工程師。別看老人現在身體這麼硬朗，他退休前的大半世人生卻是在病魔纏身的痛苦中度過的。他當時患有心臟病、高血壓、膽結石、還曾因胃出血住院五次，甚至面臨癱瘓的危險。就在這樣的情況下，他決心鍛煉身體。他練過長跑和太極拳，都不管用。

1990年，一個偶然的機會，讓他接觸到了瑜珈。在開始練功時，由於年齡大，全身骨節非常僵硬，沈維德硬是憑著一股求生的慾望，堅持苦練，這一練就是二十多年。

他每天堅持專門練習兩小時以上，並將瑜伽融入行行走坐卧之中。經過這20多年的瑜伽練習，沈老面色開始紅潤，步履矯健，之前的老毛病統統都沒有了。

可以說沈老用實際行動證明了，瑜伽不僅是女人可以練的，更不是年輕人專屬，年齡等都不是理由。

經過上海中醫藥大學測試，沈老目前其生理年齡比實際年齡年輕18歲，這也得意於沈老對瑜伽十年如一日的堅持。

沈老說：「我修習瑜伽僅4年，前後就判若兩人。當周圍的人看到我輕而易舉地將兩腿交叉放到腦後，然後用雙手將整個身體騰空托起時，不禁發出陣陣喝彩。大家都感到驚訝，說我不像是一位老人，同時他們也懷疑我的年齡是否真實。他們或許認為，上了年紀的人骨頭都硬了，怎麼還能做得出這麼難的動作？有些人甚至不相信我是退休後才開始練的。」

「其實，我並非天生就這麼柔軟，這都是修習瑜伽的結果。瑜伽使人筋脈柔軟，肢體伸縮自如，且肌肉有彈性。」

沈爺爺老伴說，「他現在瑜伽已經生活化了，如果生活中缺少瑜伽，缺少練功，就比他沒有飯吃，沒有衣穿還要痛苦。」

90歲的沈老62歲開始練習瑜伽的，用20多年對瑜伽的熱愛和堅持，治好了一身病痛，滿滿的正能量。願你我同行，帶著瑜伽的正能量，繼續前進！

在年輕的時候遇上瑜伽，是上天的祝福，在年老的時候遇上瑜伽，是上天的眷顧。

瑜伽，是關於健康生活的科學。

大師說：「誰都可以練瑜伽，除了懶人「。
