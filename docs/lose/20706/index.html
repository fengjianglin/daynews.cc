越南男孩12歲時重194斤，17歲滿身肌肉網友不相信是同一人

2019-12-16 144條評論 3,286次閱讀 307人點贊

現在流行一句話叫「減肥如整容」，的確，很多胖男孩、女孩在減肥成功後都像變了個人似的，不花錢就能變漂亮，何樂而不為呢？就像文中要講到的這個男孩，從小到大都是小胖墩的他，如今在減肥成功又健身後變身型男。

據越南媒體12月16日報道，這個男孩名叫Vo Quoc Vinh，今年17歲，來自越南胡志明市，目前在加拿大讀書。從小就胖的他也經常因體型肥胖被同學們取笑，在家人的鼓勵下他成功減肥，後又經過健身，如今小小年紀身材就十分有型，還在健美比賽上拿過獎。

據報道稱，Vinh從小就是個名副其實的小胖墩，12歲的時候他的體重就達到過194斤。自從上學之後，Vinh更一直被同學們取各種綽號，例如「胖子」、「包子」等等。當時年齡還小，他並不覺得體重影響了自己的生活，也沒有被同學們的取笑所困擾。

然而，Vinh上了中學之後，隨著課業的加重，他越來越覺得有些力不從心，也沒辦法愉快地和同學們一起參與體育運動，因此他決定減肥。而家人早就覺得他的體重影響了他的健康，便積極鼓勵他減肥。那時候，Vinh僅用了一個月時間靠節食減下了40斤。

雖然瘦了下來，但Vinh覺得自己非常虛弱和疲憊，也打不起精神，所以他走進了健身房，開始了健身之旅。從2017年至2018年，Vinh僅用一年的時間就練出了一身的肌肉，胸肌、背肌、大腿肌、肱二頭肌還有6塊腹肌，這讓他於2018年在胡志明市舉辦的青少年健美比賽中獲得了金牌。

那次的冠軍讓Vinh感受到了自己減肥的成就感，從那時起他的健身計劃便更加嚴謹。如今的Vinh正在加拿大讀書，他開設了一個油管頻道分享健身有關的知識，他希望自己能夠順利地完成學業。

近日，Vinh在網上分享了自己減肥前後的對比照片，他希望自己的經歷能鼓舞到其他年輕人減肥成功。很快這組照片就在網上引起了熱議，很多網友都不相信2張照片中是同一個人。有網友表示，這前後變化簡直太大了，一個小胖墩和一個滿身肌肉的型男，誰能相信是同一個人。

還有網友表示，果然減肥如整容啊，這減肥後不僅身材更加有型，臉部線條也更加清晰了，完全變身帥哥啊！

的確，減肥對很多人來說都是個老大難題，一是無法放棄一大推美食的誘惑，同時又懶得去鍛煉。但不吃苦怎麼能成功呢，設想一下，只要堅持幾個月就能變身帥哥和美女，甚至還可能事業愛情雙豐收，何樂而不為呢？
