​吃多少碳水最燃燒脂肪，真香真相大揭秘

你好，我是健身甜心朵麗絲：只說最有料的減肥芝士!

本文作者：朵麗絲

對於什麼是「低碳水化合物飲食」並沒有明確的定義，對一個人來說「低」的東西對另一個人來說可能並不「低」。

一個人的最佳碳水化合物攝入量取決於年齡、性別、身體組成、活動水平、個人偏好、飲食文化和當前的代謝健康狀況。與久坐不動的人相比，身體活躍、肌肉發達的人可以承受更多的碳水化合物。這尤其適用於那些做很多高強度運動的人，比如舉重或短跑。

新陳代謝也是一個非常重要的因素。舉例兩種極端，將會是完全不同的結果，當人們患上代謝綜合征即使吃再多的碳水也不會發胖，如果是肥胖或者II型糖尿病，那基本就要向糖類食品揮淚告別了。

決定你每天的碳水化合物攝入量

如果你只是簡單地從你的飲食中去除最不健康的碳水化合物來源，精製小麥和添加糖，那麼在健康飲食方面，你已經做得很優秀了。

然而，為了瘦身，想要充分享受低碳水化合物飲食的新陳代謝好處，你還需要限制其他碳水化合物來源。儘管沒有科學論文確切的解釋過如何將碳水攝入量和個人需求做對比，但從眾多健身行業人士的經驗來看這麼做非常有效。

每天100-150克

這是一種「適度」的碳水化合物攝入量，適合試圖保持健康和體重的人群。

對於之前碳水攝入量很大的夥伴來講，以這種碳水化合物的攝入量來減肥是很有可能的，但我們必須計算好卡路里，拿捏好分量。

  • 你可以吃的碳水化合物:

一斤蔬菜，半斤水果，適量的健康澱粉。

所有你能想到的蔬菜均可食用，水果避免高糖類，推薦藍莓草莓柚子蘋果這些低GL的水果，也就是一個蘋果的量。澱粉類的碳水如土豆，紅薯和一些粗糧穀物，如大米和燕麥，也就是一個拳頭的量。

每天50-100克

這個範圍是需要勇氣的，如果你想毫不費力地減肥，但又想保證碳水化合物的基本營養，可以嘗試。如果你對碳水化合物很敏感，一吃零食或者主食就極易發胖，這也是一個很好的範圍來保持你的體重。

  • 你可以吃的碳水化合物:

大量的蔬菜，半斤水果，澱粉類碳水化合物的最低含量。

和之前的蔬菜水果選擇範圍一致，但是澱粉類不要選擇大米土豆等高GL的碳水，盡量選擇糙米小米或者南瓜等飽腹感強，維持時間長，血糖波動小的主食。

每天20-50克

這就是新陳代謝的好處真正開始發揮作用的地方，但必須配合營養師的指導。對於那些需要快速減肥、代謝紊亂、患有肥胖症或糖尿病的人來說，這是一個完美的範圍。

當你每天吃的少於50克，你的身體就會進入酮症，通過所謂的酮體為大腦提供能量。這可能會大幅度降低你的食慾，讓身體開啟燃脂模式。

這種範圍，必須在專業營養師指導下使用，很多女生自己使用低碳或者斷碳減肥法，最後結果是大姨媽出走，免疫力變差，掉頭髮，皮膚彈性也大不如以前，所以一旦到了這種量，就必須有營養師去保證你在營養充足的條件下去完成一些瘦身目標。

  • 你可以吃的碳水化合物:

大量的低碳水化合物蔬菜；一些漿果類的水果（比如莓類）；從牛油果、堅果和種子等食物中微量碳水化合物。

注意，這裡有一個最大的誤區，低碳水化合物飲食並不是沒有碳水化合物，蔬菜也屬於碳水化合物，這個和我們以往的認知有一些衝突。就我個人而言，當我開始注意碳水對身材的影響時，開始大量食用蔬菜，整個人的皮膚精神狀態都好很多，之前的便秘問題也不治而愈了。

總結：

對於那些經常運動或想保持體重的人來說，每天100-150克的碳水化合物可能是最理想的；對於那些有代謝問題，需要快速減肥的人來說，50-100g是很理想的區間。

你會更容易燃燒脂肪

適量的碳水化合物飲食會大大降低血液中胰島素的水平，胰島素是一種將葡萄糖(來自碳水化合物)帶入細胞的激素。

胰島素的功能之一是儲存脂肪。許多專家認為，低碳水化合物飲食之所以如此有效，是因為它們能降低這種激素的水平；胰島素的另一個作用是告訴腎臟儲存鈉，這就是高碳水化合物飲食會導致水分過多的原因。

當你減少碳水化合物，你就會減少胰島素，你的腎臟就會開始排出多餘的水分。所以，在一些人食用低碳水化合物飲食的頭幾天，通常會減掉很多水的重量，達到5到10斤。減肥在第一周後會慢下來，但這一次脂肪將來自你的脂肪儲備。

一項研究比較了低碳水化合物和低脂肪飲食，並使用DEXA掃描儀(非常精確)測量身體成分。低碳水化合物節食者減掉了大量的身體脂肪，同時增加了肌肉。

研究還表明，低碳水化合物飲食對減少腹部脂肪特別有效，大家都知道腹部脂肪是所有脂肪中最危險的，與許多疾病密切相關。如果你剛開始吃低碳水化合物的食物，你可能需要經歷一個適應階段，你的身體會習慣燃燒脂肪而不是碳水化合物。這被被國外的科學家稱為「低碳水化合物流感」，但通常在幾天內就會結束。在這個初始階段結束後，許多人都會感覺到精力也比之前好很多，這也是因為科學的飲食會讓血糖波動正常，不會引發食困。

建議：初期盡量去記錄一下自己的食物日記

如果你想嘗試一下本文的方法，那麼我建議你試著跟蹤幾天你的食物攝入量，來「感覺」一下你吃了多少碳水化合物。我自己喜歡用薄荷記錄，食物庫很豐富，非常方便。

我們都是獨一無二的，適合一個人的東西可能不適合下一個人。在不同的碳水量中去記錄一下自己的飲食體驗，找出適合自己的方法是很重要的。如果你有健康問題，那麼在做任何改變之前一定要和你的醫生談談，因為這種飲食可以大大減少你對藥物的需求!

值得注意的是，因為纖維並不算碳水化合物，可以把纖維的克數從總數中剔除。所以，計算凈碳水化合物的公式為:(凈碳水化合物=總碳水化合物-纖維)。

這個是我減肥後的效果對比圖，希望給正在減脂路上的你一些鼓勵。

每餐保證攝入一個巴掌大小的蛋白質，健康的脂肪（堅果，牛油果）和大量的蔬菜，以及半個拳頭左右的粗糧主食，中午加餐一個水果和全脂乳製品，選擇未經加工的食品。

沒有比這更簡單的減肥方法了!
