初學者晨跑，最大心率控制在60%左右最科學，精神狀態好、不疲倦

2019-12-20 85條評論 2,587次閱讀 1286人點贊

根據上一篇文章的統計，不可否認的是，習慣於晨跑的跑者人數，遠遠超過夜跑的總人數。有句老話說得好「一日之計在於晨」早上的確是跑者精力最充沛的時間段，經過一晚上的休息調整，狀態達到了最佳。多數人喜歡早起跑步，是因為早晨的空氣更清新，馬路上車少、人少。在這樣的環境下，呼吸涼爽新鮮的空氣，再加上有氧跑，就可以「喚醒」大腦，使大腦儘快從昨晚的抑制狀態轉化到興奮。雖然晨跑有提神醒腦的效果，那為什麼還會有人在晨跑後，會身體疲憊、無精打采，更想睡覺了呢？怎麼樣跑，才能達到晨跑的真正效果呢？

1、運動量

早上跑步是為了喚醒精神，保持健康。所以，跑步量的選擇一定要科學、適度，跑步距離不能太長，速度不能過快。如果跑量太大，很容易造成身體過度疲勞，導致跑者身體不能在短時間內從疲勞中恢復過來，使跑者睏倦，影響這一天的學習和工作。

有專家認為，晨跑的強度應該在最大心率的60%~65%之間。跑者可以根據自己的狀態，來判斷跑步的強度。當你跑步時，呼吸輕鬆而均勻，沒有困難。跑步後身體發熱，微微出汗，這樣的運動量最為適宜；如果跑步時氣喘吁吁、呼吸困難、出汗量大，心跳和呼吸在短時間內不會恢復到正常狀態，肯定會影響工作和學習。晨跑後，再做一些伸展四肢的運動，這將有助於恢復身體的疲勞，並提高跑者身體的靈活性。

2、晨跑時間　

生活節奏越來越快，導致晨跑後還有很多事情要做，例如：打掃房子、吃早餐、上班等，所有的事都需要時間。如果跑步後的休息時間過短，往往會造成精神緊張、做事手忙腳亂，從而影響工作或學習的精神狀態。

把跑步的時間定在早上，每天有規律的起床，留出足夠的跑步和休息時間，避免晨跑後的手忙腳亂、精神疲憊。建議晨跑之後，在開始工作或學習之前調整20~30分鐘。這段時間裡，可以處理一些雜務，然後有秩序地工作和學習，這樣效果會更好。另外，晨跑的開始時間跟上班一樣，是固定的。有利於養成晨跑習慣。每天到固定的時間，身體自然會進入跑步或學習的狀態，這樣更有利於精神集中，無論什麼都能取得好的效果。

3、根據實際情況做出調整

每位跑者的體質都是不一樣的，體質好的，運動量和強度，則可以適當的大一些；體質較弱的跑者，需要先進行運動量、運動強度較小的「快步走」，等身體素質好一些了，就可以進行跑步了，最開始速度放慢一點，距離短一點，在體質逐步提高的基礎上，再慢慢地加大運動量。

如果晨跑時生病或身體不適，一定要減少晨跑量，甚至停止。不要強迫自己去做影響健康的運動。運動量也要根據天氣和季節適當調節。當天氣不好的時候，例如，有霧的日子，颳風的日子也可以暫停晨跑。冬季天氣較冷時，可適當增加運動量（東北地區除外，太冷的地方建議進行室內鍛煉）；夏季天氣炎熱時，應適當減少運動量。
