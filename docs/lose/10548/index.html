這項羽毛球技術練不好，場上你就是挨打的活靶子

2019-12-10 1,489條評論 2,497次閱讀 977人點贊


經常使用挑球，就得把挑球挑到位，因為挑球的作用即可以用於進攻，也可以用於防守，是一項可攻可守的技術。

然而有不少球友挑球時卻常常不易到位，不到位的球就給了對手一個非常舒服的進攻機會。為了不讓別人把自己當成活靶子，必須把球挑的不給對手舒適進攻的機會，練好挑球刻不容緩。

下面就看技術分析：挑球有正手挑球和反手挑球兩種。

1、正手挑球：

①準備動作，正手握拍舉在胸前。

②右腳向前跨出一大步，左腳在後，側身向網，重心在右腳上，同時右臂向後擺，自然伸腕，使球拍後引。

③然後以肘關節為軸，屈臂內旋，並握緊球拍，用食指及手腕的力量，將球向前上方擊出。

④持拍臂隨慣性向左前上方揮拍減速，然後收拍並回動複位，還原成放鬆的正手握拍形式。

用正拍面向正前上方揮動為正手挑直線球；用正拍面向斜前上方揮動則為正手挑對角球。

2、反手挑球：

①準備動作，反手握拍舉在胸前。

②右腳向左前方跨出一大步，重心放在右腳上，同時右肩向網，屈肘引拍至左肩旁。

③然後以肘關節為軸，握拍經體前由下往上，用拇指第一指節壓住拍柄的寬面，用力將球擊出（收拍和挑球線路同理）

在實際應用中，反手的挑球更容易控制，尤其是在被動的情況下更容易發全力往後挑。而且在雙打中反手的挑後場應用更多，因為反手的防守區域更大。

挑球技巧：

1、如來球離網較遠，或擊球點較高時，拍面可稍後仰向前上方挑球；

如果擊球點離網較近，拍面就不能仰的太狠，否則容易挑到網上

2、如來球較近網，或擊球點較低時，拍面應充分後仰接近向上，而且在擊球時要用力向上「提拉」，以便揮拍多一些，從而避免球下網。

在這種情況下挑球時，擊球者首先必須充分降低自己的身體姿勢，因為擊球時如果身體重心過高，將會對手臂的充分向上揮拍有所限制。同時，應迅速地調整好擊球的拍面，使其接近與地面平行，能充分地向上用力。

3、如果對手的後場差，那就用挑後場來控制對手，讓對手進行無效的進攻來消耗對手增加對手的失誤。

4、假動作的使用，如果準備挑球時，看到對手順勢後退，那麼完全可以收住力放網前（收力屬於高進階）。

注意手腕的應用

注意事項：

1、上網步法時，膝蓋不能過膝；

2、腳尖要對著來球的方向

挑球訓練方式：

1）多球練習，一個人拋球，一個人挑；

2）一個人挑球，一個人吊/殺球，這是一種不錯的練習方式，雙方都能練習各自技術。
