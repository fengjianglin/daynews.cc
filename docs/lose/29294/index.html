「瑜伽斷食日」12月22日.薩帕拉･愛卡達西(Saphala Ekadasi)的榮耀

2019-12-21 393條評論 2,145次閱讀 257人點贊

12月22日，是瑜伽傳統中【薩帕拉･愛卡達西(Saphala Ekadasi)瑜伽斷食日】。

奉行斷食修習的瑜伽士(yogi )在這一天：完全斷食至中午，午後全天斷食五穀和豆類，可食用水果和蔬菜，喝水。

這一天，瑜伽士會儘可能的在念珠上，多念誦Hare Krishna曼陀進行梵音冥想。對那些已得到靈性導師啟迪的瑜伽士，在這一天念誦圈數由至少16圈增至25圈以上。

12月23日 上午結束斷食。【破戒時間為：12月23日07:33-10:39破戒】。意味著在這一時段內，食用少許五穀和豆類，即可完成完整的瑜伽斷食程序。注意：破戒時間過早或過晚，此次瑜伽斷食都無效。

薩帕拉･愛卡達西的榮耀

尤帝士提爾說：聖主啊，寶薩(Pausha)月月虧期間的愛卡達西名為什麼？怎樣去奉行？在這天應崇拜哪位神？佳納丹(Janardana，生物的維繫者)啊，請細細向我敘述。

至尊人格神首奎師那答道：聖王啊！我來完整地揭示寶薩·奎師那愛卡達西（Pausha-Krishna Ekadasi）的榮耀。

奉獻者在愛卡達西奉行完整的戒食要比犧牲和布施更取悅我，所以每一個人都要在主哈瑞的日子——愛卡達西日戒食。

帝士提爾啊，用全部的智慧來聽取發生在德瓦達西的寶薩·奎師那的榮耀吧，寶薩·奎師那愛卡達西亦稱薩帕拉·愛卡達西（Saphala Ekadasi），在這神聖的日子，應先崇拜管理神拿拉央納（Narayana），遵循經典推薦的戒食法，一如蛇類中蛇沙（Sesa）為最佳；鳥類中嘎茹達(Garuda)為最佳；祭祀犧牲中馬祭為最佳；河流中恆河為最佳；神袛中主維施努（Lord Visnu）為最佳；(雙)足生物中婆羅門為最佳；而所有的戒食日中愛卡達西為最佳。生於巴拉塔（Bharata）家族之最佳啊，無論誰堅定地奉行愛卡達西將成為我最親愛的人，而且從各方面得到我深深的祝福。現在請聽我詳述奉行薩帕拉·愛卡達西的過程。

在薩帕拉·愛卡達西，我的奉獻者用時令之新鮮果實崇拜我、冥想我——全然吉祥的至尊人格神首，並向我供奉水果、香料、香和酥油燈供奉我，在薩帕拉·愛卡達西供燈尤佳，奉獻者也應該嘗試徹夜不眠。

現在請聽我說與這有關的戒食和守夜帶來的利益，國王之最啊！沒什麼祭祀犧牲或朝聖能帶來與在薩帕拉·愛卡達西戒食同樣或更多的利益，如此戒食和守夜與苦行五千年的功德一樣，王中之獅啊！請聽這愛卡達西的榮耀歷史。

後來，日出到來，一匹英俊的馬走近路巴卡，在他身邊停下來，晴朗的藍天突傳來一個聲音：路巴卡，這是給你的馬，快騎著它回到你的王國，馬依斯瑪達王之子啊，因主華蘇戴瓦的仁慈，你在薩帕拉·愛卡達西戒食獲得極大的益處，你可以毫無障礙地回去了，這是你在這吉祥的日子中戒食的收穫，現在去見你的父親，恢復你皇室中的地位。

聽了天上的話，路巴卡騎上馬馳回禪帕瓦提城，因他在薩帕拉·愛卡達西奉行戒食的功德，他再次成為王子並專註冥想至尊人格神首哈瑞的蓮花足，換言之，他已成為我的奉獻者。

路巴卡謙卑地頂拜國王馬依斯瑪達，並再次接受了王子的責任。馬依斯瑪達國王把王國交給了他，路巴卡完美地治理國家，不論在什麼愛卡達西，他都滿懷奉愛崇拜至尊主，借聖奎師那的恩慈，他得到一個美麗的妻子和優秀的兒子，年老時，路巴卡就像父親一樣把整個王國交給兒子，自己去森林修行，控制心意和感官來服務至尊主，在凈化了所有的物質慾望後，他離開軀體，回歸神首，停息在聖主奎師那蓮花足旁。

尤帝士提爾啊，象路巴卡這樣接近我的人可以免除哀傷。事實上，任何人完美地奉行這榮耀的薩帕拉·愛卡達西，即使像路巴卡這樣不知不覺，在這世界之內亦變得著名，他將在死時得到完全解脫，回歸靈性星宿，這毫無疑問。另外，僅聆聽這薩帕拉·愛卡達西榮耀的人也獲得與執行皇祭一樣的功德，至少在下一世可投生至高等星宿。所以有什麼損失呢？
