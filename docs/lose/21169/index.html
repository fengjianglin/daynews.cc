羽毛球吊球技巧學習：吊球最大威脅不是落點刁鑽，而是帶有迷惑性

2019-12-19 878條評論 1,906次閱讀 71人點贊

打羽毛球按照出拍方向的話，（以右手持拍舉例）拍面發力有切或者是包切，這是一種常用吊球，還有一個是拍面抹或者是滑抹，這是一個滑板吊球，一般來說前者常見比較容易打，後者少見比較難打，但是經過我個人練習發現後者也容易打，下面小編和大家分享下。

擊球動作一定要和高遠球保持一致，就是要有欺騙性，吊球最大的威脅不是說落點刁，而是帶有迷惑性，在出球前讓對手還以為你是後場球，但出手後才發現是前場，但自己的重心已經被欺騙了，所以比較難接球。

為了讓球儘可能的貼網，這就要求在水平方向的力儘可能的小，也就是向前送的力儘可能的小，類似於在發力的時候輕輕切一下球托，要控制這樣的力量，小臂的肌肉肯定不行，所以必須要用手腕和手指的肌肉來控制。

整個動作就要這樣：開始準備，迅速用後場步伐調整到位，然後用高遠球動作，小臂迅速後倒，肘部上抬前送，小臂快速發力擊球，（一定要快速，你慢騰騰的人家一眼就看出你不是打高遠球了）在快接近的球的時候，迅速減力，但不是停住，這就是假動作了，拍面傾斜一個角度，用手腕帶動手指輕輕側切一下球托，使球向前的力能過網即可，他就會自然下墜。

專業隊常用的兩種吊球：

一、按吊球的落點來分，常見2種吊球：

（1）近網吊球，落點在前發球線以內的吊球；

（2）遠網吊球，落點在前發球線以外的吊球；

二、收吊的要點：

（1）讓吊球近網的關鍵是，把力「收」住，讓球過網既有下墜感；

（2）吊球可否發力？可以發力，在球從擊打到飛到網口這段水平距離越快越好，而一旦飛到網口附近，「如果球還繼續使勁往前飛」，就說明吊球力量大了，談到這個話題的時候矛叔點名表揚了鄭在成的收吊。

三、典型錯誤，通過「壓腕」來吊球：

喜歡「壓腕」吊球的同學關鍵在於對吊球這個「吊」落的錯誤理解，「壓腕派」無非是想通過外力，把球頭向下打出去完成吊落的弧度，實際上這樣極容易下網+手腕受傷，專業的吊球其下落的那一段距離依靠的不是外力，而是球向前飛不動了之後，收地心引力自然的下落。
