國術百科：傳統武學的兩種han胸拔背

2019-12-16 1,076條評論 2,174次閱讀 392人點贊

大家好，我是飛宏，傳統武學真傳一句話，而我願意說出這句話，希望可以幫助到大家圓滿武學的夢想。

一、您是否存在以下問題？

1、我們傳統武學的愛好者們特別注重一些傳統名詞，但是有些辭彙讓我們這些注重細節的人頗費腦筋，比如含胸拔背、涵胸拔背，這兩個詞的用詞不一樣，到底是個怎麼回事呢？

2、如果他們指代的內容不一樣，我們訓練的結果是否也會不一樣呢？

二、其實答案很簡單！

其實含胸拔背與涵胸拔背，確實是不一樣的東西，一種是外練，一種是內練，其重點功效，前者偏重於格鬥與四肢勁力，後者偏重於養生與軀幹抻拔。我們可以理解為一種是由外向內練，一種是由內向外練，完全不一樣的訓練，以及完全不一樣的配套訓練，在達到相同的終點之前，路上領略的風光，必然完全不同！

三、具體的原因！

1、注意

關於傳統武學的名詞用意頗深，很多學者都是不經意間混淆，大家更注重後代大師們的傳承與否，而不注重先代大師們的言詞本意。事實上，含胸拔背、涵胸拔背；沉肩墜肘、橫肩撐肘等辭彙之間，有太多的背後深意。有時候一字之差，意義就千差萬別。

傳武是分內向外練，以及外向內練兩種方式，而含胸拔背與涵胸拔背，就是典型代表。其代表的不僅僅是武學的不同，更代表創拳者的思路、思維、習性的不同。

有由無極樁入渾圓者，有由渾圓樁入無極者，此需切記！

2、涵胸拔背

涵胸拔背，多見於孫露堂先生的孫氏太極拳及其門徒，這個話要從無極樁說其，我老師秦文清先生曾對我明說過：無極樁是太極拳出功夫的根本，但是無數人以為太簡單了，反而練錯了或不認真練，導致沒出功夫。

孫露堂先生的武學起於他的無極樁，而他的無極樁功夫源於其中的涵胸拔背。大家仔細觀察可以發現，張烈老師演示的涵胸拔背姿勢很是怪異，撐開肋骨，收腹，頭向上頂，等等細節稀奇古怪，練起來也憋勁難受。

其實這個是源於孫祿堂先生對於內髒的訓練（他把丹道的東西嘗試著融入了武學），著重於養生目的，正確方法是，先細微調跨，讓後腰走直或略弓，此時後背督脈被拉伸，胸腹會略蜷；緊接著保持後背的直與弓，在此基礎上調直胸椎，特別是膻中穴這個地方，這樣可以拉伸前面胸腹處的任脈，督脈管陽，任脈管陰，孫祿堂先生用此法來做無極樁，力求陰陽平衡，這樣的好處是，同時拉伸任脈與督脈，同時拉伸陰陽兩脈，鍛煉小周天(小周天為軀幹部分氣血運行，主要就是任督二脈）。

然後在這個無極樁的基礎上，去做孫氏太極拳的所有動作，就會發現非常的累，但是練完之後身體會非常的舒適，功夫也會與日俱增！

然而此起點頗高，很多人又不明武學真理，於是練數十年孫氏太極拳仍然是練錯，導致不出功夫，不見養生。甚為遺憾！

這個是小周天的涵胸拔背，主養生！

3、含胸拔背

上過我早期【整勁核心體系】課程的學員都知道，我們那時候教渾圓樁，主要精髓在於調肩、調胯，（此方法現在已經被同行瘋傳，我們也已經將此方法淘汰。）調肩調胯的目的在於極力的舒展督脈，讓背部強大肌群的功能得以發揮，打出重擊能力。同時因為督脈打通，陽氣輸送至全身，很多學員在幾個星期內便得到了很好的養生效果。

這個技術里沒有拉直任脈，只是調節背部的肩跨，讓背部弓起來，讓勁力由肩跨傳至手足，很多號稱「站樁2小時沒問題的大咖」，用此法站樁不過10分鐘便承受不住，無超過半小時者。

其實這個才是真正的渾圓樁練法，把四肢勁力找到，並以此為基礎進行武學訓練，可以極速的提升武學水平。大家如果站樁三年不出功，那麼用這個方法，三個星期就可以明顯感受到重擊能力的突飛猛進！

因為此法是將肩跨勁力直接傳入四肢。而這個做法在周天法之中，被稱為「大周天」（小周天是軀幹，大周天是軀幹加四肢，全身的氣血通暢。）

實際上這個只是半個大周天，我們將大周天的精髓原理提取出來，組成了這個直接用於技擊的方法而已。

4、結論

通過這兩個講解大家不難明了，不同老師的教學意圖不同，孫祿堂先生認為以武入道，延年益壽最好，所以他以養生為根基，最後走入技擊，即使他本人是技擊當先。王鄉齋先生則雖然自己也是技擊當先，但他以武為入，後做養生。我個人原先的理念是養生可以獨求，但是技擊+養生的雙重目的的話，技擊為先、養生為後，更有利於學員成長。成材率較高！

武學中，不同的細節配合不同的練法，你不能拿這個門派的涵胸拔背配合那個門派的尾閭中正，這樣驢頭不對馬嘴，短期看能力有所提升，但很快會遇到瓶頸，荒廢一生！學者需要切記！
