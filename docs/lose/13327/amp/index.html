30天練習引體向上能增加背闊肌？看小伙挑戰後，背闊肌堅硬挺拔

天天要聞
5 天 ago

很多時候我們在工作之餘可以加強一下自己的身體素質，現在年輕人大部分是因為工作和家庭壓力讓自己處於亞健康狀態，身體每況日下，長期下去會累積很多的疾病。如果我們每天抽出半小時的時間，隔天練習一次相信你會有很多的發現。

上圖中這位17歲的小男孩叫馬蒂亞斯，堅持引體向上和俯卧撐及卷腹一年後練出腹肌，肩膀也變寬了，整體看起來肌肉增加了一倍，這樣的表現讓誰看了都想要。

如果我們每天堅持引體向上，一個月後會有什麼變化？

很多初學者在練習引體向上時一個都拉不上，全靠上肢的力量，背闊肌、肱三頭肌沒有力量是很難拉上去的。

有一些體重超胖的男子更是一個都拉不了，當我們的體重減下去時，在日常生活中加強一下背闊肌和肱三頭肌力量，那麼引體向上對於你來說並不難。

上圖中這位小伙挑戰30天引體向上，30天後會增加背闊肌寬度嗎？讓我們拭目以待！

在沒有基礎的情況下，他每天在做的時候會拉前拉一下手臂和大腿，這樣對於肌肉不會容易拉伸，每個健身愛好者都應該養成這樣的一個良好習慣。

挑戰第一天，他採用半拉狀態，這樣對背闊肌和肱三頭肌的力量要求要小，長時間這樣訓練之後，你會得到不一樣的力量。

那麼就是背闊肌越來越強壯，你會很輕鬆的拉起來引體向上，不要做一個無用的男人，要做就做一個肌肉猛男。

當小伙挑戰到第10天以後他可以一次拉到10個引體向上，本身他的體重並不是很重，當時只有70公斤，對於他來說進步很快。

瘦人在做引體向上時比較容易，當我們把背闊肌打開之後，會用背闊肌發力那麼接下來的你就容易多了。

大多數人做不起來引體向上，就是因為腰腹核心不夠穩定的原因，腰腹核心不穩定，手臂上拉的時候身體就會擺來擺去，從而造成力量不集中，引體向上也就拉不起來了。

如果單引體向上做不了建議你多做一些俯卧撐，這樣增加手臂力量和腰腹力量同時多做平板支撐，這是練習核心肌肉最佳的選擇。

我們一起來看看他挑戰30天後的改變？

當小伙挑戰30天後，我們可以清晰的發現，他背闊肌的改變更加硬朗挺拔，背闊肌寬度明顯增加。
