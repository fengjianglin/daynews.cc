生活中或者運動中你還腰疼么，知道怎麼改善么

天天要聞
2019-12-19

很多朋友在日常生活中有腰疼或者運動中腰疼的問題。

輕則影響我們日常生活品質，嚴重的會對日常行動造成嚴重的困擾。

今天賀偉就跟大家好好講講這個問題。

一，運動中腰疼

這部分同學通常是由於不良的運動體態導致的腰部壓力過大。

典型案例就是腰部反弓造成的脊柱前突，也就是典型的骨盆前傾體態。

在運動中我們需要佩戴合適的腰帶，保證我們的核心發力。

二，日常腰疼

這部分同學通常是日常久坐以及不良體態導致的腰部肌肉過度疲勞導致的。

同時跟核心力量差也有比較大的關係。

以下訓練就是針對腰部力量非常好的鍛煉，大家可以每天進行：

1，貓駝式 8次，做2組

注意放慢動作速度，全力放鬆並拉伸我們的脊椎。

2，改良卷腹 10次，做4組

腰部發力貼住毛巾，在保證支撐的前期提下，穩定腰部。

兩條腿要交替進行，一邊2組

3，鳥狗式 10次，做2組

運動過程中保證腰背平直，可以自己對著鏡子做。保持穩定。

4，側橋 10次，做4組

每邊做2組，力量較差的小夥伴可以彎曲雙腿降低難度。

三，日常注意事項

1，在日常中避免長期伏案，適當穿插休息。

2，盡量選擇合適的腰墊，讓腰部維持正常生理彎曲。

3，有氧運動盡量選擇固定單車或者游泳，減輕腰部壓力。

4，選擇一款合適的護腰，固定腰部，避免腰部壓力過大。同時保證核心發力。

另外，要強調千萬不要選擇束腰類的產品。束腰本身是違背正常生理學的，長期佩戴會對我們的肋骨造成過大的壓力以及軟骨組織變形。
