什麼是增強式訓練(Plyometric)?

簡而言之，增強訓練是一種反應性爆發力訓練。由前蘇聯體育科學家 Yuri Verkhoshansky發明。 該訓練方式利用了延長收縮周期方式來提高我們的爆發力、身體協調性及熱量消耗。

增強式訓練的核心是利用身體對於地面的反作用力，為運動過程中提供大量的動能，從而完成爆發力的開發，及高效燃脂的目的。

當時的訓練演示圖

如果你在發作用力的影響下離開地面，正在工作的肌肉組織是預拉伸的，這樣可以產生更高的肌肉強度。 這種情況放生的速度越快，產生的力量輸出也就越多。

因此增強式訓練中的「增強」一詞，是指增強動能、增強爆發力、增強消耗量，而增強的途徑或者說是「工具」就是：地面。你的負荷不再來自於傳統的物理負重，而來自於對地面施力所產生的反作用力，它在訓練我們爆發力的同時也需要身體消耗更多的熱量去完成動作。

如何利用增強式得到最大化收益

對於大部分健身愛好者來說，最大的收益就是消耗脂肪的同時保留儘可能多的骨骼肌。

  • 在傳統有氧訓練中，長時間低強度心肺訓練會消耗我們的骨骼肌，其主要目的是為了強化心肺功能，其次要效果才是燃脂；
  • 在HIIT訓練中，你需要大量的功能性訓練工具例如葯球、牛角包、戰繩、壺鈴等等，同時需要學習如何利用這些工具完成各種新穎而陌生訓練動作。

增強式訓練通常只需要非常小的運動空間，最大限度地降低訓練工具的種類，主要通過四肢對與地面的施力來提供反作用力，通過爆發力訓練達到大量能量的消耗，它既是針對快肌纖維的力量訓練（爆發力），也是高強度的有氧訓練，同時簡單易懂的動作對於大部分普通人來說更好上手。

6款增強式訓練經典動作

1.交替箭步蹲跳

完成3組，每組堅持30秒，組間休息30秒

這是最經典的增強式訓練動作，不僅可以鍛煉到你的股四頭、腘繩肌、臀大肌和核心肌群，還可以瞬間提升你的心率。與傳統箭步蹲不同，你需要驅動臀腿讓身體離開地面，在落地前儲存足夠的預收縮時間，在落地後再一次向地面施加作用力。

2. 側向抗阻跳

使用工具：彈力帶

每側完成3組，每組側跳10次，組間休息30秒

這是一個更加側重臀部外展功能的下肢訓練，你需要將彈力繩套在大腿周圍，注意不要套在膝蓋以下的部分，你需要完成側面跳躍的動作，如果彈力繩設置過低，會出現無法跳躍甚至肌肉拉傷的風險。

3. 抗阻向上起跳

使用工具：2根彈力帶

完成3組，每組12-15次，組間休息30秒

這是我個人最喜歡的增強式訓練動作，我在為期4周的增強式訓練中，逐漸意識到這個動作對於燃脂的效果非常明顯。以下會教大家如何設置這個動作

  • 如果你在健身房，可以將兩根彈力帶交叉固定在深蹲架底部
  • 如果你在家中，可以將兩根彈力帶交叉固定在床腿或桌腿下
  • 將兩根彈力帶以對角線方式套在軀幹上，固定在肩部兩側
  • 曲膝髖做一個預先深蹲的動作，用全力向上跳起，以此重複

4. 折返立定跳遠

完成3組，每組折返10次（一個來回算一次）

你不會想到，小時候體育課最常做的立定跳遠其實就是從前蘇聯體育科學家 Yuri Verkhoshansky那裡照搬過來的，其完整的訓練動作並不是看你跳的多遠，而是看你能折返幾次。

你需要在保證立定跳遠姿勢的情況下完成10次折返跳，當你可以毫無壓力的完成10次折返時，建議不用再設置次數，而是每組都力竭效果會更顯著一點。

5. 擊掌拍俯卧撐

簡單版：俯卧撐騰空

超簡單版：跪姿俯卧撐騰空

完成4組，從每組5次開始、隨身體能力提升逐漸遞增。

增強式訓練不僅只有下肢運動，上肢訓練也同樣簡單易懂。在常規俯卧撐的前提下，增加一個騰空的動作，這動作對於已經熟練掌握俯卧撐的朋友非常適合，因為你已經有足夠的爆發力將身體推離地面。你還可以在騰空中加入拍手動作，提升胸大肌內側的爆發肌力。

對於還無法完成自重俯卧撐的朋友，可以先從跪姿俯卧撐開始。

6. 俯卧撐葯球平移

使用工具：葯球

完成3組，每組8-10次

以葯球或厚一點的軟墊作為中點，在俯卧撐時將一隻手放置在目標物體上方，推起軀幹時，將軀幹推向葯球的另一側，同時另一隻手放置在物體上方，以此來回交替俯卧撐。

在完成這組動作時，個人感覺它與拍手俯卧撐最大的區別就是核心會得到顯著的刺激，當你將軀幹偏置至葯球的另一側時，位於球體上方的手臂穩定不足，需要同側核心來提高穩定性。因此這個動作也會很好的訓練到我們的腹斜肌及腹直肌。

文章總結

增強式訓練和其他訓練體系最大的區別就在沒有特定的訓練動作，只需要明白以下幾點，就可以隨時隨地開始你的冬季燃脂計劃：

  • 利用肌肉的力量將整個身體或部分身體離開地面
  • 肌肉對地面施加的力量越強，你在空中的預收縮時間越長
  • 預拉伸越長，你消耗的熱量就會越高
  • 合理安排好心率，訓練中盡量避免心率徹底平復。
