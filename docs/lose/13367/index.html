一個月時間讓自己重新變得積極起來

2019-12-11 916條評論 4,433次閱讀 597人點贊

剛在健身房看到一個頭髮花白的老奶奶也在游泳完換衣服。我在想，我像她這麼大年紀會不會還有錢來健身房游泳？

游泳完渾身舒服，主要是腦子更有創作力了，我還能寫個隨感，感覺游泳真的讓一個人會變得這麼積極，真好。

從健身房走出來，看到一個40多歲的男人在遛狗，我在想，等我結束北漂生活，我也回老家養只狗遛一下，這樣養只貓養只狗，自己會不覺得孤獨。

戰勝自己的懶惰，這是我活著的最大難題和障礙吧

我今晚在游泳前吃了一個巨大的玉米，一個大蘋果，四個紅糖餅乾，兩包青豆零食，一大杯豆漿燕麥。

遊了不到半個鐘頭的泳，估計這些熱量都消耗不完，身體又要產生脂肪肥肉。

每到晚上，我的雙腿就像灌了鉛一樣沉重。因為經絡都被厚厚的脂肪堵住了。

我才33歲就如此墮落，任由暴食殘害自己的身體。

還如何用身體的本錢繼續幹活掙錢呢

我的雙腿感覺被脂肪折磨的快廢掉了

堵住的感覺特別難受。

小腿和大腿都腫的不行。

這讓我要面對現實。現在對我最重要的是兩條腿的健康問題。

我想努力鍛煉兩條腿。讓我的兩條腿重新變成健康的筷子腿。

像我這樣的暴食症患者不得不面對和考慮自己的健康問題。
