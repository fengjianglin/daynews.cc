轉眼間2019居然要過完了，距離2020年只剩下20天了，驀然回首，今年年初寫下的2019年清單都沒做到……

就算如此，今碗君還是對未來充滿希望滴，連2019都熬過去了，還有啥熬不過？

但是…..

在即將到來的2020年，得先做好「承上啟下」的工作——搞定七大姑八大姨！

一月一日的元旦節，正是三姑六婆聚集的好時機，也意味著我們要接受他們的花式大拷問！不過，不用怕，今碗君有對付他們的招式，這就傳授給大家！

首先，要抓準時機，在他們剛進家門時，禮貌問好，隨後在迅雷不及掩耳之時，先發制人，對他們進行靈魂拷問！

用這些問題堵住他們的嘴之後，保持一個禮貌又不失尷尬的微笑，如若他們再要開口拷問，這時候就可以拿出你的重量級制敵武器，保證一招「致命」！

此時你只需要提出一箱喜慶鮮紅的6+1堅果、糕點、特產類大禮盒，瞬間就能將她們的注意引走！

那麼問題來了，各路商家都在八仙過海，各顯神通地搞事情，推出各種元旦伴手禮，精美的包裝，巧妙的搭配，實惠的價格，究竟該怎選呢？

今碗君這就為大家獻上各大品牌的節日禮盒，供大家參考！

故宮糕點禮盒

故宮作為流量大ip，不管和任何品牌聯名都會被炒的火熱，今年不甘落後，趁著元旦前夕，和稻香村聯合推出了福獸滾滾來糕點禮盒。

這禮盒從顏色、圖案到形式，一別從前的高貴典雅，皇家風範，而是走上了軟萌路線，主打福壽萌物主題，妥妥顏值派，少女吃貨黨估計根本捨不得送給三姑六婆，想自留。

一盒裡包含9款圓滾滾的糕點，三種口味，松子紅豆餡、百香果餡、椰蓉餡，口感如同烘焙的小饅頭，外實內軟，松中帶糯。而且還附送同一款賀卡，送禮什麼的簡直太合適啦~
