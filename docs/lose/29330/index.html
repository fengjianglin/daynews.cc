高爾夫球理論篇--高爾夫球為什麼從這個方向飛出去？

2019-12-20 1,320條評論 2,761次閱讀 685人點贊

球為什麼從這個方向飛出去? ( 桿面角度VS桿頭運動方向分析)

要理解球路，先談談初始方向。Trackman對 初始方向的定義是球離開桿面瞬間時球重心的水平運

動方向。影響初始方向有兩個主要因素，我們先來談談桿面角度和桿頭運動方向。

桿面角度和桿頭運動方向對球初始方向的影響:

我們現在已經知道，桿面角度對球初始方向的影響是桿頭運動方向的四倍。根據Trackman的數

據，鐵杆擊球時桿面角度對初始方向的影響在75%左右， 木杆擊球時桿面角度對初始方向的影響

在85%左右。因此鐵杆桿頭運動方向對球初始方向的影響在25%，木杆桿頭運動方向對球初始方

向的影響在15%。

因此，我們看一個例子。擊球時桿面角度-2度，桿頭運動方向+6度。由於桿面影響初始方向75%，初始方向的計算方式是:

桿面角度: ( -2*.75=-1.5) -所以桿面角度會讓球的初始方向是-1.5

桿頭運動方向: ( +6*.25=+1.5 ) –所以桿面角度會讓球的初始方向是+1.5

最終球的初始方向是這兩個影響因素的總和(-1.5+ +1.5)這結果是0，所以球的初始方向是正的。但是別忘了這個重要的事實，初始方向是正的不代表球不會彎曲。彎曲是由於旋轉軸的偏離而導致的，如果桿面角度和桿頭運動方向之間有角度差(像上圖一樣)， 旋轉軸就會有偏差。上面這個例子，球的初始方向是正的，向左彎曲。這需要在桿面中心擊球的條件下成立(後面會談到)。

桿面角度VS桿頭運動方向

桿面角度

現在我們知道了桿面角度對初始方向的影響最大，下面要討論其它影響桿面角度的因素。

擊球點:

Trackman對桿面角度的定義是:擊球時，當球被擠到最扁瞬間，桿面和球接觸的中心位置的水平指向。通俗點說就是擊球時球和桿面接觸的點的指向。對一-號木、球道木或鐵木杆，這點很重要，因為桿面不是平的，它有兩個弧面，稱為「橫向弧”和「縱向弧」。

「橫向弧”和「縱向弧」

擊球位置(續) :

由於橫向弧，擊球位置會直接影響桿面角度。擊球點靠近趾部時，會正向(向右)影響桿面角度，擊球點靠近跟部時，會負向(向左)影響桿面角度。對於半徑11英寸的橫向弧而言，向趾部偏離中心位置1/2英寸， 桿面角度會開放+2.6度。

桿面方向:

下一個影響桿面角度的因素是桿面方向。Trackman對桿面方向的定義是:擊球時桿面中心點在水平方向上的指向。

所以，一號木擊球時，如果擊球點在趾部對桿面角度的影響抵消了桿面方向，球的初始方向可能是正的(如下圖)。

球為什麼從這個方向飛出去? (桿頭運動方向輝桿方向)

桿頭運動方向:

我們先看看影響桿頭運動方向的因態，有以下三個 :

1、揮杆方向

2、擊球角度

3、縱向揮杆平面

揮杆方向

Trackman將揮杆方向定義為桿頭重心在擊球之前的水平運動方向。要理解揮杆方向會影響桿頭運

動方向不難。換句話說，如果我們揮杆方向是從內向外的，桿頭運動方向通常也會從內向外。反

之，如果揮杆方向是從外向內的，桿頭運動方向通常也會從外向內。

許多人認為揮杆方向是唯一決定桿頭運動方向的，不是這樣的還有其它因素在影響著桿頭運動方向。球為什麼從這個方向飛出去? (桿頭運動方向/擊球角度)

擊球角度

Trackman將擊球角度定義為桿面最大擠壓高爾夫球時桿頭重心在豎直面上的運動方向。這個的衡量比揮杆方向更難觀察。要了解為什麼擊球角度重要，先看下面兩張圖。這兩張圖表示了向下擊球，這是大部分鐵杆的擊球方式。注意，因為向下的擊球角度，球杆實際上是由內往外運動的。所以，我們可以說向下的擊球角度將會讓桿頭運動方向受到從內向外(正向)的影響。因此，如果你打鐵杆，向下擊打球，你必須要有一個從外向內(負向)的揮杆方向來平衡正向的擊球角度，形成一個直的起飛方向。

另外，補償向下的擊球角度所需要的從內向外或從外向內的程度會由於豎直揮杆平面而不同。下圖表示的是向上的擊球角度。換句話說，這就是向上擊球時的情況， 比如用一號木擊打架在梯上的球。注意球杆在向上的過程中也在從外向內運動。所以我們可以認為向上的擊球角度會讓桿頭運動方向產生從外向內(負向)的影響。因此，如果你打一號木，向上擊球，你就要有一個從內向外(正向)的揮杆方向來平衡擊球角度產生的負向的影響，打出直的初始方向。

豎直揮杆平面(VSP )

豎直揮杆平面定義為下桿過程中桿頭運動的傾斜平面。鐵杆VSP平均是60度，一號木VSP平均是

45度。

要理解VSP需要理解揮杆平面的幾何問題。因為一號木杆的平面通常是45度左右，你可以看到我

們會有一個等邊三角形( 上圖中粉色三角形) ，這表示兩邊是 相等的。所以我們可以得到的結論是，每一點擊球角度的變化會導致相同數量桿頭路徑的變化。如果-號木擊球角度是+3度，就需要將揮杆方向調整到+3度來打出正的初始方向。
