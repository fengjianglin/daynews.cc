魏大勛現在，嗯，帥。。。

之前是這樣的，胖胖的憨憨的，哈哈哈

18歲左右的魏大勛，體重一直在200斤以上，但兩件事他決定瘦下來。

一是知道愛美了，胖得把五官都擠變形了，再好看的皮囊也奈何不了一身肥肉。

二是魏大勛有一個藝術夢，想作為藝術工作者，他必須瘦下來。

說執行就執行，他給自己制定了嚴格的節食計劃和運動計劃，並很快實施。

在他眼裡，減肥這件事非常公平，只要付出了，就可以得到回報，控制飲食堅持運動，每天都不會白過！

魏大勛的轉變挺大的，只要你願意和他一樣努力，也可以變得和他一樣的迷人。（先從控制飲食開始）

減下來這麼帥，你有什麼理由不行呢？

大基數肥胖怎麼減？你必須知道這3點。

大基數肥胖的危害，首先，當然是影響穿衣，很多衣服買不到號，想想也是夠絕望。

其次就是影響壽命，據其資料顯示，如果按身高比計算的「理想體重」輕的話，可以活更久。

還會影響你的健康，太胖的話，對身體多器官及系統是會造成影響的，比如高血壓，高血脂，脂肪肝，關節病等中獎幾率翻倍。

關於飲食

首先要控制主食的量、限制糖和甜食，以及養成細嚼慢咽，吃七八分飽的習慣。

提高蛋白質的攝入，蛋白質是人體組織器官的重要成分，人在極端飢餓的情況下，會消耗身體的肌肉，為了保護身體的蛋白質不被消耗，日常飲食中要有足夠的蛋白質，長期吃素並不是一件好事。

關於運動

大基數前期建議集中精力調整飲食，身體負荷太大，對膝關節的磨損太大，而且損傷是不可逆的。

可參考以下運動建議：

能站就不坐，能走就不站。進行低損傷的運動，如快走，游泳等等。

要循序漸進，不要一上來就是十公里，貴在堅持。

關於堅持

大基數的人一開始減肥效果見效會慢一點，時間也會久一點，前期可以明確好自己的減肥目的，為了自己的健康和美麗，一定要堅持下去啊~
