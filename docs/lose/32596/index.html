跳繩能減肥嗎，這樣做事半功倍，你卻跳壞膝蓋

2019-12-24 1,737條評論 3,104次閱讀 861人點贊

跳繩是我們比較常見的運動之一，可能是減肥人群會選擇的運動中排名第二的運動，第一當然是跑步了。

但是很多人都有一個疑問，跳繩能減肥嗎？

能，當然能。

跳繩也是有氧運動，而且相比起其他有氧運動，比如跑步或者騎行，跳繩更加不受場地的限制，如果你的鄰居脾氣比較好，你甚至可以在家跳繩。

但是很多人在減肥成功之前就放棄跳繩了，多半是跳的膝蓋疼或者腳踝疼，更有甚者跳的腰痛頸椎疼。

那這到底是為什麼呢？

其實有氧運動我們可以簡單分成兩種，一種是跳躍類的，一種是非跳躍類的。

我們先看非跳躍類的

非跳躍類的有氧運動就類似騎行，快走等等，因為你的身體沒有一個持續騰空的過程。

非跳躍類的有氧運動優點就是對下半身的衝擊比較小，但並不是沒有傷害，只是傷害的形式從衝擊轉化成了磨損。

比如單車，很多人就把自己的膝蓋騎廢了。

我們再來看跳躍類的

有了上面的例子，跳躍類的運動就比較好理解了，它的運動表現就是身體會持續騰空，像是跳繩或者跑步，身體都是有一瞬間是騰空的，而且這個過程是持續的。

這種運動的優勢很明顯，就是運動強度更大，也就是能造成更多的熱量消耗，而且跳躍類動作能鍛煉到的肌肉更多，不僅能減肥，也能更全面的鍛煉身體其他部分的肌肉。

但缺點自然是更加明顯。因為是跳躍類動作，所以當我們身體落地的那一瞬間，勢必會對下半身造成衝擊，而這種衝擊，必然伴隨著傷害。

我們具體來看跳繩這個動作

跳繩就是典型的跳躍類有氧運動。它的跳躍程度比跑步還要密集很多。

當我們在落地的那一瞬間，首先接受衝擊的是腳踝。

腳踝是身體上很憋屈的關節，他很精細，同時也很脆弱，但是卻要承擔來自全身的重量。

如果你的腳踝很弱，或者你的體重很大，那麼在跳躍類的運動中你的腳踝就很容易受傷。

而跳繩是一個跳躍非常密集的動作，在長時間持續跳躍的過程中，你那脆弱的腳踝可是一次次的在承擔你那遙不可及的慾望。

不過腳踝雖然脆弱，但並不傻。他懂得將傷害轉移。

我們身體是一個整體，當你做一個動作需要使用多個關節的時候，你比較弱的關節就會更傾向保護自己，讓比自己強的關節承受更大的壓力。

那麼離腳踝最近的又比它稍微強一點的關節是什麼呢？

當然是膝蓋了。

膝蓋是既強大又脆弱的關節。

強大是因為他周圍的肌肉是身體上最強壯的肌肉——腿部肌群。

脆弱是因為圍繞膝蓋的基本都是韌帶，一旦損傷，很可能就是一輩子。

當我們進行持續跳躍類動作的時候，因為你的腳踝太弱，不能有效的吸收跳躍帶來的衝擊，他的屈伸就程度就會減少。

這點在跑步上尤為明顯，很多人跑步喜歡跺腳，就是跑步時腳踩地的聲音格外大的那種。

這就是腳踝在保護自己，而這種行為就是將傷害往膝蓋上轉移。

現在明白為什麼你跳繩或者跑步膝蓋疼了吧。

那怎樣解決這個問題呢？

其實經過上面的討論，我們已經差不過能得出結論了。

既然衝擊首先是來自於腳踝，而腳踝的強弱又影響著膝關節能否接受更強的衝擊，那麼我們解決問題的方法不就有了嗎？

加強腳踝

我們能想到的和跳繩類似但是又不至於傷害到腳踝的動作就是踮腳尖（提踵），如下圖。

是不是覺得很簡單，然而真的並不複雜，有些事情就是因為太簡單所以我們才想不到，也是因為太簡單而不願意去做，反而耽誤了砍柴工。

在你開始跳繩之前，先給自己的身體一個加強的機會，每天做幾組提踵，至於做幾組，每組做幾個根據自己的身體情況來決定就行。

如果這種提踵滿足不了你，還可以嘗試下圖中的提踵。

選一雙比較不錯的鞋子

這其實也是很關鍵的一點。

你可能看很多拳擊運動員或者其他訓練者穿很薄或者不穿鞋子跳繩，但請認清自己的現狀，你不是運動員，你之所以想跳繩是因為胖，你胖是因為你很久沒運動了，你有他們的身體素質嗎？

沒有的話就請選一雙比較好的鞋子吧。

我的建議是底厚皮薄。類似下圖中的鞋子。

鞋底厚可以起到緩衝作用，鞋面薄不會妨礙腳上的動作，而且這種鞋子很輕，跳起繩來腳上不累贅。

運動完還要注意拉伸

簡單介紹兩個跳繩結束之後的拉升動作，可以更好的保護我們的腳踝和膝蓋。

首先是腳踝的拉伸（小腿腓腸肌），如下圖。

然後是拉伸膝蓋（股四頭肌），如下圖。

以上兩個拉伸動作訓練完隨便找個台階或者牆面都能完成，操作簡單且效果很好，千萬不要再去壓腿了，沒什麼用的。

對於跳繩訓練的總結和建議

如果你能選擇的運動就只有跳繩或者你就是比較喜歡跳繩這項運動，那麼請你最後聽一下我的建議。

首先是體重基礎比較大的人。

放棄吧。真的，跳繩不適合你，任何跳躍類的動作都不適合你。你不如去快走或者騎騎單車。

對於體重基數不是很大的人。

我建議你先從提踵開始訓練，先練個一兩個星期，再開始跳繩訓練，剛開始跳繩的持續時間也不要太長，可以間歇性的訓練，比如跳5分鐘休息2分鐘這樣。

等腳踝的強度上去了，再慢慢的加長訓練時間。

跳繩的形式最好是雙腳交替跳，如下圖。

這種形式比你雙腳跳對腳踝的衝擊力其實更小，而且這樣跳更省力，跳的時間也更長。等你交替跳繩練的不錯的時候，再開始長時間的雙腳跳。

總之跳繩這項運動還是很不錯的，活動空間比較小，基本哪裡都能跳，前提是沒人趕你。

只要我們能正確的進行，就一定會取得比較不錯的效果。
