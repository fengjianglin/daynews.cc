肩寬腰細是很多人心目中的理想身材。在這種身材比例之下，人體會呈現出倒三角形，充滿力量感和美感。Seong健身5年，完美呈現了倒三角體形，扔垃圾都很帥氣。肩寬腰細其實跟人的天生骨架有一定關係，有些人天生髖部骨架小，或者肩部骨架比較寬，那就很容易呈現倒三角。

當然，想要呈現完美的倒三角體形，離不開後天的訓練。Seong覺得自己的肩部骨架其實不算寬，只是平時健身特別注重肩部肌肉的塑造，才有現在的體形。很多肌肉大咖，青少年時期十分瘦弱，通過後天努力訓練才變成了肌肉大塊頭，人體的塑造能力還是很強的。

擁有倒三角體形，彰顯男性魅力。Seong左右手指插上3根管子，就可以詮釋金剛狼的力量與霸氣。問題是如何打造倒三角呢？倒三角其實就是肩寬腰細，所以，我們要加強肩部鍛煉，並且降低體脂率，讓腰部贅肉減少，最後能夠練出腹肌。當然，背部訓練也是十分必要，這會讓倒三角更加立體協調。

Seong比較喜歡的肩部訓練動作是推舉：

動作要領：

雙腳與髖同寬，腳尖指向前方或微微沖外

上身保持發力，把杠鈴置於鎖骨高度

肩胛收緊，雙臂夾緊，膝髖微屈發力將杠鈴推起

慢慢放下杠鈴，重複動作

除此之外，想要肩部寬厚，建議大家還要多練練側平舉。

動作要領：

在進行練習時，肘部不要伸直，自然站立，肘部略曲

用肩部的力量帶動上臂直至前臂向上提起啞鈴

想達到最好的效果，一定要把訓練的速度控制好

啞鈴在下放的過程肩部肌肉一樣是用力的

針對腹部的訓練其實要降低體脂率，然後再做一些卷腹動作，卷腹動作，Seong比較推薦懸垂抬腿，強度比較大，但是對整體塑形效果更好。

動作要領：

保持雙手握緊單杠，使掌心朝向前方，軀幹伸直

用力收縮腹肌帶動雙腿抬高到水平位置，然後再慢慢放下

使身體回到懸垂位置後，再開始下一次動作

想要倒三角身形更加協調立體，建議在練肩和練腹的基礎上，增加練背動作。練背的動作比如引體向上、高位下拉等，練習過程中記得沉肩，找到背部發力感。
