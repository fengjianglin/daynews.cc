停止炫耀你虛假的卧推力量吧，回歸真實的訓練

2019-12-24 1,083條評論 4,541次閱讀 1397人點贊

我並不推崇在卧推中過度拱起背部。這可能會引起力量舉選手的反感，但我依舊堅持這個看法。
首先，我要強調一下幅度較小的背部拱起是肯定可以接受的，也就是說要挺胸同時下沉收緊肩胛骨，並且保持骨盆前傾的狀態。這種姿勢可以讓你的動作更穩定。

我不推崇的起橋，是故意將脊柱凸成誇張的下腰式，以縮短肩胛骨和臀部之間的距離。

無厘頭的邏輯

起橋式卧推的誕生源於力量舉比賽中一個規則運用：在舉起重量的過程中，上背部和臀部必須時刻貼緊凳子。

這個規則並沒有特別說明下背部要接觸板凳，這是是因為力量舉運動員的臀部肌肉通常比普通人更發達，受限於腰椎本身生理曲度和發達的臀大肌使得他們的下背部無法貼緊板凳，除非他們抬起雙腳。

所以，起橋是不違反規則的，進而非常誇張的起橋也並不違反規則。運動員們爭強好勝的心理促使這種卧推技巧的誕生。

如果一個人正在參加力量舉比賽，並且目標是突破自己的最好成績，那麼他可能必須要使用起橋技巧。但他不應該在日常訓練中也一直用這種方式。

如果你又僅僅是一名普通健身愛好者，只想讓自己看起來更大更壯，就更沒必要在卧推時把脊柱凸成聖路易斯拱門那樣誇張了。

受傷風險

首先，這個姿勢會讓下背部承擔更多壓力。當脊柱處於過度超伸狀態時，椎間盤和椎孔內部神經會被壓迫，增加受傷風險。

通常情況下，卧推時脊柱所承受的壓力並沒有深蹲硬拉那麼大，但也並不意味著壓力可忽略不計。如果在卧推時還夾帶了腿部驅動技術，脊柱還需要傳導來自地面的力量，那麼脊柱受到的壓力就不言而喻了。

不相信我說的嗎？你可以讓背部受傷的人在卧推時拱起背部，他們的背痛立馬就會發作。

其次，有些人可能會非常專註於這種起橋卧推，且在一周內還同時練習深蹲和硬拉，這導致脊柱在每次訓練課都要承擔大量壓力。下背部肌肉恢復很慢，因此更理想的情況是卧推日不怎麼施壓脊柱，讓下背部充分恢復。

削弱肌肥大效果

拱起背部意味著會縮短動作行程範圍，這樣可以使動作變的更容易並且能舉起更大重量。但會導致肌肉無法做到全程收縮，因此肌肉維度發展會受限。

我們都知道肌肉收縮幅度越大，受到的刺激就越充分。就好比推舉無法像卧推那樣充分刺激胸肌，反之卧推無法像推舉一樣充分刺激三角肌一樣，同理起橋式卧推也無法讓胸肌變的更高效率變飽滿。

局部幅度的動作可以作為輔助項，但不能成為訓練主項，否則你會限制肌肉生長能力。這一點對所有訓練者來說都很重要，尤其是初學者和初級運動員。

不可否認的是，從訓練狀態調整到比賽狀態挺費勁的。假設你用正常姿勢卧推150磅，而用最大程度起橋卧推是185磅的話，我個人還是建議使用正常姿勢進行訓練。因為從理論上來講，正常卧推重量力量越大，起橋卧推力量也越大；反之則不然。

高起橋卧推也許讓你的力量很快從150磅漲到250磅；但這樣就很容易過早進入瓶頸期，並且很長時間都無法突破。

起橋時最強，其餘全是弱項

杠鈴複合動作訓練最大的一個好處就是它可以調動大部分肌群，而不是只針對某一塊局部肌肉。比如深蹲對強化下肢力量比其他動作都要來的有效，同理卧推也可以高效的強化上肢力量並帶來其它上肢動作的轉移效果。

但如果你在卧推的時候必須把背拱的跟練柔術的人一樣誇張，那麼你在其它項目上比起你的我卧推水平可能就是個渣渣，因為這個技巧沒有什麼力量轉移效果。

這就是為什麼Charles Poliquin教練推薦使用較窄的握距和適當的重量來進行卧推訓練。窄握距卧推更不容易作弊，對於估算卧推真實力量也更準確，而超寬距卧推和起橋式卧推都有可能會讓你誤判實際卧推重量。

有些人認為這是「偽」力量，確實是的。雖說它在力量舉比賽中不算作弊，但也不代表它就是真理。

腿部驅動作用同樣被高估了！

我並沒有說腿部驅動沒有用，只是說它的作用被高估了而已。

對於大多數力量舉運動員來說，使用腿部驅動確實可以讓卧推成績增加10到20磅，但並不像有些人說的那樣可以從330磅卧推增加到400磅。

殘奧會運動員Siamand Rahman用實際成績證明了這一點，最近雙腿殘疾的他卧推重量達到了705磅，這個成績僅僅比身體健全訓練有素的運動員的世界紀錄低了10磅而已。

個人建議

打好紮實的卧推基礎，不要輕易模仿比賽，盲目追求起高橋的技巧。請一定使用正常的姿勢進行卧推訓練，只有在極限試舉時才可以考慮起高橋，發揮超出日常的實力。
