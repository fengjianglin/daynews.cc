健身其實是個【大坑】 你被埋到哪一層了？

天天要聞
2019-12-11

聽說過【健身是條不歸路​】這句話嗎？

這話不是我說的

誰說的 你出來 我請你哈啤酒

​現如今【健身】這坑裡的人越來越多了

雖然健身房跑路的也越來越多 健身教練套路越來越多 但這些依然無法阻止人類對於健身的熱愛

下至十幾歲春潮湧動急於尋找配偶的少年 上至堅持能不與上帝見個面吃飯聊天的大爺大媽 都是殺氣騰騰的湧入健身洪流中

簡單歸納來講 這裡其實無非兩種類型

一種是能堅持下來的 一種是中途退場的

那部分撤退的人基本上是基於各種理由瞧破了健身這種毫無用處又浪費時間精力的【愚蠢行為】 而回歸到啤酒擼串、歌舞昇平的烏托邦生活里 也算是一種勝利逃離吧

另一種堅持下來了的人群那可就慘了

​這類人群不知不覺的被健身這個【大坑】慢慢的活埋了

剛剛埋到小腿的那類人 覺得健身不單只是一種運動 而是一種時尚健康陽光的生活朝向 他們大多健身履歷一年以上 基本上都是工作之餘下班時間去健身 此類人群大多會買健身教練的課程 極其鍾愛緊身衣 名牌運動裝備+入耳式藍牙耳機為必備 健身過程那是必須圖片+視頻+朋友圈 對於他們來說 這是一種生活

被埋過膝蓋的那些人 健身其實已如吃飯睡覺一般的自然了 儼然成為了一種習慣

什麼？假期？我沒有假期 因為我得去健身 什麼？聚餐？我真不去了我怕油鹽糖影響體型 啥？夜店？不行啊 晚睡會掉肌肉的 這些人基本上健身履歷超過3年以上 有自己的健身套路和風格 身邊隨時必須必的拎個碩大的提包 裡邊各種補劑啥的 練前練後會在更衣室一通亂兌 他們一般著裝寬鬆 喜歡鎂粉不喜歡手套 連帽無袖衛衣+套頭耳機 組間隙愛對著鏡子擺造型 而且熱愛蹲腿 此類人群目前應該是所有健身房的中流砥柱了

​那些已被埋到腰了的人 健身對他們來講根本已經不是行為了 而是一種信仰

十年以上健身的經歷 那必須必的健身房扛把子了 大肩大二頭大胸 … … 這些人基本上都是行走的健身工具書 理論一套一套的並很有說服力 因為那一身腱子肉在那擺著了 專治各種不服啊 有的雖然不在行業里但健身圈的人幾乎都認識 那幾個50公斤的啞鈴基本上也就這種人能用健身已經融入了他們的血液里 想停下來那是根本不可能 沒什麼能阻擋了 不過 其實大多數這樣的人已不再瘋狂的擼鐵了 看起來反倒是比較佛系的 他們去健身房除了擼鐵也是交友聊天的好地方

再往上快埋到胸脯子的 絕對是健身圈裡精英了 這批人雖然數量相對不多 但都是把健身根植入骨髓的人 低體脂大維度 拉絲+爆筋。… … 也懂得怎樣做C 時不時的會參加比賽和收收徒弟啥的 跟職業選手相比雖不是唯一收入來源 但也主要是吃飯本事之一了 有的甚至已經做起了健身房老闆或是補劑代理商 這些人被埋得很深很深 不會再有跳出來的可能了

那麼被埋的最深的人是呢？

​被埋最深的那群人 其實為數眾多

啥？你不信？

這些人基本上健身不會低於5年

或胖 或瘦

並且最重要的是看起來可能【毫無訓練痕迹】

呵呵 沒錯 很明顯 我就是那種人

這些人從沒考慮過為什麼要堅持不懈 因為他們認為健身如飲水一般的自然

不刻意追求驚爆眼球的肌肉身材

不追求歇斯底里的運動體驗

更不會為健身而影響正常的生活工作

對於他們 健身是眾多愛好中最好的選擇

因為 健身比抽煙喝酒健康

比泡吧泡妞省錢

比唱歌跳舞時尚

比肥宅陽光… …

​

一種愛好 不可以升華到虔誠的地步 那樣會變質 除非你想把它變成職業

能夠帶給你快樂 令你身心愉悅的才是真正好的愛好 而不是讓它成為負擔

令一個人可以走的更遠的 唯有自律和積極
