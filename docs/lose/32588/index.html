力量、速度、技術——關於業餘高水平雙打三個要素的純理論剖析

2019-12-24 3,405條評論 4,868次閱讀 1977人點贊

業餘高水平雙打對決中決定性的影響因素就是以上的三個，暫時不考慮配合以及性格、心理等成分，我只在這裡從相對簡化的條件下開展一下純理論的探討。
首先我們要明確，任何的體育項目必然是身體素質和專項技術的結合，只不過兩者的比例會有多多少少的差異，跑步、舉重等項目在身體素質這個端點，乒乓球、體操等處於專項技術的這一端，其它的運動項目基本處於兩者的漸近線上。當然，任何運動項目想要取得比較好的成績，相對較高的身體素質都是一個必要的條件。現實中不存在劍宗和氣宗的對立，都是兩者的有機結合。
其次，在業餘羽毛球雙打這個項目中，達到高水平必然是題目中三個要素的結合，任何一方面有明顯的短板都不可能匹配「業餘高手」這個稱號。同時，我們又要清楚，正因為只是業餘出於興趣花錢來投入羽毛球這個運動，沒有專業的那種時間和精力以及天賦，我們必然又不可能在三個因素上都達到比較高的層次，一般的所謂高手，都只能在其中一兩項上有比較優勢，其它大概在中高水平這個檔位（能在三個要素都達到高水平，基本就接近專業層次了，也超出了我所能討論的範圍）。
好吧，在這個前提下，簡單說下上述要素在雙打對決中的意義：
力量：首先我們想到的是殺球，這是力量在進攻中的直接體現，其實就上肢力量而言，強大的力量不僅給你提供了迅猛快速的殺球，還會提高你平抽擋的出球速度以及接殺的抽推挑的威脅，從攻守兩個方面極大增強你的威懾力。另外，核心和腿部的力量素質，對你場上的移動以及身體協調、傷病的預防都具備很好的效果。如果能進行比較系統的力量訓練，將會極大提升你對抗的水平。
速度：可以簡單分為移動速度、反應速度、出手速度，它和力量因素緊密相關，其中體重對移動速度影響極大，反應的話和神經、大腦的先天條件關係比較大，一般人很難有太大的提升。出手速度一方面和力量有關，另外和準備習慣、搶點意識、跑位、預判等都有比較大的關係。總結起來，速度的制約條件包括先天神經反應速度、力量、體重、技術和意識，大家可以對號入座分析自己的不足，針對性的練習提高。
技術：不去細說單項技術，在這裡我要強調一下穩定性、變化性以及硬朗的技術風格。前兩個就不多說了，多拍相持失誤少，技術變化多是基本的取勝要素。風格硬朗是我個人的一點體會。在雙打中，速度節奏快，得分主要在中前場，一旦打出了相對比較慢速疲軟的中前場球，就很可能被對方搶住前場下壓，連續進攻最終造成己方的失分。因此，被動的時候以快推、抽球、推挑等快速過中前場的技術為主，後場高遠和放網作為最後選擇，讓對方無法輕易在中前場發力下壓，這個技術風格就稱為「硬朗」。出球慢、打落點、放網多，這是很多單打高手在打雙打的時候很容易犯的錯誤，雙打場上需要改變這種相對慢節奏的疲軟風格，才能夠獲取更高的勝率。
基本思考就這麼些了，歡迎討論。來源中羽：球齡七年半
