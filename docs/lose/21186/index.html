12.18復盤：明天繼續做高位接力嗎？不如關注這個方向

2019-12-18 647條評論 2,738次閱讀 178人點贊

閑聊

一，健身與炒股

收盤之後習慣去健身房，一天有氧，一天抗阻力訓練，循環往複。

有氧的目的是清空大腦，無論是游泳，還是橢圓機，30分鐘都可以快速放鬆神經。

而抗阻力訓練，俗稱「擼鐵」，目的是為了保持肌肉，以及身體代謝。

健身過程中，教練會指導相關動作的要點：

例如發力點在哪個部位，重心的具體位置等等。

發力點找正確，才能把一個動作做好。

健身如此，炒股亦如此。

先找到一種方法，再簡單重複，就可以。

二，市場如何調整？

市場大漲，感覺並沒有太多復盤的必要，大概的思路是明天會分歧一天，然後繼續干。

今天是高位分歧第一天，明天是第二天。

因為預判明天高位繼續分歧，那麼今天我板上把萬通智控砸完了，只留下今天開盤低吸的籌碼。

如果今天是分歧最後一天，我會無腦持有龍頭。

可惜今天不是。

也許賣飛，但今天的買點，屬於前面沒有上車的朋友。

而我，這波在他身上賺了接近30點，我要把注意力放到低位去。

明天我會繼續耐心等高位股全部分歧，然後買最後一個漲停打開的。

機會板塊

一，金融

金融今天本應該分歧，結果超強勢了，核心股是綠庭投資，南京證券，蘇州銀行。

都是次新股！

回顧2-3月份的小牛市，金融股大漲的其實也是那幾個次新：紫金銀行，中國人保，中信建投。

南京證券無疑是這一波的中軍，如果我是裡面的大資金，我一定會去頂小盤的綠庭投資，給南京證券做預期。

這就是綠庭投資漲停的意義。

哪怕南京證券滯漲，這種做預期的標的也不容易輕易結束。

昨天開盤打板的籌碼，今天鎖倉沒動。明天如果符合預期，我還會加倉。

二，連板股

1）低吸分歧：

明天高位分化，我會考慮去做核心股的分歧。

這裡的核心股，是指南寧百貨，萬通智控這些嗎？

並不是。

我指的高位，是綠庭投資、耐威科技、聚飛光電、甚至磕磕絆絆四板的星期六。

這些才是與這波行情共振產生的個股，金融、科技、傳媒，這三個炒作方向。

至於萬通智控、南寧百貨、漫步者，這些高位股已經是純籌碼博弈，下車之後，就不容易輕易再上車。

2）低位股：

今天市場分歧第一天，那麼今天的首板股，其實也值得一看。

這裡其實包含幾個方向：

妖股：寶鼎科技；

科技：共達電聲；

傳媒：愛施德；

次新：斯迪克。

以上這些機會，可以無腦做換手板。

3）次新股：昨晚復盤重點提了次新股，今天就開始蠢蠢欲動。

下午萬通智控炸板後，次新股的嘉美包裝就想卡位，可惜萬通智控成功回封，次新的嘗試沒有成功。

但從量能上來看，嘉美包裝裡面的籌碼還在鎖倉，那麼還有預期。

從龍虎榜可以看出，今天洗盤非常充分，昨天的買1到買4全部離場。

這是一個非常不錯的龍虎榜，結合量能之後，明天可以考慮做弱轉強的。
