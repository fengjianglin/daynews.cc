粘滯點剋星 | 如何用高位推舉讓你的訓練動作絲般柔順？

2019-12-12 3,178條評論 4,768次閱讀 1788人點贊

面對杠鈴，全新的重量挑戰，你雙手握住，看著負重緩緩靠近自己，你的心裡一定暗下決心：老子今天一定要把它推起來！

起初一切都很順利，重量緩緩抬升，似乎此刻就是突破自我的瞬間；然而一瞬間肌肉彷彿突然僵硬，無論再怎麼竭盡全力，負重就是紋絲不動的停在半空中，即使勉強推起，不免心有餘悸…

卡殼的卧推

又是那該死的粘滯點！行程走了一半總是「卡殼」或直接「熄火」。如何能讓推舉動作絲滑順暢？今天我們就來聊一聊克服粘滯點經典技巧——高位推舉

關於粘滯點的那些事

粘滯點：

在上肢訓練中，我們的粘滯點大多發生在推/舉動作中，從視覺上往往是一段完成運動行程的中點，根據個人情況，粘滯點的實際位置會更靠近向心收縮的前半程。當我們在粘滯點靜止時，我們可以理解為粘滯點是一種運動過程中被動出現的等長收縮。那運動中的向心與離心階段是如何造成粘滯點的呢？

離心階段就是指肌肉承受負重後被拉長遠離軀幹的過程，例如在卧推中的下降過程中胸肌就處於離心收縮

向心階段就是指肌肉抵抗負重將肌纖維拉回軀幹的過程，例如卧推中我們推起負重的時候，胸肌纖維不斷縮短的過程就是向心收縮

那粘滯點和這兩個階段有什麼關係？簡單來說就是這兩個階段產生的力是不均等的。（離心＞向心）

離心階段產生的肌張力大約是向心收縮的1.3倍。還是以卧推為例，負重下降的過程時你的肌張力會比推起時大的多。當你選擇了離心階段可以承受的重量時，往往沒有考慮向心階段能否承受住這個負荷。

因此粘滯點的位置往往處於你完成離心後，通過肌張力把肌纖維恢復到目標肌群的初始長度的那個點，因為過了這個點，你就需要進一步縮短你的肌纖維了。

高位推舉的訓練計劃

高位推舉的目的是強化我們向心階段，讓我們克服推舉動作中的粘滯點問題。高位推舉通過在粘滯點附近設置障礙（提高杠架的水平高度），讓我們只完成粘滯點出現後的向心階段訓練。

高位推舉強化的肌肉部位

  • 肱三頭肌：主要負責將負重更好的反饋到目標肌群上（胸、肩）
  • 胸肌：卧推訓練的目標肌群，強化上、中、下胸的受力均衡
  • 肩部：主要針對三角肌前束及中束前側

如何完成一次成功的高位推舉

我們在進行高位的推與舉動作時，建議將握距設置的比常規推舉窄一些，讓肱三頭更好的感受向心階段的刺激。

1. 提前一周，完成全程卧推及杠鈴肩推，記錄下自己的粘滯點位置，有條件的可以拍攝視頻記錄下來。

2. 根據第一步獲取的粘滯點位置，設置杠架的水平高度，不要過低或過高

3. 使用空杠感受半程運動，全程保持肌肉緊繃，緩慢下降負重

4. 一旦杠體接觸到杠架，盡全力迅速將負重推起，再緩慢落下，重複。

組數及次數安排

對於健身人群來說，我們應該使用最大負重1RM的70%-85%來進行高位推舉，因為這個區間也是我們日常進行正式組的負重。

完成3-6組，每組8-10次

還有一種方式是針對側重力量舉的，你可以選擇自己1RM的90%甚至95%進行，你可能會很快力竭，但請繼續執行這個負重的高位推舉，力竭階段即使無法形成負重的位移，也可以保持靜態張力，盡全力向負重施加力量。

完成5組，每組3-5次

力竭時，保持靜態姿勢維持8-10秒

高位推舉的訓練計劃案例及好處

很多人會進行半程強化訓練，但都是作為正常訓練後進行的1個補充，但這樣做是沒有收益的，你的目標是解決正常組甚至衝擊新重量時的粘滯點，因此你可以單獨安排一天進行高位推舉

因為高位推舉整體的耗時不長且相對安全，我一般會選擇在練腿日時進行這個訓練。

訓練計劃案例1

選擇卧推及肩推的1RM的90%

1A. 高位卧推3-4組，每組3-5次

1B. 高位站姿肩推3-4組，每組3-5次

*力竭後，保持推/舉動作靜態發力，維持8-10次，重複2-3次

訓練計劃案例2

選擇卧推及肩推的1RM的70%-85%

1A. 高位卧推3-4組，每組8-10次

1B. 高位站姿肩推3-4組，每組8-10次

*一旦日常正式組的重量無法完成8次時，不要為了迎合次數而去減少負重。

高位推舉的好處

1. 縮短的運動形成可以提高肱三頭的力量，同時也是突破粘滯點的關鍵之一

2. 向心階段的強化可以縮短你在離心階段的時間，其結果就是減少延遲性肌肉酸痛DOMs

3. 高位推舉減少了你處理負重的信息量，讓你更加集中在自己的粘滯點問題

4. 不僅會解決你正式組的粘滯點問題，還可以順便提高你的1RM重量

文章總結

許多人忽視甚至輕視半程動作，認為那是不完整的訓練動作，是對大重量的盲目追求。其實當主動去使用半程訓練，而不是被重量壓迫得無法全程時，你的向心階段才能真正得到強化。希望今天的內容可以幫助到大家，喜歡的話可以關注一下小弟，我們明天見！
