練瑜伽真的會影響你的身心健康嗎

2019-12-22 742條評論 2,557次閱讀 654人點贊

這幾年，周圍練瑜伽的人越來越多，女人愛美，為了塑造麯線，為了健康，選擇去練瑜伽，其實很多人也知道瑜伽只是當時阿三為了得到更好的性愛刺激而發明的一種鍛煉方式，而且現在的瑜伽廣告寫的寫很誘人：

今年將會是，你的瑜伽開啟之年，這回我可不是瞎說，你將要和3600多萬已經投入到這項美妙運動中的人一起，用心靈去感應古老的傳統，肅清體內令人討厭的毒素……

當然也可能不會，在你拿瑜伽墊之前，聽我講講瑜伽究竟如何影響你的身心。首先，瑜伽有很多很多種，但總的來說，它包括一些拉伸和冥想，無論你以前讀到些什麼，尚沒有科學證據證明，瑜伽會排出大腸里的毒素，或是任何部位的排毒，但這並不是說瑜伽對身體沒有其它幫助。

拿背疼來說吧，據估計有80%的人，一生當中都會有背痛的時候，但一項研究發現，6到12節瑜伽課之後，練瑜伽的人會感覺到腰部疼痛大幅減輕，這是因為某些瑜伽動作拉伸了腿後腱，當這些肌腱緊張時，會拉緊髖部屈肌，讓腰部緊張，但是隨著練瑜伽，腿後腱會越來越柔韌，比如，一項2015年的研究，以練習名為「哈他」的瑜伽女性為樣本，它的有些動作比較接近下犬式和三角式，她們每周練90分鐘，連續練16個星期，到最後一星期，她們向前觸碰腳趾的距離比之前縮短了4厘米，這就是腿後腱放鬆的緣故。

如果在上瑜伽課時還同時做冥想，柔韌度或許就不是唯一的好處了，冥想7到16個星期後，一項研究發現參與冥想的人血液里，C-反應蛋白大幅降低，這些蛋白質與炎症相關，當你過度緊張時，炎症會加倍活躍，時間久了，這些炎症會造成嚴重疾病，比如心血管疾病和癌症，瑜伽中的冥想有助於緩解這一問題。

研究人員猜測，瑜伽減輕壓力的方式，是通過干預中樞神經系統，以此釋放減輕壓力的荷爾蒙，此外一些研究顯示，以冥想為主的瑜伽，如瑜伽體位法，能提高愉悅感荷爾蒙的水平，比如大腦中的催產素，另外，瑜伽是一種鍛煉，而鍛煉本身就是緩解壓力的方式，美國衛生與公眾服務部，建議每周做兩個半小時到五個小時的，輕度到中度鍛煉，其中可以包括瑜伽，快走，游泳，說實話，任何強度的有規律的鍛煉，都可以減輕焦慮，提升情緒，促進睡眠，提升自尊，所以，儘管瑜伽在提升柔韌性和正念方面有優勢，但是還有許多其它活動可以讓你健康起來，但只要它能讓你動起來，或許還能讓你交些新朋友，何不試試？
