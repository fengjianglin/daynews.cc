封網，雙打中一個重要技術手段，既可以主動得分，又可以創造機會

2019-12-24 998條評論 4,059次閱讀 779人點贊

羽毛球封網是是一種戰術意識，是指在後場隊友大力扣殺或者以其他方式進攻的時候，另一名隊友要在網前封住對手可能的回球路線的一種打法，封網尤其在雙打中最常見，單打中較為少見，封網，在雙打項目中是一個重要的技術手段，它既可以主動得分，又可以為隊友創造機會，但業餘選手有一個很普遍的現象就是：網前封網為什麼往往很容易把球打下網？又要如何改變呢？

網前封網總是下網的原因，網前封網下網的原因有許多，例如舉拍準備太慢，仰拍不夠或慢了，擊球點靠後或太低，揮拍速度太慢，用大臂擊球等等，各位球友可以對照以上原因，根據自己的技術動作找出問題所在。

1、速度太慢：速度太慢或舉拍太慢會造成擊球點太靠後，所以來不及發力；仰拍不夠或者慢了也會造成擊球點過低，或來不及做閃腕的動作；揮拍速度太慢，造成期望的擊球點和實際擊球點不相符，這是最容易造成封網下網的原因。

2、未用手腕發力：很多業餘球友在做網前的技術動作時沒有養成用手腕發力的習慣，接球時往往下意識地用大臂發力，這是造成也封網下網和失誤的重要原因，平時應要有意識的用手腕發力。

網前封網的技巧：

1、握拍方式：網前封網的時候，握拍可以調整到球拍手柄前端，握拍要靠前，因為更短的距離可以使揮拍速度大大提升，可以自己握前端和後端自己試著練習一下感覺。

2、擊球方式：封前下網最直觀的問題就是擊球的時候手腕下壓太過，打的太狠，很多業餘選手把網前封網等同於網前殺球了，手臂揮動幅度太大，往往容易下網或底線出界。

正確做法：來球後，小臂迅速後倒，手肘擺動，小臂、手腕自然向前揮動，當球來到拍前時，我們需要用手指和手腕的力量來進行下壓，封網並不需要太多的腰腹力量，也不需要肩膀的力量，封網時一定要控制好球的力度，不能太凶了。

3、注意動作的連貫性：有些球友認為網前封網應該一次封網置他人於死地，其實正確的做法是：手腕手指發力封網的時候，要做好下一拍連續封的準備，不要想一拍把對方封死，在保證自己出球封網下壓後連續跟進封網，多拍封網取勝，注意動作的連貫性。

4、注意來球的節奏：封網的時候要注意來球的節奏，因為球過來的速度、力量、弧線都是不一樣的，如果來球比較慢，弧線比較高，那封網時就可以主動加力；如果來球比較快，弧線比較平，那封網時就要注意控制，不要太急於發力，無論如何，封網都要盡量搶高點、前點。
