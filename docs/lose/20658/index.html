傳統武術的盛世亂象，我們一天能花幾個小時去練拳練武？

2019-12-17 494條評論 2,123次閱讀 849人點贊

傳統武術有一個盛世亂象，我們的傳承人們，一天能花幾個小時去練拳練武？誰又會去下功夫練體能，孫祿堂（中國近代著名武術家）的兒子做了老師傅教徒弟，都是讓徒弟先跑三年步，然後再教拳。通過跑步練習體能，因為奔跑是一切武學的基礎

武藝中有很多誤傳，過去很多人說練習武術要先扎馬步，其實扎馬步和格鬥沒有任何關係，我們可以看看泰拳這種世界上最好的站立格鬥，它不扎馬步、不蹲杠鈴，但他們每天跑十公里，跳半個小時，這樣可以鍛煉肺活量、肌肉耐力、彈性和收縮力度。這就是我們傳統武術中缺乏的，可以說，我們目前的傳統武術是精華與糟粕並存的。甚至也可以說三分之二是糟粕的。古人其實非常強調實戰的，戚繼光當年就曾引進倭刀，他希望士兵多學各門類武藝，博採眾家之長，這也是一種包容的胸懷。當年是這樣的訓練模式，後世呢，我們的武術進入了一種虛無主義，很多人在屋子裡造拳。

傳統武術在當代很不好的一點就是嚴重的套路化，很多人一輩子不打一次沙袋，這是傳統武術目前最大的悲哀，這也是傳統武術自身發展的問題，首先是傳統武術出現了一種嚴重的技術異化，因為它遠離了格鬥，甚至是一種假想式的格鬥。傳統武術在發展中被中國文化綁架了，變得不實用，遠離實戰。變化出一些無意義的招數，比如所謂的見招拆招，其實呢，武術很簡單，沒有那麼複雜，在簡單中蘊含著一些玄機。其實，原來的傳統武術也有很多類似的兩人對戰訓練、練習等等，但後來的傳統武術發展中將其徹底虛無化了。

在指導思想上也嚴重偏離，過去傳統武術有對練模式，通過招式之間的不斷對抗產生條件反射，後來這一塊就忽視了，還有價值取向上，現在的傳統武術謬誤太多。有些人現在練習太極有點體會禪意的意思，但當初太極拳的創始人，想的可不是這些。武術開始的目的就是要把人訓練成一名武士。不是把人訓練成一個君子，必須是先有殺人術再講武德。傳統武術本源的價值現在也丟失了，很多人不願意在健身房揮汗如雨，覺得那都是笨人，陷入了仙俠式的幻想中，這都是大家對傳統武術價值的誤解，中國傳統的天人合一的哲學其實和武術沒有任何關係，說俗一點，你能把人打倒就是天人合一，這也是一種對武術的綁架。傳統武術的所有訓練其實都是圍繞著如何打造速度、硬度和摧毀力來進行的。而今天，我們看了太多軟拳表演。

　　也有不少人會拿所謂的傳統武術的健身性說事，這其實是很多人不願意吃苦，嚮往一直舒服優雅的練功方式，向武術真諦前進必須流汗留血。
