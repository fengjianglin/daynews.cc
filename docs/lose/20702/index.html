心意拳十二大勢：虎撲搓打實戰應用教學

2019-12-16 130條評論 2,878次閱讀 1162人點贊

1.敵左勢.與我對峙中。我主動以左手為引手.指敵面部.引誘敵人出手防護(圖5-35)。緊接著，右手穿插敵人面部(圖5-36)。

我右手下壓回拉引動敵人身體.左拳猛擊敵人當胸，同時.左腳寸步向前踩進。(圖5-37,圖5-38)

【動作要領】左手要輕靈，不必太著力，對方如反應遲鈍，我左手即穿插敵人面部，但往往不會擊中，左手未中.右手緊跟著發出，形成一手進一手回的習慣。出手中要有挫勁，回手時要有勾拉之勁.但不要抓住不放與對方糾纏。一勾即松，意在引動對方垂心，提高擊打效果。

2.敵以右拳擊我。我左勢.右手由下向前上挫開敵人右臂，緊接著左腳向前踩出一步，左拳擊打敵人右側肋部。(圖5-39,圖5-40)

3.敵左手抓持我前胸(圖5-41)。我左勢，右手穿插其面部(圖5-42)，緊接著.左腳快速向前下踩，身體旋轉.左拳直擊對方下頦。(圖5-43)

【動作要領】過步、左手持帶和右拳擊打要同時進行。技巧在於左手臂由敵人左臂穿出下壓時不必過快，要在無意問輕輕地開始，而部表悄要非常緩和，不要帶出攻擊意識，但瞬間的過步發力即變臉變色，驟發雷聲.必要一擊洞穿其身，使敵無還手之力。武術之要道在於一個「快」字，唯「快」難破。但也不全是一味地求快.要像性能優越的跑車一樣，突然加速，突然變向。這種起動速度的快.才能讓對手適應不了你的節奏，使其手忙腳亂產生恐慌。人們往往並不供怕天氣的寒冷和酷熱.但忽冷忽熱的天氣往往會讓很多人因適應不了而生病。武術也是一樣。

4.敵左勢，雙手抓握我之雙手(圖5-44)。我左勢.左腳向前踩進，腰部向右旋轉，左拳直指敵人當胸向前發力，右手下按後持與前手形成爭力，能達致步到人翻的效果(圖5-45)。敵如跳步退後.我右腳以過步發右拳，直搗其胸部。動中發力，一擊必中.中必跌仆。(圖5-46)

5.敵左勢，以右鞭腿擊我。我左勢，左腳搶進中門，左拳指著對手頭部(圖5-47、圖5-48)。緊接著.右腳過步向前猛踩一步.右拳擊敵頭部.並暗含肘擊胸腹之意。(圖5-49)
