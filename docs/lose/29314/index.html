動起來，冬日才是減膘的最佳時期

2019-12-20 250條評論 2,748次閱讀 1716人點贊

冬季

是最容易被「胖友」們無視的時候

不是不知道冬季容易「囤膘」

但是……

火鍋它不香嗎？

被窩它不暖嗎？

奶茶它不甜嗎？

至於身上的肥肉

羽絨服一穿

誰看得見

夏天還有好幾個月

暫且忘記這些不愉快的事

直到天氣轉暖

才曉得為時已晚

「冬胖夏瘦」已經被很多人認為是一種自然規律，夏天稍稍一動便會出汗，冬季天氣寒冷，多坐少動還愛吃，不知不覺就胖了。都愛在夏天減肥，但其實冬季才是減肥的良辰。

冬季減肥的優勢：

1.冬季基礎代謝增加

都知道基礎代謝高的人不容易胖，在冬天，當身體處於寒冷的狀態時，機體為了抵禦寒冷就會增加基礎代謝，冬天人體內的基礎代謝量要比夏天增加10%左右。

2.脂肪活躍度更高

研究表明，在寒冷的環境中，身體內負責耗能產熱的棕色脂肪就會促使白色脂肪燃燒，減少體內的脂肪含量，如果體內含有50克棕色脂肪，每天就能幫助人體多消耗20%的代謝。

我們要在冬季保持一個良好的瘦身效果，其過程與方法主要還是在熱量攝入與消耗之間來找出口，只不過會因為自然規律的變化而引起些許的不同，需要我們更加有意識地去控制來避免冬季長胖的現象發生，並且，只要我們能夠控制好，還會在這個冬天不胖反瘦。
