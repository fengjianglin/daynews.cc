寒冷的冬天竟然更適合跑步減肥？那我們冬天應該如何跑步？

天天要聞
2019-12-05

冷空氣在一波接一波地襲來，雖有陽光陪伴，但終究還是敵不住冷空氣的壓制，氣溫上升的動力明顯不足。全國都在大幅度降溫，冬天如約而至，面對著寒冷的天氣，不少跑者都開始成為國家一級退堂鼓演奏藝術家，對跑步打起了退堂鼓，鑽進了溫暖的棉被和棉衣下，而還有一些跑者卻無懼寒冷，開啟冬訓，為來年春天積蓄力量。

不過在這裡Chris告訴大家一個小秘密冬天其實很適合跑步減肥！

冬天跑步利於減肥？

寒冷的冬天，閑暇時間裡，按照大部分人的想法好像大家都應該舒舒服服在棉衣棉被下，暖氣暖風下，地暖火爐旁，取暖，休息。

冬天跑步竟然還更利於減肥？冬天不是長肉的季節嗎？這好像和很多人的理解認知有一定衝突，不過這確實是實際存在的！為什麼呢？

冬天人體的基礎代謝會提高

因為人體的生理特徵，使我們在低溫環境下，會反射性地引起體內的物質代謝過程增強，增加機體的氧耗。這意味著基礎代謝的增加，每天自身消耗的熱量就變多，如果配合好跑步和均衡的飲食，消耗更多的熱量，比起其他季節更容易造成一定的熱量缺口，從而更利於減肥。但是注意一點，適當的低溫可以增加基礎代謝，但是過低的溫度會造成人體基礎代謝的降低，減少我們的最大攝氧量，在跑步中造成缺氧的狀態。所以冬季跑步的時候要做好保暖。

冬天睡眠質量高

眾所周知，睡眠對於減肥來講是非常重要的，睡眠不足時，除了會增加食慾，另外胰島素對於血糖的敏感性也會下降30%甚至更多，從而產生胰島素抵抗，這樣，血液中的血糖就無法正常的進入目標組織細胞，多餘的血糖會以脂肪的形式儲存下來。而充分良好的睡眠可以幫助我們更好的減肥。而冬天日照時間短，在黑暗的環境下，人體會更容易產生利於睡眠的褪黑素，使我們可以更快的入眠，擁有更好的睡眠質量。

以上兩點更好的解釋了為了冬天跑步有利於減肥，既然冬天跑步可以利於我們減肥。

寒冷的冬天我們應該如何跑呢？

充分的熱身準備

在低溫環境下，我們運動一定要充分的做好熱身準備活動，因為低溫會使肌肉僵硬，粘滯性增大，降低關節活動度，更容易造成運動損傷；另外低溫還會降低組織的興奮性，影響到我們的運動。所以要在跑前要做好充分的準備，主要以提高我們的神經興奮度，使肌肉中血液流動加快，提高體溫為目標。為大家推薦以下熱身動作：

1. 腳尖不離地的小步跑

2. 高抬腿

3. 俯身開合跳

4. 慢速深蹲

做好保暖措施

冬季跑步保暖是最重要的事情沒有之一，良好的保暖措施，可以使我們冬季跑步時的體溫穩定；防止身體凍傷；減少感冒的可能性。

赫爾波特.史蒂凡尼在《跑步聖經》中提到，”沒有糟糕的天氣，只有不夠好的服裝！”那麼我們在冬天應該如何選擇保暖裝備措施？

穿衣建議：

冬天跑步最好選用三層法則：

1. 貼身穿著功能性材質的緊身服，穿著舒適，排汗，維持身體的”小氣候”。此為排汗層；

2. 中間穿著容易保存溫度的服裝，負責控制體溫，隔熱，輔助排汗。此為保溫層；

3. 最外面的一層一定要選擇擁有防風材料的外套，負責擋風，隔絕外界糟糕的天氣影響，輔助排汗。此為防風層。

下裝也是同理。根據以上的穿衣建議，可以最大程度保持身體機能，體內溫度，運動效果不受寒冬和大風影響，從而提高我們冬季跑步的舒適性和安全性。

其他裝備選擇：

1. 跑步手套：保護手不被凍傷，正常擺臂。

2. 帽子，護頸：當頭部和頸部沒有做好保暖的時候，至少會損失人體40%的熱量，所以要充分的做好頭部和頸部的保暖。

小結
