男子每天45個俯卧撐、深蹲和15個引體，網友：再練顏值也是硬傷

2019-12-18 519條評論 1,878次閱讀 100人點贊

健身在我們日常生活中非常常見，每天鍛煉一會可以增強身體的免疫力，增加身體素質，同時健身還會提升自信心。健身有諸多的好處，這種免費的投資對於每一個人來說都受益。但為什麼有極少數人練出好的身材，大部分人半途而廢？原因很簡單，每一個人都有懶惰心裡，認為健身不健身也不會影響什麼，相反健身還會增加自己的負擔，會感覺到乏味。

但如果我們把健身當作是一種興趣，相信大家就能夠堅持下來了。很多人之所以健身是因為自己實在是太胖自己都看不下去，還有一部分人是因為看到別人健身後一身的肌肉，高高壯壯的很有男人味，自己嘗試後發現健身真的很難。

健身不是一頭熱血，健身是一種熱愛，健身是對自我的提高，當你真正熱愛它時你就會發現他的魅力。

上圖中這位小伙堅持健身有一年的時間了，雖說看上去並不是很壯，但比一般男性身材要好很多。他最初就是想通過健身讓自己壯一些，因為自己身體的原因吃什麼都不胖，屬於干吃不胖型的。其實根本原因是他身體吸收慢，所以健身是最好的方式。

小伙為自己制定了30天的健身計劃，沒有很難的動作，都是日常生活中我們常見到的動作俯卧撐、深蹲和引體向上。俯卧撐和深蹲每天各做45個，分三組完成，一組15個。而引體向上每組5個。

從數量上看並不是很多，但堅持一個月應該也是有定的毅力才可以，當然對於他來說這並不是難事。

曾經小編誇下海口說自己每天做100個深蹲和45個俯卧撐，堅持一個月看看效果。沒想到最後只堅持了不到一個星期的時間就斷更了，為什麼？

懶惰，很多時間我們習慣了以前的生活，想要改變現在的狀態真的很難，他需要一個過程，最好有一個朋友監督。

俯卧撐是練習手臂、胸肌和肩膀最佳的徒手動作，深蹲可以練習我們的大腿肌肉，引體向上可以練習我們的背闊肌，三個動作結合起來相當的完美，可以練習全身肌肉。

小伙30天時間到了，我們看看有什麼改變？

30天之後我們發現小伙的肌肉塊頭大了不少，不知道是不是視覺的原因，還是真的在30天有改變。從數據來看每天45個俯卧撐，45個深蹲和15個引體向上並不是很多，但也不少，應該多少有效果。

我基於小夥子之前就有基礎，這個數量如果定1個月的時間相信改變並不是很大，但從對比圖視覺上改變很多。膚色也變了，有網友說在怎麼改變顏值也是硬傷。

不得不說網友的話有點過分了，其實小伙長相也是可以了大眾一枚，不是很醜就是一般人。

其實我們從細節上來看在30天的時間裡小伙其實改變很大，起碼胸肌是有明顯改變的。每天堅持鍛煉可以增加肌肉耐力，身體素質也會提高。
