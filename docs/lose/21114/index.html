肌肉訓練周琦舉起30公斤的杠鈴，詹姆斯舉輪胎。字母哥：不想炫耀

2019-12-20 1,189條評論 3,066次閱讀 1709人點贊

不分年齡，部分強弱，只要你能舉起啞鈴，你都可以去訓練肌肉。在訓練肌肉當中，每個人依舊個人能力不同，訓練時候方式也不一樣。像奧尼爾這樣的已經退役的將近50歲的老大爺，他去訓練肌肉的時候，會選擇一些比較輕一點的杠鈴，比如像這樣30公斤的小杠鈴。

3：周琦訓練肌肉當中舉的杠鈴

當然，有些球員在球員時期，也是會舉小杠鈴的，就像周琦一樣，這是因為他們要開始熱身了，別看這隻有30公斤。但是對於熱身運動，這是最好的選擇，等舉完這個30公斤的杠鈴。周琦也還會舉更重的杠鈴，畢竟作為一個職業籃球運動員，怎麼可能只舉30公斤的杠鈴！

2：勒布朗詹姆斯舉輪胎

在訓練肌肉當中，有一個訓練器材是在戶外才能訓練的，那就是大號的輪胎，有的運動員會用翻動輪胎的方式訓練肌肉。但是，像勒布朗詹姆斯這麼強大的球員，翻動輪胎已經沒有挑戰了。他年輕時候，都是抬著輪胎走的，這輪胎少說也有240斤左右。

1：字母哥不想炫耀

要說訓練肌肉當中，誰最厲害，那勒布朗詹姆斯都要服這個球員。他就是字母哥，短短的幾年時間裡面，字母哥從一個瘦弱的球員變成聯盟的肌肉最強的球員之一，靠的就是他訓練的態度。他舉的杠鈴是240公斤的，這樣訓練肌肉，才是運動員該有的樣子，才能增強自己的肌肉。
