腰痛容易聯想到是背部無力，主因是因為你的核心出了問題喔

2019-12-04 645條評論 2,452次閱讀 234人點贊

你有背部疼痛的問題嗎？腰痛是否造成生活中許多的不便呢？在許多研究指出，美國估計有85％的人都有背部疼痛的問題。

而腰痛是美國第二位常見疾病，腰痛和脊柱的問題，間接降低生活品質，像久坐、久站或提重物都常常使人疼痛難耐。

然而保持脊柱健康往往不是多數人健身計劃的一個重點，但通常都是發生了損傷或慢性疼痛後，才會關注到脊柱整體功能的重要性。

脊椎具有許多重要的功能包括支撐頭部、頸部、肩部和骨盆的穩定性，這是每天最基本的生活功能，也是生活中的關鍵，脊椎也保護了中樞神經系統，控制人體內的每一個的功能。

提高脊柱健康的四個要點

► 提高核心肌力

► 增加核心的活動度

► 改善姿勢

► 提高人體力學（在生活的各個方面）

核心肌力取決於核心的穩定性，這有助於保護脊椎以防受傷。

核心的基本構造是腹肌、腹內外協肌、背肌、臀肌、髖部和盆骨，並且連接到臀部、脊柱、肋骨和軀幹的其它肌肉的部分，核心肌群也是上肢和下肢主體間的橋樑。

好的核心將對生活和訓練，有助於增加對於腰部的保護和增加安全性，並確實掌握動作要領。

一起來打造訓練出強健的核心肌群吧！
