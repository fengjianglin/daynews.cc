減肥王減重成功後三年復胖，最終不治身亡

2019-12-24 1,053條評論 2,366次閱讀 151人點贊

微信公眾號：青竹新健康

相信很多胖友都會看減肥類型的節目，參賽選手在節目中揮汗如雨，為了甩掉一兩三斤肉而顯現出的堅毅讓許多胖友感同身受，為之動容。

而普遍節目結束後，參賽選手都能甩掉大部分脂肪，實現體重逆襲。

但節目結束後，但節目結束後，卻沒有長期的跟蹤記錄。那些減肥王的體重，還有維持住嗎？

其實反彈率高達比你想像中高得多！

減肥王減重成功後三年復胖，最終不治身亡

2007年，阿根廷減肥節目冠軍奧利瓦，在節目中從430斤減到180斤，減掉了大半個自己。然而僅僅只過了三年，體重重新飆升到500斤。今年4月，身體各個器官出現衰竭，最終不治身亡，享年34歲。

美國知名綜藝節目《超級減肥王》，召集了14位肥胖症患者進行飲食控制和運動的傳統減肥方法。如今節目過去九年，只有1位成功保持了效果，其他參賽者全部復胖，有的甚至比參賽前更胖。

這種減肥欄目，都會有個吸引人的點，就比如美國的減肥節目，第一名獲得25萬美金的獎勵，有些人一輩子都賺不到這麼多錢，而現在只需要我是個超重者，咬咬牙減個肥，還有錢賺，何樂而不為？

目標有很強的激勵作用，但一旦目標達成，如果不繼續給自己設立新的目標，人就會很快陷入一種比較頹廢的狀態當中，有些人焦慮、不知所措、甚至覺得生活失去意義。

更多人覺得，既然目標達成，我也證明了自己，就沒必要堅持減肥期間的習慣了。

減肥後

在節目結束之後，所有參賽者都無法繼續長時間維持參賽期間的高強度運動和嚴格的飲食控制。

不再有專家和教練團隊的指導是一個原因。最核心的是，隨著高強度運動的降低，以及嚴格飲食控制的放緩，瘦身成功者，他們曾經的生活和飲食習慣又恢復到了原來的狀態。

反彈

于是之前減掉的肉肉也就悄無聲息的再次回歸。《超級減肥王》這種綜藝節目對所有的減肥者造成了一種認知上的誤區，這種誤區就是讓大家把減肥當成了一場跟脂肪進行的激烈戰爭。

經過7個月的艱苦鏖戰，一朝脫胎換骨，甩掉脂肪無數，認為自己減肥成功了。

至少從短期的體型上看，的確成功了。那成功之後呢？終於就可以享受人生了！七個月的飲食控制啊，老子嘴裡都淡出鳥來了，是不是要隔三差五得好好犒勞一下自己啊！

很多肥胖者在瘦身前的飲食習慣並沒有得到根本性的扭轉，每頓飯照例會有冰淇淋甜點，心情好的時候一天會吃5個漢堡和3瓶可樂。

而晚餐之後和家人抱著大號薯條窩在沙發上看電視也是基本的娛樂活動。7個月的時間減掉了的肥肉，卻並沒有改變肥胖者多年的生活習慣。

也正因如此，在瘦身成功後，體重再度反彈基本是妥妥的。我們把這種不改變生活習慣的短期減肥，稱之為「表象減肥」或者「抽風式減肥」。

因此，我們可以得出一條結論——沒有自律的生活習慣，減肥都是無效的！
