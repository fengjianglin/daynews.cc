乒乓解惑：就算你重心交換和身體找球都做到了，沒有一致性也白瞎

2019-12-17 385條評論 4,278次閱讀 2611人點贊

很多球友總是感覺自己不是不能發力而是控制不住力，但不使勁又不能作用到球，這種情況，就是因為對力量的釋放把控不好，這樣肯定又會出現很多失誤，從側面反映出這些球友並不是用身體整體打球，而是僅僅靠單一的胳膊在打球。整個動作上有明顯的胳膊和身體分家的情況。

有必要強調的是：這裡所說的單一的用胳膊打球並不是說別的地方都沒有動作，表面上看起來重心有在交換、身體也在轉，但是由於這些動作和胳膊的動作缺乏整體性（不夠一致），所以會出現以上的情況和感覺。

引拍時撤肘

就是在向後轉身體引拍時，肘有很明顯的後撤動作，大臂運動和身體運動的相對位移太大了，導致你的大臂不能倚靠住身體，這一點是導致動作分家的核心原因。

不會用重心找球

不會用重心找球，面對來球習慣性或者是形而上學地把重心壓低了，其實並沒有理解重心的用法，往往發不上力，拉漏拉丟。它應該是隨著來球邊轉邊壓，重心的高與低是相對於來球來說的，根據來球的情況隨之調整改變的，最終達到蓄力和讓自己在一個很合理的角度去作用到球上。

很多球友就恰恰沒有用好這個技巧，重心不去找球的位置、擊球點都是在亂變的，導致最後只能用手去調整擊球的點，所以就加劇了胳膊自己亂動的這種情況，加劇了控制不住力的情況。

改正建議：

綜合以上問題所述，在以後練的時候，要先學會用在打球過程中調整重心高低，每個球都要隨動隨找。

引拍撤肘的問題有個小技巧，就是在你邊轉身體邊壓重心的時候讓自己的肘向右腿上靠，盡量靠的近一些，去體會帶上一些側壓身體的感覺！

另外再提醒一點就是轉重心的時候雙腳不要抬著腳後跟來迴轉動，這樣不利於重心穩定。
