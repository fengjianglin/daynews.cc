65歲瑜伽奶奶練瑜伽，身材不輸年輕人，越活越年輕

2019-12-18 23條評論 1,300次閱讀 260人點贊

​最近的瑜伽圈一位瑜伽姑娘——「胡柚奶奶」，引起了伽人們的廣泛關注。

65歲的她，不僅身材不輸年輕人，而且身體和精神狀態都特別好。

光看她瑜伽時「矯健」的身影，你壓根就不敢想，她已經65歲了。

堅持瑜伽14年，不論是柔韌性，還是力量，胡柚奶奶都是在線的，做起瑜伽高難度動作毫無壓力，厲害了，我的奶奶！

然而，更厲害的是，65歲胡柚奶奶還是瑜伽教練，一個月還要給學員上50-70節瑜伽課，而且是騎著電動車，來往各個點跑課。

並且在這樣的工作強度下，堅持數十年如一日。此外，胡柚奶奶還利用教瑜伽課以外的時間，不斷的學習考證。

胡柚奶奶說：「像現在特別火的空中瑜伽、孕瑜伽這些，我都有去學，還考出了教練證。活到老學到老嘛，不要覺得我老太婆了，就不懂這些。」

她的這種精神，真的值得我們每一位瑜伽老師學習。

當問起胡柚奶奶為什麼要教瑜伽，奶奶說：「那個時候瑜伽什麼的都還不流行呢，我自己學瑜伽嘗到了甜頭，希望也能讓更多的人了解這項運動。」

怪不得奶奶的課，能深受80、90後的喜歡，每節課都爆滿。65歲依然堅持學習能量滿滿的人，誰能不喜歡呢？

還有學員稱，跟著「胡柚奶奶」學瑜伽3年，已經瘦了20多斤。真心的為奶奶點贊，太太太厲害啦！

其實，胡柚奶奶學瑜伽前，身體狀況並不是很好，退休前她是基層機關工作人員，身體跟大多數人一樣亞健康，還有腰肌勞損、甲亢等疾病問題

但通過多年堅持練習瑜伽，胡柚奶奶說她的這些問題都有所改善。

「以前醫藥費不夠花，現在不僅身體好，把看病的錢都省下來了，還能有得賺。」

現在的胡柚奶奶，上瑜伽課的收入比退休工資高了兩三倍。

然而，比收入更加重要的是，瑜伽讓奶奶覺得自己很有價值，身體到心態都依然很年輕，即使65歲也能活成自己想要的樣子。

瑜伽姑娘——胡柚奶奶再一次證明，練瑜伽，與年齡無關，當瑜伽老師，與年齡無關，瑜伽和讀書一樣，都是用最低的成本，成就一個更好的自己。

2020年快到了

趕緊跟胡柚奶奶一樣

瑜伽起來吧
