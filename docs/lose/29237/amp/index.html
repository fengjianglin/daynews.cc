他從來不辦健身卡，也不會克制飲食，但是他卻是街頭之王！

天天要聞
2019-12-22

大家好，我是思遠。

健身是一項最近幾年在我國才開始一點點興起的運動，對於歐美國家而言這項運動在上個世紀的時候就已經開始流行了。而且不管是從經驗，還是訓練要領，以及營養的攝入方面他們都有著非常好的理論和實踐。而我們很多人健身除了去健身房之外，還有一種叫做「街頭健身」一般都簡稱是「街健」。街健相對比起健身房的方式來說，沒有太多的束縛，對於重量也沒有很多的要求。

他們利用少數的器材，配合更多的動作和自身的重量還進行訓練。而我們今天也是給大家介紹一位「街健之王」，他從來不去健身房，也不愛吃蛋白粉，更不用去刻意的控制自己的飲食，就練就了一身的腱子肉。是不是很羨慕呢？我們先來領略一下這位「街健之王」的風采吧！利用一根鐵柱子，然後自己的人體旗幟說來就來，這真的沒有多少人能夠做到！

而俄挺更是輕而易舉的，並且一身的黑色皮膚在陽光照耀下把自己肌肉的每一個稜角都展現了出來！

是不是很多人都認識這一位街健大神呢？沒錯，他就是街頭健身的創始人，也可以說是這個領域的傳奇人物了，他的名字叫做「漢尼拔」。

漢尼拔子長達20年的健身生涯中，真的是一次健身房也都沒有去過，對他幫助最大的就是我們在很多小區裡面都能夠見到的「單雙杠」了。在蛋白質攝入方面，漢尼拔表示：「我從來不會去*****，當然不是說它不好，只不過我覺得使用肉類會讓我更有動力。」雖然不知道是真是假，但這個回答也是很有意思了。漢尼拔也是從來都不會去刻意控制飲食的，因為他每天的訓練時間都在兩個小時以上，巨大的身體消耗不會造成自身過多的熱量轉化為脂肪。漢尼拔的除了一身漂亮的腱子肉之外，他自身的協調性也是讓人讚歎的。

看一下這一張的雙杠人體升降機，不急能夠很好的控制手臂讓自己身體上下移動，同時把身體控制成T字形，雙腿交替鍛煉自己的腹肌。

1978年出生的他，現在已經40歲的他健身20年時間了。在以前的時候，漢尼拔不僅給人家當過保鏢還做做木工。可是慢慢的有了人氣，並且開始自己街頭健身的推廣之旅以後，他更願意成為一名奶爸。把自己更多的時間留給自己可愛的小女兒！
