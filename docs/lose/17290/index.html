高強度2000米划船的3個注意事項，划船機體能測試必需知道

2019-12-13 2,260條評論 3,880次閱讀 272人點贊

前不久，WaterRower轉發了一個划船機的挑戰視頻，這個視頻來自OTF著名的教練Austin Hendrickson（ins@trainingtall），他分享了自己挑戰2000米划船機體能測試的最終成績，用時5分24.4秒。這個成績，相當於平均每500米用時僅1分21.1秒。

這個成績的背後可一點都不簡單，高強度的體能測試和平時我們中低強度的有氧划船有著天壤之別，2000米在平時的有氧中可能不過才剛剛熱身，而全力以赴的2000米體能測試，則可能直接讓你精疲力盡。

為此，Austin也給大家提出了3個長時間划船的注意事項，如果你也想嘗試來一個2000米體能測試，那麼這3點一定不能忘記了。

1、這是一段煎熬的時間。2000米划船用時在6-10分鐘左右，長時間的高強度划船，足以讓你的精神崩潰，所以，在一開始的時候，要注意控制節奏，前期應以勻速為主，最後的500米，如果身體感覺良好，再去做衝刺。

2、注意划船間隔的呼吸。高強度划船已經屬於無氧運動，所以呼吸和平時的有氧是不同的，你需要在每一次划船的開始和結束時呼氣，在拉槳和回槳之間吸氣。

3、注意中途調整姿勢。當你體能開始下降時，最忌諱的就是姿勢上的鬆懈，你可能開始出現身體前傾，或者每一次後仰時角度變大，這樣對身體的傷害是非常大的，所以，當你開始出現意志力和體能下降時，一定要調整好划船姿勢，避免受傷。

划船機的體能測試在賽艇和帆船運動員中較為常見，它可以較為直觀的反應出運動員的專項力量、力量耐力和速度耐力的水平。對於賽艇運動員來說，2000米的體能測試是一個日常訓練，目的是提高成績。

而對於我們普通人來說，2000米的體能測試主要還是以鍛煉我們耐力為主，所以，船友們如果以健康減脂為主，那麼可以作為一個趣味的挑戰偶爾嘗試下，如果以提高心肺功能為主，則可以適當的加入自己的訓練計劃中。
