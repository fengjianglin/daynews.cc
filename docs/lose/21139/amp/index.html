120kg到78kg的蛻變，不僅僅是收穫健康那麼簡單

天天要聞
2019-12-19

大家好，我叫余鵬，浙江溫州人，現在25歲。這是我現在的照片，先給小夥伴們看一下。

也許有些小夥伴會覺得我練的不怎麼樣嗎。是的，現在我還處於增肌階段，我相信我以後一定可以達到自己的目標的，今天我想要先和大家分享的是我從120公斤到78公斤的歷程，也算給自己一個里程碑式的紀念。

圖1和圖2都是我120公斤的時候，怎麼樣，是不是看上去沒救了，當時自己都嫌棄自己，看到鏡子就會厭煩。其實曾經也想過用各種方法減肥。先是節食，臣妾根本做不到好嗎。每天看著路邊的燒烤攤我真的眼淚都快彪出來了。好，那就運動減肥，跑步，登山，騎單車，都試過了，結果都是堅持不下來。（看看下面照片，是不是覺得自行車很可憐）只能繼續胖著。

眼看都快奔三了，親朋好友開始各種安排相親。別說，其實還就是相親讓我意識到了自己不能在這麼墮落下去了。大大小小相了十來次，姑娘們不出所料的都為我滅了燈。但是要說真正讓我洗心革面的還是那次和妹子去遊樂園玩。卻被告知超過了遊藝機的限重，想玩玩過山車，卻發現保險帶扣不上。看著旁邊的工作人員和妹子那尷尬的眼神，想死的心都有了。當天回家我就下定決心，這次一定要改變自己，絕不是說說而已。

擔心一個人堅持不下來，而且又不知道方法，於是開始各種查詢溫州有沒有好的健身房。一次機緣巧合，在朋友的推薦下，我來到了全國健美冠軍趙章雷老師（以下簡稱阿雷）的健身工作室。一進去就看到了阿雷在帶會員，神情非常嚴肅，如果會員動作不標準，他甚至還會嚴厲的斥責。等他上完了課，我向他表明了來意。阿雷表示可以帶我，但是一定要嚴格的照他說的做。沒有絲毫猶豫，我們就這樣開始了3個月的征程。

這三個月里，的確是很辛苦，很累。但是當你每隔一段時間發現鏡子中自己的變化，你就會發現自己做的一切都是值得的。很感激阿雷，他的嚴厲很大程度上幫助了我。還記得有一次，我剛到健身房準備開始訓練，說了一句「好累阿」，阿雷立馬板臉：你回家吧。經過我好說歹說，他才消了氣。不過現在絲毫不會擔心這種事情了，因為我已經上癮了。如果那一天不去健身房，我會感覺渾身難受，也許這也是一種強迫症吧哈哈。

在訓練上除了常規的力量訓練，阿雷還給我安排了各種有氧計劃。考慮到我的體重較大，阿雷安排了室內的單車和橢圓機的有氧訓練。另外，在周末，他還會和我一起騎著山地車去戶外。在飲食方面當然就是水煮雞胸肉西蘭花番茄黃瓜等常規食物了。另外在補劑方面，阿雷還給我安排了乳清蛋白粉作為訓練後的補充。同時我也使用了液態燃脂素來加快我的脂肪燃燒以及保證我的訓練狀態。

慢慢的，我的身材開始有了變化，小肚腩開始小了，肉也開始變的緊了。我開始有點懈怠了，但是阿雷告訴我，現在才是最關鍵的時候，如果放鬆了，就會開始反彈。於是我咬了咬牙，都堅持到現在了，不可以放棄。

終於，我成功了，我成功的減到了理想的體型。而且，我還收穫了一份愛情。也許這就是所謂的雙喜臨門吧。

上圖就是我的女朋友和現在的我。現在的我幾乎一周都會去5-6次健身房，我覺得過得很充實。非常感謝阿雷，如果沒有他們的支持我也許無法走到今天，也或許我會一直選擇做一隻小胖子。但是很慶幸我堅持下來了，而且我堅信，這條路，我會一直走下去。也許我沒有足夠的天賦去參加比賽，但是健身，會是我一輩子熱愛的東西。

最後放上一張對比圖，相信我，我可以，你們也可以的！
