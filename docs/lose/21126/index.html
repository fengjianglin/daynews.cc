明星減肥有多拼？鍾漢良只吃一口，馬思純七天不吃飯，她喝利尿劑

2019-12-19 1,282條評論 2,611次閱讀 83人點贊

大家都知道減肥是一個非常痛苦的事情，因為它需要人有很大的意志和自制力才可以。有許多人嘴上喊著減肥，但是堅持不下來，因為這件事情真的太痛苦了。不過也有許多減肥成功的案例，他們對自己都超級的狠心，才讓自己瘦下來的。娛樂圈中有很多明星都有減肥成功的經歷，為了減肥，他們真的是對自己非常的狠心。

因為明星和大家不一樣，他們的職業就要求他們必須要瘦，而且要比普通人的瘦還要瘦上好幾倍才可以，只有這個樣子才能夠上鏡好看。明星減肥有多拼呢？曾經鍾漢良在一個綜藝節目上大家在一起吃飯，然後他就只吃了一口麵包，就放下筷子說自己飽了。

其實並不是自己飽了，只是不能夠再多吃了，畢竟他還要維持身材的。然後旁邊的郭麒麟聽到他說飽了之後就非常的驚訝，就提醒他你只吃了幾塊麵包而已。然後鍾漢良就給他糾正過來是一塊不是幾塊。隨後我們就看到郭麒麟的臉上那種詫異很無奈，感覺不能理解鍾漢良的做法。

正是因為鍾漢良對自己的嚴格要求，才沒有讓自己上了歲數之後發福變胖。可是這樣真的好艱難呀，面對那麼多的美食不能吃，一般人真的控制不住。馬思純表示自己曾經在拍左耳的時候被要求要減肥，所以那個時候她7天沒有吃過飯。

為了可以快速的減肥，她不吃飯還拚命的跑步，扛不住的時候就喝藿香正氣水。聽到這些之後，感覺她真的是在拿命減肥，不過好在馬思成憑藉左耳這個角色火了起來，也算沒有辜負她的努力。有時候演員為了角色不僅僅要減肥，而且還要增肥呢。

前段時間馬思純飾演了一個角色，劇中的這個女孩兒的是一個胖胖的女孩兒，所以馬思純成為了可以扮演好這個角色就增肥20斤，那段時間她狂吃不止，很快就讓自己的體重上去了。可是在出演完這個角色之後，馬思純就必須讓自己快速的瘦下來才行，因為角色已經拍完了，她不能夠再讓自己的身材維持一個胖的狀態。

所以就在胖瘦中反覆顛倒，其實這也很損害一個人的身體健康的，但是對於明星們來說就是這麼的殘酷，沒有辦法。還有一個人更加厲害了，為了減肥喝利尿劑。

​

她就是大家都很熟悉的張柏芝，張柏芝是一個對自己要求很嚴格的人，據說在她年輕的時候拍一個角色需要非常的瘦，所以她就喝了這個東西。不得不說明星們減肥真的是太拼了，普通人但凡有一些這樣的決心，也不會變胖了。
