堅持打羽毛球，從一周到十年，我變成了這樣

2019-12-23 1,615條評論 4,429次閱讀 1568人點贊

愛上羽毛球不是一件難事，你也不應該把它當成一種修行，只要開始打羽毛球，你就會與快樂相遇。當羽毛球打到一定的時間後，腦垂體開始分泌出內啡肽，產生愉悅感，身體被輕快感佔據，甚至你都會認為自己能永遠這麼打下去！

打羽毛球1周後

這是開始改變的一周，七天時間或許無法改變很多事情，但是你的身體開始發生正向的改變，打羽毛球的初體驗讓你的心率加快、血液快速流過全身、身上多餘的脂肪開始燃燒。

雖然你的身體會感到酸痛，但是心情卻莫名地感到舒暢，因為打羽毛球會促使腦垂體分泌出快樂激素——內啡肽，就像是戀愛的感覺，你會覺得自己快要飛起來了。打羽毛球帶給你一天的好心情和更加輕快的人生態度，讓你覺得沿途的風景都是不一樣的。

打羽毛球1個月後

打羽毛球一個月了，你的呼吸都變得均勻平穩了，心跳也變得沉穩有力了。你應該已經甩掉了幾斤贅肉，慢慢地有些衣服褲子穿上去已經有點大了。而你的生活習慣也在悄然發生改變，你會發現早睡早起不再是難事。為了捍衛打羽毛球的成果，自然會吃得清淡，對高脂肪、高糖、高熱量食物的渴望也不像之前那麼強烈。

周圍的人都發現你的精神狀態變好了，而你也會覺得之前稍微動一動就氣喘吁吁的自己，現在陪孩子參加親子競賽活動居然也能領先絕大多數的父母。那是因為打羽毛球帶給你和別人不一樣的精神氣！

打羽毛球1年後

你會發現這一年幾乎沒有去過醫院，打羽毛球讓你的免疫系統在24小時內都全力運轉。體能和耐力大幅度提高，更低的心率意味著更強勁的心臟，更平緩的呼吸意味著更加高效率的心肺系統，你已經可以跑下絕大多數人跑不到的距離。

打羽毛球帶給你高速的新陳代謝，你的身體就是一台「脂肪燃燒器」，脂肪不會再囤積。而你的同齡人，大多都是珠圓玉潤，腰腹部帶著一個游泳圈，當他們在研究減肥的時候，已經堅持打羽毛球一年的你，無論是體能還是體型都是他們望塵莫及的。

你堅持打羽毛球的事應該已經成為朋友圈裡面的一個話題，你的朋友、同事、親戚甚至領導，都會覺得你是一個靠譜的人，因為你會為了一個目標而堅持努力，專註讓你變得更有毅力，當這種精神轉移到生活中、工作上時，周圍的人就會發現原來你是一個這麼有魅力的人！

打羽毛球5年後

周圍很多同齡人的身體開始走下坡路，各種小毛病頻繁來襲，而你的身材卻更加健碩了，容貌甚至逆生長。那些身材臃腫的大叔大媽們看你的眼神都是充滿了羨慕嫉妒恨。

長期堅持打羽毛球，造就了健康良好的身體狀態，也磨鍊了堅忍不拔的精神性格。五年如一日的堅持打羽毛球是對孩子最好的教育，你會發現你的的孩子以你為榮，他/她也會和你一樣為了一個目標不拋棄、不放棄。打羽毛球，成為了你的一個人生標籤，堅持打羽毛球並不是為了戰勝誰，而是為了與更好的自己相遇！

打羽毛球10年後

長期堅持打羽毛球能夠消除對人體有危害作用的自由基，延緩衰老進程，你會發現體能狀態一點不輸年輕人，傲人的耐力和體力甚至可以和自己的孩子來一場真正的較量。

你的身材體型、儀錶姿態、體能水平、精神氣質站在同齡人中，顯得那麼與眾不同，你可以以良好的健康狀態繼續下一個10年、20年、30年……的精彩人生！

打羽毛球與不打羽毛球的人，在每天看來沒有任何區別，在每月看來差異也是微乎其微，在每年看來差距雖然明顯，但好像也沒什麼了不起的。但在每5年來看的時候，那就是身體和精神狀態的巨大分水嶺。等到了10年後再看的時候，也許就是一種人生對另一種人生不可企及的鴻溝了。

打羽毛球，為了遇見更好的自己！
