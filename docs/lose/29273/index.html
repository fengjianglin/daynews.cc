英國宅男從不出門，每天在家吃外賣，卻成功減肥209公斤

2019-12-21 326條評論 2,650次閱讀 1477人點贊

據12月20日英國《鏡報》：現年38歲的保羅斯·蒂文森（Paul Stevenson）自從成功從323公斤，減掉了209公斤，他便很幸運找到了一生摯愛，還在他們當地（德比市）找到了一份新工作。

保羅斯·蒂文森（Paul Stevenson）

之前保羅斯因為害怕別人的嘲笑他的身材，在家宅了整整4年時間。

四年前，保羅斯在母親去世後，一蹶不振，每天都在家裡吃外賣，一天吃三四次，以至於身材日漸臃腫。

2015年保羅斯的體重最重時高達323公斤，連體重秤都沒法稱出體重來。

德比郡新聞報道，他減肥成功後，找到的這份工作是12年來的第一份工作，在醫院做保安，還和他交往兩年的女友訂婚了。

保羅斯的女友是他兩年前減肥114公斤後，在社交軟體認識的，之後他們就開始交往。

他很滿意現在的生活，減肥成功讓他開始了新生活，重新開始工作，還即將和一個這麼好的女人結婚，感覺太幸福了，也對未來充滿了期待。

保羅斯過去宅在家裡時，每天吃薯片，披薩等高熱量食物，體重也隨之瘋漲，因為他是德比郡的球迷，所以他害怕自己英年早逝，再也看不了球賽，便堅定的踏上了減肥之路。

保羅斯在做了切胃手術後，減重到266公斤。這個手術可以永遠改變胃的大小,這也是他還能活到現在的原因。

保羅斯平時規律運動，組建了當地的足球隊，還去荷蘭做了手術，去除了身上因減肥多出來的皮。

現在他用自己的經歷鼓勵幫助更多曾和他有同樣煩惱的人，也很快要去工作了。

對此他表示，在自己嚴重超重時，工作是想都不敢想的。別人根本理解不了這對他的意義有多重要，當他找到新工作時簡直太興奮了。

圖片來源：網路

保羅斯沒想到還有人會需要他，他甚至已經等不及想去工作了。他現在的目標是保持身體健康，好好工作。他的未婚妻也一直陪伴著他。每當他心情不好時，未婚妻就會安慰他。

他想馬上迎娶未婚妻，因為她實在太好了，特別理解保羅斯所經歷的一切。此前他不相信自己可以走出家門，出門可以說是他一次瘋狂的旅行。

我們每個人的生活都可能遇到各種困難，我們要像保羅斯一樣，勇敢改變自己，難關也就不知不覺地度過了。
