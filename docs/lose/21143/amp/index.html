別再抱怨冬季跑不快！這6個技巧，掌握一半就能度過瓶頸期

天天要聞
2019-12-19

每個夏天跑得像獵豹的跑者們，一到冷冷的冬天，都會變成蝸牛。冬天怎麼就是跑不快呢？總感覺關節都冷冰冰的，半天都活動不開。

或許你應該改改觀念，試著從睡眠、營養、熱身、力量、心理等多方面綜合提升自己了！咚妞今天給你介紹6個小技巧~


1.增加睡眠時間

有一項研究發現，耐力跑運動員連續三天將睡眠時間增加90分鐘之後，在60分鐘的計時賽中表現提升了3%。

給數學不好的同學科普一下，這相當於2小時55分的馬拉松賽成績，快了幾乎1分鐘。

即使是職業跑者，也都非常重視晚上的睡眠。這就需要跑者為自己創造一個良好的睡眠環境，嚴格按照作息時間進行休息。

2.正確的熱身

冬天因為溫度低，所以身體比其他季節更難熱起來，這時一定不要貿然加速，否則容易造成拉傷。在跑步前進行熱身，既能讓身體快速熱起來，關節、肌肉和心臟都做好了準備，也可以預防出現意外。

最簡單的拉伸就是快走或慢跑3公里左右。此外，在室內進行一些核心力量的訓練也會讓跑步過程更加安全穩健，比如平板支撐、仰卧起坐、俯卧撐等熱身項目。

在經歷熱身過程之後，身體逐漸熱起來了，肌肉和關節以及心臟都做好了開始運動的準備，關節也充分潤滑，此時跑步會更快一些。


3.嘗試間歇衝刺跑

對熱衷於參賽的進階跑者來說，短距離衝刺訓練對改善跑步效率和速度也非常有幫助。

比較科學的訓練分配是，一周進行四次訓練，兩次40-50分鐘的跑步，周末進行一次長跑，加一次10KM或者半馬的速度進行的間歇衝刺跑。

參考方案：準備活動2公里，10公里，配速要求：80%，間歇8分鐘，專項跑6公里，配速：85%，間歇6分鐘，專項跑2000米，配速：90%。

特點&適用：遞減段落距離、增加運動強度間歇訓練法的特點是段落越來越短，強度越來越高。對提高馬拉松運動員的抗乳酸能力幫助很大。

4.選擇合適的裝備

首先是跑鞋：每一位跑者根據腳型、落地特點，對跑鞋的要求不同，而不同的訓練方法也需要配備相應的跑鞋。

而短距離衝刺跑、長距離慢跑、節奏跑等就需要不同類型的跑鞋，才能讓雙腳感到最舒服。所以，跑者不能一雙跑鞋跑到底，至少也需要兩雙不同類型的跑鞋，以適應不同的訓練。

其次，冬季戶外運動中的服裝要兼顧排汗、保暖和防風防雨，直接與肌膚接觸的貼身層，必須透氣的、快乾，保證讓汗水與肌膚儘快分離，保持肌膚的舒爽。

最外層則為防風防雨層，由於它的作用是抵抗住冷風和雨水，軟殼的衝鋒衣即可。在冬天零度左右的時候跑步的話，穿這兩層就夠。

此外，平時人體大部分熱量都是從頭頂流失，所以建議在冬天跑步時可以戴一頂帽子給頭部保暖，減少冷空氣對頭部的傷害，避免感冒、頭疼等現象發生。

襪子方面，永遠不要選擇棉襪！它們無法帶走汗液水分，流汗後只會越來越冷，讓你的雙腳有凍傷的危險。

5.補給的注意事項

冬天戶外跑，補水也是個問題。冰冰涼的水，到底能不能直接喝？

一小時以內的運動可以不補水，萬一喝涼水造成腸胃不適會得不償失，如果是運動強度較大的訓練情況，補水就是必要的了。

你可以帶一些熱水，也可以把常溫水含在嘴裡一會兒再咽下去，或者使用小的塑料水壺——只要貼身放，溫度並不會太低。

而超過一個小時的跑步中，跑者需要補給一些能量，因為之前儲存的糖原幾乎耗盡。這需要跑者提前準備好食物，注意碳水化合物、蛋白質和健康脂肪的搭配，能夠快速地為跑者提供所需能量，讓速度重新拾起來。

6.自我鼓勵

這一點屬於跑者的心理素質。當跑步感到極其困難時，物質方面的補給已經無法給跑者帶來更多的紅利，此時就需要跑者從心理層面戰勝困難。

你可以通過自言自語鼓勵自己，或者用一個筆記本記錄自己的跑步訓練日記，每日打卡，也可以說一些讓自己受到鼓舞的話語，或者平時喜歡的一些語錄。精神力量不容小覷~
