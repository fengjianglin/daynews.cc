找到自己的方向，並讓自己長期的去堅持。

其實，打扮也是需要去做的健身和打扮的一個好處，就是可以讓你變得更好看，同樣它的效果也是差不多，但是健身是從你身體本質的問題去得到解決，而且達到你想要的效果。
