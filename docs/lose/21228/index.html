重獲新生！英國女子18個月減重95公斤，親媽遇到她都沒認出

2019-12-18 462條評論 3,809次閱讀 2617人點贊

有時減重對於一個人來說，不僅是獲得更高的顏值，它可以讓一個人的生活完全改變，說是重獲新生也不為過。

據《每日郵報》12月17日報道，英國白金漢郡女子伊麗莎白·沃特金斯（Elizabeth Watkins）今年38歲，雖然她現在是一個樂觀積極、熱愛運動的招聘顧問，但就在兩年前，她還是個自卑內向、幾乎不與外界聯絡的超重者。

伊麗莎白從小就很胖，她總是從食物中獲取慰藉。在開心時，她通過吃來慶祝；在難過時，她又暴飲暴食發泄。自從她父母離婚，伊麗莎白更依賴食物了。

她的體重增長至約169公斤，自己也不愛和人接觸了，她幾乎切斷了和外界的聯繫，不想被陌生人盯著看，也不願意別人可憐她。伊麗莎白只和幾個最親近的人交往。

2017年，伊麗莎白做手術幫助其減肥。手術後的一年裡，她每天只吃700大卡的食物，每個月都會瘦6公斤左右。此後，她又改成每天吃1200大卡，體重一直穩步變輕。

曾經，伊麗莎白每天都不吃早餐，但一天會吃很多糖、巧克力、蛋糕等，喝茶也要加三塊糖。現在，她開始吃蔬菜、水果，並喝無糖咖啡。

目前，伊麗莎白約為74公斤，她還尋獲此生真愛，已經訂婚了。

伊麗莎白覺得自己重生了。「人們認不出我了，甚至我自己的媽媽都沒認出我，她直接從我身邊走過去了。」

她和兩個老朋友取得聯繫，覺得很抱歉，在他們書寫自己的人生履歷，結婚生子時，自己沒有給予朋友應有的支持。

伊麗莎白還和有同樣經歷的陌生人結為好友，她現在可以到處去旅行了。「我感覺像是無所不能，我不覺得這就是減掉了95公斤的問題，我覺得我獲得了新的生命，我想充分地過好它。」

然而，伊麗莎白減重的方法未必適合所有人。她說，她也曾去健身鍛煉，但體重卻增長了。「我知道可能是肌肉增加了，但我真的需要看到數字下降而不是增加。」因此她覺得這個方法不適合它，「我就讓自己的生活更忙碌，這對我比較有效。」
