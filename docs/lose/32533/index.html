力量訓練技術要領——抗阻力訓練

2019-12-25 1,838條評論 3,126次閱讀 1112人點贊

正確的動作有利於取得安全和有效的訓練效果

  1. 抗阻力訓練基本動作技巧
  2. 抗阻力訓練呼吸技巧
  3. 抗阻力訓練保護帶的使用
  4. 抗阻力訓練中健身夥伴之間的相互保護

在抗阻訓練中安全一直以來都是重中之重，掌握基本的抗阻力訓練技術要領對於每個開始健身，正在健身的我們來說都非常重要。

  1. 抗阻力訓練基本動作技巧
  • 握姿和握距

抗阻力訓練中常用的握距有倆種一種是正握（手背朝上，掌心向下）和反握(手背朝下，掌心向上）。

例如，在卧推的時候就採用正握的姿勢，而在二頭彎舉中就採用反握的姿勢。

還有一種握姿叫對握（掌心相對），跟一般我們打招呼時握手的姿勢類似。例如，在坐姿肩上推練習的握姿。

最後在保護其他訓練者，例如在卧推訓練時保護者通常採用正反握的握姿，（一手掌心朝上，一手掌心朝下）

在正握、反握和正反握中，我們的大拇指的拇指都要鎖死杠鈴，使握杠鈴時更加穩固，這時拇指是閉握姿勢。當大拇指貼在食指第一關節時，這種握姿稱為開握，也稱虛握。

在開始進行杠鈴訓練時，雙手在杠鈴上的握距也是非常有講究的，雙手要保持一定的距離，而我們稱這個距離為握距。有四種標準的握距，分別為窄握、髖寬握、肩寬握和寬握，在大部分正常抗阻力杠鈴訓練中都是採用肩寬握的握距。有一點需要注意，因為每個人在體型上的差距，導致了握距上的差別，所以無論採用哪種握距一定要注意的是杠鈴都應該保持平衡。

2.抗阻力訓練中的呼吸技巧

所有的健身人群都應該體會到呼吸在訓練過程中的重要性。（呼氣時會增加核心力量，也就是我們常說的腹內壓，從而提高訓練者的穩定性，提高運動者的發力）

在所有的力量訓練中，用力階段呼氣，放下重物時吸氣（不用力）。

  • 瓦爾沙瓦現象（憋氣發力）

進行力量訓練時，通常訓練者會在發力時會有憋氣的現象，這時會產生瓦爾沙瓦現象。

瓦爾沙瓦現象通常會產生以下幾種情況：

  1. 氣門關閉
  2. 膈肌及腹肌的強直性收縮而使腹內壓升高
  3. 由於呼吸肌的收縮而使胸內壓升高

練習時有兩種呼吸方法：

  • 在放下重物時吸氣，在發力時憋氣，然後呼氣
  • 在開始做動作前吸氣，在做動作過程中憋氣，然後呼氣。

儘管憋氣有利於發力，但瓦爾沙瓦現象會限制靜脈血液的迴流，使心輸出量降低；還會引起血壓升高，導致眩暈疲勞、血管破裂等現象。因此訓練者在訓練過程中盡量避免憋氣。即使是訓練年限較高的的高素質訓練者，在進行進行最大負荷的練習時，盡量憋氣的時間不宜過長（最長1-2秒）。

3.抗阻力訓練保護帶的使用

在進行抗阻力訓練時，使用保護帶有倆個益處：

  • 增加腹內壓的壓力，
  • 減少脊柱的壓力

儘管使用保護帶對身體有益處，但並不是所有的力量練習，都要使用保護帶。過度使用保護帶

  • 會使腰背部和腹部過分地依賴它，而對軀幹的支撐作用相對減弱。
  • 一旦沒有佩戴保護帶進行練習，很容易造成損傷。

因此佩戴保護帶進行練習要遵循以下原則：

  • 在地面上進行訓練時，並且用最大負荷或接近最大負荷的重量，並且負荷作用於腰背部（深蹲、硬拉等）。
  • 雖負荷作用於腰背部，但不在垂直方向，就不用系保護帶,例如卧推、二頭彎舉等
