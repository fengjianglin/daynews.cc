2019-12-20 2,791條評論 4,437次閱讀 2328人點贊

我們在每一次練習瑜伽之前，都應該為自己的瑜伽練習做好身心的準備，找一個舒適的坐姿，穩定安靜地坐下，保持後背的挺直，如果感覺保持脊柱挺直有困難，我們可以坐在輔助用品上或靠牆；接下來，通過充分地呼吸，逐步放鬆全身，慢慢地把遊走的思緒、波動的情緒穩定下來，使我們的身體柔軟下來沒有一絲地緊繃，讓 我們的內心感覺充滿能量。

我們每個人都具足喜悅的本能，我們練習瑜伽，最重要的 一部分就是希望通過練習來啟發自己，開啟感知能力，使這種本能得以顯現。

瑜伽練習幫助我們抖落心靈的灰塵，讓愉悅回到內心。

瑜伽體式練習是幫助我們打開心扉，迎接恩典，以及我們與生俱來的善良、愛心、力量和美麗的極好的方式。

瑜伽體式練習是幫助我們打開心扉，迎接恩典，以及我們與生俱來的善良、力量和美麗的極好的方式

我們的每一次練習都應該問問自己：我今天是為什麼而練習，練習的目的是什麼？

我們要學會感恩瑜伽帶來的每一點變化，學會讚美，讓自己的每一次都是進步，去不斷消除阻礙我們的任何障礙。

也許我們練習的目的並不是必須要制定得如此高大，也許對於現實生活來說這些似乎是太抽象的概念，那麼我們可以建立最基本的目標，讓自己的這一次練習的心靈目的轉化為身體或者情緒上的改善，例如，改善我們的體態、改善彎腰駝背，改善僵硬緊繃的身體，或者釋放生活中的種種煩悶、怨氣以及低落的情緒等等。

為自己的每一次練習敞開心扉、深入感受……

我們練習瑜伽，最重要的原因就是希望通過練習使這種本能得以顯現，抖落心靈的灰塵，讓愉悅回到你的內心

這也是我們今天分享的瑜伽經典的戰士第三式，，這個練習可以幫助我們建立信心，體會頑強的意志力，感知力量與平衡。撫慰我們疲憊的心靈。

可增加專註力，加強手腕，髖關節，腿部肌肉的力量

方法：站姿，雙腿併攏，全身放鬆，慢慢向前彎曲上身，將一條腿向後抬平，手伸向前方，在一條直線上，保持挺直，保持以舒適度為準，慢慢返回，交換再做另外一條腿

保持時間：每側做三次，保持儘可能長，但以舒適度為準
