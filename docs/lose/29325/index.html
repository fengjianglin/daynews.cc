道德經里太極意：高下相傾——說說虛領與氣沉

2019-12-20 392條評論 4,675次閱讀 1956人點贊

有畫師描繪出家人勾著頭打坐，這是藝術表現的輕鬆；打坐，頭容不能中正安舒，而是勾著腰縮著脖子，會影響氣機在督脈的流暢。

成語說「垂頭喪氣」，即是因為頂勁領不起，而導致精神上的萎靡不振，以致不論技擊還是養生，效果都差強人意，所以拳家說「頂勁領起，精神百倍」；楊公澄甫則說「失掉頂頭懸，白練三十年」；王宗岳太極拳論表達得更加詳盡，說「虛領頂勁，氣沉丹田」，此是內氣鼓盪的表現，也是對內氣的陰陽要求。

「虛領頂勁」要求習拳者頭容正直，頸部放鬆上豎，下巴微斂。動作到位後，一是能感覺到腦後根微微有緊張感，二是做到虛頂後，能極其自然地做到下巴微收、唇齒相合。功久至一定境界，會感覺頭頂有一線力量上牽，尾部又有一股力量下沉，形成一對陰陽兩奪之勢。如此，打坐時則不至於腰腹出現塌垂現象，從而有利於實現松腹和氣沉丹田。

氣沉丹田，從身法上講，需要「虛領頂勁」、「含胸拔背」和「斂臀」形成配合，三者相輔相成。其體現，是在松的基礎上，通過含胸拔背和斂臀的共同作用，實現背部的對拉拔長與節節貫穿，同時實現「松腹」。松腹後，斂氣入骨，加之刻刻留心在腰間，日久自然實現了腹內松凈「氣騰然」。因而，「氣騰然」，既是丹田氣「實」的表現，也是氣沉丹田的結果。十三勢歌要求「刻刻留心在腰間，腹內松凈氣騰然」即出於此目的。

氣沉丹田，從心法上講，是神意聚于丹田的結果，這要求在練的過程中，「用意」把握在有意無意之間。因而，虛領頂勁、氣沉丹田，是三對陰陽的共同作用，一是心意上的「有意無意」，二是外形上的「對拉拔長」，三是勁力上的「高下相傾」。

實現了「虛頂」和「氣沉」，整個身體自然容易松沉，脊間也節節鬆開，實現對拉拔長和貫通，背肌則能做到松實的自然變轉。如此，脊背部能蓄能實，腰背部能松能緊，配合全身的「放長擊遠」，即可做到「力由脊發」，實現「蓄勁如張弓、發勁發放箭」。

由此可見，太極拳的所有要領相輔相成，缺一不可。

做到虛領頂勁和氣沉丹田，必須注意以下幾個方面：

一是注意「豎項」，脖子能豎起來，則頭容自然能正且直，不至於仰面俯首、左顧右盼，氣力運行自然順達無礙。

二是注意放鬆整條脊柱，腰椎尾椎放鬆似一節一節對齊上摞，頸椎摞在最上部；下部無礙，則能高屋建瓴，一以貫之，做到上下暢達無滯。

三是注意斂臀。從生理上講，尾骨是微上翹的，只有椎骨放鬆後，微將臀部內收，整條椎才能得以實現對拉拔長，頂勁自然容易上去。

此外，初習拳架，還要注意不可以低頭下視以求檢查動作的正確性，還有一些動作需要做到「斜中寓正」，忌低頭貓腰而失其頂勁，如陳式太極拳老架一路中的擊地捶勢，斜行勢等。

這是動功對外形上的要求，打坐、卧功或其他靜功，一樣需要注意。

可見，日常任何時事，皆不可以顧頭不顧腚，或是顧腚不顧頭。於做人，則是不亢不卑了。
