16歲中學生每月500個雞蛋，1年增重88斤，臂圍44cm

2019-12-24 1,090條評論 2,333次閱讀 761人點贊

大家好，我叫李蓋次

今年16歲，是一名高二學生

開始健身是因為個子矮，人也瘦小

而我喜歡運動，喜歡上體育課

所以想練壯點，不被同學排斥

爸媽一開始反對我健身

覺得會練壞身體，長不高

後來在我幾個月的哀求後

終於在17年11月

爸媽給我辦了人生第1張健身卡

從此開啟了我的健身之旅

一開始家裡也很反對我的飲食

他們認為每天只能吃1、2個雞蛋

不能吃太多肉

我雞蛋和肉吃多了會被說

水煮蛋只吃蛋白不吃蛋黃也會被說

後來我就自己買雞蛋在外面吃

慢慢的，爸媽看到我體型越來越好看了

就很少說我了

（健身3個月）

我第一次喝蛋白粉用開水泡

結果蛋白粉全結塊了

我還硬吞了

還記得有次快期末考試了

沒有時間去健身房

我就把不用的書用膠質纏住

上課坐在最後排偷偷練二頭肌彎舉

健身前我是很瘦小，只有48kg

健身對我的身材和健康的改變非常明顯：

我變得越來越壯（最壯時92kg）

也很少生病了

健身還改掉了我泡網吧打遊戲

和吃垃圾食品的壞習慣

第一次打比賽是今年的8月份

雖然沒有進前6，但還是很開心

賽場上還認識了一位跟我同齡的選手

練的特別好，人也很謙虛

我自己訓練從來不用大重量

多用中等重量（8-12RM）

拋開容量談重量，就是耍流氓

還是要看總訓練容量

所以我並沒有天賦

只是比別人練得更勤快

現在非備賽期正常飲食

保證攝入足夠的營養，才利於增肌

我每天吃16個雞蛋（蛋白）

早晚各8個

最滿意的部位暫時沒有

最不滿意的是胸部

整體還有提升空間

很多新手認為增肌期可以胡吃海喝

其實不然，過多的脂肪會影響代謝

增肌期也需要吃的乾淨和控制熱量

有條件的話最好少食多餐

我增肌期每天吃10-12頓

記得Terry老師說過

非賽季才是真正的在比賽

非賽季更要控制飲食，認真訓練

我嘛，非賽季還是做個快樂的胖子吧～

有一位好老師

你會少走很多彎路

往對的方向努力

我很幸運遇到了花木蘭姐姐

以後還有很長的路要走

現在一切才剛剛開始

騷年，加油!
