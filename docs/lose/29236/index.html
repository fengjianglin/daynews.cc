用現代搏擊手段改造形意拳是正途還是歧途

2019-12-22 964條評論 2,746次閱讀 1463人點贊

拳成大藝藝稱形意

2019-12-1123:07

今天說說傳武、武術，因為我因為我是寫形意拳的，也屬於傳武，所以用形意拳冠名。用現代搏擊理論改造形意拳是正途還是歧途？在闡述這個問題之前，必須明白一個概念。形意拳到底有沒有技擊功能？

其實這個問題沒有固定的答案，或者說每個人都有自己的答案。拋開個人認識差異，有一種聲音是鋪天蓋地的，就是形意拳缺乏動手能力。這種認識是當今大多數人的看法，同時也是外行的看法。

對任何事物的了解，都需要一個接觸探索的過程，所謂如魚飲水，冷暖自知。僅僅是從這一點來攔路，秉持以上觀點的人，99%以上可能都沒有資格說話。網路上流行兩種人，一種是看客，一種是噴子，噴子不必要論。看客分兩種，一種是事不關己，高高掛起；一種是哀其不幸，怒其不爭。

其實形意拳能不能技擊？稍微有點判斷能力的都能想到合理的答案。只是見不到實證又被輿論導向所左右。於是就發出了如上的聲音。這就如同中國足球，中國人能不能踢足球？毫無疑問，可是中國男足就不行。是我們中國人不行嗎？顯然不是。

再說乒乓球是不是外國人就打不過中國人，是外國人不具備這方面的能力嗎？顯然也不是。驗證的方法很簡單，在中國選人，放到外國的足球體制下或者在外國選人，放在中國的乒乓球隊之下。結果誰都知道，但是這件事不會有人去做。所以這只是假設，在現實中不成立。

傳武也一樣，沒有人把它當作乒乓球去做，但是卻有人把它當作足球去做。是傳武不行嗎？再舉個例子。

你不能拿投入100萬產出結果去比對投入一個億產出的結果。你也不能給西瓜澆錢，它就能長成西紅柿。所以合理的人做合理的事才會產生正確的結果。但中國人有指鹿為馬的壞毛病，我們最大的問題就在這裡。

用現代搏擊改造形意拳，就如同中國足球，自己鬧得喧，上不了行家的檯面。但是指鹿為馬是可能出現的，時間長了培育了一個市場，假的東西就被越來越多的人認可了，散打就是這樣。

散打能打嗎？窩裡鬧的喧。全民散打的時代，散打自然就能打了，這就如同太極操一樣。練太極操的，也認為自己是天下無敵啊。已經不需要指鹿為馬了，而是自覺以鹿為馬。

時間長了不認得鹿，只認得馬，見了真正的馬，以為是四不像。說你這是什麼玩意兒，你這就不是馬。在他們的眼裡，鹿才是馬。好吧，你說這是馬，那我們牽到台上跳一跳吧。就是真的馬，也跳不過假馬的鹿啊。所以考量馬的舞台不應該是跳台。

同理考量傳武的也不應該是現代擂台。真正的格鬥是沒有舞台的。沒有舞台會產生兩種結果，一種就可以適應各種的舞台，一種是哪一個舞台也適應不了。能適應各種舞台的是因為你把所有的方面都做到了。不能適應各種舞台，因為你哪個方面也做的不到位。

事實上，專註於一點容易成功而形意拳是個多元化的東西，就如同乒乓球形並不能代表中國體育也那麼強大，這是往大的說。往小了說，發球好不一定就能贏了比賽，你不可能光靠練發球去奪取冠軍。武術也一樣，你不可能光靠站樁，就能取得台上的勝利。

這就如同外國人打乒乓球，不是打不過中國人，而是不知道中國人怎麼練的。學了個發球，以為厲害了，結果一比賽不是那麼回事兒。但是他們會承認自己打球不行嗎？當然不會，他們會想盡一切辦法學到中國的技術，當然他們學不全，所以幾乎永遠也贏不了。他們是不是可以得出一個結論，中國乒乓球不行。不是不行是他們沒學會。

武術就如同中國乒乓球，拳家就如同外國人，想贏球你得先學到技術，學不到贏不了人就是中國拳術不行。那就如同說中國乒乓球不行一樣可笑。不是說你也按鄧亞萍的打法練你就能贏，贏不了是你沒練到，而不是證明鄧亞萍是假的。打不贏人家，就說中國拳術不行，就說郭雲深、孫祿堂是假的，傳武從來都是假的。豈不一樣可笑。

今天胡亂說一通，傳武的沒落在於規則的改變，只讓你練發球，講究的是好看，你是贏不了比賽的。若干年後，有人說我們要跟外國人學打球，拿擂台上的現代技擊手段來改造傳武就如同跟外國人學打乒乓球。
