西北棍法四大要訣，師傅不傳，徒弟想方設法也要得到手

2019-12-11 458條評論 1,603次閱讀 687人點贊

西北棍法中，有四大要訣，也就是所謂的四大秘密。在過去的棍術家眼中，均被視為不傳之秘密，也是學習者夢寐以求的東西。這就是：

纏絲扭絲,槍棍合一

掃、劈是棍的主要技法，纏絲、扭絲卻是棍之精髓。一個以剛為主，呈現狂掃猛劈的瘋魔勁；一個是以柔克剛，呈現棉里藏針的纏絲勁。

此外拳諺中說：「棍如槍,硬如鋼。」這句話充分說明，西棍當中槍棍合一的技擊內涵。

棍的形制決定了其技法特點

西北棍術主要用法以掄擊掃劈為主。因無槍頭，扎刺遠不如槍。而棍採擷槍中纏絲、扭絲動作就較完整
地體現了「陰陽要轉」、「左右要明」、「纏絲帶扎」、「扭絲帶挑」、「弧線防守」、「直線進政」、「以點觸擊」、「輪軸相配」、「左右相容」、「梢不離圓」、「圈中帶扎」、「出奇制勝」的原則。

為了控制對方，利於攻擊，往往要使用粘纏圈轉法。使棍尖作圓周運動。棍尖運轉的圓圈大小要適當。如果圈點太小，雖然運棍速度快，但扣殺點擊的勁力不強；如果圈點太大，則運棍速度緩慢，容易被對方閃避。關於圈的大小，戚繼光在他所著的《紀效新書》中認為直徑為一尺的圈點最佳。

戚繼光雖然講的是槍，但與棍的道理也相同。運棍的圈大圈小以能夠發出足夠的扣擊勁力為主。圈小勝圈大，因為圈小則速度快。

在技擊中，有以迅雷不及掩耳之勢先發制人的方法，也有後發先至的方法。

如何做到先發制人與後發先至?

舉例說明：棍法中的「雲里撥燈」就是先發制人方法。我用「雲里撥燈」招法快速點擊彼太陽穴，彼起棍必防，如不防就有被擊中的可能。彼起棍防，我乘兩棍相碰之隙，迅速回棍擊打彼腿，彼回棍向下遮攔，
我迅速用勾魂樁法擊其頭部及持棍前手，墊步棍入彼襠，挑擊陰部，這是引上打下再打上的綜合棍法。

這就是先發制人的方法。

先發制人與後發先至的關鍵點是什麼？

關鍵在於速度「棉里藏針」是後發先至的方法。我右手在前，左手持棍把，側身應敵。彼用中平槍直刺我胸部，我往左移步，翻腕扣住對方器械。上右步再扣腕點擊執棍前手。進而前刺殺身。「棉里藏針」講究「手前一尺下功夫」。這就是後發先至的招法。關鍵還是在於速度。
