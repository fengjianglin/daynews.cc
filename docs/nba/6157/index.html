詹眉踏破爵士，庫茲馬4斷3冒贏25分賽季最佳，更打亂魔獸上位計劃

2019-12-05 0條評論 21次閱讀 0人點贊

戴維斯，庫茲馬等多人生病，還是背靠背第二場，但湖人在同爵士的客場比賽中不僅沒有疲態，好狀態反而讓主隊絕望。121比96，紫金軍團毫不費力的狂勝，將自己的戰績提升到19勝3負，繼續和雄鹿一起領跑聯盟。

詹皇此役中規中矩拿下20分12助攻，戴維斯11中9得到全隊最高的26分6籃板。而湖人本場表現最出彩的卻是庫茲馬，他打出了本賽季的最佳表現，拿下賽季最高的4搶斷3蓋帽，以及全隊最高的+25的勝負分。

庫茲馬過去三場比賽都處於夢遊狀態，21投僅5中，一共只得到14分。而與此同時，德懷特·霍華德卻完成了質變，以至於主教練沃格爾在考慮讓他搭檔濃眉，並希望濃眉去為魔獸進行調整。

本就防守強硬的霍華德，在擋拆進攻上的進化是讓他獲得沃格爾青睞的關鍵。對此，勒布朗也給出極高的評價，「德懷特的職業生涯，他都是那個在低位拿球，然後所有人圍著他的風格。而現在，他是進攻機會的製造者，擋拆者，現在如果你將球給他，他就能成為進攻端的催化劑。他可以將球傳給後衛，然後擋住他的對手，他現在做的太傑出了。這是一種思想模式的進階，對整支球隊都是如此。」

而在談及主教練沃格爾所提出的讓濃眉和霍華德搭檔組成新雙塔時，詹姆斯說，「我認為AD必須要變，他需要在不舒服的環境中再找到舒適感。這也是一種思想模式的突破，你必須去做一些你過去不曾做過的事，因為這對球隊有利，對他自己也有利，我們都需要有這樣的突破精神。」

在沃格爾「防守第一」的建隊方針下，庫茲馬成為了本賽季湖人最大的犧牲者，場均出場時間下降了近10分鐘。雖然他也很努力在防守，但就像庫茲馬自己所說的，進攻才是他最能為球隊帶來幫助的點。當然，如果庫茲馬能像霍華德那樣在賽季中完成進化，他和戴維斯的搭檔無疑是更好的。而同爵士比賽在防守端的爆發至少證明，他有這樣的可能。而在同爵士的比賽中，庫茲馬的確依靠防守拿到了他本賽季最長的31分鐘出場時間，而霍華德只有19分鐘。
