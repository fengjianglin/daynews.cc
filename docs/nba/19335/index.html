記錄就是用來打破的？聯盟中的這三個記錄，只能仰望無法被打破！

2019-12-18 692條評論 3,938次閱讀 1689人點贊

NBA是一個能夠讓你無限聯想的籃球世界，在這裡有很多你能夠想到或者你根本想不到的事情在不斷的發生。在聯盟中有很多記錄，隨著時間的推移和規則的變化，許多記錄已經不止一次被刷新，而有些記錄則離打破越來越遙遠。

第一個是拉塞爾，他擁有十一枚總冠軍戒指，那個年代NBA球隊一共才八九支，所以奪冠幾率還是很大的。但是現在呢？三十支球隊，所以如今的球員們想要破老爺子這個記錄，簡直比登天還要難。

第二個喬丹的10個得分王，很多球員在生涯結束後都不知道得分王的獎盃長啥樣，自己在職業生涯能夠得到一個得分王啊，那就已經很了不得了。現役的得分王最多的還沒喬老爺的一半，而且喬丹職業生涯平均三十多分，這倆記錄都是很難打破的存在。

最後就是麥迪35秒13分了，麥蒂在比賽時間的最後35秒，投中4記三分球，其中包括一球3+1，共得13分，逆轉了與馬刺8分的分差，最終呢也是成功帶領球隊戰勝馬刺，波波老爺做夢也沒想到，如此大的優勢也能被翻吧！

美哪深夜發朋友圈，爆出來了一個震撼的內幕！希望與大家在裡面邂逅！
