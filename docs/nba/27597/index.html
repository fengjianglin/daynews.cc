NBA聯盟向30支球隊發出21-22賽季變，大變革真的來了？

2019-12-22 150條評論 1,354次閱讀 219人點贊

著名NBA媒體記者查拉尼亞報道，NBA聯盟已經向30支球隊發出變更議案，提出了未來2021-2022賽季進行改革的幾項建議；主要是四大方面：

  • 常規賽由現在的82場比賽縮減至78場
  • 額外增加賽季中錦標賽，具體議案內容是：在賽季中的11月下旬至12月中旬，NBA30支球隊都參與其中。小組賽分區比賽（分別分為4個主場賽區和4個客場賽區，然後產生6個分區冠軍以及2張外卡名額，四分之一決賽將於主場賽區進行，半決賽以及決賽將於中立球場進行。奪冠球隊每個球員可獲得100萬美金，教練獲得150萬獎金.
  • 增加NBA季後賽附加賽：即東（西）部各排名第七、八名球隊與第九、十名球隊進行附加比賽，最後勝出的兩支球隊進入季後賽。
  • 季後賽四強將根據常規賽的最佳戰績進行排名

一、首先看看常規賽由現在的82場比賽縮減至78場的建議

最近兩賽季倫納德的輪休引來無數爭議，尤其本賽季，關於NBA球員的負荷管理更是各方議論紛紛。82場比賽中間難免存在賽季密集的情況，這樣會增加球員的傷病；甚至湖人當家球星詹姆斯也曾建議減少賽程安排。

如果常規賽真能減少至78場，會不會存在一些遺憾，各項NBA單賽季記錄是不是將成為歷史？像單賽季最高總得分、總籃板、總助攻等，甚至詹姆斯破歷史第一得分記錄也可能性更小。像勇士的單賽季73勝也更加不可能了？

二、額外增加賽季中錦標賽

季中錦標賽有點像歐洲足球的杯賽，但歐洲足球杯賽往往球重視程度不高，所以足球杯賽往往可以看到強隊爆冷屬於弱隊，而球隊也多以歷練年輕球員較多。目前本賽季的NBA收視率明顯下降不少，引入季中錦標賽，增加聯盟關注度或收視率存在這樣的可能性。但如果足球杯賽一樣，NBA各球隊不太重視，多以替補球員或年輕球員；而且這與球員希望負荷管理也存在矛盾。如果實施對於旨在總冠軍的球隊會不會形同虛設？會不會聯盟設置更加明確的輪休制度以杜絕季中錦標賽成為擺設？

三、增加NBA季後賽附加賽

此項議案倒可能吸引NBA各球隊同意，這項建議也有點像足球中升級附加賽。之所以通過可能性大些，對於像競爭激烈的西部，往往球隊之間的戰績相差不大；也更增加靠後球隊（9名之後的球隊）的求勝慾望，即使常規賽進不了前八，至少常規賽可以爭奪9-10名，再通過附加賽爭奪季後賽席位；這樣也更能增加常規賽比賽的競爭力和觀賞性。

季後賽四強排名與現在賽制排名差別不大不再贅述不過這樣會不會不再進行第一對陣第八、第二對陣第七的局面，那樣黑八奇蹟是否也將消失？

對於這幾項的提議，畢竟根據NBA規定，需要通過三分之二及以上球隊及NBA球員工會同意。如果通過將引起怎樣的軒然大波？大家對此幾項提議，都有什麼看法呢？
