若湖人休賽期不等待倫納德，他們又能得到誰呢？

2019-12-25 1,487條評論 3,600次閱讀 2446人點贊

本賽季NBA已經進行了三分之一，湖人戰績很優秀，一直霸佔著西部第一的位置。然而最近剛剛早遇到三連敗，他們一遇到輸球，就開始說夏天為了等倫納德，錯過了補強。只有這些人了，當然只能打成這樣，詹姆斯或者濃眉少一個就不行。

咱們能不能回顧一下夏天的自由球員簽約簽約情況？

倫納德當地時間7月6日宣布簽約快船，自由市場7月1日開啟，中間拖了6天的時間。因為倫納德陣營需要會見球隊，也就是說無論如何也不可能在開市當天確定，這件事情參與的球隊都是了解的。市場開啟之後，1天，大量的口頭協議達成。

以上都是市場開啟之後1天之內確定的交易或者自由市場口頭協議。這個時候市場上還有誰？

不就只剩下林書豪、莫里斯、阿米爾-約翰遜、達倫-科里森、賈馬爾-克勞福德、賈里德-杜德利、諾阿-馮萊、卡巴烏、梅傑里、薩夫洛沙、尤度、法里埃德、米爾斯、傑雷布科、布魯爾、庫弗斯、特雷-伯克、泰瑞克-埃文斯、泰勒-澤勒、卡斯比、門羅、貝勒斯、卡特、扎扎、傑夫-格林、哈里斯、米羅蒂奇、莫泰尤納斯、馬里亞諾維奇、大加索爾、威爾森-錢德勒。

還有誰？這些人裡面我們想要誰？現在讓我們重選一邊，這些人我們想要誰？米爾斯還是傑夫-格林？

最多湖人有機會搶掉太陽的烏佈雷，2年3000萬的受限自由球員。他們願意給1800萬往上的年薪嗎？他是7月17日才簽約的。

大家有情緒我能理解，但是請不要不顧事實。大家埋怨倫納德耽誤湖人補強也可以，請說說，如果倫納德不拖拉湖人，他們本可以有誰？
