聚焦｜紐約尼克斯，誰才能真正拯救這支球隊？

天天要聞
2019-12-10

寫在前言：一說到尼克斯這支球隊，最直接的印象就是這支球隊近些年的發展並不出色，尤其是在球隊競爭力方面，自2012-2013賽季之後，他們就沒有進入過季後賽球隊，雖然麥迪遜花園球館被超級球星視為「籃球聖地」，但是對於擁有這座球館的主隊，卻更像一支平庸籃球隊，說實話除了90年代短暫的光榮，球隊已經很多年都沒有改觀，那麼尼克斯的發展眼光到底如何？

球隊時間線，見證球隊從巔峰到低谷

尼克斯的落寞我歸結於1999年詹姆斯-多蘭接手球隊和總經理雷登的一系列交易操作，原本尼克斯是一支非常強硬的球隊，在范甘迪的調教下尼克斯的防守堪稱聯盟頂級，尤因仍能提供不錯的得分效率，此外球隊還有中投王休斯頓和拉里-約翰遜這兩個穩定得分點，在那個年代，可以說尼克斯是一支勁旅。

尤因

但是球隊選人眼光、運作思路和球隊交易都是災難級別的表現，這也是尼克斯為何能從總決賽級別球隊淪落到「魚腩」球隊的原因。在他正式接手球隊後，1999年選秀大會用15號簽選擇了法國中鋒維斯，他從來沒打過NBA比賽，如果你對於不熟悉，那麼卡特在奧運會的驚天一扣，背景牆就是他。

卡特隔扣維斯

1999-2000賽季球隊止步東部決賽，這也是球隊最後一次踏進分區決賽的舞台，尤因場上表現已經不符合尼克斯預期標準，但是尤因的合同在當時不算小，總經理雷登通過四方交易，將尤因送走超音速，並將達德利和首輪簽送給了太陽隊，換來了一些基本得不到上場時間的球員。之後尼克斯的選人眼光才是真正讓球隊一步步陷入谷底的開始，通過選秀大會選擇了出戰0場的賴特和埃里克維諾斯，那一年的選秀大會他錯過的球員有「黑熊」蘭多夫、托尼-帕克、阿里納斯和「貓王」傑拉德-華萊士等球員。

阿蘭-休斯頓

除了選秀大會以外，雷登休賽期對於球員的續約選擇問題，影響了尼克斯未來幾年的發展。尼克斯隊史上第一份億元級別合同就是雷登提出來的，用6年1億美元合同續約休斯頓，這一個操作直接填滿球隊的操作空間，聯盟後來出現的「特赦合同」就是由此而來。

在2002年選秀大會上尼克斯選擇了中鋒內內，緊接著球隊就把內內和坎比送給了掘金，換來有潛在傷病隱患麥克戴斯，果不其然麥克戴斯在新賽季受傷，賽季報銷。2003年又是一個選秀大年，很多球隊都執著於擺爛等待好的簽位，尼克斯並不甘心，37勝45負的戰績還是沒有闖入季後賽，同時簽位也非常尷尬，用9號簽選中了斯威特尼。如果尼克斯在這個賽季選擇擺爛，能夠得到韋德、安東尼或者詹姆斯其中一人的話，尼克斯的命運可能會觸底反彈。

總經理托馬斯

雷登擔任尼克斯總經理的這幾年，對於球隊只有傷害。他用獨特的眼光和頭腦，一步步沿著他布置的軌跡走向「魚腩」球隊行列。之後「微笑刺客」托馬斯接任總經理職務後，受限於當初休斯頓的大合同，球隊可操作的空間並不是很大，托馬斯還是換來了馬布里和哈達威，讓球隊成功打進季後賽。相比於雷登，托馬斯的選人眼光明顯出色不少，選中了弗萊（8號順位）和大衛-李（30號順位）。整體來說他算是個過渡者，讓尼克斯從谷底慢慢過渡出來。

隨後沃爾什的加盟，才讓尼克斯從過渡期緩和到季後賽球隊，他最出色的操作莫過於安東尼和比盧普斯攜手加盟尼克斯，這一次的交易讓尼克斯連續三年闖入季後賽，尼克斯算是從泥潭裡慢慢「折騰」出來。但是這些年選人眼光同樣讓人感到失望，2009年用8號簽選中喬丹-希爾（後邊球員有德羅贊和霍勒迪兩位建隊核心級別球員），2011年用17號簽選中伊曼-香波特（後面有托比厄斯-哈里斯和巴特勒），2013年24號簽選中小哈達威（後面有防守最佳球員戈貝爾），2017年用8號簽選中尼利基納（身後有13號順位的米切爾），尼克斯不斷流失掉建隊核心級別的球員，並且因為交易操作，在2010年、2012年和2014年甚至都沒有首輪簽，當手上沒有牌的時候，更不用想如何重建球隊了。

尼克斯老闆：多蘭

我們回顧尼克斯本世紀的現狀，在辭掉菲茲戴爾下課後，這支球隊已經更換了13任主教練，但是只拿到了9場季後賽勝利。最終的源頭還是來自於老闆多蘭，執掌尼克斯的這些年裡他經常插手總經理的事物，以至於球迷都非常痛恨這位容易頭腦發熱的老闆。

球隊近兩年發展如何？

尼克斯再一次失去競爭力的時間點，我認為要以安東尼加盟雷霆為界限。安東尼在尼克斯的這些年裡，幫助尼克斯連續三年打入季後賽，雖然生涯後期並沒有幫助球隊打入季後賽，但是對於球隊安東尼還是非常敬業，等來了波爾津吉斯後，原本認為新老聯合會將尼克斯帶入季後賽，但是合作兩年後球隊依然在季後賽外徘徊，安東尼選擇前往俄城，去爭取總冠軍的夢想，波爾津吉斯成為「紐約新王」，賽季中期受傷導致報銷，在交易截止日前一周交易走波爾津吉斯，讓球隊徹底打破重建。

安東尼

這幾年尼克斯的交易操作依然讓人看不懂，今年休賽期他們送走了穆迪埃、小喬丹和海佐尼亞等球員，在歐文和杜蘭特都是自由球員前提下，尼克斯錯過了一次讓球隊重返巔峰的機會，介於杜蘭特的跟腱傷病，尼克斯並沒有拿出誠意給他一份頂薪合同，而是選擇簽下了蘭德爾、吉布森、波蒂斯和佩頓等球員，除了蘭德爾有不錯的潛力以外，吉布森和波蒂斯都是即戰力球員，可以即插即用補充球隊陣容完整，但是對於一支志在重建的球隊來說，這兩筆簽約還是讓人看不懂。

尼克斯球員數據

我們來看一下本賽季尼克斯的各項數據，進攻端他們是聯盟效率最差的球隊，場均得分100.6進攻效率102都在聯盟墊底，戰績倒數第二的勇士隊，場均得分還要比他們場均得分多出4.7分，同時勇士的防守效率在慢慢得到改善，本賽季勇士防守效率為113.4，而近三場的防守效率提升到了109.6，這是一個積極的信號，但是對於尼克斯來講，缺少合格的進攻發起點和梳理點是目前最大的問題，他們場均20助攻同樣聯盟墊底。

場上主要輪換陣容

從場上表現來看，在超過上場20分鐘的7組陣容裡面，僅有兩組陣容上場凈勝分是處於贏球階段的，這也意味著基本上球員在場上，大部分都是處於輸球狀態。可以看到尼克斯的防守就像「空勺」一樣，沒有任何的防守連續性，他們的防守效率112.5排在聯盟倒數第7位，菲茲代爾的大多數防守體系都建立在保護籃下和逢掩護就換人進行，偶爾高位擋拆用包夾戰術來防守對面後衛，逼迫失誤或者提前出球，但是效果並不算出色，進攻端傳切和擋拆是主要進攻方式，盲目效仿之前勇士的打法，在球員沒有出色投射能力球員的時候，這種進攻方式並不可行。攻防兩端同時存在問題，尼克斯想要復興幾乎是不可能的事情。

主帥：菲茲戴爾

今夏引援球員中蘭德爾來到尼克斯後，正打出他職業生涯近三年里最低迷的一個賽季，攻防兩端效率完全下降，一方面受到球員因素綜合影響，另一方面蘭德爾的戰術地位並不如鵜鶘，雖然場均出手數下降不足1次，但是他的出手範圍從8.3次擴大到了10.6次，三分佔比總出手的25.0%。出手距離的改變，帶來的影響是籃下命中率和真實命中率都是近三年的新低，不得不改變自己的打法，去更好的適應球隊。

蘭德爾

主帥更替，淺談下一任主帥

上面我們提到了進入新世紀以來，球隊更換了13名主教練，這比他們季後賽贏球場次還要多。菲茲戴爾的下課，我認為對於他來說其實是有利的，在他執教時期，尼克斯只取得21勝83負的戰績，勝率在尼克斯隊史29名主帥中排名倒數第一。雖然戰績不夠出色，但是很多外界聲音都認為這並不是他的責任，管理層在過去一個夏天，手握巨大薪資空間的尼克斯並沒有簽下一位大牌球員，而是囤積了一批用不完的鋒線，這讓菲茲戴爾沒有完整的陣容資源去支撐場上陣容，尼克斯戰績位列聯盟倒數，菲茲戴爾只不過是背了黑鍋。

菲茲戴爾

對於下任主帥，基德是尼克斯的目標主帥之一，在我看來如果尼克斯真的想要復興，必須在多蘭不干預球隊事物的前提下，全面推倒重建培養年輕球員，這是基德非常擅長做的事情，字母哥在生涯第四個賽季兌現天賦，其中很大原因要感謝基德對於他的培養，當時基德任雄鹿主帥時看重字母哥的天賦，雖然他還是個替補，但是基德一直視他為核心培養，同時讓他觀看詹姆斯的錄像學習打法，這也是為什麼字母哥從基德上任後，助攻端從場均2.6次猛漲到4.3次的原因，他的表現每年都是改善和進步。

基德

不得不說基德在發展年輕球員方面，有自己獨特的執教方式，對於尼克斯來說，聚集尼利基納、巴雷特、波蒂斯、羅賓遜和諾克斯等大量年輕潛力球員後，如何兌現天賦才是讓球隊走上正軌的方法。基德目前在湖人教練組效力，他心中嚮往的還是主教練的位置，如果能夠聘請基德作為新一任主帥，對於年輕球員生涯發展很有幫助。未來方向，既然管理層不捨得給明星球員頂薪合同，那麼放棄對於自由球員的爭奪，保留未來幾年的首輪簽，才是尼克斯從「魚腩」球隊過渡季後賽邊緣球隊的正確道路。

RJ-巴雷特，能夠成為球隊未來核心嗎？

尼克斯在選秀大會沒有足夠好的運氣獲得狀元簽，他們只能選擇錫安大學時期隊友巴雷特。從場上表現來看，巴雷特控制失誤能力還需要增強，並且他在場上時絕大多數以中距離和突破為主要進攻選擇，投籃命中率39.5%很難達到球迷預期的標準，作為球隊的第一持球人，他會得到很多衝擊籃下的機會，自行終結還是給隊友創造機會，他的進攻選擇還需要提高，場均3.3次助攻與23.6%的使用率並不成正比，此外巴雷特籃下的終結能力還需要提高，大學時期他籃下出手238次的命中率高達65.13%，但是進入聯盟後151次的籃下出手，僅換來了48.1%的命中率，其中外線命中率的低迷讓對手在防守選擇傾向於限制突破，他若想成為球隊核心，必須提高他的投射能力和終結球隊能力，這是目前他急需做的事情，不然他的新秀牆真可能「撞」不破，成長為球隊核心的幾率也會大大降低。

RJ-巴雷特

相比於賈-莫蘭特，巴雷特顯得有些星光暗淡，前者在灰熊首個賽季就確立了圍繞他的核心打法，並且外界詬病的三分投籃能力，也增長到了41.0%的命中率，莫蘭特對於自己的提升有著明確的目標，並且他在比賽中會如何運用自身天賦，經常可以看到他利用速度優勢突破，並在合適機會尋找投籃機會，現在來說中距離投籃命中率45.7%，禁區命中率61%也算不錯。他是一名出手的組織型球員，助攻率36.6%在新秀排行中獨一檔的存在，第二名納恩助攻率僅有18.7%，可想而知在生涯的第一年，莫蘭特就用自己的防守去影響球隊，如果能夠提高他的得分能力，有理由相信他將會是新秀里首位入選全明星的球員。

尼克斯若想擺脫「魚腩」球隊現狀，必須要沉穩下來徹底重建，不管過程有多痛苦，能否抵達終點才是人們關心的問題。（文/路威的體育世界）
