阿德交易方案出爐！美媒曝開拓者欲組四巨頭抗衡湖人，白邊成籌碼

天天要聞
2019-12-03

本賽季最讓人感到失望的球隊，第一時間肯定會想到開拓者，這支西部勁旅擁有利拉德、邁克勒姆、安東尼等球員，卻打出了8勝12負的糟糕戰績，目前排名西部第11。按照這個勢頭髮展下去，開拓者極有可能無緣季後賽，然而球隊高管並不打算坐以待斃，他們謀劃新一輪的引援工作。

美媒Fansided報道，開拓者在今年夏天得到懷特塞德，認為這位明星內線是球隊長期規劃的一員，但新賽季白邊始終無法與利拉德、邁克勒姆兼容，開拓者也沒想到戰績居然離譜到這個地步。另一方面，主力中鋒努爾基奇預計恢復情況良好，預計幾個月後就能歸隊，因此不排除將白邊送到其他球隊。

一位聯盟高管向Fansided透露，開拓者一直與馬刺探討阿爾德里奇的交易方案，開拓者願意提供懷特塞德和一個2020年首輪選秀權作為籌碼，這樣的籌碼誠意十足。波特蘭極有可能是阿德的目的地，他的職業生涯就是在這裡開始的，重回開拓者似乎是一個不錯的選擇。

Fansided還表示，開拓者本賽季內線進攻乏力，他們急需一個可以持球進攻，並且能夠在中距離提供火力支援的大個子內線，這樣可以為後場雙槍以及安東尼緩解壓力。而且當努爾基奇傷愈歸來，開拓者可以將努爾基奇和阿德一起放進首發陣容，這對前場組合可以給任何對手造成殺傷。

過去20年來，馬刺作為聯盟傳統豪取球隊，如今他們遭遇了最尷尬的時期，目前以7勝14負的戰績排名西部第12。比戰績更可怕的是，在8連敗期間馬刺被各種交易流言纏身，以阿爾德里奇和德羅贊為首，外界紛紛要求交易雙德組合，認為他們並不是馬刺追求的答案。

如果真如這位高管說的那樣，開拓者可以組成利拉德、阿德、邁克勒姆、安東尼的四巨頭組合，在四巨頭的帶領下，開拓者甚至可以和湖人快船等球隊爭奪西部頭牌的位置。而馬刺則收穫了內線保障白邊以及球隊重建的首輪選秀權，這能有效幫助他們走出失敗的泥潭。
