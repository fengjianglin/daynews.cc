單核獨行俠阻擋不了字母哥，卻擋住了雄鹿的連勝，靠什麼？

2019-12-18 167條評論 1,203次閱讀 64人點贊

序言：北京時間12月17日，已經18連勝的雄鹿隊在主場迎戰獨行俠隊，東契奇的因傷缺陣讓很多人認為本場比賽失去了很多看點，儘管雄鹿隊的當家控衛布拉德索同樣因傷缺陣，眾人還是相信雄鹿隊的陣容深度和作戰能力，先入為主地把阻擋雄鹿連勝腳步的任務交給了即將在20號對陣的湖人隊身上，而兩位東西部頭名的對抗無疑具有著更多的看點，甚至媒體已經開始宣傳二者的對抗是總決賽的預演。但是低調且強硬的獨行俠隊偏偏不信這個邪，卡萊爾帶領眾將士用無比精彩的表現在費哲廣場帶走了一場勝利。比分最終定格在120:116，比賽的最後時刻，蒙著毛巾的字母哥無比沮喪，而坐在身邊的米德爾頓面無表情，這一幕成為了本場比賽的一個剪影。

獨行俠VS雄鹿

在小編看來，本場比賽獨行俠能夠取得最後的勝利離不開以下幾位的貢獻

第一位：獨行俠主帥卡萊爾。整場比賽，卡帥堅持自己的防守策略，即一對一防守字母哥，在上線不選擇包夾協防，側重點在於壓制對手的三分球，不讓雄鹿隊的三分投開，只有到了很深的位置，獨行俠隊才會採取收縮協防，不惜付出內線球員的犯規作為代價。從比賽結果上看，卡帥的策略成功了，字母哥打出了極為精彩的個人數據，全場48分14籃板4次助攻，在內線予取予求投中17球，作為主要防守人，鮑威爾5犯，克勒貝爾6犯，芬妮史密斯4犯，字母哥罰球16次。在18連勝期間，雄鹿隊的三分球命中率高達41%，但是本場比賽全隊三分球41中11，命中率只有27%，如果不是老將科沃爾的7中5，雄鹿隊的三分會更加慘淡。

獨行俠教父：里克卡萊爾

雄鹿隊的防守一直秉持著收縮籃下，保護禁區，考驗對手的三分手感的反潮流宗旨，作為聯盟里最出色的三分投射大隊之一，獨行俠隊用精準的三分球懲罰了這種防守，全場比賽獨行俠隊41投16中的三分讓雄鹿隊的防守失去了以往的魔力，同時卡帥強調了擋拆後中距離投籃的重要性，庫里，布倫森和鉑金一次次選擇中距離跳投投中考驗著雄鹿隊的防守耐心，導致的結果是布帥直接在下半場放棄了洛佩斯這個最好的禁區守護者，率先改變了球隊的防守戰術，這種被迫性的戰術改變為本場比賽的結局奠定了基礎。

第二位：波爾津吉斯的反彈。在本場比賽之前，波爾津吉斯的表現一直不溫不火，四成出頭的投籃命中率和三成五的三分命中率只能用糟糕形容，完全喪失了傷前投射型大前的准心，難免會被掩蓋在東契奇的光芒之下。本場比賽獨自帶隊的拉脫維亞球星在攻防兩端都發揮出了極佳的水平，全場貢獻26分12籃板4助攻2次封蓋，正負值是全隊最高的+20，第一節獨行俠之所以能取得10分以上的領先，波爾津吉斯的表現居功至偉，11分4籃板2助攻的單節輸出打了雄鹿隊一個措手不及。儘管獨行俠一度遭遇雄鹿隊的強勢反撲，但是在比賽的關鍵時刻，鉑金兩記三分直接打停了比賽，其中包括一記標誌性的LOGO shot。在防守端，他的禁區守護和身高優勢是獨行俠隊最後一道屏障，這也是本賽季他一直被忽略的輸出，在與3D中鋒大洛佩斯的對抗中，鉑金在攻防兩端完爆對手，逐漸找回狀態的鉑金也不好惹，一記隔著字母哥的補扣說明了他的身體天賦依舊存在，他的表現無愧於本場比賽MVP。

鉑金閃耀密爾沃基

第三位：賽斯庫里在第二陣容里的進攻表現。賽斯庫里全場貢獻26分5籃板4助攻，其中下半場貢獻18分。在第三節的開始階段，雄鹿隊展開了強勢反撲，逐漸縮小比分差距，甚至依靠字母哥一記輕巧的搶斷上籃反超了比分，但是此時卡帥換上了庫里，庫里利用速度優勢打亂了雄鹿隊的防守陣型，同時良好的手感和自信的出手不斷懲罰著對手，一上場就起到了立竿見影的效果，連得9分，讓獨行俠在第三節結束依舊有著10分的領先優勢，到了第四節，卡帥依舊信任庫里，庫里也用7分的貢獻回報了這種信任。由於庫里的存在，獨行俠隊的替補得分56:45領先雄鹿。其中鉑金做出投籃假動作然後回傳庫里命中三分被評為本場比賽最佳三分表現。

塞斯庫里：聯盟里註定有我的位置

第四位：小編要說的最後兩位是代替東契奇出任首發控衛的布倫森和防守工兵芬尼史密斯。雖然沒有打出驚艷的數據，13分11助攻的經濟型兩雙數據，但是他在比賽節奏控制方面很好的完成了卡帥的任務，他給芬妮史密斯的兩個神級傳球可見一斑，一個是直接突破至籃下通過斜45度線傳至三分線外的芬尼史密斯完成一次成功的三分投射，還有一個是第四節籃下極小角度下背傳芬妮史密斯完成籃下2+1。這位23歲的後衛球風穩健，極少失誤，雖然缺乏一些驚艷的傳球，但是對於控制節奏方面也有自己的獨到之處，同時他所自帶的外線投射能力更讓人眼前一亮。本場比賽僅出現2次失誤，投中2記擋拆後的中距離跳投和1記三分。

布倫森會是巴里亞的接班人嗎？

作為獨行俠隊最好的外線防守球員，芬尼史密斯的作用一直被低估，效力於獨行俠隊的四個賽季都在不斷進步，本賽季投籃命中率達到了47.1%，三分球命中率達到了34.7%，他已經成為東契奇身邊一個可靠的角色，已經顯現出3D特質。令人印象深刻的是，在本賽季獨行俠對陣湖人的第一場比賽最後時刻，芬妮史密斯在常規時間的最後時刻接到東契奇的突破分球底角命中三分，如果不是最後丹尼格林的超神三分，這是足以殺死比賽的三分。他的三分球越來越穩定，尤其是關鍵時刻絕不手軟，同時超長臂展讓他在側翼防守方面擁有著先天的優勢，敏捷的移動速度和良好的防守意識讓他深受卡帥的喜歡。本場比賽11中6 ，三分球5中3，得到15分8籃板。

獨行俠隊淘寶案例之一：芬尼史密斯

總之，一場比賽的輸贏充滿了眾多偶然性，雄鹿隊本場比賽的手感確實冰涼，二當家米德爾頓全場12中4，三分球5中1的發揮確實有失水準。希望卸下連勝包袱的雄鹿重整旗鼓，準備下一場與湖人的大戰。
