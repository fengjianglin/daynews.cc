三個水貨一台戲！不簽巨星，不養新秀，這支球隊何時能崛起？

2019-12-03 0條評論 16次閱讀 0人點贊

距離紐約上一次進入季後賽，已經過去六年。

這六年間，尼克斯聘請了禪師，又把他送走了；續約了甜瓜，再把他交易了；選中波爾津吉斯，也沒將他留下。幾通窒息操作之後，尼克斯自然而然走上了重建之路，為此尼克斯如同所有重建球隊一樣，擺爛、騰空間、收集選秀權、交易年輕人。

所謂重建，無非也就兩條路。

一是簽大牌，這種方法適用於擁有大市場的球隊，成功的例子比如湖人，在得到詹姆斯和濃眉之後，湖人本賽季一飛衝天。尼克斯正是這類，他們坐擁紐約這個全球最大市場，可惜結果我們都知道，今年夏天，尼克斯不僅沒招來一個巨星，還簽下了一堆大前鋒，操作完全讓人看不懂。

二是培養新秀，這種方法適用於所有球隊，尤其是小市場球隊，比如鵜鶘、雄鹿、騎士、雷霆、黃蜂和灰熊這種幾乎沒有大牌願意前往的地方，想要崛起，只能自己培養。眼見著要成功的例子是灰熊，榜眼莫蘭特在今年的表現可謂相當驚喜，灰熊大可以他為基石建隊。

對於大市場球隊來說，一般他們的重建會選擇「簽大牌」和「培養新人」兩步同時進行。比如本賽季的熱火，巴特勒的到來讓熱火躋身季後賽行列，新秀們的崛起則成為熱火的最大驚喜，並且因此成為東部強隊。

尼克斯沒有得到巨星，崛起自然只能靠培養新秀。

得益於此前的擺爛和交易，尼克斯還是擁有一些年輕球員的，比如巴雷特，作為今年的探花秀，巴雷特賽季至今場均可以拿到15.4分5.5籃板3.3助攻，他偶爾能有一些高光表現，5次得分超過20分，不過巴雷特的整體效率並不高，真實命中率只有46.9%——效率差幾乎是所有新秀球員的通病，對尼克斯而言，這樣的巴雷特還是可接受的。

除了巴雷特之外，尼克斯去年所選中的二輪秀米切爾羅賓遜也是可塑之才，新秀賽季的米切爾羅賓遜場均可以拿到7.3分6.4籃板2.4蓋帽，本賽季則是8.8分6.2籃板1.9蓋帽，考慮到他的上場時間只有20分鐘，米切爾羅賓遜又是一個二輪秀，這樣的護框表現可以說遠超預期。

尼克斯還有三個高順位年輕人，不過他們就沒這麼幸運了。

一是弗蘭克尼利基納，2017年的選秀大會上，尼克斯用八號簽選中了尼利基納，為此他們錯過了米切爾、阿德巴約、肯納德，可惜的是，這位法國後衛至今沒有打出符合順位的表現——生涯第三年，上場時間26分鐘，首發身份，場均只能得到6.5分3.5助攻，投籃命中率38.2%，三分命中率33.3%，進攻效率完全停留在新秀賽季的水平。

有意思的一件事是，2017年的選秀大會上，手握第九順位的獨行俠原本是有意尼利基納，可惜尼利基納被尼克斯所截胡；到了今年選秀大會，曾經有消息稱獨行俠可能會用37號簽換尼利基納，最終沒有實現。換句話說，尼利基納如今的交易價值連一個次輪簽都不如了，尼克斯卻仍然不知疲倦地嘗試著。

二是小丹尼斯史密斯，在尼利基納被尼克斯截胡之後，獨行俠用九號簽選中了這位爆發力驚人的天才後衛。新秀賽季，小丹尼斯史密斯場均可以拿到15.2分3.8籃板5.2助攻，他的爆發力確實強，整個賽季送出了40次扣籃，缺點是進攻效率差，和現在的巴雷特相差無幾。

在當時，不少人對小丹尼斯史密斯還是抱著一些期待的，至少他在扣籃大賽上給球迷留下了一些印象。可惜隨著東契奇的到來，小丹尼斯史密斯徹底「失寵」，他被送到了尼克斯，和尼利基納成為了隊友。

本賽季，小丹尼斯史密斯的表現再創新低，他場均只能得到17分鐘上場時間，而且上場也沒有球權，出手量只有上賽季的一半，場均只能拿到6.0分2.9助攻。小丹尼斯史密斯的缺點還是和之前一樣，沒投籃、沒組織、沒防守、球商也不高——獨行俠放走小丹尼斯史密斯並不是沒有原因的，給他掄了一個賽季也沒掄出來，也難怪尼克斯會不給他機會。

三是凱文諾克斯，作為去年的九號秀，尼克斯為了他錯過了亞歷山大，後者如今在雷霆打得風生水起。反觀諾克斯，上賽季作為首發前鋒，他只能以47%的真實命中率拿到12.8分4.5籃板，本賽季隨著上場時間的減少，諾克斯場均只能拿到8.0分3.0籃板1.4助攻。

值得一提的是，諾克斯可以說是上賽季最差的球員，在ESPN的統計當中，全部514位球員里，諾克斯的真實在場正負值排在聯盟倒數第一，為-7.47，這個數字幾乎是獨一檔，同屆第二差的格雷森阿倫是-5.46——這種表現很讓不讓人懷疑，諾克斯是否有值得培養的價值。

年輕球員差歸差，尼克斯的重建方式也非常讓人懷疑。

若是想要出成績，休賽期他們應該花精力去簽巨星，結果不簽巨星，簽下了一堆大前鋒球員，而且還是短工為主；若是想要擺爛，就應該讓巴雷特掄，就像老鷹對特雷楊所做的那樣，結果完全輪不到巴雷特，尼克斯居然讓莫里斯和蘭德爾掄，直接剝奪了諾克斯、米切爾羅賓遜、巴雷特等人成長的空間。

只能說，這支球隊能爛這麼久真的不是沒有原因的，尼克斯管理層的思路一般人還真理解不了，他們總是在聯盟下游徘徊，簽下的球星很多都成了垃圾合同，選中的新秀也很少培養出來。希望巴雷特不要在這種環境下迷失了自己，大蘋果城想要崛起，需要點時間了。
