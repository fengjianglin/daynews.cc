完美克制！獨行俠進攻端的優勢，76人防守端的最大劣勢

2019-12-22 512條評論 1,476次閱讀 323人點贊

前言

主場14勝1負的76人迎戰缺少絕對核心東契奇的獨行俠隊，最終卻以98-117輸給對手，慘敗19分，球隊三連敗，對於休賽期大手筆補強的76人來說近期的表現足夠糟糕，在得到霍福德和理查德森後，他們還沒有形成良好的化學反應，20勝10負的戰績遠不如上個賽季。

反觀獨行俠，將自己的客場勝率提升至11勝2負，在缺少了東契奇後接連拿下東部豪強雄鹿和76人，讓球迷看到了他們強大的團隊實力，每一名球員都在球隊找到自己的定位，發揮出了最大的效能，球隊19勝9負排在西部第四，超出了球迷的預期。

兩隊技術特點分析

獨行俠：持球擋拆+突破分球

獨行俠陣容最大的特點就是靈活，他們把221cm的波爾津吉斯頂到了四號位，優秀的移動能力和遍布全場的射程讓他在四號位上佔盡了優勢，五號位的鮑威爾也是一名移動能力出色的運動型中鋒，球隊通過四五號不停的擋拆來尋找突破的機會，把對手內線球員全部拉到三分線附近造成了內線的空虛，在擋拆後波爾津吉斯的外彈和鮑威爾的內切可以撕裂對手的防守陣型，本賽季波爾津吉斯弧頂出手105次三分球，只比他在禁區內的出手少25次，34.3%的命中率也是一個強大的威脅點。

除了兩名內線球員外，場上其他三名球員都可以持球完成突破，當一名球員突破時，另外兩名球員會在兩側底角進行埋伏隨時準備接球投籃，本賽季芬尼-史密斯在底角57投20中，命中率達到35.1%，哈達威在底角54投26中，命中率達到了逆天的48.1%，整個賽季獨行俠全隊三分線外出手佔比達到了45.3%，籃下出手佔比達到了43.2%，中距離一共只出手了292次，這種打法就是建立在他們大量持球擋拆的基礎上，獨行俠全隊有19.8%的進攻來自於持球擋拆，每回合可以得到全聯盟最高的1.08分。

76人：背身單打+無球切入

76人陣容最大的特點就是高大，這讓他們在每個單點的對位上都存在身高優勢，在落入到陣地戰時，西蒙斯由於缺少投籃能力會更多的選擇在低位發起進攻，技術細膩的恩比德更是76人隊的低位大殺器，再加上球隊以4年1.09億簽下的霍福德，三人都喜歡在低位進行背身單打，本賽季76人隊有12.5%的進攻來自於背身單打，是全聯盟佔比最高的，每回合得到0.94分也可以排在聯盟第六。

先天的錯位優勢讓他們在進行背身單打時可以很容易吸引到對手的協防和包夾，三人都有著很強的傳球能力（西蒙斯場均8.1助，霍福德3.8助，恩比德3.3助），在吸引到防守的同時可以快速出球，其他球員通過切入可以輕鬆的獲得空位投籃的機會，哈里斯每次空切可以得到1.71分，超過聯盟97.6%的球員，全隊通過空切每回合可以得到1.37分，有效命中率達到了69.7%。

獨行俠的取勝之匙：以己之長攻彼之短

76人防守獨行俠：相生相剋

獨行俠是最擅長打持球擋拆的，而76人剛好又是最不擅長防守持球擋拆的球隊，所有球隊都知道76人陣容的高大，因此也知道他們陣容在防守端最大的劣勢，就是移動速度慢，本賽季76人每場比賽有26.2個回合在防守持球擋拆，是聯盟最高的，獨行俠在進攻時，會把波爾津吉斯和鮑威爾全部提到三分線外進行擋拆，這樣就非常容易形成錯位防守的機會。

當76人內線錯位防守獨行俠小個球員時，就會被輕鬆的突破過掉，這時如果其他球員選擇補防，底角的兩名球員可以獲得完全的空位機會，如果不來補防，持球者可以輕鬆的完成攻框，實在沒有機會還可以傳給由小個球員錯位防守的波爾津吉斯，他可以輕鬆的完成三分球投射，對方絲毫形不成干擾。當然76人也可以選擇不換防，在這種情況下，獨行俠的內線會利用擋拆的間隙快速完成切入，他們有著比76人內線更優秀的靈活性，可以快速的擺脫防守完成對籃筐的攻擊。

獨行俠就這樣只通過一招持球擋拆就把76人打的狼狽不堪，76人移動緩慢的特點是無法通過戰術改變來提升的，而獨行俠內線球員的移動又是非常的快速，在防守獨行俠進攻時，76人被克制的死死的。

獨行俠防守76人：聯防戰術

本賽季的76人放走了他們隊內最強的射手雷迪克，哈里斯的三分球命中率從上賽季的39.7%降至現在的30.8%，再加上對三分球無感只喜歡突破和內線要位的西蒙斯，本賽季球隊沒有一名球員可以場均命中2記以上三分的球員，所有球隊都知道76人的內線是多麼強大，而外線又是多麼的普通，在對陣76人時，獨行俠有55個回合使用了聯防，只讓76人得到了45分，在聯防時，76人的外線球員不能連續的命中投籃來給與回應，本場比賽76人隊在三分線外34投僅10中，命中率只有29.4%。

76人無法用外線投籃來懲罰獨行俠讓他們完全放空了三分線外的防守，通過聯防造成了76人突破線路的受阻和內線的擁堵，球員無法利用突破撕開防線傳給無球切入的球員，在內線進行背身單打時也會受到很大的阻礙，本場比賽76人在內線只得到38分，低於他們的賽季平均值9.9分，整體命中率只有42.2%，遠低於獨行俠的51.8%，打得非常不流暢。

獨行俠靠著聯防讓76人的進攻陷入了停滯，從本賽季平均水平來看，76人在三分線外的表現並不是非常糟糕，場均只比獨行俠低了0.5%，但是他們卻總是不能在對方使用聯防時命中三分球，不知道這是不是心理的問題，如果76人不能很好的解決破聯防，後續很難戰勝季後賽級別的球隊。

總結

76人本賽季加入了幾名新球員，磨合肯定需要時間，我們不能以一場比賽的失利就來否定76人隊的能力，但是他們在休賽期給哈里斯開出了5年1.9億的頂薪，又以年均2500萬簽下霍福德，而沒有補充射手球員和替補球員，導致他們本賽季失去了瞬間得分的爆炸力，替補球員場均只能貢獻聯盟第五低的29.5分，如果主力球員再不能形成良好的化學反應，這賽季的76人很難成為總冠軍的最有力爭奪者。
