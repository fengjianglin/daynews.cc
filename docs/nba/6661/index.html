保羅這位適用於NBA所有版本的控球後衛，誰敢在他身上賭一把？

2019-11-27 0條評論 18次閱讀 0人點贊

想搞大動作，情況不允許

距離NBA賽季中段的交易日來臨，大約還有20天左右的時間。

為什麼要提前這麼久就開始關注今年的交易，其實是想分析一下哪些球員會在賽季中期改換門庭。

但是思來想去，覺得今年賽季中的交易，可能是最沒有看點的。

為什麼這麼說呢？因為今年休賽期，球星轉會的不在少數。

這些”大魚”既然挪了窩，沒有達到目的之前，肯定不會輕易再換球隊。

畢竟夏天的轉會就已經是經過深思熟慮的事情了，想要在短短時間內就再次改變主意，這事在NBA本身並不常見。

更為巧合的是，今年夏天已經轉會了的球星們，在新的球隊里都磨合的不錯。

不管是快船，湖人還是火箭隊，目前看起來都沒有做太大調整的必要。

這對於那些處於中游的球隊來說，就是很不利的消息了。

因為想要向上衝擊排名，就需要有球星的加持。

可是目前，聯盟里的一線球星們都相對穩定，而零碎的”修修補補”，對球隊戰績的增幅，恐怕不會太大。

保羅的油箱里，肯定還有油

放眼整個聯盟，目前在單核帶隊的球星，所剩無幾。

而那些正在準備重建的球隊中，根本就沒有球星的影子。

剩下的那些有實力，也有可能”挪窩”的球員，恐怕就只有草草的幾個人。

而保羅，絕對是這一小部分人里，最有交易價值的。

為什麼這麼說呢？

因為個人覺得，保羅目前仍然屬於聯盟所有的控衛中，最為可靠的那幾位之一。

我們先捋一捋保羅的職業生涯。

他在05年進入NBA的時候，整個NBA還流行著傳統中鋒。

姚明和奧尼爾以及霍華德，都是聯盟最關注的點，後衛和中鋒的搭配還是那時候的”奪冠密碼”。

正是因為聯盟還流行著”陣地戰”，出道之後的保羅，成為了一個出色的控場大師。

所以不管是跟韋斯特搭檔，還是後來加盟的錢德勒，都跟保羅有過一段美妙的贏球時光。

等到09年庫里哈登等優秀的後衛進入聯盟之後，慢慢的進入了小球時代。

整個籃球環境開始變得”快”起來。

“思變”的保羅，後來加盟了快船，讓快船隊變成”空接之城”。

而他自己的打法，也慢慢的由傳統的組織者變成帶有攻擊性的雙能衛。

他的中投越發爐火純青，他的三分球練得越來越好。

等到哈登這一批的球員成為聯盟中流砥柱之後，籃球規則都因此發生改變。

裁判對於犯規的判罰，成為球場上能夠主宰勝負的因素。

而這時候的保羅，順勢練就了一身造犯規的本事。

時代一直在改變，但是保羅一直都未曾被時代淘汰，因為他善於改變自身來順應當下的環境。

想奪冠，本身就是一場豪賭，看誰膽子大

為什麼在討論他的交易價值之前，要先回看他的職業生涯，是因為他的經歷，讓他在自己的第15個賽季，依舊有著讓球隊肉眼可見的價值。

時代的更替，導致球場位置最難調整的，一定是控球後衛。

但是保羅的整個職業生涯，已經證明了他可以存在於任何時代。

他的態度，也證明了只要能贏球，他可以做任何事情。

他的防守，依舊可以在關鍵時刻成為球場的勝負手。

他在關鍵時刻的處理球的能力，依舊是聯盟頂級。

他在球隊處於領先時的控場能力，依舊穩如老狗。

當球隊處於落後一方時，雖然他已經34歲，但身上依稀還有”聖保羅”的影子。

任何一支需要一位合格的一號位的球隊，在得到保羅之後，都會有質的提升。

現在阻礙他加盟爭冠球隊的唯一”絆腳石”，或許就是他身上的天價合約，就看誰敢賭一賭了。
