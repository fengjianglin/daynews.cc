20勝8負東部第三，掩蓋不了熱火最大問題！他們在走上賽季老路

2019-12-20 3,559條評論 4,687次閱讀 185人點贊

在NBA，勝利通常可以掩蓋所有的問題，邁阿密熱火就是這樣，他們在對陣密爾沃基雄鹿、多倫多猛龍、休斯敦馬賽克和費城76人的比賽中，熱火都有著出色的表現，但他們還是有一個大問題存在。

上個賽季的熱火曾經遭遇過嚴重的傷病問題，戈蘭-德拉吉奇整個賽季只打了36場比賽，迪昂-維特斯只打了44場比賽，而溫斯洛、詹姆斯-約翰遜和德里克-瓊斯都錯過了一些比賽。而約什-理查德森是熱火將巴特勒帶到邁阿密的關鍵球員，他雖然在賽季的大部分時間裡都很健康，但他在賽季末遭遇了傷病，也破壞了球隊衝擊季後賽的希望。

到目前為止，本賽季的情況看起來和上個賽季差不多，德拉吉奇已經錯過了10場比賽，溫斯洛只打了10場，德里克-瓊斯也只打了13場比賽，而維特斯甚至還沒有上場，這樣的情況在短期內是不可能改變的。

熱火先簽後換得到巴特勒之後，薪金總額已經直逼奢侈稅觸發線，他們不可能在自由球員市場有什麼作為，這意味著除非他們進行涉及多筆交易的重大調整，否則熱火現在的狀況就將持續一整個賽季。

熱火現在唯一的變數就是他們的兩個雙向球員——達里爾-梅肯和克里斯-席爾瓦，雖然梅肯總共四場比賽中才只有14分鐘的上場時間，但席爾瓦已經是輪換陣容的一員，他在熱火目前的28場比賽中已經出場24次，場均得到9.4分鐘的出場時間。

雙向合約的球員最多可以在NBA球隊度過45天，然後如果他的球隊還想繼續使用他們，那就必須把他們的合同轉為聯盟底薪的正式合同，而席爾瓦的使用時間基本上已經快要到頂了，不出意外，熱火應該會留下他，所以他們肯定會付出一些代價。

解決這個問題的最好的辦法顯然就是指望這麼多受傷的球員能夠恢復過來，不過好在熱火目前20勝8負，排在東部第三的開局非常不錯，這給了他們一些緩衝時間去弄清楚關鍵球員（特別是德拉吉奇和溫斯洛）是否會能夠儘快復出。
