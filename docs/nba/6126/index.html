空砸4.8億！日本天才狂轟27+7打假偽強隊，費城雙帝被高估了？

2019-12-06 0條評論 14次閱讀 0人點贊

76人死亡五大又失靈了，北京時間12月6日，76人被魚腩球隊奇才119比113擊敗。這支今年夏天瘋狂燒錢的球隊如今只取得15勝7負的戰績排名東部第五，表現大大低於人們的預期。76人的死亡五大陣容不管用，而更令人遺憾的是雙帝組合恩比德和西蒙斯本賽季一起狀態下滑。76人的低迷成全了奇才日本新星八村壘，他轟下27分7籃板，延續了近期出色狀態。

76人今年夏天的豪賭現在看來並不奏效。在小球風暴盛行的當今NBA，76人反而打造出大個陣容。他們在夏天瘋狂燒錢志在衝冠。

球隊給了托拜亞斯·哈里斯5年1.8億美元合同，和本·西蒙斯提前續約5年1.7億美元合同，用4年1.09億美元簽下霍福德，再加上4年1251萬美元簽下賽布爾，2年980萬美元簽斯科特……76人在今年夏天的一系列操作，瘋狂砸了超過4.8億美元。

76人也打造出本·西蒙斯、約什·理查德森、托拜亞斯·哈里斯、霍福德和恩比德的五大首發陣容。而在約什·理查德森缺陣的情況下，76人主帥布雷特·布朗保持五大配置，用身高2米01的科爾科馬茲打首發，首發五虎的身高全在2米01以上。

但是，在12月6日，76人竟然在客場輸給了奇才。恩比德雖然得到26分21籃板，但是出現8次失誤。西蒙斯得到17分10助攻，出現7次失誤。本賽季費城雙帝的狀態都出現了下滑。

恩比德本賽季場均得到22.6分12.7籃板2.8助攻1.44蓋帽，得分是新秀賽季以來的最低值。上賽季恩比德場均得到27.5分13.6籃板3.7助攻和1.9蓋帽。相比上賽季，恩比德的4項關鍵數據全出現下滑。

拿到5年1.7億美元合同的本·西蒙斯表現也出現下滑，本賽季他場均得到13.2分6.9籃板6.9助攻，得分創造了職業生涯新低。西蒙斯在夏天苦練投籃，但他本賽季也就只投進了一記三分球，他的投籃仍然不靠譜。

恩比德和西蒙斯被譽為天賦異稟的雙帝組合，但現在來看，他們似乎被高估了，按照眼下的架勢，76人根本算不上爭冠行列的球隊，他們與雄鹿、快船以及湖人這三支爭冠勁旅相比有一定的差距。

76人是偽強隊？日本球星八村壘在面對76人的比賽，貢獻了全隊最高的27分，他的效率很高，18投11中，投進一個三分球，罰球1次全進，為奇才贏球立下大功。八村壘過去5場比賽得分都在15分以上，他本賽季場均得到14.1分5.7籃板1.6助攻，新秀賽季的得分比姚明菜鳥賽季的場均13.5分還要高。
