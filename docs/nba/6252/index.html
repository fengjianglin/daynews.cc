賽季至今最強陣容，4位置無懸念，中鋒競爭激烈，東契奇第一小前

2019-12-04 0條評論 17次閱讀 0人點贊

NBA新賽季已經完成了10-11月的較量，截至目前，本賽季各個位置上表現最佳的應該是下面5個人，有4個位置已經沒有任何懸念，而中鋒位置競爭則很激烈。

控球後衛：勒布朗詹姆斯，場均25.7分7.1籃板10.9助攻，命中率49%。在威少、利拉德、歐文低迷，庫里受傷的情況下，改打控球後衛的詹姆斯反而撐起了控衛的一片天，他是賽季至今的聯盟助攻王，至少這個數據能夠說明，詹姆斯打控衛也能夠吃得消。

得分後衛：詹姆斯哈登，場均38.9分5.9籃板7.8助攻1.8搶斷，場均14.4個罰球。哈登在自己的路上越走越遠了，罰球越來越多，得分也越來越多，已經沒辦法引起太多轟動了，就像威少的場均三雙意義，但是就算如此，得分後衛上也毫無競爭對手，無論是布克、米切爾、比爾、克萊、德羅贊都有很大差距。

小前鋒：盧卡東契奇，場均場均30.6分9.9籃板9.6助攻1.5搶斷，把一個20歲的小夥子放在這裡確實有點突兀，但他配得上這個位置，接近場均三雙的表現，穩定的輸出，球隊戰績也不錯，在詹姆斯和字母哥改變位置，杜蘭特受傷，倫納德常規賽划水的情況下，東契奇已經是第一小前鋒了。

大前鋒：揚尼斯阿德托昆博，場均30.8分13.4籃板5.8助攻1.5搶斷1.3蓋帽，同樣沒有任何懸念，字母哥甚至還是MVP的大熱門，無論放在哪個位置，他都是第一人的有力爭奪者，所以哪怕是表現出色的安東尼戴維斯，也得讓位。

而中鋒是最有懸念的，因為沒有一個絕對突出的人，約基奇場均15.6分10.4籃板6助攻1.1搶斷的數據較上賽季似乎還下滑了，儘管球隊戰績繼續高居西部第二。恩比德22.4分12.2籃板2.9助攻0.9搶斷1.5蓋帽同樣沒有上賽季穩定，還被小加索爾零封了。而唐斯的25.9分12.6籃板4.1助攻1.1搶斷1.4蓋帽，顯得分外顯眼，不足是球隊只是西部第七而已。

綜合表現看，唐斯還是要暫時領先，個人表現最出色也最高效，而球隊戰績上，至少目前西部第七的位置也是符合森林狼預期的，並不會太差。
