霍華德蓋帽榜超約翰遜追大本，詹姆斯為他點贊：這是心態的成長

2019-12-05 0條評論 15次閱讀 0人點贊

洛杉磯湖人隊中鋒德懷特-霍華德目前在蓋帽榜上升到了第16名。

在剛剛洛杉磯湖人以121比96輕鬆戰勝猶他爵士的比賽中，霍華德的職業生涯總蓋帽次數來到了2083次，超過了喬治-約翰遜的2082次。目前在總蓋帽榜上排在他前面的是曼努特-博爾的2086次，霍華德很可能在接下來的一兩場比賽中對他完成超越。

四分之一個賽季下來後，霍華德在所有人的面前證明了自己。在他職業生涯的全新階段中，他為湖人注入了活力。他最初被認為只不過是德馬庫斯-考辛斯的替代品，後者在夏天撕裂了他的前十字韌帶。

但從一開始，霍華德就超出了所有人的預期。作為替補陣容的一員，他已經完全接受了自己的新角色。在進攻端，他不再在低位要球背打。相反，相比之前他本賽季開發出了擋拆、空切、空接等等「藍領」技能。

對於霍華德的表現，湖人隊當家球星勒布朗-詹姆斯也是頗為滿意，在接受《洛杉磯時報》採訪時對他評價道：

「在德懷特的整個職業生涯中，他一直是那種你把球拿下來，把球傳給他，然後每個人都圍繞在他身邊打球的人。但現在他是那種如果你把球傳給他，他會幫助進攻的人。他把球回傳給後衛，等待機會，基本上他在這個位置上做得很好。這就是心態的成長，這是為了整個球隊。」

本賽季截止至目前，霍華德在19.7分鐘的上場時間裡，場均能夠得到7.0分+7.0籃板+1.5蓋帽。如果能夠保持這樣的水準，到本賽季結束時，霍華德將有望在蓋帽榜上超越本-華萊士，升至蓋帽榜第13位。
