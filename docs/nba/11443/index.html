美媒評選各要素最佳球員榜！現役竟有8人？詹皇入選最佳身體榜

2019-12-11 690條評論 2,076次閱讀 1037人點贊

近日美媒發布了一份榜單，是關於各要素最佳球員榜單的。其中一共囊括了5項內容，分別是投射榜、身體素質榜、控球榜、防守榜和大心臟榜。接下來就讓我們一起來看一下吧！

首先是最佳射手榜，上榜的四位分別是雷.阿倫、斯蒂芬.庫里、克萊.湯普森和雷吉.米勒。其中，庫里和湯神更是同效力於現今勇士隊兩人作為全聯盟最強的後場二人組，以精湛的射術著稱。粉絲們也親切地稱呼他們的組合為「水花兄弟」。兩人都在三分榜上留有姓名，湯神以1798記三分排名第16，而庫里更是以2492記三分高居第三！不管今後勇士會如何，這兩位球員都已經為自己的生涯贏得了足夠多的榮譽，將來NBA的歷史上，也會有他們留下的濃墨重彩的一筆！

其次是最佳身體素質榜。上榜的分別是邁克爾.喬丹、勒布朗.詹姆斯、凱文.杜蘭特和揚尼斯.安特托昆博。這份榜單上現役就佔據了三人！一位是雄鹿如今的當家球星，希臘巨獸字母哥，上賽季的強勢MVP；一位是不老皇帝，紫金新皇詹姆斯；還有一位則是寂靜沉潛，等待破折而出的死神KD！字母哥的身體素質就不多誇了，號稱現役奧尼爾，籃下無人能防；老詹也是鐵打的身體，17年職業生涯，時年35歲的老漢竟然還保持著近乎巨星的競技狀態！這已經是一種逆天了；最後是杜蘭特身高臂長的怪物球員，靜態天賦恐怖！在大傷之前，KD也不少為大家貢獻平框暴扣、隔扣這樣的暴力美學；不僅身體天賦出眾，他的投籃手感也是頂尖的，再加上無解的速度與身材優勢，KD成為了現役最難防守的單打之王！

然後是最佳控球榜，上榜的球員別是賈馬爾.克勞福德、阿倫.艾弗森、凱里.歐文和皮特.馬拉維奇。有著「德魯大叔」之稱的歐文能登上這份榜單想必大家都是感覺理所應當的，畢竟歐文也有著現役手活第一人之稱！而接近40歲的克6上榜，可能就出乎很多人的預料了。雖然克6年事已高，但這也絲毫不會影響他對籃球的熟悉與掌握！其招牌動作背運接跳步上籃也成為各籃球愛好者、包括職業運動員競相學習的技能。

最佳防守榜上榜的分別是邁克爾.喬丹、斯科蒂.皮蓬、丹尼斯.羅德曼和科懷.倫納德。現役倫納德成為了榜單上的獨苗。雖然動態天賦不夠突出，但是上天卻賜給了他優秀的靜態天賦！長達25公分的巨掌，高出身高21公分的臂展，再加上一堵厚牆般的寬肩，這讓倫納德成為了現役中最恐怖的防守封鎖者！強如詹姆斯、杜蘭特這樣的都被小卡的「死亡纏繞」鎖到沒脾氣！且倫納德現如今還有兩個FMVP在手，小卡的歷史地位註定不會低！

最後是最強大心臟榜

這份榜單上的四名球員分別是史蒂夫.納什、科比.布萊恩特、拉里.伯德和埃爾文.約翰遜。一位是最著名的無冕之王，號稱「風之子」，不論何時都能保持靈動與飄逸；一位是歷史絕殺第一的殺手黑曼巴；一位是裝X從未被打臉的B王之王；還有一位是場上全位置都能勝任的全面手。至於現役為什麼沒人上榜？可能是現役中還未出現能擠掉這四位大神的存在吧！

體壇風向及時了解，體育動態時刻把握！感謝您的閱讀，您的閱讀是我更新的最大動力！
