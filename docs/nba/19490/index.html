看不下去了！勇士主場球迷開始倒戈，格林持續低迷，克萊直搖頭

2019-12-17 1,020條評論 1,993次閱讀 193人點贊

這個賽季的勇士隊可以說是一個休賽期之間就跌落了神壇。本來上個賽季在西部無人能頂，一騎絕塵的他們，在這個賽季突然遭遇到了斷崖式的下滑，目前更是排在了聯盟倒數第一的位置上，這難免讓球迷們的心理有落差。而在這個賽季勇士搬進了新的球員大通中心，在這個球館中勇士並不順利，庫里、拉塞爾、格林等人先後受傷，雖然現在格林和拉塞爾已經復出，但是這兩人最近的狀態真的讓人捉急。

在今天的比賽中，勇士隊再一次以79-100大比分不敵國王隊，遭遇到了4連敗，在之前輸給了尼克斯之後，這支勇士隊似乎就沒了心氣，接連兩場比賽都被打得毫無還手之力，這場比賽在自己的主場大通中心，他們更是被在西部還不算是強隊的國王隊打的落花流水，很明顯，他們這樣的表現讓球迷很不滿意，我們不知道他們有沒有激怒球迷，至少現在勇士球迷是看不下去了。

根據勇士記者的報道，今天在比賽進行到第四節的時候，勇士隊大比分落後，這時勇士的主場大通中心出現了讓人尷尬的一幕，因為在他們主場出現了國王的助威口號。現在我們不知道這個助威口號到底是誰做的，但是肯定不會是大通中心和勇士管理層自己弄的吧，畢竟你想擺爛也不可能這麼光明正大。所以最有可能打出這些標語的還是在現場看球的球迷了。

本來剛剛搬進新的球館開開心心的，但沒想到這個賽季的勇士可以做到這麼爛，人見人欺，就連東部墊底，無欲無求的尼克斯都打不過，你說這誰不氣。何況勇士上個賽季可是打進了總決賽的球隊，這隻隔了一個休賽期你就成了這樣，這心理落差也太大了。同時勇士隊的兩大億元先生格林和拉塞爾也持續低迷，他們今天合計只拿到了15分，這哪像是拿著億元合同的球員乾的事啊，所以在球隊大比分落後的時候我們也看到了現場球迷的不滿，也看到了勇士受傷的球星克萊-湯普森在場邊直搖頭，看得出大家都對這支勇士非常失望，但還是希望他們可以早日重回正軌吧。
