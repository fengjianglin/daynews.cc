戈登要成犧牲品？美媒建議火箭補強鋒線，三人和哈登絕配

2019-11-26 0條評論 17次閱讀 0人點贊

火箭隊昨天在主場123-137不敵獨行俠，球隊遭遇了三連敗，鋒線再次成為火箭最大的問題，之前輸給快船也是這樣的問題，西部有詹姆斯、倫納德、喬治、東契奇、博格達諾維奇（爵士）這樣的鋒線球員，而火箭一位正牌的小前鋒都沒有，只能豪斯頂上，豪斯是一名射手，防守端不行。所以，美媒建議火箭交易戈登，因為威少目前打得還算不錯，他的大合同也很難交易出去。

戈登本賽季只打了9場比賽場均只有10.9分，投籃命中率只有31%，三分球命中率自由28.4%，罰球命中率只有64.3%，全部是職業生涯最低。應該是受到膝蓋傷勢的影響，目前已經缺席了7場比賽，他4年7600萬美元的合同明年夏天生效，2024年夏天到期，他已經30歲了，這個賽季的薪水為1406萬美元。

美媒給出了兩套方案，第一套：火箭與尼克斯交易，戈登換波蒂斯。

波蒂斯身高2米08，臂展2米18，今夏2年3100萬美元加盟了尼克斯，目前場均得到9.3分6.3籃板，場均命中1.2記三分球，命中率高達42%，符合德安東尼的戰術體系。尼克斯鋒線球員太多了，導致諾克斯沒有多少機會，加上尼克斯戰績那麼差，他們也想甩掉波蒂斯2年3100萬美元的合同。而尼克斯的後衛線沒有能打得球員，戈登會非常適合，他們也想衝擊一下季後賽。

第二套方案：火箭與活塞交易，戈登、未來次輪選秀權換斯內爾、和馬基夫-莫里斯

斯內爾身高1米98，運動能力強，目前場均得到9.2分，場均命中2.3記三分球，命中率高達45.5%，他的合同還剩下2年2357萬美元，這個賽季的薪水為1139萬美元。馬基夫-莫里斯在活塞場均11.3分4籃板，三分命中率高達40%，他的合同是1年360萬美元。活塞的後衛線急需一名戈登這樣具有衝擊力的後衛，再加上羅斯、格里芬、德拉蒙德，衝擊季後賽。

波蒂斯、斯內爾、馬基夫-莫里斯這三名鋒線球員都是3D類型的，都非常適合在哈登身邊打球，防守、投三分，類似於阿里扎之前扮演的角色。
