感恩節全聯盟休戰一天，東西區第一名湖人與雄鹿將衝擊十連勝

天天要聞
3 週 ago

新賽季的第一支拿下十連勝的球隊或許將在明天的比賽產生，可能是東區的第一名雄鹿隊，也可能是西區的第一名湖人隊，又或是兩支球隊都同時達成這一成就！

19-20賽季第一個十連勝！

漫長的賽季，82場比賽，30支球隊之間的相互對抗，沒有說絕對能穩贏的比賽，每場比賽都是需要球員去努力拚搏爭取得到的勝利！

此前曾分析，湖人隊在十一月份的賽程強度並不是非常大，他們很可能拿下一波十八連勝來結束本月份的征程！

不料他們沒有輸給新晉黑馬-太陽隊，也沒有輸給一向老辣的馬刺隊！

而是偏偏輸給了失去主力球星的猛龍隊！

而在那場比賽，湖人隊球員波普也是沒有得分，而在波普有得分進賬的比賽，湖人隊保持全勝，所以靠著「波普定律」湖人又收穫了一波九連勝！

在輸給猛龍之後，湖人隊似乎覺醒了，他們在接下來的比賽更為警惕，關鍵球的處理上也是遊刃有餘，庫茲瑪投得准，濃眉搶得狠，隆多更是能夠將替補陣容帶動得生龍活虎！

第一波長連勝收入囊中，在接下來對陣奇才隊的比賽，湖人隊如果可以拿下勝利，將收穫十連勝，結束十一月份的征程！

奇才隊的當家球星沃爾仍然未復出，取而代之的比爾目前場均可以拿下28.9分，是球隊的第一的得分手，曾經作為湖人隊球員的托馬斯如今也逐漸找到狀態，場均可以貢獻12分！

托馬斯-布萊恩特曾經也是湖人隊選中的新秀球員，如今在奇才隊時而充當球隊首發中鋒，場均可以拿下14.8分8.8個籃板。已經是一名相對合格的內線球員！

莫里茲-瓦格納上賽季在湖人隊還是飲水機守護神，在奇才隊也得到重要，場均可以貢獻12.3分5.4個籃板！

伊薩克-邦加也是湖人舊將，他的表現要遜色許多，場均14.8出場時間只能貢獻3.6分！

這四位湖人舊將對陣如今這支聯盟第一的湖人隊，不知心裡作何感想！

不僅衝擊十連勝，還要衝擊聯盟第一！

上賽季的雄鹿隊在東區名列榜首，而這賽季開局似乎有些不順利，輸掉三場比賽的雄鹿懸崖勒馬，現在已經是逐漸找到曾經那個熟悉的節奏！

與湖人隊一樣拿下了九連勝，十連勝的記錄或許將在對陣騎士隊的比賽中產生！

騎士隊目前的處境有些糾結，衝擊季後賽或許是有機會，但想在季後賽中有所作為幾乎是不可能的，單憑樂福一人很難完成這項艱巨的任務！

所以又有傳聞他們想把樂福交易出去，或許會從開拓者隊換來首輪簽與西蒙斯，利特爾！

這筆交易是否會產生只能靜候佳音！

雄鹿對陣騎士的比賽，是在克利夫蘭舉行，雄鹿隊全隊各項數據都是非常華麗！

場均119.4位列聯盟第二，51.5個籃板是聯盟第一，而6.6個蓋帽排名聯盟第三，助攻場均24.4個排名聯盟第十四，助攻數據還可以有所提升，這取決於外線的投射命中率與字母哥本人的投射命中率！

字母哥作為球隊絕對核心，外圍投射的命中率不高，這讓他有時會被對手放空，而收縮在內線防守，字母哥只能投射他不擅長的遠距離投籃！

全世界的球迷都在等著字母哥練成遠投的那一天，這樣他將成為這個世界上最具統治力的球星，希望在未來的日子能看到他的提升！

這樣，離雄鹿隊打進總決賽甚至拿下總冠軍就會更近一步！
