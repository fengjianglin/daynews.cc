拋開榮譽地位不談，科比的個人技術是否比喬丹更出色？

2019-12-20 38條評論 1,201次閱讀 80人點贊

作為喬丹的接班人，科比無論是在技術還是在榮譽上都已經很接近喬丹了。很多球迷也很願意那他們兩人對比，之前喬丹親口說過他只可能輸給科比的話也夠很多粉絲們留下了足夠的遐想空間。不過，如果我們拋開身體天賦，單論技術的話，是不是科比要更加優秀一些呢？

答案明顯是否定的。拋開他們兩人的天賦不說，科比的那些得分手段大部分都是偷學喬丹的，所以說還是喬丹要更強一些。論聯盟轉身跳投第一人，那肯定非喬丹莫屬。可以說，他不光把後仰發揮到了極致，甚至他的那些運球和拉杆也相當帥氣。

如果說喬丹的缺點的話，那可能就是三分球了。在喬丹那個年代很少有三分，於是喬丹幾乎也沒有怎麼練習過，到現在為止，還是沒有人能夠打破喬丹在三分大賽上創下的最低分記錄。不過話雖然這樣說，喬丹只是沒有著重的進行練習而已。而且他就算是不投三分，依舊有其他很多種方法得分，要知道他曾經拿到過10次得分王！

喬丹能夠拿到那麼多榮譽，其實他的身體素質很好，但更出名的則是他的滯空。他的身高和體重還有爆發力幾乎都不是聯盟最頂級的水準，但靠著超強的滯空，他和防守球員同時起跳，甚至能夠保持別人落地他依舊還是雙腳離地的狀態。再加上那些相當細膩的技巧，所以喬丹當年才能夠砍下那麼多分。如果說論大體的話，科比確實和喬丹有著很大的相似點，但其實內部有一些細節他是比不上喬丹的！

科比是最接近喬丹的球員，幾乎完美繼承了喬丹所有技能，同時，他還向奧拉朱旺學習了腳步，這樣一來他的得分就變得更加容易。儘管之後科比也取得了很多榮譽，但是和喬丹之間還是有一些差距，被排在喬丹之後的歷史第二分衛上！

科比當年正是因為球風屬於飛天遁地型，所以之後因為傷病才不得不慢慢開始轉型。當年的喬丹也是這樣，年輕的時候完全不會想到之後身體會不支持自己的打法。最後我們總結可以得出，如果只說技術的話，那麼無疑還是喬丹要更強一些。因為得分能力的強弱在一定程度上就決定的球員的實力，這一點科比確實和喬丹有差距！
