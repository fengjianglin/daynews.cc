不同的三巨頭，不一樣的情懷，聯盟中的那些三巨頭

2019-12-11 1,674條評論 3,589次閱讀 275人點贊

有幸經歷了三組三巨頭。

從一開始的馬刺GDP三巨頭，到凱爾特人三巨頭，到熱火的三巨頭，然後是勇士的三巨頭，三巨頭各具特色，風格不一，但最終結果都是一樣的，取得了總冠軍。

一、馬刺GDP三巨頭

GDP組合組建於2002年賽季，5進總決4次奪冠，三人不緊球場上配合默契，心有靈犀。GDP對於我們已經不再是一個單純的團體，而是馬刺隊的代名詞。人們說起GDP，都會想起那隻曾經堅韌的馬刺，那個可愛的鄧獃獃，那個法國跑車，還有那個頭髮日漸稀少的吉諾比利。三個人如今都退役了，鄧獃獃加入了馬刺的教練組，這或許是一種輪迴吧。如今的馬刺成績已經今非昔比，但是GDP的精神依然在，仍然讓後人高山仰止。GDP的偉大不止在於他們自身，還在於波波維奇。波波教練用他的睿智讓GDP融合的如此完美。

二、凱爾特人三巨頭

凱爾特人三巨頭在組建的第一年就奪得了總冠軍。三巨頭以加內特為中軸，皮爾斯內外兼具，君子雷主攻外線。內外呼應，有內有外。當時的里弗斯還知道怎麼過富日子，不像現在那樣，有了錢不知道怎麼花了。論實力，每個三巨頭都很有自己的特點，沒有什麼可比性，但是凱爾特人的三巨頭也集成了凱爾特人的底蘊和傳統，拼搏到最後一分鐘，絕不放棄，是凱爾特人的代名詞。

三、熱火三巨頭

韋德、詹姆斯和波什組建的熱火三巨頭是我最喜愛的三巨頭。組建時期正直自己大學時期，那時候沒到熱火比賽，或者到了熱火季後賽那段時間，自己總會從自習室逃出來，看他們的比賽。當時喜歡韋德的飄逸，喜歡詹姆斯的兵來將擋水來土掩，喜歡波什的靈動霸氣。三巨頭陪我走過了美好的大學時光。當然大學時光不止有他們，但是沒有他們的日子，彷彿覺得自己的生活缺少些什麼。熱火三巨頭奪得了2次總冠軍，奪冠率在幾組巨頭中是相對較高的。

四、勇士三巨頭

水花兄弟和杜蘭特組建的勇士三巨頭，可謂是比賽最流暢最好看的三巨頭。打法如行雲流水般，嚴禁而又不失活潑。三分恐怖的讓人膽寒，勇士也奪得了2次總冠軍。勇士三巨頭的形成，騎士很大程度上受到熱火三巨頭的影響，但其實力、名氣和以上幾組三巨頭還稍有差別。

其他的如喬丹、皮蓬和羅德曼等年代久遠的三巨頭，依然讓我們景行行止，但是讓他們靜靜享受自己的過往吧，我們不再去打擾他們了。

總之：三巨頭沒有實力差距之分，只有特點不同，給我們帶來的印象和感覺不同。你心目中還有那些讓你記憶猶新的三巨頭呢？說出你的想法，追憶那段失去的青春歲月吧。
