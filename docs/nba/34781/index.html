二輪秀上賽季場均4.7+1.4+2.6，本賽季場均19+3+7，喬丹要笑醒了

2019-12-24 715條評論 2,931次閱讀 279人點贊

今年夏天，喬丹的黃蜂終於沒有給肯巴-沃克開出天價挽留合同，他們只是象徵性地用3年5800萬從凱爾特人挖來了特里-羅齊爾，算是挽回一些顏面。但是新賽季拯救黃蜂的並不是特里-羅齊爾，而是之前一個名不見經傳的小將——德文特-格雷厄姆，一個2018年的次輪秀，在本賽季迎來了爆發。

上個賽季，作為一個次輪秀，格雷厄姆出戰46場，場均14.7分鐘，獲得的機會不算多，也不算少，但場均僅有4.7分1.4籃板2.6助攻的表現，總體上比較平庸。不過，當肯巴-沃克出走後，控球後衛上位置空了出來，格雷厄姆抓住機會成長，並很快擠進首發（本賽季已過去的33場比賽，前10場替補，後23場首發），他場均可以砍下19.3分3.7籃板7.5助攻，數據較上賽季暴漲了接近4倍。

毫無疑問，就格雷厄姆現在的表現而言，遠超預期，喬丹恐怕要笑醒了吧。現在的黃蜂也並非魚腩，目前以13勝20負排在東部第9，仍舊葆有進入季後賽的希望。對於格雷厄姆，半個月前，奇才主帥斯科特-布魯克斯就說「格雷厄姆可能是領跑『最快進步球員獎』的人選之一」，而最新的評價來源於籃網名將丁威迪，丁威迪也說德文特-格雷厄姆應該當選進步最快球員。

那麼格雷厄姆是怎麼看待自己的成長呢？前不久，他接受了記者的採訪，坦言：「在我長大的過程中，我從來不是那種『麥當勞全美最佳』類型的球員，也不是AAU球隊里的頭牌，所以……我是隊里個頭最小的，我必須要努力，必須要做好細節，才能得到上場機會。」另外，格雷厄姆還說保持耐心、磨礪自己、始終堅信也是自己取得如今成長的原因。
