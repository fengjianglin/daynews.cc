全數據創生涯新高！球風沉穩如30歲，他將成為雷霆崛起的根基

2019-11-26 0條評論 14次閱讀 0人點贊

他們不想交易亞歷山大。

但是為了得到喬治，為了讓倫納德安心回家，他們做出了讓步。哪怕心碎成玻璃渣，這個字還是得簽，成大事者必須放棄一些東西。

他們是誰？他們是冠軍教頭道格-里弗斯，是火眼金睛的傑里-韋斯特。但他們依舊是贏家。

坐在他們對面的，是以普雷斯蒂為首的雷管。對於雷霆來說，這一年是支離破碎。他們失去了兩個超級明星，一夜之間從季後賽球隊跌入樂透區。

前程黯淡無光。講道理，那一夜雷管們並非一無所獲，至少他們把亞歷山大招來了。

「他真的不是一名無關痛癢的球員，而是球隊非常重要的補充。他能加盟球隊，我們真的非常高興。我們能讓他更快地成為一個更好的球員，他也具備這樣的潛力。」

從普雷斯蒂的言語中，我們看到了雷霆本賽季的目標：培養新人。

或許，這是別無選擇中的最好選擇。他們手上握著一堆選秀權，並且他們還擁有保羅，他能給新生代後衛帶來太多東西。

就目前來說，亞歷山大並沒有辜負雷管的培養和期待。15場球，場均打35分鐘，他能拿到19.7分5.3籃板以及3個助攻。同時，他每場球能扔進1.7個三分球，命中率高達40.3%。

我們看到了亞歷山大的變化，看到了他的進步。

從他2018年拉斯維加斯夏季聯賽的首次亮相，到季後賽第一輪第6場對陣勇士隊能得到22 分5 籃板6 助攻的比賽，吉爾傑斯-亞歷山大在攻防兩端的能力，場上打球時的機智，節奏的變化，絲滑平順的中距離跳投和籃下終結時獨特的動作都給大家留下了深刻的印象。

新賽季揭幕戰，面對老冤家猶他爵士，亞歷山大全場攻下26分，這也是他來到雷霆的首秀。賽後米切爾表示：「他的比賽進步了，他會是難以對付的傢伙。我想還有很多人對他的看法停留在過去，但他已經起飛了。」

第二場，28分7籃板4助攻；第三場，19分9籃板4助攻；第四場，23分8籃板4助攻……數據的背後，是他的能力正在逐漸被激發，以及保羅對他的影響。

“我很感激保羅在不同的投籃訓練中考驗我，我學習到了很多。他也讓學會了從不同的方面來看待比賽，這真的很有趣。”

翻看了一下他的球探報告：

在進攻端，突破過程中能夠控制好自己的身體，雖然較為瘦弱，但是不會動作變形。終結能力也不錯，算是本屆新秀頂尖的，可以運用各種技術讓自己突破到籃下。利用身高終結。

中投比較穩定。能夠在突破過程中瞬間停下來，創造出急停跳投的空間。

視野極佳。能夠洞悉防守，傳出好球。

在防守端，能夠用他的長臂捅掉對手球員的球。破壞對方運球，對陣小個球員時優勢更大。能夠擋在對位球員和球之間。一雙長臂更是能夠很好地破壞傳球路線。

他會是一個很棒的雙能衛。

身高1.98的他，擁有超過2.10的臂展，這讓他能在防守端給對手很大的壓力。我們都知道，攻防一體的後衛，很難得。而現在亞歷山大可以師從保羅，這個後衛線上的攻防高手。

他很瘦，瘦到讓人以為弱不禁風。可是，他是一個突入禁區的好手，並且很擅長找對抗。

亞歷山大要走的路還很長。

或許先從防守端做起。他和馬刺的默里有些相似。長臂，瘦弱。默里已經是最佳防守陣容二陣成員，亞歷山大可以從默里身上學習一下。

如果亞歷山大進化完成，他會是一個明星後衛。我很喜歡他的風格。這位早熟的控球後衛在過去一個月中不斷進步，成為了一名出色的射手，突破手，組織者和傳球手。

肖恩-利文斯頓，便士-哈達威，科懷-倫納德，保羅-皮爾斯，拉簡-隆多，邁克-康利。

以上都是球探們給他按的模板，我們看到了小卡、看到了真理。這也說明，亞歷山大身上的潛力巨大，等待著雷霆去開發。

這也讓我很期待，期待他能成為一個出色的全明星，並且扛起雷霆復興的大旗。
