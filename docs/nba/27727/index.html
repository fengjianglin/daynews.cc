今日要聞：科比等人首次入選名人堂提名，努瓦巴跟腱撕裂

2019-12-21 76條評論 1,391次閱讀 443人點贊

虎撲12月20日訊 今天NBA有4場比賽，虎撲為大家總結了今日要聞。

奈史密斯名人堂公布了2020屆的首次入選提名的名單，入選的前NBA球員有科比-布萊恩特，蒂姆-鄧肯、凱文-加內特、克里斯-波什、肖恩-馬里昂、邁克爾-芬利。此外還有兩名WNBA球員塔米卡-卡欽斯（Tamika Catchings）和斯溫-卡什（Swin Cash）入選。

比賽進行到第四節還剩10分54秒時，籃網側翼大衛-努瓦巴在無對抗的情況下受傷，隨後在隊醫的攙扶下返回更衣室，在返回更衣室的途中他的右腳完全不能受力。 據NBA記者Shams Charania報道，努瓦巴被診斷為右跟腱撕裂。在這次受傷前，努瓦巴已經進入籃網的常規輪轉陣容中。 根據雅虎體育記者Chris Haynes的報道，努瓦巴將在明天接受手術。

根據Jovan Buha的報道，聯盟相關人士普遍認為哈勒爾明年夏天至少將得到一份4年均薪為2000萬美元的新合同。因為明夏自由球員市場中明星球員並不多，聯盟相關人士甚至認為哈勒爾極有可能會得到一份總價上億美元的合同。

安吉接著表示，海沃德目前是每日觀察狀態，球隊預計可以在接下來的核磁共振檢查中能得到更多信息。儘管這次傷病與兩年前海沃德重傷時是同一隻腳，但這次是腳趾受傷，與之前沒太大聯繫。

根據Heavy網站記者Sean Deveney的報道，一位聯盟高管透露，灰熊對安德烈-伊格達拉的要價沒變，要一個首輪簽。

本賽季的前29場比賽里，阿德托昆博已經16次砍下30+10，這是自1976-1977賽季NBA和ABA合併之後的最高紀錄。

5記三分球是字母哥單場三分球命中紀錄，此前的紀錄是4記，字母哥已連續21場命中三分球。

8個助攻也讓詹姆斯職業生涯常規賽助攻數也來到8967個，超越加里-佩頓（8966），位列歷史第九。

詹姆斯個人職業生涯的出場時間也來到了57279分鐘（常規賽出場時間為47230分鐘，季後賽出場時間為10049分鐘），超越了湖人名宿科比-布萊恩特的生涯出場時間57278分鐘（常規賽出場時間48637分鐘，季後賽出場時間為8641分鐘），位列歷史第三。


根據The Athletic記者Erik Horne的報道，俄克拉荷馬城一家購物中心今日發生槍擊案，在槍擊案發生的時候，雷霆球員們正在該購物中心內看電影。
