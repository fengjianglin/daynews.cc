天賦溢出的鵜鶘隊為什麼會輸不停？

2019-12-25 2,403條評論 3,848次閱讀 1456人點贊

即使蔡恩-威廉森一場比賽都沒有打，但是鵜鶘陣中明明還有朱-霍勒迪、布蘭登-英格拉姆、JJ-雷迪克等人，這些實力與天賦的集結，怎麼會打出和勇士「G-League 代表隊」相仿的戰績呢？

在與安東尼-戴維斯 的鬧劇終於落幕過後，新奧爾良手擁狀元以及一票潛力無窮的小將們，開季前被視為季後賽大黑馬， 沒有想到一個多月過後，鵜鶘卻是電梯直線向下。在與奧蘭多魔術交手的比賽中，主教練金特里 第三節吞下第二次技術犯規，狼狽出場也扭轉不了球隊連敗（119-130）的命運，可說是鵜鶘本賽季的谷底。鵜鶘從11 月底開始12 連敗，近乎整整一個月不知贏球為何物；就算目前暫時止敗，也只有7 勝23 負的慘烈戰績，和萬劫不復的尼克並列聯盟倒數第三，只贏過輸不停的勇士和老鷹。

鵜鶘應該非常想念他們半年前剛選進的超級新人。蔡恩-威廉森 本當在這個賽季出賽，讓鵜鶘的球員撫平他們失去安東尼-戴維斯 的傷痛；然而這名大物新秀的傷勢讓他從開季以來就一直作壁上觀，也讓鵜鶘一點把戲也變不出來。

在過去一波慘痛的12 連敗當中，鵜鶘平均每100 回合會輸給對手12.6 分；即使蔡恩-威廉森一場比賽都沒有打，但是紐奧良陣中明明還有朱-霍勒迪、布蘭登-英格拉姆、JJ-雷迪克、朗佐-鮑爾 以及約什-哈特 等人，這些實力與天賦的集結，怎麼會打出和勇士「G-League 代表隊」相仿的可笑戰績呢？鵜鶘到底怎麼了？

鵜鶘在前一波12 連敗當中，打的籃球一點也不吸睛。他們自從11 月21 日擊落太陽之後（太陽陣容最佳的三名球員：艾頓、路標、貝恩斯 都沒有出賽），就開始一勝難求。至今他們的其他6 場勝利，則分別由開拓者（利拉德 未上場）、勇士（拉塞爾、湯普森、庫里 都沒有出賽）、快船（什麼!??不過萊昂納德也沒上場） 、森林狼（唐斯缺陣），真正能說鵜鶘真材實料拿下的，只有萬聖節當天力克的丹佛掘金，以及陣容潛力度大大遜色的夏洛特黃蜂。

在鵜鶘所吞下的無數敗仗中，有些真的可說是一敗塗地。在主場被魔術拿下130 分，就和他們之前在4 天當中和獨行俠交手兩次，一次主場11 分之差落敗、另一場居然在德州以38 分之差被屠宰一樣！對上聯盟唯一主宰雄鹿被虐15 分合情合理、慘輸快船25 分、被衛冕冠軍贏18 分也就算了，但是就連勇士、籃網都輸到兩位數…這樣的表現與其說鵜鶘是季後賽大黑馬，還不如說是搶狀元的大黑馬。

哪裡出了差錯？

前火箭防守教練傑夫-布茲德里克

鵜鶘隊的凈效率值（net rating）排行聯盟第26，平均每100回合，將會落後對手7.01分；他們的進攻排行聯盟第20，已經不算太好；輸不知防守端根本就是紙糊的，排行全聯盟倒數第二，僅次於根本沒有防守（？）的奇才隊。

也許我們根本就對這些小將寄予太多厚望？鵜鶘輪替陣容中的 朗佐-鮑爾、布蘭登-英格拉姆、約什-哈特、肯里奇-威廉姆斯、尼基爾-亞歷山大-沃克、弗蘭克-傑克遜 以及 賈克森-海斯 全都是一些職業球齡不到4年的小夥子，再加上超級新人蔡恩-威廉森，全隊便有8人還完全無法參透能在聯盟穩定贏球的方法。

防守端的無力絕對是拖垮鵜鶘隊戰績的罪魁禍首。專欄作家Mike Prada已經點出全隊防守端的許多大問題：包括防守換人的無力、輪轉或調度過慢，甚至是基本的戰術都令人搖頭。鵜鶘隊的防守是由前火箭防守大師Jeff Bzdelk所帶領，但他的指導方式對於這些乳臭未乾的小夥子來說，似乎還是太難以理解了，導致球季至今，鵜鶘隊在平均每100防守回合中還比去年多丟了2分。

對於這種大規模的崩壞，我們很難在攻守兩端中揪出一名戰犯；然而曾經的選秀榜眼朗佐-鮑爾本季的表現確實令人相當失望──甚至連先發的地位都因此不保。在防守端，他和 朱-霍勒迪 的組合理論上和封鎖對手的外線，但這事並沒有發生。進攻端上，他改良過的跳投姿勢也沒有收到效果，本季到目前為止，他的整體命中率38%，外線也只有33.6%的水準。

缺少了超級新人的助陣，加上迷航的年輕潛力，以及防守端的一團混亂，造成鵜鶘隊至今的慘淡表現。即使蔡恩-威廉森在新年回歸，恐怕也別太期待鵜鶘能因此起飛。

一團亂當中的唯一亮點

布蘭登-英格拉姆總算打出當初被預期的身手，他甚至一口氣提升至全明星的級別。這名安東尼-戴維斯交易案中湖人包裹里的最大咖，目前場均可以轟下25分、命中率更達50%之譜，三分球不僅投得多（場均6次出手）還投得准（命中率41%），他更將自己的罰球命中率拉到八成以上（84%，場均6次出手），還外加7籃板以及4助攻。

布蘭登-英格拉姆 儼然成為鵜鶘陣中的唯一之星──至少在進攻端無庸置疑。

所以…鵜鶘該怎麼辦？

鵜鶘隊正面臨著一連串困難的決定，但其中有些或許比較簡單。在目前這個階段，鵜鶘應該當個交易截止日前的賣家，他們可以考慮為JJ-雷迪克或是 德里克-費沃斯找到下一個落腳處，以換來更多年輕拼圖或是選秀簽，以利未來的重建作業。

即使已經高齡35，JJ-雷迪克仍然保持著46%的超高三分球命中率，每場比賽出手超過7次，而且他的薪水也不貴，還剩下一年1300萬的合約。鵜鶘能利用他換來另一支首輪簽嗎？德里克-費沃斯則是一名實用性高的替補長人，而且本季他的合約也將到期。鵜鶘隊其實不急著要兜售他們兩個，不過至少可以待價而沽，聽聽其他急於補強的球隊是否會因此拿出更甜的籌碼。

更加棘手的問題在於教練團：金特里真的是球隊未來長期規劃的合適人選嘛？傑夫-布茲德里克 真的能完美複製他在火箭隊的成功經驗嗎？（當然，以目前來說是完全失敗的）這些決策將會對未來造成更大的影響，不過在蔡恩-威廉森 一場未打之前，要對此對出決定也不盡公平，紐奧良目前也只能密切觀察一切。

唯一能確定的是，即使蔡恩-威廉森傷愈復出，也難以拯救這隻困在淺灘上的鵜鶘。紐奧良原先在本季想有一番作為卻未能如願，這並沒有什麼大不了，今年賽季只是鵜鶘重建的第一年，他們還處在「過程」的開始階段；然而，開季的不順確實可以球團刺激開始思考，他們想像中的未來到底該是什麼樣子。
