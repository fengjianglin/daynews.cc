第二春的保羅還是負資產？普雷斯蒂只要做成交易就不虧？

2019-12-20 1,239條評論 3,414次閱讀 1624人點贊

克里斯·保羅因為4年1.6億的大合同被球迷批評為毒藥合同，但轉投雷霆的保羅最近似乎煥發第二春，進入12月份以來保羅場均17.4分4.9籃板7.7助攻，命中率51.4%，三分球也有40%的命中率，加上93.9%的罰球命中率，進入了180俱樂部。

但是保羅優秀的發揮也沒有打消雷霆交易他的決心，關於保羅的交易流言一直沒有停下過。而普雷斯蒂能夠把保羅成功交易出去嗎？意圖重建的雷霆能用保羅換來什麼有價值的籌碼？

  • 雖然保羅第二春，但仍是雷霆的燙手山芋

雷霆因為之前收集了不少大合同，即使是夏天交易了兩大球星，工資還是達到了要繳納奢侈稅的水平。所以年薪高達4000萬的保羅還是雷霆的燙手山芋，由於保羅的存在，雷霆無法輕裝上陣進行重建，反而保羅現在正在越打越好，帶著雷霆打到了西部第7。

但是，有沒有打進季後賽對於雷霆來說不重要，他們更關心球隊能不能在選秀大會上拿到更靠前的名次，和能不能騰出空間降低運營球隊的成本，同時在球員市場簽來價廉物美的實用球員。

而年薪4000萬的保羅和雷霆背道而馳，所以普雷斯蒂一直在叫賣保羅。

雖然保羅能夠帶領雷霆打到西部第7，但是想要靠著34歲的保羅將雷霆帶到新高度還是太過困難，而季後賽首輪游對雷霆來說沒有什麼意義，因為西部第七的背後是保羅、亞當斯、加里納利、施羅德等人組合起來足以和爭冠球隊匹敵的高昂工資，雷霆更需要年輕、性價比高的球員，重新打造球隊的薪資平衡。

但雷霆交易保羅的心思昭然若揭，對保羅有興趣的球隊也基本上都是聯盟強隊，而這些球隊沒有足夠的空間吃下保羅。

加上這賽季，保羅的合同還有兩個賽季才能到期，雖然保羅這賽季有所回暖，但也只是場均16.3分4.籃板6.3助攻的水平，4000萬年薪肯定是溢價了。

所以，強隊們在對保羅的即戰力感興趣同時，也需要掂量一下保羅的大合同對球隊未來的損害。

不過，對雷霆出手保羅存在利好消息的是，現在的保羅越打越好，這肯定會吸引更多強隊的興趣。

  • 保羅能換來什麼？

交易保羅是一件複雜、困難的事情，首先第一步是需要有球隊感興趣，而從雷霆交易保羅的流言傳出以後，對保羅感興趣的就只有熱火和雄鹿。

第二步，需要有球隊配平保羅交易的工資，而保羅本賽季3800萬的年薪也正是達成交易最困難的一點。想要配平保羅的工資，要不就是需要對方拿出當家球星，而這肯定不是想要交易保羅的球隊會出的籌碼；或者讓雷霆吃下其他爛合同，但如果是爛合同，那麼為什麼雷霆不繼續留著保羅呢？

雷霆交易保羅的需求是清理空間、換來選秀權，但以現在的情況來看，想要出手保羅就已經是一件困難重重的事情，更不要說利用保羅做出「只賺不虧」的交易；即便是保羅現在狀態復甦，一名34歲的老將也很難讓買家付出太高的成本。

如果雷霆實在想交易保羅，那麼捆綁其他有價值的球員一起出手可能是好方法，但是這樣卻違背了交易保羅的初衷，損害了球隊利益。

合理的交易方案應該是，需要保羅的強隊有需要處理的年限不長的溢價合同，同時也能夠提供一到兩個選秀權，即便是次輪選秀權，對於雷霆也是有價值的。

雖然表面上看保羅不止幾個選秀權的價值，但要考慮到買方交易保羅後續承擔的風險，所以保羅的實際價值其實沒有那麼高。

對於雷霆來說，只要不為交易保羅倒貼太多有價值的籌碼，那麼普雷斯蒂的操作就算是沒有虧，因為這能給雷霆騰出大量的薪資空間，奢侈稅就能省下不少。

但從現在的情況來看，由於保羅的溢價，沒有多少球隊會願意接盤保羅。如果普雷斯蒂能夠說服買家接手保羅，那麼將繼續坐實雷霆教父的地位。
