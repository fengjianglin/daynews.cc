勇士隊得分這麼低，拉塞爾狀態低迷，他會成為勇士隊的救星嗎？

2019-12-17 959條評論 2,173次閱讀 1227人點贊

北京時間2019年12月16日，NBA常規賽繼續進行，勇士隊在主場迎來國王的挑戰，國王隊本場比賽失誤次數達到了27次，即就是這樣失誤連連的國王隊，勇士依然無法戰勝，更是可憐的只得到了79分，最終也是以100-79告負。截至目前，勇士隊的戰績來到5勝23負，暫列聯盟倒數第1。

勇士隊本場比賽中有2節得分只有17分和18分，這樣的單節得分感覺不想NBA球隊的比賽，給人的印象中，NBA球隊單節上25分感覺是很正常的事情，可是今天的勇士隊並未達到。雖然國王隊今天有27次失誤，但是他們的投籃命中率非常高，全場只有63投，卻命中了38個，命中率達到60%，勇士隊雖然投籃次數比國王隊多，達到了77次，可是只命中了30個，命中率只有38%，這是勇士隊今天輸球的主要原因。

拉塞爾今天的狀態很低迷，全場27分鐘時間只得到了8分，投籃9中2，三分3中1，拉塞爾自從傷愈回歸之後，狀態一直不是很好，到現在還未調整過來。可是勇士隊現在無人可用，隊內沒有更好的球員可以頂替上來，只能讓狀態低迷的拉塞爾硬撐著打比賽，這也是科爾最無奈的地方，所以科爾曾經表示過，對於今年的季後賽，勇士隊或許已經不再考慮的範圍之內了，當務之急是培養更多的年輕球員，給他們機會，讓他們成長。

盧尼雖然也上場比賽，但是因為傷勢並未痊癒，給的上場時間也是少之又少，本場比賽只給了11分鐘時間，得到0分，5投全失。與去年季後賽上的盧尼相差太大，傷病已經把勇士隊折騰得千瘡百孔，隊員在傷愈後的狀態也是非常低迷，這樣科爾如何排兵布陣，沒有辦法只能讓能上場的球員都去鍛煉一下吧。

再來看看格林，曾經的「五星勇」中，格林的角色非常重要，他是勇士隊策應進攻的箭頭人物，原來的「五星勇」只剩下一個格林，他再也不是那個能策應進攻的人物，也不是勇士隊每次在場上最較真的那個格林了，現在的他，更像一個沒有脾氣的小孩子一樣，為某件事情不去爭個你死我活。雖然對於現在的勇士隊，格林也是很不滿意，也是很沮喪，但是他一個人還真帶不動這支勇士隊，或許說他沒有帶隊能力，只能在多個球星的隊伍中閃光，畢竟他沒有庫里的得分能力，也沒有杜蘭特的超高命中率，更沒有湯普森的三分技能包。

接下來的常規賽，勇士隊還有很長的路要走，還有54場比賽要打，誰會成為勇士隊的救星，扛起勇士隊的大旗，伯克斯可以勝任嗎？本賽季他場均29.1分鐘，得到15.4分，投籃命中率43%，三分命中率35.6%。
