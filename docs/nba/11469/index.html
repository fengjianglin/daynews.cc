8中2，12中2，和12中8就是不一樣，利指導三分一劍封喉！

2019-12-11 913條評論 4,791次閱讀 1957人點贊

北京時間2019年12月11日，開拓者坐鎮主場迎戰尼克斯隊的挑戰，本賽季開拓者遠沒有打出人們所期盼的那樣，9勝15負排名西部第12。

最近開拓者又是先後敗給了湖人和雷霆，但是利拉德的表現依然是十分出色，賽季至今場均36.9分鐘，26.7分4.9籃板7.4助攻，並且可以用35%的命中率場均投進3.5記三分，但是他有些比賽中，手感也會受到很大的影響，例如在對陣湖人時3分球8中2，對陣雷霆時12中2，這也對比賽的走勢影響巨大，但今天的利拉德找回了「瞄準鏡」，讓我們一起回顧一下利指導帶著「瞄準鏡」是怎麼打球的吧！

首節開始，尼克斯反客為主，先是打了一波小領先，利指導一看二話沒說，刷刷刷三分手起刀落，急停干拔不講理迎射打進，這還不算什麼，下個回合利拉德拿到籃板推進至前場，突射冷箭，籃球劃破寂靜的天空留下一道美麗的拋物線，回來再戰，又是接懷特塞德助攻，拔起來就扔，還是一樣的三分，還是一樣的不講理打進。三分投完來秀個操作，連續胯下運球，後撤步迎射也進，不過這兩分似乎沒有三分能夠把比分拉開的快啊！於是，利指導首節還有1分02秒時，藉助掩護直接拔起來就扔，同樣的摘星拿月，取下三分，0分33秒三分球再次彎弓搭箭，籃球不偏不倚，正中靶心，首節戰罷，利拉德三分球5投5中，拿下20分，幫助開拓者領先10分！

次節再戰，休息了大半節的利拉德再次回到賽場，但他火熱的手感並沒有受到絲毫影響，上來便是一記美如畫的三分跳投，再進。幫助球隊穩住局勢。回來再攻，利指導這次面對尼利基納的防守，不慌不忙，拔起來就要投，球雖然沒有成功進入籃筐，但是裁判響哨，隨後利拉德三罰三中，繼續幫助球隊擴大領先優勢。

半場結束，開拓者隊60-41領先尼克斯隊19分，利拉德砍下25分的高效發揮！

易邊再戰，利指導先是助攻安東尼三分遠投空位打進，而後利指導自己也是藉助掩護，同樣的配方，今天只「鐘意」三分，再次穩穩命中！隨後又是直接接到球，虛晃一槍，然後面對防守迎射干拔打進，超遠三分飈中。隨著利指導又是助攻西蒙斯三分穩穩打進，三節戰罷開拓者86-57領先尼克斯隊29分，末節比賽也是早早進入了垃圾時間！最終開拓者115-87主場戰勝尼克斯隊，終結了自己的2連敗，也送給了尼克斯隊10連敗！

數據方面:利拉德出戰28分鐘砍下31分6助攻，三分球12投8中，安東尼戰舊主16分，麥科勒姆13分，蘭德爾15分，莫里斯十分，巴雷特9中1僅得5分！

最後希望開拓者越來越好，尼克斯也早日能完成重建，希望球員能夠保持健康，球迷們理性觀球，身體健康闔家歡樂！我們大家明天見！一起分析精彩的球賽！
