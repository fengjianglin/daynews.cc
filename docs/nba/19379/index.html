湖人抵達印城偶遇大雪!隊員們機場打雪仗撒歡 庫茲馬被麥基KO假摔

2019-12-18 136條評論 2,903次閱讀 1358人點贊

12月17日消息，明天上午，湖人將繼續客場之旅，對手是印第安納步行者。今天，湖人抵達印第安納之後遭遇了大雪，隊員們下了飛機之後看到白茫茫的一片，麥基、庫茲馬、考辛斯等人直接在機場打起了雪仗。

​印第安納白雪皚皚，白茫茫的一片，讓人很驚訝，湖人的包機怎麼能夠在這樣的條件下降落的，不得不佩服機組人員的操控能力。

​湖人隊員們就像南方孩子第一次來北方，沒見過大雪的樣子，全都變成了一個個三歲孩子，居然就在機場打起了雪仗。

​庫茲馬更新社交平台表示：一下飛機，我像一隻野外無辜的小鹿，被一個兩米大漢麥基的雪球一擊命中。

不過庫茲馬並無大礙，只見他假摔，慢動作回放一下緩緩躺倒在地。

​考辛斯和麥基曬出了視頻，麥基和考辛斯玩起了1V1的扔雪球比賽。

​考神躲在車門背後向麥基發動著進攻。

​有趣的是，麥基似乎還隨身帶了瓶紅酒，打雪仗的時候就擱在地上，十分搶眼。

​而卡魯索等幾位隊友也用手機記錄下了這一刻。

​湖人在上一場客場之旅射落老鷹豪取7連勝，湖人目前24勝3負高居聯盟第一，看上去大家心情都很不錯。

網友們都表示，「震驚！湖人球員機場發生混戰！打得滿身是雪！」「湖人隊內不和實錘，剛下飛機就打架」「考辛斯為什麼客場也跟著?那傷不用養嗎」「走路不影響，跟著看飲水機熬心靈雞湯場邊vip座看球不香嘛」「內訌實錘」「一群長不大的孩子」「一群鐵憨憨」「湖人無論是戰績還是隊內的氣氛都是聯盟獨一檔」「庫茲馬，我只是個地主家的傻兒子」「專機，專車，年薪過億……的一群人」「湖人全員逐漸詹化」「步行者:這他媽是來打客場的嗎，還有，那瓶紅酒過分了啊……」「地上一瓶紅酒是帶給詹姆斯的嗎？」
