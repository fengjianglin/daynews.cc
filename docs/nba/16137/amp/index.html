聯盟現役4大球星招牌動作：詹姆斯的坦克突破，杜蘭特的標誌中投

天天要聞
2019-12-16

江山代有才人出，各領風騷數百年！之所以NBA球星可以馳騁賽場，往往都會獨有一門絕技，作為自己的標誌性動作，以此成為得分利器，雖然隨著一代代球星的退役，可現役球星的絕技比起前輩們也是不遑多讓。下面就來看一下大家所熟知的聯盟現役4大球星的無解絕招吧！

4.詹姆斯的坦克突破

現在詹姆斯靠一招坦克突破依然所向披靡，而在年輕時期，這一殺招被使用得更加頻繁，也更加有效。早年的詹姆斯甚至就是靠這一殺招就能拿大部分的得分，身體強壯，力量十足，讓聯盟一眾防守悍將就只能面面相覷，無可奈何。遇到阿泰斯特這樣的野獸派防守者，詹姆斯的坦克突破才被稍加限制。

3.杜蘭特的標誌中投

杜少的進攻能力自然是沒得說，身體天賦爆表的他偏偏不靠身體，他是走技術流路線的，主要擅長變向過人和外線的精準跳投，其實一開始阿杜剛進聯盟的時候進攻還沒那麼強，也常常給人防的沒脾氣，但是有一年他突然開發了大幅度變向這個技能，一下子得分就狂漲，如今他變向之後的跳投基本沒人防的住，中不中完全就看他那時的手感，因為他手臂長，個子高，變向的速度非常快，對方根本跟不上他的腳步，打誰都是錯位，隨後可上籃可中投，這就是他的可怕之處。

2.庫里的無死角三分

他的三分球可以說是讓球員聞風喪膽，出手速度極快，極准。庫里的三分可以戲耍大個子，頂著防守也能進球的三分？各種遠距離三分，真的是令人驚嘆！庫里三分真正偉大的地方在於無死角，真到了關鍵的時刻，甚至庫里只要一過中場的時候就會被對手包夾！當年杜蘭特在雷霆的時候就被庫里的三分打服了，甚至有時候庫里只是抬個頭杜蘭特就會不自覺地飛起來！

1、倫納德的死亡纏繞

萊昂納德死亡纏繞 ，萊昂納德的絕招「死亡纏繞」讓NBA球員個個聞風喪膽。他的死亡纏繞不是那種賭博性搶斷，萊昂納德搶斷數量多，而且不失位，防守質量超高。萊昂納德防守特點是穩而乾淨、兇悍而充滿技巧。萊昂納德連續兩年榮獲最佳防守球員的稱號，無疑是對他防守的肯定。連砍分如切菜的詹姆斯都說：「萊昂納德是他見過，防守自己最好的球員」。
