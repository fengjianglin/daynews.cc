3人得分30+，兩人同時拿三雙！熱火三巨頭時期都沒見過

2019-12-11 757條評論 3,788次閱讀 496人點贊

熱火今天在主場迎戰老鷹，本賽季人熱火還沒有在主場輸過球，他們的主場戰績為10勝0負。熱火也是本賽季目前僅有的三支主場保持不敗的球隊，另外兩支也是東部球隊，分別是凱爾特人和76人。面對排在東部倒數的老鷹，熱火最終通過一個加時以135-121取勝。主場戰績變成11勝0負，繼續排在東部第三。

在今天的比賽中，熱火有三位球員得分超過30分，兩位球員拿到三雙。其中巴特勒砍下20分18籃板10助攻，籃板創個人生涯新高，這也是巴特勒本賽季第三得到三雙，而且三次三雙都是在8天內拿到的。阿德巴約今天拿到30分11籃板11助攻，助攻創生涯新高，這是他職業生涯首次三雙，他也超越韋德成為熱火最年輕拿到三雙的球員。同時，這也是熱火隊史首次有兩名球員在同一場比賽都拿到三雙。

另外，新秀納恩今天打出生涯最佳一戰，砍下36分6籃板4助攻，得分創生涯新高，36分也是熱火隊史新秀單場第二高得分，僅次於沙曼-道格拉斯的37分。二年級新秀鄧肯-羅賓遜也發揮出色，全場16投12中，三分球14投10中轟下34分3籃板4助攻，同樣是職業生涯新高。單場命中10記三分也追平了熱火隊史紀錄。

今天這場比賽是熱火隊史第二次單場比賽有3名球員得到30+，上一次是人熱火三巨頭時期，在2011年3月28日對陣火箭的比賽中，詹姆斯、韋德和波什都砍下了30+，並且都至少拿到了10個籃板。值得一提的是，這場比賽沒有打加時。

不過，在那場比賽中，熱火沒有人拿到三雙，而今天熱火在3人得分超過30分的情況下，還有兩人拿到三雙，這是三巨頭時期也沒有出現過的。另外，巴特勒、納恩、阿德巴約和鄧肯-羅賓遜四人加起來一共得到120分，這在三巨頭時期也是沒有過的。

在韋德退役之後，熱火本賽季的表現非常出色，在新領袖巴特勒的帶領下，熱火的戰績排在東部前四。熱火本賽季迅速完成了重建，並且成為東部的一支強隊，也將是本賽季的一匹大黑馬。
