小卡：我已經被聯盟所有小前鋒尊重。巴特勒：你還沒有那個水準！

2019-12-15 314條評論 2,538次閱讀 747人點贊

在12月14日快船與森林狼的比賽結束後，倫納德在接受記者採訪時說到自己已經被聯盟所有小前鋒尊重，因為自己已經達到這個位置的最佳水準。本場比賽倫納德得到42分11籃板2助攻，24投10中，三分3中3，罰球19中19。

而對於倫納德的說法，巴特勒也做出了回應，巴特勒認為聯盟里能讓他在比賽中真正尊重的只有杜蘭特和詹姆斯，倫納德還沒有達到那個水準。同時巴特勒表示他在比賽中最不願意防守的就是杜蘭特，而最不願意碰到的是詹姆斯。

對於倫納德的說法，巴特勒回應到：「我不知道考瓦伊說的尊重是哪一種尊重，他是上個賽季的冠軍，也是FMVP，對此我非常尊重他，因為任何勝利者都值得尊重。但是在比賽中，能讓我尊重的3號位只有勒布朗和KD。我是一個不服輸的傢伙，所以我很少承認別人比我更強，如果我想我承認你，你就得拿出相應的水平，而目前來說，考瓦伊還沒有那個水平。這個位置的最佳水準是KD和勒布朗，這點永遠都不會改變。」

同時巴特勒還表示，KD是自己最不願意防守的球員，而詹姆斯是自己最不想遇到的球員：「對於我來說，KD或許是我最不願意防守的傢伙，他擁有完美的身高，能突破能偷懶，也有精準的三分球和出色的運球，面對KD我能做的實在有限。而勒布朗是我最不想遇到的球員，他是一個天生的領袖，也是這個聯盟最出色的領袖。就算過了16年，他也是現役最出色的球員，如果自己能選擇，任何人都不想和最厲害的那個傢伙交手。」巴特勒說道。

巴特勒作為聯盟里聞名的打球很硬氣的球員，杜蘭特和詹姆斯能得到他如此的讚賞，足以看出他們兩個的實力有多強勁。的確，杜蘭特和詹姆斯是聯盟里讓人頭疼的球員，詹姆斯出色的身體能夠直接碾壓進內線而你卻絲毫沒有辦法，杜蘭特出色的投籃會讓你防不勝防，而巴特勒在之前的採訪中也說到自己願意花一百萬來找到防守詹姆斯的方法。巴特勒對小卡的回應如此犀利也並無道理，本賽季的小卡表現實在說不上有多出色，倫納德本賽季投籃命中率和三分命中率都是生涯最低，而數據方面跟詹姆斯相比也是有差距，如此表現也的確稱不上聯盟第一小前的稱號。
