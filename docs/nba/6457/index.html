詹姆斯：我們每晚帶著冠軍心態而戰，湖人隊史無與倫比

2019-11-30 0條評論 14次閱讀 0人點贊

虎撲11月30日訊 湖人今日主場125-103輕取奇才，豪取10連勝。賽後，湖人前鋒勒布朗-詹姆斯接受了採訪。

此役過後，湖人在11月取得14勝1負的戰績，這是自從2000年3月以來的隊史最佳單月勝場數。

當被問到本月戰績是否增加了球隊爭冠的信心時，詹姆斯說：「聽著，每個人都有各自的挑戰。我們在11月打得很棒，也想在12月延續出色表現。我們擁有爭冠雄心，但那不是現在要關注的事。眼前應該關注的是下一個對手獨行俠，他們很強。我們想要不斷進步，並在每當踏上球場之時帶著冠軍心態，打出冠軍水準。我們擁有經歷過奪冠的球員，他們知道應該怎麼做。這當然有幫助，但我們就想再接再厲，繼續這個過程。這是一個宏大而漫長的過程，我們不能抄近路。但我們享受這一路走來的過程。」

當被問到通過個人成就為湖人增添隊史榮譽對他意味著什麼時，詹姆斯說：「我不關心個人，我只關心團隊成就。如果我打出上佳狀態，以正確的方式打球，那麼我的隊友就會打好。我只關心這個。我說過了，作為一個身披紫金球衣的整體，能夠成為這支球隊的一員，成為隊史成就的一部分，這是一種榮幸。但我個人只關心隊友和隊友的表現。這就是我努力訓練的原因，如果我打好自己的比賽，隊友也會從中受益。」

對於球隊打出追平隊史記錄的最佳開局戰績，詹姆斯表示：「這是一份榮幸。想想這支球隊歷史上的那些輝煌之師以及加盟的球員吧，你就會想到許多支自己見證過的湖人隊，或者沒有親眼所見但讀過或看過集錦的隊史球隊；你也會想到那麼多球員和傑出教練，這傢俱樂部的歷史是無與倫比的。能夠讓自己躋身一些隊史偉大人物之列，感覺很酷。」

本場比賽，詹姆斯上場25分鐘，16投9中得到23分3籃板11助攻2搶斷1蓋帽。
