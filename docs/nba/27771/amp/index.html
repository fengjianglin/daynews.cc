莫蘭特：灰熊隊崛起的希望，最佳新秀的有力爭奪者

天天要聞
2019-12-20

NBA常規賽已經開始兩月有餘，在狀元蔡恩-威廉姆斯因傷未能出戰的情況下，榜眼秀莫蘭特正在用自己出色表現捍衛著19年新秀的榮耀與尊嚴。全面的技術，出色的身體素質，優秀的控球能力與廣闊的球場視野，這就是莫蘭特，灰熊隊崛起的希望，也是最佳新秀的有力爭奪者。

24年太久，灰熊隊需要一個冠軍

自1995年加入NBA以來，灰熊隊已有24年的歷史了。然而如今當我們說起這支球隊的時候，人們會想起什麼？大加索爾帶隊在季後賽的多次折戟沉沙？小加索爾與蘭多夫的黑白雙熊時代？2011年季後賽的黑八奇蹟？雖然在過去的20多年裡，這支球隊也曾書寫出了屬於他們自己的故事，但是時至今日，他們卻始終未能為他們所在的城市帶來一座金燦燦的奧布萊恩杯，在溫哥華的時候沒有，在孟菲斯的時候也沒有。

溫哥華灰熊隊

作為NBA海外擴張計劃的產物，1995年，溫哥華灰熊隊與多倫多猛龍隊一起加入了NBA，成為了當時NBA聯盟僅有的兩支非美國本土球隊。加入NBA的前幾個賽季，灰熊隊的戰績非常的慘淡，單月0勝17負，23連敗，單賽季只有14勝，這都是當時灰熊隊交出的成績，其中有些數據至今依然保持著NBA的最差記錄。

建隊的第三個賽季，雖然球隊在阿卜杜爾-拉希姆與里弗斯的帶領下有了很大的轉變，但是最終當賽季也不過是僅僅取得了19場勝利。接下來的一個賽季，因為勞資協議，常規賽場數縮減到了50場，灰熊隊只贏下了其中的8場。雖然拉希姆的數據很不錯，但是球隊的戰績離季後賽還是太遠。

慘淡的戰績導致支持灰熊隊的球迷越來越少，出於對球隊前景的考慮，當時的球隊老闆邁克爾-海耶斯向大衛-斯特恩提出了球隊搬遷計劃。2001年2月12日，大衛-斯特恩仔細考慮之後同意了海耶斯的計劃。至此，溫哥華灰熊隊便成了過往，灰熊隊正式進入了孟菲斯時代。

孟菲斯灰熊隊隊標

搬遷到孟菲斯的第一年，灰熊隊在選秀大會上收穫了一個來自歐洲的大個子——保羅-加索爾。2003—04賽季，在加索爾與肖恩-巴蒂爾的帶領下，灰熊隊實現了常規賽戰績由上賽季的50負到這賽季的50勝的巨大轉變的同時，球隊也終於隊史首次打進了季後賽。在05—06賽季，保羅-加索爾還成為了灰熊隊史上第一個全明星球員。然而，雖然與溫哥華時期相比，來到孟菲斯後灰熊隊戰績是有了很大提高，但是在季後賽上，灰熊隊卻並沒能走得多遠，他們連續三年打進季後賽，但是連續三年卻都沒能突破首輪。

2007—08賽季，由於球隊戰績不佳，灰熊隊送走了保羅-加索爾，準備推倒重建。09—10賽季，由於蘭多夫的加盟以及馬克-加索爾的成長，進入黑白雙熊時代的灰熊隊再次煥發出了他們的競爭力。在蘭多夫與小加索爾的帶領下，雖然這支灰熊隊一度成為西部的季後賽常客，他們曾在季後賽里創造過黑八奇蹟，也曾打進過西部決賽，但他們卻始終未能衝出強隊眾多的西部，未能為孟菲斯這座城市帶來一座總冠軍獎盃。

小加索爾與蘭多夫

2019年2月份，進入季後賽無望的灰熊隊交易了馬克-加索爾，至此，黑白雙熊時代迎來落幕，灰熊隊再次步入重建。交易走了馬克-加索爾以後，很多球迷對於新賽季的灰熊隊已不抱任何希望，然而隨著新賽季的來臨，當莫蘭特出現在賽場上的時候，孟菲斯的球迷們才驚喜的發現，原來這才是他們真正的希望！

灰熊隊崛起的希望

孟菲斯是小球市，很難從自由市場上吸引到什麼大牌球星。今年夏天的自由市場上，杜蘭特、歐文、巴特勒、萊昂納德等，這麼多的球星，然而孟菲斯灰熊隊卻一無所獲。在引援大牌球星無望的情況下，球隊的崛起只能依靠選秀。因此，在這種情況下，能否在選秀大會上摘得未來可期的潛力新秀就顯得尤為重要了。

賈-莫蘭特

18年的選秀大會上，灰熊隊在第四順位選中了賈倫-傑克遜。新秀賽季賈倫-傑克遜展示了出了不錯的潛力，場均可以拿下13.8分、4.7籃板、1.1助攻、0.9搶斷、1.4蓋帽的數據。曾經，很多人也都以為這個攻防俱佳的年輕人未來會帶領孟菲斯走向復興；然而，隨著莫蘭特的出現，這一切就都發生了改變。

18—19賽季，灰熊隊只取得了33勝49負的常規賽戰績，並列西部倒數第二。在選秀大會的抽籤儀式上，他們很幸運的抽中了榜眼簽，並最終用它選中了莫蘭特。至此，灰熊隊迎來了他們重新崛起的希望。

莫蘭特出身於一個籃球世家，父親是雷阿倫高中時期的隊友，在莫蘭特很小的時候，父親就已經開始對他進行籃球方面的培養。然而，跟很多大器晚成的球星一樣，小時候的莫蘭特並沒有受到太多的關注，因此也沒有什麼名氣。在進入大學的時候，莫蘭特只收到了兩份大學的邀請函，這與他現如今榜眼秀的身份顯然有些不相符。

大學時期的莫蘭特

大學時雖然沒能進入到杜克、北卡等籃球名校，但是在球星不多的莫瑞州立大學，擁有更多出場機會的莫蘭特反而獲得了更好的成長。大一賽季，莫蘭特場均可以拿下12.7分、6.5個籃板、6.3次助攻。大二賽季，由於兩個隊友的離隊，得到更多機會的莫蘭特數據上有了一個更大的提升，他以50％的投籃命中率，36％的三分命中率，81％的罰球命中率，場均拿下24.5分、5.7個籃板、10次助攻。2019年4月份，莫蘭特當選了NCAA的年度最佳控衛，2019年6月，莫蘭特在選秀大會上被灰熊隊用榜眼簽選中。

新秀賽季開始之前，莫蘭特的名氣遠不如狀元蔡恩-威廉姆斯以及探花秀巴雷特。但是當2019—2020賽季開始以後，莫蘭特卻用自己場上的表現告訴了世人，自己才是這屆新秀中最好的那一個。本賽季至今，莫蘭特場均18.9分、3.2籃板、6.5助攻、1.27搶斷，投籃命中率47.5％，三分球命中率44.2％，罰球命中率79.1％。其中多項數據排在新秀榜單中的前列，隨著他的出現，灰熊隊的崛起終於有了著落。

莫蘭特目前的表現以及他的技術特點

在19年的新秀中，季前賽的主角是狀元蔡恩，常規賽伊始階段的主角是探花秀巴雷特，但是如今隨著賽程的推移，榜眼秀莫蘭特卻才成了這屆新秀中最亮的那一顆星。

本賽季已打的22場比賽里，莫蘭特在場均29.3分鐘的上場時間裡，場均可以拿下18.9分、3.2籃板、6.5助攻、1.27搶斷，0.27蓋帽。其中得分排在聯盟第34位，新秀第一位；助攻排在聯盟第17位，新秀第一位，搶斷排在聯盟第26位，新秀第二位。

此外，在投籃命中率這一塊，莫蘭特也是表現得十分出色。他的賽季投籃命中率是47.5％，排在聯盟第43位；他的二次進攻命中率是59.1％，排在聯盟第52位；他的快攻命中率是52.4％，排在聯盟第61位。對於一個才打了22場NBA常規賽的新秀球員來說，這樣的一份數據已足夠出色。

說完了莫蘭特的表現，下面再來說一說他的特點。畢竟，一個年輕球員最終到底能夠走多遠，這與他的技術特點是有很大關係的。

其一、爆發力出色。這是莫蘭特的第一個特點，雖然莫蘭特的身高只有1米91，但是他的爆發力卻非常出色，本賽季至今他已上演了多次突破到內線隔人暴扣的好戲了，太陽隊的中鋒貝恩斯就曾成為過他的背景板。出色的爆發力可以彌補莫蘭特身高不足的這個劣勢，這讓他在外線時擁有更快的啟動速度，在內線時擁有更多的終結選擇。本賽季至今，莫蘭特在內線的得分高達10.4分，排在聯盟第20位。

其二、助攻能力優秀。這是莫蘭特的第二個特點，作為一個控衛球員，莫蘭特具有出色的大局觀與卓越的傳球能力，而這兩點都是成為一個優秀控衛的前提。本賽季至今，莫蘭特的助攻率高達35.7％，排在聯盟第12位。此外，在關鍵時刻的數據統計方面，莫蘭特在比賽關鍵時刻的助攻數是0.7次，排在聯盟第13位。目前，雖然莫蘭特的助攻失誤比只有2.06，排在聯盟第77位。但是瑕不掩瑜，這絲毫不會減少莫蘭特在助攻方面天賦的展現，而且，如今他才只打了22場比賽，假以時日，他一定會做得更好。

莫蘭特投籃熱圖

其三、得分手段豐富。無論是突破內線的上籃得分，還是在外線的中遠距離投射，莫蘭特都表現得遊刃有餘。從他的投籃熱圖上來看，除了左側45°的三分與右側45°的兩分命中率略顯不佳以外，其他的位置莫蘭特的投籃命中率都相當不錯。其中籃下命中率高達52.8％，正面和右側的三分球命中率分別高達43.3％以及45.5％。

其四、關鍵時刻不怯場。莫蘭特是一個具有大心臟屬性的球員，越是到了比賽的關鍵時刻他發揮得越好。在本賽季對陣籃網的第四節最後時刻，他先是突破內線將比分追平，然後又成功封蓋了歐文試圖絕殺的一球；對陣黃蜂隊時，他在終場前0.7秒上籃命中完成准絕殺，幫助球隊以兩分的優勢險勝黃蜂隊。

莫蘭特封蓋歐文投籃

爆發力出色、助攻能力優秀，得分手段豐富，關鍵時刻不怯場，從特點上來說，莫蘭特在未來是完全有可能成為超級球星的。當然，不積跬步無以至千里，不積小流無以成江河，千里之行始於足下，目前來說，莫蘭特要做的就是繼續在比賽中鍛煉自己，提升自己，完善自己。努力把過程做好，至於結果，交給時間就行了。

最佳新秀的有力爭奪者

對於一個新秀球員來說，最佳新秀獎自然是他們每一個人都所嚮往的，畢竟這個獎是對於他們整個新秀賽季場上表現的肯定，而且每個球員的一生也只有一次可以得到這個獎的機會。

蔡恩-威廉姆斯

本賽季常規賽開始之前，雖然狀元蔡恩憑藉著自己在季前賽出色的表現，讓很多人一度以為他已經要預定今年的最佳新秀了。然而沒想到由於傷病的緣故，到現在他還遲遲未能回來。雖然之前鵜鶘隊官方給出的消息是蔡恩因為手術需要休養6—8周，但如今時間早已過去，卻依舊沒有任何蔡恩將要回歸的消息。NBA最佳新秀的評選不僅要看球員的表現，出場率也是一個重要的參考因素，恩比德新秀賽季表現遠遠好於其他新秀，但是因為他當賽季只打了31場比賽，最終也是未能獲得最佳新秀。

目前蔡恩已經因傷錯過了很多比賽，如果他還不能儘快的回來，並且打出遠好於莫蘭特的表現，那最佳新秀估計是將要與他無緣了。當然，從目前的情況來看，蔡恩要做到這兩點非常困難。先不說回來後能不能打出比莫蘭特更好的表現，光是在短期內回到賽場上這一點，就已經讓人覺得沒戲了。

蔡恩因為傷病的緣故而遲遲未歸，蔡恩的大學隊友巴雷特又因高開低走而漸漸掉隊，在10—11月份的最佳新秀評選中，熱火隊的納恩與灰熊隊莫蘭特分別當選東西部的月最佳新秀。在整個10月下旬以及11月份的一共18場比賽中，莫蘭特的場均數據是18.6分、3.2籃板、6.4助攻、1.4搶斷，納恩的數據是16.4分、2.4籃板、3.2助攻、1.3搶斷，除了球隊戰績之外，雖然同為月最佳新秀，但是納恩的各項數據均不如莫蘭特。（註：由於本賽季NBA常規賽開始於10月下旬，11月份的月度最佳新秀的數據統計包括了10月下旬與整個11月份的共計18場比賽的全部數據）

10—11月，莫蘭特與納恩數據對比

自這個賽季常規賽開始以後，莫蘭特曾在很長的一段時間裡都佔據著新秀榜榜首的位置，雖然此前曾因為受傷與背部痙攣等原因而出現過下滑，但他回歸之後憑自己出色的表現又很快回到了榜首。12月份出場的幾場比賽，莫蘭特場均可以拿下19.8分、3.25個籃板、7次助攻，數據比上個月更上一層樓。就這樣打下去，莫蘭特離最佳新秀還會遠嗎？

結束語

本賽季至今，雖然灰熊隊的常規賽戰績依舊不佳，但是他們卻找到了球隊重新崛起的希望。我相信隨著莫蘭特的成長，在不久的將來，灰熊隊一定會走向更大的輝煌。
