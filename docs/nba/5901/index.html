詹眉炸裂高效！湖人四連大勝！喜獲升級版“瘋狗”

2019-12-09 0條評論 18次閱讀 0人點贊

北京時間12月9日，回到主場的湖人延續了近期的火熱狀態，以142比125大勝森林狼。球隊的兩位當家合砍82分11籃板19次助攻，其中濃眉哥29中20，詹皇三分8中6。濃眉在攻防兩端的表現，堪稱可怕！如此炸裂高效的表現，幾乎可以讓聯盟最佳二人組合的討論塵埃落定。

雖然比分上差距較大，但比賽的過程中，森林狼並非沒有給湖人隊製造麻煩。雖然防不住湖人兇猛的進攻，但他們仍然抓住機會分別在第二節和第三節打出18比4和15比2的小高潮，保留著追分的希望，但球隊兩位領袖31投38分的表現顯然無法帶領球隊翻盤。唐斯在內線進攻方面上表現的較為掙扎，受到了濃眉的壓制。看來破解湖人隊內線防守尤其是關鍵時刻的內線防守，已成為全聯盟的課題。

本場比賽中，卡魯索的頑強拼搏再次征服了眾人，在血染護膝的情況下仍然帶傷作戰，其戰鬥精神贏得了所有人的肯定。目前看來，卡魯索已成為湖人版的「瘋狗」。與貝弗利相比，其積極性不相上下，但禿曼巴打的更為聰明， 犯規控制的更好，防守效率更高。相信在下一次得洛杉磯德比中，兩位「瘋狗」的相互撕也將成為比賽的一大看點。
