盤點NBA歷史上的黑八奇蹟：金州匪幫氣勢如虹，公牛大業中道崩殂

天天要聞
2019-12-21

NBA季後賽中，常規賽戰績排名前列的球隊總能佔據上風。一方面，這些球隊實力更強，另一方面，主場優勢同樣是不容忽視的因素。不過，有一些球隊完成過不可能的任務，以第八的身份逆襲了榜首的強隊，如此壯舉在NBA歷史上有過5次，下面我們就來回顧一下：

1、掘金淘汰超音速

1993-94賽季，在佩頓和坎普的率領下，西雅圖超音速拿下了常規賽西部第一，季後賽首輪對陣排名第八的掘金。當時的季後賽規則還是輪五局三勝制，前兩場超音速也順利拿下。不過從第三場開始，風雨突變，掘金隊迸發出讓人難以相信的能量，連續贏下三場比賽，逆轉晉級。其中「非洲大山」穆托姆博在這三場比賽中送出了31記蓋帽，堪稱第一功臣。他在第五場比賽勝利後，躺在地上開懷大笑的鏡頭也成為了歷史經典瞬間。

2、尼克斯淘汰熱火

1998-99賽季，NBA受勞資糾紛影響，常規賽只打了50場。邁阿密熱火以33勝17負的戰績拿下東部第一，季後賽首輪中對陣尼克斯。當時，紐約核心尤因已經賽季報銷，拉里-約翰遜不在巔峰，坎比初入聯盟，斯普雷維爾喜怒無常，但就是這樣的陣容，卻創造了奇蹟。在決勝之戰的最後4.5秒，阿蘭-休斯拋投絕殺熱火，讓尼克斯完成了NBA歷史上第二次黑八奇蹟。

3、勇士淘汰小牛

2006-07賽季，小牛隊常規賽士氣如虹，拿下了67勝15負的聯盟第一戰績，諾維斯基斬獲常規賽MVP，奪冠是他們唯一的目標。但是首輪季後賽遭遇了勇士。金州匪幫在老尼爾森的調教下，劍走偏鋒，進攻火力異常兇猛。再加上老尼爾森對小牛隊的了解，勇士成功壓制了小牛，最終他們4比2將對手淘汰出局，小牛隊的冠軍夢被打得煙消雲散。

4、灰熊淘汰馬刺

2010-11賽季，鄧肯在受傷後實力明顯下滑，馬刺重新打造了以帕克為核心的戰術體系，常規賽大獲成功，61勝21負的戰績讓他們排名西部第一。但是在季後賽首輪，他們面對黑白雙雄領軍的灰熊，陷入全面被動。鄧肯在內線顯得獨木難支，外線也受到托尼-阿倫、康利等人的限制。最終，以穩健著稱的馬刺倒在了第一輪。

5、76人淘汰公牛

2011-12NBA再次受勞資談判影響，常規賽縮水到66場。公牛隊打出了50勝16負的戰績，排名東部第一。季後賽首輪面對76人，似乎勢在必得。但是羅斯的再次受傷讓他們軍心大亂，76人則是超常發揮，全隊合力用六場比賽解決了公牛，從那以後，公牛隊的復興大業中道崩殂。
