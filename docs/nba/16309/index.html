美媒排現役巨星入選名人堂幾率，四人鐵定入選，歐文存爭議

2019-12-13 362條評論 1,661次閱讀 721人點贊

名人堂資格的含金量遠高於球衣號碼退役或者50大巨星之類榮譽，名人堂是偉大的豐碑，也是對於球員生涯的肯定，是除了冠軍戒指之外最好的認可。

值得一提的是2016年4月4日，NBA官方宣布，姚明正式入選2016年奈史密斯籃球名人堂，成為首位獲此殊榮的中國人。

近日，美媒體給出了現役球星入選名人堂的幾率。

其中詹姆斯、庫里、杜蘭特、小卡四人是百分百可以入選的，這點相信毫無爭議，詹姆斯堪稱喬丹後第一人自不用多說，庫里史無前例的全票MVP、總冠軍、歷史第一三分王，杜蘭特MVP、FMVP總冠軍等，小卡榮譽略少但也是兩屆FMVP得主。

排在後面的霍華德、哈登、威少入選幾率是99%。

雖然現在霍華德在湖人只是一個角色球員，當曾經的魔獸也是3連最佳防守、五次一陣、四次一防、五屆籃板王、兩屆蓋帽王單換詹姆斯的存在。

哈登和威少則是前幾年的MVP，威少場均三雙的MVP足以載入史冊，一般來說獲得常規賽MVP基本就預定了名人堂位置。

而在這份名單里最有爭議的當屬歐文，歐文有88的幾率比字母哥、羅斯、湯普森還要高。

歐文有一枚總冠軍戒指，但是其他榮譽很少，個人帶隊能力更是差強人意，而字母哥、羅斯都是MVP得主，湯普森也有三枚總冠軍戒指在手，個人覺得這四人應該旗鼓相當，在入選名人堂這一塊，羅斯雖然是最年輕的MVP但畢竟後期因為傷病，已經不算巨星行列。
