字母哥和東契奇在國際球員中地位，能和姚明比嗎？肖華在戰略轉移

就現在來說，論及姚明當年在國際球員中的地位，比現在的字母哥和東契奇還是要高的。說到地位，要考慮的因素還是非常多的。

就算是現在，我也認為，健康的姚明能打得字母哥懷疑人生。單就一點吧，字母哥投籃太差，姚明在身高佔優、身體對抗不差的情況下，字母哥要拿分，可能更多的得通過攻守轉換的時候下快攻吧。東契奇嘛，目前表現是不錯。不過，時間還太短，究竟是曇花一現，還是真如球迷所言要超越詹姆斯，沒拿總冠軍之前或者至少打進幾次總決賽之前先不要討論了。肖華在戰略轉移

雖然NBA是商業聯盟，也有很多國際球員，但不存在什麼國家間的競爭。可是，即使是這樣，我們也不能忽略這些國際球員背後祖國，具體說是祖國球迷的力量。想想，為什麼當年要修改全明星的投票規則。說到這一點，字母哥和東契奇真的就照著姚明差得遠了。

姚明是作為狀元秀登陸NBA的，這也是國人的驕傲，同時也證明姚明的實力和潛力征服了球探和聯盟，這得益於姚明本身出色的自身條件和CBA率隊奪冠的經歷。姚明是名人堂成員，這所榮譽殿堂是對NBA巨星的最大肯定，只有像喬丹、魔術師、大鳥等等這樣級別的球員才能入選。姚明的入選本身就很能說明問題了。

姚明的打法讓NBA眼前一亮，這樣的巨人，有著靈活的內線腳步，有著精準的中距離投籃，有著很好的防守意識，除了多樣的進攻，還能貢獻較多的籃板和蓋帽，與麥迪的組合更是把火箭帶到了一個新高度，儘管季後賽成績慘淡，但呼聲很高。

東契奇和字母哥是今年才強勢爆發的，就目前來看，這兩人對NBA影響較大，主要原因是這兩人都不是美國本土球星，是肖華在面對我國問題上的一個戰略轉移，是肖華的解決方案！因此這兩人在開拓歐洲市場上具有重大意義，也許將來歐洲市場開拓能抵消我國市場的影響力，那才有資格和姚明比較！

姚明在NBA這些年的貢獻確實是不容忽視的，入選了籃球名人堂，這對任何一個籃球運動員都是一個無上的光榮，就算現在姚明退役了，但是他留下足跡和球迷依然愛著這個聯盟，他給NBA帶來的財富和關注度是現在任何球員都很難超越的，這就是姚明在國際球員中的地位！
