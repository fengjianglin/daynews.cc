庫里打不了CBA？楊毅老師不要騙我們讀書少：寂寞大神打得挺好

天天要聞
2019-12-22

林書豪的影響力有多大？看看他來到CBA之後受到的待遇，走到哪裡都將現場變成他的「主場」，這樣的情況似乎只有以前的特雷西-麥克格雷迪遇到過。林書豪在國內有著深厚的球迷基礎，從「林瘋狂」奇蹟開始到跟隨猛龍隊奪得總冠軍後離開，林書豪在NBA效力時期從來都不缺乏擁躉。

轉戰到了CBA聯賽，林書豪受到的禮遇更甚。

賽季剛剛開始的時候，林書豪的表現相當穩健並且帶隊能力出色，他在場內場外都收穫著讚譽的聲音。雅尼斯談到林書豪的時候，從來都不吝讚美之詞，「他這樣的球員非常有價值，首先他是一個很好的人，我們欣賞的不僅僅是球員的能力，還有他的為人。對於球隊內部的氛圍來說，這是非常重要的。」

林書豪球迷無數

然而隨著CBA賽季的進行，林書豪卻遭遇到了「新秀牆」——首次打CBA的林書豪，近幾場比賽狀態出現了起伏，對陣浙江男籃的比賽更是狀態低迷。是什麼原因造成的這一切呢？林書豪難以適應CBA的防守強度，他近期抱怨CBA球員防守的言論引發了巨大熱議。

什麼？上賽季還打過NBA比賽的球員打不了CBA聯賽？是的，還真是這樣的。NBA聯賽跟CBA聯賽的比賽規則與打球方式不同，CBA聯賽對於小外援的要求相對比較高，並不是說打過NBA聯賽的球員就能勝任。當然，這裡說的「打過NBA聯賽的球員」通常都是不能夠進入NBA輪轉陣容的球員。

楊毅老師在談到林書豪CBA受到待遇的一篇文章中，就寫到：「這就是我為什麼曾經說過，對小外援來說，打CBA很難的。不光對能力的全面要求和每場接近40分鐘時間，你要面對的是千槍萬刃的針對性防守。隨你信不信，一名小外援在CBA面對的比賽強度，遠超他們在NBA面對的情況。」

楊毅老師公眾號截圖

而為了說明這個問題的重要程度，楊毅老師在文章中還舉了一個栗子——哦不，是舉了個例子，「我再說極端一點，勇士厲害不厲害？可是水花兄弟要來CBA都不好使。克萊自己運球攻的能力有限，庫里的身板兒，在CBA的規則和防守下的受傷幾率遠高於在NBA。」

林書豪不適應CBA防守強度，主要是因為林書豪對抗能力一直都不好，他能夠在NBA聯賽打出一片天地，運動能力、敏捷性以及球商是他的成功關鍵。林書豪近幾年受過大傷，運動能力下降明顯，他的突破威脅早已不再巔峰期水平。這樣的狀況讓林書豪在CBA賽場很艱難，賽季越打就越疲憊。

運動能力下降可以靠投籃能力來彌補，就像是現在效力山東男籃的哈德森，外線投籃能力可以彌補他運動能力的不足。然而林書豪的投籃狀況並不好，這就讓他在比賽中壓力倍增，「新秀牆」也就此出現。可林書豪在CBA的糟糕處境，可能會在斯蒂芬-庫里和克萊-湯普森身上發生嗎？

我們讀書好，不要騙我們好嗎？

「水花兄弟」打不了CBA？

斯蒂芬-庫里身體素質一般，這倒是不爭的事實，但是真的差到不如林書豪嗎？比較重要的是，斯蒂芬-庫里的外線投籃能力是頂尖水平，他僅憑這一手外線投籃能力就會讓對手的防守比較難受。而這樣的球員並非沒有成功先例，弗雷戴特就是身體素質不佳的球員，但在CBA聯賽卻打得風生水起。

除了斯蒂芬-庫里的投籃能力之外，庫里的高球商以及傳球水平同樣有助於他能夠應對防守者。林書豪跟斯蒂芬-庫里在NBA面對的防守強度截然不同，斯蒂芬-庫里是一支球隊的核心球員，他在NBA的成功並不是偶然。這樣來做比較的話，斯蒂芬-庫里真的還不如弗雷戴特和林書豪的適應能力更強？
