3巨頭來襲！庄神5次雙20本季第1 羅斯關鍵兩球+格里芬制勝三分

2019-12-07 0條評論 17次閱讀 0人點贊

當賽季才打到1/4賽程階段，活塞就已經迎來與步行者賽季第4次對決，也是雙方常規賽最後一次交鋒。活塞經過四節鏖戰力克步行者避免連敗，而球隊三巨頭自然是率隊贏球的功臣，也是取得對步行者賽季3-1的交鋒戰績。庄神德拉蒙德狂轟25+22的大號兩雙數據，賽季狂轟5次雙20本季第1人，格里芬則是轟下25+3+5，並在最後時刻打進位勝三分，外加羅斯打進關鍵兩球，以及貢獻14分6籃板5助攻扮演第六人領袖，如此三巨頭成為活塞衝擊季後賽的保障。

庄神：5次轟雙20本賽季第1人

儘管活塞目前尚未能夠躋身東部前八名之列，但德拉蒙德無疑正打出瘋狂一季，尤其是他基本已經成為本賽季聯盟雙20代言人。面對與步行者上賽季蓋帽王特納的對決，庄神可謂是打出一場完勝的比賽。儘管庄神17中9的準星還不夠理想，但他還瘋狂搏殺內線博得9罰7中，強勢轟下25分碾壓步行者禁區防守，尤其是瘋狂搶到22個籃板，幫助活塞全隊籃板以51-33凈勝18個，而他一人狂搶7個進攻籃板，已經超過步行者全隊僅5個。

格里芬：轟賽季第2高分+關鍵三分

格里芬固然因傷缺席了12場比賽，但如今他也是逐步找回自己的狀態，重新證明自己的殺傷力。在庄神打出攻防制霸表現時，格里芬本場也是奉獻17中7的準星，其中打進3記三分，外加罰球12中8，也是轟下25分與德拉蒙德並列全隊最高值。格里芬轟下他賽季第2高分，他賽季最高分是26分，僅有1分之差。

在末節進行到最後階段，格里芬在雙方打成101-101平時，他在還剩41.4秒時，打進了一記關鍵三分，也是他全場第3記三分，成功幫助活塞搶奪領先優勢，成為贏球的關鍵一擊。

羅斯：轟14+6+5扮第六人，獻關鍵兩球

除雙塔合砍50分打出極為給力表現外，羅斯同樣是活塞不可或缺的贏球功臣。本場羅斯僅打25分，已經轟下14+6+5的不俗全能數據，成為球隊的第六人領袖。在最後階段的決勝時刻，羅斯先是在雙方打成99平時，完成一次跳投取得領先，並在還剩18.4秒時，用一次犀利的突破繼續拉開比分，從而幫助球隊險勝贏球。

（文/醉卧浮生）
