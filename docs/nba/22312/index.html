保羅為全隊定製西裝 俄城男模隊正式上線 好眼光！完爆好友詹姆斯

2019-12-20 22條評論 2,668次閱讀 1304人點贊

保羅著西裝入場

​北京時間12月19日，雷霆隊主場對陣灰熊隊的比賽開始之前，俄城男模隊驚艷亮相成為媒體和球迷關注焦點。籃球運動員有身高並且身材極好，身著西裝入場很容易讓人聯想到時裝周走秀。雷霆隊官方在社交媒體上發布組圖並且寫道，「雷霆時裝周，感謝保羅為全隊定製的西裝」。

丹尼斯-施羅德

克里斯-保羅休賽期被交易到雷霆隊，雖然這筆交易讓人很受打擊，但是多年職業生涯的浸染，他很快就從沮喪情緒中調整過來。克里斯-保羅本賽季比賽效率比上賽季有著一定提升，雷霆隊在他的帶領之下能夠躋身西部前八之列，這也在某種程度上出乎了聯盟的預料。

雷霆隊兩位明星球員都在休賽期離開了球隊，尤其是雷霆隊最後一點標誌——拉塞爾-威斯布魯克的離隊，更是讓雷霆隊徹底進入到了擺爛階段。值得一提的是，威斯布魯克一直都是聯盟中的潮人，他在有些時候會引領球隊乃至聯盟的潮流，而現在雷霆隊的潮流時尚意識並沒有中斷。

史蒂文-亞當斯

保羅是一位球商和智商都比較高的球員，作為一名球隊的老將角色，儘管保羅可能會被雷霆隊交易，但保羅還在想著如何提升球隊的凝聚力。雷霆隊有機會打到季後賽，克里斯-保羅自然不願意放棄這個機會，如果球隊能夠進入季後賽的話，這對於隊中所有球員都是很重要的事情。

為隊友定製西裝的舉動，之前勒布朗-詹姆斯也做過。

2017-18賽季季後賽首輪，騎士隊對陣步行者隊前兩場比賽打成1比1平，第三場比賽之前詹姆斯就為隊友定製了西裝。系列賽的第三場比賽開始之前，騎士隊全隊球員都穿著整齊的西裝入場，當時也吸引到了聯盟關注的眼球。帥氣的球員魚貫而入，克利夫蘭男模隊成為賽前熱議的話題。

詹姆斯為隊友定製西裝

詹姆斯談到為隊友定製西裝時表示，「為了凝聚整個隊伍，提升信心，從而更好地應付季後賽的挑戰。」比較遺憾的是，騎士隊輸掉了系列賽第三戰，然而兩支球隊打到搶七大戰的時候，全隊迸發出了極強的鬥志。騎士隊最終贏下系列賽，並且連續第四次殺到了總決賽當中。

對比克里斯-保羅和勒布朗-詹姆斯為隊友定製的西裝，克里斯-保羅的眼光顯然要勝過勒布朗-詹姆斯，克里斯-保羅注重隊友的個性化，而勒布朗-詹姆斯則更為統一。從某種意義上說，這何嘗不是他們領導球隊的方式呢？
