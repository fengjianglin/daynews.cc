雙超巨計劃曝光！萊利慾挖角MVP+MIP聯手巴特勒，球迷：誰當老大

2019-11-30 0條評論 16次閱讀 0人點贊

本賽季熱火隊的表現讓人眼前一亮，到目前為止，他們以12勝5負的戰績位列東部第四，僅落後第一的雄鹿2.5個勝場，而且，這還是在巴特勒缺席了4場比賽的情況下獲得，如果沒有那四場缺陣，熱火隊贏多兩場比賽並非難事。

與上賽季未能進入季後賽相比，新賽季球隊最大的不同就是通過先簽後換得到吉米巴特勒，他的到來，讓熱火隊在韋德退役後找到另一位可以擔當球隊門面的球星。而賽季至今，萊利已經證明，他今夏的選擇是完全正確的，巴特勒的強硬特質與熱火的球隊文化相得益彰，至少從目前來看，巴特勒如願成為球隊老大，且沒有隊友不服。

而顯然，一個巴特勒還不足以將熱火隊帶入東部精英強隊行列，畢竟，在森林狼時，他也僅僅將森林狼帶到西部第八，此前羅斯受傷，他在公牛也沒能繼續羅斯受傷前公牛的輝煌。所以，熱火能有如此戰績，還在於他們今夏收穫的兩位新秀，納恩和希羅，這兩人本賽季至今表現極其出色，希羅憑藉穩定的進攻能力，已經成為熱火不可或缺的部分，而納恩雖然偶爾表現不穩定，但是，作為一位落選秀能有如此表現，已是一大驚喜。

巴特勒+納恩+希羅這三股新力量的加入，讓熱火在本賽季極具競爭力，但是，成為東部前四，甚至前三容易，想要爭冠，顯然要困難很多。而曾經打造了詹姆斯-韋德-波什三巨頭4進總決賽2奪總冠軍的神運算元帕特萊利顯然深知這一點，所以，他的奪冠計劃並非本賽季，而是在2021年夏天開始。

據熱火隊隨隊記者Barry Jackson最新透露，他被告知，熱火隊有意招攬雄鹿隊球星阿德托昆博和步行者球星奧拉迪波到南海岸聯手巴特勒爭冠。

Barry Jackson在報道中表示，萊利今夏招募巴特勒時，就已經許諾會幫巴特勒招募來至少一位球星與其搭檔，此前球隊計劃招募布拉德利比爾，但此後他與奇才續約兩年，這讓熱火的計劃暫時落空。而現在，萊利將他的「挖掘機」瞄準了雄鹿和步行者。

字母哥和奧拉迪波兩人的合同都將在2021賽季到期，而且，如果今年再不能奪冠或者打入總決賽，字母哥很可能不會選擇提前與雄鹿續約。而步行者雖然還達不到奪冠級別，但如果奧拉迪波復出後球隊戰績無法再進一步，合同到期後他也有可能尋找新的落腳點。所以，萊利計劃在2021年夏天，這兩名球星合同到期後向他們拋出橄欖枝，儘管很難同時將兩者招攬至邁阿密，但只要能夠引進其中一人，熱火隊的奪冠希望將會大增。

字母哥是上賽季的MVP，而本賽季至今的表現並不亞於上賽季，可以說他正屬於巔峰上升期，誰也不知道他的上限會是哪裡，當然，前提是他的投籃能力必須在每年都有所提升。而奧拉迪波則是曾經的MIP，從雷霆去到步行者後，奧拉迪波的表現發生質的飛躍，不僅榮膺MIP，還入選了全明星陣容。遺憾的是，上賽季的一次重傷讓他修養至今。

如果萊利的計劃最終能夠成行，很顯然，巴特勒+字母哥的組合，顯然可以媲美全聯盟任何超巨組合。即使無法引進字母哥，退而選擇奧拉迪波，巴特勒+奧拉迪波同樣會讓他們極具競爭力，屆時，隨著希羅和納恩的成長，熱火的競爭力顯然會處於聯盟頂級的水準。

當然，在該記者曝光萊利的引援計劃後，有不少球迷開始調侃，吉米巴特勒當初是為了成為球隊老大而選擇沒有球星的熱火隊，如今，萊利要引進字母哥，那麼，到時候誰當老大呢？

更有球迷調侃道，比賽中，字母哥對巴特勒說：我突你投；而巴特勒則對著字母哥說：你投我突！

嗯，如果真是這番情景，那就亂套了！
