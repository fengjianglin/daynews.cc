賽事回顧：熱火翻盤未果，敗於灰熊；雄鹿期待翻盤，卻無力回天！

2019-12-17 2,013條評論 3,462次閱讀 1159人點贊

北京時間12月17日，熱火翻盤未果，客場敗給了灰熊；雄鹿字母哥極力輸出期待翻盤，可惜無力回天。以下是這兩場比賽的綜述：

熱火111-灰熊118

灰熊（10-17）二連勝。尤納斯-瓦蘭丘納斯得了21分10個籃板，榜眼秀賈-莫蘭特20分10次助攻，賈倫-傑克遜15分。替補出場的德安東尼-梅爾頓16分，布蘭登-克拉克14分9個籃板。

熱火（19-8）翻盤未果。吉米-巴特勒得了25分8次助攻，肯德里克-努恩18分，巴姆-阿德巴約14分13個籃板，鄧肯-羅賓遜12分。替補出場的新秀泰勒-希羅22分。

兩隊實力有明顯的差距。熱火排名東部第二，而灰熊在西部倒數第三。上一場兩隊都取得勝利，熱火擊敗了獨行俠，在對方損失東契奇之後仍打得很艱難，進入了加時。灰熊則輕鬆擊敗奇才。莫蘭特和克拉克兩名新秀復出，年輕的灰熊極有拼勁。灰熊延續這種勢頭，今天一開場就與熱火大舉對攻。首節打了不到3分鐘，灰熊就以12-5取得優勢。熱火一度將比分扳平，但防守上仍有心無力。本節最後50秒，瓦蘭丘納斯一人得了5分，灰熊以36-32領先。

熱火此前戰績出色，不光靠進攻，他們防守也不錯，但今天卻找不到防守的感覺。灰熊第二節繼續猛攻，逐漸拉開差距。本節還有57秒時，克拉克連投帶罰拿下3分，灰熊以73-56領先。熱火還以一球，上半場仍落後15分。上半場灰熊命中率高達65.9%，三分球16投11中。瓦蘭丘納斯一人拿下了17分，傑克遜12分，莫蘭特7分9次助攻。熱火命中率只有48.7%。希羅得了12分。

下半場熱火總算拿起防守的武器，不再讓灰熊為所欲為。在灰熊以80-65拉開差距之後，接下來近5分鐘他們一分未得，熱火打出一波10-0後，只以75-80落後。雙方差距一度只有4分。三節過後，熱火以86-95處於劣勢。熱火一度反超，在比賽還有5分30秒時，阿德巴約上籃得手，他們以107-104領先。在此後5分鐘，他們竟然未能再投籃命中，灰熊以一波12-1收走了比賽。終場前11.8秒，巴特勒才投中三分，但為時已晚。

獨行俠120-雄鹿116

在12月17日的賽程里，最激動人心的一場比賽是獨行俠與雄鹿的對決。最後的比賽結果是獨行俠以4分之差獲得了這場比賽的勝利，這也導致了雄鹿隊這個賽季的18連勝終止了，實在讓人覺得遺憾。從雙方球員的數據比較來看，在獨行俠隊，波爾津吉斯的得分26。其他數據都比較優秀，庫里得分也是26分，其他的數據也是不錯的。唯一遺憾的就是主力東契奇因為傷病無緣這一場比賽，可是少了東契奇的獨行俠隊也是打出了優秀的比賽。讓人們值得敬佩的隊伍。

再來看雄鹿隊的比賽得分，字母哥狂砍48分，是這場比賽的最佳球員無疑了。但是其他球員在這一場比賽中的發揮不是特別好，整個球隊得分僅僅只有4人上雙。實在讓人覺得字母哥的壓力很大啊，但其他球員也要加強訓練，爭取在下場比賽里的發揮可以更好。讓我們再來回顧全局比賽，再第一節比賽開始的時候，字母哥就發揮很好。首先突破然後遠投三分，毫無懸念球進了，然後這才是熱身，後續又遠投三分一擊命中。在這歌時候，雄鹿就已經有了優勢。

但是獨行俠隊迅速達成共識，開始進行反擊。一些球員都在外線三分球頻頻得手，就這樣把比分拉開。到第一節結束的時候比分已經是36：22，獨行俠是處於領先。到第二節的時候，字母哥也迅速組織進行反擊，打出一波小高潮。然後分差被逐漸縮小，同時獨行俠的遠投能力下降，三分得分並不顯著，變換策略換進攻方式。但是雄鹿的進攻沒有停止。直到第二節結束，雄鹿只與獨行俠相差兩分。

到中場休息結束，字母哥更是越戰越勇。一系列的突破，籃下暴扣。更是加上遠投三分和突破上籃。讓得分數達到了11分，分差不斷減小。但是獨行俠也是不甘示弱，在球員的配合下，也是將優勢又擴大到了兩位數。到第三節結束，獨行俠還是領先雄鹿去10分。到了第四節雄鹿開始反擊為了最後的勝利，字母哥三分命中，但獨行俠也是積極防守，比分一直被咬死。隨著比賽接近了尾聲，雄鹿開始抗爭，竭盡全力想要扳回比賽。但是還是無力回天。
