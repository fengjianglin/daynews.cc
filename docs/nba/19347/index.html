砍23分6助！希望這是一個好開端，這位球員太不容易了，祝他好運

2019-12-18 3,512條評論 4,633次閱讀 699人點贊

Mba常規賽繼續進行，聯盟當中有一場比賽。活塞隊對陣奇才隊。這兩支球隊目前的狀態都不夠理想，他們也急需獲得比賽的勝利來提升球隊的士氣。最終，奇才隊在客場比133:119輕鬆擊敗了活塞隊。但目前奇才隊的戰績依舊很不理想，8勝17負，這成績排在東部賽區第一十二。而活塞隊的戰績也好不到哪去。奇傷隊​本賽季的戰績不理想可以理解，球隊的核心大將沃爾因傷賽季報銷，這導致球隊的戰鬥力大打折扣，他們輸球也是情理之中。而活塞隊就情況有所不同了，他們理應取得更好的戰績。可惜本賽季的格里芬狀態全無，上賽季他打封閉針出戰季後賽真是很不理智的決定。這讓他本賽季很掙扎。希望他能夠在以後調整好自己的狀態。

這場比賽，奇才隊​贏球的最大功臣是比爾。他全場24投12中三分球9投5中，轟下了35分10助攻三籃板。他無疑是這場比賽的mVP。活塞隊表現最好的球員是替補出戰的德里克羅斯。他全場17投8中，砍下22分6籃板8助攻的全面數據。這位飽經滄桑的球員的狀態回來了，他讓那些喜歡他的球迷很是欣慰啊。與羅斯相對應的奇才隊當中也有一位這樣的大將。在這場比賽當中，他受傷歸來，全場13投9中，砍下23分6助攻。小托馬斯為自己的復出迎來了開門紅，真是一個好彩頭。

為何說小托馬斯也是飽經滄桑的球員呢？在曾經，羅斯和小托馬斯都有著很高光的表現，他們都曾是聯盟叱吒風雲的球員。羅斯入選過聯盟的全明星，獲得過mVP，入選過聯盟的最佳陣容，他還帶領曾經自己效力過的球隊殺入了賽區決賽。小托馬斯也在自己所效力的球隊當中打出了極其亮眼的表現，他也入選過最佳陣容，入選過全明星。可惜，兩位天才球員都被傷病給「摧毀」了。相比較羅斯的慘痛遭遇，小托馬斯的遭遇看起來更慘。球迷們應該都知道，小托馬斯遭遇了慘痛的背叛。如今他來到了奇才隊，希望他能夠在球隊當中打出更加亮眼的表現，為自己的未來收穫一份保障的理想合同。祝他好運。

本賽季至今，奇才隊的整體表現是很不理想的，在這樣一支球隊，新秀球員和想得到合同的球員都是一個絕佳的好機會，希望小托馬斯能夠抓住這樣的機會，爭取表現得越來越好。
