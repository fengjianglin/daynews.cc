馬龍：5年了，我第一次把快和約基奇放同一個句子里

天天要聞
3 天 ago

虎撲12月15日訊 NBA賽季常規賽掘金對陣雷霆的比賽已經結束。掘金110-102戰勝了雷霆。賽後，掘金主教練邁克爾-馬龍接受了記者的採訪。

談到本場發揮出色的愛將尼古拉-約基奇時，說道：「約基奇處於全新的狀態，我的意思是，他在不斷地認識到自己的不足，他和球隊的力量教練之間建立了良好的關係，在這方面取得了良好的進展，我覺得他正在全方面的完善自己並為之努力，無論是比賽、手活還是身體狀況，他花了很多時間在讓自己的形體上。當我看著他的時候，他看起來瘦了很多，他有了更多的能量。我很高興看到這樣。」

接著馬龍繼續就約基奇瘦身打趣道：「他真的在這方面努力了，相比過去，他的身材好了一些，他看起來很快（Fast）。」此話引起眾人大笑，馬龍接著說：「不要打斷我，他看起來有多快呢？你要知道我這5年來，第一次將Fast這個單詞和約基奇放入同一個句子里，所以他真的很快。」

約基奇本場出戰35分鐘，14投11中，得到28分14籃板12助攻，這是約基奇本賽季的第4次三雙，也是他職業生涯常規賽的第32次三雙。
