最慘新秀出爐，才打24場就涼涼！首輪秀啊，格林，這下你滿意了？

2019-12-11 1,220條評論 2,157次閱讀 1133人點贊

勇士，上賽季的總決賽隊伍，雖說今夏走了杜蘭特，但是換來了拉塞爾，球隊整體陣容沒有太大變化。

但是就是莫名其妙的戰績越打越差，目前已經是以5勝20負穩坐西部倒數第一，聯盟倒數第二，實至名歸，和聯盟墊底的尼克斯成了難兄難弟。

勇士的傷病也是本賽季戰績不佳的一個重要原因，球隊傷病最嚴重的時候，只有9個人能打球了，差點就觸發了聯盟的強制性規定。

就是一支球隊至少有8人可以上場比賽，否則就要再簽約新的球員。

不過也是因為勇士的傷病，讓球隊的替補以及新秀得到了更多的出場時間，得到了更多的表現機會。

期間，帕斯卡爾表現出色，在本賽季出戰的24場比賽中，有兩次砍下30+的表現，這24場比賽場均能得到16.6分5.2個籃板1.7個助攻，作為今年的一個二輪秀，能打出這樣的表現還是挺不錯的。

而同為今年新秀的喬丹-普爾，首輪28順位被勇士隊選中，賽季至今為勇士出戰了24場比賽，偶爾有幾場表現給力，大部分比賽是以打鐵告終。

甚至一度被球迷稱作球隊的「擺爛神器」。記得在此前的一場勇士與鵜鶘的比賽中，普爾狀態低迷，全場出戰23分鐘7投0中，沒有得分入賬。

而比賽間隙，格林也是一度在場邊對狀態低迷的普爾進行訓話，格林也是表達了自己對普爾的不滿。

繼那場比賽普爾被格林訓話後，至今才爆發了一次，三分6投5中拿下20分。而就在最近結束的這場勇士與灰熊的比賽中，普爾未能得到出場時間。

其實這也是有原因的，我們來看看普爾在12月份以來的表現就知道了，進入十二月份的前四場比賽中，第一場對陣魔術，普爾出場16分鐘8投0中，僅靠罰球拿到1分。

第二場對陣老鷹，普爾出場20分鐘5投0中一分未得，第三場對陣黃蜂又是3投0中沒有得分。

第四場對陣公牛也是3次出手未能命中，僅靠罰球得到1分。四場比賽19投0中，沒有運動戰得分，僅僅依靠罰球獲得2分。

連續多場表現如此糟糕，自然會失去球隊對他的信任，所以在最近的這一場與灰熊的比賽中沒有得到出場時間，也是再正常不過了，我們也不必感到驚訝了。普爾在球隊的未來也是一片迷茫了呀。

而就在最近，有消息稱勇士將把普爾下放到發展聯盟，也是讓普爾在發展聯盟獲得更多比賽時間，來慢慢找回自己的投籃信心。

畢竟這幾場19中0，對自己的投籃信心確實也是一種巨大的打擊呀。最慘新秀也是就此出爐了呀，才打24場就涼涼了，不知道什麼時候才能被召回了呀。

一般來說能從發展聯盟打出來的大多是前鋒位置的，後衛可以說很少了，哪怕是范弗里特，也是眾多機會堆出來的，而這一點可不是隨便哪支球隊，哪個主帥就會給的！

也不知道是不是被訓得多了，才導致狀態全無，首輪秀啊，而這下格林該滿意了，再也不用訓話了，可是勇士也找不到其他可以訓話的了呀！

被球迷戲稱為「擺爛神器」的普爾下放到發展聯盟後，不知道勇士未來會不會多贏幾場比賽，對此，大家怎麼看？
