真正的強者是不會暴露出嚴重短板的

天天要聞
3 週 ago

NBA的比賽正在如火如荼的進行中，各隊的實力也漸漸的體現了出來，強弱已經分明，現在的情況和本賽季伊始基本也一致，最有可能奪冠的還是湖人隊，快船隊，76人隊和雄鹿隊。但是儘管如此，一切還都不是最終的結果，因此需要在繼續分析一下。

湖人隊詹姆斯

湖人隊有可能爭冠是因為湖人隊在休賽期進行了很大的補強，球隊的實力已經遠遠超越上賽季。本賽季到目前為止已經取得了9連勝，戰績更是全聯盟第一，這樣的實力確實有爭冠的可能。而且湖人隊的球員配置是比較合理的，在球隊的各個位置都有可用之人。中鋒有霍華德和麥基，大前鋒有濃眉哥和庫茲馬，小前鋒有詹姆斯，後衛有布拉德利，隆多，波普和「禿曼巴」。這樣的陣容深度可以打出很多變化的配合，陣容可大可小，而且球隊還有「考神」沒算，本賽季季後賽他極有可能復出。這就是目前的聯盟第一，爭冠是少不了他的。

快船隊倫納德

說完湖人隊就不得不說一下同屬洛杉磯的快船隊，快船隊在本賽季的揭幕戰就已經戰勝過湖人隊，雖然當時湖人的球員們還沒有很好的磨合陣容，但是，快船隊的核心之一保羅-喬治還沒有回歸，現在保羅-喬治回歸，快船隊的陣容也得到了很大提升，尤其是快船隊的機動性，快船隊最大軟肋就是中鋒還不夠強硬，但是他們球員的機動性很好地彌補了這一問題，替補中鋒哈雷爾最近幾場的發揮非常出色，很好地完成了教練布置的任務，數據也很亮眼，在加上路威和「黑貝」，快船也是總冠軍的有利爭奪者。

倫納德防守字母哥

介紹完西部的兩支球隊，要說一下東部的球隊，東部球隊中，雄鹿和76人也是有爭奪總冠軍的可能。因為雄鹿隊是上賽季的聯盟第一，本賽季基本保持了上賽季的球員班底，雖說交易走了布羅格登讓球隊稍顯不適，但是他們很快調整過來，現在仍是東部第一。而76人雖然現在的戰績不怎麼樣，只是取得了東部第五的成績。但是我們不得不承認76人的首發陣容仍然值得全聯盟的任何強隊重視，他們的首發就有4個「億元先生」，只是領袖「大帝」和「西帝」還不夠強勢。他們的成長決定著76人的上限，而他們的天賦毋庸置疑，因此，這兩支球隊也是有概率奪冠的。

哈登

而其他的一些球隊就顯得不如上述的這四支球隊「強勢」了，他們或多或少的存在著各種短板，有很多球隊在常規賽中的戰績是很出色的，但是到了季後賽就顯現出了差距。因為季後賽是系列賽，球隊的任何短板都會被對手針對，缺陷會被無限放大，由此，我分析只有上述四支球隊最有可能奪得總冠軍。
