欲哭無淚！詹皇結束4連客回到家，想找妻子睡覺，卻被女兒霸佔了位置

2019-11-29 0條評論 13次閱讀 0人點贊

11月29日消息，昨天，跟隨湖人結束了4連客漫長旅途之後，勒布朗-詹姆斯終於回到了洛杉磯的家中。可是當他以為老婆熱炕頭等著自己時，卻突然發現寶貝女兒朱莉搶走了自己的位置。

​湖人以全勝戰績結束四連客之旅，詹姆斯也隨隊剛剛抵達洛杉磯並返回家中休息。

詹姆斯在推特上激動地表示：「天啊，回到家裡的感覺真好！終於結束了飛行回到了家裡，我實在是太累，太困了，就好像宿醉一樣、像是在拉斯維加斯玩了一整天。哈哈，先提前向我的家人們道個歉，明天我的精神狀態可能不會太好，愛你們呦！」

​不過，老詹很快又更新了推特訴苦道，「雪上加霜的是，當我走進我的卧室、心想著終於可以和老婆睡覺了，結果我的女兒霸佔了我的位置！！她不知道什麼時候溜出了她的房間，爬上了床和她的媽媽躺在一起。那麼，現在我必須得找另一個地方去睡覺了。可是親戚們都來城裡過節，所以家裡的所有客房都滿了，[笑哭][笑哭][笑哭][笑哭]天啊，我愛我的家人。」

​詹皇和妻子薩瓦娜結婚16年，生育了3個孩子，大兒子布朗尼15歲，二兒子布萊斯12歲，小女兒朱莉5歲。其中小女兒最得寵，是爸爸媽媽的心肝寶貝，而老詹也是著名的女兒奴。

網友們紛紛表示好笑，「去你女兒房間睡啊，塌了買新的」「去找老科喝兩杯，教教老科生兒子秘訣。」「詹姆斯這家庭地位和我差不多啊！」「驚！洛杉磯某身價過億三旬老漢竟被女兒嫌棄！」「那隻能從十幾個衛生間中挑一個了」「這是啥世道，一年賺幾個億的人老是找不到地方睡覺，最難堪的是要全世界的人替他著急」「1000平米小戶型，哈哈」「本來想小別勝新婚一下，沒想到女兒不給機會」「去韋德家啊」「這家裡怕是住了一個營」「只能和布朗尼睡一塊了」「一人掙錢養全家 各種親戚」。
