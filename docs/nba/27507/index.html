美媒評過去20年狀元紅黑榜，最黑無懸念，湖人三大紅森林狼兩大黑

2019-12-23 86條評論 1,144次閱讀 64人點贊

從2000年至今，一共誕生了20個狀元，這句是廢話！有媒體對過去20位狀元進行了排行，評出了8大紅榜和8大黑榜，湖人三狀元和活塞雙狀元紛紛進入紅榜，而森林狼雙狀元則雙雙入圍黑榜。

紅榜分別是勒布朗詹姆斯、安東尼戴維斯、德懷特霍華德、凱瑞歐文、姚明、德里克羅斯、約翰沃爾和布雷克格里芬。這8位應該懸念不大，過去20位狀元里，目前只有11位入選過全明星，而以上8位都是三次以上入選全明星的球員，除此之外，就只有本西蒙斯1次、卡爾安東尼唐斯2次、肯揚馬丁1次。

具體排名方面，詹姆斯第一毫無爭議，安東尼戴維斯排在霍華德前面就有點爭議了，僅目前而言，霍華德整體榮譽和戰績，應該都要強於安東尼戴維斯的，包括多個最佳防守球員和帶隊進總決賽。而歐文因為一個冠軍排到了第四，羅斯巔峰實在太短只能屈居第六位。

黑榜分別是安東尼本內特、格雷格奧登、馬克福爾茨、肯揚馬丁、安德魯博古特、安德魯巴尼亞尼、安德魯維金斯和卡爾安東尼唐斯。

簡單地說，除了這8大紅榜和8大黑榜之外，剩下的就是艾頓、本西蒙斯、錫安威廉森和誇梅布朗，那這幾個就暫時被列在合格的名單里，總感覺森林狼雙狀元的上榜有點意外，如果說艾頓、西蒙斯和錫安因為年限太短暫時不考慮的話，誇梅布朗怎麼樣也比不上狼隊雙俠吧。

而最水狀元，當之無愧就是安東尼本內特了，相比奧登因為傷病、富爾茨還有機會，本內特真的是實打實被證明無法適應NBA的狀元，在2013這個選秀小年，安東尼本內特本就不應該出現在第一順位，或許這也害了他。而除此之外、前六問題也不大，其實最大爭議在於維金斯、唐斯和誇梅布朗。

怎麼算，誇梅布朗都應該替掉維金斯和唐斯中的一個，進入黑榜才是，唐斯雖然球隊戰績不佳，但整體表現還不錯，2次在西部入選全明星，如果誇梅布朗替掉唐斯，可能會更合理一些，不過也許媒體對唐斯的要求比較高吧。
