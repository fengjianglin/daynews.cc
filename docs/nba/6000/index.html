3.5秒鐘可以做什麼，雄鹿隊告訴你答案

2019-12-08 0條評論 16次閱讀 0人點贊

今天早上快船與雄鹿隊的比賽，第一節還剩下18秒鐘的時候，雄鹿隊執行最後一攻。喬治希爾運球到前場跟字母哥在45°做了一個擋拆掩護，帕特森戰術犯規，因為犯規次數還沒到，所以沒有執行罰球，雄鹿隊要發邊線球，重新組織進攻。這個時候里佛斯換下了蘆葦，把貝弗利換了上來，加強一下防守。一切準備就緒，這時候距離第一節結束還有5.6秒。

7號伊利亞索瓦開邊線球，防守他的是快船隊在場上的第一高度——哈雷爾，雄鹿隊的另外兩名球員站在弱側的45°和底角。字母哥在罰球線附近，防守他的帕特森臉貼臉防守他，阻止他接球。如下圖：

進攻前的落位

喬治希爾從三秒區兜出來接球，伊利亞索瓦開完球以後往底線移動，他是有三分能力的，所以把哈雷爾調離了禁區，這時候內線空無一人。然後是字母哥和防守他的帕特森，字母哥做了一個往上走的假動作晃動了帕特森的重心，然後一個反跑直插籃下，喬治希爾瞅準時機直接將球送到手裡，字母哥直接雙手大力掛框暴扣終結。

字母哥暴扣

這次防守帕特森要負絕對的責任，他不應該在開完球之後還背對著字母哥，從而沒有觀察出來內線已經被調空，才會吃晃，所以這是防守方面的一個小細節。

進攻結束之後，計時器定格在2.1秒，也就是說這次邊線球戰術僅僅用了3.5秒就打成了，這戰術素養值得某名宿好好學習。
