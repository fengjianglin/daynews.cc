格林陷入交易流言！美媒盤點：喜歡格林的4支球隊，湖人入選

2019-12-17 3,726條評論 4,664次閱讀 3057人點贊

庫里和湯普森的受傷，勇士早已陷入了慌亂！不過有趣的是，在眾多說法之中，連球隊功臣追夢格林也陷入了交易的流言！（儘管可能性不大，但真的不影響球迷各種調侃！）

NBA有一種比較有趣的說法，勇士在過去5年能夠取得如此成就，就球隊體系而言，追夢格林至少和水花兄弟並列。

但與此同時，追夢格林的一些個人表現真的讓人很難相信，這是一位簽下頂薪的全明星？

而在勇士目前的重創期，一個不能帶隊取勝的頂薪，追夢格林陷入各種球員交易的流言並不意外！美媒《ClutchPoints》還特地盤點了4支對追夢格林感興趣的NBA球隊！

上賽季勇士西決對手開拓者同樣是本賽季的失意人之一！目前他們僅僅獲得10場比賽勝利，位列西部第11！

究其原因，他們失去了球隊三大中鋒，利拉德、麥科勒姆和安東尼的組合，他們甚至連一個高質量的擋拆都做不到，除此之外，追夢格林還可以提供他們急需的防守。

如果在湖人目前的濃眉戴維斯、霍華德、安東尼和丹尼格林的陣容里，再加一位Dpoy追夢格林會產生什麼樣的防守化學反應？

雖然湖人無法簽下頂薪的追夢-格林，但和詹姆斯同為經紀人里奇-保羅旗下球員，真的不妨礙球迷對這樣陣容產生幻想。

唐斯成為了NBA最好的大個子射手，養身喬丹威金斯砸碎了保溫杯，但森林狼雙狀元組合卻無法幫助球隊更進一步，哪怕是進入季後賽。

球風偏軟的唐斯和更適合三號位的科溫頓都無法撐起球隊的防守！就以恩比德為例，格林至少說能夠掐死恩比德！

熱火的表現讓人驚訝，他們極有可能衝擊本賽季的東部冠軍，並且已經放出了邀請全明星球員加盟的信號！

這也是球迷調侃之中最得意的「流言」之一！在黑帶拳王詹姆斯-約翰遜的照顧下，追夢-格林無疑會變得更加肆無忌憚！
