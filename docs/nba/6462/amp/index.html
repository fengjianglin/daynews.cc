NBA聯盟第一人遭針對？內線場均12.8分，罰球卻僅排聯盟第16位

天天要聞
3 週 ago

進入小球時代以來，除了斯蒂芬·庫裡帶來的三分狂潮外。罰球也成了非常重要的一種得分方式。沒人排斥這種得分方式，罰球本來也是籃球運動的一部分，合理利用規則去贏的比賽是球員球商和能力的體現。

但是自從新總裁亞當·肖華上任以來，聯盟的判罰尺度似乎確實不是同一個標準。除了人盡皆知的明星哨外，明星與明星對抗時的判罰卻也引人深思。有的球員似乎很難得到哨子，讓球迷引起激烈討論。例如：斯蒂芬·庫里的圓柱體在哪？當然比賽只有當事人能判斷場上的局勢，例如詹姆斯·哈登便能很容易地得到罰球，而今年35歲的詹姆斯卻難以得到罰球。

勒布朗·詹姆斯本賽季的表現

這是LBJ老北京的第16個賽季。時光荏苒，03黃金一代也就是狀元能屹立不倒。但是35歲畢竟是35歲，在歲月面前，沒有人能不服軟，縱使像勒布朗這樣的巨獸。本賽季的勒布朗·詹姆斯場均出場25.8分7.3個籃板11.0助攻，還有1.28次搶斷和0.61次蓋帽。投籃命中率高達49.6％，三分球命中率也有35％。正負值綜合164。老漢本賽季也是完成了自己的里程碑，成為了歷史上最年輕的33000分先生。老詹的助攻總和已經來到了8860次，僅僅差140次就能達到歷史第一位30000分+9000板+9000助攻的人。

老漢的狀態是毋庸置疑的，本賽季防守相比上賽季備受詬病的叉腰彷彿又回到了12年的防守。但是不禁令人擔心的是，如此高強度的對抗下，賽季至今無論背靠背還是連續客場，詹姆斯都無一缺席，是否有點過於拼了。湖人戰績現在穩居聯盟第一。詹姆斯也許真的可以適當休息一下。上賽季便是連續高強度作戰，最後在與勇士的聖誕大戰中腹股溝拉傷，畢竟勒布朗這種風格的運動員一定要注意不能有大傷。

場均6.0次罰球，聯盟還未學會保護詹姆斯

勒布朗可以說是籃球時代的見證者，也是締造者。2003年進入NBA。在屬於科比的那十年中，勒布朗頂著光環，欲有紫金之巔奪君之位的氣勢。那時候的詹姆斯飛天遁地，無所不能。要知道那時候的防守是真上身體。生拉硬拽那是常有的事。這也就充分地展現了他過分變態的身體素質與自律！而後，庫里掀起小球風暴，多少傳統球員倒在小球風暴的改革刀下。而勒布朗依然用強悍的實力告訴外界，NBA還是我的！

但是回過頭來。不得不說，身體素質好難道也是罪？勒布朗的球風本身便是憑藉自己的身體突破為外線的隊友製造機會或者自己內線得分。但是賽季至今勒布朗的場均罰球只有6.0次，只排在聯盟的第16位。差了第一名哈登7次，與字母哥差了5次之多。要說身體素質，那為何字母哥能有11.3次罰球呢？最令人費解的是2019年對陣灰熊的比賽中，勒布朗·詹姆斯出站39分鐘得到30分卻沒有一次罰球，突破將近十餘次，賽後詹姆斯也是像記者展示了自己手上的抓痕。這些年的勒布朗一直都是被這樣防過來的。

不難看出來，一名偉大的運動員總是用他方式證明著自己的偉大。打了16個賽季的勒布朗現在罰球總數為9809次，而10個賽季的詹姆斯·哈登如今已經有6787次了。當然這裡不得不稱讚哈登的造殺傷能力。但是勒布朗-詹姆斯畢竟已經是35歲的運動員了，比賽看一場少一場。你認為對詹姆斯的判罰標準是正常的嗎？

（羅掌柜）
