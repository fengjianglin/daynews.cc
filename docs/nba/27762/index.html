里弗斯：整支球隊失去了冷靜，不能在第四節技犯離場

2019-12-20 192條評論 1,144次閱讀 202人點贊

虎撲12月20日訊 在今日結束的常規賽中，快船以117-122不敵對手。賽後，快船主教練道格-里弗斯接受了媒體採訪。

里弗斯表示：「作為一支球隊我們失去了冷靜，這場比賽對我們而言是一次很好的學習經歷。我寧願這種情況發生在現在而不是季後賽的第四節。我們知道如果這是一次學習經歷，就一定要有相應的技戰術，那是我們能夠去控制的東西。」

在談到今日比賽中球隊後衛路易斯-威廉姆斯因不滿主裁判Kane Fitzgerald的判罰而吃到技術犯規離場時，里弗斯表示：「我認為Kane，在他與路威的爭辯過程中，給了我們足夠的機會去把路易斯弄走，但是路易斯，他已經腦熱了而讓這失去了轉機，那發生了。這種事不應該在第四節發生，無論發生了什麼，我們就是不能在第四節得到技術犯規，更不用說那是我們最好的得分手之一。」

「無論路易斯的判斷正確與否，對我來說，在那個時間點，你要成為格局更大的男人然後走開，回到比賽中去。」里弗斯最後說。

(編輯：俄城格里芬)
