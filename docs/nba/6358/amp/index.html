連中128記三分！23歲新星創造球隊新紀錄，180俱樂部他要來了

天天要聞
2019-12-02

北京時間12月2日，據菲尼克斯太陽官網的消息，23歲的德克-布克創造了一項球隊的新紀錄。他在訓練場上連中128記三分球，在視頻採訪中，布克說道：「我連續投中128個三分，真是不錯的日子，現在這是我們球隊的新紀錄了！」

對於NBA球員來說，訓練場上投三分的命中率通常都很高，我們經常會看到球員連中十幾二十個三分球，但是像布克這樣能連續投中128個三分球，卻非常罕見。就像布克說的那樣，他已經創造了球隊訓練場上的三分連中紀錄。當時，盧比奧站在他的身後，見證這一球隊紀錄的誕生。

本賽季，布克的投射命中率有很大幅度的提升。賽季至今，他出戰18場比賽，場均24.5分3.7籃板6.3助攻，投籃命中率51%，三分命中率41.9%，罰球命中率95.6%，三項命中率都創下生涯新高。

如果他能一整個賽季保持這樣的命中率，他就將成為NBA歷史上又一位進入180俱樂部的成員，上賽季，當時效力於雄鹿的布羅格登進入了180俱樂部，他的三項投籃命中率分別是50.5%、42.6%和92.8%。

布克的爆發，也讓太陽的戰績比上賽季有所提升。目前他們以8勝10負的戰績排名西部第八。不過，最近一段時間，太陽狀態回落，7場比賽輸了6場，如果他們不能及時調整，恐怕季後賽門票就危險了！

布克是2015年首輪第13順位的新秀，進入聯盟後，他很快展現出自己在得分方面的天賦，已經連續第四個賽季場均得分突破22分。本賽季，布克打得更有效率，只是他的搭檔艾頓被聯盟禁賽，太陽的實力還是受到一定的影響。當艾頓回歸後，或許太陽能重整旗鼓，再次成為西部的一匹黑馬。
