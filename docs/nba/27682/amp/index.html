西蒙斯：最現實的“詹姆斯接班人”

天天要聞
2019-12-21

原創/氧氣是個地鐵

如果說誰是最好的「詹姆斯接班人」，至今無疑當屬東契奇。雖然打法不是一回事，但從數據到習慣，東契奇都最有本事接班詹姆斯，成為未來聯盟第一人，甚至他都可能從實際行動上，以激活史上最年輕常規賽MVP使命，來實現這個過程。甚至，東契奇會真正達到「接班詹姆斯」的程度，即自成一派，像詹姆斯從「喬丹接班人」的軌跡中脫離出來，朝著第一個勒布朗·詹姆斯的趨勢發展，東契奇在未來若干年也會這麼做的。

以上就是東契奇的威力，說他和特雷·楊或者西蒙斯是一生之敵，還太低估這個皇馬舊將了。

說到本·西蒙斯，才接近「詹姆斯接班人」。在結束新秀福利後，西蒙斯和米切爾都歸於平淡，只是個普通全明星。在費城76人，他當然厲害，聯合恩比德足以競爭東部決賽。但僅此而已，西蒙斯如果還是按照老套路遞進其實很難往超巨的方向發展。恩比德帶來巨大支持同時也限制發展，而自身連出手三分都能引發集體儀式感的明顯短板傷害的是球隊戰術乃至戰略，西蒙斯或許只能是個「詹姆斯接班人」。

不過就像我曾經定位邁卡威那樣，「他把普通球員的角色發揮到極致」，同理可總結西蒙斯。當然，西蒙斯可不是普通球員，而能作為最現實「詹姆斯接班人」。這樣的評價已經足夠肯定，因為無數過早修飾的「詹姆斯接班人」都沒有什麼好動態，比如埃文斯，OJ梅奧化。錫安·威廉姆森，未知。布萊德索等以身體優勢對應的，也只是常規輪換而已，不如西蒙斯。

西蒙斯能靠天賦吃飯，是詹姆斯非常肯定的全明星，相當了不起。如此職業生涯，或許就是「詹姆斯接班人」這個聯盟十大群體之一比較靠譜的形式了。他可以被認為是隊內老大（我早在輿論推廣前就給他起外號「西帝」，西蒙斯能在恩比德身邊打球而將外號保留至今也是本事），衝擊起來大具視覺效果，圈粉和賣票任務達標，很不錯的成績。

當然，說西蒙斯是最現實的「詹姆斯接班人」只是鎖定他的現階段。如果西蒙斯能攜費城76人之勢呈現出個人總決賽版本，如果還有點總決賽MVP的變數，那就算進階，穩定下來就是質變。有概率但終究略遠，而首先第一步，就是強於恩比德。不過就目前76人而言，恩比德的戰術地位還是高於西蒙斯，他想厲害除非真的實現無差別攻擊，最重要的還是那個老生常談的弱項：投籃。

目前費城就少一個絕對領袖，像上賽季巴特勒那樣能處理關鍵球。除心理素質外，還要有方式。恩比德是中鋒干不久讀秒回合控場，哈里斯級別還差點，西蒙斯就差穩定投籃。但凡絕殺，常規手段就是壓哨三分，西蒙斯，你的在哪？

如果什麼都不改變的話，西蒙斯就是最現實的「詹姆斯接班人」而已。他只是三年級但已經三年級，想要變成更偉大，跟上偶像節奏，就是要走出舒適區。在這點看，差強人意。76人主教練早已給工作量：西蒙斯至少場均中一個三分。目前，還完成不了。對此，西蒙斯給出的解釋是不想急於求成，答案就是訓練和專心打球。希望確實如此。
