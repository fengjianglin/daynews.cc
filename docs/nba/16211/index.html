拉文打了5年拿5344分，那同屆恩比德、威金斯和約基奇呢

2019-12-15 1,835條評論 3,955次閱讀 1341人點贊

12月15日，公牛主場迎戰聯盟豪強快船，儘管喬治砍下了27分6助攻的數據，哈雷爾也有30分7籃板的表現。但公牛方面全民皆兵，老大拉文拿到了31分4籃板，關鍵時刻更是頂著哈雷爾，打進了2+1的准絕殺，最終帶走勝利。這是本賽季拉文的第2次絕殺了，這位天賦異稟的超新星，也算是兌現了天賦。

在2014年的選秀大會上，拉文第13順位被森林狼隊選中。在森林狼成長了3個賽季，拉文便被交易到了公牛隊。在這裡他成為老大，上賽季場均23.7分4.7籃板4.5助攻。本賽季場均22.6分4.5籃板3.9助攻，表現都是非常出色，他有很大機會入選全明星陣容。生涯至今，拉文已經拿到了5344分，那同屆的恩比德、威金斯和約基奇呢？

恩比德可以說是14屆的最強球員，儘管早期備受傷病困擾，但復出之後就統治籃下，迅速成為頂級中鋒。生涯至今，恩比德場均可以拿到24.1分11.5籃板1.9蓋帽的華麗數據，並且帶隊2次打入季後賽，連續2年入選最佳二陣和最佳二防，是攻防一體的優秀中鋒。不過他的出勤率太低，生涯至今只拿到了4334分。

約基奇則是當年最大的驚喜，要知道當時他只是次輪第11順位，根本不被看好成為巨星。不過約基奇的個人能力很強，是組織型中鋒。不僅得分表現不錯，還擅長搶籃板和串聯球隊。上賽季場均20.1分10.8籃板7.3助攻表現全面，最終是強勢入選全明星和一陣，已經達到了巨星的高度，5年生涯，約基奇一共拿到了5407分，表現也很不錯。

威金斯是14年的狀元秀，他有著頂級的身體天賦，當年不少球迷都說他能接班詹姆斯。雖然打法比較養生，不過威金斯的能力還是很強，劃划水就拿到了頂薪合同。生涯至今6個賽季，基本是保持全勤。新賽季場均25分5.1籃板3.3助攻表現出色，他的生涯總得分已經達到了8293分。要知道超巨倫納德也才拿了8766分，不得不說威金斯的天賦還是強大。
