科比揭秘如何欺負麥迪，並稱哈登的表現是現象級的

天天要聞
3 週 ago

麥迪和科比兩位球員，一個天賦絕倫，而另一個認真努力，「並且他們兩位也入選了00年代的四大分位陣容」，可以說是絕代雙驕，在NBA也都取得了不錯的成績。下面有關於兩人在一起的一次採訪，非常有趣！

科比與麥迪做客The Jump

科比和麥迪當時做客的是The Jump，兩人在一起談了談當年的一些往事，首先主持人提問道兩人初次見面時在什麼時候，並且被問到見面時是否知道兩人是志同道合？先是科比回答道「在自己19歲的時候，當時麥迪18歲（是不是驚訝，科比竟然比麥迪大一歲）。」他對麥迪的評價是「他就像是一個怪獸，身體條件比自己出色太多了，」隨後麥迪謙虛的回答道「我要向他學習職業精神，稱自己從來沒有和這麼刻苦的人在一起交往過，自己有運動天賦，但在刻苦勤奮這一點上，是科比影響了我」。

兩人交談

兩人在一起聊天的時候有說有笑的，還有一個有趣的話題是關於坐過山車的，當麥迪說要去訓練時，科比來了一句「現在是休賽期，訓練個毛啊！」結果麥迪被科比套路了，說要一起去迪士尼，結果來到了過山車旁，麥迪被科比拉上去，據科比透露當時麥迪一直在那裡嘔吐（採訪聊到這個話題時，兩人笑得非常開心），麥迪為了找回面子，在這裡做出了解釋，說道「當時自己剛吃過午飯，得讓肚子消化一會，但科比直接把他拉上去了，結果嘔吐了一整天」

當主持人提到了2002-2003賽季有一場比賽，兩人針鋒相對，科比說道「在那一個回合，他把我晃到了，自己感覺被羞辱的不行」（科比屬於那種有仇必報的球員哦），隨後麥迪回答道「在下一個回合，科比把我們全隊都給羞辱了，面對全隊完成了扣籃」（此時科比哈哈大笑）。

湖人隊科比與火箭隊麥迪

當主持人問到連續得分30+的球員時，對著麥迪說，你14次，科比16次。並且說到了詹姆斯哈登，科比此時給出了自己的看法。

說道「從能不能贏下總冠軍的角度來說，我並不喜歡他這樣，並認為這樣打永遠拿不到總冠軍，但你必須要將球隊救於水火之中，必有這樣打才能贏得比賽，並稱他的表現是現象級的。『』

上賽季哈登的個人表現確實是無比精彩，場均轟下36.1分，這也是繼喬丹之後首位場均得分破36分的球員，雖說上個賽季後賽輸給勇士隊，但今年夏天火箭隊進行了補強，交易來了大威少，勢必要向總冠軍發起衝擊。
