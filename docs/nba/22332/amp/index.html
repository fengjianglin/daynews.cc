49+13+10！兩新星打瘋了，巴特勒甘當綠葉，6.5億豪陣成笑柄

天天要聞
2019-12-19

NBA常規賽繼續進行，熱火客場108-104險勝76人，後者主場不敗金身被打破，之前取得了14連勝。此役，納恩得到26分，阿德巴約得到23分和9個籃板，他們率隊利用第二節攻勢確立領先位置，並在最後一節頂住了76人隊的反撲，最終守住了勝利，也完成了復仇。

比賽開始之後，76人在主場還是佔據著上風的，恩比德、西蒙斯和哈里斯也是連續拿分，幫助球隊早早確定優勢，熱火這邊手感不佳，首節以19-29落後；可到了第二節，熱火隊找回手感，76人這邊則陷入到僵局，巴特勒率領球隊轟出了27-7的高潮，在半場結束以56-48反超比分。下半場比賽76人也是找回一些感覺，與熱火陷入到拉鋸戰，可惜他們並沒有縮小分差；最後一節熱火一開始一直保持著兩位數的領先，可在最後時刻76人這邊瘋狂反撲，並在最後時刻將分差縮小到2分，可惜霍福德的三分不中，之後阿德巴約兩罰全中，比賽也失去了懸念。

熱火隊這一次能夠贏球，並不是靠著巴特勒，而是來自另外的兩大新星納恩和阿德巴約，此役兩人也是打瘋了。落選秀納恩本賽季在熱火發揮十分搶眼，本場比賽18投9中，其中三分球8中4，得到全場最高的26分，外加4籃板5助攻，正負值+19，全隊最高；在最後一節他也是命中了一記神奇的打板三分球，一度幫助球隊取得16分的領先。唯一需要吐槽的就是，他在最後時刻的兩次罰球都沒進，差一點成為了球隊的罪人，好在霍福德的三分沒有打進。

在拿下上周最佳球員之後，阿德巴約延續了出色的狀態，此役16投9中，拿到23分9籃板5助攻2搶斷2蓋帽，十分的全面，再這麼打下去，全明星都有希望了。與納恩相比，阿德巴約關鍵時刻的2次罰球則全都罰進了，直接幫助球隊鎖定勝局。

此役，這兩人聯手轟下49分13籃板10助攻，是球隊贏球的關鍵，也是搶走了巴特勒面對舊主的風頭。

重新回到76人主場，巴特勒也是得到了一些噓聲，可他並沒有受到太多影響。在看到自己手感不是很好的時候，他也是甘當綠葉，將球權更多地交到納恩和阿德巴約手中；而在關鍵時刻他也是十分穩定，突破造殺傷，兩罰全中。全場比賽11投5中，得到14分7籃板5助攻。

納恩和阿德巴約只拿著不到500萬美元的薪水，相反76人這邊則都是千萬級別的球員，其中首發陣容的西蒙斯、理查德森、哈里斯、霍福德和恩比德，這五人的合同價值高達6.5億美元，可惜這一次他們卻被落選秀打敗了。再加上他們以20勝9負的成績只排在東部第五，這也讓這套聯盟最貴的豪陣成為了笑柄，並沒有讓成績得到很大的提升。
