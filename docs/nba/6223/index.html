波波維奇還是牛！廢1人用1人盡顯老道，小將狂砍28分回報他

2019-12-04 0條評論 14次閱讀 0人點贊

文/籃郭先生

NBA的常規賽繼續進行，馬刺與紅隊進行了一場對決，對2支球隊來說，都有一些傷病的影響。比如，紅隊的豪斯、戈登，都因傷缺席了，而馬刺這邊的阿德，也是沒有辦法出戰的。再加上，紅隊是西部第五的球隊，而馬刺則是西部倒數的球隊，看起來，紅隊穩操勝券，最終卻是馬刺以135-133擊敗紅隊。數據方面，紅隊的麥克勒摩11分，威少19分10個籃板10次助攻，哈登50分9個籃板6次助攻4次搶斷，卡佩拉22分21個籃板，塔克10分9個籃板，小里弗斯19分。馬刺這邊，德羅贊23分9次助攻，蓋伊14分，懷特12分，福布斯25分，珀爾特爾6分15個籃板5次助攻，米爾斯9分6次助攻，朗尼-沃克28分。

這般結果，多少是有點意外的。因為，紅隊的實力，還是很強的。甚至於，還有人表示，對紅隊來說，他們與快船和湖人並列，是西部最強的三支球隊。這樣的說法與分析，在一定程度上，是成立的。但目前來看，湖人快船，的確是西部前二的球隊。而西部第三的球隊，是掘金，第四是獨行俠，第五才是紅隊。但在很多人看來，隨著賽季的深入，隨著哈登威少的磨合漸入佳境，對紅隊來說，排名應該是可以提升的。這樣的比賽，對馬刺來說，沒有阿德還贏球，贏的還是西部第五的球隊，多少是有點意外的。

若是看了比賽則足以知道，一名球員，算是奇兵。此人，就是朗尼-沃克，出生於1998年，是一名很年輕的球員，此前因為傷病，一直都沒有進入輪換。而如今，隨著穆雷歸來，其也是沒有多少的表現機會。但是，這樣的比賽，其還是打了35分鐘，然後，18中10，三分7中4，罰球6中4，砍下28分4個籃板3次搶斷1次封蓋的數據。這樣的沃克，算是這場比賽的奇兵。甚至可以說，也因為他的輸出，阿德缺席的負面效應，基本上被湮滅了。因為，阿德籃下的事情，被珀爾特爾幹了，但得分的事情，則是被沃克做了。如此一來，球隊贏球，便也是很自然的事情。

看起來，沃克的爆發只是偶然，但其實，也是波波維奇在對陣容進行的一種調試。怎麼說呢？那就是，此前的波波維奇，在輪換陣容的時候，基本上會用卡羅爾更多一些。客觀地說，以前的卡羅爾，還是很厲害的，但來到馬刺之後，本賽季的卡羅爾，還是有點老態龍鐘的感覺。雖然，卡羅爾才33歲，但表現並不好。如此一來，波波維奇也是大膽地用1米96的沃克擔任替補小前鋒的角色，初步來看，效果很不錯。因為，沃克的衝擊力太強了，作為「小韋德」，其個人的身體素質，真的是極好的。而在德羅贊旁邊，他要遭遇的防守壓力，也沒有那麼大，最終，他爆發了，然後，幫助球隊滅掉了紅隊。

波波維奇還是牛！廢1人用1人盡顯老道，小將狂砍28分回報他。可以說，波波維奇廢掉卡羅爾用沃克頂上來，應該是這場比賽的點睛之筆了。若是沃克繼續這樣打下去的話，對馬刺來說，可是好事。因為，他可能是他們重新崛起的機會。
