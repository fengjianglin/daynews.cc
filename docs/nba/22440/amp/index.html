7記暴扣+10中10！魔獸高效20分卻被罰球拖累，詹霍連線又上演了

天天要聞
2019-12-18

北京時間12月18日，湖人客場102-105不敵步行者，無緣8連勝，客場14連勝也被終結！安東尼-戴維斯因腳踝扭傷缺席本場比賽，而霍華德的表現相當搶眼。他全場10投10中，貢獻20分6籃板2蓋帽，其中包括7記精彩的扣籃。只可惜，他的罰球老毛病成為拖累，3罰0中，關鍵時刻沒能通過罰球得分，湖人最終落敗。

眾所皆知，戴維斯並不喜歡打五號位，霍華德和麥基能夠勝任這個位置，實際上也大大減輕了濃眉的壓力。數據顯示，濃眉本賽季只有35%的時間出現在中鋒位置上，創生涯新低。這就得多虧麥基和霍華德的貢獻了。

本場比賽，霍華德在內線的效率極高。進攻端，霍華德完成了7記暴扣。詹姆斯、波普、朗多、布拉德利、卡魯索都為魔獸送出妙傳，而霍華德也不負眾望，總能把球狠狠砸進籃筐。

第一扣！波普將球往天上一拋，霍華德心領神會，雙手掛筐暴扣！

第二扣！這一扣，堪稱本場最精彩的扣籃。詹姆斯又玩了一個腦後助攻，霍華德籃下強起暴扣，讓人彷彿看到了魔術時期的他。此前一場，詹姆斯曾胯下妙傳給霍華德上演單手劈扣的機會。

第三扣！朗多在三分線外送出妙傳，霍華德高高躍起，又是一記雙手空接暴扣！

第四扣！這一回傳球的輪到了格林，格林突破時將球往籃筐方向一拋，又是霍華德騰空而起，雙手將球狠狠摁進籃筐！

第五扣！布拉德利也給霍華德送上妙傳，魔獸單手籃下稱霸式暴扣！

第六扣！卡魯索助攻霍華德，又是一記內線扣籃！

第七扣！還是詹姆斯背身拿球，霍華德偷襲底線躍起，球到人到，再次扣籃打成！

步行者雙塔的威力還是很大的，小薩博尼斯和特納都比霍華德更年輕，但魔獸可不服老，在兩個後生面前，他的扣籃如同暴風驟雨！

詹姆斯曾把霍華德和麥基稱為讓對手頭疼的「雙頭怪」，而湖人在這兩名球員上花的錢並不多。麥基的年薪只有400萬，霍華德的年薪只有256萬。只可惜，這場比賽湖人還是沒能笑到最後。
