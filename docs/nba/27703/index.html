42歲卡特都能再就業！他今年4月場均31.3分，39歲就沒球隊要了？

2019-12-21 941條評論 2,615次閱讀 197人點贊

北京時間12月20日，39歲的克勞福德在接受採訪時表示：「除了上賽季謝幕戰那場51分的比賽之外，我4月份的時候場均30分以上，在我19年職業生涯里，這是我單月場均得分最高的一個月，所以我並不認為這取決於我是否還能打球……希望好事繼續發生。」

39歲的克勞福德上賽季和太陽隊的合約結束後，就一直處於失業的狀態中。他也不止一次在鏡頭面前表態希望重回賽場，但是並沒有球隊嘗試給他機會。同樣年邁的卡特和老鷹隊簽下了最後一份合同，開拓者隊也給了備受爭議的安東尼一個機會，在合作兩周後提前讓他轉正變成保障合同。

目前看來39歲的克勞福德想要重回聯盟，的確是一件非常困難的事情。上賽季他在太陽隊場均可以拿下7.9分1.3籃板3.6助攻，但是投籃命中率卻只有39.7%，以他的能力來看這是比較失常的。他在今年4月份打了4場球，場均可以拿到31.3分2.8籃板5.8助攻。

更令人意外的是，他在四月份的投籃命中率和三分命中率分別高達55.1%和50%。當然，四月份的這四場比賽樣本太小，不足以反映克勞福德最近的狀態；而且，克勞福德還有四個月左右就將年滿40歲，指望他做出像安東尼那樣的貢獻幾乎是異想天開。

幸運的42歲的卡特得到了老鷹隊的機會，可以完成自己職業生涯的最後一季。年邁的克勞福德在場上貢獻微乎其微的情況下，只能用更衣室領袖這一點作為賣點了；而且他還獲得過「最佳隊友」的頭銜，團結隊友顯然是沒有問題的，接下來就看有沒有球隊願意給他機會了。
