壓力來到詹皇這邊，48+11+14還絕殺，湖人死敵倆替補實在太無解

天天要聞
3 週 ago

北京時間11月28日，剛剛結束的一場NBA常規賽焦點大戰當中，最終洛杉磯快船延續著他們勝利的勢頭，比賽的最後時刻，快船兩大超級替補再次爆發。尤其是哈雷爾，他用一個半空中的補籃點到關鍵籃板球之後完成了一個絕殺，幫助快船拿下對手豪取7連勝，這場比賽倫納德輪休，快船能夠贏球，真的非常的不容易。

​

本場賽前快船就已經宣布了小卡將會因為輪休缺席比賽，而這一場比賽又是保羅喬治單核帶隊，但是快船卻有兩個不是領袖勝似領袖的球員，兩個超級替補路易斯威廉姆斯和哈雷爾。這場比賽喬治之前的手感其實並不好，尤其是上半場，保羅喬治8投4中一共只拿到了10分，但是下半場比賽他砍下12分不說，重要的是這12分全部來自於三分球。整個下半場，喬治7投4中、三分線外也是7投4中砍下了12分，而且關鍵時刻命中了多個高難度的三分球。

當然最重要的是快船不僅僅有保羅喬治，還有兩大替補球員。先是在末節球隊落後一度來到7分的時候，101-94回過頭來路威三分命中，哈雷爾飛扣，喬治直接飈中三分，接下來隨著格林的一次扣籃，快船已經追到了113平。後續兜出來還是保羅喬治，泡椒再次轟進三分，隨著貝弗利的一個三分命中之後，雙方再次打平了。當時時間還剩下1分20秒，兩支球隊後續都沒有得分，陷入了短暫的得分荒。

​

但是關鍵時刻快船挺身而出的不是別人，正是兩大替補殺星。當時還剩31秒的時候，喬治先是介紹一個掩護之後出手干拔三分不中，但是格林拿到前場籃板給到路威，路威殺到籃下拋頭不中，在人群之中哈雷爾高難度補籃，命中了一個最關鍵的進球！接下來快船又完成了成功的防守，關鍵回合防住了灰熊的最後一次進攻。

最後時刻，至少在比賽的最後10秒，快船場上決定走勢的還是兩個替補球員，關鍵球也是給他們兩個來打了。結果路威雖然不中，但是哈雷爾命中了准絕殺。全場比賽，不僅有喬治的22+5+2，重點是兩大替補合砍48分，路威狂轟24分13個助攻2個搶斷，哈雷爾24分10個籃板，兩個人48+11+14，這種驚人的效率也只有快船的替補能打出來了。

​

​

在他們兩個的帶領下，快船豪取7連勝，倫納德端坐釣魚台輪休還能贏球，這正是快船最希望看到的形態。現在壓力已經重新來到了詹皇和湖人這邊，他們必須得繼續贏球，保持對對手的優勢。
