詹皇接班人來了！大兒子砍15分奪MVP，老詹興奮狂跳被裁判攔住

2019-12-16 344條評論 3,903次閱讀 1382人點贊

北京時間12月15日，詹姆斯特意飛赴俄亥俄州觀看了大兒子布朗尼代表塞拉峽谷中學與詹皇母校聖文森特聖瑪麗中學之間的比賽。布朗尼在這一戰得到15分，並且在關鍵時刻搶斷上籃得分，幫助球隊59比56險勝對手，他賽後獲得此戰MVP榮譽。詹姆斯看球非常興奮，一個勁為兒子叫好吶喊，有一次控制不住自己，裁判不得不伸手將他攔住。

湖人正在征戰客場比賽，為了支持兒子，詹皇特意坐私人飛機趕到俄亥俄州。這場高中比賽吸引了1萬多人觀看，球票一售而空，全場爆滿，而且吸引了多達280家媒體採訪。詹姆斯兒子還在高一就人氣爆棚。

詹姆斯兒子布朗尼無疑是全場的焦點人物。父母的到來，讓布朗尼更有激情與能量，他兇猛地砍分。全場比賽還剩55.6秒，塞拉峽谷中學55比56落後1分。

聖文森特聖瑪麗中學發邊線球，出現了失誤，布朗尼搶斷成功，立馬運球沖向對手籃筐，面對對手的封堵，布朗尼非常輕巧地挑籃球進得分。

這一關鍵球讓在場邊觀戰的詹姆斯興奮到不斷地蹦跳，並且大聲吼叫。身邊的裁判看到老詹過於激動，伸出了手，將老詹攔在身後，大概意思是冷靜點，變影響他跑動空間吹罰比賽。

布朗尼最終得到15分，幫助球隊59比56獲勝，他用無可爭議的表現收穫此戰MVP獎項。

比賽結束後，詹姆斯特意和布朗尼以及塞拉峽谷中學的其他球員合影留念。詹姆斯此戰在場邊觀戰一直很吵鬧，他接受採訪時特意向母親道歉。因為過去他打AAU比賽，他的母親是觀眾席最吵鬧的人之一，還經常向裁判和對手發飆。「我以前會說『媽媽，你應該坐下。你在幹什麼？你這是讓我難堪啊。』所以，媽媽，我現在看著你，我媽媽實際上就在這裡看球。我曾經當面對她說，『抱歉，我現在親身體驗到作為家長看自己孩子比賽的感受了。』所以我道歉了。」詹姆斯表示。

看完兒子的重要比賽，詹姆斯將飛赴亞特蘭大，參加明天對陣老鷹的比賽。詹皇表示，兒子的表現激勵了他，迫不及待想要打明天的比賽了。
