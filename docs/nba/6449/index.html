火箭2連勝？威登趁勝追擊，克拉克或頂替豪斯，死亡五小再發威

2019-11-30 0條評論 20次閱讀 0人點贊

北京時間11月30號，NBA將迎來感恩節之後的第一個比賽日。休斯頓火箭繼續坐鎮主場，迎接亞特蘭大老鷹的挑戰。上場比賽，火箭憑藉威登組合的發揮，主場9分險勝熱火，終結了球隊一波3連敗。那麼，明天他們能否趁勝追擊，迎來2連勝呢？在上一場比賽中，德帥的「死亡五小」發揮了巨大作用，這場比賽是否會繼續使用呢？

威少和哈登在對陣熱火比賽中，都進行了自我調整。其中有兩點的變化是很關鍵的。

第一：威少掌控球權，減少遠投，增加近框攻擊。威少和哈登都是控球大師，10年職業生涯讓他們早已定型，因此盲目改變打法是愚蠢的。但是，兩者要兼容，其中一者就必須要打無球。因此，這場比賽中，哈登就減少了球權的控制，威少就變得異常高效。

第二、哈登增加了個人攻擊。哈登一直以來都是聯盟最具攻擊性得分手，但是他也曾是聯盟助攻王。以前，哈登在比賽中都會大包大攬，什麼都去做，這就導致他體力透支。隨著威少增加球權控制，哈登更加專註得分了，少了組織的重擔，他的體力也會得到緩解。

其實，這才是威登組合最適合的樣子。威少存在投籃缺陷，就要避免這個短板充分暴露。雖然火箭是以3分為生命線的球隊，但也要學會變通。哈登作為當家球星，要想帶領球隊繼續提升，自己的打法上也要做出調整。以這樣的改變和狀態，威登趁勝追擊，取得2連勝的可能性極大。何況，對手老鷹已經9連敗，士氣低落，鬥志喪失。

上一場比賽，德安東尼使用了克拉克，收到了很好的效果。特別是球隊擺出新死亡五小陣容：威少+哈登+豪斯+克拉克+塔克。戈登傷缺這段時間，在追分階段的五小陣容一般是：威少+哈登+里弗斯+豪斯+塔克。相對於里弗斯來說，克拉克有身高、有體重，3分球和防守能力也不落於下風。這個賽季，由於塞弗羅沙的存在，克拉克並沒有進入火箭的激活名單。

據火箭方面報道，球隊的首發小前鋒豪斯，因病無法出席明天的比賽。不出意外，克拉克或頂替豪斯擔任球隊首發前鋒。上一場缺陣的卡佩拉還沒有復出的消息，假如他繼續缺陣的話，火箭可以主打小個陣容。一方面是繼續磨練這套五小陣容，另一方面也是繼續讓克拉克融入體系。塞弗羅沙很有可能在2月份之前被裁掉，屆時克拉克會獲得大量出場時間。​

在前幾個賽季，火箭的五小陣容和無限換防，在對陣勇士的時候，就收到了奇效，特別是在阿里扎時期。火箭應該重點培養克拉克，讓他成為第二個阿里扎。這樣火箭死亡五小再發威，很有希望在這個賽季鬧出點不一樣的動靜！
