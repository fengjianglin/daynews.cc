湖人大戰森林狼！五位狀元大亂斗，兩個閃亮，兩個平庸，一個糟糕

2019-12-09 0條評論 17次閱讀 0人點贊

開心！舒服！高興！爽快！渾身的舒坦啊！作為一個湖人球迷，真的是幸福，太幸福了，北京時間12月9日，湖人142比125力克森林狼，看完了這場比賽，湖人球迷是高興了，但是有一個人的心啊，卻是拔涼拔涼的，

森林狼老闆格倫泰勒，看著湖人三位狀元閃耀全場，再看看自己選的這兩個狀元，哎，都是狀元郎，差距怎麼這麼大呢。湖人大戰森林狼！五位狀元大亂斗，兩個閃亮，兩個平庸，一個糟糕，

閃亮一：戴維斯，2012年狀元郎，50分7板6助攻，不說了，戴維斯有多麼的牛，看詹姆斯的眼睛就知道了，35歲的老漢，還能娶到這麼年輕貌美的一個嬌妻，老漢真的是看不夠啊，喜歡死你了。

閃亮二：詹姆斯，2003年狀元郎，32分4板13助攻，三分球8投6中，照這個打法，詹姆斯打到40歲綽綽有餘啊。戴維斯這麼快樂，一方面是自己業務能力好，但是必須也要說一句，還是詹姆斯身體好啊，老當益壯，老了老了，火力還是這麼旺，戴維斯太喜歡了。

平庸一：霍華德，2004年狀元郎，許多人誇讚霍華德，霍華德實力是一方面，但是更多方面是霍華德場上的矜矜業業，浪子回頭，雖然狀態不及當年之勇，但是瘦死駱駝比馬大，湖人需要霍華德。

平庸二：維金斯，2014年狀元郎，19分2板3助攻，真的，看完了整場比賽，愣沒發現維金斯在打球，沒辦法，還是那個球場上沒有存在感的維金斯，這就叫江山易改，本性難移啊。

糟糕透頂：唐斯，2015年狀元郎，19分4板8助攻，啥也不說了，看畫面，見你一次打你一次！濃眉哥低位單打唐斯，人比人，真能氣死個人啊，

第三節10分35秒，湖人半場進攻，濃眉哥低位單打唐斯，一個轉身輕鬆單手打進，第三節1分25秒，還是同樣的戲碼，濃眉哥在唐斯身上再次打中，戴維斯這50分，有三分之一都是靠唐斯得的，沒辦法，維金斯還是那個維金斯，唐斯還是那個唐斯。哈士奇還是那個哈士奇，森林狼老闆交易吧。
