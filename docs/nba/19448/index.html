得分最低的5大FMVP：倫納德17.8分上榜，魔術師16分卻沒爭議

2019-12-17 1,175條評論 3,850次閱讀 2406人點贊

漫長的NBA賽季，最受關注的比賽，一定是總決賽。經過82場常規賽的角逐，經過3輪季後賽的廝殺，東西部最具實力的球隊，晉級總決賽。在總決賽的舞台上，超級巨星往往能夠打出傳世表演。而FMVP，則是對總決賽里表現最好球員的嘉獎。一般來說，拿到FMVP的巨星數據都不會差，那你知道得分最低的5大FMVP是誰嗎？

倫納德

2014年的總決賽，馬刺對熱火完成了復仇。當年GDP組合處於巔峰末期，倫納德悄然崛起。在波波維奇的帶領下，馬刺將團隊籃球發揮到了極致，最終4-1輕鬆奪冠。倫納德是馬刺表現最好的球員，場均可以拿到17.8分6.4籃板2次助攻的數據。雖然數據不算華麗，但命中率61%非常高效。不得不說，倫納德非常的幸運。

麥克斯維爾

80年代的凱爾特人王朝，擁有許多實力強大的巨星，除了伯德、麥克海爾、帕里什之外，麥克斯維爾能力也很突出。他是1981年總決賽的FMVP，總決賽場均可以拿到17.7分9.5籃板2.8助攻的數據。不過有不少球迷為伯德叫屈，當年伯德總決賽雖然場均只有15.3分，但他還有15.3籃板7助攻2.3搶斷的數據進賬。

里德

里德時刻是NBA最經典的故事之一，而在1973年，巔峰末期的里德再次帶領球隊闖入總決賽。當時里德實力大不如前，但尼克斯整體陣容更加均衡。在最後的總決賽里，尼克斯4-1淘汰了擁有張伯倫和韋斯特的湖人隊。他場均可以拿到16.4分9.2籃板2.6助攻的數據，作為球隊老大，里德捧起了FMVP獎盃。

伊戈達拉

2015年的NBA總決賽，騎士非常悲壯，先後折掉歐文樂福，最終絕地七武士艱難打總決賽。令人意外的是，騎士還取得了2-1的領先。後來勇士將伊戈達拉提上首發，很快就終結了系列賽。伊戈達拉場均只有16.3分5.8籃板4助攻的數據進賬，不過他效率很高，作用也很突出，最終當選為FMVP。反倒是球隊老大庫里，場均26+5+6，仍然只能看著隊友捧杯。

魔術師

以上4位球員，總決賽拿到FMVP獎盃都是充滿懸念的。唯獨魔術師，場均只有16分，卻是沒懸念的FMVP。在1982年的總決賽上，年輕的魔術師帶領湖人隊再次奪冠。他場均16.2的得分，在湖人隊只能排第5，不過魔術師還送出了10.8籃板8助攻外加2.5次搶斷的數據，命中率也達到了53%，毫無疑問是湖人隊領袖，彼時23歲的魔術師就拿到了第2座FMVP獎盃。
