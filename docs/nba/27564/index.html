歐洲步宗師：“閃電俠”德懷恩·韋德

2019-12-22 3,563條評論 4,739次閱讀 3484人點贊

德懷恩·韋德（Dwyane Wade），1982年1月17日出生於美國伊利諾伊州芝加哥，前美國職業籃球運動員，司職得分後衛，綽號「閃電俠」（The Flash）。

德懷恩·韋德在2003年NBA選秀中於首輪第五順位被邁阿密熱火隊選中，從而進入NBA。2006年，韋德奪得生涯第一座NBA總冠軍，並且榮膺NBA歷史上第五年輕總決賽MVP（FMVP）。2008年，韋德代表美國男籃奪得北京奧運會金牌。2011-14年間，韋德幫助熱火隊四次進入NBA總決賽，並在中間在2012-13年2個賽季取得連冠。職業生涯在2005-16年連續12次入選NBA全明星陣容並在2010年榮膺NBA全明星MVP，8次入選NBA最佳陣容（2次一陣，3次二陣，3次三陣），3次入選NBA最佳防守陣容二陣，2009年榮膺NBA得分王，2006-07年連續2次奪得NBA全明星技巧挑戰賽冠軍，2018-19賽季第13次入選NBA全明星陣容。

進攻

中投不夠穩定，手感忽冷忽熱，進攻中比較依賴突破，一對一能力很強，上肢非常強壯，啟動很快，突破犀利無比，屬聯盟頂級，第一步跨出去後聯盟8成以上的選手已經在身後了（被稱為「迷蹤步」），不懼怕速度型後衛（沒他壯）和高大型後衛（沒他快）。對籃筐有很強的攻擊慾望，突破造犯規的能力強，上籃時候的感覺很好，得分大多數來自禁區和罰球線。快攻中的傳球很到位，經常和隊友有空中接力，陣地戰有時傳球過於隨意，導致場均失誤較多。由於出色的意識有時可以搶到進攻籃板直接發動二次進攻。

防守

他的防守不錯，很大一部分得益於他6-11的臂展，防守中能夠跟上對手的步伐，時刻干擾對手。場均一個以上的蓋帽很難得。身體對抗不吃虧，比較善於協防。具有一定的後場籃板保護能力。搶斷能力也不錯，入選了04－05賽季第二最佳防守陣容。值得一提的是關鍵時刻的防守給人印象深刻，經常不知道從哪裡飛出來就把對手帽了。

身體素質

身體素質有優勢也有劣勢，優勢是速度快，而且強壯，彈跳非常強，不過很好的彈速和意識一定程度上彌補了上述缺點。體力尚可，場均不到40分鐘。身體柔韌性好，動作比較協調，很有觀賞性。由於突破過多有時會受傷，膝蓋是個問題。

心理及領導能力

心理素質很好，越到關鍵時刻發揮越好，季後賽就比常規賽統計好看。善於讀出對手特點制定相應的進攻手段，有較高的籃球智商。心理調節能力不錯，很少連續兩場發揮失常。韋德做人比較低調，是天生的領袖。

德懷恩·韋德擅長歐洲步（迷蹤步），突破犀利，喜歡背身單打，年輕時是NBA聯盟中運球過人速度最快的球員之一。

2018年9月17日，韋德宣布將在2018-19賽季結束後退役。

2019年4月11日，韋德在完成職業生涯的最後一場比賽後正式宣布退役，結束自己16年的職業生涯。
