費城每年都是最有希望的一年，但每年都能被抓住命門？

2019-12-21 462條評論 2,454次閱讀 246人點贊

在獨行俠和費城的這一場比賽之前，兩隊分別對上了綠凱和熱火。而在那兩場比賽中，獨行俠和費城分別暴露了各自最大的弱點。

獨行俠在東契奇受傷之後，面對綠凱這種防守紀律性極強，個人防守能力出眾，到比賽最後可以無限換防的隊伍，他們的個人攻堅能力稍顯不足。

說白了就是到最後沒有一個能無視防守把球干進籃筐的人，所以布朗森兩個失誤之後，獨行俠那場基本就交代了。

而費城就更明顯了，只要用聯防困死恩比德，76人缺少空間的劣勢就會被無限放大。只要外線沒人突然人品爆發，或者開了無雙，費城一般只有歡聲笑語打出GG的份兒。

因此今天開局大家正常打，費城用他們的五大陣容一個抱一個，獨行俠擋拆恩比德則收籃下給中距離的策略之下，費城還能憑藉轉換和個人能力在場上佔據優勢。

但是卡萊爾什麼人？

聯盟知名的戰術大師，曾經在2011年用聯防限制死詹姆斯，幫助小牛奪冠的冠軍教頭。卡萊爾一看場上情況不對，第一節6分57秒喊出了第一個暫停。

暫停回來，獨行俠立馬變了聯防，這一下費城又懵了。打來打去不知道該怎麼破，最後恩比德一個三分打鐵結束了這個回合。

破聯防一般都是中鋒站罰球線策應，吸引大部分防守注意力，再配合外線的投籃和底線橫動。但說來說去，沒空間什麼都白扯！

獨行俠就收在禁區這一圈，釘死了恩比德，其他人放你在外線投，你費城又能怎樣？

費城本賽季場均出手29.5次三分，聯盟倒數第五；場均命中10.7個，聯盟倒數第七；三分命中率倒是有聯盟第十的36.2%，但關鍵是您投的少啊！

綜合產量和效率來看，費城的三分和空間不說聯盟墊底，但肯定也是中下游水平。

費城全隊只有三個人三分命中率上40%，分別是內托、塞布爾和西蒙斯。西蒙斯本賽季至今為止一共出手5次三分不提。內托每場只上大概10分鐘出頭，場均出手三分不過1.3次，即使塞布爾也不過場均2.2次。

而原本被予以厚望的哈里斯好像也開始遵循「錢得了怕甚事」的原則，本賽季三分命中率堪堪三成出頭。

理查德森倒是有場均5次出手，命中率36.5%的水準，但是近五場他21投6中，也是慘不忍睹。

您說就費城這種空間，一旦對手開始聯防，他們怎麼辦？而且聯防收成一團，內線空間極度擁擠，他們高度導致的籃板優勢又被進一步縮小。

獨行俠拿到後場板開始反擊，再加上小哈達威首節爆種，五記三分全部命中，分差自然就被拉開了。

最有意思的是，第二節末段費城靠著連中四記三分將分差縮小到了個位數。

但是這四記三分分別是：恩尼斯迎著波爾津吉斯的長臂在底角命中三分、恩比德弧頂兩個三分、還有獨行俠聯防，恩比德站罰球線進一步壓縮獨行俠防線，然後傳給科爾克馬茲的三分。

除了科爾克馬茲那記，前面三記三分都是即興的，這種三分很難有延續性。

除了防守，獨行俠在進攻端同樣打的很聰明。

當布朗教練撤下恩比德，讓霍福德打五的時候，費城內線的高度一下子就降了下來。並且霍福德的優勢就是可以防到外線，獨行俠乾脆將計就計，把霍福德拉出來然後開始衝擊籃筐、造殺傷；

霍福德發現不對勁了，和恩比德一樣守在籃下不出來之後，獨行俠又開始用中投和中近距離的拋投懲罰費城。今天不管是布朗森、賴特還是小庫里，都有拋投和中投的表現；

而當恩比德再上場，卡萊爾教練選擇讓克萊貝爾去和他對位。克萊貝爾第三節上來很快命中了三分，恩比德只能盡量往外防，結果恩比德只要一上大了，他立馬順步突破，今天兩個2+1全都造在了恩比德身上。

獨行俠總比費城多看一步，多走一步，再加上恩比德徹底成了聯防下的困獸，76人整個球隊就散掉了。

第四節替補銜接段，本該是哈里斯主攻的時候，但他顯得意興闌珊，好像失去了進攻的興趣和信心；第四節還剩4分鐘左右，西蒙斯防布朗森手都不伸了，同樣是潰敗的表現。

上個賽季，巴特勒被交易到費城之後，我說過一句——西蒙斯和字母哥誰先練出三分，誰就能徹底改變東部的格局。

聯想到昨天字母哥進的三分數和西蒙斯本賽季的出手數一樣多，我真他娘的為自己那句話感到臉紅……

西蒙斯可能職業生涯也就這樣了吧，他的野心和他內心對於自我的定位遠遠達不到那種高度。而西蒙斯決定這費城的上限，所以費城可能也就這樣了吧……

再多嘴一句，如果西蒙斯練不出三分，他和恩比德在未來必有一人會離開，而且很可能不是那種好聚好散的和平分手……

謝謝讀完！
