榜首大戰慘敗雄鹿，對湖人來說或許是件好事！

2019-12-21 1,229條評論 3,265次閱讀 223人點贊

東西部榜首大戰，本以為是場巔峰對決，沒想到湖人全場都被雄鹿按在地上摩擦，最多時落後21分，最湖人終104－111負於雄鹿，字母哥登場32分鐘轟下34分11籃板7助攻，濃眉哥砍下36分10籃板+5助攻，詹姆斯也貢獻21分+12籃板+11助攻的三雙數據。比賽雖然輸了，但對湖人也許是件好事，因為此戰暴露了湖人的很多問題。

首先是替補陣容的問題，本場替補陣容的得分是4－34，雖然這一場比賽只是個例，湖人的替補陣容太過孱弱也是事實，缺少一個穩定的得分手，現在是常規賽，打的大多數都是弱隊，替補看起來還湊合，及格線以上，但是等到了季後賽，尤其是面對潛在的爭冠對手快船和雄鹿，湖人的替補完全差了幾個檔次，看來也只能期待庫茲馬傷愈後有個好的表現了。

再一個詹姆斯這幾場比賽狀態一般，明顯感覺到詹姆斯的統治力不如以前了，防守也有些力不從心了，失誤也比較多，可能是連續的客場太累了吧，畢竟詹姆斯已經35歲了，場均還登場接近35分鐘，沃格爾該適當讓詹姆斯休息休息了。

還有湖人的鋒線問題，現在的NBA是鋒線聯盟，得鋒線者得天下，湖人除了詹姆斯和濃眉鋒線根本無人可用，詹姆斯大多時候打的還是控衛，達德利移動速度太慢，唯一能用的庫茲馬還有傷，而快船和雄鹿等隊都是聯盟頂級的鋒線，如果鋒線問題不解決，湖人前景堪憂啊。目前來說伊戈達拉是湖人最好的選擇，可是灰熊漫天要價，湖人根本沒有合適的籌碼，想要得到伊戈達拉只能看天命了。

最後說一點湖人本場比賽已經有了強隊的氣質，半場落後19分的湖人下半場完全沒晃，打的不急不躁，很有章法，替補雖然水平差，但打的一點不亂，很有心氣兒，這是一支強隊該有的氣質！
