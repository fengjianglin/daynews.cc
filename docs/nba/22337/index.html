76人五大主力被用殘！關鍵球只能賭運氣 6.5億壕首發不太靠譜

2019-12-19 413條評論 1,374次閱讀 113人點贊

儘管首節比賽被打出29比19的比分，但是客場作戰的熱火隊並不氣餒，第二節比賽轟出單節37比19的比分，一舉扭轉了場上的頹勢。熱火隊將領先的局面保持到比賽結束，76人隊最終也未能反超比分，只能接受失敗的結果。全場比賽結束，熱火隊以108比104擊敗76人隊。

坐鎮主場比賽的76人隊，他們對於比賽非常有信心，賽季開始以來，76人隊主場取得14連勝，是聯盟中唯一保持主場不敗的球隊。而比賽開始之後的順境似乎也在昭示著勝利的繼續產生，只不過比賽的走向讓76人隊感到不妙，他們在熱火隊強韌的表現下失去了活力，像是離開了水的魚兒一般。

吉米-巴特勒

吉米-巴特勒戰舊主，這隻在一定程度上成為比賽話題，因為巴特勒的離開對球隊影響並不大。76人隊在他離開的情況下，也從熱火隊順道薅了一把羊毛——是的，約什-理查德森本賽季在76人隊的表現還算不錯。然而這也並不是吉米-巴特勒跟76人隊的恩怨，而是兩支球隊之間的比賽。

熱火隊是一支陣容深度較強的隊伍，雖然球員名氣上跟76人隊沒法比，但是大多都比較實用。肯德里克-納恩和鄧肯-羅賓遜就讓76人隊見識到了能力比名氣更實用，兩個人的外線投射能力出色，這讓76人隊的大個陣容在防守端吃盡苦頭。納恩本場比賽18投9中砍下26分，關鍵時刻出現兩罰不中，卻仍然堪稱熱火隊的關鍵球員。

76人隊首發陣容

相對於熱火隊的陣容來說，76人隊的陣容可謂是豪華，準確地說，76人隊的首發陣容絕對足夠豪華。費城在休賽期的時候一口氣給出三份過億合同，續約托比亞斯-哈里斯以及本-西蒙斯之外，還簽下上賽季效力凱爾特人隊的艾爾-霍福德。這樣一來球隊首發陣容中，只有約什-理查德森的合同沒有上億，首發陣容球員身背合同總額高達6.5億美元。

這套史上最昂貴的首發陣容，無疑就是76人隊的爭冠法寶，然而如此操作不免會出現替補陣容孱弱的結果。在沒有薪資空間的情況下，76人隊還是拼湊了一套由龍套球員和新秀球員組成的替補陣容，這也是沒有辦法的辦法。76人隊如此陣容配置，給他們的新賽季帶來不小的考驗。

對陣熱火隊的比賽，問題就徹底暴露出來。

霍福德防守納恩

熱火隊雖然有一些輪轉球員缺陣，比如溫斯洛、德拉季奇等等，仍可以有7-8人進行輪轉，球員各具特點可以做出一些變化。而76人隊則做不到這一點，本場比賽看似有10名球員加入輪轉陣容，但由於在場上作用有限，並沒有替補球員能夠得到主教練的信任，這成為76人隊輸掉比賽的隱患。

76人隊球員體能下滑以及整體缺乏變化，使得熱火隊在防守中更有自信。決定比賽的關鍵一次進攻，納恩就頂住了西蒙斯的突破，西蒙斯傳球給霍福德，外線投籃不中。而熱火隊則能夠充分發揮個人特點，關鍵球交給吉米-巴特勒來打，這也成為熱火隊最終收下比賽的一點原因。

吉米-巴特勒上賽季在76人隊就主要處理關鍵球，他的離開在這方面的影響不小。
