大賺！爵士用玻璃人加二輪簽換騎士場均14分得分手 騎士超市開張

2019-12-24 3,318條評論 4,548次閱讀 3106人點贊

​據ESPN報道，克利夫蘭騎士與猶他爵士達成了交易，騎士將克拉克森交易到爵士，並從爵士得到了艾克薩姆和兩個次輪選秀權。

通過這筆交易，爵士為米切爾得到了一位可靠的替補球員。消息人士透露，騎士得到的兩枚次輪選秀權分別為馬刺2022年的二輪選秀權、勇士2023年的二輪選秀權。

據報道，猶他爵士隊將在後場增加一名得分後衛，與多諾萬·米切爾和邁克·康利並肩作戰。

這是克拉克森在三個賽季里第二次被交易，為了得到小托馬斯，他在2017-18賽季被湖人送到了騎士。

騎士和爵士近幾個賽季頻繁進行交易，去年11月，爵士用伯克斯和選秀權從騎士交易得到科沃爾。

本賽季，騎士的戰績為8勝21負，正在進行重建中。克拉克森正處於合同年，他需要證明自己的能力。

2014年，爵士以首輪第5順位選中艾克薩姆，但這位24歲的年輕始終沒能打出名堂，他的合同到2020-21賽季後結束。

艾克薩姆經常受傷，他本賽季目前只打了11場比賽，場均得到2.2分和0.6次助攻。

克拉克森有一定的持球能力，特別在康利受傷的情況下，爵士特別希望得到他。克拉克森為騎士場均能得到14.6分2.4次助攻和2.4個籃板，相當出色。
