沒有歐文籃網更好？在與不在判若兩隊，其實上賽季在綠軍早有先例

2019-12-01 0條評論 18次閱讀 0人點贊

籃網在主場戰勝了凱爾特人以10勝9負的戰績位列東部第七，一掃賽季初的頹勢，而起勢的這段時間也正巧是球隊核心歐文缺陣開始，歐文已經連續缺陣8場比賽，籃網在這段時間內取得了6勝2負的好成績。

在歐文傷停之前籃網僅僅以4勝7負的戰績排名東部末流，雖然歐文打出了生涯最佳數據，但是仍然很難帶領球隊取勝，歐文這賽季場均可以得到28.5分5.4籃板7.2助攻，均是生涯最高。

從數據上來看歐文地表現無可挑剔，他的加盟對於籃網來說絕對是一個很強的補充，在實力上也是明顯強於上賽季籃網的拉塞爾，這賽季籃網的賬面實力也是有增無減。

歐文的缺陣也可以很好地佐證到底是球隊的問題還是個人的問題，球隊在失去了歐文和勒維爾兩員大將的情況下，戰績不降反升，這確實非常令人困惑，但事實上早在上個賽季的凱爾特人就有過先例。

去年的綠軍成績一直不是很令人滿意，能戰勝強隊，也能輸弱隊，經常召開更衣室會議，而同樣作為隊內唯一核心的歐文上賽季反而缺陣的時候綠軍打出了更好地戰績。

在全明星賽前歐文總計缺陣了11場比賽，在這11場比賽中綠軍贏得了其中的9場，只輸了兩場，贏的隊伍裡面也不乏強隊，當時就有聲音出來表示球隊是否沒有歐文會更好。

歐文上賽季雖然率隊4比0在首輪橫掃了步行者，但是在東部半決賽遇到雄鹿時直接被打了個4比1淘汰出局，反倒是17-18賽季歐文因傷報銷了季後賽，綠軍在幾個年輕人和霍福德的帶領下打進了東決，並與騎士鏖戰七場。

休賽期他和杜蘭特的聯手簽約籃網震驚全聯盟，雖然杜蘭特一個賽季不能打，但是所有人都相信上賽季籃網在拉塞爾的帶領下都能以東部第六的身份進入季後賽，那麼歐文肯定不會比拉塞爾差。

的確從籃球層面來看，歐文在場上能得分也能傳球，雖然防守不強，但之前的拉塞爾也不以防守見長，在串聯全隊方面其實做的也不錯，隊友也都能有所發揮，那麼可能就要拋開籃球因素來討論。

自從歐文在騎士申請離隊後就總會討論到歐文的性格問題，歐文的性格的確不是那麼的大眾化，在離開綠軍後凱爾特人球員的評價中我們可能可以找到一些端倪。

上賽季綠軍的前鋒莫里斯在加盟尼克斯後曾評價凱爾特人上賽季的情況：「在綠軍，所有人都要優先考慮歐文的情緒，全隊上下都得迎合他，這也導致了一些問題不能真正地提出來並解決。」

的確上賽季綠軍曝出了多次隊內矛盾，在球隊戰績糟糕的時候球員們也自發組織召開了隊內會議，但結果是問題仍然存在，整個賽季下來隊內問題都沒有完全解決。

不過我們相信歐文即便是有問題，但也不是有意地想傷害球隊，因為經常看歐文比賽的朋友們可以發現，歐文在場上時無論對於對手和隊友都是非常友好，會主動去拉起隊友，也會拉起對手，也很積極地和隊友擊掌。

隊友們在媒體採訪時也都積極維護歐文，這和上賽季還在綠軍時如出一轍，在去年當老大失敗後歐文也可能意識到了當老大是多麼地不容易，這也是為什麼他願意和杜蘭特一起加盟，即便只能做二當家。

歐文在遭到綠軍主場球迷噓聲在社交媒體上發得一段話引引起了不小的反響，其實我們可以從中發現歐文是一個很有自己想法的球員，包括他曾經聽上去很可笑的「地平說」言論。

歐文他在自己的世界中有一套自己的思想，也可以說是因為歐文少年得志，從小就天賦異稟，在很年輕的時候就擁有了很多人一輩子都得不到的東西。

就像羅齊爾曾說的：「歐文就像那種你從小就很羨慕的哥哥，你想擁有他得到的一切。」太早的成名加上他母親在他很小的時候去世，也許這就造成了他某些觀念上的缺失和想法上的不同。

但無論怎樣都否認不了歐文是一個傑出的球員，他也絕對會是這賽季的全明星球員，同時歐文的球風也會為他吸引大批粉絲，這賽季還早，他還有足夠的時間和球隊磨合。

這賽季由於杜蘭特的缺陣，球隊並不會有太遠的計劃，但是進季後賽是起碼要做到的，如果有歐文在連季後賽都進不了，那麼即便杜蘭特復出這支籃網也很難走太遠。

28歲正是歐文職業生涯最黃金的年齡，個人狀態已經達到巔峰，下面需要做好的就是和球隊儘快找到化學反應，找到共同贏球的方式，這需要球員和教練共同努力，我們還是很期待德魯大叔在球場大展神威，為我們多多貢獻精彩的五佳球。
