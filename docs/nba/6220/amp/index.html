NBA最新排名，湖人雄鹿並列聯盟第一，聯盟前5有四支東部球隊

天天要聞
2019-12-04

北京時間12月4日，NBA又結束了一個比賽日，下面讓我們看一下各隊的戰況和排名。

鵜鶘在主場迎戰獨行俠，獨行俠進來狀態非常好，東契奇的個人表現也非常出色。第一節，波爾津吉斯一個人送出4次大火鍋，鵜鶘依靠他們出色的外線投射與獨行俠周旋，不過本場比賽東契奇手感火熱，他全場砍下33分，在外線扔進5個三分。在他的帶領下，獨行俠在第3節逐漸建立起比分上的優勢，並牢牢控制著場上的局勢。全場比賽結束，獨行俠在客場以118:97大勝鵜鶘，延續了自己的良好勢頭。獨行俠現在14勝6負排名西部第4，鵜鶘6勝15負排名西部第14.

熱火在客場通過加時以121:110戰勝了猛龍，猛龍隊的主場不敗金身就此告破。洛瑞本場比賽復出，但是他全場18投僅2中，三分線外11次出手無一命中，全場只得到12分11個助攻。熱火隊在常規時間最後一攻錯失絕殺，但在加時賽里熱火統治了比賽，巴特勒一人主導起熱火的進攻。猛龍隊在加時賽中僅得兩分，最終在主場敗下陣來。熱火目前15勝5負排名東部第2，猛龍15勝5負排名東部第3（兩隊勝負關係不同）。

湖人開始了他們的客場之旅，他們在客場以105:96戰勝了掘金。本場比賽開打前，兩支球隊均為西部前列，這場比賽勢必激烈。上半場湖人佔據著主動，半場結束時他們領先掘金隊11分，但下半場，掘金很快發起了反撲。在第4節後半段他們一度將比分縮小到只有3分，但掘金隨後的進攻出現停滯，他們連續投籃不進，湖人則在詹姆斯的帶領下穩定住了局勢，並將領先優勢保持到了終場。

洛杉磯另一支球隊快船在主場迎來開拓者的挑戰，兩隊上半場比分交替上升，場面非常激烈。半場打完時，快船隊在主場領先開拓者3分，但在下半場快船隊加強自己的防守，進攻端他們在外線屢屢開炮，一舉確立領先優勢。而在第4節，哈雷爾發揮出色，分差一度拉開到20分。快船憑藉著下半場的發揮，在主場戰勝開拓者。目前湖人18勝3負排名西部第1，快船16勝6負排名西部第2。

在一場東部對決里，奇才在主場120:127不敵魔術隊，比爾個人獨木難支，他全場砍下42分。魔術的整體發揮更為穩定，魔術隊全隊6人得分上雙，三人得分上20，狀元富爾茨近來狀態極佳，他本場拿下20分6助攻，也隨球隊一起在客場取勝。

活塞在客場挑戰騎士，活塞的整體實力比騎士出色，上半場結束時，活塞就已經領先了騎士16分。第3節活塞的攻勢不減，他們連連進攻得手，騎士面對活塞的進攻沒有絲毫辦法。比賽在第4節也進入了垃圾時間，活塞全隊6人得分上雙，最終在客場以127:94大勝騎士。本場比賽結束後，活塞8勝13負排名東部第9，騎士5勝15負排名東部第13.

今日比賽結束後，熱火隊已經力壓猛龍衝到了東部第二，這支球隊的黑馬特質已經相當耀眼，防守起家的熱火隊，面對諸多強手都有一戰之力，這也是他們崛起的關鍵，吉米-巴特勒更是彌補了球隊關鍵時刻的漏洞，這讓熱火隊在本賽季一飛衝天。

而湖人隊在力克掘金之後，繼續穩定傲立西部之巔的位置，西部前八目前都沒有球隊更替，不過從第九的國王往後，雷霆開拓者以及馬刺這三支老牌強隊都有些迎頭趕上的意思，他們距離季後賽行列都在3場勝場差之內，季後賽希望重燃。

東部方面，乍一看，前4名的戰績都相當出色，實際上全聯盟的前五戰績里，有4支都是來自東部的球隊，只有湖人隊和雄鹿隊是並列聯盟第一，之後的熱火猛龍以及凱爾特人，勝率都是要比西部第二的快船隊要高的。
