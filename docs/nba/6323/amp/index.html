沒人能看得懂勇士的暗箱操作，水花不倒，勇士還是奪冠隊伍

天天要聞
2019-12-03

金州勇士本賽季在遷移到新球館之後，隊內球員傷病不斷，當家球星庫里，本賽季21場比賽里，就出場了四場。是新球館的問題嗎？還是真存在風水問題呢？

當然，新球館的運營，肯定是開心的，傷病是沒有辦法避免的。格林就說過「傷病是體育的一部分，我們沒有辦法去避免他」，我很認同，在體育運動裡面，有多少做的很好的職業運動員，因為傷病。和冠軍擦肩而過，無論是籃球，還是足球，田徑等。都會因為傷病導致職業生涯變短，上一賽季五星勇士，就因此沒拿到冠軍。

目前勇士，還有四人受傷，庫里、克萊湯普森、拉塞爾以及魯尼。據小道消息傳播，魯尼將在最近幾場比賽裡面回歸賽場。這對於勇士來說，是一件好事，魯尼的內線防守以及吃餅，是勇士不可缺少的一部分。水花兄弟，外線進攻，魯尼護籃板。就像14年的勇士一樣，不過那個時候的內線是博古特，博古特具有很高的身高，可以在別人包夾克萊和庫里的時候，在內線輕鬆得分。

克萊和庫里的傷病都預期在2月份到全明星最後復出，拉塞爾最晚在12月底復出。也就是說賽季即將結束了，勇士隊才會齊員狀態。勇士隊是不是在用瞞天過海的方法，在核心主力水花兄弟養傷的同時，在訓練替補席的球員。待主力回歸之後，繼續回到奪冠的整容。

目前勇士隊戰績以4-17的戰績排行聯盟倒數第一，上賽季的總決賽隊伍，一蹶不振。在主力不回歸的情況下，勇士會不會搏一搏狀元簽呢？在用狀元簽，去換一位成熟的鋒線，來頂替隊伍裡面的鋒線呢？小編認為勇士可能會去這樣操作運營。

勇士隊進季後賽的希望是越來越小，隊里沒有穩定的得分手。勇士隊在水花健康的時候，打快速反擊的能力非常強。退防守慢一點的隊伍，基本上是防不住勇士隊的進攻。健康的水花，三分根本不講理。在全明星到來之後，勇士的克萊和勇士，又在賽場上攜手同行。

在傷病回來之後，科爾應該不會讓克萊和庫里出場時間很久。讓克萊和庫里找找手感，待到下賽季初，替補席的戰鬥力練上來了。核心水花繼續帶隊，加上拉塞爾，格林魯尼。想必目前聯盟裡面最強的湖人和雄鹿都夠吃一壺，很多人不看好克萊的防守，看過比賽的人，都知道克萊運球是差一點。作為二當家的克萊，不用球權，拿到手就能出。目前聯盟這樣的3D球員非常少，甚至拿不出比克萊更優秀的。

勇士暗中養傷，實則在等待下賽季的高順位簽。拿順位簽，去換取更合適勇士的鋒線球員，搭配拉塞爾帶隊的替補席，勇士又回到冠軍爭奪的隊伍之中，這招確實高呀！！！
