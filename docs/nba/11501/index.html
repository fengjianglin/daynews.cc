掘金野心到底有多大？外線大將被擺上貨架，潛力3D或成冠軍祭品

2019-12-11 344條評論 1,813次閱讀 98人點贊

最近NBA官方給出的最新一期球隊實力排行榜上，丹佛掘金排在第10位，相比上一期的實力排名，掘金下滑了4位。近段時間，掘金的戰績出現了不小的滑坡，他們在最近5場比賽中輸掉了4場，這與他們之前的戰績形成鮮明對比。要知道，在5場比賽之前，掘金一度在10場比賽中贏下9場，這也是他們在上一期實力排名能夠排在第6位的最大原因。

縱觀掘金本賽季的表現，他們的勢頭還是很不錯的，最近輸掉的4場比賽中，也有像湖人和凱爾特人這樣的強隊，輸給他們也不能說掘金的實力差，而是這兩支球隊的實力確實更強一些。不過，掘金管理層對目前的戰績排名並不是很滿意，所以他們誕生了通過交易補強球隊的想法。

外線潛力3D被掘金擺上交易貨架

在掘金戰績開始出現下滑之後，管理層開始考慮通過改變陣容構架一掃陰霾，於是馬利克-比斯利成為了被擺上交易貨架的那個犧牲品。

根據NBA消息源，掘金已經正式開始聽取其他球隊對比斯利的報價，而且已經有多支球隊表現出了對比斯利的強烈興趣，如果掘金對其他幾支球隊的報價感到滿意，那麼比斯利極有可能在第一時間就被送走。

本賽季，比斯利在掘金的出場時間受到壓縮，賽季至今只有場均15.9分鐘的出場時間，場均數據也出現了下滑，目前是6.4分1.5籃板，但三分球命中率高達41.2%，這是讓其他球隊看中的一點。要知道，比斯利在上賽季是掘金的鋒線主要輪換之一，他能夠場均得到11.3分2.5籃板的數據，三分球命中率同樣達到40%。

顯然，比斯利是一名出色的鋒線輪換球員，而且外線三分球能力非常穩定，再加上比斯利出色的身體條件帶來的很強的個人防守能力，這在驅使比斯利朝著3D球員的方向前進。而3D鋒線是現在每一支爭冠球隊都想要爭奪的資源，就像湖人、快船、雄鹿等強隊在爭奪伊戈達拉一樣，比斯利也成為了搶手貨。

掘金交易比斯利凸顯爭冠野心

掘金管理層在球隊成績下滑的第一時間就做出了交易隊內球員的決定，這很明顯是在向球員們傳達一種理念，那就是本賽季目標不僅僅是季後賽。我們知道，上賽季掘金在競爭尤為激烈的西部取得了第二名的好成績，但季後賽意外在半決賽折戟。

或許就是因為上賽季季後賽的實力，這讓掘金管理層本賽季非常謹慎，他們的目標已經不僅僅是季後賽，而是希望通過不斷的補強球隊在季後賽中打出更好的成績，殺進西決或者衝出西部，儘管難度很大，但掘金管理層至少已經開始為之努力了，掘金已經表現出了強烈的爭冠慾望。

那麼，掘金想通過比斯利獲得哪些籌碼呢？掘金目前後衛線已經有足夠的人手，但在小前鋒的位置上稍有劣勢，他們可能會通過比斯利換取一位有單打能力的小前鋒球員，這對他們的幫助會很大，這樣一來掘金會在關鍵時刻有兩位持球手，一位新小前鋒，另一位就是穆雷，這對於球隊在關鍵時刻決勝有很大幫助。

而比斯利會得到哪些球隊的青睞呢？根據比斯利3D球員的特質，聯盟中鋒線防守能力弱且想補強鋒線攻擊力的球隊會第一時間考慮比斯利。

結語

掘金在西部突圍的難度很大，但管理層在為之努力，或許奇蹟就是留給有準備的掘金的。
