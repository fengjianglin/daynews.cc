弱隊就是弱隊！開局老詹4個失誤被打9-0，但暫停後打出27-4攻擊波

2019-11-30 0條評論 17次閱讀 0人點贊

湖人與奇才的比賽，從雙方的實力對比上，不在一個層次。賽前，湖人16勝2負，排在聯盟第一，加上最近的9連勝，氣勢如虹，而奇才呢，排在東部第十而已。不過，比賽一開場還是製造了巨大的漣漪，湖人連續失誤，其中詹皇前5分鐘就失誤了4個，而奇才趁機打出了9-0的攻擊波，之後雖然湖人有回應，但仍舊以15-6落後，其中詹皇還被新秀八村塁連打3球。

不過，在15-6落後以後，湖人喊了暫停。也就是在這次暫停後，湖人在攻防兩端都打出了統治級的表現，從7分36秒到1分11秒這段時間裡，湖人打出了27-4的攻擊波，迅速以33-19取得了領先。之後雖然奇才能回應兩球，但湖人也還有進攻，單是第一節湖人就以37-23取得了巨大領先。這開局的9-0，到中間的27-4，不得不說，弱隊就是弱隊！

第一節，湖人能挽回落後並取得這麼大領先，主要是及時調整了陣容，並保持了防守強度。投射方面，第一節，湖人24投14中，投籃命中率高達58%，其中三分9投5中。球員表現方面，濃眉哥6投4中，得到15分4籃板；隆多4投3中，拿到8分1籃板2助攻，是表現最好的兩個球員。首節，詹皇5分3助攻，但有4個失誤，表現只能說差強人意。
