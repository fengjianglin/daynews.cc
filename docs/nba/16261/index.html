瘋了！大鬍子兩場砍下109分，命中20記三分球

2019-12-14 3,339條評論 4,995次閱讀 3860人點贊

陽光明媚的周末，所有人都在關注湖人vs熱火這場焦點戰的同時，殊不知，大鬍子哈登在我們看不到的地方又火了！

如果說湖人與熱火的比賽帶給了我們足夠的心跳和焦灼，那麼隔壁火箭帶給我們的則是酣暢淋漓的感覺。大鬍子兵不血刃，帶領火箭客場130-107屠殺魔術，相比球隊的大勝，更讓我們感到暢快的是哈登開啟殺神模式，全場爆砍54+5+7，同時命中10記三分球。而如此驚人的表現，哈登在三天內完成了兩次。

時間回到12月10日，火箭主場一分惜敗國王，而在那場比賽中，哈登「僅僅」得到27分10助攻的失常表現。球隊失利外加表現平平，這無疑讓大鬍子感到懊惱，也勢必會迎來強勢反彈。如此一來，已經八連敗的騎士不幸成為火箭的刀下魂。

在火箭客場挑戰騎士的比賽中，大鬍子猶如天神下凡，速貸球館彷彿變成自己的主場，任由哈登在場上大肆屠殺。全場比賽，哈登34投20中，三分18投10中，砍下55分3籃板8助攻。雖然在第四節一開始火箭還以88-99落後，但哈登神奇的表現帶動球隊發動反擊，一波19-6過後，火箭徹底控制了比賽局面，這樣的結果也沒有讓哈登的得分成為空砍。

魔術近期雖已兩連敗，可兩場分別負於的對手的東西部老大雄鹿和湖人，而且在這兩場比賽中，魔術展現出極強的韌性，均險些在比賽最後完成逆轉。然而這樣的魔術面對火箭卻顯得毫無招架之力。僅僅在開局魔術取得了10-4的領先，之後火箭開啟得分模式，魔術在沒有翻過身來。

大鬍子哈登延續了上一場的強勢表現，本場對陣魔術，哈登再次爆發砍下54分5籃板7助攻，三分球更是達到了15投10中。這樣一來，哈登兩場比賽共得到109分，命中20記三分球。此役過後，哈登生涯三分命中數達到2144次，超越保羅皮爾斯位列歷史第八位。

也許迄今為止，人們對於哈登的印象依然停留在碰瓷、走步上，認為哈登的高得分都是通過罰球換來的。但是這兩場比賽哈登的表現徹底顛覆了人們對於哈登錯誤的認識。兩場比賽下來，哈登共出手65次，命中39次，三分球出手33次命中20次。而兩場比賽加起來罰球僅有11次。三項命中率分別達到了60%，60.6%，100%，這樣的得分以及這樣的命中率，對於一個得分後衛來說，的確是前無古人、後無來者。

如今，哈登依然以39.3的場均得分牢牢控制著得分榜首位，同時直接甩開榜單第二位的字母哥8.2分。這對於哈登競爭本賽季MVP提供了有力的支持。

明日對陣活塞，哈登又會有怎樣神奇的表現？我們拭目以待！
