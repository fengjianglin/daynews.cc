本賽季3支球隊拿下10連勝或以上的戰績，他們連勝的奧秘是什麼？

2019-12-09 0條評論 13次閱讀 0人點贊

一支能夠在聯盟中持續連勝，達到10場或以上的球隊，顯示了其良好的狀態和穩定而又合理的陣容結構。畢竟贏球易，一直贏可就沒那麼容易了。本賽季聯盟中3支球隊達成至少10連勝的壯舉，我們來看看他們都為什麼能橫掃聯盟。

凱爾特人在10月26日到11月16日之間取得了10連勝的戰績，湖人在11月13日到11月30日之間取得了10連勝的戰績，雄鹿在11月11日至今依舊延續著自己14連勝的戰績。目前凱爾特人位列東部第2，湖人位列西部第1，雄鹿位列東部第1。

首先3支球隊的共同點是3支球隊都是以防守立本的球隊，這是一支強隊的基石。雄鹿隊的防守效率目前高居聯盟第1位，凱爾特人排在聯盟第3位，湖人排在聯盟第5位。

在NBA這個天才球員雲集的地方，想憑藉著進攻打死對方，一場或許還比較容易，你想場場都如此那你就太天真了。論得分爆發力，誰准起來還不是個爸爸，但有防守就會給對方出手足夠的壓力，對方艱難出手，只能強行投中巨星球。

但是這3支球隊的防守策略又不盡相同，湖人隊無疑是這3支球隊中理論上防守配置最好的球隊。隊內入選過最佳防守陣容的球員多達6名。

外線有丹尼.格林、布拉德利、隆多，所以湖人從外線就可以給予對方持球人足夠的壓迫。

鋒線有詹姆斯和戴維斯，詹姆斯和戴維斯的鋒線遊走可以讓球隊的防守更加的靈活多變，特別是戴維斯在本賽季充分發揮了自己高機動性的特性，迅速的協防讓對手苦不堪言。

戴維斯場均1.52次的搶斷排在聯盟第12，場均2.65次的蓋帽排在聯盟第2，這都展現了戴維斯在防守端的統治力。

內線替補席上有霍華德坐鎮，兩屆蓋帽王是聯盟中其他球隊的外線突破到內線的噩夢，本賽季在場均20分鐘的出場時間裡，霍華德還是能貢獻場均排在聯盟第13的1.38次的蓋帽，足見湖人從內到外對於其他球隊的震懾。

可能優秀是會被傳染的，湖人本賽季簽下的禿曼巴，卡魯索不僅在幾場比賽中有著飛身補扣的高光時刻，更為重要的是他的防守效率高居聯盟第9位。他才是湖人隊中真正的防守效率第一人。

雖說湖人隊是這3支球隊理論上防守效率最好的球隊，但實際上雄鹿才是這3支球隊中防守更為整體，單兵能力最強的球隊。隊內5人防守效率前10，讓雄鹿從主力到替補都能給對方持續施壓。

與快船的強強對話，雄鹿將快船的雙子星卡哇伊和喬治防的全場17+5+4和13+7的輸出就足見雄鹿的防守功力。本場比賽，雄鹿死守籃下，請君入甕，當快船的球員突破到籃下時快速協防，又始終安排人堵在快船球員的傳球路線上，逼快船的球員勉強出手。

雄鹿隊對於後場籃板的保護又是自有其方法，場均聯盟第1，51.83個籃板讓快船極難搶下前場籃板對其實行二次打擊。

而3支球隊中看上去單兵防守能力最弱，隊內平均身高最矮的凱爾特人在防守上也有著自己的獨到之處。凱爾特人聯盟第3的防守效率有賴於他們強大的協防網和凱爾特人骨子裡流淌的鐵血、強悍、拼搏的血液。

當對方球員突破到凱爾特人的籃下，凱爾特人就會迅速進行包夾，逼對方出球。凱爾特人的迅速協防無處不在，正是依靠著以多防少，整體聯動的防守策略，凱爾特人沒有讓聯盟中的其他球隊在自己身上討到任何的好果子吃。

而凱爾特人防守上更為強悍的是其防守的侵略性和活力，凱爾特人本賽季隊內的球員多次上演飛身救球的好戲，有句話說的好不怕橫的就怕不要命的，而凱爾特人這種既橫又不要命的球風的確讓聯盟中的其他球隊叫苦不迭。

以上說的是3支球隊的防守，但一支球隊僅僅只有防守能夠連勝，那也贏得不夠痛快而這3支球隊進攻的火力同樣不俗。

雄鹿在進攻效率榜上排在聯盟第2位，湖人聯盟第4位，凱爾特人第9位。

湖人隊是聯盟中唯一一支隊內有2個人的得分排在聯盟前10的球隊，戴維斯場均27.7+9.0+3.3，排在得分榜第6位，詹姆斯場均25.9+6.8+10.8，排在得分榜第8位。湖人就是以打詹姆斯和戴維斯的陣地戰為主，畢竟詹姆斯的年齡和戴維斯的身高擺在那，如果打的太快，或許對方沒有被打趴下，自己的主將就先累趴下了。

但是有著詹姆斯和戴維斯這樣的強力鋒線，，生突硬鑿對方也受不了，戴維斯和詹姆斯的內線得分分別為14.3分和12.9分排在聯盟的第4位和第5位。

隊內有著這樣兩個超級巨星，把球交給最能打的人就夠了。

雄鹿以快節奏的攻防，字母哥一星帶四射的配置立威聯盟。不同於湖人反潮流的以內線進攻為主，雄鹿隊的打法是更符合聯盟的趨勢，而且是這一趨勢的集大成者。

雄鹿的回合數排在聯盟第1位，場均3分命中排在聯盟第5位。字母哥的高能反擊加上內線突破強打時的分球令聯盟中的其他的球隊防不勝防。

凱爾特人的防守鐵血，強悍，整體的協防聯動性高，他們的進攻同樣非常的整體，憑藉著多點開花的效果，凱爾特人在聯盟中爭得一席之地。隊內4個的得分在20分左右，在每場比賽中站出來的人都不太一樣，凱爾特人將團隊籃球發揮到了極致。

3支球隊的進攻和防守是其取得10場以上連勝的內因，他們所處外界輿論和自我期待是他們能連勝的外因。

湖人隊身處繁華的洛杉磯，多年未能打入季後賽，即使上個賽季詹姆斯的到來，依舊沒能打破這一窘境。相信洛杉磯的媒體和球迷已經厭倦了這樣的日子，畢竟湖人有著屬於自己的燦爛歷史和球隊文化，本賽季戴維斯的到來是湖人崛起的最佳時機。

洛杉磯已經等待了太久，詹姆斯想要證明自己還未老去，戴維斯期許自己的職業生涯能重新起航，而且他們也要向世界宣告誰才是洛杉磯真正的主人。

雄鹿隊在上個賽季就應該打入總決賽，本來一切都應該按照既定的劇情發展，沒想到猛龍粉碎了他們所有的想念。字母哥不甘，雄鹿不甘，本賽季是字母哥，是雄鹿加冕最好的機會，來年，雄鹿的續約壓力將激增。

當聚光燈漸漸遠離凱爾特人時，凱爾特人卻展現出了空前絕後的團結與血性，所有的關注和榮譽都該是自己一場一場拼出來的，凱爾特人期許著明天。

沒有人敢說本賽季的總冠軍一定是自己的，但強勢連勝的球隊用自己的表現向世人訴說著自己眼中的星辰與大海。
