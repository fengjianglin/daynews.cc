18連勝的雄鹿和場均39分的球員，熱度卻遠不如湖人和077

天天要聞
2019-12-16

如果只看數據，本賽季NBA最熱門的球隊一定是雄鹿，他們已經取得18連勝了，並且以24勝3負高居聯盟第一。但是，對於雄鹿的連勝和他們的新聞，並沒有太多的報道，大多數人更願意去關注湖人，他們23勝3負排名西部第一，聯盟第二，目前還沒有超過10連勝，他們有三旬老漢，有安東尼戴維斯，有波普定律……

在NBA收視率大幅度下跌的時候，湖人的收視率卻大漲，這就是現實。而雄鹿關注度不如湖人，其中一個原因是小城市，密爾沃基的關注度肯定不如洛杉磯，這也是當年易建聯不太願意去雄鹿的原因，而另外一個原因就是字母哥夏天的言論，在關於莫雷事件的言論上，字母哥是偏向莫雷的，這也讓新科MVP，掉了一部分的中國粉。

要說字母哥球迷不多，那是不可能的，上賽季字母哥憑藉出色表現和生涯謙遜的性格，圈了很多粉絲，但是今年夏天也讓部分粉絲轉為路人，其實本賽季的雄鹿比上賽季還要瘋狂，字母哥表現也絲毫不遜於上賽季，但熱度已經沒了。

當年科比場均35.4分，那真的是轟動性的新聞，科比連續4場50+也是驚為天人。而如今本賽季有個人場均39.3分，多次拿到50+，但是關注度是零零散散，平平淡淡。這個原因就多了，一個是打法問題，被動要罰球跟主動要罰球有本質的區別，偶爾騙罰球跟一直騙罰球也不是一個性質，哈登的打法在很多人看來缺乏美感，所以關注度不夠。

而最主要原因就是，今年夏天，莫雷發起了不尊重我們主權統一的言論，隨後刪除發文然後躲在暗地裡不敢出來發聲。央視至今仍封殺了NBA，而哪怕是不怎麼要臉的TX，也不敢直播某球隊的比賽和新聞，所以也沒什麼關注度，甚至很多人都不知道這支球隊的戰績如何、球員表現如何、有哪些球員、有沒有受傷缺席等情況，一概不知。

取而代之，西部最熱門的球員除了詹姆斯，可能就是東契奇了，這位二年級球員打出了全面的表現，率領球隊成為西部前四，而且又愛笑，看起來比較隨和，圈了不少粉絲。

其實牛人有很多，真沒有哪個明星是不可或缺的。哪怕是喬丹、科比、詹姆斯這樣的超級巨星，也都很珍惜粉絲的喜愛，他們尊重粉絲，願意花心思花精力去維護和粉絲的關係，這樣的球員才配得上粉絲的喜愛，如果把粉絲的喜愛當成一種資本，去破壞人與人之間的互相尊重，那掉粉是正常的結局了。
