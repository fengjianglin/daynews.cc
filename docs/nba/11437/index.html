本賽季被球迷認為是“弱旅”的4大球隊，真正實力遠超大家想像

2019-12-11 1,010條評論 2,708次閱讀 1221人點贊

在NBA中，大家習慣性把一些重建球隊，實力較弱的球隊，或者是沒有季後賽希望的球隊稱為弱旅，但事實上，有些球隊只不過是在大家印象中很弱，其實球隊本身擁有著季後賽級別的實力，而本賽季中，就有4支這樣的球隊，他們只是大家認為很弱，其實真正實力要遠超大家的想像。

雷霆隊

如果把本賽季的雷霆隊當成弱旅，那只有兩種可能性，一是根本就不了解本賽季雷霆的陣容，另一個就是被開局階段雷霆的表現所欺騙，看看雷霆隊現在的陣容，首發保羅+亞歷山大+加里納利+弗格森+亞當斯，替補席上還有施羅德、貝茲利、諾埃爾、納迪爾、迪亞洛等球員。

和上賽季相比，等於說喬治+威少+格蘭特換成了現在的保羅+加里納利+亞歷山大，數據方面雖然保羅不如威少，但是在效率和串聯球隊進攻方面，保羅做的要比威少更加出色，而且相比威少，保羅更適合作為一支球隊的領袖，加里納利雖然攻防兩端的實力都要遜色於喬治，但是只要能夠保持健康，還正處於當打之年的加里納利，已然可以場均貢獻全明星級別的數據，而且在三分投射方面，加里納利並不遜色於喬治，而亞歷山大恐怕是休賽期雷霆隊最大的收穫，賽季至今場均可以貢獻18.8分5.2籃板3助攻1搶斷的數據，表現要比之前的格蘭特要出色不少，至於亞當斯，經過賽季初期的沉淪，最近的表現也有所提升，場均貢獻也接近兩雙，再加上幾名能夠穩定提供貢獻的替補，相較於上賽季更加合理的陣容，這樣的雷霆能夠被成為弱旅嗎？

太陽隊

雖然本賽季中太陽隊表現出色，戰績也排名西部前八，不過在大多數球迷眼中，太陽隊依然是一支弱旅，而本賽季和聯盟季後賽球隊交手的戰績中，也只在開賽階段擊敗了缺少喬治的快船，和處於磨合階段的76人，輸掉了所有大家認為太陽應該輸掉的比賽，那太陽真的是一支弱旅嗎？

其實並不是這樣，隨著盧比奧的加盟，得到解放的布克效率方面創下了職業生涯新高，也在慢慢的學著成為一個合格的領袖，而續約太陽隊的烏佈雷，本賽季至今效率和數據方面都有所提升，成為了太陽隊三號位的希望，特別是那些新加盟的球員，盧比奧雖然得分能力欠佳，但是大局觀和組織能力卻是有目共睹的，在他的組織下，太陽的進攻變得更加合理，貝恩斯加盟太陽後，竟然迎來了第二春，從之前藍領型中鋒，一舉轉變為三分命中率超過四成的投手，卡明斯基、沙里奇、卡梅倫約翰遜等球員，也能為球隊提供足夠的三分火力，而且本賽季至今，太陽隊的場均得分排名聯盟第五，場均助攻為聯盟第一，要知道開賽階段的太陽隊，同樣經歷了休賽期陣容的大變動，也正在處於磨合期，甚至打了一場比賽之後，就失去了內線的核心艾頓，而隨著賽季的深入，陣容磨合的更加默契，艾頓回歸後給球隊提供需要的防守和籃板，這樣的太陽隊，表現甚至可能更加出色。

活塞隊

活塞隊算不上強隊，也的確稱不上弱旅，雖然從16賽季至今，活塞隊只殺入過1次季後賽，還是上個賽季勉強進入，本賽季至今在東部的排名中，落後第八魔術隊1.5個勝場，只能屈居第九，但是要知道開賽階段的12場比賽中，是一支缺少核心格里芬的活塞隊，而在格里芬回歸後的12場比賽中，球隊打出了50%的勝率，甚至在和馬刺隊的比賽中，以34分的巨大分差大勝了對手，

而且活塞隊相比上個賽季來說，肯納德一躍成為聯盟最出色的投手之一，場均可以貢獻16.3分3.4籃板4.2助攻，休賽期簽下的馬基夫莫里斯，在進攻效率方面，甚至比內線核心格里芬還要出色，而簽下羅斯，讓活塞隊終於有了一名，在關鍵時刻可以接管比賽的球員，解決了球隊關鍵時刻乏力的缺陷，如果球隊能夠遠離傷病的困惱，本賽季的這支活塞，還是值得大家去期待的。

馬刺隊

本賽季馬刺打出了堪稱最近20來年中最差的開局，也讓老帥波波維奇體驗了一把8連敗的滋味，在本賽季的西部球隊中，馬刺隊甚至可能最終無緣季後賽，但是即便如此，也千萬別把馬刺隊當成一支NBA的弱旅，因為即便是瘦死的駱駝，也遠遠的比馬大，本賽季開賽階段，馬刺能夠輸給弱旅，被活塞隊大勝34分，也能戰勝馬賽克隊、快船隊這樣的奪冠大熱，而且別看馬刺隊戰績不咋地，但是場均得分、場均籃板、場均助攻排名全部都在中游偏上，場均失誤排名聯盟24，這樣的球隊算是弱旅嗎？
