21世紀NBA三大最感人場面！每場都讓人熱淚盈眶滿含淚水

2019-12-03 0條評論 18次閱讀 0人點贊

一：科比退役。

2016年4月14日。科比在職業生涯謝幕戰比賽中打爵士，科比爆砍60分，准絕殺了爵士，賽後科比在謝幕演說中一句：「mamba out！」結束了自己的職業生涯！全世界不知道有多少人聽到這句心裡會有莫名的辛酸，再見了科比，再見了，我的青春！

二：詹姆斯騎士奪冠。

2016年總決賽，詹姆斯率領的騎士在3比1落後勇士的情況下連勝三局贏的總冠軍！賽後詹姆斯跪在地上痛哭，大喊：「克利夫蘭this is for you！」作為一個克利夫蘭出生的孩子，終於在自己的努力下把總冠軍帶到了這個50年來沒有大賽冠軍的城市！也完成了他對克利夫蘭的承諾，這一刻也是讓無數人動容！

三：玫瑰綻放，50分之夜。

北京時間2018年11月1日，森林狼主場128比125險勝爵士隊，羅斯爆砍50分，幾乎是以一己之力帶領球隊取得勝利，賽後羅斯放聲痛哭的畫面成為全世界的焦點，羅斯含淚表示：「50分對我來說意味著一切，如果沒有我不停的訓練，沒有森林狼這支球隊，沒有球迷的支持，我是很難取得這樣的成績的！」羅斯的不幸，羅斯的不屈也讓無數在生活中受到打擊的人們相信，倒下了我還是可以站起來！
