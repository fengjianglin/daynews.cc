活塞被凱爾特人完敗，吞下三連敗，沃克表現不盡人意

2019-12-21 34條評論 2,969次閱讀 765人點贊

摘要：12月21日，凱爾特人主場以114比93完勝活塞，這場比賽活塞布雷克·格里芬因為膝蓋受傷缺席，沒有格里芬的活塞很難和綠凱相抗衡，他們頂住了首節，第一節打完僅落後了兩分，第二節活塞開始崩盤，第二節打完後分差直接來到了12分，第三節比分又被擴大，來到了22分，末節成了垃圾時間，活塞最終落後21分輸給綠凱，綠凱拿下了2連勝，活塞則吞下了三連敗。

本場比賽，羅斯和梅克表現不錯，羅斯拿下了14分4助攻，索恩·梅克替補23分鐘內9中4，獲得15分7籃板1搶斷，德拉蒙德全場31分鐘內11中5，獲得12分11籃板4助攻，這場比賽活塞輸的有些慘，因為活塞在綠凱億元球星肯巴·沃克全場6中0，三分3中0的情況下還吞下了21分大敗。

沃克此次在比賽中打了30分鐘，6中0已經很久沒見了，本賽季有一場對決掘金傷退是3中0，但是那一場沃克就只打了12分鐘，可以看出本場比賽沃克是很低迷的，而沃克在如此低迷的情況下活塞還是大敗而回。

這跟球隊的綜合實力是有很大關係的，當然少帥史蒂文斯的調整也是功不可沒的，塔圖姆本次很高效，全場27分鐘，17中10，三分7中2，杰倫·布朗的效率比塔圖姆還要高，在第二節單節6中5，綠凱雙星收穫了52分10籃板8助攻5搶斷2蓋帽。

本場比賽，格蘭特·威廉姆斯打了30分鐘，創造了賽季登場的新高，上一場格蘭特·威廉姆斯進入輪換打了25分鐘，少帥看他很積極，就讓他本場比賽打主力輪換，結果卻成了超級奇兵。

30分鐘內10中8，三分3中2，得分也是賽季新高，這個1米98的鋒線小將在本場比賽的表現著實讓人眼前一亮。
