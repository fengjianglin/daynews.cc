詹皇21+12+11濃眉36分 雄鹿斬快船後再滅湖人 3原因令湖人落敗

天天要聞
2019-12-20

湖人隊被終結了客場14連勝，而雄鹿隊也被終結了18連勝，兩支球隊對決之前卸下了各自的「包袱」。兩支球隊戰績同為24勝4負，共同保持著聯盟目前的最佳戰績，稱其為巔峰對決毫不為過。兩支球隊賽季前就是總決賽熱門球隊，因此這場比賽也被外界一致稱為「總決賽預演」。

對於湖人隊不利的是，過去兩場比賽他們的外線投籃狀況太過糟糕，兩場比賽合計只有62投13中。對陣雄鹿隊的比賽，若還是這樣的外線命中率，無疑會讓球隊在比賽中非常艱難。不過好消息在於，安東尼-戴維斯缺席一場比賽之後復出，湖人隊「詹眉組合」將重現他們的殺招。

字母哥賽前就猛誇了詹姆斯，他說道：「這很瘋狂，當然我的目標之一也是希望自己可以在未來10年都以高水準打球，但他這個月就要35歲了吧？他還能那樣移動，打出那種水準，打得很聰明，這很瘋狂，這說明了他的能力。不過，他是勒布朗-詹姆斯，他與眾不同，他是外星人，所以你對此習以為常。」

MVP球員間的對決、「總決賽預演」，火星撞地球的比賽誰能笑到最後？

字母哥和詹姆斯

湖人隊掌控首次進攻球權，詹姆斯漂亮地將球傳給底角丹尼-格林，格林手起刀落為湖人隊率先取分。兩隊都在外線嘗試手感，但是多次進攻無功而返，馬修斯造犯規兩罰一中，雄鹿隊也打破了僵局。比賽開局雙方在防守端投入大量精力，進攻端也因此受到不小影響，雙方都出現了一些失誤。

詹姆斯殺到籃下被字母哥封蓋，字母哥大步殺到籃下上籃偏出，每一次進攻都顯得艱難。比賽在場上陷入僵持的時候，迪溫琴佐和字母哥各中一記三分球，雄鹿隊率先走出了泥淖。詹姆斯造犯規兩罰兩中，丹尼-格林命中三分球，湖人隊反超比分。喬治-希爾連中三分球，雄鹿隊獲得了5分領先優勢。

進攻端效率都不夠高，比賽難免陷入到膠著狀態，而雄鹿隊的輪轉陣容稍佔優勢。喬治-希爾進攻得手之後，帕特-康諾頓也得到兩分，安東尼-戴維斯進攻打成。喬治-希爾造犯規兩罰兩中，首節比賽結束的時候，雄鹿隊23比17領先湖人隊，雙方首節比賽的得分都偏低。

拉簡-隆多

米德爾頓第二節比賽開局先得兩分，湖人隊陣容的缺點被字母哥抓住，字母哥連續衝擊籃下進攻得手。雄鹿隊開局打出一波6比0比分，分差被拉大到了12分，湖人隊趕緊叫了一次暫停。回到比賽當中，勒布朗-詹姆斯迅速命中一記三分球，籃下單打再得到兩分，為湖人隊止住了頹勢。

字母哥自信地在三分線外出手命中，科沃爾造外線投籃犯規三罰三中，米德爾頓中投命中，雄鹿隊領先分差達到15分。霍華德在籃下二次進攻打成，還沒等湖人隊喘過氣來，馬修斯、米德爾頓各中三分球，領先分差迅速拉大到19分。雄鹿隊的輪轉陣容明顯佔據優勢，湖人隊也因此異常被動。

波普進攻連續得手，丹尼-格林三分球手感火熱，分差被逐漸縮小。喬治-希爾三分直接穿針，字母哥籃下造殺傷，雄鹿隊的進攻勢頭沒有中斷。馬修斯三分球直接殺進，雄鹿隊領先湖人隊到了20分。安東尼-戴維斯造犯規兩罰一中，上半場比賽結束，湖人隊46比65落後雄鹿隊。

戴維斯對抗字母哥

丹尼-格林第三節比賽開局命中三分球，波普造犯規兩罰兩中，湖人隊連得5分縮小分差。字母哥造犯規兩罰一中，雄鹿隊第三節比賽打破僵局。詹姆斯在外線直接飆中，字母哥迅速就作出回應，戴維斯也在進攻端發威，湖人隊的進攻在下半場比賽稍有起色，這段時間對於湖人隊相當關鍵。

字母哥的外線出手非常自信，布魯克-洛佩斯造犯規兩罰兩中，丹尼-格林再次三分飆中。喬治-希爾連續在進攻中得分，湖人隊追分變得非常困難，雄鹿隊仍保持著較大領先優勢。然而湖人隊在中段打出一波攻擊流，戴維斯打成2+1、波普三分得手以及戴維斯突破得分，湖人隊追至11分分差。

凱爾-科沃爾三分再中，雄鹿隊緊緊地保持著領先的勢頭，湖人隊追分時刻每一次得分都很重要。而在第三節比賽後半段，戴維斯開始在進攻端發力，連續造犯規罰球命中為湖人隊砍分。拉簡-隆多硬吊籃下，戴維斯將球點進籃筐，三節比賽結束，湖人隊81比89還落後8分。

雄鹿隊球員

末節比賽開始之後，字母哥在籃下繼續殺傷，他甚至還造了詹姆斯個人第五次犯規。詹姆斯三分球命中，米德爾頓急停跳投得手之後，字母哥也命中三分球，雄鹿隊98比84領先14分。回到比賽中，湖人隊還是靠戴維斯在進攻端打開局面，米德爾頓又是一記三分球命中，湖人隊很受打擊。

詹姆斯突破投籃得手之後，湖人隊剛剛將分差追到9分，馬修斯干拔命中三分球，分差重新到了兩位數。戴維斯打成一次2+1，雙方進入到了最後的決戰時刻。戴維斯防守犯規，馬修斯造犯規兩罰一中，詹姆斯卻在關鍵時刻失誤。最終全場比賽結束，雄鹿隊111比104擊敗湖人隊。

比賽開場之後，雙方就展現出防守端的強度，不過第二節比賽雄鹿隊率先打開進攻，並因此奠定領先的局面。湖人隊最終沒能完成逆轉，原因在哪裡呢？首先，湖人隊的防守做得並不好，尤其是被雄鹿隊在外線投開，這給湖人隊造成巨大壓力；其次，湖人隊的外線投籃依舊比較起伏，全隊三分球35投12中；再次，湖人隊全場比賽失誤過多，全場比賽共有18次失誤。

雄鹿隊的狀態有多出色？對陣快船隊和湖人隊都曾領先到20分以上，也連續擊敗了洛杉磯兩個強大對手。
