為什麼皮蓬一直覺得詹姆斯不如自己？

天天要聞
2019-11-29

比賽精彩，賽場外的話題同樣精彩

最近這段時間，NBA正在如火如荼的進行著常規賽，聯盟30支球隊為大家帶來了精彩的比賽之餘，也為觀眾們提供了很多茶餘飯後的談資。

最近這段時間，關於NBA有史以來，誰最能代表”小前鋒”這個位置，就引起了巨大的爭議，當然了，爭議的來源還是皮蓬的那句“現在的年輕人在小前鋒這個位置上只要記住我就行，沒必要記住詹姆斯等人的名字，因為在所有的明星小前鋒里，我的冠軍戒指最多”

「唯冠軍論」是長久話題

其實在NBA，”冠軍論”是一個長時間來各方都僵持不下的話題，對於一位超級巨星來說，總冠軍戒指意味著什麼不言而喻，很多球星在拿到足夠多的分數，足夠多的個人榮譽之後，總冠軍是他們唯一的追求，強如馬龍，斯托克頓，窮極一生也未曾登上過巔峰，但是誰能否認他們的偉大嗎？顯然也是不能的。

為什麼皮蓬會覺得詹姆斯不如自己？

對於誰是”歷史第一小前鋒”這個問題，很多現役的球員和媒體都發表過自己的看法，大部分的人，都覺得詹姆斯當之無愧，可是唯獨皮蓬，覺得自己才是這個位置的最佳人選，為什麼他會對自己這麼有信心呢？其實我們細細的品味這個問題，原因不外乎兩點。

首先，他需要被人關注。

作為一個已經退役許久的球星，皮蓬在場外的生活可謂是一地雞毛，在這裡我們就不細說了。作為一個已經退役的巨星，他是一些電視節目的常客，參加這些節目，能給他們帶來不菲的收益，這是他們的生財之道，在這些節目上，他們需要一些”標新立異”的觀點來支撐自己的關注度。

其實皮蓬跟詹姆斯完全是兩個不同時代的球星，他們之間沒有個人恩怨，可是現在的詹姆斯，代表著目前NBA的最高競技水平，貶低詹姆斯，強化自身，這才能顯示出自己當初作為球員時是一個多麼偉大的存在。而自己的這些”觀點”，自然而然的會引起觀眾們的爭議，當爭議存在的時候，節目的熱度自然就會高。這樣的做法，給我個人的感覺，有點”嘩眾取寵”的味道。

其次，他需要維護喬丹的歷史地位。

皮蓬的6個總冠軍戒指，來自於喬丹，這一點，相信大家不會反對。不否認皮蓬的自身實力，但是離開喬丹，他並不能算當時聯盟里的一流球星，喬丹短暫退役的那兩年，皮蓬並沒有把公牛隊帶到一個足以證明自己的高度。

作為喬丹身邊的輔助者，他自己深刻的認識到喬丹的地位對於自己的加成有多大，在歷史第一人旁邊打球，並且一起拿下兩個三連冠，這是一件多麼榮耀的事情。

可是現在，詹姆斯已經隱隱有了超越喬丹躋身第一人的架勢，不管是詹姆斯身上累積的榮譽，還是他在場外的影響力，詹姆斯正在逐漸衝擊喬丹的歷史地位，加上現在媒體的各種造勢，接受詹姆斯的球迷們越來越多，畢竟相對於喬丹來講，現在大部分的球迷都沒經歷過那個時代，而詹姆斯，正在用自己的表現征服現在的年輕人。

在這種情況之下，皮蓬站出來說自己才是第一小前鋒，除了”黑”詹姆斯幫自己製造關注度之外，無非就是想告訴大家，在他心裡，詹姆斯連自己都未必超過，又何來超越喬丹的說法呢？

對於一名職業球員來講，看到後輩們逐漸崛起超越自己，是一件比較難以接受的事情，就拿昨天的卡特來說，自己隔扣了別人一輩子，昨天被字母哥隔扣了之後還眼眶微紅，可見內心是驕傲的。皮蓬也是如此，自己的職業生涯奮鬥了十幾年，所取得的成果正在被後來的年輕人逐漸超越，內心想必是不服氣的，可是又不能場上見個高低，那在嘴上定然要表現出來，這是人之常情。

就好像現在的很多”詹黑”，他們本身比起詹姆斯相差十萬八千里，不也認為詹姆斯其實不怎麼樣嗎？而皮蓬這個本就取得了諸多成就的球星，又怎麼可能輕易的認為別人比自己更優秀呢？
