NBA故事之范弗里特和德拉維諾瓦：平凡小個子的逆襲

2019-11-27 0條評論 18次閱讀 0人點贊

范弗里特和德拉維諾瓦擁有驚人的相似之處

1.以板凳奇兵幫助球隊擊敗勇士奪冠

2018-2019賽季季後賽中，作為替補的范弗里特扮演板凳奇兵，在東部決賽中他單場投進7個三分，在總決賽中，范弗里特再次挺身而出，在最後一戰中三分球11中5拿到22分，撲滅了勇士逆轉的最後希望，幫助猛龍奪得冠軍。

而在2015-2016賽季總決賽中，也出現一個替補奇兵，他就是德拉維多瓦。騎士在東部季後賽中未逢敵手，但來到總決賽他們就得提起精神，因為他們面對曾拿到73勝的勇士，正值巔峰的庫里在三分線外不可阻擋。這時候，臨危受命的德拉維多瓦奉命主防庫里，他瘋狗式的防守令庫里節奏全失，最終幫助騎士在1-3落後的情況下完成驚天大逆轉。這時候的德拉維多瓦不過是年薪120萬美元的小人物。

2.同是落選秀，天賦平庸

范弗里特是2016年的落選秀，身高1米83的他沒有爆發力，速度也只是平均水準，他的第一份合同是2年145萬美元，其中第二年是無保障合同。

而德拉維多瓦是2013年的落選秀，落選後只能以2年130萬美元與騎士簽了部分保障合同，和范弗里特一樣，身高1米93的德拉維多瓦沒有勁爆的身體素質，沒有華麗的技巧，也沒有高超 的籃球智商。

但是天將降大任於斯人也，必先苦其心志。生涯初期的挫折並沒有磨滅兩位落選秀對未來的憧憬。

范弗里特身上最寶貴的可能就是他心中的自信了。從生涯初期開始就沒人看好他，但是他一直相信自己的投籃能力，只有手裡有球，不管是常規賽還是總決賽他都能從容的出手。對於上賽季總冠軍的表現，

他這樣評價：上賽季的總冠軍也證明了我們的陣容深度，因此我想說，那些能夠表現出色的球員一直都在，只是需要更多的機會，更加自信。

是的，范弗里特一直保持自信，並時刻保持著準備的姿態，終於在總決賽他證明了自己，本賽季的他更上一層樓，場均能貢獻18.5分7.7助攻，幫助球隊在失去萊昂納德後還能高居東部第四。

而德拉維多瓦和范弗里特不一樣，他沒有高超的外線投籃能力，但是他有無比敬業的態度，他用拚命的方式投入比賽，他身體素質並不出色，但為了消耗對方，他拚命的加強身體接觸、保持與對方的高對抗，用無休止的的熱情撲向對方，他的敬業態度和防守質量贏得了教練和球員的尊重。在2016年奪冠後，德拉維多瓦獲得雄鹿4年3800萬的合同。

還記得當年讀書的時候，NBA的口號是where amazing happen，見證奇蹟的地方。范弗里特和德拉維多瓦也用自己的生涯詮釋著這個口號。
