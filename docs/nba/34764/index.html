馬刺狂勝！阿德一數據比肩鄧肯 傳球開掛！西帝17助攻生涯之夜

2019-12-25 1,951條評論 4,583次閱讀 3236人點贊

北京時間12月24日，2019-20賽季NBA常規賽繼續進行。

馬刺客場145-115大勝灰熊，排名在悄無聲息間已回到了西部第9的位置。

要知道，此前一段時間，馬刺連戰連敗，一度有些要徹底擺爛、重建的意思。不少資深的美國記者，也從各自消息人士口中得到了類似的消息。但最近一段時間，他們卻展現出復甦勢頭。

本場之後，馬刺書寫了多項紀錄。首先，145分創下了馬刺自2010年以來常規時間（不包括加時）單場得分的新高。此外，145分是1996年格雷格-波波維奇執教以來第3高的球隊單場得分。

另外，本場，馬刺全隊的投籃命中率達到了67.4%（89投60中），創下了波波維奇執教期間的單場新高。不僅如此，馬刺全隊送出36次助攻，這也是他們的賽季新高。

賽後，在接受採訪時，波波表示，「是他們（阿爾德里奇和德羅贊）領導了球隊，很高興全隊都融入了進來。」

這個夜晚，雙德的發揮的確值得讚歎，其中，德羅贊上場29分鐘，11投10中，罰球6罰6中，高效轟下26分3籃板10助攻3搶斷。

由此，德羅贊成為馬刺隊史上首位以超過90%的命中率打出得分25+助攻10+的球星，也是NBA歷史上第15位做到這一點的球星。

至於阿爾德里奇，他表現更為搶鏡，僅僅首節，他就轟下22分。全場，阿德上場32分鐘，25投17中，三分5投3中，罰球4罰3中，砍下了40分9籃板5助攻3蓋帽。

這樣一來，阿爾德里奇在馬刺效力期間已經5次在常規賽砍下40+，追平馬刺傳奇大前鋒蒂姆-鄧肯，排名馬刺隊史第五位。

在東部，也有一位球星發揮搶鏡，76人客場125-109擊敗活塞，76人核心本-西蒙斯出場37分鐘，14投8中，得到16分13籃板17助攻。

數據顯示，這是西帝的第24次三雙，在生涯前三個賽季，就做到這一點，歷史上除了西蒙斯之外，只有奧斯卡-羅伯特森（87次）和魔術師-約翰遜（31次）兩大傳奇了。

另外，17次助攻，是西帝的生涯單場新高，此前，他單場最多送出15次助攻。而自2000年以來，西帝是球隊第三位單場助攻17+的球員，另外兩位分別是TJ-麥克康奈爾（17次）和安德烈-米勒（18次）。
