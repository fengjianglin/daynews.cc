3+3+3！湖人又築高牆，防守漏勺砍4+3神數據正名，紫金軍團太狠了

2019-12-05 0條評論 17次閱讀 0人點贊

背靠背擊敗掘金和爵士，湖人收穫2連勝的同時也收穫了信心，此前坊間一直質疑他們打硬仗的能力，這兩場勝利就是最好的回擊。防守仍然是湖人贏球的秘訣，兩場比賽都沒有讓對手得分破百，這也是湖人本賽季第8次限制對手得分不超過100分，這已經超越了18-19賽季一整個賽季的總和。

此役，湖人在進攻端轟下121分的同時，僅讓爵士得到了96分，銅牆鐵壁般的防守讓爵士感受到了絕望。可以從兩項數據中感受到湖人隊的防守有多讓人窒息，他們全場比賽送出了12次封蓋13次搶斷，反觀爵士僅有3次封蓋3次搶斷，這差距實在是太大了。

湖人有三名球員送出了3次封蓋，濃眉哥3次、霍華德3次、庫茲馬3次，另外麥基、丹尼-格林、波普各貢獻1次，全隊12次封蓋。搶斷方面最多的是庫茲馬4次，麥基和隆多各2次，濃眉、詹皇、杜德利、卡魯索、丹尼爾斯各1次，全隊14次搶斷。

值得注意的是，庫茲馬在搶斷和封蓋方面都很搶眼，一人貢獻4次封蓋3次搶斷，這是頂級防守運動員才有的數據。防守一直是庫茲馬的軟肋，這一點不能否認，在很多球迷眼中，他就是球隊的防守漏勺，他在防守意識和判斷力上的確有不足之處。但本場比賽，庫茲馬拼了，尤其是最後垃圾時間也沒有放棄防守，全場狂奔追帽對手很帶勁，他表明了自己的態度，努力有時候可以彌補一些東西。

這是湖人本賽季第4次蓋帽次數上雙，12次封蓋是賽季第二高，最高的一次是全隊14次封蓋，出現在擊敗老鷹隊的比賽中。這也是湖人本賽季第7次搶斷次數上雙，13次封蓋也是賽季第二高，最高的一次是14次搶斷，出現在本賽季首次對陣爵士隊的比賽中。

賽季至今，湖人在蓋帽榜單上高居聯盟第一，搶斷榜單上高居聯盟第五，場均失分聯盟第二少，防守效率聯盟第四，紫金軍團在防守一端太狠了，而防守已然成為他們本賽季的招牌和制勝法寶。濃眉+霍華德+麥基組成的「死亡三塔」讓湖人內線成為「禁飛區」，丹尼-格林+波普+布拉德利+卡魯索構築的外線鐵柵欄組成了湖人第一道防線。

擊敗爵士後，湖人主教練沃格爾被記者問到兩連勝期間印象最深刻的方面，他表示：我們積極地起跳、封堵投籃，再努力地通過二次起跳干擾對方的出手，我對此印象很深。
