東契奇鬱悶了！能否加速庫班交易腳步？1人適合球隊卻存交易難題

2019-11-27 0條評論 14次閱讀 0人點贊

輸給快船隊的比賽，20歲的盧卡-東契奇鬱悶不已，他在比賽中被對手的防守輪流消耗，始終沒能找到自己熟悉的比賽節奏。貝弗利撕咬了一陣之後，保羅-喬治跟他對位，倫納德也在防守中站在他的身前，任誰都難以抵擋如此高質量的防守。這三位球員的防守質量，在聯盟中堪稱是頂級，東契奇被限制住自然並不奇怪。

東契奇賽後未接受採訪

比賽中表現不佳，東契奇相當鬱悶而且沮喪，賽後直接離開球館未接受媒體採訪。東契奇雖然年紀不大，他的職業生涯起步卻非常早，對於勝利的極度渴求更是在年輕球員中罕有。在NBA聯盟經常見到效力雷霆隊時期威少的勝利慾望強烈，他會在球隊輸球時坐在球館的場邊一言不發，像是一座到了臨界值的火山。

東契奇也是如此，內心中對勝利無比渴求。

上個賽季是東契奇的新秀賽季，他打了幾場比賽之後就表示，自己遇到的挑戰並非來自比賽，而是球隊的勝率方面的困擾。東契奇說道：「輸球讓我很沮喪，因為之前我一直都是在一支經常獲勝的隊伍中。因此當我們輸掉一場比賽後，我就會感到很抓狂，這就是不同之處。」

東契奇對勝利充滿渴望

獨行俠隊本賽季的陣容實力較上賽季有了一定提升，球隊的戰績很能夠說明問題，但是獨行俠隊跟頂級強隊還有一定差距。對陣快船隊的比賽，無疑就被對手上了一課，獨行俠隊的陣容還有致命缺陷。這個缺陷就在於，球隊太過於依賴東契奇，一旦東契奇被對手限制住的話，情況就會變得危急起來。

為了應對這個問題，獨行俠隊在第三節比賽派上老將巴里亞，在一段時間裡居然成了獨行俠隊奇兵。然而尷尬之處也就在這裡了，巴里亞並不是獨行俠隊的常規武器配置，將他派上場也是沒有辦法的辦法。東契奇是球隊的發動機，但是一支球隊只有一部發動機顯然會遇到麻煩，這便是獨行俠隊的最大問題。

安德烈-伊戈達拉

獨行俠隊不知道問題出在哪裡嗎？當然他們更清楚自身缺陷在哪裡，休賽期也對此做過一些補強，然而德隆-賴特、賽斯-庫里等人雖然各有所長，卻不精於做球隊的組織者角色。而且更為重要的是，獨行俠隊還考慮過從灰熊隊交易伊戈達拉，開出的籌碼是考特尼-李和一個未來的次輪簽。

近日這筆交易再一次被提及，《體育畫報》記者Mike Fisher報道稱，灰熊隊目前的回應是他們不願意接手考特尼-李的合同。

伊戈達拉休賽期被勇士隊交易到灰熊隊，很快就得到了多支球隊的青睞，但是各支球隊都在等待灰熊隊將其買斷。灰熊隊並不願意便宜他人，不會對伊戈達拉進行買斷，只接受獲得一定回報的交易。獨行俠隊有意做這筆交易，但是提供的交易籌碼卻難以讓灰熊隊滿意。

馬克-庫班

伊戈達拉看起來很適合獨行俠隊，既能夠組織又能夠打無球，防守也是一把好手，他的經驗對於年輕的東契奇會很有幫助。儘管如此，獨行俠隊也並不想拿出更多的交易籌碼，原因在於伊戈達拉的合同只剩下最後一年——獨行俠隊即便完成交易也不能夠迅速升級為爭冠球隊，「一年使用權」對獨行俠隊來說並無太大意義。
