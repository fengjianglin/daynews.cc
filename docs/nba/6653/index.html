壞老頭！昨天把籃筐扣歪了，沃格爾：球隊禁止詹姆斯在訓練時扣籃

2019-11-27 0條評論 17次閱讀 0人點贊

北京時間11月27日，今日湖人隊主教練弗蘭克.沃格爾在投籃訓練結束後，接受了隨隊記者Mike Trudell的採訪。沃格爾在採訪時透漏現在球隊禁止隊內核心勒布朗詹姆斯在訓練場扣籃，因為在昨日對陣馬刺的比賽之前，詹姆斯把湖人訓練場的籃筐給扣歪了，以至於本來要進行的5V5合練被耽誤了20分鐘左右。同時沃格爾還透漏，詹姆斯在扣彎籃筐後不僅沒有表達歉意，還在一旁發出怪笑聲。

對此，沃格爾說道：「現在球隊禁止勒布朗在訓練時扣籃，尤其是在客場的訓練館，這個傢伙的力氣太大了，而且充滿著用不完的精力。他昨天把AT&T中心為我們準備的訓練場地給破壞掉了，他來了一記暴扣，隨後籃筐歪掉了，而我們無法及時的聯繫到AT&T中心的工作人員。以至於原本應該進行的5V5合練被推遲了20分鐘左右，雖然這對我們的訓練造成不什麼影響，但我們擔心他的手會因此受傷。」而在被問到「幹了壞事」後詹姆斯的反應時，沃格爾吐槽道：「他就呆在一旁，朝著其他人發出gigigigigigigi的怪笑聲。」

而在談到明日對陣鵜鶘的比賽時，沃格爾說道：「鑒於兩支球隊在夏天發生的事情，我可以預料到這是一場非常有趣的比賽。AD可能會想在鵜鶘球迷面前展現自己，他會得到一些噓聲，而這也會變成他的動力。布蘭頓（英格拉姆）也會想在湖人面前表現自己，所以，我非常期待明天會發生什麼。但不管雙方的球員怎麼想，我認為我們都不會輕易的取得比賽的勝利，我對此已經做好了準備。」 沃格爾說道。

老詹這個壞老頭呀，他真是太壞了！沃格爾還不知道昨天是什麼日子么？昨天是Taco Tuesday啊！雖然美國那邊比咱們要慢13小時，但也差不多是老詹搞怪的時候了。你以為這老漢是無意間把籃筐扣歪的么？錯了！他就是故意扣歪然後用那20分鐘的休息時間去拍視頻呢！建議沃格爾去查查濃眉哥或者是丹尼格林的流量使用情況就知道了，就那20分鐘，這老頭連著他們的熱點都不知道用了多少呢！
