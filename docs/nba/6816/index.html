後場：拉文、布克

布克上場38分鐘，得到35分12板9助攻，其中有3前板，可惜只差1助攻頭能完成三雙，三分4中0，但罰球11全中，隊友烏布雷發揮同樣出色，得到25分10籃板1助攻4搶斷，太陽100-98擊敗森林狼終結三連敗。

拉文率領公牛116：115一分險勝黃蜂，全場28投17中，其中三分17投13中，命中率超高，得到49分4籃板，這樣的表現理應上榜。

老鷹116：119惜敗猛龍，特雷.楊上場41分鐘，得到30分10籃板10助攻的三雙，無奈球隊整體實力不如猛龍，沒能率隊取勝。

爵士128：120戰勝鵜鶘，爵士後衛米切爾表現出色，得到37分7籃板5助攻。

前場：西亞卡姆、字母哥

猛龍前鋒西卡上場40分鐘，18投11中，其中三分6投4中，得到34分4籃板3助攻1搶斷2蓋帽，幫助球隊取戰勝老鷹。

雄鹿104：90輕鬆取勝活塞，字母哥只上場27分鐘，但表現高效，22投12中，得到28分10籃板3助攻，還有2搶斷1蓋帽。

中鋒：唐斯

森林狼2分之差惜敗太陽，但唐斯發揮出色，上場38分鐘，命中6記三分得到31分17板4助攻，不以成敗論英雄，唐斯表現值得入選。

76人113：86大勝熱火，恩比德只上場25分鐘，表現高效得到23分11籃板2助攻2搶斷2蓋帽。
