連勝是有原因的！雄鹿悍將：我們不是衛冕冠軍，得保持獵人心態

2019-12-11 203條評論 2,449次閱讀 1019人點贊

密爾沃基雄鹿的韋斯利-馬修斯近日在接受ESPN的記者伍德亞德(Eric Woodyard)採訪時表示，希望球隊不要因為他們聯盟本賽季聯盟最長的連勝記錄，和在2019-20賽季暫時的聯盟第一的戰績而自滿。要在學會保持「獵人心態」。

「雖然我們現在是東部第一，但我們不是衛冕冠軍，這才是我們想要的。」馬修斯說道。「我們希望能夠這樣稱呼自己，為了做到這一點，我們必須始終保持獵人心態。」

本賽季截止至目前，密爾沃基雄鹿隊已經豪取15場連勝，總戰績達到了21勝3負。這是他們繼上賽季取得突破性的60勝之後的又一個偉大的賽季。但在過去未久的季後賽中，他們在2:0局勢一片大好的情況下，被最終的冠軍多倫多猛龍隊連扳四場淘汰出局。

現在，邁克-布登霍爾澤執教的這支雄鹿隊比以往任何時候都更有動力去完成另一個令人印象深刻的常規賽。「字母哥」楊尼斯-阿特托昆博繼續向所有人證明他是聯盟最有價值球員，並且有希望再獲得一個MVP獎盃。

對於馬修斯來說，他作為一名定點三分投手的任務完成的相當不錯，成功的為字母哥的衝擊籃筐提供了足夠的空間。作為一名老將，他在NBA已經待了足夠久的時間，他知道為了贏得那個對於許多球員來說整個職業生涯都遙不可及的總冠軍，一支球隊整個賽季都應該保持饑渴。

職業生涯截止至目前，33歲的馬修斯出戰10個賽季748場比賽，場均能夠得到13.5分+3.1個籃板+2.2次助攻+1.1個搶斷；本賽季至今，他代表雄鹿出場24場，場均上場24.1分鐘，能夠得到8.2分+2.3籃板+1.5助攻。
