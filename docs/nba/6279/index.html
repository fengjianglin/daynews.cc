爵士沒有“爵”，他們可能邯鄲學步了

2019-12-03 0條評論 16次閱讀 0人點贊

在連續兩場大比分落後，奮起直追也只能稍微挽回點顏面的比賽後，爵士本賽季的問題不得不擺到檯面上好好研究一下了。他們防守效率是104.1，排在聯盟第11，以爵士的防守名望，顯然不夠理想，但考慮到戈貝爾缺陣那兩場的防守崩盤，艾德戴維斯缺席了大量比賽，以及小樣本下的數據波動——畢竟爵士在被猛龍暴虐前，防守還能排到聯盟第5——這一端多半能調回去。

關鍵是他們心心念念想要改善的進攻，不但沒有因為休假期的引援見到提升，反而因為多種不利因素疊加後，掉到了更糟糕的位置——上賽季爵士進攻效率還有110.3，排在了聯盟第14，現在只有105.3，掉到了聯盟第23。

爵士真的爵了嗎？


其實沒有，他們今天跟76人這場是鐵了很久，但以整個賽季看，得益於超級大腿7突博揚的神勇投射，爵士本賽季的三分命中率反而穩中有升，現在來到了37.3%，已經是聯盟第5——「爵」隊不準的詛咒，起碼對博揚是無效的，甚至賽季初鐵到無語的邁克-康利，三分準星也調回到正常，起碼在首發陣容里，他們的外線投射效率不是最大的問題。

爵士可能…邯鄲學步了…

過去我們對爵士進攻的評價大概是…一通操作猛如虎，一看命中0-5…

這話肯定誇張了，爵士的進攻，過去最大的問題不是投不進——他們已經連續4個賽季三命中率在36%以上，得益於良好的內線終結、掩護走位和傳導球體系，他們也可以打出漂亮的攻筐數據，一直是聯盟里出手分布非常魔球化的球隊，在有效命中率一項上，爵士過去三個賽季都排在了聯盟前10。他們投籃不準的印象，更多來自季後賽——畢竟大部分球迷平時並不關心爵士的比賽，而最近兩個賽季他們季後賽出局都輸給了同一個對手，特別是今年的季後賽，他們的確鐵的令人髮指。

爵士進攻過去最大的問題，還是操作的複雜程度，與實際進攻效率不成正比。一方面，爵士不斷用掩護、跑位、傳導球創造攻筐和空位三分，幫助球隊提升了投籃效率；另一方面，也因為打得複雜，失誤太多——爵士的失誤率，多年來都很高。

這個毛病在常規賽尚可接受，到了季後賽又有新的情況。17-18賽季，遭遇紅隊的無限換防後，爵士發現他們過於依賴體系的進攻，遇到拆體系能力頂級的球隊時有點玩不轉了——當然，那輪系列賽他們也沒有盧比奧。等到18-19賽季，他們再次遭遇紅隊時，發現體系球雖然能創造出機會，卻因為全隊的投籃準星被詛咒，沒辦法把機會轉化為得分，這種局面下，他們只有G4憑藉米切爾第四節的神奇carry，才帶走了一場勝利。

爵士在痛定思痛後，決心改造陣容來升級進攻：

一方面，他們需要更好的主攻手，在體系失效後，有更強的carry能力；

另一方面，他們過去三分投得好，是建立在體系創造空位出手能力聯盟頂級的情況下，陣容依然存在嚴重的空間瑕疵（1號位和4號位），球隊缺少足夠數量夠檔次的射手。

這是他們弄來康利和博揚的原因——理論上，康利+米切爾+博揚組成的主攻核心團隊，配合1~4全部擺放射手的空間，理應交出更簡單卻更有效的進攻表現才對。

結果爵士的進攻的確變得簡單了很多，他們上賽季每場比賽要傳球310.7次，本賽季掉到了282.7次。他們從一支很依賴體系創造得分的球隊，變成了個人得分能力占極大比重的球隊，助攻率從聯盟第4，掉到了聯盟第26。

再次重申，傳球少，助攻少，不代表進攻一定不好——紅隊和開拓者的傳球從來就不多，上賽季的助攻率分別是聯盟第28和第27，但他們上賽季是聯盟進攻第2和第3的球隊。畢竟，他們後場的持球人太厲害了。

爵士想要的，恐怕也是這種效果，就算康利、米球不如當初燈泡和雙槍的威力，球隊的鋒線不完爆那倆？綜合下來，就算達不到紅隊和開拓者的進攻檔次，進入聯盟前10，再配合頂級的防守，擠進爭冠行列的念頭就不是痴心妄想了。

結果爵士發現，他們期待的主攻手升級，帶來的進攻簡單有效，失誤大幅減少沒有發生——爵士的失誤率不降反勝，康利自己的失誤控制就不如以往，並且除了他跟米切爾，球隊其他球員依然控制不住失誤，博揚準星雖棒，失誤不少，戈貝爾的失誤又變多了。

反過來，爵士過去最大的進攻優勢——創造大量攻筐和空位三分機會，從而實現魔球化出手分布，現在不存在了。爵士的三分產量降低倒還好，考慮到他們三分準星也提升了，外線投籃這塊不算退步。關鍵是攻筐頻率從過去的39.3%，掉到了34.6%，攻筐效率從65.1%，掉到了62.1%。爵士的中距離出手頻率大幅提升，從一支魔球化出手的球隊，退化成了中距離出手頻率聯盟第14的球隊——而他們中距離並不准。


原因何在呢？

康利的斷崖式下滑，是爵士進攻不能承受的損失。


康利撿回了三分準星，卻沒能把個人主攻表現撿回來，並且下滑幅度之大令人著實擔憂他的身體。

至少到目前為止，康利的個人主攻表現是無法讓人滿意的，他的擋拆持球效率每回合只有0.723，真實命中率只有49.5%——這顯然都不是合格主攻手應該交出的表現。

擋拆夥伴變化雖然有一定影響，畢竟擅長傳球和外拆的小加索爾比單純吃餅的戈貝爾更養後衛，但康利跟瓦蘭合作時，表現也比現在好太多，並且爵士的空間環境也遠好於那支灰熊。

康利是身體出了問題嗎？

現在還不能確定，但運動型數據的下滑令人擔憂，他過去就不是殺到最籃下的侵略型打法，本賽季的攻筐頻率又進一步下調，籃下命中率慘不忍賭。以往最拿手的小拋投，也完全失去了威力。上賽季，康利的拋投效率還有每回合1.047分，本賽季就只有0.709分。如果這項技能的效率找不回來，康利持球進攻最重要的武器就沒有了。

如果再考慮到康利的搶斷率來到了生涯新低，就更需要擔心他的身體狀況。

個人主攻下滑還只是一部分，康利的傳控表現也不如過去了。他沒有盧比奧那個級別的視野和風騷，但他向來有漂亮的助失比，結果本賽季不但每36分鐘助攻從6.9次掉到了5.5次，助失比也從3.45掉到了2.18。

對康利傳控表現下滑，可以有多種猜測， 比如他跟隊友之間球路是否需要更多時間熟悉：

你很難說這個球是康利傳得不好，還是戈貝爾有點準備不足

或者僅僅是因為身體不如過去，更難改變陣型：

我不確定，但可以肯定的是，爵士就是不太願意像過去那樣打球了。

這點也可以在英格爾斯和米切爾身上看到。

因為博揚的到來，加上英格爾斯自己的進攻狀態不好，英格爾斯現在成了替補。當然，英格爾斯每場比賽要打27.4分鐘，雖然比過去30多分鐘要少了，但大致還是首發球員的時間待遇。

英格爾斯的戰術地位也下滑了，他從神射手變成三分30%命中率的奇葩，用了也就短短半年多的時間，爵士因為有了更好的無球戰術執行者博揚，現在主要的無球戰術都設計給博揚了。

但英格爾斯的持球狀態，其實還可以，他還是能靠極低的球權，交出不錯的傳控數據，助失比維持的不錯。事實上，英格爾斯的擋拆產量沒有變少，但他擋拆中給隊友傳球的比重降低了，過去他有64.3%的擋拆會交給隊友終結，現在這個比重是51.3%。

類似的數據，在康利和米切爾身上也是一樣的——米切爾尤其可怕，他現在只有27.4%的擋拆會選擇找隊友，成為了一個球風極為「自私」的主攻手。

米切爾、康利、英格爾斯，他們都變得比過去的自己，更加不願意傳球了，而爵士另一個主攻手博揚，他本來就不怎麼傳球的。


所以總結下來，爵士進攻陷入困境的原因是這樣的：

他們得到了更好的主攻手——康利和博揚——但前者狀態奇差，主攻表現完全不合格，以至於讓後者在爵士的搶眼發揮被完全抵消；

爵士可能因為擁有了數量更多的主攻手，不再堅持以往強調傳導球的進攻體系，主攻手們變得更喜歡各自為戰，而不是試圖多傳球創造更好的機會。這其中，康利和米切爾的主攻，多以中距離出手結束，前者的拋投完全不準，後者的中距離準星雖然不錯，也不足以跟更多去籃下終結相媲美，於是爵士從一支魔球化出手的球隊，就變成了大量中距離的球隊，這勢必會降低整體的投籃效率；

由於吃餅環境變差，戈貝爾的回合佔有率降低，失誤率大幅提升。另一個失誤大戶博揚，他的主攻戲份比上賽季又大幅提升了，吃不掉的球權傳給隊友時，因為傳球質量不行，導致了不少失誤。他們倆是爵士本賽季失誤的主要來源，也直接導致爵士簡化進攻在降低失誤上沒得到收益。

以上原因綜合之後，就造成了爵士現在進攻的窘境：


學別人後場雙槍開掄不成功，反而丟掉了自己的體系，成了邯鄲學步了。

爵士要如何走出困境呢？

先等一等球員狀態的回歸吧——康利和英格爾斯有狀態了，什麼都好辦。

如果沒有那一天呢？

我不確定沒有了盧比奧，爵士的進攻是否還能像過去那樣賞心悅目——畢竟盧比奧+英格爾斯+戈貝爾，這三人才是那支爵士真正的進攻核心，米切爾更像獨立之外的托底和空間點。現在既沒有盧比奧，英格爾斯的狀態又不足以撼動博揚+奧尼爾的鋒線組合，爵士走回頭路有些艱難。

盧比奧、英格爾斯、戈貝爾三人連線的進攻

但就不算回到原來的體系中去，繼承過往的一些優良傳統，總不是一件壞事——你永遠不應該說，用耐心去尋找更好的出手機會是一件錯事。何況即使是一個100%狀態的康利，和進步的米切爾，他們也不是哈登，不是利拉德，爵士還是應該把球打得耐心一些，多用傳球找到隊友，而不是每個回合就想著一次拋投或者干拔中距離。

米切爾迷戀的得分方式

最後說一說米切爾。

米切爾在用高達47%的中距離出手頻率，去扛30.3%的回合佔有率，然後只能交出每36分鐘3.7次助攻，1.67的助失比，即使他41%的中距離準星並不算差，所有人也清楚，這不是一個頂級持球手應該有的打法——在這個頂級持球人遍地跑的賽季里，53.7%的真實命中率本就不好看了，再沒有傳控支持，米切爾僅僅靠高產支撐進攻表現是遠遠不夠的。

下圖是本賽季聯盟里回合佔有率高於25%的得分手，橫軸為助攻率，縱軸為真實命中率。

可以看到，米切爾現在的情況：

他的助攻率只能跟一票內線得分手，或者無球打法為主的球員相提並論，對一個持球大量打擋拆的後衛來說，實在不夠看；

他的真實命中率只比一些江湖中聞名的低效得分手，和本賽季狀態嚴重下滑的幾個球員高。

我不是說米切爾得分能力不強——起碼他用還湊合的效率保證了高產。但他對進攻的貢獻，到目前為止依然停留在「保住球隊下限」的水平：

過去，爵士體系失效時，米切爾站出來保下限；

本賽季康利打得不好，米切爾大量輸出也在保下限。

可問題是，爵士想要成為爭冠球隊，僅僅有米切爾級別的進攻下限是遠遠不夠的——一旦爵士其他人把喂餅的感覺找到時，你就會發現，米切爾對進攻反而是傷害的，與76人的比賽，英格爾斯找到擋拆喂餅感覺時，比米切爾的頻頻中距離干拔顯然更有用。

進步的長兩分干拔不應該是米切爾的終極武器。米切爾要變得更會傳球，才能反哺自己的個人輸出，讓防守人陷入猜謎，才有更多攻擊籃筐，提升效率的機會，從而成為投突傳更為全面的持球人。

爵士的上限，也就寄託於此了。
