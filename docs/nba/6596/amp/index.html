年輕版阿里扎為何不用德帥固執7人輪換戈登回來後年輕人還會上嗎

天天要聞
2019-11-28

北京時間11月28日，火箭在主場以117比108輕取熱火，結束3連敗。

此役，卡佩拉和戈登雙雙缺陣，德安東尼嘗試新的戰術，讓克拉克進入輪換陣容。本場比賽，克拉克打了29分鐘，和塔克、豪斯搭檔前場，哈登和威斯布魯克搭檔後場。這套陣容像極了2年前的阿里扎、塔克、巴莫特、哈登和保羅的經典5小陣容。克拉克砍下12分8個籃板1次搶斷和1次封蓋，三分球6中2。

熱火今天完全擋不住威斯布魯克和詹姆斯哈登的衝擊。全場比賽哈登22投9中，但三分17中7拿到34分5助攻6籃板，威少則21中11拿到了27分9板7助，雙少合力砍下了61+12+15的神勇數據。

這場比賽是克拉克本賽季首次出場時間超過3分鐘，此前17場比賽，克拉克只獲得了4次在垃圾時間出戰的機會。過去5場比賽，麥克勒莫的三分球命中率僅有21.8%，這是德安東尼不得已選擇克拉克的原因。事實上，克拉克的防守效率能夠排在聯盟前20，但是三分球命中率僅有29.7%。不得不說，德安東尼更加看重側翼的進攻能力，而不是防守。
