雄鹿想用身高佔便宜，卡萊爾祭出核武器，史詩級拼身高之戰太倔強

2019-12-18 1,551條評論 4,079次閱讀 2681人點贊

北京時間12月17日，NBA常規賽雄鹿隊在主場迎戰獨行俠隊，最終獨行俠隊120-116戰勝了雄鹿隊，18連勝就此終結，雄鹿隊幾度反撲，但是最終沒有能夠如願，而卡萊爾和布登霍爾澤的鬥智斗勇貫穿了整場比賽。

其中第三節最後0.03秒的對決都不放過，在第三節還剩0.03秒的時候，字母哥獲得罰球機會，當時布登霍爾澤為了提升場上搶籃板的優勢。布登霍爾澤用大洛佩斯換下了科沃爾，用小洛佩斯換下了喬治希爾。

而克萊爾也不甘示弱：欺負我獨行俠內線沒人是吧？好就給你看看，於是卡萊爾用馬楊諾維奇換下了庫里，用波爾津吉斯換下了懷特。這樣的鬥智斗勇也把場下的觀眾給逗樂了，就連兩個主帥也都樂了，波爾津吉斯也笑個不停。

於是NBA賽場的身高巔峰對決誕生了。獨行俠隊這邊，博班身高2米21，波爾津吉斯2米21，科勒貝爾2米11。雄鹿隊這邊字母哥2米11，大洛佩斯2米13，小洛佩斯2米13。

最終字母哥輕鬆把罰球罰進，兩個主教練一通操作最後都沒有用上。
