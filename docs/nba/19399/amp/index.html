熱火全場只有一次領先客場負灰熊，巴特勒和莫蘭特精彩對決

天天要聞
2019-12-17

北京時間12月17日9:00，邁阿密熱火隊客場挑戰孟菲斯灰熊隊，熱火目前以19勝7負的戰績排名東部第2，灰熊目前僅9勝17負排名西部第13。熱火在面對灰熊時已經取得5連勝，兩隊在本賽季有過一次交手，10月24日，熱火在賽季揭幕戰中主場120-101大勝灰熊。

熱火有望拿到賽季第20勝，如果邁阿密做到這點，這也將會追平隊史第三快拿到20勝紀錄。熱火在最近9場比賽里贏下7場。灰熊最近狀態有所回暖，他們在近4場取得3場勝利，上一場，灰熊128-111擊敗奇才，莫蘭特和克拉克複出，兩大新秀在比賽中活力四射，灰熊打出57.1%的投籃命中率，創造了賽季新高。

本場比賽的焦點是巴特勒大戰莫蘭特。巴特勒本賽季場均拿到21分6.5個籃板6.8次助攻2.1次搶斷，迄今為止，他已經連續8場比賽拿到至少20分，自從三巨頭2014年解體以來，此前只有2014-15賽季的韋德能連續9場拿到20+，巴特勒只差一場就可以追平隊史近5年連續20+場次紀錄。榜眼秀莫蘭特本賽季場均拿到18.7分3.2個籃板6.4次助攻，是年度最佳新秀的領跑者。

今天NBA聯盟公布了上周東西部最佳球員，熱火的阿德巴約當選東部最佳，阿德巴約上周獲得兩個三雙，一個兩雙，成為熱火歷史上第五位可以在一周之內獲得兩次以上三雙的球員。這是阿德巴約職業生涯第一次獲得周最佳球員，同時也是熱火隊史上第十六位獲得此殊榮的球員。

熱火首發：納恩、巴特勒、羅賓遜、倫納德、阿德巴約；灰熊首發：莫蘭特、狄龍-布魯克斯、克勞德、傑克遜、瓦蘭西尤納斯 。

首節比賽開始，納恩三分投籃不中，巴特勒搶到第一個進攻籃板球；克勞德三分遠投命中，莫蘭特送出第1次助攻；然後羅賓遜三分遠投命中，巴特勒送出第1次助攻。兩個人首節就開始了較量。莫蘭特首節拿到7分5次助攻，隨後下場休息。巴特勒這時才開始發威，首節貢獻4分5次助攻2籃板1搶斷，表現全面。灰熊隊瓦蘭西尤納斯成為熱火隊的噩夢，屢投屢中，首節拿到11分2次蓋帽，率領灰熊以36-32領先熱火。而阿德巴約手感欠佳。

第二節開始，莫蘭特上來就貢獻一次封蓋，瓦蘭西尤納斯空切扣籃命中，拿下第13分；莫蘭特隨後又先後助攻布魯克斯和希爾三分遠投命中，幫助灰熊45-36將比分擴大至9分。在貢獻個人9次助攻後再次下場休息。在灰熊51-40領先10分後，熱火終於用巴特勒替換下奧利尼克上場，開始發動反攻。巴特勒上場後就搶到本場的第3個進攻籃板球，隨後他進攻造成2次犯規，4罰4中拿下本場的第9分。隨著巴特勒最後一次遠投不中，上半場比賽結束。熱火以58-73落後15分。

半場數據情況，熱火隊方面：巴特勒9分5助攻4籃板，阿德巴約9分7籃板3助攻1搶斷，羅賓遜9分，納恩7分2籃板2助攻，希羅12分2助攻，瓊斯10分。

灰熊隊方面：克勞德6分1助攻，傑克遜12分3籃板，瓦蘭西尤納斯17分3籃板1助攻，布魯克斯5分2助攻，莫蘭特7分9助攻，梅爾頓10分2籃板2助攻，克拉克6分2籃板，希爾8分

總體看來，兩隊焦點人物巴特勒和莫蘭特基本上平分秋色，巴特勒搶籃板能力更強，莫蘭特助攻能力更棒。但瓦蘭西尤納斯比阿德巴約發揮得更好，得分幾乎是阿德巴約的2倍。

隨後第三節開始，莫蘭特上來就三分遠投命中，拿到個人第10分；阿德巴約助攻納恩快攻拋投命中，努力追趕比分；緊接著莫蘭特有快攻拋投命中，還以顏色，小將真是擋不住啊！此後，巴特勒送出第6次助攻，羅賓遜三分遠投命中，熱火68-80仍落後12分；瓦蘭西尤納斯本節手感有些涼，幾次投籃不中，罰球線上也2罰不中，但他化身籃板手，連續搶下3個籃板。在第三節還剩5分44秒鐘時，巴特勒突破挑籃命中，拿下本場個人第11分；不久，莫蘭特就突破上籃命中，拿到個人第14分；在一次投三分不中後，莫蘭特又突破上籃命中，拿到16分了，幫助灰熊86-80領先巴特勒違例，防守干擾球了，灰熊以88-80領先；此後，巴特勒進攻造成犯規2罰2中，熱火82-88落後6分。

阿德巴約被判罰違例，防守干擾球後，巴特勒又被判罰了技術犯規。隨著梅爾頓2罰2中，阿德巴約3分不中，第三節比賽結束，灰熊以95-86領先9分。

最後一節，布魯克斯跳投被阿德巴約封蓋，但瓦蘭西尤納斯補籃命中，已經拿到19分9個籃板；希羅三分遠投命中，還以顏色；隨後希羅2罰2中，個人已經拿下22分，率領球隊追趕比分；瓊斯希羅在助攻Jr. 空中接力扣籃命中後，熱火102-104隻差2分；隨後瓊斯Jr. 行進間上籃命中，幫助熱火將比分追平；接著瓊斯Jr. 造成犯規2罰1中 ，熱火105-104首次領先；阿德巴約空切扣籃命中，熱火領先3分，阿德巴約拿到個人第14分。

但灰熊隊並沒有因為落後而慌亂，在莫蘭特三分投籃不中後，克勞德 搶到籃板反手上籃得分，灰熊106-107隻落後一分；隨後 莫蘭特突破反手上籃命中，拿到個人第18分，灰熊108-107再次領先；然後灰熊連續搶到籃板，最終由小傑克遜三分遠投命中，111-107擴大比分；在巴特勒後仰跳投不中後，莫蘭特搶到籃板球。

最終灰熊隊主場118-111戰勝了熱火隊。全場數據情況，灰熊隊方面：莫蘭特20分10助，瓦蘭丘納斯21分10板，傑克遜15分5板，克勞德10分6板，布魯克斯9分2板2助，克拉克14分9板，希爾8分3助。

熱火隊方面：巴特勒25分4板8助，阿德巴約14分13板5助3斷3帽，羅賓遜12分3板，納恩18分2板3助，希羅22分5板3助，德里克-瓊斯15分4斷，萊昂納德3分4板。

灰熊本賽季對於成績基本沒有任何要求，鍛煉新人成為了灰熊當下的首要目標，莫蘭特、布魯克斯、克拉克這些新秀都有不小的潛力可以挖掘。
