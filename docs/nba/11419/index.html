大甩賣！雷霆超市重新開張！4000萬先生離隊倒計時

2019-12-12 1,675條評論 4,270次閱讀 353人點贊

據The Ringer記者Kevin O’Connor報道，多位聯盟消息人士透露，保羅、加里納利、亞當斯以及雷霆隊里的其他老將都是可以被交易的。

這三位也是目前球隊年薪前三高的球員：保羅的合同還剩3年1.24億美元，亞當斯還剩2年5337萬，加里納利則剩一年2260萬。本賽季雷霆總薪資達到1.334億美元，高居聯盟第6；需繳納230萬美元奢侈稅，聯盟第5多。

不過據O』Connor報道，儘管薪資總額超過了奢侈稅線，但由於繳納金額不多，避稅並不是雷霆的重點。雷管甚至願意接手長合同和大合同，以求獲得一些其他有價值的資產。

上一場爆冷擊敗爵士之後，雷霆以11勝12負的戰績與太陽並列排西部第七，距離第六的爵士也只差一場半。這個結果相信很多人都沒想到，交易走威少、喬治後這支球隊不是要擺爛重建嗎，怎麼奔著季後賽去了？

要知道雷霆交易走威少、喬治和格蘭特等人時，換回了未來6年的13個首輪選秀權！球隊當前的目標應該是擺爛，然後獲得高順位新秀重建，眼下卻在衝擊季後賽，普雷斯蒂能不慌嗎！

保羅倒是想去爭冠球隊再拼一把，但他的合同太大了。2021-22賽季，屆時將年滿37歲的他卻要拿4421萬美元的年薪。賽季開始前熱火曾對交易得到炮哥表達出一定程度的興趣，但兩隊最終未能達成一致。

選秀剛結束時，有消息爆出國王願意接手亞當斯，獨行俠和綠軍也對他有過興趣，但最終都不了了之。當時一位聯盟高管表示：「雷霆要價太高，很難接手亞當斯的合同並放棄球員和選秀權。」

同樣來自O’Connor的報道，開拓者預計會追求加里納利！本賽季開拓者總薪資1.46億、繳納奢侈稅2562萬美元均排聯盟第一，但球隊僅以9勝15負排名西部第12。胡德跟腱撕裂後，本就薄弱的鋒線更加捉襟見肘，交易來義大利人或許是個不錯的選擇。

如今看起來最有可能先被交易出去的是加里納利，你們希望看到炮哥和海王加盟哪支球隊呢？
