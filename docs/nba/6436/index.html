137天沒交易！NBA刷新另類歷史紀錄，兩大原因令聯盟風平浪靜

2019-12-01 0條評論 14次閱讀 0人點贊

11月過去了，交易流言不斷出現，樂福、保羅、德羅贊等人都捲入其中，但真正的交易卻仍未發生。自從7月份韋斯布魯克的那筆交易以來，NBA已經137天沒有發生交易了。如果把兩次停擺賽季（1998和2011）排除在外，這已經是自1968年以來沒有交易發生的最長時間了。

簡單來說，8月、9月、10月和11月，聯盟的交易市場都風平浪靜。來看看下面這張圖表，出現停擺的1998和2011賽季，分別有210天和162天沒有出現交易，但這和聯盟因停擺中止所有運作有關。正常的所有賽季里，也就這個賽季最平靜，137天沒交易也打破了上賽季133天沒交易的紀錄。

美媒HOOPSHYPE就此事採訪了一些球隊的高層，看看他們怎樣看待如此平靜的交易市場。「在這個休賽期，很多球員更換了新的球隊，高層目前仍在評估球隊的需求，」一位NBA高管解釋道，「看看多少球星換了球隊吧，我認為這是目前沒有交易的主要原因。」

在這個休賽期，6位入選上賽季最佳陣容的球星更換了球隊，相當於過去4個休賽期的總和，這創造了NBA的紀錄，此前的紀錄是1998年的3位。這還不包括像吉米-巴特勒、安東尼-戴維斯這樣沒有入選上賽季最佳陣容的超級球星。

根據之前的報道，上賽季揭幕戰當天的所有球員里，只有43.6%的人仍在相同的球隊效力。這一比例，也是21世紀以來的最低紀錄。休賽期換隊的不少球員要到這個月中旬開始具備交易資格，NBA的總經理們有可能正在權衡各種方案憋大招。

一位高層揭示了交易市場平靜的另一個原因：雙向合同的出現。他認為，正是因為聯盟引入了雙向合同，讓球隊有了補充戰鬥力的新途徑。

以本賽季的勇士為例，他們深受傷病困擾，雙向合同成為他們應對的關鍵。持雙向合同的兩位球員獲得了不少出場時間，凱-鮑曼出場421分鐘，達米恩-李出場241分鐘。兩人是目前聯盟所有雙向球員里出場時間最多的。

現在聯盟里共有35名球員簽有雙向合同，雖然他們比不上交易帶來的那種巨大變化，但在球隊有急需時，他們可以很快派上用場，而且這種合同不用擔心未來的風險。
