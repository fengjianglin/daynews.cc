丁翁迪41分順局打成逆風局，馬刺再度逆轉籃網

天天要聞
2019-12-21

北京時間12月20日，馬刺下半場發力，在主場翻盤，以118-105擊退了籃網。

得分41的頂翁迪

馬刺（11-16）避免二連敗，下半場得了68分。拉馬庫斯-阿爾德里奇20分10個籃板，德馬爾-德羅贊12分6次助攻，德章泰-穆雷13分、6個籃板和5次助攻。替補出場的帕蒂-米爾斯投中7記三分球，貢獻27分，德里克-懷特13分。

籃網（15-11）二連勝結束。斯賓賽-丁翁迪拿下了41分5次助攻，賈里特-阿倫19分、13個籃板和6次助攻，加雷特-坦普爾10分。

本賽季馬刺是被翻盤最多的球隊，上一場打火箭，他們最多時領先了25分，但最終仍被翻盤。

在此之前，他們連續打了4場加時賽，勝出3場。雖然足夠頑強，但不穩定仍是最大的問題。

籃網仍少了大將歐文，但他們已經習慣了這種局面，上一場逆轉鵜鶘。在歐文缺陣的日子裡，他們11勝5負。

在歷史上，馬刺對籃網的戰績是71勝22負，占絕對優勢。不過如今的時代，已經跟歷史不同了，馬刺今天打得非常艱難，幾度落後兩位數。下半場他們發力，才完成逆轉。

雖然是客場作戰，籃網先聲奪人，無論進攻還是防守都壓對方一頭。首節還有3分53秒時，紐瓦巴兩罰兩中，他們以27-19領先。米爾斯連續命中兩記三分，馬刺的壓力才有所緩解。首節過後，馬刺以27-33落後。

第二節開始沒多久，紐瓦巴再次罰中兩球，籃網以40-30取得兩位數的優勢。馬刺雖然頑強反擊，但在進攻上表現一般，雙方差距一度達到14分。本節還有2分20秒時，丁翁迪近距離投籃命中，他們以56-42領先。馬刺還是靠外線追分，懷特和沃克相繼得手，上半場只以50-56落後。

上半場籃網命中率48.8%，三分球20投5中。丁翁迪拿下了19分。馬刺命中率只有40.4%，三分球17投6中。德羅贊得了10分。

第三節還有2分07秒時，坦普爾命中三分，籃網仍以81-71領先。米爾斯發威，他不但投中三分，還兩次助攻隊友，馬刺以一波12-0結束本節，一舉以83-81超出。他在這波攻擊中得了7分，另5分來自他的助攻。

兩隊回到同一起跑線。第四節過半時，籃網以94-96落後。貝里內利命中三分，馬刺連得6分後，以102-94擴大優勢。籃網近2分鐘未能得分，丁翁迪好不容易連投帶罰拿下3分，但米爾斯馬上又三分命中。

馬刺開始控制局面。米爾斯保持火熱的手感，在比賽還有1分37秒時，他又一次三分命中，馬刺以112-101領先。籃網接連投籃不中，未能再構成威脅，慘遭翻盤。
