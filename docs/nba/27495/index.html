NBA1/3賽季頒獎之進步最快球員：漢姆鶯歌脫胎換骨，西卡二級跳

2019-12-23 862條評論 3,104次閱讀 857人點贊

2019-20賽季已經打了1/3賽程，球隊和球員們也慢慢進入了角色，是時候對賽季6項個人大獎分篇做一次初步展望了。

註：筆者結合了「誰應該拿獎」（純籃球因素）和「誰會拿獎」（媒體投票人傾向）兩大因素，每個獎項選出了三個候選人，但主要的選人標準是前者。由於比賽樣本過小，部分獎項的角逐並不清晰，因此排名不分先後順序。

德馮特-格拉漢姆（夏洛特黃蜂）

上賽季主要數據：場均14.7分鐘4.7分1.4板2.6助0.5斷，回合佔有率16.6%，真實命中率45.9%

本賽季主要數據：場均34.4分鐘19.2分3.7板7.4助0.8斷，回合佔有率25.3%，真實命中率54.7%

今年重回寫手崗位的資深數據專家霍林格最近在文章中表示，格拉漢姆是毫無疑問的「年度最驚喜球員」。不僅因為他進步幅度巨大，而且是完全出乎人們意料。

我們先給大家看看他新秀賽季結束後的球員報告：

格拉漢姆上賽季只有在身前球員有傷病缺勤的情況下才會進入輪換，在後場兩個位置間遊走，有無球均有一定戲份。他初出茅廬就很敢出手三分，以每36分鐘6.4次的頻率位列控衛前半游，但准心可就不敢恭維了。雖然我們總說三分產量是衡量射手水平的重要指標，如果球員命中率只有28.1%，那還是不投為好。

有球在手的時候，格拉漢姆是傳球為先的擋拆選擇。他球風無私有控衛本能，擅長發現防守陣型漏洞，交出了超出回合佔有率水平的助攻數，還很好得平衡了失誤。

受限於矮小體型和普通的運動能力，格拉漢姆大學期間就是糟糕的終結者，進入職業賽場只會攻框更加吃力，籃下頻率和命中率都是控衛後半游水平。一個又不會攻框，跳投也不準的控衛，自然沒法保證效率。格拉漢姆45.9%的真實命中率僅優於另外5名同位置球員，ORPM也是控衛第9差的。

結果本賽季，格拉漢姆不僅延續了良好的傳球本能，投射水平和去年比是一個天一個地，場均9.3次出手三分，命中率超過40%，而且有一大堆持球投，賽季總三分命中數119記僅次於哈登，高居聯盟第二，一夜之間變成高產高效的頂級三分大神！就算霍林格總結的一樣：「沒有人能預見到格拉漢姆能打成這樣。」

帕斯卡爾-西亞卡姆（多倫多猛龍）

上賽季主要數據：場均31.8分鐘16.9分6.9板3.1助0.9斷0.7帽，回合佔有率20.9%，真實命中率62.8%

本賽季主要數據：場均36.6分鐘25.1分8.0板3.6助1.0斷1.0帽，回合佔有率29.3%，真實命中率55.5%

作為這個獎項的衛冕者，西亞卡姆最神奇的是完成了二級跳，再次向這個獎項發起強力衝擊。他的進步也由外部和內部因素共同組成。首先上賽季的老大萊昂納德離隊了，球隊沒有補進進攻手的情況下，開發進攻的坑誰來填？只能內部消化。

而作為球隊重點培養的未來新星，西亞卡姆還真站出來扛住了第一主攻手重任。他的真實命中率確實降了，但相應的回報是球權大漲，大量的單挑+低位，大量的正面持球投三分，要知道上賽季他的三分產量不高，而且幾乎全是吃餅的底角三分。如今的西亞卡姆能參與各種陣地戰戰術，打不出來還有強投解決問題的進攻大核，結合他防守端穩定的貢獻，已經是標準的全明星球員。

布蘭登-英格拉姆（新奧爾良鵜鶘）

上賽季主要數據：場均33.9分鐘18.3分5.1板3.0助0.5斷0.6帽，回合佔有率22.7%，真實命中率55.5%

本賽季主要數據：場均33.8分鐘25.3分7.0板3.8助0.8斷0.8帽，回合佔有率28.9%，真實命中率60.0%

聯盟打完30場左右，只有5名球員能做到28%的回合佔有率+60%的真實命中率，分別是哈登、字母哥、東契奇、喬治和…英格拉姆！

鶯歌生涯前三年在湖人的表現只能用失望形容，作為低效得分手，不管是主攻、傳控還是防守都沒有太多亮眼之處。三年級結束前，鶯歌遇到血栓，賽季結束後還作為濃眉交易的籌碼之一被送去鵜鶘，幾乎沒人能看好他的四年級，新秀合同後簽個中產甚至更低價碼，是完全有可能的。

鬼知道鶯歌完成了自我救贖，也打了所有人的臉，本賽季進步為高產高效的得分手。他最可喜的是大幅提升了三分產量，每36分鐘三分出手從1.9次漲到6.0次，高達38.8%的三分准心，加上罰球命中率從去年的67.5%進步到83.9%，都說明這孩子的投籃可能真練出來了。

ESPN不久前剛公布了新賽季的RPM（真實正負值），這也是業界最權威的高階數據之一。英格拉姆目前的RPM已經是小前鋒前10、聯盟前40的存在，即使賽季初很多數據會出現異常值，導致球員的RPM失真，這份答卷已經再完美不過了。
