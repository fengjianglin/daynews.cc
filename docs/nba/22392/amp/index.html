這六大全明星球員已經退役，合同卻沒結束！尼克斯還欠諾阿1929萬

天天要聞
2019-12-19

有一些球員拿了大合同後，會被球隊四處交易，然後就有球隊買斷他們，他們雖然退役了，但合同還沒有結束，在家喝茶都能拿錢。

蒙塔-埃利斯曾經在勇士也是聯盟頂級的得分後衛之一，他在2015年休賽期4年4400萬美元加盟了步行者，本來要到2019年夏天才到期，但是步行者在2017年夏天就買斷了他。這個賽季步行者還要支付給蒙塔225萬美元，一直到21-22賽季才結束。

德隆-威廉姆斯曾經也是聯盟頂級的得分後衛，2011年2月23日，籃網通過交易得到了德隆，那個時候他還是處在巔峰狀態，2012年休賽期籃網5年9800萬美元續約了德隆，2012年夏天生效，2017年夏天到期，但是在2015年夏天，籃網買斷了德隆，導致籃網每個賽季要付給德隆547萬美元，這個賽季是最後一個賽季。

2013年休賽期，約什-史密斯以一份4年5400萬美元的合同加盟了活塞。打完一個賽季，2014年12月23日，活塞裁掉約什，范甘迪給出的解釋是，史密斯和門羅、德拉蒙德不兼容，裁掉他是給年輕球員騰出位置。史密斯本次被裁掉過後，他和活塞的合同還剩兩年，價值2600萬美元，這個賽季約什還可以從活塞領到533萬美元。

羅爾-鄧入選過兩次全明星，也是頂級的3D球員，2016年夏天4年7200萬美元加盟了湖人，2018年9月2日，湖人隊官方宣布與羅爾-鄧達成買斷協議，羅爾-鄧主動放棄了730萬美元，現在湖人每個賽季要給羅爾-鄧500萬美元，直到21-22賽季才結束。

2016年休賽期，大加索爾2年3000萬美元加盟了馬刺，第二年為球員選項，到了2017年休賽期加索爾跳出了合同選項，重新與馬刺簽訂了一份3年4800萬美元的合同，今年的3月份，馬刺買斷了加索爾，這個賽季馬刺還要給加索爾507萬美元。

諾阿曾經在公牛的時候是聯盟最好的中鋒之一，2016年夏天尼克斯給了諾阿一份4年7200萬美元的合同，諾阿在尼克斯飽受傷病困擾，2018年10月14日，尼克斯官方宣布買斷諾阿，諾阿現在每個賽季在尼克斯可以得到643萬美元，直到21-22賽季才結束，一共是1929萬美元。
