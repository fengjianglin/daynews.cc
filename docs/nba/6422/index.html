雄鹿狂勝41分11連勝！反超湖人10連勝，喬丹為毒藥眼光買單

2019-12-01 0條評論 19次閱讀 0人點贊

雄鹿取得10連勝創紀錄，上一次取得常規賽10連勝還要追溯到1985-86賽季，當時雄鹿在1985年2月6日-2月28日間連贏10場比賽。北京時間2019年12月1日，雄鹿主場以137-96狂勝黃蜂迎來了11連勝，字母哥拿下26分9籃板。

這是一場一邊倒的比賽，字母哥在半場就拿到21分6籃板2助攻1搶斷。這場比賽雄鹿可謂是內外開花，他們打出了將近6成的命中率，此外三節不到就投中了14個三分球，雄鹿還通過防守壓制對手四節開始的時候得分不到80分。

字母哥繼續打出了超級的表現，上一場他們戰勝騎士的比賽，字母哥就拿下33分12籃板2助攻2搶斷2蓋帽。本賽季字母哥的數據已經高光到很多人無法追逐的地步，東契奇的場均三雙非常出色，哈登的場均得分也很勁爆，湖人那邊還有超級雙星，但是字母哥這種真正意義的單核托起整個球隊的表現，放眼聯盟無人出其左右。

在打黃蜂之前，字母哥的所有主要數據全部殺進了聯盟的前20位，得到31.1分13.7籃板6.2助攻1.50搶斷1.44蓋帽，投籃命中率達到了56%，雄鹿打出了30年來最好的連勝紀錄，而字母哥朝著自己衛冕MVP之旅正在勇往直前，就現在這個態勢而言，字母哥比詹姆斯更有資格和希望拿下最有價值球員。

當然湖人隊在本賽季的連勝也沒有停滯，他們已經打出了10連勝，也是十年來最好的戰績。現在很多人都在談及雄鹿總決賽大戰湖人的事兒，雄鹿現在的五大陣容和湖人還有很多相似，都有能夠投籃的中鋒，3D位置人才濟濟，但是湖人隊這邊有兩個超級巨星，這一點可能是雄鹿無法相比的，此外字母哥的投籃能力還是被質疑，本賽季他的三分球依然是天愁地慘。

​ 而本賽季黃蜂交易了沃克之後成為了徹頭徹尾的弱旅，他們在東部已經是第三集團，羅齊爾和沃克無法相提並論，巴圖姆作為億元先生已經淪為了替補，喬丹連續為了自己毒藥的眼光和操作買單。最終雄鹿大勝黃蜂迎來了11連勝，超過湖人成為本賽季連勝最多的球隊。
