火箭3連勝？國王2大核心缺陣，威少或連續4場三雙，德帥點評哈登

2019-12-09 0條評論 17次閱讀 0人點贊

北京時間12月10號，休斯頓火箭坐鎮主場，迎接薩拉門托國王的挑戰。此役之前，火箭擊敗太陽，收穫一波兩連勝。國王也在今天擊敗強大的獨行俠，大大提升了球隊士氣。近期，國王的狀態並不穩定，5場比賽2勝3負，也讓他們在西部第八的爭奪中，處於下風。

明天的比賽，兩隊都想取勝。就在今天，掘金不敵籃網，讓火箭躺進西部前4。要想在季後賽處於有利位置，火箭就要力保前4。對於國王來說，一場比賽的勝負，在西部的排名都會發生巨大變化。目前，國王隊以9勝13負，位居西部第10 ，距離第8的太陽僅有1個勝場差。

國王2大核心缺陣

國王雖然今天擊敗了獨行俠，但是背靠背客場挑戰火箭，壓力著實不小。上個賽季，國王在一幫新生軍的帶領之下，差一點闖進季後賽，最終以西部第9遺憾出局。這個賽季的他們，本想大展身手，全力衝擊，可惜傷病卻阻擋了他們前進的步伐。

國王后場核心福克斯，因為左腳踝扭傷，已經缺席近一個月時間。明天對陣火箭的比賽，依然高掛免戰牌。本賽季，福克斯出戰9場，場均貢獻18.2分4籃板6.9助攻，三分命中率33.3%。

受傷的還有內線核心巴格利，他因為大拇指骨折，這個賽季僅出戰1場比賽，貢獻14分10籃板。國王2大核心繼續缺陣，這也是賽季開局至今戰績不好的主要原因。

威少或連續4場3雙

火箭近3場比賽，取得2上1負的戰績，威少的表現可謂是冰火兩重天。對陣馬刺，他在第四節後半段，「幫助」馬刺連續追分，最終火箭被拖入加時賽遺憾落敗。賽後，外界對威少的質疑聲不絕於耳。但是，對陣猛龍的比賽中，他再次上演「拯救之戰」，要不是哈登上場及時制止，火箭很有可能迎來2連敗。

也許是兩場比賽的經歷，讓威少意識到自己的錯誤。在對陣太陽的比賽中，他變得非常理智，不再盲目投3分，專註搶籃板和助攻。第四節，兩次關鍵的2+1，讓火箭徹底鎖定勝局。最近三場比賽，威少全部得到3雙數據。明天對陣國王，只要他保持理智，很有可能連續4場3雙。

德帥點評哈登

上一場火箭贏下太陽，哈登的投籃效率非常低。 27中8，3分球17中3，罰球18中15， 得到34分6籃板4助攻。雖然哈登在三節狀態很差，但是在關鍵的第四節，單節得到18分。賽後，德安東尼點評哈登的表現：哈登不必做所有的事情。但是，他會在關鍵時刻做一些事情，這就是詹姆斯，他知道該怎麼做。由此可見，德帥對於哈登是十分信任的。

上一場比賽，威登組合其實做了一個很好的示範。前三節，威少帶領火箭去打，哈登作為佯攻手吸引對手的注意力。在關鍵的第四節，哈登則作為主攻手，威少作為佯攻手，這樣會讓兩個人發揮最大的作用。

火箭3連勝？國王2大核心缺陣，威少或連續4場三雙，德帥點評哈登。你們認為，火箭明天能夠戰勝國王嗎？威少是否能夠再次拿下3雙？
