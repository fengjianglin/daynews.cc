火箭才打23場比賽，德安東尼就把這四人踢出輪換，743萬白花了

天天要聞
2019-12-11

截止到北京時間12月11日，火箭隊打了23場比賽，戰績是15勝8負，排在西部第四。我們看一下火箭目前攻防兩端的數據，每百回合得到113.3分，排在聯盟第三；每百回合失108.5分，排在聯盟第十五；可以看出火箭是一支靠進攻的球隊。

哈登場均得到38分，是聯盟得分王，與上個賽季相比，哈登的三分球命中率下降了，這個賽季只有33.8%。威少目前的狀態還算不錯，場均22.4分8.1籃板7.5助攻，防守積極，唯一的不足是三分球命中率太低了，只有22.9%。有哈登和威少在，火箭隊打進季後賽肯定沒有問題，但如何在季後賽才能走遠呢？這就要看德安東尼的執教水平。

休賽期，火箭隊簽下了麥克勒摩、克萊蒙斯、塞弗洛沙、錢德勒等新人，現在來看，麥克勒摩是最管用的，最近五場比賽，他分別得到24分13籃板、11分、28分、27分、12分，戈登受傷後，麥克勒摩頂替了他的位置。不過現在已經有四人跌出了德安東尼的輪換陣容，那就是哈爾滕施泰因、錢德勒、塞弗洛沙、克萊蒙斯，這四個人的薪水加起來為743萬美元。

哈爾滕施泰因曾經和周琦的角色差不多，今年夏天為了備戰新賽季，還拒絕為德國隊出戰籃球世界盃，但是新賽季並沒有得到很多機會，最近四場比賽，只打了一場，上了2分鐘。在上個月，哈爾滕施泰因還有單場6分16籃板的表現，後來不知道德安東尼為什麼不用了，前段時間剛從發展聯盟回來，估計又要去了。

火箭隊一直想給卡佩拉找一個替補，所以簽下了老將泰森-錢德勒，賽季至今錢德勒沒有什麼貢獻，已經連續三場比賽沒有上場了。現在卡佩拉沒有替補，內內受傷、哈爾滕施泰因、錢德勒不讓上。

火箭隊簽下塞弗洛沙是想補充側翼的防守，賽季初還能得到10分鐘的上場時間，最近也不讓他打了，除非是垃圾時間。克萊蒙斯身高只有1米75，之前有過單場16分、19分的表現，也慢慢跌出輪換陣容。

目前，火箭隊的正常輪換是：哈登、威少、豪斯、麥克勒摩、卡佩拉、塔克、里弗斯、克拉克，戈登受傷了；到了季後賽，克拉克估計也不用了，基本就是八個人輪換。
