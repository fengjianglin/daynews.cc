歷史最年輕25+7+37.3%三分球命中率！英格拉姆終於成為頂級得分手

2019-12-21 88條評論 1,169次閱讀 204人點贊

英格拉姆正在打出職業生涯最佳表現，場均25.3分聯盟第13位，高於保羅喬治、西亞卡姆、德文布克等一票球星，場均得分比他更高的球員們，已經是一個又一個璀璨奪目的名字，現在，英格拉姆也躋身此列。

然而，與哈登、阿德托昆博或者東契奇不同的是，英格拉姆的職業生涯還從未經歷過一整個穩定有序的賽季，來到鵜鶘的第一年，他只是從之前的泥潭裡抬起腿，跨入了下一個泥潭。

2020賽季的鵜鶘有多糟糕？他們延續了一貫的快節奏，是本賽季節奏第7快的球隊。但是，他們的百回合107.1隻排在聯盟第20位，與此同時百回合失分高達113.9排在聯盟倒數第四，百回合凈勝分-6.8排在聯盟倒數第五；

60.4%的助攻率聯盟倒數第13位，絲毫沒有一支進攻強隊的影子；隨之而來的是聯盟中游水準的52.5%有效命中率以及聯盟倒數第十一位的55.3%真實命中率。

——整個賽季2829個半場進攻回合里，每回合只能得到0.929分排在聯盟第22位；

——標誌著進攻空間最重要的spot up（定點接球終結）回合，763個回合每回合只能得到0.99分排在聯盟第19位；

——最重要的陣地進攻方式持球呼叫擋拆一共607個回合，每回合得到0.761分，聯盟倒數第二，太陽以每回合0.747分成功墊底；

——球隊進攻表現唯一一項能進聯盟前十的選項是擋拆掩護後接球終結，也就是吃餅。

空間？沒有。擋拆發起？不行。面筐單打硬解法？一般。進攻戰術成功率？場均傳球314.3次聯盟第二多，傳球助攻率7.9%聯盟倒數第八。進攻時跑動距離10.16英里聯盟第三，平均速度4.88聯盟第一；防守時跑動距離8.6英里聯盟第二，平均速度3.87隻能排在聯盟中游，鵜鶘就是這樣一支跑的挺快的弱隊。

當身處一支進攻混亂、環境不佳的球隊中，球員會受到很大影響；同樣，如果球隊進攻有序，球員自然能從中收益，小則提升進攻效率，大則全面大幅度提升進攻數據。布蘭登英格拉姆當然是前者，身處在泥潭之中。

早在18賽季時，英格拉姆曾在鮑爾缺陣的1個月時間裡場均19分5籃板6助攻，真實命中率62%。整個賽季場均16+5+4，只二年級而已。他還曾經被推到控球後衛的位置，難以捉摸的定位並沒有讓他停滯不前；

19賽季，與勒布朗搭檔的一年，英格拉姆加強了無球選擇，空切與中距離比以往更加果決。並且隨著賽季進行，他打的越來越好。在3月份宣布賽季報銷之前的16場球，場均22.8分命中率52%三分球命中率41.7%——已經瞥見未來曙光，只可惜還沒怎麼開始，就被傷病喊停；

布蘭登英格拉姆正在度過非同尋常的2020賽季，在12月19日的比賽中，英格拉姆全場34分6籃板5助攻3搶斷1封蓋，幾乎以個人能力正面一對一打爆了考文頓。這樣的表現在整個賽季已不少見。如果他保持當前表現至賽季末，那麼英格拉姆將成為完成場均25+7且三分命中率至少在37.3%的最年輕球員（22歲）。

近三年以來能夠達成這樣成就的球員分別是，19賽季的倫納德（26.6分7.3籃板37.1%三分球命中率）、19賽季的保羅喬治（28分8.2籃板38.6%三分球命中率）、20賽季的西亞卡姆（25.1分8籃板39.2%三分球命中率）、20賽季的唐斯（26.5分11.7籃板41.8%三分球命中率）和20賽季的英格拉姆。

鵜鶘環境惡劣並沒有阻止英格拉姆進化，他於殺戮之中盛放,一如黎明中的花朵，屬於他的未來終於鋪開畫卷：

——英格拉姆本季使用率29.1%聯盟第二十位，真實命中率60%在前二十中高居前五，一個高產高效的頂級得分手。重要的是，英格拉姆每回合觸球時間僅4秒，每次觸球可以得到0.378分，是個不太占球權但每次接球都有足夠威脅的優質進攻球員；

——持球方面，面筐單打共97個回合，每回合得到1.01分，好於82%的球員；持球呼叫擋拆123個回合，每回合0.951分，好於75%的球員；鶯歌的持球產量不算多，但效率極高。運球急停跳投場均5.8次，命中率44.8%；對比16季的杜蘭特場均8.2次，命中率45.8%，效率相差無多；

英格拉姆的持球進攻發起基本在中距離，與杜蘭特很像。中遠距離位置，英格拉姆共計出手127次，命中率48%非常高效，而受助攻率只有10.2%，幾乎是自己持球終結的。他的臂展長出手點極高，加上本賽季愈發穩定的腳步和收球，讓他在中距離進攻上頗有杜蘭特的韻味。

不過，杜蘭特在運球擺脫方面較英格拉姆好上不少。這是英格拉姆在接下來一個階段需要繼續提升的地方。以及三分線外英格拉姆受助功率高達91.2%，自主進攻終結的次數整個賽季僅5次而已，缺乏持球威脅。在這個時代，三分線外的發起能力決定了得分手的產量和效率上線，杜蘭特19賽季三分線受助功率就只有58.7%。

——無球方面，定點接球終結128個回合，每回合得到1.18分，好於82%的球員；接球投籃共計115個回合，每回合得到1.209分，好於72%的球員；對比19季的杜蘭特，定點接球終結每回合得到1.35分，好於98%的球員；接球就投每回合得到1.183分，好於80%的球員。

英格拉姆距離杜蘭特的無球效率還有一定距離，但勇士的進攻環境又比鵜鶘好上不少。同時，英格拉姆他在兩側底角分別投出了45%和45.8%的三分球命中率，是個聯盟頂級的無球人了；

這就是目前為止，我們所看見的英格拉姆。他完全適應了NBA級別的對抗，雖然仍不見掛肉，但核心力量是肉眼可見的提升。本季場均突破次數13.1次，突破後出手6.5次命中率51.9%，已經是聯盟前列。與他接近的球員是科懷倫納德，場均突破13.2次，突破後出手6次命中率49.2%。倫納德在比賽時是充滿對抗性的，突破製造罰球場均2.3次。而英格拉姆在製造犯規獲得罰球的數據，已經與倫納德完全持平。

在RPM中，英格拉姆四年以來終於ORPM（進攻真實正負值）為正數且高達1.8，排在聯盟第40位，這對於一個助攻失誤比僅1.26的球員來說已屬不易。

無論從什麼角度看，英格拉姆都是聯盟里最頂級的得分手，在他22歲這年——這麼說絕非魯莽之詞——屬於英格拉姆的偉大職業生涯之旅，終於要開始了。
