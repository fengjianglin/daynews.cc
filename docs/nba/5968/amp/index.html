先得分算我輸？步行者尼克斯讓分大戰幾乎突破下限

天天要聞
2019-12-08

北京時間12月8日，在NBA一場常規賽中，步行者1分險勝尼克斯，而這兩隊最後5分鐘的表現簡直是亮瞎了諸位看官的硬化氪金狗眼！

自比賽還有5分05秒結束，米切爾-羅賓遜扣籃得分將分差追到只差4分時，雙方就陷入了一種「你得分你就是孫zei！」的心理，屢次挑戰對方的防守下限。

在這5分鐘里，雙方一共6次嘗試出手3分全部打鐵，而雙方一共貢獻了6次蓋帽，其中本場比賽拿下5次蓋帽的邁爾斯-特納一人就貢獻了4次！

最終，只有2名球員在最後5分05秒內得分，而且還都是尼克斯的球員！其中莫里斯半截籃命中，蘭德爾最後0.1秒搶到進攻籃板二次進攻製造犯規，然而就當尼克斯有希望將比分扳平拖入加時賽時，罰籃命中率不到70%的蘭德爾「不負眾望」地將球罰丟……結束了這一場拒絕得分的鬧劇。

可能在賽後會有一些人認為他們這麼打是不是有什麼陰謀之類的言論，但是只要是看過最後這5分鐘的人，相信你們一定會得出真的是這兩支球隊能力太差的結論……（陳川）
