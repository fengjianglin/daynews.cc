集齊5大NBA球星！最強加拿大男籃出爐，同組中國男籃怎麼贏？

天天要聞
2019-11-28

昨天，東京奧運會男籃落選賽抽籤結果公布，中國男籃與希臘、加拿大同處一個小組。希臘由字母哥領軍，今年世界盃因凈勝分不足無緣八強，通過落選賽進軍奧運會是志在必得的，中國隊在他們面前毫無勝算。由於多名主力未隨隊出征，加拿大在世界盃上沒有展現出應有的競爭力，但在過去一周里，已有5位NBA球員表態將為國效力，他們分別是穆雷、亞歷山大、沃克、布里塞特、巴雷特。

穆雷於2016年被掘金以首輪第七順位選中，從替補打起，如今已經成為球隊的核心。本賽季，穆雷場均可得18.8分4.6籃板4.8助攻，是掘金的頭號得分手。今年夏天，穆雷簽下一份5年1.7億美元的頂薪長約。穆雷是第一個表態參加落選賽的加拿大球員，在他的號召下，其他年輕球員也做出了同樣的決定。

雷霆老大是保羅，但戰術地位更高的是亞歷山大，威少離開後，球隊希望他能成長為頭牌球星。菜鳥賽季在快船，亞歷山大已經展現了不俗的天賦，本賽季擁有無限開火權後，個人數據井噴，達到19.3分5.3籃板3助攻。不出意外，亞歷山大將和穆雷搭檔後場，這對組合放在NBA都不容小覷，到國際賽場想必也能大放異彩。

鵜鶘新人沃克是首輪17號秀，今年21歲的他，目前還是一名替補，每場打14.4分鐘，可得6.4分2.3籃板2.3助攻。沃克身高1.96米，司職後衛，可做穆雷和亞歷山大的替補。猛龍球員布里塞特是一位落選秀，在納斯陣下，他還沒獲得穩定的出場時間。身高2.01米的布里塞特司職前鋒，大學期間場均可得13.7分8.2籃板，有望成為國家隊的重要輪換球員。

巴雷特是今年的的探花，身高達到1.98米，可打得分後衛和小前鋒。本賽季他的出場時間為33.4分鐘，可得15.1分5.6籃板3.6助攻。大學時期的巴雷特，是全美最火爆的超新星之一，雖然有人質疑他，可事實證明，尼克斯用第三順位選他是值得的。在國家隊，巴雷特或許打不上首發，但他一定能做出自己的貢獻。

NBA效力的加拿大球員遠不止這些，出名的還有森林狼的維金斯、熱火隊的奧利尼克、騎士隊的特雷斯坦·湯普森、獨行俠隊的德懷特·鮑威爾、國王隊的約瑟夫、灰熊隊的狄龍·布魯克斯、馬刺隊的萊爾斯。如果這些人中再有5名球員願意加入國家隊，那加拿大隊的天賦將提升到第一檔，可以和塞爾維亞、澳大利亞、巴西這些強隊匹敵。

當然，歷史已經證明，只靠堆砌天賦是不會走太遠的。納斯是一名優秀的主教練，這在NBA已經得到印證，相信他有能力把這些球員凝聚到一起。至於中國隊，能和歐美頂級強隊交手，是很好的鍛煉機會，要想擊敗他們，難於登天。
