籃球之神因它成笑料，這輩子恐怕都難忘掉

2019-12-23 735條評論 2,362次閱讀 1290人點贊

被稱為籃球之神的——邁克爾-喬丹，職業生涯留下了無數的輝煌紀錄，各大記錄榜單也都是隨處可見，真正的數據之王，不過大神也有失手的時候，在其職業生涯中就有一件讓他臉面丟盡，成為日後粉絲們經常拿出來嘲諷的事情。

故事發生在89-90賽季，此時的喬丹還沒有拿到一座冠軍獎盃，但是爆炸的數據和能力已經讓喬丹成為當紅炸子雞，當時的影響力不弱於現在的詹姆斯。

三分球一直都不是喬丹的強項，但是偏巧不巧的是，這個賽季喬丹卻投出了職業生涯最高的三分命中率37.6%，不知道是一時玩性大氣，還是由於賽季命中率，心理膨脹了，他居然參加了這一屆全明星的三分大賽。

這屆三分大賽中有歷史第一xx王拉里-伯德，神射手雷吉-米勒和馬克-普萊斯，不過喬丹還是這屆三分大賽最為耀眼的明星，比賽開始前，他還和伯德打起了嘴炮，幻想著拿完扣籃大賽冠軍，再來個三分大賽冠軍就完美了。

難而夢想總是美好的，現實卻是殘酷的。比賽開始後，喬丹一通操作，一頓打鐵後，全場留下了歷史第一（倒數第一）的成績，全場僅僅拿到5分，成為了那一屆最大的笑柄，不過這個成績在2011年的全明星賽中差一點被杜蘭特的6分打破，可惜最終超出1分，這個記錄仍由歷史第一人喬丹獨享。
