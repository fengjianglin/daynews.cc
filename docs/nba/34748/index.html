中投王，到底是漢密爾頓還是阿蘭？放在當今聯盟，兩人依舊吃香

2019-12-25 1,812條評論 2,867次閱讀 1550人點贊

對於籃球愛好者來說，擁有一手精準的投籃才是王道。

NBA不乏投籃高手，有如科比、甜瓜、雷-阿倫以及水花兄弟，後輩代表還有如雨後春筍般冒出來的吹楊，波格丹諾維奇，富尼耶等人。但在這裡卻突然想到了，當年NBA里中投最準的男人。

一位是中投教科書，一位是面具俠。但他們倆都有過同一個外號：中投王。

漢密爾頓，因其犀利的跑動後中距離接球跳投被稱為「中投王」。

面具俠雖然持球進攻方面稍顯遜色，不過他卻具備用之不竭的體力，以及頂級的無球跑動能力。當年的漢密爾頓被譽為是阿蘭-休斯敦的接班人，據稱當年有一次帶著狗出去跑步訓練，結果狗累趴了，他卻一點事都沒有。這是人乾的事情嗎？

他也是最讓科比頭疼的對手。蝸殼在防守漢密爾頓的時候，幾乎要跟著他滿場跑。比盧普斯一度懷疑自己的隊友是否患有先天性跑動綜合征。這全場跑下來，不去參加個長跑什麼的，都覺得有些可惜了。

漢密爾頓在職業生涯一共打了921場常規賽，場均出場32.1分鐘，得到17.1分，3.4助攻和3.1籃板，投籃命中率為44.9%。

阿蘭-休斯頓被稱為中投王一個是因為他的命中率高，而且都是在中投範圍內，二就是因為他標準的投籃動作也被譽為教科書式投籃。據說阿蘭的中投比保羅還穩。

阿蘭休斯頓職業生涯共出戰839場比賽，場均17.3分2.9籃板2.4助攻。NBA職業生涯投籃命中率為44.4%，三分球命中率為40.2%。名宿，respect！

在當今中鋒都投三分的年代，他們這樣的球員適合如今的時代嗎？且參考保羅或者湯普森。無球跑動可以給球隊帶來大量的機會，其次只要將出手距離往外擴到三分線仍有一席之地，這典型的優質3D球員，還別說，阿蘭-休斯頓和漢密爾頓的防守也還真不錯，即便將兩人放在當下聯盟，也會有很多球隊爭搶。

單論生涯投籃數據，他們兩個人都不相上下。無論其中哪一個出來都是名副其實的中投靚仔，都配得上中投王。

這些名宿就像是當年聯盟的縮影，堅韌、穩定並且從來沒有停止過對勝利的追求，你分明知道他要做什麼，但就是無法防住他的中投，對位的那種失落，或許只有切身體驗過才知道，原來無解的得分方式，還有這一種。

任何人對位都希望他們能夠因為疲憊停下自己的腳步，而當他們有一天終於停下時，已成為傳奇。

新球員們的傳奇還在繼續，只是突然懷念，當年看球的日子。

話說，你的投籃手勢又是第幾種呢？
