濃眉因傷缺陣 霍華德10投全中20+6夢回巔峰 老詹准三雙湖人惜敗

天天要聞
2019-12-18

北京時間18日8點，洛杉磯湖人做客印第安納對抗步行者。濃眉因傷缺陣，杜德利頂替首發。詹姆斯只能單核帶隊對抗東部第六防守球隊步行者，兩隊全場打的相當焦灼，詹姆斯半場得到13分7板6助，麥基和霍華德都很高效同樣拿到8分，霍華德投籃全中，成為上半場「吃餅俠」，夢回巔峰。步行者這邊，首發打得不算太好，替補上場後激發了球隊的激情。薩博尼斯半場拿到15分6板，特納11分4板。半場52-50湖人領先2分進入中場休息。

詹霍組合

第三節，步行者打得非常頑強，詹姆斯表現一般，三節戰罷，步行者78-76反過來領先湖人隊2分。湖人替補發揮神勇，霍華德依然高效，到第四節一直還保持百發百中的狀態。

小薩對抗老詹

第四節最後將近3分鐘，兩隊打成100平，步行者依靠罰球領先2分。之後老詹霸王硬上弓造成犯規，兩罰全中，打成102平。湖人關鍵時刻失准，布魯格登命中關鍵2分，36秒領先湖人。湖人馬上暫停。暫停歸來，詹姆斯三分不中，湖人進攻受阻未得分，最後11.7秒步行者球權，小薩最後兩罰一中，最終隆多三分不中，湖人客場連勝終止，三分惜敗步行者。湖人最終還是敗在了罰球上。可惜了魔獸的英勇表現。

全場比賽湖人詹姆斯准三雙20+9+9，霍華德10投10中拿到20+6，布拉德利13+3+2，卡魯索11+4+3。步行者小薩26+10+4，特納16+7。
