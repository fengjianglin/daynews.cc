詹姆斯談卡特：他可能會被認為是史上最佳實戰扣將

2019-12-16 73條評論 4,682次閱讀 1340人點贊

虎撲12月16日訊 湖人今日101-96客場戰勝老鷹，迎來7連勝。賽後，湖人前鋒勒布朗-詹姆斯接受了採訪。

談到與老鷹的42歲老將文斯-卡特最後一次常規賽交手，詹姆斯說：「我們在常規賽和季後賽都有過多次交手。我認為他對他曾效力過的所有球隊都意義重大，特別是他的猛龍歲月，他在新澤西（籃網）打球的那幾年也很棒，還有許多，比如達拉斯和這裡等。他可能會被認為是史上最佳實戰扣將，後來隨著運動能力下降，他扣得不多了，但他在下滑之後開始投三分。我想到了他的職業精神，他總是保持好身材和心理狀態，征戰了這麼多年，比賽見證了他的輝煌。」

本賽季是卡特個人職業生涯的第22個賽季，也是詹姆斯職業生涯的第17個賽季。

此前，卡特稱自己將在本賽季結束後退役。若老鷹未能打進總決賽，這將是卡特與詹姆斯的最後一次對役。

本場比賽，詹姆斯上場37分鐘，21投12中得到32分13籃板7助攻3蓋帽1搶斷；卡特上場21分鐘，10投3中得到7分1籃板2助攻1搶斷。
