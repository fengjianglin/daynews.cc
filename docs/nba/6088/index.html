明天比賽開拓者有多大概率能戰勝湖人

2019-12-06 0條評論 15次閱讀 0人點贊

大家好，我是小蝦，北京時間12月7日湖人將客場挑戰波特蘭。在小蝦認為這場比賽開拓者想取勝比較困難，畢竟湖人打打西部第九的開拓者還是綽綽有餘的，您看我分析：

開拓者是全聯盟平均每場失分第七高的球隊

截止至北京時間12月6日，開拓者目前以平均場均失分114.8分居聯盟失分榜第七位，這和開拓者糟糕的防守有著很大的關係，目前開拓者球隊里防守最好的是24歲的鋒衛搖擺人海佐尼亞，但是也排在全聯盟球員里的第126位。要知道湖人首發的進攻效率是極高的，詹姆斯和戴維斯的擋拆就可以在開拓者內線輕鬆得分了，所以防守是開拓者必須要做的一個方面。

湖人目前場均失分全聯盟第二低

新賽季改頭換面的湖人是以防守優異著稱的，尤其是恐怖的內線，場均可以送出7.4次的恐怖封蓋（蓋帽場均全聯盟第一），與此同時內線優勢帶來的連鎖反應減輕外線的防守壓力，即使布拉德利這樣防守悍將傷停情況下湖人還是有全聯盟前幾的防守效率，目前場均失分全聯盟第二低。

開拓者助攻數倒數第二

開拓者全隊的場均助攻數在全聯盟倒數第二，平均下來一場球大概20.3次助攻，而湖人這邊場均26.5次。開拓者球隊風格比較偏向於單打，如果遭遇到強防守那很有可能導致被對手打出反擊小高潮。所以說這一點也是開拓者目前來說比較欠缺的。

綜合實力來看

開拓者目前以9勝13負的戰績排在西部第九位，開拓者的優勢在於擁有後場雙槍——利拉德和麥科勒姆。二人場均可以貢獻26.6分和22.4分，這也是是目前開拓者唯一比湖人具有優勢的地方。同時本賽季安東尼的加入也讓球隊擁有了第三得分點，安東尼場均可以貢獻16.9分5.9板1.8個助攻。而湖人這邊主要得分點都在鋒線，詹姆斯和戴維斯場均可以得分26分和25.4分，而湖人的首發後衛得分比較少。如果開拓者需要贏下比賽的話，那就是雙後衛開局狀態爆炸而且沒有被波普和格林防住，同時安東尼也能夠站出來得分。

總結：

開拓者想要贏湖人的話比較困難，湖人需要拿下比賽的話只要攻防端穩定發揮即可。開拓者如果贏湖人的話，需要三個得分點都站出來。利拉德這個小夥子還是很能創造奇蹟的，還是看明天的比賽吧，我們拭目以待。
