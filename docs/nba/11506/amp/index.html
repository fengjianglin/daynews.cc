步行者超新星被完虐，二輪秀狂砍26＋8，步行者要做出改變了

天天要聞
2019-12-10

在今天步行者隊和快船隊的比賽當中，步行者再次遭到了一場慘敗。在這一次的慘敗過程當中，球隊的新星特納表現非常的糟糕，全場比賽僅僅拿下了五分。值得注意的是，最近特納的個人狀態一直都不是很好，已經有很多場比賽的得分不超過兩位數，他究竟是遇到了什麼樣的難題呢？

其實他那的個人技術風格是非常的符合當下的小球聯盟，因為他擁有非常出色的護框技術，除此之外他還有一首非常漂亮的三分遠投。目前為止，他場均2.5個蓋帽的數據仍然是名列聯盟前茅的。但是就是這樣一位球員，為什麼會在新賽季打得如此的掙扎?

其實原因也是非常的明顯的，就是因為他的個人技術缺陷。由於特納早早地就把自己看作成一個空間型的四號位，因此他就極度地依賴於自己外線的手感，再加上教練團隊的戰術安排，這就使得他逐漸的減少了內線進攻。

雖然教練團隊這樣的安排能夠提升整個球隊的進攻效率，當然這也會極大地限制特納的得分效率，所以到目前為止特納的場均得分才僅僅是11分。一旦特納沒有找到外線手感，那麼這一場比賽很有可能就會涼涼，比如在和快船隊的這一次比賽當中，他僅僅是只拿下了五分，他只有五次的投籃機會但是僅僅命中了一球，這其實就是特納在這個賽季的進攻縮影。

除此之外在和快船隊的比賽當中，特納的防守缺陷也被快船隊無限的放大。雖然他是一個非常優秀的內線護框者，但是他的一對一防守是非常的差的，因為他沒有足夠的身體力量以及腳部橫移速度。哈雷爾就抓住了特納的這個缺陷，不斷的利用自己的身體和速度優勢攻擊籃筐，從而在這場比賽拿下了26+8的優秀數據，可以說哈雷爾在攻防兩端都打爆了特納。
