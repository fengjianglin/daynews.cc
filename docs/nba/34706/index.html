這隊12月還沒贏過球！11連敗仍不是盡頭？主帥：球隊一片恐慌

2019-12-25 1,681條評論 4,997次閱讀 1097人點贊

森林狼已經11連敗了，目前，他們以10勝19負的戰績排名西部第13位。令人瞠目的是12月已經臨近結束，而森林狼在這個月里竟然還沒贏過球。北京時間12月25日，森林狼主帥小桑德斯承認，球隊現在處於一片恐慌之中。

森林狼上一次贏球，已經要追溯到北京時間11月28日，他們在客場113-101擊敗馬刺，當時的戰績是10勝8負。那場比賽後，森林狼獲得了三天的休整時間，而噩夢也就此開啟。從12月2日到12月24日，整整23天，森林狼一勝難求，他們先後輸給了灰熊、獨行俠、雷霆、湖人、太陽、爵士、快船、鵜鶘、掘金、開拓者和勇士。

11場比賽，森林狼一共輸了111分，沒有一場比賽的分差在5分以內，這確實讓人失望。唐斯因膝傷缺席了最近4場比賽，整支球隊的士氣低沉。

「這不是什麼秘密，我們現在處於恐慌之中，」小桑德斯對記者說，「但我想說的是，我們依然保持積極的心態，只是這（11連敗）確實很讓人受傷。作為一支年輕球隊，我們必然經歷這樣的困難階段，這讓我們能從中學到一些東西。」

唐斯何時回歸，仍是未知數，但如果季後賽的希望已經非常渺茫，他們或許會讓唐斯安心養傷，沒有必要急著讓這位全明星中鋒回到賽場。

要想避免在12月以全敗的戰績收場，森林狼還有3次終結連敗的機會。他們將在北京時間12月27日客場挑戰國王，29日主場對陣騎士，31日主場迎戰籃網。4連敗的國王，或許是森林狼的突破口，反倒是東部魚腩騎士近況不錯，已經取得3連勝。籃網的歐文仍未回歸，但球隊發揮穩定，近4場比賽也贏下3場。

森林狼11連敗是挺慘，但比起NBA的歷史連敗紀錄，那還有很長的一段距離。據統計，NBA歷史上最長的單賽季連敗是26場，由兩支球隊共同保持。

一支是騎士（2010-11賽季），詹姆斯加盟熱火的第一年，騎士只贏了19場比賽，並在2010年12月21日至2月10日之間連輸26場，其中，他們在1月份一場沒贏。

另一支是76人（2013-14賽季）,那是76人的擺爛賽季，和2010-11賽季的騎士一樣，76人那個賽季只贏了19場比賽，並在2014年2月1日至3月28日之間連輸26場，快兩個月沒贏過球。
