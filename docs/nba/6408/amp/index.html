半場領先17分還遭逆轉 掘金主帥談失利：我們下半場表現得很糟糕

天天要聞
2019-12-01

北京時間12月1日，NBA常規賽繼續進行，掘金在上半場結束時還領先17分的情況下被國王加時逆轉，最終97-100輸掉了比賽。面對這樣一場遺憾的失利，掘金主帥邁克-馬龍在賽後採訪中表示：「下半場國王的防守非常有針對性，我們沒有及時的適應，下半場我們打得很糟糕。」

儘管開局經歷了5連敗，但國王在之後的一個月時間裡開始漸漸找回了狀態，算上本場比賽，他們從11月開始到如今已經取得了8勝5負的戰績，排名也從西部墊底提升到了如今的第九位，勝場數和第八的太陽持平。

本賽季，國王的一改此前糟糕防守的表現，截止本場他們場均失分107.6分，排名聯盟第10位，這是過去7個賽季以來球隊失分首次排進聯盟前十的行列。此役半場結束時，國王40-57落後掘金多達17分，但下半場和加時賽中，他們強硬的防守僅讓對手得到了40分，出色的防守是球隊本場獲勝的關鍵。

此役，掘金多位核心球員在下半場迷失，約基奇全場只有7分進賬，11次出手只有3次命中。而首節表現驚艷（得到18分）的加里-哈里斯，全場只有25分進賬，下半場和加時賽中，他累計只得到了5分。和上賽季在進攻端的青春風暴相比，本賽季的國王展示了自己在防守端的天賦。

本場比賽的失利對於掘金眾將而言，無疑是難以接受的，目前他們13勝4負的戰績距離西部第一的湖人有著3個勝場的差距，和身後的快船勝場持平。接下來的賽程中他們還需要先後對陣湖人和凱爾特人兩支勁旅，稍有不少排名可能跌至第四甚至第五的位置。
