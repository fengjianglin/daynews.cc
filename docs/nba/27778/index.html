詹姆斯血灑賽場，贏球無望後，濃眉拍打地板，湖人遇賽季首次連敗

2019-12-20 3,127條評論 4,806次閱讀 584人點贊

今日，本賽季最焦點之戰上演，東西部榜首湖人雄鹿對決，必將有一隊首次迎來連敗。這也是NBA有史以來第一次進行到這個階段（28場），對陣雙方輸球場次都不足5場的比賽。字母哥在賽前表示他不想連敗，詹姆斯表示他們會避免連敗，打出湖人籃球。

濃眉因為腳踝扭傷缺陣上一場對陣步行者的比賽後，今天迎來複出。沃格爾表示濃眉仍有酸痛感，感覺並沒有特別好，但他就是想出戰。賽前，詹姆斯是穿成這樣來到雄鹿主場的。湖人官方對此配文：「是時候了。」

詹姆斯走進場時，剛開始各種帥酷。不到三秒，他就現出了逗比原形……

今天也是字母哥的哥、字母哥以及字母弟相聚的好日子。

開局湖人率先由詹姆斯助攻丹尼-格林飆中三分，打完4分鐘，伴隨著此起彼伏的打鐵聲，湖人才以5-4領先。隨後詹姆斯的左手臂被撞出了血，現場醫務人員緊急處理了下，繼續上場作戰。

老詹一次突破上籃，直接被米德爾頓摟脖，這算得上是一次兇狠犯規了。

第一節進行到還剩6分53秒時，湖人11投2中，雄鹿11投3中。詹姆斯打完8分鐘就下場了，5投0中貢獻2分4籃板2助攻。首節湖人是17-23落後，他們太注重防守字母哥，反而放空了喬治-希爾，後者單節拿下10分。湖人這邊，濃眉6中1拿到3分3籃板2助攻2蓋帽1搶斷，格林6分，波普4分，麥基2分。

第二節開局雄鹿打出一波6-0從而領先12分後，暫停回來詹姆斯終於飆中今天第一個運動戰三分進球、隨後又背身強打，將分差縮小到個位數。

隨後雄鹿下起三分雨，打出一波14-2，在第二節還有7分鐘時，領先湖人多達19分。詹姆斯打完6分鐘後，拿下7分1籃板，繼續下場休息。隆多上來，連續助攻波普格林投中三分，隨後格林自己又飆中3+0，湖人追到只差13分。

此節雄鹿打了湖人一個42-29，他們上半場以65-46領先，詹姆斯11分7籃板2助攻，濃眉8分4籃板3助攻2蓋帽1搶斷，格林12分，波普9分2助攻，麥基2分，隆多2分1籃板1助攻，霍華德2分3籃板。

第三節開始，格林率先投中三分、波普造犯規兩罰全中，字母哥兩罰中一後，詹姆斯借濃眉掩護投中三分，字母哥還一個三分，濃眉拋投2+0，湖人追到只差13分。

當濃眉造字母哥個人第四次犯規後，湖人看起來還有翻盤的機會。雄鹿繼續依靠三分球穩定局勢，打完6分鐘，湖人65-80落後。

此後濃眉打成2+1，波普空位三分有了後，詹姆斯又助攻濃眉打中。濃眉連續造犯規，接隆多助攻空接，湖人在此節結束時追到了81-89，詹姆斯16分8籃板8助攻，濃眉23分9籃板4助攻，格林18分3籃板，波普16分3助攻，麥基4分3籃板，隆多2分2籃板2助攻。

第四節前兩分鐘，詹姆斯領到了2次犯規，累計已有5次，再犯規就要被罰下場了，接下來的時間，他得悠著點打了。

雄鹿93-81領先後，詹姆斯用三分還以顏色。字母哥投中無人防守的三分，98-84領先後，湖人暫停，詹姆斯下場休息。三分多鐘後，詹姆斯重新回歸，此時湖人90-103落後。濃眉繼續造字母哥進攻犯規，他跟詹姆斯一樣都是5次。詹姆斯突破縮小到個位數分差，雄鹿馬上三分拉開，老詹又助攻濃眉2+1，比賽最後3分51秒，湖人97-106落後。

詹姆斯遠距離長傳，助攻濃眉隔扣字母哥，湖人在最後2分鐘以101-108落後。

關鍵時刻，濃眉被吹了防守犯規，懊惱的他怒拍地板。今天濃眉帶著腳踝傷，患有感冒，打滿了下半場，他真的很想贏。

​

雄鹿隨後2罰中1，湖人連續沒中，最後33秒以101-109落後，濃眉助攻格林快速三分，並對雄鹿採取犯規戰術兩罰全中後。最後26.9秒，湖人落後7分，已經無力回天。

最終比分是湖人104-111雄鹿，詹姆斯21分12籃板11助攻，濃眉36分10籃板5助攻3蓋帽1搶斷，丹尼-格林21分4籃板，波普18分3助攻，麥基4分3籃板2蓋帽，隆多2分2籃板3助攻，霍華德2分4籃板。替補得分方面，湖人4分，雄鹿34分。

字母哥與詹姆斯，東西部榜首球隊的領袖，賽後第一時間擁抱致意。

此役過後，詹姆斯拿到個人本賽季第7次三雙，職業生涯第88次。他的職業生涯常規賽總助攻數達到8967次，超越加里-佩頓，來到了歷史第9位。

今日這場比賽，最多落後雄鹿20多分，這不是湖人的真實水準，所以下半場的他們追了上來。考慮到湖人連續客場作戰的疲勞，再加上全隊開始瀰漫的感冒癥狀，這一場輸球在意料之中。打道回府，回歸主場，好好調整，接下來8場比賽7個主場，掘金快船獨行俠等硬仗在等著湖人呢。
