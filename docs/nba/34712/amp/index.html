12年他被9個球隊放棄卻從不放棄自我，心臟曾停跳75分鐘太硬

天天要聞
2019-12-25

​ 隨著交易市場的大門打開，NBA的各支球隊也開始了一系列的運作，有心在季後賽有所作為的爵士也率先開啟了交易模式。他們換來了衝擊力更好的克拉克森，加盟了替補的實力，不過裁掉傑夫-格林的舉措卻有些令人摸不清頭腦，畢竟如今的格林場均依然能夠貢獻7.8分以及2.7個籃板的表現。

​2007年的選秀大會上，格林在第5位被凱爾特人選中，隨後便被交易到了當時還未改名為雷霆的超音速隊。雖然當時隊內力捧的是同年的榜眼秀杜蘭特，但是年齡比他大兩歲並且身體更加強壯的格林卻能夠與杜蘭特平起平坐，在戰術意義上，格林某些時刻還要高於杜蘭特，而入選最佳新秀陣容更是讓格林獲得了更多的關注。

​不過由於雷霆的建隊方向改變，格林被雷霆隊交易，而交易的對象正是當年選中他的凱爾特人。儘管此時的凱爾特人已經進入了重建期，但是身體素質出色的格林依然成為了球隊鋒線上當仁不讓的領袖。

​只可惜隨後的格林卻突然遭遇了嚴重的疾病，他被查出了主動脈瓣擴大，需要進行心臟搭橋手術，並就此缺席了2011-12賽季的全部比賽。手術成功的他與凱爾特人重新簽訂了一份合約，並且還為球隊貢獻過精彩的絕殺，但是最終還是被交易到了灰熊隊。

​來到了灰熊隊的格林開始走入了下坡路，這時他的場均得分已經下滑至13.1分，但是他依然成為了灰熊發揮最穩定的四號位。不過薪金空間不足的灰熊並沒有留下格林，而是很快將其交易到了快船，而這時的格林也徹底開啟了自己的「流浪」生涯。

​其間曾經輾轉魔術的格林直到2017年夏天才有了新的轉機，志在衝冠的騎士找到了他，以一份老將底薪簽下了格林。被寄予厚望的格林似乎重新煥發了青春，而在東部決賽的最後一場，格林頂替受傷的樂福砍下了19分，幫助騎士成功晉級總決賽。

​成為自由球員的格林選擇了加盟奇才，只不過奇才糟糕的戰績並沒有讓格林有太多的表現機會，雖然他的數據有所提高，但是資金鏈短缺的奇才最終還是放棄了格林。轉戰爵士的格林一度希望藉助這最後的機會東山再起，但是爵士最終還是選擇了裁員。

​這一路走來，格林經歷了太多的曲折，而能夠在心臟手術之後還依然在賽場上征戰，也讓格林成為了聯盟勵志人物的代表。2012年格林接受了心臟搭橋手術，而這個手術會讓格林的心臟停止跳動75分鐘，但是為了能夠繼續征戰賽場，格林義無反顧的選擇接受手術。雖然格林重返賽場，但是這個手術也讓給格林丟失了自己曾經令人艷羨的身體素質，留下了一道長長的傷疤。

​雖然格林被爵士裁員，但是他穩定的三分球以及錯位換防的能力，都是小球時代所必需的。如果那些有心衝冠的球隊能夠在此時簽下格林，對於球隊的人員輪換是一個很好的補充。希望格林可以有一個完美的結局，對於他勵志的生涯也算是一個獎賞。
