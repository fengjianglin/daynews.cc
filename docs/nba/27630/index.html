狂勝15分！雙星齊轟67+11+10，甜瓜3封蓋成亮點，開拓者終進前八

北京時間12月21日：在剛剛結束的NBA常規賽中，開拓者在主場以118-103狂勝魔術15分，本場比賽過後，開拓者取得了3連勝，以12勝16負戰績頂替國王，終於打進了西部的前八。

先看看雙方的數據統計吧，魔術這邊：富尼耶16分2籃板3助攻，富爾茨10分4助攻，艾薩克10分4籃板，戈登11分11籃板4助攻，武切維奇23分12籃板，奧古斯丁17分3助攻，羅斯9分2籃板。

開拓者這邊：麥科勒姆31分8籃板4助攻，利拉德36分6助攻，貝茲莫爾11分4籃板，安東尼5分6籃板3封蓋，懷特塞德10分17籃板，西蒙斯8分6籃板。

本場比賽，開拓者的後場雙星齊轟了67分11籃板10助攻，兩人都有30+得分進賬，這是他倆第15次同時砍下30+，是NBA歷史上後場組合中最多的。這倆進攻一打開，開拓者的進攻就不怕任何球隊了，慢慢的開拓者戰績就上來了，此前一度感覺他們要跟季後賽告別了，但是現在已經打到了西部第八了。

甜瓜安東尼在首節打了僅9分鐘情況下，就受傷離場。根據最新的消息可能是左膝挫傷，還是希望甜瓜能夠健康吧，畢竟這才剛剛開啟了他個人職業生涯的轉折點。

不過甜瓜本場比賽雖然打了9分鐘，但是3次封蓋成為了亮點，最近甜瓜每場都會貢獻2-3次封蓋，這已經說明了他想要在防守上更加努力，而鋒線位置的防守也是開拓者最欠缺的，甜瓜想要通過自己彌補球隊的弱勢一環，這個態度很值得肯定。

在安東尼職業生涯的大部分比賽中，防守一直是他弱項，甚至後期已經成為了阻礙他重返NBA的主要原因之一，他也知道這一點，所以現在在這方面非常的努力。

雖然封蓋和搶斷並不能直觀的反映出一個人的真實防守效果，但是最起碼可以反映出他的態度，這一點就足夠了。
