濃眉與字母哥孰強孰弱？今日濃眉用一記隔扣給出答案

天天要聞
2019-12-21

今日NBA的榜首之爭，神話雄鹿對陣無敵湖人，想必一定是吸引了無數球迷的關注！

但是最讓我為之動容的還是在湖人將要輸球之際，濃眉送出的一記驚天隔扣！

我們知道，字母哥和濃眉是目前NBA最年輕同時最具有潛力的新興超巨，兩人都具有極其出色的身體條件和超強的籃球天賦。

所以，字母哥和濃眉到底誰強誰弱，大家一直熱衷討論。

但是，在小葉心中一直這麼覺得：目前來說，濃眉要強於字母哥，而字母哥的發展前景要比濃眉大。

首先先說身體條件，這方面小葉覺得兩人不分伯仲，兩人都有逆天的身高和臂長，並且較為接近，所以這方面算是打了一個平手。

其次要比較的就是字母哥一直所受詬病的投射能力，雖然現在他正在一步步加強練習，可是這對於目前籃下命中率高達72%的濃眉來說，這絕對是其一個致命的弱點。

還有就是我們在籃球場上極其重要的球商，3次當選蓋帽王的濃眉善於聯合隊友一起進行有效的防守，有著極強的護框能力。字母哥憑藉身高臂長，雖然也有強大的防守能力，但是其防守效率並沒有濃眉高，這很多時候是因為他沒有像濃眉那樣聯合隊友，積極補防，很多時候，也表現為是經驗的欠缺。

所以，濃眉更像是一個球隊的領袖人物，而字母哥則僅僅是球隊的一把利器。

那為什麼說，字母哥有著更遠的發展前景呢？

目前，字母哥要遠比濃眉健康，而且其現在主要是外線打法，有著比濃眉更強的控球能力，其欠缺的經驗和投射能力都可以慢慢地積累和強化。

因此，隨著時間的推移，字母哥必將成為更加強大的「希臘怪獸」。

這次比賽雖然湖人輸了，同時詹姆斯手臂也被刮傷，但是小葉相信最後的決賽，湖人一定會有更加精彩的表現！
