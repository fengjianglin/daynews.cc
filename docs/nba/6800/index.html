不懼卡泡，鶯歌24+8穩定輸出，依然不在鵜鶘的未來藍圖中！

2019-11-25 0條評論 16次閱讀 0人點贊

在今天鵜鶘109：134大比分輸給快船的比賽中。鵜鶘核心鶯歌拉姆依舊發揮穩定，得到了24分8籃板5助攻。三項數據均為球隊最高。本賽季鶯歌拉姆進步明顯，這場比賽面對全聯盟最好的鋒線防守組合，可謂是一次檢驗了。鶯歌拉姆真金不怕火煉依然打出了賽季平均水平，交出了一份滿意的答卷。

本場比賽鶯歌依然慢熱，上半場命中率慘不忍睹，面對小卡和泡腳的防守極不適應，表現的也很掙扎。球隊也大比分落後。第三節一上來鶯歌迎來了爆發，兩記中距離跳投穩穩命中，面對老隊友祖巴茨毫不手軟打成二加一，外線三分也應聲入網。短短六分鐘四投全中，得到十分。分差也追到只剩下六分。

手感火熱的鶯歌突然摸不到球了，鶯歌大空位霍勒迪選擇忽視，自己突破上籃走步，球哥，摩爾也是接球就投，毫無意外的打鐵。鶯歌又變成了長跑運動員了。反觀快船那邊戰術非常明確，就打路威和哈雷爾強點。哈雷爾如入無人之境，將鵜鶘內線打成了篩子，分差被無線拉開，此役哈雷爾化身內線巨獸，全場得到34+12。

本賽季鶯歌拉姆表現出色，可以說是鵜鶘的絕對核心，但是球隊卻始終不認可。主教練金特里在媒體公開採訪中說到，霍勒迪是球隊的領袖，鼓勵霍勒迪多出手。最近幾場比賽鶯歌拉姆減少了大量的中距離出手。也出自金特里之手，讓鶯歌多突破殺傷。鶯歌是現役最好的中距離投手之一，這樣豈不是自斷一臂？

本賽季鶯歌的發揮大大超出鵜鶘管理層的預料。在鵜鶘規劃的藍圖中 或許之後狀元錫安和霍勒迪吧！本賽季也是鶯歌的合同年，打出一個好數據，去一個真正以自己為核心的球隊吧！
