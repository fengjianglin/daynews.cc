湖人替補是累贅？數據說明一切！

2019-12-21 201條評論 1,389次閱讀 107人點贊

在洛杉磯湖人遭遇本賽季首次連敗之後，其替補球員是不是「累贅」的問題也愈發備受關注！

12月20日湖人客場挑戰密爾沃基雄鹿的比賽以104－111敗北，雖然詹姆斯出戰37分鐘、投籃19投8中、三分7中3、罰球2中2、總共得到21分12籃板11助攻，濃眉哥戴維斯出戰43分、投籃25中11、罰球17中14、總共得到36分10籃板5助攻1搶斷3封蓋，但湖人替補在比賽中僅得4分（霍華德和隆多各2分），讓「詹眉組合」顯得孤立無援，成為了賽季首次連敗的關鍵原因之一！

那麼，本賽季目前為止湖人替補的場均得分屬於什麼水平？如何看待湖人替補的問題呢？

以下是球隊替補球員場均得分的具體排名:湖人替補以場均得到33.8分而排名聯盟第22，比最差的馬賽克隊多了8.8分，比最好的快船隊少了17.5分，也就是湖人替補的得分能力整體上屬於偏低的水平！

而在湖人替補球員之中，場均得分最多的是庫茲馬11.1分，其餘就再也沒有可以得分上雙的「替補幹將」了……

要知道，在湖人和雄鹿的比賽中，雄鹿替補則是全面開花，總共得到了34分，特別是希爾的21分確實讓湖人有點防不勝防，也難怪湖人在比賽第二節就逐漸被擊潰，即便後面苦苦追趕也變得無濟於事！

畢竟賽季漫長，湖人只靠詹姆斯和濃眉哥戴維斯的雙核驅動，並不是長久之計，重點還是得要改善替補的狀態，畢竟湖人替補逐漸傾向於隱藏著不少「累贅」作用，要麼通過球員市場繼續補強，要麼想方設法提升替補效率！
