格林防守有多強？談到最後絕殺失敗，博伊倫和拉文都提到了格林

天天要聞
2019-12-07

現在看來，在2019年休賽期追夢格林選擇提前和勇士隊續約，是多麼明智的一件事情。在杜蘭特出走，水花因傷缺陣的情況下，格林支配球、做二傳手的優勢也發揮不出來了。本賽季到目前為止，追夢格林為勇士隊出戰了15場比賽，場均出戰27.8分鐘，只能夠拿下8.3分6.1籃板5.6助攻。勇士隊也僅僅取得5勝19負的戰績，排名西部聯盟倒數第1位。

格林糟糕的表現，讓很多球迷吐糟：格林不值5年1.18億。實事求是的講，個人認為格林5年1.18億美元的合同還是比較符合身價的。雖然場均8.3分實在是太低了，但是作為大前鋒場均5.6次助攻還算說的過去。當然更為重要的是格林的防守能力，在現在這個小球時代，格林的移動敏捷性和對抗，能夠使他從1號位防到5號位。格林的大合同中，有很大一部分就是他防守的價值。

就拿今天勇士對陣公牛隊的比賽來說吧，雖然格林只拿下了9+4+5的數據，但同時也送出了4次搶斷1次封蓋，在防守端做出了很大的貢獻。9+4+5卻是取勝功臣？也許很多球迷並不以為然，但是賽後在談到拉文最後絕殺失敗的時候，公牛隊主教練博伊倫和拉文都提到了追夢格林。那麼格林的防守有多強呢？無論是博伊倫和拉文都表達了這麼一個意思：不想面對格林的防守！

當時公牛隊98-100落後勇士隊2分，拉文持球面對羅賓遜的防守，正常情況下，拉文應該選擇一個擋拆，然後選擇突破還是投籃。當時溫德爾-卡特試圖上來給拉文做一個擋拆，但是拉文揮手示意拉開，一個轉身後直接干拔出手3分不中，勇士隊以100-98戰勝了公牛隊。那麼，拉文當時為什麼沒有選擇擋拆呢？我們來看看賽後的採訪他們是怎麼說的！

公牛隊的主教練博伊倫是這樣解釋的：「我們希望根據對手的防守站位來選擇我們的進攻手段，畢竟我們不希望擋拆後是德雷蒙德-格林換防，最後結果就是這樣。」拉文在採訪時說的話，也大概表示出了這個意思：「原定的計劃是我和馬爾卡寧擋拆，但是格林不停地在準備換防。很明顯我們不希望換防後跟他一對一，所以我選擇了不走擋拆。當我看到還剩3秒時，我發動了進攻。我認為我的出手角度不錯，只可惜力量稍大。」

從博伊倫和拉文的話中，我們大概可以得出這樣一個結論：公牛隊的最後一攻，本來是要一個擋拆戰術的。但是格林不停的換防擋拆人，造成了拉文選擇擋拆之後就要直接面對追夢格林的防守。顯然博伊倫和拉文都認為選擇1V1打格林不是個很好的選擇，因此選擇了不要擋拆在外線干拔三分。由此可見格林在防守端的影響力。

那麼，大家認為格林的技戰術能力，值不值5年1.18億美元呢？
