湖人第三巨頭出爐！本賽季他出戰7勝0負，在隊內享有無限出手權

2019-11-27 0條評論 14次閱讀 0人點贊

本賽季的湖人隊可謂是無人可擋，今天剛剛完成了一波8連勝，其中最功不可沒的或許就是波普了吧，正是因為「波普定律」的存在，本賽季紫金軍團才可以走得如此順暢。然而現在湖人隊內似乎又突然出現了另外一個神定律，那就是「隆多定律」。

根據統計，自從隆多復出以來，湖人隊已經經歷了整整7場比賽，而在這些比賽當中，紫金軍團則是全部拿下勝利。換句話說，目前隆多是全聯盟唯一一個勝率100%的球員，而如今這支湖人隊也正是因為他的回歸，而變得更加強大了。

在本賽季還沒開始之前，外界普遍認為湖人的第三巨頭應該是庫茲馬，甚至也有人認為是霍華德，總之雖然爭議聲很大，但他們兩個的呼聲始終是最高的。但從本賽季的表現來看，以上兩個人似乎都已經出局了。首先是庫茲馬，他是湖人今夏唯一留下的天才新秀，可經過了一個夏天的磨練之後，他卻似乎絲毫沒有進步，不僅防守能力還是那麼差，就連賴以生存的進攻能力也退化了。截止到目前，他場均只能夠得到12分3.6籃板，命中率44.9%，三分命中率32.3%，這樣的表現顯然無法讓他成為球隊的第三巨頭。

至於霍華德呢，雖然瘦死的駱駝比馬大，但是如今他在湖人隊更像是一個藍領中鋒，場均只能夠獲得3.5次的出手機會，雖然總體來說發揮得還算不錯，但這樣的球隊地位，說他是第三巨頭似乎有點勉強。更值得一提的是，在今天的比賽中，他甚至被靈活的阿爾德里奇徹底打爆，在出場18分鐘的情況下1分未得，同時還領到了5次犯規。不得不說，如今的霍華德雖然是湖人必不可缺的奪冠拼圖，但絕對還不足以能夠擔任球隊的第三巨頭。

​

所以說，如今最適合擔任湖人第三巨頭這個位置的還得是33歲的拉簡-隆多，作為一名總冠軍控衛，他的資歷與分量還是比較能夠壓得住其他球員的。雖說本賽季隆多都是出任替補控衛的位置，但他卻是全隊除了詹姆斯以外，唯一一個進攻發起點了，所以相比於霍華德跟庫茲馬，他在球隊的戰術地位顯然更高，總不可能讓詹姆斯打滿48分鐘吧。

除此之外，本賽季隆多最薄弱的投射能力得到了很大的提升，就比如今天對陣馬刺隊的比賽，他的三分球就是3投3中，砍下了13分5籃板3助攻，成為了球隊8連勝的第三功臣，僅次於詹姆斯與戴維斯。截止到目前，他場均的三分命中率高達43.5%，能夠砍下9分3.4籃板5.3助攻，在球隊當中他的出手權甚至不亞於任何人，主要取決於他想不想出手罷了。就比如在上一場對陣灰熊的比賽中，他就全場出手了15次，比庫茲馬+波普+霍華德合起來還多，甚至要多於戴維斯。

總之呢，目前湖人的第三巨頭基本可以確認就是隆多了，除非庫茲馬能夠後程發力，至少得發揮到跟他上賽季一樣的水平，否則想要搶走這個位置，恐怕也是無望的。
