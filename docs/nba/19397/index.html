現役5大最強雙人組排名：哈登威少僅排第四，榜首化學反應太好了

2019-12-18 676條評論 1,655次閱讀 364人點贊

隨著勇士王朝的解體，NBA似乎又回到了當初多方豪強爭霸的局面，聯盟中不再有過去十年那樣的多巨頭超級球隊，取而代之的是更多的雙核球隊。只不過同樣是雙核，球隊依舊有強弱之分，有些球隊雙核配合就很好，而有的則沒有1+1>2的化學反應。今天我們就從目前的表現來看下現役五大最強的雙人組合。

5、布克盧比奧

在現今NBA各個位置都非常模糊的時代，太陽這個二人組卻是少見的各司其職的組合：傳統控衛加上傳統分衛。當年科比獨自帶領湖人時曾有過與保羅聯手的機會，可惜最終沒有成功，而布克加盧比奧的組合就很像弱化版的科比加保羅。兩人也有非常好的化學反應，有了盧比奧掌控全局，布克可以放心做自己擅長的事情了。太陽隊本賽季的成績也是水漲船高，可以說這個二人組合也許實力不算最頂級，卻是最適合彼此的那個。

4、哈登威少

這個賽季馬賽克隊的比賽似乎沒有想像中那麼好，哈登在開賽低迷後一路走高，場均得分來到了38.9分，與之相反的是開賽表現不錯還能維持場均三雙的威少卻一路走低，儘管現在還有場均22.8的得分，但命中率已經跌到42.6%，三分命中率更是只有22.8%。二人似乎並沒有找回當年雷霆時期的默契，但兩位MVP還是在缺兵少將的前提下將球隊帶到了西部前四，只能說還是倆人的衝擊力太強了。

3、喬治倫納德

這個二人組合跟哈登威少的組合差不多，喬治開賽就因傷缺席了十餘場比賽，倫納德養傷，背靠背只打一場，二人同時在場的比賽其實不多，因此人們開始質疑這對組合究竟能不能產生良好的化學反應。不過雖然配合不是很多，兩人本賽季也都是穩定發揮，喬治場均能得到24.8分，命中率44.8%，倫納德則是場均25.8分，命中率45%，快船西部第二的成績便已經說明了喬治倫納德聯手的威力。

2、字母哥米德爾頓

雄鹿本賽季取得18連勝好成績，戰績東部第一。相比於上面那些剛剛組建的雙核，字母哥與米德爾頓完全不用擔心默契方面的問題，而且二人一內一外，非常搭配。表現出來的就是字母哥這賽季數據已經是恐怖的31+12.8+5.3，米德爾頓雖然只有18.6分，但命中率也是驚人的50%與41.3%。多年的默契加上內外搭配讓兩人都有了恐怖的效率，雄鹿也得以穩定起飛。

1、詹姆斯戴維斯

聯盟第一的戰績，詹姆斯能跟戴維斯搭檔得如此之好，可能開賽前誰都沒有想到。只能說超級球星就是萬能的，在賽場上沒有他們適應不了的情況。二人都根據對方特點改變了自己過往的打法，結果也都取得了極大的成效。在詹姆斯主掌控，戴維斯主終結的情況下，湖人的化學反應著實令人驚喜不已，一點不像剛組建的球隊，成績也是直接飛躍到聯盟第一。
