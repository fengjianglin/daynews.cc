湖人繼續補強防守悍將伊戈達拉，針對雄鹿字母哥和快船小卡

2019-12-23 1,981條評論 2,939次閱讀 286人點贊

目前，伊戈達拉被交易到小牛、快船、湖人的傳言和幾率最大！

灰熊放話伊戈達拉的要價可以放低，只要一個次輪簽就行。然而據美媒的報道，當獨行俠興沖沖的拿著考特尼-李和次輪簽向灰熊報價時，灰熊又反口了，說他們還是要首輪簽，就算是首輪末段也行。

灰熊在得到一哥的時候從勇士那拿到了一個首輪簽。現在出手又要一個首輪簽，這樣一來獨行俠不幹了，交易宣告破產。

如果灰熊最終沒辦法交易伊戈達拉，那隻能買斷，屆時湖人的機會就來了。

此前，伊戈達拉也放出消息要和詹姆斯聯手，這樣一來湖人的確佔了很多優勢！

然而戰績來說，湖人雖然是西部第一，但是本賽季輸給了雄鹿，丟掉了全聯盟第一勝率寶座

可以看出，沒有專門能夠限制字母哥的人，一哥作為當年勇士王朝的功臣，主要以防守對方核心人物為主，而且非常成功，比如限制杜蘭特詹姆斯，這樣聯盟第一人的絕對核心，一哥都是有一套！

同樣，面對同城勁敵，小卡也需要有人專門限制才成，伊戈達拉如果加盟湖人，無疑是給湖人一劑強心劑，讓湖人可以解脫詹姆斯和濃眉的防守壓力，全新投在進攻方面，想必到時奪冠更加有保障了
