范弗里特為詹姆斯·哈登在火箭的糟糕表現進行了辯護

2019-12-05 0條評論 16次閱讀 0人點贊

在周四與詹姆斯·哈登和休斯頓火箭隊會面之前，多倫多猛龍隊後衛弗雷德·范弗里特對聯盟頭號得分手大加讚賞，他的高強度進攻讓很多人對籃球產生了兩極分化。

「我們已經嘗試了所有你能想到的防守，所以這不是我們第一次做這事，除了詹姆斯一對一時有點像外星人，」當被問及他們對這位前MVP的防守計劃時，范弗里特對TSN的喬什·勒文伯格說。

哈登本賽季場均39.5分，這一數據將超過他去年36.1分的場均得分。本賽季他參加了20場比賽，其中16場他都得到了至少32分。

但外界一直在批評哈登，說他在製造犯規方面的能力無與倫比。他場均13.1的罰球數，遠遠超過了聯盟中任何一個球員的平均水平。然而，在范弗里特看來，這就是為什麼哈登和他天文數字般的表現如此令人印象深刻的部分原因:

「你試著從他身上學到東西，每個球員都夢想著像他那樣製造犯規……他很特別，」范弗里特說。

范弗里特在本賽季取得了突破，他在本賽季場均18.7分，7.4次助攻和3.8個籃板——所有這些都將創造職業生涯新高——而每場比賽上場37.8分鐘，是聯盟中上場時間最多的球員。他可能沒有阻止哈登的秘密武器，但是他會盡自己最大的努力讓多倫多取得勝利。
