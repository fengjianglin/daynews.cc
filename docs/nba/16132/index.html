騎士樂於助人！一戰助攻雄鹿延續1記錄，5天後或被湖人終結？

2019-12-16 524條評論 1,582次閱讀 312人點贊

北京時間12月15日，NBA常規賽繼續上演，東部迎來一場毫無懸念的對抗。東部第一的密爾沃基雄鹿隊主場迎戰克里夫蘭騎士隊，果不其然雄鹿以125-108擊敗騎士隊，取得了18連勝的驕人戰績。這也是本賽季至今最長的連勝紀錄，此前湖人十連勝也被終結，現在只有雄鹿一枝獨秀。

本場比賽可以用毫無懸念來說，單從排名上來看兩隊的差距還是很大的，本場未開始前。雄鹿隊取得23勝3負的戰績排在東部的第二位，截止昨天還和湖人戰績打平並列第一的戰績。而騎士方面，6勝19負排在東部倒數第三，他們本賽季已經是任人宰割的球隊了，輸給雄鹿隊一點也不讓人意外。

本場比賽字母哥依舊出色，砍下了29分4籃板4助攻的全面數據，也正是在他的帶領下雄鹿本賽季高歌猛進。另外一位大將米德爾頓拿到24分和6次助攻，幫助字母哥緩解了不少壓力，本場勝利也少不了他的貢獻。騎士方面，樂福斬獲12分10籃板的兩雙數據，個人表現上只能說是一般，畢竟他可是全明星球員，現如今讓人頗有幾分失望。

騎士今天真的成了老好人，雄鹿今天的18連勝是本賽季最長的連勝紀錄了，此前湖人的10連勝已經被終結。雄鹿距離隊史上最長連勝紀錄還剩2場，他們能將連勝繼續延續至18連勝已經很難得了。尤其是東西部豪強雲集，每勝一場難度還是非常大的。單看賽程的話，雄鹿接下來的難度還是不小的。

12月17日將在主場迎戰獨行俠，這場比賽明面上還是雄鹿贏面大，尤其是在東契奇受傷的情況下，讓他們的贏面更大。當然雄鹿也並不是就一定能獲勝的，布萊德索因腓骨撕脫性骨折將要缺席至少兩周，首發控衛的位置上就變得很薄弱了。雄鹿很可能會將連勝紀錄延續到19連勝，但想要破掉隊史最長連勝難度還是很大的。

12月20日雄鹿將在主場迎接洛杉磯湖人隊的挑戰，湖人現在勢頭正盛，球隊近期也是取得了6連勝，是現在的第二長連勝紀錄。再加上他們現在磨合的也是非常不錯，雄鹿在缺少布萊德索的情況下想要擊敗湖人難度還是很大的。東西部的老大強強對決，必定是一場火花四濺的比賽，你覺得是湖人終結雄鹿的連勝，還是雄鹿能繼續上演連勝呢？歡迎下方評論區留言討論。
