休賽期失去FMVP萊昂納德的衛冕冠軍，如何在新賽季持續高歌凱奏？

2019-12-04 0條評論 17次閱讀 0人點贊

多倫多猛龍面對頂級防守強隊爵士，在上半場就領先了40分之多，全場轟下130分大勝而歸。此役過後，猛龍的戰績達到了15勝4負，百回合得分聯盟第五，百回合失分聯盟第二，百回合凈勝分高達8.8分，僅次於聯盟第一的雄鹿。猛龍目前為止的表現相當於一支賽季62勝的頂級強隊。休賽期失去了FMVP倫納德和最強3D配角之一的丹尼-格林兩名首發的猛龍是如何做到的呢？

上賽季倫納德和格林常規賽合計出場4200分鐘，其中倫納德在場的時候會扛走30%的進攻球權，他們倆人的離開對猛龍來說最首要的難題就是如何消化掉這部分時間和球權的虧空。納斯採用的方法簡單暴力但有效——讓餘下的其他主力球員打更多的時間扛更多的球權。

上場時間上，范弗里特新賽季首發出場，時間從上賽季的27分鐘暴漲至38分鐘，西亞卡姆從31分鐘漲至36分鐘，OG-奧努諾比從19分鐘漲至29分鐘，鮑威爾從18分鐘漲至27分鐘，就連職業生涯已經步入最後階段的小加索爾也漲了5分鐘。增加主力球員，縮短輪換毫無疑問能提升球隊在單位時間內的強度。

球權分配上，由於洛瑞只出場了八場比賽就受傷無法出賽，猛龍的進攻主要任務都落在了范弗里特和西亞卡姆身上。范弗里特整體水準上較上賽季並沒有明顯變化，但他的回合佔有率從20%提升到了25%，從這個角度講，大幅度提升球權卻沒有效率上的下滑，本身就是一種不小的進步，已經成長為一個合格的強隊副攻手。而西亞卡姆則是再一次迎來了全面進化，他的球權從20%暴漲至30%，已經達到了頂級球星的水準。他的個人得分效率略有下滑，但仍保持在平均線以上，更值得讚賞的是他作為球隊的持球核心，承擔了更多傳控組織任務來為隊友創造機會，助攻率超過了90%的鋒線球員，而失誤卻沒有變多。三分穩定性這個原本令人擔心的問題不僅沒有出現問題，反而有了更多的非助攻出手，命中率也更高了。

球隊的整體打法上由於更喜歡中距離出手的倫納德的離開，猛龍的出手分布徹底走向了魔球化。小加索爾在進攻端雖然個人主攻大幅度下滑，但三分投射保質保量，給范弗里特和西亞卡姆提供了極好的攻筐空間，猛龍籃下出手比例聯盟第4，三分出手比例聯盟第6，中距離出手比例僅次於火箭和籃網聯盟第三少。根據出手分布計算的期望命中率，猛龍聯盟第二。也就是說，儘管失去了一個頂級球星，但猛龍還是依靠內部挖潛和優化進攻策略填補了這個空缺。

防守端對猛龍來說則相對沒有什麼真正的難題。上賽季他們就完全不依賴倫納德的防守，甚至倫納德不在場時猛龍的百回合失分還更少。原本的防守底子就好，又縮短了輪換，增加主力的出場時間，猛龍維持防守並不意外。但猛龍目前的百回合失分比上賽季還少了5分，這其中的變化依然值得稱道。

球員層面上，小加索爾把精力更多地投入在了防守端是一方面，OG-奧努諾比的進步也是重要原因，但防守體系的搭建最重要的還是全隊的策略和執行。就如同勇士經歷了14-15總決賽的後三場後真正發掘了死亡五小在15-16賽季轟下歷史級的73勝。猛龍在經歷了一場奪冠之旅後強化了他們在季後賽獲得經驗。他們十分強調護筐，限制對手籃下出手比例聯盟第三，限制籃下命中率聯盟第一，甚至會為了護筐把總決賽針對庫里的Box-1聯防也帶到了總決賽，2-3聯防和局部夾擊也極其頻繁，造失誤聯盟第五。

頻繁守聯防和夾擊往往會導致漏三分，猛龍的對手底角三分出手比例聯盟最高。但猛龍從他們的東決對手雄鹿那裡學到了一個全新的理念——只要放的是頂級射手，強調護筐捨棄三分的收益會更大。放底角三分並沒有傷到猛龍，反而讓他們限制對手有效命中率聯盟第一。

當然猛龍並非是完美的，他們籃板很差，打到目前，對手的弧頂三分有些失常，存在回歸的可能性，從常規賽一開始就打短輪換能否長期維持都會成為猛龍的續航隱患。我們可以繼續觀察，看看這支失去了FMVP的衛冕冠軍還隱藏著多少實力。
