什麼叫天賦？對不起，全聯盟只有詹姆斯和維金斯才有的屬性

2019-12-18 73條評論 1,014次閱讀 64人點贊

32分13籃板7助攻3蓋帽，老詹手肘受傷基本上也就這個數據了，當然了隨後在快攻中詹姆斯又展現了一個摺疊暴扣，隨後又強勢封蓋隊友隆多。你沒看錯，詹姆斯的手肘有傷！有傷為啥還能這麼玩？因為人家的身上才是滿滿的天賦啊，什麼叫天賦？不是你跳的高、跑得快、爆發猛就叫天賦的，這些充其量只能叫身體素質，真正的天賦什麼樣？就像詹姆斯和維金斯這樣，為什麼要說維金斯，慢慢的往後看你就知道了！

詹姆斯什麼時候進入的聯盟？03年，除去這個賽季的話滿打滿算是打了16個賽季了，其中出戰場數最少的一個賽季是上賽季，只打了55場的比賽，休戰了一個月左右的時間，這一次的休戰時間為詹姆斯16年職業生涯中的最長時間了。同時這次休戰是因為腹股溝拉傷的傷病問題，腹股溝拉傷是什麼傷病？說白了就是老詹的運動量過大了，是一種韌帶性的拉傷，這種傷病和旁人無關，完全是自己的問題。這也就是說16年的職業生涯中詹姆斯從沒有因為什麼身體對抗的緣故受到大型的傷病，要知道詹姆斯的打法可是殘暴派的，這一點就不用多說什麼了。

縱觀詹姆斯的前十六個職業生涯中打滿賽季全勤的只有一次，其餘的賽季基本上都是七十來場比賽左右，說白了就是詹姆斯可以打滿全勤，但是為了季後賽就沒有那個必要了，偶爾跟自己輪休那麼幾場比賽，所以說這十幾個賽季下來詹姆斯幾乎是鐵人了。這麼說還不直觀，他的常規賽生涯中場均出場時間是38.5分鐘，季後賽是42.1分鐘，我們熟知的鐵人科比，常規賽的場均出場時間是36.1分鐘，季後賽是39.3分鐘。以後誰在說老詹划水，這份數據是可以狠狠的打臉的。看到現在你應該明白什麼叫天賦了吧，不是說身體素質爆炸就敢叫天賦的，奧登的身體素質和格里芬的身體素質都是一等一的好，結果是什麼基本上都是被傷病影響了。真正的天賦不僅僅要求你的身體素質好，最重要的是耐打和持久，一個賽季兩個賽季的比賽狂暴不算厲害，有本事一直穩定下去，這才叫天賦！

這一點維金斯就做的很好，你說人家枸杞養生，五個賽季中三個賽季全勤，一個賽季打了81場一個賽季打了73場，每個賽季都能平均貢獻1500分以上，這才叫天賦。並且人家維金斯的場均出場時間可是達到了場均36分鐘，妥妥的也是盡全力了，所以根本不存在什麼養生之類的，人家就是天賦好。天賦這個東西並不是每個人都有的，這種東西只能是運氣和基因以及後天的訓練的三合一。目前來看，聯盟中能夠擁有這種天賦的球員除了詹姆斯就只剩下維金斯了！
