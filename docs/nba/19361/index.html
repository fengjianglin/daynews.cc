NBA歷史得分榜裡面排名第一的人，到底是有多麼恐怖的實力？

2019-12-18 273條評論 1,553次閱讀 117人點贊

NBA歷史得分第一人為賈巴爾，其得分為38387分。這個數據至今無人能夠打破，即使是得分能力超強的科比和喬丹也難以超越。可見賈巴爾在得分能力是偉大的，得分能力是十分恐怖的，傲視群雄。其得分能力強，主要有以下三點。

第一，賈巴爾的絕活，勾手投籃，被人們稱作天鉤。相對於現在籃球運動員的勾手投籃，賈巴爾能夠在罰球線的位置進行勾手，投籃空間十分大。而且這勾手是無解的，防守運動員無法對他進行有效的干擾，所以他的投籃命中率很高。勾手投籃，不僅高效簡單，而且不需要高強度的對抗。背身單打勾手，這動作雖然簡單，但講究手感的柔和度。而對於賈巴爾剛好能夠掌握住這種投籃手感。

第二，賈巴爾具有很強的身體素質，具有很好的防守和搶斷能力。在NBA中，除了那天勾以外，賈巴爾還有力量和身體強壯而出門。由於具有很好的身體素質，身體一直處於健康狀態，所以賈巴爾的出勤率十分的高。高場次的出勤，也幫助他能夠更好的進行得分，除了得分能力，延長職業生涯長度也是一大關鍵。

第三，賈巴爾遇到關鍵三年。本來賈巴爾在有退役的想法的時候，其歷史得分只要35000多分，但是賈巴爾還是堅持再打了3年。正是由於這三年，賈巴爾又拿了3000多分。這三年，讓他的歷史得分很難再被超越。
