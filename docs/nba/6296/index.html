造流量？NBA周最佳是他？下坡路已是必然？

2019-12-03 0條評論 18次閱讀 0人點贊

今日NBA公布了上周的東西部周最佳球員，雄鹿揚尼斯-阿德托昆博和開拓者卡梅羅-安東尼分別當選。

上周阿德托昆博幫助球隊取得了4勝0負的戰績，場均得到34.8分11.3籃板4.0助攻1.5蓋帽。這是阿德托昆博本賽季第二次、職業生涯第11次當選周最佳球員。

上周安東尼幫助球隊取得了3勝0負的戰績，場均得到22.3分7.7籃板2.7助攻0.7搶斷。這是安東尼職業生涯第17次當選周最佳球員，他上一次當選周最佳球員還要追溯到2014年3月10日。

阿德托昆博當選東部周最佳球員可謂是不遑多讓，可是西部周最佳竟然是卡梅隆安東尼卻有點名不副實。

上周安東尼幫助球隊取得了3勝0負的戰績，場均數據也有22.3分7.7籃板2.7助攻，按說這數據在21世紀第一個十年也是很好的一周表現。但是最近幾個賽季，NBA數據井噴，這個數據著實有點拿不出手。即使對比上周西部其他球員，其數據和戰績上也不算最好的那個。

作為17-18賽季MVP——詹姆斯哈登，在上周的三場比賽中場均貢獻42分6籃板8助攻，幫助傷病纏身的球隊取得了2勝1負的戰績。其表現可謂神勇，更有三節打卡轟下60分的驚人表現。

同為東南賽區的達拉斯獨行俠隊的當家球星——二年級球員盧卡東契基，在上周的表現也讓人驚嘆。上周三場比賽，盧卡東契基有兩場比賽砍下40分+10助攻，場均數據35分7.7籃板9助攻的全面表現帶領球隊2勝1負。上賽季，達拉斯獨行俠隊還是西部墊底球隊，這賽季由於盧卡東契基的優異表現，現高居西部第四。盧卡東契基的優異表現也讓自己被各大媒體納入了本賽季的MVP的討論範圍中。

戰績上，詹姆斯哈登和盧卡東契基雖然沒有卡梅隆安東尼所在的開拓者好，但是無論是球星咖位還是個人數據，這二位均有過之而無不及。

其實，上周周最佳還有一個最佳人選，那就是在湖人的安東尼同屆狀元——勒布朗詹姆斯。在洛杉磯湖人隊上周的三場比賽中，勒布朗詹姆斯場均貢獻28.3分3.7籃板12助攻，幫助球隊豪取三連勝。湖人也因為連勝一直牢牢佔據西部榜首的位置。無論戰績還是個人數據，勒布朗詹姆斯無疑比卡梅隆安東尼更適合是上周的西部最佳球員。

NBA上周的西部最佳球員的評選無疑是不能服眾的。當然，NBA這樣做自然有他自己的原因。

據NBA官方統計，聯盟本賽季的平均收看觀眾已經跌破１００萬人次。賽季至今為止，美國本土最大直播平台ＥＳＰＮ已經有９場比賽的收看觀眾低於１００萬。而另一個全美直播平台ＴＮＴ的數據顯示，他們不僅已經連續６場比賽直播收看觀眾人數低於１００萬人次，其中一場比賽更是跌倒了７６萬。這樣的收視率在ＮＢＡ歷史都是十分罕見的。

由此可見，由於堅持雙標力挺休斯頓球隊總經理，ＮＢＡ不只是在中國遭遇了史無前例的「冰川期」，在美國本土大本營也同樣不受待見。

這次，如此評選的周最佳，無疑是想通過製造話題，吸引各大媒體的大肆討論，引導觀眾關注。但是這種低級營銷手段，短時期尚可，長期下去必定會是飲鴆止渴。ＮＢＡ若不能處理好中國市場，下坡路會是必然。
