去年新秀得分上雙數，1人力壓東契奇特雷-楊！卻被無情擺上貨架

2019-12-24 300條評論 1,901次閱讀 422人點贊

早些時候，NBA這邊當天進行的11場比賽依舊還在進行當中，雖然比賽有11場，可是我們國內這邊能夠看直播的卻是有2場。

而且這一天也有一則畢竟勁爆的消息出爐，那就是騎士和爵士達成了本賽季第一筆交易，爵士這邊送出艾克薩姆和2個次輪簽從騎士那邊交易過來克拉克森。

看來爵士這個賽季是有衝冠的想法啊！本來康利只要打出來，爵士也不至於去和騎士做交易，可沒辦法，康利這個賽季打的是真水。

而這一天騎士也是有著一場比賽的，他們當天是在自己的主場迎戰老鷹，兩支球隊都可以被稱為魚腩隊，畢竟一個東部墊底一個排名第13。

因為克拉克森已經準備好要送去爵士那邊了，所以近幾場比賽他應該都不會參戰，不過就這一場比賽來看，老鷹在東部墊底也不是沒有原因的。

這一場比賽當中輸給騎士之後，又一次地拿到了8連敗，這才結束10連敗多久？又向著那個方向去了？

2018年的新秀表現最好的4人應該就是東契奇，特雷-楊，亞歷山大以及塞克斯頓了，而且他們4人現在，在他們自己的球隊當中都是位居得分榜第一位的！

東契奇和特雷-楊都已經成為球隊的老大了，得分能力都這麼強！那麼去年這幾位新秀得分上雙數到底誰更多呢？

小編也是去查了查數據，雖然東契奇和特雷-楊成為了球隊大當家，可是在得分上雙的場次上，東契奇和特雷-樣都不是第一！

有1人力壓了東契奇和特雷-楊，而這個人就是塞克斯頓，騎士和老鷹這一場比賽結束之後，塞克斯頓和特雷-樣的得分都上了雙。

不過這一次得分上雙之後，塞克斯頓已經是第100場比賽得分上雙了，位居這個榜單的第一，而特雷-楊現在也才98場。

東契奇那邊的話，因為傷停了幾場，現在是以92場得分上雙位居第3，最後第4的就是亞歷山大73場，看來亞歷山大雖然有潛力，可和他們還是有一段距離。

而且值得一提的是，塞克斯頓這2個賽季當中一共是出場了111場比賽，其中首發出戰有101場，這樣就已經拿到了100次得分上雙，真的有點過分了。

不過東契奇還是很可惜，他2個賽季一共才上場了97次，他要是和塞克斯頓他們上場次數同樣多的話，那現在這個榜單榜首的位置估計得變一變了，不過這也沒辦法， 發生了就沒辦法再更改。

雖然塞克斯頓力壓東契奇和特雷-樣，可他卻高興不起來，騎士這邊的話，正打算重建球隊。

並且此前也有消息傳出，他們隊內除了加蘭德之外，其他所有人都有可能被交易，塞克斯頓真的太難了！明明表現的這麼好，卻要被無球的擺上貨架。

不過騎士這麼做確實也不是沒道理的，反正都已經在聯盟倒數了，還不如重新來過。

你們覺得騎士這個賽季到底會送出多少球員出去？
