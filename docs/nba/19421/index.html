今日美國媒體《體育商業期刊》評選出了21世紀第二個十年的數個最佳獎項。金州勇士隊獲得了2010-2019年代最佳運動隊獎。

勇士隊官推隨後發布了一個視頻慶祝獲獎，視頻內沒有出現杜蘭特的身影。


格林隨後給杜蘭特打電話：sorry bro，你來之前我們就已經做好視頻了。

開拓者對陣太陽的比賽進行到第一節還剩2分12秒時，開拓者球員安東尼在場邊與裁判交流著什麼，隨後兩人最後握手致意。

安東尼：這場吹好點，我知道你家地址。裁判：好的大佬。

今天雷霆主場迎戰公牛，最終以109-106險勝對手。保羅表現出色，高效砍下30分9籃板8助攻2搶斷。其中在關鍵的第四節，保羅單節命中5記三分，幫助球隊實現26分大逆轉。

賽後普雷斯蒂找到打通了老闆電話：我們真的得交易保羅了！

雄鹿獨行俠比賽第三節最後0.3秒，字母哥執行罰球，雄鹿這邊換上了洛佩斯兄弟準備搶板，然後獨行俠這邊，立馬拿上了大博班+波神。
