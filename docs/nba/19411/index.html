1小時花掉100萬美金的奧尼爾給那些新秀們的忠告

2019-12-17 814條評論 3,133次閱讀 83人點贊

在今年的NBA選秀中，首輪30位新秀球員被球隊選中，而他們的新秀合同每年的薪水都超過了100萬美金，而第二輪30位新秀球員新秀合同每年的薪水則不低於聯盟最低工資標準56萬美金。

被選為狀元的球員薪水肯定要高一些，他們的工資標準取決於當年工資帽的多少。18年狀元艾頓和太陽簽了2年的合同，總價值高達1770萬美金，而第一年的薪水則是820萬美金。

名人堂球員奧尼爾，曾經也是狀元秀，他在一檔節目中對這些一夜暴富的新秀球員給出了他的忠告。這也是他希望告訴曾經20歲的自己的話，20歲的他曾經在1個小時花掉了1百萬美金。

「孩子們，假如說這是100美金」奧尼爾指著一張紙說道：「你想做的是把它分成兩半，50美金用來存起來，不要碰它，也不要看它，這樣你還有50美金可以花掉。但是聰明的人，比如說那些億萬富翁，他們則會把這50塊再分成兩半，一半存起來，另外25美金用來滿足消費。你可以用這四分之一的錢來買房子，買車子，買飛機或者是去旅遊。」

簡而言之，奧尼爾的意思就是說把75%的收入用來為將來退休打算，25%的收入用來日常開銷。

這也算是奧尼爾的切身體會和肺腑之言。在他20歲的時候，曾經一下子買了3輛賓士和眾多的珠寶等奢侈品，一個小時內花掉了100萬美金。

後來他遇到了一個理財顧問，這個理財顧問給他講：「你現在正值職業生涯巔峰，覺得掙錢是一件很容易的事情。但是你要知道聯盟有很多傢伙打不上球以後，就變得一無所有，我不想看到你將來也變成這樣。」

奧尼爾也是谷歌和蘋果的早期投資者，現在退役以後還擁有眾多的連鎖店，餐館以及洗車店等。
