狂轟32+5+8！字母哥天神下凡滅魔術，戈登狂打鐵，而富爾茨打瘋了

2019-12-10 704條評論 2,789次閱讀 909人點贊

在今天結束的一場比賽中，雄鹿主場以110-101勝魔術，豪取十五連勝。目前雄鹿戰績為21勝3負，和西部的湖人平起平坐。就看誰先輸球交出聯盟第一的寶座。今天雄鹿面對魔術還真是難纏，在末節魔術曾一度將分差縮小到6分，可米德爾頓再次成為了殺神，他關鍵時刻命中三分殺死了懸念。

本場比賽字母哥和米德爾頓的雙子星合力砍下了52+24+12的數據，字母哥32分15籃板8助攻，米德爾頓20分9籃板4助攻。

今天首節開始魔術倒是虎虎生風搶佔開局，富尼耶和富爾茨這爺倆一人一個中投先聲奪人。但雄鹿也立刻回應。字母哥持球衝擊上籃打進，大洛和馬修斯也連中2記三分。雄鹿開始追分。但富爾茨和富尼耶再次聯手貢獻2次上籃合砍4分幫助魔術領先2分。節末雄鹿突然下起了三分雨。

伊利索瓦連中三分球、迪溫琴佐也中三分，首節雄鹿以31-20領先。第二節，魔術的替補殺神羅斯開始發力，他和戈登班巴聯手發力縮小分差。

可雄鹿太厲害了，洛佩斯和馬修斯一人一個三分，又把分差拉開，魔術反而是落後了18分。節末富爾茨吸引防守，富尼耶三分飆進，之後又強打3+1。魔術半場追到了只差12分。

下半場雙方陷入了絞殺，這邊字母哥和米德爾頓飆進三分，那邊戈登也命中三分，富尼耶中距離再進回應。富爾茨也大開殺戒，他中距離再次命中，魔術不斷迫近分差。但洛佩斯總是可以在魔起勢的時候命中三分。字母哥也再次衝擊打進。三節雄鹿80-73領先。

末節艾薩克多次上籃命中，富尼耶也三分再中，羅斯強打3+1。魔術將分差又縮小到6分，可米德爾頓神兵天降三分命中完成收割。但魔術的表現依舊讓雄鹿印象深刻，他們差點就完成了翻盤。

全場比賽富尼耶砍下了26分是魔術表現最佳球員，富爾茨則大放異彩，他砍下了13分5籃板9助攻。在串聯方面富爾茨是真的厲害，9次助攻他讓富尼耶可以專心進攻，而不用去管組織。

而且比賽中，富爾茨多次突破展現了自己的爆發力，雖然幾次被帽了。但我們都知道，雄鹿的長臂怪極多，護框能力很強。富爾茨的表現已經給人留下了深刻印象，只是今天戈登手感不佳，12中3隻拿到10分，但貢獻了14個籃板。

進攻端戈登表現一般，防守和籃板倒是不錯。另外艾薩克在防守端令人驚艷，他得到了10分9籃板和5次蓋帽讓雄鹿吃盡了苦頭。

另外魔術還有一個奇兵，就是特倫斯-羅斯，他瘋狂砍下了23分，在過去魔術能重返東部前八的位子，不僅僅是因為富爾茨和富尼耶，羅斯也是居功至偉。目前魔術在東部第八，很有希望殺進季後賽。他們戰績11勝12負，和第九的活塞10勝14負拉開了距離。
