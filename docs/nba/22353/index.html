羅齊爾末節19分險造大逆轉！勒夫16+14+7騎士險勝黃蜂

2019-12-19 1,565條評論 3,797次閱讀 2532人點贊

NBA常規賽12月19日繼續進行，最終騎士主場以100-98戰勝黃蜂。

首節開始，騎士隊全民皆兵6-2開局。格拉漢姆三分但是奧斯曼里突外投連拿5分，騎士一波13-2拉開分差。隨後雙方大打對攻，你來我往，首節戰罷騎士以31-21領先10分。

次節上來，澤勒連續打成2+1縮小分差，騎士這邊波特和克拉克森合拿6分穩住局勢。隨後雙方瘋狂打鐵中艱難取分，羅齊爾連拿4分後騎士打出一波8-1再度拉開分差。澤勒連續上籃命中，黃蜂回敬一波7-2！但是湯普森、塞克斯頓又接連進球，半場戰罷騎士以53-41領先12分。

易邊再戰，騎士上來就是一波14-2領先到24分！！黃蜂奮起追趕回敬一波7-0縮小分差，勒夫三分穩住局勢。雙方開始瘋狂打鐵，羅齊爾跳投、補籃又搶斷得手助攻麥基吉拋投命中，一波12-2縮小分差。三節戰罷騎士以76-64領先12分。

末節開打，騎士先打出8-4拉開分差，黃蜂這邊羅齊爾和格拉漢姆連續飆中三分追至10分！互有得分後湯普森勾手命中，但是羅齊爾爆發連續飆中三分，黃蜂一波11-4追至只差5分！湯普森罰中2球穩住局勢！羅齊爾再飆中三分！！南斯罰中1球後羅齊爾又飆中三分！！！！最後21秒騎士領先2分！塞克斯頓傳球出界！！！羅齊爾投丟！騎士險勝！

黃蜂這邊，澤勒15分10板，羅齊爾35分

騎士方面，奧斯曼18分，塞克斯頓23分，勒夫16分14板7助，湯普森14分13板
