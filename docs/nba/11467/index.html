射落8記三分！支線劇情無礙利拉德扮演主角

2019-12-11 1,360條評論 4,747次閱讀 1161人點贊

12月11日，波特蘭開拓者坐鎮主場迎戰紐約尼克斯。

開拓者現已遭遇兩連敗，他們盼能重回勝利正軌，而剛巧此役造訪波特蘭的是身處九連敗漩渦的尼克斯。值得一提的是，本場還將迎來安東尼與舊主尼克斯的重逢戲碼，這也令此役的勝敗關係變得更具意義。

開拓者有意在進攻端放權安東尼，但安東尼開局狀態較為低迷，先是單挑巴雷特干拔失手，此後又出現掩護犯規。見狀如此，利拉德只能自己先行扛起進攻大旗，外線起手搭弓射箭正中靶心，並由此提振士氣逆改戰局。

這記三分就此啟動利拉德的遠攻模式，此後他攻得興起連投連進，用匪夷所思的三分火力瘋狂轟擊尼克斯防區。無可奈何的尼克斯只能被迫改變防守策略撲防外線，而利拉德則是見招拆招，驀地啟動突破晃暈防守再使出後撤步跳投，一通操作打得尼克斯越加發懵。愈戰愈勇的利拉德此時已完全無視任何防守，他在首節戰罷前再度發威三分續火，越投越遠，越投越難，還沒等尼克斯回過神就又是兩記三分貫空穿網！

利拉德僅首節就已射落5記三分斬獲17分，而開拓者也憑藉利拉德的瘋狂攻勢建立起兩位數的領先優勢。

尼克斯不甘就此潰敗，他們隨後掀起反撲盼能止住頹勢。可利拉德豈容尼克斯不服，他在重新披掛上陣後話不多說三分又進，令尼克斯徒呼奈何。迫於無奈，尼克斯只能不斷賠出犯規以阻斷利拉德的三分火力，而利拉德也笑納尼克斯的罰球厚禮，連罰連中毫不失手，帶領開拓者將領先優勢越拉越大。

眼看分差愈發懸殊，利拉德在易邊再戰後收斂鋒芒，切換傳球模式策動隊友火力，獲益於此的安東尼也接應利拉德妙傳連進兩球，舉隊的進攻火力壓得尼克斯難以喘息。三節打卡已是板上釘釘，可利拉德臨下場前還是耍起花活，迎著撲防飆起超高難度三分，而令尼克斯絕望的是，無論他們如何防守，利拉德還是能把球穩穩送進籃筐。

利拉德此役僅出戰三節就已命中8記三分並貢獻出31分6助攻的高光數據，憑藉其瘋狂火力成為開拓者在主場以115-87大勝尼克斯的首功之臣。雖說安東尼戰舊主是此役的主線劇情，但利拉德在波特蘭仍是毋庸置疑的第一主角。
