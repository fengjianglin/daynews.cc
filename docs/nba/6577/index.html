狂轟49+22+8，兩巨頭又打瘋了，威金斯你過分了，這麼欺負馬刺啊

北京時間11月27日，NBA常規賽繼續進行，馬刺坐鎮主場迎來森林狼的挑戰，這場比賽說是主場作戰，其實馬刺內心慌得一批，森林狼已經不是弱隊了，兩位狀元充分發揮天賦，把自己的特點打出來，雖然也不是超級強隊，但是能夠成為季後賽級別的球隊，這也算是不小的成功了。馬刺就不行了，這兩天球隊內部發生矛盾，德羅贊取關馬刺官方賬號，穆雷也開始逼宮，或許過不了多久，就要發生天大的交易。

​比賽開始之後，唐斯和阿爾德里奇開始互飈三分球，兩位大中鋒搶戲了，唐斯手感爆棚，再次打中三分並在內線強起得分，一下給馬刺打懵了，不過馬刺真是老油條，首節結束，馬刺居然反超2分。第二節，森林狼立馬覺醒了，連續用三記三分球打破僵局，這還不算完，威金斯繼續帶領球隊打出11-2的小高潮，唐斯打成2+1，半場戰罷，森林狼57-47領先馬刺。

易邊再戰，德羅贊終於開始發揮，連續拿到5分，不過格拉漢姆和卡爾弗的兩記三分，讓馬刺感到絕望，隨後馬刺小將突然站了出來，連拿9分之後，三節戰罷，森林狼只領先馬刺3分。第四節，關鍵時刻唐斯3+1驚醒馬刺，雖然此後仍然可以得分，但是威金斯也給出回應，里突外投拿到5分，最終，森林狼113-101擊敗馬刺拿下比賽，拿下兩連勝。

​數據方面，馬刺還是依靠雙德取分，其他人表現一般，奇兵沃克也只是幾分鐘的表現，阿爾德里奇22分6籃板，德羅贊20分7籃板5助攻，沃克11分，蓋伊10分11籃板，米爾斯8分，懷特9分6助攻。而森林狼方面，威金斯26分8籃板2封蓋，唐斯23分14籃板6助攻，卡爾弗10分5助攻，奧科吉11分8籃板，迪奧普16分4籃板。

​這場比賽又是森林狼兩巨頭的標誌性勝利，唐斯和威金斯狂轟49+22+8，又開始打瘋了，這個賽季只要兩人發揮正常，基本上就可以贏球。威金斯你過分了啊，趁馬刺病、要他們命，這麼欺負老邁的馬刺隊，確實讓人感到可惜啊，豪強就此淪落，波波維奇也不再神奇。
