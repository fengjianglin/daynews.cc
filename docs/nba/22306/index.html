都知道湖人隊目前的弱點是三分球，昔日總冠軍神射隨時待命

2019-12-20 350條評論 2,007次閱讀 471人點贊

目前，全聯盟最強的球隊就是雄鹿隊和湖人隊了，他們各自在自己的領域，也就是東西部，傲視群雄，其他球隊根本無法跟他們媲美，但是，也別小看了其他球隊，因此，當雄鹿隊和湖人隊的連勝紀錄被打破之後，所有人都在找尋他們的弱點。

雄鹿隊單核帶隊，哪裡最強，哪裡就最弱，都在字母哥身上，這一點不用多說，那麼湖人隊呢？湖人隊的弱點也很明顯，這一次，濃眉哥缺戰更是暴露了他們最大的弱點，那就是三分球，三分球成為了他們唯一的弱點。

據統計，本賽季湖人隊場均投中10.9個三分球，排名聯盟第22位，三分命中率35.7%，排名聯盟第16位。作為一個聯盟第一戰績的球隊，這項數據顯然是不及格的，甚至是非常糟糕，也就是湖人隊的唯一弱點。

不過好在，濃眉哥和詹姆斯的組合威力強大，掩飾了這些缺點，但是並不代表沒有，因此，湖人隊也迫切希望得到補強，而美媒建議湖人隊交易補強五大球員，分別是奇才隊的貝爾坦斯、馬刺隊的米爾斯、雷霆隊的施羅德、鵜鶘隊的雷迪克、尼克斯隊的莫里斯。

這五人，顯然都是三分好手，能夠補強湖人隊的三分弱點，但是交易這五人，對於湖人隊目前的陣容改變需要一些風險，而且也並不是那麼容易交易得到，因此，如果從自由市場上找尋合適的幫手，或許對於現在的湖人隊，更加友好。

例如昔日詹姆斯的帶刀護衛JR史密斯，都知道史密斯是一個神經刀，但是不否認的是，關鍵時刻史密斯三分准起來也是相當可怕的，甚至曾經是總決賽神射，幫助過詹姆斯奪冠，立下了赫赫戰功。

有消息稱，JR史密斯並沒有放棄，並沒有選擇退役，通過訓練師曬出了最近的訓練視頻，34歲的JR依然保持著相當出色的手感，在多個位置折返出手，仍然能夠穩穩命中。很顯然，史密斯還沒有放棄籃球。

看到魔獸霍華德和安東尼重回巔峰，想必JR史密斯也是渴望能夠回歸的，而如果詹姆斯招募JR史密斯，一點難度都沒有，比起五大引援目標，要好得多，現在，就看詹姆斯和湖人隊是否真的有意了。
