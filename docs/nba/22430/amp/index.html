42+4+7！還是輸給尼克斯，論演技，老鷹真是誰都不服

天天要聞
2019-12-18

北京時間12月18日，NBA又迎來了一個新的比賽日。雖然說今天的比賽場次不錯，而且比賽的球隊不能滿足所有球迷的口味，但不可否認，今天還是有一場很精彩的比賽。亞特蘭大老鷹隊客場挑戰紐約尼克斯隊，即使是兩支東部排名墊底的球隊，但他們在球場上奮力拚搏的精神依舊是很有看點的。沒有巨星球員，但他們有一群年輕熱血的優秀球員。

在兩支球隊的首節比賽中，作為主場作戰的老鷹隊就被尼克斯隊打了一個措手不及，並以尼克斯隊41-27拿到了了14分的巨大領先。不得不承認，尼克斯隊這支坐落於紐約的球隊還是有不少的地理優勢，但也不可否認他們球隊中個個得分都很突出。本以為在次節比賽中，特雷-楊能夠帶著老鷹隊迅速組織有效的反擊，但出人意料的是，尼克斯隊依舊以勢如破竹之勢再次兩領先的的優勢擴大，最終在半場比賽結束之後，尼克斯隊以77-53拿到了24分的領先優勢。

必須為他們點一個大大的贊，都是兩支東部墊底的球隊，尼克斯能夠這樣輕鬆地在半場比賽中拿到24分的領先，難道不是一種爆發的表現嗎？隨著易邊再戰，老鷹這邊依舊沒能及時組織有效的反擊。在第三節比賽中，雖說兩隊球隊一度出現了比分交錯領先的局面，但不可否認，還是尼克斯隊這邊保持了領先的優勢，最終在三節比賽結束，尼克斯隊又拿到了1分領先，最終以109-84領先25分進入到末節的比賽。

42+4+7！還是輸給尼克斯，論演技，老鷹真是誰都不服。對於有著這樣比分的領先，我們不能在說末節對於他們雙方都是關鍵的。相信比賽進行到了這裡，已經沒有了任何的懸念，紐約尼克斯隊勢必會拿下今天的比賽。而隨著第四節比賽時間的持續消耗，最終尼克斯隊以143-120拿下了今天的比賽。大勝23分對於尼克斯隊來說，絕對是新賽季接連敗績的最大收穫。

狂轟143分！尼克斯大勝23分，又一探花打瘋了，但贏家卻是老鷹。然而我們必須承認，特雷楊砍下42+4+7還輸球了，真的是有點假，成功擺爛成功，成為了了東部倒數第一的球隊，不得不說，他們的演技還真的強！但不得不說，尼克斯隊這邊的探花新秀巴雷特今天確實表現突出，投籃13中10，拿到了27+6+1的數據。
