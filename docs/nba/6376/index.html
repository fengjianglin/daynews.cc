31歲歷盡劫波的羅斯，到底歸來了嗎？

2019-12-02 0條評論 15次閱讀 0人點贊

德里克·羅斯，31歲了。

NBA生涯第十二年——如果不算他傷停的整個2012-13季，第十一年。

眾所周知，他的一切都來得早——無論是榮耀，還是傷痛。大一帶孟菲斯進NCAA全國決賽，惜敗給查爾莫斯那記三分球。

狀元身份進NBA。年度新人。第一場季後賽就是天勾以來最高的36分。帶公牛與衛冕冠軍凱爾特人打出本世紀最精彩的七戰系列賽之一（可能沒有之一）。三年級拿到史上最年輕MVP，帶一支傷病困擾的公牛拿到常規賽62勝。那年他止步東部決賽，與此同時，與他同年同屆的威少在西部決賽戰達拉斯。

2011-12季，羅斯倒下。威少打了生涯第一場總決賽。

2016-17季，羅斯在尼克斯度過自己第一個非公牛賽季時，29歲的威少拿到常規賽MVP。

那年秋天，羅斯說曾經的自己已經逝去了。

到此為止，羅斯和威少的故事，一個少年富貴卻遭遇挫折，一個血氣昂揚後發制人。

然後，又兩年過去了。

其實從數據上看，羅斯在2014-15季就已歸來：那年他為公牛出戰51場，場均18+3+5，季後賽還和勒布朗大戰過一個系列賽。下一年，66場，場均16+3+5。

之所以不顯，是因為終究還不如巔峰期的自己。

到上季，他在森林狼打出了單場50分。場均27分鐘里18+3+4。

然後是本季：場均24分鐘里16分5助攻。

往細一點看。

本季羅斯每36分鐘25分8助攻，命中率49%，真實命中率57%——他的MVP賽季是每36分鐘24分7助攻，命中率45%，真實命中率55%。

數據上，說羅斯正在打生涯最高效的進攻，也沒錯。

即，在31歲時，經歷漫長掙扎的羅斯，至少是數據上，打回來了。

上季羅斯在森林狼，投出了37%的三分率——實際上不止於此：16尺到三分線44%，10到16尺50%，籃下命中率是可怖的65%。

本季三分率33%，長中投42%、短中投50%，籃下命中率是可怖的72%。

發生了什麼？

羅斯與威少一樣，早年有一手急停中投。起跳高、出手點高，為了對抗而練的。

出手上有個不太協調處：下腰彎膝時，手已經早早抬起了。好處是出手點高，壞處是，視覺上看，從起跳到出手，有點脫節。

從上季開始，羅斯舉球的動作比較流暢了，下腰彎膝也沒那麼深了。

彎膝時球還在腰，起跳到最高點球托起，出手。

所以視覺上看去，起跳低了，滯空少了，但動作流暢得多。

於是遠射也穩了。

（順便，勒布朗本季的投籃，也有點這樣子……）

除了投籃姿勢外，另一個區別。

羅斯年少時，出了名的低重心。體現在視覺上，就是兩腿遛開來跑，真正飛毛腿。

平時慢悠悠，到需要時壓腿亮勢、大幅度拉球、大步搶身位突破、急停跳步扭身、壓膝強起、暴扣。

（這一點，他、埃文斯和沃爾師兄弟三人都一樣……）

MVP賽季，羅斯投中711個籃，其中32個扣籃。罰球投籃比35%。動如烈火，縱躍如飛，泰山壓頂。

MVP時期的他，跟年輕時的勒布朗與韋德有些像，「我們要比你高、比你快、比你壯，完全凌駕於你。」

上季，羅斯投中363個籃，其中只有1個扣籃。罰球投籃比21%。效率高了，打得巧了。

本季他場均9.6次擋拆完成進攻，佔總進攻56%，是全聯盟第三依賴擋拆的攻擊手（前面是巴里亞和奧古斯丁）。

每場14次突破，命中率是穩健的55%，但只有1.4次罰球。

如果您仔細看羅斯現在的打法，無論攻防兩端，他的大幅度壓膝蓋下重心的動作，都變少了。

叫個掩護，溜開腿，一個大幅度加速突破。跳起懸空若有對手阻隔，強行上籃變少了，更多是低拋給籃下隊友。

急停中投變少了，跨步拋射和騎馬射箭變多了。

減少了急停急起、強投強突、轟炸籃筐、貼人死防。

巧打，躲打，繞著打。

以及，這麼說很奇怪：

羅斯現在打球的方式……不那麼背負著一切了。

在受傷前，羅斯得分很多，但基本沒有「他打得不合理」、「他粘球」、「他自私」之類的評論。因為他梳理全隊，梳理得謹慎。

從大學開始，他就是個懂得掌握節奏、懂得找隊友、有視野的組織後衛。他每場都穩穩地打球，盡量協調全隊，分享球權，關鍵時刻自己接管，偶爾做出一兩個嚇死人的動作。吉米·巴特勒頭兩天還說，自己當年訓練時看羅斯做的那些事，嚇呆了；結果到比賽里，羅斯還是能把那些招式使出來。

之前也提到過：在芝加哥受傷前，羅斯其實沒打自己喜歡的套路。

他大一帶孟菲斯進全美決賽那年，打的是DDM體系——dribble drive motion，「持球突破動態進攻」。這玩意說白了就是空間拉開，持球突破，需要能投籃的長人。網隊當年打這個體系時，成全了哈里斯生涯巔峰賽季，易建聯單季射進48個三分球（他NBA生涯五年一共就進了75個）。

但羅斯在公牛拿MVP那年，身邊布滿的是博甘斯、鄧、布澤、吉布森和諾阿。那年公牛唯一靠譜的速射手是科沃爾。

所以那年公牛的打法說白了：全隊防守搶籃板，羅斯一個人單挑突擊。

那時羅斯打得很猛。防守端，他單防基本是全覆蓋施壓打法（巔峰期德隆和巴朗也這樣）；進攻端，一個人帶一群內線。

科比在那兩年夸人有鬥志，就舉例羅斯和保羅——我們都知道科比所謂的好勝，標準有多高。

那就是MVP時期的羅斯：攻防都用盡全力，卻又打得無私而正確。2011年東部決賽敗給熱，也沒法子：不是他打得不對，實在是能為還沒到那份上，被勒布朗收了。

從上季開始，羅斯開始順應時代潮流了。

時代需要擋拆和三分，好，那就擋拆三分吧。

隊友拉開了，持球擋拆突。

這個時代的空間拉開，遠好過他MVP的年份。他也明白這點。

以前他會更多看隊友的切出，給隊友製造傳球線路，現在簡單了：拉開空間，溜開腿，突破。

這就是過去兩年羅斯的打法：

他不再攻防兩端、組織協調、接管得分，每一回合都侵略如火，認真地背負芝加哥公牛了。

他現在的打法，很地道的第六人，像個職業槍手。「給我球，給我個掩護，拉開空間，我給你們分數」。

以前那個肉搏時代，他光芒萬丈，拼到受傷。到這個刷分相對容易的時代，他也看開了。

2016年11月，羅斯說，那個vintage mvp self，過去了。他指的，是2011年夏天，還不知何謂命運磨難的羅斯。

那時他飛得那麼高，不知道世上還有什麼可以阻擋他。

頭兩天接受芝加哥那邊採訪時，羅斯說他現在打球，為的是自己和自己的家庭。他說在最絕境的時候，只有家庭是站在身邊的。

他說現在時代不同了，現在有「負擔管理」了。他說如果他當年有負擔管理，也許……也許他依然在芝加哥公牛健康打球呢？

當然，他一再說：過去的，已經過去了。

31歲了。德里克·羅斯數據很好看，甚至效率高過MVP賽季，溜開腿時依然迅速。

但那個巔峰期侵略如火、全力以赴、隨時隨地都背負一切的羅斯，其實沒有歸來。他只是多少看清了時代，於是換個路數打球：更方便、更順應潮流，也更健康。

這樣也好。

輝煌過的受難者，轉型會更痛苦。除了身體的傷痛，他們還得放下自尊，盡量拋棄已有的套路。

轉型失敗的就消失了，湮沒無聞；轉成功的也未必就完全快樂了。年少成名的痛苦就在這裡：之後的人生，每一步都在走下坡路。

所以羅斯會一邊說「都過去了」，一邊忍不住念叨「如果當初」。

但具體行動上，他到底還是在努力認真地打球，而且打得很好——了解他所經歷一切的人都知道，這比什麼都重要。
