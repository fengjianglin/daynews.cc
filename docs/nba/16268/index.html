東西部並駕齊驅，相遇之前誰輸誰丟人？

2019-12-14 325條評論 1,475次閱讀 443人點贊

北京時間2019年12月14日，結束的兩場比賽，湖人客場迎戰熱火，雄鹿客場挑戰灰熊，東西部榜首，均有驚無險的拿下了這場比賽的勝利，但是贏得並不輕鬆，比賽結束後，兩隊再次以一樣的戰績繼續領跑東西部，這讓球迷們不禁期待，誰先掉隊？或者以同樣的戰績直到兩隊相遇嗎？

在接下來的賽程中，湖人繼續客場迎戰老鷹和步行者，這場比賽雖然都是連續客場作戰，但是對手並不強勢，加上湖人最近勢頭正盛，應該可以以全勝的戰績備戰20日的東西部榜首之戰。

東部榜首雄鹿，在14日面對灰熊之後，迎接連續兩個主場，分別對戰騎士和獨行俠，對戰騎士應該可以輕鬆拿下，但是最近手熱的東契奇領銜的獨行俠，雄鹿想要拿下比賽也不會太輕鬆，好在都是主場迎戰，尤其20日碰撞湖人，也是主場，所以雄鹿的贏面還是比較大。

20日即將發生的東西部榜首之戰，也被譽為總決賽的演練，縱觀兩隊的打法，實際上是很不一樣的，湖人整體整容較高，有著霍華德、麥基、戴維斯等絕對實力的內線球員，在進攻端多以內線突破為主，配合格林、卡魯索、隆多、詹姆斯的外線和傳導進攻能力來發起進攻，值得一提的是，戴維斯，霍華德加上隆多，都是防守悍將，導致湖人隊的防守非常強悍，可以說攻防兩端配備都非常齊全，多以防守然後轉快攻來得分。

反觀雄鹿，則以字母哥內線單核帶隊，進行拉扯，為外線拉開空間，之後米德爾頓、布萊德索等球員進行外線進攻，為主，如果字母哥遭遇單人防守，則很快進行單打，突破進攻，要麼得分，要麼迫使對方犯規造成殺傷。

值得一看的是，鋒線上的當家球星，4號位的對決，戴維斯和字母哥，究竟誰強誰弱，將會在20日的比賽中揭曉。

要論陣容的完成程度和豪華程度，顯然湖人更具備優勢，但是畢竟客場作戰，究竟「詹眉連線」強勢，還是「希臘怪獸」更勝一籌，孰強孰弱？12月20日我們拭目以待。
