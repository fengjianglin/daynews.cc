NBA2KOL2|爆裂猛獸-德安德烈·喬丹

2019-11-27 0條評論 18次閱讀 0人點贊

在夏天的休賽期的交易市場中，布魯克林隊在迎來了凱里·歐文和凱文·杜蘭特的加盟，同時也迎來一位非常強硬的內線球員加盟，他就是德安德烈·喬丹。

猛獸「小喬丹」

1988年出生的德安德魯·喬丹在2008年通關選秀進入NBA，以第二輪第五順位被洛杉磯快船隊選中，也是在快船隊，與當時正值巔峰時期的「白魔獸」布雷克·格里芬組成當時震懾內線區域的雙人組，再加上同樣也是處於巔峰時期的「CP3」克里斯·保羅的穿針引線，當年的洛杉磯快船隊也被球迷們冠以」空接之城「的稱號。

在休賽期的交易市場中，小喬丹以自由球員的身份加盟了布魯克林籃網隊，他場均能為球隊貢獻8.2分，10.4個籃板和1.2個蓋帽的數據。「小喬丹」有著非常不錯的身體條件和爆發力，但是他的進攻手段比較單一，多為快攻跟進扣籃和二次進攻，他的罰球也是比較糟糕。不過他的面框防守能力也是不錯的，觀眾們都能多次在比賽中看到小喬丹在給對方進攻球員上演蓋帽好戲，以他的身高，體重以及爆發力的優勢，足以能夠讓他成為一個能夠震懾內線的蓋帽高手。

在2K Online2的遊戲世界裡，小喬丹也是能夠經常在街頭模式或是王朝模式見到他的身影。在滿突破+羈絆的加成下，小喬丹的能力值能夠達到94的數值，同時他還擁有7個徽章加成，2個扣籃徽章+5個防守徽章。作為一名藍領型內線，小喬丹的防守能力非常不錯，他的進攻籃板和防守籃板分別能夠達到92和117，他的原地上籃和扣籃數值很高，分別達到了96和115。他的防守數值比較平均，不過他的運球和控球的數值很低，加上運球速度和橫向移動速度比較低，不推薦持球進攻，不然很容易被斷球能力值高的防守球員斷球，推薦擋拆配合之類的打法。

接著來看看他的持球突破扣籃，由於速度數值及運球速度數值非常低的原因，從動態圖我們也能夠看得出來，是非常不推薦拿小喬丹進行持球推進的，不過他的扣籃動作觀賞性也是比較不錯的，大風車扣籃等一些比較有觀賞性的扣籃動作也是能夠做出來的。

然後時他的投籃動作，投籃常用手為左手，雖然從投籃動作能夠看出出手手感柔和，但是小喬丹的各項投籃數值都非常低，且他的常用進攻手段多為原地上籃和扣籃。所以不推薦進行中遠投投籃嘗試，偶爾能夠蒙進的情況也非常得小，在三秒區內可以嘗試下，不過要在空位的情況下。

最後來看看他的低位進攻，雖然也不是常用進攻手段，不過在打錯位的時候也是可以使用的一種進攻手段，加上小喬丹的力量數值也達到110了，但是用來做一些例如沉底步或是背身後仰的動作還是比較困難，偶爾可以用來進行低位勾手的，成功率也是比較大的。

在遊戲中小喬丹也是可以作為平民陣容的選擇之一，滿突價格為800W左右，作為一名面框能力不錯的藍領球員，小喬丹在街頭或是王朝模式中也有很多人使用，小編目前也是常用小喬丹拿來打街頭，無論是護框還是拼搶籃板方面，也是非常不錯的一名球員。
