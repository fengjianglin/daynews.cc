雄鹿防守悍將布萊索因腓骨骨折休戰兩周，球隊能否延續勝利？

2019-12-15 788條評論 1,934次閱讀 498人點贊

就在密爾沃基雄鹿以125-108戰勝克利夫蘭騎士的幾分鐘後，球隊宣布了首發控衛埃里克·布萊索的不幸傷病消息。

雄鹿隊的公關部門透露，布萊索在昨天對陣孟菲斯灰熊的比賽中右腓骨撕脫骨折，在一次誤傷中，隊友韋斯利·馬修斯撞上了布萊索，他的膝蓋與布萊索的脛骨發生了強烈接觸。

布萊德索本賽季場均15.0分，5.7次助攻，場均上場時間只有26.7分鐘，他再次成為本賽季聯盟最佳防守的主力。

埃里克·布萊德索在周五晚上右腿受傷。經MRI檢查，為右側腓骨撕脫骨折，預計他將缺席大約兩周。

密爾沃基雄鹿隊已經連勝18場，這是球隊歷史上第二長的連勝。他們正以24勝3負的戰績迎來球隊歷史上最好的賽季開局，但現在他們將面臨一個艱難的賽程，因為他們的一線隊全守備陣容將缺乏完陣。

在對陣騎士隊的比賽中，二年級的後衛東特·迪溫岑佐在本場25分鐘得到15分、5個籃板、4次助攻和3次搶斷，並沒有讓人失望。

身高6英尺4英寸的迪溫岑佐在後衛位置上的身材並不差，他在防守端的出色控球能力讓他能夠給對手製造混亂。

雖然他已經看到了明顯的進步，從他的新秀賽季。迪溫岑佐在2019- 2020賽季的三分線外命中率高達35.3%，而上賽季他的三分線外命中率只有26.5%。

也許最重要的是，他得到了主教練的信任，在本賽季對陣克利夫蘭騎士的比賽前，他首發了10場比賽，代替了因大腿挫傷而缺席了很長時間的米德爾頓。

鑒於布萊索將缺席兩周，我們有理由認為，他能否在2019年餘下的時間裡上場還是個問題，而接下來的幾場比賽算得上是強強碰撞，能否延續勝利真的很難說。
