感恩節到了，30支球隊應該感謝點什麼（上篇）

2019-11-29 0條評論 13次閱讀 0人點贊

今天是感恩節假日，NBA休戰一天，藉此機會，我們來盤點一個多月打完，30支球隊應該「感謝」點什麼？本系列分三篇發，每篇10支球隊。

亞特蘭大老鷹：冷麵殺手特雷

楊是老鷹重建的領導人，在一支沒有競爭力的球隊里，球迷沒什麼比看到一顆年輕新星的升起更開心的了。楊目前場均26.6分4.5板8.7助攻，還有一些瘋狂的連續運球畫面，以及過半場就扔的超遠三分表演。也許楊今年還不是A級球員（防守太糟糕了），但未來遲早有機會做到。

波士頓凱爾特人：丹尼-安吉

就在大家看起來，以為凱爾特人就當安東尼生涯末期的收容所時，交易專家安吉唯一的增員，是用類型相似的沃克頂替了歐文，選擇繼續用布朗+塔圖姆+斯瑪特三人組，用來堵住球隊各種空缺。不行你看，凱爾特人新賽季又是歡天喜地的12勝4負開局，在東部又可能搞出點動靜了。安吉一直以來都受到粉絲們的詬病，但只要凱爾特人能繼續贏球，風評如何早就無所謂了。

布魯克林籃網：暢想2020-21賽季

在這個時間點，看看是時候告訴過度熱情的歐文黑（凱爾特人球迷，尼克斯球迷），杜蘭特明年就要加入這支球隊了。但目前球隊的困境——歐文受傷，勒夫特受傷，9勝9負，歐文和在凱爾特人一樣，又和球隊有摩擦——籃網球迷需要期待的，就是2020-21賽季的無限可能。

夏洛特黃蜂：老實說？什麼都沒有需要感謝的

黃蜂的工資單，依然是現代NBA里最滑稽的開支——巴圖姆一年2,500萬？比永博一年1,700萬？馬文-威廉姆斯1,500萬？太慘了。而最慘的，還是他們休賽期和凱爾特人實質上的控衛互換，用3年5,670萬買了個替補，換掉了一個全盛時期的控衛。自從球隊名字從山貓換回黃蜂后，沃克可能是球迷唯一愉快的記憶了，然後球隊為了給羅齊爾一年2,000萬，把沃克送走了（沒給羅齊爾頂薪，可能只是因為球隊薪金空間扛不住了）。

芝加哥公牛：吉姆-博伊倫的到期合同

紙面上看，公牛表現應該更好的。他們去年前18場是4勝14負，今年是6勝12負。公牛陣容明顯提升了（科比-懷特，賽迪斯-楊，薩托蘭斯基，一年完整的波特，更好的替補），但博伊倫是個很爛的教練。事實上，只要博伊倫能得到公牛管理層加-弗曼和約翰-帕特森的鼎力支持，已經能說明他肯定不是主帥位置的合適人選了，但好消息是，本賽季結束後他的合同就要到期。

克利夫蘭騎士：約翰-貝勒明

總有人需要來收拾爛攤子，而騎士是非常幸運的，他們找到了一個66歲的新秀主帥，用來調教這群完全沒開發出潛力的新星們。貝勒明是21實際第一個10年的大學賽場里，最負盛名的主帥。他讓騎士球員在場時更努力了，打球速率更快了，溝通更順暢了，也確實讓球隊變得更好了。戰績沒有完全反映出來，但騎士主帥建立的文化，對重建非常重要。

達拉斯獨行俠：盧卡已經成長為巨星

盧卡爆發是個意外？錯了！這名斯洛維尼亞的控衛，只差0.5次助攻就能達成場均30+三雙（30.1+10.0+9.5），而且帶著獨行俠排在西部第四。別忘了這只是東契奇的二年級，他也只有20歲。身邊擁有一個重新恢復生機的波神，是個合格的吃餅人選，讓東契奇身邊的隊友資源和上賽季不可同日而語，這個小伙也確實能利用到全新的獨行俠進攻空間，打出超新星的表現。

丹佛掘金：休賽期沒有過度的動作

休賽期的其他西部球隊都有大動作，嘗試去爭搶突然空出來的西部老大王座，掘金…選擇繼續賭自己之前的那套陣容？好吧，掘金就是這樣做的。他們休賽期沒有做任何一筆亮眼的操作，開局的打法，依然強調著2018-19賽季的成功之道。掘金節奏依然很慢（連續兩年都是第27），防守效率繼續進步（從第10變到第1），限制對手得分依然頂級（對手場均得分從第6低變到最低）。在13勝3負的開局後，沿用之前的比賽方針，看起來是球隊正確的選擇。

底特律活塞：高三分命中率

去年悲慘的季後賽被橫掃之旅後，本賽季活塞又是6勝12負開局，簡直就是NBA中段班球隊的代言。但是今年活塞有一個異常數據，他們的三分命中率是聯盟第2。活塞三分出手不多，但是命中率高達39%，其中球隊老大格里芬只打了5場球，三分命中率24%。活塞沒有別的什麼擅長，投好三分是好的開始。

金州勇士：廣告標語，回憶，更多廣告標語

勇士球迷，當你今年在嶄新的大通中心看到主隊被吊打時，托著下巴就好了。今年看不到這支球隊進總決賽了，希望能有一個樂透簽來彌補大家的傷痛。
