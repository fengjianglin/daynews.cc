布朗26+5+5 塔圖姆26+5 德拉蒙德12+11 綠軍輕取活塞

2019-12-21 2,451條評論 4,684次閱讀 3250人點贊

直播吧12月21日訊 凱爾特人今日坐鎮主場迎來活塞，首節開局兩隊狀態相當一直膠著，不過第二節綠軍展現出了強大的實力，依託著主場濃烈的歡呼聲，凱爾特人開局不久便打出一波14-2的攻勢迅速拉開比分。後半節面對活塞的追趕，布朗站出來連飆三記三分，包括最後時刻的一記3+1，凱爾特人也藉此守住12分的領先進入下半場。第三節前半段活塞還能憑藉穩定的進攻咬住比分，但隨著後半節攻勢回落，場上局面被綠軍徹底接管，末節開局一波9-2過後，凱爾特人確立起27分的領先讓比賽早早失去懸念。最終，凱爾特人114-93輕取活塞拿下比賽。

雙方數據

活塞（11-18）：德拉蒙德12分11籃板4助攻6搶斷，大莫里斯10分6籃板2封蓋，米凱柳克12分2籃板，羅斯14分4助攻，加洛韋10分2籃板，梅克15分7籃板。

凱爾特人（19-7）：布朗26分5籃板5助攻3搶斷，塔圖姆26分5籃板3助攻，威廉姆斯18分3籃板，坎特8分18籃板，泰斯8分6籃板4封蓋。

首節開局活塞連續在外線尋找機會，開局前三次出手全部是在三分線外並命中前兩記遠投，不過隨後幾次進攻活塞不是失誤就是打鐵，綠軍則在之後逐漸接管比賽，奧傑萊和塔圖姆相繼命中三分迅速反超比分，活塞的進攻依舊沒有改變，除了馬基夫再度命中三分外，其他的幾次進攻都無功而返，綠軍前半節表現只能全是中規中矩，因此分差沒有拉的太大。後半節隨著替補輪換的上來，活塞逐漸進入正軌，羅斯連拿4分幫助活塞追分，活塞也在塔圖姆的帶領下堅守領先，梅克最後時刻命中三分，活塞將劣勢縮小至2分結束第一節。

次節開局梅克的2+1打成一度完成反超，但這1分的領先都還沒有捂熱，綠軍便迅速掀起反擊攻勢，凱爾特人一眾替補不斷衝擊籃下打出一波14-2的攻勢迅速反超比分。後半節雙方逐漸回到僵持狀態，米凱柳克和布朗接連命中三分為活塞追分，綠軍這邊布朗手熱回敬兩記三分。末段加洛韋再度命中三分，然而布朗的手感還在繼續，一記更加精彩的3+1來結束上半場，半場戰罷，綠軍60-48領先活塞。

休息回來，斯內爾在外，德拉蒙德在內，兩人聯手拿分幫助活塞繼續追分，但綠軍這邊勢頭不減，泰斯接連禁區拿分讓活塞毫無辦法。對攻之中活塞進攻表現並不差，米凱柳克更是續上火力連中三分，奈何防守才是癥結所在，他們進球的隨後一個回合總是被打反擊，布朗手感還在命中三分，塔圖姆連拿5分穩守兩位數領先。後半節活塞連僅存的進攻也開始回落，綠軍則在泰斯的2+1進球後再掀進攻高潮，布朗連續製造犯規命中4記罰球，加上格蘭特-威廉姆斯最後的撤步跳投進球，凱爾特人在第三節結束時取得了22分的領先。

末節開局凱爾特人繼續掌控比賽，坎特的補籃和威廉姆斯的三分為綠軍繼續擴大領先。活塞的整體攻勢仍然未見好轉，羅斯的跳投進球是活塞前5分鐘里僅有的進球，而綠軍期間一波9-2的小高潮繼續擴大領先。加洛韋的三分為活塞稍稍止血，但多達25分左右的分差已經讓比賽早早失去懸念。之後雙方几乎都是替補作戰，比分也沒有太大的變化。最終，凱爾特人114-93輕取活塞拿下比賽。

雙方首發

活塞：布魯斯-布朗、米凱柳克、斯內爾、大莫里斯、德拉蒙德

凱爾特人：沃克、布朗、塔圖姆、奧傑萊、泰斯
