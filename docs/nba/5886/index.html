5個狀元一台戲，這比賽看得過癮，膝蓋流血了還要打，你是個漢子

2019-12-09 0條評論 18次閱讀 0人點贊

北京時間12月9日，NBA常規賽繼續進行，湖人主場大勝森林狼迎來4連勝。

本場比賽可謂是眾星雲集，同時聚齊了21世紀以來的5個狀元。2003年的詹姆斯，2004年的霍華德，2012年的戴維斯，2014年的維金斯和2015年的唐斯。

比賽過程也如我們期待的那樣大開大合，雙方瘋狂對攻，湖人得到142分更是他們自1989年以來的常規賽單場最高分（不包括加時），這比賽看得過癮。

戴維斯和詹姆斯繼上一場合砍70+得分後，本場再次合砍70+，也成為了科比和奧尼爾之後的首對連續兩場比賽合砍70+的組合。霍華德5投3中得到8分6板的數據也是可圈可點，很好地完成了自己的任務。

相比於湖人的3位狀元，森林狼雙狀元今天的表現就差得多了，兩人均得到19分且整場比賽幾乎都被湖人壓制，看來他們跟前輩的差距還是有些大啊。

本場比賽除了5個狀元，發揮最亮眼的當屬禿曼巴卡魯索，登場30分鐘11投6中，3分球3投2中得到16分4板4助2斷的數據，很好的彌補了隆多缺陣帶來的影響。

第二節比賽還剩8分40秒時，卡魯索在一次對抗中膝蓋受傷流血不止，就在大家都以為他會在暫停時被換下場時，他卻表示根本沒那個必要。經過短暫包紮後就重新返回了賽場，而此時鮮血已經滲透了他的護膝，不得不說，卡魯索真硬漢也。

本場比賽後，湖人將迎來一波東部5連客，分別對陣魔術、熱火、老鷹、步行者和雄鹿，這對於他們來說是一場不小的考驗。

熱火、步行者和雄鹿都不是好惹的，希望湖人能夠繼續為我們帶來精彩的比賽。
