2020屆籃球名人堂榜單出爐，WNBA兩名女職業球員入圍候選名單

天天要聞
2019-12-21

北京時間12月20日，奈史密斯名人堂公布了2020屆首次入選的NBA巨星球員名單，其中前NBA球員來自湖人科比·布萊恩特、馬刺蒂姆·鄧肯、凱爾特人凱文·加內特、以及熱火克里斯·波什四人入選名單，而其他進入候選名單中的球員有太陽肖恩·馬里昂、小牛（現獨行俠）邁克爾·芬利、以及兩名WNBA球員塔米卡·卡欽斯、斯溫·卡什入選。

2020屆名人堂

科比·布萊恩特—湖人

科比

科比的一生都貢獻給了洛杉磯湖人，而正是這一名桀驁不馴的小子，給湖人帶來了5座總冠軍獎盃，其中1999-2002賽季科比聯手奧尼爾實現湖人的三連冠，2208-2010科比聯手加索爾成就湖人2連冠，數據方面生涯得分33643籃板7047助攻6306，20年職業生涯1346場比賽中，場均上場36.1分鐘可以貢獻25.0分5.2籃板4.7助攻以及1.44搶斷，生涯5次總冠軍，2次FMVP，1次MVP以及2枚奧運金牌。

蒂姆·鄧肯—馬刺

鄧肯

鄧肯的職業生涯一生貢獻給了聖安東尼奧馬刺，這一名被譽為「石佛」的聯盟大前鋒，在技術以及身體方面非常的出眾，帶領馬刺取得5座總冠軍獎盃，19年職業生涯總計1392場比賽，獲得26496分15091籃板4225助攻，場均出場34.0分鐘可以貢獻19.0分10.8籃板3.0助攻以及2.17封蓋，鄧肯職業生涯奪得5次總冠軍，3次FMVP，2次MVP。

凱文·加內特—凱爾特人

加內特

加內特的職業生涯始於森林狼，後轉戰凱爾特人、籃網、終於森林狼，其中加內特也是聯盟史上極具出色的大前鋒，21年職業生涯出場1462場比賽，生涯貢獻26071分14662籃板5445助攻，場均出場34.5分鐘可以貢獻17.8分10.0籃板3.7助攻以及1.4封蓋，生涯1次總冠軍、1次MVP、以及1枚奧運金牌。

克里斯·波什—熱火

波什

波什職業生涯始於猛龍，終於熱火，波什的生涯是稍有遺憾的，而遺憾在於波什本來正值當打之年，卻經歷了罕見的球員傷病問題，最終只能選擇退役，職業生涯13年出場893場比賽，總計得到17189分7592籃板1795助攻，並在熱火時期收下2次總冠軍，場均貢獻19.2分8.5籃板2.0助攻以及1.0封蓋，生涯獲得2次總冠軍以及1枚奧運軍牌。

候選名單人員：肖恩·馬里昂，職業生涯場均上場34.5分鐘，可以貢獻15.2分8.7籃板以及2.0助攻、1.3封蓋，職業生涯共獲得1次總冠軍，入選4屆全明星；邁克爾·芬利職業生涯場均上場34.5分鐘，場均貢獻15.7分4.4籃板2.9助攻以及0.9搶斷，生涯獲得1次總冠軍、2屆全明星。

另外今年格外引人關注的是首次提及兩名女性職業籃球運動員上榜，其中卡欽斯被評選為美國女子職業籃球賽歷史上最優秀的15人之一，卡什被譽為女版「詹姆斯」，曾多次獲得WNBA總冠軍，是一名技術非常全面的女性籃板球員。

對於本年度NBA提名的奈史密斯名人堂球員，候選名單中馬里昂以及芬利值得大家的討論，而兩名女性運動員被提名，意味著女性的地位同樣值得被尊敬。
