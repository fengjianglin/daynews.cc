瘋狂1扣+3記大帽！超人霍華德熱血回歸，生涯封蓋數越2米31長人

2019-12-17 3,628條評論 4,936次閱讀 803人點贊

北京時間12月16日，湖人101-96擊敗老鷹，豪取7連勝！他們在客場已經取得14連勝。德懷特-霍華德替補出場，得到2分8籃板1助攻1搶斷3蓋帽。本戰過後，霍華德的生涯蓋帽數達到2088個，超越2米31中鋒曼紐特-波爾，升至歷史蓋帽榜第15位。

目前排在第14位的是布拉德利（2119個蓋帽），第13位是大本（2137個蓋帽），如果霍華德保持目前的勢頭，他有望在本賽季超越這兩位前輩。

霍華德這場比賽只命中一個運動戰進球，但這一球卻是毫無疑問的10佳球級別。第二節結束前2分2秒，詹姆斯殺入禁區，好似腦後長眼，突然一個胯下擊地傳球，霍華德跟進接球飛身暴扣。

這一扣，讓全場沸騰起來，不禁讓人想11年前霍華德在扣籃大賽的表演。當時的他，身披超人斗篷，也有一記如此瘋狂的劈扣，讓人印象深刻。一瞬間，讓人感覺那個無所不能的超人又回來了！

霍華德不再是當年的霍華德，他也不再是球隊的核心，但現在的他，更適合這支湖人隊。他在角色球員的位置上貢獻著自己的力量，只要他的名字被叫到，他就會在球場上全力投入，為球隊的勝利拚命，而完全不在乎個人的數據。

進攻端，他不再是球隊的主要選擇，但是防守端，只要他在場上，他就要努力成為球隊禁區的屏障。對老鷹一戰，霍華德送出三記大帽！

第二節剛開始1分3秒，霍華德就蓋掉了瓊斯的上籃！

37秒後，又是霍華德釘板大帽帕克的上籃！

到了第四節，萊恩想要隔扣霍華德，魔獸可不答應，送上了他的第三次大帽！

這樣的霍華德，湖人怎麼能不喜歡呢？別忘了，他簽的只是一份無保障的合同，而且是霍華德自己要求的。不為別的，就為了證明他可以在湖人隊里實現自我救贖，也可以成為湖人隊衝冠的重要拼圖。
