大清洗啊，湖人三少要崩，為了擺爛，鵜鶘對他們下手了

天天要聞
2019-12-22

湖人三少創業未半而中道崩殂。

可能哥幾個會問技巧君，

鵜鶘不是只對霍樂迪動手嗎？

怎麼湖人三少就崩了呢？

不好意思，

鶯歌，安好。哈特，無恙。

但是球哥，要走人了！


據名記Brian Windhorst在播客節目中透露，

如果報價合適，

鵜鶘願意送出後衛郎佐-鮑爾！

本賽季球哥場均10.0分4.7板5.5助1.1搶斷，

命中率37.1%，三分33.1%，罰球56.0%。

瞅瞅這投籃命中率，三分命中率，罰球命中率，

不交易球哥交易誰啊。

練了這麼久，依舊沒有長進。

這位被寄予厚望的榜眼註定是要和投射無緣了。

話說新奧爾良似乎不怎麼受射手歡迎，

以至於霍勒迪在不久前才破了

三分命中數的記錄，

超越佩賈（553）獨佔隊史榜首！

這祖上是真的窮！

論鵜鶘的外線炮台，

遠的講最讓人影響深刻的是佩賈，

但那時候球隊還不叫新奧爾良鵜鶘，

而是叫新奧爾良黃蜂！

往近的講，能扛起射手大任的就只有

圓臉登和安德森吧，

一個在馬賽克隊當起了超級炮台，

另一個則是越投越迷茫。

而今年引進的JJ雷迪克算是外線上

一個突破型進展吧，

但是最近他也和霍樂迪一起陷入了被交易的流言。

不過說到底，

球隊戰績差不能把鍋全摔在球員身上啊。

鵜鶘如今能成為「擺爛大軍」中的佼佼者，

主教練金特里貢獻不少。

一支球隊好比一盤棋，

球員這顆棋子走不走得好還得看持棋人，

所以棋走臭咯總不能分鍋的時候拿球員來湊。

可金特里後台硬啊，

同為擺爛大軍，

人家尼克斯主帥大衛-菲茲戴爾為了

避開春運過於擁擠，

已經提前買好票回家陪妻兒了。

金特里卻愈發光彩，穩如泰山，

在球場揮斥方遒，

在為球隊爭奪狀元簽的事兒上，

那叫一個不遺餘力！

所以從目前來看，

鵜鶘就這麼個情況，

除了金特里、鶯歌和錫安，

誰都會隨時走人！湖人的哈少也不例外！

而球哥無疑是話題性最大的人物，

畢竟是湖人三大榜眼中混得最差的一個。

鶯歌現在朝著大合同進發。

拉塞爾雖然受傷後狀態有所起伏，

但好歹也是戴著「勇士大腿」的標籤啊，

打得也是美滋滋。

只有球哥沒有打出外界的期望。

也不得不說球爹的包裝能力之強，

前期在陣勢和噱頭上毫不含糊，

把自己兒子打造成未來巨星，

到頭來卻給了球迷極大的落差。

在擺爛大隊里還表現得如此掙扎，

球哥也是時候正式自己，

或許新奧爾良不屬於自己的舞台，

沒必要戴著「湖人三少」的名頭硬扛到底。

拉塞爾和鶯哥已經證明了湖人的眼光還不賴，

技巧君也相信球哥早晚有一天，

也能打出屬於自己的天賦！
