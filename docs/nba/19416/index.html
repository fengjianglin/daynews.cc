布克談艾頓復出：他已經坐不住了，全隊都準備好了

2019-12-17 686條評論 4,576次閱讀 812人點贊

虎撲12月17日訊 太陽中鋒德安德烈-艾頓此前的25場禁賽期即將結束，今天太陽眾人談到了艾頓復出在即。

主帥蒙蒂-威廉姆斯說道：「德安德烈回來以後，我們會制定相應的比賽計劃。顯而易見，我對於他即將復出感到很開心。」

「我認為艾頓復出能讓所有人的上場時間恢復正常，但願這樣的話大家都能恢復最好狀況。」

德文-布克說道：「他已經坐不住了，全隊都準備好了，這是令人激動的時刻。他缺席的每一場比賽，我們都在默默支持他，我知道這很艱難，因此他對於即將復出已經迫不及待了，我們對於即將與他並肩作戰也有同樣的感受。」

「我們對此非常期待，很顯然，這會給我們帶來完全不同的動力，此前我們一直缺乏這方面，但也有許多人站了出來，我們都很期待艾頓復出。」

米卡爾-布里奇斯說道：「你會從錯誤中吸取教訓，他之前犯下了重大錯誤，現在能做的就是吸取教訓，有所成長。我知道這會讓他更加成熟，等他回來，他會變成一頭野獸。」

「他很激動，一直在說禁賽還剩多少場，我們為他感到開心，迫不及待想看到他回來。」

艾頓將在明天對陣快船的比賽中復出。

(編輯：窩瓜)
