這怕是近年來，勇士隊所遭受的最大羞辱，水花不在，勇士隊真不行

2019-11-29 0條評論 18次閱讀 0人點贊

勇士隊近幾年那可真是風光無限啊！統治了聯盟好幾年，雖然不是每年都拿冠軍，但是要是論絕對實力的話，勇士隊那是當之無愧的聯盟最強，死亡五小也是打遍聯盟無敵手。但是本賽季由於勇士隊陣容的變動和水花的受傷，他們現在已經毫無當年之勇，甚至有些虎落平陽的凄慘之意。

昨天他們以104比90拿下公牛後，可把公牛隊教練組氣壞了，公牛教練組取消了球員感恩節的假期，給隊員們進行了加練，因為在公牛教練組看來，球隊輸給聯盟墊底的勇士隊這是他們所不能接受的，甚至是有一些恥辱的。但是我們大家反過來想一下，公牛隊因為輸給勇士隊而感到恥辱，這讓勇士隊情何以堪，這對於勇士隊來說是不是更大的恥辱呢？當初聯盟其他球隊是贏下勇士隊為榮，輸給勇士隊屬正常的心理態度，而現在呢？現在聯盟的其他球隊，是贏下勇士隊為正常，輸給勇士隊為恥辱的心理態度。這還真應了那句：「落魄的鳳凰，不如雞：虎落平陽，被犬欺」的千古名句啊！現在勇士隊想擺脫這種恥辱，唯一的辦法就是等水花兄弟攜手歸來，朋友們，你們說呢？
