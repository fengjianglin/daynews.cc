無問西東之：聖安東尼奧的輝煌與窘迫

2019-12-17 48條評論 3,689次閱讀 2303人點贊

聖安東尼奧馬刺隊是一支位於美國德克薩斯州聖安東尼奧的職業籃球隊。其前身達拉斯灌木叢隊於1967年成立並加入美國籃球協會（ABA），1973年搬至聖安東尼奧並改名馬刺隊，1976年加入美國男籃職業聯賽（NBA）。現從屬於NBA西部聯盟的西南賽區。馬刺隊是在1976年ABA與NBA合併後保持完整的原ABA聯賽的四支球隊之一，也是原ABA聯賽中唯一一支奪得NBA總冠軍的球隊。1976年以來，馬刺隊共計獲得5次NBA總冠軍，6次西部冠軍，以及21次賽區冠軍。是除波士頓凱爾特人、洛杉磯湖人、芝加哥公牛外奪冠次數最多的NBA球隊。馬刺隊史只有四次未能進入季後賽，馬刺隊利用四支樂透簽相繼選中了埃爾文·羅伯特森（1984年第7順位）、大衛·羅賓遜（1987年狀元）、肖恩·埃利奧特（1989年第3順位）和蒂姆·鄧肯（1997年狀元），以及從弗吉尼亞紳士隊買入了21歲的喬治·格文，當然還有我們的巴特爾，其中羅賓遜和鄧肯總共為球隊帶來了5座總冠軍獎盃！鄧肯更是被譽為NBA歷史最佳大前鋒！

「冰人」的時代（1976–1985）

喬治格文被譽為「冰人」，1977-1978賽季，格文在常規賽最後一場比賽得到63分，從而鎖定得分王，並在其後兩個賽季蟬聯，實現得分王三連莊，雖然球隊都進入了季後賽，但始終沒能突破西部。1984-1985賽季後，喬治格文被交易到了芝加哥公牛，格文時代終結，馬刺也進入了重建。

「海軍上將」與「石佛」的傳承（1989-2003）

1989年，剛剛服完兵役的大衛羅賓遜加盟聖安東尼奧，加上前幾個賽季重建中確定的以拉裡布朗為主教練和主教波波維奇的教練組，馬刺重返強隊行列！大衛·羅賓遜在賽季的每個月都拿下了月度最佳新秀，常規賽場均24.3分和12.0個籃板的數據在隊內均排名第一！1994-1995賽季，大衛·羅賓遜更是場均27.6分、10.8個籃板、3.2個蓋帽，榮膺常規賽MVP併入選最佳防守陣容一陣。但是，馬刺還是沒能突圍西部！直到1997-1998賽季，史上第一大前蒂姆鄧肯的降臨，馬刺從此翻身！1998-1999賽季，由鄧肯和羅賓遜組成的雙塔開始發威，歷史上首次獲得NBA總冠軍，鄧肯以27.4分、14個籃板和2.2次蓋帽當選為總決賽MVP。2002-2003賽季，聖安東尼奧再次捧杯，鄧肯更是包攬常規賽和總決賽MVP!賽季結束後，「海軍上將」正式退役！馬刺開啟GDP時代！（2001賽季托尼帕克加盟馬刺，2002賽季馬努吉諾比利登錄聖安東尼奧，只是在當時還不是絕對的主力球員）

三駕馬車（2003-2016）

隨著帕克和馬努的成長，加上穩定的」石佛「，馬刺的三駕馬車開啟了新一輪的冠軍爭奪，被譽為GDP時代！2004-2005賽季，馬刺以GDP為核心，輔以布倫特巴里/羅伯特霍利/納茲爾穆罕默德/格倫羅賓遜，老中青結合的陣容過五關斬六將，最終經過7場血戰再次登頂！鄧肯第三次拿下總決賽MVP（儘管大家都覺得馬努應該拿那個MVP才完美）！2006-2007賽，馬刺4-1淘汰掘金，4-2越過太陽，4-1輕取爵士挺進總決賽，同初出茅廬的小皇帝會試總決賽，並兵不血刃的橫掃騎士奪冠，帕克成為首個外籍總決賽MVP，鄧肯更是留下一句」未來是你的「後昂首離開!2013-2014賽季，老邁的馬刺在波波的調度下全民皆兵，常規賽更是沒有一人的場均出場時間超過30分鐘，總決賽上馬刺又碰上賽季從自己手中搶走總冠軍的熱火（詹姆斯熱火時代），這一次聖安東尼奧人沒有再次空手而歸，他們以5場凈勝70分的優勢4-1淘汰熱隊，再一次捧杯，卡哇伊成為第一個90後總決賽MVP！

後GDP時代

2016年7月11日，鄧肯宣布退役；2017-2018賽季後，帕克加盟黃蜂，馬努宣布退役；從此，三駕馬車告別馬刺，GDP時代最終結束！馬刺進入雙德時代，隨著2018夏季倫納德出走多倫多，德羅贊加盟馬刺，聖安東尼奧仍保有雙德（阿德+贊德），馬刺成為了一支平民球隊！本賽季的馬刺已經明顯追趕不上其他球隊的步伐，在強手如林的西部再難佔有一席季後賽之地，本賽季至今以9勝15負居12位，季後賽堪憂！但聖安東尼奧人沒有停下追趕的腳步，一步一步往前走，一場比賽一場比賽地打，他們的精神」石佛「回來了（助教），他們白髮蒼蒼的波波還在堅守，他們球館高掛的5面冠軍旗幟在飄揚！
