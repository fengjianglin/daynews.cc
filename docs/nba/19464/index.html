觀察｜新奧爾良鵜鶘，新賽季交出一份糟糕答卷

2019-12-17 115條評論 1,114次閱讀 77人點贊

寫在前言：休賽季的鵜鶘和聯盟大部分球隊一樣，「改變」是形容這支球隊的關鍵詞。從戴維斯決定離開的那一刻起，鵜鶘坐了一趟從季後賽二輪球隊到樂透區球隊的「過山車」。總經理鄧普斯被迫下台，接任他的是大衛-格林芬，在拿到狀元簽並選中錫安的那一刻，本以為球隊挑選了一個能夠決定球隊下限的球員，夏季聯賽緊接而來的受傷讓鵜鶘的前景再次變得富有戲劇性，新賽季到目前為止，遭遇12連敗的鵜鶘更是交出了一份糟糕的答卷。

戴維斯離隊，球隊開始更新換代

戴維斯先是將經紀人換成里奇-保羅，這位極具商業頭腦的經紀人，試圖為戴維斯的生涯做出一次重大改變。在戴維斯通過里奇-保羅發聲決定離開新奧爾良的時候，總經理鄧普斯顯然沒有做好充分準備工作，交易截止日期過後，戴維斯仍然在鵜鶘隊內，使得這支球隊和戴維斯都陷入一個尷尬的境地。

鵜鶘新星

由於鵜鶘保護戴維斯這一交易資產，他的上場時間遭遇嚴格限制，這樣帶來的結果就是主場充斥著球迷的噓聲，更衣室氣氛的緊張以及被外界忽視的蘭德爾與霍樂迪的出色表現，黯然失色的鵜鶘艱難結束上賽季的比賽，同時他們流失了逐漸兌現天賦的蘭德爾。

戴維斯的帶隊能力確實是外界質疑的地方，雖然是大前鋒位置上最出色的球員，但是在鵜鶘的7個賽季里只率隊打入兩次季後賽，而他與考辛斯組成的雙塔陣容也伴隨著考辛斯跟腱受傷無疾而終。憑心而論戴維斯與鵜鶘分道揚鑣從某一個層面上說也是雙贏的局面，戴維斯追求更高層次的目標，基於前幾個賽季的表現，鵜鶘也可以打破圍繞戴維斯展開的重建，去找尋另外一個能夠決定球隊下限的天賦型球員。

湖人「三少」在新奧爾良到底過得怎麼樣？

郎佐-鮑爾，遠不如湖人表現出色

鮑爾的腳踝成為阻礙他奔向成功的最大隱患，在湖人2個賽季只打了99場比賽，如此低頻的出場率很難想像這是一個剛進入聯盟不到三年的新人，在對陣籃網的比賽中鮑爾後場持球推進，在無對抗的情況下右腳踝連續扭傷兩次，隨後鮑爾被迫下場接受治療。雖然並沒有選擇休戰，但是對於鮑爾來說從湖人到鵜鶘，腳踝傷病成為羈絆他前進的原因之一。

鮑爾右腳踝無對抗扭傷

在鵜鶘征戰的19場比賽來看，鮑爾對於球隊攻防兩端並沒有明顯的提升，反而受限於投射能力的短板，在場時讓球隊的進攻變得更加複雜。雖然他有出色的進攻梳理能力，但是得分能力的不出眾再加上缺少詹姆斯這樣能吸引防守注意力的隊友，讓他在場上並不能造成很大的進攻威脅，反而損失場上進攻空間。

鮑爾的優勢在於能夠打快速攻防轉換，上次我們提到他與庫茲馬之間能夠形成非常好的默契配合，庫茲馬出色的運動能力，就能將鮑爾在轉換進攻的優勢就能體現出來，尤其是四分衛長傳和行進間快速轉移球防守。但是這一切的基礎立足於防守，新賽季鵜鶘防守效率114.9場均失分120.1分，這兩項體現的防守數據都排在聯盟倒數第二位，甚至比上賽季的湖人還要糟糕，必須承認一部分原因來源於戴維斯和蘭德爾的離開，使得球隊整體高度和護框下降，進而導致球隊防守力不足，費沃斯和奧卡福的護框能力的確很差，兩人防守效率都在115以上。但是在湖人以防守著稱的鮑爾，本賽季的防守效率也下降到了116.0，這是鮑爾職業生涯首次防守效率跌到110以上。

鮑爾四分衛長傳

沒有基礎防守和高強度的壓迫持球人去逼迫對手犯錯，鮑爾的優勢自然就得不到施展的空間，再加上一直困擾鮑爾的糟糕投籃手感（新賽季37.4%的投籃命中率僅比新秀賽季好一些），原本缺少戴維斯後內線就是一大劣勢，更加崇尚空間打法的鵜鶘，鮑爾目前的狀態顯然不能讓金特里滿意。他在場上並不能給對手製造很大的殺傷力，金特里也沒有圍繞他去建立戰術，近幾場比賽鮑爾掉出了首發陣容。

鮑爾

鮑爾也是急切需要證明自己的球員，他在場上並沒有打出比湖人時期更好的進攻表現，原本外界稱讚的大局觀、把控場上節奏的能力和聰明的打球方式，都沒有在鵜鶘展現出來。在場上他更多的是利用突破去尋找外線隊友，或者行進間找到處在更好得分機會的隊友，對於他來說這種打球方式終究是有缺陷，一旦鮑爾的投籃能力得不到提升，就無法利用自身進攻威脅去創造隊友進攻空間，他的上限終究會被限制，對於鮑爾來說，儘快豐富自己的技能包才是眼下最重要的事情。

哈特：「哈學友」打出精彩表現

被湖人球迷親切稱為「哈學友」的哈特，在鵜鶘打的還算不錯。作為一個30號選秀順位的球員，對於自身定位非常清楚，就是向3D方向發展。來到鵜鶘後哈特場均能夠打出11.5分6.0籃板的數據，他的防守積極性在鵜鶘整體裡面是非常好的，但是我認為防守習慣還需要改善，從湖人到鵜鶘都能夠經常看到他在對手身上吃虧，幾次讓對手造成2+1或者3+1的犯規。新賽季16.4%的回合佔有率甚至比湖人還要多，一方面金特里本身就會使用後場球員，哈特在他手下打出了職業生涯目前為止最出色的表現，另一方面要承認在後場球員里，哈特現在比鮑爾要出色。

哈特

此外哈特的傳球能力也在慢慢的提升，哈特其中一個優勢點在於他的突破，尤其是攻防轉換和對手防守立足未穩的情況下，他能夠利用自己的身體素質殺入內線索取得分，雖然新賽季他的場均助攻並沒有很大的起色，但是從場上表現來看，哈特對於突破分球的時機掌握越來越熟練。對陣活塞時哈特不看人秒傳海耶斯的那一球，真的體現了能突破能傳球的能力，現在哈特正逐漸向第六人發展。

哈特秒傳海耶斯

對於哈特並不需要寄予太高的期望，他的天賦確實不如霍勒迪和鮑爾等後衛球員，但是依靠他出眾的身體素質和球場上的態度，他贏得了金特里的信任，希望哈特能夠成為鵜鶘穩定的第六人。

英格拉姆:正兌現自身天賦

首先我們要明確一點，這個賽季處在英格拉姆的合同年。對於鵜鶘來說，530萬的合同打出全明星級別的表現，無疑是戴維斯交易籌碼中最值得球隊驚喜的球員。這個賽季英格拉姆的表現堪稱現象級。本賽季至今英格拉姆場均能夠貢獻25.0分7.0籃板3.7助攻，成為鵜鶘隊內為數不多能夠持續貢獻穩定輸出的球員。

最直觀的就是三分命中率的提升，湖人時期英格拉姆的三分投射能力並不出色，但是加盟鵜鶘以來，英格拉姆卻成為隊內最穩定的三分點之一，在揭幕戰開賽後的首周，聯盟內僅有9位球員在回合佔有率超過29%情況下，能夠保持60%以上的真實命中率，而那份榜單中最年輕的就是英格拉姆，新賽季的首周英格拉姆的個人數據就呈現井噴式的爆發，而他能夠佔據一席的原因就是，夏天苦練換來的三分球提升。在湖人的三個賽季英格拉姆終結進攻的方式更傾向於殺向籃下和依靠自身臂展優勢的中距離投籃，三分球對於他來像一個難題，不到迫不得已不會選擇出手三分。以至於前三年他場均命中三分數還不足一次，總共386次三分出手僅換來127次命中，在現在的小球時代，英格拉姆可以說是一股「清流」，到了鵜鶘以後，更注重空間進攻的他場均能夠命中2.3記三分球，甚至比上賽季的場均出手次數還要多。

英格拉姆兩個賽季對比

除了三分球以外，他在進攻端的選擇也逐漸豐富起來，過去英格拉姆能夠利用他的臂展和先天的身體天賦，大跨步殺向籃下或者面對防守人干拔中距離投籃，但是現在我們能夠看到他在持球擋拆、單打和背身投籃等方面的結合運用，本賽季定點投籃佔比總進攻的22.3%，每回合拿到1.26分，而上賽季僅佔比13.0%，每回合只拿到0.88分，體現他無球的提升，持球方面單打佔比16.9%，上賽季這個進攻僅佔比14.2%，產量與效率都在相應提升。在持球和無球打的轉換中，英格拉姆正逐漸提高自己的進攻表現，這樣帶來的影響是既能增加自己進攻選擇，不至於被對手提前預判站在進攻路線，也能帶動場上的隊友參與進攻，為隊友創造進攻機會。

英格拉姆

英格拉姆的表現無疑是湖人「三少」最為出色的，一方面得益於他的天賦，2米21的臂展搭配身高足可以讓他在面對大部分防守人時，都能佔據高度優勢保證投籃減少干擾。另一方面金特里對於他的使用顯然比沃頓更好，減少他的運球次數和增加進攻方式，讓他無論在陣地戰還是轉換進攻中，都能依靠自身優勢打出現象級表現，讓英格拉姆終於在合同年開始兌現自身天賦。

英格拉姆與金特里

霍勒迪對於球隊的意義

現在有消息稱霍勒迪正在被鵜鶘擺上貨架，這位曾被鵜鶘視為建隊核心的球員，新賽季以來的表現並沒有進一步得到提升，反而得分、籃板、助攻和投籃命中率方面都呈現下滑現象。29歲的霍勒迪本應該處於職業生涯黃金期，但是他在場上表現出一定的掙扎，助攻率、失誤率都有不同程度上升，真實命中率更是從上賽季的55.5%下降到52.3%，創造近五個賽季的新低。

霍勒迪

徹底打破重建對於鵜鶘反而更有利於培養年輕人，另外英格拉姆在比賽中展現的精英級表現，讓球隊萌生出交易霍勒迪的想法，大膽設想一下，給予英格拉姆球權的前提下，他會達到怎樣的高度呢？對於爭冠球隊來說，霍勒迪的打法能夠極快融合陣容之中，他高水平的防守能力更適合在季後賽乃至總決賽的舞台上進行展現，本賽季鵜鶘防守效率雖低，但這是霍勒迪無能為力的事情。對手大可以避其霍勒迪防守鋒芒，選擇防守弱點進行瘋狂攻擊，從而抵消霍勒迪在防守端的優勢。

霍勒迪持球

對於29歲的霍勒迪來說，他沒有多少時間等待年輕球員成長，沒有機會去陪伴一支球隊完成再一次的重建之旅，如果霍勒迪能夠交易到一支極具競爭力的球隊來說，能夠為鵜鶘換來一些年輕籌碼，這是一場雙贏的交易，另外對於潛在下家球隊來說，霍勒迪的加盟不需要球隊進行大範圍的戰術調整，他能夠保持穩定輸出，無球和持球方面他能擅長，霍勒迪的未來是去還是留我們拭目以待。

錫安，或許並不能作為球隊的核心優先順序

對於鵜鶘和錫安本人來說，夏季聯賽的受傷都在意料之外。但是我們從杜克時期來看，今年兩次的重大受傷顯然已經有傷病隱患信號，他擁有與身高並不相符的體型，讓他的膝蓋承受超負荷的衝擊力和支撐力，顯然不符合通向偉大球員的身體發展規律。

雖然他被稱為下一個「詹姆斯」，但是從傷病恢復來看，原本預計的傷病恢復日期向後拖延，在接受半月板手術後錫安至今歸期未定，或許這個賽季我們不能看見錫安在NBA的正式首秀，另外英格拉姆的年齡和發揮具有一定的優勢，在一個不能保證健康的新秀和兌現天賦的新生代球員之間，鵜鶘未來核心選擇或許並不是錫安，考慮到這是英格拉姆的合同年，在他能夠持續打出精英級表現的前提下，鵜鶘很可能給出一份頂薪合同，並且優先於培養英格拉姆為核心組建陣容。

錫安受傷

回顧2016屆選秀球員，西蒙斯和穆雷都拿到了自己的頂薪合同，英格拉姆還在質疑與期望聲中前進，或許加盟鵜鶘會成為他職業生涯的轉折，雖然現在來看新賽季球隊戰績過於糟糕，但是對於英格拉姆來說，及時兌現天賦才是他證明自己的立根之本。

鵜鶘擁有全聯盟最為羨慕的潛力新星，在一切天賦兌現前總要遭遇一段黑暗日子，相信過程才能有望成為聯盟強隊，這份「糟糕」的答卷，是每個重建球隊必須要經歷的。（文/路威的體育世界）
