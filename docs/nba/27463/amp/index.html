豎中指！罵娘！這位NBA球星與球迷在場外幹上了

摘要：是可忍孰不可忍？（歡迎關注空口有評NBA）

撰文|空空&編輯|楚琦

在北美四大職業聯賽中，NBA的看台距離球場最近，場邊的觀眾，甚至可以聞到球員身上男性荷爾蒙的味道，看清他們的毛孔。

這是好事，也是壞事。

說它好，是因為這樣的設計，讓球迷能近距離接觸球星，更有代入感；說它不好，是因為球迷和球員距離太近了，為一些不必要的言語、甚至是肢體衝突，埋下了隱患。

最典型的例子，就是由一杯啤酒引發的奧本山宮殿群毆事件，如果觀眾距離球場沒有那麼近，這杯啤酒就扔不到阿泰臉上，或許這一事件根本就不會發生。

因為這事，不少人挨了重罰，NBA從此加強了對球員的管束，嚴禁他們和球迷起衝突，但對球迷，聯盟似乎一直沒有太多辦法。

今年三月，在猶他爵士的主場，一位男性球迷沖著威少大喊。

“給老子跪下，就像你們（黑奴）過去那樣。”

六月的總決賽，坐在場邊的勇士小老闆馬克-斯蒂文斯，更是直接上手，推了猛龍後衛洛瑞一把……

類似的例子還有很多很多。

沒錯，斯蒂文斯被罰了50萬美元並禁止現場觀賽一年，那位出言不遜的爵士球迷更是被終身禁賽，可這似乎都不能阻止下一個球迷跳出來，攻擊球員。

空哥曾經看過一個視頻，視頻中，一位坐在騎士替補席斜對面的男性，不斷挑釁詹姆斯，問他什麼時候能不再哭鼻子了，詹姆斯聽見後，讓他閉嘴，結果那球迷來勁了，BB個不停，十分得意，好像在向同行的朋友們炫耀，你看，連詹姆斯都我惹毛了。

和前面的情況不太一樣，這位兄台段位更高，他嘴裡沒嘣出一個髒字，雖然弄得人很煩，但你還真拿他沒轍。

更不公平的是，同樣的話，球迷對球員說或許沒啥，但如果反過來，球員輕則罰款，重則禁賽。

所以很多時候，面對球迷的挑釁、甚至是羞辱，球員們只能憋進肚子里，裝作沒聽見。

但NBA球員也是人，有的球員就忍不了，比如說奇才隊的小托瑪斯。

小托瑪斯雖然個子很小，但非常要強，絕不是逆來順受的主。

今天，在客場挑戰費城76人的比賽中，他就因為兩位球迷越界的言行，徑直走上了觀眾席，口頭教育了這兩位球迷。

在NBA，球員走上觀眾席是大忌，最終，在初步了解事情經過後，小托瑪斯和兩位球迷均被罰出場，目前NBA正在調查此事，後續很可能會對涉事雙方追加處罰。

這兩位球迷究竟做了什麼事，逼得小托瑪斯冒險走上看台？

事情是這樣，在第四節還剩2分53秒時，奇才落後十幾分，小托瑪斯走上罰球線，結果兩罰一中，現場一片嘆息聲。

原來，在76人主場，有個由連鎖快餐店溫迪（Wendy’s）贊助的活動：如果對方球員連續罰球不中，那麼現場觀眾第二天就可以憑球票，免費到連鎖店領取一小杯聖代。

就是一小杯冰淇淋嘛，本來也沒啥，但有兩名球迷不幹了，在小托瑪斯執行完罰球回防的路上，這兩人沖著小托瑪斯雙手豎起中指，連續罵了三句”F**k you，b**ch”。

據場邊其他觀眾回憶，當時，這兩人身上有酒味，明顯喝多了。

在小托瑪斯看來，你平時怎麼罵都行，但沖著他豎中指罵娘這種事，過分了。

“在我們老家，沒人敢用這個詞罵人。”

於是，在隨後的暫停期間，小托瑪斯突然走上了看台，當時，那兩位球迷還在起鬨。

“我們的溫迪聖代怎麼辦。”

小托瑪斯徑直來到那兩位球迷跟前。

“別做這種事！你球迷就要有球迷的樣子，別做這種事。我是球員不假，但我首先是個人。”

按照小托瑪斯的說法，那人顯然沒料到會發生這麼一幕，所以支支吾吾地道歉了。

“對不起，我們只是想要聖代。”

在小托瑪斯走向球迷時，現場安保試圖把他往回拉，但小托瑪斯示意他自己沒別的意思。

“我只是得和他四目相對，讓他知道，你不能不尊重我。我不傻，我不會衝上看台揍人，我不會這麼沒教養。”

小托瑪斯認為，如果是在大街上，在光天化日之下，他倆絕對不敢這麼罵他。

“我在聯盟里也混了很久了，球迷們想說什麼都行，但別指名道姓地罵我，因為我從來不會對你這麼干，我覺得這個行為越界了。”

“我有孩子，有家人，別對著我罵那個B打頭的詞。”

“我都不知道他們有什麼好不高興的，當時我們落後15分，最後輸球的也是我們。”

事後，小托瑪斯的隊友C.J.邁爾斯連發數推，聲援小托瑪斯。

“你們一面侮辱成年人，一面因為他們受不了，說他們太脆弱，有很多話，如果你在街上對著陌生人，絕對不敢說，噴垃圾話和噓聲我們都能理解，但這也有界限。”

“有人說，因為球員賺得多，所以這都不是問題，不應該小題大做，但這種理論其實很扯淡。”

“很多人臆想說’如果我能賺那麼多錢，那我什麼都不在乎，什麼問題都能迎刃而解’，但在現實生活中，不是這麼回事，錢的確可以解決很多問題，但絕對不是萬能的。”

“說實話，這種人如果真有錢了，會變成啥德行，我想都不敢想。”

小托瑪斯說，他打了這麼久NBA，還是頭一回碰到這樣的事，他知道，接下來聯盟肯定要調查此事，但他始終覺得，自己沒做錯。

“我沒有不尊重任何人，作為一個男人，我只是走上前，看著他眼睛，告訴他這麼做不合適，我沒有嚇唬他，嘴裡也沒有嘣出過一個髒字。”

“所以如果聯盟調查起來，我會重複上面的話，希望他們能理解。”
