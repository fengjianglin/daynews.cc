「驚喜之人1」雷霆的後衛男團，猛龍的運動達人

2019-12-24 75條評論 1,214次閱讀 191人點贊

開始一個小系列，打算把30支球隊本賽季最驚喜的球員，結合著比賽寫一下——很難說這個系列具體在寫比賽復盤，還是球員本身，但把球員的功能性和閃光點放在具體的比賽里，會比單獨寫大樣本發揮要更生動一些，所以你也可以把這個系列看成重點在於球員的比賽拼盤系列。

​

猛龍的運動達人們

哈，猛龍這賽季最大的驚喜難道不是范喬丹+西亞卡姆+OG阿奴諾比這組未來建隊核心已經挑起大梁，堪當重任？

那就不算驚喜了，這哥仨兩個賽季前就已經不是泛泛之輩，當萊昂納德和丹尼格林離開猛龍，球隊開啟了後總冠軍時代開始，你幾乎就可以料定這哥仨必然要開啟一段不尋常的奮鬥歷程，意外只是，一切來得還挺順暢，角色升級帶來的壓力，看來沒有難倒猛龍這些順位低但天賦不俗的年輕人們。

好吧，納斯狠操的球隊三大支柱——西亞卡姆、洛瑞和范弗利特——本賽季都遭遇了傷病，小加也要休息，陣容深度遭遇的考驗可不輕。但到目前為止，傷病還沒有打垮這支球隊，深度本是休假期對猛龍預測最為薄弱的環節，結果納斯又從板凳末段掏出了秘密武器，靠著本來不在計算內的生力軍，幾次打出了極為閃耀的奔跑、跳躍、激情的屬於年輕人的籃球——多倫多猛龍，在48分鐘的時間裡提醒你NBA的籃球賽不是半場攻防，攻擊和防守半徑都應該在28米x15米尺度而不是14米x15米，雖說全場緊逼和強推快攻都不是新鮮東西，但像猛龍今天這樣，可以把兩樣武器玩出終極殺招的效果，從而逆轉乾坤填平30分的，真的不多。

在猛龍與獨行俠的比賽前，你當然不會猜到這樣難以置信的比賽過程——獨行俠第三節差不多就把猛龍打花了，77已經給文兒打了電話，花生米都帶上等出門了，結果猛龍第四節干出47比21埋葬了獨行俠，文兒只能把燙好的一壺二鍋頭自己幹了。

但有這樣一件事我們還是可以想得到：

獨行俠已經用三分球射死了雄鹿和76人，即使沒有東契奇，他們可怕的5-out全拉開，萬箭齊發的威脅就在那裡。獨行俠的持球人又是那樣穩健，球隊傳導球流暢順滑，失誤率聯盟最低，而猛龍是全聯盟最熱切的主動改變防守陣型的球隊之一，他們既給持球人壓力，又籃下嚴密協防，造失誤和護筐全都要，代價當然是漏掉全聯盟最多的三分球。

猛龍的防守與獨行俠的進攻，是一場賭局——獨行俠的穩健持球+流暢傳導球+大量射手布局與猛龍最自信的防守侵略性+全員護筐+極致外線輪轉之間的賭局。對猛龍來說，這賭局之險，就像絕無神硬接無名的萬劍歸宗，即使對面少了東契奇就像無名叕不滿血，自己沒有西亞卡姆和小加，那當然也不在最佳狀態。

所以，當獨行俠前三節射出37中15，40.5%的三分命中率，而猛龍造失誤的成績單是10次時——按猛龍平時的水準，三節應該造12~13次，對手三分又進了這麼多，造失誤還需要更多才行——猛龍當然吃不住了，而他們自己的進攻又平平無奇，他們沒有西亞卡姆，范弗利特和伊巴卡也不在狀態，比賽交代了才應該是正常的劇情。

替補陣容改變了局面。

納斯的熱血陣容、激情防守又拿出來了——回想一下長人林滅大帝那一次——即使沒有西亞卡姆，猛龍還是能擺出能量滿滿的五人組，由防守最有激情的小個後衛洛瑞率領，布歇和朗戴衝鋒陷陣，特倫斯-戴維斯和馬爾科姆-米勒從旁幫襯，猛龍開始絕地大反擊！

所以，逆轉的理論基礎在哪？

理論基礎就是，猛龍用拉滿的防守能量，讓獨行俠整體運動能力偏弱，沒有東契奇缺少硬解法的毛病再一次暴露了。雄鹿和76人是防守策略更加保守的中鋒深蹲類型，他們不給侵略性，更專註保護籃筐，這是獨行俠相對喜歡的防守類型——包括猛龍的伊巴卡，這支球隊少數防守覆蓋面積只有一畝三分地的球員，也是獨行俠相對不怕的類型——但當凱爾特人拿出單防換防壓迫性，特別是祭出小陣容時，獨行俠就打不動了，猛龍這次，細節不同，但原理相似。

猛龍更加激進，他們從後場發球開始全場緊逼——比賽還有一節時間，這樣守體能難道不出問題？

沒問題，因為除了洛瑞，這些球員都是替補，都是猛龍最有比賽能量的生力軍，布歇和朗戴是可以追著每一個進攻者奔跑，瘋子一樣的防守者：

布歇，就像他的名字一樣，根本「不歇」。他會的東西真的不多，但就是一股子用不完的體能+修長的身材+加玩命的起跳，球場的每一個區域都可能出現他的封蓋，進攻端就是視死如歸的往裡沖，拚命搶板，前場籃板率超過同位置98%的球員！

布歇順下天賦強吃，欺負布羅科夫

朗戴-霍利斯-傑弗森，不是很高，但臂展極佳，在籃網就有打5號位與對手大塊頭肉搏的經歷，往裡能扛，往外可以纏鬥後衛，真正從1號位防到5號位的戰士型球員，把防守能量覆蓋28米x15米的跑不死的硬漢。他身高一般，還缺少射程，但他是全聯盟里，極少數具備了這兩個特點，還基本沒拖累球隊進攻的配角球員——極有能量的空切，勇猛的前場籃板沖搶，並有著遠高於一般配角球員的處理球能力，可以在需要的時候展示一些低位技術和傳控能力——一個有一丁點輔助技能的絕對近戰輸出。

朗戴的空切

獨行俠怕的，就是這種運動能力的全面碾壓，當猛龍完成全場覆蓋的防守壓力後，獨行俠有些慌了——一場普通常規賽，他們感受到的是總決賽級別的防守拼勁：

事實上，猛龍和獨行俠就是全聯盟風格最極端的兩支球隊，猛龍對攻防轉換的強調是極致的，而獨行俠是一支典型的陣地戰球隊，他們推反擊缺快馬，退防不差倒不是因為退防速度快，而是失誤極少，並且內線經常站高位回得來。

結果猛龍把獨行俠喜歡的，雙方一字排開，老老實實打紳士籃球的節奏破壞了——猛龍就是搶跑封+沖莽投的土匪打法。

猛龍的快攻搶投

即使落到陣地，猛龍替補陣容的能量小子們，依然能讓獨行俠步履維艱——沒錯，獨行俠第四節的三分準星糟透了，這裡有運氣的成分，但看看猛龍的撲防吧，布歇的長臂難道沒有干擾到獨行俠的三分手嗎？

飛身冒小庫里的三分

距離冒到賴特就差一個指甲蓋

沒有東契奇的獨行俠，面對猛龍的運動天賦型防守陣容，開發進攻變得無力了。

到處是布歇（25號），到處是朗戴（4號）的身影——雖然跳來跳去會撲空，但他們前赴後繼，而這就是納斯要的瘋狂的防守。

猛龍的翻盤，最大功臣還是洛瑞——他用奇蹟般的三分硬解，幫猛龍啃下了第四節的攻堅，毫無疑問是這場的第一球星，防守端也是猛龍勁量小子組合的一員——但布歇和朗戴兩個在現代籃球看來，技術風格都不那麼可能出現在正常陣容里的球員，改變了比賽節奏，而這並不是他們第一次做到，納斯的攻防全局思想，以侵略性防守帶動反擊，以運動能力、推進速度補空間的打法，配合長臂怪咖和鐵血戰士，在比賽某些時段的變奏中，的確可以出奇制勝——這種教練策略與球隊運動健將配置的和諧，恐怕才是猛龍最大的驚喜。

雷霆的後衛男團

當快船人員齊整時，倫納德，喬治，貝弗利們，大抵還能壓住路威和哈雷爾的防守問題——路威藏一下，如果對面要找，倫納德、喬治、貝弗利或者哈克萊斯都是極有經驗的擠掩護者，哈雷爾籃下吃點虧，球隊鋒線也都是有經驗的協防者，快船的終極陣容有點遺憾，也談不上致命缺陷，大部分場合肯定是夠用的。

但當你沒有倫納德又沒有貝弗利的時候，防守弱點就有點藏不住了，而他們遇到的對手，又是可以擺出全方面立體化打擊的雷霆後衛男團。

雷霆很慌啊，這麼打要季後賽了，尷尬。

可這也不意外啊，咱們休假期就說了，雷霆單場比賽的戰力就是季後賽級別的，他們能不能進季後賽就兩個影響因素：

第一，保羅和加里納利要休多少場——他們倆到目前為止出勤良好；

第二，雷霆本來是想演的，沒想到打著打著攤牌了。

雷霆趕上對面快船背靠背——你懂的——很頭疼，加里納利歇一歇也算壓壓驚了，不然快船對位上還會更痛苦，畢竟當雷霆擺出王牌陣容時，持球點太容易打到快船痛點。

雷霆的王牌陣容是什麼？

保羅，亞歷山大，施羅德，加里納利，亞當斯，這五人在場的86.2分鐘里，百回合凈勝對手29.9分，是雷霆超過25分鐘出場時間的陣容里，最強的一組。施羅德為何本賽季正負值冠居全隊？他有點像金州勇士死亡五小的伊戈達拉，不是最強的那一塊拼圖，卻是終極體的最後一塊拼圖，當施羅德替下弗格森時，就是雷霆決戰的開啟。

當然，今天雷霆沒有加里納利，但快船缺倫納德和貝弗利兩張牌，弱點也變多了：

於是，亞歷山大和施羅德就發飆了：

亞歷山大，雷霆的希望之星——當然，我並不確定他是雷霆未來第一核心的培養目標，雷霆那麼多選秀權，未來誰知道呢——他不能算雷霆本賽季最大驚喜，畢竟他上賽季在快船就展示了天賦，他能做到什麼，大家也清楚。本賽季，亞歷山大吃下了更多的球權，但54.2%的真實命中率扛住24.4%的回合佔有率並不驚奇。

亞歷山大是個風格不算太現代化的後衛，他的三分球出手不多，超過93%同位置球員的中距離出手頻率，說明投籃分布還有很大的改善空間。

那麼亞歷山大好在哪呢？

不俗的侵略性之外，他的體型和運控節奏感結合的極佳，你可以看看他自信的單挑技術：

以及利用臂展找板的感覺：

投籃分布這東西，隨著三分更純屬更有自信，未來會改善的，現在的疑問只是，雷霆對亞歷山大未來的計劃是什麼？持球打法的側翼嗎？

誰知道呢，畢竟雷霆這支隊到底要怎麼走太難講了——進可季後賽，退有一把選秀權，聯盟最不確定的未來就是他們了。

施羅德打出身價的賽季，也讓雷霆又增加了一些不確定性——說好負資產的，現在看，施羅德反而成為了一些缺少控衛球隊的追求對象。

施羅德交出了生涯最好的真實命中率，56.4%，他作為第六人，反而是球隊回合佔有率最高的球員，當大腿也不是這一場比賽了。

施羅德整個職業生涯都不以效率見長，他過去很擅長突破，攻筐頻率和助攻率都不低，但籃下效率平平，侵略性平平，助失比平平，三分球也平平，中距離倒是強項，但因為他完美的迴避了所有跟提升效率有關的關鍵詞，所以施羅德的進攻也就一直是平平的水準，防守又不好，讓他1550萬的年薪，也就變得性價比平平了。

施羅德這一季，有這樣幾個進步：

籃下命中率是生涯最佳的63%，是不是真的漲球了咱們再觀察，但雷霆空間進步了，球隊也稍微增加了一些中鋒提高位策應，後衛空切的玩法，施羅德是受益人之一；

中距離是生涯最准一季，14英尺外長兩分命中率達到了54%！

施羅德的合同也只剩本賽季和下個賽季了。

好吧，我的意思是，他現在的賣相不錯。

開超市，還是季後賽攪個局，雷霆進退皆可，保羅這回暖的一季，要不，就跟他們好好乾一架吧——萬一對上XX或者XX隊，大夥真的挺想看的。
