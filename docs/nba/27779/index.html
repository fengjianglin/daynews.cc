深度解析｜猛龍隊輕搖直上，為什麼一遇小卡變成蟲

2019-12-20 82條評論 1,047次閱讀 129人點贊

猛龍隊是唯一一支身處異國他鄉的美職籃球隊，球隊所在多倫多是除了首都渥太華之外最大的城市，這裡又被譽為世界最安全，宜居的城市，雖然猛龍隊在這樣的城市裡，但是他們缺少媒體渲染和關注，甚至會被遺忘。想要讓聯盟注意到自己，球隊成績必須好，球隊必須有球星來帶動流量，而萊昂納德的到來打破了平靜如初的水面。

在猛龍隊的萊昂納德

萊昂納德加盟猛龍隊是情非得已

萊昂納德因為傷病問題和馬刺隊鬧得沸沸揚揚，一向乖巧懂事的萊昂納德在這件事情上不肯退讓半步，就算老爺子波波維奇出面也沒能改變萊昂納德出走的決心。聖安東尼奧自己培養的FMVP不想就這樣白白為他人做嫁衣。馬刺隊結合球隊面臨年齡老話的問題必須引入年輕的中鋒以補鄧肯，小加索爾之殤，而在培養中鋒上馬刺隊是成功的，這只是管理層必須引進的籌碼之一，卡哇伊的出走我們不能徹底擺爛，我們還需要即戰力，縱觀聯盟裡面很多球隊都想得到萊昂納德，籌碼不匹配問題一直困擾著馬刺管理層，直到猛龍隊總經理的出現這比交易才正式達成。猛龍隊拿出當家球星德羅贊+博爾特爾+2019首輪受保護簽位，換取馬刺隊的小卡+丹尼-格林。這比交易雙方球隊各得所需，可謂皆大歡喜，受傷最深的就是萊昂納德，因為他根本就不想去猛龍隊，他心心念念想去的快船隊沒有去成，竟然去了國外，這讓小卡想證明自己的絕心更加堅決。

萊昂納德和格林是交易的籌碼

猛龍隊因為萊昂納德的到來打破了自己的魔咒

猛龍隊翻開它的歷史成績我們總能發現些問題，在過去的五年當中，猛龍隊常規賽64.1%勝率，東部第一，而在過去的五個賽季季後賽里猛龍隊21勝30負勝率41%，可以看出來在聯盟和廣大球迷心中，猛龍隊常規賽是龍季後賽是蟲一點不假，垃圾兄弟的得名也就是由此而來的，而這一現象發生了好多年，猛龍隊上下對於這樣的成績很是不滿意，終於在2018年12月1日烏傑里解僱了當年的最佳教練凱西後聘用了助理教練納斯為新任猛龍隊主教練。納斯的到來徹底改變了猛龍的進攻端和防守端，萊昂納德在進攻端有充分的開火權，輪休針對性的保護給了萊昂納德足夠的休息時間，而烏傑里的豪賭贏了，因為小卡在季後賽的表現我們被徹底征服，他是那種能夠關鍵時刻殺死比賽，左右比賽走勢的巨星，相比於垃圾兄弟關鍵時刻的隱藏小卡這方面絕對是做了球隊當家球星該做的事。

在球隊主力整容中西亞卡姆在進攻上有更多的表現機會，洛瑞作為猛龍隊的老

萊昂納德比垃圾兄弟強很多

將關鍵時刻絕不手軟，范喬丹給予了足夠的信任，小加索爾，伊巴卡在防守端是對手的噩夢，這樣的陣容在季後賽拆解了76人隊，掰下鹿角，總決賽戰勝了無杜勇士隊成功問鼎聯盟。

沒有小卡和格林的猛龍隊在本賽季初戰績上沒有受到任何影響主要是這幾個原因

我們先來看看本賽季猛龍隊前20場比賽數據，在過去的20場比賽里排名聯盟第四，凈勝分聯盟第三，場均失分聯盟第七，場均助攻聯盟第九場均25次，場均籃板46.5個聯盟第六，場均搶斷聯盟第五8.5次。在看了這些數據後你會發現這支猛龍隊雖然比不了上個賽季開局16勝4負隊史最佳戰績，但是本賽季的猛龍隊在攻防兩端絕對另對手膽寒主要是因為球隊的進攻和防守有了變化。

在進攻端西亞卡姆在球隊的戰術地位被肯定，他擁有絕對的開火權，這樣一來西亞卡姆在進攻端場均得分大放異彩一度排在了聯盟的前十，西亞卡姆以場均36.8出場時間得到了24.7分+8.3籃板+3.6助攻+1蓋帽+0.9搶斷比上個賽季16.9分+6.9籃板+3.1助攻+0.9蓋帽+0.7搶斷，有了全面的提升，在三分球命中率上36.8%和上個賽季沒有變化，但是要知道上個賽季的西亞卡姆三分球一般是接球定位投三分，而這個賽季是主動進攻，可見這是猛龍隊一大改變。猛龍隊另一主力范弗里特本賽季的數據為18分+3.8籃板+7助攻+2搶斷，上個賽季的數據為11分+2.6籃板+4.8助攻+0.9搶斷，在三分球命中率上個賽季為39.5%而這個賽季提高到37.8%罰球命中率上賽季84.3%提升到本賽季87.6%可見范弗里特是猛龍隊戰績好的另一個主要原因，而在場均搶斷上范弗里特以場均2次搶斷排在了聯盟第三的位置可見范弗里特在本賽季的防守上下足了功夫。

西亞卡姆上個賽季和本賽季數據對比

為什麼要誇猛龍隊是他們做的真不賴。猛龍隊上個賽季的奪冠陣容毋庸置疑首發和替補都不差，而這個賽季在沒有了萊昂納德和格林，在賽季初洛瑞和伊巴卡有傷不能打，可以說猛龍隊首發去其四，這要是放到普通球隊直接擺爛或者放棄了，而猛龍隊為所有球迷展示了總冠軍應有的底蘊，他們以范弗里特+鮑威爾+西亞卡姆+阿努諾比+加索爾為首發陣容，這個陣容一看太普通了，甚至沒有魚腩球隊有名氣，但是不能小看他們，這個陣容在防守端可以密不透風，在進攻端可以撕開口子，下面就讓我們一起去看看猛龍隊的攻防兩端。

進攻端的變化：猛龍隊採用的是平民戰術，每個人都要分享球，每個人都有投籃能力，說沒有用我們看一組數據，在得分上西亞卡姆24.7分，洛瑞18.9分，范弗里特18分，鮑威爾14.1分，伊巴卡12.2分，在助攻上范弗里特以7次助攻排在前面，加索爾的策應能力這時候體現出來了他場均送出3.6次助攻，西亞卡姆助攻次數也是3.6次，戴維斯的助攻為1.8次，我們從助攻和得分上可以看出來球隊在輪轉和分享球上做的很好。這還沒有完全民皆兵不可怕，可怕的是他們還有很高的場均命中率，托馬斯場均54.1%命中率，傑弗森場均51.8%命中率，米勒有50%命中率，阿努諾比49.8%命中率，這樣一來在輪換陣容上猛龍隊不吃虧因為進攻效率不佳時候命中率來湊。

快船隊球員數據分析

防守端的變化：猛龍隊這個賽季的防守主要採用的是區域聯防和夾擊，這樣的防守策略從效果上來看不錯，因為在鋒線上西亞卡姆，阿努諾比，布歇，戴維斯，傑弗森，都是移動速度快，防守區域性強的球員，這樣的防守策略配上勁爆身體素質，我想他們的防守不好破，湖人隊的7連勝就是止於此的。我們再來看看防守端數據在搶斷上范弗里特場均2次搶斷能夠排在聯盟第三，阿努諾比這個賽季是猛龍隊重大發現他能貢獻1.3次搶斷，洛瑞也能搶斷1.3次，鮑威爾1次，西亞卡姆1次。在蓋帽上加索爾場均1.2個蓋帽是由於他的經驗為他帶來的，伊巴卡防守悍將1個蓋帽少點傷愈歸來會提升的，阿努諾比0.9個，西亞卡姆0.9個，布歇0.7個這是絕對輪換主力在防守端的表現。

我們看完猛龍隊在攻防兩端的改變不由得感嘆納斯教練在調教球員上做的真好，讓大家暫時忘掉了卡哇伊離去給猛龍隊帶來的余傷，但是真的能夠忘掉卡哇伊嗎？猛龍隊真的不在乎關鍵時刻沒有卡哇伊嗎？答案是否定的。

這是猛龍隊防守端和命中率

猛龍隊的痛在遇到快船隊的時候舊傷複發

在本賽季的猛龍隊和快船隊兩場比賽快船隊都戰勝了猛龍隊，而在這兩場比賽里萊昂納德都沒有選擇輪休，而是要親手手刃舊主，第一場比賽快船隊主場以98-88贏下比賽，在這場比賽之前兩隊在東西區表現都很好，戰績同為7勝3負，這場比賽也是萊昂納德走後第一次和老東家交手。比賽一開始納斯就讓猛龍隊打的很有侵略性，進攻端本場比賽上了很多新人不斷的增加防守強度讓快船隊很不適應，特別是卡哇伊上半場只通過罰球得到4分，同時還有4次失誤，萊昂納德有這樣的表現大家也不要驚訝！因為猛龍隊對小卡的防守強度太大了，雙人包夾，到了底線3人包夾，小卡在艱難情況下傳球，很多時候是失誤，即使不失誤隊友在接球後面臨的是猛龍隊瘋狂的餓撲，手感不好命中率很低，首節比賽輸了8分，上半場結束只領先5分球。下半場比賽是這場比賽真正分水嶺，特別是第四節整節比賽猛龍隊只得到了10分，而快船隊得到了25分，過於兇狠的防守換來的代價是快船隊打你的七寸體力，在最後一節，快船隊用了四個鋒線球員，一名後衛，機動性更好的球員徹底拖垮了猛龍隊的最後的防線，輸球是必須面對的事情。

快船隊猛龍隊第一次交手數據

小卡輸了數據贏了比賽

小卡數據難看只得到了12分11籃板9助攻3搶斷9失誤，但是沒有小卡的牽制猛龍隊不會有夾擊，猛龍球員夾擊籃板球必然沒有保障，我們來看看兩隊籃板球對比快船隊66個猛龍隊38個整整輸了28個籃板球贏球會變得很難，雖然整場比賽快船隊命中率很低37%但是不斷的籃板球讓快船隊有了重新進攻的機會，所以說猛龍隊在遇見小卡的時候過分害怕和擔心自己的實力，才會有了瘋狂的防守包夾，讓自己在攻防兩端失去了原有的平衡。

猛龍隊主場的歡呼聲激勵了卡哇伊求勝慾望猛龍還是沒有解決萊昂納德這一關

在賽前猛龍隊高管說到；不允許有現場球迷對萊昂納德報以噓聲否則和便是猛龍隊的敵人。果然現場給予萊昂納德最好的認可，萊昂納德也從洛瑞手中接過了屬於自己的總冠軍戒指。比賽在一片友好中開始了，在上一場比賽中猛龍隊的包夾限制小卡是輸掉比賽的關鍵，而這場比賽猛龍隊不在是瘋狂包夾，而是試探性包夾，小卡持球進攻猛龍隊不再多人防，守小卡中距離解決戰鬥。快船隊在進攻端不順時候球到了小卡手裡，小卡突破在罰球線附近徘徊，區域聯防我就轉移球，不夾擊我就單打你，幾次單打成功，納斯教練坐不住了果斷給予壓力，這時候快船隊外線不在像上一場手感冰涼，而是主力球員集體給予回應。全場比賽快船隊7人得分上雙，如果沒有喬治的13中2投籃命中率我想這場比賽猛龍隊會輸的更慘，在命中率上快船隊44.2%命中率，猛龍隊35.2%命中率，在三分球命中率上快船隊30%猛龍隊22.2%想贏球難上加難了。

快船隊猛龍隊第二次交手數據

這兩場比賽的輸球讓猛龍隊真正意識到自己的硬傷是哪，也讓猛龍隊重新認識了自己有小卡和沒有小卡球隊真的是兩個樣子兩種狀態。不過猛龍隊這支球隊還是給予肯定的在洛瑞，伊巴卡不能打，小卡，格林出走陣容不整情況下打出這樣的成績，只有一個原因我們是總冠軍球隊，我們總冠軍戒指上刻著World Champions。
