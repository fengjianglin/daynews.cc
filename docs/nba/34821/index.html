單節22分，完美壓制隊友發揮！想奪權嘛，不用那麼絕吧

2019-12-24 840條評論 3,657次閱讀 767人點贊

這劇本可以的！前三節比賽，黃蜂和凱爾特人打的有來有往，不分高下

以為第四節要看好戲了，沒想到卻被一個人打花了….

沒錯，辣個惡人就是塔圖姆！第四節他一人獨得22分，原本還是很膠著的場面，硬是被一個人一波流帶走了，就問你這屬於什麼水平？

本來想以一個三年級新秀和你們好好相處的，沒想到換來的是嘲諷和質疑，現在我不裝了，我是你們的爸爸！

連續的單打得分

手感來了，當著你面還是能把球給投進。

很騷的是，塔圖姆自信的時候，逼成這樣還是敢出手，還是能進。

嗯，沒錯，是那股熟悉的問題。

全場塔圖姆29投15中，轟下39分12個籃板卻只有2個助攻的數據。

所以，這徒弟越來越精髓了，內味越來越濃了。

當然了，手感這麼好，換作是楊某人別說傳球了，不多投幾個已經很給面子了。

39分是塔圖姆職業生涯的新高，雖然塔圖姆的定位是一名得分手，楊某人知道你們覺得39分看上去不是特別多，但是對於任何一個球員來說39分都是來自不易的，只是我們被某些人慣壞了而已。

最重要的一點是，單節22分比全場39分更值得去吹啊，塔圖姆幾乎用一己之力把勝利牢牢握在手上。

所以，無疑這場比賽是塔圖姆生涯最強的一戰。

這也是塔圖姆連續第三場20+的比賽，同時塔圖姆還終結了杰倫布朗連續2場20+的記錄。為什麼楊某人用終結兩個呢？

這不是明擺著的嗎？塔圖姆上去一人一波流帶走比賽，杰倫布朗在替補席喊完666之後，再重新上場的時候球隊已經大比分領先了。

垃圾時間刷分不是我布朗一貫的風格，前三節已經有14分進賬的布朗，第四節只出手了一次僅得2分，最後全場得分定格在了16分。

一個人打崩對面全隊，迫使布朗上不了場，即使上場了，因為球隊大比分領先了，從而讓布朗愧於出手，想奪權嘛，不用趕盡殺絕吧！

好賤一男的，楊某人宣布今天在凱爾特人隊內「老大」爭奪賽上取得了領先的位置的是，塔圖姆！（手動狗頭保命）

好了，不開玩笑。看塔圖姆和布朗爭取就圖一樂，綠軍真正的核心還是法爾，今天由於又是大勝，法爾又獲得了一定的上場時間。

而且，他也是給力啊，一上來就給主場觀眾表演了，遮天抓帽！雖然有點像干擾球了，但管他呢，帥就完事了！

法爾真的是凱爾特人陣中的一個活寶啊，不說和其他隊友的關係好了，就算是球迷對法爾也是蜜汁喜歡啊。

上一場法爾在綠軍主場的首秀，全場觀眾眼到垃圾時間到了，更是大喊著：we want tacko，甚至少帥也少有的跟著主場觀眾調皮起來啊。

害，這種主場氣氛真的是愛了！

用一句綠軍球迷的話來形容對法爾的愛吧：法爾永遠都不會水！
