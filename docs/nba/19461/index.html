4人20+難求一勝 連輸12場跌至倒數第二 要靠打勇士來終結連敗嗎？

2019-12-17 74條評論 1,631次閱讀 200人點贊

北京時間12月16日，NBA常規賽繼續進行，鵜鶘主場119-130不敵魔術，遭遇了12連敗。本場鵜鶘的霍勒迪、雷迪克。英格拉姆和約什-哈特都得到了20+的得分，但依舊難挽球隊落敗的命運。

儘管休假期送走了球隊的內線王牌安東尼-戴維斯，但鵜鶘獲得的報酬相當不菲，英格拉姆、鮑爾和哈特三位青年才俊的加盟，以及選秀大會上選中的流量新星狀元錫安，他們仍被外界一致看好，甚至有不少球迷認為他們有機會進軍季後賽。

但在將近1/3賽程之後，鵜鶘的戰績是讓人失望的，在本場失利之後他們已經遭遇了12連敗，排名跌至了西部倒數第二，僅強於傷兵滿營的勇士。從賽季初的信心滿滿到如今的連戰連敗，鵜鶘的表現著實讓球迷大跌眼鏡。

此役坐鎮主場面對實力一般的魔術，鵜鶘在抵抗了半場之後便潰不成軍，最多時落後對手25分。儘管陣中4人得分過20，霍勒迪砍下29分4籃板8助攻，依舊沒有換來一場勝利，鵜鶘遭遇尷尬的12連敗，這也是他們隊史的最長連敗紀錄。

來不及悲傷的鵜鶘將在接下來的兩戰先後面對籃網和森林狼，這同樣是兩支實力不俗的強敵。不過也有好消息，那便是在這兩場比賽之後，他們將面對聯盟墊底的勇士，此前老鷹和尼克斯都在他們身上終止了自己的10連敗，難道這一回鵜鶘的連敗紀錄也需要靠勇士來終結了嗎？
