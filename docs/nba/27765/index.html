總決賽強度！湖人打雄鹿，詹姆斯濃眉單節10投1中，老將單節10分

2019-12-20 3,124條評論 4,568次閱讀 1191人點贊

北京時間12月20日，NBA焦點大戰，東部第一雄鹿隊主場迎戰西部第一湖人隊。此役之前，雙方都是24勝4負，戰績一模一樣，而且從近況來看，雄鹿和湖人也都先後輸球了，終止了連勝紀錄。雄鹿之前輸給了獨行俠，而湖人則是輸給了步行者。兩隊都是輕裝上陣，沒有心理包袱，比賽必然很好看。兩隊本賽季至今都還沒有連敗過，但是今天的比賽結束後，總有一支球隊要接受連敗的事實。比賽非常精彩，截止目前，第一節打完，湖人17-21落後4分。

湖人單節僅僅得到17分，這在在如今這個時代，非常罕見。當家球星詹姆斯發揮不佳，單節5次投籃全部打鐵，三分球1投0中，罰球2罰2中，只拿到了2分，不過他還有4籃板2助攻進賬。另一個大腿，濃眉哥，今天賽前臨時決定復出的，首節狀態也不好，5投1中，罰球2罰1中，只拿到了3分，另外有3籃板2助攻1搶斷進賬。

湖人雙核發揮低迷，單節加起來10投1中只拿到5分，有些慘淡。湖人的其他角色球員，發揮也一般。中鋒麥基打了8分鐘，拿到了2分，丹尼格林3投2中拿到了6分，波普4投2中拿到了4分。替補所有都沒有得分，全是掛零。全隊進攻打不開，防守端還遭遇了雄鹿了年輕群鹿的猛烈衝擊，有些吃不消。當然了，總體來看，今天的比賽，攻防兩端，真的是總決賽強度！

雄鹿這邊，字母哥單節4投2中拿下5分。去年的新秀迪文琴佐拿下了5分，替補上來的老將喬治希爾發威，單節4投3中，三分球3投2中，拿下了10分！值得一提的是，喬治希爾本賽季三分球命中率50.5%，高居聯盟第一，而且這還是在對陣獨行俠三分球4投0中的情況下，仍有如此變態的數據。

這支雄鹿隊，喬治希爾、科沃爾，都是詹姆斯的老隊友，彼此非常熟悉，賽前老詹也跟科沃爾擁抱了。但是，到了比賽中，大家可不管你是不是熟人。今天這場比賽，註定是火星撞地球，詹眉組合也不可能一直這麼低迷，而且雄鹿這邊，字母哥也還沒怎麼攻擊。到底比賽會變成什麼走勢，咱們持續關注。
