單手抓球誰最帥？倫納德大手抓球隨意晃人，麥迪抓球美如畫

2019-12-24 1,676條評論 3,946次閱讀 2520人點贊

說到單手抓球，相信大多數球迷朋友都知道這個籃球場上的經典動作。可以說，這個動作的實際操作難度其實並不大，但是，必須要天賦好才行。如果你的手掌不夠大的話，哪怕技術再好也無法做到，因此這個技巧還是比較難完成的，但是對於NBA裡面的巨星來說，就真的不要太簡單了！對於我們來說，單手抓球，最多是在同伴面前展示一下，但是對於他們來說，卻是應用到實戰方面的！那麼，你見過最帥的單手抓球操作是哪個場面？

小卡靈巧晃人堪稱極限

說到單手抓球，對於小卡來說根本不是一件難事。我們知道小卡的大手是NBA歷史都數一數二的，在這次進攻之中，小卡就用單手抓球完成了一次完美的晃人，直接晃開了對手的中心，堪稱極限操作。

麥迪將美感演繹到極致

說到單手抓球，在這方面麥迪是將美感演繹到極限的男人。的確，他的球風本就是那樣，漂移靈動，如果他做好這個姿勢的時候，小心，下一步他的突破就要來了。

奧尼爾退役也能搞笑抓球

說到奧尼爾，在球場上這個鯊魚倒是比較少用這個姿勢，畢竟他只要接球完成內線扣籃就可以了。但是退役之後在場邊的奧尼爾玩起單手抓球，還是非常輕鬆的，畢竟他的手，在NBA裡面也是數一數二的，只不過，怎麼看，怎麼覺得這個姿勢非常搞笑呢？

科比面無表情堪稱冷酷

說到單手抓球，不玩花架子的科比面無表情，對著對手顯示出一種冷酷的感覺。我玩這招就是為了過你，而不是秀！可以說，巔峰時期的黑曼巴的確給人一種不怒而威的感覺！

喬丹霸氣外露讓人生畏

老流氓不光晃防守人，連攝像大哥都晃。

說到籃球之神喬丹，也許是最早將單手抓球讓世界知道的人，我們說喬丹只有一個，他的這招論氣勢，真的誰都學不來，讓人生畏，畢竟，他的進攻，就連只是單純的跳投你也防不住，更別說技巧了。

KG展現球場力量感

說到單打時的KG，那身高臂長的優勢也是一下子就體現了出來，這個大前鋒雖然不是肌肉型的，但是卻給人一種球場上的力量感，可以說讓人感覺十分可靠，這球交給我來打就行，其他人看我操作，準備接球空位投籃！

J博士堪稱震古爍今

說到J博士歐文，可以說是單手抓球第一人了。當年，他完成了一個堪稱變態的上籃動作，被認為是NBA經典之一，可以說讓人感覺十分驚訝，簡直震古爍今！
