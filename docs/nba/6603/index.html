詹皇超科比最年輕3萬3！濃眉哥創歷史，一人一句來祝賀他

2019-11-28 0條評論 13次閱讀 0人點贊

鵜鶘今日坐鎮主場迎來湖人的挑戰，這是兩支球隊在夏天完成重磅交易後的首度交鋒，同時也是戴維斯生涯第一次以客場球員的身份回到新奧爾良。

面對舊主的濃眉哥表現的非常積極，首節肆虐禁區拿下14分，不過鵜鶘眾將表現的更好，作為濃眉哥曾經征戰沙場的搭檔，霍勒迪首節同樣砍下15分，在他的帶動下，鵜鶘首節轟進9記三分打了湖人一個措手不及，鵜鶘也藉此取得13分領先。次節濃眉哥雖然延續狀態再砍13分，奈何鵜鶘的多點開花不給他們起勢的機會，半場結束時鵜鶘仍然手握著10分的領先。下半場的湖人一度落後鵜鶘15分，但末段庫茲馬及時站出來拯救球隊，他在第三節最後時刻和第四節初命中3記三分砍下11分並幫助球隊追平比分。

末節，丹尼爾斯連續取分幫助湖人開局打出6-0將分差追至4分，鵜鶘命中率稍有下降，2分鐘內一分未得，庫茲馬里突外投連得5分幫助湖人追平比分，湖人前4分鐘打出一波12-2的攻擊波，英格拉姆兩罰全中為鵜鶘穩住局勢，最後5分鐘，詹姆斯開始發力連得7分幫助湖人打出一波9-0將比分反超，雷迪克命中三分還以顏色。鵜鶘連續內線造成殺傷將比分再次追平，比賽進入拉鋸戰，雙方比分交替上升，最後時刻庫茲馬左側底角命中關鍵三分，詹姆斯被哈特搶斷，但雷迪克空位三分不中，隨即濃眉兩罰一中，霍勒迪發球失誤使得鵜鶘喪失機會，濃眉繼續罰球兩罰全中，英格拉姆搏命三分不中，全場戰罷，鵜鶘110-114湖人。

賽季截止至目前，加上本場比賽，湖人已經在四場比賽期間曾落後對手至少15分（本場16分），這四場比賽，湖人最終全部逆轉成功，獲得了勝利。

詹姆斯迎里程碑！破科比紀錄成最年輕33000分先生

憑藉著詹姆斯末節的15分，湖人完成16分逆轉，114-110險勝鵜鶘。詹姆斯在本場比賽中拿到29分11助攻，並砍下個人職業生涯的第33000分，超越科比成為NBA歷史上最年輕的33000分先生。

在與鵜鶘的比賽之前，詹姆斯NBA生涯總得分為32979分，距離33000分大關只差21分。而在對陣鵜鶘的比賽進行到第四節距離比賽結束還有5分29秒時，詹姆斯憑藉一記3分球拿到該場比賽的第23分，從而達成里程碑，打破了湖人名宿科比37歲零5個月的最年輕紀錄，以34歲零11個月的年紀再次締造了屬於自己的傳奇。這是一種競爭，更是一種傳承！接下來，詹姆斯將向最年輕34000分紀錄發起衝擊，這項紀錄是38歲190天，保持者是馬龍。

需要指出的是，屬於詹姆斯的最年輕紀錄已經超過40項，未來，他還是要創造屬於自己的紀錄。

在NBA歷史得分榜上，詹姆斯目前排名第四，前三位分別是賈巴爾（38387分）、卡爾-馬龍（36928分）和科比（33643分）。在這4位33000分球員中，詹姆斯達標所用場次最少，而科比最多。對鵜鶘這一戰是詹姆斯參加的第1216場常規賽，賈巴爾獲得33000分用了1237場比賽，馬龍是1277場，科比是1310場。

詹姆斯的下一個目標就是歷史第三的生涯總得分，目前科比的得分是33643分，這一項記錄也是非常有機會在本賽季被打破，但是詹姆斯想要成為歷史總得分第一，超過賈巴爾的38387分的記錄還是有些困難的，詹姆斯必須要保證3-4個賽季的出色發揮才能夠超越賈巴爾。對於此你們有什麼看法嗎？

我沒有機會看見喬丹的時代，但有幸見證著詹姆斯的偉大！

關鍵搶斷+罰球殺死比賽！濃眉首次面對舊主就砍下41分 打破KD等人的歷史紀錄

美國傳統的感恩節前夜，安東尼-戴維斯回到了新奧爾良這個夢想開始之地。

作為詹姆斯最得力的幫手，「濃眉」故地重遊新奧爾良是本場比賽最大的看點。這是安東尼-戴維斯被交易至湖人後首次回到新奧爾良。由於此前的不歡而散，濃眉今天受到了鵜鶘球迷們的特別待遇，在他進場熱身時，現場觀眾給他送上了噓聲。當然濃眉哥不管這些，作為鵜鶘隊史上的得分王、籃板王、蓋帽王，戴維斯本場比賽將朝著一項新紀錄發起衝擊。在聯盟歷史上，球員面對舊主首戰的得分紀錄（39分）由凱文-杜蘭特、斯蒂芬-馬布里、丹尼-安吉三人共同保持。

首節開局，他打得很積極，助格林上籃後，又搶斷海耶斯，快攻空接扣籃得分。8分24秒，他在邊線拉出禁區干拔中投命中，但鵜鶘的三分雨很快把比分拉開，戴維斯即便節末不斷獲得罰球機會，半分鐘時間裡還連續快攻扣籃得分，但湖人整體的追分效率很有限。他首節11投5中拿到14分，效率值卻是-10。次節4分23秒，他還命中湖人本場第二記三分，著實不易。沒有勒布朗，他扛起了得分責任，一人連拿9分。整個上半場湖人發揮最為出色的是戴維斯，對陣老東家他半場19中10砍下27分4籃板2搶斷。

在第三節最後時刻，濃眉接到隊友妙傳上演暴扣，隨後在鵜鶘發球之際，AD快速搶斷得手完成上籃，兩秒時間連拿4分。三節過後，濃眉已經轟下36分8籃板2搶斷。末節，濃眉重新上場後接到詹姆斯的空中接力後雙手暴扣得分。在比賽剩下2分40秒時追平比分。最後關頭濃眉哥成為場上主角，他4次罰球3次命中，加上關鍵的封蓋和搶斷，幫助湖人完成逆轉，114-110擊敗鵜鶘。

此役，濃眉出場37分鐘30投15中，三分4中1、罰球12中10，砍下41分9籃板1助攻3搶斷1蓋帽。

濃眉打破更換門庭之後首次面對舊主的得分紀錄，此前這一紀錄由杜蘭特（2016年VS雷霆）、馬布里（2000年VS森林狼）和安吉（1989年VS凱爾特人）共同保持。

賽後濃眉哥戴維斯接受採訪，他表示一切只有感恩！

湖人在這一波9連勝里，射手好即全隊好基本成了定律，此前苦戰國王、灰熊這些勝場跟湖人差挺多的對手，無不是因為全隊在外線瘋狂打鐵。8記三分是湖人9連勝里單場命中的最低數字，在如今聯盟任何一支球隊怕是都很難在這種外線輸出下贏球，不過湖人雙星能力太犯規，關鍵時刻一攻一防照樣拿下勝利。
