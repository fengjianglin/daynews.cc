東契奇領銜傷病潮 盤點NBA近兩日傷病，格里芬最慘

天天要聞
2019-12-16

北京時間12月15日，NBA常規賽已經緊鑼密鼓的進行了快兩個月的時間，各支球隊和球員身體也漸漸呈現出疲勞狀態，因此最近的傷病情況開始增多。

在昨天湖人客場戰勝熱火的最後時刻，詹姆斯在一次防守時和熱火的吉米-巴特勒發生肢體碰撞並倒地，導致右肘受傷。今天，湖人隊公布了詹姆斯的傷病報告，表示詹姆斯目前右肘仍然酸痛，明天客場與老鷹隊的比賽中出戰成疑。詹姆斯已經在NBA連續征戰17個賽季，本賽季打到到現在，35歲高齡的他打滿了湖人的26場比賽，還沒有缺席過1場比賽，敬業精神令人欽佩，是該讓他這個鐵人休息一下了。

如今在聯盟大紅大紫的東契奇被球迷們譽詹姆斯的接班人，才二年級的他就成了MVP的有力競爭者，如果不出現大的意外，他在NBA將前途無量。但在今天，他著實把球迷們虛驚了一場：在今天獨行俠主場迎戰邁阿密熱火的比賽中，東契奇在帶球突破時，不慎踩到了對方後衛努恩的腳面邊，扭傷腳踝痛苦倒地，從慢鏡頭的回放中可以看到，東契奇的右腳腳踝已經呈現出明顯的90度彎曲，場面讓人揪心，隨後他單腳跳著返回更衣室。

在東契奇受傷後，獨行俠陣腳大亂，以108:122不敵熱火。隨後獨行俠官方也在第一時間公布了東契奇X光片檢測結果：陰性！

與上面兩位相比，下面這兩位則沒有這麼幸運。

在昨天雄鹿戰勝灰熊的比賽中，雄鹿首發控衛埃里克-布雷德索在比賽中與灰熊的布魯克斯相撞後傷退離場，今天，根據美國媒體報道，雄鹿隊宣布了布雷德索的核磁共振檢查結果：右腿腓骨撕裂性骨折，他也因此至少傷停2周時間。

因去年的優異表現，布雷德索今年3月剛和雄鹿隊續約了一份4年7000萬美元的合同。本賽季，布雷德索出戰了雄鹿隊的全部26場比賽，場均得到15分、5.7次助攻和4.9個籃板。他的缺席，或許會讓雄鹿的連勝紀錄就此止步。

而布雷德索的骨折和格里芬相比還算輕的，在今天活塞的一場比賽中，格里芬遭遇左膝酸痛，提前退場，本場比賽他只出戰 15分鐘，7投0中，拿到2分2籃板2助攻。

在上賽季，格里芬狀態回暖，拿到近幾個賽季最高的24.5分，7.5個籃板，4.5助攻的明星數據，但中途左膝再次不幸受傷，為了球隊，堅強的他帶傷強行復出，幫助活塞闖進季後賽，這也導致了他左膝的傷情惡化，上賽季結束後他接受了膝蓋手術，復出之後並沒有恢復手術前的水準，本賽季數據一落千丈，目前為：16.4分、4.6個籃板和3.3次助攻，命中率只有36.7%。

而今天他左膝再次受傷，其職業生涯或將再次受到巨大影響，真是當年的格里芬有多強悍，如今的他就讓人有多遺憾！

NBA的輪休制度出台後，萊昂納德曾不止一次選擇輪休背靠背比賽，以減輕身體負擔，也因此飽受爭議，但看到上面幾位遭遇後，想想他是多麼的聰明！

另外，據夏洛特黃蜂隊官方宣布，新秀PJ-華盛頓因手指骨折也將缺席一段時間。
