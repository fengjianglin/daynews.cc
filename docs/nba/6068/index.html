傷病：勒夫生病缺陣，恩比德臀傷明天休戰

2019-12-07 0條評論 18次閱讀 0人點贊

今天NBA一共有10場比賽，各隊更新了傷病名單：

凱文-勒夫（生病）不會參加今天和魔術的比賽。勒夫今天甚至沒有出現在球館，他還可能缺席周日和76人的比賽。特里斯坦-湯普森也因為生病將在賽前決定。

喬爾-恩比德（左臀挫傷）將不會參加明天和騎士的比賽。在昨天輸給奇才後，恩比德告訴記者臀部不適，他將接受治療。

阿隆-貝恩斯（左小腿）不大可能參加明天和火箭的比賽。這將是貝恩斯連續缺席第4場比賽，此前他還因為臀部傷勢連續缺席5場比賽。

賈-莫蘭特已經參加了一定程度的訓練，他將在周日和爵士比賽賽前重新評估。

德懷特-鮑威爾在上一場和森林狼比賽第四節左臂受傷，他將參加周一和尼克斯的比賽。

馬庫斯-斯瑪特將參加今天和掘金的比賽，並表示自己將主防約基奇一段時間。斯瑪特因為腹部傷勢缺席上一場和熱火的比賽。

公牛前鋒奧托-波特接受核磁共振檢查，顯示左腳骨出現水腫，將在兩周後複查。塞迪斯-楊因為個人原因缺席今天和勇士的比賽。

奇才客場對陣熱火，小托馬斯（左小腿拉傷）和伊安-馬辛米（右跟腱）將缺陣。
