老詹搖搖頭！湖人短板太明顯了，不如簽下這位CBA球員

天天要聞
2019-12-24

湖人主場104-128不敵掘金，苦吞三連敗。

夏天幾乎掏空了球隊的年輕血液，換來了夢寐以求的安東尼-戴維斯後，在沃格爾的執教下極快的產生了化學反應，戰績一路高歌猛進。

本場比賽之前，湖人是以傲人的23勝5負戰績，高居聯盟第二，僅次於雄鹿。

然而在前些天濃眉因傷休戰，球隊輸給步行者後，這支備受關注的隊伍，短短几天就將自己的弊端全部暴露出來了。

雖然說那場比賽，湖人僅僅是以3分惜敗，詹姆斯也拿到了20+9+9的准三雙數據。

然而這個數據的隱藏下，卻是投籃20中8，40%的慘淡命中率，刷新了過去18場的比賽新低。

同時三分6投0中，並且附帶了5個失誤。

那麼湖人的問題到底出在哪裡呢，簡單來看幾點：

一：缺少更多的持球主攻點

對陣步行者的比賽里，除了詹姆斯，得分最高的是霍華德的十投全中20分，但他大部分的進攻都是通過吃餅得到的。

而其餘球員，都在10分左右浮動，並且大部分都是接球投籃的得分。

二：內線防守過度依賴濃眉

湖人的三大內線，濃眉場均2.7蓋帽，麥基場均1.4蓋帽，霍華德場均1.3蓋帽。

本賽季湖人場均蓋帽7.13次高居聯盟第一，104.9的防守失分聯盟第五。

乍一看，大家會覺得這是個恐怖的內線禁飛區。

的確，濃眉在的時候，可以依靠自己的移動速度，臂展以及大範圍的防守面積，有效限制對方的上籃得分，三大內線互補互成，構造了恐怖的籃下防守。

濃眉也是本賽季最佳防守球員的熱門候選人。

但是濃眉不在的時候呢？

麥基防守端容易起跳，並且頂防是弱項，霍華德雖然經驗老道，但是移動腳步大不如前，防擋拆防不出去。

結果就是打步行者那場比賽，讓對面的小薩博尼斯拿了26分10籃板，4前場板的豪華數據。

三：軟弱無力的替補群

在連勝期間，詹眉強大的吸引包夾能力，給替補創造了很多空位投籃的機會。

然而一旦遇到同樣的防守強隊，替補的得分能力就顯得相形見絀了。

上一場對陣雄鹿，湖人替補得分是驚人的4比34，直接被打爆了…

四：除詹眉外，沒有第三持球進攻點

庫茲馬作為濃眉交易里唯一留隊的年輕人，也不知道是融不進沃格爾的戰術體系，還是自身原因，各項數據退化的讓人覺得離譜。

而波普，雖然球迷們天天調侃著 「 波普定律 」，但是他的持球進攻實在是不太夠看的。

夏天續約的隆多，引援的布拉德利，丹尼-格林，都不是能持球造殺傷的球員。

最後沒錢了，在自由市場上籤了個「 杜老大 」，打打架還行，進攻上真指望不上什麼…

三連敗期間暴露的問題，總結就是隊詹姆斯和濃眉兩大核心的過度依賴。

所以當面對頂級強隊的時候，替補提供不了足夠的火力支援，從而導致輸球。

四：湖人該如何改變？

湖人本賽季的第一目標是總冠軍，但是前提得先保證詹眉的健康。

其次在面對頂級強隊時，替補要給力點。

如果替補還是不行，最好的補強方法就是通過交易，或者自由市場淘寶了。

但問題是，湖人現在已經超奢侈稅太多了，並且好一點的籌碼里，隆多，波普都擁有交易否決權…

想要交易簡直難上加難，而且在這個薪水價位的球員里，他們的性價比已經很高了。

但也不是說不能交易，之前湖人在考辛斯的受傷里，得到了一個175萬的傷病特例。

倘若湖人膽子夠大，他們可以用布拉德利477萬的薪資為主體，在工資帽的設定上，是可以換到一名600-700萬左右薪資的球員的。

目前這個價位的球員，國王的別利察是不錯的選擇，就是不知道人家會不會換了…

要說戰績，湖人三連敗了，但還是排在西部第一。

可是這期間替補表現出來的東西是完全不合格的，湖人的目標是冠軍，倘若沒完成任務，他們就是白白損失了以英格拉姆為代表的青年才俊和選秀權。

所以得穩，穩到季後賽不能出一點差錯，稍微出了一點差錯，湖人的這盤棋，就會滿盤皆輸。
