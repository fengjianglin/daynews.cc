加內特：我們曾經擊垮過詹姆斯，那是因為沒有將他放在眼裡

2019-12-18 1,912條評論 2,970次閱讀 251人點贊

北京時間12月18日，NBA名宿凱文-加內特在一檔節目中表示，當年的綠衫軍曾經擊垮過勒布朗-詹姆斯，他們沒有將詹姆斯放在眼裡。

節目中，主持人比爾-西蒙斯回顧了2012年季後賽。他說道：「當時德里克-羅斯受傷了。忽然之間，凱爾特人的感覺就像是『哇，隨便哪支球隊晉級了——』。邁阿密熱火，誰知道呢。熱火那時候還沒有奪冠呢。凱爾特人淘汰了76人，隆多命中了三分球。」

加內特接話道：「他當時狀態變得更好了一些。」

比爾-西蒙斯表示：「但那些可是關鍵的三分球啊。接著是和熱火的系列賽，你們打成2-2平。在邁阿密進行天王山之戰，凱爾特人贏了，他們開始到處抨擊勒布朗。對不對？你們試圖擾亂他的頭腦，你們感覺那時候可以擊垮勒布朗。」

加內特立即說道：「我們確實擊垮過勒布朗，所以別在這裡瞎說了。你理解他是怎麼去到克里夫蘭，怎麼去到邁阿密的吧，比爾？你還記得嗎？」

比爾-西蒙斯尷尬地回應：「啊，這是事實，你們在2010年擊垮了他。」

加內特繼續表示：「所以別再說什麼了，其實是媒體在抨擊他。凱爾特人時期，我們根本就沒有將勒布朗放在眼裡。我們不害怕勒布朗，我們也不覺得他可以擊敗我們五人。就是這樣。他努力地想要組隊，因為他不希望壓力都在自己一個人身上。你明白嗎？」

比爾-西蒙斯是綠衫軍鐵杆粉絲，他仍然為2012年凱爾特人的失利感到惋惜。他說道：「但我現在仍然會抓狂不已，我感覺我們本來可以擊敗他們的啊。」

加內特接著說：「當時他（勒布朗）去到邁阿密，他可不想繼續克里夫蘭那樣的日子。對吧，你知道我在說什麼吧？」

比爾-西蒙斯接話道：「韋德確實弄傷了隆多的肘部，他甚至都沒有被禁賽哦。」加內特對此說道：「聽著兄弟，對我們來說，那是很敏感的。很多人認為這是體育的一部分。如果一支擁有韋德、克里斯-波什和勒布朗，另一方有我、托尼-阿倫，或許再加上保羅（皮爾斯）和隆多。兩方相遇時會是很酷的一幕，會互相尊重，但不會有什麼兄弟情。不會出現什麼，『嘿，你家人還好吧？』」
