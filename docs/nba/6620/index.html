快船防守人多勢眾 獨行俠困擾 湖人放棄他虧了

2019-11-28 0條評論 17次閱讀 0人點贊

大家好，今天只有兩場比賽，分別是掘金對陣奇才，另一場是快船對陣獨行俠。首先進行的是掘金和奇才的比賽，兩支球隊很顯然他們的實力不在一個層面上，掘金隊攻防都很不錯，這場比賽讓奇才隊也只拿到了104分，對於全聯盟進攻第二好的奇才來講，顯然這個比分並不能讓球隊取得勝利，而球隊今天的三分球命中率也只有25%，主將比爾更是10投2中狀態也並不好。

掘金憑藉著自己整體強大的實力，還是比較輕鬆的戰勝了奇才隊，雖然約老師自己的進攻慾望並不高，但是拿下了20個籃板和5次助攻。奇才對今年夏天得到了兩個非常重要的得分手，一個是上賽季在湖人的瓦格納，另一個則是從馬刺而來的貝爾坦斯。這兩個球員都是奇才對撿到的非常優質的球員，一個是因為湖人隊想要得到倫納德而放棄了他，另一個則是波波維奇，犯糊塗想要把他送走交易到莫里斯，然而莫里斯臨時跳票去了紐約。

瓦格納本賽季的真實命中率非常高，達到了73.9%的命中率，作為一個中鋒，再拉開空間上做的非常好，他的三分球命中率達到47.4%，雖然這個命中率可能不足以保持到賽季結束，但就目前而言這種命中率，對於奇才來講戰術地位還是非常明顯的，能夠幫助球隊拉開空間。他的內線命中率也提高，達到了84.3%，這在中鋒當中也是極高的命中率，可以看出奇才這支球隊，因為空間優化的非常棒，使他在籃下終結的時候干擾並不多，當然他自己的出色的手感，也讓這項數據顯得非常的好看。對此，我只想說湖人的舊將真好用。

作為大前鋒的貝爾坦斯在替補席上發揮也非常棒，他的三分球一向非常不錯，今年達到了44%的命中率，而三分球出手頻率幾乎是上賽季的兩倍場均達到出手7.6次，完全是一個得分後衛的出手數，這也幫助他的真實命中率提高到了64%，就指這一項數據，馬刺就不應該放棄他。雖然他的防守並不好，但他已經是聯盟中大前鋒數一數二的射手。奇才撿到這兩名球員，真的是運氣非常好，雖然奇才的防守非常差，但是就比賽觀感來說，奇才隊看起來還是非常好看的。

另一場比賽獨行俠還是以較大的比分輸給了快船隊，快船隊本場比賽首節先用貝弗利防守東契奇，隨後用哈克萊斯防守他，這也給他造成了非常大的影響，同時這場比賽犯規也比較多讓他在場上時顯得有些猶豫，表現比較一般，只得到了22分，同時還送出了7次失誤。還是能看出來，快船隊在其他位置上對於他傳球路線的判斷還是有一定的心得的，雖然主防他的球員，並不是兩個巨星，但是兩個巨星依靠著協防和其他輔助性的防守，依然把獨行俠打的毫無脾氣可言。在另一側攻防對決中，獨行俠還是缺少一個頂尖的3D類型的球員，目前的電風扇還是防守比較一般，無法限制兩個巨星的發揮。整體而言，從這場比賽來看獨行俠的辦法並不多，不知道到了季後賽兩支球隊如果碰面的話，比賽將變成如何，至少從這場比賽來看獨行俠一點機會都沒有，快船隊完勝。
