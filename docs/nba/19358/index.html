得分最高的5大FMVP，杜蘭特35.2分也無緣，鯊魚統治籃下

2019-12-18 3,219條評論 4,369次閱讀 892人點贊

追逐總冠軍的過程中，需要面對很多困難。許多巨星整個生涯，連總決賽都沒進過，也有不少巨星，通過抱團才實現了冠軍夢想。可見，在NBA帶隊奪冠並不容易。至於拿下FMVP，那是球隊+個人的雙重巔峰，無疑是偉大的。以下是得分最高的5屆FMVP得主，他們都奉獻了歷史級的得分表現，巔峰杜蘭特35.2分都無緣上榜。

92年喬丹

籃球之神喬丹是季後賽大殺器，沒人能像他一樣統治總決賽舞台。90年代，喬丹6次帶領公牛闖入總決賽，6次全部奪冠，甚至沒有給對手搶七的機會。在1992年的總決賽上，喬丹對位滑翔機德雷克斯勒，雖然也是名人堂級別的球員，但幫主還是無情將他打爆，場均35.8分4.8籃板6.5助攻，還將德雷克斯勒的命中率限制到了40%，差距實在太大了。

02年奧尼爾

鯊魚也許不是成就最高的中鋒，但他卻是最具統治力的。巔峰奧尼爾，曾經帶領湖人隊完成三連冠，當年科比實力不錯，但奧尼爾才是絕對老大。連續3年的時間裡，奧尼爾打爆了各路內線，大本、穆大叔、鄧肯等歷史級防守悍將，同樣被斬落馬下。02年總決賽上，奧尼爾場均可以拿到36.3分12.3籃板2.8次蓋帽，命中率達到了60%。

69年韋斯特

NBA歷史上唯一的敗方FMVP得主，是69年的韋斯特。作為遠古大神，韋斯特得分能力很強，他是總決賽總得分王，曾經多次率隊進入總決賽。在1969年的總決賽上，雖然湖人最終輸給了凱爾特人，不過韋斯特表現太出色，場均砍下37.9分4.7籃板7.4助攻，征服了各界球迷，當選為總決賽MVP。

00年奧尼爾

又是奧尼爾，00年是大鯊魚奧尼爾最巔峰的賽季。那時候的他28歲，完全就是不可阻擋的存在。常規賽帶領湖人拿到了67勝15負的完美戰績，毫無懸念的拿走MVP獎盃。到了季後賽舞台上，湖人隊同樣是連克強敵問鼎總冠軍。當年總決賽，奧尼爾可以轟下38分16.7籃板2.7蓋帽的無敵數據，命中率更是高達61.1%，可以說是對手的噩夢。

93年喬丹

巔峰時期的巴克利，是聯盟超級巨星，他帶領的那支太陽隊，實力也足夠強大。巴克利MVP賽季，帶領太陽殺進了總決賽，場均27+13+5的表現堪稱完美，但依舊無法戰勝公牛。因為喬丹實在太無解了，在93年的總決賽舞台上，喬丹場均砍下了41分8.5籃板6.3助攻的華麗數據，命中率50.8%，成功帶隊完成三連冠，打的對手心服口服。
