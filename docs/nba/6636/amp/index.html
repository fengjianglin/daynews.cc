近四個賽季 火箭的建隊思路逐漸轉變 這筆交易促使球隊環境惡化

天天要聞
2019-11-27

時代的潮流從來不是以個人意志力為轉移，10年前，中鋒是聯盟的稀缺品，低位單打以及護框能力成為各支球隊的核心資產，而在當代傳統的內線在聯盟生存不易，空間型的內線供不應求，時代的潮流瞬息萬變應接不暇。

德安東尼來到火箭的4個賽季，火箭隊的戰績雖然長時間位於西部強隊的行列，但是球隊的構架和建隊思路發生了變化，這變化肉眼可見。

2016-2017賽季，火箭的配置為首發哈登、貝弗利、阿里扎、安德森和卡佩拉，板凳陣容有戈登、布魯爾/路威、哈雷爾、德克爾、內內。

火箭的建隊思路為，利用哈登的絕對實力，為其配置足夠多的射手，當時火箭的射手群有阿里扎、貝弗利、安德森、戈登、路威等多位優質射手，火箭囤積大量射手的目的是球隊做的博弈，每場比賽，這麼多射手不可能都不準，在射手數量和質量遠遠超出其他除勇士的球隊，火箭的進攻空間足夠出色，為哈登轉型控衛的首個賽季，助攻達到職業生涯新高。

火箭除了在優質射手上做了足夠的問題，在攻防平衡上也動了一番心思，阿里扎、貝弗利都是攻防兼備適合無限換防體系的球員，貝弗利能彌補哈登防守端不利的情況，為哈登分憂解難，安德森當賽季的護框和防守還在線，卡佩拉持續的進步讓人看到這支球隊的希望。

2017-2018賽季，火箭的配置為首發哈登、保羅、阿里扎、安德森、卡佩拉，板凳陣容為塔克、戈登、傑拉德·格林、巴莫特、內內。

火箭從哈登單核變為燈泡組合雙核的構架，由於本賽季安德森的退化，在賽季中下旬，塔克替代其先發，火箭的陣容構架，是一星一餅三射，戈登、阿里扎、格林、巴莫特是足夠優秀的投手，再加上保羅和哈登的三分威脅力十足，火箭除了中鋒位置，其他位置都具備極強的三分投射能力，火箭利用兩位超級巨星的牽制力，演變出極多的戰術，優質射手和吃餅型內線，是優化球星戰術的最佳補充。

而在防守端，塔克、巴莫特、阿里扎、保羅等人的防守水平和防守實力使得火箭攻守兼備，火箭在防守兩端無縫嫁接。

2018-2019賽季，火箭的配置為哈登、保羅、恩尼斯/戈登、塔克、卡佩拉，板凳陣容為傑拉德·格林、法里埃德、香波特、豪斯、小里弗斯、內內。

由於保羅退化，單打能力退化，很多時候需要戰術支撐，再加上保羅傷病頻繁，火箭更多是哈登成為球隊絕對的球星，但是球隊的空間點準度和效率下降，恩尼斯在火箭時期進攻不穩定，格林、豪斯、小里弗斯都不穩定，促使很多時候需要哈登個人吃掉很多球權，哈登的球權佔有率得到大幅度提升，交出歷史級別的數據。

同時由於法里埃德、格林、小里弗斯、香波特攻守不兼備，促使火箭在季後賽在進攻與防守之間不斷做取捨，火箭三分的穩定性下降，攻守兼備下降，哈登利用哈登個人絕對的實力將火箭拉到另一個高度。

2019-2020賽季，火箭的配置為哈登、威少、豪斯、塔克和卡佩拉，替補陣容為本·麥克勒莫、小里弗斯、錢德勒。

以這樣的陣容來看，在進攻端火箭的空間更加惡化，威少、本·麥克勒莫、小里弗斯的三分穩定性極差，三分命中率在30%左右，屬於不合格的投手，在外線毫無牽制力，而在防守端腳步移動速度偏慢的特點成為全隊的問題，再加上火箭的無球漏人，面對無球大神防守跟不上，防守問題頻發。

從這種角度而言，火箭的陣容配置，無論是進攻端，還是防守端出現較大的退化，目前火箭退化的原因是威少的交易，目前威少並沒有證明自己，而戈登開賽至今的狀態極其糟糕，其他球員的實力沒有達到攻守兼備的程度，從這種角度而言，哈登的數據有可能更加出色，這樣的環境促使哈登不斷凈化，但從側面來看，火箭的管理層工作得非常不到位，建隊思路出現偏差，促使引援引援越糟糕的局面。
