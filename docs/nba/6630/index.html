輸球又輸人！波波維奇賽後這一細節引發爭議，老爺子真的老了？

2019-11-27 0條評論 14次閱讀 0人點贊

最近對於馬刺來說並不好過，加上今天這場比賽失利，他們在最近10場比賽中輸掉了9場。馬刺作為一支連續22年都打進季後賽的球隊來說，屬於他們的時代已經過去了。老對手湖人在經歷了多年的沉淪之後，在本賽季終於迎來了過去10年來的最佳開局，15勝2負的戰績讓他們高居聯盟第一。相比之下，馬刺最近過得並不太好，年過70高齡的波波維奇在男籃世界盃中痛失金牌，在回到NBA之後，這支馬刺也沒有了以往的銳氣！

在這場比賽中，手握「雙德」陣容的他無奈敵不過35歲的詹姆斯，最後時刻球隊慘遭湖人逆轉，這讓波波維奇感到極其不高興。接二連三的失敗讓他甚是惱火，在比賽結束之後，老爺子更是疑似拒絕了湖人主教練與他握手的致意，只是簡單地揮了揮手。這種行為讓大多球迷感到不解，畢竟波波教練一向是以高情商待人，此番直接拒絕後輩讓人不免覺得他有些小家子氣，甚至有球迷認為波波維奇簡直就是輸球又輸人。

其實，波波維奇之所以在賽後沒有和湖人教練組握手，主要是因為球隊打得太差了，老對手湖人的戰績如今高居聯盟第一，而馬刺則是6勝12負，排在西部倒數位置，很有可能連季後賽都進不了。即便賽前接受媒體採訪時他還興高采烈地稱讚了詹姆斯一波，並表示自己可不想被他打得找不著北，可最後結果真的是輸掉比賽時，波波維奇可沒有心情再來開玩笑了。鬱悶的他怎麼也想不到如今的馬刺竟然如此地不堪。

眾所周知，波波維奇堪稱NBA主教練的傳奇，一向被人尊重，無論他今天的表現是有意還是無意，我們只知道之前連續22年都進入季後賽的馬刺很有可能要終結這一神奇記錄了。每個人都會有情緒，哪怕面對著自己非常喜歡球員的球隊時，輸球始終都會讓自己感覺很煩躁，波波維奇正是如此才有可能沒有和湖人教練組握手致意！

另外，湖人在今天展現了他們的統治力，在濃眉哥狀態不佳的情況下，詹姆斯用實力捍衛了球隊的榮譽。沃格爾教練則是很好地完成了他的工作，所以他應該不會在意這件小事情身上。球迷朋友們對於波波維奇疑似拒絕和湖人教練組握手致意怎麼看呢？
