5年1億美元？曝東契奇將簽下一份天價合同！專屬簽名鞋也不遠了

2019-12-22 712條評論 3,384次閱讀 1072人點贊

北京時間12月22日，據美媒報道，獨行俠球星盧卡-東契奇已經接近和喬丹品牌簽下一份為期多年的贊助合同。這份合同預計將很快官宣，NBA記者Will Cain透露，這是一份5年1億美元的合同，而且東契奇將很快擁有屬於自己的簽名鞋。

東契奇還不滿21歲，他在本賽季已經成為一位MVP的候選人。此前，東契奇在2017年曾和耐克簽了兩年的球鞋合同，但那份合同已在今年9月30日正式到期。從那時起，東契奇在球鞋合同上成為自由球員。

美媒報道稱，彪馬和UA都對東契奇很感興趣。東契奇在訓練和比賽中曾經穿過庫里7代，而在熱身時也穿過彪馬的球鞋。

不過，東契奇更青睞喬丹品牌的球鞋，不只是在比賽中，他在場外參加活動時，也經常穿這個品牌的球鞋。據悉，東契奇將成為喬丹品牌在今年簽下的第二位重磅球員，此前，他們在夏天曾經用5年7500萬美元的合同簽下了錫安，那也是新秀有史以來得到的最豐厚球鞋合同。

目前，喬丹品牌致力於簽約一些有潛力的年輕球員。比如凱爾特人的塔圖姆、奇才的八村壘也已經和這一品牌簽約。

Bleacher Report認為東契奇其實在兩周前就已經暗示了自己的選擇。他們在社交媒體上寫道：「兩周前，盧卡（東契奇）說你不能拿任何人和邁克爾-喬丹相比。今天，盧卡就要和邁克爾-喬丹簽約了。」

賽季至今，東契奇場均貢獻29.3分9.6籃板8.9助攻。對熱火一戰中，他扭傷了腳踝，預計缺席兩周時間。如今，他已經開始投籃練習，恢復進展迅速。本賽季，他曾連續20場比賽至少得到20+5+5，超過了喬丹，歷史上僅次於羅伯特森。
