霍華德無腦失誤後，朗多舉動讓人感動！湖人隊的氣氛好的讓人嫉妒

2019-12-15 2,238條評論 3,428次閱讀 83人點贊

湖人又取得了一場勝利，113-110，擊敗了熱火的魔鬼主場，6連勝，以23勝3負繼續排名聯盟第一。

詹姆斯28分9籃板12助攻8失誤的准四雙，濃眉哥33分10籃板3蓋帽，麥基9分10籃板5蓋帽，丹尼-格林9分，波普15分，布拉德利10分，他們的表現都很不錯。

但我今天想說的是本場比賽表現並不算十分出色的朗多和霍華德。

朗多在養傷了2場後今天復出，本場比賽之前他的三分球命中率高達47.5%，但這場比賽他的手感顯然不在線，5投0中，其中三分球4投0中，一分未得，只拿到2板4助攻1搶斷，還有2個失誤。

霍華德3投2中只拿到4分8籃板，比賽中還出現了一個非常低級的失誤。他保護下了後場籃板，然後在沒有任何壓力的情況下大力長傳找朗多，結果直接扔出界，送給了熱火一次進攻機會。

這真是一個非常業餘的失誤，失誤後鏡頭給到霍華德，他也是一臉的懊惱，非常的苦悶。然而接下來的一幕讓人感動，湖人全隊沒有任何人責怪霍華德，作為這次失誤的另一名男主角朗多還主動走到霍華德身邊，和魔獸擊拳，讓他別在意。

是的，這就是湖人現在的氛圍。隊友打得好，每一個球員都會為他高興，包括替補席上的考辛斯。隊友打得不好，大家不是相互指責，而是告訴他，不要緊，下一個回合好好打，我們能贏的。

詹姆斯、朗多、杜德利，這3個更衣室的老油條，他們就像潤滑劑一樣，把湖人的更衣室整理得整整有條。

在全聯盟都不敢要霍華德的時候，詹姆斯對湖人管理層說，簽下他吧，我對他有信心。在訓練中，詹姆斯讓隊友指出他的不足，他會虛心去改。在丹尼-格林在場上開小差的時候，詹姆斯：「DG(丹尼-格林）你在散步呢，把那該死的上籃投進去！」丹尼-格林不好意思的笑了。

朗多就是湖人場上的教練，場下的半個教練。他和詹姆斯都是充滿大智慧的球員，即使是上賽季湖人那麼混亂的局面，朗多都始終堅定的相信詹姆斯。

杜德利就更不要說了，平時沒事發發大合照，讓大家更團結。當霍華德重返魔術，被昔日的小弟圍攻時，來活了。老杜一挑四，把魔術四個小兔崽子都推到了一邊。

「我不在乎我的隊友是不是做錯了，這就像是你保護自己的兒子，或者說是你家中的孩子，如果他們遇到了麻煩，你會站出來為他們做主，無論老師是不是針對他。因為那是我的兒子，而德懷特·霍華德是我的隊友，我將這個視為同樣的情況。」

NBA歷史上，實力已經達到了，但因為球隊內部出了問題最終導致失敗的案例比比皆是。甚至湖人的隊史上就出現過不少次。OK最後一年，霍華德第一次湖人之旅，都是這樣的情況。

湖人坐擁詹姆斯、濃眉哥、霍華德3狀元，還有丹尼-格林、麥基、朗多、布拉德利、庫茲馬等實力角色球員，聯盟第一似乎理所應當。但要知道這個球隊，超過一大半是第一次打球，他們還在磨合階段就能打出這樣的成績，真的可以說的上是一個大奇蹟。毫無疑問，球隊團結的氣氛在其中起到了非常大的作用。

氣氛好的球隊，不一定能奪得總冠軍。但能奪得總冠軍的球隊，氣氛一定是好的！
