盤點本賽季首批高薪低能的代表球員，雄鹿虧本嚴重，尼克斯最慘

2019-12-16 789條評論 2,559次閱讀 1126人點贊

在今年夏天的時候很多球員都獲得了大合同，什麼才能夠被稱為大合同，關鍵就看這份合同的總價值有沒有上5000萬，一旦上了5000萬之後對於一名球員來說這輩子應該都花不完了，可是想要獲得大合同並非是一件容易的事情，這還要取決於他們上個賽季的表現情況。

然而有些人拿到大合同之後似乎就沒有那麼上心了，關鍵是心態也有一定的變化，這就容易導致他們打不出應有的狀態，我們也叫高薪低能球員，那麼賽季已經打了20多場比賽了，各位球員的狀態大家心中也有一個定數了，接下來我們就一起來盤點一下幾位高薪低能的代表球員。

首先要提到但就是穆雷了，掘金隊的穆雷在今年夏天簽下了一份超級合同，5年總價值高達1.7億美元，顯然這是一位頂級球員才有的待遇。從上賽季他場均21分的表現來看，似乎不值得這樣的合同，不過鑒於他才22歲，還有提升的空間，所以掘金也願意給穆雷這樣一個機會。然而在本賽季，穆雷並沒有打出1.7億的身價，場均只能夠得到17.1分，比起上個賽季的數據不升反而降了，球隊戰績也不如上個賽季了，不得不懷疑他本賽季開始打起了養生籃球，好在他還有時間去證明自己，所以對於穆雷的低效狀態估計只是暫時性的。

接下來要說到的就是雄鹿隊的大洛了，大洛上賽季還拿著底薪合同，然而來到雄鹿之後他卻能夠砍下12.5分4.9籃板，尤其是他36.5%是球隊非常需要的，所以在賽季結束之後雄鹿給送上了一份4年5200萬大合同，大洛也可以憑藉這樣的合同養老了。既然是養老，那也就只能夠打出養老的數據了，目前他場均只得到10分，三分球命中率降至30%，顯然他上賽季的狀態只是曇花一現了，而他引以為豪的三分球也失去了准心，並且按照他目前的狀態想要改進應該是一件很困難的事情，不得不說雄鹿虧本很嚴重啊。

最後要說的就是蘭德爾了。蘭德爾上賽季21.4分8.7籃板的數據怎麼也算是不錯，而尼克斯給他開出的3年6300萬美元的合同怎樣看也是物超所值，不過這位打法強硬的球員本賽季變成了軟蛋，場均得分下降到了17分不說，投籃命中率也降至44.1%，更令人失望的是蘭德爾本賽季越打越往外了，他的三分球出手數增加到了3.5次，然而只能夠命中0.9個，引以為豪的內線命中率也降至51.75%，成為了一名十足的軟蛋球員，加上尼克斯目前的戰績，顯然蘭德爾讓他們腸子都悔青了，6000千萬竟然簽來了一名軟蛋。
