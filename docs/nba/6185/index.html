保羅：為重建球隊打球並不是最壞的結果

2019-12-05 0條評論 15次閱讀 0人點贊

克里斯-保羅今年夏天被交易給了重建的雷霆，一開始有消息稱他不會去新球隊報到，而是希望能通過交易或買斷的方式加盟一支更有競爭力的球隊。但最終，他還是留了下來，這也是他職業生涯第一次遠離他的家人——他的妻子和孩子目前住在洛杉磯。

今天接受採訪時，保羅表示，為一支重建球隊效力，並不是最壞的結果：「我知道情況可能會比這還要糟糕得多，還有別的人也在面對類似狀況。但那從來不是我的常態，我和妻子談過，我們認為在洛杉磯會給孩子們帶來穩定的生活。」

為了照顧保羅，雷霆會給他一些假期回洛杉磯與家人團聚，比如感恩節。保羅對此表示感激：「我很幸運，他們一有空就過來看我，我的父母、哥哥每個主場比賽都會來看。球隊待我很好，給我很多方便，所以我能經常看到我的家人。」

雷霆目前8勝11負，整體表現超出預期，但仍沒能達到季後賽的標準，對此，保羅表示：「我會努力掌控好我能掌控的事情，對我來說，那就是每天晚上為比賽做好準備，認真訓練，好好打球。」
