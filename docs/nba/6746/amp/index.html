湖人“第4主帥”上線！隆多現場開課，二當家戴維斯也得乖乖聽著

天天要聞
3 週 ago

11月26日，NBA常規賽，西部一場大戰，湖人隊客場挑戰馬刺。目前，湖人隊已經取得了7連勝，以14勝2負的成績，高居全聯盟榜首。本場，面對實力並不太強的馬刺，湖人隊目標只有一個：8連勝。

但真正打起來之後，湖人隊想輕易拉開比分也沒那麼容易，甚至在某些時段里，還極其被動。本場比賽，詹姆斯的狀態還是很不錯的，攻擊慾望也很強。然而，二當家安東尼-戴維斯似乎不夠興奮，在與阿爾德里奇的較量中幾乎被碾壓。或許是身體狀況不佳、或一些小傷病拖累了他吧。

在第二節還剩4分鐘時，場上發生有趣的一幕。看到安東尼-戴維斯陷入泥潭，表現掙扎，隆多終於憋不住了，他將濃眉哥叫到跟前，開啟了「現場教學」時間。隆執導上線，哪怕像戴維斯這樣的大牌兒，也得乖乖聽著。

湖人隊在教練方面，資源豐富。正印主帥沃格爾就不說了，助教團隊中也是卧虎藏龍。基德曾經率領過籃網、雄鹿，培養過字母哥。霍林斯也曾是灰熊主帥，帶隊打進過西部決賽。網友曾調侃沃格爾壓力巨大，身後總有人盯著他的帥位虎視眈眈，干不好隨時可能被人取代。這還是明面兒上的，場上還有兩位「隱形主帥」呢，隆多和詹姆斯，同樣讓沃格爾壓力巨大。

隆多喜歡開課，這個大家都習以為常了。詹姆斯雖然不經常教學，但在騎士的時候，也曾有過將泰倫盧擠到一邊的經典畫面。所以，湖人隊其實有「五大主帥」，覆蓋場內場外，隨時有人掌舵，這是一件幸福而又煩惱的事情，該聽誰的呢？

湖人隊兩大隱形主帥的福利，在本場比賽體現的淋漓盡致。隆多和詹姆斯兩大發動機，代表著湖人隊兩套輪轉陣容，尤其是隆多+戴維斯的組合，可以讓詹姆斯安心在場下休息。今天，戴維斯情緒不高，隆多挂帥就更加理所當然了，終於可以肆無忌憚地做自己了。
