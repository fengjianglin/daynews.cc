新領袖會晤昔日老大 錫安渴望助鵜鶘復興

2019-11-29 0條評論 15次閱讀 0人點贊

在湖人來到新奧爾良的那個夜晚，鵜鶘安全總監蘭迪·格林納普追上快要離開球館的安東尼·戴維斯，趕緊和老夥計寒暄一番，而站在他旁邊的還有另一個人，狀元秀錫安·威廉姆森。格林納普想讓鵜鶘的新領袖見見球隊的昔日領袖，完成這次火炬的傳遞。

「我告訴他要保持健康」，濃眉透露了兩人的談話內容，「毋庸置疑，他們在等待他的歸來，他們一直在堅持到他歸來。」

在10月21日接受右膝外側半月板關節鏡手術之後，錫安尚未打過一場NBA常規賽。戴維斯重返新奧爾良的那晚，身穿一套海軍藍西裝的錫安坐在場邊觀看了濃眉的完美表演：41分9籃板，以及那次殺死比賽的搶斷。

錫安的康復時間是六到八周，眼下這個時間點即將到來，但主教練金特里表示，錫安何時復出仍然沒有確切的時間。鵜鶘執行副總裁大衛·格里芬透露，球隊會謹慎行事，更傾向於八周的復出時間表，也就是在聖誕節前。目前，錫安已經可以承受部分負重，但還沒有重返球場進行對抗性訓練。

「他渴望回到球場」，金特里說，「他對待康復的態度非常非常謹慎認真，他已經迫不及待了，我們也很想讓他破籠而出，因為他是一名可以改變比賽的球員，誰不想讓他離開那裡？」

「對於他來說，最難的是打了季前賽，而現在卻不能出戰」，金特里繼續說道，「他肯定是一名非常特別的球員，因為他是將團隊放在第一位的球員，團隊為上的球員可以在這個聯盟取得成功，並且對他們的團隊產生影響。」

現在回過頭來看，鵜鶘似乎有點兒慶幸上賽季發生的鬧劇。

2018-19賽季開始之前，戴維斯就曾向球隊提出過交易申請。儘管鵜鶘之後解僱了總經理戴爾·鄧普斯，並且選中了錫安，但依然無法改變濃眉離開的執念。當濃眉在7月份被交易到湖人的時候，他實現了自己的願望。

同時，鵜鶘也解脫了。「他不想在這裡，對嗎？」格里芬反問道，「是時候繼續前進了，和湖人以我們滿意的方式達成交易。」主教練金特里回憶說：「很難，那種大環境很艱難，那不是第一次也不是唯一一次，那是NBA的家常便飯。你需要儘力處理，繼續前進。」

現在的鵜鶘有充分的自由相信他們已經走在正確的道路上。這個賽季，他們打出了一個新的口號「永不屈服」——這是古老的新奧爾良短語，與印第安狂歡節有關。這個口號在鵜鶘的球館和訓練場上隨處可見。在每個主場賽前，吉祥物皮埃爾也會敲著大鼓高呼這個口號。對於這個口號的解讀，格里芬說：「這反映了我們如何要打球，如何成為一個團隊，如何達到那樣的標籤。」

這只是格里芬自4月份上任以來諸多調整中的一個。過去的休賽期，鵜鶘聘請了阿隆·尼爾森擔任球員護理和表現副總裁，前NBA球員特拉楊·蘭登出任總經理，前WNBA球星斯溫·卡什就任籃球運營和團隊發展副總裁，前NBA後衛塞申斯和前特納體育安全總監奇科·羅賓遜也前來任職。此外，等待鵜鶘的還有一個全新的訓練館。

「這就是格里芬帶給這裡的文化改變」，金特里說，「球員們知道這裡的一切都是一流的，以前是沒有的。這能以一種更積極的方式塑造整體感覺和環境。」

在所有這些改變中，格里芬認為唯一不需要改變的就是主教練。格里芬和金特里曾在太陽共事過一段時間，他認為金特里就是帶領鵜鶘完成重建的正確人選。

「每個人都在說，為什麼不從一名新教練重新開始呢？」格里芬解釋說，「我不知道有多少教練能和我一起打進分區決賽，而且還是以一套相似的陣容。毫無畏懼的金特里曾在2010年率領太陽打進西部決賽，他還是他。我的工作就是盡最大努力為他提供支援，因為我親眼見過他出色地完成了工作。換帥絕不是一個好主意，當你想要了解一種文化的時候，你需要與一名志同道合的人開始交流。」

鉚足全力的鵜鶘已經打造了一個相當合理的陣容，兼具老將們的經驗和年輕人的天賦。朱·霍勒迪的合同直到2021-22賽季才到期，雷迪克去年夏天簽訂了一份兩年合約。明年夏天，22歲的英格拉姆將成為受限制自由球員，而格里芬已經做好準備，鵜鶘計劃讓鶯歌長留新奧爾良。

看看鵜鶘的配置吧。詹姆斯之後最有天賦的年輕人錫安，全聯盟最好的一批年輕人——鮑爾、哈特、賈克森·海耶斯、亞歷山大-沃克。此外，鵜鶘還從湖人的交易中獲得了三個選秀權，從騎士得到了另外一個， 19歲的巴西前鋒迪迪·盧扎達也在等待鵜鶘的召喚。

「我們有一些優秀的年輕球員，還有一批步入中年的球員，我們在選秀方面有很多選擇，我們可以建立這支球隊，就像格里芬所說，這是可持續的勝利」，金特里說，「對我來說，最重要的是持續贏得勝利。除非能夠持續性贏球，否則無法談論總冠軍。你要走好每一步，一步一步來，這一切不會在一夜之間發生。」

而這一切的復興又和一個人有關：健康的錫安·威廉姆森。

「沒人知道像他這樣的球員能達到什麼樣的頂峰，因為他是一頭獨角獸」，格里芬說，「曾經有人像他一樣嗎？我不知道他會成長為什麼樣。我知道的是他只關心贏下比賽。讓我興奮的是，我們有機會追逐一些特殊的成就，而他將成為其中特別的一員。」

雖然錫安現在只能作壁上觀當一名看客，但用不了多久，他就能在冰沙國王中心聽到歡呼聲，享受屬於他的特殊待遇。而當那個時刻來臨的時候，鵜鶘期待著他能立即展現出自己的影響力。

「他是一頭怪獸，擁有錫安真的很棒，他確實震驚了我」，霍勒迪說，「顯然，19歲的他剛從大學畢業，內心還是個孩子，但他的處理方式就像是最好的職業球員。」

「他真的很努力，他的感覺也很好，」格里芬說，「他對於比賽無窮無盡的快樂，燦爛的笑容，那是我們需要的。我們需要那份能量。」
