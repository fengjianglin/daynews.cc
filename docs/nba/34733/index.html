勇士要讓庫里和格林發揮最大功效，他們需補強這一個位置

2019-12-25 812條評論 1,723次閱讀 803人點贊

勇士隊16年失冠也是因為後三場追夢格林禁賽以後博古特受傷，戰鬥力不能集中起來爆發造成的。這就像玩遊戲似的，你三次團戰每場缺一個大， 別人每次開團都有五個大，大家都正常發揮肯定難贏。

17-18賽季有了杜蘭特可以彌補中鋒的缺陷，可以補足戰鬥力，可以掩蓋中鋒位置的不足。

19年失冠的原因主要有三

1、管理層不應該為了一個帶傷的表妹而放棄麥基

（1）畢竟是沖三連冠的緊要關頭，考辛斯未磨合，而麥基配合兩年；

（2）考辛斯重傷恢復前景不明，麥基很健康；

（3）不是所有的隊伍都是火箭，有哈登和保羅；

（4）大衛-韋斯特年老退役，但是麥基並不老。

2、 教練沒有控制好場上情緒，不應該放任杜蘭特和瘋狗互相撕咬

教練沒有合理使用庫里，長年累月的跑無球 ，這還是那個日天的庫里嗎？科爾一個角色球員出身把超級球星用成了替補球員也是沒誰了。 多米諾骨牌效應，考辛斯倒下、杜蘭特過度消耗倒下，如果有健康的麥基去分擔場上時間，靠他的高度速度就可以拖到主力恢復體力，怎麼會打快船被拖傷。

3、沒有中鋒，格林的補防功能毫無意義

還不如換一個能護框有速度的，格林是要在有中鋒護框，協防才有用，不然的話，勇士的防守體系破了個大洞，誰也補不起來。

未來展望

即使庫里湯神恢復，但是看著這個管理層和科爾就讓人頭疼。科爾只打80-95分能力的隊伍，但是現在給他個70分的隊伍，他沒能力把他們帶到80分90分去。看管理層能不能補上一個合適的中鋒，自從博古特受傷以後，這個問題一直沒解決，現在杜蘭特走了，為什麼還不補上，沒中鋒，要格林有什麼用？

另外，顧問、隊醫的缺口很難補上，庫里湯神過於佛性，不會爭取組織更好的隊伍。

總體來說，勇士的未來還是比較悲觀,
