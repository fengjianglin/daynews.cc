光拿錢不幹活？很快就能擺脫6大垃圾合同，這些球隊想想就開心

2019-12-18 1,111條評論 2,051次閱讀 923人點贊

每年自由市場都會湧現出一批又一批的大合同，有些符合球員的身價，有些則淪為阻礙球隊發展的垃圾合同。簽了垃圾合同的球隊自然是苦不堪言、悔不當初，只能一忍再忍，等到垃圾合同到期球隊就能重見光明。這次我們就來盤點一下明年夏天到期的垃圾合同，看看誰能榜上留名？

比永博 4年7200萬美金

上帝為你打開一扇窗就會為你焊死一道門，這句話放在喬丹身上再合適不過了。作為籃球運動員，喬丹被譽為籃球之神，而作為球隊管理層喬丹就是當之無愧的小白了。先不談他將球隊培養了許多年的全明星控衛肯巴沃克轉手送人，單看比永博這年薪1700萬美元的大合同，你應該就能感受到喬老爺子的天真了。好在比永博這份合同賽季結束後便到期了，要不然喬老闆還得在上邊虧錢。

帕森斯 4年9444萬美金

我們現在提到帕森斯第一時間想到的不是他那出色的球技以及帥氣的面孔，而是花邊新聞以及讓球隊苦不堪言的大合同。不得不說帕森斯有一種神奇的魔力，他總能在合同年打出漂亮的數據，讓許多球隊口水流不停。結果灰熊就上當了，給帕森斯提供一份4年9444萬美金的合同，後來的故事大家也都知道了，灰熊花了大代價把拿錢躺傷的帕森斯交易到了老鷹隊。不過在合同年帕森斯場均只能得到3.5分，約等於720萬美金1分，拿著童工合同的特雷楊想必已經哭暈在廁所了。

特納 4年7000萬美金

擦乾眼淚的楊不悔剛從廁所出來就遇到了隊友特納，特納問清楚了事情的緣由露出了笑容驕傲地說道：「幸好我比帕森斯場均多得0.4分。」你可能不會想到本賽季場均只有3.9分2.1籃板的特納居然是10年的榜眼，雖然在76人場均能夠得到13.3分6.3籃板，但萬萬沒想到的是這已經是他的巔峰。此後特納輾轉綠軍、步行者和開拓者都沒有證明自己的天賦。不過我想特納是喜歡開拓者這個隊伍的，畢竟在這裡他拿到了4年7000萬美金的合同。

克拉布（4年7483萬美金，明年到期）

心情十分鬱悶的楊少俠走回更衣室的路上碰見了另一個隊友克拉布，克拉布十分友好地安慰小楊：「你看我每天得到5.8分就已經很開心了，每天砍下27.8分的你為什麼悶悶不樂呢？」是的，拿著1850萬美金的年薪，場均得到五六分就能完成任務，這種錢多事少的工作誰不想干呢？今年場均5.8分的克拉布已經可以宣布這是他職業生涯最後一份千萬合同了。突然楊不悔似乎想到了什麼：明年夏天他們三人的合同全部到期，馬上就要和如此友愛的隊友們分別了，那可真是太舒服了呀。

奈特 4年7200萬美金

自從詹姆斯離隊，騎士重新走上了收集狀元簽的老路子，全隊上下兢兢業業已經豪取6勝20負排名東部倒三，其中後衛奈特功不可沒。球隊薪金排名第三（年均1564萬美元）的他，場均交出4.9分的成績單，排在全隊的第十位。要知道奈特明年夏天合同到期，但還是為了球隊擺爛大計鞠躬盡瘁。奈特可能覺得這樣就會有人覺得：他是顧全大局，而不是能力不行。

貝茲莫爾 4年7000萬美金

開拓者今年夏天送走了特納迎來了同等價位的貝茲莫爾，後者向開拓者管理層證明了贏球輸球和特納似乎沒什麼關係。即便杯子貓場均得分比特納高了一倍，球隊戰績卻不如去年，能不能進季後賽都是兩說。目前排名西部倒數第四的開拓者似乎對贏球沒有什麼好辦法，唯一值得高興的是明年夏天貝茲莫爾的合同到期，他們終於能在市場上籤簽人了。
