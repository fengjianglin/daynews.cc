霍弗德笑談球盲鑒定器 恩比德與歐文很不同

2019-12-06 0條評論 14次閱讀 0人點贊

美東時間12月5日，奇才主場對陣76人。按照慣例，客隊都會在比賽日上午進行公開的投籃訓練。而有趣的是，今日76人並沒有把訓練地點放在奇才主場Capital One Arena，而是放在了籃球名校、也是76人傳奇巨星阿倫•艾佛森的母校——喬治城大學的體育館裡。而對於這一安排，76人方面給出的解釋很「流川楓」：「因為離酒店近。」

在訓練結束後筆者也代表體壇對76人球星，本賽季剛剛從凱爾特人轉會到76人的艾爾•霍弗德進行了採訪。遇到霍弗德的時候他正敷著冰袋坐在場邊休息，而他也很耐心而熱情地接受了採訪。

體壇：「中國球迷稱你為『球盲鑒定器』，因為雖然你的數據並不出眾，但你對球隊的作用非常大。只有真正懂球的人才能看出你的作用。」

霍弗德：「你知道，我總是在嘗試任何能幫球隊取勝的事情，我們在場上做的事情，都是為了幫助球隊取得勝利，這對我來說是最重要的。」

體壇：「波士頓和費城有何不同？」

霍弗德：「兩隻球隊有不同的風格，在費城我們會更多進行背身單打：恩比德、西蒙斯還有我自己，而凱爾特人更多強調的是移動，這兩隻球隊有不同的風格，這是最大的差異。」

體壇：「歐文和恩比德，你認為他們在領導球隊方面有差異嗎？」

霍弗德：「是的，有差異。歐文非常擅長中距離得分，可以撕破對手的防線，在場上可以做很多事情。而恩比德是有統治力的中鋒，他可以背打，可以把球傳出去，或者把球傳給內線的其他人，他是一個內線大殺器。」

體壇：「你從2007年開始就已經在聯盟征戰，你是如何保有如此長的生涯的？」

霍弗德：「你知道，我一直竭盡所能保養我的身體，我一直很愛護自己。NBA的賽季非常長，對球員要求很高，最重要的是讓自己感到舒適，我一直堅持這樣的一種生活方式。我只是確保做著正確的事情，做一切能幫助我恢復的事情，讓我感覺舒服的事情。」

體壇：「你一直被稱為很聰明的球員，你是如何提高自己的？」

霍弗德：「持續從比賽中學習。你知道比賽的風格一直在改變，自從十二三前我剛進入聯盟至今，比賽的風格真的發生了劇變。你需要不斷做出調整，就像現在大家都開始扔三分，你就也需要扔三分，這是籃球運動員需要在場上做到的。」

體壇：「你認為今年76人可以奪冠嗎？」

霍弗德：「我們需要打出更好的防守，在防守上投入更多。而在進攻端，我們需要對彼此更加了解，不斷學習和成長。我認為這會讓我們在角逐冠軍的路上佔據有利位置。」

最後霍弗德也向所有中國球迷打了招呼：「你好，中國的所有人，我所有的粉絲，我是艾爾·霍弗德，請繼續支持76人隊，我們將會努力去贏得冠軍。Go 76人！」

筆者也向霍弗德介紹了他的中文名如何念，以及他的名字和「魔獸」霍華德在中文中易混淆的問題。而霍弗德很認真地嘗試念了好幾遍自己的中文名字。

很感謝為人儒雅隨和的霍弗德接受體壇的採訪，而今晚76人隊以113比119不敵奇才隊，霍弗德拿下11分5籃板，沒能幫助球隊取勝。

正像他說的那樣：「我們需要在防守端投入更多。」如何加快磨合，提升攻防兩端，76人的爭冠之路任重而道遠。
