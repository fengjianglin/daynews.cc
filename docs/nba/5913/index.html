騷操作！魔術隊為何不願給奧尼爾頂薪？看完你就明白了

2019-12-09 0條評論 18次閱讀 0人點贊

奧尼爾是NBA歷史上最偉大的內線球員之一，在他的職業生涯中他獲得了很多的榮譽，其中四個NBA總冠軍跟三個FMVP就是對他實力最好的證明。那麼曾經在魔術就已經成為頂級中鋒的奧尼爾，為何魔術管理層不選擇給他頂薪而選擇放走呢？

在當時奧尼爾加哈達威的組合，兩年把魔術送到總決賽的舞台，雖然都未能奪冠但是以證明他們的強大。所以奧尼爾在合同到期的時候，是想留在魔術然後拿一份過億的長期合同，然而，奧尼爾的期待卻換來的是一份7年6900萬美元的合同。

甚至還在主城做了一份調查，問粉絲們奧尼爾到底值不值得一份億元合同，讓人覺得驚訝的是大多數人都認為奧尼爾不值。要知道當時很多不如他的球員，都可以拿到一份億元合同，如此不公的待遇也是讓奧尼爾最終選擇出走。

還有一個原因是，當時雖然還是中鋒的時代，但是由於喬丹的出現，讓人們知道了後衛也能帶隊奪冠。所以哈達威也順利地成為了當時球隊的重點培養對象，在城市的威望也開始高於奧尼爾，要是給奧尼爾頂薪，哈達威肯定不願意，所以二選其一管理層選擇了哈達威。
