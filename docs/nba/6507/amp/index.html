足球國度喀麥隆，是怎麼發現恩比德、西亞卡姆這樣的籃球天才？

天天要聞
3 週 ago

如果說起喀麥隆，大家第一印象自然是足球。這個被譽為「非洲雄獅」的國度，共7次晉級世界盃決賽圈，比任何非洲國家都多。同時，他們是第一支打進世界盃八強的非洲球隊，另外曾贏得過5次非洲杯冠軍和2000年的奧運金牌。

可是，就在這樣一片人人都熱愛足球的土地上，卻綻放出兩朵奇異的籃球之花，恩比德與西亞卡姆。如果說現在評選現役NBA前十的內線球員，他們兩人一定會在其中。畢竟恩比德場均12.1個籃板位列聯盟第8，除了賤兮兮的性格和碎嘴外，他是76人最重要的球員；而西亞卡姆在上個賽季獲得最快進步球員獎後，這個賽季真正迎來了爆發。場均26.0分位列聯盟第9，正負值為+173冠絕全聯盟。

可事實上，如今在聯盟里風頭正盛的兩個人，人生都差一點與籃球失之交臂。而改變他們人生軌跡的是這個人——巴莫特。

如今的恩比德，已經是大帝。但如果沒有巴莫特的話，他現在可能只會在歐洲打排球，更不會被那麼多人熟知。恩比德的父親是一名手球運動員，所以恩比德繼承了父親優秀的運動天賦，同時因為家境優渥，恩比德從小有更多的機會接觸體育運動。

他最開始接觸的和每個喀麥隆孩子相同，都是足球。但是隨著身高暴漲，恩比德並不適合足球了，於是他開始接觸排球。當時他的父親已經開始打算將恩比德送去歐洲接觸更加專業的排球技術。

就像每個個子高的男生都會被問會不會打籃球一樣，恩比德也並非完全不接觸籃球。當時，巴莫特回到喀麥隆舉辦籃球訓練營，恩比德也是第一次接觸籃球比賽。當時的恩比德還是個害羞靦腆的男孩，面對NBA正牌球員巴莫特，他只是認真地訓練。

但巴莫特在訓練營中發現了天賦出眾的恩比德，畢竟很多動作別人學七八遍都不行，而恩比德則能迅速掌握。隨後，巴莫特將恩比德帶到美國，開始了自己的籃球之路。

而與恩比德不同的是，西亞卡姆則沒有那麼好的家庭條件。但他的父親特查莫-西亞卡姆是一名狂熱的NBA球迷，他會把四個兒子喊到一起，跟他們講述NBA天才球員的故事。而他的夢想則是四個兒子中，能有人進入NBA。

但小時候的西亞卡姆對籃球並不感興趣。「哥哥們一直勸我，所以我會打一會兒籃球，」帕斯卡爾說，「但我不喜歡，從沒打算以此為職業。」

父親知道後，希望讓西亞卡姆成為一名牧師。但相比籃球，牧師更讓他抵觸。而在2012年，巴莫特在家鄉舉辦了籃球無疆界訓練營，他的三個哥哥拉著不情願的他一起去參加。而當時在籃球無疆界的活動中，還在掘金的籃球運營總裁馬賽-烏傑里看中了他。

這位日後為猛龍打造出贏得第一個總冠軍陣容的人，也是當時巴莫特訓練營和NBA籃球無疆界非洲活動的主理人，也在4年後親手選中了西亞卡姆。隨後，西亞卡姆同樣選擇了來美國打球的方式，而他在美國的生活，大部分費用都是巴莫特提供的。

即便如今成為猛龍核心的西亞卡姆，依然覺得不滿足：「我總是會做一些事情來確保每天都能打磨自己的球技，我總是感覺我起步的時候已經很晚了，這就是我的心態，我必須想辦法趕上那些五歲就開始打球的人。」

可以說，如果不是巴莫特發現了恩比德和西亞卡姆，喀麥隆男籃將少了兩個頂級球星。而在喀麥隆的土地上，或許還有很多像恩比德與西亞卡姆這樣的年輕人等待著被挖掘。

有的時候，天賦確實最容易被看見，但同時也最容易被埋沒。千里馬常有而伯樂不常有，巴莫特就像啟蒙老師一樣，引領著恩比德和西亞卡姆走上了籃球的道路。

在體育的世界裡，有很多關於這樣的故事，但相信這樣的故事不止發生在職業運動員身上，也曾發生在你我身邊。
