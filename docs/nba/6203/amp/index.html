霍華德安東尼越打越好，JR心動了，立刻採取了行動

天天要聞
2019-12-04

看到霍華德安東尼錄訓練視頻後，都順利找到了工作，而且如今混得風生水起，JR也忍不住了，曬出了一段訓練視頻，為下一份工作做好了準備。

文/聰古說球

霍華德和安東尼經歷了上賽季的落魄、迷茫之後，都在新賽季找到了自己的新工作，並重新證明了自己，8月27日，霍華德與湖人簽下一份底薪合同，今年11月20日，安東尼與開拓者簽下一份底薪合同。

霍華德加盟湖人之後，已經成為湖人替補席上不可或缺的一員，他出色的防守以及護筐能力，是湖人成為西部第一的主要原因，安東尼也一樣，他加盟開拓者之後，已經坐穩了首發的位置，因為他的出色表現，被NBA官方評選為西部周最佳球員。

看到霍華德和安東尼不僅找到了工作，而且發揮那麼出色，去年同樣落魄、被老東家放棄，至今還失業的JR史密斯忍不住了，終於採取了行動。

失業在家的JR與他的4個女兒合影，值得一提的是，他和科比一樣，都是4個女兒，沒有兒子

近日，NBA訓練師Chris Brickley在推特上曬出JR-史密斯的最新訓練視頻，JR在視頻中主要訓練了投籃，只見他穿著休閑服和籃球鞋，在球場上各個投籃點奔跑投射（各種定點、接球跳投三分），投了幾十次籃，命中率高達95%左右。

從視頻來看，JR依舊保持了很好的體型和競技狀態，霍華德和安東尼沒找到新工作之前，都是通過不斷的努力訓練，有很多訓練視頻相繼流出到網上，很快吸引了球迷、媒體、球探的關注；

​最終功夫不負有心人，霍華德和安東尼都找到了適合自己的球隊，而且他們也沒有辜負球隊的信任，他們出色的表現，最大程度幫助了球隊提升了實力。

截止到目前為止，霍華德出戰21場比賽，幫助球隊取得18勝3負的聯盟第一戰績，在不到20分鐘的有限出場時間裡，可以得到7分6.9籃板（2.1進攻籃板）1.43蓋帽，其中命中率高達73.8%為聯盟第一，蓋帽聯盟第14，正負值聯盟第46，他和濃眉、麥基組成的內線，是聯盟其他球隊的噩夢。

在安東尼未加盟之前，開拓者只取得5勝9負聯盟墊底戰績，勝率只有可憐的35.7%，得到安東尼後，開拓者勝率提升到42.9%，代表開拓者的7場比賽中，安東尼場均16.4分6籃板2助攻，投籃命中率44%，三分命中率37%，上周率領球隊豪取3連勝佳績，個人更是場均轟下22.3分7.7籃板2.7助攻數據，他上一次當選周最佳還是2014年3月10日，那時候的他還是尼克斯的老大。

如今看到霍華德和安東尼在聯盟混的風生水起， JR看到心痒痒，似乎想走的老路，於是終於忍不住跑去訓練，並將視頻流了出來，就是希望能夠得到球迷、媒體和球探的關注，目前的他還是自由球員，他的投射能力還很強。

1985年9月9日出生的JR史密斯，目前還不到35歲，作為曾經克利夫蘭騎士隊的冠軍射手，他的球風充滿著激情，而且專門為大場面而生，騎士勇士連續4年的總決賽中，JR一直都是騎士隊中最穩定、最冷血的三分射手之一。

目前的JR依然非常渴望打球，他這種經驗豐富又有三分投射能力的球員，非常適合那些志在奪冠，同時又缺少穩定三分射手的球隊，說道這裡，想必大家已經猜到哪些球隊適合JR了吧。

詹姆斯暴力隔扣，現場觀戰的JR，眼神里滿滿都是愛和崇拜
