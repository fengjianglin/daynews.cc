熱火戰術大有不同？斯波：說得太過了，有六成是上賽季的

天天要聞
2019-12-21

虎撲12月20日訊 近日，熱火主帥埃里克-斯波爾斯特拉接受了記者的採訪，談到了球隊本賽季在進攻端的進步。

本賽季，熱火進攻效率值排在聯盟第九，平均每百回合能得到110.1分。球隊也取得了20勝8負的優秀戰績。

而作為對比，熱火上賽季進攻效率值排在第21，平均每百回合得分為106.7分。

「改善進攻是教練組每個人的頭號任務。每支球隊都有不同的長處。作為教練組，我們今夏想要集中注意力改善球隊進攻。」

「另外我們認為我們進攻端的進步也很大程度上得益於球隊的人員構成。是的，我們有了一位頂薪球員，這很清楚。這就是我們之前全力追求吉米（巴特勒）的原因。然後，我們球隊的球員也很適合吉米。有投射，有組織者，還有就是巴姆（阿德巴約）的進步。所有這些都幫助我們的進攻朝著更有效率的方向進步了。」

有人認為熱火本賽季進攻戰術大有不同，對此，斯波說：「說的太過了。如果你去問其他球隊的球探，我們打的很多戰術都是上賽季的。不一樣，戰術不可能完全一樣，但是有六成都是一樣的。這六成是我們這個賽季打得很好的戰術，因為球員們的表現讓這些戰術看上去很不錯。」

談到巴姆-阿德巴約的進步，斯波教練說：「這是關鍵。球隊的很多進步都不是一夜之間完成的，阿德巴約在過去兩年里都在進步，我們上賽季和阿德巴約一起做了很多努力。但是，這個夏天，阿德巴約把所有時間都花在了改善進攻技巧上。」

「然後，我們得到了像吉米這樣的球員，還有鄧肯（羅賓遜）肯德里克-納恩以及泰勒（希羅），他們彼此成全。阿德巴約現在的進攻技能能夠激活他們。」

(編輯：流浪的光)
