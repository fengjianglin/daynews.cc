半場14分 人間大炮再啟動！不怵快船超級替補群 奇才半場只輸1分

2019-12-09 0條評論 16次閱讀 0人點贊

快船隊大比分輸給雄鹿隊之後，繼續客場的賽程，前往華盛頓挑戰奇才隊。奇才隊的實力並不夠強，這給了快船隊反彈的最好機會，但是奇才隊光腳不怕穿鞋，從來就不會坐以待斃，之前對陣76人隊的比賽，奇才隊就將東部豪強拉於馬下。奇才隊靠進攻維持生命線，一旦他們在進攻端打開就很有威脅。

貝爾坦斯在對陣76人隊比賽中

奇才隊雖然只有布拉德利-比爾一名明星球員，但是新秀八村塁以及替補得分狂人貝爾坦斯都有不錯表現。尤其是本賽季交易得到的貝爾坦斯，更是球隊意外撿到的一塊寶貝。貝爾坦斯上賽季效力馬刺隊的時候，他的外線投籃就比較出色，在奇才隊獲得更多機會的情況下，貝爾坦斯越打越出色。

貝爾坦斯被球迷稱為「人間大炮」，他的外線投籃極准，這架高炮台在外線的火力源源不斷。奇才隊能夠擊敗76人隊，貝爾坦斯三分球13投7中就起到了關鍵作用，他讓76人隊的防守陣型被徹底打散。外線投籃是貝爾坦斯的主要得分手段，賽季場均得到14.4分，而他每場比賽可以命中3.7記三分球。

曾是波波維奇愛徒

對陣快船隊的比賽，貝爾坦斯首節比賽打了三分鐘，只有一次出手就命中三分球，首節比賽得到3分。貝爾坦斯作為輪轉球員，第二節比賽才是他的主戰場，而貝爾坦斯的表現也沒有讓人失望。第二節比賽繼續在三分線外不斷開炮得手，快船隊對他的防守相當被動，甚至還在他外線投籃時出現犯規。

八村塁曾稱讚貝爾坦斯，「他是我見過的最好投手」，貝爾坦斯的精英級別射術，就讓快船隊嘗到了厲害。快船隊的防守很難對得上這尊高炮，貝爾坦斯的強投也因為身高優勢而很難被干擾到。上半場比賽，貝爾坦斯打了10分鐘比賽，7投4中（其中三分球6投3中），罰球3罰3中，貢獻全隊最高的14分。

哈雷爾和路威

眾所周知的是，快船隊替補陣容的進攻非常兇猛，坐擁兩大板凳匪徒，快船隊總是能在輪轉階段轟塌對手半塊陣地。然而奇才隊的替補球員毫不遜色，本場比賽奇才隊靠著替補陣容的出色表現挽回劣勢，貝爾坦斯半場比賽的正負值高達+12。奇才隊也是因為替補陣容的優勢，半場比賽僅僅落後快船隊1分。
