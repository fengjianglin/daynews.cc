瘋狂1夜！恭喜湖人擊敗快船，濃眉喜訊，半獸人被裁，德帥難續約

2019-12-10 1,043條評論 2,126次閱讀 57人點贊

12月10日，NBA常規賽正在火熱進行中。在觀看比賽的同時，NBA昨夜又傳出了不少重磅的新聞。

瘋狂一夜！恭喜湖人，擊敗快船，濃眉哥迎來了喜訊，火箭隊舊將半獸人被裁，德帥難續約。

NBA最新的戰力榜

NBA公布了最新一期的戰力榜，其中雄鹿隊繼續領跑。而以21勝3負，領先雄鹿隊0.5個勝場，戰績領跑聯盟的湖人隊，則繼續排在第二名。

湖人隊繼續高居戰力榜的第二名

17勝7負的快船隊，暫時排在西部第二的位置，他們的戰績是17勝7負。從上周戰力榜的第四名，上升至第三名。

快船隊排在戰力榜的第三名

這意味著，湖人隊還是能夠在戰力榜上擊敗快船隊。湖人隊有詹姆斯和安東尼戴維斯坐鎮，最近取得了4連勝的戰績。

而快船隊，則在近期先後被馬刺隊和雄鹿隊擊敗。從實力和比賽結果來看，萊昂納德和保羅喬治領軍的快船隊，並沒有想像中的那麼強大。

湖人隊在戰力榜上力壓快船隊

在昨天對陣森林狼隊的比賽里，轟下了50分的湖人隊內線悍將安東尼戴維斯，迎來了喜訊。NBA官方公布，熱火隊的吉米巴特勒和湖人隊的安東尼戴維斯，分別當選為最新一期的東西部賽區周最佳。上周的濃眉哥，場均可以轟下35分8板3帽，他在攻防兩端都打出了強勢的表現。

濃眉哥當選了周最佳

得到周最佳，這對於濃眉哥而言，是實力的高度肯定。在詹姆斯的幫助之下，濃眉哥已經成為了聯盟里最具有統治力的內線球員。

火箭隊方面，也有兩個消息。根據Sportando記者Emiliano Carchia報道，CBA浙江廣廈已經裁掉了法里德。

CBA廣廈裁掉了法里德

半獸人法里德，上賽季還在為火箭隊效力，在卡佩拉受傷期間，法里德一度打出了驚艷的數據和表現。不過，法里德在季後賽並不受到主帥德安東尼的重用。一年合同到期之後，沒有球隊簽下法里德，他只能選擇加入CBA。

火箭隊舊將法里德被裁，這意味著法里德有機會重返NBA，繼續在NBA尋找下家。

同時，The Athletic記者Sam Amick和John Hollinger聯合報道，德安東尼下賽季繼續執教火箭的可能性很小。

德安東尼難以被續約

消息源稱，除非火箭本賽季取得突破性的進展（比如火箭總冠軍），不然費爾蒂塔是不會打開自己的錢包來留下德帥的，消息人士的原話稱，德安東尼在下個賽季執教火箭的可能性「很小（slim）」。

德安東尼所帶領的火箭隊，擁有著很好的進攻。但是，哈登與威斯布魯克的組合，卻難以打出強勢的防守，這是他們戰績下滑的主因。火箭隊想要在季後賽走得更遠，就必須尋求交易，繼續提升防守。

火箭隊德安東尼與哈登、威斯布魯克

畢竟，防守更好的球隊才有機會在季後賽走得更遠，才可能距離NBA總冠軍更近。
