134-109，快船大勝馬刺。比賽一上來，德羅贊率先打進2+1，為球隊開局。但萊昂納德也不示弱，立馬中投還以顏色。隨後，喬治再添一把火，連突帶投，幫助快船取得領先。好在，德羅贊單節7中6砍下13分，及時止血，沒有讓快船將比分擴大。但是次節路威廉姆斯又來了，先是打成2+1，之後又連續三分命中，分差瞬間就來到兩位數。易邊再戰，萊昂納德和喬治一頓操作，將分差擴大到20分開外。反觀馬刺這邊，由於三分線沒有啥穩定的火力輸出，面對快船的聯防，始終沒有啥好辦法，並出現了大量的失誤。而在防守端，身體天賦又不夠，波波維奇甚至用起了只要萊昂納德和喬治一持球，就瘋狂包夾的戰術。但也沒有出現好的效果，反被萊昂納德各種秒傳祖巴茨吃餅。最終，快船火力全開，三節打卡下班。

數據方面，萊昂納德26分7籃板9助攻，喬治11分5籃板4助攻，路威20分4助攻，哈雷爾21分8籃板，祖巴茨15分5籃板，哈克萊斯15分2籃板。馬刺這邊，德羅贊24分3籃板3助攻，阿爾德里奇12分4籃板，穆雷10分5籃板6助攻，貝里內利17分2籃板，懷特10分3籃板5助攻，米爾斯9分。隨著GDP退役，波波維奇也在考慮不執教了，昔日伴隨很多球迷青春歲月的馬刺王朝，真的要落幕了。首先，這支球隊目前單論天賦，確實是比不了。後衛多，但都優缺點明顯，在缺少鋒線的情況下，阿德漸漸開始進入職業生涯末期。比賽中，波波維奇試圖用戰術去彌補，現實很殘酷。值得注意的是，落後17分上沃克，50秒後換下，賽後波波朝著快船替補席豎起大拇指

在比賽第三節，萊昂納德面對德羅贊的防守，馬刺正準備去包夾，萊昂納德馬上秒傳祖巴茨，後者籃下打進。之後，喬治吸引防守，大範圍傳球給到貝弗利，大空位三分沒有投進，可是祖巴茨搶下籃板，二次進攻輕鬆放進。緊接著，萊昂納德搶斷一條龍，即便穆雷和福布斯很快退防到位，但小卡還是單手暴扣。萊昂納德打完了，這正痛著呢，喬治打成高難度2+1.無奈之下，波波維奇大手一揮，這哥倆只要一拿球就包夾。可萊昂納德又秒傳貝弗利，反手上籃打進，哈克里斯又接喬治傳球，三分線外空位不投，突進去又是一個暴扣。

眼看著又防不住了，在6分07秒的時候，德羅贊造成犯規，站上罰球線，此時馬刺落後到17分。波波維奇換下福布斯，換上沃克。意圖很明顯，福布斯雖然能投，但防守完全對不上。沃克能持球攻，有三分，防守也還行。尷尬的是，上來面對的是萊昂納德的防守，突到內線，喬治又過來了，這兩人防守啥水平，看著都怕。只好傳到外線，結果被哈克里斯搶斷，又是一條龍暴扣，還造成犯規。解說員都忍不住表示：「這是特別泄氣的一球，剛把你換上來，這球哪怕你接球蹦當扔了，沒進，效果都比這強。」話音未落，沃克是投了，但三分沒進，差點又被萊昂納德打成反擊。5分17秒，僅僅上場50秒，波波維奇直接換下沃克上米克斯，防守我不要了，落後這麼多，乾脆就飈吧，結果快船三節打卡下班。

賽後，波波維奇和快船主教練里弗斯以及助教擁抱致意完，朝著快船替補席豎起大拇指，轉身離開，低頭回更衣室。沒辦法，兩隊實力差距實在是太大了。一個萊昂納德就防不了了，你還有一個喬治。用包夾吧，這哥倆還能傳。更要命的是，就目前馬刺的鋒線，拼天賦，哈克里斯也不一定處下風。此外，替補還有一個王炸組合，路威廉姆斯和哈雷爾，雙雙20+。後衛線上，德羅贊，穆雷，懷特，三分都不是他們拿手技能包。里弗斯一擺聯防，攻不進，防不住，就是馬刺全場掙扎的寫照。再看看快船這邊，後衛線，路威廉姆斯能攻，貝弗利能防，沙梅特能投。這也難怪波波維奇豎起大拇指，然後默默低頭回更衣室。想想以前的馬刺，帕克，吉諾比利，格林，萊昂納德，鄧肯，還有一個特別能攪和的迪奧，滿滿的都是昔日的影子。
