雙槍43投僅15中，開拓者全場僅4記三分！他們在主場給鵜鶘送溫暖

2019-12-24 25條評論 1,282次閱讀 210人點贊

北京時間12月24日，波特蘭開拓者在主場以94-102負於新奧爾良鵜鶘，此役，安東尼砍下23分9籃板的高效數據，但是麥科勒姆和利拉德發揮欠佳，兩人43投僅15中，利拉德更是三分線外10投9中，最終開拓者輸給了鵜鶘，球隊的四連勝也戛然而止，目前球隊仍然排在西部第八。

此役，開拓者面對西部倒數第二的鵜鶘，他們竟然全場只投進了4個三分球，三分球命中率低得可憐，只有13.79%。除此之外，他們竟然讓鵜鶘在籃板球方面搶下55個，而開拓者只有48個，助攻端，鵜鶘也以23-17領先於開拓者。更重要的是，這場比賽最後時刻霍樂迪的關鍵三分球幫助球隊鎖定勝局，殺死比賽。

本場比賽，利拉德和麥科勒姆失准，雖然麥科勒姆還能夠得到20分，但是他22投僅9中，命中率不高。相比之下， 球隊老大利拉德更是失去准心，21投6中，其中三分球10投0中的糟糕手感讓全隊的三分球命中率都奇低。在本賽季開始階段，開拓者曾度過了一個艱難的時刻之後，球隊目前終於殺進了西部前八，而意外輸給鵜鶘，開拓者也相當於給對手送上溫暖。

值得一提的是，本場比賽，原本開拓者是有機會能夠將比分追平的。當第四節過半的時候，巴澤摩爾命中三分球，開拓者將比分追成91-95。可惜關鍵時間他們手軟，接下來5分鐘未能投中一球。鵜鶘打出一波6-0，在比賽還有3分36秒時以101-91領先。鵜鶘此後也連續投籃不中，直到比賽結束，都沒有運動戰得分。開拓者也未能抓住機會，利拉德在終場前58.8秒命中一球，他們仍以94-101落後。然而，最後時刻，開拓沒能夠把握住關鍵的防守籃板，利拉德上籃不中，在最後6分39秒的時間裡，開拓者只投中了一個球，而鵜鶘在最後三分鐘也只是靠罰球得到了1分。最後兩隊的表現可以用菜鳥互啄來形容，不過，最終鵜鶘還是拿下了比賽的勝利。

在輸掉這場比賽之後，開拓者仍然排名西部第八，而鵜鶘以8勝23負排名西部倒數第二。
