黑色星期三！詹皇湖人創3大最差紀錄，宿敵快船卻解鎖3個全新成就

天天要聞
2019-12-19

北京時間12月18日，湖人和快船今日同時進行了比賽，結果卻截然不同，洛杉磯快船輕鬆地打爆了西部本賽季的黑馬太陽隊，而湖人隊則是非常不幸的被對手步行者隊終結了自己的7連勝。湖人和快船形成了截然相反的對比，而且快船還創造了很多全新的記錄，跟湖人正好也形成了鮮明的對比。

首先是快船方面。快船全場大勝太陽隊21分，而且整場比賽快船的三大巨頭表現都非常出色，得分同時突破20分。倫納德拿到了20分6個籃板2個助攻，路威拿下了20分2個籃板8個助攻，第4節88秒連得11分，而保羅喬治拿到了24分3個籃板1個助攻，但是卻有3個搶斷，重點是他們三個人的出場時間，沒有人超過28分鐘。

這樣一來他們也就創造了一個全新的記錄，本賽季至今為止，當快船的這三大巨頭同時拿到至少15分的時候，球隊目前7戰全勝一場都沒有輸過。快船創造了一個全新的勝率100%的記錄，同時洛杉磯快船已經正式拿到了主場的10連勝，已經追平了隊史主場方面的第3長的連勝記錄。最後一點本場比賽的快船球員麥格魯德還拿到了11個籃板球，創造了個人職業生涯的最高。一場比賽，他們完成了三個個人或者球隊的成就。

快船隊這邊是歡天喜地完成三大成就，而且收穫了一個賽季100%的贏球記錄，另外一邊呢？洛杉磯湖人反而是非常的糟糕，創造了很多很恥辱的記錄。首先湖人輸給步行者之後，已經遭遇了面對步行者的客場7連敗，上一次這支球隊能夠在步行者的主場贏球，還得追溯到2013年3月份的時候，距離現在已經過去了超過6年半！

從那時開始，湖人就從來沒有在步行者的主場戰勝過他們。上賽季詹姆斯也曾帶隊造訪過步行者，結果被42分暴打，可以說是非常的恥辱。除此之外，湖人的客場14連勝正式宣告終結，原本他們有機會挑戰NBA歷史的客場16連勝記錄，現在也沒戲了，而且這還是他們本賽季第1次在洛杉磯的主場之外輸球。

這份記錄也已經徹底終結了，兩項記錄全部終結。另外還有第3點，本賽季至今為止，湖人的最後三分鐘分差3分以內的比賽中，曾經打出過10連勝的成績，只要最後三分鐘進入關鍵時刻，湖人就沒輸過。然而這場比賽他們輸了，不敗金身徹底被破，湖人的三大記錄已經全部被終結，快船則是有三個新的里程碑，兩支球隊的待遇確實完全不一樣。
