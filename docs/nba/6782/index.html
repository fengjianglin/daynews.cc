可笑！裁判太照顧你們了吧，這麼明顯的違例都不吹，規則有什麼用

2019-11-25 0條評論 14次閱讀 0人點贊

北京時間11月25日，NBA常規賽繼續進行，快船主場戰勝鵜鶘迎來5連勝。

此役哈雷爾發揮出色登場28分鐘18投13中得到34分12板2冒的數據，在他的帶領下快船一眾替補球員正負值竟然達到了+135，而首發球員合計正負值卻是-19，這差距也太大了。

快船今天雖然取得了一場大勝，但本場比賽的一些判罰卻並不能令人滿意。尤其是第4節比賽還剩3分32秒的時候，倫納德帶球過半場，很明顯可以看到在計時器還剩15秒的時候他的雙腳都還在後場。

​正常情況下裁判應該吹罰8秒違例，但裁判竟然沒有任何表示，不能因為快船大比分領先就不吹了吧，這是哪門子道理，裁判也太照顧你們了吧。如果這麼明顯的失誤都不吹，那麼規則還有什麼用呢，可笑。

這已經不是第一次了，早些時候快船對陣雷霆，第4節關鍵時刻加里納利的一次突破明顯被哈克萊斯打到手了。裁判本來吹了犯規，但里弗斯挑戰後裁判竟然改判了，球迷都能明顯看出的打手犯規裁判為什麼看不明白呢，搞不懂。

對於一些模稜兩可的情況我們當然允許裁判有自己的看法，畢竟人不是機器，總會出錯。但是那些明顯的失誤裁判為什麼會看不到呢，難道他們的工作不就是做出正確判罰，讓比賽儘可能公平公正嗎。

為什麼NBA的收視率一直在下降，我相信裁判因素絕對是其中的一個原因，作為球迷，我們只想看到公平的比賽，這麼難嗎？
