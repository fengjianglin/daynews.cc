湖人全隊抗議裁判，詹姆斯聲稱習慣了，魔術師認為老詹將成MVP！

2019-11-25 0條評論 18次閱讀 0人點贊

詹姆斯本賽季逆生長的表現已經引起了廣泛的關注度，繼之前麥迪把詹姆斯列在MVP排行榜首位後，今天魔術師約翰遜也給出了自己心中的MVP候選人，詹姆斯第一、字母哥東契奇並列第二、3-10位分別為哈登、倫納德、濃眉、約基奇、恩比德、塔圖姆、西亞卡姆以及米切爾。

若詹姆斯持續高效發揮將湖人戰績穩居在聯盟前列，確實有可能時隔六年拿下MVP。昨日，湖人一分險勝灰熊的比賽，詹姆斯砍下30分6籃板4助攻2搶斷，帶隊豪取7連勝依舊坐穩聯盟頭把交椅。不過多少球迷在最後階段嗑起了速效救心丸？熟悉的節奏又來了，詹姆斯的關鍵三分不中、庫茲馬的黃油手讓對方打進、最後1.7秒的接球失誤，有驚無險的比賽多多少少充滿著戲劇性。

戲劇性的還有比賽的判罰尺度。詹姆斯一共出手27次命中14球，罰球次數卻為0。這是詹姆斯職業生涯第14次出現單場沒有罰球的情況，生涯首次得分30+卻沒能獲得罰球。你能看到詹姆斯在屢次衝擊禁區造成身體碰撞卻沒有罰球後，改變了進攻方式，增加了外線出手的次數。

詹姆斯在比賽中多次向場邊的裁判示意對方打到手了，手臂都有了刮痕，但裁判無動於衷。賽後，包括主教練沃格爾以及隊友們在內的多人都對老詹的遭遇表示了同情與憤慨。

沃格爾表示這令人沮喪，詹姆斯一直都在攻框，如果這樣都是0罰球，那確實有必要通過合適的渠道向聯盟管理層反應這個情況了。丹尼-格林表示詹姆斯的身體過於強壯，也不懂得表演，所以裁判很難看到他被犯規。毫無疑問，詹姆斯這樣的攻擊性，每一場比賽至少應該有8-10次罰球。

杜德利說得就更直白了，他表示詹姆斯真的不會像其他球員去騙你或者夾住你的胳膊製造犯規之類的，他只是想壓制你，但很多時候詹姆斯未能得到正確的判罰，這很艱難。

詹姆斯賽後在更衣室向記者們展示了自己的胳膊劃痕，有四五次擦傷是最近兩場比賽完全沒有吹罰過的。他說自己已經習慣了，裁判的判罰尺度不會阻止他繼續保持攻擊性。「我就是努力向籃筐施壓、向防守施壓、就是打好我的比賽。」

詹姆斯本賽季場均罰球5.6次，排在聯盟第24位，第一是哈登的場均14.2次。裁判對詹姆斯的判罰尺度一直以來都是討論的話題，詹姆斯生涯被惡犯的集錦可能比某些球員的高光集錦還要長。不過老詹自己看開了就行，球隊現在7連勝，14勝2負的開局是詹姆斯生涯單賽季前16場比賽的最佳戰績。

詹姆斯表示賽季還很長，球隊渴望進步，對於理想狀態而言，大家還仍未滿足。不過對球隊現狀很滿意，大家感到極其興奮。球隊昨晚搭乘飛機離開孟菲斯時，詹姆斯喜笑顏開，心情大好。

湖人已經抵達了聖安東尼奧，備戰明日與馬刺一戰。今天詹姆斯活躍在社交平台，他曬出韋德夫婦現場觀戰布朗尼高中籃球賽並為其歡呼的照片並寫道：「當我在客場征戰，我的兄弟姐妹為我堅守陣地！你們不知道這對我有多大的意義！這張照片非常不真實，很瘋狂，但真切詮釋了為何我一直都想告訴你們，我們的關係是超越籃球的，這完全是家族事務。」

隨後韋德轉發這張照片並寫道：「勒布朗，我們知道你現在很忙，所以我們會讓你安心！」

不愧是一家人，這樣的兄弟情真心讓人羨慕。從父代傳到下一代，一直在延續。一生能有如此知己，足矣。
