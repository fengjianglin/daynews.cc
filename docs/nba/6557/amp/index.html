兩次避戰綠軍，歐文該向詹皇學習！場均33+10，真是老東家的噩夢

天天要聞
3 週 ago

在NBA聯盟里，球星換隊一直都是很平常的事情，但像歐文一樣換隊之後卻選擇避戰老東家就很少見了，最近根據著名記者Shams的透露，歐文因為肩傷不會出戰籃網主場對陣凱爾特人的比賽，如果算上生涯歷史記錄，他6次面對老東家時只出戰了1場比賽。

​而作為歐文的隊友，詹姆斯的表現就值得歐文借鑒了，因為他面對老東家時可是毫不手軟，他面對的壓力甚至比歐文要高多幾倍！先說加盟熱火隊面對騎士的首戰吧，當時克利夫蘭幾乎所有人都響起了對詹姆斯的噓聲，甚至連波什也直言自己的生命安全都受到了威脅。

​詹姆斯的表現呢？全場砍下38分5籃板8助攻1搶斷1蓋帽，以118-90送騎士失利，把全場打得鴉雀無聲！再來看回歸騎士後對陣熱火隊的比賽，同樣的詹姆斯也毫不手軟，全場拿下30分4籃板8助攻，不過這場比賽顯然韋德還是笑到了最後，後者拿下31分5籃板5助攻2搶斷1蓋帽，成功搶戲詹皇。

​最後在加盟湖人之後，詹姆斯也再次迎來了對陣騎士隊的比賽，不過這次球館裡響起的只有掌聲，畢竟當時34歲的詹姆斯已經給克利夫蘭拿下了隊史第一冠，這樣的表現怎麼還會有球迷黑他呢？

​不過令人又愛又恨的是，這位老漢依然沒「放過」騎士，全場砍下32分14籃板7助攻1蓋帽，帶領湖人以109-105戰勝騎士隊，在三場面對老東家的比賽里，詹姆斯場均砍下33.3分7籃板10助攻，表現可以說是相當不手軟了。

​所以歐文在連續避戰的情況下，是否要像詹姆斯學習一下，如果做不到讓所有人都喜歡你，那麼你踏上球場拿出表現用勝利堵著黑子的嘴是最好的做法，而在社媒上發表長文只會讓球迷、媒體進一步的看不起你，你們認為呢？
