湖人雙榜眼發威難求一勝！鵜鶘這球隊要成為新星墳墓？

天天要聞
2019-12-06

鵜鶘又輸球了，在剛剛結束的一場NBA常規賽中，鵜鶘敗給了太陽隊，此役鵜鶘超星英格拉姆在眉骨被打開之後，上演王者歸來，全場拼下了21分6個籃板4次助攻，僅僅出手十次並在最後6犯離場，另一名前湖人榜眼郎佐-鮑爾此役同樣發揮出色，全場貢獻了20分6個籃板11次助攻，雙榜眼發揮難求一勝，鵜鶘這支球隊要成為新星墳墓，為啥這麼說？

真領袖兩罰不中+關係戶怒輸10分！英格拉姆想贏球得問隊友同不同意！

自從金特里明確表示英格拉姆不是球隊領袖之後，鵜鶘的真領袖就開始發威了，咱們平心而論，霍樂迪今天打的不錯，除了關鍵時刻自己抹了自己的脖子之外，沒啥大錯誤，但問題就是這個，主教練奉你為球隊的領袖，你得給出個表現啊，兩罰不中，被包夾之後的失誤，這能行啊？

一個最直觀的數據，英格拉姆在加時賽，直到他罰下之前，就獲得了一次出手權，那個回合，英格拉姆達成了2+1追平了比分，隨後霍樂迪和雷迪克的神操作成功的讓英格拉姆的努力成為了無用功，這不是一個領袖級球員應該做的事兒。

再看關係戶，亞歷山大-沃克，我沒有查資料，不知道他和金特里之間是不是親屬的關係，或者說是金特里年輕時候犯的錯誤，暫時就把他定為關係戶，沃克這小伙兒不錯，本賽季場均出手7.2次，能得到6.5分，這還算好的，畢竟你看他數據就知道他是一個深坑。

來看另一個關係戶梅利，這大哥場均出手4.8次，得6分，他就比較厲害了，你只看他數據你不知道他坑成什麼樣兒，舉個例子，他是一個移動更慢、防守能力更差、射術更不準，全面弱化了兩個等級的、現在的安德森，注意是現在的，安德森巔峰時期都比梅利不知道好了幾個檔次，梅利今天上場，鵜鶘輸了10分，沒有什麼準的籃兒、也沒防守。

英格拉姆想要贏球，不先問問這四個大哥，是絕對不行的。

湖人雙榜眼發威難求一勝！鵜鶘這球隊要成為新星墳墓？

為啥要這麼說，今天的鮑爾是超水平發揮了，砍下了20+6+11+3+2，攻防全能還有組織，英格拉姆出手10次拿了21分，雙榜眼是這個效率，鵜鶘還輸了，這種輸球的環境是不利於新星發展的。

輸球不可怕，也不存在缺不缺誰的問題，現在鵜鶘的問題是啥？英格拉姆、哈特這些球員拼了老命，一次又一次摔在地板上，將比分給追近，將比賽給打回到同一起跑線，然後呢？金特里會果斷叫一個暫停打斷他們的士氣，然後在暫停之後，霍樂迪和雷迪克會擔當起球隊的所有進攻，霍樂迪還好，只是進了三分線就勇往直前不再回頭，但雷迪克直接就科比附體了，漂移高難度。

不到兩分鐘的時間，這兄弟兩人就能將分差重新拉到十分以上，然後繼續養生，交給英格拉姆、哈特這些球員去一個接一個的往籃下沖，然後如此反覆，這不是黑，也不是說看不上金特里，這就是鵜鶘現在的情況，咱們有一說一，英格拉姆今年場均25+7+4，三項命中率49.2%+42.3%+83.8%，誰要是說這個數據，這個效率是在傷害球隊，那心得多黑，事實就是，鵜鶘全隊現在只有英格拉姆這一個高效率的得分點，他享受的是頭號球星的防守待遇。

那麼，如此反覆，久而久之，這些年輕球員會怎麼想？英格拉姆是沒問題，每場的關鍵時刻明顯就是不給他球，他還傻傻的跑位、策應、掩護呢，那除了英格拉姆之外的年輕球員會怎麼想，他們會覺得，好好打球沒什麼用，反正球隊不是為了贏球。

久而久之，球隊的年輕人會失去這個贏球的拼勁兒，這不是危言聳聽，擺爛球隊是怎麼擺爛的？NBA歷史上沒有任何一支球隊是打壓在場上的球員而擺爛的，他可能會把陣容不合理化、絕對球星交易走或者輪休，但只要球員在場上拼出機會，他們是絕對不會打壓球員的，鵜鶘的擺爛，是完完全全不合理的擺爛。

鵜鶘這支球隊，按照我個人的理解，不至於次成這樣兒，金特里要是這麼玩擺爛，也別英格拉姆、錫安-威廉森，前車之鑒就是戴維斯不願意為你打球，以後呢？未來呢？你都沒有，及時醒悟吧，打壓球員的表現是最大的錯誤。
