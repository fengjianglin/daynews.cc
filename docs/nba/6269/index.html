科爾又玩新花樣了，改造新核，擺怪陣，創新低，贏球排第4

2019-12-03 0條評論 18次閱讀 0人點贊

一場東西部「第一」之間的較量，最終是以25分的分差落寞，老鷹結束了10連敗，勇士吞下3連敗，這真是一場名副其實的菜雞互啄，賽前老鷹戰績是4勝16負，和尼克斯並列排在東部倒數第一，勇士就更不用說了，4勝17負，穩定佔據著西部倒數第一，最後還是勇士「更勝一籌」，拿下這場關鍵失利。

比賽輸贏不重要，倒是有一點值得一提，勇士哪怕不是強隊了，淪落到了如今的地步，但也不妨礙他們繼續創造紀錄，本場他們僅得到79分，創造了本賽季的最低得分紀錄，成功在聯盟各項紀錄統計上佔有一席之地。

上一場驚險輸給魔術後，也不知道是不是因為對上弱旅老鷹，生怕出點什麼意外，科爾讓球隊領袖格林輪休了，據說是腳後跟有傷，發炎了，從上一場對魔術的比賽中，反正是看不出來的，好吧，既然有請假條，那聯盟也無法阻止人家輪休，在證據齊全的情況下，我們也不好說勇士是不是為了賽季戰略而做的調整。

不過呢，今天勇士缺少了格林，卻迎來了盧尼的復出，自從揭幕戰打快船登場了10分鐘後，盧尼一直傷停到今天，這才僅僅是盧尼本賽季的第二場比賽，看來傷得是真不輕啊，回來是回來了，科爾卻給他設置了時間限制，最多給15分鐘，最終是打了14分鐘，科爾這方面倒也是說到做到，人看上去了壯實了不少，只是久疏戰陣，只貢獻了2分2籃板2蓋帽，表現一般。

這場比賽從科爾的排兵布陣看，他就沒打算贏，盧尼一回來就排進了首發，而且是一個極其怪異的陣容，三個內線考利斯坦、盧尼、帕斯卡爾搭檔羅賓遜以及伯克斯，把鮑曼貶為替補，這是一個沒有控衛的陣容，由伯克斯去擔任，且看得出科爾有意把帕斯卡爾往三號位推，這應該是他有意把帕斯卡爾培養成三號位球員吧，整場比賽下來，帕斯卡爾也確實大部分時間都在打三號位。

比賽是輸了，但科爾的目的達到了，他對新核心帕斯卡爾打法的改造也確實收到了效果，小夥子沒有辜負科爾對他的培養，全場出戰35分鐘，高效拿下24分+9籃板+6助攻，很全面，投籃命中率也挺高，11投9中，三分球2投2中，罰球5中4 ，勇士場上的其他球員感覺跟他就不是一個檔次的。

下賽季核心球員全部回來後，其他人還能不能進入輪換不好說，但帕斯卡爾是一定可以得到輪換機會的，他在大學打的是四號位，現在科爾開始對他進行改造，把他培養成三四號位都能勝任的球員，因為庫里、克萊、格林都回來後，球隊缺少的正是三號位球員，帕斯卡爾繼續能有這樣表現的話，那勇士下賽季的陣容可就完備了。

如果看勇士的比賽還一味地盯著輸贏關係的話，那建議少看，勇士現在是拿高順位簽排第一，培養年輕球員排第二，讓有傷的核心球員得到充分休息排第三，第四才是贏球。
