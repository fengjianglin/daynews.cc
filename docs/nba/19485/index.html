格林，想念杜蘭特么？7分3助，上空籃晃暈隊友，又是一場慘敗

2019-12-17 90條評論 2,436次閱讀 1164人點贊

北京時間12月16日中午結束的一場NBA常規賽，勇士主場迎戰國王，此前勇士的戰績為5勝22負，國王的戰績為11勝14負。這場比賽依舊不存在什麼懸念 ，79-100，勇士遭遇四連敗，國王也沒怎麼發力，第一節和第三節將比分拉開後，勇士難挽敗局。今天在比賽中出現的一幕，可能會讓很多勇士球迷感到心酸，這個放後面聊。

數據方面，勇士這邊，羅賓遜7分，格林7分，斯坦14分5籃板，達米安李13分5籃板5助攻，拉塞爾8分8助攻。國王這邊，博格達諾維奇25分5助攻，希爾德19分4籃板4助攻，巴恩斯15分，巴格利17分6籃板。

第一節上來，勇士舊將巴恩斯連拿4分，隨後斯坦和格林連續回應，雙方交替領先。但節末勇士命中率下滑嚴重，而國王依靠博格丹+巴恩斯+巴格利的組合，打出一波13-2的高潮，拉開分差！第二節勇士在伯克斯和拉塞爾的帶領下打出一波12-0高潮，隨後阿里扎連拿5分，巴恩斯和希爾德在半場結束前讓雙方分差重新拉開！半場勇士42-51國王。

進入第三節，國王外線發飆，希爾德、巴恩斯、約瑟夫連續命中三分，雖然勇士依靠克里斯、斯佩爾曼、埃文斯能夠得分，但是分差越來越大，三節結束時，國王已經領先勇士19分。進入第四節，博格丹兩記三分殺死比賽，最終勇士主場21分慘敗！

格林，今天出戰26分鐘，7中3，拿到了7分2籃板3助攻1搶斷，正負值-6。實話實說，他的防守還是有作用的，但是有效時間裡他只拿到4分，垃圾時間命中的一記三分讓球隊沒有太多改變，但相對於替補來說，格林至少今天不背鍋。

在第二節還剩下3分多鐘的時候，格林完全搶斷，國王球員反應不及，格林加速衝到籃下後，面對空籃，高打板！結果這球連籃筐都沒碰到，直接彈回來，像極了《灌籃高手》里的開場中，湘北隊內一個挨著一個的打板訓練！

關鍵是補防回來的羅賓遜也被他晃到，並沒有起跳跟進，而且身前還有兩名對手球員。這球如果說是想打板上籃，那偏的也太厲害了，明顯出手時手腕用力過猛！但如果說是想打板空接找身後的羅賓遜，這也很難為他，畢竟他不是杜蘭特，不是麥基！羅賓遜可做不到隔著兩個國王球員把球硬砸進籃筐裡面！緊跟著，勇士又被對手打了一個追身反擊！

帕斯卡爾繼續因臀部傷勢缺戰勇士的比賽，現階段勇士的進攻端終結點似乎只剩下拉塞爾、達米安李、伯克斯，而且關鍵是拉塞爾狀態很一般，格林也幫不上什麼忙，這支勇士真是有點可憐！此前科爾還曾安慰自己：「我們今年就是為了培養年輕人，早在庫里受傷前就制訂了這個策略。」但不管是媒體也好、球迷也罷，對於這支勇士的期待其實已經沒有那麼高了：別輸得太慘！

也許格林在那一次上空籃時，想起了杜蘭特，即便自己的出手不著邊際，身後的杜蘭特或者其他人總能夠用個人能力彌補格林拙劣的投籃，但現在格林身邊只有羅賓遜、斯佩爾曼、埃文斯、克里斯這些年輕人，他們可做不到隨時替你填坑啊！

別太想念杜蘭特，現在杜蘭特在籃網很愜意，每天和歐文一起，鼓鼓掌，激勵一下隊友，又是一場勝利！
