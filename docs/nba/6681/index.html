替補得分哪家強？快船排第1，湖人排第20，火箭替補得分已不能看

2019-11-27 0條評論 18次閱讀 0人點贊

新賽季已經打了1個多月，目前東西部排在第一的分別是雄鹿、湖人，而三大主要數據排第一的分別是哈登（場均得到37.9分）、德拉蒙德（場均16.8籃板）、詹皇（場均11助攻）。但是現在的問題是，截至目前，新賽季哪支球隊的替補得分最強？經查，快船的替補以場均51.12分，排名聯盟第一。

快船本賽季場均可以得到114.2分，排名聯盟第6。雖然快船的場均總分並不低，但是就全聯盟替補得分佔比來看，他們是最高的。快船替補很強，主要在於哈雷爾、路威這兩大超級替補。本賽季路威繼續高光表現，場均可以得到22.5分3.6籃板5.8助攻，表現比兩大巨星來之前還好；哈雷爾則繼續成長，本賽季場均可以砍下19分7.5籃板2.6助攻。

值得一提的是，快船也是僅有的替補場均得分超過50分的，而同城死敵湖人的替補場均僅34.65分，排在聯盟第20位。都說湖人本賽季替補陣容很有深度，也有霍華德、庫茲馬、波普、奎因-庫克等一眾實力悍將，但是與快船相比，差距巨大。如果本賽季達到最後湖人與快船爭奪西部的總決賽名額，這點差別或是致命的。

其餘球隊替補得分，奇才46.86排第二，活塞43.65排第三，獨行俠43.63排第四，熱火43.5排第五，馬刺42.44排第六，公牛41.44排第七，鵜鶘41.24排第八，黃蜂40.22排第九，灰熊40.13排第十。至於火箭隊，他們的替補場均得分僅有23.59分，排在聯盟倒數第一，已到了不能看的地步。現在大家也可以理解為什麼哈登要狂砍得分了，因為替補實在進攻能力太差！
