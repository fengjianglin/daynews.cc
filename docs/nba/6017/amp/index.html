快船不行咱可以！湖人3連勝後戴維斯霸氣發聲，雄鹿這下危險了？

天天要聞
2019-12-07

湖人隊再一次在客場實現了連勝！

隨著湖人隊剛剛結束本賽季的第23場常規賽之後，湖人隊拿下了其中的20場勝利，而在這20場勝利當中，湖人隊主場贏了9場比賽，客場贏了11場比賽其中客場11連勝，對於如今的湖人隊來說，或許更加擅長打客場，畢竟湖人隊此前在主場接連輸給了猛龍隊和獨行俠隊！

而湖人隊更擅長打客場，或許也更加有利於12月份的賽程：在12月份賽程中，湖人隊還剩下6個客場，其中還包括1個5連客。而在剛剛，率領湖人隊拿下3連勝的戴維斯在接受採訪的時候，再度霸氣發聲：

「我們感覺我們有能力在任何客場贏球，我們是一支在客場表現出色的球隊，當你在客場作戰的時候，你就需要更加團結一致」–戴維斯表示到。

對於其他的球隊來說，或許更加希望能夠在主場打比賽，畢竟熟悉的球館、熟悉的籃筐以及熟悉的支持自己的球迷，但對於湖人隊來說，連續征戰客場，或許還是能夠有利於這支全新的湖人隊的–有利於這群本賽季剛剛組合在一起的球員彼此之間能夠更好的是磨合、去熟悉、去增強彼此之間的感情！

而對於湖人隊來說，在回到主場打森林狼隊之後，又將要開始5個客場之旅了，值得注意的是，湖人隊在此番5連客之旅中，將會打雄鹿隊–那個在今天大勝快船隊的雄鹿隊，而對於戴維斯的這番話，不知道雄鹿隊有沒有聽到？

在快船隊不敵雄鹿隊之後，相信很多的球迷非常期待湖人隊打雄鹿隊，畢竟湖人隊和快船隊都是西部最強的球隊，而如今隨著快船隊客場慘敗給雄鹿隊之後，球迷自然會想看湖人隊打雄鹿隊的比賽了。而在戴維斯霸氣發聲之後，或許要對詹姆斯說聲恭喜了：

「雄鹿隊最大的優勢是在內線，畢竟有字母哥這個本賽季MVP大熱門，但別忘了，湖人隊的內線優勢同樣巨大，進攻端有戴維斯，防守端有戴維斯+麥基+霍華德，即便是字母哥，想要在湖人隊的這3個內線球員前面翻江倒海，難度還是非常大的，而且，湖人隊還有一個雄鹿隊無法匹配的詹姆斯」

從理想情況來看，湖人隊打雄鹿隊這下穩了。但在比賽還沒有開打之前，誰也不知道比賽的勝負結果，而且湖人隊打雄鹿隊也會是5連客的最後1個客場之旅，如果湖人隊想要保留更多的體力去打雄鹿隊，或許又要多打花幾個對手了！
