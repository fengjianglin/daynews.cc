叕輪休，精緻的利己主義者，離聯盟第一人位置漸行漸遠

2019-12-11 455條評論 1,618次閱讀 482人點贊

北京時間12月10日，快船客場挑戰步行者，萊昂納德又輪休了，好在快船在喬治的帶領下力克步行者，戰績位列西部第二。本賽季的萊昂納德基本上不打背靠背的比賽，在已經過去的25場比賽中，出場13次，勉強夠一半。平均出場時間也只有31min。

遙想上個賽季季後賽，猛龍在萊昂納德的帶領上擊敗勇士，歷史性的取得總冠軍，順利奪得FMVP，在詹皇逐漸老去，新皇還未登基的當下，群雄並起，擁有總冠軍加成的萊奧納德隱隱然有異軍突起，邁向聯盟第一人寶座的趨勢。加上本賽季攜手喬治加盟本就是季後賽球隊的快船，球迷無不認為，一艘巨船即將起航，萊昂納德的職業生涯榮譽將會更加輝煌，然而，本賽季的萊昂納德一直輪休，引起了球迷的口誅筆伐。

網路圖片

球員保護自己的身體確實無可厚非，一味的輪休確實傷了球迷的心，縱然有人會說，打球是一份職業，只要老闆同意就可以，可是無論從事任何職業，對職業的應該有的敬畏之心還是不能丟的。喬丹、科比、詹姆斯都是出了名的拚命三郎，基本上沒有非傷病的輪休，兢兢業業充滿了對籃球的熱愛。球場上的表現和職業精神是其通往更高歷史地位的唯二選擇。

網路圖片

猛龍的總冠軍，萊昂納德確實是絕對的總冠軍。可是也不要忘記了全明星控衛洛瑞，內線大閘加索爾、伊巴卡，飛天遁地的西亞卡姆，超神歸來的范喬丹。團隊的勝利團隊的貢獻是最大的。更何況總決賽還是面對已成殘陣的勇士隊。天時地利人和缺一不可。

網路圖片

想要更大的成功，猛龍的一冠說服力不夠強。難以比肩11年諾維茨基的一冠。萊昂納德可能真的是不在乎數據，不在乎職業生涯的高度。巔峰時期，在關鍵的比賽中出出力，延長職業生涯，賺更多的錢，做一個精緻的利己主義者，何樂不為。不過，這樣確實是聯盟第一人寶座漸行漸遠了。存在即是合理，路怎麼選擇怎麼走就是了。
