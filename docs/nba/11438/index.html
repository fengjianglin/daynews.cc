一戰十記錄，熱火打瘋了！卡特里程碑變“里程悲”

2019-12-11 880條評論 3,757次閱讀 1183人點贊

本場比賽，熱火坐鎮主場迎戰亞特蘭大老鷹隊。兩隊實力還是有一定的差距，熱火目前以18勝6負的戰績排在東部第三，而老鷹則是以6勝18負的戰績排在東部13。本場比賽熱火真的是打瘋了，首發5人有四人得分上雙，其中三人30+，兩人大號三雙。納恩36分6籃板4助攻，羅賓遜34分3籃板4助攻，阿德巴約30分11籃板11助攻，巴特勒20分18籃板10助攻，首發五人一共砍下124人，替補僅兩人得分，一共11分。

這一戰，熱火隊也是直接刷新了十項記錄。首先是幾項個人記錄，納恩砍36分刷新個人生涯單場得分新高，羅賓遜拿下34分也創造了自己個人生涯得分新高，這兩人都是今年的落選秀，你敢相信兩個落選秀合砍70分嗎？這劇本真的太浮誇了，簡直令人難以置信。另外阿德巴約拿下30分11個助攻刷新了自己生涯單場得分和助攻的新高，本場阿德巴約也是拿到了自己生涯的第一個三雙。

本場比賽巴特勒化身籃板痴漢，拿下了18個籃板，也刷新了自己的個人單場籃板數新高。羅賓遜本場的10記三分球也是刷新了個人生涯單場三分數新高，另外他的4記助攻也創造了自己個人單場助攻新高。看完這些個人記錄後，再來看看本場比賽創造的隊史記錄，巴特勒與阿德巴約兩人同時拿到三雙的數據，這是球隊歷史上第一次達成此項成就，還有一項首發得到124分創球隊首發得分最高。

一戰十記錄，註定這個夜晚是屬於熱火的。然而，在熱火創造紀錄的同時，老鷹的卡特也迎來了自己生涯的一個新的里程碑，這一戰是卡特出戰的第1500場常規賽，也是歷史上第五位常規賽突破1500場大關的球員。卡特今年是他征戰NBA的第22個賽季，目前也是征戰NBA時間最長的球員，現在他每過一天都是在刷新一項紀錄呀，這位本人半神不知道還要打到什麼時候才退役去了呀。

本來卡特高高興興的迎來生涯的里程碑，奈何熱火硬是強行將卡特的里程碑變為「里程悲」，加時一波18比4強勢收割比賽。熱火全隊太不給老卡特面子了，畢竟卡特都是一把年紀的人了。哈哈哈，說歸說，鬧歸鬧，熱火本場比賽打得真的是太精彩了，就連熱火三巨頭時期沒有達到的成就，今天都被他們給達成了。對於熱火本場比賽的表現，各位有什麼看法？
