球弟的未來不是夢，拉梅洛-鮑爾正在向著狀元秀的目標打造自己

天天要聞
2019-12-04

如果你作為一個籃球迷，沒有看過一分鐘本賽季的澳大利亞NBL比賽，你肯定不是一個人。但是如果你喜歡的NBA球隊（例如，勇士隊）正在聯盟底部苦苦掙扎，那麼最近的一些發展就值得你注意了。儘管18歲的拉梅洛-鮑爾作為球弟的名氣可能比他本人更大，但在這裡，值得強調的是蘊藏在它背後的真材實料。隨著他在效力伊拉瓦拉老鷹隊的最近兩場比賽中拿到三雙，鮑爾正在增強他成為2020年選秀大會狀元秀的潛力。並且他不是通過球爹的大肆吹捧來推動他的預期，而是他的出色表現。

拉梅洛-鮑爾在六月份成為第一個被摘走的球員，這樣的場景並不完全在意料之外。籃球界自今年夏天以來就一直在關注他，球隊的決策者們已經紛紛跨過太平洋以一探究竟。作為一名在助推進攻方面有著非同尋常的內在天賦的高個子控衛（2米01），鮑爾在球員各方面的成長曲線都領先於他的年齡已不是什麼秘密。球隊公認他是一位樂透秀的前景，但是還很少有人確切認識到他已經能做出怎樣的貢獻。

在NBA圈子中，關於頭號簽人選的共識還沒有怎麼形成。安東尼-愛德華茲在毛伊邀請賽上的出色表現無疑增強了這位喬治亞後衛的口碑。而孟菲斯中鋒詹姆斯-懷斯曼將被禁賽到明年一月，很多球隊期待著近距離觀察他。兩人都很優秀，但對於狀元秀的整體感覺仍然不溫不火。正如東部的一位總經理所說：「那兒有什麼，我們就只能選什麼。」隨著時間進入到12月份，情況確實在有所發展。相比上賽季，那個時候錫安-威廉姆森大約用了一個星期就讓所有人的選擇變得很簡單。

考慮到所有這些，也就更難以忽視鮑爾正在做的事情。值得注意的是伊拉瓦拉本賽季只有3勝6負，排在聯盟墊底的位置，所有三場勝利都是來自一個對手凱恩斯。上周鮑爾正是在凱恩斯身上得到了32分，13次助攻和11個籃板的三雙。然而，他仍然是整個NBL中產量最高的球員之一：在11月的五場比賽中，鮑爾場均得到23.2分，8.4次助攻，9.2個籃板和1.8次搶斷。

他肩負著伊拉瓦拉沉重的進攻回合，在前NBA後衛阿隆-布魯克斯受傷缺陣後更是如此。事實上他的努力在短時間內也帶來了肉眼可見的進步。他甚至在防守端也取得了長足的進步，看起來他至少有機會勝任指派給他的防守任務。

拉梅洛缺乏精英級的運動能力，但是他這麼年輕就如此出色表明這可能不是一個問題。我們正在見證一位幾個月前剛滿18歲的年輕人在充滿競爭的職業賽場上的真實表現，並且對他在一個近年來被精英級外線投籃創造者重新定義的快節奏的NBA中的前景感到樂觀。

雖然很明顯鮑爾會在一個相對的選秀小年被很早選中，但是有很多人質疑將他與第一順位綁定在一起。「一個狀元秀投籃命中率不足40%，三分球命中率不足30%？太瘋狂了吧。」西部的一位總經理強調道。

截止周二，鮑爾的投籃命中率為38.9%，而三分球命中率只有27.9%，此外場均不到四次罰球，命中率70%。在上周六面對凱恩斯以及另一位潛在的樂透秀RJ-漢普頓拿到三雙的那場比賽中，他三分球11投1中。看鮑爾打比賽，你可以看到他的進步，但是賽季結束時更多的統計數據將更有說服力。

樂觀地說，鮑爾作為一名球員的自然成熟會帶來效率上的逐漸提升，並且更多的天賦也會導致更容易的得分機會。他仍在設法找到得分導向和自我成就之間的微妙平衡。但是很明顯他在進攻組織方面的感覺將轉化為潛在的精英時尚。為了最大化他的傳球天賦，他也必須讓自己成為一個持久的得分威脅。他的大哥朗佐-鮑爾在適應NBA的過程中正在遭受這一轉變的困擾。

拉梅洛有能力傳出每一個球，但是只有伴隨著跳投和終結能力的提高，結果才會變得不同尋常。他更高大，投籃動作也更乾淨，在相同的舞台上突破和創造空間的效率會比朗佐高效的多。因此，當他的投籃變得更出色，並且統計樣本也支持這一說法時，拉梅洛的超常的控球能力將會發揮更加美妙的作用。如果他既是一個出色的傳球手，又能隨心所欲地得分，那就再好不過了。但如果他的投籃命中率得不到改善，那麼他將很難融入某些球隊的體系之中。

對於一些球隊管理層來說，另一個讓他們猶豫不決的方面可能是球爹拉瓦爾-鮑爾的嘴炮功夫。朗佐-鮑爾早期的職業生涯在一定程度上就是被球爹的口無遮攔傷害了。但在某些時候，這種噪音也可以被無視。

就像小勒布朗-詹姆斯作為加州塞拉峽谷的高一新生會有15場比賽通過網路直播。德懷恩-韋德的兒子扎伊爾也在為該隊效力，潛在的樂透秀布蘭登-波士頓和齊亞爾-威廉姆斯也在隊中。成名要趁早是新一代球員的新常態，對於球隊來說，只需要適應它即可。往好處想，拉梅洛在他爹無所顧忌的口出狂言幾年後正在開始重塑自己的形象。如果一支球隊真心認為他配得上狀元秀，那麼他們無論如何都會摘走他。
