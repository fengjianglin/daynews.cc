恩比德不懼萊昂納德，沒了小加索爾助力，76人能戰勝快船

2019-12-01 0條評論 18次閱讀 0人點贊

歡迎來到《雨說體育》，今天費城76人隊戰勝了印第安納步行者隊，全場下來喬爾恩比德砍下32分11籃板4助攻，無愧為球隊取勝的第一功臣。雖然當下76人的戰績在東部看來不算很突出，但考慮到76人在休賽期經歷了較大的調整，此前的外線核心巴特勒出走熱火，球隊引進了內線球星艾爾霍福德，因此戰術體系的調整無法避免，戰績方面有些起伏也是在情理之中的。

放眼當今的整個聯盟，76人無疑是東部非常有力的總冠軍門票爭奪者，而在西部呢？快船也擁有同樣的資格，那麼如果說這兩支球隊在總決賽相遇的話，到底誰能夠佔據上風呢？對於這個話題，可能你咋一看會感覺快船能佔據上風，確實，快船的聲勢明顯更猛，兩大鋒線攜手，再配合上哈雷爾與路威，球隊的深度與高度都讓人羨慕，看起來他們是不可阻擋的。

反觀76人呢，恩比德與霍福德的組合有點兒逆當今潮流，我們知道自從金州勇士隊崛起以來，聯盟中就越來越強調速度與三分球了，攻守轉換的能力是一支球隊賴以生存的重要取分手段，在這種大背景下，內線雙塔還有出路么？當年的鵜鶘雙塔，現在的活塞雙塔，無一例外效果都不是很理想。按照這個角度去衡量，似乎76人也是非常危險的。

首先76人的東部晉級之路肯定會更加困難一些，但如果我們不考慮晉級過程，僅僅假設76人能闖進總決賽，並與快船隊交手的話，這兩支球隊誰強誰弱呢？如果是這樣的話，那我感覺76人是絕對有機會的，他們甚至可以佔據上風。為什麼這麼說？因為快船隊最薄弱的點就是內線了，祖巴茨與哈雷爾雖然不錯，但防守端能力都非常有限，祖巴茨腳步不夠靈活，無法跟防到三分線，哈雷爾雖然靈活，但高度太矮，恩比德若是低位背身，哈雷爾完全無法起到限制的作用。

而且不要忘了，76人隊不僅是有一個恩比德，他們還坐擁一個霍福德呢，這兩個能力全面，能低位能投射的全面中鋒，對手要如何應對呢？而內線一旦被打爆，球隊就很容易陷入到犯規麻煩當中，畢竟以小防大的話，不犯規是難以阻擋對手的。喬治與萊昂納德雖然都是防守悍將，但無奈他們都無法搖擺到中鋒位置，快船的防守最強點無法與對手形成有效對位。

其實在上賽季的時候，猛龍隊為什麼能夠與76人較量到最後一刻並最終取得了勝利？說實話，萊昂納德固然很強，但中流砥柱還得說是小加索爾，如果不是小加索爾這麼一個恩比德唯一懼怕的內線防守人在猛龍的話，那麼猛龍也是無法戰勝76人的，恩比德的低位威脅被小加索爾壓住太多了，這才是比賽真正的勝負手。

但如今呢，快船隊內可沒有小加索爾，單靠萊昂納德去限制恩比德是不現實的，因此如果交手的話，我更看好76人隊能夠取勝，他們只要闖進總決賽的話，還是有希望奪得總冠軍的。
