杜克大學，NBA球星生產基地！但真的沒有巨星，全部是謙謙君子

天天要聞
2019-12-09

杜克大學是全美最有名的籃球大學之一，每年都會有很多的球員通過選秀進入NBA。

2019年NBA選秀大會

在今年的選秀大會上，狀元蔡恩-威廉森就是杜克大學的，目前在鵜鶘還沒有上場打球。尼克斯在第3順位選擇的巴雷特也是杜克大學的，目前場均得到14.3分5.3籃板3.3助攻，但投籃命中率只有39.5%，三分命中率只有31%，罰球命中率只有53%，效率很低很低。老鷹在第10順位選擇的雷迪什，場均8.1分，投籃命中率只有慘淡的31.8%，三分命中率只有慘淡的25%，被譽為今年的第一水貨。

2018年NBA選秀大會

國王在第2順位選擇了馬文-巴格利，是一個不錯的球員，上賽季場均得到15分8籃板，新賽季打了一場比賽就受傷了。公牛在第7順位選擇了溫德爾-卡特，身高2米06，內線球員，新賽季場均得到12.3分10籃板。去年爵士隊在第21順位選擇了格雷森-阿倫，防守非常兇悍的那個，休賽期被交易到了灰熊，新賽季場均得到7.6分，命中率也是慘不忍睹，只有38%。

2017年NBA選秀大會上

凱爾特人在第2順位選擇了塔圖姆，塔圖姆有巨星的氣質，新賽季場均可以得到21分7籃板。去年活塞在第12順位選擇了後衛肯納德，新賽季場均16.4分3.5籃板4.3助攻，三分命中率高達41%，迎來了大爆發。其他的弗蘭克-傑克遜、賈爾斯都一般。

2016年NBA選秀大會

湖人在第2順位選擇了英格拉姆，這個賽季在鵜鶘兌現了天賦，場均24.6分7籃板4助攻，明年夏天能夠拿到一份大合同。現在在湖人打球的庫克是落選秀，他也是杜克大學的。

2015年NBA選秀大會

奧卡福在第3順位被76人選中，76人後來放棄了他，輾轉籃網、鵜鶘，現在上場時間也不確定。溫斯洛是在第10順位被熱火隊選中的，進聯盟防守能力就被肯定，和巴蒂爾有點像，巴蒂爾當年是老K教練最喜歡的學生。

現役還有

歐文、胡德、賈巴里-帕克（老鷹）、梅森-普拉姆利（掘金）、賽思-庫里、雷迪克等等都是來自杜克大學。

退役的

退役的就比較多，巴蒂爾、羅爾鄧、布澤爾、鄧利維、布蘭德、格蘭特-希爾等等。

我們能夠發現，杜克大學確實是球星生產基地！但真的沒有巨星，全部是謙謙君子。
