卡萊爾下了一盤八年的棋？東契奇傷缺波庫發威，一數據仍聯盟墊底

天天要聞
2019-12-24

獨行俠只要還是里克·卡萊爾執教，就一定還是一支西部不可無視的力量，因為正是他鑄造了10-11賽季的史詩級一冠（用一句通俗易懂的話說：「永遠不要低估一顆總冠軍的心」）！

​

10-11賽季卡萊爾執教的獨行俠（那時還叫小牛隊），是當時全聯盟平均年齡最大的球隊。不成想，他們不僅在常規賽打出了57勝25負西部第三的佳績，更在季後賽中，首輪擊敗了明星雲集的開拓者（阿德+傑拉德·華萊士+安德烈·米勒+羅伊+巴圖姆），次輪橫掃科比領銜的湖人隊，半決賽輕取三少率領的雷霆，總決賽上力壓熱火「三巨頭」（詹韋波）奪得總冠軍。

​

奪冠後的五年，老臣們退役的退役，離開的離開，退居二線的退居二線，總之，除了諾維斯基之外，都相繼離開了，更有甚者溜達了一圈又回來了（泰森·錢德勒）。在這五年里，諾維斯基儘管仍能率隊四次殺進季後賽，卻均是一輪游。隨著諾維斯基生涯尾聲的到來，獨行俠在最近的三個賽季，均被擋在了季後賽大門之外。

​

細心的球迷一定會發現獨行俠在10-11賽季奪冠之後，有一項數據統計成階梯式的增長，那就是球隊的失誤數。有數據顯示，獨行俠在奪冠的10-11賽季，場均失誤數是位列聯盟中游水準的，在此之後的五年間，奪冠的老臣們相繼離開，陣容中再沒有經驗豐富的老將了，又沒有潛力新星湧現出來，才下坡路一走到黑！

最近的三個賽季，諾維斯基職業生涯漸漸走到了盡頭，獨行俠的戰績也是一落千丈，無緣季後賽，當然，還有另外一個客觀原因存在，那就是最近的三個賽季，西部諸強變的更加的強了。

​

值得一提的是，即使獨行俠戰績在西部位列倒數，但是，場均失誤數卻始終保持在聯盟倒數第三~第一的位置上（16-17賽季，場均11.2次位列倒數第二；17-18賽季，場均11.8次於森林狼並列聯盟倒數第一）。

在休賽期，獨行俠用上億的大合同簽下了一年多沒有打過比賽的波爾津吉斯，又得到了上賽季季後賽有亮眼表現的「庫里哥哥」，當然，還有幾位角色球員，如小哈達威（尼克斯），萊特（灰熊），等等。

​

獨行俠新賽季的先發五虎，一位二年級的菜鳥（東契奇），一位藍領球員（鮑威爾），其餘三位均為新援，你會看好這樣一支「東拼西湊」的球隊成為西部豪強雲集的「黑馬」嗎？

當然，獨行俠賽季至今的表現是很好的證明，東契奇受傷之前，打出了巔峰詹姆斯的風采，率領球隊一度躋身西部前三之位，在受傷缺陣的五場比賽中，波爾津吉斯打出了昔日全明星的風采，場均可轟22.4分13.8板2.6個蓋帽的數據。

​

​

在波爾津吉斯的進攻統治力下，賽斯·庫里也好，小哈達威也罷，都在外線找到了很好的節奏，此外，還有一人不得不說，他就是布倫森，東契奇受傷之前，布倫森甚至淪落到了第二替補的地步，東契奇因傷缺戰，布倫森臨危受命出任首發，在過去的五場比賽中，他場均14.6分5板9.2次助攻的數據，助攻失誤比超過了3-1，真的是令人刮目相看。

​

一句俗話說得好，千軍易得一「帥」難求呀！卡萊爾無愧冠軍教頭之名，保持健康的獨行俠本賽季絕對稱得上是一匹「黑馬」。
