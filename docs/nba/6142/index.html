噩耗！北京首鋼前隊長吉喆因肺癌不幸離世，年僅33歲

2019-12-05 0條評論 17次閱讀 0人點贊

曾幾何時，五棵松籃球館總會響起「吉吉吉吉babybabybaby」的BGM，這是北京首鋼籃球隊前隊長吉喆進球後的專屬配樂。

去年這個聲音始終沒有在球館響起，因為吉喆因傷缺席了整個上賽季。可即使這樣，大家也都認為分別只是暫時的，總有一天三吉都會回來。

正因如此，當吉喆離世的消息在今天突然傳來的時候，我們錯愕、震驚、難以接受。

以後人們在五棵松球館再也聽不到那首屬於吉喆的BGM了。

1986年，吉喆生於遼寧瀋陽，2007年從遼寧男籃租借到北京男籃。2011-12賽季，吉喆幫助球隊在總決賽中以4-1戰勝廣東，奪得首鋼歷史上首個CBA總冠軍。

2012年，吉喆正式轉會首鋼。隨後的13-14、14-15賽季，首鋼兩奪總冠軍，創下了四年三冠的北京王朝，吉喆功不可沒，他也曾擔任過北京首鋼的隊長。

上賽季因為膝傷嚴重，吉喆賽季報銷，他還在微博上透露會前往美國治療。其實當時吉喆已經確診患有肺癌，只是他沒有選擇把這個消息向球迷公開。當時吉喆本人、他的家人，以及俱樂部上上下下都覺得困難只是暫時的，他一定能戰勝病魔重回賽場。

然而就在今天（2019年12月5日）凌晨2點14分，與肺癌頑強抗爭一年之後，吉喆還是不幸病逝於中日友好醫院，年僅33歲。

北京首鋼籃球俱樂部如此評價吉喆：2007年吉喆來到北京首鋼男籃，2米02的他個子不算太高，但基本功紮實，很快贏得了所有人的喜愛與信任，他用自己的身軀守護著北京籃球的禁區。無論是在球隊人員不整、戰績不佳的低谷時期，還是在球隊眾志成城、四年三冠的高光時刻，吉喆始終勇於擔當，用他的中投，他的三分，他的籃板，默默為球隊貢獻出他所有的能量。

吉喆離世的消息傳來之後，眾多CBA球員、名宿集體在微博上緬懷他。吉喆的老隊友馬布里稱他為：我的兄弟、我的隊友、我的戰友、我的隊長……

翟曉川也在微博上回憶了與吉喆的最後一面：

曾經國家隊里的隊友，聯賽上的對手——易建聯說：願天堂有你喜愛的籃球。

12月8日，北京主場迎戰深圳，就像焦健說的那樣：希望大家可以穿上吉喆的51號球衣，到球場上送他最後一程。
