詹皇傳球太秀了！名記高喊“Lemagic”，有誰注意到霍樂迪的表情

天天要聞
2019-12-19

隨著安東尼·戴維斯因傷缺陣，詹姆斯承擔起了對陣東部季後賽球隊步行者的對戰，而比賽一開打詹姆斯就展現出了與過往不同的狀態，由於身邊沒有戴維斯的存在，詹姆斯更多的將處理球的權利掌握在自己手上，而不斷的切入撕裂對手的防線，但是更加讓人印象深刻的則是詹姆斯在上半場的幾次精彩的傳球，這些傳球讓湖人內線的霍華德、麥基等人都輕鬆吃餅，而詹姆斯的一個腦後傳球給霍華德完成暴扣更是引起了熱烈的討論，著名記者黎雙富更是高喊「Lemagic」將詹姆斯與魔術師約翰遜作比，更加有趣的是在一旁防守的霍樂迪的表情更是瞬間上了熱門討論的榜單。

在上半場的一次突破中，詹姆斯接到卡魯索的傳球空切進入內線，而詹姆斯則是高高挑起之後一個腦後傳球傳給了站在身後的霍華德，接到球的霍華德則是輕鬆躍起完成了暴扣，詹姆斯的這記秀實力的傳球讓現場的解說不禁談起了湖人的名宿魔術師約翰遜，甚至談到了這樣的打法像極了曾經湖人的「showtime」時期，而著名的記者黎雙富更是用「Lemagic」來形容詹姆斯，很顯然這位35歲的老將仍舊在質疑聲中用自己殘存的實力去向球迷展示自己在球場上的創造力。

更加有趣的是在詹姆斯完成這記傳球之時，步行者隊的後衛阿隆霍樂迪就直挺挺的站在一旁，從慢鏡頭回放來看霍樂迪全程是張著嘴瞪大眼睛在看著詹姆斯完成這記傳球，直到霍華德完成了暴扣霍樂迪才緩過神來，很顯然這位年輕人被眼前的這位中年人嚇到了，都知道詹姆斯在過去很強，也都知道在如今有很多人看不上這個35歲的老將，但是沒有想到眼前出現的卻是像極了過去詹姆斯的那個詹姆斯，甚至還是有過之而無不及，難怪霍樂迪會如此驚訝了，只能說老漢仍舊在用自己的餘光展現自己，這讓人著實眼眶有些許的濕潤了。
