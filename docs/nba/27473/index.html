1.9億雙人組合砍7+7！喬丹神操作坑慘魚腩，垃圾合同扎堆太罕見

2019-12-23 3,521條評論 4,490次閱讀 894人點贊

北京時間12月22日，2019-20賽季NBA常規賽繼續進行，黃蜂主場不敵爵士。本場比賽球隊的1.2億先生巴圖姆復出，並賽季首次出任首發。最終他轟下了5分3板，另外一位高薪球員，合同總額達到7200萬（包含400萬激勵條款）的比永博，則是拿到了2分4板。

合同總計1.9億的的雙人組，總計拿到了7分7板，兩大垃圾合同這樣的表現，真的讓人大跌眼鏡！更為重要的是，正是因為黃蜂垃圾合同扎堆，不斷留下不該留的人，導致他們在休賽期續約時，無法拿出讓沃克滿意的價碼，只能白白送走球隊真正的核心。如今這支球隊，只能在樂透區掙扎，陪伴這些「垃圾合同」走到最後。

本賽季巴圖姆首戰受傷，隨即陷入了長時間的休戰，即便是登場也很難打出符合價值的表現，場均3.4分配上36%的投籃命中率以及29.4%的三分命中率，這樣的表現，實在是對不起自己的5年1.2億的合同。

更為扎心的是，當初他與管理層簽署這份合同的時候，20-21賽季還是球員選項。如今這個情況，他萬萬是不可能跳出合同的。因此下賽季黃蜂只能繼續忍受他一年，這位法國魔術師已然是毒瘤級別的存在了。

比永博則是在一個系列賽中發揮出色，得到了魔術的垂青，4年7200萬的合同到手之後，他兜兜轉轉來到了黃蜂，全然沒了昔日的風采，本賽季至今場均7.6分5.6個籃板，上賽季更是只有4.4分和4.6個籃板入賬。

兩個垃圾合同，一個出自喬丹之手，一個則是來自黃蜂的交易。實際上，不僅是這兩人，黃蜂陣中目前的馬文、邁基吉等人，因為傷病、年齡等原因，多少都有些垃圾合同的意思。

在垃圾合同扎堆的情況下，黃蜂本賽季的薪資達到了1.242億，排在聯盟中游，實力和薪資完全不匹配，還被迫放走了沃克，這支球隊以及管理層，當真是NBA最奇葩的存在。
