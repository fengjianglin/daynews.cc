狀元榜眼簽選中這10位水貨，他們靠什麼騙過各大球探的眼睛？

2019-12-23 133條評論 1,051次閱讀 143人點贊

選秀大會對於NBA球隊來說就像是一場賭博，賭對了最少可以獲得一名優質的輪換球員，而賭錯了，則是人財兩失，雖然NBA各支球隊為了避免賭輸了的情況，建立了非常完善的球探系統，但這也避免不了水貨球員的存在，那這些水貨球員憑什麼騙過球探和各大選秀網站的眼睛，成為高順位甚至是狀元，原因很簡單，那是因為這些水貨球員，本身就具備著出色的天賦，下面小編就給大家證明一下，10大水貨新秀的天賦究竟有多出色。

1、誇梅布朗

為什麼誇梅布朗能夠成為NBA歷史上第一位高中生狀元？來看看高中時期誇梅布朗的數據，20.1分13.3籃板5.8蓋帽，完全是統治級別的數據，並且在2001年當選為全美最佳球員，當時的誇梅布朗身高2米11，運動能力出色，已經具備了NBA級別的對抗，不滿19歲的年齡，各項體測數據和魔獸霍華德相比都不算遜色，在當時還是得內線者得天下的時代，這樣的天賦怎麼能不讓聯盟各支隊眼饞，只不過他的心智卻還不成熟，再加上巨大的壓力，才導致最終淪為水貨。

2、米利西奇

當年的米利西奇為何能力壓大學無敵的安東尼成為榜眼秀？看完下面兩點大家也許就明白了，首先是老尼爾森的評價，作為一手培養出納什、諾維茨基等巨星的教練，老尼爾森在算的上是NBA中的一大伯樂，能讓老尼爾森作出「他的身材、體格、速度、靈巧性和力量，都讓人驚異，如果你想要個中鋒或者大前鋒，那麼達科絕對是不二人員」的評價，足見米利西奇的天賦有多出色，甚至老尼爾森還曾說過米利西奇的天賦還要比同年齡的諾維茨基更加出色，還有在06年世錦賽中，米利西奇場均16.1分9.8籃板2.8蓋帽，其中蓋帽第一，籃板第三，這兩項數據，比諾維茨基和姚明都還要出色，而且，這還是米利西奇在NBA度過了三年打不上比賽後交出的成績。

3、奧洛沃坎迪

98狀元，被稱為「大夢接班人」，17歲才接觸籃球的奧洛沃坎迪擁有著2米13的標準身高，而且在97-98賽季的NCAA中，才接觸籃球不久的他就可以貢獻出22.2分11.2籃板2.88個蓋帽，這三項數據，全部在NCAA大中西賽區排名第一，再加上接觸籃球之前，奧洛沃坎迪曾經練過田徑、板球和橄欖球，速度、靈活性、對抗性俱佳，這也是為什麼他會被認為是一個天才的中鋒球員。

4、本內特

如果有人真的相信，騎士隊是因為本內特試訓時非常禮貌，就選擇本內特成為狀元的，那他就是個傻子，如果本內特真的沒有天賦的話，或者是騎士隊完全看不上13屆新秀，完全可以將狀元簽交易出去，畢竟一個狀元簽完全可以換到一個優質的首發球員，要知道大一賽季的本內特，就可以貢獻16.1分8.1籃板1.2蓋帽的數據，還有37.5%的三分命中率，並且12次砍下兩雙，入選了全美大一新生最佳陣容，而且本內特運動能力出色，對抗性可以和NBA無縫銜接，進攻技術全面，可以打兩個前鋒的位置，持球突破侵略性強，運球能力不遜色一些後衛球員，培養得當的話，未必成為不了下一個拉里約翰遜。

5、鮑維

喬丹成為了籃球之神，而鮑維最終成為了一個笑柄，兩者的差距是如此之大，但是在當年哪個內線稱霸的時代，開拓者的選擇並不算是一個完全的錯誤，身高2米16，腳步移動、奔跑速度、彈跳力和中投俱佳，在84年選秀大會中，本身就是僅次於奧拉朱旺的第二中鋒，甚至是如果開拓者沒有選擇喬丹和鮑維中的任何一人，那麼公牛也極有可能擁手中的探花簽選擇鮑維，而新秀賽季場均10分8.6籃板2.8助攻2.67次蓋帽的數據，也證明了鮑維的潛力，只不過傷病始終這麼著這個大個子，否則以鮑維的天賦，足以成為一名全明星級別的內線球員。

6、莫里森

如果說選擇誇梅布朗是將喬丹半個身體裝進棺材，那莫里森不僅將喬丹剩下的半個身子也裝進了棺材，而且還給棺材蓋上了個蓋子，不過在當年的選秀大會中，卻沒有人認為喬丹的選擇是錯誤的，畢竟當時的莫里森心態成熟穩重，球場極高，得分手段全面，是聯盟歷史上唯一一個被以拉里伯德作為模板的新秀球員，特別是他在NCAA的表現，完全稱得上是統治，並且莫里森的求勝欲極強，三分投射也算是頂級，還真有些小伯德的影子，只不過在進入NBA後，適應不了NBA對抗強度和比賽節奏，最終淪為了水貨探花。

7、喬史密斯

95年選秀大會，堪稱優秀大前鋒井噴的一年，麥克戴斯、拉希德華萊士、加內特、科特托馬斯，但是當年的狀元，卻是屬於喬史密斯，而喬史密斯也被大家公認為是水貨狀元，其實喬史密斯並不是水貨，身高2米08的史密斯運動能力十分出色，技術也失分全面，特別是進攻端的表現完全配的上自己狀元的身份，新秀賽季就可以貢獻15.3分8.7籃板1搶斷1.6蓋帽的數據，在最佳新秀評選中排名第二，只不過史密斯的求勝欲不強，也找不到自己在球場上應有的定位。

8、斯威夫特

單論身體天賦的話，就連00年選秀狀元馬丁都無法和斯威夫特相比，在大學時期，他勁爆的扣籃就為他贏得了「鯊魚第二」的美譽，更是被很多專家和媒體一度認為，他至少可以達到肖恩坎普的高度，特別是利用聯盟頂級運動能力，他可以在球場上輕鬆的將球送進籃筐和完成蓋帽，是一個僅憑天賦、身體素質和運動能力就能讓聯盟30支球隊欲罷不能的野獸，但是斯威夫特只有天賦，卻沒有腦子。

9、塔比特

2米21的身高，2米29的臂展，超人一等的防守判斷力，同等身高中還算出色的運動能力，籃下還擁有強大的威懾力，當年很多選秀網站和專家都認為，即使塔比特技術粗糙，對抗能力還需加強，但他的選秀順位也不會跌出前五順位，而出色的封蓋能力，也為塔比特贏得了「穆大叔第二」的稱號。

10、巴格納尼

巴格納尼是水貨，說實話有些冤枉巴格納尼，如果把他放在現在的聯盟中，那巴格納尼絕對配的上頂薪級別的薪水，畢竟相比於大洛佩茲，巴格納尼腳步靈活、投籃手感出色，技術全面，能里能外，還擁有同等身高中極為出色的彈跳，擁有著成為空間型內線的全部條件，只不過對抗能力較差，而當時的聯盟，相比於三分，更看重內線球員的對抗性和籃板。
