活塞93-114不敵凱爾特人，綠軍雙子星狂轟51殺瘋了，羅斯拼盡全力

天天要聞
2019-12-21

北京時間12月21日，NBA聯賽持續開戰當中。在今日的聯賽安排中，眾多球迷們肯定是被一場焦點之戰給吸引到了目光，那就是來自於凱爾特人與活塞球隊之間的交火。捫心自問，這兩支球隊之間的實力差距還是較大的，畢竟凱爾特人可是東部排名第二的球隊，放在整個聯盟也是實打實的強隊，但是反觀活塞，則是因為新賽季的糟糕發揮也淪落到了東部第十的位置。兩相對比之下，真是可謂一個天上一個地下。但是本場比賽為什麼還能廣受如此多的球迷們關注呢？原因無非是活塞球隊擁有著羅斯這位球員，作為去年mvp選手的羅斯，作為一名不屈不撓的球場戰士，廣大球迷們依然相信在羅斯的帶領下，活塞球隊依舊有著與凱爾特人一戰的資本。

雖然在賽前大家對活塞球隊滿懷希冀，但是事實未免還是殘酷了些。本場比賽最終是由凱爾特人以114-93的比分優勢大勝活塞，比賽全程毫無轉機，讓人感到絕望。

其實看過本場比賽的球迷們無疑是最清楚了，在本場比賽中，活塞球隊在首節比賽還是非常的不錯，他們使用了新戰術，從專攻內線變成了在外線尋求機會而且收效不錯，一度取得賽場領先。雖然凱爾特人及時的組織了反打併且成功反超比分，好在活塞球隊韌性不錯，在比賽的後半段死咬住了比分不鬆口。在首節比賽結束時，雙方比分差僅為2分。

雖然在後續的比賽中，活塞終於派上了羅斯，羅斯也倚靠著強大的個人能力開始突擊凱爾特人內線尋求得分機會。但是不得不說，缺少了格里芬的活塞球隊在內線方面，確實是沒有了以往的統治力了，而活塞球隊如果想要單靠羅斯和德拉蒙德來撐起球隊的內線大門，明顯是痴人說夢。

在後續的比賽中，面對全員爆發的凱爾特人，活塞球隊再難有招架之力。雖然羅斯在苦苦支撐，但是其對內卻少有除他以外能夠站出來挑大樑的人物。而反觀凱爾特人這邊，塔圖姆和布朗這些年輕球員倒是表現不斷，不知道兩相對比下來，羅斯會不會對球隊感到心寒。不管怎麼說，後勁不足的活塞球隊根本不足以成為凱爾特人的隊手，在上半場比賽結束時，凱爾特人已經領先12分，第三節比賽結束時，比分差更是來到了恐怖的22分。看到活塞球隊在前三節比賽埋下如此之大的坑，不少的球迷們就已經知道本場比賽已經毫無希望，徒留垃圾時間。果不其然，最終活塞球隊還是以93-114不敵凱爾特人，倒在了著21分巨大分差的面前。

正如我們之前所說的，凱爾特人的雙子星在本場比賽中確實表現給力，布朗和塔圖姆的發揮的確給人一種亮眼的感覺。在本場比賽中，這兩人各自拿到了26+5+5和26+5+3的數據，兩人合計也是打出了52分的好數據，基本長承包了球隊一半的進攻任務。除了優秀的數據以外，我們還注意到在本場比賽中這兩人的進攻還具有著高效率的特點，其三分命中率和投籃命中率都非常的不錯。靠著兩位球員如此優異的發揮，凱爾特人能夠取得比賽勝利也不是我們不能提前預想到的。

當然除去凱爾特人這邊的歡聲笑語以外，活塞球隊里的羅斯無疑是非常的讓人心疼。身為一位老將，他曾經身披榮耀，可是作為一名球員，他卻在今天的賽場上奮勇拼搏卻無力回天。在本場比賽的末節中，雖然比賽早已失去懸念，但是我們還是能夠看到羅斯一次次衝擊著對方的內線以求尋找那唯一的得分機會。不得不說，羅斯絕對是值得我們尊敬的偉大球員！
