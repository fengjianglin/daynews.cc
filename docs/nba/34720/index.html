輸給最爛的球隊，自己去做最爛的球隊！11連敗的森林狼志在狀元？

2019-12-25 2,609條評論 3,922次閱讀 658人點贊

北京時間12月24日，森林狼客場挑戰勇士隊，送給勇士賽季首場連勝，自己的連敗也來到了11場。再也沒有賽季初猛如虎的表現了，回到了以前的死氣沉沉。

今年年初加內特跟森林狼兩位當家唐斯和維金斯有一場座談會，在這場座談會上維金斯立下宏願在本賽季進入全明星。在賽季初維金斯確實怒摔枸杞保溫杯，拿出了必進全明星的態度，前18場比賽10勝8負，妥妥的季後賽，妥妥的全明星！然後大家都知道了，11連敗，季後賽和全明星都化作了泡影，而且今年是維金斯的合同年，頂薪看來也沒戲了。

森林狼是一支沒有希望的球隊，因為兩位當家球星讓人沒了希望。2017-2018賽季巴特勒憑藉一己之力帶領森林牛進入季後賽，然後就沒有然後了，打完那個賽季就慌不擇路的跑了。要知道全聯盟最硬吉米-巴特勒在森林狼的時候只是名義上的三當家了，沒有錯，是給這兩個貨當三當家！

森林狼是時候再次重建了，今天輸給了勇士就是很好的表現嘛，就要輸給最爛的球隊，自己去做那支最爛的球隊。在加內特離去之後森林狼就變成了哈士奇，一變十多年，已經沒有了狼性，只會賣萌了！
