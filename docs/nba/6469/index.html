恥辱！4大魚腩輸球，唯勇士被打花 半場狂丟71分，真是要爭狀元簽

2019-11-30 0條評論 13次閱讀 0人點贊

北京時間11月30日，nba常規賽繼續正常進行。在已經結束的比賽中，東西部包括老鷹、尼克斯、灰熊和勇士在內的四支魚腩球隊全部輸球！而在這四場比賽中，昔日的豪門球隊勇士輸球方式最為恥辱，半場就瘋狂丟掉71分，全場也是一場慘敗，也難怪有球迷評論：為了狀元簽臉都不要了！

老鷹與步行者的比賽十分激烈，歷經一個加時才分出勝負，最終主場作戰的步行者在主場以1分的優勢險勝！儘管遺憾輸掉了比賽，但是老鷹全隊在比賽中所展現出的韌勁讓人欽佩，其中球隊核心特雷-楊更是28投16中，轟下49分6助攻的漂亮數據，今晚的他們絕對可以昂首離開。

尼克斯與76人一戰，主場作戰的尼克斯一開始就給了76人很大壓力，上半場結束球隊也是大比分領先。下半場再戰，76人在西蒙斯、恩比德雙核的帶領下發起瘋狂反撲，比分很快就被追上，此後兩隊陷入拉鋸戰。但奈何76人對關鍵球的把握要更好一些，最終也是以101-95的比分收穫勝利。

面對以防守著稱的爵士，主場作戰的灰熊並沒有畏懼。上半場的他們同樣取得了兩位數的領先優勢，但奈何下半場沒有頂住爵士的反撲，優勢轉瞬即逝，最終只能在主場無奈接受一場敗仗。

在庫里、湯普森、拉塞爾等主力接連缺戰的情況下，全明星前鋒格林率領的勇士打出了新賽季以來最恥辱的一戰！面對東部勁旅熱火，球隊第一節就被打出了41-21的進攻浪潮，半場更是丟掉了71分之多，比賽也隨之早早進入了垃圾時間，一場慘敗在所難免！而本場比賽過後，勇士也牢牢佔據著西部倒一的位置，大有爭奪狀元簽的勢頭，昔日的豪門球隊淪落到如此境遇不免令人唏噓！
