假如組個世界聯隊，能打得過美國隊嗎

2019-12-16 264條評論 1,404次閱讀 95人點贊

世界籃球的發展每年都在進步，男籃世界盃美國隊遭遇滑鐵盧就是很好的證明，他們僅僅排名第七名，創歷史最低。儘管哈登、杜蘭特、詹姆斯等超巨都沒去，但如果現在組個世界聯隊和美國NBA球星隊PK，也許美國隊並不能占太多便宜。

以下僅代表個人觀點

世界聯隊首發：

C 恩比德，PF 字母哥，SF 博格達諾維奇 ， SG 維金斯， PG東契奇

美國隊首發：

C 濃眉，PF 詹姆斯，SF 杜蘭特 ， SG 哈登， PG庫里

唐斯、庄神、萊昂納德、保羅喬治、歐文、克萊湯普森、利拉德

擁有詹姆斯杜蘭特萊昂納德的美國隊整體實力依舊比世界聯隊要強，但考慮詹姆斯年紀大了，杜蘭特傷愈恢復情況未知，而且世界聯隊的內線實力總體要比美國隊強，字母哥、東契奇等快速崛起，所以這隻世界聯隊還是能跟美國隊搏上一搏的。
