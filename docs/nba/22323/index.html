科比兩度上榜！盤點NBA那些驚天逆轉的比賽

2019-12-20 1,643條評論 3,348次閱讀 1142人點贊

在今天剛剛過去的比賽中，雷霆逆轉戰勝灰熊，一度落後24分的雷霆，完成大逆轉，值得一提的是雷霆也成為了過去20年以來首支連續兩場至少逆轉24分的球隊，那完成24分逆轉有多難，今天就盤點NBA那些驚天逆轉的經典比賽：

爵士vs掘金

1996年常規賽，爵士坐鎮主場迎戰掘金，但擁有「黑白雙煞「的爵士在上半場落後了34分，進入第三節仍一度落後到36分，作為球隊老大的卡爾-馬龍看不下去了，聯手霍納塞克在第三節以36：15追回了21分，但球隊仍落後13分，在第四節爵士展現出了西部強隊的氣勢，憑藉兇悍的防受成功限制對手只得了18分，最終爵士也以107：103掘金完成了36分大逆轉，這場比賽也成為了NBA歷史上有記錄最大的常規賽逆轉記錄。

公牛vs國王

2009年12月22日，作為當時東部新貴的公牛挑戰西部魚腩球隊國王，雙賽實力顯而易見，比賽也早早失去懸念，上半場結束公牛領先24分，在第三節開場公牛又打出一波12：1小高潮，分差拉開到35節，就在眾人以為比賽失去懸念時，羅斯的同年四號師弟埃文斯站了出了，帶領國王開始了反撲，在第四節比分就贏了公牛23分，最終國王也以102-98完成逆轉，不知這份聖誕禮物對於羅斯作何感想。

快船vs灰熊

2012年，作為老冤家的快船和灰熊又一次在季後賽中相遇，但是比賽在開始不久就早早失去懸念，灰熊在前三節比賽中一度領先快船27分，在第四節還有八分鐘時仍手握24分領先優勢，比賽進入垃圾時間，但是快船替補球員上場後，風雲突變，在尼克楊一分鐘命中三記三分後，快船打出一波26-1的高潮，最終借勢快船收走比賽，以99-98逆轉24分戰勝灰熊，這也成為了NBA季後賽史上最大分差的逆轉。

湖人vs小牛

2002年當時不可一世的「ok組合「帶領湖人迎戰小牛，但比賽似乎沒有那麼簡單，在前三節湖人結束時竟落後小牛27分，但是到了第四節科比爆發，8投8中聯手奧尼爾打出一波44-15的小高潮，帶走比賽，完成了27分大逆轉。

湖人vs凱爾特人

2008年總決賽第四場，在科比的帶領下湖人確立了24分的領先優勢，但是下半場凱爾特人在防守端給湖人持續施加壓力，湖人命中率持續下降，第三節結束湖人的領先優勢也來到了2分。第四節比賽開始，雙方展開攻防大戰，但是在皮爾斯的帶領下，凱爾特人最終獲得勝利。凱爾特人也完成了NBA歷史上總決賽最大逆轉。
