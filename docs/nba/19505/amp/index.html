12連敗，造27次失誤卻輸21分，6分鐘1分未得！這就開始拼演技了啊

天天要聞
2019-12-16

這賽季有許多讓人非常失望的隊伍，有些是新加入的，有些則是一如既往的令人失望，比如馬刺和開拓者，上賽季都是打進了季後賽，開拓者更是殺到了西部決賽，但是到了這賽季，這兩隻隊伍則是在西部末尾掙扎。

像勇士，上賽季還是總決賽球隊，這賽季搖身一變成了徹底擺爛的隊伍，轉變之快令人咋舌，而鵜鶘和尼克斯則是一如既往的爛，尤其是尼克斯，就像有人評論的一樣，尼克斯是一支令人絕望的隊伍！

先來說說鵜鶘，鵜鶘在休賽期得到了英格拉姆、哈特和鮑爾，在選秀大會上選中了萬眾矚目的錫安，但是錫安卻是在一場季前賽中傷了膝蓋，更有傳聞曝出他賽季已經報銷了。

但是英格拉姆卻給了鵜鶘驚喜，場均能拿到25分7籃板3.7助攻，但是也就這點驚喜了，英格拉姆數據好看，但是鵜鶘卻一直輸球，在輸給魔術之後，鵜鶘迎來了12連敗，創造隊史最長連敗，本賽季聯盟最長連敗！可謂是把鵜鶘釘在了恥辱柱上。

勇士這賽季的變化可謂是天翻地覆，所謂風水輪流轉，用在勇士身上再好不過，上賽季的總決賽球隊，5年拿了三個冠軍，這賽季卻是淪為了西部墊底的存在，球隊更是直接加入了擺爛的隊伍。

在與國王的比賽中，勇士造成國王27次失誤，最後卻是輸了21分，就連勇士主教練科爾都在賽後表示，那幾乎不可能發生。是啊，勇士是一支轉為打破不可能而生的隊伍，以前的死亡五小，三分球的浪潮，73勝等都是勇士一手締造，而如今卻有了這樣尷尬的事發生，出來混遲早要還的啊！

尼克斯擺爛，已經是老生常談了，不是一個兩個賽季的事情了，但是無論怎麼擺爛，選秀順位再高，他們選中的球員都是莫名其妙的水掉，唯一好點的波爾津吉斯如今也是在獨行俠飽受非議。

之所以稱之為爛泥，是有一定道理的，而在尼克斯與掘金的比賽中，尼克斯在最後6分鐘內原地斷電，竟然1分未得！有可能是看到了勇士與國王的中場比分吧，似乎全隊都在演蘭德爾和莫里斯，因為隊內只有他們兩個看起來想贏球。

這三支隊伍總能以各種各樣的原因輸球，而這賽制隊伍也是互相比誰的戰績更慘，賽程尚未過半，就開始互飆演技！

目前，勇士似乎看起來更勝一籌，鵜鶘和尼克斯都是比勇士多勝一場，而且勇士比這兩隊都還要多輸兩場比賽。

要知道，對於擺爛的隊伍多勝一場比賽造成的差距是很大的，畢竟，大家都在輸球，你多贏一場的話，你就得祈禱你的競爭對手要贏別人一場，這是很難的。

鵜鶘、尼克斯和勇士，這三支隊伍目前來說由於戰績太過接近，不能保證誰能拿到狀元簽，但是就目前這三支隊伍的擺爛力度來說，只能說狀元簽太香了吧！這三支隊伍也都不要以為自己穩了，畢竟還有老鷹和騎士在那虎視眈眈呢！
