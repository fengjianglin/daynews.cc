52+10+8！綠軍兩巨頭痛擊活塞，沃克6中0，又給歐文上一課

天天要聞
2019-12-21

北京時間12月21日，凱爾特人在主場以114-93大勝活塞，收穫到兩連勝。凱爾特人這邊，布朗26分5籃板5助攻3搶斷，塔圖姆26分5籃板3助攻，威廉姆斯18分3籃板，沃克2分11助攻；活塞遭遇三連敗，德拉蒙德12分11籃板4助攻6搶斷，梅克15分7籃板，羅斯14分4助攻，米哈伊柳克12分。

比賽開始之後，兩隊的狀態相當，比分也是一直緊咬著，首節比賽打完，凱爾特人僅以28-26領先；可到了第二節，凱爾特人這邊開始爆發，一波14-2的攻擊波直接拉開分差，之後活塞也是苦苦追趕，無奈布朗連續命中三分球，半場比賽打完已經領先到了12分。到了下半場活塞一開始還能咬著比分，可凱爾特人這邊攻勢很猛，分差也是漸漸被拉開，三節打完領先到了22分；最後一節凱爾特人沒有給任何機會，隨著分差再被拉開，比賽也是早早失去懸念。最終，凱爾特人114-93輕取活塞拿下比賽。

此役，凱爾特人這邊雙探花發揮十分出色，聯手轟下了52分10籃板8助攻，是球隊贏球的關鍵所在。先來看看塔圖姆他在比賽一開始就進入到狀態，打滿首節12分鐘，單節11中6，拿到13分；而到了第三節再迎來爆發，單節5中3拿到11分，幫助球隊直接將分差拉開，提前鎖定勝局。最後一節他也是沒有再出場，輕鬆幫助球隊拿下勝利。

布朗這一邊則正好與塔圖姆相呼應，他則是在第二節迎來爆發，單節6中5，三分球4中3，單節轟下15分。正是他的爆發幫助球隊拉開分差，而在第三節也是貢獻了7分，最後一節也是作壁上觀。雙探花這一次是真正的痛擊了活塞，讓對手一點辦法也是沒有。

看著雙探花發揮出色，而這一次沃克則迷失了。本場比賽他的投籃手感全失，6次出手無一命中，還出現了7次失誤，按道理球隊核心這樣的比賽應該很難贏球的，可球隊居然還21分大勝。這也要感謝沃克的改變，當他看到自己手感不佳時，他並沒有一味的選擇強攻，而是將注意力都放在組織上，全場送出了11次助攻，這也成功幫助了球隊。

而從今天這場比賽來看，凱爾特人選擇沃克還是非常正確的，他的確比歐文更適合這支球隊。與歐文不同的是，沃克並不是一個強勢的核心，就比如今天，當他手感不佳時，而隊友手感火熱，他就會主動放權，讓隊友去打。這一點歐文還是需要去改進的，沃克也是給他上了一課。
