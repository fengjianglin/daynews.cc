卡特阿密最後一戰邁遭巴特勒0比22羞辱，一成就略遜喬丹歷史第四

天天要聞
2019-12-11

老鷹12月11日做客邁阿密是卡特職業生涯最後一次和熱火的比賽——除非交易，抑或他還能在季後賽來到這裡，但這兩種性已經微乎其微——老飛人本可以帶著勝利離開，終場前59.9秒，老鷹還以117比111領先，但在隨後的6分鐘里，稚嫩的老鷹被對手打出了一波22比0的高潮，最終121比135落敗。

熱火的超級反攻始於鄧肯·羅賓遜的三分球，終場前54.7秒，他接巴特勒的助攻投中了他本場比賽的第8個三分球，全場，他三分球14投10中，得到34分。隨後，巴特勒頂著卡特的封蓋再次三分命中率，僅用30秒，熱火就將比分追成了117平，雙方就此進入加時賽。

然而，卡特和老鷹的噩夢此時才是剛剛開始，在加時賽的前3分36秒里，巴特勒帶領著他的球隊又連續得到16分，一波22比0高潮徹底殺死了比賽。此役，巴特勒得到20分18籃板10助攻，阿德巴約也有30分11籃板11助攻的三雙，熱火在主場拿下11連勝，是聯盟唯三保持主場全勝的球隊，另外兩支是11連勝的凱爾特人和13連勝的76人。

準備在邁阿密光彩謝幕的卡特發揮的已經很棒，他得到12分4籃板3助攻。這也是他第36次以對手身份出現在邁阿密，是聯盟歷史第三多的，僅次於皮爾斯的42次和帕特里克·尤因的37次。卡特職業生涯面對熱火一共得到了714分，排名在喬丹的722分之後是歷史第四。排名歷史前二的依舊是皮爾斯和尤因，他們分別得到了867分和754分。
