本賽季142萬美元新人合同的羅伊斯·揚，右肘過度扭傷將休賽4-6周

2019-11-26 0條評論 19次閱讀 0人點贊

​ESPN 推文的羅伊斯·揚（Royce Young）的右肘過度扭傷後，雷霆搖擺人Hamidou Diallo將在4-6周內重新評估。

迪亞洛在上周五對陣洛杉磯湖人隊的比賽中受傷。

勒布朗·詹姆斯開車上籃，與迪亞洛取得聯繫。當他跌倒時，迪亞洛隨後與隊友邁克·穆斯卡拉（Mike Muscala）相撞。他迅速退到更衣室。

Diallo因左膝酸痛缺席了三場比賽，剛回來。在去年的季後賽中，他在同一肘部進行了外科手術。

上個賽季初，這位2018年的二輪手是雷霆隊輪換的常規組成部分，但在全明星休息後表現不佳。在今年的12場比賽中，迪亞洛（Diallo）場均21.3 MPG，場均8.8 PPG，3.8 RPG和1.5 SPG。缺席期間，阿卜杜勒·納德（Abdel Nader）和狄恩特·伯頓（Deonte Burton）可能會在幾分鐘內看到上升。

來自肯塔基州的得分後衛與球隊的合同是新人合同，本賽季為142萬美元，明年夏天為166萬美元。
