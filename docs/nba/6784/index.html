評選本賽季退步最快球員，年薪千萬個位數得分

2019-11-25 0條評論 19次閱讀 0人點贊

NBA是一個新老交替極快的聯盟，每個賽季都有很多球員實力突飛猛進，當然也有球隊實力一瀉千里。進步的球員一目了然，退步的球員總是悄無聲息。接下來就一起看看悄悄退步的球員。

蘭德爾：蘭德爾是湖人出來的球員，由於蘭德爾本人想要一份長期合同，而湖人當時只能提供一份短期合同，最後導致蘭德爾加盟鵜鶘。上賽季蘭德爾場均21.4分8.7籃板3.1助攻，並且很多時候都是替補出場。休賽期蘭德爾3年6300萬美元加盟尼克斯，目前場均16.4分9.1籃板3.8助攻。按道理來說蘭德爾在尼克斯有絕對的出手權，但是這數據不升反降有點耐人尋味。

康利：康利也是手握億元合同的人，本賽季薪資為3251萬美元。上賽季康利還是場均21.1分6.4助攻3.4籃板，本賽季場均15.4分3.3籃板4.4助攻。康利本賽季表現起伏不定有時候能拿20+，有時候也能入賬個位數。康利作為一個後衛，投籃命中率僅有38.1%，罰球命中率只有78.3%。不知道是爵士的打法，還是康利個人原因造成數據下降，現在的爵士希望有更多人站出來得分。

阿里扎：阿里扎離開某支球隊後，消息就越來越少。上賽季阿里扎場均能得到14.1分5.3籃板3.8助攻，本賽季阿里扎淪為替補，場均4.2分3.5籃板0.8助攻。國王現在需要經驗豐富的球員帶動年輕球員，而阿里扎並未做出表率。不知道是阿里扎年紀大了還是怎麼回事，這數據以後替補都混不上。

巴圖姆：阿里扎現在還能混混替補，巴圖姆現在有可能連替補都混不上了。上賽季巴圖姆場均出場31.4分鐘，場均貢獻9.3分5.2籃板3.3助攻。本賽季只出場6次，場均出場22.5分鐘得到3分5.2籃板4.3助攻。巴圖姆本賽季年薪2557萬美元，明年2713萬美元，這完全不成正比！

戈登：威少的加盟似乎並未影響到哈登，但還是有一人受到影響，那就是戈登。戈登的場均出手在生涯平均數以下，球權一直在下降。上賽季還能場均16.2分，本賽季場均不到11分。再來看看命中率，投籃命中率比上賽季下降了10%，三分命中率比上賽季下降了6.4%，助攻是生涯最低數據。戈登在鵜鶘的比賽中受傷，預計缺席6周時間，不知道回來後是否會找到狀態。
