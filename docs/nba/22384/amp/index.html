球哥是否羨慕英格拉姆？同樣是被交易，球哥已經難兌換天賦了

天天要聞
2019-12-19

鵜鶘隊本賽季是準備大展宏圖的，因為在季前賽上，狀元錫安威廉姆斯發揮出色，英格拉姆也是小試牛刀，兌換了自己的天賦，鵜鶘隊雖然失去了戴維斯，但是可以看出他們球隊的天賦異稟，未來大業可期，甚至不要未來，本賽季就可以。

但是結果，錫安威廉姆斯受傷了，甚至很有可能賽季報銷，因此，鵜鶘隊選擇了擺爛，不過好在，英格拉姆本賽季兌換了自己的天賦，也算是鵜鶘隊的收穫了，而鵜鶘隊擺爛的原因還有，球哥鮑爾沒有兌換天賦，令人失望。

目前，英格拉姆場均25分，小杜蘭特之名已經享譽聯盟，而當初，鵜鶘隊交易戴維斯，得到的籌碼裡面，英格拉姆並不是最令人期待的，他們更渴望的是鮑爾，這個迎來生涯第三個賽季的控衛，因為球爹的關注，備受關注。

球哥除了話題比較多以外，天賦也是不錯的，出道至今，也是被譽為小基德的存在，但是遺憾的是，球哥加盟鵜鶘隊並沒有兌換出比湖人隊更好的天賦，反而迷失了自己，甚至更球爹鬧翻了，處境真的不好。

但是最重要的還是，球哥的天賦為何遲遲無法兌換，不太引起關注的玻璃屬性問題，他在投射能力上的困擾似乎已經是老生常談了，職業生涯球哥最高的投籃命中率只有40.5%，三年整合起來更是只有37.9%。

雖然，球哥的組織進攻能力不錯，防守能力也不錯，說他沒天賦，那確實也是過份，但是職業生涯經歷來看，三年過去了，球哥確實不如英格拉姆，個人性格內斂，進攻慾望不足難成大器，這句話倒是有點道理。

為何說是進攻慾望不足呢？模板基德巔峰時候起碼還會場均出手16次，反觀鮑爾至今場均最多也只是出手10.8次。而且還是在鮑爾急需證明自己的時刻，都是這樣的表現，如果拿到大合同，還會有表現嗎？

當然，現在說球哥變成水貨還為時以早，畢竟對於球哥來說，他的職業生涯經歷也是夠強悍的，就跟他的投籃一樣，雖然很醜，突然有一天，看他的發揮還不錯，畢竟球哥，還很年輕，還有時間兌換天賦。
