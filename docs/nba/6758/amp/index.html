阿德巴約21+13+5 巴特勒21+7 羅齊爾19+9 熱火送黃蜂五連敗

天天要聞
2019-11-26

直播吧11月26日訊 NBA常規賽，黃蜂今日前往客場挑戰熱火，客隊目前遭遇四連敗，上場比賽更是成就了拉文的三分絕殺，而主隊雖然上場比賽同樣遭遇失利，但仍高居東部第三，此役溫斯洛繼續因傷缺戰。比賽上來黃蜂迅速反客為主一波7-0開局，但熱火及時調整反超比分，次節熱火外線手感全開，逐漸將分差拉大到兩位數結束半場。易邊再戰，黃蜂依舊沒能找到限制對手的辦法，場上分差被繼續擴大，比賽也早早失去懸念，最終熱火在主場大勝黃蜂，送給後者五連敗。

雙方數據

黃蜂（6-12）：羅齊爾19分9板5助，格拉漢姆16分8助，澤勒10分2板，比永博11分5板，蒙克13分2板2助，華盛頓8分4板，布里奇斯9分2板2助，馬丁4分1板，巴圖姆3分2板4助3斷。

熱火（12-4）：巴特勒21分3板7助，阿德巴約21分13板5助，納恩19分2板4助，希羅18分2板，奧利尼克15分16板，羅賓遜6分2助3斷，德拉季奇6分3板9助，席爾瓦7分5板，萊昂納德4分4板。

首節上來，澤勒率先順下暴扣得手，布里奇斯跟進命中追身三分，羅齊爾罰球線拋投命中，黃蜂打出一波7-0完美開局，巴特勒翻身後仰為熱火拿到首個進球，納恩和羅齊爾分別射進三分，巴特勒自投自搶雙手補扣將球打進，阿德巴約籃下強打2+1，熱火迅速實現反超打停黃蜂，調整過後華盛頓劈扣得手，希羅外線命中遠投，巴圖姆底角也中三分，此後場上哨聲頻頻，雙方多次站上罰球線。首節結束，熱火36-31領先黃蜂。

次節開局，德拉季奇借掩護上進反籃，巴圖姆射進三分，但希羅立馬在外線給出回應，比永博跟進飛身上籃得手，蒙克突破拉杆打成2+1，奧利尼克外線連中兩記三分，澤勒禁區內拋投得分，暫停調整過後納恩遠投命中，巴特勒搶斷一條龍打成2+1，熱火逐漸將分差拉大到兩位數，羅齊爾終於命中止血三分，澤勒跟進上籃打成2+1，奧利尼克延續手感再中三分，布里奇斯橫移籃下拋投得手。半場戰罷，熱火69-58領先黃蜂。

易邊再戰，納恩外線繼續命中三分，隨後又投進急停中距離，華盛頓跟進上籃得手，布里奇斯突破奉獻精彩雙手暴扣，萊昂納德空切回敬暴扣2+1，格拉漢姆外線造成三分犯規，暫停調整過後萊昂納德籃下接球暴扣得手，阿德巴約吃餅再打2+1，此後雙方加大對抗強度，多次出現犯規，馬丁二次進攻上進兩分，格拉漢姆反擊再次將球打進，奧利尼克上進反籃後場上分差達到二十分，蒙克連中兩記三分止血。三節結束，熱火98-82領先黃蜂。

末節上來，希羅命中急停跳投，阿德巴約籃下吃餅雙手暴扣，羅齊爾拿到前場籃板同樣扣進兩分，暫停調整過後羅齊爾再度打成2+1，格拉漢姆殺到禁區拿到兩分，阿德巴約籃下暴扣打成2+1，隨後又上進反籃，黃蜂則連續出現失誤，德拉季奇快下輕鬆將球打進，布里奇斯終於打進止血兩分，但比賽已經早早失去懸念，最終熱火在主場大勝黃蜂，送給後者五連敗。

雙方首發

黃蜂：格拉漢姆、羅齊爾、布里奇斯、華盛頓、澤勒

熱火：納恩、巴特勒、羅賓遜、萊昂納德、阿德巴約
