「楊毅專欄」林書豪在CBA受欺負了嗎？

2019-12-19 1,024條評論 2,061次閱讀 1045人點贊

林書豪發出的一條社交媒體，左眼下方的劃痕和「救命，今晚又是籃球+拳擊」的表達，以及北京電視台的同行們剪出來7分鐘上海隊對林書豪的防守細節動作，把「國內球員針對林書豪」這個話題，和上海隊主防林書豪的羅旭東，推到了風口浪尖上。

書豪的支持者們怒吼：憑什麼欺負我們書豪？

前幾天，我正好在上海。上海對首鋼和北控這連續兩個主場，我都在現場做了解說。有一個細節故事，說給大家聽聽。上海打北京首鋼那場球，打到半場休息，羅旭東主動去找了林書豪。羅旭東對書豪說：我這個人打球就這樣，對抗比較硬，小動作比較多。但我不是針對你，我平時就這樣，防誰都一樣。你心裡不要有什麼想法啊，我下半場還是這麼防。

多有意思的表達。

我不知道你們怎麼想。我是覺得，在說這番話的時候，羅旭東其實挺真誠的，從裡面也能看見他對書豪的尊重。不然，他完全沒必要去說。兩軍交兵，各為其主，你死我活，羅旭東和書豪本無個人交情，我管你怎麼想呢。同時，羅旭東這番話里也給出了三個信息：第一，我是個專防小外援的防守悍將，我的防守習慣就這樣；第二，CBA的整體環境就這樣；第三，在CBA打球的小外援，平時都是這個待遇。

注意啊，如果你因為我在羅旭東身上使用「真誠」這個詞而感到不適，我想強調一下，我並不站在這些使用超規兇狠動作的運動員這邊。這一直是外援來到CBA艱難適應的部分。國內不少運動員在防守時使用的揮肘、頂胯、抬膝蓋、墊腳、對腳、絆人動作，是他們從體校開始就學會的招數和習慣。倒退20年，很多教練都會教這些動作。

我到現在仍然記得，我職業生涯最初採訪的幾場球之一，北京首鋼的外援阿爾斯通（不是火箭那個），瘋狂追打瀋陽部隊隊一名以下黑腳著稱的前鋒，阿爾斯通實在忍不了了。

後來在北京廣安體育館，奧神隊的前衛五虎之一，老將苑志南，堪稱狠招這一行的教師爺，一記鐵膝蓋，直接頂斷了快速突破的首鋼隊外援克里斯的右膝十字韌帶。20年來，CBA一直在力圖規範防守動作，保護運動員。國際籃聯修改的規則對進攻運動員的保護，也比過去嚴格得多。這從現在違體犯規吹罰的頻繁程度上就可見。CBA防守超規動作的使用，已經比過去少多了，但想根治還有漫長的過程。況且，國際籃聯的規則和NBA相比，本身就允許更激烈的對抗，進攻運動員更難獲得空間（NBA罰球線以上禁止手測，有防守三秒拉開籃下空間），很多激烈動作發生在狹小空間內，也造成吹罰的困難。

再說一遍，我同樣痛恨那些壞動作。但我認可羅旭東表達里的那一點，就是這些防守，不是從林書豪這裡才開始的，也不是林書豪一個人面對這樣的防守。CBA的每個小外援，每個外線持球的頭牌，包括郭艾倫，都在面對同樣的防守。在打首鋼之前，上海打的就是遼寧。羅旭東怎麼防的林書豪，他也怎麼防的郭艾倫。

這就是為什麼，在面對「林書豪在CBA是不是被針對」這個話題時，你所見到的幾乎所有籃球行業內的人們（記者，評論員，運動員，教練），都和林書豪的支持者們意見不同的原因。行業內的統一觀點，就是你只有面對這些，而且這不是書豪的獨特待遇。因為他們了解CBA，了解籃球，而不像書豪的支持者里，很多以前從來不看CBA，甚至從來不看籃球。

你看見上海防守書豪的這7分鐘視頻，如果感覺很觸目驚心，就是因為你沒有近距離看過CBA的多少比賽。其實你把CBA其他小外援面對防守的細節剪出來，北控福格，浙江鄧蒙，山東哈德森，廣東馬尚，你把他們面對防守的細節剪出來，大家都一樣。誰身上都多少條血道子，只是他們都是黑人運動員，你輕易看不出來。

CBA幾乎每支球隊都有專防外援的防守悍將，超規動作都不少，比羅旭東厲害的多得是。不光是國內的，外援里像福特森，防守小動作就很多。咱們就不一一點名了。越到激烈的比賽，越到季後賽，對抗越兇狠，超規動作越多。我給大家講一個真實的故事。幾年前有一次CBA總決賽，因為兩邊的小外援能力都極強，都是對方的眼中釘肉中刺，兩邊球隊內部都定了一條「策略」，極致兇狠的防對方小外援，所有動作無所不用其極。而且如果有機會，爭取把小外援「幹下去（你們自己理解啊）」，誰能把小外援幹下去有重獎。

站在這個角度上，不管你喜不喜歡，CBA就像一座叢林。這裡的文化有些像叢林法則，沒有那麼多規範，就是恃強凌弱——其實對抗項目的文化都是如此，NBA在新千年以前的規則和文化也是如此，但CBA更強烈和明顯。很多時候，這裡比的是誰橫，誰混，誰更硬。羅旭東也這麼防郭艾倫，為什麼看上去沒有他防林書豪這麼大優勢呢？因為艾倫也橫，他干艾倫，艾倫也干他。

CBA最具江湖情境的對抗，發生在當年北京首鋼對山東的比賽里。山東的悍將睢冉以防小外援著稱，那身板，那兇猛，當世聞名。睢冉把他一身的功夫，給馬布里全用上了。那老馬也不是善茬兒啊，老馬多狠呀，把一身橫練的功夫，也給睢冉用上了。北京打山東，場場這倆人往死里打。有一場，打到後面玩命了，倆人全急眼了。老馬飛起來一鐵肘，把睢冉直接打飛了，當時老馬就被罰出去了。

外界都認為老馬和睢冉結了天大的梁子。結果最後老馬退役的時候，睢冉發微博致意，讚賞老馬的強橫，說「防過你是我的榮幸」。老馬也回復，對睢冉大加讚賞，說他是個真正的鬥士。你說這場面，像不像水滸傳裡頭，豹子頭林沖劫道，碰上了青面獸楊志，倆人大殺大砍一番，最後倒成哥們兒了。CBA，就是這樣一處江湖氣很重的所在。沒有混橫之氣，難以在這裡真正立足。

這就是我為什麼曾經說過，對小外援來說，打CBA很難的。不光對能力的全面要求和每場接近40分鐘時間，你要面對的是千槍萬刃的針對性防守。隨你信不信，一名小外援在CBA面對的比賽強度，遠超他們在NBA面對的情況。我再說極端一點，勇士厲害不厲害？可是水花兄弟要來CBA都不好使。克萊自己運球攻的能力有限，庫里的身板兒，在CBA的規則和防守下的受傷幾率遠高於在NBA。

所以我還說過，其實林書豪不是太適合打CBA。一是因為他的個性——書豪人實在太好了，哈佛高材生，溫文爾雅。書豪是來自台灣的美國人，但他確實和老北京隊的風格很合拍，規規矩矩，打學生球。在CBA這個江湖上，你不夠凶，不夠狠，你就是秀才遇到兵。書豪說，我還是乖乖打球。可在CBA，太乖的外援就很難生存。

第二，還因為書豪現在的實力和特點。我說過很多次，我無比尊重書豪，但客觀上他現在的強攻能力，在小外援這個群體里是比較靠後的。書豪的三分球不準，這很不利於打沒有防守三秒的國際籃聯比賽，這意味著他只能靠節奏打到中近距離防守密集的區域去進攻，那裡本身就有太多對抗，全是胳膊全是腿。同時，書豪的絕對啟動速度也不快，這給了防守人更多機會去給他上身體。很多小外援，速度快，三分准，刀刀殺人，國內球員不是不想逮，是想上身體上不著。

我前面說的那次總決賽，那兩個小外援打完系列賽，除了全身幾百個血道子，都沒受硬傷。為什麼？兩個人都是速度快，力量極好。用防守人的話說，「我連摸都摸不上他」。就算離近了給對抗，也是國內球員彈出去。幾個回合下來，你還敢近他身嗎？而書豪，是經常在對抗里倒地。防守人一看，對抗我佔便宜，那就更狠了，十八般兵器全上了。

競技體育，弱肉強食。不知道這在書豪的支持者們看來，算不算針對和欺負書豪。

這兩天，這個話題被激起之後，書豪的不少飯圈粉絲的反應，引起了很多籃球從業者和資深籃球迷的不滿。其實真沒必要。這是書豪為CBA做出的重要貢獻，讓更多人，更多以前甚至不看球的人，開始關注CBA。她們購買林書豪的球衣，成為CBA聯賽的消費者，這在以前不可想像。任何一個行業要擴大圈層和影響力，都要經歷這個過程。

在中國，競技體育的核心受眾，也就是能真正解讀比賽，關注點在比賽本身的群體，只佔總數的15%左右。影響力要擴大，顯然需要更多非核心受眾，特別是女性受眾進來，才能形成破圈和社會影響。這是這一季CBA其他運動員，任何外援和林書豪無法相比的能力。因此CBA的確要保護書豪。其實在大多數比賽里，你可以發現裁判很注重吹罰對書豪的犯規和對抗，哪怕有一些常規對抗也吹了。像上海那一戰非常寬鬆的尺度，在本季北京首鋼的比賽里是很少出現的。

在騰訊體育首鋼對上海那場球轉播的下面，我發現了這樣一條回帖，它的點贊數最高。這條回帖和下面的第一條回復，就是兩個群體不同邏輯最鮮明的體現。可能他們彼此之間永遠不會認同，但我還是覺得，其實挺好的。
