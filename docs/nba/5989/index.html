空手套白狼？尼克斯希望得到樂福、德羅贊 多蘭不願付出選秀權

2019-12-08 0條評論 16次閱讀 0人點贊

​近年來，紐約尼克斯的一系列交易讓老闆多蘭成為笑柄。今年夏天，尼克斯曾重點追求歐文和杜蘭特，但因多蘭臨時擔心杜蘭特的傷病情況，兩位球星最終加盟了籃網。

本賽季，尼克斯的戰績非常糟糕，多蘭決定解僱原主教練菲茲戴爾。根據最新消息，多蘭並未對未來感到灰心，尼克斯計劃在賽季中期交易市場上大展身手，用大交易來扭轉命運。

根據SNY記者伊恩·貝格利報道，12月15日之後，今夏簽約的NBA球員可以被自由交易。目前，尼克斯正在關注「很多球員」，這些球員能夠提升球隊實力，幫助尼克斯改善戰績。

據報道，尼克斯最希望得到的球員是騎士前鋒凱文·樂福、馬刺的德羅贊和雷霆的施羅德。

紐約本賽季戰績為4勝18負，是東部最差的戰績。分析人士認為，菲茲戴爾成了尼克斯的替罪羊。

今年休賽期，尼克斯簽下了很多內線球員，包括蘭德爾、吉布森、波蒂斯和莫里斯，隊內還有羅賓遜和諾克斯兩位年輕大個子球員，內線十分臃腫。

尼克斯缺少一位明星球員，莫里斯場均得到18.3分6個籃板，表現最為搶眼。

尼克斯希望得到明星球員，但不希望送走選秀權和年輕球員。
