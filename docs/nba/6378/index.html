西亞卡姆35+5+5范弗利特21+11猛龍血洗爵士取七連勝

2019-12-02 0條評論 17次閱讀 0人點贊

NBA常規賽繼續進行，猛龍主場130-110大勝爵士。

康利開局命中三分、戈貝爾補籃得分，小加索爾連續回應三分再有扣籃，西亞卡姆也中三分後和阿努諾比各上籃得分，小加索爾再中三分，西亞卡姆再有上籃和三分，猛龍首節打出一波30-5的瘋狂一波流，一度領先了21分，首節就37-20領先。

第二節傑夫-格林上籃2+1然後扣籃得手，范弗利特連續三分回應、霍勒斯-傑弗森打成2+1再有上籃，分差已經來到了25分。之後特倫斯-戴維斯再中三分，西亞卡姆和鮑威爾也中三分，節末鮑威爾和范弗利特各中三分，西亞卡姆壓哨再進三分！半場猛龍77-37領先了40分。

易邊再戰後奧尼爾和康利都中三分，米切爾連續中投得分後康利中投也中。之後戈貝爾連續籃下得分，米切爾和康利又都中三分，博揚和英格爾斯也連彪三分。這一節爵士瘋狂砍下49分，不過三節過後仍然86-107落後。第四節爵士始終未能將分差縮小到17分以內，比賽早早失去懸念。

全場比賽康利得到20分，戈貝爾12分11籃板，博揚-博格達諾維奇11分7籃板，米切爾16分，傑夫-格林19分。猛龍這邊西亞卡姆得到35分5籃板5助攻，范弗利特21分11助攻5籃板，鮑威爾15分4搶斷，小加索爾11分6助攻，伊巴卡13分，特倫斯-戴維斯13分，霍勒斯-傑弗森10分2蓋帽。
