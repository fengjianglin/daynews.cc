厚積薄發？克萊恢復投籃訓練，狀態比杜蘭特更好，球迷：商量好的

2019-12-11 1,211條評論 3,078次閱讀 2157人點贊

北京時間12月11日，NBA常規賽繼續緊張地進行著，這一天雖然沒有勇士隊的比賽，但是他們卻迎來了一個好消息，球隊的第二巨頭克萊-湯普森正式恢復投籃訓練！據勇士隨隊記者Logan Murdock報道，勇士球星克萊-湯普森出現在了訓練場，並完成了幾組投籃訓練。這對於戰績慘淡的勇士隊來說，也算得上是一個好消息了。

​眾所周知，北京時間7月3日，克萊在洛杉磯成功接受十字韌帶撕裂修復手術，當時預計克萊將傷缺5-7個月，時至今日，克萊已經休息了5個月的時間了。從畫面中看來，克萊的投籃手感保持得相當不錯，外線接連命中三分，可謂是彈無虛發。另外還有一點，克萊的恢復狀態看起來似乎要比杜蘭特更好。

在前段時間裡，籃網隊也公布了一段杜蘭特恢復投籃訓練的視頻，不過，在杜蘭特的訓練中，基本上都是定點投籃，沒有任何的運球動作，更沒有任何的起跳動作。然而，在克萊的訓練視頻中可以明確看到，克萊已經可以正常起跳投籃，而且，克萊還簡單地進行了幾次胯下運球，然後撤步投籃。如此看來，再過一兩個月，克萊很有可能就要迎來複出了。

另外，勇士隊的當家球星庫里目前也正處於恢復階段，一個月前，庫里遭遇了左手第二張掌骨骨折，並且接受了手術治療，當時預計將會缺席3個月的時間。而且，庫里前段時間裡接受了第二次手術，已經將手中的固定鋼板取了出來，預計將會在明年的2月份再次進行傷勢評估。

也正因如此，庫里、克萊兩人的復出時間引來了球迷的熱議。因為按照這樣的時間推算，克萊從7月開始傷停，7個月之後，他的傷勢評估時間也是在明年的2月份，和庫里的評估時間基本相似。這樣看來，難道勇士隊這是要厚積薄發，等到庫里、克萊全都復出之後，拉塞爾、格林等人也將磨合得差不多了，之後便將開始瘋狂的連勝模式？

對此不少球迷表示：「你們哥倆商量好的吧，一起休息，一起複出！」「沒必要了，復出之後，磨合為主，休賽期再拿個不錯的選秀，來年再戰！」對於勇士隊來說，確實如此，即便庫里、克萊在明年傷愈復出，也沒必要再去爭搶什麼排名了，不如磨合為主，戰績為輔，再拿個不錯的選秀簽，豈不是美滋滋！
