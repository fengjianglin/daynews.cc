西部徹底亂了！勇士不甘沉淪，有支球隊硬要靠攏，四檔劃分已明確

2019-12-10 1,698條評論 4,526次閱讀 152人點贊

西部徹底亂了！勇士不甘沉淪，有支球隊硬要靠攏，四檔劃分已明確！

北京時間12月10日資訊；剛剛結束了今天的所有比賽，與昨天相同，聯盟又安排了9場比賽，其中快船110-99炸死你哼步行者，綠軍110-88大勝騎士，雄鹿110-101戰勝魔術，猛龍93-92險勝公牛隊，活塞105-103絕殺鵜鶘，國王119-118絕殺休城球隊，森林狼109-125不敵太陽，雷霆104-90擊敗爵士，灰熊110-102戰勝勇士，單是西部球隊就佔了六場。這讓西部格局徹底亂了，勇士隊不甘沉淪，卻有一支球隊硬要靠攏，四檔劃分已經明確！

格林

勇士隊雖然在主場8分不敵灰熊隊，但整場比賽能夠看到勇士球員們想要贏球的決心，無奈球員們手感太差，尤其是末節追分階段，灰熊給了機會，出現不少失誤，但勇士這邊就是投不進能怎麼辦？曾經勇士王朝球員之一的格林在上半場就轟下14分，表明了自己的決心，但整個下半場就拿到2分，真是有心殺賊無力回天，得分一直都不是格林的殺手鐧，然而看看身邊隊友，還有誰能站出來？輸掉主場比賽，又坐穩聯盟倒數第一的位置，但看得出來，勇士並不甘心就這樣沉淪下去。

格林

在勇士想要改變現狀的時候，卻有一支球隊硬要向勇士靠攏，他就是擁有今年狀元錫安-威廉姆斯的鵜鶘隊。這支球隊同樣是飽受傷病困擾，錫安至今還未打過一場常規比賽。初步預計也要等到明年復出了。原本遭遇八連敗，想要利用主場優勢止住頹勢，沒想到被羅斯壓哨絕殺了，吞下九連敗即將坐穩西部倒數第二的位置，與灰熊已經有0.5個勝場的差距。原本以為今夏與湖人完成交易後，鵜鶘會用行動證明濃眉的選擇是錯誤的，但還是失敗了，主教練金特里賽後也表示：「我們正在最難的一段時間。」

英格拉姆

不過雷霆能夠戰勝爵士隊讓人感到一些意外，儘管爵士最近狀態不佳，但整體實力還是強過雷霆，然而就在這樣的基礎上，富有經驗的保羅還是率隊擊敗了對手，逐漸穩固西部第七的位置。如果保羅最終帶隊進入季後賽，就不知道某人的臉疼不疼。與此同時，西部球隊已經被化為四檔，其中湖人隊以21勝3負的成績獨一檔，後面快船到西部第八的太陽為一檔，而國王隊到西部第十三的灰熊為一檔，鵜鶘和勇士單獨為一檔。

西部球隊劃分

當然賽季還很長，後面還會發生一些簽約、交易的現象，最終所有球隊的成績會發生怎樣的變化也不能確定。球迷朋友們對此怎麼看？歡迎大家在評論區留下自己的觀點。
