巔峰的他連續多年場均20+10，如今卻無法在名人堂佔一席位

天天要聞
2019-11-27

提起上世紀90年代的NBA頂級控衛，你會想起誰？斯托克頓（John Stockton）絕對是一個吧，還有「手套」佩頓（Gary Payton），蒂姆-哈達威（Tim Hardaway）也可以排上號，再晚一點出道的基德（Jason Kidd）自然也是。

但你知道嗎，太陽名宿「KJ」凱文-約翰遜（Kevin Johnson）也應該位列其中，他在巔峰的那幾年甚至被認為是聯盟第一控衛。當他進入NBA之時，完全顛覆了大家對一號位的定義——當時很少控衛被當作球隊頭號得分手，但約翰遜做到了，他在24歲的時候就成為太陽的得分王和助攻王（場均20+10）。那麼，約翰遜到底有多強呢？為什麼他的職業生涯很少被人提及、也依然無法進入名人堂？

在加州大學（UC Berkeley）打了四年籃球（場均14.4分4.4助攻）後，約翰遜參加了1987年NBA選秀。那一年選秀擁有很多潛力新星，包括「海軍上將」羅賓遜（David Robinson）、皮蓬（Scottie Pippen）和米勒（Reggie Miller）等，而約翰遜則在首輪第7順位被騎士摘走。

其實那時候騎士陣中已經有一個年輕的首發控衛普萊斯（Mark Price），所以初來乍到的約翰遜只能充當替補競爭上場時間。根據媒體報道，約翰遜對於自己長期坐板凳、場均只打24分鐘感到不滿，他和普萊斯之間的抗衡愈演愈烈。

雖然普萊斯對外界說這是一種有利於球隊的良性競爭，但事實是騎士也不想同時擁有兩個得分大於組織的控衛（所以為什麼要選約翰遜？），加上他們想增添一些即戰力，於是新秀賽季還未打完，菜鳥約翰遜就被交易離隊了。

約翰遜和另外幾位替補隊友被交易到太陽，換取全明星前鋒南斯（Larry Nance）。很多人斷言，約翰遜有著無與倫比的天賦，能力不亞於同屆的任何一個潛力新星，在新的球隊一定可以大幅進步。他們說對了，在克利夫蘭場均7.3分3.7助攻的約翰遜準備在鳳凰城起飛了。

一年後，約翰遜用爆炸性的表現轟下場均20.4分12.2助攻，命中率超過50%。他球風夠燃、擅長突破，最致命武器是快如閃電的第一步和極具侵略性的攻筐。同時，約翰遜的球場視野非常好，連大學時期被視為最大短板的跳投技術也突飛猛進。賽季結束後，約翰遜捧回了最快進步球員獎，入選賽季二陣。這是一個不少的驚喜，因為沒有多少人在生涯第二年就拿到這些榮譽。

得知自己獲獎後，約翰遜說MIP是一個很特別的獎項：「我總是告訴其他人，不要在意你從哪裡起步，重要的是你在哪裡收穫。」

22歲的約翰遜幫助球隊殺入了1989年季後賽，場均得到23.8分12.3助攻。雖然在西部決賽上輸給了魔術師（Magic Johnson）和湖人，但約翰遜憑藉出色的發揮獲得了一致好評，被認為是下一個聯盟的現象級控衛。事實上，NBA歷史中只有約翰遜、魔術師（2次）和「微笑刺客」托馬斯（Isiah Thomas）做到過單賽季20+分12+助攻。

進入第三年，約翰遜場均依然打出22.5分11.4助攻，再一次選入二陣的同時首次出現在全明星賽場。球迷和對手開始重點關注這個身高只有6尺1寸（185cm）的爆炸控衛，尤其是他攻擊籃筐的慾望和能力，還可以在突破中送出助攻。作為比較，約翰遜就像是90年代的威斯布魯克（Russell Westbrook），一個跳投更精準的威少。

1990年季後賽，太陽又在分區半決賽和湖人碰上了。這一次，魔術師還是魔術師，但約翰遜更勝一籌，他場均21.8分11.2助攻的表現力壓對手——屢屢在第四節關鍵時刻挺身而出——率領太陽4-1淘汰湖人。系列賽結束後沃頓（Bill Walton）評價約翰遜摧毀了魔術師的演出，已經是聯盟首屈一指的控衛。

可惜，太陽隨後無法在西部決賽戰勝開拓者，儘管約翰遜再次貢獻場均20+11的亮眼數據。不過這時的約翰遜是名副其實的巨星，太陽陣中乃至西部、甚至NBA的第一控衛。

1990-92兩個賽季，由於陣容老化和其他對手實力增強，太陽遭遇了一段低谷，雖然約翰遜還有場均22.2+10.1和19.7+10.7的輸出，球隊也背靠背打出常規賽50+勝場，但季後賽先後在首輪和次輪就出局。於是，急於求變的太陽在1992年夏天通過交易將巴克利（Charles Barkley）招致麾下。

這對於球隊來說是天大的好消息，因為巴克利令太陽重回西部強隊行列，不僅創造常規賽62勝紀錄、殺入總決賽，「查爾斯爵士」本人還拿下MVP。只是對於約翰遜來說，太陽自此變成了「巴克利的球隊」：他的出手明顯減少了，降至場均11.5次，每晚上場時間也只有33.5分鐘（太陽生涯最低）。

更慘的是，1992-93賽季開始前，約翰遜被診斷出患有疝氣，不得不休戰了大半個賽季，而且還因為疝氣引起的大大小小傷病一直打打停停。

約翰遜曾坦白表示自己一開始並不喜歡巴克利，但之後兩個人成為了好朋友。巴克利的防守牽制力太強了，約翰遜發現自己不需要再與對方的彪形大漢在突破中硬碰硬，MVP隊友的火力支持也令他的傳球更有效率。

而在巴克利看來，這個每晚和對方三四名後衛激烈對抗，並勇于飛入禁區的小個子也非常值得尊敬。

1993年的季後賽上，火線復出的約翰遜依舊生猛，只是在與公牛苦戰6場的總決賽上他受困於傷病沒有打出百分百狀態，還增添了不少新的傷患，包括繩肌、腹股溝、小腿等地方拉傷，全都是影響運動力的部位。

接下來的四個賽季，約翰遜每年要缺席20到30場比賽，但他是一個硬漢，總是儘可能地克服傷病回到球場，而且從沒有缺席過季後賽。1993-95兩年的季後賽上，約翰遜又回到了第一控衛的水準，場均分別砍下20.1分9.6助攻和22分9.3助攻（命中率高達57.3%）。1994年季後賽他在「大夢」奧拉朱旺（Hakeem Olajuwon）頭上的那記暴扣，更是他炸裂球風的縮影，成為NBA不斷重放的經典瞬間之一。

1996-1997賽季，巴克利離隊，約翰遜迎來了最後的光輝。他出戰70場比賽（近6年最多）場均20.1分9.3助攻。但因為再次查出患有疝氣，約翰遜沒有了以往的勁爆突破和籃底攻擊，轉而出手更多中投和三分（場均扔2.9個三分）。

再一年後，約翰遜徹底被傷病擊倒，加上基德賽季中轉投太陽，才31歲的KJ已然不是太陽核心控衛。1997-98賽季後約翰遜宣布退役，儘管1999-00賽季曾短暫回歸，也只是英雄遲暮。

那麼，巔峰時期連續多年20+10的約翰遜，到底有多強呢？不可否認，約翰遜在最好狀態的那幾年擁有第一控衛的實力，也是聯盟前十的巨星，在競爭激烈的90年代，他能夠在好幾個賽季進入MVP候選討論，這足以說明一切。當我們評價約翰遜，就像是評價一個像艾弗森（Allen Iverson）一樣快、像羅斯（Derrick Rose）和威少一樣「爆」的球員。

不過根據數據網站Basketball Reference的統計，約翰遜進入名人堂的幾率只有19%，可以理解為幾乎不可能入選。這又是為什麼呢？一個很重要的原因，是約翰遜的生涯和巔峰期都太短了。

滿打滿算，約翰遜在NBA真正打球的日子不過11個賽季；當他在太陽早期冒起，全明星隊友錢伯斯（Tom Chambers）或多或少掩蓋了他的光芒；到了個人巔峰之際，巴克利又空降鳳凰城；之後頻繁的傷病嚴重影響了約翰遜，後期基德的接棒更直接終結了KJ的生涯。

約翰遜是那種典型的有絕對實力、但缺少光環的球員，他生涯3次全明星、5次年度陣容（4次二陣，一次三陣）、MIP。真心希望他能夠被更多人談論起，並在未來於名人堂留下屬於他的名字。

值得一提的是，退役後的約翰遜開啟了政客生涯，他曾是薩克拉門托的市長，為地區教育和民生髮展做出不少貢獻。
