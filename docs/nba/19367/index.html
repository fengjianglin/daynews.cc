本賽季印象深刻4大錯覺：恩比德進攻遇強則弱，哈登得分全靠刷？

2019-12-18 566條評論 3,689次閱讀 918人點贊

不知道老球迷是否還記得，當年的「德州三大錯覺」，具體來說就是「紅隊強，獨行俠隊弱，馬刺隊老」，之所以說這些是錯覺，那是因為後來的事實證明了這些全部都是錯誤的，其中最弱的獨行俠隊在10-11賽季拿到了總冠軍，最老的馬刺隊在13-14賽季仍然能夠奪冠，其實本賽季至今，聯盟也出現了幾大錯覺，而且比當年的德州三大錯覺，錯的還要離譜。

1、哈登命中率太低，出手太多

本賽季參加的27場比賽中，哈登場均可以得到38.9分5.9籃板7.4助攻，牢牢佔據著得分榜榜首的位置，但是這樣的得分在很多球迷看來，是哈登用高出手刷出來的，誠然，哈登本賽季場均要出手25.1次，投籃命中率為44.3%，三分命中率為35.9%，的確出手有點多，命中率也不算出色，但如果因此就產生哈登高得分時刷出來的錯覺，就大錯特錯了，在本賽季所有被當作進攻核心的球員中，哈登的進攻效率為123排名第一，要高於字母哥的118，哈登的真實命中率為62.9%，也要比字母哥的61.8%更加出色，這讓的效率和真實命中率，能是刷出來的嗎？

2、德拉蒙德很強

本賽季至今，綽號「庄神」的德拉蒙德場均依然可以貢獻17.7分16.6籃板2.9助攻1.72次搶斷和1.84次蓋帽，牢牢佔據著籃板榜榜首的位置，甚至數據已經可以和中鋒時代的那些頂級中鋒相媲美了，但是如果因此就認為庄神很強的話，那就是一個最大的錯覺了，別的不說，作為一名中鋒球員，庄神現今的得分手段還只有吃餅和二次進攻，真實命中率也只有56.8%，而在防守端，庄神的數據怎麼看都是頂級防守中鋒的數據，只不過庄神的數據，要遠遠高於在所能提供的貢獻，這也是為什麼庄神的防守數據出色，卻一直沒有入選最佳防守陣容的原因。

3、獨行俠擁有兩巨頭

隨著波神的傷愈復出，讓很多獨行俠的球迷，認為東契奇身邊終於有了一個出色的幫手，獨行俠隊也擁有了自己潛力無限的雙巨頭組合，不過在現在看來，這還只是一個錯覺，本賽季的波神，在參加的25場比賽中，場均可以貢獻17.2分9籃板2蓋帽的數據，但是命中率卻只有40.1%，三分命中率也低至33.6%，而且在比賽中，波神作為一名有著2米21身高出色運動能力的內線，有三分之一的出手在三分線外，即使偶爾進入籃下，也盡量避免和對手產生身體對抗，特別是在防守端，波神的場均兩次蓋帽，只不過是看起來很美罷了，對於球隊整體的防守貢獻，幾乎可以忽略不計。

4、恩比德進攻很強

還沒有進入NBA前，恩比德就被認為是「大夢」奧拉朱旺的接班人，而進入聯盟後的出色表現，更是被一大批專家和媒體認為是聯盟第一中鋒的不二人選，攻守兩端最均統治力的球員，特別是在他在比賽中偶爾展現出來的內線腳步，更加讓一批人認定了他會是下一個奧拉朱旺，只不過在本賽季看來，至少在進攻方面很強的恩比德這也只是一個錯覺，看看本賽季已經結束的比賽，只要是和對方球隊的頂級中鋒交手，恩比德進攻端的表現都會出現下滑。

和猛龍交手兩次，面對小加，恩比德一場0分一場比賽10分，和掘金交手兩次，面對約基奇一場比賽22分，一場比賽19分，不過命中率分別只有44.4%和35.3%，和爵士隊交手兩次，面對戈貝爾一場16分一場27分，不過命中率只有38.5%和31.2%。而恩比德的整體命中率，本賽季也不過只有46.1%，為什麼會出現這種情況，一是因為恩比德下肢力量不算出色，內線頂不動力量出色的防守，籃下腳步銜接也不夠嫻熟，二是因為恩比德雖然有外線三分能力，但是命中率還遠沒有達到能夠產生足夠威脅的地步，三是雖然恩比德的進攻手段不少，但卻沒有一項到爐火純青的，四是因為連續的傷病，讓恩比德的運動能力出現了不小的下滑，特別是為了加強對抗性和力量增重後，這點更加明顯。
