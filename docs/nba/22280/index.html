NBA被捧殺5位球員：過度吹捧毀2大天才，一場陰謀讓榜眼淪為水貨

2019-12-20 1,921條評論 2,859次閱讀 1371人點贊

對於那些年輕的天才籃球運動員來說，最害怕經歷的有兩種情況，一種就是被過分的指責，畢竟這會對年輕球員的自信心產生極大的影響，就像曾經的歐洲天才米利西奇，而另一種情況就是被過分的吹捧，這不僅讓球員很容易在吹捧中迷失自己，更可能因為達不到吹捧中的預期，被大量的媒體和球迷所指責嘲笑，這就是大家常說的捧殺，下面小編就給大家帶來NBA歷史上被捧殺的五位球員。

NO1-朗佐鮑爾

提到朗佐鮑爾，不得不提起他的父親「球爹」這位出色的話題製造者，就如同國內的網紅一樣製造者一樣，通過各種誇張的言論和過分的推崇，讓鮑爾成為2017選秀大會中人氣和熱度最高的球員，甚至蓋過了之前的狀元大熱富爾茨和約什傑克遜，這種過度吹捧的營銷方式，的確讓鮑爾在選秀大會中得到了成功，但是隨之而來的，則是聯盟大部分球員的敵視，有很多同位置的球員都在和鮑爾的第一次交手中，拿出了120%的實力，只為了教育教育這位17年的榜眼秀，而當鮑爾的表現沒有達到所吹捧的程度時，那隨之而來的，就是大部分媒體和球迷的質疑、嘲諷和指責，畢竟誰讓當年你爹把你吹的那麼高？其實單論天賦和實力的話，鮑爾完全有可能成為「下一個賈森基德」，只不過球爹的捧殺，讓他失去了成長的最佳環境。

NO2-梅奧

相比於球爹一個人的吹捧，當年的梅奧簡直是被全美媒體給捧上了天，全美第一高中生，詹姆斯和喬丹的結合體，下一個聯盟統治級的後衛，這些太過於誇張的吹捧，讓梅奧在這些吹捧中迷失了自己，NCAA時毫無進步，依然在吃著高中時期積攢下的老本，再加上沒有在繼續增長的身高和不算頂級的運動能力，讓梅奧大學時期的表現已經出現了下滑，而在08年選秀大會中被灰熊隊以探花簽選中後，第一個賽季表現出色的梅奧更是膨脹到認為他要比球隊老大蓋伊更加出色，他才更適合成為球隊的老大，但是隨著托尼阿倫的到來，淪為替補的梅奧不僅數據和效率出現了下滑，更是出現了打架和服用禁藥的醜聞，這也讓灰熊隊最終對梅奧失去了信心，而時候不管是在小牛還是在雄鹿，梅奧再也沒有打出新秀賽季的表現，更是因為再次服用禁藥，被徹底關上了NBA的大門。

NO3-范霍恩

如果沒有被過分捧殺的話，按照范霍恩的實力和數據，完全可以成為一名出色的二當家球員，只不過因為「紅衣主教」奧爾巴赫製造的一場陰謀，將范霍恩推倒了全美媒體的面前，在97年選秀大會之前，奧爾巴赫就將鄧肯當作了凱爾特人隊延續輝煌的希望，但是作為狀元的不二人選，只有探花簽的凱爾特人根本就不可能得到鄧肯，在這種情況下，老狐狸奧爾巴赫動用了他在聯盟和媒體圈的龐大人脈，將范霍恩吹捧成為下一個「拉里伯德」，未來的聯盟統治者，企圖改變前面兩支球隊的目標，而經過了媒體的過度吹捧，范霍恩的熱度甚至超過了鄧肯，不過奧爾巴赫的如意算盤卻失策了，馬刺依然堅決選擇了鄧肯，而范霍恩，卻不是下一個拉里伯德，反而因為之前過度的吹捧讓他偏軟的球風，災難般的防守和無法勝任核心的位置，成為了球迷和媒體諷刺的關鍵，更是被認為是一名水貨榜眼，卻忽略了巔峰期的范霍恩也是一位場均能砍下21.8分8.5籃板的出色球員。

NO4-特爾菲爾

9歲時的特爾菲爾，已經成為了紐約街球界最出色的球員之一，而在高中時期他更是聲名鵲起，成為了和詹姆斯齊名的天才少年，甚至連說唱天王JAY-Z，著名導演斯派克李都是他的球迷，這也讓他成為了全美所吹捧的對象，在高中畢業之後，相信自己實力的特爾菲爾並沒有選擇大學而是選擇參加了04年的NBA選秀，但是特爾菲爾卻忽視了高中聯賽和街球聯賽是兩種天差地別的比賽，他的對手也要遠比他之前所面對的對手更強，這也讓特爾菲爾在比賽中暴露出了終結能力差和傳球視野窄的缺陷，而且和大多數年少成名的天才一樣，他控制不了自己的情緒，隨時隨地的都在會球場上發泄自己的不滿，根本就沒考慮過比賽的形勢，最終成為了聯盟中的流浪漢。

NO5-比斯利

大學時期的比斯利堪稱人型兇器，在天賦和實力方面，絲毫不遜色於安東尼和杜蘭特這兩位歷史超巨級別的鋒線，特別是在運動能力方面，比斯利甚至比安東尼和杜蘭特還要出色，這也讓比斯利在美國本土受到了極大的關注，被很多媒體和專家認為是下一個聯盟巨星，哪怕是按部就班的成長，進入全明星也只是時間問題，但是媒體和專家的吹捧並沒有讓比斯利將這些讚美變成前進的動力，反而讓比斯利的內心極度膨脹，養成了天老大他老二的性格，讓比斯利進入聯盟之後，不僅沒有成為聯盟中的巨星，反而是場內場外負面新聞不斷，最終淪落到無隊敢要的地步。
