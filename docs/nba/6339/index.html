厲害！創造主場最強開局，放棄隊史最佳得分，快船四巨頭110分

2019-12-02 0條評論 16次閱讀 0人點贊

在北京時間12月2號的NBA常規賽中，主力球星全員出陣的快船隊，他們在主場斯台普斯球館150：125大勝奇才隊，在上一場輸給馬刺隊之後，快船隊球員將自己的怒火轉移到奇才隊身上，最終以一場史詩級別的常規賽比分擊敗奇才。本場比賽快船隊四巨頭火力全開，首發陣容中的倫納德21投14中，全場豪取34分6籃板3助攻，另一位首發巨頭保羅喬治21投10中得到31分8籃板1助攻。

至於替補陣容裡面的兩位巨頭槌員，路威此役10投6中貢獻22分1籃板8助攻，哈雷爾15投9中收穫23分15籃板1助攻。因此倫納德、保羅喬治、路威、哈雷爾這4大主力球員，他們這場比賽合計入賬整整110分，幾乎快趕上了奇才隊的全隊得分。當然，快船隊其他球員也都是實力不俗，最終快船隊合計拿到單場150分，輕輕鬆鬆簡簡單單獲得了最後的比賽勝利。

其實這場比賽快船原本可以追平或者打破隊史最佳得分，因為在目前的快船隊史上面，這支球隊的NBA單場最高分是152分，本場比賽距離這一隊史紀錄只差2分。而在比賽的最後時刻，快船隊獲得了最後的進攻機會，當時比賽時間還剩下20多秒，快船隊完全能夠發起一次進攻。如果快船隊2分出手命中，那麼他們將追平隊史單場得分。如果快船隊能夠投中外線三分球，那麼快船隊將正式超越此前的單場152分。可是最後關頭快船隊球員卻放棄進攻，從而沒有追平或超越隊史紀錄。

不過雖然快船隊和隊史單場得分紀錄擦肩而過，但他們還是成功創造了一項隊史全新數據紀錄。由於快船隊主場贏下比賽，在19-20賽季至今，快船隊的主場戰績已經來到12勝1負，在過去的13場快船隊主場常規賽裡面，快船隊僅僅只輸給雄鹿隊，剩下的12場比賽快船隊全部守住自己的主場。而主場12勝1負的常規賽開局戰績，這也是目前為止快船隊的NBA常規賽最佳主場開局，在隊史榜單中留下了濃墨重彩的戰績紀錄！
