干翻湖人擊敗雄鹿！誰還敢在他們面前稱老大？繼續補強瞄準籃板王

2019-12-17 440條評論 1,562次閱讀 559人點贊

誰會想到，命運跟雄鹿隊開了一個玩笑。

雄鹿隊打出一波18連勝，創造球隊隊史第二長連勝戰績的時候，本以為能夠將連勝紀錄繼續拉長的情況下，卻倒在了對陣獨行俠隊的比賽當中。當所有人都把目光投向了雄鹿隊跟湖人隊的巔峰大戰，獨行俠隊卻成為攪局者，雄鹿隊18連勝被獨行俠隊終結掉，創造連勝紀錄的腳步被無情地逼停。

獨行俠隊本賽季的戰績相當出色，順利地從重建之路走上了正軌，擺脫了後諾維茨基時代多年不進季後賽的弱勢地位。本賽季從廢墟中成長起來的獨行俠隊不再是魚腩球隊，甚至於他們還成為「連勝球隊的殺手」，在結束雄鹿隊18連勝之前，還親手終結了湖人隊十連勝的戰績。

這似乎不只是「巧合「二字就能夠解釋清楚的。

字母哥空砍

獨行俠隊固然不是一支完美的球隊，他們不但會被強隊快船隊痛扁，還在本賽季兩次輸給尼克斯隊，他們的表現總是有些起伏不定。然而獨行俠隊卻是不能夠被輕視的球隊，任哪支球隊也沒有絕對把握說一定可以戰勝獨行俠隊，原因在哪裡呢？獨行俠隊的陣容有深度並且執行力很強。

里克-卡萊爾接受採訪時表示，「這就是為什麼在這個聯盟，你需要控衛的陣容深度。雄鹿隊也在經歷類似的事情，他們的埃里克-布萊德索受傷了，但他們還有喬治-希爾，以及一個可以解決這方面的體系，因為他們有揚尼斯-阿德托昆博和克里斯-米德爾頓這樣的組織前鋒。」

獨行俠隊也是如此，東契奇因傷不能夠出戰，但是杰倫-布朗森首發效果不錯。而且獨行俠隊保持著整體性，他們的角色球員雖然談不上頂級，卻有著很強的韌性。他們可能不會在每場比賽中都表現出自身實力，但若是應對不上他們的強度的話，他們總是能夠做出讓人瞠目結舌的事情。

波爾津吉斯表現出色

儘管獨行俠隊高居西部第三位，但是球隊清楚自身處在什麼位置上面，他們還沒到喊出爭冠口號的時候。馬克-庫班多年的經營，使得獨行俠隊有了一個穩定的基礎架構，東契奇、波爾津吉斯承載著球隊未來五年甚至十年的希望。在他們兩個人的周圍搭配角色球員，然而球隊還需要變得更好一些。

ESPN記者沃神近日就透露，獨行俠隊正在物色一名大個子球員，在波爾津吉斯身邊鎮守內線。沃神還表示。獨行俠隊正在尋找的球員類型，就是像哈雷爾或德拉蒙德這樣的強硬派內線。

我們都知道，馬克-庫班一直以來都比較偏矮強硬內線，從之前的埃里克-丹皮爾、泰森-錢德勒到「魔獸」-霍華德、德安德烈-喬丹等等，都在一定時間裡效力過或者被追逐過。儘管波爾津吉斯能夠護框也能夠封蓋，但他更適合出現在四號位，若是能夠為他搭配一位藍領球員，在一定程度上可以提升波爾津吉斯的作用。

這也就能夠理解，獨行俠隊為何在內線進行補強。

德拉蒙德前景不明

哈雷爾跟安德烈-德拉蒙德當然不是一種球員，但是他們的確都有著強硬的一面，尤其是哈雷爾的強硬一直都是快船隊比賽中的重要武器之一。不過相對來說，快船隊交易哈雷爾的可能性比較小，這是因為哈雷爾的效率比較高，快船隊不會錯失這樣一位替補席上的悍將。

唯一讓快船隊感到擔憂的是哈雷爾的到期合同，他現在的合同只有600萬美元年薪，下一份合同勢必要大漲，快船隊是否能夠承擔得起？其實這個問題也不存在太大的疑問，只要快船隊能夠在這個賽季在爭冠過程中處在有利位置，快船隊是不會吝嗇奢侈稅支出來保留爭冠陣容的。

德拉蒙德卻是一個更為可能的交易對象，這位籃板王的處境比較尷尬，媒體普遍認為，德拉蒙德很難拿到頂薪合同。NBA記者詹姆斯-愛德華茲三世稱，活塞方面目前正在考慮如何處理德拉蒙德的續約問題，將其交易或放他以自由身離開都是可能的選項，一切都要看活塞隊是否能在本賽季扭轉不利局勢。

​若是如此，獨行俠隊並不是沒有機會得到這位籃板王。

東契奇、波爾津吉斯以及德拉蒙德聯手的話，獨行俠隊的實力是否能夠得到更大提升呢？
