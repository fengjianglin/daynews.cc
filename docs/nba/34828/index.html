德國戰車：NBA偉大的大前鋒—諾天王(德克·諾維茨基)

2019-12-24 2,757條評論 3,695次閱讀 1347人點贊

德克·諾維茨基（Dirk Nowitzki），1978年6月19日出生於德國維爾茨堡，德國職業籃球運動員，司職大前鋒，綽號「諾天王」/「德國戰車」，現任FIBA球員委員會主席，是公認的繼拉里·伯德之後頂尖的白人球星。

1998年NBA選秀，德克·諾維茨基在第一輪第九順位被NBA密爾沃基雄鹿隊選中，不久被交易至達拉斯小牛隊（現達拉斯獨行俠隊）並效力至今。2007年，諾維茨基帶領小牛隊打出隊史67勝15負戰績，並榮膺常規賽MVP。2011年，諾維茨基領銜的小牛隊以4比2擊敗熱火奪得球隊首個NBA總冠軍，同時當選總決賽最有價值球員（FMVP）。

2019年3月19日，達拉斯獨行俠隊主場對陣新奧爾良鵜鶘隊的比賽中，獨行俠前鋒德克·諾維茨基出場3分鐘，已經得到4分。在得到這4分之後，諾維茨基職業生涯常規賽總得分已經達到31420，超越威爾特·張伯倫升至NBA歷史第6位。

2019年4月8日，獨行俠今天客場對陣灰熊，德克·諾維茨基出場7分鐘，2投1中，已經得到2分3籃板1助攻1蓋帽。加上這3個防守籃板，諾維茨基職業生涯防守籃板數達到10002個，突破10000大關。

2018年12月27日，獨行俠122-119戰勝鵜鶘，德克·諾維茨基出場12分鐘，6投2中，得到7分1籃板1助攻2封蓋。算上本場比賽，諾維茨基職業生涯常規賽出場數已經達到1477次，超越卡爾·馬龍，排名升至NBA歷史第4位。並且憑藉這場勝利，諾維茨基常規賽生涯勝利場數達到了900場，排名歷史第六。

2018年3月31日，獨行俠以92-93不敵森林狼，諾維茨基本場比賽首發上場，這是他本賽季出戰的第75場比賽。據統計，諾維茨基超越了卡里姆-阿卜杜爾-賈巴爾和羅伯特·帕里什（均為74場），成為NBA史上在生涯第20個賽季出場數最多的球員。

2018年3月18日，獨行俠客場挑戰籃網，諾維茨基首發出戰。加上本場比賽，諾維茨基生涯常規賽出場數達到1463場，超越凱文·加內特（1462場），獨享歷史第5位。

2018年3月7日，獨行俠主場以118-107戰勝掘金，諾維茨基出戰22分鐘，11投6中得到17分4籃板1助攻。加上這4個籃板，諾維茨基生涯籃板球總數達到11259個，超越多爾夫·謝伊斯（11256個），升至歷史第27位。諾維茨基本賽季三分命中總數也來到了116個，超越傑森·基德，位列39歲以上球員單賽季三分命中數第一。

2018年3月1日，獨行俠主場迎戰雷霆。比賽進行到第二節時，諾維茨基得到了個人第7分，使得自己職業生涯總得分來到了31,000分。諾維茨基成為了NBA史上第6位獲得這一成就的球員。

2018年3月5日，獨行俠主場迎戰鵜鶘，諾維茨基上半場出戰11分鐘，5投5中得到15分1籃板。加上這5個運動戰進球，諾維茨基生涯運動戰進球數達到10977個，超越埃爾文·海耶斯（10976個），升至歷史第8位。

2019年4月10日，獨行俠主場戰勝太陽，德克·諾維茨基的三分命中數超越了喬·約翰遜，升至歷史第11位。同時這也是諾維茨基職業生涯最後一個主場比賽。

2019年4月10日，在完成職業生涯最後一個主場比賽的賽後，獨行俠老闆馬克·庫班承諾將為德克·諾維茨基建造雕像並退役其41號球衣。次日，諾維茨基在完成職業生涯謝幕戰後宣布退役。同年9月14日，FIBA（國際籃聯）官方宣布德克·諾維茨基當選FIBA球員委員會主席，任期為2019-2023年。

德國戰車的運球雖然並不精巧但非常穩健，他的投籃–無論是中投還是遠射都是精英水準，即便把所有位置上的球員都拉過來也沒幾個人敢說比他更准。德克改變了比賽，改變了這個位置的定義，他是一個異類，一個偉大的異類。 （《Dime》雜誌評）

諾維茨基是聯盟歷史上進攻技術最全面，高產又高效的內線球員之一。諾維茨基是最偉大的投手之一：靈活的腳步，2米13的身高，無比柔和的投籃手感，他能在對抗中很好的控制身體，後仰投籃技術應用得更像得分後衛。 （新浪體育評）

德克·諾維茨基對比賽有著極高的熱情，這也使得他很難離開賽場。他是一個傳奇，一個活著的傳奇。他所要做的就是鞏固他的名人堂地位，他還將在歷史得分榜上繼續上升。 （文斯·卡特評）

德克·諾維茨基是籃球運動歷史上偉大的運動員之一。德克·諾維茨基在打籃球的技術和品質上超越了很多人。德克·諾維茨基打球的方式改變了籃球運動，7英尺高的大個子都模仿他的打球方式，也改變了大前鋒這個位置在籃球運動中的定義。（獨行俠隊主教練卡萊爾評）

只有蒂姆·鄧肯，科比·布萊恩特，勒布朗·詹姆斯，德克·諾維茨基這四個球員，能在我那個年代取得成功。（邁克爾喬丹評）
