三連敗！76人開始交易吧，這三份上億的合同性價比太低

天天要聞
2019-12-21

今天，獨行俠在客場113-98大勝76人，76人遭遇了三連敗，恩比德得到了33分17籃板的大號兩雙，其他球員隱身了，西蒙斯只有12分6籃板8助攻，理查德森10分6籃板4助攻，霍福德9分，哈里斯7分。要知道獨行俠沒有東契奇啊，讓對手得到了117分的高分，防守太差，進攻還不行。

76人遭遇三連敗，12月16日76人在客場89-109不敵籃網，這一場比賽恩比德沒有打，西蒙斯20分，哈里斯17分，沒有恩比德鐵定輸球，但籃網也是缺兵少將。12月19日76人在主場104-108不敵熱火隊，恩比德22分19籃板的大號兩雙，西蒙斯17分6籃板6助攻，理查德森17分，哈里斯20分，霍福德9分。輸掉的這三場比賽，76人都是防不住，自己打不進，對於聯防沒有破解的辦法。

76人有四份上億的合同，恩比德、西蒙斯、哈里斯、霍福德，恩比德肯定是物有所值，但其他三人的性價比太低。

西蒙斯是5年1.7億美元，今天這場比賽在追分時刻他沒有上場，因為他沒有投籃能力。賽季至今，西蒙斯場均只有14.4分6.7籃板8.1助攻，投籃沒有什麼進步，關鍵比賽就會被針對，季後賽更是如此。西蒙斯是一位很好的組織者和防守者，但想進一步太難了，76人想奪冠，西蒙斯必須能投籃。

哈里斯5年1.8億美元，今天這場比賽只得到7分，他的表現時好時壞，賽季至今場均19.7分6.8籃板3.2助攻，但是三分命中率只有慘淡的31.7%，哈里斯是一位不錯的角色球員，但1.8億的合同就太大了。哈里斯與西蒙斯打不到一塊，兩個人都需要球權，而且哈里斯不是那種有大心臟氣質的球員，沒有巴特勒那種領袖氣質。

霍福德是真的老了，上一場比賽得9分，今天又只有9分，他是這支球隊經驗最足的球員，卻沒有站出來幫助球隊。霍福德已經33歲了，4年1.09億美元的合同才生效，賽季至今場均只有13.4分6.8籃板3.9助攻，數據中規中矩，但命中率不高，性價比太低。

今天這場比賽76人只命中了10記三分球，命中率只有29%，「死亡五大」，場上空間太小，外線又沒有穩定的投射，不說打進總決賽，在東部就過不了，交易市場已經開啟，趕緊交易吧。
