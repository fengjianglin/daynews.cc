聯盟第一人詹姆斯，也曾打過替補！你知道當時發生了什麼嗎？

2019-12-01 0條評論 13次閱讀 0人點贊

在科比退役之後，詹姆斯便接過了聯盟現役第一人的頭銜，儘管他已經快要35歲了，但詹姆斯仍然保持著出色的競技狀態，他的職業態度一直以來也都是被人所稱讚的。很難想像一名球員可以向詹姆斯這樣將狀態保持如此長的時間，他幾乎從進入聯盟後就開始一直處於聯盟內頂尖球員的行列。

當初詹姆斯頂著狀元秀的名頭強勢進入聯盟，他也得到了極高的關注度，因為自己出色的身體素質和極強的對抗能力，詹姆斯當時也是被球隊視作是未來領袖的頭號人選。而其實在詹姆斯的職業生涯中，他還曾打過一次替補，這是為什麼呢？

這就要從詹姆斯早年第一次在騎士效力時說起了，聯盟內有名的蓬蓬頭瓦萊喬那是在騎士隊內是內線的重要球員，他的籃板能力十分出色。在後來詹姆斯率領球隊衝擊總冠軍時，瓦萊喬也是球隊中重要的一員。而瓦萊喬當時在新秀合同將要結束的那個賽季因為續約問題與騎士的高層發生了分歧。瓦萊喬認為自己的表現配得上一份聯盟中等水平薪資的合同，但騎士隊的管理層並不太願意給予他等價的薪水，因此瓦萊喬與騎士一直沒有簽下新的合約。

而瓦萊喬不願續約的舉動招致了克利夫蘭騎士球迷的反感，他們認為瓦萊喬沒有歸屬感，每當瓦萊喬登場時都會遭到球迷的噓聲，這樣的待遇也讓瓦萊喬萌生了離隊的想法。而當時的詹姆斯已經是球隊的當家球星，在出現這樣的情況時，詹姆斯義不容辭的站了出來，幫助瓦萊喬解決問題。他主動要求替補，讓瓦萊喬首發。在自己傷愈歸來的那場比賽中等到第六分鐘才出場，好讓自己出場時的歡呼聲足以蓋住瓦萊喬下場時的噓聲，並以此來表示瓦萊喬對於球隊的重要性。

詹姆斯的這一舉動讓瓦萊喬十分感動，這個來自巴西的大男孩也是做出了留隊的決定。之後也是成為了騎士衝擊總冠軍時詹姆斯的得力助手。但後來在詹姆斯離隊前往熱火之前，兩人被曝出不和的傳聞，詹姆斯當時在比賽中指責瓦萊喬假摔，兩人也是一度火藥味十足。後來瓦萊喬也是加盟了勇士，與詹姆斯的關係恐怕再也沒有了緩和的可能，但當時詹姆斯的做法還是十分值得稱讚的。
