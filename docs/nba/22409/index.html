30+7又轟30+10！雷霆妥協，這位高管評價保羅真狠，讓人唏噓不已

2019-12-19 1,248條評論 2,870次閱讀 1346人點贊

在這個夏天克里斯保羅被交易到雷霆，原本以為球隊要重建了。沒想到新賽季保羅真的是詮釋了什麼是巨星影響力。在最近雷霆高歌猛進，已經殺進了西部第7的位子。戰績為12勝14負，不得不說保羅真的是打臉了所有人。

很多人說保羅拿的是一份垃圾合同，34歲的老將拿4年1.6億美金？這原本就不合理，但這不恰恰是說明保羅的影響力巨大？

在34歲的年齡，還能拿到一份頂薪，現在聯盟中除了勒布朗詹姆斯，也就是保羅了。而且保羅太好用了，本賽季甚至有重返巔峰的味道。

本賽季保羅身體還非常健康，他場均31.3分鐘可以砍下16.2分4.6板6.3助攻1.7搶斷，命中率為46.6%，三分命中率為38.1%。場均出手只有11.4次，保羅完全不耽誤雷霆的重建，也不耽誤年輕人亞歷山大。

基本上在關鍵時刻，球都是給亞歷山大，實在沒轍了才給保羅。要是讓保羅主導球權，現在雷霆戰績可能會變得更好。在進入12月份以來，保羅狀態爆棚。2次砍下30分，1次20分。帶隊取得了5勝3負的戰績。

30分+30分+20分，保羅已經慢慢找回了狀態。要是保羅不是拿4年1.6億，4000萬年薪。而是拿4年1.2億，3000萬年薪，這各大強隊早搶著要了，保羅實力可是在康利之上。只是可惜的是保羅的表現並沒有讓雷霆更加容易交易他，他們計劃還是落空了。

根據美媒《露天看台》記者Eric Pincus報道，他在最近採訪了一位東部分區球隊的管理層人員，詢問了雷霆在交易市場的狀況。這位管理層談到了保羅加里納利和亞當斯，雷霆的三位巨頭。

這位東部的管理層對這三人做出了評價，其中保羅的價值被極度貶低。這位管理層說道：「沒有人想要CP3（克里斯-保羅）。亞當斯的合同也很糟糕。或許只有加里納利可能是唯一能讓雷霆更好交易，甚至能得到回報的球員！」

從昔日的第一控衛，到現在無人問津，保羅真是令人感到唏噓。此外《露天看台》還報道了關於熱火想要保羅的事件。不過雙方基本上談崩，一位球員經紀人透露，熱火雖然想要保羅，但他們向雷霆提出了的報價非常不切實際。基本上，保羅不可能加盟熱火聯手巴特勒。

另外雷霆也選擇妥協了，東部高管表示。最近雷霆在戰績上的表現，現在已經不能確定。雷霆是準備大甩賣重建呢？還是他們真的想進季後賽？

可以說保羅帶隊連續獲勝，已經動搖了管理層的心。他們看起來在無法交易保羅的情況下，會繼續去衝擊季後賽，選擇妥協。這對於保羅倒是一個好消息，怎麼樣保羅都要進季後賽吧？沒有保羅的季後賽還叫季後賽嗎？
