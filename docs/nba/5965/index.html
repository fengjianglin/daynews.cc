湖人領跑，西部現3大陣營！4強爭上半區，7隊大混戰力拚前八三席

2019-12-08 0條評論 16次閱讀 0人點贊

北京時間12月8日，NBA常規賽繼續進行。隨著爵士主場擊敗灰熊，今日的五場激戰全部結束。獨行俠大勝鵜鶘之後，以一波五連勝鞏固了西部第二的位置。

目前，狂野的西部展現出三大陣營。湖人強勢領跑，獨行俠、掘金和快船緊隨其後，四大豪強在上半區展開激烈的競爭。另外西部第六的爵士至第十二的國王，七支球隊的勝場差距並不大，每支球隊都是季後賽的有力爭奪者，七隊將展開大混戰力拚前八三席。另外，西部的第三陣營灰熊、鵜鶘和勇士處境依舊掙扎，基本很難再對身前的球隊造成衝擊。

隨著比賽的深入，化學反應日漸濃烈的獨行俠愈發勢不可擋，一路衝到了西部第二的位置。今日面對鵜鶘又送上了一場46分的大勝，延續著火熱的狀態。反觀西部老大湖人，在昨日大勝開拓者之後，又收穫了一波三連勝，穩居西部第一。

另外兩支豪強掘金和快船在近來狀態出現起伏，分別輸給了東部強敵凱爾特人和雄鹿。不過，以兩隊的整體實力和陣容厚度，依舊具備著足夠的衝擊力。湖人、獨行俠、掘金和快船四強將繼續在上半區展開慘烈的競爭。

爵士在近期狀態不佳，遭遇了一波三連敗，球隊的戰績一路下滑。不過，本場面對實力明顯在自身之下的灰熊，球隊打出了一場漂亮的團隊籃球，全場7人得分上雙，止住連敗。

值得一提的是，排在爵士身後的森林狼和太陽最近一段時間起伏都非常大，反倒是雷霆、馬刺和開拓者等開局低迷的球隊開始發力，幾支球隊之間的差距也在進一步的縮小。七支球隊將展開大混戰，全力爭搶第六至第八的三個季後賽席位。

相比開拓者和馬刺開始起勢，勇士和鵜鶘的狀態依舊沒有好轉。目前西部也是出現了三大陣營，隨著比賽的深入，各大陣營的比拼將更加慘烈。
