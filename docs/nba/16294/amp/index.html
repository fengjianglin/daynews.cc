里弗斯：當球員和球隊好聚好散會得到歡迎 萊昂納德的做法很職業

天天要聞
2019-12-14

北京時間12月12日，快船客場挑戰猛龍，而兩天前的一場比賽（12月10日）快船剛從印第安納全身而退。這兩場比賽，正好也是球隊兩大核心保羅-喬治和科懷-萊昂納德曾經效力過的球隊。不過兩人受到的待遇可以說是天壤之別，離隊後已經是第三次造訪印城的喬治繼續遭到全場噓聲，而幫助猛龍歷史性奪冠的萊昂納德，領取戒指的同時收穫了歡呼。

「當你和球隊好聚好散的時候，你會得到你想要的（歡迎）。」快船主帥里弗斯告訴The Athletic。「科懷從來沒有說過他打算長留（多倫多）。他一次也沒有說過這句話。所以，他並沒有誤導任何人。他為了球隊竭盡心力。他被交易到了多倫多而不曾抱怨過一個字。他做的任何事情都挑不出毛病，所以你能做的事情有且只有歡呼。」

被交易到猛龍的萊昂納德盡全力幫助球隊奪冠，至此他對這座城市已經毫無虧錢，因此在離開的時候也並沒有受到球迷們的為難。快船主帥里弗斯說，看看詹姆斯在2016年奪冠前和奪冠後回到克利夫蘭時球迷們的反應，就能夠理解喬治和萊昂納德所得到的待遇為何不同。總而言之，只要職業球員盡心儘力，球迷們自然懂得成全。

「就像勒布朗加盟騎士的時候，第二次。他們為他的回歸而歡呼。他們贏得了總冠軍。」里弗斯說。「這就是生意。而科懷，他做的一切都非常的職業。他從來沒有評論過多倫多。他確實享受這座城市。他愛那裡的人們。而另一面是，他回家了。這不像是他加盟什麼死敵之類的事情。他去了西部，他回家了，這是他過去幾年都一直試圖去完成的事情。我想球迷們表現的很克制很得體，因為我想他們都能理解這一點。」

里弗斯還表示，萊昂納德雖然只在多倫多待了一年時間，但這一年時間卻在他整個職業生涯中都是相當厚重的一頁。不僅是猛龍球迷會記得這歷史性一冠，萊昂納德這個看起來沒什麼情感的機器人，也會記得許多與隊友的點點滴滴。

「它是永恆的。」里弗斯說。「我們建立了這樣的聯結。我們一同經歷了這段旅程，所有的勝利，所有的高峰低谷，所有的得到和失去，從9月，到最後交易發生，到這樣那樣的事情。所有的這些都交織在一起，你記得這一點。你記得那段日子和隊友一塊出去遊玩，你會記得那些幸運，記得你從傷病中恢復，記得所有的那些倒霉事。這對我們兩人而言都是如此。」

冠軍很偉大，但是冠軍不代表一切。萊昂納德已經在多倫多證明了自己，他未來還需要用更多冠軍證明自己，里弗斯說當球員由於任何原因請求被交易的時候，球迷們要學會從職業的角度去看這個問題，因為職業球員的青春實在是太寶貴了，不該浪費。

「這很艱難，但是這就是我們招募他們的原因，」里弗斯告訴The Athletic關於追逐冠軍的故事。「有些偉大的球員還沒有贏得什麼東西。為了不讓自己的年華被虛擲，如果你沒有偉大的隊友，你很難在聯盟里贏下什麼東西。你就是不能。偉大的球員通常只有三到四次機會去贏得總冠軍。我想勒布朗上賽季學到了這一點。就好像，『X，我那麼牛X，但是我沒有些牛X的隊友還是不行。』所以他們才會出走，讓自己的身邊多一些可靠的隊友。」

回首自己的球員時代，里弗斯從未贏得哪怕一次總冠軍，要麼被更好的球隊擊敗，要麼被厄運打倒。但是，作為主教練他為綠凱贏得隊史第17次冠軍（2008年），確實讓他感到充實而圓滿。但在此後，他僅在2010年打進過總決賽一次，並且執教快船時從未邁過二輪大關。

里弗斯說，歷史上奪冠的球隊不多，而優秀的球隊如過江之鯽，且往往轉瞬即逝。這也就是為何當代球員更願意自己掌握自己的命運，不浪費寶貴的時間。里弗斯率領的綠衫軍就是三位超巨生涯晚期聯手的典範，這也間接導致了詹姆斯的出走，而里弗斯也見證了杜蘭特因雷霆管理層運營的失敗出走勇士的案例，對於巨星改換門庭的決定他比任何人理解都深刻。

「每年，我都祈禱自己能夠贏得總冠軍，」里弗斯說。「在我13年的職業生涯里（作為球員），我有過兩次機會。一次是在尼克斯，一次是在馬刺。在13年里，我有過兩次真正的機會，這就是我跟我的隊員們說的。」

「老兄，我一直回想當年雷霆進總決賽的那時候，當時他們輸給了熱火，我記得傑夫-范甘迪當時說：好吧，他們未來五年還會捲土重來的。然而雷霆再也沒有回來過。在這個聯盟里，你不能將成功視作輕而易舉。你真的不可以。」
