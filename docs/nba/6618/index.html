美媒排名：NBA季後賽最佳2人組合！詹韋第三，格林真值頂薪

2019-11-28 0條評論 16次閱讀 0人點贊

美媒《82games》通過各支NBA球隊單賽季季後賽在場勝負值，排列出NBA前9名2人組合！詹韋第三，能猜對的老球迷有多少？

如果你曾經對庫里的NBA歷史地位有所疑慮，不妨看一眼這份榜單！在美媒《82games》統計中，擁有庫里的組合在近20年9對超級雙人組合中獨佔4席！

16-17賽季庫里和湯普森，季後賽同時在場433分鐘凈勝對手178分排名第9；庫里和杜蘭特同時在場的471分鐘同樣凈勝對手178分；

而NBA14-15賽季，庫里和格林同時在場688分鐘凈勝對手180分排在榜單中第7名！

第6名，湖人OK組合能夠打出紫金王朝，靠的就是實力！00-01賽季，這是科比和奧尼爾最甜蜜的一年，OK組合在季後賽大殺4防，2人同時在場的620分鐘，湖人共計贏了對手187分。

第5名，這也是一個意外點！凱爾特人三巨頭中，最高效的不是球隊老大皮爾斯，而是加內特和雷阿倫的組合！

07-08賽季NBA季後賽，兩人在場時要幫助球隊贏了204分，這對組合對綠軍的空間加成已經毋庸置疑！

在上一篇文章中我就提到，03-04賽季的活塞首發5虎是NBA近20年季後賽，球隊勝負值最高的首發陣容！

而其中又以無球後衛漢密爾頓和本-華萊士最為重要！兩人在830分鐘里，球隊贏了對手216分，位列榜單中第4。

千呼萬喚始出來，第三名，詹韋組合！11-12賽季是2人的巔峰，也是詹姆斯首次奪得總冠軍的年份！詹姆斯和韋德同時在場的799分鐘里，球隊勝負值為+217分！

格林值不值頂薪？看完這份數據你就不會意外了！16-17賽季是格林和庫里組合的巔峰！勇士季後賽同時在場的455分鐘里，球隊勝負值達到了217分！

這一份數據，或許會對接下來庫里復出後，沒有湯普森的勇士隊帶去一點安慰吧？

在近20年當中，馬刺GDP組合在早期算是不太起眼的一對組合！但當他們的生涯結束之後回過頭看看，又完全是另一番風景！

在馬刺三巨頭中，又以鄧肯和吉諾比利的雙人組最為高效！02-03賽季季後賽，鄧肯和馬努同時在場的568分鐘里，凈勝對手238分位列NBA第一！這份排名，各位球迷意外嗎？
