集體崩盤！西部3大黑馬，合計19連敗 雙狀元徹底掉隊，保羅躺贏

2019-12-23 2,071條評論 3,565次閱讀 1390人點贊

北京時間12月22日，2019-20賽季NBA常規賽繼續進行，在這個比賽日中，西部的三大黑馬繼續頹勢——森林狼輸給開拓者、國王不敵灰熊、太陽也是主場輸球。至此這三隊都是延續連敗的腳步，三隊合計19連敗，直接導致14勝14負的雷霆，以不小的優勢穩穩的位居前八，保羅真可謂是「躺贏」。

這個比賽日的最後一場爭奪，是森林狼客場對陣開拓者，此前已經遭遇9連敗的森林狼，此役繼續以單核陣容出戰，唐斯缺陣之下他們在開局陷入落後，易地再戰持續追擊，末節無限接近翻盤，但最終仍然功虧一簣。

輸掉這場比賽之後，森林狼遭遇10連敗，目前是聯盟最長的連敗，戰績滑落到了10勝18負，已經快「追上」西部倒數第3的灰熊，這支賽季初一度排在西部第一的球隊，宣告徹底掉隊。

在他們身前的太陽也是「不甘落後」，賽季初同樣是無比驚艷，如今亦是陷入連敗的泥潭，6連敗之下戰績已經來到了11勝18負，縱使馬刺慘敗快船，依舊能夠排在第10壓制太陽。

國王在迎回核心福克斯之後，卻意外的遭遇3連敗，這支一度追擊至西部第7的球隊，如今再度落後到了前八追趕者的角色。在這個比賽日擊敗了森林狼的開拓者，則是穩固了自己的優勢。

至此，西部賽季初讓人眼前一亮的四大黑馬，除了獨行俠已經全部掉隊——森林狼10連敗、太陽6連敗、國王3連敗。三隊合計19連敗，遭遇了崩盤式的表現。

反觀賽季初低迷只能靠「虐菜」贏球的雷霆，卻已經將戰績提升到了14勝14負。這支一度以重建為目標的球隊，打出了五成的戰績。但客觀上說，按照往年的競爭環境，五成勝率在西部著實不夠看。但隨著西部樂透區集體低迷，也讓保羅「被迫」穩坐前八，想在短期內掉下來還真挺不容易。
