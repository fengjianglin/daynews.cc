伊戈達拉有新消息，樂福有意開拓者，雷霆或有交易，火箭心儀1人

2019-12-11 2,540條評論 4,241次閱讀 1147人點贊

伊戈達拉有新消息，樂福有意開拓者，雷霆或有交易，火箭心儀1人

文/董體育

北京時間12月11日，NBA安排了4場比賽均為開打，但是NBA消息卻是琳琅滿目，一起來看看今天NBA最新消息。灰熊隊伊戈達拉終於又有了新消息，湖人一直盼望灰熊能與老將安德烈-伊戈達拉完成買斷，但看起來他們的願望很難成真。據The Ringer記者Kevin O』Connor報道，灰熊仍然一心想要通過交易送走一哥，甚至願意為此接納一下長期合同。勇士今年夏天通過先簽後換送走凱文-杜蘭特，因此觸發了硬工資帽的限制，不得不用一支未來首輪簽作為代價，將伊戈達拉送到灰熊。因為灰熊致力於重建，和一哥達成一致，他將不必到球隊報道，只需要在家中等待交易的發生。看看對伊戈達拉心儀的湖人隊和快船隊接下來會有什麼大動作吧！

騎士隊樂福一直處於交易的風口浪尖上，據The Ringer記者Kevin O’Connor報道，如果騎士決定交易凱文-勒夫，這位經驗豐富的大個子更願意為開拓者效力。勒夫就在波特蘭城外的奧斯威戈湖附近長大，因此如果加盟開拓者，將為他提供一個回家打球的機會。不過，勒夫3年1.2億美元大合同剛剛開始生效，這一數字將讓騎士很難找到一個能與薪水相匹配的交易夥伴。本賽季，勒夫場均得到15.6分和10.5個籃板。O’Connor還提到，開拓者可能也會在交易市場上向雷霆前鋒達尼羅-加里納利展開追求，在羅德尼-胡德受傷賽季報銷之後，可三可四的加里納利同樣能為波特蘭的鋒線提供更好的選擇。一旦樂福加盟開拓者隊，他們將會有利拉德+麥科勒姆+安東尼+樂福的超級組合，屆時西部競爭會更加激烈。

雷霆隊已經擠進了西部前八的位置，但是根據The Ringer記者Kevin O』Connor報道，新賽季開始至今，雷霆總共打出了11勝12負的表現，排名西部第七。儘管季後賽仍有希望，但球隊仍然願意交易隊中包括克里斯-保羅、達尼羅-加里納利、史蒂文-亞當斯在內的幾名老將。雷霆目前的薪金總額仍在奢侈稅線以上，不過據O』Connor報道，逃脫奢侈稅的處罰並非當務之急，他們本賽季的繳稅金額不多，而接下來兩個賽季因為一些合同到期，將不用再交稅。事實上，雷霆在這些潛在的交易中，想要收穫更多未來資產，為此他們願意接受長期合同。不知道雷霆隊本賽季的計划到底是什麼，是交易之後的重建還是努力拚進季後賽呢？

火箭隊心儀1個球員也得到了曝光，昨天The Athletic記者Shams Charania也有報道稱，火箭願意犧牲未來的選秀權去交易鋒線球員。根據The Ringer記者Kevin O』Connor報道，火箭隊對森林狼前鋒羅伯特-考文頓興趣十分濃厚。隨著詹姆斯-哈登再次打出MVP級別的表現，火箭迫切想要抓住機會衝擊總冠軍，因此希望對球隊相對薄弱的側翼進行補強。而考文頓，作為聯盟中最頂尖的3D球員，也吸引了他們的注意。今年擁有雙MVP威少+哈登之後，火箭隊非常強勢，鋒線成為了最大的軟肋，拭目以待他們的交易動作吧！
