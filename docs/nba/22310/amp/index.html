退役後還有錢領的球星：狼王每年500萬，湖人造2垃圾合同

天天要聞
2019-12-20

眾所周知，NBA球星們的收入很高。對於現役球員來說，哪怕只是底薪的邊緣人，年薪也在200萬美金左右，絕對的人生贏家。大部分球員，退役的時候也都履行完了合同，所以退休後是沒有工資可領的。不過也有一些球星，他們的合同令管理層頭疼，因為退役多年之後，球隊仍然要付不少的薪水，甚至還要佔用薪金空間。

史密斯被裁5年還有錢拿

約什史密斯曾是聯盟的勁爆扣將，巔峰時期的他實力不俗，也是一位明星球員。不過14-15賽季，巔峰不再的史密斯就被活塞隊裁掉了。有趣的是，史密斯被裁5年仍有錢拿，活塞今年仍要支付533萬美金給他。當年史密斯3年4000萬美金的薪水，活塞使用了延伸條款，足足分了6年支付，史密斯也早已退役了。

籃網還在為德隆買單

德隆曾經是和保羅齊名的控衛，他的巔峰在爵士，最成功的時期則是籃網，當年他簽訂了5年9850萬美金的大合同。但傷病過後狀態下滑明顯，最終籃網為了躲避奢侈稅，選擇裁掉德隆。籃網同樣使用了延伸條款，雖然現在德隆已經淡出NBA2年時間，但籃網仍然要支付他547萬美金的薪水，算是為當年買單了。

湖人造2垃圾合同

科比大傷之後，湖人算是爛了幾年。自由市場大牌無數，但湖人始終是簽不到。正所謂病急亂投醫，當年湖人用大價錢簽下了莫茲戈夫和魯爾鄧。莫茲戈夫的合同是4年6400萬美金，後來搭上拉塞爾才送走了這個垃圾合同，輾轉黃蜂魔術之後，莫茲戈夫被球隊裁掉。未來3年，魔術都要支付莫茲戈夫557萬美金的薪水，非常的悲催。

至於魯爾鄧，則是爛在了手裡。2018年湖人硬著頭皮買斷了魯爾鄧的合同，同樣的，湖人未來3年都要支付魯爾鄧499萬美金的薪水。對於志在爭冠的湖人來說，這500薪金空間太礙事了。

加內特退役7年還有錢拿

巔峰時期的加內特，是聯盟超級巨星，他當過多年的聯盟薪金王。凱爾特人時期，加內特幫助球隊奪冠，並且再次續約。後來加內特放棄了交易否決權，被交易到了籃網。但他也和凱爾特人達成協議，退役之後綠軍將分7年返還他3500萬美金，每年達到500萬。所以退役7年，加內特都有錢領，實在是令人羨慕。
