沒你我一樣也行！單核威少創史詩紀錄，沒喬丹的皮蓬帶隊西部第3

2019-12-10 1,885條評論 3,611次閱讀 1814人點贊

在NBA中球員流動是非常正常的現象，當然僅限於普通球員，明星球員每次換東家都會鬧出不小的動靜。如果隊里還有另外一個明星球員的話，那就更熱鬧了，畢竟在鏡頭前不知道說了多少次的兄弟情，突然間就走了，擱誰誰尷尬。處理這種尷尬的情況最好的辦法就是拿出出色的表現來證明他的錯誤。今天我們就來盤點一下，那些離開搭檔也能優秀的球星。

雷霆的分崩離析如果真要追根溯源的話，那說的就多了，但是杜蘭特的不告而別仍是俄城球迷不可接受的，不過好在他們還有威斯布魯克。對於威少而言，杜蘭特的離開打擊實在太大了，兩個人在俄城八年，沒想到竟會是這個結局。心有不甘的威少在接下來的賽季里幾乎是用透支職業生涯的方式在打球，同時還創造了連續賽季三雙的史前成就，這應該就是威少最好的發泄了。

如果OK組合不出問題，也許他們還能多稱霸幾年，不過倆人迥異的性格，能維持到三連冠已經是奇蹟了。送走鯊魚後，在帶隊成績上掙扎許久的科比終於又捧起了奧布萊恩杯，而且還是兩連冠。用科比自己的話說，我終於比沙克多一個冠軍了，這下他沒話說了吧。

納什二次加盟太陽後，就立刻將太陽帶到了總冠軍爭奪者行列，同時自己也拿到了兩連MVP。在納什的光環下，許多人都忽視了斯塔德邁爾，覺得他就是一個普通的扣將，離開納什什麼也做不到。沒有想到的是在之後加盟尼克斯後，小斯的表現一點都不差，獨自帶隊都能把尼克斯帶到季後賽行列，徹底讓質疑他的人閉上嘴巴。

儘管巴克利經常吐槽皮蓬拿的六個總冠軍都是老二冠軍，啥用沒有，但真讓皮蓬帶隊的話，其實一點都不差。在喬丹退役的賽季，皮蓬獨自帶隊也拿到了55勝，名列西部第三。季後賽打到東部半決賽惜敗尼克斯。這個成績怎麼看都算很好的了，要知道沒有皮蓬的時候，喬丹差不多也是這個成績。

這張圖應該可以算得上見者傷心聞者流淚了，那個賽季開拓者的成績其實並不算差，但首發四人都像約好了一樣同時離隊，就剩下了利拉德一個。接下來的賽季，球迷們本來以為開拓者要奔著狀元簽去了，誰都沒想到在實力急劇下降的情況下，利拉德生生把球隊帶進了季後賽，只能說利指導實在是太硬了。
