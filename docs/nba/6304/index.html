昔日聯盟第一控衛胖成球，缺陣一年卻拿上億合同，球迷：史上最爛

2019-12-03 0條評論 18次閱讀 0人點贊

作為世界籃球水平最高的殿堂，NBA聯盟是很多職業球員夢想的所在，在這裡聚集著世界上對頂尖的籃球運動員，他們用一場場精彩表現來回饋現場的球迷朋友們。在NBA聯盟如果發揮出色，那麼你就會得到球隊的重視，並將你樹立為核心球員，甚至還會為你送上一份價值上億元的合同。但是有些球員在拿到大合同之後因為一些傷病的原因，導致無法上場打球，令球迷們非常惋惜，但是接下來這位球員就是因為上述的原因遭到了球迷們的謾罵，這是怎麼一回事？

他曾經被譽為聯盟第一控衛，因出色的身體素質而冠絕聯盟，他就是沃爾。作為2010年NBA選秀的狀元，在出道時被很多奇才球迷寄予厚望，認為他能夠幫助球隊重返季後賽的舞台，並且還有機會爭奪總冠軍。

事實也如球迷們所預想的那樣，奇才在沃爾的帶領下一步一步地走上正軌，是東部一支不容小覷的隊伍，並且還是季後賽的常客。本來一切事情都在往好的地方發展，可突如其來的一場意外讓球隊產生了翻天覆地的變化。

在2018年12月的一場常規賽當中，沃爾不幸在賽場上跟腱撕裂導致賽季報銷，而且至今還沒有一個準確的康復時間。現如今的沃爾因傷閑賦在家已經有一年的時間，整個人都已經沒有了往日的風采，身材更是比以往胖了很多，完全出乎球迷們的預料。

在沃爾受傷之前，奇才隊才剛剛和他達成了4年1.7億美元的超級大合同，而現在受傷最大的莫過於球隊管理層了，即使他復出之後也沒有可能回到巔峰時期。不少的球迷也是對此紛紛調侃：「這絕對是NBA歷史上最爛的合同，保羅的合同都比他值」。對此，各位網友是什麼樣的看法，歡迎大家在下面的評論區留下你們的看法，感謝大家的支持！
