心疼！剛進NBA的19歲新秀，就被要求場均20+，直接憋出抑鬱症

天天要聞
2019-11-30

23分，10個籃板，2次封蓋，這是英格拉姆被交易到鵜鶘之後，首次面對老東家湖人取得的成績單，雖然數據看似不錯，但效率上，英格拉姆只有21投4中，三分球11投僅2中，最後關頭，當英格拉姆再次三分偏出之後，鵜鶘也就此失去翻盤的希望。

還沒等比賽哨響，英格拉姆就已經轉身低頭離開，21次出手啊，看得出來，他太想證明自己了，尤其是在老東家面前，因為他和湖人之間的感情，遠勝於目前湖人隊中的任何一員。

英格拉姆是湖人的鐵杆球迷，是科比的超級迷弟，只要有時間，他就會關注偶像的任何一場比賽，哪怕是理髮期間，英格拉姆也會注視偶像的一舉一動，更幸運的是，2016年選秀大會上，手握榜眼簽的湖人，恰好選中了這位才華橫溢的年輕人，而那時，科比剛剛打完自己職業生涯的最後一場。

儘管湖人隊中還有2015年的榜眼拉塞爾，但幾乎所有湖人球迷都認為，英格拉姆才是科比真正的接班人，雖然前者無論是球風，還是身材，都神似杜蘭特。

不僅是球迷，湖人的管理層，也將科比的更衣室柜子，繼承給了英格拉姆，可見他在紫金軍團中的地位，究竟有多麼高。

但令人遺憾的是，英格拉姆的成長沒有達到預期，甚至連最佳新秀1陣都沒混上，很快，尖刻的批評接踵而來，是啊，湖人畢竟是聯盟里的豪門，那裡的聚光燈比任何球隊都要耀眼，足以讓英格拉姆的弱點無處遁形，哪怕英格拉姆在第2個賽季，打出場均16+5+3的數據，也依然挽回不了他在湖人球迷心中的形象，似乎只要不是場均20+，你就不配披上紫金戰袍一樣。

面對巨大的壓力和源源不斷的交易流言，英格拉姆的自信心備受打擊，一度患上了跟德羅贊一樣的抑鬱症，2018年他捲入喬治的交易緋聞中，2019年他又被爆出交換戴維斯，儘管湖人一再強調英格拉姆是非賣品，但在王朝面前，他還是被犧牲了，雖然英格拉姆效力湖人時間並不長，但我想他當時心情，應該是和德羅贊一樣的。

本想終老湖人，傳承偶像的意志，可那裡的人，根本不給你成長的時間，哪怕是3年都嫌長。

本賽季，英格拉姆雖然沒能在湖人面前證明自己，但他過去13場的表現，應該征服不少湖人球迷了吧，場均25.9分，7.4個籃板和4.0次助攻，命中率48.1%，三分球命中率42.4%，是聯盟里僅有的6位，場均至少25+7+4的球員，也是這6人中，僅有的2位2項投籃命中率都超過40%的球員。

英格拉姆不是水貨，不是軟蛋，更可怕的是，他今年才22歲，上限還遠不止於此，也許湖人贏得了現在，但長遠看，他真的失去了一位「曼巴精神」的傳承人。
