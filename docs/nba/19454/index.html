12連敗！鵜鶘隊史最恥辱紀錄 373分！勇士臨時工輸出居首

2019-12-17 600條評論 1,668次閱讀 189人點贊

北京時間12月16日，2019-20賽季NBA常規賽繼續進行。

鵜鶘主場119-130不敵魔術，數據顯示，本場輸球後，鵜鶘已經遭遇12連敗，而這也是他們隊史經歷的最長連敗。

事實上，本賽季開始之初，鵜鶘一度被看好有能力攪亂西部格局。畢竟，今夏，他們在濃眉哥的交易中得到了英格拉姆+球哥+哈特三大潛力新星，另外，他們還有今年的狀元錫安-威廉姆森坐鎮，霍樂迪、費沃斯、雷迪克等老兵也都是優質的即時戰力。

遺憾的是，賽季打到現在，由於傷病等原因，鵜鶘始終沒能湊齊最強陣容，近期，他們的士氣也受到了嚴重打擊，無論進攻，還是防守都成了一盤散沙。

同樣悲催的還有勇士，主場79-100不敵國王之後，勇士又遭遇了一波4連敗，如今，他們5勝23負，排在聯盟倒數第一。

不過，賽後，卻誕生了一項相當有趣的紀錄。

數據顯示，截至本場球，達米安-李與凱-鮑曼合計已砍下373分。由此，他們成為在NBA所有球隊中，雙向合同球員得分最高的隊伍，排名第二的馬賽克隊為93分，第三名熱火則得到91分。

坦白說，勇士目前的糟爛表現，與杜蘭特今夏的離開，與庫里、湯神的傷停有著直接的關係，而賽季打過四分之一，他們也基本擺出了一副要擺爛，到明年捲土重來的架勢。

不過，在連戰連敗之餘，勇士板凳席上卻多少收穫了一些希望。

有憂就有喜，在湖人客場與老鷹的比賽中，儘管，全隊遇到了不小的挑戰，但最終，湖人還是以101-96擊敗對手，拿到7連勝。

本場球之後，湖人又收穫了多項紀錄，首先，紫金軍已取得客場14連勝，目前，這個連勝場次排名NBA歷史第3，僅次於1971-72賽季的湖人（客場16連勝）和1994-95賽季的爵士（15連勝）。

另外，在過去15個客場比賽中，勒布朗-詹姆斯拿到403分，安東尼-戴維斯入賬401分，兩人得分都成功達到400+，至此，詹眉成為NBA歷史上首對在賽季前15個客場都得分400+的隊友。

不僅如此，本場，湖人全隊送出14次搶斷，追平了賽季最高紀錄，且本場，湖人出戰的10人全部都有搶斷入賬。

但此役，湖人也有負面數據，首先，他們本場三分球31中5，命中率為15.6%，創造球隊賽季新低。此外。湖人全隊出現22次失誤，創賽季新高，此前，紫金軍單場最多送出19次失誤。
