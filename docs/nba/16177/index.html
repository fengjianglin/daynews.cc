骨折來的太不是時候！大將錯過NBA榜首大戰，雄鹿被湖人搶走先機

2019-12-16 2,392條評論 4,912次閱讀 2174人點贊

12月15日，密爾沃基雄鹿隊125-108輕取騎士隊，獲得了18連勝。揚尼斯-阿德托昆博帶傷上陣砍下29分，雄鹿隊也24勝3負的戰績，排名NBA聯盟第一。而西部的湖人隊今天沒有比賽，他們目前比雄鹿隊少賽一場，以23勝3負的戰績排在聯盟第二。5天之後的12月20日，雄鹿隊將在主場迎接湖人隊的挑戰。這一場NBA的榜首大戰，誰贏誰就能確立聯盟第一的優勢。

一直以來，因為西部的湖人隊勢頭太盛，所以很少人注意到，東部的雄鹿隊戰績同樣非常出色。戰勝騎士隊比賽之後，雄鹿隊已經獲得了18連勝，這是本賽季NBA最長的連勝紀錄，也是雄鹿隊隊史第二長的連勝紀錄。雄鹿隊歷史最長的連勝，是1970-71賽季的20場，他們也在那個賽季獲得了NBA總冠軍。

不過，雖然今天的比賽雄鹿隊贏了，但他們卻收到了一個壞消息。球隊後衛埃里克-布萊索被確診為右側腓骨撕脫性骨折，將缺席未來兩周的比賽。布萊索是在上一場與灰熊隊的比賽中撞傷了右腿，當時他自己也沒當回事，但想不到卻發生了骨折。本賽季，布萊索26場比賽全部首發，場均能得到15分4.9籃板5.7助攻。他的缺陣，將會對雄鹿隊的備戰產生影響。而更糟糕的是，雄鹿隊正好在未來幾場遭遇幾支強隊的挑戰。

12月17日，雄鹿隊將在主場迎戰獨行俠隊。這本來是個難纏的對手，但因為盧卡-東契奇意外受傷，所以雄鹿隊拿下的把握應該比較大。但接下來的12月20日，面對湖人隊應該是一場硬仗。之前兩支球隊都只輸了3場球，雙方几乎是並列聯盟第一。誰先輸掉第4場，就將把優勢拱手讓人。而打完湖人隊，雄鹿隊還要接連迎戰步行者隊和76人隊。特別是76人隊，目前已經獲得5連勝，排名上升到東部第二。恩比德和西蒙斯可不願意隨隨便便輸給字母哥的。

當然了，布萊索骨折，最大的受益者還是洛杉磯湖人隊。湖人隊在12月份的比賽其實挺艱難的。因為第一個月的賽程太輕鬆，他們在接下來不得不還債了。12月份，湖人隊要挑戰多支聯盟強隊，還要連續去東部打客場。這種強度被湖人隊球迷稱為「魔鬼賽程」。果然，12月的第一場球，湖人隊就輸給了獨行俠隊，10連勝被終結。

這一次湖人隊的5場連續東部客場比賽，剛打到第2場。而12月20日挑戰雄鹿隊，是其中的最後一場。湖人隊的年齡偏大，詹姆斯馬上就要年滿35歲。安東尼-戴維斯最近也一直在遭受傷病的困擾。本來雄鹿隊以逸待勞，應該在競爭中佔盡優勢。但布萊索的意外受傷，恐怕會成為他們的隱患。從陣容打法上來講，湖人隊還是比較克制雄鹿隊的。雄鹿隊的進攻都是圍繞著字母哥為核心展開，不像湖人隊有詹姆斯和戴維斯兩個巨星，容錯性更高。字母哥雖強，但他更喜歡從外線持球突破發起進攻。本賽季的湖人隊卻是以防守立足的，要面對湖人隊詹姆斯和戴維斯的輪番防守，字母哥肯定不會打得特別舒服。

所以，缺少了布萊索這個戰術支點的雄鹿隊，要想主場戰勝士氣正盛的湖人隊，恐怕會非常困難。只能說，布萊索傷的不是時候啊。
