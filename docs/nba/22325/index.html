無東契奇獨行俠憾負綠凱！卡帥調整已趨於完美，綠軍三巨82分太強

2019-12-20 749條評論 2,010次閱讀 662人點贊

無東契奇獨行俠在客場爆冷了密爾沃基雄鹿隊，終結了雄鹿的18連勝，12月19日獨行俠將在主場迎戰東部第三的凱爾特人，無數的獨行俠球迷盼望著這支沒有東皇的獨行俠能繼續爆冷，天不隨人願啊，這場比賽獨行俠已經竭盡全力了，可還是差一點，最終綠凱以109-103擊敗獨行俠，沒有東契奇的獨行俠未能完成連續爆冷的奇蹟，卡萊爾已經充分的展現了自己的執教能力，波爾津吉斯與塞斯-庫里被打造成新的內外線雙核，這兩人又一同砍下了20+，不能怪球員們不給力，真的是對手太強大，綠凱的三巨頭得分全部上20+，卡萊爾真的儘力了。

終場前1分26秒，塞斯-庫里上籃命中分差還差著8分，沃克失誤小哈達威關鍵三分不中，卡萊爾並沒有放棄比賽，開始打犯規戰術，威廉姆斯兩罰全中，德朗-賴特強了一個兩分，隨後卡萊爾讓全隊進行全場緊逼致使杰倫-布朗失誤，波神6犯規離場布羅科夫登場就命中三分將分差縮小到5分，雖然繼續犯規沃克兩罰一中，布羅科夫製造了杰倫-布朗三分犯規，結果三罰兩中，只在最後時刻獨行俠已經咬的很緊，不過杰倫-布朗關鍵兩罰全中擬定勝局，若小哈達威那個三分能進，這場比賽勝負就不好說了，卡萊爾的臨場指揮與戰術調配已經相當出色，他儘力了。

這場比賽綠凱勝在三巨頭一同爆發，這跟對陣雄鹿不同，對陣雄鹿時獨行俠扼制出了除字母哥外的所有核心得分點，而這場比賽綠凱除了沃克外的杰倫-布朗、塔圖姆都沒能限制住。沃克全場18中9，三分10中5，罰球11中9，拿下32分5籃板3助攻1蓋帽，命中率50%，三分命中率50%；杰倫-布朗全場17中8，三分7中2，拿下26分9籃板1搶斷1蓋帽，命中率47.1%；塔圖姆全場23中10，三分10中2，拿下24分8籃板1助攻1蓋帽，這三人合力拿下82分22籃板，得分佔比全隊75.2%，綠凱三巨頭真的很出色，尤其是肯巴-沃克，一人撕裂了獨行俠的防守。

波爾津吉斯在沒有東契奇的三場比賽中都拿下20+，這場比賽他全場19中8，三分7中2，拿下23分12籃板2助攻1搶斷3蓋帽，命中率42.1%，他在場時凈勝6分的，可惜的是布倫森與小哈達威的進攻效率太差，布倫森13中4，拿下10分11助攻7籃板，小哈達威17中4，拿下13分3籃板，兩人合計30中8，若能多命中兩個球局勢都改變了，布倫森空有11次助攻，塞斯-庫里13中8，三分8中4，拿下20分4籃板2助攻也付出一切。

在最後的一分鐘啟用1分鐘都沒有打過的布羅科夫也力證了卡萊爾的神奇，布羅科夫登場一記三分，一次製造三分犯規，1分鐘里拿下5分，這就是一個主帥用人的決策能力，若此役取勝卡萊爾將被聯盟所盛讚，只可惜最終輸球了。從最近的三場比賽能看出缺少了東契奇的獨行俠最主要的問題還是進攻，東契奇是一個能場均拿下30+9+9的巨星，獨行俠的防守體系已經建立，主要問題就是進攻，卡萊爾就算再神奇，也無法每一場都幫助球員們打出高效的表現。
