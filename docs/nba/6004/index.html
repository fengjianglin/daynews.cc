湖人“漸入佳境”？我來澆點冷水

2019-12-08 0條評論 15次閱讀 0人點贊

拉施特《史集·成吉思汗傳記》提到：我們廝殺吧，勝者為汗！強者崇拜心理是人的通病，因為被強者征服也是一種榮譽。之所以喜歡NBA，也是從喜歡喬丹、喜歡詹姆斯開始的。然而當發現英雄從巔峰慢慢滑落，強者不再是聯盟中唯一，才發現喜歡漸深，已不滿足於對單場得分等數據的期待，看著支持的球隊，慢慢變好，漸入佳境也成為一種享受。

取得三連客的勝利之後，是否要學著杜德利發出：我們是不是還沒有碰過任何一支強隊啊？這是極不應該的。首先不可否認三客場勝利的含金量。丹佛掘金，NBA聯盟中海拔最高的主場球館；猶他爵士，想想當年的麥迪姚明難以逾越的季後賽剋星；開拓者，最近五場僅一負，氣勢正盛。再者，湖人的幾大「頑疾」似乎都得到了提升：1、三分球，最近兩場47%、48%的命中率，連生涯里往往被人放空的隆多，三分命中率都到了52%，驚嘆不已；2、防守效率，自布拉德利受傷後，湖人防守體系就下降了一個檔次，對手的三分線命中率飛升，是否還記得鵜鶘的三分雨，然而最近幾場的外線防守都做得很好，防守互換、防守補位都很到位。3、球隊第三得分點，缺少穩定的第三得分點，一直是讓湖人最為頭痛的事情，但是最近的比賽中，形成了群狼效應，總有幾位球員得分上雙，其他球員的得分效率也在提升。一切都顯得那麼完美。

但是，我來澆點冷水。第一點三分球的問題解決了嗎？並沒有，場均11個三分球，仍然位於聯盟中下游，第18位，對戰掘金，只投進了7個，對獨行俠7個。對內並沒有穩定的外線射手，最強射手鐵林依然不在狀態，場均1.5個三分，在當今的聯盟規則下，沒有三分投射能力，就打不出有層次的進攻體系。隆多的三分需要更堅決，更有侵略性，不止空位三分能力；格林需要找回三分球能力。

第二點防守效率真的提高了嗎？也沒有，除兩巨頭外，替補悍將卡魯索、霍華德，得分能力不足；首發波普、麥基的防守能力均在100名開外；開拓者進攻效率為109.7，今天在強防守的湖人面前拿了113分，形成反差的是，在與快船的比賽中只拿了97分，湖人防守質量並不高。或許布拉德利的回歸會讓防守效率重新提升。

第三點形成穩定的得分點？更是差強人意，庫茲馬並未形成強有力的得分輸出，與前一賽季的場均18分下滑太多；湖人替補席場均貢獻35.6分，排在聯盟第18位，與同城的快船更是不在一個檔次上。

最後，35歲的老漢與「玻璃人」體質的戴維斯，沒有負荷管理，背靠背、場場33分鐘以上，這樣真的好嗎？放上一張胡德受傷的圖片，引以自省。
