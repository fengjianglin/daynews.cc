好男人齊聚，盤點NBA歷史上前五位的三分神射

天天要聞
2019-12-17

據虎撲報道，在今日結束的一場常規賽中，湖人客場以101-96擊敗老鷹，豪取7連勝。根據NBA官方的統計，此役過後，詹姆斯職業生涯常規賽三分球命中數達到1788個，超越拉沙德-劉易斯（1787個）升至歷史第18位，排在詹姆斯前面的是JJ-雷迪克（1790個）。在當今的小球時代，球隊擁有優良的三分投手就變得格外重要，今天就來盤點一下NBA歷史上前五名的三分「神射手」。

1. 雷·阿倫

作為96黃金一代的成員，「君子雷」在他19個賽季中投中了2973個三分，2001年獲得NBA全明星三分球大賽冠軍，在2008年幫助凱爾特人奪冠，2013年幫助熱火奪冠，最為經典的是2013年NBA總決賽第六場，熱火主場2-3落後馬刺。在比賽常規時間結束前5.2秒，雷阿倫接到波什籃板，在身體失去平衡且有兩人防守下投中續命三分，成功將比賽拖入加時賽。

2. 雷吉·米勒

在長達18年賽季，共投進了2560個三分球，排名NBA歷史榜的第二位，曾經在比賽最後時刻的8.9秒連拿八分，帶領步行者神奇逆轉，後被球迷稱為「米勒時刻」。

3. 斯蒂芬·庫里

作為21世紀10年代偉大投射之一，2015年全明星三分球大賽冠軍的庫里，職業生涯一共投進了2492個三分球，正是庫里的出色手感也帶領勇士三奪總冠軍，現年31歲的他在剩下的職業生涯里，如果庫里保持健康，有理由相信他在這份榜單上的排名會繼續上升。

4. 凱爾·科沃爾

現效力於雄鹿的科沃爾在過去職業生涯里投進了2381個三分球，在2013~2014賽季創造了連續127場投進三分球的記錄，同年，科沃爾婉拒參加當賽季的全明星三分球大賽，這已經是科沃爾連續兩賽季拒絕三分大賽，事後科沃爾坦言他在全明星周末期間要陪伴愛妻和女兒，是聯盟出名的「好男人」。

5. 傑森·特里

外號「噴氣機」的傑森·特里在19年的職業生涯中投進了2282個三分球，特里經常作為球隊的「第六人」上場，幫助球隊在進攻端打開局面，在2008~2009賽季榮膺NBA年度最佳第六人，2011年賽季隨獨行俠隊獲得NBA總冠軍。
