08-09賽季韋德場均30.2分，MVP評選僅列第3，前兩名是誰？

2019-12-06 0條評論 19次閱讀 0人點贊

已經退役的NBA球星中，有人雖然獲得過很多個人榮譽，卻從來沒有染指總冠軍，典型代表是巴克利，有人雖然多次奪得總冠軍，但個人榮譽方面卻有缺陷，比如韋德。16年職業生涯，韋德3次奪得總冠軍，一次當選FMVP，一次當選全明星賽MVP，10次入選全明星先發，三次入選最佳防守二陣，兩次入選最佳防守一陣，唯獨沒有獲得過常規賽MVP。在MVP評選中，韋德曾兩次殺進前五，但最終都失之交臂，其中08-09賽季最為遺憾。

那一年，27歲的韋德正值巔峰，場均出戰38.6分鐘，可以得到30.2分5.0籃板7.5助攻，投籃命中率為49.1%，效率值高達30.4（職業生涯之最，僅次於詹姆斯的31.7），真實命中率達到57.4%。賽季結束後，韋德榮膺得分王，還入選了最佳一陣、最佳防守二陣，但在MVP評選中，最終屈居詹姆斯、科比之後，位列第三。

08-09賽季的詹姆斯，場均可得28.4分7.6籃板7.2助攻，投籃命中率和三分球命中率分別為48.9%、34.4%，效率值為31.7，真實命中率為59.1%。當賽季，詹姆斯第五次當選全明星先發，還分別入選最佳一陣、最佳防守一陣，在MVP評選中得票率高達37.3%，職業生涯首次當選MVP。

08-09賽季的科比，場均可得26.8分5.2籃板4.9助攻，投籃命中率和三分球命中率分別為46.7%、35.1%，效率值為24.4，真實命中率為54.5%。當賽季，科比第11次入選全明星先發，分別入選最佳一陣、最佳防守一陣，在MVP評選中得票率為22.2%，緊隨詹姆斯之後。

論個人表現，韋德比這兩位有過之而無不及，之所以惜敗，是因為球隊戰績欠佳。2008年夏天熱火用榜眼簽選中比斯利，將老邁的奧尼爾送到太陽換來馬里昂，賽季中期還從猛龍交易來狀態全面下滑的小奧尼爾，再加上哈斯勒姆和查爾莫斯，他們是韋德身邊最得力的幫手。比起科比身邊的加索爾、拜納姆、奧多姆、費舍爾，這陣容著實寒酸。即使這樣，韋德還是率領球隊拿到43勝39負，排在東部第五。而詹姆斯的騎士、科比的湖人分別是東西部第一。

就像吉諾比利2005年總決賽錯失FMVP一樣，韋德錯失08-09賽季的MVP，將被後人永遠當成憾事來說。對韋德本人來說，如果有那座MVP做點綴，他的職業生涯必定更加完美。可人生就是這樣，哪有那麼多完美之事？
