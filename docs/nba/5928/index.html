又是加時，昨天的比賽讓人餘味難忘

2019-12-09 0條評論 15次閱讀 0人點贊

隨著國王隊加時賽末段在一個回合內四次投球不中，馬刺艱難的贏下了這場勝利。各位猜猜此刻的更衣室里老酒鬼是斥責對於前場板的保護，還是表揚最後階段的防守呢？

在比賽的最後時間，攻防兩端的關鍵人物是，默里。

在比賽的最後階段，默里一次成功的搶斷後，接米爾斯傳球暴扣得手，一次和米神形成配合幫助米神搶斷，一次萬軍叢中取上將首級搏得一次跳球，一次擋拆中的中距離和造犯規為馬刺加時賽總共五分的得分中佔據了四分。與此同時，在加時賽開局階段，他連續兩次突破不成功，浪費兩個進攻回合。

默里的防守確實如同上上賽季一樣，給了我們非常多的驚喜，不管是繞前防守巴恩斯，還是防守時的距離判斷，建立在良好的防守習慣上的身體優勢，能夠幫助球隊非常多。可是他的進攻卻顯得比較尷尬。在他與阿德/坡頭擋拆時，由於自身沒有三分威脅，防守球員就把他當做了西蒙斯，縮在阿德面前，他只好出球，浪費掉六秒左右的進攻時間。雖然默里擁有比較不錯的傳球視野，例如最後從高位大範圍橫穿右側底角米爾斯的那一球，可是他也需要在自身突破有威脅的情況下，才能發揮這一特點。

看得出來，老酒鬼希望默里的進攻逐漸成為馬刺的常規選項，在保證效率的情況下，至少可以消耗一部分球權，像是上賽季掘金的默里在球隊進攻中的定位。默里如果想要成長起來，必須練習出三分球，讓自己的突破具有更大的威脅，發揮自己的傳球能力，讓球隊進攻變得更為多元。

昨天非常值得稱讚的是常規時間內老酒鬼的三分扳平戰術，以及球員的執行力。阿贊和阿德擋拆，隨後阿德外彈，阿贊選擇中距離跳投或者突破對方大個子或是給阿德創造了中投空間。這一套進攻選擇可以說是阿贊來了馬刺以後最為常用的擋拆配合。常見的選擇也是上述的三個。

因此沃頓的布置也顯得非常合理，在擋拆後，原本防守阿贊和阿德的兩個組都去撲向阿贊，防守貝里內利的防守人再補向阿德。——機會來了。

通過回放，能夠很清晰的看出來，這個戰術正是為了兒子的三分球而設立的。在兒子獲得空位之前，阿德的彈出立馬改成了為兒子做一個無球擋拆，擋得結結實實，阿贊的傳球給得舒服，兒子手起刀落，三分命中，扳平比分。

在保證執行力的情況下，這是兩個教練的過招，沃頓還是太年輕，被老酒鬼贏下一局。

阿贊三分五中三，甚至還有了三分被犯規，坡頭的護框依然鐵血。阿德的前場板真硬，米神的大膽的放掉防守人搶斷真狠。球隊經歷過和馬賽克球隊以及快船的大戰後，似乎有了新的精神面貌。

一切都在變好。
