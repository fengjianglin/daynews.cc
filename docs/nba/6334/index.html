志在奪冠的湖人，應是十全九美掩一瑕？獨行俠僅揭開一疵

2019-12-02 0條評論 17次閱讀 0人點贊

今天我們只看湖人在防守端的選擇問題，僅從一個問題看，當然還有很多問題，但問題總要一個一個的解決，才能達到重質保量的效果。

湖人敗給了獨行俠，其實從第一場的交手中，我們也可以看到，散漫的湖人其實和獨行俠一個水平上的，但是等你真正想和別人打了，對方的手感已經養成，就很難在壓制下去了。

本場湖人採取的防守策略顯然和打奇才一樣，那就是包夾持球人，但是東契奇畢竟是獨行俠的不二持球發起人，這一點和比爾是不同的，比爾在職業生涯的前7年主要是無球無主，沃爾掌握著奇才的主要控球時間。所以，他們也忌憚東契奇的出球能力，但問題就出現在這裡，包夾就應該果斷嚴密，包而不夾的最直接後果就是東契奇有足夠的體型和空間去支配球權，導致湖人大面積的漏防，以至於獨行俠上半場的三分球炸開了花，全靠三分球把差距給彌補回來。

一、包夾防守和包而不夾的形式防守

首先，小編要在這裡講一下，小編自己認為的包夾是什麼含義：包夾就是指防守人直接對持球的傳球路徑進行全面封鎖，然後如果持球人運球，極大地可能會導致失誤，如果持球人將球繼續放在手裡，防守人逼搶，要麼造成一次對方主動暫停，要麼造成持球人失誤，這是最好的包夾效果。當然，最好的防包夾效果就是持球人迅速後撤，然後擴大持球可操作空間，迅速出球。

我們看如下的幾張圖片，

我們首先看湖人給東契奇的包夾，這裡第一張東契奇失誤，戴維斯和卡魯索在上線很高的位置就給了東契奇包夾，給他上了身體，要知道東契奇處理球能力強，但身體還是差了些，在包夾的情況下，也只能被迫出球，所以他跳起來傳球。由於跳起傳球自身平衡不好掌握，再加上傳球受到防守人的干擾，所以球的速度並不快，因此弱側的格林能夠在鮑威爾不知覺得情況下，完成了搶斷，這一回合的防守成功，戴維斯和卡魯索的包夾是第一功臣，而格林的協防走位意識則是最終決定性因素。

我們再看第二張湖人給東契奇的包夾，這次進攻從上線位置發起，到了底線的位置，實行了包夾。這種包夾看似人多勢眾，但是實際上鳥用沒有，為什麼，我們看東契奇的身邊有四個穿白衣服的球員，而東契奇身上掛著幾個湖人的球員呢，一個也沒有，也就是說，湖人給了東契奇一個虛假的包夾，這個包夾位置不深，你若是不理他，他前面是隆多，直接硬吃，你若是去理會他，包夾根本沒有形成合圍，所以即便是4人包夾，湖人的包夾圈也實在是太大了，這就送給東契奇一次三分球的助攻。

再看，包而不夾給了東契奇很好的處理球空間，導致大空位，進攻從圖中「人」的位置發起，到了底線包而不夾，能夠將球很好的傳出來，最後出現大空位。

詹姆斯迅速從弱側包夾，由於空間都給堵住了，持球人只能選擇擊地傳球，但是擊地傳球速度慢，所以詹姆斯能夠包夾完成後，迅速歸位防守，完成積極防守。

二、包而不夾的後果就是後場籃板保護高度缺失

其實這個問題並不是湖人這一場才出現的問題，在過去的19場比賽中，我們如果留意會發現，對手的前場籃板和湖人的前場籃板差不多，最令小編影響深刻的是湖人客場對陣公牛，公牛208內線小溫德爾卡特，一個人搶暴了霍華德駐守的內線，當然這並不是霍華德一個人的錯。但是湖人的內線組合是如此的強大，再加上詹姆斯也是一位以防守籃板著稱的球員在，這是非常不應該的。湖人丟前場籃板，但是內線丟分並沒有那麼誇張，是因為湖人的內線球員機動性很好，所以他們在對方搶到前場籃板時，能迅速歸位，保護籃下，所以對方在湖人的籃下進攻效率並不高。

湖人在換防和包而不夾之中最大的失誤：對方將湖人的高度優勢全部拉倒外線，湖人內線只能有庫茲馬或詹姆斯來頂抗，即便是這二位能力再好，我們也要注意到，他們畢竟並不是真正的內線球員。小編印象最深刻的是，湖人和快船的第一場比賽，麥基和戴維斯被拉倒外線，內線臂展220的哈雷爾統治力十足。當然，湖人也利用這一點搶了對手很多的籃板，但湖人的志向是總冠軍，這種情況能出現嗎？對比爵士，218的戈貝爾和官方身高198的奧尼爾搭檔，也沒有後場籃板這般被瘋搶過。

三、借鑒爵士將是部分時段防守的提升

其實我們參考爵士，看他們是如何保護自己的禁區，保護自己的籃板。為什麼戈貝爾能夠帶著4個小個子，打出如此的穩定的防守表現。

首先我們要看一下爵士的四個外線球員，185的康利，191的米切爾，203的博格達諾維奇和198的奧尼爾，這四位的身高和湖人的四外線身高，對比一下，196的波普，198的格林，206的詹姆斯，遠超208的戴維斯。更高大、運動能力也強，可以說全面壓制，難道問題真的只出現麥基和戈貝爾的身上，其實不然，我們看爵士如何防守。如下圖：

爵士在防守端似乎一萬年也不變的防守策略就是，戈貝爾永遠不會上前去換防對方的小個子，這主要分兩種情況。一種情況是面對勇士和開拓者這種投射大隊，對手在打戈貝爾的擋拆時，如果防守持球人的球員被完全擋住了，這時候由於持球人有很強的的投射能力，所以施耐德教練要求戈貝爾必須上前，完成包夾，這是實打實的包夾，讓對方不得不停球，注意，這裡戈貝爾一定會用巨大的身形擋住持球人。至於順下的球員，則由弱側的爵士球員頂上，一旦持球人停球，戈貝爾會迅速回撤，再次回到籃下；這樣，即便是持球人沒有停球，由於防守持球人的球員已經到了能夠在側身貼防的位置，所以持球人是無法完成投籃的，要麼拋投，戈貝爾在空中等你，要麼就球傳出去，這就浪費了一次進攻的寶貴時間。另一種情況則是更簡單了，當持球人投射能力並不強時，戈貝爾並不會上前，他會呆在肘區的位置，讓防守持球人的球員要麼努力搶過，要麼從身後跟防，這樣持球人也無法完成正常投籃，至於說是空接內線，你以為戈貝爾的大長臂是白長的？

很多人以為這一點對哈登不起作用，但是要注意的是，哈登進聯盟時為198，19歲的年紀還在長高，而一直防守哈登的是191的盧比奧，在身高和體重上差了太多，所以哈登可以輕而易舉的將盧比奧頂開，不用顧忌盧比奧給他帶來的投籃影響。如果很難理解，那麼就想像，你在側身防守一個比你高比你胖的持球人，你若下手，他就造你犯規，你想要的伸手影響他投籃，他比你還高還胖，就這個身體的圍度就已經比你的臂展長，你怎麼去影響。所以這是很對的防守方式，只是爵士的外線太矮了，而哈登的進攻屬於典型的刺蝟，明明把你扎破出血了了，犯規的卻是防守球員。

鑒於湖人更高的體型，更強的運動能力，為什麼湖人不在某些時刻，嘗試這種防守方式呢？或許就是他們過於強調換防，過分看中己方球員的協防能力，過於看中球員大防小的一對一防守，最終導致球員位置模糊，若是對方抓住這一點使勁揍，比賽真是好看了。

我們看賴特持球進攻，霍華德處在一個能夠影響拋投的位置，也能夠迅速回到籃下搶籃板的有力區間，防守賴特的人一直緊緊貼著賴特，讓賴特沒有辦法正常投籃，只能選擇拋投或鑽內線，這樣就進入霍華德的防守空間。而格林從弱側回來，方式空切的鮑威爾，戴維斯則站在弱側兩個人位置的中間，選擇防守即將傳過來的球，波普在球的強側，所以他保持自己的位置，但是注意身後人的位置變化，多完美的防守陣型，太完美了。

小編沒有看下半場的比賽，因為實在是太困了，堅持不下去了，但是看了湖人在籃板上的缺失，真是痛心，江湖傳言：得籃板者得天下，lakers，且行且珍惜。
