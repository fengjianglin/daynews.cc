巴恩斯賽季新高30分 穆雷迷之選擇錯失扳平 國王逆轉掘金止連敗

2019-12-01 0條評論 17次閱讀 0人點贊

北京時間12月1日，NBA常規賽繼續進行，薩克拉門托國王主場迎戰丹佛掘金，國王下半場追回了17分劣勢，將比賽拖入加時並最終笑到最後，100-97逆轉擊敗掘金，終結了兩連敗。

比賽回顧：

掘金今日前往客場挑戰國王，客隊目前狀態正佳取得六連勝，排名也上升至西部第二，而主隊則遭遇兩連敗，暫列西部第九，此役巴格利繼續因傷缺戰。比賽上來經過短暫拉鋸後掘金突然發力，哈里斯手感火熱單節砍下18分幫助球隊建立起兩位數的領先，次節掘金繼續擴大優勢，半場結束時分差已經達到17分。易邊再戰國王發起迅猛反撲，分差一度被縮小到三分，但掘金及時調整重新拉開分差，末節國王重新追上比分，比賽懸念保持到最後，關鍵時刻穆雷絕殺球被霍姆斯封蓋，比賽進入加時，加時賽雙方依舊保持拉鋸，但穆雷最後時刻被防住沒能出手，最終國王在主場加時逆轉對手，終結後者六連勝。

技術統計：

國王方面，巴恩斯30分5個籃板，希爾德21分7個籃板，霍姆斯13分10個籃板；

掘金六連勝遭終結，哈里斯25分，穆雷15分6次助攻，巴頓14分12個籃板，約基奇7分10個籃板6次助攻。
