洛杉磯湖人隊：3個性價比交易來提升球隊實力，更好的爭奪總冠軍

2019-12-14 650條評論 2,215次閱讀 125人點贊

洛杉磯湖人隊在攻防兩端都做得很好。交易截止日之前，我們現在來聊聊三名老兵可以幫助湖人的地方。

湖人隊是總冠軍的有力爭奪者，他們擁有兩名最優秀的球員：勒布朗·詹姆斯和安東尼·戴維斯。他們得到了很好的角色球員。丹尼·格林，德懷特·霍華德，賈維爾·麥吉，肯塔維奧斯·卡德維爾-波普和其他人本賽季都打得很好。

但是每一支球隊都會遭遇逆境，湖人是否有能力應對？正如他們所有的角色球員所提到的那樣，他們是否還能有足夠深的輪換陣容和角色球員的發揮。

NBA是一個競技場，你不能操控比賽，你也不能重新來過，是一個能者居之的舞台。隨著勒布朗年齡的增長和對總冠軍的渴望，湖人不能承受任何一個失誤。

由於工資帽的問題，這支球隊只能去撿便宜了。所以我們聊一聊可能給湖人帶來貢獻的球員，都是關於老兵的最低合同，這可能會幫助湖人。它們都是低成本的，而且不會花費大量資產來購買。

勒布朗永遠不會像現在這樣一直保持著身體的巔峰，因為時間老人從來沒有被打敗過。因此，現在是時候佩林卡用盡一切辦法幫助實現這一目標了。

3.諾倫斯·諾爾

你可能會想，為什麼湖人需要再增加一個中鋒？他們已經有了霍華德和麥基這兩個優秀的球員，他們現在打得很好。

你這麼說也是對的。但是在這個聯盟中，傷病經常發生，霍華德和麥基在他們的職業生涯中也遇到過。增加像納倫斯·諾爾這樣的球員，可能會在球隊受傷的情況下有所幫助。

雖然不會有太多的上場時間，但是諾爾是全能的，如果需要的話，他可以在4號位打上幾分鐘。湖人可以找到方法把這三個人合併在一起，或者仍然保持一個人受傷兩個人的輪換。

諾爾在俄克拉何馬有一項不可交易條款。現在還不確定他是否會長期留在雷霆。如果普雷斯蒂和他談續約的事，而他不願意接受，他將不得不考慮將他交易，以獲得一些價值。

在這個賽季，他的數據有很大的增長，本賽季20場比賽場均7.5分。他的投籃命中率為64.8％，場均上場18.3分鐘。每晚蓋帽1.6次，他的防守看起來也很出色。

納倫斯-諾爾和塔倫·霍頓-塔克的交易對雙方都有利嗎？

個人理解是，湖人隊得到了一個他們可以在今年之後重新簽約的中鋒，這可以幫助他們現在贏球。雷霆得到了一個年輕的新秀，可以慢慢的成長和歷練。但是雷霆有時間發展，湖人沒有。

2.凱爾·奧奎因

這對76人和湖人來說都是一個小交易。但是對於兩支球隊來說都是可行的，因為他們的球員都有新的開始。一個簡單的賈里德·杜德利換凱爾·奧奎恩。

這一交易可以幫助76人得到了一些外線投籃機會，而湖人得到了一個好的更衣室球員和一個更好的輪換球員的存在。杜德利已經被慢慢被替換了輪換陣容。

杜德利目前場均上場7分鐘，在23場比賽中只有14場上場。奧奎恩場均上場9.2分鐘，而且他在DNP的13場比賽中也只打了9.2分鐘。

兩名球員交換位置，幫助一個團隊在他們需要更多幫助的領域。

兩支球隊都要等到12月15日才能達成協議。也許每個總經理都看到了交換DNP球員的好處，以幫助彼此更好地擴充球隊的實力。

1.亞歷克·伯克斯

亞歷克·伯克斯在過去的休賽期以一份老兵的最低合同與俄克拉荷馬雷霆隊簽約。球隊有拉塞爾·威斯布魯克和保羅·喬治。當時，伯克斯還想為一個有能力爭奪冠軍的球隊效力，爭取贏得一枚總冠軍戒指。喬治和威斯布魯克被交易了，雷霆給了伯克斯一個重新考慮的機會，他去了金州勇士隊。

勇士隊本賽季很快就退出了競爭。他們一次又一次的受傷，現在看起來他們註定要進入前三的選秀名單。展望未來，他們將更有可能成為交易的名單。

由於許多球員缺陣，這名老將在本賽季有了長足的進步。他已經證明了他仍然有能力為球隊得分。他將有機會證明，當勇士覺得最佳交易時機來臨，他可以作為一個比較重要的籌碼。

在本賽季的21場比賽中，伯克斯場均得到15分4.4個籃板和2.8次助攻，他的表現非常穩定。他的投籃命中率是42/35/87.8，湖人也需要這種穩定輸出的球員。

他的進攻指數是100.2，防守指數是109.6。這兩個數據都令人擔憂，但你知道勇士目前處於一個擺爛的賽季。在那支球隊中，並沒有很多球員的數據是正的，因此這並不具備太過重要的參考數據。

伯克斯將會是一個非常適合在湖人打球的球員。他將為他們提供穩固的板凳得分和一個精明的老將。
