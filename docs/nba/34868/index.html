聖誕大戰賽程出爐，除了洛杉磯德比，這場比賽也別錯過了

2019-12-24 2,278條評論 3,943次閱讀 810人點贊

再過不久就是今年的聖誕大戰了，但是比起往年聖誕大戰的各種巔峰對決，今年的聖誕大戰好像有些缺意思，黃金時間的比賽只有兩場，分別是9點湖人對快船和11點半掘金對鵜鶘，但是這幾支參加聖誕大戰的球隊除了雄鹿都不怎麼開心，那麼今年的聖誕大戰到底能打出怎樣的效果呢？我們不妨在這裡先前瞻分析一下。

首先來看九點的黃金檔洛杉磯德比之戰，而就在剛剛結束的湖人主場對陣掘金的比賽上，詹姆斯由於胸廓肌拉傷缺戰了比賽，而沒有了老詹的湖人可又一次感受到了什麼叫做絕望，主場狂輸24分可還行？這還是在濃眉貢獻了32分11籃板的情況下輸的分，這足以證明即使是有了安東尼戴維斯，缺少了老詹的湖人依舊很難贏球，所以即將到來的聖誕洛杉磯德比，老詹如果回不來的話，湖人將是苦不堪言，首先，這場比賽雖然是在湖人的主場，但畢竟兩隊共用一個球館，如果湖人表現的不好，那麼當天湖人可能會把自己的主場拱手讓給快船，所以如果湖人想要贏下這場球，老詹恐怕還真得回來，否則在聖誕大戰上主場被快船打花，那可就太難看了。

而除了洛杉磯德比以外，聖誕日的另外一場備受關注的比賽不是11點半的掘金對陣鵜鶘的比賽，而是3:30開始的東部雙強的對抗——76人對陣雄鹿。雖然這場比賽的直播時間對我們來說不太好，但結果依然牽動著很多球迷的心。作為東部的兩支豪強，雄鹿和76人目前的狀態可是天差地別，雄鹿目前憑藉著一波三連勝已達到聯盟第一的戰績，76人卻剛剛從三連敗的陰影中走出來暫列東部第五，但今年76人的豪華戰艦陣容卻讓很多球迷對新賽季倍感期待，所以此次聖誕大戰的東部強強對決，76人能否展現出自己的真正實力，也是非常值得期待的結果。

所以總的來看，今年的聖誕大戰值得關注的兩場比賽就是洛杉磯德比與東部兩強的對決，而這兩場比賽都有球隊面臨著尷尬的局面，想要證明實力的76人能否戰勝強大的雄鹿還不好說，詹姆斯若回歸不了，湖人應該如何守住主場的顏面更是讓人頭疼，所以即便是今年的聖誕大戰，沒有太多的強強對決，但是這兩場充滿矛盾的焦點之戰，也能夠讓球迷們在聖誕之日一飽眼福。
