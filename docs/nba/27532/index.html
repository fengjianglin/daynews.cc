送禮物也就算了，關鍵還是定製版！歐文真的應該向他好好學習一下

2019-12-22 154條評論 1,487次閱讀 152人點贊

新賽季以來，要說到哪支球隊的變化最大，在筆者的心目中，不是湖人、快船，更不是獨行俠、勇士，而是波士頓凱爾特人！因為和上個賽季相比，他們不但實力更加強悍了，更關鍵的是，他們更衣室的氛圍更加融洽了。而這一切的變化，都要歸功與肯巴-沃克的到來。

最近，肯巴-沃克提前為全隊送上了聖誕節禮物，每人一台XBOX。不過，送禮物也就算了，肯巴-沃克送出的還是定製版禮物，雖然每個人的禮物都一樣，但是，在大家的禮物盒上，都印著各自的球衣號碼，這一點，真的是太貼心了。

在收到老大哥的禮物之後，大家紛紛上前擁抱致意，肯巴-沃克更是笑得像個孩子一樣開心。不過，這樣的一份禮物對於海沃德來說，那真的是意義非凡，眾所周知，海沃德除了職業球員的身份之外，還是一個瘋狂的遊戲迷。在之前手部遭遇骨折之後，海沃德可以說是天天打遊戲，在迎來複出之後，他更是坦言道：「打遊戲讓我的手部傷勢恢復得更快了！」

但是，反觀凱爾特人隊上個賽季的更衣室，這其中的差距真的是太大了。歐文在上個賽季的表現，也是引來了不小的爭議風波，除了球隊的戰績之外，他們更衣室的氣氛也是相當糟糕。尤其是上賽季凱爾特人隊的羅齊爾，他對歐文的態度，可謂是非常不滿意的，場上不服氣，場下不理睬。無奈的是，在歐文離開沒幾天，羅齊爾就被凱爾特人隊交易走了。

​俗話說得好：「一個巴掌拍不響！」雖然羅齊爾可能不是唯一一個對歐文不滿意的，但他絕對是上賽季凱爾特人隊最不安分的球員。不過，現在的凱爾特人隊看起來好多了，雖然肯巴-沃克和海沃德都是球隊最年長的球員，但是這兩人對於領袖這一角色，似乎都不太執著，他們的角色更像是一位長者，僅此而已。

所以說，面對如今這般和諧的凱爾特人隊更衣室，想必歐文也會有些尷尬吧。不過，話又說回來，雖然歐文現在身處籃網隊，但是他在這方面真的應該向肯巴-沃克好好學習一下了。畢竟，他現在的處境其實也挺尷尬的，自己還處於傷停階段，但球隊卻越打越好。歐文倒不如趁著這段時間，好好把隊友之間的關係梳理一下，豈不是更好嗎？大家覺得呢？
