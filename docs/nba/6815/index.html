一支球隊有2個新秀進新秀榜前五，這隻能說明運氣來了擋都擋不住

2019-11-25 0條評論 16次閱讀 0人點贊

NBA球隊的崛起有很多種方法，像綠軍三巨頭和熱火三巨頭這樣，通過交易得到成熟的球星，然後直接就跨入到爭冠行列是少之又少。畢竟依靠這種模式進行進入爭冠行列，是來得快、去得也快。更多的球隊是選擇用自己培養的球星，去實現爭冠的夢想。比如馬刺培養了GDP組合，勇士培養出了庫里、湯普森和追夢格林。

想要依靠自己培養的球星來奪冠，那麼獲得足夠優秀的新秀將是關鍵中的關鍵。近日NBA官方公布了最新一期的新秀榜，不得不說灰熊隊確實選到了一個足以用來做建隊基石的賈-莫蘭特。最新一期的新秀榜中莫蘭特再次位列第一，莫蘭特在場均只出戰28分鐘的情況下，能夠貢獻19.1分3.1分6助攻，而且三分命中率還達到了41.9%。就像康利所說，莫蘭特更像一個經驗豐富的老將。

灰熊有一個莫蘭特就足以讓其瘋狂，如果把莫蘭特的上場時間設定在場均36分鐘，那麼他可以貢獻24.5分7.7助攻4板，這樣的表示絕對值得培養。不過，在最新一起的新秀榜中，灰熊還不少最幸運的球隊，因為熱火隊有兩個新秀進入了榜單前5，這隻能說明運氣來了擋都擋不住，也能說霉了幾年熱火終於要翻身了，畢竟他們曾付了聯盟最多的薪水，卻無法打進季後賽。

在榜單中，熱火隊的肯德里克-納恩高居第二，他是熱火隊內僅次於巴特勒的得分手，是熱火高居東部第三的重要一員。除了肯德里克-納恩之外，熱火隊的泰勒-希羅也排進了榜單第五，他是一個天生的投手，未來前途不可限量。相信熱火在兩個火爆新秀，以及阿德巴約和溫斯洛的帶領下，總有一天能成長到爭冠行列之中。
