創隊史近32季最差紀錄！馬刺場均失分115+ 5場丟130+防守沒救了

2019-12-23 56條評論 4,874次閱讀 1286人點贊

北京時間12月22日，馬刺不僅是輸球，更是在主場慘敗被快船瘋狂吊打，且全場比賽在主場被對手狂轟134分，如此形同虛設的防守猶如一盤散沙，馬刺的狼狽可見一斑，而如今這支馬刺根本不再是昔日那支令人熟悉的聖城戰隊。馬刺如今賽季場均失分達到115+，創造隊史最近32季最差紀錄，賽季28場比賽已經5次單場失分達到130+慘不忍睹。

馬刺在波波維奇執教下，他們是全聯盟最穩定的豪門戰隊，尤其是能夠根據風格不斷調整球隊風格，但防守一直是馬刺沒有丟掉的精髓。奈何，近兩季的馬刺已經不再是一支防守強隊，甚至是成為一支超級防守爛隊，本賽季的防守表現較之上賽季更加慘淡，也是馬刺陷入掙扎的關鍵。

要知道，馬刺在2017-18賽季當中，他們賽季場均失分為99.8分，將對手限制到場均得分不足100分，如此兇悍防守令人忌憚。只是上賽季馬刺的防守已經大幅下滑，他們場均失分已經飆升至110分，場均失分提升了超過10分，無疑是令人不敢相信之事。

只是本賽季馬刺在再一次令所有人震驚，一如他們在主場被快船打爆。全場四節比賽當中，馬刺除最後一場垃圾時間只丟24分外，其餘三節均是丟分超過30分，前三節分別是丟分33分、39分與38分，三節比賽已經被對手狂轟110分，輸球也就不足為奇。

馬刺在本場再度丟掉134分後，他們本賽季僅僅才打了28場比賽，已經有5場比賽失分達到130+。馬刺此前4場失分高達130+，分別是丟了135分、138分、132分與133分，而上賽季馬刺82場比賽有8場失分超過130+，本賽季還剩下54場比賽，馬刺已經5場失分達到130+，大幅刷新如此恥辱數據的概率極大。

馬刺如今場均失分達到115.7分，在全聯盟場均失分排名當中排在倒數第6位，僅優於奇才、鵜鶘、老鷹、森林狼、灰熊5支魚腩大隊，也是佐證馬刺的悲劇。

馬刺如今場均失分達到115+，創造隊史最近30年最差值，而他們隊史上一次更差的防守表現，還是32年前的1987-88賽季，當時馬刺場均失分達到118.5分，無疑馬刺想要衝刺季後賽席位，他們需要提升防守，否則很可能連續打進季後賽紀錄會中斷。
