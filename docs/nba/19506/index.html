提早倒數退役日子，盤點那些瀕臨被小球潮流淘汰的內線球員

2019-12-16 1,997條評論 4,368次閱讀 729人點贊

約翰-亨森

曾經ACC最佳防守球員，在密爾沃基雄鹿隊字母哥揚尼斯·阿德托昆博沒到來之前幾乎一人撐起雄鹿隊內線，兇悍的防守封蓋以及不講理的單手隔扣時常進入十佳球，可惜這種暴力的打法讓亨森沉重的雙膝傷痕纍纍逐漸喪失爆發力，再加上粗糙的低位進攻技巧以及跟不上擋拆節奏的步伐使這位曾經的千萬合同中鋒日漸掉價。雖然曾在2017-2018賽季開發底線三分球能力，可是有著字母哥、大洛佩茲的雄鹿隊早已沒有了他的位置，不久便被隨手轉給克利夫蘭騎士隊。本賽季，處於合同年的亨森佔據著克利夫蘭騎士隊973萬的空間，相信這也是騎士在去年願意接納他的原因，而亨森目前僅能貢獻5.1分2.6籃板的數據難以在今年夏天吸引其他隊伍為他提供一份保障合同。如果說亨森在全明星周末前後被正在追求重建的騎士隊里無故裁掉，也不會讓人覺得意外。

凱爾-奧奎因

奧奎因是聯盟比較好用的角色球員，在進攻端對籃板的控制、出色的對抗能力、高位策應及掩護能力都是聯盟中數一數二，可是他無解的三分砸框能力也是讓他的教練頭疼的地方，生涯至今只有22.9%的三分命中率拿出來在如今自由球員市場里真的比較嚇人，加上在防守端隨著年齡增長逐漸緩慢的腳步讓他顯得格外累贅，特別在外線上早已無法與後衛進行一對一。奧奎因本賽季只剩下203萬美刀的底薪合同，直至目前僅能為費城76人隊貢獻3.4分、3.6籃板，因此在明年夏天應該甚少球隊會選擇屆時才31歲的鋒線球員作為輪換，或者奧奎因會選擇打法與其相似的前NBA球員昆西·埃希到歐洲或者中國聯賽淘淘金已逐步奏響職業生涯尾聲。

戈爾吉-吉昂

為明尼蘇達森林狼隊效力了7個賽季的塞內加爾中鋒吉昂本賽季出場機會還算比較多且穩定，這主要是因為目前森林狼隊內四、五號位置人員不足，但吉昂暫時並沒有交出讓人滿意的成績單。本賽季至今吉昂場均為球隊上場12.4分鐘拿下4.4分4籃板1助攻，這數據遠低於其職業生涯的7.8分 6.1籃板1.4助攻。失去了籃下攻擊慾望吉昂，在三分線上同樣迷失了自己昔日的底角準星，本賽季暫時只有30%的三分球命中率一直在拖低球隊進攻效率。森林狼在2016年與吉昂簽下了4年6400萬美元的大合同，算上本賽季和下個賽季，吉昂還能在森林狼隊拿到3352萬美元的薪金，但對比其數據以及場上作用，估計又會被說是垃圾合同模板的又一傑作。雖然吉昂已經觸到其天賦上限，但不在其他方面為球隊作出貢獻的話，黯然淡出NBA只是遲早的事。

伊安·馬辛米

法國人伊安·馬辛米可以說是在NBA取得成功，主要還是其為球隊救火的功能幫助了馬刺、小牛（獨行俠前稱）、步行者闖過了不少傷缺內線的難關，可馬辛米本身同樣也是玻璃人屬性。在進攻端馬辛米的角色只有吃餅以及沖搶前場籃板，低位擋拆、高位策應的戰術幾乎與他無關，可以說是單憑三招斧闖天下的NBA版程咬金。不過在後場他的防守籃板確實是教練們的救心丹，出色的拼搶意識和對抗能力深得波波維奇在內的名師欣賞。可惜來到職業生涯晚期逐步被小球潮流所拋棄，緩慢的腳步移動、硬邦邦的投籃手感甚至還有低於7成的罰球命中率，你很難相信現在聯盟尚有這種射術不精的球員。更讓人吃驚的是，馬辛米本賽季的薪酬占著華盛頓奇才隊薪酬空間的九分一，高達1545萬美元，幸好是合同年，在明年夏天將他清理掉絕對是意料之內情理之中。並且馬辛米明年即將年滿34歲，這個年齡段的老將估計連歐洲聯賽、CBA等都難以看上眼。

烏杜尼斯-哈斯勒姆

邁阿密熱火隊隊魂隊長哈斯勒姆出人意料地開始了自己在NBA的第17個賽季，目前暫時只有兩場的出場，表現雖然沒有如文斯·卡特一樣老且辣，但在更衣室的精神領袖能力連隊內核心吉米·巴特勒都尚未達到。不過哈斯勒姆同樣是小球潮流的犧牲品，縱觀他兩次的奪冠賽季都是遊走於四、五號位，這對於身高僅2.03米的哈斯勒姆來說可謂是場場都是賣命肉搏。特別是後期詹姆斯、韋德、波什三巨頭時代的熱火哈斯勒姆的位置已經開始尷尬，無法為隊友投籃拉開空間之餘還在防守端因腳步慢拖累全隊節奏，只能上場揀幾個籃板球或者傳出一兩球策應。當然體育競技的拼搏韌勁、場內外高尚道德都是哈斯勒姆的標籤，為籃球傳承再作自己的努力一樣讓人敬佩。此前哈斯勒姆已經宣布這個賽季將會是其職業生涯終點，籃球潮流打法的更替對其來說已經並不重要。
