11場比賽轟出湖人第三巨頭！三分命中率48.4%，他已化身全能戰士

2019-12-04 0條評論 13次閱讀 0人點贊

北京時間12月4日，今天湖人正式開啟了自己的3連客之旅，首先迎來的對手便是位列西部第2的丹佛掘金隊，最終紫金軍團在詹姆斯與戴維斯的帶領下，以105-96驚險擊敗對手，避免了自己的兩連敗。

本場比賽詹姆斯組合依舊發揮得極其出色，詹姆斯18投9中，砍下了25分6籃板9助攻，而戴維斯同樣是18投9中，得到了25分10籃板。正是在他們兩人的帶領下，湖人隊今天才能夠力挽狂瀾，擊敗西部大豪門掘金隊。

如果說詹眉組合是贏球的第1與第2大功臣的話，那麼今天贏球的第三功臣就非拉簡-隆多莫屬了。今天隆指導替補出場25分鐘，6投4中，三分球2投2中，砍下了11分6籃板6助攻1搶斷的全面數據，這樣的效率甚至不會輸給詹姆斯。更值得一提的是，最近5場比賽，隆多的三分球一共是11投8中，他已經成為了本賽季湖人隊內，三分最準的球員了。

截止到目前，隆多場均能夠砍下8.3分3.1籃板5.5助攻，命中率45.5%，三分命中率更是高達48.4%，成為了湖人替補席上最強大的支柱。眾所周知，隆多一直以來都是以「全能」兩個字著稱的，因此外界甚至贈與他「隆中鋒」、「隆指導」等愛稱，這就更加證明了他的全能。但是以前的隆多還是存在著唯一一個缺陷，那就是沒有投籃能力，但是在加盟湖人之後，為了適應詹姆斯體系，他竟然開發出了自己的三分能力，如今他已經正式成為了一名「全能戰士」！

在本賽季還沒開始的時候，外界始終都在爭論一個話題，那就是湖人的第三巨頭到底是誰？而當時呼聲最高的莫過於庫茲馬，其次當然還有霍華德、波普等等，但是如今看來似乎有一個人更加合適，他就是總冠軍控衛拉簡-隆多。

不得不說，隆指導僅用11場比賽就證明了自己才是湖人的第三巨頭，他在隊內的出手權甚至不會輸給任何人，這個主要還是取決於他想不想出手罷了。要知道，在之前面對灰熊的時候，隆多一度單場出手了15次，甚至領先於戴維斯的14次，這就足以證明他在球隊當中的地位。

當然，小雨之所以認為隆多是湖人的第三巨頭，除了他的個人表現碾壓其他人之外，最重要的一點還是他有足夠的資歷與地位。作為一名33歲的總冠軍控衛，他的成就與能力皆能夠壓得住隊內的這些球員，所以他來擔任這個位置遠比庫茲馬等人更為合適。
