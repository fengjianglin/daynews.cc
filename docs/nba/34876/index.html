庫里的三分球為什麼這麼准？庫里親承訓練技巧，你也可以做到！

2019-12-23 897條評論 3,128次閱讀 121人點贊

斯蒂芬.庫里，控球後衛，效力於金州勇士，小球時代開拓者，作為聯盟頂尖的控球後衛，庫里擁有著一手無解的三分球能力，有三分能力不稀奇，但他的三分射程覆蓋整個半場，甚至後場也能投進！

這麼準的三分球，是怎麼做到的呢？庫里出了視頻對此開放了他的訓練技巧，我們來了解一下。

首先他有著整個聯盟中獨特的投籃姿勢，並且由於他的投籃姿勢使得他可以很快地出手三分，出手速度達到0.2秒！驚人的數字。投籃姿勢我們是學不了了，不過能夠保持很高的命中率卻有一定的技巧，這和投籃姿勢沒有聯繫！那就是手感！打球的朋友們都知道，手感對於投籃來說很重要，那麼庫里是怎麼樣訓練投籃手感的呢？

由於沒有視頻素材，就以文字的形式來描述。

第一，籃下單手投籃，用手腕的力量投，不要打板，就投空心籃，一直重複訓練，你會發現你的投籃手會很有感覺，然後你用上左手扶球的時候就知道了。平常打球的時候我都會這樣先練上一會，效果真的不錯，不然也不會寫出來分享了。

第二點，有了投籃手感還不行，你還得有球感，打球的朋友應該都能理解，球感對於運球的重要性，球感好了，你摸到球的感覺都會不一樣，你不會覺得球很硌得慌，你會覺得球很輕，這是我平常打球的感覺，不知道大家有沒有這樣的感覺。言歸正傳，怎麼樣訓練呢，站立不動，雙手持球手打直將球放於胸口，然後讓它自由下落，在它落地前用雙手抓住，這是一個。另外一個難度較高，正直站立，雙手持球將球置於後腦勺，讓它自由下落，然後雙手穿過胯下接球重複訓練，這個難度較高，但效果特別好！

第三點，投籃的發力，也就是為了出手的穩定性，訓練方法很簡單，有基礎的球員都可以做，跟上面練習球感的姿勢相似。雙腿筆直站立，雙手持球，手打直將球置於與胸口平行，讓球到達腿部位置時抓住，這個過程腿需要彎曲做出投籃準備，抓到球後直接投出去，難度也比較高，需要循環訓練。

庫里所發出視頻的內容就這幾個要點，這隻能說是對於他很有用，畢竟他的投籃姿勢跟常人都不一樣，所以如果你覺得這種訓練方法效果不好的話也不要太在意，不是你的問題也不是訓練方法的問題，只是不適合而已。不過小編平常打球的時候就經常用這些方法訓練一笑會，效果還可以。

最後，庫里提到了一個重點，每一個球員都適用的重中之重，那就是堅持！沒有誰有天賦，天生就會打球，沒有誰運氣隨時都很好，隨時都能投進，只有訓練出真正的實力，才是硬道理，最後祝願每一個喜歡打球的朋友們，不受傷病困擾，打球能開心就好！
