21+2！奇才輸球，比爾34分，恩比德承認吧，76人和雄鹿差距出來了

天天要聞
2019-12-22

北京時間12月22日，76人主場對陣奇才，最終76人以125-108成功擊敗了奇才，這也讓76人結束了自己的3連敗。本場比賽76人這邊球員比較齊整，反觀奇才這邊是很多球員缺席比賽，所以76人也是佔據了一個不小的優勢，只不過在這場比賽里76人依舊是打得不太輕鬆，畢竟76人的目標依舊是劍指總冠軍，但是在比賽中的狀態卻並沒有表現出來。而恩比德承認吧，76人和雄鹿差距出來了。

本場比賽一開始，兩隊打得難解難分，但當奇才開始失誤的時候，76人打開了自己的進攻手感，幫助球隊拉開了一些比分，儘管之後奇才追近了比分，但是76人第二陣容發揮不錯，再次幫助76人拉開比分。第二節比賽里，76人進攻狀態不減繼續拉大比分，而奇才這邊貝爾坦斯和比爾三分不靈，只能看著76人把比分進一步擴大，但末尾階段奇才找到進攻手感，半場比賽結束的時候奇才落後12分。下半場比賽里，奇才繼續追近比分，只不過在第四節比賽中76人發力，拉大了兩隊之間的比分，最終奇才也是無力回天輸掉了比賽。

在這場比賽里比爾也真的是儘力了，全場比賽下來砍下了36分8籃板6助攻的數據，只不過現在的奇才確實缺少了太多的球員，尤其是在內線上更是成為漏洞，加上在比賽中奇才槽糕的防守，哪怕比爾在打出了一個不錯的狀態，依舊是難以帶領球隊取勝。同時在這場比賽里貝爾坦斯面對76人的防守狀態也比較低迷，全場比賽13中3隻得到了9分9籃板3助攻的數據，可以說這樣的神射手啞火的時候，奇才想要贏球更是一件艱難的事情。不過小托馬斯在這場比賽里有著一些不錯的表現，全場比賽下來得到了20分3助攻的數據。

反觀76人這邊，在面對奇才這樣的東部弱旅還是打出一個不錯的進攻數據，西蒙斯14分8籃板11助攻，理查德森21分2籃板3助攻，哈里斯16分6籃板4助攻，霍福德13分9籃板3助攻，恩比德21分13籃板3助攻。儘管看著76人的首發陣容都打出了不錯的進攻，但是從整體來看76人的首發陣容進攻依舊不是太好，畢竟他們在這場比賽裡面對的是防守槽糕的奇才，所以在這樣的情況下，76人有這樣的進攻數據也就不太出奇了。

而對於這樣的情況，恩比德承認吧，76人和雄鹿的差距出來了。在本賽季沒有開始之前，看著76人和雄鹿的陣容，在很多球迷看來這個賽季的東決對手就是76人和雄鹿，但是從現在來看想要讓這件事情發生是有著不小的難度，畢竟現在76人和雄鹿之間的差距出來，這個差距就是76人固然是很強的實力，但是他們在比賽中卻難以有更多的專註度，不能夠在比賽中展現出48分鐘的專註度，反觀雄鹿在比賽中卻能夠打出這樣的狀態。就跟這場比賽一樣，76人本來是拉大了比分，但最終卻讓奇才追近比分，打到第四節的時候才逐漸讓勝利浮出水面。
