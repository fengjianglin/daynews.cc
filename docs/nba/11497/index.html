評選具備成為聯盟前5實力的年輕球員：塔圖姆無緣，莫蘭特上榜

2019-12-11 1,989條評論 4,357次閱讀 1298人點贊

眾所周知，NBA是一個從來不缺少超級巨星的聯盟，當喬丹第二次退役之後，科比和奧尼爾所帶領的湖人隊拿到了三連冠，在科比處於暮年的時候，庫里和他的勇士隊正式崛起，當老一輩球星退役，聯盟又會湧進多位極具潛力的球員，這也正是NBA長久不衰的主要原因。在現役聯盟，這樣的情況也在延續，當詹姆斯等超級巨星狀態下滑時，以下幾位年輕球員，在未來都具備進入聯盟前5的實力。

老鷹隊的特雷楊，在新秀賽季，他曾因為投籃命中率不高，得分效率低，而遭受到外界的質疑，不過在全明星周末之後，他逐漸適應了聯盟的比賽強度，接連打出統治級的表現，最終憑藉著場均19.1分8.1助攻的表現，全票入選了最佳新秀陣容。

本賽季，特雷楊的進步非常明顯，無論是在進攻端，還是在串聯球隊方面，他都已經達到了聯盟頂級水平，賽季至今，他場均能拿到28.8分4.1籃板8.4助攻，場均得分排在聯盟第4位，助攻數排在聯盟第5位，投籃命中率為46.2%，三分球命中率為38.8%，場均命中3.5個三分，今年只有21歲的他，還有很大的提升空間，在未來幾年，他具備成為聯盟頂級控衛的潛質。

莫蘭特，今年選秀榜眼，雖然在開賽之前，並沒有多少人看好他拿到最佳新秀，外界一致認為蔡恩才是最佳新秀的最佳人選，不過賽季至今，莫蘭特場均拿到19.1分3.1籃板6.4助攻，命中率為46.3%，三分球命中率為43.2%，在最後一節，他場均拿到8.5分，排在聯盟第2位，莫蘭特用場上的表現證明了自己的實力，他是本屆新秀當中表現最好的球員。

莫蘭特處理球方式非常成熟，得分效率高，能自己進攻，也能幫助隊友完成得分，關鍵球能力很強，今年只有20歲的他，如果能保持這樣的成長速度，他將會成為一名非常特殊的球員。

布克，很多人在談論他的時候，都會覺得他是一位得分能力強，但沒有在帶隊方面證明過自己的球員，其實也確實如此，在過去幾年，太陽隊戰績都不是很好，儘管布克能單場拿到70分，但也無法改變外界對於他的看法。

而本賽季，布克在得到足夠多的幫手之後，他在帶隊方面也展現出了自身實力，目前球隊戰績排在西部第8位，布克場均拿到25.5分4籃板6.3助攻，命中率為51.4%，三分球命中率為41.5%，罰球命中率為90.8%，現在的布克已經不再是一位純得分手，他正在成為一名出色的領袖，值得一提的是，布克今年只有23歲，在這個年齡就有這麼出色的表現，實屬不易。

​

東契奇，賽季至今，他場均拿到30分9.8籃板9.2助攻，命中率為47.6%，拿到7次三雙，表現出色，在外界看來，他已經步入了聯盟前5的行列，如果單看數據的話，確實如此，不過個人覺得，目前東契奇雖然數據出眾，但還沒有達到聯盟前5的級別，至少他還沒有在季後賽當中證明過自己的實力，不過，作為一名20歲的球員，在自己第二個賽季，就能拿到場均三雙的數據，在聯盟歷史上，也是非常罕見的，在未來，東契奇如果不受傷，那麼他絕對有潛力成為聯盟前5級別的球員，甚至有可能成為頭號球星。
