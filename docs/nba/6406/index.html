球隊觀察｜夏洛特黃蜂贏球難，改變防守才是正道

2019-12-01 0條評論 16次閱讀 0人點贊

看了幾場黃蜂的比賽，想談談這支球隊目前的問題。

不知道現在黃蜂的球迷是哪個年齡層，又是因為什麼選擇在如此的大環境下喜歡上了黃蜂隊。你問我什麼大環境嗎？很明顯，那就是「雙星」球隊正當道，老牌強隊又崛起的時代。勇士的五年統治方一落幕，湖人便拉來濃眉重整旗鼓，快船也集合兩位攻防大師意欲稱王，東面的雄鹿不甘落寞仍在連勝，堅持底蘊的綠凱也頗具競爭，甚至是除了這些球隊，你仍可以說勇士還有希望，火箭機會更大，或者是猛龍還很「猛」，雷霆也很「雷」。

在這環境下，堅持看著黃蜂的比賽，看著他們輸掉很多球也不泄氣，我得先給球迷們點個贊。

而後再來說說，他們的問題出在哪裡。

沃克離開的直接影響：進攻艱難，失去方向。

球隊最大的核心是肯巴沃克，這點在過去幾年都是毋庸置疑的，這對黃蜂來說意味著：球隊擁有絕對進攻強點，關鍵球也有第一處理人選。

而今年沃克選擇離開黃蜂后，黃蜂多少顯的群龍無首，球隊的進攻上出現了很大的問題，這些問題包括：

  1. 失去穩定輸出點：肯巴沃克在上個賽季場均35分鐘的出場時間裡得分是25.6分，球隊的使用率是31.5％，勝利貢獻值是7.4。他在進攻端的進攻選擇是多樣的，既可以後撤步也可以變向突破，亦或是推進節奏轉換進攻—-兼具外線射程和攻框能力，而他一離開的直接影響就是黃蜂隊再沒有第二個人像他一樣提供穩定的輸出：羅齊爾有射程卻缺乏對籃筐的衝擊，二年級的格雷厄姆有腳步和技巧，卻總在對抗上吃虧。說白了，沒有人可以持續進攻，意味著球隊得分失去了保障。拿最近三年的數據來比較，17-18和18-19賽季，黃蜂隊的得分是場均108.2和110.7，而在今年的21場比賽過後，他們的數據是105.5分。還有一個直接的變化體現在罰球上，這也可以說是沃克的功勞。前兩個賽季，他們分別有20.2和18.4個罰球，這個賽季已經降到了15.2，因為沃克的離去減少了黃蜂對籃筐的衝擊，這一數據的下降不可避免。
  2. 戰術執行困難：失去了強核心後，戰術變得不明確，這是經常出現的問題。而黃蜂在適應這個改變時，這一問題同樣顯得突出。羅齊爾運球到前場後，受個人能力和隊友所限，戰術往往都是簡單兩個：球在外圍不停傳導，看似再找最好的機會，最後幾秒又是落到一個人手裡單打；或者是在高位叫一個擋拆後，重新回球給中鋒或大前，完成一次簡單投籃。這樣的戰術過於簡單，又很難改變，因為繁複的戰術很難叫成，在相對渙散的球隊中執行一個複雜的戰術又很困難，時間一長，進攻端連連受挫又導致了執行力降低，惡性循環讓黃蜂叫苦不迭。

藍色為得分，橙色為罰球數

不過好在他們沒有就這麼頹廢下去，看似絕境之下，有人站出來，也有人在想辦法。

格雷厄姆，這個年輕人在去年應該還沒什麼人知道。他在2018年第二輪的34位被黃蜂選中的，是一位1米88的後衛。去年他代表黃蜂出戰了46場僅3場首發，14.8分鐘的時間裡能拿下4.7分1.4板2.6助的數據，作為一個二輪秀來說還算不錯；本來是平平無奇混口飯吃，這沃克一走卻好似成就了他，目前他代表黃蜂場均出戰33.2分鐘，基礎數據全面上漲到18.2分3.5板7.5助，這啥概念？這放在NBA中，完全可以被認為是巨星成長之路了：出場時間漲了一倍多，數據是3-4倍的翻啊，使用率從17.0％增加到了24.7％。

他個人給黃蜂帶來的變化也挺直接：0.7→3.4，三分命中數的飆升，使得黃蜂從進攻困境中走出來一點，至少可以在外線有所作為，這也是他們顯而易見的改變，進攻端既然無法往內深入，就可以往外擴，整個球隊的三分命中數也在增長，這賽季至今他們三分出手數和命中數分別為743和270，都位居聯盟第4，但三分命中率排在第10，或許還有上升空間。

球隊雖然目前輸球比贏球場次要多，但有球員站出來總歸是好事，況且，他們還有很多點是可以改變的，僅僅簡單粗暴的依靠進攻肯定行不通，還得改變他們羸弱不堪的防守才行。

防守怎麼變？

他們的防守效率差不是這麼一兩年了，上賽季防守效率是倒數的，這賽季呢？防守效率為114.4，排在全聯盟第27位，場均能讓對手攻下114分，可以說是城門大開。

所以他們的防守問題到底有哪些？半場防守鬆懈，貫徹不了教練的防守策略是一點，這是態度問題；其次，陣容不行，這是硬體問題，幾個位置調來調去，都達不到滿意的效果。

他們的陣容，是絕對能將防守水平提到聯盟中游的，只要不是在聯盟里吊車尾了，戰績都不會像現在這樣，或是跟前幾年一樣，離季後賽漸行漸遠。

那要怎麼改變呢。

1、調整防守策略：古人有雲，知己知彼方可百戰不殆。籃球場亦是如此，假如你面對的球隊，是像掘金、魔術或者步行者這種，沒什麼三分能力的球隊，可以嘗試用2-3聯防—-最基本的聯防戰術來防幾個回合試試成效，既然三分不準，就收縮內線保護籃板；遇到以前的勇士、現在的火箭或者達拉斯這種「三分大隊」，改用1-2-2聯防強調外線夾擊，掐死三分；又或者是碰到內線大閘肆虐籃筐時，3-2聯防對內線的包夾作用又體現出來了。聯防也好，盯人防也好，混合防守也好，我想說明的都是，籃球從來不只是身體，也同樣體現著球商和大局觀，何況黃蜂還沒到那麼差勁的地步，因此改換防守策略或許在未來對他們能有幫助。再舉個例子，總決賽上猛龍主教練別出心裁用了一招BOX-1，一盯四聯的防守讓庫里吃了苦頭，第四節沒有出手也沒有助攻。

2、正視防守態度：很多東西都是「一口氣」能調過來的，態度就是。最近幾場比賽，其實可以看得到黃蜂已經開始在防守上做出努力，這樣的努力體現在腳步能夠跟上，半場跟防不再懶散，協防可以及時上面。但仍要指出的是，過去的比賽習慣還是讓他們在防守上會時常鬆懈—-防守籃板為30.95，聯盟倒數第一；防守籃板率70.6％，聯盟倒數第三。球員的防守態度真是個很虛幻的詞？不是。黃蜂現階段要做到的是：堅決貫徹教練的防守，有意見可以提，但在防守端執行力要達到百分之百。

羅齊爾

一支球隊要怎麼去成長呢，我看就得像黃蜂這樣。

他們正在經歷困難的時刻，過去幾年一直陪伴他們的絕對核心離開了，這對球隊的文化和建設，進攻和防守的變化都影響很大，但要經歷這樣的時刻，才抵得上未來獲得榮耀的時刻。

他們面對進攻端無力的時候，有一個叫格雷厄姆的小將站了出來，拿起球開始進攻。他們面對防守端孱弱的時候，又必須將所有人聯合到一起，眾志成城，去防下每一個球。

這就該是球隊的成長，更何況，所處的球隊在東部，意味著挽救餘地還很多，經歷一切調整後，仍有殺出重圍的可能。而黃蜂的球迷們，也更需要耐心等待，看看他們的變化，將帶給你們怎樣的驚喜。
