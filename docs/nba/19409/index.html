3年9000萬的大前鋒恐已和掘金走到了盡頭，超新星被視為未來首發

2019-12-17 290條評論 3,669次閱讀 1114人點贊

上賽季掘金不僅打進了季後賽，更是一舉打進西部半決賽，從此這支位於西北高原的球隊就徹底完成了重建。雖然說掘金隊重建的核心是約基奇和賈馬爾-穆雷，但是掘金能如此迅速取得成功，也離不開9000萬大前鋒米爾薩普的幫助。在2017年掘金果斷開出3年9000萬的大合同，搶先簽下了自由球員米爾薩普。他也給這支年輕的掘金隊帶來了最寶貴的東西——經驗。

本賽季是米爾薩普的合同年，而他和掘金的緣分恐已經走到了盡頭。首先米爾薩普已經34歲，本賽季結束之後就年滿35歲，未來狀態如何將直接影響他的續約，畢竟35歲是一個坎；其次3000多萬的年薪已經不是掘金能夠負擔的，下賽季穆雷和約基奇年薪都超2800多萬，還有哈里斯的1900多萬以及巴頓的1300多萬，掘金也面臨巨大的薪資壓力。

其三可能才是最重要的是，那就是那就是兩大替代者恐已經躍躍欲試。近日NBA名記表示對掘金來說小邁克爾-波特是不可交易的對象，掘金將其視為未來的首發大前鋒。不管小邁克爾-波特未來能不能夠成長為一個合格的首發大前鋒，起碼現階段的格蘭特就是一個擁有首發實力的替補，畢竟他曾是雷霆的首發大前鋒。

小邁克爾-波特在選秀前被視為超級新星，而且有問鼎狀元的實力，可是因為傷病選秀順位一降再降，後來更是缺席整整一個賽季。本賽季小邁克爾-波特還在逐漸適應掘金的體系，場均出場時間也只有8.9分鐘，但是不可否認他的天賦還在。未來小邁克爾-波特或許無法達到選秀前的期望，但是一個首發之位應該不算難事。
