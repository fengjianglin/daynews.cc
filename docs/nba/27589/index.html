矛盾重重！NBA醞釀4大改革，東西分區破壁在即，雞肋or良藥？

2019-12-22 135條評論 1,215次閱讀 68人點贊

由於場內場外多種因素的影響，在過去幾年，NBA的收視率急速縮水，據美國媒體《體育商業日報》近日發布的報告顯示，在NBA的收視率方面，TNT較之上個賽季下滑了23%，ESPN方面下滑了20%，而地方體育網總體下滑了13%。

面對如此棘手的問題，聯盟也在尋找解決的辦法，其中就包括對於現有賽制的改革。

據Shams等多名美媒記者聯合報道，NBA正在醞釀4大改革，東西分區破壁在即，但是有關改革的討論卻矛盾重重，很難判斷是雞肋，還是良藥。

提議1：常規賽縮減至78場或79場，其中包含錦標賽的分區比賽（下文詳解）；

提議2：增加季中錦標賽，獲勝的球隊每名球員獲得100萬美元的獎勵，教練組獲得150萬美元的獎勵（6個分區決出冠軍，加上剩餘球隊中戰績最佳的兩支球隊，8支球隊進行單場淘汰決出冠軍。）；

提議3：增加季後賽附加賽，目前討論的方案為，附加賽分為兩輪，東西部分區分開進行，每個分區的7、8、9、10名參與，第一輪中第7名與第8名對戰，第9名與第10名對戰，7、8名勝者獲得季後賽名額，9、10名勝者，獲得進入第二輪與首輪7、8名失敗一方競爭剩餘一席季後賽的資格；

提議4：季後賽從第3輪開始打破東西部分區，按照常規賽戰績進行對陣安排。

不難看出，這一次聯盟為了增加收視率所計劃作出的改革尺度非常大，不僅涉及到常規賽的場次安排，同時涉及到季後賽的對陣，推出全新的季中錦標賽等一系列方案，甚至提出季中錦標賽冠軍每人100萬美元真金白銀的獎勵。

因此，在消息曝出的第一時間，就迅速成為業內人士與球迷熱議的焦點，其中有叫好認可的，也有唱衰悲觀的。

那麼這一系列的調整是雞肋還是良藥？

部分球隊的高管與大城市球隊老闆對此感到擔憂。

因為常規賽的縮減，意味著主場收入的縮水，如果按照目前的計劃實施，那麼每個賽季每一支大城市球隊將減少2個主場比賽，相當於270-400萬美元的真金白銀不翼而飛。

同時，賽季中期的錦標賽，對於年入千萬的明星球員來說沒有吸引力，相比於5天的休息，這份獎金非常雞肋，明星球員的不在意，自然無從提及增加收視率，因為他們才是收視率的保證。

只不過，對於聯盟大部分的球員來說，季中錦標賽的獎金還是非常誘人，因為聯盟中拿下大合同的球員總歸是少數，聯盟不足400萬的球員佔據50%，更是有142名球員年薪不足200萬，因此這份獎金對於其中大部分的球員來說具吸引力，但是他們並不能決定收視率的好與壞。

值得一提的是，相比於季中賽與常規賽縮水，季後賽的兩項調整，目前反對的聲音並不多，反而獲得大量球迷的支持。
