2.31米巨人穿著《火影忍者》特製彩繪鞋現身NBA賽場

2019-12-14 463條評論 2,786次閱讀 560人點贊

現今NBA最高的球員，是波士頓凱爾特人隊的新秀塔科·法爾（Tacko Fall），而他也是一位狂熱的動漫愛好者。在12日凱爾特人隊的比賽前夕，他就被攝影鏡頭捕捉到，穿著一雙非常顯眼的特製《火影忍者》球鞋。

塔科·法爾在NBA官方登記的身高是2.31米，雖然本季至今他只有一場出賽紀錄，當天對上印第安那步行者隊的比賽也沒有機會上場；但賽前練習時，他的驚人身高還是相當引人矚目。除了身高之外，他身上另一個亮點就是腳上的球鞋，攝影鏡頭拍到他的左腳鞋上繪有《火影》主角漩渦鳴人，而且還是「仙人模式」的狀態；右腳則是鳴人的師傅、木葉三忍之一的自來也。

近年隨著日本動漫席捲全球，NBA年輕球員也不吝於展現出對卡通動漫的喜愛，其中一種表現方式就是穿著相關主題的特製球鞋；例如先前效力於薩克拉門托國王隊、本季加盟金州勇士隊的中鋒威利·考利-斯坦（Willie Cauley-Stein），也曾穿過《火影忍者》主題的球鞋，他的鞋上圖案一腳是鳴人，另一腳則是鳴人的對手兼好友宇智波佐助；另外他還穿過繪有《高達》的球鞋出賽。

威利·考利-斯坦也曾穿著《火影》特製鞋。

威利·考利-斯坦穿著繪有《高達》的球鞋出賽。

至於最受NBA球員歡迎的動漫作品，首推歷久不衰的《七龍珠》。包括布魯克林籃網隊後衛斯賓塞·丁維迪（Spencer Dinwiddie）、明尼蘇達森林狼隊的前鋒喬丹·貝爾（Jordan Bell），以及上季新人王、本季MVP大熱門的達拉斯小牛隊前鋒盧卡·東契奇（Luka Doncic），都曾穿著過《七龍珠》相關設計的特製鞋款出賽。

本季打出優異表現的籃網後衛丁維迪，過去曾穿著《七龍珠》特製鞋款出賽。

喬丹·貝爾對《七龍珠》的熱愛是出了名的，也曾穿過相關特製鞋款。

相信這種特製的動漫鞋款，未來還會在NBA場上陸續出現，如果有機會量產售賣，或許也是一大商機呢。
