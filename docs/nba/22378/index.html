圖片不會騙人！NBA巨人體型太誇張，臂展2米59之人如巨型蜘蛛

2019-12-19 3,689條評論 4,617次閱讀 2945人點贊

我們都知道，NBA聯盟可以說就是一個巨人的世界。作為世界上水平最高的籃球世界，他對球員的身體素質也提出了極其高的要求，因此，每位運動員不僅要有著可怕的天賦和籃球商，沒有打基礎的素質，也是不能在聯盟立足的。沒錯！像小刺客托馬斯這種常人身材進入聯盟的，確實是少之又少。而這個聯盟的頂尖運動員，更有著讓人感到恐懼的誇張身材，可以說就像故事書里走出的怪物一樣！那麼，這些球星究竟都有誰呢？圖片不會騙人，下面告訴你答案

博班站著扣籃輕輕鬆鬆

2米24的博班把身高與臂展結合的完美，球場上隨時可以來一次站著扣籃，對手怎麼防？看著就好了。

2米31的波爾如同大蜘蛛

擁有2米31身高的同時，波爾還有著一雙長達2米59的NBA歷史中的最長臂展。如圖一樣，他就像球場中一隻巨型蜘蛛。

在游泳池裡又如異行世界中的巨型水怪。

麥克海爾酷似長臂猿

我們都知道，麥克海爾是凱爾特人的名宿，作為當年綠軍三巨頭的重要成員，他就是大鳥伯德的護衛，可以說，雖然是白人，但是麥克海爾的能力卻絲毫不輸給運動天賦普遍更好的黑人，不管是內線拿分還是保護籃板做得都相當到位，並且也服從教練安排，即使放在替補增強板凳火力也從無怨言。其中，他的防守工作更是看家本領，而這都多虧了麥克海爾的那雙長臂，可以在防守端為對手帶來堪稱強力的干擾。說實話，從圖片上看，麥克海爾簡直就是長臂猿！

碎骨機肩膀像城牆

我們大多數球迷不知道的是，NBA歷史上，只有一個人做到了新秀賽季就拿到MVP，這個人就是碎骨機昂塞爾德，作為曾經來中國跟中國名宿穆鐵柱打友誼賽的美國球員，碎骨機在美國享有崇高的聲望，說實話，作為子彈隊的名宿，他可以說是華盛頓最偉大的球星。除此之外，他的身材也只能用恐怖來形容，尤其是他的肩膀，說是城牆，沒有一點問題，因為太寬了，哪怕是NBA隊友，也相形見絀。

大姚寬厚超過鯊魚

說實話，我們都知道大鯊魚的維度到底有多恐怖！可以說，奧尼爾是一個百年難得一遇的籃球奇才，不僅因為他的技術和天賦，因為他的身體實在是太特別了，沒有人能做到像他那樣強壯並且靈敏，在籃下，他可以輕易虐殺大多數對手內線，然而，在這方面，大姚可以說是為數不多在巔峰時期，可以讓奧尼爾稍稍吃癟的內線，因為大姚本身比奧尼爾高出不少。另一方面，在無比強壯的奧尼爾面前，大姚的身材實在不算多瘦弱，看這張合照你就明白了——甚至大姚還要顯得稍微粗壯一點，這也是為什麼那麼多明星跟大姚合照，都顯得無比瘦小的原因！就連奧尼爾，也比不過他啊！

張伯倫讓終結者吃癟

隨意舉起一個普通人，輕鬆轉動

現在說到張伯倫，都是活在人們記憶中的傳奇球星，其實張伯倫的統治力不只在球場之上，在球場下，這位超級名宿依舊留下了堪稱無比輝煌的「戰績」。其中，哪怕退役之後，他也依舊保持著自己的威名！那個時候，他跟施瓦辛格是好兄弟，兩人還合作過電影，而據施瓦辛格這個專業的健美運動員說，張伯倫的身材可以說是恐怖無比，就算站在終結者身邊，也顯得大了一圈，並且，他的力量也十分驚人，甚至能夠單手舉起他！

伊頓讓馬龍像後衛

說到伊頓，可能現在大多數球迷會感覺有點陌生，但是，提起當年那支爵士，可以說是無人不知無人不曉了。儘管在跟喬丹對抗的時候，伊頓已經退役了，但是作為馬龍和斯托克頓的戰友，伊頓在八十年代也給聯盟留下了深刻的印象，最恐怖的就是他的身材以及單賽季場均5+蓋帽的NBA歷史記錄！當時魔術師就說了：「想要奪得總冠軍，你首先得擊敗猛獁象伊頓！」這全是因為他在籃下巨大的震懾力，他的身材就像一堵高牆，你根本不知道該怎麼突破他的防線，就連和馬龍的合照，也讓馬龍一瞬間嬌小了不少。
