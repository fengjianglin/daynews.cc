西部第十丨擁有雙狀元的森林狼為何還會7連敗，枸杞哥不香了嗎？

2019-12-16 729條評論 2,097次閱讀 523人點贊

「天賦」和「運氣」自古以來都是一對歡喜冤家，很少有人能夠兩者兼備，除非他擁有足夠的後天實力能夠撐得起隨時到來的運氣。當然一支球隊的高低程度，從某種意義上來講，除了擁有一位較為明智的主教練。更多的是球隊核心的數量，雖說三個臭皮匠頂一個諸葛亮。但是籃球比賽只能夠上場五人，三個頂一個，另外兩個能頂住其他四個嗎？所以這句話的用處顯然在球場上並不合適。

中鋒時代已經離我們遠去，小球時代已經到來，球星數量的多少能夠決定一支球隊表面上的高低。但是，凡事總有意外，例如擁有雙狀元的森林狼。表面上他們擁有著不凡的實力，但回歸到最初的遊戲規則上來說：籃球是五個人的賽事。雙狀元仍然不頂用，這其實是和球隊的化學反應和其他位置上球員能力的高低有關；

北京時間12月14日，森林狼和快船隊上演了一場精彩的比賽，最終喬治和倫納德的發揮要更勝一籌，快船獲勝送給了森林狼隊一場七連敗的尷尬紀錄。在賽季開始之初，森林狼的狀態還是非常不錯的，在一段時間之內居然佔據了西部榜首的位置，唐斯和維金斯這對雙狀元組合也是打的非常出色，讓球迷們也是看到了這支球隊的未來和希望，但是隨著比賽的逐漸深入，他們的狀態開始出現了起伏不定的問題，先是唐斯和恩比德打架遭遇禁賽處罰，隨後維金斯的親人去世也缺陣了幾場比賽。

從這些事情發生開始，他們註定就迎來了生命中的轉折點。下坡路就像一位沒有及時保養的四十多歲女人的皺紋一樣開始逐漸加深，目前他們戰績僅僅是10勝15負，排在西部的第十位。如果他們不及時做出調整的話，季後賽對於他們來說就是天方夜譚了。所以我們可以借著他們高能低位的樣本數據，來具體分析一下球隊所存在的詬病。

進攻效率出眾，但防守端問題巨大

最近的七連敗期間，森林狼在進攻端的表現還是可以接受的，尤其是維金斯和唐斯的狀態都還是比較不錯的，但是他們最大的問題就是出現在了防守端，對於一場籃球比賽的勝利來說，防守不到位，那絕對能夠成為災難級的難題。在這七場比賽當中，他們的失分都沒有低於110分的，其中還有4場120+，1場130+。甚至在面對湖人的時候失分達到了142分。這樣的防守狀態想要贏的比賽無疑是痴人說夢話！

雖然森林狼里的個人單防能力都還不錯。但是整體防守效率並不是個人防守能力的累加，這並不是一個1+1=2的簡單數學題，所以他們的防守效率在最近已經迫近到了聯盟墊底的程度（森林狼截止至今全隊的防守效率值為111.4，排在聯盟第22位）【防守效率越高，說明防守效果越差】。在森林狼的比賽中最大的問題是面對對方的擋拆進攻，尤其是在鋒線的擋拆上，無論是奧科吉還是卡爾弗在擋拆的換防上，他們兩個人的身高由於非常有限，所以經常會吃一些虧。

我們從上面兩次的防守端就能夠看出，在面對擋拆防守時，後衛線上的防守球員都被甩到了身後，進攻球員直接面對的就是中鋒，而對方面前就會有很大的空間，無論是直接投籃還是傳球尋找第二機會都是非常合理的。我們看到唐斯的身後有著非常大的利用平方點，會被使用得非常輕鬆。所以在擋拆過程中，森林狼的防守就顯得有些捉襟見肘了，整支球隊的防守體系很多時候建立的並不是非常的徹底，球隊的協防也不是很明確，這也是這幾個賽季以來森林狼一直存在的問題。防守端的詬病並非只有他們球隊有，但是像他們這種擁有雙狀元的球隊，顯然不應該成為這個模樣。所以他們如果想要更進一步成長，就必須要在防守端進行更多的改造。

後衛線實力不足，串聯能力一般

對於一支球隊來說，五個位置總是需要五位較為優秀的球員才能夠匹配得當。當然並非所有球隊都能夠如願以償，這時，取長補短就成為了首要的事。你可能只擁有一位較強的鋒線球員，但是卻能夠多一名外線射手。這樣的陣容一樣能夠成為黑馬球隊，但是當鋒線無比強勢，可缺少了射手的威脅，那麼就會變得讓人難以忍受。

一個好的控球後衛對於一支球隊來說同樣非常重要，例如本賽季的菲尼克斯太陽，他們本賽季在引進了盧比奧之後，球隊整體上提升了一個檔次，在球隊的掌控和比賽節奏的把握上也是可以帶領球隊贏得更多的場次。但是森林狼卻沒有一個比較靠譜的控球後衛。他們在賽季開始的階段比較喜歡使用老控衛蒂格，但是一段時間之後啟用了新秀卡爾弗。卡爾弗和奧科吉組成了球隊的後場搭檔，但是兩個人最大的問題就是都不會投籃，他們兩個人在球場上的時候，球隊的空間會被壓縮的比較厲害。我們下面看一下兩個人的投籃熱區。

上面兩張圖就是兩人的投籃區域效率圖，兩個人的投籃區域幾乎都是低於聯盟的水平，尤其是作為外線球員，兩個人幾乎都沒有三分的穩定手感。在防守他們的時候，很多防守球員都會留給他們很大的空間，從而去夾擊維金斯和唐斯，也給主要的進攻球員造成了很大的進攻壓力。一個完美的外線炮台能夠承擔起球隊三分之一的責任，就像湖人的丹尼格林一樣，他在三分線外的威脅完全可以吸引防守球員，這時，無論是詹姆斯還是濃眉哥都可以單人強攻或者突破後分球，他們至少不用頂著兩個人的壓力。

此外，森林狼替補的後場雙槍是蒂格和內皮爾，這兩個球員是聯盟的老油條，但是兩個的巔峰時期都已經度過。兩個人的屬性都是偏向於進攻端的，在進攻的組織上實在是比較一般。而且他們兩人都是依靠自己的突破來為隊友創造機會，但是如今兩個人的突破並不是特別的犀利，所以森林狼的四名後場球員都不是特別的會組織球隊的進攻。球隊的失誤數是在聯盟的倒數的位置，助攻和失誤比也是非常的高，所以整體來說球隊沒有太好的組織者，進攻端在關鍵的時刻通常都會是維金斯來控制比賽的節奏，但是往往會影響他在進攻時的表現，所以森林狼需要在後衛位置上進行一些改變。

全隊三分命中率為33.0%排在聯盟第28位，助攻率為56.4%排在第23位，防守籃板率為72.5%排在第20位。單從這幾個數據就能夠證明上面所說的，防守強度不足，外線投射威脅能力差。全隊的主要進攻方式多靠雙狀元，助攻率較差是由於控衛串聯球隊能力有限。

個人單打進攻單一，球隊配合效率不高

森林狼在進攻端的打法也是相對比較單一的，主要還是採取個人進攻比較多。這也和他們球隊後場組織能力一般息息相關。球隊的進攻主要是依賴維金斯和唐斯在上線的擋拆進攻來完成的一些配合，但是這幾乎是僅限於森林狼的簡單配合了。在更多的時候，球隊的擋拆進攻效率都不是特別的高。他們在進攻的時候都會採用比較簡單的擋拆來發起，但是因為後衛線球員的投籃能力比較一般，所以只能選擇突破。

這樣的話防守防守球員非常容易進行判斷，大多數的時間都會選擇從後面繞過擋拆，不給森林狼球員太多的突破空間。最終只能通過傳球來重新進行組織或者交到維金斯或者唐斯的手裡，讓他們單人發起進攻。而奧科吉和卡爾弗的終結能力都是比較一般，很難直接給對手形成太大的威脅，所以整體來說他們的進攻手段比較單一，這樣的進攻雖然看上去效果不錯，但是非常容易被對手所針對。所以在關鍵的時刻往往過於依賴於自己的主要核心球員，這樣的情況會對讓雙狀元承受更大的壓力，而球隊的上限就成了完全取決於他們倆人之間發揮的程度了。

從上面兩次進攻來看，第一個進攻是奧科吉在進攻端的時候面前擁有如此之大的空間，並不是他們的進攻配合有多麼的出色，而是對手根本沒有注意他這個點的位置，更多是去夾擊了在強側的唐斯，所以最終這個球奧科吉也沒有把握住。這一次的進攻也就以失敗所告終，外線的終結能力比較一般的情況下，唐斯更多的時候只能選擇個人的強力進攻，他很多時候並不是不想或者不主動與隊友尋求一些配合，主要還是球隊的其他球員很多時候很難在這個時候挺身而出。

第二個進攻，內皮爾持球突破之後可以說進入了一個比較困難的位置，面對比自己高出很多的西蒙斯，他很難有直接上籃的機會。那麼這個時候隊友就應該迅速過來進行一些接應，但是從圖面上來看，只有一名森林狼的球員在禁區之內準備接應，而其他的三位球員居然都在三分線為站樁無動於衷，這樣進攻之間的呼應實在有一點令人失望，所以森林狼隊這樣的表現也應該不是特別令人感到意外了。團體配合之間的融洽程度會間接的決定一場比賽的勝利，單人強勢不如多點開花。這也是為何森林狼無法成為一支助攻率較高的球隊的原因之一，他們的進攻和戰術衍生態過於少了。

樂福的去向是否能夠決定森林狼的命運？

最近在自由市場上樂福的去處也成為了一個焦點，而其中一個消息的傳出就是他有可能重新回到自己曾經征戰過得明尼蘇達，這也是引起了人們對於森林狼更多的期待。但是目前來看森林狼的問題主要是在於防守端，樂福的防守也是一直被人們多詬病的，他和唐斯的組合不知道能不能重新找回自己防守狀態，當然這也只能是一個預測。兩個人能否擦出更多的火花這一點是非常值得期待的，當然這是在樂福成功回到森林的前提下來說。

但是除了在自由市場上物色好的球員以外，球隊的內部發掘也是非常重要的，卡爾弗雖然還存在著一些問題。但是他的努力還是有目共睹的，他如果可以快速的成長，他未來一定會成為森林狼外線的一個比較好的防守球員和攻防轉換的搖擺人，當然這只是他的第一個賽季，無論是投籃穩定性還是組織上都比較一般。但是他的存在感非常的強，他可以通過很多不同的方式來幫助球隊。所以卡爾弗和奧科吉都是外線的重要球員，如果能有一位好的導師的話，兩個人的前途還是無量的。

森林狼的未來目前還是比較不明朗的，畢竟樂福的未來還是八字沒一撇的事，而卡爾弗的成長也需要更多的時間。對於雙狀元來說，他們還能否有時間等待這一切也都是未知的，所以還是希望森林狼能夠重新找回狀態，畢竟雙狀元的組合還是值得我們所期待的。上帝最擅長的操作就是給予你運氣之後給你磨難，這是森林狼現在所需要面對的問題。進攻效率的高度大部分都取決於雙狀元的發揮，但是防守端的漏洞卻成為了球隊難以彌補的難題。他們的進攻動態想要多元化，必定需要控衛擁有較為廣闊的視野。此外，球隊外線所能夠具備的威脅程度同樣也會成為勝負的關鍵點之一。朝著季後賽的夢想前進，森林狼永遠不缺乏血性的鬥爭。
