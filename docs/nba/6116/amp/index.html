單節21分轟44+9+8，最後17分鐘6分尷尬！這是布克跟超巨的距離

天天要聞
2019-12-06

北京時間12月6日，整場比賽，布克在前三節火力全開，他24投13中，拿到了44分9助攻8籃板。不過，第四節和加時賽，他只得6分。這就是他跟超級球星的差距。

太陽跟鵜鶘迎來了一場對攻戰。上半場，兩隊就合砍144分，半場結束，布克轟下了17分5助攻，但是，鵜鶘有4人上雙，雷迪克17分、霍樂迪15分、海耶斯11分、英格拉姆11分。球哥也有不俗表現，送出了7分5助攻。

下半場回來，布克在第三節大爆發。一上來，他就跳投命中。隨後，追身三分也有，連得了5分。球哥三分還擊，英格拉姆頭部挂彩，受傷下場。布克開啟個人表演，借掩護之後，突破上籃得手，隨後，他錯位單吃梅利，一記原地干拔跳投，輕鬆命中，太陽領先了14分之多。

布克快攻一條龍暴扣，緊接著，三分再中，完全不可阻擋。整個第三節布克轟下了21分，幫助太陽三節結束領先了16分之多。遺憾的是，第四節布克啞火了，這一節只得4分。

鵜鶘單節凈勝16分，英格拉姆絕殺不中，比賽進入加時。加時賽，布克依然迷失。好在隊友站出來，盧比奧連續跳投命中，隨後，他助攻卡明斯基暴扣，還剩1分鐘，太陽領先了6分。鵜鶘犯規戰術，布克走上罰球線，兩罰兩中，太陽笑到了最後。
