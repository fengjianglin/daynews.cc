不簡單！快船瘋狂輪休倫納德仍然西部第2，里弗斯太有遠見了

2019-12-10 3,006條評論 4,688次閱讀 2142人點贊

快船和步行者迎來了一場常規賽的較量。最終，快船以110:99贏下了步行者。保羅喬治出場35分鐘26投10中，三分16中7，拿下了36分9籃板5助攻。路威本場比賽手感不佳，19投僅僅2中拿下6分。哈雷爾再次延續了自己高效的表現整場比賽22投12中拿下26分8籃板。步行者這邊得分最高的是布羅格登，他出場30分鐘拿下20分2籃板，投籃15中7。麥克德莫特表現不俗，三分8中5，拿下了17分。

本場比賽的交鋒兩隊實力確實有一定的差距，快船以18勝7負的戰績排在西部第2位。而步行者在競爭力稍差的東部，以15勝9負的戰績排在第6位。因為奧拉迪波的傷病問題，步行者在場上時沒有一個明確的核心。導致球隊的進攻不流暢，在球隊關鍵的時候，沒有人能夠站出來得分。布羅格登雖然投射能力出色，但畢竟不能作為球隊的當家核心，相比保羅喬治還有一定的差距。最終步行者落敗，只能說是意料之中。

但是，本場比賽保羅喬治全場出手16次三分球命中7次。可以說這個賽季的保羅喬治，場均出手將近10次三分球，相比以往幾個賽季，有了很大的提升。從比賽過程就能看出，隨著小卡的缺戰，保羅喬治在場上打得愈發從容，也能夠展現出自己超級巨星的實力。而當小卡回歸的時候，保羅喬治在場上打得會更加猶豫，不能隨心所欲的出手，在進攻的時候也總會顧及隊友的存在。導致前段時間，保羅喬治的進攻狀態，一直不是很穩定。

對於快船來說，這應該算得上是幸福的煩惱。隊里的得分手太多，球權分配是難以解決的問題。再加上隊內沒有傳統的組織後衛，難以將其他隊友的進攻調動起來，導致快船在場上進攻經常會出現斷電的情況。沒有一個明確的核心點，基本上是保羅喬治和路威還有小卡輪流進攻。這樣的進攻顯得沒有層次感，同樣在手感不佳的狀況下，對於球隊會有一定的危害。

可怕的是，在快船動不動就輪休倫納德的情況下，他們依然能在西部排名第二。說白了，快船並不在意西部第一的位置，相比來說他們更在意球員的健康，保證球員可以安全地出戰季後賽，季後賽才是他們的目標。至於常規賽，想必里弗斯的目標是西部前4拿到季後賽主場優勢就行了。
