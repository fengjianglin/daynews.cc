湖人開場被打15-4！八村塁4分鐘狂砍8分，戴維斯無情送血帽

天天要聞
3 週 ago

北京時間11月30日，NBA常規賽，湖人隊坐鎮主場迎戰華盛頓奇才。湖人隊仍然延續著良好勢頭，他們已經取得驚人的9連勝了，戰績來到16勝2負，制霸全聯盟。本場，坐擁主場之利，對手的實力又不是特彆強，10連勝是湖人隊唯一的追求。

但是，開場後，湖人隊卻遭遇當頭棒喝，他們沒想到客場作戰的奇才，一上來就那麼兇猛，完全是反客為主。在八村塁的暴走下，湖人隊開場被打了一個9-0，然後11-2，接著15-4，一直被死死壓制。到了15-4之後，本場的第一個暫停來臨，湖人隊才稍稍能夠喘一口氣。

在開場被碾壓的這4分鐘里，湖人隊的失誤太多了，屢屢被對手輕鬆打反擊，來去之間，分差就被拉開到了兩位數以上。

在11-2期間，八村塁一人獨得6分，奇才也主打他這個點，即便面對詹姆斯也絲毫不怵。又一次，八村塁想要籃下暴扣，不過這一次他悲劇了，碰上了憤怒的安東尼-戴維斯，直接硬生生將他摁下，八村塁狼狽摔倒在地，霸氣的濃眉哥還是兇狠地瞪了一眼躺在地上的八村塁，非常提士氣。

這一球，讓奇才、讓八村塁認識到了湖人隊的可怕，巨星就是巨星，暫時不發威，卻不是病貓，忍受不了被一次次捋虎鬚。

但八村塁不氣不餒，下一回合又殺回來了，右側輕鬆上反籃命中，砍下開場4分鐘里的第8分，也正式將比分帶到了15-4，湖人隊確實經歷了一個狼狽不堪的開場。

不過，作為全聯盟榜首的球隊，湖人隊絕非這個水平。果然，暫停回來之後，湖人隊就馬上還了對方一個14-2。湖人隊擁有詹姆斯、安東尼-戴維斯這兩大巨星，就擁有一種底蘊，他們有後發制人的實力，是任何對手都不敢掉以輕心的。
