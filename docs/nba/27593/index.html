24小時連傷6人！兩大狀元郎先後倒下，場均25+8+4新星無限期休戰

2019-12-22 830條評論 2,219次閱讀 781人點贊

北京時間12月21日，在過去的24小時裡面，對於NBA來說是黑暗的，因為接連有6位實力強勁的球星出現了傷病。首先是來自於籃網的大衛-努瓦巴，在昨天籃網對陣馬刺的比賽中，他在毫無外力衝擊的情況下意外受傷，隨後被確診為右腿跟腱撕裂，賽季報銷。本賽季努瓦巴作為替補登場，在場均僅能夠獲得13.3分鐘出場時間的情況下，得到了5.2分2.3籃板，命中率52.1%，三分命中率42.9%，是一名非常優質的角色球員。

其次，就是剛剛宣布復出的太陽隊狀元核心德安德烈-艾頓，被禁賽了25場好不容易得到了復出的機會他，沒想到在首場比賽就遭遇了右腳踝扭傷，今天對陣雷霆的比較他就無緣上場了，至於啥時候能復出，目前還沒有確切的時間。本賽季艾頓只出戰了2場比賽，場均能夠砍下18分11.5籃板1.5助攻，命中率54.5%，絕對有潛力能夠成為下一個全明星中鋒。

接近著就是另外一位狀元郎，他是來自於森林狼的唐斯。根據跟隊記者Jon Krawczynski帶來的最新消息，他由於膝蓋受傷，將缺陣接下來數場比賽，至於什麼時候復出，同樣跟沒有定數。本賽季唐斯迎來了自己的巔峰賽季，場均能夠轟下26.5分11.7籃板4.4助攻1.1搶斷1.3蓋帽，命中率51.4%，三分命中率高達41.8%，是一名非常適用於小球時代的四、五號位。

最後就是最近倒了霉運的猛龍隊，在上一場對陣活塞隊的比賽中，他們同時有三人受傷。第一個是建隊核心西亞卡姆，他遭受了跟上賽季詹姆斯一模一樣的傷病——腹股溝拉傷，根據球隊官方給出的最新消息，他接下來將會無限期休戰。本賽季他場均能夠轟下25.1分8籃板3.6助攻1搶斷1蓋帽，命中率45.7%，三分命中率39.2%，基本已經提前預定了明年的全明星陣容。

第二個則是球隊的第4得分手諾曼-鮑威爾，他由於左肩半脫位，跟西亞卡姆一樣，也將接受無限期休戰的安排。本賽季他場均能夠得到14.4分3.9籃板1.7助攻1.1搶斷，命中率49.5%，三分命中率38.9%。

第三個就是冠軍中鋒馬克-加索爾，在上一場對陣活塞隊的比賽中，他就曾因傷提前退場。而如今根據官方帶來的最新消息，小加索爾的傷勢被確診為腿筋拉傷，接下來同樣也將缺席多場比賽。作為一名34歲的內線，本賽季他的狀態確實下滑了很多，但是防守能力依舊是聯盟的頂尖水平。截止到目前，他場均能夠砍下6.6分6.6籃板3.4助攻0.8搶斷1.1蓋帽，命中率36.4%，三分命中率35.2%。

毫不誇張地說，在24小時內傷的這6個人，艾頓+唐斯+西亞卡姆+鮑威爾+小加索爾+努瓦巴，他們都可以組成一套奪冠陣容了。最後我們也希望聯盟的傷病潮趕快退散，受傷的球員能夠儘快回來。
