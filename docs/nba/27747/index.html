連克洛杉磯雙雄，密爾沃基雄鹿隊，詮釋了什麼才是矛與盾的結合體

2019-12-20 3,150條評論 4,097次閱讀 89人點贊

選題背景：

北京時間12月20日，雄鹿主場以111：104力克湖人隊，正式宣布球隊以25勝4負的戰績真正位列聯盟之巔。

揭秘——雄鹿隊成功背後的秘密！

戴維斯火線復出，字母哥力挽狂瀾

在此之前，雄鹿隊和湖人隊均以24勝4負的戰績位列東、西部的頭名，而今天的這場比賽，可謂是本賽季以來的世紀對決！

——巔峰對話，一觸即發！

在湖人輸給步行者賽後，缺席的戴維斯曾表示：「並不想看到球隊連敗，自己一定要幫助球隊扭轉頹勢。」

本場賽前，湖人官宣戴維斯將出席本場比賽。

  • 簡單回顧——整體實力不濟，湖人惜敗雄鹿

雙拳難敵四手，就是本場湖人敗北的真實寫照。縱觀整場比賽，雄鹿隊多點開花，10人有得分入賬，5人得分上雙，反觀湖人隊，僅有7人得分入賬，得分點過於集中。

巔峰對決

從整體來看，儘管湖人缺少庫茲瑪、考辛斯，但雄鹿同樣也缺少主力控衛布萊德索，用的來看，還是湖人的陣容厚度遠不及雄鹿。

數據方面：

湖人——戴維斯36分10籃板5助攻，詹姆斯21分12籃板11助攻，格林21分4籃板，波普18分3助攻，麥基4分3籃板。

雄鹿——阿德托昆博34分11籃板7助攻，米德爾頓15分6籃板，洛佩斯10分4籃板3封蓋，馬修斯13分4籃板，希爾21分，迪溫岑佐5分6籃板6助攻。

  • 三大因素——雄鹿力克湖人，勇奪聯盟第一寶座

本場比賽，雄鹿之所以能夠力克湖人，不光是字母哥一人的功勞，更是全隊的努力。總的來看，雄鹿的取勝之匙源於以下三點。

因素一：

字母哥本場的內外結合，可謂出神入化，防不勝防。本場比賽，字母哥一人獨得34+11+7的全面數據，其中三分線外更是8投5中，而單場5記三分也是刷新了個人生涯單場三分命中數的紀錄（此前為4記）。

所向披靡的字母哥

從三不沾到5記三分，字母哥的自信從來就沒有失去過。

因素二：

多點開花，雄鹿的多拳出擊，拳拳「致命」。

全隊10人得分入賬，雄鹿打得就是讓你找不到北。沒有防守重點，人人都能得分，是本賽季雄鹿的成功之道。

其中，替補出場的希爾更是表現驚艷，全場12投7中，三分球5投3中，獨取21分，他才是本場雄鹿取勝的X因素。

因素三：

攻防兼備，雄鹿隊亮出了他們的頂級防守。

本場比賽，從雄鹿隊對詹姆斯的防守來看，顯然他們賽前就做足了功課。從內線洛佩茲兄弟的鎮守，到外線馬修斯、字母哥等人的盯防，詹姆斯本場的效率並不高——全場19投8中，外加4失誤。

防守端的表現

尤其是到比賽末節，詹姆斯進攻端頻頻受阻，只能靠戴維斯艱難取分。

  • 小結

雄鹿隊的一星四射，星光閃耀，但他們的陣容厚度更讓人「垂涎三尺」，合理控制失誤，抓住每一個轉換進攻機會——這或許是他們的取勝之道。

而賽季至今，球隊有12人場均出場時間上雙，唯有字母哥出場時間超過30分鐘。或許正是這般的陣容深度，才能襯托出雄鹿隊18連勝背後的秘密。

雄鹿隊成功的背後，自有他們的致勝法寶

本賽季的雄鹿，儘管失去了布羅戈登，但好在他們補充了馬修斯、洛佩茲等人，球隊實力不減反增。

  • 一星四射——字母哥星光熠熠，攻防兩端愈加成熟

自上賽季獲得常規賽MVP後，本賽季的字母哥又是更進一步。本賽季至今，字母哥場均獲得31.8分（聯盟第二）12.8籃板（聯盟第六）5.3助攻1.3搶斷1.2蓋帽的全面數據。

字母哥歷年數據變化

歷年數據變化

作為2013年的新秀，在短短的7年間，從一位無名小卒到最快進步球員，再到最有價值球員，他的進步我們有目共睹。

每一年，字母哥都會給我們煥然一新的面貌。

上賽季，字母哥以高達60%的有效命中率在鋒線上已經是佼佼者。本賽季，字母哥在三分線上自信更是空前之高——賽季至今，字母哥場均出手三分5.1次，比之前任何一個賽季都高（上賽季2.8次），更值得振奮的是，字母哥的場均三分球命中率達到了近年來新高的33%，尤其是弧頂的三分甜點區。

本賽季，字母哥投籃散點圖

如果字母哥的三分球命中率再能提高到36%，那麼他將無解。

有了字母哥牽制，雄鹿隊其他球員才能萬箭齊發。

本賽季至今，雄鹿隊共有9人場均命中三分數超過1次——場均35.8%的命中率位列聯盟中游，但場均命中14.1個聯盟第二。

而場均25.9次（聯盟第七）的助攻，如此出色的轉移球能力，能夠幫助他們找到更好的出手機會。

本賽季，雄鹿隊投籃散點圖

本賽季的雄鹿，在常規輪換中，幾乎人人都能投三分，細看他們本賽季投籃分布，似乎「魔球理論」的觀念逐漸深入人心——內線和三分出手頻率令人刮目相看。

在布登霍爾澤的悉心條件下，彷彿讓我們看到了曾經號稱「東部馬刺」的老鷹隊。

  • 全員發力，陣容深度令人驚嘆

在布登霍爾澤的戰術體系下，最重要的就是全面，人人都能分享球，人人都能成為終結點，這樣的體系像極了馬刺，更何況，他們還有個希臘怪獸字母哥。

本賽季，雄鹿隊助攻網路圖

本賽季，湖人隊助攻網路圖

從上圖可以看出，雄鹿隊的進攻主要還是圍繞布萊德索和字母哥展開的，但不像其他隊那樣（如湖人隊）過於集中。由此可見，雄鹿隊的轉移球還是做得相當不錯，而場均26次的助攻同樣可以佐證。

出場時間的合理分配，這樣才能打出更具效率的進攻。

在雄鹿隊的陣容中，常規輪換就可達12人，陣容深度令人咋舌。別看場均得分上雙的只有3人（見下圖），但人人都能得分的雄鹿更加可怕。譬如今天對陣湖人的比賽，替補出場的希爾一上場便連續命中兩記底角三分，全場一人獨得21分；反觀湖人替補，全場僅僅4分，創造了本賽季聯盟替補單場得分新低，這或許就是兩支球隊本場所體現出的最大差距。

本賽季，雄鹿各球員數據

不僅如此，替補席上的康諾頓、伊利亞索瓦、布朗、科沃爾等人都是一把得分好手。

楊毅老師在評價本場比賽中就表示，「雄鹿替補陣容的衝擊力是他們本場最大的優勢，所謂的衝擊力，就是指速度、彈跳、爆發、力量。反觀湖人，又老又慢的杜德利，身高有限布拉德利和卡魯索。」

由此可見，雄鹿隊的優勢也就不言而喻了——首發如果還可以和我抗衡，那麼就來拼替補。

要知道，希爾作為久經沙場的老將（6支球隊，12個賽季），生涯場均可貢獻11.2分3.2助攻，三分球命中率達到38.3%，作為一名替補控衛綽綽有餘。伊利亞索瓦作為空間型四號位，職業生涯（6支球隊，14個賽季）場均可得到10.5分5.8籃板，但是他給我們最大的印象還要屬那一手三分和造進攻犯規。康諾頓作為衝擊力十足的鋒線，攻防兩端都能為球隊貢獻不少，本賽季，當他在場與不在場時，球隊攻防兩端的正負值為+6.9。

全民皆兵的雄鹿，乍眼一看，還傻傻地以為是馬刺隊…

  • 攻守兼備，矛與盾的結合才是他們的致勝法寶

不僅是進攻端，雄鹿在防守端的表現同樣是聯盟頂級。

1.天生麗質——字母哥

字母哥在防守端不用多說，天生就是一名防守悍將——身高臂長，出色的力量和移動能力，讓他能夠從1號位防到5號位。

本賽季，當字母哥在場時，球隊每百回合讓對手得到99.7分，不在場時，對手則可以得到107.5分。

2.洛佩茲兄弟

進攻端不力，防守端來湊。

可否知道，大洛還是全明星球員，可是本賽季的大洛，進攻端依然找不到准心，三分線外命中率還不到3成（29%），場均只入賬9.6分。但是，防守端的貢獻足以——限制對手禁區內命中率在5成以下，這一數據比當今聯盟最佳內線防守人濃眉哥還低，這一點就足夠了。

與此同時，小洛佩茲作為本賽季的新援，矜矜業業的搶好籃板就是他的本職工作。而在洛佩茲兄弟的保駕護航之下，再加上字母哥，雄鹿的內線禁飛區自然拔地而起。

有了如此的內線屏障，雄鹿的鋒線群大可在外線翻江倒海。

3.是金子總會發光——馬修斯

馬修斯在經歷大傷之後，儘管爆發力已經回不到當年，但他依然是雄鹿那個最香的餑餑。

在談到自己加盟雄鹿隊時，馬修斯表示「這支球隊已經足夠優秀，希望來了之後不給球隊添亂。」

本賽季至今，馬修斯場均出場24分鐘，得到7.7分2.4籃板1.3助攻。以打橄欖球出身的馬修斯，力量上在聯盟鋒線上絕對不會吃虧，當他在場時，球隊的防守效率為99.7，不在場時防守效率則為105.5——可以說，馬修斯在球隊防守端的作用不亞於字母哥。

譬如說，在今天對陣湖人的比賽，馬修斯在防守詹姆斯的任務上一馬當先，完全按照教練的旨意去做——繞前搶過，絕不換防——這讓詹姆斯始終難以擺脫馬修斯的貼防。

雄鹿隊攻防效率排名，橫坐標：進攻效率；縱坐標：防守效率

本賽季，雄鹿的防守效率已經來到了聯盟第一，這是建立在進攻效率同樣排在聯盟前三的基礎上做到的。

攻守兼備，當矛與盾相結合時，這才是真正的總冠軍特質，這就是他們的致勝法寶！

東部冉冉升起的巨星，未來是你的

當看到這樣的一張圖片時，不禁感慨時光易逝，詹姆斯之後，誰又會接過詹姆斯手中的火炬？

籃球火炬的傳遞

本賽季的字母哥，還在進步，而他的上限又在哪裡？

但，未來是你的！

可以肯定的是，本賽季的雄鹿，將會有更大的機會走得更遠，儘管東部豪強如雲，但真正能夠和雄鹿抗衡的也不超過兩支，要論球隊的整體實力以及陣容配置，雄鹿隊還是獨一檔。

重新起航

回顧上賽季的東決，或許給字母哥留下的不僅僅是失利，更多的是一種激勵，曾記否，當年詹姆斯同樣經歷過如此境地，正是輸給現隊友霍華德帶領的魔術。

所謂天時地利人和，本賽季的字母哥，確實迎來了爭冠良機。時機易逝，字母哥應該好好珍惜當下——手握致勝法寶，連克洛杉磯雙雄，如能保持，總冠軍還遠嗎？
