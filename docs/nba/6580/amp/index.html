NBA快船篇！矛與盾在手，總冠軍似乎勢在必得

天天要聞
3 週 ago

快船隊因為擁有萊昂納德與保羅–喬治這兩個聯盟頂尖巨星以及聯盟最強替補組合哈雷爾和路威，他們被外界看作是本賽季最大的奪冠熱門，奪冠概率甚至比湖人隊還高。賽季初由於萊昂納德的輪休與保羅–喬治的傷病，快船隊並沒有展現出太強的統治力，而在這期間湖人隊卻狀態火熱，排名穩居西部第一併與快船拉開差距。正是因為這種差距，很多人認為快船揭幕戰擊敗湖人只是運氣好，他們並沒有那麼強，其實不然。

萊昂納德與保羅-喬治雙巨頭

萊昂納德在一波短暫的輪休之後終於開始與保羅-喬治一起並肩作戰，從兩人第一場對陣凱爾特人到最近的一場對陣獨行俠，這四場比賽他們都取得了勝利，並且越打越好。從這幾場比賽中我們可以看到快船所展現出的很多東西，他們的進攻策略，防守策略，往細的說就是各種擋拆配合，防守針對等方面的東西。不過，我認為這些都不是多麼大的優勢。最感到震撼的是快船隊的火力總是源源不斷，這就是他們最大的優勢之一。

萊昂納德

快船首發球員中擁有萊昂納德與保羅–喬治這個兩個得分高手，目前前者場均25.8分8籃板5.4助攻，後者場均24.8分5.9籃板4助攻。對於這兩人來說，數據並不能詮釋他們的能力，因為如果有必要，他們都具備得分30+的能力。至於替補方面，快船擁有路威和哈雷爾這一對組合，目前前者場均22.5分3.6籃板6.2助攻，後者場均18.9分7.6籃板2.4助攻。路威和哈雷爾這兩人雖與首發雙巨頭在個人能力方面差距較大，但也具備場均25+的能力。這些數據不僅說明快船得分能力強、得分點多，更說明了他們可以接管比賽的人很多，萊昂納德可以，保羅–喬治可以，路威可以，甚至有時候哈雷爾也可以。

快船替補雙核之一：路威

在萊昂納德與保羅–喬治並肩作戰的這四場比賽中，球隊分別拿下了107分，122分，134分，114分。這樣的數據不是在面對弱隊時拿下的，而是在面對凱爾特人、火箭、獨行俠這樣的強隊時拿下的，這說明快船的得分能力超強。從實力比賽情況來看，一方面對手無法限制快船得分。對手即使使出全力來防守，也最多只能限制萊昂納德與保羅–喬治之一，根本不可能同時限制兩人，而且這也可能會使得快船其他球員有更多的空位進攻機會。而路威和哈雷爾這兩個替補得分手，對方也無法有效的限制，畢竟他們在替補球員中算是實力頂尖的。

萊昂納德與保羅-喬治

另一方面，快船的火力源源不斷。萊昂納德、保羅–喬治、路威、哈雷爾這四人都具有接管比賽的能力，每一場比賽也都至少有兩人能夠站出來，而如果有三個人站出來，那比賽可能已經結束了。具體到其中的一場比賽，那就是誰能夠投進球誰來打，投不進了就換下一個，這樣的火力從首發延續到替補，從第一節延續到最後一節，總是源源不斷。

快船替補雙核：哈雷爾和路威

說完火力，再說說快船的防守。快船的防守就如同他們的進攻一樣犀利，單防能力和整體防守能力都很頂尖。現在是常規賽，雖然這種防守能力並未完全體現，但毫無疑問是存在的，而且是他們最大的優勢之一，也是對手最忌憚的地方。

快船在外線擁有貝弗利、保羅–喬治和萊昂納德三位防守能力頂尖的球員。貝弗利在面對絕大多數球員時都沒有身高和體重優勢，但此人很善於貼身纏鬥，而且很拼，這會給對方核心球員造成很大的體力壓力與心態壓力。保羅–喬治與萊昂納德這兩人的防守方式不同於貝弗利，這兩人擁有身高，體重和臂展，只要速度不落後對手，基本都可以給對手造成很大的投籃壓力、運球壓力以及突破的壓力。除此之外，快船在必要的時候也會對對方核心球員進行全場包夾，這在對陣單核陣容球隊時最為有效。

快船防守悍將貝弗利

在昨天對陣獨行俠的比賽中，快船對東契奇進行重點防守甚至包夾，最終使得東契奇僅僅14投4中拿到了22分8籃板6助攻，還出現了7次失誤，遠遠低於他在之前比賽中的發揮。對陣獨行俠的比賽可以看作快船季後賽防守策略的一個縮影，他們一定會想方設法去限制對方核心球員的發揮，單防不行就採用包夾的方式，他們有足夠的人員可以擔此重任，這樣會對對手造成很大的打擊，最起碼發揮會受到影響。

獨行俠東契奇

想要取得比賽的勝利，尤其想要在季後賽這種高強度模式下取得勝利，無非就是拼進攻和防守，打好自己的進攻而限制住對方的進攻就能最終贏得比賽。快船隊擁有最強的進攻火力，也因為有首發雙巨頭和替補雙核而使得在進攻端很難被針對和限制。在防守端，快船的外線整體防守和單防都很頂尖，這使得他們在季後賽會給其他球隊的外線球員造成很大的麻煩，甚至是毀滅性打擊。

如今陣容齊整的快船雖磨合不佳，但依然給人一種「我可以限制你，而你限制不住我」的感覺，這種強大可以媲美前幾個賽季的勇士隊。不過，快船想要奪冠並不容易，因為擋在他們前面的還有湖人這座大山。湖人在詹姆斯和戴維斯的帶領下穩居西部第一，他們最大的優勢是內線，而快船最大的劣勢也是內線，兩隊碰撞，內線的發揮將很大程度決定比賽的勝負。
