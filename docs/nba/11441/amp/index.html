勇士用拉塞爾和雷霆交易，換來亞當斯，提升內線能力？

天天要聞
2019-12-11

​

雷霆將亞當斯放上交易架，目的是為了獲得更多的未來資本，目前雷霆手中已經有很多個選秀權，而目前勇士的情況很複雜，由於擁有了庫里和湯普森，拉塞爾顯得有一些多於，這對於勇士來說是一個不錯的選擇，他們是可以選擇和雷霆進行交易的，他們可以將其他中鋒全部送走，雷霆還表示，為了獲得未來資本，他們是可以接收一些負資產，這對於勇士目前薪金狀況來說，是一個不錯的選擇，亞當斯提供的內線防守能力，可以讓勇士重回防守強隊行列！

​

亞當斯今年26歲，和勇士這一批巔峰球員差不多，而且兩年之後，他的合同將會到期，拉塞爾交易走，換來亞當斯，就兩人對換，勇士也可以騰出300萬左右的薪金，由於雷霆想要年輕資本，而拉塞爾今年23歲，和目前他們球隊中的年輕球員亞歷山大差不多，而且亞歷山大未來是打二三號位，所以和拉塞爾不衝突，而且勇士還可以順便將考利斯坦送走，又騰出兩百萬左右的薪金，勇士甚至還可能從雷霆手中獲得一個首輪選秀權！

​

若勇士能夠交易來亞當斯，未來他們內線擁有格林，魯尼，亞當斯，這樣的組合防守端得到了保證，亞當斯內線終結能力很強，而且亞當斯還是有策應能力，擋拆能力也很強，勇士只要籃板球得到保證，庫里和湯普森在外線就會獲得更多的出手空間，亞當斯搶前場籃板球的能力很強很強，而且更加重要的一點，就是當庫里被突破之後，內線有了足夠的保障，庫里在外線防守還是很強的，一旦突破之後，有格林和亞當斯在內線，對方很難完成進攻！

​

亞當斯對於勇士來說是一個非常不錯的選擇，他比之前勇士的中鋒都要好太多，而且還可以開發亞當斯更多的能力，對於勇士來說，即使留下亞當斯，格林庫里和湯普森，下賽季薪金超過這個賽季的開拓者，替補陣容也不會太差，而且勇士手中還有一些年輕球員，這些年輕球員在這一個賽季會得到很多機會，肯定會有一些球隊看上這些球員，也可以換來一些不錯的角色球員，交易來亞當斯，目標就是總冠軍，勇士可以交易這些年球員，明年勇士選秀順位很不錯，也是交易籌碼之一！
