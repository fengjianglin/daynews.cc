湖人返回洛城備戰木狼存三憂一喜：隆多傷情不明，悍將未痊癒

2019-12-08 0條評論 18次閱讀 0人點贊

在湖人大勝開拓者後，全隊踏上了回洛杉磯的航班，全隊都顯得有些疲倦，不過還是挺放鬆的，畢竟在負於獨行俠後取得了艱難的客場三連勝，三個對手還都不是魚腩球隊，分別擊敗了掘金、爵士、開拓者，9分險勝掘金，25分大勝爵士，23分大勝開拓者，這三場比賽之後沒有人再言湖人為「偽強隊」，什麼只會打弱旅遇到強隊猛龍、快船、獨行俠就輸球，這支湖人的硬實力得到了普遍認可，接下來他們將在主場備戰森林狼的挑戰，森林狼目前西部第八也是強隊，不過對於湖人來說備戰森林狼就是艱難的開始，有三個隱患在等待湖人。

拉簡-隆多的傷情就是第一個隱患，他在對陣開拓者的上半場結束後向隊醫表示自己左腿酸痛，隨即隊醫檢查隆多左腿筋拉傷，回到洛杉磯後將進行更詳細的檢查，目前還沒有結果，隆多的傷情不明，腿筋拉傷一般來說需要傷停2-4周，就算是很輕微的拉傷也不可能立馬就復出，湖人接下來的賽程可不松，隆多這個目前湖人最准空位投手+板凳組織核心傷停對於湖人整體的輪換、硬實力都有很大影響，隆多本賽季場均8.5分5.8助攻3.5籃板1搶斷，命中率46.2%，三分命中率52.8%，場均命中1.5個三分球，空位三分命中率高達61.5%，缺少了他對於湖人來說當然是巨大的隱患。

布拉德利因為右腿的骨裂已經缺席了三周的時間，之前預計的2-4周看起來至少也要四周了，根據NBA記者蒂姆-麥克馬洪的報道，布拉德利將對他的右腿傷勢進行重新評估，若評估通過，他大概率會在12月13日（北京時間12月14日）對陣熱火的比賽中復出。也就是說他必然缺席對陣森林狼的比賽，隆多+布拉德利都缺席對陣森林狼，沃格爾需要讓庫克、丹尼爾斯兩人承擔更多的責任，本賽季庫克與丹尼爾斯狀態時好時壞，而且防守表現很差。最慘烈的是若布拉德利評估失敗，他將繼續在傷病名單中不能登場。

假設隆多傷勢較為嚴重，布拉德利的傷情評估失敗，那第三個噩耗就格外扎心，因為在12月9日湖人主場打了森林狼後，是一個五連客之旅，沒錯剛剛打了三連客在主場小憩一場就要開啟五連客，雖然這一次五連客沒有背靠背，但魔術、熱火、步行者、雄鹿都是東部前八，雄鹿跟湖人還都是聯盟戰績第一的球隊，隆多與布拉德利的缺席對於接下來的魔鬼客場之旅存在更嚴重的影響。

當然湖人也有一個好消息，在這一波三連客之後，湖人的進攻也變得強悍起來，湖人進攻效率升到了111.8聯盟第五，從開賽初期的聯盟第11提升到了聯盟第五，湖人最強的防守也拾回102.8聯盟第三，攻防效率都進入了聯盟前五，這一點跟雄鹿隊很像，雄鹿是攻防效率都進入聯盟前三的球隊，湖人接下來的客場之旅要碰雄鹿了，雄鹿剛剛擊敗了快船，湖人VS雄鹿之戰也被譽為總決賽預演，若隆多與布拉德利都不能回歸，那湖人贏球的概率真不大。
