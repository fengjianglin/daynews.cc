三大球星三分球11中0，球迷卻把輸球的責任歸咎於兩屆最佳主教練

天天要聞
2019-12-18

昨天雄鹿隊輸給沒有東契奇的獨行俠爆出不小的冷門，畢竟雄鹿隊在此之前已經取得了18連勝，面對沒有東契奇的獨行俠應該是唾手可得的勝利。強如雄鹿都能輸給沒有東契奇的獨行俠，只能說明NBA中無弱隊，任何強隊只要大意那麼必將遭受懲罰。在雄鹿輸球之後，有球迷把責歸咎於兩屆最佳主教練布登霍爾澤。

實事求是地說基德發掘了字母哥的天賦，而真正帶給字母哥質變的還是布登霍爾澤。可是本場比賽之後，有球迷分析失利的原因說：雄鹿三分在極其不準的情況下，布登霍爾澤竟然還在第四節放任球員賭手感，在三分線外浪投，而不去打防守反擊和衝擊內線，所以主教練布登霍爾澤需要為輸球被鍋。

球迷所言不無道理，雄鹿隊全場三分球41投僅11中，命中率低至26.8%，其中三大球星三分球11中0（分別是：大洛佩茲4中0、迪文森佐3中0、希爾4中0）。可是雄鹿隊在第四節還有誰能夠持球突破，曾經是字母哥和布雷德索輪番持球強突，可是在布雷德索受傷之後，字母哥又沖了一整場了也已經精疲力竭了。

雄鹿輸球最大的原因，應該是突如其來的傷病打亂了布登霍爾澤固有的部署。雖然布登霍爾澤及時把迪文森佐提上首發的位置，但是迪文森佐還是無法完全接過布雷德索的攻防任務。即使迪文森佐送出了10板9助攻的准兩雙數據，但是迪文森佐沒有布雷德索的衝擊力，始終無法分擔字母哥衝擊內線的責任。
