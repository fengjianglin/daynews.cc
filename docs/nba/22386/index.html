打籃球僅僅只是副業，原來NBA竟有這麼多低調的富二代 你知道幾個

2019-12-19 524條評論 2,296次閱讀 1332人點贊

現在看NBA中大多數的明星都是家財萬貫的，但在他們的原生家庭條件很不好，像詹姆斯、哈登、杜蘭特這些球星，在他們的童年都是靠母親一個人來支撐起整個家庭。但也有原本就是富二代的球星，打球賺取的薪金對他們來說就像零用錢一樣，無關重要，因為他們的家境很富裕。

第一位薩博-塞弗羅沙，目前效力於NBA休斯頓火箭隊，職業生涯的巔峰時期是在雷霆時期，當時是雷霆三少中不可缺失的得力助手，防守端下手非常的快，但是他的名字還是讓人很難記住。

但薩博-塞弗羅沙的家境可以說不是一般的好，薩博·的父親帕特里克·塞弗羅薩是來自南非的音樂人，母親克麗絲汀是瑞士的一位畫家，外公更是瑞士的一家連鎖店CEO，曾表示要買下塞弗羅沙效力過的所有球隊，如果以一支球隊15億美金的價格計算，那麼初步推斷他家的資產應該超過45億美金，而塞弗羅沙一年500萬的年薪，確實像是零花錢一樣。

第二位內內-希拉里奧，效力於NBA休斯頓火箭隊，今天已經37歲的內內出生於足球國家巴西，在巴西是最出名的籃球代表，而內內也不斷的在巴西推廣籃球文化，因為有一天籃球能和足球相提並論。

但內內還有一個身份就是巴西國內大型企業的接班人，父親的公司佔據了巴西市場的一半，父親也一直在要求內內回國接班，但即便內內生涯受過很多的傷，在08年的時候還被檢查出睾腫瘤，但發現的及時，切除掉之後就沒有事了。比起金錢來，內內還是很鍾愛籃球的。

第三位就是大衛-李，現在已經退役了，而他的父親是美國一家大型連鎖餐飲業的CEO，而他的爺爺曾光是捐款就高達7000萬美金。

最後一位艾爾-法魯克-阿米奴，目前效力於NBA奧蘭多魔術隊，艾爾-法魯克·阿米奴繼承了奈及利亞的皇室血脈，他的名字「艾爾-法魯克」的意思是「君主降臨」
