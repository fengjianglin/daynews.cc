北京時間12月14日，NBA常規賽的比賽繼續打響，熱火隊在主場迎來了湖人隊的挑戰，兩支強隊之間的強強對話，這個賽季的湖人一路高歌排名西部第一，熱火隊新賽季則是扮演了黑馬的角色，熱火主場現在更是堪稱魔鬼主場，新賽季打到現在，熱火主場11連勝，主場至今未嘗敗績，而湖人客場戰績也是十分不錯，目前客場比賽12勝1負，本場比賽湖人這邊凱爾-庫茲馬因腳踝扭傷繼續缺席本場比賽。

前熱火隊名宿波什也是來到現場觀看本場比賽，賽前詹姆斯與昔日好友深情相擁，首節的比賽開始後萊昂納德三分拿到場上首分，巴特勒搶斷一條龍上籃打中，熱火取得了5：0的開局，詹姆斯助攻麥基暴扣得手為湖人打破僵局，格林三分命中，羅賓遜三分回應，熱火開局打的氣勢如虹，納恩突入內線隔著麥基封蓋左手暴扣得手，詹姆斯三分命中後湖人將分差追成了12：12平，兩隊的進攻打的大開大合，十分的好看。

瓊斯三分球命中，熱火保持著微弱的領先優勢，詹姆斯搶到籃板後一條龍上籃不中，麥基釘板大帽希羅，詹姆斯得球後突破上籃出現失誤，熱火隊打出反擊巴特勒得手，詹姆斯也是在跟裁判示意對方有腳踢球，但是裁判並沒有理會，對於裁判的表現，詹姆斯有點氣憤，回過頭來他強突內線造犯規兩罰一中，比賽還剩下3分鐘的時候，在沒有任何徵兆的情況下詹姆斯直接回更衣室了，很多球迷在猜測詹姆斯是不是受什麼傷了，過了一會詹姆斯重新回到場上，這也是讓人鬆了口氣，布拉德利拋投命中，湖人取26：25反超了比分，這也是湖人本場比賽第一次取得領先，兩隊分差交替，卡魯索突破禁區上籃命中，第一節的比賽打完，湖人30-29領先熱火隊1分。
