今夏最失敗的交易出爐！2000萬場均2+3，波波維奇的操作不敢恭維

2019-12-02 0條評論 17次閱讀 0人點贊

本賽季馬刺沉淪與今年夏天的交易有千絲萬縷的關係。馬刺上賽季的失利本想今年夏天通過交易來補強，然而卻事與願違。

今年夏天馬刺為了得到心儀的莫里斯復出了慘重的代價，賠了夫人又折兵形容波波維奇老爺子最合適不過了。

莫里斯放了馬刺的鴿子。雖說莫里斯與馬刺達成協議，馬刺與莫里斯簽訂一份2年2000萬的合同但是最終莫里斯選擇和尼克斯簽訂一份一年1500萬的合同。如果真的能夠簽下莫里斯馬刺的外線和防守一定會得到很大的提升，莫里斯生涯三分命中率36.8%，本賽季高達52.4%聯盟第一。畢竟NBA屬於商業聯盟球員選擇自身的最大利益這也是合情合理的事情，只能為老爺子感到無奈啊。

送走了貝爾斯坦。可以說馬刺出品必是精品，本賽季加盟奇才後貝爾斯坦徹底打了波波維奇老爺子的臉。替補出場的貝爾場均得到12.8+4.5+1.8的數據，投籃命中率45%三分命中高達44.1%。要知道像貝爾斯坦這樣空間型的高大內線在這個小球時代是很吃香的。然而本賽季馬刺之所以打出這麼糟糕的戰績原因之一就是外線三分不準。兩位當家球星多以三分線以內的2分見長，從而造成了內線的擁擠，試想一下如果有貝爾斯坦的話情況就好多了。

為了得到莫里斯馬刺送走了貝爾斯坦和一個簽約權得到籃網的猛獸3D卡羅爾。並與卡羅爾簽訂一份2000萬美元的合同。卡羅爾這位老將上賽季仍然保持了34%的三分命中率，同時擁有不錯的防守能力。馬刺簽下卡羅爾就是為了補強外線的火力問題和防守，看似卡羅爾就是非常合適的人選。然而事與願違，卡羅爾本賽季出現了低迷。本賽季一共替補出場9次場均只能得到2分3個籃板，投籃命中率只有28%三分命中率低至25%，這樣的數據只能說慘不忍睹。

本賽季到目前為止馬刺7勝14敗的戰績排在西部的倒數第四，雖說賽季剛進行了1/4的樣子但是照這樣發展下去連續22賽季闖進季後賽的紀錄就是在本賽季結束了。這也讓我不僅想起了波波維奇老爺子交易走了小卡換來了德羅贊。原先一直佩服的老爺子現在對他的操作真的不敢恭維了。
