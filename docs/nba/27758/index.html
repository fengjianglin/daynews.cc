兩悍將遭逐 快船被逆轉！里弗斯喊冤：裁判該保護路威 卻驅逐了他

2019-12-20 1,729條評論 3,794次閱讀 488人點贊

​快船被火箭完成15分大逆轉，貝弗利和路威被驅逐出場，主教練里弗斯賽後直言球員們需要冷靜。

「我們全隊失去了冷靜，這場失利對我們來說是個很好的教訓。我寧願現在就開始出現問題，而不是在季後賽。我們要從中吸取教訓，尤其是技術犯規方面，那是我們能控制的。」里弗斯說。

本場比賽，貝弗利六犯離場，他因不滿判罰與裁判理論，最終連吃兩個技術犯規遭到驅逐。

除此之外，貝弗利和威斯布魯克同樣有過節，貝弗利被罰出場後，威斯布魯克做出再見手勢，裁判果斷給了威少一個技術犯規。

里弗斯開玩笑說：「我們得保護好更衣室，如果那麼做，一切就相安無事。」

本場比賽，喬治得到34分和9個籃板，倫納德得到25分和9個籃板。

第四節比賽，路易斯·威廉姆斯因不滿裁判凱恩·菲茨傑拉德，吃到了兩次技術犯規，也被驅逐出場。

「我認為凱恩應該保護他，卻讓路易斯離場了。路易斯火冒三丈，但沒法挽回了。這種情況不該在第四節發生，無論怎樣，我們在第四節都不該吃到技術犯規，更不用說是我們最好的得分手了。不管路易斯對還是錯，我們是時候向前看了。」里弗斯說。
