NBA：失去對抗，變得索然無味，才知道喬丹時代才是真神

2019-12-01 0條評論 13次閱讀 0人點贊

競技體育的魅力就是身體的對抗，把身體的機能發揮到機制，呈現最好的狀態，才是它誘人的地方。

小時候看拳擊比賽，一點都不喜歡，就兩個大老粗在場上揮拳相加，其實對於拳擊來講那就是一種力量的表現，瞬間的爆發力可以擊倒對方，一片歡呼。

NBA新賽季開賽至今，受到一個腦殘經理人的影響，NBA和中國的關係降到了歷史的冰點，過去的幾個賽季我們都在討論現在的NBA越來越沒意思，再加上現在的局面，再去看NBA比賽，發現這個時代的NBA已經沒有多少觀賞性了。

過去的幾年是勇士王朝最輝煌的幾年，那一個再十幾年前是聯盟魚腩的球隊，找到了一個捷徑，三分球。三分球的效率遠遠高於兩分球，中距離，以及中鋒的籃下強打，於是勇士隊開啟了真正的小球時代，雖然在很久的時期裡面，德安東尼的球隊就是快打旋風，不過那個時候贏得進攻不足以拿到總冠軍，彼時的德安東尼不合時宜。

勇士在科爾的調教下，庫里和湯普森成為了這個時代的英雄，他們不需要在籃下強打，不需要持球突破，只需要等在三分線外舉手投籃。籃下的肉搏已經沒有了，中鋒成為了藍領球員，不要得分，只需要你搶下籃板，得分交給外線的投手門吧。勇士成功了，而且建立的勇士王朝，沒辦法，其他的球隊想要打敗勇士，就必須把進攻拉到三分線以外。NBA的新時代來了，屬於小球的新時代，然後你就會發現，現在的NBA比賽就像在球場上跑步一樣，投籃，向前奔跑，再投籃，再繼續向前奔跑，不需要戰術，不需要團隊，僅僅一點點掩護就可以，舉手投籃，比賽結束。

索然無味，現在的NBA比賽有點像看中國足球一樣，中國足球永遠沒有帶球前進，互相傳球過中場，只有一大腳，或者足球天上飛來飛去。

人們再討論庫里的歷史地位，庫里再小球時代是不錯的球員，得益於體系的建立，沒有了小球時代的體系，他什麼都不是，簡單地說，他不在勇士，他就是一個普通的球員。能不能進名人堂，按照籃球的發展來講，他取得的成就是可以進名人堂的，但是要比肩歷史上那些偉大的球星，是無法比擬的。

這一個時代，我們才能感覺到，什麼是真正的籃球之神，喬丹之後的NBA，就是一直再走下坡路。喬丹時代的NBA才是真正的籃球，那是一個講究身體對抗，身體素質，團隊協作，無論是中鋒和後衛，每一個位置都不可以弱化。為什麼喬丹是神，而不是球星，他是再萬千巨星的面前能夠終結比賽的人。雖然我們沒有像這個時代，看過喬丹的每一場比賽直播，但是我們隔著屏幕能夠感覺到，那個時代的籃球才是真正的籃球。

後喬丹時代的籃球開啟了中鋒和後衛模式，在湖人王朝期間，奧尼爾和科比的搭檔，一個內線一個外線，大殺四方。就如姚明時代也是一樣，和天才麥迪的搭檔，再繼續雙塔式時代出現，猶如馬刺的王朝也是有鄧肯這樣出色的內線作為支柱。再後來的NBA開啟了抱團模式，凱爾特人三巨頭再次改變了聯盟的模式，因為光靠一兩個人無法拿到總冠軍。

籃球是快速的發展，也許母親的籃球確實適合當下時代的潮流，但是它缺少了太多的精神，人們已經不再像以前那樣留念它了，那種激烈的對抗，戰略戰術的運用，那種團隊的協作和分工。在今天被完全的簡化為三分，三分，以及快速進攻和奔跑，失去 籃球的本真，索然無味。

所以，討論多年的喬丹和詹姆斯，誰的成就高，兩個人的成就都很高，但是，喬丹這一尊神是真的無法超越。這個時代的任何球員在喬丹的時代，都會被虐成渣渣。
