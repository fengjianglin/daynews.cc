凱爾特人油漆區防守聯盟第二 黃蜂雙後場效率感人，客場或將慘敗

2019-12-22 990條評論 3,036次閱讀 1116人點贊

夏洛特黃蜂 vs 波士頓凱爾特人（主）

夏洛特黃蜂上場比賽107-114輸給爵士，最近4場輸了3場，更令人擔憂的是黃蜂進攻端的狀態。自12月14日85-73擊敗公牛開始，黃蜂百回合得分只有101.7分，排名聯盟同期倒數第三位。這和球隊近期的傷停有一定的關係，PJ華盛頓的缺席對黃蜂有比較大的影響，本賽季PJ華盛頓場均12.3分，三分命中率40.6%。他的缺席導致黃蜂進攻端效活力下滑，這是黃蜂進攻效率降低的一個重要因素。

​

此外，本賽季表現驚艷的格雷厄姆在被提上先發後效率有明顯下滑，雖然他依舊維持了19.1分7.5助攻的基礎數據，但投籃效率並不理想，目前真實命中率只有55%。另一名控衛羅齊爾效率更差，真實命中率更是只有53.5%。現在的黃蜂進攻不利，而且比賽節奏還是聯盟第二慢的球隊，他們很難打出很高的得分。

波士頓凱爾特人上場比賽是114-93戰勝活塞，連續兩場贏球。凱爾特人這賽季更多的還是以防守帶動進攻，球隊百回合失分是聯盟第五少的103.7分，球隊場均在油漆區的失分只有41.2分，聯盟第二少。而黃蜂這邊場均卻有44.8%的進攻來自油漆區，凱爾特人的防守剛好對上了黃蜂進攻的強點。

進攻端，凱爾特人這賽季表現不錯，百回合得到110.8分，但海沃德的傷病總是斷斷續續，這會影響到球隊進攻端的穩定性。目前海沃德依舊是存疑狀態，而且就算他復出，比賽狀態也很難保持在一個很高的水準上。

本場推薦210.5小分。

彩蛋凱爾特人讓12.5分。
