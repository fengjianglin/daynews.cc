離開湖人的年輕人，哪個更適合現在的湖人，球哥重複，克6最合適

2019-12-23 1,214條評論 3,061次閱讀 1561人點贊

這幾年，湖人其實選了不少的年輕人，蘭德爾、克拉克森、拉里南斯、德安吉洛拉塞爾、隆佐鮑爾、布蘭頓英格蘭、約什哈特、凱爾庫茲馬、卡魯索、托馬斯布萊恩特等。但最終留下來的只有庫茲馬和卡魯索，當然他們都為湖人的重新崛起做出了巨大貢獻。

但是如果回頭看，當年離開湖人的這些年輕人，哪個會比較適合現在的湖人呢？

首先，表現比較好的應該是拉塞爾、英格拉姆和蘭德爾，他們都具備在其他球隊打首發的實力，拉塞爾甚至還入選了全明星。但是從目前湖人的陣容和戰術來看，這三個可能都不太適合如今的湖人，拉塞爾和英格拉姆還是比較需要球權的，在如今的湖人，他們並不需要一個長期持球的球員，他們需要一個替補的第三得分點，既能夠引領第二陣容的得分，又能在首發啞火的時候挺身而出。

這個模板其實很明顯了，就是隔壁的路易斯威廉姆斯，讓拉塞爾和英格拉姆去打這個角色，還是有點屈才了，勢必引起他們的不滿，影響球隊氛圍。所以從需求講，克拉克森也許是最適合湖人的球員，無論是在科比時期，還是在騎士，克拉克森一直都非常敢投，而且穩定性還不錯，要比庫茲馬高出很多。

首發詹姆斯、丹尼格林、布拉德利、安東尼戴維斯、麥基，替補隆多、克拉克森、波普、庫茲馬、霍華德，整個替補陣容的分工很明確，隆多是絕對控球核心，霍華德是防守核心+內線終結，庫茲馬、克拉克森和波普就負責跑出機會，出手投籃就行，湖人在有些場次比較被動，就是因為沒有第三個得分點，克拉克森的到來可以稍微緩解這個問題，他可以稱為窮人版的路威。

而球哥鮑爾和約什哈特，其實功能已經有點重複了，有詹姆斯和隆多，湖人完全不需要球哥，一個外線不穩定的傳統控衛。而拉里南斯、布萊恩特只會是球隊的第三內線，雖然合適，但不足的出場時間同樣會引起他們的不滿，所以如果可以選一個回來，最合適的還是克拉克森，當然了，如果選名字的話，可能可以選布萊恩特，畢竟跟那個名宿同名。
