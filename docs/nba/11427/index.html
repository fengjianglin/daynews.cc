熱火三巨頭早期的核心是誰？總決賽最後時刻 只有韋德敢對抗命運

2019-12-12 586條評論 2,357次閱讀 1115人點贊

很多老球迷都說現在的聯盟發生了很多的改變，這種改變不只是打法的改變，更多的是球員的改變，他們見過2001年總決賽上艾弗森的勇者無畏，他們見過科比在三連冠時期的輝煌，他們見過熱火的三巨頭時代，也見過勇士隊的忽然崛起。

這個夏天，韋德和諾維茨基雙雙離開了NBA，他們二人在很多人的青春當中都扮演了重要角色，江山代有才人出，老將們總會離開這個聯盟，然後漸漸從球迷們的視線遠離。

三巨頭時期的熱火隊曾像勇士隊一樣，連續在多個賽季里統治著聯盟，那時的韋德、波什以及詹姆斯都正處於職業生涯的巔峰期，想要在季後賽里打敗這支球隊是一件很困難的事情。

隨著波什和韋德的相繼退役，熱火三巨頭如今只剩下詹姆斯還在聯盟里征戰，不過很多熱火隊的球迷至今仍會議論誰才是熱火三巨頭時期的核心球員。

從整體表現來看，詹姆斯是那時當之無愧的球隊核心球員以及領袖，但在三巨頭剛成立的時候，在逆境中站出來拯救球隊的球員常常是韋德。

在與小牛隊總決賽G6最後的關鍵時刻，熱火隊在比賽還剩下不到5分鐘結束時，落後著小牛隊8分，但熱火隊接連幾次的進攻都顯的很猶豫，包括詹姆斯在內的許多球員都不斷傳導著籃球，沒人願意背負壓力投籃，這個時候站出來的仍然是韋德，當時央視的解說員便說道打到最後只剩韋德了。

雖然熱火隊最終還是輸給了小牛隊，但韋德向所有人證明了他的大心臟，熱火隊魂非浪得虛名。
