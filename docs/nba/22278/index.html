32＋5＋3！肯巴·沃克打瘋了！獨行俠群俠沒有再度創造奇蹟

2019-12-20 1,498條評論 2,963次閱讀 1871人點贊

北京時間12月19日，NBA常規賽又迎來了一場焦點之戰——凱爾特人vs獨行俠，最終凱爾特人以109:103，客場險勝獨行俠。本場比賽的看點是缺少東契奇的獨行俠能否再度創造奇蹟，因為在上一場比賽中他們把不可一世豪取18連勝的雄鹿斬下馬來。本場比賽又面對東部第二的雄鹿隊，那麼本場比賽的看點就是獨行俠的團隊籃球能否再度創造奇蹟呢。

從歷史戰績看獨行俠在近期5場比賽中只取得了一場比賽的勝利，總戰績還是綠衫軍領先。本場比賽獨行俠依舊是以外線三分為主要進攻點。前兩節他們投得還算中規中矩，但是第4四節命中率下滑導致最終輸掉了比賽。

其實雙方都知道贏球的關鍵就是誰掐住對方箭頭人物誰就可以取得比賽的勝利。上一場很遺憾獨行俠沒有掐住字母哥，讓字母哥內線翻江倒海。但我放你字母哥別人卻如夢遊沒有得分。而雄鹿隊是哪個人都有防住。最終獨行俠成功阻止了雄鹿的18連勝。

本場比賽焦點在於綠軍遏制住群俠，還是獨行俠遏制住肯巴·沃克。在第一節和第二節比賽獨行俠有個表現出色的X因素就是塞斯·庫里。前兩節他的三分球頻繁命中，而反觀其他人的發揮就沒有之前對著雄鹿噴井的表現。獨行俠一看行了，別人不管就管你賽斯·庫里！讓你接球算我輸！果然在綠衫軍精心套路下，獨行俠塞斯庫里明顯在第三和第四節需要追分的時候命中率和出手次數下降。而綠衫軍的轉折點出現了2次。第一次在是在第二節比賽結束4:50秒的時候，沒錯又是塞斯·庫里一記違體犯規讓一直領先的獨行俠縮小比分。

而第三節獨行俠前半節表現不錯，後半節卻莫名啞火，綠軍抓住機會打出一波14：4的小高潮，一舉反超比分，末節綠軍在拉鋸戰中不斷擴大領先。也就說綠衫軍就只用了這兩次的轉折點擊敗了獨行俠隊。

本場比賽獨行俠依舊是沒有防住關鍵人物肯巴·沃克。不僅如此這回沃克隊友也很給力，布朗26分18投8中，塔圖姆24分23投10中，聯合砍下50分。沒錯字母哥看到綠衫軍沃克隊友的表現也會哭泣吧！我的隊友可不太可！但是獨行俠這次群俠沒有像上場比賽6人得分上雙，波爾津吉斯+哈達威+布倫森+庫里一共得到66分！emmmm….數字挺6但戰績不太6。獨行俠輸球也就在所難免了。不過群俠這幾場的表現確實出色，現在的獨行俠可以和任何強隊五五開！
