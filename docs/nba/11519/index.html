泡椒大戰老東家絕不手軟，卻遭球迷如此對待？

2019-12-10 208條評論 1,159次閱讀 74人點贊

步行者上場在客場險勝尼克斯以後，今日在主場迎來西部第二的勁敵快船，而快船在對陣雄鹿的焦點之戰中大比分輸掉比賽以後，迅速調整好狀態在昨日以大比分戰勝了奇才。今日東部第六與西部第二相遇，這也是泡椒本賽季加盟快船後首次回到戰鬥了七年的印第安納波利斯，步行者的球迷會有如何反應呢？我們都很好奇。

縱觀歷史，泡椒在2010年以第十順位被步行者選中後從第二年開始每一場都在首發陣容里（除了2014年在備戰奧運會時的隊內訓練遭遇了小腿90度骨折缺席了該賽季的比賽），可見步行者對泡椒的重視以及球迷對泡椒的熱愛，或許「愛之深，責之切」便是球迷們在泡椒兩年前離開步行者後一直很難原諒他的原因吧。

今天註定是一場精彩的對決，剛剛到達球迷入場時間座位就漸漸被填滿，在比賽開始前介紹球員階段，當播報員讀到泡椒名字的時候，一向比較文明的步行者球迷發出了震耳欲聾的噓聲和NBA球賽中經常出現的干擾球員的謾罵聲，看來兩年的時間並沒有磨滅步行者球迷對兩年前泡椒離開轉會雷霆選擇的失望和憤怒。畢竟當初步行者用了七年的時間將泡椒從一個順位第十的球員培養成一位聯盟出色的球星，而他卻做出了這樣的選擇。

不過泡椒畢竟是NBA巨星，除了那些沒有原諒的球迷，現場還是有很多不遠千里從洛杉磯趕來看比賽的球迷和鍾愛泡椒而原諒他的選擇的步行者球迷，只要他進球現場還是會有很多支持者頂著壓力站起來叫好。即使對陣老東家，泡椒也絕不手軟，上半場上場17分鐘便拿下全場最高21分以及5籃板2助攻1搶斷，第三節繼續趁熱打鐵拿下13分，最終總分拿下36分贏得比賽。或許這是泡椒想要在舊主面前證明自己的方式吧，尊重對手最好的方式便是全力以赴。

觀察現場球迷發現今天現場又來了不少身穿紅色球衣的球迷，他們來自哪裡又為了誰而來呢？一看今天對陣的球隊便想起快船隊一位出色的板凳球員蒙特雷茲·哈勒爾來自路易斯維爾大學，雖然當初選秀順位不高，但這五年進步明顯，尤其是從火箭轉會快船後上場時間的增加激發他的潛能，作為替補為球隊他一直有著可觀的貢獻。

今日是快船隊在本賽季常規賽第一次在印第安納波利斯打球，離這裡不遠的路易斯維爾大學的球迷當然願意來到現場支持一下老校友。而今天哈勒爾的表現也並沒有讓大學校友們失望，在第一節最後和第二節前兩分鐘里僅僅8分鐘時間裡9投4中得到了8分外加2個籃板。整場比賽哈勒爾總計拿到26分和8個籃板1個助攻，無疑成為全場最佳板凳球員，一個球隊是否具有絕對的競爭力板凳的深度也是一個重要評判之一。在賽後看到哈勒爾抱著兩個孩子走向球員通道，一個身穿路易斯維爾大學球衣的球迷遞給哈勒爾一件球衣希望他能簽名，令人驚訝的是他真的接過球衣說他回更衣室會簽好給她，也許大學情懷永遠是一個球員永遠無法拒絕的銘記。

賽後記者採訪環節，在問到是否對步行者球迷全場噓有如何感受，泡椒回答到：「這樣的噓聲激勵著我，讓我更有力量也讓比賽更加有趣。」他今天的表現也的確如此，另外他還回應了廣大球迷都很關注的關於當初離開步行者的原因，他回答到在未來的某一天會公布一切，或許到那個時候該被噓的那個人不是我。

泡椒當初為何離開已經無足輕重，如今他已經是聯盟前十的巨星和一名優秀的領導者，拋開他與步行者的糾葛放眼他的成長，無一不在告訴我們他很強，他還會更強。他會證明每一個質疑過他的人他們錯了，他也一定會對得起每一個支持他的人，未來可期。
