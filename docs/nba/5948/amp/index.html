順位不代表未來，盤點近十年已遭棄用的樂透秀

天天要聞
2019-12-08

NBA選秀向來是個技術活，每支球隊都十分注重這個環節的工作。NBA聯盟官方，以及每支球隊都有自己的球探團隊，他們的數量龐大並且極為特殊，他們的工作非常重要、不可代替，也不為人知。他們根據自己細緻的觀察對可能進入NBA聯盟的球員做出預測，他們的預測很大程度上就決定了這些球員未來的選秀順位。

百密一疏，這些球探也不是站在上帝視角，他們總會有出錯的時候。即使他們給球隊表達自己的了解，認為某位球員可以在NBA取得成功，但有時候存在太多變數，即使再高順位的球員，他們也有可能在未來一事無成。那麼今天我們就來盤點一下近十年來，表現的最差的幾位高順位球員吧。

  • 韋斯利·約翰遜：偏科嚴重的防守悍將，難成時代逆流

現在談及這個名字，想必很多球迷第一時間想起來的是他被哈登晃倒的窘況：在2018年3月1日快船的一場比賽里，哈登用一個後撤步將韋斯利·約翰遜晃倒在地，並且遲遲等了五秒鐘才出手投籃。你可以說哈登進攻技巧高超，但你也必須承認約翰遜防守態度專註。

實際上，這位2010年首輪4號秀，能在這個位置被選走就是因為他的防守。他在讀完大三才參加選秀，預測的模板是慈世平，由此我們就可以看出這位球員並不是擁有足夠的潛力才讓他成為了四號秀。卡梅羅·安東尼的這位師弟，在大三那年場均貢獻16.5分、8.5個籃板、1.66次搶斷和1.83次封蓋，兩項命中率分別：50.2%和41.5%。對於這位球員的未來，專家預測他能成為極品3D型球員，在那個時代能有如此的前瞻性眼光，實在非常棒了。

不過，約翰遜進入了NBA聯盟以後就像完全忘記了如何投籃一般，這位科比最信任的小弟之一依然在防守端活力無限，但是他的進攻簡直災難，整個職業生涯只有40%出頭的命中率。其實在2018年離開快船以後，他就處在這個聯盟的邊緣了，最終在今年4月初被奇才裁掉，並在今年夏天選擇加盟了希臘的籃球聯賽。

  • 德里克·威廉姆斯：進攻端的猛獸，始終找不到自己的定位

2011年選秀大會，來自亞利桑那大學的德里克·威廉姆斯力壓如今克萊·湯普森、科懷·倫納德、肯巴·沃克和吉米·巴特勒等多位巨星，成為那一屆的榜眼秀。關於這位活力四射的前鋒，他最大的賣點就是他出色的身體素質。

這位前鋒在大二那一年場均貢獻19.5分和8.3個籃板，憑藉萬夫莫當的身體素質，他在大學聯賽的禁區為所欲為，雖然只有2米03的身高，但是他擁有出色的力量和彈跳，在身體對抗能力較弱的大學聯賽，威廉姆斯就是十足的霸主。

但是在NBA，一切情況都發生了變化。首先，他得為自己做出明確的定位，就像大學時期充當四號位的「糙男」倫納德，到了NBA依然要乖乖頂替三號位，並刻苦磨練自己的技術。大學聯賽的溫床讓威廉姆斯一時間難以找到自己的定位，打四沒技術，打三也沒有身體優勢，這讓他左右為難。同時，他來到的是全世界最頂尖的職業籃球聯賽，這裡可沒人會在身體對抗上怵他，身體素質的優勢也不再明顯。沒有技術、身體優勢不明顯，讓這位前鋒未來的籃球之路越來越窄，終於在2017年離開騎士後，難以繼續在這個聯盟生存。

  • 吉默·弗雷戴特：不世出的籃球天才，在NBA卻鬱郁不得志

在大學聯賽，吉默·弗雷戴特早已名聲在外，人送外號「寂寞大神」，最終在2011年首輪第10順位被選走，但是這位球員如今卻已經在CBA聯賽效力三年。

關於這位天才後衛，真的是身體限制了天賦的典型，這位後衛在NCAA擁有無比光輝的履歷：大四那年場均28.9分、4.3次助攻和1.32次搶斷，三項命中率分別為45.2%、39.6%和89.2%，帶領楊百翰大學取得20勝1負的成績。他在2010年夏天就入選了美國國家隊陪練隊，在2011年被題名為西部山區聯盟年度最佳球員，同時也被CBSSport題名為全美年度最佳球員。

但是五年的NBA職業生涯，弗雷戴特場均只能貢獻6分，命中率為41%。在對抗水平更高的聯賽，弗雷戴特身體的弱勢被體現得淋漓盡致，他只有1米88的身高，而且爆發力非常平庸，他幾乎沒有憑藉爆發力過人的能力。如果他能高10厘米，他完全可以成為像雷迪克那樣的投手，但是他太矮了，只能出任一號位。然而身體對抗能力太差，他幾乎會被對位的身體素質猛男生吞活剝。

  • 安東尼·本內特：史上最水狀元，進球對他來說太難了

在2013年選秀大會開始前，誰都不曾預料到安東尼·本內特會當選狀元，或者奧拉迪波，或者穆罕穆德，亦或諾埃爾，總之輪不到本內特，然而騎士隊就偏偏用狀元簽選走了本內特。當騎士隊日後回憶起這件事時，他們解釋說是本內特在試訓時言行舉止彬彬有禮，正是這吸引了管理層。

雖然本內特在NBA進個球都費勁，生涯毫無成功可言，但是本內特的大學生涯還是可圈可點，他在選秀前也是穩定的小綠屋成員。他在大一時場均就能貢獻16.1分、8.1個籃板和1.23次封蓋。某種程度上，這位前鋒有幾分勒布朗·詹姆斯的影子，能三能四、可攻可守，本內特頗為全能。

不過在進入了NBA以後，他的毛病一覽無遺，他的身體素質在NBA大環境下顯得非常平庸，跑跳能力並不出色。同時防守能力太差了，在外線因為自身的體重跟不上小個子的腳步，然而在內線受限於身材，護框能力極為有限。尷尬的定位，讓本內特逐漸迷失自己，並在四年後就離開了這個聯賽。

放眼這些不能適應NBA聯賽的高順位秀，他們或者是受限於身體素質，或者就是因為技術不夠精湛。能像倫納德那樣完成完美轉型的球員也寥寥無幾，畢竟天賦和努力很小概率會出現在同一個人身上。
