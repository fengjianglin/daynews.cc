尼克斯進攻勢頭良好 約基奇引領掘金快攻

2019-12-15 82條評論 1,787次閱讀 533人點贊

紐約尼克斯vs丹佛掘金(主)

紐約尼克斯上場比賽客場103-101擊敗國王，西部之旅連贏兩場。尼克斯最近的改觀是在進攻上，佩頓的作用至關重要。佩頓最近兩場場均15分4.5助攻，場均出場時間也提升到了28.5分鐘，而尼利基納則變成了一個假首發。這樣尼克斯將有更多的時間讓佩頓來控制節奏，這對球隊的進攻來說益處多多。不過佩頓的防守一直備受詬病，他在防守端的影響力顯然沒有尼利基納那麼高。

此外，尼克斯擁有聯盟最實用的鋒線群，蘭德爾、莫里斯、諾克斯、波蒂斯，這些人都充滿了能量，這會給掘金的防守製造困難。特別是馬庫斯-莫里斯本賽季展現出了極強的空間能力，本賽季他場均能命中2.8個三分，命中率高達48.2%。這賽季的百回合進攻正負值也達到了+1.3，莫里斯將成為打亂掘金防守體系的關鍵人物。

丹佛掘金方面，今天是110-102戰勝雷霆，尼古拉-約基奇28分14籃板12助攻，賽果麥克-馬龍對約基奇也是大加讚揚，他表示「5年了，我第一次把快和約基奇放同一個句子里」，從賽後的數據看，當約基奇在場時，掘金回合數能多出1.26個，這在往常是不多見的。

此外，由於米爾薩普受傷，格蘭特升到了第一陣容，這樣球隊的內線護筐能力有所下滑。普拉姆利反倒沒有分到更多的時間，進攻能力較強的埃爾南戈麥斯得到了一些機會。最近麥克-馬龍教練對進攻的重視程度還是很高的，掘金近期在進攻端會有一波反彈的勢頭。

本場預設大小分為205.5分，推薦大分。
