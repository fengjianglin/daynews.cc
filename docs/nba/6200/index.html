輪休？明天輸球？湖人至少6人生病，戴維斯真是硬漢

輪休？明天輸球？湖人至少6人生病，戴維斯真是硬漢

文/董體育

湖人隊今天雖然客場105：96擊敗了西部第二的掘金隊，但是真正的考驗在明天，湖人隊背靠背要飛往鹽湖城挑戰爵士隊。不得不說布拉德利傷停之後歸期未定，讓湖人隊的外線防守捉襟見肘，如今可謂是屋漏偏逢連陰雨，根據NBA記者Dave McMenamin報道，詹姆斯賽後估計隊內至少6名球員生病了，包括他自己、戴維斯、拉簡-隆多和凱爾-庫茲馬。主教練沃格爾表示這是一種流感，不幸的是球隊內的很多球員都感染上了，今天的戴維斯帶傷出戰砍下了25分10籃板的數據幫助球隊拿下比賽，不得不感嘆戴維斯真的是一個硬漢。

詹姆斯接受了採訪，談到隊友安東尼-戴維斯帶病堅持比賽的表現，詹姆斯說：「他整年都是這麼做的。帶傷出戰，像今晚這樣帶病出戰。他是年度最佳防守球員，今晚他又在低位單防約基奇時展示了這點。出現換防時，防到賈馬爾-默里，一對一防守小個球員，一次又一次防守成功。他打出了野獸般的一場比賽。」格林接受採訪談到了帶病出戰的安東尼-戴維斯格林說：「他是我們的領袖之一，他不只用言語還用行動發揮領袖作用。當然，他以前也抗爭過一些傷病，他還一直忍受了扭傷和磕碰。他有理由輪休，但他不想缺席任何比賽，他想抗爭一切。他不只讓我們知道他站在我們身後，也讓聯盟知道他不會以任何理由輪休。」

不得不說戴維斯就是一個硬漢的代名詞，有充分的理由去輪休，有充分的理由讓自己恢復身體，但是作為一個職業球員，既然能出戰比賽他就不缺席，為他這種職業精神點贊，可是湖人隊面對大面積的球員生病，會不會選擇輪休呢？35歲的詹姆斯一直堅持打高強度比賽，明天的背靠背會不會因為生病戰略性輪休呢？這個我們不得而知，這個要看他自己的身體狀況和球隊的安排，但是有一點是肯定的，帶病出戰比賽肯定會影響球員的狀態，明天湖人隊客場與爵士隊的比賽恐怕是要輸球了吧！

目前湖人隊18勝3負位列西部第一的位置，而爵士隊12勝9負排在西部第六，可是已經2連敗的爵士隊肯定不願意接受3連敗，並且面對目前聯盟當紅球隊湖人隊，他們肯定會奮力一搏，來證明自己的實力。不過爵士隊也有傷病影響，根據NBA記者Andy Larsen，爵士官方更新傷病報告，後衛邁克-康利因腿筋緊張將缺席明日主場對陣湖人的比賽。屆時兩支球隊會擦出什麼樣的火花，我們拭目以待。湖人隊球員遭遇的並非是傷病，或者是球隊的輪休，而是美國如今也流感爆發，多名湖人球員已經不幸中招，因為對於感冒，NBA球員都採用的是不吃藥的自愈療法，因此恢復過程會很緩慢。明天上午10:00與爵士隊開打，我們拭目以待！
