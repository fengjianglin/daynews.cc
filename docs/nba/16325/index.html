波波維奇：在我結束執教生涯之前，我想執教詹姆斯一次

2019-12-13 63條評論 1,971次閱讀 152人點贊

時光悄然飛逝，才短短几個月，NBA新賽季常規賽就已經過去將近三分之一了，上賽季結束後諾天王與韋德離開了這個賽場，這個賽季結束後加索爾以及卡特很能有可能也會離開這個賽場，除了這些巨星的退役，執教巨星們的這些教練也在一天天老去，比如現役年齡最大的NBA主教練波波維奇。

作為馬刺隊史的傳奇教練，他的整個執教生涯為馬刺帶來了無數的榮譽，波波維奇整個執教生涯也見證了無數個巨星的誕生與離去，在這20多年的執教生涯里波波維奇為馬刺帶出來一支鐵打的團隊，作為波波維奇的愛徒鄧肯，兩人聯手更是為馬刺奪得5座總冠軍，不僅如此，波波維奇還帶出了帕克、吉諾比利以及倫納德這幾位巨星。

但是現在波波維奇也即將面臨退役，但是老爺子在退役之前還有一個心愿，那就是想親自執教詹姆斯一次，近日波波維奇接受採訪表示道：我個人非常欣賞以及喜愛詹姆斯，這是大家都能看出來的，我整個執教生涯中執教了鄧肯、帕克、吉諾比利以及倫納德這些巨星，但在我結束執教生涯之前，我想執教詹姆斯一次。

波波維奇喜歡詹姆斯這是廣大於眾的，兩人私下的關係也非常好，屬於那種亦師亦友的關係，此前當詹姆斯受到質疑時，波波維奇也總能站出來為其出頭，在公開場合時每當提到詹姆斯，波波維奇都會讚不絕口，即使兩人也曾在總決賽中作為對手針鋒相對，但每次相遇後，波波維奇都會與詹姆斯來一次親切的擁抱，或許詹姆斯早已將波波維奇當作自己的禪師，而波波維奇看待詹姆斯更像是看待自己孩子一樣喜愛。

​但最遺憾的是兩人沒能在NBA中聯手，這兩位在各自領域都是屈指可數的巨星，沒在一起合作真是聯盟的一大遺憾，光是兩人的球商就足以震懾整個聯盟，如果在09-10賽季結束後詹姆斯沒有去邁阿密，而是去了聖安東尼奧，恐怕現在詹姆斯早已經不是三冠王了，對此球迷們有什麼看法呢？
