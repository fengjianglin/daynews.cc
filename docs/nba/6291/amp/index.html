西部“雙星”群雄割據！“詹眉”場均51.8分並非第一

天天要聞
2 週 ago

相比較字母歌主宰的東部聯盟，顯然西部競爭更加激烈。自本賽季勇士徹底瓦解之後，巨星抱團的時代一去不返，聯盟的生態也更加平衡。尤其是西部聯盟，多支球隊打造「雙子星」組合，讓整個西部進入群雄割據的時代。

賽季至今已經進行了將近四分之一的賽程，西部各支豪強依舊是殺得難解難分，這對於我們球迷而言無疑是一場饕餮盛宴。而提到雙星組合，洛杉磯湖人可謂是雙星球隊的典型代表。休賽期成功引進濃眉哥戴維斯，讓兩位狀元得以聯手。截止目前，兩人的組合也的確產生了良好的化學反應。兩人場均能夠貢獻51.8分，這樣的輸出算得上穩定而高效。然而儘管兩人能夠場均聯手貢獻50+，但仍舊不是得分最高的雙子星。現在我們就來看一看西部幾組雙星組合的表現。

第五位：東契奇&波爾津吉斯

場均47.8分19.1籃板11助攻 奪冠指數：★★

年僅20歲的東契奇在本賽季打出了現象級的表現，他的打法和全面的能力儼然不像一位二年級生能夠做到的。截至目前，東契奇場均能夠貢獻30.6分9.9籃板和9.6助攻的超級准三雙數據。獨行俠老闆馬克庫班對於東契奇的喜愛更是無以言表，甚至說出了「就算離婚也要留住東契奇」的狂言。

事實證明庫班的選擇是正確的。錦上添花的是，賽季初庫班再次引入歐洲球員波爾津吉斯，這對「歐洲雙子星」在本賽季打出了極好的表現。目前兩人場均聯手貢獻47.8分，同時還能搶下19.1個籃板以及送出11次助攻。在雙子星的帶動下，獨行俠本賽季勢頭強勢，以13勝6負的戰績排在西部第四。

要知道，東契奇年僅20歲，波神也僅有24歲，獨行俠的前景一片大好。

第四位：利拉德&麥科勒姆

場均49.3分9.4籃板11.1助攻 奪冠指數：0

上賽季西部決賽開拓者慘遭勇士橫掃，儘管本賽季開拓者引入大白邊彌補內線努爾基奇的空缺，然而賽季初開拓者似乎依舊沒有從上賽季的失利中緩過神，球隊的表現完全不同於人們的預期，甚至一路跌到西部倒數第二。球隊的進攻選擇依然過度依賴後場雙槍，截至目前，開拓者雙槍場均能夠貢獻49.3分9.4籃板以及11.1助攻。

論球隊後場，開拓者絕對不懼怕聯盟任何一支球隊，然而鋒線上的攻防能力幾乎是聯盟墊底。好在球隊簽下了曾經的全明星小前鋒安東尼，這在一定程度上讓球隊的配置更加均衡。如今的開拓者正逐漸找回感覺，以一波三連勝將成績提升至第11位。雖然本賽季依舊奪冠無望，但相信如今的開拓者「三叉戟」能夠帶給我們更多驚喜。

第三位：倫納德&喬治

場均49.3分14.1籃板8.9助攻 奪冠指數：★★★★★

上賽季還僅僅算得上一支草根球隊的快船，在季後賽金州勇士的地板搶下兩場勝利讓人們不禁刮目相看。休賽期，快船在背後的驚人運作瞬間讓球隊成為奪冠大熱門。倫納德與喬治的聯手加盟也給同城湖人帶來了巨大的衝擊。

相比其他雙星球隊，快船的整體陣容顯然更加合理。賽季至今，兩人場均可以聯手貢獻49.3分14.1個籃板以及8.9次助攻。雖然兩人的得分不是最多的，但千萬不要忽視快船的板凳陣容，目前快船以53.3分的替補得分排在聯盟首位。因此，快船也是本賽季最接近總冠軍的球隊。

第二位：詹姆斯&戴維斯

場均51.8分16.5籃板14.3助攻 奪冠指數：★★★

當戴維斯宣布加盟湖人之後，這對「雙狀元」組合迅速吸引了所有人的關注，而湖人對也把賽季目標直接對準總冠軍。論實力，兩人在聯盟絕對數一數二。兩人不僅擁有超強的得分能力，戴維斯還守護者球隊的籃板，而詹姆斯卻貢獻著生涯最高的場均10.9次助攻，他們讓湖人的鋒線深度領跑聯盟。賽季至今，兩人聯手能夠場均送出51.8分16.5個籃板和14.3次助攻。

無法否認兩人的全面，但也無法忽視兩人的自身不確定因素。詹姆斯的年齡讓賽季後期的狀態變得未知，而戴維斯自帶的玻璃屬性也無疑是漫長賽季的X因素。除此之外，隊內除了雙星之外，能夠站出來的球員並不多。如今湖人隊內第三得分手是庫茲馬的11.3分，除此之外再無球員得分上雙。這無疑給本賽季力爭奪冠的湖人蒙上陰影。

第一位：哈登&威斯布魯克

場均61分13.7籃板14.7助攻 奪冠指數：★★★

要說聯盟「雙星」得分誰最強，那還得看休斯頓「雙MVP」組合。兩大數據狂人聯手，必定會掀起一場腥風血雨。賽季初，也許我們還會擔心火箭的球權分配問題，然而現在來看，之前是多慮了。曾經連續三個賽季場均三雙的威少來到火箭做出了很大的讓步，這也是最正確的選擇，畢竟這裡不是雷霆，火箭是哈登的球隊。

截至目前，兩人能夠聯手送出61分13.7個籃板以及14.7次助攻，這樣的耀眼數據也讓火箭在西部始終保持著較強的競爭力，如今火箭以13勝6負的戰績與獨行俠並列西部第四。

如今的西部沒有了勇士的一家獨大，每支球隊都試圖在總決賽中分一杯羹。這樣的比賽正是我們最期待的，我們也相信之後西部的硝煙會更加瀰漫。

（文/愛籃球的老李）
