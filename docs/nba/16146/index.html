「樂動體育」看來快船這次真累了，公牛以109-106取勝

2019-12-16 269條評論 3,378次閱讀 1749人點贊

芝加哥公牛隊在爆出了幾次輸掉比賽的領先優勢後，在今天得以翻盤。

公牛今天以109比106戰勝人手不足、疲憊不堪的洛杉磯快船隊的比賽中，扎克-拉文得到31分，並將一場決定性的三分球換下。

芝加哥隊在不到兩分鐘的時間裡落後5分。在106平的情況下，公牛隊在還剩不到6秒的時間內將比分扳平。拉文在三分線附近拿到球，右路開球，在還剩2秒時被蒙特雷茲-哈雷爾犯規。

「我們演了很多大戲，」拉文說我想這就是它的意義所在，拍戲。

「我們一直打得很好，但我們只是在比賽的最後兩分鐘、三分鐘沒能取得勝利。希望我們能開始把它們串起來

勞里·馬爾卡寧得到13分和17個籃板，塞迪斯·揚得到17分，丹澤爾·瓦倫丁得到16分。

哈雷爾得了30分，喬治為洛杉磯得了27分。快船隊連續贏了四場。

公牛隊在第三節早些時候繼續大漲，以75:61領先，連得17分。這75分比芝加哥在周五晚主場以83-73輸給夏洛特的比賽中多出2分。

快船隊以12-1的比分將比分差距縮小到76-73，並在第三分鐘的最後一分鐘由哈雷爾三分球扳回81-79。拉文隨後命中3分，芝加哥隊以84比79領先進入第四名。

最後3分鐘，喬治罰中兩個罰球，以98分打破平局，蘭德利·沙默特在不到3分鐘的時間裡命中3分，以103比98領先快船。

洛杉磯在一分鐘後上升了五分，拉文和瓦倫丁連續命中三分球——夾在喬治的兩個罰球中的一個——在還剩47.9秒的時候以106平。
