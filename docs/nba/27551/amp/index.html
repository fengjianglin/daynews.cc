就算少了三名首發，單節也能虐你17分，冠軍相球隊依然是他們

天天要聞
2019-12-22

猛龍前幾天一下傷了三名大將，未來的超巨西亞卡姆無限傷停，後場首發後衛鮑威爾也無限傷停，內線首發小加索爾也要傷停數周，這麼多火力點瞬間消失了。

難道要像勇士一樣，重演悲劇嗎？兩個衛冕冠軍球隊都要一夜之間墊底嗎？

不少球迷看衰猛龍，但掌柜也分析說了，他們不會輕易地認輸，他們依舊擁有衛冕冠軍的底蘊，他們還有洛瑞、范喬丹、伊巴卡這幾位大神，怎麼會輕易的認栽呢。

掌柜說了，他們要站出來得分，面對強敵奇才，他們真的做到了！122-118頑強地戰勝了奇才！

有些球迷就留意了對面奇才的八村塁，有沒有什麼表現，很可惜的是，他也被隊友誤踢，腹股溝傷了，至少缺席五場，最近場均13分，作為一個新秀來說，還算不錯的。

再來看看猛龍的頂樑柱，洛瑞這門小鋼炮，寶刀未老，砍下26分9助攻1搶斷，火速救援的伊巴卡也貢獻了23分10籃板3蓋帽，內線真少不了你，小加索爾你還是打替補吧，哈哈哈。

我們的范喬丹發揮穩定，也貢獻了自己的場均值表現：18分8助攻1搶斷。

阿努諾比18分7籃板。

第一節， 比賽就呈現一邊倒的局面，猛龍這邊火力真是猛，怪不得姓猛名為龍，單節40分，這趕上人家兩節得分了，反觀奇才23分，17分的差距，開局挖得也太大了吧。

猛龍全隊打出了非常流暢的配合，這一節，全隊的手感都不錯，命中率高達61.9%，個個都是神槍手，再看奇才這邊，29.17%的命中率，你們好意思拿出來嗎？這不等著被虐嘛。

猛龍精神領袖洛瑞一人就拿下12分，展現出了一名老將的風采，這是半場未結束，就要提前進入垃圾時間了嗎？奇才也不至於首節就繳槍了吧。

第二節他們開始站穩的陣腳，手感的天平傾向了奇才這一邊，命中率上升到了40%，不停的衝擊籃下，獲得了多次罰球機會，這一節雙方算是打了個平手，奇才29:28猛龍，追回了一分。

第三節奇才的內線優勢開始顯露出來，一個伊巴卡怎麼搶得過對方，奇才這一節就搶到了16個籃板，前場籃板就搶了5個，這一節又追回了不少分，奇才也是一支打硬仗的隊伍，不然怎麼叫勁敵呢。

看奇才這狀態，這是要上演超級大逆轉的劇本了？四節比分，只有第一節落後太多了，前面挖的坑太大了，後面三節就算比多方多幾分，也難以補上了，奇才教會了我們，不要把第一節不當比賽看，不要把首節當熱身，除非後面三節能打出首節這樣逆分差，不然你就等著奇蹟吧。

眼看首節手握巨大的勝利優勢，這三節時間，不知不覺就被消耗殆盡了，這對於傷兵滿營的猛龍來說，很傷士氣，這可是提升士氣的開頭啊。

奇才最後一節繼續狂飆三分球，猛龍的失誤開始增多，採用了衝擊籃下的戰術，還好獲得了15次罰球機會，只罰丟了3個，最後算是艱難抗下了奇才這一波強大的反攻，打成這樣，猛龍的冠軍底蘊，你還值得懷疑嗎？

讓你三名大將，我們也能繼續打出自己的東西，即使你打順風球，我們也能跟你打硬仗，拼下來。

奇才的比爾也挺失落的，眼看能上演翻盤的好戲，只能接受空砍37分的結果。

但不難看出猛龍的失誤比較多，外線的命中率也掉得快，急需外線投射的補強，相信三名首發回歸後，能繼續扛起猛龍這面大旗！

（羅掌柜）
