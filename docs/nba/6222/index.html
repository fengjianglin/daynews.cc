火爆！詹姆斯球迷和約基奇球迷場外鬥毆，同行女伴也相互扭打

2019-12-04 0條評論 14次閱讀 0人點贊

NBA常規賽，一場西部強強對話，洛杉磯湖人隊客場挑戰丹佛掘金，最終他們以105-96戰勝對手，鞏固了西部第一的排名。兩隊上一場都輸了，本場就力爭避免連敗，結果湖人在多人生病的情況下，還是艱難戰勝了對手，著實不容易。

兩位「全能戰士、三雙達人」勒布朗-詹姆斯和約基奇之間的對決，也是本場比賽的一大看點。儘管兩人並不直接對位，詹姆斯更多對穆雷，而約基奇要去死磕安東尼-戴維斯。但兩人作為各自球隊的核心，是否能有效組織進攻並帶隊獲勝，也是考量的一大方面。最終還是詹姆斯更勝一籌。而就在兩人在場上拼個你死我活的時候，兩人的球迷在場外也不閑著，也爆發了激烈衝突。

有網友拍到一段小視頻，在掘金看台上，一名身穿詹姆斯湖人23號球衣的小伙兒，和一名身穿約基奇掘金15號球衣的壯漢，爆發了激烈衝突：二人先是爭吵，接著在女伴率先動手後，也終於忍不住了。

身穿約基奇球衣的女伴先動手的，然後身穿詹姆斯球衣的女伴還擊，接著23號球迷瞅准機會，對著15號球迷的頭部，來了重重一拳。令人意外的是，15號球迷竟然沒有太激烈的回擊，而是繼續指著對方打嘴炮兒。倒是23號球迷女伴依依不饒，往上走的時候還不忘再給對方一拳。

隨後，旁邊的觀眾趕緊過來拉架，嘴裡大喊著「NO」，勸雙方都控制一下情緒，別那麼大火氣。雙方這才罵罵咧咧各自走開，這一幕真是太火爆了。不得不說這名23號球迷還是挺有膽的，這可是掘金主場，就不怕觸犯眾怒被群毆嗎？其實，掘金球迷對詹姆斯的態度還是挺友好的，甚至在他最後罰球的時候，還高喊「MVP」，但詹姆斯不能代表球迷，約基奇也不能，所以每個人的行為還是要由自己負責。
