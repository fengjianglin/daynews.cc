26分+單臂隔扣！倫納德3度造訪，終攻克聖城 噓聲中通殺30個客場

天天要聞
2019-12-23

北京時間12月22日，2019-20賽季NBA常規賽繼續進行，在快船對陣馬刺的比賽中，倫納德以客隊身份第3次造訪聖城，終於拿下了一場勝利。全場比賽他轟下了26分7板9助4斷，在噓聲中送上單臂隔扣，攻克聖城破除心魔！

上戰來到聖安東尼奧，倫納德和快船全隊都陷入了低迷，他本人23投8中狂打15鐵，快船也給低迷中的馬刺送上了「溫暖」。再往前追溯，當其身披猛龍戰袍時，依舊是在客場輸給了馬刺。

連續兩次造訪聖城輸球，這一次他終於贏了——頂著漫天噓聲，倫納德開場便穩穩命中中投，隨後他在轉換進攻中再度命中三分。最為精彩的時刻出現在首節後段，倫納德沿底線突破，面對高塔珀爾特爾，他毫不留情的直接送上單臂隔扣，勢大力沉的摧殘著馬刺的籃筐。

單節比賽他送上了9分2板2助1斷的完美數據，終於找到了對陣老東家的節奏。隨後的次節，倫納德依舊保持著高效率，最後時刻他強突籃下妙傳喬治，後者三分不中，他直接送上補扣，單節再度拿下9分，此時快船已經逐漸將分差拉開。

易地再戰的倫納德，持續著攻防一體的表現，搶斷快攻送上一條龍暴扣之後，快船的領先優勢已經達到了17分。本場比賽倫納德不僅是在進攻端表現高效，防守端也是將死亡纏繞進行到底，送上了4次搶斷。這次搶斷穆雷的暴扣，正是得益於他出色的判斷和優越的臂展。第三節單節，倫納德全面爆發轟下了8分5助2斷，數次和祖巴茨的連線徹底打花了比賽，快船單節凈勝16分之後，比賽懸念徹底宣告終結。

倫納德末節作壁上觀，完成三節打卡，同時也第一次以客隊球員的身份，在這片他無比熟悉的場地享受勝利。至此他已經完成了對於30支球隊客場的通殺，這場勝利他等了足足一個多賽季。
