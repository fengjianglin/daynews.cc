傳說中的兩巨頭成坑，並意外雙殺公牛！科爾和勇士都失算了

天天要聞
2019-12-07

本賽季勇士擺爛，基本是定的了。但今天勇士和公牛的比賽，卻打得非常激烈，最終勇士贏下了這場比賽。上半場勇士還是落後公牛一分的，甚至第三節結束後還落後6分，然而到了第四節，勇士卻突然爆發了，克里斯、博克斯和斯佩爾曼連續得分，這也讓勇士反超了2分隨後勇士一直保持比分知道最後一分鐘，以2分戰勝了公牛，並且在這個賽季實現看雙殺公牛，這是讓很多人都感到意外的地方。

對於這場勝利，相信勇士是無法接受的，因為此前科爾就表示過球隊想要前五順位，這場勝利不是他們想要的。當然了，這場比賽勇士的兩大巨頭還是坑了，格林本場比賽出戰30分鐘，只得到了9+5+4的數據，而這樣的數據顯然是不能讓人滿意的。甚至勇士的內線根本無法限制公牛的進攻，這裡格林是需要背一些鍋的。

別忘了，格林的合同可是高達一億，但就從本賽季的表現來看，顯然是不值一億合同的，這也是讓無數球迷都很擔憂的地方。除了格林，還有就是拉塞爾了，他在本賽季的表現依然也很低迷。

就從今天的比賽來看，拉塞爾甚至還出現過單節0分的表現，的確無法讓人滿意。而拉塞爾的合同也是高達了1.17億，居然是全場8投2中，他的毒瘤程度甚至比格林還要大。或許在本賽季結束時，勇士真的可能會換掉拉塞爾，因為對於拉塞爾，勇士和科爾都失算了，雖然這也是無奈之舉。
