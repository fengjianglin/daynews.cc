10年總決賽科比場均28.6分8籃板，皮爾斯雷阿倫加內特什麼數據？

2019-11-28 0條評論 14次閱讀 0人點贊

加內特在2010年健康回歸，凱爾特人隊三巨頭在2010年整體實力有所下滑，但是球隊整體配合更加默契，除此之外球隊後衛隆多已經成長起來，凱爾特人在東部決賽擊敗魔術進入總決賽。凱爾特人總決賽對手是湖人隊，兩支球隊在總決賽展開防守大戰，總決賽七場比賽兩支球隊各自僅有一場100+得分，凱爾特人主力中鋒帕金斯在總決賽第六場因為傷病賽季報銷，最終湖人隊經過搶七大戰83-79凱爾特人隊，湖人隊完成兩連冠，總決賽科比場均獲得28.6分8籃板3.9助攻，凱爾特人三巨頭在總決賽是什麼數據呢？

加內特在2009年遭遇傷病賽季報銷，2010年加內特健康回歸，10年加內特已經沒有巔峰期速度和爆發力，總決賽加內特沒有打出08年表現。在帕金斯受傷之後球隊內線防守責任更多是加內特來完成，總決賽第七場在湖人隊防守之下加內特全場得到17分3籃板，總決賽33歲的加內特場均15.3分5.6籃板。

2010年皮爾斯依然是球隊進攻核心，總決賽皮爾斯更多面對阿泰斯特的防守，總決賽七場比賽皮爾斯得分全部上雙，其中在總決賽第五場皮爾斯得到27分帶領球隊贏下天王山之戰，整個總決賽皮爾斯場均18分5.3籃板3助攻，能夠看出來總決賽皮爾斯面對阿泰防守打的異常艱難，最終凱爾特人隊在3-2領先的情況下被湖人隊逆轉。

雷阿倫在總決賽上攻防兩端更多面對科比，總決賽第二場比賽雷阿倫全場三分11投8中得到32分，雷阿倫出色表現帶領球隊獲得勝利，之後湖人隊主教練讓科比主防雷阿倫，之後五場比賽雷阿倫三分27投4中，雷阿倫三分失準是球隊輸掉總決賽原因之一，總決賽雷阿倫場均獲得14.6分2.7籃板。

隆多在2010年已經成長起來，常規賽隆多場均獲得13.7分9.8助攻，隆多成長是凱爾特人進入總決賽關鍵，總決賽第七場隆多全場得到14分8籃板10助攻數據，總決賽七場比賽隆多場均13.6分6.3籃板7.6助攻全面數據，2010年凱爾特人陣容年齡整體偏大，球隊主力中鋒帕金斯受傷是總決賽轉折點，最終凱爾特人經過搶七大戰惜敗給湖人隊。

科比在2010年和加索爾配合更加默契，常規賽科比把更多進攻交給加索爾和隊友來完成，科比在西部決賽帶領球隊淘汰太陽隊進入總決賽，總決賽面對凱爾特人隊，湖人隊在總決賽一度2-3落後，之後兩場比賽科比分別得到26分11籃板，23分15籃板，科比在手感不佳情況下把更多精力放在防守端，總決賽科比場均獲得28.6分8籃板3.9助攻，場均28.6分兩支球隊最高。

加索爾經歷過2008年，2009年總決賽之後更加出色，加索爾和隊友之間配合更加默契，總決賽加索爾5場比賽獲得兩雙數據，其中總決賽第六場得到17分13籃板9助攻，總決賽第七場得到19分18籃板4助攻，其中9個前場籃板，加索爾在總決賽第七場很好保護球隊籃板，整個總決賽加索爾場均獲得18.6分11.6籃板2.7蓋帽全面數據，加索爾出色發揮幫助湖人獲得兩連冠。
