馬刺射日 米爾斯1.7秒准絕殺

2019-12-15 26條評論 1,398次閱讀 75人點贊

北京時間12月15日，NBA墨西哥賽再戰-場，馬刺加時121-119險勝太陽。

馬刺隊5人得分上16分，米爾斯26分5板加時賽還剩1.7秒時跳投准絕殺，穆雷18分7籃板5助攻，阿德和德羅贊各18分;太陽隊卡明斯基22分7籃板4助攻，盧比奧25分13助攻，薩里奇19分17籃板4助攻，烏布雷17分8籃板，第四節還剩3.2秒投中三分把比賽拖入加時。

技術統計

比賽回顧

首節還剩5分8秒時，懷特出現傳球失誤，被米卡爾-布里奇斯搶斷。還剩2分11秒時，接到泰勒的傳球，貝恩斯拋投得手。還剩1分8秒時，懷特在奧科博投籃時犯規，送給太陽2次罰球機，會。馬刺內線優勢巨大，本節共搶下22個籃板，包括6個前場籃板，其中珀爾特爾一人就貢獻5個籃板球。馬刺在防守端築起銅牆鐵壁，此節送4記蓋帽，將對手的投籃命中率壓制到30.77%。首節戰罷，馬刺以28-18領先。

次節回來還剩7分44秒時，德羅贊手滑丟球，被盧比奧搶斷。還剩2分鐘時，朗尼-沃克妙傳，米爾斯三分遠投命中。還剩7秒時，朗尼-沃克在烏布雷投籃時犯規，送給太陽2次罰球機會。太陽對籃板發起瘋狂進攻，本節共搶下19個籃板，包括4個前場籃板，其中薩里奇- -人就貢獻4個籃板球，憑藉籃下優勢反超取得4分領先優勢。太陽多次衝擊籃筐，獲得11次罰球機會，但準星不足，僅罰中6球，命中率為54.55%。半場結束，太陽將比分反超至48-44。

進入第三節還剩11分46秒時，盧比奧出現傳球失誤，被穆雷搶斷。還剩10分49秒時，接到盧比奧的傳球，卡明斯基三分遠投得手。還剩7分18秒時，米卡爾-布里奇斯在穆雷投籃時犯規，送給馬刺1次罰球機會。太陽打出流暢配合，此節全隊共完成9次助攻，其中盧比奧一人就為隊友送出5次傳球。太陽頻飆三分，全隊本節投10粒三分球，但命中率僅為30%。馬刺多次衝擊籃筐，獲得12次罰球機會，罰中10球，罰球命中率達到83.33%。三節打完，太陽僅以2分的領先優勢進入下一節。

第四節還剩9分2秒時，朗尼-沃克在米卡爾-布里奇斯投籃時犯規，送給太陽2次罰球機會。還剩8分28秒時，貝恩斯手滑丟球，被阿爾德里奇搶斷。還剩3秒時，盧比奧喂球，烏布雷三分遠投命中。太陽內線優勢巨大，本節共搶下17個籃板，包括3個前場籃板，其中薩里奇一人就貢獻6個籃板球。太陽三分如雨，全隊本節投11粒三分球，但命中率僅為27%。兩隊最後關頭戰成109-109平，攜手進入加時賽。

比賽進入加時階段還剩1分19秒時，盧比奧手滑丟球，被米爾斯搶斷。最後黑客，米卡爾-布里奇斯在德羅贊投籃時犯規，送給馬刺1次罰球機會。比賽結束，馬刺以121-119擊敗太陽。

雙方首發陣容:

馬刺:阿爾德里奇、德羅贊、穆雷、萊爾斯、福布斯

太陽:盧比奧、卡明斯基、薩里奇、烏佈雷、布里奇斯
