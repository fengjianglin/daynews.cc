熱火無緣聖誕大戰，巴特勒或是偽巨星，韋德：戰績好也不行？

2019-12-25 609條評論 2,077次閱讀 231人點贊

聖誕大戰的到來了，讓很多球星都非常興奮，特別是職業生涯首次打聖誕大戰的肯巴-沃克，直言這是一個特殊的時刻。可見球星打聖誕大戰，都是一種被認可吧！

當然，不是每一支球隊都可以去打聖誕大戰，比如熱火、騎士這兩支曾經的聖誕大戰常客，都因為失去了詹姆斯，從而淡出了聖誕大戰的名單。本賽季戰績不錯的熱火隊，還是不能回到聖誕大戰的行列，那麼，入選聖誕大戰的球隊，到底需要具備什麼條件？

本賽季聖誕大戰的對決，分別是鵜鶘VS掘金，快船VS湖人，休城VS勇士，雄鹿VS76人，綠軍VS猛龍。答案非常直白，兩個條件非常重要，一個是戰績，一個是商業價值巨大的球星，比如鵜鶘隊為何能進入聖誕大戰名單？常年戰績不佳的鵜鶘隊，可能是僅僅擁有一個人氣狀元錫安-威廉姆森，但戰績出色的球隊，一定可以打聖誕大戰嚒！

韋德在社媒上表達了心聲，被認為是為熱火鳴不平，因為韋德如此說道：聖誕大戰應該是靠爭取的，讓我們在賽程的安排中增加一些驚喜元素，打得好的球隊應該出現在聖誕大戰中，就像我們獎勵賽季中期的全明星賽一樣。

非常直白的意思，韋德如同在問聯盟一個問題「打聖誕大戰戰績好也不行？」。當前熱火戰績東部第3也無緣聖誕大戰，對於韋德來說確實非常諷刺，如果巔峰韋德還在這一支東部第3的熱火隊，聯盟安排聖誕大戰，真可能放棄韋德嗎？當然不會了。

想打聖誕大戰，戰績好確實也不行，更能看出來巴特勒或是偽巨星，一個錫安的影響力，似乎還在巴特勒之上，那麼，在聯盟征戰多年的巴特勒，確實太無奈。巴特勒算不算巨星，可能維金斯與唐斯最具有發言權。

還有一個原因是熱火為何不能打本賽季聖誕大戰，因為聯盟對於戰績優秀，更多還是出於上賽季的考慮，但一定要求這一支球隊擁有人氣巨星，看看西部墊底的勇士隊，只是聯盟做夢都沒有想到庫里也傷停，更不敢相信勇士的戰績會一落千丈。
