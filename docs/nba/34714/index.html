不給快船得逞，湖人或會同時輪休詹眉，有一個人也會失望透頂

2019-12-25 123條評論 1,043次閱讀 90人點贊

NBA聖誕大戰一觸即發，在美國這就是他們的年終大戰，過年必備的下飯菜，而最引人關注的一場便是湖人主場大戰快船，說是湖人主場，其實也就是場地裝飾的差別而已，都是斯台普斯中心，兩隊都免了旅途的勞累，從這點來看，又都是主場作戰。

早些時候的消息，湖人官方給出的傷情報告是，詹姆斯和濃眉對於這場聖誕大戰都是出戰成疑的狀態，what？這倆都不打那還有什麼意思呢？對於球迷來說，那肯定是非常遺憾的，在夏天剛出賽程表時就盼望著了，到頭來撲了一場空，尤其是那些早早預定了高價球票的球迷，那就真的失望了。

如果詹姆斯和濃眉不打，失望的還有另外一個人，那就是肖華，他比誰都會更加失望，出了夏天那檔事後，每一次可以籠絡球迷的機會對他來說，都顯得異常珍貴，何況現在NBA的收視率在每況愈下，當聖誕大戰都無法給觀眾呈現出精彩激烈的大戲的話，那NBA真的很難挽回它在美國體育行業里的地位了。

同時對手快船也是如此，為何這麼說呢？湖人是快船本賽季在西部最大的競爭對手，更是同城死敵般的存在，揭幕戰雖然贏了，但那根本算不了什麼，那支湖人剛剛形成，磨合尚未完成，而現在不一樣了，打了30場比賽後，目前的這支湖人基本已經成型，除非後續有交易什麼的（可能性很小），快船更想擊敗的是現在的湖人，而且是陣容完整的湖人，一來可以驗收自己的實力，二來也可以繼續給同城對手下馬威。

其實快船想在聖誕大戰上擊敗湖人的意義遠不止本賽季，而是藉此機會誇大其在洛杉磯的影響力，論歷史來說，他們沒法跟湖人相提並論，但誰又想一直活在對手的陰影下呢，現在他們擁有著球隊有史以來最好的陣容，也是最好的奪冠時機，並且很可能會延續5年左右，比湖人還要來得更長，所以，快船太想在關注度超高的聖誕大戰中再次擊敗湖人了，上一場倫納德輪休了，跟湖人一戰大概率會出戰，屆時快船就是全員出戰。

而已經三連敗的湖人呢？當然想結束連敗，但球隊一定還是以球員健康為重，尤其是核心球員，為了一場聖誕大戰，從長遠來看，不太值得，詹姆斯目前有兩處已知的傷病，胸部拉傷以及腹股溝舊傷複發，今天雖然參加訓練了，狀態看似不錯，但隱患不小，畢竟跟快船一戰強度絕對不亞於跟雄鹿，濃眉也參加了訓練，也是有傷在身，上一場打掘金時又扭了一下膝蓋，後面明顯感到不適，如果是季後賽，那沒什麼好說的，出戰一點懸念沒有，但哪怕是聖誕大戰，那也只是一場普通的常規賽而已。

如果湖人同時輪休了詹姆斯和濃眉的話，那就相當於不給快船得逞，就算你贏了，那也是擊敗一支殘陣的湖人，不論在同城球迷面前，還是賽季心理上，說服力都不夠，只是有點苦了湖人球迷，看不到詹姆斯外，還苦吞四連敗，在勝負關係上還要繼續被快船壓制著。

當然，自從詹姆斯年紀上來後，也變得極其感性，他比球迷珍惜他還要更加珍惜球迷，在球迷那裡對他是看一場少一場了，而在詹姆斯看來也是同樣的，給球迷打一場少一場了，也不排除為了讓球迷在如此盛大的節日里看到自己，他會選擇帶傷出戰，如果他帶傷出戰，那濃眉應該也會響應吧，只是那樣會給湖人的心上了一根緊繃的弦。
