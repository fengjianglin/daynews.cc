聯盟中讓人不喜歡的四位球員，詹杜均上榜，勇士冠軍功臣實在過分

天天要聞
2019-12-20

聯盟中讓人不喜歡的四位球員，詹杜均上榜，勇士冠軍功臣實在過分。眾所周知，NBA作為全世界籃球競技水平最高的職業聯賽，這裡的球員都是最頂級的，隨之而來的就是這裡的比賽是異常激烈的，對抗激烈程度讓人咋舌，每個人都想要獲得勝利，這樣一來，有很多球員為了冠軍和勝利就不擇手段，做出的一些選擇讓很多球迷有些反感。

第四位就是現役球星詹姆斯，其實詹姆斯的偉大之處自然不用多說，無論是獲得的榮譽還是帶隊能力都是歷史級別的，唯獨讓人不看好的就是他為了冠軍，在巔峰時期和韋德波什抱團，雖然獲得了冠軍但還有有人不看好他。

第三位同樣也是湖人球星考辛斯，巔峰時期的考辛斯運動能力確實一流，說他是聯盟前三中鋒也不過分，但是缺點就是頭腦太不冷靜，喜歡打快樂籃球，從來不把勝利放在第一位，這讓很多看重勝利和比賽的球迷非常的討厭。

第二位是今年夏天剛剛從勇士轉回籃網的杜蘭特，他給人的形象一直都是低調，籃球能力很強，但是他當初也是為了冠軍離開了雷霆，加盟了勇士，讓很多他的球迷從那個時候開始不再看好他，但願他能夠在籃網重新開闢自己的王朝。

第一位是勇士的冠軍功臣追夢格林，大家這名球員也是褒貶不一，有人認為他是職業精神非常強大的球員，但是有人認為他為了贏球不擇手段，曾經踢別人的襠部，戳詹姆斯的眼睛讓他受到了很多非議，希望他之後能夠改一下這些毛病。
