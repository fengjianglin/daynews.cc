小賈倫-傑克遜：我們要學會如何守住領先優勢

2019-12-21 980條評論 4,669次閱讀 1845人點贊

虎撲12月21日訊 今天，灰熊客場107-114不敵騎士。賽後灰熊前鋒小賈倫-傑克遜接受了記者的採訪。

談到自己今天的表現，傑克遜表示：「其實就是根據對手對我的防守。並沒有很刻意的調整，主要是考慮到對手在三分線外放給我一些機會。在比賽剛開始的時候有一些衝擊內線打成，但騎士很快提升了防守強度，讓突破變得更困難。他們有很多封蓋好手——約翰-亨森，特里斯坦-湯普森，這讓往內線打變得更困難。」

談到球隊上半場球的轉移，傑克遜說：「這差不多是最重要的事情，要保持球在場上不停地運轉傳導。這樣當你在弱側的時候，會更容易，因為更多的跑動會幫助得到更多的突破機會，或者幫助隊友創造不錯的空位機會。」

談到領先然後被對手逆轉，JJJ說到：「就像你能很快建立比分領先一樣，當你沒能按照正確的方式打比賽時，領先優勢被蠶食也是很快的。當我們開始出現一些問題的時候，對手開始不斷命中。而這時候，你需要做的是提升防守的強度，讓對手打得不舒服。因此，我們可以從這場失利中學到很多，要知道在比賽中能夠建立領先優勢總是讓人鼓舞的，我們需要學會在比賽中如何守住領先優勢。」

本場比賽，傑克遜出場34分鐘，得到24分3籃板2封蓋。

(編輯：科比年輕)
