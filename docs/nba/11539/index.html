26歲第四次50+，會是下一站天王？濃眉哥職業生涯五場高光表演

2019-12-10 331條評論 1,875次閱讀 137人點贊

12月9日，濃眉哥在對陣森林狼的比賽中得到50+7+6+4的數據，職業生涯第四次50+，憑藉他的表現，湖人142-125大勝森林狼，26歲的濃眉成了隊史第二年輕的50分先生，第一年輕的是科比。那麼，在未來幾年，濃眉哥能夠繼續打出高效數據，接棒詹姆斯聯盟第一人的地位嗎？

五：單場25分+11籃板+3助攻+3搶斷+10蓋帽

2018年3月12日，鵜鶘與爵士的比賽中，戴維斯面對戈貝爾的強悍防守，輕鬆砍下25分+11籃板+10蓋帽，拿到職業生涯首個三雙，特別是10個蓋帽的數據，展現了強大統治力。

四：50分+7籃板+6助攻+4搶斷

第四場就是12月9日的比賽，詹眉合砍82分，其中戴維斯29投20中，罰球10罰10中，取得50+7+6+4的數據，打爆了同是狀元的唐斯。

三：50分+15籃板+5助攻+5搶斷+4蓋帽

2016年1月26日，在NBA揭幕戰比賽中，濃眉哥在面對掘金的比賽中大發神威，震驚聯盟，34投17中，罰球17罰16中，拿下了50+15+5+5+4的全面數據。

二：53分+18板+3助攻+1搶斷+5蓋帽

2018年2月27日，在一場西部對決中，鵜鶘在落後17分的情況下，最終125-116擊敗太陽，豪取6連勝。濃眉哥戴維斯出戰39分鐘，29投16中，瘋狂砍下53+18+3+1+5的數據，展現了驚人的天賦。

一：59分+20籃板

2016年2月21日，面對活塞，以及東部最好的「神獸」德拉蒙德，戴維斯全場34投24中，命中率高達70%以上，拿下了59分創造職業生涯新高，此外還搶下了驚人的20個籃板，成為進入21世紀以來除奧尼爾之外的第二人，震驚了當時的NBA。

江山代有才人出，各領風騷數十年。健康的濃眉哥在NBA絕對是BUG一般的存在，攻防兩端極具統治力，相信假以時日必成大器。隨著現役聯盟第一人詹姆斯的老去，濃眉哥完全有實力接棒第一人的位置，大家怎麼看呢？就讓我們拭目以待吧！
