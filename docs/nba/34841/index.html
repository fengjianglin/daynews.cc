貧攻型球隊的煩惱！魔術隊在引援速成與靜待天賦之間該如何抉擇？

2019-12-24 1,413條評論 3,833次閱讀 2745人點贊

東部季後賽邊緣球隊的煩惱——高不成低不就|魔術比上不足比下能否有餘？

新賽季的魔術隊依舊保持著低調處世的一貫作風，休賽期軍備競賽只是小打小鬧，還得時不時提防著管理層運作風評重回老路，繼續以季後賽首輪為目標同時注重隊內潛力挖掘，球隊靠著有意交易補充馬刺德羅贊的風言風語和前不久因霍華德重回安利中心仍遭受噓聲對待進而勾起的一波陳年往事來撐起流量。

想來不難理解，畢竟上半區天雷勾動地火打得再不可開交也與奧蘭多無關。無論是自由市場投入力度，核心陣容人員配置還是球隊整體預估競爭力，魔術都遠不及雄鹿，76人，凱爾特人，延續冠軍氣質的猛龍，救火員前赴後繼的步行者以及一躍成為東區黑馬的熱火。

目前魔術以12勝17負位列東部第八，出於現實考慮球隊應專註於7，8名席位爭奪並扮演好守門員的角色，如何在籃網活塞以及黃蜂混戰之間分一杯羹才是球隊眼下之憂。

魔術的上限如此，究其原因在於球隊缺乏足夠咖位的持球攻擊手，同時當家核心天花板已是觸手可及。

但球隊始終未做出相應行動的原因便在於多年魚腩靠著小本經營重返季後賽，好不容易喘口氣的管理層自然沒足夠魄力勵精圖治，希冀於以大破大立方式來提升競爭力，這是聯盟中游球隊的通病——不求有功但求無過。

傷病來襲，核心迷失|固步自封已成奢望，擅防貧攻的魔術該何去何從？

上賽季後半程的魔術隊憑藉聯盟第一的防守效率高歌猛進，期間攬獲22勝9負戰績最終如願搭上末班車。重視防守籃板守轉換降速逼陣地，籃下囤積重兵側重干擾規避犯規風險以及善用鋒線防守資源於禁區之內無差別掃蕩，這基本是東區前列隊伍的防守大綱。

但與雄鹿猛龍及熱火出於戰略考慮允許給予對手一定量的三分投射不同的是，魔術在限制對手三分球方面頗為專註，從限制對手空位三分球出手到命中率均處於聯盟中上游（上賽季魔術限制對手三分球命中率為34.7%）。在魔球大行其道的當下，掐死三分線與籃下自然是上上之選，那麼中距離相對薄弱區域只能倚仗鋒線外圍不斷輪轉來儘可能彌補戰術漏洞。

防守體系得以重建主教練克利福德自然功不可沒，進攻端則更多依賴球員內部打法改造。魔術主打內外結合同時控制失誤強調球的轉移，這樣是建立武切維奇與戈登為代表的前場球員擁有一定水準的策應能力及擋拆後的處理球得以保障。

如此一來便是在原有基礎上加磚添瓦，於是續約功臣武切維奇與特倫斯·羅斯，招徠防守悍將阿米奴便是順理成章。預想中魔術將繼續打造鐵桶陣藉此立足於東部時，傷病卻不請自來。

上賽季魔術得以在末段後來居上的另一個原因便在於球隊核心球員健康情況得以保證，首發五虎加第六人合計缺席16場比賽，而本賽季賽程剛滿1/3，武切維奇，戈登以及艾薩克三人便佔滿傷病指標，加之新援阿米奴半月板撕裂，球隊處境艱辛可想而知。

此外，成功留隊的最佳第六人候選，場均以38.3%命中率投進2.7記三分球的特倫斯·羅斯因諸多原因個人多項數據回落，魔術替補席上穩定輸出15+的火力攻擊點不再突出。

受困於傷病影響抑或是合同年反噬如期而至，內線軸心武切維奇由上賽季的20.8分12籃板3.8助攻下降至17.4分11.2籃板3.4助攻，命中率下滑7個百分點。

這二位本被視作球隊首發及二陣容進攻戰術支撐的救命稻草三分線外表現呈斷崖式下滑，致使魔術場均三分球命中數跌落至聯盟倒數第四，場均得分103.5僅優於尼克斯位列聯盟倒數第二，貧攻型球隊折其一臂自然無以為繼。

屋漏偏逢連夜雨，隊內為進攻兜底的二人先後迷失，魔術規劃中的穩步前行之路恐怕走不踏實了。窮則思變，魔術會作何打算呢？

外引強援，意向引援德羅贊能否擔起魔術進攻重擔？

即便是在這個魔球打法風靡的年代魔術依舊喜好古典中距離，中距離得分佔比為11.2%聯盟第八，試圖招徠德羅贊這樣一名中距離高手來消化球權，而他的到來對於缺乏主攻手的魔術來說會是一種良性補充嗎？

本賽季至今德羅贊場均貢獻20.7分5.4籃板4.7助攻並保有49.9%的投籃命中率，擁有26.3%的回合佔有率以及22.6%的助攻率。

交易主體籌碼大概率是魔術傲人的前場鋒線群，且不考慮到德羅贊能否留隊和因交易致使天賦與即戰力損失問題，單說德羅贊空降奧蘭多後首發出任2，3號位對於球隊的影響增益能否彌補實力折損。

歸根結底來說如今德羅贊已不是這個時代下的主流全明星球員了。隊內唯一一名三分球命中率超過4成的後場核心富尼耶（場均以42.3%命中率貢獻2.8記三分球）是魔術三分線上最後的遮羞布，難不成要摘下他換成遲遲無法以三分遠射回應偏見的德羅贊嗎？短短半個賽季的磨合，德羅贊恐怕無法將這支魔術帶入季後賽，如此一來是死棋一步。

內候新王，手握多張彩票的魔術何以兌現天賦升級攻防？

引援不成就應安心於內部挖掘，而魔術天賦積累基本便集中於戈登，艾薩克，富爾茨以及班巴身上了。儘管自外界來看魔術對於年輕球員的培養未能展現出足夠魄力與耐心，可事實並非如。

部分人認為班巴成長受阻的首要原因在於魔術一紙4年長約續下武切維奇以致於其出場時間無法保證，而他的賣點在於尚未兌現的體型天賦與投射可塑性，若是從即戰力角度考量班巴絕非是能夠撐起季後賽球隊首發五號位的人選。

此外魔術一號位上扶正狀元郎，32歲有功無過的奧古斯汀退位讓賢后富爾茨展現出合格控衛水準，26分鐘內以47.5%命中率交出11.6分2.7籃板4.1助攻1.21搶斷的數據，助失比與沃克丁威迪相當，除了個人初露崢嶸的攻框能力外富爾茨令球隊傾心放權的是他以奮進態度支撐的防守。從某種意義上來講，魔術未來押寶之人便是這身披藍白之星，肉眼惺忪之重生狀元郎了。

球隊唯一非賣品，被冠以下一個西亞卡姆的艾薩克憑藉封蓋榜前三，大擺火鍋盛宴的視覺衝擊進入人們熱議範圍內。從定點底角三分延展至弧頂，掛肉增重於對抗性與靈活性間尋求平衡並開始著手於小幅度提升持球攻佔比，由於上述種種基本是對照著猛龍奪冠年之前西亞卡姆的蓄力之作，人們對於22歲的艾薩克足以復刻其軌跡充滿信心，至少防守端天賦已在肆意展現。

至於戈登恐怕已經從當初最被看好的奧蘭多之光滑落著第二檔期許之列了。自扣籃大賽橫空出世之後戈登便和格里芬聯繫起來，於是在技能樹上加點投射，組織以及持球攻，但格里芬的運控單打技巧終歸是戈登可望不可即的。向全能鋒線方向培養往往收益與風險並存，即便是在這個容錯率極高的時代也不允許足具衝擊殺傷力的球員以29.4%的命中率每場嘗試3.9次三分球，穩定性缺失之餘個人技術粗糙才是戈登致命弱點。

奧蘭多左右為難|熬過陣痛期坐等雨過天晴抑或是大破大立重拾魔力？

「左手當下」與「右手未來」往往是諸多季後賽球隊所要面臨的必選題，前者有戰績壓力後顧之憂，後者充斥著不確定性遙不可及。

於魔術而言，升級主攻手意味著嘗試全新陣容構架，戈登持球能力有限而艾薩克側重防守，班巴遙遙無期忽略不計，富爾茨受限於球權與空間需求左右為難，短期內尋求隊內升級勢必會給球隊製造出不良反應，咬緊牙關熬過即可。

一支奉行平均主義的球隊急功近利的後果便是放任戰績滑落與體系崩塌，以無限開火權去賭一位扛旗持球攻擊核心的誕生。那是擺爛球隊的特權，顯然這絕不會是魔術管理層期望看到的。說到底即便是要承受天賦枯萎之痛，魔術斷然不會選擇邁出大破大立這一步。細水長流靜待天賦發芽便可，手握戈登武切維奇遞減合同可立於進退自如之地，何必貿然自斷後路？
