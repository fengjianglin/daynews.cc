第一三分神射，科沃爾效率值超過字母哥，真的物美價廉

2019-12-17 1,045條評論 4,465次閱讀 3468人點贊

NBA常規賽繼續進行，東部聯盟排名第一的雄鹿隊坐鎮主場迎來西部豪強獨行俠的挑戰，結果缺少頭號球星東契奇的獨行俠反客為主，他們眾志成城，全隊共6人得分上雙，頂住雄鹿的反撲以120:116擊敗了主隊，終結了後者的18連勝。波爾津吉斯和塞斯庫里發揮出色，分別得到26分，頂替東契奇位置的布倫森得到13分、11次助攻的兩雙數據。

​

雄鹿方面，字母哥拼盡全力，他全場31投18中，罰球16罰11中，得到全場最高的48分和14個籃板，但其他首發人員表現低迷，沒有一個得分上雙而且效率相當糟糕，比如中鋒大洛佩茲8投1中，其中三分球4投0中。但有一位老將的表現卻非常出色，他就是科沃爾。他替補出戰25分鐘，8投6中，其中三分球7投5中，貢獻17分、3次助攻2個籃板、1次蓋帽，是全隊得分第二高的人，他的效率值為正16，全隊最高。

​

第一節獨行俠三分發揮神勇，早早取得了兩位數的領先優勢，比賽還剩最後7秒，科沃爾被換上來，次節10分左右，康諾頓帶到左側高位，橫傳甩給外線的科沃爾，後者壓一步遠距離跳投兩分。回頭獨行俠進攻，塞斯庫里外線三分球出手，但被科沃爾直接帽下，科沃爾抓板自己推進，前場吊籃底下，字母哥打成2加1。7分38秒，前場迪文岑佐左側弧頂橫移突分右側底角，科沃爾快速三分跳投命中，將比分追成36:46，直接打停獨行俠。隨後，字母哥控球過半場中路弧頂持球，科沃爾及時擋拆，字母哥輕鬆走右側衝進去，一馬平川戰斧灌籃。4分18秒，伊利亞索瓦給到左側科沃爾，翻身三分又一個。

​

​

科沃爾幾乎打完第二節，半場結束，他共出場10:18秒，4投3中，三分3投2中，得到8分2助攻1蓋帽。下半場，科沃爾繼續延續上半時的出色表現，效率依然很高，再中三個三分拿到9分，全場一共貢獻17分。三分、防守、擋拆、策應，科沃爾做到自己所能做到的所有，全場拼下5次犯規，對於一位老將來說，這樣的表現實在不可多得。本賽季他場均出場時間只有15分鐘，但本場足足上場25分鐘。雄鹿最近連戰連捷，與科沃爾等一幫替補的出色發揮不無關係。

​

出生於1981年3月的科沃爾已經年滿38歲，職業生涯輾轉76人、爵士、公牛、老鷹、騎士、雄鹿多隊，除了在76人和老鷹，其他大多時間都是打替補。生涯場均出場25.6分鐘，得到9.8分、3.0個籃板、1.7次助攻。投籃命中率為44.2%，其中三分球命中率高達42.8%，本賽季依然能保持36%的三分命中率。本賽季雄鹿志在總冠軍，科沃爾256萬美元的合同物美價廉，雖然已經38歲了，但他依然是總冠軍球隊的拼圖。
