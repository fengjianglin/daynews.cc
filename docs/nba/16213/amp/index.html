30+10+5+3！冠軍球隊又贏了，走了一個倫納德，還有一個倫納德

天天要聞
3 天 ago

北京時間12月15日，布魯克林籃網隊又迎來了一場新的比賽，客場挑戰上賽季的冠軍球隊多倫多猛龍隊。可以說，他們也稱得上是東部兩支相對較強的球隊：多倫多猛龍隊剛剛獲得總冠軍，但奪冠大功臣倫納德走了；布魯克林籃網隊在休賽引進了杜蘭特和歐文兩個超級巨星球員，奈何雙雙傷退了。這樣的局勢之下，對兩支球隊來說還算是很公平的。

在首節比賽中，作為主場作戰的多倫多猛龍隊就把持住了主場的優勢，以38-31領先了對手7分進入到次節的比賽中。值得一說的是，猛龍隊的大前鋒西亞卡姆在首節比賽中就拿到了關鍵的14分，絕對有球隊領袖的風範！必須承認，經過上賽季冠軍經歷的洗禮，多倫多猛龍隊中的很多球員都成長起來了，都有獨當一面的能力。

隨著次節比賽的進行，雙方就出現了一波又一波比分齊頭並進的尷尬局面，彷彿次節的比賽就已經達到了最後決戰的高潮。然而隨著籃網隊這邊的一波強勢反擊，最終在第二節的比賽中，他們以29-27追上了兩分，半場以多倫多猛龍隊65-60領先布魯克林籃網隊5分結束。值得誇讚的是，西亞卡姆在次節比賽又拿到了11分，半場總共為球隊貢獻了25+33+3的數據。

易邊再戰，猛龍隊依舊能夠發揮出一支冠軍球隊該有的風範，還是以比分領先的優勢將籃網隊壓在身下。但籃網隊始終是一支還不錯的球隊，即使兩大巨星球員都不在場上，但他們依舊能夠發揮出不一樣的實力。最終在第三節比賽時間走完的時候，猛龍隊以25-23再次拿到2分的領先，總比分以90-83領先7分進入到關鍵的末節。7分的領先優勢，對於猛龍隊來說，基本已經是穩操勝券了，最終在末節的比賽中，猛龍隊以110-102拿下今天的比賽。

30+10+5+3！冠軍球隊又贏了，走了一個倫納德，還有一個倫納德。今天的比賽中，猛龍隊能夠取勝，西亞卡姆絕對算的上是球隊的頭號大功臣，看的喜愛30+11+5+3的數據，絕對能夠堪比倫納德的數據。確實，倫納德在成功當上北境之王之後，選擇了離開，但他所給猛龍隊帶來的影響力卻一直存在，就像是今天的西亞卡姆一樣，同樣能夠拿到不錯的數據。
