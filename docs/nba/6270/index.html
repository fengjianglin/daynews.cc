先輸雄鹿，再負猛龍，後敗76人，NBA又一支偽強隊誕生了

2019-12-03 0條評論 19次閱讀 0人點贊

開拓者一直以來都被看成是NBA偽強隊，他們雖然擁有雙槍利拉德和麥科勒姆，但卻多次在季後賽首輪被橫掃出局，本賽季開拓者的表現又一次讓人大失所望，他們連戰連敗，排名也一跌再跌。其實不止開拓者，在當今NBA還有一支球隊堪稱是偽強隊，該隊最近5場比賽輸了4場，而且遇強難勝，接連輸給雄鹿、猛龍和76人等強隊。

在11月26日，該隊在客場以118-122輸給了雄鹿，讓字母哥暴砍50分14籃板6助攻。緊接著該隊又在11月28日客場對陣步行者的比賽中以102-121慘敗而歸，全場比賽讓步行者5人得分上雙，其中有3人得分20+。

輸給雄鹿和步行者之後，該隊獲得了喘息的機會，他們在11月30日客場挑戰重建的灰熊，雖然最後比賽是贏下來了，但是過程非常艱難，一度落後了灰熊15分。戰勝灰熊之後，該隊又連遇強敵，結果都輸了，在12月2日以110-130慘敗給了猛龍，隨後在背靠背的比賽中以94-103輸給了76人。

那麼這支球隊是誰呢？那便是猶他爵士，爵士本賽季動作頻頻，雖然流失了盧比奧，但他們得到了博格丹諾維奇、康利和穆迪埃等多名實力球員，再加上米切爾、戈貝爾和英格爾斯，總體來說實力增進不少。然而賽季開始情況卻朝著截然相反的方向發展，球隊整體磨合不佳，沒有表現出應有的戰鬥力，他們進攻不流暢，進攻效率105.3排在聯盟第23位，防守也沒有那麼出色，防守效率104.1排在聯盟第11位。此外新援康利退步太快，已經不復當年之勇，核心米切爾似乎已經到了上限，個人表現非常不穩定，多次打出24中9，23中5和22中7的糟糕表現。綜合來看，顯然現在這支爵士很難被稱為是真正的NBA強隊，他們更像是NBA偽強隊。
