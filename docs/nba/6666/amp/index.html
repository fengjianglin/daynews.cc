布雷特-布朗：希望西蒙斯加盟澳洲男籃，我們有望拿金牌

天天要聞
3 週 ago

虎撲11月27日訊 根據此前的報道，76人主帥布雷特-布朗將再次擔任澳大利亞男籃主帥。

今天布朗在接受採訪時表示，他目前的首要任務仍是帶領76人取得成績：「我需要得到兩位老闆的同意，Josh Harris和David Blitzer，他們都很棒。以及我們的總經理埃爾頓-布蘭德，他也是背後的推動者。澳大利亞主帥的工作是我越了解越想做的事情，如果沒有這三人的批准，我無法做到。」

「我在費城的生活十分繁忙，目前我的精力集中於執教76人，然後當時間到了的時候我就會做澳大利亞主帥的工作。」

布朗還表示，他希望76人後衛本-西蒙斯能夠參加東京奧運會，但他也並不能確定。此前西蒙斯曾表示計划出戰。

「我無法向國家隊引薦本，我會把決定權交給他。本還沒有對此作出正式決定，我相信他最終會來，但我會讓他來決定。而且我們都會希望此事成真，我認為一切都會水到渠成。過去幾年他一直是我麾下的控衛，如果最終如我們所願，我會再次把球交給他。」

「希望本的加盟以及其他球員的支持能夠讓我們贏得金牌，這也是我接受這份工作的唯一原因。」

談到球隊實力，布朗說道：「當我看著這支球隊的陣容，我感到自己與他們聯繫緊密，從帕蒂（米爾斯）到馬修-戴拉維多瓦，到喬（英格爾斯），再到阿隆（貝恩斯）等人。就像看著後視鏡一樣，我還記得早些時候與這些年輕人一起的日子，現在有機會與他們再續前緣，這對我很有吸引力。」

「澳大利亞體育的精神在全世界都深受認可。當我在NBA與他人聊起澳大利亞男籃，我都有此體會。這個國家的化學反應和精神，競爭性與友誼，這些都是我的動力，是每個教練都想要帶入自己球隊的東西，這都是澳洲體育文化里固有的特性。」

(編輯：窩瓜)
