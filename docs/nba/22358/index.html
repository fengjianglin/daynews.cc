詹姆斯的粉絲都多大牌？一個個比他能賺錢，股神巴菲特都看他打球

2019-12-19 249條評論 1,254次閱讀 133人點贊

毋庸置疑，當今NBA聯盟中最大的人生贏家就是勒布朗-詹姆斯，他不僅在聯盟取得了諸多常人難以企及的榮譽，更是球隊當之無愧的領袖，超級巨星。而這樣偉大的詹姆斯自然也吸引了不少大牌粉絲，他們的大牌程度甚至比詹姆斯還要誇張，甚至賺錢也比詹皇厲害太多。那麼詹姆斯的球迷有多牌面呢？接下來我們就來看看吧。

蕾哈娜：全世界的人都知道蕾哈娜是知名歌手，全世界的人都知道蕾哈娜是詹姆斯的頭號女粉絲，當然「大帝」恩比德自然也知道，畢竟他曾醉心於蕾哈娜無法自拔。蕾哈娜的音樂成就那是相當的高，她擁有數十首冠軍單曲，多次蟬聯格萊美、全英音樂和MTV大獎，世界各地的粉絲不可計數。然而蕾哈娜只想成為詹姆斯的粉絲，不管詹姆斯在哪座城市，場邊總少不了蕾哈娜的身影。

Jay-Z：如果問美國人「誰是最偉大的說唱歌手？」你一定會源源不斷地聽到Jay-Z這個名字。Jay-Z不單單是一位嘻哈歌手，同時也是唱片製作人和企業家。到目前為止他一共獲得了21次格萊美獎，並在19年福布斯100名人榜上排名第20,。而這位堪稱美國娛樂圈教父的人卻是詹姆斯的鐵杆球迷，他經常在場邊觀看詹姆斯的比賽，與詹姆斯的場下關係也是非常親密。此外，Jay-Z更是多次在歌曲中diss那些批評詹姆斯的人，比如禪師。

碧昂斯：碧昂斯也是美國家喻戶曉的女歌手，她的作品多次獲得格萊美獎，其中在2017年還獲得最佳國際女歌手的獎項。在2018《福布斯》雜誌評選的全球名人榜中收入排名第二，更為恐怖的是這位天后級別的歌手選擇嫁給Jay-Z，教父天后的強強聯合可謂所向披靡。在Jay-Z的帶領下，碧昂斯也成為詹姆斯的球迷，夫妻倆經常攜手去看詹姆斯的比賽。

巴菲特：巴菲特的名字世人皆知，他是世界上最著名的投資商，投資領域遍及股票、電子現貨、基金等行業。在2016年的彭博全球50大最具影響力人物排行榜排名第9，在當年的《福布斯》年度富豪排行榜中排名第3位。不過就是這樣一位深受金融界崇拜的成功人物居然是詹姆斯的粉絲。從下圖我們不難發現，巴菲特在與詹姆斯在一起時所露出的表情可以說是很迷弟了。

卡洛斯·斯利姆·赫魯：可能大家對這個名字不太熟悉，但他可是曾經的世界首富，2008年《福布斯》富豪榜排名第二，超越了比爾-蓋茨，到了2011年，卡洛斯以535億美元的資產壓倒股神巴菲特，登頂全球首富。但就算是世界首富也要被詹姆斯場內外的表現征服，他經常在社交軟體曬出與詹姆斯會面的照片。看那心滿意足的表情，你就知道卡洛斯對詹姆斯的崇拜程度了。

威廉王子夫婦：除去上面所說的娛樂歌星以及世界首富之外，詹姆斯還有兩名更大牌的粉絲——威廉王子夫婦。幾年前威廉王子大婚可謂是賺足了大眾的眼球，灰姑娘和王子的愛情故事讓許多人再次相信愛情。這對「普普通通」的夫婦來到美國時還特地觀看了騎士的比賽，因為他們的偶像詹姆斯在騎士隊，賽後夫婦二人也是開開心心地邀請詹姆斯合影留念。可以看出，無論是怎樣大牌的身份，在自己的偶像面前都是一副「受寵若驚」的表情。
