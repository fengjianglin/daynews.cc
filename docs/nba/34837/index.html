慘敗24分又如何？湖人球迷仍高喊“MVP”，難怪禿曼巴罰球8中8

2019-12-24 1,146條評論 2,402次閱讀 325人點贊

湖人主場迎戰掘金的比賽，已經出現連敗的湖人隊，能否徹底反彈，成為了不少湖人球迷的期待。不過掘金還是在客場輕鬆擊敗湖人，帶著6連勝狀態挑戰湖人西部第一的位置。

128比104，詹姆斯坐在場邊觀看了這一場比賽，由於胸肌拉傷的原因，詹姆斯只能眼睜睜看著湖人隊吞下一場慘敗，整整輸掉了24分，只是慘敗24分又如何？湖人球迷壓根沒有放在心上，反而在末節比賽開啟了自娛自樂的高潮，仍高喊「MVP」響徹斯台普斯中心球館。

末節比賽，綽號禿曼巴的卡魯索走上罰球線，就在卡魯索準備罰球的時候，斯台普斯中心球館開始高喊「MVP」，聲音經久不絕，讓掘金隊眾人都感到震撼。

跟隨著湖人球迷「MVP」高喊的節奏，卡魯索在罰球線隔空打爆了哈登，完成了單場罰球命中率100%的操作。看來想要讓湖人球員罰球命中率高，「MVP」的呼聲真的不能少，當卡魯索穩穩命中罰球那一刻，不少網友都吐槽：難怪卡魯索罰球8中8太准了。

全場比賽打了22分鐘，卡魯索1投0中，憑藉著罰球8中8砍下8分2籃板1搶斷1蓋帽。

對於湖人本賽季衝冠的道路上，卡魯索可以擁有禿曼巴的稱號，但他始終不是黑曼巴。那麼，卡魯索可以在MVP呼聲下，做到罰球一次不丟，確實非常難得了。

如果不是詹姆斯傷停，濃眉哥膝蓋受傷，這一場比賽的對決，勝負確實很難預料，不過湖人角色球員的強勢回應，同樣是湖人所需要的。相比常規賽西部第一的戰績，湖人更注重季後賽的衝刺，那麼詹姆斯與濃眉哥真的要保持健康才行，畢竟湖人真正的MVP才是最重要。
