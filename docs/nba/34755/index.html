瘋狂！天才五號秀被交易，湖人舊將接近簽約，公牛後衛賽季報銷

2019-12-25 1,683條評論 4,614次閱讀 3119人點贊

北京時間12月24日，NBA瘋狂一夜來臨。時隔165天，聯盟又一筆交易誕生了：騎士後衛克拉克森與爵士後衛艾克薩姆互換東家，騎士得到兩個次輪選秀權！這筆交易，是今年7月12日保羅和威少互換東家以後，聯盟首筆交易。長達165天的交易乾旱期，也宣告結束了。另外，湖人舊將喬納森-威廉姆斯接近加盟奇才了。奇才申請了傷病特例，球隊準備用這個空間來簽約喬納森-威廉姆斯。最後還有一則噩耗：公牛後衛馬克斯-斯特魯斯右膝受傷，賽季報銷，預計休息8-12個月。

消息比較多，咱們具體一個一個來說說。騎士和爵士這筆交易，應該說是爵士賺了。雖然艾克薩姆當年貴為選秀大年的天才五號秀，但進入聯盟以來，傷病不斷，打打停停，爵士已經等不起了。他兩個膝蓋都動過手術，這對於一個依靠爆發力的後衛來說，太艱難了。而且，在如今這個全民投三分的時代，艾克薩姆的投射一直沒有練出來，這也是為何爵士在去年夏天跟他續約了一份3年3300萬美元的合同後，仍然在一年之後放棄他的原因。

再來說說克拉克森，湖人舊將，湖人當年慧眼識珠挑選的二輪秀。克拉克森新秀賽季打得很爆炸，場均11.9分3.2籃板3.5助攻，入選了最佳新秀一陣。

此後，湖人也在過渡期重點培養過克拉克森，但是湖人最終發現，克拉克森打亂戰是高手，但作為核心球員來使用，他還是難堪大任。於是後來將其送到了騎士。在騎士的兩個多賽季，克拉克森的曝光度急轉直下，雖然偶爾也會拿到30+的高分，但總體來說，不溫不火，沒有當年的銳氣了。

再來說說湖人另一個舊將喬納森-威廉姆斯。據奇才記者Fred Katz報道，消息人士透露，奇才隊接近簽下喬納森-威廉姆斯。奇才本賽季遭遇了傷病潮，多名主力球員都因傷缺陣了，CJ邁爾斯（手指）、麥克雷（手指）、沃爾（跟腱）、瓦格納（腳踝）、貝爾坦斯（肱四頭肌）、八村塁（腹股溝）、托馬斯布萊恩特（右腳應激反應）。球隊沒辦法，只能連續向聯盟申請傷病特例，聯盟酌情考慮，也都通過了。

喬納森威廉姆斯上賽季在湖人打了24場比賽，場均6.5分4.1籃板，態度非常好，拼搶非常積極。有過打凱爾特人單場18分10籃板，打勇士單場17分13籃板的高光表現。奇才簽下他，來頂替受傷的另外兩個湖人舊將托馬斯布萊恩特和瓦格納，是一個不錯的選擇。

最後來說說悲催的斯特魯斯，公牛今年夏天用一份雙向合同簽下的一名後衛。上賽季他在凱爾特人效力，也是雙向合同，穿28號球衣。今年夏天，公牛將其簽下，但賽季至今，斯特魯斯還上場打過球，就在訓練中遭遇左膝十字韌帶撕裂和骨頭挫傷，賽季報銷。根據經驗，斯特魯斯預計將缺陣8-12個月。
