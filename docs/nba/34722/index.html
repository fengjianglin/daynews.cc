NBA聖誕大戰，你所不知的幾大趣事

2019-12-25 118條評論 1,111次閱讀 63人點贊

時間很快，2019年接近尾聲，聖誕節馬上到來。這是一個現在在國內也火起來的西方節日，NBA球隊在聖誕節也會有焦點大戰，算得上是常規賽裡面的一場重頭之戰，是常規賽中關注度最高的比賽之一。

聖誕大戰已經延續有五十多年的歷史了，而NBA官方也會在常規賽開始之前根據夏季人員調整後的球隊實力來安排賽程。可以說能夠參與聖誕大戰的球隊，都是實際強勁或者兼顧關注度高粉絲極多的球隊。因此，每年的聖誕大戰對於球迷來說也稱得上是一場豪門盛宴。

那麼聖誕大戰存在哪些你可能不清楚的趣事呢：

1、NBA聖誕大戰已經進行了56年了，目前聯盟30支球隊中只有孟菲斯灰熊和夏洛特黃蜂從來沒有參加過聖誕大戰；

2、庫里，生涯參加過7次聖誕大戰，共計得到86分，場均12.3分，對於一個拿過得分王的人來說，的確有些說不過去，可能那天「小學生」都忙著快樂去了；

3、麥迪，是近30年的單場得分王。在02年聖誕大戰魔術104-99擊敗活塞的比賽中,26中14,拿下46分。可是他竟然只參加過兩次聖誕大戰，場均43.5分；

4、中國人，唯一一次參加聖誕大戰是在03年，姚鯊再次相遇，最終球隊99-87戰勝湖人，姚明拿下18分8籃板2蓋帽，奧尼爾拿下22分10籃板3蓋帽；

5、近30年來聖誕大戰只出現過兩次三雙，自然是三雙常客詹姆斯（27+11+10）和威少（14+13+10）了；

6、紐約尼克斯，是歷史上參加聖誕大戰最多的球隊，達到了50次（22勝28負）；連續38年參加聖誕大戰（一部分原因是NBA總部所在地）；

7、2004年聖誕大戰，是NBA歷史上出動警衛最多的聖誕大戰，因為奧本山宮殿的當事人球隊在事件一個月之後再次相遇了；

8、傳承，1978年的聖誕大戰，76人對陣尼克斯，76人隊中兩位球員:亨利-畢比和喬-布萊恩特的妻子剛好都在那一年都生下孩子，名字分別叫邁克-畢比和科比-布萊恩特；時隔24年之後，2002年聖誕大戰，國王對陣湖人，畢比和科比同時登上聖誕大戰舞台，這無疑是最好的傳承。

今年的聖誕大戰也有很多焦點之戰，比如雄鹿對陣76人、洛城德比等等（希望詹眉都沒事都能打，希望小卡不要輪休）。
