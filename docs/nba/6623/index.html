傷病改變一切，濃眉惋惜和考神合作一年，季後賽曾有望跨過勇士？

2019-11-28 0條評論 18次閱讀 0人點贊

考辛斯在湖人的生涯並不順利，雖然簽約許久但他還未代表球隊出戰過一場比賽。在簽約之後他就在訓練中遭遇韌帶撕裂的傷病，復出還是個未知數。而養傷期間，傳出家暴醜聞，有因為涉嫌家暴被發出逮捕令。

麻煩纏身使得表妹心事重重，可他和湖人的合同還沒有結束，湖人眾將將他抱以歡迎的態度。

在鵜鶘時做隊友的濃眉哥戴維斯就很有感觸，他透露表妹是個很好的球員，即使不能上場比賽，也會盡量多參與他們的訓練，他想為為這支湖人出一份力。

表妹此前就在社媒上發除一個心酸的表情，湖人主場連勝隊友都非常高興，只有他在悶悶不樂。他非常想代表球隊出戰，每個人都沉浸在勝利的喜悅中，他也想幫助球隊贏球一起歡呼。

戴維斯雖然現在和表妹同在湖人隊，但在鵜鶘的時候本來也是前景無限。只可惜還是傷病改變了球隊的命運，他自己也覺得相當可惜。

「我們究竟能成就多少只能存在於想像之中了。我們倆當然很強，再加上隆多還有霍樂迪。真希望當時他能保持健康，我們一起去成就一些特別的事情。」

當時的鵜鶘在季後賽和勇士交戰正是缺少表妹，AD認為如果沒有傷病他們的組合甚至可以嘗試跨過勇士。兩個都是巨星型內線，加上外線的防守大閘隆多和霍樂迪，紙面上也是非常強大。

如今在湖人，AD依然兇猛，詹姆斯尚在巔峰的後半段，只要沒有重大傷病他們還是西部奪冠的大熱門。

「在湖人又得到了一次機會，這次我們都不想浪費機會。」
