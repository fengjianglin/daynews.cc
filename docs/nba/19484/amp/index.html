盯上3屆籃板王？名記稱獨行俠醞釀球員交易，組三巨頭劍指總冠軍

天天要聞
1 天 ago

如果和上賽季的NBA球隊戰績排名相比，如今獨行俠真的已經是今非昔比，上賽季無緣NBA季後賽的獨行俠，現在則是順利拿到西部第3名的常規賽排名。而獨行俠全力培養的超新星東契奇，在這個賽季也是成功躋身聯盟超巨球星的行列。隨著球隊戰績的大幅提升，同時擁有足夠薪資空間的獨行俠，他們似乎也是開始盤算起組建三巨頭陣容，從而讓球隊時隔多年再度劍指NBA總冠軍。

最近，據NBA名記扎克-洛韋在節目中爆料，獨行俠目前想要通過交易得到一名內線大個子球員，從而和全明星球員波爾津吉斯組成內線雙塔。而活塞隊實力球星、NBA3屆籃板王德拉蒙德，目前則是獨行俠管理層的目標球員。如果獨行俠管理層真的能夠得償所願，最終成功將德拉蒙德帶到達拉斯，那麼獨行俠將組成東契奇、波爾津吉斯、德拉蒙德的三巨頭陣容，即便是在目前的NBA聯盟裡面，這套巨頭陣容也是完全具備奪冠實力。

當然，獨行俠也是完全擁有實現這一交易的可能性，因此德拉蒙德的球員合同即將到期，明年夏天德拉蒙德就將成為NBA自由球員。到時候活塞隊將會面臨一個艱難的選擇，如果給德拉蒙德一份長期頂薪合同，考慮到活塞隊的薪資情況，以及德拉蒙德和格里芬的帶隊戰績，活塞隊未來很長一段時間或許都無法翻身。而如果白白放走德拉蒙德這位NBA全明星球員，活塞隊管理層肯定也是不願意，因此交易到獨行俠得到選秀簽或潛力新秀，這其實也是相當不錯的球隊操作。

當然，除了活塞隊的德拉蒙德，據扎克-洛韋透露還有一名NBA球員同樣也是獨行俠的心儀目標，而這個人就是快船隊的替補內線哈雷爾。而和德拉蒙德的情況非常相似，哈雷爾的合同也將會在明年到期，雖然本賽季快船隊肯定不會交易哈雷爾。但是到了明年夏天的時候，獨行俠就可以揮舞著高薪合同主動招募哈雷爾，到時候或許獨行俠將會和快船隊進行一場NBA搶人大戰！
