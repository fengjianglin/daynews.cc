他們是西部最佛系強隊？被雄鹿橫掃，球隊集體養生為衝冠打基礎？

2019-12-08 0條評論 13次閱讀 0人點贊

在NBA現有體系的賽程之下，東部球隊和西部球隊一年只會碰面兩次，雙方各有一次主客場的機會。然而在聯賽開始剛剛一個多月的時候，本賽季東部奪冠熱門雄鹿隊和西部奪冠熱門快船隊的兩場比賽已經打完。在這兩場比賽中雄鹿都是以較大優勢擊敗快船隊，這樣也讓球隊在本賽季未來對陣快船的比賽之中有了更多的底氣。

在此前雙方的第1次較量之中，快船隊的核心球員萊昂納德選擇了輪休，這在當時甚至還引發了全聯盟的討論，用著名解說員楊毅老師的話來說那就是作為聯賽mvp級別的運動員，萊昂納德在面對為數不多的和字母哥的較量之中選擇輪休，確實有些說不過去。已經退役的NBA名宿特雷西麥克格雷迪，在接受採訪的時候也對萊昂納德的表現表達出了不滿，他甚至用自己在職業生涯中對戰艾弗森的例子，來表達和明確了自己的態度。

不過本場比賽萊昂納德和保羅喬治全部出戰，按說應該是一場勢均力敵的較量，可是在比賽之後快船完全沒有找到狀態，第四節雙方的比分差距甚至達到了40分。全場比賽快船隊的核心萊昂納德得到17分、保羅喬治只有13分，神奇的路威也只有11分，整支球隊在比賽中只得到了91分確實有些罕見。雄鹿則維持了自己的一貫水準，由於球隊正處於隊史的最佳連勝場次之中，所以接下來的每一場比賽雄鹿隊都會拼盡全力去贏球。

從實力上來看雄鹿隊有很希望成為東部第一，然後在季後賽中拿到東部冠軍殺入總決賽。而快船隊也是西部冠軍的熱門人選，所以兩隊在總決賽中見面也是有一定幾率的。如果考慮到常規賽的兩場比賽雄鹿隊都是大勝對手，那麼在接下來的比賽中心裡優勢自然比較明顯。

當然從快船的氣質來說，他們本賽季確實比較佛系，且不說已經引發公憤的輪休表現，就是萊昂納德和保羅喬治在採訪中表現出來的態度也不是非要取得多好成績或者說就是要衝擊總冠軍，但是所有人都知道小卡在猛龍隊已經上演了這麼一出，誰知道他會不會又在季後賽暴走呢？
