這四名球員數據再好也拿不到大合同！一個傷病太多，一個身高太矮

2019-12-22 1,150條評論 2,712次閱讀 1698人點贊

在NBA，有些球員數據沒有那麼出色，卻是球隊不可或缺的一部分，也能夠拿到大合同。有些球員就算數據再好也拿不到大合同。

羅斯上個賽季在森林狼場均可以得到18分4.3助攻，今年休賽期2年1500萬美元加盟了活塞，薪水才731萬美元。羅斯目前在活塞場均可以得到16.5分2.2籃板6助攻，是活塞的第二得分手。羅斯現在打得再好，也很難拿大合同了，主要是傷病太多了，其實三分球也不是他的特長，這個賽季三分命中率只有31.3%，但中距離太准了。

以前在國王、太陽，小托馬斯就很能得分，16-17賽季在凱爾特人，他場均可以得到28.9分6助攻，是聯盟末節之王，聯盟最好的控衛之一，季後賽帶領球隊打到了東部決賽。即使是這樣，凱爾特人還是將他送到了騎士隊，終究是身高太矮，小托馬斯為了凱爾特人付出所有，帶傷作戰，直接毀掉了他的職業生涯。這個賽季小托馬斯在奇才隊，場均可以得到13.3分4.9助攻，投籃命中率高達45%，三分命中率高達42.7%，但沒有球隊願意給小托馬斯大合同，甚至千萬都沒有。凱爾特人曾經給小托馬斯一份5年7000萬美元的報價，被小托馬斯拒絕了。

羅斯傷病太多，小托馬斯身高太矮，那麼下面這兩名球員為什麼拿不到大合同呢？

路威可能是聯盟性價比最高的球員了，在過去的兩個賽季，他場均可以得到20分5助攻的數據，但是他的年薪只有800萬美元。路威一直就很能打，但就是拿不到大合同，說他防守不好，那庫里、歐文等人防守也不好，很難去解釋這個。

克拉克森今天在與灰熊的比賽中，17投12中，三分8中6，得到了33分，但只送出了1次助攻。克拉克森在人們的眼中就是一個數據刷子，眼裡只有籃筐，沒有隊友，拿球就是單幹，這個賽季是合同年，目前在騎士隊場均得到14.6分，投籃命中率為44.2%，三分命中率為37.1%，其實數據很好的，但之前季後賽表現太差了，這個標籤摘不掉。

羅斯、小托馬斯、路威、克拉克森，這四名球員現在數據再好，也拿不了大合同。
