白送1344萬實力戰將？NBA總冠軍徹底擺爛，湖人只能幹著急

2019-12-25 34條評論 4,983次閱讀 2267人點贊

2019年12月24日星期二，NBA常規賽進入到即將開始聖誕大戰前夕，這時的交易市場也已經慢慢開啟，很多球隊都開始為自己的補強而做出努力，就湖人隊來說，給詹姆斯尋找一個靠譜替補的問題依然沒有解決。而一哥到底能不能來還是個未知數，不過本賽季NBA的第一筆交易卻悄悄的到來了，但是卻和湖人沒有關係。

據多名聯盟人士透漏，第一筆交易是騎士和爵士之間的，並且達成了3換1的意向，具體為，騎士送出克拉克森換來爵士的艾克薩姆以及兩個次輪選秀權。這筆交易可以說是雙方互贏的，因為對於騎士來說可以降低球隊的工資，還能拿到未來的選秀權，而爵士則得到一個即戰力！

先來說騎士，自從詹姆斯離開之後，球隊就一直想要重建，重建意味著要騰出薪資空間和未來的選秀權，包括狀元爭奪戰！而克拉克森的合同年薪達到了1344萬美金，並且還有著樂福的大合同這使得球隊可以使用的籌碼都有所減少，隨著克拉克森的離開，相信樂福也會在這個賽季找到新東家，因為樂福的存在會破壞騎士的擺爛大計。

再來看爵士，今年夏天，球隊並沒有和金童盧比奧續約，而是通過大交易換來了進攻防守更均衡的康利，但是康利本賽季的發揮卻並不理想，沒有達到爵士想要的水平，但是克拉克森本賽季場均能拿到14.6分並且三分球的命中率接近了40％，這樣的即戰力很好的補充到爵士陣中。

這次交易，爵士得到一個即戰力加入到球隊，對於球隊的整體實力有很大的提升，特別是替補席的深度，甚至可以和米切爾組成雙槍，在強隊林立的西部，可以更好加強他們的競爭力，目前爵士排在西部第六位！再來看看騎士，艾克薩姆對球隊的幫助並不大，但是便宜不會影響擺爛大計，騰出薪金空間才是目前騎士的大事，所以這克拉克森可以說是騎士白送給爵士！

這筆交易是今年夏天保羅和威少互換東家之後，經歷165天的第一筆交易，這筆交易也相當於告訴大家可以動手了，那麼接下來的交易將會蜂擁而至，是否會對聯盟的格局造成影響呢？讓我們拭目以待！

但是對於三連敗的湖人，第一筆交易該怎麼辦呢？難道真的只能看運氣和驚喜了（上篇文章分析過）？
