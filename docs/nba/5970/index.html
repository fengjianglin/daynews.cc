12月8日NBA資訊！登威58分力克太陽，西蒙斯生涯新高狂勝騎士

2019-12-08 0條評論 20次閱讀 0人點贊

北京時間12月8日，NBA常規賽繼續進行，今日5場比賽

獨行俠主場迎戰新奧爾良鵜鶘憑藉第三節40-14的猛烈攻勢，最終在主場以130-84大勝鵜鶘，迎來五連勝的同時，送對手八連敗。獨行俠這邊，東契奇三節打卡26分6籃板9助攻，博班15分16籃板，小哈達威和布倫森各得14分，波爾津吉斯13分5籃板，小庫里11分；鵜鶘這邊雖有4分上雙但最高不過15分，雷迪克15分，英格拉姆14分4籃板，霍勒迪14分4助攻，海耶斯11分5籃板。
