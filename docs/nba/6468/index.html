東契奇42+9+11，獨行俠7分擊敗太陽，波爾津吉斯2+13

2019-11-30 0條評論 17次閱讀 0人點贊

北京時間11月30日訊，NBA常規賽繼續進行，獨行俠隊在客場挑戰太陽隊，這場比賽之前，獨行俠隊已經取得了11勝6負的戰績，排在西部第5位，而太陽隊戰績為8勝9負，排在西部第8位，近期球隊戰績不佳，最近5場輸了4場。

比賽首節，沙里奇跳投命中，東契奇拋投沒有，但他拼搶到進攻籃板補籃命中，貝恩斯命中三分，東契奇突破完成上籃，烏布雷三分命中，東契奇助攻隊友打成2+1，沙里奇回敬一個2+1，哈達威命中三分，兩支球隊比分交替領先，隨後東契奇接管了球隊進攻，他連續突破完成上籃，又助攻隊友完成得分，單節東契奇12投6中拿到14分6籃板6助攻，球隊在他的幫助之下，首節取得6分的領先優勢。

比賽第二節，烏布雷成為了太陽隊的奇兵，他在外線命中3記三分，單節拿到9分3籃板2助攻2搶斷1蓋帽的全面數據，替補出場的布里奇斯和約翰遜也有8分入賬，而獨行俠隊這邊，球隊進攻一般，單節只拿到22分。半場結束，太陽隊把比分差距縮小至3分。

比賽第三節，東契奇再次迎來爆發，他在弧頂位置命中2記超遠三分，隨後突破內線打成2+1，單節他7投5中拿到16分1籃板4助攻，不過他的表現並沒有幫助球隊取得領先，太陽隊這節整體發揮不錯，布克和烏布雷在外線命中3記三分，盧比奧單節也有10分入賬，三節結束，兩支球隊回到同一起跑線。

最關鍵的第四節，兩支球隊在進攻端都有不錯的表現，比分差距始終維持在2分左右，隨後，獨行俠隊在4分鐘內打出一波16-2的進攻，把比分差距拉大到14分，布克持球突破內線造成犯規，兩罰全中，庫里造成犯規，兩罰全中，貝恩斯三分不中，哈達威反擊造成烏布雷犯規，兩罰全中，盧比奧快速突破完成上籃，庫里三分不中，沙里奇造成波爾津吉斯犯規，兩罰全中，最後2分鐘，獨行俠隊領先11分，關鍵時刻，布克反擊上籃命中，東契奇造成盧比奧第六次犯規，兩罰一中，比賽也失去懸念，最終獨行俠隊在客場以120-113擊敗太陽隊，取得比賽勝利。

在上場比賽表現不佳的東契奇，本場比賽狀態回升，他在首節就拿到14分6籃板6助攻的全面數據，在第三節，他再次拿到16分，全場比賽，他首發出場37分鐘，24投12中拿到42分9籃板11助攻，他是球隊取得勝利的關鍵，小哈德威貢獻26分，波爾津吉斯狀態不佳，全場8投0中拿到2分13籃板，而太陽隊這邊，全隊4人得分上雙，布克拿到18分8助攻，烏布雷22分10籃板，貝恩斯17分7籃板。
