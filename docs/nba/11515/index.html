國王養成記 詹姆斯各種趣事 慈善家 是個近視 差點打橄欖球

2019-12-10 200條評論 2,203次閱讀 54人點贊

2018年騎士在客場拿下對陣凱爾特人的搶七，泰倫-盧賽後用自己的手機流量給詹姆斯開了熱點，好讓詹姆斯可以跟家裡人視頻。更有意思的是，之所以這樣做，是因為凱爾特人方面，並沒有告訴他們客隊更衣室的WiFi密碼。所以我們以後不要調侃「戰術大師」盧指導，不是盧離不開老詹，而是老詹的確離不開泰倫盧的流量啊。這個是不少詹密都知道的事情，也是逐漸挖掘詹姆斯各種逗比的一面。曾在比賽中搶走了觀眾的啤酒，假裝喝酒動作，結果讓這個啤酒銷量大爆；前兩天開拓者主場後退時撞到了保潔阿姨，更是主動拉起跟他合影…等等一系列事情

其實詹姆斯對自己「小氣」也不是什麼段子，因為他童年的關係，勒布朗是單親家庭，所以生活艱難，更是一度生活在貧民窟。所以養成了詹姆斯珍惜的習慣，14歲時，詹姆斯出去打比賽都要隨身帶著出生證明的複印件，因為沒人相信一個14歲的小孩能有這樣的塊頭。後來詹姆斯打出了名堂，要隨隊坐飛機出去參加邀請比賽，那是他第一次坐飛機。在飛機上詹姆斯哭了出來，教練問他原因，他說：「我想到了我在貧民窟的母親，她從未坐過飛機。」

正因為童年有這樣的經歷，所以勒布朗詹姆斯和他的母親在 2004 年建立了勒布朗詹姆斯基金會。他們兩人共同努力幫助兒童以及單親家庭。勒布朗基金會在很多條件不好的地區建造了遊樂場，並且還舉辦過年度超級自行車比賽。

詹姆斯高中之後就聲明遠播了，更是普通高中聯賽就迎來全美直播的動態。高三時，他所在球隊的比賽門票賣到了超過100美元一張，甚至超過了當時大部分NBA球隊普通常規賽的水平。在詹姆斯18歲生日那天，他母親格洛麗亞竟然利用兒子的名氣和潛力向銀行貸款，買了一輛價值8萬美元的悍馬H2給詹姆斯做生日禮物。這一行為違反了俄亥俄州高中生運動員聯合會的規定，最終導致詹姆斯被禁賽兩場。

其實勒布朗在高中時期還是一名非常優秀的橄欖球運動員，高二時被選進州橄欖球隊（相當於中國的省隊），在高三時，他曾考慮去打職業橄欖球。雖然最終沒有選擇職業橄欖球，但他依舊熱愛著橄欖球，就像喬丹熱愛棒球一樣。所以經常看到他在賽場下，玩橄欖球式開球。其實NBA很多球員都是多能的，艾弗森更是幫助自己的高中拿到了橄欖球州賽冠軍。

不過詹姆斯也是小機靈鬼，他知道自己有多出名，他也看到了自己身上的商業價值，所以他做出來一個很機智的舉動，去阿迪的訓練營就穿耐克的鞋，去耐克的訓練營就穿阿迪的鞋。最終詹姆斯和耐克簽下7年9000萬的大合同，作為對比，當年安東尼和AJ簽下的不過一年350萬。

賽場上也是趣事不斷，有一次看比賽，一次暫停，樂福有事兒大喊詹姆斯Lebron、Lebron、Lebron，連叫三聲，估計當時球館聲音也大也雜，老詹沒聽見。樂福又喊了一聲：「the king」，老詹扭頭慢慢走向樂福，當時騰訊的解說也逗樂了。

一位id為pwnetah的網友在Reddit上分享了一個關於詹姆斯的故事。pwnetah一家人住在明尼蘇達，是森林狼隊的季票擁有者，而且座位就在客場替補席後面不遠的地方。詹姆斯轉會熱火後第一次造訪明尼蘇達那天，pwnetah的媽媽戴了一對很漂亮的新耳環。詹姆斯注意到了，誇她的耳環漂亮，問她花了多少錢買的。他媽媽也是個愛開玩笑的人，誇張地回答「很多很多錢。」詹姆斯看了她一眼，「你覺得我買不起嗎？」，全家都被老詹逗笑了。這個玩笑讓詹姆斯記住了這家人，這之後的每一年，當熱火造訪明尼蘇達的時候，詹姆斯總會在身後的觀眾中找到這一家人，沖著他們摸摸自己的耳垂，笑一笑。

07年以前，勒布朗一直有300度左右的近視，但不管是學生時代還是後來進入NBA打球，自尊心一直不允許他戴眼鏡，他曾表示近視眼從未對自己的競技表現造成任何影響。在06-07賽季的總決賽中慘敗給老辣的馬刺後，詹姆斯決定通過激光手術治療近視，據說效果不錯。11年夏天，通過勒布朗的大力推薦，韋德也去做了同樣的手術以矯正視力，後來熱火開啟兩連冠。

2014年熱火客場挑戰勇士，終場前9秒庫里突破查爾莫斯打成2加1，勇士領先兩分，回來詹姆斯三分線外面對伊戈達拉的防守，後撤步三分絕殺勇士，並作出了霸氣的慶祝動作。賽後採訪，記者問詹姆斯：「你這次絕殺和五年前的一次絕殺很像啊，你記得嗎？」詹姆斯說：「噢是啊，五年前我也在這個位置絕殺過，不過那次我撤步投進的是中距離，我還記得防守我的是圖裡亞夫。」 還別說，老詹髮際線跟記性成反比啊。
