湖人備戰奇才沖10連勝存兩憂一喜：AD出戰存疑，最強殺招暴跌7位

2019-11-29 0條評論 15次閱讀 0人點贊

美國的感恩節到來聯盟休戰一天，湖人全隊也都回家過感恩節了，湖人在11月28日與鵜鶘的比賽中險勝4分拿下了9連勝，湖人戰績已經提升到16勝2負，距離湖人隊史最佳開局17勝2負只差一場。湖人目前戰績也高居聯盟第一，詹姆斯與安東尼-戴維斯兩人的組合的確太強大，對陣鵜鶘詹姆斯29+11+末節15分，戴維斯全場41+9+3斷貢獻絕殺搶斷，接下來湖人隊要備戰的是11月30日跟奇才的比賽。

湖人接下來的賽程會變得有些困難，主場對陣奇才後，要打獨行俠、掘金、爵士、森林狼、熱火5支在季後賽圈內的球隊，奇才與開拓者是湖人接下來7場比賽中的兩支弱旅了，11月30日對陣奇才的比賽至關重要，拿下10連勝對於一支球隊有著極大的鼓舞作用，這樣進入魔鬼賽程會更有心理優勢，奇才剛剛客場險勝了太陽，他們狀態不錯，湖人要想拿下奇才並沒有那麼容易，目前湖人存在兩個隱患，但也有一個利好。

第一個隱患，安東尼-戴維斯在對陣奇才的比賽中出戰存疑，根據NBA傷病報告顯示，安東尼-戴維斯右肩酸痛對陣奇才出場存在，有可能出戰，戴維斯除了右肩的傷勢外，在跟鵜鶘對戰時右手手肘也遭遇了撞傷，這讓湖人隊很謹慎，不過安東尼-戴維斯本人表示沒事，他希望出戰對陣奇才的比賽。目前尚不得知沃格爾會不會讓戴維斯打，若場均26.1分9.1籃板3.6助攻1.59搶斷2.82蓋帽的攻防殺器戴維斯不打，那湖人隊要擊敗奇才就沒那麼容易了。

第二個隱患更為嚴重一些，湖人的防守效率都持續走低，本賽季湖人隊的防守效率一度高居聯盟第一，但隨著賽程的深入戰績越來越好，他們在防守端的專註度也在下滑，尤其是最近的幾場險勝的比賽，都贏的很驚險都是靠兩個超級明星在末節完成逆轉，第三節落後成為常態，看了湖人比賽的球迷會有一種獨特的感覺，湖人防守一會兒松，一會兒緊，跟之前的一直緊截然不同，恰好奇才是打進攻的球隊，進攻效率113.5聯盟第二，若湖人還是如此「張弛有度」，那這場比賽湖人大概率要陰溝裡翻船。

利好也有一個，那就是凱爾-庫茲馬的競技狀態明顯回暖了，3分險勝雷霆那場比賽，庫茲馬在末節命中了關鍵三分，對陣馬刺的比賽庫茲馬10分3助攻2籃板發揮也不錯。對陣鵜鶘的比賽他更是超級大功臣，末節三分3中3，拿下9分1籃板1助攻，終場前1分09秒的反超三分更是價值千金，全場8中6，三分4中4，拿下16分5籃板2助攻。庫茲馬在替補席的表現越來越好詹眉的進攻壓力也就越來越小，這對於湖人來說是一個好消息。

湖人本賽季最強殺招就是防守，最近的比賽都贏的很艱辛，這跟防守下滑有很大關係，若要成為頂級強隊，不能一直依靠末節巨星的表現，再度將防守層級提升是關鍵，希望沃格爾重新強調防守，讓霍華德的登場時間稍微多一點點。
