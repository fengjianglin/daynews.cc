耐克給詹姆斯是超巨合同，給科比巨星合同，給姚明是侮辱的合同

2019-12-16 1,378條評論 3,742次閱讀 208人點贊

大家都知道只要是在NBA有點名氣的職業運動員，他們都可以和知名運動品牌簽下代言合同。簽下了代言合同以後他們不僅可以穿免費的球鞋，而且每年也可以掙到不少的錢。那些吸引了極大關注的職業運動員更是有數額非常巨大的合同，接下來就讓小編帶著大家一起來了解一下吧。

當年耐克給詹姆斯7年8700萬合同：之前美國的一位記者就爆料出了這樣一個事情，那就是在2003年詹姆斯沒有進入聯盟之前阿迪和銳步都給他提供了價格非常高的代言合同。但是由於詹姆斯從小就把喬丹看作自己的偶像，所以耐克也成功地撈到了便宜。詹姆斯以7年7700萬的保底合同和耐克簽約，再加上1000萬美元的獎金，他一共拿到了8700萬美元。但是要知道當年銳步為了詹姆斯願意給的合同價格可是高達10年1.5億，我們也只能說耐克確實非常的幸運，但是耐克給詹姆斯是超巨合同的了，下面看了科比的就知道。

要知道詹姆斯在進入聯盟的時候年紀僅有18歲，簽下如此高的合同確實是讓人很驚訝的一件事情。畢竟2003年和球隊簽下新秀合同的他4年才不過掙到了1879萬，年薪連500萬都不到。那個時候科比的代言合同也只有五年4500萬，給科比算是巨星合同，畢竟對比詹姆斯差距好大。從這一點上我們就可以看出詹姆斯到底有多麼優秀，特別是在合同方面看出差距，科比不如詹姆斯。

科比職業生涯依靠代言費一年也只能掙到900萬美元，和詹姆斯相比確實有著很大的差距，為什麼科比只不過是詹姆斯代言合同的一半呢？這其中又蘊含著什麼樣的原因？原來早些年的時候科比和阿迪是合作夥伴，但是由於阿迪的設計理念和科比內心的想法不同，所以這兩個人也沒有長久的合作下去，而和阿迪宣布解約以後科比更是一年內不能夠簽約其他的球鞋品牌，所以在那段時間裡他也沒有代言的收入。

其實在NBA賽場上除了這些美國本土的職業運動員以外也有很多其他外籍球員，這些外籍球員同樣有在NBA賽場上做出偉大成就的職業運動員。像中國男籃的姚明，他在NBA賽場上打出了非常好的表現，職業生涯也賺到了不少的錢，那麼姚明當年簽下的球鞋代言合同又有多少錢呢？

當年姚明進入NBA的時候就吸引了很多人的關注，那個時候耐克也對姚明打起了主意，但是給姚明是侮辱的合同。2002年姚明成為狀元秀以後耐克提供的合同報價僅僅有四年160萬，要知道這樣的數字對一個狀元秀來說顯然是一個侮辱性的工資，所以姚明果斷拒絕了耐克的代言合同。雖然說耐克最後感到後悔，也用7年1.2億挽回姚明這個表現強大的職業運動員，但是姚明卻沒有再次和耐克合作，他和銳步簽下了6年6000萬以上合同，這也使他職業生涯賺到了不少的，通過姚明這個表現我們也可以看出他確實是一個非常有魄力的人。

一個職業球員天下多大的合同與他的賽場表現有著很大的關係，那麼大家有什麼樣的看法呢？歡迎在評論區下方留言分享哦。
