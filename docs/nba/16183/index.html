萊昂納德缺席比賽是為了養傷，路威和貝弗利缺席比賽是因為反抗

2019-12-16 3,008條評論 3,989次閱讀 2805人點贊

快船輸給了公牛

隨著喬治錯失一記扳平比分的三分球，快船爆以106-109輸給了公牛。這個結果應該出乎了所有人的意料。畢竟快船高居西部第二，公牛屈居東部倒數，兩隊的實力完全不在一個等級之上。然而結果就是這樣，快船輸了，還是被對手准絕殺！

出現這種結果的原因有好多種，也包括一些場外因素。

不得不說，公牛的拉文在今天的表現可圈可點，三分線外可以突放冷箭，突破禁區也能造成殺傷，全場比賽打完，拉文狂砍31分，並在最後時刻打成殺死比賽的2+1，實在逆天。除了拉文表現出色之外，其他人也均有建樹，馬爾卡寧貢獻14分17個籃板，楊貢獻17分，瓦倫丁貢獻16分，可以說，公牛贏得了一場團隊勝利。

快船這邊，內線悍將哈雷爾狂砍30分7個籃板，喬治得到27分6次助攻，沙梅特11分，帕特森11分6個籃板。光看這些數據，快船的整體表現還算不錯。殊不知，他們疏忽了自己的立足之本，沒有將防守強度持續到最後一刻。

在比賽的最後10秒鐘，雙方戰成106-106平，公牛的拉文執行最後一攻。只見他僅僅利用一個掩護，就擠出了突破空間，並在哈雷爾的協防到來之前完成上籃動作，最終打成2+1。

輸球的背後

不是快船不想防，而是他們心有餘而力不足。在比賽開打之前，快船官方對外宣布萊昂納德，路威，貝弗利將會缺席本場比賽。球隊三名大將缺席比賽，快船輸球似乎也有情可原了。

萊昂納德缺席比賽的原因我們知道，畢竟他的傷病還沒有完全康復。

路威繼續缺席比賽的原因應該和萊昂納德有關。據消息人士透露，萊昂納德和路威可能產生了隔閡，兩人在隊內訓練賽中沒有任何配合。除此之外，近日路威也在自己的社交媒體上更新了一條動態：「我不是為了他而就在這裡，兄弟！」

除了路威之外，貝弗利也沒有出席本場比賽，他的行為似乎在間接地支持路威。兩人曾在休斯頓打球，後來又一起來到快船，其感情肯定非同一般。

種種跡象表明，快船似乎產生了更衣室矛盾，並且牽扯人數不止一人。如此一來，快船將有可能送走路威等人，以討好萊昂納德和喬治。畢竟，萊昂納德為首的鋒線組合才是快船的最利矛頭！

毫無疑問，萊昂納德帶領著快船是奔著總冠軍去的，但想要贏得總冠軍的前提要先贏得人心，確保更衣室穩定和諧。不然，一切都是笑談！如果不能及時解決更衣室的問題，希望快船的管理層快刀斬亂麻，儘早考慮一下取捨問題，將損失降到最小化。

截止目前，快船的戰績來到20勝8負，位居西部第二。
