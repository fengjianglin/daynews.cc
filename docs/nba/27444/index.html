字母哥18+19+9+2+2，雄鹿大勝步行者繼續穩坐聯盟第一

2019-12-23 120條評論 1,192次閱讀 162人點贊

北京時間12月23日，雄鹿在主場迎來步行者的挑戰，憑藉著揚尼斯·阿德托昆博18分19籃板9助攻2封蓋2搶斷的准三雙數據，雄鹿以117比89大勝步行者，終結了對手的五連勝。

馬修斯拿到19分，大洛佩茲拿到17分8籃板2封蓋，喬治·希爾拿到17分9籃板4助攻。步行者這邊小薩博尼斯拿到19分18籃板5助攻2封蓋2搶斷的全面數據，布羅格登拿到10分10助攻。

開場步行者就給雄鹿來了個下馬威，小薩博尼斯三分線外出手命中，拿到本場比賽的第一分，也帶領步行者打出10-0的小高潮。雄鹿在字母哥的率領下逐漸找回狀態，隨著節末字母哥的一條龍灌籃，雄鹿將分差追到只落後1分，以21比22結束第一節。

第二節雄鹿進攻狀態大開，特別是替補出場的喬治·希爾，他單節三分球3投3中，獨得13分，雄鹿也單節得到38分，在半場結束時反超步行者4分。到了第三節，雄鹿更是打出一波24-7，將分差瞬間擴大到21分。

雖然小薩博尼斯在節末命中三分帶領球隊打出一波11-0縮小分差，但是第四節大洛佩茲連續命中兩記三分，幫助球隊打出一波19-9，也讓比賽失去了懸念。

賽前布羅格登在接受採訪時曾表示：「如果雄鹿想要我，如果他們像步行者那樣看重我，我本很願意為他們打球。」本場比賽也能看到布羅格登強烈的進攻慾望，但是雄鹿隊也非常了解他，對布羅格登做了針對性的防守，本場他19投僅5中，沒能在雄鹿主場證明自己。

在收穫本場比賽勝利之後，雄鹿隊以27勝4負的戰績穩居聯盟第一，雄鹿主教練布登霍爾澤也收穫了執教以來的第300場勝利。
