本賽季至今發生6大慘案：恩比德被防到吞零蛋，無加時158分還輸球

天天要聞
2019-12-03

2019-2020賽季雖然才剛剛進行了將近4分之一的比賽，各支球隊還處在陣容的磨合期，有不少球員還在尋找比賽的狀態，但是令人沒有想到了是，就在這20來場比賽中，就誕生了不少震驚聯盟的慘案，下面小編給大家介紹一下賽季至今發生的6大慘案。

快船隊49分血洗老鷹

2019年11月17日快船坐鎮主場對陣來訪的老鷹隊，在這場關注度不高的常規賽中，主場作戰的快船隊在沒有加時的情況下，全場狂砍150分，並且只讓對手老鷹隊拿到了101分的得分，以49分的巨大分差血洗了老鷹隊，創造了開賽階段最大的一場慘案，而本場比賽中，剛剛傷愈復出的喬治表現出色，在僅僅登場20分鐘的情況下，就17投10中，三分11投6中，罰球11罰11中砍下了37分4籃板3助攻2搶斷1蓋帽的數據，不僅效率奇高，更是讓老鷹隊無法防守。

恩比德首吞零蛋

上賽季季後賽中，大多數球迷心中的聯盟第一中鋒恩比德，就在和小加索爾的比賽中連戰連跪，被防守的大失水準，本以為在11月26日，本賽季76人對陣猛龍的首場比賽中，恩比德能夠一雪前恥，證明自己聯盟未來第一中鋒的實力，可沒有想到的是，恩比德不僅再次跪了，而且跪的更為徹底，甚至完全可以稱得上是開賽至今的第二大慘案，在那場比賽中，面對小加索爾的防守，恩比德全場11投0中，三分4投0中，罰球3罰0中，職業生涯首次得分掛零，如果不是還有13個籃板撐場面的話，恩比德估計真沒有臉面去對76人球迷了，不過再這樣下去的話，恩比德對位小加索爾時會不會產生心理陰影就不好說了。

馬刺被活塞34分血洗

2019年12月2日馬刺客場對陣活塞，本來很多人認為，這將會是一場「菜雞互啄」的比賽，畢竟開賽初期不管是馬刺還是活塞的表現都令人大失所望，可是誰都沒有想到，活塞這隻「菜雞」變成了猛虎，而馬刺因為阿爾德里奇的缺戰，菜到了連菜雞都不如的程度，最終活塞隊以132比98，34分的巨大分差血洗了馬刺隊，而在比賽中，甚至連活塞隊的底薪球員伍德，都可以面對馬刺隊的防守予取予求，以78.6%的投籃命中率砍下了28分10籃板2助攻1搶斷1蓋帽的數據，這簡直是直接逼著波波維奇老爺退役的節奏啊。

奇才隊單場158分輸球

之前聯盟流傳著一句話，叫做「如果單場砍下120分還沒有贏球，那可能是碰上了勇士」，但這句話放在奇才隊身上就有點不合適了，因為在沒有加時賽的情況下，奇才隊單場砍下158分依然輸掉了比賽，還有比這更慘的慘案了嗎？可沒辦法，誰讓他們在比賽中碰到了手感一旦順了，簡直是人擋殺人佛擋殺佛的馬賽克隊，僅僅是哈登、威少和卡佩拉三人，在這場比賽中就聯手貢獻了97分，最終馬賽克隊以159比158，1分的優勢險勝奇才。

雄鹿隊44分血洗尼克斯

老球迷應該都聽過這句話，叫「NBA中無弱旅」，但是在現在的聯盟中卻有些不在適用，在12月3號，東部第一雄鹿隊對陣東部倒數第一尼克斯隊的比賽中，尼克斯全場比賽沒有展現出一絲的還手之力，半場結束時就落後雄鹿隊27分，等到第三節比賽結束，雄鹿隊更是將分差擴大到了38分，最終雙方的比分結果是132比88，雄鹿隊以44分的巨大分差血洗了尼克斯，創造了本賽季開賽至今的又一大慘案。

雄鹿41分血洗黃蜂

本賽季的雄鹿隊簡直變成了神話小說中的九色神鹿，上文剛剛提到了12月3號的比賽中雄鹿44分血洗了尼克斯，其實早在兩天前，12月1號對陣黃蜂的比賽中，雄鹿就已經創造過了一出慘案，在那場比賽中，雄鹿隊始終佔據著賽場上的優勢，在第三節比賽結束時，就以101比75領先黃蜂隊，讓比賽失去了懸念，第四節比賽盡出替補的雄鹿隊不僅沒有被黃蜂隊追進比分，反而分差越拉越大，甚至連公認的水貨本德爾都在比賽中砍下12分，最終雄鹿隊以137比96，41分的巨大分差戰勝了黃蜂隊。
