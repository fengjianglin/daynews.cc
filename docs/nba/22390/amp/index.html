湖人輸球甚至兩連敗都不可怕，若3連敗，可能得用總冠軍定律了

天天要聞
2019-12-19

輸球了，湖人隊在客場對陣步行者的比賽中拿到了本賽季的第四場失利，目前已24勝4負的戰績與雄鹿並列聯盟第一。更有趣的一點就是，湖人將在下一場比賽中對陣雄鹿隊，對手也是剛剛被獨行俠終結了18連勝。聯盟的賽程安排真是湊巧啊。

換一句話說：不管湖人與雄鹿在下一場比賽哪一支球隊贏球，輸球的一方都將遭遇本賽季的首次連敗。而客場102-105輸給步行者，也是湖人本賽季第一次客場輸球，下一場比賽能否止住呢？

對於湖人來說，在常規賽輸一場比賽並沒什麼大不了的，而且還是在連續客場之旅中輸球，同時球隊的核心之一濃眉也因為傷病問題缺席了比賽，庫茲馬也缺席了。本場比賽濃眉的缺席確實影響了湖人的進攻與防守，沒有濃眉步行者可以肆無忌憚的夾擊詹姆斯，重點控制湖人的外線三分，所以本場比賽詹姆斯的表現算不上好，湖人的三分球也投出了31中8的慘淡手感。

但是關於詹姆斯與濃眉對於湖人，何人為老大問題的討論，實屬扯淡。畢竟沒有哪支球隊通過一場比賽就直接否認一名球員的。

在常規賽輸球並不可怕，可怕的是球隊在連敗後沒有緊迫感，尤其是不能出現長連敗（3連敗即算長）。

在絕大多數總冠軍球隊中，常規賽三連敗的球隊少之又少，如果沒有記錯的話有2002年與2010年三連總決賽後的最後一個賽季。其中在2011年湖人常規賽拿到了57勝的戰績，可是球隊出現了4次三連敗，當年湖人也沒能進入總冠軍的爭奪行列。

本賽季的湖人，截止目前還未出現過連敗。雖然在下一場對陣雄鹿的比賽中可以遭遇賽季的首次連敗，但是問題也不算太大。一旦出現連敗，那麼湖人絕對不能鬆懈。如果客場不敵雄鹿，那麼湖人在回到主場後對陣掘金的比賽中，決不能大意。

因為三連敗對於一支將總冠軍視為目標的球隊，是一道坎。就像在2009年湖人拿到了65勝，那個賽季球隊雖然沒有打出驚天動地的長連勝（幾乎都是5-7場的連勝），但是球隊總能在關鍵時刻展現出緊迫感。就像不敵馬刺與魔術遭遇兩連敗後，球隊在第三場對陣東部第一的騎士時，用半場就搞定了騎士。

下一場對陣雄鹿，雖然雙方的連勝算是神同步終結，但是東西部第一之間的比賽依舊充滿看點。雙方將在周五進行比賽，屆時濃眉戴維斯應該也大概率能上場打球，而詹姆斯的狀態或許也能有所提升。下一場湖人雙星挑戰上賽季MVP球員字母哥，會有怎樣的火花飛出呢？

湖人、雄鹿都是本賽季總冠軍的競爭者之一，火星撞地球！拭目以待吧。
