昨天砍42分累著了？小卡再休戰，芝加哥庫里絕殺球領跑全聯盟

2019-12-16 300條評論 1,208次閱讀 191人點贊

今日，洛杉磯快船來到芝加哥背靠背挑戰公牛，昨日他們剛剛經過苦戰拿下了森林狼，「王炸」組合的實力在昨天發揮的淋漓盡致，當代「喬丹和皮蓬」一同發威，齊砍40+，喬治拿下46分7助攻，小卡拿下42分11籃板，其中罰球19罰19中，在創下了個人罰球生涯最高紀錄外，也刷新了快船快船隊史上在沒有罰丟情況下命中罰球數的最高紀錄。

或許是昨日用力過猛的緣故，或許是沒把芝加哥公牛放到眼裡的緣故，小卡在今日的背靠背作戰中選擇休戰，本賽季至今為止，小卡一共為快船出戰了20場常規賽，尤其在喬治復出之後，小卡三天一小休五天一大休，有理由或是沒理由總是要選擇休息，看來是在為季後賽保養自己的身體和蓄力。

對於芝加哥公牛而言，年輕球隊居多，扎克.拉文被視為球隊的絕對核心，雖然到今天為止，公牛僅僅排在東部第11的位置，但是相信他們在經過一段時間的沉澱和積累過後，一定會再次重新回到奪冠的行列，一直被冠以扣籃王稱呼的扎克.拉文，在球迷的印象當中永遠都是球風飄逸，能突能扣，可本賽季為止，他的成長似乎格外迅猛，甚至上演單場13記三分球，搶斷三分絕殺的重頭大戲。

本場比賽，公牛以下犯上能拿下勝利，扎克.拉文功不可沒，在關鍵的追分時刻，拉文先是面對哈雷爾在三分線外運球一記後撤步干拔三分拔進，沒有讓比分拉開，最後一個回合，強行突破內線，在快船最強硬的身軀哈雷爾防守之下，對抗後上籃打進2+1，完成准絕殺，這兩次進攻難度之大，含金量之高，是檢驗一個巨星成色的重要標準。

而反觀快船這邊，小卡，路威，貝弗利，三大將全選中休戰，喬治大將軍一人苦苦支撐，雖然小將沙梅特及時復出，5投4中，狀態還算不錯，但對於喬治的幫助還是遠遠不夠，在比賽最後階段，喬治兩次進攻失手，特別是最後一個回合，幾乎在空位的情況下，出手三分球偏出，錯失絕殺的機會，對此他個人也表示十分遺憾。

據統計，到今天為止，芝加哥公牛是第一次戰勝勝率百分之50以上的球隊，作為13號順位，1.98米的身高，拉文沒有給這個順位和身形的球員傳承丟人，41.1的三分命中率，24歲芝加哥庫里的未來無限光明。
