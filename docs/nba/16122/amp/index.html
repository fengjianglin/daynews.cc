10年前NBA五大控衛！新秀羅斯上榜，德隆力壓保羅，96一代霸榜

天天要聞
2019-12-16

NBA進入21世紀之後，迎來了一個控衛崛起的盛世，一個控衛擋拆主導進攻的時代正在悄悄到來。05-06賽季時，讓我們還在感慨一個時代能有納什和基德兩位控衛大師有多難得時，沒過幾年保羅和德隆的一時瑜亮就把15+10的控衛標配數據推到了20+10。

而羅斯和威少的勁爆雙煞又把這個標準定在了25+8。事實上，在我看來，那時候就是控衛小盛世，而保羅的經久不衰，威少和羅斯的年少成名都保證了他們把天賦帶到了今天。

第五名：克里斯-羅斯

羅斯初入聯盟便立下了赫赫戰績，季後賽第一場，面對全聯盟防守第一的凱爾特人，羅斯就狂砍36分。此前一個新人，第一場季後賽就得到類似紀錄的是：賈巴爾，1970年。

與凱爾特人鏖戰七場，羅斯場均得到20分6籃板6助攻。當時與羅斯對位的是，巔峰時期的隆多。

進入NBA聯盟第二年的羅斯便已經成長為一個全明星球員了，當賽季他場均能夠得到20.8分3.8籃板6.0助攻，並且在本戈登等人離開之後，獨自帶隊依舊殺入了季後賽。儘管首輪不敵詹姆斯帶領的騎士，但是季後賽的羅斯場均可以得到26.8分3.4籃板7.2助攻。

三年級，NBA史上最年輕的常規賽MVP。場均得到25分7.7助攻。當時的羅斯可能是NBA歷史上，身體最好的組織後衛。

191公分，臂展203公分，原地起跳34英寸，助跑跳40英寸，84公斤卧推10次，底線折返跑11.7秒，3/4場衝刺3.09.

第四名：隆多

09-10賽季，隆多可能是東部最好的控衛，在三巨頭年齡逐漸增大的情況下，隆多場均可以得到13.7分4.4籃板9.8助攻和2.3搶斷，投籃命中率是50.8%，凱爾特人常規賽也取得了50勝32負的戰績。

到了季後賽，在三巨頭已不服當年之勇的情況下，隆多憑藉著場均15.8分、5.6籃板與9.3次助攻，將綠軍重新帶回總決賽。

皮爾斯後來回憶那一年季後賽時說：「他當時真的是扛著我們前進，如果我們最終奪冠，他應該是總決賽MVP。」

第三名：克里斯-保羅

2005年第四順位被黃蜂選中的小個子後衛，能跑能跳，上天入地，無所不能。加入黃蜂后，立竿見影，把上賽季還遠離季後賽的黃蜂硬是帶到了西南區第一，西部第二的位置。保羅也得以回報，入選最佳陣容一陣，最佳防守二陣，MVP第二，僅次於科比。

如果是前兩年的話，我們會說保羅是當時全聯盟最好的控球後衛，不過要是單輪2009-10賽季，保羅左膝半月板大傷導致他缺席了大量的比賽，整個賽季只出戰了45場比賽，場均得到18.7分4.2籃板10.7助攻，那一年黃蜂也只贏了37場比賽，最後無緣季後賽。

第二名：德隆-威廉姆斯

這一個賽季，因為保羅的傷病，德隆是更好的那個，當賽季他場均能夠拿到18.7分4籃板10.5助攻，率領爵士取得了53勝29負。

季後賽，德隆又帶領爵士擊敗了掘金，系列賽當中德隆場均可以得到25.8分11.2助攻，儘管次輪輸給了湖人，但是的德隆依然可以場均拿下22分8.8助攻。

榜首：納什

2009-10賽季的納什已經是36歲的高齡了，賽季初，媒體普遍看衰太陽，認為他們能進入季後賽就燒高香了。然而納什憑藉著場均砍下16.5分3.3籃板11助攻，當選當賽季助攻王，並且率領太陽取得了54勝的西部第三戰績。

季後賽36歲的納什帶著球隊一路殺入分區決賽，狠狠打了專家的臉。只是他的職業生涯仍然難逃悲劇色彩。西部決賽和科比帶領的湖人鏖戰6場，最終被湖人淘汰的那個夜晚是納什第118場季後賽，他砍下了21分9助攻，在從未進入過季後賽的球員中，他的得分位列第一。

賽後納什哭的不能自己，他知道自己再也沒有機會完成進入總決賽的夢想了！那一年，湖人進入總決賽後，擊敗了凱爾特人，科比拿下了生涯第五冠。

然而對於96黃金一代來說，2010年是一道分水嶺，此後即使科比和納什並肩作戰，他們也沒能再次殺入分區決賽，屬於他們的時代就這樣悄然落幕了！
