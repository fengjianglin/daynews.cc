湖人快船到底誰強？看完你就知道了

2019-12-18 650條評論 2,464次閱讀 140人點贊

對於快船我們都知道，喬治和萊昂納德都是鋒線，兩人都得有球，這就得在場上看看誰更加有手感，那就會有個問題:兩人都要出手試試誰更有手感，但是我們大家都知道，在比賽中根本就沒有那麼多的機會來讓你試出手感，這是毋庸置疑的。

快船不想湖人這邊，戴維斯第一主攻點，如果戴維斯這點被限制住，詹姆斯還能在接上。在詹姆斯休息的時候，隆多能把湖人的第二陣容接著續上火力。反觀快船這邊贏球的時候特別好看，但是這全年下來，不可能一帆風順，等到比賽進入季後賽，球隊什麼都會發生改變。比如說：防守強度會加強，針對性會提高，個人的手感也會隨著比賽的進行發生變化，但只有一點事永遠不會改變，那就是你個人的高度，還有就是你的臂展。高度臂展一般都是乘二，也就是說翻著倍的長。另外一個到了季後賽當中，當防守強度加大之後，陣地就多了，這節奏就會適當的慢下來，那麼這個時候高度優勢就會更加的明顯。現在全聯盟在高度這塊能和湖人匹配的隊伍都在東部，那就是雄鹿和76人隊。

另外還有一個重要的因素，快船的陣容還沒有達到完美的陣型，意思就是還可以補強，還有簽一個內線球員，但是湖人這邊不需要了，因為有戴維斯這種級別的鋒線，還能蹦，能跳，能搶，有活力。快船的哈雷爾的確很猛，但是身高也很有限，剛好缺一個像貝弗利這種防守型球員，萊昂納德和喬治都可以組織球隊，但他們都不是控衛，在1號和5號位能不能加強是快船現有的問題。

所以從陣容上來講，湖人的陣容堪稱完美，放眼全聯盟，奪冠的第一熱門讓然是湖人隊！
