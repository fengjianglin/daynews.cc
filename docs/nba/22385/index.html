球星必殺技第一期，杜蘭特常用技術講解！

2019-12-19 2,113條評論 3,532次閱讀 2436人點贊

凱文·杜蘭特，美國職業男籃運動員，效力於布魯克林籃網隊，司職小前鋒。

2014年當選常規賽MVP，2017年NBA總冠軍及總決賽MVP，2018年NBA總冠軍及總決賽MVP！四屆得分王，歷史最年輕得分王！

常用技術第一招：變向急停跳投

通過平常看比賽得出的感覺就是，速度節奏很平穩，不快不慢，但假動作很難看出，大跨步的交叉變向後突然起跳投籃，加以身高的優勢，防守球員的手根本沒有辦法干擾到他的投籃，球進不進完全來源於kd的手感！

常用技術第二招：大跨步交叉變向突破

利用身高腿長優勢，大跨步的運球使防守球員失去平衡，然後一步或兩步突破上籃，或扣籃。經常看到kd一步過人，然後送上暴力扣籃！

常用技術第三招：外線強投

這可是kd打球風格中欣賞性最高的方式，漂移的投籃方式，球總是在他的手中能划出完美的弧線然後空心入網，這是我最喜歡kd的得分方式，就是無解，就是頂著你的頭上得分！

常用技術第四招：急停跳投

快速運球的情況下突然停住腳步起跳利用對手反應不過來的情況下出手，這招是每個打球的球員都會的，關鍵是kd不會被干擾，並且投籃手感極其的柔和，這就是為什麼人們說杜蘭特防不住的原因了吧！

常用技術第五招：背身單打翻身後仰

我們時常可以看到杜蘭特若側要球，試探步，三威脅，或者直接過掉，再或者直接干拔！但我最喜歡的還是翻身跳投，動作嫻熟，手感柔和，穩定出手，然後皮球應聲入網，這樣的球，好球！

常用技術第六招：借擋拆接球空位出手

場上的kd總有人緊跟著他，不敢放他空位，但他總會找到機會擺脫防守接到隊友傳球輕鬆得分，這也是每個球員都會的，但kd命中率是真的高啊！

常用技術第七招：外線接球強投

作為一名小前鋒，擁有2.06米的身高，在外線坐等接球投三分也是kd經常使用的得分方式，本來就防不住，還出手這麼快！

看kd打球怎麼久了，平常打球也是愛模仿他的動作，但是一直被模仿，從未被超越啊，kd是一名技術型球員，他的打法不依靠身體，但他的速度是他厲害的一個基礎，如今跟腱斷裂勢必會影響到他的速度，但就憑他那一手精準的投籃就足以讓聯盟顫抖！

最後祝願kd早日康復，死神的氣息依然瀰漫……
