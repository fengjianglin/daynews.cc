全員得分送騎士7連敗，海沃德復出10中7，開局23場，正式開始擺爛

2019-12-10 2,409條評論 3,831次閱讀 2095人點贊

北京時間2019年12月10日，波士頓凱爾特人主場迎戰克利夫蘭騎士隊的比賽，此役在上個月遭受骨折傷病的綠軍大將戈登-海沃德迎來了復出。面對實力較弱的騎士隊，綠軍誓必要在用一場大勝給海沃德復出送一份大禮。

比賽一上來綠軍就展示了自己出色、鐵血的防守，同時在進攻端也發揮出色，第一節比賽就以29-18領先騎士2位數之多。

此後每節比賽都將騎士的分數壓制在20多分左右，最終在主場以110-88取得一場大勝，取得4連勝將自己的名次提升到東部第二，同時也送給了騎士隊7連敗，現在的連敗場次僅次於聯盟倒數第一的尼克斯隊。

詹姆斯走的這2年騎士又回到了年年爭奪狀元的年代了，在加上前段時間將主力核心凱文-樂福擺上貨架，剛開局23場比賽，騎士隊就開始宣布正式擺爛了。

而樂福自己也不爭氣，連續幾場比賽都表現低迷，對陣活塞拿下8分、對陣76人拿下6分、這場比賽對陣綠軍又只拿下7分，賽季至今的場均數據直接降到了15.6分，近10個賽季最低，看來前段時間和女友出去漂流落水被送醫院後，狀態就大打折扣啊！

反觀綠軍這邊全隊12人上場比賽，12人均有得分入賬，沃克命中5記三分球，拿下全場最高的22分還送出了7記助攻。

雙探花杰倫-布朗和塔圖姆近期的表現是越來越出色、越來越穩了，兩人本場比賽加起來一共貢獻了39分18籃板4蓋帽的數據，攻防兩端都非常的出色。

光著3人本賽季至今就可以貢獻62.9分的得分，也是東部唯一一支隊內有3人場均得分超過20+的球隊，就連東部第一的雄鹿隊隊內都沒有這樣的情況。

別忘了他們陣容中還有海沃德，這名曾經爵士的白人球員領袖。這名全明星的球員，在沒有遭受骨折傷病前，海沃德的表現也是非常的好的。

經過一個賽季的調整他漸漸找回了自己的狀態，場均18.3分6.9籃板和4.1次助攻的數據，得分和助攻均排在球隊第二，依舊是這麼的全面。

復出的第一場比賽，海沃德也有著上佳的表現，那麼久疏遠賽場一回來能有這麼好的狀態也再一次證明了他回來了。

出戰26分鐘10次出手命中7球，拿下14分5籃板4助攻，多次妙處隊友得分引爆全場。

作為當今聯盟最頂尖的組織型前鋒，海沃德的復出也大大的解放了沃克在進攻端，讓他能更專註的去得分，也大大提升了綠軍在首發上的高度和陣容的深度。

斯瑪特搭檔沃克時，綠軍這外線的身高確實是太矮了，總的來說這個賽季的海沃德已經慢慢的找回了狀態，雙探花進一步的成長，眾將之間的磨合也漸入佳境，綠軍現在這鋒線配置完全不遜快船啊！

雖然沒有了歐文，反而就得他們更加整體、更加強了，這賽季的綠軍仍舊是很有競爭的一支球隊！
