復出14天經歷太瘋狂！甜瓜自曝曾考慮退役：我不怪那兩支球隊

2019-12-04 0條評論 17次閱讀 0人點贊

過去14天，對於安東尼而言，實在是一段瘋狂的經歷。他回到了NBA，他重新成為了首發球員，他在一場比賽里三分8投0中，卻在另一場比賽里11中9，他成為了西部周最佳球員……在芝加哥打客場時，現場的公牛球迷甚至在他被換下場後還高喊著：「WE WANT MELO（我們要甜瓜）!」

一切如夢似幻，安東尼在接受知名主持人尼科爾斯採訪時感慨道：「真的太棒了，畢竟我有一年沒打比賽了，感覺一切都發生了轉折，精神上情感上都是如此。」

過去兩個賽季，安東尼的生涯很不順利，他聽到了太多負面的評價。「自私、糟糕的隊友、更衣室毒瘤……」說到這兒，安東尼覺得很無奈，他不知道這些評價因何而來，「我甚至都做好準備了，做好準備離開賽場。」

主持人尼科爾斯重複了一遍：「你做好準備離開了？」

安東尼遲疑片刻後還是說道：「是的，我當時是準備離開了。因為每一天早上醒來太艱難了，不知道該期待什麼，不知道該做些什麼，不知道會發生什麼。」

過去這一年裡，安東尼甚至養成了散步的習慣。他需要通過這種方式來思考自己的境遇，去排解這種鬱悶。在採訪中，安東尼表示自己不怪那兩支放棄自己的球隊，因為他知道一切都是生意。

前往開拓者打球，意味著和自己的家人暫時分離，特別是自己的兒子。之前一場比賽，安東尼的妻子開啟視頻，讓他們的兒子通過攝像頭看到球場上的安東尼。不過，他們都為安東尼感到高興，因為安東尼終於找回了久違的快樂和熱愛。

對於如今的安東尼來說，考慮退役已經成為過去時。現在的他，只想好好繼續打球，為球隊做貢獻。當媒體提到「告別之旅」的話題時，安東尼笑了。「為什麼？為什麼我剛回來你們就想讓我離開了？要知道我休息了一整年，我的身體狀態好得很！」
