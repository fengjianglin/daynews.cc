唐斯發來賀電！恩比德被小加打爆！生涯第一次這麼慘

天天要聞
3 週 ago

誰是當今聯盟第一中鋒？約基奇？唐斯？恩比德？懸念不會太大，當表妹大傷之後，很多人會把票投給恩比德。但是，大帝卻打出了生涯最差一戰。

北京時間11月26日，猛龍主場101比96戰勝費城76人。恩比德11投0中，罰球3中0，14次出手全打鐵！0分13板2助0帽4失誤5犯規。

本場比賽，猛龍形象大使Drake也來到場邊觀戰，他還給恩比德鼓起了掌，並喊道：「MVP！」早在上賽季的季後賽，兩個人的梁子就結下了。當時猛龍領先近30分，大帝下場，Drake在場邊模仿他的慶祝動作。旁邊觀眾也跟著起鬨……

去年大帝接受採訪時，曾經表態：自己最喜歡的歌手是Drake。換位思考一下，當你最喜歡的明星，在場邊嘲諷自己，這心裡陰影面積該有多大？

嘲諷完大帝之後，Drake心情大好，跟范喬丹以及女兒合影。小姑娘可能有點懵：這倆大鬍子這麼像，到底哪個才是爸爸？當然了，後面搶鏡的洛瑞也是亮點。

此前恩比德生涯最低得分為5分，而且那場球只打了9分鐘。從5分降到0分，這一切都是拜小加所賜，從上賽季季後賽開始，他就在防守端死掐大帝。這場比賽小加索爾數據也不咋滴，8投1中，3分6板9助2帽1失誤3犯規。

可就算兌子，猛龍無疑也是賺大了。恩比德賽後顯然很沮喪：「我從未想過自己會在NBA比賽里拿0分，但現在這就是事實。猛龍在防守端無處不在，他們身高體長，包圍了禁區！」

第四節關鍵時刻，恩比德在罰球線接球，持球往籃下殺！當他剛轉身，阿奴諾比就站好了位置，進攻犯規！兩次關鍵回合，大帝一次打鐵一次失誤，又是一臉茫然。教主其實很期待這倆球隊在季後賽相遇，針鋒相對，有恩怨也有故事。小加以後可能要換個座右銘了：吃飯睡覺打大帝。看完這場比賽，唐斯也許要給小加發個簡訊：哥，我想跟你拜個把子。
