陳林堅命中關鍵三分瓊斯45分 福建勝吉林取四連勝

2019-12-21 88條評論 2,301次閱讀 1132人點贊

北京時間12月20日，CBA常規賽第19輪繼續進行，福建主場迎戰吉林。福建小外援杰特首秀砍下22分15次助攻，陳林堅得到26分並命中關鍵三分。吉林這邊，瓊斯空砍45分無力回天。

全場比賽結束，福建121-114力克吉林斬獲四連勝。

數據

福建：杰特22分15次助攻，陳林堅26分，黃毅超16分，王哲林28分10個籃板5次助攻，康寧漢姆18分；

吉林：崔晉銘10分，瓊斯45分6個籃板6次助攻，代懷博15分，姜宇星10分，霍爾曼19分13個籃板；

首節比賽，王哲林開場打成2+1助福建以5-2領先，代懷博立即命中三分追平，兩隊很快進入狀態，大打對攻。下半節，吉林改守聯防，王哲林連續攻擊籃下得手，他單節拿到11分和4次封蓋。吉林這邊不甘示弱，姜偉澤和崔晉銘合砍5分建立領先，王哲林節末連送妙傳追分。首節比賽結束，吉林24-23領先福建。

次節比賽，康寧漢姆中投得手帶隊反超，霍爾曼回敬上籃，福建傳球失誤，吉林打出5-0小高潮！下半節，崔晉銘與陳林堅對飆三分，杰特三分還中助福建打平。吉林命中率下滑，王哲林獨得4分！瓊斯艱難拿到2分，福建打出8-2！上半場比賽結束，福建47-41反超吉林。

第三節比賽， 瓊斯屢中拋投追分，無奈福建手感滾燙內外開花，王哲林打成精彩2+1！霍爾曼內線強攻取分，福建則連中三分不斷擴大優勢。吉林投籃持續失准，主教練王晗不滿判罰被吹技術犯規，福建一舉領先12分！霍爾曼強攻籃下得手，王哲林帶球突破再打2+1！第三節比賽結束，福建89-77領先吉林。

末節比賽，吉林開局命中率奇高將分差縮小到6分，福建被迫叫暫停。暫停回來，黃毅超命中三分，福建趁機還了一波7-0！吉林加強防守，瓊斯連傳帶突再度迫近比分。福建失誤頻頻，吉林將比分追至104-104！福建隨後連中兩記三分，瓊斯和代懷博拿到6分，王哲林上籃得手，瓊斯三分投進助吉林落後2分！關鍵時刻，陳林堅命中制勝三分，吉林卻發球違例。全場比賽結束，福建121-114力克吉林斬獲四連勝。

雙方首發

福建：杰特、陳林堅、何重達、黃毅超、王哲林

吉林：崔晉銘、姜宇星、瓊斯、代懷博、鍾誠
