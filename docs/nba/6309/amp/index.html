鐵血的“腎鬥士”——阿朗佐莫寧

天天要聞
2019-12-03

邁克爾-喬丹統領了九十年代，他帶領芝加哥公牛完成兩次三連冠，讓眾多英雄好漢空手而歸。但實打實講，九十年代是中鋒盛世，哈基姆-奧拉朱旺、帕特里克-尤因、大衛-羅賓遜、沙奎爾-奧尼爾四大中鋒名震江湖。其實四大中鋒除外，還有不少實力派中鋒，阿朗佐-莫寧就是其中的代表，而且莫寧的職業生涯更曲折、更勵志，也更讓人欽佩。

位列奧尼爾之後的榜眼秀，新秀賽季不輸奧尼爾，而且他與球隊的矛盾也潛移默化地改變了聯盟的格局。

1988年加入聯盟的夏洛特黃蜂在當時是一支年輕的球隊，年輕意味著的戰績方面無法獲得突破，球隊的選秀權就會比較靠前。90-91賽季他們只獲得了26勝56負的成績，之後他們獲得了狀元簽並選中拉里-約翰遜。接下來的91-92賽季，黃蜂取得了31勝51負的成績，他們又很幸運地獲得了榜眼簽，在奧蘭多魔術毫無懸念地選中奧尼爾之後，黃蜂選擇了來自喬治城大學的阿朗佐-莫寧。

現如今球迷會用「開天眼」的方式調侃「有喬選喬、無喬選鯊」，其實已經從側方面反應了奧尼爾的天賦。新秀賽季的奧尼爾交出了場均23.4分13.9個籃板1.9次助攻3.5次封蓋的表現，可實際上新秀賽季的莫寧並不輸奧尼爾。莫寧在新秀賽季交出了場均21分10.3個籃板1次助攻3.5次封蓋的成績單，雖然無緣最佳新秀，但莫寧的表現還是獲得了肯定。接下來的兩個賽季，莫寧都保持了高水準的輸出，攻防兩端的他都相當優異並且已經成為全明星中鋒。職業生涯第三個賽季結束後，莫寧與球隊之間產生了矛盾，而且莫寧與球隊的矛盾也潛移默化地改變了聯盟的格局。

1996年夏天，奧尼爾選擇加盟洛杉磯湖人，同時湖人用當打之年的中鋒維拉蒂-迪瓦茨換來了科比-布萊恩特，後來奧尼爾與科比的「OK組合」帶領湖人拿到了三連冠，可以說湖人在1996年夏天的操作從一定程度上改變了聯盟的格局，而湖人的這筆操作與莫寧也有一定關係。

94-95賽季結束後，莫寧與球隊主教練產生了矛盾，所以他拒絕將自己的合同延長。擔心合同到期後人財兩空的黃蜂在1995年11月3日將莫寧、皮特-邁爾斯、阿隆-埃利斯交易至邁阿密熱火，換來了場均可以貢獻22.3分4.6個籃板2.3次助攻的小前鋒格倫-萊斯以及馬特-蓋革、克哈里德-李維斯等人。按說球員交易是很正常的情況，可是當交易完成後，黃蜂的內線開始吃緊。隨著萊斯在95-96賽季成為全明星球員，黃蜂既堅定了以他為核心的想法，同時得到一名中鋒的想法也越來越強烈。

擁有已經成為全明星的萊斯，黃蜂並不會將太多的精力投入到年輕球員，所以他們才會在1996年夏天與湖人完成交易。雖然此後科比在湖人成為歷史級的球星，但當時黃蜂的選擇並沒有錯誤，擁有全明星球員萊斯，他們不會對一個未來充滿未知數的高中生科比感興趣，已經比較成熟的中鋒迪瓦茨更適合當初的黃蜂。所以，莫寧在1995年夏天與球隊的分離以及他被交易至熱火從某種程度上影響了黃蜂未來的選擇，也是促成黃蜂與湖人交易的關鍵因素。

正值巔峰卻被病魔困擾

被交易後，莫寧開啟了熱火生涯。他與蒂姆-哈達威以及賈馬爾-馬什本的組合讓熱火成為東部的一支勁旅，可惜當時的熱火實力強勁卻遇到了第二次回歸的喬丹以及同樣強悍的紐約尼克斯等球隊，熱火的戰績沒有進一步突破，不過莫寧領銜的熱火從來沒有退縮過。要知道，莫寧身高只有2.08米，屬於矮個中鋒，他能夠稱霸內線的基礎就是強悍的身體以及強硬的作風。

可是，就是這般強悍的人卻在職業生涯巔峰時期遇飽受病魔困擾。98-99賽季以及99-00賽季，莫寧連續兩個賽季榮膺聯盟蓋帽王以及最佳防守球員，他也逐漸迎來了職業生涯的巔峰。就在莫寧希望繼續帶領熱火衝擊冠軍的時候，2000年10月，他被診斷出患有嚴重的腎臟病症，之後他便缺席了00-01賽季的大部分比賽，此後在2001年3月份復出打了13場比賽，狀態已經出現下滑。接下來的01-02賽季，莫寧打了78場比賽並有場均15.7分8.4個籃板1.2次助攻2.5次封蓋的表現，可賽季結束後，莫寧的病情再次惡化，他也不得不接受腎臟移植手術，同時也缺席了整個02-03賽季。

之後莫寧與新澤西籃網簽約，但是在2003年11月25日，醫生診斷莫寧已經無法從事高強度運動並勒令他停止一切活動，莫寧也被迫宣布退役。但是此後的莫寧仍然沒有放棄重新回到賽場的信念，他決定再一次復出。只不過籃網已經不能再等待莫寧，他們為了得到文斯-卡特將莫寧送到了多倫多猛龍。此後莫寧拒絕去猛龍報道，而猛龍最終以莫寧無法通過體檢以及不願融進球隊在2005年2月11日將其裁掉。

再回熱火，煥發第二春，終圓冠軍夢！

2000年被診斷出患有腎病的時候，30歲的莫寧正迎來職業生涯的黃金時期。當他在2005年被猛龍裁掉的時候，莫寧已經35歲了，他職業生涯最好的時光已經被病痛消磨。如果是常人，估計職業生涯會隨著被猛龍裁掉而終結，或者說在剛被診斷出患病的時候已經終結。可是莫寧並非常人，被猛龍裁掉後，他再次加盟熱火，成為了奧尼爾的替補。

接下來的05-06賽季以及06-07賽季，莫寧很好地擔任起奧尼爾替補的角色。05-06賽季他在場均20分鐘的出場時間裡可以送出2.7次封蓋、06-07賽季他在場均20.4分鐘的出場時間裡送出2.3次封蓋，這都是相當高效的表現。當然了，莫寧在進攻端的表現已經大不如前，可是他能夠站在場上就已經是個奇蹟，甚至還能夠在防守端有貢獻，這已經足夠，而且莫寧還在05-06賽季圓了自己的冠軍夢。

替補生涯也有閃光之處、英雄離場方式更令人欽佩

再次回到熱火後，莫寧在大部分時間裡只能是奧尼爾的替補，但是這仍舊無法掩蓋莫寧的光芒。2005年11月29日，熱火主場迎戰尼克斯，莫寧頂替出現傷病的奧尼爾出現在首發陣容中，他在24分鐘的出場時間裡4投3中得到9分7個籃板並送出9次封蓋，9次封蓋追平莫寧職業生涯單場封蓋紀錄，可是那場比賽的9次封蓋卻令人感動。在那場比賽之前，莫寧有7次單場送出9次封蓋的表現，可那都是在換腎之前的表現，而2005年11月29日的9次封蓋是莫寧換腎之後的又一次單場9次封蓋。還有2006年總決賽第六場，替補出場的莫寧在14分鐘的時間裡貢獻8分6個籃板5次封蓋，這些都是莫寧在用生命打球的閃光之處。

不僅如此，莫寧的謝幕方式也令人欽佩。2007年12月20日，熱火客場挑戰亞特蘭大老鷹，首節比賽莫寧與老鷹隊馬里奧-韋斯特相撞，他的右膝髕骨肌腱完全撕裂。當時熱火的球員烏杜尼斯-哈斯勒姆、教練帕特-萊利和隊醫都衝過去查看莫寧的傷情，在發現傷情非常嚴重的時候，他們決定將莫寧抬上擔架。就在這個時候，莫寧掙扎著站了起來：拿走該死的擔架。之後他平靜地望著萊利、哈斯勒姆和德維恩-韋德說道：聽著，我一生經歷的太多太多，我不認為這是我最後一次走下球場，如果有一天我不得不爬著離開，我也要走出去，沒有人可以用擔架把我抬走，永遠沒有……

之後，莫寧喊來韋德攙扶自己而且還重重地捏了捏韋德的肩膀，當時韋德已經成為總決賽MVP，莫寧捏了捏韋德的肩膀似乎將拼搏的火炬傳遞給了韋德，那個3號和33號的背影永遠留在了球迷的腦海中。

與四大中鋒相比，莫寧的身材並不佔優勢，2.08米的身高相對較矮，這使得他在比賽中有些吃虧，但是莫寧在健身房用汗水鑄就了強悍的身體，再加上頑強的意志，莫寧在比賽中從來都是鐵血硬漢。後期的腎移植更是讓莫寧生不如死，在那段艱難的歲月里，莫寧需要像吃飯一樣每天吃下一大把藥片，但是莫寧從來沒有放棄回到賽場的信念。喬丹在職業生涯里曾經有過退役又復出的情況，類似1993年的退役或許有獨孤求敗的味道，可是莫寧不同，他幾次被醫生勒令離開賽場卻又掙扎著回到賽場，內心那份對籃球的愛已經完全可用行動詮釋，他是真正的硬漢。

2009年3月31日，熱火為莫寧退役球衣；2014年4月7日，莫寧入選奈-史密斯籃球名人紀念堂；2019年3月27日，入選2019屆FIBA籃球名人堂，這是英雄該有的榮譽，莫寧是真正的鬥士！
