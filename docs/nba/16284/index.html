18分鐘砍11+6，詹姆斯大發神威！12分鐘10分，熱火瓊斯輸出高效！

2019-12-14 243條評論 4,054次閱讀 461人點贊

北京時間12月14日上午，NBA常規賽迎來熱火與湖人的對決。熱火正取得三連勝，而且過去七戰贏了六場，其中有三場比賽是通過加時鏖戰拿到勝利的，由此可見熱火的韌性還是很強的。熱火上一戰以135-121擊敗老鷹，戰績來到18勝6負，主場豪取11連勝，熱火展現出恐怖的進攻水準，首發有三人得分超過30分，兩人砍下三雙數據，比如阿德巴約貢獻30分11個籃板11次助攻，巴特勒送出20分18個籃板10次助攻。熱火本賽季曾以80—95輸給過湖人，今日一戰應該會打出更好的進攻表現。

湖人上一戰在客場以96—87擊敗魔術，將連勝場次擴大至五場，戰績來到22勝3負，追平隊史最佳開局紀錄。勒布朗詹姆斯在與魔術的比賽中拿到25分11個籃板10次助攻，安東尼戴維斯20投6中貢獻16分12個籃板和6次助攻。湖人將在今日迎來五連客的第二場比賽，詹姆斯將重回邁阿密，值得一提的是，湖人在客場正經歷12連勝。

首節上來，倫納德，羅賓遜三分入網，巴特勒行進間扣籃，納恩突破扣籃，急停跳投，阿德巴約空切上籃，湖人這邊，麥基扣籃，勾手擦板，格林，詹姆斯三分入網，麥基，戴維斯補扣，比分來到14平。暫停回來，巴特勒突破上籃，急停跳投，行進間上籃，瓊斯三分入網，湖人這邊，戴維斯空中接力，詹姆斯突破上籃，比分來到24：21，湖人暫時落後3分。暫停回來，戴維斯補扣，布拉德利漂移中距離，撤步跳投，卡魯索突破上籃，熱火這邊，希羅急停跳投，比分來到30：29，湖人暫時領先1分。

次節開戰，阿德巴約突破上籃，納恩行進間上籃，瓊斯行進間扣籃，湖人這邊，布拉德利急停跳投，比分來到41：32，湖人暫時落後9分。暫停回來，詹姆斯，戴維斯突破上籃，比分來到38：41，湖人暫時落後3分。暫停回來，席爾瓦空切上籃，勾手擦板，奧利尼克上籃，跳投，希羅三分遠投，巴特勒行進間扣籃，湖人這邊，波普勾手打進，補籃命中，空切上籃，霍華德補扣，戴維斯三分入網，比分來到59：51，湖人暫時落後8分。

熱火：巴特勒10分3藍板3斷，奧利尼克8分3藍板3助攻，希羅5分2藍板，瓊斯10分2藍板2助攻2搶斷，阿德巴約8分4藍板3助攻，萊昂納德3分1助攻，羅賓遜3分1藍板，納恩6分3助攻。

湖人：詹姆斯11分6藍板3助攻，濃眉哥15分5藍板，麥基6分6藍板，波普6分1藍板，格林3分，布拉德利6分2藍板，霍華德2分2藍板，隆多2藍板4助攻。
