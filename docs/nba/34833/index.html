執教300勝！雄鹿主帥喜迎里程碑 沖第3座最佳主帥獎追恩師波波

2019-12-24 304條評論 1,778次閱讀 441人點贊

北京時間12月23日，當雄鹿在主場以28分優勢大勝步行者，雄鹿主帥布登霍爾澤也是喜迎里程碑，他正式迎來執教生涯第300勝里程碑。如今布帥正率領雄鹿一枝獨秀領跑全聯盟，無疑他有望蟬聯最佳主帥獎，也將是衝擊生涯第3座最佳主帥獎，從而追平恩師波波維奇的獲獎次數。

布登霍爾澤作為學院派教練的代表，他的教練生涯起點是1996年開始在馬刺擔任助教，成為NBA名師波波維奇的門徒。布帥在經過13年磨練後，在2009年成為馬刺首席助教，並在馬刺任職17年。

直到2013年布帥開始正式成為NBA主教練，他在2013年開始執教老鷹，並且幫助老鷹成為東部勁旅之一，尤其是在2014-15賽季幫助老鷹取得創隊史紀錄的60勝，也是在該賽季首次奪得最佳主帥獎。

及至2018-19賽季，布帥開始執教雄鹿，並且上賽季率領雄鹿取得創隊史紀錄的60勝，他也是順利斬獲生涯第2座最佳主帥獎。在正式出任主帥的6個賽季當中，兩次奪得最佳主帥獎的布帥無疑成為NBA現役優秀教練的代表人物。

本賽季布帥繼續執教雄鹿高歌猛進，哪怕上賽季雄鹿在東決2-0領先情況下，竟然連輸4場被猛龍大逆轉淘汰，但新賽季的雄鹿無疑在繼續升級提升。字母哥越來越全能，雄鹿的攻防統治力也是更加強悍，還在此前接連吊打洛城雙雄。

布帥也是率隊在本賽季收穫27勝4負的戰績，不僅高居東部第1名，同時也是穩居全聯盟第1名，畢竟湖人在接連傷病侵襲後，如今戰績下滑已經被雄鹿逐步拉開差距。

布帥自此達成生涯執教300勝里程碑，他目前累計執教戰績為300勝223負，勝率達到57.4%。雄鹿保持如今的態勢，本賽季再獲60勝問題不大，甚至有望取得更好的成績，而布帥則是有望蟬聯最佳主帥獎，一旦他再次獲獎就將追平恩師波波維奇的獲獎次數。

（文/醉卧浮生）
