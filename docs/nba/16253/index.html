鵜鶘是怎樣與東部第二的76人纏鬥到最後？

2019-12-14 787條評論 4,800次閱讀 2854人點贊

鵜鶘11連敗是預料之中的事——字母哥不打的雄鹿鵜鶘都能慘敗，霍福德不在的76人就更不用說了。

這場看首節的趨勢，又有些朝著慘敗的趨勢走，而最終鵜鶘卻出乎意料地把比賽懸念保留到了最後45秒雷迪克的那記「神奇」三分之前，不可謂不殺人誅心。

面對東部豪強76人，鵜鶘為何能打得這麼頑強呢？

費沃斯的回歸

鵜鶘球迷心心念念的費沃斯在千呼萬喚之後終於於本場迎來了復出。由於久疏戰陣的緣故，他在進攻端的感覺顯然還需要再找找，當然大家對他的期望也不在個人得分上。

看看這球，鶯歌和費沃斯手遞手，後者用身體嚴嚴實實的擋住西蒙斯，鶯歌繞掩護後來到中距離干拔輕取兩分。

本場前高達47.6%的中投命中率，可以預見如果能有不錯的擋拆球員，鶯歌的中距離跳投威脅將會讓很多球隊在防守端付出代價。但鵜鶘本賽季的內線，唯一掩護質量還算不錯的費沃斯，卻在此前缺陣了13場。

當鵜鶘能有像費沃斯這樣會打擋拆的球員在場時，好處還不止於讓鶯歌在中距離獲得更多得分機會：一旦鶯歌的中距離吸引到對手內線的防守注意力，他也能獲得不少吃餅機會。

更難能可貴的是，一向擋不住人的奧卡福，今天彷彿是看了費沃斯的擋拆後開了竅。

第二節開端，奧卡福給球哥來了個結結實實的掩護，讓後者得以擺脫內托的防守。大帝負責鎮守內線，看球哥跑出三分線自然不會再追，內托在被擋之後一直沒能跟上球哥的步伐。最終，球哥在左側45度角開炮命中三分。

出色的團隊配合

霍勒迪運球到罰球線附近，傳給右側45度角落位的鶯歌，面對理查德森的撲防鶯歌把球傳給低位的海耶斯，海耶斯不停球再給退到弧頂的霍勒迪，霍勒迪看到防哈特的哈里斯過來補防，馬上把球塞給哈特，哈特一個假動作後直接過掉疲於奔命的哈里斯奔襲籃下得手。

一次進攻，經過4次傳導球，在過了4人之手後最終出機會得分。鵜鶘的配合要能這麼多打打，大家何必非要頂著防守人強投呢？

球哥今天有一次助攻讓人印象深刻，他在右側45度角三分線外無視內托的防守強突到籃下，面對大帝的補防把球傳給身後的費沃斯，費沃斯順勢上籃打成2+1。

當然，如果球哥的基本功再好一些，籃下終結能力再強一些，這樣的突破分球對他來說只能算是常規操作。

防守可圈可點

大帝今天24+11+5的數據看起來相當不錯，但投籃僅僅16投6中，看了比賽就知道，鵜鶘在大部分時間對大帝的針對性防守做得很成功。

大帝低位接球背打費沃斯，球哥果斷夾擊迫使大帝出球。大帝停球後，費沃斯舉起手干擾大帝傳球視線，球哥則不停叨球分散大帝注意力，鵜鶘其他三人分別擋在可能的傳球路線上。大帝無奈只能冒險傳球給弱側底角，不想卻被早有準備的沃克成功截下。

76人想要給要到籃下的大帝做球，鵜鶘這邊除了盯防大帝的費沃斯，沃克球哥趕忙過來掏球讓大帝不能舒服接球，好不容易拿到球的大帝，轉過身來又要面對費沃斯的防守，最終倉促出手被籃筐斷然拒絕。

類似的防守還有幾次，就不一一列舉了。話說回來，鵜鶘要是早這麼防，還能十一連敗？

總的來說，鵜鶘這場球在攻防兩端都有一些亮眼表現，雖然這樣的表現不管是只從本場而言還是放眼整個賽季都很難有持續性，但如果年輕人們能從中嘗到甜頭並讓類似的攻防表現更多的出現在比賽中時，相信他們會在接下來的比賽中以肉眼可見的速度成長起來。
