詹姆斯承認：當年總決賽輸給小牛，是因為被對手激怒了

2019-12-12 1,049條評論 3,944次閱讀 136人點贊

2011年總決賽，在眾人完全看好的情況下，擁有3巨頭的熱火2-1領先的情況下連輸三場，遺憾錯失總冠軍戒指！而影響最大的詹姆斯終於談到了當時遇到的問題！

若要說詹姆斯生涯最遺憾的一年，那也莫過於這一年總決賽！熱火組成NBA最豪華的三巨頭，卻輸給了諾維斯基單核帶隊的獨行俠（當時還稱為小牛隊）

2010年夏天，詹姆斯躊躇滿志在全美直播中宣布「將天賦帶到東海岸」，這本身就引起了非議，再加上總決賽發生的家庭問題和「復讀機」對手，勒布朗已經完全失控！

詹姆斯在最後3場關鍵戰役，連續得到17分、17分和8分，成為了當年最大的談資。

不過有趣的是，在早些時候面對NBA記者Dave關於「心態」的提問，詹姆斯非常坦誠的說道：自從2011年輸給小牛隊，我開始意識到我的心態是有問題的！

「我完全被現場的球迷給激怒了！」詹姆斯回憶道：我想要找回場子，不合理的出手讓我逐漸失控，即便是過去8年，那幾個夜晚仍然印象深刻。

而在這這一年的總決賽，詹姆斯最後3場的命中率不到38.6%，三分球共計11投1中，但出手此次卻達到了系列賽最高！

從那以後，詹姆斯學會了如何在2萬人的咆哮中正確打球：我開始明白，身體上的強大是不夠的。

其實詹姆斯的狀態早有端倪！

在代表騎士和死敵凱爾特人的季後賽中，因為受不了母親格洛里亞的碎碎念（同時也怕母親被聯盟處罰），詹姆斯直接對著母親咆哮讓他閉嘴，結果還是死敵加內特和皮爾斯過來攔住了老詹。
