NBA最魔性的球隊！不論你看好還是看衰雷霆，他都能打到你的臉

天天要聞
2019-12-11

今年夏天，雷霆陸續失去了喬治和威少，而得到的只是保羅和亞歷山大，於是人們都覺得雷霆這是準備擺爛了。結果賽季已經過去了三分之一，雷霆非但沒有擺爛，反而還擠進了西部前八，即便是輸球的場次也有很多是拼到最後。事實上，這就是雷霆這支球隊最魔性的地方，無論你看好還是看衰他們，你都會被無情打臉。

截止到目前，雷霆隊排名一共取得11勝12負，排名西部第7。這支雷霆隊里只有保羅一個全明星級別的球員，但他已經34歲了，早已不是當年的那個第一控衛了。可就是這樣一支雷霆隊，竟然可以在西部如此激烈的競爭下還不掉隊，這就已經足以用神奇來形容了。

而這也是這支雷霆最魔性的地方，那就是不論你是看好還是看衰他們，你最終都會被他們無情打臉！上賽季，雷霆的工資總額排名聯盟前三，並且因為倆年繳納奢侈稅而觸發了超級奢侈稅，但他們並沒有因此而走得更遠，最終只能眼睜睜看著利拉德在他們頭上完成那記偉大的絕殺。

而在更早的一個賽季，當時雷霆幾乎成為了那個休賽期的最大贏家，先後通過一些當時看起來不痛不癢的籌碼換回了喬治和安東尼，再加上亞當斯等人，那支雷霆隊的紙面實力比起擁有四大天王的勇士隊也不虛。結果那個賽季他們也只是艱難進入季後賽，首輪就被爵士隊4比2淘汰。

2016年夏天，杜蘭特離開雷霆加盟了勇士，當時幾乎所有人都不看好雷霆能夠進入季後賽。結果那個賽季威少突然爆發，開啟了連續三個賽季場均三雙的序幕，並且幫助雷霆打進季後賽。和之後兩個賽季的首輪游不同，當時的威少在人們眼裡是一個孤膽英雄，他也通過歷史級別的數據從哈登手裡搶走了MVP。

事實上，雷霆雙少時期也是如此，當人們看到杜蘭特和威少紛紛受傷的時候，都會覺得這支球隊沒有希望了，但他們總是有人可以站出來，總是可以打出驚人的表現。但在2016年大比分3比1領先勇士被人們無限看好的時候，他們又被對手翻盤了。

雷霆三少時期，他們曾經打進過總決賽，但那個賽季他們其實並不被看好，西部的馬刺、湖人都比他們更有冠軍相，但他們就是能夠在西部脫穎而出，並且和擁有三巨頭的熱火大戰5場。第二年失去哈登之後，他們得到了當時比哈登能力更出色的馬丁，結果人們都認為他們還能衝擊總冠軍，但從那之後卻再也沒能打進總決賽。

沒錯，這支雷霆隊就是這麼魔性。當你看好他們的時候，他們總是會有各種各樣的問題，但當你不看好他們的時候，他們又總會莫名其妙的團結。所以夜侃君覺得雷霆球迷倒是不妨來個曲線救國，以後不論他們陣容多麼豪華合理，都別去看好他們，很可能會受到不錯的效果呢！
