巴克利：76人還不如猛龍 毫無機會進總決賽 奧尼爾：恩比德不夠拼

天天要聞
2019-12-11

北京時間12月11日，據《NBC》網站報道，前NBA球員巴克利在做客體育節目時談到76人，他表示對於之前在休賽季預想76人會進總決賽的預測，感到機會渺茫，同時他和奧尼爾也對恩比德進行了評論。

NBC網站報道截屏

「恩比德確實是NBA最難的球員，但是我們談論他的時候，不像在談論東契奇、字母哥和戴維斯一樣，我們不會像評價這幾個人一樣去評價恩比德，這讓我感到沮喪。」巴克利在節目中說道，在之前的休賽季，76人不單以5年1.8億美元的合同留下哈里斯，還以4年1.09億美元引進全明星中鋒霍福德。當時巴克利曾經認為這支重金打造的76人，會突破上賽季的季後賽成績，殺進總決賽。

但本賽季開賽至今，76人的表現差強人意，這支豪華軍團在開賽季時戰績一敗塗地。雖然之後表現有所回暖，但巴克利和奧尼爾還是對其非常失望，奧尼爾更是直指恩比德的不足，「你可以做的很出色，但是你確實不夠努力，你是想成為偉大的球員，還是只是個優秀的球員，如果只是想過優秀的球員，就繼續每場比賽只拿22分吧，如果是像成為偉大的球員，那每場比賽至少給我拿到28分、30分，字母哥就是想成為偉大的人。」奧尼爾說道。恩比德本賽季場均22分10籃板和6助攻。

76人目前以18勝7負，排名東部第4，而此前不被看好的凱爾特人位列東部第2，猛龍排在東部第5。對此，巴克利說道：「當時我們都在說，雄鹿和76人有機會打進總決賽，當時是這兩支球隊，但現在凱爾特人打得非常好，猛龍也是，我不想把話說絕，但我已經開始有這種感覺了。」
