39分9板3帽！濃眉攻防兩端展現統治力 爭MVP+DPOY還只是空想嗎？

2019-12-07 0條評論 18次閱讀 0人點贊

湖人隊對陣開拓者隊的比賽，賽前被安東尼與詹姆斯的再次賽場相逢帶入溫情，但是離愁別緒只存在於場下。場上無兄弟只有對手，詹姆斯帶領湖人隊延續著近期的強勢表現，安東尼-戴維斯更是在場上極富侵略性。全場比賽出戰32分鐘，21投12中，罰球15罰13中，得到39分9個籃板2次助攻2次搶斷以及3次蓋帽。

詹姆斯和戴維斯

作為湖人隊休賽期最重要的一筆引援，安東尼-戴維斯承載著湖人隊復興的希望。從上個賽季開始，湖人隊就已經為打造「詹眉」組合而努力，直到休賽期才終於如願以償。湖人隊對戴維斯滿是期待，身穿紫金戰袍的戴維斯又何嘗不是對未來充滿想像呢？他們的合作，是彼此夢想的啟航。

進入聯盟第八個賽季，安東尼-戴維斯雖然聲名在外，卻始終沒能拿得出過硬的成績。這對於安東尼-戴維斯來說，是不能接受的事情，一名球星只有帶隊打出出色戰績，才更能夠被人所接受。安東尼-戴維斯在被質疑的聲音中前行，像是一個孤獨的旅行者，抬頭仰望星空心中無限心事。

積極地防守

被鵜鶘隊交易到了湖人隊，安東尼-戴維斯的夢想被激活，像是小孩子吃到了一顆香甜的草莓糖。戴維斯加盟到湖人隊之後，詹姆斯將球隊的主攻角色交給戴維斯，戴維斯成為湖人隊最重要的核心球員。戴維斯的表現沒有讓人失望，本賽季場均能夠貢獻26.6分，成為湖人隊隊內得分王。

安東尼-戴維斯在進攻端是當之無愧的大殺器，湖人隊在很多時候都靠他打開局面。而戴維斯可不是只做這些，在防守端他做得還要更多，更是湖人隊的防守大閘。戴維斯給自己的防守提出了更高要求，他要做霍華德跟加內特的結合體——霍華德那般防守端統治油漆區，干擾對手的投籃；又如加內特那般堅韌。

防守端要做得更好

戴維斯在攻防兩端表現得都很出色，深得隊友和媒體的認可，這是對他努力的最好回饋。戴維斯帶病出戰幫助湖人隊擊敗掘金隊，隊友拉簡-隆多評價戴維斯說道：「他每晚都吸引包夾，現在他也能投三分，他為隊友創造機會。球場內外都是傑出領袖。我支持他當選MVP和DPOY。」

在NBA歷史上能夠同時拿到MVP和DPOY的球員屈指可數，安東尼-戴維斯的確是很有希望做到這一點的球員，2017-18賽季戴維斯就在兩項評選中都進入到了前三位。戴維斯並不缺乏能力，他只是需要一個平台，而湖人隊則為他提供了最好的機會。這並不是空想，戴維斯為此在努力地奮鬥著。
