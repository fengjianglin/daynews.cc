詹姆斯，雖千萬人吾往矣，何懼之有

2019-11-27 0條評論 15次閱讀 0人點贊

究竟什麼才能真正定義勒布朗·詹姆斯？

是文字上的氣勢磅礴與澎湃洶湧，還是音樂中的雷霆萬鈞與叱吒風雲，亦或是圖片里的波瀾壯闊和氣貫長虹。

當你想要很好的去描述一個人時，你可能會從他非凡的特質上去著手，用一段段精美華麗的文字，用一曲曲烘託人物氣質個性的音樂，用一幅幅刺激感官視覺的圖畫。

如果是一般人，這樣幾方面的描述可能已經足夠。可如果對象是勒布朗·詹姆斯，這些手法似顯得那麼平庸啊。

因為，單憑文字、音樂與圖畫，根本不足以來形容勒布朗·詹姆斯。面對這樣的英雄，再華美的文字與音樂，再恢弘大氣的圖畫，彷彿都只是寥寥眾生，根本無法還原其在直觀上給你帶來的最直接的感受。

2018年4月26日克利夫蘭的速貸中心，天王山的氣氛瀰漫在場館內的每一個角落。這個賽季，對於克利夫蘭人而言，是艱難的。兩套完全不同的人員配置，體現著騎士管理層的深深無奈。

賽季之初，人們感嘆著，這是一支全新的騎士，歐文的離開不僅沒有降低球隊的水準，反而在人員配置上，讓整個克利夫蘭更是期待。

以勒布朗，韋德，羅斯，小托馬斯，樂福等全明星球員組成的陣容，使得整個克利夫蘭地區充滿歡呼。

人們說，這是騎士隊史上最出色的陣容了。從賬面上看，新騎士帶給人們的直觀感受不弱於曾經湖人的豪華F4。顯然，這樣的人員搭配，讓騎士更有底氣去掀翻西部的王者勇士。

那時，騎士似乎期待又一座奧布萊恩杯了。

後面的故事，大家都知道。由於陣容磨合與戰績壓力，韋德、羅斯與小托馬斯被相繼交易，騎士迎來了大換血。多位年輕球員的加入，讓騎士看起來煥然一新。可這樣的直觀變化，依然沒有給騎士帶來顯著性的提升與幫助。

跌跌撞撞，以東部第四的身份進入季後賽。起起伏伏，在首輪就遭到步行者的頑強抵抗。在勒布朗帶隊的這些年裡，還從來沒有在季後賽第一輪遇到這樣的艱難險阻。

首輪被逼入天王山，誰也沒有想到。

比賽進入最後的一分鐘，客場作戰的步行者與騎士打平。勒布朗持球從右側45度角殺入，面對賽迪斯·楊的防守，勒布朗居然掉球出了底線。賽迪斯·楊完成了防守，逼迫勒布朗出現嚴重失誤，把比賽主動權攥到了己隊的手裡。

拼殺了40分鐘的勒布朗累了，他撐了撐膝蓋，臉上流露出了一絲沮喪，可隨後，勒布朗馬上回到了替補席，與球隊商量著防守。

最後8.7秒，勒布朗提到中場線附近。面對步行者的全明星領袖奧拉迪波，勒布朗步步緊逼，在奧拉迪波突入籃圈的一剎那，勒布朗一記釘板大帽扇飛了奧迪的上籃。隨後騎士守住球權，喊出暫停。

終場前3秒，勒布朗在邊線接到格林的發球。面對博格丹諾維奇的延誤與賽迪斯·楊封到指尖的防守，勒布朗虛晃一步高高躍起，在距離弧頂的三分線還有一步的位置，稍稍帶點後仰，投出了最後的絕殺球。

這記如同炮彈一般的三分球轟進了網窩，一旦眼神稍不注意，你甚至無法反應皮球已經被扔進網袋。

速貸中心沸騰了，沒有人能安靜的坐下來，（除了步行者的替補席）人們高舉著雙手，享受著勒布朗帶來的所有驕傲。

鏡頭裡，邁爾斯·特納一臉懵懂，彷彿不相信眼前發生的一切。他反覆看著大屏幕上的回放，以確定這顆球是在沒有任何違例下被投進的。

在步行者球員還在耷拉著腦袋，一再回看進球的瞬間時，勒布朗跳上了看台，雙拳怒錘著胸口，發泄著之前所積累著的全部怒氣，似乎在說：

雖千萬人吾往矣，何懼之有！
