現世報來了！勇士狂輸熱火科爾就差明說，聯盟恐已無格林容身之地

天天要聞
2019-12-02

雖然已經時隔兩天，但是回頭再想想勇士和熱火的比賽依舊讓人痛心。稱霸聯盟多年，不僅在邁爾密以105-122大敗，而且是敗在了他們曾最擅長的三分球。坦白講，這已經不是勇士隊今年第一次大比分輸球，可賽季至今16負4勝的戰績看著真讓人揪心。

​事情發展到這一步確實充滿戲劇性，誰能想到之前5個賽季中獨霸聯盟的王朝球隊，居然隕落得如此徹底如此的快速。或許已經接受將要擺爛的事實，所以賽後面對採訪時也不再滿面愁容，反而有點釋放重擔後的坦然。

「雖然挺著比較古怪，但確實很享受這個賽季，球隊里的年輕球員發揮得非常出色」科爾面帶微笑的輕鬆回應道。巧婦難為無米之炊，核心陣容的流失讓科爾也是無力回天，科爾之所以現在還能笑得出來，或許已經把目光瞄向未來。好在帕斯卡爾和喬丹普爾兩位新人的表現令人眼前一亮，這或許也是科爾目前唯一可以慰藉的點吧。

話雖然沒有明說，可從科爾的話中也能看得出來，隊里的「老將」已經讓他傷透心了。這位老將不是別人，正是把勇士「禍害」到現在這種境地的罪魁禍首-格林。其實從最近科爾對格林的使用上就能看出點門道，出場時間被限制，雖然依舊首發，可戰術核心已經跟他沒多大關係了，很多時候格林只是新秀們的陪跑者罷了。

古人有云：自作孽，不可活！格林就是這種人的典型。逼走杜蘭特就算了，為了自己利益還能理解，但看到球隊戰績已經滑落谷底還要一味作死。本身球隊戰績下滑所有人都在懷念杜蘭特，但他偏偏站出來放炮說自己之前數據下滑是被杜蘭特搶了球權，真是可笑之極。現在呢？球隊大佬們都掛病號了，作為球隊老大的他場均7分6籃板6助攻，說好的生涯巔峰呢？真不知哪來的勇氣說的那些話。

經過這麼長時間的檢驗，格林已經證明了他就是跟在庫里和湯普森身邊的一個偽巨頭而已，離開體系他什麼都不是，如果現在勇士不要他，估計聯盟中真的沒有他的容身之地了，沒有哪支球隊傻到會把個禍害帶回去的地步，曾今不可一世，如今現世報說來就來，可悲啊！
