可怕的年輕天賦，他們的上限在哪裡！未來聯盟的新勢力？

2019-12-06 0條評論 13次閱讀 0人點贊

北京時間12月6日NBA賽事不多，而今年備受關注的黑馬對陣年輕無上限的鵜鶘無疑是焦點之戰，果然沒讓球迷失望加時過後才分勝負，最終今年的黑馬太陽更勝一籌139-132客場戰勝鵜鶘，暫時在前八的位置得到了呼吸權，不過這場比賽的的觀賞性絕對是有目共睹，同樣是兩個年輕球隊潛力無限。

布克為首的太陽一直是進攻犀利的球隊，布克的能力更像是進攻端的一把尖刀，盧比奧是運籌帷幄的將領，這就成就了現在太陽黑馬的瘋狂，在狂野的西部前期能穩住前八的位置多少球迷都覺得很驚訝，而且他們的體系還很年輕如果再在交易市場開啟後來點補強，季後賽的席位應該有他們。不得不說一句盧員外真香！！！

相較於同樣年輕的鵜鶘，當鶯歌每場一樣那麼穩定，當球哥投進4個三分，當他們首發全民皆兵5人得分上雙4人上20的時候，我覺得這樣的球隊肯定應該引起聯盟強隊的注意了，當他們全員健康，當那個狀元郎回歸的時候，這又是一個怎麼樣的球隊，他們的未來是一個怎麼樣的，沒人說的清楚，至少我覺得季後賽還是可以進的。

狀元郎胖虎會是一個怎麼樣的成就沒人知道，可能是奧登類型的玻璃人，也可能是天賦比肩詹姆斯的怪物，他的上限也決定了鵜鶘的上限，未解之謎就看他復出會不會給人驚喜了！鵜鶘這股新勢力會一鳴驚人嗎？
