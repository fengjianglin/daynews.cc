他們在德安東尼手下打不了球確都在新球隊打到了主力他真不會用人

2019-11-30 0條評論 14次閱讀 0人點贊

本場比賽勇士隊雖然輸掉了比賽，但是他們還是有發揮出色的球員的，比如他們的替補球員馬基斯-克里斯。克里斯本場比賽上場24分鐘，3投1中，罰球8罰8中砍下了10分5籃板6助攻，表現非常全面。說實話，克里斯的得分和籃板數據我們並不感到驚訝，我們感到驚訝的其實是他的助攻數。作為一名身體素質勁爆，但技術相對粗糙的內線球員，在這場比賽中我們可以看到他在策應方面的進步，我想這也是科爾打算讓他好好開發的另一項技能。

波特蘭的復甦不能說完全是因為新援安東尼一個人的功勞，但他卻居功至偉，他為這支球隊注入了渴望贏球的慾望。本場比賽，安東尼上場37分鐘，16投8中，三分球3投1中，罰球7罰6中，得到全隊第二高分23分，並搶下11籃板，送出4次助攻，完成1次搶斷，正負值+10，全場最高

安東尼正用自己的實際行動證明火箭當初的做法是多麼的愚蠢，一位歷史級別的NBA巨星怎麼也輪不到成為背鍋俠，更不應該被冷藏，被無情地交易，況且還是苦苦追求了幾年的追求者。火箭你個害人精，差點因此毀掉了安東尼的職業生涯，活該你現在不受關注，真是天道輪迴！
