火箭背靠背戰活塞，庄神卡皇最強籃板之爭，哈登能否連續三場50+

天天要聞
2019-12-15

在剛剛結束的NBA常規賽中，休斯頓火箭做客奧蘭多輕鬆戰勝魔術，哈登連續高效發揮，出戰36分鐘，31投19中，三分球15中10，罰球6中6，砍下54分5籃板7助攻，連續兩場得到50+，投進10記三分，並且罰球都在6個以下，歷史第一人。明日，火箭將回到休斯頓主場迎戰活塞，哈登能否延續自己火熱的手感，幫助球隊拿下三連勝？

活塞的內線有德拉蒙德和格里芬坐鎮，護框和籃板非常強，德拉蒙德的籃下終結能力很好，而外線有曾經的MVP羅斯，帶領加洛韋、肯納德、斯內爾、米哈伊柳克等三分命中率超過40%的球員，全隊的三分命中率達到39%，排在聯盟第一。正是內外線都有穩定的得分球員，所以活塞的進攻效率在不斷提高，近幾場比賽，活塞的進攻效率排在聯盟前三。

活塞上一場111-122負於獨行俠，兩隊在第二節互相打出了一波14-0的進攻高潮，被獨行俠打出進攻波時，活塞進攻外線濫投，防守所有人夢遊。雖然整場比賽德拉蒙德和羅斯發揮出色，不過格里芬和肯納德的水土不服，加上全隊常出現斷電，沒能抵擋住東契奇和庫里的進攻，輸掉比賽。羅斯又找回了些當初MVP的影子，場均可以貢獻16.2分5.7助攻，他成為活塞外線不可或缺的一名老將。

火箭方面，威斯布魯克極有可能因為負荷管理原因選擇休戰，威少場均貢獻22.5分8籃板7.4助攻，投籃命中率42.2%，三分命中率23%，投籃效率慘不忍睹，但是他的衝擊力極強，突破上籃無人能擋，關鍵時刻能為球隊漲士氣，現在的威少在逐漸融合球隊，減少了不合理投籃，球隊也因為威少和哈登的改變提升了進攻效率。

威少如果缺戰，小里弗斯將會頂替他首發，小里弗斯因為流感缺席了火箭兩場比賽，今日與魔術的比賽復出，不過在16分鐘的時間裡3投0中，一分未得，剛復出的小里狀態欠佳。而最近進入輪換的克拉克繼續出色發揮，14分鐘貢獻9分3籃板，他正在慢慢擠掉老將塞弗洛沙的出戰時間。

本場比賽的一大看點是：卡佩拉VS德拉蒙德，兩人是本賽季籃板最出色的球員，場均總籃板、進攻籃板、防守籃板數據方面，德拉蒙德都是高居聯盟第一，在11場比賽中，至少得到15分15個籃板。而卡佩拉在場均總籃板和進攻籃板都排在聯盟第二，防守籃板聯盟第三，籃板能力與庄神有點差距，不過卡皇曾連續5場籃板至少20個，連續8場籃板至少19個，如今聯盟球員無法相比的數據。明日一戰，誰能在籃板球上發揮更好，也是球隊贏球的重要因素。

另一看點：哈登VS格里芬，兩位都是非常出色的得分手，然而最近的比賽兩人的狀態天差地別，哈登連續50+得分，三分球命中率奇高，格里芬的命中率則是超鬼了，連續兩場投籃命中率低至20%以下，三分球更是合計13中1。

冰火兩重天，格里芬能否迅速走出低迷，哈登能否延續得分盛宴，繼續創造NBA歷史紀錄，值得關注！
