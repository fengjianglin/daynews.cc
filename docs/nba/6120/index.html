2500萬場均3.6分，巴圖姆再刷下限，黃蜂的垃圾合同們都怎麼樣了

2019-12-06 0條評論 17次閱讀 0人點贊

垃圾合同年年有，黃蜂似乎每年都特別多。在2016那個垃圾合同集體井噴的夏天，黃蜂成為自由市場最大「贏家」之後，喬老闆的眼裡就常含淚水了。

三年之後，當那些垃圾合同都開始紛紛進入到逐年遞增的狀態時，他們的狀態也都在有規律地逐漸遞減，其中巴圖姆的合同，成為了歷史級別的垃圾。

今年，巴圖姆終於還是失去了他的首發地位，賽季至今出戰12場，但場均仍有12.8分鐘的上場時間。而今年，巴圖姆的工資是2557萬。

在還算可以的出戰時間內，巴圖姆場均3.6分4.3籃板3.2助攻，投籃命中率跌破4成，三分球命中率直接來到了27.6%。

拿著隊內最高的工資，得著隊內第十的分數，這就是巴圖姆今年的狀態。對於黃蜂來說更加可怕的是，他在明年夏天還有球員選項，價值2713萬。

對於巴圖姆來說，他的好日子還有一年，而對於比永博來說，今年就是他4年7200萬合同的最後一年。

說起來你可能不信，這位因為一個系列賽而拿下大合同的球員，今年正在打出生涯最佳的表現。場均7.6分創造了生涯最佳，永遠不要低估一顆在合同年刷合同的心。

拿著1300萬的邁吉基，也開始和巴圖姆肩並肩了。賽季至今只為黃蜂出戰了2場，場均拿下3.5分。遺憾的是，我們很可能再也看不到他的投籃姿勢了。

除了上面這三位以外，拿著1500萬場均7.9分的馬文-威廉姆斯似乎也有點垃圾合同那味，不過場均四成的三分球命中率還說的過去，現在的黃蜂畢竟是年輕人的天下了。

這就是黃蜂隊的現狀了，拿著最貴的工資，挨著最狠的打，不過他們賽季至今能排在東部第九還是超出了一些人的預料。要想變得更好，期待下明年夏天的解套吧。
