巨星為何“不好相處”？天才總有個性，科比、加內特概莫能外

天天要聞
2019-12-22

在NBA中每一位巨星都是天賦卓絕的天才，球技絕對是頂尖的存在。但再厲害的球星也不代表他們在生活中也是一個受歡迎的人，有不少的巨星都有自己的生活「怪癖」。

1、 用情不專：布雷克·格里芬大渣男。格里芬是一個脾氣比較暴躁的人，曾經在餐廳里毆打快船隊的設備經理。並且格里芬和克里斯·保羅相處也並不融洽，當然保羅也不是個好惹的人，兩人的矛盾日積月累只能分道揚鑣。還有就是他有一位前前女友，在有了孩子到了談婚論嫁的時候，格里芬卻在和另一位女星暗通款曲，最後甩了前前女友。不過格里芬離開快船之後，他就被女星甩了。

2、 自大：朗多這小子看起來就是一個非常硬朗狡詐的傢伙，其實他最大的問題是他的自大。曾經因為和主教練爭執戰術問題和教練叫囂，還曾經辱罵裁判，當然還有向克里斯·保羅吐口水等等。在朗多和湖人隊簽約的時候，他得意洋洋的表示，給你們五分鐘時間，如果不給我更多的薪水我就滾蛋了。

3、 嘴不閑著：加內特垃圾話無底線。垃圾話曾經是NBA賽場上一種另類的藝術，但在加內特這裡卻成為了一種變態的攻擊手段。他會在對位的時候提及對手的傷心事，比如當年鄧肯的母親剛剛去世，他就問候鄧肯的母親。還在對安東尼的比賽中告訴安東尼，你的老婆舔起來像甜甜圈。之後一位新秀表白加內特說你是我的偶像，但加內特的回應卻是Q**D！

4、 太嚴厲：科比對他人的極致嚴厲。科比無疑是一位偉大的球星，但在生活中他並不是太受歡迎，因為他總是對隊友有極為嚴厲的要求，只有實力強大，足夠努力的球員才能得到科比的認同。當年的死神帕克就被科比狂懟，在你籃球打的更好之前不準和我說話。科比自己也承認，自己好勝的競爭思維和想要讓隊友變更強的想法難以平衡。當然對科比影響最大的是鷹郡事件，雖然事後很多人認為這是一場仙人跳，但還是對科比造成了很大的負面影響。

（洛紙）
