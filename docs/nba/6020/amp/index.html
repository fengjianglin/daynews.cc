同樣是“涅槃重生”，魔獸能否像安東尼一樣，得到球隊的尊重？

天天要聞
2019-12-07

北京時間12月7日，波特蘭開拓者隊官方宣布，球隊已正式將卡梅隆-安東尼的合同轉為全額保障合同。相信很多球迷朋友們和小編一樣，在得知了這個消息後熱血沸騰，melo用自己的方式打破了媒體對他的質疑，同樣也贏得了開拓者管理層對他的信任。

這份保障合同雖然只有216萬，但卻也是瓜哥重回NBA應該得到尊重。與此同時，同樣在本賽季作為「涅磐重生」的球員代表德懷特-霍華德仍在隨時面臨著被裁掉的危險。

半個月前，很多媒體還在批評卡梅隆-安東尼無法放下身段，並表示他應該像霍華德那樣，用努力和誠懇去換取一個上場的機會。可如今，安東尼僅僅用了半個月的時間，用他最後的倔強去趕超魔獸，並搶先一步得到保障合同。

「魔獸」霍華德在考辛斯受傷後，火速馳援洛杉磯湖人隊，簽下無保障底薪合同，本賽季他場均可以得到7分7籃板1.5次蓋帽，防守效率隊內第三，僅次於詹姆斯和安東尼-戴維斯。就目前他的表現來講，雖然數據不像安東尼那樣搶眼，但對球隊的貢獻絲毫不遜瓜哥。

低頭有的時候不是因為自卑，而是要看清自己腳下的路。他就是霍華德，曾經披著超人戰袍的扣籃王，全聯盟唯一單換詹姆斯的男人。霍華德用最低微的態度，最誠懇的表現，贏得了所有球迷的尊敬。

安東尼的轉正，無形中給了湖人帶來了極大的輿論壓力，湖人隊能否將魔獸的合同改為全額保障呢？讓我們拭目以待。
