腸子悔青？老鷹這些年錯過的巨星可湊軍陣容 如今卻是東部墊底

2019-12-17 486條評論 1,874次閱讀 246人點贊

北京時間12月17日，據美媒報道，亞特蘭大老鷹如果在過去幾年裡能夠留住正確的球員，那麼他們或許已經擁有了一支全聯盟頂級的陣容。由於錯過了不少出色的球員，甚至是類似於東契奇這樣的巨星，老鷹或許現在腸子都悔青了。

對此，美媒做了一項統計，在近10年時間裡，老鷹分別錯過了艾爾-霍福德、丹尼斯-施羅德、盧克-東契奇、凱里-烏布雷這些不錯的球員。在2007年，老鷹曾選中了霍福德，當年身為探花秀，霍福德也為老鷹征戰過幾年，但是老鷹沒有等到霍福德真正迎來巔峰的時候，他們就把霍福德放走。

2013年，老鷹用首輪第17號簽選中施羅德。如今，施羅德在雷霆已經能是風生水起，但是當年由於沒有和老鷹管理層的關係相處得太好，施羅德並不像留在老鷹，也不願意參與到球隊的重建中，因此最終施羅德還是希望離開老鷹。　　

在2015年的時候，當時老鷹在首輪第15順位選中了烏佈雷，但是沒想到老鷹隨後就將其交易到了華盛頓奇才，奇才隊和小凱利·烏布雷正式簽約，合同為期4年價值923萬美元。後來，烏布雷去到了太陽，並且在這個賽季打出了高光表現，太陽隊也一躍成為了西部前八。看到曾經同為魚腩的球隊如今已經搖身一變成了西部前八，而老鷹卻還在東部墊底的位置上苦苦徘徊，真是讓人無限唏噓。

或許，這麼多年以來，讓老鷹管理層覺得最難過的就是去年夏天放走東契奇吧。去年夏天，老鷹曾在首輪第三順位選中了東契奇，但是最終他們用東契奇換來了特雷-楊以及一個受保護的首輪簽。當時這個交易評級中，獨行俠得到了A的評級，而老鷹的交易評級只有D。

雖然上個賽季，獨行俠同樣沒有排名靠前，但是這個賽季，在東契奇爆炸的表現以及身邊擁有了波爾津吉斯之後，球隊終於衝到了西部第三的位置。要知道，相比起特雷-楊而言，東契奇可以打後場兩個位置，而且還能打側翼，他十分全面。儘管特雷-楊本賽季仍然能夠交出一份不錯的成績單，但是球隊的戰績糟糕，也讓球迷們對球隊有些失望。回想起這些年老鷹所放走的這些球員，湊出來的首發陣容足以可以殺入到季後賽了啊！
