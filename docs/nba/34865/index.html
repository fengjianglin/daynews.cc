籃球晚報：猛龍大逆轉獨行俠 湖人遭遇三連敗

2019-12-24 527條評論 2,528次閱讀 526人點贊

【今日戰果】

洛瑞32+8+10 鉑金19+12 猛龍30分大逆轉獨行俠取5連勝

猛龍今日坐鎮主場迎戰獨行俠，客隊上場比賽大勝同樣是東部勁旅的76人，暫列西部第五位，而主隊目前正經歷一波四連勝，排名也上升到東部第四，此役東契奇、西亞卡姆和小加索爾缺戰。比賽上來猛龍率先進入狀態，一波9-0完美開局，分差也逐漸拉大到兩位數，次節獨行俠連續發起反撲，半場結束時反超猛龍9分。易邊再戰獨行俠又是一波13-0，分差一度達到三十分，末節猛龍在洛瑞的帶領下半節抹平23分分差，比賽懸念重生，關鍵時刻布歇切入扣籃得手，布倫森絕殺球不中。最終猛龍在主場30分大逆轉獨行俠，迎來五連勝。

巴頓14+13+5 米爾薩普21分 濃眉32+11 掘金送湖人3連敗

湖人今日坐鎮斯台普斯中心球場迎來掘金的挑戰，詹姆斯因為胸廓肌拉傷缺席了今日的比賽。首節比賽成為兩位大中鋒的對決，濃眉哥和約基奇各自拿下11分，兩人的精彩發揮也是兩支球隊的對攻縮影，雙方一直僵持到第二節，一直保持微弱領先的湖人在第二節節中遭遇短暫得分荒，掘金借著這期間一波反擊扭轉局面，半場結束時掘金手握2分領先。下半場的湖人搶開局並一度完成反超，但領先還沒有焐熱就被掘金連續打出反擊，後者抓住湖人啞火的機會送出12-0的攻勢一舉拉開兩位數的領先，末節掘金開局再送17-7的攻勢，分差來到23分也讓比賽失去懸念。最終，掘金128-104輕取湖人拿下比賽，並送湖人3連敗。
