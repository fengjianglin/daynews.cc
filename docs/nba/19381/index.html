23+8！200萬底薪男打出3000萬身價，開拓者這波操作真是賺翻了

2019-12-18 1,471條評論 2,400次閱讀 1244人點贊

北京時間12月17日訊。常規賽進行到54天，開拓者客場奔赴托金斯球館挑戰太陽。太陽上賽季的兩大主將德文布克和艾頓都在做「膝蓋保養」，雙雙缺陣。目前的球隊老大暫時是烏佈雷，雖然太陽先聲奪人，以一波13-3的攻擊波開局。但隨著利指導、麥科勒姆和200萬底薪男卡梅隆安東尼逐漸找回比賽節奏，最終以111比110領先一分，逆轉比賽。

數據方面：太陽隊烏布雷得分最高，37分鐘21投10中貢獻24分0籃板2助攻，沙里奇14分7板1助攻，卡明斯基17分5籃板5助攻，表現平平。開拓者這邊利指導狀態神勇，21投9中取27分1籃板5助攻。麥科勒姆40分鐘場均30分7籃板6助攻，打出全明星身價。老將底薪男安東尼成為開拓者第三巨頭，35分鐘貢獻23分8籃板1助攻。200萬的合同打出了3000萬的身價，開拓者這波操作真是賺翻了。

不得不說，開拓者提前給安東尼轉正是明智之舉，這次開拓者真沒有看走眼。底薪男安東尼依然具有全明星身價，特別是關鍵的第四節發揮堪稱經驗。在太陽小將烏布雷連續命中三分的情況下，安東尼立刻關鍵球給予回應，給太陽小哥上課，一波9-0的衝擊波穩住比分，最後利指導一劍封喉鎖定比賽，開拓者眾將打卡下班。

讓人感覺昔日的那個「甜瓜」似乎回來了，還是熟悉的配方，還是熟悉的味道。卡梅隆安東尼以底薪的方式重新在開拓者找到工作之後，場均16.8分+6.1個+1.3助攻，不得不說開拓者真是個聰明的生意人。同時也恭喜甜瓜，完成了自我救贖。
