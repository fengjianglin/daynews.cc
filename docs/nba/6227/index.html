關鍵時刻找誰不好，偏偏要打濃眉，問裁判要哨真搞笑，掘金該輸

2019-12-04 0條評論 13次閱讀 0人點贊

北京時間2019年12月4日，今天湖人在客場105-96擊敗掘金。

這場比賽之前湖人的戰績高居聯盟第一，很多球迷諷刺湖人打不了硬仗，因為11月份對於湖人而言相對比較甜蜜，背靠背少，強敵不多，大多數都是聯盟中下游的球隊，所以檢驗不出湖人的真正水平。進入12月份，湖人的魔鬼賽程這才開始，輸獨行俠，打掘金明天還要背靠背打爵士，對手都不是善茬，如果說打掘金是檢驗湖人的標準，那這場比賽算是個很好的開始。

本場比賽湖人最多手握兩位數的領先優勢，但在第四節被頑強的掘金追了上來，比賽最後的3分多鐘，約基奇造詹姆斯犯規兩罰全中，此時掘金僅落後湖人1分。或許是上賽季的西決給各個球隊留下錯覺，小個拉開空間打大個，軍訓成果穩拿比賽，所以關鍵時刻掘金的進攻戰術就是找湖人內線，也就是專門打濃眉這點。但是呢，軍訓的結果和馬龍教練預想的完全不一樣，約基奇打的不興奮，在內線頂不動濃眉，穆雷突進內線也被濃眉干擾，比賽的最後濃眉就是湖人的內線保障。想要在他頭上拿分，真的想多了。在濃眉身上沒討到好果子吃的穆雷和約基奇，他們開始向裁判抱怨，看著他們像裁判要哨的樣子屬實有點搞笑，不過話說回來，乾乾淨淨的防守裁判為什麼要給你響哨呢？

賽後據湖人主教練沃格爾透露：濃眉在中場的休息時間接受了靜脈注射，他想要上場堅持比賽。賽前有記者透露湖人更衣室內多名球員出現類似流感的癥狀。既然濃眉在防守端做出了大貢獻，鑒於身體情況，在進攻端我們就不能要求他做得更多了，生死時刻濃眉關鍵球未能打進，詹姆斯天神下凡完成補扣，這兩人現在就是湖人贏球保障。

看了下數據統計，本場比賽詹姆斯、濃眉各出手18次，二人的得分均為25分。而掘金呢最高得分是默里的22分，約基奇本場得到13分5籃板8助攻，很多球迷說掘金這場該輸，約基奇已經不再是之前靈活的大胖子，他的表現很掙扎，而且這賽季約基奇的球權也少了很多，以前掘金的每次進攻球都要讓約基奇處理。對此賽後約基奇也承認了這點，並表示自己會沒事的！
