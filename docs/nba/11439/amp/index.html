今日要聞匯總：樂福傾向效力開拓者，保羅等多人可被交易

天天要聞
2019-12-11

虎撲12月11日訊 今天NBA共有4場比賽，虎撲為大家總結了今日要聞。以後的每一天，我們都會在傍晚6點左右帶來今日NBA資訊要覽。

今天熱火經過加時以135-121擊敗老鷹，熱火多點開花，共有10項紀錄誕生。其中巴姆-阿德巴約和吉米-巴特勒均砍下三雙，肯德里克-納恩、鄧肯-羅賓遜和阿德巴約均拿下生涯最高分。

今天老鷹歷經加時以121-135惜敗熱火，老將文斯-卡特迎來生涯第1500場常規賽，他三分10中4，砍下12分3籃板4助攻。

此前騎士已經將凱文-樂福擺上貨架，據名記報道，樂福傾向於為他的家鄉球隊開拓者打球。

據名記報道，克里斯-保羅、達尼羅-加里納利、史蒂文-亞當斯和球隊里的其他老將都是可以被交易的，雷霆願意接手長合同和大合同以求獲得一些其他有價值的資產。開拓者預計會追求加里納利。

據報道，多支季後賽球隊在關注著森林狼前鋒羅伯特-科溫頓的可交易性。

據報道，奇才的戴維斯-貝爾坦斯、黃蜂的馬文-威廉姆斯、鵜鶘的JJ-雷迪克、灰熊的安德烈-伊格達拉和尼克斯的馬庫斯-莫里斯都是可以被交易的。但奇才有興趣長期留下貝爾坦斯。
