將你最喜愛的五個位置上“球星”組成一隊如何？我的可以拿冠軍

天天要聞
2019-12-20

NBA是陪伴我成長的聯賽，而NBA里的球星就是我熱愛籃球的信仰。我相信每位球迷朋友都擁有著自己喜愛的球星，但你們有沒有試想過把五個位置上你最喜歡的球星都組在一起？

雖然沒有歷史上最強五個位置的球星組在一起的隊伍強，但畢竟是五個位置上我最喜愛的球員呢，組成的隊伍也是非常強悍，拿總冠軍肯定是綽綽有餘。

在這裡我就跟大家分享一下，我最喜歡的五個位置上的球員他們是誰吧，這樣的組合在一起，化學反應會如何？

中鋒位置：姚明

雖然大家可能會認為姚明在NBA中鋒位置上並不是最強，但我說的就是我最喜愛的球星。奧尼爾和姚明之間，其實糾結了很久，因為發自內心覺得奧尼爾會強一些，但摸著自己的良心還是選擇了姚明。

雖然在統治力上姚明可能還有所欠缺，但身為傳統中鋒，他的能力可以說是非常全面了，砍鯊戰術可不會出現在姚明身上，而且姚明還有著身高以及射程的優勢，在籃下還有細膩的勾手，可以說放在陣容中鋒位置上，我心裡十分的安穩。

大前鋒：凱文樂福

我是從詹姆斯在騎士時期三巨頭的時候喜愛上樂福的，雖然樂福相比那些傳奇大前鋒還有現役的安東尼戴維斯及格里芬等球星的能力上會差一些，但他的精神還是值得我欽佩，雖然也有考慮過加內特，但畢竟和我不是一個年代的，我還很年輕。

當初騎士在季後賽，樂福可以說是兢兢業業，感覺臟活累活全乾了，本來辛辛苦苦的防了一球，到了進攻回合還要內線單打，經常還要打中鋒的位置，為了團隊，為了戰術，為了勝利，樂福值得存在於我心目中的大前鋒位置。

小前鋒：勒布朗詹姆斯

這是我最沒有猶豫的一個位置，哪怕是在鋒線有許多很強的球星，但詹姆斯的聯盟第一人沒得跑，正好我也身處這個時代，見證了詹姆斯長達十幾年的巔峰，身穿的23號球衣也正是我的幸運數字，不僅是在球場征服我，在場下也經常傳出滑稽的視頻，簡直實力圈粉。

得分後衛：特雷西·麥克格雷迪

一對眼神足以讓我把最喜愛的得分後衛名額留給他，先不說能力，光是慵懶及銷魂的眼神足以讓我忘記不了他，更何況他創造了歷史，也創造了奇蹟，35秒13分現在也依然是球迷朋友們津津樂道的話題。

麥迪用自己飄逸的籃球風格，令人看了忍不住驚嘆的干拔，在我心中埋下了深深的根，以至於心中的低位還超過了科比。

控球後衛：凱里·歐文

從魯德大叔的視頻出來，歐文就已經實力圈粉了，精湛的控球技術，在現役聯盟里沒有人可以與之抗衡，不少熱愛籃球的朋友運球的目標就是向歐文看齊，在球場上運球的感覺就像是從動漫里走出來一般，那樣的神奇，用一句話來形容就是人球合一。

很不湊巧，我最喜愛的五個位置上的球星組合出來竟然有一絲熟悉的感覺，這不就是姚麥組合+騎士三巨頭嗎？還別說，真就那麼的湊巧，就這麼樣一支大家見識過，也知道有多麼強大的組合就這麼出來了，每一個位置上都是強點。

這就是我喜愛的五個位置上的球星，自認為容錯率還行，拿總冠軍還是穩了！
