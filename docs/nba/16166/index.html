美媒評技術最好的十大巨星：利拉德第7，歐文第3，詹皇排名第幾呢

2019-12-16 2,089條評論 3,761次閱讀 631人點贊

第一：杜蘭特

近兩個賽季表現十分出色的字母哥沒能上榜，這也引起了許多球迷的質疑，不過字母哥的出色發揮主要也是得益於他那恐怖的身體條件，如果真要看個人技術的話，字母哥的確還是要比榜單上的這些球員差一些的。

杜蘭特毫無懸念的排在了第一位，他的進攻手段實在是層出不窮，在身體素質十分出色的同時，杜蘭特還擁有著如控衛般精妙的控球技術，而且他的進攻效率也是令人驚嘆的。不過如今因為傷病，杜蘭特仍舊無法回到比賽中，我們也希望他可以早日康復，帶給我們更多精彩的表演。而他的新隊友歐文是排在了第三位，歐文一直都以個人技術出色而見長，他的球風十分華麗，控球技巧更是極為嫻熟，歐文打球極具觀賞性，而且得分效率和進攻手段也讓人感到驚訝。

本賽季表現十分出色的約基奇排在第四位，他迄今為止的表現絕對是令整個聯盟都感到震驚的，他已經展現出了成熟的比賽方式和無與倫比的個人能力，他也是聯盟新時代中鋒的代表球員。此外，得分王哈登哈登排在第五的位置上，他的得分能力實在令人感到恐怖，而他本賽季仍舊延續著出色的表現，這也讓哈登很有可能會連續三年拿到得分王的榮耀，而出色的得分能力自然離不開他個人技術的支持，排在第五倒也算合乎情理。

詹姆斯本就是全能戰士的代表，他幾乎無所不能，一名小前鋒球員，卻擁有著完全不輸一流控衛的組織能力和視野，他的比賽方式也令人為之驚嘆，你似乎永遠都無法想到，還有什麼是詹姆斯無法做到的。最後，倫納德與喬治也都是個人技術十分出眾的球員，他們兩人尤其是在防守端有著十分出眾的表現。
