美媒評NBA歷史最沒爭議的5大MVP，詹姆斯上榜 1人創神跡

天天要聞
2019-12-19

MVP，這是除了總冠軍，每個賽季最被關注的一大獎項。MVP是常規賽最有價值球員獎，說白了就是最好的球隊里，表現最好的球員，才有可能最後獲獎。MVP含金量，真的比總冠軍高。有總冠軍不見得進名人堂，那有MVP就是必進，最起碼目前是這樣。

每一年MVP競爭都很激烈，有時候最後3位人選，得票率差距就在毫釐之間。但是也有毫無爭議，以壓倒性優勢勝出，加冕當個賽季MVP。近日美媒就評選出了，NBA歷史最沒爭議的5個MVP。小編看了下這5人，巔峰若是組一隊，總冠軍可以拿到手軟。

2016賽季庫里

庫里生涯最巔峰賽季，絕對是在2016賽季。這一年他表現真的是堪稱完美，拿下得分王和搶斷王，單賽季投進402記三分，還帶隊打出73勝9負歷史第一戰績。勇士在常規賽幾乎是無敵存在，一波流進攻打崩了對手。那一年勇士是最好球隊，而庫里是勇士表現最好球員，因此他征服了所有人，以100%得票率當選了MVP。100%得票率當選MVP，這在NBA73年是唯一一次，這一神跡根本無法被打破。

1996賽季喬丹

90年代看NBA，其實就是為了看喬丹。儘管當時有巴克利，「四大中鋒」，還有滑翔機，但是他們跟喬丹一對比，真的是失色不少。1996賽季，喬丹出戰82場，場均30.4分拿下得分王，入選1陣和1防，還帶隊打出了72勝10負前歷史最好戰績，他最後以98.6%得票率拿下MVP。當然季後賽喬丹也是大殺四方，奪冠拿下FMVP，他是那一年最大贏家。

2013賽季詹姆斯

看到詹姆斯不奇怪，他是現役「老流氓」，幾乎各種榜單都有他的名字。2013賽季詹姆斯還在熱火，這一年他帶隊打出了，歷史第2長的27連勝，自己也是場均26.8分8籃板7.3助攻，數據非常的全面。2012賽季詹姆斯已經是MVP，13賽季他延續了強勢表現。如果不是尼克斯的安東尼表現也好，最後投票分走了詹姆斯一票，那詹姆斯也是全票獲得MVP，最終詹姆斯99.8%得票率獲獎。

2004賽季加內特

全能戰士加內特，森林狼時期是他的事業巔峰。加內特2004賽季，場均24.2分13.9籃板5助攻1.5搶斷2.2蓋帽，他在場上是無所不在。森林狼也打出了58勝24負，西部第一的戰績。加內特加冕籃板王，入選1陣和1防，常規賽的加內特表現無可挑剔，他以99.1%得票率獲得MVP。

2000賽季奧尼爾

2000賽季奧尼爾場均29.7分13.6籃板3蓋帽，在內線根本沒人限制住他。當時科比還處於成長階段，湖人還是奧尼爾的球隊。常規賽面對各種包夾，他都能輕鬆破防，湖人也打出了67勝15負聯盟第一戰績。奧尼爾拿下得分王，入選1陣，以99.8%得票率獲得MVP。本來他可以是歷史第一個全票MVP，結果記者因奧尼爾罰球太糟為由，把一票給了艾弗森，這讓奧尼爾跟神跡擦肩而過！
