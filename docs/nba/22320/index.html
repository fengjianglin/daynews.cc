賽前預測：詹姆斯稱不想連續輸兩場，湖人擊敗雄鹿的概率有多大？

2019-12-20 1,099條評論 2,090次閱讀 864人點贊

在輸給步行者之後，湖人的客場連續紀錄被終結，7連勝也被終結，詹姆斯在接受採訪的時候說了上一次被獨行俠終結連勝的時候濃眉說過的話，那就是：不想連續輸兩場。

試問，誰想連續輸兩場啊？況且，你詹姆斯是西部第一，不想輸兩場，人家字母哥也是東部第一，當然也不想輸兩場啊，尤其是這場比賽還是東西部之間的最強碰撞，誰都不願意輸球好吧。每個人都不想輸球，這是真的，但這種話一般放在心裡就好。然而，作為一個超級球星，當家球星，既然都已經放話了，如果真的就輸球了，就是打臉，面子上過不去啊。

難道說？詹姆斯要盡全力贏下這場比賽？

在詹姆斯盡全力的情況下，湖人有多大的概率贏下這場比賽呢？這就要分情況了，要分濃眉到底是打還是不打。

如果濃眉要打，那麼湖人尚且可以和雄鹿五五開。

湖人和雄鹿都是攻防兩端都比較均衡，而且有防守尖兵，有進攻強點，當然，缺陷也有一些，就看在場上球員能否抓得住對方的弱點打了，有時候你明明知道他的弱點就是那個，但就是打不成功啊。比如，大洛佩慈的移動很慢，濃眉可以背打他，可以投他，把他調出三分線外之後，詹姆斯可以放心突破內線。再比如，字母哥雖然在開發三分球，但命中率還不高，不夠穩定，實在防不住的時候，可以採用聯防，讓他投三分，這是下下策。

之所以說濃眉在，就能五五開，主要是發揮他的優點，除了打洛佩慈之外，濃眉在第二陣容還可以打其他內線，小洛佩慈不太好打，但是利用移動速度，打攻防轉換，這也是湖人的優點。別忘了湖人現在的替補有卡魯索和布拉德利兩名防守悍將，如今湖人的替補陣容，進攻不太好，但是防守穩得一批。這些是湖人的優點。

當然，字母哥其實也不那麼好打，也不那麼好防，如果字母哥去防守濃眉，給他上身體，濃眉本身就有傷，也不太好在低位打。字母哥籃下的爆發力驚人，洛佩慈的卡位能力聯盟一流，所以還是要考驗湖人兩個中鋒的護框能力。

濃眉如果能夠打，至少在輪換階段給詹姆斯足夠的休息時間，在第二陣容至少會有一個進攻強點和戰術點，其實濃眉得多少分是次要，畢竟有傷在身，還是少做內線肉搏和低位單打戰術，防守端能夠協防到位干擾對方的擋拆跳投就行了，我們都知道，麥基和霍華德最大的弱點就在防擋拆了，中距離基本上就是放他們投了，米德爾頓和希爾可喜歡這種防守了。

如果濃眉不打，那麼湖人贏球就靠天意了，40%以下的概率吧。

內線護框就只能靠霍華德了，籃板球可能會被爆，雄鹿的籃板球本來就排名聯盟第一，如果沒有濃眉，那雙方的差距就更大了。另外，雄鹿的三分球出手次數排名聯盟第四，命中數排名聯盟第四，投籃命中率排名聯盟第二，回合數聯盟第一，快攻得分聯盟第一，內線得分聯盟第三，利用失誤得分聯盟第六。綜合起來就是，雄鹿內外都可以開花，濃眉若在，內線有守住的可能，不在，內線會很慘，外線的大小洛佩慈，伊利亞索瓦投三分的話也防不出去，若是在出現點失誤，可能就是一波流的事情，不會再像打老鷹那麼簡單了。

對於詹姆斯來講，面對防守效率聯盟第一的雄鹿，突破內線不會很輕鬆，就像上一場比賽那樣，詹姆斯多次扛著對方3個人的防守突破內線得分，兩三個回合之後就沒有力氣了，不是可持續的打法。必須得靠外線射手打出三分威脅，與詹姆斯之間建立聯繫，第二陣容沒有內線強點，必須投進外線三分，湖人才有那40%的機會贏球，否則就算詹姆斯能夠堅持得住第一陣容，等到他回來的時候，可能比賽已經花了。

不管濃眉打不打，湖人有一點必須要做好，那就是控制失誤！
