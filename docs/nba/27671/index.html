剛要被交易就想要退役？凱文樂福真挺牛的，還要帶上詹姆斯歐文！

2019-12-21 3,085條評論 4,769次閱讀 1375人點贊

對於騎士這樣一支擺爛球隊來說，選秀權和年輕球員才是球隊想要的，1.2億薪水的凱文樂福實在不是球隊想要的，所以他們近期一直在兜售！

對於很多球隊來說樂福都是一個不錯的補充，他能夠打四五號位，籃板球能力出色，遠投好，還有一手非常強的傳球策應能力。綠軍名宿麥克海爾就認為球隊應該追求他，增強球隊的投射能力，同時他和沃克的擋拆也將是非常好的殺招，唯一需要考慮的是如何配平薪水！

除此之外掘金和開拓者也對他有興趣！當時騎士給他這樣的合同就讓人看不懂，據美媒猜測他們在詹姆斯離開後認為球隊仍有競爭力，所以沒考慮到重建，希望樂福能幫助球隊打入季後賽，然後再通過交易補強重新崛起吧，但事與願違，上賽季他們只贏了19場球而已，本賽季到現在也只贏下了6場球！

近日他接受了隨隊記者Jason Lloyd的採訪，他談到了16年奪冠是他生涯最美妙的時刻，在此之前拿過了世界盃冠軍和奧運會冠軍，但在NBA拿冠軍是最難的，因為一年的征程有太多的不確定性，慶幸的是還是在1:3落後的情況下逆轉奪冠的，創造了奇蹟！

他表示「我和隊友們終結了克利夫蘭52年的冠軍荒，我們奪冠的故事也會永遠留在這裡。每當看著球場頂端的總冠軍旗幟以及那些分區冠軍旗幟，我就會想如果自己的球衣能掛在邊上那就太夢幻了。我認為詹姆斯、歐文、我以及湯普森的球衣應該被騎士退役。」

其實他說的不無道理，16年那支騎士創造了輝煌，帶來了克利夫蘭唯一的總冠軍足以載入史冊，當年表現出色的球員都應該被重視，這支球隊的歷史本就不輝煌，那是一支傳奇的騎士隊，而這四個人都發揮了至關重要的作用！
