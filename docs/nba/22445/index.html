庫班9千萬出售旗下公司，他會賣掉凈賺了20億的獨行俠嗎？

2019-12-18 559條評論 2,073次閱讀 96人點贊

​近日，億萬富翁NBA達拉斯獨行俠老闆馬克-庫班將他投資的體育科技公司出售給了MLB洛杉磯道奇隊的老闆。

美國投資公司富商、美國職棒大聯盟洛杉磯道奇隊股東托多-波利擁有的Atrium體育，收購了體育數據及視頻分析公司Synergy體育科技。馬克-庫班是這家數據公司的最大股東。

Synergy體育科技為很多NBA和MLB以及不少大學籃球和棒球隊提供數據分析。托多-波利的公司收購價格約為9000萬美元。托德·伯利上月還曾詢價收購切爾西,但被阿布一口回絕。

Atrium體育透露，被收購後將拓展新的業務，其中包括人工智慧領域在體育中的應用。今年早些時候，Atrium曾與國際籃聯FIBA展開合作。

Synergy體育科技應有錄像設備捕捉場上球員的移動。教練員應用數據去了解球員的表現、對手以及評估球員的潛能。數據提供給球隊、體育媒體、球迷等。Synergy過去一年中曾捕捉並分析了超過60000場籃球比賽和15000場棒球比賽。

​兩年前，在一檔播客節目中，庫班曾表示數據分析是近年最被高估的體育科技。同時，他認為最被低估的是人工智慧及其衍生物。

「因為現在人工智慧變得越來越高效了，這樣人工智慧就會狠狠地擊敗數據分析。另外，由於數據分析會成為人工智慧的一部分，所以人工智慧更是不會消失的。未來的數據分析部門要學習人工智慧、機器學習、深度學習、神經網路等等，同時還要精通這些能力。但從歷史上說，談到數據分析，還要說回我剛買小牛的時候，那時候我們剛開始使用先進的球員正負值，你不得不恰當地使用這些數據，並且必須能夠做到：『 OK，哪些數據對我很重要？我該如何權衡這些變數，我將從這些變數中得出什麼結論？』」

「至少剛把它應用到小牛的時候，我們還擁有巨大的優勢。通過數據的分析，我們會要求球隊去練一些發揮不佳的陣容，而球員並不知道這個陣容配合不好；也會要求球隊去練一些發揮不佳的戰術，但球員並不知道這個戰術打得不好，這幫助我們贏了很多比賽以及季後賽。隨著時間的推移，越來越多的球隊意識到數據分析的重要，球員市場也變得更有效率。然而，這些數據並沒有成為它應該有的樣子，而是變為了：『好吧，你能找到能打出你所需風格的球員嗎？』並且直到今天仍然是最大的挑戰。你可以擁有世界上最偉大的數據分析部門，但現在要想靠數據分析來獲得比其他團隊大的優勢，是非常困難的。也許會有幾個球隊在數據分析上相對落後，但你不會從任何一個聰明的球隊上拿到優勢。繼續向前發展，我認為你把什麼數據融入到你的模型中；如何清理你的數據；如何訓練你的數據，無論它是結構化的、非結構化的；如何標記它，所有這些都將是非常重要的。至少從我的角度來說，我希望我們能學到、看到未曾想像過的東西，畢竟我們不能在傳統的數據分析環境中感知到這些。」

​本賽季NBA開始前，庫班接受福克斯商業網路採訪時表示，自己是亞馬遜(Amazon)的大投資者，持有近10億美元的亞馬遜股票。

庫班於1983年創立計算機資訊公司MicroSolutions，並在網路經濟最繁榮的時候轉賣給了美國最大的在線服務公司。1995年，他和大學時代的好友兼商業夥伴托德·瓦格納想出了通過互聯網來轉播體育比賽的主意，於是成立了著名的BROADCAST網站。1999年7月，他將網站賣給了雅虎，自己凈賺20億美元。

​2000年，庫班以2.8億美元的價格購買下在NBA魚腩球隊達拉斯獨行俠，成為球隊歷史上第三任老闆。庫班手下的獨行俠成為進入21世紀後，NBA最成功的球隊之一，並在2011年獲得隊史上首次總冠軍。如今獨行俠的估值早就超過了20億美元。

投資獨行俠隊20年，庫班獲得了近10倍的收益。但庫班對籃球和這支球隊的狂熱人盡皆知，很難相信他會輕易賣掉這支苦心經營的球隊。
