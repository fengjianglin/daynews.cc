億元格林6投7+2被打爆，甜瓜17+8真香！開拓者球迷高舉三巨頭標語

天天要聞
2019-12-20

北京時間12月19日，開拓者主場迎戰勇士，結果沒有意外，他們送給對手5連敗。整場比賽，甜瓜17投7中，拿到了17分8籃板。對面的格林6投2中，只得7分5籃板2助攻。這一戰，格林被大白邊打爆了，後者14投8中，16分23個籃板。

甜瓜最近漸入佳境，過去兩場比賽，面對掘金，他16投8中，拿到了20分9籃板。跟太陽的對決，他16投8中，三分6中4，拿到了23分8籃板2助攻3蓋帽。比賽的第四節，甜瓜一人拿到了10分。

面對勇士，甜瓜再度成為球隊的第三火力點。比賽的第一節，甜瓜就保持有高效表現，他出手不多，3次出手命中2球，幫助球隊第一節領先了勇士11分，為比賽定下了基調。

第二節一上來，甜瓜送上暴扣。開拓者快攻反擊，甜瓜一馬當先，接到隊友的妙傳，高高躍起，輕鬆雙手空接暴扣，點燃了開拓者主場。如今的甜瓜已經成了開拓者球迷的寵兒。

隨後，甜瓜低位被動，虛晃之後，直接干拔跳投，美如畫，這是甜瓜的招牌絕技。隨後，甜瓜繼續爆發，錯位單吃斯坦，強打2加1。2分30秒左右，甜瓜連得了7分，繼續點燃主場。

結果上半場，甜瓜就拿到了12分。對面的億元先生格林只得2分，另外，開拓者雙槍也打開了，利拉德20分5板7助，麥科勒姆13分。開拓者三駕馬車火力全開。

下半場的比賽，甜瓜又送上2加1，單挑戲耍對手，連續交叉步後干拔跳投命中，開拓者球迷興奮舉著三巨頭的標語。不過，這一戰勇士也相當頑強，三節結束，他們追到只差5分。

第四節雙槍和大白邊接管了比賽。大白邊懷特塞德最後時刻如有神助，命中遠投，自己都不敢信，還剩39秒，他快攻暴扣得手，開拓者領先了9分，笑到最後。
