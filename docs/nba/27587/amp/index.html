熱火凱爾特人雙雙贏球，死亡五大的76人最終的排名會如何？

天天要聞
2019-12-22

北京時間2019年12月21日，NBA常規賽多隊展開廝殺，東部季後賽球隊凱爾特人、熱火、猛龍、76人均有參戰。熱火、凱爾特人、猛龍三支球隊都戰勝對手，唯有76人主場不敵獨行俠。不得不說，西部黑馬獨行俠成為檢驗隊伍成色的精準儀器，雄鹿、湖人紛紛落馬，76人這支個人認為擁有聯盟最強首發的死亡五大陣容依然不敵獨行俠，讓人不得不對76人感到擔憂！

昨天寫了小客（自稱）認為的西部季後賽隊伍及排名，今天我們來聊聊東部聯盟的季後賽球隊吧！其實賽季之初，相信很多朋友認為熱火在送走大白邊懷特賽德後，即使簽約了頂級鋒衛搖擺人吉米巴特勒也不會打出太過出色的成績，事實卻讓很多人大跌眼鏡，就像西部獨行俠異軍突起一樣，熱火在東部聯盟就是那個攪局者！今天看了熱火的比賽，讓我想起了04年打敗湖人四大天王的底特律活塞隊，全民皆兵，攻防兼備的體系加上冠軍教練斯波爾斯特拉，真的有可能會是東部最強攪局者甚至會給雄鹿等造成威脅也不好說。吉米這個賽季的轉變，儼然成為熱火年輕球員的主心骨，每個隊員都信心倍增，進攻不拖泥帶水，防守更是積極，著實讓人眼前一亮！

凱爾特人今天大勝活塞，整個比賽過程猶如砍瓜切菜，出色的整體配合，無論進攻還是防守的整體性十足，這是綠軍的傳統，出色的年輕鋒線，核心後衛肯巴沃克也是輕鬆了很多，不得不說凱爾特人與熱火很像，大牌放權，全民皆兵，讓對手防不勝防，且看看塔圖姆和布朗都可以搶到籃板後快速推進就知道凱爾特人球員的實力不俗，就連中鋒泰斯也在三分線外小試牛刀。神運算元史蒂文斯教練調教下的這支凱爾特人讓聯盟里沒有一支球隊敢輕視！

再看看另一支東部強隊76人在主場卻被失去了東契齊的獨行俠制服，最近獨行俠在失去東契齊後，那個兩米20的男人更是展現了出色的攻防能力，他彷彿在證明波神還是那個波神，傷病回歸的第一個賽季就充分發揮了自我的能量，成為西部最強黑馬。相反，76人本賽季迎來了霍福德和理查德森，紙面上的首發陣容實力冠絕聯盟，賽季初打得風生水起，一度被認為是東部霸主雄鹿的最強對手，隨後卻接連失利，城池不保，目前已經掉至東部第六，實在讓人不解！

你心目中東部聯盟的最終季後賽排名會是什麼，請留下高見，大家探討！
