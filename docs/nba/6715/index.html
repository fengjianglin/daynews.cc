這可能是西蒙斯生涯第一次投絕平三分；小加絕對是恩比德一生之敵

2019-11-26 0條評論 14次閱讀 0人點贊

每日一吐槽，您開心就好。

東契奇近期火熱的狀態，也讓好事者挖出了巴克利曾經對於東契奇的評價，巴克利說：「我不相信國外的對抗，我並不是在針對海外的球員，我只是單純的不了解他們的對抗是怎樣的。因為一名18歲的球員拿到了MVP。那隻能告訴我，他遭遇的對抗很爛。」

巴克利也沒說錯，18歲的小孩不可能統治NBA，但20歲或許可以(狗頭)。

東契奇：NBA比歐洲聯賽好打多了。

今天對陣爵士，字母哥出戰38分鐘，31投17中，砍下50分16籃板7助攻率隊取勝。


看完上一條吐槽，字母哥說：「我可以證明，世界盃比NBA難打多了。」

今天76人客場不敵猛龍的比賽，恩比德全場11中0沒有得分入賬，對位的小加得到3分。


小加：得分的事情我不太懂，你也別想懂。

據說唐斯賽後瘋狂點贊恩比德的數據新聞，並且@ 了恩比德11次。

76人對陣猛龍的比賽，最後時刻76人還有機會投中三分絕平球，哈里斯三分不中後，西蒙斯拿下籃板球，並且在LOGO附近來了一記轉身後仰三分，結果你也知道了。

這可能是西蒙斯生涯第一次也是最後一次投絕平三分了，事實證明只要活得夠久，你還真的什麼都能看到。
