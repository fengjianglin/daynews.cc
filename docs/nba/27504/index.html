端正態度！郭艾倫盲目進攻立即被換下 郭士強：遼籃要統一執行力

2019-12-23 232條評論 1,425次閱讀 309人點贊

​北京時間10月22日，CBA常規賽第20輪迎來一場焦點戰，福建主場對陣遼寧。遼寧下半場一舉擊潰福建，郭艾倫砍下25分，巴斯高效地拿到24分，賀天舉還上演了單節獨得16分的好戲。而儘管王哲林得到19分和11個籃板，但福建命中率只有37%。

全場比賽結束，遼寧以101-85大勝福建。遼寧結束了兩連敗，福建的四連勝則被終結。

面對福建，遼寧志在結束連敗，郭士強對球隊的每一次攻防都格外謹慎。第二節還剩2分16秒時，福建大舉反擊，郭艾倫此時出現了一次不理智的進攻選擇，郭士強馬上叫了暫停，並用趙繼偉換下了郭艾倫。

由於郭士強的謹小慎微，遼寧表現得沉著冷靜，他們全場在籃板和助攻上都完勝福建。

「我們之前兩場比賽有些東西沒有做好，這場比賽大家首先在統一執行力上做得非常好。尤其在進攻節奏和半場的組織配合上，要比前兩場進步很多。」郭士強說。

本場比賽，遼寧讓福建的命中率只有37%、三分命中率僅為28%，郭士強賽後表揚了隊員們的防守：「再一個，今天防守的針對性、防對方的擋拆做得非常好。」

​福建今晚開局不利，陳林堅只打了1分鐘便離場，讓球隊實力嚴重受損。

凱撒賽後表達了遺憾：「對於我們來說，遼寧是一支非常強的隊伍，他們有很多國家隊球員，包括重金簽下的史蒂芬森。我們非常困難，陳林堅打不了讓我們這場比賽非常非常困難。他們全隊在一起打了很多年，非常有默契。缺席陳林堅的情況下，我們很難和遼寧較量。」

儘管上半場一度追平，福建遺憾地下半場崩盤負於遼寧，他們的四連勝戛然而止。不過，凱撒認為連勝結束未嘗不是好事。

「結束連勝也不是壞事，更讓我們認清自己，為接下來的比賽做更多的準備。」凱撒說。
