確認出戰！詹皇連夜與球隊會和，輕傷不下火線，這35歲老漢真敬業

北京時間12月15日：昨天湖人在客場跟熱火比賽打完以後，詹姆斯沒有乘坐球隊飛機趕赴亞特蘭大備戰明日的比賽，反而是自己乘坐私人飛機去俄亥俄州觀看自己大兒子在高中的比賽，在今天早些時候，詹姆斯還出現在了球隊傷病名單上，明日出戰比賽成疑。

不過在觀看完了兒子比賽後，詹姆斯又連夜出發，飛往亞特蘭大與球隊會和，備戰明日對陣與老鷹隊的比賽。詹姆斯在被採訪時候問到肘部傷勢，他表示：「我很好，我的感覺總是很棒，我相信明天會出席比賽的。」

詹姆斯肘部的傷勢是在昨天比賽的最後時刻，與熱火球隊巴特勒相撞造成的，不過目前來看問題並不大。

老詹昨天比賽一結束，就趕去看自己大兒子在高中的決賽，他不希望自己能夠錯過兒子的所有比賽。今天詹姆斯的兒子在比賽中發揮也非常出色，在最後時刻完成了一個搶斷快攻上籃，打進了制勝球，最終拿到了MVP獎盃。

賽後詹姆斯與兒子的隊友以及教練紛紛進行了合影，作為一名父親老詹也是非常出色了，隊友們也都理解詹姆斯的決定，允許他缺席了今天的訓練賽。

湖人本賽季目前為止的賽程其實相對比較密集了，明日的比賽將會是他們東部五連客場之旅的第三場，來回的飛奔隊員們肯定都會有些精神疲憊，但是詹姆斯卻依舊充滿了活力。

35歲的老詹本賽季目前為止一場比賽也沒有缺席，他也不想缺席，輪休不是他的性格，職業生涯最大一次因傷缺陣還是上個賽季腹股溝扭傷，休息了一個多月。

明日的輕傷不下火線，讓越來越多的球迷佩服這位35歲的老將了，詹姆斯可以說是NBA最敬業的球員之一，17年職業生涯一直非常的自律，這對於任何人來說都是一件非常難以做到的事情。

他這種態度不僅僅值得球員們學習，對於各行各業的人來說，詹姆斯的敬業態度和自律精神都是值得學習的。
