韋德職業生涯最愛4個人卻沒詹皇：能依靠他們，而且他們不會離開

天天要聞
2019-12-21

北京時間12月20日，近期熱火傳奇名宿德維恩-韋德參加了一檔最新的電視節目，在這檔節目中，韋德談到了一個重點內容，就是他職業生涯里最愛的4個隊友，不過這4個隊友中卻沒有出現詹姆斯的名字，這引發了美國網友的熱議。

兩個曾經的NBA冠軍名宿巴恩斯和斯蒂芬-傑克遜有一檔播客節目，這檔節目曾經請來了不少NBA球星參加，例如當時路威就做客過他們的節目。而這一次，他們又請來了熱火的三冠王、也是隊史最偉大的傳奇名宿德維恩-韋德參加他們的節目。在節目當中，韋德公開提到了他職業生涯里最喜歡的隊友到底是誰。

這次韋德給出的答案讓很多人意想不到，因為居然沒有他最好的兄弟詹皇。韋德首先提到的名字毫無懸念是哈斯勒姆，不僅僅韋德是熱火的隊魂，哈斯勒姆也是熱火隊史最偉大的球員之一。當他退役的時候，他的球衣一定可以高掛在美航球館的上空：「我最喜歡UD（哈斯勒姆），毫無疑問的，他可是邁阿密先生。」

接下來還有什麼韋德喜歡的隊友呢？韋德還提到了錢寧-弗萊，他認為弗萊也是他最愛的隊友之一，另一個人則是詹姆斯-瓊斯。韋德曾經和瓊斯合作在熱火拿到了兩座總冠軍，不過當詹姆斯決定二重回騎士的時候，瓊斯也選擇緊隨領袖詹皇的步伐加入到了騎士，並且在騎士又伴隨著詹皇拿到了一座總冠軍。

瓊斯在騎士陪伴了詹姆斯三年，不過他在熱火人和韋德做隊友做了6年時間，也是歷史上最準的三分射手之一，職業生涯雖然場均只有5.2分1.8籃板，但在缺少他的情況下，詹姆斯從來沒有拿到過總冠軍。韋德最後提到的名字是埃迪-豪斯，埃迪-豪斯同樣是當年凱爾特人的總冠軍成員之一，他職業生涯的最後一個賽季和他職業生涯的前三個賽季都送給了邁阿密熱火。他也是韋德非常喜歡的一個好兄弟，同時也是韋德的替補，生涯最準的一個賽季，豪斯曾經有45.4%的三分命中率。

韋德提到的這另外四個人，都是拿到過總冠軍的球員，而且他們有一個特點，全部都是角色球員。韋德為什麼喜歡他們呢：「當你是這個聯盟里的超級巨星的時候，你身邊總是要依靠一下這種人，你需要這樣的好哥們兒。你可以跟他們談一切的事，而且不管發生什麼他們永遠都會在你身邊，他們不會離開。你可以向他們抱怨，他們會傾聽，並且給你一個可以依靠的地方。」

這就是韋德為什麼欣賞這麼幾個人。儘管他們是角色球員，但是韋德作為超級巨星，需要的也是這種能夠幫他排憂解難的角色球員。值得一提的是，這次的採訪過程中，韋德全程沒有提到自己的好兄弟詹皇，並沒有說他最喜歡的4個隊友里包括詹姆斯，這還是引起了一定關注的。
