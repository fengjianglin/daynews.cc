聯盟第一易主，湖人漁翁得利，字母哥進空砍群，挑戰詹皇三大幫手

天天要聞
2019-12-17

東契奇缺陣，但獨行俠卻掀翻了東部第一。12月17日，NBA常規賽，獨行俠在客場120比116擊敗雄鹿，終結對手的18連勝。如此一來，聯盟第一易主，湖人升至第一位。按照賽程安排，12月20日，湖人和雄鹿相遇，這場比賽前，湖人還要挑戰步行者。

本賽季，湖人和雄鹿都展現出很好的狀態，兩隊的成績都是24勝3負，雄鹿更是豪取18連勝。如果可以連續擊敗獨行俠和湖人，他們將刷新隊史的連勝紀錄。就當外界以為雄鹿和湖人要來一場決戰的時候，獨行俠先成了攔路虎。

東契奇因傷缺席，但是獨行俠在客場依然打出了氣勢，波爾津吉斯在內線發揮出色，小庫里在外線提供足夠強的火力，儘管字母哥的數據很豪華，但是獨行俠還是在第四節保持10分左右的領先。然而比賽最後時刻出現了波折，雄鹿的替補瘋狂追分，他們一度看到逆轉的希望，字母哥重新回到場上，但這也無法改變敗局，獨行俠120比116拿下了比賽，終結雄鹿的18連勝。

這場比賽，字母哥交出了48分14個籃板的超強數據，但他還是沒能帶隊延續連勝的步伐。需要指出的是，雄鹿的三分手感低迷，全隊41次三分出手，只有11次命中，老將科沃爾成為唯一穩定的三分點，他的三分球是7中5，其餘球員的三分都很低迷。字母哥6中1，米德爾頓5中1，大洛佩茲4中0，這樣的三分，真的很難贏球。這個問題真的像極了湖人，上一場湖人擊敗老鷹的比賽，除了詹姆斯命中4個三分球外，其餘隊友的三分球是21中1，真的是手感低迷。

輸球後的雄鹿丟掉了聯盟第一的寶座，湖人漁翁得利，升至聯盟第一。按照賽程安排，18日，湖人將對陣步行者，20日，湖人將挑戰雄鹿，東西部的第一對決，詹姆斯和字母哥的碰撞，到底誰能成為主宰者呢？拭目以待。

當然，這場比賽的一個關鍵還是兩隊的三分。湖人的客場之旅比較疲勞，而他們的三分近期的確是很低迷，需要全隊調整手感，尤其是射手格林，必須要找回自己的手感。至於雄鹿，三分同樣很重要，如果再像這場比賽一樣低迷，那真的很難與湖人抗衡了。字母哥會遇到濃眉、霍華德和麥基的輪番限制，屆時雄鹿要做出怎樣的應對，也值得關注。一場意外的輸球，讓雄鹿卸下了連勝的壓力，這對他們來說或許不是壞事，東西部的榜首大戰，這次或許會有更精彩的故事了。
