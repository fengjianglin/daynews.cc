詹眉領銜的湖人到底有多強，即將開啟的東征或許能夠給出答案

天天要聞
2019-12-12

北京時間12月11日，四連勝的湖人將於明天開啟五連客之旅，而且對陣的全部是東部實力強勁的球隊。詹眉領銜的湖人到底有多強，即將開啟的這一波東征或許能夠給出答案。

本賽季，聯盟多年的西強多弱的格局似乎有很大的變化，西部聯盟傳統強隊馬刺、勇士、爵士等實力大減，而太陽、森林狼、國王等黑馬雖然戰績大幅提高，但是競爭力也不大。而積弱多年的東部聯盟展現出強大的實力，雄鹿戰績冠絕聯盟，猛龍、76人、凱爾特人是聯盟絕大多數球隊不願意碰到的對手，其他球隊也是實力強勁，無論面對任何對手都要一爭高下。

湖人此次東征的首個對手是奧蘭多魔術隊。雖然魔術隊無論是球隊戰績還是隊內巨星的成色都無法與湖人相提並論，但他們目前仍然以11勝12負的戰績位列東部前八，同時球隊核心武切維奇的缺陣對他們的影響似乎並不大， 上一場客場與聯盟第一的雄鹿鏖戰到最後一刻便是證明，此番主場作戰，他們必定會給湖人造成很大的麻煩。

打完魔術後，湖人馬不停蹄地移師詹姆斯首冠的誕生地邁阿密，在那裡他們將遇到排名東部第三的熱火。巴特勒的加盟讓熱火再次擁有一位超級巨星，儘管無法與詹姆斯、韋德等人相提並論，但是他關鍵時間的穩定表現讓熱火更具有冠軍相。詹姆斯再次回到邁阿密，肯定是歡呼伴隨著噓聲，他表現的好壞直接決定湖人能否客場帶走一場勝利。

12月16日，湖人將迎來東正的第三個對手亞特蘭大老鷹。老鷹是聯盟中比較有歷史的球隊之一，但近幾年都是聯盟有名的魚腩球隊之一，賽季初更是被快船、火箭等強隊狂屠30分以上。然而，就是這樣一支整體實力不濟的球隊，上一場居然與邁阿密熱火打得難分難捨，所以湖人要想客場戰勝擁有小庫里特雷-楊的老鷹也不是板上釘釘的事。

東征的最後兩場分別是實力強大的步行者和雄鹿，無論能否取勝，都將是一場惡戰。雖然印第安納步行者的核心奧拉迪波至今都未出場，但是他們的戰績仍然高居東部第六，可見球隊整體實力之強，特別是內線上的特納和小薩博尼斯組合，能攻能守，必定會給湖人造成很大的麻煩。

而密爾沃基雄鹿實力冠絕聯盟， 球隊攻防兩端都是聯盟一流水準，且十分均衡，特別是去年的MVP字母哥表現更加完美，最大的短板三分球能力也有所提高。但是客場作戰的湖人也不是沒有機會，濃眉哥、麥基和霍華德組合的強大內線防守聯盟第一，若能在一定程度上限制住字母哥的發揮，讓其一星四射的戰術威力發揮不出來，客場取勝的機會將會大得多。

俗話所NBA無弱隊，特別是湖人此次東征的五個對手中有四個是季後賽球隊，紫金軍團雖然實力強大，但是要想全身而退也十分困難。詹眉領銜的湖人到底有多強，目前狀態正佳的紫金軍團能否延續自己的良好狀態和連勝，五連客的東徵結束後方可知道答案。對此，各位網友怎麼看？歡迎一起分享。（完）
