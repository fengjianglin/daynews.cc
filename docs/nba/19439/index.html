第8周最佳球員：熱火小將首次周最佳，詹皇湖人首次周最佳

2019-12-17 2,029條評論 3,426次閱讀 651人點贊

NBA官方宣布，上周東西區最佳球員分別由熱火潛力新星阿德巴約（Bam Adebayo）及湖人超級球星詹姆斯（LeBron James）分別獲選。

阿德巴約上周場均20分、11.3籃板、8.7助攻、1.3搶斷的全能成績，幫助熱火在上周取得2勝1負的成績。他在面對老鷹的比賽中更砍下30分、11籃板、11助攻的三雙成績，超越韋德（Dwyane Wade）成為熱火隊史最年輕拿下三雙的球員，隨後更在對上獨行俠的比賽再砍18分、11籃板、10助攻的三雙成績並幫助球隊拿下勝利。

至於詹姆斯則是在上周場均28.3分、11籃板、9.7助攻和1.3搶斷的數據，並率領湖人上周三戰全勝。在面對魔術的比賽砍下25分、11籃板、10助攻的三雙成績，而緊接著在面對熱火的比賽中也拿下了28分、12助攻、9籃板的准三雙，率領湖人終結熱火主場連勝。而本次獲選是詹姆斯生涯第62次獲選單周最佳球員，也是改披紫金戰袍後首次獲選周最佳球員。
