麥基練習“擦玻璃”，濃眉哥、庫克對飆三分，詹姆斯帶傷堅持訓練

天天要聞
2019-12-23

北京時間12月22日，洛杉磯湖人隊全員休整完畢之後，紛紛出現在了訓練館中。在之前的報道中顯示，湖人隊在進入12月份以來，基本上就沒有進行過一次全隊訓練，只有在與熱火隊的比賽之前，湖人隊球員進行了唯一的一次投籃訓練。如今，趁著比賽還沒有到來，湖人隊眾將士趕忙開始了各自的訓練科目。

濃眉哥在這次訓練中，主要還是進行了個人投籃訓練。上一場敗給雄鹿隊的比賽中，濃眉哥25投11中，雖然拿到了全隊最高的36分，但是濃眉哥這樣的表現依舊非常不滿意。除了個人投籃訓練之外，濃眉哥還和庫克進行了對飆三分的訓練。不可否認，字母哥上一場8投5中的三分手感，對濃眉哥的衝擊還是比較大的。

另外，隆多也現身訓練館，進行了一系列的對抗投籃和外線三分的訓練。在上一場比賽中，湖人隊替補席一共只得到了4分，這樣的情況令很多球迷感到不滿，大家紛紛將隆多視為了最大的罪人。但是，在湖人隊全場追分的情況下，隆多作為一名並不擅長得分的球員，真的是也挺無奈的。

不過，相比之下，麥基的訓練方式挺令人意外的，調侃地來說，就是在跪地練習擦玻璃！不過，對於麥基的訓練方式，有球迷也是給出了明確的回答：「這是對腹部力量的高度訓練！」對於麥基這樣的內線球員來說，他的訓練科目無疑就是力量了，畢竟，鎮守籃下這份差事，對球員的體能消耗還是非常大的。

除此之外，最令人驚訝的就是詹姆斯了，早些時候，湖人隊剛剛公布了球隊的傷情報告，而詹姆斯也是位於其中。不過，在全隊訓練結束之後，詹姆斯還是帶傷堅持來到了訓練館，和球隊的工作人員一起進行了一些背身投籃的訓練。這種精神，真的是挺令人感到敬佩的！

不過，筆者個人建議：無論如何，還是休息一兩場比賽吧，賽季還長，健康最重要！雖然球隊目前遭遇了2連敗，急切需要一場勝利來鼓舞士氣，但是，從長遠的角度來看，還是休息吧，大家覺得呢？
