歷史首次！兩隊同一天凈勝對手超45分，本賽季已有8次凈勝40+

2019-12-08 0條評論 18次閱讀 0人點贊

76人今天坐鎮主場迎戰騎士，最終他們以141-94大勝騎士，凈勝對手47分，這也是騎士本賽季第二次被對手凈勝40+。前兩節比賽，76人分別凈勝18分和23分。在上半場比賽結束時，76人已經領先騎士41分，比賽也失去了懸念。在今天這場比賽之後，76人的戰績變成16勝7負，騎士遭遇六連敗，戰績變成5勝17負，兩隊形成鮮明對比。

恩比德和理查德森今日缺席，西蒙斯今天表現出色，全場砍下34分7助攻，得分創個人生涯新高，還命中了生涯第二記三分。除了西蒙斯之外，斯科特和伯克分別得到21分，哈里斯15分，奧奎因10分11籃板。全場比賽，76人一共送出43次助攻。而騎士今天全隊的命中率只有37.5%，加蘭17分，湯普森12分，塞克斯頓和克拉克森各得到10分。

在本場比賽之前，獨行俠在主場以130-84大勝鵜鶘，凈勝46分。這是NBA歷史上首次出現，兩支球隊在同一天凈勝對手45+的情況。算上今天這兩場比賽在內，本賽季已經出現8次凈勝分超過40分的情況，不過本賽季還沒有出現分差達到50分的比賽。

​

其中，獨行俠獨佔3次，並且刷新了隊史紀錄，而雄鹿本賽季也有兩次凈勝對手40+的表現。老鷹和騎士本賽季則分別遭遇了兩次凈負40+，他們兩隊本賽季也分別排在東部倒數第二和第三。

本賽季分差超過40分的8場比賽分別為：

獨行俠142-94勇士，凈勝48分；

獨行俠143-101騎士，凈勝42分；

快船150-101老鷹，凈勝49分；

火箭158-111老鷹，凈勝47分；

雄鹿137-96黃蜂，凈勝41分；

雄鹿132-88尼克斯，凈勝44分；

76人141-94騎士，凈勝47分；

獨行俠130-84鵜鶘，凈勝46分
