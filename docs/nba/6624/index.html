濃眉首戰老東家 快船背靠背欲屠灰熊

2019-11-28 0條評論 14次閱讀 0人點贊

明天NBA將會進行14場比賽，我們精心為您對三場比賽進行情報分析，分別是籃網客場挑戰凱爾特人、快船背靠背對陣灰熊以及湖人面對鵜鶘的比賽。

籃網vs凱爾特人 08:00

傷病名單

籃網：杜蘭特（缺席）、歐文（缺席）、小喬丹（每日觀察）、勒韋爾（缺席）

凱爾特人：沃克（每日觀察）、海沃德（缺席）、泰斯（缺席）

最近五場得分/失分

籃網：102.8分/102.0分

凱爾特人：99.4分/98.0分

情報分析

不知道真的是因為肩膀傷病還是個人原因，歐文將不會身披籃網的球衣重回北岸花園。不過，對於籃網隊來說，這似乎並不算一個很糟糕的消息。雖然歐文的隱匿讓本場比賽流量大減，但是在過去六場比賽中，籃網隊取得了5勝1負的出色戰績。在大比分不敵步行者之後，球隊更是取得了四連勝的勢頭。

這段時間裡，丁威迪的表現非常出色，憑藉著出色的數據和球隊戰績表現，他甚至在上周生涯首度當選周最佳球員。過去五場比賽，這位核心後衛場均可以貢獻24.8分，幫助缺少歐文和勒韋爾的籃網在一定程度上維持了高效的進攻。

綠軍方面則有些不太順利，球隊核心沃克遭遇脖頸扭傷，狀態不定。球隊在進攻端表現得非常掙扎，過去五場百回合101.0分的得分和場均99.4分的得分均位列在聯盟墊底。唯一的好消息是，球隊的防守保持在了很高的水準。如果明日沃克能夠恢復狀態，塔圖姆也可以延續近來的出色狀態，主場作戰的綠軍還是更有機會帶走勝利。

快船vs灰熊 09:00

傷病名單

快船：沙梅特（缺席）、萊昂納德（出戰成疑）

灰熊：伊戈達拉（缺席）、莫蘭特（出戰成疑）、安德森（出戰成疑）

最近五場得分/失分

快船：113.4分/103.8分

灰熊：107.6分/117.2分

情報分析

對陣獨行俠，快船眾將用實際表現告訴我們什麼才叫令人窒息的防守。全場比賽，在貝弗利、小卡和喬治的連環鐵索下，東契奇14次出手僅有4次命中，三分線外8投全部打鐵，並出現了7次失誤。

最近五場比賽，快船隊100.2的防守效率位列聯盟第二，莫蘭特本將會成為東契奇之後又一位即將遭殃的新貴，可由於是背靠背之戰，小卡大概率將會輪休，再加上此前遭遇的背傷，莫蘭特很有可能與自己的「月考」擦肩而過。明日一戰，在雙方各有主將缺席的情況下，快船勝率更高。

湖人vs鵜鶘 10:30

傷病名單

湖人：布拉德利（缺席）、考辛斯（缺席）

鵜鶘：費沃斯（每日觀察）、威廉姆森（缺席）、米勒（缺席）、哈特（每日觀察）

最近五場得分/失分

湖人：117.4分/109.4分

鵜鶘：115.2分/117.4分

情報分析

湖人對鵜鶘絕對是明日14場比賽中最焦點的一場比賽，球隊核心安東尼•戴維斯將會在加盟紫金軍團後首次回到沙冰王中心對陣老東家。目前，本賽季狀態出色的湖人仍舊以8連勝的表現佔據聯盟頭名，再加上外線投射的回暖以及高強度防守的加持，球隊的實力遠勝於傷病滿營的鵜鶘。

對於主隊來說，他們最大的希望無疑是本賽季爆發的英格拉姆，他場均可以貢獻26.1分、7.2個籃板和4.2次助攻，關鍵時刻也屢屢上演大心臟表現。

有意思的是，在此前接受採訪時，濃眉曾表示自己與詹姆斯就再戰老東家一事進行過溝通，詹姆斯也提到了自己重回克利夫蘭時單場38分屠戮救主的無情表現。明日一戰，濃眉很有可能用統治級的發揮，來面對漫天的噓聲。
