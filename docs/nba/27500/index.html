喬丹之後，NBA真正做到享譽世界也就3人 庫里杜蘭特不夠格

2019-12-23 588條評論 2,946次閱讀 1805人點贊

喬丹是NBA的救世主，沒有他的出現，NBA真不會有現在的全球化盛世。喬丹是NBA第一個，真正享譽全球的超級巨星，在那個信息不夠發達的80—90年代，喬丹讓全世界球迷認識了他，也有太多人因為喬丹愛上了籃球。他能成為NBA歷史第一人，不僅是有2次三連冠這些神跡，更多還是在於影響力，以及對NBA推廣作用。

喬丹退役之後，NBA並沒有出現青黃不接的現象。不管是內線還是外線，都出現了很多歷史級別的巨星。但是真正算是享譽世界，即使不看籃球，不看NBA都能被廣大群眾認識，喬丹之後目前就3人，擁有這樣的巨大影響力。這3人都是歷史級別超巨，他們的名號是響噹噹，看這篇文章的朋友，至少有一大半，是這3人的粉絲，可見這3人影響力有多大。這3人究竟是誰，小編就不賣關子了，馬上揭曉答案。

奧尼爾

NBA超級中鋒有很多，但是奧尼爾只有一個。他是最獨一無二的中鋒，他是公認沒有模板的巨星。奧尼爾是籃下霸主，是他完美詮釋了，什麼叫做「得內線，得天下」的真理。他的霸氣打法，征服了太多人，中鋒擁有各種神力，能夠摧毀一切防守，都是得益於奧尼爾勁爆表現。中鋒在球隊地位這麼高，也是大家被三連冠時期奧尼爾征服，他就是近代籃球史上，最偉大中鋒沒有之一。他身上的娛樂元素也是獨一無二，即使你不看NBA，說不定哪天在頒獎典禮，或者在電影中，就能看到奧尼爾的身影，他絕對是NBA歷史上，最有影響力的中鋒！

科比

作為最接近喬丹的男人，科比的人氣又怎麼會低？科比高中時期就算是明星球員，進入NBA雖然是打替補開始，但是很快他的生涯就步入了正軌。從2000年之後，科比「小飛俠」名號被越來越多人認識，之後就是8號時期無解進攻，最後就是24號時期的「黑曼巴」。科比20年有太多球迷陪他成長，如果你20歲喜歡科比，看到他退役都40歲了，這樣的情結難以割捨。科比全球粉絲量，真的是億來計算，現在有太多巨星，那都是看科比打球長大。NBA第一個搞退役巡演，也就是科比開了這個先河，他有這個實力和影響力。科比的影響力很深遠，特別是「黑曼巴」精神，更是激勵了很多年輕人不斷前進！

詹姆斯

老詹是上榜唯一現役球員，其他的像杜蘭特，庫里，東契奇，倫納德等人，他們知名度確實也高，但是跟詹姆斯一比，差距就太大了。詹姆斯是征戰17年的老將，這17年他積累了太多粉絲。他的各種紀錄和逆天表現，不斷征服詹黑，他的影響力也是不斷擴大。跟詹姆斯當隊友，那都會沾光不少，這一點從JR.史密斯和湯普森身上就能看出來。不看NBA你肯定不知道東契奇，倫納德，杜蘭特，但是絕對知道詹姆斯，這就是影響力的體現。
