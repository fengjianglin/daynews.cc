拋開籃球場外亦精彩：躋身娛樂產業的詹姆斯影響力同樣巨大

2019-12-10 0條評論 18次閱讀 0人點贊

在上賽季詹姆斯宣布以自由球員身份加入湖人時，一度有傳聞認為詹姆斯此舉是為了他在娛樂產業的發展。然而實際上詹姆斯旗下的娛樂製作公司的發展已經無需他前往洛杉磯來坐鎮了。那麼詹姆斯在娛樂產業的影響力究竟有多大呢？

2008年，詹姆斯以斯普林希爾這個名字創建了一家新公司，並以該公司的名義出品了關於他自己高中歲月的紀錄片《不只是一場比賽》。或許對於當時的詹姆斯來說，斯普林希爾娛樂只是確保他擁有自己傳奇故事的改編創作權，他當時也並沒有意識到，這家公司能夠幫助他闖入好萊塢這個造夢的地方！

僅僅十年過後，斯普林希爾娛樂成為了有詹姆斯主演的《空中大灌籃2》的幕後製作公司。值得一提的是，這部電影將由曾經執導過《黑豹》的導演瑞恩-庫格勒擔任製片人。

科比在退役後曾憑藉其指導的動畫片《親愛的籃球》榮膺奧斯卡金像獎；斯蒂芬-庫里旗下的製作公司Unanimous傳媒曾於去年與索尼協議影業簽下了一筆合作協議，不過相關事宜還在早期籌劃中；杜蘭特也如詹姆斯一樣擁有自己的自媒體平台Thirty Five傳媒。不過他們都無法與詹姆斯的斯普林希爾相提並論。

總所周知，工作室、網路平台以及娛樂製作公司是詹姆斯希望在自己的商業版圖中著力構建的部分。詹姆斯也很注重自己在大眾媒體前的形象。關於這一點，從2015年他客串出演的電影《生活殘骸》便可得知。

不過詹姆斯的想法從不止如此，如今他與斯普林希爾的團隊在該領域屢創佳績，此外，很多好萊塢知名製作人都是詹姆斯的球迷。

自創立以來，斯普林希爾持續性地取得突破和成功。在6年的時間裡，他們從詹姆斯的高中紀錄片開始，一步一個腳印地向前穩步前進，而這期間，《星路多懊悔》也見證了這家公司地逐漸成熟。

如今的斯普林希爾正處在瘋狂的成長期，公司在華納影城附近的華納工作園區擁有一棟兩層建築的使用權。詹姆斯多年的好友以及商業合作夥伴馬弗里克-卡特也是斯普林希爾娛樂的聯合創始人。他通常會每周前往公司一到兩次來處理相關事務，而詹姆斯的次數則要少上許多，他絕大多數情況下都會在賽季結束後再趕來。

行動主義是詹姆斯身上不可磨滅的特質之一，去年，詹姆斯和杜蘭特曾在節目上公開抨擊美國總統特朗普，隨即福克斯新聞的主持人批評稱詹姆斯應該「閉上嘴好好打球」。而在該事件發生後不久，斯普林希爾方面就宣布製作一部分三個部分的紀錄片《閉上嘴好好打球》，並將在Showtime上播出。

在斯普林希爾成立的幾年時間裡，詹姆斯和卡特與娛樂圈的聯繫越來越多。兩人也與另外兩位密友米姆斯和里奇-保羅有著「四騎士」的綽號。他們四人與著名製作者、對沖基金經理人、波士頓紅襪及利物浦主席有著密切的合作。

而至於《空中大灌籃2》成功讓詹姆斯的斯普林希爾娛樂躋身好萊塢，據悉該影片計劃投資1億美金，將在2021年7月16日上映。

對於詹姆斯來說，他對娛樂產業充滿了激情，不過他也一定會一遍又一遍地強調他不會分心於籃球之外的事情。斯普林希爾娛樂的很多項目已經啟動了多年，他們如今已經意識到，自己無需詹姆斯過多的參與相關的活動。當然，這並不意味著他們不需要詹姆斯了。
