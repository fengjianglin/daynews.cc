16中9全場砍22+7，老鷹爛他卻重生，真別忘了曾經叫板維金斯之人

2019-11-26 0條評論 13次閱讀 0人點贊

北京時間11月26日，NBA常規賽繼續進行，老鷹主場迎戰森林狼，兩隊都是近年來累計了大量的天賦的青年軍，其中休賽期的老鷹隊更是在選秀大會上與用了一系列的操作拿下了許多天賦新人，其中不乏亨森雷迪仕這樣的天賦新人，其中開賽季的連勝由於特蕾楊的腳踝傷病中斷，目前已經遭遇了6連敗，本場比賽森林狼隊依靠著雙子星唐斯和維金斯的出色發揮，以125-113拿下比賽。

數據統計方面：

森林狼數據：卡爾-安東尼-唐斯28分13籃板8助攻；安德魯-威金斯25分6籃板；傑夫-蒂格8分8助攻；賈勒特-卡爾弗14分5籃板。

老鷹數據：特雷-楊37分6籃板9助攻；賈巴里-帕克22分6籃板；布魯諾-費爾南多13分4籃板；阿倫-克拉布7分7籃板。

雖然老鷹隊在本場比賽輸球後已經遭到了6連敗，但是值得一提的是，老鷹本場比賽的首發小前鋒賈巴里帕克發揮的很不錯，他全場比賽16投9中，拿到了22分6籃板，與對面的維金斯的25分6籃板不相上下，尤其是兩人多次直接對位，屬實讓人唏噓不已，曾經作為2014年的天之驕子，兩人分別作為狀元秀和榜眼被騎士（隨後被交易至森林狼）和雄鹿拿下，幾年來，兩人遭遇了截然不同的職業生涯。

維金斯雖然一直被人詬病比賽的態度，還有球迷為他親切地取了一個「養生」曼巴這樣地嘲諷外號，但是維金斯的天賦是有目共睹的，不用發全力都能在NBA的賽場上場均拿到20左右，休賽期更是簽下了億萬美金的大合同，但是帕克卻連續遭遇的傷病，早早被雄鹿隊放棄，近幾年淪為角色球員多次輾轉球隊，但是如今在老鷹隊，帕克浴火重生，在一個年輕的球隊中，他要承擔的遠不止恢復身體，他要重新證明自己是那個可以依靠未來的人。
