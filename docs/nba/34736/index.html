退役後名人堂+退役球衣？現役僅5人基本預定，只安東尼存在變數

2019-12-25 155條評論 4,026次閱讀 1636人點贊

籃球終究是一碗青春飯，到了一定年齡之後球員都要面臨退役，於是退役後的一些榮譽就成了即將退役的球員以及他們的球迷們最關心的事情了。毫無疑問，NBA球員退役後的最高榮譽無非是退役球衣和名人堂，可以這麼說，現役只有5個人能基本同時預定這兩件事，那接下來就和大家一起聊聊吧！

安東尼變數最大

作為03一屆的代表人物之一，安東尼生涯前7個半賽季都在掘金隊度過，那也是安東尼最好的時期。作為NBA十屆全明星，一次得分王，再加上他自身的影響力，無論是名人堂還是被掘金退役球衣都有希望，但又有些懸，畢竟他的職業生涯里沒有MVP級別的榮譽。所以說，安東尼恐怕是變數最大的一個，或者說是唯一存在變數的。

加索爾將以湖人球員身份退役

據悉，湖人本賽季的第一筆簽約以及基本定下來了，但並不是之前人們心心念念的伊戈達拉，而是科比兩連冠時期的二當家加索爾。在他被裁掉之後，湖人將會與他簽下一份1天的合同，隨後加索爾將會以湖人球員的身份退役。看得出來，湖人隊家嫂的貢獻還是心存感恩的，所以退役球衣也基本不存在問題，而加索爾生涯的成就再加上他在國際上的影響力，進入名人堂更不在話下。

水花兄弟沒有懸念

作為勇士隊老闆收購球隊後選擇的第一名球員，克萊-湯普森幾乎把自己生涯最好的時間都給了勇士隊，過去5年三冠他絕對功不可沒。今年夏天，湯神與勇士簽下了5年長約，而這基本意味著他的巔峰期都將在勇士度過了，所以退役球衣自然沒有懸念，而他近日名人堂則更加沒有懸念了。

作為小球時代的引領者，作為NBA史上唯一全票MVP，現役號召力僅次於詹姆斯的巨星，庫里的榮譽已經不需要再羅列了。雖然很多人都對他存在誤解，認為他只能投三分，但庫里的偉大並不會因此而遭受質疑。和湯神一樣，庫里基本不會離開勇士，退役球衣也基本是板上釘釘，名人堂更是沒有懸念。

詹姆斯或有三件球衣被退役

以詹姆斯的成就和影響力來說，進入名人堂根本就不在討論範圍內了，不會出現一丁點的變數。而他唯一的變數是到底能退役幾件球衣？目前來說，騎士隊的球衣已經是板上釘釘了；雖然他只為熱火效力了4個賽季，但卻為球隊帶來了兩座總冠軍，而熱火又比較喜歡退役球衣，所以詹姆斯在熱火的球衣也大概率會被退役。

如此一來，最大的懸念就是湖人隊的球衣了。作為NBA的傳統豪門，想要被湖人退役球衣並非易事，甚至即便本賽季率領湖人奪冠了也未必可以成為籌碼。
