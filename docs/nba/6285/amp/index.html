「專訪」鐵林：中國行加速球隊融合 濃眉DPOY LBJ虛心接受批評

天天要聞
2019-12-03

直播吧12月3日訊 湖人後衛丹尼-格林接受了《今日美國》的專訪，期間他談到了中國賽之行對球隊的幫助，以及勒布朗接受批評為球隊定下基調等話題。

Q：20場比賽，17勝3負，你認為球隊目前的狀態如何？你們的目標肯定不止於好的戰績，球隊處於實現大目標的有利位置嗎？

鐵林：我覺得是，顯然我們的表現已超出了預期，但也知道自己可以做得更多。我們還有很長的路要走，這就是令人興奮的地方。我們還沒有打出自己最好的表現，還有很大的成長和改善空間，我們這賽季並沒有整整48分鐘集中注意力。

Q：相較於在克利夫蘭，你注意到勒布朗的領導方式有何不同嗎？他比以往更多發聲了嗎？

鐵林：他總是直言不諱，他擁有冠軍DNA。多年來，他的成熟不僅體現在作為一名球員方面，為人方面也是。我們有著兄弟情誼，我們團隊很合拍，他知道我們在場上的重要性。在他職業生涯這個階段，最讓我驚訝的是，他是一個願意接受批評&學習的人，而且他真的會傾聽隊友的意見。你看不到那些超級明星，甚至是那些時代最偉大球員願意公開接受批評、傾聽隊友意見的。他不只是下達命令，而是接受命令，是的，他是一個出色的領袖，他不僅僅是關注勝利，他關心整個團隊和大局。

Q：有最近勒布朗接受批評的故事嗎？有沒有他為了改善球隊而接受批評的例子？

鐵林：有許多次，不只是一次，當詹姆斯在場上犯錯誤時，他都會主動包攬責任，在球隊視頻課上，他會說『我應該在這裡卡位的，那個球員被漏放了。』我們會告訴他『勒布朗，你要做這些那些，我們需要你更具侵略性，需要你多傳球。』他會以積極的態度聽取我們的建議，並且將它應用到比賽當中來幫助球隊。

Q：濃眉對球隊防守態度的影響有多大？

鐵林：毫無疑問，他已是年度最佳防守球員。他用行動、發聲、溝通領導球隊，他的標準對每個人都是一樣的，包括勒布朗，他就是你看到勒布朗在場上拼搶、做一直在做的事的原因。他一直在推動勒布朗。我們彼此聯繫，但這兩人的聯繫更為特別。AD一直在以身作則，用自己的語言和言行領導球隊，這就是我們防守如此堅固的原因，我們努力在48分鐘內表現得更好，但他就是我們的防守基調。

Q：雖然即將面臨西北三連客，但與快船的聖誕大戰也近了，你們有將這視作救贖之戰嗎（揭幕戰輸給快船）？

鐵林：我們期待每一次跟對手交手的機會，不僅是快船，我們還輸給了猛龍、獨行俠。我們跟獨行俠已經交手兩次，我們曾在一次膠著的比賽中擊敗過他們，這些球隊都將被記錄在我們的賽程表上。西部有眾多出色的的球隊，東部勝率達五成的球隊也多，我們將迎來一次艱難的客場之旅，接下來要打掘金、爵士和開拓者，這過程就會很有趣。當然，快船是一支極出色的球隊，他們打得非常好，這對我們來說是一場很棒的挑戰。

Q：與生涯的其他次經歷相比，這賽季的開始有何特別之處？

鐵林：我的意思是，我覺得我從來沒有過這麼好的開季紀錄。我曾效力於一些特殊的團隊，這比什麼都重要，比紀錄更重要的是我們之間的聯結，我們之間的化學反應。我曾是一些出色球隊的一員，我們在球場的化學反應是難以置信的。我認為中國之行對我們幫助很大，我們在場下的融合是美妙的。我們彼此交流，在群里中嘮嗑，在外面閑逛，我們一起規劃一些事情。我從未在一個聯結如此迅速的球隊，大家在這麼早就可以一起出去玩，是的，這就是特別之處，我們在場外玩得很開心，這點我永遠不會忘記，這是我們眾人的回憶，我們會緊緊守著這份回憶，因為我們是這樣的一群球員，我們在場下的聯結是那麼特別。

（Sera）
