9天客場之旅太累！詹皇只想回家放天假，一動作回應字母哥5記三分

2019-12-21 227條評論 1,386次閱讀 409人點贊

北京時間12月20日，湖人104-111客場不敵雄鹿，遭遇兩連敗，這是他們本賽季第一次遭遇連敗。詹姆斯得到21分12籃板11助攻，拿下賽季第7次三雙，總助攻數超越佩頓。賽後的詹姆斯顯得有些疲憊，只有提到回家的話題時，他的臉上才露出了孩子般的笑容。

過去9天，湖人隊一直在路上，他們連續打了5場客場比賽，取得了3勝2負的戰績。這是對體能和精神的一次挑戰，更何況在客場之旅中，將近一半的湖人球員身患感冒，其中包括詹姆斯自己。

輸給雄鹿後的詹姆斯，簡單分析了一下比賽，認為失敗的主要原因就在第二節。他覺得球隊在那一節犯了太多錯誤，導致最終客場失利。在那節比賽中，湖人得到29分，但讓雄鹿轟下42分。

雄鹿記者提問詹姆斯，如何看待字母哥投中5個三分球創生涯新高詹姆斯沒有說話，而是做了一個脫帽致敬的動作，然後問那個記者：「你知道這是什麼意思了嗎？那就是我想說的。」

接下來，湖人終於可以飛回洛杉磯，休息兩天，他們的下一場比賽是在北京時間下周一，對手是掘金。詹姆斯提到了自己的計劃。

他打算在明天好好休息一天，球隊沒有安排訓練，他也不打算碰籃球，讓自己徹底放空。「我想念妻子、孩子和家人了，」詹姆斯感慨道。畢竟，9天沒有回家，對於一個戀家的男人而言，確實是一種折磨。

戴維斯帶傷出戰，為了這場比賽也是太拼了。他在下半場打滿了24分鐘，沒有下場休息。雖然第一節打得不好，但後面幾節，戴維斯的貢獻很突出。本場比賽，他貢獻了36分10籃板5助攻3蓋帽。丹尼-格林投中7個三分球，貢獻21分，可惜湖人替補本場發揮糟糕，只有4分進賬。

輸掉這場比賽，湖人依然領跑西部，這不是什麼世界末日。畢竟，客觀來說，這是客場之旅的最後一戰，球隊身心俱疲，確實不在最佳狀態。賽季還得繼續下去，接下來，掘金、快船這樣的勁敵會接踵而至，湖人需要沉下心來，面對新的挑戰。

對陣勝率在五成以上的球隊，湖人6勝5負；對陣勝率不到五成的球隊，湖人18戰全勝。打弱隊，他們從未失手，現在，他們要努力提升對陣強隊的表現。
