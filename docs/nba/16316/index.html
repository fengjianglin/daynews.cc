“格林公式”，是導致勇士解體的原因，和歷史上最強的垃圾話嗎？

2019-12-13 976條評論 3,431次閱讀 1258人點贊

球場上很多東西可以直接影響球員的心態。

可以是釘板大帽，也能是顏射三分，可以是在你臉上投進絕殺球，也可以是運球終結你的腳踝。

但是垃圾話，是老帝覺得最能影響球員情緒的大殺器，運用恰當的話，更是可以直接擊潰對方的心理防線。

垃圾話是沒素質的表現，但同時它在球場上也是一種「 藝術 」。

比如出自伯德口中那句經典的：「 你們想好誰要做第二了嗎？」

以及奧尼爾那句：「 我奶奶跟著喬丹都能過奪冠 」，實屬殺人誅心。

4年3冠的勇士，毫無疑問是這個時代的王朝球隊。

倘若73勝那一年，勇士奪冠，那將成為歷史單賽季常規賽戰績最好的冠軍球隊。

而就是這樣一支王朝球隊，隨著格林對著杜蘭特說出那一句：「 在你來之前，我們拿到冠軍了！」後，導致勇士直接崩塌。

格林的這句話吧，其實在垃圾話里，只能算個弟中弟。

但是他有幾個時間段特別巧合，老帝來帶大家仔細品味品味：

1：噴的是隊友，還是內心最敏感的杜蘭特。

球場上的垃圾話，一般都是作為對手對噴，格林倒好，直接開口就對杜蘭特噴起來了。

杜蘭特是個什麼樣類型的球員呢，實力雖然強的沒話說，但是內心脆弱又敏感，特別喜歡在網路上和球迷對噴。

格林這一「 噴 」，說沒在KD內心留下陰影，我是不信的。

2：說的是大實話

杜蘭特吧，個人榮譽差不多都拿了，總決賽也進了，但就是沒拿到過冠軍。

而在上一年裡，再次帶隊打進西部決賽的杜蘭特，就是3-4惜敗給了勇士隊，迫切渴望冠軍的杜蘭特才「 跳槽 」來到勇士的。

杜蘭特在勇士打球，本身在球迷看來就是種「 抱大腿 」的行為，結果有隊內矛盾了，你什麼不好說，偏偏說個 「 在你來之前，我們就是冠軍了 」這種大實話。

在老帝聽來呢，這句話的意思就是：「 有沒有你，我們都能奪冠，你不重要 」。

這句話的殺傷性，在我看來完全是 「 生化武器 」。

假若杜蘭特內心強橫一點，和勇士表態他和格林不能共存，勇士大概率是留KD的。

可惜了，上賽季的總冠軍球隊，本賽季直接俯衝狀元簽。

而此時格林這句垃圾話又可以拿出來用了：「 在我們從冠軍變魚腩之前，公牛就已經干過了。 」（註：98-99賽季，喬丹第一次退役，公牛從冠軍球隊直接13勝37負東部墊底）。

兄弟們平常在球場上最喜歡說什麼垃圾話呢，歡迎評論區發帖留言。
