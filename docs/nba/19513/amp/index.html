快船隊真有看起來那麼強嗎？三大隱患非常致命

天天要聞
2019-12-16

今年的快船隊可謂是下血本兒了，連續的交易到了當今聯盟的兩大巨星卡哇伊和泡椒，目標就是這個賽季的冠軍。從賽季開始到現在的表現也是非常不錯的，以20勝8負的戰績位列西部第二，僅差第一湖人隊4個勝場。但這一切都是紙面的戰績，實際上快船還是存在很大的弱點的，要想奪冠還是需要補足自己的弱點的！

從三個方面來概括快船隊現在存在的隱患：

第一：快船隊的內線：大家都知道快船現在的首發中鋒是祖巴茨，然後就沒有一個傳統的高個子中鋒，而且祖巴茨的腳步移動很慢，如果被拉出去了就很容易失去防守位置。再就是靠替補的哈雷爾來打五號位。雖然哈雷爾的身體條件好，有著不錯的得分能力，但是哈雷爾的身高只有203cm，對於內線的防守和籃筐的保護是不夠的。在首發中鋒祖巴茨下場休息後，快船隊內線的防守就非常薄弱了。

第二就是路易斯.威廉姆斯了，他雖然有很強的得分能力，但是卻很占球權，進攻方式多為持球單打。而且現在的快船隊需要球權進攻的人太多了，保羅喬治、倫納德他們都是需要持球進攻的人，當他們同時在場的時候，球權的分配就有很大的問題。還有就是路威和倫納德的矛盾，以前路威就討厭倫納德這個人，近期的矛盾越來越嚴重了，估計路威是要被交易了，畢竟倫納德現在是不可能被交易走的。

第三：就是倫納德的輪休：從常規賽開始到現在，倫納德已經輪休了8場比賽，一共28場比賽輪休了8場。表面上快船隊的官方是說：「倫納德累了，需要休息」。但是沒有那麼簡單，倫納德的輪休和他腳踝、膝蓋的傷有著很大的關係，不能連續進行高強度的比賽。如果進入季後賽了，那時的比賽強度要比現在的常規賽大的多，防守也變得更加兇狠了。在他輪休的這8場比賽中，輸了4場比賽，所以傷病這個隱患也是非常致命的。

今年的洛城的兩支球隊都非常的有實力向總冠軍發起衝擊，小編希望能夠看到人員齊整的兩支隊伍正面交鋒，不要再像去年的總決賽那樣了，希望運動員們都遠離傷病。
