美記向衛冕冠軍獻計，搶下伊戈達拉可依靠鋒線群限制字母哥

2019-11-29 0條評論 14次閱讀 0人點贊

北京時間11月29日，美國媒體《露天看台體育》記者扎克·巴克利撰文提到，上賽季奪得總冠軍的猛龍，並沒有因為萊昂納德和丹尼·格林的出走淪落至季後賽末段，他們目前仍然取得13勝4負，東部第3的戰績，距離第1的雄鹿，也僅有1.5個勝場差。

猛龍當前的表現還是非常不錯的，本賽季他們的進攻效率值為111.4，排在聯盟第5位，防守效率值也有103，排在聯盟第4位，均能排入聯盟前五。就他們當前的狀態而言，的確還是可以在東部取得一番作為的，要知道，他們在周二的比賽中，還以101-96擊敗了76人。

扎克·巴克利認為，衛冕冠軍猛龍仍然可以考慮如何提升本賽季的競爭力，或許他們會在賽季中期，從灰熊那裡換取伊戈達拉。當大多數競爭球隊還在等待灰熊買斷伊格拉達這個不可能發生的情形時，猛龍完全可以佔據主動，去換取這位老將，讓這支打法成熟的球隊擁有更強的競爭力；在東部，字母哥這位強力鋒線球員就需要做出有效的限制，得到伊戈達拉，猛龍可以憑藉強大的鋒線群，輪番抵抗字母哥，這樣一來可以讓西亞卡姆在防守端解放，將能量更多釋放在進攻端。甚至，他們若是衝出了東部，對抗詹姆斯、萊昂納德、喬治這類鋒線球員也能有保障。

巴克利直言，若是猛龍得到伊戈達拉，完全有機會去看到他們的上限；為此，巴克利還列出了一筆交易方案，猛龍可以送出鮑威爾+麥考+2020年的首輪簽，換取伊戈達拉。
