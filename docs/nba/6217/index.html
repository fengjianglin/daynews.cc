霍華德亮劍！快船湖人同病相憐，榜首再生懸念，洛城球迷扎心了

2019-12-04 0條評論 18次閱讀 0人點贊

西部第一與西部第二的戰鬥已經打響，湖人做客丹佛高原。

掘金的魔鬼主場大家都有所耳聞，由於地處高原之上，往往使得一些客場來戰的球員產生高原反應，進而在比賽中難以發揮最佳狀態，導致輸球。而目前為止，掘金已經取得主場五連勝，對主場來說，可謂是信心倍增。

考驗湖人的時候到了，這場比賽不僅要克服高原反應，而且想要止住連敗，必須拿下這場比賽。但是，掘金也是要靠這場比賽止住連敗，所以，這必定是一場焦點之戰。

掘金與湖人的內外線實力其實是相差不大的。掘金內線擁有全明星中鋒約基奇，外加普拉姆利，外線投射穆雷和米爾薩普都是比較給力的。

湖人這邊的內線堪稱聯盟最豪華，三狀元內線濃眉、霍華德加考辛斯，外加腦袋以下全頂配的麥基，外線擁有冠軍射手格林和又帥又能打的庫茲瑪。

另外還有個能從一號位打到五號位、「十幾年沒一點進步」的詹姆斯。

比賽中，出現了很有意思的一幕，觀眾、解說都露出了驚訝的表情。 原來是霍華德在比賽中投出了一記三分球。

大家都沒想到，魔獸居然還有這一手，大家記憶里的魔獸都是在油漆區和別人肉搏。確實，像霍華德這種傳統型中鋒是不需要三分的。

但是你可能不知道，這個賽季加盟湖人以來，霍華德就已經多次在比賽結束後申請加練，練中投，練三分。在現在這小球時代，許多中鋒都有三分，像恩比德、唐斯、約基奇等三分命中率都還比較可觀。

魔獸這個賽季做出了許多改變，也贏得了球隊以及球迷的認可。雖然霍華德首次亮劍自己本賽季第一個三分沒中，但是我們相信以後肯定會進的。

本場比賽，詹姆斯濃眉合砍50分，強硬拿下勝利，鞏固自己的西部第一寶座。

詹皇像是在告訴掘金，妄想得到我的寶座，湖人西部第一是不可動搖的。比賽開局詹姆斯就強勢爆發，連續命中5球獨得11分。

這場勝利對於湖人來說至關重要，不僅沒有讓自己連敗，而且也勢在奪取聯盟第一的王座。

在湖人上一場主場爆冷輸給獨行俠，緊接著雄鹿大勝東部魚腩尼克斯後，雄鹿的戰力超越湖人，坐上了聯盟第一的寶座。

想必詹皇是不會這麼輕易讓出這個位置吧。湖人本場比賽勝利後，戰績來到18勝3負，和雄鹿戰績一樣。聯盟榜首花落誰家，懸念陡生。字母哥想必也在暗暗準備接下來了戰鬥了。

在榜首爭奪的這場戰鬥中，接下來的賽程對湖人是很不利的。後面的14場比賽中，有10場客場。大家都知道，在主場作戰的勝率一般都是高於去客場作戰的。

不過，湖人不幸中的萬幸是，同城的兄弟快船居然也陪著自己，快船接下來的12場比賽中有9場是客場，真的是同病相憐，難兄難弟呀！

就是可憐了洛城的球迷呀，這麼久見不到自個家的主隊，不知道會不會要哭呀。洛城球迷紛紛表示太扎心了！你能預測接下來誰會奪得聯盟第一寶座嗎？歡迎留言討論。
