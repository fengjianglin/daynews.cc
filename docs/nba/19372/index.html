湖人集體組團打雪仗，考辛斯與麥基互丟，老詹的表情亮了

2019-12-18 2,413條評論 4,227次閱讀 2225人點贊

贏下老鷹之後，湖人全隊在亞特蘭短暫休息了一天便搭上了前往印第安納的飛機，這是東部客場之旅的倒數第二站。由於印第安納的地理位置偏北，所以這裡的冬天要比其城市來的更早一些，而湖人隊員早已經有所準備，他們更換了更加的保暖的衣服，剛下機場時，場外已經堆銀砌玉，行人剛剛踩過的腳印更是清晰可見。

或許是洛杉磯常年不下雪的原因，看見如此的美景湖人眾將竟有些欣喜若狂，特別是快樂男孩麥基，和考辛斯玩起了打雪仗的遊戲。其餘隊友則紛紛拿起手機拍攝，欣賞這雪花飛舞的美景。另一邊庫茲馬在走路的過程當中，慘遭隊友突然襲擊，就這樣身高六英尺體重200磅的男人被一個雪球擊中之後竟順勢倒下，如此出色的演技讓人看的忍俊不禁。還在下機的詹姆斯內心正想：我怎麼會有如此逗比的隊友？

其樂融融的氛圍為球隊更衣室營造了良好的環境，眾志成城的湖人眾將正在為同一個目標而努力奮鬥。千萬不要小看了更衣室的氛圍，歷史上有很多偉大的球隊就是因為內訌瞬間走向失敗的。比如當時鬧到有你沒我的OK組合，沒有了奧尼爾的湖人，從總冠軍球隊立馬成為了西部倒數。比如去年的金州勇士，格林與杜蘭特比賽中的爭吵，也為阿杜後來的離開埋下了伏筆。另外還有今年的馬刺，德羅贊陰鬱的性格與關鍵時刻的軟，導致球隊戰績止步不前，內部矛盾已逐漸散開，聯盟最老牌的強隊終於走向沒落。

隆多，曾經鐵血綠軍的首發控衛，在聯盟也是出了名的刺頭，當年和雷阿倫有過私人恩怨，甚至鬧到了有你沒我的地步，最終雷阿倫加盟了熱火，在東部對抗老東家。如今雷阿倫已經退役，但隆多仍然與保羅有過矛盾，去年常規賽，兩人就上演了一場激烈的拳擊賽，最終遭到了聯盟的罰款與禁賽，今夏和湖人重新簽下一份合同，並握有否定交易權。

考辛斯，脾氣最暴躁的大男孩，在國王時期，考神就憑藉著內線的統治力，成為了當時聯盟第一中鋒。能力是有但脾氣也不小，經常在場上能和其他球員動起手來，與隆多一樣不是很待見保羅，因為CP3場上的一些小動作讓性格直爽的考神非常不滿。在過去除了能在得分、籃板的榜單上看到他的名字之外，技術犯規次數更是位列頭名。受傷之後，脾氣有所收斂，加盟勇士之後徹底放飛自我，考辛斯終於體會到了贏球的快樂，然而這次報團並沒有如願以償拿到總冠軍。今年加盟湖人，又因為傷病未能上場比賽，現在的考神只能安安心心坐在板凳席上耍毛巾了。

霍華德，曾經的更衣室毒瘤，仗著自己無與倫比的天賦，來到湖人隊的霍華德，以為他能夠代替科比成為球隊老大。然而這一切只是火花的個人想法，對於球迷來說，科比才是湖人永遠的主人，而且兩人由於在性格上很難相處，導致更衣室矛盾重重。那時候大家常說，愛玩愛笑的霍華德遇上了態度認真的科比，矛盾一觸即發，隨著傷病的侵襲，那時候被期待最高的湖人F4慘遭季後賽一輪游。

原以為當一支球隊同時擁有隆多、考辛斯、霍華德這些更衣室炸彈時，湖人的氛圍會非常糟糕，然而事實卻截然相反。這些所謂的刺頭成為了球隊的開心果，他們放下了傲氣，將快樂傳遞給每個人。其實這就是詹姆斯的偉大之處，從過去的熱火到騎士再到如今的湖人，每個詹姆斯的隊友都對他讚賞有加，期待新賽季的湖人能夠一飛衝天！
