防守效率堪比喬丹！韋德接班人成外線大閘，又一後衛DPOY誕生？

天天要聞
2019-12-22

自最佳防守球員獎設立以來，總共有22名球員榮膺該殊榮，其中有15位是中鋒或者大前鋒。內線球員作為護框者，在DPOY的競逐中有著天然優勢，後衛球員只有喬丹、佩頓、庫伯等5位球員拿過。強如科比、韋德，在他們的榮譽薄也沒有這一項。現役NBA，哪位有希望成為第6個後衛DPOY獲得者？對很多人來說，這或許是一個陌生的名字，他就是公牛隊的克里斯·鄧恩。

今天進行的比賽中，公牛在客場以119比107擊敗活塞。賽後，公牛主教練博伊倫談到了鄧恩的防守。「鄧恩在場上很善於溝通，他是我的年度最佳防守球員候選人。」博伊倫說道。這場比賽鄧恩在進攻端發揮一般，得到9分3籃板4助攻，但在防守端送出4次搶斷。截止目前，在場均22.8分鐘出場時間裡貢獻1.9次搶斷，是聯盟中搶斷率最高的球員。

鄧恩在公牛都算不上核心球員，在整個聯盟中更是小人物一個，但在大學時期，他是全美矚目的希望之星，曾入選NCAA最佳一陣。之所以稱鄧恩為韋德接班人，除了因為他本人視韋德為偶像外，還有他們都是讀完大三才參加的選秀，打得是同一個位置，都是首輪5號秀，身高都是1.93米，都是精英級的外線防守者。另外，鄧恩剛入聯盟時也曾身披也是3號戰袍。

論進攻天賦，鄧恩遠不及韋德，他缺乏身體爆發力，技術也不是很細膩，能在2016年選秀大會上以第5順位被森林狼選中，是因為當時的森林狼主教練錫伯杜極為重視防守。可惜的是，錫伯杜始終沒有在森林狼建立起自己理想中的防守體系，鄧恩的價值也沒有發揮出來。被交易到公牛後，經過兩個賽季的歷練，鄧恩終於在本賽季展現出了防守大閘的風采。

公牛不是一支擅長防守的球隊，他們的防守效率（百回合凈失分）為111.4，全聯盟排在第22位。但這沒有埋沒鄧恩的作用，他個人防守效率為101，也就是說，當他在場時，森林狼每百回合少失10.4分，這是一個令人咋舌的數據。作為一名後衛，防守效率達到這個級別是很難得的，強如喬丹，單賽季最好的防守效率值也只有102。

即使如此，鄧恩獲得本賽季最佳防守球員的可能性很小，就連入選最佳防守一陣也難度很大。因為他不是那種防守中軸，個人防守沒有帶動起球隊防守體系。過去10年的最佳防守球員，如戈貝爾、小加索爾、格林、萊昂納德，他們所在的球隊，都因他們的存在，防守效率排到了聯盟前列。
