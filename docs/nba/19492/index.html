15投僅得10分！5屆全明星淪落為雞肋，4年1.09億成毒藥合同？

2019-12-17 986條評論 3,223次閱讀 1226人點贊

被奧尼爾、巴克利批評之後，恩比德知恥而後勇，連續砍下高分，率領球隊取得一波5連勝。今天客場挑戰籃網，即使恩比德輪休缺陣，76人依舊是天賦更好的一方，可最終的結果卻是爆冷輸球。面對沒有歐文的籃網，76人打得極為掙扎，全場僅得89分，整整輸了20分。西蒙斯、哈里斯分別得到20分、17分，表現還算中規中矩，隊內另一位「巨頭」霍福德則堪稱災難。

霍福德出戰27分鐘，15次出手僅命中5球，只有10分入賬，正負值為-13。在恩比德缺陣的情況下，這位5屆全明星球員沒有承擔起內線攻堅重任，被身高臂長的賈萊特·阿倫防得失去了比賽節奏。賽後接受採訪時，霍福德反省道：「從身體方面來說我倒是沒什麼問題，但是有一些跟不上節奏或者不太適應的情況。我對於自己在進攻端的很多處理都感覺不錯，只是投籃沒能命中。」

單就本場比賽來說，霍福德的表現完全配不上球星身份，甚至有雞肋之嫌。事實上，本賽季來到76人之後，他一直沒有找到最佳狀態。光看基礎數據，場均13.6分6.9籃板3.9助攻，似乎不必上賽季（13.6分6.7籃板4.2助攻）差，但高階數據揭露了真相，真實命中率從60.5%降到54.4%，效率值從20.2降到18.1。6年前，霍福德還是場均可以貢獻18.6分8.4籃板的內線小霸主，如今33歲的他，競技狀態似乎已經開始下滑。

今年夏天，霍福德以自由球員的身份簽約76人，合同數額為4年1.09億。在擁有恩比德、西蒙斯兩位超星、確定續約哈里斯的情況下，76人沒有引進3D球員，而是選擇把錢花在霍福德身上，看中的是他的球星屬性。霍福德表現不如預期，不知他們的管理層會不會心生後悔？未來3年，霍福德的年薪都在2500萬美元以上，36歲那個賽季還要掙到2650萬美元。人們說保羅的4年1.6億是「毒藥合同」，霍福德這個毒性也夠大。

76人給托比亞斯·哈里斯的，也是一份毒藥合同。5年1.8億美元，對一個沒有過全明星經歷的球員來說，無疑是溢價的。看得出，76人就想豪賭一把，目標就是儘快獲得總冠軍。可問題是，即便勇士王朝已經不復存在，湖人、快船、雄鹿都比他們更接近總冠軍。一旦賭輸，完全沒有迴旋餘地。

對76人來說，既然選擇把寶押在霍福德和哈里斯身上，就算一時看不到希望，也要一條道走到黑。現在看起來，76人的實力是不如湖人、雄鹿強勁，但隨著賽季深入，或者到了季後賽，他們的「死亡五大」豪陣或許會發揮奇效。在終極答案揭曉前，我們還不能否定霍福德，更不能直接看扁76人。
