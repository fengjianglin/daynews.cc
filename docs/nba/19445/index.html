黃蜂新核心狀態極其低迷，12投僅1中真能率隊殺進季後賽？

2019-12-17 2,264條評論 3,869次閱讀 1103人點贊

北京時間2019年12月16日，坐鎮主場的印第安納步行者隊迎來了黃蜂隊的挑戰，這一場比賽雙方的實力差距其實並不是特別大，但是最後卻打出了107:85的比分，這確實是出乎了球迷們的意料！那麼黃蜂隊這一場比賽到底是為什麼遭遇到了這樣一場慘敗呢？

其實很大程度上是因為球隊的核心球員發揮的卻是不是很好，全場比賽得分最高的是科迪·澤勒，他拿下了19分12籃板的數據！而除了澤勒之外，黃蜂隊最倚重的兩位後場核心羅切爾和格雷厄姆的發揮卻極為糟糕！

格雷厄姆全場比賽登場37分鐘，投籃18投僅3中，三分球14投3中，僅僅得到了12分6籃板4助攻的數據。而羅切爾的發揮就更加糟糕了，他登場25分鐘，投籃12投1中，三分球3投0中，僅僅得到了3分2籃板3助攻的數據！雙核如此糟糕的發揮也就能夠證明為什麼黃蜂隊遭遇到了一場慘敗了！

本場比賽確實證明了羅切爾和格雷厄姆的不穩定，尤其是羅切爾，作為黃蜂隊休賽期最為重要的一筆簽約，他被球迷們寄予厚望。但是這一場比賽如此糟糕的發揮確實令人大跌眼鏡，雖然一場比賽不能證明什麼，但是上個賽季在凱爾特人隊中他的狀態就是起起伏伏，所以可能黃蜂隊的管理層已經對放棄肯巴·沃克，簽下羅切爾這個決定有一些悔意了吧！

黃蜂隊本賽季雖然沒有太大的目標，能夠進入季後賽就已經是非常好的成績了，但是從目前的狀態來看，想要進入季後賽確實還是有一定的難度的，尤其是球隊的核心發揮還是非常不穩定的情況下！目前他們距離東部第八有1.5個勝場的差距，希望羅切爾能夠完成自己作為球隊核心應該去做的事情，這樣才能讓黃蜂隊在接下來的比賽中有更好的狀態！
