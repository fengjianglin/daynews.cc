名宿炮轟庫里不是真正控球後衛！怪進攻太強，場均6.5助香不香？

2019-12-16 207條評論 1,202次閱讀 234人點贊

儘管斯蒂芬-庫里目前因傷缺陣，但這位金州勇士的超級巨星仍然是人們關注的焦點，但庫里是否是聯盟中最好的後衛已經不具有任何爭議，他的履歷說明了一切問題，兩屆常規賽MVP、六次入選年度最佳陣容、三屆總冠軍得主，聯盟中沒有其他後衛能和庫里競爭。

自2013年打出突破性表現以來，庫里一直是聯盟中最好的控球後衛之一，然而他因為自己的打法而導致自己在場的位置備受質疑，很多人指責他並非真正的控球後衛，而是一名得分後衛佔據了控球後衛的位置，而NBA歷史上最偉大的控球後衛加里-佩頓也持有同樣的觀點。

最近佩頓在一檔節目中公開表示，庫里不是一名真正的控球後衛，他認為聯盟中僅有的真正控球後衛是保羅和隆多。佩頓的論點非常傳統，他認為控球後衛不應該是以得分為主，更確切地說，控球後衛的工作是帶動隊友的進攻和打出很強的防守表現。

佩頓所處的時代確實如此，綽號「手套」的他不僅是他那一代最好的球員之一，而且可以說是有史以來最好的防守球員，他是聯盟中唯一一位當選過最佳防守球員的控球後衛。不過小編認為，這些並不代表佩頓比庫里更像一名控球後衛，庫里可能每場比賽都能得到30分，但在過去的六個賽季里，他的場均助攻次數高達6.5次。

庫里之所以很容易受到人們質疑，只是因為他的存在徹底改變了控球後衛的定義，他不再像隆多那樣僅僅是依靠傳球帶動球隊進攻，而是成為了得分和傳球的雙重威脅，雖然小編也很難反駁佩頓對一名真正控球後衛的定義，但庫里和其他人一樣，雖然他超強的進攻能力掩蓋了他的組織能力，但是除開他的得分不談，他的傳球依然可以說是一名優秀的控球後衛，之所以受到佩頓的指責，只能怪庫里的得分能力太強。
