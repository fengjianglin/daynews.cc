沃頓指教能力被低估 真核回歸國王或將迎反彈

2019-12-21 218條評論 1,264次閱讀 214人點贊

薩克拉門托國王vs印第安納步行者(主)

薩克拉門托國王上場比賽客場102-110輸給黃蜂，福克斯在缺席了17場比賽後終於復出。他替補出場高效的拿到了19分8助攻的數據，這樣國王此前備受困擾的攻擊力得到一定緩解。此外馬文-巴格利最近復出後狀態也不錯，場均10.5分6.5籃板，他同樣為第二陣容帶來了活力。目前這支國王陣容趨於完整，他們目前的戰力不容忽視。特別是沃頓，還是一個非常善於調教防守的教練，這正是國王上賽季所欠缺的東西。進攻端，這支球隊其實不缺乏天賦，他們在未來的一段時間將會有一個戰績的爆發期。國王甚至有機會在這賽季的西部拿到季後賽的有力位置。

印第安納步行者上場比賽是105-102擊敗了洛杉磯湖人，迎來了4連勝，這會在市場上引起很大的反響，加上步行者目前19勝9負的戰績也會為他們加分不少。但值得注意的是，安東尼-戴維斯的缺席對湖人整體實力影響巨大，因此步行者這場勝利的含金量其實是有所不足的。

另外，步行者這支球隊總體上還是頭重腳輕的配置，內線雲集了太多的輪換球員，薩博尼斯、特納、利夫和比塔澤。但外線球員的整體戰力以及輪換的質量還是有些欠缺。布羅格登這賽季的產出還不錯，場均18.9分7.6助攻，但是效率和在雄鹿時期相比還是有差距的。他的真實命中率跌到了58%，而且防守端是需要內線球員補漏的。此外，上場受傷的傑里米-蘭姆將大概率缺席比賽。本賽季蘭姆場均能貢獻14分，他的產量很難有人來彌補。TJ-沃倫的傷勢也不太理想，本賽季他場均能貢獻17.6分，這兩人如果同時缺席，步行者進攻端將遇到大麻煩。

本場競彩步行者讓5.5分，指數給的太淺了，推薦國王受讓5.5分。
