湖人隊再迎利好！沃帥為詹姆斯減壓，佩林卡兩大轉變助力紫金軍

天天要聞
2019-12-09

湖人隊目前仍然高居西部榜首，而且和身後球隊也拉開了差距，現在的湖人隊兵強馬壯，球員的身體狀況和隊內氛圍也非常棒。湖人隊再迎利好！沃帥為詹姆斯減壓，佩林卡兩大轉變助力紫金軍。

詹姆斯出場時間創新低

現在湖人不光是一直贏球，關鍵在於主教練沃格爾還嚴格控制主力球員的出場時間。根據數據統計顯示，本賽季勒布朗-詹姆斯的場均出場時間「只有」34.7分鐘，創職業生涯新低，要知道老詹生涯場均出場時間高達38.5分鐘，生涯初期時不時能上40分鐘。沃帥深知常規賽沒有必要讓球員太拚命，他知道老詹壓力很大，於是便儘可能減少他的出場時間。

詹姆斯

老詹的確還保持著極佳的身體和競技狀態，可等到月底他就年滿35周歲了，體力自然遠不如年輕的時候，常規賽過度消耗他顯然不利於備戰季後賽，甚至還會增加受傷的風險。可出場時間減少並沒有影響老詹的數據和效率，他場均依然可以貢獻25.6分6.9籃板10.7助攻，PER、真實命中率和使用率等數據依然名列前茅，實在讓人感嘆詹姆斯不但天賦異稟，而且是何等的刻苦和自律。

沃格爾

實際上沃帥可不只是偏心老詹一個人，湖人隊的多名主將出場時間均明顯少於過去。「濃眉」賽季至今場均出場34.1分鐘，僅高於新秀賽季和上賽季，其中上賽季還是他主動申請交易，出場時間才受到限制的；隆多出場時間同樣是生涯新低，到了生涯暮年，他更多開始扮演年輕球員的導師角色；丹尼-格林場均出場25.3分鐘，創13-14賽季以來新低；剩下的像霍華德、布拉德利和麥基等人，出場時間同樣也受到限制。

當然這不只是沃格爾的功勞，另一方面總經理佩林卡休賽期的引援著實給力。到現在為止湖人有多達12名球員場均出場時間超過10分鐘，哪怕有主要輪換球員休戰，也總是有球員能夠頂上。去年夏天湖人引進的很多球員與詹姆斯完全不搭，整支隊伍都找不到幾個合格的投手，而今年夏天佩林卡的引援和操作就很有針對性，也許有些球員不一定是名氣最大天賦最高的，卻可能是現階段最適合這支球隊的，這是佩林卡帶給湖人的第一大轉變。

魔術師

此外上個賽季除了傷病，「魔術師」約翰遜的大嘴巴或多或少也影響了湖人的更衣室氛圍，明明交易還沒有達成，結果更衣室內人人自危，心思都不在球場上了。而這個賽季我們有經常聽見佩林卡發聲嗎？真的很少。他只是默默在背後支持著球隊，隨時都在尋找合適的機會補強陣容，這也很大程度上使得紫金軍團一直在正確的軌道上穩步前行，成為總冠軍的有力爭奪者。
