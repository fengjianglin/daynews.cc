詹姆斯究竟是怎樣一個人？隆多說心裡話，他是徹底看明白了

2019-12-21 110條評論 1,016次閱讀 87人點贊

詹姆斯究竟是怎樣一個人？隆多說心裡話，他是徹底看明白了！

湖人今天客場以104比111不敵雄鹿，主要是雄鹿打出全民皆兵，湖人大部分人發揮不好，也許和連續客場導致身體疲勞有一定關係，另外防不住字母哥也是事實。

本場比賽，詹姆斯登場37分鐘，19投8中，全場得到21分12籃板11助攻；相對而言，詹姆斯今天發揮一般，第一節詹姆斯的發揮可以說是很糟糕，詹姆斯第一節在連續吃了字母哥和洛佩斯兩個火鍋之後，詹姆斯的進攻停止了，首節詹姆斯1分未得。

次節，10分10秒詹姆斯打洛佩斯，詹姆斯拉出來，並且後撤一步命中一記三分，接著詹姆斯一條龍飛身上籃打進，中段詹姆斯右邊突破得手，最精彩的是1分36秒詹姆斯中路撕開防線，從人縫中闖過，將皮球挑進籃筐，半場結束，湖人以46比65落後。

下半場易邊再戰，9分40秒詹姆斯左邊弧頂一記三分再次命中，奇怪的是，詹姆斯5分鐘陷入得分荒，直到2分35秒詹姆斯右邊後撤跳投命中，這一節詹姆斯僅僅得到5分，三節戰罷，湖人以81比89繼續落後。

末節，詹姆斯一上場就連續犯規，領到5犯，由於遇上犯規麻煩，詹姆斯隊禁區的進攻幾乎消失，雖然詹姆斯9分48秒又一次射中三分，當時對雄鹿並沒有很大的殺傷力，沒有詹姆斯強硬的進攻，湖人的火力明顯減弱，雄鹿的進攻潮水一般，三分雨密不透風，一起一落，雄鹿佔據上風，最終湖人以104比111不敵雄鹿。

眾所周知，之前NBA球星凱文-加內特發表了一些關於詹姆斯的言論，在湖人與雄鹿的比賽之後，記者採訪隆多，問到此事。

隆多笑著回答：「他（加內特）在到處點火是吧？」朗多這是開玩笑。

隆多接著說：「我愛勒布朗，他是最喜歡的隊友之一，他是一個非常無私的傢伙。」

隆多表示，我就喜歡詹姆斯，我就是愛詹姆斯，為啥？因為詹姆斯是一個無私的人，隆多在無私前面加上「非常」兩個字，可見詹姆斯有多無私。

隆多繼續說：「老實說真的對他恨不起來，不過在過去那種交手的日子，你就會有那種心態，這和在更衣室和他在一起的感覺不同，但你能感覺到在場內外他是多麼真誠的一個人。」

朗多表示，以前和詹姆斯交手，確實對詹姆斯有一點意見，但是自從成為隊友和交心之後，就有不一樣的感覺了，你根本對詹姆斯恨不起來，詹姆斯無論場內和場外都是一個很真誠的人，朗多是一個實在人，他是實話實說，詹姆斯究竟是一個什麼樣的人，隆多的話，也讓大家看明白了，是一個無私、真誠的好人！
