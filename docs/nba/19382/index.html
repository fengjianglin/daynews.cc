開玩笑還是真心話？路威社交媒體回應球迷，快船真的要內訌了嗎

2019-12-18 1,516條評論 2,999次閱讀 1275人點贊

最近，洛杉磯快船隊球員路威在社交媒體上面曬出自己的照片，而這也吸引了不少球迷的目光，其中一名NBA球迷就留言稱快船隊當家球星倫納德想要交易路威。非常出人意料的是，路威竟然在社交媒體上回應了這一留言，而路威給出的回答是「幸好他（倫納德）不是總經理！」路威在社交媒體上的這一番回複信息，最終也是瞬間成為球迷們討論的熱點。

當然，到底應該如何理解路威的話語，現在來看也是存在著兩種完全不同的意思。首先第一種就是路威在調侃球迷，他的回答僅僅只是在開玩笑而已，路威並不認為自己和倫納德之間存在著問題。而另外一種則是恰恰相反，路威公開說出了自己的真心話，由於此前「不會跑戰術」的採訪言論，路威和倫納德確實已經產生矛盾問題，這兩位快船主力球員現在公開進行對陣。

而到底是開玩笑還是真心話，目前估計除了路威本人，其他人也只能是進行猜測。但是平心而論，路威可謂是聯盟中的實力派球員，雖然在快船隊陣容之中，路威僅僅只是一名替補球員，但是在帶隊戰績和場均數據上面，路威都是標準的NBA球場巨頭表現。除此之外，路威的賽季合同也只有800萬美金而已，他算得上是快船隊陣中性價比最高的球員之一。

至於快船隊的頭號球星倫納德，雖然也是當之無愧的現役超巨，但由於自身的傷病隱患問題，倫納德的出勤率一直不是非常好看，快船隊很多時候都是喬治、路威等人辛辛苦苦帶隊。現如今傳出倫納德、路威兩人不和的傳聞，這也是讓人感到非常的意外，難道目前球隊戰績高居西部賽區第二名的快船隊，他們真的要出現球員內訌的鬧劇嗎？這一切究竟是子虛烏有還是千真萬確，我們也只能繼續等待最新的消息報道！
