合送14次失誤，個個都很致命！哈登威少就是國王隊的卧底

天天要聞
2019-12-10

文/籃郭先生

NBA的常規賽繼續進行，紅隊與國王進行了一場對決。對國王來說，賽季初期的表現，有些慘不忍睹，球隊的傷病情況，也不容樂觀，福克斯、巴格利等球隊核心，都是因傷缺席的。但即便如此，沃頓帶領這樣的國王，還是殺入到西部前十的位置，這一點還是有點厲害的。若是福克斯、巴格利等人歸來，球隊的即戰力，還會更加恐怖。而這樣的比賽，國王也是以119-118險勝紅隊，數據方面，國王隊的約瑟夫14分6次助攻，希爾德26分，巴恩斯19分8個籃板，別利察17分5個籃板，霍姆斯16分8個籃板，博格丹諾維奇13分，阿里扎8分7次助攻。紅隊這邊，塔克10分19個籃板，豪斯11分，卡佩拉13分17個籃板，麥克勒摩12分4次助攻，克拉克11分3個籃板。

相信許多人注意到一點，那就是，數據統計裡面，沒有哈登與威少的數據。這是故意的，因為，他二人是這一次的「男主角」，是有很多細節值得一說的。這樣的比賽，哈登貢獻27分10次助攻8次失誤的數據，而威少的效率很高，17中13，三分4中2，貢獻34分8次助攻6次失誤的數據。二人從數據來看，或者說，若是拋開失誤數來看的話，二人的貢獻，還是有點大的。那就是，二人聯手貢獻了61分18次助攻。這樣的組合，在NBA的後衛組合裡面，也算是頂配的組合了。畢竟，能有這麼多輸出的組合，還是很少的。甚至可以說，隨著克萊庫里的受傷，這樣的組合，應該獨樹一幟的。

不過，考慮到失誤，還是有些扎心的。這樣的比賽，二人一起貢獻了14次失誤，而國王全隊呢？總共才有9次。也就是說，哈登1個人的失誤次數，和國王隊全隊差不多。再加上威少還有6次失誤，以及全隊還有另外的五次失誤，對紅隊來說，是有點傷的。而對國王來說，則是贏球的機會。而最終，他們也是憑藉三分絕殺了紅隊。但其實，對紅隊來說，本來是可以拉開分差的，也是可以終結比賽的，甚至可以說，若是失誤少一點，國王根本沒有絕殺的機會。遺憾的是，2名核心球員的失誤太傷人。所以，球隊也是給了對手絕殺的機會，最終對手沒有給紅隊任何的機會，別利察手起刀落。

這樣的比賽，客觀地說，若是找背鍋俠的話，哈登肯定是算一個的，而威少雖然進攻效率很高，但也送出6次失誤，所以，也是要被提及的。此外，豪斯的防守形同虛設，也是一個原因。但是，豪斯只是角色球員，影響力是很有限的。而哈登威少，則是不一樣的。他們是球隊的核心，只是遺憾的是，他們如此多的失誤，也是實力扮演卧底的節奏。因為，對一支球隊來說，核心球員的一切，都是會被放大來看的。

合送14次失誤，個個都很致命！哈登威少就是國王隊的卧底。或許，也是因為這二人的失誤次數太高，且防守不是很好，所以目前的紅隊，也只是西部第五的水平。但其實，他們的實力，應該是西部第三的實力。只是遺憾的是，他們目前的發揮，還沒有達到預期。
