又一巨頭誕生！詹姆斯親自承認，湖人“偽三巨頭”正式成立了

2019-12-10 2,615條評論 4,465次閱讀 3290人點贊

三名球員砍98分！放在以前，這種數據只有擁有三巨頭的球隊才能夠做到，比如，擁有全明星首發陣容的勇士隊，庫里，杜蘭特和克萊。再比如，擁有詹姆斯，樂福和歐文的三巨頭陣容。但湖人明明是雙核，只有兩個巨頭，卻也砍下了98分。

難不成，湖人還有第三個巨頭？

如果不看比賽結果的話，球迷還誤以為這個所謂的第三巨頭是庫茲馬，但事實上，這個人卻是上個賽季連輪換都進不了，甚至本賽季是他第一個正式的NBA生涯，他叫卡魯索，是卡哇伊的卡，也是禿曼巴的禿。

和森林狼的這場比賽，詹姆斯和濃眉合砍82分，卡魯索替補出場打了30分鐘，11中6，三分球3中2，得到16分4籃板4助攻2搶斷0失誤。隆多因傷休戰，卡魯索臨危受命，承擔起了更多的責任，同時他不負所望，他沒有辜負沃格爾的信任，更加沒有辜負斯台普斯球迷在他被換上場之時所送給他的掌聲和歡呼聲。

甚至於，在這場比賽結束之後，詹姆斯親自發推把卡魯索的投降P在了他自己和濃眉之間，並且附言湖人三巨頭合砍98分。

ESPN在賽後也表達了對卡魯索的讚揚，也是把卡魯索歸到湖人「三巨頭」的行列，官方媒體把他和詹姆斯，濃眉相提並論，這是一種贈予他的極大的榮譽。

當然，只要是稍微清醒一點，我們都會知道，這只是一種調侃，就跟前幾個賽季湖人球迷狂呼「湖人總冠軍」一樣，就算湖人是西部墊底，樂透級別的球隊，也總是有人再喊「湖人總冠軍」，現在，卡魯索只是進入聯盟的第一個賽季的新人，把他喊成湖人第三巨頭，是對卡魯索的一種期望，一種欣賞，一種信仰和喜愛。

本來以為湖人第三個爆發點會是庫茲馬，但是沒想到庫茲馬沒有來得及爆發，卡魯索就先上位了。究其原因，不是庫茲馬不行，而是卡魯索太拼了，他知道湖人需要什麼，他自己該幹什麼。

就拿這場臨危受命的比賽來講，隆多不在，湖人的第二陣容總需要一個進攻發起點，一個組織比賽的人，濃眉最好的進攻選擇，不是在三分線外持球打，而是需要在高位背身接球，在低位卡好位之後，需要有一個組織後衛把球安全送到他手裡。能做到這的人，除了詹姆斯，隆多之外，也就只有卡魯索了。

拼到膝蓋出血，卡魯索依然不下火線，會更衣室簡單處理之後，立馬回到場上，同時現場球迷給予他最熱烈的掌聲，以前這種掌聲是給庫茲馬的，現在給了卡魯索。

卡魯索是角色球員，他所帶給湖人的幫助並不是主宰比賽，而是幫助兩大核心主宰比賽。

詹姆斯是打持球的球員，但有時候也需要另外的持球點幫助他分擔持球壓力，可以讓詹姆斯做到在「場上休息」。濃眉需要喂球，在和森林狼的那場比賽中，當場上沒有詹姆斯和卡魯索的時候，很明顯地看到湖人的進攻沒有重心了，庫茲馬和庫克都是終結點，霍華德和杜德利也只會傳導球而已，所以濃眉處在一個很尷尬的位置，而等到卡魯索帶傷回歸之後，迅速找到了濃眉。卡魯索很清楚自己的定位和存在的意義。

除了喂球之外，卡魯索能夠簡單的組織進攻，能夠投定點三分，籃下終結也有了提升。最大的作用還是在防守，老實講，湖人隊外線的防守，侵略性最大的是布拉德利，但卡魯索的防守侵略性也不必布拉德利差多少，而且一看就是非常有經驗的防守悍將，這正是湖人需要的地方，比賽的活力，積極性就從這些方面表現出來。

如果非要說卡魯索是湖人的一個「巨頭」，認真講的話，顯然達不到巨頭級別，但處於對他的喜歡和鼓勵，他已然成為了湖人球迷最喜歡的角色球員，已經被視為了第三巨頭。

不過，還是輕點吹比較好，萬一卡魯索飄了怎麼辦。
