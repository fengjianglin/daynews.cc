本賽季首筆交易達成！騎士為何送走新星得到場均僅2.2分控衛？

2019-12-24 987條評論 2,721次閱讀 1324人點贊

北京時間12月24日，據美媒體報道，克利夫蘭騎士隊和猶他爵士隊已經達成了一筆交易，騎士隊送出後衛喬丹-克拉克森，並從爵士隊得到控衛丹特-艾克塞姆和2個未來的二輪選秀權。

在2018年交易截止日之前，克拉克森被湖人交易到騎士，而如今，克拉克森處於合同的最後一年，他在本賽季的薪水是1343萬美元，他也是2014年的46順位新秀。本賽季，克拉克森代表騎士打了29場比賽，場均可以得到14.6分、2.4籃板以及2.4助攻。

而丹特-艾克薩姆是2014年的首輪5號秀，他被爵士隊寄予厚望，自從進入到NBA之後，艾克塞姆一直受到傷病的困擾，2018年夏天，艾克塞姆和爵士續約了一份3年近3000萬美元的合同。本賽季，艾克塞姆代表爵士隊打了11場比賽，場均出場7.6分鐘，得到2.2分和0.6次助攻。爵士送出的兩個選秀權分別是2022年來自馬刺的二輪選秀權和2023年來自勇士隊的二輪選秀權。

值得一提的是，由於詹姆斯離開球隊之後，騎士就進入了重建的過程，而在今年夏天，球隊並沒有太多的動作，甚至在交易市場上沒有做任何事情。因此，本賽季以來，騎士的戰績沒有任何的起色，8勝21負的戰績讓球隊管理層感到十分失望，多位球員已經被擺上貨架。克拉克森作為球隊的主要輪換球員，他是一名不錯的外線射手，三分命中率達到37.1%，是個不錯的板凳得分點。

而對於現在的爵士來說，18勝11負的戰績排名西部第六，球隊希望得到進一步的補強，而艾克薩姆本賽季並沒有太多的出場機會，他一共只打了83分鐘，場均得分2.2分。他和克拉克森都是2014年選秀出身，而艾克薩姆是樂透新秀，然而他受到了傷病影響，膝蓋韌帶撕裂。

由於在今年夏天，爵士花費了高薪得到康利，但是球隊的戰績不如上個賽季，因此他們需要的是能夠繼續補強，而克拉克森的薪資不菲。所以，騎士也需要通過釋放薪資空間去讓球隊進一步完成重建。
