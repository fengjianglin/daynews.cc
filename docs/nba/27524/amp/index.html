名記透露詹姆斯身上出現多處傷病，包括腹股溝！球迷：怎麼又是他

天天要聞
2019-12-22

北京時間12月22日，對於廣大詹姆斯球迷來說，算得上是既無奈又難過的一天了。據了解，湖人隊當家球星詹姆斯遭遇傷病困擾，下一場對陣掘金的出戰概率僅為25%，換句話說，詹姆斯可能將迎來新賽季第一次常規賽缺席了。

目前，湖人更新了傷情報告，詹姆斯的胸廓肌拉傷，他可能缺席明天與掘金隊的比賽。據知情人士透露，雖然詹姆斯沒有接受核磁共振檢查，但這不是一個輕傷，而且也不是所謂的「負荷管理」。根據沃格爾透露，詹姆斯這次受傷發生在與步行者的比賽中，但是儘管如此，詹姆斯還是堅持打完了與雄鹿隊的比賽。

據溫德霍斯特報道，詹姆斯目前身體出現多處傷病，不單單是胸廓肌拉傷那麼簡單，另外還包括上賽季讓他缺席許多場比賽的腹股溝傷勢。看到是溫德霍斯特的報道之後，不少球迷紛紛表示：「怎麼又是這個胖子！」「溫德霍斯特？那詹姆斯應該沒事！」「溫德霍斯特的消息可信度幾乎為零，甚至是負數！」

不可否認，溫德霍斯特的報道真的是令人無奈，在前段時間裡，有消息顯示，湖人隊主教練沃格爾表示庫茲馬根本不適合現在的湖人隊！巧合的是，這則消息也同樣出自溫德霍斯特，但是大家仔細想一想，庫茲馬前段時間的表現雖然不盡人意，但他畢竟是湖人隊休賽期里留下來的唯一年輕力量。而且，庫茲馬在看到溫德霍斯特的報道之後，也已經給出了明確的回復：「井底之蛙！」

再者來說，沃格爾身為湖人隊的主教練，雖然球隊遭遇了新賽季至今的第一次2連敗，但是他有可能面對記者的採訪，說出這樣傷士氣的話嗎？另外，詹姆斯在新賽季開始之前，已經明確表示過，自己之前遭遇的腹股溝拉傷，已經百分之百恢復了。

​詹姆斯這樣的球員，對自己的身體情況應該是非常清楚的，如果沒有十足的把握，相信詹姆斯也不會信誓旦旦地說出百分之百這樣的話。難道，溫德霍斯特就這麼想成為湖人隊的「御用記者」嗎？繼庫茲馬、沃格爾之後，詹姆斯終究也是沒能躲過他「所謂的報道」啊！不過，還是祝願老詹可以早日康復！

那麼，問題來了：如果下一場比賽濃眉哥獨自帶隊，大家覺得湖人能夠戰勝掘金隊嗎？
