某球隊高管：想交易得到特里蒙特-沃特斯，他可以在NBA立足

2019-12-25 2,114條評論 3,076次閱讀 549人點贊

虎撲12月24日訊 雅虎體育記者Keith Smith報道，一些聯盟高管和球探近期在對發展聯盟的考察時頻繁提到了一個球員的名字，那就是凱爾特人的新秀後衛特里蒙特-沃特斯。

一位球隊高管表示：「他現在是NBA級別的替補後衛了，我告訴我們球隊經理要交易得到他，因為他比我們現在的人選要出色。」

而另一位球隊高管則表示：「我們也會詢問（沃特斯的交易可能性），但波士頓方面不會傾聽。他們為什麼要這樣做呢？那個孩子真的很棒，他是個子很小，但他特別聰明。他速度足夠快，可以創造出手機會，而且他還是一個進攻機會製造者。如果這賽季後期他還沒得到合同的話，下賽季他一定能拿到一份標準合同。」

最後，一位球探表示：「我錯過了他，那是重要時刻，當時我認為他對於這個聯盟來說太矮了。但他很強硬，而且速度很快。我不知道他的未來能超越什麼，但他至少是一個NBA的替補控衛，這是有價值的，因為現在幾乎每支球隊都在打三後衛。」

沃特斯身高1.78米，體重79公斤，在今夏選秀大會上次輪第51順位被凱爾特人選中，隨後他與綠衫軍簽下了一份雙向合同。

本賽季至今，沃特斯只代表凱爾特人出戰過1場比賽，在20分鐘里得到7分2籃板3助攻1搶斷2蓋帽。

在本賽季的發展聯盟比賽中，沃特斯代表緬因紅爪出戰了13場比賽，場均32.5分鐘內可以得到20.1分3.2籃板7.5助攻2.3搶斷，並在11月份榮獲了發展聯盟最佳球員。
