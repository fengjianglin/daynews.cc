NBA近十屆選秀最強之人！維金斯養生也沒對手，東契奇獨一無二

2019-11-29 0條評論 17次閱讀 0人點贊

選秀是NBA球隊補充人才的重要途徑，但選秀也是要運氣的，順位越高顯然越可以選中自己認為天賦越高的球員，但選中後能否兌現天賦卻沒有人知道。最近十年，一大批球員通過選秀進入NBA，有的留下也有的離開，有的成為巨星，也有的泯然於眾人。那接下來就和大家一起聊聊這十屆選秀各自的最強之人吧！

2010年：保羅喬治

2010年選秀的狀元是沃爾，但最能代表10年新秀的球員卻是保羅喬治。事實上，沃爾具備一切成為巨星的潛質，但最終卻因為傷病而沒能達到巔峰。除了沃爾，考辛斯、海沃德等人也都打出了不錯的表現，但顯然他們如今的地位都和喬治相去甚遠。

2011年：倫納德

2011年為NBA輸送了很多巨星級別的球員，狀元歐文、克萊-湯普森、肯巴-沃克、吉米-巴特勒，此外還有武切維奇、莫里斯兄弟等人。而這些人當中成就最高、聲望最高的無疑是倫納德，雖然湯神比他冠軍數量更多，但倫納德有兩個FMVP，而湯神的3次總冠軍一次是二當家，剩下兩次則是三當家。

2012年：濃眉哥

相比2011年選秀，2012年選秀的星光相對暗淡了一些，但也輸送了比爾、利拉德、追夢格林等人。這些人當中，最強之人當然是濃眉哥戴維斯，理由不需要解釋。

2013年：字母哥

如果說2012年只是星光暗淡的話，那2013年就完全可以說是行管慘淡了！首先出了一位史上最水狀元之一本內特，再能拿得出手的也就奧拉迪波、CJ和戈貝爾了。這幾個人實力都不錯，但都沒達到巨星水準，如果不是字母哥太強勢的話，這一屆選秀恐怕會成為史上最水的之一。

2014年：維金斯

2014年選秀也出現了很多優秀球員，但最強之人最終只能落到兩個人頭上，一個是維金斯一個是恩比德。他們位置不同，隊友不同，因此無法最直觀的比較。但不可否認的是，維金斯的天賦更高，即便他本賽季場均得分上升到25+，可人們仍然不認為他已經達到上限。反觀大帝已經基本接近上限了，所以即便是養生時期的維金斯，他也可以代表2014年的新秀。

2015年：唐斯

2015年新秀當中，唐斯無疑是目前為止最強的，只有布克可以對他的地位造成衝擊，但顯然布克的綜合能力是不如唐斯的。

2016年：西蒙斯

2016年新秀們的發展速度是讓人咋舌的，僅過去3年就已經有5人簽下了億元大合同，西蒙斯和穆雷更是簽下了1.7億以上的頂薪合同。毫無疑問，這一年新秀當中西蒙斯是最強的，儘管他還沒有完全開發出三分球。

2017年：米切爾

2017年新秀的整體天賦其實還不錯，然而狀元富爾茨和榜眼球哥都因為種種原因沒能兌現天賦，塔圖姆和真正的球星也還有一線之隔，所以給人整體感覺成材率並不高。毫無疑問，第十三順位的米切爾是這一年新秀中的代表人物。

2018年：東契奇

從去年開始，東契奇就幾乎刷屏了NBA，可以說他是2018年新秀中獨一無二的存在。即便艾頓、巴格利也都不錯，即便特雷-楊一直在追趕他，但就目前而言2018年的新秀們沒有人能超過東契奇。

2019年：莫蘭特

2019年的新秀當中天賦最高的自然是狀元蔡恩-威廉森，但遺憾的是他無法保證健康，賽季已經過去4分之一了，可他回歸依然遙遙無期。而在這段時間，莫蘭特悄然成為了2019年新秀的代表人物，他的數據、能力、球隊地位都是同年新秀中最好的。當然了，如果蔡恩能夠順利回歸的話，莫蘭特的地位會被衝擊，但前提是蔡恩能保持健康。
