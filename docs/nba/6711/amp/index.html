搭檔利瓜太舒服？開拓者大將復出首戰13+12，攻防給力更獲盛讚

天天要聞
3 週 ago

​

自從卡梅隆-安東尼穿上開拓者球衣以來，他就一直能得到首發出戰的機會。

至於開拓者中鋒哈桑-懷特塞德，前段時間他一直在處理右臀部酸痛的傷病，而且在缺席開拓者前兩場比賽後終於能重返賽場。

​

在北京時間11月26日開拓者對陣公牛的比賽，這也是安東尼加盟開拓者以來的第4場比賽，也是首次能同時搭檔利拉德，麥科勒姆，胡德和懷特塞德等首發球員的比賽。在這場比賽，開拓者最終以117-94戰勝公牛，這也讓球隊結束前段時間的4連敗表現。

「我很高興能回到這裡，這是我第一次與利拉德和安東尼一起比賽，所以這絕對是很棒的感覺」懷特塞德說道。

​

懷特塞德的復出為開拓者帶來攻防兩端最急需的賽場能量，籃板和防守實力。

​

這位30歲的球員在這場比賽中能得到23分鐘上場時間，並且有8投6中得到13分12籃板和2封蓋的數據表現。

懷特塞德在球隊戰勝公牛後表示，能在場下觀察隊友的發揮確實對自己更好地適應球隊節奏有很大的幫助。

「我從旁觀的角度看到一些事情，而且我真的覺得我可以幫助，我確實也想做到這點，這就是重點之一。」

​

「他在攻防兩端的表現都非常出色，他在封蓋，摘下籃板以及籃筐附近終結進攻上都能讓球隊得益」開拓者主教練斯托茨說道。

​

自耦從開拓者在北京時間11月20日不敵鵜鶘以來，懷特塞德就一直缺席比賽，而在那場比賽中懷特塞德還是能打出14分和14籃板的兩雙數據。

「夥計們在投籃，我只想為他們擋拆出空位，只想盡我所能摘下籃板，我真的認為我們可以成為一支危險的球隊」懷特塞德說道。

​

懷特塞德的復出也意味著開拓者能在防守端和籃板上都得到更大的優勢，所以就算安東尼確實開拓者戰勝公牛的關鍵所在，但懷特塞德在搭檔利拉德和安東尼的情況下似乎也得到有更好表現，甚至是成為球隊最後一塊拼圖的那名關鍵球員。

​

正如懷特塞德自己所說，同時搭檔利拉德和安東尼確實是非常有趣的一件事。
