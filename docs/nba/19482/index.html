庶民的上籃！追夢無厘頭進攻復刻櫻木，球迷：這球打算傳杜蘭特？

2019-12-17 1,968條評論 4,256次閱讀 2748人點贊

北京時間12月16日，勇士主場79-100不敵國王，遭遇4連敗，過去9場輸掉8場。追夢格林打了26分鐘，只得到7分2籃板3助攻。沒有庫里和湯普森，格林在組織進攻方面的作用大打折扣，他的表現根本不像是一位全明星球員。

賽季至今，格林場均9分5.9籃板5.8助攻，投籃命中率41%，三分命中率26.5%。籃板、助攻和兩項命中率，都創下近6個賽季的新低。賽後，格林很失望，他說：「我們輸了20分，我們從沒打成這樣，我們爛透了，無論哪個方面都是如此。」

格林自己也打得很差，在比賽的一個快攻回合，他甚至連簡單的上籃都沒能命中。只見格林大步殺向禁區，他的上籃竟然直接扔向了籃板高處，和籃筐相去甚遠！有球迷指出，格林的這個球和《灌籃高手》里櫻木花道的那記庶民上籃，真有八九分相似。

漫畫中的櫻木花道，也是在一次快攻中用力過猛，直接將球扔向籃板高處，結果還是流川楓跟進將球補中。勇士隊可沒有流川楓，這個本該輕鬆拿到的分數，就被他們錯過了。

奧尼爾的「五大囧」節目，在社媒上發出了格林上籃的這個視頻來調侃他。球迷們也紛紛留言，不少人提到了KD（杜蘭特）。

「我猜他忘了KD已經和籃網簽約了。」一個球迷調侃道。

「這球是打算給KD的，」一個球迷寫道。

「KD，但……等等他已經不在這兒了，」另一個球迷回復道。

杜蘭特曾經在一次採訪中坦言，自己和格林之間的矛盾，是他離開勇士的原因之一。此時此刻的格林，不知道心中是不是有些無奈。

勇士仍在繼續輸球。不敵國王后，勇士的新球館大通中心戰績變成2勝10負。這是什麼概念？聯盟30支球隊里，大通中心的主場戰績是最差的，沒有之一。

作為上賽季打進總決賽的球隊，勇士的懸念已經不是能否進入季後賽，而是能拿到什麼順位的選秀簽。
