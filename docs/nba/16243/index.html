扎扎·帕楚利亞至少26次惡意致使球員受傷，NBA官方為什麼不管？

2019-12-14 325條評論 1,713次閱讀 801人點贊

扎扎·帕楚里亞，1984年2月10日出生於喬治亞第比利斯，前喬治亞職業籃球運動員，司職大前鋒/中鋒，現任NBA金州勇士隊顧問。這位臭名昭著的NBA球員已經在9月6日正式退役了，回顧扎扎的職業生涯簡直就是劣跡斑斑啊！

那麼對於帕楚利亞這種多次惡意傷人的球員，聯盟為什麼不管呢？

首先看看NBA對惡意犯規的定義：惡意犯規，指的是一些沒有必要卻又可能對被犯規者造成身體傷害的行為，中文一般稱為「違反體育道德的犯規，惡犯分為一級和二級。

一級惡意犯規，是指不必要的接觸在球成活球後，與對方隊員發生非法的身體接觸，是侵人犯規。一級惡意犯規通常是指防守方故意左右擺動，對進攻方進行故意的重度接觸以及後續動作。

二級惡意犯規，指不必要的或過度的身體接觸，通常是故意的重度碰撞以及後續動作。裁判在判罰二級惡意犯規時，一般都要審查錄像，並和對方主教練進行溝通和告知。

我們可以看到對於惡意犯規並沒有一個很明顯的界定，主要還是靠裁判的主觀判斷，在球場上情況瞬息萬變，裁判也有漏判的時候。

就如下圖一樣，扎扎·帕楚利亞整個人壓在威少的腳踝上，一個124.7公斤的中鋒整個壓在腳踝上，這是非常危險的，一不小心就會導致球員賽季報銷了，然而裁判並沒有給惡犯，原因是帕楚利亞的膝蓋被隊友的腳鉤了一下，可能是因此失去平衡，並不是故意的…..

帕楚利亞也因這個動作引起很多球員的不滿，就連保持中立的歐文也表示：「他這不是打球強硬，而是臟！」事實上聯盟早就開始注意這位臭名昭著的球員了，甚至成立過專門的調查組，但最後也是不了了之。

其實聯盟對惡犯的處罰力度是非常大的，如下：

兩種惡意犯規都會讓對方獲得兩次罰球的機會，並得到控球權，通常被稱為「兩罰一擲」。二級惡意犯規者，將在犯規後被立即驅逐出場；如一名球員在單場比賽內，被判兩次一級惡意犯規，也符合被驅逐出場的條件。

此外，二級惡意犯規者在單賽季首次被判罰之後，還會被聯盟追加處以最高不超過35000美元的罰款，並根據聯盟審查，決定是否禁賽。

那麼問題來了，這樣一位臭名昭著的球員為什麼還能有球隊簽呢？甚至還有球員為他辯解呢？（杜蘭特就曾為他辯解）

每個球員在球隊都扮演著自己的角色，得分手、防守悍將…聯盟厲害的球員太多了，有些球員基本沒幾個人能防住（喬丹、科比、詹姆斯…)，球隊要想限制這樣的對手得分就只有犯規了，這是個又累又髒的活，很少有球員願意做，而帕楚利亞恰恰是為數不多的願意擔任這個角色的球員，所以就目前來看，球隊是需要這樣的球員的。

NBA自「壞孩子」時期過後就加強了對球員的保護，對惡犯的處罰力度也是非常大的，但小俗覺得在裁判的判罰方面聯盟做的還是不夠，應該多鼓勵裁判看回放，對這種惡意犯規傷害球員的行為應該採取零容忍的態度！
