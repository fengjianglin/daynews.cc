施羅德31分保羅18分關鍵中投雷霆24分大逆轉勝灰熊

2019-12-20 95條評論 1,416次閱讀 384人點贊

NBA常規賽繼續進行，雷霆主場126-122逆轉灰熊。

亞當斯和亞歷山大各有上籃後瓦蘭丘納斯和莫蘭特也籃下得分，首節雙方不斷強突籃下連續得分交替領先，節末米爾頓勾手得分再中三分，首節灰熊32-27領先。第二節莫蘭特和施羅德各中三分後莫蘭特連續得分、狄龍-布魯克斯再中三分球，莫蘭特空接暴扣灰熊領先了16分。弗格森三分止血後保羅連續得分，節末加里納利也連續得分，半場灰熊68-59領先。

易邊再戰後莫蘭特2+1再進中投帶領灰熊12-0開局將分差拉開到24分，保羅和亞歷山大連續籃下得分、加里納利再連中三分球，施羅德也連續中投得分，雷霆不斷縮小分差。節末內德爾也中三分、亞歷山大壓哨完成搶斷中投，三節過後灰熊100-93領先。

第四節亞歷山大和施羅德合力打出一波8-2將分差縮小到1分，瓦蘭丘納斯上籃穩住局勢後亞當斯上籃再進、施羅德中投，雷霆反超了比分。克拉克連續上籃，保羅回應三分球，之後克拉克連續得分，不過施羅德、亞歷山大和保羅都有上籃，最後一分鐘雙方打平。保羅最後52秒中投得分，之後莫蘭特連續上籃不中、克拉克補籃也沒有，施羅德秒穩穩四罰全中將分差拉開到4分，之後克拉克三分不進，比賽結束。

全場比賽瓦蘭丘納斯得到24分9籃板3助攻，莫蘭特22分5籃板6助攻，狄龍-布魯克斯15分，布蘭登-克拉克19中13砍下27分，賈倫-傑克遜9分。雷霆這邊保羅18分，亞歷山大20分9籃板，加里納利20分，亞當斯13分10籃板，施羅德31分7助攻，諾艾爾10分。
