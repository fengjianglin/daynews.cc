NBA最新排名:掘金緊追湖人，獨行俠快船進前四，國王要進前八？

2019-11-25 0條評論 16次閱讀 0人點贊

NBA常規賽繼續進行，最近狀態火熱的獨行俠擊敗了馬賽克隊，延續了他們的連勝腳步。快船隊耶穌全員出戰，沒有選擇負荷管理，對手是背靠背作戰的鵜鶘，憑藉著替補球員的強大火力，快船也繼續著連勝的步伐。此外，籃網隊擊敗了尼克斯，贏下了這一場德比戰，迎來了3連勝，歐文缺陣後取得了4勝1負的成績，勝率也提升到了50%。

排名方面，雄鹿隊7連勝後排名第一，凱爾特人，熱火和猛龍都是11勝4負分別排在2到4位，76人隊多輸一場位居第五，籃網隊8勝8負排名穩定在第七。

說完了東部再看看西部，掘金隊擊敗了太陽以後繼續緊追榜首的湖人隊，而快船和獨行俠排在第三四位。國王隊在擊敗了奇才隊以後目前距離西部第八的森林狼和西部第七的太陽都是半個勝場，看來賽季剛剛開始表現不佳的他們也逐漸的找回了狀態，還是很有希望衝擊季後賽的。排名後幾位的還是勇士，開拓者，馬刺等。

你覺得本賽季哪兩支球隊可以進入最後的總決賽呢？歡迎大家在評論區留言發表自己看法哦！
