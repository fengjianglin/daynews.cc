小男孩：新奧爾良鵜鶘VS奧蘭多魔術，印第安納步行者VS夏洛特黃蜂

2019-12-16 2,067條評論 3,947次閱讀 1489人點贊

新奧爾良鵜鶘VS奧蘭多魔術

NBA 2019-12-16 星期一 4:30

場地：新奧爾良球館

新奧爾良鵜鶘情報信息

新奧爾良鵜鶘目前常規賽取得6勝20負的戰績排名在西部第14位，球隊的勝率23.1%，場均得分112.6分，場均失分119.7分。鵜鶘在上一場對陣76人的比賽中，比賽一開始便被76人拉開比分高達14分，第2節雖有反攻，但無奈76人實力強大，最終以總比分109:116不敵76人。在上場比賽中英格拉姆得到全隊最高分32分，並貢獻6個籃板，雷迪克得到19分3籃板3助攻，霍勒迪得到12分6籃板10次助攻，哈特得到10分5籃板3助攻，費沃斯得到8分8籃板。本場比賽對陣魔術，雖在過去的兩場比賽中戰平，但本賽季鵜鶘難有人扛起進攻大旗，難求一勝。

奧蘭多魔術情報信息

魔術目前常規賽取得11勝14負的戰績排名在東部第8位，球隊的勝率44%，場均得分102.5分，場均失分103.8分。奧蘭多魔術在上一場對陣湖人的比賽中，首節開始便被湖人取得26:9的領先，第二、三節魔術隊抓住湖人失誤機會，打出18:3的小高潮，但最後不敵湖人強大，以87:96落敗。在上場比賽中魔術隊艾薩克得到19分8籃板，弗尼耶得到18分2籃板5助攻，戈登得到14分14籃板4助攻，穆罕默德班巴得到13分4籃板，本場比賽對陣是西部弱隊鵜鶘，相信魔術會有出色表現贏取勝利。

綜合分析

魔術隊中鋒武切維奇本場比賽處於賽前決定的狀態，魔術隊近期戰績下滑，最近三場比賽不勝，主場作戰的鵜鶘近期已經遭遇11連敗，在球隊的傷員逐漸回歸的情況下，戰績沒有任何回暖，本場比賽看好魔術

比賽推薦：【大219.5】

印第安納步行者VS夏洛特黃蜂

NBA 2019-12-16 星期一 6:00

場地：銀行家生活球館

印第安納步行者情報信息

　步行者目前已經掉出東部四強之外，以17勝9負的戰線排名東部第六，球隊的目標肯定是希望能重返四強，據消息奧拉迪波目前已經基本康復，前段時間已經在發展聯盟尋找狀態，以目前步行者的陣容，如再能加上奧迪滿血歸來，球隊還是有很強的競爭力，目在唯一的擔心就是奧拉迪波復出能打出什麼樣的表現，畢竟已經一年沒上場比賽了，要恢復到以前的水準還是有一段很長的路要走。

夏洛特黃蜂情報信息

黃蜂目前以12勝16負的戰績排在東部第九名，依然保留著對季後賽的衝擊，上一輪在客場贏下公牛之後已經是取得了三連勝，但就在上場比賽這之後對於黃蜂有個不好的消息，球隊今年的新秀PJ華盛頓因手指骨折預計要缺席接下來的五場比賽，華盛頓本賽季一出戰28場比賽，場均上場29.4分鐘能得到12.2分5.2籃板1.8助攻，是球隊穩定的首發前鋒，他的缺陣對球隊衝擊季後賽的排位非常不利。

綜合分析

上賽季對陣黃蜂也是2勝1負，而本賽季的一場交鋒當中，步行者在前3節領先14分的情況下，被黃蜂在第4節打出了一個35：21，拖入加時，最終被黃蜂2分險勝！值得一提的是，黃蜂本賽季多次上演這種逆轉好戲，所以步行者想要擊敗黃蜂，以防對手雙殺，還是需要提防下半場崩盤。

比賽推薦：【夏洛特黃蜂+11.5】
