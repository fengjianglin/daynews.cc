瘋狂一夜！一天5場比賽出現10+分差逆轉，雷霆創20年歷史紀錄

2019-12-19 475條評論 2,921次閱讀 1119人點贊

在觀看NBA比賽時，試問誰不喜歡在落後兩位數的情況下，來上一波驚天翻盤的進攻狂潮呢？12月19日，NBA迎來了一個瘋狂的比賽日，一共9場比賽，誕生了5次翻盤逆轉，讓球迷們如同坐過山車一般高潮迭起。

1、雷霆VS灰熊：24分逆轉

西部第7的雷霆，本應輕鬆搞定聯盟倒數的灰熊。但灰熊從首節就佔據了上風，第三節更是打出了一波12-0的攻勢，將分差拉開到24分。這對於一般的球隊來說，就是進入垃圾時間，等待比賽結束的哨聲響起。

不過，這可是雷霆。這些年他們經歷了無數的翻盤和被翻盤，就在上一場比賽剛玩了一把20+分的大逆轉，心理素質非平常隊能比的。經過施羅德和保羅的先後出手，一節多的時間，雷霆就抹平了分差，還倒贏了4分，以126-122取勝。上一場比賽，雷霆逆轉公牛26分，連續2場24+的神奇逆轉，他們創造了過去20年的歷史紀錄。

2、公牛VS奇才：18分逆轉，加時取勝

奇才在前三節穩紮穩打逐漸拉開比分，第四節最多時曾領先公牛18分。對於當時的奇才來說，勝利似乎只是時間問題了。沒想到全場進攻不太順的公牛，忽然打出9-0的攻擊波，隨後不斷蠶食分差，隨著拉文罰球命中，比賽竟然拖入了加時。在加時賽，比賽的勢頭傾向了公牛，隨著溫德爾-卡特關鍵時刻拿下4分，公牛以1分之差險勝奇才。

3、熱火VS76人：落後12分不算事

熱火和76人是東部的佼佼者，雙方戰績極為接近，這一場比賽足以讓他們互換排名。因此，76人的主力們從首節就打出了威懾力，很快就領先了熱火12分。但76人主力的瘋狂努力，很快就被替補敗光了。第二節76人替補啞火，巴特勒和納恩率隊打了一波25-5。半場過後76人反而落後了8分。此後76人雖然多次反撲，但頑強的熱火頂住壓力，多次在76人看到追評的希望時命中得分，最終熱火以108-104笑到了最後。

4、掘金VS魔術：19分的分差只需幾分鐘抹平

魔術球員的身體素質排在聯盟前列，連丹佛高原的硬漢們也難以應對，魔術在本場曾一度領先19分。但在第三節魔術忽然斷電，而米爾薩普一個人連得14分，掘金瞬間就抹平了分差，第三節結束時，掘金反而領先了8分。此後掘金保持領先優勢，一直到最後，他們也以113-104輕鬆取勝。

5、凱爾特人VS獨行俠：最多落後10分，三人20+輕鬆翻盤

​失去了東契奇，獨行俠的球員配置立刻捉襟見肘。儘管塞斯-庫里在第二節爆發，打出了10分的領先優勢，但球員實力被碾壓，凱爾特人翻盤只是時間的問題。果然，綠軍的沃克、布朗、塔圖姆都砍下了20+，第三節結束後凱爾特人完成了反超，最終凱爾特人以109-103戰勝了獨行俠。
