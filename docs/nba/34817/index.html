十年前MVP票選前四的球星近況如何？票選第四已退役，MVP跌落神壇

2019-12-24 3,081條評論 4,346次閱讀 1757人點贊

10年的時間對於NBA聯賽來說是一個漫長的過程，10年之前的10-11賽季，庫里的場均得分還沒有破20分，湯普森和格林還沒有進入NBA聯賽。而在這10年間，庫里連莊MVP，勇士王朝建立，連續五年殺進總決賽，拿下3個總冠軍，之後又因為密集的傷病轟然倒塌，勇士再次淪為魚腩球隊。10年間，NBA聯賽的風格也發生了很大的變化，三分投射越來越多地被強調，攻防轉換成為球隊取分的最重要環節，曾經鐵血的防守也消失在這個籃球的舞台之上，那麼10年前叱吒NBA賽場的頂尖球星現在過得怎麼樣呢？本篇文章筆者就為大家介紹10-11賽季MVP票選前四的球星現狀。

MVP得主：德里克-羅斯

10-11賽季，羅斯出戰了81場常規賽，場均貢獻25分4.1籃板7.7助攻，帶領公牛隊拿下62勝20負的全聯盟最好戰績，他不僅用實力征服了挑剔的芝加哥球迷，也贏得了美國媒體和電視評論員的青睞，加冕歷史最年輕的MVP。就當所有人認為羅斯會向更高的榮譽發起衝擊時，他的進步在最高潮處戛然而止，傷病不但煎熬著羅斯的身體，還把他拉下神壇，公牛、尼克斯和騎士無情地把羅斯拋棄。

但是，玫瑰總有綻放的時候，羅斯在森林狼隊完成轉型，第六人也許是現在更適合他的角色，本賽季羅斯在活塞隊大部分以替補的身份出戰比賽，場均能拿下16.5分，是聯盟最高效的替補得分手之一。

MVP票選第二：德懷特-霍華德

10-11賽季，霍華德出戰了78場常規賽，場均貢獻22.9分14.1籃板2.4蓋帽，場均得分創下生涯最高，連續三年攬下最佳防守員，入選雙一陣，是那個賽季當之無愧的聯盟第一中鋒，霍華德能夠單換詹姆斯的傳聞也是在那個時期逐漸傳開的。但是霍華德過於樂天的性格帶來的負面影響逐漸顯現，沒有教練和隊友喜歡一個長不大的孩子成為他們的球隊領袖，霍華德因此和多個老東家鬧翻。

直到這個賽季，霍華德才意識到自己已經是34歲的老將了，如果再不端正自己的態度，籃球生涯可能就要結束了。本賽季霍華德痛定思痛，主動和湖人隊簽訂一份無保障的底薪合同，專註於防守和籃板，雖然場均數據只有6.8分6.9籃板，但是他的職業態度比以往的任何時候都要端正。

MVP票選第三：勒布朗-詹姆斯

10-11賽季，詹姆斯出戰了79場常規賽，場均貢獻26.7分7.5籃板7助攻，帶領熱火隊取得了58勝24負的東部第二好戰績。可能是以全美直播這樣的高調方式宣布離開騎士隊太過於難以接受，美國媒體和電視評論員不願意將選票投給詹姆斯，詹姆斯在MVP票選中僅僅屈居第三。

10年的時間過去了，詹姆斯彷彿有免疫衰老的魔力，35歲的他仍然是NBA的代表球員，在最新的MVP評選榜單中僅次於字母哥和東契奇，排名第三。

MVP票選第四：科比-布萊恩特

10-11賽季，科比打滿82場常規賽，場均貢獻25.3分5.1籃板4.1助攻，帶領湖人隊取得了57勝25負的戰績，排名西部第二位。作為湖人隊隊史最偉大的球星，科比早在2016年就告別了NBA舞台，退役戰爆砍60分也成為了NBA最難打破的記錄之一。湖人隊為了表彰科比對球隊的貢獻，為他退役了曾經穿過的8號和24號兩個號碼，相信不久的未來，科比的雕像也會豎立在斯台普斯球館之外。

退役之後的科比並沒有完全離開籃球，他開辦了曼巴學院，倫納德、喬治、歐文等眾多球星都是科比的門徒。
