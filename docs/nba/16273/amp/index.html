1.8億先生強勢復甦 31+5助球隊拿下5連勝 要想挑戰雄鹿少不了他

天天要聞
2019-12-14

北京時間12月14日，一場結束的NBA常規賽中，76人主場116-109力克來訪的鵜鶘，拿到了近期的5連勝。此役，76人前鋒托比亞斯-哈里斯表現出色，全場20投12中，得到了31分5籃板2助攻，是球隊取勝的重要功臣。

上賽季，志在季後賽大展宏圖的76人先後通過交易得到了巴特勒和哈里斯兩位全明星級別的鋒線球員。但搶七不敵猛龍失利，讓球隊今夏發生了不小的變化，在巴特勒遠走邁阿密，雷迪克加盟鵜鶘的情況下，球隊管理層選擇以5年1.8億的天價續約了27歲的哈里斯。

對於這樣的一紙續約，外界普遍並不看好，一位從來沒有入選過全明星、沒有進入過最佳陣容，沒有在季後賽證明過自己的球員，憑什麼拿到如此高昂的薪水？而在開賽之初，哈里斯低迷的投籃手感，也讓這些言論得到了「證實」。

但隨著賽季的深入，哈里斯已經逐漸找回了自己的狀態，過去的8場比賽中，他7次得分上20，本場更是高效的砍下了31分，76人在拿到了5連勝之後，排名已經升至了東部第二的位置，僅次於一路連勝的雄鹿。

本賽季東部的格局中，被外界普遍認為將是雄鹿和76人的「二人轉」，在在將近1/3賽程過後，確實是這兩隊名列前茅。而作為追趕者的76人，他們要想挑戰雄鹿，衝出東部晉級總決賽，甚至奪取最後的總冠軍，少不了哈里斯出色的表現。
