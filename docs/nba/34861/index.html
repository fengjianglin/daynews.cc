罕見的80-80，猛龍30分大逆轉之後，東西部呈現勢均力敵的局面

2019-12-24 825條評論 1,775次閱讀 178人點贊

在今天結束的比賽，猛龍主場完成30分大逆轉，以110-107戰勝獨行俠，對於猛龍而言是一場史詩級的逆轉，而對於本賽季常規賽而言，也有著不一樣的意義。因為憑藉這這場勝利，本賽季至今東部球隊和西部球隊的交手戰績，變成了80-80，東西部呈現勢均力敵的局面。

自從邁克爾喬丹退役之後，西強東弱一直是NBA的常態，自從1997-1998賽季以來，西部第八的勝率從來沒有低於五成，甚至還出現過48勝34負無法進入季後賽的情況，可以說競爭非常激烈。但本賽季因為開拓者和馬刺的低迷，加上勇士的隕落，西部開季之後一直被東部壓著打。

而如今開拓者已經有所復甦，西部也逐漸在縮小差距，在爵士最近2連勝東部球隊之後，西部80-79完成了反超，今天猛龍再一次扳平了比分。賽季進行了1/3，東西部80-80打成平手，這在20幾年來還是首次出現。

分球隊來看，雄鹿肯定是東部的大兵，它對陣西部球隊戰績是10勝2負，包括戰勝湖人、快船等強隊，輸掉的2場比賽分別是輸給爵士和獨行俠，而東部第三的熱火妥妥拖了後腿，對陣西部球隊就只有6勝6負，凱爾特人7勝3負，猛龍7勝4負，步行者7勝2負，76人是7勝5負，籃網是6勝6負，除此之外，就只有騎士3勝3負，勝率是達到五成的。

西部這邊沒有絕對優勢的隊伍，最好的是湖人和快船，對陣東部球隊都是8勝3負，湖人最近遭遇東部球隊的2連殺，再加上之前輸給猛龍，總共輸了3場，除此之外，掘金9勝4負，火箭8勝4負，獨行俠8勝6負，爵士6勝4負，開拓者5勝5負，太陽4勝4負，森林狼6勝3負，這些球隊勝率都至少達到5成，森林狼雖然遭遇10連敗，但對東部球隊戰績還不錯。

而雷霆3勝4負，國王4勝9負，馬刺4勝7負，灰熊4勝8負，鵜鶘1勝9負，勇士2勝7負勝率低於5成，從目前情況看，東西部的格局比較類似，都是前六比較強，從第七名開始都實力一般，而底部也都有球隊在擺爛，所以目前雙方差距不大。

明天爵士客場挑戰熱火，西部第六對陣東部第三，不知道哪一邊能再次取得領先。
