權威美媒評00-09年最佳陣容！科比詹皇入選一陣，姚明獲榮譽提名

2019-12-18 2,874條評論 4,714次閱讀 3014人點贊

北京時間12月18日，權威美媒Bleacher Report（下稱B/R）評選了2000-09年這10個賽季的聯盟最佳陣容。入選最佳陣容一陣和二陣的各有五位球員。另有五位球員獲得榮譽提名，中國球員姚明名列其中。

根據B/R的評選，在那個10年區間里，獲得榮譽提名的是卡特、麥蒂、韋德、姚明和大本。入選最佳陣容二陣的球員包括納什、艾弗森、皮爾斯、諾維茨基和加內特。入選最佳陣容一陣的球員包括基德、科比、詹姆斯、鄧肯和奧尼爾。

B/R點評姚明時，把他稱為禁區內「7尺6寸，體重310磅的作弊碼」。他的塊頭已經極具優勢，更可怕的是他還有如此精湛的球技。他的腳步很出色，雙手都可以得分，中距離投籃很准。遺憾的是傷病縮短了姚明的生涯，前三個賽季只缺席兩場的他，在這10年的後4個賽季只有一次出場超過57場。他健康時的數據，以及國際影響力，讓他應該得到提名。

再來看看被評入一陣的這五位巨星——

以這10年為範疇，沒有人送出比基德更多的助攻。他的傳球總是極富想像力，前隊友理查德-傑弗森表示基德總有那麼幾個傳球是你從來沒見到過的。在那10年里，基德是唯一一個累計至少10000分7000助攻5000籃板的球員。

科比是一個瘋狂的得分手，2005-06賽季，他場均得到35.4分，接近1986-87賽季的喬丹（場均37.1分）。在這個10年區間里，他是聯盟的得分王，比第二名的艾弗森多出近2000分。他的鬥志，更是和喬丹相似，詹姆斯曾告訴記者，他之所以那麼努力，因為他知道科比一直在努力，當他想要偷個懶時，他總會想到科比，然後就會馬上回到訓練場。

詹姆斯2003年進入聯盟，並沒有完整經歷那10年，但這依然無法阻止他成為一陣的成員。詹姆斯的天賦毋庸置疑，比盧普斯曾表示活塞曾用盡所有的方法，可就是無法阻擋詹姆斯，要知道當時的活塞是聯盟防守最好的球隊。詹姆斯的數據正負值8.5，是這個10年區間里排名最高的球員。

能把諾維茨基和加內特擠到二陣的球員，只有蒂姆-鄧肯。鄧肯球衣退役時，波波深情地對他說：「謝謝你讓我執教你，蒂米。」鄧肯的球風樸實無華，卻極其實用。他的打板投射堪稱絕學，假動作、腳步和傳球技巧在大前鋒中也堪稱頂級。他是馬刺恆強20年的關鍵人物。

奧尼爾是一陣的中鋒，通常情況下，沒有誰能限制他在內線的威力。納茲-穆罕默德曾說過對上奧尼爾，他唯一的目標就是別讓奧尼爾扣籃，但通常情況下這是辦不到的。攻防兩端，他都是巨無霸，在中鋒時代，他的統治力是難以匹敵的。
