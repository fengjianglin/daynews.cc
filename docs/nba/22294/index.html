35分4搶斷+末節19分，綠軍放走歐文沒啥，送走這顆小炸彈才是真虧

2019-12-20 28條評論 1,429次閱讀 299人點贊

北京時間12月19日，NBA常規賽繼續進行，今日黃蜂客場挑戰騎士，最終以98-100不敵對手，不過球隊核心後衛羅齊爾表現出色，他全場比賽上場37分鐘，27投12中砍下生涯新高的35分3籃板2助攻4搶斷，並且在第四節獨得19分，雖然最終投丟了絕殺球，但他可怕的得分爆炸力也讓人印象深刻。

今年夏天，凱爾特人幾乎經歷了一波大換血，其中最讓人難忘的，就是他們失去了全明星控衛凱里·歐文，不少人因此為凱爾特人惋惜，如果能夠留下歐文，或許他們本賽季會是比雄鹿更強的球隊。

但是實際上，歐文在凱爾特人一直無法真正融進球隊，在場上他的打法和球隊戰術格格不入，在場下他怪癖的性格也沒能讓隊內的年輕人心服口服，因此他的離開其實對凱爾特人來說損失不大，特別是後來填補歐文空缺的沃克，本賽季打得非常出色，場均23.5分4.3籃板5.2助攻，並且他的堅毅低調的性格完美契合這支凱爾特人，如今已經成為綠衫軍真正的領袖。

不過凱爾特人球迷們在感嘆沃克真香的時候，還是會有些遺憾，不是因為他們失去歐文，而是他們失去了羅齊爾，今年夏天，凱爾特人和黃蜂打成了先簽後換的交易，綠軍送出羅齊爾，黃蜂送出沃克，而兩人本賽季都打出了出色的成績。

羅齊爾本賽季場均17分4.7籃板4.3助攻，並且他才25歲，比沃克年輕4歲，合同上，沃克的合同為4年1.41億美元，羅齊爾的合同為3年5800萬美元。可以說其實從性價比上，羅齊爾是更好的選擇。

當然我們不否認沃克是更好的球員，也更適合現在的凱爾特人，但是如果夏天凱爾特人再仔細運轉一番的話，或許他們有辦法同時得到沃克和羅齊爾，而羅齊爾在將來可能會成為全明星球員，雖然夏天羅齊爾離開部分願意是因為想要首發，但是以安吉的能力，想要安撫羅齊爾也並非辦不到吧。
