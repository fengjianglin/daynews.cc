2020年NBA名人堂，科比等三尊大神的入選推薦人你知道嗎？

2019-12-20 2,287條評論 4,495次閱讀 916人點贊

北京時間12月20日，NBA奈史密斯籃球名人堂官方宣布了2020年名人堂球員候選人名單，科比、加內特、鄧肯等球星入圍。

具體的入圍名單如下：

男性球員：科比、鄧肯、加內特、波什、邁克爾-芬利和肖恩-馬里昂。

女性球員：塔米卡-凱金斯和斯文-卡什。

拋開別的先不提，科比、鄧肯、加內特三人鐵定能夠進入名人堂。時間的確過得非常快，2016年科比退役戰轟下60分彷彿就在昨天，之後我們也經歷了科比球衣退役等等。現在三人終於快要進入名人堂，作為曾經80、90後那些看著他們從新秀打到退役的球迷，心中或許再一次澎湃起來。

我們不禁又要浮想聯翩，為他們「瞎操心」：進名人堂他們三人的最佳推薦人呢？

首先是科比：

20年一人一城，8號和24號各十年，紫金軍團從來都不缺乏巨星，他是自魔術師之後再次讓湖人復興的功臣，魔術師和科比雙方都表示過對方是湖人史上最強之人；從小是喬丹的小迷弟，也通過自己的努力得到喬丹九成以上的真傳，喬丹多次在媒體前表示超喜歡科比。如果非要找推薦人，我想最好是魔術師和喬丹兩人同時推薦，應該可以傳為一段佳話。

其次是鄧肯：

19年一人一城，15屆全明星、15屆最佳防守陣容、5個總冠軍，都是在馬刺拿到的，而且和波波維奇亦師亦友，而且比較有意思的是，鄧肯入選，波波維奇也會入選。鄧肯如果要推薦人，十之八九是羅賓遜，這兩位雙搭的傳奇歷歷在目，曾共同率領團隊兩次奪冠，之後羅賓遜更是以無私的精神將領袖角色交接給了鄧肯。

最後是加內特：

狼王雖然沒有前面兩位那麼誇張的職業生涯，可是年輕的狼王無所不能，遊走於五個位置，完全有足夠的資格進入名人堂。他的推薦人則很有可能是麥克海爾。麥克海爾是加內特籃球職業生涯的領路人，成就了加內特後來的傳奇。雖然兩人有過矛盾，一度斷交，但如今兩人已經冰釋前嫌，如果麥克海爾能夠推薦，我想，掌聲和歡呼都會賺足。

其實客觀來說，自從老一代球星一個個退役，很多人對於NBA的熱情在衰減，加上現在NBA風格大變，雖然球迷一代又一代不間斷，可是似乎還是失去了擠著看報紙、圍著看電視的激動和感覺。現在很多人之所以還在堅持看，可能是因為詹姆斯等還在，或許不久之後，大家都會淡出NBA，既然在這之前，能夠見證或許是史上最強的一屆名人堂，顯然是一件幸事。

你心中三位各自的推薦人又是誰呢？
