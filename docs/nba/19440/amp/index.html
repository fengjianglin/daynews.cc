都說他坑了綠軍，杰倫卻主動為歐文洗白：鍋是教練和全隊的

天天要聞
2019-12-17

波士頓凱爾特人和凱里-歐文已經分道揚鑣，但杰倫-布朗並沒有忘記那些短暫而不甚愉快的日子。

2018-19賽季的凱爾特人被認為是NBA總決賽的有力競爭者。他們被認為是東部聯盟的強隊，東部版的金州勇士。然而，由於種種原因，他們整個賽季都起伏不定，最終僅僅以第四名的身份闖入季後賽。

儘管他們在第一輪就橫掃了一支沒有維克多-奧拉迪波的印第安納步行者隊，但在與密爾沃基雄鹿隊的東部半決賽中，他們卻慘遭吊打，被新科MVP字母哥領銜的雄鹿輕鬆淘汰。

此後，歐文受到了鋪天蓋地的指責和質疑。因為在沒有他時，凱爾特人成功插入東部決賽，並和勒布朗-詹姆斯率領的克利夫蘭騎士大戰7場才遺憾落敗。

而在有他的情況下，卻如此輕易地在次輪被淘汰，前後如此巨大的反差對比，讓他的領導能力受到了巨大的質疑，他甚至捲入了與凱爾特人新星杰倫-布朗的一場關於為什麼球隊在賽季中期如此艱難的爭論。雖然當時布朗和歐文還遠未達成共識，但事實遠非如此。

但在最近一次的採訪中，布朗實際上把責任從歐文身上轉移到了凱爾特人的管理層，教練和諸多球員身上。

「每個人都會有自己的看法。我不認為凱里（歐文）應該在乎，也許他在乎，也許他不該像他那樣在乎。他就是歐文，沒有人是完美的。」

「凱里（歐文）受到了很多指責，他不應該受到這樣的指責。有些人不能後退一步，這不是他的錯。這不是他的錯。這是球隊和教練的錯。他受到很多指責，因為他是超級明星。但這些責任都應該在球隊和教練組身上。事情都過去了，現在凱里（歐文）在布魯克林的一個更好的地方，他的根在那裡。他會沒事的。」

在歐文離開之後，凱爾特人在沒有他的情況下取得了進步，並在賽季初期以17勝7負(70.7%勝率)的戰績讓所有人刮目相看。而布魯克林籃網也以14勝12負的成績位列東部第七——當然，這裡面大部分的比賽，歐文都因傷缺席，這讓他略微有些尷尬。

話雖如此，有一件事是肯定的——不是所有的責任都應該由歐文來承擔，因為籃球比賽，並不是一個人的事。
