一個倒數第一，一個全隊第二！森林狼七連敗需要馮萊而非迪昂

2019-12-18 46條評論 1,067次閱讀 51人點贊

明尼蘇達森林狼在今年休賽期用為期一年的合同引進了兩名內線球員——喬丹-貝爾和諾阿-馮萊，再加上原有的戈爾吉-迪昂，這就導致了森林狼的內線面臨非常激烈的競爭。但迪昂和貝爾由於各種原因很難得到固定的出場時間，而馮萊則是他們中境況最好的一個人，每場比賽都能得到15分鐘的上場時間，幫助唐斯緩解內線的壓力。

但是唐斯停賽兩場給了迪昂機會，而貝爾雖然在10月和11月沒有太多的上場時間，在過去的5場比賽中，他也能得到相當穩定的上場時間。不過迪昂和貝爾上場時間的增加是以犧牲諾阿-馮雷為代價的，在過去的12場比賽中，他有9場比賽只在垃圾時間出場。不過在這段時間裡，森林狼僅僅取得了3勝9負的戰績，在本賽季開局強勢的情況下退出了季後賽圈。

雖然小編也認為森林狼主教練桑德斯將貝爾的輪換次序安排在馮萊前面是合理的，但是將迪昂也排在馮萊前面就很難理解了。儘管在11月份的比賽中，迪昂確實打出了一些不錯的表現，但是他的凈評級是全隊常規賽輪換球員中最低的，只有凱蘭-馬丁、喬丹-麥克勞林和杰倫-諾威爾比他低（抱歉，這些球員是誰小編都不知道），相比之下馮萊本賽季的凈評級為+0.5，排名全隊第二，也是除了傑克-萊曼之外全隊唯一凈評級為正數的球員。

森林狼的防守體系要求經常換防，而馮萊的機動性使他更適合桑德斯的體系，他跑動能力顯然比不能再跑快攻的迪昂強得多，主教練桑德斯非常強調防守端的換防和進攻端的提速，但迪昂的技術並不適合這兩種風格。森林狼需要一些不同的東西來打破目前七連敗的尷尬局面，把迪昂的出場時間給馮萊可能會起到意想不到的效果。
