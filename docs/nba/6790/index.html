杰倫·布朗：頭部受傷都會使很嚴重的 沃克並不需要著急回來

2019-11-25 0條評論 14次閱讀 0人點贊

近日，凱爾特人主教練布拉德·史蒂文斯接受了媒體的採訪，談到了球隊後衛肯巴·沃克的受傷情況。

史蒂文斯表示：”我當時的第一反應就是，他還能動嗎？我在巴特勒大學的最後一年，有一個孩子頭部撞在了柱子上，出現了暫時失去行動能力的情況。那是我經歷過的最可怕的事故，所以這是我最先想到的事情。後來我看到他動了一下，但是很明顯他被搞得一團糟。”

隨後，史蒂文斯補充道：”但是這很嚇人，頭部受傷、頸部受傷都是很嚇人的。說實話，這就是他周二那天出戰成疑的原因。我覺得他可以帶傷出場，但我們還是想確認一下他受傷的情況，因為頭部受傷是很敏感的。”

沃克的隊友杰倫·布朗也接受了記者的採訪，他表達了自己對於肯巴·沃克受傷的看法。

布朗說道：”我們很高興事情沒有像預想的那樣嚴重，我們以前在聯賽中也看到過這樣的情況，我以前也有過一次類似的狀況。所以他沒有太嚴重的傷病是件好事。因為任何時候，只要有人躺在擔架上離開，情況一般就很糟糕了，你不想看到這種情況。當時我正背對著他們，我不知道沃克經歷了什麼。但我能看到謝米·奧傑萊的眼睛，我能看到他的擔憂。所以那時我就知道情況可能會很嚴重。任何時候，像這樣的事情–背部，頭部受傷顯然是很嚴重的。”

談及沃克何時回歸球隊，布朗最後說道：”他不急著回來，當然，如果他感覺很好，他可以隨時回來。但有時像這樣的事情你需要喘口氣，好好休息一下，直到完全準備好再回去。”

本賽季至今，沃克場均出場32.3分鐘，得到21.1分4.5籃板4.7助攻。
