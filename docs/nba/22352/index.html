家境貧寒逆境中成長，力壓科比當選狀元，這是艾弗森的傳奇故事

2019-12-19 1,454條評論 2,723次閱讀 822人點贊

我們都知道，要說聯盟哪個球星是矮個子傳奇的話，艾弗森，費城的答案可以說是當之無愧。職業生涯期間，艾弗森完成了堪稱偉大的成就。不僅奪得過MVP，並且還在總決賽跟巔峰OK較量過，可以說，雖然生涯沒有冠軍，但仍然是NBA最著名的孤膽英雄，雖敗猶榮的大將軍，也是無數球迷心中的偶像！那麼，答案艾弗森成長的故事究竟是怎樣的？讓過人君帶你一睹為快！

家境貧寒逆境中成長

很多人不知道艾弗森的名字是怎樣來的，其實，艾弗森是他母親的姓氏，因為他的父親在艾弗森小時候就拋棄了他們母子倆，所以艾弗森實際上是在單親家庭長大的，而阿倫正是他父親的名字。小時候，艾弗森家裡很貧窮，他的媽媽要打很多份工，才能勉強維持生計。但是這個堅強的母親，從來都沒有讓艾弗森挨餓，一直都堅強地面對生活，獨自撫養艾弗森，這也給艾弗森留下了深刻印象，並且非常感恩母親。

知恩圖報送鞋給繼父

很多人不知道艾弗森究竟是怎樣走上籃球這條道路的，其實，這多虧了他的繼父。在他的母親找到新男友之後，他教育艾弗森如何打籃球。並且，他看出了艾弗森的天賦，對他十分嚴格，這才使得艾弗森在球場上嶄露頭角。後面，他的繼父不幸入獄，生活十分窘迫，艾弗森去看望他，發現他沒有一雙鞋子，就將自己的鞋子送給了他，自己最後走回家了。這樣知恩圖報的人，值得我們尊敬。

偏愛橄欖球優秀過人

我們都知道，艾弗森最後進入了籃球領域的名人堂。然而，其實在上學時期，艾弗森最愛的運動是橄欖球，因為這充滿了激情和對抗，並且，當艾弗森衝鋒的時候，沒有人能跟上他的速度。但是，他的母親看好他進入籃球領域，但艾弗森覺得籃球太娘娘腔了，一點對抗都沒有，直到他發現籃球不像橄欖球，可以在球場上無限單挑對手證明實力，才愛上了這項運動。要知道，就算是橄欖球，他也收到過獎學金，絕對優秀過人。

打球獨被抨擊實力證明

我們都知道，艾弗森經常被人們批評打球太過獨。實際上，這不是他的錯，而是他的風格就是如此。跟馬布里不同，艾弗森的風格就是自己戰鬥，因為隊友往往都沒有他的實力，對他來說：單挑就是最有效率的方法，就算批評他，他也依然頑強屹立。在外界的質疑下，他用場均得31.6分、搶8.7個籃板還有9.2次助攻的表現，打了所有人的臉。沒辦法，這就是實力的絕對壓制！

謙遜態度打動76人

我們都知道，在艾弗森之前，他的大學喬治城是沒有一位球員在讀完大學之前參加選秀大會的，然而，艾弗森急於改善自己的家庭，並且他妹妹麗薩的病情也日益加重，因此他十分渴望得到合同來賺錢。了解了他的情況，大學主教練最終決定放他去追夢。而76人的管理層一開始也很猶豫，他們需要控衛，但不知道選他還是馬布里好，這個時候，艾弗森在試訓時表現出來的溫和和謙遜打動了他們，要知道，他們曾經試訓過科比，但是科比一直擺弄著自己昂貴的太陽鏡，態度比較高傲，最終讓76人放棄了這名新秀。

驚人好勝心嚇倒裁判

我們都知道，艾弗森可以說是聯盟最有鬥志的一名球員。從他在球場上的表現你都可以看出，絕對不輕言放棄，哪怕最後一球都分秒必爭，面對奧尼爾這樣強壯的對手都不會退縮。在聯盟里的時候，有一次比賽，裁判不停地錯判，激怒了這位球星，他直接走向了技術台的裁判，大聲呵斥著這個無理的人，沒想到裁判準備吹技術犯規將他直接驅逐，這時候，裁判只看見艾弗森死死盯住他，像一頭餓狼一樣，要將他撕碎一般。被嚇呆的裁判也是不敢再針對這位MVP，這就是艾弗森的好勝心！

新秀獎缺席陪伴友人

前面的故事我們說到了，沒有艾弗森偉大的母親，就沒有艾弗森的輝煌。因此，艾弗森對自己的母親也是十分感恩。同樣地，艾弗森之所以能成長起來，也靠著身邊兄弟朋友們的支持。因此，在聯盟準備給艾弗森頒發最佳新秀獎盃的時候，76人卻發現他們的球員突然不見了。打電話一問，人們才知道艾弗森在陪自己的兄弟。我們都說，苟富貴勿相忘。實際上做到的人，又有多少呢？而在拿到這個獎項之後，艾弗森表示，這個獎是他母親的，沒有母親的栽培，就沒有他的今天。這樣的孝心，讓人感動不已。
