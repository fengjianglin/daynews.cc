三投兩中的全明星“清新脫俗”，不甘平庸的教父是否“蠢蠢欲動”

2019-12-22 2,124條評論 3,149次閱讀 1528人點贊

今天熱火輕鬆擊敗弱旅尼克斯，繼續穩居東部第三。打開某app發現熱火當家球星巴特勒僅9分入賬卻獲得了9.3分的高評分。我開始猜測，他應該是投中了致勝球或者絕殺吧，但我立馬又反應過來了，這場球早就花了，最後熱火是大勝。找不到答案的我繼續往下滑動手機屏幕。

滑到數據統計欄，看到巴特勒出場28分鐘也是正常，眼神又往右邊橫移了一下我大吃一驚，這場比賽他竟然3投2中，真是視數據如糞土啊。但是大家別以為他拿著3000多萬不幹活哈，他的正負值是全場最高的+29。其實這就是巴特勒的比賽方式，比如他在賽季初打老鷹的第二場比賽中，10投2中，僅得5分，這要是換做其他球星那肯定是坑了，但巴特勒不會啊，因為他還有9籃板11助攻6搶斷3封蓋入賬，正負值是全場最高的+21。在這個哈登威少動輒出手30到40次的年代，巴特勒真是個全明星中的「異類」。

2011年選秀大會上巴特勒在首輪第30順位被選中，菜鳥賽季僅能交出場均2.6分1.3板的數據，之後他立足於防守的同時不斷地雕琢自己的進攻技巧，才成長為今天這位攻防一體的全明星級球員。從悲慘的童年到進入NBA，從場均2分的菜鳥到進步最快球員再到奧運冠軍、4屆全明星、入選最佳陣容和最佳防守陣容，所付出的努力和歷經的艱辛只有他自己知道，所以他要比別人更加珍惜來之不易的當下。他的憂患意識讓他總是成為球隊里最刻苦的人，他的競爭性讓他始終報有更高的要求。30歲的巴特勒，一位如此優質的領袖，他會帶領熱火在東部攪局嗎？

萊利從1995年接手熱火，無論是作為主教練、總經理還是球隊老闆，二十多年來熱火無緣季後賽的情況屈指可數，從莫寧到韋德再到熱火三巨頭，這位領導湖人show time時代的偉大教頭，給熱火隊史平添三座總冠軍，熱火教父從來不允許自己的球隊平庸太久，這一次巴特勒會是那位真命天子嗎？雖然今年熱火呈現出「黑馬「的姿態，但從陣容上看，未來幾年東部差不多就是雄鹿、凱爾特人和76人的三家之爭，熱火如何才能成為攪局的那支球隊呢？

從歷史上看，強隊標配往往是超級明星+防守悍將（惡人級），比如，喬丹身邊的羅德曼，科比身邊的阿泰，庫里身邊有格林。從小球時代來看，強隊標配還要加上優質射手最好是優質3D球員，比如，16年騎士隊的JR，上賽季猛龍隊的丹尼格林，還有最頂配的3D球員克萊湯普森。「惡人」去鎖對方頭牌，射手給超級明星拉開空間，這兩類球員也是除超級明星外NBA最不愁工作的人啦。

可惜的是，兩年之內自由市場上沒什麼「大魚」也沒什麼「惡人」，如果交易的話，從目前來看大家都過得很好也沒什麼可交易的機會。也就是說熱火在巴特勒這兩年巔峰里很難打造出一支超級強隊，但有機會成為一支勁旅，下面我們來看看這支球隊在未來兩年里有沒有和其他強隊叫板的實力。

熱火陣容分析

外線持球手：巴特勒、德拉季奇、納恩

巴特勒，球隊核心。本賽季場均20.4分6.4板6.8助2.1斷，後三項數據全部為生涯最高。在進攻端他一半以上的出手是在離籃筐10英尺以里的範圍內，他本就是一個喜歡衝擊籃筐的球員。雖然今年命中率略低，但他仍然是熱火在關鍵時候最可靠的進攻點。在防守端他從不惜力，防守DIFF%高達-5.2（聯盟球員在面對巴特勒的防守時命中率平均下降5.2個百分點），尤其是要經常在外線主防對面的明星球員。進攻防守一把抓，絕對的球隊領袖！

德拉季奇，斯洛維尼亞國手，當年來中國與男籃打熱身賽還名不見經傳，一晃十幾年了，已經是位經驗豐富的老將了。本賽季場均命中2.3記三分，命中率41%，而且自己製造空間出手三分要比接球就投的比例高一點，而且命中率還略高達到了41.7%。也就是說他無論是持球攻還是接隊友傳球投籃效率都不錯，是熱火的一個進攻強點。

納恩，今年名聲大噪的一位落選秀，目前為止場均貢獻16.4分3.6助攻，命中率45.3%。擅長持球攻，無論是三分球還是兩分球，持球攻投籃效率都要強於接隊友助攻後的投籃。

外線射手：泰勒、鄧肯羅賓遜

泰勒希羅，19歲，今年的13順位，是個後衛。他場均扔進2個三分，命中率達到37.8%。他三分接球就投命中率高達46.4%，把握空位機會的能力很強，但是靠自己製造空間出手的三分只有25%的命中率，這方面能力還有待提高。13順位都不是「孬種」，這是以一位未來可期的年輕人！

鄧肯羅賓遜，這位球員不是中鋒，不是大前鋒，更不是狀元，總之跟馬刺沒有關係，他是熱火在落選秀中淘到的一位優質外線球員。本賽季場均命中3.2記三分，命中率高達45%,可以說是本賽季聯盟里最好的三分射手之一。在他的所有投籃出手中73.7%的佔比是接球就投的三分球，命中率達到45%以上，所以這是一位像科沃爾、丹尼格林那樣的純投手。

藍領球員：溫斯洛、小瓊斯

溫斯洛，目前處於傷停狀態，但從本賽季出場的10場比賽看他在防守中做的非常不錯。防守差值DIFF%達到了-5.6%（聯盟球員在面對溫斯洛的防守時平均命中率下降5.6個百分點）。可以作為熱火外線防守尖兵，擔負防守重任。

小德里克瓊斯，又是一位落選秀，最近得到了重用，打的特別積極，記得之前打灰熊的比賽中在末節關鍵時候十幾秒里完成兩次搶斷且幫球隊打成反擊，最近的幾場比賽他獲得了很多機會也越打越好。他在在本賽季能讓對位者的命中率平均下降9.7個百分點，三分線分對對手的壓迫性更大，能讓對手平均命中率下降17.3個百分點。

內線球員：奧利尼克、邁耶斯倫納德、阿德巴約

奧利尼克，作為身高超過210cm的大個球員，他場均能以41.1%的命中率扔進1.5個三分，能為球隊進攻創造更好的進攻空間，這是他最大的特點。但是防守端做的還是遠遠不夠的。

邁耶斯倫納德，同樣是位身高超過210cm的高射炮，他的三分命中率達到47.4%，但他生涯場登場時間一直在15分鐘左右，就是一位普通的輪換球員。

阿德巴約，22歲，身高206cm，臂展達到220cm，目前他正越打越好成為這支熱火隊內線最好的球員。本賽季場均15.6分10.6板4.6助攻1.45搶斷1.24封蓋，非常全面，尤其是拿了幾場三雙後展現了他在組織方面的優秀。他不但在內線可以護筐，防到中遠距離和三分線外他照樣可以限制對手的命中率，可以讓對手的平均命中率下降4到5個百分點。

陣容總結

可以看到，熱火無論是外線球員還是內線大個子大多都有著一手出色的三分球，這也讓他們以38.5%的球隊三分命中率高居全聯第一。巴特勒、德拉季奇再配上幾位出色投手和防守尖兵，其實熱火的外線配置是非常不錯的。反而是內線奧利尼克和倫納德他們飄著打的球風讓熱火內線顯得過於空虛，防守端的貢獻顯然不夠，一個阿德巴約還不足以抗衡擁有強力中鋒的球隊。

建議和預測

對於熱火來說，他們引進一位內線大閘是非常有利的，會讓他們在東部的競爭當中更有底氣。目前聯盟里只有一位比較合適的球員，就是活塞隊的「庄神」，他明年的合同是球員選項，如果本賽季不能與活塞達成續約協議，他很有可能跳出合同成為自由球員。屆時，熱火有機會簽下這位籃板怪獸。那樣的話，未來兩年熱火可能會成為東部的攪局者。
