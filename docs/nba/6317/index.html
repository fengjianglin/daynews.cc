湖人大敵或添重磅強援！西部高管做預測，馬刺1換2送阿德到爭冠隊

2019-12-03 0條評論 17次閱讀 0人點贊

NBA新賽季最讓人感到意外的球隊，馬刺隊肯定會在列，這支西部老牌勁旅「跌跌不休」，目前僅以7勝14負的戰績排在西部倒數第四。內憂外患的馬刺真的到了近20年來最危險的時刻，波波維奇治下的馬刺難道本賽季真的要無緣季後賽了嗎？

深陷泥潭的馬刺也被交易流言纏身，圍繞在德羅贊和阿爾德里奇身上的交易緋聞滿天飛。美媒HoopsHype探討了關於阿爾德里奇的交易價值，雖然馬刺將阿德下賽季的合同轉為全額保障，起初並沒有打算交易他，但馬刺也沒想到新賽季戰績會這麼差，不排除在交易截止日前送走他。

美媒採訪了多位NBA高管，其中一位西部高管認為阿爾德里奇被交易到擔丹佛掘金隊。這位西部高管向HoopsHype提到，丹佛可能是阿爾德里奇的目的地，用阿德交換馬利克-比斯利和普拉姆利是一種可行的交易方案。

同時，他表示，約基奇本賽季有些慢熱，掘金儘管戰績還不錯，但進攻端不穩定，還是存在一些問題，阿德的投射能力可以幫助約基奇減輕壓力。如果掘金認為這對內線二人組合可以作為現代NBA的首發前場，那麼這一交易可以讓米爾薩普擔任替補，那樣的話，他可以在第二陣容中給對手造成傷害。

目前，掘金以13勝4負的戰績排在西部第二，緊追湖人，是湖人在西部的強勁對手。掘金的場均失分和防守效率都高居聯盟第一位，是聯盟防守最強的球隊之一，但在進攻端，掘金並沒有打預期出的表現，場均只得到106.7分，排在聯盟第24，進攻效率也僅排在第18。

如果真如這位西部高管預測，阿德最終會前往丹佛，但約基奇+阿德+默雷的三人組看起來威力十足。從阿德目前的能力來看，他無法成為一支球隊的第一核心，但作為一個第二或第三核心，阿德還是一個相當不錯的選擇，作為一名有出色投射能力的空間型內線，他仍然有不小的利用價值。
