賽季命中率27%+25%，6項數據為0！這位28號秀堪稱擺爛神器

2019-12-02 0條評論 15次閱讀 0人點贊

早些時候，NBA當天的幾場比賽都差不多進行完了，而在已經完結的多場比賽當中，湖人以14分的分差輸給了獨行俠。

現在雖然還是聯盟第一，但勝負場卻是和雄鹿一樣的，兩支球隊並列第一，再不加把勁估計聯盟第一的位置都不保了。

這一場比賽也能說明一個問題，湖人的三分球是真的差，這一場比賽僅僅命中7記三分，而獨行俠那邊達到了17記！僅這一項數據就輸了30分！

馬刺以34分的分差輸給活塞也是沒想到，雖然馬刺是有擺爛的跡象， 但輸這麼多給活塞，真的有點說不過去了。

看來勇士想要拿狀元簽的話，馬刺的幾個阻礙之一，勇士在這一天也是有著一場比賽的，是在客場對陣魔術，最終勇士以96-100的比分輸掉了比賽。

現在勇士也是成功的壓過了尼克斯和老鷹，4勝17負再次獨坐聯盟墊底的位置，勇士這個賽季真的太難了，誰都能欺負他。

這一場比賽格林表現的也是一如既往，做為隊內唯一一個能上場的億元先生，完全起不到幫助球隊贏球的作用，這筆錢勇士真的給虧了。

而勇士在今年選擇的新秀當中，帕斯卡爾是不用說了，他已經打出了成績，而普爾就真的有點難受了，這場比賽普爾8中0僅的一分！

一球都沒打中也就算了，還是0籃板，0助攻，0搶斷，還不止，還有0失誤，0犯規，甚至連正負值都是0，6項數據為0！

要不是靠著罰球拿到一分，他就得成為隊內出場的球員當中，唯一一個沒得分的球員了，他也不僅僅只是這一場表現不好了。

本賽季的投籃和三分球命中率，分別為27%+25%！場均上場25分鐘都是在幹嘛？是在是太菜了，都沒有一點培養價值，勇士到底怎麼選的人，隨機選也不至於這麼差吧？

他也就能在現在的勇士獲得一點出場時間，去其他隊估計都得被裁掉或者下放了。

勇士這位28號秀真堪稱擺爛神器！有了他拖著後腿，勇士就多了幾分拿到狀元簽的希望，不得不說現在勇士確實需要這樣的「人才」。

勇士現在想要拿狀元簽的競爭對手有著尼克斯，老鷹，馬刺等等，沒有幾個拖後腿的球員，還真難在他們之中殺出來。

有意思的是，這位28號秀還曾經說命中率差並不是生死攸關的事，確實是這樣，感覺普爾命中率越差勇士這邊還越高興。

不然普爾這麼差的表現，為什麼還能得到這麼多的上場時間？雖然明面上一直說不滿意現在的狀況，但從勇士給普爾這麼多上場時間，以及贏了公牛之後科爾把戰術板都摔爛了。

可以看出來，感覺勇士現在是真的沒有在考慮季後賽啥的了，現在這樣也是最正確的抉擇，擺爛然後去拿個狀元簽，下個賽季在崛起，這個賽季的話，僅憑現在的陣容，在聯盟裡面根本掀不起一點波瀾！
