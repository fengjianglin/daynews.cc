NBA歷史十大小前鋒：安東尼第十，前三無人能撼動

2019-11-30 0條評論 18次閱讀 0人點贊

TOP10：卡梅隆-安東尼

安東尼職業生涯至今，得分已經超過了25000分，籃板數也超過了6000個，自進入聯盟以來，他一直保持著出色的競技狀態，更是有「進攻萬花筒」之稱。

TOP9：里克-巴里

如今這個時代的球迷應該都會對里克.巴里這個名字不太熟悉，但若是提到「端尿盆」這一經典的投籃動作，相信凡是會參與到籃球這項運動的人都十分熟悉，而巴里就是這個動作的代表球員。他是上世紀六十年代最出色的小前鋒，職業生涯總得分超過了25000分。

TOP8：保羅-皮爾斯

皮爾是是一名在場上「韌性十足」的球星，無論是詹姆斯還是科比，在場上遇到皮爾斯時都會感到無比的頭痛。而他還曾在2008年率領凱爾特人擊敗了湖人最終奪冠。

TOP7：埃爾金-貝勒

貝勒的職業生涯帶著幾分憂傷的色彩，他八次率隊打進總決賽但都鎩羽而歸。他職業生涯場均27.4分，13.5籃板，4.3助攻，並十次入選最佳陣容。

TOP6：多尼米克:-威爾金斯

儘管威爾金斯也沒有拿到過總冠軍獎盃，但這仍然難掩他的優秀，他被譽為是NBA歷史上最偉大的扣將之一，職業生涯場均24.8分，6.7籃板以及2.5助攻。

TOP5：朱利葉斯-歐文

J博士的大名可謂家喻戶曉，他曾三次奪得總冠軍，七次入選最佳陣容，職業生涯場均22分6.7籃板，3.9助攻。

TOP4：凱文-杜蘭特

杜蘭特的橫空出世一定程度上改變了聯盟的局勢，能夠將他招致麾下的球隊都可以擁有著衝擊總冠軍的實力。他已經四次拿到得分王，五次入選最佳一陣。

TOP3：斯科特-皮蓬

與喬丹生於同一時代的皮蓬就是當時聯盟中最好的小前鋒，他與喬丹並肩戰鬥，創立下當年的公牛王朝。他無論是在進攻端還是防守端都可以給予球隊極大的幫助。

TOP2：拉里-伯德

伯德是上世紀八十年代最出色的球員之一，他與魔術師約翰遜成為了那個時代的標誌性球員。他職業生涯拿下過三座總冠軍，兩次FMVP，三次MVP。

TOP1：勒布朗-詹姆斯

毫無疑問，詹姆斯目前已經是歷史第一小前鋒了，他的能力毋庸置疑，連續八年率隊打進總決賽就是最好的證明。他強悍的身體素質，頂級的大局觀，視野，還有一流的技術，讓他目前已經足以排進歷史最佳。
