太狂了！迎著戈貝爾就是一記暴力隔扣，熱火的落選秀都這麼不講理

2019-12-25 639條評論 3,122次閱讀 1469人點贊

本場比賽，東部最強黑馬熱火隊坐鎮主場，迎戰西部第六的爵士隊。爵士隊近段時間來可謂正是士氣大漲的時候。

本場比賽開始前，爵士已經取得了五連勝，此役志在豪取六連勝。而熱火，在本場比賽開始前已經取得2連勝，那麼熱火勢必要捍衛主場，奪得三連勝，向東部第二的位置發起衝鋒。

爵士與熱火兩隊就紙面實力來說，是不相上下的，甚至爵士更勝一籌。畢竟熱火的首發五人中還有兩名是落選秀。

在本場比賽中，出現了令人嘆為觀止的一幕，在距離上半場結束還有2分鐘左右的時候，熱火搶斷爵士，納恩中路持球推進。

隨即找準時機分球給小德里克-瓊斯，後者接到球後從左側底線突破發起進攻，藉助阿德巴約完美的擋拆掩護。

直殺籃下，迎著戈貝爾直接高高躍起就是一記雙手暴力炸框！這一幕，令主場所有球迷興奮不已。

全體球迷大聲歡呼起來，熱火替補席更是一個個的都做出了非常驚恐的表情，簡直難以置信，球場氣氛已然達到了一種巔峰。

德里克-瓊斯自己也是雙手抱頭，在場上狂奔，自己都不敢相信自己的這記暴力隔扣，簡直太不可思議了，當然這些都難以掩飾瓊斯自己內心的欣喜與激動。

看到這記扣籃，不禁又想起莫蘭特隔扣樂福那一球了，可惜那一球沒有扣進，扣進必是十佳球之首呀。

但是瓊斯本場比賽的這記隔扣，不說是十佳球之首，但是絕對是已經鎖定了本日十佳球、本周十佳球、乃至是本月十佳球。

狂啊，這年輕人真的是太狂了，要知道你隔扣的可是2屆最佳防守球員以及聯盟蓋帽王的法國高塔戈貝爾呀。話說這下戈貝爾也真是丟了大面子呀，堂堂最佳防守球員慘遭一位底薪落選秀恥辱隔扣。

這場比賽還真是戈貝爾的恥辱一戰呀，說出去臉面真的有點掛不住呀，真心為戈貝爾感到悲傷。

不得不說，熱火的這幾個落選秀，球隊管理層真的是撿到寶了。一個納恩，一個羅賓遜，一個瓊斯，前面兩個現在是坐穩球隊首發的位置了。

而且在球場的表現也是非常出色的，給人的感覺完全看不出是兩位落選秀。在過去的有一場熱火對陣老鷹的比賽中，這兩位落選秀還有過合砍70分的高光表現。

再加上這場比賽的瓊斯暴力隔扣戈貝爾，必須要給熱火管理層的眼光點個贊。

熱火的落選秀簡直了，真的是太不講理了。但是他們越不講理，球隊老大巴特勒倒是越開心，每天躺著贏比賽，還要有多舒服，怕是巴特勒每天晚上做夢都要笑醒呀。

就在上一場熱火對陣尼克斯的比賽中，巴特勒全場僅3次出手得到9分，球隊還大勝尼克斯15分，試問，這巴特勒還要有多爽，幾個落選秀帶著大哥趟。

在聯盟中30支球隊中，巴特勒可能就是最舒服的老大了吧。球迷朋友，你們對熱火的落選秀都怎麼看？
