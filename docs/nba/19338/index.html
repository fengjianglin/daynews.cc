北京時間12月17日08:00，活塞將在主場對壘奇才，這是兩隊本賽季第二次交鋒，在本賽季首次較量中，活塞輸給了奇才，這次再度遭遇，格里芬可能傷停，德里克羅斯能否帶領活塞一雪前恥？活塞在本賽季開局階段飽受傷病困擾，前19場球輸了13場。隨著傷病情況有所緩解，活塞近來戰績開始提升，最近7場比賽取得了5場勝利。但是，活塞當家球星格里芬再度遭遇膝傷，他在休賽期接受手術的左膝疼痛，有可能缺席對奇才的比賽。

德拉蒙德上一場因為眼部感染休戰，活塞將庄神的狀態設定為每日觀察，對奇才參賽存疑。活塞接下來6場比賽有5個主場，這是他們進一步提升戰績的好機會。奇才最近4場比賽全敗，過去8場輸了7場。奇才本賽季客場戰績是3勝11負，最近5場客場比賽全輸。奇才前場遭遇傷病襲擊，瓦格納左腳踝扭傷，托馬斯布萊恩特右腳受傷。奇才後衛小托馬斯因為左側小腿扭傷已經連續休戰5場，他有可能在對活塞一戰中復出。

首節上來，莫里斯三分遠投，空切上籃，急停跳投，梅克殺傷內線，伍德漂移投籃，布朗補籃，奇才這邊，邦加上籃，八村壘，馬辛米扣籃得手，跳投命中，托馬斯反手上籃，比爾三分遠投，比分來到17：13，奇才暫時領先4分。暫停回來，肯納德跳投命中，莫里斯，羅斯，伍德突破上籃，莫里斯勾手打進，羅斯三分遠投，伍德空中接力，奇才這邊，八村壘，貝爾坦斯突破反手上籃，比爾漂移投籃，托馬斯，比爾撤步跳投，小布朗反手上籃，貝爾坦斯，比爾三分入網，斯科菲爾德空切扣籃，比分來到43：31，活塞暫時落後12分。

次節開戰，米哈伊柳克三分遠投，斯內爾跳投，羅斯殺傷內線，奇才這邊，史密斯跳投命中，馬修斯，貝爾坦斯三分入網，比分來到54：38，奇才暫時領先16分。暫停回來，米哈伊柳克三分遠投，伍德扣籃得手，肯納德勾手打進，三分遠投，莫里斯突破挑籃。奇才這邊，比爾，托馬斯三分遠投，比分來到61：51，奇才暫時領先10分。暫停回來，加洛維兩記三分遠投，一記中距離跳投，肯納德漂移中距離，奇才這邊，比爾殺傷內線，托馬斯，馬辛米跳投命中，比分來到67：61，活塞暫時落後6分。
