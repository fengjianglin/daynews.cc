美媒建議快船用祖巴茨換合適中鋒，4選1，想法很好，但完全不等價

2019-12-09 0條評論 15次閱讀 0人點贊

洛杉磯的兩支球隊，已經成為本賽季最大的熱門，湖人就是缺少第三個穩定得分點，快船就是缺少一個強勁的內線，他們的缺點已經毫不隱藏地暴露在大家面前，今天要說的就是快船，快船裡面正牌中鋒就是祖巴茨一人，手感是不錯，但終究走的是技術流，哈雷爾很拼，但身材跟不上。

而今天就有美媒建議快船，用祖巴茨去交易適合小卡的中鋒，並且還列出了4個備選對象，分別是特里斯坦湯普森、馬克加索爾、史蒂芬亞當斯和尤納斯瓦蘭修納斯，都是有蹲位的防守型內線，而且都具備NBA首發的實力，確實是很適合如今的快船，但是！這完全不等價好嗎？如果那麼輕易就能換到，快船早就幹了，還等著美媒去建議。

先說薪資，祖巴茨是648萬，TT是1800萬，瓦蘭修納斯1600萬，馬克加索爾和亞當斯都是2500萬以上，薪資上就不匹配，另外祖巴茨的實力也不在這幾人之上，並且潛力也比較一般，合同也比較長，論即戰力不如上述4人，論對於重建的意義也不如上面4人，這個建議基本很難實現，尤其TT和馬克加索爾還是到期合同，這兩個快船想都別想，瓦藍和亞當斯倒是還有希望，但快船已經拿不出更多籌碼了。

他們的籌碼都拿去交易保羅喬治了，所以想通過交易去補強，不是一條很行得通的道路，自由市場簽老將倒是個更好的選擇，遺憾的是德懷特霍華德已經被湖人搶走了，考辛斯也被湖人拽在手裡，寧願養著，也不裁掉留給快船，如今自由市場上，好像也沒有合適的中鋒可選了，不得不說快船的內線問題，確實非常棘手，不知道傑里韋斯特有什麼妙招可以解決，畢竟今年是快船比較好的機會，在明年開始，還得面臨哈雷爾的續約問題，空間會繼續被緊縮。
