盤點各個時代最強陣容，如果他們進行一場比賽，誰會脫穎而出？

天天要聞
3 週 ago

江山代有人才出，各領風騷數百年。縱觀NBA歷史，每個時代都有幾個象徵人物，他們承載著一代球迷的青春和回憶，他們是那個時代的球迷選擇熱愛籃球與NBA的理由。每個時代的球員都創造了屬於那個時代的球風和潮流，他們上演著不同的傳奇故事，他們共同追逐那一座奧布萊恩杯，或如願或失意，但對於球迷來說他們都創造了偉大，都值得被銘記。

飛人喬丹、魔術師約翰遜、小飛俠科比、小皇帝詹姆斯等等，他們各自領銜著各自年代最好的陣容進行一場比賽，結果會如何呢？首先盤點一下80年代、90年代、00年代、10年代NBA的四組最佳陣容。

80年代：伊塞亞托馬斯、魔術師約翰遜、拉里伯德、摩西馬龍、天勾賈巴爾

對於80年代這套陣容現如今的球迷可能沒機會見證他們的偉大，但一定對他們的傳說不陌生，他們的名字如雷貫耳，魔術師約翰遜是NBA歷史上最佳控衛，伊塞亞托馬斯是「壞小子軍團」的領袖，拉里伯德擁有出色的投射能力，兩大名人堂球員天勾賈巴爾、摩西馬龍組成超級內線。

90年代：約翰斯托克頓、邁克喬丹、查爾斯巴克利、卡爾馬龍、大夢奧拉朱旺

喬幫主領銜的這套90年代陣容似乎沒有任何短板。喬丹的偉大毋庸贅言，斯托克頓是NBA歷史助攻王和搶斷王，馬龍和巴克利兩位具備最變態身體天賦的球員組成鋒線組合，其恐怖程度難以想像。奧拉朱旺的夢幻腳步會讓其他內線無計可施。

00年代：艾弗森、科比布萊恩特、加內特、鄧肯、奧尼爾

00年代這套陣容極具得分爆炸力，科比歷史第二得分後衛，奧尼爾和科比並肩拿下三連冠是歷史最佳二人組。加內特和鄧肯都是攻防俱佳，鄧肯更擅長組織，狼王擁有全面的進攻技術手段，而AI毫無疑問，是場上最讓人防不勝防的一位。

10年代：斯蒂芬庫里、卡哇伊倫納德、勒布朗詹姆斯、凱文杜蘭特、德懷特霍華德

10年代這套陣容投射能力超群，庫里和杜蘭特是目前進攻效率最高的兩名球員，倫納德攻防兼具，作為兩屆最佳防守球員，倫納德同時擁有超強的中距離投籃能力，號稱「投籃機器」。詹姆斯是歷史上最全面的球員。10年代NBA進入了小球時代，這套陣容恐怕不太擅長身體對抗，但他們極具靈活性，杜蘭特更是打誰都是錯位，因此實力不容小覷。

話說你認為他們之間如果進行一場比賽，那麼誰會脫穎而出，贏得最後的勝利呢？
