湖人機會來了，頂級防守中鋒被擺上交易架，是壓制快船最好選擇

天天要聞
2019-12-11

​

根據The Ringer記者Kevin O’Connor的報道，多位聯盟消息人士透露，雷霆的克里斯-保羅、達尼羅-加里納利、史蒂文-亞當斯和球隊里的其他老將都是可以被交易的。目前雷霆超過奢侈稅線，但是Kevin O’Connor被告知避稅並不是雷霆的重點。多位聯盟消息人士透露，事實上，雷霆願意接手長合同和大合同以求獲得一些其他有價值的資產。而目前來說，比較有交易價值的是亞當斯，亞當斯在內線的威脅很大，至於保留和加里納利或者其他老球員，交易相對較難一些！

​

亞當斯其實比較符合湖人的，目前湖人在內線擁有麥基和霍華德，還有可能賽季報銷的考辛斯，其實麥基在季後賽不是一個太可靠的球員，雖然麥基表現得不差，但是在季後賽，他還是不太值得信任，而亞當斯是經歷過季後賽考驗的球員，他在季後賽是一名合格的內線球員，湖人這個賽季主導的應該是防守，他們應該提升防守能力，而湖人的進攻主要是圍繞著詹姆斯和戴維斯，還有其他的一些射手，但是湖人的射手群目前為止，沒有一個人得分上雙！

​

而且湖人整個後衛線，三分球命中率唯一超過40%的既然是隆多，但是隆多三分球出手少，目前來說丹尼格林，隆多和布拉德利三個後衛是不能動的，丹尼格林三分還是比較靠譜的，布拉德利的防守，是比較靠譜的，目前是後衛橫行的時代，很多球隊發起都是後衛，所以布拉德利對於湖人很重要，而丹尼格林其實是為了減少詹姆斯的防守壓力，隆多是球隊唯一的後衛，季後賽會是比較重要的一點，而且這三個人都很能防守！

​

湖人若能夠得到亞當斯，內線防守問題解決了，讓戴維斯將更多的精力放在進攻端，而且亞當斯的前場籃板很厲害，可以讓湖人投手群出手變得更加果斷，亞當斯是湖人進軍聯盟第一防守強隊最重要的一環，而且還可以壓制最大對手快船，快船在面對有厲害的高大內線時，打得很難受，亞當斯可以限制住路威和哈雷爾的組合，當然湖人需要付出一些代價，比如庫茲馬，如果不想交易庫茲馬，還可以用未來的選秀權，順便還可以處理掉考辛斯，甚至直接將麥基交易走，未來兩個首輪選秀權，加上今年湖人選中的二輪新秀，雷霆可能會同意這一筆交易！
