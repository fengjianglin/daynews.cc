跟腱撕裂重出江湖！繼科比杜蘭特後，1600萬先生也再次遭遇眷顧

2019-12-08 0條評論 17次閱讀 0人點贊

北京時間12月7日，NBA常規賽繼續進行著，西部第一的洛杉磯湖人隊前往客場挑戰西部第九的開拓者隊。由於兩隊之間有著一定的實力懸殊，因此在開打前很多球迷都認為這是湖人能夠穩贏的一場勝利。儘管很多球迷都預料到了最後的結果，但相信很多人對於比賽中的一些花絮是無法補腦的。今天湖人隊雖然戰勝了開拓者，但對於兩隊來說更壞的消息還在後面。

本場比賽第1節還剩三分鐘時，開拓者球員胡德在拼搶籃板後，無對抗的情況下突然倒地，並且用手碰了碰他的跟腱。當時的場面一度騷動起來，隨後在隊友和工作人員的攙扶下，胡德才慢慢的離開。而這心碎的一幕，也不由得讓人想起了上賽季杜蘭特跟腱斷裂時的場景。

中場休息時，開拓者官方正式宣布，胡德左腳跟腱撕裂。本賽季胡德和開拓者隊這邊達成了一份兩年1600萬美元的合同，場均能夠貢獻11.5分， 3.5個籃板，1.6次助攻是球隊繼利拉德麥克勒姆外最穩定的得分點，同時也是開拓者隊非常看重的側翼小前鋒。這次受傷很有可能導致他本賽季報銷，同時也讓開拓者的鋒線輪換變得捉襟見肘。

眾所周知，跟腱斷裂對於籃球運動員來說簡直是毀滅性的打擊，幾乎是籃球運動員的頭號殺手。在NBA聯盟中有很多球員都遭受過這樣的傷病，就連科比杜蘭特這樣的頂級巨星也難逃此惡運。

上賽季總決賽中，杜蘭特跟腱斷裂，至今還處於傷病恢復中。而前NBA球星科比在2013年湖人對陣勇士的比賽中，同樣由於跟腱突然斷裂，痛苦倒地，之後科比的狀態逐年下滑，於是在兩個賽季後就選擇了黯然退役！

遭遇過此種傷病的球員狀態都會大大折扣，科比、考辛斯、蓋伊等球員就是最好的證明！好在胡德今年只有28歲，即使痊癒後不能回歸到巔峰狀態，但也不至於昭示職業生涯的結束！但願他能早日康復吧！

本場比賽，湖人隊這邊雖然贏球了，但是球隊也發生了一件揪心的事情。比賽進行到第3節的時候，湖人隊的隆多也受到了傷病的眷顧，隆多遭遇左腿筋拉傷。雖然腿筋拉傷相比於跟腱斷裂不是太大的問題，但是這種傷病對於很多NBA球員都是不容小覷的，腿筋拉傷很容易讓以前的傷病複發。要知道，當年保羅就是因為腿筋拉傷，火箭才在搶7大戰中輸給了勇士，遺憾的與總決賽的地板失之交臂！

本場比賽對於兩隊可以用一個詞來形容，那就是慘烈。前一秒還是生龍活虎的隊友，後一秒就可能宣告NBA生涯的終結。今天隆多和胡德的受傷，已經給NBA聯盟敲響了警鐘。畢竟常規賽還很漫長，對於所有球員都應該且打且珍惜！
