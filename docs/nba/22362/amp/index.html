湖人一重要因素不如雄鹿或成比賽勝負手 為何湖人仍不用悲觀

天天要聞
2019-12-19

（本文所用數據均來自於12.18的NBA官網）

西部第一的湖人和東部第一的雄鹿終於要碰面了，而且兩支球隊都是24勝4負。

說來也是巧合，本來睥睨東部、豪取18連勝的雄鹿面對沒有東契奇的小牛居然跪了，饒是東區頭號球星字母哥全場轟下48分14籃板仍難阻德州牛仔的衝擊。

一天後，客場已取14連勝的湖人在戴維斯缺席的情況下，憑藉著向來被詬病的替補席的支援，與堅韌的印城人戰至最後一刻，居然因為罰球太次，遺憾落敗。

這場比賽的贏家將獲得聯盟第一的稱號，輸家只能苦吞兩連敗。就目前而言兩支球隊從未經歷過連敗，如此看來，小牛和步行者當初是卯足了勁要讓湖人雄鹿拿出背水一戰的決心啊。

就現階段從數據上以及人員構成而言，湖人的贏面稍小。

兩支球隊進攻端特點鮮明，風格迥異。

一隻重籃下，用傳統比賽方式磨陣地，籃下出手聯盟第六，禁區得分聯盟第三，輔以準頭並不出色的三分球打擊。

一隻則是除了當家球星人均射手的三分大隊，三分球無論是出手次數還是命中次數均排聯盟第三，而且揚尼斯今年對於投三分的自信提升顯著。

除此之外，因為有了勒布朗和揚尼斯，兩個球隊抓快攻能力都是一流，雄鹿快攻得分聯盟第二，湖人快攻得分聯盟第三，差值不過0.6分。

在防守端兩支球隊大同小異，屯積長人，狠抓籃下防守，湖人場均讓對手在油漆區里拿到43.4分，聯盟第五，雄鹿更甚，場均讓對手在禁區內只得39分，聯盟第一。

比賽最大的看點是兩對頂尖前場組合之間的較量——勒布朗+戴維斯pk揚尼斯+洛佩茲。

不知道詹眉擋拆拆開後順下攻擊雄鹿禁區挑戰字母哥恐怖臂展四處掃蕩+場均2.5個蓋帽的護框高手洛佩茲效果如何，同時也考驗字母哥要怎麼踏入湖人內線得分，只要是詹姆斯或者戴維斯單防字母哥突破，他得分就不會太容易，加之麥基/霍華德的蓋帽威懾，最好能逼迫希臘怪獸多在中遠距離出手。

不過之所以說揚尼斯+洛佩茲是頂尖的前場組合，不止在防守端，還在於大洛能拉到三分線以外，這樣大概率會形成字母哥一對一突破，萬一雄鹿連續用擋拆帶走詹姆斯或者戴維斯，對上湖人的後衛直接碾進去得分的後果不可想像。

聯盟第一之爭，大概率會變成一場看內線攻防的比賽。

不過我覺得這場比賽的勝負手會出在三分線上，兩支球隊防三分防的都不好，尤其是雄鹿，當其他球隊面對雄鹿時，三分命中率會飆升到41%！如果湖人能把握住這一點，贏面會很大。

參考上一場的小牛，小牛正是靠三分投死了雄鹿，尤其波爾津吉斯顏射字母哥的那兩個。

只不過很可惜，湖人本身沒有這麼奢侈的投射資源，另外這五場比賽的三分命中率只有31%……

希望比賽中全隊的手感能回升到正常水平吧。

另外，字母哥突進去可以分球給外面四個射手，前面說過了，雄鹿的射手團產量高達全聯盟第三。對湖人外線輪轉換位和撲防能力來說，大考驗到了。

雙方擁有同一等級的防守體系，雄鹿場均失分更多，但是前面解釋過了，問題出在三分線外的防守上，這一點恰巧是湖人很可能利用不上的防守弱點。

以進攻體系而言，湖人要略遜。雄鹿禁區打不開，還有遠投能力，湖人不行，這也是湖人贏面小的原因之一。

可是呢作為湖蜜不必悲觀。

戴維斯曾在主場輸給小牛後說我們不會連輸兩場，接下來湖人帶著流感連續征戰客場取得了七連勝，包括攻克了丹佛高原、猶他鹽湖城，也打破了熱火主場的不敗金身。

敗走印第安納之後，全隊都表示要把目標放在下一場前往密爾沃基的重頭戲上。我們要相信勒布朗和戴維斯的心氣，勒布朗和戴維斯就是聯盟最好的二人組。我們也要相信整支球隊的化學反應和決心，今年我們真的很強。

密爾沃基，我們來了！

勢必凱旋迴到斯台普斯中心！
