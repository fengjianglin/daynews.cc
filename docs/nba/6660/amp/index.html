斯蒂芬A：52歲的自己也能像恩比德一樣自由得分，可惜膝蓋不允許

天天要聞
3 週 ago

在昨天76人客場挑戰猛龍隊的比賽中，以大帝自稱的恩比德居然在比賽中獲得了一個鴨蛋，在此之前恩比德的單場最低得分為5分。作為一名內線球員，恩比德在比賽中出手了11次，並且還獲得了3次罰球的機會，遺憾的是他都錯過了。

可以說昨天76人的失利，與大帝糟糕的表現有著直接的關係。賽後也有猛龍隊的球迷和記者嘲諷恩比德這一次不知道是不是又吃了什麼不幹凈的東西，導致他全身無力。而猛龍的球迷則在社交媒體上曬出了上個賽季76人季後賽被絕殺後，大帝在球員通道中痛哭的圖片。

今天斯蒂芬A史密斯作為NBA的評論員，有著中國楊毅之稱的他。在節目中也公開的嘲諷了恩比德一番。他表示：「看見了恩比德在球場上的表現後，自己堅信52歲的自己也能夠像他一樣自愛NBA的賽場上自由的得分。不過令人遺憾的是自己的肩袖撕裂已經膝蓋問題困擾了自己3年之久。」

76人本賽季有著很大希望衝出東部的球隊之一，但恩比德的表現簡直讓人大跌眼鏡。76人要過雄鹿這一關，必須要恩比德打出良好的表現才行。不過賽季才剛剛開始，恩比德還有時間去改善目前他的缺點。不過還有一點值得恩比德慶幸的是，他與唐斯之間的衝突幸虧發生在這場比賽之前。要不然唐斯可能也會像范喬丹那樣嘲諷大帝，能夠百分之百防守住恩比德，只有他自己。

在輸給猛龍隊後，目前76人11勝6負排在了東部第5的位置上。而恩比德場均可以貢獻21分11.8籃板2.9助攻和1.3蓋帽，他目前還需要解決面對像小加索爾這樣內線的應對能力。不然76人是無法在季後賽中取得勝利的。
