小哈達威27分 鉑金22+18 恩比德33+17 獨行俠送76人三連敗

天天要聞
2019-12-21

直播吧12月21日訊 NBA常規賽，76人今日坐鎮主場迎戰獨行俠，客隊上場比賽不敵凱爾特人暫列西部第五，而主隊同樣狀況不佳，遭遇兩連敗後排名東部第五，此役東契奇繼續因傷缺戰。比賽上來獨行俠迅速反客為主取得領先，其中小哈達威首節三分5中5爆砍19分，次節獨行俠一度將分差拉大到兩位數，但76人及時調整半場結束時分差回到個位數。末節76人始終未能縮小分差，比賽也逐漸失去懸念，最終獨行俠在客場拿下76人，送給後者三連敗。

雙方數據

獨行俠（19-9）：哈達威27分4籃板、波爾津吉斯22分18籃板3蓋帽、布倫森11分3籃板7助攻、庫里11分、芬尼-史密斯11分6籃板4助攻2搶斷、克萊伯10分3籃板2助攻、鮑威爾9分5籃板4助攻、萊特8分7籃板4助攻4搶斷、布羅科夫8分。

76人（20-10）：恩比德33分17籃板2助攻、西蒙斯12分6籃板8助攻3搶斷、理查德森10分6籃板4助攻2搶斷、霍福德9分2籃板3助攻、哈里斯7分2籃板2助攻、斯科特7分5籃板3助攻、伯克7分、恩尼斯6分、科克馬茲5分、內托2分。

首節上來，西蒙斯率先空接拿到兩分，芬尼-史密斯底角射進三分，但霍福德立馬追身遠投給出回應，哈達威罰球線命中急停跳投，西蒙斯殺到禁區拿到兩分，暫停調整過後波爾津吉斯跟進空接暴扣得手，布倫森突破挑籃命中，理查德森弧頂射進追身三分，恩尼斯跟進拉杆上籃得手，芬尼-史密斯搶斷奉獻精彩暴扣，哈達威迎著防守強行挑籃打成2+1，隨後又連中兩記三分，恩比德低位勾手打成2+1，克萊伯轟進壓哨遠投。首節結束，獨行俠35-28領先76人。

次節開局，內托突破上籃得手，哈里斯殺到禁區勾手命中，霍福德低位投進高難度後仰跳投，鮑威爾跟進空接暴扣得分，霍福德底角射進三分，但波爾津吉斯立馬在正面遠投給出回應，暫停調整過後布羅科夫罰球線投進中距離，波爾津吉斯自投自搶再拿兩分，場上分差達到兩位數，西蒙斯禁區內勾手打成2+1，恩尼斯底角射進三分，恩比德跟進輕鬆得手，克萊伯突破奉獻精彩暴扣，恩比德再中三分。半場戰罷，獨行俠63-57領先76人。

易邊再戰，恩比德低位命中半截籃，隨後又射進三分，波爾津吉斯跟進輕鬆扣籃得手，布倫森借掩護拋中兩分，克萊伯弧頂命中遠投，理查德森反擊低手上籃拿到兩分，克萊伯突破拋投打成2+1，暫停調整過後波爾津吉斯滑翔上籃得手，科克馬茲溜底線上進反籃，哈里斯罰球線命中撤步跳投，庫里正面突破拋中兩分，隨後又迎著防守射進遠投，個人連得七分幫助獨行俠再次將分差拉大到兩位數。三節結束，獨行俠88-77領先76人。

末節上來，庫里繼續火熱狀態拋中兩分，理查德森外線射進空位三分，恩比德禁區接球暴扣得手，克萊伯突破上進反籃，斯科特射進追身三分，暫停調整過後伯克罰球線命中半截籃，小哈達威外線射進不講理三分，76人始終未能縮小分差，斯科特急停跳投得手，萊特空切籃下將球打進，波爾津吉斯順下拿到兩分，布倫森再中高位跳投後獨行俠一波小高潮將分差繼續擴大，比賽徹底失去懸念，最終獨行俠在客場拿下76人，送給後者三連敗。

雙方首發

獨行俠：鮑威爾、波爾津吉斯、史密斯、布倫森、哈達威

76人：恩比德、西蒙斯、理查德森、霍福德、哈里斯
