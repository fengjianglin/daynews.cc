又是一場慘敗！33分鐘砍32+11+1+4+5，他根本帶不動湖人隊

2019-12-23 804條評論 2,212次閱讀 223人點贊

掘金客場挑戰湖人。湖人在經歷了一波客場作戰之後，回到斯台普斯，但有兩個壞消息，他們經歷了兩連敗，都輸給了東部比較強的球隊，他們本場比賽失去了詹姆斯。對於掘金來講，這應該是一場復仇的機會。

首節比賽，湖人沒有詹姆斯做球，進攻顯得有些停滯，出現的機會不多，湖人的第一次進攻就是隆多突破被慢悠悠的約基奇蓋帽，掘金第一個回合就打成反擊得分。湖人的前四分，都是通過濃眉強打的到的，一個高難度中距離跳投，一個籃下強打米爾薩普上反籃，外線雖然有機會，但是射手狀態不佳，沒有投進。

掘金欺負湖人退守不及時，連續打成反擊，勢頭很高。但是暫停回來之後，湖人加強了防守，三分球也投開了，迅速還了一波8-0。後面憑藉著卡魯索的防守和濃眉的拼搶，湖人在第一節領先掘金5分，雙方打成23-28。

次節比賽，湖人的第二陣容沒有隆多組織，也沒有濃眉這個進攻強點，是非常弱勢的一個陣容。庫克和庫茲馬在持球端的處理工作非常低效，湖人出現的機會並不多，但是好在湖人的防守不錯，掘金的進攻效率也不高，打了5分鐘，掘金才反超比分，這已經是超出預期的結果了。

不過沃格爾沒有換陣容，而是讓球員自己調整，回來之後庫茲馬加強了個人進攻，出手果斷，翻回領先優勢，打停了掘金。濃眉在最後5分鐘回來，連續打掘金內線，但是效率不是特別高，其他球員也沒有提供得分幫助，在半場結束之前湖人把分差縮小到2分，雙方打成55-53。

第三節比賽，湖人和掘金進入交替領先模式，但是在濃眉進攻出現問題，打不進的時候，掘金就抓住反擊機會，靠著米爾薩普的連續得分，掘金最大把分差拉開到10分。濃眉領到個人第四次犯規下去之後，湖人的進攻更加困難了，庫茲馬打了幾個，但是湖人的防守被爆，末段更是出現了莫名其妙的兩個失誤。第三節結束，掘金把分差拉開到了13分之多。

末節比賽，湖人一開始就出現失誤，分差逐漸擴大到18分，沃格爾請求暫停之後，沒有延緩掘金的進攻，反而湖人的防守還出現問題，掘金三分線外連續命中，濃眉的膝蓋貌似也出現了問題，不敢打得太拼，分差很快被拉開到23分。比賽還剩4分半的時候，湖人仍然落後20分以上，沃格爾選擇了投降。最終雙方打成128-104，湖人迎來了三連敗。

濃眉打了33分鐘，23投13中，三分球4投1中，得到32分11籃板1助攻4蓋帽5失誤，正負值-13，詹姆斯單獨帶隊的第一場比賽，湖人輸了，濃眉單獨帶隊的第一場比賽，湖人也輸了，看來，湖人想要贏球，還是得靠兩個核心共同發力才行啊，一個核心帶隊，球隊的劣勢被瞬間暴露出來，角色球員的整體效率還是要在超級球星的身邊才能夠發揮出來。

全場比賽，湖人的進攻出現了很大的問題，從比賽一開始就打得很不流暢，湖人全靠濃眉的個人能力在得分，沒有詹姆斯，湖人就沒有進攻創造者，隆多頂多算是一個組織比賽的人，而不是像詹姆斯這種可以通過突破來增加威脅吸引防守創造進攻的人，所以湖人的戰術其實非常少，掘金的防守很有針對性，濃眉就算有天大的本事，一旦他下場，局勢就會被逆轉過來，這就是為什麼掘金在首節比賽落後，後面三節壓著湖人打的原因。湖人把隆多提上首發，第二陣容也沒有濃眉，所當進入輪換階段的時候，面對防守效率排名聯盟前三的掘金，湖人基本上就是一個送分陣容了。

下一場打快船，如果詹姆斯不復出，濃眉傷勢沒好，很有可能會四連敗。接著還要打開拓者，獨行俠，太陽，這些都是季後賽級別的球隊，回想上個賽季的湖人隊，也是在聖誕節的時候出現了轉變，細思極恐啊！
