18連勝?字母哥：就算是30連勝也沒用，今天被一個中學生打敗

2019-12-17 913條評論 2,487次閱讀 1055人點贊

北京時間12月15日，NBA常規賽繼續進行。雄鹿主場迎戰騎士隊，最近雄鹿是高歌猛進，一路過關斬將，就連快船也被雄鹿胖揍一頓，完全被打蒙了。

比賽開始以後，米德爾頓和阿德托昆博內外線開火率領球隊獲得9-2的開局，騎士這邊依靠戴拉維多瓦的三分和湯普森的補籃追近比分，不過阿德托昆博暴扣得手，雄鹿趁勢一波8-0再次擴大優勢，最終半場雄鹿69-53領先對手16分。下半場直接進入了垃圾時間，最終，雄鹿125-108戰勝騎士，收穫十八連勝。

全場比賽，加蘭10分5助攻；樂福12分10籃板；湯普森12分9籃板；克拉克森13分；迪溫琴佐15分5籃板；米德爾頓24分6助攻；揚尼斯-阿德托昆博29分；喬治-希爾11分；羅賓-洛佩茲16分6籃板；艾森-伊利亞索瓦11分。

賽後，字母哥接受了採訪，談到球隊18連勝的時候，字母哥調侃說：「今天全美所有籃球迷都在關注塞拉峽谷中學和聖文森特聖瑪麗中學，詹姆斯的兒子布朗尼已經上了各大媒體的頭條，我們就算30連勝，也沒有這樣的關注度。這就是詹姆斯的影響力，他的一舉一動都在影響著全世界的籃球愛好者，我們贏了騎士，卻被一名高中生打敗了，哈哈。」
