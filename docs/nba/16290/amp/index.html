場均30+9+9，甚至比當年喬丹老詹都猛，維金斯唐斯鶯歌應向他看齊

天天要聞
2019-12-14

12月13日獨行俠和活塞在墨西哥站展開交戰，最終獨行俠是以122:111贏下了比賽，戰績來到了17勝7負，繼續鞏固了自己西部第3的位置。本場比賽他們領袖東契奇發揮神勇，24投14中，拿下了41分+12籃板+11助攻的超級大三雙，而這數據也讓他超越了詹姆斯，成為21歲前拿下40+三雙數據最多的球員（他2次，詹姆斯1次）。賽季至今他場均拿下30.4分+9.9籃板+9.3助攻，而他這樣的數據比當年的詹姆斯和喬丹都出色，他的天賦讓人嘆為觀止。

在今天的比賽開始時，東契奇的手感並不好，開局2次出手全部打鐵，球隊也被活塞打了一個9:2，嚇得卡萊爾趕緊叫了一個暫停。而在比賽回來後，他多次強殺籃下為球隊止血，當比賽進行到還剩7分42秒時，他則是迎著布朗飆中3分，得到本場比賽運動戰的第一分，單節比賽他里突外投，得到了15分+7籃板+3助攻，球隊也以34:29領先。進入第二比賽後，活塞開始瘋狂追分，在比賽還剩3分鐘時他們落後了7分，而此時作為領袖的東契奇則是站了出來，他多次強殺內線完成得分，單節比賽他5中3，得到了8分+3籃板，球隊以65:58結束上半場。

進入下半場後，他在開局連續助攻包威爾和波神在籃下得分，而他自己也在三分線外發炮，單節比賽他6中4拿下了11分+2籃板+6助攻，球隊也95：85結束前三節。在第四節比賽，他拿下了7分+2助攻，並且沒有給活塞任何機會，最終穩把穩的贏下了比賽。

今年的東契奇才20歲，但是他的成熟和老練遠超他的年齡，他知道自己能做什麼，什麼時候該做什麼，他的得分排名聯盟第3，助攻聯盟第2，籃板13，而且他不僅是數據出色，球隊的戰績也是名列西部前茅。如果按照如今的他的戰績和數據來衡量的話，他毫不弱於同時期詹姆和喬丹，喬丹在生涯的第二個賽季場均只能拿下22.7分+3.6籃板+2.9助攻，而詹姆斯則是27.2分+7.4籃板+7.2助攻，球隊也止步於季後賽的大門。

東契奇是如今聯盟中天賦較為出色，球商高，但是又很努力的球員，而且天生具有領袖氣質。而在他如此優異的表現之下，讓人不得不想起跟他一樣天賦異稟的幾個球員：維金斯、唐斯和英格拉姆，他們身體天賦不比東契奇差，但是帶隊和領袖能力卻相差甚遠。至今森林狼取得了10勝14負排名西部第10，作為領袖的維金斯在打了幾場好球後又開始養生；唐斯則是越打越軟，最喜歡飄在外面投籃，害怕對抗。

英格拉姆的些據看著還行，場均24.9分+7.1籃板，而很多球迷和專家也把鵜鶘的糟糕戰績歸結到了金特里的身上。但是實話實說，英格拉姆也有不可推卸的責任，他在場上基本無精打采，作為球隊公認的領袖，他需沒有扛起起球隊。

如今東契奇為所有年輕球員做了一個榜樣，無論是場上對待比賽的態度，還是在場下刻苦的程度他都無可挑剔，作為同樣是年輕的一代的唐斯、英格拉姆和維金斯都應該向他學習
