作為落選秀年薪曾超同屆狀元西蒙斯，如今更是有機會獲得億元合同

天天要聞
2 週 ago

NBA一直都是一個不問球員出身的地方，一切都是依靠實力說話，只要有實力就有地位和億元合同。相反不少淪為邊緣人物，甚至去海外打球的是出身杜克、肯塔基以及UCLA等名校的球員。本賽季在猛龍隊打上首發的范弗里特就是出自名不見經傳的威奇塔州立大學，在威奇塔州立大學打了四年球之後，范弗里特參加選秀的時候已經年滿22歲，最終遺憾落選。

范弗里特從來都不是輕言放棄的人，落選之後他代表猛龍隊打夏季聯賽，最終打動猛龍隊獲得一份兩年145萬美元的部分保障合同。他新秀賽季不斷奔波於猛龍發展聯盟下屬球隊和猛龍隊，整個新秀賽季他場均僅獲得可憐的7.9分鐘，但這也鍛造出范弗里特不服輸的勇氣以及更加堅強的意志。范喬丹之名，也是從這個時候悄然拉開序幕。

范弗里特通過不懈的努力，在2018年夏天獲得一份2年1800萬的合同，這讓他在18-19賽季的年薪甚至超過了當屆狀元郎本-西蒙斯。18-19賽季對於范弗里特而言算得上是先悲後喜，他季後賽在東決之前的表現都不能讓人滿意，甚至東決前三場，他合計只有11投2中。可是在東決的後三場比賽中，他合計17投14打了雄鹿一個措手不及，這樣的表示一直延續到總決賽。

近日知名薪資專家Bobby Marks表示範弗里特下份合同的薪資將會在2500萬-3000萬之間，也就是說范弗里特有機會拿到億元合同。雖然說范弗里特是從發展聯盟打上來的，但是就算拿億元合同也應該理所應當，畢竟他的表現已經征服了所有猛龍球迷。在洛瑞逐漸老去之後，猛龍隊在控衛上更加需要范弗里特，更重要的是范弗里特和猛龍新核西亞卡姆有著良好的化學反應。
