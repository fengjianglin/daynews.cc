雙德齊爆發，馬刺吃餃子，灰熊“劫富濟貧”助馬刺升第九

2019-12-24 256條評論 2,250次閱讀 118人點贊

NBA常規賽繼續進行，灰熊主場迎戰馬刺，馬刺依然徘徊在前八之外，狀態低迷，迎戰灰熊是他們贏球的好機會。比賽一開始阿爾德里奇就火力全開，上半場拿下28分，幫助馬刺以72-61取得領先，下半場阿爾德里奇繼續爆發，德羅贊也開始得分，馬刺一下子拉開比分，最終以145-115大勝灰熊，罕見的取得一場大勝，在聖誕節前夕，吃頓餃子。

德羅贊和阿爾德里奇一起爆發了，阿爾德里奇全場得到40分9籃板5助攻，德羅贊11投10中得到了26分10助攻，灰熊的莫蘭特得到19分，瓦蘭修納斯得到18分12籃板，值得一提的是憑藉這場勝利，馬刺升至西部第九位，緊緊追趕難兄難弟開拓者，而這真的要感謝灰熊的「劫富濟貧」。

在過去10場比賽，灰熊取得了5勝5負的戰績，整體上很不錯，更重要的是，他們連續戰勝了馬刺的競爭對手太陽和國王，今天又輸給馬刺，里外里為馬刺奉獻了1.5個勝場的追趕距離，可謂是雪中送炭啊。

比賽開始，德羅贊和阿爾德里奇各進一球，傑克遜和莫蘭特則回敬2個三分，雙方開局都不錯，但馬刺很快爆發，福布斯連續投中三個三分球，阿爾德里奇在內線連續打進，馬刺一波21-6之後，以25-12領先到了兩位數，隨後灰熊外線開始得分，但阿爾德里奇都能做出回敬，首節結束，馬刺46-31領先15分。

第二節比賽，蓋伊三分命中，瓦蘭修納斯連得6分，謹慎的波波維奇馬上喊了暫停，暫停回來，馬刺依然沒限制住灰熊，阿倫連中2個三分，莫蘭特打成2+1，灰熊將比分追至53-57。阿爾德里奇重新回到場上，他連得6分，馬刺回敬一波8-0，繼續拉開比分，克拉克連得6分，萊爾斯三分命中，德羅贊也兩罰全中，馬刺72-61領先結束上半場。

易邊再戰，阿德延續火熱手感，德羅贊錦上添花，連得4分。隨後灰熊多次發起反擊，但馬刺總能及時作出回應，本節末段，德羅贊連得6分，馬刺108-89領先到了19分，最後一節，蓋伊和阿德聯手得到5分，馬刺領先到20分以上，儘管灰熊一度再把分差縮小至13分，但馬刺又是一波15-0，徹底帶走比賽，最終145-115大勝灰熊。
