狂轟78+22+11！三巨頭助沃克複仇老東家，歐文替身要後悔了

天天要聞
2019-12-23

北京時間12月23日上午結束的一場NBA常規賽，凱爾特人主場迎戰黃蜂，本場比賽之前，凱爾特人以19勝7負的戰績位列東部第二，黃蜂以13勝19負的戰績排名東部第九，比賽最後的結果沒有懸念，119-93，凱爾特人主場26分大勝黃蜂！雖然海沃德和斯馬特本場比賽繼續缺戰，但雙方的實力相差確實太大了。

數據方面，凱爾特人這邊，塔圖姆39分12籃板2助攻，布朗16分9籃板5助攻，沃克23分1籃板4助攻，威廉姆斯12分。黃蜂這邊，格拉漢姆23分10助攻，比永博18分，布里奇斯15分6籃板，羅齊爾14分4助攻。

第一節初段格拉漢姆轟進三記三分，羅齊爾自投自搶，直接打停凱爾特人。暫停回來，布朗爆發，連續利用突破和中距離拿到8分，而節末格拉漢姆和馬丁又是一波三分雨，單節黃蜂領先凱爾特人7分。第二節回來，塔圖姆連拿5分，沃克也強投三分命中，節末威廉姆斯和沃克率隊打出一波小高潮，半場凱爾特人完成反超！第三節塔圖姆繼續衝擊籃下，雙方陷入膠著，沃克面對老東家並沒有手軟，但分差並沒有拉開，三節結束，凱爾特人84-77黃蜂。

最後一節，塔圖姆不可阻擋，先是勾手打進，隨後中距離也得手，黃蜂暫停過後，塔圖姆繼續突入籃下造殺傷，而節末塔圖姆持續砍分表演，連續轟進三分，單節貢獻13中9轟進22分，黃蜂被直接打花，比分定格在了119-93！

而塔圖姆一戰創下兩大紀錄：39分的得分是他單場得分生涯新高，此外末節22分也是個人單節得分紀錄，此前是15分，堪稱凱爾特人大勝的第一功臣。實際上此戰發揮出色的不僅僅只有塔圖姆，布朗在被史蒂文斯改造成得分後衛後，本賽季有了質的轉變，從上賽季的場均13+4.2飆升到19.9+6.9+2.3+1.2，得分籃板助攻搶斷樣樣全能！而且布朗的三分球命中率已經提升到37.7%！

當然，最穩定的依舊是沃克，16中8，全場轟進5記三分，輕鬆拿下23分4助攻打卡下班，凱爾特人三巨頭合力貢獻78分22籃板11助攻的表現，脆弱的黃蜂顯然無力抵抗。

再來聊聊羅齊爾，本場比賽出戰38分鐘，17中6，三分線外5中0，實話實說，這個數據著實有點一般，在場正負值-21全場最低，和沃克的對位中從身體對抗到腳步靈活性，不是一個級別，關鍵是沃克的防守態度明顯要比羅齊爾好很多！今年夏天，歐文離開，當時羅齊爾在各個媒體上有意無意的製造歐文是球隊矛盾起源的說法，他也公開喊話安吉，希望能夠留在凱爾特人，萬萬沒想到交易市場大門開啟後，立刻被當成了沃克先簽後換的籌碼！

當然，來到黃蜂他也不虧，畢竟喬丹給了他一份年薪接近2000萬的大合同，但他的表現著實難讓人滿意，何況在這支球隊，小將格拉漢姆通過自己穩定的表現成為球隊核心，不論是得分還是助攻都比羅齊爾高出不少！此外，羅齊爾很難再去想金錢之外的事情，因為黃蜂這支球隊，上限就這樣了。但如果按照這種表現打下去，羅齊爾恐怕會更加平庸，昔日的歐文替身，後悔么？
