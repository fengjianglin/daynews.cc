選秀大年！18屆新秀前5順位重排：歐洲新王狀元，小庫里也進前3

2019-11-29 0條評論 16次閱讀 0人點贊

​

2018年選秀大會是非常有趣的一年，因為出乎意料的情況是，有些被認為是必選的球員能打出遠超預期的表現，而有些球員卻沒能達到預期。很明顯，有些球員並沒有在適當的位置被摘下，那麼接下來我們就來看看2018屆新秀重排結果：

​

第5順位：馬文-巴格利三世

國王將馬文-巴格利看作是榜眼秀的首要人選，巴格利確實也被球隊寄予厚望，但目前為止，他並沒有兌現天賦。他進入油漆區的能力確實是他在進攻端非常關鍵的部分，所以巴格利目前能有場均14分和10籃板的兩雙數據。

但和本屆選秀大部分內線球員一樣，巴格利並沒有太好的三分命中率，所以隨著時間的推移，巴格利能開發出外線三分投射也並不奇怪。

​

第4順位：賈倫-傑克遜

賈倫-傑克遜是在第4順位被灰熊摘下的球員，他對灰熊來說確實是非常不錯的選擇，目前他已經成長為球隊得分王之一，並且在防守端也能有非常出色的表現。

上賽季在小加索爾身邊待了半個賽季後，傑克遜似乎能從這名老將全明星身上學到一些東西，並且成為本屆選秀中最好的新秀球員之一，但他也還有成長空間。如果傑克遜能成為更穩定的得分手和射手球員，那麼我們完全可能看到他在未來賽季成為自己位置上最出色的球員之一。

​

第3順位：德安德烈-艾頓

艾頓是在第一順位被太陽海峽的球員，儘管外界對他的天賦還有一些質疑，但他仍是2018年選秀大會中最好的球員之一。這並不會說艾頓的新秀賽季表現不好，而是其他球員能有更好的表現。

目前為止，艾頓還是有很好的內線得分能力，而且只要他能接近籃筐就能有很好的外線威脅。不過他主要的問題還是在投籃和防守端，所以如果未來賽季他能開發出足夠好的外線投射能力，這肯定會有更好的未來。

​

第2順位：特雷-楊

特雷-楊是在第5順位被獨行俠摘下的球員，不過緊接著就被送到老鷹並且為獨行俠換回東契奇。在選秀大會前，特雷-楊就被用作與庫里進行比較。

不過從生涯第二個賽季開始，特雷-楊就逐漸找到自己在外線打球的節奏並且目前看來他有可能在未來幾年內成為聯盟最好的射手球員之一，當然他的防守端仍是問題所在。本賽季至今，特雷-楊場均能得到26.6分8.7中和1.5搶斷的數據表現。

​

第1順位：盧卡-東契奇

上面我們已經提到，東契奇是在第3順位被老鷹摘下但最終被送到獨行俠的球員。東契奇職業生涯可以說是有歷史級別的開端，他能新秀賽季就打出自勒布朗-詹姆斯以來從未見過的數據表現。

東契奇的新秀賽季能有非常頂級的發揮，他最終奪下最佳新秀獎項。而本賽季至今，東契奇場均更是能砍下30.1分9.5助攻和10籃板的數據表現。

​

東契奇是非常全能的球員，目前他的主要缺陷還是在防守端，但我們完全能期待東契奇在各方面都能繼續有所成長，並且最終成為聯盟最好的球員之一。
