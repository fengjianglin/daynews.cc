詹皇打服的五位巨星，三次化敵為友，一人如今放棄底薪也要投靠他

2019-12-08 0條評論 17次閱讀 0人點贊

北京時間12月8日，NBA常規賽賽程已經過了四分之一，目前奪冠大熱洛杉磯湖人隊目前以20勝3負的成績位列西部第一，領先了第二名的獨行俠3.5個勝場，還有不到一個月就將年滿三十五歲的詹姆斯狀態依然火熱，統治力和領導力依然滿格，現在來看看他當初擊敗的那一眾巨星的現狀，不禁讓人有些唏噓。

德懷恩-韋德與詹姆斯同屬於03黃金一代，儘管韋德沒有詹姆斯那樣「天選之子」的光環，但打了兩年就聲名鵲起的他，完全沒有理由去和一個同樣是新星的詹姆斯俯首稱臣，但隨著兩人的數次交手和場下的密切交往，詹姆斯的人格和球技讓韋德深深折服，最終促成了兩人在邁阿密完成了偉業，儘管韋德如今已經退役，但不妨礙兩人維持這這段「夫妻般」默契的兄弟情義。

在詹姆斯與韋德熱火聯手期間，凱爾特人一直是他們最主要的對手之一，多次在季後賽將綠軍淘汰之後，雷-阿倫選擇離開波士頓加入熱火，上演這樣「反叛」的戲碼成為了雷-阿倫無悔的決定，他認定跟隨詹皇可以再奪一冠，的確，他在與馬刺G6最後時刻的那記追平三分，給這段化敵為友的好戲，寫下了濃重的一筆。

老詹身邊另外一個化敵為友的例子，就必須要提到德里克-羅斯了，當年初生牛犢不怕虎的他，在季後賽與熱火三巨頭打的天翻地覆，但命運寫給羅斯的劇本卻滿是悲劇，兩次膝蓋大傷讓他慢慢淡出了巨星行列，但羅斯最難能可貴的，便是他從年少輕狂到少年老成的急速切換，放下身價，來到詹皇身旁全心做一名替補，即使在騎士他沒能迎來第二春，但從那開始，他的第二段精彩故事就已經寫好了序言。

在詹姆斯的絕對統治力面前，有人選擇了臣服，但也有人選擇了規避，在泡椒在步行者的最後一年，當再次被詹姆斯的球隊淘汰之後，喬治沮喪的表示連續多年被一個人擊敗，這太讓人失望了，不到一個月，泡椒遠走西部，在雷霆打出了新天地，但世事難料，如今他和詹皇又處在了同一座城市，季後賽中他又將繼續面對這座他一直都沒有翻過去的大山。

近日，NBA名記Ramona Shelburne在節目中透露，湖人球員霍華德當初與球隊簽下非保障合同的提議，是他自己主動提出的，與上面幾個人都不同，霍華德曾經單核帶隊擊敗過詹姆斯的球隊，他有理由繼續不低頭，但如今卻選擇放棄保障底薪來投靠，也許每一個人在連續多年敗在同一人手下之後，會做出不同的選擇，或臣服，或逃避，或堅持，或投靠，但無論如何，變得更加成熟，更渴望榮譽的他們都做出了正確的決定。

（殘艦之毅）
