戴維斯抱團？經紀人回擊質疑：他在鵜鶘每晚能得40分 卻無法奪冠

2019-12-18 728條評論 2,221次閱讀 923人點贊

里奇·保羅是當今NBA最有名望的球員經紀人之一，詹姆斯和戴維斯等球星都是他的客戶。今年休賽期，在保羅的操作下，戴維斯終於如願加盟湖人聯手詹姆斯。

如今，湖人的戰績領跑全聯盟，但也有不少人質疑戴維斯離開鵜鶘的做法很不職業，有「抱大腿」的嫌疑。

在接受《GQ》記者克萊·斯基普採訪時，保羅為自己的客戶戴維斯進行了辯解，他表示籃球是戴維斯從事的職業，而球迷們僅從娛樂的角度看問題，雙方的視角是不同的。

「他們不了解AD的處境，鵜鶘已經很多年沒有打出好成績了，他需要改變一下處境。很多時候，周邊的人會發出質疑，因為這是籃球比賽。從小到大，我都認為籃球是種娛樂，直到你進入籃球行業內，才發現這是一項幾十億美元的生意。」

戴維斯為鵜鶘效力了7個賽季，保羅認為戴維斯是時候改變了，加盟湖人是正確的選擇。他還指出，球隊不該總是掌握著權力，處境艱難的球星同樣有權做出改變。

「從AD的角度來看，他只是覺得是時候改變了。這沒什麼不對的，作為一名球員，人們不了解他的生活。他每天晚上都能拿到40分，但他不能去追逐總冠軍，因為球隊沒那種能力。如果你足夠強大，我敢打賭你一定會選擇改變。要麼改變現狀，要麼換一支球隊。」
