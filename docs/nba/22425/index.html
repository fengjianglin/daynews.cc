尼克斯所有球員都可交易 計劃簽下猛龍隊魂 以便爭奪字母哥

2019-12-18 834條評論 4,243次閱讀 1620人點贊

尼克斯在今年夏天接連錯過安東尼·戴維斯、杜蘭特和歐文等巨星之後，簽署了馬庫斯·莫里斯、朱利葉斯·蘭德爾、鮑比·波蒂斯，韋恩·埃林頓、塔伊·吉布森一大批老將來填補陣容。雖然給了他們高出市場價的合同，但是為了保持陣容的操作靈活性，這些人只有一年的合同是可保障的。如今，這些僱傭兵已經被尼克斯擺上貨架，如果籌碼合適隨時都可以被交易走。

混亂的尼克斯隊中並沒有非賣品，他們的年輕球員同樣可供交易選擇，比如凱文·諾克斯、米切爾·羅賓遜、伊納斯·布拉茲代基斯，甚至是RJ·巴雷特。但是外界看來，紐約陣中唯一具有交易價值的只有馬庫斯·莫里斯。東部某球隊的一位高管在談到莫里斯時說道：”馬庫斯對於任何一個冠軍競爭者來說都是有價值的，他很堅韌，也可以投籃，並且有豐富的季後賽經驗，只是不要指望馬刺隊會再追隨他了。”

由於揚尼斯·阿德托昆博的合同將在2021年到期，尼克斯將目標對準了2021年。他們有足夠的薪金空間簽下多名球星，所以不會願意在2020-21賽季之後再增加任何薪水。紐約尼克斯計劃先從管理層下手，他們已經裁掉了主教練費茲戴爾。而據美國《體育》雜誌的弗蘭克·伊索拉爆料，尼克斯前線高管史蒂夫·米爾斯以及斯科特·佩里的職位也岌岌可危。多倫多猛龍隊總裁馬賽·烏傑里是尼克斯當前的首要目標，後者曾運作倫納德的交易，幫助猛龍隊獲得隊史首個總冠軍。

烏傑里已經拒絕了猛龍隊的續約，顯然他對於大蘋果城更有興趣。尼克斯的一些高層也認為獲得烏傑里是拿下揚尼斯-阿德托昆博的第一步，他們計劃以烏傑里為核心搭建制服組，改善球隊管理層氛圍，再由烏傑里尋找合適的主教練，爭取在明年夏天贏下字母哥的爭奪戰。
