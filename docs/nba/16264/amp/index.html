湖人從落後14分到逆戰反超發生了什麼？有誰注意，他又上場了

天天要聞
2019-12-14

北京時間12月14日，NBA常規賽繼續進行，今天洛杉磯湖人客場挑戰邁阿密熱火，全場比賽，主場作戰的熱火表現出色，他們上半場利用整體防守迫使湖人出現失誤，隨後利用反擊快攻一度將領先優勢擴大到14分；而到了下半場，湖人在詹眉組合帶領下將內線優勢發揮的淋漓盡致，最終以113-110逆轉熱火再次豪取6連勝。

上半場比賽，熱火把他們的主場優勢發揮到了極致，在主場球迷響徹全場的歡呼聲中，巴特勒帶領隊友在防守端展現出了極強的韌性，熱火全場展現出來的高強度身體對抗讓湖人很不適應，他們不斷的迫使湖人出現失誤，全場比賽，湖人失誤多達19次，三分是14投僅3中，而熱火利用這些失誤多次打出快攻反擊。

全場比賽，湖人失誤多達19次，而主隊則只有8次，湖人的失誤數是對手的2倍還多，數據顯示，熱火也很好地利用了這一點，他們利用湖人的失誤多次打出快攻反擊，得分高達29分，而湖人利用熱火失誤反擊得分則只有10分，兩者間的差距差不多是3倍，而這也是熱火上半場一度取得14分領先的關鍵。

到了下半場，湖人開始展現出他們的內線高度優勢，安東尼-戴維斯和麥基連續封蓋對手上籃，全場比賽，湖人僅是這兩個內線雙塔就送出了8次封蓋，而熱火全隊相加也只有2次；除此以外，勒布朗-詹姆斯也開啟了進攻模式，他連突帶傳帶領湖人打出一波14-5的高潮抹平分差，並且逐漸的掌控了場上局勢。

第四節比賽，詹姆斯多次單打瓊斯，由於他已經連續投進幾個三分，熱火隊只能無奈的採取包夾策略，此時詹姆斯敏銳的判斷出了對手的防守，不斷的給隊友送出助攻，下半場比賽，湖人投中了9記三分，再加上濃眉、麥基和霍華德不斷的衝擊前場籃板，湖人最終驚險的逆戰取勝，全場比賽，湖人搶了50個籃板，比熱火多了16個，前場籃板更是搶了14個，而熱火只有5個。

湖人從落後14分到逆戰反超發生了什麼？有誰注意，他又上場了！在第四節還剩下6分多鐘的時候，詹姆斯幫助球隊確立了6分的領先優勢，隨後下場休息，然而僅僅過了兩個回合，熱火馬上捉住機會不斷的縮小分差，眼看局勢不對沃格爾只能馬上派上詹姆斯，而他回來後效果也是立竿見影，幾次低位單打瓊斯幫助湖人最終鎖定勝局。

縱觀全場，主場作戰的熱火全民皆兵的頑強防守讓人稱讚，而湖人也將他們的雙超巨和內線高度優勢完全發揮了出來，但不得不說，詹姆斯對於湖人太重要了，第四節只休息了一分多鐘，局勢差點就被逆戰了，有球迷調侃說，沃格爾終於學到了泰倫盧絕招，遇事不決上老詹，累了就叫著暫停讓他休息幾十秒！
