掘金主場大勝魔術！默里完爆狀元富爾茨，全隊創造1項驚人紀錄

2019-12-19 814條評論 2,624次閱讀 457人點贊

北京時間12月19日，近來勢頭不錯的丹佛掘金隊在主場迎來奧蘭多魔術隊的挑戰，掘金隊作為西部近幾年來一支實力不俗的隊伍，在高原主場打比賽更是勝率極高。話雖然這麼說，但是本場比賽奧蘭多魔術隊顯得異常堅韌，全隊有4人得分上雙，狀元秀富爾茨在30分鐘的出場時間內也有10分入賬，對於一個剛剛從傷病中回歸的新秀來說表現可謂是中規中矩。

而對面丹佛掘金隊的控衛賈馬爾-默里，顯然是一個狠角色，這幾年來默里在丹佛掘金隊的作用顯得越來越重要，個人的得分和表現也逐漸穩定下來，今天的默里出場36分，得到了33分5籃板1助攻，命中率過五成，表現十分搶眼，也可以說是完爆了對面魔術隊富爾茨的表現。

而本場比賽，丹佛掘金雖然坐鎮主場並且面對實力並不太強的奧蘭多魔術隊，比賽前半程卻顯得有些吃力，在第三節開始一段時間裡，魔術隊一度領先丹佛掘金隊19分，場上比分為50:69，然而接下來比賽形式卻發生了扭轉，掘金隊在隨後的6分鐘內打出了24:0的超強攻擊波，而這一波攻擊波直接為後來掘金隊的勝利奠定了基礎，也不得不讓人感嘆掘金的得分爆發力。

24:0的攻擊波，想想就有些驚人，不僅要自己拿到24分，而且還讓對手一分未得，這是攻防兩端均達到了極致才能產生的效果，掘金上一次打出24:0還要追溯到2016年11月26日對陣芝加哥公牛隊的比賽，那時他們在第一節17-31落後，隨後在第二節比賽當中以41:31反超，而這兩次攻擊波也創造了掘金隊史的紀錄。

目前掘進隊已經取得了4連勝，接下來又將挑戰森林狼和洛杉磯湖人隊，或許又會是兩場惡戰了。
