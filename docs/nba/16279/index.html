雙星轟61+19！詹姆斯賽後道歉，並感謝戴維斯，韋德也忍不住開口

2019-12-14 292條評論 4,259次閱讀 2569人點贊

在今天結束的一場比賽中，湖人客場以113-110力克熱火，客場取得13連勝。而熱火的主場11連勝被終結了。今天比賽詹姆斯和戴維斯聯手砍下了61+19+13的數據，戴維斯得到33分10籃板，詹姆斯28分9籃板12助攻並出現了8次失誤。

可以說上半場的詹姆斯表現是極其糟糕的，但下半場詹姆斯恢復正常的水準，調整能力還是出色。今天比賽過程熱火確實堅韌，在巴特勒的率領下太強勢了。

從首節開始熱火的倫納德就命中三分，巴特勒連續攻擊籃筐+中距離。詹姆斯和麥基也不斷回應。首節湖人以30-29領先。第二節熱火反動猛攻，納恩持球一個加速突破到籃下得分，瓊斯和阿德巴約也勇不可當。分差被熱火拉開到9分。

詹姆斯和戴維斯開始發力，兩人合力砍下了6分幫助球隊止血。但巴特勒又打瘋了，一個衝擊暴扣霸氣側漏。帶隊打出高潮，熱火領先達到14分。關鍵時刻波普和戴維斯命中2記三分止血。

半場湖人落後熱火8分，下半場湖人終於開始發力。波普中距離命中，戴維斯中距離命中，格林三分也抬手就射，詹姆斯和戴維斯再次聯手命中2記三分。湖人打出了一波29-8的攻擊波。直接從落後8分到反超熱火11分。但巴特勒再次站了出來，強行將分差迫近到3分進入末節。

第四節詹姆斯和霍華德的二人轉主宰了比賽，他們再次將分差拉開到9分。之後雙方陷入拉鋸戰。在最後三分鐘湖人仍然領先8分，納恩和羅賓遜連續三分讓湖人嚇出一身冷汗。

但關鍵時刻戴維斯表現還是令人驚嘆，他三分命中，又命中關鍵罰球，可太累了。幸好巴特勒最後三分不進，湖人收下比賽。

今天巴特勒儘力了，他得到23分4搶斷還是無力回天。賽後詹姆斯心情不錯，在談到自己今天表現的時候，詹姆斯還公開道歉對自己上半場的表現感到不滿意，並感謝了戴維斯對他的支持。

詹姆斯說道：「上半場我的表現很糟糕，我搞砸了一切對此很抱歉。但下半場我就是在努力調整心態，並打好下半場。我的隊友們一直在鼓勵我，他們告訴我要堅持做我想做的事情。戴維斯在支持我，考辛斯也在支持我，他們都在告訴我，沒事你可以慢慢去調整，我們相信你。」

這就是湖人的氛圍，人們都說詹姆斯廢隊友。什麼隊友不開心什麼的。可本賽季湖人的氛圍說明了一切，戴維斯和詹姆斯如魚得水。反觀騎士現在就不行了，湯普怒噴主教練，樂福想加盟開拓者。

湖人和熱火的表現，也讓韋德非常興奮。在賽後韋德在社交媒體上寫道：「湖人對陣熱火的比賽，是我這賽季看過最好看的比賽。在十二月中旬就感覺像是季後賽了！」

韋德的話不僅僅是對熱火的稱讚，也是對湖人所展現的實力感到驚訝。
