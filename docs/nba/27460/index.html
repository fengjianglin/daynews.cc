如此不錯的球員，森林狼卻不要！難怪他們是扶不起的阿斗

2019-12-23 1,869條評論 3,276次閱讀 899人點贊

119:107！！！公牛隊輕鬆擊敗了活塞隊。目前公牛隊收穫了兩連勝，球隊的戰績是12勝19負。這成績排在東部賽區第十。這賽季的公牛隊看起來不想擺爛了，球隊想衝擊一下季後賽。活塞隊目前遭遇了4連敗，球隊的戰績是11勝19負。這成績排在東部賽區第一十二。這支活塞隊在本賽季至今的表現是讓外界很失望的，本以為他們會比上賽季好，沒想到下滑的嚴重。球隊核心大將格里芬的狀態更是讓人很難受。上賽季的季後賽，他就不應該打封閉針上場。現在，他正在承擔著打封閉針上場所帶來的重大負面影響。格里芬很有可能就此沒有了全明星級別的實力。為啥格里芬會這麼傻呢？聯盟當中打封閉針上場參加比賽之後出現嚴重負面的影響比比皆是，他為何還要明知不可為而為之。

​這場比賽，公牛隊的大將拉文的表現很出色。他出戰37分鐘，投籃15中9，三分7中5，罰球14中10，砍下全場最高33分，外加5籃板5助攻1搶斷。而且在比賽關鍵時刻，拉文連續命中兩記不講理三分瞬間殺死比賽。這位球員在公牛隊表現得風生水起，他在球隊當中已有過多次這樣精彩的表現。看著現在的拉文，再看看森林狼隊那兩位狀元，兩位狀元讓人很失望。尤其是維金斯，他仍舊不改養生曼巴的習性，該怎麼玩就該怎麼玩。比賽的輸贏和他有什麼關係呢？

​想當初，巴特勒在森林狼隊時，他還把球隊帶入了季後賽。但由於巴特勒在森林狼隊球員當中格格不入，而且球隊管理層也不支持巴特勒。最終巴特勒被交易出去。現在巴特勒在熱火隊表現的很好，他帶領球隊也收穫了很好的戰績。再看看森林狼，目前他們已經10連敗了。本賽季的他們也很有可能無季後賽。球隊當中的兩位當家數據看起來不錯，但球隊始終無法贏球。當初，拉文就應該被球隊留下來，維金斯這樣的球員就應該被交易。可惜森林狼隊管理層真是不行啊。

​希望拉文能夠在公牛隊表現得越來越好，帶領球隊打入季後賽。也希望他能夠入選聯盟的全明星賽，入選聯盟的最佳陣容。
