丁威迪31+7籃網加時送鵜鶘13連敗！英格拉姆23中7

2019-12-18 113條評論 2,184次閱讀 824人點贊

NBA常規賽12月18日繼續進行，最終籃網客場以108-101戰勝鵜鶘。

首節開始，雙方4-4平後丁威迪上籃後又助攻哈里斯上籃幫助籃網領先。鵜鶘這邊霍樂迪同樣得分、助攻扳平！雙方手感不佳，頻頻打鐵中斷斷續續拿分，錢德勒三分、拋投幫助籃網稍稍領先，不過隨後連續強攻籃下得分，首節戰罷雙方打成21-21平。

次節上來，互有得分後哈特連續飆中三記三分領先！但是隨後7分半鐘鵜鶘再無得分！籃網這邊丁威迪連續得分打出一波15-0反超！半場戰罷籃網以42-35領先7分。

易邊再戰，鵜鶘開始反擊，鮑爾連續得分率隊打出一波12-3反超2分！哈里斯三分、丁威迪跳投再度領先。但是英格拉姆連續造犯規，鵜鶘又打出12-4再度反超！互有得分後三節戰罷，鵜鶘以64-61領先3分。

末節開打，霍樂迪連續籃下打進拉開分差！但是籃網外線發威接連命中三分扳平！互有得分後丁威迪連續得分、助攻幫助籃網反超！費沃斯快速拿下3分，但是哈里斯和丁威迪又連續上籃得手一波7-2拉開分差！但是霍樂迪上籃得手又助攻鮑爾命中三分追至1分！互有得分後霍樂迪打成2+1反超！！阿倫暴扣扳平！隨後雙方接連打鐵，比賽進入加時。

加時賽開打，丁威迪上籃、罰球連拿4分！互有得分後英格拉姆罰中1球，丁威迪上籃再中領先到5分！鵜鶘連續投丟，坦普爾再罰中2球鎖定勝局！

籃網這邊，丁威迪31分7助，哈里斯24分，阿倫12分14板

鵜鶘方面，哈特14分9板，英格拉姆23投7僅中拿到22分11板，霍樂迪21分
