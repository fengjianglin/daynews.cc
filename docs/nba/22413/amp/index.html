本準備抱大腿的，沒想到自己卻成為大腿了，這些球星太難了！

天天要聞
2019-12-19

老兵不死，只是凋零。這賽季的聯盟，發生了一幕幕感人的畫面，讓人不僅感慨體育的魅力，我們盤點一下，本來只是想打球繼續自己的籃球夢，報大腿來的，但沒想到沒抱成，自己卻變成大腿了。實在太難了。

霍華德04年的狀元，司職中鋒，身體素質超強，打球暴力。因此被人稱為「魔獸」，職業生涯得到無數榮譽，由於自己的性格愛玩的問題，生涯後期幾乎沒有球可打，沒人能相信他能夠改變自己性格，也沒有球隊願意要他，但是湖人給他機會了，他也沒有辜負球隊的期望，一次次征服球場，征服那些看不好他的人。賽季至今，很多關鍵時刻都能夠站出來，本來只是想抱詹姆斯的大腿，得到一做冠軍，沒想到自己卻成為大腿。也恭喜魔獸找回自己，能夠在湖人開心比賽。

安東尼03年第三順位被選中，司職小前鋒，職業生涯效力於掘金，紐約，雷霆，火箭，現在效力於開拓者。作為03黃金一代的代表數人物，他生涯前期一直是球隊的核心人物，打法多樣，被稱為進攻萬花筒，由於打法比較喜歡單打，因此帶隊能力一直被詬病。安東尼的轉折點應該是離開紐約，加盟雷霆沒有成功奪冠，之後加入火箭隊。後期被火箭裁掉，導致自己之後無球可打，但是他沒有放棄過打球的夢想，一直想通過自己的努力得到球隊認可，功夫不負有心人，終於，時隔300多天後進入聯盟，加盟開拓者，但是沒想到，自己卻成為球隊的大腿了，讓開拓者現在走出泥潭。

羅斯08年狀元，生涯前期打球勁爆，最年輕的MVP。由於自己的打法，傷病很快就有了，但是人們永遠沒忘記這個風城玫瑰，他也沒有放棄過自己，一直與傷病作鬥爭，誰也沒見過巔峰的玫瑰，也不知道能夠達到什麼地步。但是現在能夠看到他打球已經很知足了，他後期幫助森林狼進入季後賽，關鍵時刻的能力，也成為球隊的大腿。現在效力於活塞，也成為球隊不可缺少的人，關鍵時刻絕不手軟。

還有那些球員，本來只想好好打球，現在卻成為球隊的大腿？
