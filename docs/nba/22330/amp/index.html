綠軍記者Q&A：認為海沃德會和球隊續約 安吉最喜歡羅齊爾

天天要聞
2019-12-19

直播吧12月19日訊 今日，凱爾特人記者Jay King在美媒The Athletic上和球迷進行互動，聊到了一些關於球隊的話題。現在，就隨小編一起來看看。

球迷：諾倫斯-諾埃爾會是一個選擇么？顯然想要像史蒂文-亞當斯和邁爾斯-特納這樣的球員得付出很大的代價，不過諾埃爾似乎是一個很有吸引力的替補內線。

JK：我只是不確定諾埃爾對凱爾特人而言是不是一個升級。不過，他的技能很誘人。

————————

球迷：你能預見像霍姆斯、戴德蒙甚至特納這樣的大個子球員加進球隊么？

JK：大個子的獵取很奇怪。是的，凱爾特人可能會使用尺寸更大的球員。但由於他們的薪資情況，這就是一個不容易構建的交易，從而來讓他們得到一個重要的升級。事實上，大多數晚上他們的表現都很不錯，丹尼爾-泰斯很棒。

————————

球迷：TT有機會被騎士買斷么？因為我對放棄斯瑪特、杰倫或者前五中任何一個去換中鋒都不太樂意。

JK：TT會是一個很好的選擇。我不知道他被買斷的可能性有多大，不過他的籃板和防守會對凱爾特人很有幫助。

————————

球迷：海沃德，扣動扳機（交易他）？

JK：海沃德打得太好了。這就是為什麼很難找到一筆適合凱爾特人的大交易的部分原因：大多數交易都需要他們最好的球衣之一，而且他們最好的球員都很出色。

—————————

球迷：海沃德會和凱爾特人續約么？（還剩一年半合同）

JK：是的。我認為這對他而言是個好的情況，他喜歡波士頓（凱爾特人）的人們，雖然我也曾就關於霍福德說過這樣的話。

————————

球迷：你對海沃德的腳傷有長期的擔心么？不要像史蒂文斯說的「海沃德的腳傷已經困擾他一段時間」。

JK：事實上，我不知道。凱爾特人在投籃訓練時並沒有提醒我們海沃德在處理傷痛，所以今天晚些時候我會知道更多信息。

————————

球迷：唐斯有可能加入凱爾特人和杰倫、塔圖姆並肩作戰么？如果我們要給塔圖姆頂薪的話能負擔得起唐斯的合同么？我從來都不是濃眉交易的支持者，因為我不想犧牲杰倫和塔圖姆或者斯瑪特，不過你認為我們能在不出他們仨的情況交易得到唐斯么？我們需要斯瑪特來敲打唐斯的防守意識，所以不要把他包括在這一樁假設的交易之中，儘管他已經捲入了交易傳聞之中。你認為我們也會試圖留住格蘭特-威廉姆斯么？我覺得他會是凱爾特人未來多年的重要成員。

JK：直接提到「凱爾特人最終能得到唐斯么？」我尊重它，這是你的大夢想。你想讓凱爾特人交易得到NBA最好的年輕球員之一，而且不放棄他們的任何主要球員。

唐斯的合同到2023-24賽季才到期，所以追求可能需要持續數年，森林狼應該會儘可能地與之合作。如果他們最終沒能創造出一個贏家，唐斯可能會在合同到期後離隊…在那之前，森林狼可能需要交易他。也許丹尼-安吉會在那時排上隊，或許他有足夠的資產和年輕人來發動攻勢。在這之前，任何事情都有可能發生，所以我不會花太多時間為這種可能性感到興奮。

————————

球迷：你認為這支球隊里安吉最喜歡的球員是誰？

JK：羅齊爾（現黃蜂球員）。

————————

球迷：我說塔圖姆從上賽季到現在在進攻上矯枉過正是不是不公平？上賽季，他有很多單打還有長兩分；現在則是更多的三分和上籃，但他仍然覺得他們是以「好東西」為代價的。看了這個賽季的塔圖姆，感覺他在努力尋求合理的出手，也正在實現。

JK：塔圖姆想要改變他的投籃方式，其中一個羞羞的小秘密就是他人沒有很好的投籃結合。毫無疑問，他已經減少了一些長兩分，增加了三分球出手，不過他依舊需要學會如何為自己創造最好的投籃機會。比如，他的拋投怎麼就沒怎麼進呢。

————————

球迷：如果現在的凱爾特人球員有望未來被退役球衣，你覺得誰最有可能？

JK：我會站塔圖姆這邊。他很年輕，可能會在波士頓有很長的職業生涯。但那也是一段很長很長的路。

————————

球迷：我是不是瘋了？我覺得完全健康的凱爾特人這賽季就能奪得總冠軍！

JK：我覺得這個賽季的東部是開放的。東部聯盟每支球隊都有自己的缺陷。這（凱爾特人奪冠）不是不可能的，不過我認為其他球隊（雄鹿和76人）是東部的熱門，而湖人和快船在西部也很難被擊敗。

（碧塔海的魚）
