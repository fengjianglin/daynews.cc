他帶隊背靠背奪NBA總冠軍 小時因體虛放棄籃球 但被媽媽無情扇醒

天天要聞
2019-12-25

在我們看來，NBA球星各個都是光鮮亮麗的，他們靠著自己喜歡做的事情和方式去掙到很多很多的錢，這也是很多球迷和普通人羨慕的地方，但是NBA球星背後真的沒有心酸的故事和往事嗎？如果你這麼想，那就大錯特錯了，NBA每個球員能夠擁有如今的成就，背後都有常人想像不到的心酸！

NBA有這樣一個男人！他得分能力超強，被人們稱為死神！和兄弟一起打球多年，為了總冠軍離開母隊，最終他收穫到了2屆NBA總冠軍以及2座總冠軍MVP的獎盃！這個人就是凱文·杜蘭特，你可能會說杜蘭特的天賦肉眼可見，小前鋒位置讓人羨慕的身高和速度，聯盟頂級的臂展和投射能力！杜蘭特的身體素質並不是最出眾的，但這足以讓杜蘭特成為NBA中的超級巨星！

你可能會說這就是天賦吧！但背後球迷不知道的是：杜蘭特小時候曾經因為打籃球偷懶的事情被母親打了一耳光，而那一耳光也讓杜蘭特與籃球有著剪不斷的聯繫！

小時候的杜蘭特因為身體虛弱而不想打籃球被母親打了一耳光，他的母親沒有別的意思，只是希望杜蘭特可以依靠籃球來改變自己的命運，畢竟他們生活的條件非常的艱苦！

從那以後杜蘭特也是不負母親的期望，努力訓練，並且以邁克爾·喬丹為自己的榜樣！11歲那年，杜蘭特帶領馬里蘭州普林斯喬治縣的美洲虎隊拿到全國冠軍杜蘭特在決賽下半場更是拿到了18分為比賽奠定的勝利奠定了基礎！

也是從那場比賽之後，杜蘭特告訴自己的媽媽我會好好的打籃球，並且有朝一日我要打進NBA！高中時期刻苦的訓練讓杜蘭特非常不適應，也是他的媽媽幫他度過難關！

這也是為什麼杜蘭特在當年獲得常規賽MVP的時候，頒獎典禮上說出：我不是MVP，媽媽你才是真正的MVP！杜蘭特的今天是他的媽媽給予的，並且改變杜蘭特的人也是他的媽媽，如果沒有兒時的一巴掌，杜蘭特可能現在並不會出現在NBA的賽場！

勇士時期的杜蘭特遭到很多球迷的謾罵，但是杜蘭特在採訪中曾告訴記者，媽媽告訴過我：你要清楚和明白自己想要什麼！曾幾何時，杜蘭特在獲得FMVP的時候，第一個擁抱的人永遠是自己的媽媽！

祝願杜蘭特可以早日康復，球迷期待著你的涅槃重生！
