4人20+，打不贏8人上雙！魚腩12連敗卻扎心：擺爛也比不過勇士

2019-12-17 1,661條評論 2,819次閱讀 1084人點贊

北京時間12月16日，nba常規賽繼續正常進行。在之前結束的一場比賽中，英格拉姆領銜的青年軍鵜鶘在主場以119-130的比分不敵魔術，遭遇尷尬的十二連敗，而這十二連敗也刷新了鵜鶘隊史最長的連敗紀錄。

此役，鵜鶘核心英格拉姆發揮出色，全場得到21分6籃板6助攻的不錯數據，霍樂迪也有29分4籃板8助攻進帳。在二人的帶領下，鵜鶘五人得分上雙，包括哈特、雷迪克在內的多名球員都有不錯發揮。但奈何魔術全隊整體表現更加出色，八人得分上雙的他們結束連敗的同時也送給了鵜鶘十二連敗。

本場比賽過後，鵜鶘戰績也是來到了6勝21負，穩居西部墊底之列的同時，已經落後西部第八的太陽6個勝場之多，可以說季後賽希望渺茫。儘管延續著聯盟最長的12連敗，但鵜鶘仍然不是全聯盟戰績最差的球隊。目前，傳統強隊勇士因為傷病和球員轉會等原因元氣大傷，以5勝22負的戰績排在聯盟倒一的位置，大有衝擊狀元簽之勢，12連敗的鵜鶘在勇士面前也只能靠邊站！

不得不說，鵜鶘隊的運氣是真夠差的。休賽期，球隊核心戴維斯遠走洛杉磯，新選的天才狀元錫安截至目前為止還是一場未打，更有消息曝出這名被寄予厚望的天才少年極有可能會缺席新賽季的所有比賽！本寄希望在英格拉姆、錫安等人的帶領下重返季後賽的鵜鶘就此夢碎，只能繼續去爭奪更高的樂透簽位。

然而，狀元簽的爭奪同樣激烈，包括勇士、騎士、尼克斯、老鷹在內的多支球隊都對此虎視眈眈，鵜鶘想要拿下狀元簽，並沒有想像中那麼簡單。儘管如此，鵜鶘仍然擁有著美好的未來，隨著英格拉姆等人的快速成長，未來幾個賽季他們將會是聯盟的一支勁旅，我們能夠做的就是耐心等待。
