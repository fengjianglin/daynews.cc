騎士老闆:當年湖人想用科比換詹姆斯，但我覺得非常不合理。

2019-12-08 0條評論 13次閱讀 0人點贊

北京時間12月8日，今日騎士隊老闆丹.吉爾伯特在接受《克利夫蘭老實人報》記者Joey Morona採訪時，透漏了在2007年時，湖人隊管理層想用科比布萊恩特來交易勒布朗詹姆斯一事，吉爾伯特表示當年科比要求湖人交易自己的事情眾人皆知，而且人們都普遍的認為科比差一點被交易到公牛。但實際上當時騎士才是科比最可能的下家，只不過被自己給否決了，因為自己告訴當時湖人的總經理庫普切克，科比還不配單換詹姆斯。

對此，吉爾伯特說道：「科比在2007年要求湖人交易自己的事情每個人都知道，而且大家普遍都認為他差一點被交易到芝加哥，但實際上我們才是科比最有可能的下家。我不清楚他們的球隊具體發生了什麼事情，當時我正在飛往巴爾的摩的飛機上，我接到了米奇（庫普切克）打來的電話，我很驚訝他沒有給丹尼（費里，當年騎士的總經理）打電話而是選擇直接和我通話，電話接通後他就直接告訴我：讓我們來談談關於勒布朗和科比的交易吧。」

「他把一系列的交易方案和備案都告訴了我，並且對我表達了他們的誠意和決心。這讓我非常吃驚，因為通常在談判桌上，你不會把備選方案告訴對方，這會讓你在交易中陷入被動，但是很顯然，米奇渴望得到勒布朗，同時又急切的想要送走科比。」吉爾伯特說道。「但即便米奇如何表達他的誠意，我們能在這筆交易中得到的只有科比一個球員，所以我毫不猶豫的告訴他：我了解到你的誠心，親愛的米奇，但是科比還無法單換勒布朗，他是我們的非賣品，這個聯盟沒有任何球員夠資格單換他。」吉爾伯特說道。

07的科比和詹姆斯都處在巔峰期，並且詹姆斯還單核率領騎士隊進入了總決賽，雖然被老辣的馬刺橫掃出局，但當年的詹姆斯儼然就是這個聯盟的未來之星！而科比在2004年-2007年這段時間，年年季後賽一輪游，加上年齡關係，吉爾伯特拒絕湖人的請求也就在情理之中了。不過隨後科比就得到了二連冠，帶領湖人重新走上了巔峰，而詹姆斯在10年離開騎士，但是在16年為騎士帶來第一座總冠軍獎盃，所以吉爾伯特當年的決定是否正確，那就是仁者見仁智者見智了。
