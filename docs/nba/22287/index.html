本賽季4大最佳“退步”獎，一老將悲催，3000萬先生讓人失望。

2019-12-20 144條評論 1,190次閱讀 114人點贊

賽季至今，NBA很多球員都找到了自己的狀態，他們很多人都打出了非常優異的表現，讓人印象深刻。不過，在同一時間段，一些球員的狀態也大不如前，有的甚至和上個賽季判若兩人，下面就讓我們一起來看一下今年的4大退步球員。

4.鮑爾

人們再也不願意花更多的精力去關注鮑爾了，鮑爾的父親也不願用他那張大嘴了博得公眾的注意力，因為現在的鮑爾對於鵜鶘來說猶如一個雞肋，棄用太可惜，但在場上他無法給球隊帶來十足的幫助。

想看一個新秀的成色一般要三年時間，生涯第三年，在球隊更年輕化，機會較湖人更多的鵜鶘，鮑爾非但沒有進步，反而更加迷失。10+4.4+5.5的數據看起來足夠全能，但背後是38%+34%的命中率和每場一次的罰球，鮑爾在場上打得沒有絲毫侵略性，他無法突進內線造殺傷，只靠組織和防守無法將他的價值最大化，在場上，鮑爾無法給鵜鶘穩定的火力支援，他那飄忽不定的投籃讓球隊頭疼不已。在這支年輕的球隊同樣有很多有天賦的年輕球員，鮑爾還能有多少機會呢？

3.蘭德爾

又是一位湖人球員，蘭德爾在上賽季打出了現象級的表現，每場出手不到15次就能拿下21分，打得非常高效。本賽季在尼克斯蘭德爾簽下了大合同，但在上場時間和球隊地位提升的情況下蘭德爾的發揮反而退步，得分、命中率、蓋帽全面下滑，在尼克斯，蘭德爾本來被寄予厚望，人們希望蘭德爾能承擔起球隊的責任，但就目前來看，蘭德爾的經驗和實力還不足以承擔這一責任，也許二當家才是他最好的歸宿。

2.格林

格林似乎在總決賽輸給騎士後一直在退步，當然我們指的是進攻端。可本賽季，格林在穩定維持了自己糟糕的進攻表現外，防守穩定性還進一步下滑。蓋帽近6年來首次跌破1次，籃板不到6個，要知道當年他可是DPOY。

勇士一開始也沒有太在意格林的進攻，畢竟球隊里有庫里和湯普森，可是當這兩尊天神倒下後，勇士的進攻變得搖搖欲墜。而這時候人們發現，這個入選過全明星的球員居然沒有自主得分的能力。他的三分命中率已經連續第二年不足30%，勇士的外線空間變得越發狹窄，但格林依舊有「廣闊」的空間，因為對手敢放投他。

問題是，即使下賽季庫里和湯普森等人全員健康出戰，勇士又能走多遠呢？如果像上賽季一樣在場上面臨4打5的局面，勇士的進攻必然是步履維艱。

1.康利

賽季開始前，人們對爵士簽約康利讚賞有加，爵士近幾年來的表現有目共睹，防守端他們堅如磐石，但進攻卻麻煩不斷。只靠米切爾一個人無法支撐起球隊的進攻，支配球球員的缺失註定了爵士的上限。因此，當組織能力出色，進攻穩定的康利加盟爵士後，人們相信在新賽季爵士能打出非常優秀的表現。

新賽季至今，爵士106.4的失分位於聯盟第8，這的確是一個出色的數據，但進攻端他們的發揮相當糟糕，107的場均得分排在聯盟後半段，與之對應的是康利的糟糕進攻。本賽季康利場均得分不到14分，命中率不到37%，罰球也跌破80%。更要命的是，康利在場上的狀態起伏極大，他可以在對猛龍的比賽砍下20分，也可以在對老東家灰熊時「砍下」8分，而之前的兩場比賽，他的得分也沒有上雙。

人們不清楚康利本賽季為何會有這種糟糕的發揮，康利一直被認為是聯盟的頂級控衛，32歲的他看起來也沒有明顯的下滑趨勢。唯一可以解釋的就是自己的狀態了
