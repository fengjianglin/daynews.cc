“活塞五虎”以防守著稱，“死亡五小”以進攻聞名，哪個更厲害？

2019-12-21 3,179條評論 4,466次閱讀 995人點贊

​

作為球迷，對自己心目中喜歡的球星總有一個排名，對球隊實力和組合的實力的比較永遠都有話題度。那麼如果讓2004年的「活塞五虎」與2018年勇士「死亡五小」打一場？究竟誰能奪得勝利？

讓我們先看看活塞五虎都有誰，比盧普斯，他首輪探花秀、總決賽MVP、5次入選NBA全明星陣容。拉希德·華萊士，他首輪四號位，4次入選NBA全明星陣容。理查德·漢密爾頓，他首輪7號秀，3次入選NBA全明星，被稱為「中投王」。普林斯，他首輪23號秀，綽號「小王子」，4次入選NBA最佳防守陣容二陣。

​

勇士方面的「死亡五小」大家都熟悉不過，庫里，他新秀賽季入選最佳新秀第一陣，率隊三次獲得NBA總冠軍，兩次榮膺常規賽MVP，6次入選最佳陣容、6次入選全明星賽西部首發陣容。克萊·湯普森，他是頂級的三分射手，全能型球員，5次入選全明星陣容。伊戈達拉，他彈跳驚人，超人的臂展。格林，他是頂級的鋒線防守者、籃板出色。杜蘭特，他是NBA得分王與常規賽MVP、二次奪得NBA全明星MVP。

​

單從陣容上來看勇士的「死亡五小」肯定能贏過「活塞五虎」，杜蘭特有死神的稱號，得分能力能進入歷史前五，加上歷史級別的神射手庫里，這樣的進攻能力火力誰又能擋得住？

當然了當年的那支活塞，只有經歷過的人才知道有多恐怖，很少有球隊能夠破解以防守著稱的「活塞五虎」的鐵桶陣。

​

下面來看看一對一的pk，庫里vs比盧普斯，小學生明顯更有優勢，比盧普斯或許能藉助身體優勢限制庫里的突破，但是絕對防不住庫里的三分球。湯姆森vs漢密爾頓，平分，兩人的實力相近，擅長無球跑動，空位投籃。伊戈達拉vs拉希德華萊士，這對組合實力差距頗大，華萊士可能會毫不留情的把伊戈達拉打成塞子。

本華萊士vs格林，這隊也不用比，格林完敗。杜蘭特vs普林斯，雖說普林斯有身高也有臂展，雖然瘦但是力量也不小，但是杜蘭特肯定還是能贏他，死神的稱號可不是白叫的。

​

其實勇士的組合主要是庫里和杜蘭特太無解了，不過活塞五虎也不一定就會敗，這主要看吹罰尺度，按照十五年前的尺度，勇士打不完比賽可能就沒人可用了。按照現在的尺度，活塞打不完比賽可能就沒人可用了。 不考慮尺度的話，活塞窒息的防守能給勇士造成相當大的麻煩，進攻卻弱不少，所以還是勇士贏面更大。
