瘋狂西部！前五隊25連勝，快船連升三位，八隊爭門票，勇士遠觀

2019-11-25 0條評論 17次閱讀 0人點贊

北京時間11月25日，今天的NBA比賽並不多，一共5場比賽，但比賽過後，西部的形勢更加複雜，相比於東部的強弱分明，實話實說，西部各球隊之間彼此差距並不大，包括一直坐穩第一的湖人，即便7連勝都沒甩開身邊的掘金，包括快船也通過一波五連勝宣告強勢回歸！簡單看一下戰況。

獨行俠137-123完勝西部某隊，東契奇狂砍41分10助攻6籃板，對手在第四節一度追到只差5分，東契奇上場後轟下13分，兩度助飛波神，隨後打卡下班！他也成為了獨行俠隊史上第一位連續4場砍下30+10的球員，成為NBA歷史上連續4場砍下30+10第二年輕的球員（第一是杜蘭特）。

國王客場113-106擊潰奇才，籃網103-101險勝尼克斯，歐文繼續缺戰，籃網繼續贏球，未來籃網要接連面對騎士和凱爾特人，歐文都不能打，名宿帕金斯質疑歐文怕被球迷指責，所以選擇作壁上觀。不過籃網在歐文缺陣後戰績4勝1負，排名飆升到東部第七。掘金主場116-104擊敗太陽，太陽的盧比奧繼續缺戰，少了最強大腦，太陽在下半場崩盤，掘金在穆雷和巴頓的帶領下拿到五連勝，排名緊咬湖人。

快船134-109狂勝鵜鶘，倫納德26分，喬治18分，但這哥倆在場時球隊還輸球，哈雷爾暴走狂砍34分12籃板，幾乎以一人之力打穿鵜鶘內線，快船收穫五連勝，但依舊解不開泡椒和倫納德同時在場效率低下的難題。

排名方面，西部簡直太瘋狂了，排名第一的湖人14勝2負，最近10戰9勝，七連勝後勝率達到87.5%，但並沒有甩開掘金。掘金12勝3負排名西部第二，也是10戰9勝，最近一波五連勝，防守效率高居聯盟第二，他們正在演繹團隊籃球。排名第三的快船12勝5負，泡椒復出後，球隊收穫五連勝，當然，他們的主場比賽最多，一共17場比賽，12場主場，戰績11勝1負。憑藉今天的勝利，快船從西部第五一下來到西部第三的位置，劍指掘金！

排名第四的獨行俠也迎來五連勝，在東契奇的帶動下，獨行俠今年不可阻擋，除了兩次輸給尼克斯之外，剩下的比賽該拿的全部拿下，而且遇強則強，東契奇才20歲，很難想像他的上限到底在哪裡。排名第五的爵士是11勝5負，他們也迎來三連勝，但因為沒有比賽，所以排名往下掉。西部前五名狂取25連勝，不要說輸球，即便你沒比賽，可能都守不住你的位置。

但從第七名開始，就出現很多未知性了。因為盧比奧的缺陣，太陽最近4場戰績為1勝3負，排名直線下滑，目前和森林狼一樣都是8勝8負，但身後的國王已經迫近，和他們僅僅相差0.5個勝場。同樣的，目前哪怕是西部倒數第二的開拓者，距離第八的森林狼也就差了3.5個勝場，常規賽才打了五分之一，完全存在各種各樣的可能性，以開拓者的實力，他們還是具備沖入季後賽的希望的，包括馬刺，包括雷霆，他們都有可能躥升。

​但唯一確定的恐怕只有勇士，BR給出的預測中，勇士能夠殺入今年季後賽的概率已經為0，球隊目前健康的球員只有8名，包括拉塞爾、庫里、克萊都是無法登場的，而格林也因為這樣那樣的小問題連續缺席比賽。正如格林之前所說，傷病就是比賽的一部分，接受現實！灰熊在結束了勇士的八連敗後，勇士目前3勝14負，距離開拓者都有很大的距離，主場打完雷霆和公牛之後，勇士又要開啟殘酷而可怕的東部客場之旅，現在能指望的只有帕斯卡爾、伯克斯、鮑曼這些人，這支勇士，可能現階段也不需要格林。
