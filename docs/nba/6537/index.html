湖人總冠軍版圖裡沒有“你們”！3方交易出爐！一球員預被交易

2019-11-29 0條評論 17次閱讀 0人點贊

本賽季，湖人開局擺出「高調」姿態，碾壓對手贏來連勝，坐穩聯盟第1寶座。但，賽季初至現在，湖人龐大的教練團隊，似乎發現了球隊目前陣容仍需要再度調整，使攻防兩端，發揮的再出色一些。

在11月份的所有比賽中，湖人所獲勝的比賽大多數都是在10分以內。這不禁，讓球迷產生幻想。球隊實力到底如何？今年能否衝冠？等等一系列問題。

繼續補強，勢在必行了。據媒體爆料，湖人正密謀3方交易，針對外線帶來的問題，球隊希望引進外線防守悍將，來進一步鞏固球隊的防守實力。

從昨天對陣鵜鶘的比賽，外線差點被鵜鶘射穿。外線防守再次被拉響警報！當然，此前悍將布拉德利的缺陣一定程度上確實影響了球隊外線防守效果。波譜的出戰，進攻端是解決了，但是防守能力在這場比賽中暴露無遺！

目前，湖人的籌碼不足。想直接引進伊戈達拉，沒有可能。但若是再招一支隊伍，各取所需。到時，湖人會極有可能獲得想要的。而交易的籌碼，很有可能是用杜德利和波普去做交換。但是考慮波普的進攻方面能力，很有可能是用杜德利加選秀權作為交換的籌碼！

NBA就是一個商業市場，不知道明天會發生什麼，不知道明天會躺在哪個城市的客廳。在利益面前，球隊會做出犧牲。
