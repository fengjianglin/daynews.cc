獨行俠野心有多大？庫班為東契奇瞄準第二幫手，雙塔或呼之欲出

天天要聞
2019-12-19

截至今天已經是交易市場開啟的第3天，聯盟多支球隊開始為了補強而蠢蠢欲動，不管是爭冠檔次球隊湖人、快船和火箭，還是季後賽球隊獨行俠、熱火都在全力爭取在賽季初期得到補強。

按照目前的情況，湖人已經佔據了引進伊戈達拉的先機，而快船也在盡最後一絲力氣爭取，火箭則是在尋求一位鋒線防守球員，目前考文頓、蘭德爾都是潛在目標，熱火則是一直和雷霆商討保羅的交易。最後，獨行俠的動作更讓我們震驚，庫班正在醞釀一筆改變西部格局的交易產生。

庫班瞄準德拉蒙德，東契奇或迎第二幫手

近日，NBA名記Woj在節目中爆料，獨行俠並沒滿足現在西部第三的成績，他們依舊希望在交易市場開啟的這段時間裡得到補強，而他們的目標就是補強內線，為波爾津吉斯帶來一位搭檔，給東契奇配備一位最適合的冠軍拼圖。

目前，庫班把目標鎖定在了東部活塞陣中的主力中鋒德拉蒙德，獨行俠希望打造一套內線雙塔的陣容，配備上東契奇的神奇表現在本賽季對總冠軍發起第一次衝擊，獨行俠的野心之大可見一斑。

本賽季，德拉蒙德場均可以得到17.7分16.6籃板1.72次搶斷1.84次封蓋，他在內線的威懾力在整個聯盟里無出其右。可以說，如果波爾津吉斯能夠和德拉蒙德組成內線雙塔，他們一里一外的組合能夠在西部掀起一股不可忽視的浪潮。

值得一提的是，德拉蒙德在活塞的生活並不開心，儘管活塞處在東部季後賽行列，但顯然無法參與東決的競爭，這讓德拉蒙德情緒很低落。所以，當獨行俠開始示好德拉蒙德的時候，他本人是不可能拒絕獨行俠的邀請的，現在的問題就只剩下活塞管理層是否同意把德拉蒙德送走了。

其實，活塞現在的陣容實力還不錯，在羅斯加盟之後很好的彌補了得分點不足的缺陷，但整體賬面實力也確實無法成為總冠軍熱門球隊。如果活塞管理層考慮到德拉蒙德的職業生涯，也有可能會同意將其送至獨行俠，但這也要看獨行俠能夠給出什麼交易籌碼了。

其實，活塞並不願意在其合同結束之後給予大合同，NBA記者曾透露，活塞絕不會給德拉蒙德提供一份5年1.9億的續約合同，也正因此，雙方存在分道揚鑣的可能性。

但是目前德拉蒙德還在合同之中，獨行俠想要得到他依舊需要通過交易才能實現。接下來，就看庫班如何醞釀這筆交易了，獨行俠有可能會成為西部爭冠大熱之一。
