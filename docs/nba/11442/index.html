伯德：曾以為喬丹獨一無二，到詹姆斯出現知道籃球之神不只有一個

2019-12-11 1,642條評論 2,997次閱讀 2022人點贊

北京時間12月11日，今日前NBA球員，傳奇名宿拉里.伯德在出席一檔播客節目時，談到了湖人球星勒布朗詹姆斯和傳奇巨星邁克爾喬丹之間的比較。伯德表示自己曾經以為邁克爾喬丹是歷史上獨一無二的存在，但直到詹姆斯進入這個聯盟，自己才發現籃球之神不只有一個。伯德認為雖然現階段詹姆斯還和喬丹有一定的距離，但是在詹姆斯退役後，他的地位一定能和喬丹等同甚至是超越。

對此，伯德說道：「我曾經以為邁克爾是NBA歷史上獨一無二的存在，我們沒在各自的巔峰時期較量過，當他成名時，我已經是一個快退役的老傢伙，但我知道即便是巔峰時期的自己也無法戰勝他。看著他打球的方式，看著他那些非人的技巧，我確信他就是籃球這項運動的最頂峰，不管後來這個聯盟怎麼發展，都不可能會出現第二個邁克爾。但是當勒布朗出現在這個聯盟，並且開始統治這個聯盟時，我才知道籃球之神不只有一個。」

「勒布朗擁有驚人天賦和聰明的大腦，他不像邁克爾那樣擁有與生俱來的得分能力，但是他比邁克爾更加全能。在我看來，他就是邁克爾和魔術師的結合體。在他的身上我看到了巔峰時期邁克爾的影子，他們對於球場的統治力幾乎同等。或許現在勒布朗和邁克爾之間還有一些差距，但是當勒布朗退役時，他一定會站在和邁克爾等同的高度，甚至是超越邁克爾。」伯德說道。「更令人尊敬的是勒布朗已經成為了孩子們的榜樣，他為這個社會做出的貢獻，給那些貧窮的孩子們帶來希望，都能讓他成為籃球界的代表人物，他在用籃球讓這個世界變得更好，所以不管從哪方面來說，勒布朗都是第二個籃球之神。」

一直以來伯德都很看好詹姆斯，或許是因為自己沒做到的原因，伯德固執的認為詹姆斯一定能夠超越喬丹。但說句實在話，詹姆斯在球場的統治力和喬丹真的存在一定的差距，這差距或許直到詹姆斯退役都彌補不了。那麼詹姆斯能超越喬丹么？除非湖人接下來三連冠，否則兩人的歷史地位將不會產生改變。但是詹姆斯配得上籃球之神的稱呼么？完全配得上！現在的詹姆斯不僅僅是湖人的當家球星，他還是這項運動的代表人物，年輕球員以他榜樣，孩子們以他為學習目標。他對於這世界和社會所做出的貢獻，要遠大於他在球場上給球迷們帶來的表演。用籃球讓世界變得更好的人，才是真正的籃球之神！
