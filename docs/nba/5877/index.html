119+36，小卡君臨天下，4巨打瘋了，又迎8200萬神射！

2019-12-09 0條評論 14次閱讀 0人點贊

12月9日，快船隊在客場以135比119大勝奇才隊，實現了強勢反彈。繼上一場28分不敵雄鹿隊時候，快船隊迅速實現了強勢崛起，走出了失利的陰霾。

快船隊大勝奇才隊

本場比賽，可謂是小卡君臨天下，4巨打瘋了，又迎8200萬神射，狂轟119+36！火箭成輸家！

該場比賽的對決，快船隊卡哇伊萊昂納德和保羅喬治兩大巨星先聲奪人，他們在上半場就開始火力全開。伴隨著兩大巨頭率先進入狀態，快船隊的替補兩大巨頭路易斯威廉姆斯以及哈雷爾，也在第三節找到了進攻狀態，並擴大比分優勢，一度領先對手多達21分，第四節繼續保持強勢攻防的快船隊，就此鎖定了勝局。

保羅喬治與萊昂納德

小卡在今天君臨天下，他出戰32分鐘，18投12中，三分球6投3中，罰球8中7，得到了34分11板5助1斷的大號兩雙數據。小卡的投籃命中率高達67%，三分球命中率為50%，而他的2分球是12投9中，命中率高達75%。小卡拿到了全場最多的籃板，轟下了全場最高的得分，經歷了上一場的失利，讓小卡勢在必得拿下本場比賽。防守端，他讓對位的八村塁只有14投5中，邦加4投0中。可以說，萊昂納德制霸了全場，他在攻防兩端都是全場最出色的球員。

萊昂納德統治攻防兩端，君臨天下

萊昂納德雖然不善言談，但是他對於本場比賽的贏球慾望，還是非常強烈的，這正是他能夠火力全開，君臨天下的原因。

除了萊昂納德大爆發之外，喬治也打瘋了，他轟下了全場第二高的27分，還有6板6助4斷的全面數據。能夠砍下高分，還送出全場最多的助攻與搶斷，不得不說，保羅喬治也是在攻防兩端大爆發的球員，他的正負值為全場最高的+35，這是他狀態火熱的體現。

保羅喬治也有出色發揮

替補出戰的兩大巨頭路易斯威廉姆斯和哈雷爾，分別轟下了18分1板6助，以及20分5板。快船隊的四巨頭，在今天同時爆發，都打瘋了，這是他們大勝對手的關鍵。

哈雷爾與喬治打擋拆戰術

需要關注的是，快船隊在今天還迎來了8200萬射手的集體歸來。身背4000萬合同的貝弗利，他5投3中，三分球5中3，得到了9分7板4助的全面數據。作為一名頂級的3D球員，貝弗利在今天雖然出手次數不多，但是投籃命中率和三分球命中率均高達60%。

在防守端一如既往穩定的貝弗利，如果能夠在進攻端保持如此高的命中率，那麼他會成為快船隊衝冠的重要拼圖。

快船隊貝弗利與哈克里斯

而4200萬合同在身的哈克里斯，在沉寂多場之後，終於迎來了爆發，他首發上場36分鐘。6投4中，三分球5中3，得到了11分6板1助2斷1帽的全面數據。作為一名空間型3D前鋒，哈克里斯本場比賽的得分，是本賽季新高。

在進攻端不爭搶球權的哈克里斯，一直是快船隊最低調的首發球員。他在本場比賽的投籃命中率為67%，三分球命中率為60%，可以說是火力兇猛。

哈克里斯、貝弗利，兩名薪金總和為8200萬的神射手，聯手快船隊4巨頭喬治、小卡、路威、哈雷爾，共計轟下了119分36板22助7斷2帽，還有14記三分球，他們帶領快船隊取得了這場大勝。

快船隊眾將集體爆發

在今天贏球之後，西部賽區的競爭對手獨行俠隊和掘金隊，分別輸給了對手，這讓快船隊重返西部第二。火箭隊，目前排在西部第四，但是他們落後快船隊1個勝場。

火箭隊，將成為快船取勝的輸家。一方面，路易斯威廉姆斯、哈雷爾、貝弗利，都是火箭隊舊將，他們為快船隊的強勢崛起發揮了重要的作用。他們的爆發，讓火箭隊在西部賽區多了一個奪冠對手。

火箭隊成為輸家

另一方面，快船隊正在擴大與火箭隊之間的勝場差。目前，快船隊佔據著西部第二的位置，恐怕火箭隊只能暫時在上半區邊緣徘徊，難以重返西部前三了。
