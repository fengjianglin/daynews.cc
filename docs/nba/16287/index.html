什麼叫歷史最佳，喬丹一人跟雷霆四少的榮譽對比，完全不落下風

2019-12-14 59條評論 1,564次閱讀 633人點贊

最近有媒體曬出一張圖——史上最大的「如果」，下面是雷霆四少的個人榮譽集合：

伊巴卡：2個蓋帽王，3個一防，1個冠軍

威斯布魯克：2個助攻王2個得分王，8個一陣，8次全明星，1個MVP

哈登：1個助攻王2個得分王，6次一陣，7次全明星，1個MVP

杜蘭特：4個得分王，9個一陣，10次全明星，2個冠軍，1個MVP，2個FMVP

如果不是伊巴卡在季後賽打了封閉強行復出，影響了他的未來生涯，雷霆四少毫無疑問都會是全明星球員。不僅如此，雷霆三少人手一個MVP，分開之後各自成為球隊核心，都有著不錯的表現。所以很多人在討論，如果雷霆四少沒有分開，雷霆會取得怎麼樣的成就。

當然這篇文章的主角是邁克爾喬丹，喬丹幾乎就是公認的NBA歷史最佳，但有些球迷還是認為喬丹能夠成為歷史最佳，是因為有情懷的因素在裡面。但其實並不是，無論是個人表現，球隊戰績，個人榮譽，個人效率，以及對NBA和對籃球的影響，喬丹都是頂級的，詹姆斯很優秀，但恐怕哪怕是詹姆斯最資深的球迷，都不敢說詹姆斯超過喬丹了。

以下是喬丹的個人榮譽：10個得分王3個搶斷王，14個全明星，5個MVP，6個冠軍，6個FMVP，10個一陣，9個一防。

單項數據王方面：喬丹是13個，雷霆四少加起來剛好也是13個。至於一陣和一防方面，喬丹是10+9，雷霆四少是23+3，雷霆四少多了7個，但是喬丹的優勢是均衡，攻守兩端都是一流的，這一點雷霆三少肯定做不到，杜蘭特算最接近的，至於哈登，防守已經不是重點了。

然後是全明星，喬丹當然也是落後的，14VS25。喬丹的優勢是硬榮譽，6個冠軍，而雷霆四少才3個；6個FMVP，雷霆四少才2個；5個MVP，雷霆四少也才3個。

整體上喬丹少了7個一陣或者一防，少了11次全明星，但多了2個MVP，3個總冠軍，4個FMVP。很明顯，雷霆四少只有在一陣或者一防、以及全明星次數上領先喬丹，領先的原因是因為人多，但是重要榮譽方面，喬丹完全實現了一人碾壓四人的存在，有些東西我們確實沒見過，但從這些存在的榮譽和數據上就能感受到，喬丹確確實實配得上歷史最佳的稱號。

或者在對比下，喬丹生涯個人榮譽，大概就等於現在的詹姆斯+現在的杜蘭特吧。所以沒超越喬丹不是恥辱，能夠被拿來跟喬丹對比，已經是一種榮耀了。
