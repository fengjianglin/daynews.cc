馬刺在哈登有爭議的非扣籃比賽中贏得了2倍的勝利

天天要聞
2019-12-04

詹姆斯·哈登被裁判官員揮手灌籃，這是第一個跡象表明這可能不是休斯頓火箭之夜，沃克獲得了職業生涯最高的28分，聖安東尼奧馬刺克服了哈登的50分，在周二的加時賽中戰勝了休斯敦135-133。

在周六對陣亞特蘭大的比賽中，哈登打出了本賽季最高的49分鐘，結束了60分的比賽，但在第二次加時賽中，他因兩次被指控而被吹口哨。第二場比賽還剩0.8秒，對手是德馬-德羅贊，這是聖安東尼奧的勝利。

沃克和塔克

在此之前，在第四節的一個奇怪的時刻，官員抹去了哈登的扣籃，剩下7：53，休斯頓領先102-89，官方裁定該球沒有清除圓柱體，但船員隊長詹姆斯·卡珀斯賽後說，事實上，它確實清除了球門，並且應該是一個成功的外場進球。

卡佩爾斯說，火箭沒有要求在30秒的時間內對教練提出質疑的機會進行審查，在這段時間裡，他們一直在爭論並要求澄清，這一拙劣的號召粉碎了休斯敦的勢頭。

我們只是停止了比賽，他們獲得了信心，哈登表示，火箭在下半場以22領先，在第四節以兩位數的優勢領先，除了最後的4：18。

沃克是一名第二年的後衛，他在職業生涯的大部分時間裡都在替補席上耐心地等待著，在他第一次延長上場時間的時候，他接任了這位球員，

這是信心，沃克表示，我的隊友讓我繼續前進，一旦我感覺到了，我就會放手去玩我的遊戲，我知道我能做什麼，不能做什麼，我很高興我們打了隊球，我們贏得了這場勝利。

德羅贊拿下23分，9次助攻和5個籃板，熱刺在主場連續第二次贏得勝利，打破了火箭隊的兩連勝紀錄。

他們踢得比我們更快，但更有緊迫感，尤其是站起來了，火箭控球後衛拉塞爾·韋斯特布魯克說。

哈登在第二次加時賽中得了6分，其中包括克林特·卡佩拉的罰球失誤，還有1：41，休斯頓以131比128領先，布萊恩·福布斯以3分的比分以131分的比分扳平比分，兩支球隊在比賽剩下的時間裡交換了罰球。

哈登上籃

除了在最後的30秒里，德羅贊在罰球中4投3中，最後一次在哈登身上被罰進了最後的罰球。

這就是他是誰，他是球隊的領導者，沃克說，我們希望他能在關鍵時刻進球，做出重大貢獻，這正是他所做的，他為我們贏得了這場勝利。

卡佩拉拿下22分21個籃板，奧斯汀里弗斯和韋斯特布魯克分別為休斯敦貢獻19分。韋斯特布魯克30投7中，但增加了10次助攻和10個籃板.

馬刺沒有首發大前鋒阿爾德里奇，他因右大腿疼痛錯過了他的第二場比賽，聖安東尼奧在主場擊倒洛杉磯快船四天後，以集體的努力擊敗了火箭隊。

聖安東尼奧主教練波波維奇說：這是一支我們認為我們擁有的球隊，我們必須一直這樣踢下去，但我們並沒有做到這一點，所以，希望我們已經開始想辦法了。

福布斯獲得25分，雅各布·波爾特爾得到15個籃板，5個蓋帽和5個助攻，魯迪·蓋伊在23分鐘內獲得14分。

在加時賽的最後兩分鐘里，休斯頓以7比2的比分擊敗了聖安東尼奧，迫使球隊進入第二節，第一次加時賽結束時，哈登的跑步者從籃板和籃板上跳了下來。

沃克在第四節得了18分，其中包括連續8分結束這段時間，並迫使加時賽。沃克的跑動開始於小巷–接二連三的扣籃，然後連續3分打成115平手–都還剩11.5秒。

沃克

他是這支球隊中最有活力的球員，德羅贊說，他可以做一些我們很多人都做不到的事情。他只是去玩他的意志和能力，然後他開始了。

火箭官方：哈登在比賽中扭傷了左腳踝，在5：26落後於球場。卡佩拉似乎踩到了哈登，試圖救球，哈登慢慢地站起來，小心翼翼地在球場上慢跑，但是除了第一節的最後2：27，其他的比賽都打完了。

馬刺官方：沃克之前職業生涯的最高紀錄是4月3日對陣丹佛的16分和2月6日對陣金州的26分鐘，周二他打了35分鐘，福布斯在被罰出局前得了25分，而第二次加時賽僅剩48.5秒。

下一場比賽

火箭：周四晚上在多倫多。

馬刺：周五晚上主場迎戰薩克拉門托。
