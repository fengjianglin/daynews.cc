湖人第三巨頭成籌碼？還是躲不過交易這一刀！但湖人不會急於交易

2019-12-22 34條評論 1,253次閱讀 63人點贊

湖人隊兩場不勝，交易流言漫天飛舞。

客場接連輸給步行者隊、雄鹿隊，湖人隊本賽季首次吃到連敗戰績，使得湖人隊賽季開始以來積攢的良好口碑下滑。聯盟之中似乎從來都是如此，贏球的時候做什麼都對，而輸球的時候看哪裡都是問題。湖人隊輸掉「總決賽預演」，賽季開始以來取得24勝5負戰績，依舊是全聯盟第二好成績。

然而湖人隊真的夠完美嗎？至少有兩件事情是湖人隊難以辯駁的——其一，湖人隊對陣強隊的比賽戰績不佳，五場輸球對手分別是獨行俠隊、步行者隊、猛龍隊、快船隊以及雄鹿隊，清一色都是各自分區排名前五的球隊。湖人隊賽季初期就被質疑賽程輕鬆，直到現在也沒能完全證明自身實力。

其二，湖人隊的表現起伏不定，每場比賽都需要「詹眉組合」打出足夠出色的表現，角色球員表現好的時候輕鬆一些，其他時候比賽就比較困難。湖人隊的三分球能力仍是他們的痼疾，自從上賽季詹姆斯加盟湖人隊開始，球隊就沒能配置其足夠多的外線火炮，這導致球隊的進攻時常斷電。

美媒報道截圖

從這兩點來看，交易流言的出現也並不奇怪，身處於聯盟之中，任何一支球隊都是如此。強如聯盟第一的雄鹿隊，不也是跟一些球員的交易流言聯繫在一起嗎？交易流言的出現會有兩種情況，對於球隊現狀不滿意或者球隊還有更好的提升空間，雄鹿隊的交易流言屬於後者，而湖人隊不幸成了前者。

湖人隊幾天前一度跟鵜鶘神投手JJ-雷迪克聯繫在一起，就是因為外界對湖人隊外線投籃比較擔憂所致。

若是尋求交易的話，交易籌碼無疑是湖人隊遇到的最大難題，休賽期為了得到安東尼-戴維斯，湖人隊將年輕球員以及選秀權一股腦地梭哈。而他們在休賽期簽下的自由球員當中，具備較高交易價值的球員也並不多，因此年輕球員凱爾-庫茲馬越來越多出現在交易的流言當中。

在比爾-西蒙斯的訪談節目中，ESPN記者布萊恩-文霍斯特就談到了凱爾-庫茲馬的一些情況。他在節目中表示，湖人隊主帥沃格爾覺得庫茲馬並不適合湖人隊的體系。由於一系列傷病導致庫茲馬跟球隊磨合時間不夠，但是庫茲馬的缺席對湖人隊戰績卻沒有產生太大的影響。

湖人隊「三巨頭」

賽季開始之前，凱爾-庫茲馬還被稱為湖人隊的「第三巨頭」，球隊地位僅次于勒布朗-詹姆斯和安東尼-戴維斯。主教練沃格爾就很看重庫茲馬，「他可能是我們隊中第三好的球員，有了他我們可以有更多選擇。在庫茲馬、詹姆斯和幾個後衛幫助下，讓戴維斯更多去打五號位變得極富吸引力。所以當他完全恢復時，我們會有幸福的『煩惱』。」

庫茲馬休賽期一直都在跟隨美國男籃集訓，最終因為受傷提前告別美國男籃，但是他的能力卻得到了波波維奇的認可。庫茲馬對於新賽季的征程比較自信，「上個賽季我的投籃不是很好，但是我場均依然得到接近20分。如果我的投籃變得更好，並且繼續提升我的防守，我認為我可以成為湖人的第三巨頭。」

但是這一切在新賽季還沒有出現。

比較悲哀的事情在於，凱爾-庫茲馬之前一直都是交易安東尼-戴維斯的潛在籌碼之一，最終庫茲馬親手幸運抽到的四號簽代替他被交易到鵜鶘隊。可是這份幸運並未能始終守護他，當湖人隊新賽季出現些許狀況的時候，庫茲馬還是沒能逃脫被捲入交易流言的命運。

湖人主帥沃格爾

雖然這還只是虛幻的交易流言，美國媒體TradeNBA卻極其認真，該媒體甚至還設計了兩套交易方案。一套方案是打包庫茲馬和波普跟鵜鶘隊交易JJ-雷迪克，另一套方案則是跟活塞隊交易馬基夫-莫里斯，活塞隊要送出兩個首輪簽補償。前一個交易方案若是成真的話，庫茲馬絕對會相當鬱悶。

交易方案很容易做出來，但交易卻不是簡單的事情。

眼下交易庫茲馬無疑會比較難，這主要是由庫茲馬的交易價值決定的。庫茲馬新秀賽季就讓聯盟為之驚嘆，上賽季更是成為詹姆斯身旁的最強得分手，但是庫茲馬的上限卻存在質疑。而在另一方面，庫茲馬本賽季正處在低谷當中，此時他的交易價值當然要比正常時候低上一些。

湖人隊不會輕易就做出交易的決定，他們還有比較長的時間來進一步檢驗球隊真實能力，畢竟湖人隊仍然是聯盟戰績最好的幾支球隊之一。湖人隊接下來還有一些硬仗要打，擺脫了客場旅途的疲憊之後，湖人隊能夠打出怎樣的表現呢？庫茲馬也是球隊要繼續考察的球員，他可是湖人隊本賽季的第三得分點。
