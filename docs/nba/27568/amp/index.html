這是重建隊？雷霆3人20+，豪取3連勝+戰績回5成 保羅8投躺贏

天天要聞
2019-12-22

北京時間12月21日，2019-20賽季NBA常規賽繼續進行。布克回歸率領太陽，在客場挑戰雷霆。最終主隊3人20+，保羅在全場僅僅8次出手的情況下，仍然幸福「躺贏」。這場勝利，是雷霆最近11場比賽的第8場勝利，豪取3連勝之後他們的戰績來到了14勝14負，首次回歸五成。

賽季初被定義為「重建隊」，一度只能在鵜鶘、勇士這種魚腩身上刷勝場的球隊，如今穩居西部前八，季後賽的希望越來越大，不得不感慨這支球隊實在太神奇了。

面對太陽，雷霆除了在開局，幾乎是全場保持對於對手的壓制，球隊獲勝的最大功臣無疑是亞歷山大、加里納利和施羅德。

小將亞歷山大打出了生涯的代表作，面對太陽羸弱的內線，他一次次的衝擊籃下，全場18投13中，轟下了生涯新高的32分，外線投籃也是罕見的神准，2次三分出手全部命中。

施羅德持續暴走式的表現，上場轟下了31分的雷霆超六，本場的發揮依舊穩定且全能，全場比賽攻下了24分9板6助，連續兩場高效的發揮，讓雷霆穩穩手下勝利。

加里納利儘管在比賽中，一度與貝恩斯鬥氣吃到技術犯規，但這位義大利鋒線的表現，還是非常穩定，全場轟下22分。

三名隊友發揮驚艷，保羅則多少有些「躺贏」的味道。雷霆大腦全場僅僅出手8次，在隊友個個神勇的情況下，他也樂意給自己的小弟做嫁衣。

拿到本場比賽的勝利，雷霆的戰績來到了14勝14負，反觀太陽則是遭遇5連敗，這支賽季初的大黑馬，戰績直線下滑。目前來看，雷霆若是不進行大的交易拆散現有陣容，季後賽真的希望很大！
