10中1差點掉鏈子！獨行俠險些毀他手裡 罰球不中卡萊爾表情亮了

2019-12-18 1,953條評論 3,411次閱讀 1895人點贊

獨行俠隊擊敗雄鹿隊，終結了雄鹿隊18連勝的腳步，雄鹿隊隊史最長連勝紀錄為20場，這一次沒能夠追上當時的紀錄。獨行俠隊堪稱聯盟連勝終結者，之前他們還曾終結了湖人隊10連勝的腳步。在東契奇因傷缺陣的情況下，無論是對陣熱火隊還是對陣雄鹿隊，獨行俠隊都表現得極其強硬。

卡萊爾賽後說道：「我在聯盟里已經很久了，我很清楚18連勝是一支優秀球隊很好的證明，但是你不可能整個賽季都不輸球。我們只是覺得，如果我們能夠在比賽中站出來，真正專註於自己（的比賽狀態），制定一個非常簡單的計劃，我們就能夠給自己一個贏球的機會。」

獨行俠隊真的做到了。

字母哥大殺四方

雖然雄鹿隊幾乎整場比賽都比較被動，比賽進入到最後一分鐘的時候還落後著11分——那還是在喬治-希爾兩罰兩中的情況下。一般來說，這樣的情況基本就宣判了雄鹿隊死刑，奇蹟可能會出現但那可是奇蹟，但是雄鹿隊不甘心連勝紀錄被終結，還是要做最後一次努力。

最後一分鐘比賽打得異常膠著，雄鹿隊將比賽切割得非常零碎，就是想趁著混亂的形勢創造奇蹟。雄鹿隊的努力並不是徒勞無功的，用了大約半分鐘的時間，伊利亞索瓦打成2+1，雄鹿隊還落後8分。半分鐘時間追8分？這更是一項難度極高得到任務，但是雄鹿隊仍然沒有放棄。

小蒂姆-哈達威

對雄鹿隊不利的情況在於，球隊整場比賽三分球命中率極差，關鍵時刻不敢指望著三分球命中率能夠回升。

杰倫-布朗森出現失誤，伊利亞索瓦迅速得到兩分，雄鹿隊採取犯規戰術，賈斯汀-傑克遜兩罰不中，雄鹿隊燃起了一線希望。喬治-希爾得到兩分，雄鹿隊繼續犯規，不過布朗森兩次罰球全都命中。字母哥大跨步殺到禁區迅速得分，小蒂姆-哈達威被送上罰球線，兩罰兩中。

字母哥再次快速得到兩分，繼續對小蒂姆-哈達威犯規，這一次兩罰只有一中。關鍵時刻，字母哥投籃得手造成獨行俠隊防守犯規，然而加罰一球沒能命中，失去了一次絕佳機會。雄鹿隊落後3分，小蒂姆-哈達威第三次被送上罰球線，第一次罰球不中，場下的卡萊爾吸了吸鼻子，表情有些不自然。

好在，小蒂姆-哈達威第二罰命中，獨行俠隊鎖定了勝局。

卡萊爾有些緊張

小蒂姆-哈達威本場比賽表現並不好，全場比賽10投只有1次命中，不過在最後時刻還是被卡萊爾換上場。卡萊爾關鍵時刻用人非常細緻，不斷地進行人員調整，而小蒂姆-哈達威就是被他派上場罰球之人。小蒂姆-哈達威關鍵時刻頂住壓力6罰4中，不過最後兩罰中的第一罰不中，讓卡萊爾心頭一緊。

雄鹿隊當時還有一次暫停可用，若是小蒂姆-哈達威兩次罰球都不中的話，雄鹿隊還有絕平比賽的機會，那樣對獨行俠隊來說就非常不利了。小蒂姆-哈達威罰中最後一球，也算是沒有讓卡萊爾失望，本場比賽手感如此糟糕，卡萊爾還是敢將他派上場，不能不說卡萊爾用人非常大膽。
