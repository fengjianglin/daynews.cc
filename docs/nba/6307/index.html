10球9扣，老詹挨揍，小卡力劈華山遭無視，安東尼夢回2014

2019-12-03 0條評論 16次閱讀 0人點贊

卡哇伊說體育12月3號消息，10球9扣，老詹挨揍，小卡力劈華山遭無視，安東尼夢回2014。

轉眼之間，新賽季已經過了四分之一的時間，聯盟的格局也變得更加的明朗。上賽季西部豪強開拓者至今還在季後賽的邊緣苦苦掙扎，西部非季後賽球隊的湖人一騎絕塵，領跑聯盟。雄鹿還是那個東部第一，失去小卡的猛龍依舊強悍。鐵打的球隊，流水的球員，NBA每日十佳球，也有看點。

就拿昨天的每日十佳球來說，10個球，有9個是扣籃，這樣的十佳球也是令人瞠目結舌。有的人可能會問，那還有一個呢？正所謂10球9扣，唯獨詹姆斯挨揍，第10球則是詹姆斯被芬尼-史密斯追帽的場景。平日我們看慣了10佳球里老詹追帽別人，如今風水輪流轉，這次也輪到他成為背景了。

第九球至第一球，全部是扣籃，10佳球成為了扣籃的盛宴。既然說是扣籃，昨天最精彩扣籃之一的倫納德迎扣八村塁和貝爾坦斯一球卻沒有出現在10佳球里，這也是讓人匪夷所思的地方。說實話，卡魯索接濃眉哥助攻空切扣籃都能上，憑啥小卡的不能？要說難度和精彩程度，明顯都是倫納德的那個更好。

想了半天之後，終於發現了原因，快船和奇才那邊的比賽還沒有打完，十佳球的評比可能已經出來了。倫納德的那一扣發生在比賽的第四節，當時在美國已經是晚上九點多了，估計是工作人員等不及要下班了，就沒有等所有比賽都打完，就開始評比了。

小卡那記平地驚雷，力劈華山的扣籃，就這樣遭到了忽視。其實，我們在以前看十佳球的時候，經常會發現這樣的現象，一些我們覺得不錯的球並沒有出現在十佳球里。具體的評選規則，我們也無從得知。所以想多了也沒什麼用，還是好好看球就好。

再來說說今天早上的一則好消息，那就是安東尼當選上周西部最佳球員。這對於35歲的甜瓜來說，無疑是一件喜事，沉寂了那麼久，如今終於看到了一絲曙光。安東尼上一次拿到這個榮譽的時候，還是2014年3月，那時候的杜蘭特還在雷霆，倫納德還沒有拿到自己的第一個FMVP，5年一晃，時過境遷。

甜瓜本人得知這個消息之後，也是十分的高興，在社交媒體上也發了一條動態來表達自己的心聲。隨後韋德也發文祝賀安東尼獲得這項榮譽，曾經的風塵四俠，韋德已經退役，看著安東尼這個「老傢伙」還在場上繼續的戰鬥，他是打心眼裡替他高興。
