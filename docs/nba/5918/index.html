NBA新賽季五位最強二當家！濃眉登頂，一人把老大演成“毒瘤”

2019-12-09 0條評論 16次閱讀 0人點贊

NBA新賽季五位最強二當家！濃眉登頂，一人把老大演成「毒瘤」。

第一、濃眉哥

現役最強二當家，安東尼-戴維斯，目前湖人老大還是詹姆斯，精神領袖，未來肯定是戴維斯。目前戴維斯場均得到26.6分9.1籃板3.1助攻2.73蓋帽，是湖人得分王，投籃命中率高達49.5%，三分命中率也有34.2%，罰球命中率高達86..3%，小球時代，濃眉打誰都是錯位。數據顯示，詹姆斯給濃眉的球，超過了之前的所有隊友，他也多次在公開場合表示，濃眉就是球隊核心。

第二、保羅-喬治

現在喬治扮演的是3D球員的角色，場均得到22.7分6.2籃板3.3助攻，三分命中率高達43%，其實喬治的個人進攻能力比倫納德強，特別是在投籃這一項，喬治去任何一支球隊都可以當老大，早年間在步行者我們已經見到了他的帶隊能力。現在他更習慣當二當家，想奪冠就要做出犧牲。

第三、西亞卡姆

猛龍的老大是洛瑞，這一點是不用質疑的，他是球隊隊魂，洛瑞這個賽季煥發了第二春，場均20.5分4.5籃板7.1助攻，三分命中率高達38%，現在他已經是總冠軍球隊的指揮官了。洛瑞扮演著詹姆斯一樣的角色，西亞卡姆是這支球隊的未來，上個賽季就是進步最快球員，這個賽季進步更大，場均25分8.6籃板3.8助攻，三分命中率高達37.6%。

第四、英格拉姆

霍勒迪是鵜鶘老大，顯然他不是一個合格的球隊領袖，而是一名得分手，但本賽季效率極低，場均只有19.2分4.9籃板6.7助攻，但三分命中率只有33%。英格拉姆是球隊首席得分手，但你是初來乍到，明年夏天就要成為自由球員，建議英格拉姆去其他球隊，等錫安回來，英格拉姆也許會受到影響。

第五、丁威迪

丁威迪今天在主場105-102擊敗了掘金，球隊豪取三連勝，丁威迪得到24分4籃板8助攻，沒有歐文，籃網打得更好，丁威迪帶隊比歐文帶隊成績好多了。杜蘭特不在，歐文成球隊老大，但已經連續缺席了12場比賽，期間籃網只輸了3場球，所以很多人都說歐文是「毒瘤」，特別是波士頓凱爾特人球迷。如果歐文回來後，籃網又輸球，那就很尷尬了。
