4大NBA“惡名昭彰”的防守悍將，每個人都被很多球迷討厭

2019-12-25 604條評論 3,275次閱讀 2290人點贊

NBA是一個身體碰撞很激烈的聯盟，能在這裡立足的球員不僅是身高出類拔萃，身體素質更是一等一。這樣一群強壯的肌肉大漢在一個幾百平的場地競爭，免不了一番肉搏爭鬥。所以，一個優秀的防守者永遠是一個完美的冠軍拼圖，這樣的防守者每支球隊都爭相招募。不過，這類防守者中，有的是名譽天下，但有的是臭名昭著。臟活累活通常形容的是防守段的工作，可對於這類人來說，確確實實是骯髒的手段，即使他們入選過最佳防守陣容，當選了最佳防守球員，也改不了人們對他的印象。

布魯斯-鮑文

球迷們最熟悉的鮑文是在馬刺時期的鮑文，那時候鮑文是馬刺側翼防守大將，防守能力是頂尖的。生涯共入選過8次最佳防守陣容，足見他在防守端的影響力，在馬刺奪得過3次總冠軍，在個人榮譽上是無可挑剔的，可在聲名上，鮑文稱得上的臭名昭著的。鮑文的防守很多小動作，扯衣服，扯褲子那是小事。特別是對上對面的主力，鮑文不止一次下死手。2007年的季後賽馬刺對陣太陽，鮑文全系列在盯防納什，做出惡劣的犯規，直接用膝蓋頂撞納什的腹股溝，直接將納什撞翻在地，還可隱約看出納什的眼淚。最後納什因傷停賽，馬刺得以晉級。不僅僅是納什一人遭受過鮑文的惡意犯規，卡特，科比等人都有不過相同經歷。即使鮑文已經退役很久了，可每次討論到骯髒防守，他總是第一個被提及。

扎扎-帕楚利亞

帕楚利亞可能是現役最令人討厭的球員了，除了所在球隊球迷，怕是其餘球迷都對他有怨言。帕楚利亞最大的成就，可能是2016年季後賽一腳終結了倫納德，讓勇士有驚無險的挺近季後賽，拿到總冠軍。很多人都說，帕楚利亞才是那一年的FMVP。當時帕楚利亞對倫納德的犯規被無數次回放，可終究只是一個普通犯規，甚至連技術犯規都沒有。波波維奇老爺子都忍不住要怒噴扎扎，也申請過讓聯盟調查犯規事件，可最終還是不了了之。扎扎的那一腳，幾乎打散了整個馬刺，倫納德傷退一年，一年後因為傷病原因與馬刺鬧翻，最後遠走多倫多。可以這樣說，一腳解決了勇士在西部最強的敵人，一勞永逸。

德雷蒙德-格林

追夢格林是是2017年的最佳防守球員，他的防守是出了名的好，真正意義上能從一號位防到五號位的球員。當然，他的臟跟他的防守一樣出名。最喜聞樂見的，當然是踢褲襠手段。2016年季後賽勇士對陣雷霆，追夢格林在一次搶籃板下落時左腳突然上踢，踢中亞當斯的腹股溝令其受傷。這不是追夢格林第一次了，他在搶籃板總有上踢的惡習，還口口聲聲說是人體運動學原理。如今不僅球迷不喜歡格林，有一些球員都對他嗤之以鼻。

羅恩-阿泰斯特

阿泰斯特即現在的慈善-世界-和平，慈世平。當年阿泰可是一個狠角色，能在2004年，在四大中鋒，三大大前手中奪得最佳防守球員的鋒線球員，能是善茬嗎？阿泰身體素質極強，在鋒線上沒多少人比他強壯的，在防守端經常以身材欺負別人。最著名的奧本山宮殿事件，令的剛剛復興的步行者跌落地獄。還有阿泰的肘子也是非常危險了，當年一肘子打在哈登頭上，幾乎打得哈登腦震蕩，隨後休息了很長時間才恢復過來。
