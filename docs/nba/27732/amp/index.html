中期交易為何"難產"？美專家給出5大原因，狂歡之後的平靜

天天要聞
2019-12-21

NBA中期交易大幕已經拉開，交易流言出現了很多，但進入實質性操作階段的方案卻屈指可數，這樣的冷清很有可能會延續到交易截止日。上賽季的交易截止日，一共誕生了14筆交易，19支球隊參與其中，34名球員改換門庭，非常熱鬧。不過本賽季中期恐怕很難出現這樣的局面了，究竟是什麼原因造成的呢？

最近，The Athletic的NBA專家John Hollinge撰文分析了中期交易遇冷的5個原因。首先是各隊的奢侈稅壓力比較小，不急於做交易。相比於補強球隊，”省錢”是老闆們更在意的迫切需求，NBA的本質還是生意。本賽季各隊的薪金都比較健康，目前只有4隊需要繳納奢侈稅，而這4隊的財力基本都能夠承受。

第二個原因是各隊的薪金空間有限。NBA目前有29支球隊的薪資空間超過了工資帽，只有老鷹離工資帽還有500萬美元的差距，不過並沒有什麼實際意義。一方面，老鷹擺爛不需要補強；另一方面，這筆錢也不足以用來進行一些實質性的交易。今年夏天，大多數球隊都已經完成了引援目標，中期交易的空間被壓縮。

第三個原因是很少有必須送走的球員。專家認為：”2016年夏天很多自由球員經歷了一次狂歡，當初和球隊簽下的毒藥合同大多數都已經到期”，最近兩年誕生的毒藥合同又比較少，所以各隊交易的意願都不太強。這也是中期交易的特徵之一，如果沒有特別迫切的需要，很多球隊寧願觀望，能不冒險則不冒險。

第四個原因是各隊可供操作的空間都不大。湖人今夏交易到了濃眉哥，快船則交易到了保羅-喬治，對未來資產的透支很嚴重，中期交易會更謹慎。對於沒有爭冠需求的球隊來說，他們又沒有薪金壓力和毒藥合同，需要的重建資產同樣是別隊也需要的，所以就造成了如今的尷尬局面，大家都拿不出好籌碼來交易。

第五個原因是很多球隊已經將目光投向了2020年夏天的自由球員市場。和中期交易相比，明年夏天的自由市場顯然會有更多的選擇。濃眉哥、英格拉姆等人都有機會進入明夏的自由市場，除此之外，富尼耶、哈雷爾以及范弗里特這樣的實用型球員也不會缺乏追求者。以上的這5個因素共同作用，導致了市場冷清。
