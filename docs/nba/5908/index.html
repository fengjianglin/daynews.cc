濃眉哥第一節9投17分！更不講理的是，34歲詹皇戰斧隔扣2米08悍將

2019-12-09 0條評論 15次閱讀 0人點贊

北京時間12月9日，湖人主場迎戰森林狼。濃眉哥第一節9投7中，轟下17分。詹姆斯第二節一上來就送出隔人暴扣，這樣的雙人組真的太無解了。

狀元大對決，濃眉哥遭遇了唐斯。比賽一上來，濃眉哥就打開了進攻。搶到籃板之後，濃眉哥一條龍帶到前場，頂著防守上進，還造犯規，完成了2加1。隨後，他低位背打唐斯，轉身小勾手也有。

詹眉連線也來。森林狼追到7平，詹姆斯反吊籃下，濃眉哥快攻輕鬆上進。唐斯終於作出回應，跳投命中，濃眉哥馬上雙手暴扣回應。又是詹姆斯，長傳連線濃眉哥，後者再度上籃得手，湖人打停了森林狼。

暫停回來，濃眉哥依然不可阻擋。接到詹姆斯的擊地傳球，濃眉哥罰球線翻身跳投命中，他連續6投命中。此後，濃眉哥終於投丟了，不過，整個第一節，濃眉哥9投7中，狂轟17分。

第二節一上來，詹姆斯也爆了。還剩9分21秒左右，詹姆斯帶球推進，對位吉昂。森林狼內線悍將身高達到了2米08，相當強壯。

然而，詹姆斯突然加速，直接殺到內線，隔著吉昂，直接一記狂野劈扣。逆天的是，詹姆斯這次加速突破，有些過大了，突破到籃下，高高躍起，身體已經超出了籃筐，但是，詹姆斯在空著擰著身體，回頭望月式，將球摁進籃筐。身體素質實在異於常人，很難想像詹姆斯還剩21天，就年滿35歲了。

在詹姆斯完成這記殘暴隔扣之後，板凳席上的隊友全都抱成一團，太興奮了。湖人主場也成了歡樂海洋。
