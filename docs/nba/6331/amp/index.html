兩巨雙星，一戰狂砍110+30，優點突出，缺點致命，想衝冠還需補強

天天要聞
2019-12-02

快船隊的兩巨雙星本賽季可是厲害啊！首發兩巨倫納德+喬治讓他們的首發陣容實力強勁，替補雙星路威+哈雷爾讓他們的替補陣容深厚無比。快船隊本賽季之所以這麼厲害，關鍵就是他們的兩巨四星組合，太具統治力了。說句實話就路威+哈雷爾這對替補組合，在別的球隊都能當首發雙星使用了，但是在快船他們只能做倫納德+喬治的替補，從這點就可以看出，快船隊的陣容有多強。

在今天與奇才一戰中，他們以150比125大勝奇才，拿下比賽勝利，這一戰他們完美詮釋了他們到底有多強。倫納德拿下34分6籃板，喬治拿下31分8籃板，哈雷爾拿下23分15籃板，路威拿下22分1籃板。兩巨四星合砍110分30個籃板。從這就可以看出快船隊的這套陣容，在進攻端的天賦簡直是爆棚。兩巨砍完，雙星砍，給你玩的是車輪戰術，這那支球隊抗的住啊！

雖然他們在進攻端的實力非常強，但是朋友們有沒有發現他們在組織端的能力卻非常弱。本場比賽兩巨雙星一共才有可憐的13次助攻，全隊助攻也才只有22次，比奇才足足少了10次。不只是這一場比賽他們在組織端表現不好，而是賽季以來他們在組織端的表現就一直不好，場均23.4次助攻排在聯盟第20位，56.3%的助攻率排在聯盟第25位。在組織端表現不好，就意味著快船隊主要依靠單打去得分，去贏球而不是去依靠團隊的力量。而近幾年奪冠的勇士隊，騎士隊，猛龍隊他們最終能奪冠靠的全是球隊整體的發揮，靠的是團隊的能力，打的也是團隊籃球。所以說，快船隊要想衝冠，他們還必要須去補強一位組織後衛，將球隊各球員串聯起來，讓球隊進攻更加流暢，打出團隊籃球。朋友們，你們說呢？
