傷退！日本混血新星腹股溝拉傷 無緣連9場上雙仍給中國籃球拉警報

2019-12-18 256條評論 1,914次閱讀 278人點贊

北京時間12月17日，在奇才客場挑戰活塞一戰，奇才的首輪秀、日本混血新星八村塁遭遇悲劇，他僅出戰17分01秒時間，就因為腹股溝拉傷提前傷退。八村塁在離場前貢獻3中3的兩分球，三分球則是1中0，外加罰球1中1，得到7分2籃板的數據。

作為奇才重點培養的新星之一，八村塁的表現無疑是可圈可點，他在本場比賽之前場均能夠得到14.2分，已經是奇才陣中僅次於比爾與貝爾坦斯的第3號得分手。八村塁在對決快船一戰，更是瘋狂轟下30+9的數據，完成日本球員在NBA的首次30+得分，外加他此前已經連續8場比賽得分上雙。

只是在如今對決活塞一戰，八村塁固然保持著極高的進攻效率，17分鐘時間奉獻4中3的準星，除一記外線三分不中外，其餘兩分球出手全部打進，也是已經得到7分的數據，距離連續9場得分上雙只有一步之遙。

奈何，八村塁無奈提前因為腹股溝傷退，而奇才官方也是宣布他不會在本場比賽回歸，而具體傷情需要等待進一步的官方消息。

值得一提的是，八村塁在此前已經連續8場比賽得分上雙，他本場僅差3分，就能夠連續9場得分上雙，繼續刷新生涯連續得分上雙場次紀錄，卻因為提前傷退讓如此紀錄戛然而止，無疑是極為遺憾之事。

當中國球員在本賽季已經消失於NBA賽場，周琦選擇回歸CBA效力，而其他中國球員尚未能夠有球員登陸NBA，就連華裔的林書豪也是加盟CBA效力，無疑NBA的「亞洲之光」只能是由日本球員撐起來，甚至是此前還上演了NBA歷史首次日本德比，由八村塁大戰渡邊雄太，自然也是給中國籃球拉響警報。
