尤度狂送8封蓋6搶斷，北京大勝廣廈

2019-12-18 1,070條評論 2,572次閱讀 492人點贊

北京時間12月17日，北京男籃在主場迎來了只能以單外援出戰的廣廈男籃，憑藉著尤度18分11板8封蓋6搶斷的防守端出色發揮，北京隊以105比95贏得了比賽勝利。

林書豪拿到20分5籃板9助攻，翟曉川拿到21分5籃板。廣廈這邊孫銘徽拿到20分4籃板9助攻，胡金秋拿到19分10籃板。

根據此前歐洲媒體的消息，廣廈隊小外援阿隆·傑克遜已與球隊解約，加盟以色列聯賽，而剛剛代表廣廈出現第二場比賽的普拉姆利和球隊還沒有產生一定的默契，比賽開始廣廈隊先得到球權，普拉姆利沒有和隊友想到一起出現了傳球失誤。方碩利用掩護出手中距離，反而命中了本場比賽的第一球。

不過普拉姆利還是有不俗的個人能力，在第一節還剩8分多鐘時，普拉姆利從三分線運球，轉身，隔著翟曉川和秋天的防守完成一記暴扣，又在林書豪持球快攻中，跟上他的速度並將球切掉。

第一節後半段，換上全華班的廣廈男籃打得更有整體性和積極性，憑藉劉錚的快攻壓哨上籃，廣廈在第一節以24比22領先北京隊2分。

第二節上半段，雙外援的北京男籃明顯更有優勢，隨著林書豪造成一記2+1，北京男籃打出一波14-4的高潮。不過林書豪還是沒能擺脫「鼻青眼腫」的命運：第一節他在進攻端被防守人用手打到面部；第二節他又在防守端造進攻犯規時被進攻球員肘到臉。

所幸這一切沒有影響林書豪的發揮，他在第二節末尾連續助攻尤度兩次、上籃命中2分、造成對方違體、搶到前場籃板助攻朱彥西三分，幫助北京以54比37在上半場領先廣廈17分。

尤度在防守端給廣廈隊進攻造成了很大的困難，他在內線的防守讓廣廈大多只能選擇外線投籃，但他們手感不佳，上半場三分14投僅4中，失去了進攻節奏的他們又在第二節出現了6次失誤，單節只得到了13分。

下半場開始，尤度先接林書豪的空接命中，又在籃下分給底角朱彥西命中三分，北京隊將領先優勢擴大到了22分。第三節雖然懸念不大，但是兩邊都貢獻了一些精彩的片段，特別是孫銘徽晃過林書豪後完成了一記滑翔暴扣。不過在隨後的防守回合中他試圖封蓋翟曉川失去重心摔倒在地，所幸沒有受傷，在下場調整後又重新回到場上。三節比賽結束，北京以86比60已經領先廣廈26分。

第四節比賽還剩8分半左右，在李春江執導的提醒下，趙岩昊單打林書豪造成後者犯規，林書豪也累積6犯被尤度替下場。隨後的比賽廣廈沒有放棄，也將分差最終追回到了10分，但還是沒有改變比賽的結局。
