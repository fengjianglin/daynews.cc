從場均139分被防成57分，“死亡五小”栽過跟頭，猛龍真核心出爐

2019-12-06 0條評論 20次閱讀 0人點贊

剛結束不久的NBA比賽里，猛龍與火箭的對決，這一場比賽雙方都打的不錯。結果是火箭贏得了比賽的勝利，比賽之後也出現了一幕非常暖心的畫面。

一位小球迷在場邊舉牌，標語上提到了火箭的球星拉塞爾-威斯布魯克，小球迷的牌子內容是「我的心屬於猛龍，但是威斯布魯克才是我心目中的英雄！

威少也被這位小球迷的舉動暖到了，將自己的戰靴送給了這位小球迷。

在比賽里威少出戰35分鐘，出手27次7中，三分6中0獲得19分13籃板11分2搶斷，猛龍在這一場比賽中，使用出他們的秘密武器，針對哈登的「實驗性」防守，半場實施了包夾策略，收穫了不錯的效果。

雖然在進攻方面猛龍這一場比賽的表現不是很好，但在防守端發揮出了很好的效果。

猛龍主教練納斯在賽後接受了採訪，他表示本場比賽對哈登的「實驗性」防守十分有效，並且在後續的比賽中還會研究各種防守戰術。

這樣的教練誰不愛啊，能夠制定自己的獨家秘笈，每場比賽的戰術都是全聯盟獨一份啊！

本賽季猛龍的比賽中，對戰湖人、快船、開拓者、76人到現在的火箭球隊，球隊的核心都沒有打出很好的成績，現在看來納斯才是猛龍隊的真核啊！

火箭對戰猛龍，哈登僅出手11次獲得23分。快船的萊昂納德獲得12分，開拓者利拉德獲得9分，湖人詹姆斯獲得13分，76人的恩比德是最慘的，獲得生涯最低得分0分！

這5人場均得分加起來有139分之多，在對戰猛龍的比賽里，這5人加起來只獲得了57分。特別是恩比德，被防得一分都沒有得到，面對猛龍恩比德真的是有苦說不出啊！這幾位都是聲名赫赫的球星啊，都沒有辦法去破解。尤其是在上賽季的季後賽中，面對擁有「死亡五小」的勇士球隊，猛龍也毫不畏懼，展開了BOX1戰術，這項戰術是第一次出現在聯盟之中。

這項戰術的體現為1人緊貼4人聯防，那時的庫里在賽場中十分難受，一點都發揮不出他能夠得分的優點。看來猛龍這位主教練納斯戰術鬼才的稱號實錘了啊！

目前猛龍的戰績是15勝6負，排在東部第4的位置。

在猛龍接下來的比賽中又是對戰76人球隊，不知道這一次能否再次把恩比德打出上一場的數據。

當然本賽季也才經過了一個多月，到時隊伍的各種短板會逐漸的被磨平，屆時NBA的比賽還會更加的精彩，不知道那時的你是否會守在屏幕前觀看自己喜歡的球隊嗎？
