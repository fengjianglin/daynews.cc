交易窗口即將開啟！四大全明星面臨易主，湖人、快船力爭“一哥”

2019-12-10 94條評論 1,027次閱讀 69人點贊

在NBA，球員流動是再正常不過的事，球隊想要補強，球員想找個更好的歸宿，交易是最好的方式。還有5天時間，新賽季「早期交易窗口」就要開啟，那些休賽期剛簽下合同的球員，將有可能被交易。1/4賽程已過，各種交易流言不絕於耳，總結下來，主角是德羅贊、阿爾德里奇、保羅、樂福這四位全明星球員。另外，湖人、快船對伊戈達拉的爭奪也看點十足。

1、德羅贊

北京時間12月3日，德羅贊清空了自己的社交媒體，他上一次這麼做，是在被猛龍交易到馬刺時。於此同時，他的隊友德章泰·穆雷在ins上取關了馬刺官方賬號。這兩跡象表明，馬刺在戰績掙扎的情況下，正在考慮是否將球隊推倒重建。德羅贊是一位古典派得分後衛，極少投三分，有一手精湛的中投，本賽季場均可得21.4分5.7籃板4.8助攻。無論在猛龍還是馬刺，德羅贊都沒有證明自己是一位超巨，做爭冠球隊的二當家或許更適合。

2、阿爾德里奇

阿爾德里奇當初之所以選擇加盟馬刺，是因為看中了聖城的贏球文化，可波波維奇並沒有履行承諾圍繞他建隊，萊昂納德的離開也是他始料未及的。回望過去4年所發生的一切，阿爾德里奇去馬刺並不是一個明智的決定。現在，34歲的他將面臨又一次抉擇，繼續留在馬刺註定與冠軍無緣，趁亂離開或許會有「人挪活」的效果。本賽季，阿爾德里奇場均可得18.9分7.0籃板，依舊處於准全明星水準。

3、保羅

保羅是一個倔強而又好勝的球員，即使在羸弱的雷霆，他還是把這一面展現得淋漓盡致。在保羅的率領下，雷霆過去6場取得5場勝利，戰績上升到西部第七。即使這樣，雷霆的未來里還是沒有保羅，保羅的未來也不在雷霆。截止目前，保羅場均可得16.0分4.3籃板5.9助攻，數據較巔峰期明顯下滑，但他的控場能力、領袖作用，對每支球隊來說都是財富。和保羅傳過緋聞的有熱火和雄鹿，無論去哪，都比待在雷霆強。

4、樂福

據名記Shams爆料，樂福已經向騎士提出交易申請，而且要求加盟一支爭冠球隊。爭冠球隊就那麼幾支，主力框架都很穩定，都無法吃進樂福的大合同。樂福來說，能去一支季後賽球隊是更現實的目標，本賽季場均可得15.6分10.5分的他，換個能贏球的環境，或許能找回巔峰狀態。有報道稱，特雷·楊要求管理層引入強援，他若和樂福組成雙子星，足夠讓人憧憬一番。

雖然這四位是全明星球員，但他們的交易去向都不如伊戈達拉更受人關注，因為和他扯上關係的是湖人和快船。快船擁有頂級鋒線配置，他們對「一哥」的需求不如湖人迫切，而「一哥」本人也更傾向於聯手詹姆斯。現在的問題是，灰熊只想通過交易送走伊戈達拉，湖人還沒有合適的籌碼，需要神操作才能成就這樁美事。
