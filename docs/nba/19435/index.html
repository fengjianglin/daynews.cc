NBA一周熱點：尼克斯爛泥扶不上牆 雄鹿群雄逐鹿

2019-12-17 257條評論 2,351次閱讀 470人點贊

賽場大事場外風雲，NBA熱點不斷。在過去的一周里，球賽依然精彩紛呈，球員同樣精彩絕倫。羅斯別利察在同一天分別上演讀秒絕殺。雄鹿繼續保持連勝勢頭，獨孤求敗誰能終結？話不多說，我們一起來盤點上周NBA熱點吧。

羅斯花有重開日

在活塞與鵜鶘的比賽里，替補出場的羅斯前三節表現並不是十分理想，僅僅得到4分，但到了第四節羅斯如有神助，單節8中7得到17分。最後時刻更是連得4分，用一記陀螺轉身中投絕殺比賽，看的球迷都熱血沸騰，曾經MVP史上最年輕MVP彷彿又回來了。或許現在的球迷已經忘記了曾經的玫瑰有多殘暴,但是羅斯會提醒你：」 請原諒我的髒話，我生來就是干這個，絕對認真，這就是我該做的事。」

花有重開日，人無再少年。雖然現在的羅斯已經無法找回曾經的自己，但玫瑰總會在寒冷的冬日裡綻放。

尼克斯爛泥扶不上牆

在炒掉菲茲代爾後他們的確在進步——尼克斯在輸給雄鹿44分後，以37分輸給了掘金。當然菲茲代爾只不過是一個背鍋俠，尼克斯的爛早已經是歷史傳統，不信？我們來盤點一些尼克斯這些年來的鬼魅操作：2010年為了簽下詹姆斯、韋德和波什清理薪資空間，結果來的是小斯；12年選擇簽下39歲的老基德，而不是德拉季奇；14年僱傭菜鳥教練費舍爾；16年為諾阿提供7200萬的大合同；17年處理安東尼合同猶豫不決，導致重建失敗；19年在那麼寬鬆的薪資空間，那麼多大魚的情況下，沒有簽下杜蘭特、倫納德、沃克、歐文中的任何一名球員。

每支球隊的爛各有不同，勇士的爛在於傷病，鵜鶘的爛在於重建，而尼克斯的爛在於傳統——爛泥扶不上牆的爛。

湖人擊破熱火主場不敗

在巴特勒加盟之後，熱火在東部有一人之下萬人之上的勢頭，特別是主場戰績，新賽季險些保持著不敗的戰績。不過也被西部第一的湖人所擊敗，不得不說湖人的實力真的強。這場比賽詹姆斯得到28分9籃板12助攻的准三雙數據，而戴維斯也得到33分10籃板3封蓋，並且投進了一記殺死比賽的關鍵三分。現在的湖人不單單實力強悍，而且隊內氣氛也異常好——雖然詹姆斯上半場送出了7個失誤，後面甚至打得畏手畏腳，但是戴維斯告訴他「我們願意與你赴湯蹈火」，然後才有詹姆斯下半場的狀態復甦。

沃格爾：我等了七年，就是要等一個機會，我要爭一口氣，不是想證明我了不起，我只是要告訴人家，我現在也有詹姆斯了。

群雄逐「鹿」的時代到來

賽季初我們還一度以為新賽季會是宇宙戰艦的快船時代，誰知道原來是群雄逐「鹿」的時代到來。現在雄鹿的戰績一枝獨秀，頗有獨孤求敗的感覺，一波18連勝球迷和其他球隊都望而卻步。場均得分121.2分聯盟第一，場均防守效率101.5聯盟第一，場均籃板51.3個聯盟第一，還有各種各樣位列聯盟前茅的數據，實力詮釋什麼叫攻防俱佳。更為令人恐懼的是，即使球隊絕對核心字母哥休戰，雄鹿依舊可以輕取鵜鶘。

這樣的雄鹿，難道就沒有球隊能終結嗎？本周五湖人客場挑戰雄鹿，湖人：不知道我頂不頂得住呢？
