沒濃眉真難！替身5中1被爆+詹皇客串4號位也不行 遭星二代26+10虐

天天要聞
2019-12-18

北京時間12月18日，當湖人頭號得分手濃眉哥因傷缺席，湖人在客場慘遭步行者兩位數逆轉惜敗，四號位被對手打爆成為重要敗因之一。頂替濃眉哥首發的杜德利僅有5中1僅得到3分表現，外加詹姆斯等客串四號位效果也不佳，相較於對位的「星二代」小薩博尼斯15中10，轟下26+10+4的華麗數據，無疑是被對手給虐得體無完膚。

作為湖人今夏最重要的一筆交易引援，哪怕湖人被倫納德「放鴿子」無緣組成三巨頭，但詹眉組合的強大戰鬥力，無疑還是讓湖人展現出衝冠的態勢。在賽季前15個客場當中，詹姆斯與濃眉哥均是客場得分突破400分大關，成為歷史上首對能夠達到如此標準的雙人組。

只是濃眉哥在上一場湖人力克老鷹之戰，因為右腳踝不慎扭傷，導致他無緣出征對陣步行者的比賽。湖人接下來將迎來一場焦點大戰，他們將客場挑戰字母哥領軍的雄鹿，如此榜首大戰無疑才是重頭戲，自然也是濃眉哥缺席的關鍵因素，他需要留到如此比賽發力。

湖人在賽前無疑還是被看好擊敗步行者，畢竟他們此前已經打出一波7連勝，並且客場更是豪取14連勝，詹姆斯也是歷史上唯一一位30歲以上，還能夠場均奉獻26+7+10的球員，更是令人信服湖人有能力在客場繼續創造神奇。

奈何，湖人僅僅只打了小半節多熟悉的吊打對手好球，他們在首節曾經以24-14壓制對手取得兩位數領先優勢。只是再往後，湖人再也未能夠繼續神勇下去持續吊打，而是陷入到與步行者的瘋狂纏鬥當中。即便是末節湖人曾經再度反超比分，卻在關鍵時刻被布羅格登的神奇連續得分打垮，也連續錯失關鍵三分，讓湖人只能是客場飲恨。

作為頂替濃眉哥空缺的杜德利難辭其咎，作為湖人首發四號位他僅有5中1的準星，並且在場上只是一名3D球員，他在外線的三分出手4中1，導致僅有3分進賬。1.98米的杜德利打四號位無疑是相當勉強，畢竟對位2.11米的小薩博尼斯，他根本沒有任何身高對抗優勢，也是被小薩博尼斯瘋狂碾壓打爆。

儘管湖人還用詹姆斯客串過四號位，外加霍華德與麥基等也是在四五號位遊走，但整體上並未能夠限制小薩博尼斯的威力，導致被對手輕鬆轟下26+10+4打爆四號位，自然也是湖人輸球的重要敗因所在。
