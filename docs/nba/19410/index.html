NBA小故事：麥迪遜廣場花園鬥毆事件 安東尼被禁賽15場

2019-12-17 576條評論 4,886次閱讀 1712人點贊

直播吧12月17日訊 在13年的今天NBA發生了一起著名的鬥毆事件——麥迪遜廣場花園鬥毆事件。這起鬥毆的主要成員安東尼、JR-史密斯和內特-羅賓遜也都被處以嚴厲的禁賽處罰。

在美國時間2006年12月16日，掘金做客麥迪遜廣場花園挑戰尼克斯。在比賽還剩1分15秒的時候，掘金119-100領先尼克斯，但是掘金絲毫沒有偃旗息鼓的意思，主力球員仍在場上。

當JR-史密斯快攻上籃的時候，尼克斯新秀馬迪-科林斯將其狠狠地按了下來。JR-史密斯被這個惡意犯規激怒了，當即衝過去和馬迪-科林斯對噴垃圾話。

裁判抱住了馬迪-科林斯防止事態擴大，尼克斯球員內特-羅賓遜趕來為新秀出頭。安東尼從後面推了一把內特-羅賓遜，隨後JR-史密斯掙脫了大衛-李，朝內特-羅賓遜衝過去，兩人當即扭打了起來。

這時掘金場上的球員也很快的衝過來，場上一片混亂。正當工作人員和球場的安保人員將兩隊的球員分開時，場上的衝突又進一步擴大。

安東尼在和馬迪-科林斯對噴的時候突然一拳打在了馬迪-科林斯的臉上，這也使得尼克斯球員大怒，他們全部沖向安東尼，安東尼見勢不妙往後場跑。尼克斯球員賈萊德-傑弗里斯一馬當先地衝過去想抓住安東尼，結果被馬庫斯-坎比絆倒。最後場上的10名球員全部被驅逐出場。

最後，聯盟對涉事球員進行了嚴厲的處罰。安東尼被禁賽15場，JR-史密斯被禁賽10場，內內被禁賽1場。內特-羅賓遜被禁賽10場，馬迪-科林斯被禁賽6場，賈萊德-傑弗里斯被禁賽4場，傑羅姆-詹姆斯被禁賽1場。
