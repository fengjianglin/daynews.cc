馬刺輸球！米爾斯：蓋伊更衣室評價詹姆斯，大家聽完空氣都凝固了

2019-11-26 0條評論 13次閱讀 0人點贊

北京時間11月26日，NBA常規賽繼續進行。洛杉磯湖人客場挑戰聖安東尼奧馬刺，雙方鏖戰四節，詹姆斯個人砍下33分。最終湖人以114：104贏下了比賽，豪取8連勝。

比賽開始之後，湖人隊獲得首次進攻球權，詹姆斯單打德羅贊率先取到兩分。隨後兩支球隊比賽很膠著，進入到下半場以後，詹姆斯開啟了進攻模式，面對馬刺的5人，幾乎挨個打了一遍，蓋帽、暴扣、3分、助攻，樣樣都有，一套組合拳直接帶走了比賽。

全場比賽，勒布朗-詹姆斯33分3籃板14助攻；安東尼-戴維斯19分12籃板6助攻；肯塔維厄斯-考德維爾-波普14分；拉簡-隆多13分5籃板3助攻；丹尼-格林6分9籃板。阿爾德里奇30分5籃板；德羅贊24分4籃板5助攻；布林-福布斯13分3籃板5助攻；雅各布-珀爾特爾4分8籃板；帕蒂-米爾斯6分。

賽後，米爾斯接受採訪時，被問到更衣室剛才很安靜，發生了什麼？米爾斯：「蓋伊說詹姆斯真的太強了！他在我們的主場，現場球迷狂喊MVP，我的天，我們是不是太菜了！蓋伊說完以後，大家都感到很慚愧，一時間，空氣都凝固了！」本人覺得，蓋伊說得不對，不是你們太菜，是詹姆斯太強了！賽季至今，詹姆斯場均助攻11次，聯盟第一，而且場均25.6分的得分，無所不能的詹姆斯征服了太多對手！
