18日大數據：關鍵罰籃不中榜魔獸詹皇前2！吹楊空砍 尼克斯吃餃子

天天要聞
2019-12-18

今日賽況：

國王102-110黃蜂 湖人102-105步行者 老鷹120-143尼克斯 籃網108-101鵜鶘 魔術102-109爵士 太陽99-120快船

1、NBA官方評選今日最佳數據：丁威迪31+7當選

NBA官方評選出了今日比賽的最佳數據，丁威迪31+7當選。今日比賽，丁威迪26投11中，三分8中1拿下31分7助攻的數據，幫助籃網加時戰勝鵜鶘。此外，這場比賽也是本賽季丁威迪第4次得分30+。

贏球卻有爭議！常規時間籃網的最後一攻引起了鵜鶘隊上下的強烈不滿。當時兩隊93平，丁威迪持球進攻，在進攻時間還剩2秒時出手三分，但是球並未碰筐，亮燈後球還在空中飛行，最後球砸到的也是籃板，然後才掉到了鮑爾手裡。

賽後鵜鶘主帥金特里透露自己當時三次要求裁判回放那一回合，但都被拒絕了。主裁Kevin Scott賽後談到未停表時表示：「在那時裁判們確信球沾筐了，所以未停表。」談到未進行回放，Kevin Scott表示：「不能回放，因為立刻交換了球權並沒吹死球，所以是無法看回放的。」

2、湖人客場14連勝被終結 本賽季首次在洛杉磯之外輸球

湖人客場102-105憾負步行者，客場14連勝紀錄被對手終結。此外，這也是湖人本賽季首次在洛杉磯之外的地方輸球。湖人的客場14連勝位居歷史第三名，前兩名分別是1971-72賽季的湖人（客場16連勝）和1994-95賽季的爵士（客場15連勝）。本場比賽過後，湖人本賽季的戰績變為24勝4負，仍位居西部第一。

3、雄風猶在！霍華德10中10砍20分 出手10+命中率百分百生涯第三次

湖人客場102比105惜敗給步行者，7連勝被終結。今日戴維斯因傷缺陣，霍華德扛起湖人內線大旗，全場他出戰27分鐘，10投全進砍下20分6板2帽。這也是霍華德職業生涯第三次單場至少出手10次且命中率百分百，此前兩次分別為2010年2月25日對陣火箭的11投全進30分16板3助以及2015年11月5日對陣魔術的10投全進23分14板2助。

4、詹姆斯生涯罰球命中數超越諾維茨基 升至歷史第六

湖人客場102-105不敵步行者。此役，詹姆斯罰球線上8罰4中，生涯罰球命中數來到了7243個，超越德克-諾維茨基（7240），升至歷史第六位。目前排在詹姆斯前面的是喬丹，他生涯共命中7327個罰球。

5、沃格爾從步行者下課後 率隊重返印城5戰全負

湖人102-105客場不敵步行者，7連勝被對手終結。有意思的是，這是沃格爾從步行者下課後（15-16賽季後）第5次率隊重返印第安納，然而至今他仍未在這裡取得過一場勝利。此前4場比賽分別是：2016年11月15日，魔術69-88客負步行者；2017年1月2日，魔術104-117客負步行者；2017年11月28日，魔術109-121客負步行者；2018年1月28日，魔術112-114客負步行者。

6、尼克斯單場得到143分 隊史近13年來最高得分

尼克斯在主場以143-120大勝老鷹，送給對手五連敗。尼克斯單場得到143分，是隊史近13年來的最高得分，上一次是2006年12月28日的151分。在尼克斯隊史上，常規賽的單場最高得分是152分。

7、帶隊吃餃子！巴雷特27分6籃板 得分生涯新高

今天尼克斯主場迎戰老鷹，最終尼克斯收穫了21世紀以來的最高分，以143-120擊退了來犯的老鷹。全場新秀RJ-巴雷特一共出戰了28分鐘，13投10中（三分3中2）得到全隊最高的27分6籃板1助攻1搶斷，進攻端出色的他，還上演了對雷迪什的精彩單臂隔扣。值得一提的是，27分也刷新了他生涯的得分紀錄，之前最高為2019年10月27日對陣凱爾特人的26分。

據報道巴雷特（19歲）和米切爾-羅賓遜（21歲）成為隊史首對21歲以下且同場得分20+的二人組。本場比賽，這兩位球員得分均創職業生涯新高。尼克斯隊RJ-巴雷特出場29分鐘，得到27分6籃板1助攻1搶斷；米切爾-羅賓遜出場29分鐘，得到22分13籃板1搶斷2封蓋。

8、湖人本季最後3分鐘分差3分以內比賽首次輸球 此前為10連勝

湖人在印城3分不敵步行者，七連勝&客場14連勝被終結。據統計，這是湖人本賽季最後3分鐘分差3分以內比賽首次輸球，此前已經拿到10連勝。目前，湖人以24勝4負排名西部第一，下一場將是球隊東部五連客最後一役，對手是東部領頭羊雄鹿。

9、遺憾空砍！特雷-楊斬落42分4板8助

今天老鷹客場挑戰尼克斯，在大比分落後的情況下特雷-楊打滿末節且得到15分率隊縮小分差，無奈球隊此前挖坑太大，最終還是120-143大比分不敵尼克斯，球隊遭遇5連敗。全場特雷-楊一共出戰了41分鐘，29投16中（三分13中7），罰球4中3 ，拿下全場最高的42分外加4籃板8助攻，雖然數據很搶眼，但是他的在場凈勝分為-18為全場最低。

10、魔術單場3失誤 平隊史最低紀錄

魔術102-109不敵爵士。本場比賽，魔術僅僅失誤3次，這追平了隊史失誤最低紀錄數。2002年1月19日魔術對陣雄鹿和1996年4月1日魔術對陣尼克斯，這兩場比賽魔術均僅送出3次失誤。本場比賽過後，魔術12勝15負位列東部第8名。

11、鵜鶘慘遭13連敗 繼續刷新隊史紀錄

鵜鶘101-108加時不敵籃網，慘遭13連敗。這樣一來，鵜鶘的隊史連敗紀錄被刷新到了13場，在此次連敗之前，他們的隊史連敗紀錄為11場。本賽季至今，鵜鶘6勝22負位列西部第14名。

12、小卡泡椒路威均砍下15+時 快船7勝0負

快船120-99戰勝太陽，豪取主場10連勝。本場比賽，小卡拿下20分6籃板，喬治拿下24分3籃板，路威砍下20分8助攻。據統計，本賽季當三人同時得到至少15分時，快船取得了7勝0負的佳績。快船已經在主場拿到了10連勝，追平了隊史在主場的第三長連勝紀錄。快船隊史在主場的最長連勝紀錄是13場，是12年11月28至13年1月9日創造的，另外快船還拿過一次主場11連勝，是1978-79賽季創造的。

13、太陽連續11場29+助攻 歷史第二隊

太陽99-120不敵快船，慘遭4連敗。值得一提的是，太陽本場共送出30次助攻，這也是他們連續第11場送出至少29次助攻，他們是歷史上第二支做到這一點的球隊。歷史上第一支做到這一點的球隊是1988-89賽季的黃蜂，他們曾連續13場送出至少29次助攻。本場比賽過後，太陽11勝16負，位列西部第10名。

14、過去10年關鍵時刻投丟罰球次數排行：魔獸&老詹前二 哈登第三

湖人客場不敵步行者，這場比賽最後時刻詹姆斯和霍華德連丟4記罰球，成為輸球的關鍵因素，賽後美媒統計了過去10年關鍵時刻丟罰球排行榜：

2、詹姆斯 122次 命中率75%

3、哈登 110次 命中率82%

4、小喬丹 106次 命中率57%

15、13年前的今天：大將軍&科比上演對飈大戰 前者狂砍60分

13年前的今天，奇才客場加時以147-141擊敗科比率隊的湖人。「大將軍」阿里納斯與科比上演對飆大戰，加時賽更是獨取16分（湖人全隊15分）力克湖人。此役，阿里納斯出戰49分鐘，投籃32中17，三分12中5，罰球27中21，奉獻60分8籃板8助攻2搶斷；科比出戰47分鐘，投籃24中15，三分11中7，罰球10中8，貢獻45分8籃板10助攻。

同時，阿里納斯成為繼張伯倫後第二個面對湖人砍下60+的球員，也是唯一在科比頭上砍下60分的球員。

16、28年前的今天：騎士狂勝熱火68分創歷史最大分差紀錄

在28年前的今天，騎士主場迎戰熱火。該場比賽上半場騎士以73-53領先熱火20分，第三節騎士打出了33-14，將分差擴大到了39分。第四節熱火的狀態依舊不佳，被騎士打出了一波42-13,，單節再次凈勝29分。最終騎士以148-80血虐熱火。68分的分差也創造了NBA歷史最大的分差紀錄。
