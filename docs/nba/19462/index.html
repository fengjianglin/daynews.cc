2010年湖人雙塔成NBA內線絕唱，拜納姆之後再無進攻型傳統中鋒

2019-12-17 582條評論 1,983次閱讀 253人點贊

2010年的黃綠大戰，一直被視為最經典的總決賽之一，因為那個系列賽充斥著競爭、肉搏與對抗。很多人甚至會評價，「2010年之後再無肉搏戰」，這只是一句誇張的語句，來形容2010年肉搏戰的激烈，可曾想，竟然成真了！

2010年之後，真的再也看不到肉搏戰了。實際上2011年就是一個轉折點，那一年小牛用精準的投射能力，先後擊敗了湖人、雷霆和熱火，奪得總冠軍。那一年小牛至少還有錢德勒這樣的防守中鋒，還有諾維茨基這樣的超級內線。

而2012年開始，內線再也不是主角了，熱火三巨頭捲走2個冠軍，14年的馬刺，鄧肯也只是甘做綠葉，隨後的勇士更是打起了死亡五小的戰術，而16年的騎士同樣不是靠內線的球隊，哪怕是上賽季的猛龍，馬克加索爾和伊巴卡，更多的作用還是防守。

從某個角度講，拜納姆之後再也沒有進攻型的傳統中鋒了，包括霍華德也是！再早的比賽不知道，但21世紀以來，鄧肯、加內特、姚明、奧尼爾、諾維茨基、韋伯這些全明星內線，一直就是場上的焦點，從1999年-2010年，NBA12個總冠軍里，奧尼爾拿了4個，鄧肯拿了4個，加內特拿了1個，活塞拿了一個，而最後2個是湖人拿的，他們的內線是保羅加索爾和拜納姆的雙塔。

那時候我們深信不疑，得內線者得天下，不是因為赤木剛憲的洗腦，而是實實在在的從比賽中得出的經驗，強如科比，同樣要有內線的支持才能奪冠。而湖人2010年的雙塔，成為了NBA內線的絕唱，很多人認為湖人之所以能擊敗凱爾特人，是因為第七場帕金斯的缺席，讓湖人在籃下徹底刷爆了籃板，這是事實，也再一次說明了內線的重要性。

在湖人雙塔之後，沒有任何一支球隊的內線能夠在球隊里有這麼高的地位，都沒有，小牛隻有諾維茨基，馬刺也開始變得向外線靠攏，至於熱火和雷霆，都是以外線為主。我們沒辦法判定，是小球擊敗了傳統內線，還是傳統內線的沒落成就了小球，但我們可以清楚地知道，德拉蒙德格林是肯定防不住拜納姆的。

拜納姆是NBA最後一個進攻型傳統中鋒了，雖然他的心智不是很成熟，但就內線進攻而言，拜納姆無疑是當年NBA最強的，姚明在籃下的攻擊力都不如拜納姆，但他的中距離更准，至於霍華德，他的自主進攻能力，真的跟拜納姆不是一個檔次的。

在拜納姆之後我們很難看到，有球員正兒八經地在籃下卡位要球，而外線球員也在認真地等他落後，給球，然後背打，或轉身、或虛晃、或切入暴扣、或輕巧勾手，沒有，看不到了，一方面沒有這樣的球員，另一方面你在卡位的時候，可能哨聲就已經響了。所謂時勢造英雄，一起淘汰傳統內線的還不僅僅是小球，還有裁判的選擇。

奧尼爾也曾經說過，如今NBA只有3個傳統中鋒，馬克加索爾、小喬丹和霍華德，小喬丹不會進攻，霍華德只會左手佯攻，只有馬克加索爾有點威脅，這話一點都不誇張，巔峰奧尼爾，可以打得德拉蒙德格林懷疑人生，這不是說德拉蒙德格林防守不好，而是當年的NBA，身高和體重真的是一種優勢。

老球迷和老球員們喜歡厚古薄今，是因為他們知道在他們熟知的那個年代裡，現在的球員根本不可能打得這麼舒服，也打不出什麼名堂來，這個是真的！

新球迷們喜歡說老球員在現在NBA也不一定能夠那麼出色，這也是真的！因為兩邊的規則和判罰標準不一樣，註定了兩個時代的球員要有不一樣的選擇方向，以前你需要身體、技術，現在你只需要投得准就行，沒那麼對抗給你，因為裁判也不允許。

而2010年竟然成為了一個分割線，湖人就成為了內線的絕唱。
