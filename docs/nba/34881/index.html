湖人三連敗！濃眉膝蓋扭傷，老詹因傷缺陣！更險峻的考驗在三天後

2019-12-23 1,128條評論 2,214次閱讀 146人點贊

在此前遭遇胸廓肌拉傷後，詹姆斯帶傷打完與步行者和雄鹿的比賽，這也說得通為何老詹在那兩場缺少了往日的攻擊性。昨日，詹姆斯缺席了球隊的公開訓練，並只和訓練師練習了簡單的背身單打轉身後仰。今日，湖人官方宣布，詹姆斯因傷將缺戰與掘金一戰，這也將是詹姆斯本賽季首次缺席常規賽。

在連續出戰29場，場均34.8分鐘拿到25.8分7.5籃板10.6助攻後，詹姆斯終於因傷缺戰比賽了。注意，這裡是因傷缺席，而不是輪休。帶傷出戰將隱患重重，老詹可以利用最近幾天時間來康復，打好與快船的聖誕大戰，奠定下一波贏球的信心。

此前因左腳踝扭傷缺戰5場的庫茲馬在今日迎來複出，上場時間控制在20分鐘以內。湖人派出的首發陣容為隆多、布拉德利、丹尼-格林、濃眉和麥基。

詹姆斯在賽前發布了一張帶有文字的圖片，上面寫道：「完全沒有理由！無論他們多麼希望你失敗，請不要讓這阻止你前進。」

臨近開場的球員通道熱場環節，庫茲馬：「我應該在車底，不應該在這裡。」

開局後，麥基絲毫不怯場，面對全明星級別中鋒約基奇背轉身打成2+0。

不需要上場，詹姆斯愜意地坐在場下觀賽，當一回球迷。

哈里斯這球都快走回丹佛了，說實話，我朋友圈要是有他這樣的人，微信運動估計永遠也拿不到第一名。

濃眉單打約基奇，也顯得輕鬆。

天氣漸漸轉涼，庫茲馬請大家吃火鍋，你要鴛鴦鍋還是麻辣鍋？

第一節最後階段，濃眉撿到前場籃板後的補扣，有點暴力。

首節湖人28-23領先，濃眉11分5籃板，隆多5分2籃板2助攻3搶斷，布拉德利3分，麥基2分4籃板，卡魯索4分1搶斷，庫茲馬3分，格林2籃板1助攻。

波普在第二節開局一分鐘後飆中三分。

禿曼巴的膝蓋不知道磕誰身上，又禿嚕皮了。

庫茲馬：「聽最近有流言說，主教練認為我不適合湖人？要賣我？荒唐么不是。」

第二節最後階段，濃眉還是像第一節最後那樣，搶到搶場籃板雙手補扣。

半場湖人53-55落後，濃眉19分8籃板1蓋帽，隆多5分4籃板2助攻3搶斷，卡魯索6分，庫茲馬9分3籃板2蓋帽1搶斷，霍華德6分4籃板1助攻，波普3分。

第三節，裁判成為了主角。濃眉的好帽被吹了犯規。湖人一度被對方打出10-0落後兩位數。

這一節的進攻基本是濃眉與庫茲馬在苦苦支撐。

濃眉這一下又把膝蓋給掰了，趕緊下來接受隊醫檢查。

此節結束，湖人78-91掘金。濃眉27分10籃板3蓋帽，庫茲馬14分4籃板2蓋帽。第四節開始，濃眉帶傷留在場上，掘金連續投進三分，湖人很快落後20分，比賽接近崩盤。最後4分鐘，湖人依舊落後20分，主教練沃格爾才將濃眉換下場。

比賽花到麥基居然從後場持球去到前場，命中了個人本賽季第一個三分球。

最終湖人104-128掘金，濃眉32分11籃板1助攻4蓋帽，隆多5分4籃板8助攻3搶斷，庫茲馬16分4籃板2蓋帽，格林5分4籃板2助攻，麥基7分5籃板2蓋帽，波普6分3籃板2助攻，卡魯索8分2籃板，霍華德8分7籃板，庫克2分2籃板，丹尼爾斯7分，杜德利2分1籃板。

輸了比賽不要緊，最重要的是濃眉膝蓋沒有大礙。三天後，聖誕大戰對陣快船，更嚴峻的考驗等著湖人。
