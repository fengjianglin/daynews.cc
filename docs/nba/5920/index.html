東契奇超喬丹成歷史第一人，高興太早！西部第2恐一輪游水平

2019-12-09 0條評論 17次閱讀 0人點贊

東契奇，正在創造自己的神話故事。北京時間2019年12月9日，獨行俠主場以106-110負於國王隊，東契奇拿下27分7籃板8助攻，連續19場比賽至少得到20分5籃板5助攻，超越排名第一的邁克爾-喬丹歷史第一，喬丹曾經在1988-89賽季曾連續18場得到這一數據。

之前東契奇三節轟下26分6籃板9助攻， 連續18場打出20+5+5追平邁克爾-喬丹，很多人都認為他打出歷史第一的這個連續紀錄只是時間問題，現在東契奇一個人將獨行俠帶到這樣的高度，是沒有人想過的，如今他也是MVP榜單的第二號人物。

獨行俠現在的陣容，你甚至除了波爾吉尼斯之外都叫不上來什麼名字了，這個球隊在開賽之初只是被討論能否進入季後賽，而如今這樣的陣容進入了西部第二的位置，誰都做不到，東契奇的表現確實很神奇。不過國王在這場比賽反客為主的表現更神奇。

半場國王贏了獨行俠20分讓所有人大跌眼鏡，半場他們讓獨行俠才得到46分，而希爾德，別利察以及霍姆斯的表現都非常驚艷，半場都拿下兩位數的表現。東契奇一個人果然是帶著一個爛隊在前行，波神才4分，要不是小哈達威拿下17分，獨行俠可能更慘。而東契奇半場10分4籃板5助攻，依然是朝著超越喬丹的紀錄前行的。

下半場開始之後，東契奇繼續開始表現，雖然球隊的表現還是差點，但是東契奇第三節開始不斷的得分，三節不到的時候終於轟下20+5+5的數據，成為了這個數據的歷史第一人。實際上如果算上aba，1960-61賽季的大O羅伯特森曾經創造了分別為19場、25場、29場的紀錄，不過這一次東契奇超越的是NBA喬丹版本的紀錄。超越了喬丹的紀錄固然不錯，但是獨行俠現在的問題也很多，那就是一人球隊的狀態越來越明顯。東契奇在比賽最後幾分鐘力挽狂瀾的表現必須點贊，但最終還是功虧一簣。

這個球隊甚至連波爾吉尼斯都靠不住，如果只是這樣的狀態，即便是拿到了西部第二甚至東契奇拿下MVP獎盃，獨行俠在本賽季的前景也不會很好，頂多就是季後賽的一輪游。
