勇士隊榮獲“第一”，告訴我們兩個道理，人生亦是如此

天天要聞
2019-12-05

因為那個懂球的胖子，筆者對NBA的關注度下降了一些，以前是每場球必定要了解一下，或者是看視頻，或者是看戰報，反正每天玩手機花的時間，一般都與NBA有關。現在不一樣了，當然，現在偶爾也看球，但是明顯消耗的時間要少很多。

詹姆斯帶隊坐穩西部第一

也正是如此，對最近NBA的一些情況並不是太了解，今日閑來無事，刷了一刷NBA，看了看榜單，湖人西部第一，雄鹿東部第一，兩者戰績相同18勝3負，還不錯嘛。不過當我往下拉的時候，就尷尬了，在西部的榜單中一直沒發現勇士隊的身影，直到拉到最下面，才看到勇士，尷尬的倒數第一，戰績為4勝18負，目前榮獲「聯盟第一」，估計這個結果也是打了不少專家的臉，筆者也是其中一個，當然，不是說筆者是專家，而是沒想到勇士隊能夠差到這個地步。

太可怕了，就幾個月不看NBA，勇士隊就淪落到這個地步了，前幾年說到勇士隊就聯想到總冠軍，什麼連勝啊，73勝啊，季後賽一路橫掃啊，看得都麻木了，前兩個月，誰跟你說勇士隊會是倒數第一的球隊，你可能會想這個人有病吧，不懂球就別亂說。但是，事實上從一支超級宇宙隊，到倒數第一原來也這麼的近。本賽季勇士隊這麼差勁，最重要的原因毫無疑問是傷病，先是湯普森的賽季報銷，又到庫里傷停，格林和拉塞爾也相繼傷停，這四位億元先生不能打，對勇士隊來說真的是打擊非常大，從一支全明星球隊，跌成平民球隊，能不差勁嗎？一批拿著底薪的球員和其他球隊的億元球員來拼，能贏才怪。雖然格林後面也能打了，不過他並不是勇士隊的救世主。

杜蘭特出戰，庫里湯普森傷缺

從勇士隊的情況來看，告訴了我們兩個道理：第一，三十年河西三十年河東，沒有常勝將軍。過去強，不代表現在強，更不代表未來強，一支球隊想一支贏下去是非常難的，一支球隊一直保持一帆風順也是很難的，現在勇士正在遭遇著最大的挑戰，等他們挺過了這段艱難的時日，待庫里湯普森回歸，他們的戰鬥力還是非常不錯的。

第二，傷病如猛虎，身體好才是真的好。雖然說今年休賽期，杜蘭特離隊，但是庫里、湯普森和格林勇士的三大法寶還在，他們依然是奪冠的熱門，但是因為湯普森、庫里接連受傷，讓他們兵敗如山倒，不得不說，再厲害的球員，再傷病面前也是黯然失色。其實這兩點和人生也有點相似，很難有一帆風順的人生，難免有各種磕磕碰碰，在這個過程中，身體健康才是最重要的，身體好才是真的好。

最後，想說的一點是，本賽季西部的球隊競爭力都非常強，勇士隊極有可能成為一支連續幾年進入總決賽，而後無緣季後賽的球隊，您覺得呢？
