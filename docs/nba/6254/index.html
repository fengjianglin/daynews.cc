卡皇復出！哈登已讓所有人信服！馬刺得分王相信哈登將得到100分

2019-12-04 0條評論 16次閱讀 0人點贊

北京時間12月4日早上9點半火箭客場對戰馬刺，這是雙方本賽季的第一次對決，上賽季常規賽雙方3—1火箭占絕對優勢，其中有一場31分大勝對手。

火箭這邊目前以13勝6負戰績繼續排西部五，最近球隊取得兩連勝，在上一場與老鷹的比賽中更是47分狂勝，當家球星詹姆斯-哈登在前三節的時間砍下60分。談及哈登的表現，馬刺隊員德羅贊說道：「我很確定，如果他下定決心，他可以（單場）拿下100分。不過我不會支持他在明天比賽中做到，他可以改天再做。」哈登和德羅贊都是在洛杉磯長大，在高中時就是籃球場上的對手和兄弟，兩人也同為2009屆選秀球員。

「即便是在大學（亞利桑那州大），他（哈登）也曾在一段時間內領跑全國（得分），」德羅贊說道，「所以對我而言，這（哈登的得分盛宴）並不奇怪。」對於自己的兄弟賽前如此毒奶，哈登說道「我就是保持攻擊性，利用對方的防守為隊友創造機會，我們正在想辦法努力破掉對方的防守策略，看看對方的防守時機以及什麼時候上來包夾我，作為一個職業球員我很享受這個時刻」。很顯然，最近各支球隊對哈登的得分盛宴很是頭疼，不得不採取包夾戰術，但這並沒有困擾著哈登。

卡佩拉會在本場比賽中復出，此前他因病缺席了休斯敦的最近兩場比賽，卡佩拉賽季場均拿到14.6分14.7板1助0.87斷2冒的全面數據，最近七場比賽更是至少抓下19個籃板，他的復出給球隊注入防守和活力。然而，小前鋒丹紐爾-豪斯， 他可能會因為自己的疾病而繼續缺陣，可能幾天後才會回來。 卡佩拉錯過了對邁阿密和亞特蘭大的兩連勝，而豪斯錯過了第二場比賽。

馬刺隊這邊本賽季沒有達到預期，目前戰績僅僅取得4勝14負排西部第十二，與同區重建的灰熊隊平起平坐，在球隊坐擁雙德組合取得這樣的成績顯然不合格，最近因為輸球原因，球隊內幕更是出現內訌現象，昔日的德州豪強退步之快令人痛惜。如果球隊接下來沒有走出困境，估計會選擇交易隊中核心球員也不意外。馬刺本賽季防守效率聯盟倒數第四，場均丟掉115分，這顯然主帥波波維奇無法接受。

球隊在夏天引進了優質3D悍將卡羅爾，但目前僅僅交出2.5分和2.7板，投籃命中率28%三分命中率25%的慘淡數據，後衛德章泰-默里本賽季也沒達到預期，被主帥波波維奇按在了替補席，賽季僅僅9.6分6.1板4.2助，三分命中率將要跌破20%。雙德組合還是保持穩定貢獻場均為球隊拿下40.5分12.2板7助的數據。阿爾德里奇每次打火箭隊必爆發，明日之戰火箭想要贏球必須重點防好這個點，但好消息是，阿德目前出現大腿酸痛，大可能不會出戰本場比賽。米爾斯也是亂戰高手必須提高警惕，每次打火箭都能貢獻關鍵球。

雙方預計首發陣容：

火箭：維斯布魯克、哈登、麥科勒姆、塔克、卡佩拉

馬刺：福布斯、德里克-懷特、德羅贊、蓋伊、珀爾特爾
