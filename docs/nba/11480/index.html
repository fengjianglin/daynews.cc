怒噴裁判！砸飛技術統計！馬龍為何憤怒？三細節證皆為一人

2019-12-11 123條評論 1,087次閱讀 56人點贊

文/水清清

北京時間12月11日，76人與掘金比賽行至第二節7分18秒處罕見一幕發生，掘金主教練馬龍罕見的怒噴裁判、砸飛助理教練技術統計，直接吃T，那麼是什麼原因讓他如此憤怒？三細節印證皆為一人！

邁克馬龍怒了

馬龍雖不是溫文爾雅型主帥，但也不是點開的性情式主教練，為何突然間暴怒呢？畫面如下：

約基奇進攻，未吹霍福德犯規

如上圖，比賽行至7分19秒，掘金33-46已落後13分，這時馬龍換上約基奇，希望他加強內線進攻，而後者也果斷執行馬龍戰術，從左底線揉至油漆區，頂著霍福德防守強行攻擊籃下，結果在霍福德干擾中皮球彈筐而出，裁判未作出霍福德防守犯規判罰，於是馬龍怒了。

馬龍怒噴裁判

用王猛老師的話形容，「馬龍是要吃了裁判啊」，嗯，他直接沖入場內，對底線裁判一陣狂吼，大體意思是這還不吹打手犯規？助理裁判一看馬龍情緒失控，於是馬上介入至二者中間，試圖勸說馬龍，可結果則更罕見。

馬龍揮拳發泄，將技術統計砸飛一地

馬龍顯然余怒未消，並且主裁判已經作出技術犯規手勢，氣急的馬龍一揮拳，結果上來勸他的助理教練技術統計成為犧牲品，直接砸飛一地。

那麼馬龍僅僅是因為這一次判罰的不滿才如此惱怒？不盡然，一切皆為約基奇。

一、至這次出手為止，約老師登場的11分鐘時間裡竟然沒有一次出手，如此表現讓馬龍極其不滿。

二、對面的恩比德此時已經轟下12分，兩人懸殊差距是球隊出現13分被動局面直接原因。

三、作為球隊領袖，約老師本賽季不僅數據出現滑坡，且賽場積極性始終不高，以致球隊戰績出現巨幅下滑，目前以14勝7負跌至西部第四了。

這就不難理解馬龍為什麼如此動怒了，他不僅僅因為這一次判罰，而是想以此為代價，喚醒隊員、尤其是約老師的戰鬥意志，效果？約老師隨後4投2中取4分4籃板6助攻，引領掘金半場將比分追至50-55，保留著逆轉希望。

嗯，一位主帥的技術犯規，實際上往往是戰術一部分，除給裁判施壓外，更主要的是為喚醒球員鬥志。
