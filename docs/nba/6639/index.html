NBA中的“錦鯉”球員，其中有位“粘著”詹姆斯豪取三冠軍戒指

2019-11-27 0條評論 17次閱讀 0人點贊

NBA當中有很多球星追逐了一生，也沒有拿到過總冠軍，而有那麼一些球員，雖然並不是球星，但是他們卻隨隊拿到了很多枚冠軍戒指，可以說有他們在，就是球隊的吉祥物。接下來我們就來看看NBA中的「錦鯉」球員，他們負責「躺著奪冠」！

10、亞當·莫里森，2枚

亞當·莫里森是一位NBA的邊緣球員，但是由於他正好加盟了湖人隊，在幾乎沒有出場的情況下拿到了兩枚總冠軍戒指，他可以說是一位飲水機球員，在長下揮舞著毛巾為球隊加油助威。

9、姆本加，2枚

姆本加同樣也是追隨湖人隊拿到了兩枚總冠軍戒指，當時是湖人兩連冠的時候，姆本加充當拜納姆的替補，打打醬油就獲得了總冠軍戒指，真的令那些終身無冠的球星羨慕不已。

8、邦納，2枚

邦納是一名三分球射手，他有著一定的實力，由於她的皮膚髮紅所以被稱為「紅曼巴」，在2007年的時候，邦納作為馬刺隊的替補球員取得了一枚總冠軍戒指，在那之後他一直留在馬刺隊並且在2014年又一次隨隊拿到了總冠軍，因為他心甘情願的做一個飲水機管理員，邦納幾乎沒有為球隊做出任何貢獻，只是躺贏就拿到了冠軍。

7、馬克-馬德森，3枚

這一位仍然是在湖人躺贏的球員，並且他在「OK組合」的創立的湖人王朝中連續三年追隨湖人，並且拿到了總冠軍。他在比賽中幾乎沒有什麼出場機會，也未能給湖人隊做出貢獻。

6、詹姆斯-瓊斯，3枚

眾所周知，瓊斯是勒布朗-詹姆斯的「貼身小弟」，雖然在年齡上他比詹姆斯大，但是他就是一直「粘著」詹姆斯混到三個冠軍。詹姆斯在熱火隊時他就是詹姆斯的隊友，跟隨詹姆斯拿下了兩個總冠軍戒指，在詹姆斯回歸騎士以後他也追隨詹姆斯，到了騎士又拿到了一個冠軍。

5、迪克-辛普金斯，3枚

辛普金斯也是在公牛王朝過程中追隨公牛隊拿下了三個總冠軍戒指，他就是喬丹身邊一個名不見經傳的球員，在球場上也發揮不了多大的作用，但是他的運氣實在是太好了，連續三年追隨公牛王朝獲得總冠軍。

4、帕特里克-麥考，3枚

麥考是最近幾年中最有名的「錦鯉」球員，他從新秀入選以來，不到幾年，就已經拿下了三個總冠軍戒指。前兩個是隨勇士隊拿下的，第三個冠軍是在他離開勇士轉投猛龍後，猛龍又獲得了總冠軍。NBA流傳著這樣一句話「得麥考者得總冠軍」。

3、弗蘭克-佩普-索爾，4枚

索爾也是隨著湖人隊拿到了第一枚總冠軍戒指，並且在之後他追隨著喬治麥肯繼續征戰NBA，並且又拿下了三個NBA總冠軍。

2、詹姆斯-洛斯庫托夫，6枚

這位球員終於不是湖人隊的球員了，他是如今總冠軍數量第一的凱爾特人的「吉祥物」。在凱爾特人拿到六個總冠軍的時候，總有一位這樣的球員默默的看護著飲水機，也許當時沒有飲水機，看護著保溫桶吧！他就是綠軍球員，洛斯庫托夫。

1.羅伯特·霍里，7枚

羅伯特.霍里，是榜單唯一一個憑藉自己實力幫助球隊拿下總冠軍的角色球員，他一共有6枚總冠軍戒指，一隻手都帶不滿。他的總冠軍戒指含金量非常高，他在比賽中經常能夠在關鍵時刻，投進重要的三分球，要麼幫助球隊絕殺對手，要麼幫助球隊鎖定勝局，可以說他是一個非常有能力的「錦鯉」球員。
