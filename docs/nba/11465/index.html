熱火紀錄之戰讓特雷楊自嘲！巴特勒花式嘲諷，格林湯普森也湊熱鬧

2019-12-11 1,669條評論 4,129次閱讀 1513人點贊

北京時間12月11日，熱火隊主場歷經加時賽以135-121大勝老鷹隊，取得3連勝。雖然熱火隊在第四節還剩1分鐘時落後6分，但他們之後連中兩記三分球將比賽拖入加時賽，而且還在加時賽打出16-0的進攻高潮奠定勝局。熱火隊此役首發除了梅耶斯·倫納德外其他四人得分都上20，首發總計得到124分，創造熱火隊史最高紀錄，熱火隊還創造了其他多項紀錄，可謂紀錄之戰。

肯德里克·納恩此役26中14、三分球10中3砍下36分6籃板4助攻（36分是個人生涯最高紀錄），鄧肯·羅賓遜16中12、三分球14中10砍下34分3籃板4助攻（34分、4助攻以及10記三分都是個人生涯最高紀錄），巴姆·阿德巴約18中13砍下30分11籃板11助攻2搶斷（30分和11助攻都是個人生涯最高紀錄，個人生涯首次三雙），吉米·巴特勒19中8、三分球3中1砍下20分18籃板10助攻（18籃板是個人生涯最高紀錄），這也是熱火隊歷史上首次同時兩名球員完成三雙，納恩和羅賓遜還成為NBA歷史首對同砍30+的落選秀隊友。

值得一提的是，在第四節還剩1分鐘時，特雷·楊巧妙助攻隊友得分，當時老鷹隊以117-111領先熱火隊6分，楊難掩興奮向觀眾席怒吼：「It’s over! It’s over!」（比賽結束！比賽結束！）。之後發生的事情我們都知道了，熱火隊連中兩記三分將比賽拖入加時，並統治加時賽。

在賽後，特雷·楊發推自嘲：「……啊哦（一臉懵逼、攤手以及笑哭的表情）」。吉米·巴特勒賽後在社交媒體上發了特雷·楊慶祝比賽結束的視頻，並寫道：「這個男人特雷·楊，是一位來自未來的預言家，他是對的，比賽結束了。」

有趣的是，德懷恩·韋德調侃了巴特勒，他評論道：「你這人有問題，但我喜歡。」除此之外，勇士隊的德雷蒙德·格林和克萊·湯普森都來湊熱鬧，追夢格林發了一串笑哭和捂臉的表情，而湯普森也發了一串笑哭的表情，還寫道：「跟他們談談吉米」。
