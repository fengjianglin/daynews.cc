安東尼談妻子現場觀戰：家庭對我的支持意義重大

2019-11-26 0條評論 14次閱讀 0人點贊

虎撲11月26日訊 在今天開拓者客場以117-94戰勝公牛的比賽中，開拓者前鋒卡梅羅-安東尼發揮出色

賽後，安東尼在場邊接受媒體採訪。

安東尼首先談到自己今天的比賽：「我在逐漸找回自己的感覺和節奏，我就是一天一天來。上場比賽對陣騎士我打得不好，畢竟我是時隔一年直接4天打了3場比賽，但今天我感覺自己找回了彈跳，打得不錯。今晚的比賽我們必須拿下，是一場在客場之旅中不錯的勝利。」

談到芝加哥球迷對自己的支持，安東尼說道：「這些支持讓我感到不可思議，我認為可能很多人喜歡看到我繼續做好自己的表現。我的努力為自己贏得了這樣的時刻，但這只是一場比賽，我們會繼續努力。」

談到妻子拉拉場邊觀戰，並與兒子基揚視頻連線，安東尼說道：「有時候我們沒有很珍惜這樣的時刻，把這種事情視作理所當然。但這次，我妻子在場邊，我兒子在視頻里，他們都在這裡，我家庭對我的支持都在這裡，這意義重大。每次當我們有機會這樣做的時候，我們會抓住的。」

談到本場勝利，安東尼說道：「這至關重要，每次在客場取勝，尤其是以勝利結束客場之旅都很重要。」

安東尼今天上場31分鐘得到25分8籃板2助攻。
