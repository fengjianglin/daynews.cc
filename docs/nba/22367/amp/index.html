07年科比場均31.6分5.4助攻，常規賽MVP諾維茨基什麼數據？

天天要聞
2019-12-19

科比是繼喬丹之後最偉大得分後衛，作為一名外線球員科比擁有出色進攻和防守，職業生涯11次最佳一陣，9最佳一防就是最好證明，科比職業生涯20年全部在湖人隊度過，期間在2006年，2007年獲得得分王，2008年獲得常規賽MVP，2007年科比場均獲得31.6分5.4助攻5.7籃板數據，最終科比無緣常規賽MVP，常規賽MVP最終被小牛隊諾維茨基獲得。

1.德克-諾維茨基（小牛）

2006-2007賽季場均.24.6分8.9籃板3.4助攻

諾維茨基職業生涯全部在小牛隊度過。小牛隊經歷過2006年總決賽洗禮之後整體配合更加默契，球隊在常規賽打出67勝15負聯盟第一戰績，小牛隊領袖諾維茨基憑藉出色表現獲得常規賽MVP，這也是諾維茨基職業生涯首次獲得常規賽MVP。

2.史蒂夫-納什（太陽）

2006-2007賽季場均.18.6分3.5籃板11.6助攻

納什在2005年，2006年連續兩年獲得常規賽MVP，2007年常規賽納什表現依然出色，賽季場均獲得18.6分，投籃命中率達到53.2%，常規賽納什場均獲得11.6助攻聯盟第一，納什出色表現帶領太陽獲得61勝21負西部第二戰績。

3.科比-布萊恩特（湖人）

2006-2007賽季場均.31.6分5.4助攻5.7籃板

科比在2006年，2007年連續兩年獲得聯盟得分王，2007年28歲的科比無論是技術還是身體都達到職業生涯巔峰期，常規賽科比場均獲得31.6分聯盟第一，賽季科比帶領湖人獲得42勝40負戰績，科比出色表現帶領湖人進入季後賽，賽季結束科比進入最佳一陣最佳一防。

4.蒂姆-鄧肯（馬刺）

2006-2007賽季場均.20分10.6籃板3.4助攻

2007年30歲的鄧肯處於職業生涯巔峰期，常規賽鄧肯出場80場比賽，帶領馬刺隊獲得58勝24負西部第三戰績。鄧肯在賽季結束之後進入最佳一陣，最佳一防。並且帶領球隊獲得總冠軍。

5.勒布朗-詹姆斯（騎士）

2006-2007賽季場均.27.3分6.7籃板6助攻

2007年是詹姆斯職業生涯第四個賽季，詹姆斯在常規賽帶領騎士隊獲得東部第二戰績。季後賽詹姆斯帶領球隊在東部決賽淘汰活塞隊進入總決賽，東部決賽第五場詹姆斯得到48分9籃板7助攻數據，22歲的職業生涯首次進入總決賽。

6.特雷西-麥克格雷迪

2006-2007賽季場均.24.6分5.3籃板6.5助攻

2006-2007賽季27歲的麥迪依然保持巔峰期狀態，常規賽麥迪出場71場比賽，麥迪帶領球隊獲得52勝30負戰績，季後賽首輪被爵士淘汰出局，麥迪在季後賽場均獲得25.3分5.9籃板。
