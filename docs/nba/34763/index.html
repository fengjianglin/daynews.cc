都怪倫納德？白白送走實力新秀，如今場均11分6板，湖人隊上當了

2019-12-25 1,514條評論 4,473次閱讀 849人點贊

根據最新的媒體報道消息，在當初倫納德和湖人隊的加盟談判中，倫納德團隊可謂是獅子大開口，不光想要一份NBA頂薪球員合同，他們甚至開出許多難以想像的要價籌碼。而拒絕倫納德舅舅違規要求的湖人隊，最終也是在倫納德爭奪戰中徹底出局。現在重新回顧這次球員招募，湖人隊真的是虧大了，因為在休賽季裡面浪費時間等待倫納德，不僅讓湖人失去招募其他球星的機會，甚至導致自己培養的實力新秀球員被主動放棄。

在2019年的夏天，湖人隊管理層將奇才隊拉到濃眉哥的交易之中，而湖人隊將瓦格納、邦加、瓊斯等三名球員1個二輪簽，白白送給了奇才隊，再加上濃眉哥主動放棄的400萬保證金，湖人隊最終成功騰出了一份頂薪合同空間。當然，這一切就是為了幫助球隊簽下倫納德，但無奈湖人隊管理層的計劃並沒有實現，他們最後沒有得到倫納德，可是瓦格納、邦加等湖人球員卻是實實在在已經送到了奇才隊。

而在本賽季的NBA常規賽中，正式效力於奇才隊的瓦格納，他打出了場均11.6分6籃板1.3助攻的場均數據，常規賽投籃命中率58.1%，並且瓦格納還具備外線三分遠投能力，他擁有39.3%的賽季三分命中率，場均可以投中1個三分球。更重要的是瓦格納現在還是二年級新秀球員，他的合同薪資只有206萬美金，並且接下來的兩個賽季裡面，奇才隊依舊可以享受瓦格納的新秀球員合同紅利。

如果當初倫納德沒有拖著湖人隊，湖人隊完全不需要交易掉瓦格納等新秀球員，讓自己白白失去一名空間型內線。假如瓦格納依舊留在湖人隊中，那麼瓦格納和濃眉哥的內線組合，絕對能夠大幅提升湖人隊的內線實力和深度，而且也可以拉開湖人隊的球場空間。當然，湖人隊管理層當初也是自己過於心急了，在沒有得到倫納德準確加盟承諾的情況下，他們就早早送走一大票潛力新秀，最終導致湖人隊非常遺憾失去了場均11+6的瓦格納！
