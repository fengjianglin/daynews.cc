過氣全明星和准全明星帶隊，兩個超巨還在養傷，完整籃網得有多強

2019-12-23 91條評論 1,440次閱讀 454人點贊

北京時間12月22日，籃網主場122-112擊敗老鷹，目前他們以16勝13負的戰績排名東部第7，領先排名東部第8的魔術4個勝場。雖說現在就去算戰績有點早，但以籃網目前的勢頭來說他們想要打進季後賽機會很大。

本場比賽丁威迪砍下39分6個籃板6次助攻，小喬丹得到12分20個籃板6次助攻，這兩位可謂是籃網獲勝的首功之臣。其實在第三節比賽結束的時候，籃網還落後對手兩位數的分差，但末節他們在保證穩定進攻的同時還打出了頑強的防守，單節37-14凈勝了23分。

末節丁威迪7投6中得到16分3個籃板2次助攻，小喬丹搶下了11個籃板，兩人一里一外，carry全隊。誠然對手老鷹隊並不算是強隊，但完成這樣一場大逆轉還是足夠說明籃網的實力，小喬丹是過氣全明星中鋒，但他廉頗老矣，尚能飯否，丁威迪則是正在打出全明星級別的表現。

說丁威迪打出全明星級別表現的是科比，他今天來到籃網主場觀戰，賽前在和丁威迪交流的時候給了他這樣的評價。本賽季丁威迪在場均出場剛剛30分鐘的情況下得到22.7分2.9個籃板6.2次助攻，這數據卻是已經是全明星級別，值得一提的是，本賽季的前11場比賽，丁威迪打得還是替補，場均出場時間更少。

今夏籃網簽下了歐文和杜蘭特，不過這兩位超級巨星現在都還在養傷，沒有他們的籃網已經是季後賽級別球隊，一旦這兩位回歸，籃網到底會有多麼強，簡直不敢想像，總冠軍對於他們來說也並非奢望。
