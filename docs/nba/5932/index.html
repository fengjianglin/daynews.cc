休城三號位恐易主！底薪7號秀屢次成球隊奇兵，豪斯的壓力來了

2019-12-09 0條評論 15次閱讀 0人點贊

北京時間12月8日，NBA常規賽繼續進行著。在今天上午剛剛結束的一場比賽中，西部強敵菲尼克斯太陽隊前往休斯頓客場挑戰火箭隊。本賽季的太陽一掃過去魚腩球隊的陰霾，一躍成為西部的一匹強勁黑馬。本場比賽雙方打得不可開交，火箭在最後時刻差一點崩盤，好在威少站出來才才讓火箭隊最終以115:109有驚無險地拿下本場比賽，喜迎兩連勝。

本場比賽豪斯重回首發，首節開始火箭先獲得球權，當豪斯面對太陽兩名球員的防守下衝擊內線打進一球後，火箭隊這邊突然得分停滯下來，接連的失誤讓太陽打出一波5:0的小高潮反超比分。第2節回來，在威少的帶領下，火箭打出一波7:2的小高潮，將比分追至30:33。半場戰罷火箭帶著十分的領先優勢進入中場休息。

易邊再戰，火箭的進攻勢頭開始下降，而太陽隊這邊開始了追分狂潮。太陽打出一波13:2的小高潮追平比分。第4節決戰時刻，火箭的進攻突然像斷了電似的，太陽隊趁機打出7:2的小高潮，再次追平比分。兩隊進入了僵持階段，分數也是相互反超，最終憑藉威少在最後三分鐘的發力，火箭在主場以115:109拿下一城。

本場比賽，火箭隊之所以能夠有驚無險地拿下本場勝利，與哈登威少的穩定發揮有著直接關係，正是哈威組合末節聯手狂砍28分，才把太陽反撲的勢頭給徹底澆滅。全場比賽詹姆斯哈登雖然命中率比較低，27投8中，但是依然得到34分六籃板四助攻三搶斷。威少全場比賽18投10中，拿下24分14籃板，11次助攻的三雙數據。除了哈登威少的出色表現外，麥克勒摩的表現絕對稱得上是火箭的奇兵，本場比賽麥克勒莫15投10中，砍下27分！

要知道，這位13年的的7號秀近4場比賽已經投進22記三分球。隨著他的三分球命中率的不斷回暖，也讓自己更對得起小雷阿倫的稱號！

本賽季麥克勒摩雖然和火箭隊簽訂的只是一份部分保障合同，但是他正在用實際行動來為自己贏下一份大合同，截至到目前為止，麥克勒莫的進攻效率值達到118.8，排名聯盟第二，而它的凈效率只達到13.6，位列聯盟第十，同時是全隊最高！

這位昔日的7號秀拿著202萬美元的底薪卻打出了全隊效率值最高的表現，毫無疑問，火箭在夏季的這場豪賭賭贏了！

如今隨著麥克勒莫的狀態不斷回暖，三分球命中率也在不斷提升。對於火箭這支擅長三分投射的球隊來說，無疑是一個很良好的信號。不得不說，近兩場比賽麥克勒摩打出了出色的表現，轟下了三巨頭一般的數據，無論是以首發還是替補3號位出場都有著穩定的發揮，如此豪斯和戈登的壓力也相繼來了，畢竟戈登和豪斯都是球隊的小前鋒人選！那麼大家覺得，麥克勒莫能擠掉豪斯的首發位置上位嗎？
