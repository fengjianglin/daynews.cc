造濃眉哥犯規！搶斷詹姆斯！底薪老將決定生死，並羞辱了湖人雙核

2019-12-21 1,963條評論 3,434次閱讀 1678人點贊

文/籃郭先生

NBA的常規賽繼續進行，湖人大戰雄鹿，堪稱NBA這個武林的武林盟主之爭，因為，誰贏了，誰就獨佔聯盟第一的位置，誰輸了，暫時就位列聯盟第二的位置。任何一個聯盟，都有自己的盟主，過去這幾年，勇士扮演盟主的角色。但如今，應是群雄爭霸的時代。最終，這場比賽，雄鹿獲得了比賽的勝利。數據方面，湖人隊的戴維斯36分10個籃板5次助攻3次蓋帽，詹姆斯21分12個籃板11次助攻，丹尼格林21分，波普18分，朗多2分3次助攻5次失誤。雄鹿這邊，字母哥34分11個籃板7次助攻，米德爾頓15分6個籃板，洛佩斯10分4次搶斷，喬治希爾21分3個籃板。

這樣的對決，不敢說是絕對是巔峰對決，但也至少是季後賽級別的防守，季後賽級別的強度。對2支球隊來說，雖然沒有什麼宿仇，但聯盟之一的競爭壓力，還是擺在那的，源於此，作為球迷，自然期待與關注，那就是，大家都想看看，最終獲得比賽勝利的球隊，到底是哪一支？因為，這在某種程度上來說，也可能是總決賽的預演。雖然，總決賽是明年6月的事情，但作為球迷，還是希望提前能有一些判斷。初步來看，雄鹿底蘊豐厚，一星四射的打法在字母哥這裡，變得如魚得水、爐火純青了。至於效果如何，或許還有待季後賽的檢驗。

因為，常規賽的對決，或許沒有那麼大的價值。畢竟，有球隊曾獲得73勝，但也只是亞軍而已。所以，常規賽的成績，是常規賽的成績，季後賽笑到最後，那才是真正厲害的球隊。如今，一場常規賽而已，看起來沒有那麼重要，但季後賽也需要常規賽一場勝利一場勝利地堆砌。這樣的比賽，看起來，是詹姆斯、濃眉哥、字母哥等人的舞台。但其實，也有攪局者。甚至於，一人還扮演著超級奇兵的角色。此人，就是馬修斯，此前在獨行俠的時候，一度拿著高薪被罵得很慘。但如今的他，早已不是高薪球員，而是底薪球員。今年夏天，雄鹿隊給了他一份底薪合同。

如今看來，超值。這樣的比賽，馬修斯首發打了30分鐘，9中4，三分7中3，罰球4中2，砍下13分4個籃板2次助攻2次搶斷1次蓋帽的數據。作為一名首發球員，他的數據，自然是合格的。與此同時，他更加厲害的是最後時刻的搶戲。那時候，湖人追到7分，雄鹿好幾個回合無功而返，這時候，馬修斯殺入籃下，造成濃眉哥犯規，2罰1中，算是止住了頹勢。而接下來的這個回合，詹姆斯運球殺入籃下，被馬修斯眼疾手快搶斷下來，這樣的一個搶斷，也基本上意味著一點，湖人隊無緣比賽的勝利了。因為，時間所剩無幾了。但是，分差還是不小的。這一攻一守之間，我們看到了馬修斯關鍵時刻的穩健。看到這樣的馬修斯，你或許只能感嘆：真香。而現實，也的確如此。

造濃眉哥犯規！搶斷詹姆斯！底薪老將決定生死，並羞辱了湖人雙核。這般馬修斯，也證明了自己的價值。對他來說，這個賽季若能繼續保持這樣的發揮，明年夏天的時候，估計就不是底薪就能拿下的球員了。
