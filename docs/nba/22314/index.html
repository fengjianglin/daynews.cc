被傷病毀了的五大NBA天才，若非傷病他們必成超巨

2019-12-20 852條評論 2,292次閱讀 387人點贊

眾所周知，NBA從來不缺乏天賦異稟分球員，但是並不是每個天才都能成為超級巨星，很多天才就在他們到達巔峰前就飽受傷病的折磨，沒能到達巔峰成為超級巨星，今天我們就盤點一下被傷病摧毀的五大NBA球星。

安芬尼-哈達威

哈達威年輕時被稱為喬丹的接班人，二年級時他就趁著喬丹退役進了最佳陣容一陣，三年級時他再次入選一陣，並和奧尼爾聯手打進總決賽只可惜敗給了大夢率領的火箭，在喬丹復出後他也被認為是當時的第二分衛，但是傷病卻沒有使他成為下一個喬丹，甚至沒有到達自己真正的巔峰就提前隕落了。

特雷西-麥克格雷迪

麥蒂是中國球迷非常熟悉的球員，他的天賦自然不必多說，歷史上沒有幾人的天賦可以與他相提並論，但是傷病使他的成就大大折扣，因為先天性脊椎側凸在進入NBA前醫生就告訴他最多能在NBA打四年，可見他的傷病有多麼嚴重。在NBA麥蒂在飽受傷病折磨的情況下進過七次最佳陣容，得到過兩次得分王和一個進步最快球員，有過35秒13分的神跡，並於2017年進入名人堂，很難想像如果沒有先天性疾病麥蒂能取得多麼大的成就。

布蘭登-羅伊

羅伊曾被譽為是科比的接班人，他在生涯的前四個賽季就兩次入選最佳陣容三次入選全明星，還當選過最佳新秀，被稱為黃曼巴，但是由於嚴重的膝傷他不得不摘除雙膝的半月板，只打了六個賽季就匆匆退役。

德里克-羅斯

作為NBA歷史上最年輕的MVP羅斯也沒有抵住傷病的侵襲，巔峰時他變向不減速，扣籃不掛框，受傷前他拿過最佳新秀，MVP，進過最佳陣容一陣，全明星，第一年打季後賽就把當時強大的凱爾特人逼到搶七，MVP賽季東決上演三英戰呂布，和熱火三巨頭大戰五場。可惜因為傷病這一切突然戛然而止，雖然現在的羅斯似乎煥發了第二春，但是我們永遠無法看到玫瑰的巔峰了。

格雷格-奧登

奧登無法到達巔峰絕對是NBA歷史上的一大遺憾，在大學時他就被譽為是奧尼爾與鄧肯的結合體，在NCAA決賽上他單手打爆了諾阿和霍福德領銜的內線，在選秀大會上力壓杜蘭特成為狀元。但是在NBA傷病一直纏繞著他，職業生涯一共只打了105場常規賽，如果他不受傷或許真的會統治NBA吧。
