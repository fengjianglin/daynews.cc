NBA十個鮮為人知故事：字母哥曾當小販養家，喬丹地位在奧運體現

天天要聞
2019-12-20

NBA巨星雖然是運動員，但是在全世界人氣方面，不要說那些小鮮肉，即使是和一些影帝歌后相比，都絲毫不遜色，但是作為球迷來說，大家真正了解這些巨星球員嗎？下面，小編就給大家帶來關於NBA巨星的10個鮮為人知的小故事，知道7個是懂球帝。

1、92奧運會唯一全部首發的球員-邁克爾喬丹

喬丹在聯盟中的地位怎麼樣？下面這個小故事應該能讓大家了解一二：92年奧運會，為了一雪前恥的美國籃協，和想要NBA走出美國的聯盟高層一拍即合，派出了歷史上最豪華的巨星陣容，而作為球隊主教練的戴利而言，既是一種幸福，也是一種不幸，幸福是因為可以執教這麼多巨星，不幸的是這麼多巨星該如何分配上場時間和首發位置？而最終的的結果是，在八場比賽中，其他位置的球員輪流坐莊，而美國隊首發得分後衛則是鐵定的喬丹。

2、字母哥的悲慘童年

出身於籃球強國希臘的字母哥，為什麼在13年選秀大會中，是以一個身材瘦弱，技術粗糙的形象進入到大家視野的？要知道歐洲球員基本功方面都很紮實，希臘自身的青訓系統也很完善，其實原因很簡單，字母哥並不是希臘本土居民，他的父母是奈及利亞移民，非法偷渡到了希臘，之後才有了字母哥四兄弟，雖然曾經作為職業運動員的父母，遺傳給了他們兄弟四人出色的運動天賦，但卻沒辦法給他們一個保證溫飽的家庭，字母哥小時候，幾乎天天餓著肚子，還要擺攤賣小商品補貼家用，直到15歲，才開始正式接觸籃球這項運動，這也是為什麼進入NBA時的字母哥，不僅瘦弱，而且技術粗糙。

3、伊巴卡的決定

雷霆最對不起的球員是誰？不是威少、不是杜蘭特，更不是哈登，而是為雷霆隊賭上了自己職業生涯的伊巴卡，在2014年季後賽開賽前，因為膝蓋傷勢，伊巴卡已經被球隊宣布了賽季報銷，沒有了伊巴卡的防守，雷霆隊的內線被馬刺隊爆的渣都不剩，不管杜蘭特和威少如何努力，也沒有改變連輸兩場的比賽結果，眼看球隊一個賽季的努力就要付之東流，已經報銷的伊巴卡，選擇了打封閉回歸球場，幫助球隊，雖然依靠伊巴卡的出色發揮，幫助雷霆隊贏得了一場寶貴的勝利，但是這次決定，也讓伊巴卡復出了極大的代價，讓他在黃金年齡，就失去了當年的頂級彈跳力。

4、創造奇蹟的米勒

說起NBA歷史上最頂級的「殺手」，估計很多老球迷腦海中出現的第一個球員，就是雷吉米勒，這位90年代最為大的殺手，在同年時期卻是一個雙腿殘疾的的孩子，雙腿扭曲，臀部畸形，腳踝內拐，甚至被醫生判斷為能夠走路就是奇蹟，但是米勒的家人沒有放棄小米勒，而米勒自己也沒有放棄自己，經過多年穿戴支撐物的鍛煉後，米勒終於可以向正常孩子一樣奔跑，而這時米勒的姐姐，將弟弟引領到了另一條道路上，那就是籃球，從擊敗自己的姐姐開始，到來去挑戰高高在上的籃球之神，不放棄，不認輸的米勒，創造了一個又一個的奇蹟。

5、拉塞爾

換一個輕鬆的，說個拉塞爾老爺子的小故事，當時還是球員的拉塞爾，在打完客場比賽之後，馬上就前往了下一個比賽場所，而經過長時間的飛行，這些球員早已飢腸轆轆，所以在到底目的地之後，拉塞爾就和同房間的薩姆瓊斯，點了烤薄餅這種食物當作宵夜，只不過球隊當時的主教練奧爾巴赫卻認為，烤薄餅是一種垃圾食物，會影響球員比賽的狀態，規定了球員在比賽日禁止食用這種食物，結果餅剛送到房間，奧爾巴赫就進來了，拉塞爾剛剛就咬了一口，結果被奧爾巴赫給抓了個正著，並要對拉塞爾兩人進行罰款，這時的兩大巨星反駁道，他們吃餅的時間並不是比賽日，可是他們倆怎麼斗得過老狐狸奧爾巴赫，老狐狸一臉嚴肅的告訴他們，現在已經是凌晨一點，是第二天了，拉塞爾和瓊斯無言以對，只好乖乖的交了罰款。

6、格蘭特希爾的付出

為了籃球，球員可以付出什麼？這個問題格蘭特希爾可以給大家一個意料之外的答案，因為希爾為了他的籃球生涯，險些付出的生命，眾所周知，加盟魔術之後的希爾從之前的人氣王，淪為了聯盟的玻璃人，但是希爾始終沒有放棄自己的職業生涯，所以希爾一次次的在傷愈，復出，受傷中輪迴，但是在2003年手術後，希爾遭遇了可能致命的細菌感染，差點讓他性命不保，之後進行了半年的抗生素注射，才讓他徹底脫離了危險，但這也沒有讓希爾選擇放棄。

7、麥基牌廁紙

再來個輕鬆的，這個小故事是關於麥基的，只不過麥基成為了惡搞的對象，當時效力於勇士隊的麥基，被格林將頭像印在了廁所的衛生紙上，並且在社交媒體上對此進行調侃，用以報復之前麥基將他的頭像印在被子上的行為，而球隊老大哥伊戈達拉也加入了麥基的調侃中，不過上廁所時被麥基盯著的感覺真的好嗎？但這的確是一個惡搞的好辦法，畢竟沒有什麼比讓自己討厭的人吃便便更爽的行為了。

8、0號特工阿里納斯

「Hi，我是吉爾伯特·阿里納斯，這是我的故事。職業生涯的前40場，我是在板凳上度過的。他們認為我打不上比賽，我想，他們根本沒看到我的天賦。他們覺得我就是個0，一無是處。但是我並沒有坐在那裡怨天尤人，而是不斷的訓練，訓練。在沒有人相信你的時候，你的任何努力都會為自己加分。這已經不是我能否打好籃球的問題了，而是我要證明他們是錯誤的。 現在我仍然穿著0號球衣，因為我要告誡自己每天都要努力」很多老球迷應該都熟悉這段廣告詞，也熟悉這個廣告，雖然這是阿迪達斯宣傳所用，但也代表了阿里納斯進入聯盟之初的故事。

9、步行者三分王是教練

如果一支球隊中，擁有這雷吉米勒和穆林兩大投手，那誰是最終的球隊三分王？而且巧了，這兩人的確同時在一支球隊效力過，而且經常會在訓練中或者結束後進行一些帶有賭注的三分比賽，不過比賽的最終勝利者，卻不是這兩位歷史頂級的投手，而是他們當時的主教練拉里伯德，他們只能在比賽結束後，眼睜睜的看著這個球員時期囂張無比的白人，在退役之後依然完虐自己，之後還會拿走代表勝利者的鈔票。

10、托馬斯和魔術師約翰遜的故事

在80年代的NBA中，所有人都知道魔術師約翰遜和微笑刺客托馬斯是一對好友，而在魔術師被查出艾滋病之後，托馬斯也和約翰遜在球場上進行了貼面親吻的禮節，這讓大家見證到了雙方的友誼，但是在92年奧運會的大名單中，托馬斯卻被排除在了最終名單之外，而作為好友的約翰遜，不僅沒有提出異議，反而是幕後推手中的一員，直到後來通過媒體的爆料，大家知道了當時在媒體面前爆料魔術師是同性戀，他的HIV病毒是一名男人傳播的人居然就是托馬斯，這也是為什麼魔術師會給出這樣的反擊。
