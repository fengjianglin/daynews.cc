聚焦｜本賽季的猛龍是為何能在缺少兩大鋒線的情況下一路高歌？

天天要聞
2019-12-01

六月份，多倫多猛龍收穫了他們隊史第一座總冠軍，舉國歡慶。不過休賽期科懷-倫納德和丹尼-格林兩大鋒線的出走卻給球隊澆上一盆冷水，這兩人對於球隊的重要性不言而喻，具體表現為：倫納德在上賽季季後賽中交出了多項數據（包括得分、TS%、USG%、WS、BPM等）皆全隊第一的答卷，格林在季後賽中交出了場均三分命中數球隊第三、DBPM球隊第二的答卷。

此番改變，使大家在本賽季初對這支猛龍並不十分看好，認為其充其量就是一季後賽球隊，在聯盟激不起什麼風浪。不過賽季至今，猛龍所展現出來的實力令人訝異：

  • 球隊戰績14勝4負，東部第二，聯盟第四；在洛瑞和伊巴卡受傷期間，球隊打出了8勝2負的絕佳表現，其中不乏湖人、76人等頂級強隊；目前球隊喜提六連勝；
  • 球隊進攻效率聯盟第7，防守效率聯盟第1，凈效率聯盟第3；
  • 球隊搖身一變三分大隊，場均三分命中數聯盟第4，三分命中率聯盟第1；
  • 球隊快攻得分聯盟第1，利用失誤得分聯盟第2；
  • 球隊有效命中率聯盟第6，真實命中率聯盟第5。

目前來看，猛龍所展現出來的實力絕非是一支季後賽球隊那麼簡單，他們所展示出來的決心和贏球的慾望指引著他們向更遠大的目標邁進。

球隊取得如此成就，蘊含如此大的能量，原因為何？

其一，尼克-納斯教練的用人和戰術。主教練是一支球隊的燈塔，他的戰術理念指引著球隊前進的方向，球隊獲得如此成就，納斯教練功不可沒。作為冠軍教頭，納斯教練在用人和防守戰術的布置上很有一套：

  • 納斯教練的用人哲學。從以往的比賽過程來看，納斯教練所展示出來的用人哲學是「用人不疑，疑人不用」。在上賽季季後賽中，納斯教練的「三後衛陣容」被人一度質疑，可他一直不為所動，結果以「真香」告終。而本賽季，納斯教練直接將球隊放權給剛剛四年級的西亞卡姆，其20.4次的出手次數和29.5%的使用率冠絕全隊，「放開掄」也成為西亞卡姆本賽季的標配。在球隊兩員大將凱爾-洛瑞和賽爾吉-伊巴卡受傷缺陣後，納斯教練果斷將兩人時間分配至霍利斯-傑弗森、特倫斯-戴維斯和克里斯-布歇三位火槍手，這三個板凳匪徒也不負厚望，撐起了猛龍的替補席。

  • 納斯教練的防守戰術。范弗利特在球隊對陣湖人和快船時表示要制裁曾經的隊友格林和倫納德，結果令前者全場5投0中，令後者全場12投2中。為什麼會出現這種情況呢？這就不得不提納斯教練的防守戰術－局部夾擊、全員協防和快速退防。事實上，全聯盟也就只有猛龍有這樣的防守條件，因為球隊中絕大多數球員都有很好的機動性和運動能力，例如西亞卡姆、阿奴諾比、鮑威爾、布歇等，他們能夠支撐得起這套防守體系。在面對對方明星球員時，他們可以局部夾擊賭對手三分水準；能夠快速協防不給對手創造投籃舒適區；在對手快攻時他們也能做到快速落位回防。如此，造就了他們聯盟第一的防守效率。

局部西亞卡姆、傑弗森和布歇形成夾擊

其二，歸功於「征南大將軍」西亞卡姆。

洛瑞在採訪時曾說道：「帕斯卡爾，他今年一定會入選全明星的，他在持續成長變得越來越好，他會進化成超級巨星的。」的確，半路出道的西亞卡姆正在用天賦征服著這個世界上的球迷們，上賽季剛剛奪得進步最快球員的他在本賽季的進化下莫非能夠衛冕？西亞卡姆本賽季持續進化的點有哪些呢？

  • 增加使用率和出手次數，承擔球隊責任。「放開掄」是一種能力，並不僅僅只是「投就完了」那麼簡單，當一名球員長時間的保持高使用率和高出手次數，勢必會遭到對手的「特殊照顧」，所以對於聯盟四號位中出手次數排名第2、使用率排名第4的西亞卡姆來講，「放開掄」並不是那麼容易得，其結果就是本賽季至今只打出了55.7%的真實命中率，位於聯盟中游水平。不過賽季還長，相信西亞卡姆能夠調整過來，為球隊承擔起更多責任。
  • 改善進攻方式。本賽季如此高的使用率不是之前賽季轉換進攻、定點投籃等能夠彌補的，所以西亞卡姆本賽季在擋拆持球、面框單打和背打等個人進攻上有所提升：

可以發現，西亞卡姆降低了轉換進攻和定點兩項進攻方式的佔比，本賽季此兩項較於2018-19賽季分別下降7.5%和7.2%，而在擋拆持球、面框單打和背打上分別增加了7%、9.1%和2.6%。本賽季的比賽中，能看到很多西亞卡姆與小加索爾擋拆後持球單打或者面對對面內線時面框單打的鏡頭，當然也有錯位背打對手小個的畫面，說明西亞卡姆本賽季開始注重提升個人自主進攻的佔比。

  • 提高三分出手比例。本賽季西亞卡姆在三分線外的出手佔比由上賽季的22.6%提升至31.1%，場均命中三分數量由1.0個提升至2.4個，在聯盟所有四號位中排名第4位，已經進化為一名高產的空間型四號位。並且在進攻方式上也有所改變，之前賽季的三分多來自受助攻的定點或底角三分，而本賽季的賽場上出現過西亞卡姆面框三分的情景，誰能想到如今的西亞卡姆竟然能做出「在防守隊員面前運球後投三分」的動作。
  • 領袖氣質。本賽季的西亞卡姆在球場上所具備的領袖氣質是之前所不具備的，特別是在洛瑞和伊巴卡受傷期間。在對陣湖人的最後時刻，西亞卡姆連續兩記轉換進攻讓比賽失去懸念；在對陣76人最後一分鐘時球隊落後2分，西亞卡姆拉開一對一殺往籃下，強打恩比德命中還有加罰，完成反超，之後再次破壞西蒙斯的吊球並帶走比賽。何為領袖？在比賽最後時刻能掌控比賽的即為領袖。

其三，OG-阿奴諾比狀態回暖。

阿奴諾比一直是猛龍全力培養的球員，上賽季因為倫納德的到來及傷病原因改打替補，之後狀態有所回落；並且在上賽季末由於傷病原因未能出席任何一場季後賽。本賽季阿奴諾比慢慢找回狀態，賽季至今場均能拿到11.3分5.6籃板1.6助攻1.2搶斷的全面數據，三分命中率高達47.7%，真實命中率高達63.2%（聯盟所有三號位中排名第五）。

阿奴諾比也是一個出色的防守者，2.20米的臂展及出色的運動能力使得他在防守端遊刃有餘，單防、協防和退防無所不能，2.8的DBPM也能排名球隊第二。

其四，替補球員們出色的狀態。

  • 鮑威爾。自凱爾-洛瑞受傷之後，鮑威爾開始接替他登上首發位置。在首發的10場比賽中，鮑威爾場均可以砍下15.9分，投籃命中率48.3%，三分命中率41.5%，表現出色；在同魔術的比賽中，鮑威爾18投12中高效砍下全場最高的33分，帶領球隊獲得最終勝利。
  • 霍利斯-傑弗森和克里斯-布歇。在伊巴卡受傷缺陣之後，這兩位挺身而出，彌補伊巴卡不在場時的空缺，從最近十場比賽來看，兩人的工作做得超出預期。近十場比賽，傑弗森場均可以拿到10.5分6.9籃板，其中包含3.2個前場籃板，為球隊的二次進攻提供機會；布歇場均可以拿到9.1分6.6籃板（3.2個前場籃板）0.9蓋帽，防守端表現出色。
  • 特倫斯-戴維斯。在鮑威爾頂上首發之後，這位22歲的年輕後衛獲得表現機會，並且其表現也是令人眼前一亮。在最近替補出場的十場比賽中，場均可以拿到10.6分4籃板3助攻的全面表現，三分命中率高達45.2%，為球隊銜接段的進攻提供源源不斷的火力支持。

板凳匪徒們出色的表現，也是近期猛龍戰績一路高歌的重要原因。不過等洛瑞和伊巴卡傷愈歸來，如何使用這幾名出色的替補，也許會成為納斯教練甜蜜的煩惱。

賽季至今，這支猛龍所發揮出的能量令人刮目相看，他們積極、有活力且無所畏懼，他們勇於擔責且從不退縮，他們身臨險境卻永不言棄。也許，這就是冠軍底蘊吧！
