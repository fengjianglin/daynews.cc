曾經NBA最年輕MVP到如今的角色球員，羅斯到底經歷了什麼？

天天要聞
2019-12-23

羅斯，曾經NBA歷史上最年輕的MVP球員，曾經球隊的絕對核心。又有誰能想到傷病會這麼快帶走了他最擅長的進攻方式，從球隊核心到四處流浪，從巔峰跌落谷底，經歷這一切又有多少人能夠重新站起來繼續前行，但是羅斯一直在堅持著曾經的夢想，不放棄、不氣餒、不抱怨，直到今天似乎找到了曾經熟悉的味道，雖然玫瑰不能再次絢麗綻放，但是卻留給人間宜人的芬芳。

羅斯

1、流浪並不代表放棄，流浪一樣可以選擇堅持！

從芝加哥公牛到尼克斯、騎士、森林狼，到如今的活塞，誰也無法體會羅斯的心情，曾經媒體也報道過，羅斯多次萌生過退役，但是最終還是因為籃球夢選擇了堅持。就在上個賽季在森林狼對戰爵士隊的比賽上，羅斯得到了50分，主場球迷一起高喊：MVP！MVP！MVP！此時的情景既熟悉而又陌生，比賽結束的一刻羅斯哭的像個孩子，而此刻所有累積的壓抑情緒得到了釋放，不知道坐在電視機前的你或者看到這段視頻的你是否跟我一樣流出了眼淚。而這一切告訴了我們，流浪並不代表放棄，流浪一樣可以選擇堅持！

羅斯怒砍50分之夜

2、傷病可以帶走我的輝煌，但是不能帶走我的熱愛！

自2011年羅斯獲得常規賽最有價值球員之後，傷病就已悄悄地降臨。2012年的四月份，羅斯不幸遭受到了膝蓋十字韌帶撕裂，公牛宣布羅斯賽季報銷。本以為經過一個夏天的恢復，玫瑰還能夠再次綻放，沒想到在2013年10月份復出沒打幾場就遭受了半月板撕裂，又在之後的比賽中半月板頻頻的出問題，傷病就這樣一點一點侵蝕著羅斯的身體，拿走了羅斯引以為豪的技能，最終公牛放棄了曾經的核心，選擇了重建。從巔峰跌落谷底，羅斯經歷了前所未有的痛苦，是該放棄還是堅持？羅斯選擇了堅持，堅持自己的熱愛，堅持自己的初心。而這一切告訴我們，傷病可以帶走一個人的輝煌，但是不能帶走一個人的熱愛！
