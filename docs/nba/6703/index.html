猶他爵士：深度分析陣容磨合與體系球風，能否充當西部攪局者？

2019-11-26 0條評論 15次閱讀 0人點贊

作為NBA從未奪過總冠軍的11支球隊之一，猶他爵士難免讓人感嘆，畢竟在90年代末連續輸給芝加哥公牛，也成為了「黑白雙煞」約翰·斯托克頓與卡爾·馬龍的遺憾，而球隊上一次打進西決要追溯到06-07賽季，12年前的季後賽，讓無數中國球迷見識到了德隆·威廉姆斯與卡洛斯·布澤爾的厲害。

老一代爵士

可是作為1974年成立並進入NBA的球隊，爵士卻從未捧起過奧布萊恩杯，從1983年到2002年，他們連續19個賽季常規賽保持50%的勝率並打入季後賽，所以在球迷印象中一直保持著西部勁旅的角色，但令爵士心塞的是季後賽的征途中，他們一直難以走得更遠，也從未摸到過總決賽的地板，因此談到西部攪局者，猶他爵士就會浮現在球迷腦海中，畢竟硬實力上較為穩定，只不過困擾於小球市的限制，他們難以突破。

23歲的多諾萬·米切爾是如今爵士的當家球星，進入聯盟第一個賽季就能砍下場均20+得分，證明了球隊管理層在首輪第13順位的眼光，他少年老成的球風打法，也被鹽湖城的球迷喻為「小韋德」，但畢竟進入NBA才第三個賽季，雖然擁有成為巨星胚子的潛力，但也得經過比賽的磨鍊。

猶他爵士在確定以多諾萬·米切爾為建隊基石後，在19年夏天放棄了西班牙後衛里基·盧比奧，招募來博楊·博格丹諾維奇與傑夫·格林，彌補了原先薄弱的鋒線實力，算上交易得到的前灰熊核心控衛邁克·康利，爵士在引援上做足了功課，也給了主教練奎因·斯奈德更多機會，來打造更為強勢的陣容，畢竟想從西部脫穎而出，必須先擠進季後賽行列，猶他爵士並非洛杉磯湖人，能夠吸引來勒布朗·詹姆斯與安東尼·戴維斯等成熟的超巨球員，他們只能依靠球隊現有的陣容來磨合，打造出能夠獲勝的體系球風。

爵士球員

缺少得分爆發力，板凳陣容的磨合有待提高

眾所周知，爵士的進攻向來缺乏天賦，多諾萬·米切爾的存在價值就是彌補球隊巨星球上的空缺，但從球隊整體的得分能力來看，爵士並沒有所謂的得分爆發力，場均得分107.1僅排在聯盟第22位，處於NBA的下游水平，一方面是球隊體系的影響，另一方面則是球員的得分能力。

由於博楊·博格丹諾維奇的加盟，主教練斯奈德將喬·英格爾斯安排到了替補席，頂替他的則是羅伊斯·奧尼爾，而本賽季兩人的場均得分並沒有上雙，米切爾場均25.9分是隊內的最強得分手，緊隨其後的是博格丹諾維奇的20.7分，而邁克·康利與魯迪·戈貝爾的內外組合，場均得分加起來還沒達到30分，這就是爵士隊的現狀，康利與米切爾容易遭到對手的嚴防死守，而鋒線上的得分點均已投籃為主，內線的戈貝爾習慣於吃餅或二次進攻的得分，爵士缺少籃下的得分成功率，更缺少球隊的得分爆發力。

爵士球隊數據

在猶他爵士的第二陣容里，伊曼紐爾·穆迪埃、喬·英格爾斯與傑夫·格林是絕對的核心球員，但球隊場均26.44的替補得分位列聯盟倒數第2位，只高於休斯頓火箭隊，但要知道後者擁有場均得分接近40分的詹姆斯·哈登，而爵士並沒有能夠與之媲美的球員，所以猶他的板凳深度成為了最大隱患，沒有替補球員的進攻火力支撐，季後賽當然難以前進。

從現有的賬面陣容來看，喬·英格爾斯有能力去競爭最佳第六人的頭銜，畢竟上賽季首發出場了82場比賽，能夠砍下場均12.1分5.7助攻4籃板1.2搶斷的全能數據，更有39%的三分球命中率，或許是角色轉換後的不適應，讓他在輪換階段極的表現極為掙扎，這是主帥奎因·斯奈德必須去解決的問題，光擁有實力強勁的板凳陣容，也得去好好磨合，穩定住比賽局勢，否則在西部季後賽里靠僅有的首發五人打比賽，這絕對走不遠。

康利

球風硬朗紀律性強，立足於陣地戰體系

猶他爵士一直被稱作「鐵軍」，因為他們所呈現出的球風很硬朗，更是保持了超強的球隊紀律性，畢竟常年沒有超巨球員的存在，他們只能依靠團隊籃球來生存，即便眼下擁有米切爾，爵士依舊強調球隊的整體性，貫徹了硬朗的球風以及超強的紀律性，這也是他們生存在西部賽區的資本，否則早已淪為魚腩球隊。

爵士以場均102.4的失分排在聯盟第2位，他們依靠防守的整體壓迫性，避免了許多無謂的丟分，外線的康利雖然沒有身高與對抗上的優勢，但他101.2的防守效率也排在了聯盟第55位，更不用說內線大閘戈貝爾的數據，99.6的防守效率位居聯盟第33位，29.2%的後場籃板率與19.6%的籃板率均排在聯盟前5位，用個人的防守能力帶動全隊，這就是猶他爵士一再強調的紀律性，所以他們能夠打出硬朗的球風，這也是功勛教頭傑里·斯隆為爵士留下的一筆巨大財富。

戈貝爾進階數據

除了球員自身的硬實力外，NBA聯盟最注重的就是球隊氛圍，無論是球場上的領袖，或是教練席上的指揮，就連更衣室領袖都會是影響比賽的因素，爵士受困於鹽湖城的小球市場，他們不像紐約和洛杉磯，沒有太多的流量曝光度去吸引球星，僅有自身的選秀與培養。所以從老闆到球隊後勤人員，都明白球隊紀律性的重要。因此爵士的硬朗球風，就成為了他們的標籤，既然沒有進攻上的天賦，那就從防守做起，想贏球就得避免無謂的失誤與丟分，就像缺少得分爆發力的屬性一樣，只能基於用防守來鞏固勝利。

硬朗的球風，超強的球隊紀律性，高效的投籃與防守效率，這都是爵士在球場上所呈現出的比賽方式，可從客觀的數據上來看，本賽季爵士場均100.84次回合數排在聯盟第20位，但懂球的球迷們都明白，爵士的一切球隊屬性特性，都給予在他們陣地戰的體系中，這樣在季後賽的比賽中，才有機會與對手糾纏。

戈貝爾防守

攻防兩端的投入，季後賽存在隱患

爵士是本賽季NBA最會投三分的球隊之一，場均37.7%的三分命中率排在聯盟第7位，323個罰球命中排在聯盟第4位，能夠看出他們穩定高效的投籃能力，畢竟在現有陣容的球員里，僅有多諾萬·米切爾擁有超強的自主得分創造力，所以身邊隊友的投籃能力才顯得更為關鍵。

除了投籃之外，爵士隊的防守能力依舊是聯盟頂級水平，場均101.4的防守效率排在聯盟榜首，這不僅僅是場均丟分，還包括了搶斷、蓋帽或是籃板球的全面表現，主教練奎因·斯奈德給球隊打造的防守戰略體系，讓他們在主場戰勝了擁有「字母哥」阿德托昆博的雄鹿，要知道本賽季的雄鹿場均得分高達119.8分，恰好是位居球隊場均得分榜的第一，這一組矛與盾的對決，爵士是笑到最後的贏家。

戈貝爾是聯盟最出色的內線防守者，對於爵士的價值不低於米切爾，靠著戈貝爾的一己之力，爵士的場均防守籃板高達37.65個，排在聯盟第2的位置，75.7%的防守籃板率排在聯盟第5位，也從側面反應出球隊在防守端的專註，用戈貝爾作為防守的內線核心，帶動球隊保護籃板球的整體性。不過細心的球迷一定發現，在爵士的進攻籃板數據上，卻僅有23.6%的進攻籃板率與場均7.71個進攻籃板，全都排在聯盟倒數的位置。

爵士在攻防兩端的比例投入，是季後賽的隱患質疑，要明白進攻才是最好的防守，雖然能夠抓下很多防守籃板，但如何轉化為得分才是關鍵，所以從攻防兩端的籃板數據看來，這其實是爵士比賽的一個縮影，因為他們將太多精力花在了防守端，自然而然會忽略對進攻籃板的拼搶，而要做到快速退防，就得穩定投籃命中率，放棄對進攻籃板的爭搶機會，所以從本質上反應了球隊在防守上的體力消耗，勢必會影響球隊在季後賽進攻的正常發揮，這一點上的吃虧，也是他們難以提升球隊戰績的關鍵。

猶他爵士 米切爾

結語

猶他爵士的現狀，充其量扮演西部攪局者的角色，米切爾決定了球隊的實力上限，但終究還未呈現出超巨的比賽方式，這就是小球市的難題，招募不了大牌球星，還得擔心自家培養的球員被挖走，所以以球隊整體性為主，打造出可以延續的體系球風就顯得格外重要。

兩次輸給邁克爾·喬丹不是意外，而是超級巨星的能力體現，在當下的小球潮流里，爵士利用「法國吳彥祖」戈貝爾的巨大優勢，支撐起球隊攻防兩端的內線實力，讓上賽季表現出色的喬·英格爾斯來帶領輪換陣容；米切爾依舊是爵士最具天賦的球員，畢竟新加盟的邁克·康利並非全明星球員，無論爵士的陣容如何調整，他們依舊保持著一貫的體系球風，這也是在西部的激烈競爭中，猶他僅有的優勢，做好西部攪局者的工作，他們或許有機會第三次邁進西決的舞台。
