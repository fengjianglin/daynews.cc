防守勝利貢獻聯盟第一！防守最好中鋒？球迷：忘記小加索爾了吧？

2019-12-19 589條評論 2,179次閱讀 812人點贊

早些時候，NBA這邊進行的9場常規賽比賽當中，已經有著幾場比賽是已經比完了的，不過當天這麼多場比賽當中，沒有一場是可以威脅到湖人和雄鹿他們第一的位置。

現在湖人和雄鹿都拿到了24個勝場了，不過湖人下一場比賽是在客場對陣雄鹿，也就是說湖人和雄鹿總有一支球隊拿到他們賽季第一個2連敗。

這也是雄鹿和湖人這個賽季的第一場對戰的比賽，說是聯盟第一之爭也不為過，誰贏了就能獨佔聯盟第一的位置！

而當天東部這邊的黑馬球隊熱火也是有著一場比賽的，他們是在客場對陣76人，熱火這邊因為前一個比賽日輸給了灰熊之後直接從東部第2的位置掉到了第4。

這一次和76人的比賽結束之後，熱火以4分的差距險勝了76人，再一次的回到了東部第2的位置。

而76人這邊呢輸了這一場比賽之後，本賽季聯盟所有球隊當中再無主場不敗球隊！不過76人能撐到最後確實非常厲害了。

而這一場比賽之前，76人這邊的主帥布朗說了一段讓人感到意外的話，布朗說現在恩比德是聯盟防守最好的球員！這個確實吹得有些過分了。

可能布朗是看到現在恩比德的防守勝利貢獻聯盟第一才這樣說的吧，不過聯盟這麼多的中鋒裡面，恩比德真的算是防守最好的中鋒？不見得吧。

約基奇，唐斯他們呢？就算在比賽之前吹了一波恩比德可比賽還是輸了，更何況恩比德還有一位苦主呢！

看到這一則消息之後，許多球隊也紛紛表示，布朗啊，你忘記小加索爾了吧？布朗這一次確實好像忘記了什麼，小加索爾把恩比德直接防得一分未得還記得嗎？

布朗還真是敢說，雖然防守勝利貢獻這個數據是恩比德第一，可聯盟防守最好的中鋒，真不能從這個數據上就認定是他，還有很多因素有變動的。

就比如面對的對手是誰實力怎麼樣，而且小加索爾把恩比德防得死死的又怎麼說？布朗，你還是等恩比德拿到DPOY再來說他是防守最好的中鋒吧。

而76人這邊輸掉這一場比賽之後，並不是從第2掉到第3位，而是直接掉到了東部第5，這裡也可以看出來現在東部的競爭確實比西部大一點。

雖然整體實力比不上西部那邊，可各支球隊都在很想拿下每一場比賽的勝利。

而且從76人現在的陣容來看，他們還是很有希望去爭奪一下總冠軍的位置，或許希望不大，但夢想還是要有的，萬一實現了呢？

本賽季的常規賽也才進行到1/4多一點，還有著很多場的比賽未進行，最終的結果到底會怎麼樣誰也不知道。

而且現在也有很多球隊打算進行交易了，到時候有可能很多球隊的陣容會產生變動，實力也只會變得更強。

畢竟不管是哪支球隊，都不可能交易一個隊球隊一點作用的球員回來吧，除非是真的想去拿狀元簽，不然應該都會借著這交易的機會拼一拼。
