NBA或迎重磅改革！肖華想法遭眾嘲，球迷：還不如搞個全明星單挑

天天要聞
2019-12-22

北京時間12月21日，本賽季的NBA遭遇了收視率寒冬，無論是在美國本土還是在其他國家，NBA的收視率都在下降，因此有消息稱，NBA聯盟正在想辦法拯救這一情況，其中一個辦法就是做出改成改革，讓比賽更具有看點。

而根據記者Shams Charania的消息報道，NBA已經向各球隊發送2021-22賽季變更建議，建議內容包括：

1、賽季常規賽變為78場比賽

2、賽季內設錦標賽。

3、季後賽設置7-8號種子的比賽（即為分區7-8名與9-10名之間將進行爭奪季後賽7-8順位的比賽）。

4、季後賽四強將根據常規賽的最佳戰績進行排名。

這其中減少比賽場次幾乎是所有球隊的共同願望，因此早在幾年前，就有不少球員和教練抱怨一個賽季82場的常規賽實在太多了，密集的賽季帶來的是傷病風險的增加，因此有球員曾表示，60場左右的常規賽更加合理，雖然這次NBA只是建議將常規賽縮減到78場，但少打四場也能獲得將近一周的休息時間，相信會受到球員和球隊的歡迎，球隊也不用費盡心思搞輪休了。

此外更改排名機制和季後賽球隊機制還算不錯，但具體如何還有待考驗。

但是另一個設置錦標賽的建議可能就非常雞肋了，錦標賽賽制更改建議為：在11月下旬至12月中旬，所有30支球隊都參與其中，小組賽分區比賽（4個主場賽區，4個客場賽區），共產生6個分區冠軍以及2張外卡名額，四分之一決賽將於主場賽區進行，半決賽以及決賽將於中立球場進行。獎金池為每位冠軍球員1百萬美元，教練為150萬美元。

毫不客氣地說，這會是一個非常雞肋且受到排斥的賽制，因為這其中有幾個問題：這個錦標賽冠軍有什麼意義？相當於當賽季的另一個總冠軍嗎？這個錦標賽冠軍和總決賽冠軍是一個性質嗎？

如果這個錦標賽冠軍和總決賽冠軍是一個性質，那就等於有球隊可能一個賽季得到兩個總冠軍，這樣將來的球員的總冠軍積累數就會更快，那麼歷史地位該怎麼評判呢？以前球員用3個賽季才能拿到3連冠，將來的球員用一個半賽季就可以嗎？

如果這個錦標賽冠軍和總冠軍不是一個性質，只是獎金獎勵，那麼對於那些志在奪冠的球隊來說，可能會出工不出力，剩下的就是菜雞互啄，說白了，這就是個安慰獎。這種獎項幾乎不會得到只認總冠軍的球迷和球員的的認可。

其實你也可以將這個錦標賽看成是另一個性質的全明星，但它的效果可能還不如全明星，儘管這些年全明星也越發單調，但畢竟噱頭還是不錯的，因此有不少球迷表示：肖華與其費盡心思弄錦標賽，不如想辦法開發全明星新玩法，比如全明星鬥牛和半場3V3無疑是不錯的想法。

而這次改革內容雖然還只是建議，但可以看出肖華的決心，不過消息一出，球迷們對於肖華和NBA的這個改革想法幾乎都是清一色嘲諷和不屑，認為比起大衛·斯特恩，肖華的想法都太幼稚和想當然了，看看以往他對全明星賽制的改革和在賽季結束後頒發MVP獎項所引起的尷尬，幾乎可以肯定，這又將是一次失敗的想法。
