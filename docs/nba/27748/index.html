“多餘的卡椒”：快船需明確的是該不該把“卡椒”從路威身邊拿走

2019-12-20 2,340條評論 4,396次閱讀 2861人點贊

洛杉磯快船或許是本賽季湖人之後最受球迷矚目的球隊，依舊保留了上賽季季後賽實力的班底，招攬到上賽季風頭最勁的總決賽MVP倫納德，交易來常規賽雙一陣的保羅-喬治，還簽下了諸如哈克利斯、帕特森、麥格魯德等新鮮血液，鋒線上人才濟濟的他們紙面實力堪稱聯盟頂級，甚至許多美國專家、媒體都預測他們會是新賽季戰績最好的球隊。不過賽程過了四分之一後我們卻發現雖然快船的戰績不差，依舊算是頂級豪強，但離許多人的預期還是相去甚遠…

  • 11月30日97：107大敗於馬刺倫納德得到19分，喬治得到5分，路威得到7分；
  • 12月7日91:119慘敗於雄鹿倫納德得到17分，喬治得到13分，路威得到11分；
  • 12月4日雖然贏下開拓者，但是倫納德僅僅得到11分，路威只得到9分；
  • 12月10日雖然贏下步行者可是路威只有19投2中得到6分（倫納德缺席）。

自從11月27日快船「三巨頭」正式上線後至少保證有兩個超級得分點出戰的11場比賽（對陣公牛倫納德和路威都因傷缺席）里快船就至少有這4場比賽「三巨頭」作用1+1+1是小於3的！平均每3場比賽就會出一次狀況。而他們同時出戰的比賽三人表現都能算得上A+的在我看來只有兩場…

看文章的標題可能你已經知道我想說什麼了：里弗斯最幸福的是他有三個能夠獨當一面的外線超級得分手；但他也有一個最大的煩惱就是不知如何合理地分配三人的出戰以及球權分配。所以我們提三個問題：

①三個需要有球權在手才能發揮出最大威力的外線得分機器同時放在場上真的能夠做到1+1+1大於3嗎？

②如果三人要同時上場該由誰持球？

③如果不能達到預期效果是否應該將他們拆開後平均分配球權？

「掌舵手」路威的「失聯」

這支「銀河戰艦」缺少一個傳統的一號位已經不是什麼秘密，全隊場均觸球數（67.7次）、場均持球時間（5.2分鐘）、每次觸球控球時間（4.59秒）以及每次觸球後運球次數（3.74次）等衡量球隊「掌舵手」的數據排在第一的不是首發控衛貝弗利，也不是「卡椒」，而是球隊三號得分點、替補席上的尖刀路威。一個星期前當我發現這個問題時挺奇怪的，因為很多球迷還口口聲聲說「路威球權太少了所以發揮不出上賽季的水平」…

那麼這個全隊控球時間最長的「掌舵手」究竟是不是影響「三巨頭」同時發揮的根本所在？先來看一個數據：

  • 路威場均傳給倫納德的成功傳球數是6.4個，實際助攻數是0.5次；
  • 場均傳給喬治的成功傳球數是3.9個，實際助攻數是0.5次；
  • 場均傳給貝弗利的成功傳球數是4.5個，實際助攻數是0.4次；
  • 場均傳給哈雷爾的成功傳球數是12.2個，實際助攻數是2.2次；
  • 場均傳給格林的陳宮傳球數是3.8個，實際助攻數是0.6次。

以上就是路威作為一個場均持球時間最長的實際控球者在開賽以來和球隊隊友之間互動前五的球員數據。

可能你會說他們同時在場的時間都不一樣，數據差異大不是很正常？那我們修正一下數據，讓全隊所有球員處在「同一起跑線上」比較一下：

上圖所示，在修正過後，「假設」路威和以上十位隊友分別在場的時間相同的情況下場均實際助攻哈雷爾2.20次依舊排在全隊第一，而場均實際助攻倫納德1.10次甚至還略微小於帕特森和格林的1.10次僅僅排在全隊第六…可是我們看路威在【相同時間裡場均成功傳球給喬治的次數卻高達13.71次超過哈雷爾排名全隊第一，場均成功傳球給倫納德11.78次排名全隊第三】，這就是說雖然在相同時間內路威傳給「卡椒」球的次數並不少，但轉化為實際助攻的效率卻非常低，遠低於傳給哈雷爾的球甚至還低於傳給哈克利斯、帕特森以及格林的。

在相同時間內，路威場均成功傳給「卡椒」二人的傳球次數合計是25.49次，合計實際助攻數是2.58次，傳球/助攻轉化率是10.12%！沒什麼概念？我們來做個對比：「假設」詹姆斯和戴維斯同時在場的時間與路威＆卡椒一致的話，他場均成功傳球給戴維斯17.01次，實際助攻數是3.29次，傳球/助攻轉化率是19.34%。看看詹眉這個成功傳球次數和實際助攻數的轉化率…所以快船的1+1+1怎麼可能大於3呢？

假如一定要「三巨頭」同時在場誰更適合持球？

回答完第一個問題後我們再來討論一下第二個問題，假如非要三人同時在場的時候應該由誰持球做進攻發起點？路威真的不適合作為持球點嗎？回答這個問題之前我們先來看一張快船隊【傳球質量】圖：

上圖是快船本賽季傳球方面的數據統計，橫軸為潛在助攻傳球比，縱軸為創造潛在助攻後隊友把握性（即命中率），藍色虛線是根據快船球員傳球能力做出的預測，虛線以上表示傳球質量超過球隊平均，虛線以下表示傳球質量一般。

這張圖我們可以得出的結論是路威的傳球質量很高啊，24.77%的潛在助攻傳球比還有高達58.88%的潛在助攻命中率，比平均值高了4.76%的命中率（球隊在24.77%的潛在傳球比下平均潛在助攻命中率應是54.12%），遠高於喬治、倫納德以及貝弗利。

但我覺得這個圖只能表明路威傳球質量不低，但是無法證明他就是五小、尤其是「三巨頭」同時在場時的最佳持球點，因為我們應該先再看一張快船死亡五小陣容互相之間的場均傳球分布圖，分析一下五小在場時相互間的傳球（助攻）結構：

上圖球員名字前的數字表示他場均傳給另外四人的成功傳球數之和；兩位球員之間弦面積越大、越寬表示他們之間相互傳球越頻繁，比如路威和哈雷爾之間的白色部分面積最大、最寬，表示他們兩人之間連接最頻繁；弦色表示場均成功傳球和場均成功接球數的關係大小，比如和貝弗利連接的四條弦均為黑色，代表他傳給另外四人的成功傳球數均多於他接到另外另外四人成功傳球數，喬治前面三條弦為黃色則表示他傳給哈雷爾、倫納德和路威的球都大於他接到這三人成功傳給他的球。

結合上面兩張圖我得出的結論有三點：

路威潛在助攻命中率高是因為調內線餵給哈雷爾的餅多，甚至超過了喬治和倫納德餵給哈雷爾的餅數之和。如果我們把所有人傳給「吃餅俠」哈雷爾的數據都拿掉的話（因為計算過程複雜，所以我只公布個答案），路威的潛在助攻命中率低於球隊平均值1.47%，喬治高於球隊平均值1.33%，倫納德低於球隊平均值1.29%，貝弗利低於球隊平均值2.16%；

五人同時在場的時候貝弗利是最符合球隊優秀持球點特徵的人，因為「出球」遠比「回球」多，不會出現「球給他就結束一個進攻回合」的情況。但是考慮到他的潛在助攻命中率不高，也就是傳球質量不高，所以可以作為五小進攻端的第一或第二持球點；

喬治出人意料地成為最好的持球點選擇，至少在三巨頭同時在場且貝弗利不在場的情況。

「多餘」的「卡椒」

實話實說吧，上面分析了那麼多數據無外乎證明一點：路威不能和「卡椒」同時在場，非要在場的話應該由喬治做進攻發起點…

這個結論我已經說過不止一次了，我非常反感有人「信口開河」地認為路威狀態不如上賽季是因為球權太少…請問您真的查了數據嗎？路威球權已經是全隊第一了還要怎麼個增加法？所謂的「球權」只是看出手而不是看實際佔有的控球時間等數據？球都在他手上由他支配著你跟我說他狀態不如上賽季是因為球權太少？哪怕看出手數他15.8次也比自己上個賽季的15.2次更高啊？

路威的問題在哪裡？在於他只能和哈雷爾產生化學反應，「三巨頭」或者「死亡五小」同時在場的話路威作為實際控球者依舊只能單方面和哈雷爾聯繫，完全無法讓喬治尤其是倫納德參與其中。

貝弗利為什麼不會有「不良反應」？因為即便貝弗利也是球隊綜合持球率前三的球員，但是他的定位不同，他完全可以做一個純粹的持球點串聯起另外四名隊友的進攻，可是路威不行，如果他要承擔更多終結任務的話很難和外線的隊友聯繫起來，不單單是本賽季，即便之前幾個賽季依舊如此。其實很多外線終結點都很難和同是外線的隊友聯繫起來，不僅僅是他一人，另一個典型是離開騎士後的歐文。

之所以會出現這種情況的原因我覺得在於打法，以上個賽季的路威和歐文為例，我們拆解一下看看他們的進攻方式有什麼共同點：

擋拆接球、手遞手和繞掩護這三項需要隊友配合戰術的得分方式不僅僅兩人使用頻率低（只有歐文的繞掩護使用頻率「勉強」超過10%），甚至效率也不算太高，沒有任何一人/項超過聯盟70%的球員。可是在擋拆持球方面兩人都是使用頻率最高且得分效率排名均超過了聯盟80%的球員。

擋拆持球是最簡單粗暴的戰術配合，都不需要隊友有很好的回傳球能力，只要有一個高大的內線延阻防守球員就足夠。所以「卡椒」兩人雖然擋拆後的【威脅】與【回傳球能力】要高於哈雷爾，可是這種配合對於路威來說不僅僅不熟悉，甚至效率也一直不算太好，哪怕是擋拆後路威傳球給順下的「卡椒」效果也一定不如給順下的哈雷爾高。所以這會讓「卡椒」在以路威為發起點的進攻中顯得「多餘」…

快船的最佳選擇

不過坦白地講，前面我說「假如一定要三巨頭同時上場的話應該由喬治持球」只是一種安慰…因為雖然從數據上來分析喬治是最適合的持球點，可是現實告訴我們：喬治持球就是個無底洞啊…

  • 因為路威因為傳球而非自己被斷或運丟造成的失誤比高達67.44%（應該也可以再次證明路威不是一個好的進攻持球點）；
  • 倫納德因為傳球而造成的失誤比是32.88%；
  • 喬治因為傳球造成的失誤比雖然不及路威可是也達到了56.92%。

你要注意，喬治基本上持球時間只有路威的五分之三，所以如果增加持球時間的話…你真的敢把球交給喬治？

所以，關於文章開頭的最後一個問題「該不該把『三巨頭』拆開？」我已經有了自己的答案了。雖然將球交給喬治可能會是「三巨頭共存」的一個美好嘗試，但是我認為這或許會給他們帶來更多的問題，很可能只是「飲鴆止渴」。我依舊堅持認為快船最好的方式是把「卡椒威」拆開，讓路威和哈雷爾帶替補，哪怕縮減路威的出場時間！

因為【路威身邊沒有倫納德和喬治但是有哈雷爾的時候】命中率提高了6.1%；且因傳球而造成的失誤比降低了37.63%，只剩下29.81%；所有有數據統計的進攻方式投籃效率平均提高了2.74%；同出場時間下突破次數上升了1.88次，突破成功率提高了4.93%。可能唯一的負面影響就是三分命中率低了將近4%…

當然，我並不是認為他們仨永遠不能同時在場，只是我覺得在特殊時間段且對方內線威脅並不大的情況下才可同時祭出「三巨頭」。否則順的時候三巨頭會令人感到恐懼，不順的時候至少有其中一個「巨頭」就是擺設、1+1+1就是小於3！

我雖然不認為快船本賽季表現糟糕，但是也確實說不上符合預期，儘管傷病是其中的重要原因之一，可我看到他們根本上就存在著陣容的適應性問題。打法的不兼容註定很難依靠強制磨合解決問題，及時調整陣容的適配以及球權分配問題對他們來說或許更是當務之急。
