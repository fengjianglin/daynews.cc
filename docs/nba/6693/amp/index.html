逆轉24分取勝！17次失誤！火箭的心裡的苦，你終於品味到了

天天要聞
3 週 ago

NBA比賽繼續如火如荼的進行。如今的聯盟，眾球隊在引援上， 都非常注重球隊的攻防平衡，這是衝擊季後賽，甚至是問鼎總冠軍的重要砝碼。但也不乏有些攻強守弱的球隊，也可以拿到不錯的戰績，比如休斯頓火箭隊。但是在最近接連面對掘金、快船等防守出色的西部豪門時，即便擁有頂級進攻火力的火箭隊也要吃癟。今天的比賽，又有一支球隊，嘗到了火箭的苦。

在老鷹主場對戰森林狼的比賽中，擁有超新星特雷楊的主隊與擁有兩大狀元秀維金斯和唐斯的森林狼大打對攻，特雷楊全場比賽瘋狂出手28次，一人砍下37分6籃板9助攻，首發球員賈巴里-帕克得到22分7籃板2助攻。對面的維金斯和唐斯與不遑多讓，前者得到25分6籃板，後者得到28分13籃板8助攻。

比賽上半段，主場作戰的老鷹依靠強有力的攻勢，一鼓作氣建立起12分的領先優勢，但是之後森林狼防守上的優勢開始顯現出來了，不論是籃板球、搶斷，還是蓋帽數據，都壓制住了主場作戰的老鷹。況且今天的老鷹，在比賽場上頻頻失誤送機會，全場比賽總共送出17次失誤。最終讓客場作戰的森林狼逆轉24分，以12分凈勝分笑納勝利。

此役過後，根據數據統計，老鷹隊現在的防守效率排在全聯盟的倒數第3位，與他們的常規賽戰績有些吻合。目前老鷹隊以4勝13負的糟糕戰績排名東部倒數第2位，依照這種趨勢走下去，他們可能和魚腩球隊尼克斯一樣，都會早早的退出季後賽之爭。

該注重防守了，進攻強如火箭這樣的西部豪門都因為防守上的劣勢在最近遭遇連敗，更不用說進攻效率同樣倒數的老鷹隊了。火箭的苦，特雷楊，你終於體會到了。
