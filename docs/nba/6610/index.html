希羅22+8、阿德巴約17+11，熱火不敵404！“三巨頭”84+19+14

2019-11-28 0條評論 15次閱讀 0人點贊

北京時間11月28日，NBA2019-2020賽季常規賽如火如荼地進行中。來自邁阿密的熱火隊客場挑戰位於休斯頓的404隊。最終經過一場激烈的角逐，熱火隊在新秀希羅狂砍22+8的數據下，不敵休斯頓的404隊，球隊再吞一敗嗎，本場比賽巴特勒缺陣！

今年的熱火隊，備受關注的除了巴特勒的加盟，還有兩位新秀希羅和納恩。其中納恩更是在媒體上表示，他今年將會是最佳新秀。可是，本場比賽納恩的表現並不理想。相反，在打替補的希羅表現的非常出色，不僅差點將敗局已定的熱火隊從生死邊緣拉了回來，還打出了22+8的高效數據。作為一名來自2019年13順位的選秀後衛，其光明的前途似乎在跟前輩科比、米切爾等人看齊。

而相比於熱火隊，一直遭遇連敗的休斯頓404隊的狀態也備受球迷關心。本場比賽，連敗後的404隊表現出來的鬥志非常的旺盛。雖然球隊的主力中鋒卡佩拉缺陣，但球隊的小前鋒豪斯表現的非常搶眼，不僅拿下了23+4+2的豪華數據，儼然成了火箭的第三個巨頭。

而臨時頂上來的克拉克，表現也非常的穩健，全場貢獻12+8的表現，對得起他這個新秀二年級的菜鳥。

與其說是新人的發揮讓404隊拿下了比賽，倒不如說是雙核的穩定輸出為贏下比賽打下了堅實的基礎。詹姆斯哈登砍下34+6+5的數據，威斯布魯克拿下27+9+7的數據。雙MVP的組合，讓世人很難對這支球隊刮目相看。

如今賽季已經過了將近四分之一，各隊的戰績也初見端倪。作為狂野西部的一份子，404隊處在前八行列非常的可觀。而熱火隊，在東部的戰績也一直處在季後賽的行列。相信，在擁有巴特勒、德拉季奇、希羅、阿德巴約等球員的前提下，熱火隊今年一定可以挺近季後賽。而404隊更進一步，進入西決，也很有希望。
