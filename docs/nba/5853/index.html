國王背靠背挑戰火箭，阿里扎重回豐田中心，威登率隊將獲三連勝？

2019-12-10 0條評論 19次閱讀 0人點贊

北京時間12月9日上午8點整，在剛剛結束的一場NBA常規賽中，排名西部第10的青年軍薩克拉門托國王客場以110比106擊敗達拉斯獨行俠隊，爆了一個不大不小的冷門，東契奇在最後時刻本有機會扳平比分，但隨著他最後一投不中，國王最終憑藉整體優勢拿下比賽。

國王隊在比賽結束後隨即前往豐田中心，挑戰德州的另一隻勁旅休斯頓火箭，阿里扎作為休城當年巴哈馬五小中的核心鋒線，繼上賽季之後重回豐田中心，那麼威登組合能否率隊取得三連勝呢？

哈登賽季至今場均可以得到38.5分7.4助攻6.1籃板，但在上場對陣太陽隊的比賽中投籃效率不佳，全場27次出手只命中其中的7球，而這其中很大一部分原因是因為他遭到了對手針對性的防守。

哈登的得分能力毋庸置疑，在球隊本賽季的八連勝期間，他多次打出40+的表現，而在本賽季得分超過40分的比賽中，火箭只是在客場以兩分劣勢敗走聖城，其餘的比賽均獲得勝利，哈登對於休城的重要性不言而喻。

火箭在之前的三連敗期間，每當哈登遭到包夾的同時，場上隊友的表現如同斷電，球隊在進攻端陷入困境，而威少也無法掌控比賽的節奏，休城場上的運轉一時間陷入癱瘓。好在主教練德安東尼及時變陣，加上威少逐漸適應球隊節奏的基礎上，火箭在最近5場比賽取得4勝1負的戰績。這期間不得不提到角色球員麥克勒莫，他在最近4場比賽中，三分球45投22中，命中率達到驚人的48.8%，不僅幫助球隊緩解進攻端的壓力，而且在威登身邊甘願做臟活累活，不論首發還是替補，快攻三分樣樣都有。

威少目前場均可以得到21.9分8.4籃板7.5助攻，雖然數據相較於以往出現不同程度的下滑，但作為球隊的第二選擇，加上需要適應全新的球隊，他的表現也在情理之中，借用哈登的採訪來說，威少還會繼續變得更好，這是他職業生涯首次離開俄克拉荷馬城，20到25場的比賽磨合一定是不夠的，也許這需要一整年的時間。

的確，威少在本賽季只拿到6次三雙，而火箭在這其中取得5勝1負的戰績，不過他在其餘比賽里的表現，休城的戰績可謂是喜憂參半，但他為球隊帶來的變化是任何人都不能詆毀的。

火箭作為聯盟過去幾個賽季進攻節奏最慢的球隊之一，由於威少個人的打法所致，球隊本賽季的進攻節奏已位居聯盟中游，他們最明顯的變化來自於籃板球和快攻得分。火箭上賽季場均籃板只有42個，排在全聯盟倒數第三，這間接導致休城快攻得分上的不足，放眼當今小球打法盛行的年代，快攻得分已經成為一支球隊進攻的常規武器。隨著威少的加盟，火箭本賽季場均可以搶上48.2個籃板，排名聯盟第三，隨之而來的就是球隊快攻得分的增加，而威少的籃板球保障完成了全部的推進工作，也為火箭增添了與上賽季完全不同的風格。

火箭本賽季主場取得9勝2負的戰績，勝率高達81%，雖然球隊目前遭遇不同程度的傷病問題，但隨著威登組合的深入磨合，球隊的體系日漸成熟，而國王當家球星福克斯和首發中鋒巴格利均因傷病將缺席明天的比賽，這對於球隊的攻防兩端將產生不同程度的影響。

鐵柱認為，儘管火箭在整體實力明顯佔優的情況下，也不能輕視薩克拉門托的青年軍，火箭本賽季曾多次因為輕敵被對手反超甚至輸球，加上阿里扎對於火箭隊的了解以及防守上對哈登的限制，無疑為明天的比賽增添了幾分不確定因素，而威登組合能否率隊取得三連勝呢？讓我們拭目以待。
