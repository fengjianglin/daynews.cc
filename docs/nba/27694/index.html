深度分析∣當今聯盟打法風格及球星發展方向

2019-12-21 629條評論 3,854次閱讀 2863人點贊

上世紀80年代末，前任NBA總裁大衛·斯特恩帶著幾盤錄像帶，在央視大樓用耐心和等待敲開了中國市場的大門。30年來，NBA和中國籃球迷彼此需求，締造了體育界空前罕見的繁榮盛景，越來越多的中國人因為NBA更加熱愛籃球，看NBA比賽構成了大部分國人球迷的青春與情懷。

NBA在中國市場風靡

30年來，NBA聯盟在中國蓬勃發展，在國內的熱度直線上升。聯盟各項規則不斷完善，各球隊的打法風格也經歷著翻天覆地變化，每個時代的超級巨星技術特點也大相徑庭，唯一不變的也許只有不同時代球迷對這項世界頂尖籃球聯賽的喜愛與狂熱。

30年來NBA比賽打法風格的變化

國內最早一批能夠通過影像資料或者視頻直播看到NBA比賽是在90年代初，整個90年代是NBA在國內從起步到迅速發展的重要時期。那個年代恰恰是籃球之神邁克爾·喬丹唯我獨尊、統治聯盟的時代，國內最早的籃球粉絲群體基本都將喬丹視作唯一偶像，能夠欣賞到籃球之神的賽場魅力是那個年代球迷的幸運。進入21世紀，喬丹退役並老去，OK組合，馬刺GDP，姚麥組合，以詹姆斯為首的03黃金一代，水花兄弟等一批超級球星以傳承的方式陪伴著中國球迷的青春，為不同時代的球迷帶來美好的籃球記憶。

近30年來，NBA聯盟的打法風格在不斷發生變化。我們不妨先通過數據來直觀看一看其中的奧秘。樣本選自不同時代、但在聯盟歷史上締造輝煌榮耀的傳奇球隊，分別為95-96賽季（締造王朝）的芝加哥公牛隊、12-13賽季（創造27連勝）的邁阿密熱火、15-16賽季（締造73勝9負紀錄）的金州勇士隊。

三支傳奇球隊基礎數據

三支歷史級的球隊擁有各自的特點，95-96公牛隊和12-13熱火隊場均得分和失分，與15-16勇士隊相比較低，投籃命中率相差不大，這意味著勇士隊的進攻節奏更快，效率更高，但防守端的失分也非常高。比賽節奏的加快導致對手的得分整體水平也提升不少，同樣凈勝分10分以上，95-96公牛隊對手得分僅92.9分，而15-16勇士隊對手得分高達104.1分，這與那支的公牛隊進攻得分相差無幾，現今聯盟得分的容易程度似乎在提升。

傳奇球隊的三分球

從數據可以看出，三支歷史級強隊最明顯的變化在於場均三分球出手數和命中數，90年代的公牛隊場均三分球命中僅6.6次，13年的熱火三分命中為8.7次，而16年的勇士隊場均三分命中竟然高達13.1次。也就是說15-16賽季的勇士隊場均依靠三分球得分高達39.3分，佔總得分比重達34.2%；而賽季95-96公牛隊三分球佔總得分比例僅18.8%。數據似乎在告訴我們：相比於上個世紀的比賽，當今NBA聯盟各支球隊更喜歡、也更善於利用三分做主要進攻手段，進攻可利用區域和攻擊範圍拓寬很多。是否真的如此呢？下面再來看一組數據。

76人和勇士近10賽季三分球表現

這組數據選取兩支球隊近10個賽季的三分球表現，樣本分別為金州勇士隊和費城76人隊。從數據中明顯看出，近10年來兩支球隊三分出手數和命中數整體上都在逐年上升，尤其14-15賽季以後兩支隊場均出手三分數都在25次以上，16-17賽季以來的近幾個賽季兩支球隊三分命中數都在10次以上，也就是說球隊場均利用三分球得分至少在30分以上。近兩個賽季76隊一直被外界詬病為不善於利用三分球得分的球隊，很多專家甚至認為這將成為他們奪冠路上的”絆腳石”，然而就是這樣一支球隊上賽季的場均三分出手仍在30次以上且場均命中10.8次。

勇士隊近十賽季三分球

76人隊近10賽季三分球

分析到這裡，我們不得不承認，當今聯盟各支球隊在充分利用三分球作為主要得分手段。相比較於90年，21世紀初更多利用內線和中距離進行得分，如今的聯盟更加註重24英尺外的投籃得分。因為三分球的得分效率更高，如果一支球隊三分球命中率只有30%（並不高），和兩分球命中率45%相等價。隨著聯盟球員整體三分命中率的提升，這種遠距離的進攻方式被認為更具有效率。本賽季的比賽中，三分線外的出手比例高達37.9%，16英尺到三分線之間的手比例僅8%，10英尺到16英尺之間的比例為9.2%，而10英尺之內的出手比例為44.9%。這體現了近幾年聯盟”魔球”理論的盛行，即出手得分集中在三分線外或者油漆區內，這種得分方式被認為是高效的。越老越多的球隊開始重視三分球的價值，並在打法上傾向於利用三分球完成得分。

聯盟整體打法風格的巨大轉變和當今聯盟球星的技術風格變化相輔相成，趨勢化的打法風格對球員提出新要求，想要成為巨星或者在當今NBA佔有一席之地，必須要適應球隊的要求。各位置球員的發展方向也和10年前相去甚遠。這是聯盟發展到這種程度帶來的必然結果，球員們只能適應趨勢發展，豐富自己的技能，才能在聯盟實現價值。

當今聯盟後衛球星的發展方向

後衛這個位置主要負責球隊串聯和得分任務。前面已經分析過，當今聯盟的打法風格使得頂級的後衛要有出色的三分球能力，擁有不俗三分球能力的後衛可以更好的拉扯球場空間，防守人必須從過半場就投入精力。優秀的後衛利用三分球威懾力，能夠豐富進攻端的選擇，突破內線造成殺傷還是助攻空位隊友都會變得更加容易。

水花兄弟引領後衛風潮

近10年內最出色後衛當屬斯蒂芬·庫里，生涯三分球命中率高達43.5%，巔峰賽季場均命中5.1次三分球。他的隊友克萊·湯普森同樣出色，生涯三分球命中率41.9%，巔峰賽季場均3.5次三分球。憑藉兩人突出的三分投射能力，勇士隊打造屬於自己的傳控團隊籃球體系。毫不誇張的說，以水花兄弟為基石的勇士隊引領了當今聯盟的打法風格，是當今這種籃球體系打法中最成功、最出色的球隊。庫里和湯普森為後衛球星設立了發展的標杆，不論PG位還是SG位，一定要有出色的三分球能力。持球進攻強，擁有極高的三分威脅，能夠帶動隊友，最強模板就是庫里。無球能力強，防守強硬三分精準，最強模板是湯普森。無論哪種風格的後衛，練就出色的三分球能力成為後衛球員的發展方向。

現今聯盟最頂尖的SG

哈登成為近幾年炙手可熱的巨星，很大程度上依賴其出色三分球能力，除了優秀的突破節奏和腳步，造犯規能力突出外，哈登最具威脅的就是精準的三分球，上賽季場均36.1分，其中13.2次三分出手命中4.8次，場均依靠三分球可以砍下近15分。出色的三分球能力使得哈登擁有更多的進攻選擇支配，球場可以利用空間增加，防守者必須提到很高位置對他進行壓迫，這樣他可以輕鬆突破後分球給空虛的內線或者無人盯防的外線隊友，三分球的威懾力幫助哈登獲取更輕鬆的自主進攻或者串聯球隊的機會。相比於另外一位巨星，也是他的隊友韋斯特布魯克，身體素質似乎更加出色，卻在媒體和球迷心中沒有達到哈登那樣的評價高度，原因在於不具備出色穩定遠投的能力。

不同時代頂級後衛的風格轉變

當今頂尖後衛更愛三分球

可以拿不同時代歷史級別的後衛和當今頂尖的後衛進行數據比較，樣本選擇四位MVP球星當賽季的統計數據，分別為95-96賽季喬丹、07-08賽季科比、15-16賽季庫里和17-18賽季哈登。喬丹當賽季場均砍下30.4分，三分出手比例僅為14.16%，科比當賽季砍下28.3分，三分球出手比例24.76%，接近四分之一。庫里和哈登當賽季場均30分左右，哈登三分出手接近50%，而庫里三分比例已超過總出手數的一半。相比於老一輩後衛，近些年聯盟頂尖後衛更喜歡並且善於利用三分球得分。

現今後衛更善於利用三分解決問題

當今聯盟相對優秀的後衛，都擁有不錯的三分球能力，不具備三分球能力的後衛也在開發這項技能。本賽季場均三分出手，特雷楊9.1次、比爾7.9次、利拉德9次、米切爾6.3次、拉文7.6次，得分榜前列的後衛無一不擁有較高的三分出手和相對不錯的命中率。一向以不會三分著稱的隆多本賽季場均也出手三次三分，並且命中率高達43.8%，從不投三分的本西蒙斯本賽季也命中兩球。投三分不僅是當今聯盟對後衛提出的新要求，也成為一種”時尚”。

當今聯盟鋒線球星的發展方向

鋒線球員擁有比後衛球員普遍的身高優勢，又擁有比純內線球員出色的速度和靈活性，相對來講更為全面。當今聯盟的鋒線球員發展方向多樣，卻都能各自充分發揮作用，主要可分為幾類。

優秀的組織型前鋒

  • 兼備強悍的進攻能力和出眾的組織能力：身體素質和天賦爆棚，擁有出眾的球場視野和比賽掌控力，能夠利用強大的進攻能力牽制對手，進而為隊友創造進攻機會，能夠及時將球分享出去，組織能力堪比頂尖控球後衛。擁有這種能力的球星屈指可數，最典型的代表為勒布朗·詹姆斯。近些年來，詹姆斯統治聯盟正是依靠其超凡的引領球隊進攻的能力，聯盟三分投籃的盛行與詹姆斯的能力非常匹配，給他配備足夠的射手群，他可以利用突破能力吸引防守為外線創造輕鬆出手機會。詹姆斯三次問鼎奪冠都是射手群+一位頂尖單打球星（韋德或歐文）的配置，在這套陣容配置中詹姆斯可以充分發揮自己的優勢。

頂尖攻防一體的鋒線

  • 兼備頂尖的進攻能力和不俗的防守能力：擁有極強的個人單打能力，關鍵時刻攻堅能力強、得分不手軟，防守能力出眾。典型的代表為杜蘭特和倫納德，如果沒有出色的組織能力，那麼將進攻磨礪的不可阻擋，防守端能夠發揮支柱作用，這種攻防一體的鋒線也是鳳毛麟角，並且能夠取得成功。像杜蘭特和倫納德這種球員，任何爭冠球隊都渴望並不可或缺的巨星，他們在關鍵場次決勝時刻的表現從不讓人失望。

備受推崇的3D型鋒線

  • 兼備出色的防守能力和三分球能力：上面論述的特點需要依賴出眾的天賦，達到那種境界需要不俗的身體素質兼具堅持不懈的努力。而第三類球員則可以從天賦平平的球員中培養出來，也就是我們常說的3D球員。防守和三分球都是可以通過努力進化出來的，這類球員儘管稱不上球隊的核心，但絕對能夠貢獻重要力量，實現在賽場上的價值。比如伊戈達拉，丹尼·格林，考文頓等是這類球員的典型代表。

要論鋒線球員的發展方向，如果沒有絕對的天賦，無法擔任出色的組織前鋒，也無法打造頂尖的攻防一體能力，那麼苦練三分球並培養優秀的防守習慣將成為鋒線球員最有價值的成長途徑。

當今聯盟內線球星的發展方向

曾幾何時，內線球員是聯盟最搶手的球員，90年代”四大中鋒”的時代，內線霸主在賽場予取予求，風光無限，當今聯盟似乎很難發掘出擁有如此”統治力”的內線巨星。然而筆者並不這樣認為，正因為聯盟打法風格的變化對當今的內線球員提出新的要求，內線球員尤其中鋒不需要像曾經那樣硬抗籃下取分，籃下對抗的減少和禁區內砍分的降低，給球迷一種統治力不強的錯覺。

超級內線攻擊方式的轉變

選取21世紀的傳奇內線奧尼爾（01-02賽季）、鄧肯（03-04賽季）和新生代（18-19賽季）的恩比德、約基奇為樣本，比較他們得分方式和投籃選擇上的不同。從數據中可以看出奧尼爾賽季禁區內出手1047次，三分線外出手1次；鄧肯禁區內702次，三分線外12次；恩比德禁區634次，三分線外264次；約基奇禁區內726次，三分線外271次。當今的聯盟優秀的中鋒在進攻上擁有更廣擴的範圍，三分球能力在平均水平之上，且禁區內命中率上並不比傳統中鋒遜色，禁區命中率方面，奧尼爾62.9%，恩比德61.8%，約基奇60.3%。

由此可以看出，當今聯盟內線球員在球場上承擔的角色更多樣化，而不是單一的守護藍框，進攻僅停留在禁區內。但毋庸置疑的一點是，優秀的內線球員必須擁有出色的護框能力和籃板保護能力，這點和傳統中鋒的特點不謀而合。

中鋒也擅於組織進攻

擁有出色組織能力的內線在聯盟炙手可熱，約基奇率領下的掘金隊上賽季一度高居西部第二，依靠他的串聯能力，掘金隊整體運轉非常流暢，他更是場均送出7.2次助攻，刷新球迷對中鋒場均助攻的認知。上賽季小加索爾在猛龍隊也是扮演出色組織者的角色，有了內線的串聯，猛龍的團隊進攻有聲有色。

中鋒也可擅長中遠投

擁有外線投射能力、技術全面的內線在聯盟擁有重要位置，恩比德、濃眉哥、唐斯、波爾津吉斯等一眾內線憑藉全能的技術和出色的投射能力在各自球隊扮演不可或缺的支柱角色。進攻端既有籃下衝擊力又能夠提到高位遠投釋放內線空間，這樣的球星容錯率極高，能很好的搭配不同類型的隊友。

出色的內線藍領

如果沒有出色的組織能力又不具備不錯的投射能力，那麼作為內線球員想要立足聯盟，必須要有出眾的護框能力和籃板保護能力，進攻端不承擔重任，靠偶爾的”吃餅”完成終結。這樣的球星作為角色球員，也能發揮重要作用。比如魔獸霍華德，小喬丹，卡佩拉等，這些擁有不錯防守能力球星在季後賽的關鍵時刻能夠幫助球隊渡過難關。

聯盟比賽風格已完成終極進化？

近幾年來，隨著小球風格的盛行，越來越多球隊開始重視三分球的價值。油漆區內和三分線外的得分成為進攻的主導方式，兩者佔據80%以上的出手權。”魔球”理論似乎已經統治聯盟，中距離投籃似乎正在逐漸消失。曾幾何時，”籃球之神”喬丹和”黑曼巴”科比賽場上優雅美如畫的後仰跳投如今幾乎銷聲匿跡，甚至部分球迷認為各支球隊已經找到最有效的進攻方式，NBA比賽已完成終極進化。

中投仍是當今聯盟巨星的殺手鐧

筆者認為，說NBA比賽風格已完成終極進化，實屬有些誇張。籃球比賽需要不斷創新，不斷變化，不斷與時俱進。籃球比賽風格和當今球員技術特點與發展方向協調發展，球員的技術打法和能力造就了如今的賽場風格，賽場風格的變化也對球員提出新的要求。三分球已成為球隊得分的主要進攻選擇，然而凡事都不是絕對的，中距離投籃依舊是當今很多超級巨星的”殺手鐧”，上賽季季後賽東部決賽中，倫納德正是用無解的中距離進攻改變賽場走勢，倫納德在5-14英尺出手為37次，15-24英尺33次，而25英尺外只有26次，且15-24英尺命中率高達51.5%。中距離投籃依舊是很多超級巨星最擅長最穩定的得分手段，幫助勇士兩連冠的杜蘭特也是這一領域的頂尖高手。

新賽季精彩繼續

三分球價值被充分挖掘，球場進攻空間被更充分利用是聯盟比賽風格進化的體現，但不代表傳統的技術風格要被淘汰，強硬的內線單打，柔美的中距離投籃依舊佔有重要位置。現如今，更樣多的進攻方式和技戰術被開發和融入到比賽中，這是新時代籃球發展的需要和必然結果。

作為熱愛籃球的球迷，我們不必為籃球比賽風格變化而爭論不休，我們只需要靜靜的欣賞比賽帶給我們的快樂，享受新一代籃球巨星帶給我們的激情與鼓舞，享受那些曾經輝煌的偶像帶給我們的回憶與青春。
