利拉德，一個被低估了的男人，如今的NBA巨星

天天要聞
2019-12-20

1990年7月15日利拉德出生於加利福尼亞的奧克蘭。他們的家庭算不上富裕，但也不至於無法解決溫飽問題，從小利拉德就喜歡上了打籃球，他和小夥伴們常常會對著一棵大樹進行投籃練習，甚至是暴扣。距離拉德回憶說，在大樹上進行扣籃，是我表哥開創的，因為他們都比我大，每次我都是站在旁邊看著他們跳起來扣籃，我是其中最矮的一個，不能扣籃，所以我只能跑到一邊，對著大樹進行投籃。突然有一天在利拉德剛回家，進門的時候，奶奶就急忙跑過來，告訴他，那棵大樹被砍掉了，聽到這個消息的時候，此時的利拉德感到非常的沮喪，因為沒有了那棵大樹，他也不知道接下來要到哪裡去練習投籃。後來爺爺知道了這件事之後，就對他說，孩子，我們有一個牛奶箱子，我可以把它的底部去掉，然後給你做一個籃筐，接著他的爺爺又把這個箱子扔在了電線杆上，這就成為了利拉德童年時的第一個籃筐，從此以後他也深深的愛上了籃球。

雖然如今的利拉德早已成為了nba球星，但是在他通往成功的這一條道路上，卻充滿了荊棘高中期間利拉德前後轉了兩次學，因為那個時候，他的表現並不出眾，他是在一段時期里淪為了學校籃球隊的角色球員。

1990年7月高中時期絕大多數的籃球媒體和球探都不看好利拉德，在他們那一屆的全美高中生評級裡面利拉德僅僅是以兩星級的球員而最高的評級，則為五星，這就意味著在高中階段利拉德並不是一名出色的球員，因此，在他高中畢業之後，都沒有收到任何一所籃球名校的獎學金入學邀請，其實那個時候利拉德也非常渴望去到一所籃球名校里打球，但是為了減輕家庭的經濟負擔，最後他還是選擇了小聯盟的韋伯州立大學，那是一所紀律嚴明的學校每一位球員的管理都是非常嚴格的。時任校籃球隊主教練的貝克納常常會給利拉德說，你的表現還不夠好，你必須要更加努力一些大學期間，利拉德每周有6天的時間，需要進行力量練習，游泳和投籃訓練。

一段時間，利拉德每天都會覺得身體特別的累，感覺都快吃不消了，有一次利拉德覺得自己已經無法再堅持了，於是他在比賽前夕沒有徵得主教練的同意，就給自己放假了，沒想到主教練後來直接衝到了樓上，當著所有隊員的面把它給揪了出來，訓斥你知道為什麼絕大多數人都會失敗嗎？因為他們總是忍不住用一生所求來換一時之需，聽完主教練的這一番話之後，利拉德反思了很久，他也明白了其中的道理，從此以後。每天都會非常努力的訓練，就算再苦再累，也沒有說過要休息，在韋伯州立大學的4年時間裡，利拉德在貝克納教練的培養之下，從一名無緣nba的球員最後在2012年以6號秀的身份進入到nba。
