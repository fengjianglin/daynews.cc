回顧NBA歷史最長的33連勝：比爾-沙曼教導有方，張伯倫無私奉獻

2019-12-04 0條評論 15次閱讀 0人點贊

1971-72賽季，湖人隊的33連勝之所以會發生，很大程度上是因為球隊擁有經驗豐富、才華橫溢的老兵核心。除此之外，這是一個被稀釋的聯盟，當時足可以被稱得上是勁旅的球隊除了湖人，就只有雄鹿和子彈隊了。

而且由於ABA聯盟的興起，聯盟里球員們的流動性很大。從1969年到1972年的三個賽季里，就曾經出現過聯盟里最長的13個連勝之中的四次。分別是33場連勝、20場連勝、18場連勝和16場連勝。這難道僅僅是巧合嗎？絕對不是。

首先，比爾-沙曼在當年接管了湖人隊，他可謂是聯盟里第一個「真正」的教練。那個時候，聯盟里的比賽日是這樣的：NBA球員們通常提前一個小時來到球館，先四處瞎逛一會兒，然後一邊抽著萬寶路香煙，一邊漫不經心地聽著教練做出諸如：「別讓威利斯搶到籃板」，「別讓門羅擊敗我們」之類的完全可以忽略不計的廢話。

而沙曼則不同，他在執教過程中非常追求細節，強調體能與反覆操練的重要性——這是如今聯盟里教練們非常習以為常的執教模式，但在六七十年代里，他們被大大地忽略了——沙曼還強制性地要求手下球員們每天都要進行拉伸訓練，讓球員們吃健康食品，戒除煙癮，而且還安排球員們參加賽前適應場地的投籃訓練，這樣他們就能適應不同場地的籃筐以及賽場環境，此外還給他們養成了每天觀看比賽錄像的習慣。

這些都是現代教練們經常做的事情，對於當時湖人隊這支老兵球隊而言，細節決定了最終的成敗。需要額外提及的一點是，沙曼也是調教張伯倫最好的一位教練。作為教練，他是第一個成功說服張伯倫進入拉塞爾式籃板、蓋帽機器模式的人，他還針對後者專門設計了一些心理戲，以此來穩住威爾特的情緒，控制他的想法，讓他感覺自己與身邊所有發生的事情都息息相關。

自此之後，張伯倫犧牲了無數次出手機會，但他也成功地把自己與「無私」和「團隊精神」順利掛鉤。所有這一切的一切，都讓1972賽季的湖人隊變得日益強大。所以，單從字面上來考慮，這支球隊不可能比1969賽季的湖人隊更強。當你考慮到聯盟被稀釋，沙曼在球隊的影響力，以及張伯倫的賽場重生，就看到了這最終的33連勝結果。
