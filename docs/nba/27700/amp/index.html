亞洲球員NBA單場30＋多難？林書豪僅5次，那姚明、阿聯、周琦呢？

天天要聞
2019-12-21

對於很多中國球員來說混NBA本來就是一件非常難的事情，畢竟由於身體素質的原因他們在對抗中長期處於下風，所以得分自然非常難了，至於砍下30+以上的得分更是難上加難，以林書豪為例，他在NBA效力的幾個賽季中，僅拿到5次30+，那麼姚明、阿聯、周琦是什麼情況呢？

林書豪5次，林書豪在NBA一共效力了10個賽季，剛開始只是一個落選秀，最後還是勇士給了一次機會，只不過這個時候都是替補身份出場罷了，2011年NBA停擺期間短暫的在CBA打球之後回到了NBA。之後又加盟紐約尼克斯，在這個地方他開始展現自己的實力，開始穩定地坐穩首發的位置，而30+也是從這個時候開始的。只不過之後來到籃網之後傷病太多 也是在這裡拿下最後一個30+。

姚明49次，姚明作為NBA最成功的球員，他的實力本來就是統治級別的，奧尼爾之後就是名副其實的第一中鋒了，即便是當年強悍的霍華德等幾個人面對姚明都是輸多勝少。而且姚明的技術比較全面，特別是籃下技術可以算是中鋒裡面的翹楚，加上7尺6的身高很多時候投籃讓對面沒法防，所以整個職業生涯一共拿下了49次30+，只不過還是傷病太多最後早早地離開了NBA。

易建聯1次，實際上易建聯的身體天賦和素質在NBA還是能夠立足的，當年第六順位被雄鹿選中不是沒有原因的，而且第一個賽季就有很多高光的表現，不過拿下30+就是他表現最好的一次，之後和很多亞洲球員一樣被傷病困擾，征戰5個賽季之後回到了CBA，不過即便是回到CBA他也是保持了很好的競技狀態。

周琦0次，至於周琦肯定是裡面天賦最高的，但是成就也是最低了，最大的原因還是沒有姚明和易建聯的那種自控能力，所以單薄的身材完全沒法在NBA立足，最高得分僅僅只有6分。
