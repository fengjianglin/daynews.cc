致命搶斷+上籃+MVP！！！布朗尼真的不給老爹面子啊

天天要聞
2019-12-16

北京時間12月15日，據媒體報道，勒布朗-詹姆斯的大兒子布朗尼在今天進行的一場高中比賽中得到15分，他在第四節最後階段完成致命搶斷和得分，並且成為本場比賽的MVP！！！

詹姆斯與他心愛的大兒子布朗尼

今天，Sierra Canyon中學和詹姆斯的高中母校聖文森特聖瑪麗進行了一場全美關注的高中籃球賽。

Sierra Canyon中學擁有詹姆斯的大兒子布朗尼和韋德的兒子扎伊爾。

由於湖人隊今天沒有比賽，詹姆斯帶著妻子專程前往俄亥俄觀看這場比賽。

布朗尼在今天這場比賽中的發揮非常出色，特別是第四節還剩不到1分鐘，Sierra Canyon中學以55-56落後1分，但是布朗尼在中圈完成了一次致命搶斷，他隨後自己運球快攻上籃得分，幫助本隊反超1分。此時，場邊的詹姆斯激動地跳了起來，並為兒子加油。

最終，Sierra Canyon中學以59-56擊敗聖文森特聖瑪麗。布朗尼全場得到15分，這是他高中生涯的單場最高得分，而且被評為本場最有價值球員。

賽後，場上出現了非常溫馨的一幕，詹姆斯拉著他的高中隊友和發小一起和拿著本場MVP獎盃的布朗尼合影留念，這可以說是真正的籃球傳承。

詹姆斯到現在還沒退役一個很大原因就是在等他的兒子布朗尼，布朗尼的表現讓詹姆斯很是欣慰，非常期待詹姆斯在幾年後在NBA球場上與他的兒子布朗尼來一場戰鬥，希望布朗尼技術能快速長進，早日與父親站在同一球場，加油吧，小布朗尼！！！
