一句話形容香蕉船四兄弟現狀，球迷：“詹瘋、韋退、瓜甜、炮貴”

天天要聞
2019-12-22

NBA著名的”香蕉船四兄弟「，是由韋德、保羅、安東尼、詹姆斯組成，這四位都是響噹噹的人物。之所以叫”香蕉船四兄弟「，是因為幾人出去遊玩，同坐一個香蕉船，後來大家起了這個名字。最近很多觀眾讓我用一句話概括四兄弟近況，若總結得不好，歡迎大家踴躍發言。

詹瘋：34歲的詹姆斯依然是NBA的門面，場均拿到25.9分7.4籃板10.6助攻，帶領湖人隊拿到西部第一的成績。同屆很多球員都已經功成身退，而詹姆斯還在帶領後輩衝擊總冠軍，這一切都太瘋狂了。你永遠不知道詹姆斯的極限在多少歲，永遠都在下一屆。

韋退：韋德拿著3屆NBA總冠軍、一屆總決賽MVP已經功成身退，手裡拿著香檳酒看著孩子逐漸成長，滿眼都是自己小時候的樣子。韋德的身體也慢慢開始發福，等到進入名人堂時，或許已經胖得認不出了。

瓜甜：安東尼當年加入雷霆，本以為能衝擊總冠軍，可總是事與願違，最後成為了背鍋對象， 觀眾表示：「這瓜沒熟」。離開雷霆加入火箭後，十場比賽就被否定，一系列的操作讓安東尼險些無球可打。新賽季開始後加入開拓者，場均得到16.8分6.2籃板1.2助攻，開拓者的成績也漸漸有了好轉，觀眾表示：「這瓜包甜」。

炮貴：保羅如今也是34歲，本賽季場均拿到16.3分4.4籃板6.3助攻，數據與上賽季相差並不太大，但是雷霆能穩在西部前八，這就是最成功的地方。很多人都說雷霆為什麼不交易保羅，試問保羅的合同聯盟如今有幾支球隊能吃下。保羅本賽季年薪3851萬美元，20~21賽季4136萬美元，21~22賽季4421萬美元。因為經濟不景氣專家都在預測NBA下賽季工資帽下調10%左右， 保羅這份合同可以說是天價，這個炮真貴。

一句話總結”香蕉船四兄弟「，」詹瘋、韋退、瓜甜、炮貴「，大家如果有更好的話，可以在下面留言。
