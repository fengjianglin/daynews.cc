無奈，新賽季被傷病毀掉的四支球隊！三支本要爭冠總冠軍

天天要聞
2019-12-22

新賽季被傷病毀掉的四支球隊！三支本是總冠軍競爭者。

勇士在休賽期的計劃是以庫里、拉塞爾、格林為核心，衝擊季後賽，然後讓克萊在季後賽之前回來，繼續衝擊總冠軍，我們也相信他們是季後賽球隊，畢竟底蘊在。但是庫里僅打了4場比賽就受傷了，場均只有20.3分，三分命中率只有24.3%，防守壓力太大。拉塞爾、格林也是打打停停，拉塞爾雖然能得分，但改變不了大的局勢，格林就是體系球員。勇士現在是奔著狀元簽去了，克萊和庫里都沒有必要回來，好好休息，下個賽季再來。

籃網現在的戰績為15勝13負，排在東部第七。沒有杜蘭特，籃網只能是季後賽球隊，有了杜蘭特就是總冠軍級別的球隊。歐文只打了11場比賽肩部就受傷了，場均得到28.5分5.4籃板7.2助攻。勒韋爾只打了9場比賽拇指韌帶損傷，場均得到16.8分5籃板4助攻。努瓦巴也接受了跟腱斷裂修復手術，賽季報銷。其中歐文和勒韋爾都是主要的得分手，幸虧有丁威迪，不然球隊早已跌出了前八。

活塞現在還是處在東部第十的位置，雷吉-傑克遜只打了2場比賽背部就受傷了，場均只有5分。格里芬只打了15場比賽，場均只有16.3分4.4籃板3.3助攻，命中率只有37.3%。羅斯也輪休了一些比賽，活塞本來該是季後賽球隊，現在打進季後賽也困難了，羅斯不應該在這樣的球隊浪費時間。

開拓者今天戰勝魔術後，取得了三連勝，終於是回到了西部前八。開拓者上個賽季打進了西部決賽，新賽季狀態低迷，逐漸好轉，開拓者這個賽季的傷病很嚴重。努爾基奇是上個賽季受傷的，到現在還沒有回歸，他的防守對開拓者太重要了。另一個內線科林斯左肩脫臼，只打了3場比賽，場均可以得到9分4籃板2助攻。12月7日，胡德在與湖人的比賽中遭遇左跟腱斷裂，賽季報銷。以開拓者的實力，打進季後賽問題不大，其實他們的目標是總冠軍！

傷病讓NBA沒有那麼精彩，特別是西部，如果勇士和開拓者沒有傷病，西部會更加好看！
