狂降65位！快船老大命中率達不到平均水準，養生打法變衝冠隱患？

2019-12-08 0條評論 14次閱讀 0人點贊

賽季至今，快船百回合得到110分，進攻效率位列聯盟第10位。如果萊昂納德的投籃更准一些，快船本可以排到更高的位次。這位球隊的頭號球星，賽季至今的投籃命中率急劇下滑，如果以目前的投籃命中率來看，他正在經歷生涯最鐵的一個賽季。

數據顯示，萊昂納德賽季至今出戰17場比賽，投籃命中率只有43.6%，在所有符合排名標準的144名NBA球員里僅列第106位。上賽季同樣的一份排名，萊昂納德以49.6%的命中率位列第41位。他的命中率排名比上賽季下降65位。在這個賽季前，萊昂納德命中率最低的一季是2017-18賽季，他只打了9場比賽，投籃命中率46.8%，在其他至少出戰58場比賽的賽季里，他的命中率就沒有低過47%。

數據顯示，本賽季至今，他的有效命中率只有47.5%，是生涯第一次低於50%，聯盟里有99名至少200次出手的球員，小卡的有效命中率只排在第84位。

另外，他的真實命中率只有53.5%，也是生涯第一次低於56%。在聯盟里是什麼水平？聯盟平均值是55.9%。換句話說，小卡的真實命中率連聯盟平均水準都比不上，身為上賽季的總決賽MVP，這確實有些尷尬了。

這個賽季如此不準，小卡的問題出在哪兒呢？官網給出數據分析，小卡在三秒區內的投籃命中率下滑，是主要原因。

生涯前7個賽季，小卡在合理衝撞區的投籃命中率是67.6%，2018-19賽季同樣是67.6%，而本賽季至今，55投33中，只有60%。在三秒區的其他位置，生涯前7個賽季是47.7%，2018-19賽季是46.5%，而本賽季至今是有43.6%。數據來看，小卡在三秒區內的把握明顯不如生涯前幾個賽季。

一項比例的巨大變化，值得注意。前7個賽季，小卡在合理衝撞區的出手比例佔三秒區出手的63%，上賽季這一比例微幅下降，但也有62%，而本賽季，卻狂跌到35%。換言之，小卡在籃下的出手比例大幅度減少，他正打得越來越養生？

毋庸置疑，小卡是一個中投能力很強的球員，但再準的中投，也不如上籃和扣籃命中率高。如果小卡明顯減少籃下進攻，只是專註於投射，那麼他的命中率問題可能會變得更嚴重。這一點，也體現在他的罰球率下跌，造犯規能力有所下降。如果小卡不能走出命中率的低谷，當快船面對強隊時，他們很可能會像打雄鹿那樣陷入困境。
