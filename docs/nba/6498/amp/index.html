火箭一大軟肋難彌補！小將四場球只打16分鐘，德帥該多給他機會

天天要聞
2019-11-30

今天是感恩節，NBA沒有比賽，但火箭隊也在積極備戰，接下來的他們要面臨很多挑戰，尤其是隊中多位大將的傷病顯然是有很大影響的。火箭一大軟肋難彌補！小將四場球只打16分鐘，德帥該多給他機會？

上賽季火勇大戰時，球迷們對卡佩拉的表現非常失望，覺得他只要吃不了餅，基本上就沒作用了，休賽期時卡佩拉甚至還陷入交易流言中。但卡佩拉不在，火箭隊還真挺頭疼的，因為光看陣容名單，他們有四名中鋒球員，可巴西中鋒內內本來是打算退役的，本賽季至今還沒上場打過球。老將錢德勒實在是年事已高，狀態上也是今非昔比，每場出場個10分鐘左右也就差不多了，被逼無奈德帥甚至長時間打小個陣容，讓塔克和克拉克臨時客串中鋒，這肯定要吃虧啊。所以這個時候卡佩拉的作用就體現出來了，他的高度、內線終結能力、籃板球和護框，現在的火箭隊沒有人比他更出色了。

但火箭真的沒別的選擇了嗎？那也不是，只是德帥又把小將哈爾騰施泰因給「遺忘」了。這也是德帥用人的老問題了，上賽季哈爾騰施泰因在全明星賽後基本就沒出過場，要麼是在發展聯盟鍛煉，要麼就是坐在替補席上看隊友表演，生生一點機會都不給實在是有點誇張了。過去幾場球哈爾騰施泰因的出場時間也很少，上一場面對熱火出場了10分鐘，可總共也就出手了一次，自然也就別指望他能打出多耀眼的數據了。再之前三場球哈爾騰就更慘了，對獨行俠打了2分鐘，對掘金打了4分鐘，和快船一戰更是沒有得到出場機會，基本都是在比賽失去勝負懸念後才能上場，看來德帥還是不夠信任他啊。

實際上本賽季哈爾騰施泰因已經證明過自己了，面對開拓者他3投全中拿到9分，再之前一場他面對森林狼隊拿到6分，但卻搶下驚人的16個籃板球，效率是非常高的。更別說哈爾騰還有不錯的策應以及投射能力，可以大大豐富球隊的進攻戰術，德帥應該讓他和卡佩拉分擔中鋒位置上的大部分時間才對。接下來就算卡佩拉回歸，狀態可能也需要重新調整，再說你也需要給卡佩拉準備一名靠譜的大個子替補才對啊。我覺得哈爾騰施泰因毫無疑問是最合適的選擇，就是不知道德帥自己啥時候才能想通了。
