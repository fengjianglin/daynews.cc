爭冠局面愈發混亂？聯盟最被低估球員上貨架，他比樂福還受關注

2019-12-21 199條評論 1,681次閱讀 516人點贊

隨著賽季的進行，那些志在衝冠的球隊也已經垂垂欲動，很快就要又要到了一年一度的中期交易，現在每一筆交易和簽約都有可能讓爭冠局面發生改變，美媒體《fadeawayworld》就列出了十大人選，其中德羅贊、樂福以及亞當斯等人均位列其中！

在這份榜單中，樂福德羅贊伊戈達拉等球員等紛紛上榜，可謂星光熠熠，這樣的實力球員無論到哪支球隊，實力都會大幅提升。

第一檔的明星球員有德羅贊和樂福！德羅贊本賽季在馬刺過得並不舒心，球隊也可能本賽季無緣季後賽，德羅贊關鍵時刻疲軟的毛病會是交易的一大障礙，但是他出色的中距離是現在聯盟最稀缺的東西，在季後賽和總決賽的生死戰中，中距離會成為最多的得分手段，德羅贊仍然會受歡迎。

至於樂福更不用說，騎士要交易他已經不是什麼秘密了，掘金開拓者已經表示有意得到樂福，但是他們一定要付出首輪簽的代價才能做到！

第二檔的實力球員有伊戈達拉，亞當斯，加里納利，克勞德，莫里斯等人！

這些雖然不是全明星球員，但是絕對是最佳的總冠軍拼圖。特別是在現在鋒線球員極其吃香的年代，幾位球員幾乎都不愁下家，當然交易難度也很大，比如伊戈達拉亞當斯等人，合同較大其他球隊願不願意付出高昂的籌碼也是巨大的問題，相反的克勞德的交易難度會小很多，伊戈達拉克勞德這樣的球員是現在湖人最渴望也是最迫切需要得到的球員類型，湖人一定會嘗試接觸！

第三檔就是一些拼圖替補球員，這類的代表有蒂格，特里斯坦-湯普森等人，合同較大但是本賽季發揮也是一般，最關鍵的是，這類型的球員在奪冠熱門球隊並不吃香，湯普森防守不錯但是態度是個問題而且合同不小。蒂格本賽季發揮實在慘不忍睹，森林狼需要降低籌碼估計才能交易出去！

不過在所有的潛在交易球員名單上，有一個人卻是可以獨一檔的！那就是鵜鶘的霍樂迪！

鵜鶘方面已經明確表示，他們已經把霍樂迪擺上貨架，他們寧願留下雷迪克也不願意留下霍樂迪。球隊除了狀元郎和鶯哥都可以交易。

為什麼說霍樂迪是獨一檔呢，而且會改變聯盟格局呢？

首先，因為身處鵜鶘，讓霍樂迪的關注度一直較小，但事實上霍樂迪是少有的攻防兼備的球員，他不想樂福那種有著明顯的防守弱點，也不想克勞德湯普森那種進攻手段缺乏，他能突能傳，能組織能進攻，防守端更是曾經入選過最佳防守陣容！是不是很意外，身體瘦小的霍樂迪也曾是最佳防守陣容球員，而且本賽季防守能力依舊出色，只不過鵜鶘整體糟糕的防守一定程度拖了後腿！不敢說從一防到三，但是一二號位防守沒有任何問題。

其次，說霍樂迪聯盟最低估球員不是胡言亂語的！

之前美媒《The Athletic》曾對現役球員發起投票，票選聯盟最被低估的球員，霍勒迪以16.3%的得票率高居第1位，他的實力早就得到了同行認可。

賽季至今，霍樂迪場均拿下19.5分外加5籃板、6.8助攻、1.7搶斷，而且得分手段沒有死穴，三分命中率35.4%場均突破17.4次聯盟第6位，場均突破得分8.6分！能突能投的特質適合任何超巨搭檔，單打每回合拿下1.125分，超過聯盟90%的球員，個人得分能力極強。

當然，霍樂迪的合同不小，5年總價值1.31億美元！下賽季他將有超過2600萬美元，在2021-22賽季還有球員選項，但估計霍勒迪也沒有放棄這份合同的理由！所以接手他的球隊需要做好準備，但是霍樂迪才30歲啊，他還在生涯巔峰！攻防兩端都非常適合在短期內有爭冠意願的球隊。

而在鵜鶘宣布將霍樂迪放上貨架以後，霍樂迪的熱度已經遠超之前的伊戈達拉和樂福等球員，他比伊戈達拉進攻更好，比樂福防守更好，誰會拒絕一個沒有缺點的球員呢？超高的性價比和個人實力已經成為決定本賽季總冠軍走向的風向標！

快船已經蠢蠢欲動，他的進攻比貝弗利要好，防守被路威出色，無私的球風適合倫納德和喬治。如果能得到霍樂迪，快船將擁有三位攻防兼備的球星，那樣他們將更加勢不可擋！

當然， 快船的競爭對手也很多，熱火掘金都希望能得到這位最被低估球員，無論這三支球隊誰得到霍樂迪，都將迅速成為冠軍最大熱門。04年拉希德華萊士加盟活塞，攻守兼備的他成了活塞該賽季奪冠的終極版圖。08年加索爾空降洛杉磯湖人直接殺進總決賽此後兩次衛冕，這次，霍樂迪會複製傳奇嗎？
