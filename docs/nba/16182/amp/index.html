NBA十大最著名的數據刷子：樂福威少上榜，55分先生命中率極低

天天要聞
3 天 ago

出色的數據對於NBA球員來說，那就代表著大把大把的鈔票，各種各樣的榮譽和比較高的聯盟地位，所以打出漂亮的技術統計收集大部分球員每場比賽中奮鬥的目標，不過在球場上，有些球員可以通過出色的數據和個人能力，帶領球隊取得勝利，而有些球員，雖然場均數據十分漂亮，但是基本上和數據無關，這類球員也有一個統稱，那就是「數據刷子」，下面小編就給大家介紹一下，NBA歷史上最著名的十個數據刷子。

1、詹寧斯

作為2009年的十號秀，詹寧斯新秀賽季的風頭甚至蓋過了哈登和庫里這兩位未來的聯盟巨星，特別是在09年11月26號的比賽中，詹寧斯砍下了55分，創造了多項紀錄，成為了NBA歷史上最年輕的50分先生，但是在哈登和庫里都成為巨星的情況下，成名更早的詹寧斯連一份底薪合同都沒有獲得，主要原因就是，詹寧斯不僅是數據刷子，而且效率極低，在詹寧斯的NBA生涯中，他場均命中率最高的一個賽季只有41.9%，命中率超過四成的賽季只有兩個，特別是11-12賽季，詹寧斯場均可以拿到19.2分，但是投籃命中率卻只有41.9%，三分命中率為33.7%，雄鹿隊更是無緣季後賽。

2、蒙塔埃利斯

作為一名二輪秀，埃利斯的表現已經足夠出色，特別是巔峰時期的得分能力，更是達到了准全明星的級別，但是蒙塔埃利斯和全明星球員最大的差距就是，全明星球員能夠帶領球隊取勝，但是埃利斯不管得多少分，基本上都和球隊的戰績沒有多大的關係，他也沒有能力帶領球隊進入到季後賽，而且大部分時間中，埃利斯的眼裡，除了籃球和籃筐之外，在沒有其他的隊友。

3、樂福

效力於森林狼時期的樂福，是聯盟中的兩雙機器之一，拿到過最快進步球員和籃板王的獎項，還3次入選全明星，更是成為新世紀以來，第一個單場砍下30分+30籃板的球員，但是樂福數據和戰績卻是兩個完全相反的對立面，他的數據越出色，球隊的戰績反而越差，特別是他為了搶到籃板，經常會放棄自己防守的球員，而且在森林狼效力的6個賽季中，一次都沒有帶領球隊進入過季後賽。

4、威少

說威少是數據刷子，一定會得到很多喜歡他的球迷反駁，甚至這些球迷會說其他人怎麼不刷出來一個賽季場均三雙看看，的確威少連續三個賽季場均三雙的壯舉很偉大，但是即便如此，也不能改變一個事實，那就是威少在球場上有時候會刻意的去追求某項數據，這也是為什麼大家喜歡說，雷霆隊當年的戰術打法是什麼，主要是看威少缺少了哪項數據。

5、格里芬

作為09年的狀元秀，單看格里芬的數據和球場上的表現，基本上沒有什麼太大的毛病，但是你要是仔細分析一下的話，會發現一個有趣的現象，那就是比賽的勝負大部分情況下，和格里芬真的沒有多大的關係，在快船隊效力時期，格里芬就容易在關鍵時刻出現表現乏力的現象，這點甚至被保羅公開批評過，而被交易到活塞之後，本來希望格里芬和德拉蒙德能組成出色的雙塔，實現戰績的突破，但格里芬加盟之後，活塞隊的戰績和他沒來之前基本上沒有什麼變化。

6、德拉蒙德

如果一名球員場均可以貢獻17.3分15.6個籃板1.7次蓋帽的數據，卻連最佳防守陣容都進不了，這是不是就說明了一個問題，在效力與球隊8個賽季期間，只有兩次帶領球隊進入季後賽，這是不是也說明了一個問題，綜合起來說，德拉蒙德砍下這樣的數據，對於球隊本身的防守和戰績來說，基本上沒有多大的關係。

7、隆多

當年的隆多，也是一個出色的數據刷子，他最為出名的舉動就是，明明面對空籃卻不選擇得分，而是等到隊友來到後助攻隊友扣籃，最為著名的一次就是2012年凱爾特人對陣活塞的比賽中，三節戰罷比賽已經失去了懸念，但是為了完成連續34場助攻上雙的壯舉，作為綠軍主力後衛的隆多，在垃圾時間中仍然呆在場上，直到刷完10助攻，完成助攻上雙後，才心滿意足的下場。

8、斯塔克豪斯

在大多數老球迷的印象中，斯塔克豪斯都是以得分高手的印象存在，不過這位出色的「得分高手」，職業生涯命中率只有40.9%，而到了季後賽之後，他的命中率更是跌到了36.9%，這是一個得分高手應該有的命中率嗎？而且在斯塔克豪斯的巔峰時期，從來沒有帶領過任何一支球隊殺入過季後賽。

9、蘭多夫

在加盟灰熊隊之前，蘭多夫是NBA聯盟中實實在在的數據大刷子，基本上比賽是否能贏，是否可以進入季後賽，這點和他基本上沒有絲毫關係，他上場之後，最大的目的就是打出出色的賽季，以獲取更大的合同，這在當年是公認的，而且絲毫不需考慮自己的進攻效率和命中率，特別是05-06賽季，蘭多夫以48.3%的真實命中率和每百回合99的進攻效率，場均拿到18分。

10、埃爾文海耶斯

在被馬賽克隊選中之後，海耶斯拿到過得分王和籃板王，場均數據最差的一年也有25.2分14.5個籃板，但是球隊卻以半賣半送的情況將他交易到了子彈，而得到的回報只有一個跳投手傑克馬林，為什麼馬賽克隊會做這樣明顯賠本的買賣，因為球場的海耶斯，不僅不按照教練的安排打球，而且非常迷戀自己的數據，甚至有時候會為了一個蓋帽和一個籃板，而遠離自己的防守球員，特別是除了菜鳥賽季外，海耶斯再也有帶領球隊進入過季後賽，不過好在加盟子彈之後，球隊的核心昂塞爾德願意站在海耶斯身後，為他補防，幫他搶籃板，讓海耶斯可以盡情的做他想做的事，這才改變了聯盟對海耶斯數據刷子的看法。
