DPOY！戴維斯數據平平卻統治攻防兩端，詹姆斯樂開花，第四冠有望

2019-12-05 0條評論 16次閱讀 0人點贊

121-96，湖人大勝爵士取得2連勝。上半場湖人是反客為主，65-47吊打主場的爵士，完全看不出這是一支打完背靠背，並且球隊多人生病的球隊。全隊是三軍用命，活力滿滿，外線有詹姆斯和格林和一群替補群開炮，內線有霍華德，麥基和戴維斯，錘爆爵士的內線，內外開花令湖人一度取得18分的領先。而爵士則打得不像爵士，全隊死氣沉沉，沒有了昔日犀利的防守，被湖人的反擊快攻是打得找不著北。半場湖人65-47爵士。詹眉兩人合砍31分，戴維斯15分，詹姆斯16分。

下半場米切爾連中3球，戈貝爾雙手暴扣也有，傑夫一條龍殺內線上籃追到11分。戴維斯籃下背身打進，詹姆斯再上一個籃，湖人穩住局面。波普得分繼續，2個空藍一模一樣的劇情，分差擴大到19分。博格達3分繼續救命，戈貝爾2+1，米切爾擠開防守打進，爵士追到12分。困難時刻看戴維斯，翻身跳投，突破造罰籃，雙手暴扣，防守端還送上大火鍋，統治攻防兩端，撲滅了爵士反撲的勢頭，實在太無解了。三節結束湖人領先19分，爵士白忙活了，主場的球迷也噓聲四起，非常不滿爵士的表現。

第四節詹姆斯帶隊打了4分鐘左右就下場休息了，20分12助的數據打卡下班。隆多連得7分拉開到22分，比分懸念就是終結。而戴維斯第四節都沒有上，26分鐘拿到26分6板3帽的數據，雖然數據不是很爆炸，但是看了比賽的球迷都知道，戴維斯的統治力是無法用數據來表示的，他本賽季絕對有望衝擊最佳防守人。而詹姆斯自然是非常高興的，這個搭檔太給力了，第四冠還會遠么？
