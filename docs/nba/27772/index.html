字母哥“超常發揮”！這就是你們說的無敵狀態？可他還有進步空間

2019-12-20 273條評論 4,518次閱讀 3506人點贊

雄鹿隊坐鎮主場迎來湖人隊挑戰，這一場常規賽比賽引發了全聯盟的關注目光，被冠以「總決賽預演」足以說明一切。有些巧合的是，兩支球隊在碰面之前都終結了各自的連勝紀錄——湖人隊客場14連勝被步行者隊終結，而雄鹿隊的18連勝戰績被獨行俠隊強硬地在密爾沃基阻止。

之於雄鹿隊來說，18連勝戰績被終結固然可惜，然而常規賽的漫長賽程，一場比賽勝負並不能決定整個賽季的命運。但是對陣湖人隊的比賽卻完全不同，這將在一定程度上檢驗雄鹿隊的成色——既然把眼光放在了總決賽舞台，就該讓自己在常規賽變得更加強大，而湖人隊就是理想的常規賽對手。

字母哥更是如此，他是MVP的熱門候選者，就該在「詹眉組合」面前展現出自己強硬的一面。

雙方兩大王牌

比賽開始之後，雙方都擺出了各自對待比賽的強硬態度，防守端的激烈絞殺讓雙方在場上陷入僵持。詹姆斯在籃下的進攻受限，而字母哥也在嚴防之下出現上籃偏出的情況，這是一場高強度的對抗，屬於真正男人之間的對話。這樣的一場比賽誰都不能退縮，狹路相逢唯有勇者能夠勝出。

字母哥的首次得分是在三分線外，一記三分球打破了自己沉寂的手感，像是闖進了秘密的私人花園。而很快他又在籃下尋覓到進攻機會，搶到進攻籃板輕鬆將球放進，不過這也是他在首節比賽中僅僅得到的5分。首節比賽4投2中，字母哥為雄鹿隊貢獻5分2個籃板以及4次助攻，還給了詹姆斯一記封蓋。

字母哥三兄弟

第二節比賽開始後的輪轉陣容階段，字母哥重新登場便在籃下強硬地打擊湖人隊防守，雄鹿隊也因此將領先優勢擴大。詹姆斯連砍5分，字母哥一記三分球給出了回應，他在進攻端給球隊以巨大支撐。第二節比賽字母哥打了9分鐘，7投4中貢獻11分6個籃板以及2次助攻。

由於雄鹿隊上半場比賽領先較多，第三節比賽也沒能將分差迫近太多，字母哥因此獲得了半節多的休息時間。儘管第三節只打了5分鐘比賽，字母哥卻兩次命中三分球，第三節比賽再得到7分。字母哥得分比較均衡，每一節比賽都能夠在進攻端做出貢獻，雄鹿隊始終擁有著穩定得分點。

字母哥上籃得分

末節比賽打了僅僅三分鐘時間，字母哥在進攻端砍下7分，打斷了湖人隊第三節比賽末段的追分勢頭。值得一提的是，字母哥再一次命中三分球，這讓湖人隊的防守很受打擊。最終雄鹿隊主場擊敗湖人隊，字母哥出戰32分鐘的時間，19投11中，貢獻34分11個籃板以及7次助攻。

本場比賽字母哥在三分線外「超常發揮」，三分球8投5中，外線火力相當兇猛，單場命中5記三分球也創造了字母哥個人比賽紀錄。之前總有人說字母哥若是能投三分球會更加恐怖，本賽季字母哥外線投籃就有著不小的提升，場均5.2次三分球出手，命中率達到33.8%。

雖然字母哥的外線投籃能力一般，但是字母哥努力的結果卻顯而易見。即便是作為聯盟中MVP級別的球員，字母哥也並沒有自滿，他還在提升著自己的表現。始終努力提升自己的能力，永遠在學習中進步，正是聯盟中頂級球員能夠保持長盛不衰的統治力的重要原因。
