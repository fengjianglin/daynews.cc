雷霆有季候賽的實力靠隊內球員的能力，現在他們來了準備好籌碼吧

2019-12-11 122條評論 1,089次閱讀 69人點贊

今天雷霆將加里納利，保羅和亞當斯正式擺上了貨架。現在西部第七的雷霆可以說真的是有著進季後賽的實力，這個一點和雷哥之前的預測相同，雷霆就是要打出一些成績，然後讓其他球隊看看，我們這些球員這麼能打，有想要的，可以拿出籌碼，咱們慢慢商量。

前兩天傳出樂福被擺上貨架的消息，雷霆完全可以看菜下飯碟，樂福能交易出去，我加里納利差那了？樂福場均15.6分10.5個籃板2.6次助攻，場均可以命中2.1記三分，加里納利場均18.3分5.7個籃板2.1次助攻，場均能夠投中2.7記三分。數據相同的情況下，加里納利比樂福便宜600萬左右，並且交易索要籌碼肯定是比樂福低，還有最重要的一點就是，加里納利是到期合同，這個合同一旦到期，可以情理薪金空間。不妨礙下賽季補強。

由此可見雷霆現在對於加里納利的交易一點都不慌，就在家裡喝著茶水等著生意上門， 雷哥認為加里納利的交易前景是要比樂福樂觀不少的。不知道現在開拓者是什麼心情，是不是已經安奈不住要給雷霆打電話了，畢竟後場雙槍加安東尼和加里納利的前場，大白邊坐鎮中鋒，這個配置在西部還是非常有競爭力的，想要這套陣容只需要復出一些選秀權和年輕球員，為什麼不能博一下呢？

保羅有交易市場嗎？肯定是有交易市場的，但是能吃下保羅的球隊卻沒有幾個，但是雷霆的意思也表達的挺明確，為了得到年輕球員和選秀權，雷霆可以接受一些垃圾合同，畢竟在雷霆老闆本內特看來，能用錢解決的事都不是事。不然也不會一直叫著高昂的奢侈稅。熱火現在高居東部第三，但是真論其球星成色，也就是巴特勒了，所以熱火要是想著有所作為，那完全可以嘗試一下，假如真是有這場交易，那麼雷霆應該並不樂意跟熱火交易，畢竟手握熱火未來的首輪選秀權，熱火成績好，選秀排名低。所以這樣的交易，熱火可能需要更多一點的籌碼。你們怎麼認為呢？韋斯特可能會出現在交易當中。

除了熱火，雄鹿也是有可能得到保羅的，血布不錯，但是相比較保羅帶給球隊的，雄鹿是非常需要的，年輕的字母哥成長了不少，但是要是在加上老炮的經驗，這樣的組合想想也是可怕，當然薪金也是非常的美麗，但是結合上賽季，雄鹿這賽季很可能登頂奪得總冠軍，這種情況下，豪賭一把也不是不行。萬一成了呢。還有森林狼有交易保羅的可能性嗎？

最後再說到亞當斯，想要了解更多的讀者可以去看雷哥之前的文章，已經分析過鴨蛋的交易前景，結合這個賽季的表現，可以說更加沒有市場了，之前分析過的凱爾特人和國王依然有可能，但是雷霆這比交易完全可以說是賤賣了，比如一個杰倫布朗，再比如國王的戴德蒙加博格丹諾維奇，但是我認為國王並不一定給，所以亞當斯可以說是負面的操作。

最後還有就是施羅德，現在來說也是交易不交易兩可，我接受各隊隊施羅德的報價，交易不交易就看我普萊斯蒂怎麼決定了。

加里納利為上保羅為中亞當斯為下，雷霆假如能交易出去保羅就已經成功了，因為亞當斯要是交易不出去就留著自己用。畢竟交易保羅出去以後，可以騰出一定的薪金空間。

這就是雷霆這三位球員的交易前景，能力一般水平有限但是誰也擋不住熱愛，歡迎大家留言討論，喜歡的可以點點關注。
