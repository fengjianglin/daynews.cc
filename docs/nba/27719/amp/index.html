快船主帥因輪休杠上詹皇，名人堂球員力挺詹姆斯

天天要聞
2019-12-21

詹姆斯日前受訪被問及負荷管理時，他坦言健康就會出賽，並認為輪休對他而言並不合理。採用負荷管理理論輪休方針的快船主帥里弗斯也反酸詹皇的評論，他認為兩支球隊的管理方針大不相同，湖人應該就是小皇帝說了算，此話一出掀起軒然大波。

「出勤率」是考核NBA超級巨星的一項標準，這一點大家也都有所了解。球星技術無論多好，如果他的身體始終無法保持健康，那也是不行的。在所有人的印象中，彷彿每個賽季都能打滿82場比賽的詹姆斯就是這個聯盟「第一鐵人」，而勒布朗的出勤率確實很高，達到了95.1%左右。上個賽季的傷也導致出勤率有所下降。

​

當然，不少球迷認為倫納德輪休是也因為傷病所致，卡哇伊是股四頭肌肌腱炎，是由於長期的跑跳運動從而周而復始地給連接股四頭肌肌肉和膝蓋的股四頭肌肌腱不斷增加負擔，反覆勞損而造成的。隨之而引起疼痛，傷病甚至撕裂。這是一種肌腱的傷病。所以，倫納德的傷病很可能並不是像大家原本所想的那樣，由於他長時間超負荷訓練引發的勞損弄成的。而是由於曾經的某次股四頭肌挫傷沒有治療得當。其團隊認為，這次得挫傷發生於2016年3月，與熱火對陣時被溫斯洛撞傷。而在2017年3月，卡哇伊再次遭遇股四頭肌挫傷。傷病恢復需要一個過程，一切都要循序漸進

此前聯盟官方發言人對快船前鋒科懷-倫納德輪休一事發表了聲明。「在聯盟的輪休政策規定下，倫納德並不是一個健康的球員，他在快船的傷病列表上被列為膝蓋傷勢治療。聯盟辦公室在經過與體育治療主管的商討後，選擇相信快船隊醫的判斷，那就是倫納德目前的身體狀況不宜於出戰背靠背的比賽。」

而且今年的快船的目標是衝擊總冠軍，雖然卡哇伊不經常打比賽，但是快船依舊保持著西部第二的位置，他們有著深厚的替補陣容，路威，喬治，哈萊爾都是砍分好手。要想爭取到今年的總冠軍，當家球員的身體狀況就必須要保持好。

就昨天的一檔綜藝節目，名人堂球星佩頓上節目談及此事時，表態力挺勒布朗，痛批里弗斯沒有說實話，認為負荷管理應該是倫納德主動要求的，這是他會來快船的原因之一，因為球隊允許他這麼做。

佩頓又對詹姆斯表示：「現在說到勒布朗，我向你致以全部的讚美，勒布朗，因為你教給這些年輕人比賽的意義。你都35歲了，而你說你不知道自己在nba中還剩下多少場比賽，而且人們前往現場看你打球，因為你有著超高的人氣。這麼多的孩子去看你打籃球，你能讓他們失望嗎？當你的年薪達到2500萬到3000多萬美元時，你有義務在健康的情況下打球，那是你該做的事。

我們很多群眾朝九晚五地工作，工資卻拿的很少，他們也一直在幹活；有人一天工作9到10小時，而你卻不能工作兩個半小時？兩個半小時都不行？你都不訓練的，說什麼呢，你連訓練都不參加嗎？所以你們都在說些什麼？別再拿負荷訓練說事了，走向籃球場做你該做的事，因為你拿了3000萬年薪，不要說那些屁話了！勒布朗，你是唯一，繼續加油！」

那麼大家覺得誰的言論對呢？是該保持身體健康狀況重要，還是為了不讓觀眾失望而上場更好呢？歡迎留言討論。
