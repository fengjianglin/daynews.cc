15日馬刺VS太陽，米爾斯加時0.4秒絕殺能否成為下一個“馬努”？

天天要聞
2019-12-16

聖安東尼奧馬刺對陣菲尼克斯太陽隊，兩隊歷經一個加時，馬刺隊三分險勝太陽隊。本場比賽，太陽隊當家球星布克休戰，比賽打得相當激烈，第一節和第二節兩隊各有亮眼表現，半場馬刺44分比48分落後4分進入中場休息，後半場，兩隊是針尖對麥芒一直打得非常焦灼，第四節還剩2分10秒時，馬刺隊米爾斯弧頂一記關鍵三分將分差拉到4分，比賽剩1分15秒盧比奧關鍵三分失准，幸虧籃下的貝恩斯補籃命中，兩隊打成104平，16.7秒阿爾德里奇籃下關鍵補籃打成2+1，可惜罰籃不進領先太陽3分，給了太陽隊烏布雷2.6秒命中三分把比賽拖進加時的機會。加時賽以穆雷的跳投拉開序幕，場終前1分鐘兩隊又一次打成116平。隨後盧比奧一記45度關鍵三分領先2分。喜劇性的是接下來又以一次致命的接球失誤白白給了馬刺一次球權，德羅贊拋投兩分打進將比分又一次打平，還有一次加罰，罰籃不進，雙方拼搶籃板，最終馬刺球權，米爾斯實在不想加班，「馬努」附體一般穩穩命中一記中投，0.4秒絕殺直接帶走比賽。

太陽在少了布克的情況下，5人得分上雙，最終還是沒有拿下最後的勝利，盧比奧全場20投10中拿下25分3板13助，助攻數破4000個大關。卡明斯基22分7板4助，薩里奇19分17板4助。馬刺隊阿爾德里奇出場37分鐘18投7中得到18分7板1助，德羅贊出場33分鐘15投7中同樣得到18分，另外有6板3助，穆雷高效的同樣拿到18分7板5助，替補出場的米爾斯加時0.4秒絕殺太陽，全場出戰31分鐘，23投10中，拿到全場最高的26分，另有5板1助入賬。

你認為如果替補出場的米爾斯持續高效的表現，有希望成為下一個第六人「馬努」嗎？
