東契奇22+7波神19分 雙子星這次躺贏！一人9中9成關鍵先生

2019-12-05 0條評論 15次閱讀 0人點贊

東契奇狀態火爆，他帶著獨行俠隊在本賽季打出了不俗的表現，擊敗鵜鶘隊之後，獨行俠隊14勝6負位列西部第四位。東契奇拿到了自己的首個月最佳獎項，這並沒有讓他有任何的自滿情緒，對陣鵜鶘隊的比賽，僅僅三節就砍下33分18個籃板以及5次助攻的數據。

如此表現自然得到全聯盟的關注，幾乎每場比賽之前，他都會成為對手重點談論的球員。對陣森林狼隊的比賽之前，傑夫-蒂格就表示，「他（東契奇）很可能是目前NBA最好的球員，因此你得試著讓很多的人去防守他。我不是很確定這是否奏效，看起來現在沒有人能夠阻止他了，因此希望我們能夠找到努力限制他的方案。」

森林狼隊新賽季開局時相當勁猛，不過此後表現逐漸滑落，目前10勝9負排在西部第七名的位置上。森林狼隊的狀態起伏不定，之前曾五場比賽僅僅取得1勝4負成績，不過最近兩戰都取勝，似乎重新回到了正確的軌道當中。對陣前來挑戰的獨行俠隊，森林狼隊能夠延續勝勢嗎？

波爾津吉斯

獨行俠隊掌控首次進攻球權，鮑威爾首次嘗試外線投籃就命中，考文頓同樣在外線穿針還以顏色。東契奇、鮑威爾繼續在進攻端殺傷，獨行俠隊打出11比3領先的開局，開局之後手感火熱。維金斯三分命中，小蒂姆-哈達威也在外線飆中，森林狼隊仍然處於落後的位置，但是球隊的進攻卻逐漸好了起來。

卡爾弗連續得分，唐斯也在進攻端蘇醒，格拉漢姆三分得手之後，森林狼隊追至僅僅落後1分分差。賽斯-庫里進攻得手，內皮爾造犯規兩罰兩中，森林狼隊緊緊咬住比分。最後一分半鐘，科勒貝爾進攻端打得比較出色，獨行俠隊打出6比2的比分收尾。首節比賽結束，獨行俠隊30比25領先森林狼隊。

德懷特-鮑威爾

第二節比賽開始，卡爾弗造犯規兩罰兩中，森林狼隊隨後卻遭遇到一波流打擊。獨行俠隊的輪轉陣容手感火熱，打出一波8比0的比分，不經意間將分差拉開到了11分。維金斯兩罰兩中，森林狼隊止住血，但是獨行俠隊的進攻感覺仍然不差，鮑威爾三分命中，獨行俠隊還是領先著12分。

唐斯發威在內外線連續取分，森林狼隊打出一波打出7比0，很快將分差縮小。喬丹-貝爾兩罰兩中，森林狼隊還落後5分，東契奇打成2+1，考文頓則命中三分球，比賽打得相當膠著。考文頓的手感相當火爆，將森林狼隊從泥潭中拉了出來，蒂格最後時刻兩罰兩中，森林狼隊56比53領先結束上半場比賽。

盧卡-東契奇

森林狼隊第三節比賽開局手感火熱，維金斯、卡爾弗連中三分球，打出一波8比2將分差擴大。波爾津吉斯命中三分球，但是唐斯的進攻難以阻擋，森林狼隊始終保持著領先局面。不過獨行俠隊連得6分，還是在努力將分差縮小，領先5分的森林狼隊趕緊叫了暫停進行調整。

鮑威爾連續進攻得手拿到4分，分差被縮小到只剩下1分，唐斯命中三分球成功止血。森林狼隊繼續在進攻端發力，打出一波9比2的攻擊波，分差被繼續地拉大。獨行俠隊又是鮑威爾站了出來，他幫助獨行俠隊不斷地追分，芬尼-史密斯投籃得手，第三節比賽結束，獨行俠隊80比84落後。

賽斯-庫里

芬尼-史密斯末節比賽率先得分，維金斯造犯規兩罰兩中，德隆-賴特、布朗森連續得分，獨行俠隊追平了比分。獨行俠隊的輪轉陣容表現不錯，不斷在外線發炮命中投籃，獨行俠隊反超森林狼隊4分。關鍵時刻，獨行俠隊替補陣容的三分雨發揮作用，比賽也進入到了決戰時刻。

場上出現意外情況，鮑威爾在防守時受傷回到更衣室。東契奇在比賽最後時刻，終於命中外線投籃，稍後他又在外線造犯規三罰三中，獨行俠隊領先8分。獨行俠隊佔據了場上主動，森林狼隊努力地追趕奮力一搏，最終無果。全場比賽結束，獨行俠隊121比114擊敗森林狼隊。

獨行俠隊前三節比賽表現都不夠好，很重要的原因在於雙子星東契奇和波爾津吉斯進攻狀態不佳。鮑威爾成為支撐球隊進攻的重要球員，9投9中彈無虛發砍下24分，獨行俠隊沒有被對手將分差拉得更大。末節決戰的時刻，獨行俠隊的外線投籃回暖，逆轉在客場取得一場勝利。
