威少變了！火箭終結3連敗，豪斯末節起衝突，威少上演暖心舉動

2019-11-28 0條評論 18次閱讀 0人點贊

今年夏天，休斯頓火箭隊對巴特勒展開了猛烈地追求，奈何落花有意流水無情，最終巴特勒選擇了加盟邁阿密熱火隊自己當老大。不過火箭隊今年夏天的引援工作並非一無成果，他們通過交易送走了老將保羅，迎來了更年輕、更具有衝擊力的球員威斯布魯克，他和哈登組成了雙MVP組合。今天熱火和火箭隊迎來了兩隊新賽季的首場交鋒，最終火箭隊117-108擊敗了對手。

這場比賽巴特勒因傷缺席比賽，火箭隊主力中鋒卡佩拉也因傷缺陣，比賽一開始，兩隊打得難捨難分，直到比賽進入第二節，火箭隊憑藉著當家球星哈登的瘋狂表現單節得到了39分，而熱火失去了巴特勒在進攻端出現了很大的問題，第二節僅僅得到了19分，火箭隊憑藉著單節20分的優勢領先進入了下半場。進入下半場，第三節兩隊進入僵持階段你來我往，分差沒有進一步擴大，而第四節熱火突然醒悟，展開了追分大戰，不過挖下的坑太大了，火箭隊憑藉著哈登的幾個神仙球，依舊笑到了最後。

全場比賽，火箭方面得分王依舊是哈登，哈登砍下34分6籃板5助攻，威少27分9籃板7助攻，豪斯21分4籃板2助攻，塔克7分7籃板，克拉克12分8籃板。熱火方面得分都比較均勻，全隊6人得分上雙，阿德巴約15分10籃板8助攻，羅賓遜13分、德拉季奇13分、希羅22分、奧利尼克18分、溫斯洛10分。

威少變了！火箭終結3連敗，豪斯末節起衝突，威少上演暖心舉動。值得一提的是，比賽進入到了第四節，熱火正在上演追分的好戲，而豪斯在一次突破的過程中遭遇到了對手的衝撞跌到在地，豪斯一怒之下拿起皮球砸向對手，兩人眼看就要拳腳相加，關鍵時刻威少挺身而出當在了兩人中間，把豪斯拉開，之後和豪斯經過了一番交談，終於讓他冷靜了下來，避免造成了更大的衝突。很顯然，來到火箭隊之後威少變了，變得更成熟了，他不但不再那麼容易上頭了，還能想方設法讓隊友冷靜下來。
