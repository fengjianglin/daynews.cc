哈登招保鏢為了打架，詹姆斯的保鏢是安東尼小弟，安東尼保鏢無數

2019-11-29 0條評論 18次閱讀 0人點贊

大家都知道，人怕出名豬怕壯，越是有錢的人就越注重自己的安全。無論是演員，歌手，老闆，球星，只要身價達到了一定程度或知名度，那他們就會請一些專業的保鏢來保護自己。看到哪些商業圈和娛樂圈的名人，每次出行身邊都跟著一大批黑衣人，那便就是他們請的專業保鏢。在NBA中，別看那些球星都長得高大魁梧一個，看起來沒人能奈何他們似的，但他們在出行的時候，也是有保鏢保護的，那麼NBA誰的保鏢實力最強呢？

庫里保鏢 庫里保鏢有一個綽號」火箭「，是因為他保鏢的彈跳能力非常出色，可能金州拉文都比不過，庫里保鏢真實的名字是叫沃克，關於沃克這名保鏢有一個有趣的故事，曾經他為了將一個罪犯捉拿歸案，追了他整整四條街區，後來沃克做了庫里的保鏢，並成為了勇士的安全顧問，平日里和庫里形影不離。

哈登保鏢 哈登因為當年阿泰在球場上把他打成腦震蕩，他對阿泰非常不滿，但想去找阿泰理論，但因為阿泰身體比自己要強健許多，怕打起來又吃虧。哈登就給自己專門配了一個保鏢，就是為了以後和別人打架時派上用場。雖然說從外表上看哈登的保鏢一身肥肉，但是據說是特種兵出身。

詹姆斯保鏢 詹姆斯在一個賽季的總決賽前後，他的家裡經常遭到他人的破壞，因此讓在打總決賽的詹姆斯非常不安，不過當時贊詹姆斯的好友安東尼說：你不要擔心，你的事我來處理，你只要專心比賽就好了。然後安東尼就吩咐他手下小弟去追查，不到一天的時間，那個破壞詹姆斯家裡的人就被找到了。詹姆斯表示感謝，就要安東尼小弟做了自己的保鏢。

科比保鏢 科比的這位保鏢從外表上看似乎是一位大叔，給人的感覺沒有任何殺傷力，不過可千萬不要小瞧這人，這位保鏢和哈登的保鏢一樣，都是特種兵出身，據說每天的薪水達到一萬美元。

安東尼自帶體系 紐約亂不亂，瓜哥說了算，安東尼在美國有著很強大的黑幫背景，在籃球場上，只要是被安東尼眼神一蹬的人，都會立馬變得有點慫，安東尼保鏢無數，平時安東尼出門都不用帶保鏢，打一個電話好幾車的小弟馬上就開車到門口，所以安東尼自帶保鏢體系。
