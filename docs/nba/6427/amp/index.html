17分大逆轉！國王逆轉戰勝掘金戰績追平太陽，近13戰8勝5負

天天要聞
2019-12-01

北京時間12月1日，NBA常規賽的比賽繼續進行，國王在主場迎來了掘金隊的挑戰，最近的掘金隊狀態十分的不錯，球隊正在經歷一波6連勝，目前掘金取得了13勝3負的戰績，今天的比賽掘金只要拿下國王，他們就將會創造一個新的記錄，隊史最佳開局，而國王狀態這邊狀態稍顯不佳，連續的兩個客場比賽里，國王遭遇到了兩連敗，他們渴望主場用一場勝利來止住頹勢。

首節的比賽開始後，哈里斯造犯規罰球拿到了本場比賽首分，霍姆斯上籃回應，兩隊一度打成了6：6平，但是此後國王隊進攻端陷入停滯，哈里斯三分命中，巴頓連得4分， 哈里斯籃下強打打成2+1，憑藉著這波小高潮掘金取得了11分的領先優勢，8分鐘的時間哈里斯砍下了15分，首節比賽打完掘金領先國王10分，第二節的比賽掘金延續著自己很不錯的手感，分差一度被擴大到了17分，半場比賽打完掘金隊領先17分。

下半場的比賽開始後，約瑟夫一記三分拉開了國王隊反擊的序幕，別利察打成2+1，希爾德追身三分也有，巴恩斯連續得分，國王隊單節轟出了23：15的小高潮，三節比賽打完國王隊將分差縮小到了8分，末節比賽掘金隊再次將分差擴大到了14分，本以為比賽沒有什麼懸念了，但是在比賽最後時刻希爾德連中兩記三分，22.9秒希爾德上籃幫助國王隊追平比分，此後兩隊都沒有得分，比賽進入加時，加時賽兩隊打到最後一攻才分出勝負，9.5秒鐘的時候國王隊領先3分，掘金有球權，巴恩斯最後一防立下大功，穆雷突底線，巴恩斯的跟防非常到位，到籃下又有協防過來，穆雷想再分球的時候已經沒有時間了，最終全場比賽結束國王100-97險勝掘金。掘金輸球的很大原因在約基奇這邊，全場比賽不在狀態，特別是在第四節5投0中並且接連失誤導致了球隊遭遇逆轉，全場比賽，約基奇11投僅3中得到7分10板6助1帽和4次失誤。

今天的國王全民皆兵，球隊有五人得分上雙，巴恩斯30分、希爾德21分、霍姆斯13分、別利察10分、博格達諾維奇10分、在輸掉這場比賽後，掘金無緣自己的隊史最佳開局，而國王則是拿下了一場勝利，他們用一場超級大逆轉來結束了自己的連敗，在贏下這場比賽後國王隊的戰績來到了8勝10負，現在國王的戰績和西部第八太陽相同，在10月里國王狀態十分的低迷，開局的五場比賽全部輸掉， 沃頓一度被罵的十分的慘，11月份的國王也是漸入佳境，13場比賽沃頓帶領國王取得了8勝5負的戰績。
