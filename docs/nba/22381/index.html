究竟是不是非賣品？鵜鶘的13連敗，假日哥的3個去向

2019-12-19 190條評論 1,172次閱讀 134人點贊

在本賽季開始之前，雖然鵜鶘送走了安東尼戴維斯，但不會有人認為鵜鶘會成為聯盟最差的球隊之一，隊內交易來了一批年輕人，甚至還包括人氣爆棚的狀元威廉森，加上交易來了神射手雷迪克，而且主教練金特里又是非常擅長進攻的教練，因此大部分專家都認為這是一支充滿年輕活力的球隊，打起快攻來會非常具有觀賞性。

但隨著賽季的進行，鵜鶘的狀元威廉森因為受傷常規賽一場沒打，球隊戰績卻一路下滑，本以為濃眉走後，球隊新領袖假日哥霍樂迪會帶領鵜鶘隊打的不錯，但13連敗後6勝21負的戰績實在太慘，隨著交易日即將開啟，曾經的非賣品霍樂迪可能被鵜鶘擺上貨架。

據美國媒體爆料，鵜鶘的非賣品變成了威廉森和英格拉姆，同時也不願意交易雷迪克，但是假日哥霍樂迪可能被交易。本賽季作為戰術核心地位的霍樂迪場均能得到19.5分5籃板6.8助攻這樣不錯的數據，但是並非核心的鶯歌拉姆卻在本賽季爆發了，場均24.9分7籃板3.6助攻，幾乎走向了頂薪的陣營，而上賽季幾乎沒有人看好這位天賦酷似杜蘭特的球員。雖然假日哥打的不錯，但相比於大衛格里芬的MVP級別還有一段距離。

朱霍樂迪的身體非常強壯，雖然投射不是頂級，但是防守、籃下對抗能力都非常強。如果鵜鶘要交易他，有三支球隊很適合他，第一，太陽隊，休賽期簽下了盧比奧，太陽隊的進攻似乎一下被盤活了，德文布克也更能專註進攻，太陽本賽季戰績位於西部第九，有機會進入季後賽，霍樂迪會是很好的選擇。

第二，熱火隊，戰績19勝8負，東部第四，吉米巴特勒來後球隊已經發生改變，如果霍樂迪能來，球隊更會發生質變，甚至成為奪冠球隊，但是熱火隊會願意交易一到二個充滿潛力的新秀嗎?

第三，森林狼，擁有兩個狀元的他們本賽季初打的確實不錯，維金斯也越打越好，但是這支球隊似乎還是看不到未來，如果假日哥能夠加入，森林狼是有衝擊季後賽的能力的。
