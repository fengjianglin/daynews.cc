又贏了！老鷹僅差5分，霍華德創下里程碑

2019-12-17 2,817條評論 4,046次閱讀 1517人點贊

12月16日，此前已經贏了6次的湖人再次上場，場面一度非常刺激。總的來說雙方的表現非常穩定，不過在三分球方面湖人表現不是很好，不過接下來狀態就迅速回歸，James一人之力得到23分，成功領先老鷹7分。湖人隊狀態愈來愈好，第三節的時候一度領先對手10分。

然而對方在關鍵時刻連續發力，將比分差縮減到4分。在最後一刻，雷迪什不知為什麼沒有投中，最後湖人以5分之差擊敗了對方。這次的比賽對霍華德是非常關鍵的，他在這一戰中創下了新的里程碑。其實在此之前湖人隊已經是6連勝了，以23:3的勝負比穩居西部俱樂部的首名，而老鷹就很慘了，已經在東部墊底，甚至還不如尼克斯。

對於這樣的結果，其實很多觀眾都認為毫無懸念，早就應該想到了。畢竟這兩個球隊從配置和球員實力上來說都相差很大，湖人隊的戰績一直非常高，高居西部第一，然而老鷹隊在今年已經成了東部的倒數。即便如此，這次的賽場上湖人隊贏得不是非常順利，比賽本身並沒有因為兩支球隊的實力懸殊而平穩。或許是湖人隊太過自傲，在上半場外線投籃表現特別糟糕，而且全隊隊員失誤過多，導致輸給了老鷹隊不少分值。

現在比分已經非常明了了，從賽後數據統計表上來看，可以知道湖人隊在上半場雖然表現不佳，但是籃板球上的表現還是非常好的，至少比老鷹隊好得多。此次湖人隊一共搶下了56個籃板，老鷹只有42個，在禁區得分方面老鷹比湖人少20分。詹姆斯的三分球是上半場最大的亮點，除此之外還有胯下妙傳助攻以及霍華德的暴扣。上半場比賽過程還是讓人看得很揪心的，湖人隊的後衛上籃實險些遭到隊友的「攻擊」。

當然這也與湖人隊的整體狀態有關，上半場比賽中湖人隊沒有將比分徹底拉開，而且外線投籃表現非常糟糕，上半場的三分球一共投了14次，然而得分的只有兩次，並且這兩個三分球都是詹姆斯投的。

第二節比賽中，湖人隊的狀態有所回歸，之後和老鷹隊就一直處於膠著的狀態。第三節的比賽非常有懸念，在最後老鷹隊竟然打出了一波小高潮，直接把比分拉近。在決戰的那一刻，雙方的分差保持在一個非常微小的差距狀態。最後時刻老鷹隊的雷什掉鏈子，關鍵的三分球沒有投中，所以湖人拿下了這次比賽的勝利。不知道大家對於這次的比賽有什麼看法?你覺得結果如你所料嗎？
