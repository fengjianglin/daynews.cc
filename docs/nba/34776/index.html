20年前艾弗森叱吒風雲加冕MVP，當年票選前四的球星還有誰？

2019-12-24 92條評論 1,743次閱讀 379人點贊

20年的時間對於NBA來說是一個漫長的過程，20年前，大多數的球隊還是把陣地進攻作為得分最重要的方式，每個球隊都必須配備攻擊力極強的內線球員來撕裂對手的銅牆鐵壁。20年後，NBA聯賽的風格和20年前大相徑庭，內線球員淪為配角，控衛和鋒線球員掌控著大多數的球權，三分球才是取勝的最大利器。那麼20年前，哪些球員能夠代表NBA的最高戰力呢？本篇文章筆者就帶大家回顧00-01賽季MVP票選前四的球員。

MVP得主：阿倫-艾弗森

艾弗森是96黃金一代的代表人物，桀驁不馴和孤寂高冷是艾弗森的代名詞。00-01賽季，效力NBA聯盟第五年的艾弗森開始步入生涯巔峰，他出戰了71場常規賽，場均貢獻31.4分4.5籃板5.5助攻和2.8搶斷，加冕那個賽季的得分王和搶斷王，還入選了最佳陣容一陣。

在艾弗森的帶領下，76人隊取得了56勝26負的東部最好戰績，那是76人隊時隔12個賽季，再次取得單賽季50+的勝場。憑藉個人數據和球隊戰績的雙重表現，艾弗森力壓鄧肯和奧尼爾等內線長人，成為喬丹之後第一個奪得MVP的後衛球員。

MVP票選第二：蒂姆-鄧肯

作為馬刺隊隊史最偉大的球星，鄧肯為馬刺隊贏得了無數的榮譽，自己也被球迷們稱之為「歷史第一大前鋒」。在大學磨礪了四年的鄧肯出道即巔峰，生涯第二個賽季就帶隊奪得馬刺隊隊史首冠。00-01賽季，鄧肯打滿82場常規賽，場均貢獻22.2分12.2籃板2.3蓋帽，入選雙一陣。

任何一支球隊都想要鄧肯這樣既能攻防一體，又能無條件服從教練安排的球員作為建隊基石。00-01賽季，馬刺隊憑藉鄧肯的高效發揮，拿下58勝24負的聯盟最好戰績。

MVP票選第三：沙奎爾-奧尼爾

奧尼爾是NBA歷史上統治力最強的中鋒球員，他在21世紀初統治了NBA聯盟，帶領湖人隊拿下三連冠。雖然奧尼爾經常在常規賽划水，但是他的數據依舊恐怖。00-01賽季，奧尼爾出戰了74場常規賽，場均拿下28.7分12.7籃板2.8蓋帽，入選了那個賽季的一陣和二防。

有奧尼爾強大的內線牽制力，湖人隊其他球員在攻防兩端更加輕鬆，當奧尼爾在內線遭遇多人夾擊把球分給隊友的時候，隊友只需要把握住空位出手的機會即可。00-01賽季，奧尼爾帶領湖人隊取得了56勝26負的戰績，位居西部第二。

MVP票選第四：克里斯-韋伯

韋伯就像是奢華版的阿爾德里奇，他既可以像阿爾德里奇一樣命中中距離直臂跳投，還能送出阿德做不到的華麗助攻，他是那支華麗國王隊的靈魂人物。00-01賽季，韋伯拿到了生涯最好的數據表現，他出戰了70場常規賽，場均拿下27.1分11.1籃板4.2助攻，入選了那個賽季的最佳陣容一陣。

當一個大個子球員能像控衛一樣精準地找到隱藏在角落裡的隊友，這個大個子所在的球隊進攻肯定非常流暢。韋伯就是那位能像控衛一樣打球的大個子，在他的帶領下，國王隊55勝27負，排名西部第三位。
