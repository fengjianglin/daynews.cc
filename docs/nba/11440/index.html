NBA自甘墮落球員，一人本該是喬丹的一生之敵，卻因毒品英年早逝

2019-12-11 687條評論 3,926次閱讀 1091人點贊

我們都知道許多NBA球員都是通過籃球來改變了他們貧苦的生活，很多球員出身貧民窟，他們從小沒有過過一天好日子，在加入聯盟後一夜暴富，有了很多的金錢，自然就會被一些以前沒有見過的奢靡生活和美色腐蝕，很多球員因此也漸漸墮落，從而無球可打離開了NBA。今天我們就來看看NBA七大自甘墮落球員。

第七位，艾迪·庫里

艾迪·庫里是一位非常統治力的內線球員，他擁有這非常強壯的身體，在內線里稱霸，並且得到了尼克斯的大合同，但是他並沒有因此變得更加努力，而是開始了紙醉金迷的生活，對身體也不在控制，導致嚴重發胖，但是他並沒有奧尼爾那樣的超級天賦，因此慢慢沉淪。

第六位，文·貝克

文·貝克是一位全明星球員，他在雄鹿是能夠場均拿到20+10的火爆數據。要知道那是在一個崇尚防守的年代。真正毀掉他的還要說是98年聯盟停擺，在這個期間由於無球可打，因此他也不再注意控制體重，開始發福，當聯盟回歸正軌的時候，他卻偏離了軌道，慢慢的他也就被球隊拋棄，變成一個無球可打的前全明星球員。

第五位，巴郎·戴維斯

巴郎·戴維斯是第一代「大鬍子」，他和今天的哈登有著相同強壯的身體和精準的投籃。他的突破和三分常常能把對手打得找不著北，並且還帶領勇士創造了黑八奇蹟，把不可一世的小牛打回老家。但是有無後來他拿到大合同以後變得無比懶惰，體重也開始增長，身體無法負擔這個重量，開始出現傷病，最終只能離開。

第四位，安東尼·沃克

安東尼·沃克是曾經96黃金一代6號新秀，他在新秀賽季是和科比、艾弗森一個級別的新秀一陣成員，並且在凱爾特人中也是絕對球星的存在，是無數球隊爭相追逐的球星，但是在生涯末期他也是沒有很好的控制體重，導致早早退役，並且由於不懂理財，揮霍了1億的身家。

第三位，肖恩·坎普

肖恩·坎普是一位非常勁爆的大前鋒，在比賽中能夠扣籃就絕對不上籃，他的彈跳能力是聯盟頂級，但是在停擺賽季的時候也是沒有很好的控制飲食，並且疏於訓練，因此最終因為體重增長，失去里之前的爆發力。並且他生活作風方面也非常自由，有很多私生子。因此在回到NBA以後無法適應只能退役。

第二位，艾迪·格里芬

埃迪-格里芬本是一個非常有潛力的新星，但是他並沒有脫離了美女和酒精的侵蝕，在25歲時就因為超速駕駛葬送了年輕的生命。

第一位，倫·拜亞斯

倫·拜亞斯是一位絕對意義上的籃球天才，在大學聯賽中是喬丹的對手，在能力上面與喬丹不相上下，一個人對陣喬丹的三人組。但是在86年被NBA選中榜眼以後他卻沒有去參加凱爾特人的訓練，因為在那之前他就因為吸毒過量死了，真是一種莫大的遺憾。如此的天賦就被毒品葬送了，否則在NBA中可能能看到和喬丹抗衡的局面。

看到這麼多本該大放異彩的球星最終泯然眾人，特別是拜亞斯本該是喬丹的一生之敵，卻因毒品英年早逝，實在令人揪心。
