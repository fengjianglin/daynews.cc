杰倫-布朗：第二節的防守幫助我們拿到勝利

2019-12-05 0條評論 15次閱讀 0人點贊

虎撲12月5日訊 凱爾特人今天主場112-93擊敗熱火，全場比賽綠衫軍前鋒杰倫-布朗發揮出色，首發出戰34分鐘的他20投10中得到31分4籃板1助攻1搶斷，賽後布朗接受採訪談起了球隊的表現。

綠衫軍今天第一節20-28落後，但他們在第二節單節打出32-16將比分反超。

「第二節球隊的防守幫助我們拿下了比賽。」布朗如此說道。

過去6場比賽，杰倫-布朗場均可以得到22.5分，投籃命中率高達53%。

布朗表示，最近一段時間對手總是先針對肯巴-沃克和傑森-塔特姆進行防守，那麼在他們後面的自己就利用這些優勢去得分，而隊友沃克也肯定了布朗在球隊中的作用。

「當杰倫找到狀態，我認為他就是球隊的X因素。當他打出來，這肯定給了球隊更多選擇，而且這個賽季他一直非常棒。」沃克如此說道。

比賽第三節，杰倫-布朗在命中一個底角三分後還轉身向著熱火替補席做了個「飛吻」的動作。

賽後在談到這一點時，布朗笑著說道：「那是送給凱利（奧利尼克）的，我知道很多人在那邊，但這是給凱利的。他是我的菜，我們之間充滿著愛。他是波士頓球迷喜歡的人之一，所以我只是努力給他展示一點愛。」

(編輯：劉旭)
