美國楊毅打趣：保羅如果這樣打的話會被雷霆隊交易的

2019-12-17 1,420條評論 2,436次閱讀 918人點贊

北京時間二零一九年十二月十七日nba常規賽繼續進行已經兩連敗了雷霆隊今天坐鎮主場迎戰公牛的挑戰。最近兩支球隊狀態依然不好。但隨著雷霆隊近期的表現。良好，現在已經來到西部第八的位置，但本場比賽雷霆隊招數的考驗極大，在落後最多26分的情況下，憑藉保羅在最後時刻超常的發揮，在主場以109:106逆轉拿下公牛隊。繼續保持著季後賽的競爭名額

比賽一上來。場面的局勢已經出現了一邊倒的狀態。雷霆上來不是失誤就是打鐵，而公牛隊則捉住了機會，連連打出反擊，公牛一波9:0開始迅速建立了領先的優勢雖然雷霆有亞當斯和弗格森及時三分為雷霆止血，但依然不能逆轉場上的局勢，公牛隊由拉文再次帶領公牛打出一波反擊小高潮一波22:6的瘋狂攻勢一舉確立了20分的領先優勢，首節比賽公牛以37:16領先雷霆21分

第二節上來，雷霆隊連得7分稍稍迫近了比分，但公牛對這邊手感依然火熱，瓦倫丁和阿西迪亞諾克在命中三分。比分一直在20分左右。此後雙方打起來拉鋸戰，雷霆隊依然在本節沒有追回比分，半場結束，依然以49:68落後19分進入下半場

下半場一開始雷霆隊由加里內利的罰球和保羅的三分幫球隊追分。公牛隊拉文連得5分給予回應，此後雷霆的進攻打得有聲有色，連續四個回合都是通過內線取得的，通過一波小高潮將比分追到14分。但拉文還是公牛隊最穩定的點，一人連拿7分穩住局勢，但本節末端，雷霆隊再有保羅和亞當斯接連命中憑藉一波11:2的小高潮，將比分追到只剩下8分結束第三節

第四節上來瓦倫丁和馬爾卡寧的進球。再度將比分擴大，但雷霆保羅對站出來，連續兩計單打命中三分，瞬間將比分縮少到4分。此後保羅似乎有神幫助一樣外線再度命中兩計三分，本節保羅借一人之力狂砍19分，幫助雷霆隊反超比分。雖然公牛隊最後時刻有拉文一人拿下六分，幫助公牛隊追平了比分。但年輕的公牛在最後時刻依然付出了年輕的代價，由小卡特最後卡未被吹犯規，喪失了最後反撲的機會，最終雷霆隊在主場以109:106逆轉了公牛，取得了勝利

本場比賽，雷霆隊似乎有如神助表現最好的依然是那位35歲老將保羅全場出戰36分鐘情況下12投9中命中6個三分全場得到30分10個籃板8個助攻2個搶斷。加里內利16投6中也有22分6個籃板4個助攻。替補中的施勞德和弗格森也有18分和12分的貢獻。諾伊爾和亞當斯也有各9分的進賬

其實公牛隊本場比賽也發揮的也不錯，也高達6人得分上雙，其中表現最好依然是核心拉文全場25投15中得到全場最高的39分外加2個籃板3個助攻2個搶斷的優異表現，卡特和馬卡寧也有13分和12分的貢獻，替補中瓦倫丁和阿斯迪亞科諾也有10分和12分的進賬

本場比賽最大的亮點是35歲老將保羅再次以勁爆的表現。證明外界自己依然是那位第一控衛在全場12次都出手情況下就得到30分10個籃板8個助攻的准三雙成績。在夏天被交易到雷霆隊，很多外界認為保羅這一次的冠軍夢已經失去了，都認為，雷霆隊已經走向擺爛的路上，但35歲的保羅依然持著那一顆冠軍的心扛著雷霆隊慢慢的前進，不被外界認為的情況下已經第西部第八的位置，這場比賽過後。有美國楊毅之稱的史密斯地調侃保羅這樣打的話或許會被雷霆高管交易的，不知道大家怎麼看呢，歡迎大家在下方留言
