詹皇又變廢為寶？湖人256萬龍套變奇兵，三分命中率48%變利器

2019-12-04 0條評論 16次閱讀 0人點贊

湖人隊在終結了十連勝之後開啟了12月份的魔鬼賽程，但是再度面對西部的強隊掘金時湖人隊展現出來的表現令人讚歎。湖人目前在西部除了快船之外，確實沒有其他球隊能夠抗衡。

隨著比賽的深入，目前來看湖人隊的表現較為出色，球隊的角色球員表現也是可圈可點。在比賽中就連隆多在本賽季的表現也是十分出色，個人職業生涯的三分球命中率達到生涯最佳。

在面對掘金隊的比賽中，隆多的三分球2投2中，而這也是他本賽季投中的16記三分球。在本賽季里，隆多平均每場比賽能夠命中1.4記三分球，可以說是湖人隊外線較為穩定的投射點，這也很好的幫助了球隊拉開空間。本賽季隆多的三分球命中率達到48.4%，職業生涯新高。

賽季至今，隆多場均登場20.5分鐘，可以得到8.3分3.1籃板5.5助攻0.91搶斷，投籃命中率45.5%。這也說明了隆多目前的中距離投籃命中率還不如三分球命中率，隆多需要更多的去進行調整。

如今在湖人隊陣容中，隆多的定位更多的變成了球隊的一個防守者以及球隊在替補階段的組織者。這兩點對於隆多來講是輕而易舉的，這也是過去職業生涯中隆多能夠做到的事情。

但是對於隆多來講，在職業生涯中去進行一個定點投射，這就需要他跟詹姆斯一起出戰時，當詹姆斯掌控球權，隆多需要去做到的地方。本賽季來看，隆多的表現可圈可點，逐漸在向這個方向靠攏。

作為過去基德在職業生涯末期在小牛隊期間打出了職業生涯不錯的三分球命中率，如今隆多與基德一樣開發了自己的三分球，對於湖人來講這是去奪冠的利好。
