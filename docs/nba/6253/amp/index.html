王朝覆滅

賽季開始剛剛過去一個半月，這一個半月NBA卻呈現了新的格局。如果一年前的今天你說勇士隊明年進不了季後賽，我相信你會被球迷的吐沫直接淹死。畢竟如日中天的宇宙勇怎麼看也不可能會進不了季後賽，但是僅僅一個半月就讓我們接受了這個事實，今年的勇士隊確實進不了季後賽了。

從六月份杜蘭特在總決賽第五場遭遇跟腱斷裂和第六場克萊遭遇十字韌帶撕裂僅僅過去五個多月，勇士隊卻早已從王朝淪為魚腩。六月三十號杜蘭特宣布加入籃網為了歐文的友誼使他做出了這個決定，勇士王朝宣布解體。庫里在十月三十一號對陣太陽的比賽中，一次突破內線衝撞後倒地左手落地遭遇左手掌骨折預計缺席3個月。本賽季勇士遭遇了重創核心球員全部受傷，使得賽季開始就失去了爭奪季後賽的資格。

四方爭霸

隨著勇士王朝的解體，聯盟諸強爭相補強進入爭冠行列。

  1. 西部快船引進倫納德和保羅喬治兩位聯盟頂級鋒線組成最強鋒線組合，倫納德手握上賽季總決賽MVP併入選常規賽最佳陣容二陣。保羅喬治上賽季常規賽MVP前三候選人併入選最佳陣容一陣，二人組合配上在季後賽和宇宙勇廝殺六場的路易斯威廉姆斯和哈雷爾組成四大天王，兩位首發兩位替補，快船現有的陣容應該也是聯盟最強陣容之一，無論板凳深度還是首發厚度都屬聯盟前列。
  2. 洛杉磯的豪門湖人也在休賽期簽來安東尼戴維斯和詹姆斯組成詹眉組合，一內一外堪稱無敵之勢。總冠軍球員丹尼格林的加入也讓湖人在外線獲得一名優質的3D球員，上賽季的格林三分命中率聯盟第一併且還擁有非常優秀的外線防守能力。火花霍華德的「悔過」讓他重新加入湖人，本賽季霍華德放棄了出手權增加了擋拆搶籃板使得湖人的內線實力大幅度增強。
  3. 上賽季聯盟第一的雄鹿本賽季依然勢頭正猛不可阻擋，字母歌帶領球隊拿下12連勝，雖然休賽期缺失了布羅格登但是字母歌的進一步進化使得雄鹿並沒有缺少太多的戰鬥力，在東部依然是一枝獨秀。在倫納德西歸，杜蘭特受傷字母歌在東部應該是大殺四方無人能擋，沒有能與之匹敵的超級巨星。
  4. 被倫納德在搶七絕殺當成背景板的76人，休賽期也沒有落人下風，巴特勒的出走讓他們引進新的全明星中鋒霍福德和熱火的理查德森組成聯盟死亡五大。在東部只有雄鹿能對他們產生威脅，其他隊伍都好像差那麼一點。
  5. 衛冕冠軍猛龍因為倫納德的出走喪失了球隊關鍵時刻決定比賽的巨星，雖然西亞卡姆的快速成長讓他們依然保持這優質的戰績。但是在季後賽遇上戰術針對的時候，猛龍沒有一個可以掌控比賽的超級球星所以球隊實力會低上一檔，畢竟西亞卡姆相對於字母歌和恩比德還是差點成色。

新星崛起

隨著杜蘭特、庫里、湯普森等全明星的受傷，新賽季的聯盟超新星們似乎看見了全明星的大門在向他們招手。東契奇、特雷楊雙星閃耀，賽季開始至今東契奇和特雷楊都位列聯盟得分榜和助攻榜前四，打出了超級巨星的數據，東契奇更是帶領獨行俠打進西部前四有很大機會帶領獨行俠重新打進季後賽。

閃耀的新星可不止這兩位，魚腩多年的太陽終於隊季後賽有了想法，休賽期引進盧比奧和烏布雷使得太陽從聯盟墊底上升至西部第八。太陽當家球星德文布克70分先生本賽季也有望首次進入全明星。

東部猛龍隨著倫納德的離去，球隊迅速擺正西亞卡姆的核心位置，雖然失去了倫納德和丹尼格林但是猛龍在西亞卡姆的帶領下依然穩坐東部第二。隨著西亞卡姆的大幅度進步，這個賽季也應該是全明星的熱門人選之一，公牛的拉文籃網的丁威迪都同樣打出很好的數據都有機會入選今年的全明星。

總結

競技體育就是如此的殘酷和精彩，我們悲痛於杜蘭特、湯普森等人的重傷，我們也欣喜東契奇、特雷楊、西亞卡姆等人的成長。一如標題寫的那樣「年年歲歲花相似，歲歲年年人不同」，物是人非沒有人知道明天會有誰會倒下也沒人知道明天會有誰會站出來！
