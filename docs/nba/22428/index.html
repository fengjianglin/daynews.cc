狂轟143分創39年新高！魚腩4戰3勝終於爆發，他才是最大贏家？

2019-12-18 541條評論 3,476次閱讀 2498人點贊

北京時間12月18日，尼克斯主場143-120大勝老鷹，過去4場比賽贏下3場。這支東部的超級魚腩，更換主帥後，倒是顯得面貌一新。在尼克斯隊史上，非加時比賽得到143分，已經能夠排到第三位，僅次於1980年的149分和1977年的144分。換言之，143分是近39年來尼克斯在不加時比賽里得到的最高分！

換帥如換刀，還真是如此。自從邁克-米勒成為球隊的臨時主帥後，尼克斯取得了3勝3負的戰績，勝率達到50%，而此前菲茲戴爾本賽季的帶隊戰績是慘不忍睹的4勝18負。

數據方面，尼克斯也是全面爆發。本場比賽，巴雷特27分6籃板，莫里斯22分，米切爾-羅賓遜22分13籃板2蓋帽，蘭德爾17分11籃板，諾克斯17分，全隊投籃命中率55.8%，三分命中率46.4%，這哪像一支魚腩球隊的數據？老鷹一方，特雷-楊空砍42分，亨特19分，卡特15分。

回顧這場比賽，尼克斯的優勢基本在上半場就建立了。首節，他們打出了41-27的高潮，獲得了14分的領先優勢。莫里斯在第一節就拿下13分。次節，尼克斯輪到巴雷特發揮，他在這一節得到9分，半場結束時，尼克斯77-53領先對手24分。下半場，兩隊得分差不多，老鷹只追回1分，最終以23分的分差輸給對手。比賽中，尼克斯最多領先老鷹31分！

過去4場比賽，尼克斯贏下3場，他們分別擊敗了勇士、國王和老鷹，輸給掘金那一場，也只輸掉6分。這支球隊的精神面貌有了很大改觀，看來換了個主教練，是有不少幫助。

尼克斯狀態回暖，最大的贏家當屬臨時主帥邁克-米勒。雖然球隊只是任命他暫時擔任主帥一職，而且正在物色正式主帥的人選，但如果米勒能夠率隊打出不錯的表現，也不排除他被最終扶正的可能。
