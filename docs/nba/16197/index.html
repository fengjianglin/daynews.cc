馬刺隊連續四場打入加時賽，更衣室領袖爆發，漂移中投絕殺太陽隊

2019-12-15 597條評論 1,566次閱讀 60人點贊

12月15日，馬刺隊和太陽隊在墨西哥城展開交鋒，這是兩支球隊在本賽季的首次交手，太陽隊球星德文-布克因傷缺陣。兩支球隊在上一場比賽都遭遇敗仗，馬刺隊被東部倒數第二的騎士隊拖入加時，並最終輸球，主教練波波維奇對此非常不滿。太陽隊則是以108-115輸給了西部弱旅灰熊隊，賽後盧比奧坦言這是球隊在本賽季最糟糕的失利，球隊必須振作起來。可以說，本場比賽兩支球隊都是力爭勝利，比賽過程也是非常很膠著，兩支球隊交替領先，常規時間戰至109平，加時賽最後時刻，米爾斯命中漂移中投才殺死比賽，馬刺隊以121-119驚險戰勝太陽隊。

第一節比賽，馬刺隊率先找到投籃準星，米爾斯替補上場，單節砍下8分，馬刺隊以28-18領先結束第一節比賽。第二節，太陽隊逐漸找回手感，盧比奧三分、上籃相繼命中，太陽隊以32-30反超比分，太陽隊多點開花，單節打出30-16，以48-44領先進入下半場。

第三節比賽，兩隊的內線防守形同虛設，卡明斯基利用阿爾德里奇防不出來的弱點，連續兩次投籃命中。馬刺隊則抓住卡明斯基和薩里奇同時在場護框能力不足的弱點，頻頻攻擊籃筐，默里、萊爾斯、德羅贊、懷特、沃克都有突破得分入賬，不斷縮小比分差距。太陽隊這邊，盧比奧率先站出來，利用出色的組織能力幫助球隊，先後助攻烏布雷三分命中，薩里奇籃下輕鬆取分。珀爾特爾命中壓哨上籃，幫助馬刺隊以78-80緊咬比分。

第四節比賽猶如坐過山車般精彩刺激，貝里內利漂移中投，蓋伊上籃，默里反擊暴扣，沃克反擊上籃，馬刺隊球員打出一波8-0的得分高潮，一舉將比分反超為86-80。太陽隊在2分半鐘的時間裡，一分未得，直到卡明斯基轉身上籃得手止血，隨後太陽隊大個子開始接管比賽，薩里奇籃下和三分線外均有進球，貝恩斯則送上兩記暴扣，馬刺隊阿德則是頻頻打鐵，太陽隊以93-92反超比分。

第四節後半段的比賽，兩隊展開拉鋸戰，馬刺隊這邊，阿德中投，德羅贊突破，米爾斯連中兩記三分，太陽隊這邊，貝恩斯相繼命中三分和上籃，薩里奇、盧比奧四次罰球全中，雙方戰至104平。最後1分鐘，蓋伊三罰全中，貝恩斯上籃得手，阿德籃下放籃得手並且造成貝恩斯推人犯規，但是罰球不中，給太陽隊留下了扳平比賽的機會。太陽隊前場發球，烏布雷接盧比奧傳球，面對阿德、默里的撲防，穩穩命中底角三分。隨著蓋伊翻身跳投不中，比賽進入加時。

加時賽，馬刺隊默里先得兩分，之後兩隊頻頻打鐵，阿爾德里奇補籃打破僵局，太陽隊烏布雷也打成2+1。米爾斯運球失誤後將功折罪，搶斷盧比奧，命中反擊三分，馬刺隊佔據主動。太陽隊新秀約翰遜扣籃得手，追平比分，阿德兩罰一中，又丟罰球，盧比奧建功，三分線外撤步穩穩命中，太陽隊119-117反超。太陽隊成也盧比奧，敗也盧比奧，關鍵時刻，盧比奧接球出界失誤，球權回到馬刺隊手中。德羅贊直殺籃下，高拋打進並且造成布里奇斯犯規，但是德羅贊加罰不中，將絕殺的機會拱手讓出。

在爭搶籃板的過程中，籃球打在太陽隊球員的身上出界，馬刺隊有3.8秒的時間終結比賽。波波維奇最信任的米爾斯接球，借阿爾德里奇的掩護，罰球線漂移出手，中投絕殺太陽。

本場比賽，馬刺隊球員阿爾德里奇、德羅贊、默里各取18分，沃克和蓋伊也有得分上雙的表現。球隊贏球的最大功臣還是老將米爾斯，米爾斯整場比賽保持侵略性，23投10中，三分14中6，拿下26分5板1助攻，並且命中決勝中投。
