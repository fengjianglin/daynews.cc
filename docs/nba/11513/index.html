保持冷靜！湖人不會讓他們歷史性的開局來影響他們的注意力

2019-12-10 1,145條評論 4,196次閱讀 1471人點贊

對於湖人來說，留住現在的狀態是他們成功的關鍵！

洛杉磯湖人在2014到15賽季中拿下了21場勝利（注意這是整個賽季的）。而本賽季開賽至今，湖人在詹姆斯和戴維斯的帶領下已經取得了23勝3負的最佳戰績。這不僅證明了拜倫.斯科特領導的湖人隊是多麼糟糕，同時這個數字也證明了這支球隊在本賽季已經有了一個非常好的開端。

湖人在歡呼

在上周他們輸給獨行俠隊後，湖人隊分別與掘金，爵士和開拓者進行了比賽，沒有任何意外的是弗蘭克·沃格爾帶領湖人贏得了這三個客場勝利。湖人隊現在擁有西部最好的21勝-3負的戰績。

湖人上一次的最佳開局的是在2008-09賽季，也就是他們擊敗魔術的那一年。此外，11連勝的客場紀錄也是他們自1972-73賽季以來最長的連勝記錄。用球迷的話說，他們不僅有出色的戰績，更有良好的精神狀態，但是難得可貴的是更衣室里的每個人到目前為止都保持著冷靜的頭腦。就像主教練沃格爾在接受採訪時被問到：他是否認為湖人的球隊具有歷史意義時說。「對我來說，我每天醒來都是0勝-0負。我要求我的團隊也這麼做，我們只看眼前的這一天」。隨著湖人以142比125擊敗森林狼，弗蘭克·沃格爾可能是獲得今年最佳主教練的領跑者之一。

詹眉連線

沃格爾還補充到：「媒體可以談論這些事情，但在內部，我們實際上只是專註於讓每天變得更好。」

除了主教練優秀外，他們的兩大核心更是球隊的火車頭，他們取得的化學反應令人吃驚。詹姆斯與戴維斯兩人是互補的，因為他們都是能夠做出正確發揮的球員，而且他們都願意傳球。

詹姆斯與戴維斯

並且湖人的陣容也趨於完善，隨著賽程的深入，他們之間的默契也越來越足。當丹尼·格林, 凱爾·庫茲馬,和特洛伊·丹尼爾斯以及奎因·庫克在外面的時候，他們能給內線留下足夠的活動空間，當然，如果他們接到球，他們也會毫不猶豫的把球投出去。而布拉德利和卡魯索則更注重防守，自布拉德利傷退後，湖人的防守效率在逐漸下降；而波普是兩者兼而有之能攻能防；隆多的回歸則可以為詹姆斯爭取更多的休息時間；麥基和霍華德的配對使得戴維斯不需要長時間的任職中鋒。

湖人球員相互鼓勵

  • 他們接下來將去挑戰魔術隊，此後還有熱火，這也是東部的種子球隊。但是球迷們都在期待湖人與快船和雄鹿的對決。

還記得詹姆斯曾說。「我打得太久了。我只關心我們今晚做了什麼，以及我們明天會如何變得更好。這對我們來說是一次很好的客場旅行，從奧蘭多開始。」離這個賽季結束還有很長一段時間，湖人管理層也還有大量的工作要做，但我們不得不讚揚佩林卡的運作，當湖人沒有追求到倫納德時，他們用冷靜的思維方式來組建了這支隊伍。可以預見的是戴維斯和詹姆斯與湖人隊將在本賽季迎來一個偉大的賽季，戴維斯累積的獎項與數據就是一個最好的證明，他們把目光放到了明年6月的奧布萊恩杯上，如果他們繼續保持目前的水平，他們將有很好的機會做到這一點。
