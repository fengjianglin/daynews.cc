倫納德42+11喬治46分 雙核爆發仍贏得驚險！快船隊一比賽數據揪心

2019-12-14 728條評論 1,665次閱讀 486人點贊

在多倫多大比分擊敗猛龍隊，快船隊的六連客之旅3勝1負，從對陣雄鹿隊潰敗到如今三連勝，快船隊的調整能力比較出色。雖然賽季開始之後因為傷病、輪休以及磨合等因素，快船隊的戰績受到了一定影響，但是悄然之間快船隊還是拿到了西部第二好的球隊戰績，僅次於榜首的湖人隊。

雖然贏下了對陣猛龍隊的比賽，但是貝弗利卻在比賽中受傷，防守時撞到馬克-加索爾身上，遭遇腦震蕩傷退。在談到球隊傷病的時候，路易斯-威廉姆斯樂觀表示，「我們擁有足夠的深度，如果有人缺席，其他人也會站出來並帶來能量，因此這對我們非常重要，但願我們還是能儘快迎回所有人。」

客場之旅還在繼續，快船隊奔赴明尼蘇達挑戰森林狼隊，對手森林狼隊近況不佳，已經吃到了六連敗。唐斯談到這一情況時說道：「我們在賽季開始時成績不錯，但我們球隊目前正在下滑。他們沒把我們當回事……我們已經向他們展示了我們能做得多好。所以，我們不能再鬆懈了。」

森林狼隊會在對陣快船隊的時候反彈嗎？

蒙特雷茲-哈雷爾

貝弗利缺陣的情況下，特倫斯-曼頂替進入到首發陣容之中。森林狼隊首次進攻，安德魯-維金斯打成一次2+1，顯示出了不錯的進攻狀態。快船隊雙巨也迅速進入到狀態，保羅-喬治進攻得手，科懷-倫納德連得4分，快船隊6比3領先森林狼隊。唐斯也在進攻端做出貢獻，雙方比賽開始時都靠著各自核心打開局面。

然而森林狼隊的防守太過鬆散，保羅-喬治跟科懷-倫納德打得太過輕鬆，兩個人各得5分打出一波10比2，快船隊以16比9領先森林狼隊。倫納德和喬治再連得7分，快船隊領先優勢擴大到了12分，維金斯造犯規兩罰兩中，唐斯命中一記三分球，森林狼隊兩位年輕球星帶隊追趕。

倫納德再中一記三分球，輪轉階段被換下場。然而在路威、貝弗利都缺陣的情況下，快船隊的輪轉陣容出現了麻煩，很難在進攻端得分。森林狼隊轟出一波11比3的小高潮收尾，快船隊只靠罰球拿到了3分，而維金斯帶隊越殺越猛。首節比賽結束，快船隊33比29領先森林狼隊。

科懷-倫納德

第二節比賽開局，雙方都出現了一些失誤，哈雷爾在籃下進攻得手，快船隊率先在第二節比賽得分。保羅-喬治命中一記三分球，戈爾吉-吉昂給出回應，雙方保持著僵持的態勢。兩隊在進攻中造對手犯規，比賽的局勢沒有任何改變，安德魯-維金斯兩罰兩中，森林狼隊仍然落後6分。

不過在膠著的時候，保羅-喬治連續進攻得到4分，快船隊將分差拉大到了兩位數。保羅-喬治的進攻手感還在，不過考文頓命中三分球又上籃得手，幫助森林狼隊止住血。快船隊的進攻效率下滑，球隊陷入到了得分荒當中，約什-奧科吉打成2+1，唐斯兩罰兩中，森林狼隊追平了比分。

唐斯命中三分球反超比分，場上形勢眼見著不妙，倫納德在進攻端挺身而出，一人連續得到9分穩住了局面。保羅-喬治也在進攻中帶來幫助，考文頓仍然還是森林狼隊重要得分點，儘力幫助森林狼隊不被繼續拉大。第二節比賽結束，快船隊65比58領先森林狼隊，倫納德和保羅-喬治聯手砍下54分。

卡爾-安東尼-唐斯

下半場比賽開局，保羅-喬治抖落出武器庫中的各中進攻手段，三種進攻方式全都打成轟出一波7比0。第三節比賽只打了一分半鐘，快船隊將分差擴大到了14分，森林狼隊被迫叫了暫停。然而還沒有按住保羅-喬治，祖巴茨的進攻火力也來到，森林狼隊愈發變得被動起來。

維金斯打成2+1，保羅-喬治進攻得手，倫納德造犯規兩罰兩中，森林狼隊信心受到一定打擊。倫納德打得順風順水，而維金斯零星地進攻保持著進攻火力，快船隊一直都領先著20分左右。哈克里斯命中一記三分球，快船隊領先優勢擴大到了21分，比賽完全在快船隊的掌控之下。

保羅-喬治三分球命中，哈雷爾罰球命中，快船隊領先達到27分之多。森林狼隊靠罰球得到3分，保羅-喬治卻再次轟進一記三分球，森林狼隊得分之後快船隊就能給出回應。直到第三節最後一分鐘，傑夫-蒂格一個人打出6比0的比分，三節比賽結束，快船隊102比81領先森林狼隊。

保羅-喬治

末節比賽開始之後，唐斯率先在進攻端發力，連續得到8分之後，森林狼隊的信心有所提升。傑夫-蒂格進攻得手，維金斯再得到兩分，森林狼隊將分差縮小到14分，快船隊的領先優勢被不斷地蠶食。傑夫-蒂格兩罰兩中繼續追分，倫納德命中三分球，唐斯迅速給出回應。約什-奧科吉進攻得手，分差追至只有10分。

森林狼隊打到最後時刻，終於爆發出一些血性，不過他們把握機會能力並不夠強。然而快船隊犯的錯誤更多，唐斯外線投籃造了倫納德3+1，將分差迫近到了5分。被動局面之下，哈雷爾連續在籃下打成兩次進攻，快船隊終於緩過一口氣來。維金斯進攻得手，保羅-喬治也打成進攻，快船隊仍領先8分。

維金斯飆中三分球，哈雷爾突到籃下上籃得手，而唐斯扣籃過程中倒地，稍微休息之後重新投入比賽。快船隊關鍵時刻失誤不斷，維金斯強打得手，分差再被縮小到了4分。倫納德兩罰兩中，傑夫-蒂格造犯規同樣兩罰兩中。森林狼隊對小德里克-沃頓犯規，沃頓兩罰一中，不過森林狼隊的進攻回合卻出現失誤。

森林狼隊只能再次犯規，保羅-喬治兩罰兩中，比賽失去了懸念。最終全場比賽結束，快船隊124比117戰勝森林狼隊。

快船隊前三節比賽大比分領先，最多時領先27分，但是最後一節比賽卻被森林狼隊瘋狂追分。快船隊在場上太過鬆懈，末節比賽頻繁失誤，全隊總共19次失誤，這無疑是讓快船隊揪心的數據。

（文/顏無鏘）
