傷勢加重！進入觀察名單，有誰知道99年的他已經是一位年輕奶爸了

2019-12-02 0條評論 14次閱讀 0人點贊

莫蘭特大家都知道，是灰熊2019年的榜眼秀，是本賽季表現最好，最熱門的新秀，場均拿到18.6分3.2籃板6.4助攻，目前排名新秀榜的第一位。但在11月26日灰熊輸給步行者的比賽中，他持球進攻，突破上籃被特納蓋帽後身體失去平衡，落地時背部撞到攝影師受傷。

在灰熊接下來對陣快船和爵士的比賽中，莫蘭特都被列入賽前決定是否出場名單，但最後都帶傷作戰。而在這兩場比賽中，都明顯看到莫蘭特的表現受到背傷的影響，而灰熊為了謹慎起見，安排莫蘭特輪休一段時間。

莫蘭特擁有萬花筒般的過人技巧，打球風格又像威少，非常爆炸又很有激情。不僅個人進攻能力出色，組織和傳球能力也非常優秀，只有20歲的他球風卻很成熟，未來可期！希望他的傷病能趕緊痊癒，不然真的太可惜了。

而有誰知道，99年的莫蘭特已經有是一位年輕奶爸了！ 11月29日西方感恩節，他在社交媒體上曬出自己帶女兒的動圖，的確是年少有為啊。不過這種操作在NBA球星中也見怪不怪，老詹也是20歲有的兒子，而98年的塔圖姆兒子已經能在賽場上走路了！
