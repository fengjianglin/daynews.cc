首發正負值-96替補+81，快船再現奇葩數據，沒路威的哈雷爾太難了

2019-12-16 1,949條評論 4,672次閱讀 3618人點贊

今日快船做客芝加哥挑戰弱旅公牛，最終他們以106:109憾負對手。本場他們的替補發揮神猛，5人獲得上場機會，砍下56分，其中哈雷爾一人就砍下了30分，他們的正負值全部為正，合計高達+81。而首發就些慘不忍睹，僅得到了50分，正負值全部為負，5人合計-96，他們的低迷是球隊輸掉比賽的主要原因，也不得不說哈雷爾真帶領的替補太難了。

本場萊昂納德繼續背靠背輪休，而貝弗利和路威也沒有出場，但沙梅特傷愈復出，里弗斯排出了特倫斯曼+沙梅特+喬治+哈克里斯+祖巴茨的首發。首節比賽他們首發還是打的不錯，在比賽還剩3分43秒開始輪換時，他們以19:18，領先1分。而哈雷爾等替補​被換上場後則是開始大發神威，特別是哈雷爾，他連續強攻籃下，單節比賽他拿下了7分，帶領球隊以29:24結束戰鬥，這一節他們比首發多贏了4分。

第二節開始後，哈雷爾繼續帶領替補拉大分差，傑姆羅兵遜打成2+1，帕特森進3分，在比賽進行到還剩4分06秒時，他們以52:38領先了14分，此時哈雷爾已經拿下了17分。然而在喬治等首發回來後，分差被公牛漸漸追平，半場結束，他們僅以57:55領先，這節首發在場輸了12分之多。

下半場開始，首發繼續挖坑，隨著喬治打鐵失誤、祖巴茨失誤，哈克里斯打鐵，分差被公牛追平並逐漸拉開，時間還剩7分04秒時，他們從領先2分到落後9分。之後又是哈雷爾等替補上場才將比分迫近，三節結束，他們將比分追到了79:84，這一節哈雷爾拿下了全隊最高的9分。

最後一節開始後，哈雷爾帶隊開繼續趕比分，他連續在籃下強打馬爾卡寧造成犯規，而傑姆羅賓遜也命中3分，比賽進行到還剩7分41秒時，隨著麥克魯德2罰全中，他們將比分追到了91:91，然而在首發漸漸回來，他們最終輸掉了比賽。

整場比賽下來，快船替補的表現讓人嘆為觀止，他們5人出場，全部有得分進賬，得到了56分+搶下了22個籃板+送出11助攻+5個搶斷+4封蓋+僅8次失誤，這些數據全部比首發出色，不得不說太奇葩了。而在這些人中，哈雷爾表現得最瘋狂，他24投9中，罰球14罰12中，拿下了30分+7籃板+2搶斷+3封蓋，正負值為+15，然而在這樣的情況下還是難阻球隊失敗。

這個夏天快船的替補成了一道​靚麗的風景線，他們整個球隊場均能夠得到115分，其中替補得到51.21分，佔全隊得分的44.53%，得分和佔比都是排名聯盟第1，而且只要他們替補發揮失常，球隊基本上很難贏下比賽。

而他們替補之所以如此兇猛，主要是因為哈雷爾和路威太厲害，路威賽季至今場均拿下19.9分+6.2助攻，哈雷爾19.1分+7.6籃板。而他們2個同時搭檔時威力才是最猛的，但是如今路威不知什麼原因已經被輪休了2場，而如果今天他在的話，或許球隊就不會輸，哈雷爾應該很想念他吧！
