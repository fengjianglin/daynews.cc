保羅沒有看走眼！底薪悍將怒搶3前場籃板，德安東尼終於開竅了

2019-11-28 0條評論 14次閱讀 0人點贊

要說本賽季最受關注的球隊，在賽季開始之前，必然是馬賽克隊，即便是湖人擁有詹姆斯和濃眉，但是因為有著姚明情結，加上威少的加盟，所以馬賽克必然是最被關注的球隊，要知道在全美直播場次之中，馬賽克隊也是高居聯盟第三名，然而因為無腦總經理的行為，使得球迷們也是徹底放棄了馬賽克，不過可惜了威少和哈登，畢竟他們表達了自己的立場，也是讓球迷們非常感動！

哈登的表現也是非常穩定，除了開賽的幾場比賽略微有點打鐵之外，再次打出了現象級的數據，場均得分也是一度非常接近40分，而隨著傷病潮的來襲，哈登的表現一路帶著球隊前行，也是一度達到了西部第二，僅僅比湖人落後0.5個勝場，不過隨著威少的不斷打鐵和替補的連續不給力，掘金也是對哈登採用了季後賽的包夾戰術，而從那以後，球隊遭遇3連敗，威少的表現也被球迷們稱之為卧底行為，這是準備用實際行動趕走總經理。

哈登的表現雖然非常搶眼，幾乎憑藉著一己之力帶領球隊一度打出了全聯盟得分榜單第3的位置，不過再看一組數據，就非常尷尬了，賽季開始到現在，全聯盟替補得分最少的隊伍就是馬賽克隊，真的是讓球迷們感到無可奈何。而德安東尼也是非常固執，尤其是最近幾場比賽，小里弗斯打得非常差勁，但是依然被信任，而且幾乎場均30分鐘的上場時間，但是換回來的只是一次又一次打鐵。

小里弗斯最近的手感全無，近兩場比賽3分線外5投全失，而賽季的三分命中率也是跌到了28%，在無法幫助球隊獲勝的情況下，德帥似乎並沒有準備進行改變，然而就在對陣熱火的比賽中，德安東尼終於開竅了，在麥克勒莫並沒有取得預期效果的情況下，德帥啟用了克拉克這位小將，而事實證明，德安東尼選擇對了，保羅沒有看走眼！

這位底薪悍將獲得了29分鐘的出場時間，雖然三分線外只有6投2中，但是整體來看10投5中的克拉克非常高效，取得了12分8籃板1助攻1搶斷1封蓋的全面數據，證明了保羅確實沒有看走眼，當初正是因為克拉克不錯的表現，在生日夜當晚，保羅還親自送了一輛賓士獎勵這位小將，真是滿滿的幸福感。而更重要的是，這位底薪悍將怒搶3前場籃板，積極性也是讓球迷們眼前一亮，球迷們更是調侃，這樣的態度才是起到5年2億頂薪作用的！

在上個賽季，正是因為克拉克替補出場後投進的三分球， 幫助球隊打平了三分紀錄，而隨後又邁卡威打破了球隊三分紀錄，這位打平三分紀錄之人終迎爆發，對於球隊來說也是一件好事，至少板凳席上證明了有人有能力得分，如果這樣下去，也許球隊的戰績才會趨於穩定，也希望這位小將能夠獲得越來越多的機會吧。
