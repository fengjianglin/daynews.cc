前任與現任之爭？安東尼和約基奇誰才有資格在掘金退役球衣？

2019-12-15 1,230條評論 2,154次閱讀 771人點贊

提到掘金你會想到球星是誰？我想安東尼和約基奇這兩個名字一定繞不過去，他們二人一個是前任掘金老大一個是現任掘金老大，雖然有著年代的差距但是兩人卻因為同穿15號球衣而被放到一起比較，最近安東尼在採訪中的時候表示：掘金應該退役他的球衣。而如今15號戰袍正披掛在掘金新王約基奇的身上，那麼安東尼和約基奇誰才有資格在掘金退役球衣呢？

安東尼的掘金生涯

安東尼在2003年的選秀大會上在首輪第三順位被掘金選中開啟了他的NBA之旅，在掘金安東尼學會了如何成為一位NBA超級巨星和絕對領袖，在掘金效力的8個賽季的時間裡安東尼完美的兌現了他的進攻天賦，進攻萬花筒也成為了甜瓜進攻端的最佳表彰。也正是由於安東尼的到來讓名不見經傳的掘金隊成為了一支NBA強隊，而且安東尼還帶領掘金殺進了西部決賽，這也是掘金隊史最佳戰績！雖然沒能帶領掘金殺進總決賽但是如此成就安東尼在掘金退役球衣應該手拿把賺了，但是有個叫約基奇的出現了讓這件球衣的歸屬又出現了懸念！

​掘金新王約基奇

相比於安東尼的年少成名約基奇的成名之路就顯得坎坷了不少，作為一位二輪秀約基奇的成長速度著實驚人，尤其在去年約基奇首次打進NBA季後賽並且在季後賽的表現非常搶眼，約基奇的大局觀和團隊意識讓很多球迷都喜歡上這位塞爾維亞胖子。懶散的球風和頂級的球風讓很多球迷感嘆如果約基奇更加認真的訓練，打球更加努力一點會不會帶領掘金殺進總決賽甚至奪冠呢？如今的掘金屬於他們的新王約基奇！

​安東尼和約基奇誰才有資格在掘金退役球衣？

從個人能力和生涯數據來看安東尼都碾壓約基奇，所以掘金退役球衣的選擇大概率會選擇安東尼，而約基奇也並非沒有機會除非他能帶領掘金殺進總決賽甚至奪冠那麼他就更有資格退役那件15號球衣！

​關於安東尼為什麼想要在掘金退役球衣原因無處得知，其實這麼多年以來站在甜瓜的立場想想，自己帶領雪城大學在NCAA大殺四方，將雪城大學搖身一變變成強隊並奪得冠軍的時候，詹姆斯還在高中征戰，同一年選秀，瓜哥卻不是狀元，難免會想自己居然不如一個高中生，憑啥？進入聯盟第一年，無論是帶隊成績還是個人數據，都比詹姆斯要好，最佳新秀依然不是自己，憑啥？我們中立的想一想，其實這本來就不公平，對於任何人而言，心裡都會有落差。進入聯盟後，在進攻端，他的單打技術，投射能力，比同時期的韋德，詹姆斯都要強，別人還在學習教科書的投籃，自己就已經是教科書。或許，這就是命吧，有些人天賦異稟，能在職業生涯裡面遇到不錯的陣容，不錯的教練，能夠時刻保持競爭的心，刻苦訓練，展現自己的天賦。而有些人，天賦也很好，卻沒有遇到合適的陣容、合適的體系，還有合適的球隊。很顯然，甜瓜屬於後者。這種苦悶只有甜瓜本人最清楚！
