心疼詹姆斯！詹皇叉腰大口喘氣，防絕殺被撞倒，湖人太不爭氣

2019-12-14 31條評論 1,091次閱讀 146人點贊

詹姆斯真的還年輕嗎？西部第一的湖人對陣東部第二的熱火，在比賽的最後時刻，詹姆斯的狀態不禁讓人心疼！

同為超級小前鋒帶隊，上賽季被評為NBA現役十大球星第10位的熱火當家前鋒吉米-巴特勒在開賽前對湖人的詹姆斯可是放下了狠話。

「對於我們的戰績是否意外？等我們成為聯盟第一的時候，我再告訴你！」吉米巴特勒在ESPN專訪中說道。還未在主場輸過球的熱火遭遇湖人，果然比賽要比預想中還要激烈！

熱火在半場一度領先了多達10分，而憑藉詹皇第三節的強行登場，單節11分優勢追平！

而末節最後2分鐘，即便是詹姆斯剛的2個回合比分就被追近，連NBA主持人都調侃道：聯盟最好的球隊，結果詹姆斯一刻都不能下場，湖人確實太不應該。

歲月真是不饒人，比賽最後2分鐘的一幕成為了詹姆斯的縮影！在邊線球時雙手叉腰大口喘氣，引不少球迷心疼。

比賽最後4秒，湖人領先三分，詹姆斯錯位阻擋迎面撞來的巴特勒，結果一個狠勁直接被小自己一個身板的巴特勒撞飛在地，遺憾巴特勒沒能抓住機會錯失追平機會，而35歲的老皇帝或許早已不復當年模樣！
