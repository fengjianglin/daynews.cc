王哲林該加把勁了！周最佳被周琦易建聯包圓 周琦乃MVP有力爭奪者

2019-12-13 1,185條評論 3,876次閱讀 2328人點贊

北京時間12月13日，CBA聯賽官方公布第5周周最佳球員，本土球員易建聯以及外籍球員多米尼克-瓊斯當選。易建聯帶領廣東男籃取得3戰3勝成績，場均得到21.3分11.3個籃板3次搶斷以及2.7次蓋帽；多米尼克-瓊斯帶領吉林男籃也取得3戰3勝成績，場均得到47.7分8.3個籃板11.7次助攻以及1.7次搶斷。

易建聯和瓊斯當選周最佳

多米尼克-瓊斯本賽季首度當選周最佳球員，在周最佳外援的評選當中，富蘭克林兩次獲得周最佳，亞當斯、賽克斯各獲得過一次周最佳。相比於周最佳外援的變化較大，本土周最佳球員的懸念就小多了，五周的周最佳球員易建聯三次當選，而另外兩次則屬於周琦，兩個人包攬了前五周的全部周最佳。

作為國內球員中的翹楚，易建聯和周琦的表現非常搶眼，而另外一位明星內線王哲林卻沒有拿到過一次周最佳。王哲林沒有拿到過周最佳，跟球隊的成績有著很大關係，近三個主場贏下兩場比賽，福建男籃賽季至今總共只贏下四場比賽。球隊成績如此不堪，王哲林賽季場均25.3分10.7個籃板也只能是空砍。

而另一位國內球員郭艾倫狀態低迷，遼寧男籃本賽季的表現也並不出色。

郭艾倫賽季狀態起伏不定

郭艾倫本賽季的表現起伏不定，而遼寧男籃更是因為更換外援的磨合，球隊的成績受到了一定影響。從賽季初的連敗當中走出來之後，遼寧男籃仍然沒有找到最好的比賽狀態。最近幾場比賽，遼寧男籃整體性非常強，也讓人看到了球隊的競爭力，但是遼寧男籃的實力是否能夠衝冠，卻還需要碰一下強隊來檢驗。

而對於福建男籃來說，外援的選擇是造成球隊陷入困境的原因之一，埃里克-格林打了四場比賽被換掉，斯塔德邁爾的能力江河日下。而現在又換成了康寧漢姆，福建男籃還是在嘗試外援人選，「鐵打的泰-勞森，流水的搭檔」。外援的選擇不確定，這讓福建男籃的磨合以及球隊的防守很受打擊。

王哲林上賽季榮膺常規賽MVP

王哲林上個賽季拿到了常規賽MVP，然而本賽季球隊的情況卻對王哲林衛冕相當不利。儘管在CBA聯賽球隊戰績不是硬標準，上賽季福建男籃只排在第七名，王哲林還是拿到了常規賽MVP。而在往前，2016-17賽季山東男籃排在第八名，但是丁彥雨航也拿到了MVP。

福建男籃至少要保證球隊進入季後賽，王哲林才可能會有機會，而這一次衛冕MVP的困難卻要更大，周琦將成為MVP的熱門候選者之一。周琦本賽季場均能夠貢獻23.1分12.1個籃板以及2.8次蓋帽，攻防兩端都有著重要貢獻，新疆男籃本賽季戰績不錯，周琦將成為MVP的有力競爭者。

王哲林，要加把勁了。
