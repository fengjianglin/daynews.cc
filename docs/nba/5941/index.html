5分鐘兩隊合砍3分，這比賽太假，NBA遲早被蕭華毀了，你就瞎搞吧

2019-12-09 0條評論 15次閱讀 0人點贊

北京時間12月8日，NBA常規賽繼續進行，步行者客場104-103戰勝尼克斯送給對手9連敗。當然這場比賽本身沒有什麼值得過多討論的地方，雙方各取所需，但是比賽最後5分鐘簡直刷新了我對於NBA的認知。

第四節比賽還剩5分鐘結束的時候，步行者104-100領先尼克斯。正當大家準備欣賞緊張刺激最後5分鐘比賽的時候，高潮來了。兩隊開始瘋狂打鐵和失誤，僅特納一個人就蓋了尼克斯球員4次，這期間僅莫里斯命中了一記中投。

本來最後時刻尼克斯還是有機會扳平比分的，蘭德爾在比賽還剩0.1秒結束的時候獲得2次罰球機會，只要兩罰全中就進入加時賽，結果蘭德爾偏偏2罰1中，尼克斯就這樣輸掉了比賽。

可能有人會說他本來罰球就不準，2罰1中也是正常現象。可是你看了他第二個罰球就知道，根本就不是瞄著籃筐去的，隨手一扔就完事了。

自從NBA和菠菜公司合作後，這樣的假球已經出現了不止一場了，蕭華你就瞎搞吧，NBA遲早毀在你手裡。
