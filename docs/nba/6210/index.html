低迷！28分鐘9投2中僅得9+6凈負-14，安東尼對位卡椒慘被打爆

2019-12-04 0條評論 13次閱讀 0人點贊

北京時間2019年12月4日NBA常規賽繼續進行，NBA來了一場重頭戲。最近狀態火熱的開拓者，在客場迎戰近十場八勝，西部第三的快船。目前開拓者8勝11負，排名西部第11離西部第八的太陽還有1.5個勝場。此站他們不可怠慢，重新為一個季後賽名額而奮鬥著，但最終快船還是依靠替補的全面發揮，在主場以117比97大勝開拓者，從而終結對手的三連敗，繼續穩固球隊的西部第三的位置

比賽一開始，雙方球員就進入進攻的狀態。快船有萊昂納德和喬治率先發難內線突破外線三分怎麼投怎麼有，以15:8取得先機，但隨後開拓者立刻給予回應，在利拉德和安東尼的三分和罰球的帶領之下。在，本節末端打出了一波小高潮。在首節打完，開拓者以30:29還能領先結束第一節，第二節上來，雙方還是打得你來我往不分上下，的比分來到40平。此時快船隊的替補哈雷爾和帕特森兩個人站出來。幫助快船隊又打出一波小高潮以57-51重新獲得六分的領先優勢，但隨後開拓者。在利拉德的三分帶領之下勉強的迫近比分，半場結束，快船以62:59反超比分進入下半場

第三節上來，場上的局勢發生一邊倒的狀態，快船隊由喬治和祖巴茨帶領下打出一波小高潮，瞬間將比分拉到十分以上。此時雖然有麥克勒姆的三分止血，但快船的外線狀態依然火熱連續兩記三分投進。此後懷特塞德和利拉德的突破為開拓者迫近了比分。但本節末端還是哈雷兒站出來在內線不可阻擋頻頻為快船得分，在第三節結束時幫助快船89:76領先進入決勝節在第四節上來之後。此時，開拓者在進攻端陷入了得分荒。快船又在哈雷爾和帕特森的內外聯動又攻出一波7比0小高潮。將比分擴大到20分以上。雖然開拓者由替補大將西蒙斯連續反擊追趕別人，但最後時刻喬治里突外投也頻頻做出回應。穩穩的控住比分，再也沒有給開拓者反撲的機會，最終在主場以117比97大勝對手，從而終結對手三連勝。進一步鞏固球隊在西部第三的位置

本場比賽，快船隊可謂全民皆兵12人上場共有11人得分其中。其中最大的功臣莫過於喬治全場出場28分鐘14投9中得到25分4個籃板3個助攻。替補大將哈雷爾也有不俗的表現全場14投11中共砍下26分9個籃板，帕特森也砍下19分8個籃板的貢獻，貝弗利和威廉姆斯分別有7分和9分的進賬。另一位核心萊昂納德本場比賽發揮的一般，全場15投3中僅得11分9個籃板5個助攻另外還有2個搶斷

開拓者本場失利的原因莫過於他們的後場雙槍發揮的一般，利拉德全場15投5中僅得16分7個助攻2個籃板，麥克勒姆雖然17投9中得到20分3個籃板，但在場上是凈負23分全隊最低，懷特賽德和西蒙斯也有18分和11分的貢獻，令人最關心的是安東尼本場僅9投2中僅得9分6個籃板

其實本場比賽的亮點莫過於安東尼。面對新一代的聯盟小前鋒的對決。從賽後的結果和數據來看。我們感慨知道安東尼確實老了，全場在出場28分鐘的情況下9投2中僅得9分還有6個籃板，在以往來說相當於低迷狀態，相對於對面的快船保羅喬治和萊昂納德分別得到25分和11分，簡直被打爆，但我們作為球迷來說，昔日的安東尼是全聯盟觀賞性的前鋒，但隨著年齡增長身體上的機能和防守能力隨著年齡的下降。這是事實，但在35歲之際還能在場上打出這樣精彩的表現是我們作為球迷所希望看到的，雖然這場比賽輸了，但是我們還能看到安東尼在場上拼搏著，為自己最後的冠軍夢而奮鬥
