湖人全隊淡化對雄鹿關鍵之戰 霍華德：贏下這場不等於贏得總冠軍

天天要聞
2019-12-19

幾乎所有人都認為湖人隊與雄鹿隊的對決是聯盟大事件，只是洛杉磯的更衣室里不這麼認為。不管是對籃球發展史，還是NBA的偉大球員和球隊，勒布朗-詹姆斯都很有研究。他不僅在眾多經典比賽中擁有美好回憶，還包括他研究的那些過往經歷。

因此，當他在當地時間周二（北京時間12月18日）晚上得知湖人隊要在周四打一場多麼罕見的比賽時，這確實激起了他的一點興趣，屆時他們將前往密爾沃基去打比賽，而在這場比賽中，對陣的雙方都保持著聯盟最佳戰績。

但這樣的噱頭，也只是激起了詹姆斯一點興趣。據統計，周四湖人隊和雄鹿隊的比賽，這兩支球隊是在NBA完整賽季中打到12月19日或之後，第二次出現雙方僅有不超過4場失利的情況。另一次發生在1993年12月28日，當時23勝3負的火箭隊在西雅圖（20勝3負）擊敗了超音速隊。

「這挺酷。」詹姆斯說。

僅此而已。如果你想讓詹姆斯覺得82場常規賽中的某一場比賽尤其重要，那還是先看看他的過往經歷吧。他參加了239場季後賽和1226場常規賽。他已經經歷過所有能經歷的東西。

所以當地時間周四（北京時間12月20日）晚上，TNT直播的這場湖人隊（24勝4負）對上雄鹿隊（24勝4負）的比賽，可能只會讓籃球咚咚作響，但勒布朗不會覺得有什麼特別。

「這是我的第17個賽季了。」詹姆斯說。「像這樣的常規賽，兩支球隊只是想辦法努力打得越來越好，但是沒有什麼別的含義。我也不覺得有什麼特別的。」

在球隊輸給步行者隊後，詹姆斯並沒有更願意去談論雄鹿隊。在沒有安東尼-戴維斯的情況下，湖人隊打得很糟糕，他們罰球17罰僅8中，三分球31投8中，導致湖人隊客場連勝14場的紀錄就此終結。

「我能說的就是讓我們迎接下一場比賽吧。」後衛艾弗里-布拉德利說。

但對陣雄鹿需要湖人多付出一些努力。因為雄鹿隊同樣也輸掉了最近的比賽，獨行俠此前在密爾沃基以120比116戰勝了雄鹿隊，但雄鹿隊在主場的戰績是13勝2負，在過去10場主場比賽中更是9勝1負，平均每100次進攻比對手多得12.4分，這是聯盟最高的凈勝分。湖人隊以8.7分位居第二。

湖人教練弗蘭克-沃格爾說:「在整個82場比賽的完整賽季中，不管是什麼時候，兩支戰績最好的球隊碰面，總是能打出賽季最佳的比賽。所以希望兩隊能夠貢獻精彩的比賽。」

這將是一場非常棒的常規賽。但是湖人隊滿載著老將們季後賽走得更遠的期待，他們並不會去數著日子等待與密爾沃基雄鹿隊的常規賽對決，或者是思考擊敗雄鹿隊的方式有多麼重要。如果非要給這場聯盟第一之爭，再增添一些誇張的含義，你可以把它當做一場常規賽的重要節點去拿下，並讓一支強隊保持專註，但近期湖人險勝老鷹，又輸給步行者的表現好像又不像這麼回事。

沃格爾並不是不屑於去完成這樣的重要節點任務，隨著湖人隊的不斷前進，他制定了一些小目標來保持球隊的狀態。布拉德利因右腿出現應力反應而休戰了13場比賽，教練在隊內發出了「埃弗里-布拉德利挑戰」，讓球隊在每場比賽中打出聯盟最佳的防守等級，這一防守等級將延續到整個賽季。

中鋒賈維爾-麥基上周說，這（布拉德利缺陣）讓球隊「更有動力去防守，而我覺得去年我們正是防守出現了問題。」

如果說湖人隊用雄鹿隊來激發自身動力，顯得他們在貶低雄鹿隊。

「每場比賽都很重要。」中鋒德懷特-霍華德說。「每一刻都很重要。」

拉簡-朗多稱對陣雄鹿隊的比賽只是「又一場客場比賽」。亞歷克斯-卡魯索幾乎沒有表現出更多的熱情，他稱雄鹿隊是「一支非常好的球隊」，並補充說湖人隊也是「一支非常好的球隊」。丹尼-格林至少還承認這將是「一個有趣的比賽」，並說「不管它精彩與否，人們都會很興奮地觀看這場比賽。」

如果還要最後強調一下最重要的點。那就是對於聯盟和球迷來說，湖人隊對雄鹿隊的比賽將是一個有意義的時刻，這是常規賽往往缺少這種時刻。對詹姆斯來說，這可能只是「相當酷」的一場比賽，對他的隊友來說，這可能也沒有什麼特別的意義。這是82場常規賽中的一場比賽。

但是在媒體（和社交媒體）眼中，這很重要。至少還是有一點重要性的。如果湖人隊在印第安納帶走一場勝利，而同時雄鹿隊拿下了達拉斯的比賽，這可能會顯得更具含金量。如果戴維斯能確保登場，比賽可能也會更有吸引力；然而，沃格爾說，戴維斯只是「有可能」上場。

儘管如此，這很有可能是明年6月份NBA總決賽的一場預演賽，現在這場比賽的結果也具有特別的意義，那就是獲勝者將在12月份成為聯盟中保有最佳戰績的唯一一支球隊。這就真的是非常重要了。對嗎？

「誰在乎這些？」霍華德說。「沒人在乎。隨便誰說我們什麼都沒關係。我們知道我們必須夜以繼日地去做什麼才能贏球。我們知道這對每個人來說都是一場重要的比賽，但即使贏下這場比賽也不會讓我們贏得總冠軍。」
