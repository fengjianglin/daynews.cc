北京時間12月21日上午8:30，76人將在主場迎戰獨行俠，盧卡-東契奇將在費城與獨行俠會合，他會在這場比賽中復出嗎？

視頻直播

比賽性質：NBA常規賽

比賽對陣：獨行俠VS76人

比賽時間：2019-12-21 星期六 9：00

直播地址：點擊觀看

預計首發

76人：西蒙斯、理查德森、哈里斯、霍福德、恩比德

獨行俠：布倫森、小哈達威、史密斯、波爾津吉斯、鮑威爾

交鋒戰績

兩隊過去5次交手，76人戰績3勝2負。雙方最近一次交手，4月2日76人客場102比122落敗。

76人

76人上一戰主場104比108輸給熱火，主場不敗金身告破。恩比德得到22分19籃板，連續4場比賽打出20+10數據。托拜亞斯·哈里斯得到20分4籃板，過去5場比賽有4次得分20+。西蒙斯上一戰得到17分6助攻。76人近期進攻火力有所下滑，最近6場比賽場均得到105.2分，低於球隊本賽季場均得分109.1分。76人全員健康沒有傷兵。

獨行俠

獨行俠目前18勝9負排在西部第5位，場均得分117.6分，場均助攻24.6，場均籃板47.9，場均失分109.2分，場均失誤12.1。上一場比賽獨行俠在主場以103-109惜敗於綠軍，顯然，東契奇的缺席給球隊帶來了不小的影響，雖然波爾津吉斯全場拿到了23分和13個籃板，杰倫-布倫森拿到10分7籃板11助攻的兩雙數據，但遺憾的是獨行俠最後還是輸球。

恩比德大戰波神

在東契奇缺陣期間，波爾津吉斯成為獨行俠的主要進攻火力點，過去三場比賽他都打出兩雙表現，過去四場場場得分20+。恩比德上一場比賽拿到22分和19個籃板，那是他本賽季的第18個兩雙。相信恩比德和波爾津吉斯的表現，將在很大程度上決定比賽的走勢。
