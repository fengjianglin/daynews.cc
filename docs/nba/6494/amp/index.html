衛冕冠軍還有幾成功力？13勝4負爭當第二，將第一中鋒釘上恥辱柱

天天要聞
3 週 ago

賽季至今哪支球隊是被外界低估的呢？走了新科FMVP萊昂納德，聯盟頂級「3D」格林的衛冕冠軍猛龍隊無疑是其中一支。

​賽季至今，猛龍以13勝4負的戰績與凱爾特人肩並肩列與東部第二之位。如果說猛龍血洗聯盟倒數第一的尼克斯28分，是實力的碾壓，那麼，迎戰聯盟赫赫有名的「死亡五大」費城76人呢？

上賽季季後賽猛龍隊是憑藉一記三分絕殺在搶七大戰中送走費城76人的，還讓「比德大帝」淚灑現場，成為外界吐糟的話柄（唐斯就曾以此取笑過恩比德）。本場對決，儘管沒有了「卡哇伊」萊昂納德，恩比德領銜的費城76人恨不得撕碎這隻猛龍。

​「比德大帝」萬萬沒有想到，原本以為是場復仇戰的比賽，會成為職業生涯的「恥辱夜」。猛龍一開始范弗利特就接到小加索爾的傳球，三分直接出手命中，之後比賽完全進入了衛冕冠軍的比賽節奏當中，兩分鐘後，西亞卡姆左側45度角的迎射3分命中，猛龍打出8-0的夢幻開局，之後哈里斯，西蒙斯等人為球隊止血，穩住局勢。但是，西亞卡姆的狀態實在火熱，范弗利特上籃不中後，直接一個輕巧的補籃，個人進攻方面也絲毫不手軟，單打哈里斯一個輕巧的轉身換開防守人之後，舔籃命中，隨後，又一條龍上籃還造成「大帝」犯規。

​費城准五星的76人當然不會輕易放棄，憑藉韌性和天賦將分差迫近，將懸念拖進了末節。在末節還剩下最後一分鐘時，76人以96-94領先猛龍兩分，最後時刻西亞卡姆又站了出來，再次面對「大帝」打成2＋1，球隊重新奪得領先優勢，「大帝」又一次成了背景。

一度被譽為「第一中鋒」的恩比德，不僅在防守端被打成了背景牆，在進攻端同樣被掐成了「小綿羊」。在小加索爾的防守下，所有的進攻手段都不靈了。在32分鐘的出場時間內，11投0中，一分未得，13籃板算是唯一的生存感，可5次犯規也是令人唏噓不已。

32分鐘，11次出手，得到掛零，在歷史上也沒幾個能做到，尤其是全明星球員。當然，也成了恩比德生涯的「恥辱夜」。

​好消息是，隨後的一場，恩比德面對國王的比賽中，高效砍下了33分16板的數據，創下了另一個歷史第一。在砍下0分的情況下，下一場得分超過30分的紀錄。

儘管如此，也無法掩蓋「大帝」低迷這個信號。恩比得雖然能單吃聯盟中大多數的中鋒，但碰到小加索爾這種不懼身體對抗，且富有經驗的老將往往都會在低位吃癟，這一方面上將，很難與聯盟第一中鋒稱號相掛鉤了。

​球迷一句話說得好，上賽季「雙卡雙待」的猛龍是冠軍球隊，本賽季「單卡單待」的猛龍，同樣具備衝擊東決的實力。
