打國王三人出場40分鐘？6人輪換，德安東尼離下課真不遠了！

2019-12-25 650條評論 3,824次閱讀 2424人點贊

北京時間12月24日，又是一年的末尾，休斯頓火箭將客場挑戰國王隊，兩隊的實力差距顯而易見，火箭高居西部第三的位置，而國王隊三連敗後又被馬刺反超，已經來到了西部第十的位置。如果國王隊還想要留住季後賽的懸念，那麼就必須儘快的結束自己的連敗之旅，但顯而易見，面對休斯頓火箭隊不是一個好的選擇。

最終休斯頓火箭隊客場113-104擊敗國王隊。首先說下兩隊數據，火箭隊這邊，哈登34分5籃板5助攻，威少28分7籃板6助攻，特克5分10籃板4助攻，豪斯14分2籃板，卡佩拉15分14籃板2助攻，麥克勒莫8分1籃板，里弗斯7分。國王隊這邊，福克斯31分9籃板6助攻，希爾德10分3籃板4助攻，巴恩斯9分4籃板，巴格利11分6籃板，霍姆斯12分14籃板3助攻，阿里扎10分6籃板2助攻，別利察10分2籃板2助攻，約瑟夫9分6籃板2助攻，19-16！三人出場時間40+，哈登威少累壞了，又只剩6人輪換了。

福克斯一人的得分就比火箭全隊更多，但是看出場時間，打滿末節的福克斯，全場只打了36分鐘，這已經是國王隊全隊上場時間最多的球員了。但是火箭隊，一度領先25分的火箭隊，3人上場時間40+，分別是威少和塔克以及卡佩拉，哈登的出場時間也高達38分鐘。所以說哈登和威少才會這麼累。常規賽才過去三分之一，接下來會有更多的難關，如果保持這樣的輪換和出場時間，火箭依舊是重頭再來的結果。
