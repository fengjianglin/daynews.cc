唯一兩雙！三分升至43.1%，不止東契奇，獨行俠6年前眼光就很不錯

天天要聞
3 週 ago

正在進行的NBA常規賽里，新一周的比賽有許多球隊都打出了非常理想的戰績。

湖人目前穩坐西部榜首，榮獲8連勝！東部第一雄鹿也不甘落後，以8連勝還擊。

而排在雄鹿之後的熱火球隊也取得了新一周的第一場勝利，雖然在上一場比賽中輸給了擁有最貴首發的76人，但這東部第二的位置也還是保住了。

熱火在這一場對戰黃蜂的比賽中，展現出非常好的手感，無論是投籃還是三分都取得了很好的數據。

本場比賽熱火的投籃命中率為51.9%，三分為41.9%。巴特勒在這場比賽中出手11中6，三份2中0很遺憾，本場收穫21分3籃板7助攻1搶斷1封蓋。

納恩出手10次7中，三分6投5中，全場三分命中數最多！而阿德巴約本場出手14中8，收穫21分13籃板5助攻1封蓋。

值得一提的是，本場在替補席還有一位兩雙獲得者，他就是奧利尼克，他本場收穫15分16籃板1助攻2搶斷，作為替補席唯一獲得兩雙的球員，他的命中率也非常高！奧利尼克在本賽季出戰16次，場均獲得8.3分5.6個籃板1.6次助攻。

在本賽季的奧利尼克發揮有些不太穩定，但慶幸的是。他克服了這個困難，在熱火與老鷹的那場比賽獲得0分之後他就發生了改變，他的手感不佳就幫助球隊積極防守，他的想法很簡單，就是要幫助球隊取得勝利！

而在上賽季奧利尼克的三分命中率為35.4%，他在本賽季目前的三分命中率是43.1%。

才短短半年的時間就能提升一個檔次，並且在6年前進入聯盟就一直處於瓶頸，現在終於突破了。當然這位球員也證明了自己5000萬合同的價值！

凱里-奧利尼克是在2013年被獨行俠以首輪第13位選中進入NBA賽場。

他在賽場上的表現可圈可點，看來獨行俠的眼光可不止在東契奇上表現，現在還有一位奧利尼克啊。

畢竟這位球員是在6年前就被選中的，雖然只是13號秀，但現在表現出來的狀態還是對得起順位的名頭！

熱火目前的狀態能夠保持住就好，不過在首發陣容上，我認為可以另闢蹊徑，畢竟這位邁爾斯-萊昂納德最近幾場的表現實在是讓人大跌眼鏡。

不過還是希望熱火這支球隊保持住好的狀態，能夠帶來更多精彩的比賽就好了。
