爆冷！五星陣容遭吊打，替補席杜蘭特搶鏡，把歐文一把推進場內

2019-12-17 346條評論 1,376次閱讀 119人點贊

109-89，籃網主場大勝76人，以兩隊實力來開，76人絕對是全面佔優，畢竟今夏76人成功得到霍福德和理查德森，再成功續約哈里斯，加上恩比德和西蒙斯，妥妥的聯盟頂級豪華陣容，籃網雖然也得到杜蘭特、歐文和小喬丹，但杜蘭特本賽季因傷報銷，歐文新賽季也受困於傷病，籃網真實戰力並不強，面對強敵76人，籃網想要拿下對方可能性不大。

但事實呢？籃網大勝了對手20分，並且四節比賽76人從沒有一節佔據優勢，從比賽一開始，籃網就全面壓制76人，四節比分分別為28-23、29-20、26-23、26-23，可以說籃網整場比賽都是吊打76人，76人這麼豪華的陣容為何毫不反抗之力的？

最主要的一個原因就是恩比德這場比賽輪休，恩比德的作用毋庸置疑，缺了恩比德，76人就像是換了一支球隊一樣，攻防兩端都很難打開局面，就算面對實力明顯不如自己的籃網也是如此，內線打不開局面，外線三分失准，致使76人兵敗如山倒。全場比賽，76人只搶了38個籃板球，籃網搶了51個籃板球，在內線的拼殺上，76人是完敗，三分球76人26投只有5中，攻防兩端都暴露出很大問題。恩比德就像是76人的黏合劑，只要他在場，以他為攻防核心，球隊就能正常運轉，發揮出每一個球員的水平，76人也才是聯盟頂級強隊，但沒有了恩比德，76人各種運行不暢，爆冷輸球也就可以理解了。

全場比賽，76人一方西蒙斯20分5籃板3助攻2蓋帽、哈里斯托拜亞斯17分6籃板3助攻2蓋帽、理查德森11分4籃板、霍福德10分9籃板5助攻3蓋帽、斯科特8分、內托8分3籃板3助攻。籃網一方丁威迪24分6助攻、喬-哈里斯16分、坦普爾13分4助攻、小喬丹13分11籃板2助攻、普林斯8分6籃板4助攻2搶斷、艾倫8分8籃板、盧瓦烏8分5籃板、恩瓦巴6分。

籃網今天打得太順，坐在場邊觀戰的杜蘭特和歐文多次為隊友加油喝彩，在第三節還有最後3分多鐘的時候，籃網進攻，丁威迪喂餅，艾倫空接暴扣得手，場邊籃網替補席也直接炸鍋了，小喬丹蹦起來慶祝，歐文和杜蘭特也擠作一團，場邊位置太窄，大家都想要更大空間慶祝，杜蘭特毫不客氣，直接推了一把，歐文被推進了場內。

尷尬的是歐文差點撞到裁判，這是要強行領個技術犯規嗎？辛虧裁判沒有理會，歐文趕緊退回到替補席。杜蘭特這一把有點用力過猛了，是不是想要歐文趕緊復出打比賽啊？杜蘭特：你看看，你天天在場邊陪著我看球，球隊戰績反而越打越好，我都沒面子了，你趕緊復出得了，去幫助球隊繼續高歌猛進才是。

籃網這邊整場都是歡聲笑語，76人就有點慘了，被籃網這套陣容完爆，球隊還是存在很大問題，怎麼解決沒有恩比德在場時的漏洞呢？是不是主教練用人有問題啊？這麼豪華的陣容打得卻這麼掙扎，大家覺得呢？
