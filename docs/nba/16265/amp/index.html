能好好說話嗎？為爭鄧肯是不是超巨，籃壇名宿馬健與球迷又懟又罵

天天要聞
2019-12-14

12月13日，一位認證為「科比真話粉絲大咖」的網友——「想說卻還沒說DE還很多」，如此艾特籃球名宿馬健說：

「如果沒看見『不吹不黑』這幾個字，我真想給你兩嘴巴子」。

原來，在其所附的一個小視頻中，是馬健曾經參加的一檔《不吹不黑》的籃球節目，在與其它嘉賓爭論時，馬老師堅稱「鄧肯一直算不上是超巨，沒法跟詹姆斯相比」的觀點，於是，可能這位是鄧肯粉絲的球迷，就前來狂懟馬健了。

不過，面對這位網友無禮之言的「給兩嘴巴子」，馬健顯然惱了，他回復稱：

「你除了在網上吹點兒牛B，估計到我面前跟傻B一樣，可能話都不敢說甚至尿褲子啦……」。

——確實，雖然馬健有了一把年紀，可是作為體育人，且就憑其如此高的個子，真到了面對觀，還真有可能嚇得鍵盤俠們先尿濕了褲子吧？

不過，到底曾拿到5座總冠軍獎盃，3個FMVP的鄧肯，這位NBA聯盟歷史上的「第一大前鋒」，算不算是一位「超巨」呢？其實，在不同球迷的認知中，或只是仁者見仁、智者見智的事吧？

或可能，鄧肯這位顯得太過老實的球星，在名聲、市場、形象等的開發與影響力上，與公認的「超巨」，如喬丹、科比、詹姆斯等相比，會略有爭議——或許，這正是馬健這位籃球專家會有如此觀點，並不認可鄧肯是「超巨」的原因吧？

首先，這次球迷與馬健的罵戰，顯然是這位科比的粉絲所引發——你竟要給人家馬老師兩大嘴巴，這顯然是不對的。

真理或共識，難道是靠打出來的嗎？如果通過打，才能讓別人同意你的觀點，則這樣的科比粉絲，你也太強權主義了吧？幸虧，這位球迷還不是軍事迷啊！

馬健老師因此節目得罪了很多鄧肯的粉絲，但是，對於不同觀點，在網上仍可以論戰，只不過，絕不能是線下的「實戰」——因為觀點不同，就要對異於自己的人動手，這類暴力思維者，其實在現實中最可怕，也最危險！

當然，馬老師作為籃球名宿，面對如此眾多的鍵盤俠，估計也應司空見慣了吧？您老是不是都要給一一罵回去呢？當馬老師也是滿口髒話地回擊時，這是不是有點讓你自己的水準，也給拉低到了跟鍵盤俠一樣的層次呢？

所以，作為公眾人物、名人的馬健老師，還是應以講理為主。有了真正的說理，在有理有據之下，還是能說服大多數人，哪怕是和而不同，也不錯嘛——對於較少數的網路噴子們，其實對待他們最好的武器，就是根本不要理會他們：這樣做，則足以表明：噴子們，你們算個毛啊！瞧，我根本看不起你，所以才不理會你們呢！

因此，無論是網友們，還是名宿的馬健們，都請好好說話吧！
