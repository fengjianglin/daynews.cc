詹姆斯：揚尼斯每年都在進步，雄鹿引入了適合他的球員

2019-12-18 566條評論 3,057次閱讀 842人點贊

虎撲12月18日訊 湖人今日102-105客場不敵步行者，終結7連勝。賽後，湖人前鋒勒布朗-詹姆斯接受了採訪。

當被問到下一場與雄鹿的對決是否感覺很特殊時，詹姆斯說：「不，我都打了17年了，像這樣一場常規賽，只是對陣的兩支球隊都希望提升自己。不，它的感覺並不特殊。」

談到雄鹿前鋒揚尼斯-阿德托昆博躋身頂級球員的過程，詹姆斯說：「他進聯盟多少年了？7年。我不確定，他上賽季贏得了MVP，前一季他也很棒，在那之前也挺不錯的，大概第3個還是第4個賽季的時候。他天賦異稟，他有能力搶到籃板後持球推進，傳球，得分，突破，為防守施加許多壓力。自從進入聯盟後 ，他每年都在取得進步，他們在他身邊搭配了許多合適的拼圖，最大化發揮他的天賦。就是他們的管理層做得很棒，引入了適合他球風的球員，那是最重要的一點。我認為揚尼斯每年都在進步，但球隊圍繞他建隊引入特別適合他比賽風格的球員，這樣他才有條件去取得進步。」

本場比賽，詹姆斯上場35分鐘，20投8中得到20分9籃板9助攻。北京時間12月20日早9:00，湖人將客場挑戰雄鹿。

(編輯：Oceans)
