“富富”得正！富兒子魔術重生記，1億雙控打造東部黑馬

2019-11-29 0條評論 13次閱讀 0人點贊

真是樹挪死，人挪活。富爾茨連續六場比賽得分上雙，戰勝騎士的第4節中段命中的關鍵三分，富兒子在魔術算是找回自我了，至少有所期待。

在費城76人當狀元的日子，真的不好過，已經有了恩比德和西蒙斯這樣的天之驕子，當年留給富爾茨的機會更少，兩年加上受傷，他才打了33場比賽，差一點就毀在了那個球隊，這個球隊對於一些人而言也是多災多難的存在，恩比德和西蒙斯也是經歷了大傷之後歸來的，恩比德甚至好幾年都沒打球。

這幾年費城連續毀掉了幾個天才，包括此前的邁卡威，然後奧卡福，諾埃爾這樣的天才在這裡都遭遇了傷病，並且從高光的出道走向沉淪，都沒有用上多久的時間，富爾茨作為一個狀元在費城沒有被折磨瘋已經是幸事了。所以在「要未來還是要現在」的博弈中，最終費城放棄了富爾茨，但現在看起來就是雙贏的局面。

對於費城而言西蒙斯的存在是無法撼動的，即便他可能成為歷史上最不擅長也不會投三分的一號位，但是他也有可能打出魔術師那樣的表現，以及作為現在一號位的終極轉換終結者，西蒙斯在費城成為未來是毋庸置疑的。而富爾茨去了一個比較弱的隊伍可以得到更多的機會，這本就是他想要的結果，現在就是各需所需。

而富爾茨在魔術的日子戰績雖然一般，但是也好歹算是進入了東部前八的位置，有機會打一打季後賽。本賽季17場比賽富爾茨打了12場首發，場均11.4分生涯新高，雖然他的三分球命中率21%還是天愁地慘，但是好在是富爾茨在比賽中找回了那種大家期待的狀態，將對手晃出底線，投中關鍵三分，富有激情的衝擊力和轉換，這是富爾茨在進入nba之後真正意義上的一次開始。

魔術現在的球權都在富爾茨和福尼耶手裡，「富富得正」的組合雖然在聯盟的後場二人組不算顯眼，但是魔術現在進攻發起點的分配讓他們在本賽季的進攻端還是有了進步。福尼耶作為一個攻擊手和富爾茨的搭配中還是有了更多的空間去進攻，兩個人類型差不多，但是富爾茨更有衝擊力，而福尼耶更聰明一些。

​ 現在兩個人同時打出了生涯最好的表現，魔術現在也有了自己三角形，加上「富富得正」的組合之外，武切維奇依然穩定，富爾茨現在正在恢復自己的信心，這是他建立節奏的第一步，而這之後相信富爾茨會越來越好，畢竟他才21歲，魔術從費城得到了富爾茨自然相信他的潛力，而且魔術這些年通過交易改變命運的例子很多，所以富爾茨或許會在這裡真正的開啟自己的巨星之路。
