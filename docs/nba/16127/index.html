武切復出魔術八人上雙輕取鵜鶘，英格拉姆21+6+6

2019-12-16 802條評論 1,928次閱讀 386人點贊

尼古拉-武切維奇傷愈復出，魔術八人得分上雙，主場130-119輕取鵜鶘，結束三連敗，繼續以12勝14負的戰績排在東部第八位。

武切維奇因為腳踝扭傷缺席了此前的11場比賽，今天傷愈復出並首發登場。

上半場比賽，魔術僅以58-57領先1分，但他們在第三節開始發力，尤其是該節結束前2分22秒，鵜鶘連續吃到技術犯規，主教練埃爾文-金特里甚至因此被驅逐出場，魔術在替補控衛DJ-奧古斯丁組織下趁勢打出一波小高潮，以99-84領先15分進入最後一節。這一節，魔術轟下41分。

最後一節，魔術一直穩穩控制著場上局勢，沒給對手留下任何可趁之際，最終順利收下這場主場勝利。

武切維奇最終得到20分9個籃板，奧古斯丁貢獻17分8次助攻，喬納森-艾薩克21分11個籃板2斷1帽，魔術10人輪換當中只有阿隆-戈登得分不足兩位數。

鵜鶘方面，布蘭登-英格拉姆21分6板6助攻，朱-霍勒迪29分8助攻，JJ-雷迪克23分，約什-哈特也得到20分。
