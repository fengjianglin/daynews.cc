馬刺隊用波波維奇討厭的方式贏了快船，但是球隊前景依舊讓人擔憂

2019-12-01 0條評論 13次閱讀 0人點贊

11月的最後一天，聖安東尼奧馬刺隊迎戰洛杉磯快船。馬刺以107比97戰勝快船，給自己的11月份畫上了一個句號。

本場比賽的數據方面，馬刺隊六人得分上雙，其中懷特10投6中，三分球7投5中，砍下17分3籃板3助攻成為獲勝功臣，阿爾德里奇同樣獲得17分8籃板7助攻，德羅贊15分9籃板5助攻，福布斯12分，替補登場的米爾斯和萊爾斯也分別砍下13分和10 分，全員皆兵，每個人只要上場就會為球隊做出貢獻，符合波波維奇和馬刺隊一貫的風格。

另外本場比賽兩位替補球員的發揮起著至關重要的作用，分別是珀爾特爾和貝里內利，前者當晚被波波維奇放在替補陣容中，在與快船二陣容內線的纏鬥中取得了巨大成效，將哈雷爾成功限制住，使聯盟最強的二陣容今晚不再有優勢。貝里內利在第二節關鍵時刻一攻一助貢獻5分奠定了馬刺領先的格局，功不可沒。

快船隊方面本場比賽迎來卡治組合首秀，但是球隊整體效率不高，投籃命中率僅為39.13%，3分命中率為36.36%，球員集體啞火是今晚失利最主要的原因，四大主力萊昂納德、喬治、哈雷爾、蘆葦集體命中率下滑，四人加起來共56投18中，32.14%的命中率。輸球也是意料之中的事。

為什麼說馬刺隊贏球是以波波維奇最不喜歡的方式贏下比賽的呢，波波不止一次的表達出對三分球的討厭，然而本場比賽馬刺隊三分球34投14中，是本賽季命中球最高的一場，正是通過關鍵時刻的三分球表現才取得了勝利。

隨著聯盟的發展，比賽的節奏越來越快，三分球的戰術地位，得分比重越來越高，場均三分出手越來越多。相同次數的出手，35%命中率的三分球要比50%命中率的兩分球得分還要多，效率可見一斑。但是馬刺隊的場均三分球出手25.6次排名聯盟第20位左右，場均三分球命中數為9.1排名聯盟第18位，命中率為35.3%排名聯盟第18位。波波作為主教練並不推崇進行更多三分球出手和三分球戰術，隊內球星德羅贊和阿爾德里奇三分球能力實在有限，註定了球隊的主要的得分方式。

為什麼說馬刺隊的前景仍然堪憂？

  1. 首先，隊內氛圍恐要生變。整個11月份，可以說是這十幾二十年經過的最糟糕的一個月，本月馬刺隊總共經歷了17場比賽只贏了4場分別是勇士、雷霆、尼克斯和剛剛戰勝的快船，其中還包括了一次8連敗。球隊一旦陷入了輸球的境地中，媒體就會將各種矛盾衝突管你是真的還是假的曝光出來，最近德羅贊清空個人賬號上所有內容，穆里取關馬刺隊官媒，不論事情真假，原因如何，都將對馬刺隊很不利，甚至還有一些交易傳聞出現。這次贏球希望能夠帶來一些改變。
  2. 其次，球隊的球員配置。目前馬刺隊7勝13負排名西部第12位，即使勉強能夠進入季後賽也沒辦法在季後賽中取得突破。自GDP三駕馬車退役後，萊昂納德出走，馬刺隊只剩下阿爾德里奇和德羅贊兩位明星球員，在球隊關鍵時刻需要球員站出來的時候沒有一位讓人放心的球員。阿德雖然有一手漂亮的翻身跳投但是比賽中與對方內線肉搏，體力消耗巨大，同時容易被對方球員盯死，甚至最後時刻都無法接到球。德羅贊執行關鍵球的效果也不是很好，身高以及技術特點容易被限制。

3.波波維奇的執教方式。波波維奇想讓球隊處於一個平均主義的狀態，每個人都參與到進攻中，誰都可以獲得機會得分。不會讓某一個人去扛起球隊，得分都不會超過30。這種方式有好處但是也有壞處，關鍵時刻處理球不夠堅決，沒有英雄球的打法。

馬刺隊正處在一個非常難受的階段，以現有陣容很難在賽季中取得突破。是努力的爭取季後賽名額以延續連年進入季後賽的記錄還是推倒重建以獲得更好的選秀順位？馬刺是時候做出選擇了。順便提一下，20年前，他們擺爛選中一個小夥子讓他們擁有十幾年的爭冠實力。
