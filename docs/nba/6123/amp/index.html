被犯規不吹，走步也不吹？詹姆斯：這是我生涯里最糟糕的一個走步

天天要聞
2019-12-06

客場以121比96大勝爵士之後，詹姆斯終結了自己在鹽湖城的客場7連敗。但是相比比賽的結果，詹姆斯在比賽中的一個動作，在比賽結束後引發了廣泛的討論。

詹姆斯的這個動作出現在於爵士比賽的第一節，當時詹姆斯運球剛剛通過半場，隨後他左手持球走了三步有餘，然後又開始繼續運球。站在詹姆斯正面防守他的博格達諾維奇看得格外清楚，立馬高舉雙手向裁判示意詹姆斯走步，很多觀眾也注意到了詹姆斯的這個違例動作，紛紛從觀眾席上站起來大聲抗議。

不過，站在邊線附近的裁判沒有絲毫的表示，就任憑詹姆斯繼續運球組織進攻。「這真的是我所經歷過的最糟糕的一件事，可能也算得上是我在職業生涯中所做過的最糟糕的動作之一了，」詹姆斯在比賽結束之後說，「其實我當時本人都沒有意識到，直到中場休息的時候，我們教練組的一位助教給我看了當時的錄像，我才意識到自己幹了什麼。」

據詹姆斯介紹，他當時之所以停止了運球，是被另外一邊正在跑位和對抗的米切爾和波普吸引了注意力。「我當時正在思考，觀察場上的局面，我看到了KCP（波普）和多諾瓦（米切爾）正在對抗，」詹姆說，「我覺得自己當時已經做好了傳球的準備，而我的大腦感覺上突然卡殼了，就好像暫時掉線了一樣。」

本賽季NBA開賽至今，圍繞裁判的話題已經不是第一次出現，而詹姆斯此前也曾因為在一場比賽中一次罰球都沒有拿到而表達過埋怨。而這次詹姆斯雖然因為裁判的錯漏判而獲利，但他並沒有因此而得意。「我為裁判們要經歷這樣的事情而感到遺憾，他們可能因此要寫一些書面的內容或者類似的東西，但這個吹罰真的不太好，」詹姆斯說。

除了這個漏吹的走步之外，詹姆斯這場比賽中還有一個爭議鏡頭。那就是在比賽最後階段，已經被換下場的詹姆斯因為隊友的成功防守而歡欣鼓舞，他被拍到腳上沒有穿鞋，在場邊肆意慶祝。這個舉動，被爵士主場的解說員認為是一種不敬，也在網路上引發了討論。

詹姆斯在賽後也針對這件事給出了解釋。最近幾個客場比賽，詹姆斯都會把自己在比賽中穿的球鞋送給小球迷，所以在他被換下場之後，有兩個鹽湖城的小球迷就走到場邊，希望詹姆斯能夠把球鞋送給他們。詹姆斯欣然接受，脫下球鞋後還簽了名，將兩隻球鞋分別送給一男一女兩位小球迷。而針對網路上的批評，詹姆斯也做出了回應，表示自己並非對爵士隊不敬，只是在為隊友慶祝。
