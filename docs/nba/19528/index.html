那些曾被稱為第二個詹姆斯的球員，後來都怎麼樣了？

2019-12-16 323條評論 4,431次閱讀 623人點贊

​近日，ESPN本賽季第一次公布了賽季球員真實正負值，詹姆斯以RPM值9.7分高居榜首，並且以超高的優勢力壓第二名哈登和第三名字母哥。

詹姆斯出道已經17年了，「出道即巔峰」是無數球迷對於詹姆斯統治力的讚歎。詹姆斯在聯盟中數十年如一日般統治著NBA，在聯盟中，天賦成為了年輕球員參加選秀時一項重要的參考標準。

近幾年來，我們已經被媒體介紹過無數個「他是下一個可以成為詹姆斯的球員」，詹姆斯已經被眾多籃球媒體排在了NBA歷史第二人的位置上，而他還在繼續創造著歷史，流水的聯盟鐵打的詹姆斯。詹姆斯豈是那麼容易比肩的？下面我們就來盤點那些曾經被成為第二個詹姆斯的球員。

35歲還能飛

O.J.梅奧

O.J.梅奧2008年NBA探花秀，在孟菲斯灰熊隊開啟了自己的職業生涯。梅奧高中時期的表現就吸引來全美的目光，他在玫瑰山基督高中開始了自己高中籃球生涯。八年級的時候，梅奧場均能得到20.5分。他連續兩年率隊奪得俄亥俄州冠軍，本人也連續獲得州籃球先生稱號。梅奧新秀賽季就可以砍下18.5分3.8籃板3.2助攻的數據，但出道即巔峰，梅奧被不自律的自己所打敗。

曾經的梅奧

毫無意識梅奧是一名天才後衛，他在高中時期就被很多人稱之天賦可比肩詹姆斯，但是梅奧一直都是一個問題球員，他曾經和托尼-阿倫因為賭博在球隊的專機上大打出手，後來又多次和毒品沾邊，在2016年7月2日，梅奧因違反禁毒協議被聯盟取消比賽資格，禁賽2年，後來他沒有再能返回NBA，2019年5月，NBL聯賽湖南勇勝隊官方宣布簽下O.J.梅奧。

泰瑞克·埃文斯

泰瑞克·埃文斯是2009年NBA的4號秀，他在高四年級時參加麥當勞全美高中生明星賽就成功當選MVP，2008年埃文斯當選美聯社評出的賓夕法尼亞州年度最佳高中球員，後來他逐漸漲露頭角，獲得越來越多的關注。

曾經的全能新秀

埃文斯新秀賽季就可以砍下20.1分5.3籃板5.8次助攻，成為繼大O·羅伯特森、喬丹、詹姆斯之後，第四位在新秀賽季就可以砍下20+5+5的球員，在那時，他被很多球迷認為是後衛中的詹姆斯。後來因為球隊給他的定位不準確，再加上傷病的問題，埃文斯再也沒有回到自己的巔峰。在輾轉多支球隊後，埃文斯在灰熊隊找到了曾經的自己，但是好景不長。在2019年5月18日，泰瑞克·埃文斯因違反禁毒協議被NBA聯盟取消比賽資格兩年。

安德魯·威金斯

威金斯是2014年的NBA狀元秀，在一進入聯盟後，威金斯就和天賦一詞常常被捆綁在一起，根據當年ESPN透露出的威金斯的體測數據，威金斯的垂直彈跳高度達到了1.12米，超過了巔峰時期的喬丹和麥迪。因為出色的身體天賦和身體素質，威金斯在身體機能方面絲毫不遜色於當年初進聯盟的詹姆斯。威金斯在生涯開始時就常常拿來和詹姆斯對位比較。

威金斯突破

威金斯一直被詬病不是強硬的球員，他不喜歡身體對抗，在比賽中的也缺乏激情和專註度，常常被球迷們質疑他出工不出力。在今年夏天，他被森林狼名宿加內特狠狠的教育了一番後，本賽季威金斯開始覺醒，賽季至今他場均可以砍下25+5+3+1的數據，他的所有數據都在飛漲，而可怕的是，他的天賦還遠遠沒有被兌現。

本·西蒙斯

西蒙斯是2016年的NBA狀元秀，他因足部傷勢缺席2016-17賽季的全部比賽後，他在下一賽季捲土重來，2018年本·西蒙斯擊敗多諾萬·米切爾和傑森·塔圖姆榮膺2017-2018賽季年度最佳新秀。

西蒙斯因為出色的傳球能力和身體素質被視為是下一個詹姆斯。西蒙斯的天賦有多高？西蒙斯作為一名控球後衛卻完全沒有三分能力的情況下還能殺入全明星，並且生涯場均貢獻16分8.2籃板8.1助攻的全面數據。

西蒙斯還需要進步

西蒙斯傳球能力能力出眾，球風無私，常常可以突破在內線砍分，但是因為中遠投能力的缺失，他在場上會傷害到球隊的進攻空間，迫使同隊的恩比德作為中鋒拉到外線。西蒙斯天賦很強毋庸置疑，但是要想讓76人在季後賽走得更遠，西蒙斯必須提高自己的三分水平，否則76人在季後賽難以進一步取得突破。

蔡恩·威廉森

蔡恩因為匪夷所思的爆發力、力量以及速度被認為最像身體巔峰時期的詹姆斯，甚至可以說，他的身體素質的變態程度可以在一些方面已經超過了詹皇，因為他中鋒的體重在再加上出色的彈跳能力，他足以在任何人面前完成生吃。

蔡恩因過大的體重一直被人所擔心他的膝蓋，事實的確如此，膝蓋並不是鐵打的。因為膝蓋的傷勢，蔡恩被傳可能被「賽季報銷」。還有流言傳出，蔡恩絲毫沒有注意自己的體重，他完全不理會球隊的建議，體重別沒有下降。

強壯的胖虎

NBA狀元魔咒似乎開始應驗，每6年要出現一個水貨狀元。距離2013年本內特被選中後，蔡恩似乎又要重蹈覆轍。到底是跨時代的巨星還是又一個水貨狀元，我們只能通過時間來給我們答案。

NBA在一個籃球最高殿堂，在這個舞台中從來都不缺少天賦，詹姆斯在2003年進入聯盟後，年年都在打磨自己的技術，直到35歲的年齡，詹姆斯都在完善自己的技術。想成為下一個詹姆斯，你不僅要有他那恐怖的身體，更重要的是，你還有需要有他那日復一日的進取之心。

最怕比你優秀的人還比你努力！
