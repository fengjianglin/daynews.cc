綠軍“隊寵”的兩種選擇：最大區別就是他們對“團隊籃球”的理解

2019-11-28 0條評論 18次閱讀 0人點贊

感恩節前夜的兩場焦點戰分別在新奧爾良冰沙王中心和波士頓TD北岸花園球館打響，可遺憾的是更加吸引我的布魯克林籃網對陣波士頓凱爾特人「新舊寵兒」之爭卻少了一位主角…時隔半年，凱里-歐文不僅沒有代表籃網出戰「老東家」，沒能在花園球館上演「歐文vs.沃克」的兩位新老綠軍接班人之間直接對話，甚至都沒有跟隨球隊前往波士頓…不過雖然缺少了主角，但「歐文和沃克之爭」依舊成為了今天球迷討論的最大流量來源。

可能也正是因為球迷的支持？這場聚焦了所有球迷目光的「老東家之戰」沃克表現神勇，出戰31分鐘24投13中（三分球10中6）得到39分6個籃板4次助攻幫助球隊121:110大勝歐文的籃網，用實際行動回饋了球迷的厚愛。

歐文和沃克究竟誰更適合做球隊老大、究竟誰更適合凱爾特人是這個夏天圍繞著這兩位現役頂級控衛間最熱門的話題。而賽季戰至今日，至少從戰績上來看肯巴-沃克是【更加適合凱爾特人】的，狂喜的波士頓球迷對於兩隊現狀的滿意度更是溢於言表，賽前的觀眾席上早早來到北岸花園的球迷不僅手持著代表「沃克＞歐文」的標語，甚至還在比賽中大喊「歐文高名氣大於實力」…

對於凱爾特人球迷來說他們確實「有理由」、「有勇氣」做出這些舉動，因為在賽前看好這支綠軍的球迷並不多，畢竟不單單是球隊最重要的當家球星的「背離」，內線支柱霍福德的離開更是讓他們禁區變得「捉襟見肘」。而籃網雖然杜蘭特因傷無法出戰，但擁有歐文、小喬丹、丁威迪、勒韋爾、哈里斯、阿倫等一干實用球員雖然衝冠無望，可是戰績也應不至於太差吧？甚至在有了「加強版」拉塞爾的歐文之後說不定會有新的化學反應？

而現實卻出乎很多人的意料，籃網開局不順依舊遊走於季後賽邊緣，綠軍卻一路高奏凱歌甚至一度領跑全聯盟。也許這就是「傳說中」的化學反應吧？那麼為什麼沃克入主綠軍之後會比上個賽季歐文的綠軍更具競爭力？對比歐文，沃克究竟有哪些不同點讓他更適合這支球隊？

不粘球更容易融入團隊籃球

史蒂文斯的執教風格想必大家都很清楚，有兩個特點：第一是防，以鐵血著稱；第二是攻，以團隊配合帶動。防守端我們暫不討論，因為實際上雖然他們的防守不弱於上賽季，但和歐文、沃克都關係不大…

而在進攻端很多球迷覺得沃克助攻數都還沒歐文多，所以認為歐文應該比沃克更適合團隊籃球？其實不能只看助攻方面的數據，因為助攻、潛在助攻或者助攻傳球比等數據是判斷一個球員串聯質量的數據，而要判斷某個球員或某隊是否屬於團隊型應該看：①出球率；②球員平均每次觸球時間；③球員每次觸球後運球次數。

比如在我上一篇分析開拓者的文章中提到的勇士隊，自有數據統計以來的過去7個賽季他們場均出球率高於聯盟平均值、球員平均每次觸球時間都短於聯盟平均值、球員每次觸球後運球次數都少於聯盟平均值。所以過去的勇士給我們最大的印象除了三分以外就是快速、行雲流水的傳切球配合。

我們再看看兩人在凱爾特人時期的數據對比：

  • 上賽季歐文場均63.1次成功接球、51.3次成功傳球，出球率【成功傳球次數除以成功接球次數】是81.3%；而本賽季沃克場均成功接球60.5次、成功傳球49.5次，出球率81.8%。實際上出球方面數據相差並不大，但是在「控球」方面，兩人的數據差就大了。
  • 上個賽季歐文場均觸球次數達到76.8次，本賽季沃克是73.7次，不過因為都是控衛，後場接球數和其他位置球員會有所區別，所以我們以前場接球數來判斷他們的「粘球程度」。歐文場均31.1次前場觸球，球隊場均前場觸球229.8次，歐文佔了球隊13.5%的前場球權；而本賽季沃克場均前場觸球次數是25.5次，球隊場均前場觸球219.8次，他只佔了球隊11.6%的前場球權，兩人前場「球權率」相差了將近2%。
  • 同樣只計算前場數據的話歐文場均觸球時間4.04秒，每次運球次數3.34次；沃克場均前場觸每次觸球時間3.49秒，每次運球3.10次。

我們看直觀的助攻數據確實感覺歐文上賽季6.9次場均助攻明顯優於本賽季沃克的4.6次，可實際上並非那麼簡單，首先控球多了、觸球多了自然出【潛在助攻】的機會也多；第二如今的沃克因為身邊年輕球員的成長可以不需要像上個賽季的歐文那樣「集大權於一身」，他可以視對手防守的變化在進攻發起者、中轉員以及終結者三個角色中來回切換，他在一場比賽中的任務不會像上個賽季的歐文那麼集中，這對於現聯盟的新型「攻擊型控衛」來說是一大福音，比如庫里就是如此，當本賽季球權任務強加在他身上時我們發現不僅勇士的流暢程度大打折扣，他自己的效率也下降得厲害。

而這種「權利」的釋放實際上正是歐文最缺乏的，因為他不善於融入年輕人的團隊籃球。

團隊籃球或許只是簡單的信任隊友

沿著前文提到的「團隊籃球」這四個字、沿著歐文的助攻數據，我查詢了他和沃克的數據對比發現一個很有趣的現象，凱爾特人上個賽季留守隊里的幾大主力球員走了一半，留下的都是他們最大的希望、最年輕的潛力股，尤其是以塔圖姆、杰倫-布朗和斯瑪特為甚。但是這三人卻沒有分到多少歐文場均51.3次成功傳球和6.9次助攻的「羹」！特別是對比起沃克來說，簡直差得有點離譜，就像是實行了兩種【對待年輕人政策】…

先來看一張我的統計：

這是一張對比18~19賽季歐文和本賽季的沃克【實際助攻】以及【成功傳球】給斯瑪特、布朗、塔圖姆三人的數據圖表。每一個條形柱狀左邊數據是助攻數，而右邊數據為成功傳球數，可以看到，除了歐文實際助攻給塔圖姆的次數高於沃克以外，其餘任何數據都低於沃克許多。而歐文之所以實際助攻塔圖姆數據高於沃克主要原因在於塔圖姆上賽季命中率高了將近5個百分點…換句話說要不是塔圖姆本賽季「鐵」了，歐文和這三位的「場上交流」遠不及沃克！

不誇張，真的可以說是「遠不及」，因為如果你再看看他們接歐文成功傳球佔比的話就會發現這個差距之大，特別是和布朗之間，幾乎不來電。

  • 上個賽季歐文場均成功傳球數是51.3次，可是成功傳給這三人的比例分別是14.7%（斯瑪特）、6.3%（布朗）和15.0%（塔圖姆），這三人合計只佔了他成功傳球數的36.0%；助攻佔比分別是7.4%、8.5%和15.9%，合計比例是31.8%。
  • 而本賽季沃克對這三可是「恩寵有佳」，場均49.5次成功傳球，但是傳給他們三人的比例分別是23.8%、14.9%和18.4%，合計比例高達57.1%；助攻佔比分別是13.2%、19.1%和16.2%，合計比例高達48.5%。

【在同時出場的時間佔比上雖然也會有差異，但是這和時間關係並不大，主要是和布朗同時在場時間差距大了些，和斯瑪特、塔圖姆幾乎出戰時間差不到5%】

成功傳球數佔比相差21.1%，助攻數佔比相差了16.7%。雖然都傳言上個賽季綠軍更衣室矛盾頗多，但是從這些數據來看除了因為霍福德的離開使得綠軍更多倚仗外線這幾位新人讓沃克和他們有了更多交流外我想最根本的原因還是在於兩人對待新人的態度不同吧？也許是不信任？也許是年輕人不配合？

歐文的單打能力雖然毋庸置疑，但是對年輕人的不信任是造成歐文上個賽季「形單影隻」成績不佳的最關鍵因素。這個賽季他們的外線「理論上」來說比上個賽季要更弱些，畢竟還走了個羅齊爾，可是他們在外線的配合效率提高了一個台階。這不僅僅是年輕球員的成長帶來的收穫，我覺得更多的是因為沃克懂得如何調動起年輕人的衝勁，不需要的時候打下手，需要的時候再衝鋒陷陣。

更好的團隊籃球是創造更好的進攻模式

歐文單打能力是否頂級應該不用我再問了，但是我如果問歐文的出手選擇（並非指最後關鍵時刻非他出手的情況）是否也是頂級呢？可能很多人還是會回答我說「是」？其實我也會這麼回答…因為出手選擇得看是誰出手，像歐文那些高難度的上籃，你換現役的任何一個人去我都會覺得——這麼浪費球權真的好嗎？可歐文就是能擱進去！

但我還是得說一下，如果考慮到球隊其他隊友的話，在非關鍵時刻或許歐文還有更好、更合理的處理球方式，或者以他的能力可以打出更多的戰術更優的團隊籃球。

舉個例子，歐文的突破能力現役聯盟說第二沒人敢說第一吧？他上個賽季場均11.2次突破也排在聯盟前茅，可是場均突破後的成功傳球數只有4.0次，實際助攻數只有1.0個；而本賽季沃克同樣是場均11.2次突破，不過人家突破後的成功傳球數達到4.2次，實際助攻數1.2個。當然，這裡不是糾結那多出來的0.2次成功傳球和0.2次助攻，而是對於突破後【助攻的質量】的思考。

上個賽季歐文在綠軍突破後為隊友創造的機會質量很低（注意，這和他突破後自己得分效率很高沒有任何關係）！他上個賽季267次突破後的成功傳球隊友命中率只有26.2%，沃克雖然也不高可好歹達到了29.3%。也就是說在團隊籃球的基礎下打突破分球這一戰術上沃克創造的機會要好過歐文，如果你再考慮到場均次數上的差距的話。

歐文的突破威脅對於防守球員來說壓力一定是大於沃克的，可人家就是能夠利用突破打出比歐文價值更高的「團隊戰術效率」。或許這和歐文的性格有關吧？除了這個細節外歐文和沃克很大的區別就是自己的有球端「借用」團隊配合後的出手區別。

同樣是在綠軍，並且上個賽季他們全隊平均的擋拆、掩護頻率還要高於本賽季9%，可是上個是賽季歐文擋拆持球的頻率只有29.4%，PPP只有0.99分，你要知道史蒂文斯的戰術板上最多的就是利用擋拆以及交叉掩護配合的戰術，更重要的是上個賽季有霍華德、貝恩斯的綠軍質量可是遠高於本賽季的。那本賽季的沃克呢？43.6%的擋拆持球頻率還能保持1.04的PPP！

其實我們沒有理由也沒有證據能夠證明沃克比歐文好，但是不管是實際球隊戰績還是詳細數據分析、拆解，至少可以證明沃克更適合打團隊籃球也更適合凱爾特人。賽季開始前我雖然不是很看好籃網，但也不是很看好凱爾特人…因為我印象中沃克更適合做獨來獨往的終結者，他很少向我們展示在團隊籃球中的作用。我沒想到過他可以做發起者，可以做終結者，還可以做一支球隊在進攻端最好的「槓桿」，平衡了球隊進攻端有球、無球的能力，激發了史蒂文斯團隊籃球的最佳反應，而這也是早期綠軍王朝的建隊思想。

我崇拜團隊籃球，也一直試圖證明團隊籃球要強於「天賦籃球」，或許下次該做個聯盟團隊籃球和天賦籃球的詳細數據分析了。
