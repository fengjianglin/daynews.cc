能打的一個不留！湖人三少成功擠走球隊大當家，狀元怕是回不來了

天天要聞
13 小時 ago

早些時候，NBA這邊在這一天又將進行多場常規賽的比拼，當天的幾場比賽當中，有直播的就只有2場比賽，一場是獨行俠對陣雄鹿，一場是開拓者對陣太陽。

獨行俠對陣雄鹿還是很有意思的，本來獨行俠將會成為雄鹿連勝歷程上的一大絆腳石。

可他們隊內最核心球員東契奇在比賽當中把自己弄傷了，還不清楚到底多久才會回歸，不過比賽當中什麼都有可能發生，雖然獨行俠贏面不大，但也是有希望的。

而作為本賽季第一個拿到12連敗的球隊，鵜鶘這邊在當天並沒有比賽要打，正好利用這一天反思一下，不過他們也確實比較難。

這個賽季因為錫安不在，球隊的實力已經顯得有點弱了，賽程還對他們不利，近10場比賽當中，他們兩遇獨行俠，還和湖人，雄鹿以及76人對戰過。

所以說賽程的不利，和他們拿到12連敗有很大關係，而他們下兩場比賽分別對戰籃網和森林狼，鵜鶘勝算也不大啊！

不過最近鵜鶘那邊傳出來一則消息讓小編很是驚訝，NBA的名記Zach Lowe在一次訪談當中表示，鵜鶘可能會在交易日截止之前把霍樂迪和雷迪克送走。

雖然說霍樂迪想要留在鵜鶘，可是該記者卻表示，他的職業生涯已經到了一個階段了，鵜鶘如果要重建的話，他不會想留在這樣的鵜鶘隊里。

確實想不到鵜鶘會是這種打算，霍樂迪和雷迪克怎麼說也是現在隊內得分榜上的第2和第3位啊！

能打的一個都不要？這套路和尼克斯有點像啊！尼克斯或許將會把他們隊內的得分王莫里斯給交易走。

不過鵜鶘重建確實也是比較好的選擇，錫安不在，現在這樣的陣容又打不出成績，只能試著交易幾個對球隊有幫助的球員回來。

而他們隊內有交易價值的也就那麼幾個，鶯歌，哈特，鮑爾這湖人三少，以及霍樂迪和雷迪克。

現在看來鵜鶘是有培養新星的意思啊，不然也不會將霍樂迪和雷迪克交易走了。

這邊也是恭喜湖人三少成功擠走球隊大當家，這是鳩佔鵲巢啊！就是不知道霍樂迪和雷迪克最終的歸屬是哪支球隊了。

而爆料鵜鶘可能會交易走霍樂迪和雷迪克的記者也表示，自己認為雄鹿可能已經盯上雷迪克了，如果他進入交易市場，雄鹿願意交奢侈稅去拿下他。

而霍樂迪他沒有說，不過小編挺想看到霍樂迪他們三兄弟在同一支球隊打球會是一種什麼場景，就是不知道步行者那邊會不會選擇拿下霍樂迪。

不得不說狀元簽可真搶手，勇士尼克斯也就算了，現在鵜鶘老鷹以及騎士都不甘落後，勇士那邊是真的沒辦法。

鵜鶘這邊挺有意思的，把他們倆交易走，在培養鶯歌他們的同時還能有望拿個狀元簽，鵜鶘要是真拿到狀元簽的話就有點意思了。

那他們下個賽季隊內有可能直接有2名狀元在場上打比賽，當然也有可能選擇繼續培養鶯歌和錫安。

說到錫安，按最開始的消息，這段時間應該會應該復出才對，但最新的消息確實，到目前還沒有進行恢復性訓練，看來這個賽季是涼了，回不來了，下賽季再爭個最佳新秀吧！
