3連敗！湖人慘敗讓詹姆斯眉頭緊鎖，一大壞消息更令湖人蒙上陰影

2019-12-24 2,991條評論 4,136次閱讀 901人點贊

3連敗，湖人104-128慘敗掘金！上半場由於戴維斯的無解，加上掘金的外線手感的確不是很好，因此湖人在大部分時間還是處於領先的。但是到了下半場，當掘金外線都開後，陣容的深度也比湖人好，而湖人這邊只有戴維斯和庫茲馬兩個得分點，一對比差距就體現了出來，第三節掘金贏了11分奠定優勢。第四節哈里斯和比斯利又打瘋了，兩人連續得分一度將分差拉開到了23分。本以為沃格爾會放棄本場比賽讓戴維斯休息，但沒想到沃格爾也是很固執，不顧濃眉的傷病繼續派他上場，連帶著庫茲馬也上了，這也是令人十分不解，難道這場勝利如此重要？比球員的健康還重要？還剩4分31秒湖人依舊落後20分，沃格爾終於不再堅持將戴維斯換下。

數據：巴頓14分13板5助2斷、米爾薩普21分、約基奇18分6板5助、加里-哈里斯19分5助4斷、比斯利16分2板2助、普拉姆利10分5板5助、濃眉32分11板4帽、庫茲馬16分4板。

輸球不可怕，湖人也輸得起，但是一大壞消息卻令湖人蒙上陰影。那就是戴維斯在比賽中撞到了膝蓋，表情是很痛苦。雖然之後堅持上場，但是我們也看得出來戴維斯明顯緩慢很多，走路都不太利索了，因此這才是最大的麻煩。詹姆斯就有傷在身，而戴維斯一直就有傷，如今再添傷病，如果兩大超巨都不能打了，那真是難以想像湖人回變成怎麼樣？不會又像上賽季那樣突然崩盤吧？

本場比賽詹姆斯也來到現場觀戰，一開始老詹的心情還是很輕鬆的，因為沒有他，湖人開場打得還不錯。但是隨著比賽的進行，老詹額頭的皺紋是越來越多，眉頭緊鎖，越看越不對勁，這不是很上賽季的湖人很像么？更恐怖的是湖人下一場將迎戰洛杉磯快船，這又是一個硬茬子，貌似四連敗好像不遠了？
