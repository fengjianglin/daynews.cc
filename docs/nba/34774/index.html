第五順位，天之驕子？西部強隊等了他5年多了，終於被磨光了耐心

2019-12-25 3,368條評論 4,569次閱讀 2612人點贊

近日，據NBA多名消息人士透露，爵士與騎士達成交易協議，將艾克薩姆以及兩個次輪簽位送往克利夫蘭，換來到期合同球員克拉克森。這也是繼威少與保羅互換東家的交易之後，165天以來聯盟的第一筆交易。

而隨著這筆交易開始，或許新賽季的季中交易的大幕也即將拉開，艾克薩姆的交易，或將成為本賽季交易期暴風雨的前夜了。

作為澳大利亞的天才少年，艾克薩姆曾在耐克籃球峰會帶領世界聯隊出戰，以16分3籃板2助攻的成績帶領隊伍擊敗美國隊。在2014年在首輪第五順位被猶他爵士隊選中，艾克薩姆擁有良好的靜態身體天賦，身高1.98米臂展2.07米可以說這個身體條件在後衛當中相當的出色，在這個小球時代也是一名天生的鋒衛搖擺人，能夠輕鬆完成1-3號位的換防。

當時的爵士管理層也正是看重著這名球員極佳的身體天賦、換防彈性以及出色的得分能力，才將其選中作為球隊的未來所培養，在新秀賽季便給予他場均22.2分鐘的出場時間，然而他場均4.8分交給球隊一個糟糕的答卷。在此後的幾個賽季當中雖然偶爾有亮眼表現但還不足以匹配球隊給予他的出場時間，以及薪金合同。

對於爵士這一支老派風格的球隊，注重防守，生涯早期的艾克薩姆雖然在進攻端無所建樹，但是他憑藉著其出色的靜態身體天賦，出色的臂展，橫移速度較快，防守持球人時壓迫性較強，善於進行對防守人進行盯防纏鬥，其作為一名首發後衛在防守端的賽場表現還是平均水平以上的。

然而隨後艾克薩姆又暴露出了新的問題——傷病，右膝韌帶撕裂、腳踝、肩部的傷勢等等，「玻璃人」這是球迷們對他的一個標籤，反反覆復的傷勢影響著艾克薩姆籃球水平的提高以及對於球隊陣容的適應。

縱然艾克薩姆有著優秀的身體素質以及十足的潛力，但是一次次的傷病顯然正在拷問著球隊管理層，艾克薩姆真的值得繼續培養嗎？

「有心栽花花不成，無心插柳柳成蔭」，2017年手握第十三順位簽的爵士選中了多諾萬·米切爾，最初人們以為這只是一個速度較快，普普通通的年輕後衛，但是第十三順位出品的後衛又怎會普通？年僅一年級的米切爾聯手最佳防守球員戈貝爾，率領球隊闖入季後賽，首輪淘汰了被大部分人所看好的俄克拉荷馬雷霆，次輪落敗於火箭。

富有年輕活力，擁有即戰力的米切爾與多年傷病反覆的艾克薩姆形成了鮮明對比，球隊對於他的耐心也被磨光了，爵士將培養重心隨即轉移到了這個一年級新秀身上，同時為了加強後場火力的同時彌補防守，在今年休賽期球隊引進了准全明星後衛邁克康利。而艾克薩姆作為後場替補的第一選擇，反覆傷病最終被球隊搭上兩個次輪簽所交易。

交易而來的克拉克森雖然相較於身體天賦沒有艾克薩姆這麼出色，身高臂展有限，但是在騎士生涯的表現還算是可圈可點，場均出場23分鐘14.6分，作為一名志在季後賽球隊的第一替補後衛綽綽有餘，並且不容易出現傷病，這也許是爵士最為看重的一點，畢竟板凳輪換深度對於一支球隊也是至關重要的。至於艾克薩姆，這位曾經第五順位的天之驕子，未來何去何從，是否能夠重新追得上人們的期待，或許還需要一點點幸運女神的青睞。
