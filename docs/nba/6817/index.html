鵜鶘爛隊氣質太濃，除了要解僱金特里，還需這樣做

2019-11-25 0條評論 13次閱讀 0人點贊

作為一個資深NBA球迷，開賽以來也看了很多鵜鶘比賽，其中也發現了很多問題。小編覺得主要體現在這兩方面：

1、管理層

大衛-格里芬其實還是很不錯的，但是這個賽季的操作最迷的就是內線沒有擼來一個黑粗硬，直接讓鵜鶘內線變成萬人捅。所以鵜鶘陣容巨大缺陷被對手痛擊，不僅僅是主教練的鍋，管理層也要背。

2、主教練

我覺得此人真的是個大混子，前任管理層真的太菜了，竟然會放棄蒙蒂選擇金特里。他金特里有那水平嗎？他沒有，你硬拉上去這不是在亂整么。主要反映在這幾方面：

（1）就是沒有構建防守體系，防擋拆沒有辦法，並且防守站位亂作一團，經常出現防守換位撞在一起。

（2）紀律性執行力方面的要求，整個鵜鶘隊就是爛隊文化，毫無紀律可言，誰都可以單打，戰術分工不明確，你讓雷迪克傑克遜去持球打這不是為難別人么。該設計戰術的英格拉姆和雷迪克卻不涉及戰術。

（3）在二次進攻籃板上更是毫無布置，球在四個人範圍內都能讓貝弗利給搶了，不知道金特里平時怎麼跟球員講解籃板卡位與協調配合的。

（4）當然，最讓人擔心的還是金特里培養新秀的能力，今年的沃克跟海斯都是璞玉，但目前真沒看到天賦在哪，只看到了無紀律性的打球，莽。蔡恩不是那種全能型球員，組織不是特別出色，可以說，他的天賦都在身體素質上，而技術上太需要一個會培養的教練來精雕細琢了，而且鵜鶘未來有很多選秀權。

倘若不解僱金特里，那麼再多的新秀都會埋沒。而且新奧爾良不是大球市，很難吸引大牌。

所以說，鵜鶘不解僱金特里，不重組陣容，簡直就是在浪費蔡恩的天賦。至於英格拉姆，其實已經打出來了，放養都可以，所以鵜鶘怎樣對他都不擔心。真為蔡恩擔心，鵜鶘的爛隊氣質太濃了。

英格拉姆真沒遇到擅長防守的教練，所以單防那種靠節奏打得都還可以，但是對於速度快的真沒辦法，其次，防擋拆也是大災難。英格拉姆要想成為攻防一體的巨星，還是得遇到好的防守型教練啊！
