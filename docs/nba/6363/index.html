52+19+10！湖人王炸組合真儘力了，對比卡萊爾，沃格爾還是太年輕

2019-12-02 0條評論 15次閱讀 0人點贊

沒人想到，高歌猛進的湖人會倒在獨行俠腳下。今天早些時候結束的一場NBA常規賽，湖人主場不敵獨行俠，10連勝被終結。可能很多人都沒有想到，湖人主場、對手排名西部第五，最終卻迎來了一場14分分差的大敗仗。詹姆斯不甘心，濃眉哥不甘心，霍華德不甘心，但是那又怎樣呢，說到底只是一場常規賽。就像賽後詹姆斯說的那樣，球隊不會因此陷入沮喪，會從錯誤中吸取教訓，賽季還很漫長。

競技體育從來都有一個鐵律：贏球的時候一切都好，輸球的時候什麼問題都全暴露了。比如今天這場比賽， 湖人替補疲軟，內線保障不足，防守對方關鍵球員不力等問題，都被放大了。主教練只會用詹姆斯和濃眉哥組成的「王炸組合」，兩人合起來拿下了52分19籃板10助攻，真的儘力了。要知道，湖人今天全隊整場比賽也不過只得了100分！詹姆斯和濃眉哥既當爹又當媽，進攻防守一肩挑，還是換不回一場勝利。

其實真要說對手獨行俠有多強？不見得吧！獨行俠只有一個東契奇，雖然今年夏天續約了波爾津吉斯，但是波神畢竟是大傷回來的，已經不如當年在尼克斯那麼猛了，而且上一場面對太陽隊的比賽，波爾津吉斯全場8投0中只得到2分，近期狀態也不好。另外獨行俠還有誰？小哈達威，很多人都不認識的芬尼史密斯，以及鮑威爾、克萊伯、博班、巴里亞、賈斯汀傑克遜等人。

剛剛上文列舉的這些人，要真放在自由市場，不見得有多少球隊願意要，但是在名帥卡萊爾麾下，他們都是有用之人。2011年卡萊爾封神之作，就是重用特里、巴里亞，在總決賽擊敗三巨頭帶隊的熱火，讓人印象非常深刻。這位名帥控制欲極強，跟湖人替補後衛隆多還有過不愉快的合作經歷，但是偏執的性格背後往往是出眾的才華，卡萊爾就是這麼一個人，物盡其用，就是他的執教哲學。

反觀湖人隊主帥沃格爾，雖然當年在步行者，率領那支擁有喬治、大衛韋斯特、羅伊希伯特、蘭斯史蒂芬森、喬治希爾的步行者，也有跟熱火叫板的經歷，但是始終未能翻過熱火那座大山。現在沃格爾來到湖人，本賽季前19場比賽，湖人順風順水，沃格爾得到的誇讚也不勝枚舉，但真要說沃格爾錦囊里有多少妙計？不見得。

其實從今天的排名布陣，就可以看出，對比卡萊爾，沃格爾還是年輕。替補方面重用8投1中的庫茲馬，縮減霍華德的出場時間，用容易吃晃的麥基去頂防鮑威爾，都是下下策，最終也讓湖人付出代價。鮑威爾11投6中砍下15分9籃板，霍華德只得到12分鐘出場時間，湖人遭遇一場慘敗……這麼看來，詹姆斯和戴維斯，要衝擊總冠軍，除了要帶一帶隊友，還要帶一帶教練啊！
