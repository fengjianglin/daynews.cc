烏傑里做過多少神操作，值得尼克斯用兩個首輪換？

2019-12-17 2,316條評論 4,667次閱讀 2731人點贊

在所有人都以為倫納德出走會讓猛龍直接重建，但是2019-20賽季的猛龍非但沒有因為核心離隊變得分崩離析，還以17勝8負的表現排到東部第五，如果放在西部，猛龍可以排在西部第3。

猛龍可以在倫納德出走後繼續保持東部強隊的實力是因為有很好的陣容整體性，猛龍2019-20賽季得分上雙的球員有6人，出場時間超過15分鐘的有9人。全民皆兵的猛龍有很強的厚度，這也讓他們有實力去和任何一支強隊掰手腕。

猛龍能夠保持這樣的競爭力，和他們的總經理烏傑里的戰略眼光分不開，因為上賽季的猛龍隊史首冠也正是烏傑里一手策劃而成。也由於烏傑里的優秀，屢次崛起未果的尼克斯把引援的目光放在了制服組上，而烏傑里就是他們的目標之一。根據尼克斯隨隊記者報道，如果尼克斯想要讓烏傑里來掌管球隊，那麼他們需要給至少兩個首輪簽給猛龍，後者才願意放行。

烏傑里的經理生涯做了什麼神操作，值得尼克斯用兩個首輪簽交換？

送走安東尼，打造炮轟掘金

烏傑里在聯盟的第一份總經理工作是在掘金，2010年他接過了掘金總經理的位置，而在上任後不久他就做出了一筆震驚聯盟的三方大交易，把當時萌生去意的安東尼送到了尼克斯，同時還打包了比盧普斯等4名球員和一個2016年的首輪簽。

這筆交易在當時看來是讓重建的象徵，但是換來了加里納利等優質球員的掘金開始全民皆兵，在縮水賽季打出了38勝28負的表現，排在西部第6。而當時的掘金和現在的猛龍有異曲同工之處，2011-12賽季的掘金有6人得分上雙，也挖掘出了不少有實力的角色球員，像科里·布魯爾、魯迪·費爾南德斯、莫茲戈夫……

在2011-12季後賽首輪，這支平民掘金還和當時如日中天的湖人首輪打到了搶七。

2012-13賽季的掘金，烏傑里進一步通過交易升級了掘金的陣容，送出阿弗拉羅和艾靈頓，加上2013年的次輪和2014年的首輪，從76人換來了合同年的伊戈達拉。而伊戈達拉的加入將掘金的炮轟進攻威力提升了一個層次，2012-13賽季的掘金打出了106.1分聯盟第一的場均得分，57勝25負的戰績也是西部第3。

因此，烏傑里也拿到了2012-13賽季的年度最佳經理。但是這支掘金在季後賽首輪被勇士下克上，而烏傑里也在2013年的夏天轉投猛龍，開啟了另一段神奇歷程。

重用德羅贊、洛瑞雙核，打造北境之王

烏傑里加入猛龍後也迅速地通過交易來重整球隊，首先他將義大利狀元巴尼亞尼送到了尼克斯清理空間，又把當時球隊的二號得分手魯迪·蓋伊交易到國王，確定以德羅贊和洛瑞為核心打造陣容。

烏傑里的兩筆交易從現在回看極其明智，不但甩掉了兩份大合同，同時也打下了猛龍崛起的基礎，如果猛龍當時繼續信任巴尼亞尼和蓋伊，那麼也許不會有後來的崛起。

雖然2013-14賽季的猛龍在季後賽首輪搶七輸給了籃網，但烏傑里的重整陣容讓猛龍有了明確的陣容核心，而德羅贊和洛瑞日後也都成長為全明星首發後場。

目光如炬，烏傑里選人有一套

烏傑里為猛龍帶來了不少被低估的球員，比如2014年他用約翰·薩爾蒙斯換來了諾蓋拉和路·威廉姆斯；在猛龍的路·威廉姆斯立刻綻放出作為超級第六人的潛質，打出了場均15.5分的表現，成為僅次於德羅贊和洛瑞的第三桿槍。

2015年，猛龍又以首輪第20順位選中了德隆·賴特，這名後衛日後也成為了猛龍重要的替補，也是馬克·加索爾交易中的重要籌碼。

2016年，猛龍在選秀大會收穫頗豐，首輪9順位選中奧地利中鋒珀爾特爾，27順位選中了現在的基石球員西亞卡姆，和落選秀范弗利特簽約。

珀爾特爾除了在2017-18賽季成為球隊實用的替補中鋒，也在倫納德的交易中起到了大作用；而西亞卡姆現在已經成長為24.7分8.3籃板3.6助攻的全明星球員，范弗利特也是場均18分7助攻的球星。

兩筆交易，鑄就猛龍隊史首冠

說到烏傑里的神操作，那麼倫納德的交易不得不提，雖然後來球迷都為烏傑里歡呼鼓掌，但在這筆交易剛剛出爐時，為了一個沒有續約保障剛剛從大傷中恢復的倫納德交易走球隊旗幟德羅贊讓烏傑里背負了許多的罵名，彼時烏傑里成為了多倫多球迷的公敵。

還有用年輕的瓦蘭修納斯換來34歲的馬克·加索爾也讓許多球迷質疑——如果猛龍沒有拿到冠軍，那麼馬克·加索爾3000萬的年薪又會成為猛龍的毒藥。

但最後事實證明烏傑里是對的，倫納德的交易完全物超所值，而馬克·加索爾也沒有因為年齡而實力滑坡，他在季後賽對武切維奇、恩比德的有效防守為猛龍做出了重要的貢獻。

現在看來，倫納德和馬克·加索爾的交易正是猛龍隊史首冠的基石。
