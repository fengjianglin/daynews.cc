小托馬斯怒噴球迷有理？聯盟一判決惹爭議，球星的尊嚴誰來挽救？

2019-12-23 892條評論 2,400次閱讀 805人點贊

自從小托馬斯離開綠衫軍之後他也成為了標準的流浪漢，雖然他仍然活躍在賽場上，但是他已經沒有昔日「地表最強175」的風采，雖然本賽季數據有所提升，但是已經成為了不折不扣的角色球員，在他職業生涯走向低谷的時候，甚至連球迷也開始欺負他了。

在12月22日結束的一場常規賽中，奇才以112-122不敵76人。當比賽進行到第四節還剩2分53秒時，奇才球員托馬斯在一次暫停期間，坐在了76人的觀眾席上，並與76人的球迷發生了口角，隨後小托馬斯被驅逐出場，這也讓人大跌眼鏡。

要知道球員與球迷發生衝突的時候，大部分被驅除的都是球迷，而這一次小托馬斯卻成為了被驅除對象，因此一時間也傳聞小托馬斯可能出現了極為不當的發言，賽後，小托馬斯在接受採訪時解釋了當時到底發生了什麼。

原來小托馬斯當時罰丟了第一次罰球，之後命中了第二個罰球，場下的一名76人球迷在小托馬斯罰球的時候對小托馬斯豎起兩個中指，連續說了三次『f**k you, b***ch』。而這恰恰是小托馬斯所不能容忍的，因此暫停期間他立刻去76人球迷的觀眾席找他對質。

不過小托馬斯並沒有言語激烈，他表示到，請你不要這樣不尊重人，首先你是個男人，然後你才是粉絲。這名粉絲隨即解釋道，「對不起，我只是想讓你罰丟。」

小托馬斯繼續與這名76人球迷理論，並告訴他，我知道費城球迷有多熱情，球迷的行為並不能讓我感到困擾，但是你言論非常糟糕，如果是大街上碰面，我就在你身邊，你肯定不會說這樣的話，你這樣的行為在任何層面上都是不合適的。我不應該不被尊重。球迷們可以想說什麼就說什麼，但是不要談論我的家人，也不要說那個詞。

然而就是因為這一段發自肺腑感言，卻被當值裁判認為小托馬斯故意找茬，他也被驅除出場。或許裁判並不知道小托馬斯受到了如此待遇，但是在球員受到來自球迷不尊重甚至惡劣辱罵的時候，顯然很難處變不驚，實際上當年齊達內就是被對方問候家人之後才暴起用頭撞人。

或許，小托馬斯因為這一行為還要面臨罰款，但是在裁判在保證球迷安全的時候，球星的尊嚴又該由誰來拯救，小托馬斯肯定不會是個案，或許在這種現場發生的時候，球星有向當值裁判解釋的機會才更人性化，而不是看到衝突就立刻驅除出場。
