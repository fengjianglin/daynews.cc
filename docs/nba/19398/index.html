能投善突，助攻能力強！場均18+6的榜眼算不算天賦都點滿了？

2019-12-17 1,974條評論 3,888次閱讀 2822人點贊

剛進入NBA的第二年東契奇與特雷楊就已逐漸步入聯盟頂級球員的行列，場均得分都位於聯盟前五位。而今年的新秀狀元蔡恩威廉森還沒進入NBA就早已名聲大振，驚人的身體素質與爆炸的彈跳讓他當之無愧成為了狀元，但是今年的榜眼的就略顯低調，可是灰熊相信自己的選擇，交易走了康利，扶正了榜眼莫蘭特。

於是，就有了本賽季的狀元威廉森受傷，而身為榜眼的莫蘭特就幾乎鎖定了最佳新秀。本以為同為杜克雙星的探花巴雷特會是另一位最佳新秀的有力競爭者，隨著賽季的進行，巴雷特的命中率在逐漸下降，也沒能達到人們期望的樣子。

相比之下，莫蘭特正在向灰熊隊和球迷們展現出他的天賦。

首先，身體素質爆炸，善於攻框，幾次驚人的隔扣讓媒體為之瘋狂，還有突破上籃絕殺的瞬間更是讓人印象深刻，總結為籃下終結能力強，善於突破，移動速度快，突破有威少方向發展的天賦。

第二，投籃能力對於新秀來說並不稚嫩。三分球、拋投雖不是聯盟頂級球員的水平，但是足夠了，三分出手不多，但有43.1%的命中率，這就已經超越很多人了，拋投雖然不是具有特雷楊的頂級水準，但是柔和的手感讓人們看到他多開的技能樹。

第三，助攻能力強。作為核心後衛培養，助攻能力必不可少，場均6.5助攻，聯盟第16，依舊是善於打擋拆順下可以助攻內線，也可拋投得分，具備中等以上後衛的傳球視野和能力，這是傳球的天賦。

本賽季的他場均18.8分6.5助攻3.1籃板帶領著年輕的灰熊隊，不僅是提速了，還讓灰熊隊的打法也發生了改變，對於灰熊管理層而言，本賽季的他們已經很成功了，隊內的年輕人都展現出自己的天賦，莫蘭特拋去不談，小加侖傑克遜、克拉克、狄龍都越打越出色，就在最近他們還擊敗了勢頭正勁的熱火隊，這支灰熊隊絕對未來可期。
