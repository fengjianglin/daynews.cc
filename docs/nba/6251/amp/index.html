生涯10次帶隊東決，詹姆斯9次闖進總決賽，唯獨這一次輸給了他

天天要聞
2019-12-04

在NBA生涯早中期的騎士隊和熱火隊時期，詹姆斯的球隊都是屬於NBA東部賽區，而在這一時期裡面，詹姆斯也是連續帶隊制霸東部聯盟。從05-06賽季第1次進入季後賽東部決賽，目前為止詹姆斯整個NBA球員生涯已經是10次帶隊東決，而其中9次詹姆斯都是順順利利闖進了NBA總決賽。唯獨在08-09賽季的時候，詹姆斯在東決遭遇失利出局，輸給了當時巔峰狀態的「魔獸」霍華德。

在這一年的NBA常規賽中，詹姆斯常規賽場均28.4分7.6籃板7.2助攻，投籃命中率48.9%。而且當時詹姆斯基本上打滿了整個常規賽，82場比賽詹姆斯僅僅只缺席1場，最終詹姆斯帶領騎士隊取得66勝的聯盟最佳常規賽戰績。到了當年的NBA季後賽裡面，騎士隊依舊是所向披靡，在首輪和半決賽全部橫掃對方球隊晉級，勢如破竹闖進了東部決賽。

但不曾想在東決的第1場比賽，霍華德帶領的魔術隊就給了騎士隊一個下馬威，首場比賽霍華德和劉易斯合砍55分20籃板，並且兩人合計是33投23中，單場投籃命中率非常高效，因此魔術隊以1分的微弱優勢擊敗騎士隊。儘管詹姆斯拿到單場49分的超神數據，但可惜還是未能扭轉乾坤取得系列賽首勝。

雖然在之後的第2場，詹姆斯帶隊拿到35分4籃板5助攻，並且成功戰勝魔術隊，將兩隊的系列賽比分扳成1：1。但是由於球隊整體實力上面的差距，魔術隊和霍華德還是笑到了最後，魔術隊通過6場艱難的東決系列賽對決，送給了詹姆斯NBA生涯至今唯一的東決失敗戰績。值得一提的是，當年魔術隊在NBA總決賽對手正是擁有科比的湖人隊，而由於這一次的東決失利，詹姆斯非常遺憾錯過了和科比的巔峰對戰！
