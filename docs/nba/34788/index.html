NBA1/3賽季頒獎之最佳新秀：莫蘭特一騎絕塵，熱火落選秀成驚喜

2019-12-24 526條評論 3,616次閱讀 572人點贊

2019-20賽季已經打了1/3賽程，球隊和球員們也慢慢進入了角色，是時候對賽季6項個人大獎分篇做一次初步展望了。

註：筆者結合了「誰應該拿獎」（純籃球因素）和「誰會拿獎」（媒體投票人傾向）兩大因素，每個獎項選出了三個候選人，但主要的選人標準是前者。由於比賽樣本過小，部分獎項的角逐並不清晰，因此排名不分先後順序。

賈-莫蘭特（孟菲斯灰熊）

本賽季主要數據：場均29.3分鐘18.2分3.2板6.6助1.2斷，回合佔有率27.7%，真實命中率55.0%

說到聊到莫蘭特生涯開局的內容，就離不開他得分產量和效率的結合，這也是看一名球員得分能力的最好指標。下表是2004-05賽季至今，所有新秀的回合佔有率（橫軸）和真實命中率（縱軸）表現。

莫蘭特已經在橫軸最右邊的團隊里了（球權最高），周圍的名字是東契奇、歐文、格里芬、米切爾和特雷-楊，基本都是當年最佳新秀的水平。NBA是個能者多勞的聯盟，小小年紀能扛得分重任的品質是非常寶貴的，有聯盟平均的效率也可以接受。

不過莫蘭特大學時期射程並不好，在NBA也僅限於能回應空位三分的水平，三分產量並不高，如果開局超出預期的三分命中率（40.4%）保持不住，效率可能會回落。

至於傳控數據，雖然失誤不少，但莫蘭特能承擔大量持球的同時，還有不俗的助攻率，已經非常值得稱讚了。本賽季莫蘭特已經有5場10助以上的表現，同屆其他球員是0次…

可以說，莫蘭特在最佳新秀角逐戰里已經沒有真正對手，只要在剩餘賽季保證好出勤率和表現，拿獎幾乎是板上釘釘的。

肯德里克-納恩（邁阿密熱火）

本賽季主要數據：場均31.0分鐘16.4分2.8板3.6助1.0斷，回合佔有率23.5%，真實命中率54.5%

雖說莫蘭特有點一覽眾山小的意思，但熱火落選秀納恩的表現也值得一看。他是熱火超強淘寶團隊的又一力作，進入NBA立刻兌現即戰力，持球表現完全超出預期，運動能力和投射的結合，讓他交出了驚人的得分產量。除了球隊老大巴特勒，納恩23.5%的回合佔有率高居熱火全隊第二，成為外線重要主攻手。

納恩和莫蘭特表現相比也是有差距的，而除了這兩名球員…這屆新秀甚至都沒人有資格拿到最佳新秀選票了。本來我們還期待狀元秀錫安-威廉姆森，但他兩個多月不打球，就算復出後表現出色，總出場數可能也達不到評獎標準。至於其他高順位新秀，適應職業賽場速度極慢，好一點的球員也只是勉強能打替補的水平，還有不少爛到深不見底、不屬於NBA輪換水平的存在。
