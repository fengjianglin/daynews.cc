西部豪強集體輸東部，湖人沒比賽卻收穫好消息，這下總決賽穩了？

2019-12-16 1,064條評論 4,951次閱讀 3461人點贊

在昨日的比賽中，湖人與熱火苦戰到最後，終於成功從客場全身而退，收穫到一場酣暢淋漓的勝利。同時，球隊取得了6連勝，也破掉了熱火主場的不敗金身，繼續穩固住自己西部第一的位置。而在今日，雖然湖人並沒有比賽，但是西部的數支強隊卻「助攻」了湖人一把，集體輸給了東部，幫助湖人再一次鞏固了西部第一的位置。而昨日詹姆斯手肘稍微有些受傷，經過了冰敷之後，本人表示可以出戰下一場比賽，這對球隊而言也是個好消息，更重要的是，在今日的比賽中，湖人儘管沒有比賽，也接連能夠收穫數個好消息。

首先，在今天的比賽中，西部前五除了掘金以及無比賽的湖人之外，其餘快船、火箭、獨行俠全部輸給了東部。而第一個好消息固然是湖人與身後的球隊差距越拉越大，在快船獨行俠同時輸球後，目前湖人領先了快船4個勝場差，也意味著接下來湖人的壓力將會小很多，不需要因為穩固榜首位置而透支球員的體能。儘管今日東部第一的雄鹿拿到了歷史第9的18連勝，但是戰績依舊與湖人差不多，而周五的比賽，將會為大眾揭曉，在湖人面前，到底是否存在著所謂的強隊。

第二個利好的消息是獨行俠東契奇的傷病，當然，對於球員傷病，我們都是持祝福的態度，也希望東契奇能夠早日康復歸來，畢竟這歷史級別的二年級表演還未落幕，倘若能夠帶獨行俠越走越遠，那麼獲得MVP也不是一件不可能的事情。但是對於湖人而言，東契奇受傷，獨行俠的戰績自然會下降一些，因此，湖人在西部的優勢就越來越大，甚至能夠說，整個西部暫時沒有一支球隊能夠撼動湖人西部第一的位置。

而第三個利好的消息是看到了火箭與快船的實力，儘管此役火箭的威少與快船的倫納德都沒有上，但是同樣，他們的對手也都不是很強。活塞的庄神也沒有上場，在羅斯爆發砍下20+12時，活塞就這樣贏下了火箭。同樣，公牛也只是東部第11的隊伍，就算倫納德不在，也不至於被拉文絕殺吧。況且比較有趣的是，在被絕殺後，喬治說出了與去年被利拉德絕殺後一樣的對白。去年季後賽被利拉德絕殺後，喬治面對採訪時直言：「我覺得他那球不是一個合理的出手。」而在今日被拉文絕殺後，又說出：「我認為他那球走步了，但是裁判沒有吹」這樣的話，明顯對於對手取得勝利表示不服氣。

不過，對於湖人而言，看到這樣的場景也是件好事，畢竟不管是火箭還是快船，他們都不太穩定，這在一輪系列賽中是比較致命的事情。而湖人在面對強隊的時候基本都能獲得勝利，更不用説弱隊了。如此看來，湖人總決賽是穩了？也未必，這兩支球隊的爆發力也很強，倘若稍有不慎，很容易就會陰溝裡翻船，加上湖人目前隊內缺少具有外線爆發力的球員，而且一位良好的鋒線防守者也難找，面對這兩支球隊，還得看賽季中湖人有沒有更驚人的操作。
