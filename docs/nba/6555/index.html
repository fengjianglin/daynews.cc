事實證明格林根本不是核心，一部分球迷強行巨頭，是誰最清楚

2019-11-29 0條評論 17次閱讀 0人點贊

自從勇士在聯盟嶄露頭角以後的第二個賽季，格林就開始被某些球迷成為勇士的巨頭之一，他在水花兄弟在場的時候，經常會拿到三雙或者四雙的數據，讓人特別驚訝，從而連續三次入選了全明星，但是當潮水退去的時候，誰會最快顯形？那就是最吃體系的體系球員。

開賽至今，勇士已經打了19場比賽，但是戰績卻是15負4勝，聯盟倒數第一，格林在場之後效率奇差，更不用說助攻和防守，沒有了水花的牽制，格林打的異常艱難，防不住人，進不去球，失誤更是多，而且每次在關鍵時刻怒噴裁判，從而吃T，導致球隊輸球。

咱們來仔細看下格林的定位，在勇士隊伍，因為有水花兄弟，格林經常在拿到球以後下快攻，水花兄弟跑位，格林弧頂傳球，隨便一下就是一個助攻，為什麼?因為水花兄弟太准了，助攻經常隨便上十，在隊友防守出現漏洞的時候及時補上，進行協防，所以他的主要功能就是下快攻，協防補防。

16賽季以後阿杜加盟，格林減少了出手次數，其實也是他不太擅長投籃，從而主動放棄了投籃，對手研究出這個漏洞以後，直接放棄他的三分，直接去包夾庫里，因為庫里才是這個球隊的發動機和最強進攻點，從而聯盟就有了一個名言，「球隊可以輸，庫里必須死」，不為什麼，因為這是所有球隊得出的寶貴經驗和結論。

還記得快船季後賽第一場突然去包夾杜蘭特，沒想到杜蘭特得分慘淡，效率直線下滑，但是庫里卻打瘋了，狂砍36分，球隊二十分大比分贏下快船，第二場果斷換戰術，繼續包夾庫里，恩，結果你看到了，庫里效率雖然很高，但是球隊卻沒有輸，阿杜一個人的單打根本解決不了勇士輸球的問題，雖然球隊輸了，但是更加證明了這句話是對的，只要掐住庫里，球隊還是能贏一兩場的，如果不掐，只能被橫掃。

格林作為一個被放空的人，從而也就成了勇士最大的漏洞，今年沒有水花兄弟，沒有了伊戈達拉等老將，格林再也撐不起一個所謂的防守體系，因為他的缺點實在是太大了，他的數據就是建立在隊友的基礎上，他的速度和激情是優點，但是有的時候反而成為了缺點，而且他上場以後這群新的隊友，打球及其不適應，非常擁堵，打球也不果斷，他沒有拉開空間的能力，隊友來回奔跑，打的效率極低，所以上一場贏了公牛的比賽，格林出場時間只有24分鐘，因為他上場以後實在是太讓人無語，非常影響進攻。

所以說只有當潮水都退的時候才能知道誰在裸泳，想當年庫里13賽季，帶領上個賽季倒數第二的勇士直接進入季後賽第二輪，跟馬刺血戰六場，第一場就狂砍44分11個助攻，波波當時只能狂呼，彷彿看到了喬丹，那時候的格林菜鳥一枚，剛剛進入聯盟，一直坐冷板凳，可能都沒有想到自己一個二輪秀以後會成為一個所謂的巨頭，那個賽季的克萊是新秀第二年，科爾還在勇士，伊戈達拉還在掘金。就是那一年庫里打服了所有人，科爾由此看到了機遇，自己寫了一本庫里使用說明書，打動了勇士高層，其實科爾和庫里才是格林的伯樂，庫里不會介意你的能力不行，他會根據你來調整自己的打球方法，這就是勇士球隊無與倫比，橫掃聯盟的最大基礎。

所以，我們靜靜的等待水花兄弟的復出，勇士還是那個勇士！
