聖誕將至保羅贈全隊定製西裝！亞當斯個性十足，迪亞洛笑靨如花

2019-12-19 312條評論 2,108次閱讀 412人點贊

對於俄克拉荷馬州雷霆的小夥子們來說，聖誕節似乎提前到來了。

克里斯-保羅為他的每名隊友都定製了西裝，盡顯慷慨。而他的隊友們都非常開心地換上了保羅送的這一身禮物，甚至大聲表示，如果不好好地秀一番，簡直就是罪過。

首先，特倫斯-弗格森和納倫斯-諾埃爾：

弗格森回到了陣容中，他看起來已經準備好了。他這一身的格子拉長了他的整個身體，讓身體更顯修長。另一方面，諾埃爾的禮服襯衫更為端莊，但仍然引人注目，駝色西裝和褲子，看起來非常寬鬆，不失輕鬆活潑。

接下來是達尼羅-加里納利和丹尼斯-施羅德：

加里納里似乎有點措手不及，半是「放鬆」的手勢，半是對未來的憧憬，但這套西裝足夠嚴肅，足以彌補這一切。他這一身並沒有太多的閃光點，但有時候經典是最好的選擇。

另一方面，施羅德這一身則十分奪目。那件象牙色的高領毛衣上壘起吼吼一層，裡面的背心與夾克和褲子搭配得很好，再套上薑黃色的襪子和亮閃閃的流蘇裝飾的厚紅木色鞋。他的翻領上甚至還別著一枚小小的金別針，用來固定他的衣領。


邁克-穆斯卡拉和哈米杜-迪亞洛：

穆斯卡拉看起來不錯，這裡主要說說迪亞洛——迪亞洛臉上的表情基本上就是送他禮物的原因。他那能給身邊人帶來力量的笑容，好像他沒有其他方法來控制他所感受到的快樂。一件淡色的格子西服和一件樸素的石板灰色高領毛衣，卻顯得有那麼一點點違和……

謝伊-吉爾傑斯-亞歷山大和達里厄斯-貝茲利：

亞歷山大的裝扮小編實在不知道該怎麼吹，黑墨鏡、厚紅唇、藍褲子、棕色皮鞋……嗯，頗具時尚感，身材不錯。

貝茲利的一身條紋西裝與黑框眼鏡搭配起來就顯得舒服很多，再配上修長的身形和昂首挺胸的步姿，頗有種知識分子的感覺。

阿卜杜勒-納迪爾和史蒂文-亞當斯：

納迪爾的炭灰色襪子和脆白鞋搭配起來非常不錯。他的領帶有點歪，但亦或是故意為之，讓他看起來更顯有趣。當然，史蒂文-亞當斯是個奇蹟。亞當斯，這個被球迷戲稱為「海王」的男人，他穿這身完美的西裝就像穿人字拖和短褲一樣舒服。

你有沒有想過，一件海藍寶石禮服襯衫會好看，更不用說引人注目了？在亞當斯身上，這一點得到了完美的體現。在那滿頭亂髮、修剪過的鬍子和一條非常有力量的深藍色領帶的映襯下，史蒂文-亞當斯像聖誕樹上的星星一樣閃閃發光。

最後，克里斯-保羅：

保羅決定以這樣一種禮物來與隊友們慶祝聖誕節的到來。他這個賽季給雷霆帶來的不僅僅是一個定製裁縫，更是為一支大多數人認為今年將錯過季後賽的球隊帶來了出乎意料地改變。

在剛剛結束的比賽，保羅10投6中得到了18分+5次助攻，帶領球隊以122比126險勝孟菲斯灰熊，進一步鞏固了自己西部第7的位置。

如果在失去了保羅-喬治和拉塞爾-威斯布魯克之後，這樣的雷霆真的在被紅隊棄之不要的保羅的帶領下，成功殺入季後賽，甚至對陣上紅隊或者快船，那畫面，簡直太美啊……
