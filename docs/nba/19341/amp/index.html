22+6+8，只用28分鐘！恭喜羅斯，終於碰到了一個好教練

天天要聞
2019-12-18

北京時間12月17日，底特律活塞在主場迎來了華盛頓奇才的挑戰。活塞的戰績好於奇才隊，上一場比賽更是擊敗了休斯頓火箭隊，球隊的狀態十分火熱，但是格里芬在上一場比賽受傷，本場比賽無法出戰，德拉蒙德因為眼睛傷勢，依舊會缺陣，所以活塞現在基本上可以說是德里克-羅斯單核帶隊，這也讓本場比賽重新充滿了懸念。

德里克-羅斯在上一場比賽中，送出了12個助攻，在沒有格里芬的下半場，羅斯用自己的領袖能力幫助活塞隊拿下了勝利，再一次展現了昔日MVP的風采。本場比賽一開始，奇才隊就在客場瘋狂得分，第一節就拿到了40分，視活塞的防守於無物。不過好在活塞隊第一節也拿到了31分，分差僅僅只有9分。到了第二節，活塞在羅斯的帶領下，成功將分差縮小到了3分。

到了下半場，奇才隊依舊在不斷的得分，球隊進入了全民皆兵的狀態，儘管德里克-羅斯不斷的站出來命中關鍵球，分差還是被拉開到了10分。隨後羅斯站出來不斷的助攻隊友，三節結束的時候，分差回到了5分。第四節一回來，羅斯又不斷的得分，活塞成功將分差追到一球之差。隨後羅斯下場休息一會，活塞沒有讓比分擴大，等到羅斯回來之後，比爾卻徹底打開了，三分雨不斷下，分差也來到了兩位數，最終活塞119-133不敵奇才隊。

22+6+8！活塞雙塔缺陣，8年過去了，羅斯再次上演單核帶隊。在雙塔缺陣的情況下，德里克-羅斯就是這支球隊當之無愧的領袖，上半場羅斯的表現不溫不火，但是到了下半場，我們再一次看到了MVP時期的風采，他不斷的殺入籃下，不斷的串聯球隊，一如八年前，他帶著那支芝加哥公牛走上東部第一的寶座，一切恍如昨日。與其說活塞輸掉了比賽，不如說上帝偷走了羅斯的雙腿。

22+6+8，只用28分鐘！恭喜羅斯，終於碰到了一個好教練。本賽季的德里克-羅斯如此出色，離不開一個人，那就是主教練凱西。凱西是一個十分出色的教練，曾獲得過最佳教練的榮譽，在得到羅斯之後，凱西就對羅斯的使用十分謹慎，羅斯的上場時間也基本上控制在三十分鐘以內，但是決勝時刻，一定會放上德里克-羅斯。遺憾的是，凱西沒有在傷病之前遇上德里克-羅斯，不然也會是一段佳話。

花有重開日，人無再少年。德里克-羅斯的確回不到那個22歲的年紀了，但是玫瑰依舊會在每一個地方綻放，因為德里克-羅斯，從未向命運妥協。
