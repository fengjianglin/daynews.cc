拿著45萬美元的年薪 卻干著比韋德還強的工作 熱火隊這真賺了！

2019-12-20 3,017條評論 4,665次閱讀 1210人點贊

今天熱火又贏球了，而且贏的還是排名的直接競爭對手76人。東部競爭激烈，隨著凱爾特人和猛龍分別戰勝各自對手，直接將原本的第二名76人擠到了第五名，而熱火也只是短暫的在第二名的位置上停留了兩個半小時，轉眼就被凱爾特人趕走。

今天熱火獲勝的第一功臣當屬肯德里克.納恩，這位今年的「大齡」新秀，出場41分鐘，全隊18投9中，三分球8投4中砍下26分4籃板5助攻，正負值全場最高+19，最「難能可貴」的是只有一次失誤，沒有一次犯規！

作為一名落選秀，肯德里克雖然大學期間表現優秀，但由於參加選秀時已有23歲，年齡已經是偏大了，很多球探認為他潛力不大，加上又有對女性暴力的不良前科。他竟然落選了，雖然勇士短暫的給過他機會，但也僅僅幾場比賽之後就將他裁員。

但是金子總會發光，納恩在發展聯盟和夏季聯賽的優異表現引起了熱火隊的重視。他們開賽前給了他一份3年300萬的底薪合同，但第一年只有45萬的保障部分，剩下的要靠自己的表現爭取。

開賽的前5場比賽，納恩就讓整個聯盟眼前一亮，他總計得到112分。這個得分甚至超越了韋德和比斯利，成為了熱火隊史第一人。更是整個聯盟07-08賽季以後，除了杜蘭特之外，唯一拿到這個數據的人。

這樣的人，卻拿著這樣的白菜合同。萊利估計做夢都能笑出聲來！這不是他第一次淘到這樣的寶，當年的白邊哥懷特塞德，也是以落選秀的身份被熱火選中。還有本賽季同樣坐穩首發的鄧肯.羅賓遜表現也很出色。

納恩如果延續這樣的表現，不僅會讓熱火成為東部有力的競爭者，而且在這個大合同滿天飛的年代，成為億元先生也不是夢！
