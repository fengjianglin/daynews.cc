狂野西部的那些被期望進入前八的球隊能否打進季後賽？

2019-11-29 0條評論 18次閱讀 0人點贊

（本文數據截止至11月27日）

NBA新賽季已經過去了一個月，我們能夠確定的事情有這麼幾件：洛杉磯兩支球隊的表現都符合賽季前的預期，盧卡-東契奇真的太出色了，金州勇士……算了，我們還是別提他們為妙。由於勇士失去了過去五年「獨孤求敗」的競爭力，很多人都認為本賽季的西部將會成為真正的狂野西部。賽程已經過去了五分之一，從目前來看這種想法是正確的。

現在我們可以把各支球隊劃個檔次了。洛杉磯湖人、洛杉磯快船、丹佛掘金、猶他爵士和休斯頓火箭將會是西部最有力的競爭者，而得益於東契奇舉世無雙的表現，達拉斯獨行俠似乎也能佔據一個西部前六的席位。

到目前為止，這六支球隊應該是穩進季後賽的。其餘兩個季後賽席位的競爭是開放的。火箭現在排在西部第6位，排在火箭之後的是菲尼克斯太陽和明尼蘇達森林狼。賽季初，幾乎沒有人能預測到這兩支球隊可以打入季後賽，他們的預期戰績分別是聯盟的倒數第6位和倒數第7位。在這兩支球隊後面，還有超過六支球隊的排名距離西部第六的勝場數只在3.5場之內。

如果現在斷言哪兩支球隊將會佔據剩餘兩個季後賽席位的話，那麼還為時過早了，畢竟西部的局勢太複雜了。有些球隊，比如勇士——他們擁有一個歷史級別的擺爛任務——而孟菲斯灰熊，也是擺爛大軍中的一支，因為他們兩支球隊的賽季前目標都不可能實現了。他們如今的目標是爭奪懷斯曼（原諒我，這也是一場激烈的競爭）。再比如俄克拉荷馬雷霆，儘管謝伊-吉爾吉斯-亞歷山大的成長速度令人驚喜，然而球隊主要的任務依然是在克里斯-保羅和達尼羅-加里納利的帶領下和勇士爭奪高順位選秀權。除去這幾支球隊，剩下的就是森林狼、太陽、薩克拉門托國王、聖安東尼奧馬刺和波特蘭開拓者了。賽季進行五分之一之後，根據他們目前的表現，我們可以通過信心指數的方式來為他們打入季後賽的可能性做個預測。

明尼蘇達森林狼（9勝8負）

在我們開始分析森林狼的表現之前，我想要給安德魯-威金斯發個簡訊表達歉意。安德魯，我對我當初說你是個「廢物」和「在攻防兩端都毫無價值」和「絕對配不上一份頂薪合同」和「無論你是不是還沒到25歲，用什麼詞語來評價你都是別有意味的」道歉。本賽季14場比賽過後，威金斯場均得分是職業生涯新高的25.2分，投籃命中率也達到了46.9%，同時，他的統計表正負值也首次達到了正數。他的助攻數和籃板數都是職業生涯的新高。

卡爾-安東尼-唐斯的表現也是不容忽視的，他場均拿下26.5分，他的得分數、籃板數、封蓋數、罰球數和三分球命中數都是領跑全隊的。雖然曾經的唐斯經常會打出展現他無窮潛力的精彩比賽，但是如今，也就是他職業生涯的第五個賽季，他終於成為了一隻真正的獨角獸。萊恩-桑德斯的快節奏進攻體系十分有利於唐斯的發揮，看起來整支球隊也走在正確的道路上。

周一，森林狼以125-113擊敗了亞特蘭大老鷹，這可能是球隊新征程的起點。在對陣老鷹的比賽中，6號秀賈勒特-卡爾弗代替傑夫-蒂格出任首發。雖然他的傳球或許不及經驗豐富的蒂格，但是面對有點難纏的老鷹時，他卻表現得遊刃有餘。儘管卡爾弗的成長對於森林狼能否打入季後賽意義重大，但是整支球隊的成功還是依仗於唐斯和威金斯雙人組的發揮，無論是從比賽計劃，還是從數據表上看，他們倆都是球隊最重要的支點。到目前為止，唐斯和威金斯兩個人共同出場的時間為295分鐘，他們倆的共同凈效率已經相比於上個賽季翻了一番還不止。作為球隊陣容中使用率最高的球員，他們倆發揮的高下就會決定球隊是在明年春季一路高歌還是迎來一波連敗了。兩個賽季之前，森林狼打破了球隊多個賽季缺席季後賽的尷尬紀錄。雖然森林狼在上賽季沒有打入季後賽，但是這支全新的森林狼確實要開啟球隊歷史的新篇章了。

信心指數：80

菲尼克斯太陽（8勝8負）

太陽是賽季初最令人驚訝的球隊，雖然他們的勝率在過去五場比賽之後回到了50%，但是他們之前的戰績一度是7勝4負。儘管贏下快船和費城76人的比賽讓人瞠目結舌，然而這兩場勝利絕不僅是幸運女神的眷顧而已。德文-布克場均拿下24.8分，三分球命中率是職業生涯新高的42.9%。里基-盧比奧是布克理想型的後場搭檔，盧比奧場均送出聯盟第六高的8.1次助攻。阿隆-貝恩斯、達里奧-沙里奇和凱利-烏布雷也在蒙蒂-威廉姆斯的體系中找到了自己的定位。他們戰績的下滑在一定程度上是由於傷病導致的，盧比奧和貝恩斯加起來一共缺席了過去九場比賽中的八場比賽，還有一部分原因則是球隊實力使然。賽季初，人們預測太陽的勝場數會少於30場，如今太陽50%的勝率早已完全超過了之前的預期。

德安德烈-艾頓將在12月中旬從禁賽中解禁復出，從紙面實力上來看，他的回歸將進一步顯著增強太陽的整體實力。如果按天賦排名的話，那麼艾頓是太陽陣容中第二出色的球員。在競爭季後賽席位的關鍵節點，他將會如何提前融進球隊的體系是值得關注的。毫無疑問，處於最佳個人狀態的艾頓具有在內線翻江倒海的實力，他也可以成為球隊最出色的封蓋手。處於最差個人狀態的艾頓則會因為他缺乏投射的弱點而破壞太陽逐步提升的進攻（他們的進攻排在聯盟的第9位），太陽的進攻也會因此停滯下來。

信心指數：60

薩克拉門托國王（7勝9負）

上賽季的國王受到人們的一致好評，他們的進攻風馳電掣，再加上二年級後衛德阿隆-福克斯的嶄露頭角，他們打出了十多年以來球隊的最佳戰績。雖然福克斯在本賽季再次成為了人們關注的焦點，但是他給人們的反饋卻不是那麼的積極。來自肯塔基大學的福克斯本賽季的表現不再驚艷，在他出戰的九場比賽中，球隊的表現看起來是雜亂無章的。在福克斯因為腳踝受傷而缺席比賽之前，國王的戰績只有3勝6負。而在之後，他們打出了一波4勝2負的戰績，戰勝了波士頓凱爾特人和太陽，輸掉的兩場都是客場比賽，其中包括一場對陣湖人的比賽。

在福克斯缺席的六場比賽中，國王的凈勝分提高了5.1分，他們目前的戰績距離西部第八也僅有一個勝場而已。福克斯並不需要急於復出，國王賽季的慢熱開局可能也還跟福克斯不適應新帥盧克-沃頓的體系有一定關係。上個賽季，國王的進攻節奏排在聯盟的第3位。本賽季，他們的進攻節奏只能排在聯盟的倒數第3位，對於擅長轉換進攻的福克斯來說，這無異於一劑難以下咽的苦藥。雖然國王的核心陣容是由福克斯、巴迪-希爾德和還在養傷的馬文-巴格利這幾位年輕人組成的，但是他們最近的勝利卻要歸功於經驗豐富的里喬恩-霍姆斯、哈里斯-巴恩斯和博格丹-博格丹諾維奇。上個賽季的國王已經展現出了他們具有在西部更上一層樓的潛質。如今的問題是，他們能否延續上個賽季的表現呢？

信心指數：40

新奧爾良鵜鶘（6勝11負）

本賽季大部分關於鵜鶘的話題都集中在缺席比賽的球員身上，而不是上場比賽的球員身上。遭遇了半月板傷病的蔡恩-威廉森至今還未完成自己的常規賽首秀。不過，這無法阻止他的隊友們帶領球隊高奏凱歌。離開洛杉磯聚光燈的布蘭登-英格拉姆似乎在新奧爾良完成了重生。英格拉姆場均拿下26.1分，場均出手5.7次三分球，三分球命中率達到了45.9%。

雖然朱-霍勒迪在進攻端的慢熱讓人有點擔心，但是好在他終於找回了自己的手感。作為鋒線老將的JJ-雷迪克最近在首發陣容中代替了郎佐-鮑爾的位置，他和英格拉姆的搭檔能夠拉開進攻空間，這十分有利於球隊突擊手的衝擊內線。真正的問題是，當威廉森復出之後，一切會如何呢？在受傷之前，威廉森季前賽的表現令人驚訝，卻也符合人們的預期。如果他能完全康復的話，那麼他的火熱狀態是值得人們的期待的。不過，在威廉森缺席的這段時間裡，英格拉姆時球隊無可置疑的頭號攻擊手，他回到二號攻擊手位置的過程將會決定本賽季是鵜鶘又一個重建的賽季，還是鵜鶘重返季後賽的賽季。根據大衛-格里芬的消息，威廉森的復出已經指日可待了，即使球隊的磨合可能會出現些許問題，但這不應該成為鵜鶘不敢讓未來的超級明星大展身手的疑慮。

信心指數：50

聖安東尼奧馬刺（6勝12負）

在十多年的長盛不衰過後——認真的說，確實是十多年——時間老人的腳步終於追上了格雷格-波波維奇和他的馬刺。馬刺上一次缺席季後賽還有追溯的1996-97賽季，那時候因為大衛-羅賓遜的傷病他們選擇為了得到蒂姆-鄧肯擺爛一年（他們竟然真的成功了）。再往前推，他們缺席季後賽是1988-89賽季。不過，在奪得五次總冠軍，經歷20年的輝煌，湧現多位名人堂球員之後，馬刺的這段旅程可能真的要走向終點了。

馬刺的勝率只有三分之一。球隊中的兩位明星球員不再年輕，他們倆的打法又是落後於時代的。馬刺場均三分球出手24.9次，這是聯盟的最低值，不過他們場均命中13記三分球的數據倒還算不錯。考慮到本賽季馬刺的防守只能排在聯盟的第26位，球隊的三號得分手竟然是布林-福布斯，我們可以說那支偉大的馬刺已經離我們遠去了。雖然波波維奇曾經也在混亂的球隊名單中製造過奇蹟，但是這次的挑戰要遠遠困難於之前的任何一次。

信心指數：10

波特蘭開拓者（6勝12負）

上個賽季，開拓者突破了人們給他們預設的球隊上限，自1999-2000賽季之後再一次打入了西部決賽，他們上一次打入西部決賽時，球隊的前場核心還是拉希德-華萊士。儘管艾爾-法魯克-阿米奴、埃文-特納和莫-哈克利斯等幾位重要輪換球員在休賽期離開了球隊，內線球員扎克-科林斯和尤素夫-努爾基奇也處於傷停狀態，然而人們普遍認為，在競爭開放的西部，由達米安-利拉德和CJ-麥科勒姆這對後場組合領銜的開拓者依然是極具競爭力的。不過，從目前來看，事實與人們的想法相差甚遠。

開拓者的表現一度十分掙扎，他們在賽季首月遭遇了兩次四連敗，憑藉著周一以117-94戰勝芝加哥公牛的比賽，他們終結了最近一次的四連敗。當未來黯淡無光時，開拓者毫不猶豫地把希望寄託到了卡梅羅-安東尼身上，安東尼上一次出戰NBA還是去年的11月。到目前為止，安東尼依然還是那個安東尼，他的場均得分為16分，他在周一的比賽中拿下了25分，出手20次，比利拉德和麥科勒姆分別多出手10次和4次。哈桑-懷特塞德的比賽風格令人難以捉摸，他在進攻端的貢獻完全無法彌補他在防守端的失分。有流言顯示，開拓者對於交易得到克里夫蘭騎士的凱文-樂福很有興趣。樂福可以成為球隊的三號得分手，他在進攻端可以馬上有效彌補懷特塞德的不足，而且在努爾基奇和科林斯復出之後，樂福和他們的共存也沒有什麼問題。在得到樂福之前，開拓者只能依賴安東尼了。

信心指數：30
