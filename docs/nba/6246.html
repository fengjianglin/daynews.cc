直播12月4日火箭VS馬刺前瞻，火箭馬刺球探報告。火箭十連勝之3？

2019-12-04 0條評論 1次閱讀 0人點贊

2019-12-04 09:30 NBA常規賽 火箭VS馬刺

預測：火箭勝 12-18分。火箭12月賽程甜蜜，只有猛龍和快船是戰績較好的強隊，很有可能十連勝。

看直播關注我，發私信給我：火箭。

馬刺隊球探報告

1.馬刺自從開局在4勝1負後，取得了3勝13負的糟糕戰績。馬刺隊的防守效率只排在聯盟的第26位，但是他們場均失誤次數是聯盟第二少的。

2. 德羅贊已經連續7場比賽得分超過20分，他也是馬刺本賽季唯一一名每一場比賽得分上雙的球員。他本賽季場均可以拿下21.6分5.5籃板4.5助攻。

3. 魯迪蓋伊昨日完成了他職業生涯的第900場比賽，他和詹姆斯是自2006-07賽季以來，聯盟中唯二可以拿下擁有15000分、5000籃板、1000次搶斷和500次蓋帽的球員。本賽季場均可以拿到11.3分6.0籃板。

4.阿德因為右側大腿酸痛，明日出戰疑問。

火箭隊球探報告

1.火箭的場均得分120.3分，排在聯盟第二位，僅次於雄鹿隊的120.8分，火箭擁有聯盟最高的三分球出手數和最多的三分球進球個數，但是三分球的命中率不高，只有34.3%，排在聯盟第26位。

2. 火箭中鋒卡佩拉由於上呼吸道感染在床上整整躺了一周，預計將會復出對陣馬刺，目前連續7場籃板19+.

3. 威斯布魯克過去四場比賽的投籃命中率為45.9%，過去三場比賽中，他只嘗試出手了8次三分球投籃，這是一個好現象。

4. 哈登在賽季前19場比賽中已經拿下了739分，這是自1962-63賽季張伯倫以來，最高的得分；上賽季哈登對陣馬刺隊場均可以拿下37分。

5. 豪斯最近因為傷病原因缺席了過去火箭10場比賽中的6場，對陣馬刺的比賽出戰成疑。
