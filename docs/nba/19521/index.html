詹姆斯道歉了！十年前他和母親在場邊吵架，這事全怪皮爾斯？

2019-12-16 1,468條評論 2,739次閱讀 294人點贊

眾所周知，詹姆斯一直都將「我不只是一名運動員」掛在嘴上，他還從事了很多慈善工作，並且會幫弱勢群體發聲。此外，詹姆斯也是一個非常出色的父親，休賽期的時候經常能在「小詹」布朗尼的比賽場邊看到他的身影，即便是在常規賽中，詹姆斯也會抽出時間現場觀看布朗尼的比賽。

有意思的是，往往布朗尼的比賽，賽場上發生了什麼都不會是重點，因為詹姆斯總是會在場邊「手舞足蹈」奪走所有球迷的關注——每當布朗尼或者布朗尼的隊友有漂亮的進球，詹姆斯都會第一個激動地站起來，甚至「違反規則」走進場上，他的動作也經常被自己的球迷調侃，稱「這位家長請你冷靜」。

考慮到父母對於自己孩子的感情，大部分球迷對此還是能夠理解的，不過詹姆斯卻意外地在鏡頭面前向自己的母親道歉。

布朗尼最新的比賽，小詹所在的球隊成功擊敗了詹姆斯的高中母校，賽後詹姆斯接受了記者採訪，表示：「我需要向我的媽媽道歉，因為在我打高中比賽的時候，我媽媽總是人群中聲音最大的，我會陷入瘋狂，我的對手甚至裁判都會瘋狂，我曾經對她說快點坐下，你這樣太讓我尷尬了，老媽，現在我要道歉，我現在切身體會到父母的感受。」

事實上，不只是高中，哪怕是詹姆斯進入聯盟之後，他的母親都會在場邊觀看自己兒子的比賽，這其中還有個故事，看完你大概就知道為什麼詹姆斯要道歉了。

事情發生在2008年季後賽，當賽季的騎士在詹姆斯的帶領下取得了45勝37負，排在東部第四，詹姆斯場均可以拿到30.0分7.9籃板7.2助攻，當選得分王。季後賽當中，騎士首輪擊敗了奇才，次輪遇上了凱爾特人，彼時的凱爾特人擁有加內特、皮爾斯和雷阿倫的三巨頭，騎士在前三場比賽過後大比分1比2落後。

第四場比賽在騎士主場進行，詹姆斯一如既往地包辦著騎士的一切，第二節中段，騎士獲得快攻機會，詹姆斯加速超過了皮爾斯和加內特準備得分，沒想到身後的皮爾斯居然抱住了詹姆斯，由於衝擊力過大，兩人一直衝到了場邊才停止，巧的是，詹姆斯的母親剛好就在那個位置。

眼見自己的兒子被犯規，詹姆斯的母親忍不住了，她站了起來，甚至要用手去擋加內特，詹姆斯見狀立刻讓自己的母親坐下，不過他的口氣並不是特別好，這一幕也被鏡頭抓拍下來，看起來像詹姆斯在「怒吼媽媽」。

有一個說法是，詹姆斯之所以會如此激動，是因為看到了他的母親站了起來，並且要用手去攔加內特，這樣做的話可能會讓她終生無法進進球館——不管真相如何，都改變不了的一個事實是，詹姆斯一直都是一個孝子，出身於單親家庭的他經常感謝自己的母親。

同時需要說的是，雖然是皮爾斯的犯規導致了這一幕的發生，但是皮爾斯絕不是惡意犯規，這一點從他一直抱著詹姆斯也能看出來。詹姆斯和母親的爭吵開始後，皮爾斯也是拉著詹姆斯，回到賽場之後，詹姆斯還特意找到了對手，並且感謝了對方。

年少輕狂，怕家長在場邊大喊大叫讓自己尷尬，這個階段相信很多人都經歷過，詹姆斯也是如此，如今，他的兒子已經踏上了賽場，他也和他的媽媽一樣，不顧形象在場邊為孩子加油，為人父母的不易，真的要切身體會才知道。
