進攻防守都是聯盟第一，這樣的雄鹿隊太可怕，字母歌配得上MVP

天天要聞
2 週 ago

在今天上午結束的一場NBA常規賽中，米爾沃基雄鹿隊以132:88 44分狂勝紐約尼克斯隊，他們不僅展現出了可怕的統治力，自己也收穫了12連勝，在上一場比賽取勝之後，他們是自1984年至85年以來首次拿下11場連勝記錄，今天贏球之後，這個12連勝對於整個面無敵雄鹿對對史來講都是極為罕見，甚至是成為創造記錄的戰績！

截至目前他們的21場比賽里取得了18勝3負的出色成績，位居東部第1位，繼上個賽季成為全聯盟戰績最佳的球隊之後，本賽季的雄鹿隊並沒有停止腳步，向著他們進一步的戰績發起了衝擊！

雄鹿隊為何連續兩個賽季都展現出了極高的統治力？

當我們走進數據之後，可以清楚的發現他們統治了進攻和防守兩端。對於籃球這項運動而言無非就是，讓自己得更多的分，讓對手得更少的分。顧名思義就是成就在進攻和防守兩端。

對於一支球隊而言，如果只注重進攻，打出華麗的進攻，沒有高效的防守，那麼他們存在的也只是觀賞性，或許並不能得到出色的戰績。對於一支球隊來說，如果他們只注重防守，沒有進攻的話，無論是常規賽還是季後賽，他們都不可能走得太遠。

全聯盟30支球隊每年的總冠軍只有一個，在NBA這個世界籃球大舞台上競爭異常的激烈，對於籃球這項運動而言，雖然進攻和防守很難兼得，但是想要奪得最後的總冠軍，想要進入總決賽，想要打出出色的戰績，必須在進攻和防守兩端都下足功夫，不要求兩端同時做到最好，但是兩者不可缺1。要重視進攻和防守的平衡性，不可只進攻而不防守，同樣也不可只防守沒有進攻。

攻守兼備才能成就王者，然而米爾沃基雄鹿隊他們就是如此做到的，聯盟第一的驕傲戰績就是從進攻和防守同時抓起做到的。

進攻端：
截至目前雄鹿隊的場均得分高達120.8分，高居聯盟第一，他們的投籃命中率達到了48.3%，同樣是全聯盟第1位。雄鹿隊的進攻效率達到，113.8高居全聯盟第2位。從這一系列的數據，我們可以清楚的看出在進攻端，雄鹿隊可謂是會當凌絕頂的存在。
防守端：

當我們走進雄鹿隊的防守數據之後，不由自主的會給他們豎起大拇指。截止目前雄鹿隊的防守效率為101.9，高居全聯盟30支球隊中第1位。他們的場均防守籃板為41.57個高居全聯盟第1位。
綜合進攻和防守兩端我們可以，輕鬆的看出雄鹿隊是全聯盟在攻防兩端最具統治力的球隊。籃球的勝利無非就取決於進攻和防守，雄鹿隊竟然做到了兩者兼得戰績如此出色，也是理所當然。

無論一支球隊在攻防兩端打出怎樣的表現，它都是陣容基礎的呈現。這一切都是教練的布置和球員在場上的表現所決定所做到的。

下面我們一同來看雄鹿隊用怎樣的陣容做到了他在進攻端的統治力。
雄鹿隊在進攻端毫無疑問的是以字母哥阿德托昆博為首。他是這支球隊的靈魂進攻防守核心，他的全能就是在進攻端和防守端的統治力。
首先我們來看字母哥，這個上賽季奪得常規賽MVP的男人在本賽季有了進一步的進化和提升。
本賽季的字母歌場均可以得到，30.8分+13.4次籃板球加5.8次助攻，投籃命中率達到了恐怖的56.5%。他的快攻，他的衝擊力，他在籃下的終結能力幾乎是無解的。雖然他的中遠投的能力，還是沒能夠達到出色的水平，但是也一直在進步當中，要知道本賽季的這麼個字母哥年僅24歲呀，在他身上的看到的天賦，看到那種爆棚力爆炸力還沒遠遠沒有達到他的最高水準。現在的字母哥像極了剛入聯盟初期的詹姆斯一樣，無與倫比的身體素質和衝擊力。

字母哥在進攻端的統治里表現在：

他擁有著無與倫比的身高，臂展，彈跳，移動速度，身體素質和衝擊力。

他的身高達到了2米11必然達到了2米24。體重為110公斤，渾身充滿著肌肉線條。難得可貴的是他在這個身高的情況下，有著聯盟頂級的運動天賦，移動速度。2米11的身高完全是一個中鋒大前鋒的身高。他不僅僅擁有著這樣的身高優勢，但是他還擁有著同樣作為這個身高球員下沒有的移動速度和衝擊力。
最終導致的結果就是有他高的沒他快沒他彈跳力強，有他快的，沒他高沒他衝擊力大。這樣才造就了一個在進攻端在罰球線以下近乎無解，近乎無人可防的字母哥。

他不僅僅將個人出色的身體天賦僅僅展現在進攻端，本賽季他場均13.4次籃板球高居全聯盟第四，並且還有5.8次的助攻，此時此刻彰顯出了他無與倫比的全能性！

在這樣一個字母哥的帶領下，雄鹿隊在進攻端形成了一星四射的陣容，在雄鹿隊的比賽里，常常都是以字母歌為首，為進攻核心和組織進攻核心，另外配備一個衝擊力較強的控球後衛就是布拉德索，另外三個人都有著不錯的外線投射能力！這樣就可以依靠著布萊德索字母哥為首的衝擊力吸引對手包夾，傳出空位機會！
與此同時在這麼個身邊的配的這幾個得分手都表現出了不錯的貢獻和效率，其中米德爾頓目前場均17.4分+5.4次籃球加3.1次助攻，布萊德索場均15.3分+5.1次籃板加5.6次助攻，洛佩斯場均10.7分+4.9次籃板球+1.6次助攻！另外還有喬治希爾，馬修斯，伊利亞索瓦等悍將，提供著替補席上的火力！
這樣以來整個雄鹿隊的進攻以快攻防守反擊，速戰速決，高效率得分為主！他們在進攻端的天賦速度是令人無比羨慕的！

一支球隊的進攻為龍頭，一支球隊的防守為基礎，在上面我們一同看到了雄鹿隊，在進攻端的表現和展現力，下面我們一同分析他們是如何在防守端同樣做到聯盟最出色的呢？

還是那句話，場上的展現場上的執行，依靠陣容基礎，依靠他們的球員來決定！在防守端，首先我們要肯定的是雄鹿隊擁有著多位聯盟頂級的防守球員，加上他們的主教練不登霍爾澤，給予整支球隊的防守體系打造和技戰術配合，才成就了一個如此出色的雄鹿防守！
雄鹿隊隊內在防守端的核心還是以字母哥為主，這個男人進入了上賽季的最佳防守陣容第一陣容，其中還兩次進入過最佳防守第二陣容！字母哥的防守優勢就是他的身體條件，加上他自身擁有的機動性，幾乎是可以從1號位防到5號位！

字母哥在防守外線的時候，他擁有著比外線球員更高的身體，擁有著不落後於外線球員的機動性移動速度，加上他個人的2米11身高和2米24壁展，常常對於對手的外線形成了一個死亡纏繞！

當字母哥防守內線的時候，恰恰他的身高又不落後於對手的內線球員，身體對抗彈跳臂展都不落下風，更為關鍵的是他腳下的移動速度要比大多數中鋒大前鋒的移動速度要快，這就是字母哥為何在防守端同樣出色的原因。

然後我們一同來看在他身邊配備的這些防守換將，其中一位就是雄鹿隊的當家控球後衛，布萊德索，這個1米85的男人在上個賽季竟然進入了最佳防守第一陣容，布萊德索早期被稱為小詹姆斯是有原因的，就是他擁有著爆棚的身體天賦，雖然身高只有1米85，但是他的臂展達到了2米02，體重達到了93公斤，也就是說他在外線防守之時對抗移動速度和必展成為他對於外線防守的殺手鐧！
另外在鋒線位置上，他們還擁有著聯盟出色的頂級3d球員，米德爾頓和馬修斯不僅僅能夠體現外線的火力，還能夠給予球隊外線防守很大的幫助！

對於雄鹿隊整支球隊的防守體系打造還是死亡5大為主，這死亡5大或許字母哥作為鎮守內線的防守球員，其他球員身高都在兩米左右要的就是他們防守端的移動速度不落下風，並且能夠打出有效的防守反擊，推起快攻，從防守端制勝，從進攻端終結！

這就是雄鹿隊為何連續兩個賽季目前高居聯盟最佳戰績的原因，因為他們同時兼得了，決定籃球賽場上勝負的兩樣最重要的東西就是進攻和防守，並且他們都做到了聯盟最佳的水平！上個賽季雄鹿隊大游進入總決賽大有奪冠之勢，只是無奈在東部決賽中2:0的領先情況下，被猛龍隊逆轉，這是他們為年輕所付出的代價和教訓！

對於他們而言，他們的天賦不落後於任何一隻球隊，最弱的一環或許只是在季後賽中的比賽經驗了，也希望作為一個出色的主教練，布登霍爾澤能夠繼續為年輕球員們為他的隊員灌輸更好的籃球思維！

他們目前是全聯盟最出色的球隊，但是想要奪得最後的總冠軍，前途慢慢任重而道遠，需要解決的事情還有很多！最後只能祝願雄鹿隊好運，祝願字母哥加油！
