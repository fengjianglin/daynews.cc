重排09年選秀，格里芬仍是狀元，哈登榜眼，國王沒選錯，只是用錯

2019-12-21 2,745條評論 4,807次閱讀 2697人點贊

2009年本是選秀小年，如今卻誕生了2個MVP，成為質量最高的一屆新秀之一，不得不說確實是很驚喜，那麼重新排2009年選秀，順位會有怎樣變化呢？

狀元：快船選擇布雷克格里芬（實際選擇布雷克格里芬）。

當年快船陣中有克里斯卡曼這位全明星內線，有小喬丹、坎比這樣的藍領內線，有巴朗戴維斯、埃里克戈登兩位出色的後衛，對於快船而言，缺的是小前鋒，而如果沒有小前鋒，快船首選應該是一名可以充當核心的球員，從09一代球員里看，看起來更像核心的球員就是格里芬了，庫里和哈登也許更出色，但他們真的偏「浪」，沒有格里芬那麼成熟。

榜眼：灰熊選擇詹姆斯哈登（實際選擇塔比特）。

灰熊已經有了康利、梅奧和蓋伊三位外線，同時也有了蘭多夫和馬克加索爾兩個內線，怎麼看灰熊都不缺人，但既然有榜眼，灰熊可別浪費掉，他們最佳選擇應該是一位可以從第六人打起，並且有潛力的球員，同時最好要有點防守，畢竟這是灰熊的球風，相比之下，當年哈登的選秀報告里，防守肯定比庫里好的。

探花：雷霆選中庫里（實際選擇哈登）

同樣的，雷霆擁有威少、杜蘭特、伊巴卡、塞弗羅薩、傑夫格林，首發陣容也滿了，他們的選擇會跟灰熊一樣，選一個打第六人並且有潛力的球員，但是他們對防守沒有絕對的要求，反而更希望進攻強一些的，庫里肯定是最好的選擇了。

第四順位：國王選擇泰瑞克埃文斯（實際選擇埃文斯）

選擇本沒有錯，只是用錯了，非得把控衛上的詹姆斯，用成小前鋒上的「侏儒」，國王徹底用廢了埃文斯，一個新秀場均20+5+5的球員，真是可惜了。

第五順位 ：森林狼選擇朱霍樂迪（實際選擇盧比奧）

森林狼很富裕，連續擁有第五和第六順位，而此時森林狼有樂福、艾爾傑弗森兩位潛力內線，同時還有布魯爾這位小前鋒，森林狼缺的是2個後衛，當時選擇盧比奧看中的是他的潛力，但盧比奧的打法在後來有點落後了，選擇霍樂迪會更合適一些。

第六順位：森林狼選擇德羅贊（實際選擇弗林）

選盧比奧不算錯，但選弗林一定是錯的，既然已經有霍樂迪一個控衛了，第六順位選個得分後衛妥妥的，霍樂迪+德羅贊+布魯爾+樂福+傑弗森的陣容，其實還不錯。
