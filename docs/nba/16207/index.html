今日NBA：馬刺公牛絕殺拒絕連敗，東契奇開場便崴腳憾負熱火

2019-12-15 2,219條評論 4,379次閱讀 727人點贊

北京時間2019年12月15日，NBA的比賽照常進行，今天的比賽也是出現了許多驚心動魄的時刻，比賽也是一場比一場精彩，一場比一場好看。

首先是在這個賽季狀態不佳的聖安東尼奧馬刺隊，不論是面對勁敵還是弱隊，他們表現得都十分的不出彩，作為連續二十多年都晉級季後賽擁有深厚底蘊的球隊，他們在德羅贊和阿爾德里奇的配合中失去了之前那麼強的競爭力。

在今天馬刺隊也是面對實力並不強的太陽隊，馬刺隊雖然遠赴客場作戰，可是比賽一上來他們還是很快的進入比賽的節奏，第一節也是獲得了十分的領先，不過在第二節馬刺隊也是把比分優勢全部送了回去。

知道常規賽的最後一刻馬刺隊也沒有握有制勝之計，比賽也就此進入加時賽，在加時賽中，馬刺隊的米爾斯投出了制勝的一球絕殺太陽隊獲得了比賽的勝利，面對太陽隊打的如此焦灼，馬刺隊的狀態確實是要好好的調整了。

在今天的比賽中，德羅贊得到18分、阿德得到18分、默里得到18分、替補出戰的米爾斯得到全場最高的26分，替補隊員的發揮是今天馬刺隊能夠贏下比賽的關鍵所在。

反觀太陽隊，盧比奧全場得到25分13助攻、卡明斯基22分、沙里奇19分17籃板、烏布雷17分，布克缺戰，艾頓依舊處于禁賽期，太陽隊在艾頓不在的情況下依舊保有很強的氣勢，太陽隊今年肯定不再是魚腩球隊了。

而在另一場的比賽中，快船隊倫納德、路威、貝弗利等人缺席比賽，快船隊喬治帶隊客場挑戰公牛隊，公牛隊的發揮一直有很大的氣氛，他們也是連敗勇士隊展現出他們的下限十分低。

不過在今天的比賽中，公牛隊的拉文在最後時刻完成了救贖，制勝的上籃並造犯規讓公牛隊最終以109：106贏下快船隊，在這場比賽中，拉文得到31分、馬爾卡寧得到13分17籃板、卡特得到14分，替補出場的萬倫丁和賽迪斯楊分別拿到16、17分。

而快船隊今天只有11人輪換，喬治拿到27分、沙梅特11分、哈勒爾替補出場得到30分，帕特森11分，快船隊因為輪換在今天顯得捉襟見肘，被拉文絕殺也不算是十分丟臉的輸法。

而在獨行俠隊與熱火隊的比賽中，東契奇僅一分多鐘就崴腳離場也是直接讓比賽懸念降低，不過獨行俠隊在沒有東契奇的情況下表現出了超強的求勝欲，儘管上半場落後23分，不過在常規時間內，獨行俠隊竟然把比分扳了回來，雙發也進行到加時賽，沒有主力的獨行俠隊最終也是雖敗猶榮，118：122他們也是惜敗熱火隊。
