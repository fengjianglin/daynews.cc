湖人熱火賽後分析，兩隊攻防俱佳，超巨決定比賽方向

2019-12-16 322條評論 1,301次閱讀 123人點贊

熱火與湖人這場比賽到底有多麼精彩，韋德賽後一句話就挑明了，他稱這是本賽季最佳比賽。之所以會稱其最佳，跟比賽的過程有很大的關係，因為兩隊不僅是在最後時刻才分出勝負，比賽中的很多細節也值得大家去回味，接下來我們就一起分析一下比賽中的各種亮點和數據。

先來說說這兩隊的背景吧。湖人現在是西部頂級的強隊，目前已經獲得了5連勝的戰績，並且這5連勝中，湖人的進攻端有了明顯的改善，除了與熱火那場比賽中只得到96分之外，與森林狼和開拓者那兩場比賽分別砍下了142分和136，即便是與爵士那場砍下的121分依舊高於湖人本賽季的平均水平，這也體現了湖人近期所作出的改變。

而熱火這邊也差，他們最近也獲得了三連勝，高居東部第三的位置，僅次於76人和雄鹿，即便是綠軍都在他們的戰績之後。熱火也是一支進攻強隊，場均得到111.9分排名聯盟第12位，這跟球隊團隊作戰的風格有關，他們不僅有巴特勒在，還有本賽季的黑馬球員納恩，2019屆的新秀希羅，外加阿德巴約等人均有很強的進攻能力，所以這兩支球隊的對陣自然不會缺乏看點。

比賽一開場兩隊的核心人物就開始了solo，詹姆斯與巴特勒沒有通過換防對位，而是進行了直接對位，在巴特勒強硬防守之下，詹姆斯的進攻也受到了一定的限制，令人有些意外的是，熱火即便是在湖人要求詹姆斯和濃眉哥單打時候並沒有人去協防或者是包夾，憑藉他們去單挑，而這樣的做法也得到了一定的效果，被迫讓詹姆斯和濃眉哥只能夠靠快節奏的比賽去得分，詹姆斯更是對這樣的打法不太習慣，第一節就送出了7次失誤。

然而即便是如此，熱火都沒有佔到什麼便宜，只因湖人的麥基、布拉德利、卡魯索等人屢次衝擊熱火內線得分，首節比賽結束時湖人還是領先熱火一分。

第二節比賽熱火換成了阿德巴約+希羅+納恩這樣的陣容，顯然是想要加強球隊的進攻火力，尤其是納恩和阿德巴約屢次進攻湖人籃筐，當時的霍華德並沒有什麼招架之力，很快熱火就將比分反超了。防守端熱火也開始對詹眉組合包夾，從而造成了湖人球員多次失誤，熱火也抓住這個時機很快將比分擴大至兩位數。

可是下半場風雲突變，一下子讓湖人成為了主動，原因就是詹姆斯狀態復甦了，他不僅多次進攻打中，在遭遇夾擊之後第一時間將球給傳了出去，湖人也是多點開花迅速將比分抹平。熱火還是依舊用著納恩和阿德巴約的組合，就這樣兩隊的比分也開始了交替領先直到最後一刻。在比賽最後時刻熱火的納恩和羅賓遜兩記三分球將比分追至還有1分分差，此時還有不到10秒鐘，最後波普被送上罰球線之後兩罰穩穩命中，讓湖人得到了3分的領先優勢。

最後一攻肯定是巨星來擔任了，吉米巴特勒最後一攻頂飛了詹姆斯之後接球，可是他面對的是湖人另一位核心濃眉哥，第一次晃動沒有徹底將濃眉哥晃開，在最後時候的時候還是被濃眉哥的長臂所影響，最終三分不中遺憾告終。

全場比賽兩隊能攻能防，比賽打得十分精彩，湖人贏下比賽的關鍵點就在於擁有的巨星上，尤其是巴特勒最後一攻先後面對詹姆斯和濃眉哥時所展現出來的乏力也是很明顯的，所以熱火還是輸在了巨星數量上。
