簽下他們真賺大了！快船有3位低薪高能，庫里4400萬合同拿2座MVP

2019-12-21 234條評論 4,221次閱讀 553人點贊

在NBA中，既然會出現那種遠超球員實力的溢價合同，自然也就不缺少那種最為超值的合同了，下面小編就給大家帶來最近幾個賽季中，聯盟出現的最超值的幾個合同，當然那些底薪合同和新秀合同就不在這個範圍之內了。

庫里4年4400萬

在2012年12月1日，庫里和勇士隊提前達成了續約協議，合同為4年4400萬，這份合同在2013年夏天生效，在2017年結束，要知道當時的庫里還不是之後的「庫日天」，更是在11-12賽季打了26場比賽後，就因傷賽季報銷，所以這份合同在當時看來，還是比較符合庫里身價的，但是誰能想到，在這份合同生效的第一個賽季庫里的數據就上升到24分4.3籃板8.5助攻，職業生涯第一次入選了全明星和最佳陣容，而且在這份合同內，庫里一舉成為聯盟頂級巨星，兩次獲得常規賽MVP，同時也兩次帶領勇士隊獲得NBA總冠軍，要知道他的這份合同中，最後一年的年薪也不過1211萬美元，在當時也就是一名不錯首發的年薪，這也讓這份合同成為近幾個賽季中最超值的合同。

哈雷爾2年1200萬

被馬賽克隊交易到快船隊的哈雷爾，在18年休賽期，和球隊簽下了一份2年1200萬美元的合同，其實在當年簽下這份合同時，已經稱得上是一份比較超值的合同了，因為當時的哈雷爾在場均17分鐘的出場時間內，就可以貢獻11分4個籃板的數據，命中率更是達到了64%，而這份合同生效後的第一個賽季，哈雷爾在保持出色效率的同時，數據更是達到了16.6分6.5籃板2助攻1.34次蓋帽，成為了快船隊的主要得分手之一，更是險些當選當賽季的最佳第六人，本賽季作為哈雷爾的最後一個賽季，他的數據更是有了進一步的增長，場均刻可以貢獻19分7.5籃板，按照這個水準，2020年休賽期想要簽約哈雷爾，至少需要1500萬以上的年薪才有可能。

PJ塔克4年3188萬

以塔克的表現來說，任何一支球隊都願意為塔克送上一份千萬級別的年薪，但是17年馬賽克隊簽下塔克時的合同，只有4年3188萬，雖然在球場上塔克的數據比較一般，但是他是聯盟少數貢獻大於數據的球員，特別是上個賽季，在球隊遭遇了傷病潮時，如果說哈登支撐起了球隊的進攻，那塔克就是一個人支撐起了球隊的防守，特別是在一些比賽的關鍵時刻，正式憑藉塔克關鍵時期的籃板和防守，才幫助球隊最終有驚無險的贏下了比賽。

路易斯威廉姆斯3年2400萬

為什麼本賽季的快船隊能夠在保證球隊幾名關鍵球員仍在的情況下，交易得到喬治，簽下倫納德呢？最主要的一個原因就是，球隊著數份非常超值的大合同，就像在快船隊兩奪最佳第六人，以替補身份成為球隊之前進攻核心的路易斯威廉姆斯，看看威廉姆斯加盟快船之後的表現，場均數據每個賽季都在20分5個助攻左右，這樣的數據覺得配得上千萬級別的年薪，但是在18年休賽期，路威和球隊僅僅簽下了一份3年2400萬的合同，別說路威比賽中貢獻的數據，僅僅是他關鍵時刻的表現，就配的上這份年薪了。

貝弗利4年2300萬

除了路威和哈雷爾之外，之前的快船還有著一份非常超值的合同，那就是貝弗利在馬賽克時期欠下的4年2300萬美元的合同，在比賽中的貝弗利雖然口碑褒貶不一，但是不能不能否認的是，貝弗利是快船本賽季之前最出色的外線防守球員，也是現役聯盟球員中最出色的3D型控球後衛之一，2次入選了最佳防守陣容，而且貝弗利和塔克一樣，都是那種不能用數據衡量貢獻的球員。

肯巴沃克4年4800萬

​沃克可以說是喬老闆管理生涯中最出色的一次操作，作為黃蜂隊自己選秀自己培養的球員，逐漸成為球隊核心的沃克，在2014年和球隊簽下了4年4800萬美元的合同，在這份合同生效之後，沃克從沒有一個賽季的場均數據低於20分5次助攻，不僅入選了全明星，更是帶領球隊殺入季後賽，而和沃克數據效率相似的歐文，同時期的合同要比沃克的高出500多萬。
