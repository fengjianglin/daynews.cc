真爛！鵜鶘4名首發球員，半場比賽25投1中，球隊遭遇13連敗

2019-12-19 977條評論 3,936次閱讀 886人點贊

北京時間12月18號，鵜鶘隊在NBA常規賽中迎戰籃網隊，而此前已經是12連敗的鵜鶘隊，他們最終並沒有在主場贏下比賽，而是以101：108不敵籃網隊，將球隊的尷尬戰績上升到常規賽13連敗。這場常規賽裡面鵜鶘隊球員的投籃可謂糟糕至極，全隊合計是108投37中，投籃命中率只有34.3%，而鵜鶘隊本場比賽的三分球命中率則是27.5%。

尤其是在比賽的上半場階段，鵜鶘隊更是打出讓人瞠目結舌的命中率數據。當上半場2節比賽打完的時候，在鵜鶘隊的5名首發主力球員裡面，除了半場比賽9投3中的霍勒迪，其餘4名首發球員合計25投1中，這4人的半場命中率達到了匪夷所思的4%。其中球哥鮑爾是半場5投0中，肯里奇-威廉姆斯半場比賽7投0中，費沃斯半場則是3投0中，只有英格拉姆收穫10投1中，勉強命中了1次半場投籃出手。

老實說，雖然也見過不少NBA的「打鐵大戰」，但是這麼糟糕的球隊命中率，在NBA整個歷史裡面也是不多見。當然，在下半場和加時賽中，鵜鶘隊球員還是找回了自己的投籃手感，特別是鵜鶘隊核心球員英格拉姆，他在上半場10投1中的情況下，之後交出了13投6中的球場數據，最終得到全隊最高的單場22分10籃板1助攻，展現出自己的調整能力和球星氣質。至於其他的3名鵜鶘隊首發球員，最終他們也是同樣完成投籃得分。

平心而論，現在這支鵜鶘隊真的非常非常爛，鵜鶘隊雖然通過交易濃眉哥得到了NBA潛力新星，但是現在鵜鶘隊還是徹徹底底的NBA魚腩球隊，並且在NBA常規賽打出13連敗的鵜鶘隊，他們的戰績也快要趕上西部倒數第一的勇士隊。如果接下來鵜鶘隊管理層不能通過球員交易扭轉乾坤，從而改變球隊現階段的糟糕處境，那麼19-20賽季對於鵜鶘隊而言，或許基本上可以算是提前結束了！
