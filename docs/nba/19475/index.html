西部亂上加亂！黑馬醞釀重磅交易，名記爆料追18+17猛獸組三巨頭

2019-12-17 1,634條評論 4,710次閱讀 64人點贊

要說聯盟本賽季成色最足的黑馬，當屬達拉斯獨行俠隊了。這支上賽季的魚腩球隊打出了17勝8負的開局力壓多支豪強高居西部第三位，僅次於洛杉磯雙雄湖人快船。創造一連串紀錄的東契奇帶領獨行俠在一夜之間崛起，成為聯盟的熱門球隊，這也讓老闆庫班有了更大的野心。

獨行俠本賽季的目標已經不單身闖入季後賽了，而是有了更大的野心，為了讓球隊走得更遠，獨行俠開始醞釀重磅交易尋找爭冠拼圖。據ESPN記者WOJ在一檔節目中爆料，獨行俠目前正在積極尋求交易得到一位大個兒球員來搭檔波神，第一目標就是活塞中鋒「庄神」德拉蒙德。

儘管坐擁兩位身高超過2米20的高塔波神和博班，但這兩位中鋒都有自己的局限性。替補中鋒博班是邊緣球員，非常規輪換，而主力中鋒波神大部分時間都飄在外線，防守端也並非強力內線。因此為了進一步加強內線厚度，以及為波神分壓，解放他的進攻能力，獨行俠將目標瞄準了藍領德拉蒙德。

德拉蒙德的合同將在2021年到期，2020年夏天擁有球員選項，他大概率將跳出合同試水自由球員市場追求一份頂薪肥約。但NBA記者James L. Edwards III報道，活塞不太可能給他提供5年1.9億的頂薪，最有可能會給他開出2000-2500萬美元的年薪。另外，活塞方面目前正在考慮如何處理德拉蒙德的續約問題，將其交易或放他以自由身離開都是可能的選項

賽季至今，德拉蒙德場均砍下17.7分16.6籃板的兩雙數據，同時還有1,72次搶斷和1,84次封蓋，是一名優質的藍領中鋒。儘管自主進攻能力有限，但吃餅能力和出色的二次進攻能力還是他能夠場均貢獻接近18分。對於獨行俠而言，德拉蒙德可以呆在波神身邊干臟活累活，而有東契奇的存在，也可以將德拉蒙德的吃餅能力最大化。

東契奇+波神+德拉蒙德的三巨頭看起來美如畫，儘管交易聽起來還面臨許多困難，但今年休賽期已經發生了很多樁看似根本不可能的交易。如果這筆交易能夠成功，獨行俠的實力就會提升一檔，成為西部的一支勁旅，到時候也會讓狂野的西部形勢更加混亂，也更熱鬧了。

另外，ESPN名記同時還透露，除了德拉蒙德，獨行俠潛在的獵物還有快船替補大殺器哈雷爾。本賽季是哈雷爾的合同年，獨行俠管理層認為在哈雷爾合同到期後，快船不會花太多錢來留下他，到時哈雷爾將成為他們的首要選擇。
