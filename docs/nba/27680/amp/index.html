2019-12-21

都知道成為一名NBA巨星是非常困難的，他們需要付出許多的努力，但是收穫也是非常豐富的，他們享受著光鮮亮麗的人前。創造了許多偉大的紀錄，但是也有許多不願面對的紀錄，因為，光彩的背後總是有些黑暗點點。

名宿科比夠偉大吧？歷史第二得分後衛，湖人隊退役兩件球衣的偉大功臣，職業生涯榮譽無數，擁有60億粉絲，但是科比也有不願面對的紀錄，那就是打鐵王，職業生涯曾經單場打鐵記錄（30次）和總打鐵記錄（14481次），都是很難被打破的尷尬紀錄。

名宿鄧肯是歷史第一大前鋒，帶領著馬刺隊奪取了五座總冠軍，在大前鋒的位置上，無人可及，能攻善守的鄧肯是馬刺隊隊史最偉大的巨星，但是就是這樣偉大的巨星，誰能想到，鄧肯生涯總共被對手封蓋了1488次，比排名第二的蘭多夫多了207次。歷史被蓋帽王。

名宿卡爾馬龍，雖然職業生涯沒有獲得總冠軍，但是也是超一流的大前鋒，在喬丹時代能夠跟喬丹對抗的超級巨星，歷史第二得分王，但是卡爾馬龍也有一個缺點，那就是歷史失誤王，生涯的失誤總數（4524次）是NBA球員中最多的。

名宿雷阿倫是目前為止最偉大的三分射手，畢竟庫里還未退役的情況下，還為超越雷阿倫，雷阿倫的三分總數還是第一，雷阿倫的三分總數是第一，而三分打鐵王也是雷阿倫，投的多，丟的多，不得不說，庫里三分，確實強於雷阿倫。

名宿拉希德-華萊士，怒吼天尊的名號可不是亂叫的，或許是個性使然，沒有哪個球員的技術犯規能夠跟華萊士相提並論，尤其是現在技術犯規需要罰錢的情況下，更不可能了，畢竟，傻子才跟錢過不去。

當然，還有許多名宿犯了許多錯誤，但是這些名宿，之所以犯錯誤，是因為需要創造更美好的紀錄，美與壞總是一起來的，沒有辦法，世界上是沒有真正完美的人，要不然，強如喬丹為何要選擇退役，因為厭倦了。
