人設崩塌？記者再爆猛料，萊昂納德再拿三個總冠軍都洗不白了

2019-12-25 2,617條評論 4,005次閱讀 2273人點贊

萊昂納德這次真的是百口難辯了！

據說，聯盟已經收到了舉報信，舉報萊昂納德在休賽期的時候，向猛龍，快船和湖人這三支球隊提出過分要求，相信球迷朋友都已經有所耳聞了。最終萊昂納德和快船簽約了，至於快船有沒有接受萊昂納德的要求，還是說這些都是記者們瞎編的，這就不得而知了。記者Sam Amick說，萊昂納德提出來的這些要求讓湖人很不滿。而聯盟也聲明，對於違規簽約，對於違反工資帽的簽約，要深入調查。反正在關鍵比賽前夕，爆出了這個猛料，還是挺有意思的。

萊昂納德的人設這次是真的崩塌了。來回顧一下他的一些難以理解的操作。

首先是保密問題，同時和三支球隊聯繫，工作做得非常保密，但是最終和一支球隊簽約，這在無形之中會耽擱其他球隊的引援。有人說，又不是萊昂納德叫湖人和猛龍等的，是他們自己要等的，跟萊昂納德有什麼關係？

這就牽扯到一個職業道德的問題了。確實不是萊昂納德要求湖人猛龍必須要等到那一天，但是既然萊昂納德早就想要了要和快船簽約，為什麼要吊著其他兩支球隊，為何不直接告訴他們呢？還有，提出要求讓湖人不要交易濃眉這個事情又怎麼解釋呢？

然後，關於「負荷管理」這個事情，又鬧得滿城風雨。

里弗斯前一天說萊昂納德要打，第二天萊昂納德說不打。這到底是聽誰的？一會兒說是受傷不打，一會說是輪休不打，搞得里弗斯很沒面子，聯盟也沒面子，最終球迷被玩得團團轉，蒙在鼓裡。詹姆斯側面回應了這個話題，說不輪休不僅是渴望比賽，熱愛籃球，還是為了那些來現場看他比賽的球迷，不知道是不是在為球迷發聲？

資深籃球評論員楊毅老師說了，不要在乎這個人到底怎麼樣，我們看的是球，關心的是這個球員的競技水平，帶來的精彩表演，而不是這個人。然後下面一群網友發表意見：那你為什麼不去播火箭的比賽？PGONE的說唱水平也不錯啊，為什麼要封殺他啊？

其實這和萊昂納德的事件是一個道理。根據馬克思主義的辯證關係來看，萊昂納德的球技和他的人品，是相互獨立的，但又是互相依存的。

球迷因為他打球打得好，才會喜歡這個球員，而喜歡這個球員，就會過多地關注這個人的日常生活，球場之外的東西，這是對於一個自己喜歡的球員的最常規的關注，所以球迷不可能只欣賞他的球技，而不去關注他其他方面，對其他的事情一概不過問。他的球技吸引了球迷，他的人品對球迷的這種感情起到一個深化或者弱化的作用。

站在楊老師的角度，他說的話當然沒問題，他是職業玩家，作為一個職業玩家，一般是不會直接暴露自己喜歡哪個球星的。站在商業的角度來看，他只負責說球，為什麼要去關注他的其他生活呢？哪裡有時間去關心這個事情呢？所以站在他的角度，也勸球迷關注點放在場上，而不是場外。這是可以理解的。

但作為一個有靈魂的，真正懷有熱愛之情的球迷，是絕對做不到忽略場外之事的。不管萊昂納德球打得有多好，不管他是不是聯盟第一人，反正就算他在快船拿三個總冠軍，也洗不白這些操作了。
