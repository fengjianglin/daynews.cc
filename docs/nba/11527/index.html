聯盟第一家超市開張！場均18+6鐵鋒能加入湖人助詹姆斯衝冠

2019-12-10 368條評論 1,374次閱讀 296人點贊

賽季已經進行了四分之一，NBA第一家超市正式宣布開張了，尼克斯總裁已經表示，他們將會尋求可以提升球隊的交易，當然，這是委婉的表示，尼克斯目前4勝19負，聯盟倒數第一，比勇士的戰績還要差，這種表態，無異於直接宣布超市開始營業了。

那麼尼克斯，有湖人現在最缺的，詹姆斯替補，是誰呢？上個賽季效率於凱爾特人的神鋒，本賽季的尼克斯得分王馬庫斯-莫里斯，本賽季的莫里斯場均能貢獻18.6分5.8個籃板，場均以50%的三分球命中率命中2.9記三分球，他能幫助詹姆斯衝冠！

那麼湖人，有沒有交易得到他的可能呢？有，但庫茲馬一個人不行，因為莫里斯的年薪，高達1500萬美元，湖人需要啟動三人以上，才有可能得到莫里斯，最直接的，他們需要送出波普+布拉德利/麥基+庫茲馬，來得到馬庫斯-莫里斯+阿隆佐-特里爾/米切爾-羅賓遜，這筆交易為啥有可能呢？因為尼克斯管理層傻啊，你看這些年他們給球員的合同，就是冤大頭，同為大城市，沒準尼克斯就願意成全湖人呢？

說回正題，湖人不會去出動波普、布拉德利這樣的球員去換取一個詹姆斯的替補，是不值得的，湖人的真正目標，應該是特里爾和尼利基納，兩位1米96的球員，在當今聯盟，1米96是可以擔當鋒線的，特里爾在尼克斯也經常打小前鋒，為啥要換他倆？因為能防。

不要高估庫茲馬的交易價值，他只是一個27順位的大齡新秀而已，如果湖人能用他換到特里爾+尼利基納，完全可以有後續操作，用尼利基納+二輪這樣的球員去換取雷霆的弗格森，這是一筆可操作的交易。

本賽季特里爾遭遇到了傷病，但依然可以貢獻7.5分，三分命中率44.4%，尼利基納的表現歲不如預期，卻也能貢獻6+3+2+1，並且尼利基納的防守是後衛裡面頂級的，就算後期換來弗格森，一位擁有標準小前鋒身材的防守衝擊型鋒線，也是湖人穩賺不賠的。

如果湖人真的可以完成這筆交易，那他們首發就有布拉德利、格林、詹姆斯、戴維斯、麥基，替補席上的穩定輪換有隆多、波普、特里爾、弗格森和霍華德，這堪比豪陣，如果賽季中期簽下被買斷的伊戈達拉，伊巴卡，那湖人就將處於不敗之地！
