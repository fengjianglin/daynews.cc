4場0分後，出場時間暴增14倍！遭遇本賽季最大難題卻獲意外驚喜

2019-11-25 0條評論 18次閱讀 0人點贊

在NBA常規賽中，號稱為本賽季最大的一匹黑馬太陽球隊，在本周的第一場比賽失利，以104-116輸給了主場的掘金球隊。

本場比賽中太陽的盧比奧與貝恩斯因為傷病原因還是沒有出席這場比賽，在上一場比賽中太陽與森林狼的對決中有布克的帶領，太陽還能勉強拿下比賽。

但在這一場比賽裡布克發揮低迷，場均出戰33分鐘只拿下了12分1籃板5助攻的數據。

真是前一場天神下凡，後一場猥瑣發育還被打的沒脾氣。

雖然太陽輸掉了這一場比賽，但在這一場比賽中太陽的替補席位卻出現了一名非常讓人驚喜的球員！

他就是奧科博，在這一場比賽主力奧科博出場28分鐘收穫16分2籃板4助攻，本場比賽也是他這個賽季目前的最高得分。

其實在本賽季奧科博的場均出場時間僅為10.5分鐘，他的賽季首秀也只有僅僅的2分鐘。

埃利-奧科博在本賽季出戰11場，場均獲得4.2分、1.1個籃板和2.4次助攻。

尤其是他在本賽季的11場比賽中有4場比賽獲得0分，這對於一位得分手實在是不好的數據。

可是奧科博沒有就此放棄，在經歷了4場0分後終於有所起色。

因為太陽目前無人可用的情況下，奧科博以替補的方式出戰，在鵜鶘與太陽的比賽中終於拿到了本賽季的上雙分數11分！

雖然沒有取得勝利，但從出場時間來看奧科博也已經大大地提升了。

而在太陽對戰掘金的這一場比賽，奧科博終於抓住了機會，獲得28分鐘的時間從賽季開局的出場時間，這整整暴增了14倍啊！

尤其是這一場比賽中奧科博展現出來的實力也讓人記住了他。因為奧科博是2018年的次輪秀，關注度也沒有那麼高，所以他的存在感就顯得比較低。

雖然奧科博在上賽季的數據也不是很好，但他是屬於比較努力的那種，天賦不夠就只有勤加苦練了，畢竟勤能補拙嘛！

奧科博這位球員也才22歲，他還是有很大的機會去證明自己，而太陽目前的狀況剛好能夠給予這位小將發揮的機會。

由於太陽狀元艾頓的禁賽，盧比奧與貝恩斯的傷病，太陽遭遇了本賽季最大的難題，不過卻發現收穫了大驚喜！

而這種情況下太陽很有可能會另闢途徑，從替補席選出一位適合球隊打法的球員，要知道太陽在本賽季的野心可不小啊！
