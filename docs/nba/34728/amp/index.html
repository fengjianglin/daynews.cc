​英雄相惜英雄，此人曾贊小卡會成巨星，小卡推特只關注他一人

天天要聞
2019-12-25

喜歡看NBA的人都知道，這個賽季風雲變幻，各路球星紛紛加盟自己想去的球隊而這個夏天最令人期待的球星肯定是剛剛在猛龍奪冠的萊昂納德，作為NBA總冠軍球隊的絕對核心，他受到了許多球隊的追捧，而他最終也是選擇了洛杉磯快船隊。

其實萊昂納德是一個低調、話很少的球星，他和別的球星不一樣，其他NBA球星有了名氣實力和地位後，都是很喜歡和媒體接觸，希望進一步地擴大自己的影響力，為自己爭取更多的利益。

例如巨星杜蘭特，他雖然實力出眾，被譽為聯盟第二人，僅次於詹姆斯，可他也是非常關心社交媒體的，他還用過小號來發泄自己的不滿，經常和球迷互懟，可以說是非常有個性的人了。當然，杜蘭特還是非常尊重自己的母親，剛進聯盟時候還背著書包，像一個三好學生，只是後來才慢慢形成自己不羈的個性。

萊昂納德進入聯盟這麼多年，他一直是低調謙遜的人，面癱臉是他的標誌，其實自從他父親去世，他就變得孤立話少，是他的舅舅一直鼓勵和支持自己，萊昂納德也是非常尊重自己的舅舅，經濟上的管理也是有他的一部分樣子。所以當年萊昂納德和馬刺談崩的原因之一也是萊昂納德的舅舅不滿意馬刺的所作所為。另外他們也是想要回到家鄉，在洛杉磯這樣的大市場尋找更好的代言。

其實萊昂納德是一個非常顧家，希望能在鄉親父老面前打球的人。不然他也不會離開剛剛奪冠的猛龍隊，回到自己的家鄉洛杉磯。而且他也是非常尊重一位曾經鼓勵過自己、支持過自己的前輩，那位前輩就是有著華麗球風的克勞福德。

低調王者萊昂納德，推特只關注一位球星，不是詹杜庫更不是哈登：萊昂納德剛進聯盟時定位是3D球員，防守尖兵。而克勞福德：我堅信你會成為巨星，震驚整個聯盟的。萊昂納德非常感激和尊重克勞福德，所以即便有很多厲害的NBA巨星，在他心中，這位前輩才是最值得他去尊敬的，小卡推特只關注克勞福德他一人，這麼一位NBA球星，可謂是英雄相惜英雄。

不要小看克勞福德這位老將，這是一位以華麗的球風而聞名的球星，是最佳第六人，而且他的人品在聯盟也是備受讚譽的。他的得分能力還是很強勁的，他還單場砍下過51分，這可是39歲的老將啊。這和他多年的努力和自律是分不開的。

努力的人終會得到他人的敬重，你認真地對待生活，生活也必然不會辜負於你。所謂英雄都是惺惺相惜的，你看到的只是小卡和克勞福德的成就，卻不知道他們在背後付出多少努力，灑下多少汗水，只是為了在這個競爭殘酷的聯盟生存與發展。希望他們都達到自己想要成為的那個人，不忘初心，繼續前行。
