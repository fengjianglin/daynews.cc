30+11！猛龍小卡進化，首輪末位秀再升級，他將成下一個巴特勒？

2019-12-15 1,070條評論 4,687次閱讀 2387人點贊

12月15日，猛龍主場迎戰籃網。猛龍在首節拉開分差後，就牢牢佔據了比賽優勢，最後猛龍以110-102戰勝籃網。西亞卡姆上半場就轟下了25分，全場26投10中，三分球5中3砍下了30分11板5次助攻3次搶斷和3次蓋帽，輕鬆將籃網斬落馬下。

由於猛龍今夏失去了倫納德和丹尼-格林，猛龍的實力下滑嚴重，如今猛龍能夠佔據東部第5的位置，靠的是西亞卡姆在本賽季一飛衝天。西亞卡姆並沒有太多華麗的進攻技巧，他在本賽季雖然背框、面框和腳步都有所進步，但最擅長的還是融進球隊的打法，強行快攻沖入內線上籃以及空位三分。這種打法雖然不吸引眼球，但勝在效率頗高。

在首節籃網打出12-2的進攻高潮後，西亞卡姆憑藉三分完成了反超，隨後在首節還剩3分鐘時，西亞卡姆先是一個在三人防守下的強行快攻上籃得分，再加上一個底角三分，讓猛龍徹底拉開了比分，佔據了領先優勢。

本場比賽中段，丁威迪持球單挑西亞卡姆，丁威迪一個變向後向內線衝去，雖然西亞卡姆跟上了腳步，但是胳膊碰到了丁威迪，被打了一個2+1。這讓西亞卡姆有些不爽，籃網老大生吃猛龍老大的場面實在抹不開臉面。隨後兩個回合，西亞卡姆主動要球，他在弧頂面對丁威迪先是一個加速衝進內線上籃成功，下一個回合又主動找丁威迪防守，一個背後運球急停後，西亞卡姆在丁威迪面前穩穩命中了一個中距離投籃。

西亞卡姆的經歷與的熱火老大巴特勒極為相似，在他們進入NBA時，並不被人看重，兩個人都是在首輪末位被選中，在前兩年都只是角色球員。但隨後他們抓住了主將受傷和離開的機會，巴特勒在羅斯受傷時增加了球權，成為了公牛的新老大，由於去年倫納德經常輪休、今年洛瑞受傷，西亞卡姆也逐漸成長為球隊領袖。他們的逆襲之路簡直一模一樣。

​如今，西亞卡姆場均24.5分8.3板3.5助攻，儼然成為了球隊老大，而這似乎還不是他的極限，今天他半場就砍下了25分，還有足夠的表演空間。也許沒多久，西亞卡姆也會成為下一個巨星了。
