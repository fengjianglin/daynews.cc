一不小心就跳高了！詹姆斯失誤肩膀平筐，喬丹頭撞板後下場敷冰塊

2019-12-20 1,779條評論 4,085次閱讀 563人點贊

在12月18日的一場比賽中，持球進攻的格拉漢姆知道自己點起了追防過來的福克斯，抬手準備造犯規時卻發現什麼都沒碰到。這時候他才反應過來，原來福克斯直接從自己頭上飛了過去。這下格拉漢姆就直接驚呆了，連掉球都沒反應過來。

我們經常說NBA是籃球的最高殿堂，這裡彙集了全球最好的籃球運動員，無論在哪方面，他們都是最頂尖的。不過這個最頂尖總得有個衡量的尺度，不能光說。今天我們就來看下他們最頂尖的彈跳是什麼樣子，那真的是一不小心就從你的頭上飛過去了，下面就來一起看下吧。

大學時期的喬丹不講理的滯空高度讓他輕鬆送給對手大火鍋。躲閃不及後腦勺直接撞在了籃板上，只能默默下場休息敷冰。

我們都知道詹姆斯跳得高，至於到底有多高呢，他也不參加扣籃大賽，似乎就只能說說了，但一次比賽中的失誤球卻不小心暴露了他變態般的彈跳。這個空中接力不知道隊友怎麼想的，竟然傳了一個這麼高的球，不想失誤的詹姆斯就只能全力起跳先把球拿下來，結果高高躍起的詹姆斯肩膀都平筐了。

今年最沒有懸念的事情應該就是狀元的人選了，畢竟與蔡恩相比其他人都有些拿不上場面，因為他的身體天賦實在太出眾了，以至於一直拿來跟詹姆斯比較。單就彈跳來說，蔡恩確實達到了詹姆斯的高度。不僅平常的扣籃，就算是在籃板球爭搶中，蔡恩也是跳到了一個匪夷所思的高度，肩膀都要超過籃板下沿了。

這個時候喬丹又出現了，為了表現自己的與眾不同，他乾脆直接來了個平筐跳投，這就是著名的「The shot」。從圖中我們可以發現，喬丹的頭都已經平筐了，硬是等防守球員落地他才把球投出去。就是比你跳得高，這讓防守球員到哪說理去？

摸籃板上沿一直是流傳於球迷中的一個傳說，許多球員都曾被傳出過可以摸到籃板上沿，最後都證實是假的。不過也不是真的沒人能做到，最起碼張伯倫是真的有可能。這張照片中的張伯倫握拳就已經接近籃板上沿了，手掌伸開的話，或許他就可會摸到籃板上沿第一人了。

NBA中最激動人心的就是各種隔扣了，各路飛人們都有將防守球員扣翻在地的表現。興許是隔扣得太多球員了，這次字母哥就選擇了一個十分誇張的方式，只見他在快攻空接中直接從小哈達威的頭上飛了過去完成扣籃。飛過去就算了，採訪的時候還說根本沒注意到小哈達威，看來是真的是飛太高了。
