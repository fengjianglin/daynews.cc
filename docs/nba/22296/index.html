詹眉組合和OK組合誰更強？兩萬球迷參與投票，沒想到單方面碾壓

2019-12-20 1,159條評論 5,003次閱讀 3076人點贊

湖人本賽季截至目前的表現無疑是令人驚訝的，在上賽季沒能拿到季後賽入場券的情況下，他們在休賽期進行了合理的人員補充，而且收到了顯著的成效。在詹姆斯和濃眉哥的帶領下，湖人新賽季一騎絕塵，一直高居聯盟第一。而球隊整體的狀態和表現也是令球迷十分滿意。他們對於比賽的控制方面做的十分出色，在領先的時候能夠很好的把控場上節奏，而在落後時，也可以通過全隊的努力來完成逆轉。

而詹姆斯和濃眉哥的雙核配置也是十分合理，兩人在場上的表現也都十分出色。而他們目前在本賽季常規賽賽程已經結束1/3的情況下，僅僅輸掉了3場比賽，而前段時間，也有不少球迷都表示湖人很有可能會打破勇士此前創造的單賽季73勝場的記錄。而且照這個勢頭下去，詹姆斯距離自己的第四座總冠軍獎盃也不再遙遠了，而湖人也將很有希望能夠在科比退役之後，再一次打進總決賽，站上最高領獎台。

而對於湖人球迷來說，他們無疑是感到無比興奮的，他們期待著球隊能夠將這樣的態勢保持下去，拿到更出色的戰績和表現。而也有許多球迷，已經開始將詹姆斯和濃眉哥的組合視作為湖人最強二人組了，不過，這一說法也同樣遭到了許多球迷的質疑和反對，他們認為湖人歷史最強二人組依舊是科比與奧尼爾。詹姆斯和濃眉還沒有任何的榮譽到手，現在就這麼說的話，實在太過草率。

而就此，有媒體也是針對這一爭議發布了一項投票活動，投票內容就是奧尼爾與科比的組合與詹姆斯濃眉的組合，誰更強？而在該投票活動發布之後，也是有接近兩萬名球迷參與了投票，只不過，投票的結果卻是走著一邊倒的勢頭，有超過3/4的球迷都依舊堅持認為OK組合才是湖人最佳二人組。

而這樣的投票結果也是符合常理的，畢竟科比與奧尼爾攜手可是為湖人拿到了三座總冠軍的獎盃，而且還是三連冠。他們連續四年打進總決賽，完全的統治了西部，而且在當時的聯盟中，幾乎沒有球隊可以與之相抗衡。而詹姆斯如今都已經35歲了，濃眉哥的影響力自然也是不如科比和奧尼爾，況且這也只是他們合作的第一個賽季，季後賽的表現才是對他們最好的考驗。
