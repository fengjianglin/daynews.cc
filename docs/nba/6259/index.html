雄鹿是東部最好的球隊么 與其他東部豪強對比 有什麼優勢和缺陷

2019-12-03 0條評論 18次閱讀 0人點贊

北京時間12月3日，雄鹿在主場以132:88大勝到訪的紐約尼克斯，東部排名第一的雄鹿面對東部排名倒數第一的尼克斯，兵不血刃的將分差拉開到兩位數追不上的鴻溝，東部最強球隊與東部最弱球隊之間的差距肉眼可見。

本場比賽結束後，雄鹿迎來12連勝，12連勝是雄鹿本賽季最長的連勝，也是整個聯盟最長的連勝，戰勝尼克斯以後，雄鹿以18勝3負，超過湖人，戰績位列全聯盟第1，那麼目前這支雄鹿是全聯盟最強的球隊么，在東西部與競爭對手有什麼差距么？

從目前來看，雄鹿本賽季輸球的三場比賽，分別輸給爵士、熱火和凱爾特人，由於雄鹿目前處於東部，東部球隊的威脅應該更大，東部球隊的即戰力對比，應該更重要。

從目前的排名以及雄鹿輸球的對手來看，東部第2的猛龍，目前15勝4負，與雄鹿相差2個勝場，本賽季兩隊交手過一次，雄鹿以115:105戰勝猛龍，東部第3的熱火，14勝5負，東部第4的凱爾特人，14勝5負，東部第5的76人15勝6負，這4支東部的球隊與雄鹿有直接競爭關係。

從各隊的配置以及即戰力分析，不考慮臨場發揮，這4支球隊能對雄鹿競爭總冠軍構成威脅么？

雄鹿VS猛龍

上賽季兩隊在季後賽的交手歷歷在目，猛龍用萊昂納德主防字母哥，卸掉了字母哥勁爆的衝擊力，利用瘋狂的快速轉化，防得雄鹿外線面對高壓完全失准，而猛龍依靠萊昂納德超強的發揮，西亞卡姆和范弗里特的發揮，在雙方大打防守戰的情況下，憑藉主攻手的全面性和多樣性，戰勝了雄鹿。

本賽季，萊昂納德和丹尼·格林離開猛龍，但是猛龍並沒有呈現光速下滑的趨勢，西亞卡姆的快速成長以及阿奴諾比的復出，其防守的紀律性和執行力並不比上賽季差，其在防守端快速輪轉的特質，並沒有因為萊昂納德和格林的離隊而降低，反而強度更上一個層次，面對主攻乏力的球隊，猛龍能防得讓其得不了分。

而在進攻端，猛龍依靠團隊籃球，注重時機的合理性，其三分命中率為40.2%，為全聯盟三分命中率最高的球隊，比第二的活塞高出0.9%，其36.2次三分出手，比活塞多了4.9次出手，看得出猛龍走的是兼顧效率兼顧產量的團隊籃球。

從對位而言，猛龍西亞卡姆能對位字母哥，小加索爾能對位布魯克·洛佩斯，洛瑞能對位米德爾頓，范弗里特能對位布萊德索，阿奴諾比對位馬修斯，兩隊先發陣容都可以對位，關鍵看球員的臨場發揮。

雄鹿VS熱火

熱火面臨的第一個問題是誰主防字母哥，阿德巴約還是奧利尼克？無論是誰，從單防的角度而言，都有些困難，我們需要看其他球員的對位情況，雄鹿拿誰來主防吉米·巴特勒？馬修斯的腳步移動速度根本跟不上吉米·巴特勒，似乎更有可能是用米德爾頓主防吉米·巴特勒，米德爾頓能防住吉米·巴特勒么？似乎有點難度，從這種角度而言，雙方的頭號球星，都有對方單防解決不了的問題。

主要看其他球員的支撐，目前熱火第二得分手是納恩，第四得分手是希羅，這兩位球員都是新秀，熱火的不確定太多，而雄鹿洛佩斯、伊利亞索瓦、康諾頓三名重要輪轉球員三分命中率在30%左右，有回暖的空間，不確定也很多，雙方誰都有可能擊敗另一支球隊。

雄鹿VS凱爾特人

從防守端來看，凱爾特人的防守資源極其豐富，有斯瑪特、杰倫·布朗這樣的頂級側翼防守人，也有泰斯這樣蓋帽率和後場籃板率大幅度提高的內線球員，坎特本賽季在防守端又有一定的進步，凱爾特人採用包夾的策略，能夠應對字母哥的衝擊，凱爾特人本賽季的防守紀律性以及防守輪轉也能做到防守快周轉。

主要問題還是在進攻端，海沃德又受傷了，本賽季狀態有一定回暖，如果復出後，狀態存疑，同時肯巴·沃克和坎特都是防守端有漏洞進攻端特別厲害的球員，非常考驗斯蒂文森教練，從配置和陣容的試錯率來看，雄鹿明顯比凱爾特人好，但是臨場發揮又是另外一回事。

雄鹿VS76人

用恩比德主防字母哥，用霍福德主防洛佩斯，用西蒙斯主防布萊德索，用里查德森主防米德爾頓，用哈里斯主防馬修斯，從首發陣容的實力對比來看，其實76人比雄鹿還要好一些。

但是76人的替補實力完全拿不出手，替補場均得分最高的科爾克馬茲場均只有9.1分，而雄鹿板凳陣容的深度和廣度明顯強於76人，但是如果到了季後賽，又是首發陣容佔據大量出場時間的系列賽，替補的輪轉陣容出場時間必然不會很多，雄鹿與76人的差距主要在替補陣容的銜接段，但是在季後賽，這樣的差距明顯會被縮小，從這種角度來看，雄鹿的優勢並不明顯。

雖然迎來12連勝，雖然本賽季東部球隊的魚腩球隊更多，但是球隊的即戰力更加集中，雄鹿本賽季輸給熱火和凱爾特人說明其在東部的地位並不穩固，面對另外4支球隊實力並沒有明顯高出多少，反而使得本賽季東部頂級球隊的競爭更加激烈。
