三分最強的扣籃王？職業生涯遭逢僵局，拉文要怎麼做才能更進一步

天天要聞
2019-12-20

印象中，扎克·拉文似乎在聯盟中待了很長時間，他的名字也從默默無聞到世人皆知。只有24歲的他目前已經來到了職業生涯第6個賽季，同屆選秀球員中幾乎沒有比他更年輕的人，而他此前也遭遇過眾多球員未曾經歷的事情。

兩度拿下扣籃大賽冠軍，從明尼蘇達到芝加哥，球隊地位也從第三選擇成為了絕對核心。此時此刻的他已經是球迷們口中的老球員，從十字韌帶撕裂的傷病中恢復過來，他將競技狀態提升了一個等級，可始終未能取得更大的成就。觀察到他的榮譽榜單，一切也只是停留在全明星賽和新秀時期的獎項罷了。

要說拉文的名氣大不大，他以一己之力再次讓扣籃大賽成為全明星周末最值得期待的表演，現役球員中或許只有還沒退役的老卡特能夠在扣籃這件事情上壓他一頭。要說他的數據優不優秀，連續兩年場均得分穩定在23分以上，個人數據和效率在東部的同位置球員上都是排名前列的。但你要說拉文為什麼沒能帶領公牛崛起，又或者說他的能力還能不能更上一層樓？那麼現在還沒有人可以給出準確的答案。很顯然他遭遇到了職業生涯中最大的僵局，他的數據與上賽季基本無異，也沒有更多的手段提升自己的實力。對他而言，要做些什麼才能夠突破這一個僵局呢？下面就讓我們來詳談一下。

三分最強的扣籃王？進攻範圍越來越靠外

在說到拉文目前陷入的僵局之前，不妨讓我們先說一下他現在的打法方式和特點，也便於更加深入地了解這個球員的特性。儘管我們將他視為歷史最傑出的扣籃王之一，但比賽中的他呈現出截然不同的風格，扣籃不過是他的特長之一而已。

個人數據統計

從新秀賽季開始，他的三分投射就成為了佔比最大的得分手段，那時候他場均就會投出2.2次三分。當他在森林狼的最後一個賽季，他的場均三分出手達到了6.6次，整體佔比總出手次數足有43.3%，也許這一點就足以讓許多球迷感到驚訝了。

來到芝加哥的三年他也仍舊延續了這個習慣，這個賽季正是他將個人風格發揮到極致的時候，以39.5%的三分命中率場均出手7.6次三分，這樣的命中率和命中數都讓他成為了聯盟最出色的外線投手之一。毫不誇張地說一句，在歷史上這麼多拿到過扣籃王稱號的球員中，他的三分能力理應是最強的。即便是卡特生涯最高的三分出手數也只是5.3個，命中率也稍遜於拉文。

幾年前的聯盟對三分的鼓勵程度還沒有現在這麼高，那時候的拉文也沒有這麼痴迷於三分，他的遠距離兩分佔比曾一度高達29.2%。這令他的打法風格更像是古典分衛，可是他卻在發展過程中逐漸偏離了傳統分衛的軌道，如今的他實行著聯盟最潮流的魔球理論。這個賽季的三分佔比有41.9%，反觀籃下出手佔比也有35.4%，剩下不到25%才是中距離投籃。

賽季出手佔比

他的三分球不需要過多依賴隊友助攻，大部分都是持球過程中的運動戰三分，干拔跳投的樣子還頗有當年科比的一番風味。魔球理論的好處我們也曾說過多次，在同等出手次數下，有著魔球習慣的球員能夠得到更多的分數，畢竟籃下和三分都是最高效的得分方式。拉文也就是堅信著這個道理，只可惜現階段的他還沒有完全發揮出魔球的好處。

或許是由於傷病的問題，他選擇突破的次數明顯變少了，整體的攻擊範圍都是越來越靠外了。正是從以往的打法改變到如今的魔球打法使得他面臨現在的困擾，而他沒辦法突破困境的原因也十分簡單，下面這兩個觀點也許能夠解答大家心中的疑問。

內線終結能力不足，缺乏有效的進攻手段

拉文有著後衛線上最標準你的身材比例，1米98的身高配上2米04的臂展，這在他初入聯盟的時候為他增添了許多優勢。而他的彈跳高度和速度甚至是滯空時間也給他帶來很多方便，這讓他曾連續兩年的內線命中率都超過了60%，可往後卻逐漸變低了，這個賽季的內線命中率也只是51.1%。他的扣籃能力強大並不代表他的內線終結能力就一定強大，縱使是詹姆斯也不能每次內線突破都完成扣籃，拉文顯然也不可以。

而他的內線命中率之所以這麼低，主要是因為他除了扣籃之外就沒有更好的終結手段了，不管是上籃還是拋投都缺乏相對應的柔和手感，這就是他目前陷入僵局的第一個原因。這也導致他失去了魔球帶來的最大好處，那就是高效的得分，他的真實命中率還有各項進階數據也能夠明白這一點。

六個賽季當中，他真實命中率最高的時候反而是在第三個賽季，來到公牛之後的真實命中率最高只有57.4%。然而那時候的拉文還沒有開始轉變到魔球打法，他又是怎麼樣得到生涯最高真實命中率的呢？這是由於當時的拉文還熱衷於沖往內線，雖然他的場均罰球次數遠不如現在高，但是在當時較低的出手次數下獲得場均3個罰球已經算不錯了。

傷病的陰影還是會時不時影響他的表現，現在的拉文沒有了當初的身體對抗能力。他很少敢於做動作，所以只能夠白白錯過一些以往很容易就能夠命中的球。如果從進階數據還有各項效率來看，16-17賽季的他才是巔峰狀態。現在他的場均得分雖然對比那時候上升了4.3分，可是卻多用了3次出手，這種得分效率顯然是不夠高的。

失誤統計

反觀哈登這位魔球大師，即便他的在賽季初段一度低迷，但如今的內線命中率也提升到了56%。即便哈登在身體失去平衡的情況下仍舊能夠將球打進，他的突破能力也是不需要質疑。再看看拉文，他一方面減少了自己的場均突破次數，另一方面也沒有了當初的滯空能力和爆發力，兩者結合下他又怎麼可能更進一步呢？他在持球三分這方面確實很有一套，但是持球突破能力卻不算特彆強。

在遭遇包夾的時候很容易就會被搶斷，場均3.3次失誤就是這麼來的。並且他沒有辦法和隊友間產生有效的溝通配合，擋拆掩護過後就只能突破，就連以往熟悉的中投都捨棄了。說到底他還是沒能擁有更多樣化的進攻手段，而且更加沒有一種威脅巨大的方法減緩對方對他的防守壓力。沒有多一種有效的進攻手段成為了限制拉文的第二個原因，兩個原因相加之下就讓他的個人能力看起來像是在原地踏步了。

球隊位置徘徊不定，等待球隊尋來搭檔

可能是拉文出色的身材加上聯盟眾多雙能衛例子的影響，這讓公牛管理層認為他也能夠成為雙能衛，可是卻沒有考慮到拉文是否具備足夠的持球能力還有組織串聯技巧。去年他場均可以拿到4.8次助攻，這已經是生涯最高點，而這個賽季只能有3.8次助攻進賬。

他其實更加偏向於純粹的得分手，組織方面的工作他不是特別精通，也很難去學會。這也能夠和上面提到的突破問題扯上關係，為什麼拉文的突破顯得這麼費勁呢？其實就是因為他不知道該怎麼和隊友配合，隊友為他做出掩護的時候也不知道該往什麼地方去，到底是傳球還是直接突破都沒能做出好的選擇。然而他的回合佔有率卻是聯盟第14高的，也就是大部分時候球都在他手上，讓這麼一個不太懂得組織串聯的球員來過多持球顯然不是一件好事。

進階數據統計

尷尬的球隊定位很容易就會讓拉文沒辦法把握住成長機會，他已經改變過一次打法風格，頻繁地改變風格也是不切實際的方法。其實拉文現在需要的東西不是很多，他只是要一個能夠在進攻端釋放他持球時間的控衛罷了。現在公牛隻有一個場均助攻比拉文高的球員，可即便是他也沒有很好的處理球能力。這就讓拉文的無球能力逐漸被遺忘，實際上他的無球能力並不差，更多地反跑和空切反而可以降低他在內線受到的防守壓力。

當防守人的注意力不在他身上的時候，那麼他自然可以更輕鬆地得分。就像是去年的布克一樣，現如今拉文只要一拿球就會被包夾，而破包夾能力和出球能力都不夠好的他很容易就會被鎖死。在盧比奧到來後，布克已經打出了生涯最佳表現，那麼拉文什麼時候才能夠迎來屬於自己的盧比奧呢？這隻能等公牛管理層的操作了。

結束語

身處一支持續擺爛的隊伍很難會得到球迷的關注，尤其是他只簽下了均價2000萬的合同，這讓他的起步比起同齡的球星慢了很多。事實上他能夠走出傷病並且連續兩年打出23分的表現已經很艱難了，但倘若他一直停留在這個階段，那麼他依舊不能得到更多機會。這已經是他第二份合同的第二個賽季，他至今還沒進入全明星正賽，要儘早突破僵局才是明智之舉。
