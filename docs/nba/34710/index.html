真性情的威少，正是這支火箭隊需要的潤滑劑

2019-12-25 50條評論 1,591次閱讀 636人點贊

曾經，二哥和三弟是MVP的競爭對手，那時的我，雖然沒有加入討伐二哥的大軍，但對二哥的三雙也是不以為然的，甚至是嗤之以鼻的。靠中鋒幫著卡板有什麼可推崇的。

今年二哥來了火箭，真正的逐場看過來，才發現二哥的沖搶能力真強，卡位能力真強，對抗中搶板能力真強，後衛中的羅德曼的感覺。拿板後的推反擊真的犀利。原來曾經以為威少拿籃板只是靠隊友卡位真的是偏見。

我們都有喜歡的球員，我們都希望自己喜歡的球員是無所不能的，是戰無不勝的，於是慢慢的，我們把自己的期望投射在自己喜歡的球員身上，我們認為喜歡的球員成為了自己理想中的樣子，為了維護他，也為了維護自己的美好理想，但自己卻成為了自己當初討厭的樣子。

來到了火箭的威少，讓我認識到自己當初的傲慢與偏見。是時候放下不切實際的投射，活成想要的自己才是最重要的。我想要自己善良、公正、客觀、勇於承擔、敢於認錯！

請原諒我當初的傲慢與偏見，不過我已經付出了代價，我永遠失去了第一時間欣賞史詩級威少的機會，希望我能做回想要的自己，不再錯過其他見證偉大的機會。
