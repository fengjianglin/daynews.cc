下一個倫納德？科爾稱自己用波波維奇培養小卡的方式培養23歲新秀

天天要聞
2019-12-19

史蒂夫科爾作為教練來說經驗相對來說比較少，但是他卻成為了NBA最傑出教練之一。與其說是他成就了勇士隊，不如說是他們互相成就。但作為教練，他的很大一部分經驗都來自於他的老師——馬刺主教練波波維奇。波波維奇的履歷就不必說了，成就了馬刺王朝，強調傳球的重要性。而最近，科爾稱自己在用波波維奇培養倫納德的方式培養新秀帕斯卡爾。本賽季的帕斯卡爾場均16.1分5籃板1.7助攻，具有爭奪最佳新秀的能力。

我們先說回馬刺培養倫納德，波波忍痛割愛交易了自己的愛徒喬治希爾，得到了倫納德。當時的馬刺管理層包括波波就已經看到了倫納德的得分能力，身體天賦，但是缺點也很明顯，對於比賽中如何帶動隊友，如何傳球方面一無所知，傳球視野也很有限。

據記者爆料，當時的波波維奇故意讓小卡充當一名傳球手的角色，讓他把球傳到側翼，看到每一位隊友。而科爾正是採用了相同的方法來培養帕斯卡爾。帕斯卡爾也是一名天生的得分手，得分能力很強，但是在不削弱進攻的情況下，增進傳球的能力並不簡單，如今的科爾正在向他灌輸波波的經驗。

要知道，上賽季來到猛龍的倫納德幾乎成為聯盟頂級巨星，並帶領猛龍奪得總冠軍，他場均能送出5.1次助攻，而這一數據在他新秀時期只有1.1次。本賽季來到快船，小卡場均仍然能送出5個助攻，自己場均也能得到25.5分，已經進入聯盟頂級球星的行列。

當然，要把帕斯卡爾培養成這樣的傳球手絕非易事，作為新秀他的第一個賽季已經非常出色了。但是要成為和倫納德那樣的球員，他還有很長的路要走。
