你以為這些是謠言？其實是真的！97喬丹遭人投毒，麥迪訓練不努力

天天要聞
2019-12-23

網路時代的興起有利也有弊，有利的方面是大家可以通過網路，快速的了解到自己想知道的信息，而弊端的一面就在於，很多目的不純的人為了博取眼球，利用網路傳播速度快的特點，大量的製造虛假新聞，來迷惑眾人，不過有些新聞可能太過於出人意料，或者說是有些聳人聽聞，造成了這些新聞被當成了謠言，比如下面的這些。

卡特泄露球隊戰術

對於卡特的球迷來說，小編首先要道個歉，因為卡特職業生涯中這個最大的污點的確是真的，在2004年11月19日猛龍對陣超音速的比賽中，終場前28秒時猛龍隊還落後超音速隊7分，並不是沒有發生奇蹟的可能性，但是在猛龍暫停布置完戰術後，出現了讓人意想不到的畫面，卡特站在超音速替補席前，對著超音速替補席喊道「下面是一個閃光戰術」。接下來猛龍隊其他球員果然掩護卡特跑了一次閃光戰術，反應過來的超音速替補球員馬上大聲的提醒自己的隊友，而在比賽結束後，有三名不願意透露姓名的超音速球員在接受採訪時，確定了這件事情的真實性，同時也在當時引起了聯盟的震驚。

易建聯年齡造假

早在阿聯還效力於NBA期間，就曾經傳出過有關阿聯年齡造假的新聞，不過這卻被當年的國內球迷嗤之以鼻，認為是不可能的一件事情，認為的謠言，可是經過實際調查，阿聯的年齡的確存在著更改，阿聯中學時期的學籍表上，清楚的寫著出生於1984年，而國家二隊球員的註冊列表中，阿聯的出生日期同樣是1984年，還有當年的醫保卡事件，廣東省社會保障基金管理局的網站上，有兩個易建聯，一個在1984年出生，另一個則在1987年出生，而84年出生的易建聯，在2006年時停止了繳費，這些都能證明，阿聯年齡的更改已經得到了證實，但是這也已經不重要了，畢竟阿聯對於中國籃球的貢獻是有目共睹的。

麥迪訓練不努力

之前就有謠言稱，麥迪訓練不努力，不僅會在訓練中休息，還經常減少一些其他球員都必須訓練的科目，其實這個謠言是真的，麥迪的確經常出現這種情況，只不過其中的原因並不是因為麥迪偷懶，而是因為麥迪患有「先天性脊柱炎」這種疾病，很多和背部有關的訓練項目，麥迪是不被允許完成的，因為這樣的訓練非常有可能給麥迪造成終身癱瘓的後果，而麥迪在訓練中休息也是因為這個原因，一旦背部出現不適，麥迪是被要求需要強制性休息的。

喬丹97年被投毒

喬丹97年的流感之戰，已經成為了聯盟歷史上的經典，但是喬丹因何患病，成為了當時人們的一大猜測，其中被人投毒這個謠言，可信度最高，流傳的也是最廣，不過喬丹當年的御用訓練師格羅弗在接受媒體採訪時，表示他100%可以確定，當年的喬丹的確是被人投毒了，而且講出的理由十分充分，要知道當時爵士隊主場所在的城市並沒有多少酒店，而喬丹所點的披薩外賣很久才送到，而且是5個人一起送上來了，並且這份披薩只有喬丹一個人食用，等到凌晨兩點時喬丹已經出現了劇烈不適反應。

奧尼爾打過科比

​科比和奧尼爾當年的矛盾人盡皆知，甚至在當年有謠言稱奧尼爾曾經打過科比，雖然很多人不信，不過奧尼爾打過科比這件事確實是真的，並不是媒體造謠，在湖人三連冠期間的訓練中，奧尼爾和科比發生了爭執，而奧尼爾重重的給了科比一拳，後來奧尼爾在面對採訪時也曾經說過，他當年差點宰了科比，這無疑確定了奧尼爾曾經對科比動手的真實性，而另一位當事人科比，也在面對媒體採訪曾經回應過這件事，甚至還開玩笑稱，當年他一定是瘋了，才敢去挑釁這樣一個大傢伙。
