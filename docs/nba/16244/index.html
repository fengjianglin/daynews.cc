巴克利評現役前5巨星：倫納德第5，哈登第2，庫里第幾呢？

2019-12-14 2,835條評論 4,883次閱讀 359人點贊

在科比退役以後，詹姆斯就理所當然的開始獨享現役第一人的頭銜，而他在前幾個賽季的出色表現也是讓他完全無愧於這一頭銜。不過，隨著年齡的增大，詹姆斯的統治力略有下滑，而聯盟中也開始漸漸湧現出更多表現出色，天賦出眾的新星。而尤其是在上賽季倫納德帶領猛龍擊敗勇士，奪得冠軍以後，他成為第一人的呼聲也變得更高了起來。此外，在總決賽中受傷離場的杜蘭特同樣已經向我們展現出了他的強大之處，當然，還有連續兩個賽季拿下得分王的哈登，他也同樣是第一人的有力競爭者。

而對於這個話題，聯盟名宿，被球迷們稱為「大嘴」的巴克利也是在一次採訪中給出了他的看法，他評選出了自己心目中的聯盟現役排名前五的巨星。而對於這些曾經在聯盟中叱吒風雲的傳奇巨星們來說，他們的選擇或許也會更加的具有參考性，那麼下面我們就來一起看一看巴克利心中的現役五大巨星都分別是誰。

巴克利在被問到這個問題後，第一時間表示他反對美國媒體所評選出來的那個排名，他並不認為庫里能排進前五，在他看來，庫里的排名應當是在第七位。而當時的主持人對於巴克利的回答也感到驚訝，反問道庫里可是三次奪冠的球員，而且還帶隊拿下過73勝場的記錄。

但巴克利稱庫里卻從未拿到過FMVP，而且他在防守端的表現不足以影響到比賽的結局，這也就是球星和巨星差距。而且巴克利認為勇士是有著四位球星的，庫里的成功更多的依賴隊友和體系，拋開這些的話，他也只是一名出色的射手而已。

而之後，巴克利就給出了自己心中的現役前五巨星，他們分別是詹姆斯，哈登，杜蘭特，字母哥，倫納德。巴克利表示這幾個人都可以排在庫里的前面，而倫納德是一位在攻防兩端都可以影響到比賽勝負的球員，他在去年總決賽中的表現也是令人感到驚訝的。字母哥是上賽季常規賽的MVP，在這個小球時代中，他在內線就是絕對的霸主。他的終結能力和身體素質都是十分罕見的，如果他可以順利的成長起來，未來聯盟的第一個非他莫屬。
