末節連砍7分終結湖人客場14連勝，這可是被稱作總統的男人

2019-12-18 109條評論 2,108次閱讀 1167人點贊

看布羅格登打球，你會感到踏實。今年夏天，他將自己的踏實和穩健帶到了步行者。

如果你對這種感覺依舊不強烈，那麼在看過今天步行者對陣湖人比賽的最後時刻，想必你會同意我的。

比賽最後3分18秒，步行者落後5分，布羅格登借薩博尼斯掩護，在右側45度三分出手，皮球穩穩命中，將分差追至兩分。

2分21秒，布羅格登再次選擇和薩博尼斯配合，在後者擋拆下，突至三秒區內，小手一抖，拋投命中，追平比分。

41秒，面對霍華德的防守，小打大，奪步上籃，空中閃躲，反手打板命中，幫助步行者領先兩分，奠定比賽勝局。

儘管不是明星球員的地位，但最後這幾個球，打出了明星級別的表現。穩如巨輪的布羅格登，帶著步行者擊退了西部第一的湖人。

此役勝利，他們終結了湖人的7連勝，以及湖人的客場14連勝，同時完成了自己對陣湖人的主場7連勝。

本以為明天湖人對陣雄鹿的比賽，將上演各自分區老大誰能終結對方連勝的戲碼，沒想到開打之前，雙方先把連勝包袱卸下了。

霍華德今天10中10打出完美表現，但還是被布羅格登搶了風頭。

說回布羅格登，他在最後時刻的關鍵表現無疑是決定比賽的勝負手。對於一直擁有詹姆斯和濃眉的球隊來說，湖人是十分擅長打關鍵球的。

湖人在本賽季最後三分鐘，分差在3分以內的比賽里，一共10勝1負，今天之前，十戰全勝，今天折在了布羅格登手裡。

目前，步行者以19勝9負的戰績位列東部第6，除了雄鹿一騎絕塵外，步行者和76人、綠軍、熱火及猛龍戰績相差不大，同屬第二梯隊。

要知道，這是一支缺少了核心奧拉迪波的步行者。而填補他的空缺，並幫助球隊取得不錯戰績的球員自然就是布羅格登。

今年夏天，布羅格登以4年8500萬美元的合同加盟步行者，這筆簽約是雙方都滿意的結果。現在看來這筆簽約對於步行者來說，確實是值得的。

布羅格登賽季至今場均出戰31.5分鐘，場均拿下18.9分4.5籃板7.6助攻，其中籃板和上賽季持平，得分和助攻都刷新職業生涯賽季新高，助攻是上賽季的兩倍有餘。

數據的提高，也能理解，畢竟上賽季在雄鹿，他身邊是字母哥、米德爾頓、血布等人，上賽季他的使用率是20.7%，本賽季提高至26.3%。

機會變多是一回事，能抓住機會也是一回事，我們不能否認布羅格登對機會的把握，以及獲得機會後的穩定發揮。

在奧拉迪波缺陣的步行者，布羅格登是這支隊伍的領袖和核心。從結果來看，他完成得不錯，今天的表現更是論證了這一點。

雖然這是一支濃眉缺席，打了折的西部第一，但就像步行者主教練所言：能夠擊敗聯盟戰績第一的球隊，這意味著很多。

這個24歲以36順位進入聯盟，1966年以來首位當選最佳新秀的大齡二輪秀球員，現在第一次成為了一支球隊的老大，也有可能只是暫時的。

老成的布羅格登很難成為一名超級球星，他的打法並不華麗，他的天賦也並不那麼出眾。


但是他在很長一段時間都會是人們口中的好球員，也會是好的領袖，畢竟大家都叫他「總統」。
