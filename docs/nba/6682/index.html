湖人備戰鵜鶘欲奪9連勝！AD肩傷未愈存隱憂，兩致勝定律仍延續

2019-11-27 0條評論 14次閱讀 0人點贊

北京時間11月26日，湖人在客場擊敗了馬刺拿下了8連勝，跟馬刺的對決不再是險勝1分、3分了，而是贏了馬刺10分，這場比賽詹姆斯的發揮極為出色，全場比賽24中13，三分7中4，拿下33分14助攻3籃板1蓋帽，第三節單節送出了7次助攻，詹姆斯將全隊串聯，湖人6人上雙擊潰馬刺，拿下8連勝後湖人戰績15勝2負依舊是聯盟第一，距離湖人開局最佳戰績17勝2負還差兩場勝利，在08-09賽季湖人在科比的帶領下曾打出湖人隊史最佳開局戰績17勝2負，如今詹姆斯有望帶隊衝擊這個紀錄，當然拿下9連勝是目前最關鍵的一步。

湖人隊官方在11月27日上傳了球隊踏上新奧爾良的征程，打鵜鶘隊是四連客的最後一個客場之旅，之前湖人並未跟鵜鶘交過手，也就是說這是AD大交易後，英格拉姆、哈特、鮑爾首次面對舊主湖人，也是戴維斯首次打舊主鵜鶘，這場比賽的關注度很高，很多球迷都期待英格拉姆與詹姆斯的對位，戴維斯面對好友奧卡福、霍勒迪會打出什麼樣的數據。當然鵜鶘的硬實力跟湖人差距很明顯，鵜鶘戰績6勝11負西部第11，進攻效率109.7聯盟第10，防守效率114聯盟第27，他們最弱一環是內線防守，坐擁AD、麥基、霍華德的湖人優勢很明顯。

不過安東尼-戴維斯的肩傷依舊是一個隱患，目前尚不知他在對陣鵜鶘的比賽中是否登場，NBA傷病報告在11月27日早上7點前還未更新，戴維斯在對陣馬刺的比賽前就是右肩酸痛出戰存疑，可他還是頂著疼痛打了33分鐘，19中7，三分6中1，拿下19分12籃板6助攻2蓋帽，肩膀的傷勢影響了AD的投籃手感，面對鵜鶘這樣的內線防守弱隊，若是健康的AD拿下40+都不成問題，可AD最近飽受肩傷困擾，他若無法打出統治力，就又得讓詹姆斯盡全力衝擊三秒區了。

這第9場勝利很重要，前面湖人已經連續擊敗了太陽、勇士、國王、老鷹、雷霆（2場）、馬刺，擊敗鵜鶘後回到主場第一場是打奇才，10連勝就要來臨了，不過四連客+背靠背的湖人隊體能是一個大問題，好在這支湖人隊有兩項致勝定律傍身，按理說是不會輸給鵜鶘這支年輕球隊的。

第一項定律：波普定律

波普定律目前依舊有效，當波普得分超過0分時湖人沒有輸過球，波普在與快船、猛龍的比賽中都得0分，恰巧兩場比賽都輸了，之後的15場比賽波普都有得分湖人都贏球了。雖然這個定律很扯淡，可目前的確還沒有破功，其次在布拉德利受傷後波普的狀態越來越好，最近的6場比賽場均14.2分2.7籃板 1.2助攻1搶斷，命中率61.2%，三分命中率61.5%，場均2.7個三分球，波普成為湖人第三得分點，當他有如此高效的狀態，湖人贏球的概率當然會增大。

第二定律：隆多定律

隆多定律也是不敗定律，在隆多11月13日復出後湖人保持不敗，隆多本賽季缺席了10場比賽，湖人8勝2負，隆多復出的7場比賽湖人全勝，隆多復出後減輕了詹姆斯的組織進攻壓力，減輕了波普、丹尼-格林的後場防守壓力，加之他本人的進攻還打的有聲有色，他在替補席成為了神機軍師，他復出的7場比賽，場均9.7分3籃板5助攻1.3搶斷，命中率42.3%，三分命中率45%，場均1.5個三分球，隆多隻要繼續登場大概率就是繼續贏球。

雖然丹尼爾斯的發揮算不上致勝定律，但在布拉德利傷停，卡魯索還未痊癒的情況下，丹尼爾斯能在17分鐘里6中4，三分3中3，拿下11分1搶斷的奇兵表現，這對於湖人也是好事一件，空間更大，詹姆斯的傳球選擇更多，這支湖人拿下9連勝應該沒有太大的問題。
