多人感冒還能贏？除開意志力，湖人兩替補功不可沒

2019-12-04 0條評論 13次閱讀 0人點贊

要說美國本土最冷的城市是哪座，丹佛絕對榜上有名。據悉，丹佛是美國海拔最高的城市，丹佛城市海拔距測量達到了1600米之多。由於正好是一英里的長度，美國人稱它為「里高城」。也正因如此，丹佛異常寒冷。在NBA球員的心中，丹佛的城市球隊掘金隊的主場，也就成了遠近聞名的魔鬼主場。這不，今天前來里高城做客的洛杉磯湖人隊，幾名主力球員都感冒了，湖人差點就栽在了掘金這群小夥子手裡。

還好最終有驚無險，湖人眾將憑藉頑強的意志，通過了身體和對手的兩重關卡，以105-96的比分帶著勝利離開了寒冷的里高城。

本場比賽之前丹佛掘金距離湖人也就一個勝場，所以本場比賽很大程度上也是榜首之爭。由於丹佛高原特殊的環境，坐鎮百事中心的丹佛掘金自然佔據了天時地利人和。然而比賽一開始，坐享安逸的掘金隊球員大意失荊州，一開始就被湖人打了一個10-0，就此開始了漫長的追分之旅。湖人隊雖然遭遇到感冒危機，但是球員們卻越挫越勇，在上半場始終壓制著掘金隊。半場結束，他們甚至取得了兩位數的領先優勢。

湖人的感冒危機終究成了攔路虎，下半場湖人隊員體力下降。掘金的後場雙子星穆雷和哈里斯趁機帶領球隊進入追分模式，掘金一度反超比分。然而姜還是老的辣，詹姆斯連續取分又將分差再次拉開到8分。到了最後一節，掘金依然是抓住湖人球員體力不足的問題，逐漸將比分縮進。危急關頭，又是34歲的老詹站了出來，他的一記扣籃又喚醒了湖人隊友的鬥志。一個5-0的小高潮，湖人收走了比賽。

據湖人隊三旬老漢勒布朗-詹姆斯透露，在今天比賽之前，包括他在內的至少六名湖人核心球員都有感冒的癥狀。比較明顯的有他，隆多，庫茲馬和戴維斯，而在他們當中，最嚴重的無疑是安東尼-戴維斯。濃眉在中場結束的時候，甚至需要輸液。但他今天仍是帶病砍下了全隊最高的25分，力助球隊拿下勝利。對比之下，掘金陣中的核心約基奇在健康的情況下，反而在與濃眉的對位中迷失了自我，全場僅拿下13分。

其實拋開傷病，湖人今天能獲勝很大程度上也依仗了湖人兩名替補的發揮，其中一位就是老將隆多，隆多今天出戰25分鐘，拿到了11+6+6的全面數據。最重要的是他的三分球兩投兩中。在上半場掘金反撲階段，隆多不僅自己命中三分，在湖人推防守反擊的時候，他總能找到處於投籃最佳位置的隊友。掘金沒能在上半場將比分扳回來，跟隆多將湖人有效串聯起來也有很大關係。值得一提的是，湖人的「感冒大軍」，隆多也是其中一員。

另外一名關鍵替補是卡魯索，雖然「禿曼巴」今天全場僅拿到6分4籃板。但是數據不能完全體現一個人的作用。禿曼巴今天在防守上很好地限制了掘金的巴頓，後者全場11投僅兩中。而且在比賽進行到最後幾分鐘，湖人一度有被逆轉的危險，後面能打出5-0的小高潮，跟卡魯索造了穆雷一個進攻犯規也有關係，比賽最後的時間，沃格爾把卡魯索留在了場上，這本身也是肯定了「禿曼巴」的重要作用。

能夠在「魔鬼賽程」的開端就造訪「魔鬼主場」的情況下,拿下這場來之不易的勝利，對於湖人內部的信心提升是巨大的，相信這場比賽帶來的信心可以幫助湖人在接下來的比賽更加地遊刃有餘，期待他們接下來的表現。
