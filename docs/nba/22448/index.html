恩比德：我在練習低位腳步，過去幾場這方面有所欠缺

2019-12-18 1,191條評論 2,797次閱讀 540人點贊

虎撲12月18日訊 今天76人中鋒喬爾-恩比德在訓練時接受媒體採訪，他談到自己的低位進攻能力。

恩比德說道：「我在練習我的低位腳步，過去幾場我在這方面有所欠缺，現在想要找回來，然後努力變得更好，因為我實際上可以表現好得多。這賽季我幾乎每次都會面筐，這也是因為對方會包夾我，所以每次拿到球，我都會看看防守會做什麼。只要不是雙腳都在禁區，我就會面筐，但最重要的是迅速移動。」

「在低位的時候，最重要的事情就是保持平衡，整個賽季我都覺得我沒做好這點。我需要降低重心，為一切做好準備，否則就容易失誤，而且我能看出自己的重心是高還是低。最重要的就是平衡、重心和決策。」

「對手一整個賽季都在雙人包夾甚至三人包夾我，我需要習慣這一點，季後賽里也會這樣，所以我要繼續為此努力。上賽季他們會放空本（西蒙斯）來包夾我，這賽季很多時候他們也在做一樣的事情，但他們可能會從任何人那裡過來，你也不知道他們到底從哪來，必須做好調整，做出正確決策。」

(編輯：窩瓜)
