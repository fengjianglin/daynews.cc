威登組合68投18中，替補沒有球權合砍21分，火箭隊終結二連勝

2019-12-04 0條評論 18次閱讀 0人點贊

火箭隊終結二連勝

今天NBA常規賽繼續進行，火箭隊客場挑戰同區對手馬刺隊。火箭隊剛剛在主場大勝老鷹隊，球隊核心哈登三節比賽狂砍60分，而馬刺隊最近的狀態不好13場輸了11場。比賽之前很多球迷認為火箭隊會很輕鬆的戰勝馬刺隊，但是比賽的過程肯定會讓火箭球迷很失望，火箭隊在最大領先21分的情況下，被馬刺隊完成了逆轉，特別是第四節火箭隊進攻突然斷電被馬刺隊打了34比18逼進加時，最終火箭隊在客場通過雙加時135比134輸給了馬刺隊終結了二連勝。

威登組合68投18中

火箭隊的打法就是球星單打，威少和哈登都是NBA的巨星，都拿過常規賽MVP。所以火箭隊的球權大多數都是在這兩位手上，俗話說成了威登，敗了威登。這場球就是這樣，威登組合一共投了68球，而火箭隊一共投了112球，威登組合佔了60.7%的出手權。而且只命中了18球，命中率只有26%。哈登這場球出手了38次投籃命中了11球，其中三分球20投只有4中，投籃命中率只有28.9%.，三分球命中率只有20%。哈登今天狂砍了50分，但是罰球24罰24中拿了24分，如此慘淡的投籃命中率還是執著的投籃。另一位MVP威少的投籃命中率更加慘，30投7中，三分球6投1中，投籃命中率只有23%，三分球命中率只有16.6%。威登組合投成這樣，還執著的去投，這樣的火箭隊如何去競爭總冠軍。

替補沒有球權合砍21分

很多球迷都說火箭隊的替補得分不行，但是你看技術統計都可以看出來，火箭隊的替補沒有球權，在球場上就看著威登組合玩，他們不想玩了就接球投一個。今天這場球也一樣，火箭隊出場了5個替補一共才投18次籃，還沒有哈登投3分球出手次數多，特別是里弗斯出場了40分鐘，才8次投籃而且命中率非常高投中了6球，三分球6投5中，今天小里的手感那麼好威登組合都沒有多給他一些出手權。今天火箭隊的替補只有兩人得分里弗斯拿了19分，錢德勒拿了2分。表面上說今天輸球是火箭隊的替補得分不給力，但是最主要的還是威登組登投和太多了，在火箭了作替補太可憐了，有能力也無法發揮出來。
