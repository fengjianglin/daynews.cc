經紀人：貝爾坦斯對奇才很有信心，他現在不會考慮合同

2019-12-19 1,777條評論 2,870次閱讀 831人點贊

虎撲12月18日訊 奇才前鋒戴維斯-貝爾坦斯的經紀人Arturs Kalnitis在近日接受媒體採訪時表示，貝爾坦斯對於自己在球隊中的定位感到非常滿意。

「湯米-謝潑德（奇才總經理）在打造這支球隊方面做了很棒的工作，球員們都感覺備受關注，這是全方位的非常棒的體驗，」Arturs Kalnitis說道，「戴維斯（貝爾坦斯）來到這支球隊的兩周之後，他打電話告訴我，『這裡很棒。』」

「他對於奇才非常有信心，如今球隊的情況真的為他提供了展現自己能力的機會，如果他下賽季留在華盛頓，球隊迎來健康的約翰（沃爾）之後會非常有趣。戴維斯真的相信奇才會成為東部一支勁旅，我非常確信這對於他而言是非常關鍵的考慮因素。」Arturs Kalnitis接著說道。

貝爾坦斯現有的合同將於明年夏天到期，談到屆時貝爾坦斯的去向，Arturs Kalnitis說：「對於他而言，這將是一個有趣的夏天，因為很多球隊都會高度關注他，每支球隊都需要他這樣的球員去衝擊總冠軍。」

「以我對他的了解，他現在完全不會考慮合同的事情，他現在專註於全力以赴為奇才提供幫助，這隻關於贏球。」Arturs Kalnitis說道。

2018年休賽期，貝爾坦斯與馬刺簽下了一份2年1450萬美元的合同，本賽季薪水為700萬美元。今夏在一筆三方交易中，貝爾坦斯被馬刺交易至奇才。

根據此前消息報道，聯盟某高管表示貝爾坦斯明年夏天合同到期後將會得到年薪在1500萬-2000萬美元的合同。
