詹皇的2個幫手，一年內被送至同一地方，蘇群：騎士繼續增援爵士

2019-12-25 336條評論 2,122次閱讀 1074人點贊

雖然2014年詹姆斯回到騎士之後，只在2016年為騎士帶來一個冠軍，但是他所在的球隊，一直保持著爭冠實力，也年年打進總決賽。只是當2018年詹姆斯離開後，騎士的陣容也就坍塌了。去年這個時候，騎士選擇將37歲的科沃爾交易到猶他爵士，從爵士隊換來了阿萊克-博克斯以及2個未來的二輪選秀權。

不過，騎士的重建並未完成，交易也不會停下。最新的消息是，騎士和爵士達成協議，將克拉克森送出，換來艾克薩姆+兩個次輪簽。據騎士記者Chris Fedor報道，多位聯盟消息人士透露，兩支球隊在上周就開始商討這筆交易，而騎士得到的是2022年（來自馬刺）和2023年（來自勇士）的次輪簽。

這筆交易發生後，國內籃球名嘴蘇群老師也第一時間給出了自己的評論：「騎士隊繼續增援爵士，把克拉克森送到了鹽湖城，所得只是埃克薩姆和2個第二輪選秀權。上次騎士已經增援一個科沃爾。」只可惜，上次增援的科沃爾沒有派上什麼用場，今夏也再次離開。好在克拉克森還年輕，目前才27歲，正是一個球員的成熟期，本賽季克拉克森場均14.6分2.4籃板2.4助攻，有望成米切爾身邊可靠榜首。

今夏爵士花了很大努力得到康利，但康利新賽季總體表現掙扎，場均僅13.6分3.3籃板4.6助攻，與上賽季相比可謂一落千丈，何況康利的年薪高達3200多萬。克拉克森的到來後助於提升後場實力。另一方面騎士送走克拉克森，恐怕主要是考慮到他的合同（1344萬）將在明夏到期，現在交易還能撈回一點爾東西。總體上，騎士一年內2次增援爵士，主要回報是3個次輪秀，後詹姆斯時代重建難啊。
