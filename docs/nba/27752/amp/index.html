12中7高效砍下21分 33歲老將打出生涯第二春 雄鹿想奪冠少不了他

天天要聞
2019-12-20

北京時間12月20日，NBA常規賽一場焦點大戰，東西部榜首的湖人與雄鹿狹路相逢，本場之前雙方戰績均為24勝4負，本場將直接決定誰是當下聯盟的頭名。最終經過四節比賽的較量，雄鹿主場111-104力克湖人，拿下了這場焦點對決的勝利。

本場比賽，字母哥和詹姆斯兩位新老球星的對決無疑是最大的看點，而本場雙方都有出色的表現。字母哥全場19投11中得到了34分11籃板7助攻，而詹姆斯稍遜一籌，19投8中，得到了21分12籃板11助攻。更重要的是，字母哥的雄鹿在這場對決中笑到了最後。

本賽季字母哥繼續著自己強勢的表現，場均31.7分12.8籃板5.3助攻，是MVP的最有力爭奪者。但本場比賽的勝利，除了字母哥之外，雄鹿需要感謝老將喬治-希爾的出色發揮，全場他替補出戰，12投7中，三分球5中3，罰球4中4，高效的拿下了21分3籃板1助攻。

上賽季中期，火線加盟球隊的喬治-希爾就已經有不俗的表現了，尤其是在季後賽中，場均11.5分3.5籃板2.8助攻，在對陣凱爾特人和猛龍系列賽中都有驚艷的發揮。這也是今夏為何球隊願意以3年2900萬合同留下他的重要原因。

而本賽季，33歲的他在雄鹿打出了自己生涯的第二春，場均10.0分3.1籃板3.0助攻，投籃命中率和三分球命中率分別達到了53.4%和50.6%。今夏在布羅格登離開之後，雄鹿的後場多少顯得有些單薄，但希爾的強勢表現很好的彌補了這一不足，本賽季雄鹿想要衝擊總冠軍，少不了他出色的發揮。
