倫納德：我年收入2870萬，杜蘭特：我6540萬，那詹皇和庫里呢？

2019-12-09 0條評論 20次閱讀 0人點贊

職業體育運動員一直以來都是一個高薪行業，但這也是符合現實情況的，作為職業運動員，他們的職業生涯往往都難以持續太久，而且在此期間，還要飽受到傷病的困擾，有很多實力出眾的職業運動員都是因為傷病無奈結束了職業生涯。而這些運動員再結束職業生涯後，除了教練員等工作的話，往往都是很難再找到一份新工作的。因此，他們拿到高薪也是理所當然的。

而像籃球，足球這樣影響力巨大的運動，他們中頂尖的職業球員一般都會擁有巨大的影響力，而他們薪資和收入也是要遠遠超過行業平均水平的。就拿NBA聯盟球員來說，現役第一人詹姆斯一直以來都保持著出色都競技狀態，而他在全球範圍內的影響力也是無與倫比的，據福布斯排行榜中顯示，詹姆斯目前的年收入高達8900萬美元，在世界體壇中排名第八。也是NBA球員中賺的最多的。

據統計，詹姆斯目前的年薪為3600萬美元，而他還有很多的廣告和代言合同，此外，詹姆斯還做過許多的投資，投資領域涉及體育，地產，餐飲等方面，而這8900萬美元的年收入都只是保守估計了。詹姆斯之前入股足球俱樂部利物浦後，僅僅數年過後，他在利物浦的股份就已經翻了數十倍了。那麼，下面我們再來看看聯盟中其他幾位巨星的年收入。

庫里作為目前聯盟中的第一控衛，他在近幾年的表現也是堪稱完美的，而在福布斯排行中，庫里的年收入排在第九位，僅次於詹姆斯，他目前在勇士隊的年薪高達3800萬美元，此外，他還有許多的代言合同，年收入共計為7980萬美元。

而本賽季轉會籃網的杜蘭特他的年收入排在第十，為6540萬美元。杜蘭特儘管如今因傷或將缺席本賽季的大部分比賽，但他在此前的職業生涯中擁有著極為出色的表現，他此前還曾與耐克簽下了一份十年總價值3億美元的合約。除了他們幾人以外，像哈登，保羅，歐文，字母哥，威少等人也是都排名福布斯收入榜中的前列，當然，還有目前聯盟中炙手可熱的倫納德，他在上賽季未結束之前的年收入就已經接近了3000萬美元，如今加盟快船的他預計年收入也將提升到5000萬美元以上了。
