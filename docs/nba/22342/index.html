失去“077”不敵“078”，獨行俠遭逆轉憾負凱爾特人

2019-12-19 2,320條評論 4,357次閱讀 2138人點贊

北京時間12月19日，獨行俠主場迎戰凱爾特人。憑藉凱爾特人「078」組合合砍82分，他們以109比103獲得比賽勝利。

沃克拿到全場最高的32分，還貢獻了5籃板3助攻，布朗拿到26分9籃板，塔圖姆拿到24分8籃板。獨行俠這邊波爾津吉斯拿到23分13籃板，塞斯·庫里拿到20分4籃板。兩隊都要重要球員缺陣，除了之前因傷休戰的東契奇和斯瑪特外，海沃德因左腳酸痛也缺席了比賽。

開場兩隊都嘗試了大量的三分，但是凱爾特人沒有找到手感，被對手打了一波14-5，暫停回來凱爾特人加強了沃克的進攻，後者連得10分，率領球隊打出一波12-2反超了比分。沃克第一節也獨得16分。第一節結束時，憑藉塔圖姆的壓哨中投，兩隊以29比29打平。

第二節比賽塞斯·庫里猶如哥哥附身，外投內突連連得分，獨行俠也增強了對沃克的防守，包夾迫使其出球，第二節還剩4分鐘左右，沃克在被包夾的情況下傳球失誤，獨行俠通過反擊造成對手犯規，取得了10分的領先優勢。第二節結束，獨行俠以55比50領先凱爾特人5分，在獨行俠針對性的防守策略下，沃克第二節一分未得。

第三節凱爾特人加強了防守強度，特別是針對庫里這一點，讓獨行俠第三節全隊只得到18分。第三節末格蘭特·威廉姆斯命中三分，幫助凱爾特人反超1分，三節比賽結束，凱爾特人以76比73領先3分。

比賽進入到第四節，獨行俠東契奇缺陣的影響暴露的非常明顯，由於沒有一個很好的進攻發起點，獨行俠隊的進攻沒有什麼章法，第四節後半段連續出現兩次失誤。沃克抓住機會連造兩次犯規，幫助凱爾特人以95比85領先10分。雖然塞斯·庫里連續命中兩記「神仙三分」，但沃克也連中兩球給予回應。就在比賽懸念的火苗即將熄滅之時，獨行俠迫使凱爾特人出現底線球發球失誤，一回合連得5分將分差縮小到5分。但凱爾特人暫停後穩住了局面，最終有驚無險的拿到了比賽勝利。
