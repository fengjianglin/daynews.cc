比詹姆斯受傷更嚴重的，是湖人繞不開這三道難題

2019-12-23 2,908條評論 4,944次閱讀 196人點贊

吹誰誰黑，這個定律似乎顛撲不破。

前些日子，湖人隊順風順水時，他們儼然已是總冠軍的頭號熱門候選，又是防守固若金湯，又是陣容搭配猶有後招，那段時間的七連勝簡直就要，醉人。

然後，連敗來了。

然後，詹姆斯受傷要休戰了。

最新的消息是，北京時間周一主場打掘金，詹姆斯基本確定不打，原因是胸肌拉傷。據說，是打步行者時受傷的，上一場打雄鹿，詹姆斯已經是帶傷作戰。

所以，就看之前剛剛因為腳踝不適休息過的濃眉哥了，就看據說有望復出的庫茲馬了。

空氣中，是不是已經有了三連敗的氣息？

這句話可能有些絕對，但你可能也找不出反駁的理由：沒有詹姆斯，湖人就不是湖人。

濃眉哥當然是基石，但他並不能夠帶動整支球隊戰術體系，至少是現在。當下的湖人隊是「兩巨頭」驅動，其實跟當年的「OK時代」有得一比，但詹姆斯的作用更接近當年奧尼爾，濃眉則是一天天成長的科比。

對於濃眉來說，這當然是考驗亦是機遇。但對於湖人來說，這隻能是一個漫長賽季里、恐怕最棘手考驗的開始。

好不容易，主帥沃格爾在一個多月時間裡，把看上去東拼西湊的湖人隊捏合出了出人意料的效果，即便不完美，但絕對夠力量。可如果，完善的事情還沒來得及做，傷筋動骨的事情又來了，你說這該是多大的麻煩？

現在的湖人，確實有必要好好想想，接下去要去克服的困難有多少。

第一，當然是傷病。

詹姆斯並不是傷病號，在本賽季湖人打完的29場里，他保持了全勤。與他同樣創造這一紀錄的，還有丹尼·格林、波普、麥基和霍華德。然而，其他人的數據就沒那麼好看了—

濃眉哥缺席兩場，這於他而言已經是不錯的出勤率；庫茲馬缺席9場，本來是指望他成為球隊第三極的；隆多缺席12場，所以有段時間詹姆斯的負擔特別重；布拉德利缺席14場，單防對方鋒線殺器的任務只能交給格林一人。對了，還有考辛斯，一場沒打，而且誰也說不清他本賽季能不能打……

對於一支今年夏天開始重新捏合的球隊來說，傷病無疑是最討厭的東西，可是以湖人球員的年齡結構和傷病史論，不斷有人缺勤是一件在所難免的事情，要知道上賽季隆多就只打了45場，布拉德利也只打了63場。

最重要的是，馬上35歲的詹姆斯，身體耐受度肯定不會像25歲那樣了。所以，即便接下去的這場球，詹姆斯只是想讓自己調整一下，湖人隊恐怕也要想清楚了，剩餘的50多場常規賽，是不是非要每場必拼？

第二，是被念叨過一萬遍的鋒線補強。

這個問題，從一開始就並非一個答案，因為有人說湖人的鋒線還缺一個能夠串聯、又能夠防守的球員，又有人說湖人的鋒線卻足夠犀利的投手，聽上去，有點兒南轅北轍。

潛台詞卻是，湖人缺的不止一個。

前面那種說法，目標直指伊戈達拉，他如果能加盟，絕對是攻、防兩端都能助詹姆斯一臂之力，尤其是進攻方面，湖人現有的幾位格林、布拉德利、波普乃至杜德利，基本都是自己跑位、等人喂球的打法，太容易被對手掐住脈搏。

後一種，則是賽季初從雷迪克開始，就沒中斷過這樣那樣的流言。截至目前，湖人場均進10.9個三分球，排名聯盟第22位，顯然不是強隊水準。

再看隊內，雖然有多達7名球隊的三分球命中率超過35%，可即便是詹姆斯之外場均投進三分球最多的丹尼·格林，本賽季三分命中率達到不算差的38.4%，他也難以避免深一腳淺一腳，過去5場比賽里有兩場竟然是三分球掛零。

也因此，其他球隊中有不少三分投手，都被熱心人跟湖人扯上關係，最近幾天連國王隊夏天砸下大合同的博格達諾維奇，都成了「考慮對象」。說實話，病急亂投醫，這絕對不是強者的風範。

第三，還是要說說，球隊的風格氣質。

有詹姆斯，有一眾老將，湖人在經驗和防守上篤定不會吃虧，打陣地戰也不會吃虧，可這樣的··陣容配置和球風照樣還有缺陷—厚重有餘，靈動不足。

當今聯盟的潮流是強調攻守轉換，強調快速攻擊，湖人遇上實力尋常的球隊能夠以慢制快，遇上強敵就可能壓制不住對手的節奏。

賽季之初輸給猛龍，庫茲馬就明言「我們現在的致命缺點就在於攻守轉換」，過去兩場輸給步行者和雄鹿，也還是吃虧在這方面。對雄鹿一戰，湖人被搶斷14次，還被對手投進16個三分。

快與慢，重與輕，優劣從來都沒有定然，可如果湖人解決不好傷病問題，或者解決不好外線火力的穩定性問題，所謂節奏就可能落入被動。

是不是，好歹準備一個應急方案，讓濃眉打中鋒、庫茲馬打四號位、詹姆斯打三號位？

總覺得，這支湖人隊的潛力，還可以再挖一些，再挖一些。
