什麼叫同天賦不同命？威少大傷升級，羅斯則隕落，對比令人唏噓

2019-12-19 1,660條評論 3,902次閱讀 2097人點贊

對於NBA這個全世界職業籃球的聖地來說，最不缺少的就是天賦出眾的球員，很多在普通球迷眼中天才球員般的存在，其實在NBA中屬於大眾貨色，不過在NBA中，即使天賦相同，在聯盟中能夠達到的上限，在聯盟中命運也有很大的差別，下面小編就給大家帶來，什麼叫NBA中的同樣的天賦不同的命。

羅斯和威少

羅斯和威少，同樣是08年選秀大會中的高順位新秀，同樣的天賦異稟，同樣的運動能力頂級，同樣的衝擊力驚人，同樣一進進球隊就被當作核心培養，同樣有著不善於外線投射的缺陷，也同樣都遭受過足以影響職業生涯的傷病……，但是這兩位相同之處甚多的天才後衛，受傷後的命運卻有著巨大的差距，大傷後的羅斯，運動能力和實力出現了比較嚴重下滑，而且用了幾年的時間才徹底走出了大傷之後的影響，但是卻再也不是那個歷史上最年輕的常規賽MVP了，而威少在大傷之後，不僅沒有受到什麼影響，反而就如同返廠升級了一般，從全明星球員一躍成為聯盟巨星，而他的大部分榮譽，也全部都是大傷之後拿到的，不得不承認，兩人真是同樣的天賦不同的命。

格里芬和斯威夫特

斯威夫特和格里芬，一個是00年選秀的榜眼秀，另一個是08年選秀的狀元秀，這兩名同樣司職大前鋒的球員，進入聯盟之處都憑藉了頂級的天賦和運動能力，以及勁爆的扣籃被拿做和當年的NBA巨星坎普作為比較，只不過這兩位球員的命運卻大相徑庭，雖然職業生涯中格里芬遭受了不少的傷病困擾，轉型後的打法也經常被人詬病，但不可否認的是，他依然是現役排名前五的大前鋒，全明星級別的球員，而斯威夫特卻成為聯盟歷史上最著名的水貨榜眼，缺乏的球商讓他空有頂級的天賦和運動能力卻發揮不出來，不但早早的離開了聯盟，甚至在CBA聯賽中，都很難成為頂級的存在。

拉里約翰遜和巴克利

拉里約翰遜進入聯盟時，就被認為是巴克利的接班人，兩人的天賦和特點也相差無幾，同樣以不足2米的身高司職大前鋒，同樣擁有著強壯的身體和出色的力量，同樣進攻手段能里能外，同樣擁有著頂級的運動能力，同樣是同位置中的佼佼者…..，只不過相較於職業生涯巔峰時期一直沒有遭遇到巴克利，拉里約翰遜卻在巔峰時期遭遇到了極為嚴重的傷病，這不僅讓約翰遜沒有達到巴克利的高度，更讓他早早的就因為傷病選擇了退役。

吉米巴特勒和拉扎爾海沃德

拉扎爾海沃德可能很多球員都沒有聽過這個名字，他是一位具有不錯體型和強壯程度的小前鋒，打法十分強硬和具有侵略性，只不過運動能力和和天賦方面卻不算出色，而選秀時期的巴特勒也同樣如此，並且兩人的整體天賦十分接近，但是雖然海沃德也是一名十分勤奮和努力的球員，但卻沒有巴特勒那種級別的上進心和動力，如果說海沃德是將自身的天賦發揮到了80%左右，那巴特勒就是發揮到了絕對的極限，這也是為什麼天賦相差無幾，海沃德只是一個角色球員，而巴特勒卻成為了聯盟巨星。

保羅加索爾和米利西奇

「年少成名，天賦頂級，擁有著7尺的內線球員標準身高，不次於外線球員的靈活性和移動能力，出色的投籃手感和籃下技巧，能夠拼搶籃板和完成補防，但是對抗能力較弱，力量和單防能力有待提升」，這段話，除了可以用來形容初入聯盟時期的加索爾外，還能用來形容NBA中最為出名的水貨榜眼米利西奇，同樣作為高順位新秀，天賦方面也相差無幾的情況下，加索爾一進進球隊就被當成未來的核心培養，特別是沒有戰績壓力的情況下，可以讓加索爾在場上任意的發揮，在比賽中去快速的成長，而米利西奇卻在自己的菜鳥賽季，遇到了NBA歷史上最為古板，也是最不喜歡使用新秀球員的教練拉裡布朗，再加上球隊志在奪冠，這讓新秀時期的米利西奇根本就沒有得到頂級新秀應有的培養資源，反而是出場時間寥寥無幾，白白浪費了職業生涯初期最寶貴的那幾年時光。
