15戰砍619分 場均41+ 得分能和張伯倫比較 哈登太狠

2019-12-25 816條評論 3,275次閱讀 121人點贊

字母哥連續佔據官方MVP榜第一位，哈登儘管進入了前5，可是哈登頂多排到第3或第4，已算不上MVP陪跑者了。毋庸置疑，上賽季哈登場均36.1分都未能當選MVP，本賽季繼續在場均得分上拉開得分王位置，哈登還是非常難當選MVP，只是承認哈登很強就那麼難嗎？按照哈登如今在得分端的表現，哈登排在第一位，不應該具有爭議。

從哈登12月份的數據表單，可以看得見哈登在場上屬於多功能型，儘管得分能力讓哈登得到很大的話題性，不過哈登在籃板、助攻與搶斷的數據上，同樣是外線球員的佼佼者之一。

另外，根據數據統計顯示：哈登本賽季前15場客場比賽，15戰砍下610分，場均41.3分，實現了賽季前15場客場比賽場均得分40+的神跡，要知道上一個達到此得分數據的球員，還是上古神獸張伯倫。

得分上能把張伯倫逼出來比較，哈登太狠了！自從張伯倫以後的得分手，能夠如此頻繁與張伯倫進行比較，強如科比都沒有做到，除了單場81分，哈登才是NBA得分數據破壞者。本賽季，哈登已不只一次逼得張伯倫的得分數據，被拿來做對比，可見哈登確實是當代最強得分手。

值得一提，哈登高爆炸數據的背後，還隱藏著一些驚人操作。

1、別說哈登出手太多，要是不具備強悍的體能，投個25次以上，可能已經累到不行，命中率更沒有保障。2、比起得高分空砍的得分手，哈登單場砍下40+，勝率80%歷史第三。3、哈登高出手不破壞球隊化學反應，不管哈登怎麼打，真是命中率都保持60%以上，這是一個非常誇張的數據。

哈登可以在高得分的同時，還能帶領球隊贏得比賽，躋身於西部豪強球隊，這可是德文-布克這位70分先生所不能具備的。在得分能力上，哈登太狠，或許以後的NBA最強得分手稱號，真可能只屬於哈登吧。
