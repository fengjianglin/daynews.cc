6人能防守奧尼爾，小將班巴肌肉強壯，霍華德：6人加起來都扛不住

天天要聞
2019-12-21

現役的很多的內線球員，都沒有跟奧尼爾和姚明等人一起打過比賽，當時奧尼爾和姚明是作為內線最強的球員代表，他們之間的對抗，比現在的三分球對飆可是好看多了。就拿奧尼爾來說，當時的內線球員當中，沒有幾個人能扛住奧尼爾。姚明是其中一個，那麼要是換成現役的球員當中，有哪幾個內線球員能扛得住奧尼爾，下面我列舉了6個能扛得住奧尼爾的內線球員。

6：勒布朗詹姆斯

首先，扛住奧尼爾這樣的內線球員，最重要的就是有力量，而說到力量這個詞語，在現役的球員當中，也就是勒布朗詹姆斯的力量是比較強的。勒布朗詹姆斯雖然不是跟奧尼爾一個位置的球員，奧尼爾是中鋒球員，而勒布朗詹姆斯卻是一個小前鋒。但是，詹姆斯這種球員，是五個位置都可以防守的，就算是奧尼爾這樣級別的內線，勒布朗詹姆斯只要是使盡全力，還是有很大的機會防守住奧尼爾的。

5：小將班巴

很多人可能說，現役的內線球員當中，沒有幾個能看得上眼的，跟奧尼爾那個年代的內線球員差距太大了。這個是有一點差距，不過，年輕的球員當中，還是有很多優秀球員的，跟奧尼爾一樣，也是從魔術隊選秀道路來到NBA的班巴，是一個優秀的內線球員，而且他的肌肉也很強壯，甚至比加內特的肌肉還要強壯，要是在多加鍛煉，班巴把自己的肌肉量在增加一點，也是可以扛住奧尼爾的。畢竟，班巴跟勒布朗詹姆斯相比，他的身高更高，體重也更重。

4：大白邊

要說現在的內線球員，在身體素質和體重方面，能跟奧尼爾差不多的球員真的不是很多。而大白邊這種類型的球員，已經算是聯盟裡面身材比較高大而且是強壯的。再來就是大白邊本來就是一個以防守能力比較強的球員，要是遇到靈活的內線，大白邊可能還不知所措，但是，遇到奧尼爾這樣的內線球員，只是要硬扛住奧尼爾，大白邊還是有很強的力量去跟他對抗的，但是，扛得住不一定是防住，大白邊防不住奧尼爾的，奧尼爾的腳步更加的靈活。

3：唐斯

唐斯也是現役的內線球員當中，年輕一代的佼佼者，從剛開始的選秀狀元，到現在的球隊核心球員，森林裡球員唐斯一路走過來，都是很努力的。尤其是唐斯這幾年的肌肉變化，是有明顯的效果。肌肉增強了不少，而擅長進攻的唐斯，雖然防守端不是最強的。但是就憑著這一身的腱子肉，唐斯要是真的用力防守奧尼爾，也是有很大的機會扛住奧尼爾。

2：戈貝爾

在現在的內線球員當中，進攻最好的一個球員應該是76人隊的恩比德，因為他有很好的內線進攻手段，還有外線的投籃手感。但是，要是說到防守最好的一個球員，那肯定是拿到了最佳防守獎盃的戈貝爾了，作為一個法國的中鋒球員，戈貝爾的職業生涯可以算是成功的，戈貝爾的身體素質也很強，而且還很高。要是扛住奧尼爾，對於現役最強的防守球員來說，應該不是什麼難事。

1：霍華德

當然，上面這些球員，都沒有正面的跟奧尼爾交鋒，除了勒布朗詹姆斯，在自己的職業生涯，勒布朗詹姆斯也是很少跟奧尼爾直接對位的。而現在在湖人隊的霍華德，當年可是跟奧尼爾有很多直接對位的機會。他知道防守奧尼爾有多難，要是叫霍華德去扛住奧尼爾，估計他會說，就算是上面的6個人加起來，也是扛不住奧尼爾那頭大鯊魚的。因為，奧尼爾巔峰的時候，霍華德都只能算是小鯊魚一頭。真的能防守奧尼爾的球員，在現役的球員當中，或許只有霍華德一個人有可能，上面的幾個球員要是防守職業生涯末期的奧尼爾還是有機會的，但是，要是換成是職業生涯巔峰的奧尼爾，真的不是這麼容易就能扛得住的。沒有絕對的力量，會被撞開三米開外。
