盤點本賽季那些淡出觀眾視野的球星們

2019-12-18 1,150條評論 3,467次閱讀 1365人點贊

球星總是人們關注的焦點，大多數球迷看球就是看球星。在NBA，有的球星常年保持良好的狀態，年年進全明星，在球迷心中存在感極強，比如詹姆斯、哈登、倫納德等，也有的球星曇花一現，由於傷病、競技狀態等各種原因逐漸淡出觀眾視野，特別是如今的聯盟競爭激烈，人才輩出。本賽季東契奇、莫蘭特、特雷楊、西亞卡姆等年輕球員大放異彩，在本賽季得分前十的球員中，90後有七人，由此可見，新一代的年輕人逐漸要統領這個聯盟了。這批年輕有為的球員崛起，就意味著另一批球員的淡出，接下來就盤點本賽季漸漸「失寵」的球星。

1、沃爾

作為2010屆的狀元郎，沃爾也曾打出過高光的表現，在他的巔峰期是不折不扣的全明星球員，奈何他的職業生涯飽受傷病折磨。上賽季開賽不久接受了左腳後跟手術，預計缺席6-8個月，後來在家中不慎滑倒，導致左跟腱撕裂，又將缺席一年，預計本賽季末復出。頻繁的傷病嚴重影響了沃爾的職業生涯，這次有遭遇跟腱撕裂這種大傷，他的未來打上了一個大大的問號。沃爾在2017年夏天與奇才簽訂了一份4年1.7億的合同，這也導致了他受傷期間球隊難以引進強援，只能靠比爾單核帶隊，奇才這兩賽季戰績平平，本賽季也恐無緣季後賽。傷缺時間太久，球隊戰績慘淡，使沃爾的關注度大減，比爾在他受傷期間表現搶眼，早已坐穩了奇才當家球星的位置，等他復出之後要想重回巔峰恐怕難上加難了。

2、考辛斯

同樣是傷病困擾，考辛斯本賽季已經幾乎沒有關注度了，上賽季加盟勇士組成五星戰隊也未能奪冠，本想來湖人重返巔峰，奈何休賽期再發生意外。他很可能已經賽季報銷了，即使本賽季復出，也就是想上賽季那樣了，狀態平平，不能給球隊帶來很多幫助。更何況這賽季的湖人兵強馬壯，一路高歌猛進，內線麥基穩坐首發，魔獸完成自我救贖，表現搶眼。考神即使復出也不一定能進入常規輪換了，曾經的第一中鋒如今也逐漸被觀眾淡忘，實在令人心痛啊。

3、追夢格林

為啥會是追夢格林呢？原因很簡單，在水花兄弟傷缺的情況下，格林毫無疑問是勇士的絕對核心領袖，他帶隊打出聯盟倒數第一的戰績。他曾經也是全明星球員，有人還說他是聯盟前十球員，現如今水花兄弟不在的時候場均9分6板，這就是一個普通替補球員的數據，球隊最需要他的時候他沒法輸出。現如今，很少有人看勇士的球了，要扭轉局勢只有等到庫里或者克萊回來了。由此可以看出，格林更加依賴水花兄弟，缺少了水花兄弟，他太難了。

4、朗佐-鮑爾

球哥以榜眼得身份被湖人選中，並被寄予厚望，加上有一個能說會道並且能「打爆」喬丹的老爹，他在湖人期間備受矚目。如今被交易來到鵜鶘，很多人認為球哥可以放手大幹了，賽季初他也打出過幾場亮眼的比賽，但是三分之一的賽程已過，他相比前兩個賽季還是原地踏步，場均10分、4籃板、5助攻。和他一起交易來的英格拉姆場均25分7籃板，儼然成為球隊未來，觀眾們似乎對他失去了耐心，沒人在討論他的表現了。還有一個重要原因就是球哥和球爹鬧翻了，球爹不再為他打廣告了……
