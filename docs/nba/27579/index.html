美媒全明星預測出爐，字母東契奇票王，約老師落選引爭議

2019-12-22 2,598條評論 3,584次閱讀 2679人點贊

不知不覺，賽季已經進行了一小半，聖誕大戰很快就將開始，全明星投票也即將開始。雖然全明星賽越來越沒有看頭，但是全明星資格作為一個球星榮譽，也得到了很多球迷和媒體的關注。

這個賽季，以老詹為首的老牌球星繼續發揮穩定，東契奇等新人也有不錯表現，那麼這個賽季東西部哪些人將會入選全明星，哪些人又將被選為全明星首發呢？近日，美國媒體FW進行了全明星預測，一起看看是否和你想像的一樣。

西部首發：東契奇、哈登、倫納德、詹姆斯、戴維斯

先說東契奇，毫無疑問，東契奇這個賽季的表現完全配得上全明星首發，甚至他可以成為西部的票王。第一個賽季，東契奇打出了准明星級的表現，但是獨行俠的戰績並不理想，而這個賽季的東契奇不僅打出了現象級的表現，而且獨行俠也高居西部前四。

35歲的詹姆斯依舊沒有下滑的意思，在過去的29場比賽里，詹姆斯場均拿下25.8分，7.5籃板和10.6助攻，如果不出意外，詹姆斯將拿下自己職業生涯首個助攻王，而湖人這個賽季的表現也讓湖人球迷對這支球隊有了更多盼頭。

濃眉哥則是詹姆斯最好的幫手，甚至在某些方面濃眉的價值比詹姆斯更大。濃眉哥不僅能依靠自己的個人進攻天賦極大的幫助詹姆斯解決進攻端的問題，而且他在防守端的價值更加無法估量，本賽季的濃眉甚至有競爭最佳防守球員的能力，全明星首發幾乎板上釘釘。

倫納德可能是這一組唯一一個有懸念的首發，不是因為倫納德不夠優秀，而是西部優秀的球員太多，比如他的隊友喬治。而且，在經歷過夏天的自由市場決定後，倫納德的受歡迎度也受到了一定的影響，不過瘦死的駱駝比馬大，倫納德依舊有實力進入全明星首發。

替補：利拉德、布克、威少、喬治、維金斯、唐斯、戈貝爾

這些人里利拉德和喬治有殺入首發的可能，布克，維金斯和唐斯在這個賽季進步比較大，入選全明星也問題不大，戈貝爾作為爵士的內線防守核心並且也曾入選過全明星，也有可能被選入全明星。值得一提的是約基奇並不在美國媒體的預測範圍內，不知道他們是出於何種考慮，至少在我看來狀態逐漸回暖的約老師是很可能爭取到一個全明星名額的。

東部首發：沃克、巴特勒、字母哥、西亞卡姆、恩比德

由於歐文和杜蘭特的傷病，東部的全明星競爭相對要輕鬆一些。沃克是綠軍這個夏天最大的收穫，他場均拿下23.5分，4.3籃板和5.2助攻的全面數據，在東部後衛中已經屬於金字塔頂尖的球員了。

巴特勒與沃克類似，更換了球隊雖然沒有打出非常豪華的數據，但是他們卻給球隊帶來了巨大的成功，20.9分，6.5籃板，6.8助攻，2.1搶斷，這樣數據你還能對巴特勒要求更多嗎？

字母哥的首發位置也板上釘釘了，在雄鹿戰勝湖人後，字母哥更像是加冕為王的勝利者。這個希臘怪物這個賽季不僅身體素質繼續爆炸，而且三分球也不再是球隊的累贅，字母哥顯然正在成為一個歷史上絕無僅有的球員。

自從倫納德離開猛龍後，西亞卡姆扛起多倫多猛龍，尤其是在洛瑞受傷期間，西亞卡姆更是球隊的絕對巨星。上賽季已經當選最快進步球員的西卡，這個賽季沒有停下腳步，場均25.1分，8.0籃板和3.6助攻，西卡已經慢慢學會用巨星的方式帶領球隊前進。

誠然，恩比德這個賽季的表現並不盡如人意，但是這並不代表他不是一個頂級中鋒。他依舊是76人的防守支柱，同時也是76人的第一得分點。而且，隨著恩比德逐漸找回狀態，東部很難有內線球員成為他的威脅。
替補：特雷·楊、本·西蒙斯、羅斯、比爾、塔圖姆、拉文、德拉蒙德

​特雷楊和比爾所在的球隊都有共同的問題，那就是球隊戰績太差，但是他們個人數據都非常華麗，入選替補全明星應該問題不大。西蒙斯和塔圖姆都屬於球隊的二當家，也是非常有潛力的年輕球員，球隊戰績出色，個人數據穩定，他們入選也無可厚非。羅斯雖然是替補，但是場均16.6分6.0助攻的成績加上球迷的呼聲，也可能讓羅斯再次入選全明星。拉文本賽季打出了很多次出色的表現，德拉蒙德場均搶下驚人的16.7個籃板，這些都可能成為他們入選全明星的資本。

以上就是所有全明星的預測，各位球迷朋友們覺得哪些已經十拿九穩，哪些又存在爭議呢？
