太臟！哈登隊友惡意“砍”特雷楊脖子 被吹惡意犯規還斥責其假摔

2019-12-02 0條評論 15次閱讀 0人點贊

犯規是有限度的，如果是故意犯規，而且還是惡意犯規，自然逃不開公眾的法眼。日前，哈登所在的404隊與亞特蘭大老鷹進行了一場常規賽，兩個隊之間還是有很大的差距的。當第二節比賽開始了四分鐘，老鷹才在整個比賽中搶到第二個籃板，除了404隊百發百中的原因外，老鷹隊在籃板方面也很鬆懈。

兩個隊之間的差距迅速擴大到25分以上，404隊也換上了替補，哈爾滕斯泰因終於有機會表現了，得益於卡佩拉缺席，哈滕終於進入了輪換陣容。如果是論積極性的這個方面，哈滕毫無疑問沒有問題，這也是他能夠擠掉周琦的原因之一。當然，哈滕在球場上的骯髒行為，真的讓人討厭！

在老鷹的轉換進攻中，特雷-楊一路帶球狂奔。此時哈騰出現在中場的圓圈附近，看到他追不上特雷-楊。他就張開左臂，並且明顯地向上抬起了，哈滕的手臂徑直「砍」在了特雷-楊的脖子上。

特雷-楊隨後痛苦倒地，並且氣得狠狠將球砸到地上。哈滕則面帶無辜伸出雙手，示意裁判沒有違規，然後對著特雷-楊，斥責他在假摔演戲。

當然，裁判也立即觀看了錄像，最後做出惡意犯規的決定，特雷-楊獲得2罰1擲，你們又是怎麼看待404隊的球員如此髒的表現的呢？
