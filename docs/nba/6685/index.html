想參照歐洲足球模式？哈登聯手塔克給蕭華澆一盆冷水

2019-11-27 0條評論 17次閱讀 0人點贊

據之前消息，NBA總裁蕭華可能醞釀對現有的賽程進行改革。

NBA記者沃基和Zach Lowe隨後報道這一議題說，NBA和國家籃球運動員協會（NBPA）在感恩節後的比賽窗口找到了共同點，這個窗口將延長到12月中旬，淘汰賽之前的比賽也將計入每支球隊的常規賽日程。

不過，球員們顯然對這一計劃不感興趣，他們似乎並不支持NBA最近提出的在2021-22賽季為所有30支球隊及時增加一個年中錦標賽的提議。比如詹姆斯哈登和PJ塔克。

今天哈登和塔克在豐田中心的訓練之前都向記者們談到了這個問題。

哈登和塔克在周二豐田中心的訓練之前都向記者們談到了這個問題，Fox26的馬克伯曼報道了兩人的採訪。

塔克說表示，我們在為NBA總冠軍而戰，我不想為別的事而比賽。還有什麼？沒有別的了。（這）像是安慰或什麼的？我不知道，我們打這些比賽是為了贏得NBA總冠軍。

而被問及對NBA在賽季錦標賽中被淘汰的談話/討論的看法時，哈登起初說他對此一無所知，然後補充表示，我們在上大學嗎？這所大學？

不過，NBA總裁（Adam Silver）經常提出在未來舉辦季內錦標賽的想法，將其與歐洲足球模式進行比較。然而，球員們對此反應冷淡，但聯盟相信仍然有辦法讓球員相信這個想法的優點。

NBA這個想法，參照歐洲足球模式的想法最終能夠落地成型嗎？目前來看似乎不太可能，不知大家怎麼看！

此外，在對陣邁阿密熱火的比賽中，內線大將卡佩拉可能會因病缺席，他目前被列為可疑人員，如果屆時餅皇真的缺戰，這將大大不利於馬賽克結束連敗，兩支球隊首次交鋒，熱火大勝29分。
