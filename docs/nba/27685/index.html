終於等到湖鹿大戰，可惜老詹輸了，雄鹿三分直接教做人

2019-12-21 567條評論 1,553次閱讀 274人點贊

北京時間12月20日，湖人在客場挑戰雄鹿，最終以104-111輸掉了此次比賽。目前這兩支隊伍的戰績都排在聯盟頂端，賽前就又很多球迷十分期待這場提前上演的決賽，可以說誰能贏下這場比賽，誰就能夠穩住在聯盟中的霸主地位。此前兩隻隊伍都有輸球，這場比賽再輸掉就會迎來2連敗，可惜，最終還是老詹輸了！

比賽開始之後兩隊之間並沒有爆發出激烈的進攻，反而是從一開始雄鹿就佔據上風。儘管濃眉哥在這場比賽中復出，而且砍下36分10籃板5助攻，老詹這場比賽也拿到了21分12籃板11助攻的三雙數據，可惜還是沒能幫助湖人取得勝利。

這場比賽輸掉之後，也讓湖人的缺點更加明顯了。其實從此前的湖人的表現來看，新賽季他們的三分能力並不是很高。面對同樣強大的對手，比如此次比賽中的雄鹿，三分球往往就是最重要的追分利器。此次比賽湖人全場只有丹尼格林命中7個三分，其餘球員加起來只有23投5中，命中率甚至不到25%。而反觀雄鹿，全場三分命中率超過了40%，直接教對手做人！甚至連三分一直都不夠出色的字母哥，在本次比賽中也有8投5中，可見湖人正是輸在三分球上。

除此之外，湖人本次輸掉湖鹿大戰還和他們此前的賽程有一定原因。此前湖人經歷了4連客之旅，隊伍疲憊不堪，在對陣雄鹿之前並沒有太長的休息時間，所以本次輸掉比賽也是可以理解的。而且就算是輸了球，對於湖人來說也不是什麼大問題。相反的，如今湖人2連敗更能讓他們發現問題的所在，賽季到現在才剛剛進行了一半不到，未來還有不少時間留給他們調整。

這麼來看的話，今年湖人很可能會考慮在賽季進行的差不多是時買來幾位射手，彌補一下板凳深度。畢竟現在進入了小球時代，投射能力對球隊來說極其重要。這個賽季老詹的罰球也讓人不是很滿意，數據為近幾年來最低，相信未來老詹也會根據這一點做出一定的調整和改變。目前湖人的外線並沒有特別出色的專職控衛，未來可能還會通過交易引進。這次湖鹿最終以老詹敗北結尾，但說白了也只是一場常規賽而已，說不定湖人能夠在今年季後賽中完成復仇呢？
