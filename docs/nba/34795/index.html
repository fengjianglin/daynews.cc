實力擔當！禁賽25場後首秀27+10+0失誤，特雷楊迎回強力幫手

2019-12-24 623條評論 2,473次閱讀 212人點贊

北京時間12月24日，老鷹客場挑戰騎士，球隊首發內線科林斯因為服用禁藥被禁賽25場後終於迎來了回歸。在禁賽之前，這名17年的19號秀本賽季前5場比賽場均可以送出17分8.8板2帽且三分命中率高達47.4%，可以說是隊內僅次於特雷楊的二當家。毫無疑問，科林斯的復出能給予老鷹極大的幫助。

從同屆球員來說，科林斯已經屬於佼佼者之一了，上賽季二年級的他場均就可以送出19.5分9.8板。今天這場比賽，科林斯剛回來就立即展現了自己的實力。首節比賽9中5砍下11分1斷1帽，特雷楊同樣得到11分。上半場戰罷，科林斯得分提升到18分，特雷楊則是18分5板6助攻。

雖然最終3分分差遺憾輸給了騎士，不過今天老鷹三名年輕球員的表現絕對值得稱讚。科林斯復出首秀就20中12三分5中2高效拿到了27分10板的兩雙數據，另外還有2斷2帽且沒有任何失誤。特雷楊同樣砍下30分6板11助攻，而今年的四號秀亨特19中10也貢獻了23分。老鷹隊的三名新星這一戰合計送出80分！

還是那句話，特雷楊終於迎回了強力隊友，科林斯能在進攻端給特雷楊分擔不小的重任，兩人是老鷹隊重點培養的對象，也是老鷹隊的兩大基石球員。
