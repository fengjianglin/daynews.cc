今日NBA賽場｜掘金擒森林狼，凱爾特人拔掉活塞，獨行俠大勝76人

2019-12-21 40條評論 1,836次閱讀 272人點贊

北京時間12月21號，NBA常規賽照常進行

森林狼100-109掘金，約基奇22+10+10送森林狼九連敗

掘金在主場迎戰森林狼，開場掘金手感火熱，一波14-4迅速建立起兩位數的領先優勢，雖然森林狼做出回應，但隨後默里和巴頓聯手取下10分，森林狼首節落後13分。次節比賽， 掘金在籃下連中兩球，分差重新變為17分，兩隊此後在進攻端形成對峙。雖然威金斯連得8分，幫助森林狼縮小分差，只是森林狼並未將這股勢頭延續下去，隨著蒂格上籃得手，掘金領先10分結束上半場。

下半場回來，吉昂命中三分，森林狼在第三節開始後不久追回7分，雖然約基奇命中三分，但吉昂保持手感，他又中兩個三分球，一人獨得8分，森林狼將分差縮小到2分，節末默里連得5分，掘金領先7分第三節。末節比賽，掘金隊連拿4分，擴大領先優勢，迪奧普跳投命中為球隊止血，隨著比斯利、穆雷和約基奇輪流投中三分，一輪9-0的小高潮，掘金103-84領先，最終掘金以109-100戰勝森林狼。

本場數據

森林狼：吉昂18分6籃板；威金斯19分6助攻；蒂格13分；諾阿-馮萊9分7籃板。

掘金：約基奇22分10籃板10助攻；默里28分6籃板5助攻；巴頓13分4籃板；米爾薩普11分4籃板。

活塞93-114凱爾特人 雙探花綠軍送活塞三連敗

活塞客場挑戰凱爾特人。綠軍開局5-8落後，奧傑萊與塔圖姆各中一記三分反超比分，兩隊展開對攻大戰，比分交替上升，首節打完，凱爾特人以28-26領先活塞2分。次節比賽，活塞開局打出5-2攻勢反超比分，隨後活塞進攻受阻，綠軍多點開花杰倫-布朗帶領替補陣容打出一波11-0的攻擊波，隨後你來我往，半場結束，凱爾特人以60-48領先活塞12分。

下半場回來，塔特姆在內線連得4分，杰倫-布朗在外線火力全開，三節打完，凱爾特人已經以91-69領先活塞22分之多。末節比賽，羅斯單人連得4分，格蘭特-威廉姆斯暴走一人連砍7分，綠軍已經取得100-73領先27分的巨大優勢，活塞沒有放棄，但無奈落後分差過大，凱爾特人穩穩收下比賽。最終，凱爾特人主場114-93大勝活塞。

本場數據

活塞：德拉蒙德12分11籃板；-羅斯14分4助攻；斯內爾7分；梅克15分7籃板。

凱爾特人：沃克2分3籃板11助攻；杰倫-布朗26分5籃板5助攻；塔特姆26分5籃板；坎特8分18籃板；格蘭特-威廉姆斯18分3籃板。

獨行俠117-98大勝76人波神籃板數創生涯新高

首節比賽開始，西蒙斯空接開局，史密斯和霍福德對飆三分球，西蒙斯聯手恩比德拿下8分帶隊取得13-5的開局，獨行俠趕緊叫暫停。暫停過後，獨行俠改聯防，76人一時不適應，獨行俠趁機追分，手感火熱內外開花，波爾津吉斯暴扣，小哈達威5記三分得手，，首節打完，獨行俠隊35-28領先7分。

第二節，獨行俠的球員繼續發揮出色，波神里突外投，獨行俠用一波14-1的攻勢再一次拉開比分，在本節後半段，76人隊總算狀態回暖，西蒙斯打成2+1，恩比德和科爾克馬茲也連續三分命中，半場結束，獨行俠63-57領先6分

第三節，下半場回來，恩比德連得5分，76人隊將分差追到僅差1分。這時布倫森站出來2助攻加1記拋投，幫獨行俠隊穩住局面。恩比德撐起76人的進攻，但卻沒能帶動起76人整體的反撲，塞思-庫里最後時刻連拿7分，三節打完，獨行俠隊88-77領先11分。第四節上來，塞斯庫里，小哈達威，波神等人的手感依舊，頻頻進球，防守端波神大帽恩尼斯，76人雖然沒有放棄比賽，但是擋不住獨行俠的進攻，最終獨行俠隊117-98大勝76人隊。

本場數據

獨行俠：小哈達威27分4籃板；波爾津吉斯22分18籃板3封蓋；布倫森11分3籃板7助攻；多里安-芬尼-史密斯11分6籃板4助攻。

76人：恩比德33分17籃板；西蒙斯12分6籃板8助攻；霍福德9分3助攻；哈里斯7分2籃板。
