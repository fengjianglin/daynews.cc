威少加盟火箭成效如何？擺脫1.6億老將成功一半，戰績+數據不算差

2019-12-08 0條評論 14次閱讀 0人點贊

威少這筆交易對火箭來說是否成功毫無疑問會是影響球隊本賽季以及未來的一個問題。而本賽季至今球隊僅打了25%左右的比賽，但還是有一些數據能證明這筆交易算是成功。

評估交易是否成功的關鍵因素是火箭究竟犧牲了哪些球員去推動這筆交易，並且還應該有哪些代替方案。在今年休賽期這筆交易中，火箭送出手握1.6億美元合同的克里斯-保羅，他在上賽季的整體表現可以說是生涯最差的情況，而且很明顯他已經走上生涯下坡路。

2018-19賽季期間，保羅場均得分和投籃命中率均是生涯新低的表現（場均15.6分和41.9%投籃命中率）。當然，哈登上賽季場均36.1分所佔據的出手權很大程度上也是保羅整體數據下降的關鍵所在，畢竟上賽季他場均僅有12.4次出手。

不過值得一提的是，保羅也有在出手次數少於上賽季但卻能闖入全明星的賽季表現，而且更重要的是，上賽季保羅場均2.6次失誤是生涯第2差的表現。

所以在手握高達1.6億美元的合同但卻無法在搭檔哈登情況下打出高效表現，擺脫保羅對火箭來說絕對是合理的選擇。

相比之下，威少在年少時就與哈登成為好友，而威少在今年休賽期抓住加盟火箭並且搭檔哈登的機會。

本賽季至今，威少場均嫩得到21.7分，這確實是比保羅能貢獻更多的得分，只是39.9%肯定算不上是高效的投籃命中率，更何況威少場均4.5次失誤也是他自2016-17賽季以來的最差表現。

威少在火箭陣中還是能改善球隊籃板球的問題，目前火箭場均48.1個籃板已經是聯盟第3高的表現，要知道上賽季的火箭在場均籃板這項數據上僅能排在聯盟第26的位置。還要就是火箭在過去21場比賽中能有14勝7負的戰績，這也比上賽季前21場比賽10勝11負的戰績要好得多。

威少和保羅是完全不同的球員，這其中威少是有更好運動天賦的球員，但保羅是有更好傳球能力和穩定性球員，更重要的是，在火箭依賴三分投射的戰術體系下，保羅穩定的中投能力原本應該是火箭非常理想的補充。

只是相比起保羅和哈登之間緊張的關係，火箭得到威少後最大的收穫是他能與哈登有較為良好的友誼。

從合同角度來看，能擺脫34歲的保羅那份還有3年1.24億美元的合同已經是成功一半，這相比起31歲的威少明顯是劣勢，而且威少完全可能在合同期內（5年2.06億美元，2023年夏天到期）為火箭帶來爭冠級別的實力。

考慮到火箭目前所遭受的傷病困擾，現在很難評估這筆交易情況，畢竟火箭在過去一段時間都面臨著戈登，格林甚至是豪斯的缺席。而威少最大的優點之一就是能讓身邊射手得到更好的得分機會，所以這支火箭在威少加盟後究竟能達到怎樣的高度還是個未知數。

但就目前來看，威少的加盟確實能為火箭帶來優勢，但也還有很大的提升空間。
