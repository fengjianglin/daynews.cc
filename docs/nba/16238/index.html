11連敗！這隊徹底崩了，唯一超巨離隊倒計時，他們真留不住人啊

11連敗！這隊徹底崩了，唯一超巨離隊倒計時，他們真留不住人啊

北京時間12月14日，新奧爾良鵜鶘隊客場以109-116不敵費城76人隊，迎來了本賽季的11連敗。上賽季辛辛苦苦擺爛得到的狀元簽選中了威廉森，可沒想到後者打了幾場季前賽直接因傷缺陣，無奈本賽季鵜鶘隊只能繼續擺爛。經過這一場失敗後，目前他們以6勝20負的戰績排在西部聯盟倒數第二位，稍強於殘陣勇士隊，不出意外鵜鶘隊的奮鬥目標依然是樂透，就現在來看，這支球隊真的是徹底崩了！

雖然說鵜鶘隊近來狀態不佳，但不可否認英格拉姆的表現還是值得肯定的，場均24.9分7.1籃板，投籃命中率接近五成，三分球命中率超過四成，英格拉姆的進步是肉眼可見的。在狀元威廉森復出之前，他是這支鵜鶘隊當之無愧的核心球員，也是個人能力最強的球員，值得一提的是，到明年夏天英格拉姆就成為了自由球員，而在那時的自由市場上根本沒有真正的超巨，連球星都非常罕見，因此英格拉姆必然會十分搶手，不敢說他是唯一的最大牌球員，至少他可以算得上其中之一，綜合現在來看英格拉姆離開鵜鶘隊的可能性比較大。

有消息稱尼克斯方面有意以頂薪招攬英格拉姆，這對於英格拉姆的吸引力還是很大的，明年夏天尼克斯隊擁有充足的薪資空間去簽約球星，他們可以在簽下英格拉姆的同時再招攬一名球星，雖然尼克斯隊現在的實力與鵜鶘隊不相上下，但是英格拉姆在尼克斯隊會有更好的發展前景，紐約這種大球市也是新奧爾良所無法相比的，因此當明年夏天尼克斯隊為英格拉姆拋出頂薪的橄欖枝，後者很有可能加盟尼克斯隊。

說到這裡對鵜鶘隊不由得心生憐憫，此前苦苦培養出來的濃眉哥要走，費盡心思交易來了英格拉姆，「球哥」鮑爾和哈特。好不容易有個英格拉姆打得不錯，合同到期後卻又留不住，鮑爾與哈特的表現糟糕，選中的狀元又因傷缺陣，鵜鶘隊的未來著實堪憂啊。英格拉姆走後，如果威廉森還不能夠及時復出，鵜鶘隊中很難找出比較穩定的球星，甚至說他們連一個合適的球隊核心人選都沒有，不知道這樣的鵜鶘隊何時才能摸到西決甚至是季後賽的地板。
