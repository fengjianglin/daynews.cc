細數NBA十大另類球員：中鋒拿助攻王，1米98打中鋒，外線拿蓋帽王

天天要聞
2019-11-27

俗話說的好「龍有龍道，鼠有鼠道」，在什麼崗位就要完成什麼工作，但是總有些人，干著和他實際崗位不怎麼沾邊的工作，在NBA中就存在著不少這樣的球員，明明不該出現在這個位置上的人出現在了這個位置，明明不是這個位置拿的這個獎項，卻最終成為了獎項的獲得者，什麼是很奇葩？下面小編就給大家帶來NBA歷史上的十大另類球員。

1、蓋帽王基里連科

聯盟蓋帽王的獎項，大多數多屬於球隊的內線球員，因為他們距離三秒區更近，身高也佔據著優勢，可以迅速的回到籃下完成補防，不過之所以說是大多數，那是一位在NBA賽季蓋帽王的名單中，出現了一個另類，這個另類球員就是被稱為「AK47」的基里連科，在04-05賽季，基里連科以場均3.3次蓋帽的數據獲得蓋帽王的榮譽，要知道球場上的基里連科，可是實打實的小前鋒，不知道他成為蓋帽王，有沒有令聯盟的一眾內線蓋帽高手汗顏。

2、搶斷王追夢格林

就像蓋帽王是內線球員的專屬一樣，搶斷王同樣是外線球員的專屬，因為大部分球權都掌握在外線球員手中，他們的靈活性也要比內線球員更加出色，這也讓外線球員更容易得到搶斷，但是這其中也有另類，這個球員就是德雷蒙德格林，在16-17賽季，格林以場均2次搶斷的數據成為了當賽季的搶斷王。

3、助攻王張伯倫

只要踏上球場，張大帥似乎能做到那些他想要做到的一切，不過最另類的一點就是，張大帥還獲得過助攻王，場均8.6次助攻僅次於大O羅伯特森的場均9.7助攻，排名第二。然而當時的NBA的助攻王是以賽季總助攻數，張伯倫這個賽季打滿了82場比賽，而大O只打了65場比賽，最後張伯倫成功當選了這個賽季的助攻王，成為NBA歷史上唯一一個中鋒助攻王。要知道在NBA助攻王的歷史上，幾乎是清一色的組織後衛，特別是張大帥的位置還是作為終結點的中鋒，以中鋒身份搶控衛的飯吃，張大帥不另類誰另類。

4、最矮的中鋒海耶斯

1米98的身高屬於標準的分衛身高，用來打小前鋒都要需要考驗球員的臂展和運動能力，但是馬賽克隊舊將查克海耶斯，卻在姚明傷病報銷之後，以1米98的身高和極為平庸的運動能力，頂到了球隊首發中鋒的位置，而且效果還十分不錯，更是打出了一次三雙的數據，特別是和爵士隊兩大全明星的較量中抓下18個籃板，這樣的表現，完全不輸於那些身高7尺的藍領中鋒。

5、沒有得分能力的鋒衛搖擺人羅伯森

籃球運動說到底，都是一場關於得分的比賽，得不了分，說什麼都是白搭，特別是作為一名外線球員來說，擁有一手投射能力是必不可少的，但是這對於雷霆隊之前幾個賽季的首發鋒衛搖擺人來說，得分堪稱是一種奢望，畢竟這位除了上空籃外，其他任何得分方式都靠運氣，連罰球都能出現三不沾，甚至在球場上，對手都不屑於防守羅伯森。

6、空間型大中鋒洛佩茲

聯盟歷史上，空間型內線並不在少數，但是能達到大洛佩茲這種程度的，完全沒有，作為當年的背身單打小能人，7尺大漢，現在的洛佩茲在進攻端是能不進三分線就不進，場均三分出手甚至比兩分出手還要多，特別是作為當年籃網隊的內線大殺器，洛佩茲現在的場均罰球還不足三次，籃板也不足5個，要不是還有一手不錯的蓋帽能力，洛佩茲的數據簡直比外線都要外線。

7、最高的控衛西蒙斯

當年魔術師約翰遜以2米06的身高出任組織後衛，已經創造了聯盟最高控衛球員的紀錄，誰能想到，西蒙斯以2米08標準的大前鋒身高，出現在76人隊控衛的位置上，而且除了三分之外，不管是組織進攻還是防守，西蒙斯做的都還十分不錯，也讓對手球隊在防守西蒙斯時陷入了如何對位的麻煩。

8、說不清是什麼位置的波神

本賽季的波神到底在球場上打得什麼位置？說是內線球員吧，但是大部分時間波神都在三分線外，而且很少主動要求在內線單打對手，說是外線球員吧，你們誰見過身高2米20的外線？但是40.7%的命中率對於內線來說是不是太低了？拿場均2+的蓋帽又是怎麼回事？他到底打得什麼位置呢？

9、最不會組織的控衛戴爾埃利斯

組織後衛在球場上最主要的工作是什麼？答案當然是組織球隊進攻了，就連當年的艾弗森和庫里都沒有放棄組織進攻的大任，但是放在埃利斯身上就行不通了，畢竟這位身高1米88的超音速首發控衛，職業生涯可以貢獻15.7分，但場均助攻卻只有1.4次，作為控球後衛打出這樣的數據，稱為球場黑洞在合適不過的了。

10、最不喜歡得分的球員羅德曼

球員沒有不看重得分數據的，因為這能幫助球員拿到更大的合同，但是羅德曼卻是一個例外，這貨簡直把得分視為糞土，要知道作為一名自主得分能力不強的內線球員來說，得分的兩大手段一是二次進攻，一是吃餅，就吃餅而言，身邊從來不缺少巨星隊友的羅德曼應該有大量的吃餅機會，而場均4.8個進攻籃板也讓羅德曼不缺少二次進攻的機會，再加上他還能投一下三分，這證明了羅德曼場均得分上雙並不是一個特別困難的事，但羅德曼就是這麼另類，不僅不得分，更是懶得得分，單場0分28板就是一個鮮明的例子。
