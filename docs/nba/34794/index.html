無問西東之：華盛頓巫師們的子彈

2019-12-24 556條評論 3,161次閱讀 1675人點贊

華盛頓奇才隊，成立於1961年並加盟NBA，是一支屬於美國的哥倫比亞特區華盛頓為基地的NBA職業籃球隊，是美國男籃職業聯賽（NBA）東部聯盟東南部賽區的一部分。 奇才隊又稱華盛頓巫師隊，是NBA中名字變化最多的球隊，前身是巴爾的摩子彈隊，在1996-97賽季結束之後，球隊啟用了新的標誌並更名為華盛頓奇才隊。從60年代開始，球隊就是東部的強隊之一，連續打入季後賽，並於1977-78賽季奪得球隊歷史上第一座總冠軍 。不過90年代奇才隊大部分時間在扮演失敗者的角色，整個90年代只於1996-1997賽季進過季後賽。隨著吉爾伯特·阿里納斯在2004-05賽季的強勢崛起，奇才隊重新殺入季後賽。但從2008年至2011年，奇才隊連續三年與季後賽無緣，直到2013-14賽季，在約翰·沃爾等球員的帶領下，重返季後賽。

捧杯的瞬間

1977-78賽季，子彈隊在頭39場比賽贏了24場。最終以44勝38負的戰績排在馬刺隊之後，位居中央區的第二名。進入季後賽之後，球隊又重拾賽季之初的狀態，橫掃亞特蘭大老鷹隊，緊接著淘汰掉了馬刺隊和76人隊，進入了總決賽。最終子彈隊以4-3的總比分擊敗西雅圖超音速隊，成為NBA的總冠軍，彌補連續兩次在總決賽中被橫掃出局的「魔咒」（分別是1971年和1975年）。

「大將軍」駕到

2003年8月8日，阿里納斯以自由球員的身份正式加盟華盛頓奇才隊。2003-04賽季，阿里納斯為奇才出場55場比賽，場均出場時間37.6 分鐘，場均得分(19.6 分)和助攻(5.2次)，宣布華盛頓的「0號特工」就位。2004-05賽季，阿里納斯每場比賽得分25.5分，追平單季生涯新高的每場比賽籃板4.7個，平均每場5.1次助攻和1.74次搶斷，與隊友賈米森成為自傑夫·馬龍和摩西·馬龍在1987年後奇才歷史上共同出現在全明星賽球員。2005-06賽季，阿里納斯首發出場80場比賽，平均每場獲得職業生涯最高的29.3分，場均得分列NBA得分榜第四位(29.3分) ，他連續第二次入選進入NBA全明星賽。2006-07賽季，阿里納斯出場74次(73次首發)，平均每場21.7分， 5.4次助攻，4.2個籃板和1.73次搶斷。本賽季他和科比的巔峰對決成為了永恆的經典：2006年12月18日，阿里納斯誰對做客斯台普斯，湖人當家科比狀態火熱，24投15中拿到45分8籃板10助攻的准三雙，阿里納斯更加瘋狂，他全場32投17中（其中3分球12投5中）得到職業生涯最高的60分，同時還有8籃板8助攻2搶斷，率領奇才在斯台普斯中心反客為主147-141擊敗湖人。2008年，奇才與阿里納斯續約，雙方簽訂了一份為期6年價值1.1億美元的合同，但由於傷病，這份合同成了一杯毒藥。

「籃球之神」到此一游

2001-2002賽季，邁克爾·喬丹再次宣布復出加盟華盛頓奇才隊。為奇才隊效力的第一個賽季，喬丹場均得到22.9分、5.2次助攻和1.42次搶斷，為全隊最高，率領球隊比上個賽季提高了18場勝利（從19勝63負到37勝45負）。還在12月29日對陣山貓隊的比賽中，獨得51分，這是在MCI中心球館裡最高的分數。由於喬丹復出的推動作用，奇才隊常規賽的觀眾是整個聯盟最多的。球隊創紀錄地主場41場全部售罄，另有38場客場的票也全都賣光。2002-03賽季，邁克爾·喬丹宣布退役，在他最後一個NBA賽季里，喬丹是球隊唯一82場出全勤的隊員，其中首發了67場，平均每場得到20分，6.1個籃板和3.8次助攻，還有1.5個搶斷，而投籃命中率是45%，罰球命中率82%。賽季結束後，籃球之神打光了最後一刻子彈，從此雲遊！

「雙爾」也將彈盡糧絕

2010-11賽季，奇才隊在2010年選秀大會上幸運地馬上抽到狀元，並挑選來自肯塔基大學的約翰·沃爾。2012年6月29日NBA選秀，布拉德利·比爾在首輪第3順位被華盛頓奇才隊選中。從此，特區人民看到了球隊復興的希望，「雙爾」組成的「黃金雙槍」在後衛線上摩拳擦掌，準備大幹一場。2013-2015賽季，球隊連續挺進季後賽，然而最終都是在第二輪敗下陣來。最終，最終，可惡的傷病還是沒有放過這支球隊，沃爾倒下了，獨木難支的比爾扛著球隊前進，但是拼到只剩最後一顆子彈的華盛頓人離季後賽漸行漸遠。本賽季，沃爾場均得分28.3分高居聯盟第五，可是球隊卻以9勝20負排名東部第12，基本宣告退出季後賽的爭奪。12月24日，奇才戰勝尼克斯，比爾出手38次；12月22日，奇才挑戰76人，比爾出手24次；12月21日，奇才客戰猛龍，比爾出手24次；12月29日，奇才挑戰公牛，比爾出手19次；12月17日，做客活塞，比爾出手24次；12月15日，奇才負灰熊，比爾出手23次。。。。。。瘋狂的輸出，卻換不來回報，比爾或許也累了！

兄弟，你倒下了還有我，但我不知道能不能撐到你回來？！

致平凡的我們：凡事儘力而為，但有時拼至刺刀見紅只為無愧於心！
