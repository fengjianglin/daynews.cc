3步扣籃被吹邊線球！和蔡恩一樣，字母哥終於被自己的天賦耽誤了

2019-12-10 36條評論 1,069次閱讀 113人點贊

今天魔術和雄鹿的比賽開局就發生了非常匪夷所思的一幕——字母哥在球隊的一次快攻當中被身旁的富尼耶拉到了手，隨後裁判吹罰了富尼耶的犯規！

有意思的是，字母哥當時並沒有停速，直接一個加速殺到內線，來了一記勢大力沉的掄扣！可隨後裁判表示——字母哥的扣籃進球無效，先犯規！只是普通犯規！

這字母哥就有點鬱悶了！趕忙用手指做出了畫圈圈的動作，當然，不是「畫個圈圈詛咒裁判」，而是提醒裁判，這球我3步上籃起步了，應該是個2+1！

可是裁判在回看錄像之後依然只給了個普通犯規，發邊線球！對此，字母哥也只能抿嘴表示無奈了！

話說，有聽過因為自己的天賦太強，而被耽誤的嗎？的確，NBA本賽季就有1例——蔡恩·威廉森。

今年剛進入NBA，在鵜鶘拍攝定妝照的時候，蔡恩就被發現有至少2個英格拉姆那麼大！

而在本賽季鵜鶘打湖人的時候，濃眉哥戰舊主，在更衣室過道遇到了年輕的蔡恩的時候我們也是發現，蔡恩居然有2個濃眉哥那麼厚！

的確，蔡恩是歷史罕見的天賦之子，他身體的壯實程度甚至連詹姆斯都不及！

就說一點——同樣是聯盟新贊助商耐克製作的立體剪裁的球衣，詹姆斯都沒辦法把袖口撐出褶皺，胸口撐起小山，但是蔡恩就可以！

不過同樣值得一提的是，因為其過於龐大的體重卻只有2m左右的身高，蔡恩自進入聯盟起膝蓋就一直在受傷，看起來就是個玻璃人！而受傷期間，蔡恩的體重進一步增大了！現在他看起來胖成了個球！很多球迷都參照日本的相撲，戲謔他為「行走的大肥肉」！

所以，天賦雖強，但最終卻沒有發揮正向效果，這可不就是被耽誤了嘛！而應該說和蔡恩一樣，字母哥的這次上籃也是；因為換個角度看慢鏡頭，字母哥是3步起了沒錯，可是卻是從三分線外1m多就開始起3步了！

三分線外1m多就開始起3步！關鍵是最後還暴扣！可以想像字母哥這步幅有多大？快一個我們普通人的2倍了吧！

但是大雖大，快攻可以很快，有時候也容易沒辦法及時轉彎、剎車；更有時候，遇到沒見過世面的裁判，也得不到恩惠。
