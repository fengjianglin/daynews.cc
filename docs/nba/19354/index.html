三分球8中0！馬刺花3年才找到的“庫里”，就這麼被用廢了？

2019-12-18 232條評論 2,078次閱讀 458人點贊

還記布林-福布斯嗎？上賽季，馬刺遭遇近20年來的最大危機，少主倫納德連同頂級3D格林一起離隊，帕克轉投黃蜂，吉諾比利宣布退役，新星默里賽季報銷，儘管厄運一個接一個，但神奇的他們仍然闖入季後賽，並在首輪跟掘金大戰7場才漸漸敗退。

馬刺之所以能打成這樣，福布斯功不可沒，落選秀出身的他，在場均28分鐘的出場時間裡，交出一份11.8分，2.9個籃板和2.1次助攻的成績單，命中率46%，三分球命中率43%，場均能扔進2.1記三分球，那個賽季的福布斯，是馬刺隊進攻最高效，三分最准，也是最高產的外線火力點。

更關鍵的是，福布斯不佔球權，使用率17.4%，排在全隊第10位，但他的真實命中率58.6%，卻排在後場球員中的第一位。季後賽中，福布斯更加高效，得分10+，2項命中率均超過了48%，儘管馬刺的後場競爭激烈，但顯然，福布斯已經率先脫穎而出，哪怕是默里傷愈復出，福布斯依舊是雷打不動的首發。

25場比賽，25場首發，全隊除德羅贊和阿爾德里奇之外，只有他能享有這樣的待遇，但這一次，福布斯卻辜負了老帥的信任，合同年的他並沒有繼續蛻變，反而打得越來越不像自己，成績單雖然還是11.9分+2.5個籃板+2.0次助攻，可命中率卻只有40%和34.4%，過去2戰，更是14投僅1中，三分球8投0中。

也許是太想在合同年賭把大的，如今的福布斯，不再只滿足於無球，打法甚至有點像2屆MVP庫里，空位三分不投，清一色頂著防守扔超遠，時不時還給你來個單打……

福布斯作為落選秀，不是沒有道理的，身高不足，運動能力也十分有限，控球和傳球都不擅長，他上賽季之所以能如此高效，主要是因為他在做自己最擅長的事，那就是打無球，當定點射手。

上賽季，福布斯使用最多，也是最高效的進攻方式就是定點投籃，佔比36%，每回合能得到1.22分；其次是轉換進攻、手遞手進攻等等，每回合也都能拿到1.2分；至於單打，佔比僅為7.6%，每回合只能得到0.83分，然而本賽季，福布斯的使用率已經逼近20%了，超過了正牌控衛德里克-懷特，甚至比第二陣容的核心米爾斯還高，排在全隊第5位。

NBA的任何一位後場，都希望能像庫里那樣的打球，但天賦是個好東西，它不是你努力就能彌補的，就如同身價，一個2億美元，一個2年600萬美元，一個MVP，一個落選秀出身，不是貶低福布斯，只是希望他能認清自己，過去2場比賽，他的表現已經令波波維奇很不滿了，上一場更是直接末節雪藏，加時賽也一秒沒打。

2場比賽的出場時間，都被控制在20分鐘以下，要知道，防守端的福布斯，已經是聯盟倒數15人中的一個，如果他的進攻還一團糟的話，結果可想而知，若不是波波維奇手上無人可用，怎能讓福布斯如此放肆，唉！馬刺全隊現在，只有3人場均能進至少1個三分球，福布斯就是其中之一。

另外2人，分別是老將貝里內利和米爾斯，波波維奇是真沒轍啊，首發5人中，怎麼著也得有個能投三分的吧，但好消息是，新秀朗尼-沃克正在逐漸兌現天賦，福布斯，你快醒醒吧，小心一念成魔，自毀前程啊。
