亮瞎眼！2.13米麥基打進12年第3記三分 網友調侃該上5大囧

2019-12-24 389條評論 2,189次閱讀 1206人點贊

北京時間12月23日，2.13米的湖人神塔麥基會投三分嗎？在賽季出戰的29場比賽，麥基沒有任何一次三分出手，但在湖人慘敗輸給掘金一戰，麥基出手了賽季首次三分，並且成功命中打進賽季首記三分，也是他職業生涯12年第3記三分，無疑是極為令人震驚一幕。至於全場麥基的表現一般，他只有6中3得到7分5籃板2蓋帽1助攻數據，卻吃到4次犯規。

在末節比賽已經提前進入到垃圾時間後，麥基在第四節還剩下3分多鐘時，持球推進到外線三分時，麥基並沒有選擇分球給對手，而是直接選擇干拔三分出手。要知道，掘金球員在目睹麥基殺過來時，還主動往兩旁退散，無疑就是預測麥基接下來會傳球。未曾想，麥基竟然是自己直接選擇三分出手，而皮球也是應聲入網打進三分，完成他賽季首次三分命中，成為極為令人「震驚一幕」，而湖人主場的球迷也是有不少人起立為他歡呼。

要知道，本賽季打了整整30場比賽，麥基才出手了如此一次三分球，卻神奇打進三分。至於麥基已經在NBA效力12年，他職業生涯常規賽至今合計才打進3記三分球，合計準星只有28中3。

麥基生涯首次打進三分是2012-13賽季效力掘金時，當時在2013年1月1日掘金大勝快船之戰麥基打進生涯首記三分。至於第二記三分則是上賽季，麥基在湖人大勝太陽之戰再中一記三分。

除此之外，麥基的生涯打過5次季後賽，僅僅只有1次三分出手不中，畢竟對於2.13米的麥基而言，出手三分並不在他的業務範圍之內，讓他去投三分也是頗為強人所難。

只是麥基能夠打進三分，還是令球迷極為震驚之事，甚至有網友呼籲麥基應該再度登上奧尼爾的「五大囧」節目，畢竟如此一幕讓所有人都「囧」然一笑。

（文/醉卧浮生）
