比爾35+10難掩小刺客高光，最強175傷停五戰爆發，23+6對轟羅斯

2019-12-18 354條評論 3,814次閱讀 924人點贊

北京時間12月17日，NBA常規賽繼續進行，華盛頓奇才隊客場133-119擊敗底特律活塞隊，結束了四連敗的窘境，本賽季的奇才與活塞都算是東部弱旅，實際上活塞是有一定的季後賽可能性的，但因為管理層的一些思路他們還在猶豫到底是放棄本賽季還是繼續衝擊季後賽，格里芬、德拉蒙德等人交易流言頻發侵蝕人心，此役德拉蒙德與格里芬還缺陣，格里芬缺席的理由為左膝酸痛，庄神缺席的理由是左眼過敏，這支活塞的領袖就成了德里克-羅斯，而奇才這邊比爾是核心，傷停五場比賽的先發控衛小托馬斯也復出，綜合實力奇才就壓過活塞一頭了。

活塞在上一場比賽中可是擊敗了西部豪強馬賽克隊的，而且也是在庄神與德拉蒙德傷停的情況下，只能說活塞沒有領袖後缺乏穩定性，這場比賽讓奇才轟下133分，全場88中49命中率56%，三分球32中18命中率56%，這個進攻效率實在是太高了，而活塞86中41命中率48%，三分球39中13命中率33%，差距相當明顯，而且活塞隊的兩個核心都打的有聲有色，布拉德利-比爾在這場比賽中轟下了35分全場最高，他一人就改變了活塞的防守。

比爾此役開場就暴走，首節單節打滿12分鐘，9中6，三分7中4，轟下15分6助攻，他帶領奇才轟下43分，單節凈勝活塞12分衛比賽的勝利奠定基調，在末節他8分鐘，4中3拿下11分2助攻終結比賽，全場比賽24中12，三分8中4，拿下35分10助攻3籃板1搶斷，命中率50%，三分命中率50%，拿下本賽季第二次30+10助，比爾的表現讓活塞難以防守。不小托馬斯此役復出對於這場勝利也有很大幫助，比爾無法掩蓋他的光芒。

小托馬斯在12月4日對陣魔術轟下20分7助攻5籃板後就因為小腿傷勢傷停了5場比賽，此役小托馬斯復出首發登場，布魯克斯為他制定最多24分鐘的登場時間，但小托馬斯表示自己已經沒有大礙了，為了贏得勝利想要多打一些時間，最終他26分鐘，13中9，三分7中5，拿下23分6助攻1籃板1蓋帽，命中率69.2%，三分命中率71.4%，打出了一場效率值極高的比賽，在關鍵的第四節，單節6分鐘，4中4，三分3中3，轟下11分2助攻1籃板成為比爾最好的幫手，那個地表最強175又回來了！德里克-羅斯此役作為活塞的核心都沒能壓制他。

羅斯此役全場29分鐘，17中8，三分5中2，罰球4中4，拿下22分8助攻6籃板1搶斷，命中率47.1%，三分命中率40%，羅斯這數據已經相當全面完美了，可效率值不及小刺客，此役的小刺客還幫助奇才拿下了最後的勝利，他這場比賽光芒蓋過了羅斯。羅斯與小托馬斯都是因為傷病隕落的球星，兩人也些許共同之處，當這兩人在場上對位時不少的球迷會為之感到一陣心酸，兩個當年的頂級控衛因為傷病淪落至此，好在兩人都沒有放棄，羅斯在今夏獲得了2年1500萬的合同，小托馬斯獲得1年232萬的底薪，不過本賽季小刺客效率值暴漲，場均25分鐘，13.2分5.1助攻1.8搶斷0.44蓋帽，命中率46.2%，三分命中率43.3%，場均2個三分球，他在明夏有希望拿下一份中產合同。
