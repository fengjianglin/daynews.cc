袁方怎麼看-湖人和快船到底誰更強？

天天要聞
3 週 ago

這個話題，應該有機會一直吵到今年的西決。

如果像擺沙盤一樣去排兵布陣和分析，兩個隊還真的是難分伯仲，但我認為兩個隊互相比較的大原則有這麼幾個。

第一點，也是最最最重要的一點就是，湖人快船誰更強，要看誰能更好地管理球員的健康和傷病，特別是各自核心的身體健康，誰能管好誰就能幹掉對方，或者至少贏來和對方在同一條起跑線上公平競賽的機會。兩邊的四大超巨，一個比一個有問題和有隱患。詹姆斯看起來生龍活虎，健康得不得了，和他去年聖誕大戰受傷之前的樣子是一樣的，彷彿是有不老之法術，但畢竟是一名真真正正的老漢了，他不傷則已，一旦傷了，他所需要的恢復時間是超過常人的。保羅-喬治剛剛傷愈復出，按理說應該比較皮實，或者最起碼看概率的話應該不會再輪到他了，但他和詹姆斯面臨的挑戰一樣，就是能不能維持住這種健康，湖人和快船能否各自管理好詹姆斯和泡椒的健康狀態，將具有決定性的作用。再來說的就是「濃眉哥」安東尼-戴維斯和卡哇伊-萊昂納德了，這倆哥們都屬於「皮糙肉厚心更硬」那種類型的，什麼意思？他倆人為了到心愛的球隊打球，可是各自得罪了一大片人啊，但他們倆經此一遭，反而無所謂了，但是他倆可都是實實在在有傷病的，都屬於帶傷上陣，並且隨時有可能複發。戴維斯的肩膀和卡哇伊的膝蓋，可能是湖人和快船能否更進一步的絕對保障，兩個隊拼上所有，也要管理好他們倆的傷病。至於其他人，也並非完全可以置之不理，庫茲瑪也比較玻璃，丹尼-格林也是老將，另外一邊替補雙煞路威和哈雷爾看起來比較結實正當年，但誰都不敢大意。我們最希望看到的是兩個隊齊齊整整站上西決的賽場。

第二點，就要看誰能更好地管理好球隊隊內的情緒，這一點湖人隊處境更為複雜與兇險，快船屬於看熱鬧型的，但也不能大意。洛杉磯，是非之地，體育娛樂之都，有任何一點風吹草動，都能有數百名記者快速響應衝進球館，同樣一件事情，發生在孟菲斯和發生在洛杉磯是有著天壤之別的。湖人隊這幾年吃了無數虧，隊內隊外都是瓜無數，吃瓜群眾是無辜的，但球員也是無辜的，一旦心情情緒被影響，這球就打不好了。

第三點，要看教練。沃格爾迄今為止表現不錯，而且畢竟是擁有聯盟中的絕對頂配助理教練團，沒理由不胸有成竹，里弗斯那邊關鍵就是在於熟，在於沒有任何後顧之憂。看哪個隊能為他們的教練創造更好的工作環境去潛心研究戰術而不是擔心受怕壓力過大。

這三點是有可能在大方向上直接影響兩隊對陣結果的，其他的東西，比如臨場發揮，比如角色球員的表現，等等等等，都是現場上相對不可控的事情，就不用咱們多操心去分析了，只能靠球員們自己打了。
