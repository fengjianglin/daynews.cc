熱火打響“字母保衛戰”萊利5年7億花對了？雄鹿1.7億先生是天坑

2019-12-05 0條評論 19次閱讀 0人點贊

​2021年的自由市場，字母哥將會是第一大魚，如今很多球隊已經開始未雨綢繆。

​《邁阿密哨兵》現在也曝出，熱火隊正在積極騰出薪金空間準備出手爭奪字母哥，萊利這樣的人一旦參戰，就一定是個大場面和大手筆。

​2014年之後，熱火從波什的毒瘤合同一直到現在，很多人都看不明白熱火的布局，沒有一個真正意義的巨星，不給韋德頂薪，但是這裡的角色球星到任何球隊，都有可能是主力。德拉季奇，約翰遜，溫斯洛，奧利尼克，阿德巴約，加上巴特勒，現在的熱火很像一個敢死隊。萊利過去5砸了7個億在這些人身上，最終修修補補留下了現有的陣容。

​均衡，攻防有序，這是熱火在過去幾年追求的一個目標，戰勝勇士的比賽熱火全隊多點開花，共有6人得分15+，本賽季熱火竟然有7個人場均得分上雙，他們現在的整體看起來和雄鹿很像，但實際上一點都不像，這是一個防守頂級戰術素養更高的球隊，關鍵時刻持球點很多。

​雄鹿在過去的幾年憑藉字母哥的成長已經成為了頂級球隊，但是即便如今領銜東部，可依然很多的隱患。他們的進攻火力聯盟第一，但是防守是中下游的水準，接近第三梯隊。和湖人一樣他們取得十連勝，雄鹿上一次取得常規賽10連勝還要追溯到1985-86賽季，當時雄鹿在1985年2月6日-2月28日間連贏10場比賽。

​但更多人看好湖人，快船，東部的綠軍和費城，雄鹿嗎？還差點意思。就是因為他們的陣容是真正意義的單核，看起來都是當打的球星，但是沒有一個是超一流的，湖人有雙核，快船是超級雙核，綠軍也有，費城有五大，但是雄鹿這邊的打法大概就是圍繞字母哥的一星四射，五大的陣容雖然都能投籃，但是在上個賽季被淘汰，已經證明他們球星的成色遠遠不夠。

​最恐怖的是，雄鹿現在運行空間的能力，遠不如熱火靈活，你一個米德爾頓1.7億已經堵死了很多路，這種3D雄鹿敢砸這麼多錢？米頓無非就是准一流的球星，關鍵時刻這個球隊持球的人只有字母哥，血布也差點意思，但是反觀熱火呢？萊利這幾年找的這些人，正面看都是當打的狠角色，反過來看，就都是能夠出手的一把好牌。

​你和任何球隊交易，幾個狠角色加一起一定更好操作，熱火現在的這些人都在年限里，身體倍棒，可用之才，不管是單方面的還是拽上一個第三方交易，熱火都擁有綠軍安吉那樣的交易資本。字母哥在今年如果衛冕MVP然後再次折戟季後賽，那麼他和濃眉的路線就可以一致了，一定去找一個真正的巨星搭檔，巴特勒這種攻防一體的狠角色，是個完美選擇。

​所以無論從運作，巨星搭檔還是球隊的根基球星的布置，熱火現在確實有資格叫板所有對手，他們甚至比任何對手都渴望字母哥的到來，重塑這個球隊的冠軍之風。
