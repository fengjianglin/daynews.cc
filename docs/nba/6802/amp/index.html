錫安即將復出，但在東契奇莫蘭特面前，他還是未來聯盟第一人嗎？

天天要聞
3 週 ago

錫安快要復出了！

今天，TNT評論員雷吉-米勒帶來了最新消息，錫安很有可能將在12月中旬復出，距離現在已不遠。

作為2019屆的狀元，天賦出眾的錫安威廉姆森還沒進聯盟就被無數人追捧。ESPN把他排在百大球員榜第42位；2K給出最高的97的扣籃值；NBC體育預測未來5年聯盟前五十的球員把他排在第五；露天看台評選的23歲以下球員TOP10，福布斯公布的2019年球鞋代言收入排行榜等等，錫安也都是榜上有名，而且排名都不低！

當然，他的表現也是可圈可點。季前賽，他為鵜鶘出戰4場場均能得到23.3分6.5籃板，投籃命中率高達71.4%。在他的帶領下，鵜鶘客場大勝老鷹，客場逆轉公牛，主場險勝爵士，客場力克馬刺，豪取四連勝！表現極其出色！

但是因為體重和打發，傷病也隨之而來。事實上，上賽季在一場杜克大學對北卡的焦點大戰中，他在一次持球突破時用力過猛，蹬破了自己的球鞋，並因此滑到，膝蓋受傷。在夏季聯賽時，錫安與其他球員相撞導致膝蓋受傷，隨後鵜鶘隊讓他缺席了剩餘的夏季聯賽。

北京時間10月19日，《露天看台》消息，因為右膝酸痛，錫安-威廉姆森將缺席數周的比賽。3天後的10月22日，鵜鶘官方宣布，錫安將接受右膝外側半月板撕裂的關節鏡手術，預計需要6-8周的時間才能康復。

一個月後的今天，頂著天才之名的錫安即將歸來，但他發現似乎這個聯盟有著太多比他出色的天才了。

東契奇，18年探花秀，新秀賽季場均21.2分7.8籃板6助攻，榮膺最佳新秀。本賽季，才生涯第二個賽季的他依然表現得像一個超級巨星！還不到21歲的東契奇，截止今天已經場均可以砍下30.6分10.1個籃板9.8次助攻1.3次搶斷的全面數據，賽季16場比賽已經有場砍下了三雙。場均助攻榜上排名第二，僅次於詹姆斯！

對陣馬刺，他砍下42分10籃板11助攻的超級三雙，成為21歲以下拿下40+三雙的第二人，比肩詹姆斯！

對陣勇士，他出戰25分30秒，第四節並未出場就砍下35分10籃板11助攻，成為NBA歷史上首位在26分鐘內砍下35+三雙的球員，同時，東契奇生涯30+三雙的次數達到6次，超越詹姆斯的5次，成為NBA歷史上20歲拿到30+三雙最多的球員。

今天對陣擁有雙MVP的XX隊，他再次打出讓人驚訝的表演。全場東契奇出場34分鐘就轟下41分6籃板10助攻的超強數據，關鍵的最後一節，他單節8投5中拿到13分2助攻，徹底撲滅了對方的勢頭。

這一場比賽之後，他僅用88場達成2000+500+500的數據，僅次於大O奧斯卡-羅伯特森。同時他也成為NBA歷史上第二年輕的連續4場30+球員，僅次於杜蘭特，NBA過去35年也只有4人打出過這樣的數據。

老鷹的特雷楊也不差。本賽季楊出戰15場比賽，場均可以得到25.9分4.4籃板8.7助攻1.6搶斷，三分球命中率34.9%。

客場爆冷擊敗掘金的比賽，他三分球13中8，狂砍42分11次助攻4籃板。這是他職業生涯第二次拿到至少40分10次助攻，在上賽季對陣公牛的比賽里，特雷-楊曾拿到過49分16次助攻。在NBA歷史上，此前只有兩名球員在22歲之前多次拿到至少40分10次助攻，分別是勒布朗-詹姆斯（4次）和邁克爾-喬丹（2次）。

這是特雷-楊連續三場比賽拿到至少30分10次助攻，在進入21世紀以來，此前只有四位球員能夠至少連續三場拿到至少30分10次助攻，分別是哈登、韋斯布魯克、詹姆斯和韋德。而特雷-楊也是NBA歷史上最年輕連續三場打出30+10（得分以及助攻）的球員。

說完去年這兩位天才，再說說錫安的同屆生門。

榜眼莫蘭特，賽季至今為灰熊出戰14場，場均可以得到19.1分3.1籃板6助攻1.2搶斷，三分命中率41.9%。對陣籃網的比賽，他全場得到30分9次助攻，第四節最後時刻他突破得手把比分追平，隨後又高高躍起封蓋了歐文欲圖絕殺的中投，加時賽最後時刻也是他吸引防守注意力，幫助克勞德三分出手一劍封喉。

對陣黃蜂，他在終場前0.7秒上籃命中完成准絕殺，灰熊119-117險勝黃蜂，取得兩連勝。

7號秀科比懷特，賽季場均13.9分3.6籃板2.3助攻，三分命中率35.6%。對陣尼克斯的比賽，他在末節三分球8投7中，成為NBA歷史上新秀球員在單節比賽中命中的最多的三分紀錄。同時他也是本賽季聯盟第一位單節20+的新秀，同時也是本賽季聯盟第一位末節單節20+的球員。

次輪秀帕斯卡爾，場均16.8分5.1籃板1.6助攻，對陣開拓者他砍下34分13籃板，34分是本屆新秀中最高得分。

還有熱火的落選秀納恩，場均16.8分2.5籃板3.2助攻，三分命中率37.6%。

看起來，復出後的錫安不要說未來聯盟第一人了，本賽季的最佳新秀他都有可能拿不到了！
