繼小加巴特勒之後，波神又把大帝打得強顏歡笑，76人成偽強隊？

天天要聞
2019-12-21

人生要有不較勁的智慧，當恩比德再次遇到小加時，他選擇以軟服硬，沒有正面抗衡，逃避，有時也是一種自我慰藉，如果依舊像上賽季季後賽那樣不撞南牆不回頭，估計還得惹一身傷，對心靈的打擊將會更加嚴重，加上大家老拿被小加打哭這樣戳心眼的話嘲諷他，承受力稍弱一點的話也許會毀了他，大家還記得德羅贊嗎？慘痛的先例教訓，好不容易逃離了東部，沒想那個老漢依舊窮追不捨，跑來西部包攬天下了。

為什麼要先說這樣一段苦情戲呢？答案是今天的大帝差點又被打哭了，面對沒有東契奇的獨行俠，76人竟然以一場19分的慘敗而告終，難以置信，今天這場失利，恩比德的表現其實還說的過去，33分17籃板的貢獻已經不小了，但是這次他差點哭的真正原因，是球隊整體問題，配置看上去已經算聯盟頂尖水平的他們，卻打出了魚腩球隊的表現，而且不僅僅是今天這一場球，從開賽以來，已經出現無數次這樣的情況，真的很想知道他們究竟是哪裡出了問題！

三分投射致命傷，「西不投，恩不哭，哈不進，霍不得，」表面上擁有四巨頭的76人，已經被球迷這般調侃，巨頭裡面竟然沒有一個擁有穩定的三分輸出，三分命中率最高的西蒙斯，40%，這命中率是怎麼來的想必大家心知肚明，人家壓根就不敢投，投的少運氣好就是了，另外三位爺，非常和諧，都把三分命中率保持在30%左右，鐵，實在是鐵，在當今聯盟回合數如此多的情況下，三分投成這般鐵樣，贏球的難度可想而知，擁有再好的配置，投籃不進球，比賽不得分，巨頭不就成了擺設了嗎？這時候再看看鵜鶘的雷迪克，真不知道費城球迷有多想他！

在上一場輸給熱火後，76人在東部的形勢已經岌岌可危，如果真的連熱火都吃不消，其他幾支強隊擁有更多的球星，到時候他們放遠投，守聯防，縱使你內線再強估計也無濟於事，今天的獨行俠初用這種聯防，雖說小試牛刀，但結果頗豐，相當受益，恩比德速度慢，力量弱的缺點也是76人破聯防的一個難題，如果在籃下足夠強硬，機動性夠強，牽制夠大，76人還是有一定改變的空間，他們本賽季第四節得分聯盟倒三差，失誤聯盟第二多，沒有了巴特勒這樣一個老大哥，關鍵時刻又成了大問題，最可笑的是，等到拼罰球的時候還得雪藏西蒙斯，你們說說這叫什麼事？全隊的籃子要是再練不出來，估計真的要走向偽強隊的道路了，屬實可惜。

高挑型投射人才，2.21米的波神揚手投三分，試問聯盟有幾人能夠干擾他？這隻獨角獸在沉寂一年多之後，得益於東契奇的受傷擁有了千載難逢的機會，如今的獨行俠在趾高氣揚的勝利中不斷前行，他不需要像在尼克斯時全場徒勞，現在的隊友可以幫助他勝利，在贏球中找狀態，通過這幾場球，有人覺得是東契奇抑制了波神的發揮，今天18個籃板創了生涯新高，以前東契奇在的時候都在搶波神的籃板，其實不然，波神之前的狀態是真的不穩，但其中有沒有東契奇的因素暫不下定論，但就籃板而言，東契奇是更應該拿的，他拿到籃板可以快速推進，組織反擊，而波神卻不行，不是嗎？共存，一直以來都是生物界的一大難題，東契奇和波神究竟適不適合只有他們自己能下定論，就像卡椒，至今沒有化學反應，快船不照樣打得風生水起嗎？

交易市場打開時，曾爆出獨行俠有意伊戈達拉，今天達拉斯的新聞早報特地出來澄清此事，有東契奇我們不需要一哥，別自作多情了，看來，整座城市都愛上了這個小鮮肉，一哥來的確也沒有適合的位置，現在只希望77能痊癒歸來，打好將來的每一場球，指不定哪一天，聯盟就變成他的了，還有，希望76人可以真正做出調整，大帝賽後的強顏歡笑讓人看了真的無比心酸啊！
