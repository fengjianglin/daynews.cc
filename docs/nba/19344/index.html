NBA球員原來還有自己的替身，世界上的無奇不有，網友都說他不信

2019-12-18 560條評論 3,011次閱讀 609人點贊

​籃球場上的撞臉無處不在，不僅有雙胞胎兄弟，而且還有一些類似遠方親戚的人；比如大家比較熟悉的莫里斯兄弟、黃蜂隊的澤勒兄弟、掘金隊的普拉姆利兄弟，當然還有MVP字母哥兄弟；如果兩個毫無聯繫的球員如果長得很像，究竟會是怎樣的觀感體驗呢？

森林狼的枸杞哥維金斯剪了板寸，大家都會以為他是前小牛的傳奇人物約翰遜；作為2014年的狀元，維金斯這賽季可謂打破保溫杯不在泡枸杞，雖然森林狼的戰績不怎麼樣，但是維金斯的實力大家也是有目共睹。

凱爾-科沃爾與中國人氣流行歌手周杰倫幾乎一模一樣，我怕這髮型也一定是在同一個理髮店弄出來的吧，如果周杰倫到了美國那會是這樣的景象？恐怕昆凌也是十分擔心呀。

大家好我是詹皇，詹皇就是我，我就是張震岳，戴上墨鏡的詹姆斯跟張震岳幾乎很難分辨出來，連他們迷人的微笑都是一模一樣，要說這張震岳如果不是唱歌唱得好，完全可以去NBA當詹姆斯的替身呀，前段時間詹姆斯拍攝空中大灌籃正愁沒有替身呢；建議以後詹姆斯中國行的把張震岳也叫上。

克萊湯普森是我深受們中國球迷喜歡的運動員之一，我們中國真的是一個神奇的國家，地大物博，人口眾多，湯普森也是經常來中國，難道他是來看他在中國讀書的小弟嗎？

美國有濃眉，中國當然也有，國內的初中生不僅是眉毛和戴維斯相似，他的臉型也都像極了濃眉哥，就連濃眉哥本人也在社交媒體上面轉載了這張照片；要說以後一定要來中國看看這個小夥子。

格里芬現在在活塞隊打的也是相當苦悶，於是就放棄了NBA千萬年薪來到我們美麗的中國參加了人民解放軍，看著格里芬在大雪中仍然堅守著崗位。著實令人佩服，會打球的球員不一定是好士兵，但是格里芬絕對可以。

利文斯頓是前勇士的冠軍成員，跟隨勇士拿到了3次總冠軍，跟他比較像的是16年退役的老將安德烈-米勒；同時這兩人跟我們中國著名歌手十分相似，在這個世界上他們三人絕對是神奇般的存在；米勒打法傳統，比較穩重，助攻也十分厲害，利文斯頓那招干拔跳投令所有勇士球迷懷念，而莫文蔚的歌聲一直回蕩在我們耳邊

勇士隊的追夢格林被大家熟知，布蘭登巴斯也是老一代球迷的熟人了，在凱爾特人拿到過NBA總冠軍，CBA的球迷一定不會陌生，他目前效力於CBA遼寧隊。

傳說小次郎沒有兄弟，但我表示不信，要找到跟神龜十分相似的人可謂難上加難，但不代表不存在，他是否是大威少的兄弟呢？或者是私生子？哈哈哈

肯巴沃克跟沃爾，如果你剛剛看NBA那一定不好分辨，肯巴沃克已經轉隊到了凱爾特人，而牆哥沃爾因為傷病如今還在修養當中，趕快傷愈復出吧，不然你都被人認成別人了。

歐文跟梅奧，如果不是兄弟那一定是冤家，梅奧是被寄予厚望的天賦探花秀，新秀賽季就能砍下場均18.5分，入選2008-09最佳新秀陣容，但是這人行為敗壞一度陷入毒品和美女的深淵裡，跑到中國鄉村打娛樂籃球；而歐文就不一樣了，早已經成了冠軍後衛，如今也攜手杜蘭特準備在籃網干出一番事業

說完了NBA我們來看CBA，這五官告訴你他不是易建聯恐怕你要打我；同時易建聯還兼職起了鐵匠，看來退役後的易建聯一定不愁沒有工作。
