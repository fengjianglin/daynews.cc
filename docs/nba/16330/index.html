東契奇41+12+11羅斯19分 獨行俠三原因勝活塞 1人成取勝奇兵

2019-12-13 285條評論 2,033次閱讀 225人點贊

獨行俠隊主場被國王隊擊敗之後，五連勝被對手終結掉，而在西部積分榜上，16勝7負的獨行俠隊位列西部第三名。而在這場比賽之後，獨行俠隊對陣活塞隊的比賽移至墨西哥城進行。活塞隊近期戰績不錯，最近5場比賽只輸了對陣雄鹿隊的比賽，取得了1勝4負戰績。

當然這場比賽的困難不僅在於比賽本身更在當地高海拔，里克-卡萊爾賽前就表示出擔憂，「我們必須明白這在某種程度上會有影響，除非球隊提前兩周過去，否則很難完全適應，這是基於科學的事實。我們三年前在這裡打過球，有過不錯的一段經驗，我們需要更快地進行人員輪轉。」

而對陣活塞隊的比賽，防守是比較重要的事情，最近幾場比賽活塞隊打得比較整體，總是有球員能夠在比賽中站出來。東契奇賽前就談到德里克-羅斯，「他是個非常不可思議的球員，過去是，現在也仍然是。我們必須得認真對待。他非常靈活，在場上能做很多事，所以我們必須認真防他。」

獨行俠隊跟活塞隊會打出一場怎樣的比賽呢？

盧卡-東契奇

比賽開始之後，活塞隊獲得首次進攻球權，托尼-斯內爾三分球命中首開記錄。東契奇投籃不中，托尼-斯內爾繼續嘗試外線投籃卻沒能命中，德拉蒙德在籃下進攻得手。小蒂姆-哈達威為獨行俠隊率先取分，不過德拉蒙德和格里芬繼續在進攻端得分，活塞隊開局9比2領先，獨行俠隊被迫叫了暫停。

獨行俠隊首節比賽手感不佳，甚至在暫停之後的兩次罰球不中。在球隊進攻打不開的情況下，東契奇的侵略性發揮作用，然而兩次造犯規都是兩罰一中，連續六次罰球只有兩中。東契奇背傳送出助攻，鮑威爾輕鬆扣籃得手，而隨後東契奇也找到了投籃手感，將分差縮小到了兩分。

隨著進入到最佳的進攻狀態，雙方在場上展開了對攻的局面，比賽打得激烈而又火爆。科勒貝爾命中三分球，獨行俠隊領先5分的分差，而活塞隊則打出一波6比0的比分完成反超。東契奇造犯規三罰三中，賽斯-庫里連得到5分，伍德為活塞隊得到兩分。首節比賽結束，獨行俠隊34比29領先活塞隊。

波爾津吉斯

伍德第二節比賽繼續得分，不過賽斯-庫里的外線投籃火力兇猛，這為獨行俠隊保持著領先的局面。獨行俠隊頂不住伍德的進攻，但是賽斯-庫里的進攻狀態更為出色，小庫里連得8分打出8比4的比分，獨行俠隊將分差拉大到9分。亂轉陣容階段，小庫里讓迫使活塞隊叫了暫停。

活塞隊進攻狀態提升，德拉蒙德和馬基夫-莫里斯進攻不斷得手，不過賽斯-庫里的進攻感覺仍然火熱。布朗森也加入到對攻戰團當中，活塞隊很難佔到任何的便宜。不過波爾津吉斯狀態不佳，攻防兩端都難以做出貢獻，隨著德拉蒙德在籃下肆虐，活塞隊逐漸將分差迫近。

活塞隊打出一波14比0的進攻高潮，一舉反超並取得7分領先的局面。獨行俠隊再次暫停之後，東契奇打成2+1為球隊止住血，獨行俠隊的進攻也逐漸活泛起來，東契奇和賽斯-庫里各命中三分球，獨行俠隊回應一波14比0的進攻高潮。上半場比賽結束，獨行俠隊65比58領先活塞隊。

德里克-羅斯

下半場比賽開始，德懷特-鮑威爾進攻得手，波爾津吉斯也得到兩分，獨行俠隊將領先優勢繼續擴大。活塞隊的進攻陷入到了停滯的局面，波爾津吉斯造犯規兩罰兩中之後，德拉蒙德終於為活塞隊在第三節比賽得分。獨行俠隊的戰鬥慾望比較強烈，而活塞隊則有些活力不足。

東契奇內外開花連得到5分，德懷特-鮑威爾進攻也得手，獨行俠隊將分差拉開到18分。局面對活塞隊極端不利，肯納德和德里克-羅斯各中一記三分球，但是波爾津吉斯的進攻狀態有所上升，活塞隊難以將比分迫近過多。獨行俠隊再打出一波8比0的比分，領先優勢已然達到了24分之多。

德里克-羅斯三分命中，東契奇迅速給出回應，活塞隊似乎難以破解僵局。不過在最後的兩分半鐘形勢突變，德里克-羅斯和馬基夫-莫里斯各中兩記三分球，德里克-羅斯更是在比賽1.9秒時命中三分球，活塞隊打出一波16比2的比分。三節比賽結束，獨行俠隊95比85領先活塞隊。

賽斯-庫里

末節比賽活塞隊追分，馬基夫-莫里斯成為活塞隊的急先鋒，但是他的罰球三罰只有一中。賽斯-庫里、布朗森各命中一記三分球，獨行俠隊將比分擴大到了16分。馬基夫-莫里斯進攻得手，盧卡-東契奇再中三分球，活塞隊的反撲被獨行俠隊壓制住。波爾津吉斯投籃得手，托尼-斯內爾命中一記三分球。

比賽還有最後的懸念，活塞隊也沒有放棄比賽，但是獨行俠隊的進攻保持著不錯效率。東契奇扣籃得手，德里克-羅斯命中三分球，東契奇突破打板命中，獨行俠隊掌控著場上局面。活塞隊追分無望，提前換下主力球員投子認負，全場比賽結束，獨行俠隊122比111擊敗活塞隊。

雖然本場比賽有一些起伏，不過獨行俠隊幾乎整場比賽都保持著領先，他們哪些方面做得好呢？首先，獨行俠隊活力要強過活塞隊，儘管活塞隊內線有籃板王德拉蒙德，獨行俠隊籃板球爭搶卻完全勝過活塞隊；其次，獨行俠隊進攻效率更高，尤其是外線投籃手感不錯；再次，賽斯-庫里成為獨行俠隊奇兵，他在比賽中表現得相當高效。
