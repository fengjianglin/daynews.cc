20中5，13中2！錯失關鍵絕殺！才2場啊，球隊新大腿就被打回原形

天天要聞
2019-11-26

早些時候，NBA當天的11場常規賽都已經進行完畢了，比賽的結果也大都不出所料，馬刺輸給了湖人，可能又得開始他們的連敗之路了。

不過雷霆竟然是以3分之差險勝勇士，這一點確實讓人沒想到，雷霆打現在這樣的勇士還這麼費勁？

更勁爆的是當天76人與猛龍的比賽當中，恩比德上場了31分鐘一分未得，難道真的是對猛龍產生陰影了？雖然被打哭過，但也不至於0分啊！

從國王結束了5連敗並且表現出色後，小編也是一直有關注他們的比賽，這一天國王是在客場對戰綠軍，此前國王是在自己的主場贏過綠軍一次的。

這一場比賽確卻輸了，這就體現出了主場的重要性了！當然勇士那主場不在這裡面，大通中心實在是太毒了！

有意思的是雙方的2場比賽都是以1分之差勝出的，難道現在的國王和綠軍的實力相當了？不至於吧！

這一場比賽國王這邊僅希爾德一個人就拿到了41分並且命中了11記三分，26次出手有21次是三分，看來那1000萬的獎勵希爾德是非拿不可了。

當然了比賽當中有表現好的也就有差的，這一場比賽博格丹的狀態有點迷啊！出手是20中5，三分是13中2，這場比賽的命中率25%+15%！

命中率這麼低還一直堅持出手，並且大部分還是三分，看來還是有點高估博格丹了。

命中率還是次要的，這一場比賽國王把最後的一記絕殺球交給了他，但是很可惜他丟失關鍵絕殺，導致球隊輸掉比賽。

而且他最後的決賽是選擇投三分！知道自己本場比賽三分那麼鐵，還在這麼關鍵的時刻投三分，這是故意的？

博格丹在打出30+的得分數據的時候，還被很多人認為是國王的新大腿！才多久？這才2場比賽啊！

球隊新大腿就被打回原形了，直接的導致自己球隊輸掉比賽，你都已經28歲了啊！還這麼衝動？

當然也有很多人會問，為什麼把最後的決賽交給手感不好的博格丹，其實你只有看多了國王的比賽你就會知道。

哪怕福克斯在場，最後能決定勝負的球大部分都是由博格丹來執行的，雖然他在隊內的命中率不是最高的，但大部分決勝球確實是他出手這是事實。

也不知道國王那邊教練是怎麼想的，這種球不是應該交給那場比賽手感最好的球員嗎？13中2都讓他去絕殺？
