最新MVP榜前十：沃克擠進前三，連勝助飛詹姆斯，卡椒擊落東契奇

2019-11-30 0條評論 16次閱讀 0人點贊

北京時間11月29日晚，就在剛剛NBA官網公布了最新一期的MVP榜前十名，接下來我們就來一一了解一下。

首先位列榜首的一樣的來自雄鹿隊的字母哥，賽季至今他場均能夠轟下31.1分13.7籃板6.2助攻1.5搶斷1.4蓋帽，可謂是攻守兼備。除此之外，他率領的雄鹿隊目前以15勝3負的成績，暫居東部第一名，按照聯盟的慣例，如果字母哥想要衛冕MVP的話，那他本賽季的表現勢必得比上賽季還要好才行。而從目前的情形來看，不管是個人數據還是球隊戰績，字母哥都取得了非常大的進步，因此他暫居MVP榜榜首也是情有可原。

而排名第二的則是勒布朗-詹姆斯，賽季至今他場均能夠轟下25.8分7.3籃板11助攻1.3搶斷，成為了聯盟的助攻王。眾所周知，MVP有一個至關重要的標準，那就是球隊的戰績，而目前湖人通過一波9連勝，已經穩穩地坐上了全聯盟第一的位置，因此詹姆斯的排名也得到了很大的提升，此前曾一度被東契奇給超越。

而排在第三的就讓大家比較意外了，他就是來自於凱爾特人的肯巴-沃克，賽季至今他場均僅能夠得到22.3分4.7籃板4.6助攻，命中率只有41.2%，不過三分命中率卻高達40.4%。不得不說，不管從個人數據還是在場上的效率來看，沃克都確實不算出色，但他為何能夠突飛猛進，直接闖入MVP榜前三呢？原因很簡單，因為他近一周的表現確實超乎了所有人的預料，所幸在重傷歸來之後，面對歐文的籃網隊，他轟出了39分6籃板4助攻，直接率隊擊翻籃網，幫助波士頓的球迷狠狠地出了一口惡氣。所以沃克的排名之所以可以飛速上升，球迷的情緒也起了很大的促進作用。

至於先前一度排到第2的東契奇，由於慘遭卡椒組合蹂躪之後，他的排名也瞬間跌至第4名。賽季截止至今，他場均能夠轟下30.1分10籃板9.5助攻1.5搶斷，大家都幻想著他能夠打破羅斯的記錄，成為NBA歷史上最年輕的MVP，就是不知道最後這個願望能否實現了。

其他人：哈登第5、西亞卡姆第6、約基奇第7、倫納德第8、戴維斯第9、唐斯第10。
