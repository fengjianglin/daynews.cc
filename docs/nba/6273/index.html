沃格爾的瘋狂實驗，戴維斯的波什化，或許是對湖人最好的答案

2019-12-03 0條評論 20次閱讀 0人點贊

北京時間12月3日，在洛杉磯湖人即將開啟3連客之前，我們或許可以先來預測下湖人接下來的變化，而今天我想要說的，是關於戴維斯的波什化，具體的定義為：戴維斯的場均籃板會變少，更多地在外線出手，就像當年樂福和博士一樣，作為一個三分炮台為詹姆斯拉開空間。

我知道很多人會以戴維斯的波什化來嘲諷詹姆斯廢隊友，搶數據，但實際上，戴維斯的波什化才是對湖人最好的選擇，這一點湖人主教練沃格爾其實非常清楚，那麼接下來就讓我來說說原因。

沃格爾對戴維斯的要求

本賽季沃格爾對戴維斯提出了兩個硬性指標：多進行低位單打、多投三分。看似矛盾，其實很正常。

首先，戴維斯其實不是一個低位技術頂級的內線球星，他的低位背打技術比較一般，他喜歡面框進攻和投籃，但是沃格爾很清楚，戴維斯有著成為內線大殺器的一切身體條件，而低位技術是在季後賽重要的攻堅手段，如果戴維斯練不出來，他可能就會像之前在鵜鶘那樣，數據好看，但是其實關鍵時刻對場面的影響很小。

當然沃格爾不會要求戴維斯整晚都在內線硬鑿，他還需要戴維斯將自己的進攻空間擴大，因此在此前的4連客期間，沃格爾表示他曾要求戴維斯每場比賽中至少要投5記三分，他認為戴維斯可以穩定地命中三分，為球隊打開空間，改變對方對他的防守方式。

沃格爾之所以這樣做，不僅是因為他希望濃眉完成進一步的進化，同時也知道這樣對湖人和詹姆斯的好處更大。

從波什到樂福到濃眉，到詹姆斯最怕的對手

不知道你有沒有發現，在和詹姆斯合作之前，波什、樂福、濃眉都是屬於內線進攻比列相當高的球員，但是和詹姆斯合作之後，波什和樂福最讓人印象深刻的打法就是——投三分。一次兩次是偶然，三次就算得上是必然了。

其實不管是熱火、騎士還是湖人，都非常清楚空間型大前鋒才是最適合詹姆斯的內線，因為他們能在詹姆斯突破時拉開空間，特別是他們打5號位的時候，可以把對手的大中鋒帶出去。

仔細想想吧，詹姆斯職業生涯最怕的對手是什麼？倫納德的死亡纏繞？杜蘭特的無敵中投？庫里的逆天三分？都不是，儘管這些球員很厲害，但是一對一外線互爆，詹姆斯從來不在怕的！

除開2017-2018真的打不過的兩次超級勇士，詹姆斯職業生涯輸球的季後賽都有一個類似的特點：

·2015年輸勇士，伊戈達拉、格林鋒線確實讓詹姆斯夠嗆，但勇士最後的殺招是堵在內線的博古特，詹姆斯其實真不怕五小，你看2016年博古特傷後，騎士直接連扳3場。

·2014年輸給馬刺，倫納德確實給了詹姆斯不小的消耗，但真正讓詹姆斯難受的是站在內線的鄧肯。

·2010年輸給凱爾特人，加內特才是那個讓詹姆斯難受的人。

·2009年輸給霍華德，這個真不用多解釋了吧？

此外，當年步行者能跟熱火打得難解難分，最厲害的不是喬治，而是希伯特簡直打出了東部第一中鋒的感覺。

所以你可以發現，詹姆斯不怕任何外線，但他最怕對手先用強力鋒線消耗他，然後逼他到籃下後，裡面有個大個子在那堵著他，明白了這一點，你就明白一個能投三分，拉開空間的內線有多重要了。

戴維斯的波什化，才最能發揮詹眉組合的威力

實際上，湖人要想真正發揮詹眉組合的威力，戴維斯的波什化是必須的，拿現在來說，詹姆斯和濃眉為什麼很少打擋拆？因為沒意義！詹姆斯是三號位，濃眉是四號位，擋拆完兩人面對的也是對手的三四號位球員，根本沒法形成錯位，同時擋拆完詹姆斯和濃眉都是一個往籃下走，對方的防守球員也跟著往籃下走，內線徹底堵死。

但是如果濃眉在擋拆完能拉到外線投三分呢？那對方的防守球員就不得不跟著出去，直接解放內線空間給詹姆斯，他進可強突造殺傷，退可分球讓外線投三分。

所以這也是為什麼沃格爾要求戴維斯投三分，戴維斯也願意投三分的原因，本賽季他場均投3.4個三分球，是職業生涯新高，而隨著賽季的進行，我相信這個數字還會增加。

最後再說一點，波什化對濃眉最大的好處其實是減少內線搏殺，這樣可以讓他受傷的幾率更小一些，因為濃眉本身有傷在身，有穩定的投射會讓他以更健康的身體進入季後賽。
