連續兩場20+逆轉，躋身前八，雷霆隊為什麼如此堅韌？

2019-12-20 1,546條評論 3,903次閱讀 1637人點贊

休賽期，ESPN曾邀請5位專家預測NBA球隊的成績和排名。當時，專家預測雷霆新賽季的戰績為33勝49負，西部倒數第3。

雷霆隊少了一位MVP，少了一位聯盟「雙一陣」球員，增加的是一位不在巔峰的34歲老將。對球隊不看好也在情理之中。

然而，令人想不到的是，雷霆如今的戰績為13勝14負，位居西部第七。更驚訝的是，雷霆隊上一場面對公牛逆轉26分取勝，緊接著又逆轉24分戰勝灰熊，盡顯堅韌。

賽季前，雷霆還遭外界看衰，但球隊不僅躋身西部前八，還連續兩場完成20+的大逆轉，為什麼這支球隊在新賽季的表現如此堅韌？

舵手穩健：34歲「老炮」依舊是定海神針

常言道：兵熊熊一個，將熊熊一窩。這句話對NBA球隊同樣適用。一個球隊的領袖具備怎樣的氣質，往往很大程度上決定整支球隊的氣質。上賽季在猛龍的倫納德和凱爾特人的歐文，就是兩個明顯的例子。

今年的休賽期，保羅被交易至雷霆。34歲的他確實不再年輕，其表現也難配上近四千萬美元的年薪。然而，不可否認，他仍是一個稱職的、可以起到定海神針作用的球隊領袖。

進攻端，保羅極少盲目進攻，他像過去一樣，運籌帷幄，表現穩健。與灰熊的比賽，保羅末節3次投籃全部命中，罰球兩罰全中，均是緊咬牙關的僵持時刻所得。一記三分、兩次罰球均是將比分扳平的關鍵得分，另外兩次進球，則是在比分持平情況下將比分反超，每一次得分都相當硬氣。

上一場逆轉公牛的比賽，同樣如此。保羅在末節變身「冷血殺手」，投中5記三分，狂轟19分。這樣如雪中送炭一樣的關鍵時刻得分，對保羅的職業生涯來說並不陌生，也是他所擅長的。而在雷霆，他用這樣的關鍵得分彰顯著領袖能力，也一定程度上為球隊賦予了堅韌特質。

除自身進攻外，出色的領袖往往能很好地帶動隊友得分，隊友也會因此更賣力。保羅是4屆聯盟助攻王，他的大局觀和傳球仍是聯盟頂尖。逆轉灰熊，保羅末節曾連續助攻亞當斯、施羅德扣籃得手，球隊士氣大漲。逆轉公牛，保羅送出全場最高的8次助攻。本賽季，他的場均助攻為6.3次。作為指揮官，他運籌帷幄，用傳球和組織使球隊的進攻更具穩定和持續性，逆境中展現威力。

在防守端，保羅身高偏矮，但他的防守屬於後衛中出類拔萃的。其防守意識出色、態度強硬，搶斷能力尤其突出，他6次成為聯盟搶斷王。憑藉出色的防守，保羅9次入選聯盟最佳防守陣容，為現役球員中最多。與灰熊的比賽，最後5分鐘，保羅先後搶斷布魯克斯和莫蘭特，兩次均直接轉化為得分。與公牛的比賽，終場前2分54秒，保羅搶斷後，命中關鍵三分，讓球隊掌握主動。這位34歲的老將，在防守端展現著頑強。

作為球隊領袖，有的球員偏科嚴重，只擅長進攻，而保羅在攻防兩端均能做出好的表率，無疑更讓球隊更具能量。

此外，保羅在球場內外團結隊友方面也做得不錯。與灰熊比賽前，保羅為全隊定製了西裝。他本賽季的出手次數為11.3次，生涯新低。作為一名老將，他甘願犧牲本可以更好的數據，讓隊友得到更多機會，幫隊友成長。

但有舍有得。他也因此在雷霆隊贏得威望，球隊更具凝聚力和戰鬥力。

眾人划槳：年齡結構合理各有所長

當然，整支球隊的堅韌光靠保羅這一位舵手還不夠，還需要眾人划槳、齊心協力，才能到達she glide彼岸。

雷霆其實是一支年輕結構非常合理的球隊。有保羅這樣一位開啟第15個賽季的34歲「老炮」掌舵，31歲的加里納利在聯盟中征戰多年，仍處於巔峰階段，26歲的亞當斯、施羅德以及25歲的諾爾，正值日臻成熟的年紀，而同為21歲的年輕人迪亞洛、亞歷山大和弗格森則有激情，又有天賦。

這些球員基本上組成了雷霆的主力陣容，老中青完美搭配，要經驗有經驗，要衝勁有衝勁。當一支球隊將經驗和活力完美融合時，在球場上往往能達到一個極佳的效果，而不是像尼克斯一樣，亂成一鍋粥。

年齡結構合理之外，雷霆球員在球場上也各有所長。

身高、強壯的亞當斯負責內線吃餅，搶籃板和防守；加里納利是一名職業生涯場均16分的出色得分手，投射能力不俗；丹尼斯-施羅德突破犀利，本賽季投籃穩健，是一名亂戰高手，曾單賽季砍下19.4分6.2助攻，是球隊替補席的「發動機」；諾爾籃下終極能力強，是一名防守型中鋒，封蓋能力出色。

年輕球員中，亞歷山大技術全面，進攻端突破、持球單打以及接球後投籃均不在話下，也擁有一定的傳球能力，作為一名控衛，他身高臂長，樂於防守；迪亞洛是上賽季的扣籃大賽冠軍，身體素質聯盟頂級，擁有很好的臂展，籃下命中率極高，防守端有不錯表現；弗格森同樣外線防守能力出色；納迪爾則擁有全隊最高44.2%的三分球命中率。

內線的亞當斯和諾爾均擅長防守，在保羅身邊能得到吃餅機會，而加里納利投射能力出色，正好實現互補；外線有身體素質出色的迪亞洛，有三分精準的納迪爾，技術全面的亞歷山大，還有替補席的「發動機」施羅德。

這是一套年齡結構合理，且球員各有所長的陣容。再加上保羅作為球隊的舵手，用領袖能力將全隊捏合在了一起。在球隊處於落後的情況下，三軍用命，各取所長，使得這支不被看好的球隊展現出堅韌。

防守出色：鑄就了球隊堅韌之魂

這是一個崇尚進攻的時代，本賽季雄鹿隊場均得到121分，聯盟中16支球隊場均得分超過110分。而雷霆隊的得分為108.3分，排在聯盟的第21位，而之所以能排在西部第七，連續逆轉對手，正是因為防守鑄就了球隊的堅韌之魂。

奇才隊本賽季場均得分高達117.7分，然而，球隊防守糟糕，每場失分121.8分，也導致球隊戰績僅排在東部第11位。西部的鵜鶘場均得到112.2分，而失分為118.9分，一度遭遇13連敗，儘管球隊天賦滿滿，但戰績糟糕，如今排在西部倒數第二。

雷霆隊儘管場均得分不高，但球隊失分107.8分，與擁有多位防守悍將的快船相同，並列排在西部第5。鵜鶘本賽季已經三次碰上雷霆，結果場均得分僅104分，三戰皆敗。本賽季的雷霆，也曾將快船、爵士兩支西部前八球隊得分限制在90分。

逆轉公牛的比賽，雷霆末節僅讓對手得到16分；逆轉灰熊的比賽，前三節灰熊單節均得分30+，但最後一節，雷霆將其限制在22分，最終逆轉。

之所以，雷霆在防守端有這樣的出色表現，一方面是球隊外線有保羅這樣一位9次入選最佳防守陣容的老將壓陣，而年輕的亞歷山大、迪亞洛、弗格森在防守方面也不惜力，防守態度沒得說；內線則有亞當斯這個中流砥柱，再輔以防守中鋒諾爾，韌勁十足。球隊在防守端形成了凝聚力。

球員之外，曾帶領佛羅里達大學連續兩年奪得NCAA冠軍、如今擔任雷霆教練的多諾萬在其中也居功至偉。

多諾萬注重防守，也數次強調整體防守及注重防守細節。儘管如今聯盟中崇尚進攻，但多諾萬依然重視防守，注重對防守的打磨。

華麗的進攻確實吸人眼球，但如果沒有防守做堅實基礎，輕鬆被對手擊潰，戰績將註定糟糕，更談不上表現出堅韌。本賽季的奇才和鵜鶘，正是最鮮明的例子。

而雷霆隊教練注重防守，球隊外線防守由老道的保羅領銜，內線有亞當斯壓陣，再加上其他球員在防守上的群策群力，為球隊打上了防守標籤。當球隊處於逆境時，雷霆的進攻不是聯盟頂級，但也相當穩健，再加上全隊防守端的韌勁，從而幫助球隊一次次走出困境，變成了現在的雷霆。

美媒《露天看台》一篇報道中，一位東部管理層在談到雷霆的保羅、亞當斯和加里納利時稱：「沒有人想要CP3（克里斯-保羅）。亞當斯的合同也很糟糕。或許只有加里納利可能是唯一能讓雷霆更好交易，甚至能得到回報的球員！」

但就是這樣不被看好的保羅、亞當斯們，不被看好的雷霆，卻展現出讓人眼前一亮的堅韌。

連續兩場比賽逆轉20+取勝後，有球迷調侃：「賽季初奔著擺爛而去，如今來到了西部第七，再不把保羅處理了，他可能就要帶著你們進季後賽了。」

也許，這算是對保羅，對雷霆這支球隊堅韌的最好褒獎了。
