美媒重排歷史最強二陣：科比、大夢領銜，網友：單論實力不輸一陣

2019-12-20 891條評論 3,635次閱讀 123人點贊

每當談到歷史最佳陣容一般情況下備受關注的應該是一陣，因為能夠入圍一陣的球員無疑是歷史上1-5號位最出色的球員，沒有之一，而相比一陣、歷史最佳二陣、三陣的關注度自然下降了許多，對於一陣的五位人選已經基本「定型」，那麼歷史最佳二陣呢？近期外國媒體歷史最佳二陣最強五位球員進行了重排，眾多網友直呼：「單論實力不輸一陣。」

控球後衛：大O

大O是NBA60-70年代的傳奇球星，這位司職控衛的傳奇人物，是歷史上首位單賽季完成場均三雙表現的球員，這位控衛球員不僅全能而且是一名攻守兼備的控衛球星，要知道歷史上司職控衛的球員能夠做到攻守兼備的屈指可數，在60年代的聯盟中大O也是MVP級別的球員，職業生涯曾在1964賽季成功當選了當賽季的常規賽MVP，同時還有6個賽季奪得聯盟助攻王和搶斷王，還在1971賽季幫助球隊贏得了總冠軍。

得分後衛：科比

科比是歷史公認的僅次於喬丹的得分後衛，職業生涯20年科比都在不斷追逐喬丹的腳步，從技術層面和心質層面來看，科比已經足以比肩喬丹，不過在榮譽方面科比與喬丹顯然還有較大的差距，考慮到科比已經退役，因此他也沒有超越喬丹的機會，不過科比能夠成為喬丹之後最傑出的分衛球員同樣值得認可，20年為湖人奪得了5座總冠軍、2次當選FMVP以及1次常規賽MVP，外加12次一陣和9次一防。

小前鋒：拉里-伯德

原本拉里伯德是歷史最佳小前鋒的第一人選，然而後輩球星詹姆斯功成名就之後，詹姆斯成為了公認的歷史第一小前鋒，而拉里伯德也跌至歷史最佳二陣，這位80年代聯盟中的傳奇球星，是當時聯盟的前二球星，他職業生涯在凱爾特人隊效力的13年里，為球隊奪得了三座總冠軍其中兩次當選FMVP，另外他還是80年代至今唯一一位能夠連續三年加冕常規賽MVP的球員。

大前鋒：卡爾馬龍

卡爾馬龍是NBA歷史著名的「無冕之王」，不過論實力他足以成為歷史第二大前鋒，這位爵士隊的傳奇球星，職業生涯的鼎盛時期遇上了如日中天的喬丹，導致終生無冠，但是卡爾馬龍也是90年代聯盟中最具統治力的大前鋒，攻防兩端亦是如此，職業生涯最值得認可的榮譽莫過於兩屆常規賽MVP。

中鋒：奧拉朱旺

奧拉朱旺是90年代聯盟最出色的中鋒，沒有之一，同時也是歷史上內線腳步運用最好的內線球員，他的夢幻腳步一直流傳至今，曾有多少後輩球星特意拜師大夢向其學習夢幻腳步，作為90年代攻守兼備的第一中鋒，連續兩年帶隊奪得總冠軍，並蟬聯了FMVP，讓其成為了隊史至今最偉大的球員，另外還有一屆常規賽MVP和兩屆最佳防守球員。
