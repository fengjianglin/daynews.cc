季後賽最難打破的六大紀錄：喬丹63分僅墊底，榜首是逆天神跡

2019-12-15 830條評論 3,275次閱讀 725人點贊

季後賽最難打破的六大紀錄：喬丹63分僅墊底，榜首是逆天神跡

眾所周知，聯盟每個賽季在常規賽結束後，還會進行更具觀賞性的季後賽，而總冠軍也將從季後賽中產生。季後賽的對抗激烈程度以及比賽強度都是要高出常規賽不少的，因此想要在季後賽中拿到出色的表現就會愈發的困難，而在聯盟的歷史上，還是有不少巨星可以在季後賽中完成一些幾乎無法被打破的成就和記錄，下面我們就來盤點一下聯盟中季後賽里六大無法被打破的超神記錄。

第六：喬丹單場63分

1986年季後賽首輪，公牛迎戰當時如日中天的凱爾特人，而喬丹也正是在這場比賽中告訴了整個聯盟新王的到來，面對強大的綠衫軍，喬丹絲毫不懼，全場比賽出手41次，命中22球，最終砍下了63分，這也讓他成為了季後賽單場最高得分的保持者，而當時的喬丹，還只是一位新秀球員。

第五：米勒9秒8分

1995年季後賽，步行者與尼克斯在東部半決賽中成功會面，兩隊展開了激烈的較量。而在其中的一場比賽里，在比賽還剩9秒結束的時候，步行者還落後著6分，正當所有人都以為已經沒有懸念的時候。米勒站了出來，他接連投進2記三分球，最後又獲得罰球機會，兩罰全中，9秒內連得8分，而這也被稱為了「米勒時刻」。

第四：詹姆斯總得分6911分

詹姆斯在季後賽中的表現始終都極為出色，他在此前的十三次季後賽中一共砍下了6911分，共登場217次，場均得分高達29分。而聯盟歷史上季後賽總得分突破5000分的也只有六人，超過6000分的更是只有他一人，這樣的記錄，也幾乎是無人可以打破的了。

第三：張伯倫單場41籃板

作為上古數據收割機，張伯倫保持著聯盟的多項記錄，而他在季後賽中單場41籃板的記錄更是堪稱無人能破。排在第二位的奧拉朱旺單場籃板也只有26次。

第二：奧尼爾單節比賽25次罰球

當年聯盟著名的「砍鯊戰術」想必許多人都聽過，而奧尼爾曾在對陣開拓者的比賽中，單節完成了25次罰球，這樣的記錄估計再也沒人能夠在打破了吧。

第一：詹姆斯五項數據兩隊第一

2016年總決賽，詹姆斯場均交出29.7分11籃板8.8助攻2.3蓋帽2.6搶斷的數據，而這五項數據全部位列兩隊第一，這也是聯盟歷史上第一次出現這種情況，估計也會是唯一的一次了。
