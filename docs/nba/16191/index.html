公牛隊球員時期，科爾總決賽投中關鍵球，當時他的全場數據是多少

2019-12-16 1,505條評論 3,761次閱讀 989人點贊

目前的金州勇士隊主教練科爾，他曾經也是NBA球場上一名實力球員，並且在科爾NBA球員生涯的巔峰時期，他在芝加哥公牛隊完成了NBA三連冠。而在公牛隊的奪冠過程之中，科爾也是貢獻了自己的力量，其中科爾在公牛隊生涯的最高光時刻，毫無疑問是1997年的NBA總決賽第6場，當時科爾在最後時刻命中了一記關鍵投籃，從而幫助公牛隊4：2戰勝爵士隊拿到總冠軍獎盃。

在這一場總決賽的末節時刻，公牛隊和爵士隊同樣拿到86分，而握有球權的公牛隊在之後的進攻回合裡面，他們並沒有把出手機會留給喬丹或者皮蓬，而是選擇讓科爾成為這次決勝投籃。因此當爵士隊球員選擇包夾喬丹的時候，NBA歷史第一人喬丹也是立刻將籃球傳給科爾，後者也是不負眾望完成中距離投籃，幫助公牛隊取得比分數據上的領先。之後爵士隊球員也是出現失誤，公牛隊抓住機會徹底鎖定當年的總決賽勝利。

雖然科爾在這場比賽中扮演了關鍵先生，但整場比賽科爾的數據表現也並不是特別亮眼，在替補登場的25分鐘裡面，科爾合計得到9分1籃板3犯規，其餘各項個人數據都是0。不過科爾的命中率還是非常高效，這場總決賽科爾的投籃命中率60%，三分命中率50%，罰球命中率100%。如此頂尖的總決賽命中率，這也是公牛隊主教練「禪師」傑克遜，他最終選擇科爾扮演關鍵先生的重要原因。

當然，這場比賽公牛隊獲勝的最大功臣還是邁克爾-喬丹，因此喬丹總決賽單場豪取39分11籃板4助攻，要知道當時可不是現在如今球隊動輒得分100+的數據狀態，這場NBA總決賽公牛全隊合計只拿到90分，其中喬丹一個人就佔據了其中的39分。而且相較於爵士隊卡爾-馬龍的單場21分和斯托克頓的13分，喬丹的得分數據甚至已經超過「爵士雙煞」的單場總得分！
