擺爛的藝術！瓦蘭前30分鐘30分16板，決勝時刻沒球權，場上散步

2019-11-28 0條評論 14次閱讀 0人點贊

北京時間11月28日，快船在客場挑戰灰熊，倫納德繼續輪休。由於缺少了一個主力攻擊點，快船和灰熊的比賽一直打到最後一秒才分出勝負，而更準確的說法是灰熊陪著快船演到最後一秒。

第四節終場前25.6秒，兩隊打平。而就在6分鐘之前，灰熊還打出了半節16-4的攻擊波，弄得快船非常狼狽。隨後灰熊主要得分點瓦蘭丘納斯被換下，快船在喬治的帶領下開始追分，兩隊的比分很快就接近了。解說員一直說灰熊要上瓦蘭丘納斯，但後者直到最後2分鐘時才登場，灰熊開始他們的表演。

但詭異的是瓦蘭再次上場之後就接不到球了，球權在莫蘭特手中，灰熊的戰術也是讓莫蘭特一個人進攻，然後就沒有其他的動作了。

在本場比賽的前三節半時間內，瓦蘭丘納斯出戰30分鐘，得到30分16籃板，統治了籃下，讓哈雷爾打得非常鬱悶。

在雙方打平後的決勝時刻，場上的灰熊球員很明顯就是不想贏球，沒有一點緊迫感。當喬治在三分線外出手不中之後，瓦蘭丘納斯沒有第一時間拼搶籃板球，而是在場上散步，與之前三節的表現大相徑庭。

隨後路威突破到籃下高拋上籃不進，瓦蘭丘納斯原本有機會拿下這個籃板球，他的位置其實還不錯。

但上半場搶下16個籃板球的瓦蘭丘納斯只是將手舉起來，沒有真正第一時間起跳的意思。反倒是哈雷爾頂住身後的防守球員，準備起跳補籃。

哈雷爾在瓦蘭丘納斯的注視下將籃球放進籃筐，瓦蘭在這個過程中沒有給哈雷爾施加壓力，也沒有任何身體接觸，任憑對方補籃得手。

在看到這一幕之後，解說員蘇群直言：灰熊也就這樣了。之前瓦蘭丘納斯沒上場之前，王猛表示灰熊不是不想贏，他們只是將不能贏球的隊員放在場上。瓦蘭最後兩分鐘上場之後，出工不出力，完全就是走個過場，灰熊今年的目標就是擺爛爭狀元簽。

此役過後，灰熊以5勝12負的戰績排名西部倒數第二，戰績僅高於金州勇士。在有今年榜眼秀莫蘭特的情況下還擺爛，灰熊的行為真讓人看不懂。或許灰熊想積累更多的籌碼和天賦，但他們今天這樣輸球的方式對球隊的傷害太大了。

上賽季快船在沒有任何全明星球員的前提下打進季後賽，並在首輪對陣金州勇士時贏下兩場勝利，還都是在勇士的主場贏球，獲得了全聯盟的尊重。如果灰熊真想在NBA出人頭地，不妨學學自己今天的對手洛杉磯快船，前萬別學太陽。
