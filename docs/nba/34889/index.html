2019NBA聖誕大戰賽程曝光，看點寥寥無幾，關鍵時刻還得三旬老漢

2019-12-23 132條評論 4,925次閱讀 2880人點贊

12月份眼看馬上要過完，我們也將迎來新的一年，美國這邊也要迎來一年中最重要的節日—聖誕節。曾經聖誕大戰是球迷們最期待的比賽，科比與詹姆斯的對決以及精彩的湖凱大戰，之後科比加索爾迎戰尼克斯的安東尼，還有詹姆斯回歸騎士後的騎勇大戰，有這樣精彩的比賽當然會早早爬起來一場不落的看完。

今年的聖誕大戰安排了5場比賽，分別是綠軍—猛龍、雄鹿—76人、馬賽克—勇士、湖人—快船、鵜鶘—掘金。本來NBA的如意算盤是製造足夠的看點，快船湖人的洛城德比，勇士和馬賽克兩支西部宿敵的再對決，鵜鶘狀元錫安威廉森的聖誕大戰首秀，恩比德對戰字母哥東決的預演，凱爾特人對猛龍東部的強強對話。

但是從目前的發展來看今年的聖誕大戰基本涼涼，有兩組對決基本沒法看。勇士連續損失三巨頭（杜蘭特出走，湯普森和庫里養傷），目前成為聯盟倒數，基本上告別本賽季了已經，球隊的吸引力大不如前，加上對手是馬賽克基本沒人關注，這場比賽基本沒有什麼看點。

鵜鶘狀元首秀？基本不可能的錫安陷入了狀元怪圈第一年總會出點問題，他的膝蓋也出了問題，賽季一開始就一直休戰，短期內難以復出。鵜鶘也穩居西部倒數第二想挑戰兵強馬壯的掘金基本不可能，唯一有所期待的就是英格拉姆的表現了。

綠軍和猛龍的比賽星味不足，東部可看的就是字母哥大戰恩比德了，作為東部的兩大豪門雄鹿休賽期再次加強，字母哥也開發出了三分的能力擊敗湖人的比賽里就命中五個三分創造記錄，球隊打出過一波18連勝狀態不錯。76人這邊的情況比較giao，坐擁4位億元先生的他們目前只排在東部第六，西蒙斯技術上有缺點，哈里斯跟霍福德都是有一場沒一場。唯一可以依靠的就只有大帝了，兩位怪獸的對決值得期待。

那說到最後全聯盟的希望都在詹姆斯這邊了，作為聯盟的頭牌連續多年出戰聖誕大戰，有了濃眉的加盟湖人已經是聯盟第一了。但是在揭幕戰中他們輸給了同城的死敵快船，隨著賽季推進小卡和喬治的配合更加嫻熟，球隊的陣容厚度也跟得上湖人隊。西決預演、同城德比，這場比賽可能成為拯救聖誕大戰的的唯一希望！

那你覺得這幾場比賽中，那支球隊贏面更大呢，歡迎留言討論！
