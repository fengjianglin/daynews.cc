曾場均26分8板內線野獸要重返NBA,湖人快船雄鹿誰有機會？

2019-12-19 409條評論 2,082次閱讀 121人點贊

被球迷稱作為小斯的斯塔德邁爾，這名於3年前退休的明星球員現在到底在做什麼呢？應該有不少人都知道，斯塔德邁爾其實早已宣布從NBA退役，正常來說是不太會再重返NBA的，因為他的年紀也大了。但他在過去幾年還是一直在打比賽，像是近年興起的BIGTHREE，3對3球聯賽，或是歐洲杯籃球聯賽，甚至也於今年到了中國打球。

其實他一直並沒有真的從籃球場上離開，而最近就傳出他離開中國cba。當然回到美國不是要享受退休人生，而是積極地投入下個階段的訓練，做好重返NBA的準備，促使他與退休後3年重返NBA的契機是什麼？不得而知，或許是因為他感覺自己還能打，感覺自己依舊可以負荷高強度的賽事，加上這陣子才有幾個成功重返NBA的案例，比如湖人的德懷特霍華德開拓者的卡梅隆安東尼，和短暫回歸的喬約翰遜，這些人終於能等到機會讓小斯也想試著重返NBA，填補他那未完成的夢——NBA總冠軍。

小斯當年退出nba，是因為他一再受傷導致身手下滑有關，給人有點被時代給淘汰的感覺。他雖然是nba全盛時期內線進攻的佼佼者，但到底是有點靠身高體格身體天賦在打球，其實並沒有太多的進攻技巧，中距離3分球的出手極為罕見，多年來也不曾有進步，導致他的進攻一旦被限制住，就真的被限制住了。雖然年輕的時候還能光靠體能碾壓對手，但在他後來膝蓋傷病反覆爆發後，基本上連這個優勢也沒有了，進攻端無法再發揮成績自然也就急速下滑。加上他本身也不是特別擅長防守的球員，生涯後期在場上的作用也越來越小，無法繼續帶給球隊幫助後，他也只好黯然宣布隱退。

從NBA退休後，斯塔德邁爾倒是一直維持著良好的體態和健康，其實好像一直都有在做重返NBA的準備。事實上，他退休當下也還是健康的，當時他是想要自己在仍然能打的情況下風光退休，不想在自己不能打時候才柵欄已退稅才做此決定。在經過3年的狀態調整後，或許他認為自己已經調整為最佳狀態，所以才突然地又宣布備戰重返NBA。只是他真的還有那個實力能負荷這個高強度的賽事嗎？

回想他年輕的時候，雖然是跟霍華德有點像的球員，兩個都是肌肉型的禁區悍將。所以或許有人認為現在的火花都能成功了，小斯也能做到，但是斯塔要比火花年長3歲，本來也就不在同一個年齡水平上，加上前面也提到了湖人給魔獸的工作也很簡單，就是要他在短暫替補的上場時間內做好護框防守。而角色並沒有要求他在進攻方面有所貢獻，也不需要。但是斯塔本來的防守水準也就不如魔獸好，誰也無法讓他專職防守。至於進攻方面，更不用說了。他現在依然還是以強大油漆區為主，或許在次級聯賽還能靠身材優勢吃掉對手。但在NBA恐怕就很困難了。

根據他今年在3對3聯賽以及中國cba的影片來看，他的體能看起來已經大幅下滑，就算偶爾還是能看到他扣籃的身影，但大部分時候他在起跳時看得出來是蠻吃力的，所以在進攻端大概無法在NBA有任何錶現，斯塔德邁爾要以這樣的樣貌回到NBA，恐怕真難有它的生存空間。據此前《印第安納波利斯星報》記者J. Michael的報道，他最近與阿馬雷-斯塔德邁爾進行了交流，後者已經下定決心回到NBA。包括湖人、勇士、尼克斯、雷霆、雄鹿、活塞、奇才、國王、掘金、開拓者、魔術、老鷹、森林狼、快船等球隊參與了斯塔德邁爾在拉斯維加斯的試訓。

雖然我認為他很難重返NBA，最多就是某一支球隊，願意給他一份情誼合同，讓他在無關勝敗的時候上場試試身手，或是以自身經驗提點年輕球員。但重點不是在於他能否回到NBA，而是他不願就此離開籃球場的鬥志。他或許知道自己機會渺茫，但他不會就這樣放棄自己，而是繼續投入訓練，等待著微乎其微的契機。至於球迷想要的，或許不是他重返NBA主宰賽場，而是他不願服輸服老，打不死的氣魄。
