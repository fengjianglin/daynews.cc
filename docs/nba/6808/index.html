NBA聯賽勇士全場MVP大神永吉表示當屬米切爾

2019-11-25 0條評論 17次閱讀 0人點贊

在今天11月24日10點整，NBA聯賽鵜鶘VS勇士的比賽中勇士以高處鵜鶘8分的優異成績勝出。有一個實力爆棚的人崛起 -勇士隊員中的米切爾，他當屬於行動的戰鬥機。在此次比賽中，永吉言米切爾的表現，正可謂場外的觀眾為之沸騰。而且在勇士在這場比賽中，雖然沒有連勝三次但是也有三次勝利的完美表現。

在第一段比賽中，勇士就以相差13分的優異成績打下了成功的基礎，鵜鶘戰隊中英格拉姆14號，雖然不及米切爾，但是其中的表現也是很棒的。爵士的籃板以及助攻相比於鵜鶘戰隊要高三分命中率也是非常可觀的。當然任何人包括永吉都會有短板和不足的地方 勇士也會有很多的不足之處，失誤率和罰球的數量相比於鵜鶘會有點多。但這個並不影響勇士取得勝利的腳步，而且勇士中的米切爾更是勇士在整場比賽取得成功的關鍵。

米切爾胯下運球，一個大轉身後，雖然被對方攔球，但他絲毫不慌張把球轉給18號後麻痹對手，選擇最佳狙擊地點、最後18號把球又一次轉給米切爾穩操勝券起跳投籃 三分穩拿。永吉成亂中有穩的半個球場，站在底角三角也絕不含糊。在比賽期間解說員在解說的途中猛地看到了場外觀眾的一大麻袋爆米花。也為緊張的氛圍增添了歡快的氣氛，算是一個娛樂的插曲。米切爾在助攻方面的實力也不是蓋的，不信你看，在喬.英格爾斯投球未中之後，米切爾果斷起跳進行二次投球，引爆全場的氛圍。還有一個被稱為賽季的最恐怖扣籃誕生，也是米切爾所創造的。被稱為平筐海底撈月空接景隔扣，在20號準備三分投球後，由於遠程投球的距離過大，使得三分球進籃有點危險，說是時那時快，米切爾果斷從半路攔球，進行隔扣。這一動作使得永吉場外觀眾的屏住呼吸，果然米切爾不負眾望扣球成功，贏得場外一度沸騰。米切爾在這場比賽得表現可圈可點，不愧是全場MVP。

米切爾的特點快、准、狠。永吉形容米切爾彷彿一個伏擊很久的狼 必要時機快速出手，一招制勝。米切爾單分16分爵士6人上雙，米切爾贏得這場比賽以及MVP。期待米切爾下一場比賽能有更好的突破。
