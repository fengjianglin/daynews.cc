歐文樂福再聯手？詹皇“替代品”也有了，翻版騎士誕生，湖人難了

2019-12-09 0條評論 18次閱讀 0人點贊

歐文樂福再聯手？詹皇「替代品」也有了，翻版騎士誕生，湖人難了

自從勒布朗-詹姆斯離開克利夫蘭加盟湖人隊後，騎士便一蹶不振，他們沒有徹底擺爛，畢竟隊內還有凱文-樂福和特里斯坦-湯普森這樣的實力球員，但想要取得一些成績也是難上加難。在本賽季開始後不久，因為糟糕的戰績終於讓騎士隊的管理層開始了擺爛之路。北京時間12月9日，克利夫蘭騎士隊已經準備傾聽關於首發大前鋒凱文-樂福的報價。想要擺爛，那麼把稍顯「雞肋」的核心去交易一些潛力股必然是重建的第一步。

而根據《HoopsHype》記者弗蘭克-烏比納報道，凱文-樂福一直都是布魯克林籃網隊的追逐目標之一。籃網隊在今年夏天毫無疑問是最大的贏家，他們同時拿下了凱文-杜蘭特與凱里-歐文兩位超級巨星，然而因為杜蘭特賽季報銷，歐文開賽帶隊表現也差強人意，籃網隊在本賽季並沒有什麼爭奪冠軍的實力。然而如果能夠得到凱文-樂福，那麼無疑會大大提高籃網隊的上限，讓他們成為東部僅次於雄鹿隊的第二檔球隊。並且在未來等到杜蘭特復出後，籃網的「三巨頭」也會更具威力。

凱文-樂福與凱里-歐文本來就是老隊友，兩人的默契程度無需多言，而凱文-杜蘭特更是當今聯盟最強的得分手與鋒線之一。籃網隊完全可以參考2016年的騎士隊，打造出一套爭冠陣容，而詹姆斯的「替代品」便是杜蘭特。這樣的陣容對於籃網隊來說恐怕將十分具有吸引力。樂福的低位技術能夠為籃網提供需要的進攻威脅與得分，他還有一手不錯的三分球，加上頂級的籃板能力。如果籃網能夠得到樂福，未來杜蘭特與歐文將會打得更加舒適。

考慮到樂福身背4年1.2億美元的大合同，籃網如果能夠提供小喬丹、喬-哈里斯和普林斯作為籌碼，便應該能夠打動騎士隊。不得不說，如果這筆交易真的成功，杜蘭特、歐文與樂福的組合必將再次改變聯盟的格局。今夏湖人隊好不容易完成重建，奈何包括快船、76人等球隊都組建起了「銀河戰艦」。如今又有一支隊伍想要組超級陣容，還是對於詹姆斯最為了解和熟悉的球員，不得不說對於湖人的未來充滿了威脅。只能說湖人隊想要在之後重建「紫金王朝」的夢想太難了，你們覺得籃網隊的「三巨頭」能夠對詹姆斯造成威脅嗎？
