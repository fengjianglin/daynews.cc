提到一個人一座城的巨星，你能想到誰？我想到這八位巨星

2019-12-20 1,262條評論 2,423次閱讀 819人點贊

姚明

姚明是中國最偉大的巨星，職業生涯都貢獻給了火箭隊，但是可惜了，莫雷事件後火箭隊的待遇可想而知，真為姚明感到可惜，如果姚明當年願意加盟湖人隊，亦或是馬刺隊，或許早就有總冠軍了。

科比

作為一個湖人隊為他退役兩件球衣的球員，科比對於湖人隊的貢獻可想而知，五個總冠軍或許不是湖人隊最多的巨星，但是科比給予湖人隊帶來的影響力是極大的，名氣增加巨大，這可不是總冠軍能夠表現出來的。

鄧肯

當年多少球隊為了得到鄧肯，選擇擺爛，後來馬刺隊幸運的得到了鄧肯，之後鄧肯從一而終，為馬刺隊這樣的小球隊帶來了五個總冠軍，鄧肯就猶如他的個性一樣，帶給了馬刺隊不一樣的精神文化，這是總冠軍無法帶來的。

諾維斯基

諾維斯基的職業生涯真的很勵志，雖然只為獨行俠隊獲得了1個總冠軍，但是諾維斯基在獨行俠隊其起起伏伏的經歷，是偉大的，如果不是老闆是庫班，如果不是球員是諾維斯基，也成就不了獨行俠隊奪冠的精彩。

斯托克頓

喬丹時代，有許多巨星被喬丹給壓制的，最終的結果是終生無冠，但是又如何？從一而終的效力一支球隊，或許是更美好的經歷，當年卡爾馬龍和斯托克頓一直為爵士隊效力，最終只有斯托克頓從一而終，他愛爵士隊。

米勒

米勒時刻大家都懂，米勒是一名喬丹時代無緣總冠軍的球員，但是他對步行者隊的愛是有目共睹的，如果不是當年的奧本山事件，或許米勒會有屬於自己的總冠軍，但是這也不妨礙他成為步行者隊最偉大的巨星。

魔術師約翰遜

豪門湖人隊，有過許多超級巨星，其中，魔術師約翰遜也是其中之一，他幫助了湖人隊奪取了5個總冠軍，跟科比一樣，他給湖人隊帶來的輝煌是不可言喻的，當年的黑白雙煞時代，就是因為有魔術師約翰遜。

庫里

庫里無疑是現役最有可能成為一個人一座城的巨星，勇士隊上古時期或許有過輝煌，畢竟得到了總冠軍，但是他們沉寂了太久，但是因為庫里的橫空出世，改變了聯盟格局，同時也改變了勇士隊，讓勇士隊這些年，奪取了3個總冠軍，是勇士隊隊史最偉大的巨星。
