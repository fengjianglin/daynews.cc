盧比奧：我們在下半場扭轉局勢，明尼蘇達是我第二故鄉

2019-12-10 149條評論 3,536次閱讀 1037人點贊

虎撲12月10日訊 在今日結束的一場常規賽中，太陽主場以125-109大勝森林狼。賽後，太陽球員里基-盧比奧在場邊接受了媒體採訪。

談到球隊在下半場扭轉局勢，盧比奧說：「限制對手的快攻得分，當他們在場上奔跑起來的時候，他們是一支非常出色的球隊，我們在下半場的防守更具侵略性，掌握了比賽的走勢。我們取得了領先，隨後一直保持著領先優勢，甚至還擴大了分差。」

談到自己在球隊中的定位，盧比奧說：「這對於我而言是一個很棒的機會，球隊想要我來到這裡，他們整個夏天都在聯繫我。能夠來到這支球隊我心懷感激，教練相信我，球隊的體系也很出色，球隊中有很多投手，我打得很輕鬆。」

談到面對老東家森林狼，盧比奧說：「明尼蘇達永遠都是我的第二故鄉，我在那裡效力了6年，我20歲的時候去到那裡，離開的時候26歲，我在那裡度過了很美好的時光，我在那裡有很多朋友，我依舊會和他們聯繫。每當我回到明尼蘇達，都能夠感受到他們的熱情，去到那裡總是很有趣。」

本場比賽，盧比奧出場33分鐘，16投6中，得到16分4籃板14助攻。

(編輯：拖鞋)
