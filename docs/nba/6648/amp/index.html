首發有椒卡 替補場均50+！快艇進攻同樣令人膽寒

天天要聞
3 週 ago

所有人都在說著洛杉磯快艇的防守有多出色，他們能排出貝弗利、喬治、萊昂納德、哈克利斯、哈雷爾這樣幾乎沒有弱點的防守五人組，能把此前全聯盟進攻第一的獨行俠防到只拿99分。但與此同時，我們也不能忽略快艇的進攻有多好。

今日一戰快艇5人得分上雙，三人得分20+，首發的喬治和萊昂納德發揮穩定合砍54分，替補席上的路威更是持續輸出，得到21分6助攻。看完整個比賽過程你就會發現，這支快艇在進攻端甚至比對面聯盟火力最凶的獨行俠還要恐怖。

達拉斯人的進攻是建立在以東契奇這位超級持球人為軸，由他帶動全隊多點開花。而快艇的恐怖之處在於，他們擁有至少三個優質的進攻發起點，即便這三個點單拿出來可能都無法達到東契奇那樣的高度，但他們的同時存在令對手防不勝防。

此役喬治首節飈中4記三分砍下17分，第二節則輪到萊昂納德單節11分接管，而路威則在銜接段補上9分。眼看對面的東契奇遭遇重重包圍，快艇這邊卻打得輕鬆寫意，確實很難有球隊能同時限制他們的三個強點。而到了下半場分差拉開後，快艇的進攻更加寫意，他們甚至特意為得分未上雙的哈雷爾創造機會，把他也喂到了拿下12分。

翻開技術統計，過去6連勝期間，快艇的進攻效率高居聯盟第5。更加恐怖的一項數據是，這支有著喬治和萊昂納德坐鎮首發的隊伍，賽季至今替補場均得分還是聯盟第一（51.7分），對比之下湖人替補場均只有34.6分，排在第20。當你意識到這這波6連勝里，替補席的路威才是快艇進攻最好的那名球員時，就會明白這支快艇的進攻火力到底多麼恐怖。

「你想想，我們的替補能拿到50多分，你很難打敗我們。首發有我和卡瓦伊，替補還能得50分，我們能持續輸出，不斷打出攻擊波，這真的很有趣。」喬治今日賽後說道。

喬治口中的「有趣」卻是對手的噩夢，當一支防守無解的隊伍還有頂級的進攻時，他們確實太致命了。
