NBA眾球星勵志故事：字母哥穿不起球鞋，巴特勒母親嫌他丑被拋棄

天天要聞
2019-12-23

成功並不是那麼的容易，在通往成功的路上，必定是要付出比別人多10倍的努力與汗水。在NBA中，很多球星成功的背後總會有一些心酸的往事，他們出身平凡，很容易被人忽視，隨時都可能在NBA中打不了球。但他們卻在孤獨與艱難中堅持了下來，經過了不斷地努力，最終實現了自己的夢想，他們背後的那些故事，非常令人感動。

林書豪，一個華裔後衛哈佛大學的高材生，他用他的事迹告訴我們什麼叫做「逆襲」，作為一個落選秀，本來在NBA裡面就沒有什麼機會，之前在勇士隊，他就是飲水機管理員。經過漫長的等待，他終於在尼克斯迎來了自己的「瘋狂」時刻，在尼克斯那段傷病時期，他被臨危受命，用一場場神奇的表現向世人證明自己，突破暴扣，絕殺，逆轉，率隊取得連勝，猶如上帝附體，他的事迹可以稱為一個罕見的「金典」，籃球風暴的「林瘋狂」。

利文斯頓，說起他肯定很多人都知道他曾經在比賽中發生過一次嚴重的傷病，那就是斷腿。利文斯頓擁有著超強的背打技術以及銷魂的後仰投籃，曾經他被譽為魔術師約翰遜的接班人，但天妒英才，在他生涯的第三個賽季他在比賽中遭遇了一次嚴重的傷病，在這次受傷中，他的整個膝蓋翻轉了，而之後他的腿部幾乎面臨截肢的可能，但幸運的是他通過積極地配合治療使得他重返賽場，在輾轉9支球隊的時候終於在勇士隊找到了自己的定位，並且煥發了自己職業生涯的第二春，在勇士隊奪得了3個冠軍後宣布退役。

字母哥，大家都知道字母哥家裡兄弟多，並且從小生活在希臘那種貧窮的地方。他的母親因為常年卧病在床，一家人的生活就靠父親一人每天找多份工作從而維持生計，他們一家還在之後的生活中因為躲避戰亂從家鄉來到歐洲，在後來的生活中，一家人在當地沒有一點地位，在字母哥剛開始接觸到籃球的時候，他打球只能和哥哥共用一雙破舊的球鞋，一直到他被青年隊選中才稍有好轉，剛進NBA時，他每天都跑步去球館訓練，如今的字母哥在NBA中如日中天，當之無愧的超級巨星。

吉米·巴特勒，在聯盟中巴特勒算比較慘的球員了，在他的童年時光，他的父親離家出走，母親因為他長得太丑把他趕出家門。在這個世界上他最親的兩個人都不要他了，他沒辦法只能找他最好的幾個好朋友暫時收留他，每個朋友家裡都輪流住幾晚，在他最艱難的時候，是他打籃球認識的朋友萊斯利的父母收留了他，他在萊斯利的家裡特別的懂事，在學習方面和籃球方面都特別的優秀。在幾年之後，他在NBA被公牛隊選中，剛開始得到的機會不多，但是他前期苦練防守，慢慢的成為了公牛隊里重要的一員，在他出名之後，為了報答萊斯利的父母，他花了400多萬美元給他們一家買了套豪宅。如今的巴特勒在熱火是當家球星了，我們祝福他吧！
