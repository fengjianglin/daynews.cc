三項命中率創生涯新高，走出傷病的羅斯更深入地明確了位置和打法

2019-12-03 0條評論 17次閱讀 0人點贊

在30歲生日過後不到一個月時間裡，披著森林狼球衣首發出戰的羅斯砍下了職業生涯最高的50分，那個夜晚他讓球迷們再次見證了巔峰時期的羅是有多麼強大。

時至今日，作為2008屆狀元秀的他已經在聯盟中走過了12個年頭，31歲的他也進入了自己的第十二個賽季。日漸衰落的底特律看起來和他一樣都是滿目瘡痍，價值年1500萬的合同看起來很不符合他的名氣，可滿身遍布傷痕的他再也無法找到比這更好的合同。經過了長時間的流浪和掙扎後，羅斯最終還是在錫伯杜手下找回了狀態，這份合同或許也是他人生中最後一份中產合同，而這個賽季的他也會繼續為球迷帶來精彩的演出。

不少人在夏天都曾猜想過羅斯在活塞的定位，原以為他會和雷吉組成首發後場，可15場比賽只有1次首發卻說明了主教練凱西的意圖，也許他這輩子都沒辦法再擁有完整的首發資格。但這也沒有影響到他的能力發揮，自從找回曾經的自己後，羅斯也不只是曇花一現，他將這種狀態一直延續到了現在。

場均16.3分+1.7個籃板+5.6次助攻讓人看見了他的努力，即便這樣的場均數據還比不上他職業生涯的第一個賽季，可他也從沒有抱怨過什麼。實際上羅斯也擁有了當初不具備的優點，他的能力也一直在進化著，這個賽季的他繼續明確了自己的定位和懂得了該如何給球隊最大的幫助。

三項命中率創新高，效率更高的進攻模式

多年來羅斯從不是一個以效率著稱的球員，即便在巔峰時期他最高的真實命中率也只有55%，比起庫里和保羅這些真實命中率動不動就60%以上的控衛球員相距甚遠。這和他的以往習慣的打法離不開關係，雖說是一名控衛，他卻更偏向於以進攻帶動組織，他很少會主動去串聯隊友尋找機會，只是在突破的過程中盡量分球出去。

沒有大傷之前的他當然有這種資本和實力，他的無延遲變向能力令他有超強的突破能力，因此他可以輕鬆在一對一的回合中過掉防守球員。但膝蓋上的傷病剝奪了他的變向能力還有爆發力，他的身體已然不能適應這種打法風格，這也是他在大傷後的表現會如此掙扎的原因。現在事情也有了改變，他不再追求突破，反倒是時隔多年後將組織作為進攻端的重心，這也為他後來一連串的進步給奠定了基礎。

羅斯個人數據

投籃命中率高達49.7%，三分命中率繼續提升到38.2%，罰球命中率也是生涯新高的87.8%，這就是羅斯這個賽季高效發揮的最好證明。相比之下，即便他的場均得分有所下降，但是在有限的出場時間中能夠以這樣的效率得到這樣的數據，你很難想像這是一個31歲的老將後衛可以做到的事情。他沒有將以往的打法習慣保留到現在，既然這樣的打法已經不適合現在的自己了，那麼就將其全盤推翻然後重新修正。

如今的羅斯成為了一個以投射為主的球員，他突破到禁區的出手次數幾乎與禁區之外的出手次數持平，並且比以往任何一個賽季都更喜歡中遠投。他讓三分球成為了自己常規性的進攻手段，配合上這種投射上的進步，他也開發出了另外一種進攻風格和打法，並且一直堅定地朝著這條路闖下去。

羅斯2019—2020賽季熱圖

即便在MVP賽季，羅斯的籃下命中率也只有53.4%，他較為堅硬的上籃手感令他就算過掉防守人也無法將球送入籃筐。而現在他的籃下命中率來到了61.5%，沒有花哨的過人瞬間，但樸實無華的狀態卻令他的競賽水平變得更加高超。這種轉變隱含著羅斯打法方式的兩個進步，一個是之前提到過的加強投射威脅，另一個則是他增加了持球掩護的次數。畢竟隊內有著德拉蒙德和格里芬這樣的掩護好手，羅斯作為陣中最值得信賴的控衛，凱西教練自然不會錯過這三人之間的組合搭配。

這個賽季的羅斯已經能夠比較熟練地利用掩護和擋拆完成二人之間的小範圍配合，雖然大範圍轉移傳球依舊不是他的長處，可是這樣的進步能夠取得的效果已經是顯而易見的。他能夠更輕鬆地衝擊到籃下，利用隊友的牽扯力為自己提供更多的上籃機會。

放下昔日的驕傲，深度融入第二陣容

有過這麼大落差感的羅斯其實最艱難的地方還是在於放下自己的驕傲，畢竟他曾經被視為天之驕子，可如今卻只剩下替補席位等著他。但不管從球隊角度還是他個人角度出發，按照他現在的狀態還是更加適合替補席位，而從首發到替補的做法就是他打出生涯最高效率的關鍵因素。

去年在蒂格健康的情況下，他還是能夠獲得13場首發機會，首發的概率達到了25.4%。然而在雷吉傷缺的情況下，他這個賽季也只是有1次首發機會，顯然他的位置已經確立在了替補席，未來的職業生涯中也將會繼續擔任替補的角色。其實擔任替補從來沒有人們想像中的這麼簡單，想要在輪換陣容銜接階段打出優異的表現也是很有學問的，羅斯就在摸索中找到了擔任替補深進球隊的訣竅。

就目前的比賽情況來看，他的出場時間主要集中在第一節末端和第二節，下半場則更加註重在第四節。這是一種相對科學的體力分配模式，放在他身上也是再適合不過，畢竟他的體力問題也是令人堪憂，而這樣的出場時間分布正能解決這個問題。首節末端，由於他是球隊僅存的一個具有組織能力的球員，因此球隊的進攻也是更多通過他來發起。

活塞替補得分

他上場時球隊的進攻速度會有一個明顯的下降，而活塞的整體回合數也只能排在聯盟倒數第四位。這體現出了他的控場能力，不同於聯盟中大部分替補的亂戰模式，由羅斯來掌控的活塞隊替補堪稱是聯盟中最穩健的替補。他們會尋求大量的配合，而羅斯也在不斷地梳理隊友之間的站位，這使得活塞的替補席場均可以拿下44.5分，這可是聯盟第4的高度。

用這種較為特殊的方式來給球隊帶了更多的貢獻成為了羅斯最值得稱道的地方，他不會太過佔用出手次數，但是卻已經放下了身段融入到替補席當中。同時這也能夠掩蓋掉他防守端的缺失，上賽季他的防守效率排在全聯盟後衛中的倒數位置，如今106.8的防守效率即便同樣不高，但至少也在聯盟後衛的平均水準。

由於他以往喜歡利用身體天賦來進行防守，這也導致了他的防守習慣一度十分差勁，可是現在他也學會了不少厲害的防守技巧。至少他現在已經不會太容易就跟丟進攻球員，還能在面對高大球員的時候實現搶斷，他只需要防住對方的替補後衛，這樣的防守任務對他而言並不算艱難。觀察到這種種跡象，可以說接受替補位置對羅斯而言是一舉多得的好改變。

出場時間仍受限制，嘗試穩定身體狀態

也許細心的球迷也早已發現他場均只有23.3分鐘的出場時間，這在同隊內也只能排在第6位，但這並不代表他的戰術地位會因此下降。由於他身上依舊存在不小的傷病隱患，這也影響令他的價值大幅度下降，出於保護球員的態度，凱西也只能容許羅斯出場短暫的時間。

這已經是老生常談了，反正只要是提到他，我們總是沒辦法避開傷病這個話題。如今的聯盟更加是增添了對於球員負荷管理的明文規定，保證健康才是首要大事，至於出場多少時間也顯得不太重要了。可是這還是會在一定程度上給羅斯帶來一些負面影響，而這個賽季我們也能看見他的一大問題，那就是狀態的不穩定。

說到底他還不屬於「微波爐」類型的球員，他的手感需要通過持球時間來尋找，而這和他的替補位置也有點背道而馳的味道。15場比賽中，他有過31分的賽季新高，也有過2分的冰點時刻，這取決於他那場比賽的狀態是否良好。

作為一名替補球員，最理想的狀態就是快船的威廉姆斯，他能夠隨時隨地拿到分數，而狀態的沉浮就會限制住羅斯在替補位置上的作用。關鍵在於他能不能將火熱的手感隨時保持，而這也需要建立在身體健康的前提下，所以說他只能拿到中產合同也是有原因的。但至少他也去嘗試著接受角色上的改變，想要成為一個最傑出的替補還是要經歷更多的考驗。

寫在最後：

踏出改變的第一步往往才是最艱難的，羅斯已然在替補席上走過了兩個賽季，他也在這個過程中明確了自己未來的打法和位置。這對他來說無疑是天大的好事，過往的榮譽和打法風格不會幫助他繼續在聯盟中立足，只有逐漸的適應時代才不會被淘汰。

看著甜瓜又一次變甜，卡特也繼續翱翔，玫瑰又怎麼可能不會繼續綻放呢？羅斯已經找到了自己的路，剩下的工作就是保持健康和挖掘出更多未知的才能。
