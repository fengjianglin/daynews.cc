今日要聞：猛龍創隊史最大逆轉記錄，小托馬斯被禁賽2場

2019-12-24 414條評論 3,146次閱讀 1999人點贊

虎撲12月23日訊 NBA今日共5場比賽，虎撲為你整理了今日要聞。

本場比賽，猛龍最多時落後對手達到30分（第三節剩餘2分32秒以55-85落後於獨行俠），這創下了猛龍隊史最大逆轉紀錄。此前，猛龍的最大逆轉紀錄是在北京時間2010年12月擊敗活塞時創下的逆轉25分。

根據76人隨隊記者Keith Pompey的報道，76人官方已經對與奇才後衛以賽亞-托馬斯發生口角爭執並對後者進行辱罵的球迷做出了處罰。 76人官方禁止兩名辱罵小托馬斯的球迷進入富國中心（76人主場）觀戰一年，同時也吊銷了將季票轉讓給辱罵小托馬斯兩名球迷的季票擁有者的季票資格。

在談到球隊後衛斯蒂芬-庫里的傷勢恢復狀況時，科爾表示他已經開始在場上進行與籃球相關的訓練活動（有氧運動）。 科爾還表示庫里之後會更多地跟隨球隊，但他目前還沒有開始接正式開始籃球訓練。

據The Athletic記者Chris Kirschner報道，老鷹內部一位消息人士透露，雷霆中鋒史蒂文-亞當斯有可能會在本賽季交易截止日前被交易，而他也一直被老鷹方面討論。

根據NBA記者Malika Andrews的報道，籃網主帥肯尼-阿特金森透露，凱里-歐文目前還沒有得到參加有身體對抗訓練的醫學許可。

根據NBA記者Tim MacMahon報道，獨行俠後衛盧卡-東契奇目前恢復良好，有望在北京時間本周五對陣馬刺的比賽中復出。

凱爾特人前鋒傑森-塔特姆發揮出色，他登場36分鐘，29投15中，拿下39分12籃板2助攻3封蓋，創造了其職業生涯得分和出手次數新高。值得一提的是，塔圖姆在第四節單節砍下了22分。

這是他們過去一周里3次完成分差15+的逆轉，球隊本賽季第4次。逆轉公牛26分，逆轉灰熊24分，以及逆轉勇士15分。雷霆本賽季完成4次分差15+的逆轉，與湖人並列為聯盟為最多。

熱火球員邁耶斯-倫納德在推特上回應吉米-巴特勒前幾日發布的「籃板通緝令」。 倫納德在推特上寫道：「賞金我已經付清了，#金錢##金錢##雙手合十#（表情），很高興看到新的一天！」

今日，開拓者前鋒卡梅羅-安東尼接受了媒體記者的採訪。 在談及自己的膝蓋傷勢時，安東尼表示自己是在此前對陣魔術的比賽中不慎與武切維奇相撞傷到的膝蓋。「一切都還好，就只是膝蓋撞到膝蓋後的挫傷而已。」「我會早點到球館看看自己感覺如何，我傾向於出戰明天的比賽。」安東尼表示。
