NBA退役巨星們的標誌性動作：除了麥迪飄逸干拔還有哪些（一）？

2019-12-23 1,028條評論 4,344次閱讀 3012人點贊

每個人在世界上都是獨一無二的在NBA也是如此！麥迪飄逸的干拔聯盟還能找出第二人嗎？答案是沒有的。那麼今天我們來細數NBA的巨星們的標誌性動作吧！

喬丹：跳投、吐舌頭

如果說科比的跳投美如畫那麼喬丹就是那幅畫，持球背框翻身跳投命中畫面是如此的美妙！他的標誌性跳投成了八九十年代我們對籃球的回憶。

運球過人面框暴扣前總要吐個舌頭是喬丹第二個標誌性特點，有點像可愛的哈士奇喲，不知道小夥伴們還記得嗎？

斯蒂夫納什：舔鞋底

納什的運球極其嫻熟經常突破衝擊籃下也屢屢造成對手的犯規，所以罰球是少不了他的，然而他的罰球動作總是令人詬病的，每次罰球前總要用手摸摸鞋底然後再用舌頭舔舔摸過鞋底的手，這畫面太美小編我不敢想！特殊癖好還是啥原因小編一直很好奇。

艾弗森：crossover和聆聽

全世界都知道艾弗森突右邊可是就是防不住他，他能用同樣的方式過你100次。

諾維斯基：金雞獨立

德國老司機的金雞獨立似的跳投在聯盟可是一絕，幾乎沒有人能防得住，曾今就靠著這一絕技為小牛隊帶來了一座含金量極高的總冠軍。

麥迪：無可複製飄逸的干拔

​接球干拔、突破干拔、搶斷干拔！沒錯麥迪他來了，他帶著飄逸的干拔來了！曾今是聯盟四大分位之一的他得分能力超強加上飄逸的干拔為他吸粉無數！
