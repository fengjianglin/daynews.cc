居安思危！“湖鹿”大戰所折射出的諸多問題值得湖人深思啊

天天要聞
2019-12-23

兩強相遇勇者勝！洛杉磯湖人與密爾沃基雄鹿，東西部各自的霸主，大戰前，兩隊的戰績都是24勝4負並列聯盟榜首，這一戰本該會是一場「火星撞地球」的較量！

老詹vs字母

然而劇情似乎並沒有按照計劃如期而至…

在比賽開始前的8分多鐘的時間裡，兩隊還是處在旗鼓相當的局面，等到老詹第一次下場進入輪換時間的時候，局面似乎有點急轉直下，以至於到了第二節，湖人更是被動挨打，場面變得一發不可收拾，最多時球隊已經落後了21分之多…

而到了下半場老詹只休息了1分多鐘，而濃眉帶著小傷更是1秒都沒有歇，但即使這樣還是沒能挽回劣勢，最終湖人以104比111敗走密爾沃基，輸掉了這場可能是總決賽預演的榜首大戰！

老詹與濃眉

這場比賽是湖人本次5連客的最後一站，球隊在上一場就已經吃到敗仗，102比105輸給了攻防兩端同樣出色的步行者。再往前還與強硬的熱火進行了一場堪稱季後賽級別的較量，比賽打得非常辛苦，強度之大，讓老詹真的顯出了「老態」。

不過我倒認為，此次湖人隊的客場之旅是幸運的，因為能夠與聯盟中這些攻守都比較出色的球隊打比賽，對於球隊來說是一次查漏補缺的絕佳機會。

要知道這種機會可是可遇不可求的，像這種級別的球隊，整個聯盟也就那麼6、7支左右，而此次的5連客就遇到了其中的3支，實屬幸運。

老詹持球突破

在這5場比賽中，球隊先以9分優勢「拆穿」魔術，接著3分優勢艱難「熄滅」了熱火，然後5分優勢「射死」了老鷹，接下來3分劣勢敗走印第安納，以及最後一場7分兵敗密爾沃基。

在這5場比賽中球隊暴露出了很多問題，尤其是與雄鹿的這場「巔峰對決」，就像在顯微鏡下打比賽一樣，缺點被放大了無數倍。

比輸球更讓人擔心的是那些「硬體」配置。雖然賽程剛過了三分之一，但是對於本賽季志在奪冠的湖人來說已經是迫在眉睫。球隊必須儘快想辦法彌補差距，做到未雨綢繆，否者總冠軍真的只是一紙空談。

劃重點——湖人隊此次敗走密爾沃基所暴露出的諸多問題！

相形見絀的輪換陣容

眾所周知，湖人的板凳席在聯盟中一直屬於「平民」的存在(得分排名中等偏下一點)，在比賽中如果沒有被對方板凳完爆就算是燒高香了。

結果本場比賽呢？「湖鹿」板凳得分對比是「扎眼」的4比34！！！

這哪是被爆啊？這簡直是被爆成渣了都…

隆多挑籃

替補只有霍華德與隆執導各拿了2分，其他人的統計欄上都是0。此役過後湖人替補得分33.79，排名直降到聯盟的第22位，這下「平民」都算不上了。

1.板凳席沒有一個有攻堅能力的進攻組織者

湖人替補席後衛線上有隆多、卡魯佐、布拉德利、庫克(本場沒有打)等人。

隆多組織能力毋庸置疑，但是也得分什麼場景。像在行進間快速推進過程中，或者陣地戰對面防守人站位不佳的情況下，隆執導的「眼觀六路耳聽八方」串聯隊友的能力能得到很好的體現。

隆執導

但是，同樣在陣地戰，如果對面防守採用深度蹲坑，極致協防，給一步放外線的策略，空間完全被鎖死，就像雄鹿這場，那隆執導也沒什麼好辦法了…

如果他自己有一定得分能力的話，這樣還能把雄鹿的陣型拉扯出來，空間會得到釋放，同時隊友機會也能相對多一些，但可惜，隆指導並沒有這樣的能力…

隆多

之前倒是有幾場三分命中率甚至超過50%的優異表現，但是說實話有點海市蜃樓的錯覺。熟悉隆多的都知道，這根本不是他正常乾的事兒。

果然最近4場比賽，隆執導三分線外是穩定的13中2…

布拉德利

而卡魯佐、布拉德利這兩人，組織上基本用處不大。進攻呢？今天這兩位絕大多數對位的都是雄鹿身高兩米以上的鋒線球員，身材比你高，比你大，同時移動速度還不比你慢，所以進攻端也真是沒什麼機會，在得分上，兩人今天都掛了零。

2.派出「小陣容」的時候，身高體型劣勢明顯，導致對面鋒線球員可以肆無忌憚地發揮

湖人每場比賽慣用的8、9套陣容，其中以濃眉打5號位的「小陣容」為例，就像今天在第三節使用的這套：卡魯佐、布拉德利、波普、杜德利、濃眉。

前面4個人的平均身高只有191，而雄鹿呢？幾乎每個陣容擺在場上都有將近兩米的身高，而且運動能力照比湖人的小陣容相比一點都不差，這對於湖人來說就很尷尬了。

你會發現場上有兩三個位置，人其實是對不上的，這不是換不換防的問題，是常規來看就對不上，然後就導致雄鹿像米德爾頓、布朗等鋒線球員，在湖人的這些小個球員面前迎面就射三分，根本視你為無物。其實這個問題歸根結底也是湖人隊陣容的硬傷之一——鋒線人員儲備不夠。

在這點上讓我不禁想起了今年的男籃世界盃，咱們國家隊的陣容配置，鋒線上資源同樣匱乏。小丁、周鵬都因傷打不了，可蘭、阿布都等人又得不到教練的信任，能夠真正起作用的只有小川一人，大多數場上擺出的都是3後衛陣容，這就給對手足夠的空間去大打小。

下面這球就是個典型，米德爾頓從後場帶球到前場面對卡魯佐的防守，迎面就射三分，球進後也直接打停了湖人。大家可以對比下那個出手點與封蓋點的高度差，這也就不難理解為什麼米德爾頓能這麼「目中無人」的投籃了。

米德爾頓三分線外迎射卡魯佐

換防之後呢？差距更明顯了，經常把波普、隆執導換到字母的身前，可想而知會是什麼畫面？

字母一發力，波普已經躺地上了，而濃眉的站位已經不在防守位置，雖然腳步移動快，但是字母似乎更勝一籌。籃下的隆執導看姿勢是忌憚字母把球傳給右側底角的康諾頓，而準備向右側外線進行補防，這樣籃下就處於空無一人的狀態，那字母豈不是想怎麼玩就怎麼玩了？

不過說實話，就算身高185的隆多在籃下，對於字母來說基本也跟沒人一樣。

字母面對波普的防守

下面這個球，說實話湖人有點無奈，為什麼這麼說呢？看雄鹿這進攻站位，字母一人弧頂拿球，其他4人全部在三分線外，就連中鋒大洛也是如此。這可不是隨便站的，這四個人可都有一定的三分能力。

字母面對隆多的防守

科沃爾就不用多說了。

左側45度角的米德爾頓賽季至今三分球命中率40.4%聯盟第33，場均能進2.1個三分球。

右側45度角的大洛三分命中率雖然只有28.3%，但也屬於聯盟中游水平，場均能扔進1.3個，至今已經命中了36記三分，要知道他可是中鋒，而且投三分真的是他常規「武器」。

左側底角的希爾三分命中率更是高達51.2%聯盟第1。

湖人的其他人根本沒法協防，就算有協防動作，大部分也是假的，所以只能眼睜睜地看著隆執導被字母無情碾壓，兩大步基本已經堆到籃下了，就像大人打小孩一樣輕鬆放籃得手。

中鋒霍華德被大洛的遠投能力牽扯出了禁區

針對「重點人」，防守策略存在問題

對於科沃爾這種級別的投手，一定要重點「照顧」。

就本場比賽來看，說沃格爾對比賽的解讀能力存在問題也好，或者說球員對技戰術的執行力不夠也罷，反正一些回合暴露出分不清重點人的現象是真實存在的。

比如第三節末段的這個回合，雄鹿外線的這「三桿槍」可是雄鹿外線最強的火力點。科沃爾雖然本賽季的三分命中率「只有」38.4%，但是他可是個歷史級別的三分神射手啊！生涯三分總命中數2389個歷史排名第4，是專靠這個吃飯的。而另外兩桿槍米德爾頓和希爾，三分球命中率一個超過40%，一個超過50%。

大洛低位打濃眉根本就是個「假象」，而右側埋伏底線的伊利亞索瓦在本回合的角色定位，充其量也就是沖搶二次籃板、補籃什麼的。而且以濃眉的單防能力，大洛根本占不到任何便宜，無需包夾協防。

但是湖人這邊似乎並沒有看清「真相」。

注意格林的視線並沒有在自己的防守人上

於是上線的格林對大洛採取了包夾，以至於湖人全隊的防線開始有點顧此失彼的趨勢…

格林錯誤判斷局勢對大洛進行包夾

大洛面對包夾，在自己臨出界前將球甩給了弧頂的科沃爾，只見後者在毫無干擾的情況下，神情堅定，泰然自若，從接球到投籃，動作連貫、姿勢優美，整個過程一氣呵成，堪稱教科書。皮球也是在空中划出了一道美麗的弧線正中靶心。

對於這種問題怎麼解決？

首先，教練得有「撥開迷霧見月明」的能力。

對於對手的排兵布陣起碼得能做到中心有數。防守端，面對對方不同的陣容，要知道進攻強點在哪？解讀對手的真實目的。進攻端也是同樣的道理，要明確對面的防守弱點在何處？，針對「弱點」去展開進攻。

其次，需要球員在場上高效的執行教練的戰術安排，同時也需要有閱讀比賽的能力，進而做到隨機應變。只有頭腦清醒，明確對方的真實意圖，才能打得更加「明白」，即使最終輸球，也會收穫滿滿。

老詹單打效率固然高效，但似乎是無奈之舉

1.除鐵林外，幾乎沒有一個穩定的外線「輸出手」。

本場比賽，湖人全隊三分35中12，命中率為34.2%，看著還算可以。

但是在這12記三分中，有7個來自於格林，此役後者三分線外12中7，58.3%的命中率，如果拋開格林的數據不看，那湖人全隊三分線外的表現就變成了23中5，命中率僅為21.7%，這就很扎心了…

也不是對手的外線防守有多麼強，根據本場比賽雄鹿的整體防守策略來看，對於外線還是會給出一些空間的，只不過湖人這邊外線的球員實在是問題太多，首先身高就是個問題，即使給你一步，防守時還是能造成足夠的干擾，甚至有時都出不了手…

隆執導近幾場已經是「原形畢露」，波普雖然有賽季38.1%的三分命中率，但是穩定性不好，時好時壞，卡魯佐本身就不擅長在外線「張弓搭箭」。另外值得一提的是，本場濃眉在三分線外也是顆粒無收，三分6中0，有點小尷尬…

2.內線球員除濃眉外，幾乎沒有自主進攻能力，面對大洛這種有投射兼顧一定技術的中鋒非常吃虧。

正所謂「知己知彼方能百戰百勝」，而湖人中鋒麥基顯然沒有意識到這一點。

像麥基這種「糙漢子」面對大洛這種會「針線活」的中鋒，說實話心裡是非常沒有底的，很慌，所以只能採取「下下策」的又摟又抱，這無疑是已經「繳槍」的行為。

像下面這個球，大洛左側低位拿球單打麥基，介於大洛有投射能力還有一定的低位技術，所以就簡單的一個虛晃想走左路，這時麥基已經不淡定了，直接上前摟人。

這球都沒了，還抱著人不放呢！這是玩曖昧呢？

面對「囧基」的「囧操作」，沃格爾瞪大了雙眼遠遠的望著這一幕，助教基德等人的表情也有點耐人尋味，至於潛台詞，大家可以自己腦補…

沃格爾一臉不滿的看著麥基的防守

聊聊自己在現實中的實際經歷。

這種情況，我想經常打球的朋友應該多少都會了解一些，我就拿自己的實戰經驗來談一談這個事兒吧。

平時出去跟別人打球的時候，一般打的都是半場陣地，3V3或者4V4，一局5個球，或者7個球的那種。

如果你自己的技術比較糙，基本功不好，那麼在防守各方面都比你強的人時，你心裡會比較慌，確實是這樣，因為你不知道他下一秒到底會做什麼，充滿著很多未知因素。

我本人的籃球水平還算可以，基本上籃球場上的「那些事兒」都會一些。但是在我遇到那種身體條件，技術能力都比較好的人時，自信心還是受到過傷害，以至於後來越打心裡越沒有底。

但我倒不會像麥基那樣直接抱人，也沒必要，因為平時打球的時候，大家主要還是以娛樂為主。這時我會盡量努力去防，把注意力集中，爭取能找到解決問題的「鑰匙」，然後隨著某一兩個回合收到效果的時候，自信心也會得到一定的恢復。

所以這種情況不要直接「繳槍」，把注意力集中，多動動腦子盡量去防，也許最終的結果會完全不同。

寫在最後

湖人本場比賽所暴露出的問題一句話總結就是：鋒線資源匱乏、外線穩定輸出點不夠以及個人與整體的防守策略問題。

針對第一點，可以通過交易，或者在買斷市場上想辦法。

針對第二點，只能是多多訓練，還有提高在比賽中解讀對手防守的能力。

針對第三點，也只能是多多研究對手，比如多看看比賽錄像，然後有針對性的進行訓練。

最後希望湖人能儘早解決這些問題，然後如果能在6月份再次碰到雄鹿的時候，來他一場真正的勢均力敵的較量！
