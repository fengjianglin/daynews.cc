湖人優於快船？詹眉3優勢遠勝卡喬，湖人可沖73勝？

天天要聞
2019-12-11

本賽季開啟之後，對於西部球隊來講湖人與快船誰更強是一直繞不開的話題。如今兩支球隊都已經進行了近25場的比賽，在這25場比賽中兩支球隊的狀態都一目了然，因此不少專家記者以及媒體認為：湖人要優於快船。

戰績

首先從戰績來講，湖人隊目前取得了21勝3負的戰績，與雄鹿高居聯盟第一，這一點上要遠勝過快船。快船目前的表現18升7負，與湖人隊相差3.5個勝場。對於湖人來講這是一個極大的優勢，快船目前在雙核還不能穩定出戰的情況下，快船隊無法去追趕。

在戰績方面，湖人隊的表現要好過快船，這一點其中有傷病的影響。喬治在賽季之初無法出戰，倫納德目前依舊需要輪休，這是影響快船隊的最大問題。

攻防

湖人隊本賽季的最大變化就是攻防兩端的變化更加均衡，首先湖人每場比賽得分114.5 ，失分104.6，每場凈勝分達到9.9分；快船隊則是得分115.1分，失分108.3分，凈勝分6.8分；

湖人隊的進攻略遜色快船，但是防守卻是比快船隊要更加出色，球隊在防守端一度成為聯盟第一。如今在球隊的布拉德利受傷之後，湖人隊的防守才逐漸下滑到目前的第五位。

韌性

考察一支球隊是否是冠軍球隊還有一個重要的點就是球隊在打硬仗的表現如何。快船面對目前排名東西部前八的隊伍，一共有12場比賽，總戰績為7勝5負；而同樣是湖人隊，在11場比賽中，湖人隊的戰績為8勝3負，比快船隊少1場比賽，但是戰績要更加出色。

能夠看到湖人隊在面對實力較強的球隊時表現更加出色，綜合來看湖人隊在這一方面的表現要好於快船隊，表現要比快船隊更加出色。如今的湖人隊只3敗，並且目前有很大的機會去衝擊NBA的紀錄73勝。
