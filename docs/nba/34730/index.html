聖誕大戰！東部更精彩 西部無懸念 洛杉磯德彼勝者為王

2019-12-25 2,507條評論 3,911次閱讀 2019人點贊

聖誕大戰一直是聯盟推崇的最精彩的籃球比賽，在聖誕節這一天準時上演，在歡渡節日的同時，上演一場場精彩的視覺盛宴，把那些最具看點的焦點賽事放在聖誕之夜，無關勝負，但絕對精彩！那麼今年的聖誕大戰有那些不可錯過的對決呢？我們一起來看看聖誕之戰的五場比賽。

五場大戰，二場比賽在東部前五名之間展開，三場比賽在西部，其中火箭VS勇士、掘金VS鵜鶘二場比賽算是西部的首尾之爭，湖人VS快船是同城德比，絕對值得重點關注。

2019-12-26 周四01:00:00時，凱爾特人VS猛龍，凱爾特人客場挑戰去年的總冠軍猛龍隊，兩隊相差0.5個勝場，凱爾特人目前東部第二，猛龍東部第四，去年常規賽兩隊的排名正好相反，此戰凱爾特人陣容整齊，而猛龍缺少球隊主將西亞卡姆，猛龍主場13勝3負，凱爾特人客場8勝6負，猛龍佔盡優勢，本隊若猛龍獲勝，將進入東部第三名，凱爾特人勝則進一步穩固第二名的地位，這是一場提前進行的東部半決定，看好猛龍主場取勝。

2019-12-26 周四03:00:00時，雄鹿VS76人，由排名全聯盟第一的雄鹿VS 76人隊，這可以說是一場提前上演的東部決定，兩隊賽季初都被寄以厚望，特別是76人隊，全明星的首發五人預測將打出優異戰績，但目前東部排名不理想，這與球隊的西帝本賽季的發揮有著很大的關係，反觀雄鹿隊，在上賽季常規賽MVP字母哥的帶領下，一路高歌猛進，曾經有過17連勝的驕人戰績，本場比賽可謂針尖對麥芒，76人隊主場12勝1負，雄鹿客場12勝2負，兩隊各位置實力對比相差不大，76人肯定不會讓字母哥好過，兩軍對壘，勇者勝，雄死誰手未可知啊？

202019-12-26 周四06:00:00時，火箭VS勇士，這是一場沒有懸念的聖誕大戰，勇士早已沒有了昔日總冠軍的風采，傷病將他們徹底擊倒，西部倒數第一的排名與明年的狀元簽惜惜相關，勇士捨不得放棄，繼續擺爛才是他們最終目標，而火箭在經歷了賽季初防守退化和不穩定後，現在威少已經適應和找到了在火箭和哈登身邊打球的感覺，漸漸恢復到了前幾個賽季的剽悍，衝擊力十足，哈登場均38.6的得分，已經無人能敵，只能翻出上古大神張伯倫與其比較了，在二人共同帶動下，火箭目前排在西部第三的位置，如果在接下來的三場比賽中拿到全勝，有望衝擊西部榜首的位置，可以說這是一場首尾之爭，結果已經沒有懸念，球迷期待的火勇大戰，看來只能等到明年了。

2019-12-26 周四09:00:00時，快船 VS湖人，一場同城德彼是多少球迷和媒體的期待，兩支同城新貴，都是本賽季重組，二隊網邏了聯盟中進攻和防守最好員中的四人，兩兩二組，湖人西部第一，快船第四，是人們預測中的西部決定的對手，本場對決，勝負的天秤已經倒向快船一方，湖人正經受著三連敗的慘痛經歷，詹皇和濃眉哥都有傷在身，能否出戰存疑，快船的小卡應該在聖誕大戰中上崗，結束輪休，小卡和喬治攻守平衡，湖人此戰難以保全，快船取勝後同樣對榜首有著巨的衝擊，有望登上榜首，湖人能不能全員出戰為球迷奉獻一場精彩的聖誕大戰呢？

2019-12-26 周四11:30:00時，鵜鶘VS掘金，幾乎又是一場所首尾大戰，西部排名正數第二對倒數第二，多數媒體預測鵜鶘隊本賽季將在一個強勢反彈，今年夏天，球隊加入了湖人二少，又有狀元錫安加盟，再加上不錯的交易引員，所以球迷期待他們能有象獨行俠一樣的戰績，但事與願違，傷病和不成熟讓這支球隊再次有了爭奪狀元簽的資本，就是安排聖誕大戰的聯盟也不會想到勇士和鵜鶘倫落到這種地步，掘金隊依然穩定如昨，戰勝鵜鶘應該在意料之中。

五場聖誕大戰，東部精彩紛呈，西部毫無懸念，只期待洛城德彼能為我們帶來不同的享受！
