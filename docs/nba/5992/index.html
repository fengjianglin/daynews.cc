滲入各行各業！美媒曝詹姆斯12大讚助商：僅耐克就支付他70億

2019-12-08 0條評論 14次閱讀 0人點贊

湖人球星勒布朗·詹姆斯是美國乃至全世界最知名的運動員之一。由於巨大的知名度和市場性，詹姆斯已經從廣告代言中獲得巨額收入，他已經成為各種公司的品牌大使。

美媒盤點了詹姆斯令人印象深刻的商業投資組合。其中很大一部分是詹姆斯多年來獲得的各式各樣的代言合同和贊助商。

耐克

作為詹姆斯最大、最賺錢的贊助商之一，耐克在2003年擊敗了競爭對手阿迪達斯和銳步，把詹姆斯拉進耐克代言陣營。當時，阿迪達斯和銳步向19歲的詹姆斯提供了一份價值8700萬美元的合同。2015年末，詹姆斯與耐克簽署了一份「終身合同」，到64歲時，耐克將支付給詹姆斯超過10億美元，摺合成人民幣就是70億元。

可口可樂

詹姆斯在2003年與這家飲料巨頭簽約，推廣雪碧和Powerade——兩種可口可樂產品。最近，詹姆斯在雪碧冬季香料蔓越莓的廣告中亮相。

Beats Electronics（節拍電子）

這家由說唱歌手Dr. Dre創立、2014年被蘋果收購的熱門電子公司，自2008年以來一直在贊助詹姆斯。這位NBA巨星經常在比賽前後戴上公司的耳機。

Blaze Pizza（大火披薩）

勒布朗是這家加州披薩連鎖店的最初投資者之一。2012年，詹姆斯投資了這家公司，從那以後，Blaze披薩就成為了《福布斯》雜誌上增長最快的連鎖餐廳。今天，詹姆斯是這家餐廳的發言人和代言人。

Upper Deck

詹姆斯自從新秀賽季開始，他就得到了這家交易卡公司的贊助。該公司擊敗競爭對手托普斯(Topps)，簽下詹姆斯，最初簽約5年，合同金額超100萬美元。

State Farm Insurance（州立農業保險）

2008年，詹姆斯與該公司簽署了一份贊助協議，和詹姆斯一起簽約的，還有克里斯·保羅、詹姆斯·哈登、帕特里克·馬豪斯、亞倫·羅傑斯等其他職業運動員。

Dunkin Donuts and Baskin-Robbins（唐恩都樂和巴斯金羅賓斯）

由於Baskin-Robbins是Dunkin ‘ Brands Inc.的一部分，所以這兩家公司合併了。2012年3月，詹姆斯與該公司簽約，在亞洲推廣唐恩都樂(Dunkin Donuts)和巴斯金-羅賓斯(Baskin-Robbins)。

起亞汽車

自2014年以來，詹姆斯一直是起亞汽車的奢侈品大使，並參與了該公司及其汽車的幾則廣告宣傳。起亞汽車也是NBA的主要贊助商之一。

Verizon（威瑞森）

這家電信巨頭和詹姆斯自2016年開始合作。但有傳言稱，詹姆斯對該公司的贊助可能會在2019年或2020年初結束。2015年，NBA放棄了Sprint和T-Mobile，與Verizon簽署了價值約4億美元的協議。

英特爾

為了推廣自動駕駛汽車，這家科技公司在2017年與詹姆斯簽署了贊助協議。詹姆斯還被用來推廣電視和其他英特爾產品。

Audemars Piguet（愛彼牌）

這家瑞士手錶公司從2011年開始贊助詹姆斯。這個奢侈手錶品牌甚至有一個受詹姆斯啟發的手錶系列，很多手錶都用了他的名字。

Rimowa

詹姆斯最近的贊助商，奢侈箱包品牌Rimowa，在2019年5月加入了詹姆斯的投資組合。該公司的「永不靜止」品牌既受到詹姆斯的啟發，又由詹姆斯擔任品牌大使。

12大讚助商，僅耐克就讓詹姆斯賺得盆滿缽滿，這也能理解為什麼詹姆斯在湖人最低迷的時候加入，詹姆斯真的是籃球的王。
