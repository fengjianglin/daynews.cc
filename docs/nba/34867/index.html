最大黑馬，逆風拿下快船後，最慌的是雷管？

2019-12-24 2,467條評論 4,062次閱讀 1989人點贊

北京時間12月23日，雷霆主場爆冷118-112擊敗快船，又是一場大比分的逆轉，現在的雷霆的韌性更是讓人佩服，不管什麼樣的對手，不管什麼樣的局面，千萬不要小看他們，今年目前為止的黑馬非他們莫屬，不過接下來會不會持續下去那就很難說了，畢竟雷管的目的更接近於重建！

目前西部第七，這樣的戰績讓人大吃一驚也讓人眼前一亮，雷霆是個高產巨星的地方，曾經的三少每個都是MVP級別，現在的雷霆又出了一個亞歷山大，21歲的小伙倒是不懼大場面，不管什麼樣的對手都會讓你深深記住他，這也是雷霆重點培養的目標，很有潛力的小夥子！

海王別看年過半百的臉，實打實才26，不過這兩年下滑嚴重，不過今天實打實 吃了一次雞，20+17對於當年的海王可能不怎麼稀奇，不過對於現在來說真算是過年吃頓餃子了，如果能持續這樣的表現我想雷管又要動搖了。

對於現在雷霆的表現，cp3還是有很大的功勞，不管是海王還是保羅都是雷霆擺上貨架急著出手的球員，他們想要的重建，可是事與願違呀，這種局面反而是騎虎難下了，交易就會打破球隊平衡倒時候百分百會擺爛，如果不交易今年進季後賽只能說有機會而不是十足把握，這是考驗雷管的時候了，大家覺得雷霆會怎麼選擇呢？
