再次大逆轉！美國楊毅：如今能阻止保羅和雷霆只有管理層的決定

2019-12-20 312條評論 2,173次閱讀 462人點贊

北京時間2019年12月19日，NBA常規賽繼續進行，其中有保羅領銜的雷霆隊在主場迎戰灰熊隊。最近雙方球隊的狀態依然不錯，灰熊隊在雙子星的出色發揮下取得兩連勝，而雷霆隊再上一場在落後26的情況下大逆轉取得勝利，狀態依然不錯，此番雙方碰面。外界認為是火星撞地球的表演，最終雷霆隊再次上演逆轉好戲，在落後高達24分情況下，在主場以126:122逆轉灰熊取得兩連勝

比賽一上來，雙方就進入了狀態，灰熊隊連續通過內線得分，你13:9搶了先機，但那天對由加里內利和諾爾內線連續上演暴扣，持續為雷霆隊追風，但此時克拉克迎來爆發掘金內線連得9分，在本節結束幫助灰熊以32:25在客場取得領先

進入第二節。發難依然是灰熊隊，他們憑藉外線火熱的手感，連中三分，瞬間將比分51:36領先到15分，這是雷霆隊依靠加里內利三分幫助球隊止血，灰熊隊我在外線手感實在火熱，連中三分此時保羅站出來了，帶領雷霆隊穩定拿分助攻加里納利連中在半場結束時。將比分59：68縮小到個位數

進入下半場之後，雷霆隊再出現進攻端突然啞火持續打鐵而灰熊隊趁機打出一波小高潮將比分擴大到20分以上。雷霆依靠加里內利連投帶罰連得九分幫助球隊追分，但此時灰熊隊狀態太好了。比分一直在20分左右。但在本節末端的時候，灰熊的狀態突然下水一樣下跌，此時施勞德得連投帶罰，幫助球隊不斷縮小比分，爾灰熊連續出現了失誤。雷霆在保羅的帶領之下，在本節末段打出一波小高潮，在第三節結束前將比分93:100落後七分進入決勝節

進入第四節雷霆保羅的和亞當斯聯手打出一波小高潮，幫助雷霆105:104將比分反超。此後雙方比分持續上升，比分來到116平。來到決戰時刻，保羅的跳投依舊穩健。在關鍵時刻每次幫助球隊力挽狂瀾。但隨著最後時刻莫蘭特的兩罰不中，加上施勞德的罰球穩鍵。這種雷霆隊在主場以126:122在落後最大分差24分情況下，逆轉灰熊隊取得兩連勝

次番主場作戰的雷霆高達6人得分上雙，其中三人得分超過20，表現最好的施勞德的全場19投10中得到全場最高的31分7個助攻個籃板。亞歷山大和加里內利分別各有20分的進賬。令廣大球迷關心的保羅在出戰30分鐘情況下10投6中得到18分5個助攻4個籃。還有2次搶斷。亞當斯和諾爾也有10分和13分的進賬

客場作戰的灰熊隊也有高達三人得分超過20。其中雙星合砍49分克拉克19投13中砍下全隊最高的27分外加7個籃板1個助攻，莫蘭特16投9中砍下22分7個助攻5個籃板。中鋒瓦蘭也有24分9個籃板3個助攻，布魯克斯也有15分6籃板6助攻的貢獻

本場比賽最大的亮點是雷霆隊再一次逆轉，再一次落後高達20分多分的情況下在逆轉勝利，這邊比賽後已經取得兩連勝，球隊排名已經來到西部第7的位置，如果雷霆隊這狀態保持下去的話本賽季的季後賽名額可是觸手可及，讓廣大球迷關心的是，保羅最長比賽再次打出出色的表現在全場僅有10次出手機會的就輕鬆得到18分5個助攻4個籃板，賽後有美國楊毅之稱的史密斯又調侃，現在可以能阻止雷霆隊和保羅的只有傷病和管理層的決定了，不知道大家怎麼看的，歡迎廣大球迷在下方留言
