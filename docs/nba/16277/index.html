老鷹3人20+不敵步行者7人上雙，吹楊30投僅中9，老鷹仍不悔？

2019-12-14 1,437條評論 2,547次閱讀 1543人點贊

在剛剛結束的步行者與老鷹的常規賽中，坐鎮亞特蘭大主場的老鷹最終以100-110主場不敵步行者。

儘管特雷楊本賽季的數據相比新秀賽季突飛猛進，和東契奇堪稱一時瑜亮，但魚腩了幾個賽季的老鷹本賽季依然沒能有任何起色。截止至目前，老鷹的戰績僅僅只有6勝19負，低至東部倒數第三。

近十場比賽里僅僅贏下兩場的老鷹對上目前高居東部第六的步行者，在比賽的一開始雙方巨大的實力差距就盡顯無疑。

在第一節一開始，雙方還能互有來回，打得難分秋色。但到了首節後半段，老鷹的手感冰涼，命中率逐漸降低，而步行者則抓住這個機會，打出一波小高潮，取得了領先優勢，以29-22領先老鷹7分結束本節。

次節上來，步行者延續了第一節不錯的狀態，麥康奈爾連得4分，麥克德莫特也三分得手，幫助步行者進一步將比分擴大。而老鷹並未找到什麼好的應對辦法，最終上半場結束，步行者以63-48取得了15分的領先優勢。

下半場開始，老鷹如夢初醒，全員火力全開。在特雷-楊和賈巴里-帕克的連續得分帶領下，老鷹開始瘋狂地追分。最終在這一截結束時，老鷹將比分追為83比86，僅僅落後3分。

末節決戰，老鷹依然展現著渴望贏下這場勝利的決心，但步行者打得十分穩健，在麥克德莫特連續兩記三分得手之後，步行者一直維持著8分左右的領先優勢，最終老鷹未能再次將比分追上，以100比110輸掉了這場比賽。

本場比賽，老鷹先發共有三人得分20+，但難阻球隊失利。其中，特雷-楊上場37分鐘，30投9中，得到23分7籃板8助攻1蓋帽；德安德烈-亨特上場37分鐘，16投7中，得到21分4籃板2助攻2搶斷；賈巴里-帕克上場27分鐘，13投9中，得到20分6籃板1助攻1搶斷。

而76人則全員多點開花，共有7人得分上雙：沃倫18分4籃板1助攻2搶斷，薩博尼斯12分14籃板2助攻，邁爾斯-特納11分5籃板，蘭姆11分4籃板4助攻，布羅格登19分8籃板12助攻，麥克德莫特16分2籃板2助攻，阿隆-霍樂迪13分2籃板，賈斯丁-霍樂迪2分4籃板，麥康納8分3籃板5助攻
