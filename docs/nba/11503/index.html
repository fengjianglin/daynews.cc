10勝38負，樂福絕望，科爾很享受？這是2年前爭冠的騎士勇士？

2019-12-11 2,067條評論 3,755次閱讀 247人點贊

騎士和勇士的比賽像國足一樣，很難再有懸念！騎士對綠軍，騎士早早投降，從第一節開始被揍了一晚上。勇士對灰熊，菜雞互啄到最後一刻，儘管NBA彩經百般看好，無奈新秀莫蘭特不懂彩經，末節一頓猛操作，把勇士按牢在副班長「寶座」上。想想2年前騎勇還在爭冠，如今聯手貢獻10勝38負，這回騎勇真的開始爭狀元了？

騎士排名

騎士自從走了詹姆斯，就打算推倒重建，今天「如願」拿下7連敗，5勝18負排名東部倒2，在爭狀元的道路上越走越穩。騎士把當家樂福也擺上了貨架，期望能在交易市場賣個好價錢。樂福說：「我們太爛了，我開始有點絕望了！」絕望的還有騎士老闆，樂福這2場打得也「太爛了」，連場得分未過10，這還是當家球星嗎？但隊里還有比他更值錢的嗎？

凱文·樂福

勇士本賽季的墜落有點意外，一開始大家還在糾結，現在好了，不用再想季後賽了，「水花兄弟」可以安心養傷了，科爾都開始享受了：「我很享受這個賽季，因為我可以專心指導年輕球員，我們有機會尋找補強球隊的球員，培養潛力巨大的球員……」不愧是冠軍教頭，輸球都能輸得如此高大上，爭狀元都能爭得這麼有逼格！

勇士排名

只有追夢格林還在掙扎，全場13次出手創個人賽季新高：「我們本該在上一場勝利的基礎上繼續前進，但今晚我們還是不能如願……」是的，勇士本賽季獲勝後第2場的戰績為0勝5負，從未品嘗過連勝的滋味。想起當年破紀錄地拿下73勝，耿直的格林真的學不來科爾那種「不能反抗，就默默享受」的超凡境界！

追夢格林

或許是意外所致，或許是氣數已盡，或許是因果報應，或許是因禍得福！科爾或許真能為勇士尋找或開發一兩位出色新秀，搭配庫里、湯普森、格林和拉塞爾，再組一套爭冠陣容。而克里夫蘭騎士這位「爭狀元」的高手，在明年選秀場還能再亮高招嗎？如果凱文·樂福真的要走，哪裡才是歸宿呢？

科爾
