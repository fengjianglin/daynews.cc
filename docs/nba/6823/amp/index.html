湖人夢碎？最重要的拼圖被「雪藏」，計劃被打亂，詹皇出面也沒用

天天要聞
3 週 ago

洛杉磯湖人隊，在本賽季開局階段打出了令人驚艷的表現，在詹姆斯和戴維斯的帶領下，湖人隊目前的戰績，已經高居全聯盟第一。對於這支上賽季沒有進入季後賽的球隊來說，一眉哥的加入，以及一眾實力球員上場的補強，確實讓湖人的實力得到了升華。詹姆斯在去年經歷了一個失敗的決賽季後，本賽季又有機會重新爭奪總冠軍，雖然，如今湖人隊的陣容已經十分完整，但還有一個人，一直是他們心心念念想要得到的，那就是Andre Iguodala。

今年夏天，因為勇士隊的核心杜蘭特出走，球隊隨即進入了重新規劃未來的階段，而Iguodala也被勇士擺上了貨架，通過交易送到了灰熊隊。目前Iguodala正處於合同的最後一年，而本賽季他的薪水，已經達到了1700萬美元。然而Iguodala卻並不在灰熊隊的發展計劃之中，對灰熊隊來說，他們的首要目的，就是培養年輕潛質球員上場迅速成長，而Iguodala這樣的老將，雖然依舊具備了足夠的實力，但顯然和灰熊隊顯得格格不入。

這樣的情況也導致了本賽季開始前，Iguodala就與灰熊隊達成了共識，Iguodala不會代表灰熊隊出戰，也不會參與球隊訓練。湖人原本希望在決賽季中，灰熊隊能夠買斷Iguodala的合同，因為若非如此，灰熊隊將會在本賽季，白白支付給Iguodala1700萬美元的薪水，而買斷無論對誰都有好處。灰熊隊可以剩下部分的錢，Iguodala也能到自己心儀的球隊，而需要Iguodala的球隊，也能在陣容上變得更強大。此前湖人與Iguodala是互相心儀的，聯盟大部分高層曾認為Iguodala未來會加盟湖人隊，因為，他在洛杉磯也有很多自己的生意。

然而11月25日，根據美媒報道，灰熊隊正迫使對Iguodala感興趣的球隊（湖人和快艇）通過交易得到他。灰熊不會買斷Iguodala，他們希望能夠得到一些選秀權之類的回報。並且，據美資深記者透露，灰熊隊已經給湖人和快艇傳遞了這個信息，Iguodala不會出現在自由市場上。這樣一來徹底打亂了湖人的計劃，被視作湖人奪冠最重要拼圖的Iguodala被「雪藏」，灰熊還提出來想要就來換的交易要求，湖人想要裸簽下Iguodala的夢想，恐怕是真的碎了。雖然，曾有消息表示詹姆斯出面邀請過Iguodala，但是，即使詹皇的面子再大，灰熊隊不放人，一切也都成了徒勞。
