詹姆斯帽隆多帽出條件反射？NBA還有很多，圖看那些經典條件反射

天天要聞
2019-12-18

最近在湖人的一場規賽中，發生了有趣的一幕。當湖人球員拉簡·隆多完成快攻上籃的時候，他的隊友勒布朗·詹姆斯竟然從背後起飛企圖給自己的隊友一個追身大帽！估計球迷都看傻了，詹姆斯是不是以前帽隆多帽出了條件反射？

好在詹姆斯及時停手，不然這可能是NBA有史以來最NB的五大囧！估計隆多也是嚇出了一身冷汗，不過替補席上的隊友早就笑開了花。賽後接受採訪的老詹表示，自己被隆多上籃之前的傳球假動作騙了，索性也假裝要追帽他嚇唬回來。

以前隆多和詹姆斯可是老對手了。當年詹姆斯率領騎士大戰隆多和凱爾特人三巨頭，幾乎每次系列賽都堪稱經典。隆多也是詹姆斯追身大帽的最悲慘受害者之一。

哪怕詹姆斯轉會熱火，隆多也依然屢屢慘遭毒手。又一次詹姆斯還把隆多狠狠帽飛了出去，可憐的隆多直接摔出了界外，狼狽不堪。

所以隆多學聰明了，每次和詹姆斯比賽，快攻上籃都記得回頭看看！這才有了他職業生涯最精彩的一記傳球！戲耍詹姆斯背傳助攻隊友得手！NBA球員身經百戰，他們的生活幾乎只有籃球。那麼一些動作自然也成了條件反射一般，成為肌肉記憶深深地刻在他們的身體里。

比如兩個著名的「演員」哈登和吉諾比利，就在一次直面對話中大秀演技。吉諾比利一動，哈登身體也隨之反射，兩人雙雙摔倒在地試圖騙倒一次犯規。球迷們看得滿臉黑線，估計心裡已經唱起來「簡單點，讓我的籃球簡單點……」

戰術跑了千百遍，就算是轉會了也忘不了那段兄弟情啊！當年詹姆斯從熱火重返騎士，面對老東家的時候竟然腦子一抽跑上去給前隊友科爾做了個掩護！而科爾也是一愣，隨即持球殺了過去。

同樣的事情也曾在加內特身上發生過。當年KG和皮爾斯轉會至籃網，綠衫軍只剩下隆多獨挑大樑。曾經的戰友不得不兵戎相見，恍惚之間KG似乎忘記了你我已是對手！看著持球進攻的隆多，竟然上去幫他做了一個擋拆！

如果說前面的行為是條件反射，那麼科比的這個反條件反射註定讓人感慨萬千。當年魔術隊的巴恩斯試圖用球嚇唬科比，然而科比竟然絲毫不為所動！這才是籃球運動員的精神！作為一個球員，怎麼能被自己心愛的籃球所嚇倒呢！

再來說說這一次「條件反射」，在2016年西部決賽中，格林抬腿提到了雷霆球員亞當斯的要害部位。

格林這一腳踢得很重，亞當斯當時只覺得下體一緊，其痛苦無以言表。而這一腳，也成為勇士悲劇的伏筆。

到了總決賽，格林又一次沖著襠去了！這一回打到的是詹姆斯。而事後聯盟也因為詹姆斯的抗議對格林的動作調查，並且決定禁賽一場。最終騎士也抓住了機會，成功1-3落後的不利局勢下完成翻盤。可以說格林的這一拳一腳，直接搞丟了勇士的總冠軍。
