歐冠八分之一決賽今晚19:00進行抽籤對陣

2019-12-16 1,733條評論 3,074次閱讀 390人點贊

　　今天晚上七點，2019-20賽季歐冠1/8決賽就將進行抽籤儀式，十六強球隊將根據同組和同聯賽迴避的原則，抽到自己在1/8決賽的對手，期待今晚的抽籤能有精彩的強強對決!

　　歐冠抽籤規則和預測

　　歐冠16強的抽籤是按照一系列規則進行的，已經在小組賽階段對陣的球隊不能被抽到一起，來自同一個國家的球隊也不能被抽到一起。小組第一名的球隊將抽取對陣第二名的球隊。

　　根據上述規則，皇馬可能會遇到拜仁、曼城、尤文圖斯、利物浦或萊比錫，而同樣獲得小組第二的馬競可能會遇到PSG、拜仁、曼城、利物浦或萊比錫。

　　小組第一出線的巴塞羅那可能會面對那不勒斯、熱刺、切爾西、里昂或亞特蘭大，而同樣獲得第一名的瓦倫西亞也有類似的可能對手：那不勒斯、熱刺，多特蒙德，里昂和亞特蘭大。

　　同時，基於本國球隊不相遇，小組賽同組球隊不相遇這兩個原則，有媒體對各球隊相遇的概率做出了模擬計算。根據模擬結果，兩個英超的小組第一曼城和利物浦，抽到馬競的概率都高達22.4%，而切爾西遇到巴薩的概率也高達23.28%。熱刺則最可能和瓦倫西亞相遇。

　　過去幾個賽季，巴薩的簽運一直上佳。目前情況來看，無論其抽到熱刺、那不勒斯、切爾西、里昂還是亞特蘭大，只要發揮正常都能夠較為輕鬆地晉級八強。雖然巴薩與熱刺相遇的概率並不是最高，但是上賽季歐冠四強對陣亞軍的較量卻備受球迷期待。穆里尼奧自帶流量，梅西自然更不必多說。同時， 亞洲天王孫興慜與梅西的較量，同樣備受期待。

　　歐冠十六強名單

　　A組：巴黎聖日耳曼(法甲)，皇馬(西甲)

　　B組：拜仁(德甲)，熱刺(英超)

　　C組：曼城(英超)，亞特蘭大(意甲)

　　D組：尤文圖斯(意甲)，馬競(西甲)

　　E組：利物浦(英超)，那不勒斯(意甲)

　　F組：巴塞羅那(西甲)，多特蒙德(德甲)

　　G組：萊比錫(德甲)，里昂(法甲)

　　H組：瓦倫西亞(西甲)，切爾西(英超)
