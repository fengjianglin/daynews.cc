三位一體才能稱王--盤點那些年國際足壇讓人印象深刻的"三叉戟"

2019-12-18 2,498條評論 4,982次閱讀 1670人點贊

在我的足球記憶中，最早聽到「三叉戟」這個稱號，是上世紀90年代末期那隻AC米蘭的三個進攻球員，分別是巴喬+維阿+薩維切維奇三人組合，雖然克林斯曼+布雷默+馬特烏斯和古利特+巴斯滕+里傑卡爾德也是讓人耳熟能詳的三人組合，但他們更廣泛流傳於世的稱號分別是德國「三駕馬車」和荷蘭「三劍客」。

小編本文不拘泥於具體的叫法，凡是屬於那個年代大家公認的三人組合，我都把它稱之為三叉戟。下面小魚兒就和大家一起來回憶近年來國際足壇著名的三人組合。排名不分先後，盡量按照時間先後順序排列。

1.瑞典三劍客（AC米蘭 諾達爾+利德霍爾姆+格倫）

2.曼聯金球三人組（曼聯 丹尼斯·勞+查爾頓+貝斯特）

3.巴西藝術三盛 （巴西 濟科+法爾考+蘇格拉底）

4.法國中場鐵三角 （法國 普拉蒂尼+吉雷瑟+蒂迦納）

5.那不勒斯魔術組合（那不勒斯 馬拉多納+喬爾達諾+卡雷卡）

6.德國三駕馬車 （國際米蘭 馬特烏斯+布雷默+克林斯曼）

7.荷蘭三劍客 （AC米蘭 里傑卡爾德+古利特+范.巴斯滕）

8.巴爾幹新三個火槍手（貝爾格萊德紅星 薩維切維奇+潘采夫+普羅辛內斯基）

9.斯圖加特三駕馬車 （斯圖加特 巴拉克夫+埃爾伯+博比齊）

10.巴西3R（ 巴西 羅納爾多+羅納爾迪尼奧+里瓦爾多）

11.皇馬BBC （皇馬 C羅+貝爾+本澤馬）

13.巴薩MSN（巴薩 梅西+蘇亞雷斯+內馬爾）

14.紅軍三個火槍手（利物浦 薩拉赫+馬內+菲爾米諾）
