德甲多特欲擒領頭羊，西甲巴薩爭勝希望大，意甲尤文力爭全取3分

2019-12-17 446條評論 3,044次閱讀 625人點贊

專家推薦：

德甲

多特欲擒領頭羊

多特蒙德VS萊比錫 兩隊都打進歐冠16強。多特將對陣大巴黎，先主後客。萊比錫則面對熱刺，先客後主。相對來說，萊比錫抽了一個較好的簽。上賽季多特主客場「雙殺」萊比錫，領先後者10分獲得亞軍。本賽季萊比錫佔據聯賽榜首，球隊磨合很好。多特落後萊比錫4分，主場對陣領頭羊，是拉近雙方差距的好機會。亞盤開出主隊讓平半中水，多特有心理優勢，看好多特主場不敗。蕭少澤

西甲

國家德比高看巴薩

巴塞羅那VS皇家馬德里 本周四凌晨，西甲本賽季首回合「國家德比」在諾坎普上演。C羅遠赴尤文圖斯，使皇家馬德里斷了「魂」，上賽季總積分竟落後巴塞羅那19分。本賽季，在「二進宮」的齊達內調教下，本澤馬強勢率領眾將欲與巴薩爭鋒。兩隊現積分同為35分，名列前兩名。

巴薩隊員 新華社資料圖

梅西、本澤馬進球數也同為12個。論狀態，聯賽中的巴薩主場7戰全勝，進球30個，火力爆棚。皇馬8輪客戰4勝3平1負，勝率不高。論交鋒，兩隊近十次對戰中，巴薩6勝2平2負（主場1勝2平1負）。威廉希爾指數現為1.67 4.33 4.60，看好平局。但畢竟梅西、蘇亞雷斯與格列茲曼是主場作戰，而皇馬的阿扎爾、馬塞洛和J羅等人傷停難以上場，巴薩爭勝很有希望。勝負彩在平手盤條件下單選3，防1即可。比分推薦：2:1、2:0、1:1。飛揚

意甲

尤文力爭全取3分

桑普多利亞VS尤文圖斯 桑普多利亞本賽季聯賽表現慘不忍睹，目前只積15分排在第16位，僅高出降級區2分。其主場2勝2平3負，只打進5球。而尤文圖斯隨著國際米蘭的崛起，本賽季也沒有了在意甲一方獨霸的強勢，目前他們和國米同積39分，只以勝負關係的優勢壓過國米，排在榜首。就目前來看，尤文暫時沒有了歐冠的拖累，可以全身心投入到聯賽中，目前這個階段是搶分的絕佳機會。到目前為止的16場聯賽，尤文場場都有進球，攻擊力很有保障。本場威廉希爾初賠5.00 3.80 1.67。賠率對於尤文還是有一定的防範。就戰意而言，還是看好尤文做客竭力取分。推薦尤文客勝。 奕佳
