「復盤」成為壓倒埃梅里的最後一根稻草，阿森納主場負於法蘭克福

2019-11-30 0條評論 22次閱讀 0人點贊

在北京時間11月29日進行的歐聯杯小組賽中，近期表現十分糟糕的阿森納坐鎮主場酋長球場對戰前來挑戰的德甲球隊法蘭克福。從近期表現來看，英超豪門的日子十分不好過，最近各項賽事6場比賽不勝，聯賽13輪比賽戰罷3勝6平3負僅積18分排名第8，進球18個上半區最少，失球19個上半區最多，攻防兩端都不能令人滿意；法蘭克福方面，由於上一賽季幫助球隊獲得聯賽第7的多名主力球員離隊，他們的日子同樣不太好過，最近4場比賽1勝3負的表現令人失望。

首發陣容

阿森納方面，主教練埃梅里排出了一套並不常見的陣型——343。3名前鋒分別是在之前的小組賽中表現十分出色的35號左邊鋒馬蒂內利、中鋒14號球隊新任隊長奧巴梅楊、77號年僅18歲的小將薩卡；4名中場分別是21號右邊翼衛錢伯斯、3號左邊翼衛蒂爾尼和28號威洛克以及由於辱罵球迷事件被雪藏多日的前隊長扎卡；拖在最後的分別是20號前德國國家隊成員穆斯塔菲、5號從大黃蜂多特蒙德來投的帕帕和23號前切爾西球員大衛路易斯。總體來說，主教練埃梅里對於本場比賽還是十分重視的，雖然對個別位置進行了輪換，但是關鍵中軸線上仍然由球隊的絕對主力坐鎮，目的很明確就是力爭在主場獲得一場酣暢淋漓的勝利。

法蘭克福方面，主教練同樣進行了變陣，與歐聯杯小組賽上一輪對陣標準列日不同地在中場增加了一名前腰的位置，意圖加強在中前場對皮球的控制。3名中衛分別是20號日本國家隊前隊長長谷部誠、19號阿根廷人亞伯拉罕和13號奧地利國家隊絕對主力欣特雷哥；在他們身前分別是10號塞爾維亞國腳科斯蒂奇、5號費爾南德斯、8號瑞士人索烏、24號現任德國國際奧達科斯塔以及擔任前腰的15號日本年輕球員15號鐮田大地；頂在球隊最前面的2名前鋒是33號從意甲豪門AC米蘭租借而來的安德烈席爾瓦和39號25歲的葡萄牙人帕先西亞。法蘭克福排出了他們的全主力陣型，在仍然落後標準列日1分的情況下，獲得平局是他們的最低要求。

阿森納主打邊路進攻，法蘭克福準備不足漏洞百出

埃梅里排出343的陣型就是為了加強邊路進攻，3名中衛中的存在可以讓2名翼衛大膽地沿邊路向前壓上參與進攻，同時中後場人員增多的陣型使得2名邊鋒薩卡和馬蒂內利幾乎沒有任何的防守壓力，在防守階段他們可以仍然停留在中線附近，儲備足夠的體力用於不斷衝擊對手的防線。這一戰術布置在上半場取得了極佳的效果，阿森納多次通過犀利的邊路進攻直接威脅到了對方的大門。

比賽僅僅開始10分鐘，阿森納通過邊路進攻獲得了一次破門良機。通過耐心的傳導皮球到了左邊鋒薩卡的腳下，憑藉著出色的腳下技術和高人一等的超強爆發力，薩卡的突破吸引了對手3名球員的包夾，在對手完全封堵住傳球路線之前送出了效果極佳的傳中。由於右側的中後衛已經被吸引到大禁區附近，法蘭克福在禁區內僅僅有3名球員爭頂並且人員分布極其不合理，造成阿森納內收到中路馬蒂內利完全無人防守。只可惜威洛克的頭槌攻門沒能打在門框範圍之內。阿森納前場球員的個人能力都十分突出，長時間英超聯賽的鍛煉使得他們不僅腳下技術優良並且具備了不錯的身體對抗能力，在歐洲賽場上即使是年輕小將的表現也十分驚艷，對抗幾名防守隊員不是問題。

比賽繼續進行，阿森納後場開出大腳，中鋒奧巴梅楊回撤頭槌後蹭給薩卡，法蘭克福的幾名後衛雖然身材高大但是速度較慢，薩卡的速度優勢十分明顯直接獲得了單刀球的機會，只是由於守門員的出色發揮才沒能打破比賽的僵局。從上面的圖片中可以看出，阿森納在進攻中投入的人員多而有序，包括從後腰位置上插上的威洛克，共有4名球員直插對手腹地（即使在進攻端弱側的馬蒂內利也在全速插向對方禁區）。反觀法蘭克福方面，中場線與後防線的距離拉得非常大，後防線3名中衛不得不面對4名速度極快的攻擊手，他們的弱點完全暴露在了對方的戰術布置之下。

上半場補時階段，阿森納終於沒再繼續浪費機會。馬蒂內利邊路突破低平球傳中，從前點位置內收到中路的薩卡踢空，後點的奧巴梅楊在完全沒有防守的情況下低射打入1球。法蘭克福的防守雖然在人數上佔據了絕對的優勢，但是並沒有貼近搶點的薩卡和奧巴梅楊，給了對手從容起腳的機會。在英超中進攻能力只能算作是中等偏上的阿森納僅僅通過3-4人的進攻小組就在禁區內獲得了如此的良機，德國球隊的防守能力實在不敢恭維。

下半場法蘭克福大舉反擊，阿森納人心渙散無人防守

進入到下半場，比分落後的德甲球隊一改上半場的頹廢表現，突然迸發了極強的戰鬥力：防守端全場逼搶，將阿森納的進攻擊毀在構建階段；進攻端主打對手肋部空當和爭奪二點球，最終由日本球員鐮田大地梅開二度逆轉獲得了勝利。

比賽進行到第54分鐘，法蘭克福發動進攻。阿森納在後場囤積了7名球員進行防守，僅僅在右邊路就有4名球員盯防對手的2名進攻球員，但是下半場替補上場的後腰法國小將貢多齊完全失去了自己的位置，造成了肋部空當十分巨大。法蘭克福小將鐮田大地聰明地發現了這一漏洞，他迅速的從原來所處小禁區前點的位置跑到了對方防守的真空地帶，在接到隊友的傳球稍稍調整之後迅速起左腳打遠角得手。阿森納幾名防守球員完全沒有了鬥志，站在原地觀看對手的跑位卻完全沒有做出任何的反應。

僅僅8分鐘之後，法蘭克福開出角球，皮球被阿森納球員頂出禁區，站在大禁區外的鐮田大地在完全沒有任何防守的情況下低射破門。阿森納控制住了角球的第一點，但是這些身經百戰、身價不菲的球員全部沒有意識到第二點的重要性，把對手埋伏在禁區外的3名球員放得一乾二淨。賽後，上面圖片中在大禁區內最外側的厄齊爾遭到了廣泛的批評，他不僅沒有站在任何一名防守球員的身邊並且沒有第一時間的沖向獲得第二點準備起腳的鐮田大地，對於本次防守的貢獻度等同於0.

總結

從賽後的技術統計來看，阿森納仍然在控球率、傳球成功率、進攻次數、危險進攻次數上佔據了一定的領先優勢，但是在最關鍵的射正次數上卻處在了落後的位置。這說明阿森納空空將球權佔據在腳下，但是面對對手的防守並沒有好的辦法威脅到球門，反倒是法蘭克福效率極高的反敗為勝。

本場比賽的失利成為了壓倒埃梅里的最後一根稻草，僅僅幾個小時之後阿森納溫格時代結束的第一任主教練埃梅里被俱樂部炒了魷魚，還被網友扣上了「阿森納莫耶斯」的帽子。回顧他入主酋長球場的18個月時間，埃梅里做錯了太多事情，他沒有為球隊引入一名可以帶領球隊的球員、儘管殺入了歐聯杯決賽卻遺憾的輸給了切爾西、將球隊多年以來的中場發動機厄齊爾放在了板凳上、反覆放棄和啟用即將離開球隊拉姆斯、不斷否定自己的決定、任人唯親地使用頂級聯賽驚艷極其匱乏的法國小將貢多齊、處理不好與媒體之間的關係等等。埃梅里真的不適合阿森納，分開對於雙方來講都是最好的選擇。
