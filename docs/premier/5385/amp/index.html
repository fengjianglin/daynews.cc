穆里尼奧第一簽！熱刺歐冠晉級後鎖定新援 為他不惜跟拜仁爭

天天要聞
3 週 ago

歐冠小組賽第五輪，坐鎮主場的熱刺逆轉戰勝奧林匹亞科斯。憑藉這場讓二追四的贏球，穆里尼奧率隊提前一輪晉級歐冠淘汰賽。如此強勢的表現，讓熱刺球迷看到復甦希望。值得一提的是，穆里尼奧因此成為歐冠歷史上首位率領六支不同球隊晉級淘汰賽的主教練。眼見於此，受到鼓舞的熱刺管理層已經著手為穆里尼奧引進強援，甚至不惜和拜仁等歐冠豪門搶人。

據英國媒體報道，熱刺正在密切關注沙爾克的年輕門將努貝爾，23歲的他被視作德甲新諾伊爾。還別說，如今熱刺最有隱憂的位置就是門將。洛里今年已經33，競技狀態隨時可能下滑。作為替補門將的加扎薩尼，雖然在對陣利物浦的比賽里一戰封神，但是下底技術明顯硬傷。

關於這一點，之前和布萊頓比賽里的丟球就是最好佐證。再加上防守起家的穆里尼奧對門將位置格外的看重，熱刺還真是應該考慮買個門將了。然而，想到埃里克森、托比和維爾通亨等人可能離隊，熱刺屆時需要補強的位置太多。哪怕列維同意大把花錢補強，恐怕也騰不出多少資金來買門將。所以，預算不多的熱刺盯上了努貝爾。由於至今都沒和沙爾克續約，被譽為新諾伊爾的他明年夏天合同到期。

不用想也知道，熱刺這是打起免簽的主意了。還別說，被調侃為轉會界活雷鋒的沙爾克這些年免簽送出大批優秀球員，如今在英超打拚的就有厄齊爾、馬蒂普等人。然而，熱刺這次想要免簽可不容易。煥然一新的沙爾克管理層，準備全力挽留努貝爾。為此，先是力排眾議地讓他當隊長，隨後又開出了漲薪十倍的續約合同，可謂誠意滿滿。

比起全力留人的沙爾克，拜仁的誘惑更要命。眾所周知，德國球員大多渴望加盟拜仁，至今沒續約的努貝爾也不例外。再加上馬特烏斯等一眾名宿的公開發聲，已讓這樁轉會無限接近成功。所幸的是，穆里尼奧的入主讓熱刺有後來居上的趨勢。

想想也是，成績和薪資待遇等綜合比較的話，熱刺和拜仁相差不多。穆里尼奧的加成作用，絕對是熱刺搶人的一大籌碼。畢竟葡萄牙人曾先後率領波爾圖、國米和皇馬問鼎歐冠，而且非常擅長調教中後場的防守球員。跟著穆里尼奧的話，絕對有助努貝爾的快速成長。所以，熱刺很有希望拿下這位德甲冉冉升起的新星。
