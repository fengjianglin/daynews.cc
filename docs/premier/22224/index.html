又傷了！皇馬鍾情索帥無意，還當自己世界第一中場啊，球迷受夠了

2019-12-18 438條評論 1,554次閱讀 522人點贊

博格巴是幾內亞裔法國足球運動員，博格巴於2009年加盟曼聯青訓營，足球天賦出眾的博格巴很快就被各大俱樂部給盯上了，於2012年夏窗被意甲豪門尤文圖斯簽下，效力於尤文圖斯期間博格巴表現出色，不僅拿下了2013年的歐洲金童獎，還多次入圍了意甲最佳陣容中。

博格巴在2016年的時候以8900萬英鎊的身價重返老特拉福德球場，這一筆高昂的轉會費也是打破了足壇里的最高紀錄，他的未來也被球迷無限的看好。

博格巴的控球能力出眾，視野也非常寬闊，作為一名中場球員博格巴絕對是其中的佼佼者，實力強大的博格巴也迅速的成為了曼徹斯特聯中的絕對核心，入選法國國家隊後，也是隨法國隊拿下了2018年俄羅斯世界盃的冠軍，之前有不少球迷都認為博格巴是現役的世界第一中場球員。

論腳下技術博格巴的確是一名出色的球員，但是心氣高傲的博格巴經常在場上託大，發揮得好的話博格巴確實能夠送出幾腳好球，如果狀態不佳那博格巴絕對是曼聯隊內丟球最為嚴重的一名球員。

博格巴強調自己的球權但又容易將球權丟失，當時的主帥穆里尼奧不得不給他安排兩個「保姆」型中場來預防他的丟球，但博格巴不僅在球場上高傲至極，球場外這位法國國腳也不會消停。

穆里尼奧是不能接受自己的球員耍大牌的，然而博格巴就是偏偏對著干那一位，最後是鬧得更衣室里烏煙瘴氣，曼聯高層無奈解僱了穆里尼奧，而留下的是博格巴。

博格巴的大牌行為屢見不鮮，缺席訓練、不願上場比賽、私生活混亂等，最令球迷驚訝的是，這位法國國腳經常詐傷。

前不久博格巴由於有傷在身無緣曼聯的各項賽事，在療養中的博格巴參加哥哥的結婚典禮上熱舞、與NBA球星傑米-巴特勒一起在美國打籃球，球迷都以為這位中場大將的傷勢恢復得差不多了，但不料的是，曼聯主帥索爾斯克亞在接受媒體採訪中表示博格巴又傷了還不能復出。

習慣詐傷的博格巴已經得不到球迷的信任了，不少球迷都表示稱博格巴這次極有可能又是詐傷，要知道博格巴最近代表球隊出戰的時候還是9月份，他已經缺席了很長一段時間了。

夏窗時博格巴與西甲豪門皇馬的聯繫較為密切，當時也是鬧著要轉會離開，並表示不會再為曼聯踢球，球隊拿出的續約合同也被其給拒絕了。

據悉，皇馬想用隊內的前鋒貝爾加現金的方式來換取博格巴，博格巴一直都是皇馬的引援目標，皇馬希望在冬窗能夠拿下這位法國國腳。

但是博格巴於曼聯的合同還有18個月，主帥索爾斯克亞也表示不會為博格巴放行，儘管博格巴無意續約。

索帥也準備物色兩名中場球員來倪補博格巴合同到期離開後的空缺，主帥是不願意放，但是曼聯球迷是巴不得博格巴感覺離開，有球迷稱不想為曼聯效力就趕緊離開吧，更有球迷稱其為球隊毒瘤，可見紅魔球迷已經受夠了。
