「12.6」今晚五大聯賽，國米上意甲榜首

天天要聞
2019-12-06

十二月已經來臨，隨著歐預賽告一段落，歐冠小組賽到達尾聲，五大聯賽的比賽也越來越多了，抓住2019年的尾巴，好好乾一波。

周五001 澳超 墨爾本城VS珀斯光榮

墨爾本城新賽季7場比賽5勝1平1負暫列次席，戰績不俗，得益於今年夏天針對中場的補強，新援諾尼、卡布雷拉和盧納提升了進攻端的創造力，主力射手麥克拉倫也發揮出色，7輪聯賽打進9球。球隊每場比賽都保持著場均2球的效率。

珀斯光榮作為上賽季常規賽冠軍和總決賽亞軍，本賽季的表現讓人大失所望，目前以1勝3平3負的成績暫列積分榜第11位，後防主力戴維森離隊對球隊防守影響極大。鋒線上卡斯特羅和哈羅德集體低迷，也是球隊失利的原因。

推薦：001勝，進球數34球。

周五013 德甲 法蘭克福 VS 柏林赫塔

法蘭克福上輪客場1比2被美因茨逆轉，遭遇兩連敗。但是主場優勢明顯，新賽季7個主場4勝2平1負，包括大比分擊敗拜仁慕尼黑、勒沃庫森等豪強，含金量十足。本輪主場必定全力反撲搶分。不過主力中鋒多斯特、主力門將特拉普傷缺，主力中衛阿布拉姆紅牌停賽，對後防影響巨大。

柏林赫塔上輪主場1比2負於多特蒙德隊，聯賽中遭遇四連敗。本賽季8個客場比賽中取得2勝2平4負，取分能力不強，主力進攻中場萊基傷缺，主力門將賈施泰紅牌停賽，同樣對後防有一定影響。

推薦：013勝

周五015 意甲 國際米蘭VS羅馬

國米各項賽事取得四連勝，聯賽中反超尤文圖斯重新佔據榜首，歐冠小組賽還剩最後一輪主場迎戰巴薩，只有將其擊敗才能確保進入淘汰賽，壓力依然很大。雙線作戰依然考驗球員耐力和球隊替補深度。

羅馬上輪客場3比1擊敗維羅納，各項賽事取得三連勝，雙方交戰勢均力敵，近10次作客取3勝4平3負。跟國米一樣，羅馬下周依然有歐戰任務，歐羅巴小組賽最後一輪主場迎戰沃爾夫斯貝格，取勝才能晉級16強。所依本場兩隊打發可能會有所保守。

推薦：015平讓平，進球數23球。

二串推薦：013勝，015平讓平，sp3.16，參考！
