米蘭挖槍手雙將？愛美麗：我只關心貢親王過得好不好

2019-11-14 0條評論 18次閱讀 0人點贊

阿森納經歷了一波5輪不勝，但埃梅里的帥位倒是異常穩固，他不僅得到了來自高層的支持，甚至還取得了不錯的試驗成果——「親兒子」貢多齊現在成為了絕對的中場核心！

本賽季12場英超聯賽，貢多齊首發並幾乎打滿全部時長，目前取得1次助攻。上賽季的另外兩大中場，托雷拉和扎卡，卻各自遇到了困難，他們在阿森納的地位一落千丈，甚至於朝不保夕：

扎卡與球迷發生衝突，現在已被剝奪了隊長袖標，而且由於全家人都遭受了「言語暴力」，扎卡在酋長球場估計是呆不下去了。27歲的瑞士人已經在米蘭市尋找住宅，明年1月份的冬窗，或許就是阿森納前隊長扎卡離開的時候。瑞士記者也稱：「轉投米蘭更有利於扎卡的職業發展。」

托雷拉，雖然與槍迷關係融洽，但無奈他不受埃梅里的待見啊！本賽季至今，托雷拉只在英超首發了4場比賽，場均時間還不到40分鐘，這絕不是托雷拉理應受到的待遇——以托雷拉的實力，他就是槍手陣中的「白坎特」，托雷拉的防守預判和跑動能力，足以讓槍手中場提升一個檔次，可惜埃梅里就是不用。

《都靈體育報》消息稱，23歲的烏拉圭人托雷拉，也很有可能在未來1-2個轉會窗內離開阿森納，說不定會和扎卡一起加盟AC米蘭？米蘭方面，由於凱西的離隊已經提上日程，因此他們也迫切希望簽下新援，來改善本賽季糟糕的成績。

如果扎卡和托雷拉雙雙離隊，最大的受益者必然是貢多齊。20歲的法國中場小將在埃梅里麾下確實成長得很快，本賽季他場均能送出1.83次搶斷、1.25次攔截和1.5次解圍，12場比賽9次關鍵傳球，87%的傳球成功率，這些數據都在把貢多齊指向未來中場領袖的地位。

然而問題是，貢多齊起來了，阿森納的成績卻毫無起色，甚至可以說埃梅里是把本該用於提升阿森納整體的精力，全花費在了培養貢多齊了？要知道阿森納中場們多數是踢邊路的，比如邊前腰厄齊爾（打中路效果不理想）、邊前衛奈爾斯、邊鋒+邊前衛薩卡等，中路球員只剩下塞瓦略斯這個租借來的球員，以及20歲的威洛克，還都是前腰。

是的，看來埃梅里下一個目標就是培養威洛克了？貢多齊+威洛克的超新星組合也挺值得期待的，就是不知道以埃梅里輸球的速度，他能否在帥位上堅持到這兩人成長起來呢？
