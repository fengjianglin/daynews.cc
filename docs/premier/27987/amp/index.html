21日大嘴足球離散：歐洲聯賽接近半程，卡利亞里客場搶分

周五早場繼續紅單，法乙卡昂0-0悶平克萊蒙特，平局博膽順利打出；早場公推最近12中12連紅，近26中22狀態穩定！周五晚場遭遇明燈，全部8場8中4，但精選主推3場明燈，索肖、特溫特和卡爾斯魯厄均無緣打出！

周六足壇賽事豐富，午間天皇杯和澳超5場亞系早場，照例作為周五晚場加送福利！晚間五大聯賽繼續進行，本周末聯賽之後，除英超外的歐陸聯賽，將陸續進入冬歇期，本周末將是2019年諸侯一個足球盛宴！早間檔分享歐洲聯賽8場指數，另附意甲烏迪內vs卡利亞里解析！周六晚場2檔推送：晚間21:30分享上半夜精選8場指數和方案；23:00分享下半夜精選8場指數和方案，歡迎搜索關注大嘴聊球號獲取！

[關於離散] 競技體育分析指數，也就是各家歐指分歧度。基本原理：百家歐指平均值與每家指數差值的絕對值的求和。離散度小則各家觀點接近，進而賽果易打出；離散度大反之意見分歧大，賽果不易打出。又因各方因素影響下指數變動，離散在特殊時刻具有更高參考價值，切勿盲目參考運用。換句話說，離散只是參考指數，而非結論性數據!以下是各場數據匯總(數據截取至11:00)：

周六012 意甲 烏迪內斯vs卡利亞里

比賽時間：12月21日22:00

個人觀點：烏迪內斯負（-1）

比分預測：1-1/0-1

意甲第17輪爭奪，本輪聯賽之後，意甲將進入為期2周半的冬歇期。本賽季意甲競爭異常殘酷，過去8年尤文一家獨大的局面正在被瓦解，目前國米正對尤文霸主地位發起強烈衝擊。而本場對陣雙方狀態天壤之別，烏迪內斯逐漸淪為保級隊，而卡利亞里本賽季成為意甲半程最大黑馬。

烏迪內斯上賽季聯賽排第12位，本賽季球隊上半程發揮欠佳，前16輪聯賽4勝3平9負，目前僅僅拿到15個積分，排名意甲第17位，僅僅領先降級區2分，保級形勢相當嚴峻。球隊前8輪主場3勝2平3負，主場表現基本正常。球隊上個主場爆冷戰平那不勒斯，但本周中客場輸給尤文。本輪烏雞一周雙賽，體能狀況是一大疑問。

卡利亞里上賽季意甲僅列第15位，本賽季球隊強勢爆發，前16輪聯賽8勝5平3負，目前那大29個積分，力壓亞特蘭大、米蘭和佛羅倫薩等傳統強隊，高居意甲第5位，落後多賽一輪排名第4的羅馬6分，球隊甚至有望衝擊下賽季歐冠名額。值得關注的是，卡利亞里前7個客場3勝4平保持不敗，和國米一起成為意甲僅有2支客場不敗隊伍。

雙方上賽季2回合交手，烏迪內斯主客場雙殺對手，最近6次交鋒，兩隊各勝3場沒有平局，而烏迪內主場近6次對陣卡利亞里4勝1平1負佔據絕對上風。本場雙方賬面實力對比來看，烏迪內全隊身價1.4億歐元，稍稍落後於對手1.9億歐。初始勝平負指數2.45 3.29 2.82，標準232指數格局下，優勢稍稍偏向主隊。即時指數波幅較小，整體指數向主隊小幅傾斜，目前均值2.44 3.29 2.92。不過離散指數一邊倒傾斜，當前主勝離散18.99高企狀態，而且有繼續上漲趨勢；平局指數穩定，客勝指數中低位，指數對主隊相當不利。綜合分析來看，本場看好卡利亞里客場不敗，本場重點關注平局打出。
