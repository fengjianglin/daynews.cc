有一種歐聯杯叫放棄，拉齊奧徹底決定“窩裡橫”

天天要聞
2019-11-28

▲盧卡庫-勞塔羅組合終於在歐冠賽場也發飆了。

■■■

昨日，歐冠小組賽第5輪全部戰罷，絕大多數比賽還是真刀真槍打得激烈，今日轉向歐聯，就沒那麼刺激了。儘管歐足聯也提升了歐聯杯的獎金，但這個賽事對不少球隊尤其是英超、意甲的強隊缺少吸引力。

以曼聯為例，今日奔赴哈薩克首都，索爾斯科爾只帶了18人，而這份名單中，有10人沒踢過一線隊。固然，這是曼聯已然拿到小組出線名額下的舉措，但作戰態度，可見一斑。

當然，這不代表曼聯就打算放棄比賽了，年輕是把雙刃劍，要知道，曼聯本賽季英超的所有進球全部來自於U23球員，所以，比賽還要具體分析。

今日公推，自然來自歐聯杯賽場，拉齊奧VS克盧日，在聯賽豪取5連勝的藍鷹軍團，是否還會「分心」歐聯賽場？

以下是未來一周的賽程預計，供關注包周版（每周不少於35場推薦，相比單場價格至少打了2.8折，絕對是超值+穩健之選）的用戶參考。

拉齊奧今年是徹底被義大利球迷認作是窩裡橫了，前兩個賽季，小因扎吉也是在歐聯杯小組進行大幅度輪換，但都能拿到出線資格，且至少都能拿到3個勝場，多少還能為意甲在歐足聯積分做點基本貢獻，但本賽季迄今1勝3負，晉級希望已然飄渺。而目前的形勢，俱樂部似乎也不打算為這一點希望抗爭了，因為聯賽更「值得」去拼。

上輪客場補時絕殺薩索洛，拉齊奧斬獲聯賽5連勝，積分穩居前三，爭奪歐冠席位勢頭良好，而12月22日去沙特打義大利超級盃，所以聯賽賽程會有影響，球隊根本不希望在受歐聯杯牽絆，會一心主攻意甲了。

小因扎吉嘴上說得好聽，「至少要為榮譽而戰」，但排兵布陣卻很誠實。今晚首發陣容恐怕只留下三名上周末首發成員，前鋒科雷亞搭檔20歲小將阿德卡尼，來自利物浦U23，本賽季在拉齊奧總共只出場過14分鐘。中場留下右邊衛拉扎里，防線仍會首發核心阿切爾比。

▲克盧日主帥可是中超老熟人。

歐指主勝1.63，仍在上調趨勢中，亞洲指數主讓0.75仍不足85水，上盤會有較高的熱度。但拉齊奧這套替補陣容本季在歐戰表現未獲證明，而羅馬尼亞領頭羊克盧日實力不弱，且有豐富歐戰經驗，上輪聯賽也消耗不大，而此戰拿分即可提前晉級，客軍以逸待勞，值得支持。
