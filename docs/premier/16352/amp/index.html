國米還能拼湊首發嗎！期盼的救命新星演奇葩首秀 12分鐘摸不著球

天天要聞
2 天 ago

國米1-1戰平佛羅倫薩後，意媒感嘆：這個夜晚，足球之神沒有將一點仁慈，用於眷顧困境中的國際米蘭。

進球，因為毫釐之間的越位被吹。

盧卡庫精彩的扛人轉身獲得良機，但「打不進比較難」的近距離爆射卻打在門將腿上。

傷停補時階段更是讓弗拉霍維奇上演一次「梅西附體」的神級奔襲爆射破門。

3分變1分，揮霍了2分領先優勢，並不是這支歐冠出局後士氣不振、身體疲憊、半數主力卧於病榻的國米今晚僅有的苦惱。真正讓國米憂慮的，也是巴斯托尼傷退，勞塔羅、布羅佐維奇染黃下輪停賽，國米戰損名單繼續擴大！

賽前我們就指出，國米6將傷停（森西、巴雷拉、阿薩莫阿、坎德雷瓦、加利亞爾迪尼、桑切斯），這個戰損名單甚至可能擴大到兩位數，因為有4名球員再拿一張黃牌，就將下輪停賽。

本戰，布羅佐維奇第27分鐘背後對卡斯特羅維利戰術犯規染黃。

勞塔羅在第76分鐘，與門將拼搶二分之一球時沒有及時起跳掃到門將面部，染黃。

兩人將雙雙在下輪停賽，再加上巴斯托尼完場前傷退，這讓孔蒂無比憂慮，現在的問題已經不是他應該如何選用狀態更好的球員，而是即便是有一個算一個全部用上，能否拼湊的出首發陣容也不好說的問題！

顯然，孔蒂對於形勢有著非常清晰的認識，所以第84分鐘，國米大帥同時換上兩人，波利塔諾和阿古梅——波利塔諾取代勞塔羅出任二前鋒，阿古梅出任三個中前衛居中的位置，可以視作是孔蒂為下一戰出場的球員做一次預演。

但兩名球員的發揮都遠低於預期——其實理智點的球迷，預期本來就不高。

13分鐘比賽，波利塔諾8次觸球，僅有2次有效傳球。

腳下對抗，3次拼搶全部失敗。

頭上對抗，2次拼搶全部失敗。

傳中成功率是0。

長傳成功率是0……

這樣的狀態，想要取代勞塔羅，除非接下來一周他的狀態出現奇蹟般的提升。

17歲新星阿古梅是國米球迷幾個月來都在期待的新星，法國國少的隊長此戰終於上演了遲到的首秀，但阿古梅的表現，證明了孔蒂一直不用他確實是有原因的。阿古梅踢了13分鐘比賽，前12分鐘，不要說傳球、過人、突破、對抗，他竟然連一次觸球都沒有。

出任三中場居中位置的阿古梅，踢的是他在法國國字型大小球隊比較熟悉的位置。

小夥子確實很拼，四處奔跑，毫不惜力。

但是效果很差，12分鐘雙方拉回拉鋸，可是阿古梅卻連一次皮球都摸不著。

直到傷停補時最後一分鐘也就是比賽第96分鐘，小將終於有了他此戰第一次——也是唯一一次的觸球。不需要太有同情心，你就能體會那些想要拍下阿古梅比賽用圖的攝影記者是多麼苦惱。

過去一個多月，很多球迷，以及義大利媒體，都在呼籲孔蒂：既然中場人荒，為何不啟用阿古梅？

批評家們更是跳著腳的痛罵：看看巴斯托尼！看看埃斯波西托！看看庫魯塞夫斯基！誰說新星就不能用！區區孔蒂一個二流主帥，就這點用人魄力，憑什麼拿比我們這些懂球牛人、在野神帥高出幾百上千倍的薪資？

孔蒂給出了他的解釋。

他一開始指出：阿古梅和埃斯波西托、巴斯托尼甚至是皮羅拉不同，那幾位新星很適應義大利足球的踢法和節奏，欠缺的只是一線隊比賽經驗。阿古梅雖然才華橫溢天賦傑出，可他不但欠缺成年比賽經驗，也對義大利足球的踢法完全陌生。孔蒂表示他非常信任阿古梅，但小夥子需要更長時間，這急不來。

他的說法並不被輿論認可。說的次數多了，孔蒂也難免沒了耐心，他後來乾脆說：那些因為國米不使用年輕人而批評的人，並不是真的為球隊好！

至少從阿古梅的首秀來看，孔蒂的判斷是有道理的。

小將確實對意甲的踢法完全陌生，如果是一個中鋒或門將，十幾分鐘摸不到球，這不算罕見。但作為一個中場樞紐位置的球員，這麼長時間觸不到球，就說明他還不太清楚自己應該出現在什麼位置，應該怎樣跑動，怎樣上搶，怎樣撤防，怎樣對抗，也就是說，他還沒有為踢意甲做好足夠準備。
