奧巴梅揚啞火，槍手如何突破困境？

天天要聞
2019-11-12

最近一段時間，槍手面臨諸多困難，目前已經在最近四輪聯賽中沒有取得過勝利，上一次聯賽獲勝還要追溯到10月6日主場1比0小勝伯恩茅斯的比賽中。而球隊的當家射手奧巴梅揚則在最近的五輪聯賽中只取得一粒進球，與開賽初期的火熱狀態形成鮮明對比，雖然依然排在英超射手榜的第四位，但接下來因為國際比賽日，奧巴梅揚要回加彭參加比賽，能否儘快恢復狀態令人擔憂。

同時，在槍手與水晶宮隊的比賽中，槍手前隊長瑞士中場扎卡因為表現不佳在被替換下場下遭到主場球迷噓聲，憤怒的扎卡把隊長袖標重重摔到地上，並對球迷爆了粗口，這更加激發了球迷對他的不滿。隨著事態的持續惡化，俱樂部剝奪了扎卡的隊長資格，自此之後沒有讓其繼續上場比賽。截至目前，這一事件仍然在繼續發酵，而根據瑞士媒體的報道，扎卡已經與阿森納徹底鬧掰，他將在明年1月轉會加盟AC米蘭，並且已經開始在米蘭城尋找房子。此次事件之後，奧巴梅揚被俱樂部及主教練埃梅里選定為槍手新隊長。

新隊長上任的奧巴梅揚雖然在之前與狼隊的比賽中打進一球，但無奈球隊依然在主場與對手戰平，沒有如願取得3分，最近的一輪比賽更是以0比2敗走狐狸城。面對著狀態不佳的奧巴梅揚，困難重重的槍手主教練埃梅里如何帶領球隊突破困境，是其能否在帥位上坐得穩的關鍵所在，好消息是，俱樂部高層放出消息，依然信任埃梅里，他依然可以集中精力破解槍手困局。

本周的國際隊比賽日結束後，槍手將在11月23日主場迎來英超第13輪聯賽，對手是排在積分榜倒數第二位的南安普敦隊，聖徒已經連續七輪聯賽沒有取得勝績，屆時將是槍手擺脫目前困境的最好時機。
