又一個歐洲大牌教練想來中國執教！直言想體驗不同文化，你信嗎？

天天要聞
2019-12-22

眾所周知，自從中超聯賽開啟金元模式，我們的中國俱樂部就有了足夠的資金，腰板也硬起來了。於是乎經常會在國際轉會市場上掀起風浪，甚至和歐洲五大聯賽的豪門搶人。有時候搶的是外援，也有時候搶的是教練團隊。所以這些年我們在中超不僅能看到世界級的大牌外援，還能看到冠軍級別的教練團隊。比如廣州恆大的里皮團隊和斯科拉里團隊，比如上海上港的博阿斯團隊和埃里克森團隊，比如魯能的馬加特團隊，比如現在大連一方的貝尼特斯團隊等等，哪個不是歐洲頂級的教練團隊，所以中超聯賽就像是個明星匯聚場。

當然了這些大牌教練能來，首先一點就是他們能獲得比歐洲高得多得多的薪資，這是對於他們最大的新引力。而且外教來中國執教就相當於扶貧，會受到球員和球迷的信任和力挺，甚至沒有質疑的聲音，所以他們的執教環境也是一級棒。所以這些都是外教願意來中國的原因。

不過這些年是來了不少外教，但是成功的卻是寥寥無幾。大部分最終都是還要本土教練最終給善後，所以來中國容易，想要在中國足球取得一些成績和成就還是很難的。但是這一點往往被很多外教忽視掉了，因為更多的人還是無法拒絕高薪的誘惑的。

最近又有一位歐洲名帥被中國足球吸引了，公開表示自己願意去中國執教。而他給出的理由不是金錢，而是想要了解不同的文化，但是你信嗎？據西班牙媒體「be soccer」報道，荷蘭足球名宿斯塔姆表示他不排除去中國執教的可能性。球員時代，作為世界最為頂級的中後衛之一，斯塔姆曾效力埃因霍溫、曼聯、拉齊奧、AC米蘭以及阿賈克斯等俱樂部。退役後的斯塔姆，先後執教雷丁、茲沃勒、費耶諾德。

10月底辭去費耶諾德的帥位後，斯塔姆目前正在尋找新的工作。在接受採訪時，他表示不介意去中國或是美國等國家隊執教。「我真的很想去美國執教，我喜歡那兒的文化。中國似乎也是一個很好的選擇，去那裡可以了解完全不同的習俗。最近，我收到了很多荷蘭以外的邀請。」

也就是說，如果有來自中國的邀請，斯塔姆可以優先的考慮，因為中西差異很大的文化深深地吸引著他。不過我們還覺得，之所以他願意來中超，也是看在更高的薪資的份兒上。當然了我們並不會拒絕任何名帥來中國闖蕩。但是我們更希望，他們來之後能夠幫助中國足球進步，而不是高高在上，帶不動最後倉皇而逃，這對於雙方都不好。

一個好的外教要能夠充分地適應和了解中國文化，只有做到這一點，他們才有可能成功。你怎麼認為呢？你認為斯塔姆有機會來中超並取得成功嗎？
