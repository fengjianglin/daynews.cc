安切洛蒂：我知道埃弗頓人有多想擊敗利物浦

2019-12-24 54條評論 1,547次閱讀 233人點贊

卡爾洛-安切洛蒂在他作為埃弗頓主帥，出席的首個新聞發布會上表示，擊敗同城死敵利物浦贏下默西塞德郡德比，並非是「不可能完成的任務」。

北京時間1月4日，19-20賽季英格蘭足總杯第3圈的比賽，埃弗頓將做客挑戰利物浦。

雙方的上一次交手就是在本月初，當時埃弗頓在客場2-5慘敗於利物浦，那場比賽成為了壓倒駱駝的最後一根稻草，太妃糖第二天就宣布了主帥馬爾科-席爾瓦下課的消息。

實際上，自2010年以來，埃弗頓還沒有在任何比賽中擊敗過利物浦，本賽季紅軍在英超聯賽中更是勢如破竹，然而，安切洛蒂本賽季就曾率隊擊敗過克洛普麾下的這支利物浦，歐冠小組賽中那不勒斯主場2-0戰勝了利物浦。

「埃弗頓人知道我對利物浦的戰績，他們會對此感到高興的，」安切洛蒂在與伯恩利的賽前新聞發布會上說道。

「（那不勒斯）本賽季擊敗了利物浦，（默西賽德郡德比）在這裡是非常重大的比賽，也會給我們很大的動力。」

「這感覺很好。說實話，我和尤爾根-克洛普之間的關係真的非常好。和他們踢比賽，總是會令讓人很興奮。

「當然，我贏過他們，而且他們並不習慣輸球。對我來說（那不勒斯贏利物浦那天）是美好的一天。我知道埃弗頓人有多想擊敗利物浦。」

安切洛蒂還被問到為何選擇執教埃弗頓：

「吸引我的是俱樂部的傳統和歷史，這是英格蘭最大的球隊之一，」安切洛蒂說道。

「我之前管理的是幾支頂級的球隊，確實是這樣。但我執教巴黎聖日耳曼的時候，那裡的項目非常棒，那吸引了我，我想埃弗頓這裡，也和巴黎一樣，具有雄心。

「但我並不是只執教過頂級球隊。我生涯初期執教的帕爾馬就並不是一支頂尖的球隊。我喜歡那種有雄心，並對如何取得成功有清晰想法的球隊。在這裡，我已經去過訓練場了，那是一個非常非常棒的訓練場。
