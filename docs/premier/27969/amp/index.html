意甲前瞻：戰熱那亞雙核缺陣，孔蒂不以傷病為借口

天天要聞
2019-12-21

烏迪內斯vs卡利亞里

烏迪內斯賽季初期發揮穩健，原本保級壓力不大，但近期連戰強敵連續不勝，已經在保級區邊緣。烏迪內斯各項攻防數據都處於意甲下游，而目前他們的最佳球員是門將穆索。賽前主帥戈蒂表示烏迪內斯還有時間，明天即使輸球但如果踢出內容也並非不可，事實是這樣嗎？

烏迪內斯的好消息是，陣中已經缺陣三場的主力邊衛塞馬和中場賈賈洛都將迎來複出，這樣拉森可以回到右翼衛，烏迪內斯唯一缺陣是中衛薩米爾，這個位置上球隊有足夠的儲備。主帥透露普塞托也已經恢復90分鐘的體能。

卡利亞里上輪被拉齊奧在補時階段連進兩球痛失好局，兩輪不勝球隊的天花板到了嗎？考慮到球隊近期的超多傷員，的確會影響馬蘭的排兵布陣。除了長期缺陣的主力外，二門奧爾森繼續缺陣，主力中場南德斯停賽、卡斯特羅受傷。一看名單，馬蘭其實除了常規主力外已經沒有後手選擇，尤其後場也不夠看。

​

卡利亞里本賽季一大進攻手段是傳中，這一點上主隊有足夠的高度，烏迪內斯喜歡堆人防守這點上卡利亞里會很難辦。卡利亞里堪稱本賽季意甲崛起實力，且客戰3勝4平未嘗一負，但本場賠率也沒有給予卡利亞里足夠的支持，本場看好烏迪內斯主場取勝。

國際米蘭vs熱那亞

國際米蘭連續三場不勝，一部分主力未愈，本場比賽又面臨布羅佐維奇和勞塔羅的停賽缺陣，孔蒂突然面臨危機時刻。人員上的危機，讓小將埃斯波西托和阿戈梅有望在本場獲得首發亮相的機會。坎特雷瓦可能被用在中場位置，他和阿戈梅中一人會獲得首發。另外左翼衛位置，阿薩莫阿傷缺，拉扎羅和比拉吉將競爭一個首發。東拼西湊組成的352讓國米整體實力下降一個檔次。

​

莫塔上任後，熱那亞的成績還不如安德雷亞佐利時期，他的帥位已經岌岌可危。近期他已經重回球隊熟悉的三後衛體系力求拿分，無奈上輪燈塔德比被桑普絕殺。本場阿古德洛復出，誇梅重傷後他對熱那亞非常重要，但勒拉熱爾、斯圖拉羅缺陣讓莫塔在中場上沒有太多選擇。

​

國際米蘭和熱那亞作為俱樂部之間的關係一直很好，甚至一段時間都是各贏各自主場，但目前國米需要競爭意甲冠軍、熱那亞需要拿分保級，本場比賽絕不該有作假成分。國米近期主場頹勢也需要一場勝利來追求反彈，雖然國米陣中傷病不少，但熱那亞除上述傷員外也還包括薩帕塔和老將潘德夫等人缺陣，考慮到這些因素，國米主勝賠率雖然一直在提高，但這很難壓低主隊熱度，順勢而為，預測熱那亞最多輸一球。
