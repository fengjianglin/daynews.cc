2019意甲最後一戰，加圖索不勝

天天要聞

▲沙漠意超杯

■■■

因扎吉大戰內斯塔，最終這場意乙大戰，以貝內文托1比0弗洛西諾內首場，昨日公推也順利紅單，並命中比分。

我在分析的最後說了，「老友交手，互相給個面子，但因扎吉在尚未鎖定晉級名額之下不會輕易放水，而且今日主場拿下，下半程做客時再送上3分，對雙方才是利益最大化，這才是朋友之間更好的安排。」

好了，因扎吉主場拿下3分，領先第三名已達14分，幾乎鎖定了一個直接升甲的名額（前兩名），那麼接下來，我們就一起看看下半程，因扎吉會不會如我所言，做客還內斯塔一個3分。

恭喜大因扎吉，提前祝賀他將重返意甲，而今天他的弟弟也將成為焦點。義大利超級盃今晚在沙特利雅得進行，上季意甲冠軍尤文圖斯對陣上季義大利杯冠軍拉齊奧，就在兩周前，小因扎吉在聯賽3比1力斬尤文，讓製造了老婦人賽季首敗，今日雙方交手，必然更加刺激。

義大利超級盃，就放入實戰版序列了，今日公推給朋友們送上2019年度的最後一場意甲賽場，那不勒斯VS薩索洛。

以下是未來一周的賽程預計，供關注包周版（每周不少於35場推薦，相比單場價格至少打了2.8折，絕對是超值+穩健之選）的用戶參考。

周日023 03:45 薩索洛VS那不勒斯

競彩勝平負 0 4.65 4.10 1.46

讓球勝平負 +1 2.25 3.70 2.30

競彩方向：讓球勝

其他方向：薩索洛（受讓0.75）

參考比分：1比1 2比2

上周末是加圖索執教那不勒斯首秀，本欄目推薦了那不勒斯的對面，結果加圖索主場1比2被帕爾馬絕殺了。紅單，但超出了我的預期，原本覺得那不勒斯有可能拿不下3分，但沒想到居然輸球了。換帥之下，球隊的狀況不是一般的低迷了。

加圖索是個可愛可敬的人，就在本場賽前新聞發布會上，還險些把本輪說成是米蘭和那不勒斯的比賽，「米蘭」二字已脫口而出，但迅速改口。不難看出，米蘭已成為加圖索身體的一部分，但這裡是南方的那不勒斯，隊伍更不好帶。

上任3天就帶隊出戰，當然不可能立竿見影，但10天過去，那不勒斯會有多少改變呢？單從對帕爾馬一戰看，那不勒斯在攻防兩端的表現都很糟糕，可以說對比安切洛蒂時期是倒退，這其中固然有換帥帶來不適以及此前和老闆內耗帶來的負面情緒，但戰術上的模糊也不可忽視。

加圖索認為，球員的頭腦里有問題，興奮不起來。這也確實是問題之一，但很難根治。俱樂部最近也在和不少球員談續約的問題，但千萬別理解為這就是德勞倫蒂斯心軟了，想跟隊員求和，說白了還是利益，果真全部鬧僵，損失最大的還是他本人。目前的方案多半是，先續約，再賣。當然，冬季也根據加圖索需要引援一兩人。

我個人很喜歡加圖索，從球員時代到其直爽的性格，但就執教水平，在教練大國義大利，還差得遠。今日對手薩索洛的教練德澤爾比，我個人還是比較欣賞。今年40歲，比加圖索還小1歲，球員時代不突出，攻擊型中場，意甲首秀恰恰就是那不勒斯，名副其實的小球員，但執教風格鮮明。

德澤爾比先後帶過巴勒莫和貝內文托踢意甲，兩支南方球隊給人留下的印象都是重攻輕守，從上賽季開始執教薩索洛，期間曾有傳聞說巴薩來考察過此人。薩索洛是典型的小俱樂部，沒有大志，能在意甲穩定生存即可，而有自己的個性標籤已算相當不錯了。

德澤爾比治下，薩索洛講求配合，尤其是進攻方面講究細節，博加負責突破，老將卡普托的搶點射門，隊長貝拉爾迪綜合能力突出，攻擊力不落俗套。而一旦防守能夠改進，這支球隊應該可以穩居中游的。

義大利Snai即時指數4.50 4.25 1.65，主降客升，平賠下調，亞洲指數客讓0.75球85水上，加圖索上任第二戰，這個盤賠還是稍有誘上之嫌。但庫利巴利仍在傷停，那不勒斯防線並不靠譜，此戰薩索洛主場值得高看一線。
