歐冠抽籤已定，預測一波哪些球隊會晉級下輪？

天天要聞
2019-12-19

第一組：多特VS大巴黎

說實話，這個抽籤對於大巴黎來說確實能算上好籤了，畢竟現在的多特有點乏力了，而大巴黎現在配置還算是很不錯的，進攻火力還是和往年一樣猛。看好大巴黎無壓力晉級

第二組：皇馬VS曼城

這兩支球隊到一起還是比較有看頭的，皇馬不是以前的皇馬，而曼城在歐冠賽場一向喜歡掉鏈子，而這個賽季在英超賽場也掉鏈子了，目前狀態也不是很好，希望能在歐冠淘汰賽開始前恢復狀態，奉獻一波精彩的比賽。而我目前還是看好曼城晉級，畢竟皇馬今年歐冠賽場表現也不強勢

第三組：亞特蘭大VS瓦倫西亞

兩支驚險的球隊，都是最後一輪取勝驚險晉級，而瓦倫西亞更是小組第一晉級。從實力上來看，我更看好瓦倫西亞，畢竟今年的瓦倫西亞有點去年阿賈克斯的感覺，黑馬一匹，小組賽一勝一平切爾西，一勝一負阿賈克斯，面對這兩支球隊絲毫不慫。而亞特蘭大就有點坎坷了。所以看好瓦倫西亞完美晉級

第四組：馬競VS利物浦

這組就有點意思啦，目前世界最好的教練之二，並且都能帶出自己風格的球隊。利物浦去年的歐冠冠軍，今年不出意外也能兌現渣叔的承若拿到利物浦最渴望的英超冠軍。而西蒙尼就不說了，一手把馬競帶成世界一流俱樂部，誰碰到馬競都會頭疼，當然，本賽季馬競狀態有點下滑，但是也是個不好啃得硬骨頭。這場比賽我看好利物浦頑強晉級，不會太舒服。並且馬競有百分之三十的機會爆冷，畢竟足球的臨場發揮也是很重要的

第五組：切爾西VS拜仁

這場對決懸念應該不是很大了，以目前車子的狀態是很難在拜仁身上佔到便宜的，拜仁歐冠小組賽雖然基本都是魚腩球隊，但是表現出來的強大進攻火力是非常耀眼的，而車子雖然晉級了，但是還是挺驚險的，而且最近聯賽也是很乏力。所以看好拜仁順利挺進下一輪

第六組：里昂VS尤文圖斯

尤文的上上籤，我感覺16強的球隊裡面，尤文最好的對手就是里昂了，拿下比賽的懸念基本沒有了，看點就在C羅能否延續自己淘汰賽之王的神話了，我還是看好總裁繼續刷新歷史的。尤文晉級，沒有懸念。

第七組：熱刺VS萊比錫

從實力來看，熱刺這個簽也算好籤了。但是我並不認為熱刺會能輕鬆的晉級，萊比錫和瓦倫西亞都是今年的黑馬，而萊比錫現在更是德甲第一，更是拿下了拜仁多特門興等德甲傳統強隊，所以實力不容小覷。但是我還是看好熱刺晉級下一輪，畢竟穆帥擅長打硬仗，但晉級不會太輕鬆。並且爆冷指數可能高達百分之四十！

第八組：那不勒斯VS巴薩

那不勒斯就有點意思了，歐冠近幾年最佳戰績晉級淘汰賽，然後主帥被解僱了，這波操作實在讓人無語，安胖的能力還是有的，雖然聯賽今年狀態稍微差一點，但是歐冠表現還是很搶眼的，莫名其妙的解僱了，淘汰賽面對巴薩。爆冷指數百分之五十，為什麼這麼高呢，因為巴薩往往先主後客的賽程最容易掉鏈子，被紅狼淘汰的那年就是，而去年同樣的方式被利物浦淘汰。而今年面對的那不勒斯又是一個神經刀，搞不好就又是悲劇重演了，這也是我唯一看不準的比賽了，感覺巴薩6，那不勒斯4.
