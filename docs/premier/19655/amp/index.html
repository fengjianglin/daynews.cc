紅牛少帥戰熱刺老將軍，神奇的1987年促使一個青年和老人的故事

天天要聞
2019-12-17

12月16日晚，歐冠16強賽淘汰賽抽籤結束了，結果讓球迷瘋狂，真是場場都精彩，因為所以球隊實力都很強。每局比賽都有其看點，接下來說的就是熱刺對RB萊比錫的比賽，說這是年輕人對老人的挑戰一點也不為過。

倫敦球隊和德甲的較量，相對而言穆里尼奧的球隊收穫了英超最好的簽位，但作為目前的德甲領頭羊，萊比錫擁有不俗的實力，少帥納格爾斯曼將迎來執教生涯的歐冠淘汰賽首秀。進攻出色的萊比錫將給熱刺的防線極大壓力。

毫無疑問，無論從歐冠經驗還是執教履歷，穆里尼奧遠勝於納帥，但納格爾斯曼作為年少有為教練的代表，本賽季率領萊比錫在攻防兩端都展現了不錯的實力。他們完全具備成為黑馬的能力。

雖然穆里尼奧上任後，熱刺的表現逐步回升，但他們的防線依舊幾乎場場丟球，而萊比錫的德國前鋒維爾納本賽季各項賽事23場比賽已經打進21球，在德甲射手榜上僅次於萊萬排名第二。同時萊比錫在德甲已經打進42球比拜仁還多1球，火力全開的紅牛面對熱刺羸弱的防線，對於防守大師穆里尼奧而言將是非常嚴峻的挑戰。

這場比賽雖然在執教經驗上穆里尼奧要遠勝於納格爾斯曼，但萊比錫這支球隊充滿著無限可能。如果他們最終晉級也不用太意外。

1987年對於普通人或許只是普通的一年，但是對萊比錫和熱刺兩傢俱樂部來說卻都有著不同的意義。在這一年熱刺主帥穆里尼奧正式結束了自己的球員生涯，隨後開始了教練生涯。也正是在這一年萊比錫少帥納格爾斯曼出生了，同時在1987年奧地利商人馬特希茨正式推出紅牛飲料，從此奠定了之後的紅牛王國。

當2004年穆里尼奧帶領波爾圖獲得歐冠冠軍的時候，納格爾斯曼還是個16歲的孩子。如今兩人將在歐冠賽場對決。

本次歐冠小組賽，紅牛和本菲卡，里昂，澤尼特一組，也許這一組實力稍微差一點，但是能取得小組第一也是不易。少帥納格爾斯曼帶領萊比錫的首個賽季戰績非常不錯。在納格爾斯曼帶隊獲得歐冠小組第一之後，知名媒體Squawka稱讚納格爾斯曼。「納格爾斯曼甚至比梅西還年輕。另外在本賽季的歐冠賽場上，出場次數超過1次的球員中，有60名球員都要比納格爾斯曼的年紀更大。」

要知道，少帥是20歲退役的，他踢了不足五年的球，之後就開始當了教練。32歲的少帥自2019年7月7日來到RB萊比錫，一共踢了27場比賽，其中包括友誼賽，德甲，德國杯，歐冠，戰績是17勝5平5敗的戰績，這個成績可以說是非常好了。

在整個萊比錫球隊中，球員整體年齡都在30歲以下，應該說是平均年齡25歲，這是一隻年輕的球隊。就是這支球隊，在德甲以10勝3平2負的戰績積33分，保持在第一的位置上。

2008/09賽季，穆里尼奧帶領國米在義大利超級盃決賽中點球戰勝羅馬奪得第一個冠軍。2009/10賽季，穆里尼奧帶領國米成功以總比數3:2淘汰巴薩，奪得歐冠冠軍。在2010年5月22日的歐冠決賽中，國米以2:0擊敗拜仁慕尼黑奪得冠軍。國米成為義大利首支三冠王球隊。三冠天下，這是穆里尼奧執教生涯的巔峰。

穆里尼奧還有很多的獎盃和數據，在這裡就不多說了，數據比較，他和少帥差距還是很大的，但是並不是絕對的，誰輸誰贏只有到最後才知道。

抽籤結果出來後，萊比錫少帥納格爾斯曼談到了這場比賽。抽籤結果揭曉之後，萊比錫少帥納格爾斯曼表示：「與熱刺的對決將非常美妙，也非常激烈。熱刺是歐洲的一支頂級球隊，熱刺在一座無與倫比的城市裡，有著世界級球星。當然這場比賽不會輕鬆，不過我們會非常期待，尤其是期待球場的氛圍。」

「熱刺有著穆里尼奧這樣一位世界級主教練，比賽會很激烈，我們必須要在兩回合的比賽中都拿出頂級表現，我們有這個能力。很美妙的抽籤，非常期待。」

馬特烏斯談到了萊比錫與熱刺的對決。馬特烏斯表示：「本賽季萊比錫是目前為止德甲最有觀賞性的球隊。他們踢著美妙的足球。球隊的戰術和陣容配置都不錯，納格爾斯曼幾乎每場比賽都有很多解決問題的辦法。」
