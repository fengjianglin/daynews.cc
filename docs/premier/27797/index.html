一潰千里！米蘭五球大敗後小門將哭了，已被國米尤文甩太遠

2019-12-23 1,718條評論 4,794次閱讀 2034人點贊

當最近4輪不敗的米蘭面對剛剛輸球的亞特蘭大，沒有人會預測到這場比賽會以米蘭慘敗收場，結果紅黑軍團卻在客場乾乾淨淨輸了個0比5，120周歲剛起步就遭遇開門黑的同時，也創下了球隊21年來最慘痛的失利。

跟隊記者賽後怒斥稱，這是一場恥辱。還有同行表示，米蘭現在需要的是換來11個伊布。評分顯示米蘭半數以上的人都不及格，最高的反倒是右邊鋒蘇索，6.5分，但蘇索本場在進攻端的表現非常平庸，他和萊奧及恰爾汗奧盧幾乎沒有任何創造性的連線。

反觀對面的真·藍黑軍團，輸給博洛尼亞其實與歐戰疲勞和主力缺陣有關，並不能反映出球隊的真實實力。本場火力全開的主隊從第一分鐘起就展現出要吃掉對手的必勝信心，戈麥斯騙過小孔蒂後爆射破門早早吹響了全面進攻的號角。亞特蘭大僅用20分鐘就把比賽變成半場攻防，戈麥斯、伊利契奇、馬林諾夫斯基大放異彩，2球1助攻的伊利契奇被評為10分滿分。

輸球後的米蘭門將多納魯馬情緒幾近崩潰，是被攙扶著離開球場的。儘管他的評分不及蘇索，但沒有人可以否認，如果沒有多納魯馬，米蘭很可能要丟更多球，儘管最後一個失球要算在他的頭上。最令米蘭球迷寒心的是，球場上沒有一個堅強有力的主心骨能帶頭穩住局面，簡單地說就是：不但踢得菜，還踢得面。無論是進攻端倚重的皮揚特克，還是此前發揮穩健的小孔蒂，都沒有打出應有的水平。

主帥皮奧利的排兵布陣也被媒體口誅筆伐，一邊繼續堅定使用凱西、恰爾汗奧盧、蘇索，一邊自殺式地換上卡拉布里亞，前三人繼續著場場夢遊的「常規操作」，後者則一上來就被對手打爆，間接導致了球隊一潰千里。當然，被小卡拉替換下場的R羅同樣表現平平，已經3個月沒有為米蘭出戰的他很可能會在冬窗離隊。

經此一戰，米蘭的排位跌出上半區，比保級區只領先不到10分，而21分的積分更是只有尤文和國米兩大死敵的一半。更殘酷的事實是，現在的AC米蘭在意甲確確實實就是中游水平，不但無法與國米尤文藍鷹紅狼拿波里們抗衡，面對中下游球隊也難言勝算，此前和薩索洛的0比0就是明證。所以，輸給能打進歐冠16強、運轉流暢又士氣高昂的亞特蘭大，很奇怪嗎？並不。

米蘭現在的頹勢，無論針對哪個位置補強都只是治標不治本，真正發揮穩定的球員寥寥，即便是換帥也要面臨巨大風險。你覺得，和本賽季發揮優異的卡利亞里、亞特蘭大相比，米蘭到底差在哪裡？
