今年荷蘭足球四大才子紛紛退役 斯內德就是其中之一

2019-12-18 139條評論 2,059次閱讀 232人點贊

今年，荷蘭隊的四大才子紛紛退役，聽到這一消息，讓觀眾既是遺憾又是不舍。遺憾的是他們還能有更長的職業生涯，不舍的是未來想看他們踢球也只能是坐在冷冰冰的屏幕看著回放。那他們自己真的捨得嗎？

8月13日，斯內德在家鄉發布了一條視頻，以這一方式宣布正式結束自己的職業生涯。

斯內德和其他三個才子一樣，年少成名並被伯樂相中，接著便在荷蘭豪門阿賈克斯出道。很快，在他23歲的時候就已經拿遍國內聯賽冠軍，這時候他的能力已是世界級。世界級的斯內德被皇馬看中並以重金把他請來踢球，這對於事業上升期的斯內德來說無疑是個一箭雙鵰的好機會。能者多勞，在很長一段時間中，斯內德得到了許多上場機會，皇馬成就了斯內德，斯內德也給皇馬帶來了足夠高的效益。

但斯內德的事業並非一帆風順，好景不長。皇馬主席換屆後不願繼承前主席的球員框架，再加上一段時間斯內德的戰績並不理想，這時斯內德不得不離開了皇馬。斯內德這匹千里馬不會就此沒落，他很快就被穆里尼奧招到門下，這時穆里尼奧才是他真正的伯樂，這時斯內德真正的巔峰期開始了。穆里尼奧麾下的國際米蘭在這一時期中不斷刷新記錄，斬獲三冠王，在時隔45年後再度拿到了歐冠的冠軍獎盃。不得不說，國際米蘭的環境很適合斯內德，在這樣一個積極的隊伍中踢球，再加上斯內德的天賦與能力，這樣的斯內德才是真正被激發出真正的潛力，有了好的環境，造就巔峰便不再會是一件難事。

巔峰過後，斯內德遭受傷病折磨，狀態不再。國際米蘭認為其價值大不如前，提出降薪，耐心消磨後決定對其使用封殺手段。隨後，斯內德在短短几年間不斷轉會，他的狀態也在不斷惡化。最終宣布退役或許這是斯內德最好的選擇，若說遺憾，斯內德的遺憾或許是即使榮譽滿滿，但他得到的國家隊獎盃僅有一座。
