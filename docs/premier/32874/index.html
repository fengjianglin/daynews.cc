2019年回顧：一沉不起的紅魔

2019-12-23 1,199條評論 4,334次閱讀 1690人點贊

2019年，即將過去，對於足球界來說，也很少以「年」來總結，因為大多數主流聯賽都是以8月、9月至明年8至9月作為一個賽季。但今天小編不妨以「年」去為一支球隊總結過去一年的表現，或者會為各位帶來一點不同的感覺，它就是紅魔曼聯。

上個賽季，曼聯在穆里尼奧的帶領下表現低沉，賽季初曾跌至英超第十三位。但自從十二月開始曼聯由名宿索爾斯克亞走馬上任後，起初是臨時主帥的身份暫時帶領球隊，但誰沒想到他的上任為球隊帶來了生氣，上任後在英超連續九場不敗，直到三月份對阿森納時才輸掉第一場比賽，可惜由於之前落後分數太多，即使表現有所回勇，但是也沒能追回之前的差距，仍落後於積分榜前面的球隊，只排在第六。

但儘管英超排名上並不如人意，但曼聯在個別賽事的表現的確踢出了血性，當中最為人津津樂道的莫過於歐冠十六強次回合逆轉巴黎聖日耳曼。但到了八強遇到西甲的巴薩，最終還是止步八強。但是對於曼聯來說，已經算是超額完成任務了。上賽季的陣容仍然有不少老將，馬蒂奇、阿什利揚、瓦倫西亞等人仍是球隊的一員，前鋒線仍有盧卡庫、桑切斯等人。

曼聯在英超最終僅以第六名結束，但其實他們曾一度有殺入前四的希望，可惜賽季尾連續多場丟分，最後四場便以兩平兩敗結束，當中有已降級的卡迪夫城、哈鎮。不要忘記他們只與第四名的熱刺相差五分，對於曼聯球迷來說，索爾斯克亞雖然帶來一丁點的希望，但最終大家仍要面對現實。但大家都認為錯不在新帥，因為他執掌時間太短，大家都認為他值得有更多的時間去重建曼聯！

夏天轉會窗往往是很多球會起死回生的一刻，曼聯這個夏天放走了不少舊將，如傷病困擾多時的瓦倫西亞、中場埃雷拉及斯莫林，另外也放走了兩位高薪球員桑切斯及盧卡庫到意甲球隊國際米蘭。但曼聯本賽季重建的工作並不多，只從萊斯特城以天價引入馬奎爾、萬比薩卡及丹尼爾詹姆斯，對於年紀老邁的防線實在是有必要進行防線重組的工作，先不說轉會身價的高昂，但的確兩位具有英超經驗及正值黃金年紀的本土球員是曼聯非常需要的。

2019/20賽季，首場賽事迎戰由蘭帕德執教的切爾西，兩支由名宿帶領的老牌球隊第一場便對決，最終曼聯以4：0擊敗對手，先聲奪人。但諷刺不過的是，勝出的曼聯與輸掉比賽的切爾西隨後在英超賽事各走一路，曼聯勝出後卻越戰越頹，隨後對狼隊、水晶宮及南安普頓都先後丟分。直至小編下筆之前，曼聯僅排在英超第八，對於球迷而言，索爾斯克亞去年帶領曼聯的復甦只是曇花一現。本賽季對於索爾斯克亞來說才是真考驗，球隊的精神面貌沒有太大改善，陣型仍是依靠個別球員的個人發揮，拉什福德、馬夏爾、丹尼爾詹姆斯三人都是單挑能力強，但一起上場時卻沒有太多的默契。

在踏入2020年前的曼聯仍有幾場英超賽事要踢，面對愈戰愈勇的紐卡斯爾了及中游球隊伯恩利，相信曼聯也沒有太大的把握全取分數，本賽季英超第三至第七位競爭激烈，究竟索爾斯克亞能否熬到賽季尾也還是一個疑問吧？
