史上五大自由球帽子戲法：沒有梅西，米哈大叔一戰封神

天天要聞
2019-11-10

自由球破門是足球世界中最高級的破門方式，也是職業球員夢寐以求的破門方式。在最新的一輪西甲聯賽中，梅西上演了自由球梅開二度，引來了全世界新一輪的吹捧和膜拜。梅西一場比賽打進兩個自由球已經神乎其神，然而在足球歷史上，還曾經有5名球員在一場比賽中上演過自由球帽子戲法，在那一刻，他們註定是這世界上最神奇的男人。

第五名：雷·麥金農

蘇格蘭球員雷·麥金農一生寂寂無名，從未入選過國家隊，也沒有在歐洲五大聯賽踢過球，單賽季進球一直處在6粒以下。但僅僅因為一場比賽，他在世界足球歷史上還是留下了自己的名字。1997年效力鄧迪聯時，雷·麥金農曾經在一場比賽中攻入三記自由球，這個自由球帽子戲法讓他平庸的職業生涯有了一抹亮色。

第四名：克里斯蒂亞諾·達·席爾瓦

在巨星輩出的巴西，克里斯蒂亞諾·達·席爾瓦可能是個小角色，很少有人知道他的存在。出道以來，這位巴西人便一直在日本J聯賽球隊柏太陽神效力，也曾在歐冠中直面中超球隊。但就是這麼一名球員，卻做到了連貝利、濟科、儒尼尼奧、內馬爾等巴西自由球高手都未曾做到的壯舉——在一場比賽中打進三粒自由球。2015年，達·席爾瓦曾在J聯賽中單場三度命中自由球，一時震驚島國。

第三名：科斯塔斯·弗朗澤斯克斯

希臘人科斯塔斯·弗朗澤斯克斯在世界範圍內可能名氣不大，但在希臘國內卻有相當的知名度，至今保持著希臘頂級聯賽自由球破門次數的歷史紀錄。1996/97賽季希臘聯賽最後一輪，弗朗澤斯克斯上演自由球帽子戲法，成為了希臘歷史上唯一做到這一點的球員。搞笑的是，弗朗澤斯克斯自由球破門屢屢得手，而在點球點前卻經常失手，為此他曾經無奈地說，「如果一次由我主罰點球，我申請加道人牆。」

第二名：朱塞佩·西格諾里

西格諾里是上世紀90年代義大利一代神鋒，效力拉齊奧期間曾經三次榮膺意甲聯賽金靴。在堪稱輝煌的足球生涯中，西格諾里還因為曾經在一場比賽中打進3粒帽子戲法被人們所牢記。1994年4月，在對陣亞特蘭大的比賽中，西格諾里在不到30分鐘的時間裡上演自由球帽子戲法，幫助球隊以3-1取勝。三個球中，前兩個是典型的越過人牆的自由球破門，第三個則是點球式的破門，希格諾里甚至沒有助跑，超長點球直奔球門左下角，門將眼睜睜看著球奔入網窩。

第一名：米哈伊洛維奇

米哈伊洛維奇是足球史上聞名的自由球大師，有著「自由球之王」的美譽，也因為職業生涯較長，被稱為「米哈大叔」。米哈大叔的自由球經典之戰出現在1998年12月13日，拉齊奧主場迎戰桑普多利亞，效力拉齊奧的米哈在這場比賽中三次利用自由球洞穿桑普多利亞的大門，幫助拉齊奧5-3拿下對手。這三個球，前兩個位置在禁區前中路，皮球繞過人牆後帶著強烈的弧線竄進球門左下角；第三個位置在禁區前右側，皮球繞過人牆後直掛球門右上角。米哈大叔曾說過，他踢球就是為了踢自由球，「我其實不是那麼喜歡踢球，但踢自由球太爽了，對我來說那才是足球。如果沒有自由球，我可能就不會選擇踢球了。」「自由球之王」實至名歸！
