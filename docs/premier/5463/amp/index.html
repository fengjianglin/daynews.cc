阿森納換帥在即！曼城助教竟成熱門候選 另一名帥學習英語做準備

天天要聞
2019-11-26

北京時間11月26日，今天的頭條留給攻防一體的坎特，還有什麼值得我們關注的新聞，我們一起來回顧一下

1、坎特超級進化，9場4腳射門收穫4球

據opta數據統計，切爾西中場坎特在最近參加的9場比賽里打進了4粒進球，這一數據和坎特在此之前參加的93場比賽的進球數持平。

從數據來看，坎特的能力顯然得到了進化，坎特在過去給球迷的印象更多是「跑不死」、中場鐵閘和超強的攔截能力，進攻方面的能力一直不是其能力之一。而在本賽季，9場比賽僅4腳射門就收穫了4粒進球，顯然坎特已經不滿足於僅僅在防守端做出貢獻了，他證明了自己可以在攻防兩端幫助球隊。

2、泰晤士報：阿森納對埃梅里失去耐心，有意阿爾特塔和阿萊格里

據報道，阿森納高層已經對埃梅里失去了耐心，在計劃邀請前俱樂部球員阿爾特塔接替埃梅里，並且此前也有報道稱阿爾特塔也願意執教老東家。

《泰晤士報》還透露，此前阿森納就有過讓阿爾特塔接任溫格的想法，只是因為他目前在曼城擔任瓜迪奧拉的助教也放棄了。

另外，阿森納方面也已經聯繫了賦閑在家的阿萊格里。據了解，阿萊格里目前正在努力的學習英語，以為將來做準備。

3、卡薩諾：伊布會揍不努力訓練的人

近期，前AC米蘭球員卡薩諾參加了Tiki Taka電視節目，談到了與伊布的往事，他表示伊布在訓練中也是一個特別嚴格的人。

卡薩諾說道：「外人可能看不到伊布善良的內心，只會感覺他很驕傲。我們效力於米蘭時，我總是在訓練中被他鞭策，因為他在訓練中也特別嚴格，他也同樣這樣要求他的隊友。如果他現在重返米蘭，我相信他可以為米蘭帶來新的風氣，可能場上作用不會很大，但他可以幫助球隊進步。」

4、斯特林進入2019年度BBC年度體育人物候選名單

在BBC目前公布的候選人中，曼城邊鋒斯特林成功入選了該名單。據了解，具體投票時間將在當地時間的12月5日晚間18點開啟。

BBC體育年度人物獎項開始於1954年，旨在表彰當年度表現優異的英國本土運動員，環法運動員傑蘭特-托馬斯是去年的獲獎者，熱刺前鋒凱恩在去年的排名中位列第3。

BBC已公布的年度體育人物候選人有：斯特林（足球）、本-斯托克斯（板球）、阿倫-維恩-瓊斯（橄欖球）、迪娜-阿瑟-史密斯（田徑）

5、蒙奇：拉基蒂奇熱愛塞維利亞，可能簽下他

據《每日體育報》報道，據了解，拉基蒂奇本賽季在巴薩的出場時間相較上賽季少了很多，因此這位克羅埃西亞中場已經開始尋求轉會。目前很多球隊都對其感興趣，老東家塞維利亞就是其中之一。塞維利亞體育總監蒙奇也談到了這筆轉會：「拉基蒂奇和巴薩的合同還沒有到期，但我知道他心裡一直熱愛著塞維利亞。現在是我們重新簽下這名球員的好時機。」

本賽季至今，拉基蒂奇共為巴薩出戰了11場比賽，只首發了1場，貢獻了1記助攻。
