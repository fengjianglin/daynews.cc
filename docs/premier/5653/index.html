C羅上演帽子戲法後在更衣室釋放哭泣，他在頒獎前情緒起伏比較大

2019-11-16 0條評論 18次閱讀 0人點贊

在金球獎頒獎典禮前夕，C羅在國家隊比賽中上演了帽子戲法，無疑為他在金球獎的評比中增加了籌碼。金球獎的投票者是記者，為這個獎項增加了「審時度勢」的性質，果然在C羅上演了帽子戲法後，上周還要求他就「換下風波」道歉的義大利記者，紛紛已經忘記了聯賽中的事情，轉頭誇獎起了C羅是「葡萄牙足球的救星」、「C羅扛著葡萄牙向歐洲杯正賽近了一步「。記者們的健忘不等於C羅把上周末的事情忘得一乾二淨了，尤其是他還是一個感情細膩的人。在義大利時間11月16日，一位與C羅關係很好的義大利記者在出席足球節目的時候，透露了C羅上演帽子戲法後在更衣室釋放哭泣，顯然他是在宣洩上周末「換人風波」的情緒。

從賽後C羅的情緒來看，他還算平穩的，他還與現場的球迷們熱情合影。不過當他回到了更衣室之後，他深知自己的這場帽子戲法可以證明他的狀態和實力沒有問題，周末聯賽換下他是「不明智的舉動」，更加關鍵的是這三個進球可以為他自己掙回輿論優勢（金球獎前夕），讓C羅不禁上演了「更衣室釋放大哭」的戲碼。由於在金球獎評選前夕，「換下風波」很有可能影響評選結果，畢竟記者是看人品的。面對這個問題，這位C羅的好友記者，也非常配合地表示：「C羅希望用自己的好狀態，神清氣爽迎接頒獎典禮。」不過這位被C羅邀請吃飯的記者也強調了：「現在是頒獎前夕，C羅的情緒起伏比較大，這可能回影響他的發揮？我想也許是這樣的吧。」

支持C羅的人們，反覆提起這個帽子戲法，就是為了表明C羅的狀態還在，他在尤文圖斯「被打入冷宮」的傳言不靠譜。如果他真的被打入了冷宮，那就是薩里「拎不清」了。平心而論，「換下風波」對C羅的評價有點極端化了，如果純粹說狀態以及技術的話，C羅相比於巔峰時期相應是有些下降的，但是無論對於俱樂部還是國家隊而言，他仍舊是「無可替代的寶貝」。所以筆者認為「換下風波」可能對C羅在金球獎上的評選會有些影響，但是不會影響C羅在尤文圖斯的出場機會，畢竟薩里還是懂球的。

這位與C羅相熟的記者說出的「在頒獎前夕，C羅的情緒起伏比較大「，或許可以解釋他在」換下風波「的一系列表現。畢竟C羅能夠有這麼高的人氣，很大程度上就是靠「真性情」來吸引球迷的。C羅的情緒起伏完全可以解釋他的爭議事件，還可以為這位頂尖球星增加親近感。筆者也不得不感嘆C羅好友記者的這一波「攻關」一下子平息了金球獎之前的爭議，甚至還為C羅增加了「人性化英雄」的設定，讓他在金球獎上「盡量獲得更多的選票」。

個人認為，C羅的帽子戲法可以說明了他近階段狀態沒有問題，不過立陶宛這個對手的實力並不能證明「C羅到底有多大的能耐」，畢竟立陶宛不是足球強國。關鍵是C羅身上承擔著葡萄牙足球的壓力，讓他雜很多時候會感到「喘不過氣來」。照理來說，葡萄牙國家隊在近幾年中拿到了歐洲杯冠軍、歐洲國家聯賽冠軍，說明了葡萄牙「黃金二代」為足壇新貴帶來了實實在在的榮譽。然而葡萄牙在國際足球大家庭的地位呢？依然不是很高，從葡萄牙拿到歐國聯冠軍，這項賽事就「神奇」地取消了冠軍直接晉級歐洲杯正賽的規則，說明了足壇大家庭並不「認可」葡萄牙這位足壇新貴——這才是C羅真正需要宣洩情緒的地方。
