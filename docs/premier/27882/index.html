今日推薦：曼聯熱刺看好，皇馬主場無憂，尤文恐遭雙殺

2019-12-22 382條評論 2,107次閱讀 1185人點贊

北京時間12月22日晚間至23日凌晨，五大聯賽相繼開戰，幾大焦點比賽也將陸續上演。比賽之前，上觀英超著力對以下4場比賽進行了分析梳理。

1、沃特福德VS曼聯，主隊不勝

曼聯進入12月份以來已經未嘗敗績，他們實現了在聯賽、杯賽和歐戰的三線俱進。並且在這個月連克曼城和熱刺，讓人感覺眼前一亮。而沃特福德方面表現實在慘淡，他們現在仍以1勝平10負的成績僅積9分位列英超倒數第一，比倒數第二的諾維奇還少了3分。作為上個賽季在英超叱吒風雲，最終超越埃弗頓隊、萊切斯特等獲得英超第八，這個賽季卻一蹶不振，沃特福德的問題嚴重到了什麼地步，可見一斑。本場比賽，面對洶湧而來的曼聯，大黃蜂難保清白之軀。

2、熱刺VS切爾西，主隊不敗

穆里尼奧和蘭帕德又相遇了，這離他們上一次在賽場相遇僅僅過了一年多。去年的英聯杯里，執教德比郡的蘭帕德在老特拉福德淘汰了穆里尼奧的曼聯，之後不久穆里尼奧被解僱。如今兩人各自執教了新的隊伍，而狀態卻截然相反。穆里尼奧的熱刺在接手後一路高歌，已經來到了第五的位置，而蘭帕德的切爾西自上個月以來在英超中僅僅收穫兩場勝利。此番師徒相遇，穆里尼奧的熱刺更被看好，而年輕的蘭帕德和他年輕的切爾西很難在這裡全身而退。

3、皇馬VS畢爾巴鄂，主隊不敗

在巴薩取勝之後，聯賽里的壓力來到了皇馬這邊，他們這場必須取勝才能保持奪冠的希望。而畢爾巴鄂是一個不容忽視的對手，他們近幾場比賽只輸了一場，顯示出這個賽季球隊的變化。考慮到皇馬隊內人員和恢復和磨合，現在他們已經進入一個良好的戰鬥狀態，加上這是西方聖誕前最後一個主場，齊達內的球隊不會放過畢爾巴鄂競技。

5、尤文圖斯vs拉齊奧，尤文不勝

這是義大利超級盃的決賽，作為尤文圖斯的死敵，雙方近6場交手取得3勝3負的成績。而且最近的聯賽交手中，拉齊奧坐鎮主場以3:1大勝尤文圖斯。現在的尤文圖斯競爭力實在太差，薩里對球隊的改造是個徹頭徹尾的失敗。如果不是迪巴拉和C羅在鋒線的靈光一現，他們很難依靠薩里的進攻體系取得勝利。這場決賽之前，拉齊奧7戰6勝，顯示了強大的攻擊力。而尤文圖斯雖然也能贏球，但是場場打得糾結難纏，令人坐立不安。此番對陣藍鷹，尤文圖斯大概率不勝，甚至被拉齊奧雙殺。
