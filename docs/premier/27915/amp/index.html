7年英超老臣成局外人！國家隊主教練看不下去，公開喊話求人情

天天要聞
2019-12-22

年終的時候，所有人都在盤點著過去一年的悲歡離合。對於小編來說，這是我第一次來到頭條的舞台上，和你們分享女性眼裡的足球世界。為了更好地呈現足球世界裡的喜怒哀樂，小編我也是蠻拼的。除了半夜起來追歐洲的比賽，還親自去到了利物浦的主場安菲爾德，以及阿森納的主場酋長球場。

因為是女性球迷，所以更願意關注的是足球周邊的故事。看多了這些分與合，小編最大的感受是，儘管這些球員們拿著我們普通人一輩子也無法企及的財富，但是他們卻依然有著你我一樣的煩惱，比如剛剛拿到工作的穆里尼奧。

在返回到熱刺時的第一個採訪中，他描繪了自己失業在家的心情，很糟糕，有些迷茫。聽上去並不像賣慘，他沒有生活的壓力，但卻始終有著其他方面的追求，這樣的追求，離我們很遙遠，卻帶著讓我們也感同身受的力量。

再比如，我們這篇文章的主人公。

過去的幾個小時里，英超賽場上，曼城捍衛了伊蒂哈德球場的榮譽，遠在英倫大地之外的利物浦拿下了自己俱樂部歷史上的第一座世俱杯冠軍。意甲的賽場上，17歲172天的國米前鋒埃斯波西托，超越巴洛特利成為21世紀意甲賽場上最年輕的首發球員。而德甲這邊，拜仁的小將齊爾克更是上演了自己的錦鯉生涯，連續兩場比賽替補出場，然後完成絕殺。

埃斯波西托必須得感謝一下他的老闆孔蒂和他的師兄盧卡庫

毫無疑問，在年關將至的關鍵時刻，足壇風起雲湧中，小將們的身影更加的出色，但是別忘了20天前，剛剛以32歲高齡拿下第六座金球獎的梅西，以及幾天前在意甲賽場上上演了讓牛頓都汗顏的彈跳力的C羅，他們還在綠茵場上書寫著青春未曾遠去，我依然老驥伏櫪的故事。

新老交替是歷史的必然，符合人類發展的規律，最重要的是成全了球迷們的幸福時刻。只是，當我們在看到那些活躍在焦點上的人物時，最不應該忘記的是就是那些已經大勢中看不到身影的球員，比如效力英超切爾西的吉魯。

本賽季的切爾西，在被限制引援的大環境下，又相繼走掉了薩里、阿扎爾等關鍵人物後，曾經的足球名宿蘭帕德臨危受命。在可打的牌不多的情況下，吉魯原本應該是有更多的機會的。但蘭帕德啟用了芒特、亞伯拉罕等新人。儘管在成形的過程中，他們也交了不少的學費，但是，經過短暫的低迷後，切爾西還是在蘭帕德的帶領下超越了熱刺、阿森納、曼聯等傳統豪門，佔據著英超第四的位置。

與此同時，重返歐冠賽場的他們，也奮力地殺出了重圍，拿到了16強的門票。

在幾頭開花的大好局勢下，蘭帕德這位英超的新人教練正在贏得更多人的尊重。

青春風暴的紅利，不光讓切爾西的青訓成果得到現實檢驗，也讓芒特、亞伯拉罕開始逐漸地走上自己的巨星之路。然而，任何事情都有兩面。青春風暴有多猛，那些老去的球員就有多尷尬。吉魯正是這種尷尬的寫照。

上個賽季，吉魯是薩裏手下的輪換的重要一員，他在中鋒位置上的作用非常明顯，儘管隨著年事已高，他不能再如在阿森納那般攻城拔寨，但是在漫長的賽季里，他依然是球隊不可或缺的對象，是切爾西拿下歐聯杯冠軍最重要的人物之一。

本賽季，隨著蘭帕德的走馬上任，曾經屢有神奇表現的他早已經淹沒在了青春小將掛起的風暴中。據統計，本賽季的他在英超聯賽中出場了191分鐘，0球0助攻。歐冠小組賽中，出場了17分鐘，0球0助攻；歐洲超級盃，74分鐘打入1球。

不管從出場時間還是場上的表現來說，吉魯已經算得上是切爾西的邊緣人物了。因為無法獲得穩定的出場機會，先前也老傳出他要離開切爾西加盟國米的消息。

按理說，俱樂部的表現讓他無法入圍競爭激烈的法國國家隊。但是，根據每次法國隊的大名單來看，吉魯依然是法國國家隊重要的一員，每一次都能得到徵召。不僅如此，今年在俱樂部的日子不怎麼好過也讓德尚看不下去，公開喊話蘭帕德，需要給吉魯一些機會，以償還自己在坎特沒有傷愈之時的不徵召。

吉魯不受重用，是年齡之殤，但在國家隊的表現，又好像是遇到了一個火眼金睛的伯樂。怎麼看，怎麼都像普通在職場上打拚的我們。效力了英超7年的老將，從曾經在英超的大殺四方，到現在的局外人，吉魯的處境，多少讓人感慨。眼下，又一個冬窗轉會季要開始了，他會迎來自己的春天嗎？
