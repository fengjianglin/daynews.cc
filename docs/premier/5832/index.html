國際米蘭社媒數據漲幅冠絕意甲 百年豪門如何變得這麼潮？

2019-11-09 0條評論 18次閱讀 0人點贊

記者里皮發自廣州

本月，足球媒體「News Tank Football」發布了歐洲足球聯賽及俱樂部社交媒體的相關數據匯總。西甲成為網上最熱門的足球聯賽，英超排在次席，意甲名列第三。在20支意甲俱樂部中，國際米蘭是上月數據上升最快的，藍黑軍團的社交媒體影響力得到顯著提升。

截止至11月1日，國際米蘭在海外三大社交媒體（臉書、推特和Instagram）上共有2370萬粉絲。這一數據讓藍黑軍團成為了意甲社媒榜單探花。值得藍黑擁躉高興的是，國際米蘭近期在社交媒體上表現強勢。上個月，國際米蘭成為了意甲20隊中臉書絕對價值提升最大的俱樂部，獲贊數比之上月多了將近100萬（漲幅5.7%）。在推特和Instagram上，國際米蘭同樣表現優異。在推特上，國際米蘭的上月獲贊數上漲了3.7%，漲幅為意甲最高。在Instagram上，藍黑軍團的上月獲贊數獲得了4.9%的上漲，這一數字位列意甲20隊中的次席。

這樣優異的社交媒體數據提升，充分反映了球迷對於如今這支國際米蘭的滿意程度。今夏，俱樂部請來名帥孔蒂，並簽下隊史最高轉會費的盧卡庫、智利名將桑切斯、義大利靈巧中場森西、烏拉圭傳奇後衛戈丁等明星球員，讓國際米蘭實力大增。義大利媒體《慢鏡頭》評價國際米蘭是今夏轉會最成功的意甲俱樂部，給出了8.5分的最高分。通過今夏的轉會操作，國際米蘭全隊身價增長了1.2億歐元，位列意甲20隊榜首。

這樣的投入帶來了立竿見影的效果。孔蒂為球隊注入了永不言敗的信念，國際米蘭整個球隊的鬥志提升，外界都看在眼裡。而在名帥的調教下，實力大漲的國際米蘭本賽季迄今在意甲11輪後，仍舊緊咬近年連冠的尤文圖斯，僅僅落後1個積分，位列聯賽次席。這使得球迷不僅繼續著現場觀賽支持球隊的熱情（國際米蘭意甲賽季場均主場觀眾數，連續4年位列第一），也在社交媒體上展現出了自己熱愛俱樂部的激情。

另一方面，國際米蘭在社交媒體上的進步，也得益於俱樂部宣傳方式方法的革新和優化。同樣在這個夏天，在俱樂部主席張康陽的主導下，國際米蘭推出新主張——「Not for everyone」。這一新主張拉近了俱樂部和新一代球迷之間的距離。球迷毫無疑問是任何俱樂部最寶貴的財產。國際米蘭正是在跨入蘇寧時代之後，進一步增強了和球迷的全方位聯繫。剛剛度過2歲生日的國際米蘭媒體中心（Inter Media House）在這個夏天製作的視頻受到了國際米蘭球迷乃至各界球迷的極大關注和歡迎。創意視頻層出不窮，讓不少球迷笑稱國際米蘭有了家自己的電影製片廠。

對於多層次、多年齡段球迷需求的重視，體現出了國際米蘭以球迷為本的運營思路。引進大牌教練和球星，使得球隊實力上漲，從而競技成績提高，進而讓俱樂部商業收入增多，並吸引更多新球迷加入藍黑大家庭，形成良性循環。國際米蘭這家意甲百年豪門重返巔峰，指日可待。
