歐冠簽位：英超兩強遭遇馬德里雙雄，究竟是"車毀"還是"仁亡"?

天天要聞
2019-12-17

2019/20賽季歐冠1/8決賽抽籤結果已經出爐，西甲英超強強對決，多特遇上了大巴黎，圖赫爾重回威斯特法倫戰舊主；皇馬鏖戰曼城，瓜迪奧拉和齊達內名帥鬥法；歐冠新軍亞特蘭大挑戰西甲勁旅瓦倫西亞；馬競遭遇新科冠軍利物浦，上演最強的矛盾之爭；切爾西重逢拜仁，重演12年歐冠決賽；里昂挑戰尤文圖斯，C羅刷新歐冠數據；熱刺對陣德甲領頭羊萊比錫，穆里尼奧重返歐冠賽場，那不勒斯挑戰巴薩，歐冠賽場上兩支球隊首次交鋒。

其中比較看點的4場比賽當屬多特VS巴黎、皇馬VS曼城、馬競VS利物浦、切爾西VS拜仁。

威斯特法倫作為盛名世界的魔鬼主場，任何球隊到了這裡都討不到便宜，現巴黎主教練圖赫爾的上一任東家正是多特，圖赫爾執教生涯中曾6次對陣多特，無一勝績，這兩場比賽他能否打破這個尷尬的記錄？羅伊斯與內馬爾的對決誰將勝出？年輕球員中桑喬和姆巴佩誰將閃耀歐冠？這都是兩隊交鋒的看點。

皇馬作為歐冠史上最出色的球隊，從來都是奪冠的熱門，曼城在英超上半程掉鏈子之後必將把重心轉移到歐戰賽場，以尋求在歐冠上取得突破，兩支頂級球隊的激情碰撞將激起無數火花，本場比賽也是瓜迪奧拉首次與齊達內交鋒，前幾屆的曼城總是在淘汰賽掉鏈子，齊達內作為教練在歐冠淘汰賽上還嘗一敗。這將是瓜迪奧拉麵臨的巨大挑戰，儘管他執教生涯中對陣皇馬9勝4平4負佔據優勢。

馬競和利物浦的比賽也將是西蒙尼和克洛普的首次交鋒，兩位具有鮮明特點的主帥。一個將防守反擊發揮極致，連續三個賽季西甲聯賽丟球最少，堪稱最佳防線；一個激活了薩拉赫、馬內、菲爾米諾，紅軍三叉戟所向披靡，場均3球的超高效率，利物浦在聯賽的絕對優勢也讓他們有精力專心投入到與馬競的對決中，這場比賽多數人看好利物浦，但是常年混跡歐冠淘汰賽的馬競絕對不容小覷。

蘭帕德作為切爾西球員擊敗拜仁奪得12年歐冠冠軍的場景依舊曆歷在目，8年後他以主教練的身份再遇拜仁，兩隊雖然都不是8年前最鼎盛的時代，但是年輕氣盛的斯坦福橋青年軍對陣小組賽六戰全勝火力全開的拜仁，究竟是”車毀”還是”仁亡”，仍然會是眾多媒體關注的焦點。

尤文和巴薩分別都抽到了實力相對較弱的里昂和那不勒斯，雖然說在賽程上具有先客後主的優勢，但是歐冠無弱旅，在沒有出線之前，一切皆有可能。
