內馬爾準備好了帶領球隊前進，主動讓點球？球天下為你繼續解說

天天要聞
2019-12-18

桑巴王子內馬爾一夏天的鬧劇最後還是以留守巴黎聖日耳曼為終。這個夏天足壇最熱的事就是內馬爾去哪？回歸巴薩重組MSN鋒線還是投到死敵皇家馬德里彌補C羅離去以後的鋒無力。最終，卡達家族為了面子沒有去放行內馬爾，內馬爾也就成為了球迷的公敵。

在比賽中當內馬爾發角球時不少球迷甚至扔東西，不斷地辱罵他。這對於一個世界級球員來說是一個困難的賽季。每一個球員都在年輕時候隨意放縱自己，真正回頭以後才發現所有的相遇都是美好的。內馬爾留守巴黎聖日耳曼或許是一個很好地選擇。

浪子回頭金不換，本賽季在內馬爾回歸後，屢屢獻出關鍵進球和助攻幫助球隊獲得了勝利。不斷地突破給製造機會換來球隊的進攻機會。也不再為了一個進球而與卡瓦尼爭奪點球。這都是內馬爾在不斷地成長。

這個賽季是在球天下（專業的足球賽事資訊平台）看到的，內馬爾在意甲的數據我們可以看出在關鍵傳球和直塞都是位列聯賽的前列。在出場時間相少於其他球員的情況下仍能決定比賽。這就是內馬爾可以僅次於梅羅以後的最佳球員稱號。

巴黎聖日耳曼的目標一直不是僅僅聯賽的冠軍，在五大聯賽的比較中，意甲是排在五大聯賽的末位。但每次在歐戰中巴黎聖日耳曼都在一手好牌中丟掉了比賽的勝利。巴黎相比於其他豪門球隊在於球隊內矛盾似乎似乎總有發生。上賽季拉比奧被雪藏一個賽季，還有小德的矛盾都被人所言辭。這對於一支要想獲得歐冠的球隊是難以相信的。其次在中場線的調控也是不盡人意，更多的需要內馬爾和迪瑪利亞站到中場去連接球隊的整個進攻。難以讓他們真正發揮到自己最熟悉的位置。

內馬爾的漸漸成熟是他更好的證明自己的的一個賽季，在天賦上他不輸給任何人，想要自己單獨帶隊證明自己的他需要在多線作戰的情況下保持健康帶領球隊去的前所未有的高度，這樣自己的選擇權才會更多。巴黎聖日耳曼要上升到一個真正的高度需要內馬爾完美的表現去帶動姆巴佩等的發揮，更好的去作為一個領袖的作用。

在剛剛結束的歐冠抽籤中，抽到了多特蒙德，這對於圖赫爾才熟悉不過的老東家，想要從中獲得勝利可謂是很艱難。這支內馬爾的巴黎聖日耳曼必須要做出更多的準備來應對了。
