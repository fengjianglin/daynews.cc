盤點：巴塞羅那近十年來的最佳與最差轉會

2019-12-16 1,796條評論 3,434次閱讀 233人點贊

點擊標題下方「關注」按鈕關注「球是這樣踢的」，讓在新的視角，為你分享最鮮活的足球信息！

上期的內容，我們給大家盤點了皇家馬德里最近十年最佳與最差的球員轉會。今天，我們繼續引述ESPN的內容，給大家盤點巴塞羅那近十年來最佳轉會與最差轉會。

近十年最佳轉會

5.哈維爾·馬斯切拉諾，2011-18年

出價：從利物浦2400萬歐元買入
售價：免費轉會至中超河北華夏
出場：334

在英超的利物浦呆了五年之後，馬斯切拉諾加入巴塞羅那。在那裡，他不得不重新證明自己，畢竟那是世界最好的球隊。在巴薩，這位阿根廷中場很快成為了一名出色的中後衛，並在他的第一個賽季就贏得了五座冠軍獎盃。就像遊戲里一樣，他成了玩家都喜歡的球員。

4.內馬爾，2013-17年

出價：從桑托斯8620萬歐元買入
出售：PSG出價2.22億歐元
出場：186；進球數：105

從桑托斯帶走巴西人，這場交易充滿了爭議，甚至到了訴諸法庭的地步。但即使拖延了好幾年，對他在球場表現沒有任何影響。進球、助攻、盤帶、球場上肆無忌憚的表演，他在巴薩表現了很高的水準。另外，他與梅西、蘇亞雷斯組成的「 MSN」組合，使他在四年內贏得了八座獎盃。當他以世界紀錄的高價轉會至PSG時，一切都變糟了。但是他今年夏天差點回去，而且現在仍然懷有返回巴薩的願望。

3.馬克·安德烈·特爾斯特根，2014-

買價：1200萬歐元，來自門興格拉德巴赫
出場數：209

當這位22歲的守門員轉會至諾坎普接替巴爾德斯位置時時，似乎有些賭博的性質。當智利球星克勞迪奧·布拉沃於同年夏天也轉會至巴薩時，「小獅子」轉會巴薩看似是一個徹頭徹尾的愚蠢舉動。然而，特爾斯特根在2016年贏得了在巴薩主力的位置，並且之後一直穩固著自己的地位，成為世界最好的門將之一。

2.喬迪·阿爾巴，2012-

出價：從瓦倫西亞購入1400萬歐元
出場次數：309；進球數：16

巴薩不會讓太多的青年隊人才流失，但是阿爾巴是在瓦倫西亞打出名堂的球員，後來才又重返諾坎普。如今，他是世界上最出色的左後衛之一，他在球場上能夠與梅西完美配合，態度兢兢業業。如此超高的性價比球員，讓加泰羅尼亞俱樂部當年為其付出的轉會費成為一種笑話。

1.路易斯·蘇亞雷斯，2014-

買價：6,500萬歐元，來自利物浦
出場次數：263；進球數：187

他在巴薩的職業生涯開端並不好，因為他在2014年世界盃上咬傷了義大利後衛基耶利尼而被禁賽四個月。坦率地說，烏拉圭人在職業生涯中經常引起爭議，但毋庸置疑，他已經成長為世界最好的前鋒之一。

近十年最差轉會

5.道格拉斯，2014-19年

出價：從聖保羅以400萬歐元的價格買入
出售：貝西克塔斯 免費
出場：8；進球：0

他在諾坎普球場上虛度了五年的大部分時光，所以錯過了職業生涯的黃金時期。他只是不夠出色，無法為俱​​樂部效力：有時候就這麼簡單。

4.菲利普·庫蒂尼奧，2018-

買價：1.2億歐元，來自利物浦
出場次數：76；進球數：21

在為看似與巴塞羅那風格完美契合的球員身上花了很多錢之後，令人困惑的是巴西人的情況沒有得到完美解決。庫蒂尼奧沒有在巴薩穩定自己的位置，在18個月後，他選擇租借加盟拜仁慕尼黑。他的買斷費用為1.2億歐元，至少意味著巴薩不會虧本。

3. 茲拉坦 伊布拉欣莫維奇，2009-11年

出價：4,700萬歐元
歐元
出場：46；進球數：22

這是世界上最昂貴的轉會之一，但是伊布只用了一個賽季就轉租到了AC米蘭。他在46場比賽中打進22球的成績相當不錯，但教練瓜迪奧拉使這筆交易變得如此糟糕。「當您購買我時，您其實正在購買法拉利。如果您駕駛法拉利，則必須將高級的燃料放到油箱中，然後駛入高速公路，然後踩下油門踏板。但瓜迪奧拉裝滿的卻是柴油，而且是在鄉下兜風。如果那是他想要的，那麼他應該從一開始就為自己買一輛菲亞特」，這是是伊布自傳中的名言。

2. 迪米特羅·齊格林斯基，2009-10

買價：從頓涅茨克礦工買入2500萬歐元
賣出：頓涅茨克礦工1500 萬歐元
出場次數：12；進球：0

在以下俱樂部名單中找出奇怪的一個：頓涅茨克礦工，巴塞羅那，第聶伯羅，雅典AEK。齊格林斯基以舉足輕重的中後衛身份抵達諾坎普球場；一年後，他以很爛的聲譽離開了。

1.阿爾達·圖蘭，2015-

出價：3400萬歐元，來自馬德里競技隊
出場：55；進球數：15

令人驚訝的是，儘管圖蘭自2017年以來就沒有為俱樂部效力，但他仍然是巴塞羅那球員。由於失寵，他在2017-18賽季沒有上場一分鐘，並被長期租借給伊斯坦布爾，最近他在那裡因在醫院開槍而被判處三年徒刑。他的巴塞羅那合同將於明年夏天到期，您可以放心，它不會續簽。
