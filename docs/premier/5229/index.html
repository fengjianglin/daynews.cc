意甲前瞻：卡利亞里堪稱大黑馬，水手不敗勢頭難以為繼

2019-12-02 0條評論 15次閱讀 0人點贊

卡利亞里隊原本屬於意甲聯賽的中游球隊，但是本賽季這支球隊可謂大放異彩，取得了7勝4平2負，積25分位列第5，這對於一支平民球隊來說實屬不易，本賽季他們先是從國際米蘭租借簽回了自己的舊部納因格蘭，這位比利時中場堪稱是這個位置上的世界前五球員之一，而且剛剛榮膺意甲上個月最佳球員。而本賽季他們引進的佛羅倫薩前鋒小西蒙尼也是一筆超值引援，目前，卡利亞里的防守端非常出色，只丟了14球。

而客隊桑普多利亞本賽季的表現讓人大失所望，本來，人們期待著他們能夠延續上賽季的驚人表現並且取得中上游的表現，但是這支球隊目前只是取得了3勝3平7負，積12分位列第17，其中在客場戰績為1勝5負，積3分位列第18位，只能為保級而戰，目前來說，誇利亞雷拉這位從C羅手中搶走金靴的不老男神沒有能夠保持原來的競技狀態。此外，桑普多利亞打入9球而防守端竟然丟掉了多達20球，可見他們攻防都非常低迷。

最近10場比賽，主隊取得了6勝4平，其中主場4勝1平，狀態堪稱時意甲最佳之一，而客隊則是在目前陷入了保級的泥沼，好在桑普多利亞目前取得了三場不敗，以往交鋒10次主隊只贏過一場，此役他們有希望復仇。預測：卡利亞里
