五大聯賽頭名，四家三叉戟，進攻足球才是這個時代的王道？

2019-12-20 585條評論 1,492次閱讀 190人點贊

五大聯賽已經接近半程，除了德甲巨人拜仁慕尼黑還沒睡醒以外，其他四家聯賽的大爺仍然是大爺！其中利物浦以17輪49分的戰績，成為了五大聯賽的利獨秀同學。

而值得一提的是這五大聯賽榜首球隊，其中四家都是以三叉戟為戰術主體。這是否能說明三叉戟才是這個當今這個時代的趨勢？下面小編為各位看官簡單介紹一下各隊三叉戟。（純個人見解，如有不同意見，歡迎糾錯）

利物浦 馬內、菲爾米諾、薩拉赫

利物浦今年聯賽的強勢開局，與進攻端的的穩定發揮是密不可分的，三叉戟的高效進球率在17輪聯賽中已經斬獲了22球。眾所周知英超是5大聯賽中競爭最激烈，對抗最高的聯賽，三叉戟在這麼高強度的聯賽中斬獲這麼多進球，可見高效！

黑色閃電馬內

司職左邊鋒的馬內是三叉戟的速度擔當，反擊的發起點，每當球隊後場斷球，馬內的快速前插，猶如一道黑色閃電，直插對手要害。擁有高速護球和帶球能力他，在對手禁區右路製造了很多麻煩，撕扯對方防線的他，可突可傳，讓對手防不勝防。

全面型菲爾米諾

紅軍三叉戟的攻擊槓桿，菲爾米諾繼承了南美球員的靈動與典型歐洲球員的紮實，司職中鋒的他，在禁區內可搶點，拉出禁區可傳威脅球，由於他在中鋒位置的問題，他的跑動同時可以幫助馬內和薩拉赫拉出更多的活動空檔，菲爾米諾雖然進球少，但可以說是三叉戟中必不可少的人。

現象級薩拉赫

法老王薩拉赫在三叉戟司職右路，驚人的盤帶能力讓每個後衛防守他的時候都壓力頗大，步頻快、腳法細膩可以讓他在狹小的空間內完成擺脫，每當球隊打不開局面的時候，他就是那個三叉戟進攻的發起者，無論靈動的盤帶還是刁鑽的射門，總能幫助球隊打開局面。

巴塞羅那 梅西、蘇亞雷斯、格列斯曼

今年巴薩在聯賽踢的並不好，作為西甲二人轉之一的他們，今年在客場僅僅拿了14分。（當然這也跟賽季初梅西與蘇亞雷斯的傷病，以及格列斯曼融合球隊需要時間有很大關係）不過即使是這樣，這個巴薩新三叉戟已經為球隊奉獻了27粒聯賽進球。

里奧-梅西

梅老闆在三叉戟的位置是司職於右路，巴塞羅那整條進攻線的組織者和終結者，由於身高的問題，除了頭槌能力稍差外，其他方面近乎完美，超強的盤帶能力一個人就可以單挑對手整條防線，見縫插針的傳球，往往能給對方球門造成很大的威脅。

一刻不閑蘇亞雷斯

蘇亞雷斯是南美球員的典範，無球跑動與門前的嗅覺，讓他在三叉戟中如魚得水，擁有超強射術的他在梅老闆牽扯對方大部分防守的時候，往往能給予對手致命一擊。而且蘇牙在無球狀態下與對方防守隊員心態上的博弈也是他的一大特點。

漸入佳境的格列斯曼

今年加盟的格子一直在嘗試融入巴薩三叉戟的體系，曾經主要踢影鋒的他，對於左邊鋒的位置也漸入佳境，超強的盤帶能力與護球能力，幫助梅西分擔了很大的進攻壓力。成為新三叉戟中重要的一員。

巴黎聖日耳曼、姆巴佩、內馬爾 伊卡爾迪

大巴黎不用多說，法甲聯賽的王者，今年伊卡爾迪的加盟，新三叉戟的成立，讓我們更加看到了大巴黎在聯賽的強勢！這支新的三叉戟17輪聯賽中一共為大巴黎奉獻了24粒進球。

絕世少年姆巴佩

不知不覺中天才少年已經來到大巴黎快3個賽季了，越發成熟的他現在司職於巴黎三叉戟的右路，一騎絕塵的速度，左右腳的均衡，無論反擊還是戰地進攻，都可以把對方的防守沖的七零八落，為禁區的伊卡爾迪創造出很多機會。

禁區之王伊卡爾迪

伊卡爾迪在禁區的恐怖能力，讓防守隊員不得不緊盯他，而是即便這樣，還是讓對手防不勝防，無球跑動與門前嗅覺，讓他在左右兩個世界級的隊友的幫助下發揮到了極致。

逐漸成熟的內馬爾

內馬爾司職於左邊鋒的位置，超強的控球能力以及頂尖的過人能力，讓他把對方的防守隊員耍的團團轉，往往一個人就拉走了對方防守的重心（被動帶嘲諷技能），從而給其他隊友創造機會。

尤文圖斯 C羅、迪巴拉、伊瓜因

尤文圖斯同樣是意甲的霸主，8連冠的他們，在本賽季遭受到了國際米蘭與拉齊奧的衝擊，新主教練的到來，一直在調試著這支「老婦人」不過最近「真香定律」的薩里漸入佳境，尤文圖斯的新三叉戟已經起航。

克里斯蒂亞諾-羅納爾多

與前面三個三叉戟不同的是，尤文三叉戟不能算是傳統意義上的三叉戟，在薩里的戰術安排里，迪巴拉是以影鋒的位置去踢的，而C羅是司職於左中鋒，但是比賽踢起來，三人會經常換位，所以沒法去定性C羅到底是在三叉戟的什麼位置，在左右邊鋒位置上的時候，C羅主要的作用主要是拉扯對方防線與內切，而當遊走到中鋒位置的時候，就簡單粗暴的扮演了一個終結者。

巴拉巴拉小魔仙迪巴拉

雖然安排的任務是以影鋒去踢，但是他也會經常拉到右邊去踢，迪巴拉的射術精湛，不管是禁區內還是禁區外，只要給予他充足的空間，他就能奉獻出一腳「圓月彎刀」，而迪巴拉同時擁有一流的盤帶能力，總能在球隊需要的時候，撕開對手的防線。

煙槍伊瓜因

伊瓜因一樣是一個傳統意義上的中鋒，門前嗅覺以及終結能力超強，在尤文這支三叉戟充當右中鋒，當C羅拉到左邊小迪拉到右邊時，又變成了搶點型中鋒，有一腳不錯的直塞球能力，為兩翼的隊友輸出關鍵球。
