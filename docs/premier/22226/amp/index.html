周三意甲推薦：布雷西亞有所回暖，薩索洛客場拿分

天天要聞
2019-12-18

昨日給大家推薦的長串很遺憾沒沒有命中，僅僅命中了三場，昨日我個人看好的不萊梅主場大敗，而且我還是用這場做膽的，實在是沒有想到啊，今日我先給大家推薦一單自己看好的意甲單場，晚場我會接著長串的，我個人看好薩索洛客場不敗，很有可能拿分，好了下面就給大家分析今日的賽事！

周三013意甲:布雷西亞 VS 薩索洛

比賽時間：2019-12-19 03：45

比賽分析：

布雷西亞在11月3日解僱了上賽季帶領球隊升級成功的主帥科里尼，代替科里尼的是偉大的後衛格羅索，布雷西亞上輪聯賽坐鎮主場3比0完勝了萊切，取得聯賽兩連勝，終止了賽季至今6輪不勝3連敗的低迷狀態，有所回升，聯賽戰罷15輪取得4勝1平10負，打進14球，排在進攻端倒數第4名，丟26球，排在防守端倒數第6名，攻防兩端均十分糟糕，目前積13分，排在第18名，處於降級區，距離安全區2分，在主場方面，賽季7個主場取得1勝1平5負，進8球丟15球，攻防都很差，綜合實力仍然很弱。

薩索洛近3輪不勝，連續遭遇瓶頸，但是與布雷西亞相反的是，球隊遭遇了魔鬼賽程，12月1日客場2比2尤文圖斯，12月8日主場2比2戰平近期意甲狀態最好的卡利亞里，12月15日客場0比0逼平近期狀態回勇的AC米蘭，儘管沒有取勝，但是這3場平局的含金量不小，這個賽季由於貝爾納迪的爆發，球隊的攻擊力得到了很大的提升，貝納爾迪（12場8球2助攻）同時激活了進攻線上的其他隊友，9號卡普托（13場7球4助攻）與法國小將博加（14場4球）都有相當不錯的發揮，球隊的問題是在德米拉爾轉會去了尤文圖斯之後，防守端開始不夠穩固了，15場27個失球影響了球隊的成績。

從戰績來看，近期2連勝的布雷西亞無疑比連續不勝的薩索洛看上去要更好一些，但是前面提到這個成績其實與賽程相關，而在三連平中薩索洛在與強隊的對抗中表現出的狀態是不錯的，機構本場的平手指數對於主隊支持力度並不大，目前主隊的指數調整到了中低，後續有到主讓平半的可能性，這是機構在用主隊的連勝在誘主，本場看好薩索洛至少不敗。

綜合推薦：讓負
