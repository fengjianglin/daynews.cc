落後扳平守住優勢，利物浦的強勢遭遇考驗，一場輸球就前功盡棄

2019-11-28 0條評論 18次閱讀 0人點贊

北京時間11月28日凌晨結束的歐冠小組第五輪的一場焦點比賽，主場作戰的衛冕冠軍利物浦隊在迎戰那不勒斯隊的比賽中，半場一球落後之下憑藉後衛洛夫倫的進球扳平比分，以1比1艱難逼和對手。本輪戰罷，利物浦隊雖然還是以1分的優勢領跑積分榜，小組出線佔據最有利形勢。只是，如果最後一輪他們客場小比分輸給薩爾茨堡紅牛隊，他們賽季開戰以來所表現出來的強勢，將前功盡棄，衛冕冠軍小組出局。

本輪比賽前，利物浦隊領先著第二名那不勒斯隊1分，領先薩爾茨堡紅牛隊5分，只要這場比賽能在主場擊敗對手，那麼克洛普的衛冕冠軍就將提前一輪小組出線。然而，主場作戰的利物浦隊，雖然開場後就展開積極的逼搶，和快速犀利的進攻。但安切洛蒂的那不勒斯隊通過傳接球的戰術一步步瓦解利物浦隊的攻勢。

比賽第21分鐘，正當利物浦隊前場進攻被斷，還沒來得及迅速退防之時，那不勒斯隊的一次反越位戰術就撕開利物浦防線，默爾藤斯的進球直接把利物浦隊踢入困境。因為一旦就以這個結果結束賽事，那不勒斯隊將要反超拿到小組第一提前出線。

好在下半場，克洛普果斷用張伯倫換下戈麥斯後就收到效果，藉助一次角球機會，中衛洛夫倫的沖頂破門，幫助利物浦隊扳平了比分。雖然此後的比賽利物浦隊不甘以一場平局收場，但客場作戰的那不勒斯隊戰術目的非常明確，那就是確保不再丟球，利物浦隊只能遺憾的以1比1平局結束賽事。

本輪比賽結束後，積分榜上利物浦隊依然還以1分的優勢領跑，但也僅僅領先1分而已。更危險的是，隨著另一場比賽薩爾茨堡紅牛隊客場擊敗魚腩亨克隊後，他們目前僅僅以3分之差落後利物浦隊，依然還有最後逆轉出線的可能。最後一輪比賽，那不勒斯隊只需在主場打平魚腩亨克隊，就將保證小組出線。縱使爆冷輸掉比賽，但只要利物浦隊客場不輸給薩爾茨堡紅牛隊，他們也將小組出線，因此可以說，那不勒斯隊的出線幾率在這輪比賽結束後形勢最好。

因為首回合的比賽，利物浦隊只是以4比3主場擊敗薩爾茨堡紅牛隊。如果最後一輪比賽利物浦隊以小比分輸給對手，那麼兩隊之間的積分就將達到相同。鑒於首回合薩爾茨堡紅牛隊打進三個客場進球，那麼他們將會以客場進球優勢淘汰掉利物浦隊。因此，利物浦隊最後一輪比賽，縱使要輸球，但也要以打進三個以上客場進球的表現才能確保出線。大家都知道，回到主場的薩爾茨堡紅牛隊進攻力很強，他們主場擊敗利物浦隊還是有很大希望的。

一旦衛冕冠軍在最後一輪輸球出局，那麼克洛普的球隊在新本賽季開戰以來所表現出來的強勢，將前功盡棄。因此五大聯賽中，目前最為強勢的，就是利物浦隊。英超聯賽打了13輪，利物浦隊僅僅平了一場，表現非常強勢和霸道。
