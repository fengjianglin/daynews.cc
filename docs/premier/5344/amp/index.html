克洛普：很抱歉利物浦歐冠沒提前出線 球迷沒法去奧地利度假了

天天要聞
3 週 ago

鏡報報道，利物浦在主場的歐冠比賽中沒能戰勝那不勒斯提前出線，紅軍主帥克洛普也為此道歉，聲稱破壞了球迷們想要去奧地利「度假」的心，球隊也損失了一次輪換的機會。與那不勒斯的比賽取得了平局，這意味著克洛普的球隊必須在薩爾茨堡拿到一分，才能確保進入歐冠淘汰賽。

如果利物浦在主場擊敗那不勒斯，那麼紅軍將會擴大小組第一的領先優勢並且提前出線。這意味著利物浦可以在12月的密集賽程中，得到輪換的機會。但是利物浦在主場戰平那不勒斯，球隊也就失去了輪換的機會。

克洛普對於球隊出線的希望還是很有信心的，他說：「目前小組晉級形勢還很開放（三支球隊都有希望晉級），我們沒有取得理想的結果，但是仍然很有希望晉級。我不想說我們在每一場比賽中都需要強度，但是我們確實在比賽中感受到了。」

這位德國教練隨後補充道：「我知道大家都希望我們本場比賽能夠取勝，鎖定小組第一，然後做客薩爾茨堡的比賽就可以當成一個假日比賽，進行輪換，以面對12月艱難的賽程。可惜今天我們沒能如願取勝，所以我想向球迷們說一聲抱歉。而球迷們也因此沒有了帶著度假的心情看球，很抱歉這樣的事情沒有發生。」

目前利物浦積10分排名第一，那不勒斯積9分緊隨其後，薩爾茨堡積7分排名第三。如果利物浦在對陣薩爾茨堡的比賽中取勝，那麼他們仍然將排名小組第一併且晉級歐冠16強。但是如果紅軍在奧地利敗北，那麼他們將無緣歐冠淘汰賽，只能去參加歐聯杯的賽事了。首回合紅軍坐鎮主場對陣薩爾茨堡，也僅僅是4-3險勝，靠著薩拉赫的絕殺才拿下3分。除此之外，法比尼奧的受傷也給利物浦增添了困難，他是球隊中場的關鍵人物，而他的腳踝在比賽中受傷。

球員的受傷，以及失去一個可以給球隊輪換的比賽場次，使得克洛普的壓力並不小。在12月的密集賽程中，紅軍每三天就將進行一場比賽，利物浦的身影將會出現在歐冠、世俱杯、聯賽杯以及英超的賽場上。
