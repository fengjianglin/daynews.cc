歐冠最新積分榜：利物浦強勢晉級，小組頭名爆冷出局，國米輸巴薩

2019-12-11 370條評論 1,785次閱讀 234人點贊

北京時間12月11日凌晨，歐冠小組賽最後一輪比賽開打，目前已經結束8場比賽。利物浦客場2-0輕取薩爾沃茲紅牛，以E組頭名身份強勢晉級十六強；那不勒斯4-0大勝比利時勁旅亨克，同樣小組出線；F組，在梅西輪休的情況下，巴薩2-1戰勝國米，以不敗戰績晉級淘汰賽，多特蒙德2-1布拉格撕拉威亞，以第2名身份小組出線。H組積分榜大亂，青年軍阿賈克斯在賽前還以2分優勢高居積分榜榜首，但本輪主場0-1輸瓦倫西亞，同組切爾西2-1拿下里爾，阿賈克斯掉至第3，爆冷被淘汰，瓦倫西亞和切爾西攜手晉級。

薩拉赫

E-H組末輪比賽已經在今天凌晨結束，積分榜排名已經正式出爐，8支晉級球隊全部誕生。A-D組的比賽將在明天凌晨，屆時歐冠十六強將全部產生。

E組：賽前該組沒有球隊確定出線，所以本輪4隊都竭盡全力。利物浦與薩爾沃茲紅牛上演對攻大戰，比賽節奏非常快，可以說是今天凌晨最精彩的一場比賽。下半場凱塔和薩拉赫2分鐘內先後完成進球，最終利物浦在客場2-0獲勝，穩穩拿到3分，積13分獲得小組頭名。

巴薩

同組另外一場比賽同樣沒有懸念，米利克戴帽，那不勒斯半場就3-0領先亨克，最終收穫一場4-0的大勝，以一分之差排名小組第2，同樣晉級十六強。賽後那不勒斯官宣主教練安切洛蒂下課，那不勒斯目前排名意甲積分榜第7名，比第3名拉齊奧少12分，更糟糕的是球隊內部有矛盾，安帥下課在預料之中。

積分榜

F組：巴薩賽前已經鎖定小組第一，所以巴爾韋德此役選擇讓隊長梅西輪休，蘇亞雷斯、特爾施特根等主力球員坐替補席，佩雷斯和盧卡庫在上半場各進一球，兩隊1-1戰平。下半場接近尾聲，17歲小將法蒂貼地斬破門，成為歐冠歷史上取得進球的最年輕的球員。

德甲勁旅多特蒙德2-1險勝布拉格撕拉威亞，在巴薩的『幫助下』，最終反超國米，上升至小組第2，驚險出線。

積分榜

G組：萊比錫客場2-2與里昂握手言和，積11分鎖定小組第一。澤尼特0-3慘敗本菲卡，從積分榜第2掉至倒一，錯失出線良機，送里昂晉級。

瓦倫西亞

H組：前5輪戰罷，阿賈克斯積10分高居積分榜頭名，比切爾西和瓦倫西亞多2分，牢牢掌握晉級主動權，本輪只要不敗就可以出線。儘管擁有主場優勢，但阿賈克斯0-1不敵西甲勁旅瓦倫西亞，輸掉生死戰，從積分榜第一掉至第三，遺憾未能出線，將對手送上榜首。

阿賈克斯

英超豪門切爾西沒有掉鏈，主場2-1拿下里爾，取得寶貴3分，升至積分榜第2，與瓦倫西亞攜手晉級。主教練蘭帕德成為成為9年來第一位執教六場小組賽後，帶隊闖進歐冠淘汰賽的英格蘭主帥，首位作為球員和教練都能殺進歐冠淘汰賽的英格蘭人。

切爾西

本賽季歐冠小組賽僅剩最後8場比賽，目前已經有14支球隊淘汰賽，明天凌晨將產生最後2支晉級球隊，頓涅茲克礦工與薩格勒布迪納摩競爭C組第2，馬競與勒沃庫森爭搶D組最後一個晉級名額。
