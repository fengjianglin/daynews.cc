歷史第一人！穆里尼奧魔力不減，國足帥位再合適不過

2019-11-27 0條評論 18次閱讀 0人點贊

穆里尼奧不需要太多讚譽。

他達成了一項偉大的成就，在執教熱刺的第二場比賽中，熱刺逆轉奧林匹亞科斯晉級歐冠淘汰賽。

穆里尼奧成為了歷史上第一位率領6支球隊打進歐冠淘汰賽的主教練。

波爾圖、切爾西、國米、皇馬、曼聯、熱刺。

並且在波爾圖和國米時期，均帶隊獲得歐冠冠軍，出線奪冠率33.33%。

我們常叫他「魔力鳥」，這是穆里尼奧四個字發音的諧音，而「狂人」則是更合適的稱呼。

畢竟他是為數不多的和溫格打過架的人~

狂這個字，現在更多帶有貶義的意思。我們經常說某個人狂，是說他不知分寸，沒什麼本事還上躥下跳，用句俗話說就是「不知道自己算老幾」。

但穆里尼奧絕對是老大，他也必須是老大，他也配得上當老大。

從波爾圖發跡之後，他就成了各大豪門的追逐目標，切爾西成為了第一個贏家，他們也很快享受到了收穫的果實。穆里尼奧執教的第一個賽季，切爾西就拿到了英超冠軍，打破了曼聯和阿森納壟斷英超的局面。05-06賽季，穆里尼奧再次率隊奪得英超冠軍，就此超越了溫格從未在英超衛冕的歷史。

也許這就是後來二人在場邊動手較量的原因吧~

可惜的是，當季的歐冠八分之一決賽，穆里尼奧的切爾西碰上了巴薩，彼時的巴薩是小羅閃耀的時代。

小羅在禁區前擺脫眾人射門得分的瞬間，是切爾西球迷永遠不願回想的畫面。

不過穆帥很快就為自己報仇了，執教國米又在歐冠遇到了巴薩。在首回合領先的情況下，穆里尼奧果斷地使用了「901」陣容，只留下米利托一人作為反擊點。上半場莫塔被罰下，穆帥大手一揮，「901」變成了「900」。

如果大家不記得這場比賽了，沒關係，記得這個經典的時刻就可以。

也許是上天覺得穆帥的人生太過精彩，自國米奪得歐冠之後，穆帥的執教生涯頻現坎坷。

執教皇馬更衣室不和，把卡西貶到替補席，引發了隊內西葡球員的分化；

回歸切爾西，剛拿到雙冠王成功續約，卻因為聯賽里戰績不佳被人開除；

來到曼聯，帶隊成為英超歷史第一支600勝球隊，最後卻靜靜地離開。

穆帥自己也說過，被曼聯解僱時是他人生最困難的時刻。

穆里尼奧的執教風格非常鮮明：信奉意志、執行力強、偏執張狂。

他的偏執能讓他在歐冠賽場擺出超級大巴，他的張狂能讓他為了球隊和隊員無所顧忌的表達意見。

穆里尼奧的性格和能力，造就了他的「吸粉」體質，他本人也非常坦誠，無所不言。

其實總結一下穆帥的執教生涯，我們會發現他適合執教低潮期但有實力的球隊，他可以將一支球隊從低谷帶到高潮。

比如平民波爾圖、當時的切爾西、「電話門」影響下的國米、低谷期的皇馬。可當這些豪門球隊復甦，大牌球星們爭奪話語權時，俱樂部高層往往更珍惜球員，所以穆帥成了犧牲品。

而在執教熱刺前的一段時間裡，穆里尼奧表達了想要執教一支國家隊的意願，他想讓自己的教練生涯更加圓滿，他想衝擊國家隊榮譽。

所以國足，你看到了嗎！

國足和穆帥，堪稱天作之合，二者互相匹配的程度非常高，絕對是一次雙贏的合作。

穆帥只要帶國足帶進世界盃，就是非常偉大的一項國家隊成就，他將成為全中國足球的功臣。

穆里尼奧是葡萄牙人，國足歸化巴西外援說葡萄牙語，毫無溝通障礙。

國足目前急需一個有威望有能力的主帥帶領球隊。

註：以上關於國足的內容純屬個人一廂情願的想像。

希望穆帥能繼續在熱刺發揮魔力，畢竟熱刺上賽季已經是歐冠亞軍了，這賽季的目標，嗯~
