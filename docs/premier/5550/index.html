效忠不同俱樂部？托蒂：我和里皮都效忠‘義大利’俱樂部

2019-11-23 0條評論 20次閱讀 0人點贊

虎撲11月23日訊 近日，羅馬傳奇球星托蒂接受採訪時，表示自己現階段不會回到羅馬，他會繼續遵循自己所選擇的道路。

托蒂：「我是一個很敏感的人，當我決定了一條道路，我就會走到底。如果現在羅馬讓我回歸，我會陷入艱難的抉擇當中。」

「我會拒絕羅馬的邀請，因為我尊重我做出的決定，我想要繼續自己所選擇的道路。我想要幫助球員，向他們傳輸我的經驗。」

談里皮

「我對他的記憶都是積極的方面。除了一名偉大的教練，我把他看作一個非常特殊的人，一位父親。我和他之間有超出普通教練與球員之間的關係，那超出了足球這個範疇。」

「我受傷（2006年世界盃遭遇重傷）後，他和我進行了交流，端正了我（消極）的思想。」

兩人代表著不同的俱樂部，如何摒棄俱樂部間敵對關係，在國家隊攜手共同前進？

「很簡單，在國家隊我們就想著國家隊的事情，不會去考慮俱樂部的事情。團結就是力量，我們的目標就是將義大利帶到更高的層面，我們渴望讓義大利成為世界冠軍。我們同屬於一支俱樂部，義大利俱樂部。」

扎尼奧洛

「我希望大家停止扎尼奧洛與我的比較，如果他持續這樣出色的表現，我也希望他繼續這樣的表現，那麼他值得收穫足球賦予他的一切。」

「我們需要讓他繼續成長，他了解羅馬精神，希望他能夠長留羅馬。」

美國財團有意收購羅馬

「我也不知道他們是否有意收購。我對此有我的疑問，當有人想要染指羅馬的時候，球隊卻很難向前，我很清楚這一點。」

「但從球迷的角度，無論誰來到俱樂部擔任主席，我都希望他能夠帶領球隊贏得一切，意甲和歐冠。」

(編輯：姚凡)
