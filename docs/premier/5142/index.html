C 羅本賽季結束是否存在離開尤文圖斯的可能性？

2019-12-05 0條評論 13次閱讀 0人點贊

在都靈城四萬多名尤文球迷的助威聲中，斑馬軍1-2輸給了掀起青春風暴的阿賈克斯，也最終止步歐冠八強。尤文圖斯的這次輸球可謂是完敗，無論是最後的結果還是場上的局勢都處於絕對的劣勢，阿涅利與阿萊格里在賽後都坦誠斑馬軍輸得服氣，阿賈克斯更配得上歐冠四強。隨著這場輸球，C羅連續八年殺入歐冠四強的紀錄戛然而止，而有關他會離開尤文的消息也不斷被曝出，雖然C羅的離隊可能不大，但是也絕非天方夜譚。對於目前的尤文圖斯來說，怎樣配置球隊的陣容更好的為C羅服務則是一個必須完成的任務。

C羅是否真的存在離隊可能？

C羅在去年夏天與尤文圖斯簽訂了一份為期四年的合同，合同在2022年到期。但是在輸給阿賈克斯後，C羅的密友阿吉雷以及母親多洛蕾斯卻分別表示C羅在輸球後的鬱悶、惱火以及失望。

義大利媒體《共和報》則爆料，C羅可能在下賽季結束前離開尤文，這樣的消息一石激起千層浪，博彩公司也為C羅新賽季是否離隊開出了賠率，繼續留在尤文的賠率為1.25，但是加盟曼聯、皇馬或者巴黎的賠率分別為7.0,10.0，以及14.0。因此從以上賠率看，儘管C羅離隊可能不大，但是如果C羅真的走人，不會讓人意外，7.0的賠率並不算很高。

尤文圖斯如何打造陣容為C羅服務?

C羅更大的可能是留在尤文，但考慮到明年葡萄牙巨星就將年滿35歲，這也意味著下個賽季可能是尤文在C羅帶領下贏得歐冠的唯一機會。因此對於目前的斑馬軍團來說，如何更好的配置陣容，打造一支為C羅服務的球隊是第一選擇，也是唯一選擇。從本賽季的陣容選擇看，尤文圖斯主打433陣型，C羅在這一位置擔任左邊鋒，但是他的位置不太固定，有時也會出現在中鋒位置進行包抄爭頂。從實質上說，這與C羅昔日在皇馬踢球時的433陣型差不多，但是總體水平低了一個等級，這需要從引援上進行補強。接下來我們從尤文的不同位置的引援與清洗進行一下分析。

後防線：基耶利尼替身難尋 烏姆蒂蒂成上選

什琴斯尼已經成為了布馮理想的繼承人，對戰阿賈克斯的次回合比賽中，什琴斯尼發揮神勇，他兩次撲出對方的必進球，同時也干擾了內雷斯的一腳挑射，如果沒有波蘭門將尤文早已被打成篩子。因此從門將位置看尤文圖斯無需買人，什琴斯尼的發揮穩定，而且替補席上還有佩林這位準國門級別的守門員。

中衛位置會是尤文圖斯補強的重點，巴爾扎利本賽季結束後退役，卡塞雷斯只是球隊應急的選擇。雖然魯加尼與球隊簽訂了一份續約合同，但是從表現看，他難以獲得絕對主力位置。基耶利尼這位尤文圖斯的隊長的位置不可撼動，而博努奇的位置也比較穩固，雖然在重要性上與基耶利尼無法相提並論，但是在頂級中衛稀缺的今天，博努奇依舊可以在尤文圖斯有一席之地。在買人選擇上，德利赫特是不錯的選擇，但是引援難度太大，而且這位球員加盟巴薩概率極高，因此這種背景下尤文圖斯簽下巴薩中衛烏姆蒂蒂是絕佳的選擇。當然，尤文也對瓦拉內虎視眈眈，但皇馬不願意放走法國中衛，而他的合同違約金高達5億歐元。

兩個邊後衛位置，可以踢兩個邊路的坎塞洛以及德西利奧應該會留下，尤其是坎塞洛，其在邊路超強的進攻能力可以彌補其防守能力的不足，在需要進攻的時候，坎塞洛甚至可以直接提到中前場發揮攻擊能力。德西利奧左右腳能力均衡，進攻防守都沒有很多亮點，但是全能性不錯。斯皮納佐拉這位左邊後衛助攻能力不錯，防守有些軟肋，但是擔任替補問題不大。桑德羅是尤文的主力左後衛，而且與尤文續約時間不長，他理應得到留隊的機會。但是從目前的情況看，在球隊需要引援資金的情況下，斑馬軍賣掉桑德羅的可能也不小，斯皮納佐拉、德西利奧都可以踢這個位置。同時如果賣掉桑德羅，引進馬塞洛當然是最佳選擇，但是在齊達內回歸皇馬的情況下，尤文圖斯引援難度太大。終上所述，在邊後衛位置，尤文最大的可能是保持陣容不變，如果確定套現桑德羅，引進阿賈克斯的塔利亞菲科或者皇馬的馬塞洛，將是一個好的選擇。

中場：尤文最薄弱環節 需徹底換血

中場一直是尤文圖斯的軟肋，在與阿賈克斯的兩回合比賽中，尤文圖斯的中場軟肋也暴露無疑。C羅在皇馬效力期間，戰艦的三大中場分別是克羅斯、莫德里奇以及卡塞米羅，這一中場配置也是皇馬贏得歐冠三連冠的關鍵，而尤文圖斯也應該從這一配置中尋找靈感，為球隊配置一條可以與三人媲美的中場。

克羅斯大局觀強、傳球精準，而且有遠射的絕活。從目前尤文圖斯陣中的中場球員看，皮亞尼奇的風格與克羅斯最為相似。但是克羅斯更全面，防守好，有硬度，皮亞尼奇相對來說主要差在防守。有的球迷認為J羅可以代替皮亞尼奇，但是J羅的盤帶擺脫能力缺乏，速度偏慢跑動能力不強、球隊前場逼搶時和防守端貢獻不大，因此只用J羅單純的代替皮亞尼奇，對於球隊的提升不大。

皇馬陣中的卡塞米羅在中場的防守作用明顯，而且有一腳遠射絕活，依照這一模板看，尤文圖斯目前陣中的中場球員僅有馬圖伊迪可以與之比肩，尤其是在防守以及積極性上，馬圖伊迪更是彪悍。而在進攻上說，馬圖伊迪擁有插入禁區的能力，但是射門能力差，尤其是遠射威脅更是幾乎為零。埃姆雷-詹的防守能力與馬圖伊迪相比差一些，尤其是速度方面是軟肋，進攻上與馬圖伊迪相比也是半斤八兩。不過詹更加年輕，跑動能力對於如今的尤文愈發重要。

鑒於此，因此如果尤文想打造與皇馬相似的中場配置，需要一位在進攻端以及防守端都擁有出色能力的中場才可以彌補皮亞尼奇以及馬圖伊迪（埃姆雷-詹）身上的軟肋。而如果比照皇馬的另外一位中場莫德里奇的模板，這位引進的球員還應該具有速度優勢以及靈動技術。

目前尤文圖斯已經確定在夏窗簽下拉姆塞，拉姆塞的進攻能力不錯，速度快突擊能力強，同時在防守上的身體比較硬朗，以上這些也是拉姆賽的優勢。拉姆塞的加盟使得尤文圖斯的中場配置更加完善，而如果能夠簽下傳聞已久的里昂獸腰恩東貝萊，對於尤文中場將是極大的補充，法國人是標準的B2B型中場，模板是亞亞-圖雷，無論進攻防守都能幫助到斑馬軍團。

前鋒：迪巴拉成雞肋 豪賭薩拉赫伊卡爾迪？

C羅在皇馬效力期間的位置作用與現在尤文圖斯的位置作用差距不大。而在中鋒位置上，本澤馬與曼朱基奇的特點有些不同，本澤馬的優勢是靈動的跑位以及回撤的拿球組織，而曼朱基奇的優勢是頭槌以及防守積極性。基恩這位小妖雖然本賽季發揮不錯，但是在對戰阿賈克斯的次回合比賽中卻展現了不會為隊友做球的軟肋，同時在禁區內的支點作用上也無法與曼朱相提並論。

如果從三連冠的皇馬與現在的尤文對比，目前尤文鋒線最軟肋的位置是邊鋒，誇德拉多下賽季會有31歲，速度耐力會走下坡路，科斯塔突破能力強，但是傳中精度不夠，而且非常容易受傷連續性不強。迪巴拉從來就不是一名邊鋒，他的存在在尤文非常雞肋。貝爾納代斯基是目前尤文最可靠的邊鋒。

而我們看看C羅在皇馬效力時期戰艦的邊鋒配置，貝爾、阿森西奧以及巴斯克斯都擁有出色的邊路技術，而且由於邊鋒群的存在，因此皇馬還可以踢442陣型，在球隊進攻時非但兩名前鋒不用頻繁套邊接應，就連邊前衛也能隨時插上去充當接應點，而在禁區內接應點多也會使得C羅得到的機會更多，畢竟C羅的頭槌能力是頂級水準。

因此從以上分析，尤文圖斯在今夏賣掉迪巴拉以及科斯塔，尋找強力的邊鋒應該是第一選擇。目前尤文圖斯與佛羅倫薩的邊鋒小基耶薩的緋聞很多。小基耶薩是一名右邊鋒，也能打影鋒或者左邊鋒的位置。速度快是小基耶薩的優點，他經常能夠憑藉自己的速度優勢突破對手防線，傳球成功率非常高，這使得在他突破成功後無論是選擇內切射門還是邊路傳中都會製造足夠的威脅。至於像費力克斯、薩拉赫等目標，雖然也與尤文傳出緋聞，但或身價過高、或操作難度太大，基本很難真正達成轉會。

在中鋒位置上，尤文一直與國米的中鋒伊卡爾迪傳出緋聞，但伊卡爾迪是一名機會主義型射手，與本澤馬大相徑庭，同C羅恐怕很難共存，引進他的戰術意義不大。另外伊瓜因將在賽季結束後租借回歸，是賣是留，尤文還需要做出定奪，事實上效力皇馬時期伊瓜因和C羅有過搭檔經歷，尤文倒不妨實驗一下這對組合。
