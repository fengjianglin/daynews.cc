謝謝你尤文圖斯，拜仁十年復興的原點

2019-12-05 0條評論 16次閱讀 0人點贊

最近十年，最頂級的歐洲足壇豪門，無疑是皇薩仁。2010年至今闖入歐冠四強的次數，這三傢俱樂部排在前三位。作為拜仁慕尼黑的球迷，今天筆者來跟大家回顧一下拜仁近10年崛起的原點。

2009年4月27日，在主場0：1輸給沙爾克04以後，時任拜仁主教練克林斯曼閃電下課，克林斯曼對拜仁的改革正式宣告破產。本打算退休的海因克斯臨危受命，帶領拜仁完成了餘下的比賽。那個賽季的拜仁混亂不堪，不僅歐冠恥辱性的0:4輸給巴塞羅那慘遭淘汰，在聯賽中還被馬加特率領的沃爾夫斯堡5：1痛擊；拜仁的未來顯得岌岌可危。賽季結束以後，拜仁邀請一代名帥范加爾教，意在徹底改變過去的技戰術打法，重新打造新的球隊。

羅本與范加爾

范加爾上任後，重用以穆勒、巴德斯圖貝爾為首的年輕球員，對托尼，克洛澤等功勛球員冷落無比，甚至將盧西奧賣給了國際米蘭；這讓拜仁的更衣室出現了不和諧因素。賽季開始後，拜仁踢出了40多年來的聯賽最差開局，即便是在引進羅本之後球隊的賽季之路還是進行的異常艱難，而歐冠聯賽小組賽兩負波爾多更讓拜仁基本上出線無望。不過也許是范加爾命不該絕，就在這內外交困的時候，同組對手尤文圖斯倒數第二輪意外敗給波爾多，這讓拜仁的歐冠之路出線了一線轉機。不過當時球隊的情況十分不好，在最後一場客場對陣尤文的生死之戰前，拜仁隊中最重要的兩位球員里貝里和羅本預計都無法上場，就這樣拜仁迎來了賽季最重要一戰。 在比賽開始之前沒有人看好拜仁能從尤文主場帶走3分，果然，第19分鐘德米凱列斯中場丟球導致拜仁0：1落後，此時的拜仁已經到了最危急的關頭。可是讓人沒想到的是丟球後的拜仁彷彿換了只球隊，眾志成城，氣勢如虹，連入4球，逆轉取勝，從尤文手中搶得了小組出線權。經此一戰，拜仁猶如打通任督二脈，在之後的比賽中屢屢化險為夷，反敗為勝，最終在2001年以後再次闖入歐冠決賽，聯賽中以5分的優勢力壓沙爾克奪冠。

尤文1：4拜仁

在此之後，拜仁逐漸找回失去的榮光，重新奠定了歐洲頂級豪門的地位。刨根問底，這一切的一切都是從2009年的客場對陣尤文的那場命運之戰開始。如果那場比賽拜仁最終沒能逆襲，那之後的道路恐怕又會是另一番模樣，至少拜仁的復興之路會更加漫長。所以那場對尤文的生死對決無疑是拜仁近十年復興的原點。
