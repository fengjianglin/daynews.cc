西甲：

西班牙人1-3馬德里競技

北京時間11月10日晚23:00，2019-20賽季西甲聯賽第13輪迎來一場焦點之戰，武磊效力的西班牙人做客萬達大都會球場挑戰強敵馬德里競技，本場比賽武磊進入先發陣容。上半場比賽武磊表現活躍，險些為球隊首開紀錄，西班牙人由達德爾打進精彩進球，馬競在半場讀秒階段扳平比分，兩隊暫時戰成1-1平。下半場比賽莫拉塔為馬競反超比分，武磊在第59分鐘被換下場，科克最後時刻為主隊鎖定勝局。最終西班牙人客場痛失好局，1-3負於馬德里競技。

英超:

利物浦3-1曼城

北京時間11月11日凌晨0時30分，2019-2020賽季英超聯賽第12輪焦點戰在安菲爾德球場打響，領頭羊利物浦主場3-1輕取衛冕冠軍曼城，新賽季開局取得11勝1平不敗成績，一枝獨秀領跑積分榜。利物浦領跑優勢擴大至8分，比曼城超出9分；藍月亮跌至積分榜第四位。法比尼奧第6分鐘遠射破門，薩拉赫第13分鐘頭槌破門擴大領先優勢，馬內第51分鐘再下一城。B席第78分鐘扳回一分。

曼聯3-1布萊頓

北京時間11月10日晚22:00，2019-2020賽季英格蘭超級聯賽第12輪的一場比賽拉開帷幕，曼聯在老特拉福德球場迎戰「海鷗軍團」的布萊頓。上半場比賽，第16分鐘，佩雷拉一腳勁射打破賽場上的僵局，第19分鐘，麥克托米奈補射造成普羅佩爾的烏龍球。下半場比賽，第64分鐘，鄧克利用角球機會將扳回一城，2分鐘後，拉什福德再下一城將比分再次擴大。最終全場比賽結束，曼聯在主場以3-1的比分戰勝了布萊頓。
