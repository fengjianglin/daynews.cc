怎麼回事？曼聯今夏離隊四人，全都爆發了！一人已奪得一冠

2019-11-17 0條評論 17次閱讀 0人點贊

今年夏天，曼聯送走了不少球員，其中斯莫林和桑切斯是租借出去的暫且不說，而盧卡庫和達米安是被曼聯賣掉的，埃雷拉和瓦倫西亞則是自由離隊的，這些人離隊之後表現怎麼樣？近日，《曼徹斯特晚報》統計了一下四人目前的情況，這四名球員全都爆發了，他們都成了各自球隊的主力，而且還有一人甚至已經拿到一個冠軍了，不得不說這對曼聯來說是很尷尬的一件事。

其中，曼聯隊長瓦倫西亞離開曼聯之後，回到了厄瓜多的基多大學俱樂部，在那裡瓦倫西亞是球隊絕對的主力右後衛，目前瓦倫西亞帶隊闖入了厄瓜多杯的決賽。當然，瓦倫西亞的表現和埃雷拉比起來還稍微差點，人家埃雷拉已經拿到一個冠軍了，加盟巴黎之後，埃雷拉就受傷了，但是只要他健康就是球隊的首發，目前他已經幫助巴黎首發了7次，同時幫助巴黎拿下法國超級盃的冠軍，而且按照巴黎的實力，本賽季的法甲冠軍肯定少不了。

達米安在曼聯是絕對的邊緣人，不過回到意甲賽場之後，他成為了球隊的絕對主力，在帕爾馬他代表球隊首發出場8次，而且還幫助球隊戰平國米，戰勝羅馬，現在達米安出色的表現也被國米看在眼裡，國米希望可以挖角達米安了，這也是對他的一種認可。至於盧卡庫，聯賽11次首發打進9球，位居意甲射手榜第二的位置，光是這些足以說明盧卡庫的表現，而且近期盧卡庫的表現越來越好，他也成為了國米球迷的新寵兒。

事實上，不止這四名球員，即便是租借離隊的桑切斯和斯莫林也表現不俗，桑切斯在賽季初也逐漸找到了狀態，4場比賽2次首發，打進1球助攻1次，可惜的是他受傷了，不過受傷前他也展現了自己的實力。至於斯莫林？意甲鏟球王，意甲爭頂王，而且本賽季他還沒有被人過掉過，光看數據，他比范迪克都要強，近期不少媒體表示，羅馬希望買斷他了。

看到這些離開曼聯的球員表現非常出色，不知道索爾斯克亞心中是什麼樣的想法呢？特別是斯莫林、盧卡庫和埃雷拉三人現在的表現越來越好，而現在曼聯卻在為尋找前鋒，尋找一個中場，尋找馬奎爾的搭檔而煩惱，如果今夏索爾斯克亞不送走這些球員，或許曼聯的陣容已經成型了！
