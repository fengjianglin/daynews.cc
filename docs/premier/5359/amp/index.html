過掉門將盧卡庫突到門前爆射 遭VAR極端判罰後國米大將造3球回應

天天要聞
3 週 ago

北京時間11月28號凌晨，歐冠小組賽第5輪繼續進行，落後多特3分僅排在小組第三的意甲豪門國米今天作客中赫體育場對陣布拉格斯拉維亞，趁著這輪巴薩對陣多特，這也是國米重新奪回出線希望的一戰。

國米開局果然也不錯，第18分鐘，國米後場斷球發動反擊，弗雷德里奇解圍失誤，盧卡庫將球卸下之後用身體優勢硬是抗開布拉格斯拉維亞2名防守球員殺入禁區右側倒三角回敲，中路馬丁內斯拍馬趕到直接凌空掃射得手，國米1：0領先。

第34分鐘，布拉格斯拉維亞後場連續倒腳，弗雷德里奇在馬丁內斯上搶的時候傳球失誤被斷，馬丁內斯單刀無私橫傳，盧卡庫跟上輕鬆推射空門得手。不過隨後主裁馬齊尼亞克表示要觀看VAR，因為這個進球之前布拉格斯拉維亞右路進攻未果，國米才反擊到前場。

那一球庫法爾右路下底傳中，德弗萊在搶點時放倒奧萊因卡犯規，馬齊尼亞克在吹掉盧卡庫進球的同時，國米還被吹罰點球，索切克操刀主罰命中，布拉格斯拉維亞1：1扳平比分。就這個球來說，德弗萊犯規問題也不大，只不過馬齊尼亞克沒有很快吹停比賽，直到盧卡庫將球打進再來冰火兩重天的極端改判也是讓國米球員很沮喪。在那之後國米氣勢上就差了一截，他們多次進攻無果，第64分鐘盧卡庫，布羅佐維奇又相繼射門被門框擋出。

直到第81分鐘，拉扎羅後場左側長傳反擊，布拉格斯拉維亞後防解圍失誤，盧卡庫高速插上將球拿住，面對出擊到禁區外的門將科拉什，盧卡庫趟一大步過掉科拉什，面對空門盧卡庫並沒有射門，而是帶到門前才大力爆射，這一爆射也終於是讓盧卡庫出了一口悶氣，激情慶祝的孔蒂也因此吃到黃牌。之後盧卡庫又做球，馬丁內斯打入鎖定勝局的一球，補時階段盧卡庫還打入1球，只不過被吹越位在先，最終盧卡庫2傳1射獨造3球，國米客場3：1擊敗布拉格斯拉維亞。
