塞維利亞詛咒？！埃梅里到馬欽 25天四位前主帥下課

2019-12-24 2,000條評論 3,465次閱讀 331人點贊

本賽季歐洲聯賽已經過半，除英超之外其他聯賽基本都進入冬休。隨著聯賽逐漸深入，換帥潮也一波波襲來。就在昨晚，西班牙人官方宣布了主帥馬欽下課，而他也成為了過去一個月里，第4位慘遭下課的前塞維利亞主帥。

11月29日 埃梅里 OUT（阿森納）

執教塞維利亞時間：2013.1.14-2016.6.12

所獲榮譽：歐聯杯冠軍x3

作為教授溫格接班人，埃梅里頂著巨大壓力接手阿森納。熬過第一個艱難賽季後，西班牙人終究沒能挺過11月。13輪英超4勝6平3負，埃梅里在槍迷一致首肯下，結束其在酋長球場17個月的工作。

12月9日 桑保利 OUT（桑托斯）

執教塞維利亞時間：2016.7.1-2017.5-31

帶領桑托斯以巴甲亞軍結束本賽季後，桑保利辭去了球隊主帥一職。桑保利去年底接手桑托斯，也算成功帶隊取得了不錯成績。他在離開塞維利亞後，近2分在阿根廷國家隊和桑托斯的執教，都只維持了一年，可謂流年不利。

12月21日 蒙特拉 OUT（佛羅倫薩）

執教塞維利亞時間：2017.12.30-2018.4.28

上周六，佛羅倫薩在主場1-4慘敗羅馬後，炒掉了主帥蒙特拉。”小飛機”今年4月上任紫百合主帥，至今僅8個月時間，一共帶隊完成26場正式比賽，戰績5勝7平14負，進26球失36球，勝率僅19%。

12月23日 馬欽 OUT（西班牙人）

執教塞維利亞時間：2018.7.1-2019.5.15

上任匆匆，下課也匆匆！馬欽在西班牙人的經歷，甚至還不如他的前任加耶戈。在執教赫羅納取得輝煌後，馬欽在塞維利亞就未能收穫理想中的成績，而西班牙人則是將他推向了深淵。
