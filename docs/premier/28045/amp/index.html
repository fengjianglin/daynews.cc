德甲賀芬咸VS多特蒙德+英冠米德爾斯堡VS斯托克城，5場分析+推薦

天天要聞
2019-12-20

​​

賀芬咸16輪聯賽戰罷取得了7勝3平6負的戰績，打進23球失27球積24分排名第9位，球隊的主場表現比較一般，8輪聯賽主場取得了3勝1平4負的戰績，打進12球失18球，場均失球數超過2球，可見球隊的防守端表現的並不穩定。

​

目前聯賽16輪過後，多特蒙德8勝6平2負，勝率只有50%，在拜仁本賽季如此糟糕發揮的情況下，多特並沒有把握好千載難逢機會，在爭冠中佔得先機，爭奪聯賽冠軍前景暗淡。今夏轉會窗口行動迅速，從德甲其他球隊引進多名實力派球員加盟，其中再次從拜仁吃回頭草，引進曾經在球隊效力的胡梅爾斯加強球隊後防線實力。

【綜合分析】

從雙方近10次的交鋒記錄來看，霍芬海姆取得1勝6平3負，平局相對較多。同時，霍芬海姆近年來主場面對多特蒙德都能夠保持不敗。另外，各位要留意，霍芬海姆近年來都能夠拿下數據。不過，從實力上看，確實是多特蒙德高出不少。亞洲玩法開出客讓半球/一球均水的指數，後市降為客讓半球生死盤，機構意圖明顯，出氣給與客隊半一的力度，對客隊較大的支持力度，阿俊看好多特順利打出。

推薦

03；30 德國甲組聯賽 賀芬咸VS多特蒙德 【多特蒙德讓-0.5】

​​

米德爾斯堡在上輪的聯賽中客場1-3輸給斯旺西，球隊最近的4場聯賽中僅僅贏下1場比賽，整體的表現低迷。雖然球隊的攻擊端在最近的5場比賽中僅有1場比賽沒有進球，但是防守端近4場比賽僅有1場零封，攻守表現失衡。值得關注的是，球隊最近的2個主場全勝，並且連續4場比賽不敗，主場具備優勢。

客隊斯托克城作為一支老牌球隊本賽季的表現依舊不理想，這已經是連續數個賽季在聯賽中需要為保級而戰，目前他們在積分榜上僅僅排在第22位，距離安全區還有2分的差距，因此天空之城在聯賽中面臨著相當的壓力。球隊本賽季的主場發揮非常糟糕，主場積分更是排在聯賽倒數第二，雖然客場成績也不理想，但是他們卻拿到了2場勝利。


【綜合分析】

各大主流機構初指給到了斯托克城平手盤搭配低賠付指數，後市受注以後大部分機構統一地將指數升到了斯托克城客讓平半球搭配超高賠付指數，結合兩隊上賽季交鋒記錄來看，斯托克城客讓平半球的指數對於客場作戰的斯托克城支持力度極大，但是這與基本面完全背離，米德爾斯堡上賽季兩次面對斯托克城均能保持不敗，判斷此番操作當是機構刻意利用斯托克城近期的兩場不敗來強拉上盤，有誘上嫌疑，因此本場比賽看好主場作戰的米德爾斯堡能夠捍衛自己的主場。
