給義大利潑盆冷水，歐預賽十連勝但含金量並不高

義大利在曼奇尼的帶領下，今年可謂是面目一新，球隊在歐預賽上取得了10場全勝的戰績，早早提前出線，球隊打進了37個進球，只丟了4個球。另外還創造了豪取11連勝的壯舉，同時2019自然年取得10場勝利也是一項新的紀錄。在剛結束的一場預選賽中9:1大勝亞美尼亞，相信義大利的球迷這輩子也是第一次看見義大利在正式比賽中打進這麼多進球。

義大利的一切都朝著有利的方向發展，要知道去年的世界盃是義大利的一場災難，球隊60年來首次缺席世界盃決賽階段比賽，當時的主教練文圖拉也成為了眾矢之的。文圖拉絕對是有責任的，包括用人不當，陣容打法一直不夠堅定，甚至在球隊第二回合落後時，都還想著換上防守球員，而不是進攻球員，連搏一搏的勇氣都沒有。

但是義大利的沒落也絕對不是文圖拉一人之鍋，其實早在06年世界盃奪冠之後，義大利就盡顯疲態，10,14兩屆世界盃都未能小組出線，大牌球星逐漸退役，新人成長的速度不夠快，現在的義大利隊中，並沒有絕對的巨星。

而曼奇尼執掌帥位，可以說是一次對的時機遇到對的人。當然曼奇尼的運氣也比文圖拉好，文圖拉當時預選賽和西班牙一組，最後獲得小組第二參加附加賽也是預料之中的事。而曼奇尼預選賽和芬蘭，希臘，波黑，亞美尼亞，列支敦斯登分在一組。這個小組，除了希臘在04年有過曇花一現之外，其他球隊在歐洲基本沒有拿的出手的戰績，而且歐洲杯擴軍，小組第二都可以直接出現，這又無形中降低了球隊的壓力。

當然曼奇尼是有能力的，但是對曼奇尼的印象更多的是停留在「虐菜」上。07-08賽季執教國米時，那個賽季國米僅僅領先羅馬3分奪冠。而在最後的幾輪比賽中，國米顯示主場未能取勝競爭對手羅馬，然後30輪主場輸給了尤文，36輪米蘭德比中又落敗，最後靠穩定的贏了幾隻弱隊後驚險奪冠。而在歐冠上國米則連續幾年被比利亞雷亞爾，瓦倫西亞這樣的西甲第二檔球隊淘汰。而在那個賽季下課後，穆里尼奧接手打磨了兩年就拿到了歐冠。到了曼城後，感覺曼奇尼執教能力還是有進步的，包括2012年雙殺曼聯，最後一輪聯賽逆轉奪冠。但在歐冠方面，曼城連續兩年小組賽都未能出線。

可見曼奇尼更適合打聯賽，因為聯賽賽程長，弱隊更多，而真正硬碰硬的比賽中，曼奇尼並沒有太多的證明自己。所以本次義大利取得這麼優異的成績，是值得肯定的，但也不可太過樂觀，明年的歐洲杯才是曼奇尼的一次大考，義大利的連勝真強還假象，明年就能見真曉。
