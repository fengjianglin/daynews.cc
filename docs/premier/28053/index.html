留洋首站去小球隊，轉會費幾十萬歐，日本國家隊主力是這樣煉成的

2019-12-20 54條評論 1,239次閱讀 244人點贊

2018世界盃結束後，森保一成為日本國家隊新主帥，併兼任日本國奧隊主帥。

森保一

上任以來，森保一對日本國家隊的換血如今已完成，年輕一代的日本國腳已挑起大梁。在目前的日本國家隊中，除長友佑都、吉田麻也這兩位經驗豐富的老將年過30外，其餘主力球員年紀都沒到30歲。

格外值得留意的一點是，除門將外，目前日本國家隊的主力球員，全都是早早便留洋，留洋第一站都是去了小球隊，並且轉會費相當低；而留洋的第二站，往往是更上了一層樓。當時，這些球員是各自J聯賽球隊的絕對主力，但當球員有機會留洋時，各J聯賽球隊普遍選擇了放行。

吉田麻也

2010年1月，21歲的吉田麻也開始留洋，從名古屋鯨轉會至荷甲球隊芬洛。

2012年8月，吉田麻也從芬洛轉會至英超球隊南安普頓，轉會費280萬歐元。

冨安健洋

2018年1月，19歲的冨安健洋開始留洋，從福岡黃蜂轉會至比甲球隊聖圖爾登，轉會費80萬歐元。

2019年7月，冨安健洋從聖圖爾登轉會至意甲球隊博洛尼亞，轉會費上升至900萬歐元。

長友佑都

2010年7月，23歲的長友佑都開始留洋，從FC東京租借至意甲球隊切塞納。

2011年1月，憑藉在意甲半個賽季與2011亞洲杯上的出色表現，切塞納正式從FC東京簽下長友佑都，轉會費為160萬歐元。

僅僅3天後，國際米蘭便從切塞納租借來了長友佑都。

2011年夏天，國際米蘭便正式從切塞納簽下了長友佑都，轉會費450萬歐元。

酒井宏樹

2012年7月，22歲的酒井宏樹開始留洋，從柏太陽神轉會至德甲球隊漢諾威96，轉會費120萬歐元。

2016年7月，酒井宏樹離開漢諾威96，自由轉會至法甲球隊馬賽。

柴崎岳

2017年1月，24歲的柴崎岳開始留洋，從鹿島鹿角自由轉會至西乙球隊特內里費。

2017年7月，柴崎岳從特內里費自由轉會至西甲球隊赫塔費。

遠藤航

2018年7月，25歲的遠藤航開始留洋，從浦和紅鑽轉會至比甲球隊聖圖爾登。

2019年8月，遠藤航從聖圖爾登租借至德乙球隊斯圖加特。

堂安律

2017年7月，19歲的堂安律開始留洋，從大阪鋼巴租借至荷甲球隊格羅寧根。

2018年7月，堂安律正式從大阪鋼巴轉會至格羅寧根，轉會費170萬歐元。

2019年8月，堂安律從格羅寧根轉會至荷甲豪門埃因霍溫，轉會費升至750萬歐元。

中島翔哉

2017年8月，23歲的中島翔哉開始留洋，從FC東京租借至葡超球隊波爾蒂芒人。

2018年夏天，此前一個賽季葡超29場進10球的中島翔哉，正式從FC東京轉會至波爾蒂芒人。

2019年2月，中島翔哉從波爾蒂芒人轉會至卡達球隊杜海勒，轉會費高達3500萬歐元。

2019年7月，中島翔哉從杜海勒轉會至葡超豪門波爾圖，轉會費1200萬歐元。

南野拓實

2015年1月，20歲的南野拓實開始留洋，從大阪櫻花轉會奧地利球隊薩爾茨堡紅牛，轉會費80萬歐元。

2020年1月，南野拓實將正式從薩爾茨堡紅牛轉會至利物浦，轉會費850萬歐元(725萬英鎊)。

原口元氣

2014年7月，23歲的原口元氣開始留洋，從浦和紅鑽轉會至德甲球隊柏林赫塔，轉會費50萬歐元。

2018年7月，原口元氣從柏林赫塔轉會至德甲球隊漢諾威96，轉會費升至450萬歐元。

大迫勇也

2014年1月，23歲的大迫勇也開始留洋，從鹿島鹿角轉會至德乙球隊慕尼黑1860，轉會費50萬歐元。

2014年7月，大迫勇也從慕尼黑1860轉會至德甲球隊科隆，轉會費160萬歐元。

2018年7月，大迫勇也從科隆轉會至德甲球隊雲達不來梅，轉會費升至450萬歐元。
