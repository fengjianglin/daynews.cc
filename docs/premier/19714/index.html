歐冠淘汰賽對手出現，8隻豪門造就了4場精彩對決，球迷們有福了

2019-12-17 846條評論 3,379次閱讀 498人點贊

北京時間12月16日，歐冠淘汰賽抽籤結果出爐，本次淘汰賽的結果也出乎大家所料，雖然沒有球迷朋友們期待已久的尤文圖斯和皇馬對決，利物浦和皇馬對決，巴薩和切爾西對決，切爾西和尤文圖斯對決這種重磅好戲的上演，但是8隻豪門依然將會為大家帶來4場巔峰對決，其中有幾場淘汰賽更是可以被稱為提前上演的決賽，是火星撞地球的對決。

最吸引大家的無疑就是皇馬和曼城的對決，這場比賽是真正意義上的提前上演的決賽，皇家馬德里本賽季狀態不錯，齊達內回歸之後給這支球隊帶來了很大的改變，年輕球員的崛起讓現在的這隻皇馬比上個賽季強大不少，本賽季也是歐冠冠軍的大熱門，而他們的對手曼城是本賽季最被看好的一支球隊，整體陣容確實非常強大，更主要的是他們的主教練瓜迪奧拉曾經9次擊敗皇馬，這是一個非凡的紀錄，兩隻球隊必定會帶來一場精彩紛呈的對決。

第2場豪門對決就是馬德里競技vs歐冠衛冕冠軍利物浦，這場比賽球迷們大部分還是看好利物浦獲勝，雖然馬德里競技整體實力不弱，但是本賽季在歐冠和聯賽都沒有太強勢的表現，更重要的是球隊目前處於陣容的換血期，夏季轉會窗口大量主力球員的離開影響了他們的實力，而利物浦本賽季更是狀態飽滿，在英超賽場上一騎絕塵，強大的陣容和控制力讓歐洲任何一支球隊都不敢與他直接對抗，本場比賽看好利物浦打敗馬競。

第三場巔峰對決就是大黃蜂對陣大巴黎，這兩支隊伍都是歐冠淘汰賽上的常客，雖然在最近幾年都沒有能夠更進一步，但是兩支球隊整體在歐洲也算還可以，當然比較之下大巴黎顯得更有衝擊力，大黃蜂軍團雖然青年軍很多，但是整體實力稍遜一籌，大巴黎上個賽季被曼聯逆轉淘汰，本賽季捲土重來一定要顛覆歷史，年輕的大黃蜂很難抵禦住大巴黎的狂轟亂炸。

第4場巔峰對決是切爾西對戰拜仁慕尼黑，這場比賽誰也不敢說一定可以贏，要知道現在的拜仁慕尼黑可是處於換帥期的動蕩，新的主教練還沒有到來，這是南大王一個重大的隱患，切爾西方面雖然也是新帥帶隊，但是蘭帕德本賽季表現出色，用成績證明他是一位合格的主教練，更重要的是切爾西的青春風暴實在太過強大了，這場比賽拜仁慕尼黑很難報當年決賽一箭之仇，還是看好切爾西方面戰勝他們。

尤文圖斯和巴塞羅那基本上是抽到了上上籤，他們的對手雖然實力不弱，但是相對於他們來說還是缺乏競爭力，亞特蘭大和瓦倫西亞兩支球隊實力不分伯仲，這場比賽也是一場激烈的廝殺，至於萊比錫紅牛和熱刺相遇，穆里尼奧一定會給這隻歷史第1次進入淘汰賽的球隊，一個深刻的教訓，相對於上面的4場比賽，這些比賽的觀賞性就不高了，就讓我們期待歐冠1/8決賽的4場巔峰較量。
