歐冠小組賽積分榜：1隊滿分2隊拿16分，5大聯賽均分8個小組第一

2019-12-12 858條評論 1,885次閱讀 312人點贊

北京時間今天凌晨，歐冠小組賽全部結束，16支出線隊全部確定。6輪小組賽中踢出了不少經典戰役，如拜仁客場7-2橫掃熱刺，利物浦主場4-3險勝薩爾茨堡紅牛，皇馬主場2-2戰平大巴黎等。而小組第一的分布也比較平均，英超2個，西甲2個，德甲2個，意甲1個，法甲1個，這也為後面的淘汰賽抽籤提供了更多的可能性。

A組中，大巴黎一騎絕塵早早出線，小組賽取得5勝1平相當強勢，皇馬積11分，這已經是他們在近二十年來，小組賽中積分較少的一次了，一場未勝的布魯日積3分拿到了一個歐聯杯資格。B組中，同樣是2強2弱的局面，拜仁6戰全勝轟進24球，早早拿到出線權，熱刺在前2戰積1分的情況下，依靠一波3連勝提前一輪獲得出線權。

C組中，曼城實力強大提前一輪鎖定小組第一，亞特蘭大和頓涅茨克礦工、薩格勒布迪納摩糾纏到最後1輪，最後成功逆襲晉級，成為歐冠改制來首支3連敗開局還能出線的球隊，這主要得益於1強3弱的小組格局。D組中，尤文圖斯和馬競出線，勒沃庫森在前3戰全敗的情況下，還能將懸念保留到最後一輪，已經十分了不起了。

E組中，利物浦和那不勒斯直到最後一輪才確定出線，第3名薩爾茨堡紅牛逼得相當緊，而亨克小組賽僅積1分，是拿分最少的球隊之一。F組中，巴薩提前拿到出線權，多特蒙德和國米糾纏到最後時刻，最終多特蒙德勝出，而國米遺憾地連續2次在最後一輪被淘汰出局，只能參加歐聯杯比賽。

G組是實力最平均的一個小組，結果萊比錫RB提前出線，而剩餘的3支球隊直到最後一輪才排定座次，澤尼特從出線在望到歐聯杯資格都沒撈到，成為唯一出局的種子隊。H組中，瓦倫西亞、切爾西和阿賈克斯糾纏到最後一輪才見分曉，結果阿賈克斯在大好局勢下出局，而法甲亞軍里爾則成為「魚腩」，6戰僅拿到1場平局，表現糟糕。
