壯士暮年有凌雲？卡瓦尼入駐床單軍團或能煥發第二春

2019-12-23 170條評論 1,388次閱讀 325人點贊

樂魚體育訊 根據迪馬濟奧個人網站的報道，卡瓦尼將與馬德里競技俱樂部簽署一份3年的合同，現年32歲的卡瓦尼合同還有半年到期，但有權力和其他任何的俱樂部簽署一份提前合同，然後在6月加入他們，雖然卡瓦尼有諸多選擇，但是他的心只屬於馬競。而且西蒙尼早就有意卡瓦尼作為自己球隊的」進球「拼圖。關於卡瓦尼加盟床單軍團，已是馬競球迷多年以前就開始盼望的事情了，如今盼了多年的理想轉會終於快要實現了！

很多球迷第一次認識卡瓦尼，應該還是在10年南非世界盃上，那時候與弗蘭和蘇亞雷斯組成的前場三叉戟大放異彩，不過那個時候，大家印象最深的是弗蘭的金球獎以及五球並列射手榜第一。在世界盃結束後卡瓦尼來到了那不勒斯，說到那不勒斯我們會想到這兩個賽季默滕斯、因西涅和卡列洪的三叉戟。而卡瓦尼正是他們的前輩，那一年他與拉維奇和哈姆西克三人在意甲大殺四方。卡瓦尼更是轟入26球，拿到意甲銀靴，併入選賽季最佳陣容，幫助那不勒斯拿到歐冠資格。

在那不勒斯出色的表現很快就吸引了豪門的注意力。土豪入主後的巴黎雄心勃勃，他們希望能夠在鋒線上找到一名射手輔佐伊布。卡瓦尼成為了最終目標。2013年7月16日，卡瓦尼以6400萬的高價加盟巴黎。而他隨後的表現也沒有辜負這筆轉會費。

很多人都取笑卡瓦尼喜歡浪費機會，甚至把他空門不進打丟單刀的畫面做成集錦。那些視頻表情包在網上廣為流傳。即使法甲影響力不高，卡瓦尼都有取代本澤馬成為世界第一吐餅俠的趨勢。但事實上，卡瓦尼加入巴黎5個賽季的進球數從來沒有低於過20個，作為伊布的僚機卡瓦尼在13/14、14/15、15/16賽季分別打入25、31、25球。16/17賽季伊布離開後卡瓦尼交出了50場49球的成績單。即使是在一場普通法甲比賽中，32歲的卡瓦尼也會傾其所有，從對方禁區衝刺到本方禁區。當別人都已經絕望的時候，他還沒有放棄。

有的如繁星般耀眼，受世人敬仰。每個孩子都會把他們作為偶像，球衣印號出現頻率最高的也總有他們的名字。有的樸實無華，為人低調。我們踢球的時候或許不會模仿他們的動作，但球場上這些人從來都不可或缺。卡瓦尼，無疑是屬於後面一種。這些年卡瓦尼總是站在伊布、蘇亞雷斯、內馬爾們的身後，他似乎太過平凡以至於那些努力付出都未得到應有尊重。確實他沒有最頂級巨星那樣的天賦，最後願卡瓦尼不要因為傷病停下腳步！
