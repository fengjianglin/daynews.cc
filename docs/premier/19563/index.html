球員觀察：尤文和利物浦關注的圖盧茲後衛貢薩爾維斯

2019-12-18 373條評論 2,967次閱讀 996人點贊

作為法國足壇的青訓強隊，圖盧茲從不缺少年輕的青年才俊，從卡普埃，西索科，本耶德爾到托迪博們，年輕的潛力股總是能在圖盧茲發光發熱，而法國青年國腳，18歲的馬蒂厄-貢薩爾維斯（Mathieu Gonçalves）則有機會成為下一個。新賽季，這位年輕的左後衛-中衛搖擺人亮相法甲賽場，雖然在職業足壇的開局不算美妙，但潛力不俗的他依然得到了來自尤文圖斯和利物浦的關注，本文正是對貢薩爾維斯的介紹。

貢薩爾維斯（圖片：法國足協官網）

球員簡介

2001年6月8日出生於巴黎的馬蒂厄-貢薩爾維斯，擁有巴西血統。貢薩爾維斯很早就加入了圖盧茲的青訓體系，在圖盧茲青年隊效力期間，年輕的貢薩爾維斯憑藉自己的全面特點得到了教練的喜愛，無論是在圖盧茲梯隊還是法國青年隊，貢薩爾維斯都是球隊左路的主力人選。

transfermarkt上貢薩爾維斯代表法國青年隊的出場記錄

2017年16歲的貢薩爾維斯遭遇了十字韌帶撕裂的打擊，不過年輕的他並未沉淪，半年之後重回賽場的他繼續穩步提升，去年夏天，他也正式成為了職業球員。上賽季他主要在圖盧茲二隊效力，而本賽季，被提拔進入圖盧茲一隊的貢薩爾維斯已經在法甲聯賽出場4次。同時，貢薩爾維斯一直是法國各級別青年隊的主力，他代表法國U16-U19都有過出場。

技術特點

身高178cm，體重68kg的貢薩爾維斯在場上是一位邊中衛搖擺人，年輕的他可以勝任左後衛、左翼衛和中衛的位置。

transfermarkt上貢薩爾維斯的個人資料

在青年隊比賽中，貢薩爾維斯展現出來的潛力是出色的：在邊路他善於上搶完成斷球，此外在進攻端，貢薩爾維斯亦能利用自己的帶球製造威脅，在比賽中，他的扣球變向擺脫非常有效。此外，貢薩爾維斯也善於利用自己在左路精準的斜長傳助攻隊友得分。

不過本賽季在圖盧茲一隊，貢薩爾維斯卻交了不少「學費」，他在出場的4場比賽里送出了2記點球，並且打進一粒烏龍球，本賽季初登法甲賽場的他，可以說發揮糟糕。在禁區內，貢薩爾維斯缺乏經驗，不管是上搶還是鏟斷動作都有些魯莽，這也讓他吃了不少虧。

Whoscored：貢薩爾維斯本賽季在防守端的數據

雖然在法甲的幾次出場里犯下了不少錯誤，但我們依然能看到貢薩爾維斯的一些特質，他在攔截球路上具有一定的天賦，此外，他果斷的上搶也能給球隊帶來幫助。作為邊中衛，貢薩爾維斯在速度和爆發力上沒有太大優勢，不過他的柔韌性，平衡能力和轉身速率還是不錯的。

轉會傳聞

雖然本賽季沒能在法甲送出太好的表現，但貢薩爾維斯依然憑藉自己不俗的天賦吸引了歐洲大俱樂部的關注。據隊報的報道，利物浦和尤文圖斯都有意在今年冬窗簽下18歲的貢薩爾維斯，作為球隊的後防儲備。其中，利物浦一直希望在左後衛位置引進未來儲備，年輕的貢薩爾維斯也進入了克洛普團隊的視野。而尤文圖斯更是一直在密切關注年輕的貢薩爾維斯，據GFFN援引隊報的報道，尤文圖斯的代表已經和貢薩爾維斯的經紀人進行過多次會面。
