告別皇馬？莫德里奇或登陸中超，金球先生冬窗被傳離開

天天要聞
2019-12-22

歐洲球員交易市場還有一周時間就將徹底拉開大幕，對頂級豪門來說，適當的補強以備戰下半賽季是最合理的方案，對合同即將到期的頂級球星來說，能夠獲得同其他球隊直接簽下大合同的機會也備受期待。然而對上半賽季表現不算突出，獲得機會不算太多的老將們來說，冬季轉會窗口自己不得不認真考慮一下職業生涯的未來了，效力於皇馬的金球先生莫德里奇，就面臨這樣的尷尬問題！

一年半前帶領皇馬連續第3年拿下歐冠獎盃，俄羅斯世界盃上帶領克羅埃西亞國家隊闖進決賽，自己更是憑藉連續穩定的表現一舉打破梅羅對金球獎長達10年的壟斷，皇馬中場大將莫德里奇在過去一年迎來了自己人生的巔峰。然而本賽季開始至今，這位明星中場在球隊中的地位大幅下降，除了賽季之初獲得了相對充足的出場時間，主教練齊達內更願意培養年輕的巴爾韋德，來為球隊的未來發展做打算！

34歲的莫德里奇新賽季代表球隊出場16次，貢獻了3粒進球送出5次助攻，客觀上數據方面魔笛依舊是那個能力突出的頂級中場，但球員加盟皇馬7年以來，過去這半個賽季可謂最失意。8次替補上場3次被替換下場，莫德里奇在球隊中的地位已經不再是那個中場核心了，除了巴爾韋德的強勢崛起，接下來皇馬還可能引進博格巴、范德貝克等中路好手，今年冬天轉會市場，如今看來正是魔笛離開的最佳時機！

球員去年夏天俄羅斯世界盃之後就被傳言可能加盟意甲豪門國際米蘭，儘管藍黑軍團一度為其開出了高達1000萬歐元的年薪，但是皇馬方面認為莫德里奇對球隊的作用太過明顯，因此交易最終沒能成行。今夏轉會市場上，AC米蘭也曾希望能夠得到這位頂級中場，為此甚至克羅埃西亞傳奇球星博班親自出馬進行遊說，但是經濟因素及合同年限最終導致了交易沒能成行，如今轉會市場上，魔笛的行情不容樂觀！

西班牙媒體《DonBalon》日前爆料，在皇馬隊內地位下降，出場時間被壓縮，年齡又來到34歲的莫德里奇，在冬季轉會市場上尋求離開。球員本人這樣的年齡，想要在米蘭城雙雄這樣的球隊中尋找位置並獲得相對較長的合同看似並不容易，然而該媒體認為，中超幾家球隊對魔笛的興趣，或許將促成這位世界級中場的離開，至於轉會費方面，為皇馬效力了7年之後，該媒體相信皇馬不會為難對方！

如果莫德里奇這位金球先生真的能加盟中超聯賽，對整個中國足球的提升都將是一件好事兒，儘管中超聯賽近年來沒少引進國際大牌外援，但像莫德里奇這樣的頂級中場卻並不多見。雖然保利尼奧、奧斯卡、胡爾克等球員同樣在歐洲足壇擁有極高的知名度，但世界足球先生加盟中超，這樣的消息確實足夠震撼！今冬轉會市場上，魔笛真的會同皇馬告別嗎，我們拭目以待！
