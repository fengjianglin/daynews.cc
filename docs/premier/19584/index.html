06世界盃戲耍布馮的米蘭名宿退役，義大利冠軍成員僅剩兩人在戰

2019-12-17 1,093條評論 4,759次閱讀 1352人點贊

要說在世界足壇哪個國家的後衛最被人青睞，那麼義大利人一定當仁不讓。即使現在義大利足球處在低谷時期，但是亞平寧後防線依然人才輩出。

不過要說到義大利後衛們最強盛的時期，那一定還是這個世紀的第一個十年，內斯塔、卡納瓦羅以及馬爾蒂尼等人創造了義大利足球的神話，當然在他們之中還有一個人不那麼耀眼，他就是扎爾卡多，一個博洛尼亞優秀右邊後衛。

義大利足球的鐵血後防

博洛尼亞出道

扎卡爾多出生在摩德納的一個小鎮上，那還是意甲的黃金年代，扎爾卡多在這樣的環境下也進步得非常快，博洛尼亞雖然算不上十分強大的球隊，但是扎爾卡多還是能夠快速成長並且獲得足壇的廣泛關注。

由於天生的身材優勢以及不錯的技術，扎卡爾多的年少時期是相對順利的，他一路從博洛尼亞的少年隊到成年隊幾乎沒有遭遇太多的阻力，只不過在2000-01賽季被租至丙級聯賽，短短一個賽季的蟄伏並沒有消磨掉扎爾卡多的意志。老教練馬科里尼對於這名球員青睞有加，他也表示扎卡爾多肯定會是能夠成功立足於意甲聯賽的球員。

回到意甲聯賽的扎卡爾多立即展現了不錯的戰鬥力，在20歲生日的前夜他代表博洛尼亞出場逼平了強大的羅馬隊。在面對「七姐妹」之一的拉齊奧時，扎卡爾多拿出了強硬的表現，他一次又一次的防守讓拉齊奧的豪華攻擊線無功而返。

與此同時，相應的義大利國家隊梯隊也有屬於扎卡爾多穩固的位置，他和帕爾馬隊的博內拉都是義大利U21的中堅力量。彼時的扎卡爾多已經受到了包括尤文圖斯和AC米蘭的豪門廣泛關注，像博洛尼亞這樣的小球會看起來很難留的下這名後防新星了。不過扎卡爾多還是表現了自己忠心以及對於職業生涯清楚的規劃，他決定繼續為博洛尼亞效力。

國家隊生涯，陰差陽錯

雖說扎卡爾多已經表現出了自己在右後衛位置上的能力，但是很多義大利名宿認為他腳下速度太慢，很難適應世界盃的快速節奏。不過里皮對於這名弟子相當看重，儘管扎卡爾多在世界盃預選賽的時候表現比較不成熟，里皮還是堅定地把首發右後衛的位置給了他。

由於首場比賽格羅索鎮守的左路遭到了對手的強力打擊，里皮決定把贊布羅塔放到左後衛的位置，本來贊布羅塔被認為是右後衛位置上的主力。不過扎卡爾多面對這樣的機會有些緊張，面對美國隊的第二場小組賽他打進了烏龍球。這也是「藍衣軍團」那屆世界盃上唯一的運動戰丟球。布馮另一次被洞穿大門就是決賽時齊達內的點球了。

里皮對於這樣的情況相當無奈，他只得大手一揮，把經驗豐富的贊布羅塔放回右後衛的位置頂替扎卡爾多，這樣的變化成全了義大利隊，也成全了黃健翔口中「偉大的左後衛」格羅索。

可以說這樣陰差陽錯的變化反倒是成全了義大利隊，沒有格羅索創造的點球，可能義大利隊就沒有拿下世界盃的機會了。

巴勒莫巔峰，踏入豪門卻為時已晚

2004年，扎卡爾多終於經歷了自己人生當中的一次轉折，剛剛升入意甲聯賽的巴勒莫急於招兵買馬，扎卡爾多作為右邊後衛新星，當時的身價還不算太高，立刻被招致麾下。在巴勒莫的生涯是扎卡爾多最輝煌的時期，他在巴勒莫總共效力了4個賽季，一直坐穩球隊的主力位置，也正是巴勒莫的歲月才讓他成為里皮最為信任的右邊後衛人選。

不過即便如此，還是沒有意甲頂級球隊想要簽下扎卡爾多，2006年世界盃雖然是他「意外」成就了「偉大的」格羅索，但是他在各大豪門球探的眼中剩下的便只是那次烏龍球失誤了。

他決定出國尋找機會，這次是德甲勁旅沃爾夫斯堡，那個時候狼堡正在經歷著球隊歷史上最為輝煌的時期之一，拿下隊史首座德甲冠軍。扎卡爾多也是球隊的奪冠功臣之一，只不過這一次，他還是沒能成為主角，整個賽季他幾乎一直只能以替補的身份出戰，不過好在獲得了自己的第一個頂級聯賽的冠軍。

扎卡爾多心生去意，他開始聯繫意甲球隊，帕爾馬和AC米蘭都對他有意，只不過扎卡爾多不敢前往米蘭。那時候米蘭還是爭冠球隊，扎卡爾多害怕自己會像在國家隊或者狼堡那樣沒有任何貢獻，於是他加盟了帕爾馬。

2013年，米蘭走向衰敗，這個時候扎卡爾多終於獲得了加盟豪門的機會。只不過米蘭已經不是那支米蘭，扎卡爾多也不是從前的扎卡爾多了。

在聖西羅度過兩年後，年過而立的扎卡爾多先後輾轉卡爾皮、維琴察、哈姆倫斯巴達、三朵花，並在7月9日，義大利奪得06年世界盃的13周年紀念日正式退役。至此，那支義大利隊中仍在延續球員生涯的只有在阿根廷「養老」的德羅西和重回尤文的「小將」布馮了。
