BOB體育與那不勒斯俱樂部簽署戰略合作協議

2019-12-18 259條評論 1,217次閱讀 247人點贊

義大利那不勒斯2019年12月17日 /美通社/ — 北京時間2019年12月12日，亞洲大型娛樂體育遊戲平台BOB體育(BOB Sports)與意甲球隊那不勒斯（S.S.C Napoli) 在義大利那不勒斯這座古老文明之城舉行了戰略合作新聞發布會。雙方公共關係發言人代表出席了此次新聞發布會並正式簽署了戰略合作協議。在接下來合作的時間，BOB體育將成為那不勒斯亞洲區域合作夥伴，與那不勒斯俱樂部攜手勇闖接下來的意甲與歐洲杯等重大魅力足球賽事。

BOB體育和那不勒斯俱樂部雙方代表人在展板上簽字

那不勒斯（Societa Sportiva Calcio Napoli)，起源於義大利南部，有著悠久而豐富的歷史。這位可以追溯到1904年的義大利足球巨人有過無數輝煌時刻，其中馬拉多納、安東尼奧·卡雷卡、西羅·費拉拉、安德里亞·卡內維爾和薩爾瓦多·巴尼等為亞洲球迷最為喜愛熟悉。那不勒斯一直以強勁的實力與不懈的拼搏證明自己，一直躋身於義大利足球界最受球迷喜愛的球隊榜單前列。這也與BOB體育追求品牌卓越與拼搏的企業精神不謀而同。此次合作更是雙方重大品牌戰略意義重要地一步。

對於此次合作，那不勒斯代表人卡洛-塞爾維翁表示：「BOB體育是一家實力雄厚的務實企業，作為亞洲行業內排名前列的品牌，擁有著行業頂尖的技術與謙虛拼搏的員工。這恰好也是那不勒斯俱樂部自身的發展中對自己要求的方向。我們感到非常榮幸能與這樣優秀的企業展開合作。作為行業領先的數據平台品牌之一他們的實力毋庸置疑。此次的合作是雙方品牌知名度國際化提升的一個重要標誌，也會成功使得那不勒斯俱樂部更為亞洲地區朋友們認識。」

那不勒斯商務合作代表人卡洛-薩爾維翁

隨後，BOB體育公關發言人尼科-麥格利亞也對這一合作發表了相關感言：「那不勒斯俱樂部就如同這座城市意義，擁有悠久輝煌的歷史。它對體育的熱情、對困難的不屈不撓，與我們BOB體育追求的企業精神有很深的共鳴。那不勒斯俱樂部願意成為我們的合作夥伴並將我們介紹給它的球迷朋友們，我們BOB體育同樣感到非常榮幸，相信在我們BOB品牌板塊不斷向世界邁進地同時能同樣給我們的用戶球迷朋友們帶來更多信心，BOB體育致力為大家帶來更多精彩地體驗。我們對此次合作都感到充滿信心。」

合約達成後雙方握手留念

BOB體育一直以報道世界範圍熱點體育賽事並提供各類線上平台遊戲為主要業務，在涉及報道的足球電競以及世界各地各種大型比賽以及電競聯賽等等，做到了實時更新實時播放。值得一提的是，BOB體育在品牌版圖拓張的短短几年內，已經成功攜手法甲里昂、德甲多特蒙德、意甲法甲里昂等歐洲俱樂部並簽訂了合作夥伴關係協議。而此次牽手的意甲那不勒斯，可以看出BOB體育品牌拓張的雄心與追求。
