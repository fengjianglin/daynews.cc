昨夜今晨，哪場最“冷”？並非巴薩客平，也非那不勒斯輸球

天天要聞
3 天 ago

北京時間的昨夜今晨，歐洲五大聯賽繼續開戰，其中歐冠小組出線的強隊，大多都取得了完勝。比如利物浦和馬競都是主場2-0戰勝對手，拜仁主場6-1狂掃不萊梅，多特4-0客勝美因茨，萊比錫紅牛3-0客勝杜塞爾多夫。不過，也有3支歐冠16強球隊沒能獲得勝利，其中巴薩客場2-2被皇家社會逼平，最令球迷關注。

巴薩客場沒能取勝，許多球迷都埋怨裁判的點球判罰尺度主客不一，更多的球迷則覺得巴薩「客場蟲」的老毛病又犯了，也有球迷認為梅西沒能像之前幾場比賽那樣狀態爆棚，是球隊沒能客場取3分的關鍵所在。

這些原因可能都或多或少存在，但巴薩之所以沒贏球，還有一個重要原因就是皇家社會並不弱，他們是聯賽的第4名（賽前），為了保住歐冠席位，皇家社會主場踢得非常積極，全場射門次數居然是巴薩的2倍，控球率、危險進攻次數也都佔有明顯優勢。巴薩客場拿1分，並沒有什麼委屈。

在周中的歐冠小組賽中，那不勒斯4-0狂勝亨克，但賽後俱樂部卻宣布安切洛蒂下課。率那不勒斯殺進歐冠16強的安帥，卻無緣率球隊征戰歐冠淘汰賽。北京時間12月15日（今天）凌晨1:30，那不勒斯坐陣主場迎戰帕爾馬。結果，安切洛蒂的下課並沒有改變球隊聯賽上的頹勢，那不勒斯依舊1-2輸給了帕爾馬，但這也算不上什麼冷門，畢竟賽前這兩隊就同積21分。加圖索的那不勒斯首秀遭遇開門黑，也在許多球迷的預料之中。

昨夜今晨，歐洲五大聯賽爆出的最大冷門，當數切爾西主場0-1輸給了伯恩茅斯（昨晚23:00開球），畢竟切爾西目前在英超排第4，而伯恩茅斯則排名第14。從全場進攻來說，切爾西不僅控球率高達61%，射門和射正次數也是伯恩茅斯的2倍。但全場唯一進球，卻是戈斯林在第84分鐘打入的一粒雜耍般的半倒鉤吊射。

這個進球，主裁判一開始判戈斯林越位，但隨後看VAR回放才發現，越位的並非戈斯林，於是改判進球有效。顯然，這一球門將凱帕的出擊有些問題，如果守在門前，戈斯林不可能這樣雜耍般破門。

賽後，切爾西主帥蘭帕德對結果也很失望，他認為自己的球隊踢得太緩慢太保守，不僅創造的絕對機會不夠多，最後還出現了愚蠢的丟球。
