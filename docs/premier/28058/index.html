尤文名宿：相信薩里的哲學 有個人能幫助尤文完成歐冠的飛躍

2019-12-20 1,635條評論 4,159次閱讀 260人點贊

拉瓦內利接受《羅馬體育報》採訪：

關於意甲聯賽

「尤文是讓我成長的球隊，我是尤文蒂諾。我在拉齊奧贏得過聯賽冠軍、義大利杯和義大利超級盃，她也在我的心中。國際米蘭有安東尼奧·孔蒂，我對他抱有令人難以置信的尊重。我要說一切皆有可能，聯賽還很漫長，我正在沉迷於其中」。

拉瓦內利，聯賽冠軍馬拉松重新出現。
「尤文跟國際勢均力敵，拉齊奧只是有點落後。競爭是三足鼎立，將會持續到底。聯賽的前四個月精彩絕倫。國際和拉齊奧的唯一污點，是分別從歐冠和歐聯中出局」。

尤文和國際有所預料，拉齊奧一鳴驚人。

「藍白軍團在中前場令人感到害怕，後場穩固。拉齊奧在組織上，不比誰遜色，多年以來至少能夠爭奪歐冠資格，位於第三名和第四名，不幸的是參賽資格多次溜走，但它總是做到出色的事情。掌聲獻給洛蒂托和塔雷，他們交給了因扎吉一支令人生畏的球隊」。

西莫內·因扎吉正在書寫歷史。

「他正在做到出色的事情。人們總是基於各種時刻準備進行斷言和批評，給出最終評價還為時尚早。需要承認他在去年贏得了義大利杯，正在聯賽中表現得好。遺憾的是從歐聯出局，錯失了一個機會，考慮到組成陣容的球員的實力應該達成出線，它為此做了準備」。

尤文沒有「殺死」聯賽，您預料到了她會如此地陷入重圍嗎？

「尤文從來不會讓我失望，他正在遵守自己的計劃。他在歐冠中晉級，八分之一決賽將面對里昂，我不會說這是一個容易跨越的障礙，但他肯定觸手可及。球隊正在為聯賽冠軍而戰，我確定三月他會在各項賽事中奮戰」。

薩里主義姍姍來遲。

「薩氏尤文正在提高，我要說會是一種逐漸增強。他獲得了球員犧牲自己的意願，相信他的理念。需要給他時間，評判一位教練必須等到賽季結束，好好等待吧」。

尤文、國際和拉齊奧，有何不同？

「國際，在身體和精神上，有所成長，從不放棄。尤文還是通常的尤文，儘管改變了比賽哲學。拉齊奧也學會了相信自己到最後一刻，它在卡利亞里的補時最後四分鐘獲勝，這是幸運的，感謝撒丁球隊，他們多次錯過擴大比分，但球隊堅持到了最後。拉齊奧在過去會錯失機會，它完成了向前的重要一步」。

尤文重新發現了迪巴拉。

「迪巴拉，是的。但整支球隊令人難以置信。我希望道格拉斯·科斯塔儘快回歸。從天賦上來看他是世界最強之一，他會讓尤文在歐冠中也能實現質的飛躍」。

勞塔羅-盧卡庫，孔蒂的雙重武器。

「勞塔羅正在打出一屆超出任何邏輯的聯賽，他正在展現自己是個優秀的冠軍球員，是個拉動者。盧卡庫正在等待他的時刻，他是安東尼奧喜歡的重量級前鋒，取得過重要的進球。誕生了一對配合良好的組合。但要注意……」

注意什麼？

「競爭是三足鼎立，但加圖索的拿波里會重新參與進來，不要忘了羅馬。里諾是我們的最佳教練之一，去年他在AC米蘭做到了夢幻般的工作。他證明了自己跟西莫內一樣能幹和年輕。他們兩人都是蓄勢待發」。
