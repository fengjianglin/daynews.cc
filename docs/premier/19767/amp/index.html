歐冠16強抽籤儀式是史上最具含金量16強對決 英超仍一統天下？

天天要聞
2019-12-16

感謝您的「關注」，您的支持是我們足球咖團隊不懈追求的最大源動力！足球比賽是信仰，更是一種修行。感謝您們一如既往的支持和關注，

歐冠16強抽籤儀式就將進行。這是歐冠歷史上首次由五大聯賽球隊包攬十六強，可以預見的是，每場比賽都有可能是強強對話，這或許將是近些年最激烈的一次十六強戰了。

抽籤規則

根據歐冠16強戰的抽籤原則，同國和同組球隊將迴避。經過仔細分析後，今晚的抽籤或許將產生眾多強強對話。另外，各小組排名第一出線的球隊為種子隊，以小組第二身份出線的球隊為非種子隊，在1/8決賽階段，種子隊將先客後主進行比賽。本賽季歐冠1/8決賽首回合的比賽將在明年2月18、19、25和26日進行，次回合的比賽將在明年3月10、11、17和18日進行。

抽籤看點

1- 史上最強16強戰

由於來自五大聯賽的球隊包攬了本賽季的歐冠16強，這就意味著，1/8決賽中一定會有強強對話產生。這是五大聯賽球隊史上首次包攬歐冠16強席位，1/8決賽的含金量也就可想而知了。球迷們也戲言，或許這就是皇馬主席弗洛倫蒂諾一直希望組建的歐洲超級聯賽吧！極端情況下，皇馬將會面對尤文、巴薩將會面對切爾西、拜仁將會面對馬競、利物浦將會面對多特……

2- 再現英超盛世？

上賽季的歐冠和歐聯杯中，4支來自英超的球隊全部殺入最後的決賽，最終利物浦2-0擊敗熱刺，拿到歐冠冠軍，而切爾西則拿下阿森納，問鼎歐聯杯。本賽季的歐冠小組賽中，利物浦、曼城、切爾西和熱刺這四支來自英超的球隊全都順利小組出線，其中利物浦和曼城還是以小組第一的身份順利出線，如果一切順利的話，英超四隊全部闖入歐冠8強的極端情況也不是不可能發生。今年的歐冠賽場上，還能再現英超盛世嗎？

3- 誰能抽到大禮包？

雖然五大聯賽的球隊包攬了本賽季的歐冠16強，但顯而易見的是，瓦倫西亞、亞特蘭大、萊比錫和里昂的實力相對較弱。傳統豪門球隊肯定都希望在1/8決賽中碰上一支「魚腩」球隊，儘可能以最小的代價收穫勝利，闖入歐冠8強。

4- 誰能成為最大黑馬？

顯然，歐冠16強中的最大黑馬當屬亞特蘭大，在小組賽前三輪全部輸球的情況下，亞特蘭大奇蹟般的最後時刻拿到小組第二，殺入歐冠淘汰賽。如果能抽到實力相對較弱的萊比錫或是瓦倫西亞，並延續自己的神奇表現，亞特蘭大闖入歐冠8強也不是沒有可能的事情。

另外，隊史首次闖入歐冠淘汰賽的RB萊比錫也不容小覷，目前萊比錫以33個積分排名德甲榜首，如果能和近況不佳的那不勒斯抽到一起，萊比錫並非沒有一戰的可能。

5-巴薩切爾西冤家再聚首？

根據抽籤規則，巴薩16強的潛在對手有熱刺/亞特蘭大/那不勒斯/里昂/切爾西，毫無疑問，巴薩大戰切爾西是球迷們最想看到的。兩支球隊是歐冠場上的冤家，尤其是2009年在小白在斯坦福橋絕殺一役最為經典。2012年，托雷斯則在諾坎普長途奔襲完成絕殺同樣讓球迷印象深刻。根據國外專家的計算，巴薩最可能的對手就是切爾西，概率達到了23.377%！

6-C羅重回伯納烏？皇馬有望遭遇尤文

自C羅轉會後，球迷們一直爭論不下。這筆轉會到底是皇馬虧了，還是C羅虧了？有球迷說皇馬失去葡萄牙人後已經不再是歐洲頂級豪強，還有人說C羅少了皇馬眾星喂餅，踢得越來越掙扎。因此如果皇馬和尤文相遇，也將是看點多多。C羅重回伯納烏，他會受到怎樣的對待呢？

7-利物浦復仇？紅軍或再碰皇馬

對於利物浦球迷來說，2018年的歐冠決賽無疑非常遺憾。薩拉赫的受傷以及卡利烏斯的黃油手，讓利物浦把冠軍拱手相讓，紅軍也不得不也見證皇馬連續三年登頂歐冠。如今的利物浦已經成為歐洲風頭最盛的球隊，毫無疑問，他們也希望再一次與皇馬正面較量，證明自己並不輸於對手。

8-每年一度的「弱弱對決」？萊比錫碰面亞特蘭大？

經常觀看歐冠聯賽的球迷們會發現，幾乎每年小組賽都有一個實力平均的「魚腩」小組，而淘汰賽也總會有一對「弱弱相遇」。縱觀今年的16強球隊，萊比錫、亞特蘭大和里昂無疑是相對較弱的一方。萊比錫的潛在對手包括：皇馬/熱刺/亞特蘭大/馬競/那不勒斯/切爾西，而亞特蘭大的潛在對手包括巴黎/拜仁/利物浦/巴薩/萊比錫/瓦倫。因此可以看出，這兩支球隊是非常有希望在淘汰賽相遇的。

9-曼城再抽好籤？

仔細回顧過去幾個賽季的抽籤後，不難發現，曼城在抽籤方面一直運氣不錯，這也讓其他球隊的球迷們十分嫉妒。上賽季他們抽到了幾乎最弱的沙爾克04，再往前一個賽季則是抽到了「大禮包」巴塞爾。2016-17賽季，曼城遭遇摩納哥，再往前一個賽季則是基輔。此次抽籤曼城的潛在對手是皇馬/馬競/那不勒斯/多特/里昂，他們能如願抽到實力最弱的里昂嗎？讓我們拭目以待！

10-巴黎繼續8強噩夢？

巴黎球迷經常自嘲：每年辛辛苦苦拿到小組第一，抽籤總抽「炸藥包」。確實如此，大巴黎實力強勁，但是他們總是在淘汰賽「硬」不起來，經常首輪就被淘汰。上賽季被曼聯意外淘汰，之前一個賽季則是被皇馬擊敗。16-17賽季則是遭巴薩大逆轉。
