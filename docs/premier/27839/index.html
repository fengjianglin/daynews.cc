AC米蘭在意甲最黑暗的三場比賽都踢成啥樣？今天一一道來

2019-12-23 216條評論 3,540次閱讀 2485人點贊

⚽️AC米蘭隊史上經歷過數次低谷，最黯淡的時光莫過於貝盧斯科尼接手之前的80年代初期，他們曾經遭遇過兩次降級。不過說起米蘭隊史在意甲聯賽遭遇的慘敗，說來也就只有這3場比賽。

⚽️【No.1，1997年4月6日】當時身為聯賽衛冕冠軍的AC米蘭在經歷了隊史最輝煌的一段王朝時期後，出現一次十分明顯的低谷，卡佩羅親手建立的王朝徹底覆滅。這個賽季他們在主場1-6慘敗尤文圖斯，不僅讓尤文搶走了聯賽冠軍，更是留下了隊史在意甲聯賽最慘痛的失利，這場比賽也是兩隊意甲交鋒史上比分最為懸殊的一場比賽。當時AC米蘭名將雲集，參加這場比賽的就有馬爾蒂尼、博班、維阿、巴雷西等著名球星。

⚽️當時還屬於U23的小將維埃里獨中兩元，尤文圖斯破門的還有尤戈維奇、齊達內和尼古拉-阿莫魯索。馬科-西蒙尼為AC米蘭打進了留有顏面的唯一進球。這場比賽結局極其殘酷，維埃里一戰封神，並且在賽季結束後得到了西甲馬德里競技的高薪合同，而米蘭時任隊長巴雷西則就此隱退江湖，黯淡離開了意甲舞台。米蘭的第一個超級王朝時代就此結束。

⚽️【No.2，1998年5月3日】1-6慘敗僅僅過去一年，當時的AC米蘭依然沒有走出低谷，這個賽季他們的狀態依然糟糕，聯賽收官階段他們組客場0-5慘敗羅馬隊。當時大打攻勢足球的羅馬隊上半場就送給AC米蘭一個創紀錄的4-0，坎德拉、迪比亞喬、塞爾吉奧、德爾維奇奧先後進球。榮升隊長的馬爾蒂尼則再次經歷慘敗。1-6輸給尤文的陰影還未散去，他就又一次無奈看著球隊遭虐。

⚽️這場比賽小將托蒂雖然沒有進球，但是卻完成了多次精彩的突破和助攻，讓米蘭吃盡苦頭。客場0-5的慘敗也是當時米蘭隊史上在意甲聯賽遭遇的最大失利。連續兩個賽季創下主場和客場的恥辱紀錄，當時的米蘭陷入了深深的低谷。不過好在他們在下一年的意甲聯賽結束了這一噩夢，扎切羅尼和比埃霍夫的到來讓這段陰暗短暫的歷史得以改寫。不過這兩場慘敗他們輸給尤文和羅馬兩支傳統強隊，也不算是太過於丟人的黑暗史。

⚽️【No.3，2019年12月22日】2019年的聖誕節即將到來，已經8年沒有拿過意甲冠軍的AC米蘭，在歐冠16強球隊亞特蘭大的主場藍色競技球場再度遭遇慘敗。這場比賽的上半場，米蘭「僅僅」以0-1落後，但是比賽進入到下半場，AC米蘭遭遇到他們在意甲聯賽最灰暗的一個下半場比賽。下半時他們在22分鐘短暫時間裡4次被亞特蘭大攻破球門，創下了隊史又一奇恥大辱。

⚽️雖然客場0-5的比分不是隊史最慘，只是與21年前在羅馬奧林匹克球場的慘敗持平。但實際情況，目前的AC米蘭處境更加糟糕，沒有實力超群的核心球員，這一次慘敗難以重新當年第二年就奪冠的神奇一幕。也就是說現在的米蘭真的無力翻身。亞特蘭大也無愧於本賽季歐冠黑馬的本色，5球打爆米蘭讓眾多米蘭粉絲痛罵和飆淚。

⚽️隊史3次慘敗，這一次的打擊無疑最大。AC米蘭復興變成空談，同時救火主帥皮奧利也不排除有再次下課的危機。如此看來上賽季屈居聯賽第5，加圖索被解僱真是千古奇冤。這也難怪賽後有球迷惡搞，引用范志毅痛批國足的經典畫面，改為米蘭名宿舍瓦的痛罵，以此來嘲諷如今再次深陷低谷的AC米蘭。不過話說回來，意甲第4大勝意甲第9，這結局有毛病嗎？一個連歐聯杯都沒踢上，一個則闖進了歐冠16強。
