足壇未來10年將叱吒風雲的小將，除了姆巴佩還有他們

2019-11-10 0條評論 18次閱讀 0人點贊

已經捧起大力神杯的姆巴佩，無疑是這個年齡段球員中的佼佼者。除了他，還有一切不容忽視的新生代在強勢崛起。

1，若昂·菲利克斯

1999年出生的菲利克斯比姆巴佩還小一歲。1.26億歐元的身價和在馬競的迅速融入，讓人對他的未來充滿期待。同為葡萄牙球員，他也有了C羅接班人的稱號。

2，維尼休斯

來自巴西的維尼休斯年齡更小，2000年7月出生。是個從貧民窟走出的足球天才。

羅德里戈·席爾瓦·德·戈埃斯

又是一名來自巴西的球員，又是效力於皇馬。本月6號皇馬對陣加拉塔薩雷的歐冠比賽，羅德里戈上演帽子戲法並為本澤馬助攻。成為了勞爾之後成為歐冠歷史上第二年輕上演帽子戲法的球員。羅德里戈出生於2001年1月，真的是英雄出少年！

4，蒂莫·韋爾納

蒂莫·韋爾納（Timo Werner），1996年3月6日出生於德國斯圖加特。現在是RB萊比錫的當家前鋒。韋爾納從U15開始，代表各級別國青隊42場比賽攻入33球。韋爾納腳下技術出色，有突破能力，身高1.81米的他還有著百米11秒1的驚人速度。拜仁和利物浦等豪門已經盯上了他，如果能加入頂級俱樂部，韋爾納也將有機會綻放更大的光彩。

5，扎尼奧洛

義大利終於又出了一個青年才俊。1999年出生的扎尼奧洛，司職中場。現在在羅馬隊已是鐵打的主力，也入選了義大利國家隊。現在意甲的關注度比較低，扎尼奧洛在中國球迷中不那麼聲名顯赫。看過他比賽的會有深刻印象，傳球很有想像力。 義大利從來不缺好門將，好後衛，鋒線也常有頂級球員出現。但是中場大師，這些年除了皮爾洛就沒有別的球員達到頂級的高度。希望扎尼奧洛能成長起來！

6，弗倫基·德容

今年剛加盟巴薩的荷蘭中場德容（1997.5.12出生），已經證明了自己的實力。自身的技術風格十分適合巴薩，曼城這種技術流球隊的戰術打法。相比尤文圖斯高價入手的德利赫特，兩名荷蘭小將的表現反差可是夠大。估計尤文現在已經很鬱悶引援走眼……德容也已成為荷蘭國腳，被歐足聯評為歐洲國家聯賽最佳青年球員併入選歐洲國家聯賽決賽階段最佳陣容。他的對於荷蘭足球和巴薩，都是未來的希望。

7，丹尼爾·詹姆斯

詹姆斯1997年11月10日出生於威爾士。加入到了曼聯以來他的表現就讓人感到非常的滿意。如今他已經為曼聯上陣了十幾場英超的賽事，已經射進了三個球。威爾士球員中最大名鼎鼎的吉格斯和貝爾都是速度奇快，加上如今的詹姆斯，這還真是一個出「球場跑車」的國家。曼聯1500萬英鎊簽下詹姆斯，物超所值都不能形容，真的是撿漏了。相信成熟之後詹姆斯將給曼聯帶來更大幫助。

8，莫伊塞·基恩

莫伊塞·基恩（Moise Kean），2000年2月28日出生於義大利韋爾切利。出身於尤文圖斯青訓。司職前鋒。有象牙海岸血統的小基恩身體條件出色，在尤文時期表現也足夠亮眼。今年夏天從尤文圖斯轉會到了埃弗頓，尤文圖斯的這個操作讓本隊球迷非常不解，不知道為什麼要賣掉這個潛力股。本人比較喜愛義大利足球的原因吧，對義大利的球員關注比較多。目前論名氣，小基恩和菲利克斯，德容這些天價球員差很多。尤文大牌雲集，小基恩機會不多。2018-19賽季，莫伊塞·基恩代表尤文圖斯在意甲出場13次，進6球。看看替補上場就能進球的小基恩，我們的武球王效率真得提高啊……

9，奧斯曼·登貝萊

1997年出生的登貝萊，論名頭，論身價，這位億元先生甩開詹姆斯，小基恩幾條街。他的個人突破能力極強速度極快，他一個人往往就可以把對手的中後場兩條線推後10-20米。然而轉會巴薩二年多以來已經6次受傷。真是詭異，來巴薩之前，登貝萊在雷恩和多特蒙德一共就因傷修養過10天。不知道這位法國邊鋒什麼時候能站出來，讓巴爾韋德的用兵能有更多選擇。只要別真的變成玻璃人，他的未來還是值得期待。畢竟他還如此年輕。

10，傑登·桑喬

2000年3月25日出生於英格蘭。場上位置是邊鋒。英格蘭人少有出去闖蕩的勇氣，而桑喬17歲就來到了德國，並且征服了大黃蜂球迷。桑喬能在狹小的區域完成過人，帶球速度也非常快。是英格蘭少有的秋風華麗的球員。18/19賽季，桑喬入選了德甲的最佳陣容，打進12球，並送出聯賽最高的18次助攻！現在的桑喬，已是各大俱樂部最為垂涎的天才少年。

梅羅絕代雙驕的時代不知還能持續多久，球員必然要迎來更新換代。除了以上這些，還有哪些小將是未來成就不可限量的呢？歡迎大家補充，交流。
