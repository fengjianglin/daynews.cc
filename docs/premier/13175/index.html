米蘭城沒有永遠的勝利者，相互對望

2019-12-12 50條評論 1,026次閱讀 54人點贊

米蘭（Milan），古羅馬時期被稱為米迪歐蘭尼恩（Mediolanium），是義大利第二大城市，世界時尚與設計之都和時尚界最有影響力的城市，世界歷史文化名城，是義大利人口最密集與工業最發達的地區。歐洲著名足球隊AC米蘭足球俱樂部、國際米蘭足球俱樂部歷經了百年風風雨雨，發展成為世界上頂尖的足球豪門，米蘭德比也是世界上最具底蘊和最精彩的同城德比。

AC米蘭：成立於1899年，傳統隊服是紅黑相間的間條衫，聖西羅球場，82955人，義大利足球甲級聯賽冠軍 18次， 義大利杯冠軍 5次， 義大利超級盃冠軍 7次， 歐冠冠軍 7次， 歐洲超級盃冠軍 5次， 歐洲優勝者杯冠軍 2次 ，洲際杯/豐田杯/世俱杯 4次 。

國際米蘭：成立於1908年3月9日，傳統隊服是藍黑相間的間條衫，梅阿查體育場，80018人，義大利足球甲級聯賽冠軍 18次，義大利足協杯冠軍7次 ，歐冠冠軍3次 ，歐洲聯盟杯冠軍3次 ，洲際杯/世俱杯冠軍3次 ，義大利超級盃冠軍5次。

AC:隊史上著名球星有馬爾蒂尼，舍甫琴科，卡卡、皮爾洛、里維拉、阿爾塔菲尼、內斯塔、范巴斯滕、科斯塔庫塔、巴雷西，古力特等

國米：隊史上著名球星有羅納爾多，薩內蒂，維埃里，馬特烏斯，克林斯曼，法切蒂，梅阿扎，貝爾戈米，托爾多等

百年恩怨的同城死敵，給我們帶來過無數的經典對戰，眾多球迷的意甲情懷大多數產生在這兩傢俱樂部之中，或許喜歡AC米蘭，或許深愛國際米蘭。但這些都沒有所謂，沒有人能看到他們戰爭的終結，他們也沒有永遠的勝利者，在米蘭城裡，相互對望，也希望明年兩個隊向歐洲冠軍發起再次衝擊，距離上一次真的太久了，喚起老球迷的米蘭熱情。
