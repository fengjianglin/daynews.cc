帕爾馬主帥：認同加圖索說要保持攻守平衡；庫盧太厲害了

天天要聞
2019-12-15

虎撲12月15日訊 北京時間今天凌晨結束的意甲聯賽第16輪的一場較量中，達維爾薩帶領帕爾馬客場2-1戰勝那不勒斯，賽後達維爾薩也在採訪中談到了加圖索說帕爾馬進攻轉化率高的話題。

達維爾薩說道：「傳射建功？這就是我們為什麼不願意在冬窗出售庫盧塞夫斯基的原因。儘管機會不多，但是我們仍創造出了3次極具威脅的破門機會，因此我認為這場勝利是我們贏得的。」

「我們不是不會控球，但是當你的球隊擁有庫盧塞夫斯基、熱爾維尼奧這樣的球員時，當你的對手是強大的那不勒斯時，你必須以防守反擊起步，但我們沒有在全場比賽都堅持防守反擊，而是在關鍵時刻加快了比賽節奏，並取得了效果。」

「我們不認為這是本賽季的轉折點，對陣布雷西亞的保級大戰尤為重要，我很認同加圖索的看法，我們必須在比賽中保持攻守平衡，保持謙卑的態度，並在訓練場加倍努力。」

「加圖索說我們進攻轉化率高？因為我們擁有兩個強力邊鋒，要想在幾乎沒有輪換的情況下保持體能是很困難的，但是我們做到了這一點。我告訴斯普羅卡蒂不要做太多奔跑，他的任務更多是在策動，而熱爾維尼奧和庫盧塞夫斯基真的是厲害。」

「上半場我們踢得很好，下半場隨著體能的下降，我們的表現也有所下滑，但是熱爾維尼奧多次造成威脅，4次形成單刀，這一點也是我們在訓練中強調的，我們的關注點不該在控球率。」

「要說控球率，在對陣卡利亞里的比賽中，我們控球率就很高，但是當你在場上沒有合格中鋒的時候（科內柳斯比賽第17分鐘傷退），一味地追求控球率只能是自取滅亡。因為這樣做只是幫助對手消磨了自己的時間，並且很難造成真正的威脅。」

「帶領帕爾馬從丙級到甲級到衝擊歐聯杯？現在說這個還太早了。上賽季我們在前半程也發揮得很好，但是很可惜下半程遭遇了傷病潮侵襲。這賽季，我們上半程就遭遇了很多傷病，但我們有了一個更完整的陣容。」

(編輯：姚凡)
