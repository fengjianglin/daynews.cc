馬爾蒂尼—內斯塔—斯塔姆—卡福真是世界最強後防嗎？我看未必

天天要聞
2019-12-03

​

在歷史當中有許多傳奇不斷為人稱頌，然而很多傳奇在經過深度剖析之後，大家會發現其實也並沒有那麼神奇，而在足球的歷史上也有很多類似的傳奇，今天戰鬼就要和大家剖析一下傳說當中歷史上最強的後防組合——馬爾蒂尼、內斯塔、斯塔姆、卡福。

四個神級後衛的史上最強後防

後防天團4大主力一齊首發

儘管米蘭在0405歐冠決賽當中在伊斯坦布爾慘遭逆轉，但那場比賽當中米蘭排出的後防線一直為球迷所稱道，卡福、內斯塔、斯塔姆、馬爾蒂尼，無論名氣還是個人實力，可以說都是歷史上絕無僅有的後防組合，這四人當中任意一個都是本位置上最頂級的球員，這樣人員配置可以說在世界足壇的歷史上都是空前絕後的。然而戰鬼在這裡卻要說一個殘酷的事實，這個史上最強防線其實一共也沒用過幾次。

唯一能打敗這條後防線的是年齡

經典遊戲實況8里米蘭後防天團個個超神

這條防線第一次在正式比賽當中聯袂登場是2004年9月14號米蘭在歐冠小組賽客場挑戰頓涅茨克礦工隊的比賽，在那個時間節點，馬爾蒂尼36周歲，卡福34周歲，斯塔姆32周歲，內斯塔28周歲。而且，最要命的是對體能和速度要求最高的兩個邊後衛是後防四人中年級最大的兩個。而且米蘭當時主打4312陣型，球隊沒有專職的邊前衛球員，在進攻的時候需要邊後衛前插助攻拉開進攻寬度，這就要求兩個邊後衛要較多的前插，顯然這樣的要求對兩位老將，尤其是36歲的馬爾蒂尼來說實在是太過苛刻。這樣的年齡結構註定了這個組合不可能被太多使用，要最大限度的保障老將們的體能。

輪換得當，體能充沛，後防天團無所不能

36歲的馬爾蒂尼不但是球隊後防核心更是精神領袖

時任米蘭主帥的安切洛蒂顯然知道這條防線的問題所在，因此他把4個人拆開使用，絕大多數比賽當中4人里只有3人在場上，內斯塔、馬爾蒂尼、斯塔姆在中衛位置上進行輪換，左路卡拉澤、塞爾吉尼奧和潘卡羅三人輪換，斯塔姆在一些比賽當中會踢到右後衛位置和卡福輪換，這樣的安排保障了三位老將的體能，也保護了容易受傷的內斯塔。而在關鍵的歐冠淘汰賽比賽當中，四人一齊上陣，在歐冠八分之一決賽和歐冠4分之一決賽當中，米蘭4場比賽全部零封了曼聯和國米，四人天團首發3場，體現了後防天團的超強實力。截止到2005年4月20日，米蘭後防天團聯袂首發9場，球隊8勝1負，完成了7場零封，僅失2球，其中被米蘭零封的球隊當中包括國米、曼聯、巴薩這樣的頂尖豪門，唯一一場輸球是聯賽0:1負於利諾沃。截止到此時此刻，米蘭後防天團的表現堪稱完美。

黑色五月，後防天團土崩瓦解

一個13一個31，先後從拉齊奧加盟米蘭的世界級中衛

然而到了聯賽和歐冠衝刺階段的5月，風雲突變。米蘭整體狀態突然下滑，整個5月份在各項賽事當中僅僅取得1場比賽的勝利。米蘭的後防天團也沒能阻擋球隊下滑的頹勢。首先是在歐冠半決賽第二回合當中，米蘭1：3客場敗於埃因霍溫，依靠安布羅西尼的客場進球，球隊勉強進軍歐冠決賽。隨後在聯賽天王山之戰當中，米蘭主場0：1負於尤文圖斯，失去了爭冠的先機。最後是所有米蘭球迷都不願回首的伊斯坦布爾之夜，米蘭3:0領先最終被利物浦逼平，點球決戰負於對手。在這最關鍵的三場戰役當中，米蘭的後防天團4人全部首發，但也沒能挽救米蘭，3場比賽當中球隊丟了7球，其中伊斯坦布爾的後45分鐘，就丟了3球，這一數字超過了之前8場之和，後防天團被徹底擊潰。

後防天團從此絕跡江湖

03-04賽季卡福自由身加盟米蘭

04-05賽季結束之後，天團4位成員的年齡又大1歲，身體機能進一步下滑。安切洛蒂再沒有在任何一場比賽當中安排4人同時首發，後防天團從此絕跡江湖。四人同時在隊兩年，實際上一共只同時首發了12場比賽。嚴格意義上說，這根本達不到一套首發後防線的出場標準。在這12場比賽當中，米蘭8勝1平（歐冠決賽負於點球決戰）3負，丟9球。從數據上看這個防線的質量仍然值得肯定，但距離「世界第一後防天團」這樣的稱號仍然顯得名不符實。

由此我們不禁希望，現實世界如果能像遊戲一樣，球員不會衰老，或者衰老的沒有那麼迅速明顯，再或者後防天團早兩年相會，這條防線該是怎麼樣的恐怖？正是這樣浪漫的想法，讓米蘭的後防天團留下了歷史上最強的後防組合的傳說。
