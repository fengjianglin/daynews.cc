強大中超！中超最強中衛盛讚中超：曾低估中超水平，實際上水平高

江蘇蘇寧本賽季成為了中超big4球隊，在最後一輪反超山東魯能成為中超第四，而2018江蘇蘇寧是中超第五名，成績前進一名。蘇寧主帥已經表示希望2020賽季成績更進一步，那就是中超聯賽第三名的目標。從2019賽季最後13輪的比賽來看，江蘇蘇寧才是拿分最多的球隊，13輪居然取得9勝4平的不敗成績，拿分大大超越上海上港，北京國安和廣州恆大。其中外援中後衛米蘭達也做了很多貢獻，近日米蘭達接受中國媒體採訪，盛讚了中超聯賽，認為自己曾經低估了中超水平，中超水平實際上很高。又有一個國際水平的中後衛盛讚中超，無疑是中國球迷的驕傲。

米蘭達本賽季夏季轉會期才加盟江蘇蘇寧，他本是國際米蘭中後衛，效力國際米蘭3個賽季全是主力中後衛。在效力國米之前，米蘭達是馬德里競技連續五個賽季的主力中後衛。本賽季在中超出場9次，場均貢獻1個攔截1個封堵3.8個解圍。

近日記者採訪了米蘭達，問到半個賽季的感受，米蘭達表示說實話，一開始我低估了中超的水平，以為在這裡不會遇到太多壓力和挑戰，實際情況比我想像得困難許多。中超有很多高水平球隊，還有不少世界頂級的外援。現在我要說，中超是水平很高的聯賽，我很開心在這裡踢球，因為我喜歡高水平的對抗，每次跟強者對話都會有無比興奮的感覺。問哪一場比賽印象最深，米蘭達選擇蘇寧主場3-1擊敗廣州恆大。問中超哪個前鋒最難防守，米蘭達沒有給出具體球員，而是表示面對任何一個前鋒，思想上都不能覺得容易對付，因為他們的目標就是把你甩在身後。

米蘭達肯定了中超聯賽是一個高水平聯賽，確實如此，亞足聯最新一次更新國家聯賽積分榜單。中超排名第一，卡達聯賽排名第二，日本聯賽排名第三，沙特聯賽排名第四，韓國聯賽排名第五。而且亞洲前十名總身價最高的俱樂部，中超俱樂部佔據7家，他們是恆大上港國安，大連申花蘇寧魯能。無疑米蘭達的話是真實的感受，再次為中超的強大感到驕傲。
