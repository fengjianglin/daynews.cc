格子離隊影響太大，馬競冬窗打算引進皮亞特克和貝洛蒂增強攻擊力

2019-12-03 0條評論 17次閱讀 0人點贊

貝洛蒂和皮亞特克是意甲的優秀前鋒

馬德里競技本賽季在西甲聯賽中戰績不佳，最主要的原因就是攻擊力下降。西班牙媒體報道，他們有意在冬季轉會窗口引進AC米蘭前鋒皮亞特克和都靈射手貝洛蒂。

法國球星格列茲曼在這個夏天轉會巴塞羅那，馬競雖然引進了菲利克斯，但他們的攻擊力還是出現了下滑，最近對陣尤文圖斯和巴薩的兩場比賽，他們都沒有破門得分。而在西甲15輪比賽過後，他們僅僅排名聯賽第6。

皮亞特克本賽季在米蘭的表現令人失望

由於迭戈-科斯塔本賽季深受傷病的困擾，西蒙尼的球隊目前只有莫拉塔一名正印中鋒勉力支撐。

根據《阿斯報》的消息，馬競正在轉會市場上尋找一名前鋒，並可能從AC米蘭租借引進波蘭射手皮亞特克。與此同時，該西班牙報紙還透露，都靈前鋒貝洛蒂也是一個選擇，馬競願意為這名義大利國腳開出3500萬歐元的轉會費。

瑞典老將伊布有可能在這個冬季回到紅黑軍團，因此米蘭可能選擇將皮亞特克租借出去。本賽季至今，這名波蘭射手在意甲出場14次，僅僅打進3球，而且2個為點球破門。

貝洛蒂本賽季在意甲已經攻入7球

另一方面，貝洛蒂至今出場13次打進7球，但都靈是否會在賽季中途放走他們的當家球星目前還有待觀察。

此外，報道指出馬競也在考慮引進巴黎聖日耳曼中鋒卡瓦尼和瓦倫西亞前鋒羅德里戈。 (編輯：天藍色男孩)
