意超級盃-迪巴拉破門難救主 拉齊奧3-1尤文奪冠

2019-12-23 3,679條評論 4,837次閱讀 3503人點贊

央視網消息：北京時間12月23日，第32屆義大利超級盃在利雅得展開角逐，尤文圖斯1比3負於拉齊奧。阿爾韋托首開紀錄，C羅造成門將脫手，迪巴拉追平比分。下半時，盧利奇與卡達迪破門，本坦庫爾被罰下。拉齊奧第5次奪冠，追平國際米蘭，僅次於AC米蘭（7冠）和尤文（8冠）。

意超級盃-迪巴拉破門 拉齊奧3-1尤文奪冠

開場8分鐘，因莫比萊左路橫傳，阿爾韋托禁區弧頂抽射掠過橫樑，他被馬圖伊迪踹中右腳踝接受治療，但他隨後為藍鷹取得領先！第16分鐘，阿爾韋托分球，盧利奇禁區左側下底扣球騙過德希利奧後傳中，阿萊士-桑德羅後點冒頂，米林科維奇-薩維奇回做，阿爾韋托在門前10米處推進右上角，1比0。

阿爾韋托又接因莫比萊做球，在門前11米處推射被斯琴斯尼得到。皮亞尼奇中場長傳，伊瓜因禁區內卸球回做，C羅閃出角度距門12米處抽射偏出右門柱。隨後C羅左路送球給前插的迪巴拉，迪巴拉在門前11米處左腳斜射高出。

C羅與迪巴拉踢牆配合，禁區前沿被米林科維奇-薩維奇放倒，迪巴拉在門前20米處左腳自由球稍稍偏出左門柱。尤文後防沒能將球破壞，科雷亞禁區右側斜射被斯琴斯尼封出。迪巴拉前場搶斷突入禁區做球，伊瓜因在門前13米處左腳低射偏出。阿爾韋托右側底線挑傳，盧利奇小禁區前沿頂高。

尤文第45分鐘追平！馬圖伊迪分球，C羅左側禁區前沿左腳大力低射被斯特拉科沙單手封出，迪巴拉小禁區邊緣推進空門，1比1。他在義大利超級盃4個進球全部是攻破拉齊奧球門，從而超過皮耶羅、埃托奧、舍甫琴科、特維斯（3球），成為義大利超級盃射手王。

下半時，阿爾韋托左路斜傳，科雷亞前點頭槌一蹭被斯琴斯尼得到。科雷亞右路傳中，盧利奇後點距門12米處射門送還斯琴斯尼。

誇德拉多換下德希利奧；卡達迪換下盧卡斯；拉姆塞換下伊瓜因；帕羅洛換下阿爾韋托。拉姆塞左路回傳，C羅在門前21米處抽射稍稍高出左門角。拉齊奧第73分鐘再度超出！科雷亞右路塞球，拉扎里扣球左腳傳中，帕羅洛前點頭槌一蹭，盧利奇後點距門6米處推入近角，2比1。

道格拉斯-科斯塔換下馬圖伊迪；凱塞多換下因莫比萊，他在後場左路送出直傳，科雷亞單刀突入禁區推進推進右下角，但被判越位。

阿萊士-桑德羅左路傳中，迪巴拉前點距門8米處頭槌被斯特拉科沙得到。薩里在場邊抗議吃到黃牌。尤文補時第3分鐘又連遭打擊，迪巴拉自由球被擋，拉齊奧快速反擊，凱塞多直傳，科雷亞距門12米處推射被斯琴斯尼出擊封出，帕羅洛跟進補射被本坦庫爾鏟倒，本坦庫爾吃到第2張黃牌被罰下！卡達迪禁區弧頂主罰自由球擊中橫樑下沿彈進，3比1。
