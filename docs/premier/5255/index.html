萊切斯特城vs埃弗頓:勝，平

主隊近期走勢連戰連捷，近10戰僅客場1比2不敵聯賽榜首利物浦，其餘9場比賽全勝，進32球僅失5球，其中6場比賽零封對手，攻強守穩的特點一覽無餘，聯賽目前已5連勝（僅僅有一場沒能零封對手）聯賽主場5勝1平。而對手埃弗頓目前排名離降級區不遠，且客場戰績乏善可陳，其中僅有的一場勝利也是來自於本賽季魚腩球隊南安普頓，特別需要指出的是上一輪主場面對聯賽副班長諾維奇隊，居然0比2失利。可是啊可是，如此強弱對比明顯的一場比賽卻沒有得到賠率上的支持——主隊居然只讓半一！！！這場有幺，任九3.1

國米vs費拉拉:主勝

國際米蘭這個賽季表現比較強勢，球隊聯賽11勝1平2負打進29球失12球，攻守兩端發揮完美，目前難1分之差落後於尤文圖斯排名意甲次席。周中在歐冠賽場，國際米蘭在作客3-1擊敗了布拉格斯拉維亞，雙線戰績飄紅。目前球隊射手盧卡庫在14輪聯賽射進了10球，整體狀態比較出色。費拉拉是聯賽的弱旅，2勝3平8負僅排名聯賽第19位，其中在作客方面1平5負僅取得1分，而且費拉拉作客還沒有取得進球。國米最近4戰費拉拉取得3勝1平保持不敗，其中主場2戰均取得2-0的勝果，本場比賽開出1.75指數，指數深盤本場勝負懸念不大，費拉拉作客還沒有取得進球，進攻乏力，本場可走正路國米大勝。

曼聯vs維拉:勝，平

本場競賽，曼聯中場仍然缺兵少將，博格巴和麥克托米奈因為傷病原因高掛免戰牌，馬蒂奇剛剛開始康復練習，並且正在鬧轉會的他出戰也成疑，經過歐聯的競賽，想派小將迦納救場也好像不太實際。現在真實的問題在於右路區域防衛，雖然有萬比薩卡在，可是現在各支球隊都不打右邊路而是打右肋部，這個方位現在是林德洛夫來防衛，外號「漏夫」的瑞典人這個賽季被各支球隊針對後，對立才能和爭頂才能上的缺點被無限擴大，而右邊後衛萬比薩卡常常過來協防補位也沒為林德洛夫少背鍋。至於右邊後腰方位上，上輪讓佩雷拉來打證明是徹徹底底的失利。綜合以上分析，曼聯這場需要防平，任九3.1過關
