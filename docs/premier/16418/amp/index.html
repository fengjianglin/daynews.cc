加圖索歸來命運依舊，那不勒斯補時被絕殺。

天天要聞
2019-12-15

意甲第16輪，那不勒斯主場迎戰帕爾馬。補時第2分鐘，熱爾維尼奧快速反擊右路和庫盧2過1後，左腳推射破門。8爺始終沒有繞過命運，首戰失利，開門黑。

耳邊似乎想起了二手的《命運》，歷史驚人的相似。只不過這一次更加凄慘。上一次是米蘭領先被絕平，這一次則是被熱鳥進球帕爾馬逆轉取勝。

其實，那不勒斯踢的不錯，射門比33:7。壓倒性優勢，帕爾馬通過反擊給加圖索的球隊製造困難。一個庫盧，一個熱鳥。讓那不勒斯的後防壓力山大。

盧庫，全名德揚·庫盧塞夫斯基。瑞典人。本賽季被亞特蘭大租借到帕爾馬。本場比賽1球1助攻，開場搶斷庫里巴利單刀破門，間接造成庫里巴利受傷下場。為帕爾馬立下頭功。補時妙傳熱鳥推射破門，更是居功至偉。

目前尤文、國米都對這名中場虎視眈眈。其實我米蘭也可以引進此人。就像以前國米搶米蘭的人一樣，現在可以倒過來搶國米的人。我覺得不會錯。目前庫盧16輪4球6助攻。非常亮眼的數據，19歲當紅炸子雞。

另外一個英雄就是熱爾維尼奧了，從阿森納-羅馬-華夏幸福-帕爾馬，速度沒有下降。依然是90以上，跑起來威脅極大。射門效率也高。

16輪4球3助攻。本場比賽4次射門，3次射正，1個進球。還有4次過人，射中門柱一次。

好風騷的過人，好威脅的射門。

射中立柱。

熱鳥絕殺球，拉都沒拉住。

帕爾馬就是反擊，兩把槍，結果那不勒斯兩個窟窿。8爺的命運也是夠慘的，後衛回追拉不住一個鏟球，就是下去了也值得。也許這是實況玩法。但是我覺得為了不丟球，值得這麼做。

庫里巴利，太背了。8爺賽後把失利的責任歸結於自己。沒有指責沒有抱怨，很8爺。我是有點不太適應8爺在意甲除米蘭以外的球隊執教，還沒有適應。

有點心疼他，很多名宿看好他。我從心底也是這麼想的。球員時代，加圖索就是最聰明的人，對球場局勢判斷正確。他那種不拋棄不放棄的心態感染了每個人，當教練也如此。有點李雲龍的那個意思。那不勒斯目前需要解決的就是士氣問題，8爺樂於此道。這也是為什麼找加圖索的原因吧。

不用過多表揚，也不用過多評價他的執教。士氣特別重要。嗷嗷叫像一群餓狼，誰不害怕。怕不死高位逼搶的一群瘋子，誰不丟球。除非你已經到達一個段位，守的住。要不就是控的好。如利物浦如巴薩。還是祝福吧，我喜歡8爺。

期待那不勒斯和米蘭的比賽，他會是個什麼心情。重回聖西羅，還需要時間。如果義大利杯賽能夠相遇，如果不相遇。下個賽季也許還有變化。誰能知道呢！

其實他和安胖關係最好，你知道吧。
