「專欄」彭雷：巴甲保級大戰賣個面子？

2019-11-12 0條評論 20次閱讀 0人點贊

今天看一場巴甲的保級大戰：

004 博塔弗戈VS阿瓦伊

博塔弗戈10勝3平18負排名聯賽第17位，上輪最後時刻被弗拉門戈絕殺，又趕上其他球隊拿分，博塔弗戈現在掉入降級區，但距離前面也就差一分，這場比賽贏了就能拉下來至少兩隊。

阿瓦伊3勝6平20負，客場2勝1平12負，主客場7連敗，客場5連敗，上輪被桑托斯擊敗後，距離保級區17分之多，球隊軍心已經動搖了，7輪追17分難度可以想像。

本場陣容上，雙方都有不少損失，博塔弗戈主力後衛科斯坦扎和主力進攻中場/前鋒費爾南多是停賽，加上吉爾森和馬塞洛兩名後衛傷停，桑塔納傷愈歸隊是好消息。客隊主力後衛納西門托、洛倫佐都是受傷，前場保利斯塔、弗雷拉等也是受傷，後衛莫瑞斯以及中場弗朗克則是禁賽，主力門將弗拉迪米爾解禁復出。

從指數上看，135，客勝明顯被拋棄了，而雙方主客場平局數據都很少的情況下，3.35是不是有些低呢？我覺得保級大戰，1分對雙方來說都是毫無意義的，這種情況下平局的可能性很小，無論是誰，比賽最後都會努力進攻，都有可能被對手抓漏洞。博塔弗戈從底子上說，還是更強的，就是進攻不太好，最近4場有3場不進球了，但對手是格雷米奧、克魯塞羅（最近狀態最好的保級對手）、弗拉門戈都沒有進球，對桑托斯進了一個，最近4連敗對手確實太強了都。

走主勝正路吧，阿瓦伊賣個人情算了。比分1比0。

昨日競彩貓APP推薦總結

昨天表現整體還不錯，14中10，澳超6連紅完成，不錯不錯，下次有澳超咱們再推。意甲成功實現4連紅，卡利亞里、桑普、拉齊奧都紅單了，尤文圖斯高倍讓平繼續命中，不得不說，米蘭很努力，有機會平局，可惜尤文還是有球星，一錘定音，1比0讓平，尤文圖斯10場勝利9場1球小勝，冠絕歐洲，下次無腦選就行了。

日職聯、畢爾巴鄂逆轉都紅單了，早場意甲+德甲串，卡利亞里完勝佛羅倫薩，我就說嘛，憑什麼受讓那麼多，和亞特蘭大一個道理。門興也輕鬆取勝，2串1紅單。

黃金檔意甲高倍也打出來了，桑普逼平亞特蘭大，亞特蘭大太累了，拉齊奧4比2擊敗萊切，也算正常。

公推的美職聯決賽完美打出，3比1，西雅圖奪冠，能連續公推紅單，很開心哦。錯的是曼城和曼聯，覺得曼聯單關太熱了，不想太容易想多了。曼城推的平局，曼城拚命降低客勝就是有問題，31裡面挑了1，其實如果兩個點球都判了……
