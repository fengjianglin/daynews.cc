破案了！瓜帥為何離開巴薩就拿不到歐冠了

2019-12-14 447條評論 3,294次閱讀 1744人點贊

瓜迪奧拉是小編和許多球迷最喜歡的主教練，從他初出茅廬首次執教巴薩一線隊就像諸葛亮出山一樣，光芒萬丈帶領巴薩一飛衝天，2008年下半年帶隊，第一年2008/2009賽季獲得罕見的六冠王！2009/2010賽季巴薩要不是被國米暗算一把，用盤外招客場遇到種種不合理不利因素，火山石和長時間坐大巴讓巴薩疲憊不堪，出乎意料1：3敗北，主場對國米，穆帥的確是把擺大巴擺到極致，僥倖只輸一個球。最終進決賽。2010/2011年巴薩再沒有犯錯，拿下歐冠！三年二冠，差一點三冠靠的是什麼？還是剛帶巴薩，原因是什麼，現在我們討論一下。

我們喜歡瓜帥，喜歡三年換一個豪門，做主教練能做到他的份上也算達到巔峰，穆里尼奧最狂，在瓜迪奧拉面前也狂不起來，瓜帥在巴薩取得巨大成功，就喜歡換一個環境，第二站到拜仁，聯賽冠軍肯定沒問題，歐冠差強人意，2015年歐冠進決賽的比賽就是輸給他最心愛的巴薩，還是他最得意弟子梅西淘汰，真是老天捉弄人啊。我們最關注的是為什麼拜仁和曼城拿不到歐冠？除了沒有梅西，還最缺什麼？破案了，還最缺乏強大中場！

2009到2011，巴薩最強中場哈白布是地球上最強中場，那是巴薩基石！沒有梅西的西班牙國家隊靠這強大中場拿到2008和2012年歐洲杯，2010年世界盃!這三位分工明確，把傳控玩到極致，皮球在他們腳下幾乎搶不走，這三劍客在中場獨步天下，拿冠無數。拜仁為什麼拿不到歐冠，我們看看瓜帥執教拜仁的中場鐵三角格策，阿隆索，蒂亞戈；比哈白布不止差一點！西班牙拉瑪西亞青訓出來的不光是這三位中場大師，皮克和普約爾，佩德羅，小法，梅西他們對足球理念是一致的，踢得整體足球都達到別人達不到的境界，拜仁三位達不到，就格策合格，像極了小白！那二位差不少！中場比不上巴薩中場，2015年對陣巴薩，怎可能不輸？那就是徒弟碰上師傅，巴薩的傳控是大師級別的！

我們在看看曼城的中場鐵三角：德布勞內，B-席爾瓦，大衛席爾瓦。歐洲三個國家的中場精英組合在一起，也不錯，但是和哈白布比較也是差一個檔次，融為一體的整體性差不少，德布勞內單兵能力的確首屈一指，能力特彆強，但是小團隊配合，比巴薩差一點。阿奎羅也是阿根廷頂級射手，和梅西關係相當好，但在曼城，是和梅西打的相同前鋒位置，也最多算低配版的梅西，肯定比不上梅西啊！中場比不上巴薩中場，前鋒比不上梅西，曼城怎麼能拿歐冠？如果現在曼城不解決糟糕的後衛問題，就和2017的巴薩一樣，後果很嚴重！瓜帥比我們更了解拿歐冠最需要什麼？梅西只有一個，哈白布組合也難找到，所以拿歐冠難！
