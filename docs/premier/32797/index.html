“毛抬廠”結義三兄弟連續掉鏈子，昔日三大豪門集體沉淪令人唏噓

2019-12-24 490條評論 4,280次閱讀 2886人點贊

聖誕假期即將來臨，對於歐洲足壇來說，2019年的比賽基本結束。除了需要加班的英超之外，其他各大聯賽的球星們該休息的休息，該度假的度假，等待來年1月再戰。

只是馬上進入新的一年了，喜歡「毛抬廠」結義三兄弟的球迷們確實開心不起來。對於AC米蘭、曼聯、阿森納的擁躉們來說，看完心愛球隊的前半賽季的表演，肯定都會產生巨大的疑問：這還是我曾經熱愛的那支所向披靡的球隊嗎？昔日輝煌無限的三大豪門，如今集體沉淪令人唏噓。

0比5慘敗的「大哥」

120年前的1899年，也就是大清還沒有亡國的年代，在亞平寧半島的米蘭成立了一支以紅黑色為主調的足球俱樂部，那就是AC米蘭。如果你是羅森內里，那肯定對球隊的輝煌歷史如數家珍。

不用說上古世紀的格諾利組合（格倫、諾達爾、利德霍爾姆），也不必說首任「金童」里維拉，更不用拿出威名赫赫的荷蘭三劍客（古利特、范巴斯滕、里傑卡爾德）。就光本世紀初的黃金時代，就足夠讓米蘭球迷興奮不已，舍甫琴科、因扎吉、魯伊科斯塔、皮爾洛、加圖索、安布羅西尼、內斯塔、西多夫、卡卡……，這些名字依然停留在米蘭球迷的腦海中。

只不過看到如今的米蘭現狀，恐怕會讓退役多年的兩位總監博班和馬爾蒂尼有脫衣上場的衝動，顯然這兩名傳奇名宿在公眾前強顏歡笑，心裡恐怕一直在滴血。且不論這兩名「菜鳥」大管家的運作手段如何，至少他倆還是在危難之際承擔了責任，這種勇氣還是值得肯定的。但是在球隊需要一個長期規化的階段，朝令夕改的幕後金主埃利奧特是否應該主動反省一下？

坦白說，0比5慘敗肯定不是跟亞特蘭大的正常差距。看到小本經營的真藍黑用出色青訓和恰當策略，換來歐冠16強的成績時，最近3個賽季凈轉會投入高達3.27億歐元的米蘭高層作何感想。要知道國米（1.73億歐元）和尤文（1.53億歐元）加起來，還比米蘭少0.01億歐元。可是目前在積分榜上，米蘭已經落後兩大死敵21分，這樣的巨大反差更加凸顯紅黑軍團的尷尬處境。

順帶提一句，過去三個賽季亞特蘭大凈投入-2800萬歐元，剛剛在義大利超級盃力挫尤文奪冠的拉齊奧凈投入-3000萬歐元，2017-18賽季爆冷淘汰巴薩打進歐冠半決賽的羅馬凈投入-7400萬歐元。可是看看這三支球隊的目前意甲排名，分別是亞特蘭大第5（31分）、拉齊奧第3（少賽一場，36分）、羅馬第4（32分）。

劫富濟貧的「二哥」

在結義三兄弟中，曼聯如今的情況最為特殊。擁有20個英格蘭頂級聯賽冠軍獎盃的紅魔，自打2017年後就跟顆粒無收。上賽季陷入困境後，管理層在聖誕賽程開始前就主動求變，讓爭議頗多的「狂人」穆里尼奧下課，繼任者成了球隊傳奇名宿索爾斯克亞。

平心而論，挪威人重返夢劇場之後，還是跟球隊有過短暫的蜜月期。正是曼聯的出色戰績，讓原本對他疑慮重重的管理層改變看法，主動為其轉正並且還延長合同。由於索爾斯克亞獲得了包括恩師弗格森爵士在內的一眾名宿的支持，執教經驗尚淺的他反倒根深蒂固。加上前一個賽季的出色執教成績，讓不少曼聯球迷對索聖充滿期待，只是本賽季情況突然發生變化。

截至第18輪，曼聯目前僅僅取得6勝7平5負，積25分暫列積分榜第8，落後排名第4的切爾西7分。想要時隔一年殺回歐冠，曼聯要麼就要在後半賽季上演彎道超車，要麼就在歐聯杯中一路過關斬將拿到冠軍，可這兩條路選擇哪一條都不好走。

更讓曼聯球迷又愛又恨的是，球隊在對抗強隊和弱隊截然不同表現，讓人的確很難理解。面對積分榜前四的球隊（利物浦、萊斯特、曼城、切爾西），曼聯出人意料取得3勝1平的佳績。算上力克熱刺和戰平阿森納，面對英超賽場公認的6支強隊，紅魔居然取得4勝2平的傲人成績。但是在面臨積分榜後五位時，曼聯卻只取得1勝3平2負。節禮日前遭遇本賽季主場尚未開胡的副班長沃特福德，紅魔不出意外贈送大禮，0比2讓對手如願拿到3分。

對於攻堅能力有限的曼聯來說，始終很難如願打破密集防守，這也正是球隊踢弱隊反而成績不佳的原因所在。不得不承認，索爾斯克亞的固執也讓球隊付出慘痛代價。看看心腹愛將林加德的吐餅能力，恐怕就連索聖本人都要犯嘀咕，同樣都是DNA差距怎麼這麼大呢？

勤儉持家的「三弟」

如果說米蘭的問題是管理層短視，曼聯的問題是人員配置不合理，阿森納的問題恐怕就要歸咎於溫格教授。或許正是法國名帥20多年的帶隊成功，給了老闆克倫克錯覺，這年頭少花錢依然能取得好成績。於是在阿森納常年穩居前四之後，本來也就沒打算巨額投入的老闆，開始有了小富即安的想法。只是當溫格教授抽身離開後，槍手之前被掩蓋的一切問題都浮出水面。

說入主的這些年不投入，也確實有點冤枉克倫克啦，畢竟他也有過砸錢引進扎卡、拉卡澤特、奧巴梅揚等等實力悍將的舉動。只不過球隊每個轉會窗還都神奇地沒花幾個錢，無論是主動甩賣隊長，還是分期付轉會費，只要是凈投入低於4000萬榜，克倫克都欣然接受。難怪槍手死忠們對老闆和管理層的不作為怨聲載道，甚至還有極端球迷還希望克倫克賣掉球隊主動放阿森納一條生路。

由於本賽季成績實在糟糕，管理層只能主動拿下了千挑萬選的溫格接班人埃梅里，嘗試了永貝里過渡之後，任命另一位名宿阿爾特塔成為球隊新任主帥。可是無論是永貝里還是阿爾特塔，此前的頂級聯賽執教經歷全部是0。儘管阿森納官方解釋，球隊是經過深思熟慮後選定新帥人選。只是在波切蒂諾、阿萊格里等知名教頭賦閑在家的情況下，冒險讓菜鳥教練拯救球隊，真的是明智之舉嗎？

或許對比埃弗頓高價聘請安切洛蒂，阿森納這波選帥恐怕還是考慮了經濟因素，要知道阿爾特塔再怎麼工資暴漲也只是410萬歐元左右，僅僅是安胖在太妃糖基本工資的1/3。管理層還真是把節儉的優良傳統，完美的體現在球隊的任何方面。這樣的做法已經讓隊中大牌球員萌生去意，球隊內部確實是暗流涌動。勤儉持家持家的阿森納，何時才能重現當年49場不敗的輝煌？
