從安切洛蒂到齊達內，銀河戰艦被賦予的氣質為何截然不同？

2019-12-19 1,022條評論 4,273次閱讀 1625人點贊

要是算兩次任期，齊達內在皇馬已經帶了三個半賽季了。這段時長還不夠曾經的弗格森和溫格塞牙縫，但已經算老爹博斯克以後最長壽的主教練了。在他之後，便是火不過三年的穆里尼奧，以及撐了兩年的安切洛蒂們。

近乎偏執的換帥當然是為了皇馬的競爭力，也是為了老佛爺的高要求。弗洛倫蒂諾的主帥要求很好概括：忠誠，聽話，有冠軍，能壓的住球員。只看這幾條，穆里尼奧壓不住球員，齊達內不怎麼聽話，安切洛蒂對球員顯然比對主席來的忠誠。

當然，你不能說安切洛蒂的執教不成功——執教皇馬的兩年里，安切洛蒂刷出了生涯最高勝率，皇馬的那群驚弓之鳥被漸次撫平，球隊還有一個成功的四冠王賽季——那時候被他們踩在腳下，可的是西蒙尼，穆里尼奧和瓜迪奧拉的球隊。

都說齊達內在皇馬一球定千古，但真論起榮譽來，他在尤文拿過兩次世界足球先生，連著進過兩次歐冠決賽，那時候的尤文也能統治米蘭雙雄把持的意甲，奪個兩連冠回來——那時候真正把齊達內樹立為尤文核心的，是還沒完全發福的安切洛蒂。因得人緣廣的緣故，安切洛蒂主隊換了不少，自然桃李滿天下，他和齊達內的重逢也就顯得順理成章了。安切洛蒂執教皇馬時期，齊達內是皇馬B隊主帥，平時也少不了和恩師討教。等到安切洛蒂去了拜仁重回伯納烏，在皇馬主帥席上等著他的已經成了齊達內了。

只看攻守套路，安切洛蒂和齊達內一期的皇馬確實很像：兩邊都主打433或平行中場442，防守端很少局部施壓而是注重站位，都是高控球下的反擊大隊，傳中後都有C羅和本澤馬外加第三者的搶點。這兩支皇馬也都在聯賽里唯唯諾諾，到了歐冠卻能重拳出擊的角色。

安切洛蒂在皇馬呆了兩年，雖說打造陣容還是著眼於平衡，但中場不夠強硬的防守者，阿隆索這類後腰也是文狀元改成的武舉人。但球員時期的齊達內和馬克萊萊維埃拉都合作過，太清楚這類悍匪的可怕了。因而剛成了皇馬主帥，他便提攜了卡塞米羅，二進宮後又提拔了巴爾韋德，讓莫德里奇克羅斯J羅伊斯科競爭上崗。

執教尤文時期，他把齊達內捧成了世界第一人，到了米蘭城立起了4個10號的聖誕樹，還給卡卡搭好了戲檯子，在斯坦福橋他帶出了一套打進百球的433冠軍隊，皇馬時期他創造出了一套能頂三五年的戰術體系——但離開皇馬，安切洛蒂卻自己走進了自己造的戰術死胡同，一味靠著邊翼拉開，中場平衡和鋒線提速招搖撞騙，自己卻已經沒有創新的勇氣和精力了。
