曼城向冠軍進發·布馮希望更多·索爾斯克亞看好哈蘭德

2019-12-19 1,987條評論 4,780次閱讀 3567人點贊

斯特林:沒有獎盃，曼城的進球將毫無價值

斯特林說，如果曼城沒有贏得任何獎盃，那麼他本賽季的進球都將「毫無意義」。

這位英格蘭國腳在英聯杯曼城對陣牛津聯的1/4決賽中，下半場連進兩球，幫助球隊1-3戰勝了對手。他的這兩粒進球讓斯特林在本賽季的全部進球數來到了18個。

斯特林說，「如果沒有獎盃，那我們的進球就毫無意義。我們試圖參與每一個比賽的最終競爭，不僅僅包括英超聯賽，還有足總杯和歐冠聯賽。」

斯特林

布馮想在尤文圖斯追求更多美好情感

尤文圖斯傳奇人物布馮認為，現在退休將剝奪他自己在破紀錄的職業生涯中保持的「美好情感」。

周四，41歲的布馮作為門將登場，球隊以2-1戰勝桑普多利亞，這讓布馮與保羅·馬爾蒂尼平起平坐，成為意甲出場次數最多的球員。這場比賽是他在意甲中的第647場比賽，也是他在尤文的第479場比賽，比之前的俱樂部紀錄保持者德爾·皮耶羅還多一場。

布馮說:「我繼續比賽的原因是，我和我的隊友以及俱樂部一起，仍然可以給自己許多美好的情感，而這些情感是值得保留下去的。」

布馮

曼聯主教練索爾斯克亞：埃爾林·哈蘭德是我喜歡的球員

曼聯主教練索爾斯克亞表示，他很喜歡埃爾林·哈蘭德，他與這位薩爾茨堡紅牛的前鋒有著密切的聯繫。

由於索爾斯克亞和他在莫爾德執教的球員的關係，曼聯被認為是這名少年可能簽約的俱樂部之一。眾所周知，紅魔正在尋找攻擊增援。

索爾斯克亞說，「哈蘭德是我喜歡的球員，我們一直在尋找優秀的球員，他已經有了很好的發展。而他今後的發展應由他自己決定。」

哈蘭德
