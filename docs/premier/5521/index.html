國米外租新星要漲價了，巴雷拉傷情不容樂觀，巴萊羅得到孔蒂認可

2019-11-24 0條評論 13次閱讀 0人點贊

在今天凌晨結束的南美解放者杯決賽中，弗拉門戈拿到了冠軍。在比賽中打進兩球的巴博薩獲得了決賽最佳球員，23歲的巴博薩在回到巴西之後迅速由飲水機球員變身成了球王，他在一年多的時間裡已經打進了超過50個進球，還重新回到了巴西國家隊。

現在是精明的馬洛塔當家，所以這位國米外租新星又要漲價了。之前桑托斯想低價簽回巴博薩，被國米拒絕了。馬洛塔不想和這種只想佔便宜的球隊打交道，所以把他租給了弗拉門戈。這個決定如今被證明相當明智。弗拉門戈把巴博薩當成核心，所以這位國米棄將才能全面爆發。

之前巴博薩還沒有這麼火的時候，弗拉門戈其實只用2000萬歐元就能長期留下他，結果這家同樣摳門的巴甲豪門猶豫了，如今他們再想買斷巴博薩就必須付出更大的代價了。當然了，如果他們掏不出錢，可以用17歲新星雷尼爾來換，想來馬洛塔應該會考慮的。

估計又有球迷想要讓巴博薩回來了。儘管這位新星在巴西的表現相當搶眼，但他能否在國米也有同樣的表現非常值得懷疑，畢竟兩個聯賽的水平相差比較大。巴博薩是吃餅型前鋒，還是核心打法，必須全隊圍繞著他轉。孔蒂並不喜歡這樣類型的球員，另外巴博薩的紀律性和精神屬性同樣很難讓人對他看好。這樣的球員還是趕緊逢高出貨吧。

比起不在計劃內的巴博薩，巴雷拉這樣忠心又拚命的球員才是孔蒂的最愛，然而他今天意外受傷了。更加讓人擔心的是巴雷拉的傷情不容樂觀。國米官方的消息是巴雷拉右膝扭傷，之後經過檢查才能結出具體結論。右膝扭傷最少也要兩周才能恢復，就怕是更嚴重的問題。當時巴雷拉感覺很痛苦，他甚至告訴記者自己聽到了骨折的聲音， 這可不是什麼好消息。

巴雷拉受傷確實很不幸，但比賽還要繼續，孔蒂只能把希望寄托在其它的替補球員身上了。好消息是斯帕萊蒂愛將巴萊羅得到了孔蒂的認可。34歲的巴萊羅在今年夏天拒絕了其它球隊開出的高薪，因為他想留在國米拿一個冠軍再走。這位老將在訓練中相當努力，他的體重減輕，體能也變強了。

巴萊羅今天踢了45分鐘，送出了30次精準傳球和1次成功長傳，跑動也相當積極。不過他兩次嘗試過人沒有成功，還有一次丟失球權以及三次被人輕鬆過掉。總體上來說巴萊羅的表現還是及格的，他也因此在賽後得到了6.7分的評分。孔蒂在接受採訪時點名表揚了巴萊羅，並認為他可以被信任。希望這位老將可以在本賽季實現自己的心愿，祝他好運。
