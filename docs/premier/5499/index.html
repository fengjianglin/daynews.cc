皇馬國米放棄吧，英超才是穆里尼奧唯一的棲息地

2019-11-24 0條評論 16次閱讀 0人點贊

一個月前，沒有多少人會關注這場發生在北京時間20點的「倫敦德比」，比起隔壁的切爾與阿森納，熱刺和西漢姆聯的存在感總歸是稍弱的。但是，這一切都在穆里尼奧到來之後變得不一樣了。一個最直觀的表現就是，這場比賽賽前發布會上來的記者，比熱刺參加歐冠決賽時都多。

熱刺和西漢姆聯為我們奉獻了一場精彩的比賽，孫興慜的傳射，三球的領先，最後十分鐘的緊張與刺激……但是這一切都不如穆里尼奧在場邊的表情包精彩。媒體甚至統計了這個教練出現在鏡頭裡的頻次，90分鐘的比賽，導播竟然40次切給穆里尼奧，這聽上去就有些瘋狂。

比賽剛開場，時隔十一個月，穆里尼奧再次回到英超時，他沒有什麼太多的表情，但是堅毅的表情看上去十分自信。當孫興慜完成第一粒進球的時候，他向空氣揮拳，並與助教擁抱，來釋放激情。當盧卡斯錯失單刀的時候，他捂著臉下蹲，誇張的表情展示自己的遺憾。而當對手進球之後，他大聲呼喊球員，看上去有一些緊張。

再次出現在英超里的穆里尼奧沒有一分鐘的適應期，他像之前幾年裡一樣，面對媒體，面對球迷，面對敵人，為勝利而慶祝，為丟球而遺憾。就像是魚兒游回了大海，鳥兒飛到了森林。用他自己的話說：「過去的十個月，我作為評論員一次次來到球場，但是坐上的卻是轉播席，我心裡難受，這不應該是我在的位置，我應該去的地方是球場中間，那裡才是我天生的棲息地。」

是的，穆里尼奧太熱愛這裡了，他在英超成名，他在英超落敗，他在英超走下神壇，他又在英超重新起航。經歷風風雨雨，看過雲捲雲舒，哪怕這裡曾經嘲諷他，曾經拋棄他，但是他卻永遠懷念這裡，因為只有在英超，穆里尼奧才是穆里尼奧。

在來到熱刺之前，曾經有媒體傳出關於穆里尼奧的緋聞，巴黎聖日耳曼、拜仁、皇家馬德里、國際米蘭甚至廣州恆大都有考慮過狂人，但是狂人卻一直在等待英超的機會。對於他，這個一生不懈追求勝利與激情的男人來說，沒有一個地方能比英超更有吸引力。

他受夠了皇馬的政治鬥爭，受夠了意甲的輿論環境，他更不願意去遠離足球中心的德甲與法甲，所以，他來到了熱刺，再次回到了英超。第一場比賽，他贏了，不僅僅是自己11個月來的第一場勝利，還是熱刺10個月以來的第一場客場勝利。

看上去他要做的還有很多，均衡防守，打造體系，提拔球員，穩定合同，但是，從這第一場比賽開始，穆里尼奧就已經回到了巔峰。
