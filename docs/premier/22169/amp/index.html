變天！一場2-1讓意甲榜首易主，C羅逆天進球太無解

天天要聞
2019-12-19

北京時間12月19日1:55，意甲第17輪迎來一場比賽，尤文圖斯做客挑戰桑普多利亞。比賽第19分鐘，迪巴拉禁區內右側凌空抽射，皮球直奔球門左下角，尤文圖斯取得領先。第35分鐘，卡普拉里禁區內迎球爆射，球進了。10分鐘後，桑德羅左側送出傳中，後點的C羅高高躍起頭槌破門！C羅的這粒進球太逆天了，觸球瞬間頭部離地2.56米。最終，尤文圖斯2-1擊敗桑普多利亞，登上了積分榜榜首的位置。

本場比賽尤文圖斯控球率高達70%，11腳射門4腳射正收穫2粒進球。桑普多利亞14腳射門3腳射正。在賽後whoscored的評分中，C羅得到全場最高的8.3分，4腳射門的數據對於C羅來說並不算多，但他的那粒進球價值千金。C羅已經連續5場比賽取得進球了，這是他在尤文生涯首次做到這一成就。就在人們質疑C羅實力下滑的時候，C羅總能用精彩的表現給予回應。

C羅的進球太無解了。自從2003/04賽季以來，C羅總共進了69個頭槌，比同時期任何球員都多。C羅的彈跳滯空能力在足壇是獨一檔的存在。桑普多利亞主帥拉涅利都被打服了，他說：「尤文的那種進球你能做的就只能是祝賀他們，球迷值回票價了。C羅在足球場上做到了大家在NBA里才會看到的事！」

在賽後，C羅表示這場比賽非常困難，對手表現非常出色，他對於全取3分非常滿意。C羅回應了自己的傷病問題。他說：「之前有1個月我受到了膝蓋傷病的影響，不過現在我已經好了。我的身體狀態非常出色，我想要幫助球隊贏得比賽奪得冠軍。」之前薩里因為考慮到C羅膝蓋問題，接連在比賽中把C羅換下場，讓人們意味他們兩人的關係不佳。事實證明，C羅依然是隊中無可取代的靈魂球員。

尤文圖斯在意甲第14輪和第15輪的時候，先是和薩索洛踢平，隨後1-3輸給拉齊奧，導致讓出了榜首位置。這一次尤文圖斯擊敗桑普多利亞之後，聯賽取得兩連勝，在積分榜上來到了榜首的位置，把國際米蘭擠下去。不過，尤文圖斯先賽一輪，國際米蘭還是有重奪榜首的機會。
