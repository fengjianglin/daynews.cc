19172勝負彩14場，歐冠小組賽最後一輪，誰將晉級？

2019-12-10 800條評論 2,213次閱讀 398人點贊

本期歐冠賽事，也是相當的難，雖然強弱對決有，簽於球隊的晉級情況，戰意難考。本期對一些球隊的戰意及實力和其他因素結合給予簡單提點推薦參考，希望有所幫助。本人推薦只代表個人初盤思路想法，請結合臨場及個人思路定奪，謹慎參考，紅黑勿怪勿噴。

01薩尓茨堡紅牛本賽季依靠哈蘭德狀態火爆，且紅牛火力點全開，大球率相當高，面對來訪的全歐狀態如虹的利物浦我想應該是一場精彩的比賽，以當今歐洲球壇與利物浦對功球隊大部份都是敗下陣來，如果紅牛防守反擊，不激進，會有機會，畢竟紅牛陣中也好手個人能力不錯的球員，本場兩隊都是生死戰紅牛必須勝出晉級，利物浦保平即可晉級，本場難點賽事有條件者可以全包，本場我想去防個主勝盲 點，分勝負。

02多特必須取勝才有機會晉級，只是有機會哦，且坐鎮魔鬼主場我想取勝是問題不大。

03國米本戰張康陽將坐鎮督戰，必須取勝，打平看多特的賽果，巴薩已經頭名晉級。大輪換，布斯克茨、梅西、蘇亞雷斯將休息，而皮克、羅伯托、朗格萊、拉基蒂奇也可能獲得休息，德容和格子會出戰，半主力陣容，且兩隊傳聞關係走的很近，巴薩有可能放水。這場我信陰謀論，不信實力，況且目前的國米也有一定實力，博下主勝，有條件求穩防平應該可以過關。

04本場里昂取勝將攜手晉級。本組三隊同積十分相比對陣關係，里昂取勝小組頭名，本賽季菜比錫狀態出眾，萊比錫打平頭名，但我想萊比錫不會輕易放棄頭名機會，且上場萊比錫主場輸給里昂，有一定的復仇慾望，所以本場我想走客不敗。

05本菲卡確定出局，沒有壓力，歐冠勝一場獎金240萬歐，萄超俱樂部可是差錢哦，為這獎金還是有一定動力的，澤尼特取勝還有機會，看里昂賽果。走兩頭分勝負。

06阿賈克斯打勝平都將出線，巴倫西亞必須取勝，巴倫西亞的防守較好。但巴倫西亞做客面對功擊力及表現欲強勢的年輕小將們，想拿走一場勝利應該比較難。阿賈克斯想要還債，應該是晉級後再還，主不敗了。

07切爾西須取勝才有機會晉級，里爾己經淘汰，專註聯賽。蘭帕德的年輕小將為在歐洲楊名立萬，應該是會努力，主勝。

08礦土與亞特蘭大都存在晉級可能，分勝負。

09薩格勒也存在晉級希望，但必須取勝才有機會。曼城已經頭名晉級。有所輪換。但不排除輪換球員求表現。分勝負，有條件者本場可全包。

10拜仁熱刺兩隊都一晉級。本場旨在練兵。熱刺有復仇體材，且是穆里尼奧上任的第一場歐冠客場比賽。拜仁坐鎮主場，拜仁最近狀態不是很好。防大冷分勝負。

11馬競取勝晉級，打平看勒沃賽果，火車頭已經淘汰，做客應該不會儘力，保聯賽，實力也不允許，為不看別人臉色，馬競必全力爭勝，主勝。

12勒沃須取勝有機會晉級，但必須看馬競臉色。馬競取勝應該是大概率事件。尤文聯賽目前積分落後2分，分的聯賽，本場應該有所輪換。輪換球員為求表現。實力怎麼樣就不知道了，本場首選平局，主不敗。

13布魯日歐冠已淘汰。皇馬已經晉級。布魯日可以爭奪一個歐聯資格，分勝負。

14大巴黎巳經晉級，坐鎮主場，面對本組最弱球隊。實力取勝，

乘午休時間也碼了近兩個小時，腦細胞去了幾斤，呵呵，只是基本面的東西分析，所以大家且看且珍惜，祝大家好運。
