電訊報：紐卡有意一月租借扎卡 米蘭也在關注這名球員

2019-11-14 0條評論 15次閱讀 0人點贊

直播吧11月14日訊 據《每日郵報》援引《每日電訊報》的報道，紐卡斯爾希望能在一月份的冬季轉會窗口租借阿森納中場扎卡。在早些時候，AC米蘭是簽下扎卡的熱門，據報道球員本人已經開始在米蘭城尋找房子。不過喜鵲主帥史蒂夫-布魯斯認為，扎卡會願意為紐卡斯爾效力。

在2-2戰平水晶宮隊的比賽中，扎卡被替換下場，途中他受到了球迷們的噓聲，對此他做出了一些不理智的回應，也因此被剝奪了隊長袖標。自那以後，他再也沒有為槍手出戰過任何一場比賽。

儘管他已經為此事進行道歉，但在被告知未來不太可能再為阿森納出場比賽之後，這名27歲的球員渴望離開球隊。為了確保能夠為明夏的歐洲杯做好準備，扎卡首先需要確定的就是能夠定期參加比賽。

《每日電訊報》聲稱，扎卡已經成為紐卡斯爾在一月份冬季轉會窗口的頭號引援目標，並且球員本人也已經意識到了紐卡斯爾對他的興趣。紐卡後衛法比安-沙爾是扎卡的國家隊隊友，他已經告訴了扎卡關於這支俱樂部和這座城市的事情。

布魯斯希望能夠讓球隊在中場中路的位置上有更多的選擇，並且他相信扎卡是一名能夠給球隊帶來提升的球員。考慮到接下來需要在簽約扎卡的過程中保持競爭力，紐卡斯爾已經暫緩了對其他目標的追求。但最有可能的方案就是先尋求租借，因為一月並不是一個容易永久引進球員的月份。

（馬東宇）
