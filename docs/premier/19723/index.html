國際米蘭！也該好好管管任性的孔蒂了

2019-12-17 2,163條評論 3,697次閱讀 2791人點贊

佛羅倫薩的弗蘭基球場，對國際米蘭是一塊凶地，斯帕萊蒂在這裡，已經連續兩年補時慘遭絕平。前賽季，小西蒙尼第91分鐘亂軍中扳平，上賽季更氣結，補時竟然補了11分鐘，然後吹了丹布羅西奧一個似臂實胸的「手球」，而且主裁看了半天視頻回放，還是吹了點球。

再往前一年，皮奧利更是在半場領先的情況下，下半場大雪崩出4比5，讓伊卡爾迪刷了第一個戴了帽竟然還沒贏的紀錄。

這場，國際米蘭又在一個不好的節點上被絕平，孔蒂賽後的炸雷，更坐實了賽前取消發布會，其實就是他的意思。

自絕於媒體，是過時操作

「新聞沉默」這東西，在義大利絕跡已經有年頭了。它指的是球隊與媒體的不合作，自絕於新聞。最初，是1982世界盃時貝阿爾佐特對媒體追問隊中球員超親昵的曖昧關係不勝其煩，同時也惱怒部分槍手在授意下對自己展開文字圍攻，於是義大利隊搞「新聞沉默」，將媒體排除在外。這一招，很難說沒起什麼作用，畢竟義大利最後擼了世界盃，你說啥都是好的。

後來，這一招被義大利各俱樂部頻繁借用，動輒「新聞沉默」。球隊戰績暴跌，保級吃緊？沉默！哪家報紙哪篇文章讓哪位俱樂部大人物不高興了？沉默！球隊對裁判不滿了？沉默！

但在今天，已經沒人這麼搞了，互聯網經濟讓人意識到，職業足球離不開媒體的宣傳包裝，俱樂部與媒體是相互依存、互惠互利的關係，你自絕於新聞，等於自己劃掉好大一塊流量，對形象沒什麼積極幫助。

義大利足球全方位落後歐陸，不單是競技成績，更多經濟上的，只有更多的開放包容，才會吸引更多流量，贏得更多點擊、廣告、贊助。還搞上世紀新聞沉默那套，是自絕於時代。

偏偏，孔蒂在很不合時宜的時候，為了發泄一下個人的壞脾氣，取消了發布會，向全媒體開戰了。

這是很莫名的舉止，歐冠輸球出局，尤其輸給來放水的巴薩替補，確實窩囊，面子上太不好看。但國際米蘭上下對孔蒂是非常寬鬆寬容的，小組出局，經濟損失幾千萬，也沒有怎麼問責孔蒂。有馬羅塔撐腰，孔蒂無論什麼舉動都被罩著，人比人氣死人，想想去年，斯帕萊蒂哪有這等好處？

孔蒂太倔，已經闖禍無數

被多特蒙德連扳三球大翻盤後，孔蒂就已經出格過一次，竟然炮口對準自己人，直接說引援不夠，10塊錢不夠花了。那次孔蒂沒有得到警告，導致這次他的任性，又一次發作，僅僅因為《羅馬體育報》一篇球迷來信，孔蒂就把事情鬧大，向整個媒體開戰，這恰恰說明，孔蒂表面上「性情中人」，其實非常在意外界對自己的評價，一篇豆腐塊大小的文章，就能讓他暴跳如雷。

孔蒂50了，知天命的年紀，對名帥來說，這是一個節點，過了這個點，不再是少壯，更多憑藉經驗、能力去解決問題，而不是少帥的那種衝勁。

孔蒂的執教有獨特之處，非常容易被球員接納，球隊精神風貌短期內會有非常積極的變化，這些都在本賽季的國際米蘭身上得到了驗證。

但是，阻礙孔蒂成長的，不是別的，恰恰是他的個性，孔蒂的前兩傢俱樂部，尤文圖斯與切爾西，孔蒂都取得了非常優異的成績，完全能夠繼續合作下去，但恰恰都因為「任性」，導致孔蒂與他們都崩了。

執教尤文，僅僅因為未來如何發展意見不一，三連冠後孔蒂直接撂挑子。本來，作為尤文東山再起的第一功臣，孔蒂在球迷心中地位崇高，俱樂部也沒有任何道理換帥。憑藉這些年意甲的經濟形勢，沒有一家是尤文對手，別說三連冠，七八連冠又有何難？即使孔蒂歐冠不行，看在他是中興功臣的面子上，尤文也不會主動趕人，那樣要付出相當大的形象代價，團隊和諧會有相當大的破壞。何況年年打歐冠，怎麼也練出來了，克洛普連輸六次決賽終於贏了個大的，孔蒂如果連沖七八次歐冠，總會有一次運氣好會突破的，到那時「不善歐冠」就沒人說了。

孔蒂撂挑子，用他自己的話說，是因為「10塊錢吃不起100塊餐廳」，其實尤文就算沒一百塊，五六十塊總是有的，除了當紅的C羅梅西買不起，一般次一點的球星總是買得起的，只要孔蒂自己不作，他真的可以成為「尤文弗格森」，一輩子下去。

可是孔蒂一任性，自己掐滅了本該在尤文祠堂里享受八輩子的香火。尤文對孔蒂非常惱怒，阿涅利甚至在最近的股東大會上，不點名刺了一下孔蒂：「當初賭球案，因為有些人在任職的前俱樂部有不法行為，我們一度幾個月沒主教練。」這就是說當年孔蒂卷進了假球，是尤文挽救了他。這不但表明阿涅利對孔蒂已經視為仇敵，而且尤文出身的中興元勛，恐怕再也不會像特拉帕托尼、里皮那樣，獲得二進宮的機會。

執教切爾西，孔蒂再現了他的神奇，迅速把前一年四分五裂還在保級的切爾西擰成一股繩，第一年就神奇奪冠，不消說，俱樂部到球迷當然都十分高興，孔蒂在切爾西前景大好。

可嘆的是孔蒂沒有把英超奪冠視為「萬里長征走完第一步」，相反，而是功成名就，我開始要躺在功勞簿上，秋後算賬了，對曾經流露出離隊踢中超的科斯塔痛下殺手，一條簡訊，直接毀了科斯塔，也導致切爾西資產暴跌，讓俱樂部一下子對孔蒂大為反感，到頭來兩邊為解約賠償掐得不亦樂乎。

脾氣快收不住了還不敲打一下？

孔蒂的脾氣，已經越來越收不住了。執教尤文，三年後發作，執教切爾西，一年後開始鬧，但是好歹那兩家都奪冠了。現在執教國際米蘭，半年不到，才上了個榜首就已經鬧了兩次，馬羅塔脾氣再好，也會犯嘀咕吧？

從對媒體的態度，很難看出一位教練能混得怎麼樣。穆里尼奧儘管成績已經大不如前，但毫無疑問依舊是當今教練圈第一流量擔當，就是因為他非常懂得把握媒體，通過媒體把自己宣傳出去。波切蒂諾與熱刺鬧翻，導火索是熱刺老闆見錢眼開，為拍紀錄片要把攝像頭裝到波叔辦公室，可是穆里尼奧非但不以為忤，還主動配合：來來來，我辦公桌上隨便裝，更衣室、澡堂子……夠不夠？

孔蒂恰恰相反，這次直接拿媒體撒氣，跟一個普通球迷較勁，更把報紙的一個版面、一篇文章，擴大到對整個媒體的開戰，這是越活越回去了。

國際米蘭當務之急，是沉下心來逐漸消化歐冠出局的不利影響，孔蒂這樣做，只是事情鬧得更大，主帥如此心浮氣躁，球員球隊豈能不受影響？

中資對國際米蘭，管理向來是很寬鬆，具體事務幾乎從不干涉，只在大政上拍板，但是，孔蒂此番失態，應該敲打他一下了，不然他越來越急躁出格，不就是輸場球嘛？職業足球哪天不輸球，哪天不被批幾句，有必要這樣小事化大，大事化爆嘛？

該管管孔蒂了！
