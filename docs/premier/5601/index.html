卡薩諾：國際米蘭會贏得意甲冠軍

2019-11-21 0條評論 17次閱讀 0人點贊

新華社羅馬11月20日電 前義大利國腳卡薩諾20日在接受採訪時說，他認為國際米蘭會贏得本賽季的意甲聯賽冠軍。

前義大利國腳卡薩諾（右） 新華社/法新

目前意甲已戰罷12輪，尤文圖斯以32分排名第一，國際米蘭以1分之差緊隨其後。當被問到如何看待意甲冠軍的歸屬時，曾在國際米蘭效力的卡薩諾認為他的老東家將摘取桂冠。

卡薩諾說：「現在的這支國際米蘭讓我想起了孔蒂執教尤文圖斯的第一個賽季，當時他們贏得了聯賽。如果本賽季尤文在爭冠道路上犯錯且國際米蘭持續保持威脅，那麼國米將拿到冠軍。」

卡薩諾還表示，雖然目前的這支尤文實力非常強大，但聯賽已不是球隊的首要目標。「當然，尤文有著最強的陣容，但現在他們的重心在歐冠上。」

效力國際米蘭時的卡薩諾（右） 新華社/法新

作為義大利足壇著名的「壞小子」，卡薩諾此前曾效力於羅馬、AC米蘭、國際米蘭等多支球隊，去年他宣布退役。
