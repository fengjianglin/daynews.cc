歐冠 勒沃庫森VS尤文圖斯：提前出線或放水，賽果其實早已內定

2019-12-11 1,915條評論 3,318次閱讀 1048人點贊

歐冠 勒沃庫森VS尤文圖斯

勒沃庫森：球隊目前聯賽排名第6的位置，球隊小組賽第3的位置，球隊還有機會出線的可能性，不過這只是理論上的可能。而球隊的防守問題還是很差，球隊近28場is只有6場不丟球，不過球隊連續6場不敗，而且聯賽2場力克沙爾克和拜仁強隊，球隊的士氣得到了很大的提升。而球隊進入比賽的狀態比較快，近20場比賽有17場上半場有進球。球隊採取的是4-5-1的防守反擊體系，阿根廷的前鋒阿拉里則是球隊的主力得分點，上場聯賽則是他的梅開二度幫助球隊取勝，他的競技狀態保持的不錯，而球隊的中場核心阿朗吉斯，他的中場防守能力和傳球很出色，他本賽季打進2球和3助攻的出色數據。但是勒沃庫森的邊路防守是球隊的弱點，本德爾和文德爾則是顯得速度不夠快，很容易被對手突破從而形成傳中。

尤文圖斯：球隊目前聯賽排名第2的位置，小組賽第1已經提前出線。由於上輪比賽的輸球，導致尤文圖斯的聯賽積分榜榜首讓出，而且是C羅打進了第一個球後被對手反超，顯然球隊的士氣受到了大大的減弱。值得一提的是尤文圖斯不擅長客場作戰，而且還是面對德國的球隊，他們近7次歐冠面對德國的球隊之贏下1場。由於球隊的傷病問題多，後防線的隊長基耶利尼，前鋒科斯塔，中場拉姆塞都受傷，加上之前受傷的赫迪拉，整個尤文圖斯的攻防兩端都受到了打擊。而且本場比賽戰意不足的情況下，主教練薩里或許會派出替補陣容。

歐洲各大機構給給出的是勒沃庫森讓0.25的數值，後續受到外界的影響，變成勒沃庫森讓0.5的數值，很顯然熱度是在勒沃庫森這邊，這樣的升高數值，實際是對於勒沃庫森贏球的很好保障。從離散度來看，主勝呈現一個兩個高峰三角形夾著中間的一個正梯形，而平局則是波動起伏的下降，和主勝有6次交匯點，客勝則是和主勝的趨勢一致，但是後續客勝是平行的狀態。以目前的數據推薦勒沃庫森主勝，比分2-1,3-0.本場的實單我會放在下方藍色鏈接中，包括晚上歐冠熱刺拜仁，皇馬，巴黎的比賽我也會寫出來，包括實單我會一起放在下方藍色鏈接，大家可做5串1,6串1方案參考（昨天命中國足1-2的比分，三場歐冠全紅）
