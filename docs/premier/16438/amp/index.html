周日 英超 011 曼聯（主）vs埃弗頓（客）

比賽時間：2019-12-15 22:00

　　曼徹斯特聯隊近期表現有所回暖，先後擊敗了熱刺和曼城，積分榜也升至第6的位置，索爾斯克亞的帥位暫時得到了穩固。本賽季曼聯對陣big6球隊取得了3勝2平的不敗戰績，是英超豪門之間對決成績最好的俱樂部，不過需要注意的是本賽季曼聯往往在對陣中下游球隊時失分不少，球隊的狀態飄忽不定。本場比賽拜利、門薩、羅霍、博格巴、達洛特繼續缺陣，盧克肖、林加德出戰成疑，曼聯的首發陣容和上一輪聯賽並沒有太大的變化，拉什福德和馬夏爾搭檔前鋒，弗雷德和麥克托米那搭檔雙後腰。

　　埃弗頓隊上輪聯賽主場3:1擊敗切爾西隊，球隊結束了3連敗的頹勢，目前積17分排在聯賽第15的位置。在原球隊主帥馬爾科.席爾瓦下課之後，埃弗頓名宿鄧肯-弗格森接管球隊後的首場比賽就取得了勝利，換帥如換刀效果十分明顯。本賽季埃弗頓隊客場成績並不是很好，8個聯賽客場僅拿到了1勝1平6負的成績，是英超客場成績最差的球隊。這與球隊目前傷病滿營有不小的關係，主力中場安德烈.戈麥斯、格巴明，主力中場沃爾科特，主力後腰施奈德林，主力後衛德爾夫，輪換後衛馬蒂納因傷缺席，主力後衛迪涅、主力中場科爾曼出戰成疑，多名主力球員缺陣，球隊實力大打折扣。

比賽推薦：主勝

進球數：3 / 4

周日 德甲 016 沃爾夫斯堡（主）vs門興（客）

比賽時間：2019-12-15 22:30

　　沃爾夫斯堡最近狀態下滑，過去7場聯賽輸足4場，而且球隊主場表現也明顯回落，從賽季初5場不敗到如今的兩連敗，主場氣勢處於逐步減弱的階段。雖然狼堡目前防守排名德甲第一，已賽14輪只丟14球，但他們的進攻能力堪憂，是倒數的水平，攻弱守強特性十分明顯。

　　本賽季門興異軍突起，目前力壓拜仁、多特排名積分榜榜首。門興在日前的歐聯杯中敗下陣來，無緣淘汰賽，不過這對於小馬駒而言利大於弊，球隊至少不用一心掛兩頭，可專心於聯賽爭冠。

　　在雙方最近10次德甲交手中，狼堡5勝2平3負，其中主場戰績是3勝2平保持不敗，佔據優勢。主流遊戲數據開出平手盤，同時配置狼堡低水位，但就目前在聯賽的表現而言，門興擺明是優於狼堡，機構此舉明顯是利用狼堡交手佔優而故意打壓門興，有分流門興熱度的意味，綜合考量，本場門興值得追捧。

比賽推薦：主負

進球數：2 / 3

周日 意甲 025 佛羅倫薩（主）vs國際米蘭（客）

比賽時間：2019-12-16 03:45

　　佛羅倫薩本賽季高開低走，球隊在9月份到10月初之間曾取得一波3勝3平的不敗戰績，但是進入10月底後，紫百合的狀態急轉直下，進入11月份後更是一蹶不振，上輪聯賽客場輸給都靈後，球隊近期慘遭四連敗，期間共丟9球，後防線漏洞百出，而鋒線上缺少一個穩定的得分點，主力射手普爾加15輪聯賽僅僅打進3球，而老將里貝里在經過了賽季初的短暫驚艷后，近期也歸於沉寂，主場對陣萊切更是因傷退場，本場比賽能否出戰還是疑問，本場比賽面對國際米蘭，佛羅倫薩想要擺脫連敗頹勢依然不輕鬆。

　　國際米蘭周中在歐冠小組賽主場不敵巴塞羅那，球隊無緣出線資格，這對球隊的士氣是個不小的打擊，不過回到聯賽中，藍黑軍團最近兩輪抓住了尤文圖斯丟分的機會，趁機登上榜首，但是主力中場巴雷拉和森西的先後受傷，還是影響了國際米蘭的陣容深度，近期球隊不得不派老將巴萊羅接連征戰聯賽和歐冠，周中和巴塞羅那血拚一場後，國際米蘭的主力班底消耗較大，鋒線上的盧卡庫和勞塔羅近期更是沒有喘息輪休的機會，此外最近5次做客弗朗奇球場，國際米蘭2平3負無一勝績，而且孔蒂在執教尤文圖斯時，佛羅倫薩就曾多次扮演其苦主的角色，綜合來看，本場比賽國際米蘭不能大意。

比賽推薦：讓負

進球數：3 / 4

周日 西甲 026 巴倫西亞（主）vs皇家馬德里（客）

比賽時間：2019-12-16 04:00

　　巴倫西亞本賽季初因為高層解僱主帥馬塞里諾，一度引起了陣中球員的凝聚力下滑，好在隨著賽程的深入，球員們在場上的表現還是十分職業，蝙蝠軍團最近各項賽事連續9場比賽也取得了6勝2平1負的成績，周中在歐冠小組賽客場擊敗阿賈克斯後，球隊不但獲得了一波三連勝，也成功拿到了出線資格，球隊上下目前士氣正盛，另外巴倫西亞本賽季8個聯賽主場4勝4平保持不敗，和皇馬、巴薩一起成為本賽季西甲聯賽主場還沒被攻陷過的球隊，而且從數據上來看，巴倫西亞本賽季進攻端多點開花，帕雷霍、戈麥斯、加梅羅等人都有著出色的表現，本場比賽對陣皇馬，巴倫西亞自然不想輕易服輸。

　　皇家馬德里近期表現極佳，周中在歐冠小組賽客場擊敗布魯日後，最近各項賽事連續10場比賽8勝2平保持不敗，歐冠獲得出線資格的同時，球隊在聯賽中也和巴薩並駕齊驅，陣容厚度是皇馬本賽季雙線取得佳績的關鍵，賽季初簽下了阿扎爾、約維奇、羅德里戈等人後，又留下了貝爾和哈梅斯等球員，皇馬在賽季初遭遇傷病潮後，戰鬥力依然沒有下滑，而進攻端羅德里戈、維尼休斯兩名年輕球員的湧現，使得主力中鋒本澤馬在前場的壓力下滑不少，專註得分的本澤馬本賽季也創造了在皇馬的進球數據新高，而且同樣是一周雙賽的節奏，陣容厚度更強的皇馬自然比巴倫西亞要有優勢，本場比賽皇馬有信心帶走積分。

比賽推薦：主負

進球數：2 / 3

周日 法甲 027 聖埃蒂安（主）vs巴黎聖母院（客）

比賽時間：2019-12-16 04:00

　　聖埃蒂安近期的狀態非常低迷，上輪聯賽他們客場面對蘭斯，結果面對這支防守型的球隊他們卻丟了三球，開場10分鐘就被對手打進一球，雖然哈默馬曾經扳平過比分，但是很快又被對手連進兩球，最終1比3不敵蘭斯。而在歐羅巴聯賽中，他們依然在輸球，早早出局的他們在客場0比1輸給了沃爾夫斯堡，最終小組賽一場未勝只拿到了4分。

　　17輪聯賽過後，聖埃蒂安7勝4平6負，打進21球，失23球，積25分，目前排名法甲第9位。聖埃蒂安近期的狀態非常低迷，尤其在客場已經各項賽事三連敗，雖然主場很少輸球，但是贏球也不多，大多數是平局，防守線丟球過多讓表現不錯的進攻端也有些無奈。

　　巴黎聖日爾曼上輪聯賽在客場面對勁旅蒙彼利埃，結果巴黎毫不客氣的3比1擊敗對手，上半場巴黎連續被動換人，金彭貝和蓋耶都先後下場，隨後陣容有些混亂的巴黎自擺烏龍先丟一球，下半場在內馬爾的帶領下巴黎進行了反擊，由於蒙彼利埃被罰下1人，多一人的巴黎先是內馬爾自己得分，隨後他助攻姆巴佩將比分反超，終場前姆巴佩幫助伊卡爾迪鎖定勝局。

　　16輪聯賽過後，巴黎13勝0平3負，打進35球，失9球，積39分，目前排名法甲第1。在歐冠中他們的表現也非常出色，他們主場面對加拉塔薩雷根本沒有手軟，幾大神鋒全部取得進球，卡瓦尼也在第84分鐘點球破門，內馬爾2傳1射，姆巴佩也同樣2傳1射，可以說本場比賽將巴黎強大的攻擊力展現無遺，聯賽中打進烏龍球的帕雷德斯本場表現出色，得到了8.2的高分，而頭號球星內馬爾則得到了罕見的10分滿分。
