埃斯波西托：在哪我都想當領袖 盧卡庫就像我的大哥哥

2019-11-21 0條評論 19次閱讀 0人點贊

直播吧11月21日訊 國米青訓小將塞巴斯蒂亞諾-埃斯波西托在接受《米蘭體育報》記者採訪時談到了自己的成長經歷，以及自己和隊友們之間的關係。埃斯波西托表示，在每一支自己效力的球隊中自己都想成為領袖，而盧卡庫就像是自己的大哥哥。

埃斯波西托，你在義大利U17、U18、U19都能有精彩的表現，你認為你應該入選哪一級別的國家隊？

「義大利國家隊。義大利國家隊的球衣獨一無二，義大利國家隊的隊徽別具一格，為義大利國家隊效力的那份驕傲絕無僅有。那些在平時不付出100%努力的球員不會入選國家隊，而在我效力的每一支球隊中，我都想成為領袖。」

你的童年是怎樣度過的？

「我8歲時就離開了家，那時候我就知道我不會擁有一個和大部分孩子們一樣的童年。當我來到國米後，我也意識到了這一點。」

「我的生活里幾乎只有學習和足球，但現在我已經適應了這種生活。我為足球而活，我也已經實現了我的一些夢想。」

毫無壓力地為國米效力可能嗎？

「一切都發生得很快，我可能還沒緩過神來。我為國米U17梯隊效力，然後完成了預備隊首秀，我成為了預備隊的常規主力，並得到了與國米一線隊一同訓練的機會，然後我迎來了歐聯杯首秀、歐冠首秀，然後代表國米在意甲聯賽中出場……」

完成歐聯杯首秀那天，你的家人們沒有來到現場，這是真的嗎？

「很不走運，我的父母沒有在現場觀看我們與法蘭克福的比賽。其實在本賽季剛開始的時候，他們一直都會現場觀看我的比賽，但是很不幸他們並沒有看到我的意甲首秀。我虧欠我的家人很多，可能我會在身上文一個我母親的文身。」

你認為孔蒂是一名怎樣的主教練？

「非常讓人印象深刻，他是一名非常出色的主教練。」

你和盧卡庫相處得如何？

「盧卡庫與眾不同，他非常的謙遜，我進入一線隊更衣室的時候盧卡庫非常熱烈地歡迎了我，對我來說，他就像我的大哥哥一樣。」

「我有很多需要向盧卡庫學習的地方，我和丹布羅西奧的關係也很好，我父親曾是丹布羅西奧的教練，他們對我來說都很重要。」

為國米效力對你的成長有多大的幫助？

「幫助非常大。國米更像是一個大家庭，在這裡，無論是為人處世還是足球方面的技能都會得到提升。我虧欠薩馬登先生（國際米蘭青訓主管）很多。克萊里奇先生對我的成長也很重要，是他將我帶到布雷西亞青年隊接受訓練，很可惜他已經去世了。」

你沒能參加U17世界盃，有沒有想過要參加2022年卡達世界盃？

「這是我的一個計劃，我能有今天的成就，都是因為我腳踏實地，一步一個腳印換來的。」

（鐵甲鋼拳）
