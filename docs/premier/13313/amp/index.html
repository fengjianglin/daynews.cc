漂亮！齊達內客戰巴薩2勝2平保持不敗，齊達內已找到鎖死梅西秘鑰

天天要聞
2019-12-10

巴薩和皇馬的西班牙德比，將在北京時間12月19日凌晨開打，梅西落選了客戰國米的比賽，這也是巴薩對6座金球先生的獎賞，顯然巴薩是想給國米賣個人情，甚至小獅子特爾施特根都不會首發登場，而是派上內托，這一切也都是為了8天之後的西班牙國家德比的比賽做準備，去年沒了C羅和齊達內的皇馬，與巴薩在西甲的首回合交鋒踢得相當慘烈和難看，這也導致了洛佩特吉的下課。

齊達內客戰巴薩未嘗敗績

齊達內是唯一一位執教皇馬，在巴薩客場還未嘗敗績的主教練，在諾坎普齊達內執教的皇馬還從來沒有吃虧過，因此齊達內也期待重回皇馬後繼續在巴薩主場不會吃虧，上賽季洛佩特吉執教皇馬客戰巴薩的比賽，皇馬不僅1-5慘敗，馬塞洛還遭遇了肌肉拉傷，但本賽季的皇馬和巴薩在西甲積分榜上依然呈現出你爭我奪的局面。

而齊達內在今夏對皇馬進行了陣容的補強，雖然本賽季初皇馬的新援都遭遇了傷病困擾，但隨著聯賽的深入皇馬陣中的不少新援都找到了自己在銀河戰艦的定位，這也讓齊達內多了不少的幫手，可以有更多的思路解決球場上的膠著和進球的問題，皇馬高層自然期待齊達內的巴薩第5次客場之旅可以繼續保持不敗，甚至在客場全取3分，在西甲積分榜上反超巴薩，佔據領先優勢。

齊達內找到鎖死梅西秘鑰

對於巴薩的最大威脅梅西，齊達內正在考慮用近來在皇馬陣中表現不錯的F·巴爾韋德來專門盯防梅西，梅西在近來巴薩的比賽中展現了自己的強勁，一個徹底恢復了健康的梅西所向披靡，在西甲射手榜上也成功超越本澤馬，來到了西甲射手頭名的位置，但是巴薩方面已經開始對梅西進行適當輪休，這樣做也是為了避免梅西可能出現的傷病狀況。

巴薩很清楚，一旦球場上沒有梅西，巴薩就會變成二流甚至三流球隊，而沒有了小獅子特爾施特根的巴薩，就徹底淪為了三流球隊，因此巴薩歐冠小組賽最後一輪的比賽，去米蘭城就是為了去給國米送分，巴薩已經做的仁至義盡，能不能笑納大禮要看孔蒂和國米一眾球員的努力了，而齊達內也開始提前部署F·巴爾韋德防守梅西的特訓。

巴薩球迷笑稱：也的確只有「巴爾韋德」防得住梅西

巴爾韋德是防守梅西的秘鑰，的確如此，但是F·巴爾韋德卻不是那個能鎖死梅西，真的可以鎖死梅西的人應該是巴薩主帥巴爾韋德，而不是皇馬陣中的F·巴爾韋德，巴薩球迷的意思就是，只有巴薩主帥主動犯錯，梅西才可能被鎖死，當然這就是巴薩的主動失誤了，而說到找到防守梅西秘訣的球員，如今的下場都不是太好。

上賽季歐冠1/4比賽，說找到防守梅西秘訣的曼聯後衛斯莫林，如今已經被租借去了羅馬，而在羅馬斯莫林似乎重新有了一番新的領悟，並在球場上有了更好的表現。不過對F·巴爾韋德而言，可以有防守梅西的機會，對其而言也是一次歷練和成長的機會，至少可以在防守梅西上獲得經驗，至於是否能夠鎖死梅西，可能還需要實際的比賽才能得到印證。

西班牙國家德比的比賽因為加泰局勢不穩而推遲，如果以當時的情況，皇馬絕對是處於劣勢之中，甚至齊達內的帥位都可能不保，不過現階段皇馬和巴薩應該是五五開的局面，不知道本賽季西甲的首回合交鋒，最後的贏家到底是皇馬還是巴薩。
