第101場國家德比！多特蒙德傳奇發聲：“像個男人一樣去戰鬥！”

2019-11-09 0條評論 16次閱讀 0人點贊

周六晚間，拜仁將坐鎮安聯球場迎來多特蒙德，雙方上演在德甲的第101次「德國國家德比」。這個秋天，德甲前兩極均遇到了不同程度的困難，賽季至今發揮都沒達到預期，紅黃兩強都希望用一場勝利，擺脫前一階段聯賽的低迷，並為之後2/3個賽季做好鋪墊

2010年夏天以來，雙方在德甲的16次對壘時，有一半都是兩強霸佔積分榜前兩位，這一次，拜仁「只是」位列第4，多特1分優勢領先拜仁排名次席。最近26個賽季，只有4個賽季德甲冠軍沒有歸屬南部之星或大黃蜂，拜仁拿到17冠，多特有5個沙拉盤進賬，對於德國足球的統治力可見一斑。

拜仁上輪客場1比5被法蘭克福擊敗，遭遇十年來聯賽最大比分輸球，主帥科瓦奇被下了逐客令，周中和奧林匹亞科斯以及本場國家德比，拜仁都由原助教弗里克帶隊。

歐冠中，擔任過8年德國隊助教的弗里克，在用人上釋放了明顯的信號，他並沒有啟用聯賽被停賽的博阿滕，而是遣上並不受科瓦奇重用的哈維·馬丁內斯和阿拉巴搭檔中衛，為的就是利用國家德比前唯一的練兵機會，培養防線默契。好的一面是，拜仁在連續8場正式比賽丟球後(17球)，諾伊爾終於不用再從球網中撿球。

弗里克在中場拿下了蒂亞戈和庫蒂尼奧，排出了純德班中場，重用精神屬性強大的戈雷茨卡和穆勒，非常時期，弗里克選擇信任球員的鬥志，對於這個中場配置，穆勒表示：「在較短準備時間內，這個中場的效果還是不錯的，尤其是在無球的時候。」

國家德比中，弗里克預計會啟用歐冠首發中的絕大部分人，強強對話，球員對比賽的投入程度往往比技戰術更重要，無論結果如何，弗里克都不希望看到對法蘭克福一役，拜仁鬆散的防守。

大黃蜂在國家德比前適時起勢，打出了本賽季最經典一戰，在歐冠出線天王山之戰，半場0比2落後的絕境之下，3比2逆轉國米。球隊又找回了上賽季初期「下半場惹不起」的感覺。除了進攻速率明顯加快、更多尋求射門之外，多特相比此前最大的變化其實來自於主帥法夫爾。

上周末滿62歲的瑞士人，一改往日安靜的指揮方式，大聲疾呼給球員傳遞信號，肢體語言也豐富了起來，甚至一度激動地沖入場內指揮。多特CEO瓦茨克表示：「法夫爾比原來更有生氣，我很喜歡他現在的執教方式。」

德甲第5至第7輪之間，多特連續三場在2比1領先的情況下，與對手戰成2比2，折損了6個積分，隨後兩個客場對國米和沙爾克04的比賽，多特也遠沒有發揮出應有水平，法夫爾感受到了一定壓力。

如今，瑞士人主動求變，球隊連續兩場比賽下半場發威，分別攻入3球，擊敗狼堡和國米，球隊可以帶著寬廣胸襟前往慕尼黑。萊萬此前在德甲中面對舊主多特狂入14球，波蘭人也承認：「對多特的比賽將是艱難之戰，對國米下半場的表現昭示了他們的進攻有多強。」

多特最近5次在德甲層面做客安聯全敗，得失球比3比22，體育主管佐爾克(也是史上代表多特參加國家德比場數最多的球員)呼籲：「我們之前做客慕尼黑的成績很糟糕，這裡不需要去談什麼技戰術，而是要像男人一樣去戰鬥！」

人員方面，拜仁缺少了兩大中衛聚勒和盧卡斯·埃爾南德斯，加之博阿滕作壁上觀，只能在防線上縫縫補補。多特一邊的隱憂在於攻擊線，隊長羅伊斯因為腳踝傷勢未能趕上對國米的比賽，桑喬在歐冠中大腿受傷，兩人想在德國國家德比中亮相，都要和時間賽跑。

多特蒙德用表現證明，他們已經為國家德比做好了準備，拜仁還在緊湊中做著緊鑼密鼓的準備。紅黃對碰，是多特擺脫安聯5連敗的尷尬，亦或拜仁睡獅猛醒，周六晚見分曉。
