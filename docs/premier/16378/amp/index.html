穆里尼奧執教國際米蘭時期，米蘭城曾一度被塗成了藍黑色

天天要聞
2019-12-16

每當一支球隊戰績不佳的時候，大家總是先責怪主帥，所以本賽季以來AC米蘭表現不好（進入12月以來還是表現可以的），大家也認為是「皮奧利太菜」。有多少球迷們知道，AC米蘭曾經出現過一個真正的「菜鳥主帥」呢？他就是巴西10號萊昂納多。北京時間19年12月15日，義大利體育頻道就討論了萊昂納多給AC米蘭帶來的意義。雖然萊昂納多執教紅黑軍團的階段戰績不算好，但是進攻非常流暢，打法賞心悅目，小羅也曾一度煥發第二春，邊後衛也確立了安東尼尼和阿巴特。雖然他是巴西人，但是他發揮了義大利足球的精髓，中後衛更是組建了蒂亞戈.席爾瓦和內斯塔這對強悍的組合。所以他在告別之際球迷也十分惋惜，可惜最終還是只能和球隊分道揚鑣。

其實在萊昂納多告別AC米蘭的時候，國際米蘭也告別了著名的穆里尼奧，藍黑軍團在那個賽季完成了三冠王霸業，成為了義大利足球史上至今唯一的三冠王。現在義大利體育頻道也承認，穆里尼奧執教國際米蘭時期，米蘭城曾一度被塗成了藍黑色，AC米蘭處於弱勢地位。可是就如同曾經紅黑軍團從巔峰滑落一樣，藍合軍團在電話門後的輝煌也是時候終結了，終結者就快要站出來了。既然說到了萊昂納多對AC米蘭的貢獻，不得不提穆里尼奧對意甲的格局改變。作為國際米蘭「三冠王」時期的主帥，穆里尼奧共執教藍黑軍團一共有兩個賽季，拿到五座各項賽季冠軍獎盃，老球迷們還記得嗎？

08年奧運會之後，穆里尼奧來到了亞平寧半島，開始征戰意甲賽場。08～09賽季，穆里尼奧率領藍黑軍團成功衛冕意甲冠軍。歐冠賽場上，不過在八分之一決賽，國際米蘭敗給了老對手弗格森執教的曼聯。09～10賽季，穆里尼奧迎來了自己在意甲的巔峰時刻，他一手將藍黑軍團帶上了歐洲之巔。斯內德、盧西奧、米利托，穆里尼奧按照自己的設想買來了一條中軸線。那一次，穆里尼奧的主要陣型進化為4-2-3-1。相比於切爾西時期的4-3-3，埃托奧和潘德夫的防守能力要強於羅本和喬·科爾，所以穆里尼奧的變陣具有「看菜做飯」的作用。其實萊昂納多也不弱，只不過AC米蘭當時的陣容無法萊昂納多提供這樣的條件。國際米蘭的防守球員被穆帥賦予了更多的戰術任務，他們既需要在進攻端壓制對面的邊後衛，又要協助防守對面的邊鋒。梅西和羅本兩大球星便倒在了這一戰術下。

當然在國際米蘭時期，穆里尼奧有一個重大成就——在歐冠中贏了瓜迪奧拉領銜的巴薩。正式這個歐冠冠軍，讓10年的夏天米蘭城一度被塗成了藍黑色，在藍黑軍團功成身退的穆里尼奧登上了百廢待興的新銀河戰艦。首個賽季，穆里尼奧便與老冤家瓜迪奧拉五次交手，兩負兩平一勝。穆里尼奧只是在國王杯勉強找回了一絲顏面，總的來說，穆帥最理想的執教生涯，還是在國際米蘭。
