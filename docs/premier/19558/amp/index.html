薩里：拉比奧特能首發出場 歐冠我唯一擔心的對手是皇馬

天天要聞
2019-12-18

直播吧12月17日訊 由於義大利超級盃將在12月22日舉行，因此本輪尤文和拉齊奧的比賽時間進行了調整。北京時間12月19日凌晨1:55，意甲聯賽第17輪的比賽將先賽一場，尤文將在客場迎戰桑普多利亞。尤文主帥薩里出席了賽前新聞發布會，並回答了記者的諸多提問。薩里表示拉比奧特能夠首發出場，拉涅利的球隊從來都不好對付。在談到歐冠時，薩里表示在歐冠中讓自己唯一擔心的對手是皇馬。

根據桑普的風格特點，這場比賽尤文是否能繼續使用4-3-3陣型？

「我還不知道，我會根據球員狀態做出決定。我們昨天進行了放鬆，你們至少得讓我在訓練中看看球員們的狀態，然後再做評估和決定。」

為什麼尤文的比賽提前，而拉齊奧則推遲？

「賽程就是這樣，我覺得這樣也挺好的，只不過讓拉齊奧在星期一踢比賽有點不太好，不過我們也決定不了這個，所以想了也沒用。」

什琴斯尼傷情如何？布馮會出場並追平馬爾蒂尼的紀錄嗎？

「布馮出場？可能吧，但我也不能完全確定，什琴斯尼的小問題我們今天還會進行檢查，不過核磁共振檢查已經做完了。」

C羅能夠連續進行第5場比賽嗎？

「我們看看吧，總體來說，我們前場球員最近的表現都不錯。道格拉斯-科斯塔和拉姆塞都在復出，我們會根據訓練情況以及隊醫的反饋進行綜合評估。」

拉比奧特能夠連續第3場比賽首發出場嗎？

「拉比奧特認為他現在需要進行比賽，如果他的狀態已經恢復得足夠好，他也將首發出場，這沒問題。如果拉比奧特出場，可能馬圖伊迪會做出點犧牲，我可能會讓馬圖伊迪替補出場，但無論怎樣，拉比奧特都能夠首發出場。」

關於陣型和球員選擇

「如果我們只上兩名站位比較靠里的中場，並讓他們踢滿全場，那我們將會很容易陷入困境。因為這樣球員的位置會有重合。不是說只要上三前鋒就一定能得到好的效果，攻守平衡才是最重要的，攻守不平衡的球隊很難在全場都發揮穩定且出色。」

關於歐冠抽籤結果

「現在我對其他事情感興趣。中間還要經歷冬歇期和一系列比賽，所以現在做出判斷只能針對目前的情況做出，並沒有實際意義。在歐冠我唯一擔心的對手是皇馬，但無論對手是誰，歐冠比賽總會非常困難。」

你如何看待抵制種族主義的運動？

「我們必須要提高對這方面的認識，足球應該是純粹的，種族歧視多年來一直都是一個世界性難題，這些運動的初衷都是為了呼籲大家，提高公眾意識。」

你認為對陣桑普的比賽會怎樣？

「我非常喜歡討論比賽，你終於問我關於比賽的問題了！桑普剛剛贏得了德比戰的勝利，他們士氣正旺，拉涅利的球隊總是不好對付。」

如果你派出C羅、迪巴拉和伊瓜因的三叉戟，其他球員在場上的分工會有什麼改變？

「對我們來說，最重要的是做好高位防守，並通過控球給對方施壓。我們必須注意站位，以及失去球權後立刻重新奪回球權。」

（鐵甲鋼拳）
