唏噓！梅羅身後第一球星掉價 英超三豪PK皇薩文爭奪 巴黎已覓備胎

2019-12-21 1,579條評論 4,008次閱讀 1768人點贊

據英國媒體報道，在最近的德轉公布的最新身價排行榜上，大巴黎的頭牌球星內馬爾身價首次出現了下降。內馬爾的職業生涯始於2009年，當時他從桑托斯青年隊晉陞到一線隊，這位巴西天才的身價被估值為90萬英鎊，從那時起，他的身價逐年上升，直到2016-17年他在巴塞羅那時才停滯在9000萬英鎊，此後他在2017年至2018年之間身價繼續大漲，當時PSG從9000萬英鎊增至1.62億英鎊。內馬爾一直都是足壇轉會市場上的熱門人物，不過這位梅羅之後的頭號球星目前已交出了世界第三寶座。

不過，隨著姆巴佩等眾多新星的升起，和近年傷病和負面消息不斷，這樣的內馬爾，身價出現下滑似乎也在情理之中，目前他的最新估值在1.44億英鎊。目前，內馬爾與主帥圖赫爾的關係達到了冰點，巴西人離開巴黎的念頭比任何時候都要堅決，而他的下家眾說紛紜，切爾西、曼城、皇馬、巴塞羅那以及尤文圖斯都是潛在下家，而近期利物浦也加入了這場球星爭奪戰。

《太陽報》消息，這位27歲的巴西球星明年夏天將大概率登陸英超聯賽，而切爾西、曼城和利物浦都有意簽下他。據悉，切爾西，曼城和利物浦都與內馬爾的代表保持聯繫。皇馬和尤文圖斯也保持著對內馬爾的興趣，然而，在2017年以創紀錄的1.98億英鎊的價格將內馬爾賣給PSG後，巴塞羅那仍然是內馬爾的最愛。

而巴黎方面也開始為其尋找合適的替身，他們已確定利物浦王牌馬內作為內馬爾的替補。但是他們也面臨著一場爭奪，以求將姆巴佩留在俱樂部。據傳，現年20歲法國紅星是明夏皇馬的頭號引援目標，儘管姆巴佩年紀輕輕但他的身價高達3.42億英鎊，皇馬已為其籌措足夠的轉會資金，再加上巴西神童維尼修斯，以確保能成功簽下姆巴佩。
