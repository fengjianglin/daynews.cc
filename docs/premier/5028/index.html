鬥士回歸！50歲名帥重返教練席，接受骨髓移植上月剛出院

2019-12-09 0條評論 18次閱讀 0人點贊

塞爾維亞傳奇球星米哈伊洛維奇想必大家都不會陌生，退役之後他繼續征戰綠茵場，成為了一名主教練，先後執教過多支意甲球隊，目前是博洛尼亞隊的主帥。

今年7月，50歲的米哈伊洛維奇突然在新聞發布會中公布了一個驚人消息，他身患白血病。隨後他接受了手術治療，進行了骨髓移植，並於11月22日順利出院。此前近兩個月時間，博洛尼亞一直是由助理教練塔尼加代理執教。治療期間，球員們總是會在賽前和米哈伊洛維奇通話，聽取他的建議，並給予他鼓勵。

北京時間12月9日凌晨，博洛尼亞在意甲第15輪聯賽中主場迎戰AC米蘭，米哈伊洛維奇也重新坐回了教練席，這是他自手術之後首次以主教練的身份帶隊參賽，這種與病魔抗爭到底的戰士精神甚至讓對手都為之動容。

米蘭主帥皮奧利對米哈伊洛維奇的回歸感到非常激動，他在接受義大利天空體育採訪時說道：「見到他（米哈伊洛維奇）真是一種榮幸，我想擁抱他，不過因為他的病情，他說最好不要這樣做。但這仍然令人興奮，這真是一個美妙的時刻。我希望他能完全康復起來，他是在太讓我感動了。」

比賽正式開始之前，踏進球場的米哈伊洛維奇一亮相立刻就收到了全場熱情洋溢的掌聲。

最終，主場作戰的博洛尼亞2-3輸給了AC米蘭，但這並沒有影響到球隊上下的士氣，畢竟靈魂人物米哈伊洛維奇已經回歸賽場，相信他這種戰士精神能讓博洛尼亞在之後的比賽中愈戰愈勇。
