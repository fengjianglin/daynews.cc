他是國米永遠的老爹，無心插柳引入薩內蒂，與羅納爾多父子相殺

2019-12-16 187條評論 1,216次閱讀 109人點贊

談到國際米蘭，就很難迴避掉這樣一個偉大的家族：莫拉蒂家族。對於藍黑軍團而言，莫拉蒂家族就彷彿是圖騰一般的存在，尤其是馬西莫·莫拉蒂，更是被許多內拉祖里發自內心的尊敬著，哪怕在他交出了球隊的大權之後，仍用老爹來稱呼著他。即便是縱觀足壇，恐怕也找不到第二位如此受人愛戴的球隊主席了。

他接手國際米蘭純粹是為了愛：

莫拉蒂家族與國際米蘭之間的淵源，還要追溯到上世紀的中葉。石油大亨出身的安傑洛·莫拉蒂，也就是人們口中的老莫拉蒂，因為妻子埃爾米尼亞深愛著國際米蘭的緣故，而接手了這支球隊。在他的掌舵之下，藍黑軍團迅速崛起，開啟了如日中天的大國際時代。

不過，實事求是的說，安傑洛·莫拉蒂先生雖然對國際米蘭有著巨大的貢獻，可他的實質上，卻更像是一位逐利的商人。在國際米蘭最高峰的時候，老莫拉蒂出售掉了這支球隊，這讓他賺到了盆滿缽滿。不僅如此，他還嚴厲警告自己的家人不要繼續在足球圈拋頭露面了。

但是，馬西莫·莫拉蒂卻與其父有著本質上的區別，他對國際米蘭是發自心底的熱愛，一直都有著回購藍黑軍團的打算。而在1993—94賽季時，他深愛的國際米蘭居然只在聯賽之中排名第14而已，眼看就要墜入了乙級聯賽的深淵，這讓馬西莫·莫拉蒂心痛不已，不顧父親昔日的教訓，從佩萊格里尼的手中回購了藍黑軍團。

無心插柳引入薩內蒂：

然而，相對於他那個精明的父親，馬西莫·莫拉蒂無論是在商業方面還是球隊經營方面，都遜色了不止一籌。就拿足球層面來說，莫拉蒂先生在掌舵藍黑軍團的初期，對於足球的理解就顯然有些誤會，他單純以為囤積足夠的球星就能夠幫助球隊奪冠，尤其是中前場的巨星，更是莫拉蒂先生的心頭大愛。

好在，莫拉蒂先生雖然不太擅長建設球隊，可他的運氣卻著實是不錯，屢有無心插柳柳成蔭之舉。就比如薩內蒂之所以會登陸藍黑軍團，追根溯源其實也是無心插柳所致，而這恰好是莫拉蒂先生掌舵國際米蘭之後的第一筆交易。

在1995年的夏天，莫拉蒂先生看中了當時阿根廷聯賽的明星球員蘭伯特，費盡周折才把他帶到了國際米蘭。至於薩內蒂，在那筆交易之中純粹就只是無關緊要的添頭罷了，在當時很少有人留意到。

然而，出人意料的是，被寄予厚望的蘭伯特很快就泯然眾人矣，反倒是不被看好的薩內蒂，則在梅阿查球場站穩了腳跟，漸漸成長為了球隊圖騰一般的存在，在歷史上都留下了濃墨重彩的一筆。

這是莫拉蒂先生上台之後的第一筆交易，大抵也算是他掌舵國際米蘭的一個縮影了。

與羅納爾多父子相殺：

無心引入的薩內蒂成為了球隊的傳奇，至於那些莫拉蒂先生想盡辦法才引進的球星呢？卻多半未能達到預期，有些還鬧出過不小的波瀾，羅納爾多的例子就極具代表性。

在1997年的夏天，莫拉蒂先生用2790萬美元的天價，一舉創下了當時的足壇轉會費紀錄，這才從巴塞羅那挖走了如日中天的羅納爾多。

對於羅納爾多，莫拉蒂先生簡直就像是對待自己的親生兒子一般，雖然這位巴西巨星因為傷病的困擾，在藍黑軍團養傷的日子甚至還要多出他出場的日子，可莫拉蒂先生依舊是無怨無悔，不僅四處張羅幫助羅納爾多治傷，還在他的養傷期間主動奉上大合同，幾乎就做到了極致。

如果不是莫拉蒂先生無微不至的照顧，頻頻重傷的羅納爾多未必就能在韓日世界盃的賽場上上演王者歸來的一幕。

但是，當羅納爾多重新站在世界之巔之後，這一切都變了。因為與時任國米主帥庫珀合不來，羅納爾多在2002年的夏天挾著世界盃奪冠的威勢找上了莫拉蒂先生，要求他解僱庫珀，否則這位巴西巨星就要求離隊。

雖然莫拉蒂先生把羅納爾多視若己出，可他基本的常識還是有的。一旦開了這種先河，任憑球員逼宮教練，那麼以後球隊就將永無寧日了。於是，莫拉蒂先生拒絕了羅納爾多的要求，直接導致了這位巴西巨星的出走。

更讓莫拉蒂先生傷心的是，當羅納爾多在皇馬日漸艱難之後，他居然投奔了國際米蘭的同城死敵米蘭，並且在米蘭德比中攻破了舊主的大門還肆意慶祝，這讓莫拉蒂先生怒不可遏。自此，這對父子開始了一段時間的冷戰。

雖然現如今這兩人的關係看似恢復如初，但是，錯過了，終究還是錯過了，就再也回不到從前了。

穆里尼奧幫他圓夢：

在建隊方面的缺陷，讓莫拉蒂先生很難重現父輩的輝煌，哪怕他不惜重金先後請來了羅納爾多、維埃里、阿德里亞諾等多位巨星，也很難實質性地改變這支球隊。不過，在吃過多次苦頭之後，莫拉蒂先生終於明白了自身的問題，不再痴迷巨星，轉而去尋找那些足以託付球隊建設的領袖之才了。

在這種大背景之下，莫拉蒂先生遇到了桀驁不馴的穆里尼奧，對他委以重任。

不誇張的說，穆里尼奧雖然執掌過不少球隊的教鞭，可如果說這其中對他最為信任的球隊掌舵者，恐怕非莫拉蒂先生莫屬了，幾乎做到了有求必應的地步，就連已經在巴塞羅那證明過自己不適合豪門的誇雷斯馬，只要穆里尼奧堅持需要，莫拉蒂先生還是不惜斥巨資把他帶到了梅阿查球場，就可見一斑了。

而在球場之外，莫拉蒂先生也在竭力幫穆里尼奧遮風擋雨，無論義大利媒體如何批評這位葡萄牙狂人，總是能夠看到莫拉蒂先生替他辯護的身影，這與日後的曼聯高層形成了極其鮮明的對比。

正是在莫拉蒂先生這種再無保留的支持之下，穆里尼奧才在自己的第二個賽季就打造出了攻守平衡的陣容，不僅在意甲的賽場上繼續稱霸，還在歐冠的舞台上耀武揚威，就連如日中天的巴塞羅那也被藍黑軍團斬落馬下，收穫了意甲歷史上唯一一個三冠王。

而在巔峰之後，就是急劇的滑落，莫拉蒂先生越發有心無力了起來，不得不把深愛的藍黑軍團又交到了其他人的手中，自此歸隱江湖。

不過，這並不能抹煞莫拉蒂先生對於國際米蘭的貢獻以及熱愛，他幾乎把自己的一切都貢獻給了這支球隊。在內拉祖里的心中，他就是永遠的老爹。
