流水的德比，鐵打的梅西，他到底在捍衛什麼？

2019-12-18 43條評論 2,793次閱讀 690人點贊

作為西班牙國家德比現役進球和助攻紀錄保持者，即將在明晨打響的本賽季首回合「世紀大戰」，梅西無疑仍是最高光的存在。C羅離開之後，長達9年的「相愛相殺」只剩巴薩10號獨舞，而阿扎爾、庫蒂尼奧、登貝萊等「億元先生」的集體缺席，也註定星光日漸黯淡的國家德比，仍要靠梅西「撐場子」。

9年來第二次同分進入「世紀大戰」，壓力和機遇之於兩隊不言而喻，而剛捧起金球的梅西，顯然要重現當年率隊征服諾坎普、衛冕西甲冠軍的神跡。同時，德比的系列紀錄，也將因梅西在德比中的表現，再度在年末引爆。

紀錄，只在腳下

41場國家德比，26個進球14次助攻，從19歲到29歲，2次在國家德比上演帽子戲法，在皇馬巴薩兩隊的所有現役球員中，無論進球和助攻，梅西都是獨一檔的存在。從梅西首次亮相國家德比的14年至今，「世紀大戰」舞台上，固然不乏小羅、埃托奧、哈維、伊涅斯塔、勞爾、范尼、C羅、拉莫斯等閃亮的名字，但惟一的主角，是且僅是梅西。

上賽季首回合國家德比，堪稱近年來星光最寡淡、氣氛最詭異的一次：C羅另投意甲，已經令德比坍了半邊天；梅西因此前與塞維利亞一戰手臂骨折，也只能在看台上為球隊助威，而首開紀錄的，則是當時在巴薩境況堪憂、最終遠走的庫蒂尼奧。而再上一次既沒有梅西也沒有C羅的國家德比，出現在2007年的12月23日，當時皇馬在客場1:0擊敗巴薩，當時梅西因左腿傷勢缺席，C羅則還在曼聯效力。

這次缺席，也使得梅西追平小桑奇斯、亨托、哈維、拉莫斯共同保持的國家德比出場紀錄，只能延後到本場來實現。當然，考慮到拉莫斯本場註定首發，梅西想超越皇馬隊長，仍需要漫長的拉鋸戰，但追平前任巴薩隊長哈維，亦是完成俱樂部700場紀錄之後，梅西年內達成的又一座里程碑。

對於皇馬而言，梅西是幾代主帥和球員共同扎心的存在：生涯第3場國家德比，梅西就在諾坎普上演帽子戲法，令三度落後的巴薩絕平皇馬；2016-17賽季次回合國家德比，夏天就將「三十而立」的梅西，在比賽讀秒階段中路包抄射入死角，脫下球衣向伯納烏展現紅藍10號，更是梅西亘古未有的霸氣場景。

「夢三」星流雲散至今，持續弱勢的巴薩，國家德比幾乎是惟一從不讓球迷失望的戰場，但回到場面本身，皇馬有無數次場面佔優、數據壓倒，離取勝一線之間的時刻，無奈最終總會因梅西的天神下凡，功虧一簣。也難怪此番主場作戰之前，此前在諾坎普2勝2平、心理上優勢不小的齊達內，在家門口作戰尚且拒絕透露對梅西採取何種防守策略。畢竟，所有的「針對」，在梅西面前，或許都不是天衣無縫。

獨舞，寂寞並熱鬧著

儘管在西甲射手榜上，梅西和本澤馬以12球並駕齊驅，但明眼人都看得出，和梅西同齡的背鍋之王，即便數據再華麗，也無法和梅西等量齊觀。而這場比賽，雙方數位「億元先生」也一齊缺席：皇馬一邊，仍要繼續養傷的阿扎爾，早在上周就已確定無緣加盟後的首次世紀大戰。而巴薩一邊更慘：兩位億元先生庫蒂尼奧和登貝萊，一個早就租借離隊，如今正在談判折價買斷；一個重傷賽季報銷，明年何時復出都是未知數。惟一能出戰的億元先生，只剩格列茲曼一人而已。

12月初第6次捧起金球獎，歐冠1/8決賽遭遇的又是人心動蕩的那不勒斯，眼下西甲還以凈勝球的微弱優勢領跑，年終歲末，四處收割的梅西，心緒顯然還算不錯。但一生之敵C羅即便遠離國家德比，對金球的熱望卻始終未曾止歇，近幾輪意甲、歐冠的狀態正在復甦。比起渴望「扳平」的C羅，梅西「甩開」的念頭同樣強烈，畢竟，在歐冠總進球數尤其是淘汰賽進球數很難追及C羅的現實面前，「絕代雙驕」只剩一人獨舞的國家德比，將是梅西最手到擒來的戰場。

當然，對於國家德比兩次戴帽的梅西而言，包攬「世紀大戰」各類紀錄固然重要，但仍有一座亟待翻越的高山：巴薩傳奇球星馬蒂·文托拉，曾在1935年完成了西班牙國家德比歷史上僅有的一次大四喜。儘管這已是74年前的洪荒紀錄，但對於已經無數次令球迷將神跡視作「常規操作」的梅西而言，說敢說這是有生之年，他一定無法完成的奇蹟？

決勝，梅西再熟悉不過

早年的西班牙國家德比，一家獨大的皇馬曾有戰績的壓倒性優勢，但伴隨著上世紀90年代至今，巴薩三支「夢之隊」相繼崛起，雙方戰績已經回到同一起跑線。在以往總計242次的西班牙國家德比中，巴薩錄得96勝，皇馬取勝95場，雙方戰平51次。巴薩以一個勝場的微弱優勢領先。而如果僅將舞台縮小到西甲戰場，178次「世紀大戰」雙方各取勝72場、並戰平34次，可謂難分伯仲。

命運似乎總要給皇馬和巴薩製造交鋒的懸念：自巴薩取得西甲榜首至今，他們每輪的戰績和皇馬「神同步」：但凡球隊丟分，「天靈靈地靈靈，馬競輸球皇馬平」幾乎接踵而至。上輪西甲，兩隊不但都在客場被逼平，甚至連所穿的客場球衣，都是綠色！這種「自帶默契」，已經不能用「玄學」解釋了。

儘管17輪35分並列榜首，放眼五大聯賽著實「寒磣」，但對於正在換血期的兩支豪門而言，過去幾年固然在不同戰場各有所獲，實則鮮少以如此均勢的狀況進入德比。上一次雙方以同分開打「世紀大戰」，還要追溯到2010年4月10日，當時面對執掌皇馬元年的穆里尼奧的強勢阻擊，同樣做客伯納烏的巴薩，以2:0輕取對手，並在最後7輪取得6勝1平，那場比賽，首開紀錄的仍是梅西。而在比賽尾聲，自知無望取勝的皇馬球迷開始提前退場，鬱悶的C羅在死球狀態下，憤怒地將皮球踢上了看台。

毫無疑問，這場國家德比的價值，遠遠超出了6分：在球隊重建的新周期，誰能取勝，誰就將佔據爭冠的絕對主動，以及來年重逢時更大的心理優勢。但對於梅西而言，繼續在伯納烏高光演出，他還有另一個目標要儘快達成。

2019自然年聯賽最佳射手之爭中，梅西以33球高居榜首，萊萬以30球緊隨其後，而姆巴佩也已打進了29球。想要不被波蘭神鋒追上，梅西仍要快馬加鞭，畢竟，拜仁今年最後兩場比賽的對手是弗萊堡和沃爾夫斯堡，「瘋狂萊萬」繼續刷數據的可能性不低。而梅西除去國家德比，還要面對一向以硬朗著稱的阿拉維斯，想要維持住3球優勢，本場表現至為關鍵。
