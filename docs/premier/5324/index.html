納因戈蘭：我想證明給國米看，當初放走我是個錯誤

2019-11-28 0條評論 15次閱讀 0人點贊

直播吧11月28日訊 納因戈蘭今夏被國米外租至卡利亞里，而在撒丁島納因戈蘭重新調整好了狀態，在自己發揮出色的同時，還幫助卡利亞里取得了7勝4平2負積25分的好成績，目前暫時排名意甲積分榜第4名。在接受Eleven Sports記者採訪時，納因戈蘭表示自己想要證明給國米看，當初放走自己是錯誤的選擇。

納因戈蘭表示：「我想證明給國米看，他們在今夏選擇放走我是一個錯誤。現在我踢得很好，卡利亞里的成績也很出色，對我個人來說，這非常棒，不過我也不會高興得太早。」

「退出比利時國家隊？這是我經過深思熟慮後做出的決定。國家隊是否會更換主教練已經不重要了。在俄羅斯世界盃之前，馬丁內斯和我說在比利時國家隊中，沒有我在羅馬時所踢的角色。那個賽季我們（羅馬）還殺進了歐冠半決賽。對我來說，就這樣了，後來我們也沒有再聯繫過。」

「現在我想花更多的時間來陪伴我的孩子們，照顧我的妻子，她現在正在和癌症進行抗爭，幸運的是，目前一切都很順利。」

納因戈蘭本賽季為卡利亞里出場11次，其中10次首發，貢獻3球4助攻。

（鐵甲鋼拳）
