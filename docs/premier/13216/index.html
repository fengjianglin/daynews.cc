歐戰奇葩分數！12分無緣歐冠16強，2分可以踢歐聯杯，賈府遺憾

2019-12-11 896條評論 4,791次閱讀 2814人點贊

歐戰奇葩分數！12分無緣歐冠16強，2分可以踢歐聯杯，賈府遺憾。

2019/20賽季歐冠小組賽第6輪已經打響，如今有4個小組已經結束了爭奪，E組中利物浦和那不勒斯攜手晉級，F組中巴薩和多特斬獲前二，國米連續2年無緣16強，G組中萊比錫紅牛和里昂晉級，H組中瓦倫西亞和切爾西殺出重圍，荷甲豪門阿賈克斯則是十分遺憾，他們以10分位列小組第3，值得一提的是，賈府這次10分、凈勝球+6的成績是歐冠小組賽歷史第二好的第3名。

那歐冠小組賽成績歷史第一好的是哪支球隊呢？說到這裡，我們就來看一看歐戰賽場的那些奇葩分數。

2013/14賽季，拉莫斯在歐冠決賽中補時絕平，西蒙尼的馬競在最後時刻痛失冠軍，最終皇馬在加時賽打進3球，以4-1問鼎歐冠。而在這個賽季的歐冠小組賽中也誕生了一個紀錄，當時的F組死亡氣息濃郁，多特蒙德、阿森納、那不勒斯、馬賽這4支五大聯賽的球隊齊聚一組，然而馬賽似乎已經認命，法甲勁旅6戰皆負，其餘3隊則是廝殺慘烈，最終6輪過後，多特蒙德、阿森納、那不勒斯都是獲得了12分，不過由於小分處於劣勢，那不勒斯位列小組第3，無緣歐冠16強。於是，那不勒斯創造了歐冠小組賽成績最好的小組第3的紀錄，這真是一個悲傷的故事。

有高分淘汰，自然也有低分繼續踢歐戰的極限操作。

2017/18賽季歐冠小組賽，皇馬、多特蒙德、熱刺、希臘人競技被分在了H組，這個賽季，熱刺在小組賽中迎來爆發，6輪5勝1平，以16分斬獲小組頭名，皇馬以13分位列第2，而德甲豪門多特蒙德在博斯的帶領下僅拿到了2分，不過由於小分佔優，大黃蜂力壓希臘人競技（也是積2分）獲得了小組第3。也就是說，歐冠小組賽2平4負一場沒贏的多特，最後還能接著踢歐聯杯淘汰賽，你說奇葩不奇葩？

2019/20賽季，阿賈克斯10分踢歐聯，2017/18賽季，多特蒙德2分踢歐聯，這麼一對比，賈府是有多麼的不甘啊！賽後，賈府主帥滕哈格表示：巨大的失望！我們本該進入歐冠16強，但結果卻是這樣。
