國米6中場折損5人卻可能創另類紀錄 孔蒂變回成名陣還是另謀奇招

2019-12-18 261條評論 1,584次閱讀 489人點贊

所有人的目光對準了孔蒂。

在戰損球員幾近達到兩位數、中場中路只剩下一個健康球員的前提下，國米2019年收官戰如何拼湊首發陣容，孔蒂面臨著艱難抉擇。

原本國米本賽季有六個中場球員，隨著巴雷拉、森西、加利亞爾迪尼傷病，布羅佐維奇停賽，巴萊羅在對陣佛羅倫薩的賽後查出小腿肌肉疲勞，現在五個中場都加入戰損名單。

只剩貝西諾一個中場，如何拼湊首發陣容？

天空台指出了孔蒂可能的四種解決辦法。

其一，變陣。

畢竟孔蒂職業生涯是玩424陣型揚名的，他在切爾西也常打343陣型，現在國米倒是有幾個可以踢邊鋒的人選（拉扎羅、波利塔諾等）。

其二，讓什克里尼亞爾客串後腰。

什克在斯洛伐克國家隊踢過後腰，儘管已是舊事了，但這也可以作為一種解決方案。

其三，強要幾名傷員提前復出。

《米蘭體育報》也著重介紹了目前的情況。

在國米這些傷員中，只有巴雷拉是沒有復出可能的。

也就是說，加利亞爾迪尼、森西、巴萊羅都是有可能在2019收官戰復出的。其中，巴萊羅的肌肉疲勞程度有待掃描檢測結果，加利亞爾迪尼也有可能提前復出。

甚至森西都可能成為驚喜，說不定也能有復出勤王的可能。

但顯然，第三種解決辦法，是球迷最不希望的。

畢竟此戰後就是冬歇期，國米下一場比賽是整整17天之後。為了一場比賽，將傷員的恢復周期前提2-3周，這是一件非常冒險的事，萬一傷員付出代價折損更長時間，那即便此戰拿下3分，也是得不償失。

第四種解決方案，是肯定創下紀錄的解決方案：在鋒線讓埃斯波西托頂替勞塔羅，在中場直接委派阿古梅出場，這樣兩名U18球員出場將創下國米紀錄。

埃斯波西托出場不是新聞，畢竟，如果孔蒂打多特、打巴薩都敢押寶於他，那麼打熱那亞小將出場也不奇怪。

但是阿古梅的情況不同，一方面孔蒂幾次表態阿古梅還沒有做好準備，另一方面打佛羅倫薩之戰，阿古梅出場後確實不清楚該怎麼踢。

不過，意媒翻閱歷史後，指出：國米上一次遇到如此類似的中場慘烈兵荒，還是近5年前的曼奇尼時代，當時曼奇尼也有變陣、強要傷員復出、改動球員位置、提拔年輕新星四種解決方案，當時國米大帥選擇了提拔新星，而且效果拔群。

那是2015年4月20日，米蘭德比戰前。

當時的國米主打陣型是4312，球隊中場卻戰損嚴重，瓜林、布羅佐維奇雙雙停賽，奧比輕傷，連可以客串邊中場的若納坦、長友佑都、坎帕尼亞羅也都受傷。

意媒力薦曼奇尼變陣424，或者讓維迪奇等人位置前提去打後腰。

但是曼奇尼做出驚人之舉：在德比戰這樣的大場面，直接提拔了當時年方18歲的法國新星尼烏庫里打首發——當時的尼烏庫里與如今的阿古梅一樣，這次首發之前，只有過一次國米一線隊的替補出戰，時間很短，且表現不佳。

那場德比戰，尼烏庫里表現出色，力助殘陣出擊的國米保住德比不敗。

此戰後，尼烏庫里一躍而成為首發，在六天後國米與羅馬之戰里繼續首發出場，為國米2-1取勝立下顯赫戰功。

在當時，尼烏庫里也是被視作是足壇最具天賦的超新星。遺憾的是，後來檢測出尼烏庫里患有心臟病，他缺席賽場2年半之久，就此泯然眾人。

至今很多行家還堅信，如果不是心臟疾病困擾，尼烏庫里完全可能有很璀璨的職業生涯。

無論怎樣，因為當年的臨危受命，至今國米球迷群仍然銘記著那個驚鴻一瞥的絕世天才。一個因為身體原因而無法盡情展示足球天賦的男孩，能以這種方式璀璨過，也算是他不幸人生中的些許安慰。

時隔近5年之後，孔蒂會不會複製當年曼奇尼的壯舉？這是一個問題。

當然了，曼奇尼精於相人，使用球員更具魄力；孔蒂求勝更為心切，一門心思要和尤文掰手腕，兩人風格完全不同。

但輿論難免會有所期待：如果，五年前的慘烈兵荒，成為一位天才少年將自己的名字烙進一代球迷記憶的絕佳舞台的話，那麼，現在，同樣的橋段會不會再次上演？
