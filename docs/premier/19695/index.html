一位鐵杆球迷的成長史，偉大的CR-7

2019-12-17 2,533條評論 4,553次閱讀 3333人點贊

記得小時候，每次看電視，出現五環體育，播放足球比賽，我都不看，心想這麼多人搶一個球玩，有意思嗎？第一次看足球，猶記得是02年韓日世界盃，中國隊第一次出現在世界盃賽場，雖然盡吞九球，但現在看來，這已是目前為止的最強國足，郝海東，范志毅，馬明宇，等等，從韓日世界盃後，我開始喜歡上足球，最早時候，了解足球是通過廣播，有個廣播節目～足球風暴，那時所有足球的信息都是在哪了解，漸漸的了解到了歐洲豪門，曼聯，尤文，那時的他們是歐洲頂級豪門，我看的第一場歐冠決賽是AC米蘭對陣尤文，雙方對陣120分鐘，未分勝負，直至最後點球大戰，尤文遺憾告負，可能當時是同情弱智吧，我漸漸的成為尤文球迷，同時，當時貝克漢姆所在的曼聯，由於他轉會皇馬，弗格森爵士從當時的里斯本競技買來了18歲的克里斯蒂亞諾.羅納爾多，初來英超賽場，c羅迷戀於盤帶，致使球隊時常延誤戰機，弗格森爵士對於c羅的教誨，使他受益匪淺，直08年，莫斯科的雨夜，盧日尼基球場，雖然點球大戰c羅罰失點球，但特里罰點球時滑倒，進而成就曼聯，成就c羅，09年，歐冠決賽敗於當時的宇宙隊巴薩，c羅遠走伯納烏，與梅西在西甲較量，在皇馬幾經沉浮之後，助皇馬歐冠五年內四次獲得歐冠冠軍，至此，c羅完成球星到巨星的轉變，18年，c羅轉會尤文，尤文，曼聯是我鐘意的球隊，從當年在曼聯得青澀少年，到現在在尤文的巨星，如今35歲c羅的比賽，看一場少一場，大家且看且珍惜～～～～
