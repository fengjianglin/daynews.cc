孫興慜今年第三次染紅，韓國已得罪意甲和西甲，英超是否也將失去

2019-12-23 2,321條評論 4,814次閱讀 3853人點贊

12月23日凌晨，英超第18輪，蘭帕德率切爾西在客場迎戰熱刺。最終憑藉威廉的梅開二度，切爾西2-0擊敗熱刺。但是比賽的焦點卻在的第60分鐘。本是非常普通的一次身體衝撞，孫興慜倒地後卻突然用腳去踢呂迪格，呂迪格隨即倒地。

主裁安東尼泰勒最初只是出示黃牌，但在VAR提醒下，重看後認定孫興慜有暴力行為，重新掏出了紅牌。

其實切爾西上半場就已經2:0領先了，這個惡意犯規完全沒有必要。第12分鐘，威廉左側開出戰術角球，科瓦契奇隨後將球交回，威廉左路內切入禁區後抽射命中右下角，切爾西1比0領先熱刺。

上半場補時階段，加扎尼加出擊解圍失誤撞倒阿隆索，主裁判判罰了點球。

隨後威廉操刀主罰命中梅開二度，上半場結束時，切爾西已2比0領先。

對於孫興慜這次明顯的報復動作，曼聯名宿加里-內維爾點評道：「孫興慜的脾氣太暴躁了，他向呂迪格踹了一腳，VAR的官員們做出了非常勇敢的判罰。」藍軍主帥蘭帕德也談到：「我認為孫興慜是一個優秀的球員，在外人看起來，他像是一個不錯的人。但孫興慜的動作就是紅牌，無論他動作多大隻要做出來了就是紅牌。」而穆里尼奧有明顯的護犢行為，他認為，VAR此時介入是不適當的，因為裁判第一時間不出紅牌，不算是清晰明顯的錯誤，按照規則，只有當裁判出現這類錯誤時，VAR才可以介入。

但是通過回放，我們也能看出呂迪格的表演戲份也很足，孫踢的是右肋，他卻捂的肚子，並且那一腳明顯力道不重。

據統計，孫興慜今年已三次染紅，這一「成就」在英超歷史上只有過一回，只要在今年的最後幾天里，再來一次，就能超越前輩，坐上第一了。

我們來回顧一下今年孫興慜的前兩張紅牌，第一張英超熱刺客場對伯恩茅斯的比賽中，孫興慜就在上半場因為失去冷靜報復性背後用力推搡萊爾馬被直接紅牌罰下。

第二張是和埃弗頓的比賽中孫興慜因為鏟球導致戈麥斯斷腿後果嚴重，但孫興慜的犯規初衷並非惡意，所以即使比賽中拿到了直接紅牌，但賽後經過申訴後紅牌被撤銷。

孫興慜今年的這幾張紅牌極有可能影響到他未來的發展，除了第二張紅牌，其他兩張都是因為衝動造成的，不管一個人球技如何，但是不能控制情緒將會極大縮減職業生涯。孫興慜的球技我們早有耳聞，說是亞洲一哥也不為過，但是日韓世界盃假球事件過後，韓國已經讓意甲和西甲所不齒了（日韓世界盃，韓國擊敗義大利和西班牙被判定為假球，安貞煥在內的多位球員都被義大利俱樂部當即開除）。如果再讓英超也留下不好的名聲的話，韓國將很難再在歐洲聯賽立足。
