足壇核心六大成功轉會，三項要素不可或缺，君臨天下的幕後故事

2019-12-20 1,876條評論 3,652次閱讀 1909人點贊

在足球世界裡，最好的球員都是萬眾矚目的焦點，更是每支球隊中不可或缺的核心，但是這種類型的球員也是最「痛苦」的，特別是在離開成名之地，加盟豪門球隊的時候，新的環境、新的隊友、新的打法往往會讓這些球員無所適從，如果沒有堅如磐石的心理素質、高人一籌的競技水平或者主動求變的適應能力，那麼曾經的「核心」們很有可能會變成水貨，從而一蹶不振。今天就為各位盤點90年代以來，五大最成功的核心轉會（時間排序）。

1.坎通納（天生國王，1992年利茲聯轉會曼聯）

但凡超級巨星，都有著一顆捨我其誰的雄心，有「雖千萬人吾往矣」霸氣，而坎通納則是其中的最極致者。英超元年，弗格森迫不及待的將帶領利茲聯奪得最後一個英超改制前的聯賽冠軍坎通納帶到了紅魔，在那個那個剛從海瑟爾慘案解禁，技術荒蕪的年代裡，坎通納用技術對92班的年輕人進行了啟蒙，更用氣質征服了老特拉福德，即便是那腳飛踹門停賽的8個月，這種氣質，恰恰是英國人最喜歡的——英格蘭球迷對於脾氣暴躁的莽漢性格有著不可抑制的熱愛，如果他本人恰恰是個實力出眾的人，那簡直就是全民英雄。

領子一立，自帶大哥氣場，後面跟著一群小弟橫衝直撞。不服氣是不是？不服氣乾死你。不管面對的是誰，先入場再說，足球就是這樣，不到終場哨響，誰都不知道結果如何，這種強烈的自信與他的前場核心地位結合，誕生了獨一無二的坎通納。無論是踢法還是信心，他都足以影響全隊。他在曼聯只效力了四年半，出場185次，打進82球，但卻打下了深刻的烙印。每當我們想起他的紅魔歲月，只有八個字：絕對王者，君臨天下！

2.齊達內（技壓群雄，2001年尤文圖斯轉會皇家馬德里）

在尤文圖斯時，齊達內已經是世界足球先生了，雖然皮耶羅作為斑馬王子還是球隊的象徵，但是在球場上法國人無疑才是核心。而在皇馬那支巨星璀璨的銀河戰艦里，羅納爾多、菲戈、勞爾、貝克漢姆和歐文在他的光華面前都變得有些暗淡，他是銀河一期毋庸置疑的核心。

低調的法國人，並沒有前輩坎通納那樣的狂傲，相反他顯得極為沉靜，他所依靠的就是比別人更加完美的技術，在2002年的歐洲冠軍杯決賽，在和勒沃庫森的那場比賽中，齊達內一記蕩氣迴腸的半轉身凌空怒射幫助皇家馬德里登頂歐洲冠軍。這一腳射門被皇馬球迷親切地稱為「天外飛仙」。而齊達內自己，則完成了那一代球員中皇帝的加冕儀式。

3.羅納爾迪尼奧（驚才絕艷，2003年巴黎聖日耳曼轉會巴塞羅那）

小羅也許是00年代最有才華的球員，他從巴西來到歐洲的首站是巴黎，雖然只在法國豪門呆了2個賽季，卻迅速成為巴黎球迷的最愛，被認作球隊歷史上最出色的巴黎足球王子，並使得曾經的10號才華橫溢的奧科查被迫離開，但是那時的巴黎並沒有中東土豪的支持，財政危機讓他們只能將小羅拱手相讓給巴薩。

那時的巴薩正處於球隊新老更替的陣痛期，加上死敵皇馬的銀河戰艦政策，讓諾坎普變得前所未有的蕭條，但是球場精靈的到來讓一切發生了改變，小羅幾乎憑藉一己之力對抗對手的天王軍團，第一個賽季就幫助巴薩連續17場聯賽未嘗一敗。第二個賽季，2004-05賽季，羅納爾迪尼奧和他的隊友們全面爆發，並一路領跑聯賽直到最後。羅納爾迪尼奧獲得了2004年和2005年的世界足球先生，並獲得了2005年的歐洲金球獎。

而他對於梅西的關心和扶持，更加顯示了一代巨星的高尚人格，梅西處子進球後，小羅主動背著他慶祝的場景讓人記憶深刻。若沒有小羅，也許阿根廷人不會有那麼寬鬆舒適的成長環境。

4.伊布拉希莫維奇（上帝下凡，2004年阿賈克斯轉會尤文圖斯，2006年尤文圖斯轉會國際米蘭，2009年巴塞羅那轉會AC米蘭，2012年AC米蘭轉會巴黎聖日耳曼，2016年巴黎聖日耳曼轉會曼聯）

足球歷史上，優秀球員如過江之鯽，數不勝數，但是能夠頻頻口出狂言，卻又一一做到的人也許只有他一人——自認上帝的伊布。「茲拉坦不做綵排。」 當伊布19歲的時候，他曾經幾乎加盟阿森納，槍手甚至已經為他準備了印有他名字的球衣，但最終這筆轉會沒有成行，原因是溫格想看他試訓一次。一個19歲的少年敢懟世界名帥，自信與狂妄就是一線之間。而他無疑是後者。

除了在巴薩失落的一年之外，無論是在阿賈克斯、尤文圖斯、國際米蘭、AC米蘭、巴黎、曼聯還是洛杉磯銀河，他都是當之無愧的進攻核心，「得伊布者，得天下！」就如他自己所說：「我能做的只有笑笑自己有多完美。」今年伊布在家鄉馬爾默的雕像正式揭幕，瑞典人的豪言：「到紐約看自由女神像，到瑞典看茲拉坦像！」看看就是這麼傲嬌、這麼自信，而且又這麼讓人心悅誠服！

正如那段流傳已久的對話，記者：誰能贏得世界盃預選賽資格賽？伊布：只有上帝知道。記者：向上帝提問似乎是件難事。伊布：為什麼？你正在看著上帝啊。

5.C羅（總裁霸道，2009年曼聯轉會皇家馬德里）

如果願意，C羅絕對可以在曼聯呆到退役，24歲之前的他所取得的成就就已經超越了俱樂部的前輩坎通納，只是對於足壇歷史上勝負心最強烈的男人來說，老特拉福德已經容不下他的野心了，只有伊比利亞半島的那座聖殿才是他靈魂安放之所，即便到那裡意味著一切重來。

C羅讓弗洛倫蒂諾的銀河二期成為了皇馬百年輝煌長卷中最耀眼的篇章，9年的時間，C羅一共在438場比賽打入450球，拿下了共計16個冠軍，5年4歐冠前無古人，他與梅西開創的雙雄時代即便是未來也很難再有來者。對於他的形容，不需要太多的語言，到此足夠了。

6.皮爾洛（廉頗能飯，2011年AC米蘭轉會尤文圖斯）

在來到米蘭之前，皮爾洛是一個優秀的前腰球員，但也就是優秀而已，這種檔次的球員在當年巨星雲集的國際米蘭是無足輕重的，於是AC米蘭用一個傷病纏身的科科就換來了未來十年的棟樑。在這裡，皮爾洛迎來了人生命運的轉折，安切洛蒂將他放到了後腰位置上，經典的「四個10號」同時出場的陣型正是以皮爾洛為中心，第一個「組織型後腰」由此誕生。加圖索和安布羅西尼成為他的保鏢，西多夫和魯伊科斯塔在前面作為緩衝。那些年，皮爾洛在在球場上釋放了自己的全部情感，就像是彈開的螺旋彈簧一般，不斷帶領隊伍走向勝利，

2011年，米蘭認為他老了，於是將他送給了老婦人，誰知道在這裡，他卻繼續延續著職業生涯的輝煌。他向世界詮釋了現代足球最偉大的後腰的含義，他所在的位置和所做的事情，從過去到現在，甚至是在未來，對於足球來說都是唯一，四個賽季共為球隊出場164次，打進19球(其中15球來自定位球)，並有35次助攻。幫助球隊奪得七座冠軍獎盃。
