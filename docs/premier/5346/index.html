歐冠晉級形勢：八隊晉級淘汰賽，餘下諸強第一的利物浦卻有險？

2019-11-28 0條評論 14次閱讀 0人點贊

今天凌晨，歐冠小組賽第5輪第2比賽日的8場比賽全部結束，截止目前，小組賽已經有8支球隊晉級淘汰賽。在餘下的8支球隊中，切爾西、國米等球隊佔據了晉級的主動權，而目前處於小組頭名的利物浦卻在理論上存在險峻。

八支球隊晉級淘汰賽，巴黎拜仁等5隊鎖定頭名

A組中，雖然皇馬在2-0領先的情況下被大巴黎追為2-2平局，但是在此前進行的同組比賽中，布魯日與加拉塔薩雷戰成平局，皇馬已經拿到小組第二與大巴黎一起晉級淘汰賽。

B組中，已提前晉級的拜仁用一場大勝展現出了自己出色的狀態，而穆里尼奧的熱刺在主場0-2落後的情況下完成4-2的大逆轉，以小組第二的身份晉級。

C組中，曼城主場戰平礦工，鎖定小組第一的同時晉級淘汰賽；D組中，尤文在主場1-0取勝馬競，上輪已經完成晉級的他們鎖定了小組頭名；F組中，巴薩憑藉著梅西的出色表現在主場3-1取勝多特，提前一輪鎖定小組第一；G組中，萊比錫紅牛在0-2落後的局面下於最後階段由福斯貝里連入2球博得平局，球隊雖然沒有確定小組頭名，但是已獲得晉級資格。

綜上所述，5輪比賽之後大巴黎、皇馬、拜仁、熱刺、曼城、尤文、巴薩、萊比錫紅牛均成功晉級淘汰賽。

剩餘八個名額：紅軍理論最險，切爾西國米佔據主動

C組

頓涅此刻礦工（6分）、薩格勒布迪那摩（5分）、亞特蘭大（5分）將在最後一輪拼該小組的最後一個晉級名額。賽程上，在最後一輪的對決中，薩格勒布迪那摩在主場對陣曼城，頓涅茨克礦工在主場對陣亞特蘭大。

晉級形勢最好的是礦工，只要他們主場取勝亞特蘭大就能確保一個淘汰賽席位。一旦戰平對手，則需要寄希望於曼城在對陣薩格勒布迪那摩的比賽中保持不敗。本小組的晉級形勢，算是較為複雜。

D組

馬競以7分暫列小組第二，而勒沃庫森拿到了6分，莫斯科火車頭已經提前出局。賽程上，馬競最後一輪在主場對陣已經出局的莫斯科火車頭，取勝對手就將晉級。而勒沃庫森要想晉級，不但要在最後一輪主場取勝尤文，還要寄希望於馬競主場丟分。

E組

利物浦目前以10分位居榜首，那不勒斯（9分）、薩爾茨堡紅牛（7分）緊隨其後。賽程上，那不勒斯在最後一輪主場對陣早已出局的亨克，取勝概率極大，不出意外的話將取勝晉級；紅軍在最後一輪將客場挑戰進攻實力出色的薩爾茨堡紅牛，一旦輸球很可能跌到小組第三。

另外，若利物浦、那不勒斯、薩爾茨堡三隊同積10分，比較三隊小「聯賽成績」，那不勒斯8分第一出線，薩爾茨堡和利物浦同積4分，薩爾茨堡憑小聯賽總凈勝球多排第二出線，利物浦在這種情況下會出局。

可以說，目前佔據小組頭名的利物浦在晉級形勢上較為兇險。

F組

目前國米與多特同積7分，但是國米因為凈勝球的優勢暫時壓制多特。小組賽最後一輪，國米將在主場對陣已提前鎖定頭名的巴薩，而多特則在主場對陣布拉格斯拉維亞。

國米目前比多特多了3個凈勝球，只要球隊主場取勝巴薩且多特追不上凈勝球，國米就將晉級淘汰賽，可以說，國米是佔據晉級主動的。

G組

聖彼得堡澤尼特與里昂同積7分，兩支球隊的戰績均為2勝1平2負，而且得失球也是7-6。在最後一輪的賽程中，澤尼特將客場挑戰本菲卡，而里昂則在主場對陣已經晉級但是沒有確定頭名的萊比錫紅牛。屆時，雙方要想爭奪出現資格，澤尼特在交鋒戰績中1升1平佔據優勢。

H組

阿賈克斯（10分）、切爾西8分、瓦倫西亞（8分）位居前三，切爾西在凈勝球與瓦倫西亞相同的情況下因為進球多暫時佔據第二位。

小組賽末輪的賽程為阿賈克斯vs瓦倫西亞、切爾西vs里爾，這對於切爾西來說是一個絕對的利好消息，只要取勝就鐵定晉級淘汰賽。

（開水）
