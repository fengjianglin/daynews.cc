西甲雙雄將迎巨變！皇馬已贏得未來 巴薩緊靠梅西苦苦支撐

天天要聞
3 天 ago

西甲雙雄之爭是足球史上永恆的話題，儘管巴薩在過去幾年時間內佔了上風，但這終究是過去式。皇馬巴薩都在著眼於未來，不過目前為止，皇馬似乎已經贏下了未來之爭。皇馬外租的球員一開始閃耀，而巴薩卻黯淡了許多。

盤點皇馬年輕外租球員

厄德高

厄德高本賽季被皇馬外租至皇家社會，儘管挪威中場目前只有20歲，但厄德高在皇家社會已展現出強大的實力。之前效力荷甲的維迪斯時，厄德高就在賽季末擠進了荷甲最佳陣容，值得一提的是，厄德高在當時是唯一一個不是效力荷甲豪門阿賈克斯及埃因霍溫卻入選荷甲最佳陣容的球員。而目前效力於皇家社會的厄德高表現相當驚艷，在9月更是獲得了西甲最佳球員，代表球隊出場的15場比賽中，更是打進了3球送上4助攻。在今晨對陣巴薩的比賽中，厄德高表現同樣出色，如此出色的表現也讓皇馬準備提前召回厄德高。

久保健英

久保健英曾被巴薩視為第二個梅西，不過日本天才在今夏選擇加盟皇馬。本賽季因沒有足夠的出場機會被外祖至馬洛卡。久保健英在馬洛卡已經坐穩主力的位置，作為中場球員的他擁有出色的視野、組織能力以及盤帶能力。皇馬主帥齊達內更是毫不掩飾的稱讚久保健英十分出色，將來一定會成為一名出色的皇馬中場。

阿什拉夫

現年21歲的阿什拉夫今夏被皇馬外租給多特，在加盟多特之後，阿什拉夫逐漸成為了多特的主力邊後衛，本賽季迄今為止已打進5個進球，並送上3次助攻。在歐冠對陣國米的關鍵戰中，阿什拉夫梅開二度幫助多特逆轉國米，也讓多特成功在歐冠中小組出線。

除這些外族的中場球員，皇馬隊內也用著大量出色的年輕球員。羅德里戈、維尼修斯、巴爾韋德、米利唐都逐漸展現出自己的實力，用不了多久皇馬的陣容將徹底更新換代，屆時銀河戰艦將再次起航。

盤點巴薩外租球員

拉菲尼亞

現在26歲的巴西中場拉菲尼亞在今夏被巴薩外租至塞爾塔，不過在租借之後，拉菲尼亞就因傷缺戰。這已經不是拉菲尼亞的第一次受傷。或許拉菲尼亞是一名出色的球員，但頻繁的傷病讓拉菲尼亞的狀態難以穩定。或許巴薩這次外租拉菲尼亞之後，沒有打算召回的打算。

庫庫雷利亞

庫庫雷利亞出自巴薩青訓，巴薩在今夏從埃瓦爾回購之後又外租給赫塔費，本賽季在赫塔費表現平庸，目前已代表球隊出場21次，僅僅送上4次助攻。

庫蒂尼奧

庫蒂尼奧在今夏被巴薩外租給拜仁。曾經巴薩簽下庫蒂尼奧以為他能夠代替離隊的伊涅斯塔，然而現實非常的殘酷。在上賽季受傷之後，庫蒂尼奧始終沒能在巴薩有良好的表現。即使庫蒂尼奧為巴薩進球，巴薩球迷仍對他噓聲不已。由於上賽季災難性 的表現最終讓巴薩將其外租。而庫蒂尼奧在加盟拜仁之後表現極不穩定，有時表現出色，有時堪稱災難。顯然巴薩希望拜仁將其買斷以減少損失。

相較於皇馬，巴薩外租球球員以及隊內的年輕球員顯得不那麼出彩，甚至有些糟糕。幾乎沒有外租的球員有實力在巴薩踢球，而在隊內，目前僅有德容、阿圖爾、阿萊尼亞、佩雷斯以及法蒂。然而阿萊尼亞長坐板凳，德容最近變現平平，而阿圖爾更是因為腹股溝休戰，目前巴薩的主力仍靠著梅西等一幫老將苦苦支撐。巴薩的更新換代不知要等到何時。
