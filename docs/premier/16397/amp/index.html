足壇往昔：10位實力大於名氣的小眾球星，資深老球迷的“回憶殺”

天天要聞
3 天 ago

早年間的世界足壇，湧現過無數璀璨的巨星，從馬拉多納到荷蘭三劍客、再到羅馬里奧、巴喬、羅納爾多，他們星光熠熠，都是那個時代不同時期的代表，成為了球迷們心中的偶像。

但總有一些球員，他們頭上沒有巨星的光環，卻各自在足壇上留下過不可抹去的痕迹，留給我們或多或少、或深或淺的烙印，那是我們的足球記憶，綠茵場上也因為有了他們，增添了很多色彩，他們不是傳奇，但也值得我們懷念。這些名氣不大實力不錯的小眾球星，你還有多少印象？

尼利斯/比利時

捧杯者為尼利斯

出道比利時的安特萊赫特，但是職業生涯的高光是在埃因霍溫。還曾榮獲1995年的「荷蘭足球先生」。90年代中後期，與當時年輕的范尼組成了令人恐怖的鋒線組合，在歐洲足壇享譽一時，尼利斯助攻，范尼破門，是很長一段時期埃因霍溫的常規操作。尼利斯代表國家隊總共出場56次，進球10個，於2002年退役。

埃爾南德斯/墨西哥

長發飄飄的埃爾南德斯

我們說的這個埃爾南德斯可不是「小豌豆」，而是路易斯.埃爾南德斯。對他印象最深刻的就是1998年的世界盃，4場比賽4個進球，激情的慶祝，飄逸的金髮，是20年前那個夏天一道美麗的風景。埃爾南德斯職業生涯沒有在歐洲大陸踢過球，但是依然不能阻擋他成為一個優秀的前鋒，他在1997年和1998年，兩度當選墨西哥足球先生。也因為一頭長髮，被稱為墨西哥的「風之子」。

莫斯托沃伊/俄羅斯

俄羅斯「沙皇」

被譽為「俄羅斯沙皇」的莫斯托沃伊，職業生涯大部分效力於西班牙的塞爾塔俱樂部，在效力塞爾塔的7年時間，他帶領球隊從一支名不經傳的地方球隊，變成一支在歐洲讓人不可小覷的勁旅。作為中場核心，莫斯托沃伊具有犀利的盤帶和超強的組織能力。2005年退役，至今，塞爾塔球迷仍然認為，莫斯托沃伊是他們「最後的10號」。

德賈明哈/巴西

拉科魯尼亞時期的德賈明哈

德賈明哈，一個不可一世的巴西天才。這個曾經在球場上戲耍齊達內的人，因為自己的桀驁不馴和怪脾氣，被足壇貼上了「壞球員」的標籤。德賈明哈的天賦毋容置疑，但是心高氣傲，當年在被問及為什麼不努力進入巴西國家隊，他的回答是：如果我進國家隊只是為了給里瓦爾多當替補，又什麼意義？德賈明哈創造力十足，過人技巧令人嘆服，但是他的「惡習」，阻止了他成為更耀眼的明星。

尼哈特/土耳其

尼哈特

尼哈特+科瓦切維奇的鋒線組合，是當年皇家社會在西甲賽場大殺四方的核武器。尼哈特身材瘦小，跑位靈動技術出色，科瓦切維奇高大強壯，衝擊力十足，他們天然互補。在2002-2003賽季，皇家社會直到最後一輪，才被皇馬反超，痛失西甲冠軍，但是尼哈特和科瓦切維奇的完美表現，征服了所有球迷。

施耐德/德國

勒沃庫森隊長施耐德

施耐德曾代表德國足球國家隊參加了2002，2006兩屆世界盃和2004的歐洲杯。勒沃庫森的隊長，最悲情的記憶就是01-02賽季的「三亞王」，歐冠一路殺到決賽，卻被齊達內的一腳天外飛仙斬於刀下，聯賽和德國杯賽也是痛失冠軍，當時那支擁有巴拉克、施耐德、諾伊維爾、盧西奧的勒沃庫森，令人惋惜。

拉杜喬尤/羅馬尼亞

拉杜喬尤

拉杜喬尤是羅馬尼亞1994年黃金一代的主力前鋒。在1994的美國世界盃，羅馬尼亞令人驚艷，在哈吉的帶領下，1/8決賽淘汰了阿根廷進入八強，掀起黃色風暴。只是在1/4決賽時點球負於瑞典，無緣半決賽。拉杜喬尤作為鋒線擔當，在這次比賽里打入4球。拉杜喬尤職業生涯輾轉很多俱樂部，也曾短暫效力於當年歐洲豪門AC米蘭，但始終是不得志，最後泯於眾人。

特里斯坦/西班牙

l特里斯坦

21世紀初期，拉科魯尼亞曾有個很好聽的稱號，叫「超級拉科」。當年的拉科魯尼亞異軍突起，在1999/2000賽季的西甲聯賽中，擊敗不可一世的皇馬和巴薩，成功奪取冠軍。還曾在02的歐冠比賽中，淘汰米蘭，晉級4強。當年拉科的鋒線雙子星就是荷蘭的馬凱+西班牙的特里斯坦。

莫菲奧/義大利

國米時期的莫菲奧

莫菲奧，估計能記起這個名字的球迷不算多，意甲死忠或許會有印象。一個天才型的球員，典型的前腰10號球員，直傳球很詭異，穿透力強 ，經常能傳出一腳神出鬼沒的助攻，身材瘦小，盤帶靈巧 ，技術極佳。效力過國際米蘭和帕爾馬。

扎霍維奇/斯洛維尼亞

扎霍維奇

扎霍維奇應該算是斯洛維尼亞的第一球星。在2000年的歐洲杯讓他一站成名，扎霍維奇雖然並不是出自足球強國，但他本身擁有著頂尖的技術能力，超強的攻擊慾望，當時的斯洛維尼亞可以說是扎霍維奇一個人的球隊，在這個星味暗淡的隊伍里，他是唯一的明星，但是後來也因為脾氣乖戾，被國家隊除名。

老球迷們，你還記得哪些實力大於名氣的往昔球星，歡迎來侃！
