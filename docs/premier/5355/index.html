足壇最紅新星引7豪門爭搶！西甲雙雄英超四強混戰 拜仁開天價遭拒

2019-11-28 0條評論 17次閱讀 0人點贊

北京時間11月28日，今天的頭條留給「香餑餑」哈弗茨，還有什麼值得我們關注的新聞，我們一起來回顧一下

1、圖片報：曼聯、利物浦等7傢俱樂部有意引進哈弗茨

據報道，現年20歲的哈弗茨在本賽季的出色表現吸引了多家豪門的注意力。而據消息人士透露，利物浦、曼聯、切爾西、曼城、拜仁、巴薩和皇馬都對他十分感興趣，並且他的經紀人也已經和這7傢俱樂部進行了接觸。此前有媒體報道稱拜仁曾以9000萬歐元報價哈弗茨，但沒有成功。

據了解，目前哈弗茨的身價已經漲到了1.3億歐元。

2、波切蒂諾前助教：波切蒂諾不會德語，很難融入德國球隊

近日，波切蒂諾的前助教斯蒂芬-布倫德接受了《體育圖片報》的採訪，談到了關於波切蒂諾執教拜仁的傳聞。

布倫德表示：「執教熱刺的五年中，他帶領熱刺4次進入歐冠賽區，這是很不容易的事情。熱刺的財力在英超可能只能排在第六，然而波切蒂諾讓熱刺發生了改變，在他手下，凱恩和阿里成長為了明星球員，他為熱刺培養了很多年輕球員。他有著非凡的能力。」

「但波切蒂諾並不適合執教德國球隊，一方面是他不會說德語，另外他的教練團隊也從未接觸過德語球隊。儘管他會一些英語，但他更喜歡講西班牙語。因此，他很難執教一支德國球隊，德甲霸主的執教難度更是非同一般。」

3、信使報：帕洛塔有意出售羅馬，要價7.5億歐元

據報道，在過去的幾個月里，帕洛塔一直在為出售羅馬俱樂部做準備，而目前的潛在買家有卡達家族集團，這也是巴黎聖日耳曼的老闆。美國的弗里德金集團也對收購羅馬俱樂部很感興趣。羅馬主席帕洛塔也承認有進行過相關談判。

據報道，該集團與金融顧問集團摩根大通正在對羅馬的財政情況進行調查。而從透露出的消息來看，帕洛塔對俱樂部的要價達到了7.5億歐元，這其中也包含了建造新羅馬體育場所需的8000萬歐元費用。

4、隊報：阿森納和里昂有意烏姆蒂蒂

據報道，巴薩後衛烏姆蒂蒂可能在冬窗離開巴薩，包括阿森納和里昂在內的多傢俱樂部都有意引進。此前《每日體育報》就曾報道，烏姆蒂蒂的傷病極大的限制了他在巴薩的發展，如果在不能在短時間內恢復，巴薩將在明夏引進一名中衛替代烏姆蒂蒂。

近兩個賽季，烏姆蒂蒂始終沒有從傷病中走出來，他在上賽季的聯賽中更是只為巴薩出戰了14場比賽。本賽季至今，烏姆蒂蒂也因傷僅為巴薩出場了3次。

5、天空體育：維爾通亨仍想和熱刺續約，不在乎主教練是誰

據報道，現年32歲的維爾通亨與熱刺的合同在明夏就會夏到期，儘管熱刺更換了主教練，但他仍舊希望留在熱刺，這一點並不會因為主教練的不同而有所改變。

據了解，羅馬和勒沃庫森都有意引進維爾通亨，不過在今夏，熱刺曾告訴維爾通亨他是球隊的非賣品。

這位比利時球員是目前效力熱刺時間最長的球員之一，在隊內擁有極高的威望。他在2012年就從阿賈克斯轉會到了熱刺，至今已為熱刺出戰了294場比賽。
