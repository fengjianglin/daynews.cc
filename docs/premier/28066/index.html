意甲前瞻：佛羅倫薩VS羅馬 冬歇期前最後一輪紫百合能否不敗收關

2019-12-20 88條評論 1,855次閱讀 383人點贊

喜歡大催可以點擊關注，常來看看。

冬歇期馬上就到，買人的買人賣人的賣人，轉會市場又要迎來不少看點，大催比較關心魔力鳥會買誰，老闆那麼摳，鳥叔會咋辦。

意甲：佛羅倫薩VS羅馬

比賽時間：12-21 03:45

本輪比賽完結後，要到明年的1月6日聯賽重啟，所以是2019年最後一輪比賽。由戰績欠佳的紫百合主場迎戰羅馬。意甲16輪過後，紫百合積17分排名聯賽第14名，僅領先降級區4分；羅馬目前32分排名第4，已經擠身歐冠資格，與第5名卡利亞里高出3分，後面是緊追不捨的亞特蘭大與帕爾馬，羅馬目標劍指前三與拉齊奧僅相關4分。

佛羅倫薩：

紫百合上賽季就驚險保級，本賽季開局到現在16輪後打平上賽季排名，離降級區又不遠，任務重狀態差。主場賽8輪只拿到2勝，平3負3，勝率不足30%，8個主場打進9球失10球，好在的是近3場均有進球，上輪1-1平國米拿到寶貴1分，可惜種子選手小基耶薩在上輪戰平國際米蘭的意甲聯賽中腳踝受傷，他將無法出戰，這對於紫百合來講多少是有影響。往績上近4次交鋒紫百合保持不敗且取2勝，如此糟糕的上半程聯賽，冬歇期前守平拿分戰意可期。

羅馬：

近期表現不錯的同時保持6連勝，羅馬目前擠身第4名，已經身在歐戰資格區，未來就是打好每場比賽讓自己拿到下賽季歐戰資格，這點戰意上很是強烈。客場8場勝率50%，不敗率87%，僅面對與自己實力相近的帕爾馬客場輸球，羅馬防守能力還是不錯的，客場11球進帳失5球，鋒線實力不俗。

初盤平均值：2.82/3.33/2.31

初盤即時值：3.07/3.46/2.27

考慮到紫百合近期主場戰鬥雖然只取得了2勝3平3負，但看到主場的紫百合防守還是做得不錯，就算是輸球也僅是輸1球，而上輪主場平下國米，也算是提振了士氣，本場比賽如若丟球也不會丟大球，而羅馬本場比賽應該也不會放開來攻，畢竟不敗收關拿到積分才是重點，而其對手卡利亞里本輪坐客烏雞，勝負還不好說。競彩指數：讓球勝、讓球指數：佛羅倫薩+0.25。
