C羅永不服老！連續4場進球回擊一切質疑，喬治娜看台大秀愛的力量

2019-12-16 42條評論 1,495次閱讀 342人點贊

最近一段時間葡萄牙人C羅成為各大媒體的焦點人物，作為當今足壇巨星級人物，葡萄牙人本賽季在尤文圖斯的日子並不好過，與以往賽季超高的進球率相比本賽季C羅似乎掉進了魔咒，進球率下降、被主帥薩里提前換下場，即將年滿35歲的C羅被部分人質疑，有不少人說C羅隨著年齡增長狀態下滑是正常的，之前那個進球如麻的C羅或許不再回來！但永不服輸的C羅並沒有作出言語上的回擊，葡萄牙人用場上行動證明自己依舊年輕！自己用永不服老！

意甲第16輪尤文圖斯坐鎮主場迎戰烏迪內斯，上半場葡萄牙天王巨星C羅在第9分鐘、第36分鐘上演梅開二度的好戲，幫助斑馬軍團早早地建立勝利的優勢！值得一提的是算上本場比賽，C羅已經在最近4場比賽均有進球（含歐冠小組賽末輪對陣勒沃庫森打進一球），葡萄牙人經歷了在俱樂部的一段低迷期後重新復甦，連續4場比賽進球回擊之前所有的質疑聲，自律的C羅再次證明雖然自己即將年滿35歲，但在前鋒位置上自己依舊是世界級的水準！

本場比賽還出現了有趣的一幕，第36分鐘C羅打進本場比賽第2粒進球後，激動的葡萄牙人跑到了場邊與看台上的女友喬治娜進行了互動，看台上的喬治娜為C羅豎起了大拇指！喬治娜看台上大秀愛的力量！前段時間是C羅加盟尤文後最艱難的時期，葡萄牙人被媒體推到了風口浪尖之上，甚至有人呼籲希望C羅離開亞平寧，此時喬治娜很好的扮演了賢內助的角色，她一直與C羅站在一起，本場比賽C羅跑到場邊與看台上的喬治娜互動就是對兩人關係最好的詮釋！

眾所周知C羅還是曾經那個C羅，葡萄牙人的自律行為在整個足壇都被人稱讚，雖然年齡增長但他的身體素質幾乎還保持在二十幾歲，本賽季在尤文圖斯經歷過一段時間的低迷期，一大因素就是主帥薩里的戰術讓葡萄牙人感到很不適應，近期薩里在戰術布置上作出了適當的調整，C羅重新回到自己熟悉的鋒線位置後，葡萄牙人用一粒粒進球回報了薩里，也讓所有尤文球迷吃下了一顆定心丸，那個有如神助的C羅依然在！

連續進球的C羅將一切質疑聲化於無形，同時從最近幾場比賽來看，他與主帥薩里沒有絲毫問題，薩里也在積極地調整適合C羅的戰術體系，相信以目前的狀態葡萄牙人在本賽季能夠創造出更多的不可能！
