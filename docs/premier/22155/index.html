解簽 | 馬德里雙雄下籤？英超雙雄下籤

2019-12-19 2,667條評論 4,885次閱讀 1995人點贊

​歐冠1/8決賽抽籤的最震撼對決無疑是馬德里雙雄抽到了英超雙雄，具體來說就是皇馬抽到了曼城，而馬競抽到了利物浦，西班牙媒體認為馬德里雙雄抽到了下下籤，其實話並不能這麼講，只能感嘆英超雙雄的運氣不是那麼好。

目前，利物浦和曼城是被公認為世界上最好的兩支球隊，利物浦是上賽季的歐冠冠軍，而曼城聯賽中還能壓住不可一世的利物浦奪冠。而皇馬和馬競則是近些年來歐冠的傳統勢力，在去年皇馬還奪得了5年4歐冠和歐冠3連冠，而馬競就是他們曾2次在歐冠決賽中擊敗的對手。但在今天看來，皇馬和馬競還是要弱曼城和利物浦一點點。

上賽季，利物浦在萬達大都會球場奪得歐冠冠軍。

我們說馬德里雙雄並非抽到下下籤，而是他們的對手抽到下下籤的原因是，馬德里雙雄本就是小組第二，他們要面對的小組第一本就不可能是弱隊，當然，馬德里雙雄肯定都想抽到萊比錫紅牛，但這種上上籤也是可望不可求的，如果他們不抽到曼城或者利物浦，那皇馬也可能抽到拜仁、尤文或者萊比錫，而馬競則也要抽到拜仁、巴黎或者萊比錫。除了萊比錫以外，拜仁、尤文、巴黎這樣的強隊究竟比利物浦或者曼城弱多少很難講。

皇馬和曼城將在1/8決賽中相遇。

但對於利物浦和曼城來說，他們肯定會感到自己在小組賽中費盡千辛萬苦拿下的第一這次是白拿了，他們肯定更想抽到多特蒙德、里昂、那不勒斯（利物浦無法抽到因為同小組）、亞特蘭大（曼城無法抽到因為同小組）這樣的球隊。

瓜迪奧拉沒有任何理由對抽到皇馬感到興奮，他在帶拜仁3個賽季衝擊歐冠冠軍時便是被西甲三強一一擋在決賽外。我們甚至驚奇地發現，在皇馬5年奪4歐冠的最後2次奪冠道路上，皇馬也都不是以小組第一的成績晉級淘汰賽的。但讓瓜迪奧拉可以鬆口氣的是曼城目前有的問題，皇馬也有，那就是兩支球隊都可以踢得不錯，但攻擊線的進球效率都不高，貌似臨場比賽時哪隊的前鋒更有靈感運氣更好誰就更有出線的機會。

而克洛普可能抽到了他最不想面對的對手，因為馬競是一支著名的「五五開」球隊，這支球隊幾乎在淘汰賽中面對任何對手都有五成的機會能出線。如果克洛普的利物浦是一支以善打硬仗而聞名的球隊，馬競同樣也是，兩支球隊是歐冠十六強中作風最頑強的球隊。而且今年的利物浦似乎已經從以防守穩固為特徵變為進攻摧枯拉朽的球隊，這不正合適馬競這支防守反擊專家球隊的口味嗎？當然利物浦擁有的優勢正是馬競所欠缺的，那就是馬內和薩拉赫的進球效率，馬競今年最大的問題就是踢的哪怕美如畫就是不進球，從這點上看，即便馬競還是支五五開的球隊，對利物浦也是處於明顯的劣勢。

克洛普可能抽到了他最不想面對的對手，因為馬競是一支著名的「五五開」球隊。

本次歐冠16強的最大特點便在於16支球隊全部來自五大聯賽，英超和西甲的4支球隊全部晉級。不過幸好，巴薩抽到了那不勒斯而不是熱刺或者切爾西，而巴倫西亞抽到了亞特蘭大而不是熱刺，最終16強大戰沒有直接演變為西甲大戰英超的比賽。巴薩沒有抽到英超球隊，而是抽到那不勒斯這是個可以接受的結果，當然如果能抽到亞特蘭大這樣的歐冠新軍就更好了，但這種上上籤同樣是可望不可求的事。畢竟巴薩在小組賽已經被分在死亡之組，淘汰賽首輪沒有抽到死亡對手從概率上看也是符合邏輯的。而巴倫西亞則抽到了亞特蘭大，歐冠16強分組總會有這樣實力對比差別不大的對決，而這場就是，巴倫西亞沒什麼可抱怨的。

歐冠16強分組中實力對比差別不大的對決將在巴倫西亞和亞特蘭大之間展開。

歐冠16強的抽籤我們完全可以看作是英超新勢力對歐陸老勢力的衝擊，看一下歐足聯的俱樂部積分，前4名被西甲三強外加拜仁包攬，這與目前利物浦和曼城打出最驚艷的足球不相符，英超積分最高的曼城僅排名第6，而衛冕冠軍連續2年打進決賽的利物浦只排在第8位，現在就讓我們看看英超的曼城、利物浦和切爾西能否衝破昔日豪門們把持的尊貴堡壘吧。
