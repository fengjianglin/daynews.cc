博努奇：尤文很多丟球是個人失誤，擊敗藍鷹需完美表現

天天要聞
2019-12-22

虎撲12月21日訊 尤文圖斯後衛博努奇提醒球隊，他們需要拿出完美的表現才能在義大利超級盃擊敗拉齊奧，「我們希望在這個聖誕節再奪一個獎盃」。

作為上賽季意甲和義大利杯的冠軍，北京時間12月23日凌晨00:45，斑馬軍團將在利雅得與拉齊奧競爭義大利超級盃的冠軍。本賽季尤文僅僅輸過一場球，就是12月7日做客1-3不敵藍鷹。

在賽前新聞發布會上，博努奇說道：「那場比賽教會了我們，一些小插曲可能會成為決定性的。我們很清楚拉齊奧的狀態正佳，但我們也取得了積極的表現和結果，所以我們也很有信心。」

「我們最初的目標是踢得比上一次對陣拉齊奧要好，上次交鋒我們給了他們太多的得分機會。在這裡獲勝將會增加我們的熱情和信心，我們希望在聖誕假期的壁爐台上再放上一個獎盃。」

因莫比萊在意甲16場比賽中打進17球，在歐聯杯也出場4次攻入2球。

「他攻入了很多球，為球隊工作，我們很了解他，大家都是義大利國家隊隊友，但是拉齊奧不僅僅有因莫比萊，他們踢出了自己的身份，我們需要一支完美的尤文圖斯才能擊敗他們。」

「我們必須作為一個整體去移動，限制住米林科維奇和因莫比萊的跑位。在薩里的麾下我們改變了比賽方式，雖然我們還沒有完全達到要求，但是我們已經做得很好。」

「讓我們惱火的是，我們本賽季丟掉了很多球，很多都是由於個人失誤，其他的則是這套新踢法形成過程中所要經歷的。我們知道我們必須繼續改進。」

(編輯：天藍色男孩)
