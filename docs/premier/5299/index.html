巴薩眾將談安菲爾德奇蹟，特獅：被逆轉後我只想立刻消失

2019-11-29 0條評論 14次閱讀 0人點贊

直播吧11月29日訊 據《世界體育報》報道，近日巴薩眾多球員以及主教練巴爾韋德，在紀錄片「比賽日」中談到了被利物浦以及瓦倫西亞擊敗的事情。

巴薩本次播出的紀錄片第7章被命名為「暴風雨18天」，而主要講述的就是球隊上賽季在18天之內連續被利物浦以及瓦倫西亞擊敗所發生的事情，而這兩場比賽也讓他們丟掉了兩個冠軍的爭奪權。

在球隊0-4輸給利物浦無緣歐冠決賽後，門將特爾施特根談到了本場比賽：「利物浦想在主場一開始就一鼓作氣的逆轉我們，而我們也希望他們的這口氣能停下來，但這是不容易的。我們沒能到找回巴薩應該表現出來的樣子。」

皮克表示：「在球場中，當利物浦一開場後就取得了進球時我們很困難，羅馬那場的影子出現了。」

而羅貝托表示：「利物浦開場就進攻又進攻，帶著高漲的情緒和烏雲密布般的進攻，而我們則是顯得情緒低落。我一直都有在關注計分牌，但是我不敢相信眼前發生的事情。」

而巴薩內部人員表示：「這一場的失利也對隊長梅西造成了很大的打擊，因為他一直記得自己曾在賽季初表示要將所有的冠軍帶給球迷，因此失利也讓他比任何人都傷心。」

在輸給利物浦後，特爾施特根和布斯克茨代表巴薩球員留下並感謝來到安菲爾德的巴薩球迷們。對此，特獅表示：「踢出這樣的比賽後，我想做的第一件事情就是消失。這(結果)聽起來很糟糕，球迷們遠道而來支持我們，而我們的共同目標就是進入歐冠決賽。我只想感謝球迷們的鼓勵，並為今天的比賽結果道歉。」

而布斯克茨表示：「這是一個非常苦澀的失利，一個非常悲傷的夜晚，因為它又發生了一次。我們是巴薩，所有球迷以及我們自己對於比賽都有很高的期望。」

而巴薩主教練巴爾韋德也談到了這場失利：「我總是在想，我能做什麼和改變什麼。我有一種感覺，如果我在某件事情上錯了，那麼我很可能再錯一次。所以在經過了羅馬的遭遇後，我反思了很多次，並根據那場比賽的經驗做了很多準備。」

「當安菲爾德的事情發生後，你所要做的就是和家人在一起。忘記這些事，並試圖把一切不可思議的失敗都合理化，找出他們的原因。」

當巴薩在國王杯決賽半場0-2落後時，梅西開始鼓勵隊友振作起來：「帶著雙冠王離開和輸掉決賽是不一樣的，不要忘記贏下這場比賽我們就有雙冠王了。雖然現在局面不利，但是我們不必泄氣。」

賽後，布斯克茨也回憶起了連續輸給利物浦和瓦倫的比賽，他表示：「連續錯失奪冠機會讓人很消極，但你必須接受它，保持冷靜。不要被這些事情打垮，從中學習並成長。」而皮克則表示：「即使目前我們很困難，但我們也必須著眼於新賽季和未來，我們的團隊和俱樂部依然是最頂級的。我們要對未來保持樂觀的態度，因為這支球隊已經證明了他能夠奪冠軍，而我們依然是歐洲最頂級的俱樂部之一。」

最後，梅西回憶道：「為巴薩這樣偉大的俱樂部效力，你必須不惜一切代價的去爭奪所有冠軍。我們的團隊有能力做到這一點，所以我們也將再次實現它(三冠王)。」
