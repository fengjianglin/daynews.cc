羅納爾多受傷導致巴西隊表現不佳，巴西足協在兩年內更換四名主帥

天天要聞
2019-11-13

如今的球迷們也許只知到羅納爾多在韓日世界盃上大放異彩，卻很少有人知道他其實差點錯過那屆世界盃。至少在01～02賽季開始的時候，大部分人不看好羅納爾多能夠參加世界盃，畢竟他的傷病不讓人樂觀。隨著新賽季的臨近，各俱樂部季前熱身比賽的質量也有所提升，當時的羅納爾多還沒有完全恢復。就算他完全恢復的話，算上找回狀態的時間，參加世界盃真的很不樂觀。巴西時間2019年11月12日，巴西體育頻道回顧了羅納爾多在那屆世界盃之前的「奇蹟恢復」。在梅阿查的熱身賽上，國際米蘭與來訪的奈及利亞球冠軍恩耶巴隊進行了一場比賽，藍黑軍團以7-0大勝對手，羅納爾多也取得了進球，賽後他對記者說：「我的膝蓋已經完全康復了。」

這句話被大肆報道了，畢竟這無論對於國際米蘭的新賽季，還是羅納爾多參加世界盃的夢想來說，都是一個很好的消息。當天有四萬名球迷湧入梅阿查，他們又看到了自己久違的球星。熱身賽是一回事，而聯賽是另一回事，顯然二者的水平有很大的差距，所以羅納爾多想要在聯賽中找到狀態，還需要一個過程。當時的「外星人」是多麼希望能夠參加02年的韓日世界盃，他需要在俱樂部新教練庫珀帳下發揮出水平，這是重新取得教練信任的過程，也需要花費時間這個阿根廷人取代了里皮成為了球隊的新任教練，因為上賽季藍黑軍團只取得了聯賽第五名的成績。

庫珀這位教練成名於馬洛卡，他在99年曾帶領球隊殺入歐洲優勝者杯杯決賽，不久之後他轉投瓦倫西亞，連續兩年帶領球隊殺入歐冠決賽——這在當時看來已經是奇蹟了。但是在國米，就像剛開始的里皮一樣，他與羅納爾多保持著距離.他們的關係一度比較緊張，這個羅納爾多的球迷們都是知道的。庫珀決心要在藍黑軍團完成一番大事業，自1989年以來，俱樂部還未拿過聯賽冠軍——這是有些尷尬的紀錄。不過當時陣中有許多球星，比如說薩內蒂、維埃里、雷科巴、托爾多等等，這一次他們要向聯賽以及歐冠冠軍發起衝擊。

10天之後在與巴斯蒂亞的一場友誼賽中，最後二十分羅納爾多替補登場，打進兩球，幫助球隊4-2獲勝，他沒有放過任何一次進球的機會。在巴西國家隊，羅納爾多的受傷也在很大程度上影響了球隊成績，在韓日世界盃南美區預選賽階段，巴西隊表現的不盡人意。因為羅納爾多受傷導致巴西隊表現不佳，巴西足協在兩年內更換了四名主教練，後來就是知名的冠軍教頭斯科拉里被任命為國家隊主教練，也就是帶領巴西隊拿到韓日世界盃冠軍的那位主帥。在與烏拉圭比賽前，大菲爾召集全體球員開會，來了解他們每一個人，他提出了自己的想法，充分與隊員們交流。當時受傷的羅納爾多也在被邀請之列，但那個時候國家隊隊內關係有些複雜，一部分人並不希望給羅納爾多機會。
