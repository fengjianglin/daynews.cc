克洛普慌不慌？利物浦最大的苦主，安切洛蒂上門討債來了

2019-12-25 83條評論 4,818次閱讀 3221人點贊

一說起安切洛蒂，你最先想到的是什麼？

不說100%，起碼99%的球迷都會想到一個地方，一個夜晚，一場比賽——伊斯坦布爾之夜。

安切洛蒂的一世英名，都在那個夜晚、那場比賽之後給毀於一旦。甚至因為這場比賽，安切洛蒂此前本已被遺忘的、已被成功蓋過的失敗一起被翻了出來，然後一起毀於一旦。

似乎安切洛蒂留給世人的一切，全部都濃縮在了2005年5月25日這一個夜晚。「被奉獻」了歐冠史上最經典的決賽和最神奇的大逆轉之後，安切洛蒂和那支風華絕代的米蘭永久的被釘在了足球的恥辱柱上。

與其說安切洛蒂敗給了傑拉德，不如說是敗給了命。那是安切洛蒂帶隊踢得最好的一場歐冠決賽，配合如行雲流水，攻勢似水銀瀉地，即便利物浦在6分鐘內連進3球，但場面上更有機會取得勝利的，還是米蘭。

但最終輸掉比賽，丟掉冠軍的卻是米蘭。

從此，「伊斯坦布爾之夜」就與安切洛蒂形影不離，安切洛蒂和他那支風華正茂的米蘭就此成為各類TOP10中當之無愧的打榜冠軍……冠軍的背景。

這是一種詛咒，一種伴隨一生的詛咒。

2007年5月23日，安切洛蒂帶領米蘭在雅典歐冠決賽中擊敗了利物浦，在復仇成功之後重回歐洲之巔。手刃仇敵的快感然讓意氣風發的安切洛蒂以為自己終於可以擺脫「伊斯坦布爾之夜」的折磨，然而殘酷的事實卻是：無論時間過了多久，2005年5月25日那一夜在伊斯坦布爾所發生的一切，已經成為了永恆的經典，但對於他，這卻是他執教生涯永遠抹不掉的恥辱。

——哪怕他在兩年之內就完成了自我救贖，哪怕他在當時已經無人匹及。

這樣的境遇，在他帶領切爾西成為英格蘭雙冠王時沒有改變；在他帶領大巴黎稱霸法甲時沒有改變；在他帶領皇馬重奪歐冠冠軍時也沒有改變……

一代名帥，三座聖勃萊德杯在手，當今足壇帶領兩支不同球隊獨取歐冠獎盃的唯二，依舊還是擺脫不了14年前的那場驚天大逆轉的陰影。安切洛蒂這一輩子，就交代在那個神奇的夜晚了，交代給利物浦了。

從拜仁這個有著「綠茵好萊塢」之稱的豪門黯然下課後，安切洛蒂漸漸被頂級豪門所拋棄，他只能來到在意甲都只能算作二流的那不勒斯。

但命運就是如此作弄，安切洛蒂執教的那不勒斯，兩次歐冠小組賽都和克洛普執教的利物浦「相聚」在了一起。

要說安切洛蒂有本事，那還真不是瞎吹；但要說他衰，那還真沒冤枉他。與克洛普的利物浦兩年小組4戰，2勝1平1負佔據上風，但偏偏輸掉的那場，就是上賽季小組賽最後一輪被利物浦踢出局的那一場。

強調一點：利物浦最終憑藉這場比賽的勝利小組出線，並最終上演了不亞於2005年歐冠奪冠的神奇之旅……

再強調一點：利物浦在安菲爾德上演的4球大逆轉，又再一次把安切洛蒂和他的米蘭從「睡夢」中無情地「扎」醒……

若說這世間沒有宿命的安排，誰信？

從那不勒斯下課後，安切洛蒂很快就找到了新東家，成為利物浦城市中另一支球隊——埃弗頓隊的新帥。這倒是正應了那句——不是冤家不聚頭的老話，而埃弗頓與利物浦原本就是水火不容的冤家。遠的不說，就說上賽季埃弗頓球員在逼平利物浦之後那不亞於奪冠的狂歡慶祝，這默西塞德德比中所包含的仇與恨僅從這一個畫面就能略知一二。

現在好了，一個對利物浦有著切膚之痛的主教練，一支對利物浦有著深仇大恨的俱樂部，原本根本不可能有交集的主帥和球隊，就這樣走到了一起。

看看安切洛蒂的執教履歷就明白，自從度過伊斯坦布爾那個糟糕的夜晚後，他對陣利物浦的戰績可是11戰7勝1平3負，即便算上對陣克洛普時的2勝1平1負，安切洛蒂也是佔據上風。

安切洛蒂對於帶隊奪取冠軍充滿信心，豪言將為奪取本賽季足總杯冠軍而戰，甚至對擊敗利物浦充滿信心，他甚至表示該擔心的是利物浦，——因為自己曾經擊敗過他們很多次。

的確，他擊敗過利物浦很多次，但媒體和球迷能記住的卻偏偏是他輸掉的第一次。
