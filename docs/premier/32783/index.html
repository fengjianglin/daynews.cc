2戰尤文造3球！藍鷹真核成歐洲助攻王，卻只為國家隊登場過16分鐘

2019-12-24 522條評論 4,109次閱讀 535人點贊

在沙特利雅得舉行的義大利超級盃，最終憑藉著阿爾貝托、盧利奇以及卡達迪的進球，拉齊奧3-1完勝尤文圖斯，隊史第5次加冕意超杯冠軍。

值得一提的是，短短15天前的意甲交鋒中，「藍鷹」同樣以3-1的比分力克對手。而這2場比賽中，一名球員奉獻的1粒進球和2次助攻，顯然使他成為球隊的最大功臣——他就是路易斯-阿爾貝托。這名西班牙中場以其優異的發揮，完全遮蓋住了C羅和迪巴拉這兩大天王的星光。

那麼，這名與巴西傳奇隊長重名的拉齊奧核心，究竟是何許人也呢？

球員簡介

● 路易斯-阿爾貝托（西班牙）

阿爾貝託身高182cm，體重70kg，場上司職前腰或者影鋒的同時，也可客串翼衛位置。

他有著西班牙球員標誌性的腳下技術以及向前場輸送威脅球的能力。目前在拉齊奧身披「10號」球衣的他，是球隊當之無愧的場上「發動機」。

儘管如此，這名西班牙中場在力量和身體對抗方面，依舊有些可以提升的空間。

曾經的紅軍棄將，如今的「老婦人」殺手

路易斯-阿爾貝托1992年出生於西班牙安達盧西亞大區的聖何塞斯德瓦里市。自幼便加入了塞維利亞青訓體系的他，還曾被短暫租借到巴塞羅那B隊效力。

2013年夏窗，英超豪門利物浦以800萬歐的價格將其買斷，阿爾貝托也得以順利「參軍」，並隨球隊經歷了那個光彩奪目的13-14賽季。

但儘管曾被當作中場新星培養，但在之後的幾年裡，阿爾貝托卻並沒有在「紅軍」陣中獲得足夠的出場時間，只被接連租借到了馬拉加和拉科魯尼亞等西甲中下游球隊效力。

2016年8月，拉齊奧以400萬歐元的價格將其收入隊中，如此價格甚至只有利物浦當年花費的一半。但在小因扎吉麾下，阿爾貝托卻逐漸展現出了自己的潛力。

17-18賽季開始，阿爾貝托成為了球隊的中場核心。在小因扎吉的「3-5-2」體系中，他與另一位當紅「炸子雞」米林科維奇-薩維奇在三中場的搭配中一左一右，共同支撐起了「藍鷹」的中場，中間則由另一位利物浦棄將盧卡斯領銜。

3年的時間裡，阿爾貝托總計為「藍鷹」登場114次，打入22球的同時，還送出了驚人的39次助攻，幫助球隊贏得了1座義大利杯和2座義大利超級盃，並以14次助攻加冕了17-18賽季的「意甲助攻王」。

不僅如此，在短短15天的時間裡，拉齊奧先後以兩個3-1將尤文圖斯予以擊潰，這也是「斑馬軍團」本賽季唯二的兩場失利。在這兩場比賽中，阿爾貝托奉獻了1粒進球和2次助攻的瘋狂數據，是球隊當之無愧的「大場面先生」。

值得一提的是，職業生涯8次與尤文圖斯的交手中，阿爾貝托總計獲勝4次，並送出了1球3助攻的體面數據，而「老婦人」也是他最喜愛的對手。

「藍鷹」生涯大放異彩，卻僅為國家隊登場16分鐘

比較遺憾的是，由於近些年西班牙球員人才井噴，目前阿爾貝托僅為國家隊登場過1次，還是在一場與哥斯大黎加無關緊要的友誼賽中完成的，時間也僅有16分鐘。因此，在目前的「板鴨軍團」中，阿爾貝托恐怕連邊緣國腳都難以算上。

儘管如此，以西班牙目前中場的人員配置，主帥恩里克一定會面對許多「幸福的煩惱」。暫且不說蒂亞戈、布斯克茨、薩烏爾和羅德里等豪門球員，就連那不勒斯核心法比安-路易斯的順位，恐怕也在他之前。

「參加2020年歐洲杯是我的目標。」阿爾貝托如實說。

入選國家隊雖然並非遙不可及，但考慮到他27歲的年齡，想要順利搭上歐洲杯的末班車，阿爾貝托恐怕還需要繼續拿出具有說服力的發揮和數據。

「數」說球員

根據著名數據網站《轉會市場》的統計，阿爾貝托目前的身價為4000萬歐元，在拉齊奧隊內僅次於7000萬歐的米林科維奇和4500萬歐的因莫比萊。

此外，貴為17-18賽季的意甲助攻王的他，也在本賽季保持了出色的傳球腳感。其中，阿爾貝托在意甲奉獻的11次助攻，更是使他獨步五大聯賽助攻榜，與托馬斯-穆勒並列。

結語

本賽季的積分榜，拉齊奧在少賽一場的情況下僅與榜首的國米相差6分，意甲就在這樣不聲不響的情況下，形成了三足鼎立的焦灼局面。

推翻尤文圖斯的意甲8冠統治，這名「斑馬殺手」和他領銜的拉齊奧隊，或許已在路上。
