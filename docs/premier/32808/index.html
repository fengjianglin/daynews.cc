熱刺為何穆里尼奧執教後成績不佳，看了分析，豁然開朗

2019-12-24 2,059條評論 3,178次閱讀 837人點贊

破案了！熱刺為何穆里尼奧執教後成績不佳，看了分析，豁然開朗。網友熱議一針見血地指出：從穆里尼奧傷人初始，我第一時間就說明了熱刺這賽季的攤子相當不好，原因核心自然是埃里克森的問題。埃里克森對熱刺究竟有多關鍵，看看身價就一目了然。從球員特點來看，可以說身價的90%都集中在傳球端，而幾年以來熱刺的整個中場配置也始終為了突出埃里克森的傳球最大化。互補起來就是，其他球員傳球能力簡直慘不忍睹，但跑動和保護核心能力可圈可點。作為另一個具有同樣標本化的球隊就是，前幾個賽季的國米。

你把布羅佐維奇拿掉，也簡直不可看。所以，在沒有埃里克森的情況下，為了最大程度發揮熱刺前鋒線速度優勢，自然增加了長傳。 成功率當然低，場面也難看，這就是熱刺球員的現狀。指望中場球員通過傳控去創造機會，不僅沒有能力，反而會極大壓縮前鋒活動空間，所有優勢全被抹滅。孫有速度，凱恩有終結，阿里有插上進攻，但這幾個特點都得依靠埃里克森的頂級傳球才能發揮最大化。從自主開發進攻能力方面，馬內菲爾米諾這些球員就顯然要強太多。別扯前幾個賽季什麼進攻如何，埃里克森一撂攤子，吹上天的波切蒂諾簡直秒跪。埃里克森僅僅是這賽季的問題，倒也不是什麼大事，核心問題在於列維究竟願不願意為了爭冠付出。

國米和熱刺兩傢俱樂部管理層有不少相像的地方，都是嘴面上很能逼逼，實際行動端都做得一般，當然也不能說很差。現在球隊中場無限被強隊絞殺反擊，放大了後場防守垃圾的問題。當然，那門將是真的差，昨天威廉那腳抽射沒撲到就算了，無影腳直接把我看懵逼了，大晚上熬夜，我甚至懷疑我是不是出現幻覺了。前場沒有中場的支持，拿不到球，拿到球被空間壓縮得厲害，讓孫興民，阿里和凱恩踢得很不舒服，最近感覺狀態也不好了。波切蒂諾離開熱刺的原因從我個人以為以下幾點：理念和列維不和，波叔想拿冠軍，想砸錢，列維堅持可持續發展。對教練來說巔峰期就那麼10幾年，想投錢爭取更大爭奪冠軍可能無可非議。波叔堅持認為球隊需要重建，列維認為球隊已經拿了歐冠亞軍，修修補補就有希望爭冠，拿你了冠軍你只要穩前四我也不怪你。

正是這個分歧使得二人感情上決裂了，這很容易理解，最好的朋友吵架多了早晚決裂！目前從這幾場比賽來看，波叔的眼光是基本靠譜的，現在後腰、中後衛 邊後衛都有問題，中後衛鳥叔勸說托比，爭取續約，一方面只是延後重建的時間 ，另一方面托比即使續約狀態也早就開始下降了。後腰和二個邊位的問題，更嚴重和急迫。其實恩東貝萊就是重建的後腰之一，主要看中的就是帶球推進和攔截，羅塞爾所看中的就是調度和傳球，波還是按照登貝萊和理科生的模版來重建的。這沒什麼嚇人不嚇人的，重建不可能一個窗口完成，下個窗口估計就是中後衛和右邊路，左邊路可能會最後解決。

破案了！熱刺為何穆里尼奧執教後成績不佳，看了分析，豁然開朗。沒有了埃里克森得熱刺不可能採取中路打法，所以只能抱著邊打，更悲劇的是客串打左後衛得維爾同亨運動戰基本沒有什麼進攻屬性，這時只要右後衛被壓制基本上就只能依靠長傳。左右現在熱刺的對手基本抱著左路奧里耶的位置和中路弧頂的位置打。這種局面不知恩東貝萊完全康復之後能否得到改善，與其相信很多球友說的指望列維發錢買核心不如希望穆帥把恩東貝萊或者洛索用好。
