德甲：多特蒙德vs萊比錫解析，本場數據指向清晰，乘勝追擊好時機

2019-12-17 535條評論 1,490次閱讀 173人點贊

回顧：意甲卡利亞里vs拉齊奧不負眾望，命中比分1-2。

多特蒙德本賽季德甲聯賽8勝5平2負，共打入了37粒，失球19粒。向來有魔鬼主場之稱的他們在新賽季依然出色，7個主場保持不敗且場均進球3粒以上，僅失6球。自從變陣343後，多特蒙德上輪5-0大勝美因茨，在聯賽中也是獲得了3連勝，近兩輪更是大比分且零封戰勝對手，球隊彷彿終於進入了正軌，找回了以往的穩定性與連貫性。此役對陣萊比錫，既是法夫爾新陣試驗的嚴格考量，也是德甲秋季冠軍的風向標之戰。

萊比錫本賽季德甲聯賽10勝3平2負，目前以33分佔據榜首，近期狀態極佳，在聯賽中已經連續6場全勝，其進攻端非常強悍，近40場各項賽事只有3場是交白卷，聯賽至今共打入了42粒進球，隊中最佳射手維爾納的個人進球數也達到了16粒進球，僅次於拜仁的萊萬多斯基。此役以客場迎戰多特蒙德，主客因素上對萊比錫的影響並不大，因為其主客場都很穩定，客場的表現更是聯賽中最佳的球隊。

指數解析：

相對這個賽季兩隊的表現來看，實力是相差無幾的，但在穩定性方面，萊比錫要更佔上風。不過初始遊戲指數給出了多特蒙德-0/0.5的讓步，還是要更傾向於主隊，相對本場而言，這樣的讓步也是對入手主隊加大了難度，亦可理解是阻礙，顯然機構並不看好萊比錫能從多特蒙德身上取下3分。綜合分析，本場看好多特蒙德不敗，建議：多特蒙德-0/0.5（小防和）；競彩：主勝（防平）：比分2-1，3-2。如果覺得分析的合理，請點贊加關注，參與下方討論各自的見解，萬分感謝。
