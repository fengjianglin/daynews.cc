歐冠末輪出線形勢 國米切爾西必須球贏 G組形勢最複雜

2019-12-10 2,282條評論 3,227次閱讀 1742人點贊

12月10日和11日，歐冠小組賽最後一輪將陸續開打。歐冠目前歐冠已經出線的球隊：曼城、熱刺、拜仁、萊比錫、尤文、巴薩、皇馬、巴黎聖日耳曼。各小組具體如下：

A組：巴黎聖日耳曼已經鎖定小組第一，皇馬第二，布魯日和加拉塔爭奪歐聯杯席位。

B組：拜仁和熱刺已經鎖定小組前兩名，貝爾格萊德紅星和奧林匹亞科斯爭奪歐聯杯席位。

C組：曼城已經出線，並鎖定小組頭名，礦工主場戰勝亞特蘭大就出線；亞特蘭大客場贏球，而且薩格勒不勝，亞特蘭大就可出線；薩格勒贏曼城，而且礦工不勝亞特蘭大，薩格勒布迪納摩就可出線。

D組：尤文圖斯已經出線並鎖定小組第一。馬競贏球就出線，勒沃庫森取勝，馬競不勝，勒沃庫森出線；勒沃庫森不勝，馬競輸球也出線。莫斯科火車頭取勝也排在倒數第一。

E組：利物浦客場平局出線，贏球或者兩場都平局就以小組第一出線，輸球很危險，利物浦主場4-3戰勝薩爾茨堡，主場丟球太多。那不勒斯不敗出線；薩爾茨堡小勝或者大勝都可出線，5-4以上小勝出局；亨克排名已經出局。

F組：巴塞羅那已經出線，並鎖定小組第一。國米與多特同分但戰績佔優，國米贏球鐵定出線。多特蒙德取勝，國米不勝就出線。布拉格斯拉維亞已經出局。

G組：萊比錫已經出線，平局就可鎖定小組第一。里昂贏球出線，里昂與澤尼特同時戰平，澤尼特出線。里昂平，澤尼特輸，里昂出線。里昂負，澤尼特勝平負都出線。本菲卡取勝也只有競爭歐聯杯資格。

H組：切爾西主場戰勝里爾出線，不勝巴倫西平局就可出線；阿賈克斯平局就可出線，取勝小組第一。阿賈克斯輸球、切爾西贏球，阿賈克斯將被淘汰，巴倫西亞和切爾西晉級。
