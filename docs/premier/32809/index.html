國際冠軍杯中馬競7-3皇馬竟然攪黃了貝爾加盟江蘇蘇寧的轉會

2019-12-24 1,017條評論 3,344次閱讀 2161人點贊

2019賽季夏窗轉會期間，對於中國足壇來說，最大的流量和討論莫過於孫大聖貝爾要加盟江蘇蘇寧的消息了。 記得那個時候《足球報》曾經報道，由於貝爾和皇馬教練、球迷之間的摩擦不斷，導致貝爾將結束在皇馬的生涯，將會前往中超，而江蘇蘇寧和北京國安將對其展開競爭。而最終似乎江蘇蘇寧在競爭中勝出，且貝爾加盟江蘇蘇寧也基本成定局，但是結果卻是大聖沒到江蘇蘇寧，反而來了桑蒂尼和米蘭達。

在那時，有關貝爾為什麼沒有加盟江蘇蘇寧的傳聞或者報道多的不得了，有說因為江蘇蘇寧不願意向皇馬支付貝爾的轉會費，以至於談判陷入停滯；也有說江蘇蘇寧由於和貝爾在待遇以及其他一些條件上存在巨大分歧，從而導致這樁轉會告吹；也有說貝爾加盟江蘇蘇寧則是一個曲線救國的線路，簽約江蘇蘇寧然後去國際米蘭或者歐洲球隊等等，甚至於當時白國華還發微博就此事進行道歉，可謂沸沸揚揚。總之一句話貝爾沒有來江蘇蘇寧，為什麼沒有來鬼才知道。

不過就在人們漸漸淡忘這件事情之後，江蘇蘇寧的主教練奧拉羅尤在參加一檔名為「TelekomSport TV」的節目時，熊大透露了貝爾轉會失敗的原因。而這個原因竟然是一場比賽所引發的血案。由於當時皇馬在國際冠軍杯中以3-7大比分輸給了馬競，從而導致皇馬主席要求貝爾留下來。同時熊大也透露了那時後所有談判都已經完成，就差貝爾的合同了，且對於薪水雙方也達成了一致，不過對於大額轉會費，蘇寧沒法接受。

因此貝爾未能加盟江蘇蘇寧，一來是皇馬大比分輸了比賽，皇馬主席覺得球隊進攻不利，所以希望貝爾能夠承擔起進攻端的重任，二來還是媒體所猜測的那樣，還是轉會費太高所致。

所以怎麼說呢，貝爾未能順利加盟江蘇蘇寧，就怪那場比賽，真是一場比賽引發大聖有家不能回。甚至有球迷開玩笑道，都怪本澤馬，沒能進8個球，導致貝爾回不了花果山。
