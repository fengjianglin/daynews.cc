不願再坐板凳！32歲老將離隊，床單軍團進攻無力，願為其騰出空間

天天要聞
2019-12-24

今年32歲的卡瓦尼目前效力在法甲豪門巴黎聖日耳曼，卡瓦尼在年少時就展現出了非常高的足球天賦，達努比奧俱樂部青訓營的主帥迅速地將其給簽下了。

登陸五大聯賽後，卡瓦尼在意甲勁旅那不勒斯效力期間表現出色，高光的表現迅速征服了不少的球迷，卡瓦尼在那不勒斯效力期間不僅幫助球隊拿到了義大利杯賽冠軍，且還收穫了意甲金靴獎。

卡瓦尼的技術特點不比他人，卡瓦尼球風相對來說比較清麗，也可以說是簡單粗暴，烏拉圭人的跑位和門前嗅覺處於世界級，隊友總能在禁區內看到卡瓦尼的身影，值得注意的是，就是這位烏拉圭人在意甲效力期間其身價是登頂過五大聯賽。

卡瓦尼童年時期還有一段趣事，卡瓦尼的家境並不富裕，但是卡瓦尼的父親非常支持自己的孩子追逐夢想，卡瓦尼的父親承諾他進一個球就給他買一個冰淇淋，結果那天受到激勵的卡瓦尼一下子打進了7個進球，父親也遵守諾言給卡瓦尼買了7個冰淇淋。

卡瓦尼於2013年夏窗正式加盟巴黎聖日耳曼，其轉會身價高達6400萬歐元，烏拉圭人也因此成為了2013年夏窗的標王，加盟到大巴黎後卡瓦尼的狀態依然在線，但是當時隊內擁有伊布拉希莫維奇的存在，烏拉圭人一直在伊布的陰影之下。

直到伊布轉會離開，卡瓦尼成為了大巴黎隊內的鋒線頭牌，火力全開的卡瓦尼在那個賽季一共幫助球隊打入了49粒進球，也成功的奪得了法甲聯賽的金靴獎。

頭牌還沒當夠，巴西巨星內馬爾突然加盟，俱樂部視內馬爾為絕對核心，卡瓦尼心中自生不快，由於當時的主帥埃梅里也並沒有明確聲明內馬爾是點球的主罰手，在大巴黎獲得一個罰點球的機會時，兩人是互不相讓爭罰點球。

隨後內馬爾的核心地位逐漸確立，加上同時剛加盟的小老弟姆巴佩迅速崛起，且姆巴佩與內馬爾的關係極為親近，卡瓦尼再一次失去了存在感。

本賽季由於卡瓦尼有傷在身，巴黎聖日曼在今年夏窗引進了國米中鋒伊卡爾迪，伊卡爾迪在法甲聯賽的表現非常出色，傷愈歸來後的卡瓦尼狀態依舊低迷，在中鋒的位置上，主帥圖赫爾的首選顯然不是烏拉圭人，而是狀態火熱的伊卡爾迪。

本賽季卡瓦尼在聯賽中僅出場8次，其只有4次是首發，場均出場時間也僅只有41分鐘，可見卡瓦尼已經不再被大巴黎需要。

西甲勁旅馬德里競技本賽季的防守依然出色，但是隊內的前場核心格列茲曼投敵巴薩離去，主力前鋒科斯塔又還在傷病之中，床單軍團的進攻屬實無力，西蒙尼希望在冬窗引入一名鋒線大將，卡瓦尼正是西蒙尼心中合適的人選。

馬競願意通過出售勒馬爾來為其騰出薪資空間，卡瓦尼也不願一直在大巴黎坐冷板凳，大巴黎這邊也不想與球隊的功臣鬧僵願意放人，這筆交易很可能達成。
