本周部分英超球隊將迎來聯賽杯的比賽，德甲則是一周雙賽，本期足彩包括德甲1場，德乙2場，意甲1場，西甲1場，法乙9場，開賽時間從12月21日1點30分，到21日凌晨4點。

先來看一場德甲聯賽，霍芬海姆（24分）在主場迎戰多特蒙德（30分），在18日結束的聯賽中，霍芬海姆客場2-0擊敗柏林聯合，多特蒙德則是在主場3-3與萊比錫紅牛握手言和。本賽季，霍芬海姆在主場的表現比較極端，3勝1平4負，本場首選主勝搏冷，次複選。

一場荷乙聯賽，阿賈克斯青年隊迎戰奧斯，阿賈克斯青年隊39分排名第二，奧斯倒數第三，兩隊實力目測差距很大，首選主勝。
