歐冠八分之一決賽抽籤解析：拜仁迎來複仇之戰 尤文喜獲最佳簽

2019-12-18 993條評論 1,970次閱讀 500人點贊

歐冠淘汰賽抽籤塵埃落定，16支晉級球隊各家歡喜各家愁。馬競迎戰衛冕冠軍利物浦，多特蒙德死亡小組逃生之後又迎來兵強馬壯的巴黎，熱刺迎來本賽季德甲狀態最好的球隊萊比錫紅牛，亞特蘭大和瓦倫西亞兩支潛在黑馬捉對廝殺，拜仁時隔8年再次遇到昔日歐冠決賽擊敗自己的冤家切爾西，四面楚歌的那不勒斯迎來了狀態起伏不定的巴塞羅那，青春風暴的里昂主帥加西亞迎來了昔日意甲老對手尤文圖斯，而皇馬和曼城這兩支火星撞地球的比賽更是點燃了所有球迷的激情。

歐冠八分之一決賽抽籤

馬競VS利物浦：兩大鐵血教頭的碰面

馬競VS利物浦

毫無疑問馬競是利物浦最不想遇見的對手，雙方過去4場歐戰交鋒各取1勝幾乎伯仲之間，兩大鐵血教頭西蒙尼VS克洛普，拼的不僅僅是戰術，更是球隊的必勝意志。雖然近些年成績有所下滑，但是馬競在歐冠絕對是不容忽視的球隊。對於馬競而言，利物浦並非絕對的下下籤，除了萊比錫以外，拜仁、尤文、巴黎這樣的強隊究竟比利物浦或者曼城弱多少很難講。

多特蒙德VS巴黎聖日耳曼：圖赫爾知根知底

巴黎聖日耳曼VS多特蒙德

對於多特蒙德而言，從死亡小組突圍最終進入16強尤為不易，但是最艱難的是，他們將迎來對他們知根知底的前任主帥圖赫爾帶隊的巴黎聖日耳曼.

值得一提的是，兩隊此前在歐冠從未有過交手的數據，唯一一次歐戰碰面還是在10-11賽季的歐聯杯，不過當時兩隊在小組賽各自打平並沒有分勝負，時過境遷，兩隊的整體實力都有著非常大的提升，但是本賽季內憂外患均有諸多困難。對於巴黎而言，抽中多特蒙德算是相對理想的中上籤，但是昔日教訓歷歷在目，巴黎在歐冠沒有絕對的優勢。

熱刺VS萊比錫紅牛：驚喜萊比錫書寫新歷史

熱刺VS萊比錫紅牛

萊比錫紅牛是16強中唯一一支16強的新軍，他們在歐冠往前的每一步都在創造一個歷史。雖然初生牛犢，但是萊比錫絕非任人宰割的球隊，作為本賽季德甲進球數最多的球隊，熱刺本賽季脆弱的防線，勢必將受到一番嚴峻的考驗。對於熱刺而言，在八分之一決賽抽籤中在僅剩萊比錫和巴薩兩個對手可選的情況下，規避了穆里尼奧的老熟人巴薩而選中了萊比錫不啻為一個理想的選擇。

亞特蘭大VS瓦倫西亞：對於雙方而言均是上籤

瓦倫西亞VS亞特蘭大

本屆歐冠16強全部球隊均來自於五大聯賽的情況下，亞特蘭大和瓦倫西亞這些球隊理所應當成為了諸多豪門希望碰到的對手，而兩支球隊的相遇，一定程度上對於兩隊而言均是上籤。對於亞特蘭大這支歐冠新軍而言，能夠進入16強已經是超額完成目標，瓦倫西亞本賽季在西甲表現並不理想，從整體實力來看，亞特蘭大佔據優勢。

切爾西VS拜仁慕尼黑：拜仁時隔8年迎來複仇之戰

切爾西VS拜仁慕尼黑

2012年歐冠決賽，切爾西在落後的局面下上演了致命一擊，最終通過點球淘汰了拜仁，兩隊在歐洲賽場相遇甚少，但一旦相遇大多是拜仁的佔據下風，除了2012年的歐冠決賽，本世紀另外一次交鋒是在04-05賽季的歐冠四分之一淘汰賽，最終同樣是切爾西6-5擊敗拜仁。

切爾西轉會禁令已經取消，今年冬窗將會有大筆資金投入引援，對於主帥蘭帕德的磨合陣容功底提出了不小的要求，弗里克率領拜仁的時間並不長，兩位還未證明自己的主帥狹路相逢，對於雙方而言並非壞事。

那不勒斯VS巴塞羅那：巴薩避免重蹈羅馬覆轍

那不勒斯VS巴塞羅那

巴薩抽到剛剛換帥的那不勒斯是一個好籤，從上一場意甲1-2不敵帕爾馬的比賽來看，加圖索似乎並沒有扭轉那不勒斯低迷的狀態。但巴爾韋爾治下的巴薩在聯賽艱難鏖戰，在歐冠也早已失去了往日的統治力，連續2年在歐冠淘汰賽遭到對手大逆轉。陣容實力比兩年前羅馬還強的那不勒斯未必沒有取勝的機會，背水一戰的加圖索VS戰戰兢兢的巴爾韋德，巴薩可別再重蹈羅馬覆轍。

里昂VS尤文圖斯：尤文喜獲最佳簽

里昂VS尤文圖斯

16支球隊各有千秋，但要提各大豪門最願意碰到的對手，莫過於瓦倫西亞里昂這類本賽季多線作戰舉步維艱的球隊。考慮到瓦倫西亞被亞特蘭大分走，抽到里昂的尤文圖斯喜獲最佳簽。就連出席抽籤儀式的內德維德都坦言球隊的運氣不錯。

雙方歷史上在歐戰相遇4次，尤文3勝1平保持不敗，本賽季里昂在法甲表現糟糕，僅僅位列聯賽中游，近期主攻手更是遭遇傷病賽季幾乎報銷，主帥加西亞在意甲面對尤文就是勝少負多，很難想像屆時魯迪·加西亞會有什麼好的辦法面對這支意甲老江湖尤文圖斯。

皇馬VS曼城：火星撞地球血拚到底

皇馬VS曼城

皇馬面對曼城的比賽，絕對是八分之一淘汰賽中最重磅的重頭戲，聯賽衛冕幾乎無望的曼城肯定會將主要重心放到歐冠賽場，而對大耳朵杯執念的齊達內也將就歐冠發起新一輪的衝擊。

同為頂尖強隊的曼城和皇馬在本賽季聯賽屢屢犯錯的表現，也讓本場比賽充滿了懸念，瓜迪奧拉可以鬆口氣的是曼城目前有的問題，皇馬也有，那就是兩支球隊都可以踢得不錯，但攻擊線的進球效率都不高，貌似臨場比賽時哪隊的前鋒更有靈感運氣更好誰就更有出線的機會。

本賽季的歐冠淘汰賽將於明年的2月19日點燃戰火，誰能成為下一位登頂歐洲的王者，讓我們拭目以待！
