前無古人！恆大或衝擊男足歷史首個世界盃賽冠軍

2019-12-21 202條評論 2,342次閱讀 732人點贊

近期，隨著高拉特歸化成功，廣州恆大的全華班「拼圖」又完成重要一環，隨著大量恆大歸化名單在陸續完成中，球迷們可能會想，廣州恆大的終極目標究竟是啥？

幫助中國男足打進世界盃？可能有這個目標，但是估計難度有點大，而球迷相信看到了本文標題，男足歷史首個世界盃賽冠軍，這裡沒有問題，男足歷史，包括11人男足、五人制男足等，世界盃賽冠軍沒問題，世界級的杯賽冠軍，這個杯賽冠軍就是世界俱樂部冠軍杯。

2021的世俱杯已經確定在中國舉辦，中國將以東道主的身份參與這個只有洲際聯賽冠亞軍才能參與的賽事，而廣州恆大如無意外作為中超八冠王將代表中國參加世界俱樂部冠軍杯。

可能球迷會有問題，廣州恆大的陣容，有可能衝擊世界俱樂部冠軍杯賽冠軍？

離比賽正式舉行還有一年多時間，實際上一切皆有可能。

首先是恆大歸化及「超級拼圖」計劃。埃爾克森、阿洛伊西奧（野牛）、高拉特、費爾南多（小摩托）、阿蘭已經陸續歸化入籍，那麼恆大加上朴志洙、布朗寧、保利尼奧、塔利斯卡以及羅伯特蕭，已經可以排出一支歸化和外援超過半支球隊的強力陣容，這樣的陣容在亞洲絕對頂級，前鋒線完全可以實現全洋槍，只有門將、後衛、中場配一些本土球員，就可以湊出一支牌面頂級的陣容。

其次，恆大還有一年多的時間去磨合這套陣容。由於中國男足還在征戰2022世界盃四十強賽，在後面的比賽中，大量恆大歸化球員將繼續代表國家隊和俱樂部世界級大賽中磨合，包括世界盃、亞冠聯賽、中國杯等頂級大賽舞台。

第三，要保證奪冠，恆大還需要一個王炸，面對皇馬這樣的世界頂級俱樂部，要想贏球，必須要有一個和他們同級別的王炸，這個王炸很可能是勞塔羅。

廣州恆大主帥卡納瓦羅已經公開宣布了對國米球員勞塔羅的興趣（畢竟手臂紋身是接近的），恆大能否簽下勞塔羅？首先錢肯定是一方面，但是恆大隻要有決心，掏幾個億也不是沒可能，最關鍵的是勞塔羅屬於國米，國米屬於中資蘇寧集團，鑒於廣州恆大集團和江蘇蘇寧集團之間的合作關係，促成一次中資俱樂部之間的轉會並非天方夜譚，如今廣州恆大韋世豪有望租借西甲格拉納達去「打怪練級」，也是因為對方也是中資背景。

迪巴拉和勞塔羅，有望成為下一個世界足壇絕代雙驕、球王級的人物，勞塔羅雖然個子不高，但是控球能力強大，身體能抗後衛，一腳觸球能力強，接球瞬間就能完成過人、背身過人、射門或傳球，具備了中國球員最喜歡的外援特點，可以一個人或者一兩個人配合生吃對方後衛。

第四，那麼恆大那麼多後防漏洞怎麼辦？雖然後防已經亞洲頂級，但是面對世界強隊還是感覺不放心，這個問題的解決方案可以參照中東豪門的解決方案，就是在世界俱樂部杯賽期間，去找幾個世界級後衛打短工，以短期租借的形式加盟恆大，從而完成恆大亞洲霸主級的超級拼圖。

第五，主場優勢。這些優勢包括裁判優勢，包括本土作戰各種便利，必將讓球員超出100%發揮出全部實力。

有了上面講的五點，大家覺得廣州恆大是否有機會打破隊史記錄（2013年世俱杯第四名），衝擊2021年世俱杯冠軍？
