[意甲]薩索洛攻防核心缺陣，紅黑軍周年慶典必取勝

2019-12-15 226條評論 1,978次閱讀 319人點贊
  • 本輪比賽對於AC米蘭來說具有不一樣的意義，今天米蘭要在主場舉行120周年慶典，俱樂部高層和多位名宿將現場觀戰，同時參加慶典，因此米蘭眾將士必然是戰意十足，以一場勝利捍衛球隊的榮譽，可以想像，如果不能取勝，迎接米蘭球員的將是球迷的唾罵和高層的憤怒。在賽前的新聞.發布會上，皮奧利就表示本場比賽不同以往，必須為米蘭的榮譽而戰。

（皮奧利表示本場米蘭必須取勝）

  • 在博納文圖拉復出以後，米蘭的戰績有著顯著提升。他先是在對陣那不勒斯的比賽中遠射幫助球隊扳平比分，隨後客場面對帕爾馬，也是他的遠射造成對方守門員脫手，給了特奧補射的機會,上一輪對陣博洛尼亞他的進球幫助球隊鎖定勝局，3場比賽奉獻2球I助攻，幫助球隊拿到7分，皮奧利能夠穩住帥位博納文圖拉功不可沒。
  • 薩索洛賽季至今已經丟了27球，防守能力聯賽倒數第3，主力門將孔西利缺席了近3場比賽，球隊便丟了6球，無論是圖拉蒂還是魯索，實力上都難與孔西利相提並論，而本場比賽孔西利還是無法復出，除此之外，薩索洛當家射手貝拉爾迪同樣因傷缺陣，貝拉爾迪11輪聯賽奉獻了8球2助攻,他的缺陣讓球隊失去了最重要的得分點。這兩大核心的缺陣讓薩索洛的實力大打折扣，本場比賽恐怕難逃一敗。
  • 歐洲指數在後期繼續將主勝拉低，平局和客勝都被抬高，亞洲指數已經統一抬高到主讓1球，在米蘭120周年慶典之際，機構給了其足夠支持，加上客隊核心缺陣，本場比賽米蘭必勝無疑，本場推薦: AC米蘭勝。
