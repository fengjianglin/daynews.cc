歐冠：勒沃庫森vs尤文圖斯 藥廠能否出線，命運掌握在對手手上

天天要聞
7 天 ago

歐冠：勒沃庫森vs尤文圖斯 04:00

昨天巴薩真是公正無私，「幫助」多特蒙德成功出線，德甲球隊就剩一個隊沒晉級了，今天勒沃庫森能有奇蹟嗎？

這場對勒沃庫森來說只有一條路可走，那就是主場戰勝尤文圖斯。同時開始的另一場比賽，莫斯科火車頭能否阻擊馬競，只能聽天由命了。

最近一段時間勒沃庫森成績不錯，5勝1平，對手不乏馬競、拜仁等難纏對手。用博斯的話說，其實他們一直踢得很好，只是經常得勢不得分，現在他們運氣好了起來。

其實這不過是官場話，能走出低谷，源自博斯對球隊做出的明顯改變。這位過去以「頭鐵」著稱的教練，也開始向現實低頭了。

這5場勝利，勒沃庫森控球率最高也不過56%。要知道，此前他們可是五大聯賽僅次於巴黎的控球強隊，控球率超過七成如同家常便飯。

最明顯的就是客場對陣拜仁的比賽，博斯上任一年來，首次單場在控球率上落後於對手，而且落後得非常徹底——僅有25%，但換來的卻是高達7次的射正，以及在安聯球場的客場勝利。

與控球率一起被拋棄的還有博斯的詭異陣型。此前為了加強中後場的控制力，博斯經常使用三後衛，讓能力最強的弗蘭德踢邊翼衛來組織推進，本末倒置。最近幾輪，博斯重新打回了正經的4-2-3-1，在兩翼派上迪亞比、貝拉拉比和貝利等速度型球員，弗蘭德回到中路，這期間貢獻了1球3助攻。

博斯幡然醒悟，及時調整，薩里卻仍然走在固執的道路上。棄用曼朱基奇和埃姆雷·詹、邊緣化C羅、倒腳戰術枯燥低效、頻繁甩鍋球員甚至是布馮……在不敗期間，這些並不是什麼大事，但一旦輸球，薩里受到的質疑只會越來越多。

1-3不敵拉齊奧，遭遇賽季首敗，這個結果看似冷門，實際完全在情理之中，藍鷹全場9次射正，兩倍於尤文。如果不是尤文衛冕冠軍的班底不錯，再加上迪巴拉、科斯塔連續絕殺救主，薩里的首敗，早就應該到來了。

本場斑馬軍團的戰意也明顯不如藥廠強，他們已經鎖定小組頭名。薩里一如既往拒絕大輪換，只有德利赫特未隨隊來到德國，但透支體能的做法對老將居多的球隊來說，並非是好事。尤文防線除了誇德拉多，也很難有人跟得上藥廠邊鋒的速度。

勒沃庫森出線需要奇蹟，但做好自己，才有奇蹟的可能。看好藥廠主場取勝。
