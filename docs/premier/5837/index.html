愛哭的孩子有糖吃？國米1月份將為孔蒂引進吉魯和達米安

2019-11-08 0條評論 16次閱讀 0人點贊

吉魯與切爾西的合同將在本賽季結束後到期

義大利媒體報道，國際米蘭將會在一月份引進吉魯、庫盧塞夫斯基和達米安，但看來要簽下巴塞羅那中場比達爾的可能性不大了。

孔蒂最近再次抨擊藍黑軍團在今年夏天引援不力，為了滿足主帥的要求，國米預計會在冬季轉會窗口忙碌起來。

達米安目前已經離開曼聯，加盟帕爾馬

根據《米蘭體育報》的消息，法國前鋒吉魯已經和國米進行談判，並同意加盟球隊擔任盧卡庫的替補。吉魯與切爾西的合同將在本賽季結束後到期，本賽季在斯坦福橋得到的出場機會已經不多。

孔蒂很了解這名33歲的高中鋒，2018年1月藍軍從阿森納簽下吉魯時，就是孔蒂擔任主帥。吉魯目前的年薪達到800萬歐元，但國米可以與他簽約2-3年，將工資分擔開來。

相比比達爾，蘇寧集團希望國米引進一名更年輕的中場

此外，《米蘭體育報》聲稱，國米同樣在對兩名帕爾馬球員很感興趣。庫盧塞夫斯基從亞特蘭大租借加盟帕爾馬，至今打進2球送出5次助攻，這名19歲的瑞典邊鋒在過去幾周已經吸引了國米的關注。而達米安一直都是孔蒂很欣賞的球員，他在防線上的全面性正是孔蒂所看重的。

不過，義大利足球市場網透露，蘇寧集團不會同意孔蒂引進比達爾的計劃，他們更希望藍黑軍團在1月份引進一名更年輕的中場。雖然智利人公開表達了渴望離開巴薩的新援，但巴薩是否願意放他在冬窗離開還是個問題，《都靈體育報》則指出，他能夠加盟國米的唯一途徑是巴薩同意讓他租借離隊。(編輯：天藍色男孩)
