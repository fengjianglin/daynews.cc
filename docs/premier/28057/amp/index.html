[關於離散] 競技體育分析指數，也就是各家歐指分歧度。基本原理：百家歐指平均值與每家指數差值的絕對值的求和。離散度小則各家觀點接近，進而賽果易打出；離散度大反之意見分歧大，賽果不易打出。又因各方因素影響下指數變動，離散在特殊時刻具有更高參考價值，切勿盲目參考運用。換句話說，離散只是參考指數，而非結論性數據!以下是各場數據匯總(數據截取至11:00)：

周五007 法乙 卡昂vs克萊蒙特

比賽時間：12月21日03:00

個人觀點：卡昂平負/平局（0）

比分預測：0-1/1-1

法乙第19輪爭奪，也是上半程聯賽最後1輪，本輪比賽後法乙將進入長達三周的冬歇期。目前積分榜上，洛里昂2分優勢領跑，朗斯和特魯瓦分列二三位。本場兩隊表現不算驚艷，客隊目前位列第5位，有機會衝擊升級名額；但主隊作為降班馬，目前排名10名開外，落後前三名多達15分，形勢並不明朗。

卡昂作為降班馬，上賽季法甲倒數第二不幸降級。球隊本賽季征戰法乙也不見起色，前18輪聯賽4勝8平6負，僅僅拿到20個積分，排在聯賽第13位，落後前三多達15分，重返法甲基本已成空談，而領先降級區也僅僅只有3分，目前保級壓力巨大。球隊前9個主場2勝4平3負，不過最近5個聯賽主場2勝3平不敗，整體狀態處於上升趨勢。

克萊蒙特上賽季法乙排名第10位，本賽季球隊表現明顯進步。目前18輪聯賽過後，克萊蒙特8勝5平5負，目前拿到29個積分，排名法乙第5位，落後榜首球隊10分，落後第三的特魯瓦6分，依然很有希望至少衝擊升級附加賽名額。球隊前9個客場5勝2平2負，客場戰績相當出色，其中近4輪聯賽做客2勝2平不敗，整體狀態同樣保持良好。

雙方上一次交手已是2013-2014賽季，當時卡昂聯賽1勝1平佔上風。而最近6次交手，卡昂3勝3平保持不敗，聯賽主場保持dui9克萊蒙特4連勝。不過風水輪流轉，目前雙方差距在不斷縮小。目前賬面實力對比那看看，卡昂全隊身價2620萬歐，是克萊蒙特全隊身價接近2倍，但初始勝平負指數2.71 2.93 2.69，法乙非典型222指數組合，平局受到極大打壓，雙方几乎完全均勢格局。即時指數稍稍向主隊傾斜，目前均值2.72 2.89 2.75，相對優勢回歸主隊。不過離散指數一邊倒傾斜，當前客勝指數16.84繼續維持中低位，平局指數低穩，主勝指數則稍稍高企，指數對主隊相當不利。綜合分析來看，本場看好克萊蒙特客場不敗，重點關注平局打出。
