第98分鐘絕殺！意甲上演瘋狂大逆轉，昔日冠軍8連勝距榜首僅3分

天天要聞
2019-12-17

北京時間12月17日凌晨，意甲聯賽第16輪的最後一場較量在撒丁島競技場打響，13輪不敗的意甲黑馬卡利亞里坐鎮主場迎戰豪取7連勝的拉齊奧，在開場8分鐘便一球落後的情況下，拉齊奧在傷停補時階段上演瘋狂大逆轉，5分鐘連入2球，2-1絕殺對手，小因扎吉率領的藍鷹軍團追平隊史聯賽最佳開局，距榜首僅3分。

卡利亞里是意甲本賽季的最大黑馬，上賽季在保級區苦苦掙扎的卡利亞里開局遭遇2連敗後，隨後的13輪里豪取8勝5平的不敗戰績，16輪過後力壓羅馬位列第四，而昔日的意甲冠軍拉齊奧同樣狀態火熱，在第5-7輪遭遇3輪不勝後，小因扎吉的球隊已豪取聯賽7連勝，位居意甲第三。

開場不到8分鐘，主場作戰的卡利亞里再次顯露黑馬本色，通過一次界外球機會取得夢幻開局，若昂-佩德羅禁區左側接隊友擲出的界外球頭槌後蹭，24歲的西蒙尼之子喬瓦尼-西蒙尼前點掃射破門，今年夏天從佛羅倫薩租借加盟卡利亞里的小西蒙尼打入本賽季個人聯賽第5球。

一球落後的拉齊奧隨後展開猛攻，但均無功而返，下半場，小因扎吉接連作出換人調整，洪尼、卡達迪和凱塞多先後替補出場，而卡利亞里憑藉頑強的防守，將1-0的領先優勢保持至傷停補時階段。最終拉齊奧憑藉替補球員的出色發揮，在傷停補時階段5分鐘內連入2球完成瘋狂大逆轉。

第93分鐘，拉齊奧將比分扳平，米林科維奇-薩維奇禁區內接隊友左路傳中回做架炮，27歲的西班牙中場路易斯-阿爾貝托在禁區線附近迎球抽射，皮球直竄球門左下角。

長達7分鐘的傷停補時讓拉齊奧看到全取3分的希望，果然在第98分鐘讀秒階段，拉齊奧上演絕殺，第56分鐘替補出場的洪尼左路起球傳中，第80分鐘披掛上陣的凱塞多門前頭槌攻門，將球頂入遠角，31歲的厄瓜多前鋒脫衣慶祝，拉齊奧全隊陷入瘋狂。

最終憑藉傷停補時階段的連入兩球，拉齊奧2-1逆轉擊敗卡利亞里，送給黑馬近14輪首敗的同時，自己也豪取聯賽8連勝。16輪過後，拉齊奧積36分穩居第三，得益於領頭羊國際米蘭本輪丟分，藍鷹軍團與榜首差距縮小至3分，意甲形成三強爭霸局面。

意甲開局16輪拿到36個積分，藍鷹軍團也追平了隊史聯賽最佳開局，以1973/74賽季換成三分制，拉齊奧在那個賽季開局16輪同樣拿到36分，並最終奪得隊史首座意甲冠軍。
