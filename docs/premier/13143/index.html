冷門跌出，兩大黑馬歐聯杯小組出局，球迷吐槽為窩裡橫球隊

2019-12-13 324條評論 2,381次閱讀 825人點贊

　　北京時間12月13日，歐聯小組賽第6輪比賽全部戰罷，24支出線球隊出爐了，他們將跟歐冠賽場淘汰下來的8支球隊一起組建32強，為了本賽季的歐聯冠軍展開爭奪。幾家歡喜幾家愁，有的球隊死裡逃生，有的球隊則是悲劇出局，要說最大的冷門，毫無疑問是門興跟拉齊奧的出局讓人無語了。

　　這兩支球隊本賽季在聯賽表現非常精彩，門興目前是在德甲處於領頭羊位置，領先拜仁都達到了7分，上一輪比賽，門興就是在主場2-1擊敗了拜仁，球隊在聯賽展示出了極強的爭冠水平，而在歐聯賽場，此前5輪戰罷，羅馬8分，門興8分，伊斯坦布爾7分，沃爾夫斯貝格4分。

　　另外一支球隊拉齊奧本賽季表現也是十分精彩，聯賽15輪戰罷拿到了33分，目前位居第3名，上一輪聯賽，拉齊奧在主場3-1擊敗班霸尤文圖斯，狀態非常暴躁，目前是落後尤文3分，落後國米5分。而在歐聯賽場，此前5輪戰罷，凱爾特人13分、克魯日9分、拉齊奧6分，雷恩1分，拉齊奧小組出線十分艱難。

　　最終，在決定生死的第6輪比賽里，門興主場1-2輸給了伊斯坦布爾，而另外一場比賽里，羅馬隊2-2戰平對手。手握大好出線形勢下，門興居然掉到了第3名出局，而羅馬則幸運拿到9分出線了，真是一個誰都想不到的結果，此前排在第三名的伊斯坦布爾還拿到了小組第一。

　　而拉齊奧隊則比較慘了，5輪戰罷，他們的出線主動權就沒掌握在自己手中，除了要擊敗雷恩外，還要期待布魯日輸給凱爾特人，而最終兩場比賽戰罷，拉齊奧0-2輸給了雷恩，而那邊布魯日不管結果如何，都意味著拉齊奧出局了，事實上布魯日主場2-0完勝凱爾特人，沒有給拉齊奧任何念想。

　　兩支在聯賽表現十分暴走的球隊，在歐聯賽場紛紛小組出局，這也是讓人無限唏噓，堪稱一個大冷門了，廣大球迷也是吐槽乃「窩裡橫球隊」，尤其是拉齊奧，最近幾年更是典型的送分童子，屢屢在歐聯賽場沒啥表現。

　　（代古龍）
