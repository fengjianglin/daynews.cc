解簽歐冠十六強：英超艱難，西甲好運，意法德伺機而動

2019-12-17 44條評論 1,018次閱讀 68人點贊

為什麼歐洲冠軍聯賽成為如今熱點和流量最高的賽事，因為這裡雲集了地球上最具競爭力的隊伍，本賽季歐冠十六強皆出自五大聯賽，贏下最後的冠軍就意味著俱樂部可以成為五大聯賽中國當賽季表現最好的一支隊伍。當然，爭冠過程一定會存在運氣，就像本次十六強抽籤，但如果十六支隊伍全部來自歐洲五大聯賽的，運氣似乎沒那麼管用了，更多的要依靠實力。

多特蒙德vs巴黎聖日耳曼：熟悉的陌生人

圖赫爾雖然曾經帶領過多特蒙德征戰過3個賽季，但並不了解如今的多特，黃黑軍團只剩下羅伊斯和魏格爾兩名老人。早在美因茨時期，圖赫爾便多次與黃黑軍團在德甲賽場交手。可惜，作為對手迎戰多特蒙德，圖赫爾的戰績相當糟糕，過去各項賽事6次交手，德國人均以失敗告終。當然，目前圖赫爾率領巴黎聖日耳曼卻有足夠的底氣。

多特蒙德陣容過於年輕導致球隊狀態及其不穩定，擅長發揮球隊主場優勢，但缺乏一錘定音的能力。主場佔盡優勢卻被巴薩0-0逼平，最後終結比賽的能力一般，兩戰國米球隊丟了4個球。遭遇鋒線能力強的球隊時，多特往往是失敗的一方，1-3輸給當時狀態不佳的巴薩就是代表。最後能夠晉級淘汰賽，還有感謝國米主場未能拿下替補出戰的巴薩。

本賽季巴黎表現不錯，以5勝1平的戰績力壓皇馬小組第一出線，由納瓦斯把守的大門6場比賽僅失兩球，還是客場迎戰皇馬時丟的。圖赫爾拿出這套442陣容更好的平衡了球隊攻強守弱的問題，帕雷德斯迅速成長，蓋伊成為了球隊的新屏障，中後場得到加強讓鋒線更好地投入到進攻中。拋棄卡瓦尼的新鋒線組合聯手攻入24球，重新回到邊前衛的迪馬利亞則打出6球6助的絕佳數據。

迎戰多特，圖赫爾更有信心。

皇家馬德里vs曼城：巴薩國王與皇馬巨擘的較量

有個名人曾經說過：「我變禿了，也變強了。」這句話用在皇馬和曼城的教練身上沒有絲毫違和。兩名光頭教練經歷相當類似，瓜迪奧拉首次執教巴薩便創造了不可一世的「夢三王朝」，第一個賽季拿下六冠王；齊達內初掌皇馬兵權，就打破了歐冠的衛冕魔咒，連續三個賽季捧起大耳朵杯成就傳奇。

兩隊本賽季都出現狀態起伏，二進宮的齊達內遭遇球隊陣容更替的問題，當初三連冠中軸線都不再年輕，核心莫德里奇34歲，最佳射手本澤馬32了，雷打不動的拉莫斯同樣33歲；年輕一代中只有維尼修斯和巴爾韋德保持較穩定發揮；比較麻煩的是球隊新援，阿扎爾未能達到最佳狀態，約維奇依舊處於磨合階段，門迪最近剛剛獲得信任。不過，皇馬逐漸將重心落到防守打磨上，近期成績穩定，只是由於鋒線尚未融入導致進攻不夠犀利。

曼城一方則是後防失火。本賽季曼城在英超中表現不佳，如今已落後榜首利物浦14分，球隊後防堪稱災難。大衛-席爾瓦的年紀難以承擔持續高強度比賽，京多安強強對話難以撐起中場，羅德里的控場風格也無法取代費爾南迪尼奧的防守。曼城中場控制力下滑，後防長期處於被迫回追的局面，在拉波爾特受傷以後，斯通斯和奧塔門迪就是兩個漏勺，藍月亮只能寄希望於冬窗開啟購置球員補充。

兩隊最終鹿死誰手還要看到時候的狀態。

亞特蘭大vs瓦倫西亞：黑馬之戰

對於歐冠淘汰賽來說，兩支黑馬球隊直接碰撞是一種損失，黑馬往往伴隨著極大的未知性，甚至可以說是氣運加身，亞特蘭大和巴倫西亞都具備這樣的素質。

由於分心歐冠，兩隊在自家聯賽表現都不溫不火，不過晉級路線卻完全不一樣。巴倫西亞由於雙線作戰，儘可能地將重心放在歐冠聯賽中，除了一場0-3外，球隊一直保持不敗，尤其是在切爾西和阿賈克斯的主場均以1球小勝，成功地以小組頭名出線；亞特蘭大最初是奔著陪讀來的，前三輪小組賽全敗僅丟11球，結果到了第4輪，球隊發現同組的直接競爭對手都敗在曼城強勢之下，球隊拿出了五大聯賽勁旅的實力搶分，球隊2勝1平積7分，末輪出線。

兩隊的防守都不夠好，但亞特蘭大進攻火力更強，38球位列意甲第一，而巴倫西亞更善於通過定位球破門得分。如今的歐冠，誰更擅長進攻誰更有機會往前進一步。

馬德里競技vs利物浦：利矛攻盾

利物浦本賽季狀態飈紅，17輪英超聯賽拿下16勝1平，強勢的表現也延續到了歐冠，小組賽雖然輸給過那不勒斯，但那場比賽主要還是進攻狀態不佳。馬內-菲爾米諾-薩拉赫三人組只要有一名球員在線，都能夠改變比賽進程。如今克洛普更注重球隊進攻效率，而非依賴體能鑄造終極逼搶的戰術，讓球隊享受不少輪換福利，雙線作戰卻依舊保持極強的競爭力。這也是球隊本賽季的可怕之處，再有上賽季的奪冠經驗，球隊能夠更從容地面對歐冠淘汰賽。

紅軍強勢反而顯示出馬競方面的頹勢。西蒙尼的戰術似乎已經走到了盡頭，花重金引進的菲利克斯並未撐起鋒線，進攻端創造力到了油盡燈枯的局面，陣地戰缺乏破局核心，反擊時鋒線效率不夠，聯賽僅僅攻入18球。不僅如此，球隊高位逼搶的實力下滑不少，建立出逼搶強側卻難以斷下皮球，中場球員實力不夠，導致球隊由攻轉守時速過渡期太長，進攻端長期處於少打多的局面。

結合兩隊目前的狀態，利物浦更有機會改變比賽進程。

切爾西vs拜仁慕尼黑：互交學費

切爾西很年輕，年輕到蘭帕德才41歲，隊內年事最高的佩德羅也才32。比起多特蒙德，切爾西的青春更加肆意，隊內頭號射手亞伯拉罕剛剛22歲，邊路核心芒特與普利西奇都是97後，後防更是沒有一名老將坐鎮。這也導致切爾西現在的尷尬局面，時常佔盡優勢被對手反擊進球，陷入逆風球後無法站穩腳跟，強強對話中，遭遇那些老辣的豪強就會被對手帶入節奏，最終遭遇失利。

拜仁則是一支老辣的豪門，球員有足夠的經驗，陣容結構合理老中青三代各司其職。不過教練弗里克也是代理主帥，此前多為助教，率領拜仁征戰歐冠是一項極大的挑戰。更重要的是球隊目前還需要磨練，弗里克上任放棄了科瓦奇的試驗，回歸正統的4231陣型，讓穆勒等老將重新得到了機會，上輪聯賽6-1完勝美因茨也算是近期的成果。不過，這些利好局面只是暫時的，要想站穩腳跟，拜仁需要進行更多的磨合打造出一支更強的隊伍。

一切都在明年2月，兩隊最終會進化成什麼樣。

里昂vs尤文圖斯：鋒線間的較量

兩支隊伍優點與缺點同樣明顯。里昂進攻端有德佩，荷蘭人在法甲重新找回了自己，本賽季出場13次貢獻9球2助攻，當之無愧的核心球員。此外，小將登貝萊和阿瓦爾分別是球隊鋒線妖人，前者為隊內頭號射手，後者則是過人王。相應的，球隊中後場實力較為一般，但是好在實力平均沒有明顯短板，教練對於防守端的調教相當務實，區域式防守運用是法甲翹首。

尤文圖斯坐擁C羅，伊瓜因本賽季狀態也不錯，雖然目前薩里的戰術體系中，葡萄牙人融入得還不夠，反倒是更年輕的迪巴拉近期屢次拯救球隊，但不能否認這些征戰沙場的老將，會在歐冠賽事中爆發出驚人的能量。比起鋒線豪華，尤文中場的確不夠細膩，薩里的戰術風格讓馬圖伊迪、本坦庫爾顯得粗糙，皮亞尼奇獨木難支，引進的拉比奧特和拉姆塞莫名變成替補，使得尤文面對中下游球隊很難打出犀利進攻。

面對里昂這種衝擊性踢法的球隊，一旦對手拉低姿態慢慢打反擊，尤文的勝利不會太容易。

熱刺vs萊比錫：穆里尼奧的歐冠對手很強

如今的熱刺不管是實力上還是陣容結構都延續了上賽季的水準，唯一不同的是教練從波切蒂諾換成了穆里尼奧，前者下課並非戰術水平而是無法掌控更衣室，這也是熱刺目前的隱患之一。穆里尼奧的到來能夠緩解更衣室焦躁的氛圍，但治標不治本，熱刺球星們低廉的工資和嚮往轉會的內心已然成為現實。要想發揮出熱刺真正的戰鬥力，光是一個穆里尼奧是不夠的，而距離明年二月份還有一段時間，還需要看葡萄人如何調教。

那邊的熱刺潛藏著炸藥，萊比錫卻打出了近幾個賽季最好的表現，頗有2011年克洛普說率領多特蒙德席捲德甲的氣勢。雙線作戰對於這支年輕的隊伍沒有絲毫影響，反倒是挖掘出了更多的潛力。鋒線維爾納15場聯賽貢獻16球5助攻，中場小將恩昆庫更是成為了當之無愧的核心。在主教練納格爾斯曼的領導下，球隊火力全開，控制節奏的同時把握反擊機會，進攻手段全面。面對任何對手，萊比錫紅牛都可以憑藉著自身陣容實力變換不同的進攻方式，球隊是目前的德甲第一火力42球，其中有32球是運動戰。

那不勒斯vs巴塞羅那：翻車or轉運

如今的巴薩可以說誰都敢輸，尤其是在歐冠淘汰賽這個階段，連續兩年遭遇對手逆轉讓球隊心理下滑到一個階段。本賽季在巴爾韋德的領導下，球隊戰績和實力進一步下滑，雖說小組賽保持不敗，但真正踢得比較好的只有一場半。兩戰布拉格斯拉維亞都被踢得異常被動，做客多特被對手摁著錘了一整場，交手國米直到下半場才艱難逆轉。球隊中軸線老化，跑動能力達不到歐冠的平均水準，前場缺乏年輕爆點。這種態勢下要想拿冠軍，除非所有歐洲豪強都積貧積弱，顯然不可能。

那不勒斯也好不到哪裡去，安切洛蒂帶隊交手利物浦絲毫不落下風，甚至送對手近半年多來唯一的敗績，球隊也進入了淘汰賽，結果轉身便被俱樂部老闆炒掉。究其原因是害怕雙線作戰影響球隊聯賽成績，無法維持下賽季的歐冠名額，因此拿下與自己理念不合的安切洛蒂，扶起加圖索。且不說兩名教練經驗和水平上的差距，光是著眼於聯賽，間接放棄歐冠的意思就會讓那不勒斯很難再進一步。

儘管巴薩對陣意甲球隊戰績糟糕，但目前這支那不勒斯並沒有想像中那麼難啃。
