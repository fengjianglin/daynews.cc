無皇馬不決賽！C羅超級盃後一細節表示無聲的抗議

2019-12-23 328條評論 1,285次閱讀 79人點贊

在今日凌晨舉行的義大利超級盃比賽中，上賽季的意甲冠軍尤文圖斯和上賽季義大利杯冠軍拉齊奧遭遇，最終C羅領銜的尤文圖斯1比3不敵拉齊奧。

阿爾韋托首開紀錄，C羅射門被封堵，此後迪巴拉補射追平。下半場，盧利奇與卡達迪破門，拉齊奧成為最終的贏家，實現了在本賽季意甲聯賽和超級盃中對尤文的完美，雙殺證明了義大利杯賽冠軍才是意甲的真正王者。

在本場比賽結束之前，C羅有一項傲人的記錄，那就是「決賽福將」的美譽，自2013年決賽不敵馬競以來，C羅已經實現決賽14連勝的壯舉，比賽種類包括了國王杯、歐洲杯、歐冠、UEFA超杯、世俱杯、西班牙超級盃、歐國聯杯等，其中11個是在皇馬取得的決賽勝利，接近所有決賽勝利的80%，據說C羅已經告訴多位皇馬隊友表示後悔離開了皇馬。

本賽季比賽意甲和超級盃尤文被雙殺的原因可能是戰術克制，在薩里保守的多層次縱深防守陣型4-3-1-2立足防守，因此在聯賽中能取得大部分比賽的勝利，但是卻被拉齊奧針對性的更直接更有寬度更有速度的3-5-2克制。

尤文雖然全場控球高達65%，傳球次數680次接近對方傳球373次數的兩倍、傳球成功率也91%也領先對方十個點，但是尤文這種層層推進有層次沒速度的打法，總是給對方足夠的布防準備時間，最後運動戰打成了陣地戰，包括C羅最有威脅的射門也被早已準備就緒的對方門將封堵，全場射門16腳只兌換一個進球。

作為為冠軍而戰的C羅，顯然亞軍不是榮耀而是恥辱，在比賽結束後，拉齊奧慶祝奪冠，戴上了象徵冠軍的金牌，而作為失利的尤文一方，只能戴上象徵亞軍的銀牌，和尤文隊友布馮等人戴著銀牌相比，C羅直接把銀牌抓在手裡，沒有戴到脖子上，顯然總裁對於獲得銀牌很生氣，一生不認輸只愛勝利的總裁獲得亞軍是最大的恥辱。

對於賽後C羅拒絕佩戴象徵亞軍的銀牌，你怎麼看？
