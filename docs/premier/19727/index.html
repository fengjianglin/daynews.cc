范戴克和喬·戈麥斯很好·尤文三前鋒·本澤馬想贏西甲

2019-12-17 662條評論 2,543次閱讀 1631人點贊

克洛普說，范戴克和喬·戈麥斯可以一起踢得「非常好」

利物浦主教練克洛普說，范戴克和喬·戈麥斯在利物浦防守受傷的情況下可以一起踢得「非常好」。

伴隨著馬蒂普和洛夫倫的受傷，范戴克和喬·戈麥斯在周六英超聯賽2-0戰勝沃特福德的比賽中築起了利物浦的堅固防禦。

克洛普說，他經常和喬·戈麥斯交談，並相信這位英格蘭國腳和范戴克的合作關係會很好。

喬·戈麥斯和范戴克幫助他們的球隊擊敗沃特福德，利物浦在英超聯賽中領先10分。

范戴克

薩里排除了C羅、迪巴拉和伊瓜因一起上場的可能性

尤文圖斯主教練薩里從C羅、迪巴拉和伊瓜因一起踢球中獲益匪淺，然後警告說這種事情可能不會經常發生。

三前鋒同時上場的選擇得到了回報，C羅在3-1戰勝烏迪內斯的比賽中梅開二度，其中還有一個進球是伊瓜因打進的。至少暫時來說，這個結果讓尤文暫時重回聯賽榜首。

然而，薩里暗示，正是由於烏迪內斯保守的比賽風格，他才決定將三名前鋒放在一起首發。這對我們來說是一個很有價值的選擇，但是我們不能一直使用。

本澤馬:我想和皇家馬德里一起贏得西甲

本澤馬說，在對陣瓦倫西亞的比賽中拯救了球隊之後，他希望在這個賽季帶領皇馬獲得西甲的榮譽。

由於本澤馬最後時刻的扳平比分，馬德里周一在瓦倫西亞戲劇性地將比分打成1-1平。

本澤馬說:「我想贏得這個聯賽冠軍，如果我能在比賽上幫助我的隊友，我會做到的。」

本澤馬本賽季現階段和梅西的西甲聯賽進球數相同。
