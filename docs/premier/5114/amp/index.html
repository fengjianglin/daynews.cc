今日比賽預測：國際米蘭-羅馬，國米狀態火爆而羅馬錶現飄忽

天天要聞
2019-12-06

國際米蘭上輪聯賽坐鎮梅阿查球場以2-1的比分擊敗史帕爾，由於尤文意外本森索羅逼平，紅黑軍團因此成功反超1分登頂榜首，前14輪意甲聯賽取得12場勝利，同時創下隊史同期最佳戰績。從斑馬軍團搶回榜首寶座後，國際米蘭如今需要保住自己領頭羊的位置。

聯賽至今，國際米蘭場均轟入2.21球，火力僅次於亞特蘭大和拉齊奧。自2001-02賽季後，球隊首次出現前14輪聯賽過後陣中有2名球員進球數達到8球或以上，其中上輪梅開二度的勞塔羅近3場比賽攻入5球，狀態火爆。

但上輪主帥孔蒂並未如外媒預測一樣讓小將埃斯波西托首發出戰，他堅持使用勞塔羅和盧卡庫這對前鋒組合，在密集的賽程下，兩人體能方面消耗極大。

而由於加利亞爾迪尼受傷，國際米蘭目前只有布羅佐維奇、巴萊羅和貝西諾3名中場能夠出戰，中場嚴重缺人。不過，布羅佐維奇最近狀態上佳，上輪繼續交出助攻，職業生涯第二次連續兩輪意甲聯賽送出助攻。

羅馬上輪同樣獲得一場勝利，他們客場以3-1的比分擊敗了升班馬維羅納，這場勝利也是紅狼軍團在意甲的第400場客場勝利，不過由於卡利亞里全取3分，球隊依然因凈勝球劣勢未能進入前四。

皇牌前鋒哲科這周一直因感冒發燒缺席訓練，本場比賽至今仍未確定能否出戰，一旦缺席對羅馬進攻端影響頗大。好消息是，新狼王扎尼奧洛停賽一輪後復出，他是哲科之外紅狼陣中第二號威脅球員。

姆希塔良在對陣維羅納的比賽中替補上陣，並在最後時刻攻入1球；由於帕斯托雷和小克魯伊維特傷缺，他可能會被安排在首發陣容中，而阿美尼亞國腳表示自己想要在意甲和羅馬證明自己。

扎帕科斯塔一直處於養傷狀態，而弗洛倫齊的表現也未能說服主帥豐塞卡，羅馬右後衛是迫切需要加強的地方。據意媒報道，羅馬已經將那不勒斯後衛希薩伊作為球隊一月份轉會窗口的優先引援目標。

最近4兩場客場，羅馬錄得2勝2負，表現飄忽。而紅狼雖然最近10次作客梅阿查擁有8成的不敗率，但近5次僅贏過國際米蘭1次，如今紅黑軍團士氣正盛，此役未必能夠全身而退。

  • 國際米蘭最近10場賽事勝率為80%，贏盤率為70%，贏盤能力出色。
  • 國際米蘭最近10場各項賽事僅1場雙方進球總數少於2.5球。
  • 國際米蘭是目前意甲失球第2少的球隊（14輪失13球）。
  • 羅馬最近10場賽事勝率為60%，贏盤率為60%，贏盤能力不俗。
  • 羅馬最近8場各項賽事有6場開出大球。
  • 羅馬最近10次作客對陣國際米蘭擁有8成的不敗率。

    • 綜上所述，這場比賽應該是國米可以贏下比賽，羅馬第一射手哲科上場成謎，且最近羅馬錶現飄忽，國米又是主場作戰。國際米蘭2-1羅馬，國際米蘭3-0羅馬，國際米蘭3-1羅馬。
