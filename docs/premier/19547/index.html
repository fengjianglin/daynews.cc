瓜穆相看：穆里尼奧在皇馬與瓜迪奧拉在巴薩，簽約的前7個球員

2019-12-18 574條評論 1,646次閱讀 495人點贊

瓜穆相看，當年西甲，瓜迪奧拉與穆里尼奧兩人的對決，讓整個世界為之側目，如今二人都先後離開西甲。今天盤點一下他們兩人在打造球隊過程中，購入的前7個球員。

瓜迪奧拉作為巴薩主教練簽約前七名球員：（這也是瓜迪奧拉上任第一年簽下7個球員）：

凱塔：

凱塔在塞維利亞精彩的發揮i，吸引了瓜迪奧拉。瓜迪奧拉上任第一筆簽約，就是激活非洲球員凱塔的違約金，1400萬英鎊，將凱塔帶到諾坎普。凱塔在加泰羅尼亞的四個賽季中出場188次，凱塔與瓜迪奧拉在同一年離開巴薩。離開巴薩的凱塔曾經為大連阿爾賓、巴倫西亞、羅馬和埃爾傑什等球隊效力過。如今凱塔已經退役。

皮克：

瓜迪奧拉在這一年，從曼聯弗格森手裡買來打不上主力的皮克。只用了區區500萬英鎊。皮克回歸巴薩後，一直是巴薩的主力後衛，效力至今。這一筆簽約，是瓜迪奧拉給巴薩留下的寶貴財富。

平托：

從塞爾塔轉會到巴薩的平托，在巴薩很少有出場的機會。儘管平托曾經獲得西甲最佳守門的榮譽。但巴薩的巴爾德斯太出色。平托只能擔當替補。讓平托為世人所知的是平托的口哨事件。

巴薩2-0 哥本哈根的歐冠聯賽中，平托用 吹口哨的方式模擬主裁判的哨聲，破壞了對方前鋒的進攻。這違背了公平比賽的原則，讓平托一時間成為輿論討伐的焦點。

卡塞雷斯：

在簽下皮克的同時，瓜迪奧拉同樣用1500萬英鎊簽下了中後衛卡塞雷斯。由於皮克的出色發揮，卡塞雷斯在巴薩三年時間，只出場了38次。之後500萬轉賣給塞維利亞。

丹尼·阿爾維斯：

瓜迪奧拉又一次成功的簽約。丹尼·阿爾維斯在塞維利亞出色的發揮，瓜迪奧拉繼凱塔之後，用3000萬英鎊，從塞維利亞帶走阿爾維斯。阿爾維斯在巴薩八年，一直是巴薩主力邊後衛。

亞歷山大·赫萊布：

亞歷山大·赫萊布在阿森納表現平庸，並且有傷在身，但瓜迪奧拉依然簽下了這位中場球員。事實上，赫萊布在巴薩待了一年後，就被瓜迪奧拉長期租借在外。直到合同結束。

恩里克：

巴西中後衛恩里克是瓜迪奧拉巴薩第一年最後一筆簽約。瓜迪奧拉當時對巴薩後防非常不放心，在第一年簽下了4名後衛，其中三名是中衛。事實上恩里克也是多餘簽約。至始至終都沒在巴薩出場過。

穆里尼奧從國米入駐皇家馬德里後。穆里尼奧為了打造能與巴薩抗衡的球隊。也進行一系列的簽約。

塞爾吉奧·卡納萊斯

穆里尼奧用450萬簽下了這位19歲的塞爾吉奧·卡納萊斯。第一年出場15次，之後租借給瓦倫西亞，然後再也沒有回到皇馬，永遠的賣給瓦倫西亞。

迪馬利亞：

穆里尼奧從本菲卡帶了阿根廷人迪馬利亞。只花費2500萬英鎊。迪馬利亞在皇馬成為助攻王。皇馬生涯是迪馬利亞在歐洲足壇高光的時期。

佩德羅·萊昂

穆里尼奧簽下23歲的萊昂，第一年萊昂在穆里尼奧手下出場14次，之後就被穆里尼奧徹底的拋棄。

赫迪拉：

對於聰明勤奮的中場球員，薩米·赫迪拉成為穆里尼奧的最愛。這位德國國腳在穆里尼奧三個賽季中，他每一個賽季都出場40多次。

里卡多·卡瓦略

里卡多·卡瓦略在波爾圖，切爾西與穆里尼奧共事，當穆里尼奧來到皇馬的時候。里卡多·卡瓦略在次跟隨穆里尼奧來到皇馬。卡瓦略此次更多的是跟著穆里尼奧學習如何管理球隊。如今里卡多·卡瓦略已經成為一名助教。

梅蘇特·厄齊爾

穆里尼奧花費1500萬英鎊，從德國帶來厄齊爾。事實上這一次簽約最為關鍵。厄齊爾在穆里尼奧手下成為絕對主力，每個賽季的助攻在整個歐洲名利前茅。

​努里·沙欣：

穆里尼奧第二年花費1000萬英鎊簽下沙欣，然而由於傷病，沙欣在皇馬沒有表現機會，之後被租借到利物浦，多特。但沙欣依然被傷病困擾。沒有亮眼的發揮。
