意甲今日頭版：新梅西懲罰國米 加圖索今天上任那不勒斯

2019-12-11 300條評論 2,230次閱讀 679人點贊

直播吧12月11日訊 米蘭體育報、羅馬體育報、都靈體育報是義大利的三大體育報，今天我們來看看它們各自的頭版頭條都帶來了哪些內容。

《米蘭體育報》

1.「出局！」孔蒂遭淘汰，安帥贏球仍分手。國米揮霍機會被巴薩二隊淘汰，盧卡庫進球但也浪費了三個進球。紅藍軍團在梅阿查擊敗國米2-1，17歲的法蒂打入制勝球。現在藍黑軍團撤退到歐聯杯。教練悲傷：「遺憾，非常失望。」

2.安切洛蒂遭解僱，加圖索將接手那不勒斯。那不勒斯4-0亨克晉級16強，德勞倫蒂斯解僱安帥，後者可能去阿森納。加圖索今天上任。

3.帕普-戈麥斯回到烏克蘭，亞特蘭大為歷史而戰。

4.馬爾蒂尼嘗試引進托迪博，球員違約金1.5億歐，米蘭希望巴薩大減價。

《羅馬體育報》

1.「安東尼奧不再晉級。」國米1-2巴薩，降到歐聯杯。紅藍軍團沒有上全部主力，佩雷斯首開紀錄，盧卡庫扳平，之後藍黑軍團浪費三次機會，17歲的安蘇-法蒂打入制勝球。孔蒂：「失誤太多，我們非常失望。」

2.「安帥晉級然後下課。」那不勒斯晉級1/8決賽，今天開啟加圖索時代。卡爾洛在那不勒斯的冒險以4-0亨克畫上句號，米利克帽子戲法。

3.尤文在勒沃庫森隊訓練，亞特蘭大做客烏克蘭尋求壯舉。

《都靈體育報》

1.「新梅西懲罰國米。」安蘇-法蒂，17歲，替補出場決定比賽，巴薩2-1。勞塔羅表現優秀，盧卡庫浪費機會：藍黑軍團降入歐聯杯。多特蒙德晉級。

2.「尤文，你要為阿萊格里而贏。」今晚小組賽最後一輪做客勒沃庫森，一旦贏球，歐冠獎金將超過阿萊格里的最後一年。

3.「安切洛蒂下課！」那不勒斯摧毀亨克，德勞倫蒂斯引進加圖索。

4.「加油女神！」加斯佩里尼：「如果我們贏球，我們將創造歷史。」

（我叫達芬奇）
