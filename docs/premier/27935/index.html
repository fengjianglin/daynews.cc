周六八場賽事推薦：國米往績碾壓，紅牛以攻代守奧格攻防強勢

2019-12-22 532條評論 2,397次閱讀 386人點贊

周日020 德國甲級聯賽 萊比錫紅牛 VS 奧格斯堡

萊比錫紅牛近況：

萊比錫紅牛隊在本周三凌晨進行的德甲聯賽中客場3比3逼平多特蒙德隊，在正式比賽中連續十一場比賽不敗。

從球隊近況來看，萊比錫紅牛隊在最近七場正式比賽中取得4勝3平進20球失11球，並在本賽季十個主場比賽中取得5勝3平2負進24球失13球，此役必會憑藉主場強勢進攻全力取勝反彈。

本周六主場迎來奧格斯堡隊，萊比錫紅牛隊主帥納格爾斯曼表示目前狀態非常出色的奧格斯堡隊在最近比賽中創造出很多機會，防守非常嚴密的他們很少讓給對手留下空間，在最近兩輪聯賽中展現不錯的反擊能力，此役紅牛隊必須要憑藉快速逼搶不讓對手發起攻防轉換，並憑藉出色攻防表現盡全力在主場取得勝利。

從球隊陣容情況來看，主力中衛奧爾班和科納特、主力中前衛坎普爾正在進行康復治療，其他球員都可以出場。

奧格斯堡近況：

奧格斯堡隊在本周三凌晨進行的德甲聯賽中主場3比0完勝杜塞爾多夫隊，在正式比賽中取得三連勝。

從球隊近況來看，奧格斯堡隊在最近五場正式比賽中取得4勝1平進14球失4球，並在本賽季九個客場比賽中取得2勝3平4負進12球失19球，此役將會憑藉頑強攻防全力阻擊強敵。

本周六客場挑戰萊比錫紅牛隊，奧格斯堡隊主帥施密特表示目前狀態最出色的兩隊都踢出賞心悅目的足球，他們始終在努力爭奪聯賽半程冠軍，明顯不被看好的奧格斯堡隊必須要展現穩定防守和定位球威脅，並憑藉出色表現力爭在客場取得一個好結果。

從球隊陣容情況來看，替補中前衛Fe.格策正在進行康復治療，主力右後衛福拉姆貝格、主力進攻中場格魯埃佐、主力中鋒芬恩博加松因傷缺陣，半主力中衛蘇希正在進行恢復性訓練，主力後腰R.赫迪拉累計五張黃牌停賽，半主力中前衛莫拉維克傷愈歸隊。

交戰記錄：

主隊在雙方最近6次交戰中取得1勝4平1負進5球失4球，在2019年3月進行的德甲主場比賽中爆冷0比0被對手逼平。萊比錫紅牛隊近10場7勝3平0負(其中主場3勝1平0負)，奧格斯堡隊近10場6勝3平1負(其中客場3勝2平0負)。從最近狀態來看，客隊的狀態稍好一些。

賽果分析：讓負

周六036 意甲 國際米蘭VS熱那亞

國米近況：

近10場戰績：6勝2平2負，進18球，失11球

國米上輪客場1比1戰平佛羅倫薩，終場前遭遇壓哨扳平，這場平局導致國米積分被尤文圖斯追平。本周末，國米將在主場迎戰排名第19位的熱那亞，上賽季國米主客場對其完成雙殺，總比分達到9比0之巨，而自2007年以來，國米最近12次主場迎戰熱那亞取得了9勝3平0負的碾壓優勢，自2013年以來已在聯賽主場對熱那亞完成了六連勝，如今再戰，藍黑軍團無疑仍是實力明顯佔優的一方。人員方面，桑切斯、小森西、阿薩莫阿、巴雷拉、加利亞爾迪尼傷休，勞塔羅-馬丁內斯、布羅佐維奇累計黃牌停賽，孔蒂排兵布陣會有一定難度。

熱那亞近況：

近10場戰績：2勝3平5負，進12球，失15球

在場面明顯佔優的情況下，熱那亞上輪卻在主場0比1不敵同城死敵桑普多利亞，自今年10月底以來最近7輪0勝3平4負，主帥莫塔正承受著巨大壓力，但好就好在，俱樂部高層目前依然力挺他，可話說回來，已經深陷保級泥潭的熱那亞亟待反彈。然而，本輪熱那亞卻會前往梅阿查球場挑戰國米，這是一場強弱分明的對決，上一次從梅阿查球場全身而退得追溯至七年前，而本輪雖說會有阿古德洛與潘德夫解禁回歸，但勒拉熱爾、薩帕塔、誇梅的傷休缺席還是會對排兵布陣造成負面影響。

交鋒戰績：兩隊近10次交鋒，國際米蘭6勝0平4負，進18球，失8球

賽果推薦：勝
