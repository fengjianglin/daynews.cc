佐拉：預感那不勒斯能2-1贏利物浦，這是球隊走出困境的機會

2019-11-26 0條評論 14次閱讀 0人點贊

直播吧11月26日訊 北京時間11月28日凌晨4點，那不勒斯將在歐冠小組賽客場挑戰衛冕冠軍利物浦，義大利名宿佐拉預測那不勒斯將會在比賽中擊敗利物浦，同時，這將球隊擺脫困境的一次機會。

佐拉在接受《每日郵報》採訪時說：「現在對處於困境的那不勒斯來說不是一個好時機，但像上一回合擊敗利物浦的夜晚可以重現。」

「失去了因西涅，對那不勒斯是一個真正的打擊，但那不勒斯已經證明他們知道如何擊敗利物浦。紅軍的進攻確實很強，但他們的防守並不是無堅不摧的。」

「憑藉球員的創造力，那不勒斯會給任何球隊帶來威脅。安菲爾德是世界上獨一無二的體育場，它會給球隊注入額外的動力。這樣的比賽，很難解釋賽場內外的氣氛，我覺得那不勒斯以冷靜的心態面對利物浦是很重要的。」

「然而，那不勒斯現在並沒有足夠的冷靜，因為俱樂部剛剛對球員們在訓練中的叛變處以罰款，甚至有傳言說主帥安切洛蒂可能會被解僱或辭職。」

「這就是現在足球的本質，它經常在社交媒體上被炒作。面對困難是每個俱樂部的一部分，但重要的是如何走出困境。」

「明天的比賽可以看作是那不勒斯走出困境的一次機會。我的直覺告訴我，他們會以2比1獲勝。」

（時刻）
