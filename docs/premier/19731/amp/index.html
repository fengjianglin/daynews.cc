英超雙雄對決馬德里雙雄！到底誰才是世界第一聯賽？

天天要聞
2019-12-17

歐冠16強抽籤結果新鮮出爐，最引人關注的當屬英超雙雄碰上馬德里雙雄，具體來說，就是曼城抽到了皇馬，利物浦抽到了馬競。對於英超雙雄來說，這絕對是下下籤，小組賽拼死拼活，居然命運給安排了馬德里雙雄，這可是歐冠賽場上最難纏的兩個對手。皇馬才剛剛被終結了五年四冠的輝煌旅程，馬競也兩進歐冠決賽，只能感嘆英超雙雄運氣不佳了。

至於馬德里雙雄，抽到什麼樣的對決也沒有抱怨的理由，都是小組第二的身份，完全沒有挑對手的資格。

梅西和C羅的球隊都抽到了上上籤。尤文圖斯抽到了法甲的里昂，對手剛剛損兵折將，進攻核心德佩膝蓋十字韌帶撕裂，至少缺席6個月，肯定缺席里昂本賽季的所有比賽，而且還可能會影響到荷蘭國家隊明年的歐洲杯。德佩想要趕上明年的歐洲杯，只能祈禱奇蹟發生了。德佩本賽季表現出色，在32場國家隊和俱樂部各項賽事中打進23球，平均每74分鐘就貢獻1次進球或助攻。對於尤文圖斯來說，面對缺少德佩的里昂，基本上屬於保送。

巴薩抽到了那不勒斯，這組對決更是沒有任何懸念。那不勒斯本賽季隊內「兵變」，主帥安切洛蒂已經被加圖索取代了，絕大部分的一線隊球員都和俱樂部有不可調和的矛盾，那不勒斯看上去離散夥已經不遠了。巴塞羅那這時候算是撿了一個大便宜，面對一支隊伍團結出問題的球隊，只要自己不瘋狂送大禮，晉級應該不成問題。

最想看的對決當然是曼城和皇馬，當瓜迪奧拉和皇馬聯繫在一起，這就意味著不缺看點。自從離開巴薩之後，瓜迪奧拉一直渴望在歐冠的舞台實現突破，可不管是之前帶隊德甲霸主尤文圖斯還是現在的曼城，瓜迪奧拉的球隊始終在歐冠的舞台差一口氣。這個賽季，由於遭遇了嚴重的傷病，曼城在聯賽已經掉隊，17輪過後已經落後了利物浦14分，雖然還沒有到放棄聯賽的地步，但適當的把重心轉移到歐冠，爭取在歐冠的舞台有所突破，這是比較明智的選擇。可是，命運偏偏給瓜迪奧拉開了一個不大不小的玩笑，它把歐冠的絕對王者送到了曼城的面前。皇馬可是才完成了5年4冠，俱樂部榮譽室內的13做沉甸甸歐冠獎盃一字排開，這些榮譽記錄著歐冠巨人的每一步。雖然本賽季的皇馬已經進入了重建的過程，但是，年輕人的茁壯成長已經讓皇馬回到了正軌，再加上齊達內的「玄學護體」，曼城想要過關可不那麼容易。

利物浦也抽到了自己最不想看到的對手，馬競就是屬於那種切不動、煮不熟、嚼不爛的滾刀肉，在歐冠的舞台上可以做到和誰都能「五五開」，這支西蒙尼率領的鐵血之師，他們可以做到和誰交手都能有五成出線的幾率。馬競這個賽季的最大問題就是沒有球員能夠穩定進球，17輪聯賽他們收穫了8場平局，比勝場數還要多一場。也就是說，馬競想贏球很難，可是對手想在馬競面前贏球，那更是難上加難。克洛普領銜的利物浦向來以善打硬仗著稱，可是對上了馬競，還是五五開的局面。矛與盾之間的強強對話，誰放的錯誤更少，誰就更有可能笑道最後。

穆里尼奧的熱刺也抽到了上上籤，他們八分之一決賽的對手將是德甲球隊RB萊比錫，這組對決最大的看點可能是是雙方的主教練，穆里尼奧VS納格爾斯曼，在很多專家和球迷的眼中，納格爾斯曼就是德國版的穆里尼奧。值得一提的是，納格爾斯曼只有32歲，這比很多球員的年齡還要小，但是，RB萊比錫的主帥已經廣受讚譽，被認為是當今歐洲足壇最牛的年輕主帥，很多豪門都在密切關注納格爾斯曼的動向。熱刺擁有上賽季歐冠決賽的班底，過去的夏天還進行了一定的補強，從雙方人員的實力以及名氣對比來看，穆里尼奧的球隊似乎是更大的晉級熱門。而對於納格爾斯曼來說，這也是證明自己的最好機會，作為一位主教練，還有什麼比擊敗穆里尼奧更讓人激動呢。

當然了，每年的歐冠抽籤都會有一組平民對決，這也是歐足聯的某種「公平」體現，有些像「陰謀論」的東西，大家討論討論就可以了，說不清道不明，反正就是存在，也不需要太多的解釋。亞特蘭大（意甲）vs 瓦倫西亞（西甲）是那兩個幸運兒。米蘭城的中立球迷，請開始歡呼吧，今年的聖西羅，你們終於可以有歐冠淘汰賽可以看了。而對於紅黑球迷還是藍黑球迷來說，不一樣的藍黑出現在聖西羅球場，必定給他們精神上更大的刺激。

隨著抽籤儀式的結束，真正的歐冠比賽要到來了，每一場比賽都是生死對決，都會決定很多人的命運。有些人在歐冠的舞台上成就傳奇，有些人則成為倒霉蛋，被冠上失敗者的頭銜。16強抽籤最大的看點就是英超雙雄迎戰馬德里雙雄，這兩組對決的結果也能間接的證明誰才是當今世界第一聯賽。競技體育，最終還是要靠成績來說話，論商業開發，英超當然是天下第一，可歐冠的成績始終是硬傷。即使利物浦連續2年都進入歐冠決賽，他們在歐足聯俱樂部的積分都只排在第8，前四還是由西甲三強和拜仁包攬，「皇薩仁」在歐冠的統治格局仍然沒有被打破。

期待吧，真正的高手對決將在明年2月份到來，歐冠淘汰賽才是享受足球最好的舞台。
