讓皇馬“嚇得尿褲子”的天使之翼，如今成自家球迷嫌棄的談判鬼才

天天要聞
2019-12-22

1992年11月，一個尋常的日子，一位15歲的波黑小夥子離開了自己戰火紛飛的祖國，去往德國，簽約加盟漢堡U19青年隊，從此開啟了自己的足球職業生涯，他就是哈桑·薩利哈米季奇。

在漢堡，薩利很快就被恩師馬加特提拔至一線隊，為球隊攻城拔寨，並迅速佔穩主力位置。而憑藉自己的出色表現，他也在三年後如願自由加盟拜仁慕尼黑。

薩利哈米季奇在漢堡期間

「南大王」的「天使之翼」

加盟拜仁慕尼黑之後，時任主帥希斯菲爾德便給了薩利充足的上場時間，而他的表現也確實配得上教練的信任。

儘管薩利身材並不高大，卻有著鐵血意志和鬥志，不知疲倦的奔跑、極強的爆發力，再加上嫻熟的過人動作，他成了當時所有左後衛的噩夢，時任皇馬左後衛的羅伯托·卡洛斯也不得不稱讚他是當時世界上最出色的球員之一，而「天使之翼」的稱號也由此得來。

穩定的發揮給了薩利極大的自信，而自信也確實源於其出色的技術。在2000-01年歐冠期間，拜仁慕尼黑一路順利殺入半決賽，而他們的對手正是皇家馬德里。賽前，薩利的一句「我會讓他們嚇得尿褲子」確實展現了他強大的自信。

薩利哈米季奇與拜仁奪得歐冠冠軍

拜仁最終主客場雙殺皇馬，成功晉級決賽，並最終捧得大耳朵杯。薩利雖無進球，卻給到皇馬後防線極大的壓力，成功幫隊友牽制。

在經久的比賽中，薩利同樣擺脫不了傷病的困擾。而由於傷病的原因，他的主力位置也受到動搖，並在2007年加盟尤文圖斯。

在拜仁效力的九年里，薩利哈米季奇共為球隊征戰200多場，貢獻了30多個進球，同時幫助球隊獲得6次德甲冠軍、4次杯賽冠軍、2000-01賽季的歐冠冠軍、豐田杯冠軍等，對球隊給予了極大的貢獻。

「天使之翼」在外飛翔

2007年的夏天，薩利哈米季奇自由轉會至尤文圖斯。當時尤文剛經歷了「電話門」風波，薩利是「老婦人」得以依賴的重要球員。

在尤文圖斯的4年中，薩利前三個賽季都是球隊的主力，但在第四賽季時，由於尤文圖斯意在重建、加速球隊年輕化，34歲的波黑球星不得不接受替補身份，並於賽季後選擇離開。

薩利哈米季奇在尤文圖斯期間

在離開德甲四年後，薩利應恩師馬加特的招攬，重新回到德甲並加盟沃爾夫斯堡。

「薩利是一位水平相當高的球員，他可以打多個位置。他的到來，使我們的陣容更具生命力。我敢肯定，他將用其經驗為球隊做出最大的貢獻。」

在簽下薩利後，馬加特高興的對媒體說。薩利也十分客氣：

「對我來說，這不同於為拜仁和尤文效力。是馬加特給了我繼續職業生涯的機會，我一定不會辜負他。在沃爾夫斯堡我將擁有全新的職業履歷，而這一切均歸功於菲利克斯·馬加特。」

薩利哈米季奇在沃爾夫斯堡期間

師徒的對話確實可以看出兩人的深厚感情，而薩利在恩師手下也確實不負眾望，貢獻著自己高光的表現。但由於年齡已過巔峰，在加盟沃爾夫斯堡一個賽季後，薩利最終在2012年選擇退役。

重回安聯球場

在退役之後，薩利哈米季奇並沒有選擇成為教練，而是一直在為拜仁開拓海外市場和提高國際影響力，並在2017年和當時贏得歐冠決賽的隊友利扎拉祖、埃爾伯同時擔任拜仁形象大使。

而在同年7月份，薩利出任拜仁體育主管一職，負責一線隊的管理和轉會操作。會說英語、德語和義大利語等五國語言的他，同時私下又幽默風趣，和球員關係緊密，對拜仁肯定會有很大的幫助。

薩利哈米季奇擔任拜仁形象大使

在拜仁低迷時期，薩利的工作以及性格確實幫助球隊走了出來，在海因克斯退出後，主動推薦昔日隊友科瓦奇出任新帥，並在執教初期取得良好的成績。

在「金元足球」橫行的年代，拜仁慕尼黑的轉會操作確實屬於一股清流，而這也正是拜仁高層的意志體現。在薩利任職以來，雖然錯過一些大牌明星的加盟引得不少爭議，被拜仁球迷成為「談判鬼才」。但確實不乏有一些亮眼的操作，如今在球隊站穩主力的格雷茨卡、格納布里、戴維斯、帕瓦爾和埃爾南德斯等，都是薩利哈米季奇親手操作完成，著實有一種「花小錢辦大事」的風格。

薩利與科瓦奇

開啟新的征程

在今年拜仁慕尼黑的會員年度大會上，與拜仁結緣49年的俱樂部主席赫內斯宣布退休，並由阿迪達斯前首席執行官赫爾伯特·海納接任。而薩利在赫內斯的助推下，也將與2002年7月接過拜仁空閑四年之久的體育董事一職，正式進入董事會。屆時，他將有更大的決策權來進行工作。

同時，薩利哈米季奇在拜仁時期的同期隊友，卡恩也將於明年加入董事會，並在2021年12月31日魯梅尼格的合同到期後，成為新的董事會主席。

至此，未來拜仁的高層領導已經清晰，薩利如何和他昔日隊友卡恩繼續帶領拜仁前進，讓我們拭目以待。
