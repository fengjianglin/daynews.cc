盤點近10年歐冠經典比賽：紅軍巴薩絕地反擊領銜

2019-12-19 2,491條評論 3,784次閱讀 2169人點贊

21世紀第二個十年即將結束，本賽季的歐冠比賽也到了淘汰賽階段，在等待歐冠比賽開始的同時，小編為大家盤點了近10年來歐冠最經典的比賽（按時間順序，排名不分先後），來看看有沒有讓你熱血沸騰的那一場吧。

2010年4月28日：巴薩1-0國際米蘭（歐冠半決賽次回合，總比分2-3）

這場比賽算得上是穆里尼奧教練生涯的一次偉大勝利。那支贏得歷史性三冠王的國際米蘭完美地體現出了穆里尼奧所強調的組織性、紀律性和務實。魔力鳥的比賽總是看起來是如此無趣，但又是如此讓人興奮。

穆里尼奧告訴球員們要保持陣型，縱深防守，在距離球門不到25米的位置上向巴薩施壓。斯內德如是評價這位主教練：「他給了我們其他人無法給我們的信息，這就是穆里尼奧的力量。」

那個夜晚，國際米蘭展現出驚人的統一性，即便莫塔在開場不到半小時，便被罰出場外。但穆里尼奧的國際米蘭頂住了巴薩一波又一波傳控進攻，即便皮克在最後時刻攻入一粒進球，笑到最後的還是藍黑軍團。穆里尼奧將這場比賽稱為 「職業生涯中最美麗的失敗」。

魔力鳥風光無限

2012年4月24日：巴薩2-2切爾西（歐冠半決賽次回合，總比分2-3）

十年間最讓人驚訝和最印象深刻的歐冠勝利，無疑是由切爾西製造的。切爾西在首回合以1-0的比分搶佔先機，但他們做客諾坎普卻承受著巨大的壓力。上半場特里被罰出場外，並且被巴薩攻入兩球。原本看起來切爾西已然沒有晉級決賽的希望，但誰也沒有料到拉米雷斯能夠取得一粒如此精彩的進球，同時誰也沒有料到切爾西防守端的表現會變得如此英勇。托雷斯則是在比賽最後時刻攻入了一粒讓人久久無法忘懷的進球——切爾西扳平了比分，以3-2的總比分淘汰巴薩，晉級歐冠決賽。

2014年4月29日：拜仁0-4皇馬（歐冠半決賽次回合，總比分0-5）

瓜迪奧拉將那個晚上描述為「我執教生涯中最糟糕的一天」。這是他執教拜仁的首個賽季，也是他離開巴薩之後，第一次栽了個大跟頭。

首回合做客伯納烏，拜仁0-1輸掉了比賽。所以在次回合比賽中，瓜迪奧拉試圖利用靈活的3-4-3陣型來扭轉局勢。但隨後他又失去了勇氣，切換到了4-2-3-1陣型，最後他又將陣型切換成了從未使用過的4-2-4陣型。比賽開始之前，瓜迪奧拉就要求球員們「全力以赴」。然而這場比賽最後還是變成了一場災難。皇馬控制了場上局勢，並將拜仁打成了篩子。拉莫斯兩度利用定位球破門，C羅在中場休息之前攻入一球，隨後又自由球洞穿拜仁大門。

賽後，瓜迪奧拉反思道：「我完全搞錯了，這是一個巨大的失敗。」

2015年5月6日：巴薩3-0拜仁（歐冠半決賽首回合，總比分5-3）

梅西在歐冠聯賽中有很多精彩的表現，而這場3-0擊敗拜仁的比賽，可能就是其中最值得記憶的比賽。瓜迪奧拉率領著一支幾乎達到了巔峰的拜仁出征諾坎普，在比賽大部分時間裡，瓜迪奧拉的戰術策略看起來都是奏效的，但到了比賽最後13分鐘，似乎情況發生了變化——梅西接管了比賽。

他在距離球門25碼處攻入一球，幫助巴薩1-0取得領先。接著，他又攻入了自己的經典之作——用犯罪式過人的方式突破了博阿滕，然後梅開二度。隨後，他又在比賽最後時刻用一記漂亮的傳球，內馬爾進球得分。

MSN組合

2015年6月6日：尤文圖斯1-3巴薩（歐冠決賽）

這場比賽可以說是「MSN」組合的巔峰時刻——面對阿萊格里率領的尤文圖斯，梅西、蘇亞雷斯和內馬爾將自己的職業道德和組織能力、個人天賦完美的結合到了一起。

開場不到4分鐘，拉基蒂奇便為巴薩取得領先。儘管面對尤文圖斯的瘋狂反擊，擁有夢幻般開局的巴薩在很長一段時間裡都承受著巨大的壓力，但蘇亞雷斯和內馬爾的進球，最終還是幫助球隊鎖定了勝局。同時，這場比賽是哈維代表巴薩出戰的最後一場比賽，同時也是他在歐冠決賽中取得的第四場勝利，巴薩用一座歐冠獎盃為球隊功勛告別。

2016年5月3日：拜仁2-1馬競（歐冠半決賽次回合，總比分2-2，馬競憑藉客場進球數優勢晉級）

瓜迪奧拉率領拜仁三次出征歐冠聯賽，三次失利都是如此令人遺憾，而最後一次失利或許是其中最令人遺憾的一次。那時的拜仁正處於一個巔峰狀態，但他們遇到了迭戈-西蒙尼的馬競。阿根廷人治下的「鐵血」馬競正不顧一切地想要彌補2014年歐冠決賽所留下的痛苦。

首回合比賽中，馬競1-0取得領先，但客戰拜仁，顯然不是一件容易的事情。坐鎮主場大V拜仁採用了一種只有他們才能夠踢出來的攻勢足球。他們用凌厲的攻勢死死地壓制著馬競，迭戈-西蒙尼的球隊遭受到了前所未有的痛苦。哈維-阿隆索在半個小時內就幫助拜仁將總比分扳平。三分鐘之後，托馬斯-穆勒原本有機會將總比分反超，但奧布拉克出眾的表現，拯救了馬競。

格列茲曼在下半場的進球，其實也是馬競在那場比賽中為數不多的進攻機會之一。雖然馬競的射門次數並不多，但格列茲曼的進球對於拜仁來說，顯然是致命的——比賽第74分鐘，萊萬多夫斯基攻入一球，但最終拜仁還是因為客場進球數的劣勢而被淘汰出局。

瓜迪奧拉希望能夠率領拜仁在歐冠聯賽中取得更好的成績，但這或許會成為他永遠的遺憾。

2017年3月8日：巴薩6-1巴黎聖日耳曼（八分之一決賽次回合，總比分6-5）

這場比賽算得上是現代歐冠聯賽歷史中最經典的比賽之一，巴黎聖日耳曼在首回合比賽中有著出眾的表現，他們以4-0的比分擊敗了巴薩，並第一次在歐冠聯賽中展現出了冠軍的姿態。

次回合比賽中，當年巴薩3-0領先之後，卡瓦尼攻入了一球（其實巴黎聖日耳曼還有另外兩次破門得分的機會，但他們最終錯過）。在比賽還剩下兩分半鐘的時候，巴薩仍需要三粒進球才能夠取得最終的勝利，而他們做到了：內馬爾主罰自由球破門，內馬爾主罰點球命中，然後內馬爾助攻塞爾吉-羅貝托——在傷停補時進行到4分41秒之時，塞爾吉-羅貝托攻入制勝進球，巴薩完成驚天大逆轉。

同時，這場驚天逆轉也產生了一些歷史性的影響：巴黎聖日耳曼在接下來的夏季轉會期，以1.98億英鎊的價格得到了內馬爾。

2019年4月17日：曼城4-3熱刺（歐冠四分之一決賽次回合，總比分4-4，熱刺憑藉客場進球數優勢晉級）

這場比賽的劇情可謂跌宕起伏，沒有任何一個人能夠在終場哨響起之前，預測正確誰是最後的贏家。首回合比賽中，熱刺1-0擊敗曼城，然而次回合比賽開始沒多久，他們就落後於對手——斯特林搓射遠角破門。但熱刺在落後沒多久，又憑藉孫興慜的兩粒進球，再次將總比分超出。不過很快，貝爾納多-席爾瓦就將總比分再度扳平。比賽前11分鐘，雙方就聯手為球迷奉獻了四粒進球。

這樣的情況，也是讓曼城的進攻變得愈發兇猛，他們試圖將這場比賽把握在自己手中。下半場14分鐘，曼城就將比分改寫為4-2。隨後，略倫特的進球為熱刺追回一球，將場上比分改寫為4-3。當然，曼城並沒有放棄，只不過斯特林在傷停補時階段的進球，最終因為越位而被吹掉。

2019年5月7日：利物浦4-0巴薩（歐冠半決賽次回合，總比分4-3）

安菲爾德上演過不少神奇之夜，而這場逆轉巴薩的比賽，可能就是近十年最好的。或許再也找不出另外一場比賽比這場比賽更能夠概括這支紅軍了——他們就是「精神巨人」。利物浦在首回合0-3落敗，看起來他們已然沒有了晉級決賽的希望。但克洛普在賽前告訴每一名紅軍將士：「雖然我認為這是一件不可能的事，但因為這是我們的比賽，所以我希望我們能夠有這樣的機會。」此外，受傷的薩拉赫也是穿著一件「永不放棄」的T恤出現在場邊。

比賽一開始奧里吉就為利物浦取得了進球。誠然梅西在這場比賽中也為球隊創造了三次不錯的得分機會，但都被錯過了。比賽在下半場發生了明顯的變化，維納爾杜姆梅開二度。巴薩震驚了，他們甚至都沒有將注意力全部集中在阿諾德的快發角球之上——奧里吉得球後順勢完成了射門，將總比分改寫為4-3。

2019年5月8日：阿賈克斯2-3熱刺（歐冠半決賽次回合，總比分3-3，熱刺憑藉客場進球數晉級）

即便是三周之前，熱刺在伊蒂哈德球場的表現，也沒有辦法與這場比賽相提並論。這是利物浦逆轉巴薩之後24小時，另外一場歐冠史詩級的比賽。阿賈克斯在客場1-0擊敗了熱刺，而在坐鎮主場的次回合比賽中，他們又是在上半場攻入了兩粒進球，看起來他們對於決賽門票已經十拿九穩，但事與願違。

這個時候，小盧卡斯出現了，他接二連三的進球，最終幫助熱刺在客場將總比分追成了3-3，並幫助熱刺最終以客場進球數的優勢淘汰阿賈克斯，與利物浦會師決賽。

終場哨響起之後，阿賈克斯的球員們癱坐在地上，淚流滿面。熱刺主教練波切蒂諾則是在場上跑來跑去，流下了興奮的淚水。

神兵天降盧卡斯
