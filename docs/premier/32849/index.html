尤文兩敗藍鷹非偶然，穆帥被愛徒上課

2019-12-23 3,505條評論 4,972次閱讀 1676人點贊

被米蘭恥辱性輸球弄的有點低迷，想說說昨晚的這兩場巔峰較量。尤文圖斯是意甲豪強，歐洲豪強，各項賽事僅僅一敗，1:3不敵拉齊奧，昨晚意超杯再碰拉齊奧，結果竟然如出一轍，還是1:3，能夠在半月之內相同比分送給尤文兩次敗仗的一定不是偶然，而且帶著意甲八連勝光環的藍鷹起飛了。

拉齊奧這支球隊向來是小球會思想，即便拿過多次意杯又名列意甲前茅，但始終不願意加大投入進軍歐冠，反而是功成名就時開始賣人。而現今的這些人，因莫比萊、米林科維奇、科雷亞、阿爾貝托，卻是不好出手的，正是這些人給了藍鷹和因扎吉的穩定。第一次取勝尤文是靠反擊中的速度和邊路肋部的結合，而這一次再次體現。尤文依靠C羅才贏得了很多個2:1，中場運行的慢速成為槽點。而這一次，C羅帶領阿根廷雙鋒終於沒能再次力挽狂瀾。

尤文圖斯在意甲神一樣的存在，卻害怕有控制速度進攻的球隊，打亞特蘭大會有這種感覺。可惜，亞特蘭大進攻是猛，但防守脆弱，而藍鷹是中前場的控制能力強大。尤文的防守很強，但拉齊奧這種快節奏的配合，整體的打法，乾淨快捷的出球，讓尤文後防有些顧此失彼，是盯人還是站樁，彷彿不能取捨。

這種情況不僅發生在後場，中場也一樣，本坦庫爾的染紅證明這一點。而C羅僅有爆發力，伊瓜因什麼都沒發揮，迪巴拉有靈性，所以打入補射一球，其餘什麼都沒發揮。拉齊奧的穩定超出想像，放棄歐聯杯的做法，讓尤文同樣雙線作戰卻不能苟同。成敗毫釐之間，尤文絲毫沒有變弱。

昨夜另一戰就是英超的師徒大戰，熱刺在主場0:2不敵切爾西。穆帥被愛徒蘭帕德擊敗。其實這是一場簡單的爭四大戰，切爾西最近態勢不好，如果輸球將鐵定出前四，而單純依靠亞布拉罕直接造就了打法單一。更嚴重的是，孫興民和阿里已經到了比較巔峰的狀態，穆帥又相當了解切爾西。如此情況之下，藍軍客場落敗成為大概率，蘭帕德破釜沉舟，向死而生。

這個改變就是343陣型，切爾西在熱刺主場大膽強攻。祖馬本來是失誤很多的後防球員，但變三後衛後反而激發了他的速度優勢。與其說是強攻，不如說是強守，此戰藍軍速度優勢明顯，阿斯皮利奎塔、芒特等都參與了移動的防守。這樣的混合防守逼搶下，帶來的就是熱刺的混亂。西索科、奧里耶等陷入了團圍。孫興民帶起球來就有速度，關鍵是帶不起球來。就這樣，小孫著急的選擇了染紅。

一切都結束了，威廉又恢復了曾經的活力。切爾西贏得很悲壯，但只是暫時穩住了前四。歷史上所有以弱勝強的戰例中雙方都會付出很大代價，同樣，這一戰，也許不值得，但，足以正名。
