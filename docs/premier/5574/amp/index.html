波切蒂諾下課牽動全歐，但這一條理由足以排除巴薩？

天天要聞
2019-11-22

隨著波切蒂諾的下課，熱刺迎來了穆里尼奧執教，穆帥也已經馬不停蹄地完成了第一場訓練課。穆帥出山的流量實在太大，讓人們幾乎忘記了離任的波切蒂諾——這個同樣出色的主帥，下家會是哪裡呢？

要預測波叔的下家，首先就要看各大球隊主帥的位置是否穩固。我們先來把歐洲各大豪門的主帥按照下課的可能性排個序、分個類，看看哪些豪門球隊有主帥需求：

1，帥位非常穩固。這種情況分兩種，一種是戰績優異、風格突出，說白了就是難以取代：克洛普（利物浦）、瓜迪奧拉（曼城）、孔蒂（國米）。要麼就是新來的，俱樂部不可能短時間內多次換帥，比如蘭帕德（切爾西）。

2，帥位壓力不大。儘管戰績不見得非常出色、或者比賽內容不一定精彩，但完全是在俱樂部接受範圍內。比如：薩里（尤文）、西蒙尼（馬競）。

3，帥位危機正在醞釀。這類主帥需要時刻保持專註，因為隨時都可能因一場關鍵性失利而成為危機中心，或者因為沒有進步和突破而被炒魷魚，比如：齊達內（皇馬）、巴爾韋德（巴薩）、圖赫爾（大巴黎）。

4，帥位危機已經出現。這類主帥已經聽到多次下課的聲音，而俱樂部很可能在等待他們做出改變，做不到的話隨時可能像波叔那樣走人，比如：法夫爾（多特）、埃梅里（阿森納）、索爾斯克亞（曼聯）。

如此一看，波叔最有可能去的只有3或4這樣的俱樂部，也就是說皇馬、巴薩、大巴黎、多特、阿森納與曼聯可能是波叔的下家。不過以上球隊中，巴薩是最沒有希望讓波叔執教的，這是為何？

理由很簡單：波切蒂諾有皇家西班牙人隊背景。波切蒂諾球員時代曾兩度效力於西班牙人隊，第一次7個賽季，第二次3個賽季，身為後衛的他總共為球隊打進了13粒聯賽進球。眾所周知，西班牙人是西班牙加泰羅尼亞地區的球隊，與巴薩乃同城死敵。巴薩即使換帥，也不會找波切蒂諾，當然波叔也不會願意執教巴薩。

按照賠率來看，曼聯是最有可能邀請到波叔的，賠率低至1賠2.75；科瓦奇卸任後的拜仁是1賠3；皇馬1賠3.5；隨後是大巴黎、阿森納、AC米蘭和尤文圖斯。拜仁是因為本就沒有正式主帥，因此「近水樓台先得月」，他們正在接觸波切蒂諾。而巴薩不在榜單當中，可能也跟西班牙人因素有關吧。
