中超：這位河北外援宣布退役，薪酬低，只為熱愛的足球

2019-12-14 160條評論 1,214次閱讀 306人點贊

就在今天，河北華夏幸福的外援拉維奇宣布自己退役，並在社交平台上同步發布了消息。這位現年34歲的巴黎聖日耳曼球員在2016年帶著期望加入了河北華夏幸福，並在2019賽季離開了河北華夏幸福。

說起拉維奇想必喜歡足球的朋友必定不陌生，在那不勒斯和巴黎聖日耳曼的那幾年可以說是拉維奇的黃金歲月，享受到了非常多的目光和聚光燈，但是現在的他終於在中國完成了自己最後的一戰，完美地給自己足球生涯畫上了一筆句號。

拉維奇在自己的社交媒體上感慨良多，以：他們都是這段故事中的精彩歲月。這些獨特的時刻以及無數的回憶我都將會銘記在心。我永遠感謝在這段旅途中陪伴著我的人。伴隨著喜悅的心情，我向生命中最精彩的歲月作告別。我真的很高興。」作為自己的告別語，可以說不少河北的球迷看到這兒估計都感慨良多。

可以感覺到拉維奇熱愛河北華夏幸福這個球隊，在球隊效力的過程中，一共出戰了75次，並且在這75次的出戰中拿下了足足35次的進球，可以說對於他這個年齡來說已經算非常好的一個成績了，平均兩場必進一球。除了這些戰績之外，拉維奇其實是一個真心愿意為足球和自己球隊付出的隊員。

在11月27日的比賽之中，河北華夏幸福在本賽季最後一個主場作戰中輸給了廣州恆大。拉維奇給河北打入了非常重要的一球，使得廣州恆大沒能在客場零封主場隊伍河北華夏幸福。而根據《東方體育日報》的消息表示在拉維奇因傷缺席比賽的時候，選擇了自己降薪百分之30，並表示沒有在球場上自己就不配得到這麼多的工資。使得俱樂部整體都非常的尊敬他，不僅如此球迷們也是對其讚賞有加。

在拉維奇的退役統計上，中國球迷對拉維奇的生涯評分達到了足足9.1分（滿分10分），可以說不管是以前在那不勒斯又或者是在大巴黎，還是現在來到了河北華夏幸福。拉維奇的職業生涯可以說是一個非常非常值得我們紀念和銘記的一個職業生涯。雖然現在拉維奇的傷病折磨得他無法再繼續前進，繼續踢球，但是拉維奇的足球精神，想當主角的那顆心值得我們所有球迷朋友以及球員學習。再見拉維奇，感謝你把職業生涯最後一站留給了河北。
