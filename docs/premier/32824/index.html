6年前AC米蘭德比0比1國際米蘭，卡卡薩內蒂兩位隊長賽前深情相擁

2013年12月22日，也就是6年前的今天，2013-2014賽季意甲聯賽第17輪上演焦點大戰，紅黑軍團AC米蘭客場挑戰同城兄弟國際米蘭。

這是米蘭主帥阿萊格里執教的第400場正式比賽，國際米蘭老將坎比亞索迎來第300次意甲聯賽。蒙托里沃停賽，本賽季從皇馬回歸米蘭的卡卡戴上隊長袖標，賽前卡卡與國際米蘭隊長薩內蒂深情相擁，成為這場對決最讓人難忘的一個瞬間。

​

隨著意甲的沒落，兩隊的陣容已經無法和當年的星光熠熠相比。看一下兩隊的首發和替補出場球員名單。

AC米蘭(4-3-2-1)：32-阿比亞蒂/2-德希利奧，25-博內拉，17-薩帕塔，21-康斯坦特(45’28-埃馬紐爾森)/16-波利(87’11-帕齊尼)，34-德容，4-蒙塔里/22-卡卡，8-薩波納拉(76’9-馬特里)/45-巴洛特利。

​

國際米蘭(3-5-1-1)：1-漢達諾維奇/14-坎帕尼亞羅，35-羅蘭多，5-胡安/2-若納坦，21-泰德爾(57’10-科瓦契奇)，19-坎比亞索(82’9-伊卡爾迪)，4-薩內蒂(68’17-庫茲曼諾維奇)，55-長友佑都/13-瓜林/8-帕拉西奧。

​

這場比賽中當局禁止AC米蘭死忠攜帶標語和裝備進入南看台，為了聲援對方球迷，國米死忠也收起了本方標語和裝備。對於已經學歪了的中超某些所謂的德比來說，這一幕令人深思。

​

第86分鐘，帕拉西奧打入了全場唯一進球。

全場補時第3分鐘後，蒙塔里推倒庫茲曼諾維奇，雙方球員發生摩擦，最終蒙塔里被紅牌罰下。國際米蘭主場1比0獲勝。
