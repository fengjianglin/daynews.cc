薩里：尤文的三個前鋒都是世界級，我非常希望C羅能拿金球獎

2019-12-01 0條評論 15次閱讀 0人點贊

北京時間2019年11月30日，尤文圖斯主帥薩里出席了對陣薩索洛的賽前發布會，薩里表示球隊在對陣馬競的比賽中發揮很出色，明天比賽埃姆雷-詹應該會登場。

關於C羅：

克里斯蒂亞諾上一場比賽表現得很棒，就算是在防守一端也是如此，現在他只需要繼續努力，讓自己輝煌的職業生涯增加更多的榮譽就好。 前鋒線上的輪換很正常，我們擁有三個世界超一流的前鋒，他們的狀態也都不錯。

薩里的煩惱：

有時候，讓誰出場是一件讓我很頭疼的事情；不過這樣的陣容深度，也是讓其他人頭疼的地方。關於陣容明天的比賽，埃姆雷-詹應該會獲得出場機會。

巴爾扎利的作用：

巴爾扎利回歸教練組，他將他的防守經驗教給了我們的後衛，這對德利赫特的幫助非常大，這孩子已經適應了我們的足球，也融入了尤文的傳統。

桑德羅：

桑德羅昨天參與了完整的訓練，我還不知道他能否出場90分鐘。 貝爾納代斯基也很棒，他對球隊非常關鍵，技術出色身體也強壯，所以他需要更多的機會來展現自己。

展望比賽：

明天的比賽非常危險，我們踢了兩場非常消耗精力的比賽，再加上薩索洛本身實力就很強，他們又有一個年輕又有能力的教練，我對他們非常尊重。
