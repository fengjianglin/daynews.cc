2019-12-10 330條評論 3,115次閱讀 1254人點贊

​19/20歐冠分組賽

比賽時間2019-12-11 04:00

里昂VS萊比錫紅牛

由於積分的差異，里昂與紅牛的追求不同，里昂必須要取勝才能晉級，而紅牛平局可保小組第一，首輪交手裡昂2-0客場戰勝紅牛，按理說本場里回到主場的里昂信心十足，再加上他們的戰意點，本場都是對里昂十分有利的。

紅牛的客戰要強於主場，客場的成績排名德甲第一，當前各項賽事4連紅，共打入15球，火力爆了。亞指為本場開出主讓半球主隊高水初盤，後市退盤為主讓平手主隊低水，機構對主隊走勢看衰。里昂有交手優勢，又是主場作戰，理應受到重視，但平手盤基本對主隊無信心可言，從盤路的走勢，本場推薦客隊不敗。重點可能是平局。

勝平負：讓球負

比分：1-1

國際米蘭VS巴塞羅那

國米目前積7分排名第二，還未能確保晉級，本場必須全三分才能晉級，而巴塞羅那已積到11分排名小組第一位，本場不需要積分保住小組頭名。國米在往年交手巴薩的賽事中，失敗率偏高，近6戰僅有一次取勝。亞指為本場開出主讓半一主隊低水初盤，初盤指數對主隊持較好，後市機構再升盤至一球盤主隊中水，機構對主隊進一步看好，從走勢看，本場國際米蘭3分勢在必得。

勝平負：主勝

比分：2-0 3-1
