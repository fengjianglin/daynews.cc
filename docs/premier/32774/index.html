半程領先8分，曾7連勝+11輪不敗，曼聯死敵有望15年後重返英超

2019-12-24 144條評論 1,358次閱讀 304人點贊

下賽季的英超，一個熟悉的名字即將回歸——利茲聯。

利茲聯

目前，本賽季英冠踢完23輪，賽程剛好過半，利茲聯以14勝5平4負的戰績高居積分榜第2位。雖然剛剛遭遇1場失利，但在此之前，他們剛剛經歷11輪不敗，並在其中完成一波7連勝。

相比榜首的西布朗，利茲聯僅落後3分，但他們領先身後的謝周三，已經有8分之多。

同時，利茲聯也是英冠防守最好的球隊，24輪比賽只丟掉15球，極有希望在本賽季獲得直接升級的資格。

利茲聯暫列英冠次席

目前這支利茲聯的主帥，是赫赫有名的「瘋子」教練貝爾薩。在2018年接手球隊後，雖然曾經出現過「偷窺」對手戰術的「間諜門」這樣的插曲，但現在，貝爾薩有望帶領球隊完成「復興」的最後一步！

貝爾薩

利茲聯上一次出現在英超已經是2003-2004賽季的事情。那個賽季，財政混亂導致球隊內外交困，最終慘淡降級，而在3年前，他們還曾闖入過歐冠4強。

2000-2001賽季，大衛-奧萊利利茲聯在歐冠中先後戰勝AC米蘭、拉齊奧、拉科魯尼亞，並在第一階段小組賽就擠掉巴薩，闖入半決賽。只可惜在半決賽中輸給庫珀執教的瓦倫西亞，無緣決賽。

那支利茲聯擁有維杜卡、科威爾、阿蘭-史密斯等球星，以及年輕的伍德蓋特、費迪南德，「青春風暴」一時席捲歐洲。只是當時，財政問題已經逐漸在侵蝕處在巔峰中的利茲聯，債台高築的他們在之後幾年進入惡性循環。

2002年，利茲聯以破後衛轉會費紀錄的價格將費迪南德賣到曼聯；

2003年冬天，利茲聯將伍德蓋特高價賣到紐卡斯爾；

2003年夏天，利茲聯賣掉保級功臣哈里-科威爾；

2004年，利茲聯降級，並在賽季結束後賣掉維杜卡和阿蘭-史密斯。

轉眼間，一支歐冠四強球隊分崩離析。

2006-2007賽季，在由於財政問題被罰掉10個積分後，利茲聯再次遭遇降級，來到英甲聯賽，陷入俱樂部歷史上的最低谷。直到2010年，他們才重返英冠。

利茲聯時期的費迪南德&阿蘭-史密斯

作為曼聯的死敵，雖然坎通納、費迪南德、阿蘭-史密斯等人都曾先後效力2支球隊，但利茲和曼徹斯特兩座城市之間的仇恨關係，也延續到了足球的成名。曾幾何時，利茲聯vs曼聯的這組「玫瑰戰爭」，被視作英格蘭賽場最火爆的對決。

效力利茲聯時的坎通納

目前這支利茲聯陣中，卡西利亞、埃德爾-科斯塔、巴勃羅-埃爾南德斯、巴姆福德等名將。其中，出自切爾西青訓的「學霸前鋒」巴姆福德是本賽季球隊的最佳射手。

巴姆福德

下賽季的英超聯賽，我們有希望見證最為火爆的英超「玫瑰戰爭」，這支英格蘭足壇的老牌勁旅，會繼續他們的復興之路嗎？
