荷甲榜首之爭，阿爾克馬爾客場遇阻？

2019-12-21 276條評論 1,454次閱讀 406人點贊

昨天公推的佛羅倫薩vs羅馬，羅馬還是實力高出一截，雖然客場作戰，但全隊並沒有絲毫怯場的意思，最終1-4戰勝佛羅倫薩，客場全取3分，公推順利紅單。今天給朋友們公推一場荷甲的比賽，鹿特丹斯巴達主場迎戰阿爾克馬爾。

荷甲第18輪 鹿特丹斯巴達vs阿爾克馬爾 2019-12-22 周日 02:45

鹿特丹斯巴達作為升班馬，照本賽季勢頭下去，保級問題不大，目前在聯賽中排名第11。鹿特丹斯巴達主場進攻其實還不錯，近21個主場賽事只有一場交了白卷。但是近期成績不好，近7場比賽只有1勝，贏得還是實力一般的維特斯，近4場比賽都未能取得一粒進球，這跟球隊整個狀態有關，也跟球隊實力有關。

阿爾克馬爾本賽季的成績值得肯定，目前與阿賈克斯同積41分，並列榜首。阿爾克馬爾本賽季最大的變化是防守能力提高不少，聯賽至今多達13場零封對手，其中不乏強隊，場均失球只有0.47個，為荷甲球隊中失球最少，而近7場荷甲聯賽收穫連勝，而且全部零封對手，狀態火熱。

兩隊近5場交鋒，阿爾克馬爾取得3勝2平的成績，優勢不小。值得一提的是鹿特丹斯巴達目前主場4場不敗，這對阿爾克馬爾有一定的壓力。在與阿賈克斯爭冠的路上，這樣的比賽阿爾克馬爾是必須要打下來的。本場比賽機構意見不太統一，有的開出主隊受半球，有的開出受一球，經過調整目前都統一到主隊受讓一球。本場比賽老曹相信戰意更強的阿爾克馬爾不會簡單滿足於只贏一球的危險局面。
