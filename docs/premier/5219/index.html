薩里回應丟掉榜首：我們意甲冠軍太多了，爭冠動力所以沒那麼強烈

2019-12-02 0條評論 17次閱讀 0人點贊

北京時間12月1日19:30，尤文圖斯在主場迎來了薩索洛的交手；雙方進行了意甲第14輪的較量，本場比賽尤文圖斯在主場是依靠博努奇的先拔頭籌一度領先，隨後客隊實現逆轉反超比分，最終是C羅點射扳平；尤文圖斯無法在主場取勝也讓身後的國際米蘭反超一份登頂了聯賽榜首。

本場比賽C羅是復出首發登場，最近四輪比賽未進一球的葡萄牙人最近的壓力也是非常大；特別是因為迪巴拉的狀態以及表現非常好的情況下，迪巴拉還要給C羅當替補，這也讓C羅無法進球的情況下越來越大的質疑聲席捲開來。

本場比賽雖然C羅依靠替補上場的迪巴拉製造的點球成功破荒並最終讓尤文圖斯繼續保持賽季不敗的出色戰績；但是C羅在比賽接近第七十分鐘的時候在門前擋出了隊友迪巴拉的射門，這一幕讓人實在是看不懂；雖然迪巴拉的這一腳射門不一定能夠百分百的完成破門，但是C羅在前場無法製造更多的威脅也讓人質疑他在隊內壓著迪巴拉的位置是否是正確的。

雖然如此，但是不得不說尤文圖斯在面對薩索洛的時候表現確實有些不盡人意；對比C羅，德利赫特依舊是沒能達到出色的補強，況且德利赫特還是在隊內年薪僅次於C羅的球員；如果說C羅因為年紀大了，他可能在數據上或者進球上無法為球隊帶來更多的幫助，但是C羅的精神力以及商業價值確實是隊內絕對的頭號人物，所以德利赫特的表現確實讓人無法滿意。

本場比賽也正是德利赫特在比賽下半場剛剛開始的時候出現業餘解圍失誤導致球隊最終被薩索洛一度逆轉了比分。

尤文圖斯雖然本場比賽依舊保持不敗，但是這場平局也讓各項賽事六連勝畫上了句號；當然在結束連勝的同時也丟掉了意甲聯賽的榜首。雖然國際米蘭逆轉尤文登頂了意甲聯賽榜首，但是尤文圖斯主帥薩里卻並不太在意這個結果，因為在義大利人看來，尤文圖斯未來還會有很多機會重新拿回榜首位置的機會。

薩里在尤文圖斯二比二戰平薩索洛的比賽後接受了相關媒體的採訪，薩里表示尤文圖斯贏得的意甲冠軍次數太多了，所以在爭冠的動力上沒有太過強烈，球隊對於歐冠爭冠的渴望會更加強烈一點。但是作為主教練，薩里表示自己會嘗試提升球員們的求勝欲。

「球隊已經贏得了許多次意甲冠軍，所以球隊現在對於聯賽爭冠的動力沒有歐冠爭冠動力那麼強烈了，這一點我們必須做出改變。」

對於目前丟掉榜首位置以一分之差屈居次席，薩里表示意甲聯賽還有24輪，他並不擔心這1分；薩里覺得未來會有機會反超國米，而且他表示球隊沒能取勝應該關注球隊目前所存在的問題而不是去在意目前尤文圖斯到底是意甲第一還是意甲第二。

不過在這裡還是得提醒一下這位尤文圖斯主帥，利物浦球迷將會告訴你一分有時候也會難倒英雄好漢。
