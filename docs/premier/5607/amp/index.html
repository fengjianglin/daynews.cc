不好挽留！我們的保利里奧明年會走？

天天要聞
2019-11-21

這一段時間給國足的世界盃預選賽霸屏了，讓我們球迷暫時把恆大給忘了，11.23日恆大和上港的真正天王山之戰開始了，要是沒有國足的大潰敗鬧心，球員還真的翹首盼望這場比賽，廣州和上海，多少年了，這二支城市球隊一直在上演雙城記，只有曼薩諾帶國安拿了一次亞軍。恆大大戰上港，勝負直接決定冠軍歸屬，因為這是倒數第三輪的比賽，已經在沖關了。恆大現在的狀態可以說在低潮期，傷兵滿營，受傷不能上場的球員不比首發球員少，都有二位數了。恆大要勝利，保利里奧布站出來，扛著恆大走，幾乎不可能！

許老闆放出話：2020年恆大實現全華班。保利里奧沒有中國國藉啊，明年保8和塔利斯卡還在恆大，是不是打許加印臉，中國企業家很要社會名譽。許同志也沒說保利里奧除外，是非賣品。這就說明保8明年真有可能離開恆大，從他給國際米蘭點贊看出一點苗頭，國米也不錯，歐洲算強隊，有歐冠可參加。恆大和蘇寧關係一直不錯，蘇寧是國米大股東，蘇寧少東家張朝陽現在具體接管國米，擁有拍板權的老闆。保8轉會國米，是三贏的不錯的結果。

保利里奧2017年被巴薩看重，用4000萬歐違約金直接帶走，說明保利里奧能在歐洲大牌球隊立足，保8在恆大不放的時候，狀態也相當不好，在中超比賽發揮和現在一樣，沒有鶴立雞群的樣子，就是泯然眾人矣。這種感覺不好，是要離開恆大的節奏啊，心不在恆大了。狀態好的保8，滿血上場那就是厲害，我們都能看出來。當保利尼奧一下子持續低迷的時候，說明他有想法了，想換個平台，重新再煥發第二春。恆大對於他，已經有點厭倦了。

恆大2020年的全華班，都有7名歸化球員，保利里奧的作用也大不如前了。巴西幫一大堆，為國出戰，但是我們的保8不行，沒資格代表國足比賽，雖然我們更想他進國足。走就走吧，流水不腐，恆大也不會保8離開，實力會受到多大影響。歸化高拉特和艾克森等花了不少銀子，把平頭塔和保8出售，補一補財務虧空，也不錯。保利里奧明年離開，我們恆大球迷心裡是什麼感受？他是中超和恆大引進最大牌球星，也是發揮最好，身價最高的球星，也是許家印願意用私人飛機接送保利里奧回巴西國家隊比賽的球星！再見，暴力鳥，一代恆大英雄，明年的那一天我們肯定會說。
