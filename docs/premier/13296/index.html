10日大嘴足球離散：歐冠末輪精彩紛呈，里昂主場命運堪憂

2019-12-10 167條評論 2,762次閱讀 178人點贊

周一早場繼續紅單，德乙斯圖加特3-1成功逆轉戰勝紐倫堡，正路主勝順利打出；早場公推近3檔3中3全紅，早場近16中12狀態穩定，周一晚間檔5中3，主推命中里奧阿維主勝正路，比分預測命中里奧阿維1-0！

周二足壇賽事豐富，本周歐冠將進行小組賽最後1輪爭奪，明天凌晨利物浦、國米和切爾西等豪門將為出線名額殊死一搏。另外本周英冠聯賽再度一周雙賽，共計7場比賽入圍今晚賽程，足總杯補賽1場入圍。早間檔分享歐冠+英冠等精選15場指數，另附歐冠里昂vs萊比錫解析！獲取全部15場晚場指數和觀點，以及串關和比分預測，歡迎搜索關注大嘴聊球號！

[關於離散] 競技體育分析指數，也就是各家歐指分歧度。基本原理：百家歐指平均值與每家指數差值的絕對值的求和。離散度小則各家觀點接近，進而賽果易打出；離散度大反之意見分歧大，賽果不易打出。又因各方因素影響下指數變動，離散在特殊時刻具有更高參考價值，切勿盲目參考運用。換句話說，離散只是參考指數，而非結論性數據!以下是各場數據匯總(數據截取至11:00)：

周二012 歐冠 里昂vs萊比錫紅牛

比賽時間：12月11日04:00

個人觀點：里昂負（-1）

比分預測：1-1/1-2

歐冠小組最後1輪爭奪，前5輪比賽過後，巴薩、曼城、拜仁和巴黎等豪門已經鎖定小組頭名，而國米、切爾西、國米和利物浦等球隊命運將在最後1輪下結論。本場比賽所在G組，並無超級豪門在陣，4支隊伍整體實力相當接近，是一個隱形死亡之組。目前萊比錫積10分提前出線，但尚未鎖定小組頭名，澤尼特和里昂同積7分，兩隊本場都必須全力爭勝，而積4分的本菲卡，由於相互戰績落后里昂，目前只能競爭歐羅巴名額。

里昂上賽季法甲高居第3位，本賽季球隊失去費基爾等多名核心球員，整體實力明顯下降。法甲聯賽至今，里昂戰績7勝4平6負，目前僅積25分排名法甲第7位，聯賽表現相對糟糕。歐冠賽場，球隊前5輪2勝1平2負，由於相互戰績處於劣勢，目前落後於澤尼特排名小組第三位。本場里昂唯有戰勝萊比錫，才能鎖定小組出線名額，球隊壓力自然不小。

萊比錫上賽季德甲排名第3位，新賽季球隊聘用霍芬海姆少帥納格爾斯曼執教，球隊聯賽至今9勝3平2負，力壓拜仁、勒沃和多特等豪門，排名德甲第二位，僅僅落後榜首的門興1分，很有希望衝擊德甲冠軍。歐冠賽場，球隊首輪輸球情況下，之後比賽3勝1平拿到10分，目前小組排名第一，由於相互戰績優勢，已經提前鎖定小組出線名額。但本場倘若輸球，球隊將退居小組第二位，因此本場也不容有失。

雙方首回合交手，里昂客場爆冷2-0完勝萊比錫，而這也是兩隊歷史上唯一一場交手紀錄。目前雙方賬面實力對比來看，萊比錫全隊身價高達5.4億歐元，領先於里昂全隊3.67億歐元，賬面實力客隊佔優。數據方面，初始勝平負指數2.31 3.48 3.00，勉強開到233指數組合，相對優勢給到主隊。不過受雙方整體狀態和小組形勢等因素影響，目前數據明顯向客隊回攏，目前均值2.45 3.55 2.77，主隊優勢明顯縮水，而平局目前指數高企，客勝熱陡然上升。離散指數一邊倒傾斜，目前主勝指數已經21.01完全高企，平局指數中位，客勝指數降至相對低位區間，指數對客隊相對利好。綜合分析來看，本場看好萊比錫客場不敗，重點關注客勝心理冷門打出。
