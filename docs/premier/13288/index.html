楊澍：周一全紅，硬貨復盤+歐冠免費分享一場

2019-12-10 785條評論 3,217次閱讀 1074人點贊

周一的比賽一向競猜難度不小，老楊只選了其中三場，結果三場皆紅，為這一周開了個好頭。

其中阿森納的比賽逆轉得漂亮，槍手這一場顯然是被看低的，讓球勝的獎金太過吸引，老楊還是堅挺槍手。

葡超的比賽也有心理冷門的味道，里奧阿維在少打一人的情況下拿到這一勝，可以說相當給力了。

今明兩天，歐冠將進入最後一輪的爭奪，相對來說，今晚的四個小組變數更多一些，雖說有戰意的球隊更多，競猜難度卻並不小，畢竟人都是很主觀的，帶著一些主觀想法去分析比賽的話，本身就容易有偏差。

東亞杯的比賽也已經開始。賽事分量儘管不足，不妨礙大家關注國足的一顆心。

送上彩蛋一場，希望帶著大家繼續紅：

010 歐冠 多特蒙德 VS 斯拉維亞

多特本賽季無論是德甲還是歐冠兩條戰線的表現都不算十分強勢，球隊目前以26分排名德甲第三位，距離榜首的門興有5分的差距；在歐冠中，多特5場小組賽取得2勝1平2負，暫時和國米同積7分，最後一輪將決定兩支球隊的命運。多特最後一場小組賽沒有任何選擇，只有獲勝，然後再看國米與巴薩的比賽結果，與國米的直接交鋒，多特是落於下風的。本場競彩獎金方面，主勝問題不大，不過讓球勝獎金不算低，小勝概率居多，首選讓球平。
