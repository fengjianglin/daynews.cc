世界足壇球星回顧:智利前鋒——薩莫拉諾

天天要聞
3 天 ago

伊萬·薩莫拉諾是智利足球史上最顯赫的名字。他曾效力過西班牙的塞維利亞、皇家馬德里和義大利的國際米蘭。他在皇馬時贏得過西班牙聯賽的賽季最佳射手，在國際米蘭時期幫助球隊奪得1998年歐洲聯盟杯的冠軍。並於1998年代表智利隊參加了該年在法國舉辦的世界盃足球賽。

1985年薩莫拉諾在國內的科布雷薩爾競技開始了自己的職業生涯，1988年轉戰歐洲賽場，加盟了瑞士球隊聖加侖(FC St. Gallen)。 1991年薩莫拉諾初次登陸西班牙足球甲級聯賽，效力於塞維利亞，薩莫拉諾在59場比賽中總共射入了21個球，隨後被球隊以500萬美元的價格交易去了西班牙豪門皇家馬德里。在1992年至1996年的皇馬期間，薩莫拉諾贏得過聯賽，西班牙國王杯，西班牙超級盃和歐洲超級盃的冠軍榮譽。1995年在主教練豪爾赫·巴爾達諾的帶領下，薩莫拉諾幫助皇馬取得了西甲聯賽的頭銜，並且在聯賽中射入了27個進球，在和死敵巴塞羅那的比賽中更是上演了帽子戲法，隨後薩莫拉諾榮登當屆的西甲聯賽賽季最佳射手稱號。

五個賽季之後薩莫拉諾離開西班牙，轉而在1996年到2000年期間，效力了意甲豪門國際米蘭4個賽季。當時他的隊友有尤里·德約卡夫，迭戈·西蒙尼，哈維爾·薩內蒂和羅納爾多等眾多知名球星。在1998年5月，國米在決賽中3-0擊敗了拉齊奧後獲得了歐洲聯盟杯的冠軍，在決賽中正是薩莫拉諾打進了打破僵局的第一球。

2001年薩莫拉諾轉投墨西哥俱樂部墨西哥美洲兩個賽季，在美洲對的第一個賽季中獲得了Torneo de Verano獎盃。2003年底在智利本土的知名球會科洛科洛結束了自己較為輝煌的16年的職業足球生涯。

國家隊生涯

薩莫拉諾總共代表智利國家隊出戰69場，射入34粒進球。他初次披上國家隊戰袍是在1987年6月19日，20歲的薩莫拉諾在和秘魯國家隊的一場友誼賽中出場，並且射入一球，這場比賽中智利3-1擊敗了秘魯。
