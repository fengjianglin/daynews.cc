名宿：三叉戟有點像我當年 尤文缺乏比達爾這樣的球員

2019-12-22 2,387條評論 4,081次閱讀 1196人點贊

尤文名宿拉瓦內利接受全尤文的電話專訪：

尤文圖斯和拉齊奧在二十來天之後再度交鋒，你對此有何看法？

「我想將是一場非常美妙和精彩的比賽，這兩支球隊總是嘗試踢好足球。預測尤文圖斯是熱門，他在奧林匹克球場踢了非凡的半個小時，儘管隨後結果並不積極。我們經常忘記這點。不過，拉齊奧，一直是個危險的對手，必須保持非常當心。它擁有歐冠級別的球員，比如因莫比萊、米林科維奇-薩維奇、路易斯·阿爾貝托、科雷亞，他們總是處於能夠改變比賽的水平。為了把獎盃帶回家，斑馬軍團必須竭盡他們的全力」。

薩里可能連續第三場比賽派出三叉戟，迪巴拉-C羅-伊瓜因。

「當然這是一種可用的方案，教練每天都在觀察他們，比我們更加了解相關條件和狀態的情況。不過尤文圖斯比對手多打一場比賽，因此除了三叉戟我想大家都必須把犧牲放在首位，以便保持球隊的平衡。有點就像在我的尤文圖斯所發生過的一樣」。

你相信與過去幾年相比，在中場缺少一名能夠決定比賽的球員嗎？

「我不相信缺少一名決定性的中場球員，斑馬中場水平極高，只需看到埃姆雷詹這樣強大的球員疲於找到空間就能明白這點。羅德里戈·本坦庫爾正在打出一屆精彩的聯賽，他是一名善於奔跑、頑強堅定和適應戰術的年輕球員。加上皮亞尼奇、馬圖伊迪、赫迪拉：拉姆塞——後者可以出任前腰——會難以做出選擇，還有道格拉斯，他狀態良好時能決定比賽」。

正是在中場方面，傳聞指出對阿圖羅·比達爾和萊昂德羅·帕雷德斯有點興趣。這兩名球員可以帶來更多的東西，儘管已經擁有能夠決定比賽的球員？

「如果尤文圖斯能夠重新得到比達爾，會變得不可戰勝。這點毫無疑問。他是這支球隊所缺乏的球員，有他在場上就可以一直派上重型三叉戟」。

最後，如果我們頒發黑白「金球獎」，你會給誰？

「對於2019年，毫無疑義地，頒給C羅」。
