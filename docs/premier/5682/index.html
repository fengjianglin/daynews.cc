全歐尋鋒！羅馬和米蘭將目光投向西班牙新星奧爾莫

2019-11-14 0條評論 17次閱讀 0人點贊

薩格勒布迪納摩副體育總監馬爾科-武凱利奇（Marko Vukelic）證實，達尼-奧爾莫（Dani Olmo）很可能在一月份轉會離開。武凱利奇透露：「我們知道羅馬和米蘭對他感興趣，但他們不是正在關注奧爾莫的全部俱樂部。」

這名年僅20歲的球員2014年從西甲豪門巴塞羅那轉會來到薩格勒布迪納摩，當時的他還只是阿倫-哈利洛維奇交易的一部分。

「我們已經知道了來自羅馬和米蘭的興趣。」武凱利奇在接受Calciomercato採訪時說道。

「但並不是只有這兩傢俱樂部在關注他。有很多俱樂部都對奧爾莫很感興趣，因為他是一名冉冉升起的新星，也是我們俱樂部的一名明星球員，他在這表現得很出色。」

武凱利奇透露，這位西班牙新星很可能在一月份的冬窗再次轉會，但雖然有眾多俱樂部表達了對於他的興趣，但薩格勒布迪納摩目前還沒有收到任何一份正式的報價。

「顯而易見的是奧爾莫很有可能在一月份離開，我們隨著都準備聽取關於他的報價。我們目前還在等，有很多關注他的俱樂部，讓我們看看具體的報價會什麼時候到來。」
