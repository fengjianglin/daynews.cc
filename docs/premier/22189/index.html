熱刺助教：穆里尼奧的演技在教練界是No1，但是他比不過巴薩球員

2019-12-19 221條評論 1,268次閱讀 317人點贊

英國時間12月18日，熱刺助教Joao Sacramento被邀請參加天空體育的周中播放的足球節目。首先他向穆帥的前同事們「打招呼」，表示穆里尼奧現在執教很繁忙，無法回到他喜歡的位置上（擔任天空體育解說員），接著他就與天空體育的工作人員以及嘉賓們，分享了穆帥執教熱刺以來的趣事。熱刺有著拍攝紀錄片的傳統，每當球隊取得一點成績的時候，熱刺總是要為球隊拍上一部紀錄片。穆里尼奧這位足壇「大流量」執教熱刺的時候，當然也要「入鄉隨俗」拍攝紀錄片。熱刺的訓練場、餐廳乃至更衣室，只要穆帥所到之處，無時無刻都有攝像機陪伴，就是為了給紀錄片拍攝有用的素材。

於是天空體育一位穆里尼奧的前同事就問助教：「穆帥會不會不適應攝像機圍繞的感覺？這是在日常生活中。」助教很自信地表示：「他很享受（所有舉動被拍攝紀錄片）的感覺。」記者這位助教認為，穆里尼奧的演技絕對是屬於教練界的No.1：「如果足壇有奧斯卡的話，我想教練組的最佳演技獎，肯定可以頒發給穆里尼奧了。我可以很確切地說，他的演技在教練界是No.1。但是要把球員都算上來的話，那就不一定了。我想他的演技，還是比不過巴薩球員。」這位助教說的「巴薩球員」，無疑就是那些依靠跳水騙取點球的球員了。在此，Joao Sacramento也鄭重聲明了：穆里尼奧是一位非常痛恨假摔的教練。

這一點大家都是知道的，因為穆里尼奧每一個賽季之前，為球員上的第一堂可並不是戰術課，也不是鼓舞人心的精神課，而是道德品德課。在穆帥的理念中，球員應該把體育道德擺在第一位。這包括了在場上不弄虛作假，因為穆里尼奧就曾經在個人傳記中說過：「假摔總歸是假摔，儘管這個動作可能讓他以及他的球隊，暫時取得了勝利，僥倖獲得了比分優勢，但是他們在整個足壇中的形象一經徹底敗壞了。」相信老球迷們都知道，穆帥指的就是十年前，布教授的「成名作」——倒地捂臉後偷偷觀察裁判舉動。

這個動作發生在那個賽季國際米蘭vs巴薩的歐冠半決賽中，儘管巴薩憑藉著布教授出色的演技，誇張的捂臉動作，暫時讓裁判罰下了藍黑軍團一位球員，但是這無法改變國際米蘭最終戰勝巴薩進軍決賽的事實。現在的球迷們看待那一次布教授的表演，都說抱著輕鬆、調侃的心態去看待的，然而穆里尼奧作為當時的國米主帥，他就不那麼認為了。也是那個動作後，穆帥更加痛恨依靠「誇張演技」在比賽中爭取進攻機會的球員。而巴薩在布教授的經典動作誕生之後，也獲得了「拉瑪西亞影視學院」這一延續到現在的稱呼。

所以助教這回在誇獎穆里尼奧演技出色的同時，也說了他比不過巴薩球員，實際上就是在諷刺當時的這場淘汰薩上，布教授的「誇張演技」。好在現在VAR已經普及了，雖然在越位的判斷上仍舊鬧出不少笑話，但是現在的球員們，還想玩「禁區跳水」幾乎是不可能的事情了。縱觀當今足壇，因為假摔拿黃牌的現象的確越來越少了，一方面VAR的普及讓球員們「不敢表演」，另一方面球員的體育道德普遍都有提升，這是足壇的好事，也是穆里尼奧這批教練願意看到的。我們就期待穆帥在熱刺的紀錄片吧！看看他的演技，到底能不能追上「拉瑪西亞影視學院」 這批超級巨星。
