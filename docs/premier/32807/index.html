太陽報：曼聯準備為哈蘭德開出20萬鎊周薪 10倍於當前薪水

2019-12-24 2,619條評論 3,760次閱讀 1951人點贊

據英國媒體《太陽報》消息，曼聯為了能夠簽下當紅妖星哈蘭德，準備向他開出一份周薪高達20萬英鎊的合同，而且合同里還會加入一些獎金條款，譬如進球、金靴、球隊成績等，這比他目前的待遇有了巨幅上漲。

本賽季，哈蘭德表現非常出色，歐冠賽場上出場6次，打進8球，在射手榜上僅次於萊萬，排在第二位；在奧地利聯賽，哈蘭德也有神奇發揮，14場比賽轟進16球，他出色的表現幫助薩爾茨堡排在聯賽榜首。由於表現出色，19歲的哈蘭德成為歐洲足壇的香餑餑，諸多豪門想簽下他。

志在復興的曼聯，也是哈蘭德的追求者之一，為了簽下挪威妖星，紅魔向他開出了一份高達20萬英鎊的周薪合同，外加進球、金靴、球隊成績（歐冠資格和獎盃）等獎金條款。英國媒體《太陽報》表示，哈蘭德如果接受曼聯的合同，意味著薪水暴漲10倍，從2萬英鎊周薪，漲到20萬英鎊。同時曼聯打算滿足薩爾茨堡的要求，在冬窗買斷哈蘭德後將挪威天才回租，讓哈蘭德能夠在奧超踢完本賽季。

早些時候，萊比錫紅牛主管明茨拉夫接受採訪時，暗示他們在哈蘭德的競爭落敗了，「是的，我們與哈蘭德見了面，我們對他感興趣，非常感興趣，他聆聽了這些，也很樂意轉會，他最晚希望在明夏更換球隊。想簽他，並不容易，因為曼聯、尤文、多特還有一些俱樂部都想要他。當然，我們的俱樂部也有一定的限制，因為薪資問題，我們可能無法去競購他，費用是可控的，不僅僅是薪資，還有轉會費。這都是一樣的陳詞濫調，我們可以買到一切，我們可以做任何事情，不幸的是，我們現在無法做到，否則哈蘭德就是我們的了。」
