單關簡析預測：德甲 美因茨vs法蘭克福 意甲 卡利亞里vs桑普

2019-12-02 0條評論 15次閱讀 0人點贊

北京時間2019-12-03日03:30分，德甲聯賽第十三輪的末輪，美因茨將主場迎接法蘭克福的挑戰。目前美因茨在聯賽積12分排名15位，如何保級是現在的首要問題，而法蘭克福積17分排名第十，雖沒有降級壓力，但是相比上賽季法蘭克福本能有所下滑，而在歐聯杯中形式也不算樂觀。

美因茨在本賽季表現糟糕，本賽季4勝8負積12分排名聯賽第13位，期間打進17球失31球，薄弱的防守是球隊戰績慘淡的主因，不過美因茨是本賽季至今沒有平局的球隊，勝負能力明顯。而在上輪缺少一人的情況下，美因茨客場5-1擊殺霍芬海姆，可以說是在本賽季最漂亮的一仗，球員士氣也在長期低落的情況下得到了提升，不過脆弱的後防線若不得改善，依然不能改變現狀。但是美因茨的主場相較客場要穩定得多，主場也是他們的拿分重地。

法蘭克福則是在周中的歐聯小組賽做客倫敦，憑藉鐮田大地的梅開二度，客場2:1戰勝槍手阿森納，終結了球隊三連敗同時也是直接導致了對方主教練埃梅里的下課。法蘭克福雖然今年夏天失去了上賽季狀態如日中天的三叉戟，但是帕先西亞和多斯特完美取代了他們的位置，而中場方面鐮田大地和科斯蒂奇組成的前衛線可攻可防，經常讓對手非常頭疼。本賽季5勝2平5負積17分排名聯賽第10位，期間打進21球失18球，球隊的攻守體系較為均衡。

兩隊論定位和整體實力法蘭克福要強於美因茨，指數給出法蘭克福平半讓步後市直接調整至平手盤明顯對法蘭克福信心不足。對於本場比賽，美因茨主帥拜爾羅澤上任後確實有明顯好轉，通過上輪大勝使球員狀態回暖，但是否會有穆里尼奧上任後的奇效，還需要抓緊改善球隊的防線。而法蘭克福雙線作戰，在聯賽保持穩定的同時還要顧及歐聯杯小組賽的情況，不僅體力方面有所損耗而且本場能否全力出戰還是未知綜合以上種種原因本場選擇支持美因茨保平有望取勝。比分看好：2-1，2-2，2-0

本場推薦：美因茨+0，競彩：勝平，膽大單選勝

北京時間12-3-03日45分，卡利亞里主場迎戰桑普多利亞。目前，卡利亞里積25分，聯賽排名第6;桑普多利亞積12分，聯賽排名第17。

卡利亞里10輪不敗，狀態持續高漲；桑普4輪不敗，狀態不錯。

卡利亞里本賽季表現突出低開高走目前也是處於歐冠區內，狀態較過往兩個賽季有了明顯的飛躍。自開局意甲2連敗之後，近11場意甲豪取7勝4平狀態和士氣俱佳。球隊上輪客戰萊切的比賽，球隊一度取得兩球領先，但在第81分鐘到85分鐘連續被罰下兩人，最終被升班馬連追兩球2-2逼平，連勝勢頭被終結。但球隊仍保持著11輪意甲不敗紀錄，表現僅次於榜首的尤文。球隊攻擊端表現十分強悍，本賽季14場正賽多達12場至少2球，且無1場蛋蛋，前場三叉戟攻擊力十足，整體競技狀態十分火熱。

桑普多利亞目前僅僅比降級區的熱那亞高出兩個積分，拿分迫在眉睫。球隊在上場比賽主場2:1力克烏迪內斯，保持著近4輪聯賽不敗的好狀態。桑普多利亞在開局階段表現非常糟糕，前7輪聯賽1勝6負，繼續延續著球隊去年的保級壓力。好在球隊近幾場的表現有了明顯的回暖跡象，反彈趨勢明顯。而且桑普多利亞在本場比賽結束之後，下場比賽將繼續和卡利亞里迎來義大利杯的比賽，雙方要在4天內兩次交手，對於主教練拉涅利來說著實是一個不小的挑戰。

從本賽季表現來看卡利亞里要強於桑普太多，初始指數給出卡利亞里-0.5中高水讓步後市調整至-0.25搭配中水。卡利亞里無論是在實力和狀態方面都要出色的多，他們能夠成為讓球的一方也無可置疑。後市雖說有所下調錶面上是對卡利亞里支持力度衰減但是卡利亞里近期表現出色本場主勝熱度可想而知指數的調整能為主隊減少不少熱度而且搭配中水反而對主隊有利。加上桑普憑藉目前的陣容實力想要和卡利亞里抗衡起來還是有一定的難度。因此本場比較看好卡利亞里主場取勝。比分看好：2-1，2-0，3-1

本場推薦：卡利亞里-0.5，競彩：勝
