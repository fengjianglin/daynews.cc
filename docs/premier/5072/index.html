安切洛蒂：相信球隊能走出困難時刻；換帥傳言不實

2019-12-08 0條評論 15次閱讀 0人點贊

虎撲12月8日訊 北京時間12月8日凌晨，意甲第15輪，那不勒斯客場1-1戰平烏迪內斯，賽後那不勒斯主帥安切洛蒂接受了媒體採訪，回答了記者的諸多問題。

關於本場比賽

「下半場我們表現得很不錯，最重要的是態度的改善，我想這對球隊來說是個積極的信號，但上半場我們的狀態非常糟糕。」

「我現在很樂觀，相信球隊仍然有力量走出困難的時刻。然而，上半場的表現很糟糕，缺乏靈感。」

「我本想說我們的困難時期已經過去了，但事實並非如此。我們需要一個改變的契機，希望能在下周的歐冠中出現。」

「我們必須找到方法來改變像上半場那樣的不利局面。我們嘗試讓邊後衛向前推進，並換上略倫特踢中鋒，情況有所改善。」

「在足球界里，當事情進展順利時，這是球員的榮譽，而當事情進展不順時，教練就該受到責備。這支球隊必須發揮出最好的水平，但是到目前為止我們還沒有做到這一點。」

「我認為此刻恐懼和擔憂佔了上風，這不利於我們保持清醒的頭腦和面對現實的勇氣。球隊在下半場更加努力，贏得了更多的挑戰，很好地踢完了比賽。」

「我們丟的球很可笑，因為我們很清楚拉薩尼亞在這種情況下很可怕。我已經為這場比賽做了充足的準備，如果我們保持穩定的狀態，我們就能繼續斬獲進球。」

「因西涅在上半場表現不佳，但後來球隊中沒人表現好，這也不是他的錯。但他覺得自己有責任，因為他是隊長，他和其他人有同樣的問題。我只是改變了球隊的體系，讓洛薩諾發揮更大的作用。」

關於對亨克的歐冠小組賽

「周二的比賽很重要，我們不能搞砸。我們必須沿用我們之前在歐冠中使用的打法，這跟我們在意甲聯賽中的打法不一樣。」。

此前那不勒斯的動蕩以及換帥傳言

「球隊的更衣室里沒有火藥桶。我聽說媒體對我們的報道，他們說球隊內部已經分裂，隊內經常發生爭吵，這些都不是真的，球隊現在的狀況很好。」

「我們只是嘗試重新調整自己，尤其是在防守上，因為球隊最好的表現正是基於嚴密的防守。」

「加圖索是我的好兄弟，我們最近沒有談過，我也沒問過他換帥的謠言是否真實，因為我相信主席德勞倫蒂斯跟我說的話，他說對我有信心。」

「我從未想過放棄，這正是我過去的經歷教會我的，情況越糟糕，就越能扭轉局面。」

(編輯：姚凡)
