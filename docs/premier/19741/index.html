同一天兩場神劇情3-3！一度3球領先，60分鐘開始連丟3球痛失好局

2019-12-16 133條評論 1,331次閱讀 291人點贊

在北京時間的12月15日，英冠、意甲賽場各出現了一場神劇情比賽，從0-3追到了3-3。

先是英冠第22輪的一場較量，利茲聯主場迎戰卡迪夫城。

近七輪聯賽取得全勝的利茲聯狀態正佳，貝爾薩執教的利茲聯，是本賽季英冠的沖超熱門。而卡迪夫城則位列英冠積分榜中游。

主場作戰的利茲聯，本場開局相當順利，一度3-0領先。

第6分鐘，卡迪夫城獲得角球，但角球罰出後，反被利茲聯抓住機會打出經典的快速反擊，34歲的西班牙老將巴勃羅·埃爾南德斯在本方半場右路得球後，送出一腳精準的斜長傳，左路的葡萄牙前鋒埃爾德·科斯塔快速插上，球到人到，埃爾德·科斯塔突入禁區後推射破門，利茲聯1-0。

利茲聯1-0卡迪夫城

第8分鐘，接隊友的左路傳中，班福德禁區內胸部停球後，左腳順勢射門，利茲聯2-0領先。

利茲聯2-0卡迪夫城

第51分鐘，利茲聯獲得一粒點球，班福德主罰命中，打進個人在本賽季英冠的第9球。利茲聯3-0領先。

利茲聯3-0卡迪夫城

但3-0的比分，並不保險！

0-3落後的卡迪夫城，在接下來的時間裡發起了絕地反擊。

第60分鐘，前鋒湯姆林禁區內得球後，似傳似射地來了一腳，皮球划出一道拋物線，墜入了球門，卡迪夫城扳回一球，3-1。

利茲聯3-1卡迪夫城

第82分鐘，接隊友的邊路傳中，卡迪夫城陣中身材高大的隊長莫里森頭槌破門，比分變為3-2。

利茲聯3-2卡迪夫城

4分鐘後，卡迪夫城隊長莫里森因鏟球動作而被紅牌罰下。比分落後的卡迪夫城，只剩10人應戰。

卡迪夫城隊長被紅牌罰下

比賽進行到第88分鐘時，面對來球，湯姆林跳起來後充滿想像力地用腳後跟一磕，力道恰到好處，

替補上場的德國前鋒格拉策爾直插禁區將球打進，卡迪夫城神奇地將比分扳為了3-3！

利茲聯3-3卡迪夫城

湯姆林與格拉策爾相擁慶祝

一度0-3落後的卡迪夫城，從第60分鐘開始連扳3球，神奇地將比分扳為3-3，在客場拿到寶貴的1分。

而對於貝爾薩的利茲聯而言，3分變為1分無疑是苦澀的。

喜歡蹲著看球的利茲聯主帥貝爾薩

本輪戰平後，利茲聯交出了聯賽領頭羊的寶座，退居英冠積分榜次席，落後榜首球隊西布朗2分。

22輪過後的英冠積分榜

接著則是意甲第16輪的一場較量，維羅納主場迎戰都靈。

客場作戰的都靈，一度3球領先。

第36分鐘，阿根廷球員安薩爾迪小角度爆射，都靈客場1-0領先。

維羅納0-1都靈

第55分鐘，都靈陣中的西班牙中場球員貝倫格爾，高速帶球進禁區後，面對多名防守球員，打出一腳精彩的落葉球。都靈客場2-0領先。

維羅納0-2都靈

第61分鐘，都靈隊員罰出角球，埋伏在禁區外的安薩爾迪左腳打出彈地球，梅開二度！都靈客場3-0領先。

維羅納0-3都靈

3-0領先，依然不保險！

0-3落後的維羅納，從第69分鐘開始神奇地連扳3球。

第69分鐘，都靈隊員禁區內手球，主裁判在回看過VAR後，判罰了點球。

維羅納陣中35歲的鋒線老將帕齊尼將點球罰進。維羅納1-3都靈。這是帕齊尼在本賽季意甲的首球。

維羅納1-3都靈

第76分鐘，帕齊尼禁區內精彩地轉身抽射，都靈門將西里古撲了一下後，皮球擊中立柱後彈出，韋雷補射得手，維羅納2-3都靈。

維羅納2-3都靈

第84分鐘，利用前場自由球的機會，維羅納陣中的波蘭前鋒斯滕平斯基神奇扳平比分，3-3！這是斯滕平斯基在本賽季意甲的首球。

維羅納3-3都靈

最終，3-3的比分保持到全場比賽結束。一度0-3落後的維羅納，神奇地連扳3球。值得一提的是，維羅納本場進球的帕齊尼、韋雷、斯滕平斯基，三人都是在下半時才替補出場，替補出場即進球，堪稱神奇。
