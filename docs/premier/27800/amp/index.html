面對3場丟6分的皇馬，慘遭雙殺的尤文，現在是C羅回歸最佳時機

天天要聞
2019-12-23

北京時間12月22日晚間到23日凌晨，五大聯賽上演兩場焦點之戰備受關注。首先舉行的是義大利超級盃的決賽，尤文圖斯坐鎮主場1:3負於來訪的拉齊奧，在15天內遭遇藍鷹雙殺，而且是同樣的比分輸球。稍後進行的西甲一場焦點賽事中，同樣坐鎮主場的皇家馬德里與來訪的畢爾巴鄂競技戰成0:0平，無奈收穫一場平局，已經落後榜首的巴薩2分。

對於尤文圖斯的這場比賽來說，薩里的排兵布陣再次受人詬病，這場比賽不光透露出主教練在戰術布陣和人員安排上的無能，更透露出尤文在進攻方面的疲軟。坐擁豪華三線人員的薩里，把一把好牌打成這個樣子也不是第一次了。本賽季以來尤文圖斯領先對手2球以上的比賽僅有3次，而僥倖1球小勝的比賽多達9場，更不用說還有3場難堪的平局。薩里這樣的成績在五大聯賽豪門裡都是最差，更不用說與其他聯賽榜首球隊做比較了。

而皇馬方面，更加糟糕，他們對陣畢爾巴鄂競技的這場比賽里坐擁69%的控球率，全場20腳射門，打不進一個球。本澤馬、維尼修斯、克羅斯、莫德里奇、貝爾等人一次次無功而返。加上上一場對陣巴薩時全場佔盡優勢卻不能打破僵局，對陣巴倫西亞到最後時刻才扳平比分。皇馬的進攻線在大把的浪費著機會，僅僅在這三場比賽里他們就丟了6分。更不用說早期對陣皇家貝蒂斯和巴利亞多利德的兩個平局，毫不客氣的說，皇馬的鋒線如果能夠像以往那樣把握住機會的話，他們現在領先巴薩應該是兩位數的優勢。

皇馬方面是進攻機會繁多，而鋒線無能，尤文方面是鋒線有人，但是中場輸送能力不足，這正是當下C羅面對的困境。對於皇馬和C羅來說，事實早已證明，他的離開是個雙輸的局面。可以肯定的是，如果C羅繼續留在尤文圖斯，肯定逐漸淪為薩里的消耗品，而皇馬在缺乏一個鋒線終結者的情況下，仍將處於巴薩下風。而對於34歲的葡萄牙人來說，如果想要再次製造奇蹟和輝煌，現在回歸皇馬是最佳時機。
