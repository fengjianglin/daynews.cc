恆大將簽意甲鋒霸，艾克森談做中國人，中國足球重回起點

2019-12-22 1,483條評論 4,320次閱讀 2061人點贊

恆大預簽意甲鋒霸，艾克森談做中國人，中國足球重回起點

恆大主帥有意阿根廷國腳勞塔羅·馬丁內斯，卡納瓦羅公開表達了對勞塔羅·馬丁內斯的好感。據《sempreinter》報道，廣州恆大主帥卡納瓦羅分享了他對意甲前鋒勞塔羅·馬丁內斯的看法。卡納瓦羅表示：「如果可以的話，我想明天早上和他簽約。他會成為中國足球聯賽中表現出色的球員之一。」

成為中國人的4個月後，艾克森也袒露心聲，在採訪中他表示：「離開巴西來中國踢球，和家人待在一起的時間變少了。」但是，艾克森話鋒一轉：「但我還是做出了這個決定，因為這是為了我愛的足球所要做的事情。」

折騰了八年多的時間，中國足球又重回老路。

2011年8月11日，國足1:0小勝牙買加。比賽結束後，率隊已取得六連勝的高洪波宣布交出帥印。足協請來了卡馬喬。

2019年11月14日，國足世界盃預選賽客場1:2輸給敘利亞，賽後憤怒的里皮辭職。據悉，里皮下課後國足接下來的主教練將在李霄鵬、李鐵以及王寶山三位土帥中產生。

這麼多年來，中國足球不斷折騰，教練一直從本土帶隊成績不管如何都只是臨時，洋帥都會頂替。

如今，這無疑是個令人悲傷的輪迴。
