孔蒂：“他們在成長，有機會首發登場”

2019-12-21 593條評論 4,863次閱讀 771人點贊

北京時間12月22日01:00，國際米蘭將主場迎戰熱那亞，這將是藍黑軍團在2019年的最後一場比賽。在今晚舉行的賽前新聞發布會上，孔蒂表示：

「我們擁有自己的目標，在聖誕來臨時達到更好的競技狀態，力爭更好的聯賽排名。這五個月的時間，由於許多原因，我們不斷在遇到需要去解決的重要問題，以及一些意外的出現，這讓許多球員都承擔起了責任。」

「（阿古梅和埃斯波西托）他們與我們一起訓練，而且都很年輕，他們在不斷成長，有機會在比賽中首發登場。他們擁有成為國際米蘭未來許多年中堅力量的潛質。我非常信任他們。」

「還有一節訓練課，我們會評估一下。我們知道我們現在缺乏可輪換的球員，布羅佐維奇和勞塔羅也將停賽。情況有點緊急，我們會以最好的方式處理現在的情況。我們會盡我們最大的努力，同時相信球迷們也會幫助我們以最好的方式結束這一年。桑切斯很想回來，但醫生們制定的時間表需要得到尊重。他的傷勢是一個打擊，幸運的是勞塔羅和盧卡庫一直能夠上場比賽，埃斯波西托也得到了成長。」

「我很苛刻，同時也很客觀。最近三場比賽，我們理應收穫更多。我不想去談論運氣因素，在這幾場比賽中出現過很多次情況，我們理應用更好的處理方式。我希望看到球隊能夠保持相同的渴望以及比賽強度，同時在各方面都更加專註。我們需要以優異的表現去贏得這場比賽。」

「從賽季一開始，我們的目標就是成為一支值得信賴的球隊。不論是哪一支球隊面對我們，都將面臨一場苦戰。這是我們的目標，我們正為此而努力。整個過程需要耐心，勝利有時候會欺騙你，我們想建立的是一種持久而重要的東西。」

「時間越長，我就越沉浸其中。我知道這是一個全新的計劃，我知道它不會馬上全部到位，需要一個搭建的過程。考慮到這一點，現在最重要的不是獲勝，而是留下一個印記，去灌輸一種工作文化及專業精神。」

「我們的工作還有很長的路要走。我們正在研究一些概念，需要時間來理清一些細節，這些細節有時會給你帶來額外的東西。這段旅程很愉快，我擁有繼續工作的渴望。」
