英德意墊底球隊都贏球，西班牙人隊已成歐洲五大聯賽新的墊底隊

2019-12-23 409條評論 3,974次閱讀 1838人點贊

⚽️過去一夜，歐洲五大聯賽墊底隊齊發力，尤其是德甲、英超和意甲此前排名倒數第一的球隊都取得了難得的勝利。而西甲墊底隊西班牙人的境況則更加糟糕，已經徹底淪為五大聯賽98支球隊目前積分最少的球隊，可以說是全歐倒數第一。

⚽️德甲-帕德博恩17輪12分

昨夜之前，帕德博恩積分只有9分，德甲倒數第一。但是他們在昨夜比賽中2-1擊敗了法蘭克福，拿到了賽季第二場勝利，17輪過後雖然3勝3平11負的戰績依然排名德甲最後一名，但是積分12分已經超過了西班牙人18輪之後的積分，達到了12分。德甲消滅了半程分數不足兩位數的墊底隊，帕德博恩也爆出一個不小的冷門。

⚽️英超-沃特福德18輪12分

昨夜，欺硬怕軟的曼聯再次慷慨送分，此前只贏過1場比賽的墊底隊沃特福德贏下了賽季第二場勝利。他們在主場2-0爆冷擊敗曼聯，也終於讓自己的積分超過了2位數。18輪過後2勝6平10負，繼續在英超墊底，但是積分也達到了12分。

⚽️意甲-斯帕爾17輪12分

本輪之前，斯帕爾同樣以9個積分排名意甲倒數第一，但是他們與都靈隊的比賽客場贏球，為自己贏得了一絲保級的希望。艱難拿下客場勝利後，斯帕爾甚至擺脫了墊底的局面。3勝3平11負，積分超過熱那亞1分，升至倒數第二。不過即便是最新的墊底隊熱那亞，11個積分依然比西班牙人要多上1分。德甲、英超和意甲此前3支積分不足10分的球隊齊齊發力，讓西班牙人徹底淪為五大聯賽最差球隊。

⚽️西甲-西班牙人18輪10分

追平隊史單賽季的最爛戰績，西班牙人還輸掉了與萊加內斯隊的保級生死戰。昨晚比賽之前，他們和萊加內斯同積10分攜手墊底，但是一夜過後，他們坐實了西甲倒數第一的位置，同時還將五大聯賽全歐倒數第一的帽子扣在自己頭上。歐洲五大聯賽98支球隊，目前西班牙人隊的積分最低，沒有之一。

⚽️一夜之間，五大聯賽最差球隊徹底易主，西班牙人不僅是積分最少，勝場、凈勝球也是最少的球隊。半個賽季之後，主場竟然沒有贏過球，9個主場2平7負。而西班牙人隊全歐倒數第一的位置恐怕還要延續，新年過後的第一場比賽對手是同城老大巴塞羅那，再輸一場恐怕也是妥妥的事兒。

⚽️武磊的心理落差可想而知，一年前他效力的上海上港奪得中超聯賽冠軍，個人也是獨攬中超金靴。一年之後，他效力的球隊在五大聯賽排名第98位，積分全歐墊底，甚至有可能在明年下半年直奔西乙聯賽。現在對於武磊和西班牙人俱樂部來說都面臨嚴峻考驗，想要翻身都需要有奇蹟發生。
