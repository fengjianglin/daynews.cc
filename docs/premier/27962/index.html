卡帥：勞塔羅本賽季表現讓我驚訝 薩里是思想最現代的教練之一

2019-12-21 251條評論 1,931次閱讀 819人點贊

直播吧12月21日訊 在接受義大利天空體育記者採訪時，廣州恆大主帥卡納瓦羅談到了關於加圖索、安切洛蒂因西涅、伊布、伊瓜因、薩里、德里赫特、勞塔羅等多個話題。卡納瓦羅表示薩里是當今足壇戰術思想最現代化的主教練之一，並表示這賽季勞塔羅的表現讓自己驚訝。在談到最近兩場比賽薩里都選擇讓C羅、迪巴拉和伊瓜因時，卡納瓦羅表示那是因為他們三個都是最強的球員。

關於加圖索

「我感到加圖索充滿了幹勁，他上賽季就在AC米蘭幹得很漂亮，儘管賽季結束後加圖索沒能帶領米蘭達成目標，並且離任。但是他仍帶領米蘭打出了很精彩的比賽，展現出了很棒的精神面貌，儘管他手下並沒有特別出色的球員。希望他也能在那不勒斯做到這一點。」

關於安切洛蒂

「我們沒人喜歡這個結局，我不知道那不勒斯更衣室內部究竟發生了什麼，所以我也不會對此進行評論，不過有一點是確定的，這種情況並沒有得到很好的解決。」

「我們都知道，那不勒斯是一個很特殊的地方，球員、俱樂部和主教練都需要以不同的方式來處理這種情況。有些事情搞砸了，現在球隊也為此付出了代價。」

關於因西涅

「因西涅的狀態不夠穩定，並且飽受爭議？這都是一些他必須經歷的事情，但他必須重新回到正軌，做那些自己曾一直做的事情。因西涅有很強的技術能力，他也需要在為人方面取得進步，因為他現在是球隊的隊長，他也需要承擔更多的責任。但是，我相信只要他能持續取得進球或助攻，人們就會忘記關於他一切的爭議。」

關於伊布

「伊布已經38歲了，他曾做過很多難以置信的事情，他是一名真正的領袖，但那不勒斯也需要給這個賽季的新援們更多空間，對他們更耐心地培養。伊布確實非常強，但是他已經38歲了。」

「如果一個我是俱樂部高層人員，我會在在一名38歲球員身上進行投資時多加考慮。無論如何，我相信，只要我們一起乘車在那不勒斯轉一圈，我們都會愛上那不勒斯，因為這裡非常美麗。」

關於薩里

「我認為薩里是全義大利最讓人感興趣的人，因為經過他的調教，他的球隊能夠在攻防兩端都發揮出色。他總是試圖控制比賽節奏，並保持很高的強度，薩里無疑是當今足壇戰術思想最現代化的主教練之一。」

關於尤文前場三叉戟：C羅、迪巴拉和伊瓜因

「薩里在最近兩場比賽中，都選擇讓迪巴拉、C羅和伊瓜因一起首發出場，他們幾乎可以保證他們三個在每場比賽中每人都能取得進球，與此同時，薩里只需要在防守端對球員們的站位進行調整就好了。」

「現在的足球已經與以往不太一樣了，現在不再有那種只留在前場不參加防守的進攻球員了。球隊更加強調整體性和組織紀律性，這也是一種充滿魅力的新趨勢。選擇那些最強的球員們首發是對的，C羅、迪巴拉和伊瓜因都是最強的，所以他們才會一同出現在場上。」

關於德里赫特

「在德里赫特身上，我看到了一些與我當年很相似的東西。當年我在贏得世界盃冠軍後離開尤文並加盟皇馬時，我對一切事物都充滿了期待。有段時間我幾乎在每場比賽中都總是會犯一些錯誤，比如吃牌、送點、手球等等。」

「當你加盟一支新球隊的時候，這種事情時有發生。但是我們看到德里赫特有實力，他只是暫時遇到了點小困難。他的性格、技術、身體素質都很強，他是一名非常出色的後衛。」

關於勞塔羅

「這個賽季勞塔羅的表現讓我感到很驚訝。上賽季勞塔羅剛剛加盟國米，他需要適應環境，所以沒有發揮出自己全部的實力。但是這個賽季他做到了，他的進球慾望很強烈，並且還能向自己的隊友們傳遞一種非常積極的信號。他和盧卡庫二人相輔相成，共同促進，並且都發揮得很出色。」

（鐵甲鋼拳）
