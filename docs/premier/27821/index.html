拉齊奧奪得義大利超級盃！第94分鐘自由球重炮，打穿6人的人牆

2019-12-23 1,046條評論 3,927次閱讀 2977人點贊

12月23日，義大利超級盃在沙特首都利雅得進行，由尤文對陣拉齊奧。此前，尤文曾8次奪得義大利超級盃，而拉齊奧則是4次捧杯。

比賽進行到第16分鐘時，西班牙中場球員阿爾韋托禁區內推射破門，拉齊奧1-0領先。

在上半場即將結束前，C羅的射門被撲出後，迪巴拉補射破門，為尤文將比分扳為1-1。

第73分鐘，盧利奇的後點射門，令拉齊奧再次領先，2-1。

第94分鐘，拉齊奧獲得前場自由球機會，尤文在禁區內排出6人的人牆，但依然無濟無事，替補出場的卡達迪一腳重炮，打穿人牆，將比分最終定格為3-1！

拉齊奧第5次奪得義大利超級盃。

在這個月，拉齊奧已兩次擊敗尤文，比分都是3-1！

在本月的8號，意甲第15輪拉齊奧主場3-1逆轉擊敗尤文一役，拉齊奧陣中的阿爾韋托表現同樣相當出彩，奉獻2次助攻：

先是在上半場傷停補時階段，阿爾韋托將球吊入禁區，費利佩用一記強力頭槌，為拉齊奧扳平比分。

阿爾韋托助攻費利佩

之後在第74分鐘，阿爾韋托送出長傳，助攻米林科維奇破門，拉齊奧反超比分。

阿爾韋托助攻米林科維奇

曾效力過塞維利亞、利物浦的阿爾韋托，自2016年夏天加盟拉齊奧後牢牢佔據了主力位置。值得一提的是，2016年夏天從利物浦轉會來到拉齊奧時，阿爾韋托的轉會費僅400萬歐元。如今再看，這筆轉會可謂超值。
