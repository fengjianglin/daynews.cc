12.18精選6串1，尤文取勝即反超登頂，奪冠指數仍遙遙領先

2019-12-19 62條評論 1,126次閱讀 77人點贊

周三 意甲 006 桑普vs尤文

基本面分析：

桑普拿到15個積分，意甲20強中僅排第16。桑普目前僅高出降級區2分，新賽季狀態並不理想。上輪客場1-0險勝積分榜倒數第二的熱那亞，他們方才結束了意甲兩連敗。與此前連續4輪（2勝2平）不敗相比，近期又有被打回原形的趨勢。主場數據方面，桑普本賽季意甲至今2勝3平3負，不敗率超過六成，其中包括了1-0險勝都靈，以及逼平羅馬、亞特蘭大，含金量十足。但是，8戰的得失球是5-9，進攻端疲軟，後防線也存在一定的隱患。最近一次家門口出戰，他們0-1憾負實力平平的帕爾馬，意甲連續4個主場不敗戛然而止。

尤文目前積39分，屈居積分榜次席，尤文目前以凈勝球劣勢不及國米，衛冕之路不可掉以輕心。上輪主場3-1完勝保級隊烏迪內斯，他們方才結束了連續兩輪不勝的尷尬，近期雙線作戰的情況下發揮並不穩定。客場數據方面，老婦人本賽季意甲至今5勝2平1負，8戰的得失球是11-7，攻防兩端均衡。最近一次客場之旅，他們在第69分鐘少一人作戰的情況下，1-3輸給開局不俗的拉齊奧，新賽季聯賽不敗金身告破。另外，當家射手C羅過去4戰斬獲5球，近期狀態不俗。

綜合分析：

雙方最近10次交手，老婦人7勝1平2負，佔據壓倒性優勢。其中，上賽季最後一輪意甲，他們客場0-2不敵桑普，但彼時已經提前奪冠，並無戰意。本輪聯賽是冬歇期前的最後一輪聯賽，尤文可以排出最強陣容來應戰，誓要打破隊在費拉利斯球場兩連敗的頹勢，而桑普的進攻環節沒有調整好，拉涅利的強項是防守反擊，不是進攻，本場碰到老辣的老婦人，以目前球隊的鋒線狀態很難攻破尤文的防線，所以本場看好尤文客場不敗。

周三 西甲 007 巴薩vs皇馬

基本面分析：

巴薩目前僅以凈勝球優勢，力壓皇馬領跑西甲積分榜。巴薩上輪客場被皇家社會2-2逼平，格列茲曼和蘇亞雷斯先後破門，西甲4連勝勢頭暫緩，近期狀態不可謂不佳。主場數據方面，他們本賽季西甲至今7戰全勝，對手中包括了實力不俗的塞維利亞、巴倫西亞。而7戰的得失球是30-9，進攻端依舊無解，但後防線並非牢不可破。最近一次家門口出戰，巴薩則是5-2完勝升班馬馬洛卡。另外，當家射手梅西以12球並列領跑射手榜，蘇亞雷斯也有9球入賬，新援格列茲曼則是斬獲6粒聯賽進球。

本澤馬打進絕平球，皇馬上輪客場與巴倫西亞1-1握手言和，同樣是西甲4連勝勢頭暫緩，近期狀態也不俗。客場數據方面，他們本賽季西甲至今4勝3平1負，正經歷著連續3個（2勝1平）客場不敗。其中，唯一一敗則是0-1爆冷憾負升班馬馬洛卡。8戰的得失球是13-6，攻防兩端均可圈可點。另外，本澤馬打進12粒聯賽進球，與梅西並列射手榜第一。而阿扎爾、貝爾和馬塞洛等人均遭遇不同程度的傷病，出戰成疑。

綜合分析：

此役是第10輪的一場補賽！對陣雙方目前同積35分，且都是少賽一輪。雙方最近10次交手，巴薩6勝2平2負，佔據一定上風。上一次對決中，他們客場1-0險勝皇馬。競彩官方目前給出的指數組合是1.59-3.88-3.88，還是相對示好主隊的。而且，3.88的平局和客勝指數防範性並不強，我看好巴薩此役全取3分。
