周五賽事分析：霍芬海姆大黃蜂大打對攻戰，紫百合主場對戰紅狼

2019-12-19 186條評論 1,173次閱讀 168人點贊

德甲：霍芬海姆VS多特蒙德–3/0

霍芬海姆上輪聯賽客場2-0戰勝柏林聯合，結束球隊4輪不勝的頹勢，目前積24分排在第9位，其中主場3勝1平4負，近6輪2勝1平3負；

多特蒙德上輪聯賽主場在2球領先的情況下最終被萊比錫3-3逼平，目前積30分排在積分榜第4，落後榜首4分，球隊中場核心天才少年桑喬貢獻一傳一射，本賽季各項賽事打進12球並有13次助攻，帶領球隊歐冠打進淘汰賽，將與巴黎聖日耳曼爭奪八強的席位；

聯賽中球隊在客場的成績3勝3平2負，近6輪聯賽3勝2平1負，本賽季球隊的狀態起伏比較大，

兩隊近兩個賽季交手霍芬海姆1勝2平1負與多特平分秋色，其中在主場1勝1平保持不敗，霍芬海姆取得關鍵的勝利後藉此及時止住球隊近期連戰連敗的頹勢，藉助主場的優勢，本場可能會給多特製造更多的麻煩，

而多特近期狀態起伏較大，既有4-0大勝美因茨，5-0大勝杜塞爾多夫，也有0-4負拜仁，3-3平帕德博恩，上輪更是在兩球領先的大好局面下被萊比錫扳平，所以本場看好兩隊分出勝負

意甲：佛羅倫薩VS羅馬–0/1

佛羅倫薩上輪聯賽主場在補時階段絕平領頭羊國際米蘭，保住球隊主場近5個賽季對國際米蘭的不敗戰績，目前在聯賽中積17分排在第16，其中主場2勝3平3負，打進9球失10球，6輪聯賽2平4負未嘗一勝，只打進5球卻丟掉11球；

羅馬上輪聯賽主場3-1戰勝斯帕爾，目前在聯賽中積32排名第4，其中客場4勝3平1負，打進11球失5球，近6輪聯賽4勝1平1負，打進11球失5球，

近兩個賽季雙方交手佛羅倫薩1勝2平1負，其中在主場1平1負，結合兩隊近期的狀態，本場看好羅馬客場不敗

西甲：埃瓦爾VS格拉納達–3/0

埃瓦爾近5輪聯賽僅取得1平4負，目前積16分排在第17，其中主場3勝5負，打進8球失15球，球隊傷病多，阿爾瓦雷斯，阿爾比拉，拉米斯，安赫爾均有傷本場不能上場；

格拉納達近期黑馬的成色在漸漸褪去，近6輪僅取得1勝1平4負的成績，目前在聯賽中積24分排在第9，其中客場2勝2平4負，球隊中的維科，蒙托羅的受傷對球隊影響較大；

兩隊近期球隊的傷員都較多，狀態起伏不定，埃瓦爾在主場打法較激進，且在主場沒有平局，非勝即負，所以本場看好雙方分出勝負
