周日026 西甲 畢爾巴鄂VS萊萬特

比賽時間：11月10日21:00

畢爾巴鄂實力、名氣高於萊萬特。不過球隊本季西甲12戰4勝5平3負積17分排名第11位，表現一般，跟客隊同分，沒有優勢。近7場1勝3平3負，僅擊敗倒數第2的西班牙人，狀態糟糕。這令他們今天熱度有限。當然，值得注意的是，球隊6個主場4勝1平1負，入9球丟2球，十分強勁，延續了主場龍作風。陣容上，只有主力中場貝尼亞特因傷缺席，人員基本齊整。往季交鋒，近6戰萊萬特3勝1平2負，優勢不大。不過近5個主場面對萊萬特4勝1負，優勢明顯。

萊萬特實力、名氣不如畢爾巴鄂。不過本季西甲12戰5勝2平5負積17分排名第12位，表現還行，跟主隊同分。近2場連勝，上場更是主場3-1大勝巴薩，爆出大冷。這令萊萬特關注度陡增。不利的是，客場6戰2勝4負，入7球丟10球，較為糟糕。陣容上，主力後衛托尼奧，老傷號主力後衛蘇亞雷斯，因傷缺席。主力中場羅奇納因傷出戰成疑。

機構普遍主讓0.75起步，後市主隊回報不斷走低至低回報。基本面上，畢巴雖然實力、名氣佔優，但是表現一般，跟客隊同分，而且近期狀態糟糕，無法給予玩家太強信心。另一方面，萊萬特表現不差，且近2場連勝，尤其上場主場3-1擊敗巴薩爆出大冷，關注度陡增。這種背景下。畢巴依然主讓0.75優勢讓步，尤其後市回報不斷走低至低回報。其實受到機構不小的支持。結合畢巴主場出色，萊萬特客場欠佳。綜合考慮，本場我看主勝。競足推薦：勝。

周日033 桑普VS亞特蘭大

比賽時間：11月10日 22：00

桑普多利亞上輪在保級大戰中客場對陣斯帕爾，0比0的僵局一直保持到傷停補時最後1分鐘，由詹盧卡·卡普拉里(剛剛替補上場1分鐘）打進絕殺一球，1比0險勝的同時終結了連續6輪不勝（這也是本賽季桑普的第2場勝利）。老帥拉涅利執教4輪首獲、取得1勝（2平1負），球隊上下正在慢慢恢復對拉涅利的信任。

賽後拉涅利承認，他的球隊仍處於降級區，還要為保級而繼續努力，同時也表揚了卡普拉裡帶回來的金子般的3分。

桑普鋒線老將誇利亞雷拉因為大腿問題，錯過了上輪與斯帕爾的比賽，本場應該能夠回來了，儘管還有待觀察誰會與他搭檔進攻，估計會是在加比亞迪尼與博納佐利之間進行選擇。

周中歐冠主場作戰，亞特蘭大1比1踢平榜首球隊曼城，終止了歐冠小組賽3連敗，終於拿到了有史以來第1個冠軍聯賽積分。最後10多分鐘，曼城門將被罰下，亞特蘭大以多打少佔據氣勢上的上風卻未能再進一球獲勝，令加斯佩里尼覺得有些遺憾。

在上周的聯賽中，亞特蘭大卻是自毀長城，不但送上烏龍大禮，還有隊員紅牌罰下，在主場0比2輸給了排名第6的卡利亞里，告別了聯賽8輪不敗（5勝3平），同時還中斷了連續16場進球的紀錄——近2輪聯賽1平1負不勝。

亞特蘭大6個聯賽客場4勝2平不敗（僅次於客場6連勝的國米），2場平局是踢平兩支強隊拉齊奧(3比3)、那不勒斯(2比2)——客場指數6連紅了。

主帥加斯佩里尼不得不再等一場比賽，才能等來本隊主力射手薩扎帕塔復出，後者的內收肌肌肉問題還未徹底好利索，而另一名前鋒伊利契奇上輪對陣卡利亞里時因報復性犯規吃到紅牌、被禁賽2場，這是第一場。好在荷蘭中場德魯恩解禁復出了。

從歷史往績上看，桑普多利亞對陣亞特蘭大時表現不錯，過去6場獲勝4場、近10次取得6勝1平3負佔優，近10次主場則是贏了7次，其中上賽季主場1比2負，終止主場對亞特蘭大的2連勝（2比1、3比1）——但目前兩者之間仍然存在不可否認的差距：亞特蘭大以打進30球成為聯賽最佳進攻球隊，而桑普至今只進7球（並列意甲鋒線最差）。

亞特蘭大去年在桑普的主場拿下了3分，但要注意的是，自1992年以來，亞特蘭大還沒有連續2次在客場擊敗過桑普。

關注焦點：路易斯·穆列爾（亞特蘭大）——這名哥倫比亞前鋒僅參加過2次對陣桑普多利亞的比賽，但他在與老東家的交鋒中已打進3球。穆列爾本賽季聯賽已經收穫8球，這很可能是他最後一次打首發前鋒了，因為他的同胞薩帕塔下一輪就可能復出。

上賽季：桑普多利亞1-2亞特蘭大

本場熱門方向：客隊可能獲勝/穆列爾可能有進球。

本場競彩指數是3.85、3.70、1.69，國際指數有對亞特蘭大看衰的一個退讓趨勢，因為要考慮到亞特蘭大是雙線作戰（且自1992年以來，亞特蘭大還從沒有在客場2連勝過桑普）、桑普為了保級有強大的求生欲。加之亞特蘭大意甲客場指數已達6連紅，本場我想選擇冷門保級球隊桑普不會輸給歐冠參賽球隊、競彩讓勝
