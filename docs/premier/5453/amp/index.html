歐冠強強相遇專家預測：“鳥叔”有望兩連勝 巴薩拒絕再犯錯

天天要聞
2019-11-26

本周中歐冠小組賽第5輪將展開多場激戰，目前巴黎聖日耳曼、拜仁慕尼黑與尤文圖斯三隊提前出線。本周一些小組即將迎來明朗的出線結局。明晨同樣有多場強強相遇的重磅對決，平局是不可忽視的首要選擇。

皇馬VS巴黎聖日耳曼

A組頭名大巴黎本輪客戰第二名皇馬。首先從出線形勢來說，皇馬如果輸掉本場比賽，假設布魯日取勝加拉塔薩雷，那麼最後一輪客場挑戰比利時球隊則成了生死戰。也就是說，本場比賽至少要拿到1分，皇馬才能確保出線。巴黎領先皇馬5分，高居積分榜第一已經出線，最後一輪大巴黎坐鎮主場將面對加拉塔薩雷，只要取勝穩居第一，本場比賽大巴黎可謂無欲無求。兩隊交手，皇馬上賽季歐冠雙殺對手，本賽季客場0比3潰敗。威廉為此役開出2.25 3.70 2.90，亞盤普遍以讓平半開盤。拿1分兩隊皆大歡喜，平局可以抬高有阻的嫌疑，還是更有可能。第二落點是皇馬取勝，巴黎拚命攪局的可能性不大。推薦首1次3。

尤文VS馬競

D組的尤文已經提前出線，拿到1分可確保小組第一。馬競末輪將在主場迎戰莫斯科火車頭，此戰如果取勝可以鎖定出線權。如果馬競本輪拿到1分，則末輪迴到主場保證不敗也可出線。此役和局同樣是個大家都可以接受的結果。尤文本賽季各項賽事14勝3平17場不敗，狀態好於近期客場4平1負5場不勝的馬競。威廉初賠1.91 3.30 4.33，亞盤平半至半球盤口略有分歧。馬競想要把握主動權此役不能空手而歸，看好客隊不敗，首選平局。小寶熊

「鳥叔」有望兩連勝

皇馬VS巴黎聖日耳曼。A組巴黎四戰四勝，已獲得出線資格。皇馬積7分，出線基本也沒什麼問題。雙方首回合巴黎主場3比0大勝。雙方歷史交鋒基本都在歐冠賽場，在2015年和2018年都交手過，皇馬3勝1平保持不敗。本賽季皇馬一掃上賽季的頹勢，目前與巴薩同積28分位居西甲榜首。本場比賽，皇馬讓出平半中高水。皇馬取分的動力更高一些。看好皇馬主場不敗。

熱刺vs奧林匹亞科斯。B組形勢和A組相似。拜仁12分獲得出線權，熱刺積7分。熱刺本場必須取勝，這樣才能在最後一場客場對陣拜仁時輕鬆上陣。熱刺剛剛更換了主帥，穆里尼奧賦閑多日後再次出山，首場執教客場3比2拿下西漢姆。由於穆帥的來臨，熱刺再次成為眾人關注的焦點。本場比賽亞盤開出熱刺讓球半中水，看好熱刺在穆帥的帶領下，再次取得一場勝利。

尤文vs馬競。雙方首回合馬競主場2比2戰平尤文。目前尤文積10分已經出線，馬競積7分。但本場比賽不能輸給對手，否則很可能失去小組首位。在2014賽季兩隊也在歐冠同在一組，當時馬競一勝一平，力壓尤文獲得小組第一。而在上賽季歐冠十六強對陣中，馬競主場2:0領先的情況下，被尤文回到主場3:0翻盤，C羅獨中三元。本場比賽，尤文讓半球中水，目前降為平半中水。尤文多名主力受傷或不能上場，包括C羅、基耶利尼、曼朱基奇等。雙方有打平可能，主隊不敗。蕭少澤

巴薩拒絕再犯錯

巴塞羅那VS多特蒙德。死亡之組巴薩上輪犯錯，在弱旅布拉格斯拉維亞身上只拿到一場平局，雖然並不算致命，但是這讓巴薩的出線之旅變得有些複雜。考慮到巴薩最後兩輪的對手是多特蒙德和國際米蘭，一旦出現巴薩兩連敗就將無緣淘汰賽。如果巴薩在本輪擊敗只比他們少1分的多特蒙德，那麼球隊就會提前出線，而且鎖定小組第一。此役亞盤多以巴薩讓一球盤受注，作為兩大豪門的對決明顯高看巴薩一籌。縱然巴薩擁有實力上的優勢，一球盤也略深。如果臨場堅挺一球盤，筆者認為巴薩讓球走兩頭穩妥。總體而言，更看好多特從諾坎普全身而退。推薦：巴薩讓一球0。推薦比分：巴薩1比1、2比2、2比3。徐凱
