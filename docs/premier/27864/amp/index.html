PK掉安切洛蒂的菜鳥教練阿爾特塔能充實阿森納文化嗎？

天天要聞
2019-12-22

阿爾特塔在瓜迪奧拉身邊卧薪嘗膽三年有餘，阿森納則在big6的浪潮中漸行漸遠。兩個並不相似的靈魂走到一起，雖然西班牙人曾在槍手效力5年，但單憑「緣分」一說，頗為牽強。

眼下的這支阿森納早已面目全非，溫格臨走之前奢望過的「阿森納的文化價值」慢慢變形了。已經淪為英超笑柄的阿森納需要改革領路人，他們給予了新帥十足的面子，至少官網的歡迎致辭多了幾分獻媚討好的意思：「阿爾特塔從七名候選人中脫穎而出」。

槍手歷史上第20任主帥

群眾有明辨是非的能力，也有去偽存真的權利，作為英超最年輕的主帥，阿爾特塔容易被關注，被審判，被期待。

  • 教練時代三奪歐冠冠軍的安切洛蒂不香嗎？

本賽季的那不勒斯內憂外患，未能守住「千年老二」的底褲是俱樂部主席動了殺心的根本原因。但所有人都知道今年意甲的主旋律是尤文圖斯和國際米蘭的二人轉。即便那不勒斯的球員工資在亞平寧半島高居前五，但這並不是「唯我獨尊」的必要條件，羅馬雙雄和黑馬卡利亞里也有權利享受更上一層樓的快感。

人，一旦陷入了回憶的死結，再多的眼前利益都會被忽視，那不勒斯在歐戰小組賽的征程中已經證實過他們並非沒有即戰力，不然能兩回合從衛冕冠軍手中搶回彌足珍貴的4分，只是德佬給安切洛蒂的硬任務是在意甲有所作為，即便外戰內行，還是辜負了他650萬歐的高額年薪。

本是命運的賞賜，但阿森納卻「不受嗟來之食」，哪怕安切洛蒂九年前在英倫舉起過英超和足總杯雙料冠軍，但時過境遷，落魄的阿森納以豪門自居，被解僱的義大利人卻成為昨日黃花。據傳，阿森納董事會都未曾邀請安切洛蒂面試，這種爭議當然備受指責，甚至連阿森納自家的名宿都在憤憤不平：「應該安切洛蒂給阿森納面試」。

「一個羊群已經是很散亂的組織，平時大家在一起盲目地左沖右撞。如果一頭羊發現了一片肥沃的綠草地，並在那裡吃到了新鮮的青草，後來的羊群就會一哄而上，爭搶那裡的青草，全然不顧旁邊虎視眈眈的狼，或者看不到其它還有更好的青草」。這就是羊群效應。

阿森納理直氣壯的理由無非是想選「一頭有思想的羊頭」，但其實當初拿著ppt面試的另一個西班牙人何嘗沒有打動人心，「歐聯之王」埃梅里不能入鄉隨俗，並不代表連續從拜仁和那不勒斯黯然下課的安切洛蒂也是無用之人，就連溫格在永貝里上任之初都在建言獻策：「當下的阿森納需要一位有經驗的名帥來維持」。

有些人，一旦錯過就不在

在五大聯賽都均有斬獲的安帥即便在走下坡路，但絕非紙老虎，何況英超輕車熟路，即便效果無法立竿見影，但他的經驗值卻是應付槍手當下混亂的良藥。在拜仁時期，他拿下功勛人物穆勒、里貝里和羅本過於激進，飽受批評，但年年習慣性賣隊長的阿森納並沒有絕對大佬，以好人形象示人的安切洛蒂完全可以化身硬朗的改革派，既然沒有什麼可以失去了，何必怕一無所有，可惜，阿森納董事會一朝被蛇咬，十年怕井繩，內心深處，他們已經將英超歷史勝率排第5的安切洛蒂和埃梅里劃分為一類人。

  • 零輸球的教練得到了工作

踢而優則教慢慢地成為足壇的一股主流風，遠有瓜迪奧拉之於巴薩，近有蘭帕德之於切爾西，索肖之於曼聯，阿森納選擇阿爾特塔自然存在這樣的因素。作為隊長的他深受球迷愛戴，如今以教練身份回歸酋長球場，阿森納球迷自然不會排斥，但情緒這種東西都是善變的因子，在阿森納效力九年，榮譽更多的永貝里身上不也曾被寄託過這種「DNA情懷」，但並未有任何起色的阿森納足以說明情懷並不是解藥，拉卡澤特和奧巴梅揚的王炸組合進球勢頭不減，但紙糊的防線也未曾得到修繕。

阿爾特塔和永貝里身上有太多相似之處

雞湯，這種乏味的東西並不是任何教練都可以得心應手，沒有任何執教經驗的阿爾特塔當然會以名宿的身份和球迷、球員互動，但更多人覺得他是個菜鳥，穆里尼奧一針見血：「最好的教練是那些贏得更多勝利的人，而現在最好的教練是那些輸得最少的人」，這並不是為了站隊安切洛蒂，而是直指阿森納的草率用人。

阿爾特塔的野心可見一斑，在他退役之時其實有三種選擇，阿森納青訓教頭，阿森納助理教練和瓜迪奧拉的助教，迫切需要養分的他選擇了曼城，而與阿森納再續前緣其實還有一個直接原因，那就是曼城對於他將來接任主帥的要求並不明朗。

但千萬不要將平台當作本領，千萬不要將瓜迪奧拉菜鳥賽季奪冠的例子理解為常態。何況中規中矩的蘭帕德就因為對決強隊鮮有勝績也遭受質疑，一直在曼城助教崗位上的阿爾特塔肚子里有多少墨水，並不與曼城的成績掛鉤，但就怕這是阿森納甄選領路人的錯誤標準，因為西班牙人身上除了阿森納元素外，還有太多揮之不去的標籤，譬如出自拉瑪西亞青訓，譬如一直深受瓜迪奧拉足球哲學影響，吝嗇的克倫克為成全阿爾特塔的買人計劃嗎？這是一個大大的❓

阿爾特塔此番入駐酋長，時間上肯定久於臨時主帥永貝里，何況槍手經不起一個賽季三換領路人的波折，剩下的半個賽季幾乎是阿爾特塔練兵的靶場，為此他有諸多工作需要做，諸多球員需要觀察，這是一個被董事會允許經歷挫折的周期。

阿爾特塔知道球迷愛他，但他不在酋長的日子裡，也深知這裡的球迷越來越沒有耐心。溫格、埃梅里、永貝里無一例外，作為新帥，他的第一聲發令槍硝煙瀰漫，他說：「如果我的團隊中起到了不良影響，那麼抱歉，你可能不太適合我們」。

言下之意，不破不立，或許相比其他新帥的委婉，師出有名的阿爾特塔更珍惜這份使命，不過新阿森納並未有任何進展，在與埃弗頓的比賽中，槍手的第一腳射門出現的時間節點居然在臨近結束的那一刻，為坐在看台上觀摩的阿爾特塔留下了「撥亂反正」的素材，賽前他就說過「重點觀察球員的投入度」，而另一邊，埃弗頓的新帥安切洛蒂也在思考同樣的問題，或許這是暗戰的序篇。

總之，來日方長，愛情里都講究三觀，但別忘記，人際交往的本質是價值的互換。這些天，人們給予阿爾特塔的頭銜是「實現夢想的新人」，過些日子，究竟是這一次選擇是「冒險」還是「明智」，成績永遠是上綱上線的第一把枷鎖。#我要上頭條#
