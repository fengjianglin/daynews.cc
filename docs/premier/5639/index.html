低級烏龍斷送國足世界盃夢，悲情烏龍釀悲劇，盤點那些經典烏龍球

2019-11-17 0條評論 16次閱讀 0人點贊

這幾天對於恆大鐵衛張琳芃而言勢必是難熬的一段時光，因為在對陣敘利亞時最後時刻都烏龍球，張琳芃似乎已經成為了球迷們心中的罪人。雖然作為一位中後衛，在場上失誤打進烏龍球也是在所難免的，但在國足衝擊世界盃這個關鍵節點，在國足大賽中屢屢讓人失望憤怒的這個前提下，球迷們也需要找到一個情緒釋放的出口，而張琳芃這個烏龍球著實是太搶眼了，這使他分擔了球迷們的所有”火力”。

最近幾天去張琳芃的微博上參觀一番，就可以看到憤怒的球迷們已經火力全開，各種問候張琳芃的方式都有，甚至還給張琳芃起了一個”敘利亞名宿”的外號。這粒烏龍球雖然不至於讓國足衝擊世界盃徹底失敗，但至少直接促使了里皮的下課，而中國足球也很有可能因為這個烏龍球而再次走上一條新的重建之路，所以從這個角度來看，這個烏龍球的影響還是巨大的。當然，在世界足壇範圍內，意義重大影響深遠的烏龍球並不佔少數，接下來就盤點幾個世界足壇讓人難忘的致命烏龍球。

國際米蘭 馬特拉齊

馬特拉齊作為一位中後衛是一個進球好手，出色的搶點意識和頭槌能力讓他小在職業生涯中收穫了不少進球，當然在進烏龍球上，他也是一位好手。在05到06賽季意甲聯賽中，國際米蘭客場挑戰恩波利，比賽進行到了最後時刻場上比分依然是0:0。而在恩波利一次反擊中馬特拉齊拿到了球，恩波利全隊也上前逼搶，馬特拉齊慌亂情況下選擇將球回傳門將。然而由於沒有看到塞薩爾的站位，這個球卻成了一個吊門直接飛入到了自己球門。超過五十米的烏龍球，馬特拉齊也創造了歷史。

全運會新疆隊

在2013年全運會賽場上，新疆隊打進了一個這麼多年讓球迷們津津樂道的烏龍球。那是東道主遼寧迎戰新疆的比賽，當比賽進入到下半場時遼寧隊高空球傳入新疆隊禁區，新疆隊後衛解圍失誤球向著自家球門方向而去。緊接著4號隊員的倒鉤解圍也踢空了，皮球彈地後碰到2號球員，守門員出擊的停球居然導致球速加快飛向自家大門，最終一粒經典的烏龍球誕生。這粒烏龍球也迅速成為了球迷們討論的焦點話題，而這樣雷人的自由球確實讓人哭笑不得。

哥倫比亞埃斯科巴

前文中提到的幾個烏龍球多少都帶有喜劇色彩，但最後這一個就完全不同了，在世界盃賽場上，一位叫做埃斯科巴的哥倫比亞後衛因為一粒烏龍球結束了自己的生命。在1994年美國世界盃小組賽中，埃斯科巴在與美國隊比賽中打進了一個烏龍球，而這個球也直接導致了哥倫比亞隊出局。而在他回國從麥德林市一家酒吧出來時，12發子彈射向了他，最終他失去了自己年輕的生命。警察在調查這起槍殺案後給出的結果是被害者被槍殺的原因很大可能是因為他在世界盃上的烏龍球。
