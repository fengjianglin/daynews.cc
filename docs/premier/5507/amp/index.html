“斑馬軍團”全歐唯一不敗！是玄學？還是實力？

天天要聞
2019-11-24

北京時間11月23日晚22:00，2019-2020賽季意甲第13輪比賽，尤文客場對亞特蘭大率先開戰。下半場比賽，戈森斯頭槌得分，隨後伊瓜因梅開二度並助攻迪巴拉破門。全場比賽結束，尤文圖斯客場3-1擊敗亞特蘭大獲得勝利。

16分鐘，赫迪拉禁區內手球送點，巴羅主罰點球，皮球擊中橫樑後彈進球場內，帕薩利奇頭槌補射，皮球高出橫樑。

巴羅主罰點球擊中橫樑

第56分鐘，巴羅禁區內右側的傳中，戈森斯在禁小禁區內左側的頭槌攻門，皮球入網，亞特蘭大1-0尤文。

戈森斯在禁小禁區內左側的頭槌破門，亞特蘭大1-0尤文。

第74分鐘，伊瓜因在禁區中路得球後轉身抽射，皮球入網，尤文圖斯1-1亞特蘭大。

伊瓜因扳平比分 1:1

第81分鐘，誇德拉多在禁區內右側底線附近的傳中，伊瓜因禁區中路包抄射門得手，尤文圖斯2-1亞特蘭大。

伊瓜因反超比分 1:2

補時絕殺！93分鐘迪巴拉接到伊瓜因的斜傳，殺入禁區右側，閃開射門角度後抽射得分。

迪巴拉錦上添花 1:3

德里赫特全場五次解圍，三次封堵，四次攔截、四次頭槌爭搶成功，各數據均在隊內位居前列，在《whoscored》的評分中，德里赫特得到7.8分，在後防球員中位居第一。

2018年度的歐洲金童 德里赫特

雖然薩里派上了貝爾納代斯基、皮亞尼奇、本坦庫爾和赫迪拉四名中場球員，面對著亞特蘭大的高壓防守卻始終無法將皮球控制在自己腳下。

單從比分上看，如果不看比賽說尤文取得了一場豪勝也不為過，但是過程卻是截然相反，全場75分鐘一直被亞特蘭大摁著打，直到伊瓜因扳平比分，尤文的局勢才有好轉。要不是德里赫特與什琴斯尼的超常發揮，上半場就會被打花。

雖然說比賽最終只要結果，但是如果薩里不解決中場問題，每場都是靠球星來續命，運氣和玄學不會總是有的！

賽季還很長，大家覺得尤文在聯賽會不會掉鏈子呢？在歐冠又能走多遠呢？
