如果法夫爾下課，誰出任多特蒙德主教練合適？

2019-12-03 0條評論 16次閱讀 0人點贊

客場征戰柏林赫塔，憑藉上半時的閃電兩連擊取得三分，也讓法夫爾暫時躲過下課危機，但今年多特蒙德的戰績不盡人意，聯賽排名，歐冠聯賽還面對最後一輪斯拉維亞的挑戰，就憑斯拉維亞客戰巴薩拿出的勇氣，多特蒙德不敢說一定能贏吧，而法夫爾抓住第二根稻草，那麼第三根稻草-歐冠出線，能抓住嗎？如果抓不住下課，誰接替法夫爾合適？

一、法夫爾能抓住歐冠這根稻草嗎？

先看下多特蒙德所在小組形式。

圖片為自製

從圖中可以看出，多特蒙德在凈勝球方面落後國際米蘭，雙方的勝負關係上國際米蘭也佔優，所以僅從現有情況看，多特蒙德形式不如國際米蘭。國際米蘭只要最後一輪主場對陣巴塞羅那的比賽中取得3分即確定進入歐冠淘汰賽，國際米蘭近期狀態很好，主場2:1贏下斯帕爾，成功登頂意甲，這對國際米蘭信心加成極大，所以現在球隊狀態上佳，主場面對巴塞羅那挑戰，應該是奉獻一場精彩比賽。

梅阿查球場

而多特蒙德最後一場對陣斯拉維亞，雖然客場以2:0取得勝利，但看看現在的多特，近期狀態糟糕，客戰剛剛換帥的柏林赫塔，憑藉上半場閃擊以2:1取得勝利，聯賽也只排名第五，而斯拉維亞雖硬實力不強，但至少展現出一定戰鬥力，相信多特蒙德在主場雖能取得勝利但不會輕鬆。

多特蒙德主場

實際上，多特蒙德能否出現，還要看巴塞羅那的表現才行。巴薩最後一輪做客米蘭城，能否保證不輸球？我想是的，憑藉剛剛獲得的第六座金球獎，梅西成為藍星上在金球獎項上的第一人，攜此之威，更希望在歐冠舞台上更近一步，雖巴塞羅那已經確定小組第一出線，但相信全體將士不希望以一場失利成為小組賽的收官之戰。看好巴塞羅那不敗。

綜上，我想最後該小組出線應該是巴塞羅那和多特蒙德，多特蒙德主場贏球會很艱難，而國米主場可能會是一場平局。

二、法夫爾如果下課，有誰會接任？

客觀上看，多特蒙德對比薩其馬吸引力很不足，對比馬競、熱刺等等，俱樂部內部又略顯小家子氣，對比阿森納、曼聯等落魄豪門，城市、市場也不佔優勢。所以如果法夫爾在聖誕節前後下課，我想按照俱樂部的操作思路，大概率先找救火教練，然後又在幾個聯賽里淘寶。

1、救火教練

如果真的聖誕節前後換帥，我猜可能是羅傑施密特，原來勒沃庫森和北京的那位，在勒沃庫森後期被球迷調侃將藥廠變為伐木大隊的訓練世界級，比賽小區級的名帥。

半路換帥，我想多特蒙德的賽季目標就是晉級歐冠，所以在技戰術打法上不會要求太多，而施密特的伐木戰術可能在德甲還比較適合，看看下圖。

這是13輪過後的前6球隊進球和失球的數據，多特蒙德的失球數是最多的，而進球數只能並排第三，所以只從13輪的數據看，多特防守並不穩，出現主場2:2弗萊堡的情況也正常。

羅傑施密特所擅長的高位逼搶加高速轉換，陣型以442為主，輔以4321/4312切換，利用前場的瘋跑逼搶，對邊鋒位置需求高，但對單兵作戰能力強的中鋒要求並不高，這種戰術要求簡直就是為多特蒙德量身打造，看看主場對巴塞羅那時的陣容。　

1-比爾基；5-哈基米，16-阿坎吉，15-胡梅爾斯，13-格雷羅；28-維特塞爾，6-德萊尼；7-桑喬，11-羅伊斯，23-小阿扎爾(73′,19-布蘭特)；9-帕科

也許羅傑施密特需要做的可能就是將羅伊斯的位置前提，與帕科組成雙前鋒，充分利用前場四人組的跑位吧。

當然，救火教練來說，名宿薩默爾也合適，但是否能出任除了看俱樂部的態度外，薩默爾自身的健康狀況才是根本原因。

2、長期教練

2.1、如果羅傑施密特戰績不錯，轉正也不是沒可能。但多特蒙德後續的人員儲備是否能滿足技戰術打法需要還要打問號。至少現在的消息看，瓦茨克準備冬歇期買中鋒了。抓緊吧！

2.2、 諾維奇教練——法爾克

近期新聞爆出多特蒙德有意法爾克寫消息。法爾克現為英超諾維奇的主教練，之前是多特蒙德二隊教練，在2017年5月成為諾維奇的主教練，隨後在本賽季升入英超，作為主教練，法爾克的特點是攻強守弱，善於激勵球員，善於挖掘新人的同時又會給俱樂部省錢，典型案例就是普基。這個賽季的普基我就不詳述了。我想單是省錢這一項應該就足夠贏得瓦茨克的心了。

我大概是從2011年左右開始看多特，當時是歐冠小組賽，多特客戰阿森納，看了半場的感覺是多特太強了，玩命跑，無論阿森納如何傳導球，在接球人面前永遠有多特蒙德的球員，半場結束時簡直就是半場攻防，這是當時的阿森納，可不是現在的二三流阿森納。我喜歡看高位逼搶的足球風格，所以現在的這支多特蒙德反倒是不如利物浦看的多，所以我希望多特蒙德如果換帥，能找個打繼續打高位逼搶的教頭，不要在迷戀所謂的傳控了。在羅伊斯不多的健康職業生涯里，為其也為俱樂部多贏下幾個冠軍吧！

也許有人會說波切蒂諾合適，我想波切蒂諾也合適，但他不會來多特蒙德。皇馬、拜仁、巴薩甚至尤文現在主教練的位置都不穩，作為一個因為預算而與俱樂部鬧翻的主教練，肯定會希望能到一個願意投入的球隊就職，何況又是豪門！所以我想波胖肯定不會來。

也許對於多特蒙德俱樂部、球迷、球員來說，最合適的還是那個在場邊激情慶祝打出一套組合拳的高大男人！
