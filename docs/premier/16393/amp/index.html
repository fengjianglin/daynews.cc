意甲：佛羅倫薩VS國際米蘭

比賽時間：2019/12-16~03:45

佛羅倫薩戰況

上輪主場對戰升班馬萊切的比賽中不僅主力邊鋒里貝里受傷，最終還輸掉了比賽。最近一輪客場挑戰都靈的比賽中表現其實並不差，不僅沒有被壓制反而稍佔上風，但卻被都靈率先攻入2球，只在最後傷停補時打進一球以1-2落敗而歸，遭遇聯賽4連敗，如今整隻球隊猶如被籠罩在霧霾之中。根據最新的消息稱，由於球隊成績不佳，這場對戰國米的比賽是蒙特拉挽救帥位的最後機會，如今佛羅倫薩已經在評估五名主帥候選人，他們分別是埃梅里、普蘭德利、迪比亞吉奧、迪馬特奧和斯帕萊蒂。

國際米蘭戰況

繼上賽季歐冠被熱刺逆襲後，國米在周中只能戰勝巴薩晉級歐冠淘汰賽的情況下，球隊主場1-2負於已經提前出線的巴薩，隊史首次連續2年止步歐冠小組賽，只能無奈繼續參加歐聯杯了。歐冠出局也並完全是壞事，至少國蘭可以專心與尤文在意甲中一較高下。不過上輪戰平羅馬差點令國米丟掉榜首，幸好斑馬軍團意外落敗才保住領頭羊位置，但這樣的幸運可一不可再，球隊需要時刻保持警惕。國米近兩場比賽只打進了1球，主帥孔蒂已經深受球隊陣容不足之苦，勞塔羅和盧卡庫這對前鋒組合雖然本賽季發揮出色，但過度使用明顯感到疲態。

綜合分析

國米聯賽至今作客7戰全勝，乃意甲目前作客搶分能力最強隊伍，強勁的搶分能力得益於出色的進攻表現，球隊目前聯賽作客均場入2.57球亦是冠絕意甲。雖然本賽季作客取得7戰全勝狀態相當火熱，但國米已連續5個賽季作客未能擊敗佛羅倫薩，加上中場多名主力傷退，機構開出國米讓出半球並搭配高水對客隊阻力大，結合兩隊近況，個人認為本場比賽國米難爭勝。
