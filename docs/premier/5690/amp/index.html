奧拉羅尤要加油了，你給蘇寧足球拖後腿了，看看另外2支球隊吧

天天要聞
2019-11-14

江蘇蘇寧易購集團自從進入足球領域之後，開始大力發展自己的足球王國，目前已經擁有了國際米蘭、江蘇蘇寧男足和江蘇蘇寧女足三支球隊。而且三支球隊目前的發展也還都算不錯。

不過相比之下，奧拉羅尤的江蘇蘇寧男足似乎有點拖了蘇寧足球的後腿，因為和另外兩支球隊的成績比起來，奧拉羅尤的江蘇蘇寧男足好像是成績最弱的一支隊伍。

本賽季，國際米蘭在聯賽中表現相當令人滿意，12輪比賽10勝1平1負，排名聯賽第二，只落後尤文圖斯1分，對於多年未曾染指的意甲冠軍發起了猛烈衝擊，也給意甲的霸主尤文圖斯帶來了很大的壓力，蘇寧的投入終於在本賽季看到了回報。

而江蘇蘇寧女足，本賽季的成績更是讓人驚訝。球隊在女足中超聯賽中，完全是一支獨秀，早早的就確立了奪冠的優勢，將其餘對手遠遠甩在身後。不僅如此，蘇寧女足還包攬了女足錦標賽和足協杯的冠軍，實現了賽季的三冠王大業，成為了中國女足俱樂部中的王者。眼下，蘇寧女足又將參加首屆亞洲女足亞冠聯賽的冠軍爭奪戰，一旦奪冠，將成為史無前例的四冠王。

江蘇蘇寧男足的成績雖然在中超球隊中也算不錯，但是與前兩者相比，似乎就遜色了很多，足協杯早早就被淘汰，聯賽排名第五無緣下賽季亞冠，看來奧拉羅尤真的要加油了，否則自己執教的球隊就會和其它兩支球隊的差距越拉越遠了。

當然，江蘇蘇寧男足想要在成績上能進一步突破，也需要蘇寧集團的大力支持，在資金上和球員的引入方面給予大力的支持，這樣的話才可能實現江蘇蘇寧三支球隊齊頭並進的局面，真正的創建出江蘇蘇寧的足球王國。
