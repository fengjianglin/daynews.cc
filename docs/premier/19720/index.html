鎖死司令塔，薩索洛一招致敵，米蘭需要尋找備選方案

2019-12-17 804條評論 4,533次閱讀 3259人點贊

文/花榮評球 個人原創文字作品，未經本人允可謝絕轉載，謝謝合作。部分圖片來源於網路，侵刪。

圖/米蘭官網

▲主場未能拿下薩索洛為球隊120周年慶典獻禮，賽後米蘭球員都顯得有些失落。

放在尋常背景下，主場拿不下薩索洛，對於現在這支米蘭來說算不上什麼不得了的事情。畢竟過往8次在聖西羅交手，米蘭雖然戰績佔優，但從來沒有贏得酣暢淋漓，5次勝利全是一球小勝，雙方實力對比，只在半斤八兩之間。但趕上俱樂部120周年慶典之戰，高朋滿座，勝友如雲，拿下薩索洛充當慶典煙花就成了必須要完成的任務。只可惜碰上對方門將「開掛」，自己又把握不住機會，聖西羅山呼海嘯聲中迎來一場悶平也就不足為奇了。

被掐斷的進攻火苗

圖/米蘭官網

圖/米蘭官網

▲為紀念球隊周年大慶，米蘭特地製作了紀念球衣。球衣上的文字和圖案改成了燙金字體，並在球衣正面隊徽下方添加了120的字樣。正面的贊助商標誌下還添加了「飛得更高」的祝福語。只可惜這件紀念球衣沒能給球隊帶來好運，米蘭只在薩索洛身上拿到1分，沒能以三連勝為球隊獻禮。

球隊慶典遭遇砸場子，薩索洛並不是第一個喪門星。皇馬百年大慶之際遭逢國王杯決賽，對手是此前只拿過一次國王杯的拉科魯尼亞，伯納烏都在等待雙喜臨門，球隊捧杯為慶典增色。結果拉科魯尼亞公然砸場，逆轉戰勝皇馬，毀了皇馬的喜慶氛圍。比之美林格們的遭遇，羅森內里們算是幸運多了。

圖/米蘭官網

▲球迷們在看台上高舉紀念圍巾為球隊助威。

或是球迷營造的這種大場面為球員們增添了壓力，米蘭球員腳下發緊，顯然背著包袱上陣，再未打出上一場對博洛尼亞時的高光表現。但整體而言，米蘭在上半時借著觀眾製造的巨大聲浪，依舊在進攻端給對手製造出了不小的壓力。特奧的進球被吹之前，米蘭已經製造出了數次破門機會，但佩格洛的連續撲救阻擋了南看台已到嘴邊的歡呼——米蘭早定大局的三板斧沒能打開局面，給了對手緩過神來的機會。

▲博納文圖拉在第14分鐘的遠射被撲，凱西的跟進補射再度被擋。本場比賽，米蘭在射門的選擇上多以低射為主，這給了身材並不大高達的佩格洛表現的機會。

如前所述，兩隊的實力本就在伯仲之間，薩索洛一旦回過神來，米蘭的進攻就開始受阻。最讓米蘭沒能料到的，是薩索洛直接拿出了殺招——緊盯博納文圖拉。德澤爾比這招極有針對性，一把掐住了米蘭的進攻命脈。上場對博洛尼亞，被放置在左中場位置的博納文圖拉成為了米蘭進攻的司令塔，馬尼亞內利和貝拉爾迪頻繁襲擾和跟隨博囧的結果，直接導致了米蘭在中場位置的出球不暢。

▲馬尼亞內利在比賽中緊緊跟隨博納文圖拉，進行貼身防守，於此同時，一旦博納文圖拉進入貝拉爾迪的防守區域，薩索洛邊鋒便會及時回位包夾，這導致了博納文圖拉出球上多次被斷。

▲博納文圖拉拿球之後，馬尼亞內利迅速從後場向前跟進，阻斷了博納文圖拉的傳球。

▲博納文圖拉前場拿球，正要組織進攻，右邊的貝拉爾迪迅速回防戰術犯規，阻止博納文圖拉的傳遞。本場比賽當中，類似的場面多次出現，薩索洛也憑藉這種戰術成功限制住了米蘭的進攻組織。

德澤爾比的這種設計，成功對米蘭的進攻起到了限制作用。米蘭對博洛尼亞的比賽中流暢運轉的進攻體系，在薩索洛的有意阻截下被有效切割，博納文圖拉不得不頻繁拉到邊路拿球，這無疑打亂了球隊的部署。對皮奧利而言，最大的難題在於：博納文圖拉是米蘭的進攻源泉，一旦他被限制，米蘭就很難通過他的組織來打通進攻通道。本場比賽，米蘭的79次長傳比之對博洛尼亞時的38腳長傳多了一倍有餘，博囧受限，中路進攻沒了樞紐，只能依靠長傳來組織調度是最大因由。

博納文圖拉受限，球隊的控制力度減弱，帶來的多米諾骨牌效應，就是本納賽爾的防守壓力增大。本場比賽本納賽爾的後腰位置多次受到攻擊，博納文圖拉的位置受到重點照顧，對手一旦斷球即時便能形成對後腰位置的威脅，這讓本來就不以防守見長的本納賽爾難免出錯。

▲本場比賽博納文圖拉被薩索洛重點盯防照顧，這也導致了米蘭的進攻多次被打斷，球隊的進攻效率和組織傳遞比之上一場比賽有了明顯的差距。

門將開掛≈射門選擇不佳

圖/薩索洛官網

▲頂替受傷的一號門將孔西利出場的38歲老門將佩格洛成為比賽的主導者。他在比賽中7次撲救的開掛表演，直接導致了米蘭只能無奈地帶走一分。值得一提的是，這是他本賽季首次意甲出場。

薩索洛的進攻限制本就已經讓米蘭進攻受阻，再加上本賽季首次出場的佩格洛開掛般的表演，聖西羅看台上只能一次次響起嘆息聲。但門將開掛，佩格洛狀態出色是一個因素，米蘭在射門上選擇不當則是另一原因。

米蘭本場射門23次，9次落在門框範圍以內，多是半高球和低平球，恰恰正中了身材不高，重心較低，下地速度較快的佩格洛下懷。全場米蘭最有威脅的射門，還是來自於替補上場的萊奧。萊奧本場兩腳射門，一次轉向后角上方，佩格洛毫無反應；一次要角度打向底角死角，佩格洛鞭長莫及。若是幸運女神稍稍眷顧，葡萄牙小將本該迎來自己在米蘭的第二粒進球。

▲萊奧在最後時刻的兩腳射門頗具威脅，險些為球隊打開了勝利之門。

米蘭本賽季至今打進16球，攻擊力意甲倒數第5，場均射門15.1次，射門轉化率不足6%，是為球隊進攻無力的根結。恰爾漢奧盧場均3.3次射門只換來1個頭槌進球，隨後場均奉上2.9次射門的皮揚泰克也只有1個運動戰進球入賬。球隊在射門時機把握、方向選擇方面都缺乏合理性，博納文圖拉復出之前，米蘭在遠射方面甚至沒有過破門得分，賽季至今未能通過角球手段破門，這都反應出了米蘭在終結能力上缺乏明星級的球員。米蘭苦追伊布不止，說白了最看重的還是瑞典人超強的門前終結能力。

本納賽爾在第41分鐘錯失的空門機會，最能說明米蘭這種能力上的匱乏。米蘭中場的射門選擇，時機不對，方式有誤，放在伊布、小羅，甚至帕託身上，本該是輕鬆的一擊，偏偏在本納賽爾的猶疑和調整之間化為泡影。面對對手送上的大禮還要吐餅，米蘭拿不下薩索洛，還能怪誰？

▲屢屢錯失進球讓球員們顯得十分焦灼，未能一鼓作氣拿到三連勝，米蘭接下來的征程依舊艱難。

這一夜的聖西羅未能響起「米蘭，米蘭」的歌聲，對於已經歷過120年風雨的紅黑軍團，無疑是一個遺憾。但慶典畢竟只是一時，球隊的未來才是長遠。對皮奧利和球員們來說，獻禮不在乎一場勝利，在薩索洛針對米蘭日臻成熟的A方案做出應對之時，如何在接下來的比賽中解決問題是重中之重。下一場比賽，狀態出色的特奧停賽，對手又是近來士氣高漲的亞特蘭大，皮奧利又要頭疼了。
