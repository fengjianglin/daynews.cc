尋找邊衛，羅馬有意波爾多後衛薩巴利

天天要聞
2019-12-19

據義大利媒體calciomercato的報道，羅馬將冬季引援的目光投向了法甲賽場：26歲的塞內加爾國腳、波爾多邊後衛優素福-薩巴利（Youssouf Sabaly）得到了紅狼軍團的關注，接下來的轉會窗，羅馬希望簽下這位左右通吃的邊後衛補充球隊的陣容厚度。本文正是對薩巴利的介紹。

薩巴利代表塞內加爾國家隊比賽（圖片：波爾多官網）

轉會傳聞

據義大利媒體calciomercato的報道，26歲的薩巴利得到了來自羅馬的關注。紅狼軍團希望在接下來的冬季轉會窗補充邊後衛位置，因此能左能右，相當全面的塞內加爾國腳也進入了他們的視野。本賽季薩巴利遭遇了膝蓋傷病，不過復出之後他依然是波爾多的邊路主力，目前他的身價大約在1200-1500萬歐元左右。

transfermarkt：薩巴利職業生涯的轉會記錄

1993年3月5日出生於法國的薩巴利，10歲就加入了巴黎聖日耳曼的青訓營，在2013年-2017年他先後被租借到了埃維昂，南特和波爾多，並都坐穩了球隊的主力位置。2017年夏天，薩巴利正式以400萬歐元的身價加盟波爾多，並效力至今。國家隊層面上，薩巴利是2013年法國U20世界盃的冠軍成員之一，不過他最終選擇代表塞內加爾成年隊出戰，並參加了去年的世界盃和今年夏天的非洲杯。

技術特點

身高174cm的薩巴利在場上能司職右後衛，左後衛和右翼衛多個位置，這位塞內加爾國腳非常全面，在攻防兩端都能起到自己的作用。不過由於傷病影響，近兩個賽季他的狀態有所下滑。

薩巴利本賽季的出場位置和發揮（Whoscored）

作為一名邊後衛，薩巴利的傳球線路廣泛，他的傳中能力非常出色，善於斜吊弧線球進禁區，同時他也並不局限於斜吊弧線球，低平球傳中和倒三角同樣也是他的特長。此外，薩巴利亦能在後場送出直塞球和大範圍轉移，可以說作為邊後衛，薩巴利的傳球能力是出色的。此外，薩巴利的無球插上的能力也相當突出，出現在邊路的他往往在無球時能利用自己的跑位尋找到空擋。

Whoscored：薩巴利的優點為傳球，傳中，盤帶和搶斷

薩巴利也有一定的過人能力，誠然，最近兩個賽季由於傷病的增多，薩巴利的突破沒有以前威猛（巔峰賽季的薩巴利非常喜歡利用爆發力突然加速閃開空間完成過人），不過薩巴利依然能憑藉不錯的小範圍擺脫能力完成在邊路的一對一過人。相對來說，薩巴利的連續變向過人能力不算非常出色，不過作為一名邊後衛，總體來說他的盤帶能力還是不錯的。

薩巴利職業生涯的盤帶數據（Whoscored）

在防守端，薩巴利有著不錯的預判和搶斷能力，站定位置之後，薩巴利往往能上搶完成斷球。此外，薩巴利的鏟球能力也相當突出，他的滑鏟往往能擊敗對方前鋒。相對來說，薩巴利在防守端的弱點是有時動作偏大，容易犯規。

薩巴利職業生涯的防守數據（Whoscored）

不過需要注意的是，近兩個賽季薩巴利的傷病增多，因此他的狀態也有一定幅度的下滑：傷病讓他的爆發力和移動能力不再處於巔峰，而他在盤帶和助攻上的表現也不如以前，因此在2018年世界盃之後，他的狀態一直不算非常穩定。當然總體來說，如果身體健康，左右通吃的薩巴利還是一名值得信賴的邊路大將。
