西超4大帝星神仙打架！皇馬2大王儲出世 淘汰賽幾乎鎖定死亡對決

2019-12-12 1,898條評論 3,341次閱讀 846人點贊

齊達內排出二線班底，皇馬依然客場勝出，終結歐戰對壘布魯日不勝的尷尬歷史。羅德里戈與維尼修斯相繼進球，西甲三強本輪歐冠均有超新星破門。莫德里奇近5次首發2球4助攻，與全隊同步狀態復甦，不過下輪歐冠淘汰賽皇馬有8成概率遭遇4大豪強，前景並不樂觀。

皇馬各項賽事3連勝，替補陣容立功

皇馬改寫不勝歷史，齊達內B陣立功

與巴薩一樣，皇馬不僅上輪就鎖定小組出線席位，就連座次也確定，只能是落後巴黎聖日耳曼而列小組次席，故而本輪歐冠主題就是練兵，齊達內大面積輪換，首發陣容只有瓦拉內卡塞米羅兩名絕對主力，羅德里戈與莫德里奇算是半主力，巴爾韋德卡瓦哈爾本澤馬與庫爾圖瓦替補席待命，拉莫斯克羅斯乾脆就沒來比利時。歐冠第50場出戰的卡塞米羅，連續5場踢滿90分鐘，更多也是齊祖周末客戰瓦倫西亞時基本會安排其輪休，避免再吃黃牌而停賽錯過世紀大戰，故而他能保持歐冠全勤。替補陣容出擊，皇馬上半時陷入苦戰，布魯日的高位壓迫與前場反搶屢次奏效，客隊則找不到節奏，伊斯科與魔笛進攻組織效果平平，主要威脅還是來自維尼修斯的個人突擊。

約維奇錯過單刀球機會

約維奇錯過單刀良機，是皇馬上半時唯一的破門良機，半場只有3次射門，為齊家軍本賽季各項賽事最低。好在白衣軍近期防守抗壓性比賽季初要好了不少，沒讓首回合丹尼斯連續兩次獨闖龍潭破門的尷尬時刻重現，下半時齊祖利用主隊體能下降逼搶力度削弱的良機，在邊路大做文章，防守中屢次被丹尼斯突擊身後的奧德里奧索拉，邊路套上助攻效率依然不錯，客隊兩次傳中接連製造前兩個進球，兩名巴西邊鋒與魔笛相繼破門，皇馬客場擒下1978年的歐冠亞軍隊，隊史上4次對壘布魯日頭一回勝出，終於改寫不勝歷史，也避免21世紀首次歐冠小組賽積分不足9分。

羅德里戈8射正入6球，齊祖找到貝爾替身

羅德里戈先拔頭籌

羅德里戈再次展示射術

羅德里戈與維尼修斯連續兩場首發，齊祖試驗新打法的意圖很明確，相比伊斯科首發時的菱形中場布陣，巴西雙子星兩翼齊飛的配置，雖然節奏控制上會薄弱些，可顯然攻擊性與直接殺傷力會更強。兩大新星也不負眾望，18歲菜鳥先拔頭籌，19歲的二年級生錦上添花。羅德里戈繼續展示大心臟殺手本色，打入一腳高難度抽射，本賽季4場歐冠3次首發出戰270分鐘連入4球，已追平本澤馬，並列球隊歐戰頭號射手，加上聯賽7次登場4次首發斬獲2球，巴西新星處子賽季已收穫6球，且各項賽事前8次射正目標就打入6球，外加2次助攻，直接製造8球，進攻效率高得驚人。如果貝爾傷愈復出後身體狀況無法調整到理想水準，羅德里戈將是世紀大戰右邊鋒位置的不二人選。

巴西雙子星攜手進球

維尼修斯隨後斬獲歐冠生涯處子球，也算是羅德里戈誤打誤撞的產物，他門前搶射被兩名主隊中衛梅赫勒與德利合力擋下，無巧不成書的是羅德里戈無意中腳後跟後磕變為妙傳，既讓自己收穫歐冠生涯第2次助攻，也成全同胞斬獲賽季第2球，賽後他興奮表態，「從未考慮過租借離隊」。維尼修斯特點鮮明，個人突擊能力極強，上賽季聯賽後半程與國王杯3場世紀大戰，就曾讓皮克為首的巴薩防線手忙腳亂，若是阿扎爾傷勢恢復不及無法亮相下周中的諾坎普一戰，維尼修斯就可能在左邊鋒位置首發.當然齊祖的另一變化是可能變陣，改打4312體系，讓伊斯科在前腰位置首發，那麼維尼修斯就可能得當替補，羅德里戈更適合打雙前鋒陣型。

西超新星一代PK，皇馬00後雙煞進球PK巴薩兩妖星

安蘇-法蒂打破歐冠最年輕進球者紀錄

既然是練兵，就要大膽給新秀機會，除剛傷愈不久的布拉欣-迪亞斯外，齊達內也基本派上了能用的新人，皇馬新援們基本通過考核，米利唐門迪狀態都比剛加盟時強出不少，只有約維奇依然找不準自我定位，儘管他也參與制造了維尼修斯的進球。維尼修斯盤帶突擊出類拔萃，羅德里戈運用各項技術的合理性則更勝一籌，全場他傳球48次，傳球成功率高達96.6%，很難想像這是鋒線球員交出的數據，畢竟前場球員傳球失誤率會相對較高。有趣的是，打入第3球的是34歲的莫德里奇，而兩名進球的新秀年齡總和才37歲，僅略高於魔笛的年齡。這既是兩大巴西新星首次同場進球，更是改寫球隊歷史，皇馬隊史上首次在單場歐冠有兩名不滿20歲的新星攜手破門。

西甲三強未來巨星們均在歐冠進球

本輪歐冠，皇馬巴薩均以副選陣容出戰，巴薩「二隊」客場擊敗全主力出擊的國際米蘭，最為驚艷，卡萊斯-佩雷斯與安蘇-法蒂兩位本賽季湧現的邊路新星相繼建功，均收穫歐冠處子球，法蒂還創下歐冠最年輕進球球員的歷史紀錄，西甲御林軍也不遑多讓，同樣打邊鋒的兩位巴西新星相繼破門。統計機構披露數據，歐陸足壇身價最高的00後新星，桑喬、維尼修斯與羅德里戈位列前三，弗洛倫蒂諾重金買來的兩張彩票顯然是刮出了獎，兩人評估身價總和已超昔日引進他倆時的9000萬歐元。另外本輪歐冠出線生死戰中，為馬競打入點球先拔頭籌的菲利克斯，也僅剛滿20歲，歐冠4戰2球的葡萄牙國腳，還是史上轉會身價最貴的U20球員，西甲三強圍繞未來的人才大戰，似已擴展到了20歲以下。

魔笛覺醒伴隨皇馬復甦，16強戰4大強敵會如何？

皇馬首輪歐冠淘汰賽很可能遭遇強敵

本賽季是歐冠史上頭一次，五大聯賽球隊包攬所有16強名額，西甲英超4強全出線，意甲也包攬三席，法甲兩隊，這也意味著歐冠淘汰賽將從首輪開始便空前慘烈。皇馬在1/8決賽潛在對手有5個，從統計學概率上說，遭遇英超雙雄曼城與利物浦概率最高，達21.88%，遇到尤文圖斯，即與C羅再次碰頭的概率也有20.88%，可能有18.08%的概率撞見拜仁，只有17.28%的幾率抽中實力最弱的萊比錫。換言之，齊達內的球隊有接近83%的概率遇到四大豪強之一，前途艱險。

莫德里奇遠射破門

幸而皇馬近期全隊狀態喜人，比之賽季初大有起色，各項賽事近10戰拿下8勝且保持不敗，主場戰平巴黎一役也是兩球領先後防線鬆懈被對手閃電般連追兩球。莫德里奇兩度重炮轟門斬獲皇馬生涯歐冠第5球，其中遠射破門4次，全場魔笛傳球91次，傳球成功率也達89.5%，送出2次關鍵傳球，成為中場操盤手。魔笛的狀態也是皇馬的晴雨表，賽季初他發揮奇差，齊祖無奈之下提拔巴爾韋德，讓兩人輪番首發，近期前金球先生表現顯著回升，近5場首發有4場參與制造進球，2球4助攻。皇馬未來一周將連續出戰瓦倫、巴薩與畢爾巴鄂三大強敵，迫切需要包括魔笛在內的隊內王牌拿出最佳狀態，否則這7天2客1主的魔鬼賽程，真會磨掉齊家軍一層皮。
