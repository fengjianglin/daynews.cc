卡利亞里已脫胎換骨狀態不俗 拉齊奧進攻火力兇猛力保歐冠名額

2019-12-16 1,273條評論 3,676次閱讀 2644人點贊

卡利亞里在意甲是一支中下游級球隊，成績也一直在聯賽10名以外，曾經在14/15賽季經歷了短暫的降級生涯，再次回到意甲賽場成績明顯有所提升，新賽季球隊表現優異，少賽一場的情況下以8勝5平2負暫居第5位，距離第4的羅馬也只有一個勝場，如果本場比賽能夠取勝且取得進球就可以重返前4。近期球隊士氣高昂，上輪在客場0-2落後的不利局面下，連扳2球，有驚無險的拿到1分，把自己近期戰績提升到15輪不敗，士氣高昂。卡利亞里在新賽季最大的改變還是再其進攻端，15輪31球是目前意甲第4進球大戶，特別是中場佩德羅和納因格蘭發揮出色，連勝為球隊攻入15球，前鋒線西蒙尼並沒有打出自己的狀態，仍處於調整期，這也是球隊未來隱藏要爆發的進攻點，球隊在其他位置球員也是多點開花，增加了對手的防守難度，卡利亞里本賽季再也不是一支弱隊，不過本場對陣拉齊奧，對球隊相對不利的是，主力門將奧爾森將繼續停賽，這對球隊的防守會帶來不小的壓力，而且拉齊奧進攻火力更猛，替補門將能否抵住他們的狂轟亂炸有待考察。

拉齊奧作為意甲的豪門球隊，近年成績也沒有太多進步，一直徘徊在歐聯杯戰區，上輪在歐聯杯的最後一輪，球隊也乾脆放棄了，以半主力的陣容應對，最後0-2落敗，這樣球隊也可以把全部精力轉移到意甲的賽場，雖然球隊歐戰不利，但是在聯賽的表現還是值得肯定，截至目前，球隊在意甲賽場已經聯賽12場不敗，更是贏足了8場，在聯賽中以10勝3平2負排名第3，距離第二的尤文也只有2個勝場，本輪如果可以順利拿下，距離榜首的國際也只有3分，以本賽季強大的火力，在加上穩定的防守，本賽季最低要求拿到一個歐冠名額問題不大。 本輪面對的對手卡利亞里是其手下敗將，在雙方連續10場的交手中，拉齊奧8勝2平完全碾壓對手，這也給球隊建立了巨大的心理優勢，無論在進攻還是防守上都要強於對手不少，客場拿到分數應該是在情理之中。
