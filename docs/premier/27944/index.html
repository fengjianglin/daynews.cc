巴薩決定清洗4位巨星，梅西接班人在內，換取2億歐元資金

2019-12-22 1,045條評論 2,375次閱讀 688人點贊

隨著1月轉會窗口的臨近，各傢俱樂部也積極行動了起來。根據《每日體育報》和法國《le10sport》消息，西甲豪門巴塞羅那方面已經準備好了在今年夏季對球隊陣容進行大清洗，不得不說，現在的巴塞羅那在各個位置上都有很多人才和有天賦的球員，但是這也讓巴薩的陣容十分擁擠，薪資結構不堪重負，再加上巴薩準備繼續投入資金引入強援，出售部分球員成為巴薩換取資金最好的方法。

目前有三位球員被巴薩列為1月轉會窗清洗對象，第1位是19歲的法國中後衛托迪博，這位年輕的球員在巴薩的出場機會渺茫，同位置中包括老大哥皮克，以及同為法國人的烏木蒂蒂，朗格萊，在這種情況下巴薩想將它出售，同樣對他感興趣的俱樂部有很多，這其中就包括勒沃庫森，埃弗頓，南安普敦等多傢俱樂部，其中紅魔曼聯方面願意為他支付高達2000萬歐元的轉會身價，巴薩非常滿意。

中場悍將比達爾也是巴薩的清洗對象，荷蘭人德容的加入讓巴薩的中場變得十分擁擠，球隊迫切需要清除部分老將，比達爾是最好的選擇，因為目前國際米蘭方面對他非常感興趣，雙方在歐冠最後一輪比賽時就進行商談過，兩支俱樂部幾乎已經達成了一致，只不過所提出的轉會方式不同，國際米蘭希望先租借，但是巴薩卻執意索要2000萬轉會資金，不太願意租借。

本次清洗名單中的重點對象是號稱梅西接班人的登貝萊，這位當年在多特蒙德震驚歐洲的法國天才在以1.05億歐元加盟巴薩之後，一度被認為是世界足壇未來的王者以及巴薩梅西的接班人，很多人都認為他能夠取代內馬爾，但事實證明法國人糟糕的職業態度讓他註定無法成為超級巨星，頻繁的傷病和散漫的職業態度已經讓他在巴薩失去了主力的地位，俱樂部對它的不滿是越來越多，現在巴薩已經決定將他出售，希望換取1億歐元的轉會資金，畢竟這位天才還很年輕，有潛力，巴薩認為他值一個億。

除了登貝萊這位巨星之外，目前被巴塞羅那租借至拜仁慕尼黑的庫蒂尼奧也在巴薩的清洗名單之中，在紅藍軍團的一個賽季，巴西人始終沒有找回在利物浦的狀態，他也被打上了巴薩歷史第一水貨的標籤，在今年夏季租借至拜仁之後，近期在南大王的表現是越來越好，有消息稱，只要拜仁慕尼黑方面願意拿出超過9000萬歐元的報價，巴塞羅那就願意將他轉會。

巴薩希望能夠在明年夏季轉會期結束之前將球隊多於球星進行出售，以籠絡超過2億歐元的轉會資金，這一方面是為了緩解巴薩的資金壓力，完善薪資結構，另一方面是希望能夠在明年內馬爾的轉會中擁有足夠的資金，要知道巴黎方面對於內馬爾的報價高達1.8億歐元，巴薩要想簽下他，至少需要準備2億歐元的資金，出售球員對於巴薩來說勢在必行，西甲豪門需要更多的轉會資金引入巨星，衝擊失落4年的歐冠冠軍。
