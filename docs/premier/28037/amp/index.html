遍地開花！日本足壇13虎將征戰五大聯賽，歐洲留洋深造共計50人

天天要聞
2019-12-20

隨著日本當紅國腳南野拓實加盟英超領頭羊利物浦，日本球員五大聯賽人口數增加到13人，他們的足跡遍布英超、西甲、德甲、意甲、法甲，而他們身後還有源源不斷的日本球員在荷甲等歐洲聯賽效力的球員。據粗略統計，目前共有50名日本球員在歐洲留洋。這大概就是中日足球差距的現狀。

利物浦用如此低廉的價格拿下南野拓實，讓曼聯都有些眼紅。據Goal的消息，曼聯此前也曾關注過南野拓實，並把他列為了1月的轉會目標之一。報道稱，曼聯並不知道南野拓實的解約金條款只要725萬鎊，他們認為南野拓實的價格要更為高昂。報道同時聲稱，曼聯內部人士認為，像這麼划算的買賣，曼聯都沒有進行參與，CEO伍德沃德需要對這樣的失敗負責。

截至目前，日本球員有4位在英超效力，他們分別是南野拓實（利物浦）、武藤嘉紀（紐卡斯爾）、吉田麻也（南安普頓）；西甲2位：久保健英（馬略卡）、乾貴士（埃瓦爾）；意甲1位：富安健洋（博洛尼亞）；德甲3位：鐮田大地（法蘭克福）、長谷部誠（法蘭克福）、大迫勇也（不萊梅）；法甲3位：酒井宏樹（馬賽）、昌子源（圖盧茲）、川島永嗣（斯特拉斯堡）

第二梯隊在集中在人才輩出的葡萄牙和荷蘭聯賽。其中葡超6位：安西幸輝、權田修一、龜倉龍希、菅島弘希、（波爾蒂芒人）、中島翔哉（波爾圖）、前田大然（馬德拉航海）；荷甲7位：堂安律（埃因霍溫）、板倉晃（格羅寧根）、菅原由勢（阿爾克馬爾）、中村敬斗（特溫特）、中山雄太（茲沃勒）、范維爾梅斯克肯際（茲沃勒）、本田圭佑（維特斯）

比利時甲級聯賽也是日本球員登陸歐洲的重鎮，目前大概有10位日本球員：伊東純也（亨克）、伊藤達哉（聖圖爾登）、植田直通（色格拉布魯日）、小林祐希（華斯蘭德貝弗倫）、久保裕也（根特）、鈴木優磨（聖圖爾登）、丹尼爾-施密特（聖圖爾登）、三好康兒（安特衛普）、森岡亮太（沙勒魯瓦）、豐川雄太（歐本）

其他則奧超2位：奧川雅也（薩爾茨堡紅牛）、北川航也（維也納快速）；俄超：西村拓真（莫斯科中央陸軍）；塞超：淺野拓磨（貝爾格萊德游擊）；蘇超：食野亮太郎（哈茨）；土超：長友佑都：（加拉塔薩雷）

非頂級聯賽也不缺少日本球員，其中德乙：遠藤航（斯圖加特）、原口原氣（漢諾威96）；西乙：山口瑠伊（埃斯特雷馬杜拉）、岡崎慎司（韋斯卡）、柴崎岳（拉科魯尼亞）、香川真司（薩拉戈薩）；西乙b：安部裕葵（巴塞羅那B）。
