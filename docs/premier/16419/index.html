19歲妖星狀態爆棚！加圖索被無情打臉，首戰暴露出更衣室最大問題

2019-12-15 330條評論 1,267次閱讀 283人點贊

今天凌晨意甲第17輪展開一場比賽爭奪，剛剛經歷換帥的那不勒斯主場迎戰帕爾馬，最終憑藉19歲妖星庫盧塞夫斯基的傳射建功以及熱爾維尼奧傷停補時階段的絕殺，帕爾馬客場2比1力克那不勒斯！那不勒斯主帥加圖索遭遇上任之後的開門黑，輸掉比賽後加圖索黑臉離開球場。本場比賽帕爾馬新星庫盧塞夫斯基狀態爆棚，瑞典小將沒有給那不勒斯主帥加圖索留絲毫情面，開場不久庫盧塞夫斯基便生吃庫利巴利為客隊首開紀錄！

在比賽臨近結束，又是庫魯斯夫斯基助攻熱爾維尼奧讀秒絕殺那不勒斯，最終帕爾馬客場拿下3分！庫盧塞夫斯基是本賽季意甲賽場上的當紅炸子雞，這名年輕球員已經受到了多家豪門俱樂部的關注，本場比賽新帥加圖索被小將無情打臉，那不勒斯首秀便遭遇失利！本場比賽過後，加圖索的球隊暴露出一大問題，隊員在場上的心態出現了問題。本賽季在意甲賽場上那不勒斯表現低迷，前任主帥安切洛蒂在本周中歐冠小組賽最後一輪比賽結束後下課，前AC米蘭主帥加圖索臨危受命，但義大利人臨時接手那不勒斯卻並不被眾人看好！

安切洛蒂雖然遭俱樂部解僱，主要原因是因為其與俱樂部主席德勞倫蒂斯之間的關係存在一定的問題，而安切洛蒂與那不勒斯球員之間的關係還是非常不錯，在得知安切洛蒂下課後，整個那不勒斯更衣室陷入一片沉寂，由此可見安切洛蒂還是深得球員擁戴！如今加圖索接手那不勒斯首先要面臨的就是如何能夠得到那不勒斯球員的信任，雖然在球員時代加圖索是一名悍將，但在他的執教生涯中並沒有亮麗的表現。

那不勒斯球員的能力以及球隊的整體實力在意甲賽場上還是名列前茅的，但本賽季他們彷彿失去了鬥志，聯賽至今僅排在第8位，隊內多名球員萌生去意這讓意甲勁旅雪上加霜，當然主要原因在於俱樂部主席德勞倫蒂斯前段時間製造的風波。如今加圖索要做的首先就是將球隊的氣氛調動起來，讓曾經那支鬥志昂揚永不服輸的精神重新出現在更衣室中，雖然本輪對陣帕爾馬的賽後，加圖索主動將球隊失利的責任攬在了自己身上，但球隊的根本問題仍沒有得到解決！

​賽季中途接手球隊對於任何一名教練來說都是巨大的考驗，加圖索在經歷了AC米蘭執教的經歷後，希望能夠真正的幫助那不勒斯從低谷中走出。面對一手自己心裡都沒有底的牌，加圖索遭遇了執教生涯的又一次考驗，他到底能否將那不勒斯重新帶回正軌，你怎麼看呢？歡迎在評論區。
