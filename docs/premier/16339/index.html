當今足壇三國殺，哪支球隊最像蜀國？

2019-12-16 3,405條評論 4,525次閱讀 932人點贊

2019年最成功，也是最春風得意的俱樂部是誰？答案應該是比較統一的！雖然是4：0逆轉有很大運氣成分，但是破釜沉舟的氣勢，的確有強隊風範！利物浦英超歐冠獲得次數最多的球隊，還多次進入歐冠決賽，上演過多次歐冠奇蹟。就像我們最津津樂道的東漢三國，魏蜀吳，在2019/2020賽季，誰是蜀國，誰是吳國，誰是魏國？

在當今足壇，最厲害的球隊應該是皇馬，巴薩，利物浦！他們也是獲得歐冠次數最多的幾支球隊，最近20年就差AC米蘭，7冠！拜仁5冠，阿賈4冠！今年利物浦卧薪嘗膽，從2005年到現在才拿到一次歐冠冠軍，真是等到花兒都謝了。從庫鳥離開利物浦，利物浦像受到巴薩極大的刺激一樣，奮發圖強；蘇牙被你搶去，庫鳥也搶，非要在歐冠給你難看！於是買買買，這次大手筆買後衛，范迪克和阿利松，當時最貴的中後衛和守門員完成利物浦的球員拼圖。不容易啊，14年的努力，才在克洛普的帶隊下拿到歐冠冠軍，像極了三國中的劉備建立蜀國的堅韌不拔，就像劉備和曹操進行的漢中大戰，通過勝利，奠定蜀國的最強盛時期。現在的利物浦肯定是最近14年最強盛時期，打敗了最強大的對手巴薩，拿到最後的歐冠，這就是實力！

利物浦最像蜀國，巴薩像誰？吳國！巴薩是從克魯伊夫帶隊1992年才拿到第一個歐冠，2006年小羅幫忙才拿第二個歐冠，然後2009.2011.2015在梅西帶領下拿三次歐冠，感覺梅西就像周瑜，為吳國的強盛立下汗馬功勞。依靠著拉瑪西亞青訓的一代黃金球員建立霸業，哈白布皮克梅西阿爾巴羅布特等拉瑪西亞團隊，就像長江天險，守護者俱樂部。

皇馬就像魏國，實力最強，建國最早，皇馬也是拿歐冠最早，次數也最多，13次，比巴薩加上利物浦還多2次！魏國一直在三國中實力最強，地大物博，人口最多；皇馬也是綜合實力一直很穩定在最前列，是當今足壇歷史最悠久的豪門俱樂部，看他們的替補席一直是大腕雲集，比一般俱樂部主力還厲害！主教練用的世界名帥也最多，真是從主教練和球員都是群星燦爛，就像魏國手下謀士上百，戰將千員。大家說AC米蘭比較像三國里的哪一個王，劉表怎樣？本來是完全可以三國鼎立，一方諸侯；但是由於老闆貝盧斯科尼總理下台，沒有財力，實力一落千丈。現在的米蘭多少年沒有歐冠資格，這麼大的江湖幾乎沒有他的傳說。讓人感嘆不已。

阿賈克斯像三國的誰？像不像呂布，高光的時候把以後老大曹操打的丟盔卸甲，徐州之戰就是呂布經典之戰。一開始呂布相當厲害，就像阿賈克斯，年輕風暴相當厲害！克魯伊維特，利特馬寧，德容，德利赫特青春風暴，讓我們感受到足球的魅力！大家覺得像不像？歡迎評論。
