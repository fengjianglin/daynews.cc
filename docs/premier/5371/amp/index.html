C羅在尤文變雞肋了嗎？那個歐冠王者何時才能回歸？

天天要聞
2019-11-27

迪巴拉又一次成為了尤文圖斯的救世主，在尤文主場和馬競的這場強強對話中，憑藉迪巴拉在半場行將結束前一記角度極小的直接自由球破門，尤文圖斯拿下3分的同時也提前一輪鎖定了小組第一。

然而，這場比賽下半時的一次換人調整再次成為賽後大家討論的焦點。當比賽進行到下半場第76分鐘，此時場上的 比分依舊是1-0，尤文領先，強強對話，比分可能在瞬間就被改變。在大局未定的情況下，主教練的換人調整往往會現在拿下在比賽中表現不佳的球員，而把狀態好的球員留在場上。但是，薩里卻選擇用伊瓜因換下了迪巴拉。

伊瓜因在上周末的聯賽中完成了梅開二度，而且又是薩里的愛將，理應獲得出場機會。而從比賽的表現來看，如果唯狀態論，那薩里更應該選擇用伊瓜因換下C羅，而不是在本場比賽取得進球的迪巴拉。客觀來說，在這場防守為先的比賽中，C羅並沒有太多亮眼的發揮。在下半場馬競大軍壓境的情況下，薩里的球隊需要進攻線上的球員回來救急，而以C羅目前的身體狀態和體能儲備，一旦在防守中消耗過大，在進攻端帶來的影響力只能大打折扣。

但是，薩里不敢提前換下C羅。上周的聯賽，薩里就和C羅提前溝通好了，好好輪休一場，全力備戰周中歐冠與馬競的比賽。再加上國際比賽日前尤文圖斯的兩場比賽，C羅因為連續被提前換下，已經感到非常憤怒了。好不容易，將帥不和的風波剛剛冷卻，薩里也不敢造次了。過去這段時間，估計薩里自己也想明白了，C羅這樣的球星，自己得罪不起，否則一旦翻臉，捲鋪蓋走人的可能就是自己了。

再說了，尤文圖斯在此前四輪小組賽中已經拿到了10個積分，沒有任何出線壓力，在如此容錯率下，薩里也可以任性一把，繼續容忍把C羅留在場上。

不過，薩里的這個選擇同樣招來了一片罵聲，包容C羅，這就意味著犧牲了迪巴拉。這場比賽，迪巴拉上半場打進了一粒精彩的自由球世界波，提前下場正好可以享受主場球迷的歡呼聲。可是，這給後面的比賽留下了隱患。接下來的比賽，如果薩里需要再做調整，如何在C羅和迪巴拉之間進行取捨，這是一道擺在薩裡面前的難題。

C羅的態度很明顯，不喜歡自己被提前換下場，除非他主動示意堅持不了。而迪巴拉這賽季的狀態要更穩定，已經連續三場比賽取得進球，其中兩場比賽更是打進了 唯一的進球。在薩里目前的體系中，C羅和迪巴拉誰的作用更明顯，答案其實也很明了。

C羅的球迷可能又會說了，看看在葡萄牙國家隊的C羅，進球效率依舊很高，剛剛才上演帽子戲法，而回到俱樂部就表現低迷，這是你薩里的問題，不是C羅的問題。

事實真的是這樣嗎？C羅的確在葡萄牙國家隊保持了超高的進球效率，但是，我們要理智的看待問題，葡萄牙隊在歐洲杯預選賽上遇到的立陶宛和盧森堡這些隊伍，他們是實打實的魚腩球隊，實力能和歐冠級別的隊伍相提並論嗎？另一方面 ，葡萄牙的戰術體系都是圍繞C羅服務的，場上11個葡萄牙球員，大家都是以C羅為中心運轉。幫助減輕防守壓力，進攻時享受無限開火權，在保姆式服務下，C羅自然得吃得喝。而尤文圖斯更講究整體，不依賴某一位球星的發揮，薩里也是一位強調集體的主教練，之前在那不勒斯一舉成名的前場進攻「死亡三小」，同樣也是不分主次的。C羅在尤文享受不到保姆式的服務，場上效率自然大打折扣。

C羅、薩里、尤文圖斯，目前三者並沒有處於健康發展模式中，C羅需要調整自己的比賽方式，而薩里也需要圍繞C羅制定一些戰術，俱樂部則做好中間的協調關係。一個團隊在一起合作，需要相互妥協，相互犧牲，否則，隱患可能會越來越大，最後直接影響到更衣室的和諧。

總之，C羅現在在尤文的處境非常微妙，他看上去距離人們期待的那個關鍵先生還有不小一段距離。
