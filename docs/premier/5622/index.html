傳聞：為得到姆巴佩，佛羅倫蒂諾計劃說服C羅轉會巴黎聖日耳曼

2019-11-20 0條評論 13次閱讀 0人點贊

皇家馬德里主席弗洛倫蒂諾計劃從巴黎聖日耳曼簽下姆巴佩。

姆巴佩已經成為了弗洛倫蒂諾明年夏天轉會窗口的頭號目標，他可能會利用前皇馬球星C羅來吸引這位世界盃冠軍來到伯納烏。

2018年7月，c羅以1億英鎊的轉會身價從皇家馬德里轉會到意甲尤文圖斯俱樂部，簽署了一份為期4年的合同。

然而，據Eldesmarque報道，弗洛倫蒂諾正在和C羅的經紀人豪爾赫·門德斯密謀讓這位葡萄牙巨星轉會去巴黎聖日耳曼。

據報道，C羅一直希望獲得與梅西相等的薪酬，而法國豪門大巴黎完全可以承擔如此高的薪酬。

如果C羅可以被說服轉會到巴黎聖日耳曼，那麼弗洛倫蒂諾相信皇馬可以得到姆巴佩。

尤文圖斯體育主管法比奧·帕拉蒂奇堅稱C羅不會在明年夏天離開俱樂部。

「他非常專註於他目前的目標，我一點也不相信他會在賽季結束或者更晚的時候離開，」帕拉蒂奇上個月說。

「他有合同在身，我們和他在一起很開心，他在尤文圖斯也很開心。」

與此同時，c羅似乎支持薩里在之前1-0戰勝AC米蘭的比賽中換他下場的決定。

「在過去的三周里，我受到了限制，」C羅承認。

「每個人都知道我不喜歡被換下，但我在換下後的動作沒有任何問題。我試著通過帶傷上陣來幫助尤文圖斯。」

「沒有人喜歡被替換，但我能理解，因為我在那兩場比賽中表現不佳。我不是百分之百狀態。」
