瓦爾韋德：神童法蒂是一個天才得分手，巴薩想獲得歐冠冠軍

2019-12-11 690條評論 4,099次閱讀 1920人點贊

巴塞羅那主教練埃內斯托·瓦爾韋德稱安蘇·法蒂是一個「天才的得分手」，他在歐冠對陣國米的比賽中創造了歷史。這位加泰羅尼亞的年輕球星在周二晚上光芒四射，17歲的法蒂成為了世界盃歷史上最年輕的射手，在周二聖西羅的比賽中，他攻入了最後一粒制勝球。

這名前鋒本賽季有三個進球，瓦爾韋德讚揚了這位西班牙青年國腳，他的進球是在F組比賽中盧卡庫的進球蓋過了佩雷斯的第一粒進球之後打進的。

「他們(法蒂和佩雷斯)是今年進入一隊的兩名球員。年輕球員，對安蘇來說非常年輕。比賽很複雜，特別是卡萊爾·佩雷斯，因為他們有一個經驗豐富的後衛，他首發了，但他有一個目標，因為他打得很好。安蘇經歷了一些身體上的困難和不適，但是他是一個天生的射手。」

巴塞羅那已經獲得了16強的席位，他們在國際米蘭被淘汰的情況下以小組第一的身份晉級。瓦爾韋德談到了冠軍聯賽對加泰羅尼亞人的重要性，他們上一次舉起冠軍獎盃是在2015年。

「我們對俱樂部和這場比賽負有很大的責任，」他說。「我們已經證明了你必須贏得所有的比賽，今天的比賽也不例外。」

在本賽季早些時候受到批評後，巴塞羅那成為西甲冠軍，並在所有比賽中取得了六場勝利。但是巴爾維德希望從他的球隊看到更多，他們將在周六客場挑戰皇家社會。

「我們都很好，但我喜歡以更連續的方式來看待每件事。一開始我們有困難，」他說。「這個小組階段對我們來說很困難。我們戰勝了國際米蘭，在布拉格斯拉維亞取得了勝利。但我們在這裡，我們面臨著巨大的挑戰，一周有三場比賽，我們希望能有一個好的結局。」

在經歷了一個艱難的賽季開始後，巴塞羅那在巴爾韋德的帶領下恢復了元氣。
