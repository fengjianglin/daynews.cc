別等頒獎了，2019金球獎得主已經顯而易見

天天要聞
3 週 ago

2019年金球獎的投票環節早已結束，但是具體的結果需要等到12月2日的頒獎典禮才會知道2019年的金球獎得主；不過在金球獎頒獎典禮之前，各大媒體也在不斷預測今年金球獎的得主，但是目前從各方面的情況下來看，梅西將會大概率榮獲個人第六座金球獎。

首先看一下三位競爭球員：梅西、C羅、范戴克

范戴克首先作為後衛球員就在評選當中並不佔據優勢，他沒有梅羅二人可以沾沾自喜的進球數據；范戴克在去年雖然跟隨利物浦奪得了歐冠冠軍，但是依照金球獎常年的評選來看，歐冠冠軍依舊不足以讓范戴克擁有過多的競爭優勢。

2014年諾伊爾甚至帶領德國戰車捧獲了世界盃冠軍，但是因為是其門將的原因依舊只能陪跑。梅羅二人在其影響力就已經能夠左右大部分投票，畢竟所有擁有投票權的人不可能觀看得了三個人所有的年度比賽用以對比三人的發揮，很大一部分或許取決於媒體造勢或者就是第一印象。

C羅去年的表現並不如同往年一樣突出，上個賽季是葡萄牙人轉會去到尤文圖斯的第一個賽季；C羅在尤文圖斯跟隨球隊完成了意甲八連冠並收穫了義大利超級盃，國家隊層面奪得了首屆歐國聯冠軍。C羅上個賽季在俱樂部打入28粒進球，國家隊攻入3粒；對比往年的進球數據，這一成績單並不突出葡萄牙人的實力。

況且2018年金球獎評選時，C羅的數據對比今年更是好出了不少；但是最終捧杯的莫德里奇，而C羅卻連頒獎典禮都還沒去，甚至在年度最佳陣容還沒有被主持人提及名字；與歐足聯的緊張關係也不利於C羅的獲票，不得不說尤文圖斯的影響力與關注度也遠遠不及皇家馬德里。

梅西在上個賽季雖然沒能奪得什麼重量級的冠軍頭銜，但好在至少梅西還有一座西甲冠軍獎盃；不過要說個人數據的話，梅西在上個賽季的數據是相當的驚人；為俱樂部各項賽事出場五十場的比賽中奉獻了51球19助攻，國家隊層面也有3球1助攻的表現。

在本屆金球獎的評選當中，有人說不給梅西就是恥辱，有人說不給梅西就是黑幕，有人說金球獎就是梅西獎。多數球員教練都非常認可梅西就是世界最佳。

昨日《世界體育報》記者Francesc Aguilar在社交媒體上表示今年的金球獎得主本人已經知曉了，《法國足球》已經告訴了獲獎者；雖然這名記者並沒有透露具體的獲獎人，但是他卻在社交媒體上表示：「姆巴佩認為梅西是今年金球獎的最大熱門，這是有原因的，不是嗎？」

不僅如此，最新一期的《世界體育報》封面是《法國足球》已經派遣團隊前往巴塞羅那拍攝紀錄片；而三大候選人只有梅西在巴塞羅那，這不禁讓人浮想聯翩。《世界體育報》表示梅西獲得2019年金球獎以及是有先兆了。

不僅如此，雖然頒獎典禮在12月2日，但是自11月初的時候，就有消息稱《德轉》官網早已經「透露」了梅西的第六座金球獎；在11月初開始，在《德轉》官網的梅西個人介紹頁面，上面顯示著梅西榮獲了2019年的金球獎。

從目前各方媒體的報道來看，梅西捧起第六座金球獎的概率可謂是基本實錘；包括梅西上個賽季的出色數據，雖然沒有歐冠、美洲杯等重大錦標，但是由於個人數據的出色應該對阿根廷人也造成不了太大的影響。

只要各方媒體的爆料屬實，2019年的金球獎得主已經是顯而易見。
