又1豪門轟然倒塌！被英超墊底隊4分鐘轟2球，曼聯今夜又爆冷輸球

2019-12-23 1,763條評論 3,606次閱讀 1156人點贊

北京時間12月22日22:00，英超第18輪，德赫亞黃油手，萬比薩卡送點球，曼聯客場0-2沃特福德，遭遇2輪不勝。

這是一場不小的冷門，在賽前英國機構預測沃特福德主場取勝的概率只有24%，而曼聯客場可能性高達 51%！

此役前，沃特福德在17輪過後只拿到9分排名英超倒數第1，他們只取得過一場勝利，那就是客場2-0諾維奇。他們本賽季8次主場作戰，4平4負未嘗一勝，這一場2-0，是他們本賽季主場首勝，現場2萬人狂歡。

在雙方歷史交鋒來看，曼聯最近5次英超面對沃特福德是全部取得勝利，這是他們27年英超11次交鋒第2次輸給沃特福德（9勝2負），上一次輸給沃特福德要追溯到2016年9月18日，當時客場1-3輸球。

上半場34分鐘，馬夏爾的直塞球，林加德單刀挑射高出橫樑，浪費了一次黃金機會。他在64分鐘被索爾斯克亞換下，換上傷愈復出的博格巴。

根據Opta的數據統計，林加德在英超單場0球0助攻已經超過25小時，上一次直接參与進球還是2018年12月16日對陣哈德斯菲爾德，當時他助攻博格巴打進1球。

第50分鐘，21歲沃特福德前鋒薩爾一腳射門其實沒有踢正部位，不過德赫亞卻出現了低級失誤，黃油手丟球後，讓曼聯客場0-1落後。

根據Opta的數據統計，從上賽季到現在，德赫亞已經出現過6次失誤丟球，這1尷尬數據排名英超門將第一位。

在第52分鐘，薩爾製造萬比薩卡鏟球犯規，裁判判罰點球；

第54分鐘，隊長迪尼主罰點球命中，沃特福德2-0取勝！

21歲的薩爾，今夏以3000萬歐元身價從雷恩轉會至沃特福德，成為隊史最貴球員。

不過，此前他在17輪英超只打進1球，而今晚他在4分鐘內獨造2球，在第87分鐘被替換下場時，全場2萬人起立為他鼓掌。

在AC米蘭客場0-5亞特蘭大之後，又一支歐洲豪門在本周末轟然倒塌。
