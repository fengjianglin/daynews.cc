齊祖回懟巴黎：姆巴佩就是想加盟皇馬！西媒：曼聯、尤文同搶妖鋒

天天要聞
1 個月 ago

阿斯報消息，尤文和曼聯加入了皇馬對薩爾茨堡紅牛新星小哈蘭德的爭奪中。

小哈蘭德在自己參加的前4場歐冠比賽中已經斬獲7球，這名19歲的前鋒正像C羅和梅西那樣打破歐冠的紀錄，事實上本賽季才進行到11月份，小哈蘭德就已經在17場各項比賽中攻入了23球，只是球員在歐冠中的爆發顯然更有說服力，也更能吸引歐洲大俱樂部的關注。

薩爾茨堡紅牛在1月份以區區500萬歐元的價格就從莫爾德買來了小哈蘭德，而這有可能成為歐洲冬窗史上最划算的交易之一，目前皇馬已經對這名挪威神童展示出了興趣，不過他們還要面臨尤文以及曼聯等豪門的競爭。

根據義大利媒體的報道，尤文在薩爾茨堡紅牛客戰那不勒斯的比賽中派出了他們的球探部門主管Matteo Tognozzi考察了小哈蘭德，曼聯更是不必多說，紅魔在放走了盧卡庫和桑切斯後鋒線正缺人手，而索爾斯克亞甚至是小哈蘭德在莫爾德時的啟蒙導師，此外，巴薩和阿森納同樣被認為正目不轉睛的緊盯這名當紅新星。

阿斯報指出，儘管薩爾茨堡紅牛並不歡迎任何球隊在一月份報價小哈蘭德，但他們很清楚在如今的轉會市場上可以為這名前途無量的射手貼上高價，索爾斯克亞看起來已經準備出手，尤文也正著眼於用這名挪威人來替代年老的C羅，皇馬則要面臨一番激烈的競爭。

小哈蘭德可以算是本賽季躥紅第一人了，如果說他在奧地利聯賽的進球數還有水分的話，歐冠4場7球的戰績就很有說服力了，明年幾乎可以肯定很多豪門會搶購他的。

不過皇馬不一定會買哈蘭德，從各方面消息來看，姆巴佩還是皇馬的第一目標，所以皇馬可能會集中精力追求姆巴佩。

今天關於姆巴佩的傳聞也很多，我們來看看。

巴黎體育總監萊昂納多最近在接受採訪時回擊了齊達內關於姆巴佩的言論，不過皇馬主帥似乎不打算退讓，並且齊祖最近再次重申，巴黎前鋒就是想加盟皇馬。

齊達內在最近談到姆巴佩時表示，這名前鋒「夢想加盟皇馬」，而這個表態顯然讓巴黎方面十分不滿，萊昂納多最近在被問到此事時就表示，他認為齊達內的言論有點「煩人」，並且督促齊祖不要讓姆巴佩陷入「不安定的狀態」。

然而，皇馬主帥似乎不打算在這個問題上做出讓步，並且在對埃瓦爾的賽前新聞發布會上又強調了一遍他的立場。

齊達內說：「我說的那些話只不過是姆巴佩自己已經說過的話，那就是他夢想有一天穿上皇馬的白色戰袍。

這就是我說過的話，而且今天我要再說一遍，我也願意每天都重複一遍（I’m saying it again today and I would repeat it again everyday）。

我對萊昂納多沒有什麼要說的，但我可以說我自己想說的話。」

齊祖這番表態可以說是很強硬了，那就是我想說就說，而且我說的是姆巴佩自己的原話，有什麼不對？

當然，也只有齊祖和皇馬真的有興趣購買姆巴佩的前提下，才有這麼硬扛的理由，看來這個問題上皇馬是立場堅定的。

隊報消息，姆巴佩夢想為皇馬效力的事實已經讓巴黎陷入了恐慌模式，法甲豪門準備「不惜一切代價」續約這名前鋒。

隊報表示，巴黎已經和姆巴佩的團隊就續約問題展開了溝通，但目前他們還未向球員提供新合同，不過和幾天前法國足球的消息一致，隊報也表示巴黎非常擔心姆巴佩和皇馬互有情意的問題。

齊達內在最近一周里兩次公開表示姆巴佩「夢想為皇馬效力」，也無怪乎巴黎對此感到擔憂和不滿，不過這名前鋒的合同依然要到2022年才到期，因此皇馬的策略就是寄希望於球員保持現在的立場，不與巴黎續約，因為一旦一名球員的合同進入最後24個月甚至12個月，那麼賣方就非常有可能會失去留人的主動權。

鑒於此，法國方面消息表示，巴黎希望能「不惜一切代價」來完成對姆巴佩的續約，顯然由於皇馬的虎視眈眈，法甲豪門的恐慌指數正不斷攀升。

隊報的意思是，巴黎願意不惜一切代價續約姆巴佩，但現在還沒開出合同，為什麼不開出合同，下面這個文章里有提及。

隊報消息，巴黎希望能和姆巴佩續約，但他們也清楚事情不會這麼簡單。

姆巴佩當下的合同2022年到期，而在最近齊達內的一系列發言後，巴黎前鋒未來很可能會是皇馬的引援目標，對此隊報表示，為姆巴佩提供新合同暫時還不是巴黎工作的重心，但他們知道想要續約這名法國球星會相當困難。

法媒指出，巴黎現在還沒有為姆巴佩提供新合同，因為他們無法為球員提供和內馬爾相同的3000萬歐元的稅後年薪，再者，姆巴佩的合同依然還有兩年半才到期。

不過問題在於，假如本賽季巴黎依然無法在歐冠中取得突破，那麼姆巴佩就很有可能會開始質疑自己的未來，並考慮加盟皇馬。

也就是說，不是巴黎不想續約姆巴佩，一個是他的合同還有兩年半，不用太著急，一個是有內馬爾在，巴黎也很難再開出稅後3000萬年薪的合同了，要同時養這兩個超級球星代價太大了……

皇馬的策略依然是等，最好是姆巴佩不續約，然後到明年夏天甚至後年夏天，他們就好運作了，就像買阿扎爾一樣……

馬卡報消息，巴薩最近一系列欠缺說服力的表現讓巴爾韋德陷入了千夫所指的境地，不過紅藍軍團主帥感覺自己依然擁有球隊高層的鼎力支持。

在最近的新聞發布會上，巴爾韋德表示：「未來問題不會困擾我，最近我和巴托梅烏曾一起用餐，俱樂部一直都是支持我的，我也感受到了那種尊重。

我們知道有些該贏的比賽球隊沒有拿下，而這有時候會造成軍心不穩，但我們需要堅定的重整旗鼓，並保持相應的強度。」

怎麼說不是關鍵的，關鍵是成績和場面真正有好轉才行。

高層在炒人之前，總是「支持的」……

足球市場網消息，尤文仍未放棄追求羅馬新星尼科洛-薩尼奧洛，並且計劃斥資4000萬歐元求購這名中場。

報道稱，尤文在夏天時就曾想收購薩尼奧洛，不過羅馬當時沒有任何放人的計劃，儘管如此，看起來這並沒有打消斑馬軍團購人的野心。

據悉，尤文準備再次和羅馬展開談判，並且他們願意為薩尼奧洛提供4000萬歐元的報價。
