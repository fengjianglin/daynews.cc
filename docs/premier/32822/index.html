太不要臉了！0-5後米蘭中場點贊對手 眼熱老東家風生水起

2019-12-24 2,957條評論 4,660次閱讀 1365人點贊

0-5不敵亞特蘭大，意甲昔日豪門AC米蘭輸得連底褲也沒了。而在這場載入AC米蘭史冊的恥辱性失利之後，紅黑軍團中場核心凱西還做出了令人髮指的舉動——點贊亞特蘭大隊長慶祝勝利的動態。亞特蘭大隊長戈麥斯在比賽結束之後更新了Instagram動態，他上傳了一張慶祝照片，而凱西還為他點了贊。儘管凱西曾效力於亞特蘭大，曾與戈麥斯並肩作戰，但在AC米蘭大敗的節骨眼上做出極具爭議的點贊敵將的舉動，無疑會引起紅黑球迷的群情激憤。何為無恥？凱西給出了明確地答案。

這場恥辱性的失利無疑讓距離前四漸行漸遠的AC米蘭承受著前所未有的壓力。義大利媒體《米蘭體育報》撰文分析了這場慘敗背後俱樂部從上到下應付的責任情況，其中，他們的老東家、管理層、球員和教練都分得了一口鍋。

AC米蘭的老東家埃利奧特對沖基金對俱樂部的投資是非常顯著的，但他們在三個轉會窗口的消費方式無疑是存在問題的。俱樂部CEO加齊迪斯剛剛上任，他還在尋找俱樂部最佳的運作模式，這一點是可以理解的。而在競技層面，馬爾蒂尼、博班和瑪薩拉三人合作時間也才只有半年，他們仍然需要時間，但他們在選擇球員方面顯得不夠成熟。主帥皮奧利是罪責最小的人，他才上任僅僅兩個月。當然，率隊每天參加訓練是皮奧利的責任，他不可能對這場失利不負責任。

《米蘭體育報》稱，我們不能把這場恥辱性的失利歸咎於AC米蘭的球員，這支球隊的陣容水平的確不高，球隊太過年輕，他們缺乏個性和鬥志。不過，球員們在昨天的比賽中所展現出的精神面貌是任何俱樂部都無法容忍的。一支名字叫做米蘭的球隊，在聖誕節時距離歐冠區竟然還有14分的差距。皮奧利已經發現了紮根在紅黑軍團靈魂深處多年的病根，他在擔任俱樂部主帥三周之後如是說道：「不管是贏球、輸球還是平局，對我們似乎都已經沒有任何影響了。」
