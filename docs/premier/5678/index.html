矛盾公開化？C羅直接打臉尤文跟薩里，風格轉變進球數回到10年前

2019-11-15 0條評論 17次閱讀 0人點贊

今天凌晨，歐洲杯預選賽第9比賽日B組一場焦點戰在阿爾加夫球場展開爭奪，葡萄牙主場6比0大勝立陶宛，C羅帽子戲法，皮齊、帕西恩西亞和貝爾納多進球。

在賽後接受媒體採訪時，C羅表示：「我跟你說一件各大報紙頭條級別的事情吧，隊長（指他自己）的身體狀況不是『好』，而是非常好。」C羅的這番話，意有所指：上周末的意甲聯賽中，C羅被尤文圖斯主帥薩里換下後嘴裡「喋喋不休」，並且直接離開了球場。薩里賽後表示，感謝C羅「帶傷上陣」。

尤文方面還在等待C羅的解釋及道歉，而C羅的這段話，無疑是「火上澆油」，赤裸裸地打薩里的臉。雙方的矛盾，或許將進一步激化。那麼，在俱樂部層面，是挺C羅，還是挺薩里呢？

C羅完成帽子戲法後，一系列的數據，又出來了：7場比賽13球（國家隊），連續13年進球30+……可是，大家有沒有發現，C羅的這13球，有7個球是在立陶宛身上取得的（兩場比賽）。在面對荷蘭時，並沒有取得進球。同時，C羅連續13年進球，大家有沒有看詳細數據？其今年的進球數據，僅有32球4助攻！上一次出現這種情況，還是2009年，那時候的C羅，僅有24歲。

我們再來看C羅的這三個進球，都是「一擊致命」！再聯想一下不久之前卡佩羅所說的話：C羅已經連續三年沒有一次過人成功了。其話，雖然過於偏激。但是，也道出了一個事實：C羅的踢球風格，已經轉變了。他，不再像過往那樣風馳電掣，連過數人，而是成為了一名「禁區內」的射門機器。

​

為何？隨著年齡的增長，C羅的身體狀態，已經不如過往，想要輕易地過人，難了！而射術，並不受年齡因素的左右。反過來講，還可以做到越老越精準。通過這一些列的事實，我們的承認：C羅，老了。他不再用身體踢球，而是用腦……而當尤文需要有人持球突破的時候，薩里選擇用迪巴拉換下C羅，無疑是一個正確的決定！
