復盤：萊加內斯“獨特風格”打亂西班牙人，武磊受限無所建樹

2019-12-23 1,077條評論 3,395次閱讀 1338人點贊

前言

萊加內斯主場挑戰西班牙人，這是一場提前上演的保級生死戰，因為對陣雙方都處在降級區，西班牙人與萊加內斯同積10分，但是因為凈勝球的劣勢，西班牙人排在了萊加內斯的後面，目前西甲聯賽排名墊底。

無論是西班牙人還是萊加內斯，這場比賽都是一場不能輸的「生死戰」，雙方一定會拼盡全力，拿下對手，取得寶貴的3分。

西班牙人隊進攻核心達德爾本場比賽的觸球點和射門位置

然而現實是殘酷的，西班牙人沒能在客場取得進球，無功而返，萊加內斯主場力克西班牙人全取三分；武磊在79分鐘替補登場後也表現平平。而對手萊加內斯則打出了自己的風格，在技戰術層面上也體現出極高的水準，贏下比賽也是理所當然。

萊加內斯本場比賽「獨特風格」到底是什麼？西班牙人隊很難打出「效率足球」的原因何在？

雙方陣型解讀分析

  • 西班牙人隊

西班牙人隊主教練馬欽一如既往的排出了自己心中的最佳陣型5-3-2。

如下圖所示，3中衛搭檔依然是貝爾納多，戴維洛佩斯和卡萊羅；左右兩個邊翼衛分別是迪達克與維克托戈麥斯；後腰則是馬克羅卡，在馬克羅卡身前的兩名中場分別是達德爾與格拉內羅；雙中鋒分別是卡萊里和坎普薩諾。

西班牙人本場比賽的初始陣型

在西班牙人的這套5-3-2陣型里，達德爾無疑是場上的核心。

如下圖所示，達德爾本場比賽的觸球點及射門位置。從圖上我們可以看到，達德爾本在本場比賽的觸球位置幾乎遍布全場，除此之外，達德爾的組織，串聯，射門各項數據樣樣都有。達德爾承擔了場上進攻的重任，並且在球隊由守轉攻時扮演著一個發起者的角色。

達德爾本場比賽的觸球點及熱點圖分析

  • 萊加內斯隊

    • 如下圖所示，萊加內斯本場比賽排出了與西班牙人隊一模一樣的陣型（5-3-2），但是萊加內斯的風格與西班牙人隊大相徑庭。

    萊加內斯強調的是身體力量與技戰術的結合，也可以叫它現代版的「長傳沖吊」，而西班牙人隊則繼續繼承了加泰羅尼亞大區的「傳控足球」風格，堅持走地面，從門將到頂在最前面的前鋒。

    萊加內斯5-3-2陣型

    兩支風格完全不同的球隊相遇，將會擦出什麼樣的火花呢？

    復盤

    一，萊內加斯「長傳沖吊」打身後，西班牙人難以適應

    萊加內斯作為馬德里大區的球隊，在踢球風格上更開放更注重身體力量與技戰術的結合，但是他們不光是一味利用自己的身體優勢，像早年間的英格蘭隊那樣，萊加內斯的打法非常現代化，「長傳沖吊「的同時，還有很多非常細膩的配合，做到了「粗中有細」，粗獷與細膩的結合。

    如下圖所示，這是昨晚在同一時間段進行的兩場比賽的同一個數據（高空爭奪次數）的對比。

    處在上面的是亞特蘭大與AC米蘭的比賽數據，在底下的是萊加內斯與西班牙人比賽高空爭奪次數的數據。因為萊加內斯主要以「長傳沖吊」的打法而見長，所以這場比賽在高空爭奪的次數上很多，基本是亞特蘭大與AC米蘭那場比賽的2倍。

    萊加內斯與西班牙人各有18次高空爭奪

    萊加內斯本場比賽的長傳令西班牙人很不適應，很快，西班牙人隊就陷進了萊加內斯隊的圈套........
    • 西班牙人隊的第一粒丟球

    如下動圖所示，這是西班牙人在上半場開場不久之後的丟球。

    萊加內斯後場一個大腳開到前場找自己的前鋒，西班牙人後衛戴維洛佩斯搶到了一點球，將球頂給後腰馬克羅卡，馬克羅卡在對方多人的包夾之下想快速將球解圍，但是無奈趕到了他的右腳（左腳是馬克羅卡的慣用腳），解圍踢跐了，隨後萊加內斯隊的中場球員迅速抓住這次機會，送出了一腳精準的直塞球，7號丹麥前鋒布萊斯維特前插得球後左腳推射遠角得分。

    西班牙人的第一粒丟球

    我們可以看到，萊加內斯的進攻打得非常簡練，長傳進攻讓西班牙人隊隊員在防守時顯得有些手忙腳亂，這也是萊加內斯隊最為擅長的進攻手段。

    如下圖所示，羅卡丟掉球權球之後，對方7號立刻加速啟動向身後的空當跑去。在這裡，我們來看看西班牙人隊2名防守球員的防守姿態，第一個貝爾納多，並沒有察覺到對方7號已經快速前插，自己還是一個正面防守的動作，沒有側身去跟對方的7號，這就導致了後來它自己完全跟不上對手，貝爾納多的失誤是這次丟球的直接原因；

    第二個則是在貝爾納多身旁的卡萊羅，如果貝爾納多沒有跟上，卡萊羅也可以過來補位，但是顯然在這一刻卡萊羅沒有意識到他身後的「危險區域」，如果他提前向後退兩步，或許他就可以幫貝爾納多彌補身後的位置，將丟球化險為夷。

    第一粒丟球分析

    • 長傳找中鋒是萊加內斯隊的進攻手段之一

    如下動圖所示，萊加內斯隊在上，下半場幾乎每一次進攻都是這樣發起的。從後場直接起長傳，打到對方7號布萊斯維特身旁附近，布萊斯維特可以將球卸下來直接射門，就像下面動圖的這一次進攻；還有一種進攻套路就是7號布萊斯維特頭槌回做，接下來隊友都蜂擁而至，在此區域迅速形成一個「集團軍」，最後展開配合，打開西班牙人隊的防線。

    萊加內斯隊進攻簡單直接

    那麼，問題來了，萊加內斯的「集團軍」作戰是如何實施的呢？

    二，萊加內斯「區域集團軍」作戰優勢明顯，小區域人數優勢形成致命打擊

    萊加內斯的「長傳沖吊」讓西班牙人隊主教練馬欽顯得有些無計可施，馬欽也並沒有做出太多的調整，緊接著，萊加內斯又用自己「最喜歡的方式」獲得了一次很有威脅的射門良機。

    如下動圖所示，這次進攻的發起是大力手拋球，它的效果跟大腳長傳一模一樣。手拋球開向自己人數優勢明顯的區域，隨後一大批球員蜂擁而至在西班牙人右側區域形成了「集團軍」作戰之勢（2打2局面），再加上一個精彩的2打2斜傳直插，成功的撕破了西班牙人隊的防線。

    大力手手拋球至人數優勢的區域，隨後迅速展開快速進攻

    萊加內斯隊球員大力手拋球，拋向自己人數有優勢的區域，隨後迅速展開快速進攻，幾名球員的跑位很堅決，很清晰，這樣做的目的就是能夠形成以多打少或像上圖所示的2打2的局面，這就是萊加內斯隊採用區域「集團軍作戰」的原因所在。

    再來看第二個例子，如下圖所示，這是西班牙人隊第二粒丟球前的截圖，萊加內斯隊成功地利用了西班牙人隊邊翼衛和右中衛之間的防守空當，成功的完成了一次邊路2過2撞牆配合。

    邊路2對2撞牆配合

    24號羅德里格斯與7號布萊斯維特完成配合之後，接下來他需要做的就是下底傳中，可是傳中的話有兩種選擇，1. 直接將球傳到門前。 2. 選擇一個倒三角線路的傳球。

    而羅德里格斯選擇了後者，這是一個聰明的選擇，因為如果直接傳向門前的話，門將與西班牙人的後衛都比較容易解圍，而如果選擇倒三角路線的話，不但可以迷惑門將的視線，還可以讓對方的中後衛措手不及。而26號恩尼西里也心領神會，跑到了倒三角的接球區域（如下圖綠色區域），完成了射門並取得了進球。

    倒三角傳球

    完整動圖如下，先是中場的一個過頂長傳，打到西班牙人邊翼位維克托戈麥斯的身後，隨後立即展開了2過2的撞牆配合與倒三角傳球破門。

    雖然萊內加斯隊的打法整體來看比較「粗獷」，長傳球與身體接觸較多，但是他們在關鍵位置與局部還是打出了自己的內容與一些十分細膩的戰術配合。這就是我們所說的「粗中有細」，關鍵還要看你的戰術細節，萊內加斯很好的做到了這一點。

    西班牙人第二粒丟球

    馬欽落後時勇於變陣，武磊在變陣後的4-3-3體系下受到限制

    下半場落後兩球之後，馬欽也嘗試著調整場上的陣型與戰術，他曾嘗試過5-2-3陣型，但是緊接著換上武磊之後，又轉變為4-3-3陣型。

    馬欽在主場迎戰奧薩蘇納時曾使用過4-3-3陣型，當時球隊跟現在的情況極為相似，都是處在落後的階段。具體的打法我們看下面的圖，這是主教練馬欽變陣後的陣型（4-3-3），3名中後現在變為2名中衛，戴維洛佩斯前提出任球隊後腰，剛換上場的武磊打右邊鋒，中鋒也是剛換上場的費雷拉，左邊鋒則是卡萊里。

    西班牙隊變陣4-3-3後的基本套路

    在這裡大家可能會有些疑問，讓一名中鋒（卡萊里）踢左邊鋒是不是有些不合適呢？

    其實這三名前鋒根本就不用出來活動，他們只需呆在禁區里進行搶點，給門前製造一些混亂，找準時機取得進球。所以，他們名義上是邊鋒，實際上確是「假邊鋒」，把邊路留給左右兩邊的邊後衛（原本應該是邊鋒），讓邊後衛進行傳中，3名前鋒進行搶點即可。現在你知道為什麼讓兩名大個子上場的原因了吧。

    武磊替補登場表現平平

    這一戰術，是馬欽的搏命戰術，對於現代足球來說這樣的進攻方式效率非常低。世預賽中國隊與敘利亞隊的比賽，里皮在下半場也採用過這樣的搏命戰術，但是效果都很一般，二者都沒有製造奇蹟。

    而中國球員武磊，顯然在馬欽的4-3-3搏命戰術之下無法發揮自身的優勢。因為想要在中路搶點就得要與對方中後衛身體對抗，為什麼馬欽安排人高馬大的費雷拉與卡萊里在中路搶點呢？就是因為他們的身體對抗能力強，可以爭下來一點球，從而才有機會。

    武磊的身體一直是他的弱勢，所以他在下半場的「肉搏戰中」幾乎沒有體現價值，觸球機會也寥寥無幾（如下圖所示，武磊只有2次觸球的機會），武磊自身的速度優勢也無從體現。

    西班牙人隊的進攻效率問題仍然是迫在眉睫。

    總結

    萊加內斯隊在這場比賽重新詮釋了什麼是現代版的「長傳沖吊」，長傳打到本方人數優勢的區域，隨後迅速展開快速而有序的進攻，有撞牆配合，也有倒三角配合；這些都說明萊加內斯隊不僅僅是「長傳沖吊」打法，它還結合著一些細膩的戰術配合，這才是它最終贏下比賽的原因。

    而西班牙人隊在面對「風格突出」的萊加內斯隊時，顯得有些無計可施，下半場的變陣也沒有達到預期的效果。可以說，目前的西班牙人隊已經迎來自己真正的寒冬，如果再不解決球隊的防守盯人及進攻效率問題，那麼球隊離降級也就越來越近了。
