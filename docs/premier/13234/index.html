歐冠前瞻：藥廠戰意十足，尤文客場力不從心？

2019-12-11 1,073條評論 3,400次閱讀 1804人點贊

主隊勒沃庫森此前取得了2勝0平3負的歐冠小組賽戰績，目前藥廠積6分位列第3，和身前的馬德里競技隊只有一分的差距，只要本輪主場爆冷擊敗尤文，在同時馬競不勝，這支球隊還是擁有逆襲對手而出線的機會，本賽季位列德甲第6的勒沃庫森在荷蘭人博斯的帶領之下終於擺脫了低迷，雖然他們還只是很年輕的球隊，賬面實力也是非常有限，不過能夠主場擊敗馬德里競技也證明了其爆冷的能力。

此前，尤文圖斯取得了4勝1平歐冠戰績，已經以13分的優勢確立了第一的優勢。但是目前他們的近況不佳，剛剛客場1-3不敵拉齊奧，連續兩輪未嘗一勝的斑馬軍團主帥薩里如今遭到了球迷巨大的質疑聲，耗資一億歐元引進了金球先生C羅，但是羅總卻在薩里的治下變得越來月沒有威脅，伊瓜因和迪巴拉兩位射手和C羅組合的合理性是目前尤文面對的最大難題。

根據博體體育提供數據分析，本場亞指主隊讓平半中低水，目前已經有部分公司調整為主讓半球高水，此役對於尤文來說雖然不涉及到小組排名的問題，但是目前鑒於薩里的執教理念以及戰術體系已經面臨質疑，他還是希望球隊能夠重視任何對手。而從目前遊戲指數的情況看，意甲衛冕冠軍居然遭到了無視，可見勒沃庫森在對手戰意不強的背景下有望贏球。
推介：勒沃庫森 信心指數 ★★★★
