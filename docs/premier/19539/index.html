足壇事件2010-2019，這10年大事改變足球

2019-12-18 527條評論 1,679次閱讀 487人點贊

從2010年到2019年，足壇發生了許許多多的變化，有些事你可能已經記不清楚，有些則絕對不會被遺忘。在21世紀第2個十年行將結束之際，我們回顧總結出這期間世界足壇的十件大事，它們有的振奮人心，有的引人唏噓，有的則至今眾說紛紜。但毫無疑問，這些大事件，都對國際足球這十年間乃至更長遠未來的發展變化，產生著深遠的影響。

1、瓜帥推動傳控席捲足壇

雖然「Tiki-taka」理論並不是最近十年才被提出，其代表球隊巴塞羅那和西班牙國家隊也早在上個十年的末期就已開始稱雄世界足壇，但「傳控」這種風尚在全世界徹底流行，那還是2010年以後的事情。南非世界盃，博斯克率領的鬥牛士軍團以當時巴薩的人員和戰術為基礎，將短傳滲透進攻和控球在腳「防守」的套路發揮到了極致。他們在決賽擊敗依靠防反晉級的荷蘭，也代表了傳控風格的徹底勝利。

除了西甲本身就有傳控傳統，傳控代表人物瓜迪奧拉還把這種理念帶到了德甲和英超。瓜帥和瓜式足球吸引了無數信徒， 很多人堅信「傳控就是華麗足球和成功足球」，各隊管理層對這種風格趨之若鶩，很多新晉和非新晉的教頭也成為了它的忠實仿效者。由此，足壇出現了兩個經久不衰的話題：其一，傳控怎麼破？這種足球為強隊帶來的壓倒性優勢經常會讓對手感到絕望。其二，是不是所有球隊都適合傳控？關於這一點，最近阿萊格里表示防反應該復甦，稱瓜迪奧拉把世界足壇「帶歪了二十年」。

2、貪腐醜聞籠罩足球管理部門

近十年來世界足壇最大的「政治事件」，當屬國際足聯的徹底改朝換代。2015年6月，時任FIFA主席布拉特在剛剛第5次連任後宣布辭職，並在半年後遭到了FIFA道德委員會的「禁足令」。國際足聯長達17年的「布拉特時代」徹底結束，因凡蒂諾在2016年2月被選為新一任主席。

倒掉的足壇大佬遠不止一位，過往數年甚至十數年中國際足聯涉及的各種貪腐都浮出水面。從2010年開始，英國媒體就曾多次曝光FIFA官員涉嫌貪腐，包括接受來自卡達方面的賄賂，支持他們申辦2022年世界盃。而2015年5月底，美國司法部門聯合瑞士警方，在蘇黎世突擊了FIFA官員的駐地，抓捕7名高管。這一事件震驚世界，也導致了隨後布拉特的下台以及FIFA的改組。時任歐足聯主席普拉蒂尼亦因貪腐問題遭到「禁足」，世界足壇的風雲人物從此變成局外人。

3、科技手段介入判罰

在布拉特時代，國際足聯的態度曾是「誤判也是足球的一部分」，長期拒絕引入科技手段輔助裁判。然而，不斷進步的技術和無法平息的爭議，還是令歷史潮流發生轉變。2012年，FIFA通過了使用門線技術的決議，世界足壇從此告別了各種「門線懸案」。而2016年因凡蒂諾上任後，視頻助理裁判技術也開始得到FIFA認可，並逐步在全世界推廣開來。

從最早僅在美國大聯盟的預備隊比賽中測試，到歐洲各國聯賽紛紛引入，以及在2018世界盃全面應用，VAR早已成為世界足壇的潮流。不過除了能減少冤假錯案，這項新技術的運用卻也明顯存在不成熟之處，有時反而引發了更多爭議。最典型的，就是英超2019-20賽季引入之後，VAR頻繁被用於鑒別「體毛越位」，同時卻錯過了一些明顯的錯漏判。

4、大賽忙擴軍，新創歐國聯

早在2013年，挪威足協主席就透露，歐足聯正在計劃建立一項新的國家隊賽事。2014年，舉辦歐洲國家聯賽的決議得到了歐足聯55個成員的討論通過。最終這項賽事於2018年開打，並在2019年決出了首屆冠軍葡萄牙。

目前，歐國聯尚遠未具備世界盃或歐洲杯的影響力，不過它的出現已足夠體現世界足球的一種發展趨勢，那就是各個利益團體都在搶奪市場，希望以創立新賽事或擴大原有賽事規模的方式來牟取更多利益。比如歐洲杯就在2016年從16強擴軍到了24強，非洲杯和亞洲杯也都在2019年改為決賽圈有24隊參加。除此之外，2026世界盃擴軍到48強的方案也在2017年得到了正式通過。

在俱樂部層面，除歐冠和歐聯杯之外的第三級歐戰「歐洲協會杯」已在2019年確定成立，2021年開打。另外，由24支各大洲俱樂部參加的新版世俱杯，也已在2019年確定，將於2021年在中國開戰。

5、財政公平法案難擋通貨膨脹

在前一個十年的最後階段，歐足聯原則上通過了一條以法案形式控制歐洲各俱樂部盲目燒錢、無度消費的計劃。而隨後「財政公平競賽」法案（FFP）誕生，成了近十年懸在各俱樂部頭頂的一把達摩克利斯之劍。FFP的基本原則是要求各隊「量入為出」，不可過度舉債，如果支出大於收入到一定程度，那就要受到歐足聯的制裁。這一法案的確影響了不少俱樂部的經營策略，也曾有多家球會因此受罰，包括英超曼城曾在2014年遭到罰款、意甲AC米蘭在2019-20賽季被禁止參加歐戰等等。

不過，FFP也被一些人批評為「只打蒼蠅不打老虎」，執行並不嚴格，有太多漏洞可以讓有錢老闆鑽空子，因此它的限制作用其實非常有限。最明顯的案例就是巴黎聖日耳曼敢於連續以2.22億和1.8億歐元的價格買進內馬爾和姆巴佩，該俱樂部的營收能力其實比較有限，全靠有錢的卡達老闆「左手贊助右手」。類似這樣的瘋狂燒錢，也促成了轉會市場前所未有的通貨膨脹，而今，轉會費破億已不是新聞。

6、梅羅爭雄展示西甲霸權

從2008年C羅榮膺金球獎開始，世界足壇便進入「絕代雙驕」時代，梅西和C羅的「當世最佳球員」之爭主宰足壇。兩位不斷刷新各種紀錄的超級巨星壟斷了最高級別的個人獎項，一度連續10年包攬金球。21世紀第二個十年中，只有2018年莫德里奇捧走金球，打破了二人的壟斷。

梅羅爭雄的這十餘年中，兩人在國家隊的成就相對而言不算特別出色，但俱樂部層面，他們推動西甲統治了歐洲。過去9屆歐冠，除了2012年的切爾西，2013年的拜仁和2019年的利物浦，其餘6次冠軍得主不是皇馬（4次）就是巴薩（2次），充分顯露出西甲在這十年中的極度輝煌。

7、弗格森退休，英超洗牌

2012-13賽季結束之後，執教曼聯長達27年的弗格森爵士正式宣布退休，也代表著英超的一個時代徹底翻篇。在那之前，英超無論誰與誰爭霸，紅魔都是絕對主角之一。而在那之後，英超進入輪流坐莊的時期，曼城、切爾西甚至萊斯特城都曾稱雄。「曼聯王朝」從此不見蹤影，紅魔以自身標準而言可謂一蹶不振，曼徹斯特倒向藍色一邊。當然，截至目前，英超似乎還並沒有新王朝的建立，瓜迪奧拉麾下的藍月亮雖已聯賽兩連霸，但在2019-20賽季似乎已經難敵另一股紅色——利物浦。

8、米蘭雙雄易主，意甲式微

在這個十年的最初階段，米蘭雙雄依然處在一個輝煌時代。國際米蘭2010年在穆里尼奧帶領下問鼎歐冠，加冕三冠王；AC米蘭則在2010-11賽季第18次奪取意甲冠軍，曾讓人覺得他們再一次迎來了復興。然而隨後的故事卻是兩家歷史悠久的豪門雙雙遇到經營難題，創收能力始終無法取得突破，原老闆莫拉蒂和貝盧斯科尼也難以為繼。

2013年，莫拉蒂將國際米蘭70%的股份賣給了印尼人托希爾，正式開啟了藍黑軍團易主的進程。隨後蘇寧集團又在2016年進駐，逐漸從托希爾手中接過了這家老牌俱樂部。至於AC米蘭，貝盧斯科尼在2015年就開始不斷尋求買家，並最終在2017年將俱樂轉讓給了以中國商人李勇鴻為首的投資集團。不過，米蘭雙雄都未能在易主後迅速恢復豪門元氣，國米直到2019年才重返歐冠，AC米蘭更是折騰一番之後，因李勇鴻無法償還款項，而在2018年被美國埃利奧特對沖基金接管。

雙雄的萎靡直接導致意甲在電話門後繼續走低，整個聯賽只剩尤文圖斯一家真正豪門。意甲不僅冠軍懸念近乎消失，聯賽吸引力銳減，歐戰競爭力也大幅下滑。2010年之後，意甲前所未有地連續9年無緣歐戰冠軍。

9、中資登陸歐洲，進退匆匆

曾幾何時，能在歐洲足壇引發話題的，大多是富可敵國的中東老闆、出手闊綽的俄羅斯寡頭，抑或摳門的美國商人。而從2015年左右開始，「中國資本」也成為一個關鍵詞。大到萬達入股馬競、蘇寧入主國際米蘭，小到西班牙低級別聯賽球隊被徐根寶投資，中國老闆的足跡不僅遍布英法西意荷五大國，就連捷克聯賽也有他們的資本注入。最鼎盛時期，中資涉及的歐洲俱樂部差不多超過30家。

不過中資來勢洶洶，退潮速度卻也不慢。從李勇鴻徹底告別AC米蘭，萬達從馬競退股，到夏建統轉賣維拉、鄭南雁從尼斯離場等等，中資在歐洲足壇的「風光」僅持續了兩三年。當然，退潮並沒有帶走一切中資，目前張近東的國際米蘭和郭廣昌的狼隊都有不錯表現。

10、歐洲女足躍進

女足世界盃在2015年擴軍為24強，而女足運動也在過去十年取得長足進步。最明顯的，當屬歐洲各國依託男足的俱樂部體系，對女足運動注入了更多資金和關注度。在西班牙，馬競女足和巴薩女足曾在2019年4月創造新的俱樂部女足賽事現場觀眾人數紀錄，而女足英超的2019-20賽季也在不斷刷新自己的各種觀眾紀錄。另外，曼聯和皇馬等老牌豪門近兩年也新建了屬於自己的女足隊伍。俱樂部女足的發展，還直接帶動歐洲男足傳統強國在女足國家隊層面的競爭力不斷提升。
