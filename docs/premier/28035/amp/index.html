歐冠歷史上十大經典比賽，驚天大逆轉，有夢就會成真

天天要聞
2019-12-20

第十位：切爾西逆轉那不勒斯

首回合客場1∶3兵敗，切爾西要想完成逆轉難度很大，但藍軍沒有放棄，相反三員老將書寫了傳奇，整個斯坦福橋都為老兵傳奇狂歡。

回顧整場大戰，可以說是德羅巴、特里、蘭帕德主宰了比賽進程，在最需要進球的時刻，老兵們都站了出來。第28分鐘，拉米雷斯左路傳中，德羅巴頭槌打破僵局。第47分鐘，蘭帕德開出角球，無人防守的特里前點頭槌擴大比分。2∶0，切爾西佔據客場進球多的優勢，此時那不勒斯只能加強進攻。第55分鐘，因勒扳回一城。英超藍軍再次站到懸崖邊上，第75分鐘，蘭帕德開出角球，伊萬諾維奇頭槌攻門被多塞納用手擋出，蘭帕德點球破網將兩隊總比分定格為4∶4平。與德羅巴、特里、蘭帕德這些大佬相比，伊萬顯得並不起眼，可是巴爾幹奇兵總能在關鍵時刻完成絕殺。加時賽競爭更加激烈和殘酷，老兵們在體能下降的情況下利用經驗與對手周旋，而伊萬猶如一部永動機仍然不斷奔跑尋找殺機，演繹著奇兵新傳。第105分鐘，德羅巴禁區右側轉身低傳，伊萬諾維奇勁射破門幫助切爾西4∶1勝出，總比分5∶4挺進八強。

第九位：紅軍64分鐘連中三元逆轉出線

2004年12月8日，歐冠小組賽最後一輪，利物浦必須贏奧林匹亞科斯兩球，才能出線。第26分鐘，里瓦爾多主罰自由球得手，紅軍必須在剩下的64分鐘內打入三球。第47分鐘和第81分鐘，替補出場的龐戈勒和梅洛各入一球。第86分鐘，傑拉德以標誌性的凌空怒射助紅軍晉級。小組賽的逆轉似乎是伊斯坦布爾奇蹟的預演，利物浦的韌性之強讓人震撼。

第八位：巴薩4球大勝力挽狂瀾

2000年冠軍杯1／4決賽，巴薩與切爾西狹路相逢，首回合藍軍出人意料地在斯坦福橋3比1取勝，但次回合巴薩上演了不可思議的大逆轉，5比1大勝晉級歐冠半決賽，進球的分別是里瓦爾多、菲戈、達尼和克魯伊維特。

第七位：莫倫特斯弒舊主 摩納哥力斬皇馬

首回合比賽皇馬在主場4－2擊敗對手，本場比賽中，儘管勞爾幫助皇馬首開記錄，但摩納哥隊憑藉久利的梅開二度和莫倫特斯的進球主場3－1擊敗皇馬，雙方總比分打成5－5，摩納哥隊憑藉客場進球多的規則逆轉淘汰皇家馬德里隊，皇馬5年來首度無緣歐冠半決賽，摩納哥一路繼續著自己的神奇之旅。

第六位：拉科爆發為平民球隊證明

04年歐冠1/4決賽，拉科魯尼亞隊在首戰1比4失利的不利局勢下上演驚天逆轉,主場4比0大勝衛冕冠軍義大利AC米蘭隊,躋身半決賽。誰也沒有想到發生在阿森納隊和皇馬隊的悲劇事會在AC米蘭隊身上重演,但這樣的情還是發生了。擁有首回合四粒進球和「烏克蘭核彈頭」舍甫琴科復出的「紅黑軍團」整場行同夢遊，根本沒有表現出應有的水平，只得抱憾出局。

第五位：巴薩5分鐘閃電逆轉二度奪魁

一場不期而至的冷雨沒有澆熄法蘭西大球場的如火激情，在04-05賽季利物浦製造8分鐘內連扳3球的驚天逆轉之後，巴塞羅那和阿森納05-06賽季歐冠決賽再次聯袂奉獻了一場驚心動魄的歐冠決賽。這是一場奇怪的比賽，萊曼的衝動讓阿森納開場僅18分鐘就面臨10人應戰的窘境，以少打多的槍手沒有放棄，坎貝爾一頭將亨利開出的自由球刺入了巴塞羅那的心臟。正當人們認為西甲冠軍的歐冠美夢將被巴黎的雨水衝散時，埃托奧和貝萊蒂卻在5分鐘內奇蹟般的連入兩球，阿森納在距離首座歐冠獎盃只有10分鐘時愴然倒地，而巴塞羅那卻帶著歐冠本賽季的不敗金身，昂首在巴黎加冕歐洲之王。

第四位：皇馬逆轉稱王首屆歐冠

55到56賽季歐冠決賽，皇馬出戰來自法國的蘭斯。當時的皇馬是歐洲最優秀的球隊，因此在賽前他們被普遍看好。但在決賽中，蘭斯的表現相當不錯，他們一上來就以2-0取得領先。不過在壓力之下，皇馬在傳奇巨星迪斯蒂法諾的帶領下大爆發，他們最終上演大逆轉以4-3取得比賽勝利。皇馬奪取首屆歐洲冠軍杯冠軍，同時球隊也拉開了連續五年稱霸歐洲的序幕。

第三位：巴薩驚天大逆轉巴黎聖日耳曼

2016/17賽季歐冠1/8決賽次回合繼續進行，坐鎮主場諾坎普球場的巴塞羅那足球俱樂部6-1屠殺巴黎聖日耳曼足球俱樂部，從而以總比分6-5驚天翻盤晉級八強。

巴薩的逆轉是歐冠歷史上前無古人後難有來者的大逆轉，從來沒有任何一支球隊能夠在首回合4球慘敗且沒有客場進球的情況下逆轉比賽。

坐鎮諾坎普球場的巴薩上演驚天翻盤，主場6-1屠殺巴黎，以總比分6-5晉級八強。蘇亞雷斯閃電破門助巴薩取得領先，庫爾扎瓦烏龍球和梅西點球破門一度助巴薩拿到3-0領先。隨後卡瓦尼破門令巴薩晉級前景渺茫。但內馬爾梅開二度確立了巴薩5-1領先優勢，補時階段第4分鐘，羅貝托打進本場比賽第六球，讓巴薩起死回生。第二位：冥冥之中的天意逆轉

曼聯奇蹟1999年的最後一戰，奇蹟程度也達到了頂峰。曼聯球員和球迷都懷疑，這是不是冥冥中的神的旨意，因為這天就是巴斯比爵士的90歲誕辰。這場神奇的比賽已經被世界各地的報道渲染過無數次，因此你難以再找到什麼新的故事。在比賽結束前，無數曼聯球迷都向上帝祈禱著奇蹟能發生，但也許他們事實上已經感到絕望。但替補球員謝林漢姆和索爾斯克亞第一位：紅軍勇敢之心搏回大耳朵打碎了拜仁球迷的心。第一位：紅軍勇敢之心搏回大耳朵

本場比賽註定載入史冊，作為歐洲俱樂部賽事的最高舞台，冠軍聯賽的決賽，三球逆轉的好戲在歷史上絕無僅有。實力強大的紅黑軍團在上半場早早確立勝勢，馬爾蒂尼的歐冠決賽首球和克雷斯波的梅開二度幫助球隊三球領先。事後有媒體爆料，中場休息時，米蘭將士就已經在更衣室開始慶祝。然而這樣的疏忽也間接助長了一個奇蹟的誕生。下半場比賽，利物浦隊在傑拉德的率領下發動反撲，紅軍在15分鐘內連下三城，不可思議地扳平比分。儘管此後米蘭依舊有機會絕殺，但最終未能如願。結果在點球大戰中，利物浦波蘭門將杜德克的大腿舞迷惑了米蘭的罰球者，而隨著舍瓦漫不經心的一腳射門被他擋出，宣告了伊斯坦布爾奇蹟的誕生，米蘭再一次成了逆轉好戲中的配角。
