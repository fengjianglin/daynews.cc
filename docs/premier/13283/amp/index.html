法國足球預測巴黎冬窗操作，卡瓦尼或離隊，嘗試引進馬內

天天要聞
2019-12-10

虎撲12月10日訊 《法國足球》在本期雜誌中著重討論了巴黎冬季轉會期有可能的操作。

預計離隊球員

卡瓦尼：他將很難繼續留在巴黎了，不過球員經紀人更希望他能留在賽季末合同結束後再走，並且在近期明確表示，「卡瓦尼不會在一月份離開。」目前國際米蘭、那不勒斯和馬競都對卡瓦尼展現了興趣。

德拉克斯勒：他與俱樂部的合同還剩1年半，如果他難以拿出有說服力的表現，那他不得不收拾行囊準備考慮未來了。

庫爾扎瓦：最近幾周，各方都在努力工作以便在冬天為庫爾扎瓦找到一個的令人滿意的解決方案。萊昂納多希望將球員出售到英超，標價約在500到1000萬歐元之間。

蒂亞戈-席爾瓦：他的合同即將在夏天到期，作為隊長，他必須表現得更好，以說服俱樂部給自己續約。

內馬爾：俱樂部已經開始考慮內馬爾在明年離隊的備案了，但無論如何，如果內馬爾離開，巴黎將不可能放手姆巴佩。

預計引援目標

尼斯的阿塔爾和尤文的德西利奧都是球隊關注的目標。此外，那不勒斯的情況和蒂亞戈-席爾瓦的未來將決定巴黎是否會引進他們的中後衛庫里巴利和中場阿蘭。作為阿蘭的替代，雷恩小將卡馬文加也是可能的引援。此外，如果要為內馬爾的離隊未雨綢繆，利物浦的馬內將是巴黎的目標。
