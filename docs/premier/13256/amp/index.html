國際米蘭1：2不敵巴塞羅那。未能出線。賽後雙方官微讓人深思

天天要聞
2019-12-11

北京時間12月11日凌晨4點，國際米蘭坐鎮主場梅阿查迎戰梅西，皮克休戰的巴塞羅那。最終1比2不敵狀態神勇的巴塞羅那，6輪總成績排名巴塞羅那，多特蒙德之後排名第三。未能出線。

第20分鐘，勞塔羅前場搶斷.盧卡庫左路空擋，比拉吉爆射被巴薩門將擋出。

第23分鐘，格子突傳禁區，比達爾拼搶時將球碰到卡萊斯-佩雷斯腳下，佩雷斯門前射門進球！巴薩1-0領先，這也是佩雷斯歐冠首秀以及歐冠首球。

第44分鐘，國米長傳禁區，勞塔羅護球倚住後衛後回做，盧卡庫弧頂左腳勁射，球碰後衛後略略變線竄進球門右下死角！國際米蘭1-1追平。

第86分鐘，法蒂與蘇亞雷斯擋拆二過一配合後，法蒂在禁區弧頂大力抽射，球竄進球門左下死角！巴薩2-1。法蒂攻入歐冠處子球，並殺死國米晉級可能。

法蒂的進球徹底擊垮多次進攻不利越位的國際米蘭。傷兵滿營的藍黑軍團遺憾出局。

賽後國際米蘭，巴薩羅那權威媒體集體發生。看到內容後。五穀雜陳。也許這就是足球的魅力。誰也不能笑傲群雄，面對失敗挫折，你只能變的更加強大。憐香惜玉不適合描述當今足壇！
