早報：曼聯連扳三球後終場前丟球，遭謝菲聯3-3絕平

2019-11-25 0條評論 17次閱讀 0人點贊

英超第13輪，曼聯客場對陣謝菲爾德聯。謝菲聯上下半場各入一球，隨後威廉姆斯、格林伍德和拉什福德8分鐘內連入三球助曼聯反超。終場結束前麥克伯尼幫助主隊完成絕平。最終，曼聯客場3-3戰平謝菲爾德聯。

英超第13輪，曼聯客場3-3平謝菲聯，據天空體育統計，曼聯本賽季英超所有進球（19粒）全部來自於23歲和23歲以下球員。

曼聯客場3-3謝菲聯，曼聯連續12場聯賽客場未零封，創34年來最差紀錄。

本輪英超，曼聯3-3客場戰平謝菲聯。曼聯在比賽中一度連扳三球反超比分，但謝菲聯最後時刻的進球阻止了紅魔帶走勝利。比賽後，索爾斯克亞對比賽做出了評價。

意甲第13輪，羅馬主場迎戰布雷西亞。上半場，羅馬攻勢如潮，但未取得進球。下半場斯莫林頭槌破僵後先後助攻曼奇尼、哲科破門。全場比賽結束，最終羅馬主場3-0完勝布雷西亞。

巴甲第34輪，排名第二的帕爾梅拉斯1-2負于格雷米奧，領頭羊弗拉門戈在還剩4輪的情況下領先第二名13分，提前4輪巴甲奪冠。

日前格列茲曼接受了歐足聯官網的專訪，在此次專訪中，身為世界盃冠軍得主的法國前鋒表示，他之所以選擇加盟巴薩，就是想來學習的。除此之外，格列茲曼還就隊友梅西、家庭和個人成長等話題給出了他的答案。

據西班牙《科貝電台》報道，曼聯的法國中場博格巴正在為了在冬窗轉會去皇馬而努力，他現在已經傷愈，但為了離隊而不想出場。

近日，皇馬的比利時球星阿扎爾接受了法國《巴黎人報》的採訪，他談到了關於姆巴佩和金球獎的一些話題。
