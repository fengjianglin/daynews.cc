歐冠：默滕斯破門洛夫倫扳平，利物浦1-1那不勒斯

天天要聞
2019-11-28

虎撲11月28日訊 北京時間11月28日凌晨04:00，歐冠小組賽E組第5輪，利物浦主場迎戰那不勒斯。上半場比賽法比尼奧傷退離場，默滕斯單刀率先破門。下半場比賽庫里巴利奉獻精彩解圍，米爾納助攻洛夫倫頭槌破門將比分扳平。最終全場比賽結束，利物浦主場1-1戰平那不勒斯。

比賽第6分鐘，迪洛倫佐受傷倒地，隨後並無大礙的他重新投入到比賽中。

第11分鐘，洛薩諾分球，大禁區前默滕斯的抽射被防守隊員擋出，隨後法比安的補射將球打飛。

第14分鐘，在一次拼搶中法比尼奧被洛薩諾絆倒在地。

第19分鐘，無法繼續堅持比賽的法比尼奧被換下場，維納爾杜姆披掛上陣。第21分鐘，迪洛倫佐中場附近送出長傳轉移，右路迅速前插的默滕斯殺入禁區右側後稍作調整，隨後默滕斯右腳一記漂亮的小角度抽射破門得分！0-1！客場作戰的那不勒斯率先取得進球！


不過在進球之前和范戴克的拼搶中，默滕斯存在犯規嫌疑。

第36分鐘，羅伯遜開出前場左路定位球給到禁區，門前范戴克的頭槌攻門被梅雷特穩穩抱住。

第45分鐘，馬內帶球殺入禁區後被迪洛倫佐推倒，不過主裁判並未吹罰點球。


半場補時階段，米爾納禁區左側將球巧妙一扣晃開角度，隨後米爾納的小角度抽射被梅雷特化解。

半場戰罷，利物浦主場暫時0-1落後於那不勒斯。

易邊再戰第55分鐘，約瑟夫-戈麥斯禁區右側將球回敲，大禁區前亨德森的兜傳造成梅雷特撲救脫手，門前菲爾米諾得球後的抽射被庫里巴利在門線前踢出解圍。

第61分鐘，羅伯遜送出橫傳，禁區線附近薩拉赫的左腳推射被梅雷特穩穩抱住。

第62分鐘，張伯倫大禁區前右側得球後送出一記兜傳，後點菲爾米諾的頭槌攻門將球頂偏。

第65分鐘，米爾納開出右側角球給到禁區，門前高高躍起的洛夫倫一記頭槌攻門進球得分！1-1！利物浦將比分扳平！


不過洛夫倫起跳爭頂時有推人犯規的嫌疑。

第71分鐘，薩利赫和庫里巴利在禁區內拼搶激烈。

第80分鐘，維納爾杜姆大禁區前胸部將球一停，隨後維納爾杜姆的右腳抽射將球打偏。

第88分鐘，洛夫倫前場送長傳轉移，禁區右側菲爾米諾的頭槌攻門將球頂偏。

補時階段，兩隊發生激烈衝突，主裁判最終對羅伯遜、庫里巴利和阿蘭三人分別出示黃牌進行警告。

最終全場比賽結束，利物浦主場1-1戰平那不勒斯。

比賽陣容：

利物浦（4-3-3）：1-阿利松、12-約瑟夫-戈麥斯、6-洛夫倫、4-范戴克、26-羅伯遜、3-法比尼奧（19’維納爾杜姆）、7-米爾納、14-亨德森、10-馬內、9-菲爾米諾、11-薩拉赫

那不勒斯（4-4-2）：1-梅雷特、19-馬克西莫維奇、44-馬諾拉斯、26-庫利巴利、6-魯伊、22-迪洛倫佐、5-阿蘭、20-傑林斯基、8-法比安、14-默滕斯、11-洛薩諾
