「歐冠」F組生死戰：國米的救贖+多特爆冷

2019-12-11 289條評論 1,430次閱讀 434人點贊

簡單說一下F組的情況，國米到了非贏不可的地步，即便對手是巴薩。本著和zhuang斗其樂趣窮的鍥而不捨精神，也許只有解這樣的盤才能給心靈帶來一絲慰藉，希望不要輸得太慘。

由於是同一時間段進行的比賽，我反而更擔心的是多特，如果國米一不小心2:0,3:0領先，多特估計也很可能打不穿這個初盤1.75。只是現階段早早退到1.5是何居心，難道擺明了多特最多贏一個？從主客場的轉換來看，讓球力度沒啥問題，最主要的還是要看國米這怎麼踢。所以1.75將1.5很有可能就是考慮到國米大比分領先，多特失去信心，被對手狠狠報復，最後贏球輸盤甚至直接贏不了球。

從這樣一個角度來看，國米靠啥贏球？靠啥大比分贏球？靠啥大比分贏巴薩？據說這場球創了義大利足壇票房紀錄，有超過7萬人趕赴梅阿查現場觀球，主場優勢加成+1 。再一個，據相關消息稱，巴薩將有大部分主力輪休，這也給國米注入了無限的精神能量。

盤面上，半一盤的初指完全已經脫離了合理的讓步，這本質就是一個戰意盤。在如此深盤之下，阻力其實已經非常大了。像這種戰意盤，我認為選擇順勢而為更符合雙方的綜合情況。
