上任阿森納後擺在阿爾塔特面前的頭等難題：人心

2019-12-21 449條評論 2,655次閱讀 601人點贊

昨天阿森納官方正式宣布阿爾特塔出任球隊主帥，簽約三年半，他上任的第一件事就要立即開始著手為他的執教制定規則，他的首要任務將是說服球員們留隊，或者是徹底改變球隊的陣容。因為目前阿森納隊內以奧巴梅揚為首的幾名球員正在重新考慮他們的未來，目前，阿森納隊內不止奧巴梅揚一人開始考慮自己未來，包括扎卡、拉卡澤特、托雷拉等人都存在離隊的可能。

而這將會大大減少了球隊的競爭力，這也是俱樂部、高層、球迷都不願意看到的事實，但事實歸事實，有些東西我們必須要理清，目前的阿森納確實成績不好，更衣室的情況混亂這是不容置疑的事實，球隊中很多球員並不在乎球隊的現狀，這種情況短期內也無法改變。當他在世界媒體面前亮相時，他的第一場新聞發布也給人留下了深刻的印象。

阿爾特塔對於自己的用人規則明確表示：「我想按照自己的方式做事，但通過說服他們這是讓每個人生活得更好的正確方式。每個人首先都要互相尊重，我希望人們對我要求他們做的事情負責。為了幫助他們，我需要知道發生了什麼，我需要了解他們的感受，他們缺少什麼。如果我能做到這一點，我就能幫助他們。這樣，他們就會信任我，跟著我。如果他們覺得自己是這種身份的一部分，他們就會想要留下來，因為他們會很享受，但當你痛苦的時候，我不介意你的離開。」

他很清楚，阿森納需要進行全面的改變，才能讓這個倫敦最成功的俱樂部重現輝煌。而被問到會否選出新的球隊隊長時，他表示：「我需要先在更衣室了解情況。我不急於做決定是因為這不是必需做的。我要先觀察球隊的情況，之後再慢慢做出一些能幫助球隊的決定。過去幾個月發生了很多足球以外的事情，並引起連鎖效應，對球隊的內部關係產生了影響。我們要團結起來，更衣室不容侵犯。」

誰走誰留？A8決心要鐵腕治軍

鐵腕治軍的手段肯定是有效的，但當一個新人主帥剛來就用鐵腕手段，球員會不會信服呢？況且現時我廠的眾多球員也有離心，如何穩定及那個球員需要清洗是目前需解決的問題。現時我廠的排名已然達到史上最差的一個排位，如果隊內的重要球員都出走的情況下，也不可能說一時半刻能找到替代者，所以得看阿爾塔特敢不敢大刀破斧改造球隊了。

先看看近期的傳聞，奧巴梅楊希望得到的是能打歐冠和爭奪冠軍，現在奧巴梅揚的問題是他的合同只有18個月了，阿森納必須得對於他的情況做出決定，因為如果奧巴梅揚不準備簽新約的話，那麼他們就必須最遲得在夏天賣掉他。但作為隊長的奧巴梅楊確實已為阿森納付出了很多，他對球隊現狀很沮喪，但你不能責怪他，作為如此重要的球員，阿爾特塔能不能挽留就要看他的功力了。

拉卡澤特也正處於類似的情況，很顯然他也在觀察俱樂部的進展走勢，但相比奧巴梅楊的態度，拉卡澤特會比較緘默，但若奧巴梅楊真的離開，多多少少都會牽連著他，畢竟在場上場下兩人感情十分好，而且拉卡澤特若離開也不愁下家，自己正處於黃金年齡，當然兩人離開一人都是會對阿森納產生巨大的打擊，惹能留住前面這兩黑風雙煞，對於本賽季的成績是有保障的，但也會面臨他們免費轉會的可能。

另外中場扎卡本賽季由於一些舉動已經與阿森納球迷鬧僵，他在球隊中的地位也受到了動搖，並且不再擔任球隊隊長。但是他仍希望能夠在高水平聯賽重新證明自己，他本人也傾向於冬窗離隊，他與球迷之間頗為緊張的關係也對他的生活和工作都造成了影響，扎卡也希望能重新證明自己，並且非常有可能加盟米蘭，如果此次交易能成行，小編相信也不會有太大爭議，因為扎卡在現時阿森納發揮的作用實在太低了，而且對於慣常阿爾特塔使用的陣型，扎卡的上陣次數也將會大幅減少。

那不勒斯已經以先租後買的方式報價引進阿森納中場托雷拉了，如果阿森納不同意，也準備報價3000萬歐元直接買斷。托雷拉個人也對重返意甲持開放態度，原因他在本賽季的上陣次數太少而且多為替補。儘管托雷拉願意加盟那不勒斯，但是阿森納並不想出售他，托雷拉的作用不容置疑，很多槍迷都非常喜歡他，奈何當時的主帥沒有將他放在一個重要的位置，看看後阿爾塔特的阿森納會不會重新將他重用。

最棘手的問題是球員管理

隨著阿爾特塔的上任，永貝里目前也有離開的打算，據傳他正在與馬爾默洽談球隊主教練的職務，有球迷猜測他可能會繼續留在俱樂部作為阿爾特塔團隊的一部分，但以目前來看永貝里留隊輔助阿爾特塔的可能性更大。早前永貝里已經在推特上表達了對阿爾特塔的支持：「我的想法仍然是一致的，希望大家能站在我們身後，支持整支球隊！過去幾周，能幫助我所熱愛的球隊，這對我來說是一份榮譽。我很興奮未來能和阿爾特塔以及他的團隊共事。」

對阿爾特塔來說最重要、也是最難的工作就是球員的管理工作。看看塔隊能不能有什麼手段穩住局面吧，事實上以塔隊剛入主就要讓現在這批球員服氣也是很困難，但如果全賣了，以塔隊目前的聲望又怕吸引不來球星，所以怎樣處理現今的更衣室是擺在阿爾特塔的頭等難題，現在有了永貝里的配合，相信對於穩定軍心也建立了一個良好的基礎，最後球員的走向我們拭目以待。

永貝里最後一場擔當槍手主帥

好了，最後提醒一下大家，今晚20:30分阿森納將在客場面對埃弗頓，由於備戰時間過短，新帥阿爾特塔將在場邊觀摩，永貝里將執教他最後的一場槍手比賽，所以今晚看點非常多，而且也是黃金時間槍迷切勿錯過了。主帥阿爾特塔需要面對的問題還有很多很多，如果先能扭轉到更衣室的問題，小編覺得已經是塔隊第一步的成功了。
