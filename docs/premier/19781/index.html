一夜之間，三大豪門遺憾收穫平局，皇馬最慶幸、國米最無奈

2019-12-16 3,151條評論 4,859次閱讀 495人點贊

昨夜時分，五大聯賽比賽陸續進行。其中英超、西甲和意甲有多隻球隊出征，但有三支豪門球隊都以平局收場，遺憾地未能全取三分。

英超賽場，第17輪比賽比賽中一場焦點戰在老特拉福德進行，最近頻頻斬落豪門球隊的曼聯主場對陣來訪的埃弗頓。

比賽進行至沒多久，曼聯就取得破門良機，林加德抓住對方球員湯姆-戴維斯失誤於點球點附近轉身掃射，可惜打偏了。

第10分鐘，曼聯球員詹姆斯獲得單刀機會，但最終的射門還是滑球門而出。

頻頻浪費機會的曼聯很快就遭到「報應」，比賽進行至第36分鐘，埃弗頓球員貝恩斯開出右路角球，皮球剛好打在曼聯球員林德洛夫身上，後者將球撞進自家球門，曼聯主場0:1落後埃弗頓。

易邊再戰，落後的曼聯加強進攻，球隊猛攻之下終於在比賽的第77分鐘扳平比分，格林伍德接到詹姆斯的斜塞球低射破門，曼聯1:1戰平埃弗頓。

此後，雙方再無建樹，最終曼聯全場62%控球率、24腳射門的情況下遺憾收穫平局，而埃弗頓則在新主帥弗格森帶領下連續兩場處於不敗。

意甲那邊，領頭羊國米客場挑戰紫百合佛羅倫薩，國米近期狀態不佳，在歐戰上輸給了2隊陣容的巴薩羅那，也從本賽季的歐冠賽中率先出局。

回到聯賽，他們急需一場勝利一掃頹勢，而聯賽排名第13位的佛羅倫薩就是他們最好的拿分球隊。

比賽開始沒多久，客場的國米就反客為主。第7分鐘，國米球員巴萊羅與隊友完成精彩的短傳配合後突入禁區直面門將獲得單刀機會，巴萊羅擺脫門將後低射破門，國米1:0領先佛羅倫薩。

第12分鐘，國米再獲破門良機，勞塔羅于禁區前沿完成導購破門，可惜球射得太正，被門將沒收。

第37分鐘，國米發起快速反攻，盧卡庫助攻勞塔羅推射破門，可惜邊裁舉旗在先是為越位球。

下半場易邊再戰，第58分鐘，博阿滕接隊友達爾伯特左路45度角傳中，前者頭槌高出橫樑。

比賽進行至補時階段，國米遭遇打擊，佛羅倫薩換上替補球員佛拉霍維奇，後者高速帶球突破至國米禁區甩開防守球員勁射破門。最終全場比賽結束，國米客場被讀秒絕平，實屬無奈至極。

西甲那邊，瓦倫西亞與皇馬的比賽中，同樣有補時階段的絕平發生，不過利好是在皇馬那邊。

皇馬對陣瓦倫西亞，是齊達內的球隊反超巴薩佔領榜首位置的絕佳機會，畢竟本輪比賽較早進行的一場比賽中，頭名的巴薩客場對陣皇家社會中僅收穫一分，與自己的積分僅有一分之差。

比賽進行至第6分鐘，本澤馬回敲做球給巴爾韋德，後者掃射被多梅內克擋出。第10分鐘，莫德里奇於隊友禁區前橫敲，本澤馬接球後轉身掃射同樣被擋出。

第19分鐘，卡瓦哈爾右側傳中，羅德里克後點頭槌攻門，由於力量不大被對手門將拿住；第35分鐘瓦倫西亞球員左路傳中，托雷斯門前頭槌攻門在庫爾圖瓦干擾下頂高。

下半場易邊再戰，第77分鐘主隊打破場上僵局，瓦倫西亞球員沃斯禁區底線位置倒三角傳中，索萊爾中路跟上推射破門，瓦倫西亞1:0領先皇馬。

落後後的豪門加強攻勢，在之後的時間中頻頻造殺機，終於在補時的第5分鐘將比分扳平，庫爾圖瓦頭槌攻門被對方門將撲出，本澤馬補射破門，皇馬1:1扳平比分。全場比賽結束，皇馬依靠著本澤馬的絕平客場戰平瓦倫西亞，當然，他們也錯失了反超巴薩升上積分榜榜首的機會。

一夜之間，三支意在全取三分的豪門球隊卻有著相同的平局結果，可謂不甘。曼聯被對手造烏龍球實在是「冤」、而國米在最後時刻被絕平也是無奈透頂，皇馬能在久攻不下的情況下絕平對手怕也是用慶幸來形容了。
