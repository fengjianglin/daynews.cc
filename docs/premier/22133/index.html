ESPN：巴薩中場鬧情緒，下個月將離隊！英超三豪強爭奪法甲超新星

2019-12-19 958條評論 2,211次閱讀 655人點贊

ESPN消息，比達爾和巴薩的關係正日趨緊張，尤其是智利人沒有作為首發參加對皇馬的國家德比。

報道稱，在被巴爾韋德告知不會首發參加對皇馬的比賽後，比達爾憤怒的離開了訓練場，紅藍軍團的其他球員曾嘗試讓這名中場冷靜下來，但沒有奏效，鑒於目前的情況，智利人很可能會在一月份離隊。

國米被認為是有意在冬窗收購比達爾的豪門之一，孔蒂的球隊正著眼於加強陣容，不過巴薩對球員2000萬歐元的要價被藍黑軍團認為有些太高了。

據悉，國米希望能以先租後買的形式簽下比達爾（選擇性買斷），但巴薩對這個提議不太感興趣，不過在國家德比之後，也許諾坎普方面會重新考慮這個可能性。

其實這場比賽比達爾還是替補登場了的，而且賽後球員還在社交媒體上曬照片為巴薩加油，但不得不說比達爾的確可能會在冬窗離隊，尤其是拉基蒂奇最近已經明顯重新得到重用的情況下。

只是暫時不知道比達爾的下一站在哪。

天空體育消息，利物浦、切爾西以及熱刺都在關注里爾前鋒維克托-奧斯梅恩，不過除非里爾在一月份收到一份無法拒絕的報價，否則這名奈及利亞新星至少會在法甲待到本賽季結束。

報道稱，20歲的奧斯梅恩目前在里爾很開心，並且球員現在的首要目標就是幫助球隊拿到下賽季的歐冠資格，本賽季這名前鋒在各項比賽中已經打進12球，包括歐冠中對切爾西的進球，而里爾則在法甲排名第3，落後榜首的巴黎11分。

值得注意的是，奧斯梅恩此前曾在助教若奧-薩克拉門托的指導下成長，而薩克拉門托目前正是穆里尼奧手下教練組中的關鍵人物，此外，之前從比利時簽下奧斯梅恩的里爾體育總監路易斯-坎波斯也和熱刺主帥有著非同尋常的關係。

當然，任何球隊若想簽下奧斯梅恩恐怕都要支付一大筆錢，因為這名前鋒的合同還有四年半到期，而球員已經代表奈及利亞有過9次出場，並打進了4球。

奧斯梅恩這個賽季也很火，接替佩佩的他已經在歐洲打出了名堂，未來很可能會登陸豪門。

穆帥這次執教熱刺，是從里爾教練組挖的人，莫不是這回還要繼續挖人家球員么？拭目以待。

利物浦則剛剛宣布簽下了南野拓實。

伊卡爾迪的妻子兼經紀人旺達-娜拉確認，這名前鋒不會在明年加盟尤文。

自今夏從國米租借加盟巴黎以來，26歲的伊卡爾迪拿出了非常優秀的表現，阿根廷人在出戰的16場各項比賽中打進了13球，這也讓球員明年可能加盟尤文的傳聞不脛而走，然而，娜拉在最近接受採訪時否認了這個可能性。

娜拉告訴Rete 4：「不，不會，伊卡爾迪現在是巴黎球員，他在那裡感到很安定，並且一如既往的在打進很多瘋狂的進球。

唯一讓我感興趣的就是伊卡爾迪開不開心，而他目前在巴黎非常開心。」

其實巴黎能買斷伊卡爾迪已經是對各方都最好的結果了，娜拉不要搞事情就行了……
