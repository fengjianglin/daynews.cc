歐聯杯32強：18支球隊曾進決賽8支球隊奪冠，競爭程度空前激烈

2019-12-13 1,474條評論 2,476次閱讀 287人點贊

隨著最後一場歐聯杯小組賽的結束，歐聯杯32強已經在今天凌晨全部產生，這屆歐聯杯的比賽隊伍空前強大，包括了上賽季歐冠4強的阿賈克斯、歐聯杯亞軍阿森納、去年的歐聯杯冠軍曼聯、意甲榜首國米以及連續三屆衛冕冠軍塞維利亞，還有其他具有競爭力的球隊：勒沃庫森、羅馬、波爾圖、本菲卡等等。

12支歐聯杯小組第一的球隊以及4支歐冠戰績較好的小組第三的球隊屬於第一檔，將與12支歐聯杯小組第二的球隊和另外4支歐冠戰績較差的小組第三球隊組成的第二檔球隊進行抽籤。採用如下規則:1.同小組的球隊將不會在32強淘汰賽中碰面。2.同國球隊不會在32強淘汰賽中碰面。3.第一檔球隊次回合的比賽將在主場進行。歐聯杯32強抽籤儀式將於下周一在瑞士尼翁歐洲足球協會聯盟總部舉行。

本屆歐聯杯的32支隊伍裡面，有18支球隊曾經闖入歐聯杯（聯盟杯）決賽：阿賈克斯、阿森納、本菲卡、布拉加、國際米蘭、曼徹斯特聯、塞維利亞、波爾圖、阿爾克馬爾、勒沃庫森、法蘭克福、格拉斯哥流浪者、羅馬、頓涅茨克礦工、狼隊、里斯本競技、薩爾茨堡、布魯日。其中有8支球隊奪冠：阿賈克斯1次（1991/92），國際米蘭3次（1990/91、1993/94、1997/98）、曼聯1次（2016/17）、塞維利亞5次（2005/06、2006/07、2013/14、2014/15、2015/16）、波爾圖2次（2002/03、2010/11）、勒沃庫森1次（1987/88）、法蘭克福1次（1979/80）、頓涅茨克礦工1次（2008/09）。

本屆歐聯杯小組賽總共進行了144場比賽，總進球達到了396球，場均2.75球，平均每33分鐘便有一個進球產生，在下半場61-75的時間段進球最多，在這時間段打入了71個進球，上半場的進球數為177個，下半場進球為219個。其中阿爾克馬爾和布拉加都打入了15個進球，是歐聯杯小組賽進球最多的隊伍，阿森納和塞維利亞打入14個進球緊隨其後。

本屆歐聯杯6場小組賽取得最多進球的球員是流浪者前鋒莫雷洛斯，哥倫比亞人延續著歐聯杯資格賽的火熱狀態，在小組賽打入6球，領跑射手榜，並幫助流浪者闖入歐聯杯32強，排在他身後的是塞維利亞的穆尼爾、里斯本競技的布魯諾·費爾南德斯和布拉迪斯拉發的什波拉爾，他們在小組賽中打入5球，並列射手榜第二。
