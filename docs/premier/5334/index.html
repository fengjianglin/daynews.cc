傳射建功！5場歐冠打入8球！意媒：曼聯和尤文爭奪哈蘭德

2019-11-28 0條評論 17次閱讀 0人點贊

智道11月28日訊 本輪歐冠，薩爾茨堡紅牛客場4-1完勝亨克，19歲小將哈蘭德傳射建功，球隊歐冠小組賽的積分上升至7分，只要末輪主場取勝10分的利物浦就可能獲得小組出線。薩爾茨堡紅牛之前在小組賽中客場3-4不敵利物浦，末輪再戰將成兩隊歐冠出線的生死戰。哈蘭德本賽季進球如麻，在5場歐冠（3次首發）比賽中狀態火熱，已經打入8個進球，僅此於萊萬的10球位列歐冠射手榜第二。

來自《米蘭體育報》記者尼科洛-斯基拉最新消息：曼聯和尤文都對簽下哈蘭德非常感興趣，他的「吸血鬼」經紀人拉伊奧拉已經開始運作挪威小將在明夏的轉會事宜。

該記者表示：「目前曼聯是最有能簽下哈蘭德的俱樂部，因為紅魔主帥索爾斯克亞是哈蘭德在挪威聯賽踢球的恩師，曼聯希望冬窗就提前簽下他來彌補失去中鋒盧卡庫的損失，並回租給薩爾茨堡紅牛半個賽季，明夏再正式加盟曼聯。但哈蘭德的父親（阿爾夫-因格·哈蘭德）是前曼城和利茲聯球員，這兩支球隊都是曼聯的死敵，同時加盟曼聯意味沒有歐冠聯賽可踢，因為曼聯目前13輪英超戰罷僅積17分排名第9，已經落後第四名的切爾西9分，本賽季爭四前景非常不樂觀，這會影響哈蘭德的選擇，從歐冠球隊到歐聯杯球隊，會讓球員萌生失落感。」

「除了曼聯之外，尤文也想得到哈蘭德，C羅競技狀態隨著年紀增大近期有點啞火，伊瓜因也是31歲的老將，尤文需要新的超級前鋒，19歲的哈蘭德值得期待，這是一筆可用十年的鋒線引援。今夏加盟尤文的中衛德里赫特與哈蘭德擁有相同的經紀人，那就是拉伊奧拉。」
