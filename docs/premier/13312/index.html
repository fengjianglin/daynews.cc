多線作戰不言倦，克洛普：我們依然瘋狂地雄心勃勃

2019-12-10 2,357條評論 4,852次閱讀 2780人點贊

尤爾根·克洛普堅持認為讓他的利物浦球員保持動力不是問題，因為他們在多條戰線追求榮譽的過程中「瘋狂地雄心勃勃」。

紅軍目前在英超積分榜上以8分的優勢領先，他們需要在周二客場對陣薩爾茨堡的比賽中再得1分，才能確保進入歐冠16強。

利物浦將在本月晚些時候參加俱樂部世界盃的比賽，同時也將進入聯賽杯八強，今年最後一個月的比賽將是密集而快速的。

克洛普敦促他的球員們一場一場地踢下去，首先是與薩爾茨堡的決戰，奧地利球隊需要三分才能出線。

他說：「這不是我希望過關的方式。我希望我們能踢出最好的比賽。」他在周一的賽前新聞發布會上說：「薩爾茨堡很高興上一輪他們贏了根克，我們和那不勒斯打平了，所以他們有決戰的機會。那不勒斯很高興，因為他們知道他們只需要和亨克比賽。」

「我們對此有所準備。薩爾茨堡必須戰勝利物浦。這是可能的，我喜歡這樣。我們仍然瘋狂地雄心勃勃，我們將拚命奔跑。我們來到這裡沒有想著要統治薩爾茨堡。」

「我們必須專註於下一場比賽。這很艱難，因為我們每三天踢一場。明晚是我們一生中最重要的比賽，因為沒有其他的比賽了。」

10月在安菲爾德與薩爾茨堡的比賽中，利物浦在主場4-3擊敗薩爾茨堡的比賽中失球3次，而在客隊的得分手中，年輕射手哈蘭德的進球更是令人興奮。

克洛普的隊員們在上一場對陣伯恩茅斯的比賽中完成了他們14場比賽中的第一次零失球，但是德國教練希望他的球隊能夠處理好進球如麻的哈蘭德。

「保持零失球對明天的比賽並不重要。」他說：「丟球會發生的。我認為我們防守得很好。我們在第一回合對薩爾茨堡的比賽中防守得不是很好。他們是一支令人驚訝的球隊，在反擊中取得了很多進球。我們非常了解他們的長處，我們希望能夠消除這些長處。」

「你如何防守一個好的前鋒？試著阻止球進入禁區。他真是個天才。」

利物浦將在周二進行本月的第三場比賽，在今年年底前，他們還將在兩大洲進行五場比賽。克洛普重申了對他的球員的很多要求，但是他希望他們能夠享受勝利的感覺。

「我們知道自己的日程安排。」他補充道：「兩天前我們踢了一場艱苦的比賽。一場非常艱苦的比賽。我們不期待丟分，我們不接受丟分，孩子們也一樣。」

「我們要求自己高度集中。我們正處在人生最緊張的時期，但我們會儘力去享受。毫無疑問，這很困難，但這並不能阻止我們。」
