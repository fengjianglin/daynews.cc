周一足球心水：斯帕爾VS熱那亞 維拉VS紐卡斯爾（千倍博高水）

天天要聞
3 週 ago

斯帕爾VS熱那亞

費拉拉最近兩個賽季在意甲聯賽都是勉強保級成功，本賽季開始以後，球隊的表現依然沒有太大改觀，12輪聯賽過後2勝2平8負，深陷降級區內，進攻乏力是困擾費拉拉多時的大問題，目前球隊合計才有7球入賬，除了佩塔尼亞和迪弗朗西斯科兩名前鋒外，球隊只有一名中後場球員獲得進球，本場比賽面對同處降級區的熱那亞，想要取勝難度不小，尤其是後衛托莫維奇和西奧尼克本場比賽雙雙遭到禁賽，做好防守是球隊的首要任務。

熱那亞本賽季的表現讓人失望，目前12輪聯賽過後，球隊僅以2勝3平7負的成績暫列積分榜第18位，比費拉拉多出一分，和主隊相比，熱那亞本賽季在進攻端做的還算勉強合格，庫瓦梅、皮亞蒙蒂和潘德夫幾名射手都有著一定的進球能力，相比之下球隊的防守無法讓人滿意，尤其是做客6場2平4負的同時，期間丟了17球，客場場均失球接近3個，面對實力處境和自己相差無幾的費拉拉，熱那亞想要帶走積分的話，首要任務便是改善防守，兩隊最近4次碰面，單場比賽的進球數都沒有超過2個，此戰雙方恐怕會陷入一場悶戰。
本場比賽亞指開出平手盤，即時主隊的指數處於較低的位置，兩隊都是意甲弱旅，近期狀態也都較為糟糕，作為一場勢均力敵的弱旅搶分之戰，雙方都想力爭擺脫降級區域，因此戰意方面都比較強烈，從實力方面熱那亞整體實力和名氣都稍佔優勢，不過目前的走勢卻忽視了客隊具備的優勢，無法增強客隊的信心，本場比賽看好主隊不敗。
看好：勝/平

維拉VS紐卡斯爾

阿斯頓維拉本賽季重回英超聯賽後，球隊在今年夏天花費上億歐元引援補強陣容，但是由於沒有頂級球員補進，球隊的實力並未得到質的提升，新賽季開始以後，維拉的發揮也是乏善可陳，目前12輪聯賽過後以3勝2平7負的成績暫列積分榜第17位，期間打進17球丟了20球，進攻端表現出色，前場的韋斯利、麥金、特雷澤蓋等人多點開花，但是後防線的糟糕發揮影響了球隊的拿分效率。

紐卡斯爾本賽季開局表現糟糕，球隊進入10月份後才開始漸入佳境，上輪聯賽主場擊敗伯恩茅斯後，最近5輪聯賽3勝1平1負，期間合計打進7球，和賽季初相比，進攻火力的提升是關鍵，不過仔細來看，紐卡斯爾本賽季的進球大多由中後場球員貢獻，鋒線上的若埃林通、聖馬克曼尚在適應聯賽節奏中，本場比賽客場面對火力不俗的阿斯頓維拉，紐卡斯爾不能保守，若能延續近期進攻端的良好發揮，喜鵲還是有機會全身而退。

本場比賽亞指初指開出阿斯頓維拉隊平半中低指數的讓步，對於主隊還是給予了一定的力度支持。本場比賽論經驗和狀態明顯是紐卡斯爾聯強於阿斯頓維拉，但本場比賽的指數沒有給與紐卡斯爾聯的信任，指數看好阿斯頓維拉可以終結連敗，此戰可以擊敗紐卡斯爾聯拿到勝利。本場比賽看好主隊不敗。
看好：勝/平
