C羅之後，曼聯的7號斷了傳承，這五人的總進球數不及C羅的1/3

2019-12-21 657條評論 2,385次閱讀 578人點贊

前兩天即將年滿35歲的C羅依然生猛，打進了一記不可思議的頭槌，從而引爆了世界足壇，各種各樣的P圖充斥著網路，讓人不得不服C羅的狀態和逆天的身體素質！反觀C羅的老東家曼聯，現在僅排在英超的第六，落後榜首的死對頭利物浦高達24分，爭冠就是天方夜譚，能撈到下賽季的歐戰資格估計都得謝天謝地了，相信紅魔球迷看到現在依然生猛的C羅應該會無比懷念他吧。想當年年僅18歲的C羅加盟曼聯時不過是個籍籍無名的小將，走時已成曼聯的又一個7號傳奇。C羅離隊後曼聯也有多位球員穿過7號球衣，但都沒有為紅魔作出重要貢獻，還有不少是水貨，桑切斯租借離隊後更是無人身披7號球衣。作為曼聯最傳奇的號碼，時至今日竟後繼無人，這也可以說是曼聯近幾個賽季戰績低迷的一個縮影！C羅之後，曼聯的7號斷了傳承，這五人總進球不及C羅的1/3！

邁克爾·歐文

曾經的「追風少年」歐文，離開利物浦可能是他的一個錯誤決定，去到皇馬雖然進球率高，但僅是球隊的替補，出場時間有限，僅呆了一年就重回英超了。在紐卡斯爾的4年里他飽受傷病的困擾，2009年自由轉會到了曼聯身披7號球衣，但那時的他已狀態下滑嚴重，偶爾有高光表現，但已力不從心。為紅魔出戰52場打進17球，還在曼聯拿到了職業生涯的唯一一個聯賽冠軍獎盃！

安東尼奧·瓦倫西亞

兢兢業業的瓦倫西亞在09年就加盟了曼聯，歐文12年走後弗爵爺把7號給了他，但之前表現良好的他突然就失去了靈性。那個賽季出戰40場僅打進一球，助攻也寥寥無幾，倍感壓力的他，第二個賽季果斷穿回了25號，然後狀態又回來了！

安赫爾·迪瑪利亞

加盟曼聯之前的「天使」迪瑪利亞是皇馬的主力邊鋒，在皇馬的四年表現出色，幾乎每個賽季的進球和助攻數都能上雙，那時的他正值巔峰。但來到曼聯後狀態一落千丈，當時頂著曼聯隊史最高身價的他32場比賽才進了四球，賽季末還和球隊鬧翻，一個賽季後就去了法甲！

孟菲斯·德佩

14年世界盃發揮出色的荷蘭邊鋒德佩，一時間成為了各大豪門的重點考察對象，15年夏天加盟了曼聯，本想大展拳腳的他卻在老特拉福德迷失了。在曼聯呆了兩年就去了里昂，只交出了53場九球的成績單，去了里昂後狀態反而比在曼聯更好了，現在是球隊的頭號球星。

阿萊克西斯·桑切斯

最近的一位7號就是智利球星桑切斯了，在阿森納還是球隊「大腿」的他，來到曼聯後變成了「弱雞」。諸多傷病加上超高的薪水還讓隊內失和，新賽季被租借到了國際米蘭，在曼聯出場45次只進了可伶的5球。

曼聯新賽季高開低走，英超首輪4：0大勝切爾西一度讓眾多紅魔球迷振奮，但隨著賽季繼續進行，曼聯的成績還是跟前兩個賽季沒什麼兩樣，積分還比同時期少。曼聯現如今缺少的是一位隊魂，一位能真正引領曼聯前進的球星，希望曼聯的傳奇7號能早日找到合適的人，帶領紅魔走出低谷！
