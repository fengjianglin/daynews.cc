世體：巴薩依然不放棄挖角國米利器！英媒：曼聯購第一目標遭打擊

2019-11-25 0條評論 18次閱讀 0人點贊

鏡報消息，曼聯將錯失簽下小哈蘭德的機會，因為萊比錫紅牛擁有這名鋒線新星的優先購買權。

報道稱，薩爾茨堡紅牛小將哈蘭德是索爾斯克亞在未來轉會窗中的首要目標之一，但由於薩爾茨堡紅牛與萊比錫紅牛之間的合作關係，以及球員合同中針對這家德甲球隊方面2570萬英鎊的違約金，看起來曼聯成功購人的機會已經不大。

本賽季哈蘭德已經在各項賽事中為俱樂部打進26球，而根據最近德媒的消息，萊比錫紅牛也非常渴望簽下這名挪威新星，並且相對於其他球隊，萊比錫紅牛具有先天優勢，那就是他們擁有姊妹球隊薩爾茨堡紅牛球員的優先購買權，並且他們還可以用極低的價格激活這些球員的違約金條款。

萊比錫紅牛體育總監Markus Krosche上周已經表達過對小哈蘭德的興趣，而球員的父親阿爾夫-因格-哈蘭德也表示，他建議自己的兒子應該先去德甲發展歷練，這些情況都意味著曼聯將會錯過哈蘭德，除非萊比錫紅牛在買下這名球員之後馬上以高價再轉賣給曼聯。

世界體育報消息，巴薩仍然沒有放棄從國米簽下備受好評的鋒線球星勞塔羅-馬丁內斯。

國米本賽季在聯賽中表現出色，而馬丁內斯的優異發揮也得到了各界的好評，報道稱，阿根廷人現在已經引起了很多歐洲豪門的關注，而巴薩是最近與之聯繫最密切的球隊。

消息指出，紅藍軍團的高層現在已經制定了一份4人鋒線引援名單，其中就包括了馬丁內斯，而除了這名國米球星，諾坎普方面的目標還有巴黎前鋒姆巴佩、熱刺前鋒凱恩以及曼聯球星拉什福德。

本賽季，22歲的馬丁內斯的表現非常出色，這名前鋒已經在16場各項比賽中貢獻了8球3助攻，國米去年則僅花費了2500萬歐元就從競技隊簽下了這名未來之星。

馬丁內斯肯定是巴薩的目標，但現在不會有什麼進展，賽季中國米肯定不會賣的。

蘇亞雷斯表示，他對自己在結束巴薩的生涯後前往MLS踢球持開放態度。

32歲的蘇亞雷斯近日已經和貝克漢姆的邁阿密國際聯繫在了一起，而這名神鋒在談到未來時也表示，前往美國踢球對他來講很有吸引力。

蘇亞雷斯告訴ESPN：「MLS這個聯賽在過去幾年中成長了很多，過去兩年中你甚至可以看到很多青年才俊選擇去MLS發展，主要是一些南美球員。

這個跡象表明，MLS是想要繼續成長的，他們不僅是滿足於簽下一些進入職業生涯尾期的球星，他們在尋求讓年輕球員和老球員一起在那裡發展，以讓聯賽變得更好。

我和巴薩有合同在身，並且我在這裡非常開心，但未來你永遠不知道會發生什麼，MLS是一個很有吸引力的聯賽。

可以說我的職業生涯已經長期以來在頂級賽事中取得了成功，因此接下來我尋求的是家庭的穩固以及孩子的成長，家人會跟我一起做出決定的，而美國則是一個非常理想的目的地。」

蘇神說的很清楚了，反正職業生涯也成功了，接下來考慮的是退休之後全家的生活和穩定，這個思路很清晰。

所以巴薩確實該考慮買新前鋒了。

米蘭體育報消息，那不勒斯準備在明年夏天賣掉5名球星。

那不勒斯目前為止已經陷入了一片混亂中，該隊主席德勞倫蒂斯最近在球隊表現不佳之後不讓球員們回家，而是採取懲罰性加練的行為引起了很多隊內球星的反抗，現在很多球員已經站到了德勞倫蒂斯的對立面，這讓俱樂部內部的氣氛日趨緊張。

報道稱，為了撥亂反正，德勞倫蒂斯現在已經制定了包含5人的清洗名單，他們分別是卡列洪、默滕斯、因西涅、阿蘭以及庫利巴利，據悉此5人要為最近的球員反抗負責，因此只要收到合適的報價，那不勒斯明年就會賣人。

其中，卡列洪和默滕斯的合同明年夏天就要到期，因此兩人在冬窗就可能被出售。

鏡報消息，多特蒙德有意擊敗拜仁，簽下熱刺新星特羅伊-帕洛特。

報道稱，多特蒙德已經從之前收購桑喬的交易中嘗到甜頭，這一次他們打算故技重施從英格蘭挖走另一名希望之星，那就是17歲的熱刺神童帕洛特，而拜仁、尤文以及皇馬也早已盯上了這名新星。

帕洛特本賽季在熱刺對科爾切斯特的卡拉寶杯比賽中迎來了一線隊首秀，此外這名小將還在本月愛爾蘭2-1戰勝紐西蘭的比賽中完成了國家隊首秀，並且奉獻了1個助攻，而今年2月份時，帕洛特才剛剛和熱刺簽下第一份職業合同，並且該合同2021年到期。

根據目前的消息，很多歐洲大俱樂部都希望能利用帕洛特的合同情況搶購這名新星，不過目前還不清楚熱刺新帥穆里尼奧對這名前鋒的態度以及計劃是什麼。

在接受YouTube頻道Qué partidazo採訪時，馬塞洛暗示自己不確定在皇馬的未來會如何，看起來這名左後衛不排除在未來離隊的可能。

馬塞洛最近幾個轉會窗曾和尤文密切聯繫在一起，而在接受Qué partidazo採訪時，31歲的巴西人拒絕承諾他的長期未來依然在皇馬，這引起了很多人的遐想。

馬塞洛說：「我比較傾向於只考慮短期的計劃，而現在我效力於皇馬。不過我沒有考慮過未來我會做什麼，現在距離我職業生涯的最後階段還有很長一段路要走，我們走著瞧吧。」

其實也有很多媒體解讀的是，馬塞洛說未來堅守在皇馬，至於他說的沒考慮過未來怎麼做，似乎是說的退役之後……

反正馬塞洛其實一直都是對皇馬錶忠心的，估計一時半會不會鬧離隊的。

皇馬前鋒馬里亞諾的經紀人近日否認了他的客戶將加盟羅馬的傳聞，不過他也承認馬里亞諾預計將在冬窗外租離開皇馬。

在接受Laroma24.it採訪時，馬里亞諾的經紀人大衛-阿蘭達表示：「馬里亞諾將外租兩個賽季，不過我們現在還沒有決定去哪個俱樂部。」

本賽季，25歲的馬里亞諾沒有代表皇馬出戰1場比賽，並且只有1次入選了比賽日陣容，據悉，球員的高薪是阻礙其尋找下家的因素之一，這名前鋒的年薪高達400萬歐元。

馬里亞諾明顯沒有機會了，不走留著對各方都不好，冬窗外租吧，或者有合適的報價，皇馬就賣了。

這個年齡沒有比賽時間，職業生涯就廢了……
