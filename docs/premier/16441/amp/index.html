轉會通：馬競降價曼聯獵物 旺達阻撓巴黎買斷二弟

天天要聞
3 天 ago

巴薩

西班牙媒體《世界體育報》消息，巴薩正在關注他們的舊將蒂亞戈，這位西班牙中場可能會離開拜仁。在弗利克接替科瓦奇成為拜仁主帥後，蒂亞戈和托馬斯-穆勒都被認為將離開拜仁，其中最主要的是蒂亞戈，他得不到拜仁新帥的重用。據悉，蒂亞戈與拜仁的合同2021年夏天到期，明夏進入合同年。

馬競

英國媒體《每日電訊報》消息，馬競可能降低薩烏爾的要價，前提是西班牙球星希望離隊。目前，薩烏爾與馬競的合同2026年夏天到期，合同違約金1.25億英鎊，如果西班牙球星想離開，那麼床單軍團可能降低他的標價，僅為8500萬英鎊。據悉，切爾西和曼聯都對馬競中場感興趣。

曼城

英國媒體《每日郵報》消息，曼城考慮簽回昔日青訓球星桑喬。桑喬在2017年8月離開曼城加盟多特，隨後這名英格蘭天才在大黃蜂爆發，職業生涯軌跡如火箭般上竄，一飛衝天，目前是世界足壇最好的邊鋒之一。本賽季，桑喬代表多特出場22次，打進11球助攻10次，他的身價已經超過1億英鎊。不過，曼城如果想簽下桑喬，需要面臨著利物浦、曼城和切爾西等豪門的競爭。

國米

義大利媒體《都靈體育報》消息，國米有意簽下阿隆索，切爾西為其估價超過3000萬歐元。據悉，國米有意在冬窗對邊路進行補充，他們有意簽下切爾西左後衛阿隆索，他與孔蒂曾在切爾西合作過，一起拿到英超冠軍。本賽季，阿隆索為切爾西在各項賽事中出場13次，打進1球助攻2次。

米蘭

義大利媒體《全市場》消息，米蘭雙雄考慮冬窗期間租借引進皇馬前鋒約維奇。今夏，約維奇以6000萬歐元的價格從法蘭克福轉會至皇馬，來到伯納烏後，他遲遲無法融進球隊，本賽季至今，約維奇僅打進可憐1球。目前，約維奇在皇馬已經成為絕對替補，在中鋒的位置上，出場次序排在本澤馬之後。

拜仁

德國媒體《圖片報》消息，儘管庫蒂尼奧的買斷條款是1.2億歐，拜仁和巴薩很可能商量出一個較低的價格。不過，報道同時指出，拜仁現在不急著決定是否買斷庫蒂尼奧，他們會慢慢來。早些時候剛剛一場德甲賽事，庫蒂尼奧打進3球助攻2次，幫助拜仁主場6-1瘋狂逆轉不來梅。

巴黎

義大利媒體《都靈體育報》消息，巴黎對本賽季伊卡爾迪的表現很滿意，他們希望在一月份以7000萬歐元的價格將他買斷，但是他的妻子旺達更希望伊卡爾迪加盟皇馬或尤文。今夏，伊卡爾迪租借加盟巴黎後，年薪從550萬歐元飆到1000萬歐元。本賽季，伊卡爾迪15場比賽打進12球。
