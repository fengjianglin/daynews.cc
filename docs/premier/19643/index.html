群雄紛爭！齊祖瓜帥光頭對決 巴薩熱刺抽中禮包 拜仁藍軍冤家聚首

2019-12-17 1,725條評論 4,111次閱讀 54人點贊

北京時間12月16日晚，19/20賽季歐冠16強抽籤結果出爐，皇馬與曼城PK，巴薩戰那不勒斯，切爾西和萊比錫紅牛，拜仁與切爾西等將進行八強戰的較量，隨著淘汰賽來臨，各大豪門將進入真正的對決狀態，歐冠將真正刺激起來。

1、皇馬vs曼城 光頭對決誰將計勝一籌

如果看瓜迪奧拉執教生涯對陣皇馬的戰績，是9勝4平4負，可謂是佔據巨大優勢，當然這些優勢多是發生在瓜帥執教巴薩時期，08/09賽季更是在伯納烏6-2血洗皇馬。不過在離開巴薩後，瓜帥對皇馬的戰績並不理想，執教拜仁時期唯一一次與皇馬歐冠交手就被打成了「五仁」，15/16賽季歐冠半賽季中執教曼城又以0-1總比分被皇馬淘汰，而在這2次目送晉級後，皇馬均戰勝同城死敵馬競摘得歐冠冠軍。

當然本賽季皇馬與曼城聯賽境遇都有點像，只不過皇馬經歷賽季初的低迷後，如今依舊在爭冠之列，而曼城落後利物浦14分，爭冠希望已不大了，因此歐冠定當成為瓜帥看重的榮譽戰場了，而雙方對決，也是技術足球之間的巔峰戰，瓜帥球隊向來強調傳控足球，皇馬則以簡練直接的踢法為重，誰將更勝一籌？

2、熱刺vs萊比錫紅牛 2成概率抽中上上籤 好運連連熱刺8強可期！

在出線的8個小組第1中，無疑萊比錫紅牛是實力相對最弱、大家最想抽中的球隊，但對切爾西來說，能夠抽中如此大禮包不得不說運氣成分更多，畢竟在同組以及同國迴避原則下，熱刺潛在對手包括巴薩、尤文、萊比錫紅牛、巴黎和巴倫西亞，而當第7個抽出的小組第2是熱刺時，剩下的小組第1對手就只剩巴薩和萊比錫了。

而最終抽出的是萊比錫，畢竟五五開的概率，能夠避開巴薩，還是有一定運氣成分，而以歐足聯慣常手段來看，切爾西與巴薩的相遇會比切爾西與萊比錫的相遇更具熱度，穆帥迎戰巴薩，這樣的對決會成為無數媒體追逐的焦點，好在最終穆帥還是幸運抽中上上籤，但德甲榜首萊比錫除了歐戰經驗不足外，相信並不乏戰鬥力，穆帥能否將其制服還未可知？

3、那不勒斯vs巴薩 抽中內訌隊 巴薩輕鬆笑納大禮包？

在熱刺確定對陣萊比錫後，巴薩也就確定了對手那不勒斯，與熱刺一樣，巴薩同樣是抽中了大禮包，相對其他球隊來看，那不勒斯應該是巴薩最願意抽到的球隊了，熱刺方面，穆帥與巴薩太過了解，亞特蘭大則是意甲火力最強球隊，對目前防守不算穩固的巴薩來說，有一定風險。

那不勒斯來說，此前一度是內訌不止，主席與球員關係不和，而後近來安切洛蒂下課，新帥加圖索首秀就遭開門黑，綜合看，巴薩面對那不勒斯晉級難度不大。

4、切爾西vs拜仁 冤家聚首 2012歐冠噩夢重現or拜仁完美復仇？

1/8對決中，切爾西與拜仁的較量應該更能引發球迷的感慨，兩隊上一次雙方在歐冠的對決就是在2012年5月20日的歐冠決賽，120分鐘雙方戰成1-1，點球大戰中切爾西成功擊敗拜仁奪得隊史首個歐冠冠軍。值得一提的是，本次抽籤對陣出爐後，藍軍官推就發文「我們又見面了」，而拜仁則俏皮的發了一張圖，圖中則是拜仁將「2012」和「德羅巴」2個關鍵詞屏蔽在外。

畢竟那一年，全場壓制的拜仁與冠軍獎盃近在咫尺，這時候德羅巴挺身而出，先是絕殺扳平將給了拜仁當頭一棒，後又在點球大戰中壓軸出場一錘定音徹底讓拜仁夢碎；7年後再度相遇，是噩夢重演還是完美復仇，而藍軍將士中誰又將重現魔獸式神跡表現？
