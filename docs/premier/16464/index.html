意甲今日頭版：米蘭120歲生日快樂 加圖索今晚那不勒斯首秀

2019-12-14 2,418條評論 3,703次閱讀 318人點贊

直播吧12月14日訊 米蘭體育報、羅馬體育報、都靈體育報是義大利的三大體育報，今天我們來看看它們各自的頭版頭條都帶來了哪些內容。

《米蘭體育報》

1.「米蘭120歲生日快樂。」米體提前為紅黑軍團送上生日祝福。1899-2019，曾經征服世界，現在渴望回到巔峰。跟里維拉、舍瓦和薩基一起，我們重溫屬於紅黑色的歐洲。明天在聖西羅慶祝。

2.加圖索那不勒斯首秀，在聖保羅迎戰帕爾馬。加圖索派出因西涅領銜的三叉戟。

3.國米，盧卡庫準備重新衝刺。尤文，迪巴拉挑戰好友德保羅。

4.卡利亞里傳奇里瓦成為俱樂部榮譽主席。

《羅馬體育報》

1.「里諾的那不勒斯。」加圖索今晚首秀，對陣帕爾馬，雙方積分相同。比賽前一天那不勒斯仍在集訓，他們想追回8分回到歐冠區，這是任務：很困難，但不是不可能。因此他們要有勇氣和清晰的思路：從4-3-3開始。

2.「國米和尤文的80天。」距離3月1日雙方在都靈的直接交鋒還有80天，不確定的傷病、轉會市場和杯賽。阿爾托貝利：「雙方會競爭到最後，不過斑馬軍團仍是熱門。」

3.旗幟，里瓦成為卡利亞里俱樂部榮譽主席

4.莫塔-拉涅利，令人害怕的德比。熱那亞和桑普在降級區的交鋒。

5.克洛普與紅軍的愛沒有結束。教練簽下新約，合同至2024年。

《都靈體育報》

1.「尤文還是國米？這就是小基耶薩會去的地方。」前紫百合總監科爾維諾接受本報專訪，他透露了紫百合前鋒的未來，小基耶薩一直是國米和尤文的目標。「把紫百合賣給科米索？這是德拉瓦萊德愛心行動。」

2.伊卡爾迪事件：巴黎準備立馬買斷，國米能進賬7000萬歐，但旺達踩剎車。

（我叫達芬奇）
