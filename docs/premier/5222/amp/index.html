馬爾蒂尼和卡洛斯，誰才是歷史最佳左後衛？C羅：馬爾蒂尼更強

天天要聞
2 週 ago

跟籃球比賽同樣精彩的莫過於足球比賽了，作為世界上最受歡迎的體育運動之一，足球運動也吸引無數球迷為之瘋狂，這從每一屆的世界盃中就可以看得出來。那麼在足球隊中，每個位置的球員都至關重要，唯有默契的配合才能創造更好的成績，在足球歷史中，從馬爾蒂尼和卡洛斯選擇一個球員，那誰才是歷史最佳左後衛？作為當代足壇頂尖球員的C羅給出了他的看法：馬爾蒂尼更強！

馬爾蒂尼的輝煌成就

C羅是現代最具有影響力的足球運動員之一，他有超凡脫俗的實力，也有出色的身體素質，即使33歲了，仍然可以拿到豪門足球俱樂部尤文圖斯的天價合同。而他認為馬爾蒂尼是就開始最強左後衛也是有他的原因的，1963年出生的馬爾蒂尼是一名義大利足球巨星，所有的球員時光都在AC米蘭足球俱樂部度過的，是一位相當忠實的球員。

因為實力突出，馬爾蒂尼成為了義大利國家隊和AC米蘭的雙料隊長，他的職業生涯中獲得了相當多的成就，其中就有7次義大利足球甲級聯賽冠軍、5次歐洲冠軍聯賽冠軍和5次歐洲超級盃冠軍。技術一流，人品一流的馬爾蒂尼獲得了眾多肯定，也是當之無愧的足球巨星，最無與倫比的球隊左後衛。

卡洛斯的輝煌成就

而卡洛斯也是足球歷史上的優秀左後衛，雖然他更年輕，出生於1973年，但是他的球員經歷相當豐富，不僅再在多個俱樂部都效力過，職業生涯中還出場過1127場比賽，非常強悍。小時候家庭並不富裕的卡洛斯成年後的身高也不高，只有1米68，但好在球場上的他速度快，反應能力比其他球員要優秀的多，而且大腿力量很強。作為球隊左後衛的卡洛斯在職業生涯中共獲得過3次歐洲冠軍聯賽冠軍，4次西甲聯賽冠軍和一次世界盃冠軍，2012年就選擇了退役。

二者不相上下

對於馬爾蒂尼和卡洛斯，值得肯定的是他們都是世界足球歷史上優秀的球員，從數據上相比來看，馬爾蒂尼先天條件更出色，擁有無窮的創造力，而且成就更多，會是最佳左後衛。但卡洛斯不屈服的踢球風格也帶動了球場上的激情，一定程度上推動了足球的發展，也用自己的事迹激勵了很多球迷。
