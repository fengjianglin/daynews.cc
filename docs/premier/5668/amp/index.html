放眼未來！利物浦盯上挪威21歲中場，欲將其帶至安菲爾德球場

天天要聞
2019-11-15

本賽季利物浦在聯賽中表現不俗，這支英超豪門正在逐步接近隊史首座英超冠軍。此前一直飽受詬病的利物浦中場，本賽季這些球員的表現也是可圈可點，加盟球隊第二個賽季的法比尼奧越踢越好，巴西人在克洛普的精心栽培下，如今已經步入英超頂級後腰行列。另外亨德森、維納爾杜姆兩人在近幾個賽季也是漲了不少球，此前一直處於養傷狀態的張伯倫在復出後更是有如神助，這位英格蘭國腳連續奉獻精彩遠射，張伯倫的強勢回歸讓克洛普在排兵布陣上顯得遊刃有餘。

本賽季利物浦的中場組合堪稱是近幾年最穩定的一個賽季，但作為一支豪門球隊，自然要為球隊的長遠發展做打算。根據足球市場消息報道，利物浦對比利時球隊亨克的21歲中場桑德爾-博格有著濃厚的興趣，克洛普希望能夠將這名年輕球員帶至安菲爾德，為球隊未來儲備人才！目前亨克對21歲的桑德爾標價2500萬歐元，這個價位對利物浦來說是非常實惠的了，如果他能來到英超賽場，善於栽培年輕球員的克洛普將會使其更上一層樓。

博格在2017年以200萬歐元從挪威球隊瓦勒倫加轉會至亨克，加盟亨克後，博格從板凳球員成長為首發球員，上賽季幫助球隊獲得比甲聯賽冠軍。本賽季這位21歲的中場已是球隊的絕對主力，本賽季到目前為止博格代表亨克出戰19場比賽打進2球，表現不俗！雖然在歐冠小組賽中，亨克與利物浦的兩次交手中都以失敗告終，但這名1米93的中場球員卻給克洛普留下了深刻的印象，身體十分硬朗、腳下技術嫻熟，作為中場球員博格擁有出色的大局觀意識，這些都符合克洛普的選人標準！

雖然桑德爾-博格只是效力比甲聯賽，但本賽季在歐冠比賽中，博格的出色表現還是吸引了同組對手，據悉意甲勁旅那不勒斯也對這名年輕中場感興趣，義大利媒體透露，那不勒斯對博格很感興趣，有可能達成協議，但那不勒斯方面並不急於在明年夏天就將其簽下，而是希望先與球員達成口頭協議。目前義大利媒體認為那不勒斯最有可能得到桑德爾-博格，因為博格選擇那不勒斯將會獲得更多的出場機會！

離轉會窗口還有一段時間，在這之前誰都無法下結論，利物浦面臨著那不勒斯的競爭，但很顯然作為歐冠冠軍的利物浦更加有優勢，如果博格來到安菲爾德雖然前期出場時間少，但是在這樣的冠軍球隊中，博格能夠很快的提升自身能力。歡迎在評論區留言討論。
