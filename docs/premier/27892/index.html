22日競彩大勢：倫敦德比穆帥戰舊主 皇馬馴服雄獅

2019-12-22 515條評論 1,891次閱讀 638人點贊

北京時間2019年12月22日周日競彩開24場比賽。早場意甲，亞特蘭大坐鎮主場對陣AC米蘭，盤路走勢傾向主勝；英超倫敦德比，穆里尼奧率隊熱刺戰舊主切爾西，昔日弟子蘭帕德的球隊還在為年輕交學費，熱刺不敗；西甲，皇馬連續不敗場次已經超過10場，畢爾巴鄂客場表現平淡，難以抵擋皇馬的進攻。今天的競彩大勢帶來以下三場比賽前瞻：

周日004 意甲 亞特蘭大vsAC米蘭

亞特蘭大上輪客場1-2不敵博洛尼亞，歐冠神奇出線後有所鬆懈。戰罷的7個意甲主場3勝1平3負得失球18:14戰績不及客場出色。此役他們只有主力前鋒薩帕塔（7輪6球）一個老傷號。

AC米蘭上輪主場0-0被薩索洛逼平，紅黑軍團連續4輪不敗換帥後表現開始有所起色。8個意甲客場4勝4負得失球9:9，此役主力左邊衛蒂奧（13輪4球1助）、前鋒帕奎塔（13輪1助）均累積5黃停賽。

戰罷的16輪意甲兩隊場劇數據對比：射門次數20:15，控球率54%：54%，傳球成功率82%：84%，亞特蘭大射門優勢明顯，亞指初盤開主讓半球，受注後調整為主讓半一，競彩投注建議主勝做膽。

周日019 英超 熱刺vs切爾西

倫敦德比，穆里尼奧又一次在一個新東家率隊戰舊主，對手是他此前「二進宮」的切爾西，同時藍軍教練蘭帕德又是魔力鳥昔日帶隊的得意弟子。

熱刺上周末做客最後時刻絕殺狼隊，邊鋒小盧卡斯在穆里尼奧上任後打入3球，此前聯賽他只有1球入賬，近期一直客串左後衛的費爾通亨破門。穆里尼奧執掌熱刺之後帶隊的5輪英超4勝1負，球隊排名也上升到了積分榜第5。不過，葡萄牙人帶隊的兩場強強對話都輸球了，分別是英超客場1-2不敵曼聯，歐冠小組賽客場1-3負於拜仁。左後衛本·戴維斯和前鋒拉梅拉接近康復，腹股溝傷勢還在恢復中的中場恩東貝萊此戰不會冒險復出。

切爾西上周末主場爆冷0比1輸給伯恩茅斯，近5輪英超輸了4場，蘭帕德的球隊還在為年輕經驗不足而交學費。因傷連續缺席過去兩場比賽的中衛托莫里此戰有望復出，藍軍陣容齊整。8個英超客場5勝3負得失球20:17，客場的切爾西比較重攻輕守。

戰罷的17輪英超兩隊場均數據對比：射門次數12:17，控球率52%：58%，傳球成功率81%：85%，切爾西傳控射佔優，但是亞指強開主讓平半，競彩投注建議熱刺不敗。

周日024 西甲 皇馬vs畢爾巴鄂

皇家馬德里周中的西甲補賽中做客諾坎普0比0戰平巴薩，全場皇馬防守表現出色，進攻在下半場反擊也頗有威脅，頑強守得1分，積分榜上依舊跟巴薩並駕齊驅領跑。戰罷的8個西甲主場6勝2平得失球20:6保持不敗。主力後腰卡塞米羅（16輪1球1助）累積黃牌停賽，馬塞洛、阿扎爾、J·羅德里格斯、阿森西奧、盧卡斯·巴斯克斯都繼續在傷兵營中。齊達內的球隊目前連續不敗的場次超過10場，狀態穩定。

畢爾巴鄂競技上輪0比0戰平埃瓦爾，全場球隊佔據優勢但是遭遇對手頑強拼搶，主教練蓋斯卡·加里塔諾批評自己的球隊「連續傳球難以超過3次」。周中他們打了國王杯，客場3-0完勝一支低級別球隊晉級。後衛安德爾·卡帕累積黃牌停賽，前鋒阿杜里斯傷愈復出，中場穆尼亞因、後衛德馬科斯繼續養傷。球隊本賽季27個積分有20分來自主場，做客1勝4平3負得失球6:9隻擊敗過奧薩蘇納。

戰罷的17輪西甲兩隊場均數據對比：射門次數17:11，控球率56%：48%，傳球成功率87%：75%，皇馬傳控射優勢明顯，亞指初盤開主讓一球球半，受注後升盤球半，競彩投注建議主勝做膽，博皇馬讓一球勝。
