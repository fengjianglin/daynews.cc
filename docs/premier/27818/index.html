1-3！又一場1-3！尤文15天被雙殺，34歲C羅今夜終止2神跡

2019-12-23 1,166條評論 2,438次閱讀 1221人點贊

北京時間12月23日00:45，義大利超級盃決賽，迪巴拉破門難救主，阿爾貝托、盧利奇和卡達迪各進一球，拉齊奧3-1戰勝尤文奪冠。

在12月8日意甲第15輪，尤文剛在客場1-3輸給拉齊奧；在15天後，尤文被拉齊奧以同樣的比分擊敗。

這是一場不小的冷門，在開賽前尤文一致被看好，他們爆冷輸球的可能性只有17%！

對於34歲的C羅來說，他今夜2大神跡被終止：此前他連續9年進球40+，連續12次決賽均奪冠。

這是C羅2019年最後一場比賽，尤文下一場將是1月6日主場對陣卡利亞里的意甲比賽。這樣，C羅2019年的進球紀錄定格在39球，C羅單一自然年進球數（2010-2019年）：48球、60球、63球、69球、61球、57球、55球、53球、49球、39球。

第16分鐘，盧利奇左路傳中，米林科維奇回做，阿爾貝托推射破門，拉齊奧1-0；

上半場結束前，C羅遠射導致對方門將脫手，迪巴拉跟進打空門得手，尤文扳平1-1；

第72分鐘，拉扎里右路傳中，盧利奇在後點凌空抽射破門，拉齊奧2-1；

在第94分鐘，卡達迪自由球直接破門，拉齊奧3-1鎖定勝局。
