歐冠：多特蒙德想出線，需要戰勝本場對手，還要祈求國米不勝

2019-12-10 836條評論 1,990次閱讀 1043人點贊

歐冠2019-12-11 04:00 星期三，多特蒙德VS布拉格斯拉維亞。

多特蒙德

本周中，歐冠小組賽將會迎來最後一輪的較量，其中對於目前仍在生死邊緣線上掙扎的多特蒙德而言，他們需要打醒十二分精神作最後的奮力一搏，畢竟目前他們由於勝負關係處於劣勢，被迫暫時屈居國米之後，排名小組第二，已經喪失了出線主動權。

最後一輪比賽他們除了要戰勝對手外，還要祈求國米不勝，條件缺一不可，好在他們最後一輪的對手是小組最弱的布拉格斯拉維亞，鑒於首回合他們在客場也能2-0輕鬆拿下對手。

如今回到主場，並且在上周末剛剛收穫了一場5-0大勝士氣正盛的情況下，相信他們只要正常發揮的話，基本上是三分穩拿，因此就目前而言，多特做好自己基本不成問題，唯一就看國米能不能從巴薩身上全取三分了。

布拉格斯拉維亞

事實上，這場比賽其實懸念並不大，畢竟布拉格斯拉維亞就是一支陪太子讀書的球隊，且目前已經提前出局，就連爭三都已經無望，試問他們還有什麼戰意可言？

按照實力對比，此役布拉格斯拉維亞很可能只是替補出戰，因此多特蒙德只要不大意的話，全取三分是基本不成問題的。
