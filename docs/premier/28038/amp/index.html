梅西奪獎停不下來！巴薩1.5億巨星再贏范迪克，C羅成第4

天天要聞
2019-12-20

北京時間12月20日，英國權威媒體《衛報》評選了足壇的百大巨星，該媒體在今天揭曉了排名前10位的巨星，梅西名列榜首，緊隨其後的是范迪克和馬內，而C羅在評選中遭到無視，僅僅排名第4位，利物浦成為評選的贏家，在排名前10位里，有4人效力歐冠冠軍，而克洛普的球隊即將衝擊世俱杯冠軍。

前不久，梅西剛剛拿到了金球獎的榮譽，這是他第六次獲得這份殊榮，這一次，《衛報》評選的足壇百大巨星，梅西又是名列第一位，此前，梅西還獲得了世界足球先生的殊榮，可以說，梅西在2019年度迎來了豐收年，不出意外的話，梅西還會獲得很多媒體評選的2019年百大巨星第一名。

值得一提的是，《衛報》是英國媒體，但是在評選百大巨星的時候，該媒體並沒有將英超豪門利物浦的當家球星范迪克列為第1，反而將梅西放在了榜首，這讓梅西在今年的成就更有說服力。

當然范迪克在2019年度的發揮也是非常出色，他幫助利物浦拿到了歐冠和歐洲超級盃的冠軍，接下來，利物浦還有希望贏得世俱杯冠軍，可以說，范迪克功不可沒，此前他也拿到了歐足聯賽季最佳球員的殊榮。在評選中，利物浦的馬內排名第3位，這可並不容易，因為馬內力壓的可是尤文圖斯球星C羅，後者在這次評選中僅僅拿到第4名。

在金球獎的評選中，C羅還能名列三甲，但是在《衛報》的百大巨星評選中，C羅只能排名第4，不得不說，這是一份C羅不喜歡的排名。排名第5位的是利物浦的另外一位巨星薩拉赫，埃及球星在2019年度的表現也是可圈可點。

在這次評選中，排名第6-10位的巨星分別是巴黎聖日耳曼的姆巴佩、拜仁慕尼黑的萊萬、曼城的斯特林、利物浦的阿利森和巴薩的德容。在前10名里，有4名球員來自利物浦，這也從側面印證了利物浦的成功。

《衛報》的評選雖然存在爭議，但是這次評選年度百大巨星，英媒還是邀請了來自69個國家和地區的225名足球專家來擔任評委，其中還有34名是現役或者已經退役的知名球員，因此，這樣的評選，還是很有說服力。

附《衛報》百大巨星前10：

2：范迪克（利物浦）

3：馬內（利物浦）

4：C羅（尤文圖斯）

5：薩拉赫（利物浦）

6：姆巴佩（巴黎）

7：萊萬（拜仁）

8：斯特林（曼城）

9：阿利森（利物浦）

10：弗蘭基-德容（巴薩）
