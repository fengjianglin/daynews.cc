超級比賽日｜國米主場求勝穩固榜首，馬競客戰拿分衝擊前四

天天要聞
2019-12-06

12.7日（周六），五大聯賽的早場比賽，將迎來2場焦點大戰:意甲榜首球隊國米主場迎戰強敵羅馬，事關榜首位置;西甲排名第6的馬競做客比雷亞雷爾，能否取勝事關前4位置。

法甲:里爾不敗，衝擊三甲

里爾對陣布雷斯特，里爾排名第4，布雷斯特排名第12，雙方最近5次交手，里爾4勝1負佔據優勢。里爾最近5場比賽2勝1平2負（巴黎、歐冠阿賈克斯）狀態尚可，布萊頓最近5場比賽1勝1平3負（巴黎）狀態差。

兩隊排名雖然差了8個排位，但分差也只是4分，結合最近的狀態，里爾主場贏球問題不大，起碼不會輸球，否則排名會掉2-5;而布雷斯特上輪獲勝之後結束了5輪不勝的尷尬，本輪客場作戰也有機會拿分。看好里爾主場不敗。

德甲:法蘭克福取勝，止敗

法蘭克福對陣柏林赫塔，法蘭克福排名第10，柏林赫塔排名第16，雙方最近5次交手，阿森納1勝1平3負處於下風。法蘭克福最近5場比賽1勝（歐聯阿森納）4負狀態差，柏林赫塔最近5場比賽1平4負狀態極差。

法蘭克福在5-1戰勝拜仁後，聯賽遭遇3連敗，排名也是大幅下滑，急需一場勝利;而柏林赫塔更是各項賽事7場不勝（聯賽4連敗），目前深陷降級區。看好法蘭克福主場取勝。

意甲:國米渴望勝利，坐穩榜首

國際米蘭對陣羅馬，國際米蘭排名第1，羅馬排名第5，雙方最近5次交手，國際米蘭1勝3平1負雙方打平。國際米蘭最近5場比賽4勝1負（歐冠多特）狀態正佳，羅馬最近5場比賽3勝2負（歐聯門興）狀態尚可。

國米最近狀態正佳，聯賽5連勝，在與尤文的較量中，沒有再掉鏈子，上輪聯賽由於尤文意外打平，更是趁機反超積分升至第一，本輪主場面對羅馬，一定會全力以赴的戰勝對手、從而穩固榜首位置;而羅馬最近狀態也不錯，一旦客場取勝就有機會躋身前四。看好雙方分勝負。

西甲:馬競客戰，不容有失

比雷亞雷爾對陣馬德里競技，比雷亞雷爾排名第13，馬競排名第6，雙方最近5次交手，比雷亞雷爾2勝2平1負佔據優勢。比雷亞雷爾最近5場比賽1平4負狀態極差，馬競最近5場比賽1勝1平3負（巴薩、歐冠尤文勒沃庫森）狀態差。

馬競上輪主場0-1負於巴薩後，最近5場聯賽1勝3平1負，丟了至少4分，排名也掉出了前四，本輪做客面對狀態極差的比雷亞雷爾有望拿分，至於能否取勝就看臨場發揮了;而黃色潛水艇最近遭遇了5場不勝（4場失利）的困境，考慮到下輪客場打塞維利亞，本輪主場要是還不能拿分，就直逼降級區了。看好馬競客場拿分。
