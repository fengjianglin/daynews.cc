結束的雙驕？C羅：有空去英超和意甲嗎？

2019-12-03 0條評論 16次閱讀 0人點贊

梅羅—-惺惺相惜！

在剛剛結束的2019年法國《法國足球》評選出的金球獎頒獎典禮上，不出意外的由來自西班牙巴塞羅那俱樂部的萊奧 梅西獲得，這也是梅西獲得的第6個金球獎，創造出了金球獎創辦以來的一個偉大記錄，的確，在過去的一年中，梅西個人的表現一如既往的優秀，34場西甲比賽，36個進球加13次助攻，雙雙都是西甲第一；在歐冠比賽中，10場比賽12個進球，帶領巴薩進入歐冠4強。其個人表現可以說是無可挑剔。獲獎也是實至名歸。

梅西6奪金球獎！

在梅西獲得今年金球獎的時刻，不能不提到一個人，那就是做為梅西一生的對手的C羅，在本次評選中，C羅獲得了第三，新生代頂級後衛范戴克歷史性的獲得了第2名，取代了之前長達10年之久的梅羅「二人轉」的足壇獲獎記錄，在網上看到很多評論說，以此次的評選為轉折，C羅實力不如梅西，已經滑落下去了。

在這裡，筆者想要說的是，不管出於自然規律，還是年齡增長，自身傷病等因素，一個人的狀態肯定是會從高處到低處慢慢滑落的，這點無可厚非，任何人都逃不脫這個自然規律。

但是，筆者卻不敢這麼苟同，想想C羅的經歷，輾轉三大頂級聯賽，英超，西甲，意甲，即插即用，持續高光，其超強的適應能力，無人可出其右。

C羅獲得2018-2019賽季意甲最佳球員獎！

首先，2008年就在英超曼聯獲得了英超冠軍，英超最佳球員，最佳射手等頂級榮譽，以及歐冠冠軍，

接著，2009年開始轉會西甲皇馬，10年間同樣是西甲聯賽冠軍，最佳球員，最佳射手，歐冠冠軍加身。場均1球以上的進球率，皇馬歷史上的進球王。

最後，2018年來到意甲尤文圖斯，首年就帶領尤文圖斯獲得意甲冠軍，並把最佳球員收入囊中，單從這點來看，其在三個不同的聯賽里都能發揮出自身實力，迅速融進球隊，超強的適應能力和他的身體素質一樣都是令人嘆為觀止的。

試想，過往的諸多成名的巨星，又有幾人能在改變球隊，改變環境後任能充分發揮自身實力的？

想想當年的小羅，從巴薩到米蘭。。。。。。

近者，格列茲曼，從馬競到巴薩。。。。。。

C羅，能在風格差異巨大的聯賽里，拿出頂級的表現，獲得一致的認可—-奪取最佳球員，有幾人能做到？

。。。。。。本文不是貶低梅西，也不是拔高C羅，單從二人表現來看，已是雙驕，吾輩之幸！
