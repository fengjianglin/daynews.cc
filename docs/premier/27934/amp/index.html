瘋狂4-0！國米17歲新星創造個人歷史，咆哮慶祝，球迷激動落淚

天天要聞
2019-12-22

19/20賽季意甲第17輪焦點大戰，國際米蘭主場4-0大勝熱那亞，此役100秒時間內，國際米蘭連進2球殺死比賽。憑藉到手3分，國際米蘭以凈勝球優勢力壓尤文圖斯，再次成為領頭羊。一周內，意甲榜首位置已經4次易主。此外，17歲172天，國米新星埃斯波西托成為打入意甲進球第二年輕的國米球員！

本賽季意甲，國際米蘭復甦，雖然遺憾未能在歐冠中出線，但這反而有利於藍黑軍團把所有注意力集中到聯賽中。雖然國際米蘭此前主場1-2不敵尤文圖斯，但他們沒有丟隊。

今天的比賽，國際米蘭贏得很輕鬆，上半場100秒時間裡，國際米蘭就打進2球終結懸念。第30分20秒，盧卡庫頭槌首開紀錄！第32分鐘，加利亞爾迪尼破門。值得一提的是，加利亞爾迪尼的進球就是源自於盧卡庫的助攻！

下半場，17歲的國際米蘭小將埃斯波西托創造歷史，他成為了國際米蘭21世紀以來在意甲最年輕的首發球員，並且此役取得了進球。進球之後，國際米蘭球迷齊喊這位青訓小將的名字，而埃斯波西托也是瘋狂咆哮慶祝，3-0！隨後，盧卡庫破門，4-0！

一周時間裡，意甲領頭羊4次易主！上一輪比賽，尤文3-1擊敗烏迪內斯後成為領頭羊，隨後，國際米蘭1-1戰平佛羅倫薩，憑藉1分重返榜首！本輪之前比賽，尤文2-1擊敗桑普多利亞，葡萄牙飛人C羅打進逆天滯空球，尤文重返榜首！如今，國際米蘭大勝，再次以凈勝球優勢重返榜首！

因此，本賽季意甲爭冠已經白熱化，除了國際米蘭和尤文圖斯表現出色，拉齊奧也不容忽視。但至少，國際米蘭可以以領頭羊身份結束2019年！
