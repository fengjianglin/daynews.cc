換人風波前尤文已在考察“新C羅”這4人皆有望改穿黑白衫

2019-11-13 0條評論 17次閱讀 0人點贊

【章魚流言板】明年初就將年滿35歲的C羅顯然不可能成為尤文圖斯長期計劃當中的一員，而他上周末所引發的爭議又讓人們對他在斑馬軍團的前景開始感到懷疑。當然，即便沒有這次換人風波，老婦人也需要早點為沒有葡萄牙人的日子做好打算。

尤文正在著眼於尋找一位「新C羅」。

這絕不是道聽途說，也不是C羅在義大利神聖德比中引發爭議後尤文的臨時起意。在此前的俱樂部年會上，俱樂部股東大會主席保羅·艾卡迪就向與會者透露：

「C羅這位世界第一球星來到這裡，說明尤文對球星的吸引力已經不在皇馬、曼聯、巴薩等球隊之下，任何引援都有可能，簽下德里赫特再次證明了這一點。我們接下來還計劃引進『新C羅』，而且會比C羅更年輕。」

按照艾卡迪所言，老婦人早已在關注更為年輕的超巨。儘管歐洲足壇高水平球員層出不窮，但是配得上「新C羅」名號的依然鳳毛麟角，畢竟34歲的葡萄牙人本身就是一個全球化品牌。

C羅遲早有離開尤文的一天，他離隊後留下的空當應該如何填補呢？縱觀全歐洲，也只有以下4人能讓斑馬軍團在「後C羅」時代步履穩健。

內馬爾

這是一個顯而易見的答案。效力於巴薩時，內馬爾就被視為梅西的接班人。以世界第一身價轉投大巴黎的巴西人，當然也有能力接過C羅的衣缽。在今年夏天導演了這出轉會肥皂劇之後，內馬爾已將他希望離隊的心思展露無遺，但是擺在他面前的選項實在有限。

皇馬現在有阿扎爾坐鎮，還擁有2名巴西天才新星——羅德里戈和維尼修斯；巴薩則剛剛為格列茲曼砸下重金，還有2年前作為內馬爾替代者加盟球隊的登貝萊，以及拉馬西亞培養出的最新產品法蒂。

財大氣粗的曼聯倒是也能負擔得起內馬爾的轉會費，但是志在奪取金球獎的巴西人願意加盟這樣一支連歐冠資格都拿不到的球隊嗎？而在尤文圖斯，他就不會有類似的煩惱。

大巴黎並非將內馬爾視為非賣品，但前提是有球隊願意提供足以打破世界紀錄的報價。尤文陣中目前有很多球員可以作為轉會添頭，而且，就算非得砸錢不可，斑馬軍團也不是沒有實力承擔。

基利安·姆巴佩

儘管下個月才年滿21歲，但是毫無疑問，姆巴佩已經是當今足壇能力最強的巨星之一。

大部分年輕球員都會在媒體的讚譽聲中迷失方向，姆巴佩卻堪稱「異類」。從16歲開始，法國金童就被視為未來金球獎的有力爭奪者，而他也確實兌現了自己的天賦，甚至突破了樂觀者最樂觀的期待。

雖然大巴黎是他的家鄉球隊，不過這位世界冠軍成員早已公開表示過，會考慮在合適的時機轉會離開。在眾多歐洲豪門當中，曼城對他的覬覦由來已久，也是僅有的幾支能支付如此高額轉會費的球隊之一。

不過，看看伊瓜因和C羅加盟球隊時的身價和年齡，尤文為了30歲上下的球員都願意付出1億歐元左右的資金，更可況是一位有能力在未來10年獨領風騷的少年英雄呢？

勞塔羅·馬丁內斯

本賽季勞塔羅在孔蒂手下表現出色，想從意甲爭冠直接競爭對手手中搶下這位阿根廷新星，對於尤文來說顯然不會容易，但也並非不可能。

意甲北方三強之間的關係和一般同國豪門不同，歷史上有大量球星在這3傢俱樂部之間頻繁往來，巴喬、維埃里、皮爾洛等皆是傑出代表。因此，如果尤文真的想要引進勞塔羅，至少有的可談。

單從數據來看，指望勞塔羅接班C羅似乎有些離譜，畢竟阿根廷人截至目前只為國際米蘭打進17球，代表潘帕斯雄鷹出場15次，很多人可能都不知道這位前阿根廷競技隊的前鋒到底是誰。

但是，勞塔羅只有22歲，潛力毋庸置疑。他不僅有望在俱樂部層面取代C羅，還有可能成為梅西在國家隊的接班人。本賽季，他在面對尤文、巴薩、多特等強隊時均有進球，可見他是一位不折不扣的「大場面先生」。

如果他能按照現有軌跡繼續發展下去，本賽季結束之前，整個歐洲足壇都必將熟悉他的名字。

毛羅·伊卡爾迪

雖然圍繞在伊卡爾迪身邊的爭議聲不斷，但是誰都無法否認，他擁有世界級的門前嗅覺。過去6個賽季，他一直是國米的最主要得分點；本賽季租借效力於大巴黎，他更是已在10場各項賽事中打進9球，有消息稱他已經在與卡瓦尼的競爭中獲勝，即將穩坐主力位置。

儘管原則上他還是一名國米球員，可是誰都明白，早已與俱樂部鬧翻的他重返梅阿查可能性幾乎為0，更何況藍黑軍團現在已經有了盧卡庫和勞塔羅這對攻擊組合，根本也不需要阿根廷人回歸。

其實在今年夏天，尤文就和伊卡爾迪傳出過緋聞，如果老婦人真的對「二弟」有意，只要拿出足夠的報價，阿根廷人斷然不會對自己的老東家「一往情深」。

從影響力來講，伊卡爾迪和C羅相比顯然還相去甚遠，但是這位高效射手可以成為尤文歐冠夢想的最後一塊拼圖。如果能夠如願捧得大耳朵杯，斑馬軍團的品牌價值也會更上一層樓。
