獲官方MVP！C羅完成梅開二度當選最佳，1數據成五大聯賽第1人

天天要聞
2 天 ago

北京時間12月15日22點，意甲聯賽第16輪迎來一場較量，尤文圖斯坐鎮主場迎戰烏迪內斯。開場不久，C羅就幫助尤文打破僵局，下半場C羅再取得進球+中柱錯失上演帽子戲法的機會！博努奇頭槌破門，普塞托扳回一球，最終尤文主場3-1擊敗烏迪內斯。此役獲勝後，尤文暫時領跑意甲積分榜。值得一提的是，C羅連續15個賽季進球數上雙，成為五大聯賽第一人！賽後，C羅獲得8.9分全場最高，並且當選本場最佳。

第8分鐘，伊瓜因分球給弧頂處的馬圖伊迪，馬圖伊迪嘗試一腳射門被烏迪內斯門將穆索沒收。第9分鐘，博努奇送出精準的長傳球迪巴拉禁區內射門被擋，C羅禁區左側得球後小角度抽射破門，尤文1-0領先烏迪內斯。C羅已經連續四場比賽都有進球，這是他在尤文生涯首次實現連續四場比賽都有破門。第13分鐘，伊瓜因送出直塞，禁區左側的迪巴拉跟上射門被穆索撲出。

第21分鐘，達尼洛在前場右側傳中，迪巴拉得球後直接吊門入網，不過裁判示意迪巴拉手球犯規，因此進球無效。第29分鐘，迪巴拉後場帶球突破到中圈傳球給前場的C羅，C羅回做給伊瓜因，伊瓜因射門被穆索撲出。第36分鐘，又是迪巴拉傳球，伊瓜因帶球到禁區外起腳遠射再次被穆索撲出。第37分鐘，伊瓜因送出直塞，C羅及時插上推射破門完成梅開二度，尤文2-0領先烏迪內斯。第45分鐘，尤文獲得左側角球開出，博努奇頭槌破門，半場比賽尤文3-0領先烏迪內斯。

下半場易邊再戰，第50分鐘，德保羅外圍嘗試一腳遠射偏出。第51分鐘，C羅帶球突入禁區左側分球給伊瓜因，迪巴拉接到伊瓜因的傳球後一腳搓射擊中橫樑。第55分鐘，烏迪內斯球員拉薩尼亞獲得單刀機會禁區內射門被布馮撲出。第60分鐘，本坦庫爾送出直塞找到C羅，C羅得球後殺入禁區內閃開角度一腳抽射打在防守球員身上變線，烏迪內斯門將穆索拿到皮球。第61分鐘，達尼洛低平球傳到禁區右側，伊瓜因迎球射門高出橫樑。第70分鐘，C羅連續突破後嘗試一腳低射擊中邊網。第73分鐘，福法納前場左側帶球內切到弧頂處遠射被布馮撲出。

第83分鐘，科斯塔傳球給C羅，C羅背身護球轉身起腳射門被烏迪內斯門將穆索撲出。第85分鐘，C羅弧頂處一腳遠射擊中立柱，險些上演帽子戲法。下半場比賽補時5分鐘，第94分鐘，斯特里格低平球傳到禁區中路，普塞托搶射破門幫助烏迪內斯扳回一城，最終尤文主場3-1戰勝烏迪內斯。
