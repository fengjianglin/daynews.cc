超級計算機預測歐冠1/8決賽結果，利物浦熱刺皇馬都被淘汰

2019-12-17 85條評論 1,084次閱讀 131人點贊

北京時間2020年2月19日凌晨4點，2019-20賽季歐冠1/8決賽第一回合將鳴鑼開戰。哪支球隊將在兩回合比賽中更勝一籌，哪支球隊將晉級歐冠8強，這些話題是球迷們目前比較關心的。同樣的超級計算機也對1/8決賽的結果作出了預測，這樣的預測靠譜嗎？我們不妨先來看看超級計算機是如何預測2019-20賽季歐冠1/8決賽比賽結果的。

歐冠1/8決賽對陣情況

2019-20賽季歐冠1/8決賽的8組對陣情況，分別是多特蒙德對陣巴黎聖日耳曼、皇家馬德里對陣曼城、亞特蘭大對陣瓦倫西亞、馬德里競技對陣利物浦、切爾西對陣拜仁慕尼黑、里昂對陣尤文圖斯、托特納姆熱刺對RB萊比錫、那不勒斯對陣巴塞羅那，超級計算機是如此預測兩回合比分：

超級計算機預測衛冕冠軍利物浦止步16強

多特蒙德vs巴黎聖日耳曼——第一回合：多特蒙德1-2巴黎聖日耳曼；第二回合：巴黎聖日耳曼2-1多特蒙德。巴黎聖日耳曼以4-2的總比分成功晉級8強。

皇家馬德里vs曼城——第一回合：皇馬1：1曼城；第二回合：曼城3-1皇家馬德里。齊達內未能延續神奇，兩回合曼城以4-2的總比分淘汰皇家馬德里。

亞特蘭大vs瓦倫西亞——第一回合：亞特蘭大1-1瓦倫西亞；第二回合：瓦倫西亞2-1亞特蘭大。兩回合瓦倫西亞以3-2的總比分淘汰亞特蘭大。

超級計算機的預測讓利物浦球迷揪心

馬德里競技vs利物浦——第一回合：馬德里競技1-0利物浦；第二回合：利物浦2-1馬競。雙方各自取得主場比賽的勝利，兩回合總比分2-2，馬競以客場進球多的優勢淘汰衛冕冠軍利物浦。

切爾西vs拜仁慕尼黑——第一回合：切爾西1-2拜仁慕尼黑；第二回合：拜仁慕尼黑2-0切爾西。拜仁慕尼黑的優勢較為明顯，兩回合以4-1的總比分獲勝。

​里昂vs尤文圖斯——第一回合：里昂1-2尤文圖斯；第二回合：尤文圖斯2-1里昂。兩回合比賽尤文圖斯都取得勝利，尤文圖斯最終以4-2的總比分晉級下一輪。

超級計算機預測尤文圖斯成功晉級8強

托特納姆熱刺vsRB萊比錫——第一回合：熱刺2-1萊比錫；第二回合：萊比錫1-0熱刺。穆里尼奧入主熱刺讓這支球隊受到更多的球隊，不過他沒能給熱刺帶來好運，兩隊總比分2-2，萊比錫以客場進球多的優勢勝出。

那不勒斯vs巴塞羅那——第一回合：那不勒斯0-3巴塞羅那；第二回合：巴塞羅那3-1那不勒斯。超級計算機認為這是歐冠16強賽中強弱最分明的一組對決，巴塞羅那兩場比賽都取得勝利，最終以總比分6-1的巨大優勢成為晉級8強的球隊。
