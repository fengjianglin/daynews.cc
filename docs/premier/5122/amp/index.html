周五015 意甲 國際米蘭VS羅馬

比賽時間：2019-12-07 03:45

2019-20賽季義大利甲級聯賽第15輪國際米蘭將在梅阿查球場迎戰羅馬。第14輪尤文圖斯遭薩索洛逼平，國際米蘭2-1擊敗費拉拉SPAL後，國際米蘭一舉超越尤文圖斯，成為意甲榜首。球隊最近的狀態相當出色，在意甲和歐冠聯賽雙線作賽保持4連勝。

阿根廷前鋒勞塔羅表現出色，在意甲已經取得了8粒進球，成功吸引了巴塞羅那的目光。國際米蘭為了留住勞塔羅，將提供新的續約合同，將其年薪提升到250萬歐，並去掉1.11億歐的解約金。此外另一名前鋒盧卡庫的表現也相當出色，對比在曼聯時期，盧卡庫有質的提升，在意甲已經斬獲10粒進球。盧卡庫的出色表現，與主帥孔蒂的重用密不可分。

值得注意的是國際米蘭將在下周中與巴塞羅那在冠軍杯中遭遇，巴塞羅那已經鎖定了小組第一，國際米蘭只要取勝就可以確保小組第二晉級。這場比賽對於國際米蘭極為重要。對於勞塔羅而言，更是重要，畢竟傳聞中巴塞羅那有可能引進勞塔羅作為梅西接班人來培養。

羅馬最近的狀態也相當出色，球隊在歐羅巴客場擊敗了伊斯坦布爾，球隊成為小組第一，下輪只要取勝即可順利以小組第一晉級。在意甲聯賽羅馬最近的表現也相當強勢，球隊連續8輪僅輸1場比賽，作客0-2意外輸給帕爾馬，在最近6輪羅馬拿下5場勝利。斯莫林、小克魯伊維特和哲科表現活躍，也是羅馬近期出色的原因之一。

這場比賽國際米蘭與羅馬的強強對碰，國際米蘭已經成為了聯賽榜首，球隊這個賽季目標恐怕已經瞄準了意甲冠軍，至於歐冠，球隊更多是走一步算一步的姿態。羅馬近況出色，已經頗具戰鬥力，但要想阻止如日中天的國際米蘭前進步伐，在客場羅馬恐怕做不到。

【勝平負】勝1.74

【總進球數】1、2、3

【半全場】勝勝、平勝

【比分】1比0、2比0、2比1
