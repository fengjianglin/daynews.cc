太冤！梅西禁區摔倒1秒後站起卻被判假摔吃黃，C羅給出正確示範

天天要聞
2019-11-28

球場上的梅西無疑是超神的，對方2到3名防守球員都不一定能攔住他，但他總有一個人「過」不去，那就是裁判。在歐冠與多特的比賽中，梅西罕見吃到黃牌，其原因居然是在對方禁區內假摔，不過通過慢動作顯示，梅西只是因為失去身體平衡而倒地，而且摔倒1秒後就站了起來，顯然不是要騙點。

歐冠小組賽第5輪，巴薩主場迎戰多特蒙特。本場比賽，梅西的表現堪稱夢幻，不但打進球隊第2粒進球，而且還送出2次美妙助攻，幫助蘇亞雷斯和格里茲曼各入一球。全場比賽，梅西貢獻多達11次成功過人，對方防線在他看來簡直就是多餘的。只不過，裁判卻在這樣一場近乎完美的比賽中再次搶鏡，因為他判罰梅西假摔了。

第51分鐘，梅西中路一路盤帶進對方禁區，隨後在過掉對方門將後倒在了禁區內。裁判第一時間衝過來，向梅西出示黃牌，理由是他假摔騙點。不過通過慢鏡頭顯示，梅西並非要假摔騙點，他是為了躲閃對手而失去平衡才摔倒，而且倒地後他也馬上站起身來並無要點球之意。可以說，裁判為此判他假摔並出示黃牌實在太過嚴格了。

縱觀梅西的職業生涯，他在禁區內假摔騙點的案例確實不常見，但和他並稱「絕代雙驕」的C羅卻屢見不鮮。面對類似情形，C羅給出了「正確」示範。

上賽季尤文與那不勒斯的焦點之戰，C羅就曾經製造了對方一次極有爭議的紅牌。C羅帶球反擊，那不勒斯門將梅雷特棄門出擊，他伸出右腳想破壞皮球卻一腳踢空，C羅順勢倒地，而且倒地動作也十分誇張。從針對C羅雙腿的近景鏡頭我們可以看出，梅雷特伸出的右腳，基本是從C羅雙腿之間踢過的。如果說兩人有身體接觸的話，那隻可能是梅雷特的右腳輕微觸碰到了C羅左腳。不管真相如何，C羅最後憑藉一己之力改變戰局、給球隊帶來巨大利益已成事實。

同樣類似的情況還發生在上個月。尤文主場與熱那亞的比賽。前90分鐘兩隊戰成1比1。第94分鐘，C羅禁區內拿球，面對對方的防守球員，他將球一扣，隨後與對方發生了身體接觸。然後，C羅再次習慣性倒地不起，而且倒地動作也有些誇張。慢鏡頭顯示，C羅與防守他的薩納布里亞確實有碰撞，但後者不管手部還是腿部的動作似乎都不足以推倒C羅。所以，C羅在這次身體接觸上「造點球」意圖明顯。

當然，這裡我們並不是要批評C羅不能「跳水」，更不是要讓梅西學習C羅，只是單純從比賽的角度分析什麼是假摔，畢竟每個裁判對於球員在禁區內的動作都有自己的理解。不管怎樣，梅西在這次禁區內倒地被判假摔確實讓人難以理解，隨後吃黃更是非常冤枉，他和C羅相比，錯就錯在沒有倒地後的一系列誇張動作，所以才會遭到誤判。
