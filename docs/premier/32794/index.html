歐戰獎金分布：四強巴薩比歐冠冠亞軍還要多？

2019-12-24 2,001條評論 3,621次閱讀 1089人點贊

體壇新聞12月24日訊 歐足聯最近公布了上賽季歐冠以及歐聯杯獎金的分布方案。獎金數額根據各傢俱樂部的歐足聯繫數、賽事獎金以及電視轉播分成等因素進行計算。

2018-19賽季歐冠獎金分配前十如下↓↓↓

1、巴薩（1.18億歐）

2、利物浦（1.11億歐）

3、熱刺（1.02億歐）

4、尤文（9600萬歐）

5、曼聯（9300萬歐）

6、曼城（9300萬歐）

7、巴黎（8600萬歐）

8、馬競（8600萬歐）

9、皇馬（8500萬歐）

10、拜仁（8300萬歐）

巴薩在上賽季歐戰共收穫了1.18億歐元的獎金，位居歐冠榜單首位。歐冠冠軍利物浦獲得1.11億歐緊隨其後。前六中有四家是英超俱樂部。

歐足聯繫數（下表紅色部分）基於俱樂部近十年表現評分，皇馬在這部分獲得了3550萬歐元的最高分成，巴薩獲得3400萬歐位居第二；

賽事獎金（下表黃色部分）根據俱樂部在該賽季歐冠聯賽的成績給予分配，利物浦毫無懸念得獲得了6000萬歐的最高分成；

電視轉播權（下表藍色部分）的比重在這賽季遭到了削減，獲額最高的巴黎和巴薩分別獲得了2400萬歐以及2300萬歐。

歐聯杯的獎金分配製度大體上與歐冠一致。2018-19賽季歐聯杯獎金分配前十如下↓↓↓

1、切爾西（4600萬歐）

2、阿森納（3900萬歐）

3、法蘭克福（2700萬歐）

4、雷恩（2000萬歐）

5、比利亞雷亞爾（1800萬歐）

6、塞維利亞（1700萬歐）

7、勒沃庫森（1500萬歐）

8、拉齊奧（1500萬歐）

9、AC米蘭（1500萬歐）

10、貝蒂斯（1300萬歐）
