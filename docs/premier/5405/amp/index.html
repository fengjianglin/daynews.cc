剛剛結束的2019-20賽季歐冠小組賽第五輪第一比賽日的比賽中，巴黎客場2比2戰平皇馬，由於另外一場比賽加拉塔薩雷也1比1戰平布魯日，皇馬提前一輪獲得出線資格，而巴黎上輪就出線了。B組的拜仁6比0大勝貝爾格萊德紅星隊，熱刺則4比2逆轉奧林匹亞科斯，最終熱刺也提前一輪進入十六強，拜仁同樣上輪就出線了。C組的曼城1比1戰平礦工，亞特蘭大則2比0薩格勒布迪莫，目前這個小組只有曼城出線，剩下三隊都有機會在最後一輪出線。D組的尤文主場1比0馬競，提前進入十六強，而勒沃庫森則2比0莫斯科火車頭，最後一輪馬競獲勝就能出線，他們的對手是莫斯科火車頭。
