英超四球隊全部晉級16強！熱刺和切爾西前路艱難

2019-12-11 455條評論 1,828次閱讀 253人點贊

剛剛結束的2019-20賽季歐冠小組賽第六輪的比賽中，切爾西主場2比1戰勝了里爾晉級歐冠十六強，不過由於他們和瓦倫西亞的比賽中拿到了1平1負的戰績，所以只能在同分的情況下排名小組第二。而另外一支球隊利物浦則2比0輕取薩爾茨堡紅牛，以小組第一的身份晉級歐冠十六強。而另外兩支英超球隊，曼城上輪比賽就鎖定小組頭名，而熱刺則鎖定了小組第二，英超的四支球隊全部晉級歐冠十六強。

雖然都晉級了十六強，但是四支球隊的情況不同，利物浦和曼城由於拿到了小組第一，所以他們的對手可能不會太強，他們可能遇到的強隊只有皇馬、馬競和多特，而且即便遇到這些球隊利物浦和曼城也不會懼怕對手，不過熱刺和切爾西的情況就不容樂觀了，因為他們的對手幾乎都是強隊，而其中切爾西的運氣可能是最差的，而熱刺比切爾西稍微好一點點。

按照歐冠原則，16強賽不會遇到小組賽的同組對手，同時也不會遇到相同聯賽的對手，而這種情況下切爾西的對手會從巴黎、拜仁、尤文、巴薩和萊比錫五支球隊中產生，而這些球隊的實力都超出切爾西很多，除非切爾西抽到萊比錫，不然球隊很難晉級。而熱刺的情況則稍微好一點，他們可能遇到的對手有巴黎、尤文、巴薩、萊比錫和瓦倫西亞。以熱刺目前的狀態，遇到巴黎、尤文和巴薩會非常困難，不過如果遇到萊比錫或者瓦倫西亞的話，熱刺晉級的機會還是很大的。

​

上賽季，英超球隊在歐冠賽場上的表現非常出色，所有參賽的英超球隊都晉級了十六強，而利物浦和熱刺甚至在歐冠決賽賽場上會師了，只是不知道本賽季這四支球隊的表現怎麼樣呢？歐冠賽場上還會像上賽季一樣，出現歐冠決賽賽場上的兩支球隊都是英超球隊的盛景嗎？同時，上賽季歐聯決賽的兩支球隊也都是來自英超，而目前歐聯杯賽場上狼隊和曼聯已經晉級了，阿森納只要不大比分輸球也會晉級，不知道他們在歐聯杯賽場上會有什麼樣的表現呢？
