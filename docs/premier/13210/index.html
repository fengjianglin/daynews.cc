安切洛蒂在你心中是怎樣的一位教練？能排第幾？

2019-12-12 517條評論 2,765次閱讀 867人點贊

一早起來看見安切蒂諾下課的新聞有點震驚，畢竟他是足壇有名的戰術大師與冠軍收割機，居然在那不勒斯成為「權利鬥爭」的犧牲品，最後慘遭被解僱的厄運

說起安切洛蒂，他在我心中就是一副老好人的形象，無論在哪執教，總感覺他都有點被俱樂部「欺負」的意思。在切爾西，他的球隊打進百球英超奪冠，最後難逃在車裡被解僱的厄運。在皇馬，他率領球隊奪取隊史第十座歐冠獎盃並且踢出22連勝，最後也慘遭下課。在拜仁，在那不勒斯，最後的命運也亦是如此，不僅令人感到唏噓

在執教履歷和冠軍榮譽上，安胖無疑是現役足壇最成功的主教練。他是唯一一位拿到四大聯賽冠軍的教頭，曾在米蘭、切爾西、巴黎和拜仁都拿到過聯賽冠軍；而且是唯一一位率領兩支不同球隊贏得3次歐冠冠軍的主教練，

安切洛蒂在我心中可以說是一位非常偉大的主教練，論執教能力，我個人感覺他不遜於瓜、穆、克這三位主教練，並且在經驗上比他們還要豐富，世界前五妥妥的，不知道在你們心中安胖是一位怎樣的主教練，實力能排第幾？
