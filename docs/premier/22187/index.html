馬爾基西奧：我捨不得腳下的這片綠茵場，但傷病改變了一切

2019-12-19 956條評論 1,943次閱讀 477人點贊

直播吧12月18日訊 前尤文中場馬爾基西奧在接受《米蘭體育報》採訪時表示，他想要繼續留在綠茵場上，但傷病改變了一切，身體狀態實在不允許。

馬爾基西奧在2016年4月與巴勒莫的比賽中前十字韌帶斷裂，這迫使他錯過了2016年歐洲杯和2016-17賽季初的比賽。他在採訪中提到：「只有你真正經歷過類似的傷病，你才能理解它意味著什麼。受傷之後，我的跑動能力和反應速度都大不如前，我的腦海里有一個聲音告訴我，我再也不是以前的馬爾基西奧了。」

「這並不意味著你無法重回巔峰，總有人能夠做到。但是，它確實給我帶來了很大的改變。為了加強膝蓋力量，我不得不增重，尤其是腿部，但這也使得我失去了以前的靈動，我只好改變自己在場上的踢法。在倒地的那幾秒鐘，你所感受到的痛苦往往是加倍的。我意識到，我已經無法回到之前的狀態了。那真是一段艱難的時光。」

2017-18賽季結束後，馬爾基西奧與尤文圖斯解約，隨後加盟澤尼特，他在當時就決定一年後退役。對此他表示：「在我職業生涯的最後一年，我的另一個膝蓋也受了傷，所以我很清楚自己的決定。當時我的左膝半月板撕裂，難以進行劇烈運動。不過我仍然選擇繼續，因為我覺得自己可以承受，但隨後我不得不再次進行手術，否則我的膝蓋只能撐到40歲了。」

「當這樣沉重的現實擺在你的面前時，你不可能還滿腦子只想著足球，你必須想想自己以後的生活。我無法欺騙自己，儘管我一心想留在綠茵場上，但我的身體實在不允許。」

（淺色調）
