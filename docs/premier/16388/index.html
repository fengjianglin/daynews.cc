三叉戟合體，C羅梅開二度，尤文取勝烏迪內斯

2019-12-16 309條評論 1,752次閱讀 410人點贊

北京時間12月15日22時，2019-20賽季意甲第16輪，尤文圖斯主場迎戰烏迪內斯。道格拉斯-科斯塔、拉姆塞和本坦庫爾傷愈復出，皮亞尼奇、誇德拉多停賽，什琴斯尼肩部受傷，布馮首發出場，從而超越皮耶羅，以477次出場成為在意甲出場最多的尤文球員。

尤文3-1完勝烏迪內，C羅梅開二度，連續4場比賽進球，博努奇頭槌建功。本場比賽，三叉戟頻頻連線，多次製造威脅。防守方面，德米拉爾繼續高光表現。布馮也屢次作出精彩撲救。

比賽回顧:

比賽第八分鐘，尤文後場送出長傳，直接到了烏迪內斯的禁區弧頂前沿，伊瓜因頭槌蹭了一下，球落到了禁區左側，C羅拍馬殺出，一腳凌空抽射破門得分，1:0！C羅連續4場進球——

12月1日對陣薩索洛；

12月7日對陣拉齊奧；

12月11日對陣勒沃庫森；

12月15日對陣烏迪內斯 ​

27分鐘尤文打出一次精彩反擊！來自前場的配合，伊瓜因最後拿球射門，可惜被對方門將撲到！

第36分鐘，伊瓜因直塞，一腳妙傳給到總裁，總裁推射破門！梅開二度！2:0

45分鐘，尤文左側開出角球，德米拉爾先是頭頂了一下，接著博努奇甩頭直接攻門！3-0！

易邊再戰，烏迪內斯連續獲得機會，布馮發揮出色，高接抵擋，力保球門不失。

首發出場的新人布馮以477場的聯賽出場次數，超越斑馬王子皮耶羅成為代表尤文出戰意甲聯賽場次最多的球。

第76分鐘，迪巴拉、博努奇下場，換上貝爾納代斯基和德利赫特。

81分鐘伊瓜因下場了，袖標戴在了馬圖伊迪的手臂上。科斯塔登場。

84分鐘，羅納爾多的一次射門擊中立柱，錯過本場比賽帽子戲法的機會。

臨近比賽結束，羅納爾多連續獲得射門機會，烏迪內斯的門將發揮出色，加上門柱的幫忙，力保自家球門不第四次被攻破！

94分鐘，烏迪內斯打入挽回顏面的一球。

比賽結束，尤文3:1取得比賽的勝利！
