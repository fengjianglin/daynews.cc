34歲C羅感人一幕，禁區內表演倒鉤射門，狀態不佳他仍拼盡全力

2019-12-02 0條評論 17次閱讀 0人點贊

北京時間12月1日晚上7點30分，意甲新一輪的較量在安聯球場拉開戰幕，尤文圖斯坐鎮主場迎來薩索洛的挑戰。如果單從實力和戰績上來說，西瓜軍團肯定不是斑馬軍團的對手。所以，這場比賽大家都在期待薩里的球隊能夠拿到什麼樣的勝利，剛剛復出沒有多久的C羅能不能找回自己的狀態、順便打破進球荒。事實證明，薩索洛並不好踢。經過90分鐘的激戰，尤文在主場2-2被逼平。這場比賽的不勝，讓尤文丟掉榜首，球隊的損失大了去了。

比賽中，C羅再一次進入到首發陣容。在90分鐘的時間裡，C羅發揮不算好，他在前場出現多次丟球的情況，而且跑動也沒有賽季初那麼利索。但是，也不能說他發揮糟糕，在比賽中C羅打進扳平一球，不僅如此，他在前場也做出了不可磨滅的貢獻，比如爭頂頭槌，最後時刻的搶點以及參與前場的調度。誠然，C羅本場比賽的狀態還不在最佳，有些動作甚至讓人覺得他下滑太厲害了，可他在球場上依舊拼盡全力在付出。

比賽來到第50分鐘，尤文獲得一次角球機會，皮亞尼奇在角球區罰出皮球。皮球一飛到禁區，許多球員就跳起來爭頂。在一通爭搶後，誰也沒有獲得球權，之後皮球飛到C羅身邊。眼看皮球就要落到自己的身前，C羅稍作調整，向前走了幾步後騰空而起，來了一腳倒鉤射門。遺憾的是，C羅的這腳射門沒有形成威脅，埃姆雷-詹隨後的射門也被薩索洛門將沒收。

儘管C羅的這次射門威脅性不大，但是能夠在這樣的場合里選擇用這樣的射門方式，不得不說C羅真的拼了。要知道本場比賽開始後，天降大雨，場地濕滑無比，C羅奮不顧身來這一下，也算是為了球隊傾其所有了。這一幕，讓人感動。這場比賽，C羅的狀態不太好，身體也沒有在最佳狀態，可他仍然用行動證明自己的價值。在球場上，只要他在一分鐘，他就會努力一分鐘；只要他在一秒鐘，他就會努力一秒鐘。他已經34歲了，還如此拼，球迷應該向他致敬。

任何人都擋不住時光的打磨，誰都會有34歲的時候，這個時候拚命黑被人，等你的偶像到了34歲，35歲，狀態下滑了，別人也是同樣的態度，試問你會怎麼想？所以，嘴下留情，好好看球才是最重要的。不喜歡可以，肆意詆毀就太不應該了。
