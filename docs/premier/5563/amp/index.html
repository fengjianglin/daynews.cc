AC米蘭加入了競爭簽約“切爾西”法國前鋒的行列，儘管對手是國米

天天要聞
2019-11-22

切爾西前鋒奧利維爾·吉魯並不是新主帥「蘭帕德」未來計劃的一部分，有報道說他會在明年一月轉會窗口期轉會到義大利甲級聯賽。儘管國際米蘭被認為是最有可能簽約他的對象，但他們的同城死敵AC米蘭也加入了競爭簽約法國人的行列，這使得事情變得更加有趣。

奧利維爾·吉魯

身高192cm的吉魯堪稱優質型男，眼神堅毅，稜角分明，絡腮的胡碴更添了幾分成熟魅力，發達的胸肌更是惹人注目。效力於切爾西俱樂部的吉魯，在場上是一名優秀的前鋒，難得在身高體健的同時，腳下技術也保有一定水準，可作為中鋒游弋在對手禁區，助攻能力強。與大衛·特雷澤蓋、蒂埃里·亨利等年少成名的前輩們不同，24歲才征戰頂級聯賽、25歲才入選國家隊的吉魯無疑是大器晚成的典型。在阿森納的優異表現，讓吉魯逐漸邁入歐洲一流射手的行列 。

奧利維爾·吉魯上賽季1月以1700萬歐元的轉會費從阿森納加盟藍軍，並在與他們的第一個完整賽季贏得了歐足聯歐洲聯賽的冠軍。他也是聯賽的頭號射手，切爾西在決賽中擊敗了他的前東家阿森納。

阿森納時期的「奧利維爾·吉魯」

然而，隨著前任主教練毛里奇奧·薩里的離開和蘭帕德的到來，吉魯已經完全淡出了人們的視線，本賽季只在英超出場四次。因此，這位法國前鋒正在尋找離開俱樂部的機會，安東尼奧·孔蒂的國米也成為了他的熱門目的地。

安東尼奧·孔蒂
