豪門風範！巴薩之後曼城也不“放水”熱蘇斯3球英超冠軍客場大勝

天天要聞
2019-12-12

球隊提前鎖定小組第一，最後一輪是否就「放水」給急需3分出線的球隊？這兩天西甲，英超豪門巴薩和曼城都碰到這樣的選題。巴薩昨天客場對陣國米，他們若是不出力讓國米贏球，那國米就能晉級，然後事實上巴薩也是給了國米機會，比如梅西輪休，蘇亞雷斯替補。只不過豪門就是豪門，最多也就是人員上有所調整，但賽場上還是要實力說話，國米要想贏球也得擊敗沒有梅西的巴薩。

昨天國米最終並沒有抓住機會，盧卡庫和馬丁內斯都錯失了一些進球機會，然後17歲小將一劍封喉破門幫助巴薩2：1鎖定勝局，輸球的國米只能連續兩年最後一輪在主場飲恨無緣歐冠16強。今天凌晨曼城客場對陣薩格勒布迪納摩，他們要是輸給對手，那麼薩格勒布迪納摩就極可能拿到16強入場券。開場僅10分鐘，卡德日奧爾右路45度角拿球直接吊到禁區內，奧爾默前點搶點直接小角度凌空怒射，皮球應聲入網，薩格勒布迪納摩1：0首開紀錄。

然而在丟球後曼城開始加強攻勢，很快他們的進攻就壓制住薩格勒布迪納摩，第34分鐘，馬赫雷斯禁區右側邊上拿球晃開防守後將球吊到後點，無人防守的熱蘇斯搶點頭槌攻門，皮球應聲入網曼城1：1扳平比分。易邊再戰第49分鐘，曼城前場左側一連串精妙的短傳配合，福登策應將球交給熱蘇斯，後者殺入禁區內躲閃騰挪過掉2人包夾後冷靜推射得手梅開二度，曼城2：1反超比分。

4分鐘後，曼城還是左側撕破薩格勒布迪納摩防線，門迪下底傳中，後點無人防守的熱蘇斯凌空將球擋入網窩，熱蘇斯4分鐘連進2球上演帽子戲法，在他的出色發揮下曼城連扳3球3：1領先。第84分鐘，曼城這次是右側突破薩格勒布迪納摩防線，B席爾瓦殺入禁區內倒三角回敲，福登跟上推射得手，曼城4：1鎖定勝局。

最終曼城打得比昨天的巴薩還要放得開，連入4球逆轉大勝薩格勒布迪納摩，主隊沒有收到曼城的「放水」大禮包最終無緣出線，不過曼城的「不放水」倒是成就了意甲勁旅亞特蘭大的神奇逆襲，他們從前3輪0分到最後7分鎖定小組第二闖入歐冠16強。
