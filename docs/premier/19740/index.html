C羅創造“神紀錄”尤文追上國米

2019-12-16 32條評論 3,302次閱讀 512人點贊

揚子晚報紫牛新聞訊（記者 湯敏）歐洲五大聯賽爭冠最激烈的不僅僅是西甲還有意甲。16日凌晨，在意甲第16輪，尤文主場3:1擊敗了烏迪內斯，而國米客場被佛羅倫薩1：1逼平，雙方目前同積39分。

尤文能夠擊敗烏迪內斯，C羅當居首功，他的梅開二度讓球隊取得了2:0的領先。上半場9分鐘C羅就為斑馬軍團首開紀錄，第37分鐘他接到伊瓜因的助攻再下一城，隨後博努奇頭槌破門，半場尤文圖斯就三球領先對手。這也是自2018年2月以來，尤文時隔一年多首次在意甲賽場半場攻入三球。

周中的歐冠小組賽最後一輪，C羅也為尤文圖斯打入一球，幫助斑馬軍團挺進淘汰賽。一周內C羅兩次首發並打滿全場，對一位34歲的球員來說並不容易，而更加令人讚歎的是他在比賽中連續取得進球。如果算上此前的兩場意甲聯賽，C羅已經連續四場完成了破門。而且，C羅本賽季在尤文圖斯的進球已經達到11粒。因此創造了一項神紀錄——連續15個賽季俱樂部進球數超過10+。而比C羅小兩歲的梅西此前剛剛創造了連續14個賽季進球數超10+的成績。

一個月前，從國家隊返回的C羅受到膝蓋舊傷的困擾，多次被主帥薩里提前換下，狀態也沒有達到最佳。恰在此時，老對手梅西6度斬獲金球獎，一些人不免提出疑問：雙驕大戰梅西再次領先一個身位，年近35的C羅尚能飯否？但C羅很快就在球場上證明了自我，以梅開二度向世界證明他還是那個不屈的男人。
