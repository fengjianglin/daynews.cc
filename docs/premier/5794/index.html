尤文正接觸瓜迪奧拉，新帥薩里為何在尤文站不住腳？

2019-11-09 0條評論 16次閱讀 0人點贊

意甲豪門尤文圖斯本賽季目前依舊在聯賽中處於領跑的地位，但是相信很多尤文的球迷都應該可以看出來本賽季尤文在場面上的把控能力減弱了，好多時候都是需要靠當家球星C羅的個人發揮。其實今年對於尤文而言還是有很大的變化的，球隊進一步的優化和補強陣容。

雖然球隊此前執教了5年的功勛教練阿萊格里在今年夏天和俱樂部分手，但是尤文將薩里這位曾經在意甲征戰多年的主教練帶到了球隊。之前夏天有像瓜迪奧拉之類等多名世界級的名帥和尤文圖斯傳出緋聞，但是球隊還是選擇了薩里，這個結果多少還是讓人有些吃驚的。雖然這位剛剛在夏天的歐聯杯中奪冠的主教練也是具備自己的戰術風格的，但是他對於豪門的掌握能力還是稍顯欠缺。不過對於新任的主教練，俱樂部和球迷都是寄予了厚望。畢竟薩里也曾經執教那不勒斯多年，對於意甲的環境還是非常的熟悉。

大家都知道雖然在技戰術打法上每一個教練都不一樣，但是也會有相似之處。但是對於尤文而言，阿萊格里和薩里完全就是兩種戰術風格。薩里在此前執教切爾西的時候就非常執著的使用自己的戰術打法，薩里在英超的賽場上讓切爾西的球員打起了傳控足球。

薩里此前執教的那不勒斯中並沒有什麼實力的大牌球星，但是在切爾西中，球隊可是擁有多名球星的。這就對於薩里而言很不適應，因為他需要的是調動全隊一起進行進攻，但是在切爾西，球隊擁有像阿扎爾這樣的球星，那也就不需要全員的參與。這正是薩里所反感的，因為他此前曾多次公開表現，他並不喜歡某些球員在比賽中獨自「戰鬥」。不過非常打臉的是，球隊在進攻中還只能依靠球星的單打獨鬥。雖然帶領切爾西最後拿了一個歐聯杯的冠軍也算有了一個比較好的交代，但是也暴露出了薩里的個人氣質還是不適合去執教一些大牌實力豪門的。

就像本賽季的尤文在薩里的調教下，基本可以說是「面目全非」。舉個例子，此前尤文在聯賽客場面對保級隊時，雖然球隊輪的當家球星C羅沒有出場，但是球隊在場上還是有很多主力球員的。鋒線端的迪巴拉加伊瓜因的組合已經非常的豪華。但是就是這樣一套4.96億歐的首發陣容硬生生的在客場被只有2340萬歐的萊切給逼平了，賽後很多媒體對於薩里的執教頗有怨言。外界普遍認為薩里還是沒有改掉此前的「毛病」，他的執教理念只能適合去一些中游球隊執教。要讓他執教豪門，那麼結果註定是悲劇的。

日前據義大利媒體的報道表現，目前俱樂部有意希望將瓜迪奧拉帶到球隊。目前俱樂部的高層認為目前的尤文只有像瓜迪奧拉這樣的世界級名帥才能穩住，才能恢復以往在比賽中的戰鬥力，而不是像現在這樣，大部分的比賽都是舉步維艱。
