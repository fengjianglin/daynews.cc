「歐冠戰術室第3期」75分鐘的迪巴拉，25分鐘的德里赫特

2019-11-29 0條評論 13次閱讀 0人點贊

歐冠第五輪D組的一場焦點戰，由尤文圖斯迎戰到訪的馬德里競技。11月底的義大利天氣冷到無以復加，但27號的尤文圖斯競技場依然座無虛席。

賽前尤文領先馬競3分領跑小組第一，這次的小組賽薩里的弟子們表現得算是出色，小組五場比賽不敗。不過西蒙尼最近似乎就有些鬧心，聯賽給巴薩皇馬緩過來神現在已經排在第5位，上一場歐冠又1-2不敵勒沃庫森,。從10月30日在聯賽戰平阿拉維斯之後，只有11月10日3-1勝西班牙人虐了個菜，其餘四場均非平即敗。

反觀尤文圖斯從11月開始保持著全勝的戰績，聯賽以35分排名第1和國際米蘭成為第一方陣。說到這裡不得不提一句，表妹如今是風生水起，AC米蘭卻在聯賽第12的泥淖里無法自拔，萬一要是掉到了意乙這米蘭德比就真的不知道什麼時候再能見到。戰績上的優越沒法掩蓋多名主力受傷的情況，這包括基耶利尼、阿萊士-桑德羅、道格拉斯-科斯塔、皮亞察，還有多名主力歐冠累計黃牌歐冠禁賽，包括埃姆雷-詹、曼朱基齊。這樣來看其實馬競的陣容更為齊整，中後衛上希門尼斯和薩維奇的傷退，鋒線上迭戈-科斯塔，除此之外其他球員身體狀態大都不錯，但競技狀態就很難說。

尤文圖斯1-0馬德里競技首發及輪換

尤文圖斯這邊薩里給出4-1-2-1-2的站位，門將位置上什琴斯尼，中後衛組合為德里赫特和博努奇，德西利奧和達尼諾分別鎮守左右邊路，皮亞尼奇拖後擔任後腰，中前衛是馬圖伊迪和本坦庫爾，拉姆塞司職前腰，C羅和迪巴拉壓在鋒線上。

雖然此前尤文球迷想方設法為德里赫特如何不再禁區內手球進言獻策，但隨著聯賽和歐冠的進行，這位前阿賈克斯最年輕的歐冠隊伍隊長還是展現出了身價所在，不斷地補位化解威脅使得馬競最後時間段水銀瀉地般的攻勢在最後一擊上化為烏有。

皮亞尼奇算不上防守型的後腰，在這個關鍵位置他保持了92.3%的傳球成功率也算是完成了自己的職責，但身材方面的短板使得本坦庫爾和馬圖伊迪需要不斷地來回跑動，當然這兩位球員這場比賽也是完成了自己的任務，覆蓋了場上相當大的範圍。拉姆塞表現平平，沒有製造太多的威脅。鋒線上的C羅這場比賽顯得有些力不從心，不知道這是因為年齡增長還是馬競太過於熟悉他的原因。迪巴拉全場最佳無爭議，表現活躍且狀態極佳。

西蒙尼雖然不像薩里那邊那樣傷兵滿營，但給出的站位倒是日新月異。門將位置上是奧布拉克，本場傳球成功率比什琴斯尼低了近一半。托馬斯和埃克托-埃雷拉的雙防守後腰組合此前在馬競陣中也只出現過一次，後防線上的洛迪-埃爾莫索-菲利佩-特里皮爾的組合是自從戈丁遠走意甲之後馬競比較嘗出現的後防陣容，鋒線上是莫拉塔搭配比托洛。

全場比賽尤文圖斯表現更為出色，比賽開始尤文就將比賽節奏加快，雙方球員在中場的搶斷極為積極，全場尤文11次搶斷，搶斷成功率50%，馬競更是有18次搶斷，搶斷成功率在60%。可以說只要在對方中前場拿球後稍作思考，就會有球員就會上搶。兩支球隊的打法非常相似，以邊路進攻為主，全場比賽尤文的邊路進攻佔78%，馬競的邊路進攻則佔到69%。這邊只從尤文方面來簡評這場比賽，適當提及部分馬競球員。

尤文本場的表現給人一個感覺，在馬競半場的推進比在自己半場的傳接球更讓人放心。在馬競的半場尤文的傳球很流暢，中場的馬圖伊迪、皮亞尼奇、本坦庫爾以及拉姆塞之間的配合算得上默契。馬圖伊迪個人感覺攻強於守，這也在一定程度上彌補了邊後衛德西利奧進攻屬性的不足，可以明顯看出德西利奧這個邊後衛的插上進攻腳下頻率是略顯遲緩的，尤其在與迪巴拉的靈活對比之後這種感覺更為明顯，而德西利奧也拿到了全場最多的3次失誤。

這個類似於菱形中場的站位和前場之間的交叉跑位也很頻繁，迪巴拉並非傳統意義上的前鋒，在阿根廷他和梅西的位置相重合，在尤文他也和梅西在巴薩擔任著相似的角色，偽9號的存在。更多的回到中場接應，拉姆塞會適時補上迪巴拉的位置，在鋒線上迪巴拉和C羅也在不停地換位以尋求馬競後防更多的空檔。

還是說回迪巴拉，本場比賽迪巴拉的狀態非常好，在進球前，上半場的技術動作就可以看出迪巴拉可以說是在一個比較高的競技水平上。隊友40米的長傳連停帶過得恰到好處，所以這個進球其實也不算意外，迪巴拉的左腳確實值一億歐。賽後談到迪巴拉進球時，薩里感慨：「我的第一反應是你TM居然想在那個位置直接射門？然後他就做到了，我只能向他鼓掌致意。」半年前，迪巴拉還是尤文想要掃地出門的球員，本賽季他用非常高效的表現證明了自己，這不僅包括意甲還有歐冠。

C羅今年34歲，這是他歐冠出場的第167場比賽，在卡西利亞斯的177之後，歐冠出場次數上升到第2位。那天他再次面對西蒙尼，很可惜不是這場比賽的主角，他獲得單刀的機會後沒法加速越過馬競的後衛然後射門，解說打趣道這要是換作姆巴佩估計就趟過去了。確實如此，在進攻端表現平平，他沒法有效地內切到禁區，有過很有想像力的傳球但隊友沒有心領神會，腳底下的動作看起來靈活但又有些沉重。和迪巴拉對比之後，或許他已經快要到離開世界頂級聯賽的邊緣，但沒人知道他還能堅持多久。

第63分鐘，薩里用貝爾納代斯基換下表現平平的拉姆塞，陣型更像是改為了4-3-3，貝爾納代斯基的表現其實在拉姆塞之上，雖然是打中了橫樑，這也是為什麼他30分鐘比拉姆塞60分鐘的賽後評分更高的原因。

第76分鐘，薩里準備換上伊瓜因，我以為是換下C羅，但實際上是換下迪巴拉。我其實並沒有讀懂薩里的意思，迪巴拉並沒有表現出體力不足和競技狀態的下滑，反而是C羅的表現才是更應該被替換的那個。這個換人之後，馬競最後15分鐘的攻勢徹底打開，德里赫特在這段時間表現出色，無論是面對高空球還是地面球都做出了很好的處理選擇。在最後時刻，莫拉塔本可以改寫比分，可惜門前自己腳下拌蒜，球眼睜睜地從他身邊飛過去，狂奔一整場的莫拉塔最終也沒抓住這個機會。

本場比賽裁判的吹罰尺度也是相當寬鬆，C羅被侵犯倒地的次數也居多，到比賽結束也沒受傷算是一種幸運，馬競動作也是挺大的。

菲利克斯是西蒙尼下半場進行的第一個換人，西蒙尼在下半場開始20分鐘之內用掉了所有的換人名額。話說菲利克斯上場之後有幾腳不算太具有威脅性的遠射，這幾腳打得莫拉塔是直搖頭，我也一度覺得他本場替補上來沒法發揮有效的作用。替補上場快兩分鐘沒有一次觸球，不知道的還以為馬競鋒線對這個小年輕有什麼成見，其實是找不到合適的出球路徑。

不過菲利克斯終究還是對得起自己的身價，這個外腳背傳得極具隱蔽性，可惜馬競沒有抓住機會，德里赫特的補防速度之快讓馬競的前鋒來不及調整，菲利克斯的視野和腳下技術，德里赫特的意識和解圍速度，這也算是兩個身價過億的足球天才之間的一次間接對話。

要說馬競鐵血還是鐵血，最後15分鐘攻勢也算是一浪高過一浪，尤文的失誤也在增多，在基耶利尼缺陣的情況之下，博努奇作為後防中堅和年輕的德里赫特之間的配合，讓尤文禁區雖然是風聲鶴唳但最終也沒讓馬競嘗到甜頭。不過說到底還是沒明白為什麼76分鐘換下迪巴拉，後面的場上局勢完全證明換下他是很錯誤的決定，馬競再有個5分鐘能扳平比分也說不定，不過還好德里赫特在迪巴拉下場後的20多分鐘里的表現對得起自己的身價。
