曼城又一主力遭禁賽！溫格上任FIFA總監，38歲伊布官宣離隊

天天要聞
1 個月 ago

過去的24小時，英格蘭足壇和世界足壇都有不少重磅消息，紛紛上頭條。

比如溫格接受國際足聯邀請，出任世界足球發展總監；曼城中場貝納多-席爾瓦涉嫌種族歧視，被英足總禁賽一場；38歲的伊布和洛杉磯銀河合同到期，已離開大聯盟但未透露下家。

1、溫格出任FIFA全球足球發展總監

FIFA主席因凡蒂諾在蘇黎世總部親自迎接溫格的到來。按照FIFA官網介紹，溫格將負責監管和推動世界各地男女足球運動的發展，同時教授還是FIFA足球技術諮詢小組成員，負責審查足球規則制定和修改。

溫格履新FIFA，也意味著他未來一段時間都不會執教球隊，不會接手拜仁。對於新的工作崗位，教授滿懷期待：「我非常期待接受這一重要挑戰，能夠從更廣闊的角度去分析足球我很感興趣，這項使命是全球性的。我們看到了新的國際足聯，將運動本身作為目標核心，並決心以不同方式來發展這項運動。」

視頻-FIFA主席歡迎教授

FIFA主席因凡蒂諾認定溫格是最恰當人選，70歲的溫格是足球界最受尊敬的人物之一。因凡蒂諾強調，教授的戰略眼光，能力和精力，以及勤奮的態度，是國際足聯求賢若渴的原因。

2、貝納多-席爾瓦「推特門」禁賽一場

曼城中場大將貝納多-席爾瓦被英足總禁賽一場，將無緣英超下一輪與切爾西的焦點大戰，原因是貝席兩個月前在社交媒體發布的一條推特被認定涉嫌種族歧視。

今年9月22日貝席在推特上傳了一張圖片，與隊友門迪開玩笑。但卻被英足總指控行為不當，涉嫌污辱隊友，有種族歧視嫌疑。事實上當事人門迪並未感到冒犯，還回復了貝席。英足總認定貝席違反英足總規則E3（1）、E3（2），構成「嚴重違反規則」的行為。

按英足總規定，如果種族歧視行為發生在球場上，且證據確鑿的話，球員將至少被禁賽6場。由於貝席是在場外開玩笑，所以只禁賽一場。但還將罰款5萬英鎊，另外貝席還將被強制參加英足總安排的4個月教育課程。

BBC記者Simon Stone報道，曼城俱樂部認為貝席遭到了不公平對待，而且貝席並不知道巧克力暗含種族歧視意味，門迪是貝席的好朋友，門迪本人也清楚表示這只是一個玩笑。但鑒於多事之秋，曼城不會對處罰進行上訴。

3、伊布與銀河合同到期離隊

「逼王之王」茲拉坦-伊布拉希莫維奇在大聯盟征戰的故事到了終點，北京時間今日上午，洛杉磯銀河官方宣布與伊布合同到期，雙方確定不再續約。在38歲的年紀，奉先依然是足球界超級大咖，在梅羅之外收割無數粉絲。

銀河主席在官網致敬伊布：「此時此刻，我們要感謝伊布為洛杉磯銀河和MLS作出的貢獻。自2018年加盟以來，茲拉坦為洛杉磯足球運動帶來積極影響。我們為他的工作精神和激情表示由衷感謝，我們還要感謝茲拉坦的職業精神，以及他對洛杉磯和整個北美足球圈帶來難以估量的影響。」

伊布在本人的社交媒體上寫道：「我來了，我看到了，我征服了。感謝洛杉磯銀河讓我再次感覺到活著。給洛杉磯銀河球迷：你們渴望茲拉坦，我給了你們茲拉坦。不客氣。故事還在繼續……現在回去看棒球吧。」

在加盟銀河之前，伊布在曼聯經歷了一次嚴重的膝蓋重傷，並休戰了7個月。2018年3月伊布和曼聯解約後加盟銀河，兩個賽季在MLS常規賽和季後賽一共出場58次，射落53球還送上15次助攻，平均每93分鐘就打進一球，效率可見一斑。

至於伊布的下家，目前還未官宣，而博彩公司給出的賠率來看，昔日意甲豪門AC米蘭以1.33成為第一熱門。如果伊布回歸紅黑軍團，說不準真能助米蘭走出困境。

在7年前的今天，對伊布也是一個值得銘記的日子。

2012年11月14日，瑞典隊在友誼賽中4-2擊敗英格蘭，伊布上演大四喜，包括一個禁區外超遠距離倒掛破門，可憐的喬-哈特又成為世界波背景板。

這粒進球發生在補時階段，當時瑞典3-2領先，哈特出擊到禁區外頭槌解圍沒有頂遠，機敏的伊布在離門將近30碼的地方騰空而起，背向球門踢出一記倒掛金鉤，皮球划出一道美妙的弧線入網，真乃神仙球也！
