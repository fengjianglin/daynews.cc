曼聯清洗三人可節省3700萬！索帥迫切想送走桑切斯，有人要就帶走

2019-11-16 0條評論 15次閱讀 0人點贊

曼聯幾乎拚命想要清洗掉智利邊鋒桑切斯，英媒「The Athletic」報道稱，曼聯是如此渴望徹底斬斷與智利邊鋒之間的關係，以至於只要有俱樂部想要他，給不給轉會費都可以把他帶走。

目前，桑切斯被曼聯租借給國際米蘭一個賽季，原本前4次出場表現不俗，有1個進球和1次助攻，如果繼續這麼下去，國米有望會把他買斷。不料桑切斯卻在國家隊比賽重傷，今年提前報銷。

這樣一來，國米最終是否會永久轉會引進桑切斯，已經打上了大大的問號。曼聯可能要寄望於昔日鋒霸盧卡庫幫忙遊說，比利時國腳本賽季7300萬鎊轉會國米後，進球效率不錯，打入9球在意甲射手榜暫居第二，打破了巴西「外星人」羅納爾多創造的藍黑軍團開局進球紀錄，也幫助球隊只以1分之差落後尤文圖斯。而盧卡庫和桑切斯是最好的朋友，當盧員外與新隊友布羅佐維奇差點大打出手時，就是智利邊鋒在勸架，比利時鋒霸可能說服主帥孔蒂留下桑切斯。

曼聯並不需要桑切斯的轉會費幫補財政，但埃德·伍德沃德和索爾斯克亞希望從工資單中減去他的50.5萬鎊高周薪支出，那等於每年憑空多出了超過2500萬鎊，本身就是一種利好。

除了桑切斯，曼聯明夏至少還會再處理掉兩名老將：老隊長阿什利·揚以及老化的塞爾維亞後腰馬蒂奇。揚教授下賽季就年滿35歲了，他在曼聯的「續命」生涯也會宣告終結；馬蒂奇屆時32歲，但他的覆蓋能力已經顯著下滑，而且跟索帥早已鬧翻，離隊幾成必然。

阿什利·揚的合同明年到期，他基本將會自由轉會離隊，而馬蒂奇雖然合同期也到明年夏季，但曼聯擁有一年優先續約權，可能續約後賣掉，收取一定費用。兩人周薪都是12萬鎊，清理掉他們每年能節省1200萬薪金空間。加上桑切斯的話，曼聯可以通過清洗重新獲得3700萬的工資預算。
