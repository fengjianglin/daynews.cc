外媒聚焦里皮辭職！福克斯：震驚足壇，意媒：里皮冒險之旅已結束

2019-11-15 0條評論 13次閱讀 0人點贊

國足主帥里皮昨晚突然辭職讓球迷多少覺得有些意外，不僅讓新聞發布會記者大吃一驚，而且成為外國媒體的關注焦點。《福克斯體育》認為里皮的辭職震驚足壇，而義大利媒體《都靈體育報》則表示里皮的中國冒險之旅已經結束。

里皮

里皮曾帶領意甲豪門尤文圖斯登上歐冠之巔，並且帶領義大利國家隊在世界盃奪冠，來到中國後又幫助廣州恆大拿到亞冠冠軍，銀狐里皮是世界名帥，已經無需再證明自己。里皮名氣擺在那裡，自然會受全世界各地媒體的關注，特別是義大利媒體。

福克斯體育：2020年FIFA世界盃預選賽中國隊1-2不敵敘利亞賽後，世界傳奇教練里皮選擇辭去中國國家隊主教練職務，這一舉動震驚世界足球。（該報透露里皮在新聞發布非常生氣，拍了桌子）

福克斯體育

都靈體育報：前尤文俱樂部和義大利國家隊主帥里皮在中國的冒險之旅已經結束，中國隊1-2不敵敘利亞隊賽後，里皮選擇辭職。（這家義大利媒體表示此前由於受東亞杯影響，里皮一度成為媒體關注焦點，李鐵將帶領國家隊征戰東亞杯。）

都靈體育報

我們國內球迷和媒體對待里皮的辭職持兩種態度，一部分人認為因為國足球員競技水平不是太高，基本功差，即使是世界名帥也無法改變國足現狀，為里皮鳴不平；另外一部分球迷則認為里皮拿著天價年薪，球隊輸球後直接甩鍋太不負責任，並且從比賽中看不到國足有什麼明確的戰術打法。

艾克森

里皮已經第2次辭去中國國家隊主教練職務，原因都是國足在賽場表現太差，上一次是國足征戰亞洲杯，在面對伊朗時出現多次低級失誤，最終國足慘敗伊朗，里皮賽後新聞發布會同樣直接宣布辭職。這次國足不敵敘利亞，兩粒丟球同樣是我們球員失誤造成的，里皮很難接受球隊這樣的輸球方式，所以再次辭職。

武磊

國足輸球主教練里皮肯定要承擔一部分責任，但也不能讓里皮一個人背鍋，否則國足很難成長。國足球員在場上壓力較大，非常擔心自己犯錯，所以很難發揮出自己的特點，部分球員在國家隊和俱樂部表現如共同兩人。希望媒體和球迷更理性一點，給國足更多空間，不要輸球就大罵特罵。畢竟我們在正賽中已經連續23年沒有贏過敘利亞，所以昨晚國足輸給敘利亞也不是太意外。

希望足協能為國足找到合適的主帥，如果我們能在小組賽後半程多拿分，保住小組第2應該沒有問題，仍然有希望晉級十二強賽。
