“最佳球員”到“最佳教練”！成功轉型優秀主教練的AC米蘭名宿

2019-12-19 125條評論 1,112次閱讀 207人點贊

球員時代他們憑藉精湛的球技，已經是家喻戶曉的明星球員，退役後，這些米蘭名宿逐漸轉型，在教練的位置上取得了同樣優異的成績。

安切洛蒂

安切洛蒂

球員時代曾是米蘭的主力中場，1987-1988賽季至1991-1992賽季效力米蘭，作為球員見證了米蘭王朝的輝煌，1992年退役後轉型成為一名教練，先後執教過雷吉納、帕爾馬和尤文圖斯。2001年回歸米蘭，帶隊又創造了一段難忘的光輝歲月，執教7個賽季，三次進入歐冠決賽兩次捧得歐冠獎盃，正是在安切洛蒂的改造下，皮爾洛成為了世界級的中場大師。但過於執著的使用聖誕樹陣型和固定的人員，也導致後期球隊成績的波動，最終在2009年安切洛蒂與米蘭分道揚鑣。

2009年安切洛蒂來到切爾西執教了兩個賽季，第一個賽季就為球隊帶來了英超冠軍。2011年轉戰法甲接過大巴黎的帥印。2013年執掌銀河戰艦皇家馬德里。2015年挂帥拜仁慕尼黑。2018年重回意甲執教那不勒斯，由於和球隊老闆之間的矛盾，而被昔日的愛將加圖索取代。

執教過意甲，英超，法甲，西甲，德甲，歐洲五大聯賽的豪門教練席都留下了安切洛蒂的身影，他用行動證明他是就是世界級的主教練。

特拉帕托尼

特拉帕托尼

特拉帕托尼是AC米蘭隊的傳奇人物，球員時代效力米蘭12個賽季，奪得意甲冠軍2次，冠軍杯冠軍2次。1973年退役後選擇成為一名主教練，在帥位上他依然交出了出色的成績單。執教生涯共獲得21個冠軍頭銜，率隊拿過意甲冠軍，德甲冠軍，葡超冠軍，奧甲冠軍，歐冠冠軍，歐聯杯冠軍，歐洲超級盃冠軍。他將球員生涯大部分都獻給了米蘭，而教練生涯則大部分留給了尤文圖斯，並帶領尤文圖斯取得了一個又一個的成就。

卡佩羅

卡佩羅

卡佩羅球員時代曾在米蘭效力，退役後轉型成為一名主教練。他傳承了義大利足球的注重紀律，注重防守的戰術，1：0主義的先驅者，歐洲足壇最成功的主教練之一。執教米蘭6年，4次奪得意甲冠軍，1993-1994賽季率領米蘭斬獲歐冠冠軍。2004年在尤文圖斯再次捧起意甲冠軍獎盃，來到銀河戰艦的第一年就拿到了西甲冠軍。離開皇馬後拿起了國家隊的教鞭，英格蘭和俄羅斯都下了他的足跡，但執掌國家隊的履歷遠沒有其在俱樂部時來的耀眼。2017年卡佩羅登陸中超接管江蘇蘇寧，但僅僅過了一年卡佩羅就與蘇寧俱樂部協商解約。

里傑卡爾德

里傑卡爾德

里傑卡爾德作為「荷蘭三劍客」之一，開創了米蘭輝煌的時代，1988-1989至1992-1993效力米蘭5個賽季，為球隊帶來了眾多榮譽，1995年選擇退役。退役後，里傑卡爾德轉型成為一名教練，作為荷蘭隊助理教練和球隊一起在98年法國世界盃上取得了第四名的成績。後來曾執教過鹿特丹隊，但由於成績不佳而離任。此時巴塞羅那拋來了橄欖枝，2003年正式接過巴塞羅那的教鞭，他秉承著荷蘭人全攻全守的足球理念，2004-2005、2005-2006兩個賽季他為巴薩帶來了西甲聯賽冠軍，並在2005-2006賽季一舉奪得歐冠冠軍。繼恩師克魯伊夫之後，造就了巴薩夢二隊的傳奇。但後來也是因為球隊成績下滑而離開了巴塞羅那。

里傑卡爾德輾轉多地，但執教成績都不理想，現在的他已經不再擔任教練。他說過，我已經不再以教練的身份工作了，不想一直做到60歲，我希望做點別的事情了。

舍甫琴科

舍甫琴科

「烏克蘭核彈頭」不僅是綽號，也是對他最恰當的評價，他就像重型武器一樣，隨時有可能炸裂對手的陣地。登陸意甲第一個賽季就以24球拿到了「金靴獎」，效力米蘭7個賽季出場208次，打入127球而載進球隊史冊。2012年舍甫琴科結束了自己18年的職業生涯選擇退役。2016年烏克蘭足協宣布舍甫琴科成為球隊新任主帥。舍甫琴科的執教生涯雖不長，但他執教了烏克蘭隊，在2020年歐洲杯預選賽中有著出人意料的優異成績，前7場比賽中以6勝1平的戰績提前2輪晉級，其中更有2：1戰勝葡萄牙，5：0戰勝塞維利亞的不俗表現。希望在明年的歐洲杯上舍瓦帶領著烏克蘭能越走越遠，創造屬於自己的神話。

還有一部分米蘭名宿雖成為主教練，但還未取得比較有說服力的帶隊成績。
