C羅在皇馬的前因尤文的後果！是誰讓他丟掉了連續10年40+進球？

2019-12-23 1,420條評論 4,956次閱讀 1699人點贊

2019年的意甲比賽結束，尤文圖斯也在今日凌晨的意超杯決賽輸給了拉齊奧，並且C羅無緣連續十年破40進球的記錄！這不禁讓人感慨萬千，也讓球迷懷戀C羅在皇馬的日子！

自C羅離開皇馬也有一年多，憶往昔，現在回想起來也是令人大想不到。當年勞爾等球員算是「廉頗老矣」，也讓皇馬青黃不接，尤其是老對手巴薩營中坐鎮「雙嬌」之一梅西，皇馬也急需一名能夠和梅西旗鼓相當的球員！所以花了9600萬歐元，從英超帶來了新科金球獎得主C羅！

當然，C羅是皇馬積攢了一輩子運氣換來的超值投資！自加盟皇馬以來，C羅共為皇馬打進了450球，場均超過1球！具體如下：西甲292場打進了311球95個助攻！歐冠101場打進105球31個助攻！國王杯30場22球3助攻！世俱杯6場6球2助攻！西班牙超級盃7場4球！歐洲超級盃2場2球！同時為皇馬贏下4座歐冠獎盃！3座世俱杯冠軍！2座西甲冠軍！2座國王杯冠軍！2座西班牙超級盃冠軍！3座歐洲超級盃冠軍！這是C羅9年為皇馬贏下的榮譽與個人數據！

這些冠軍與個人數據就已經表明C羅為皇馬的貢獻，其也僅用9年成為皇馬隊史最佳射手！當然，皇馬也成就了C羅，助其4奪金球獎！其中最惋惜的是18年的金球獎，如果C羅不離開皇馬，大概率還會是他的！這樣也能讓他與梅西在金球獎上平起平坐！或許也是在尤文這個賽季待的不盡人意，加上梅西也在前段時間奪得金球獎讓其引發感慨，不僅一次對身邊朋友表達後悔離開皇馬！畢竟在皇馬有助於他拿到金球獎，而且很難再找出第二個有皇馬這般影響力的球隊！

C羅離開皇馬不單單是C羅的損失，也是皇馬的損失，自C羅離開以後皇馬年度進球也出現斷崖式下滑！當年「儲王」貝爾沒能如願被扶持上去，「新王」阿扎爾身披7號，表現卻不盡人意，遲遲沒能融進球隊！雖然本澤馬此時站出來了，表現也不錯，但還是很難與C羅相提並論！從上一輪的西甲德比就能看出來，皇馬整體不錯，但還是缺少一個如同C羅那樣的決定性球員，且能把握住機會！就連老對手梅西賽前採訪時也說阿扎爾離C羅還很遠，很難替代C羅！

不管怎麼說，現在已經回不去了，只能寄託希望於明年35歲的C羅在尤文能夠得心應手，再次讓自己偉大起來！也讓尤文能夠獲得更多的冠軍讓自己榮譽加身！
