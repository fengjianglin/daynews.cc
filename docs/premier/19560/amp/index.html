安切洛蒂的米蘭本該歐冠5連冠！他答應我，蒙扎升意甲時回來執教

天天要聞
2019-12-18

AC米蘭周一120歲生日，安切洛蒂確定轉投英超埃弗頓，前經理加利亞尼接受《羅馬體育報》採訪分別談到對自己很重要的球隊和教練，「安切洛蒂曾經跟我說，等我們的蒙扎升入意甲時，給他預留位置。希望他說話算數，屆時再聯繫。」貝盧斯科尼去年9月收購意丙蒙扎，加利亞尼仍擔任經理，目前蒙扎領跑意丙A組，下賽季升乙希望極大。

《羅馬體育報》：您和貝盧斯科尼搭檔了40來年，參與米蘭120歲的31年，說說你眼中的貝盧斯科尼。

加利亞尼：我本是一介平民，長得不帥、沒錢、沒名氣，貝盧斯科尼讓我成了名人、讓我有錢並瀟洒起來。我對他不是迷信崇拜，只是敬仰。

他這一生4個身份獲得豐碩成果：一是地產商建設者，1970-79年米蘭大興房地產時，普通開發商只造公寓，貝盧斯科尼在東區造出了高瞻遠矚的米蘭新城，最先倡導和實現了小區周邊的人車分流，汽車和自行車分流。二是媒體人，3個傳媒領域同時下手，私營電視台Mediaset抗衡義大利國家電視台，義大利的默多克；

三是足球戰略家，1987年7月1日開會時說：米蘭明年拿意甲、後年拿歐冠，會場有人竊笑，有人議論他瘋了，卻都成了現實；四是政治家，1993年決定組建政黨，計劃半年內當選總理，結果次年5月11日就贏得大選當了國家元首。」

採訪標題：我的雙重勝利。一是米蘭，二是未來的蒙扎，號稱再造一個米蘭到意甲和米蘭打德比

作為米蘭球迷，最難忘的一天是？

1989年在巴塞羅那的歐冠決賽。我們經營後的第一個歐冠冠軍，賽後，我和老友在巴塞羅那的蘭布拉大道被球迷層層包圍，還被他們舉了起來飛了1.2公里。最黑暗的一天，2005年的伊斯坦布爾之夜。第119分鐘杜德克撲出了舍甫琴科必進的近距離頭槌、然後又撲出舍瓦的補射時，我預感到大事不妙，要輸了。

請您點評一下米蘭王朝的幾個教練，從薩基開始。

就一個詞，真愛。

米蘭奪取歐冠的4個朝代。1960年代的羅科2次歐冠

安切洛蒂呢？

也是真愛。如果人的一生允許有超過一個的真愛，那就讓我繼續說下去，或許不止兩個呢。時至今日我都認為，安切洛蒂執教的米蘭完全可以在冠軍杯五連冠的。我特別想記起2006年的半決賽對陣巴薩，埃托奧被逼得要去當邊後衛才能限制卡福。在足球內容上，米蘭至少不弱於里傑卡爾德那支巴薩，我覺得更高一些。當然，最美好的經歷是2003年，我們半決賽淘汰國米，決賽擊敗尤文，對於一個米蘭球迷而言，最美不過如此。

卡佩羅呢？

脾氣暴躁，但他是個好教練。

加利亞尼的賭註：安切洛蒂？

安切洛蒂之外，和您心靈相通的教練是誰？

阿萊格里！我們倆感情很深。事實也證明，我沒有看錯他。

貝盧斯科尼執意炒掉阿萊格里時，您也很反對吧？私下爭論沒？

不記得了，咱們說點別的話題。

那就說說伊布可能的重返米蘭，您有何意見？

也不聊伊布！拜託了，因為米蘭的轉會現在歸別人管。

說說您和米蘭之間，我們不知道的內幕。

2003年歐冠奪冠後幾天，我們在斯福扎古堡（米蘭地標建築之一）聚會慶祝歐冠冠軍。宴會上，我特意走過好幾張桌子去找一位著名的精神病專家，但我已不記得他的名字了。我對他說：教授，這幾天我老是夢到舍甫琴科罰那記點球，而且夢裡他都罰丟了，您看我的癥狀嚴重嗎？

那他怎麼說？

無可救藥。
