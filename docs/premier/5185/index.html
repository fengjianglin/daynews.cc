梅西再度稱王！12年來，多少位球員在金球獎爭奪中敗給梅羅？

2019-12-04 0條評論 18次閱讀 0人點贊

失去的東西終歸是要奪回來的，在連續三年闊別金球獎之後，梅西終於再度拿到了金球獎。加上這個金球獎，梅西的金球獎總數達到了6座，超越了C羅的5座金球獎，成為世界足壇歷史上榮獲金球獎次數最多的球員。

過去十二年，梅羅二人斬獲了11座金球獎，僅僅在2018年被莫德里奇「偷」走了一次。被梅羅二人壓在身下的不是某一個人，而是全世界的所有足球運動員，是梅羅二人剝奪了他們獲得金球獎的機會。

斯內德（2010年）

2010年的斯內德沒有拿到金球獎堪稱是世界足壇的千古奇冤。

如果說那一年的斯內德敗給了梅西，莫不如說成斯內德敗給了評選機制。2010年是金球獎和世界足球先生合併的第一年，這個獎項也更名為國際足聯金球獎。在獎項的評選過程中，媒體、國家隊隊長、教練各佔三分之一的比重。這個世界本沒有絕對公平，而斯內德就恰恰被那一絲不公平剝奪了金球獎。

2009年夏天剛剛轉會到國米的斯內德，在穆里尼奧的戰術體系下如魚得水。作為中場核心，荷蘭人率領國米拿到了義大利史上的第一個三冠王。

這還沒有結束，世界盃賽場上，斯內德開始率領橙衣軍團大殺四方。雖說決賽上惜敗西班牙，但斯內德雖敗猶榮。作為中場球員，斯內德在整屆賽事上打進5粒進球，並列射手榜的第一位。

斯內德在那一年的表現無懈可擊，但在金球獎最終評選中他輸給了梅西。順便說一句，斯內德是那年媒體評選的第一位。

里貝里（2013年）

里貝里和斯內德有著相同的命運，但他們輸給了不同的人。斯內德輸給了梅西，而里貝里輸給了C羅。

2012-13賽季，海因克斯治下的拜仁慕尼黑如日中天。在德國國內，他們斬獲了德甲冠軍和德國杯冠軍。在歐洲賽場，他們一舉將歐冠冠軍和歐洲超級盃冠軍收入囊中。走出歐洲，拜仁又斬獲了世俱杯冠軍。

在這一年裡，里貝里用近乎完美的表現，帶領拜仁披荊斬棘。里貝里不以進球見長，但他可以像一顆重磅炸彈一樣，撕扯對手的防線，讓對手的防守陣型土崩瓦解。他不僅是隊內的助攻王，更是德甲的助攻王，他的傳球比進球更可怕。

儘管里貝里是媒體投票的第一位，但他還是輸給了更受教練和球員喜歡的C羅。那年C羅在59場比賽中打進69粒進球，但他全年無冠。

諾伊爾（2014年）

如果說斯內德輸給了梅西，里貝里輸給了C羅，而諾伊爾則是輸給了現實。強如卡西和布馮，也未曾染指一個金球獎，從某種程度上說，諾伊爾比他們更接近。

諾伊爾重新定義了門將這個位置，他不拘泥於禁區內，他會走出禁區破壞對手的進攻，甚至走到了中圈位置。很多人會質疑諾伊爾的選擇，但事後你就會發現，諾伊爾做出了最好的選擇。

作為拜仁門將，他幫助拜仁拿到了德甲冠軍和德國杯冠軍。作為德國國家隊門將，他用他的十指關幫助德國隊拿下大力神杯。世界盃7場比賽，諾伊爾把守的德國球門僅被對手攻入4球，德國隊的世界盃冠軍，諾伊爾功不可沒。

如果說唯一一個遺憾，那一定是歐冠半決賽上諾伊爾所在的拜仁輸給了C羅的皇馬，這也為他不敵C羅痛失金球獎埋下了伏筆。

這個時代對梅羅來說，是幸運的，而對其他運動員來說，是不幸的。如果沒有梅西和C羅，可能金球獎的歷史上會多出現十幾人的名字，而如今，這些名字都被梅羅的名字掩蓋。
