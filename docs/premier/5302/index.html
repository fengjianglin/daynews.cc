槍迷盼來春天！ESPN：阿森納高層正在商討，埃梅里最快周日前下課

2019-11-29 0條評論 15次閱讀 0人點贊

智道11月29日訊 來自《ESPN》的最新消息：阿森納的高層正在討論決定球隊主帥埃梅里的未來，他可能在周日（北京時間12月1日22時）對陣諾維奇的比賽前就被解僱，目前球員和球迷都在等待官宣。

在歐聯杯的比賽中，阿森納主場被法蘭克福逆轉，阿森納已經在各項賽事中7輪不勝，聯賽層面5輪不勝（3平2負）。本賽季開始時，阿森納董事會給予了埃梅里充分的信任，更是花費8000萬歐元簽下法甲銀靴佩佩，但經過如此平庸的戰績和球隊將帥失去和睦的情況下，俱樂部高層的耐心已經到了極限。俱樂部給了埃梅里多次證明自己的機會，但埃梅里始終未能找到合適的解決辦法，球隊一直深陷低谷不能自拔。

阿森納俱樂部的高層將會在本周五討論決定埃梅里的未來，而埃梅里目前的助教弗萊迪-永貝里最有可能暫時代理主帥職位。在埃梅里取代溫格18月後，阿森納再一次開始物色新主帥，這次會是新起航，還是再沉淪。曼聯、AC米蘭等多家豪門在不停換帥後，並沒有迎來新生。
