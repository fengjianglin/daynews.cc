女排世俱杯最新積分榜：天津女排2-3之後陷絕境，出線堪憂

2019-12-03 0條評論 14次閱讀 0人點贊

北京時間12月3日，2019女排世俱杯結束了首個比賽日的爭奪，天津女排在2-0領先的情況下崩盤，2-3輸給了意甲勁旅諾瓦拉。廣東恆大女排3-1力克巴西豪門米納斯。此外，瓦基弗銀行與科內利亞諾均戰勝了各自的對手。小組賽首輪戰罷，天津女排排名B組第三，出線陷入被動，廣東恆大女排排名A組第二，掌握著出線的主動權。

A組積分榜

A組：科內利亞諾（義大利）3-1伊薩奇巴希（土耳其），廣東恆大女排（中國）3-1米納斯（巴西）

揭幕戰在科內利亞諾與伊薩奇巴希之間展開，雙方均巨星雲集，都是本屆世俱杯的爭冠熱門。兩隊經過4局較量，埃格努領銜的科內利亞諾3-1力克博斯科維奇領銜的伊薩奇巴希，其中埃格努狂轟29分全場最高，她42扣27中1攔1發，狀態十分火熱，博斯科維奇35扣12中2發，拿到了14分，韓國球星金軟景拿到全隊最高的17分。

同組的另外一場較量，在廣東恆大女排與南美冠軍米納斯之間進行，廣東恆大女排由前世界第一主攻科舍列娃領銜，米納斯則是上屆世俱杯的亞軍。兩隊經過4局較量，恆大女排3-1勝出，在出線名額的爭奪中掌握了主動權。不過，恆大女排後面兩場的對手是科內利亞諾與伊薩奇巴希，恆大女排若想小組出線，至少還需要再贏一場。

首輪戰罷，科內利亞諾與廣東恆大女排同積3分，由於小分稍處劣勢，因此恆大女排排名小組第二，科內利亞諾佔據榜首位置，米納斯與伊薩奇巴希分列三四位。不過，儘管博斯科維奇、金軟景等球星領銜的伊薩奇巴希暫時小組墊底，但是後面兩場只要保持全勝，依然有機會晉級四強。

B組：瓦基弗銀行（土耳其）3-0海灘（巴西），天津女排（中國）2-3諾瓦拉（義大利）

B組的首場較量由瓦基弗銀行對陣巴西海灘隊，朱婷曾經在瓦基弗銀行效力過三個賽季，所以中國球迷對這支球隊一點也不陌生。瓦基弗銀行曾經3次站上世俱杯的最高領獎台，近兩年更是實現了兩連冠。最終沒有任何意外，雙方經過3局較量，瓦基弗銀行3-0橫掃對手，迎來本屆世俱杯的開門紅。

天津女排首戰的對手是意甲勁旅諾瓦拉隊。天津女排不但擁有朱婷、李盈瑩、姚迪、王媛媛等多名國手，而且世俱杯期間，還臨時引進了中國女排主力副攻袁心玥，這令天津女排的網口實力得到加強。面對上賽季歐冠冠軍諾瓦拉，天津女排雖然連贏兩局，但是2-0之後卻出現了崩盤，諾瓦拉連扳3局完成逆轉，天津女排2-3遭遇世俱杯開門黑。

小組賽首輪結束之後，瓦基弗銀行與海灘排名小組前兩位，天津女排排名第三，出線形勢極為不利，後面兩場對陣瓦基弗銀行與海灘隊，天津女排沒有任何退路，否則有可能會被淘汰出局無緣四強。
