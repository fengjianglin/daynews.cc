瓦爾韋德對那不勒斯保持警惕！巴薩期待歐冠

2019-12-17 115條評論 2,148次閱讀 54人點贊

意甲球隊正在經歷一段艱難的時期，但是巴塞羅那的老闆相信他們會威脅到他的球隊在歐洲的雄心，巴塞羅那主教練瓦爾韋德強調，那不勒斯將是歐冠16強中危險的對手。

在歐冠汰賽中，加泰羅尼亞人將對陣意甲球隊，第一回合的比賽將於2月25日在聖保羅舉行。那不勒斯在國內經歷了一個艱難的賽季，他們在周六主場1-2輸給帕爾馬，排名第八，落後領頭羊國米和尤文圖斯18分。

儘管聯賽成績不佳，導致安切洛蒂離開主教練的位置，加圖索被任命，但那不勒斯在歐冠E組保持不敗，是本賽季唯一在比賽中擊敗衛冕冠軍利物浦的球隊。

瓦爾韋德相信如果巴薩想要進入四分之一決賽，他們必須保持最佳狀態，即使他們在季前賽中以4-0大勝。

「那不勒斯是一支危險的球隊，」他告訴巴薩電視台。「很明顯，他們沒有經歷最好的時刻。他們換了教練，必須一點一點地適應。

「我們將會看到他們的表現，以及每個人在二月的表現，但是毫無疑問，我們必須為比賽做好準備。這將是一場氣氛熱烈的比賽，有一個很大的體育場，球迷們就在附近。這是很正常的。

「從這個意義上說，我們已經習慣了在這種氛圍下踢球，如果能體驗一下那不勒斯作為一個城市的一切，那將是非常美妙的。」

與此同時，加圖索已經承諾那不勒斯將在這場比賽中給自己一個強有力的證明。「巴塞羅那是一支偉大的球隊，在兩場精彩的比賽中，這將是一個巨大的挑戰，」他說。「我們不會表現出恐懼。」

那不勒斯將有機會在與巴薩的比賽中檢驗自己，他們將在1月底前與意甲排名前兩名的國際米蘭和尤文圖斯進行比賽。

巴薩有一個即將到來的與皇家馬德里的德比戰，而他們也將在2月的冠軍聯賽前與馬德里競技和瓦倫西亞交手。
