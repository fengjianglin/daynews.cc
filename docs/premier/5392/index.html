歐冠小組賽第五輪第一比賽日，A-D組六隊確定晉級，五隊爭兩名額

2019-11-27 0條評論 14次閱讀 0人點贊

2019-20賽季歐洲冠軍聯賽進行到了第五輪的比賽，ABCD四組的8場比賽率先進行，八場比賽的結果分別為：

  • 莫斯科火車頭0-2勒沃庫森
  • 加拉塔薩雷1-1布魯日
  • 皇馬2-2巴黎聖日耳曼
  • 熱刺4-2奧林匹亞科斯
  • 曼城1-1頓涅斯克礦工
  • 尤文圖斯1-0馬德里競技
  • 貝爾格萊德紅星0-6拜仁
  • 亞特蘭大2-0薩格勒布

這8場比賽結束後，有6支球隊宣布直接進入下一輪，分別是A組的巴黎和皇馬，B組的拜仁和熱刺，C組的曼城，以及D組的尤文圖斯，C組和D組還各有一個名額，有待最後一輪爭奪，而且對於C組來說，現在剩下的3支球隊，包括6分的頓涅斯克礦工，5分的薩格勒布，4分的亞特蘭大，都有機會，D組則是馬競和勒沃庫森爭奪最後一個名額。

尤文vs馬競

本輪的焦點戰，毫無疑問是，皇馬和巴黎，以及尤文和馬競的比賽。

皇馬在本澤馬先進兩球，2-0領先的情況下，被巴黎在81和83分鐘連扳兩球，扳平了比分，雙方握手言和，皇馬也失去了成為小組第一的最後機會。

而尤文和馬競的比賽，尤文圖斯憑藉迪巴拉的進球，1-0擊敗了馬競，同樣也鎖定了小組出線，也讓馬競能否晉級的懸念留到了最後一輪。

B組的亞特蘭大，之前表現一直很差，只取得了1平3負的戰績，本輪終於爆發，主場2-0擊敗薩格勒布，為自己保留下了晉級可能，最後一輪可以全力拚頓涅斯克，當然了，如果薩格勒布能爆冷擊敗曼城，即便亞特蘭大最後一輪獲勝，也無法晉級，但是這樣的情況畢竟概率較小。

亞特蘭大

穆里尼奧帶領的熱刺第一次在歐冠比賽中比賽，上半場19分鐘，熱刺就丟了兩球，看起來岌岌可危，但是穆里尼奧在第29分鐘做出了神奇換人，換下了戴爾，換上了埃里克森，最後出現了一場4球逆轉，阿里，凱恩和奧利耶分別為球隊建工，幫助球隊4-2擊敗奧林匹亞科斯，並且鎖定了出線名額。

但是拜仁客場6-0大勝了貝爾格萊德紅星，萊萬送出大四喜，取得五連勝，所以熱刺確定無法爭奪小組第一，送出大四喜的萊萬目前躍居射手榜第一位。
