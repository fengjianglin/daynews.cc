歐足聯新紀錄！他曾拒絕中超大合同有望超越C羅法爾考

2019-12-18 1,976條評論 3,304次閱讀 228人點贊

北京時間12月18日，格拉斯哥流浪者前鋒阿爾弗雷多·莫雷洛斯以14個進球創造了歐聯杯的歷史。在這之前，還沒有人能在聖誕節前在歐聯賽上打進這麼多球。他也有希望超越同胞哥倫比亞人法爾考創下的歐聯賽18個進球的紀錄。

阿爾弗雷多·莫雷洛斯在小組賽打進6個進球，資格賽打進了8個進球，是歐聯杯頭號射手。14個進球只比法爾考的18球紀錄少了4個，法爾考在2020-2011賽季幫助波爾圖拿到了歐聯杯冠軍。

阿爾弗雷多·莫雷洛斯在去年曾拒絕了北京人和的邀約，人和曾給他開出稅後周薪2.7萬英鎊，800萬英鎊會的轉會費的條件。在接受《布魯之聲》的採訪時，這位哥倫比亞前鋒透露了真實原因：「我的夢想可是踢英超聯賽，因為蘇超和英超有相似之處。國家隊也對我很重要，這是我一直在關注的事情。我知道，如果我能繼續為流浪者進球，那就會迎來新的機會」。
