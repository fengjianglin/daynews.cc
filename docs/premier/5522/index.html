漂亮！國米打破三年不勝魔咒客場大勝苦主，藍黑軍團兩點值得肯定

2019-11-24 0條評論 13次閱讀 0人點贊

在今日凌晨剛剛結束的一場意甲第13輪比賽中，客場作戰的國際米蘭，在都靈的大雨夜中，憑藉馬丁內斯、德弗賴、盧卡庫的進球，在客場是3比0完勝對手。本場比賽面對都靈的勝利，是國米近6次與都靈的對陣首次獲勝，藍黑軍團上一次能夠在客場擊敗紅色公牛，還要追索到2015至16賽季，當時國米在客場憑藉著孔多比亞的進球，在客場是1球小勝都靈。而在此之後的三個賽季時間裡，國米在客場面對都靈的戰績，不是平局就是輸球。紅色公牛近幾個賽季來，已經成為名副其實的國米苦主。

​

因此國米能夠用一場酣暢淋漓的大勝，打破球隊三年客場不勝都靈的魔咒，確實值得藍黑軍團球迷們高興。在拿下苦主都靈後，國米還是以1分的微弱劣勢緊追領頭羊尤文圖斯，繼續給斑馬軍團施壓。

​

本場比賽國際米蘭的將士們，在都靈的大雨夜表現的無懈可擊，尤其是在攻防兩端的發揮，讓球迷們印象深刻。此役孔蒂在戰術的使用上，也同樣十分出色。特別是國米做到了這兩點，是相當值得肯定。

​

此役國米在客場面對都靈這位苦主，藍黑軍團並沒有因為本賽季球隊發揮好戰績佳，排名比對手高，而擺出一副強者的姿態，反倒是把自己放在一個挑戰者的位置上，去不斷衝擊都靈。本場比賽雖然國米在控球率上處於下風，但是孔蒂制定的防守反擊戰術卻十分犀利，馬丁內斯為藍黑軍團首開紀錄的進球，就是最為明顯的例子，貝西諾在中場利用自己的頭槌去攔截都靈的進攻，就直接轉化成一次精彩的助攻。國米此役的三粒進球，特別是馬丁內斯和盧卡庫的進球，其實都是得益於球隊精準的快速反擊。

​

再加上國米兩大鋒線巨星馬丁內斯和盧卡庫，延續著自己的良好狀態，再度在比賽中聯袂上演進球好戲，從而確保了藍黑軍團的攻擊力。兩人在前13輪意甲聯賽中，已經聯手打入了16粒（盧卡庫10球，馬丁內斯6球）意甲進球。本場比賽國米雖然全場射門只有13腳，射正次數也只有5次，但是得益於前鋒們極其高效的表現，以及全隊相當合理的戰術制定，3次攻破都靈的大門，球隊在進攻端的表現當然值得肯定。

​

除了在進攻端令球迷們印象深刻，本場比賽國米將士們在防守端的表現同樣值得表揚。此役是藍黑軍團各項賽事近6場比賽中，第一次實現零封對手。國米在防守端的堅固表現，讓球迷們再次看到賽季之初，那支防守固若金湯的藍黑軍團。其實此役國米能夠在客場零封都靈是相當不容易，畢竟都靈擁有貝洛蒂、維爾迪、扎扎等一大批進攻好手，但是這一次，國米後防線付出了4張黃牌的代價，還是成功做到零封對手。這一點比起國米的三分，比起盧卡庫和馬丁內斯連續不斷取得進球，還更加令藍黑軍團的球迷們興奮。畢竟在意甲的賽場上，想要衝擊最後的冠軍，防守必不可少。

​

打破三年不勝的魔咒，擊敗苦主順利拿下3分，繼續緊追尤文，國際米蘭本周的表現，自然是值得肯定！！！
