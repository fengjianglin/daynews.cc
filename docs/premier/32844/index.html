名聲沒臭！里皮辭職僅40天就收到邀請，意甲保級隊找他救火

2019-12-23 278條評論 1,834次閱讀 731人點贊

據義大利《羅馬體育報》12月23日報道，目前帥位空缺的佛羅倫薩向前中國隊主帥里皮發出了邀請，希望里皮能帶隊到本賽季結束，但遭到了銀狐的拒絕。這也是里皮離開國足後收到的第一份報價。

佛羅倫薩本賽季在意甲賽場上戰績不佳，4勝5平8負排名聯賽第14名，只比降級區高出4分。就在12月21日，佛羅倫薩宣布主帥蒙特拉下課。解僱蒙特拉之後，佛羅倫薩立即展開了新帥的選拔工作，包括前巴黎聖日耳曼主帥布蘭科、前國米主帥斯帕萊蒂、前瓦倫西亞主帥馬塞利諾等名帥都在考慮範圍之內。

蒙特拉剛剛下課

不過上述幾位教練各有各的短板，義大利透露，佛羅倫薩俱樂部主席科米索最中意的人選其實是里皮。佛羅倫薩已火速聯繫了里皮，希望里皮能帶隊到本賽季結束也就是明年夏天，然後再變換角色進入俱樂部管理層。但里皮對於這種救火性質的執教沒有太大興趣，已經拒絕了佛羅倫薩的邀請。

另一方面，里皮曾多次在接受採訪時強調，他不會再執教俱樂部，只會考慮國家隊。因此，里皮這次拒絕佛羅倫薩也在意料之中。里皮在11月15日辭去國足主帥一職後，這40天一直都在義大利休息，除了偶爾參加活動和接受採訪外，幾乎快被中國球迷所淡忘。不過以里皮的名氣和水平，相信接下來還會接到不少球隊拋來的橄欖枝，他何時能夠再出山，也是一個懸念。
