【歐冠】16強全部產生和抽籤規則，以及可能出現的強強對話。

2019-12-12 2,394條評論 4,103次閱讀 1110人點贊

北京時間12月12日凌晨4點整，歐冠小組末輪展開爭奪全部結束。至此歐冠16強全部已經產生。

a組小組第一是法甲巴黎，小組第二西甲皇家馬德里，

b組小組小組第一德甲拜仁慕尼黑，小組第二英超熱刺，

c組小組第一英超曼城，小組第二意甲亞特蘭大，

d組小組第一意甲尤文圖斯，小組第二西甲馬競

e組小組第一英超利物浦，小組第二意甲那不勒斯

f組小組第一西甲巴塞羅那，小組第二德甲多特蒙德

g組小組第一德甲rb萊比錫，小組第二法甲里昂

h組小組第一西甲瓦倫西亞，小組第二英超切爾西

至此全部16強產生，全部是五大聯賽出品，五大聯賽再一次壟斷歐洲冠軍聯賽。

西甲和英超四支隊伍參加全部晉級，所以競技水平也是這兩個聯賽比較激烈，意甲領頭羊國米連巴薩2隊都踢不過，可見一般。

而接下來晉級8強將抽籤決定，而抽籤的規則是小組第一對其他小組第二，然後同組同國聯賽不會抽上，這樣抽籤我們來看一下可能產生的結果。

a組第一的巴黎來自法甲，不會對上同組皇家馬德里和同國第二里昂，機會最大的抽籤結果可能是英超、也是最壞的結果、魔力鳥的熱刺和蘭帕德的切爾西、然後西甲馬競和德甲多特蒙德都是比較差的簽。差簽機會大

b組第一的是拜仁慕尼黑，不會對上同組的熱刺和同國聯賽的多特蒙德。其他結果對拜仁問題不大，除了皇家馬德里。差簽機會一般

c組第一曼城，不會對上同組亞特蘭大和同國聯賽切爾西和熱刺，其他結果都沒問題，最精彩還是遇到皇家馬德里。差簽機會一般

d組第一尤文圖斯，不會對上同組馬競和同國聯賽亞特蘭大，其他結果就是c羅打老東家皇家馬德里，或者英超熱刺和切爾西，差簽機會非常大。

e組第一利物浦，不會對上同組那不勒斯和同國聯賽切爾西、熱刺，唯一差簽還是皇家馬德里。差簽機會一般。

f組第一巴塞羅那，不會遇到同組多特蒙德和同國馬競、皇家馬德里，其他小組第二對有梅球王的巴薩只有對手怕，巴塞羅那沒機會差簽。差簽機會沒有。

g組小組第一德甲的rb萊比錫，不會遇到同組裡昂和同國聯賽拜仁慕尼黑，對德甲的rb萊比錫來說，是現在小組第2非常想遇到的對手了。差簽機會極大。

h組的是西甲的瓦倫西亞，不會遇到同組的切爾西和同國的馬競、皇家馬德里，遇到熱刺和多特蒙德算半斤八兩。差簽機會一般。

小編感覺，西甲皇家馬德里和歐冠亞軍魔力鳥的熱刺、和西蒙尼的馬競是小組第二中小組第一最不想碰到的。而蘭帕德的藍軍，實力還是不夠。而小組第一的利物浦和巴黎差簽機會都非常大。

朋友們，抽籤結果如何都好，你們看好誰晉級8強呢？我看巴塞羅那晉級8強之路會非常順利，你們感覺呢？
