切爾西留不住坎特，四隻豪門伸出橄欖枝，尤文是很好的選擇

天天要聞
2019-12-17

據西媒El desmarque報道，切爾西球星坎特想要離隊尋求新的挑戰，這位法國中場大將已經有點厭倦英超的挑戰，他已經在萊斯特城和切爾西拿到兩次英超冠軍，現在他的目標是要拿到歐冠，但是正在重建之中的切爾西很難滿足他這個想法，再加上他本人希望能夠尋求新的挑戰，離開英超已經成為他下一站的目標。

在冬季轉會窗即將到來之時，已經有多支豪門向這位世界第一後腰拋出了橄欖枝，皇家馬德里主帥齊達內非常欣賞他這位法國同胞，希望能夠將帶到皇馬，只不過皇家馬德里的轉會事宜一向都由老佛爺負責，在佛羅倫蒂諾眼中，皇家馬德里的後腰已經擁有卡塞米羅了，球隊似乎並不太需要另一位世界級後腰，更重要的是切爾西方面給出的報價讓老佛爺不滿意，一億歐元的轉會費讓老佛爺並不想考慮他。

皇家馬德里的死對手巴塞羅那也想要這位勤勉的法國人，本賽季巴塞羅那的後腰布斯克茨狀態起伏不定，作為為數不多的組織型後腰，布斯克茨已經沒有辦法為巴塞羅那帶來更多的改變，而德容的到來更多的是替代曾經哈維的中場組織者位置，巴塞羅那方面需要一位頗為組織性的後腰，坎特雖然以防守著稱，但是最近兩個賽季在切爾西也培養出了他優秀的進攻能力，攻守具備的坎特可以適應中場很多位置，可能不一定適合巴薩的傳控打法，但是他可以滿足巴薩的很多需求。

個人認為最適合坎特的俱樂部是尤文圖斯，斑馬軍團中場控制力實在是太弱了，皮亞尼奇一個人在中場又要組織又要防守，實在是太累了，這也讓前場的C羅，迪巴拉等人經常回撤到後場參加防守任務，限制住了尤文圖斯的三叉戟，坎特的到來可以完美的解放尤文圖斯的陣容，為尤文圖斯帶來更好的中場防守，這樣皮亞尼奇就可以專註於組織，而迪巴拉和C羅就徹底解放，尤文圖斯的實力將出現一個飛躍式增長。

拜仁慕尼黑方面也希望得到這位世界級的後腰，本賽季拜仁慕尼黑在德甲可以說是太慘了，現在排名僅僅是德甲第5位，這不僅僅是因為球隊換帥的動蕩，更重要的是拜仁慕尼黑的防守出現了很大的問題，現在的拜仁就缺少坎特這樣的工兵型中場，如果法國人願意來，拜仁慕尼黑的中場防守問題將解決，他們的球隊將變得更加強大，聯賽冠軍唾手可得，也將成為歐冠賽場上最大的競爭者。

優秀的坎特是當今足壇少有的全能型中場大師，作為世界第一後腰的中場球員最近兩個賽季在尤文圖斯展現出了出色的進攻水平，個人認為他的全能性和對球隊的重要性甚至在大部分中場球員之上，而有些防守偏弱的豪門球隊在得到他之後將如虎添翼，同時低調謙遜的性格能讓他很快地融入各支球隊，這也是為什麼眾多豪門對他拋出橄欖枝的原因，他的選擇實在是太多了，個人認為最適合他的是尤文圖斯，拜仁慕尼黑也不錯，巴薩的風格和他還是不相符合，皇馬方面有卡塞米羅，不太需要它。
