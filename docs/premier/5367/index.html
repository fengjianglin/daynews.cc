捷克國安PK義大利蘇寧，國米要用實力說話

2019-11-27 0條評論 15次閱讀 0人點贊

對於國際米蘭來說，這場比賽的三分將決定國米整個冬窗的命運，目前在小組僅排第三的國米，本場若是不能全取三分，其歐冠出線的概率將會只存在理論上可能，若是不能打入歐冠淘汰賽，球隊冬窗的轉會、續約事宜將受到不小的打擊。

據米蘭當地媒體的報道，國米若是今年闖入淘汰賽球隊將會獲得4600W歐元的獎金收入，國米的CEO馬洛塔也表示球隊和孔蒂在溝通轉會前景，近期又傳出埃里克森被國米這賽季的成績所吸引有投靠可能，後衛什克，也表示將會拒絕巴薩邀約，留在國米繼續效力，而這一切都要先以歐冠淘汰賽為前提下，這場事關國米未來的一戰，國米方面受外界操控的可能將會極小，應會用實力說話。

布拉格斯拉維亞所在城市和北京已經解除城市友好關係，箇中原因無法細說，只能說布拉格所在的這個城市新上任市長很不友好，讓其對這支隊也無法有支持之意。而此前布拉格與國米首戰和逼平巴薩皆屬於不正常賽果，這場關乎國米生死，用實力說話的前提，布拉格應無勝可言。

本場看法：國際米蘭（-0/0.5），國內遊戲：負（2.09）。
