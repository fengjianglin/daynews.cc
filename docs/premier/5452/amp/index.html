橙衣軍團強勢復甦，青年才俊嶄露頭角，未來可期

天天要聞
2019-11-26

荷蘭隊的崛起不是沒有原因的，在對於年輕球員的培養上，荷蘭足協的政策也是為青年小將提供了很大的幫助，在荷甲優秀的青訓體系下成長的小球員可以來到歐洲任意一家青睞其的球隊效力，同時俱樂部不能夠收取高昂的費用。這一做法雖犧牲了荷甲部分俱樂部的轉會收入，但通過犧牲本國俱樂部的方法，為本土出身的青訓小將提供更大平台與空間發揮自己的才能，球員既能夠選擇在本土聯賽穩定發揮自己實力，也能夠到更高一層次的平台尋求挑戰，這樣的舉措無疑是世界各國足協應當學習的。

荷蘭隊慶祝進球

荷蘭足球之所以成功優秀的青訓體系以及龐大的群眾基礎的功勞不可或缺。在優良的體系下，本世紀荷蘭隊先後培養出范佩西，斯內德、羅本、范德法特等優秀世界級球星，更是幫助荷蘭隊在世界大賽舞台上大放光彩，在2010年更是殺入世界盃決賽，但再次無法加冕冠軍使得荷蘭坐實了無冕之王稱號。在隨後的2014年世界盃里，在范加爾的帶領下荷蘭隊再度過五關斬六將，最終點球大戰敗給阿根廷，但擊敗東道主巴西隊贏得14年世界盃季軍。也正是兩年世界大賽的出色表現，球隊的主力球員漸漸老去，球隊出現了更新換代問題，球隊遲遲無法找回曾經大殺四方的狀態，也導致了荷蘭隊接連無緣歐洲杯與世界盃兩項重要賽事。

年輕小將踢出不錯表現，中生代球員逐漸成熟

正當人們認為荷蘭隊將隕落之時，一眾年輕球員與中生代球員的崛起給了荷蘭隊復興的希望。在阿賈克斯陣中，德容以及德林赫特兩位年輕小將接連爆發出出色的狀態，幫助阿賈克斯在上賽季歐冠賽場殺進四強，再度讓人們看到了荷蘭足球復興的希望。

小將德容在中場出色的發揮也受到了巴塞羅那的青睞，本賽季頂著7500萬身價轉會至西甲豪門。德容坐鎮中場主要參與球隊的梳理進攻以及穩固球隊中場的統治。在阿賈克斯時期，在德容的中場梳理下，球隊在中場的控制力有目共睹。在能夠穩定中場控球的情況下，能夠為前場提供有效的火力支援這也是德容適合巴薩的關鍵原因。隨著伊涅斯塔與哈維的離去，布斯克茨的老去，巴薩在中場渴望得到新三人組鞏固的地位。而德容的到來似乎也讓巴薩球迷看到了希望，場均出色的傳球成功率，有他在場也為梅西提與前場球星供了更多的火力支援，同時在疏導球隊中場控球上也起到了立竿見影的作用。

不論是巴薩還是荷蘭國家隊，德容的出色表現已然獲得了主帥的肯定。反之，金童獎得主德林赫特似乎發展得並不順利，在高價轉會尤文圖斯後屢屢出現的失誤似乎並不能讓人們對他的表現信服。也許是壓力過大，球隊原本是想讓老將基耶利尼等老大哥在場上對其進行指導提升他的能力，可不料傷病使得老將缺陣，而德林赫特也只能獨當大任坐鎮尤文後防線，壓力也迫使他犯下錯誤，對於手球問題是他的致命缺點，如何在高身價的前提下穩中求勝提高自己的表現是至關重要的。

好在，在荷蘭國家隊由利物浦大將范迪克帶領下，德林赫特的表現也有所好轉。畢竟，利物浦鐵衛在渣叔的調教下已然已經成為了當今世界最佳中衛，目前還與梅西C羅爭奪本年度的金球獎，在擊敗了梅西c羅獲得歐足聯最佳球員獎項後，范迪克勢必將向金球獎發起衝擊。

在渣叔治下，范迪克在後防線上宛如定海神針般的存在，他的到來徹底改變了利物浦後防線薄弱的問題，也正是在場上充滿血性的領導能力與拼勁使得利物浦的後防線以及球隊整體實力能夠得到質的飛躍。無論是國家隊還是俱樂部，范迪克都能成為球隊的後防領袖，在他的帶領下我相信德林赫特一定能夠在場上獲得更豐富的經驗。

上下兩圖：范迪克堪稱教科書一般的一防二

當然，范迪克的作用也不僅僅局限於防守端，在進攻端他也能夠成為荷蘭隊的關鍵人物，身材與體質上的優勢使得他能夠在球隊的定位球戰術他起到至關重要的作用，在球隊打不開局面時，憑藉角球或定位球使用頭槌破門。在後場的進攻組織上，范迪克也有著一定的組織進攻能力，同時在隊友接球位置不佳的情況下，有著一腳腳下能力的范迪克也能夠獨自帶球推進尋求機會。這樣能功能守的寶藏球員無疑是荷蘭隊能夠再度崛起的關鍵人物之一。

巴薩傳奇到來，球隊迎來改革轉變

荷蘭隊在後范加爾時期聘請了巴薩傳奇科曼執教球隊，科曼延續了荷蘭隊傳統的球隊陣型4-3-3，在嘗試了多名中場球員後，最終將年輕的德容定位中場核心球員，而德容出色的處理球與擺脫能力能夠使他在這個位置發揮出更加出色的作用。而在中場方面維納爾杜姆與年輕的德容搭檔，二者風格不一，在球風上維納爾杜姆在渣叔的調教下更加具有拼勁與血性，兇猛的上搶使得荷蘭隊中場的保護更加安全。同時在進攻端，維納爾杜姆能夠具備出色的前插能力，使得他能夠通過跑位為在進攻端的球員提供掩護的同時，發揮自己的進攻才能。

但相較於當年大殺四方的荷蘭隊目前荷蘭隊最為缺少的便是鋒線能力出眾的球員。當年范加爾治下的荷蘭隊之所以能夠表現出色，進攻端的火力不可忽視，在球隊的鋒線上范佩西坐鎮中鋒，身體出眾的他能夠起到球隊的支點作用爭搶高球，同時在定位球戰術上也能發揮出自身的價值，門前的終結能力更是范大將軍最為恐怖的地方。

而身後的小飛俠羅本便是以個人能力出名，在邊路有著一條”羅本走廊”，憑藉個人能力帶球突破之後內切射門，是羅本最為喜歡的破門方式。前腰位置便是由斯內德坐鎮，巔峰時期的他更是國米三冠王的核心球員，在2010年折戟西班牙後外界紛紛為他發聲，世界欠他一座金球獎獎盃。斯內德通過自己出色的視野以及傳球能力，往往能夠精確的制導將球傳送到鋒線兩位殺星腳下，這也是我們笑稱當然荷蘭隊”三人進攻，七人防守”的原因。

鋒線缺乏傳統中鋒，德佩還需提高能力

反觀現如今的荷蘭隊的鋒線球員德佩，雖然在曼聯沒有受到重用沒有發揮出自己荷甲高光的表現，但是轉會裡昂後逐漸成熟的他，也慢慢發揮出自己的天賦逐漸坐穩了荷蘭隊與里昂的主力鋒線球員位置。但他的差距與羅本范佩西還是很大的，單單是門前終結能力上，德佩時常弄丟機會便無法與范佩西相提並論，而在帶球突破方面也不及小飛俠羅本，時常的粘球使得他往往忽略了隊友的跑位，最終導致球權被斷。

荷蘭目前缺乏鋒線當家球星，在馬倫等年輕球員還需要磨鍊的情況下，德佩應當挑起重擔，面對壓力迎難而上便是突破自己的最佳方法。好在，雖然德佩無法與前輩相比，但他較為穩定的發揮也成為了如今荷蘭隊的鋒線首選，科曼將其放在偽九號的位置上，看到了德佩的組織推進壓迫能力後，科曼大膽嘗試重用德佩啟用無鋒陣，雖然沒有傳統中鋒對長傳球的接應存在劣勢，但留給德佩的時間還有很多他完全可以通過自身努力適應並熟悉球隊需要的位置。

德佩在前場細膩的腳下技術和速度上的優勢給了這支荷蘭隊更充沛的活力，出身主打邊鋒的他能夠快速的邊中切換攪亂對方防線。當然在球隊鋒線球員狀態不佳時，維納爾杜姆的作用就格外重要，具備前插能力他是能夠減輕鋒線球員負擔的有效存在。

同時，科曼治下還招進了傳統荷蘭中鋒呂克德容，在德佩與荷蘭進攻端遲遲打不開局面時能夠切換鋒線球員風格，創造新的機會。面對像法國德國這樣高強度的比賽中高球的爭搶也格外重要，這便是德容出現在荷蘭隊大名單的重要性原因。

陣容厚度不及英格蘭法國，主帥科曼成關鍵

在青年才俊接連噴涌而出的荷蘭隊，缺席兩屆大賽後再度挺進歐洲杯，這支荷蘭隊的表現無疑是我們值得期待的。但相比較與目前法國隊與英格蘭隊的豪華陣容，荷蘭隊確實還欠缺不少，兩支傳統豪強在青年才俊嶄露鋒芒的同時有成熟的中生代球員帶領著小將前進，在這點上荷蘭隊確實遜色於英格蘭和法國。這也就考驗了主帥科曼的執教能力，如何將荷蘭隊苦惱的鋒線問題得到有效地解決，球隊在時隔多年後重回歐洲賽場能夠為我們帶來怎樣的表現，也是我們值得期待的。
