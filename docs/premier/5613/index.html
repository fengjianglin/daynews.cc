郵報盤點本賽季尚未爆發的高薪球員：庫鳥、格子、桑切斯在列

2019-11-20 0條評論 13次閱讀 0人點贊

直播吧11月20日訊 本賽季至今，許多令人熟知的球星並沒有爆發出令人期待的表現，《每日郵報》就盤點了本賽季尚未爆發的那些高薪球員。其中包括有租借拜仁的庫蒂尼奧、租借國米的桑切斯以及今夏加盟巴薩的格列茲曼。

《郵報》盤點的未爆發高薪球星如下。↓

庫蒂尼奧（巴薩租借至拜仁）

由於拜仁本賽季目前表現不符合球迷的期望，隊內球星庫蒂尼奧也遭受著一定的批評。在過去一個月，庫蒂尼奧遭受了不少媒體和球迷的抨擊，在拜仁1-5慘敗法蘭克福一役中，庫蒂尼奧被形容為「隱形」。那場比賽也導致時任主帥科瓦奇遭到解僱。據郵報計算，庫蒂尼奧在拜仁的每場比賽需要花費球隊約56.3萬英鎊。

桑切斯（曼聯租借至國米）

在賽季初被曼聯租借至國際米蘭時，桑切斯一度被外界期待有職業生涯的「第二春」，然而智利人並沒有完全釋放出當年在意甲大殺四方的威力，如今不幸遭受傷病困擾。桑切斯的國米生涯目前來說還不能算得上好。

格列茲曼（巴塞羅那）

夏季重磅加盟巴薩後，格列茲曼的起步令球迷感到有些沮喪。法國人目前只取得了兩粒進球。以梅西為核心的攻擊線似乎讓格列茲曼有些水土不服。

皮亞特克（AC米蘭）

皮亞特克在今年一月份加盟米蘭後有著夢幻般的開局，前五場比賽打進六球。但本賽季的表現一直無法與上賽季相匹配。本賽季12場比賽，波蘭射手僅打進3球，效率下滑嚴重。

約維奇（皇馬）

6000萬先生自加盟皇馬以來一直沒有令人信服的表現。上賽季在法蘭克福打進27粒進球使皇馬球迷對他本賽季寄予厚望，但本賽季至今他交出的答卷僅為一粒進球。而現如今又要遭受髖關節傷病的困擾，21歲的約維奇在皇馬可謂道阻且長。
