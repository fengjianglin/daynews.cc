笑到最後才算贏，歐冠黑馬五輪第一出局，意甲勁旅五輪墊底晉級

天天要聞
2019-12-12

歐冠小組賽全部結束，沒有太多冷門的結果，三大聯賽中只有兩支球隊出局。意甲的國際米蘭被巴塞羅那以及多特蒙德淘汰，德甲的勒沃庫森被尤文圖斯以及馬德里競技淘汰，主要是因為他們分組的關係造成的。

英超和西甲成為大贏家，各有四支球隊晉級歐冠16強。英超方面有利物浦、曼城、切爾西和熱刺，西甲有巴塞羅那、皇家馬德里、瓦倫西亞和馬德里競技，本賽季在德甲威風八面的萊比錫紅牛也順利以小組第一出線。

值得一提的是，本屆歐冠16支出線隊伍都來自五大聯賽，而且八個小組的第一名實力都不弱。而無法成為小組榜首的皇馬、切爾西、熱刺和馬德里競技，將會在淘汰賽的第一輪遭遇硬仗。無論他們抽籤的對手是誰，都會出現精彩的強強對話。很顯然，各個小組的頭名球隊都不願意碰到這幾隊。

在各個小組出線的球隊當中，拜仁以六場全勝的姿態強勢晉級，這與他們在德甲面臨壓力的情況大為不同。而亞特蘭大則只有7個積分，卻能順利出線。

還有一支球隊值得關注，上屆歐冠黑馬阿賈克斯，10個積分都無法成為小組前兩名，這也是歐冠歷史上成績第二好的未晉級球隊，僅僅比多年前，三隊同積12分而被淘汰的那不勒斯要低分。

俗話說，笑到最後才算贏，亞特蘭大在小組的前五輪都是排名墊底，他們在開局的前三場比賽，分別不敵小組的三個對手，其中兩場比賽是凈負四個球，首循環結束後，他們積分為0，凈勝球高達負9。

不過最後三場，亞特蘭大知恥而後勇，主場逼平曼城，加上擊敗薩格勒布以及礦工，最後一輪奇蹟般地從小組第四變成小組第二，最終獲得出線權。

反觀阿賈克斯，他們開局十分順利，延續著上屆歐冠的強勢，連續兩場3：0分別戰勝里爾和瓦倫西亞，似乎出線不會有任何問題。甚至說，他們在最後一輪開始之前，還是排名小組第一。可惜的是，末戰在主場一球小負瓦倫西亞之後，不但失去了小組第一，而且還把出線的機會交給對手。

亞特蘭大連續五輪小組墊底，最終出線，阿賈克斯連續五輪小組榜首，最終出局，這就是足球的魅力。足球是圓的，不到最後時刻，誰也不知道結果是怎樣的。

亞特蘭大和阿賈克斯一喜一悲的故事證明了，多看足球可以領略到更多的哲理，三連敗不等於沒有希望，甚至也不需要三連勝來挽救。人生的種種悲歡離合，不也是這樣嗎，一時的得失並不算什麼，把歡笑留到最後的結局吧，香檳不要過早就打開。
