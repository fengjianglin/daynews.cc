意甲 11/10 19:30 卡里亞里vs佛羅倫薩

2019-11-10 0條評論 18次閱讀 0人點贊

卡利亞里上賽季差點降級,今年走勢也頗為神奇,開局場只拿1分後,突然破蛹成蝶,幾乎場場贏數據。目前意甲第六,得失球比例非常漂亮,進18球失10球,這個數字表現出攻守均衡,接近尤文圖斯的進20球失12球。相對於各大聯賽脫胎換骨球隊,萊斯特、門興、切爾西鄒換了新帥。卡利亞里主教練還是去年差點降級的羅蘭,筆者只能理解為,羅蘭在去年接手帥位,艱難度過奢合期,終於在本賽季找到感覺。

卡利亞里的比賽總是讓觀眾非常興奮,團隊力量鼓舞人而佛羅倫薩,本賽季也有進步,但進步來自於里貝里的加盟。總體上打強硬球隊,很難得勝。2隊分列意甲第6、第8位,差距並不大,客隊名氣略勝一籌。數據耐人尋味,主+0.25中利,變為平手,主隊超高疑點在於後數據,始終在主隊平手1.12+超高利,和+0.25超低利來回波動,這兩個位置球迷接手都非常不舒服。上平手,幾乎是不可能贏的數字阻擊,上受上,獎金低的可憐。而最終,恐怕球迷會選擇標準數居,+0.25接手。筆者認為,這是一個心理戰,數據面在兩者之間完全能開出中利、中高利合理數據。本場客隊里貝里停賽,主隊主場戰績雖不如客場優秀,但氣勢玨盛,依然有能力不敗。

卡利亞里+0
