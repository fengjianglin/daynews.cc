怎麼哪裡都有你？細數那些與國米傳緋聞的球星

2019-11-22 0條評論 17次閱讀 0人點贊

本賽季開賽前，國際米蘭引進孔蒂擔任國際米蘭的主帥，很多人都對此表示質疑，認為孔蒂的性格並不能夠駕馭國米。事實上，他們都想錯了，孔蒂帶領的國際米蘭像暴發戶一般，直接躍進聯賽前二，誓在打破尤文圖斯意甲的連冠。

因為本賽季國米暴發戶一樣的表現，加上神秘的中國財主入主國米，這給了國際米蘭施展野心的機會，國米主帥孔蒂想要引進許多強援保持國米現在火熱的狀態，並且想要衝擊更高的平台，國米開始了瘋狂的引援計劃。

明年冬窗即將打開，在冬窗打開之前，足球市場上就爆出了許多實力強勁的球員的轉會消息，有趣的是，轉會消息中，幾乎每一條都有國際米蘭的身影。

最火的大概是宣布重返歐洲賽場的伊布了吧，當伊布宣布離開美職聯重返歐洲時，國際米蘭便向伊布瘋狂放電，雖然伊布已經年邁，但是這並不影響伊布的狀態，「上帝」仍然是伊布的最佳代名詞。

除了伊布，法國國腳吉魯也爆出了加盟國米的信息。當吉魯表示不願繼續為切爾西效力時，多特蒙德、阿森納等球隊都表示願意引進吉魯，隨後國米也向吉魯表示了自己的誠意。根據媒體報道，國米戰勝了許多的競爭對手，吉魯表示願意明年冬窗加盟國米，為它效力。

國米他飄了，豪門的球員他都敢搶。巴薩球員拉基蒂奇不被巴爾韋德賞識，他本人也表示對巴薩表示失望，想要離隊，巴薩也對此表示了默許。尤文圖斯得知此事後，便願意引進拉基蒂奇，為了獲得拉基蒂奇的芳心，尤文有意出售馬圖伊迪給拉基蒂奇讓位。而國米也對拉基蒂奇展開了攻勢，表示願意以更多的錢簽下拉基蒂奇。不知道拉基蒂奇心動了嗎？

不僅願意出錢，國米還想到了免簽。國米把眼光放到了維爾通亨的身上，32歲的他打中衛，因為穆鳥的入主，可能年齡過大的維爾通亨將會被清洗，他與熱刺的合約將在明年結束，因此國米想要和維爾通亨進行溝通，將在明年夏天免簽下維爾通亨。

除了，以上球員，國際米蘭還於其他球員「有染」，讓我們期待轉會窗口的打開，國際米蘭將會在市場上有什麼樣的表現吧。
