意甲賽事:佛羅倫薩vs羅馬，趁你病要你命，羅馬欲一雪前恥

2019-12-21 1,169條評論 2,177次閱讀 1097人點贊

高開低走

佛羅倫薩在開賽中表現強眼，雖面對亞特蘭大、尤文圖斯、米蘭曾取得6場不敗，但自10月底以來狀態糟糕，已經6輪聯賽不勝，上輪聯賽憑藉小將弗拉霍維奇在比賽結束哨音吹響前的一粒進球逼平領頭羊國際米蘭，終於終結4連敗。目前積分17，高出降級區僅4分，保級並不穩。後方不穩是球隊狀態低迷的主要原因，糟糕的防守使球隊最近9輪聯賽場均丟2球以上。值得注意的是最近的四次交手佛羅倫薩2勝2平，上賽季更是出現了7:1主場血洗羅馬的場面。此戰佛羅倫薩延續記錄，奪分提氣意圖明顯。

士氣高昂

羅馬近期可謂士氣高昂，最近6場在丟4球的情況下轟入14球，4勝2平保持不敗，這要得力於哲科、小克魯伊維特和扎尼奧洛等人多點開花，而新融進球隊的小曼奇尼和斯莫林對防守端的貢獻同樣功不可沒。羅馬目前積32分排在聯賽第四位，比第五卡利亞里高出3分。不利的是中後場傷病較多，莫林、小克魯伊維特、克里斯坦特、帕斯托雷、扎帕科斯塔、桑頓都將因傷休戰。面對狀態不佳的佛羅倫薩，羅馬自然想一雪前恥。
