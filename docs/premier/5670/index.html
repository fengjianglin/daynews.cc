恭喜曼聯！或500萬購知名鋒霸，與尤文高管面談，商討博格巴去向

2019-11-15 0條評論 19次閱讀 0人點贊

索帥執教曼聯的這個賽季，也許最為頭疼的就是球隊的前鋒進球問題，尤其是前段時間馬夏爾的受傷，更是讓他覺得壓力很大。他所倚重的拉什福德，雖然偶爾有著神奇的發揮，但是大部分的時間還是不怎麼靠譜，而球隊也始終不能有一個穩定的輸出前鋒。

在曼聯這個問題最為艱難的時候，媒體也不斷將一些前鋒和曼聯聯繫在一起，曼朱基奇，伊布，甚至哈蘭德都和曼聯有過關聯了，而就現在的情況來看，最為可能的選擇還是曼朱基奇 。

義大利媒體透露稱，現在帕拉蒂奇，這位尤文的足球主管，現在已經前往了曼徹斯特 ，針對曼朱基奇的轉會事宜，和曼聯的高層進行會面。在薩里成為尤文主帥之後，這名球員到現在都沒有為球隊一線隊出場過，形勢不容樂觀。因此現在帕拉蒂奇的想法也是儘快將曼朱基奇推銷給曼聯，這也是紅魔希望看到的。

現如今，曼朱基奇的狀態是自己單獨訓練，他拒絕前往卡達等低水平聯賽踢球，現在他希望可以繼續留在頂級聯賽。此外值得一提的是，帕拉蒂奇這一次來到英國，可不就只是一個推銷曼朱基奇的工作，他還有一點就是順便打探一下博格巴的交易事情。

雖然斑馬軍團確實是很想買來法國人，但是誰也都知道，現在曼聯要是放棄 博格巴的可能性是有多低。之前尤文在今年夏天已經和其傳出了緋聞，甚至皇馬也在那個時候多次傳聞想引進法國人，但是最終都是不了了之了。

據悉，曼聯可能會以500萬的低廉的「白菜價」簽下曼朱基奇，當然尤文對於這個價格也可以接受。現在這筆交易最大的問題是球員的待遇問題，由於之前球隊在桑切斯身上真的是吃了太大的虧，導致球隊現在在面臨這樣的問題時候十分的敏感，因此他們一定要盡最大的縮減曼朱基奇的薪資待遇。

除了博格巴，帕拉蒂奇此次還希望就此打探一下威廉的轉會事情。這個切爾西球員現在和球隊合約到本賽季結束就到期，但是似乎威廉沒有多少和球隊續約的可能性，儘管切爾西也不想失去他，但更大可能是巴西人想尋求國外球隊的發展機會。

此外切爾西的埃摩森也是尤文的關注目標，但是現在切爾西依然無法進行引援，想從他們手裡買人難度還是很大的。
