歐冠又有7隊出線！阿賈克斯國米遺憾出局，這黑馬或攜手馬競晉級

2019-12-11 71條評論 2,090次閱讀 59人點贊

北京時間12月11日，歐冠小組賽E到H組進行了最後一輪比賽，結果利物浦、那不勒斯和切爾西等豪門，都順利晉級16強，而國米、阿賈克斯和里昂等豪門，則不幸慘遭淘汰。

在本輪比賽開始之前，阿賈克斯原本是晉級熱門，只要不輸給對瓦倫西亞，他們就肯定出線，遺憾的是，在本場生死戰中，他們竟然以0-1輸給對手。由於切爾西在主場順利擊敗里爾，本輪比賽過後，阿賈克斯也從小組第一掉到了第三，最終在主場爆冷出局。原本只要一場平局，阿賈克斯就肯定出線，結果卻不幸爆冷落敗，確實讓人非常遺憾。

跟阿賈克斯一樣，國米也是在主場「爆冷」，要知道，本輪比賽巴薩輪換了多名主力，其中梅西沒有進大名單，而蘇亞雷斯也被安排替補，可以說巴薩絕對有心放國米一馬，可惜他們卻沒抓住，最終在主場1-2輸給對手。相比之下，多特蒙德就爭氣多了，在主場對陣布拉格斯拉維亞的比賽中，他們最終以2-1擊敗對手，幸運超越國米，殺進了歐冠16強。在一夜過後，利物浦、那不勒斯、瓦倫西亞、切爾西、萊比錫，里昂和多特蒙德等7支球隊順利出線，而國米、阿賈克斯和澤尼特等球隊，則慘遭淘汰。

截至目前，歐冠16強僅剩下2個席位，在最後一輪比賽中，馬競將在主場迎戰莫斯科火車頭，此役只要擊敗對手，他們就將肯定出線。儘管近期狀態欠佳，但是馬競的實力依然遠超對手，相信在這樣的生死大戰中，西蒙尼的球隊不會讓球迷失望。如果沒能拿到3分，那麼馬競就得看尤文的臉色。在C組中，礦工將在主場與亞特蘭大進行生死大戰，在小組賽前三輪全部輸球之後，亞特蘭大還有出線的希望，只要此役能夠在客場擊敗對手，而小組出線的曼城又沒有做客輸給薩格勒布，那麼亞特蘭大就將創造奇蹟殺進16強。從目前看來，狀態出色的亞特蘭大，有很大的機會在客場拿到3分，接下來就看曼城願不願意幫忙了！在小編看來，最後一輪亞特蘭大這支意甲大黑馬，極有可能攜手馬競晉級，大家認為呢？
