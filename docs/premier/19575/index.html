江山代有才人出，安切洛蒂、佩萊格里尼需更新調整才能重獲成功

2019-12-17 1,275條評論 3,487次閱讀 1157人點贊

據義大利轉會專家迪馬濟奧宣稱安切洛蒂已與埃弗頓達成協議，義大利名帥將正式執教埃弗頓，只要安切洛蒂與那不勒斯談妥解約事宜，埃弗頓將會官宣安切洛蒂執教信息，從執教尤文、米蘭、到皇馬、拜仁再到英超二流埃弗頓，不僅讓人唏噓，近些年安切洛蒂的執教並算不上成功，加上在西漢姆聯隊即將下課的佩萊格里尼，真是讓人感嘆：江山代有才人出，安切洛蒂、佩萊格里尼老一代主帥執教之路越來越難。

安切洛蒂和那不勒斯主席也曾甜蜜無間

安切洛蒂、佩萊格里尼執教已走下坡路，近些年的執教算不上成功。

安切洛蒂也曾榮耀加身。安切洛蒂絕對是世界名帥，他在米蘭使用的聖誕樹陣型也曾風靡一時，對皮爾洛的使用是他執教生涯的妙筆，也曾率領米蘭和皇馬奪得3次歐冠獎盃。但他更像一個管理者，更善於做一個老好人，安帥一般不會給俱樂部老闆提出更多的要求，能將隊內的球員能力捏合成型，發揮出最大的能量，這也是為何很多球員和俱樂部老闆都非常喜歡他的主要原因。

安切洛蒂執教皇馬是最後榮光

安切洛蒂已走下坡路。2013年安切洛蒂執教皇馬，隨後率領皇馬奪得第10座歐冠獎盃，這似乎是他執教的又一次高峰，從皇馬下課後執教的拜仁卻遭遇滑鐵盧，拜仁球員對他也並不感冒，特別是在剛剛經歷了瓜迪奧拉的執教後，穆勒、里貝里、羅本對安切洛蒂非常不屑，在拜仁下課後安帥執教那不勒斯，也僅僅執教一個半賽季後就與主席矛盾再次下課，安切洛蒂的執教之路已走下坡路。

佩萊格里尼也是如此。佩萊格里尼人稱佩工中國球迷相當熟悉，佩工在黃色潛水艇比利亞雷亞爾隊、皇馬和曼城都曾取得優異戰績，但就是從2016年轉戰中超執教華夏幸福後，佩萊格里尼的執教之路就越來越艱難，在中超佩工也過的很不如意，回歸英超後執教西漢姆聯的戰績也持續低迷，本賽季17輪聯賽過後西漢姆聯僅僅排在第15名，佩工也即將下課。

佩萊格里尼執教曼城也曾榮獲冠軍

江山代有人才出，瓜迪奧拉、穆里尼奧、西蒙尼都感同身受，足壇主帥不及時更新執教理念就會被淘汰。

世界足壇更新換代更快。現如今世界足壇就如同現在的中國經濟發展，更新換代的速度非常快，任何一位主帥想抱著一套技戰術理念和戰術體系混個十幾年絕對行不通了，瓜迪奧拉曾在巴薩打造的夢三隊傳控體系，在拜仁和曼城都未能取得如意的戰績，只有融入了英超和德甲風格才能適應。穆里尼奧也在二進宮切爾西和曼聯碰壁，西蒙尼目前在馬競也失去了魔力，這些都說明目前世界足壇更新換代速度太快。

江山代有才人出。現在的世界足壇更像是百花齊放的時代，瓜迪奧拉的傳控足球依然在堅持、穆里尼奧的防守反擊也在不斷更新、克洛普的高位壓迫和重金屬足球風格依然震撼人心，齊達內、波切蒂諾、圖赫爾、羅傑斯等一批優秀主帥也在不斷進步，蘭帕德、特里、傑拉德等年輕主帥也已經嶄露頭角，長江後浪推前浪，一浪更比一浪強。

穆里尼奧和瓜迪奧拉也曾感受到危機

安帥佩工需更新理念。安切洛蒂和佩萊格里尼目前需要一定時間的休息和調整，兩位都是已經超過60歲高齡的老人，現如今的時代發展迅速，球員們的個性也更強，在管理模式方面也需要調整，同時以往的技戰術理念也需要及時更新，足球世界是殘酷的，作為足球教練壓力更大，如果兩位老帥不及時更新和加強學習的話，恐怕下步的執教之路會更加艱難。

做一個足球教練難、做一個世界級足球教練更難，要想在這個飛速發展的時代獲得冠軍，必須要及時更新戰術理念，如果固步自封不願意學習，那註定會被時代所淘汰。
