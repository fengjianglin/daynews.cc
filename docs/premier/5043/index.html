那些曾經的小梅西近況如何？恐怕要讓您失望了……

2019-12-08 0條評論 20次閱讀 0人點贊

從未在拉瑪西亞成長過的哈利洛維奇，18歲就披上巴薩球衣，實在太沉重了

本賽季，昔日被稱為「挪威梅西」的厄德高，正在努力救贖自己的職業生涯，他新賽季已經打進3球助攻4次，在Whoscored上的評分也高達7.53分，這至少說明，他新賽季在數據展示上，取得了很好的成績。

昔日曾經有很多年輕的妖人被冠以「X國梅西」的稱號，但並不是每一個人都像厄德高那樣，仍舊能夠在實現自我救贖的路上順利前行。

當年克羅埃西亞也出現了一名號稱「新梅西」的選手，他就是阿倫-哈利洛維奇。16歲時，哈利洛維奇就已經接受過包括熱刺、國際米蘭以及烏迪內斯在內的不少球隊的試訓，出生足球世家的他兄弟三人都走上了職業足球的道路，而以他的能力天賦最為出眾。

埃姆雷-莫爾也沒有成為人們心目中的樣子

18歲時，哈利洛維奇果斷選擇加盟擁有梅西的巴薩，他希望能夠在這裡接受梅西的教導。然而事與願違的是，在這裡，哈利洛維奇卻成為了一名平庸的球員。

在巴薩B隊並沒有展示出過人實力的哈利洛維奇，在隨後被租借給了希洪競技，結果在這裡他的表現也並不盡如人意，當他轉會去漢堡踢球時，身價不過只有250萬歐元。但是在德甲聯賽，哈利洛維奇出場機會寥寥，租借回西甲拉斯帕爾馬斯之後，他也沒有拿出過人的能力，就連上賽季轉去AC米蘭踢球，也被人認為是帶有摸獎性質的賭博……

厄德高還有一定的救贖機會

本賽季，轉戰荷蘭聯賽的哈利洛維奇在海倫芬隊出場10次，多以替補身份登場，總計出場時間只有區區260分鐘，已經23歲的他，已經沒有太大的希望被主流聯賽看上了……

同哈利洛維奇境況類似的，還有號稱「土耳其梅西」的埃姆雷-莫爾，這位出生在丹麥的天才球員，在19歲就加入了多特蒙德，可是在這裡他雖然獲得了一些出場機會，卻並沒有提供有說服力的表現。短短一個賽季之後，就被轉賣給了西甲塞爾塔俱樂部，而在塞爾塔的2個賽季，他只不過獲得了區區8次首發機會，新賽季又被租借回了土耳其加拉塔薩雷。

可怕的是，在加拉塔薩雷，已經22歲的莫爾竟然徹底淪為替補，昔日讓多特蒙德視若珍寶的才華，在這裡早已經消失殆盡，在新賽季中他在各項賽事中的首發次數也僅有3次，沒有進球，也沒有助攻……

大多數「X國梅西」的發展並不如預期，這似乎並不是一個偶然的現象，以哈利洛維奇和莫爾為例，他們在年紀輕輕時就被給予了過高的期待，以至於沒有經過很好的經驗積累，就被匆匆推上一線隊大舞台，而承受太多壓力的他們，本達不到高強度的比賽要求，在表現不好的情況下，又陷入了無休止的質疑，進而惡性循環，導致越來越多的「X國梅西」陷入到了傷仲永的窘境。（陳川）
