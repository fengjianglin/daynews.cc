薩里：年輕時只在周日晚上抽煙，周中幾乎從不抽

2019-12-24 1,249條評論 3,524次閱讀 2404人點贊

直播吧12月24日訊 薩里在家鄉俱樂部菲利內（Figline）100歲生日之際接受了《全市場》的採訪，尤文主帥年輕時候在這裡效力過，採訪中他透露了自己的抽煙史。薩里1959年出生在那不勒斯，不久後他便隨家人移居到了佛羅倫薩的菲利內-瓦爾達諾小鎮。

「我希望並且相信自己仍然是過去的那個薩里，我沒變，在球場上我永遠是當初的那個薩里。一個月前，我接到通知要去尼翁參加世界最佳教練峰會，我問尤文秘書： 『他們為什麼要叫我？你確定嗎？』。我的思想沒有變過。」

你跟菲利內俱樂部的關係如何？

「小時候，意甲對我來說是一個相當遙遠的世界，我兒時的偶像都是菲利內的球員。我有幸在這裡踢球，我跟童年的小夥伴們一起打進了一線隊，這是極其美好的事情。」

對菲利內的未來有什麼計劃嗎

「是的，我出生在那不勒斯，但我認為自己是100%的菲利內人。我願意為菲利內做點什麼，我跟朋友們有一個目標：那就是幫助小鎮里的所有體育俱樂部成長。」

你熱愛自行車運動？

「看起來很奇怪，我現在是足球教練，但如果晚上電視上同時有環法比賽和足球比賽時，我一定會看自行車比賽。」

你跟阿萊格里的較量？

「我記得有一次我執教的聖喬瓦內塞（意丁球隊）對陣阿伊安內瑟，當時阿伊安內瑟的教練就是阿萊格里，那場比賽0-0收尾，沒有一個進球。賽後一個球迷對我們說：『如果你倆是教練的話……』」

你從什麼時候開始抽煙？

「從我開始踢球的時候，我只會在周日晚上比賽期間抽兩根。周中極少抽，幾乎從不抽。」

（我叫達芬奇）
