斯坦科維奇鼓勵庫盧：人們都說你像德布勞內，但你要堅持做自己

天天要聞
2019-12-04

直播吧12月4日訊 帕爾馬小將庫盧塞夫斯基與國米名宿斯坦科維奇都叫德揚，二者也在網路上進行了互動，斯坦科維奇對庫盧塞夫斯基進行了鼓勵，庫盧塞夫斯基則稱斯坦科維奇是最棒的中場，是自己全家的偶像。

斯坦科維奇在接受Dazn記者採訪時說道：「嗨！德揚，過得怎麼樣？希望你一切都好，我在這裡為你送上掌聲，也祝你未來好運，我看你正走在正確的道路上，加油堅持下去吧，腳踏實地，只考慮足球，不要受外界的影響。」

「人們都說你像德布勞內，但在我眼裡，你更應該做你自己，你要形成自己的風格特點，並堅持下去，加油吧少年，祝你好運。」

庫盧塞夫斯基看到斯坦科維奇的文字後非常高興和感動，他立刻在下方評論道（斯坦科維奇的採訪被轉載至相關門戶網站）：「能收到您對我的評價簡直難以置信，我在U17青年隊對陣國米U17 的比賽中曾見過您（斯坦科維奇），那是您的兒子效力的球隊。比賽結束後儘管我很累，但我還是跑過去與您合影。」

「您在我心中是最強大的球員之一，是一名偉大的球員，一個偉大的人。您是我們全家的偶像，我們為您著迷，並曾一次次地觀看過您的每一粒進球，我現在還清晰記得當初在面對沙爾克04時您打進的中場吊射。哈哈，等我回家我肯定第一時間告訴我爸，斯坦科維奇給我發消息啦！」

此前，有報道稱庫盧塞夫斯基是國米冬窗的引援目標，庫盧塞夫斯基本賽季表現出色，他為帕爾馬在意甲聯賽中出場14次，貢獻3球5助攻。

由於發揮出色，庫盧塞夫斯基得到了眾多豪門球隊的青睞，入選了瑞典國家隊，並完成了國家隊首秀。現在他已然成為帕爾馬的絕對核心球員。今夏曼城也曾考慮簽下庫盧塞夫斯基。

（鐵甲鋼拳）
