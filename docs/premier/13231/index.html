凱叔看盤：雙方實力懸殊，別受曼城狀態影響判斷

2019-12-11 3,494條評論 4,790次閱讀 1979人點贊

歐冠 12-12 01:55 薩格勒布迪納摩VS曼徹斯特城

由於小組對手實力懸殊，曼城早早鎖定小組第一，提前完成出線任務。所以此役，曼城戰意成為左右本場比賽最大變數，外加曼城現在聯賽狀態低迷，正處於水深火熱中，瓜迪奧拉的日子也不好過。

薩格勒布迪納摩此前的歐冠小組賽當中取得了1勝2平2負積5分位列第3，此役儘管希望渺茫，但是夢想還是要有的！且此前主場歐冠比賽，球隊能保持不敗，主場還是有相當強勁克敵能力。

曼城的情況想必大家都清楚，現在聯賽近乎衛冕無望，瓜迪奧拉魔力不在，剛在英超最不能輸的曼徹斯特德比中，主場1-2不敵日漸衰落的紅魔，曼城在英超統治力已經大不如前。好在歐冠簽運不錯，小組出線易如反掌。現在目標達成，客戰克羅埃西亞班霸，瓜迪奧拉醉翁之意不在酒。聖誕魔鬼賽場仍不容有任何閃失，這是起碼的態度，4天後球隊仍需客戰剛剛回魂的槍手阿森納，難保不是一場苦戰。

根據博體體育提供數據分析，本場初盤亞指給出主隊受讓半球中高水，目前部分公司已經調整為主受讓半一中水。主隊最近10場比賽的狀態一度讓人興奮，只輸掉一場比賽而已，但是看過昨天國米對巴薩二隊的比賽可知，替補陣容甚至比主力陣容來得更加生猛，曼城仍是值得高看一線。

本場推薦：曼城 比分推薦：0:1/2:4
