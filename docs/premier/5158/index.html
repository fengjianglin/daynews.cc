布馮：當年為留在尤文踢意乙放棄了很多 希望成為年輕人的榜樣

2019-12-05 0條評論 17次閱讀 0人點贊

近日，尤文圖斯門神布馮在接受採訪時談到，當年「電話門」事件後他選擇留在尤文踢意乙，為此放棄了很多東西，同時他表示希望成為年輕人學習的榜樣。

「留在意乙的決定？在一個充滿各種言論的世界裡，那是一次好機會，能夠向所有充滿熱情的年輕人發出一個強烈的信號。當你成為第一個站出來放棄重要事情的人，你才可以發出這種信號。在那種情況下我放棄了很多。當時我28歲，處在職業生涯的重要階段，我剛剛贏得了世界盃，正在爭奪金球獎（最終布馮金球獎排名第二）。如果我做出別的決定，情況就會變得不同。但在那時，我相信只要做好自己、慷慨待人，生活就會把所有補償都還給你。現在我42歲還能登場，正是14年前決定做出的選擇。」

「幾周之前，我參加了一個廣告活動。和我一起的還有伊萬-巴索，以及兩名籃球運動員。有時候我感覺不太舒服，因為有很多運動員贏得了和我一樣多的榮譽，他們代表了義大利的最高水平，也贏得了很多重要的獎牌。但他們不夠幸運沒能保持足夠的知名度，這可能是金錢所導致的。這讓我感到有些愧疚，因為體育是絕對精英化的。我非常幸運，從事了最賺錢的項目——足球。這種機會是靠著激情得到的，我今後也會去教育看體育的孩子們這樣做。」
