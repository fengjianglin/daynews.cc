EMC易倍體育：霸道總裁c羅上演逆天頭槌，拉涅利誇讚連連

天天要聞
2019-12-20

當西班牙國家德比以沉悶的0-0結束，當梅西在諾坎普搖頭嘆息，遠在義大利的c羅不甘寂寞，用驚艷絕倫的比賽告訴球迷們他仍是那個無所不能的霸道總裁，他仍然可以用逆天的標表現讓自己的球隊贏得比分。

桑普多利亞球隊的實力一般，目前處於聯賽的中下游，距離降級區也沒有多大的差距，不過在桑普多利亞的陣中卻有一位大名鼎鼎的主教練，他就是曾率領萊切斯特城創造了震驚整個歐洲足壇的黑馬奇蹟的拉涅利。有拉涅利在桑普多利亞不容小覷。因為在聯賽中尤文圖斯和國際米蘭已成水火之爭，所以這場比賽薩里直接派上了最強陣容。

在上一場比賽中，薩里大膽地改變了以往所固執的打法，改用了三叉戟陣型沒想到收穫了奇效，於是這一場薩里繼續沿用了這一陣型。將c羅、伊瓜因和迪巴拉一同派出了場。在比賽打響之後，尤文的進攻非常積極，迪巴拉在前場的組織拿球非常活躍，他和c羅以及伊瓜因的配合也非常具有默契。

在比賽的19分鐘，迪巴拉打入了一個精彩的天外飛仙，攻破了桑普多利亞的球門，幫助自己的球隊在客場取得了領先。不過桑普多利亞也沒有放棄，他們的前鋒卡普拉里在第35分鐘的時候幫助球隊扳平了比分。之後的兩支球隊各有攻守，可以看得出在拉涅利的調教之下桑普多利亞的進步很大。

就在上半場快要結束的時候，逆天的一幕出現，在看到隊友的邊路傳中球後，c羅猛地起跳，高高躍起，然後猛地一甩，將皮球砸進了球網。c羅的這一進球幫助尤文再次領先，並且最終幫助球隊贏下了這場比賽。這場比賽雖然尤文圖斯打的並不算精彩，但是c羅的這個頭槌絕對是無比驚艷。

當看到c羅的這一進球時不止是全場的球迷，就連桑普多利亞的主教練拉涅利也目瞪口呆了，他震驚於c羅變態的滯空能力。據統計c羅的這次起跳到落地的整個時間為0.92秒，也就是說c羅在空中停留了0.92秒，這直接追平了NBA中的籃球之神喬丹的紀錄，也難怪在賽後拉涅利驚嘆說，這是只有在NBA中才會看到的逆天一幕，但是這個夜晚c羅在綠茵場上做到了。在這場比賽取得進球後c羅已經連續5場比賽破門了，過往那個自帶1-0屬性的c羅彷彿又回來了，我們期待著c羅為我們創造更多的奇蹟。
