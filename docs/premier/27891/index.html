梅西神了！5人包夾下轟世界波 西媒：50個人也沒用

2019-12-22 1,088條評論 2,946次閱讀 1202人點贊

到底幾個人才能攔住梅西？這個問題每年都會有人提起，但也沒人敢給出一個確定的答案。今天在這場巴薩同阿拉維斯的比賽中，梅西面對5人的圍追堵截，依然轟進了世界波。

這一幕出現在比賽第69分鐘，梅西接到了蘇亞雷斯的傳球，直接帶球向對方球門殺去，此時阿拉維斯5名防守球員立即圍了上來，但梅西絲毫不為之所動，宛若他身邊沒有人防守一般。左腳一撥，隨後抬腳就射，這球勢大力沉，角度刁鑽，對方門將對此毫無辦法。

這粒進球是梅西本年度的第50粒進球，也是他10年內第9次年度進球超過50球，另外，這也是梅西在諾坎普打進的第250粒進球，而他只用了234場比賽。

對於梅西的這粒進球，《每日體育報》也驚嘆說：「 別說是5個人了，就算是50個人過來包圍他，對梅西來說也沒有關係！ 梅西再一次證明，在球隊最需要他的時候，他就會挺身而出。在他攻破阿拉維斯球門的時候，共有5名球員出現在他的周圍。」
