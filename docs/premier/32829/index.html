可以替代蘇亞雷斯的3位年輕前鋒：哈蘭德上榜，第1是梅西同胞

2019-12-23 2,883條評論 4,018次閱讀 953人點贊

路易斯·阿爾貝托·蘇亞雷斯無疑是足壇過去十年最好的前鋒之一，但是再好的球員也有老去的一天，烏拉圭人自然也不例外，儘管他依然在2019-20賽季的19場比賽中攻入了12粒進球，卻無法掩蓋他已經四年多沒有打進歐冠客場進球的事實。

在今年的早些時候，下個月就要滿33歲的路易斯·阿爾貝托·蘇亞雷斯就曾公開表示他在西班牙的日子不多，所以巴塞羅那必須儘快找到一位能夠替代他的球員，那麼問題來了，在當今足壇的眾多年輕球員當中，又有哪些前鋒能夠無縫融入巴塞羅那的前鋒線，成為烏拉圭人在諾坎普的替代者呢？我們今天就一起來看一看。

第3位：埃爾林·哈蘭德 19歲/薩爾茨堡紅牛/挪威

埃爾林·哈蘭德這個名字似乎就是進球的代名詞，看過他本賽季比賽的朋友，應該都同意這位挪威足球運動員將是足壇的一位大人物。截止目前為止，挪威人為奧地利薩爾茨堡紅牛在各項賽事中出場22次，攻入令人難以置信的28球，其中包括5次帽子戲法。當你意識到他還只有19歲，並且他的這些進球當中，還有一些是在面對利物浦和那不勒斯這樣的頂級俱樂部時打進的時候，這些數據聽起來更為了不起。

雖然從身高這一點來說，1米91的挪威人並不符合巴塞羅那的血統，但是只要他能夠不斷進球，這一點並不是問題，埃爾林·哈蘭德的比賽主要圍繞身體和空戰，但是他的技術也很出色，可以在諾坎普茁壯成長。對於他，巴塞羅那已經關注了很久，如果最終能夠轉會成功的話，對於雙方都是有益的。

第2位：蒂莫·韋爾納 23歲/萊比錫紅牛/德國

雖然羅伯特·萊萬多夫斯基在德甲聯賽的開局中佔據了幾乎所有的頭條，但是蒂莫·韋爾納更像是一個沉默的刺客，截止目前為止，這位德國人已經在為萊比錫紅牛出戰的21場比賽中攻入了19球，除此之外，這位23歲的球員還有6個助攻。

相比路易斯·阿爾貝托·蘇亞雷斯，蒂莫·韋爾納傳球準確性更好，而且有需要的話，他可以成為一名邊鋒，這一點，德國人已經證明過自己了，對於這樣一名優秀的球員，加盟歐洲賽場頂級球隊只是時間問題，據一些消息人士透露，他的釋放條款已經降到了3000萬歐元，這對於巴塞羅那來說自然是一個好消息，而且就算是這個數字翻一倍的話，相信諾坎普也不會眨眼，因為蒂莫·韋爾納是值得的。

第1位：勞塔羅·馬丁內斯 22歲/國際米蘭/阿根廷

勞塔羅·馬丁內斯一直被認為是阿根廷前鋒生產線的下一個大人物，這位南美球員聰明、充滿活力、技術過人，截止目前為止，他已經在為國際米蘭出戰的21場比賽中攻入13球，從數據來說，他或許不像路易斯·阿爾貝托·蘇亞雷斯那樣多產，但是他有足夠的天賦與他的同胞里奧·梅西在巴塞羅那建立一個良好的夥伴關係。

這位22歲的球員在球門前快速思考的能力和敏銳的嗅覺，很容易就讓人想到烏拉圭人，國際米蘭本賽季已經和巴塞羅那交過兩次手，他在客場打進了一球，在國米主場，他雖然沒有進球，但是表現還算不錯。不過，國際米蘭並不願意他離開，因此巴塞羅那這邊如果想要得到他的話，必須提供一個大膽的報價來引誘義大利方面賣掉他們的寶石。

除了上面3位年輕球員之外，大家覺得當今足壇的眾多年輕前鋒當中，還有哪些能夠成為路易斯·阿爾貝托·蘇亞雷斯在巴塞羅那的替代者？歡迎在評論區留言寫下您的看法哦。
