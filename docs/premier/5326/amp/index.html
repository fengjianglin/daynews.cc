意甲今日頭版：盧-勞組合勝過VAR 尤文不能丟掉迪巴拉

天天要聞
3 週 ago

直播吧11月28日訊 米蘭體育報、羅馬體育報、都靈體育報是義大利的三大體育報，今天我們來看看它們各自的頭版頭條都帶來了哪些內容。

《米蘭體育報》

1.「盧-勞的遊樂場。」盧卡庫和勞塔羅牽手慶祝的畫面成為封面。冠軍聯賽，國米依靠最佳拍檔在布拉格完成壯舉：3-1。盧卡庫（1個進球、2個被吹、2個助攻、1腳橫樑）和勞塔羅（梅開二度）的非凡默契比斯拉維亞和VAR更強。在聖西羅擊敗巴薩能確保晉級16強，孔蒂：「將會有8萬人為我們助威。」

2.「那不勒斯的救贖。」在利物浦一場精彩的平局，現在只需要一分就能出線。默滕斯並沒有興高采烈。不過德勞倫蒂斯表示：「大家表現非常棒。」

3.歐聯杯：羅馬做客伊斯坦布爾，生死戰。

4.魔法師迪巴拉是寶貝！帶領尤文前進，保羅的改變是薩里的功勞也有女友的功勞，後者讓他重新開心。

5.恩里克回歸：「莫雷諾走了，他不忠誠。」

《羅馬體育報》

1.「巨人。」整耳欲聾的國米在布拉格贏球：3-1，現在可以憧憬出線。勞塔羅和超級盧卡庫擊敗斯拉維亞，孔蒂重新打開晉級之路，等待在聖西羅迎戰巴薩：他們已經鎖定小組頭名。

2.那不勒斯用心踢球，阻擋利物浦。默滕斯首開紀錄，洛夫倫扳平比分1-1，距離16強只有一步之遙。

3.皇馬巴薩誘惑迪巴拉。西甲豪門開始用令阿根廷人無法抗拒的魅力誘惑他。

4.小基耶薩與佛羅倫薩嘗試和好。

5.羅馬出征伊斯坦布爾，目標晉級。拉齊奧對陣克魯日。

6.恩里克：「莫雷諾不忠誠。我已經讓他走了。」

《都靈體育報》

1.「若奧-菲利克斯獲2019年金童獎。」馬競天才被選為歐洲U21最佳球員，他是第17屆得主。12月16日晚，他將在都靈領獎。

2.「不要丟掉這顆寶石！」尤文找回了最佳的迪巴拉，他用一粒精彩自由球驚艷歐洲，並且還有現象級的表現，夏天，他還被放在轉會市場中。他的合同到2022年到期，不過帕拉蒂奇正在研究留住他。

3.「一支偉大的國米。」藍黑軍團在布拉格的超級表演：勞塔羅-盧卡庫組合帶領球隊3-1取勝。

4.「那不勒斯，一場金子般的平局。」1-1利物浦，現在他們只需要一分就能出線。

（我叫達芬奇）
