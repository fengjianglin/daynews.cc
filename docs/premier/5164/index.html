索肖：連續性是球隊成功的關鍵，我們現在的方向是對的

2019-12-04 0條評論 15次閱讀 0人點贊

虎撲12月4日訊 目前為止，英超豪門曼聯本賽季的戰績不佳，原因肯定來自許多方面，但也許今夏轉會市場上的失策也是其中之一。

主帥索爾斯克亞看來，馬奎爾和盧卡庫的最終轉會姍姍來遲，阻礙了曼聯整個夏天的引援進程。

今年夏天8月5日，曼聯才以8000萬英鎊的價格從萊斯特城簽下英格蘭中衛哈里-馬奎爾，同時英超的轉會窗口關閉3天之後，盧卡庫轉會國際米蘭才最終成行。

今年夏天的季前準備中，索爾斯克亞強調，如果有球員流出就會引進相應的球員替代。但結果是曼聯除了與尤文前鋒保羅-迪巴拉和曼朱基奇還有熱刺的組織中場埃里克森傳了一整個夏天的緋聞外，啥也沒發生，盧卡庫走了就是走了。

本著一個一個來的原則，曼聯先後完成了丹尼爾-詹姆斯、萬-比薩卡、馬奎爾的轉會。差不多每一筆轉會都耗了將近一個月的時間。

索爾斯克亞早在今年2月份就擬定了球隊哪些位置上需要引援，但轉會窗結束後，曼聯還是缺一名中場中路的球員以替代費萊尼和埃雷拉離去的空缺。

紅魔主帥請球隊的擁躉們放心，關於下賽季的球隊結構，高層已有框架。《曼徹斯特晚報》將其理解為曼聯的夏季轉會目標幾近確定。

索帥說：「是的，對於球隊在6、7月份時陣容是什麼樣的，我們頭腦里有清晰的畫面。希望事情的進度好又快，像馬奎爾加入、盧卡庫轉出這樣的轉會就太晚了。我們現在有計划了，是的。」

不過，在球迷中，耐心正一點一點的被消耗。至今為止，曼聯被聯賽第一名拉開了22分之差，距離歐冠區也有8分，僅僅比降級區高了6分，14場英超聯賽只贏下了其中4場。

上賽季，聯賽踢了17場後，曼聯解僱了時任主教練穆里尼奧。而現在，如果索爾斯克亞不能確保在接下來面對熱刺、曼城、和埃弗頓的聯賽中全取9分，他在前17輪的成績甚至還不如當時穆里尼奧下課時的聯賽積分。

當時穆里尼奧中途下課很是令人吃驚，因為曼聯解僱前一任主教練是因為賽季結束後球隊沒進入歐冠。

索爾斯克亞說：「我不會對別人怎麼經營俱樂部指指點點，但是我知道球隊的『連續性』是其成功的關鍵。有的球隊對主帥更有耐心，而有的球隊則沒給足主帥時間，缺乏連續性。」

「至於我和曼聯，我們有一個合理可行的計劃，希望能開花結果，並且儘快。雖然有的時候進度不盡如人意，但是我們的方向是對的。這體現在多個方面，不止是球場上，還有幕後，這就是俱樂部現在的狀況。」

「上賽季末，球隊人心渙散、鬥志全無、信仰崩塌。現在這些都將轉變，球隊的精氣神與之前大不相同。我相信這些球員，他們可以隨時贏下任何一場比賽。」

「但是我也說過我們現在並不是萬事俱備，我們要創造更多的』一致性『。當我們的球員全部回歸，擁有一個健康完整的陣容時，我們完全可以奮起直追，對積分榜前面的球隊發起衝擊。現在，我們跟排名第5的熱刺也就相差2分。」

「上賽季我們獲得了第6名，這賽季我們力爭更上層樓。」

(編輯：流浪劍客拿把大刀 )
