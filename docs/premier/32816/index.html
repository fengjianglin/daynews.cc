尤文輸掉超級盃，C羅又少一項榮譽，C羅轉會尤文一場三輸的結果

2019-12-24 247條評論 1,152次閱讀 87人點贊

昨晚義大利超級盃在沙特國王大學體育場舉行，C羅所在的球隊尤文圖斯以1：3輸給了因扎吉執教的拉齊奧。痛失義大利超級盃冠軍.

賽後C羅憤怒異常，對隊友失望至極。賽後的領獎過程中，羅納爾多全程黑臉，官員剛剛辦法給羅納爾多超級盃銀牌後，C羅毫不掩飾的立即就摘掉被羅納爾多認為是恥辱的獎牌。並拒絕與第二名派獎官員握手。

羅納爾多作為與梅西比肩的偉大球員，羅納爾多自然不會輕易地接受失利，對於難度如此小的義大利超級盃，面對並不強大的拉齊奧。尤文圖斯居然連續兩次輸球。在聯賽輸給拉齊奧，已經讓出聯賽榜首的位置，如今在義大利超級盃中，再次1：3輸給拉齊奧。這對於從2013年以來，從來沒在決賽中輸球的羅納爾多來說，是不可接受的。對於志在奪取個人第六座金球獎的C羅來說，任何一個冠軍都會給金球評選加分。但難度最小的超級盃就這樣被拉齊奧搶走了，C羅怒不可及，就不難理解了。

歐冠冠軍偶然性非常大，並且強手如雲，C羅納爾多並無絕對把握拿到歐冠冠軍，聯賽如今被國際米蘭搶走榜首位置，尤其國際米蘭在歐冠中被淘汰，國際米蘭勢必會更加專註義大利聯賽冠軍。C羅的義大利聯賽冠軍的難度較以往或許有一些挑戰。再輸掉義大利超級盃後，C羅的不高興就自然而然了。

羅納爾多在皇馬拿到4個歐冠冠軍，獲得了五個金球獎。與皇馬相得益彰，但C羅與皇馬因為不可名狀的原因分手，羅納爾多加盟尤文。羅納爾多一直想證明自己離開皇馬也能帶領強隊的尤文奪取歐冠冠軍。

可惜事與願違。皇馬失去了C羅，皇馬步履蹣跚，失去以往在歐冠的稅利。同樣加盟尤文後的C羅，也沒能完成帶領常年歐冠四強的尤文能更進一步。尤其在新賽季，羅納爾多在21場比賽中只打進12球，並且有的比賽，會被主教練薩里提前換下場。羅納爾多在尤文狀態不佳是不可辯駁的事實，或許是薩里的戰術制約了羅納爾多的發揮，或許是羅納爾多年事已高的，無論是那種原因，C羅已經不再是曾經在皇馬那個人擋殺人佛擋殺佛的王者巨星了。而尤文在獲得C羅後，成績也並沒有提升。相反這賽季可能四大皆空。

皇馬，C羅，尤文，目前看起來是一場三輸的結果。

有輸家當然也會有贏家。正如羅納爾多的經紀人說的那樣，不離開皇馬，羅納爾多會比梅西早一步拿到第六座金球獎。
