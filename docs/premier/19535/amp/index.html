意甲12-19桑普多利亞VS尤文圖斯“水手坐鎮主場-至少我們還有夢”

天天要聞
2019-12-18

桑普多利亞VS尤文圖斯

時間：12-19 01：55

場地：路易吉·費拉里斯球場

意甲第17輪老婦人尤文圖斯客場挑戰水手桑普多利亞球隊，目前老婦人在聯賽中16場比賽12勝3平1負39積分高居第二名，水手現在只有4勝3平9負積15分排名中下游16名。從兩隊的成績來看，毫無疑問老婦人的實力在水手之上，但是球場上什麼情況都有可能發生，良好發揮，全力取勝才是王道！

從目前的情況來看每一場勝利對老婦人來說都很重要，因為本賽季國際米蘭復甦，目前的積分和老婦人同樣39分，並且以凈勝球的優勢位列首榜，值得一提的是，老婦人本賽季客場被拉齊奧逆轉擊敗外，還被萊切和薩索洛逼平，這也是落後國際米蘭根本原因，本賽季可以說國際米蘭已經成為老婦人競爭聯賽冠軍最大的對手之一，而本次在弱隊身上搶分是最好的機會。

在上一輪老婦人主場以3比1的成績擊敗烏迪內斯，在這場為球隊建功的是C羅和萊昂納多·博努奇，由於C羅梅開二度，並在此刷新個人數據，是加盟老婦人後第一次連續四場比賽進球。要知道C羅已經三十多了，在沒有傷病的情況下，他一定會選擇出戰任何一場比賽，屆時的比賽期待C羅還能進球。

水手本賽季發揮不佳，意甲六輪後，水手排名墊底，此後，克勞迪奧·拉涅利出任水手主教練，並帶隊脫離降級區。在上一輪對陣帕爾馬0比1的成績惜敗，比賽中罰丟點球無緣幫球隊扳平比分。不過本賽水手主場防守極佳，8個主場僅丟9球，是意甲主場中防守最好的幾個球隊之一，相反水手進攻也是很乏力的，8輪主場只贏過都靈和烏迪內斯。不過提一下水手上賽季在主場2比0擊敗過老婦人，心理上佔據一定上風。

老婦人本賽季最大的問題就是防守不給力，聯賽16場比賽丟了16球，場均丟1球的表現並不能讓人滿意，在歐戰強者如雲，老婦人想要走到最後，必須得加強防守這方面問題，還需要更加努力。不管怎麼說當務之急還是得先拿下水手！

競彩推薦：尤文圖斯 勝

亞盤推薦：尤文圖斯 -1

大小推薦：大2.5

比分推薦：2-1 3-1

以上純屬個人見解分析，最後推薦根據臨場盤口而定，歡迎各種評論轉發！
