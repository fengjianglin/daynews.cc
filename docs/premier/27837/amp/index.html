48年恥辱紀錄誕生！曼聯4分鐘丟2球被墊底隊打爆，門神罕見黃油手

天天要聞
2019-12-23

19/20賽季英超第18輪焦點戰，曼聯客場對陣墊底隊沃特福德，下半場短時間內（4分鐘），曼聯連丟2球。場邊，索爾斯克亞直接黑臉。尤其是第一個丟球，德赫亞黃油手，直接捂臉不敢見人。曼聯過去13場無法零封，創下48年恥辱紀錄！

第50分鐘，薩爾禁區里射門，力量不大、速度不快，德赫亞想直接抱住，但是沒有抱穩，皮球竄進球門。丟球之後，德赫亞把臉埋在草皮里，無臉見人！

這樣一粒丟球讓曼聯防線徹底慌了，隨後，萬比薩卡送點，主裁判果斷判罰點球，站在點球點前，迪尼右腳爆射中路，時間定格在54分鐘！

也就是說，短短4分鐘時間，曼聯連丟2球。打進第二球後，沃特福德全隊瘋狂慶祝，而主隊球迷徹底嗨了。要知道，沃特福德可是副班長，而曼聯此前擊敗了曼城，但這場比賽，曼聯防守瞬間短路！

本賽季，曼聯、阿森納、AC米蘭演著「三國演義」，此前，AC米蘭0-5被爆打，如今，曼聯也陷入絕境。對於曼聯來說，這是恥辱比賽！

曼聯過去13場無法零封，自1971年以來紅魔最差表現，也就是說，曼聯創下了48年的恥辱紀錄。這樣的曼聯，讓人感到失望。本賽季，曼聯好一場壞一場，他們兩次擊敗切爾西，並且擊敗了衛冕冠軍曼城，但面對中下游球隊，曼聯確實送分童子！

至於沃特福德球迷，他們很久沒有這麼活躍了，即便他們本賽季真的不幸降級，這場比賽也將是美好的回憶。賽前，誰能想到，曼聯會踢得這麼差，下半場，曼聯4分鐘丟2球。和AC米蘭一樣，球迷也吐槽么「曼聯也臉都不要了！」
