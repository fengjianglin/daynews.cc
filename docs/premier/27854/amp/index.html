利物浦首奪世俱杯難開心，因為“一生之敵”重返英超了

天天要聞
2019-12-22

要說今年五大聯賽最「紅」的球隊，嗨妹認為非利物浦莫屬了。

昨晚進行的世俱杯決賽中，依靠菲爾米諾加時賽的進球，利物浦以1-0的比分戰勝南美勁旅弗拉門戈，奪得了隊史首個世俱杯冠軍，看來紅軍真的是不怕遠征難啊！

在渣叔克洛普的率領下，本賽季的利物浦在英超聯賽中可謂勢如破竹。雖然昨晚的「老二、老三」之爭，曼城3-1戰勝了萊斯特城。但頭號競爭者的勝出依然不能阻擋紅軍奪冠的喜悅，因為他們已經在積分榜上領先10分之多，在少賽一輪的情況下可以說是優勢很大了。

難道就沒有人能治得了利物浦了嗎，今年的英超就這樣大結局了？嗨妹感覺沒那麼簡單，因為那個男人來了，確切的說是一個老男人，而且是一個與紅軍有著「血海深仇」老男人！

北京時間12月21日，英超球隊埃弗頓官方宣布，安切洛蒂正式執教球隊，雙方簽約至2024年。

不錯，嗨妹上文說的男人正是這位義大利名帥，每當提起這位氣場十足，頗有黑手黨老大派頭的老帥時，相信很多利物浦球迷的心情都十分複雜，讓我們先來看看他們又有哪些神評論！

安切洛蒂和利物浦之間的恩怨估計很多球迷都再熟悉不過了，每當有媒體盤點足壇各種驚天逆轉的時候，那場在伊斯坦布爾進行的歐冠決賽絕對會出現在多數榜單的第一位，這場比賽的確是對「逆轉」這個詞語的最佳定義。接下來就讓嗨妹撥動時間的指針，帶大家重新回顧一下那場震鑠古今的比賽吧！

2005年的歐冠決賽在土耳其的伊斯坦布爾舉行，由利物浦對陣AC米蘭隊，當時執教米蘭的正是安切洛蒂。相信在本場比賽之前，安帥怎麼也不會想到，他和他的隊員會在這個夜晚經歷多麼巨大的情緒波動。

看了雙方的首發陣容名單之後，嗨妹才真的體會到了現在米蘭球迷的心情，不得不說，你們是真的慘！

回到這場比賽，單單從球員名氣上看，米蘭這邊完全就是碾壓之勢，那時的傑隊和龍哥還沒有達到今天的高度，而巴羅什、科威爾雖然也都實力不俗，但你總不能把他們和卡卡和舍甫琴科相提並論吧，況且由馬爾蒂尼、內斯塔、斯塔姆、卡福領銜的這條後防，完全可以競爭足球歷史最佳陣容了，所以當時外界也是一邊倒的看好米蘭。

比賽的進程印證了大多數人的猜想，開場僅1分鐘，馬爾蒂尼門前掃射將球打進，幫助米蘭收穫夢幻開局；第39分鐘，米蘭兩大鋒霸舍甫琴科、克雷斯波成功連線，後者禁區內推射建功；而到了第44分鐘，卡卡前場送出一腳極具穿透力的直塞，充分展現出了未來第一人的風采，克雷斯波接球後冷靜挑射再下一城，3-0，米蘭上半場沒有給利物浦任何機會。

一場歐冠決賽中，AC米蘭僅在上半場就取得了3球的優勢，據說半場休息的時候，球員們已經打開香檳慶祝了，作為球隊的主教練，相信此時的安切洛蒂的心情是這樣的。

不過利物浦的綽號是什麼，紅軍啊！紅軍不怕遠征難，萬水千山只等閑，別說3球，4個球又怎麼樣，這個時候干就完事了！（大巴黎表示說的很對）

精神建設搞得再好，場上還得需要人站出來解決問題。有道是滄海橫流方顯英雄本色，傑拉德——這支球隊的精神領袖吹響了紅軍第一聲反攻的號角！

第53分鐘，里瑟將球傳到禁區，傑拉德門前躍起頭槌破網，利物浦扳回一城；僅僅3分鐘過後，斯米切爾大力低射再進一球，此時的利物浦已經完全殺紅了眼；第60分鐘，傑拉德禁區造點，阿隆索主罰點球被撲後，機敏補射將球打進，3-3！

不到10分鐘的時間，利物浦在3球落後的絕境下生生扳平比分，伊斯坦布爾之夜這出大戲迎來了最高潮！嗨妹估計安切洛蒂這時候的心情應該是這樣的。

經過了120分鐘的激戰之後，兩隊依舊3-3平，比賽迎來了殘酷的點球大戰。而此時的米蘭已經完全被利物浦的逆天表現打丟了魂。最後一粒點球，舍普琴科面對杜德克的靈魂舞步將球罰失！剎那之間，全場的利物浦球迷完全陷入了瘋狂，伊斯坦布爾之夜永載史冊，毀滅你，與你何干！

這場蕩氣迴腸的歐冠決賽最終以利物浦奪冠而告終，不過安切洛蒂和紅軍的恩怨卻遠遠沒有結束。不知道是不是這場比賽打開了安帥心中的封印，在之後歲月中，安帥曾率領4支不同的球隊7次戰勝過利物浦。

伊斯坦布爾之夜兩年之後，安切洛蒂的米蘭和利物浦在歐冠決賽中再次相遇，不過這一次安帥總算找回了場子，依靠因扎吉的兩粒進球，AC米蘭2-1戰勝對手贏得歐冠獎盃。

2009/10賽季，時任切爾西主帥的安切洛蒂率隊雙殺利物浦，拿下了當賽季的英超冠軍。14/15賽季，安切洛蒂執教的皇馬在歐冠小組賽再次雙殺了紅軍。而就在近兩個賽季，安切洛蒂執教的那不勒斯又兩次在歐冠小組賽中與利物浦同組，他兩次都在主場帶領球隊擊敗了老冤家。

安切洛蒂此次重回英超肯定不是沖著利物浦來的，這位早已名滿天下的主帥勢必有著更大的目標。

但是不是冤家不聚頭，他們的相遇勢必會掀起新一波的恩怨情仇。2020年1月4日的足總杯，安切洛蒂將首次率領埃弗頓面對利物浦，讓我們一同期待這一人一隊，又會給我們帶來怎樣的故事吧！
