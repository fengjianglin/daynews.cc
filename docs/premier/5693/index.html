每日讀報：韓國“范迪克”仍然有望赴英超PK紅軍范迪克

2019-11-14 0條評論 18次閱讀 0人點贊

1、前義大利中場里卡多·蒙托利沃即將退役，他對自己在AC米蘭的職業生涯感到失望。但他對AC米蘭沒有怨恨。自從他與米蘭的合同在2018-19賽季結束後，34歲的他就一直沒有效力過別的俱樂部。

2、莫拉塔透露他離開車路士原因，因為受傷那段時間被西班牙國家隊排除在外，持續的媒體壓力和各種批評讓這位西班牙天才難以承受。他在代表藍軍出場72場只進了24球。

3、熱刺左後衛丹尼-羅斯攤牌熱刺，表明不會離開熱刺立場，他已經被熱刺通知不會有續約合同，俱樂部希望他離開，但他堅持要呆到2021年合同到期。

4、巴塞羅那已經不考慮蘇亞雷斯替代者哈蘭德轉會競爭，認為8500萬英鎊的價碼太高，這使得曼聯有機會簽下這位高產射手，他本賽季爆發吸引了諸多豪門。

5、拜仁又盯上了曼城9000萬球星薩內，在一月份的轉會窗口期，薩內可能會再次成為拜仁慕尼黑的挖角目標，薩內和曼城合同將在2020-21賽季結束時到期。

6、本賽季被延遲的西班牙首回合國家德比時間敲定，確認了12月18日巴薩主場PK皇馬。

7、曼城看上了巴塞羅那年輕小將費爾波，23歲小將今夏轉會到巴薩，在巴爾韋德統領下缺少出場機會。

8、曼城隊中場球員貝爾納多·席爾瓦被英足總禁賽一場。因為貝爾納多·席爾瓦因9月份時候在社交媒體上發布了一條帶有種族歧視的推文，這位前摩納哥球員在推特上發了一張隊友本傑明·門迪年輕時的照片，還有附上了一張來自西班牙某糖果品牌過時照片，這張照片因帶有種族主義色彩而遭到批評，雖然隨後很快被刪除但還是給媒體曝光傳播帶來了負面影響。這將導致B席將缺戰曼城主場對陣切爾西的比賽。

9、拜仁門將諾伊爾表示不太相信與佩普·瓜迪奧拉在拜仁慕尼黑重聚，拜仁現在漢斯·弗里克的帶領下「一切都很順利」。

10、切爾西邊鋒哈德森-奧多伊稱斯特林是他榜樣，他很有可能取代斯特林出戰英格蘭對陣黑山的歐洲預選賽中，此前斯特林聯賽對戰利物浦因與喬·戈麥斯矛盾而被取消了出場資格。

11、利物浦已經加入了曼聯、巴黎聖日耳曼和皇家馬德里對桑喬的追逐競爭，這位英格蘭邊鋒希望明年夏天離開多特蒙德。

12、曼城解僱了負責訓練設施的負責人，原因是他沒有將這座價值2億英鎊的球場保持在合格水平。

13、葡萄牙主帥費爾南多·桑托斯透露C·羅納爾多將代表葡萄牙出戰對陣立陶宛的歐洲杯預選賽，在最近的兩場意甲聯賽比賽中，羅納爾多被替換下場，球迷對他的健康狀況產生了懷疑。在預選賽前夕，桑托斯否認了關於羅納爾多受傷的傳言，並評論了關於斑馬軍團球星的傳言。

14、切爾西邊鋒威廉是尤文圖斯和巴塞羅那的目標，他的合同將在明年夏天到期。巴薩希望威廉明年夏天自由轉會，將在明年一月與巴西人就明年夏天自由轉會到諾坎普展開談判。

15、曼城已經向裁判主管邁克·萊利正式投訴了邁克爾·奧利弗在對陣利物浦的關鍵敗仗中的表現。同時曼城已經加入布萊頓和阿斯頓維拉的行列，致函裁判主管邁克•萊利，表達了對VAR爭議的擔憂，之後各俱樂部將召開關鍵會議，討論VAR爭議的進展。

16、沃特福德仍然有意向從北京國安簽下韓國後衛金玟哉。這名22歲的中後衛在去年1月加盟國安，這一年他給俱樂部和國家隊都留下了深刻的印象，為他贏得了「「韓國范戴克」的綽號。

17、盧卡斯·托雷拉正在考慮他在阿森納的未來，他的合同將在一月或夏天到期。

18、鐘塔西強調他仍然完全專註於為曼聯踢球，不會受到不利消息干擾。

19、據報道，馬里亞諾.迪亞茲拒絕離開皇馬。

20、萊比錫主帥朱利安-內格爾斯曼表示，如果沃納認為離開俱樂部是正確的選擇，他將鼓勵他離開。

21、據報道，巴黎聖日耳曼已經確定了超級巨星姆巴佩的替代者。

22、據《每日回聲報》報道，伯恩茅斯目前還沒有與利物浦就瑞安·弗雷澤的出售進行談判。

23、萊斯特城對流浪者隊前鋒阿爾弗雷多·莫雷洛斯有意向。現年23歲的莫雷洛斯出道於麥德林獨立，還效力過赫爾辛基，2017年夏天加盟流浪者並效力至今。目前他已經代表哥倫比亞國家隊出場過3次。

24、多特蒙德希望在一月份簽下一名前鋒，傳聞切爾西的吉魯、尤文圖斯的曼祖基奇和下個月將離開洛杉磯銀河隊的伊布都是他們潛在目標。

25、米蘭對埃弗頓前鋒莫伊茲·基恩很感興趣，並且已經和經紀人米諾·雷奧拉進行了談話，討論了在一月份轉會的可能性。這名19歲的球員自從離開尤文圖斯以來，他的四場英超聯賽中都沒有進球。據報道，基恩因為違反俱樂部紀律而缺席了周六對陣南安普頓的比賽。

26、那不勒斯準備向薩爾茨堡紅牛隊報價前鋒埃爾林·哈蘭德，據報道提交了一份約3500萬英鎊報價。

27、據《米蘭體育報》報道，由於熱刺對佩萊格里尼感興趣，羅馬向這名中場球員提供了一份新合同。

28、皇家馬德里準備了3.4億英鎊的一攬子豪購計劃，以確保明年夏天能從巴黎聖日耳曼將姆巴佩帶到伯納烏。

29、里昂體育主管朱尼尼奧透露，曼聯享有對孟菲斯.德佩的購買優先權，到目前為止他們還沒有收到相關俱樂部任何報價。荷蘭邊鋒在三年前以1500萬英鎊的價格從曼聯加盟里昂後有表現出色，最近與熱刺4000萬英鎊的轉會聯繫在一起。
