歐冠出局雖遺憾 但打破10號魔咒的勞塔羅配得上昂首離開梅阿查

天天要聞
2019-12-11

缺兵少將的國際米蘭最終還是沒有能夠在主場戰勝巴薩，再加上另外一場比賽中多特蒙德取勝布拉格斯拉維亞，藍黑軍團連續第2個賽季倒在了歐冠小組賽，無緣晉級。但無論如何，球隊的10號球員，阿根廷前鋒勞塔羅-馬丁內斯還是可以昂首走出梅阿查球場，比賽中他已經拼盡全力，國米有威脅的進攻幾乎都和他有關係，然而孤膽英雄卻無力改變球隊的命運。

10號對於一支球隊往往意味著核心、巨星和難以替代的角色，但這在瞬息萬變的足球世界中卻並非鐵律。在一些球隊之中，10號球衣不僅不是絕對主力的象徵，甚至還帶有某種令人難以解釋的魔咒現象，例如國際米蘭的10號。

但本賽季，勞塔羅用自己優異的表現打破了這一魔咒，讓藍黑色的10號球衣也能夠燦燦生輝。意甲15次出場8個進球，歐冠6次出場5個進球，曾連續4輪比賽破門得分，這樣的藍黑10號你怎能不愛？

梅亞察球場的10號球衣似乎從來都不是什麼搶手貨，甚至是燙手山芋，每一名身穿這件球衣的球員在藍黑軍團都歷經磨難，結果都讓人唏噓。這確實很奇怪，但至今都找不到答案。我們不妨就回首一下過去10年間那些藍黑色的10號們。

斯內德：莫拉蒂時代最成功的10號

斯內德是從伊布拉希莫維奇那裡繼承來的10號球衣，但瑞典人隻身穿這件戰袍踢了1場熱身賽就轉投巴塞羅那了。荷蘭人在2009年夏天被大肆招兵買馬的皇家馬德里掃地出門，慧眼識珠的穆里尼奧將他帶到了義大利，就此成就了他與這支球隊在進入21世紀之後的巔峰，斯內德也成為了馬西莫-莫拉蒂入主國際米蘭之後最為成功的10號。

在斯內德之前，國際米蘭10號球衣的主人中不乏大牌球星，像義大利中場卡博尼、「外星人」羅納爾多、「憂鬱王子」羅伯特-巴喬、日後的米蘭名宿西多夫、「國王」阿德里亞諾等等。這些隊員的國米生涯有好有壞，有歡笑也有淚水，但始終距離人們想像中的樣子有著差距。

羅納爾多在國際米蘭足夠強悍，但身穿10號的他卻連續遭遇重傷，直到後來改穿9號才有所好轉，但依舊沒能拿到意甲冠軍，最終淚灑替補席的畫面也定格為足壇經典。巴喬的國米生涯更加鬱鬱寡歡，和里皮的個人恩怨導致他有心無力，難以打出頂級狀態。阿德是高開低走的典型，後期已經徹底墮落。

這就是國際米蘭的10號，直到斯內德身穿這件戰袍帶領球隊登上歐洲之巔，才改變了人們對於它的看法。然而，在斯內德和俱樂部之間產生了不愉快並最終遠走土超後，這件球衣又經歷了多位主人，卻始終沒再能找回往日輝煌，和球隊一起陷入了低迷之中。

科瓦契奇：不得不放手的希望之星

科瓦契奇在2013年冬天由薩格勒布迪納摩加盟國際米蘭，當時他只有19歲，被視作莫德里奇之後格子軍團的下一個中場核心。當時的國際米蘭已經從巔峰迅速滑落，步入低迷期。不過當時的主帥斯特馬拉喬尼很信任科瓦契奇，給了他不少鍛煉的機會。於是，那之後科瓦契奇正式接過了荷蘭人留下的10號球衣，這代表了一種傳承，更代表了一種信任，克羅埃西亞天才被看作球隊重建道路上的重要人物。

然而，國際米蘭在馬扎里的手下顯得十分掙扎，從始至終都沒能走上正軌，球隊的成績比起前任斯特馬拉喬尼來說也沒有太明顯的進步。整個2013-2014賽季，科瓦契奇雖然在意甲聯賽出場32次，但大多數時間都是替補登場，最終他只奉獻4次助攻，沒有取得進球。時間進入到2014-2015賽季，隨著球隊人員的再一次變動，科瓦契奇得到的機會比此前多了很多，他在意甲出場35次打進5球並有5次助攻，儼然成了國際米蘭的中場核心，這也是他在藍黑軍團發揮最出色的賽季。

優異的狀態讓皇家馬德里向科瓦契奇伸出橄欖枝，彼時已經深陷財政危機的國際米蘭為了能夠填補財政虧空，最終選擇將他出售給銀河戰艦。後斯內德時代的首任10號球員卸任，留下的是一聲嘆息。畢竟在這之前，球隊已經將庫蒂尼奧賤賣給了利物浦，此番再失去一名青年才俊，球迷們心中不免五味雜陳。

約維蒂奇：被傷病摧毀的黑山妖星

2015年，帶著在曼城各項賽事只打進11球的尷尬成績單，昔日紫百合佛羅倫薩的希望之星約維蒂奇重返意甲聯賽，加盟國際米蘭。出於對他的信任，也為了顯示他重振雄風的決心，約維蒂奇被俱樂部授予10號球衣，雙方簽下1份長達4年的合約。

在佛羅倫薩時期，約維蒂奇風華正茂，是意甲聯賽中融技術、意識、速度甚至是顏值於一身的偶像級明星。然而加盟曼城之後，約維蒂奇卻得不到時任主帥佩萊格里尼的重視，再加上阿圭羅在藍月亮陣中不可撼動的地位，JOJO很快就迷失自我，加盟國際米蘭實際上是他自我挽救的出路。

約維蒂奇在國米的首秀是完美的，憑藉他的進球，國際米蘭主場1:0力克亞特蘭大取得賽季開門紅。此後，這支老牌豪門拿到了賽季開局階段的5連勝，大有中興的跡象，約維蒂奇更是在5場比賽中打進3球，儼然成了藍黑大腿。但很快，國際米蘭在當賽季第6輪主場1:4不敵佛羅倫薩，JOJO在那場比賽中因傷沒有出場，等到在第8輪復出時，他卻狀態全無，球隊也漸漸恢復到了那種渾渾噩噩的狀態之中。到了賽季中段，約維蒂奇已經得不到曼奇尼的信任而淪為板凳球員，而且惱人的傷病也再次找上了他。整個賽季下來，他只有26次出場，僅交出打進3球助攻3次的成績單，表現難以令人滿意。

2016-2017賽季，在弗蘭克-德波爾手下，JOJO徹底被邊緣化。半個賽季只拿到了可憐的67分鐘出場時間，即便球隊中途換帥也沒有能夠改變他的命運，皮奧利同樣認為JOJO不是自己需要的前鋒。就此，黑山人和國際米蘭10號的緣分走到了盡頭。2017年初，約維蒂奇轉投塞維利亞，但他的職業生涯並沒見起色。此後，他傷傷停停，再也不見往日的威風。

若昂-馬里奧：不配10號的10號

這位「大神」可以說是國際米蘭歷史上最糟糕的10號，因為他的表現根本配不上10號球衣。如果說他的幾名前任還有過高光表現，那麼這位馬里奧連高光是什麼都沒見過。

2016年夏天，國際米蘭豪擲4000萬歐元從葡萄牙體育買進這名當時只有23歲的葡萄牙中場球員，他也被看作國際米蘭尋求突破的重要棋子。然而，他的表現只能用糟糕透頂來形容。從賽季初被絕對信任的主力，漸漸淪為可有可無的輪換球員，還未高開就已經低走。若昂-馬里奧在國際米蘭的首個賽季只留下了30次出場、3粒進球和5次助攻的平庸數據。起初，人們對於他的定位和期待是球隊的中場核心，但到頭來卻發現這不過是一個美好的幻想，根本就不切實際。

第2個賽季，斯帕萊蒂治下的國米冬窗期間從巴薩租借來拉菲尼亞以增強中場的控制力，馬里奧則乾脆被租借到英超球隊西漢姆聯。即便到了英超聯賽，馬里奧同樣沒能打出身價和水平。雖然13次出場有12次是首發，並有2粒進球，但西漢姆聯對於買斷馬里奧並無任何興趣，最終他只能回到梅亞察。

2018-2019賽季，馬里奧代表國米在意甲出戰20次，僅打進1球，表現毫無起色。伴隨著能夠重新穩定參加歐冠，球隊中場陣容開始大換血，馬里奧的藍黑生涯也走到了盡頭。2019年夏天，他正式轉會俄超莫斯科火車頭，去歐洲二流聯賽繼續追求自己的球星夢想。

小結：藍黑10號註定多災多難 勞塔羅值得期待

斯內德似乎只是上天送給國際米蘭的一個驚喜，在短暫的輝煌之後，藍黑色的10號戰袍很快回歸平庸。這些球員如同走馬燈一般，他們在這支球隊有著各種各樣的經歷，離開的理由也是千奇百怪，但卻殊途同歸——無法復興10號球衣昔日的輝煌，也無力幫助國際米蘭開創下一個新時代。現在，這個重任落在了勞塔羅-馬丁內斯的身上，本賽季在孔蒂手下，他和盧卡庫的鋒線組合越來越有感覺，或許在時隔10年之後，國際米蘭的10號球衣又會迎來一位真命天子，一位真正的巨星。
