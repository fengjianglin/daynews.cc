本田圭佑先憑藉良好的競技狀態贏得維特斯的一紙合同

天天要聞
2019-12-08

　　自今年6月份從澳洲球隊墨爾本勝利離開之後，本田圭佑一度無球隊問津，淪落到在推特上尋求再就業。在經過一番努力下，本田圭佑先是跟著荷甲勁旅維特斯進行訓練，最終憑藉自己的表現與良好的競技狀態打動教練組，贏得了維特斯的一紙合同。

本田圭佑

　　當地時間11月24日，輕鬆取得勞工證後的本田圭佑身披33號球衣，在荷甲第14輪維特斯客場0-2不敵鹿特丹斯巴達的比賽中首發出場81分鐘，這是昔日日本天王時隔186天參加的第一場正式比賽。

　　本場比賽本田圭佑出現在4231陣型中的前腰位置，負責球隊的前場串聯組織等任務。可以看到的是，33歲的本田圭佑寶刀未老，傳球意識等方面仍舊是球星級別的存在，在場上發揮的遊刃有餘，甚至險些收穫重返歐洲的處子球。

本田圭佑中場隨球轉身傳球

本田圭佑險些破門

　　雖然並沒有幫助球隊拿下比賽，但本田圭佑的表現已經征服教練組。6天後，在荷甲第15輪維特斯客場2-3海倫芬的比賽中，本田圭佑首發打滿全場，與上一場出現在4231陣型中的前腰位置。據數據顯示，本田圭佑本場比賽攔截五次（全場最高），傳球成功率高達92%，賽後被評分為全隊最高的7.7分，表現非常出色。

　　事實上，雖然維特斯這個球隊在荷甲範圍內都不算是什麼聲名遠赫的豪門，但諸多球迷對其並不陌生。2015-17年，不到20歲的中國足球超新星張玉寧便在這支球隊效力，並且創下了中國球員登陸歐洲聯賽最小年齡的紀錄。

　　沒有對比便沒有傷害！效力維特斯兩年的張玉寧因為實力不濟等原因所獲機會並不多，兩個賽季僅出場24次（首發3次），總出場時間為593分鐘，打進了3個進球，大部分時間悶坐在替補席獲踢預備隊。而33歲的本田圭佑登陸荷甲前兩場比賽便出場2次（2次首發）首發，總出場時間為171分鐘，上場時間已經有趕超張玉寧的勢頭。

張玉寧的留洋生涯並不成功

　　是的，你沒看錯！就是這麼一位連日本國家隊門檻都無法進去的本田圭佑，竟然還能夠在歐洲頂級聯賽球隊擔任「大腿」級別的主力。而被譽為中國足球未來10年中鋒位置最優秀球員之一張玉寧，早已結束歐洲留洋生涯回到中超效力。

　　值得一提的是，本田圭佑現在已經不再是專心踢球的職業球員，如今他還有柬埔寨國家隊主帥這個身份。

　　當然，本田圭佑的的履歷是張玉寧所無法比較的。這位昔日日本天王出道於大阪鋼巴青訓，曾效力過名古屋鯨魚、莫斯科中央陸軍、AC米蘭等名門。當年本田圭佑在AC米蘭效力更是身披象徵核心球員的10號球員，稱為亞洲足球的驕傲。

　　最近10年，以香川真司、本田圭佑、長友佑都、岡崎慎司等名將為首的日本黃金一代，在亞洲乃至世界足壇叱吒風雲。隨著時間的推移，本田圭佑這批人早已經淡出了國家隊，以柴崎岳、大迫勇也、堂安律為代表的日本中生代球員撐起了新的門面。此外，日本還有久保健英、三好康兒等小將在歐洲聯賽奮戰，未來他們將成為新一批日本黃金一代。

日本黃金一代

　　而我們呢？國足現在仍舊靠著與本田圭佑同一時代的蒿俊閔、張琳芃等人撐起門面，中生代球員只有武磊、顏駿凌寥寥幾人拿的出手，相對年輕的張玉寧、韋世豪等人還難以獨當一面，諸多U23更是要憑藉U23政策才能在聯賽有所機會。

　　哦，對了！39歲的鄭智還是是國家隊常客。

　　中國足球呀，這一切之殤何時難愈？球迷們何時才能走出痛惜之境？或許，還要很久吧……

　　（寒少）
