歐冠的五個偉大紀錄，C羅讓梅西也得望其項背，利物浦13戰進41球

2019-12-17 1,203條評論 3,197次閱讀 71人點贊

要說歐洲冠軍聯賽那可是公認的全世界最具影響力以及最高水平的俱樂部賽事，歐冠賽場也是足壇巨星證明自己實力所在地方，梅西、C羅、齊達內等等這些巨星無不在這裡留下了他們的光輝事迹，「外星人」羅納爾多雖然沒有奪得過歐冠冠軍，但他在老特拉福德的帽子戲法也證明了實力，他只是運氣差了點！現在，我們就來看看歐冠歷史上的五個偉大紀錄吧！

​

皇馬共13次奪冠

縱觀歐冠的賽場上，成就最為突出的那肯定是非皇家馬德里莫屬了。皇馬成立於1902年，絕對的老牌豪門了，從球隊里走出了許許多多的世界球星。在00年的時候，它還被FIFA評為是20世紀的最偉大的球隊！從皇馬的隊史來看，他們一共拿下了十三個冠軍。放眼整個歐洲足壇，能夠和皇馬匹敵的也就只有意甲豪門AC米蘭了。

​

C羅連續六次獲歐冠金靴

要說歐冠的賽場上最出色的球隊是皇馬的話，那最出色的球員肯定是C羅。雖然說梅西和他並稱絕代雙驕，兩人統治著足壇長達十幾余年，迄今為止，C羅拿下五次的世界足球先生，梅西略勝一籌拿了六次。但在歐冠的賽場上，第一人還得是C羅。他在這個賽場上可是創造了連續六次拿下金靴的記錄，C羅的這一成就讓梅西也得望其項背。梅西也不差，他拿了四次的歐冠冠軍，進球數也快趕上羅總裁了，我們作為球迷能見證這兩位超級球星的風采也是我們的幸福！

​

皇馬5年奪4冠

作為老牌的足壇豪門，皇馬在改制後的歐冠賽場上除了十三冠的紀錄之外，還有著五年四冠的偉大成就，也是第一支衛冕的球隊，這些記錄就連拜仁和巴薩都不得不甘拜下風啊！不愧是歐洲足壇底蘊最強的豪門，這也是皇馬能不斷吸引球星加盟的所在。

​

利物浦13戰共進了41球

還記得在17-18賽季的時候，利物浦在13戰里共進了41球！就單看這個數據我們就不得不說讚歎利物浦的進攻火力了。雖然在最後的決賽中被皇馬擊敗，但也值得敬佩！18-19賽季他們捲土重來，終於捧回了大耳朵杯，本賽季在英超高奏凱歌的他們，距離他們的首座英超聯賽冠軍已然不遠，歐冠他們也誓在衛冕！

​

10.2秒進球

歐冠的經典紀錄自然也少不了最快進球紀錄了。還記得當時在07-08賽季的時候，拜仁對戰皇馬，強強對決，火花四射。比賽開始才10.2秒，拜仁的馬凱就以迅雷不及掩耳之勢進了一球！當時別說觀眾了，就連當時在賽場上的皇馬球員都還沒有反應過來，想想都知道有多迅速了。
