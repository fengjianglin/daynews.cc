歐冠抽籤結果讓一隊竊喜！從瀕臨淘汰到重獲新生只用了36天

2019-12-17 721條評論 4,025次閱讀 297人點贊

引人矚目的歐冠1/8決賽抽籤儀式落下帷幕，這也是五大聯賽首次包攬歐冠16強席位。抽籤結果自然不乏皇馬VS 曼城以及馬競VS利物浦這樣的重量級對決，幾支球隊就算晉級也要掉一層皮。正所謂有人歡喜有人愁，有一支球隊在抽籤結果出來後卻暗暗自喜，它就是加斯佩里尼執教的亞特蘭大。

抽籤儀式結束後，亞特蘭大主席佩卡西在接受採訪時激動地說道：「能夠避開一些豪門球隊，抽到瓦倫西亞對我們來說也是最理想的對手了，這是一份很不錯的聖誕禮物。」

首次參加歐冠的亞特蘭大曾被譽為「神秘之師」，上賽季聯賽打進77球冠絕整個意甲。因為球場問題小組賽主場遷至聖西羅球場，這也為「真藍黑」贏得了不少的關注度。但令人意外的是，火力強大的他們歐冠前3場比賽只打進2球，卻被灌了足足11球。3連敗的開局讓人大跌眼鏡，沒有人能想到這支歐冠新軍還有晉級的機會。

然而奇蹟卻發生了！自11月7日主場逼平曼城開始，在短短的36天時間裡他們逆天改命般的進軍16強。在客場3球完勝頓涅茨克曠工後，球員們在更衣室里載歌載舞的慶祝，球隊班機從烏克蘭飛回義大利已是凌晨3點，但這並不能阻止球迷們燃放煙火迎接心中的英雄，整個貝爾加莫市成了歡樂的海洋。

在加斯佩里尼的調教下，有越來越多的球迷開始喜歡這支非典型的義大利球隊。與其他意甲球隊不同的是，亞特蘭大在保持攻防體系的同時攻防都非常流暢。雖然這支球隊沒有耀眼的球星，但幾乎每一個人都有持球能力。他們在進攻端快速簡潔的踢法贏得了非常多的射門機會，並且有著相當的射門精準度。在聯賽前16輪比賽里，亞特蘭大123次射正球門領先尤文圖斯16次高居第一，同時29個運動戰進球顯示了球隊極強的協同作戰能力。在防守端積極展開高位逼搶，另外小範圍圍搶的效率也很高，他們164次搶斷的數據同樣位居聯賽首位。

儘管淘汰賽的對手巴倫西亞實力不可小視，在剛剛結束的聯賽中險些掀翻皇馬，但劫後重生的亞特蘭大更能放下包袱。在鋒線核心杜萬.薩帕塔長期缺陣的情況下，他們在主場曾狂射尤文22腳，一度讓對手在防守上疲於奔命。待淘汰賽開始時，薩帕塔和伊利契奇將傷愈復出，在帕爾馬錶現出色的庫盧塞夫斯基也有望租借回歸，屆時球隊完全有機會向歐冠八強發起衝擊。
