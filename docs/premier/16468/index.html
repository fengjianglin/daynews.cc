1季兩戰，尤文結束恐德！成功的秘訣在與什麼？積累

2019-12-14 60條評論 1,113次閱讀 201人點贊

歐冠賽場上尤文又贏了，一場2比0乾淨利落干翻藥廠。六戰五勝一平，強勢出線！但是對陣勒沃庫森的兩場比賽除了給小組出線增加了六分以外，也在幫助尤文擺脫了一個窘境。

要知道。本賽季之前尤文對德國球隊的戰績一直都不算太好。在自己輝煌的意甲八連冠這些年，尤文僅僅在14-15賽季進決賽那年主客場雙殺「有特殊情況」的大黃蜂。除此之外的6場對陣德國球隊，3平3負。甚至當年小組墊底的門興，也能給尤文上上課，兩場悶平，讓尤文丟掉小組第一位置。德甲掌門就更別提。

其實十幾年前，尤文對陣英超的球隊很吃虧，無論是曼聯、阿森納、利物浦、切爾西。都曾經從尤文身上踏過。但是尤文更新打法。逐漸找回場子。在八連冠這些年對陣英超球隊拿到了8戰5勝2平1負的優異戰績。

同樣，在這幾年在歐冠的錘鍊下，他們也逐漸找到對陣德國球隊的方法。從2012-13賽季對陣拜仁被雙殺到2015-16賽季的主客場4比4。尤文的變化和成長是肉眼可見的。直到本賽季遇到藥廠，並拿出兩戰進5球丟0球的主客場雙殺戰績。

逐年成績上漲，逐年在歐冠頂級行列出現。漸漸地，積累到一定程度，成功就會到來，就像當年的拜仁一樣。
