C羅示範教科書式跑位，眨眼間啟動擺脫防守，三個人愣是沒有反應

2019-12-18 28條評論 1,043次閱讀 77人點贊

意甲第16輪，尤文在安聯球場迎來了烏迪內斯的挑戰。此次面對下游球隊，薩里的球隊自然是想拿到一場大勝，一方面是為了追分，畢竟聯賽積分榜第一位不是自己；一方面是為了給球迷一個說法，上一場聯賽輸球輸得有點慘。好在這場比賽尤文全隊上下一改頹勢，用酣暢淋漓的進攻拿到了一場久違的大勝。這場比賽後，尤文依舊排在積分榜的第二位，但是積分已經成功追上國際米蘭。

這場比賽，C羅成為球隊獲勝的最大功臣，他在90分鐘的時間裡上演梅開二度。值得一提的是，這是C羅本賽季代表尤文上演的第一個梅開二度。所以說，這場比賽是本賽季以來C羅在俱樂部發揮最出色的一場比賽。上演梅開二度後，C羅的聯賽進球數來到9個，接近上雙了。其實，他原本有機會上演帽子戲法，可惜他的運氣實在是太糟糕了。不過，通過這兩個進球，大家發現他找回了自己昔日的狀態，尤其是第二個進球，堪稱教科書式的進球。

比賽第37分鐘，尤文在前場調度。德西利奧將球傳給伊瓜因後，伊瓜因開始從邊路帶球來到中路。注意當伊瓜因來到本坦庫爾身邊的時候，C羅正在背靠著球門，而且他身邊站了三名防守球員，這三個人始終在盯著C羅。下一秒，C羅突然啟動，從人縫中向前跑，這時候伊瓜因也送出皮球。最後，C羅成功接到皮球，一腳推射破門。值得一提的是，在C羅轉身前插進行跑位的時候，三個防守人成為了擺設，他們愣是沒有任何反應，連防守的動作都沒有來得及做出來。

不得不說，C羅的這次跑位太絕了，堪稱教科書式的神級跑位。本來他慢騰騰地散步，還處在防守者的視野里，可眨眼睛他就瞬間啟動，成功突破防守。當然了，這個球伊瓜因送得也相當精妙，這絕對體現了傳球功底。一直以來，C羅之所以能夠成為超級射手，其中一個非常重要的原因就是他的跑位不可思議，神乎其神。他不靠蠻力，靠的是球商。一般人，還真的跑不出這樣的機會。

跑得出來，關鍵也得打進，這就是天生的殺手。C羅親身示範教科書式的跑位，他用這個進球告訴所有人他並沒有老，也向全世界宣告自己已經王者歸來。這一次上演帽子戲法，下一次那必須就要來一個帽子戲法了。
