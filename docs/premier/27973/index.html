難道想回爐培訓？卡帥渴望買來國米前鋒！5大歸化球員還不夠用？

2019-12-21 245條評論 1,173次閱讀 205人點贊

卡納瓦羅的高情商在於，面對記者談吐自如，而且很融入中國文化。率隊最終奪得恆大隊史第八冠，他在接受採訪時坦承，自己在「學習班」里也確實在「學習」，他表示和俱樂部高層其實就是很平常的一次交流，隨後一直在間歇學習如何克服球隊的傷病，打好對陣上港的天王山之戰。結果，恆大贏了上港，完成了對爭冠對手的賽季「雙殺」，這是奪冠之旅的分水嶺。

奪冠之後，卡納瓦羅當然有理由享受一下假期的輕鬆。他已經接受了多家媒體採訪，也多次談及自己執教中超的感受以及未來將回到歐洲執教。不可否認，卡納瓦羅如果幫助恆大繼續奪冠，再鍛鍊出更多新人，他和恆大將實現「互利互惠」。中超將讓他名利雙收，同時也是卡納瓦羅重返歐洲的一大跳板。近日，卡納瓦羅接受了義大利媒體《天空體育》的採訪。

談及那不勒斯現狀，卡帥說，「加圖索遭到質疑，但他去年在米蘭幹得很出色，我們都希望他在那不勒斯也能做到。因西涅必須重新再做自己一直在做的事情：進球並貢獻助攻，大家會忘記所有爭議。」談及伊布回歸傳聞，卡帥表示，「伊布是一位不可思議的球員，是真正的領袖。但那不勒斯必須考慮要培養新球員，伊布雖然偉大，但他已經38歲了。」

卡納瓦羅還在採訪中談及國米前鋒勞塔羅，「我之前聽說勞塔羅和斯帕萊蒂之間存在著問題，我對那感到很驚訝。勞塔羅是一名出色的球員，他的進球慾望很強，在賽場上做了很重要的工作，並且他還讓搭檔盧卡庫受益。」聯想到恆大陣容，卡納瓦羅甚至坦言，「如果可以的話，我明天一早就想簽下他。在我看來，他是那些能夠在中國賽場發揮非常出色的球員之一。」

除了蔣光太之外，恆大還擁有非華裔歸化球員高拉特、艾克森、阿蘭、費南多、洛國富，再加上「保塔」組合和多名本土實力強將，還不夠用？對於卡納瓦羅的發言，有球迷調侃道：「卡帥說話要小心，難道又要進學習班回爐培訓？」此前據滬媒報道稱，恆大下賽季將不會再引入外援。而且目前恆大等俱樂部仍在跟足協進行協商，俱樂部希望足協能放寬非華裔球員註冊與上場人數。

卡納瓦羅所稱讚的國米前鋒勞塔羅·馬丁內斯，1997年出生於阿根廷，偶像是哥倫比亞前鋒法爾考。2018年7月3日正式加盟國米，雙方簽了5年長約。從2017年1月起始，勞塔羅共代表阿根廷U20出場11次，打進7球。今年5月，勞塔羅代表國家隊征戰美洲杯，美洲杯1/4 決賽戰罷，阿根廷同巴西、智利、秘魯闖進四強。勞塔羅入選了《WhoScored》評選的最佳陣容。
