義大利的教練培訓聖地：沒有任何培訓教材，通過論文答辯才能畢業

天天要聞
2019-12-22

科維爾恰諾訓練基地，坐落於佛羅倫薩的郊區，美麗的菲索爾山的山腳，它是義大利各級國家隊的訓練基地，同時也是義大利的足球技術中心，國內唯一一個具備頒發歐足聯職業級等級教練的認證機構。

每年都有很多夢想執教義大利職業聯賽的人聚集到科維爾恰諾培訓。這裡是一個教練走向成功的起點，義大利足球史上許多偉大傳奇教練都曾在這裡受訓。

義大利籍的教練不僅數量多而且質量高

在2018-2019賽季，意甲聯賽中共有19傢俱樂部僱傭了本土教練，這一數字在歐洲五大聯賽中排第一。西甲聯賽緊隨其後，有18傢俱樂部聘請了西班牙教練，法甲有6名非法國籍的教練。英超聯賽只有5名英格蘭教練。

義大利籍的教練不僅在歐洲範圍內數量多，而且執教成績也相當優異。在英格蘭，曼奇尼、涅利和孔蒂分別在2012年、2016年和2017年率隊獲得英超冠軍，孔蒂還在2018年帶領切爾西獲得了足總杯冠軍。2019年的上半年，薩裡帶領著切爾西獲得歐聯杯冠軍。在法國、西班牙和德國，剛剛擔任埃弗頓主教練的安切洛蒂贏得了兩個聯賽冠軍，兩個國內杯賽冠軍和一個冠軍聯賽冠軍。

在義大利，阿萊格裡帶領尤文贏得了5個聯賽冠軍，4個義大利杯冠軍，並兩次進入歐冠決賽。其他成功的例子還有執教拉齊奧的西蒙尼·因扎吉，亞特蘭大的加斯佩里尼，AC米蘭隊主教練馬可·詹保羅。

上面提到的這些教練，除了他們的國籍一樣，還有另一個共同點:他們都在科維爾恰諾接受過教育。

到科維爾恰諾參加培訓的很多人都曾是職業球員，2018年7月，皮爾洛、吉拉迪諾、巴蒂斯圖塔等人曾在這裡學習，而在今年率領廣州恆大隊奪得中超冠軍的卡納拉羅也是這一期的培訓學員，今年中超聯賽下半段，當恆大讓卡納瓦羅去恆大中心學習時，難怪卡帥會很理直氣壯的說，我不需要學習，我的教練員證書是全球最高級別的。

科維爾恰諾的創辦理念：創新

1957年，兩個在佛羅倫薩極負盛名的投資人創辦了科維爾恰諾。其中一個是意甲佛羅倫薩俱樂部的投資人Luigi Ridolfi。另一個是 曾擔任義大利足球協會的副主席的Dante Berreti，。Ridolfi和Berreti希望在義大利創造一些獨特的東西，他們有一個共同的目標——把來自不同領域、背景和信仰的教練聚集在一起，分享想法，使之成為創新思維的溫床。

在科維齊恰諾培訓的學院，培訓主要有兩個方面。一是為期一個月的培訓。所有培訓的教練必須住在學院，每周培訓四天。他們需要在培訓中形成自己的執教風格，調整自己獨特的意識形態，證明自己值得通過嚴格的教練考試。第二是由學院負責人和他的團隊組織的面試。

科維爾恰諾與其他地方的教練資格認證最不同的部分在於每個受訓者都需要撰寫論文。論文的主題由教練根據需要自行選擇，但內容必須是翔實和獨家。在面試中，教練需要解釋和闡述他們的論文，指出撰寫論文的價值和目標，它如何在現代足球中發揮作用以及對未來足球發展的借鑒。

倫佐·烏里維埃里(Renzo Ulivieri)是教練培訓項目的負責人，也是主導教練論文答辯的人。烏里維埃里今年79歲，曾是那不勒斯和帕爾馬的主教練，在接受《紐約時報》的採訪時，他解釋了科維爾恰諾的一些做法。他講到，在教練考試中有兩句話是完全禁止的。一個是「在我的時代」;另一個是「我的足球」。

因為在科維爾恰諾的創辦理念中，他們希望學員能夠不斷尋求創新，過時的想法是被禁止的。思想必須是關於今天和明天的，而不是昨天起作用的東西。

而為了體現學院對於學員創新能力的嚴苛要求，烏里維埃里講了另外一個例子。在科維爾恰諾沒有任何的培訓書籍來給受訓教練學習。「來學習我們課程的教練沒有收到任何書籍。」因為這些書沒有任何意義，如果我要寫一本書，那可能要花兩年的時間。所以當我把它給你的時候，它已經兩歲了。已經過時了。」

他繼續說道:「足球有一些最核心、很基本的原則，這些原則雖然很古老，但他們一直存在，我們需要教導教練從新學習和領會這些知識，但除此之外，其他所有的內容都需要從零開始。我不能交給哪些教練「我的足球」，因為它已經是50年前的事情了，我需要交給他們十年後的足球，教會他們預測未來。

在烏利維耶里42年的職業生涯中，總共有22次執教經歷，其中包括在博洛尼亞20多年的執教。作為一名經歷過降級、管理超級巨星、面對爭議、在聯賽中不受關注和執教過義大利足球各個級別隊伍的教練，烏利維耶里經歷豐富、聰明而且和藹熱情，如今被看做是足協教練培訓負責人的最理想人選。

傳奇教練們在科維爾恰諾的畢業論文

孔蒂在2006年畢業時，他的論文題目是「關於4-3-1-2陣型的思考與視頻的數字化應用」，強調了4-3-1-2陣型的利弊、防守和進攻的設置、過渡以及球隊在有球和無球的情況下的運作方法。

孔蒂分析了比賽中可能存在的幾種情況，並考慮了一些非比賽因素，如心理對陣型的影響。論文一共38頁，孔蒂使用了大量圖表，儘可能呈現它所想到的細節問題，被看做是如今孔蒂執教風格的完美體現。

盧西亞諾·斯帕萊蒂的論文，題為「3-5-2陣型在現代足球比賽中的使用」，正如它的名字所寫的，這是一篇關於著名陣型的分析。他解釋了為什麼3-5-2陣型在現代足球比賽中如此重要，並給出了一些關於正確使用3-5-2陣型的建議和方法。在職業生涯的後期，斯帕萊蒂成功地運用了3-5-2陣型在羅馬取得了成功，並在俄羅斯帶領澤尼特隊贏得了俄羅斯聯賽的冠軍。

1997年，安切洛蒂的論文題為《足球的未來:更有活力》，強調了這項運動在未來將會發生的變化，並通過一些圖表，分析了球場內外活動的增加將如何改變未來的足球運動。

曼奇尼2001年的論文研究的是攻擊型中場在比賽中的作用和重要性。在他的執教生涯中，特別是在曼城，曼奇尼很好地利用了有創造力的中場球員。

薩里的論文題為《圍繞比賽進行周期化訓練》，他寫道:「通常一周有7次訓練。其中兩次是身體素質訓練。兩次是糾正上一場比賽錯誤的訓練，一次是針對全隊戰術的訓練，另一次是針對個別區域球員的特殊訓練(比如防守、中場、進攻)。」論文的內容很好的體現了薩里對於訓練計劃和內容安排的重視。

在科維爾恰諾，所有畢業教練的論文都完全開放，新的學員可隨時參考和閱讀。烏利維耶里希望年輕的教練們能夠從中獲得靈感，一代代傳承並發展出新的足球理念。

科維爾恰諾的傳承

科維爾恰諾有一座足球博物館。2000年正式開放，博物館由六個房間構成。第一個房間里保存著20世紀30年代贏得的兩屆世界盃冠軍以及幾名球員的原始球衣。第二個房間是一些足球傳奇人物的紀念品，包括馬拉多納和貝利。第三個房間是是關於義大利世界盃的一些比賽和裝備。第四間、第五間和第六間則是最近的輝煌:1968年的歐洲杯、1982年和2006年的世界盃，以及1994年和2000年的世界盃。這是一個熠熠生輝的地方，它頌揚足球高於一切，在保護過去的同時展望未來。

近幾年，由於安切洛蒂的在拜仁的失敗,曼奇尼在國米下課,以及前尤文圖斯主帥圖拉帶領的義大利隊未能打進2018年俄羅斯世界盃決賽圈，使得社會上出現了很多對於科維爾恰諾的批評聲。

但在支持者看來，科維爾恰諾的培訓方法是正確的。從英格蘭到德國，從法國到俄羅斯，從已經奪得的世界盃、歐洲冠軍聯賽冠軍和國內聯賽冠軍，無數次的勝利已經證明了這個機構的能力。

他們需要的是堅持和等待，等待新一代的義大利教練的出現，把足球帶到一個新的高度，重塑義大利足球的輝煌。

(文/歐陽凌羽)
