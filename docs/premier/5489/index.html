意甲綜述：拉齊奧絕殺薩索洛 紫百合不敵維羅納

2019-11-25 0條評論 17次閱讀 0人點贊

北京時間24日晚，意甲第13輪全面打響。其中，薩索洛1-2拉齊奧，博洛尼亞2-2帕爾馬，維羅納1-0佛羅倫薩。具體賽果如下 薩索洛1-2拉齊奧 因莫比萊連續8場破門，凱塞多終場絕殺 本場比賽在三色旗球場打響。第34分鐘，科雷亞中路直塞，因莫比萊禁區得球扣開防守推射遠角得手，0-1！第45分鐘，佩盧索助攻卡普托破門，1-1！第91分鐘，阿爾貝托妙傳，凱塞多完成絕殺，1-2！最終藍鷹客場驚險取勝。 博洛尼亞2-2帕爾馬 哲馬伊利絕平，兩隊握手言和 本場比賽在達拉拉球場打響。第17分鐘，庫盧塞夫斯基為客隊先拔頭籌，0-1！第40分鐘，奧爾索里尼助攻帕拉西奧扳平比分，1-1！第71分鐘，庫茨卡妙傳，亞科波尼再度為帕爾馬超出比分，1-2！第95分鐘，哲馬伊利接到帕斯助攻絕殺比分，2-2！最終兩隊握手言和。 維羅納1-0佛羅倫薩 迪卡爾米內製勝球，紫百合客場敗北 本場比賽在維羅納主場打響。上半場兩隊互無建樹。易邊再戰，第66分鐘，法拉奧尼助攻迪卡爾米內打破僵局，1-0！此後兩隊均無建樹，最終維羅納主場小勝。 (編輯：姚凡)
