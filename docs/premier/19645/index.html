強強對話！歐冠16強抽籤結果出爐，克洛普執教生涯首次交鋒西蒙尼

2019-12-17 2,940條評論 4,496次閱讀 565人點贊

北京時間12月16日19時，2019-20賽季歐洲冠軍聯賽1/8決賽抽籤儀式在瑞士尼翁的歐足聯總部舉行。衛冕冠軍利物浦抽中西甲勁旅馬德里競技，克洛普和西蒙尼兩位世界級主帥將第一次帶隊碰面。西甲豪門皇家馬德里遭遇藍月亮曼城，巴塞羅那對陣那不勒斯。

由於本賽季歐冠16強席位均被五大聯賽球隊佔據，所以這也比以往增添了不少看點。在1/8決賽中，西甲球隊將分別與英超和意甲球隊展開較量。其中馬德里競技坐鎮萬達大都會球場迎戰利物浦，這裡是紅軍今年六月份獲得歐冠冠軍的地方，西甲英超兩大戰術大師西蒙尼和克洛普的首次交手格外引人關注。「銀河戰艦」皇家馬德里將與瓜迪奧拉率領的曼城展開強強對話，鑒於瓜帥的前巴薩主帥身份，這場比賽同樣充滿看點。梅西領銜的巴塞羅那將迎來意甲勁旅那不勒斯的挑戰，巴薩這兩年的歐冠之旅給人的記憶並不美好，此番面對從來沒有在正賽中對陣過的那不勒斯，俱樂部主管阿莫爾在抽籤儀式後表示希望能夠在諾坎普解決戰鬥。此外西甲球隊瓦倫西亞對陣黑馬亞特蘭大的比賽同樣充滿著不確定因素，其中晉級的一隊極有可能在8強戰中成為攪局者。

根據賽程，歐冠八分之一決賽首回合將於2020年2月的18、19、25和26日展開，次回合則於3月10、11、17和18日進行。
