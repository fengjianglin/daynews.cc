將團長送去羅馬，可能是紅魔下的一步妙棋

2019-11-30 0條評論 16次閱讀 0人點贊

曼聯本賽季花巨資引進了馬奎爾，以他為核心來打造本隊的後防線。他本人表現可謂是中規中矩，他的到來加強了曼聯後防線的硬度和出球能力，現已被確認為曼聯未來的隊長人選。

然而作為馬奎爾中衛搭檔這個位置，目前就顯得不夠明朗了。本來作為馬奎爾搭檔第一人選的林德洛夫，以其出色的出球能力佔得了先機，然而抗壓能力弱，以及對抗能力不強等問題又在他這打上了問號。隊中另幾名中衛，不是技術粗糙沒有出球能力，就是玻璃體質不堪重用。相反因為中衛人員太多，本賽季租借去了意甲的團長斯莫林，在羅馬倒是混的風生水起。現在都在羅馬當上隊長了。

根據足球網站whoscored的數據，斯莫林以本賽季場均評分7.53分排在所有球員中的第7位，甚至高過了過去幾個賽季被譽為意甲第一中衛的庫利巴利。要知道在曼聯8700萬歐元引進馬奎爾之前，曾準備以1.5億歐元引進庫利巴利，是因為那不勒斯不賣才沒有成行。根據數據統計，斯莫林前12輪意甲每90分鐘正定成功5.5次，是庫利巴利的2倍。搶斷成功率是100%，同樣強於那不勒斯中衛。除此之外，傳球成功率也是團長勝出。紅狼希望以1300萬歐元的價格買下他。儘管曼聯的心理價位大概在1700萬歐元左右，但與庫利巴利的身價比起來，只有那不勒斯中衛的十分之一。

曼聯近幾年的引援敗筆很多，迪馬利亞、德佩、盧卡庫等都成為紅魔的匆匆過客，就連現在還在陣中的博格巴也難言成功。而現在動輒上億的引援價格，更是使引援工作需要慎之又慎。將不在建隊計劃中的棄將團長送去意甲，通過團長的表現似乎能從側面判斷出意甲目前的水平，而是否值得為心儀的中衛庫利巴利掏出這1.5億歐，答案似乎也挺明顯了。意甲現在球員水分還是很大的，比如前幾個賽季身價一度被炒到1億，受多家豪門關注的前鋒貝洛蒂，目前可能就值5000萬，還沒有球隊願意為他出手。

為馬奎爾選個好搭檔，不管是內部挖潛，還是說從外部引進，必然是曼聯未來需要辦的事。而對於即將打開球星超市的那不勒斯，曼聯是否要出手引進庫利巴利，還是應該多考慮考慮。畢竟庫利巴利已經28歲，只比團長小2歲，而且這1億多的身價是否真的值，不說否定吧，至少現在也是個問號了。
