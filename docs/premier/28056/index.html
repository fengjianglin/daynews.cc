多特有實力客勝霍村，紫百合主場有優勢，羅馬後防吃緊恐丟分

2019-12-20 421條評論 1,751次閱讀 817人點贊

上期回顧：前天東亞杯日本輸給了韓國，還是身材上吃了虧，韓國定位球威脅不小。另外一場意甲推薦還是相當穩，薩索洛客場2-0完勝布雷西亞，就是實力上的差距。近期戰績不太理想，後面一定繼續認真總結提高。還是穩紮穩打不貪高賠，保證命中為先，為大家帶來更多精準分析！

周五016 霍芬海姆VS多特蒙德

球隊之前客場令人驚訝的爆冷擊敗拜仁慕尼黑開始，豪取聯賽五連勝，但好景不長，近4場聯賽1平3負，幸好本周中客場「爆冷」擊敗狀態火熱的升班馬柏林聯合，結束聯賽4場不勝。本賽季是霍芬海姆失去功勛主教練少帥納格爾斯曼的首個賽季，且多名主力球員離隊，本賽季註定艱難前進。聯賽前6輪，霍芬海姆表現「正常」，僅取得1場勝利。

目前聯賽16輪過後，多特蒙德8勝6平2負，勝率只有50%，在拜仁本賽季如此糟糕發揮的情況下，多特並沒有把握好千載難逢機會，在爭冠中佔得先機，爭奪聯賽冠軍前景暗淡。今夏轉會窗口行動迅速，從德甲其他球隊引進多名實力派球員加盟，其中再次從拜仁吃回頭草，引進曾經在球隊效力的胡梅爾斯加強球隊後防線實力。

霍村上輪戰勝擁有魔鬼主場的柏林聯合，取得這場勝利後霍村終止五輪不勝，多特各項賽事4連勝後，上輪被萊比錫3-3逼平，看來球隊只能會找軟柿子捏，近幾個賽季多特都無法從霍村主場帶走勝利。亞盤受半一起步，是近幾年多特作客最深盤口之一，後市利用基本面降盤給上盤降溫，看好多特帶走三分。

推薦：負 sp：1.66

周五018 佛羅倫薩VS羅馬

球隊核心小基耶薩自傷愈復出後，仍需要時間來找回腳感，他在復出後僅僅交出一記助攻；而與國米一役他只踢了60分鐘就被提前換下，疑似受傷或有機會缺席本場比賽。佛羅倫薩儘管上輪戰平國米，但主場已經連續4輪不勝，當中甚至有0-1不敵升班馬萊切，主場表現低迷，另外紫百合過去10次主場迎戰羅馬不勝率達8成，沒有任何優勢。

在過去的兩個月里，羅馬在意甲聯賽中得到了19分，只有同城的拉齊奧做得更好，由於上一輪卡利亞里落敗，因此紅狼以3分的優勢成功反超闖入前四，陣中士氣正值高昂。雖然羅馬最近中後場傷停較為嚴重，但他們依然展現不俗的防守，至今場均失1球與尤文、拉齊奧並列意甲聯賽防守第2佳，本場斯莫林有望復出，進一步增強後防實力。

紫百合已經6輪不勝，上輪他們戰平了擁有奪冠實力的國米，羅馬上輪3-1戰勝意甲弱旅費拉拉，近6場各項賽事4勝2平保持不敗，不過球隊半場0-1落後費拉拉，防守仍存隱患。近期戰績和硬實力都是羅馬佔優，澳門只有平手起步，這對於去羅馬過於便宜，大部分亞盤公司受平半，歐賠只有232賠率組合，對主隊有提防，看好紫百合不敗。

推薦：讓勝 sp：1.54
