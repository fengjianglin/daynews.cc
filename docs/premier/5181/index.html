西媒分析C羅金球失敗原因：他依舊是最好之一，只可惜離開了皇馬

2019-12-04 0條評論 13次閱讀 0人點贊

2019年足壇最後一項大獎已經揭曉，阿根廷天王梅西力壓眾多球星，捧起了本年度的金球獎獎盃，而他也以6次獲獎的紀錄傲視群雄。在本次評選當中，荷蘭後衛范迪克以7分的分差屈居第2，梅西的老對手C羅位列第3，近9年來首次跌出金球獎的前兩名。

客觀來講，本次金球獎的前3名得主在過去一年當中的表現都各有千秋，因此對於最終排名外界也是「仁者見仁智者見智」。西班牙《馬卡報》在今天就表示C羅原本是有可能拿到本年度的金球獎的，只不過是在沒有離開皇馬的前提下。

去年夏天，C羅以1億歐元的價格從皇馬轉會到尤文，在為這家意甲豪門效力的一年中，C羅聯賽打入28球，歐冠攻入6球，最終拿到了意甲冠軍並殺入了歐冠八強。國家隊層面上，C羅的表現更加完美，在他的率領之下葡萄牙一路過關斬將，拿到了首屆歐洲國家聯賽的冠軍。

​但是最終，C羅只排在金球獎評選的第3名，分數上也被前兩名落下了一大截。《馬卡報》評價說：C羅體會到了過去10年「其他球員」所體會的痛苦，他也像迪馬利亞、厄齊爾、伊瓜因一樣，在離開皇馬之後職業生涯走上了下坡路。

《馬卡報的》說法雖然有些片面，但不得不承認還是有一定的道理。在過去數年前，C羅與皇馬一直都在互相成就，C羅拿金球獎的年份基本上也都是皇馬取得不錯成績的賽季。但是今年這種關係被中斷了，皇馬因為失去了C羅曾變得一蹶不振，直到近期才算找回了一些狀態；而C羅也因離開了皇馬，導致在影響力上有所下降。

最後，這家西班牙報紙總結說：C羅依舊是當今足壇最出色的球員之一，這一點毫無疑問，但是與皇馬分道揚鑣讓他失去了跟梅西在同等條件下掰手腕的機會。另外還有讓人更加擔心的是，C羅連續兩年缺席金球頒獎典禮，也很容易讓他在擁有投票權的媒體中留下「話柄」。
