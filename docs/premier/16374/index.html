尤文3-1烏迪內斯，誰注意到C羅評分？接近滿分！球迷：還能火三年

2019-12-16 2,042條評論 4,905次閱讀 2413人點贊

北京時間12月15日晚，意甲第16輪一場焦點戰，尤文圖斯主場迎戰烏迪內斯，最終，尤文3-1烏迪內斯，此役，C羅上演了梅開二度，尤文在積分榜上也暫時超過了國米。

本場比賽，薩里在前場排出了C羅+迪巴拉+伊瓜因的三叉戟組合，這樣的安排想必也會有人質疑，畢竟在之前有的比賽中，三人似乎不是足夠的默契，但這場不一樣，三人配合嫻熟，尤其是C羅，也徹底找到了進球的感覺，狀態非常之好。

比賽開始後不久，C羅就為球隊贏得了一個自由球；第7分鐘，C羅策動進攻，伊瓜因再交給馬圖伊迪，後者大力轟炸門被沒收；第8分鐘，尤文取得領先，C羅跟進凌空抽射入網。

第11分鐘，迪巴拉小角度射門被封堵出了底線；在這之後的一段時間，場面相對沉悶；第20分鐘，迪巴拉吊射破門，不過裁判示意他手球犯規在先、進球無效，並出示了一張黃牌；第27分鐘，C羅展現了他的傳球能力，C羅回敲，伊瓜因的射門被化解；第35分鐘，伊瓜因再次射門又被撲出。

終於，在第36分鐘的時候，尤文擴大了比分！伊瓜因一腳精妙直塞，C羅順勢推射入網。

第44分鐘，尤文又進球：迪巴拉角球開出，德米拉爾頭槌擺渡，博努奇頭槌入網。

上半場，尤文就3-0大比分領先。下半場一上來，C羅塞球，迪巴拉的射門被封堵；第51分鐘，又是C羅的傳球，迪巴拉的怒射稍稍高出了橫樑；隨後，C羅還在防守端做出頭槌解圍；第59分鐘，C羅抽射製造威脅；第62分鐘，C羅和迪巴拉玩起了撞牆配合；第69分鐘，C羅的低射擊中了邊網；第83分鐘和第84分鐘，C羅連續兩腳射門都非常有威脅。

尤文3-1烏迪內斯，誰注意到C羅評分？在whoscored的賽後評分中，C羅獲得了9.2分的高分，接近滿分！這個評分比較合理，除了全場的2個進球之外，C羅也送出了不少的妙傳，而且在最後階段，C羅險些上演了帽子戲法，堪稱全場最佳。有球迷也調侃：還能火三年。希望C羅延續好狀態！加油，總裁。
