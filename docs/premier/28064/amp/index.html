北京時間12月21日3點45分佛羅倫薩主場迎戰羅馬，雙方能擦除怎樣的火花，讓我們期待他們帶來的精彩比賽。

佛羅倫薩本賽季16戰4勝5平7負，進球20失球24，積17分排名14，主場8戰2勝3平3負，進球9失球10。本賽季16場意甲除了2場蛋蛋，其他14場下半場全有球。

羅馬本賽季16戰9勝5平2負，進球29失球16，聯賽積32分排名第4，客場8戰4勝3平1負，進球11失球5。羅馬贏球能力還是高出對手佛羅倫薩不少，本賽季雙線作戰，但16場意甲也贏出9場，是對手勝率的2倍多，實力面優勢不小。

雙方近10場交戰佛羅倫薩4勝2平4負。佛羅倫薩主場也毫無優勢可言，跨賽季近19個意甲主場只贏2場，即便都是本賽季的贏球，但本賽季主勝率也就25%，羅馬跨賽季近10個意甲客場多達5場平局，其中3場蛋蛋，為近期意甲客場蛋蛋王。

初盤開出主隊受讓平半球，水位也有超高水調整為中高水，大小球升盤降水。佛羅倫薩超小球不少，近10場意甲多達4場最多1球的超小球，即便在意甲也屬於很高的超小率。羅馬客場半大能力低下，跨賽季近12個意甲客場也就4場半場有球，即便在意甲也屬於很差的半大率。佛羅倫薩近期狀態下滑嚴重，反觀羅馬近況出色，近6仗各項賽事錄得4勝2和的佳績，加上5次贏盤，來勢洶洶的羅馬此番出征有望再次贏球贏盤。
