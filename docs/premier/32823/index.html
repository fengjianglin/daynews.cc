巴薩是梅西的，那麼巴薩會不會用梅西去換內馬爾呢？

2019-12-24 319條評論 2,729次閱讀 1725人點贊

大家覺得巴薩會不會用梅西換內馬爾，這個假設成立嗎？按梅西目前當然不可能離開巴薩，我也覺得梅西在巴薩退役的結局對雙方都最完美，但那是球隊繼續保持競技成績的前提之下，別看現在這些球迷頭拱地的高喊梅球王萬歲，如果哪天球隊的成績下降，他們翻臉會比誰都來得更快，我還記得學生時代看的ac米蘭，馬隊與米蘭的關聯夠緊密了吧，但是在安胖後期米蘭成績下滑時，馬隊也開始表現不穩，那些所謂的鐵杆球迷很快就翻臉了。

我還記得當時報紙上採訪那些義大利球迷，他們是怎麼說的，馬爾蒂尼？曾經偉大過，不過也是時候改換換人了，這種氛圍下，也傳出了一些馬爾蒂尼的轉會緋聞，我記得先後有曼聯和皇馬的消息，甚至進入了商談詢價的階段。 最後還是馬隊本人和父親先後表態這件事才最後作罷，所以，我對商業足球時代的球隊和球迷都不要太報幻想了。

c羅在皇馬的地位是跟梅西在諾坎普比不了的，c羅在皇馬能被自家球迷噓哭，梅西在巴薩那就是真正的國王，第一，2.2億是不可能2.2億的，這輩子都不可能2.2億的，換梅西又不值，拿歐洲金靴，隊魂核心換內馬爾是肯定不值的，所以要換肯定要梅西下滑才有可能，再加上，梅西如果健康，踢到35應該不是問題，到時候內馬爾也29了，正在合同年，巴薩真想要直接免簽， 況且現在足壇年輕又強的前鋒太多了，巴薩光是近兩年選擇很豐富，內馬爾這樣年齡較大、玻璃人體質、有叛徒名聲（爭議），還2.2億身價的球員，拿什麼跟桑喬，斯特林，登貝萊，菲利克斯，姆巴佩等一眾只賣一半價錢的未來之星比？

即使內馬爾再強進步再多，這個結果都是不可能發生的 ，除非梅西自願離隊，梅西光在球隊里待著不踢球，也能替球隊吸引大量的關注，因為他已經是個傳奇 你手中有座獎盃時你會拿它去換另一座獎盃嗎？當然是留著，然後儘可能去獲得更多的獎盃，且不論梅西的象徵價值，僅就競技層面的意義而言，如果內馬爾夠得上梅西的話，當初早就在巴薩就取梅西而代之了，根本沒必要走。

對於球迷來說，提到巴薩，你最先想到的人是誰？巴薩逆轉巴黎的海報很能說明問題，對於絕大部分，是梅西 而且，梅西之於巴薩，從哈白布的錦上添花，尤其是到了現在，六金球，這種價值不言而喻 內馬爾很強，我承認，但他現在的整體表現依舊不如梅西，而且，當初走的方式確實不好看，會員球迷和我們這些外圍球迷考慮的東西可是不一樣的，會員球迷能接受嗎？

再去看看內馬爾走後當地球迷反應就知道了，說句最難聽的話，能讓梅西離開的，只有他自己，梅西的續約情況甚至會影響主領導的地位的，那句話其實很對，皇馬，是弗洛倫蒂諾的，曼聯，是弗格森爵士的，巴薩，是梅西的。
