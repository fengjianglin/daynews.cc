5佳球：庫蒂尼奧再現穿雲箭 薩拉赫0角度神仙球

2019-12-16 1,606條評論 3,768次閱讀 227人點贊

過去一周，歐冠和歐聯杯兩項賽事的小組賽階段告一段落，五大聯賽包攬了歐冠16強，國際米蘭生死戰告負與曼聯和阿森納一併征戰歐聯杯；周末聯賽，利物浦擴大榜首領先優勢，曼城客場3-0大勝阿森納，皇馬和巴塞羅那在國家德比到來前紛紛戰平，拜仁慕尼黑上演大逆轉，尤文圖斯在積分榜上再次反超國際米蘭。在新的一周來臨之際，搜狐體育盤點過去一周，評出本周5佳進球。

最佳進球一：薩拉赫0角度神仙球

12月10日，歐冠小組賽第六輪，利物浦客場2-0擊敗薩爾茨堡紅牛的生死戰，薩拉赫第58分鐘打進一記神仙球：亨德森後場長傳，薩爾茨堡紅牛後衛解圍失誤，薩拉赫斷球後，擺脫兩名防守球員後（包括對手出擊的守門員），在禁區右側非常狹小的角度下，完成了0角度破門，幫助利物浦鎖定勝局。

最佳進球二：拉希察世界波

12月14日，德甲聯賽第15輪，不來梅客場1-6慘遭拜仁慕尼黑逆轉的比賽，客隊球員拉希察第24分鐘打進一粒漂亮的進球：克拉森傳球，拉希察突破博阿滕攔截後禁區弧頂內勁射破門，拜仁慕尼黑守門員諾伊爾無能為力。

最佳進球三：戈斯林背身勾射絕殺

12月14日，英超聯賽第17輪，伯恩茅斯客場1-0爆冷擊敗切爾西的比賽，戈斯林打進一粒絕妙進球：第84分鐘，勒爾馬頭槌往前頂，戈斯林點球點背對球門後退，隨後一記勾射，攻破切爾西城池。這個進球，一度被邊裁認定越位在先，在經過VAR裁定後發現，戈斯林並不越位，是個好球。

最佳進球四：庫蒂尼奧美妙吊射

12月14日，德甲聯賽第15輪，拜仁慕尼黑主場6-1逆轉不來梅的比賽，庫蒂尼奧第63分鐘完成一記美妙的吊射破門，阿拉巴後場長傳，巴西球星前插到點球點左側，面對出擊的不來梅守門員輕鬆吊射，幫助南部之星3-1領先。

最佳進球五：庫蒂尼奧穿雲箭

還是同一場比賽，庫蒂尼奧在第78分鐘接佩里西奇禁區左側傳球，盤帶幾步後，在禁區弧頂左側遠射，穿雲箭的威力再現，皮球划出一道弧線擊中右側立柱彈進球門。庫蒂尼奧上演帽子戲法的同時，也幫助拜仁慕尼黑鎖定比分，主場6-1瘋狂逆轉不來梅。
