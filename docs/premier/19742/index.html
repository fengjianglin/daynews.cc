「米蘭專欄」親愛的AC米蘭，生日快樂

2019-12-16 223條評論 1,165次閱讀 109人點贊

AC米蘭在聖西羅進行了建隊120周年慶典，但紅黑軍團的這個生日過得並不開心。與薩索洛的比賽，筆者數十次（保守估計）捶打沙發和桌子，其中十次是因為佩戈洛撲出了米蘭的射門，兩次是因為萊昂擊中橫樑和立柱，伴隨著裁判吹響終場哨，球迷們一聲長嘆，然後紛紛在朋友圈等社交平台為米蘭送出一聲生日祝福。

也許，這個略顯「糟心」的生日就是米蘭近些年的縮影——選擇錯誤，策略模糊，運氣糟糕。事實上，這已經是紅黑軍團本賽季為數不多能夠創造出絕對機會的比賽，然而高質量的射門要麼被對方門將高接抵擋撲出，要麼被主場門柱拒之門外，特奧轟入了一記世界波，又因為凱西的手球被判無效，本納塞爾拿到了空門的機會，瞬間的遲疑就目送機會溜走……

球迷們當然可以一如既往地去怪蘇索的「不作為」，畢竟他得到了《米蘭體育報》賽後的最低平分，也可以去指責凱西，是他的手球讓特奧的進球作廢——這個手球對球的運行軌跡幾乎沒有影響。但問題的關鍵還是在於，皮奧利沒有走出自己執教的舒適區，他的變化還不夠多樣，尤其是在無法打破僵局之時。

恰爾汗奧盧踢成了前鋒，皮亞特克日常跑去禁區邊緣策應，凱西的位置有些尷尬，中場失勢時，本納塞爾在壓迫下不斷地回傳失誤，這是皮奧利的選擇錯誤。當比賽進入下半場後半段，米蘭似乎在全力猛攻，又好像要穩固後防，在三分與一分間徘徊，這是皮奧利的策略模糊。而面對實力並不強的薩索洛，米蘭總共射正了12次（要知道米蘭前15輪射正數也不過60多次），卻沒有一次有效地攻破對方的城門。

與薩索洛一戰米蘭的數據

這確實和米蘭過去幾年走過的彎路很像，不是嗎？

從中資時代開始，米蘭所買的球員往往無法踢出加盟前的水平，一度淪為球星黑洞，這是選擇錯誤。在現代足球商業化發展的時期，貝盧斯科尼固守著家族企業的經營模式，錯過了發展的黃金階段，而中資到來也是急功近利地瘋狂投入，忽視了競技體育科學發展的軌道，於是球隊越陷越深，這是策略失誤。上個賽季，米蘭最終以一分之差無緣歐冠，這是運氣不佳。是多重因素，讓米蘭陷入了如今的困境。

即便如此，在米蘭120歲生日到來之際，依舊有超過6萬名球迷湧入了聖西羅，將這塊聖地裝扮成紅黑色的海洋，音樂響起，旗幟飛揚，大概對於羅森內里們來說，這樣的吶喊比勝負本身更加重要。就像AC米蘭，對於米蘭球迷早已不是一支球隊，而是一份信仰。球隊可以輸，也可能不可遏制地墜向深淵，但在吐槽之後仍舊會繼續守候著。

畢竟，紅黑色的低谷不過短短十年，與120年的滄桑史詩相比，不過是光陰一瞬。在百廿光陰之中，米蘭有過44年無冠的低迷時刻，有過兩次降級的恥辱記憶，也有過上世紀尾聲沉淪的時光，但最終都像馬爾蒂尼說的那樣：「我們曾經跌倒過很多次，但無論跌倒多少次，我們都會重新站起來。」

卡卡在ins中寫道：「能夠成為這120年歷史的一部分，我感到非常自豪，祝賀你，AC米蘭！」

舍甫琴科在接受採訪時說道：「這段歷史是無法忘懷的，120周年慶典對所有人來說都是一個節日，米蘭是獨一無二的，它是一部永遠也不會完結的偉大電影。」

安切洛蒂則在推特上表示：「生日快樂，米蘭！你不再是一個孩子了，但那些回憶，那些榮譽，那些激情和那份愛永遠都在。一直有人追隨你，擁戴你，你也因此永遠年輕。」

在與薩索洛比賽的中場時間，包括薩基、卡佩羅、多納多尼、博班、馬爾蒂尼、巴雷西、里維拉、塔索蒂、羅西、因扎吉、迪達、安布羅西尼、阿巴特、科斯塔庫塔在內的多名名宿進入聖西羅的草皮進行合影留念，他們每一位傳奇身上都承載著關於紅黑色的印記，我們因為他們而愛上米蘭，然後開始了漫長的守候。

從2009年到2019年，米蘭總共只拿到過一個意甲冠軍和兩個義大利超級盃冠軍，對於一支18次奪取意甲冠軍和7次君臨歐洲的王者之師來說，這樣的成績實在慘淡，但也正是在這樣的過程中，我們明白了一個道理：好的壞的，都是AC米蘭。也許故事的開始我們是因為華麗而愛上她，但隨著時間推移，喜歡變成了愛，衝動變成了感情，然後我們和米蘭一起歡呼，陪米蘭一起沉默，黑暗的時光，也成為了歷史的一部分。

沒能戰勝薩索洛，比賽的結局並不完美，無法以最好的姿態迎接慶典，這或許就是生活的真相。對於米蘭來說，過去的十年不算美麗，但也正是這種起起落落更加凸顯出感情的純粹與真實，2019年的最後一場比賽，米蘭還將迎來勁敵亞特蘭大，隨後將開啟下一個十年的征程。

那些光榮夢想，那些黯然神傷，那些張揚歲月，那些沉寂時光。親愛的AC米蘭，120歲生日快樂！無論過去，現在，還是未來，我們一直在這裡。
