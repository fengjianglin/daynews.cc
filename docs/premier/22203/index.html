申花舊將後悔加盟尤文：我為自己感到羞愧，這是最糟糕的一次經歷

2019-12-18 373條評論 2,004次閱讀 128人點贊

球員時代，阿內爾卡幾乎拿遍了所有集體榮譽，並先後為阿森納、皇馬、利物浦、和切爾西等歐洲豪門效力。雖然早已退役，但回想起在尤文的那段經歷，阿內爾卡仍無法釋懷：「很不幸，這是我球員生涯中最糟糕的一段經歷，因為我只踢了45分鐘，還從未在主場球迷的面前登場比賽。」

外媒截圖

2013年年初，阿內爾卡以欠薪為由，向上海申花提出解約，但沒有被接受。在無法獲得自由身的情況下，阿內爾卡只能以租借的形式，與尤文圖斯簽約5個月，租借費為60萬歐元。

阿內爾卡

剛來到尤文時，阿內爾卡對新東家極盡溢美之詞，這位著名的「足壇浪子」甚至公開表示，他在尤文找到了家一樣的感覺：「我非常驕傲可以成為尤文圖斯的球員，我有一些前隊友曾在尤文效力，他們告訴我，這是一家偉大的俱樂部，當我走進更衣室時，就像是回家一樣。義大利足球舉世聞名，這裡有很多偉大的球員，在意甲奪冠的難度非常大，但尤文可以在任何比賽中都取得勝利，我希望自己可以儘快融進球隊。」

阿內爾卡

當時，尤文的主帥是以激情著稱的孔蒂，而阿內爾卡的競爭對手，則包括武齊尼奇、喬文科、誇利亞雷拉和馬特里。因為長期沒有參加系統訓練和正式比賽，阿內爾卡的競技狀態並不好。時任總經理馬羅塔曾呼籲球迷，多給法國人一點耐心：「就現階段來說，阿內爾卡還很難打上主力，他需要慢慢恢復狀態，究竟如何使用他，還得看主教練的想法。」

阿內爾卡

在之後的五個月里，阿內爾卡幾乎把板凳坐穿，時至今日，他仍對這段經歷耿耿於懷：「我決定接受尤文的邀請，因為它是一家大俱樂部，有很長的歷史，也因為它是意甲衛冕冠軍。很不幸，這是我球員生涯中最糟糕的一段經歷，因為我只踢了45分鐘，還從未在主場球迷的面前登場比賽。每當有人跟我提起尤文時，我都會迅速轉移話題。我為當年的轉會感到羞恥！在我效力過的所有球隊中，我都有過進球，但都靈（指尤文圖斯）除外。我不是不尊重這家偉大的俱樂部，但我不敢說我曾經在尤文效力，因為這段經歷很糟糕。」

阿內爾卡
