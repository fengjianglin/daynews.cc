恆大第一簽？卡帥心儀的鋒線新援浮出水面，直言明天就想簽下他

2019-12-21 282條評論 1,678次閱讀 305人點贊

近日，廣州恆大主教練卡納瓦羅在接受《天空體育》的採訪時表達了他對勞塔羅的欣賞之情。卡納瓦羅直言對勞塔羅很欣賞，並且表示如果可以的話，我都想明天就把他簽下。如果這個新聞報道屬實，不知道卡納瓦羅對勞塔羅猛誇，能否讓恆大下定決心引入這位阿根廷鋒霸。

恆大目前外援配置有保利尼奧、塔利斯卡、朴志洙，這三位實力不俗的外援是恆大本賽季拿到中超冠軍的功臣。媒體報道足協下賽季外援政策不會進行調整，依然是允許每支球隊註冊四名外援。所以恆大要是有意引進勞塔羅這位阿根廷鋒霸，應該是具備一定的操作性。不過恆大能否同意卡納瓦羅的要求，從國米引入勞塔羅就是一個巨大未知數。因為之前卡納瓦羅一直呼籲恆大，給他配備一名高水平中鋒。卡納瓦羅技戰術體系中一直急需強力中鋒助陣，之前他在出任權健主帥時，就力主球隊從德甲引入了強力中鋒莫德斯特。

卡納瓦羅上賽季接手恆大之後，多次要求球隊給他招一名強力中鋒。結果恆大高層對卡納瓦羅的要求一直熟視無睹，直到本賽季中超夏季轉會時才從上港引入了歸化強援艾克森。已經過了巔峰期的艾克森，顯然不是卡納瓦羅中意的中鋒新援。從卡納瓦羅多次呼籲恆大給他配備強力中鋒，球隊一直沒有滿足他的要求，這就說明卡帥在恆大引援沒有太多的決策權。

由於卡納瓦羅的引援需求一直得不到恆大積極回應，所以卡納瓦羅只能通過媒體隔空向球隊高層喊話和施壓。卡納瓦羅這次接受《天空體育》專訪時明確表示想簽下勞塔羅，恆大或許該認真考慮一下卡納瓦羅的引援需求。勞塔羅效力國米時表現不錯，多次在比賽中有關鍵進球。所以勞塔羅的表現除了吸引恆大主帥卡納瓦羅的注意，還引起了歐洲頂級豪門巴薩的青睞。

巴薩想要引進這位阿根廷鋒霸來球隊接班蘇亞雷斯，蘇亞雷斯已經是年過三十的老將。所以恆大要是真有意簽下勞塔羅，恆大就需要趕緊出手簽人。否則卡納瓦羅欽點的恆大鋒線新援，就會被財大氣粗的巴薩搶走。希望恆大在未來轉會市場上，能夠給球迷帶來驚喜。

（羅掌柜）
