國米歐聯杯對手意外換帥，曾遭武磊球隊7-0雙殺，張康陽收大禮

2019-12-17 1,223條評論 2,475次閱讀 1294人點贊

近段時間，國米的狀態並不太好， 歐冠小組賽最後一輪輸給巴薩，無奈歐冠小組賽出局，回到聯賽中的「藍黑軍團」，也沒能找回勝利的感覺，客場1-1被佛羅倫薩絕平，各項賽事連續3場比賽不勝，意媒認為，國米陣容太單薄是制約他們取得好成績的根本因素，要知道，與佛羅倫薩的一戰，捉襟見肘的孔蒂甚至啟用了年僅17歲的中場小將盧西恩-阿古梅。

不過，昨晚的歐聯杯抽籤，國米算是得到了一個不錯的好消息，因為他們的1/16決賽對手，是來自於保加利亞的球隊——盧多戈雷茨，即便國米現階段人員不夠充足，甚至下輪聯賽停賽+傷缺，可能多達9人之多，但所謂「瘦死的駱駝比馬大」，面對這個對手，國米的勝算還是很大的。

而且，就在歐聯杯抽籤結束不到一小時，盧多戈雷茨方面就官方宣布換帥。值得一提的是，這支球隊在小組賽階段，與武磊效力的西班牙人同組，西班牙人首回合主場面對盧多戈雷茨，瘋狂打進6球，6-0大勝，次回合客場挑戰，武磊首發出場，隊友坎布薩諾攻入唯一進球，1-0帶走3分。也就是說，國米的淘汰賽首個對手，曾經兩回合0-7被西班牙人雙殺，實力差距可見一斑。

因此，對於國米來說，抽到盧多戈雷茨算是好籤了，雖然關注度不高，但第一距離不算太遠，避免舟車勞頓，第二實力不算強，可以讓主力輪換休息，專心聯賽。而且對手剛剛換帥，磨合期還有段時間，等到每年2月再戰，如此倉促地面對意甲領頭羊，恐怕盧多戈雷茨自己也知道希望不大。

而對於國米來說，冬窗即將來臨，球隊人員短缺的問題很快就可以得到彌補，此外，包括森西、桑切斯等主力球員，邁過每年1月，也基本上可以滿格復出，屆時，對於孔蒂的排兵布陣也將帶來極大的幫助。張康陽已經定下了目標，要求孔蒂不能輕視歐聯杯和義大利杯的戰線，至少要保障拿下一冠。而淘汰賽首輪抽到這個不錯的對手，對於張康陽來說也算是好消息。
