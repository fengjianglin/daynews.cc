一敗再敗！選拔隊0比1不敵韓國！最後一戰關乎顏面

2019-12-16 1,081條評論 2,027次閱讀 1106人點贊

全方位的受虐之後，只有比分不讓人絕望。

今日頭條

韓國 1比0 中國

比分不是過程的合理反映


東亞杯第二輪一場比賽在韓國同中國之間進行。第10分鐘，國足反擊，張稀哲送出斜傳，譚龍右路跟進後傳中，董學升搶前點墊射稍稍偏出立柱。第13分鐘姜至鵬後場失誤，黃仁范遠射被劉殿座一擋擊中橫樑彈出底線。隨後韓國利用角球取得領先，金玟哉前點甩頭攻門得手。雖然本場比賽中國隊只有這一個丟球，但是全場比賽中國隊全面被動。下半場大舉前壓後，不斷被韓國隊打出威脅反擊，只是由於對手浪費了太多機會，0比1的比分才得以保持到結束。

賽事速遞

西甲

瓦倫西亞 1比1 皇馬

皇馬錯失登頂機會


西甲聯賽第17輪，皇馬客場挑戰瓦倫西亞。第78分鐘，瓦斯右側肋部下底倒三角回傳，中路跟進無人盯防的索萊爾輕鬆推射破門。補時最後時刻，皇馬扳平場上比分。皇馬左路角球吊入禁區，庫爾圖瓦禁區內高高躍起頭槌攻門被多梅內克封出，隨後本澤馬在門前亂戰中近距離勁射破門。最終，皇馬錯失登頂機會，依舊與巴薩同分排名第二。

英超

狼隊 1比2 熱刺

維爾通亨絕殺，熱刺瘋狂追分


英超第17輪，熱刺客場挑戰狼隊。第8分鐘，小盧卡斯禁區外得球後憑藉個人能力突入禁區，他小角度晃開空檔一腳爆射將球打進球門上角。第66分鐘，狼隊在中後場完成搶斷，希門尼斯帶球推進後橫傳右路，將特勞雷稍作調整突然起腳一腳世界波洞穿球門。第90分鐘，埃里克森角球開出，維爾通亨點球點附近甩頭攻門將球打進。最終，熱刺在客場取勝。

新聞速遞

一場失利，里昂損失兩名大將

里昂官方發布通告，球隊隊長德佩左前十字韌帶撕裂，球隊21歲中場阿德萊德右膝蓋前韌帶斷裂。兩人的具體傷缺時間還待確定，但預計至少缺席6個月時間。德佩和阿德萊德都是在球隊0-1負於雷恩的比賽中受傷的。

英媒：哈蘭德表態願加盟曼聯

《鏡報》消息，哈蘭德已經向曼聯主帥索爾斯克亞表示願意加盟曼聯。本賽季19歲的哈蘭德已經在各項賽事中為薩爾茨堡紅牛打進28球，其中包括8個歐冠進球，身價已經高達8000萬英鎊。目前，奧地利聯賽已經進入為期兩個月的冬歇期，媒體認為哈蘭德會在冬季轉會期就離隊。

精品預告

周一足球賽事再度回歸平靜，實戰版籃球專家趙原，將為彩民帶來籃球賽事的精選分析和推薦。

盤路回顧

雖然輸掉了比賽，但是中國隊在面對日韓時取得了贏盤的戰績，也實屬不易。面對中國隊，韓國隊主讓1.25，雖然韓國隊早早就先開紀錄，不過由於鋒線隊員對於機會的態度過於隨意，導致遲遲無法再度進球。或許，大勝中國隊也不足以讓保羅·本托感到滿意，畢竟最後一場對陣日本，才是韓國隊爭冠的關鍵之戰。

五大聯賽冷門頻頻，法甲里昂主場0比1不敵雷恩，客勝賠率高達5.00。西甲方面，本賽季表現出色 塞維利亞，也是在主場不敵比利亞雷亞爾。主隊全場傳中次數高達68次，依舊無法逃過5.15倍的客勝賽果。
