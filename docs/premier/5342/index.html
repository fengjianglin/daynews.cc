歐冠奇景！兩榜首隊危機四伏，恐最後時刻出局，第二名卻形勢大好

2019-11-28 0條評論 15次閱讀 0人點贊

11月28日，歐冠小組賽結束了第5輪全部比賽的爭奪，又誕生了2支新的出線球隊，分別是巴薩和萊比錫RB，目前一共有8支球隊拿到了16強的席位。不過，這8個席位的分配並不均衡，有兩個小組的出線名額全部誕生，還有兩個小組的競爭十分激烈，至今沒有一支球隊能夠突圍，包括衛冕冠軍利物浦所在的E組。

貴為上賽季的歐冠冠軍，利物浦本賽季小組賽卻遭遇了開門黑，首輪便在客場0-2不敵那不勒斯。不過，在隨後的3場比賽中，利物浦豪取3連勝，排到了積分榜首。然而，本輪小組賽面對那不勒斯，利物浦再度栽了跟頭，主場1-1驚險戰平對手，不僅沒能提前出線，而且導致自己處在極為被動的位置。

收穫這場平局之後，利物浦積10分繼續排在榜首。最後一輪小組賽，利物浦將前往客場挑戰薩爾茨堡紅牛，對手積7分。雙方的首回合交鋒，利物浦4-3險勝。因此，最後一輪小組賽，如果薩爾茨堡能夠在主場贏1球，而且丟球數少於3個，就能成功晉級。反倒是位於小組第二的那不勒斯形勢非常好，積9分的他們最後一輪主場迎戰墊底球隊亨克，戰平即可出線。

與利物浦面對同樣困境的還有H組的榜首球隊阿賈克斯。本輪小組賽開始前，阿賈克斯、瓦倫西亞、切爾西3支球隊同積7分，平分秋色。不過，本輪聯賽瓦倫西亞2-2戰平切爾西，兩隊都只拿到了1分，阿賈克斯則是客場2-0成功擊敗里爾，暫時登上了小組積分榜首，對身後兩支球隊的領先優勢達到了2分。

然而，這樣的領先優勢並不保險。最後一輪小組賽，阿賈克斯將在主場迎戰瓦倫西亞，這將是一場硬碰硬。阿賈克斯如果不敗，可以順利出線，但是如果主場落敗，結果就不好說了。而排在第二名的切爾西則是形勢非常明朗，他們最後一輪坐鎮主場硬仗僅積1分的里爾，拿下這場比賽晉級應該不會有太大障礙。

本賽季的歐冠聯賽，的確是景象非常奇特，一些並不是死亡之組的小組，被硬生生踢成了死亡之組，比如利物浦所在的E組。當然，造成當前的局面，與利物浦自己有非常大的關係，對陣那不勒斯哪怕是贏下一場，都不會是現在這個局面。
