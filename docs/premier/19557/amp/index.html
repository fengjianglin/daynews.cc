歐冠8強前景分析！4大豪門對決，兩隊提前預定八強，頭號黑馬無緣

天天要聞
2019-12-18

隨著歐冠小組賽的比賽結束，最終出線的十六支球隊也全部出爐了，而最終的淘汰賽對決也已經產生了。可以看到的是，根據最終的抽籤結果，多特將對陣巴黎聖日爾曼，皇馬對戰曼城，亞特蘭大對陣瓦倫西亞，馬競與利物浦相遇，切爾西同拜仁展開強強對話，里昂將面對尤文，熱刺對陣RB萊比錫，那不勒斯與巴薩相遇。隨著十六強對決的產生，最終的晉級名額也是備受球迷期待的，接下來我們就從中為大家解析一下各支球隊的簽運，以及晉級的形勢。在這其中，有4組豪門球隊之間的對決引起了球迷的關注，而尤文巴薩則大概率的預定8強席位。不過作為首次進軍歐冠淘汰賽的黑馬球隊RB萊比錫來說，就不太走運了，畢竟他們遇到了老辣的穆里尼奧，進而有近乎出局的危機。

4組豪門對決其一便是多特對陣大巴黎，這一組對局很有意思，畢竟兩支球隊都是擅長進攻的，兩隊的比拼會是大開大合，若是從進攻角度來對比的話，大巴黎的「四億妄為」組合絕對是更勝一籌的，而他們的中後場實力同樣超過多特，因此大巴黎進軍八強的概率很高。其次，陣容實力更強的拜仁遇上防線羸弱的切爾西，也有很大的勝算，因此拜仁有望在12年歐冠決賽輸球後，復仇藍軍。而另一組利物浦同馬競的豪門對決也是非常有看點的，不過兩隊目前的境遇截然不同，畢竟利物浦狀態高漲，而馬競則是進攻乏力，防線水平同樣大不如以前，因此衛冕冠軍晉級的希望很大。

皇馬同曼城的豪門對決也是很有意思，不過考慮到皇馬在歐冠中的巨大經驗，以及齊祖的玄學加成，因此銀河戰艦很有可能贏下這組強強對決。另外，豪門球隊中，尤文巴薩的簽位絕對是最好的，畢竟尤文碰上了傷兵滿營的里昂，巴薩遇到了陷入內憂外患危機的那不勒斯，因此這兩家球隊大概率晉級下一輪比賽。瓦倫西亞同亞特蘭大隊的比賽也是看點十足，這兩家球隊實力上或許沒有太大的差距，不過瓦倫西亞隊有著更多的歐冠經驗，因此他們有更大的晉級希望。最後，首次晉級歐冠淘汰賽的RB萊比錫，他們其實已經證明了自己，不過在歐冠奪冠經驗豐富的穆里尼奧面前，這一對手恐怕很難給上賽季的歐冠亞軍熱刺製造威脅，因此穆里尼奧的球隊大有晉級八強的機會。綜上，本賽季的歐冠八強很有可能是大巴黎，拜仁，利物浦，皇馬，尤文，巴薩，瓦倫西亞以及熱刺。球迷們怎麼認為呢？歡迎留言討論！
