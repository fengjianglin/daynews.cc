國足負日本三隊！肖國際主場"巴"氣側漏

2019-12-10 2,177條評論 4,961次閱讀 3840人點贊

昨日回顧

謝邀，人在美國，剛下飛機。昨晚英超，處於風口浪尖的阿森納終於客場3：1逆轉西漢姆，終結9場不勝的尷尬。破廠球員已經到了贏一場就可以瘋狂慶祝的地步，可悲可嘆！公推2連紅！

關注

中國1比2不敵日本

周二傍晚，中國在東亞杯首戰中對陣日本。上半場，中國沒有太多機會，對方則利用反擊機會射入一球；下半場，三浦弦太為日本再下一城，董學升最後時刻扳回一球，最終中國1比2不敵日本。

比賽開始7分鐘，中國禁區內尋找機會，右路窄角度位置董學升完成一腳射門，不過被撲出，但他也已經越位了。13分鐘，中國角球機會，譚龍跟進頭鎚，可惜高出橫樑。16分鐘，日本門前製造威脅，町中槙之輔跟進射門擊中了門柱。29分鐘，日本取得了領先，反擊機會中日本完成精彩配合，鈴木武藏抽射破門，中國0比1日本。30分鐘，姜至鵬做出危險犯規動作，吃到黃牌。35分鐘，日本反擊機會，禁區內中路鈴木武藏得球，一腳爆射，不過球被擋出來。40分鐘，中國角球機會，不過球在前點被對方解圍。補時階段，董學升接到傳中嘗試頭鎚，但還是頂偏了。半場結束，中國0比1日本。
換邊再戰， 52分鐘，中國禁區內的機會，明天嘗試一腳抽射，不過射中了橫樑。70分鐘，日本再次取得進球，這是角球的機會，三浦弦太頭鎚破門，中國0比2日本。75分鐘，日本繼續前場尋找機會，外圍遠射被于大寶破壞。81分鐘，中國禁區內的機會，譚龍的射門稍稍射偏。90分鐘，中國取得進球，董學升接到長傳頭鎚，球打到立柱彈入破門，為中國扳回一城。最終中國1比2不敵日本。

今日推薦

周中首個比賽日的歐冠杯小組賽最後1輪精彩紛呈，當中多場強強對話，不過最為重磅的當然是F組的1場較量，意甲豪門國際米蘭將坐鎮主場迎接西甲豪門巴塞羅那的挑戰，比賽具體時間為周三凌晨。由於雙方近期整體發揮均讓人滿意，各自勝率頗高，此役絕對是1場含金量極高的對陣。

憑鋒線組合馬丁內斯與盧卡庫的聯手建功，國米上輪小組賽作客輕取捷克球隊布拉格斯拉維亞，同輪由於多特不敵巴薩，因此藍黑軍團在F組的積分榜上實現反超大黃蜂，國米目前以微弱優勢排名次席，佔據出線主動權。只要藍黑軍團末輪全取3分，他們就一定會進軍到十六強，全隊上下戰意甚濃。其實前5輪小組賽戰罷，國米總共收穫9球，是該組攻擊力最強者，孔蒂主打的雙前鋒組合確實效果極佳，他們唯一需要擔心的便是防守問題，畢竟7個失球對比巴薩是略遜一籌。

這是國米隊史上第13次征戰歐冠杯小組賽，在歐洲豪門中並不算多，並且最近9場歐冠杯比賽，他們的戰績僅為2勝3平4負，爭勝能力一般。但此役藍黑軍團別無選擇，一旦同時間開打的多特最終戰勝布拉格斯拉維亞，國米就必須要全取3分才能守住另外一張晉級十六強的入場券。作為最後一支奪得歐冠杯的義大利球隊，他們肯定不想主場飲恨。過去9次在歐戰賽事中主場作戰，國米豪取5勝3平1負的佳績，而且此前9次在歐戰賽事中主場迎戰西班牙球隊，藍黑軍團保持不敗，戰績為3勝6平，這無疑成為他們此役擊退巴薩的最大資本。

巴薩提前1輪鎖定小組第一，末戰對於他們來講意義不大。一來至少球隊絕對不會在該場比賽中死磕國米，萬一讓球員受傷便得不償失；二來巴薩還不如成全藍黑軍團想要進軍十六強的願望，畢竟對自己的位置影響不大，他們此役的鬥志肯定是不及國米。在下一階段淘汰賽未來到之前，球隊目前可以高枕無憂。既然如此，必定要將重心回歸國內賽事，聯賽方面他們已經被老對手皇馬追平積分，未來一段時間內重新拉開分差、擴大領先優勢才是球隊的當前任務。在首循環面對國米時，正是前鋒蘇亞雷斯上演梅開二度，幫助巴薩逆轉對手，烏拉圭射手此役將成為隊中的主力攻擊手。讓人擔憂的是，球隊最近15次在歐戰賽事中出征的戰績僅為5勝6平4負，並不算出色，而且他們歷史上走訪義大利也僅6勝11平7負，尤其是2011-12賽季之後，巴薩最後8次作客挑戰意甲球隊一勝難求。陣容方面，前鋒登貝萊與後衛塞梅多等人因傷倦勤；前鋒梅西和後衛皮克均輪休。

數據顯示，巴薩近17場歐冠16場不敗，雙方近4次交手，3次總進球2球或以下。巴薩近10次對國米9次不敗。本場主讓半一升一球高水，更多強調戰意，82是一個相當保守的教練，在小組第一已經穩如狗的情況下，本場必定慫的一筆，走主勝。
