舊將：我不後悔離開切爾西 阿扎爾並沒有比我強

天天要聞
2019-12-14

直播吧12月14日訊 日前，薩索洛邊鋒博加接受了米蘭體育雜誌《體育周刊》的採訪，他表示對於離開切爾西不後悔，並堅稱隊友阿扎爾、佩德羅和威廉沒有比我強。

現在22歲的博加在2018年的身價僅為300萬歐元，不過如果他繼續保持出色表現，切爾西還是保留了對他回購的選擇權。

本賽季迄今為止，博加在13場意甲比賽中貢獻了4個進球和2 次助攻，其中包括對陣尤文圖斯時一個漂亮的吊射。

博加說：「我12歲來到切爾西，因為他們的球探在馬賽俱樂部里發現了我。剛來到切爾西的頭兩年，從馬賽的海邊到倫敦的大霧，我很難適應這邊的天氣，食物也是一個嚴重的問題，因為我早餐從吃穀類食品到一盤雞蛋和培根。這些都是不容易消化的。」

「我在切爾西青年隊的隊友是亞伯拉罕，他現在是一線隊的中鋒，還有奇克，穆桑達……我們一起贏得了歐洲青年聯盟杯和U21超級聯賽。」

「我時不時在思考，我是否應該在切爾西等待更長的時間，我必須要經常比賽，但是我在隊中的出場順序排在阿扎爾、佩德羅和威廉之後。我覺得不是他們比我強，而是他們在球隊的等級體系中比我領先。」

「我不後悔離開切爾西，當時的主教練孔蒂沒有跟我說太多，當我在對陣伯恩利的處子秀時，他只是對我說：『輪到你了』。」

（時刻）
