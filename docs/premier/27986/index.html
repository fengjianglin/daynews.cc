19178期任九推薦：萊比錫想拿半程冠軍 國米可能給盟友送分

2019-12-21 74條評論 1,710次閱讀 154人點贊

昨天的任九不出意外的黑了，法乙單選涼涼，德乙防不勝防，多特居然輸球，哎，今天深盤不少，爭取紅起來。

紐卡斯爾vs水晶宮

紐卡近期狀態不錯，雖然上一輪客場輸球，但是主場近幾輪不敗，更是平了曼城贏了曼聯。水晶宮也近4輪不敗，兩勝兩平。兩隊近況還算可以的情況下亞盤初盤給出主讓平半盤，這個盤整體來說還算正常，但是也有個別盤口給出平手盤，居然沒有考慮紐卡的主場優勢，這個盤很讓人費解，後期升到主讓平半，這樣的話給我的直觀感覺紐卡這場有問題。目前平半盤下盤低水，上盤水位有下調趨勢，感覺紐卡不勝。而紐卡主場平局居多，這樣的話客勝也許是個盲點，我想博一把客勝，賭水晶宮送紐卡主場第二敗。

諾維奇vs狼隊

諾維奇重新來到英超似乎還是不能適應，這個賽季明顯的為保級而戰。近況也很差。此番面對整體表現還不錯的狼隊，看樣子也是輸面很大啊。亞盤初盤客讓半球，這個盤是狼隊這個賽季客場讓的最深盤，主隊作為升班馬及倒數第二的球隊，這個盤還算正常。但是後期下盤水位下降，頗有降盤的趨勢，下盤沒有什麼拉力的情況下盤面這樣走給我的感覺狼隊還是無法撐起這個半球盤，這場比賽防範狼隊不勝，走10.

曼城vs萊切斯特城

這場比賽曼城如果在丟分，那麼這個賽季想要拿到冠軍就很難了。兩隊近況不用多說。亞盤初盤給出主讓一球球半，這個盤是除了萊切城奪冠那個賽季以外最低的盤了。也可以看出萊切城這個賽季的表現也讓曼城有所忌憚。後期盤口升到球半盤，有些低開高走的味道。兩隊周中都有比賽，但是曼城相比較要輕鬆一些，所以萊切城相比較不佔優勢。因此這個升盤也算合理一些。這場比賽下盤的關注度應該不低，低開高走也許會起到阻盤的作用，不如膽子大一些，博3.

美因茨vs勒沃庫森

勒沃庫森已經聯賽兩連敗了，美因茨則還是保持著賽季無平局的狀態。這次兩隊交鋒亞盤初盤給出客讓半一，這個盤對比兩隊交戰歷史盤面有些偏高了。並且近期狀態也不足客隊撐起這個盤。後期盤口上升到半球盤，上盤低水，還有恢復到半一盤的可能。盤口高開低走，給人的錯覺上盤可出。但是考慮到兩隊是一個派系，這場比賽位覺得還是防範一下為好。同派系有可能美因茨能拿到賽季首場平局，走10，膽大的博0.

拜仁vs沃爾夫斯堡

拜仁這個賽季老是給小弟送分，不知道這次會不會在送分了，亞盤降盤，錢多的主小心防範，我這貧民玩家只能博3了，賭拜仁贏球輸盤（兩隊交鋒拜仁主場似乎有規律的贏盤輸盤，按規律這場要贏盤）。

沙爾克04vs弗萊堡

沙爾克這場比賽我挺糾結的。這個賽季沙爾克表現不錯，但是佛萊堡這個賽季表現也很好，就是近期狀態一般。這場比賽亞盤初盤給出主讓半一，如果對比兩隊近況來說還算正常，但是考慮到歷史交戰沙爾克優勢也不是很大。但是防範更頭大，近幾個賽季弗萊堡也沒在沙爾克主場佔到過太大便宜，但是這場比賽我又感覺有0，所以這場比賽究竟防1還是防0根據自己喜好吧。我準備跟著感覺走，防0

萊比錫紅牛vs奧格斯堡

萊比錫能否拿到半程冠軍，這場比賽很關鍵，但是客隊奧格斯堡近況很不錯，近6場比賽5勝一平，進15球丟4球。而且上賽季3次面對奧格斯堡都打平了。這場比賽亞盤初盤給出了兩隊交鋒歷史最高盤球半兩球，這個盤如果僅僅依靠萊比錫現在力拿半程冠軍來撐有些勉強。所以這場比賽建議防範雙方激戰成和。各取一分，因為門興拿3分也難。

柏林赫塔vs門興

作為拜仁的小弟，為大哥排憂解難是肯定的。赫塔兩連勝，狀態有所好轉，但是仍舊是一個弱隊的體現。但是面對門興，赫塔每場比賽都要拼個你死我活。雙方近幾個賽季基本上都分勝負。這場比賽亞盤初盤給出客讓平半，看來也是對於赫塔多為忌憚，並沒有強行造勢門興。後期上盤水位下降，有升盤的趨勢，這樣的話勢必將熱度引導門興。所以這場比賽建議分勝負。

國米vs熱那亞

熱那亞作為國米的盟友，國米每個賽季都要適度照顧一下。上個賽季為了力保國米拿到歐冠資格，熱那亞兩場全輸送分明顯，這個賽季，熱那亞形勢很不樂觀，按照國米投桃報李的理念，這場比賽很可能會送分。亞盤初盤給出球半，盤面還算正常，後期降盤到一球球半，降得沒有道理啊。所以這場比賽個人建議走30，1分送的沒有價值，但是也不排除送1分，畢竟國米也是關鍵時期。所以怎麼防你們自己定，我準備走30.
