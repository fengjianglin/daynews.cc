意甲-盧卡庫造3球埃斯波西托首球 國米4-0熱那亞終結3場不勝

天天要聞
2019-12-22

直播吧12月22日訊 北京時間12月22日凌晨1:00，2019-2020賽季意甲聯賽第17輪繼續進行，國際米蘭主場迎戰熱那亞。上半時比賽坎德雷瓦助攻盧卡庫首開紀錄，隨後加利亞爾迪尼破門迅速擴大比分。下半時比賽埃斯波西托點射建功打進個人意甲首球，隨後坎德雷瓦助攻盧卡庫再下一城錦上添花。最終，國米主場4-0擊敗熱那亞，終結各項賽事三場不勝的頹勢。

國際米蘭近期表現略顯低迷，球隊先是在歐冠賽場主場不敵巴薩無緣晉級十六強，隨後又在聯賽賽場客場戰平佛羅倫薩遭遇兩連平，積分榜上目前少賽一場落後榜首的尤文3分位居次席。但熱那亞的日子同樣不好過，上輪主場不敵桑普多利亞後，熱那亞在遭遇聯賽七輪不勝的同時深扎保級泥潭。此役首發方面，盧卡庫搭檔小將埃斯波西托先發出戰，貝西諾與比拉吉聯袂登場。

比賽開始第3分鐘，什克里尼亞爾中場挑傳，皮球被防守球員搶先伸腳攔截。第5分鐘，羅梅羅拼搶中從背後撞倒埃斯波西托，主裁判鳴哨示意犯規。第7分鐘，加利亞爾迪尼前場搶斷後順勢送出一記直塞，但傳球力量稍大，皮球直接出了底線。第8分鐘，埃斯波西托左側肋部卸下皮球後左腳順勢抽射，皮球被拉杜倒地穩穩沒收。↓

第9分鐘，比拉吉左路持球擺脫防守後嘗試斜45度傳中，盧卡庫與貝西諾兩點包抄均沒有碰到，皮球稍稍偏出右側立柱出了底線。第13分鐘，克里希托邊路防守中放倒貝西諾，主裁判鳴哨示意犯規，國米獲得前場定位球機會。1分鐘後，比拉吉主罰右路定位球吊入禁區，加利亞爾迪尼高高躍起甩頭攻門，皮球被拉杜側撲沒收。↓

第16分鐘，吉利奧尼外圍持球稍作觀察後將球吊入禁區，漢達諾維奇出擊穩穩將球拿到。第20分鐘， 國米左路角球吊入禁區，皮球被防守球員頭槌破壞。第22分鐘，國米右路傳中，比拉吉左路跟進順勢將球墊向中路，加利亞爾迪尼包抄甩頭攻門，拉杜奮力將球擋出橫樑。↓

第23分鐘，國米左路角球吊入禁區被破壞，坎德雷瓦外圍迎球凌空抽射，皮球被拉杜穩穩收入懷中。↓

第26分鐘，埃斯波西托左路角球吊入禁區，盧卡庫包抄沒有頂到，皮球直接出了右側底線。第28分鐘，坎德雷瓦右路持球內切後起腳遠射，皮球高出了球門橫樑。第31分鐘，國米率先取得進球！盧卡庫前場斷球後分球，坎德雷瓦跟進將球吊入禁區，盧卡庫包抄到位甩頭攻門將球頂進球網，國米1-0熱那亞。↓


第33分鐘，國米迅速擴大場上比分！國米左路大力邊線球擲進禁區，盧卡庫禁區內倚住防守球員將球回做，加利亞爾迪尼跟進大力抽射擊中防守球員身體發生折射後竄進球網，國米2-0領先熱那亞。↓


第35分鐘，巴斯托尼拼搶中抬腳過高放倒吉利奧尼，主裁判果斷向巴斯托尼出示黃牌。第37分鐘，坎德雷瓦右側肋部突入禁區晃開防守後起腳勁射，皮球被門前的隊友盧卡庫擋出了底線。第39分鐘，盧卡庫右路持球內切晃開防守球員後起腳低射，皮球偏出了右側立柱。↓

第43分鐘，羅梅羅拼搶中推倒埃斯波西托，主裁判鳴哨向羅梅羅出示黃牌。第45分鐘，國米獲得禁區前定位球機會，比拉吉主罰直接攻門，皮球擊中人牆後高出橫樑。

半場結束，國米暫時2-0領先熱那亞。

易邊再戰第46分鐘，羅梅羅後場回傳失誤，盧卡庫禁區內將球攔截稍作調整後起腳射門，皮球被拉杜奮力上前用身體封堵。↓

第50分鐘，熱那亞反擊中左路傳中，禁區內巴斯托尼穩穩將球攔截。第53分鐘，克里希托主罰定位球直接攻門，皮球被人牆封堵。第56分鐘，熱那亞後場長傳球吊入禁區，漢達諾維奇出擊輕鬆將球拿到。第60分鐘，國米禁區前搶斷後分球，禁區內埃斯波西托包抄射門沒有形成威脅，拉杜輕鬆將球沒收。第61分鐘，薩納布里亞外圍連續調整後起腳遠射，皮球彈地後被漢達諾維奇撲出底線。↓

第63分鐘，阿古德洛禁區內放倒加利亞爾迪尼，主裁判果斷鳴哨判罰點球並對阿古德洛出示黃牌。第64分鐘，國米打進本場比賽第三粒進球！小將埃斯波西托主罰點球踢向左側，雖然拉杜判斷對方向但無法阻止皮球入網，國米3-0領先熱那亞。↓


第66分鐘，薩納布里亞左路得球稍作觀察後起腳遠射，皮球被漢達諾維奇奮力擋出橫樑。第68分鐘，薩納布里亞外圍繼續嘗試遠射，漢達諾維奇將球收入懷中。第71分鐘，國米打進本場比賽第四顆進球！坎德雷瓦外圍分球，盧卡庫禁區內得球踩單車晃開防守後爆射得手，國米4-0熱那亞。↓


第74分鐘，熱那亞獲得前場定位球機會，薩納布里亞主罰直接攻門，皮球高出了球門橫樑。第78分鐘，國米做出本場最後一次換人調整，迪馬爾科替補出場換下比拉吉。第80分鐘，拉扎羅後場回傳失誤，阿古德洛順勢前插搶斷成功晃過出擊的漢達諾維奇後小角度射門，皮球擊中邊網出了左側底線。↓

第84分鐘，盧卡庫前場分球，克里希托搶在前插的阿古德洛之前將球剷出了底線。第86分鐘，埃斯波西托禁區前橫敲，盧卡庫故意讓球，森西跟進迎球抽射，皮球卻擊中了來不及躲閃的埃斯波西托出了底線。

全場比賽沒有補時，主裁判直接吹響了結束的哨音。最終，國米主場4-0擊敗熱那亞，終結了各項賽事三場不勝的尷尬。

國際米蘭首發：1-漢達諾維奇、37-什克里尼亞爾、6-德弗里、95-巴斯托尼、87-坎德雷瓦（75’19-拉扎羅）、8-貝西諾、20-巴萊羅、34-比拉吉（78’21-迪馬爾科）、5-加利亞爾迪尼（72’12-森西）、30-埃斯波西托、9-盧卡庫
