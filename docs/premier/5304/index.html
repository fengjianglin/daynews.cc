歐冠第5輪，2晚精彩對決，是否讓你很過癮？談一談你最關注的比賽

2019-11-29 0條評論 18次閱讀 0人點贊

歐冠小組賽第5輪，在11月27日、11月28日展開。2晚16場精彩比賽，是否讓你直呼過癮，競技體育的緊張、刺激、精彩激烈程度帶來的滿足感，使我們常常快樂興奮，你也有這樣的感覺嗎？

2018-2019賽季歐冠總決賽由2支英超球隊之間展開。分別是英超利物浦隊和英超托特納姆熱刺隊，利物浦2:0戰勝熱刺，捧得2018-2019賽季歐冠冠軍獎盃，隊史第6次奪得歐冠冠軍。

2019-2020賽季歐冠賽事現已展開，目前已經進行到歐冠小組賽第5輪。一共分為8個小組，由A組-H組，4支球隊為一組，共32支球隊。

熬夜看球：

連續2天的精彩比賽，好多比賽都在凌晨4點開賽。如果想看球，就得從溫暖的被窩裡爬起來，看完之後差不多洗漱下，吃完早飯，就該上班了。估計很多人感同身受吧！

談一談你最關注的比賽：

1、歐冠第5輪備受關注，穆里尼奧回歸，帶領熱刺主場迎戰奧林匹亞科斯。這也是穆帥執教熱刺以來的第2場比賽，歐冠首戰。能否取得首勝，對穆帥來說是相當期待的。不過有點戲劇化的比賽發生了，主場被對手先入2球，之後又連入4球，逆轉取勝。穆帥高高地跳起，慶祝勝利。

2、大巴黎對皇馬的比賽，估計很多人都很關注。大巴黎歐冠賽事目前為止未嘗敗績，皇馬近期也發揮出色，阿扎爾精彩表現，18歲羅德里戈讓人期待。大巴黎的內馬爾、姆巴佩、伊卡爾迪，不用說這是一場從星雲集的比賽，最終2:2兩隊握手言和。

3、利物浦對那不勒斯的比賽，利物浦作為上賽季歐冠冠軍。主教練克洛普、前場三叉戟馬內、薩拉赫、菲爾米諾，後防範迪克等眾多好手。在歐冠還是聯賽當中，都有很好的發揮，最終1：1與那不勒斯握手言和。

4、巴薩對多特，巴薩上一輪0：0，沒有進球。這一輪能否打破進球荒，備受關注。最終蘇亞雷斯、梅西、格列茲曼分別進球，3:1擊退多特。

還有很多精彩的比賽，看的很過癮。談一談你最關注哪場比賽？歡迎在下方評論區評論交流。
