歐冠抽籤揭曉，除了C羅，哪還有上上籤？

2019-12-17 138條評論 1,346次閱讀 359人點贊

北京時間12月16日，歐冠16強分組對陣結果已經正式出爐，面對這樣的結果，肯定是有球隊歡喜，有球隊擔憂了，而著名足球評述員詹俊也發表了個人見解，詹俊認為本次的抽籤結果中，英超球隊只有熱刺抽到了好籤，其他英超球隊都遇到了勁旅，從結果一看，熱刺對陣RB萊比錫，真的是上上籤嗎？

而利物浦則將會面對馬競，以目前馬競的形勢來看，利物浦想要出線，雖然會有難度，但實際上難度並不大，馬競是出了名防守強悍的，但如今利物浦前場攻擊線不容小覷，想要撕破馬競的防線，說實話並不困難。

相反馬競雖然防守了得，但攻擊力卻相當一般，本賽季西甲17輪聯賽戰罷，馬競才取得了18個進球，比巴薩的43球和皇馬的33球差太遠了。

所以，利物浦只要做好後防上的工作，並且在進攻的時候有足夠耐心，是有機會去的進球的，現階段利物浦的後防線，絲毫不輸馬競，而且前場攻擊力也比馬競更強。

本次歐冠16強由五大聯賽球隊包攬，所以任何一支球隊都不存在所謂的「大禮包」，熱刺抽到的RB萊比錫，目前排在德甲第一名，15輪聯賽轟入42球，比拜仁還多，談什麼上上籤？曼城遇上皇馬更別提了，火山撞地球，想要突圍可謂死裡逃生，誰都沒有好結果啊！

當然，熱刺有穆里尼奧，實力漸入佳境，面對這一支德甲勁旅，確實更加值得高看， 所以詹俊認為是上上籤，也無可厚非。

如果非得說哪支球隊運氣好，那麼我選擇尤文，里昂本賽季發揮並非特別理想，尤文想要拿下對方，實屬不難。
