莫塔避談換帥傳聞：我不關注媒體，只考慮我的隊員們

2019-12-22 433條評論 2,770次閱讀 543人點贊

直播吧12月22日訊 北京時間12月22日01:00，意甲第17輪，熱那亞客場挑戰國際米蘭，最終0-4告敗。此前已有媒體稱烏拉圭教頭洛佩斯將取代莫塔上任熱那亞主帥，賽後被問到如何看待自己的未來時，莫塔對此避而不談。

自10月22日出任熱那亞主帥以來，除了開局2-1戰勝布雷西亞那場比賽外，莫塔僅取得一場勝利，即在義大利杯3-2擊敗意乙的阿斯科利。目前，17輪聯賽戰罷，熱那亞僅積11分，排在聯賽最後的位置。

在接受義大利《天空體育》採訪時，他談道：「我們嘗試推進至對方半場，但在他們打進第一個球之後，我們便難以再控制住場面。對手很強大，他們毫無疑問擁有爭冠的實力。」

「我為隊員們所付出的努力感到驕傲，即便球隊處於困境，比賽結果不盡如人意，但他們仍在努力百分百地為球隊付出。我只考慮我的隊員們，比賽結束後，我在更衣室看到他們，每個人都很難過，這是很正常的，我將會和俱樂部談談。」

「我承認我們有失誤，這是因為我總要求隊員們從後場開始組織進攻，把球控在腳下，並且保證進攻時的人數優勢。我們在傳球時犯了些技術性失誤，但只要我們團結一致，就能夠彌補回來。正如我所說的，今天我們所面對的是一支實力強勁的對手，從比賽一開始他們就不斷地給我們製造麻煩，我們很難踢出自己的東西。」

此前有烏拉圭媒體報道稱，烏拉圭名宿迭戈-洛佩斯已經接近出任熱那亞主帥，12月27日將舉行新聞發布。

莫塔對此稱：「我不看烏拉圭的報紙，也不怎麼關注義大利媒體。就像我之前說的那樣，我為球員們的努力感到驕傲，即使這聽起來有些奇怪，我仍這樣認為。有些事情需要改變，我相信我們能夠做到。我每天想到的只有球員，對其他並不怎麼感興趣。」

（淺色調）
