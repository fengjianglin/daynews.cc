流年不利！尤文丟冠後C羅薩里再曝不和 想要追國米難上加難

2019-12-25 1,130條評論 4,606次閱讀 1893人點贊

C羅和尤文主帥薩里的隔閡越來越深。根據《共和報》的消息，丟掉義大利超級盃的冠軍讓C羅非常生氣，除了憤怒，他還對薩里的戰術安排以及換人決定提出了質疑。尤文近況不佳，如今師徒之間又產生了矛盾，如此狀態再想要追趕國米，實在太難了！

這賽季，C羅和薩里已經多次被爆出不和傳聞。這一次，義大利超級盃的失利更是讓二人本就緊張的關係變得更加惡化。《共和報》指出，除了有許多媒體外，尤文內部有一些隊員也對薩里表達了懷疑。

其中，對薩里提出質疑的最大牌的球星就是C羅。據悉，C羅對薩里的排兵布陣並不滿意。此外，他認為薩里的換人也有問題，比如薩里不該在第66分鐘的時候將伊瓜因換下。《共和報》稱C羅在意超杯比賽後的更衣室里非常憤怒，有些隊友不能理解C羅的感受，這讓葡萄牙人感到極度的失望。

已經有不少專家稱薩里配不上尤文，建議尤文趕緊將薩里解僱，不然斑馬軍團這賽季很可能四大皆空。這次輸給拉齊奧後，薩里甚至辯解稱並不是尤文踢得不好，而是對手實力太強。本賽季，尤文已經輸給拉齊奧2次，且這2場敗仗都發生在最近的半個月內。再加上聯賽里因為凈勝球少而被國米壓制，尤文如此窩囊，薩里難辭其咎！
