尤文巴薩2輪進8球，阿根廷4鋒霸全包攬，但仍比不過梅西

天天要聞
3 週 ago

北京時間11月23日（昨天）22時，在意甲第13輪的一場比賽中，尤文客場3-1戰勝亞特蘭大。本場比賽C羅輪休（薩里說是準備歐冠），伊瓜因在前鋒線上搭檔迪巴拉。兩位阿根廷鋒霸不負眾望，包攬了球隊的3粒進球，其中伊瓜因上演了梅開二度。而在上輪意甲的一場強強對話中，尤文主場1-0擊敗AC米蘭，而打入制勝一球的，正是第55分鐘替補C羅出場的迪巴拉。

在昨天凌晨進行的法甲第14輪首場比賽中，巴黎主場2-0戰勝里爾，阿根廷2鋒霸伊卡爾迪和迪馬利亞包攬2粒進球。巧合的是，在上一輪法甲巴黎客戰布雷斯特的比賽中，也是這2個阿根廷人包攬了巴黎的進球。第37分鐘迪馬利亞單刀挑射破門；第84分鐘，替補上場的伊卡爾迪打入准絕殺的1球。賽後，天使獲得全場最佳球員。

也就是說，最近2輪歐洲五大聯賽，無論是法甲領頭羊巴黎，還是意甲領頭羊尤文，陣中都有2位阿根廷前鋒包攬了各自球隊的全部進球（兩隊共進8粒），這樣的情況還是很罕見的。要知道，巴黎陣中有更有名氣的內馬爾、姆巴佩和卡瓦尼，而尤文的頭號射手是C羅。出現阿根廷4鋒霸包攬球隊2輪進球，有另4位名頭更大的鋒霸不在狀態或有小傷、讓阿根廷前鋒獲得更多機會的原因。

不過，即便是迪巴拉、伊瓜因、迪馬利亞、伊卡爾迪最近2輪如此出彩，如果跟梅西相比，仍有一定的差距。昨天晚上，西甲上演首尾大戰，巴薩客場對陣萊加內斯。可不要小看副班長，本場比賽，萊加內斯第12分鐘就1-0領先，幸虧梅西本場依舊搶眼，下半場助攻蘇牙扳平比分，第80分鐘比達爾利用對手烏龍助攻，打入制勝球。梅西賽後獲得全場最高的8.8分（全場最佳）。

而上一輪西甲，巴薩主場4-1大勝塞爾塔，梅西上演帽子戲法，賽後被評為全場最高的9.7分（全場最佳）。2輪聯賽進3球，另4名阿根廷鋒霸當然望塵莫及。

要知道這兩輪西甲之間，而夾著國際比賽周，梅西代表阿根廷踢滿了對陣巴西和烏拉圭的友誼賽，梅西在這兩場比賽中都取得了進球，幫助阿根廷1-0巴西，2-2烏拉圭。32歲的梅西連續4場打進5球，而且梅西在場上的妙傳和過人，更是令另4名阿根廷鋒霸自嘆不如。

但願在明年的南美世預賽和美洲杯上，迪巴拉、迪馬利亞能成為梅西的鋒線搭檔，加上阿圭羅和勞塔羅，阿根廷的鋒線組合一定令對手不寒而慄，阿根廷隊完全有能力再現昔日的輝煌！
