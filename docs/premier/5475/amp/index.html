國米舊將3分鐘2球，拉門戈2-1河床，弗拉門戈獲得南美解放者杯

天天要聞
3 週 ago

連續2個賽季的解放者杯決賽，都出於安全考慮而臨時更換場地，十分接近衛冕的河床隊，卻在最後時刻被巴爾博薩連進2球，痛失隊史上的第5座解放者杯。或許誰都沒有想到的是：上任僅5個月的熱蘇斯，就帶領弗拉門戈站上南美之巔！

時隔38年之後，弗拉門戈再度殺進解放者杯決賽，身為衛冕冠軍的河床隊，則是五年內第三次參加「南美巔峰之戰」。乍一看，弗拉門戈似乎處於劣勢，可如果翻看「紅黑軍團」的花名冊，你就會驚詫於這支球隊的陣容實力，像門將阿爾維斯、右後衛拉菲尼亞、左後衛菲利佩·路易斯和前鋒巴爾博薩，都曾經在歐洲的頂級豪門或准豪門效力，而主帥熱蘇斯更是叱吒葡超多年。

此役，弗拉門戈以4-2-3-1應敵，今夏從歐洲回歸的菲利佩·路易斯和拉菲尼亞分居左右兩閘，阿勞和熱爾松搭檔雙後腰，身體感到不適的里貝羅出任右前衛，巴爾博薩頂在單箭頭的位置上。

11月24日凌晨04:00，2019-2020賽季南美解放者杯決賽，由河床對陣弗拉門戈。上半場比賽，博雷禁區內掃射為河床先拔頭籌;下半場比賽尾聲，巴博薩的進球為弗拉門戈扳平比分,，補時階段弗拉門戈上演絕殺。最終常規時間比賽結束，弗拉門戈2-1神奇逆轉河床，38年後再次贏得解放者杯。

第14分鐘，河床大禁區右側送出橫傳，弗拉門戈禁區內沒有將球解圍，博雷機敏跟上掃射得分，河床1-0弗拉門戈。

第21分鐘，馬蒂亞斯-蘇亞雷斯左側送出橫傳，德拉克魯斯跟上射門被破壞。

第37分鐘，博雷大禁區弧頂右腳爆射稍稍偏出左側立柱。

第41分鐘，德拉克魯斯與拉菲尼亞空中激烈拼搶。

第44分鐘，河床前場送出精妙直塞，可惜越位在先。

第46分鐘，帕拉西奧斯飛鏟。

半場戰罷，河床1-0暫時領先弗拉門戈。

易邊再戰第47分鐘，弗拉門戈大禁區右側遠射太正被阿庫尼亞沒收。

第51分鐘，馬蒂亞斯-蘇亞雷斯左側大禁區線上遠射打偏。

第53分鐘，博雷大禁區弧頂兜射稍稍偏出。

第57分鐘，弗蘭門戈禁區內連續射門被封堵，里貝羅小禁區前沿跟上爆射阿庫尼亞送上精彩一撲。

第63分鐘，卡斯科大動作犯規，弗蘭門戈教練氣憤異常。

第68分鐘，帕拉西奧斯大禁區弧頂遠射稍稍打偏。

第71分鐘，恩佐-佩雷斯犯規染黃。

第79分鐘，河床發動快速反擊，普拉托遠射打偏。

第81分鐘，河床右側下底送出倒三角，普拉托跟上的射門偏出右側立柱。

第85分鐘，河床發動反擊，右側將球轉移至中路，普拉托帶球調整的遠射太正被門將沒收。

第89分鐘，巴博薩後插上破門為弗拉門戈扳平比分，河床1-1弗拉門戈。

第92分鐘，巴博薩大禁區線上跳起桑巴挑球晃開兩名防守球員順勢左腳爆射絕殺，河床1-2弗拉門戈。

第95分鐘，帕拉西奧斯惡意犯規被紅牌罰下。

第96分鐘，巴博薩抱怨裁判被罰下。

出場陣容：

河床：阿爾馬尼、蒙鐵而、馬丁內斯、皮諾拉、卡斯科、恩佐-佩雷斯、費爾南德斯、帕拉西奧斯、德拉克魯斯、博雷、馬蒂亞斯-蘇亞雷斯

弗拉門戈：迭戈-阿爾維斯、拉菲尼亞、羅德里戈-卡約、菲利佩-路易斯、巴勃羅-馬里、里貝羅、熱爾松、威廉-阿朗、德阿拉斯卡埃塔、布魯諾-恩里克、加布里埃爾-巴爾博薩
