AC米蘭恥辱性慘敗！球員高層和投資方都難辭其咎

2019-12-24 839條評論 4,651次閱讀 268人點贊

0-5！這就是米蘭在2019年的最後一場所交出的成績單。

這一戰，AC米蘭創造了俱樂部歷史上的3大恥辱紀錄——根據飛鯨體育足球歷史資料庫的統計，這是1937年以來，AC米蘭在意甲對陣亞特蘭大的最大比分失利，而上一次被亞特蘭大單場打進5球還要追溯到1950年3月12日，當時球隊以2-5實力；這是AC米蘭意甲歷史上第4次單場凈負5球，上一次是在1998年5月3日，客場0-5不敵羅馬；最後，這也是AC米蘭在過去21年以來各項賽事的最大比分失利。

雖然在賽前已經預料到這會是一場艱難的賽事，但相信沒人能夠想到，AC米蘭居然會幾乎以毫無還手之力的態勢被亞特蘭大所屠殺。對於紅黑軍團而言，這簡直就是一場災難，一場徹頭徹尾的恥辱。

其實在此前的4場比賽中，米蘭連續保持不敗，戰平那不勒斯和薩索洛、力克帕爾馬和博洛尼亞，似乎正在逐漸擺脫賽季初期的低迷表現。米蘭希望能夠用一場勝利告別2019年，並迎接俱樂部在邁入120歲後的首場比賽，結果卻是當頭棒喝。

整場比賽，AC米蘭看起來根本就不是亞特蘭大的對手。加斯佩里尼的球隊是一台完美運轉的機器，而皮奧利的球隊則似乎從未進入到比賽狀態當中，米蘭看起來沒有一個零件是管用的。面對亞特蘭大潮水般的攻勢，紅黑軍團完全被碾壓了，也跟不上亞特蘭大的節奏，防守輕易崩盤，中場位置站不住，進攻完全沒有威脅……

意甲聯賽17輪過後，AC米蘭僅僅只拿到了可憐的21分，這個分數比積分榜前兩位的國際米蘭和尤文圖斯足足少了一半，即使距離賽季初所制定的歐冠資格區域，也已經有著多達14分的差距。

造成如此大的差距，球員、教練、管理層甚至俱樂部的所有者，都逃脫不了責任。

一直被認為是米蘭復興希望的埃利奧特集團，做的最大一個決策便是聘請了原來阿森納的CEO加奇迪斯。他們希望能夠參照槍手在過去的多年運營模式賺取收益，並大力開發紅黑軍團在商業領域的價值。

可是，一段時間過去了，對比同城死敵國際米蘭在中國不斷收穫的的巨額贊助，我們並沒有看得到俱樂部在這些方面能夠取得太多進展。而在過去的三個轉會窗口中，他們超過2億的投資力度不可謂不小，雖然由於財政公平的規定，沒有太多一擲千金式的引援，但引援的數量並不少，可是能夠發揮作用的球員卻寥寥無幾，更別說能在市場上帶給俱樂部多少經濟的收益。誰都知道埃利奧特未來會將米蘭再次出手，但是目前觸發的條件和下家的類型都還看不到清晰的答案。

從萊昂納多到馬爾蒂尼，再到博班，米蘭名宿前赴後繼，一個又一個成為了俱樂部在技術領域的管理者。除了身份上象徵著米蘭的傳承和過去，似乎在實際競技層面上的操作，並沒有得到太多的證明。受制於加奇迪斯在財政控制和方向把控等方面的最終決定權，他們的工作也受到了制肘，只能在有限的層面上得以施展。

如今的米蘭，再也不是曾經那支叱吒風雲，可以引進頂級球星的豪門球隊。現在的技術團隊既要保證新加盟的球員未來能夠有升值的空間，也被要求球隊的成績必須過得去。在這樣的壓力之力，他們在建隊的思路上面，也無法完全按照主教練的想法進行打造。雖然新引進的球員不能說是差的選擇，但他們卻並不適合現有的戰術體系。引援不堪用，青訓出品也沒有更多的新鮮血液補充，反倒是把洛卡特利和庫特羅內一個又一個地送走。一線隊碌碌無為，上賽季預備隊的成績還遭遇了一個大倒退，直接以降級告終。

現在米蘭的陣中，缺乏合格領導者，缺乏一個有號召力的核心，缺乏有經驗球員的米蘭。球隊表現明顯沒有血性，沒有個性，心理脆弱。一旦陷入落後的狀態，球員就會陷入單打獨鬥，完全看不出這是一支爭奪歐冠席位的球隊應有的水平。

不管是博努奇從尤文短暫加盟後，再選擇打臉般的離去，還是伊瓜因來投短短半年之後，便宣告要遠走高飛，其實都證明了一個結果：米蘭已經不是引進一到兩個冠軍級別的球員能挽救的了。他們也很清楚這一點，不論是出於職業敏感度或者經濟層面的考慮，根本就不會考慮將米蘭復興的巨大責任扛到自己的肩上。

如果你所效力的球隊是米蘭，如果你身披紅黑間條衫，那麼你就應該全身心地為這件球衣奉獻所有的力量。然而，他們在賽場上所表現出來的精神面貌，絕對是任何俱樂部都無法接受的。散步般的防守，夢遊式的進攻，毫無所謂的表情，正如皮奧利所表示：「無論輸贏，似乎對我們的球員來都沒有任何影響。」

真真正正能夠配得上這一身球衣的，或許就只剩下多納魯馬和特奧等寥寥數人。雖然我們一直詬病多納魯馬受到經紀人的影響，屢屢在續約問題上與俱樂部產生不愉快，但起碼他還會哭，還會為了這個成績而自責。作為現今義大利的一號國門，他還從來沒有踢過歐冠，而且起碼在未來幾年都看不到有參加這項賽事的希望希望，或許他在米蘭的日子，也不會太過長久了。

至於教練，或許是過去這麼多年，在米蘭最容易受到指責的人物。從阿萊格里到米哈，再到因扎吉和加圖索，都是如此。怪責任何一個人，都是一件很容易的事情。可是對於紅黑軍團這一傢俱樂部和豪門來說，主教練或許更多只是一個替罪羊。阿萊格里離開之後，帶領尤文多次問鼎意甲桂冠並重回歐洲頂級球隊行列；上賽季打進前五的加圖索也受到了德佬的肯定，剛剛接過了那不勒斯的教鞭；在米蘭開始執教生涯的因扎吉，正在意乙聯賽率領貝內文託大殺四方，遙遙領先高居積分榜第一。這也證明，米蘭的根本問題，也許從來就不是教練的問題。

很多球迷曾以為的貝盧斯科尼離開之後，李勇鴻入主會成為米蘭復興的希望，然而卻被證明是鬧劇一場，之後期待埃利奧特能夠扭轉頹勢，帶領米蘭走出困境，可如今卻又成了漫長煎熬的起點。
