里皮：我以最好的方式教中國人踢球，六到七個月內可能重新執教

天天要聞
2019-11-22

離開中國之後，里皮似乎有數不清的話要說。日前，「銀狐」接受了《ilsussidiario》的專訪，並對自己的「中國之旅」進行了總結：「從職業角度來說，我以最好的方式教授中國人踢球，也贏得了很多（冠軍）。而對我個人來說，我非常高興能領略這個國家的歷史和文化。」

里皮

在國足負於敘利亞後，里皮憤而辭職。在妻子和兒子的陪同下，「銀狐」已回到義大利休息。不出意外的話，他會在半年後重新執教：「毫無疑問，我現在不會再執教俱樂部，尤其是義大利的俱樂部。或許在六到七個月內，我會重回教練席，並繼續在海外執教。」

里皮

作為前尤文主帥，里皮不可避免要談到意甲的爭冠大勢。在他看來，尤文圖斯更有優勢：「尤文圖斯，是排名歐洲前三或歐洲前四的球隊。國際米蘭和尤文圖斯的競爭，令人感覺非常刺激。我相信，這會持續到最後一輪，但我必須要強調的是：尤文的奪冠概率高達60%到70%，在聯賽和歐冠賽場上，他們至今都沒有輸球。」

里皮和馬達洛尼

令里皮感到欣慰的是：曾被他帶到世界之巔的義大利隊，又在曼奇尼的任內，找到了自信和統治力：「曼奇尼是一位非常高效的主教練，在技術和戰術上，他都非常成熟，他非常勇敢，徵召了一些甚至沒有參加過意甲的球員，並對他們委以重用。這是一支年輕的隊伍，但他們的表現非常出色。最為重要的是，他重新賦予義大利隊歸屬感。義大利隊一直在贏，但能持續多長的時間，還是未知數。我希望看到更好的義大利隊，這是我的期望，也是每一個義大利人的期望。」

雖然最後的結局，充滿了遺憾，但對里皮來說，在中國的執教經歷並不算失敗：「我非常熱愛足球這項運動。從職業角度來說，我以最好的方式教授中國人踢球，也贏得了很多的冠軍。而從我個人來說，我非常高興能領略這個國家的歷史和文化。」
