3-1逆轉，南美銀河戰艦進世俱杯決賽，利物浦爭冠第一對手

2019-12-18 106條評論 1,424次閱讀 81人點贊

　　為了世俱杯比賽，克洛普跟利物浦放棄了聯賽杯，結果利物浦U23隊主帥帶領的青年隊0-5慘敗給維拉，這個結果也讓人相當失望，而在另外一邊，克洛普跟利物浦即將迎來世俱杯半決賽，對手是蒙特雷。能否闖進決賽讓人期待，若利物浦進入決賽，利物浦能否奪冠也要先問一支球隊，那就是南美冠軍弗拉門戈。

　　12月18日的凌晨，世俱杯半決賽先賽一場，弗拉門戈迎來利雅得新月，後者是2019賽季新科亞冠冠軍，擁有喬文科、戈米。亞洲頂級強隊面對南美頂級強隊，實力對比如何讓人期待，然而90分鐘比賽戰罷，比賽沒有啥懸念。

　　上半場比賽利雅得新月率先取得進球，弗拉門戈0-1落後，下半場比賽弗拉門戈發動了反擊，開局僅僅1分鐘就追平了比分，這個進球也是掀起了反擊的號角，比賽第78分鐘和第82分鐘，弗拉門戈打進2球，3-1完成逆轉。

　　弗拉門戈率先闖進決賽，對手將是利物浦和蒙特雷的勝者。對於利物浦來說，他們絕對是世俱杯冠軍的奪冠熱門，但面對擁有巴爾博薩、里貝羅、恩里克、拉菲尼亞，號稱「南美銀河戰艦」的弗拉門戈，利物浦也需要掂量一下了，

　　這麼說吧，利物浦實力絕對是冠軍熱門，但對於志在跟利物浦會師世俱杯決賽的弗拉門戈來說，他們也想要給利物浦製造一點麻煩，如今弗拉門戈已經來到了決賽現場，就等利物浦能否撞線了，我們也期待這場決賽快點到來。

　　2005年，利物浦在歐冠完成神奇的逆轉，伊斯坦布爾奇蹟逆轉AC米蘭奪得歐冠，隨後的世俱杯比賽里，利物浦卻在決賽0-1輸給了聖保羅，球隊3個進球都被判罰無效，也是無緣第一座世俱杯冠軍，如今再一次出發，利物浦能否拿到第一座冠軍，讓人期待，

　　（代古龍）
