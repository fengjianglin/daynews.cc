恆大報價加圖索，與那不勒斯競爭？球迷：又一個抬價行為

2019-12-10 0條評論 22次閱讀 0人點贊

據《都靈體育報》稱，那不勒斯主席德勞倫蒂斯今天將在羅馬與加圖索會面，目的很明顯，目前那不勒斯成績不夠理想，只排在第7的位置，更是遭遇近9輪比賽不勝的尷尬。這樣的情況危及到了安切洛蒂的帥位，要知道，作為意甲豪門，那不勒斯儘管無法撼動尤文的冠軍地位，但穩住歐冠席位，他們還是沒有問題的。

安切洛蒂執教那不勒斯兩個賽季，球隊一直處在平穩當中，只是本賽季他們的表現讓球迷不太滿意，看情況，那不勒斯已經動了換帥的念頭，而他的前任主帥薩里同樣在尤文不夠開心，只是那不勒斯等不了啦，與加圖索會面，無疑讓安切洛蒂壓力非常大。歐冠層面，那不勒斯也沒有確保晉級，雙線的成績都很一般。

而這家意媒還表示，廣州恆大也將會對加圖索報價。在我看來，這純屬無稽之談，雖然恆大本賽季沒有拿到亞冠，但是卡納瓦羅至少拿到了聯賽冠軍，儘管說聯賽冠軍無法確保卡納瓦羅的帥位，但按照恆大過往的選帥標準，加圖索入主恆大，幾乎不太可能。

過往賽季，用恆大來抬價的行為已經屢見不鮮，而《都靈體育報》所言，毫無疑問，又是一個利用恆大抬價的虛假報道。
