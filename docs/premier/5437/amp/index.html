穆里尼奧迎主場首秀，熱刺能否再添一勝

天天要聞
2019-11-26

今日足球賽事雖不多但質量相當不錯，只因為有歐冠賽事。而穆里尼奧也將在主場完成帶隊熱刺的首秀。此役熱刺的對手是來自希臘的豪門奧林匹亞科斯。熱刺方面想必各名球員都想在穆家軍中贏得魔力鳥的信任，那麼奧林匹亞科斯能否給熱刺製造麻煩呢？

貝爾格萊德紅星vs拜仁慕尼黑

拜仁目前在小組中獲得12分，提前出線；而貝爾格萊德紅星則是僅僅拿到3分，此役就算獲勝也可能被淘汰。雖然貝爾格萊德紅星依然排名國內第一，但在主客場被近況不佳的熱刺以9-0的比分大勝，足見看出在歐戰中紅星實力還是有很大差距。不過貝爾格萊德紅星在自家主場，近15場歐戰僅僅輸掉2場，近22場也就輸掉3場，可見紅星主場還是有優勢的。而2007年歐聯杯主場不敵拜仁，也是紅星近12次歐戰主場面對德國球隊唯一的敗仗，其餘場次8勝3平。而紅星隊內的主力射手帕夫科夫（18場8球）則是預計缺席此役，中場烏卡諾維奇則是在周末聯賽中復出。拜仁方面球隊換帥後取得3連勝，包括4-0大勝多特，3場比賽都沒有失球。拜仁小組賽全勝，跟大巴黎是目前小組賽全勝的球隊。另外拜仁已經連續11場歐戰客場保持不敗（8勝3平），沒能取勝的對手包括歐冠第一豪門皇馬、上賽季歐冠冠軍利物浦和上賽季黑馬阿賈克斯，拜仁在客場還是有一定把握。此役拜仁做客給出-1.5球較低回報，後期升為-1.75球較高回報，雖然升指但同時升水；而競彩官方主隊受讓2球勝的獎金為2.02，跟拜仁凈勝至少3球的獎金2.6相比，賠付較低。我看好目前戰意並不算多的拜仁輸掉指數。

推薦：貝爾格萊德紅星最多輸1球

托特納姆熱刺vs奧林匹亞科斯

穆里尼奧接手熱刺後首秀客場擊敗西漢姆聯，可以說開了一個好頭。熱刺在歐冠本組中緊隨拜仁排名第二，目前4戰後積7分，領先第三名的貝爾格萊德紅星4分。此役熱刺如果獲勝將提前出線。在被拜仁打入7球後，熱刺利用紅星找回面子，兩回合9-0勝出。熱刺在歐冠主場打法比較激進，近13場贏下9場但卻輸掉其餘4場，但對手拜仁、阿賈克斯、巴薩或是尤文，實力均非奧林匹亞科斯可以相提並論。此役熱刺方面，傷停依舊是洛里、恩東貝萊、拉梅拉和維爾通亨；不過熱刺鋒線兩槍孫興慜和凱恩狀態不錯，孫興慜也繼續斬獲進球。奧林匹亞科斯目前已經出局，歐冠客場遭遇6連敗。而奧林匹亞科斯此前14次歐戰客場對陣英格蘭球隊取得1勝1平12負的糟糕成績，證明跟英格蘭球隊的實力差距。這場比賽坐鎮主場的熱刺給出-1.75球讓球指數，作為穆里尼奧帶隊主場首秀，我認為球隊肯定會拿出更積極的表現；另外熱刺客戰對陣奧林匹亞科斯也是曾取得2球領先，我認為這場比賽熱刺至少能夠贏2球。

推薦：熱刺

莫斯科火車頭vs勒沃庫森

莫斯科火車頭和勒沃庫森目前各積3分，可以說兩隊此役誰能勝出誰還能保留一絲出線希望。兩隊首回合交鋒，坐鎮主場的藥廠讓出1.5球指數，結果1-2不敵莫斯科火車頭，但那場比賽火車頭的2粒進球全部是來自勒沃庫森球員的非受迫性失誤。目前莫斯科火車頭近6場比賽中只贏下1場，便是上場聯賽客場擊敗坦波夫，期間包括2場歐冠均以1-2的比分輸給尤文圖斯。火車頭上場擊敗勒沃庫森結束了對陣德國球隊的7連敗，但在主場球隊依舊不敵德國球隊，已經吃到4連敗且得失球為1-9，可以說火車頭天生懼怕德甲球隊。而勒沃庫森近期跟俄羅斯球隊交鋒沒有佔到優勢，近5次交鋒4平1負。近期勒沃庫森取得2勝1平的戰績，可惜上場德甲在主場未能擊敗弗萊堡。此役勒沃庫森做客給出-0.25球較低回報，對近期狀態略有起勢的勒沃庫森有所看好。另外對於勒沃庫森而言此役還是球隊第100場歐冠比賽，作為歐洲第22支完成如此殊榮的球隊而言戰意較強。

推薦：勒沃庫森

亞特蘭大vs薩格勒布迪納摩

亞特蘭大做客面對薩格勒布迪納摩被對手打出4-0，可以說此次在主場亞特蘭大需要證明自己的實力。在本組目前排名中，曼城以3勝1平的戰績遙遙領先，薩格勒布迪納摩跟頓涅茨克礦工同積5分，而原以為會以小組第二出線的亞特蘭大則是墊底，僅僅獲得1分。亞特蘭大上場聯賽剛剛在主場1-3慘敗給缺少C羅的尤文，近5場賽事一勝難求。雖然亞特蘭大近年在意甲中經常打出多進球，但此役僅僅是亞特蘭大第40場歐戰，經驗略顯不足。薩格勒布迪納摩則是歐冠賽事已經超過40場，球隊近期在客場戰績不錯，4勝3平3負，本季資格賽至今客場5戰2勝2平1負，僅2球小負曼城。這場比賽指數開得中規中矩，坐鎮主場的亞特蘭大讓出1球指數，可以說還是給予亞特蘭大足夠的信心。從戰意來看，亞特蘭大出線基本不太可能，而亞特蘭大還有多少實力能夠復仇成功呢？從首回合賽況來看，亞特蘭大客場4球完敗，肯定跟不擅長跟對手踢比賽有關。我認為這場讓一球穿盤太有難度。

推薦：薩格勒布迪納摩最多輸1球

達拉斯獨行俠vs洛杉磯快船

兩支球隊近期狀態均不錯，獨行俠方面自從擊敗猛龍以來球隊豪取5連勝，球隊德州內戰贏球前4連勝都是在主場，可以說獨行俠連勝期間跟賽程優勢有一定關係。期間東契奇發揮十分出色。快船則是在擊敗鵜鶘後同樣迎來5連勝，期間球隊5場比賽限制對手得分在110分以內，保羅喬治復出後，快船替補席實力更強。本場比賽指數是做客的快船讓0.5分，我認為雖然獨行俠本賽季突飛猛進但球隊替補席上沒有穩定輸出點，跟快船替補席上擁有哈雷爾和路威兩名猛將有較大差距，我看好快船再勝。

推薦：洛杉磯快船
