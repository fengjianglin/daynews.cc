巴薩日本小將梅開二度，已連續7場首發，曾是亞冠冠軍隊主力

2019-12-16 1,229條評論 2,398次閱讀 660人點贊

南野拓実加盟利物浦的傳聞愈演愈烈，另一名日本球員在昨晚卻已經在歐洲豪門閃耀！西乙B聯賽第17輪，巴薩B隊3-1戰勝拉努西亞，20歲的日本小將安部裕葵完成梅開二度，幫助球隊取勝。

安部裕葵的這2粒進球都非常精彩。上半場，在球隊落後1球的情況下，安部裕葵在禁區正面踢出一腳凌空貼地斬，扳平比分。

下半場，在一次反擊中，安部裕葵接隊友助攻，在禁區外重炮轟門，直掛球門死角。

這是安部裕葵夏天加盟巴薩B隊以來，首次在比賽中完成梅開二度。目前，他已經為巴薩B隊打進4球，最近7場聯賽，安部裕葵更是場場首發，已經在西甲豪門梯隊站穩腳跟。

其實，在開啟旅歐生涯之前，安部裕葵就已經是日本國內豪門鹿島鹿角的絕對主力了。在球隊2018年隨隊奪得亞冠冠軍的過程中，安部裕葵發揮了重要的作用，決賽兩回合更是全部首發。

安部裕葵夏天加盟巴薩

前不久與國際米蘭的歐冠最後1輪小組賽前，曾有傳言安部裕葵將會隨一線隊出征，雖然最終這名20歲的小將沒能出現在球隊的歐冠大名單，但相信未來在巴薩一線隊看到安部裕葵的身影，也只是時間問題。

南野拓実、久保健英、安部裕葵

最近半年，日本年輕球員在歐洲賽場不斷帶來驚喜，在久保健英加盟皇馬、安部裕葵加盟巴薩後，南野拓実與利物浦的緋聞更是令人羨慕。下賽季，我們或許就阿靜看到不止1名日本球員在歐洲頂級豪門立足了。
