意甲最新積分榜：國米遭絕平微弱優勢領跑，C羅2球尤文全取3分

2019-12-16 165條評論 1,574次閱讀 664人點贊

昨晚今晨，意甲聯賽第16輪比賽繼續進行，領頭羊國際米蘭客場1-1戰平佛羅倫薩，遭遇2連平，憑藉凈勝球優勢繼續居首；尤文圖斯主場3-1擊敗烏迪內斯，C羅梅開二度；羅馬主場3-1擊敗斯帕爾，升至積分榜第4位，雙方各罰入一粒點球。

國際米蘭客場1-1戰平佛羅倫薩，第8分鐘，布羅佐維奇送出直傳，巴萊羅晃過後衛後低射近角得手，他進球後拒絕慶祝，以示對老東家的尊重；第39分鐘，盧卡庫右路傳中，勞塔羅推射破門，但是VAR認定此球越位在先；補時第2分鐘，波利塔諾丟球，弗拉霍維奇長途奔襲爆射破門，上演絕平好戲。

尤文圖斯主場3-1擊敗烏迪內斯，第9分鐘，博努奇後場長傳製造混亂，C羅低射破門首開紀錄；第37分鐘，伊瓜因送出直塞，C羅推射梅開二度；第45分鐘，尤文角球開出，博努奇頭槌破門擴大比分；補時最後階段，客隊普賽托破門扳回一分。

其他比賽中，羅馬主場3-1擊敗斯帕爾，客隊佩塔尼亞打進點球，隨後托莫維奇打進烏龍球扳平，佩蒂特和姆希塔良連入2球逆轉，羅馬殺進前4；維羅納主場3-3戰平都靈，主隊3名替補各入1球，連扳3球戰平；AC米蘭主場0-0戰平薩索洛，博洛尼亞主場2-1擊敗亞特蘭大。
