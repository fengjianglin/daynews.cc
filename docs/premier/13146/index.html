凱西：想在米蘭效力多年 若想離隊我早就走了

2019-12-13 1,988條評論 4,982次閱讀 2410人點贊

直播吧12月13日訊 在接受《慢鏡頭》的採訪時，AC米蘭中場凱西回應了關於自己的轉會傳聞，他表示他在米蘭很好，他想在這裡效力多年。

關於教練

「我跟皮奧利相處得非常好，跟我在米蘭遇到的其他教練一樣。我的行動就是想讓所有人明白，教練對我沒有任何敵對，我對教練也沒有任何反對。」

關於落選對陣尤文大名單

「我跟皮奧利之間沒有任何問題，跟他合作我感覺很好。我沒有踢對陣尤文的比賽，當時我受傷了，另外我在訓練中遲到了，所以教練沒把我列入大名單，沒有任何大問題。之後教練讓我踢比賽就是證明，他沒有針對我，反之亦然。」

關於米蘭賽季初的低迷

「我們走了一些錯路，但現在我們正在進步，我們正在為了實現目標而努力。打進歐冠？我認為現在一切仍是開放的。」

關於轉會市場的聲音

「如果我想走，我早就走了……我想在這裡效力多年，米蘭是我從小就開始支持的球隊，對我來說我實現了自己的夢想，我永遠會為紅黑兩色全力以赴。」

關於兒時的偶像

「我尤其喜歡西多夫。另外我一直喜歡埃辛，當他加盟米蘭的時候，我非常高興。」
