博格巴經紀人怒斥曼聯貪婪，已喪失傳統！博格巴服軟不會強行轉會

2019-12-21 2,516條評論 3,839次閱讀 2424人點贊

曼聯主帥索爾斯克亞在英超賽前新聞發布會上，公開否決了博格巴冬季轉會的可能性。「他不會在一月份被出售，」索帥表示，「我希望他今年結束前可以復出，但我不會強迫他，我不能冒險導致他舊傷複發。」

此前，博格巴被認為拒絕為曼聯出場，想要以此方式迫使曼聯允許他轉會皇馬。但由於曼聯強硬應對，博格巴的經紀人米諾·拉伊奧拉改變態度，公開承認博格巴會留下為曼聯爭奪冠軍。

拉伊奧拉接受英國《每日電訊報》採訪時說：「我認為保羅（博格巴）真的回了家，他本可能去其他地方，但他用心選擇，回到了曼聯的家。保羅是個非常、非常美妙的人。舉個例子，他不能對奧萊（索爾斯克亞）生氣，因為奧萊還是球員時，作為老將，他對年輕的保羅非常好，他從奧萊那兒得到的愛和尊重，意味著他不能對奧萊生氣。」

2016年時，博格巴以8900萬鎊創紀錄轉會費從尤文圖斯重返曼聯，拉伊奧拉被曝從中賺取了2400萬中介費。但他堅稱自己和博格巴並非為錢而來，曼聯能買到他是因為對球員志在必得，不在意花錢。

「我不在意什麼創紀錄轉會費，」拉伊奧拉說，「我在意的是，他是不是做出了對職業生涯有利的正確決定？然後我才考慮，這是不是值得？他回曼聯，不是因為他們給了最多的錢。他去曼聯，因為他想去曼聯，然後他們標出價格，那就是我介入的時候。市場決定價格。你知道，這就像勞斯萊斯，客戶會進來問：『勞斯萊斯多少錢？』如果你要問這種問題，說明你買不起。這不是錢的問題，而是你想要還是不想要。如果你想要，你就會買下它。」

拉胖堅稱，博格巴渴望幫助曼聯重返巔峰：「保羅希望成功，他希望快樂，想要贏得冠軍，他很樂意在曼聯取得這些成就。是的，皇馬對他非常感興趣，但曼聯不讓他去。我們並沒有因此鬧事，他沒有因此鬧情緒，因為我跟他是步調一致的。我們知道其他球員可能採取極端手段，但保羅不喜歡那樣。所以我們採用了不同的做法。」

但博格巴的經紀人明確表示，對轉會皇馬沒能成功感到遺憾：「皇馬確實有興趣，但他們（曼聯）不允許他轉會，我非常遺憾，因為我認為保羅是法國人，齊達內則是法國歷史上重要的人物。但保羅並不是我從內維爾和所有憤怒的前曼聯球員口中聽說的那樣的人，『他應該為這個感到羞愧，為那個感到羞愧』，羞愧啥？他每天努力，做到最好。他沒有抱怨無緣歐冠。他是不是完美的？沒人能完美。他是不是要為失敗負責？他感覺是的。如果你在隊里，就要為失敗負責。」

拉伊奧拉還怒斥曼聯俱樂部現在過於貪財，已經失去了傳統的特點。「弗格森執教時，你至少可以看出這是他的球隊——曼聯的打法，積極主動，球員都知道應該如何踢一場比賽。」這位充滿爭議的經紀人說，「所以我說，你必須在俱樂部確立你想要的特質，然後就像樂高積木一樣，尋找合適的部件，你需要明確性。事實上，現在我看著曼聯，覺得他們更多體現著美國精神，而不是歐洲足球。我的意思是，有時候俱樂部更在意經濟收入，而不是競技成績。那是非常美國化的思維，他們需要有人平衡財政和成績。」

和許多人一樣，拉伊奧拉覺得曼聯需要聘請體育總監，「俱樂部需要一名專家，一個體育總監，在球隊、教練和老闆之間充當橋樑。今天，你必須要有專家，因為教練已經沒有足夠時間管理球探和其他任務，我感覺曼聯就缺少這個。」

不過，素以貪婪著稱的拉伊奧拉批評曼聯貪婪，這一幕有些令人哭笑不得。他的抱怨或許也基於曼聯對博格巴的標價，英媒稱如果皇馬或者尤文想要買下博格巴，至少需要給出1.5億英鎊轉會費。
