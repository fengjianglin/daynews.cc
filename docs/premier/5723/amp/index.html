意甲今日頭版：C羅沒有任何道歉 米蘭關注曼朱&默滕斯

天天要聞
2019-11-12

直播吧11月12日訊 米蘭體育報、羅馬體育報、都靈體育報是義大利的三大體育報，今天我們來看看它們各自的頭版頭條都帶來了哪些內容。

《米蘭體育報》

1.「克里斯蒂亞諾後悔。」對陣米蘭被換下場後，尤文巨星跟教練嘗試緩和。C羅在社交網路上發出了一條緩解緊張的動態，球迷們支持薩里。葡萄牙人減少爭議：「這是一場重要的勝利。」尤文不會罰款，但C羅從葡萄牙歸來後需要在更衣室內做出解釋說明。

2.維亞利和曼奇尼：國家隊的兄弟。多麼棒的組合，27年後，兩人再次共事。維亞利擔任領隊。

3.盜賊入室，阿蘭為家人擔憂害怕，現在他考慮告別那不勒斯。

4.「國米攻勢。」在歐洲，國米是唯一一支一直進球的球隊。寶貝巴爾博薩追平巴西傳奇球星濟科單賽季進球紀錄。

5.從忍者（納因戈蘭）到皮亞特克，分析聯賽中狀態爆棚和狀態低迷的球員。

《羅馬體育報》

1.「我就是我。」葡萄牙巨星登上封面，C羅事件，他對尤文沒有任何道歉。一條沒有脾氣的簡短ins，然後飛往葡萄牙。俱樂部這樣做：沒有任何罰款。

2.「報復那不勒斯。」檢查院對阿蘭家中遇竊、澤林斯基妻子汽車被破壞開啟調查。

3.弗洛倫齊讓曼奇尼和豐塞卡分裂。曼奇尼把弗洛倫齊召回國家隊：「對我來說，他是一名重要的球員。」而羅馬教練認為：「對我而言，他是一個選擇。」

4.維亞利的藍色之心：「我會這樣幫助年輕人。」

《都靈體育報》

1.「薩里和CR7的真相。」獨家！本報給你們透露C羅踢米蘭下場時說了什麼，尤文沒有採取任何罰款，只是擔心他的膝蓋在國家隊比賽中情況。本報給你們揭秘C羅膝蓋上的繃帶。

2.本賽季意甲更精彩要感謝馬蘭的卡利亞里。

3.國米，孔蒂的引援名單。

4.米蘭關注默滕斯和曼朱基奇。

5.西斯塔納：從意丁到布雷西亞，再到國家隊。

（我叫達芬奇）
