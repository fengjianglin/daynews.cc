周三精彩二串一推薦：​法甲 第戎-蒙彼利埃

法甲002 第戎 – 蒙彼利埃

比賽時間 12-5 02:00

第戎這支球隊上賽季在聯賽積分榜18名，本賽季在聯賽積分榜17名，第戎目前是在全力以赴的爭取分數，為保級做準備。第戎雖然說主場戰績還算不錯，但是客場確實是比較平淡，最近三個客場都是失利了。第戎本賽季的發揮並不理想，僅以4勝3平7負的戰績排在第17位，處於降級區邊緣。值得一提的是，第戎的4場勝利含金量極高，其中一場是在主場2-1擊敗巴黎聖日耳曼，上輪也是在主場擊敗了雷恩，主場能力值得肯定。

蒙彼利埃上輪法甲客場0-0悶平聖埃蒂安，近5輪比賽2勝3平保持不敗，他們近10輪法甲4勝4平2負，10場比賽總計只有7粒失球；蒙彼利埃近10個主場擁有6成勝率（6勝3平1負），僅負於雷恩，球隊表現不錯。球隊最近6個主場5勝1平，期間連續掀翻里昂和摩納哥兩大勁旅，6場比賽總計只有2粒失球。蒙彼利埃近9輪比賽只有3個上半場有過失球，近5輪法甲只有1個上半場有過進球，近6個法甲主場只有1個上半場沒有進球，只有1個上半場有過失球。

蒙彼利埃現雖暫排法甲第6位，但積分上距離第二名也僅有5分之差，球隊依然是歐冠資格的有力競爭者，利好消息是球隊以10個失球成為聯賽防守力第三強的球隊，不過攻擊力略顯欠缺一些才打進14個球，與他們的排名有些不符。而且蒙彼利埃客場成績排名倒數，還沒有客勝的記錄，下輪還沒面對大巴黎，今場面對護級分子，必須要拿出更好的狀態。

賽事預測：蒙彼利埃

比分參考：0-1，1-2

意杯1:00萊切vs史帕爾

比賽時間：2019-12-05 1:00

賽事分析：

史帕爾在本賽季意甲聯賽中表現不佳，在過去14輪聯賽中只取得2勝3平9負的成績，目前排在積分榜第19位，已經深陷降級區當中，特別是在客場方面，7場客場比賽只取得1平6負的成績，至今還未獲得首勝。

萊切是本賽季的升班馬球隊，在上一輪意甲聯賽中，萊切做客出戰佛羅倫薩，最終在客場1-0擊敗對手，球隊近期接連挫敗意甲強隊，在過去7場聯賽中，萊切取得1勝5平1負的成績，僅僅只是敗給拉齊奧，在面對尤文圖斯、AC咪蘭時都頑強逼平對手。球隊本賽季8場客場比賽取得3勝2平3負的成績，客場表現還算不錯。

兩隊在本賽季聯賽中首次交手時，史帕爾在主場面對萊切時，被對手以3-1擊敗。史帕爾近期已經連續7輪聯賽不勝，球隊狀態極其低迷，加上球隊的保級壓力較大，恐怕無法分心意杯賽事。而萊切近7輪聯賽只有1場敗仗，球隊不敗能力極高，加上球隊客場表現不錯，本場比賽做客狀態低迷的史帕爾，萊切有一戰之力。

 精彩推薦推薦：勝
