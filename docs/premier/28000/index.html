【1三哥競cai】多特1-2冷負霍村，羅馬4-1紫百合

2019-12-21 2,184條評論 3,356次閱讀 683人點贊
  • 註：昨天兩場分析都穩穩命中，今天就帶大家回顧一下昨天的賽事

【德甲】格策破門，多特客場1-2遭霍村絕殺逆轉

北京時間12月21日今天凌晨3:30，德甲第17輪，多特蒙德做客挑戰霍芬海姆。上半場格策推射破門，阿什拉夫遠射打中橫樑，半場多特一球領先霍村。下半場回來，阿達米揚補射扳平，終場前克拉馬里奇頭槌絕殺。最終，多特蒙德在客讓0.5球的情況下，1-2遭到霍芬海姆逆轉，雙線5輪不敗被終結。

【意甲】哲科獨造3球，羅馬客場橫掃4-1紫百合

北京時間12月21日凌晨03:45，意甲第17輪，羅馬客場對陣佛羅倫薩。上半場，哲科和科拉羅夫相繼破門，佛羅倫薩這邊由巴德利攻入一球，羅馬客場2-1領先;下半場兩隊易邊再戰，佩萊格里尼、扎尼奧洛破門，將比分定格在4-1。最終，羅馬客場4-1戰勝紫百合，雙線7場不敗，紫百合則慘遭聯賽7輪不勝。

【官宣】阿爾特塔正式出任阿森納主帥，簽約3年半

北京時間12月20日昨晚，阿森納官方宣布，阿爾特塔正式回歸出任主帥，合同為期三年半。2011年阿爾特塔加盟阿森納，他在酋長球場效力了5年時間。2016年7月3日，時任阿森納隊長阿爾特塔宣布退役並加入曼城教練組，擔任藍月亮助教至今。據天空體育消息，阿森納將向曼城支付200萬英鎊賠償金。

【豪門】尤文公布意超杯大名單：基耶利尼時隔4個月回歸

北京時間12月23日周一凌晨0點45分，義大利超級盃正式打響，尤文圖斯對陣拉齊奧。尤文已經公布了意超杯大名單，C羅、迪巴拉領銜幾乎全主力陣容。值得一提的是，基耶利尼時隔4個月重回陣容，但無法上場。基耶利尼在8月底遭受了右膝前十字韌帶撕裂的傷病，預計傷缺6-7個月。

【關注】442評選年度百大巨星：梅西+范迪克+薩拉赫前3，C羅第4

北京時間12月21日消息，近日歐洲知名足球雜誌《FourFourTwo》評選出2019足壇百大巨星，其中梅西高居榜首，范迪克和薩拉赫位列2、3，C羅排在第4，姆巴佩第5，進入前10的還有奧布拉克、萊萬、德布勞內、斯特林和馬內。比較意外的是，內馬爾僅排名第23。孫興慜排在第30位，他也是唯一進入前100的亞洲球員。

【情報】英超重點傷停：曼城4主力受傷 博格巴難復出

北京時間12月21日今晚，英超聯賽第18輪陸續打響，焦點戰包括熱刺vs切爾西、埃弗頓vs阿森納、沃特福德vs曼聯和曼城vs萊切斯特等。最新的傷停當中，曼城4名主力受傷——阿奎羅可能復出，薩內、大衛-席爾瓦和拉波爾特缺陣，熱刺則缺少洛里斯、拉梅拉和本-戴維斯，曼聯的博格巴和羅霍、切爾西的奇克、阿森納的科拉希納茨也將缺陣。
