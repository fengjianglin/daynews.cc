曝西班牙人曾想引進張琳芃，恆大開中超市場價，轉會最終破裂

2019-12-04 0條評論 18次閱讀 0人點贊

北京時間12月4日，足球記者豐臻透露，2年前，西班牙人俱樂部曾想引進張琳芃，但恆大方面開出中超市場價，這讓西班牙人無法接受，轉會破裂。也就是說，如果當時恆大放人，如今，武磊和張琳芃都可能成為西班牙人隊友，這畫面，中國球迷不敢想。

巔峰時期，張琳芃是亞洲最頂尖的後衛，甚至和皇馬、切爾西、國際米蘭傳出緋聞。2015年，張琳芃與英超豪門切爾西傳出了緋聞，成為了當時最熱鬧的新聞，但張琳芃最終沒有轉會成功。

與此同時，同樣是那一年，國際米蘭計劃引進中國球員張琳芃，而里皮也建議藍黑軍團簽下恆大右後衛，最終，恆大開價800萬歐元，國際米蘭打了退堂鼓。

如今，豐臻透露，西班牙人兩年前找恆大談引進張琳芃，恆大開出了一個中超市場價，交易沒做成。現在設想一下，兩年前張琳芃的中超市場價，可能不會低於一億人民幣，因此，轉會沒有成行也是正常的。

武磊之所以能夠加盟西班牙人，實力是一方面，更重要的是，上港基本白送武磊留洋，因為他的轉會費只有200萬歐元，約合人民幣不到2000萬！

因此，張琳芃至今未能實現留洋夢想，而武磊已經逐漸在西甲閃光，兩人的命運如今天差地別。值得一提的是，張琳芃已經30歲了，並且狀態下滑，他也很難實現留洋夢想了。

此外，張琳芃在國家隊失誤，導致國足1-2不敵敘利亞，如果他有機會2年前去到西班牙人，狀態可能就不是現在這樣了。可以說，張琳芃的職業生涯還是留下遺憾，本來，他可以像昔日隊友武磊一樣，在西甲揚名立萬。
