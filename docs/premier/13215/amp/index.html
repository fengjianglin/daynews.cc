瘋狂暗示？小組第二連續3年拿歐冠，皇馬欲反擊，穆帥或創紀錄

天天要聞
2019-12-11

歐冠小組賽第6輪的比賽已經與今天凌晨正式打響，在最後一輪小組賽結束後，各小組的出線形勢也將逐漸明朗。在已經結束比賽的小組中，利物浦攜手那不勒斯成功晉級；多特蒙德則在小組賽最後一輪反超國米拿到小組第二，和巴薩一同進入淘汰賽。其餘的晉級球隊包括了來自德甲的RB萊比錫，里昂，瓦倫西亞和切爾西。而在明天凌晨，其餘四個小組的比賽也將全部進行，屆時歐冠十六強將全部揭曉！瘋狂暗示？小組第二連續3年拿歐冠，皇馬欲反擊，穆帥或創紀錄。

本次歐冠決賽舉辦地

有趣的是，今天國外媒體曝出了一項數據。那就是在過往的三個賽季，拿到歐冠冠軍的都是排名小組第二的球隊。2016-17以及1027-18賽季，皇家馬德里兩度奪得了歐冠冠軍。而在去年，紅軍利物浦也是時隔多年後再次拿到歐冠。這個定律引起了人們的無限遐想，這是不是一種暗示呢？因為目前排名小組第二的球隊，有很多都具備足夠的奪冠實力。

多特蒙德

首先是F組的多特蒙德。德甲「大黃蜂」絕非浪得虛名，他們作為歐冠的常客，經常在這個賽場製造出大動靜。當年在克洛普的帶領下，他們曾在半決賽擊敗皇馬闖入決賽，最後惜敗給拜仁慕尼黑。雖然近年來實力有所下降，但在本賽季的歐冠小組賽中，他們也曾逼平過強大的巴塞羅那。這支新老結合的球隊，對任何球隊來說都是一個難纏的對手。

貝爾

再來看看A組，目前皇家馬德里以8分排名第二。雖然這個小組的比賽還沒有結束，但是小組排名已經沒有了懸念。身後的布魯日和加拉塔薩雷無力爭奪出線名額，身前的巴黎聖日耳曼又領先自己5分。無欲無求的皇馬最後一輪很可能進行較大的輪換，為之後的淘汰賽留力。要說到小組第二最終拿歐冠的事，皇馬是最有發言權的。他們在之前連續兩個賽季以小組第二出線，並最終闖入決賽。分別以4-1擊敗尤文圖斯以及3-1擊敗利物浦，兩度奪得冠軍。雖然目前的皇馬不是歐冠賽場最強的球隊。但「銀河戰艦」向來是遇強則強，擁有豐富歐戰經驗的他們，絕對不可小覷。

穆里尼奧

最後是B組。這個小組的情況和A組相同，排名第二的托特納姆熱刺同樣是追不上前面，也不必擔心後面，提前坐穩了小組第二的位置。不過，熱刺在中途經歷了換帥，倘如沒有經歷那場2-7的慘敗，現在的小組第一是誰還真不一定。不過，自穆里尼奧入主以來，球隊就有了煥然一新的變化，在最近5場比賽中贏了4場。明天凌晨，他們即將迎來複仇之戰，客場挑戰拜仁。不過，這場比賽其實無關痛癢。後面的淘汰賽，才是對穆里尼奧的考驗。不過，「魔力鳥」最不怕的就是挑戰。執教生涯斬獲無數榮譽的他，此番會不會帶領熱刺再度殺入決賽並奪冠呢？球迷們紛紛表示，此時的穆帥有一個大膽的想法！

媒體調侃

而除了以上的球隊，目前的小組第二還包括里昂，切爾西，那不勒斯和馬德里競技。這些球隊沒有一個是好惹的，這也讓本賽季的歐冠變得越來越好看。那麼，球迷們認為，誰將會是今年的歐冠冠軍呢？
