涉及德甲榜首之戰，誰能更勝一籌？

2019-12-17 321條評論 2,903次閱讀 1135人點贊

北京時間2019-12-18凌晨03:30分，德甲聯賽第十六輪，

基本面

多特蒙德主場迎接萊比錫紅牛的挑戰。目前在德甲聯賽再通過十五輪的交鋒中，門興客場不敵沃爾夫斯，而萊比錫紅牛通過客場3-0大勝杜賽多夫逆襲門興暫列榜首，使德甲的競爭更具挑戰性。隱忍三個賽季，萊比錫紅牛力壓群雄，多特蒙德將是你的墊腳石

作為上屆冠軍熱門拜仁和多特蒙德的表現，多特蒙德雖然不算穩定，但是本賽季目前卻依然保存著爭冠的能力，反而上屆冠軍拜仁，在本賽季的表現卻不盡人意，目前在聯賽僅排第五，而與榜首的積分差距已高達六分。而作為本次比賽，無論是對於多特蒙德還是萊紅牛都是關鍵一戰，由於其後對手實力不強，多特蒙德之後分別面對霍芬海姆.奧格斯堡.科隆.，萊比錫紅牛則要面對奧格斯堡.柏林聯合.法蘭克福。由於之後比賽對手實力一般，本場比賽可能是穩定榜首或逆襲的關鍵時刻，兩隊必定非常重視本場比賽。

德甲豪門多特蒙德本賽季多線作戰均有亮眼發揮，歐冠層面球隊小組賽最後一輪主場2-1力克捷甲豪門斯拉維亞，而國米主場不敵西甲巨人，從而幸運的驚險涉關闖入淘汰賽。聯賽層面15輪過後積29分暫列第三，與領頭羊差距4分。多特近期狀態不俗，上輪做客美因茨的比賽中球隊4球大勝對手後豪取本賽季第二次各項賽事四連勝，多特擁有魔鬼主場之稱，賽季至今12個各項賽事主場9勝3平保持不敗，諸多豪強到訪均未能討到便宜，強如巴薩、國米這樣的歐洲豪門同樣在此折戟，而在聯賽中多特是賽季至今主場唯一保持不敗的球隊，場均打入3.3球和場均失0.86球的攻防數據俱佳。

而萊比錫紅牛在近幾個賽季一直十分強勢，可以說在德甲的地位越來越穩固，尤其是在本賽季的表現讓球迷眼前一亮，本賽季球隊同樣多線作戰但絲毫沒有乏力現象，先是在歐冠以小組第一身份直接晉級淘汰賽，又在聯賽層面球隊以33個積分領跑積分榜。上輪客場3-0輕取弱旅杜塞爾多夫後豪取各項賽事10場不敗，球隊擁有五大聯賽最強攻擊火力，聯賽場均打入2.8球，球隊形勢一片大好。

本場比賽事涉德甲榜首之戰，兩隊都不容有失，近期兩隊的狀態都十分火熱，不過從兩隊定位上來看多特還是處於優勢的兩隊最近幾次交鋒中萊紅牛卻難求一勝，尤其是在上賽季分別被多特蒙德主客場5-1擊敗，本次兩隊交鋒萊比錫紅牛或許存在一定心理壓力。指數方面給出多特讓平半而且搭配中高水此讓步相比歷史交鋒是歷史最弱交鋒指數明顯對多特取勝信心嚴重不足，因此本場走萊紅牛至少不敗才是首選。

比分看好：1-1，1-2，2-2

本場推薦：萊紅牛+0.25，
