天津女排墊底，其中三國手受意甲冠軍大帥青睞，除朱婷還要這兩人

2019-12-09 0條評論 19次閱讀 0人點贊

北京時間12月9日，此次的女排世俱杯國內兩支女排球隊均為出線，也是讓觀眾大失所望，雖然兩支球隊在成績上表現不理想，但是不代表中國女排實力弱，在隊伍中還是有很多具備優秀實力的球員，而在近日歐洲亞軍意甲冠軍的科內利亞諾女排教練聖雷利接受採訪，談到中國女排的球員，聖雷利表示看好三位球員，並且希望奧運會後有機會非常想與她們合作。

聖雷利表示雖然本次的賽事上，天津女排與廣東恆大女排表現一般，但是優秀球員的光芒總是鋒芒的，並且希望自己能夠有機會執教她們，朱婷是首選，然而聖雷利還指出除了朱婷以外，還有另外兩名球員，分別是袁心玥與李盈瑩，年僅19歲的李盈瑩，在左手進攻方面表現突出，並且年紀的優勢在未來的發展空間非常大，而身高達到2米01的袁心玥，在攔網方面讓他記住了這位女孩，如果三人都能夠有機會出現在他的球隊里，那他會非常開心。

聖雷利表示也明白做到這點很困難，幾位優秀球員都是國內球隊的核心主力，俱樂部方面不會輕易讓她們走的，想完成轉會難上加難，然而這對於袁心玥與李盈瑩來說這是能見識國際舞台的大好機會，而當初河南女排就為了中國女排與朱婷的個人發展，忍痛割愛放走朱婷這樣的核心主力，這樣的舉動讓人佩服，中國女排才有了這樣一位超級隊長。

如今同樣的問題擺在了天津女排與八一女排面前，如果真的有機會促成袁心玥與李盈瑩留洋發展，他們是否也能做出河南女排一樣的舉動？如果奧運會過後袁心玥與李盈瑩都能走向國際舞台，那麼中國女排的實力將會再上一個台階，大家對此有何看法，歡迎評論區留言~
