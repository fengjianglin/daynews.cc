羅馬上輪客戰國米，賽況不落下風，最終0比0全身而退，雖說沒能取得各項賽事三連勝但從梅阿查球場拿分也可喜可賀，而在本周中的歐巴羅聯賽小組賽最後一輪，羅馬主場2比2戰平沃爾夫斯貝格，賽後羅馬將帥均承認本隊發揮欠佳，但最重要的是能夠出線。本周末，羅馬將在意甲第16輪主場迎戰弱旅費拉拉SPAL，對於羅馬來說，這絕對是一場復仇之戰，上賽季羅馬痛失歐冠資格的關鍵原因之一就是主客場被費拉拉SPAL雙殺，上半程主場0比2失利，下半程客場1比2再輸。因此，哪怕本周中出戰了歐羅巴聯賽，羅馬也肯定會在本場比賽拼盡全力。不過，小克魯伊維特、克里斯坦特、帕斯托雷、扎帕科斯塔等人將因傷缺席，詹盧卡-曼奇尼禁賽。

隨著上輪主場0比1不敔布雷西亞，費拉拉SPAL聯賽已經連續8輪不勝，期間只取得了2平6負的糟糕戰績，目前僅積9分排名墊底，本輪將會前往奧林匹克球場挑戰積29分排名第5位的羅馬。出戰本場比賽之前，費拉拉SPAL在本賽季此前7個聯賽客場0勝1平6負，進1球失11球，此乃球隊最直觀的短板而羅馬一向都擅長於主場作戰，由此看來，費拉拉SPAL本輪前景實在是凶多吉少。更為槽糕的是，球隊如今傷情非常嚴重，雷查、迪弗朗西斯科、埃斯佩托、薩拉、達歷桑德羅、法里斯都將因傷缺席本場比賽。

兩隊近4次交鋒，羅馬2勝0平2負，進7球，失5球。
