12.15 意甲：佛羅倫薩vs國際米蘭 佛羅倫薩戰術保守國米難突防線

2019-12-15 3,288條評論 4,276次閱讀 1440人點贊

佛羅倫薩在上賽季的時候是意甲聯賽中平局最多的球隊，平局場次多達17場，也正因為平局多佛羅倫薩這才得以保級成功，在新賽季佛羅倫薩的戰鬥力並沒有改變太多，沒有改變其實就是在變弱，現在佛羅倫薩經過了15輪聯賽之後只拿到了4勝4平7負的戰績，排名第12位，同比上賽季排名沒有太大的變化。

佛羅倫薩在主場的戰術上很猥瑣的，這種防守足以讓很多球隊都頭疼，本場佛羅倫薩去面對剛剛歐戰結束的米蘭還是有一些小優勢的。

國際米蘭本賽季開局以後爆發很不錯，不過國際米蘭在面對尤文圖斯和巴薩這種豪門球隊來說還是有著一些差距的，但是國際米蘭本賽季取分能力很穩，這讓國際米蘭在尤文圖斯失誤的情況下反超了尤文圖斯2分暫居榜首之位。

國際米蘭現在明面上優勢很大，但是上一場面對巴薩的輪換陣容居然還以1:2落敗了，也因此國際米蘭在歐戰小組賽沒有出線，國際米蘭士氣上必定會受到一些影響，本場比賽國際米蘭難拿3分。

本場比賽給出3.20 3.45 1.92的指數組合，指數上現在給出國際米蘭優勢方，後續指數上現在對於平負指數的支持力度也算不錯，不過現在亞路上在給開出0.5的情況下對於國際米蘭的支持力度並沒有那麼大，本場比賽佛羅倫薩更加值得去信任。
