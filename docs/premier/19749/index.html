絕殺！穆里尼奧為自己正名自己還是那個世界頂級的教練

2019-12-16 621條評論 2,246次閱讀 263人點贊

何塞·穆里尼奧聲稱，在毛里西奧·波切蒂諾的統治下，老熱刺會在對陣狼隊時屈服。

儘管最初通過盧卡斯·莫拉取得領先，但托特納姆在很長一段時間內都處於劣勢，在阿達瑪·特拉奧雷扳平比分後，托特納姆通過揚·沃頓根在最後時刻取得了勝利。這使得穆里尼奧在不到一個月前接替被解僱的波切蒂諾之後的五場比賽中贏了四場。穆里尼奧說：【「狼隊在角球和自由球上都有很強的統治力。我們贏了，因為我們在防守定位球上處理得很好。」】

穆里尼奧和熱刺慶祝戰勝狼隊

這場比賽局面上明顯是主場作戰的狼隊更佔優，但是兩位主教練都不敢輕易做出換人調整。在比分一比一的時候，這樣的比分是兩支隊伍都不接受的，穆里尼奧率先做出換人調整，他把開場八分鐘就取得進球的小盧卡斯換下，換上發定位球能力出色的埃里克森，事實證明穆里尼奧這個做法的無比正確性，正是埃里克森上場後的兩次定位球幫助熱刺實現最後的絕殺。

絕殺

穆里尼奧賽後表示：

【這是困難的。我們試圖把狼帶離他們的舒適區，但是他們有像火車一樣的球員。「當我看到迪爾擊中門柱時，我想如果我們現在不進球，我們就會遭殃，而我們做到了。」「最後，弗頓根的頭槌為我們贏得了勝利。「這場勝利是我們強大的心態造成的。真正意義上的團隊概念。只有「團隊」才能贏得這場比賽。比賽對我們來說很困難。】
