本納賽爾失空門萊奧兩中框 AC米蘭0-0薩索洛

天天要聞
2019-12-16

北京時間12月15日晚間22點，意甲聯賽第16輪繼續進行，AC米蘭主場迎戰薩索洛。特奧妙傳，恰爾汗奧盧失良機，此後特奧進球無效，本納賽爾錯失單；下半場易邊再戰，佩格洛屢獻神撲，萊奧替補兩度擊中門框，全場比賽雙方均無建樹。最終，主場作戰的AC米蘭0-0戰平薩索洛，雙方握手言和。

主場作戰的AC米蘭貴為聯賽豪門，近幾年表現不符合預期，受制於進攻線火力明顯不足，加上後方也屢屢丟球，本賽季的表現更顯尷尬，目前僅僅排在聯賽第十的位置，距離前四名差距相當大。而客隊薩索洛作為平民球隊，目前排在聯賽第14的位置，球隊的進攻火力十足，但後防很難有好的發揮，此役面對米蘭，薩索洛勢必加強防守，爭取更多的積分。

上半場比賽薩索洛率先開球，開場一分鐘，孔蒂長傳球給前場，皮亞特克帶球直殺禁區，一記爆射打在邊網上，同時越位在先。8分鐘，馬隆大禁區外圍送出直傳，跟進的卡普托近距離搶射打高了。

10分鐘，托利安防守中踢倒特奧，裁判出示黃牌警告。14分鐘，博納文圖拉大禁區前沿起腳遠射，門將撲救脫手，跟進的凱西近距離補射被沒收

23分鐘，馬隆挑傳球給前點，貝拉爾迪得球橫向擺脫，一記遠射被封堵。28分鐘，特奧禁區禁區左路橫敲中路，恰爾汗奧盧點球點附近掃射偏出。32分鐘，凱西禁區內停球後，順勢分球外圍，特奧得球一記世界波，皮球直進球門，可惜凱西手球在先，進球無效。

41分鐘，薩索洛開出戰術定位球，基里亞科普洛斯大禁區前沿遠射被擋出，此後本納賽爾獲得單刀機會，過掉出擊的門將，一記推射沒打上力量，錯失絕佳機會。

半場比賽結束，主場AC米蘭0-0戰平薩索洛。

下半場易邊再戰，47分鐘，貝拉爾迪分球給左路，博加停球後稍作調整，一記勁射打偏了。

52分鐘，蘇索開出左側角球，後點皮亞特克頭槌攻門，皮球被門將沒收。54分鐘，恰爾汗奧盧禁區線前沿停球後，直接射門可惜太正了。

60分鐘，博納文圖拉禁區線附近將球掃向門前，跟進的皮亞特克倒地鏟射踢空了。70分鐘，皮亞特克分球給身側，跟進的恰爾汗奧盧勁射被撲出，博納文圖拉補射同樣被擋出。

73分鐘，小孔蒂前場右路起腳傳中，禁區內皮亞特克頭槌攻門，可惜頂偏了。82分鐘，本納賽爾給前場左路，萊奧得球一記勁射，皮球擊中門框彈出。

87分鐘，特奧帶球推進，順勢分給身側，萊奧禁區弧頂一記低射，皮球擊中門框。

時階段，雙方互有攻守，94分鐘，特奧禁區左側勁射被封堵，此後帕奎塔再次射門同樣被擋出，兩隊始終沒有取得進球。

全場比賽結束，AC米蘭0-0薩索洛。

米蘭：99-小多納魯馬、12-孔蒂、13-羅馬尼奧利、22-穆薩基奧、19-特奧、79-凱西（54』39-帕奎塔）、4-本納賽爾、5-博納文圖拉、10-恰爾汗奧盧、8-蘇索（84』7-卡斯蒂略霍）、9-皮亞特克（77』17-拉斐爾-萊奧）

替補未登場：25-雷納、90-A-多納魯馬、2-卡拉布里亞、18-雷比奇、20-比格利亞、31-卡爾達拉、33-克魯尼奇、46-加比亞、68-里卡多-羅德里格斯

分類: 英超
