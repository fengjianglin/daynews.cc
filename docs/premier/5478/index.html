聚焦意甲！軍機處多維度關鍵情報帶您看：費拉拉vs熱那亞

2019-11-25 0條評論 13次閱讀 0人點贊

昨日總結

昨天我們投票選擇的是德甲 奧格斯堡vs柏林赫塔，最終奧格斯堡主場4-0大勝柏林赫塔，軍機大臣推薦的平局未能命中，但近期狀態不錯，大家值得關注。

在投票環節中，18%的網友看好奧格斯堡主場取勝，42%的網友看好兩隊打平，40%的網友則看柏林赫塔客場取勝。十一月中，投票首選打出的比例不高，今天是否會延續這一走勢呢？大家可以關注一下。

​

軍機處發起投票的262場比賽中，網友最看好的賽果打出125場，而不被看好的結果打出137場。

所以，軍機處建議：您在結合本場情報專家觀點的情況下，在比賽臨場前再次點進文章看一下我們網友投票的結果，或許會對您預測比賽有很好的幫助！

今天中國競彩開售8場比賽，除了免費軍情外，軍機處同時還會覆蓋中國競彩開出的其他賽事情報，助您玩賺所有比賽！下面帶來意甲 費拉拉vs熱那亞的比賽密報~~

​

狀 態

費拉拉斯帕爾五輪不勝

費拉拉斯帕爾在上輪聯賽中客場0-0戰平烏迪內斯，連續五輪不勝（2平3負），目前積8分排在聯賽第19位。而8分已經是球隊在意甲前12輪的第二高分，僅僅低於1954/55賽季（勝場按三分計算）。球隊本賽季兩場勝利全部來自主場，拿到了7個積分，佔比87.5%聯賽最高。

熱那亞連續15個意甲客場不勝

熱那亞在上輪意甲聯賽中客場0-0戰平那不勒斯，連續三輪不勝（1平2負），目前積9分排在聯賽第18位。球隊在過去的15個意甲客場都沒有贏球（7平8負），是當前意甲聯賽的客場最長不勝，球隊上次客場贏球還是在今年1月末客場3-1戰勝降級的恩波利。

交 鋒

費拉拉回歸意甲後不輸熱那亞

自2017年回歸意甲之後，費拉拉斯帕爾在過去四次與熱那亞的交手中保持不敗，不過除了首次交手主場1-0取勝之外過去三場都是1-1的平局。

​

攻 守

費拉拉進球全部來自主場

費拉拉斯帕爾在本賽季一共只有7球進賬聯賽最少，不過這7個進球全部來自主場，本賽季主場球隊只有上次0-1不敵桑普多利亞的比賽被零封。

熱那亞本賽季客場從未領先

在本輪比賽開始之前，熱那亞是本賽季失球最多的球隊，達到26球，其中17球來自客場聯賽最多。熱那亞是本賽季僅有的三支在客場從未取得過領先的球隊，另外兩支是費拉拉斯帕爾和桑普多利亞。

​

傷 停

費拉拉斯帕爾兩大中衛停賽

費拉拉斯帕爾兩名中後衛托莫維奇（11場）和喬內克（9場）在上輪聯賽中均領到了黃牌此役將累積黃牌自動停賽一場，他們的位置將由費利佩（6場）和薩拉（7場）頂替。此外球隊右前衛達歷山德羅（4場）和左前衛法雷斯則繼續因傷缺陣。

熱那亞頭號射手受傷

熱那亞頭號射手庫亞梅（11場5球）在國際比賽日期間受傷將至少缺席六個月的時間，此役將與前鋒薩納夫里亞（5場）、前腰薩波納拉（4場）和左後衛巴雷卡（6場）一同缺陣，球隊左後衛克里希托（7場2球）將回歸首發。

球 員

熱那亞主力右後衛故地重遊

本場比賽熱那亞主力右後衛吉利奧尼將故地重遊。在2016/17賽季吉利奧尼曾經效力於費拉拉斯帕爾，一共只代表球隊出戰了7場意乙聯賽349分鐘。

軍機大臣推薦：本場比賽軍機大臣看好費拉拉斯帕爾主場贏球，推薦：勝。
