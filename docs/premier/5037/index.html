歐冠杯 04:00 國際米蘭VS巴塞羅那 巴塞羅主力輪休 藍黑勢在必得

2019-12-09 0條評論 19次閱讀 0人點贊

2019-12-11 04:00 星期三

場地：朱塞佩·梅阿查球場

歐冠小組賽F組第6輪強強較量展開，藍黑軍團國際米蘭主場VS宇宙隊巴塞羅那客場，2支球隊戰績國際米蘭戰5場2勝1平2負積7分與大黃蜂多特蒙德並列第2名，巴塞羅那戰5場3勝1平1負積11分排名榜首，國際米蘭本場只有全勝確保出線，作為主教練安東尼奧·孔蒂也希望球隊取得好戰績，球隊現戰意高昂。上場逼平紅狼羅馬後，國際米蘭需一場勝利重回正軌。需要注意是體能問題，連續1周賽程讓國際米蘭體力受到下滑，安東尼奧·孔蒂到時臨場換人非常關鍵。

巴塞羅那情況比國際米蘭好很多，目前已出線情況下還能確保榜首。本場對巴塞羅那來說無關痛癢，埃內斯托·巴爾韋德輪換一些重要球員給他們足夠休息時間備戰聯賽，因後面要面對藍白軍團皇家社會、銀河戰艦皇家馬德里2支球隊，埃內斯托·巴爾韋德知道若聯賽這2場不能拿下的話自己帥位就不穩定。所以巴塞羅那取勝心不會很強烈，更多用來考察年輕球員與戰術演練。

國際米蘭眾將也紛紛說必須把巴塞羅那拿下勝利的，可看出戰勝之心多麼強媒體則爆消息巴塞羅那會輪換球員，隊內核心裡奧·梅西大概率不出場。巴塞羅那禁區外進球數10次，全歐洲最多的，國際米蘭這也進7球不差巴塞羅那很多，所以都需把防守注意在遠射上。

國際米蘭傷病多，阿萊克西斯·桑切斯、斯特凡諾·森西、萊昂德羅·巴雷拉、羅伯托·加利亞迪尼確定缺席，安東尼奧·坎德雷瓦待定，除去阿萊克西斯·桑切斯，其他都是中場球員。前面賽中我們可發現萊昂德羅·巴雷拉等人傷病讓國際米蘭中場控力下降。巴塞羅那也5球員缺席，但已確定出線，對巴塞羅那影響不大。

對安東尼奧·孔蒂來說，國際米蘭還有改善空間 球隊也爆出引進哈維·阿隆索、奧利維爾·吉魯等人，因一但像現在一樣遇傷病困擾，國際米蘭缺輪換球員，對1支球隊雙線戰來說，必須冬窗有新球員加盟。巴塞羅那現陣容有足夠深度，加球隊近幾個轉會窗口花費太多錢，東窗基本確定不太有大動作。

競彩推薦：和

亞盤推薦：巴塞羅那+0.75

大小推薦：大3

比分推薦：2：2、（防1：1）
