意甲今日頭版：CR7想要博格巴 國米收到恐嚇信

2019-11-17 0條評論 15次閱讀 0人點贊

直播吧11月17日訊 米蘭體育報、羅馬體育報、都靈體育報是義大利的三大體育報，今天我們來看看它們各自的頭版頭條都帶來了哪些內容。

《米蘭體育報》

1.「大師曼喬。」薩基稱讚國家隊教練：「國家隊給俱樂部做出了榜樣。」前米蘭以及義大利教練薩基表示：「有些球員在聯賽中感到吃力，在國家隊表現更好。」多納魯馬和維亞利來到威尼斯：「看望受災的人民。」

2.藍黑軍團緊張：國米和孔蒂收到子彈和恐嚇信。米蘭檢察院開始調查。

3.卡伊洛和特巴斯：不同意超級歐冠。都靈主席和西甲主席反對超級歐冠計劃。

4.皮亞尼奇受傷告別波黑，尤文周六客戰亞特蘭大可能無法使用波黑中場。

5.轉會市場：斯莫林，庫盧塞夫斯基和其他的租借球員，誰會走，誰會留。

《羅馬體育報》

1.「曼喬的勇氣。」國家隊主帥用自己的方式挑戰意甲。10連勝，一共使用52名球員。整個國家的人民重新愛上國家隊：曼奇尼就是這樣消除無緣世界盃的恥辱的。2020歐洲杯：德國、奧地利、克羅埃西亞和荷蘭也拿到入場券。義大利U21擊敗冰島：3-0，庫特羅內梅開二度。

2.孔蒂遭到恐嚇：一封帶著子彈和威脅信息的信指向國米教練，檢察院開始調查。

3.藍衣軍團代表來到聖馬爾科廣場：「威尼斯能渡過難關。」

《都靈體育報》

1.「CR7想要博格巴。」英國報紙沒有疑問：C羅已經對尤文高層表達了對法國中場回歸的熱情。

2.「加油威尼斯，我們跟你們在一起！」由足協主席格拉維納、領隊維亞利、球員多納魯馬組成的國家隊代表昨日來到聖馬爾科廣場。

3.針對孔蒂的恐嚇：國米總部收到威脅信。

4.雄雞貝洛蒂爆發：「我要在國家隊和都靈進球。」

5.歐青賽預選賽：U21義大利3-0冰島。

（我叫達芬奇）
