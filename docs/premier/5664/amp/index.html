瘋狂7球大殘殺！21歲小將1分鐘首秀進球，西班牙無壓力挺進歐洲杯

天天要聞
2019-11-16

北京時間11月16日凌晨3點45分，歐洲杯預選賽F組第9輪正式打響，本次由西班牙主場坐鎮對陣馬爾他的客場挑戰，還有羅馬尼亞主場坐鎮迎戰瑞典的客場挑戰。

西班牙上輪做客1-1逼平瑞典，羅德里戈在補時階段扳平比分，由於競爭對手羅馬尼亞同樣戰平，鬥牛士軍團得以提前兩輪鎖定出線資格。而馬爾他上輪做客0-1不敵法羅群島，小組賽遭遇7連敗，小組賽八輪戰罷僅攻入兩球，本場面對強大的西班牙恐怕要遭遇大比分失利。

羅馬尼亞近期5輪賽事取得3勝1平1負，羅馬尼亞近5輪主場賽事取得3勝1平1負，球隊近段歐預賽表現穩健，此役坐鎮主場面對小組排名第2的出線主要競爭對手羅馬，若能取勝將可以掌握出線主動權。而瑞典近期5輪賽事取得2勝2平1負，瑞典近5輪客場取得3勝1平1負，球隊近段歐預賽同樣表現不俗，球隊目前第2名的優勢只領先第3名1分，而最後一輪將在主場面對魚腩球隊，因此本場只要保持不敗就可能以小組第2名晉級。

西班牙7-0馬爾他

比賽時間開始，第23分鐘，西班牙開出右側角球造成禁區內混亂，莫拉塔接到赫拉德-莫雷諾頭槌後蹭，門前搶點將球打進，1-0。

第28分鐘，西班牙開出左側角球，拉莫斯頭槌攻門被撲，莫雷諾補射將球打進，但主裁判示意進球無效。

第41分鐘，西班牙再下一城，莫雷諾禁區前沿不停球送出直塞，卡索拉前插到禁區內推射破門，2-0。

隨著裁判一聲哨響，半場比賽結束，上半場西班牙掌握著場上的絕對主動權，並且分別由莫拉塔和卡索拉破門，目前在比分方面，西班牙主場2-0暫時領先馬爾他。

下半場易邊再戰，第60分鐘，西班牙這邊，帕科在前點的位置往後一蹭，將球給到了後點的保·托雷斯直接頭槌頂空門得手，3-0！

第62分鐘，經過了短短2分鐘，西班牙又取得進球了，前場右側的莫雷諾的橫敲給到禁區內，薩拉維亞跟進一腳射門，將球打入大門，4-0。

第68分鐘，西班牙佔據了全場絕對的主動權，蒂亞戈的直塞球，穿透了馬爾他整條防線，小將奧爾莫在禁區內冷靜過掉對方門將，將球送進空門，5-0！

第70分鐘，又是西班牙進攻配合，羅德里的斜長傳找到禁區左側貝爾納特，不停球直接給到對方禁區中路，越過對方門將，莫雷諾禁區中路跟進打空門得手！6-0，全場比賽變得毫無懸念了西班牙提前獲得了歐預賽出線晉級。

第84分鐘，西班牙徹底打瘋了，這是蒂亞戈的助攻給到了納瓦斯，在禁區右側外圍起腳遠射，使出一記電梯球，直接下墜到對方大門內，7-0！

隨著裁判一聲哨響，全場比賽結束，最終西班牙主場7-0血虐馬爾他，開起了屠殺模式主場將馬爾他打得體無完膚，順利拿下3分，在小組賽中以第一名的優勢力壓瑞典，提前一輪出線晉級2020年歐洲杯！

羅馬尼亞0-2瑞典

比賽時間開始，第18分鐘，瑞典這邊的進攻配合，福斯貝里在對方左側的底線位置一記巧妙地將球挑傳到對方禁區中路，馬庫斯·貝里後點插上高高躍起，直接頭槌破門，1-0。

第34分鐘，瑞典在下一城！馬庫斯·貝里在弧頂外一記直塞球，給到了誇伊森直接連人帶球突到對方禁區門前，起腳掃射破門，2-0！

隨著裁判一聲哨響，半場比賽結束，上半場瑞典隊在場上的表現佔據著絕對主動權，並且分別由博格和庫埃森破門，目前在比分方面，瑞典客場2-0暫時領先羅馬尼亞。

下半場易邊再戰，瑞典在上半場擁著2粒進球的優勢，啟動了鐵桶陣模式，瑞典球員們打得相對較為保守，稍稍往後撤了。而作為主場羅馬尼亞由於處於落後劣勢，加強全方位的進攻，來打穿瑞典的鐵桶陣！

隨著裁判一聲哨響，全場比賽結束，最終瑞典客場2-0擊敗了羅馬尼亞，憑藉著誇伊森和馬庫斯·貝里的進球，幫助瑞典拿下勝利，在歐預賽F組排行榜積18分，提前一輪小組賽出線晉級2020歐洲杯。
