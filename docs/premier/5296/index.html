利物浦賽前發布會 克洛普：太真實了 埃梅里下課反映教練命運

2019-11-29 0條評論 17次閱讀 0人點贊

當地時間周五，克洛普出席本輪英超主場對布萊頓的比賽的賽前發布會。會上他談到了前不久剛離開阿森納的教練埃梅里，還有球隊中場主力法比尼奧的傷情。

關於埃梅里下課

當我看到新聞時，我感到很驚訝，但這是真實的足球世界。對於（阿森納）發生了什麼，我沒有頭緒，但顯然老闆對表現不滿意，所以做了一個決定。這就是我們（作為主教練）的命運，只要你有合同在身。

我祝福他（埃梅里）一切順利，他是一個很出色的教練，這一點他在不同國家都已證明過。現在阿森納必須找到解決辦法，有很多教練可供選擇，包括一些年輕人，雖然可能有些缺乏經驗，但他們可以試試。

關於法比尼奧的傷（官方確認將因腳踝韌帶損傷缺陣）

這絕對是壞消息，對於傷缺時間我沒有100%把握，但可以確定的是他將無緣聖誕賽程，休養需要一定時間。他的傷是我賽後最關心的問題，我們當然要找人替代他，但還是希望他儘快回來。

關於中場人選

納比·凱塔沒問題，在那不勒斯賽後除了法比尼奧外大家都沒受傷。這對大家來說都是機會，亨德森、維納爾杜姆和拉拉納都踢過這個位置，我們也可以改變陣型，失去法比尼奧這樣的球員對我們而言是一大損失，但不是只有我們有傷員，這是我們需要面對的事。

關於火力點分散

不久前大家討論的還是我們如何依賴於三叉戟，我並不關心到底誰進球，只要球進球門我就很開心。當然，這（火力點分散）是好事，畢竟這可以讓我們沒那麼容易被看透，但我們本應進更多球才對。

關於布萊頓

波特工作做得很棒，他的球隊將會很難對付。布萊頓有一整個星期用來備戰，他們來到這裡會儘力搶分，這是100%可以肯定的。我們必須在三天前才剛和那不勒斯鏖戰的情況下應對他們的挑戰。

關於最近的戰績

我們沒有考慮13場比賽里贏了12場，只是100%地專註於下一個對手。現在我們只想儘可能展現自己的水平，布萊頓可不只有勇氣，他們訓練有素，在比賽中一直保持著競爭力。
