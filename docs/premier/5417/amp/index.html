本期勝負彩為歐冠第五輪，前四輪大冷都還沒有，本輪是否有大冷到來？本人推薦只代表個人初盤思路想法，請結合臨場及個人思路定奪，謹慎參考，紅黑勿怪勿噴。

03紅星VS拜仁

拜仁目前歐冠四戰全勝，小組已提前出線，本場取勝將提前鎖定小組第一，球半兩球盤下博個膽，目前兩隊在歐冠賽場中沒有平局，有條件者可以防個平哦。

04熱剌Vs奧林匹

熱刺本場比賽是穆里尼奧入主白鹿巷的第二場賽事，第一場歐冠賽事，看好穆里尼奧歐冠首場賽事取勝。埃里克森上場聯賽替補，傳聞埃里克森不續約的話，將會淪為替補，或者進不了大名單。是否是穆里尼奧的煙霧彈呢？

05亞特蘭大VS薩格勒

本場比賽將在中立場踢，各機構都開出一球盤口。以亞特蘭大本賽季的尿性，內戰行外戰欠缺，防冷分勝負了。

06曼城VS礦工

實力盤下，主勝。

09亨克VS薩爾茨紅牛

亨克比甲球隊目前聯賽不盡如意，十六輪後排第九，進失球及勝負相當，薩紅牛本賽季相當強勢火力大開，各項賽事場均進球近三，個人看客不吸，本場客勝有博，求穩防平

10利物浦VS那不勒斯

利物浦做客那不勒斯失利而回，本場坐鎮主場，有復仇慾望，利物浦的主場一向強勢。沒有幾個球隊能從安菲爾徳球場全身而退，我想那不勒斯小組賽中也很難雙殺利物浦，那不勒斯如果能從利物浦主場帶走一分，應該就算成功的了，主勝有博的餘地，主不敗

11巴薩Vs多特

本賽季巴薩有點不穩定，各項賽事目前已經輸了三場，歐冠賽場上不敗。多特的青春風暴和高位逼搶應該是有機會給巴薩製造出麻煩，個人看分勝負。走兩頭。

12斯拉維VS國米

本賽季國米在孔蒂的帶領下。還是比較穩定，我想國米做客斯拉維拿走分數應該是沒有問題的，客不敗。

13萊比錫VS本菲卡

萊比錫各項賽事五連勝。本菲卡進十場賽事沒有平局，拐點是否到了。防平過關。主不敗
