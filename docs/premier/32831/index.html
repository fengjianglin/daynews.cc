大揭秘！為什麼C羅離開皇馬加盟尤文？稅務問題或加薪成主因

2019-12-23 434條評論 1,992次閱讀 1022人點贊

2018年，C羅從皇家馬德里轉會到尤文圖斯，這一消息震驚了世界足壇，而且似乎來得非常突然。事實上，考慮到C羅本人就在一年前表示他想在伯納烏結束自己的職業生涯，這讓許多偶然的觀察者感到驚訝。

這位葡萄牙隊長成為了馬德里在歐冠中統治地位的基石，激勵他們獲得了四次冠軍，他以450粒進球的俱樂部進球紀錄載入史冊。那麼，考慮到球場上的一切都很順利，是什麼讓C羅離開西甲來到了意甲和都靈山呢?

為什麼C羅離開皇馬加盟尤文?

C羅聲稱他離開皇家馬德里去了尤文圖斯，因為他覺得他在白衣軍團主席弗洛倫蒂諾·佩雷斯的眼裡「不再是不可或缺的」。他補充道，尤文圖斯表現出了對他的渴望，他覺得自己比皇馬更強大，同時指出齊達內的離開也起到了一定的作用。

2018年，C羅在接受法國足球採訪時表示:「我覺得在俱樂部里，尤其是在主席，我不再像一開始那樣受到重視了。最初的四五年，最初的四五年，我感覺自己就像明星。現在主席看著我，彷彿在說，如果你明白我的意思，我就不再是不可或缺的了。這就是我想離開的原因。」

當然，C羅所陳述的離開西班牙首都的理由不一定能說明一切。在他即將離開俱樂部的時候，有報道稱，在皇馬贏得2018年歐洲冠軍聯賽冠軍後，C羅希望加薪。前皇馬主席卡爾德隆說，內馬爾的到來，據說也是前曼聯球星做出決定的原因之一。

這位前皇馬主席在接受特立尼達維拉電台採訪時說:「C羅在贏得歐冠後要求加薪，但弗洛倫蒂諾拒絕了。C羅也很生氣，因為他聽說皇馬願意出很多錢買內馬爾。弗洛倫蒂諾告訴他，如果他帶來1億歐元的支票，他就會放他走，事情就是這樣。」

然而，C羅否認他的轉會是由於報酬糾紛，他說:「如果一切都是為了錢，我就會去中國，在那裡我的收入是在尤文或皇馬的5倍。」我來到尤文不是為了錢。我在馬德里掙得一樣多，如果不是更多的話。」

C羅是否因為稅務問題離開皇馬?

關於C羅轉會尤文圖斯的另一個假設是，他是西班牙一個備受關注的稅務法庭案件的焦點人物。該案件於2019年初結案，羅納爾多支付了1900萬歐元的罰款。

當對他的起訴在2017年開始時，西班牙的報道稱，葡萄牙人曾因他的待遇威脅要離開這個國家，很明顯，這個問題困擾著他。值得注意的是，2018年初，他在社交媒體上抨擊「假新聞」，指責詆毀他的人試圖「破壞」他在職業生涯中經歷的「美好時刻」。

然而，皇馬主席佩雷斯隨後否認了這名球星離開馬德里的說法。他在2019年5月對昂達·塞羅說:「有傳言說他因為稅務問題想要離開，但事實並非如此。他想要改變，想要改善他的合同，所以他在尤文圖斯找到了機會，每個人都很高興。」
