21年前今天AC米蘭德比2比2國米 巨星大PK 南看台巨型TIFO震撼全場

1998年11月8日，也就是21年前的今天，1998-1999賽季意甲聯賽上演米蘭德比，紅黑軍團AC米蘭坐鎮聖西羅主場迎戰同城兄弟國際米蘭。當時的AC米蘭剛剛從米蘭王朝之後的低谷中走出，國際米蘭則因為莫拉蒂的瘋狂投入招攬了多名巨星，可惜無法形成一個整體。

比賽之前，聖西羅南看台AC米蘭球迷展示的巨型TIFO震撼全場！

兩隊主帥扎切羅尼和路易吉-西蒙尼都排出了最強陣容，全力一拼！看一下兩隊星光熠熠的首發和替補出場球員名單。
