西甲繼續強勢，還是英超、意甲重回巔峰，歐冠八分之一決賽見分曉

2019-12-17 671條評論 1,682次閱讀 293人點贊

昨天結束的19/20賽季歐冠八分之一決賽抽籤儀式上，西甲豪門皇家馬德里遭遇英超衛冕冠軍曼城，而另一支來自西班牙首都的球隊馬德里競技，則和歐冠衛冕冠軍利物浦相遇。同樣來自西甲的巴塞羅那則遇到剛剛換帥的那不勒斯，歐冠常客巴倫西亞和歐冠黑馬亞特蘭大遭遇。西甲四強對戰英超、意甲聯軍，這無疑是首輪歐冠淘汰賽的最大看點。

在過去的20年間，歷年歐冠冠軍如下：

皇家馬德里（2000）

拜仁慕尼黑（2001）

皇家馬德里（2002）

AC米蘭（2003）

波爾圖（2004）

利物浦（2005）

巴塞羅那（2006）

AC米蘭（2007）

曼聯（2008）

巴塞羅那（2009）

國際米蘭（2010）

巴塞羅那（2011）

切爾西（2012）

拜仁慕尼黑（2013）

皇家馬德里（2014）

巴塞羅那（2015）

皇家馬德里（2016）

皇家馬德里（2017）

皇家馬德里（2018）

利物浦（2019）

可以看出在過去20年間，皇馬共拿到6次歐冠，巴塞羅那拿到4次歐冠，兩隻來自西甲的球隊一共10次站在歐洲之巔，和其他球隊的總和持平，而利物浦、曼聯、切爾西組成的英超代表團共拿到4次歐冠，米蘭雙雄代表意甲拿到3次，德甲獨苗拜仁拿到2次，剩餘一次屬於穆里尼奧的波爾圖，屬於足球世界裡神奇的2004年（歐洲杯希臘神話）。不得不承認，進入21世紀以來世界足壇的主流是西班牙風格，統治歐洲足壇的是西甲聯賽，C羅和梅西兩位天之驕子把最好的青春歲月留在了西甲，便是最好的證明。

然而隨著傳控足球日漸衰落，巴塞羅那也失去了往日的統治力，C羅離隊後攻擊力大打折扣的皇馬，即便引入多位強援也未能恢復歐冠三連時期的霸氣十足。今年6月利物浦與熱刺會師歐冠決賽，兩隻英超球隊半決賽分別踩著傳控足球鼻祖巴塞羅那，以及傳控足球小師弟阿賈克斯過關，就證明了西班牙風格的足球，已經在逐漸被淘汰。利物浦式的瘋狂逼搶、快速反擊才是足球世界裡長盛不衰的秘訣。

本賽季歐冠八分之一決賽，皇馬將遭遇藍月亮曼城，雖然過往的交鋒中，銀河戰艦佔據壓倒性優勢。可如今的曼城早已不比當年，而且對方陣中還有一個皇馬剋星–瓜迪奧拉。要說利物浦最怕哪支球隊，還真就是馬德里競技，同樣充滿激情的教練，同樣瘋狗式的踢法，利物浦的衛冕之路，想必不會輕鬆。

那不勒斯剛剛換掉安切洛蒂，巴塞羅那的老對手加圖索上任，向來熱衷暴力美學的義大利悍將，能否為新東家帶來驚喜？被羅馬逆轉的場景還歷歷在目，遇上類似的對手，巴塞羅那能否洗刷恥辱，重新拿出宇宙隊的風采？巴倫西亞對陣亞特蘭大，可以算是本輪8場對決中的菜雞互啄，即便哪一支球隊晉級，未來的路都會非常艱險。

是西甲強勢依舊，還是英超再接再厲，又或是意甲重新登頂，本賽季歐冠八分之一決賽，我們拭目以待。
