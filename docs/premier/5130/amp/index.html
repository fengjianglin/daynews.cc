西媒：皇馬準備現金加新星再求博格巴！梅西：退役的那天正在迫近

天天要聞
2019-12-06

西媒El Desmarque消息，皇馬準備報價4200萬英鎊加維尼修斯來求購曼聯中場博格巴。

皇馬整個夏天都在嘗試收購博格巴，不過曼聯的立場一直十分堅定，那就是他們沒有出售這名中場的計劃，然而，看起來這並沒有打消齊達內的購人熱情，並且報道稱皇馬主帥甚至經常催促高層幫助他簽下自己的心頭愛，而馬德里方面似乎也已經制定出了一套新的引援計劃。

El Desmarque表示，在一月轉會窗開啟之後，皇馬將為曼聯提交一份球員加4200萬英鎊現金的報價，作為籌碼的球員則正是本賽季在伯納烏失寵的維尼修斯，不過，這家西媒指出曼聯希望得到雙倍的現金外加維尼修斯，這個還價恐怕很難令皇馬點頭同意。

19歲的維尼修斯在2018年從弗拉門戈加盟了皇馬，並且隨後在42場比賽中打進5球，不過本賽季，這名巴西新星似乎並不受到齊達內的重用。

其實不是可靠消息，皇馬或者說齊祖想要博格巴是真的，但加上維尼修斯是不現實的，皇馬方面的消息源表示，雖然維尼修斯本賽季沒有太多機會，但皇馬依然看好他，甚至不願將其外租，更不要說當籌碼送走了……

馬卡報消息，巴薩人最擔心的時刻可以說就是梅西做出退役決定的那天，而這名足壇巨星最近則親口談到了這個話題。

在奪得職業生涯第6座金球獎之後，梅西就自己的未來發表了評論，巴薩球星當時表示：「我很清楚自己多大年齡了，所以這種時刻也就更加令人享受，因為退役的時刻已經在逐漸迫近，你很難再重複這樣的輝煌。

我希望，並且願上帝保佑，我還可以踢很多年，不過我32歲了，正在走向33歲的路上，所以正如我之前所說，一切取決於我身體上的感覺怎麼樣。

今天我個人感覺比以往任何時候都好，希望我可以繼續將其延續下去吧。」

從梅西的話中，感覺他確實在考慮退役的問題，但他也說了，自己現在身體上感覺很好，所以幾年內梅西應該不會退役的。

而且巴托梅烏最近也說了，梅西至少還會在巴薩效力幾年的。

伊布最近暗示他將重回意甲，並且會加盟一支「必須再次走向勝利之路」的球隊。

在確認離開洛杉磯銀河之後，38歲的伊布和米蘭密切聯繫在一起，而目前來看，瑞典人的下家的確很可能會是一家意甲球隊，對此，伊布本人也談到了他的未來。

在接受GQ義大利版採訪時，伊布說：「我會加盟一支必須再次走向勝利之路的球隊，一家必須重拾其輝煌歷史的球隊，這傢俱樂部將尋求挑戰所有的對手。在義大利見吧。

對一名球員來講，事情不是選個球隊那麼簡單，還有其他方面的因素要適合才行，甚至我的家庭也要感興趣。」

從伊布的話里，能看出他下家應該是意甲，並且非常有可能是米蘭，博洛尼亞也對伊布感興趣，但問題是球員一家不一定願意去那生活，而且米蘭畢竟底蘊和潛力要大得多。
