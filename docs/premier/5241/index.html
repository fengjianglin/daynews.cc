布馮生涯罕見一幕：黃油手害尤文丟球！41歲高齡首發，滿臉自責

2019-12-01 0條評論 17次閱讀 0人點贊

北京時間12月1日，意甲焦點戰，尤文主場迎戰薩索洛，在上半場比賽中，埃姆雷詹失良機，但很快博努奇遠射打破僵局，此後，薩索羅快速扳平比分，前45分鐘，C羅的表現平平，並沒有在進攻端體現太大的作用，半場比賽結束，主場作戰的尤文1-1戰平薩索洛。帶著這樣的比分，進入下半場之後，尤文很快又犯錯了。

而且這一次，是尤文後防非常恥辱的連續失誤，先是，德里赫特在大禁區弧頂，一腳倉促橫穿解圍，正好把球送到了對手腳下，卡普托拿球之後，在禁區前沿發力抽射，不過，由於沒有踢准部位，此球有些綿軟無力，不過，此時，令人萬萬沒想到的是，在德里赫特失誤之後，站在門線上的老將布馮，也出現了黃油手的情況。

41歲的尤文門神，面對隊友的失誤橫傳，突然有些倉皇失措，退守近門柱防守有些踉蹌，而當薩索羅前鋒卡普托禁區線上低平抽射之後，布馮倒地側撲，本該是一次穩拿的撲救，可惜，出現了意外一幕，皮球從布馮的腋下變向穿過，彈地之後滾入了門線之內，就這樣，尤文1-2落後了！

丟球之後，布馮表情頗為尷尬，看得出來，對於這樣的失誤，41歲的老門神有些不敢相信。值得一提的是，此役尤文面對薩索羅，有一個看點，就是兩隊的門將對比，一便是意甲最年輕門將圖拉蒂（18歲），一邊是最年長門將布馮（41歲），他們之間年紀相差23歲，要知道，2001年，布馮轉會至尤文圖斯的時候，那年圖拉蒂才出生。很可惜，這兩人此役都沒能實現零封。
