巨龍騰飛：穆里尼奧率領波爾圖登上歐洲之巔，從此一飛衝天。

2019-11-21 0條評論 18次閱讀 0人點贊

穆里尼奧真正執教生涯開始於葡超的小球會——利亞拿足球隊。

其實在此之前他有更高的跳板，那就是葡超豪門本菲卡，然而因為與主席和董事會意見不合，憤然辭職，僅僅執教本菲卡9場比賽。

不過到了利亞拿之後，穆帥得到了充分的支持，在他的鞭策下，利亞拿歷史性的拿到了球隊第五名的歷史最好成績。

正是因為在利亞拿優異發揮，穆里尼奧得到了波爾圖的青睞，對於有仇必報的穆帥來說，沒有什麼比加入死敵復仇更痛快的了，作為本菲卡的死對頭，39歲的穆帥果斷接過了波爾圖的教鞭。

當時波爾圖已經三年沒有拿到冠軍了，誰都沒有對「菜鳥」穆里尼奧抱任何信心，然而讓人沒想到的是，第一個賽季，他就率領波爾圖拿下三冠：葡超冠軍、葡萄牙杯冠軍、歐洲聯盟杯冠軍。

這是波爾圖第一次獲得聯盟杯，當時決賽在塞維利亞進行，波爾圖的決賽對手是凱爾特人。

當時波爾圖球迷有他們最好的中場——德科。「葡萄牙的齊達內」是歐洲足壇給德科的稱號，正是他的兩記世界級的助攻，讓球隊90分鐘內戰成2：2，當時歐足聯推出了銀球制勝，隨著加時賽波爾圖前鋒德雷攻入銀球，幫助球隊歷史上首次獲得聯盟杯冠軍。

這也讓穆里尼奧名聲大震，不知道本菲卡的董事會看到此情此景作何感想。

不過穆帥不會去想這些，因為他和他的球隊有更高的追求和野心——歐冠。

在小組賽中，波爾圖沒受到太大考驗，在戰勝貝爾格萊德遊記後，提前一輪進入淘汰賽。再結合前一年剛拿到聯盟杯，當時媒體認為，波爾圖將是本年度歐洲賽場最大的一次黑馬。

穆帥在訓練上嚴格要求球員的跑位，在立足防守的基礎上採取高位逼搶，讓對手出錯從而撕破對方的陣型破門得分，球員們也做到了穆帥的要求。

在歐冠1/8決賽上，他們遇到了當時的英超霸主曼聯。人們對波爾圖晉級普遍看低，開場的走勢也證明了這點，第14分鐘，曼聯就在客場1：0領先。不過波爾圖的麥卡錫梅開二度，幫助波爾圖在主場有驚無險的取勝。

曼聯帶著一個客場進球回到老特拉福德，而且又是曼聯先取得進球，這個比分一直保持到常規時間90分鐘，所有人都以為曼聯會成功進入下一輪比賽。然而沒想到91分鐘遭到科斯蒂尼亞絕殺，令人印象深刻的是，穆里尼奧在場邊狂奔50米，激動的心情不言而喻。

在通往決賽之前，他們還有一個攔路虎，那就是超級拉科！2004年的歐足聯積分排名拉科位居第5位，超級拉科以華麗的打法被大量球迷所欣賞。

這賽季歐冠最經典的一場比賽就是拉科和AC米蘭奉獻的，相信球迷也都知道拉科客場1：4完敗AC米蘭，然而回到主場里亞索球場，拉科4：0逆轉翻盤將AC米蘭淘汰出局。

波爾圖也面臨非常嚴峻的考驗，他們主場，被拉科0比0逼平，基於拉科與AC米蘭的表現，大部分人對波爾圖的客場之旅並不看好。但是這就是命，德科在禁區被鏟倒，裁判判罰點球，德雷一蹴而就，這也結束了拉科將近兩年歐冠主場不失球的記錄。

波爾圖連續兩年接連挺進聯盟杯和歐冠決賽。

進入決賽是球隊是本賽季冠軍杯的兩匹黑馬，波爾圖和摩納哥。這也是十三年來第一次決賽的兩隻球隊不是來自英、德、意、西聯賽的球隊。

波爾圖將高位逼搶和快速反擊發揮到極致，乾淨利落的以3：0將摩納哥挑於馬下。

兩年時間內先後率領球隊奪得聯盟杯和歐冠獎盃，穆里尼奧展現了他的魔力。

在執教波爾圖過程中，穆帥總是人們關注的焦點，桀驁不馴的他自然散發著魅力，這也使得他的球員願意跟隨他，在穆帥的領導下，波爾圖空前的統一團結。

當然對於魔力鳥來說，波爾圖滿足不了他的野心，他的眼界只會更高……
