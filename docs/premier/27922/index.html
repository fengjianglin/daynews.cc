3歐冠+四國聯賽冠軍，安切洛蒂為何選擇埃弗頓？利物浦或是關鍵

2019-12-22 1,466條評論 4,887次閱讀 1491人點贊

昨晚，英超又傳來重磅消息，此前在那不勒斯下課的安切洛蒂正是執教埃弗頓！一位是世界頂級教頭，一個是沒落英超豪門，他們是如何做到聯姻的呢？沒有一個教練不想展示自己的偉大，同樣的，沒有一支俱樂部甘於平庸，而利物浦的存在更是讓老帥多了几絲興趣！

執教生涯20冠，安切洛蒂名聲在外

執教足跡遍布五大聯賽，安切洛蒂早已名聲在外。早在1999賽季便率領尤文圖斯奪得國際足聯托托杯（為未能獲得歐洲主流聯賽歐冠或聯盟杯參賽資格的俱樂部提供，於2009年廢止）冠軍，此後進入21世界後安切洛蒂迎來大爆發。

自2003年開始，AC米蘭與安切洛蒂一同創造了一個米蘭盛世，在此期間安切洛蒂在2003-04、2007-08賽季兩度帶領AC米蘭問鼎歐冠，此外在2003-04賽季帶領AC米蘭奪得意甲聯賽冠軍。除此之外世俱杯、義大利超級盃以及義大利杯也都有染指，安切洛蒂在AC米蘭總共為球隊帶來6個冠軍。2009年安切洛蒂執教切爾西，為球隊奪得英超、足總杯和社區盾杯。此後安切洛蒂又帶領巴黎聖日耳曼奪得法甲聯賽冠軍。2013-14賽季安切洛蒂輾轉西甲帶領皇馬奪得國王杯、歐冠、歐洲超級盃和世俱杯。在挑戰德甲的日子裡安切洛蒂同樣帶領拜仁慕尼黑奪得德甲、和德國超級盃冠軍！

安切洛蒂執教能力毋庸置疑，因材施教是他的看家本領，如果在西甲也能聯賽稱雄那安帥將會做到五大聯賽大滿貫！但球迷不會因為缺少一個西甲獎盃忽視安帥的成功！

老牌勁旅求新生，簽約安帥第一步

盛世英超名不虛傳，安切洛蒂的到來也讓英超又多了一位世界名帥，以安切洛蒂的實力接受世界上任何一支豪門都沒有問題為何卻單單選擇在英超都稱不上強勢的埃弗頓呢？

作為英超老牌勁旅，在英超聯賽中唯唯諾諾的埃弗頓祖上也闊過！九次問鼎英格蘭頂級聯賽冠軍，五次捧起足總杯冠軍，一次歐洲優勝者杯奪魁，只不過後期因為沒有搭上金元足球的快車逐漸的落寞下來。埃弗頓想要崛起瘋狂砸錢不現實，尋找一位世界名帥為他們制定復興計劃才是行之有效的辦法，因此他們選擇了安切洛蒂。對與已經吃遍歐洲豪門的安切洛蒂來說選擇一支豪門並不難，但挑戰性相對較較小，埃弗頓身處英超情況則要複雜的多，如果能帶領埃弗頓完成復興那安切洛蒂無疑會更加成功！更大的挑戰也讓老帥最終決定拿起埃弗頓的教鞭，一心想要重塑輝煌的埃弗頓會給足安切洛蒂權利，這一點也是彌足可貴的！

不可忽視利物浦，老冤家再聚首

再優秀的教練都會有成為背景的一刻，安切洛蒂自然也不例外，對與安切洛蒂來說一生之痛便是2005年的伊斯坦布爾之夜！而對手正是利物浦！

2005年歐冠決賽安切洛蒂所帶領的AC米蘭在3-0領先的情況下被貝帥率領利物浦頑強逆轉成為歐冠歷史上最強的背影，而一年之後的歐冠決賽又是領支球隊聚首，此次安切洛蒂沒有放鬆2-1拿下利物浦登頂歐洲。十多年來，安切洛蒂與利物浦開啟了一場曠日持久的追殺之戰，自伊斯坦布爾之夜開始安切洛蒂總共與利物浦交手12次，在此期間安切洛蒂7勝1平兩負，在此前的歐冠小組賽中安帥所率領的那不勒斯兩回合一勝一平對利物浦保持不敗！不得不說，安切洛蒂正是利物浦的剋星！

當然，埃弗頓與利物浦的恩怨也不得不提，默西塞郡兩支球隊雖然利物浦高高在上，但埃弗頓才是這個地區的老大哥，多年德比恩怨並沒有消失在歷史長河中，拿下安切洛蒂也算是深深地刺激了一把利物浦！

安切洛蒂想要成功不容易，利物浦想要繼續保持強勢也不容易，希望早日看到兩支球隊交手，也期待安切洛蒂能給我們帶來驚喜！
