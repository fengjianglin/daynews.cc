「專欄」梁利鋒：對於歐冠，國際米蘭是拼一發還是放一下？

2019-11-28 0條評論 15次閱讀 0人點贊

歐冠小組賽進入決定去留的第5輪，一定有好戲，但也要關注戰意……

0 11 布拉格斯拉維亞VS國際米蘭

前4輪戰罷，國際米蘭4分排第3，布拉格斯拉維亞2分小組墊底，前面的巴薩和多特蒙德分別是8分和7分。從賽程安排上看，國際米蘭如果最後兩輪取得全勝，就能獲得出線權；而布拉格斯拉維亞如果想進淘汰賽，不僅要確保後兩場全勝，還要指望巴薩本輪擊敗多特蒙德。對比晉級形勢，國際米蘭更好。

不過戰意方面，國際米蘭對於歐冠淘汰賽資格，可能不是特別看重——小組第3獲得歐聯杯淘汰賽參賽權，更如雞肋。向來不擅歐冠的藍黑主帥孔蒂不可能公開表示「歐冠可棄」，但目前意甲的爭奪形勢，對國際米蘭是比較有利的。先拼聯賽冠軍，再圖歐冠復興，也是俱樂部對孔蒂的期待。之前兩個歐冠客場，國際米蘭都是先進球的情況下被逆轉，比賽踢得不差，但戰略目的性不強，沒有表現出「一定要出線」的信念。

布拉格斯拉維亞本賽季歐冠整體表現可圈可點，特爾皮紹夫斯基教練的搶逼圍戰術面對強者非常奏效，主場對多特蒙德和巴薩都是佔據場面優勢，尤其對西甲冠軍，如果不是特爾斯特根「開掛」，捷克球隊完全應該拿分、甚至取勝。本場對國際米蘭，斯拉維亞必定要借主場之利為小組第3全力拚殺，主力中衛霍沃爾卡遭受重傷是一大損失，但後腰紹切克活動能力極強，可確保防守體系有條不紊。

國際米蘭目前傷員較多，尤其中場，森西和加利亞爾迪尼還不能復出，幾天前對都靈又折了巴雷拉(膝蓋受傷)——布羅佐維奇更是超負荷運轉，隨時可能倒下。孔蒂賽前將自己的球隊形容為「新軍」，也是放低身段、為球隊減壓的態度。斯拉維亞前兩個歐冠主場全敗，國際米蘭前兩個歐冠客場全敗，本場定有一尷尬被終結。競彩指數2-3-2配置，主勝還在不斷下調，「強讓」的國際米蘭取勝難度漸漸增大。如果臨場無劇烈指數變化，斯拉維亞不敗應該是十拿九穩。
