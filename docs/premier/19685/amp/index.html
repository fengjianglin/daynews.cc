歐冠16強分組已定！多場重磅對決，一場比賽將會非常火爆

天天要聞
2019-12-17

北京時間12月16日，2019-20賽季歐冠聯賽16強淘汰賽的抽籤儀式正式落幕，衛冕冠軍利物浦將會面對馬德里競技，皇家馬德里則是抽中了上賽季的英超冠軍曼城，多特蒙德將會碰上老朋友圖赫爾率領的巴黎聖日耳曼。

除此之外，巴塞羅那將會面對那不勒斯，拜仁慕尼黑vs切爾西，瓦倫西亞vs亞特蘭大，尤文圖斯與熱刺則是抽中了上上籤，他們將會分別面對里昂與RB萊比錫。淘汰賽首回合的第一場比賽將會在2020年2月19日開始。

作為本次抽籤抽出的最重量級的對決，「藍月亮」曼城將率先做客伯納烏挑戰皇家馬德里。這也是兩名世界名帥，齊達內與瓜迪奧拉在教練生涯中首次相遇。瓜迪奧拉在執教巴薩時，曾多次擊敗皇馬，沒有任何一位教練擊敗皇馬的場次比瓜迪奧拉多。而那時齊達內還並沒有開啟玄學執教，2015-16年賽季，歐冠半決賽，齊達內率領的皇馬，正是擊敗了曼城，才晉級的決賽。皇馬歷史中在歐冠賽場共4次遇到曼城，皇馬2勝2平保持不敗。

抽籤儀式後，皇馬公關總監布特拉格諾對曼城，表達了自己的看法：「近些年來，曼城投了非常多的錢，但我們是皇家馬德里，可以對抗任何對手，尤其是在我們球迷的支持下。」兩隊的首回合較量，將在2020年2月27日進行。

而另一場焦點大戰，多特蒙德將會碰上法甲豪門巴黎聖日耳曼。目前大巴黎的主教練圖赫爾，在執教巴黎前，圖赫爾的老東家便是多特蒙德。圖赫爾在執教多特蒙德時，帶領球隊獲得了1次德國杯冠軍。然而圖赫爾在離開前，與多特蒙德的主管佐爾克和高層瓦茨克均產生了無可調和的矛盾，圖赫爾和當時的多特首席球探米斯林塔特不和，而佐爾克是米斯特林塔特的支持者。

而本次抽籤最幸運的就是由C羅領銜的尤文圖斯了，他們將會面對法甲里昂。本賽季里昂開局不利，位列法甲中游。兩隊曾在2016-17賽季小組賽中相遇，當時尤文圖斯取得了1勝1平的不敗戰績。上一輪聯賽，德佩與雷內-阿德萊德均受到了前十字韌帶撕裂的重傷，前者更是有可能會缺席明年的歐洲杯，里昂的實力不足以威脅到尤文圖斯，所以尤文基本可以鎖定一個晉級名額。
