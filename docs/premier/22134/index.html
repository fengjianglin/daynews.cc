不只會進球！C羅戲耍一年前歐冠夢魘對手，一記妙傳亮瞎眼

2019-12-19 570條評論 2,164次閱讀 132人點贊

C羅雖然轉投加盟尤文，但並未發生什麼斷崖式的下滑，雖然沒有在皇馬時期那樣進球如麻，但尤文的助攻手少也是其中一個原因，但上賽季的C羅還是拿到了尤文隊內的最佳射手，但上賽季C羅在尤文的歐冠小組賽首輪卻遭遇了夢魘，在與瓦倫西亞的比賽中，C羅因為摸頭殺被主裁直接紅牌罰下，而時隔一年的時間，穆里略也過得並不好，如今已經是桑普多利亞的一員，而C羅也完成了對穆里略的「報復」。

穆里略1年前是C羅夢魘

18/19賽季的歐冠小組賽首輪，是C羅代表尤文的首場歐冠比賽，讓尤文可以花費1億歐的價格，買進33歲的C羅，尤文方面自然也是看中了C羅在歐冠賽場上的實力，C羅自然也心領神會，並希望通過自己在歐冠賽場上的表現，回報尤文高層對自己的信任，但歐冠小組賽首輪C羅卻遭遇迎頭一盆涼水，因為C羅摸摔倒在地的穆里略頭的一個動作，讓當值主裁直接出示紅牌將C羅罰下場。

C羅自然覺得很冤，因為自己並沒有做什麼出格的動作，而媒體方面也在解讀C羅到底在摸頭的動作的時候，有沒有揪穆里略的頭髮，不過C羅最終還是遭遇了禁賽1場的處罰，而下場前C羅流下的熱淚也表明，自己真的沒有想過犯規，而裁判的舉動也讓葡萄牙人接受不了，隊友紛紛上前安慰，而副主席內德維德也親自來到球員通道，安撫情緒幾近崩潰的C羅，希望讓其情緒得到緩解。

1年後C羅戲耍穆里略

1年之後的意甲賽場上，C羅雖然因為膝蓋不適遭遇了踢得並不如意的11月，但隨著C羅的傷勢逐漸恢復，在過去尤文連續4場比賽中，C羅都取得了進球，甚至與烏迪內斯的比賽中，C羅半場就上演了梅開二度的好戲，險些上演自己在尤文意甲生涯的第一個帽子戲法，不過與桑普多利亞的比賽，C羅卻直接對位遇到了自己的夢魘穆里略，這個上賽季還在瓦倫西亞的球員。

穆里略的這一年過得也是猶如過山車，在瓦倫西亞的表現很不錯，甚至還讓C羅在歐冠拿到了紅牌，穆里略在19年冬窗被巴薩租借到球隊中，但雷聲大雨點小，穆里略在巴薩只有4次出場，在新賽季開始前穆里略為了獲得更多的出場機會，選擇來到了意甲租借並附帶強制買斷條款，成為了桑普多利亞陣中的一員，比賽下半場第47分鐘，穆里略在單防C羅的過程中被C羅輕鬆的過掉，雖然隨後C羅的傳球，沒有和伊瓜因形成連線，但不得不說過去了1年的時間，C羅還是那個強勁的C羅，而穆里略已經不再是那個可以防得C羅發脾氣的穆里略了。

C羅秒傳亮瞎眼

除了打入那粒震驚桑普多利亞主場球迷的霸氣頭槌外，C羅在進攻中也充當了傳球手的角色，這也是薩里的體系中最缺少的，上半場比賽第41分鐘，C羅在大禁區左側拿球，在連續晃動後，挑傳禁區，而在這條線路上有伊瓜因和迪巴拉兩人，但兩人的反應還是稍微慢了一些，都沒有碰到皮球，如果有一個人碰到皮球，皮球都會被撞進球門，面對隊友浪費的機會，C羅也只能無奈地笑了笑。

但薩里在與桑普多利亞的比賽中，依然沿用了上輪意甲的陣容，在鋒線位置上依然派出了C羅、迪巴拉和伊瓜因3人一同首發的陣型，顯然薩里雖然認為這種踢法只會在特定對手，以及特定時間裡使用，但還是認可3人在鋒線上的配合，而3人之間的互動，也讓尤文的前場不再那麼死氣沉沉，好的氣氛會讓球員得到更好的發揮，特別是像C羅這樣的頂級球員，只要踢開心了，各種高難度的進球不在話下。

C羅就是這麼霸氣，雖然金球獎再次被梅西趕超，但只要不受到傷病困擾，聯賽下半場的C羅還會給球迷帶來更大的驚喜，歐冠冠軍？那並不是開玩笑。
