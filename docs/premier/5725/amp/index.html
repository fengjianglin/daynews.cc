早有徵兆？瓜迪奧拉明年離開曼城回歸巴薩，曼城花1億鎊挽留

天天要聞
2019-11-12

被爭冠對手利物浦直接已經下天王山之戰，賽季進行1/3已經落後9分，瓜迪奧拉今年遇到的困難前所未有，去年掉鏈子最後靠著追分逆轉成功衛冕，今年還要分心歐冠聯賽的希望已不大。

最新的法國和美國媒體消息稱，瓜迪奧拉將孤擲一注爭取歐冠，原本無意引援的曼城也將大力支持，冬窗將首先花費一億英鎊進行補強中軸線。因為明年瓜迪奧拉合同就將到期，而他也沒有要續約的跡象，明年或將正式離開曼城。

瓜迪奧拉花了三年多時間想將曼城帶到豪門地位，奈何歐冠兩次好機會都輸給了英超球隊，今年將是他的最後一年，所以他在打利物浦之前的歐冠小組賽都沒有輪換主力，如果沒有歐冠冠軍曼城始終就不是頂級強隊，那麼對瓜迪奧拉來說他就是有瑕疵的。但是時刻到了，他也沒辦法，就跟當初離開拜仁一樣。

瓜迪奧拉從來都沒有長留一個俱樂部的打算，他個人希望慢慢征服五大聯賽，正如很多大牌教練一樣。在巴薩待了4年，隨後拜仁慕尼黑執教3年，今年是他在曼城開始的第4個年頭，賽季結束就會追平最長執教紀錄。4年一輪迴，瓜迪奧拉會選擇以他的方式開啟另一段旅程，有可能的下一站有巴薩、拜仁和尤文。

瓜迪奧拉執教了巔峰的巴薩，兩奪歐冠，但是他在那個時候並沒有特別的出彩，畢竟是宇宙隊巴薩換個教練蹲著不指揮也能照樣贏球。如今，巴薩同樣兩年被逆轉，隊內飽受巨大壓力，巴爾韋德排名世界第一水分卻很大而且明年合同也將到期，梅西年齡也不小了，這給了他回歸的條件。

另外，瓜迪奧拉的妻子回到西班牙進行生活，留下瓜迪奧拉一人在英國，這是讓他改變主意離開的更重要原因。他的妻子可不只是老婆這麼簡單，類似於經紀人，當初尤文邀請瓜帥時她就親自去義大利接洽。而且，瓜迪奧拉經常回到西班牙參加活動，一些其他因素的訴求也將讓他回到巴薩。

尤文之前和瓜帥有過接觸，私下有沒有協議不得而知，瓜迪奧拉在拜仁當地名下房產還沒有變賣，所以不排除去這兩隊的可能，但是綜合看回歸巴薩是眾望所歸。當回歸這一刻來臨，各大俱樂部又得因換帥翻天的變化，梅西與瓜迪奧拉再次相遇那可能真是為同一個理想而奮鬥。
