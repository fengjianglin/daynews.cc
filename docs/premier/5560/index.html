曼聯隊內兩大失意之人！一人失寵欲逃離，另一人逐漸成為球隊英雄

2019-11-22 0條評論 18次閱讀 0人點贊

上輪聯賽曼聯主場戰勝布萊頓，拿到寶貴的3分球隊在積分上升至第7位，索爾斯克亞拯救了自己！挪威人率領的曼聯本賽季表現十分糟糕，聯賽首輪4球大勝切爾西的比賽令球迷拍手叫好，此後曼聯就陷入低迷。昔日的英超霸主在弗格森退休後，始終無法回到當初的統治地位，不論是穆里尼奧還是有著紅魔血統的索爾斯克亞似乎都難以駕馭這支曼聯，近期球隊狀態有所回升，也讓球迷看到了一絲希望。今天我要說的是曼聯隊內兩大失意之人！一人失寵欲逃離，另一人逐漸成為球隊英雄，他們就是馬蒂奇和弗雷德！

一朝天子一朝臣，隨著去年冬天穆里尼奧被曼聯解僱，新帥索爾斯克亞走馬上任，塞爾維亞人馬蒂奇的命運從此發生了改變，備受穆里尼奧喜愛的馬蒂奇在葡萄牙人麾下是絕對主力，但自從挪威人入主老特拉福德後，馬蒂奇並不受索爾斯克亞喜愛，本賽季麥克托米奈的崛起更讓馬蒂奇坐穿了板凳，本賽季至今馬蒂奇僅為曼聯打了5場比賽，受不到重用的塞爾維亞人心灰意冷，他本人已經決定在接下來的轉會窗口尋求離隊。

當然30歲的塞爾維亞中場並不缺乏追求者，意甲的AC米蘭以及西甲的馬德里競技都希望能夠引進這位實力派球員，近日隨著穆里尼奧成為熱刺主帥，更有不少聲音呼籲馬蒂奇再次投靠恩師！在曼聯隊內此前狀況比馬蒂奇還要慘的弗雷德則通過近期的表現逐漸打動主帥索爾斯克亞，當初以6000萬歐元從頓涅斯克礦工加盟曼聯的巴西人被曼聯球迷寄予厚望，但高昂的轉會費也讓弗雷德背負著巨大的壓力。

加盟曼聯前，在烏克蘭聯賽弗雷德異常神勇受到多家豪門俱樂部的追捧，然後在老特拉福德他卻始終無法找到狀態，失去耐心的穆里尼奧用實際行動表明他已經將弗雷德打入冷宮，此時這位身價6000萬歐元的巴西人被曼聯球迷貼上了名副其實的「水貨」標籤。上賽季中途索爾斯克亞到來，但弗雷德依然沒有迎來真正的機會，直到本賽季曼聯全隊狀態直線下滑，索爾斯克亞無奈之下再次啟用巴西人，但這次弗雷德抓住了機會，在歐聯杯對陣阿斯塔納的比賽中表現不俗，隨後在曼聯的12場各項比賽中巴西人首發了9場！

上輪英超聯賽戰勝布萊頓的比賽中，弗雷德出色的狀態征服了老特拉福德球迷，賽後他被評為全場最佳，成為球隊獲勝的英雄！目前他與麥克托米奈組成的中場成為索爾斯克亞所倚仗的一對組合，在經過長時間的沉睡後巴西人終於蘇醒。歡迎在評論區留言討論。
