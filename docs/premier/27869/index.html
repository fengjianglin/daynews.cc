比利時球員周薪TOP10：中超佔據三席位

2019-12-22 1,082條評論 2,324次閱讀 1204人點贊

我們說起足球，肯定想到的是巴西、阿根廷等，一直以來世界排名也是巴西等居首，但是最近國際足聯更新的最新一期男足國家隊的世界排名，比利時則高居榜首，由此可見比利時的足球也是相當的牛。而比利時的球員也是遍布全球，最近比利時媒體HLN對比利時球員的周薪排名進行了更新，其中排在前10的基本都是效力於5大聯賽的球員。不過繁花似錦的中超卻依舊有3名球員擠進了TOP10。

其中排在第一位的則是，今年夏天剛剛轉會皇馬的阿扎爾，他的周薪來到了48.5萬歐元，阿扎爾今年夏天以1.5億英鎊加盟皇馬，目前8場西甲比賽，僅取得了1球和1助攻，總感覺這樣的數據跟身價有所不匹配呀。

第二則是來在我們大中超山東魯能的費萊尼，他在今年的2月份以1000萬英鎊轉會至山東魯能，在本賽季為山東魯能打進了8個進球和3個助攻，更是成為魯能戰術體系中不可或缺的一員，他的周薪為42萬歐元。

第三位則是曼城的德布勞內，這位比利時球員在2015年以5400萬英鎊轉會到了曼城，成為了曼城的一線球員，本賽季為曼城在聯賽中出場17場，打進6粒進球並貢獻11次助攻，為此他的周薪來到了32.5萬歐元。

第四位則是來自皇馬的庫爾圖瓦，這位比利時國門，在2018年8月份從切爾西以3500萬歐元加盟到了皇馬，已經成為皇馬的主力門將，目前周薪為28萬歐元。

第五位是來自國際米蘭的盧卡庫，這位比利時前鋒在2019年8月以8000萬歐元從曼聯轉會到國米，本賽季在意甲出場17場，打進12球並送出1個助攻，目前周薪為26.5萬歐元。

第六位登貝萊，來自大中超廣州富力，登貝萊在廣州富力出場26次，打進1球送出4次助攻，場均1.8次關鍵傳球也說明他在廣州富力的作用，目前富力給了他周薪為24萬歐元。

第七位依舊是來在大中超，那就是一天到晚鬧著要離開球隊的大連一方的卡拉斯科，雖然卡拉斯科天天鬧「革命「，但是本賽他對於大連一方的作用是毋庸置疑的，他的周薪達到了20萬歐元。

而來自熱刺的阿爾德韋雷爾德，那不勒斯的默滕斯和多特蒙德的維特塞爾分別以17.5萬歐元、14.5萬歐元和1.4萬歐元排在第8、第9和第10位。
