巴薩領跑球衣贊助榜！英超前十佔6席，C羅加盟後 尤文提升2600萬

天天要聞
2019-12-20

近年來，各大體育運動品牌與俱樂部簽訂的球衣贊助是水漲船高。記得2015/16賽季，曼聯俱樂部與阿迪達成了十年7.5億英鎊的天價贊助，紅魔每個賽季拿到7500萬英鎊在當時的歐洲足壇首屈一指。可如今，曼聯已經從榜首跌至第三，而同屬英超的BIG6球隊也逐步縮小了與曼聯之間的差距；另外，C羅加盟後尤文提升了2600萬英鎊。

據最新數據顯示，擁有梅西的巴薩是現役足壇得到球衣年贊助費最高的球隊。在2015/16賽季拿到隊史的第24座西甲冠軍後，運品牌耐克決定為合作夥伴獻上一份新的合約。為此，耐克將每年為巴薩提供總價值高達1億英鎊的球衣贊助，這讓西甲豪門超越曼聯登上球衣贊助榜單第一。對於耐克來說，巴薩戰績突出，再加上有影響力超高的梅西加持，這是個雙贏的決策。

在吸金能力上，巴薩的死敵皇馬同樣不落下風，銀河戰艦在歐冠上的巨大成功讓各類贊助蜂擁而上。作為本隊球衣的長期合作夥伴，阿迪與皇馬已從1998年合作至今，期間也是數度續約。皇馬目前以200萬英鎊之差落後於巴薩，排名第二。

此前的榜首球隊曼聯目前已經降至第三，而第四至第六也均是來自於英超的球隊。英超兩連冠曼城今夏更換了球衣贊助商，彪馬取代了此前的耐克。為此，英超冠軍每年得到了總價值6500萬英鎊的球衣贊助。阿森納本賽季轉投耐克，贊助也從彪馬時期的3000萬英鎊/年翻了一番，達到6000萬英鎊，這使得阿森納一舉追平切爾西，並列第五。

排名第七的是意甲霸主球隊尤文圖斯。要知道，尤文此前未能進入前十，但在18/19賽季C羅加盟老婦人後，阿迪馬上提前與意甲冠軍進行了續約，贊助費從此前的2000萬英鎊升至目前的4600萬英鎊，C羅效應再次從中體現。而位居尤文身後的是上賽季的歐冠冠軍利物浦，紅軍的4500萬英鎊還不足皇馬的1/2。不過有消息稱，利物浦將在下賽季停止與NB的合作關係，正有崛起之勢的紅軍有望再次回歸阿迪大家庭，而超越曼聯升至英超第一。

另外，排名前十的俱樂部還包括了拜仁（4200萬英鎊）和熱刺（3000萬英鎊）。值得一提的是，在這份榜單的前十中，獨佔六席的英超無愧商業價值最高的聯賽，而次榜單中五大聯賽唯缺法甲。
