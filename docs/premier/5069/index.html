瘋狂7連勝！意甲超級黑馬橫空出世，爆冷擊敗8.3億豪陣奪冠熱門

2019-12-08 0條評論 14次閱讀 0人點贊

拉齊奧上賽季38輪比賽取得17勝8平13負積59分排名聯賽第8，比奪冠的尤文圖斯真正少了31分！但是，本賽季拉齊奧卻成為意甲的超級黑馬，成為爭冠第一集團的一份子，有些不可思議。在剛剛結束的意甲第15輪的比賽，拉齊奧在主場迎戰坐擁8.3億豪陣的奪冠大熱門尤文圖斯，竟然爆冷以3-1擊敗對手，終結了尤文圖斯聯賽14輪不敗紀錄。

這樣一場失利對於尤文圖斯來說在預料之中，因為本賽季尤文圖斯有太多場比賽都是驚險獲勝，要知道運氣總不會站在尤文圖斯這一邊。本賽季尤文圖斯有9輪聯賽都是凈勝對手1球，包括：1-0擊敗帕爾馬、都靈和AC米蘭，2-1擊敗維羅納、布雷西亞、國際米蘭、博洛尼亞和熱那亞，4-3擊敗那不勒斯。而在上一輪聯賽，尤文圖斯在主場涉險逼平薩索洛，已經距離輸球很近了，這一次終究還是輸給了拉齊奧。

拉齊奧在意甲第8輪的時候，和亞特蘭大踢了一場非常奇葩的比賽，球隊在0-3落後的情況下硬生生把比分扳平，以3-3結束。從那時候開始，拉齊奧的狀態就大爆發，從第9輪聯賽一直贏到現在，取得瘋狂7連勝。包括：2-1絕殺佛羅倫薩、4-0血洗都靈、2-1險勝AC米蘭、4-2戰勝萊切、2-1絕殺薩索洛、3-0完勝烏迪內斯、3-1擊敗尤文圖斯。

拉齊奧能夠取得如此出色的成績，與1名球員的逆天發揮分不開，他就是因莫比萊。他至今已經進了17球，是五大聯賽中進球最多的球員。比尤文圖斯當家射手C羅整整多了10球。不過，在這場拉齊奧與尤文圖斯的比賽，因莫比萊的表現並不出色，他有6腳射門沒能收穫1粒進球，還罰丟了1粒點球。

現在意甲的爭冠形勢變得越來越撲朔迷離，有點群雄爭霸的意思。國際米蘭15輪12勝2平1負積38分排名榜首，尤文圖斯15輪11勝3平1負積36分排名次席，拉齊奧15輪10勝3平2負積33分排名第3。前3名的積分差距都比較小。隨著賽季的深入，板凳深度和球員傷病會對球隊的成績帶來非常大的影響。從球隊的陣容和實力來看，尤文圖斯依然是奪冠的最大熱門。
