19:30博洛尼亞VS帕爾馬

博洛尼亞

【球員】核心球員奧爾索利尼上輪為博洛尼亞打進唯一1顆進球，而他也入選了上周的義大利國家隊，並在9-1大勝阿美尼亞的歐預賽中攻入1球，狀態出色的他此役有望繼續首發。

【傷停】博洛尼亞兩名後衛達尼洛、馬蒂亞巴尼在本場對戰帕爾馬的比賽當中，由於雙雙累積黃牌的緣故而自動停賽，幸好日本國腳富安健洋目前已經傷愈，可以為球隊出戰。

帕爾馬

【球員】庫盧賽夫斯基新賽季在帕爾馬成為了絕對主力，12場聯賽貢獻2球5助攻，引起了全歐洲的關注。這名新星入選了最新一期的瑞典成年國家隊名單，並在上周完成國家隊首秀。

【傷停】科內柳斯在對陣羅馬一役頂替受傷的因格萊塞作為首發中鋒出戰，並在傷停補時打進1球鎖定勝局，不過他在代表丹麥國家隊出戰歐預賽中受傷，又折損一名鋒線主力。
