利物浦前鋒薩拉赫的問題，以及批評家們是如何將其誤解的

2019-11-29 0條評論 16次閱讀 0人點贊

利物浦前鋒穆罕默德·薩拉赫在冠軍聯賽對陣那不勒斯的比賽中表現平平，受到了批評

譯自：利物浦作家ByIan DoyleChief

在安菲爾德，人們聽到了利物浦的喘息聲，他們正在為贏得歐冠淘汰賽的勝利而努力，卻錯過了一個機會。

這次是默罕默德·薩拉赫猶豫了，他直接從那不勒斯門將亞歷克斯·梅雷特的禁區內射門，卻沒有戰勝門將。

這是這名埃及人的不佳表現，這個賽季曾經的護身符拚命地想找回狀態。

至少越來越多的評論家希望人們相信這種說法。

毫無疑問，薩拉赫在最近幾場比賽中並沒有像前幾場比賽那樣頻繁地折磨對方的防線——在周三1-1戰平那不勒斯的比賽中，他也遠遠沒有發揮出最佳狀態——但這其中有幾個原因。

尤其是在萊斯特城的哈姆扎·喬杜里的一次醜陋的挑戰結束時，他的腳踝受傷了。

那差不多是七周前的事了。

從那以後，薩拉赫錯過了英超聯賽對曼聯的比賽，並在上周末對陣水晶宮的比賽中坐在替補席上。

很明顯，他的傷勢仍然困擾著他，這個問題可能在精神上和身體上都造成了限制。

另一個事實是，對手非常清楚這個問題。

然而，薩拉赫對利物浦主帥克洛普來說太重要了，不能讓他休息很長一段時間。

事實上，那些相信對手防守的人已經「發現」薩拉赫忽視了本賽季埃及人的數據。

在18場比賽中，他攻入9球，貢獻5次助攻。

12個月前，他在18場比賽中打入9球，4次助攻。

比去年還好?詆毀者將會對此感到困惑，特別是考慮到那些進球包括對陣阿森納、托特納姆熱刺和曼城的進球。

他們當然會提到薩拉赫的第一個賽季，這個邊鋒在20場比賽中打進15球，4次助攻。

然而，很少有球員能夠在一個賽季又一個賽季都保持這樣的產出，尤其是當他不是中路前鋒的時候。

這並不是巧合，薩拉赫在周中被那不勒斯的卡里杜·庫利巴利在禁區內用手推搡，使得薩迪奧·馬內在對方邊路得分。而埃及人所扮演的角色略有不同，即使是在右翼。

有趣的是，在對陣那不勒斯的比賽中，當利物浦改用4-2-3-1陣型時，薩拉赫得到了機會。

克洛普在去年冬天使用了同樣的方法，不僅利用了一個狀態良好的沙奇里，也增加了薩拉赫的進球威脅。

在擔任中鋒的12場英超聯賽中，他打入了12球，6次助攻。

法比尼奧有可能在邊線上呆上一段時間，這可能會讓利物浦回到那種狀態。

不管怎樣，本賽季薩拉赫唯一的問題是受傷了——而且他還想繼續為球隊貢獻自己的力量。
