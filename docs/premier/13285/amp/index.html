周二010、歐冠—多特蒙德VS布拉格斯拉維亞

比賽時間：2019年12月011日04:00

今晚歐冠小組賽最後一輪即將打響，除了A組、B組鎖定出線之外，其餘小組皆有未能全部揭曉小組賽突圍隊伍，這樣的情況在獵鷹推薦這場F組俗稱「死亡之組」中也有發生。目前F組中除了巴薩鎖定出線名額之外，國米以7積分暫列該組第二名，緊隨其後的是多特蒙德以6積分位列第三位，所以今晚不光是多特需要戰勝布拉格斯拉維亞，還得看巴薩與國米那場比賽的結果出來才知道F組第二位出線隊伍是誰。這樣的情況在今晚比賽的很多小組中都會發生，驚喜得留到最後那一刻揭曉。

好了，先來說下獵鷹推的這場比賽，多特作為德甲傳統強隊，雖然本賽季表現出的情況是後防線不夠穩定，防守掉落至德甲中游水平，但進攻火力方面還是一如既往的強勢，場均進球數達2.36球。在主場作戰更是強勢到底，目前是德甲唯一一個能主場不敗的球隊，而且在聯賽中近期重返勝軌，目前重心在於歐冠出線之戰中。客隊布拉格斯拉維亞是捷甲聯賽中的球隊，無論是各個方面在這死亡之組都處於絕對劣勢，目前僅獲2積分排名F組最後一名，出線已無望，今晚面對主場的多特輸球是必然，只能祈求少輸當贏。

巴薩方面已經確定梅西輪休，皮克也未隨隊出征，這樣的做法外界調侃是：你賣我勞塔羅，我放水放你進歐冠。可能外界這樣的看法會對今晚多特有一定的影響，因為只要國米贏了巴薩就能鎖定小組出線名額。但是反觀如果今晚多特不去爭取一把，那麼連出線的希望都不復存在，雖然得看巴薩的臉色，但如果國米不爭氣，連讓了梅西的巴薩都戰勝不了，那麼多特出線就概率很大。今晚大黃蜂為自己而戰，出線得靠自己努力爭取，幸運只會降臨在做好準備的人身上，獵鷹看好今晚大黃蜂主場打穿盤口。
