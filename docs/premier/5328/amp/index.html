菲利克斯高票當選2019金童，將在12月16號前往都靈領獎

天天要聞
2019-11-28

歐洲金童獎是由義大利《都靈體育報》於2003年創辦，評出年度最佳年輕球員。其中馬德里競技的年輕葡萄牙球員若奧·費利克斯上賽季表現出了不俗的實力，成功的拿下了金童獎。幾個月前還在巴塞羅那青年A隊的安蘇·法蒂在這個榜單中排名第六。

菲利克斯在所有國際媒體投票中，獲得了332票，毫無懸念的高票當選了2019年歐洲金童，他也是繼去年德里赫特之後又一位獲獎的年輕球員，上賽季菲利克斯憑藉在本菲卡出場26次打入15球助攻7次的優異成績，夏窗以1.2億的天價加盟馬德里競技，在加盟馬競後出場13次，打入3球貢獻1個助攻，他也希望能夠藉此獲獎機會機會重回本菲卡的巔峰狀態。

多特蒙德19歲新星桑喬，在上賽季坐穩多特首發為球隊貢獻了13個進球16個助攻，並在新賽季開始後延續著出色的狀態，賽程還沒過半已經為球隊打入6球助攻8次這樣優異的表現也讓他獲得了175票，排在這份榜單的第二位置。

第三名是勒沃庫森中場哈弗茨，他得到了75票，他上賽季為球隊出場42次打入20球助攻6次，優異的表現讓他得到了75張選票，他在勒沃庫森的出色表現也引起了包括拜仁、皇馬等歐洲豪門的注意，有消息稱下賽季拜仁會砸重金引進這位年輕的德國中場。

第四名是來自於薩爾茨堡紅牛的哈蘭德，這位挪威的年輕球員本賽季異軍突起，已經上演了5個帽子戲法，包括了歐冠附加賽中的出色表現讓他在金童獎投票中獲得了74張選票位列第四名。

第五名是上屆金童獎得主，夏天以1億身價從阿賈克斯轉會到尤文圖斯的德里赫特，上賽季德里赫特憑藉著出色的防守，以及不錯的進攻能力，以隊長的身份帶領阿賈克斯闖進歐冠四強，成為了歐洲足壇最熱門的新星之一，他獲得了71張選票。

第六名則是巴薩小將法蒂，上賽季他還在青年隊，由於季前賽的出色發揮被巴爾韋德招入一線隊，並在梅西受傷的那段時間有著不錯的發揮，剛剛17歲的他代表著拉瑪西亞的未來，他在本賽季打入2球送出1個助攻，他獲得了49張選票。

第七名是來自曼城的福登，這位英格蘭中場有著出眾的天賦，在瓜迪奧拉的調教下越發靈性，瓜迪奧拉甚至還說福登是我們唯一的非賣品，就算收到5億歐元，福登獲得了46張選票。

第八名是來自於米蘭成名已久的多納魯馬，這也是入選這份榜單中唯一的一個門將，多納魯馬年少成名，一度成為米蘭城最後的一堵牆，可惜多年的沉淪加上資金運轉不周，AC米蘭沒能在聯賽或者歐戰取得更大的突破，多納魯馬獲得了37張選票。

第九名是來自羅馬的扎尼伽洛，他擁有著出色的盤帶技術，本賽季他的盤帶成功率在五大聯賽僅次於梅西，成為了羅馬的主力為球隊帶來5個進球2個助攻，併入選了義大利國家隊，在歐預賽中取得了自己國家隊的處子球，他獲得了36張選票。

第十名是來自於埃因霍溫的馬倫，本賽季他在埃因霍溫出場19次打入16球助攻3次，場均造1球的超高效率讓他成為了豪門俱樂部的關注對象，他獲得了35張選票。

金童獎由40名國際媒體記者組成的評審團投票，選出每年表現最佳的21歲以下的球員，每個評審員有5次投票名額，第一張選票10分，第二張選票7分，第三張選票5分，第四張選票3，第五張選票1分。
