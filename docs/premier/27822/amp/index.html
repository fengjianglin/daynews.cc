C羅啞火！一場3-1讓尤文錯失賽季首冠，意甲豪門第5次奪冠

天天要聞
2019-12-23

​北京時間12月23日凌晨0點45分，義大利超級盃決賽準時打響，尤文圖斯對陣拉齊奧。全場比賽戰罷，拉齊奧憑藉阿爾貝托、盧利奇和卡達迪各入一球，迪巴拉扳回一球，補時階段，本坦庫爾被紅牌罰下，最終拉齊奧3-1擊敗十人尤文圖斯，從而獲得義大利超級冠軍，這是拉齊奧隊史上第5次獲得該項賽事冠軍。

第8分鐘，盧利奇傳球給到禁區外的阿爾貝托，阿爾貝托起腳遠射稍稍高出橫樑。阿爾貝托射門時被馬圖伊迪鏟倒，主裁判向馬圖伊迪出示黃牌。第16分鐘，盧利奇晃過尤文球員德西利奧後起腳傳中，米林科維奇頭槌擺渡，因莫比萊傳球，阿爾貝托一腳推射破門，拉齊奧1-0領先尤文圖斯。第20分鐘，盧利奇傳球給到阿爾貝托，阿爾貝托帶球殺入禁區內起腳射門被尤文門將什琴斯尼沒收。

第22分鐘，伊瓜因傳球給C羅，C羅帶球到禁區內右腳射門偏出。第25分鐘，C羅傳球給到迪巴拉，迪巴拉左腳爆射高出。第30分鐘，C羅為尤文贏得一個前場自由球，迪巴拉主罰自由球選擇射門偏出。第33分鐘，阿爾貝托帶球突入禁區起腳射門被什琴斯尼撲出。第45分鐘，馬圖伊迪傳球給C羅，C羅一腳遠射造成拉齊奧門將斯特拉科沙撲救脫手，迪巴拉及時跟進補射破門，半場比賽尤文1-1戰平拉齊奧。

下半場比賽開始，第49分鐘，C羅帶球內切後橫傳給迪巴拉，迪巴拉左腳低射偏出。第52分鐘，盧卡斯挑傳到禁區內米林科維奇頭槌攻門太正被什琴斯尼沒收。第54分鐘，科雷亞下底傳中，禁區左側的盧利奇右腳射門被什琴斯尼沒收。第69分鐘，拉齊奧獲得右側角球機會，卡達迪開出角球到禁區內科雷亞頭槌攻門偏出。

第71分鐘，拉姆塞分球給弧頂處的C羅，C羅一腳遠射稍稍偏出。第73分鐘，拉扎里傳中，盧利奇門前左腳凌空爆射破門，拉齊奧2-1再次領先尤文。第75分鐘，誇德拉多挑傳，拉姆塞頭槌攻門被斯特拉科沙穩穩將球拿到。第82分鐘，科雷亞獲得單刀機會一腳抽射破門，不過裁判示意科雷亞越位在先進球無效。

第90分鐘，博努奇禁區內頭槌攻門稍稍偏出。補時階段，本坦庫爾犯規，兩黃變一紅被罰下，第94分鐘，拉齊奧獲得前場自由球機會，卡達迪主罰自由球直接射門皮球飛進球網，卡達迪的進球也殺死比賽懸念，拉齊奧3-1戰勝尤文。值得一提的是，意甲第15輪拉齊奧也是3-1擊敗尤文，尤文兩周內被對手雙殺。
