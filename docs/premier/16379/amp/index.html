三豪門兩喜一憂！C羅破門尤文3-1，曼聯1-1扶貧，熱刺2-1絕殺

天天要聞
2019-12-16

北京時間12月15日22:00，英超和意甲進行了多場比賽，結果三大豪門兩喜一憂，其中C羅率領尤文在主場3-1大勝烏迪內斯，而曼聯則不幸在主場1-1憾平埃弗頓。至於熱刺，本輪他們做客2-1絕殺狼隊。

在本輪比賽中，尤文主場迎戰烏迪內斯，結果薩里罕見讓C羅、伊瓜因和迪巴拉聯袂首發，這個安排也讓尤文的進攻實力徹底釋放。此役剛開始9分鐘，迪巴拉射門被擋，隨後C羅不停球直接補射，皮球直接飛進死角，為球隊取得領先。在浪費了數次機會之後，第37分鐘又是C羅站了出來，接到伊瓜因的直塞之後，這位超級巨星單刀破門，上演了梅開二度的好戲。隨後博努奇頭槌錦上添花，幫助尤文在主場3球領先對手。在比賽最後時刻，烏迪內斯打進一球，將比分改寫為1-3。加上本場比賽，C羅已經連續4場各項比賽破門，在擺脫了膝蓋傷勢之後，他終於徹底找回了狀態，相信接下來C羅的進球，還會源源不斷的到來。

本輪英超，曼聯主場對陣埃弗頓，剛開始幾分鐘，林加德就在禁區內轉身掃射，可惜皮球偏出。隨後詹姆斯獲得單刀機會，結果皮球滑門而出。第36分鐘，久攻不下之後，曼聯自己出現失誤，在防守角球時，林德洛夫不幸將皮球擋進自家大門，因為這個烏龍球，曼聯也在主場以0-1落後。下半場第78分鐘，替補登場的格林伍德在禁區前低射破門，最終幫助曼聯在主場1-1戰平對手。本賽季，曼聯對陣強隊時戰績彪炳，可惜一遇到中下游球隊卻頻頻丟分，本輪戰平對手之後，他們又不幸在英超扶貧，如果曼聯每一輪的對手都是熱刺和曼城，相信他們肯定排名英超第一。

在另一場比賽中，熱刺做客對陣狼隊，第8分鐘盧卡斯靠個人能力在禁區內連續突破最終爆射破門。遺憾的是，領先之後熱刺反而被對手壓制，在上半場的比賽中，狼隊創造了數次機會，可惜都一一浪費。下半場第67分鐘，狼隊搶斷後快速反擊，結果特勞雷在禁區前打出一腳精彩的遠射，憑藉這個世界波，狼隊也在主場以1-1扳平比分。在比賽補時階段，熱刺獲得角球機會，結果維爾通亨頭槌頂進大門，幫助球隊在客場2-1絕殺對手。不得不說，穆帥確實就有這樣的魔力啊，正是他換上埃里克森，製造了這次絕殺傳中，熱刺有這樣的神奇主帥，確實太幸福了！
