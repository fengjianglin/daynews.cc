歐洲五大聯賽期中考，C羅數據難及格

2019-12-24 1,446條評論 3,264次閱讀 1325人點贊

隨著聖誕節的到來，五大聯賽相繼進入暫時的休戰期，第一競彩總結了上半賽季五大聯賽所有球員、各支球隊的一些有意思的數據。

球員篇

進球

本賽季到目前為止歐洲五大聯賽射手榜前五：萊萬、維爾納、因莫比萊、瓦爾迪、梅西，C羅沒進前十！

球隊貢獻度排名榜首的是丹尼·因斯11球（南安普頓），占球隊進球數的52%，緊隨其後的是因莫比萊17球（拉齊奧45%）、本耶德爾13球（摩納哥32%)、萊萬19球（拜仁41%）、瓦爾迪17球（萊斯特城41%）

場均射門數前五

儘管C羅在射手榜沒能進入前十，但在場均射門數方面，C羅以每場5.6腳射門排名首位，緊隨其後的是內馬爾4.7次（巴黎聖日耳曼）、萊萬4.6次（拜仁）、巴洛特利4.5次（布雷西亞）、巴拉爾迪4.4次（薩索洛）

進球與射門兩項數據對比，不得不感嘆的是C羅也許真老了！

場均盤帶數前五

年過30的梅西場均盤帶數5.5次依舊傲視群雄，其餘4位分別是內馬爾4.9次、特勞雷4.6次（狼隊）、扎哈4.6次（水晶宮），馬克西姆4.5次（紐卡）。

場均助攻數前五

目前的助攻王是效力於意甲拉齊奧的阿爾貝托，本賽季到目前為止貢獻11次助攻，周末他剛剛幫助拉齊奧3比1贏下尤文拿到義大利超級盃冠軍。

與他助攻次數並列的是拜仁的托馬斯·穆勒，隨著年紀的增大，托馬斯·穆勒已經逐漸轉變成助攻型前鋒，有點像當年魯尼的感覺。

接下來就是效力於曼城的德布勞內（10次）、效力於多特蒙德的桑喬（9次）、以及同效力於多特的小阿扎爾（7次）。

桑喬已被英超豪門切爾西、曼聯盯上，他在大黃蜂的職業生涯或許已經進入倒數，而小阿扎爾本季轉會多特顯然又是一筆雙贏的買賣。

場均攔截數

萊斯特城中場恩迪迪以及里卡多·佩德羅分別以4.4次、4.2次排名第三第四，可以想見萊斯特城本季的崛起並不是偶然。

球隊篇

場均控球率前五

德甲拜仁慕尼黑以62.7%的控球率高居榜首，其餘分別是巴黎聖日耳曼（61.8%）、曼城（60.9%）、巴塞羅那（60.8%）、勒沃庫森（59.7%）

五支球隊都是技術型球隊。勒沃庫森能上榜主要還是因為較量博斯的關係，此人偏好全攻全守的激進足球，不會防守，但以目前藥廠球員的能力，還不足以支撐如此激進的打法，聯賽到了後半程，藥廠有可能開啟神坑模式，需要特別注意。

傳球成功率前五

大巴黎傳球成功率90.2%，這與法甲球隊實力較弱有一定關係，排名第二到第五位的分別是曼城（88.6%）、巴塞羅那（87.9%）、拜仁（87.3%）、皇馬（87.3%）

場均射門數

曼城20.8次排名第一，緊隨其後的是那不勒斯（20.1次）、亞特蘭大（19.9次）、拜仁（18.6次）、尤文圖斯（18.2次）

高空球爭奪成功率

畢爾巴鄂競技59.5%排第一，尤文圖斯（58.1%）、薩爾茨堡（57.4%）、皇馬（57.1%）、帕爾馬（56.1%），尤文圖斯主要是有C羅、薩爾茨堡主要是因為球隊當紅炸子雞哈蘭德、皇馬主要是以因為在鋒線上有本澤馬這一點。

哈蘭德目前面臨著豪門的選擇，而一旦他離開，薩爾茨堡紅牛在高空球爭奪方面的數據必然會下降。

2019/20賽季上半賽季歐洲五大最佳陣容

陣型：4-4-2

守門員：戈蒂埃（布雷斯特）

後衛線：佩雷拉（萊切城）、范迪克（利物浦）、斯莫林（羅馬）、戴維斯（拜仁）

中場：梅西（巴薩）、德布勞內（曼城）、迪馬利亞（大巴黎）、維爾納（萊比紅牛）

鋒線：姆巴佩（大巴黎）、萊萬多夫斯基（拜仁）
