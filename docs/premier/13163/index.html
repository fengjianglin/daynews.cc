歐冠小組賽盤點！小國米7分逆襲 賈府10分竟與不勝妖隊征戰歐聯

2019-12-12 830條評論 2,271次閱讀 255人點贊

北京時間今天凌晨，歐冠小組賽全部結束，十六強悉數產生，亞特蘭大、馬競搭上末班車，這是歐冠歷史上首次出現16強球隊都是五大聯賽球隊的現象，其它國家聯賽球隊全部被擋在了十六強外。英超、西甲共八支球隊全部晉級，世界第一聯賽之爭愈演愈烈。下面我們就來盤點一下歐冠小組賽一些有趣數據。

命最硬球隊：亞特蘭大

亞特蘭大今晨客場3-0擊敗頓涅茨克礦工，最終以小組第二的成績晉級16強。前三輪小組賽中，亞特蘭大連續輸給了薩格勒布迪納摩、頓涅茨克礦工和曼城，三戰全敗1分未得。後三輪比賽，他們先戰平曼城，連續戰勝迪納摩和礦工，最終積7分獲得小組第二出線。他們也成為歐冠改制以來，第二支小組賽前三輪0分，最後卻能獲得出線權的球隊。上一支完成這樣壯舉的球隊是2002/03賽季的紐卡斯爾聯隊。

最悲催球隊：阿賈克斯

與亞特蘭大形成鮮明對照的是阿賈克斯，H組是出線形勢最膠著的一個小組，只要在最後一輪打平就能出線的阿賈克斯在主場遭爆冷，他們親手將巴倫西亞送上小組第一的寶座，而他們卻以6戰3勝1平2負積10分的高分無緣歐冠16強，作為去年歐冠的黑馬，本賽季他們成為積分最高的小組第三。與其同分的熱刺、多特、馬競都順利晉級了淘汰賽。

最狗屎運球隊：布魯日

這支比甲勁旅在所在A組在一勝難求的情況下力壓加拉塔薩雷成為小組第三獲得歐聯淘汰賽資格，他們6戰取得3平3負積3分，只打進4個進球。而聖彼得堡澤尼特成為小組賽積分最高的墊底球隊（7分墊底），這與前者形成巨大的反差。

火力最猛球隊：拜仁

拜仁 B組小組第一 打進24球/失5球

歐冠小組賽末輪，拜仁主場3-1擊敗熱刺，至此拜仁小組賽取得6場比賽全勝的成績。他們小組賽6場比賽中總計攻入24球，包括7-2熱刺，6-0紅星這樣的大比分勝利。在歐冠小組賽階段攻入24球，這也是歐冠歷史上小組賽的第二多進球，只有17-18賽季的巴黎（25球）在小組賽中比本賽季的拜仁進球更多。

防守最好球隊：巴黎

巴黎 A組小組第一 打進17球/失2球

巴黎成為16強小組賽中失球最少的球隊，他們6戰拿下5勝1平力壓皇馬穩居小組頭名，球隊攻防兩端表現俱佳，他們僅在客場2-2逼平皇馬的比賽中失球，其餘5場均保持零封對手。而亨克與貝爾格萊德紅星成為本賽季歐冠小組賽中防守最爛的球隊，他們的失球均達到了20球。
