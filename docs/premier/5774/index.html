周日040利物浦VS曼徹斯特城

利物浦上周末最後時刻逆轉維拉，保持6分領先優勢，周中歐冠，他們主場2比1小勝亨克，主帥克洛普輪換了後衛洛夫倫和羅伯遜、前鋒馬內和菲爾米諾，上周末輪休的後腰法比尼奧回歸，中場維納爾杜姆和張伯倫進球；中衛馬蒂普依舊因傷缺席。

曼城上周末同樣是2比1艱難逆轉南安普敦，周中歐冠，他們做客1比1戰平亞特蘭大，前鋒斯特林破門，主力門將埃德森半場因肌肉問題提前被換下，替補門將布拉沃染紅後，後衛沃克一度替補出場當了10分鐘門將；埃德森周末是否參賽還需要觀察，但周中已經因傷缺席的隊長D·席爾瓦估計是趕不上了，後腰羅德里和中衛拉波爾特也將繼續缺席。

周日052尤文圖斯VSAC米蘭

尤文圖斯上輪客場1比0擊敗都靈，仍以1分優勢領跑，主場5戰全勝但只打進12球，周中歐冠客場補時破門2比1擊敗莫斯科火車頭已提前出線；後衛德利赫特因腳踝傷勢缺席周中歐冠，但他有望在聯賽中復出，後衛基耶利尼仍在養傷。

AC米蘭上輪主場1比2不敵拉齊奧，30年來首次在聯賽主場輸給對手，皮奧利上任後4輪1勝1平2負，其中客場1比2不敵羅馬；中場卡斯蒂列霍在上輪拉傷大腿肌肉要休養10天，前鋒蘇索和中場比格利亞因傷也可能繼續缺陣，今夏加盟的前鋒雷比奇本場或將進入首發陣容，但媒體披露他目前體重超標，此外還有媒體報道稱球員曾向技術總監馬爾蒂尼建議請回上賽季的主教練加圖索。

進球數推薦：【3、4】
