願你歷盡千帆歸來仍是少年，再見，永遠的金童永遠的托雷斯

2019-11-21 0條評論 14次閱讀 0人點贊

十歲那年，費爾南多·托雷斯就與卡爾德隆球場展開了不解的恩緣，他加入了馬德里競技少年隊，這個伊比利亞男孩就此開啟了成長之路。雖然當時托雷斯瘦小但是身體修長，速度飛快，在試訓中千里挑一被球探選中，很快他就就脫穎而出，開啟了他夢想征程的起點。

2001年5月，17歲的托雷斯在馬競與萊加內斯的比賽中替補出場，完成了職業生涯的處子秀，那時的馬競處於西乙聯賽，托雷斯的橫空出世讓俱樂部看到了希望。經過兩個賽季的蟄伏，馬競終於回到了西甲賽場上，那時隊中的老將對這個突然到來的小夥子並不友好，稱他為”那個小孩”，這也是托雷斯後來綽號”聖嬰”的由來。在當時的馬競主帥阿拉貢內斯的提拔下，他在自己的第二場西甲中就取得了進球，第一個西甲賽季中更是為馬競打入13個進球，幫助馬競畏懼聯賽11位，這對於升班馬來說，已經是一個很不錯的成績了

。

他那與年齡不相符合的成熟和球感，逐漸贏得了隊中大哥們的信任，而在恩師阿拉貢內斯的眼中，托雷斯還擁有著一些其他的特質，比如頭破血流不下火線的頑強品質、比如在對方挑釁下奮力還擊的領袖氣質，這些都讓托雷斯在馬競的地位漸漸提升。曾經極為靦腆的小男孩開始演變為大場面先生，在02/03賽季西甲20輪馬競3∶0巴薩的比賽中，托雷斯禁區內抹過德波爾射門得分的畫面依舊曆歷在目，這是他早期的代表作，從此他一步步成為了巴薩剋星。第二個賽季，托雷斯以年僅19歲成為了馬競歷史上最年輕的隊長，昔日的聖嬰已經成為了床單軍團的領導者。主教練換成了多年後執教中超的曼薩諾，他接受了阿拉貢內斯的建議，將托雷斯在球隊中的地位不斷提升。憑藉著出眾的速度和技術，少年隊長托雷斯開始兌現他的足球天賦，在接連不斷的貢獻世界波的同時，20歲的他在那個賽季成為了西甲的第三射手。出道即巔峰，這是托雷斯的寫照，一頭金髮的托雷斯陽光明媚，讓沉寂多年的卡爾德隆再次煥發除了活力。在第一段旅途中，托雷斯為馬競在西甲4個賽季出場145場，場場首發為球隊總共打入63球。

在06年世界盃一年後，托雷斯前往安菲爾德尋找新的挑戰，正像在卡爾德隆起步一樣，托雷斯很快就征服了安菲爾德的球迷，在加盟不到一個月的時間裡，托雷斯就為球隊貢獻了一個帽子戲法。在馬競不能參加的歐冠，托雷斯在這個新舞台也大放異彩，在小組賽對陣波爾圖打開了自己歐冠的進球賬戶。那時與傑拉德、阿隆索的配合尤為默契，那是一個最好的托雷斯，也是最完美的金童時代，那時的他也不過才23歲，利物浦球迷為他量身定做的”托雷斯之歌”數次響徹安菲爾德的夜空。為利物浦效力的四個賽季中，托雷斯為球隊帶來了65個進球，在多年未能獲得頂級聯賽冠軍的影響下，托雷斯選擇了加盟英超新貴切爾西。

隨著達格利什入主安菲爾德和蘇亞雷斯的加盟，托雷斯在2010年世界盃後以創造紀錄的5100萬英鎊的身價加盟了藍軍切爾西。在切爾西的托雷斯並不像之前的轉會那樣順利，或許是被高價的轉會費所束縛，在斯坦福橋的前三個月中，托雷斯未能取得一個進球，這與他在利物浦場均0.66球的數據大相徑庭，他一次次被提前換下，斯坦福橋的球迷也發出陣陣不和諧的噓聲，在加盟藍軍後的第14場比賽，托雷斯才為新東家打入1球。在第二賽季里，托雷斯在與曼聯的紅藍大戰里晃過德赫亞，面對空門時卻將足球踢飛，這個巨大的空門失誤也讓托雷斯背負著水貨的罵名，對手的嘲諷，自己的無奈，托雷斯在那天的老特拉福德，徹底的淪為了笑柄。然而，沉舟側畔千帆過，病樹前頭萬木春，在英超賽場不如意的托雷斯卻在歐冠賽場浴火重生，在歐冠半決賽次回合，再次上演了巴薩剋星，托雷斯的絕殺，親手淘汰歐冠冠軍巴薩，晉級決賽。在決賽中，第88分鐘托雷斯在前場拼到一個角球，切爾西憑藉著德羅巴的角球進球扳平比分，並在點球大戰中贏了拜仁，成功登頂歐洲。在切爾西與馬競的歐冠比賽中，托雷斯打入一球卻選擇舉起雙手不慶祝，以顯示他對老東家的尊重，在馬競球迷眼中或許永遠都是那個長不大的孩子，終有一天，卡爾德隆的大門會再次為聖嬰而開。

在切爾西不算順利的職業生涯後，他輾轉AC米蘭，十場僅入1球的尷尬數據表明他只是聖西羅的匆匆過客。在他視線的遠處，是那夢開始的地方，2015年1月一條回家之路就這樣突然開啟，托雷斯重新加盟馬德里競技，身披19號戰衣，迎接他的是卡爾德隆45000名球迷山呼海嘯的歡迎聲，願你歷盡千帆，歸來仍是少年，這是很多人對托雷斯離去多年的真實感言。那個含著手指慶祝的金髮少年重新回到了人們的視野當年托雷斯的隊友西蒙尼已成為了他的主帥，這段隊友變師徒的傳奇故事又將他們的緣分繼續。在西蒙尼的手下，托雷斯恢復到了年輕時的最佳狀態，在國王杯主場對巴薩的進球後親吻卡爾德隆草地的一幕，更是令人為之動容，卡爾德隆的記憶永遠溫暖人心。儘管在回歸馬競後只能作為一名替補球員登場，但他一直不辱使命，他的進球也會在適時的時候到來，在歐冠淘汰賽中再一次成為巴薩的夢魘，他的進球成功幫助馬競再一次殺入歐冠決賽，可惜的是在決賽中點球大戰中惜敗皇馬，終究未能幫馬競帶來一座歐冠獎盃。2018年5月，當托雷斯最後一次走進卡爾德隆球場，全場氣氛令人震撼，最後一次代表馬競出場，金童打進2球，與自己輝煌的馬競生涯做出了告別。

有些驚訝的是，托雷斯選擇了日本J聯賽的鳥棲沙岩做為自己職業生涯的最後一站，在那個遙遠的陌生國度，隨著年齡的增長，傷病的增多，昔日的金童再也不復當年之勇，在那裡，他打入職業生涯的第300球，並在退役前10天打入生涯最後1球——301球。就像一個完美的終點，35歲的托雷斯走過時光，走過青春，他曾經盛名之下難負眾望，他也曾經陰霾過後重現笑容，那著名的”托雷斯之歌”從安菲爾德傳到卡爾德隆，再傳到鳥棲沙岩，一直傳到了告別時刻。
