卡利亞里對桑普多利亞和瓦斯科達伽對克魯塞羅

2019-12-02 0條評論 16次閱讀 0人點贊

卡利亞里對桑普多利亞

時間 3日03:45

卡利亞里13輪戰罷拿到25分，以勝負關係優勢力壓同積25分的羅馬和亞特蘭大暫排意甲第4，較上賽季同期多拿10分，打進25顆進球，火力僅次於尤文、拉齊奧以及國米。上一個主場5-2力克佛羅倫薩，近期主場狀態極好，主場防守實屬一般，新賽季加上熱身賽11個主場只有一場零封，其中面對進攻低下的布雷西亞和維羅納都能丟球，導致無法贏球。但球隊仍保持著11輪意甲不敗紀錄，表現僅次於榜首的尤文。在意甲屬於很有球，本賽季14場正賽多達12場至少2球，且無1場蛋蛋，在意甲屬於很良心的球隊。

桑普多利亞13輪過後拿到了12分暫列意甲第17位，少賽1場領先倒數第3的熱那亞2分，較上賽季同期少拿4分。桑普多利亞聯賽場均有14.7腳射門，而卡利亞里聯賽場均只射11腳，暫為意甲第2低。上輪憑藉多踢一人的優勢，主場逆轉擊敗烏迪內斯，連續4輪聯賽保持不敗，目前已經暫時遠離了降級區。近期反彈力度不低，近4場意甲2勝2平保持不敗，而之前5場慘遭1平4負。球隊近3場比賽指數全勝。桑普多利亞之前6輪聯賽有5場丟球，而近3輪聯賽有2場零封，防守狀態有一定起色。可是桑普多利亞客場能力低下，本賽季6個意甲輸了5場，唯一不敗對手還是倒數第2的斯帕爾，含金量也很低，跨賽季7個客場聯賽無法在上半場取得進球。

雙方球隊在最近的10場交鋒下，主隊的卡利亞里1勝4平5負，劣勢極大。上賽季主場0-0與桑普多利亞互交白卷。但是卡利亞里本賽季強勢的表現，鋒線火力強勢，加上最近10場主場沒有輸過桑普多利亞，這場是單關，所以我選平局。

達伽馬對克魯塞羅

時間 3日07:00

達伽馬上輪客場0-1不敵聖保羅，最近3輪首次輸球，目前以積44分的成績排名第12位，只要保住當下排名，達伽馬就將獲得下賽季南美俱樂部資格賽席位，而如今聯賽只有最後3輪，本輪坐鎮主場迎戰積36分排名第17位的克魯塞羅，本賽季首回合，達伽馬曾客場0-1失利，上賽季聯賽主場交手2-0勝出。然而，自2007年以來，達伽馬最近10次主場對陣克魯塞羅只取得了2勝1平7負的糟糕成績，而本賽季此前17個聯賽主場，達伽馬只拿23分，主場成績排名巴甲第17位。本輪雖說常規主力均可出戰，但想要全取3分，難度勢必不低。

克魯塞羅上輪主場0-1不敵升班馬阿拉戈斯體育中心，這是一個災難性賽果，因為對手同為保級球隊，而在聯賽最後3輪的情況下，克魯塞羅排名倒數第4位，恰好處於降級區，在其前方有37分的塞阿拉、積41分的弗魯米嫩塞。本輪克魯塞羅一旦客場不敵達伽馬，保級形勢將會更加嚴峻，對於這支老牌勁旅來說，走到如今這步的主要原因是球隊勝率是在太低，雖說前35輪只輸13場，比排名第8位的巴西國際也只多輸1場，但如此同時卻收穫了多達15場平局。本輪客戰達伽馬，對方仍有較強的戰意，克魯塞羅必將面臨一場惡戰。人員方面，除了羅德里吉尼奧傷休，其他都能出場，陣容齊整是克魯塞羅的唯一利好。

綜合各方面情況來看，要我推薦就是克魯塞羅客場贏球，最少也能客場帶走一分。

個人觀點，僅供參考。
