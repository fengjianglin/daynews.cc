西媒：皇馬早已看上瓦倫西亞希望之星！曼聯主帥暗示當紅新星加盟

2019-12-17 1,914條評論 4,490次閱讀 553人點贊

阿斯報消息，瓦倫西亞神童費蘭-托雷斯長期以來都是皇馬的目標，並且銀河戰艦已然三顧茅廬求購這名未來之星。

報道稱，儘管只有19歲，但托雷斯已經登上了皇馬的引援名單，事實上馬德里豪門在這名小將12歲時就曾嘗試簽下他，並且在其16歲時也曾敲響過球員的大門（甚至邀請託雷斯來皇馬訓練過），而皇馬最後一次嘗試收購托雷斯則是西班牙小將2017年在印度參加u17世界盃時。

阿斯報指出，銀河戰艦的舉動造成的直接後果就是瓦倫西亞將托雷斯迅速提拔進了一線隊，並且通過兩次操作將球員的違約金從800萬歐元提高到了1億歐元，現在蝙蝠軍團正在繼續嘗試和托雷斯續約，因為球員當下的合同2021年就要到期。

所謂英雄出少年，托雷斯此前被評為了u19歐青賽的最佳球員，並且這名新星在過去的7場比賽中貢獻了4球2助攻，西班牙人至今已經代表瓦倫西亞出戰了72場比賽——要知道現在托雷斯才只有19歲而已，在蝙蝠軍團100年的歷史上，還沒有其他年輕人在這個年齡就出戰如此多的比賽。

所以，這樣的希望之星引起皇馬的關注毫不奇怪，並且未來恐怕還會招來其他豪門進一步的追求。

托雷斯確實是西班牙未來的希望之星，皇馬看上他是不奇怪的，但目前還沒有很可靠的報道，看看再說吧。

都靈體育報消息，巴黎有意在一月份買斷租借自國米的前鋒伊卡爾迪，不過球員的妻子兼經紀人旺達-娜拉希望皇馬和尤文也能追求這名前鋒。

伊卡爾迪今夏從國米租借加盟了巴黎，目前來看這筆交易屬於三贏，球員得到了比之前多一倍的薪水，並且阿根廷人在10場法甲比賽中已經打進8球，發揮出色，鑒於此，巴黎很可能會用這名26歲的球星來取代日漸老邁的卡瓦尼，並且他們可以在冬窗激活伊卡爾迪價值5800萬英鎊的買斷條款。

不過根據都靈體育報的說法，娜拉依然在為他的丈夫尋找其他選擇，尤其是她希望伊卡爾迪可以引起皇馬和尤文的興趣，但巴黎體育總監萊昂納多很可能會迫使球員方面儘快做出決定。

伊卡爾迪的職業生涯顯示他是一名非常穩定的高產射手，阿根廷前鋒在代表國米出戰的219場比賽中打進124球，其中在2017-18賽季的34場意甲比賽中就打進了29球。

娜拉確實是個很會搞事的女人，不過這個傳聞不靠譜。

而且皇馬對伊卡爾迪是沒興趣的，很顯然。

索爾斯克亞表示，曼聯目標小哈蘭德已經就未來做出了決定，並且球員「知道自己想要什麼」。

根據挪威媒體VG的報道，索爾斯克亞在上周五早上飛赴薩爾茨堡會見了小哈蘭德，一家私人飛機先是從東米德蘭機場抵達了曼徹斯特，隨後曼聯主帥在早上06：49分乘坐該飛機前往了奧地利，報道稱，挪威人此行是為了說服19歲的小哈蘭德在未來加盟曼聯，最近有傳聞表示球員已經前往德國和多特蒙德以及萊比錫紅牛有過會談。

這家媒體進一步指出，索爾斯克亞在午餐時間乘坐飛機返回了曼徹斯特，而在曼聯1-1戰平埃弗頓的比賽後，紅魔主帥也被問到了這個問題。

對此，索爾斯克亞說：「我不會對傳聞發表評論，就是這樣。小哈蘭德知道自己想要什麼，並且也知道自己將會怎麼做。」

本賽季，小哈蘭德在22場比賽中打進了28球，而曼聯的優先任務就是在接下來的轉會窗引進新前鋒和中場。

也就是說索帥真的沒有否認去奧地利會見小哈蘭德的傳聞，看起來確實不排除他會加盟曼聯的……

馬卡報消息，拉基蒂奇對自己在巴薩的未來已經有頗多質疑，但球員還是決定至少在諾坎普留到本賽季結束。

拉基蒂奇賽季初期明顯淡出了巴薩的首發陣容，報道稱球員此前也在嚴肅考慮於一月份離隊的事宜，不過值得注意的是，克羅埃西亞人最近接連首發參加了對多特蒙德、馬競、馬洛卡、國米以及皇家社會的比賽，並且表現也相當不錯，目前所有的跡象都顯示，拉基蒂奇可能會重新回到巴爾韋德的計劃中。

馬卡報表示，事實上拉基蒂奇其實一直都在強調，他非常希望能繼續為巴薩效力，鑒於球員現在再次成為了球隊倚重的人，一月份轉會離隊看來已經不在這名中場的計劃中。

米蘭經理博班在俱樂部120周年慶典上接受了義大利天空體育的採訪，並且談到了有關伊布的話題。

博班說：「我很榮幸能成為米蘭歷史的一部分，俱樂部會不會在轉會市場上帶來一個禮物？我不知道，今天我還和馬爾蒂尼聊過，我們正在觀察好幾個位置，球隊會看看哪些地方是需要補充的。

伊布？這個問題上我和馬爾蒂尼的進展是一樣的，是不是這筆交易現在可能性越來越小了？是的，這就是現實。」

太陽報消息，切爾西有意在一月份收購本-懷特，這名中衛本賽季從布萊頓租借效力於利茲聯，並發揮令人印象深刻。

懷特目前的合同2022年到期，並且布萊頓並不想失去這名極具天賦的後衛，不過報道稱，假如切爾西願意支付2500萬英鎊的報價的話，他們也願意考慮放人。

本賽季，藍軍已經在英超中丟掉了25球，而22歲的懷特被認為是可以加強切爾西防線的理想人選，這名小將在16歲時遭到了南安普頓的解約，不過布萊頓當時為其提供了一份學徒合同，現在來看，海鷗軍團對球員的信任已經收穫了回報，並且目前為止，懷特已代表布萊頓完成了2次一線隊亮相，2次都是在2016年的卡拉寶杯中。

值得一提的是，雖然在場上主要司職中衛，但假如有需要的話，懷特也可以勝任右後衛位置，而這名球員的潛力和年齡看起來十分符合蘭帕德的選材標準。
