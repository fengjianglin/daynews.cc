歐聯看點：武磊缺席，曼聯全隊輪休，一豪門需防冷

2019-11-28 0條評論 17次閱讀 0人點贊

　　北京時間11月28日至29日凌晨，歐聯杯小組賽倒數第二輪繼續開戰。本賽季狀態起伏的曼聯將客場挑戰阿斯塔納；武磊所在的西班牙人做客費倫茨瓦羅斯；近期狀態不佳的阿森納跟法蘭克福上演強強對話；意甲勁旅羅馬客場迎戰伊斯坦布爾。

　　【阿森納VS法蘭克福】

　　比賽時間：11月29日凌晨04：00

　　阿森納近期狀態可謂是跌入谷底，上輪聯賽主場2-2戰平南安普頓之後連續5場不勝。排名也跌落到第八的位置，且落後前4多達8分，爭4難度不小。阿森納新賽季防守很一般，本賽季13場英超也就零封2場，其中1場還是進攻魚腩紐卡，此外面對降級區的沃特福德和維拉每場都能丟2球，此外近2場歐聯杯麵對葡超中游的吉馬良斯也連續丟球。

　　球隊戰績不佳，球員跟主教練的關係也變得微妙起來。在上周六與南安普頓的比賽中，阿森納球迷高呼「我們希望埃梅里滾蛋」，儘管拉卡澤特在傷停補時階段為球隊扳平比分，但是終場哨想起時，球迷發出了巨大的噓聲。據多家媒體報導，如果阿森納在對陣法蘭克福的比賽中失利，埃梅里的位置將會面臨巨大威脅，這將是他最後一根救命稻草。

　　法蘭克福本賽季狀態起伏不定，新賽季在德甲聯賽中僅取得5勝2平5負的糟糕戰績，輸球率達到42%，而歐聯杯則有50%。在近3場正賽連續輸球後，對球隊士氣的打擊不小。需要注意的是法蘭克福本賽季客場戰績僅為1勝4負，此外上個歐聯杯客場也負於比甲強隊標準列日，導致出線形勢也不穩，但是法蘭克福這支球隊自帶爆冷屬性，德甲第十輪5-1屠殺拜仁讓人大跌眼鏡，此番客戰挑戰阿森納，雖然取勝難度不小，但也並非全無機會。

　　【阿斯塔納VS曼聯】

　　比賽時間：11月28日23:50

　　哈薩克超聯賽兩周前已經落下帷幕，雖然阿斯坦納末輪主場0-2不敵伊特什，但他們此前已經提前鎖定聯賽冠軍，實現了聯賽4連霸。不過阿斯坦納在歐戰也就是三流球隊，該隊作為歐冠資格賽球隊，首輪就被克魯日淘汰，被迫打歐聯，而到歐聯小組賽4場全敗，這四場比賽中僅打進1球，丟球數高達14粒，近期2回合面對阿爾克馬亞狂失11球，阿斯坦納歐戰之旅也提前結束了，已經提前兩輪無緣淘汰賽，難有戰意。

　　曼聯上一輪小組賽3-0輕鬆戰勝貝爾格萊紅星之後已經提前取得晉級名額，所以對於本場比賽的勝負並無重視，此番出征哈薩克，索爾斯克亞選擇讓所有主力球員留在家裡休戰，帶上了青年隊為主的陣容出戰，由此可見，本輪的歐聯只是例行公事而已。

　　賽前發布會上，索爾斯克亞早早公布球隊首發的七名球員，準備明牌出戰。在索帥公布的7名先發球員分別是老門將李·格蘭特、右後衛伊森·萊爾德、中衛迪肖恩·伯納德和圖安澤比、左後衛盧克·肖、中場林加德和萊維特。而詹姆斯·迦納、安赫爾·戈麥斯、庄塔西和梅森·格林伍德，預計也會先發。

　　【費倫茨瓦羅斯VS西班牙人】

　　比賽時間：11月29日凌晨1:55

　　武磊所在的西班牙人本賽季聯賽雖然戰績不佳，深陷降級區，但是在歐聯杯賽場上非常兇猛。在上一輪小組賽6-0橫掃盧多戈雷茨之後已經提前鎖定出線資格，為止本場比賽主教練馬欽也是輪休一部分主力留力聯賽。

　　值得一提的是，在西班牙人的輪休名單中，武磊在列。在上輪聯賽赫塔費的比賽中，武磊首發出場並打滿全場，為球隊攻入了至關重要的進球，也在賽後獲得了全場最佳球員的榮譽。毫無疑問，武磊近來神勇的發揮，征服了球隊新任主帥馬欽，逐漸奪回球隊主力前鋒的位置。而在球隊聯賽保級形勢艱難的情況下，馬欽也勢必要把更多精力放在聯賽當中！

　　費倫茨瓦羅斯目前名列匈甲聯賽第一，主場5勝1平，很是強勢。小組賽雖然暫列第三，但首戰西班牙人卻是主場1比1戰平，這也是雙方歷史上唯一的交鋒記錄。目前費倫茨瓦羅斯依舊擁有出線希望，主場面對無欲無求的西班牙人，贏球機會很大。
