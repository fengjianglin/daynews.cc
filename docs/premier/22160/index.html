國米又要坑盟友？意媒給球迷吃定心丸，孔蒂收到好消息

2019-12-19 227條評論 2,729次閱讀 479人點贊

國米這回似乎又要坑盟友帕爾馬了，為什麼要說「又」呢？因為帕爾馬之前從藍黑軍團簽下的卡拉莫實在有點坑，先是嚴重違規，狀態不穩定還有玻璃屬性。即使是這樣，帕爾馬仍然要付出超過千萬歐元的買斷費，確實有點虧。

而根據意媒最新的消息，國米不但想帶走帕爾馬目前的大腿庫盧塞夫斯基，還試圖挖走對方的體育總監法賈諾，他們有意為這位極有能力的人才提供一個更好的平台。值得一提的是意媒還透露法賈諾和孔蒂關係極好，今年夏天羅馬也想挖走他，但是失敗了。

帕爾馬本賽季保級無憂，甚至還有機會進軍下賽季的歐洲賽場，這也是他們想把庫盧塞夫斯基留到賽季結束的主要原因。事實上就算帕爾馬願意放人，亞特蘭大也不會讓國米撿便宜，他們希望能從這位新星身上得到足夠的回報。而庫盧塞夫斯基的經紀人也在不停地用意甲其它球隊和英超豪門來給自己的客戶抬價。總結下來就是國米想拿下他必須要大出血。

另外法賈諾確實很有實力，不過國米現任體育總監奧西利奧剛剛續約，他也為俱樂部做出了不少貢獻，真的沒必要換人。不知道除了體育總監這個職位，國米還能安排什麼樣的工作給法賈諾？

雖然意媒很喜歡爆料國米的負面消息，不過他們最近連續爆出了有關這支球隊的利好，也給藍黑軍團的球迷吃了定心丸。《米蘭體育報》表示馬洛塔手握五千萬歐元的冬窗預算，他很可能會給孔蒂帶來多名強援。《羅馬體育報》表示國米和烏迪內斯的談判非常順利，德保羅很可能會在明年夏天加盟。還有多家媒體表示比達爾吵著鬧著也要來投奔孔蒂。

總之在意媒這些記者的口中，國米形勢一片大好，孔蒂不用擔心無人可用了，球迷們也不用擔心這支球隊會再次在下半程掉鏈子了。雖然明知這些意媒不是太靠譜，但還是希望他們說的這些都能實現。

再厲害的強援現在也來不了，孔蒂只能依靠現有的球員來應付下一場對熱那亞的比賽，不過他今天收到了好消息。根據最新的消息，巴斯托尼和坎德雷瓦已經恢復健康，並參加了合練，之前肌肉疲勞的巴萊羅則在進行個人訓練，他有一定機會出戰下一場比賽。
