沒機會了！熱刺冬窗欲脫手埃里克森 標價不到4000萬鎊

2019-12-05 0條評論 13次閱讀 0人點贊

儘管穆里尼奧成為了熱刺的新任主帥，但這並沒有改變熱刺大腦埃里克森想要離開球隊的想法。根據英國媒體《倫敦晚旗報》的最新消息，熱刺主席列維願意在1月以低於4000萬鎊的價格出售埃里克森。《倫敦晚旗報》還指出，埃里克森的下家極有很能會從西甲豪門皇馬或者是意甲班霸尤文中產生。

眾所周知的是，埃里克森與熱刺之間的合同將在本賽季結束後到期，儘管俱樂部此前開出了周薪超過20萬鎊的新合同，這個收入也讓埃里克森成為俱樂部工資最高的球員之一，但有消息稱，這位丹麥中場依舊拒絕續約。在埃里克森看來，他想去追求新的挑戰，最好是意甲或是西甲。

值得一提的是，熱刺主席列維原本是打算在今年夏天以8000萬鎊的轉會費將埃里克森賣給曼聯的，但最終這筆買賣無疾而終。《倫敦晚旗報》指出，列維決心不讓埃里克森在明年夏天自由轉會，並表示這位丹麥中場能夠在1月份低價離隊。

在這種情況下，只要有球隊願意在冬窗出價4000萬英鎊左右，那麼列維就會考慮放人。目前還不清楚是否會有球隊在冬季轉會窗口求購埃里克森，因為這位丹麥國腳在明年夏天能夠以自由轉會的方式離開。此外，在明年1月份的時候，埃里克森還可以和英格蘭之外的俱樂部簽署預備合同。
